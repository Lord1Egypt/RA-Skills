## Description: <br>
AI project management powered by CellCog. Knowledge workspaces, document upload, AI-processed context trees, signed URL retrieval. Works standalone or as CellCog chat context. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and external agents use this skill to create CellCog project workspaces, upload and organize documents, read AI-processed context trees, generate signed document URLs, and supply project context to CellCog chats. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Selected project documents may be sent to CellCog for processing and persistent project context. <br>
Mitigation: Confirm before sharing confidential documents or starting CellCog chats over sensitive project data. <br>
Risk: Signed document URLs grant temporary unauthenticated access and remain valid until expiration. <br>
Mitigation: Use the shortest practical expiration and avoid posting signed URLs in public logs or shared chats. <br>
Risk: The skill can guide project and document deletion actions. <br>
Mitigation: Confirm destructive operations with the user before deleting documents or projects. <br>
