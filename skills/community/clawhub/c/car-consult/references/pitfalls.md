# 避坑指南

## 一、4S店常见套路

### 价格套路

**1. 低价引流，强推高配**
- 表现：广告价是低配，进店说"没车"
- 应对：坚持要广告车型，或要求书面确认无车

**2. 落地价含糊**
- 表现：只报裸车价，不提其他费用
- 应对：坚持问"落地价一共多少"，要求明细

**3. 贷款"免息"陷阱**
- 表现：说免息，但有高额手续费
- 应对：问清楚总还款额、手续费、实际利率

**4. 加装礼包水分大**
- 表现：送"万元礼包"，实际成本低
- 应对：折现或要求具体品牌型号

### 合同套路

**1. 定金不退**
- 表现：口头说可退，合同写"定金"
- 应对：坚持写"订金"或注明可退条件

**2. 交付时间模糊**
- 表现：合同不写具体日期
- 应对：要求写明日期+逾期赔偿

**3. 赠送项目不写进合同**
- 表现：口头承诺，提车时赖账
- 应对：所有赠送写进合同

### 提车套路

**1. 库存车当新车卖**
- 表现：库存超过6个月的车
- 应对：检查轮胎生产日期，要求额外优惠

**2. 展车/试驾车当新车**
- 表现：里程少但有使用痕迹
- 应对：检查座椅磨损、漆面状况

**3. 强制店内上牌/保险**
- 表现：说"外面办不了"
- 应对：可自己上牌/买保险，或谈价格

## 二、新能源专属陷阱

### 续航虚标
- **表现**：CLTC续航600km，实际只能跑400km
- **真相**：CLTC工况偏理想，实际打6-8折
- **应对**：看车主实测，按70%估算

### 电池衰减
- **表现**：几年后续航明显下降
- **真相**：三元锂衰减快于磷酸铁锂
- **应对**：选质保政策好的品牌（首任车主终身质保）

### 保费暴涨
- **表现**：第二年保费涨很多
- **真相**：新能源车险定价高，出险更贵
- **应对**：对比多家保险公司，安全驾驶

### 充电不便
- **表现**：没家充，快充排队
- **应对**：提车前确认充电条件

### 品牌跑路
- **表现**：新势力品牌倒闭，售后无门
- **应对**：
  - 选有车企背景的品牌
  - 看月销量（5000+安全线）
  - 避免首款/小众车型

## 三、贷款陷阱

### 利率障眼法
| 说法 | 实际 | 计算 |
|------|------|------|
| "年费率3%" | 实际年化约5.5% | 费率≠利率 |
| "两年免息" | 可能有手续费 | 问总还款额 |

### 计算真实利率
```
真实年化利率 ≈ 月供 × 期数 - 贷款额 ÷ 贷款额 × 2
```

示例：贷款10万，36期，月供3100
```
真实利率 ≈ (3100 × 36 - 100000) ÷ 100000 × 2 ≈ 11.5%
```

### 建议
- 能全款就全款
- 对比银行车贷 vs 4S店贷款
- 问清楚：总利息 + 手续费 + 提前还款违约金

## 四、验车陷阱

### 外观检查
- [ ] 光线充足处看漆面
- [ ] 检查划痕、凹陷、色差
- [ ] 检查轮胎生产日期（DOT后4位：周+年）

### 内饰检查
- [ ] 座椅有无磨损
- [ ] 各功能按键测试
- [ ] 空调、音响、车窗

### 新能源专属
- [ ] 电池健康度
- [ ] 充电口盖开合
- [ ] 充电线缆齐全

### 证件检查
- [ ] 车辆合格证
- [ ] 购车发票
- [ ] 保修手册
- [ ] 两把钥匙

## 五、其他注意事项

### 保养套路
- **表现**：缩短保养周期，推荐"深度保养"
- **应对**：按说明书保养周期，拒绝过度保养

### 质保陷阱
- **表现**：说"终身质保"，但限制条件多
- **应对**：看清质保条款，哪些不在范围内

### 二手车风险（新能源重点）

#### 电池陷阱
- **表现**：标称续航500km，实测只能跑300km
- **真相**：电池健康度（SOH）是二手新能源最核心的估值因素
- **应对**：
  - 要求看第三方电池检测报告（查SOH、充电次数、快充占比）
  - 去官方售后做免费电池检测
  - 了解该车型电池更换成本（部分车型换电池≈半台车价）
  - 优先选磷酸铁锂（寿命更长）而非三元锂
  - 快充占比高的车电池衰减通常更严重

#### 质保延续陷阱
- **表现**："首任车主终身质保"听起来好，但过户后失效
- **真相**：大部分品牌质保不随车转移，或只有8年/15万公里基础质保
- **应对**：
  - 问清剩余的厂家质保期限和覆盖范围
  - 了解官方认证二手车（CPO）是否额外提供延保
  - 特斯拉、蔚来等部分品牌支持质保随车

#### 调表/隐瞒事故
- **表现**：表显里程少，实际已经跑了更多
- **应对**：
  - 检查轮胎磨损/刹车盘磨损与里程是否一致
  - 查4S店维保记录（出险记录）
  - 新能源车可查电池充电记录（很多品牌APP可查）
  - 第三方检测机构（查博士等）做全车检测

#### 平台交易风险
| 渠道 | 风险 | 建议 |
|------|------|------|
| 瓜子二手车 | 部分车源有"保障服务费"隐性成本 | 问清总费用，对比线下 |
| 懂车帝二手车 | 个人车源较多，无平台保障 | 坚持走第三方检测 |
| 车商/个人交易 | 无保障，事故车/泡水车风险最高 | 必须找独立检测机构 |
| 品牌官方认证二手 | 价格偏高，但有质保 | 最省心，适合第一次买二手 |

#### 新能源二手车的"甜区"
- **车龄**：3-5年（首任车主快速折旧期已过，残值趋于稳定）
- **里程**：4-8万公里（电池一般还在健康期）
- **品牌**：特斯拉/比亚迪的二手市场最成熟，估值最透明
- **价格带**：8-15万是二手新能源的甜区——这个价位可以买到原价20-30万的车

#### 新能源二手 vs 燃油二手
- 贬得快，但购入价也低 —— 如果开到报废，二手新能源可能比买新的划算
- 燃油二手没有电池焦虑，但同价位可能车龄更老/配置更低
- 政策风险：部分老款新能源不再享受路权优惠（需查当地政策）

#### 建议优先级
1. 品牌官方认证二手车（最省心）
2. 大平台（瓜子/懂车帝）+ 第三方检测
3. 个人车源 + 第三方检测 + 查维保记录
4. 车商/二手市场 — 风险最高，不推荐新手
