# 法律尽职调查清单（DD Checklist）

> 本清单供起草调查报告时交叉检查，确保各模块无遗漏。

## 基础文件
- [ ] 目标公司营业执照、公司章程（历次修正案）
- [ ] 发起人协议/股东协议
- [ ] 工商档案（全套）
- [ ] 股东名册及出资证明
- [ ] 批准证书/行业准入文件
- [ ] 组织机构代码证（已合并）

## 股权及公司治理
- [ ] 股权结构图（穿透披露至最终受益人）
- [ ] 股权质押/冻结/查封查询结果
- [ ] 股东会/董事会/监事会议事规则及会议记录
- [ ] 董事、监事、高管名单及任职资格
- [ ] 一致行动协议/表决权委托协议
- [ ] 对赌协议/估值调整协议

## 业务及资质
- [ ] 主营业务说明
- [ ] 经营许可证/资质证书清单
- [ ] 特许经营合同
- [ ] 重大供应商合同（前五名）
- [ ] 重大客户合同（前五名）
- [ ] 代理/经销协议

## 资产
- [ ] 不动产清单及权属证书
- [ ] 租赁合同（出租及承租）
- [ ] 知识产权清单（专利、商标、著作权、域名）
- [ ] 知识产权年费缴纳凭证
- [ ] 知识产权许可/转让合同
- [ ] 主要设备清单

## 财务及税务
- [ ] 审计报告（近三年）
- [ ] 银行借款合同及还款计划
- [ ] 对外担保合同
- [ ] 纳税申报表（近三年）
- [ ] 税收优惠批准文件
- [ ] 税务稽查/自查报告

## 劳动人事
- [ ] 员工花名册
- [ ] 劳动合同样本
- [ ] 社保/公积金缴纳凭证
- [ ] 高管劳动合同及竞业限制协议
- [ ] 保密协议
- [ ] 股权激励方案及协议

## 合规
- [ ] 诉讼/仲裁/执行案件清单
- [ ] 行政处罚决定书
- [ ] 环保批文（环评、验收、排污许可）
- [ ] 安全生产许可证
- [ ] 消防验收意见书
- [ ] 进出口经营备案

## 关联交易
- [ ] 关联方清单
- [ ] 关联交易合同
- [ ] 关联交易内部审批文件
- [ ] 关联方资金往来明细
