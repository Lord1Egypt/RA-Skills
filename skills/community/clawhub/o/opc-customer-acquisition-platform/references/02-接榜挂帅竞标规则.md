# 接榜挂帅竞标规则

> 本文档详细说明竞标流程、评标标准、挂帅人机制和流标处理方式。

---

## 一、竞标模式详解

### 1.1 四种竞标模式对比

| 竞标类型 | 适用场景 | 竞标方数 | 竞标内容 | 时长 | 适用等级 |
|---------|---------|---------|---------|------|----------|
| **快速抢单** | 单一任务、紧急需求 | 不限 | 直接报价 | 24小时 | D级 |
| **方案竞标** | 有一定复杂度的需求 | 3-5家 | 方案+报价 | 3-5天 | C级 |
| **综合竞标** | 常规项目需求 | 5-8家 | 方案+案例+报价+信用 | 7-10天 | B级 |
| **联合体竞标** | 大型复杂项目 | 2-3组 | 团队方案+分工+报价 | 10-15天 | A级/S级 |

### 1.2 快速抢单模式（D级）

**适用条件**：
- 项目金额 < 5,000元
- 单一交付物
- 周期 < 7天

**流程**：
```
1. 项目发布 → 系统自动进入【快速抢单】状态
2. 竞标开放 → 符合条件的成员可直接抢单
3. 资格预审 → 系统自动验证信用等级（≥C级）
4. 抢单成功 → 先到先得，自动锁定
5. 确认接单 → 抢单成功后2小时内确认
6. 启动执行 → 进入【执行交付】环节
```

**规则**：
- 同一项目每人只能抢单1次
- 抢单后2小时内未确认，自动释放
- 被释放的项目可重新抢单
- 累计3次抢单后放弃，信用扣分

### 1.3 方案竞标模式（C级）

**适用条件**：
- 项目金额 5千-2万元
- 2-3个交付物
- 周期 7-15天

**流程**：
```
1. 项目发布 → 进入【方案竞标】状态
2. 竞标公告 → 公示期3天
3. 方案提交 → 竞标方提交竞标书（方案+报价）
4. 方案评审 → 评审委员会评审
5. 结果公示 → 公示期1天
6. 确认挂帅 → 挂帅人确认
7. 启动执行 → 进入【项目分包】环节
```

**竞标书要求**：
- 报价单（含分项报价）
- 初步执行方案
- 进度计划
- 团队介绍（如有）

### 1.4 综合竞标模式（B级）

**适用条件**：
- 项目金额 2万-10万元
- 3个以上交付物
- 周期 15-30天

**流程**：
```
1. 项目发布 → 进入【综合竞标】状态
2. 资质预审 → 竞标方需提前报名，通过资质预审
3. 竞标公告 → 公示期5天
4. 方案提交 → 提交完整竞标书
5. 初审 → 淘汰不符合条件的方案
6. 复审 → 进入方案评审（可安排答辩）
7. 综合评分 → 按评标标准打分
8. 结果公示 → 公示期2天
9. 确认挂帅 → 挂帅人确认
10. 启动执行 → 进入【项目分包】环节
```

**竞标书要求**：
- 公司/团队介绍
- 类似项目案例（≥3个）
- 完整执行方案
- 详细报价单
- 团队成员简历
- 信用授权书

### 1.5 联合体竞标模式（A级/S级）

**适用条件**：
- 项目金额 > 10万元
- 多模块、大团队
- 周期 > 30天

**流程**：
```
1. 项目发布 → 进入【联合体竞标】状态
2. 联合体招募 → 符合条件的团队组队
3. 联合体报名 → 提交联合体申请书
4. 资质审核 → 审核联合体整体资质
5. 方案提交 → 提交联合体竞标方案
6. 方案评审 → 评审委员会评审+答辩
7. 综合评分 → 按评标标准打分
8. 结果公示 → 公示期3天
9. 联合体确认 → 确认联合体牵头方
10. 启动分包 → 进入【项目分包】环节
```

**联合体申请书要求**：
- 联合体章程
- 各成员介绍
- 分工方案
- 责任划分
- 收益分配方案

## 二、评标标准（5维模型）

### 2.1 评分体系

| 评分维度 | 权重 | 评分项 | 数据来源 | 满分 |
|---------|-----|-------|---------|------|
| **信用评级** | 30% | 信用等级 | 股权合作Skill-信用台账 | 30分 |
| **历史业绩** | 25% | 项目数量+好评率 | 股权合作Skill-贡献台账 | 25分 |
| **方案质量** | 25% | 完整性+创新性+可行性 | 评标委员会 | 25分 |
| **报价合理性** | 15% | 与指导价对比 | 获客分包平台-指导价体系 | 15分 |
| **时间承诺** | 5% | 是否满足工期要求 | 竞标书 | 5分 |

### 2.2 各维度评分细则

#### 信用评级得分（30分）

| 信用等级 | 等级系数 | 最终得分 | 加分项 |
|---------|---------|---------|--------|
| **S级** | 1.2 | 30×1.2=36分（封顶30分） | 无需加分 |
| **A级** | 1.0 | 30×1.0=30分 | 无需加分 |
| **B级** | 0.9 | 30×0.9=27分 | 无需加分 |
| **C级** | 0.7 | 30×0.7=21分 | 无需加分 |
| **D级** | 0.5 | 30×0.5=15分 | 无资格参与 |

**加分项（最高+5分）**：
| 加分项 | 条件 | 加分 |
|--------|------|------|
| 近期无投诉 | 近3个月内无投诉记录 | +2分 |
| 连续履约 | 连续完成3个项目以上 | +3分 |
| 特殊贡献 | 为社区做出特殊贡献（经认定） | +2分 |

#### 历史业绩得分（25分）

| 评分项 | 评分标准 | 得分范围 |
|--------|----------|----------|
| **同类项目数量** | 每有1个同类项目 +3分 | 0-15分 |
| **好评率** | 每10%好评率 +2分 | 0-10分 |
| **扣分记录** | 每有1次扣分记录 -5分 | -∞~10分 |

**示例计算**：
- 同类项目3个 → 3×3=9分
- 好评率90% → 9×2=18分（封顶10分）
- 扣分记录0次 → 0分
- **历史业绩得分** = 9+10 = 19分

#### 方案质量得分（25分）

| 评分项 | 满分 | 评分标准 |
|--------|-----|----------|
| **方案完整性** | 10分 | 包含所有必要模块：背景分析、需求理解、执行方案、风险预案 |
| **创新性** | 8分 | 有独特的解决思路或增值建议 |
| **可行性** | 7分 | 方案可落地执行，有明确的里程碑 |

**评审要点**：
| 评分项 | 优秀(80%+) | 良好(60-80%) | 一般(40-60%) | 较差(<40%) |
|--------|------------|--------------|--------------|------------|
| 完整性 | 完整无遗漏 | 基本完整 | 部分缺失 | 严重缺失 |
| 创新性 | 有明显创新 | 有一定创新 | 常规方案 | 无创新 |
| 可行性 | 完全可行 | 基本可行 | 存在风险 | 难以执行 |

#### 报价合理性得分（15分）

| 报价与指导价关系 | 得分 |
|------------------|------|
| 等于指导价中位值 | 15分 |
| 指导价80%-中位值 | 13-14分 |
| 指导价中位值-120% | 13-14分 |
| 指导价60%-80% | 10-12分 |
| 指导价中位值+20%-50% | 10-12分 |
| 指导价<60% | 5-8分（可能被判定为恶意低价） |
| 指导价>150% | 5-8分（可能被判定为虚高） |

#### 时间承诺得分（5分）

| 时间承诺 | 得分 |
|----------|------|
| 承诺时间 < 要求时间 | 5分 |
| 承诺时间 = 要求时间 | 4分 |
| 承诺时间 ≤ 要求时间×1.2 | 3分 |
| 承诺时间 > 要求时间×1.2 | 1分 |
| 承诺时间 > 要求时间×1.5 | 0分（不满足基本要求） |

## 三、评标流程

### 3.1 评标委员会组成

| 项目等级 | 评标委员会 | 人数 |
|----------|----------|------|
| **C级** | 导师+1名成员 | 2人 |
| **B级** | 盟主+导师+1名成员 | 3人 |
| **A级** | 三长老+2名特邀 | 5人 |
| **S级** | 三长老+理事会代表+2名特邀 | 6人 |

### 3.2 评标流程

```
┌─────────────────────────────────────────────────────────────┐
│                    评标流程                                  │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【准备阶段】                                                 │
│  ├── 收集所有竞标书                                           │
│  ├── 整理竞标方资料                                           │
│  └── 分配评审任务                                             │
│                                                              │
│  【初审阶段】                                                 │
│  ├── 格式审查（淘汰不合格标书）                               │
│  ├── 资质验证（验证信用、业绩真实性）                         │
│  └── 列出通过初审的竞标方名单                                  │
│                                                              │
│  【评分阶段】                                                 │
│  ├── 各评委独立打分                                           │
│  ├── 计算各竞标方平均分                                       │
│  └── 生成初步排名                                             │
│                                                              │
│  【复核阶段】                                                 │
│  ├── 复核评分异常情况                                         │
│  ├── 处理评委争议                                             │
│  └── 确定最终排名                                             │
│                                                              │
│  【定标阶段】                                                 │
│  ├── 确定推荐挂帅人                                           │
│  ├── 撰写评标意见                                             │
│  └── 提交审批                                                 │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 评标时间要求

| 项目等级 | 评标时限 | 备注 |
|----------|----------|------|
| C级 | 2天内 | 可不安排答辩 |
| B级 | 5天内 | 可选安排答辩 |
| A级 | 10天内 | 必须安排答辩 |
| S级 | 15天内 | 必须安排答辩+现场考察 |

## 四、挂帅人机制

### 4.1 挂帅人职责

| 职责类别 | 具体内容 |
|----------|----------|
| **统筹协调** | 负责项目整体统筹和进度把控 |
| **对外代表** | 对外代表项目组与客户沟通 |
| **分包管理** | 主导项目分包和团队组建 |
| **交付责任** | 承担项目交付的主要责任 |
| **分账执行** | 负责项目分账的执行和监督 |

### 4.2 挂帅人权益

| 权益类型 | 具体内容 |
|----------|----------|
| **挂帅补贴** | 基础贡献权重 + 10%挂帅补贴 |
| **优先信息权** | 优先获得新项目信息 |
| **三长老资格** | 可担任三长老角色（需信用A级以上） |
| **品牌背书** | 以"挂帅人"身份获得社区品牌背书 |

### 4.3 挂帅人资格要求

| 项目等级 | 信用要求 | 经验要求 | 其他要求 |
|----------|----------|----------|----------|
| C级 | ≥C级 | 完成≥1个项目 | 无 |
| B级 | ≥B级 | 完成≥3个项目 | 无 |
| A级 | ≥A级 | 完成≥5个项目，其中同类≥2个 | 有团队管理经验 |
| S级 | ≥A级 | 完成≥10个项目，其中同类≥5个 | 有大型项目管理经验 |

### 4.4 挂帅人变更

| 变更原因 | 处理方式 | 生效条件 |
|----------|----------|----------|
| 主动请辞 | 启动重新竞标 | 原挂帅人申请 |
| 无法胜任 | 降级为普通成员，重新竞标 | 三长老认定 |
| 严重违规 | 取消资格，启动退出程序 | 违约事实确认 |
| 不可抗力 | 紧急指定临时负责人 | 紧急情况 |

## 五、流标处理

### 5.1 流标判定标准

| 流标原因 | 判定条件 |
|----------|----------|
| **竞标方均不达标** | 所有竞标方综合得分 < 60分 |
| **竞标方数量不足** | 有效竞标方数量 < 规定的最低数量 |
| **客户取消项目** | 客户主动取消需求 |
| **恶意竞标** | 发现竞标方串通、围标等行为 |

### 5.2 流标处理方式

| 流标原因 | 处理方式 | 后续步骤 |
|----------|----------|----------|
| 竞标方均不达标 | 重新发布，降低门槛 | 调整需求或预算后重新竞标 |
| 竞标方数量不足 | 自动延长竞标期 | 延长期结束后重新评估 |
| 客户取消项目 | 项目终止 | 公示原因，退还保证金 |
| 恶意竞标 | 取消竞标资格 | 扣除信用分，启动处罚 |

### 5.3 流标后的项目重启

| 重启条件 | 重启流程 |
|----------|----------|
| **调整后重启** | 修改项目信息 → 重新审核 → 重新竞标 |
| **指定执行** | 经三长老批准 → 指定执行方 → 简化流程 |
| **永久终止** | 发布终止公告 → 公示原因 → 结清费用 |

## 七、快速检查清单

参与竞标前，确认以下事项：

```
□ 仔细阅读项目需求，确保理解准确
□ 评估自身能力是否满足要求
□ 准备完整的竞标书材料
□ 确保信用等级符合要求
□ 评估时间是否能满足要求
□ 确认报价在合理范围内
□ 了解评标标准和权重
□ 按时提交竞标书
□ 保持联系方式畅通
```
