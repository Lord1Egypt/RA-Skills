# Pulling Manipulation Safety Protocol

## When to Use
- 患者存在关节错位或功能障碍
- 诊断明确
- 患者体质壮实
- 无关节或脊柱骨性病变、发育不良、强直僵硬或严重畸形
- 非年老体弱或久病体虚者

## Execution Steps
1. **术前评估**：确认无骨性病变、关节强直、严重畸形等禁忌情况。
2. **术者姿势**：站势稳定，沉肩垂肘，两手用力稳实、协调。
3. **扳动原则**：
   - 方向可为同一或相反，但不可硬扳或施暴
   - 幅度严格限制在正常生理活动范围内
   - 由小到大、循序渐进，不强求
   - 动作果断迅速，整体缓和准确
4. **部位特异性操作**：
   - **颈项扳法**：先牵引，再旋转至最大限度后扳动
   - **腰部斜扳法**：侧卧位，被动旋转至极限后反向扳动
   - **胸背扳法**：可用拇指顶扳或膝顶扳，配合患者深呼吸
5. **术后观察**：操作后密切观察患者反应，如有不适立即停止。

> 注意：即使对体质壮实者也须审慎操作，防止跌仆或意外伤害。
