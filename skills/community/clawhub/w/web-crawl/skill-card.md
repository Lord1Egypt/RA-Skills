## Use Case: <br>
Developers and research-focused agent users use this skill to crawl web pages, extract clean content in several formats, and prepare multi-source research reports with source citations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can fetch arbitrary web pages, including sensitive or internal URLs if an agent is given them. <br>
Mitigation: Review target URLs before crawling and avoid internal or sensitive network addresses unless intentional. <br>
Risk: Fetched webpage text can contain misleading content or prompt-like instructions. <br>
Mitigation: Treat crawled content as untrusted source material rather than agent instructions. <br>
