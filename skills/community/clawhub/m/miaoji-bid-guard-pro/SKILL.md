## 定位升维

**敌人**：只看单次ACoS、不规划长期ROI曲线的"点杀"思维

**升维打击**：从"建议出价多少"升级到"设计完整广告战役"

免费版告诉你"这个词建议出价$1.2"，Pro版告诉你"这个季度这组词的ROI会怎么走，什么时候该进攻、什么时候该防守"。

---

## 核心定位

广告护城河Pro版是免费版的战略升级版，提供完整的90天ROI预测曲线和多活动协同方案。通过季节性出价系数表和关键词攻防矩阵，帮助卖家在Prime Day/Q4等关键战役中占据优势位置。

**与免费版的本质区别**：
- 免费版：单次出价建议 → 现在应该怎么做
- Pro版：全程战役规划 → 这90天会怎么走、每步棋怎么下

## Pro专属功能

### 1. ROI预测模型

基于品类特性和运营阶段，预测90天广告ROI变化曲线：

```
【90天ROI预测曲线】
━━━━━━━━━━━━━━━━━━━━
品类：[XXX]
阶段：[新品推广/稳定期/旺季冲刺]
起始ACoS：XX%

📈 ROI预测分段：
| 阶段 | 时间段 | 预期ACoS | 预期RoAS | 策略基调 |
|------|--------|---------|----------|---------|
| 测试期 | D1-14 | 35-40% | 2.5-2.8 | 宽泛测试 |
| 优化期 | D15-30 | 28-32% | 3.1-3.5 | 精准聚焦 |
| 扩张期 | D31-45 | 22-26% | 3.8-4.2 | 放量测试 |
| 稳定期 | D46-60 | 20-24% | 4.1-4.5 | 稳定投放 |
| 冲刺期 | D61-90 | 18-22% | 4.5-5.0 | 旺季备战 |

📊 关键里程碑：
- D14：关键词点击量≥500，有效率≥70%
- D30：ACoS降至30%以下
- D45：核心词自然排名进入前3页
- D60：整体ACoS稳定在25%以内
- D90：准备进入旺季冲刺模式
```

### 2. 多活动协同策略

品牌广告+效果广告+展示广告联动方案：

```
【多活动协同矩阵】
━━━━━━━━━━━━━━━━━━━━

🎯 协同目标：新品上市 90天 GMV从$0到类目前200

📊 活动配置：
| 活动类型 | 预算占比 | 目标 | 配合节奏 |
|---------|---------|------|---------|
| 品牌广告(SB) | 20% | 品牌曝光+ASIN关联 | 全程 |
| 视频广告(SBV) | 25% | 种草转化 | D30起放量 |
| 效果广告(SP) | 45% | 直接转化 | 全程优化 |
| 展示广告(SD) | 10% | 再营销 | D45起启动 |

🔗 协同逻辑：
阶段1(D1-30)：SP测试为主，积累数据
  → 识别高绩效关键词
  → 识别高效受众
  
阶段2(D31-60)：SP放量 + SB助攻
  → 高绩效词启动SB保护
  → 高效受众启动SBV种草
  
阶段3(D61-90)：SD收割 + 全面协同
  → SD再营销已购/浏览人群
  → SB+SP+SD联动，锁定转化

⚠️ 协同红线：
- SP ACoS>35%时，SB不追加预算
- 某个词SP出价>ACoS目标2倍时，SB暂停该词
```

### 4. 关键词攻防决策矩阵

基于自然排名位置的进攻/防御/撤退决策：

```
【关键词攻防矩阵】
━━━━━━━━━━━━━━━━━━━━

📊 决策框架：

         自然排名位置
         1-3位    4-10位   11-20位   21位+
┌──────┬─────────┬─────────┬─────────┬────────┐
│点击率 │                     │                    │
│ >15%  │ 🟢观望  │ 🟡观察  │ 🟢进攻  │ 🔴强攻  │
├──────┼─────────┼─────────┼─────────┼────────┤
│点击率 │                     │                    │
│ 8-15% │ 🟢防御  │ 🟢进攻  │ 🟡进攻  │ 🟡进攻  │
├──────┼─────────┼─────────┼─────────┼────────┤
│点击率 │                     │                    │
│ <8%   │ 🔴撤退  │ 🟡防御  │ 🟡观望  │ 🟡进攻  │
└──────┴─────────┴─────────┴─────────┴────────┘

🔵 决策说明：
- 🟢进攻：提高出价10-20%，争取更多曝光
- 🟡观望：保持现状，持续监测数据
- 🟢防御：维持当前出价，保护已有位置
- 🟡防御：适当降低出价，优化ACoS
- 🔴撤退：大幅降低出价或暂停，止损
- 🔴强攻：高溢价+广泛匹配，快速获取数据

📋 攻防执行清单：

进攻模式激活条件：
□ 自然排名 >10位
□ ACoS < 目标ACoS × 1.2
□ 转化率 > 类目均值
→ 执行：出价+15%，匹配模式调整为广泛

防御模式激活条件：
□ 自然排名 1-5位
□ ACoS > 目标ACoS × 1.5
□ 流量已饱和
→ 执行：出价-20%，匹配模式收窄为精准

撤退模式激活条件：
□ 自然排名 1-3位
□ ACoS > 目标ACoS × 2.0
□ 订单贡献<10%
→ 执行：出价-50%，或暂停该词
```

## 不适用场景（Do Not Use When）

- ❌ 需要实时竞价调整（建议使用Sellics/Perpetua等自动化工具）
- ❌ 极小预算账户（<$500/月，数据量不足难以预测）
- ❌ 只想看单次出价建议（建议使用免费版 miaoji-bid-guard）
- ❌ 已有完整广告代理服务（可能与现有策略冲突）
