# 中国平台广告违禁词速查表

## 一、绝对化用语（广告法第九条）

这些词在任何平台都禁止使用，违者可能被罚款20-100万元：

### 禁止词
- 最：最好、最大、最高级、最便宜、最流行、最先进
- 第一：全国第一、销量第一、排名第一、行业第一
- 极：极品、极致、顶级、顶尖、绝版
- 独：独家、独创、独有、唯一
- 绝：绝对、绝无仅有、绝佳
- 首：首个、首选、首款、首次
- 万能、全能、包治、根治

### 安全替代
| 禁止词 | 安全替代 |
|--------|----------|
| 最好 | 非常好、备受好评 |
| 销量第一 | 销量领先、热销 |
| 顶级 | 高端、优质 |
| 独家 | 特色、甄选 |
| 绝对有效 | 深受认可、广受好评 |
| 包治 | 有助于、辅助改善 |

## 二、医疗/健康类违禁词

### 禁止词（非医疗资质不可用）
- 治愈、根治、药到病除、疗效
- 处方、药效、临床验证（非医疗产品）
- 减肥、瘦身、降脂（需特证）
- 抗癌、防癌、治疗

### 化妆品特殊限制
- 美白（需特证，否则用"提亮"）
- 祛斑（需特证，否则用"淡化"）
- 防晒（需特证，否则用"隔离"）
- 抗皱（需特证，否则用"紧致"）
- 医学护肤（禁止，用"科学护肤"）

## 三、金融类违禁词

### 禁止词
- 保本、保收益、零风险
- 稳赚、翻倍、暴利
- 内幕消息、独家渠道
- 100%收益、年化XX%

### 安全替代
| 禁止词 | 安全替代 |
|--------|----------|
| 保本 | 稳健型 |
| 零风险 | 低风险 |
| 稳赚 | 历史表现良好 |
| 翻倍 | 增长显著 |

## 四、平台特殊规则

### 小红书额外违禁
- 引导加微信/VX/私聊
- 价格对比（"比XX便宜"）
- 虚假种草（未使用就推荐）
- 医疗美容效果承诺

### 抖音额外违禁
- 诱导站外交易
- 夸大收益/效果
- 低俗/擦边内容
- 未标明的广告内容

### 淘宝/京东额外违禁
- 价格欺诈（原价无依据）
- 虚假库存（说缺货实则不卖）
- 引导线下交易
- 侵权/山寨描述

## 五、快速自检方法

写完内容后，搜索以下关键词，出现任何一个都需要修改：
1. "最" + 任何形容词
2. "第一" / "No.1" / "NO.1"
3. "绝对" / "一定"
4. "根治" / "治愈" / "100%"
5. "包" + 效果承诺（包好/包瘦/包过）
6. 任何医疗效果承诺（非医疗产品）

**原则**: 用具体数据替代绝对化表述。说"98%用户反馈改善"比说"100%有效"安全得多。
