# Party Mode Meeting

Party Mode is an OpenClaw skill for running a sequential multi-agent roundtable.

Use it with:

```text
/party <topic>
```

The skill asks OpenClaw to select 3-4 available agents, invite them one at a time, allow short reactions or passes, and return a final transcript with a brief wrap-up.

This skill has no external dependencies, credentials, scripts, or tools. It is instruction-only.
