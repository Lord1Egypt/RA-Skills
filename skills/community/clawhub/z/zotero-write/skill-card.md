## Description: <br>
Write, tag, annotate, and edit a local Zotero SQLite database. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
Researchers, librarians, and developers use this skill to add tags, notes, metadata updates, and new items to a local Zotero library through a Python SQLite helper. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill edits a local Zotero SQLite database, so an incorrect path, open Zotero process, or unintended field update could damage or overwrite library data. <br>
Mitigation: Confirm the database path, close Zotero before writing, keep the backup requirement, and test on a copy before using an important library. <br>
