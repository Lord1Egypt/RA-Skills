# Amazon A+ Content Generator

智能生成亚马逊A+内容，一站式完成模块方案、场景文案和多语言本地化。

## 快速开始

**输入格式**：
```
产品类目 + 核心卖点（3-5个）+ 目标语言
```

**示例**：
```
保温杯 + 保温持久/食品级/轻便/多场景 + 德语
```

## 功能特点

### 智能模块匹配
根据产品类目自动推荐最优的A+模块组合

### 场景化文案
将产品参数转化为真实使用场景的描述，打动目标用户

### 多语言本地化
支持德语、日语、法语、西语、意语等市场的文化适配

### 合规自检
自动检查图片规格、文字规范和模块标准

## 支持的类目

- 服装类
- 电子类
- 家居类
- 美妆类
- 宠物类
- 食品类
