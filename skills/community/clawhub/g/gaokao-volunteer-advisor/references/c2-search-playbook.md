# 检索剧本：逐条可抄的搜索策略

## 通用原则

1. 优先搜索官方来源（省考试院、学校官网）
2. 负面信息搜索用多个关键词组合，覆盖不同角度
3. 搜索结果要交叉验证，单一来源不能作为结论
4. 标注每个数据的来源和年份
5. 查不到就说查不到，不编造

## 阶段一：录取数据搜索

### 搜索位次和分数线

```
{省份} {年份} 一分一段表 省考试院
{省份} {年份} 普通高校招生计划 省考试院
{学校名} {年份} 各专业录取分数线
{学校名} {年份} 投档线 最低位次
```

### 搜索招生计划

```
{省份} {年份} 普通高校招生计划
{学校名} {年份} 在{省份}招生计划
{学校名} {年份} 本科招生章程
```

### 搜索选科要求

```
{省份} {年份} 普通高校招生选科要求
{学校名} {年份} 选科要求
```

## 搜索结果处理

1. 每条信息标注来源（官方/媒体/学生反馈）
2. 负面信息区分：已证实（裁判文书/官方通报）vs 网传未证实
3. 数据标注年份，非当年数据标注"仅供参考"
4. 冲突信息以官方来源为准
5. 涉及安全的信息，即使未证实也要标注"值得警惕"
