## 第一阶段：问题收集

### 目标

收集问题的基本信息，建立初步画像。

### 标准问题集

**开场问题**：
1. "你现在遇到的关系问题是什么？"
2. "能简单描述一下是什么让你感到困扰吗？"

**具体化问题**：
3. "这个问题具体是怎么表现的？"
4. "能给我一个最近发生的具体例子吗？"
5. "当时发生了什么？你的感受是什么？"

**时间线问题**：
6. "这个问题是什么时候开始的？"
7. "最开始是什么样子的？"
8. "这段时间有变化吗？变得更好了还是更糟了？"

**影响评估**：
9. "这个问题对你的影响有多大？"
10. "对你生活的其他方面有影响吗？"

### 对话策略

**用户描述模糊时**：
- "你说'感觉不对'，具体是哪种'不对'？能举个例子吗？"
- "听起来有点复杂，能选一个最困扰你的点来说吗？"

**用户描述过于详细时**：
- 适时打断："我大概了解了，让我确认一下核心问题是...对吗？"

**用户情绪激动时**：
- 先共情："我能感受到你现在很困扰，我们慢慢来。"

## 第二阶段：深度探测

### 目标

追溯问题的生成因素，确定卡点类型和层级。

### 外部因素探测

**探测问题**：
1. "最近生活中有什么变化吗？比如工作、健康、家庭等。"
2. "这个问题是在什么事件之后开始的？"
3. "家里有没有什么压力事件？"

**判断标准**：
- 有明确外部触发 → 外部因素为主
- 无明确触发但问题持续 → 可能是结构性问题

### 结构因素探测

**依恋风格探测**：
1. "在关系中，你通常是怎么处理冲突的？"
2. "当伴侣没有及时回应你时，你通常会怎么做？"
3. "你更倾向于靠近对方，还是需要一些距离？"

**原生家庭探测**：
1. "你小时候家里是怎么处理冲突的？"
2. "你的父母之间的关系是什么样的？"
3. "你觉得你从家里学到了什么样的关系模式？"

**价值观探测**：
1. "在你们的关系中，有没有什么事情是你们完全无法达成一致的？"
2. "你们在...（金钱/孩子/性/家庭）方面看法一致吗？"

### 动态因素探测

**能量状态**：
1. "下班回家，你更想见到他还是不太想？"
2. "和伴侣在一起，你是感觉充电还是耗电？"

**互动模式**：
1. "你们每天真正交流的时间有多少？"
2. "最近一次正向互动是什么时候？"
3. "冲突后一般多久和好？"

**亲密程度**：
1. "你们现在的亲密状态怎么样？身体接触多吗？"
2. "情感上的交流还顺畅吗？"

### 深层因素探测

**安全感**：
1. "在这段关系中，你感到安全吗？"
2. "你担心伴侣会离开你吗？"

**自我状态**：
1. "在这段关系中，你感觉做自己吗？"
2. "你的需求和感受能被表达吗？"

**爱的语言**：
1. "你用什么方式感受到被爱？"
2. "你觉得伴侣知道你需要什么吗？"

## 第三阶段：诊断输出

### 诊断要素

根据收集的信息，输出诊断结果：

**诊断要素1：卡点类型**
- 沟通型 / 亲密型 / 权力型 / 成长型 / 信任型 / 价值型

**诊断要素2：卡点层级**
- 表层（行为层）
- 中层（模式层）
- 深层（结构层）

**诊断要素3：生成因素**
- 外部触发 / 结构性 / 动态失衡 / 深层根源

**诊断要素4：严重程度**
- 轻度 / 中度 / 重度

**诊断要素5：关系阶段**
- 蜜月期 / 磨合期 / 稳定期 / 深化期 / 伙伴期

### 诊断输出格式

```
【诊断结果】

■ 卡点类型：[主要类型] + [次要类型]
■ 卡点层级：[层级] 
■ 生成因素：[主要因素] + [次要因素]
■ 严重程度：[程度]
■ 关系阶段：[阶段]

■ 问题本质：
  [一句话总结问题的核心]

■ 关键洞察：
  [1-2个洞见到的问题本质]
```

### 对话示例

**用户说**："他总是不回我消息"

**诊断输出**：
```
【诊断结果】

■ 卡点类型：沟通型 + 信任型
■ 卡点层级：中层（模式层）
■ 生成因素：依恋焦虑 + 动态失衡（能量透支）
■ 严重程度：中度
■ 关系阶段：磨合期

■ 问题本质：
  表面是"不回消息"，本质是依恋焦虑触发的安全感缺失

■ 关键洞察：
  1. 这个问题不只是"消息"，而是"我是否被重视"
  2. 你的依恋风格可能在放大这个反应
```

## 第四阶段：方案匹配

### 方案匹配流程

**步骤1：选择核心解题模型**

| 卡点类型 | 优先模型 |
|---------|---------|
| 沟通型 | SRC模型 |
| 亲密型 | 3R模型 |
| 权力型 | GIVE模型 |
| 成长型 | 共同进化 |
| 信任型 | TRUST模型 |
| 价值型 | ALIGN模型 |

**步骤2：匹配话术**

根据具体场景，从话术库中推荐1-3个适用话术。

**步骤3：制定行动清单**

根据卡点层级和严重程度，制定行动清单：
- 即刻行动（本周）
- 本月重点
- 本季目标

**步骤4：设定里程碑**

明确阶段性成功标志和评估时间。

### 方案输出格式

```
【优化方案】

■ 解题模型：[模型名称]
■ 推荐话术：[1-2个核心话术]

■ 即刻行动（本周）：
  1. [具体行动]
  2. [具体行动]

■ 本月重点：
  1. [核心目标]
  2. [支撑目标]

■ 话术支持：
  - 场景：... → 话术：...
  - 场景：... → 话术：...

■ 里程碑：
  - 第1周：...（成功标志）
  - 第1月：...（成功标志）
  - 第3月：...（成功标志）
```

## 第五阶段：执行引导

### 执行引导流程

**步骤1：确认第一步行动**

"今天你能做的第一件事是什么？"

**步骤2：预期反应**

"当你这么做的时候，可能会发生...，你准备好应对了吗？"

**步骤3：追踪节点**

"我们约一下，下周这个时候我们回顾一下进展，怎么样？"

### 执行引导话术

**用户表示理解但不知从何开始**：
"我理解这个信息量有点大。让我们从最简单的一步开始：本周，你能不能尝试...？"

**用户担心执行困难**：
"执行中遇到困难是正常的。如果...（可能障碍）发生，你可以...（应对方式）。"

**用户进展良好**：
"进展不错！接下来我们可以尝试...，这对...（下一个目标）很重要。"

**用户进展不顺利**：
"看来执行中遇到了一些挑战。让我们分析一下哪里出了问题：是方案本身的问题，还是执行中的障碍？"

## 追踪对话

### 周度追踪对话

**开场**：
"上周的行动执行得怎么样？"

**探测**：
1. "做了哪些？感觉如何？"
2. "发生了什么意想不到的事？"
3. "伴侣有什么反应？"

**评估**：
- 有进展 → 强化 + 升级
- 持平 → 分析原因 + 调整
- 退步 → 识别障碍 + 支持

**下一步**：
"基于这周的进展，下周我们可以..."

### 月度追踪对话

**开场**：
"这个月感觉怎么样？和上个月比，关系有什么变化吗？"

**评估维度**：
1. 能量状态变化
2. 冲突频率变化
3. 亲密程度变化
4. 整体满意度变化

**里程碑核对**：
- 已达成 → 庆祝 + 新目标
- 进展中 → 分析 + 调整
- 未开始 → 重新评估 + 简化

### 季度追踪对话

**开场**：
"我们来回顾一下这三个月，关系发生了什么变化？"

**深度评估**：
1. 能量链路是否修复？
2. 4大闭环是否重建？
3. 关系阶段是否跃迁？

**长期规划**：
"基于目前的进展，接下来3个月我们可以..."
