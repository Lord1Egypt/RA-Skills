## 化石能源任务清单

化石能源按子域组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：资源勘探 → 开发生产 → 加工转化 → 市场销售

## 依赖拓扑摘要

### 一、域内链路

**资源勘探链路**: T1-1 → T1-2 → T1-3 → T1-4
**开发生产链路**: T2-1 → T2-2 → T2-3 → T2-4
**加工转化链路**: T3-1 → T3-2 → T3-3 → T3-4
**市场销售链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整化石能源链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T3-1 → T3-2 → T3-3 → T3-4 → T4-1 → T4-2 → T4-3 → T4-4

**反馈改进链路**: T4-1 → T1-2（市场分析反馈到资源评估）

**安全驱动链路**: T2-3 → T2-1（安全环保驱动开采技术改进）

更多组合根据具体任务动态推导。
