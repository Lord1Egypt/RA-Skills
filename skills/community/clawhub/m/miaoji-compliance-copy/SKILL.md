# 秒技·合规文案重写 (miaoji-compliance-copy)

## 技能定位
**输入**：粗糙的Amazon产品文案（标题+五点描述）+ 产品品类
**输出**：秒级清洗后的3重保障文案（去违规+加场景+Rufus可读）

## 核心能力

### 1. 合规性检查与清洗
- 扫描Amazon平台禁止词（如"Best Seller"、"Free Shipping"、"治疗"、"顶级"等）
- 扫描过度营销词（如"第一"、"最强"、"保证"等绝对化用语）
- 生成替换建议，确保文案符合亚马逊2026年审核规则

### 2. Rufus搜索适配改写
- 植入用户意图词（场景词+人群词）
- 参数罗列→场景化描述转换
- 加入主观属性词，提升Rufus推荐权重

### 3. 一键重写输出
- **标题重写**：含场景词+意图词，符合搜索算法
- **五点描述重写**：痛点→方案→利益结构
- **搜索词补充**：关键词矩阵建议

## 使用流程

### Step 1: 输入原始文案
```
产品品类：[如：露营帐篷/护膝/咖啡机等]

原始标题：
[粘贴标题]

五点描述：
1. [第一条]
2. [第二条]
3. [第三条]
4. [第四条]
5. [第五条]
```

### Step 2: 智能分析
技能自动处理：
1. 违规词扫描 → 生成问题词列表
2. Rufus适配度评估 → 识别缺失要素
3. 场景词注入建议 → 推荐场景标签

### Step 3: 重写输出
生成3个版本的文案：
- **版本A**：标准合规版（平衡型）
- **版本B**：场景强化版（适合Rufus搜索）
- **版本C**：转化导向版（适合广告投放）

## 输出格式

### 合规检查报告
```
🔍 合规扫描结果

❌ 发现问题词：
| 原词 | 问题类型 | 建议替换 |
|------|----------|----------|
| "Best Seller" | 平台禁止词 | "人气之选" |
| "保证有效" | 过度营销 | "用户反馈良好" |

✅ 通过项：[已通过检查的要素]
```

### 重写文案
```
📝 版本A：标准合规版

【优化标题】
[合规且含场景词的标题]

【五点描述】
• [第一条：痛点引入]
• [第二条：产品特性]
• [第三条：使用场景]
• [第四条：差异化优势]
• [第五条：购买引导]

🔑 搜索词建议：
[场景词1] / [场景词2] / [人群词] / [用途词]
```

## 技术说明
- 基于亚马逊2026年审核规则分析
- 深度理解Rufus搜索推荐逻辑
- 支持中英文文案处理

## 适用场景
- 新品Listing编写
- 现有Listing优化
- 合规性自查
- Rufus搜索排名提升


## 交付标准

优秀的输出应该让你一眼看到：

- 每个违规词都有安全替代方案
- 修改后文案保持原意，转化意图不变
- Rufus对话式表达自然流畅
- 标注每个修改的具体原因
- 提供至少2个表达选项供选择

如果输出缺少以上任何一项，建议补充后再交付。

## 安全边界声明

本技能为纯文案建议工具，所有文案修改建议需卖家手动复制到亚马逊后台确认后提交。本技能不连接任何亚马逊API接口，不修改任何Listing内容，不存储任何账户凭证，不自动同步数据。
