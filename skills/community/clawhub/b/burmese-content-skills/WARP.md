# WARP.md

This repository contains a Burmese writing skill for AI tools.

The main purpose is to transform AI-generated Burmese writing into natural conversational Burmese.

## Main Files

- SKILL.md → Core writing instructions
- README.md → Documentation
- examples/ → Burmese writing examples

## Target Writing Types

- Burmese Facebook posts
- Tech blogs
- Motivation content
- Knowledge sharing
- Storytelling
- Productivity posts
- AI-generated Burmese cleanup

## Burmese Writing Philosophy

The writing should sound like:
- a real Burmese person
- casual conversation
- social media content
- modern Burmese communication

Avoid:
- government Burmese
- textbook Burmese
- newspaper Burmese
- robotic translations

## Recommended Tone

- direct
- relatable
- emotional
- simple
- human
