## Description: <br>
Use when an agent needs to score itself or another agent with the Agent Spectrum six-axis framework, run the quick or deep edition, identify the resulting type and faction, render both the Hexagon Block and Coordinate Card Block, and return the strict result in the user's language without mixed-language labels. <br>

This skill is ready for commercial/non-commercial use. <br>
