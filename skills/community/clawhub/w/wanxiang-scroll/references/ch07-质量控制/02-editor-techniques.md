# 编辑技巧

改稿方法、节奏调整、逻辑修补。

---

## 文笔润色：去模板化

**核心原则**：
1. 拒绝AI常用词：综上所述、值得注意的是、总而言之、因此、然而、此外、再者
2. 增加口语化表达：说到底、说白了、我跟你讲、那、可、对了
3. 加入非逻辑性跳跃：人类思维有时候是跳跃的
4. 情感表达更直接：使用感叹词、反问句

---

## 风格精炼：植入瑕疵

完美是人类不存在的，AI写得太完美反而可疑。故意植入小瑕疵：
- 小的语法错误（但不影响理解）
- 口语化表达
- 偶尔的重复
- 不完整的句子

**示例**：
```
完美AI版：他是一名经验丰富的医生，在医学领域有着深厚的造诣。
人类化版：他是医生，老医生了。看病准得很，就是脾气不太好。
```

---

## 情绪化表达

AI的情绪表达太理性，人类更情绪化：
- 使用感叹词
- 使用反问句
- 使用省略号
- 使用口语

**示例**：
```
完美AI版：这让他感到非常沮丧和失望。
人类化版：那感觉，真的...太难受了。就像是...算了，不说了。
```

---

## 非线性叙事

AI的叙事太线性，人类的思维更跳跃：
- 突然插入无关但相关的内容
- 跳跃式的时间线
- 插入回忆和联想

**示例**：
```
完美AI版：他走进房间，看到了那个人，然后想起了过去的回忆。
人类化版：他推开门。那人坐在那儿，还是老样子。恍惚间，他又回到了三年前那个雨夜...
```

---

## 装饰词去除

精简文本，去除无效的装饰词和形容词：
- 删除空洞的形容词：美丽的、漂亮的、优秀的
- 删除重复的修饰词：非常十分特别
- 删除"的/地/得"的滥用
- 用具体名词替代形容词

**示例**：
```
原文：他是一个非常英俊的年轻人，有着非常漂亮的眼睛，非常温柔的性格。
精简后：他帅，眼睛好看，性格温柔。
```
