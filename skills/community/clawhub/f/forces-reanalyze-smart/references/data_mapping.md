# 表格结构 & 数据映射

## 文档结构

目标文档第三节是一个固定结构的嵌套表格。

### 表格头（11列）

| 列 | 字段 | 属性 |
|----|------|------|
| 1 | 时间 | — |
| 2 | 工单总数 | — |
| 3-4 | 无需优化 | colspan="2" |
| 5 | 待产品优化 | — |
| 6 | 外部原因-技术派工 | — |
| 7 | 外部原因-需技术排查 | — |
| 8 | 外部原因-无需技术排查 | — |
| 9-10 | 内部原因 | colspan="2" |
| 11 | 超时 | — |

### 月份组（33个单元格）

每个月数据占3行×11列=33个单元格。

**第1行（主行）** — 索引 0-10

| 偏移 | 列 | rowspan |
|------|-----|---------|
| 0 | 月份名 | 3 |
| 1 | 工单总数 | 3 |
| 2 | 无需优化左 | 3+2 |
| 3 | 无需优化右(留空) | — |
| 4 | 待产品优化 | 3 |
| 5 | 技术派工 | 3 |
| 6 | 需技术排查 | 3 |
| 7 | 无需技术排查 | 3 |
| 8 | 内部原因左 | 3+2 |
| 9 | 内部原因子列 | — |
| 10 | 超时 | 3 |

**第2行（子行）** — 索引 11-21

| 偏移 | 列 | 说明 |
|------|-----|------|
| 9(20) | 列10 | 延期处理 |

**第3行（子行）** — 索引 22-32

| 偏移 | 列 | 说明 |
|------|-----|------|
| 9(31) | 列10 | 无法重现 |

### 全局索引计算

```
月份组起始索引 = 11 (表头) + 33 × 月份序号
               = 11 + 33 × N

N=0 (02月): 11
N=1 (03月): 44
N=2 (04月): 77

单元格索引 = 月份组起始 + 行偏移 + 列偏移
  主行 列1 = 月份组起始 + 0
  主行 列2 = 月份组起始 + 1
  ...
  子行1 列10 = 月份组起始 + 20
  子行2 列10 = 月份组起始 + 31
```

## 数据格式化规则

参考已有月份的数据格式，新数据应保持一致性。

| 字段 | 格式 | 示例 |
|------|------|------|
| 工单总数 | 纯数字 | `644` |
| 无需优化 | `{n}（产研介入{m}）` | `160（产研介入94）` |
| 待产品优化 | `{n}（{m}）` | `42（20）` |
| 技术派工 | `{n}` 或 `{n}（{m}）` | `0` |
| 需技术排查 | `{n}（{m}）` | `79（48）` |
| 无需技术排查 | `{n}（{m}）` | `208（28）` |
| 内部原因 | `{n}（产研介入{m}）` | `138（产研介入87）` |
| 已解决 | `已解决{n}（{m}）` | `已解决54（27）` |
| 延期处理 | `延期处理{n}（{m}）` | `延期处理66（46）` |
| 无法重现 | `无法重现：{n}（{m}）` | `无法重现：18（14）` |
| 超时 | `{n}（{m}）` | `88（58）` |

注意：括号使用中文全角括号 `（）`，不添加"个"字后缀。产研介入参数格式看具体列——有些是"产研介入"，有些只有数字。
