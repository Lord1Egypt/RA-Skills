# 生产现场调研指南

## 调研目标
- 评估生产现场管理水平（5S/标准化/目视化）
- 识别生产过程中的七大浪费
- 评估OEE与产能利用状况
- 评估异常响应能力（Andon/停线管理）

## 调研重点

### 1. 现场管理
- 5S执行现状（现场走一圈拍照记录）
- 目视化管理水平（标识/看板/颜色管理）
- 标准化作业覆盖率（SOP上墙率/执行率）
- 产线布局（U型线/直线/CELL化？）

### 2. 生产效率
- 各工序Cycle Time（实际vs标准）
- 瓶颈工序识别
- OEE数据（是否有采集？真实值是多少？）
- 换模/换线时间（SMED水平）
- 生产节拍(Takt Time)管理

### 3. 物料流转
- 线边仓管理（物料摆放/先进先出）
- 物料配送方式（推送到工位/领料？）
- WIP（在制品）管理现状
- 物料搬运距离与频次

### 4. 异常管理
- 异常响应机制（Andon/呼叫铃/对讲机？）
- 停线管理（谁有权停线？停线频次？）
- 异常数据记录与分析
- 问题升级流程

### 5. 人员管理
- 多能工比例
- 人员技能矩阵
- 员工培训体系
- 绩效考核（计件/计时？）

### 6. 信息系统
- MES系统使用深度
- 报工方式（扫码/手工/自动采集？）
- 生产数据实时性

## 常见问题诊断库

| 问题现象 | 可能根因 | 典型影响 |
|---------|---------|---------|
| 5S形同虚设 | 缺乏持续检查/管理层不重视 | 效率损失/安全隐患 |
| 现场WIP堆积严重 | 批量生产/前后工序不平衡 | 占用资金/掩盖问题 |
| OEE低 | 故障多/换型长/质量损失 | 产能不足/成本高 |
| SOP执行率低 | SOP不实用/培训不足/监督缺失 | 质量波动/效率不稳定 |
| 异常响应慢 | 无安灯系统/责任不明/升级路径不清 | 停线损失/问题扩大 |
| 换线时间长 | 内部/外部作业未区分/缺乏SMED | 柔性不足/批量被迫大 |
| 物料搬运浪费多 | 布局不合理/配送无标准 | 非增值时间多/效率低 |
| 多能工比例低 | 培训体系缺失/薪酬激励不足 | 人员调配困难/柔性差 |
| MES数据不准 | 手工录入/滞后/未与设备集成 | 决策依据不可靠 |

## 数据采集项
- 5S评分（现场拍照记录）
- 各工序C/T（实测3-5次取平均）
- 换模/换线时间
- OEE（或分段数据计算）
- 日/月产出数据
- WIP数量（关键工序间）
- 异常停线次数与时长
- 多能工比例
- SOP覆盖率与版本
