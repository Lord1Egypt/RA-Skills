## About

This project systematically integrates the core knowledge system of Medical Cell Biology, covering **146 key skills** across **20 major categories**. Topics span cell structure and function, molecular mechanisms, signal transduction, cell cycle regulation, stem cell biology, cell death and aging, organelle dynamics, membrane transport, gene expression regulation, disease mechanisms, and cutting-edge research technologies.

**Target Audience**: Medical students, researchers, clinical practitioners, educators

**Textbook Reference**: People's Medical Publishing House "Medical Cell Biology" 7th Edition

**⚠️ Disclaimer**: These skills cover cell biology concepts, molecular mechanisms, pathological analysis, and experimental techniques. They should not be used as independent medical or experimental guidance.

Mitigation: Use outputs only as educational or research references. Verify recommendations against current guidelines and domain consensus.

## Quick Start

### Installation

CLI:
```bash
openclaw skills install medical-cell-biology-7th-edition
```

### Usage

Each skill contains four sections:
1. **Trigger conditions** — When to use this skill
2. **Execution steps** — Standardized SOP
3. **Precautions** — Contraindications and warnings
4. **Reference documents** — Supplementary detailed materials

### Example Queries

- "Explain the molecular mechanism of microtubule dynamic instability."
- "Analyze how [cdk-cyclin-complex-activation-mechanism] activates the cell cycle."
- "Use [mitochondrial-apoptosis-pathway-analysis] to evaluate how anticancer drugs trigger apoptosis."
