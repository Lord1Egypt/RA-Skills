# 糖胺聚糖与蛋白聚糖结构功能分析

## 何时使用
- 分析软骨、角膜、基膜等组织的机械性能或通透性异常
- 探究发育、衰老或疾病（如肿瘤、遗传性代谢病）中细胞外基质（ECM）成分变化
- 解释凝血障碍、血管生成抑制或生长因子信号传导异常
- 评估组织工程材料中ECM模拟设计的合理性

> **注意**：仅适用于存在细胞外基质的组织环境，不适用于血液中游离细胞等无ECM场景。

## 执行步骤

### 1. 识别糖胺聚糖（GAG）类型与结构特征
- 确认GAG是否为以下六种之一：透明质酸（HA）、硫酸软骨素（CS）、硫酸皮肤素（DS）、硫酸乙酰肝素（HS）、肝素、硫酸角质素（KS）
- 判断是否含硫酸基团（HA除外）及负电荷密度
- 评估其水合能力与空间占据效应（尤其HA可结合大量水分子）

### 2. 分析蛋白聚糖（PG）组装方式
- 确定核心蛋白种类及其定位（细胞表面、基质或基膜）
- 判断GAG链是否通过Xyl-Gal-Gal-GlcUA连接至核心蛋白丝氨酸残基
- 识别是否形成聚合体（如软骨中PG通过连接蛋白与HA组装成巨大复合物）

### 3. 关联核心生理功能
- **机械支持**：评估PG对组织抗压性与弹性的贡献（如软骨）
- **分子筛作用**：分析GAG/PG负电性对水、离子、小分子及细胞因子通透性的选择性调控（如肾小球基膜）
- **信号调节**：检查细胞表面PG（如syndecan-1）是否介导生长因子（如FGF）与ECM蛋白结合
- **特殊功能**：确认肝素-PG是否参与抗凝（结合抗凝血酶），或角膜PG是否维持脱水致密状态以保障透光性

### 4. 评估病理与年龄相关变化
- 若观察到组织弹性下降、保水能力减弱，考虑年龄相关GAG转换（如胎儿高HA/CS → 成人DS/KS增加）
- 若出现GAG堆积症状（面容粗陋、关节僵硬等），排查黏多糖贮积症（如Hunter综合征）
- 在动脉粥样硬化背景下，检查内皮HS/CS减少与DS-PG增加是否促进LDL沉积
- 在肿瘤微环境中，判断HA/CS分泌是否增强并促进细胞迁移、增殖或免疫逃逸

## 输出提示
若发现组织物理特性异常、通透性失调、凝血功能障碍或肿瘤进展迹象，应优先考虑GAG/PG组成、硫酸化程度或代谢酶活性异常。
