# Prashna 单事件问事协议模板

> 版本: v1.0 | 2026-04-25
> 用途：当用户提出一个具体问题时，AI按此模板执行 Prashna 分析

> **来源标签**: 【现代演绎·Skill整合】 — 单事件问事推理协议
---

## 输入信息收集

当用户提出具体问题时，记录以下信息：

| 字段 | 必需 | 说明 |
|------|------|------|
| **问题原文** | ✅ | 用户原始提问措辞（影响宫位选择） |
| **提问时间** | ✅ | 问题完全成型的精确时刻 |
| **提问地点** | ✅ | 经纬度 |
| **问题类型** | ✅ | 事业/婚姻/财务/健康/失物/旅行/诉讼/子女 |
| **出生信息** | ❌ | 若有，用于大运/过境辅助验证 |

## 执行流程

### 阶段一：铸造星盘

```bash
# 调用引擎
python3 scripts/jyotish_engine.py prashna \
  --datetime "YYYY-MM-DD HH:MM" \
  --lat XX.XX --lon XX.XX --mode chart
```

### 阶段二：十步断卦（AI执行）

按以下十步逐步分析，每步输出结论：

#### Step 1: 星盘活力评估
- [ ] 上升星座、上升主星位置和力量
- [ ] 月亮位置和状态
- [ ] 结论：星盘基础强弱

#### Step 2: Arudha Lagna 对比
- [ ] 计算AL
- [ ] 对比 Lagna(真实) vs AL(感知)
- [ ] 结论：认知偏差程度

#### Step 3: 相关宫位分析
- [ ] 确定问题对应的宫位（见下表）
- [ ] 检查宫内行星
- [ ] 检查宫主星位置和状态
- [ ] 结论：该领域有利/不利

| 问题类型 | 主宫 | 辅助宫 | 自然Karakā |
|---------|------|--------|-----------|
| 事业 | 10 | 6, 11 | 太阳 |
| 婚姻 | 7 | 5, 2 | 金星/木星(女盘) |
| 财务 | 2, 11 | 9 | 木星 |
| 健康 | 1, 6 | 8 | — |
| 失物 | 2, 4 | 7, 11 | — |
| 旅行 | 3, 9 | 12 | 木星 |
| 诉讼 | 6, 7 | 12 | — |
| 子女 | 5 | 9 | 木星 |

#### Step 4: 征象星力量对比
- [ ] 提问者征象星 = 上升主星 + 月亮
- [ ] 所问事物征象星 = 对应宫主星 + 自然Karakā
- [ ] 力量对比：谁更强？
- [ ] 结论：征象星偏向何方

#### Step 5: Tajika Yoga 检查
- [ ] 两个征象星之间是否有 Ithasala（入相位）？
- [ ] 是否有 Ishrafa（出相位）？
- [ ] 是否有 Nakta（中间人传递）？
- [ ] 是否有 Mani（煞星阻隔）？
- [ ] 结论：Yoga 支持成功/失败

#### Step 6: Sphuta 组合
- [ ] Trisphuta 落在哪个星座/宫位？
- [ ] 是否与凶宫/凶星关联？
- [ ] 结论：综合能量走向

#### Step 7: 阻碍排查
- [ ] 征象星是否燃烧？
- [ ] 征象星是否落陷？
- [ ] 月亮是否空亡？
- [ ] 是否有 Rahu/Ketu 干扰？
- [ ] 结论：阻碍因素列表

#### Step 8: 证词权衡
- [ ] 清点有利证词数量
- [ ] 清点不利证词数量
- [ ] 结论：多数方向

#### Step 9: 时间判断
- [ ] 入相位度数差 → 时间单位
- [ ] 本位/固定/双体星座 → 快/慢/中
- [ ] 结论：预计应期范围

#### Step 10: 大运辅助（如有出生数据）
- [ ] 当前大运主星
- [ ] Prashna 指示是否与大运主题一致
- [ ] 结论：交叉验证结果

### 阶段三：综合结论

输出格式：

```
## Prashna 问事占星结论

**问题：** [用户原始提问]
**提问时刻：** YYYY-MM-DD HH:MM @ 地点

### 核心结论
[一句话明确回答：是/否/时机/方向]

### 分析依据
1. **征象星分析**：[Step 4 结论]
2. **Tajika Yoga**：[Step 5 结论]
3. **宫位状态**：[Step 3 结论]
4. **Sphuta 组合**：[Step 6 结论]
5. **阻碍因素**：[Step 7 结论]

### 时间判断
[Step 9 结论：具体时间范围]

### 注意事项
[Step 10 交叉验证 + Kunda 验证结果]

### 综合评估
| 维度 | 评估 |
|------|------|
| 星盘活力 | 强/中/弱 |
| 成功概率 | 高/中/低 |
| 时间范围 | [范围] |
| 关键阻碍 | [阻碍] |
```

## 特殊场景协议

### 失物查询专用

额外输出：
1. 物品指标星及位置
2. 窃贼分析（已知/未知）
3. 方向指引（火=东、土=南、风=西、水=北）
4. 找回概率评估
5. 建议搜索区域

### 健康查询专用

额外输出：
1. Prana/Deha/Mrityu Sphuta 判断
2. 若涉及重病：Mrityu Chakra 分析
3. 不输出确定性的死亡判断，只输出"风险等级"

### 是非判断专用

使用 KP Prashna（1-249法）作为补充：
1. 用户报数（1-249）
2. 确定上升点 Sub-Lord
3. 检查相关宫 Cuspal Sub-Lord
4. 给出 是/否/延迟 判断

## 原则提醒

1. **问题必须真诚** — Prashna shakti 仅在真实需求时激活
2. **同一问题不重复问** — 重复提问使第一个盘失效
3. **不替代专业建议** — 健康/法律等问题需声明"仅供参考"
4. **措辞影响结果** — 始终记录原始问题
5. **综合多方证词** — 不依赖单一因素
