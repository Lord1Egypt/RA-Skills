## Description: <br>
Learn how to install and migrate to Jetpack Navigation 3, and how to implement features and patterns such as deep links, multiple backstacks, scenes, conditional navigation, returning results from flows, integration with Hilt, ViewModel, Kotlin, and view interoperability. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Android developers use this skill as reference material for installing Jetpack Navigation 3, planning migrations from Navigation 2, and implementing Compose navigation patterns such as deep links, multiple back stacks, adaptive scenes, dialogs, bottom sheets, and result handling. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Migration guidance can lead to broad changes across navigation code and dependencies. <br>
Mitigation: Review the generated migration plan before allowing project-wide code changes, and verify behavior with existing navigation tests. <br>
