## Description: <br>
Batch Processor Zh helps agents batch process and convert office documents, including PDF-to-Word conversion, Excel conversion, invoice extraction, contract analysis, and repeated workplace document updates. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Employees and operations teams use this skill to automate repetitive document conversion, extraction, analysis, renaming, and template update tasks across large folders of files. It is suited for HR resume review, finance report import, procurement order checks, legal contract archiving, and similar office workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Batch processing can affect many documents at once, including sensitive HR, finance, legal, or procurement files. <br>
Mitigation: Confirm the input folder, output folder, file types, and overwrite behavior before running a batch job; keep backups for important documents. <br>
Risk: The skill depends on an external office-mcp batch conversion tool. <br>
Mitigation: Install and use it only when the office-mcp batch conversion tool is trusted for the documents being processed. <br>
