# 保险科技技能

## 核心能力

| 能力 | 说明 |
|------|------|
| 风险缺口分析 | 基于年龄/收入/负债/家庭结构计算寿险/重疾/医疗/意外保额缺口 |
| 产品对比 | 300+产品条款逐项对比，ICR性价比评分（覆盖内地+香港） |
| 理赔指导 | 全流程材料清单+保险公司沟通话术+理赔时效追踪 |
| 保单解读 | 晦涩条款白话解读，识别免责条款和等待期陷阱 |
| 社保衔接 | 社保/惠民保/百万医疗/中高端医疗多层次保障规划 |

## 触发场景

- "我30岁年收入20万该买什么保险"
- "对比一下达尔文10号和超级玛丽13号"
- "医疗险理赔需要什么材料"
- "帮我解读这份保单条款"
- "已有社保还需要商业医疗险吗"

## 执行流程

### 阶段一：风险画像
1. 收集家庭信息（年龄/收入/负债/已有保障）
2. 计算四大险种保额缺口
3. 生成家庭风险热力图

### 阶段二：产品匹配
1. 根据缺口和预算筛选产品
2. 逐项对比条款（保障范围/等待期/免责条款/费率）
3. 输出ICR性价比评分排序

### 阶段三：决策支持
1. 推荐组合方案（含保费测算）
2. 投保注意事项（健康告知/职业限制）
3. 后续服务指南（保单管理/续期提醒/理赔流程）

## 注意事项
- 分析结果仅供参考，不构成投保建议，请以保险公司正式条款为准
- 健康告知务必如实填写，隐瞒病史可能导致拒赔
- 香港保险涉及外汇管制和跨境法律风险，需谨慎评估
