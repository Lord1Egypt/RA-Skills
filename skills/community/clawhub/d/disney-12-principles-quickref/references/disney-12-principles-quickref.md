# 迪士尼十二动画法则速查表

本文件为动画片脚本创作时的法则匹配参考，供创作过程中快速查阅。

## 法则与镜头场景匹配指南

| 法则 | 最佳匹配镜头类型 | 典型应用场景 |
|------|-----------------|-------------|
| 1. 挤压与拉伸 | 特写、近景 | 角色表情变化、弹性碰撞、跳跃落地 |
| 2. 预备动作 | 中景、全景 | 角色即将起跑、出拳、跳跃前的蓄力 |
| 3. 构图/舞台布局 | 全景、远景 | 场景交代、情感氛围渲染、关键信息聚焦 |
| 4. 连贯动画&关键帧动画 | 所有镜头 | 快动作用连贯，精细动作用关键帧 |
| 5. 跟随与重叠动作 | 中景、近景 | 角色急停后头发/衣角飘动、手臂摆动延迟 |
| 6. 渐入渐出 | 中景、跟拍 | 角色起步加速、减速停止、物体弹起落下 |
| 7. 弧形运动 | 中景、跟拍 | 抬手、走路、转头、抛物线运动 |
| 8. 次要动作 | 近景、特写 | 说话时挑眉、走路时摆手、思考时摸下巴 |
| 9. 时间把控 | 所有镜头 | 快节奏（帧少）vs 慢节奏（帧多），控制叙事节奏 |
| 10. 夸张表现 | 特写、近景 | 惊讶瞪大眼、愤怒握拳、开心蹦跳 |
| 11. 扎实造型 | 所有镜头 | 确保角色有体积感、透视正确、不是扁平纸片 |
| 12. 吸引力 | 特写、近景 | 角色首次亮相、魅力展示、表情特写 |

## 节奏法则搭配建议

### 开头（Hook阶段，0-15%时长）
- 优先使用：夸张表现、挤压与拉伸、吸引力
- 目的：快速抓住观众注意力

### 发展（叙事推进，15-75%时长）
- 优先使用：预备动作、弧形运动、渐入渐出、次要动作、跟随与重叠
- 目的：让动作流畅自然，推进故事

### 高潮（关键转折，75-85%时长）
- 优先使用：夸张表现、挤压与拉伸、时间把控
- 目的：强化情感冲击

### 收束（结尾，85-100%时长）
- 优先使用：构图/舞台布局、吸引力、扎实造型
- 目的：画面美感收束，留下印象
