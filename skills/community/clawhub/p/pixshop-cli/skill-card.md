## Description: <br>
Pixshop CLI helps agents guide users through terminal commands for Pixshop image generation, video creation, photo editing, app browsing, and prompt search. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and developers use this skill to install, authenticate, and run the Pixshop CLI for terminal-based AI image generation, video creation, media editing, app discovery, and prompt search workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Installing the Pixshop CLI uses a third-party npm package. <br>
Mitigation: Verify the npm package source before global installation. <br>
Risk: Pixshop commands may upload user media to a third-party service. <br>
Mitigation: Avoid uploading sensitive, private, or unauthorized personal images. <br>
Risk: Authentication commands use Pixshop account credentials and local CLI configuration. <br>
Mitigation: Log in only to the intended Pixshop account and keep local configuration files private. <br>
Risk: Batch generation and media operations may consume Pixshop credits. <br>
Mitigation: Review batch commands and credit-consuming operations before running them. <br>
