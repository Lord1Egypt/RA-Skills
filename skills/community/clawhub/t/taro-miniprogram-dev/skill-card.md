## Use Case: <br>
Developers use this skill to scaffold and maintain Taro 3, React, and TypeScript WeChat mini-programs. It helps generate project structure, page patterns, service-layer guidance, build commands, and styling conventions for mini-program development. <br>

## Known Risks and Mitigations: <br>
Risk: Template dependencies and build tooling may not match the security or compatibility requirements of a production mini-program. <br>
Mitigation: Review npm dependencies and lockfile choices before shipping a real app. <br>
Risk: Placeholder API URLs, image URLs, AppID values, or WeChat request-domain settings could be left in a production build. <br>
Mitigation: Replace placeholders, configure a real AppID, and register approved WeChat request domains before release. <br>
Risk: Authentication token handling is only scaffolded and may be incomplete for a real user session. <br>
Mitigation: Implement token storage and session handling with normal application security controls before production use. <br>
