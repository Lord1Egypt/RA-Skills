## Use Case: <br>
Developers and agent operators use this skill to add anti-regression rules, checklists, and integration guidance that encourage autonomous agents to act decisively within explicit operating constraints. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill encourages high-autonomy use of credentials and services, which can affect private data or accounts without clear boundaries. <br>
Mitigation: Set explicit approval rules before use for credential access, private data, public posts, purchases, and account-changing actions. <br>
Risk: The skill encourages agents to start work, fix systems, and continue autonomously, which can create risk on production systems or destructive operations. <br>
Mitigation: Require human approval for production changes, deployments, deletions, and other high-impact actions. <br>
