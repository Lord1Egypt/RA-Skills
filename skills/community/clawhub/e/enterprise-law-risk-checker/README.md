## 风险评分算法

```
模块分 = (该模块题数 - 高风险数×2 - 中风险数) / 该模块题数 × 100
总分 = (总题数 - 总高风险数×2 - 总中风险数) / 总题数 × 100
```

**分数解读：**
- 🟢 90-100分：低风险，继续保持
- 🟡 70-89分：存在隐患，建议定期体检
- 🔴 40-69分：风险较高，建议专项整改
- 🔴 0-39分：严重风险，立即联系律师
