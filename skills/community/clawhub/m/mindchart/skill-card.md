## Description: <br>
Create exquisite infographics from user-uploaded files or given text. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[newtoolai](https://clawhub.ai/user/newtoolai) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users, employees, and developers use this skill to turn text or uploaded-file content into AntV Infographic syntax and rendered infographic assets. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Node/Bun dependencies and rendering packages may introduce ordinary dependency-chain risk. <br>
Mitigation: Install the listed packages from trusted sources in a project-local environment before running the skill workflow. <br>
Risk: Conversion scripts read and write local file paths provided during rendering. <br>
Mitigation: Run the scripts only with input and output paths you intend to use, and review generated SVG and PNG files before sharing. <br>
