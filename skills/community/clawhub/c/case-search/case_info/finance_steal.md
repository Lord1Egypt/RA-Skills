# 财产被盗类案件

案号：(2023)豫16刑终790号
概括：石俊杰在公交车上扒窃他人现金，二审法院认为其行为构成盗窃罪，证据充分、量刑适当，故裁定驳回上诉，维持原判。

案号：(2023)湘3125刑初149号
概括：宋某友多次盗窃某水厂电缆线，数额巨大，其犯罪行为构成盗窃罪，法院综合其前科、坦白、认罪认罚等情节，依法作出判决。

案号：(2023)川0522刑初212号
概括：先某川先以居住为目的非法侵入他人住宅，后又另起犯意秘密窃取财物，法院认为两行为相互独立，分别构成非法侵入住宅罪和盗窃罪，应数罪并罚。

案号：(2023)川1921刑初28号
概括：陈某多次入户盗窃他人财物，情节严重，其行为构成盗窃罪，法院结合其坦白、认罪认罚等情节，依法判处刑罚并责令退赔。

案号：(2023)黑1003刑初66号
概括：孙某某实施盗窃，数额较大，构成盗窃罪，且其系累犯并有多次犯罪前科，法院依法从重处罚，综合其他情节作出判决。

案号：(2023)皖1204刑初176号
概括：张某受委托为被害人申请网贷，贷款到账被害人账户后，张某隐瞒事实并秘密将款项转走，法院认定其行为对实现占有起决定作用，构成盗窃罪而非诈骗罪。

案号：广州互联网法院发布十五起涉网络游戏纠纷典型案例之九：张某诉曹某网络侵害虚拟财产纠纷案——网络虚拟财产价值的确定方法
概括：曹某侵害张某的网络游戏虚拟财产，法院通过一定方法确定了该虚拟财产的价值，判决曹某赔偿相应损失。

案号：(2021)苏01刑终866号
概括：王某偷拿被害人手机后，利用事先窥得的密码秘密转移被害人支付宝绑定的信用卡内资金，法院认为该行为系通过窃取支付密码控制资金，银行并非错误处分，构成盗窃罪。

案号：(2022)鲁0117刑初102号
概括：吴某某等人实施盗窃、非法获取公民个人信息，并为信息网络犯罪活动提供帮助，分别构成盗窃罪、侵犯公民个人信息罪和帮助信息网络犯罪活动罪，法院数罪并罚。

案号：(2021)闽民终302号
概括：荷兰收藏家奥某某持有的肉身坐佛像被确认为中国福建某村被盗的章公祖师像，因其交易违反中国法律且为盗赃文物，不适用善意取得，法院终审判令其返还佛像。
