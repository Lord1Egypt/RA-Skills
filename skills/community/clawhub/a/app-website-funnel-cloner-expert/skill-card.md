## Description: <br>
Expert "Funnel Hacker" skill that analyzes websites, marketing funnels, and apps to create brand new "cloned" versions with original content, using source sites as layout and style templates rather than copying their content. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External developers, marketers, and product builders use this skill to analyze authorized websites, funnels, and app pages, then generate original replacement copy, design guidance, implementation assets, and deployment documentation for a similar but non-copying experience. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Analyzing external URLs or pasted content can expose confidential information or content the user is not authorized to process. <br>
Mitigation: Use the skill only on sites and materials the user owns or is authorized to analyze, and avoid submitting confidential content unless the configured APIs and environment are trusted. <br>
Risk: Generated website, funnel, or app outputs can create originality, copyright, trademark, or brand-confusion issues if reviewed carelessly. <br>
Mitigation: Manually review outputs before publication, replace source-specific branding and media, verify asset licenses, and confirm the final work is original and legally appropriate. <br>
Risk: The skill requires an OpenAI API key and may use URL-fetching or browser-analysis tools during generation workflows. <br>
Mitigation: Run it in a controlled workspace, protect credentials, and grant only the access needed for the specific analysis task. <br>
