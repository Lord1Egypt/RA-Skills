## 1. 核心原理

**Prashna** = 梵语"提问"。在提问瞬间铸造星盘，基于**共时性原则**——内在关切与天象的对应。《Prashna Marga》称之为 **"prashna shakti"（提问之力）**。

| 维度 | 本命盘 | Prashna 盘 |
|------|--------|------------|
| 时间基准 | 出生时刻 | 提问时刻 |
| 覆盖范围 | 一生模式 | 单一具体问题 |
| 出生数据 | 必需 | 不需要 |

**权威经典：**

| 经典 | 核心贡献 |
|------|---------|
| Prasna Marga (Nilakantha, 17世纪) | Prashna 最高权威，32章 |
| Tajika Neelakanthi | Tajika Yoga 系统 |
| Shatpanchasika | 古典 Prashna 经典 |
| Krishneeyam | 喀拉拉邦 Prashna 流派 |

## 2. 星盘铸造与Kunda验证

### 2.1 铸造五要素

1. **精确时间** — 上升点约每2小时变化一次
2. **精确地点** — 经纬度影响上升点
3. **可靠计算** — Swiss Ephemeris
4. **Kunda验证** — 验证上升点正确性
5. **记录原话** — 问题措辞影响解读方向

### 2.2 Kunda 验证法（Prasna Marga Ch.5）

1. Lagna 经度 → 弧分
2. × 81（Kunda 常数）
3. ÷ 12 取余
4. 余数 → Nakshatra
5. 若与出生 Nakshatra（或三合）一致 → 正确
6. 不一致 → 每差1个 Nakshatra 修正10弧分

**示例：** Lagna 11°34' = 694' × 81 = 56,214 ÷ 12 → 余6 = Ārdrā

## 3. 十大核心计算体系

### 3.1 Arudha Lagna（映像上升点）

**概念：** 揭示"表象"而非"真实"。对比 Lagna（真实）vs AL（感知）。

**计算（BPHS 法）：**
1. 从上升数到上升主星所在宫位，记录宫数 N
2. 从上升主星位置数 N 个宫位 → **Arudha Lagna**
3. **例外：若落在1宫或7宫，再数10个宫位**

**速查表：**

| 上升主星宫位 | AL宫位 | 上升主星宫位 | AL宫位 |
|---|---|---|---|
| 1宫 | **10宫** | 7宫 | **10宫** |
| 2宫 | 3宫 | 8宫 | 3宫 |
| 3宫 | 5宫 | 9宫 | 5宫 |
| 4宫 | 4宫 | 10宫 | 4宫 |
| 5宫 | 9宫 | 11宫 | 9宫 |
| 6宫 | 11宫 | 12宫 | 11宫 |

> AL 不可能出现在 1、2、6、7、8、12 宫

**双主星（天蝎/水瓶）判断优先级：**
1. 不在该星座中的行星优先
2. 伴星更多者更强
3. 被木星/水星 Drishti 者更强
4. 擢升 > Mulatrikona > 本宫

**解读：**

| Lagna | AL | 解读 |
|-------|----|------|
| 强 | 受克 | 实际好，但感到焦虑 |
| 弱 | 强 | 看似好，实际有问题 |
| 强 | 强 | 内外一致，清晰 |
| 弱 | 弱 | 实际和感知都有问题 |

**Bhava Pada：** 每个宫都有 Arudha（Pada），用该宫主星替代上升主星，算法相同。

### 3.2 Trisphuta（三重合点）

**公式：**
```
Trisphuta = 太阳经度 + 月亮经度 + 上升经度  (若≥360°则减360°)
```

**Prasna Marga 修正版（更精确）：**
```
Trisphuta = Lagna + 月亮 + Gulika
```

| 落位 | 含义 |
|------|------|
| 吉星星座/宫位 | 正面流动 |
| 被凶星折磨 | 内在冲突，健康问题 |
| 6/8/12 宫 | 苦难、疾病、隐藏挣扎 |
| 木星联系 | 保护与智慧 |

### 3.3 五大 Sphuta 组合（Prasna Marga 第17-18颂）

| 名称 | 公式 | 用途 |
|------|------|------|
| **Trisphuta** | Lagna + 月亮 + Gulika | 三因子 |
| **Catusphuta** | Trisphuta + 太阳 | 四因子 |
| **Pancasphuta** | Catusphuta + Rahu | 五因子终极 |

**示例：** Lagna 43°9' + 月亮 18°29' + Gulika 64°26' = **126°4'** (Trisphuta)

### 3.4 Prana / Deha / Mrityu Sphuta

| Sphuta | 含义 | 公式 |
|--------|------|------|
| **Prana** | 生命之息 | (Lagna × 5) + Gulika |
| **Deha** | 身体 | (月亮 × 8) + Gulika |
| **Mrityu** | 死亡指标 | (Gulika × 7) + 太阳 |

**判断：** Prana + Deha > Mrityu → 寿命长；Prana + Deha < Mrityu → 危险

**基于时间的替代法：** `Y = (提问 Ghatika × 35) ÷ 日长`

| 星期 | Mrityu (Y+) | Kala (Y−) |
|------|---|---|
| 日 | +30° | −30° |
| 一 | +300° | −300° |
| 二 | +75° | −75° |
| 三 | +155° | −155° |
| 四 | +120° | −120° |
| 五 | +210° | −210° |
| 六 | +255° | −255° |

### 3.5 Gulika（古利卡）

**概念：** 土星的暗影子行星，与业力相关。

**计算：** 白昼（或夜晚）均分8份 → 土星份额结束时的上升经度 = Gulika

| 星期 | 白天(日出后Ghatika) | 夜间(日落后Ghatika) |
|------|---|---|
| 日 | 26 | 10 |
| 一 | 22 | 6 |
| 二 | 18 | 2 |
| 三 | 14 | 26 |
| 四 | 10 | 22 |
| 五 | 6 | 18 |
| 六 | 2 | 14 |

> 若实际日长 ≠ 30 Ghatika，按比例调整。1 Ghatika = 24分钟。

### 3.6 Chor Graha（盗贼行星——失物专用）

| 宫位 | 代表 |
|------|------|
| 2宫 | 失物本身 |
| 4宫 | 位置/安全 |
| 7宫 | **窃贼** |
| 8宫 | 隐藏 |
| 11宫 | **恢复** |

**窃贼识别：**

| 条件 | 判断 |
|------|------|
| 7宫主凶星 + 联系Lagna | 熟人/附近人 |
| 7宫主在 Trikasthana(6,8,12) | 隐秘/远方人 |
| 7宫主在 Kendra(1,4,7,10) | 已知/附近人 |
| 7宫凶星 | 有预谋盗窃 |
| 7宫吉星 | 意外/无意拿走 |

**物品行星对应：** 水星=文件/手机、金星=珠宝、月亮=个人物品、火星=工具/车辆、木星=书籍/黄金、土星=旧物/家具、Rahu=欺诈/网络、Ketu=疏忽遗失

**方向（月亮或指标星星座元素）：** 火象→东、土象→南、风象→西、水象→北

### 3.7 Sneha / Viraha 组合

- **Sneha（友善）：** 征象星互相友好星座/相位 → 有利
- **Viraha（敌对）：** 征象星互相敌对星座/无相位 → 不利

### 3.8 Arudha Sphuta

```
Arudha Sphuta = Arudha Rashi + Lagna的度数（仅度数）
```

### 3.9 Pranapada（替代计算）

1. 提问 Ghatika × 120 ÷ 日长 + 太阳经度
2. 调整：固定星座−120°、双体星座+120°、本位星座不变

### 3.10 Mandi vs Gulika

| 学派 | 观点 |
|------|------|
| 南印度（喀拉拉） | 更重视 **Mandi**（Gulika 时段中点的上升度数） |
| 北印度 | 更重视 **Gulika**（时段端点的上升度数） |
| Prasna Marga | 视二者为同一点 |

## 4. 征象星识别体系

| 角色 | 征象星 |
|------|--------|
| **提问者** | ① 上升主星 ② 月亮（共同征象星） |
| **所问事物** | 对应宫位的主星 + 自然 Karaka |

**自然 Karaka：** 太阳=权威/父亲、月亮=母亲/心智、火星=兄弟/房产、水星=沟通/商业、木星=智慧/子女/丈夫(女盘)、金星=婚姻/妻子(男盘)、土星=长寿/延迟、Rahu=执念/海外、Ketu=超脱/损失

**行星特殊状态 Prashna 解读：**

| 状态 | Prashna 含义 |
|------|-------------|
| 逆行 | 延迟、重返、重新考虑 |
| 燃烧 | 混乱、行星功能减弱 |
| 落陷 | 脆弱/失败（除非 Neecha Bhanga） |

## 6. 问题类型专题

### 6.1 事业/工作
- **宫位：** 10宫（事业）、6宫（日常工作）、11宫（收益）
- **征象星：** 10宫主 + 太阳
- **成功：** 上升主与10宫主 Ithasala + 木星相位10宫
- **失败：** 土星/Rahu 影响10宫 + Ishrafa

### 6.2 婚姻/关系
- **宫位：** 7宫（伴侣）、5宫（爱情）
- **征象星：** 7宫主 + 金星 + 木星(女盘丈夫)
- **和谐：** 金星强旺 + 7宫主与上升主 Ithasala

### 6.3 财务/投资
- **宫位：** 2宫（积蓄）、11宫（收益）
- **征象星：** 2宫主 + 11宫主 + 木星
- **有利：** 木星强 + 2/11宫受吉星影响

### 6.4 健康/疾病
- **宫位：** 1宫（身体）、6宫（疾病）、8宫（危险）
- **好转：** 强上升主 + 6宫主弱
- **恶化：** 参考 Mrityu Chakra（§9）

### 6.5 遗失物品/盗窃
- **宫位：** 2宫（物品）、4宫（位置）、7宫（窃贼）、11宫（恢复）
- **找回指标：** 11宫主相位 Lagna/2宫；月亮强旺；Ithasala
- **方向：** 按指标星星座元素（火=东、土=南、风=西、水=北）
- **操作流程：** 记录时刻→铸盘→定指标星→分析位置→评估找回概率→方向搜索

### 6.6 旅行/出国
- **宫位：** 3宫（短途）、9宫（长途）、12宫（海外）
- **征象星：** 9宫主 + 木星

### 6.7 诉讼/纠纷
- **宫位：** 6宫（诉讼）、7宫（对手）
- **胜诉：** 上升主星力量 > 7宫主

### 6.8 子女
- **宫位：** 5宫（子女）、9宫（福气）
- **征象星：** 5宫主 + 木星

## 9. 死亡轮（Mrityu Chakra）

> ⚠️ 仅用于重病/生命危险类 Prashna

### 计算流程

1. 提问时间 → Vighatika → **÷ 15** = Mrityu Rashi 经度
2. 求太阳、月亮、Gulika 的 Navamsa 位置
3. 组合：
   - Mrityu 太阳 = 太阳 Navamsa + Mrityu Rashi
   - Mrityu 月亮 = 月亮 Navamsa + Mrityu Rashi
   - Mrityu Gulika = Gulika Navamsa **−** Mrityu Rashi

### 断命规则

| 条件 | 结论 |
|------|------|
| Mrityu Gulika 合相 Mrityu 太阳 | ⚠️ **确定死亡** |
| Mrityu Gulika 合相 Mrityu 月亮 | 🔶 **极大危险** |
| Mrityu Gulika 独立 | ✅ **病情好转** |
