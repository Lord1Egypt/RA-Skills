## Description: <br>
Frontend Design Pro helps agents audit, polish, critique, and improve frontend UI code using typography, color, spacing, motion, interaction, and UX-writing guidance. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[antonia-sz](https://clawhub.ai/user/antonia-sz) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and frontend-focused agents use this skill to review, refine, and generate user interfaces with stronger visual hierarchy, accessibility, responsive behavior, motion, and UX copy. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may activate on broad UI or design requests and influence output even when it was not explicitly named. <br>
Mitigation: Scope the requested UI task and review the resulting design guidance before applying changes. <br>
Risk: Design recommendations may conflict with an existing product design system or brand standard. <br>
Mitigation: Prefer the project design system, accessibility requirements, and brand constraints when they conflict with the skill's general guidance. <br>
