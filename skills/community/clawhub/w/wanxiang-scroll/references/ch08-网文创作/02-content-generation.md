## 脑洞生成器

### 开书7件套 - 反套路反转脑洞

**用途**：生成反套路的脑洞创意

**核心功能**：
- 生成3个反套路脑洞
- 每个脑洞包含核心梗、反转点、冲突设计
- 覆盖不同类型的网文题材

**输出格式**：
```
脑洞1：[核心梗] + [反转设计] + [冲突设置]
脑洞2：[核心梗] + [反转设计] + [冲突设置]
脑洞3：[核心梗] + [反转设计] + [冲突设置]
```

### 万能公式书名生成器

**核心公式**：
1. 冲突+金手指（例：《废柴重生：开局获得无敌系统》）
2. 身份+职业（例：《魔道祖师：炼器师》）
3. 动作+目标（例：《逆天改命：我要成神》）
4. 反差+悬念（例：《废柴竟是神之子》）

**生成要素**：核心梗、金手指、主角身份、冲突设计、悬念设置

## 细纲扩展

### 分章细纲生成器

**指令**：
> 你是一位金牌编剧。请根据以下大纲，为我创作第一卷的详细分章细纲。每章需包含：核心情节、出场人物、关键爽点/冲突、结尾钩子

**输出要求**：
- 每章800-2000字
- 得包含冲突或反转
- 结尾需要有钩子
- 体现主角成长

---

### 情节扩写四要素法

1. **拉伸对话**：把一句话变成对话拉扯
2. **分解动作**：一个动作拆成三个细节
3. **扩散反应**：不只是主角反应，环境、NPC都要有反应
4. **加载感官**：视觉、听觉、触觉、嗅觉、味觉

**示例**：
```
情节骨架（200字）：主角收到一条神秘短信，说他的妻子出轨了。

扩写后（2000字）：
拉伸对话："叮——"手机震动，屏幕亮起。林风扫了一眼，瞳孔骤然收缩。
分解动作：他握着手机的手开始颤抖，指节泛白。
扩散反应：同事们投来诧异的目光。
加载感官：烈日当空，但他浑身冰冷。汗水顺着额头流下来，刺痛眼睛。
```
