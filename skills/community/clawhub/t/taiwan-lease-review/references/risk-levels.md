# 條款風險分級機制

## 分級標準

| 分級 | 定義 | 法律效果 | 建議行動 |
|------|------|----------|----------|
| 🔴 違法條款 | 違反強制或禁止規定（民法第71條） | 條款**無效** | 必須刪除或修改為合法版本 |
| 🟡 高風險 | 顯失公平（民法第247-1條）或違約金過高 | 可能被法院宣告無效或酌減 | 強烈建議修改 |
| 🟢 低風險 | 對一方較不利但仍在合理範圍 | 有效但需留意 | 可接受但建議告知風險 |
| ✅ 標準條款 | 符合法律規定及定型化契約規範 | 合法有效 | 無需修改 |

---

## 🔴 違法條款判斷要點

### 住宅租約（依租賃專法）

| 條款內容 | 違反依據 |
|----------|----------|
| 「不得報稅」「稅金由承租人負擔」 | 住宅租賃不得記載事項 |
| 「不得申請租金補貼」 | 住宅租賃不得記載事項 |
| 押金 > 2個月租金 | 租賃專法第4條 |
| 「提前終止罰款3個月租金」 | 不得超過1個月 |
| 「電費每度超過台電最高級距」 | 不得記載事項 |
| 「拋棄所有修繕請求權」 | 民法第429條強制規定 |
| 「不得設籍」 | 租賃專法不得記載事項 |

### 店面/商用租約

店面及商用租約**不完全**受租賃專法規範，以下仍屬違法：

| 條款內容 | 判斷 |
|----------|------|
| 違建仍收取押金 | ⚠️ 違建租約本身效力有爭議 |
| 「承租人拋棄民法第423條租賃物交付義務」 | 🔴 違法，民法強制規定 |
| 違章建築不得約定終止時不退還押金 | 🔴 違反民法第179條不當得利 |
| 排除買賣不破租賃（民法第425條） | 🟡 高風險，可約定排除但效力受限 |

### 車位租約

| 條款內容 | 判斷 |
|----------|------|
| 「停車位不得出租」限制所有權人 | 🔴 違法，物權效力優先於管委會規約 |
| 機械車位尺寸不符導致無法停車 | 🔴 給付不能，租約無效 |
| 「拋棄民法第425條」 | 🟡 高風險 |

## 🟡 高風險條款範例

| 條款 | 風險說明 |
|------|----------|
| 「承租人未告知續約意願視同放棄」 | 需有合理預告期，否則顯失公平 |
| 「出租人不負任何損害賠償責任」 | 概括免責條款可能違反民法第247-1條 |
| 「承租人需負擔房屋稅、地價稅」 | 嚴重偏離民法租賃對價關係 |
| 「違約金為租金的三倍」 | 違約金過高，法院可依民法第252條酌減 |
| 「契約期滿不得續約且無優先權」 | 商業租約中，承租人裝修投入後無法回收 |
| 「管委會辦法優於租約」 | 可能架空租約保障 |

---

## 🟢 低風險條款範例

| 條款 | 說明 |
|------|------|
| 「禁止室內吸菸」 | 合法，屬合理使用限制 |
| 「禁止非法使用」 | 合法 |
| 「遷出時需恢復原狀」 | 合法，但需注意「自然耗損」不在此限 |
| 「承租人每月繳納管理費」 | 常見約定，但須明確金額 |

---

## ✅ 標準條款範例

- 「押金於租期屆滿交還房屋後30日內無息返還」— 符合法規
- 「承租人提前終止需一個月前書面通知」— 合理預告期
- 「出租人於修繕前不得任意終止租約」— 符合民法第440條
- 「水電費按實際使用度數分擔」— 公平合理
