## [1.0.1] — 2026-05-13

### Changed
- Frontmatter `version` field now quoted as a string per ClawHub CLI requirements
- Added this CHANGELOG.md for consistency with the rest of the published skill catalog

### Notes
- No behavior changes in this release. Purely documentation and metadata cleanup.

## [1.0.0] — 2026-04-12

### Added
- Initial release
- Full-lifecycle event planning for any event type (parties, conferences, weddings, fundraisers, retreats, dinner parties, more)
- Event records with status, dates, venue, expected attendance, budget, and notes
- Task checklists with due dates, assignees, and status tracking
- Vendor directory reusable across events
- Volunteer management per event
- Smart planning templates with lead-time-aware checklists
- Cost and budget tracking with per-event reporting
- Proactive nudges for overdue tasks and budget thresholds
- Persistent storage in `event-data.json`
