# Ge Gen Pharmacological Mechanisms

## When to Use
当面对以下任一临床情境时启用本技能：
- 患者出现急性心肌缺血或存在相关风险
- 存在脑供血不足或微循环障碍
- 高血压需辅助干预
- 糖尿病伴胰岛素抵抗或并发症
- 发热需中药解热干预
- 酒精性肝损伤需保肝解酒支持

**禁忌提示**：对黄酮类成分过敏者、出血倾向未控制者（因葛根抑制血小板）禁用。

## Execution Steps
1. **确认活性成分**：识别葛根中的主要药效物质——黄酮类（葛根素、黄豆苷元、黄豆苷）、香豆素类及三萜皂苷。
2. **匹配药理作用**：根据患者主症，对应以下机制：
   - **解热**：调节体温中枢，降低发热反应
   - **改善微循环**：扩张毛细血管，提升局部微血流量
   - **抗心肌缺血**：对抗垂体后叶素诱发的心肌缺血
   - **扩张血管**：增加冠脉与脑血流量
   - **降低心肌耗氧量**：优化供氧/耗氧平衡
   - **抑制血小板凝集**：减少血栓形成风险
3. **评估扩展适应证**：结合现代研究，考虑其在以下方面的辅助价值：
   - 降血压、抗动脉粥样硬化
   - 改善胰岛素敏感性（抗糖尿病）
   - 神经保护（抗氧化、抗凋亡）
   - 解酒保肝（加速乙醇代谢）
   - 抗炎、镇痛及骨质疏松防治
4. **综合判断**：虽作用广泛，仍需辨证使用，不可替代基础治疗。仅作为多系统保护的辅助手段。

> 注意：本技能不用于替代诊断或一线治疗，仅提供药理机制支持以辅助决策。
