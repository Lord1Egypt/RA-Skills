# 义乌品质品控指南

## 一、品质管理流程

### 1.1 三级质检体系

```
第一级：源头质检（档口/工厂）
├── 外观检查：无瑕疵、色差、破损
├── 规格核对：尺寸、重量、材质
├── 功能测试：是否正常工作
└── 包装检查：标签、说明书

第二级：出货抽检（发货前）
├── 抽检比例：5-10%（首单加倍）
├── 不良率控制：<3%
├── 问题隔离处理
└── 补货/退换

第三级：第三方质检（大额订单）
├── 机构：SGS、TUV、Intertek
├── 费用：$200-500/次
└── 出具质检报告
```

## 二、质检标准

### 2.1 外观标准

| 项目 | 标准 | 检查方法 |
|------|------|----------|
| 颜色 | 与样品一致，无明显色差 | 目测对比 |
| 表面 | 无划痕、污渍、气泡 | 目测+触摸 |
| 边角 | 无毛刺、锐边 | 触摸检查 |
| 印刷 | 清晰、无错位、无漏印 | 目测 |

### 2.2 规格标准

| 项目 | 允差 | 测量工具 |
|------|------|----------|
| 尺寸 | ±3% | 卷尺/卡尺 |
| 重量 | ±5% | 电子秤 |
| 材质 | 与描述一致 | 目测/测试 |

### 2.3 功能标准

```
电子产品：
- 开关正常
- 功能完整
- 无异响
- 电池安装顺畅

机械产品：
- 运转顺畅
- 无卡顿
- 连接牢固

纺织品：
- 无脱线
- 拉链/扣子正常
- 无异味
```

## 三、包装规范

### 3.1 包装标准

```
内包装：
- 单品独立包装
- 标签信息完整（品名、规格、数量）
- 保护性材料（气泡膜/珍珠棉）

外包装：
- 五层瓦楞纸箱
- 封箱胶带加固（H型封法）
- 唛头标识清晰

标识内容：
- 品名
- 数量
- 目的地
- 易碎/防潮标识
- 箱号
```

### 3.2 包装检查

```
✅ 纸箱完好无损
✅ 封箱牢固
✅ 标识清晰正确
✅ 内衬保护到位
✅ 产品无挤压变形
```

## 四、客诉处理

### 4.1 处理流程

```
客户反馈 → 24小时内响应 → 确认问题类型
                              ↓
              ┌───────────────┼───────────────┐
              ↓               ↓               ↓
         质量问题        物流问题        沟通问题
              ↓               ↓               ↓
      补发/退款      协调物流       解释说明
              ↓               ↓               ↓
      追溯供应商      升级方案      优化流程
```

### 4.2 处理标准

```
小问题（不良率<5%）：
- 折扣补偿（5-10%货款）
- 下单优惠

中等问题（不良率5-15%）：
- 补发合格品
- 承担运费

大问题（不良率>15%）：
- 全额退款
- 货物退回
- 追究供应商责任
```

### 4.3 客诉话术

```
第一步：致歉确认
"非常抱歉给您带来不便，我完全理解您的不满。
请放心，我们会立即处理这个问题。"

第二步：收集信息
"为了更快解决问题，请提供：
- 问题产品照片/视频
- 数量
- 您期望的解决方案"

第三步：提出方案
"根据您提供的信息，我们可以：
- 方案A：补发合格品（[X]天内发出）
- 方案B：退款[金额]
- 方案C：下次订单[X]折优惠
您看哪个方案更合适？"

第四步：跟进闭环
"问题已处理，感谢您的理解和耐心。
我们会加强质检，避免类似问题再次发生。
如有其他问题，随时联系我。"
```

## 五、供应商质量评估

### 5.1 评估指标

| 指标 | 权重 | 评分标准 |
|------|------|----------|
| 不良率 | 30% | <1%=10分，1-3%=8分，3-5%=6分 |
| 交期准确率 | 25% | >95%=10分，90-95%=8分 |
| 问题响应 | 20% | 24h内=10分，48h内=8分 |
| 配合度 | 15% | 主动配合=10分，被动配合=6分 |
| 改进意愿 | 10% | 主动改进=10分，被动改进=6分 |

### 5.2 分级管理

```
A级供应商（≥85分）：
- 优先合作
- 首推给客户
- 年度返点

B级供应商（70-85分）：
- 正常合作
- 定期评估
- 要求改进

C级供应商（<70分）：
- 限制合作
- 要求整改
- 整改不力淘汰
```

## 六、质量改进

### 6.1 问题记录

```
建立质量问题台账：
- 日期
- 产品
- 问题描述
- 影响范围
- 原因分析
- 处理措施
- 预防方案
```

### 6.2 根本原因分析

```
5Why分析法：
问题：产品质量不稳定
→ 为什么？供应商原料波动
→ 为什么？供应商检测不严
→ 为什么？没有统一标准
→ 为什么？缺乏培训
→ 为什么？管理意识不足

根本原因：供应商质量意识不足
解决方案：培训+标准+考核
```

### 6.3 PDCA循环

```
Plan（计划）：制定质量目标、标准
Do（执行）：实施质检、记录数据
Check（检查）：分析数据、找问题
Act（改进）：优化流程、防止复发
```

## 七、特殊品类质检要点

### 7.1 电子产品

```
必检项：
- 通电测试
- 功能测试
- 安全检查（无漏电、发热）
- 配件齐全

认证检查：
- CE（欧盟）
- FCC（美国）
- RoHS（环保）
- 3C（中国）
```

### 7.2 纺织品

```
必检项：
- 尺寸测量
- 色牢度
- 缝线强度
- 面料成分

安全检查：
- 无尖锐物
- 无有害物质
- 无异味
```

### 7.3 玩具

```
必检项：
- 年龄标识
- 安全警告
- 小零件测试
- 边缘检查

认证要求：
- EN71（欧盟）
- ASTM（美国）
- CCC（中国）
```
