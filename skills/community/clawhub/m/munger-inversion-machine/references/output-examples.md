# Output Examples

Use these examples as shape guidance.

## Business Idea

User:

```text
Invert my idea: a paid marketplace for Claude Code skills.
```

Output shape:

```text
# Inversion: Paid Marketplace For Claude Code Skills

## One-Line Verdict
This fails if creators upload supply before buyers have a repeated reason to search, trust, and pay.

## What Has To Be True
Developers must believe skills save enough time or make enough money to justify payment, and the marketplace must already have demand.

## How This Fails
| Failure Mode | Why It Happens | Severity | Early Warning |
| --- | --- | --- | --- |
| No buyer habit | Users install free GitHub skills instead | Fatal | Listings get views but no purchases |
```

## Offer

User:

```text
Invert my offer: 72-hour website fix sprint for local businesses.
```

Output shape:

```text
# Inversion: 72-Hour Website Fix Sprint

## One-Line Verdict
The offer fails if the buyer sees website quality as cosmetic rather than lost leads.

## Stupidity Filter
Do not pitch "better design." Pitch missed calls, weak trust, poor mobile conversion, or quote-request leakage.
```

## Personal Decision

User:

```text
Invert quitting my job to build apps full-time.
```

Output shape:

```text
# Inversion: Quitting To Build Apps Full-Time

## One-Line Verdict
This fails if runway pressure forces rushed decisions before any repeatable acquisition channel exists.

## Smallest Smart Test
Pre-sell one offer or get 3 paid customers before quitting.
```
