# Max-Self-Improvement 架构详解

## 5层记忆体系统架构图

```
┌─────────────────────────────────────────────────────────────────┐
│                     User Input / Task                            │
└─────────────────────────────┬───────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  Layer 1: Sensory Memory (瞬时感知)                              │
│  - Current context window                                        │
│  - Recent 3 turns of conversation                                │
│  - Active tool outputs                                           │
└─────────────────────────────┬───────────────────────────────────┘
                              │ 晋升触发:
                              │ - 用户明确偏好
                              │ - 关键结果返回
                              │ - 错误发生
                              │ - 用户纠正
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  Layer 2: Working Memory (短期记忆)                              │
│  /memories/session_notes.md                                      │
│  - Task ID, current focus                                        │
│  - Intermediate results                                          │
│  - Pending steps                                                 │
│  - Active constraints                                            │
└─────────────────────────────┬───────────────────────────────────┘
                              │ 任务完成/晋升条件
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  Layer 3: Long-Term Memory (长期记忆)                            │
│  /memories/user_preferences.md  ─ 用户偏好 [永不删除]            │
│  /memories/domain_knowledge.md  ─ 领域知识                       │
│  /memories/workflow_library.md   ─ 工作流程库                    │
│  /memories/project/[slug]/       ─ 项目级记忆                     │
│                                                                  │
│  置信度: [unconfirmed] → [low] → [medium] → [high] → [validated]│
└─────────────────────────────┬───────────────────────────────────┘
                              │ 值得记录的事件
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  Layer 4: Episodic Memory (情境记忆)                             │
│  /memories/evolution/patterns.md  ─ 成功模式库                   │
│  /memories/evolution/lessons.md   ─ 失败教训库                    │
│  /memories/evolution/cases/[YYYY-MM]/[id].md  ─ 原始Episode     │
│                                                                  │
│  Episode 结构:                                                    │
│  情境 → 动作 → 结果 → 根因分析 → 提取模式 → 标签                   │
└─────────────────────────────┬───────────────────────────────────┘
                              │ 每5任务/异常触发
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  Layer 5: Metacognitive Memory (元认知)                          │
│  /memories/evolution/metrics.md                                 │
│  - 任务成功率                                                     │
│  - 平均完成轮次                                                   │
│  - 记忆命中率                                                     │
│  - 用户纠正频率                                                   │
│                                                                  │
│  输出: Self-Improvement Directives (行为改进指令)                 │
└─────────────────────────────────────────────────────────────────┘
```

## 自我进化循环详解

```
                    ┌──────────────────────┐
                    │    Task Received     │
                    └──────────┬───────────┘
                               │
                               ▼
              ┌────────────────────────────────┐
              │  读取记忆 (Layer 2-5)          │
              │  检索适用模式                  │
              └───────────────┬────────────────┘
                              │
              ┌───────────────┴───────────────┐
              │                               │
              ▼                               ▼
    ┌─────────────────┐           ┌─────────────────┐
    │ 有匹配模式       │           │ 无匹配模式       │
    │ 应用 patterns   │           │ 默认推理         │
    └────────┬────────┘           └────────┬────────┘
             │                              │
             └──────────────┬───────────────┘
                            │
                            ▼
              ┌────────────────────────────────┐
              │        执行任务                 │
              └───────────────┬────────────────┘
                              │
                              ▼
              ┌────────────────────────────────┐
              │        评估结果                 │
              └───────────────┬────────────────┘
                              │
         ┌────────────────────┼────────────────────┐
         │                    │                    │
         ▼                    ▼                    ▼
┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
│ Success         │  │ Partial         │  │ Failure         │
│ 提取成功模式     │  │ 更新置信度       │  │ 根因分析        │
│ 写入patterns.md │  │                 │  │ 写入lessons.md  │
└─────────────────┘  └─────────────────┘  └─────────────────┘
         │                    │                    │
         └────────────────────┴────────────────────┘
                              │
                              ▼
              ┌────────────────────────────────┐
              │  知识提升到 Long-Term Memory    │
              └───────────────┬────────────────┘
                              │
                              ▼
              ┌────────────────────────────────┐
              │  更新 metrics.md               │
              └───────────────┬────────────────┘
                              │
              ┌───────────────┴───────────────┐
              │                               │
              ▼                               ▼
    ┌─────────────────┐           ┌─────────────────┐
    │ 每5任务触发     │           │ 上下文溢出风险   │
    │ 元认知审查      │           │ 保存状态+restart │
    │ 生成改进指令    │           │ 从session恢复   │
    └─────────────────┘           └─────────────────┘
```

## 置信度状态机

```
[unconfirmed]
     │
     │ 首次验证
     ▼
[low] ───────────────────────────────────┐
     │                                    │
     │ 再次验证                           │ 反例推翻
     ▼                                    │
[medium] ────────────────────────────────┤
     │                                    │
     │ 再次验证 (累计3次+)                 │
     ▼                                    │
[high] ───────────────────────────────────┤
     │                                    │
     │ 同一模式验证3次以上                 │ 被反例推翻
     ▼                                    │
[validated] ◄─────────────────────────────┘
```

## 记忆体文件路径规范

```
/memories/
├── session_notes.md              # 当前会话状态
├── user_preferences.md           # 用户偏好 [永久]
├── domain_knowledge.md            # 领域知识
├── workflow_library.md            # 工作流程库
├── project/
│   └── [project-slug]/
│       └── context.md             # 项目上下文
└── evolution/
    ├── patterns.md                # 成功模式
    ├── lessons.md                 # 失败教训
    ├── metrics.md                 # 性能指标
    └── cases/
        └── [YYYY-MM]/
            └── [id].md            # 原始Episode
```
