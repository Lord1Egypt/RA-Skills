# 常见PPT页面设计模式

本文档整理了常见的PPT页面布局模式，可用于风格统一的页面重设计参考。

## 一、封面页模式

### 模式1：居中大标题
- **布局**：标题居中，上下留白
- **元素**：主标题、副标题、装饰线、背景图
- **适用**：正式、简约风格

### 模式2：左文右图
- **布局**：左侧标题文字，右侧配图
- **元素**：主标题、副标题、说明文字、大图
- **适用**：商务、科技风格

## 二、目录页模式

### 模式1：数字编号列表
- **布局**：左侧大数字，右侧标题+描述
- **元素**：01/02/03编号、章节标题、简短描述
- **适用**：专业咨询风格

### 模式2：图标卡片网格
- **布局**：2x2或3x2网格，每个章节一个卡片
- **元素**：图标、标题、简短说明
- **适用**：活泼、现代风格

## 三、章节过渡页模式

### 模式1：大数字+标题
- **布局**：左侧大号章节编号，右侧章节标题
- **元素**：01、章节大标题、简短引言
- **适用**：正式、大气风格

### 模式2：全屏图+文字
- **布局**：全屏背景图，文字叠加
- **元素**：半透明遮罩、章节标题、副标题
- **适用**：视觉冲击力强的风格

## 四、内容页模式

### 模式1：左文右图（经典）
- **布局**：左侧文字内容，右侧配图
- **比例**：文字约60%，图片约40%
- **元素**：小标题、段落文字、装饰竖条
- **适用**：公司介绍、项目背景

### 模式2：上文下图
- **布局**：顶部标题文字，底部大图
- **元素**：大标题、说明文字、全屏图
- **适用**：产品展示、案例介绍

### 模式3：三栏卡片
- **布局**：三个等宽卡片横向排列
- **元素**：每个卡片含图标/编号、标题、描述
- **适用**：三大优势、三个特点、三大价值

### 模式4：四象限
- **布局**：2x2四宫格布局
- **元素**：四个模块，每个含标题+内容
- **适用**：SWOT分析、矩阵图、分类说明

### 模式5：时间轴
- **布局**：横向或纵向时间线
- **元素**：时间节点、事件标题、详细说明
- **适用**：发展历程、项目里程碑、流程步骤

### 模式6：左右对比
- **布局**：左右两栏对比
- **元素**：VS分隔、左右各一组内容
- **适用**：前后对比、优劣对比、方案对比

## 五、数据页模式

### 模式1：大数字突出
- **布局**：3-4个关键数据横向排列
- **元素**：大号数字、单位、指标名称
- **适用**：核心数据、业绩成果、关键指标

### 模式2：图表+说明
- **布局**：左侧图表，右侧文字说明
- **元素**：柱状图/饼图/折线图、数据解读、关键洞察
- **适用**：数据分析、趋势展示

## 六、总结页模式

### 模式1：金句+要点
- **布局**：顶部大字号金句，底部要点列表
- **元素**：引用样式的金句、3-5个核心要点
- **适用**：价值主张、核心理念、总结升华

### 模式2：愿景展望
- **布局**：背景图+半透明遮罩+文字
- **元素**：愿景标题、目标描述、行动号召
- **适用**：未来规划、项目愿景、结尾页

## 七、通用设计元素

### 装饰元素
1. **左侧竖条**：标题或内容块左侧的彩色竖线，用于强调
2. **底部分隔线**：标题下方的细线，增强层次感
3. **顶部色块**：页面顶部的通栏色条，用于标题区
4. **底部装饰条**：页面底部的色条，可放金句或页脚
5. **编号样式**：大号数字编号，可圆形、方形、纯数字

### 卡片样式
1. **纯色背景卡**：浅色背景，无边框，柔和
2. **描边卡片**：白色背景+彩色边框，清爽
3. **阴影卡片**：白色背景+投影，有立体感
4. **渐变卡片**：渐变色背景，现代感强

### 文字层级
1. **一级标题**：32-36pt，加粗，主色或深灰
2. **二级标题**：20-24pt，加粗，深灰
3. **三级标题**：16-18pt，加粗，主色
4. **正文**：14-16pt，常规，深灰
5. **注释/小字**：12-13pt，常规，浅灰

## 八、风格匹配检查清单

- [ ] 主色调与参考页一致
- [ ] 文字颜色与参考页一致
- [ ] 字体与参考页一致
- [ ] 标题字号比例与参考页一致
- [ ] 装饰元素类型与参考页一致（竖条/卡片/编号等）
- [ ] 布局复杂度与参考页相当（形状数量）
- [ ] 页边距与参考页一致
- [ ] 图片使用方式与参考页一致
- [ ] 底部是否有装饰条（与参考页一致）
- [ ] 整体视觉感受与参考页统一
