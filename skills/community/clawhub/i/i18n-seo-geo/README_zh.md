## 这是什么？

**Gingiris SEO & GEO 双引擎增长手册** 是一份面向 2026 年搜索新现实的完整 SEO/GEO 策略指南。当今超过 30% 的搜索流量始于 AI 聊天机器人（ChatGPT、Perplexity、Claude、Gemini），仅优化 Google 已经不够。

本手册教你如何让你的内容同时在传统搜索引擎获得排名，并在 AI 搜索中被引用 — 这就是 GEO（Generative Engine Optimization，生成式引擎优化）。

与过时的 2023 年 SEO 建议（关键词密度、外链堆叠）不同，这本手册覆盖了 JSON-LD Schema 模式、E-E-A-T 写作声音系统、对比页模板、IndexNow 推送等让 AI 引擎实际引用的内容模式。

## 核心框架：SEO + GEO 双引擎

```
  传统搜索 (Google/Bing)
  ┌──────────────────────────┐
  │  关键词 + 反向链接       │ ← SEO
  │  Schema + Core Web Vitals│
  └────────────┬─────────────┘
               │
        ┌──────▼──────┐
        │  结构化内容  │ ← 共同基础
        └──────┬──────┘
               │
  ┌────────────▼─────────────┐
  │  IndexNow + AI 爬虫      │ ← GEO
  │  直接回答 + 对比表格     │
  └──────────────────────────┘
  AI 搜索 (ChatGPT/Perplexity/Claude)
```

## 五个核心原则

1. **从 BOFU 往上做** — 高意向关键词优先（定价页、竞品对比页）
2. **真实声音 = 最好的 E-E-A-T** — 创始人亲历故事、真实数字、第一手经验
3. **结构化 = 可引用** — Key Stats 表格、FAQ Schema、对比矩阵让 AI 引擎能直接引用
4. **IndexNow 是 GEO 的基础设施** — 内容更新后秒级推送，不等爬虫来抓
5. **对比页是 SEO 金矿** — 每个竞品一个对比页面，截获决策阶段用户

## 常见问题

**Q：这和通用 SEO 技能有什么不同？**
A：大多数 SEO 技能还在教 2023 年的打法（关键词密度、外链）。这本手册教你如何为新的双引擎现实写作 — 30%+ 的流量现在来自 AI 聊天机器人。JSON-LD Schema 模式、E-E-A-T 声音、对比页模板都是 AI 引擎实际会引用的。

**Q：包含 Schema 标记和 JSON-LD 模板吗？**
A：包含。提供 SoftwareApplication、Article、FAQ、HowTo、Comparison 的即用模板，以及 IndexNow 推送到 Bing/Yandex 的指南。

**Q：能帮做程序化 SEO 吗？**
A：能。关键词漏斗部分覆盖了程序化 SEO 模式 — 如何识别可规模化的模板、生成不被判定为低质量内容的落地页、以及如何结构化让它们在 AI 搜索概览中被引用。

**Q：中文内容的 SEO/GEO 策略和英文有什么不同？**
A：核心方法论一致，但百度和 Google 的算法权重不同，AI 搜索（文心一言 vs. ChatGPT）的引用偏好也不同。手册中有针对中文搜索环境的专门策略。

**Q：多久能看到效果？**
A：SEO 内容通常 3-6 个月开始显现效果，对比页因为针对高意向关键词，可能 4-8 周就能带来转化。GEO（AI 引用）的反馈更快，通常 2-4 周内可以观察到。
