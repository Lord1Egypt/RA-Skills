## Technical Checklist (Pre-Show)

Complete all items at least 60 minutes before going live.

- [ ] All product links created and pinned in TikTok Shop backend
- [ ] Each product link tested on a separate device (loads correctly, correct price displayed)
- [ ] Discount codes configured and tested
- [ ] Overlay graphics loaded (price comparisons, countdown timers, product labels)
- [ ] Camera, lighting, and audio tested
- [ ] Backup device charged and connected to the same account
- [ ] Moderator briefed on comment response guidelines and giveaway rules
- [ ] Backstage assistant has inventory dashboard open for real-time stock updates
- [ ] Internet connection speed tested (upload speed above 5 Mbps minimum)
- [ ] Physical product samples laid out in presentation order within arm's reach
