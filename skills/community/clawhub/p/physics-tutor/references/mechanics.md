# 费曼力学核心：到底在说什么

## 第一性原理

力学只回答一个问题：**东西为什么会动，为什么会停，以什么方式动？**

## 三个核心概念的关系

### 力（Force）

- 不是"推动的感觉"
- 是**改变运动状态的原因**，不是运动本身
- 一个物体可以受力但不动（平衡），也可以不受力却不停（惯性）

### 质量（Mass）

- 不是"东西有多重"
- 是**惯性的大小**——对抗改变的程度
- 质量是物体保持原状的"倔强程度"

### 运动（Motion）

- 速度是描述，不是原因
- 加速度才是关键——速度在怎么变

## 牛顿三定律的真正含义

### 第一定律（惯性定律）

> "东西不想动，也不想停"

物体没有偏好，它只是在延续它现在的状态。
不受力 → 保持匀速直线运动或静止。
**惯性不是原因，是描述。**

### 第二定律（F=ma）

- 不要背公式
- 理解：**力决定了质量如何加速**
- 同一条力，质量越大加速越小
- 这是因果关系，不是定义

### 第三定律（作用反作用）

- 不是"一个推一个拉"
- 是：**力永远是成对出现**，来自同一个相互作用
- 作用力和反作用力永远不在同一个物体上

## 费曼式理解力的关键问题

1. **谁在受力？**
2. **什么相互作用产生的？**
3. **这个力方向朝哪？大小多少？**
4. **受力之后物体会怎么动？**

## 受力分析：一切的基础

### 画受力图的标准步骤

1. **先找场**（重力一定画，其他场按题目给）
2. **再找接触**（弹力、摩擦力、支持力——有接触面才有）
3. **标方向**（重力竖直向下，弹力垂直于接触面，摩擦力沿接触面）
4. **标符号**（常用 G、N、f、F 等）

### 平衡状态

- 静止或匀速直线运动
- 合力为零：ΣF = 0
- **解题方法：** 正交分解 → 两个方向分别列方程

## 运动学：从加速度出发

### 核心公式（匀变速直线运动）

| 公式 | 适用场景 |
|------|---------|
| v = v₀ + at | 已知初速、加速度、时间 |
| x = v₀t + ½at² | 已知初速、加速度、位移 |
| v² - v₀² = 2ax | 已知初速、末速、位移（不含时间） |
| x = ½(v + v₀)t | 已知初速、末速、时间（不含加速度） |

**记不住？** 记住第一个 v = v₀ + at，其他三个用图像面积推。

### 图像法（最重要的工具）

**v-t 图：**
- 斜率 = 加速度
- 面积 = 位移

**x-t 图：**
- 斜率 = 速度
- 切线斜率 = 瞬时速度

### 平抛运动

分解为：
- 水平：匀速直线运动（vx = v₀）
- 竖直：自由落体（vy = gt）

**合速度：** v = √(vx² + vy²)
**轨迹：** 抛物线 y = (g/2v₀²)x²

### 圆周运动

**线速度：** v = 2πr/T
**角速度：** ω = 2π/T = v/r
**向心加速度：** a = v²/r = ω²r

**向心力：** F = ma = mv²/r

**重要：** 向心力是效果力，不是新的力——由其他力提供（绳子弹力、重力分力、摩擦力等）。

## 牛顿第二定律的应用

### 已知受力，求运动

**标准步骤：**
1. 受力分析 → 画受力图
2. 正交分解（一般选加速度方向为一个坐标轴）
3. 列方程：ΣF = ma（两个方向）
4. 求解

### 已知运动，求受力

**标准步骤：**
1. 运动分析 → 求出加速度
2. 用 F = ma 求合力
3. 用合力求某个未知力

## 能量角度：更普适的方法

### 动能定理

> **W_合 = ΔEk = ½mv₂² - ½mv₁²**

合外力做功 = 动能变化。

**优势：** 用功和能的角度解题，不需要知道中间过程的加速度。

### 机械能守恒

> **在只有重力/弹力做功的系统里：E₁ = E₂**

当机械能守恒时，**用能量守恒往往比用牛顿定律更简单**。

**判断标准：** 有没有摩擦/空气阻力？有没有除重力/弹力以外的外力？

## 常见误解

### ❌ 误解1："有力就运动，没力就停止"

有力不一定运动——有力也可以保持静止（平衡状态）。
没力也不一定停止——不受力时会保持匀速直线运动（惯性）。

### ❌ 误解2："速度大惯性大"

惯性只和**质量**有关，和速度无关。
质量越大，惯性越大——改变它的运动状态越难。

### ❌ 误解3："作用力和反作用力互相抵消"

作用力和反作用力**永远作用在不同物体上**，不可能抵消。
它们是一对力，但各管各的物体。

### ❌ 误解4："向心力是一种新的力"

向心力是**其他力充当的效果**。
说"受向心力"是方便说法，实际上是绳子拉力、重力分力等提供了向心力。

### ❌ 误解5："平抛运动中 v 和 a 同步变化"

竖直方向速度在变（g 一直作用），但水平方向速度不变（没有加速度）。
合速度方向和加速度方向**不同步**。

## 力学这张网的连接点

```
力学
├── 连接能量：动能定理是牛顿第二定律的积分形式
├── 连接振动与波：弹簧振子是力学+简谐运动
├── 连接电磁：带电粒子在电磁场中的运动
└── 连接热学：气体压强是分子碰撞的统计结果
```

## 力学解题的两条主线

**主线A（力→运动）：** 受力分析 → F=ma → 运动学公式
**主线B（功→能）：** 做功分析 → 动能定理或能量守恒

拿到一道题，先判断：
- 已知受力 → 走主线A
- 已知运动状态变化 → 走主线B
- 涉及高度/速度/距离且无摩擦 → 优先考虑能量守恒
