**制衡三原则**（鬼谷子）：
1. **钩箝调和**："钩"（吸引）和"箝"（约束）并用
2. **阴阳其和**：甜头（阳）和约束（阴）配套
3. **终始其义**：制衡要始终一致

| 你 vs A | 稳定 | | |
| 你 vs B | 稳定 | | |
| A vs B | 稳定 | | |

**裂缝处理策略**：
- **小隙** → 修补（私下沟通）
- **中隙** → 监控+缓冲（增加沟通频率）
- **大隙** → 利用或退出（调整联盟结构）

> ⚠️ "巇者，罅也。罅者，涧也。涧者，成大隙也。"——小裂缝会变大，不要等。

## 快速权衡口诀

1. **画地图**：搞清楚所有人的利益关系
2. **找盟友**：找到自然利益共同体
3. **算总账**：不只算短期收益
4. **设保险**：给每个关系装制衡机制
5. **盯裂缝**：每周检查一次
