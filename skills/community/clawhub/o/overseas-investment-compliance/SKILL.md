# 对外投资合规诊断Skill

## 概述

**一句话价值**：输入投资场景，**先判定资产流向方向**，再输出从"能不能投"到"怎么合规投"的完整诊断报告。

> ⚠️ **核心原则**：不是所有场景都是"境内→境外"，必须先判定方向再选合规框架。

---

## 触发条件

当用户表达以下意图时，立即激活本Skill：

| 类别 | 用户表述示例 |
|------|-------------|
| **合规诊断** | "我这个投资合规吗"、"能投吗"、"合规路径是什么" |
| **资产出境** | "怎么把钱转到海外"、"ODI备案"、"37号文是什么" |
| **外资入华** | "外国企业来中国投资"、"外商投资怎么审批" |
| **境外资产操作** | "境外公司想在香港上市"、"海外资产配置" |
| **政策咨询** | "境外投资有什么规定"、"港澳投资有什么政策" |

## 工作流程

### Step 0：资产定位与流向判定（基石）

> ⚠️ **必须首先完成此步骤**，不同流向适用完全不同的合规框架。

通过提问确认两个维度：

| 维度 | 采集问题 |
|------|----------|
| **资产当前位置** | "资产/资金现在在哪里？"（境内/境外/港澳） |
| **目标去向** | "打算投资到哪个地区？"（境外/境内/港澳） |

#### 合规框架匹配矩阵

| 资产位置 | 去向 | 适用合规框架 | 典型场景 |
|---------|------|-------------|---------|
| **境内** | 境外 | ODI备案+发改委+商务部+外管 | 境内企业出海投资 |
| **境外** | 境内 | 外商投资法+负面清单+安全审查 | 外资收购中国企业 |
| **境外** | 境外（跨境） | 目标地法规+交易所规则 | 境外资产港股上市 |
| **境内** | 港澳 | 参照执行+CEPA特殊安排 | 内地企业赴港上市 |
| **境外** | 港澳 | 港交所规则+资产所在国法规 | 境外公司港股上市 |
| **境内** | 境内 | 无对外投资合规问题 | 纯国内项目 |

> **关键提示**：只有"境内→境外"才涉及ODI备案，其他方向完全适用不同框架。

---

### Step 1：信息采集

根据流向判定结果，按适用框架采集信息：

| 维度 | 采集问题 | 必填 |
|------|----------|------|
| **投资主体** | "您是央国企、民企还是个人？" | ✅ |
| **资产位置确认** | 资产目前在哪里？（境内/境外/港澳） | ✅ |
| **投资目的地** | "投资目标国家/地区是哪里？" | ✅ |
| **投资行业** | "投资的是哪个行业/领域？" | ✅ |
| **投资方式** | "是直接投资、离岸架构还是其他方式？" | ✅ |
| **投资规模** | "预计投资金额大约多少？" | 建议提供 |

> **交互技巧**：先问清楚"钱在哪里"、"要去哪里"，再决定后续问题。

---

### Step 2：合规判定

基于流向框架+检查矩阵，输出三色评级：

| 评级 | 含义 | 下一步 |
|------|------|--------|
| 🟢 允许投资 | 政策明确支持或无明确限制 | 告知最优路径 |
| 🟡 需审批/限制 | 需满足特定条件或需多部门审批 | 告知审批路径 |
| 🔴 禁止投资 | 明确禁止或高风险不建议 | 告知原因+替代方案 |

---

### Step 3：审批路径图

根据流向框架，绘制适用审批流程：

**境内→境外（ODI框架）**：
```
发改委备案 → 商务部备案 → 外汇登记 → ODI专户
```

**境外→境内（外商投资框架）**：
```
商务部门准入审批 → 市场监管登记 → 安全审查（如触发）→ 行业许可
```

---

### Step 4：风险标记

| 风险类型 | 标记 | 说明 |
|----------|------|------|
| 地缘政治风险 | 🌍 | 目的地国审查机制 |
| 穿透审查要求 | 🔍 | 离岸架构穿透 |
| 数据安全风险 | 💾 | 数据跨境传输 |
| 技术出口管制 | 🚫 | 管制技术 |
| 外汇管制风险 | 💱 | 资金进出限制 |
| 税务风险 | 📊 | CRS、CFC |

---

### Step 5：推荐合规方案

```
【最优合规路径】（根据Step 0判定选择）

境内→境外：ODI备案 → 37号文（如涉及个人） → 外汇登记 → ODI专户
境外→境内：外商投资准入 → 负面清单核查 → 安全审查 → 企业登记
境外→境外：目标地法规 → 交易所规则 → 资产所在国审批
```

## 合规检查矩阵

### 三类主体×三种流向

| 流向 | 央国企 | 民企 | 居民个人 |
|------|--------|------|----------|
| **境内→境外** | ODI+国资委前置 | ODI备案 | 37号文 |
| **境外→境内** | 外商投资法+安全审查 | 外商投资法+安全审查 | 台胞投资保护法（如适用） |
| **境内→港澳** | CEPA+参照执行 | CEPA+参照执行 | 参照执行 |

### 行业准入检查

| 行业类别 | 境内→境外 | 境外→境内 |
|----------|-----------|-----------|
| **鼓励类**（环保、新能源） | 🟢 支持 | 🟢 欢迎 |
| **一般制造业** | 🟢 支持 | 🟢 开放（负面清单外） |
| **矿产开发** | 🟡 限制 | 🟡 限制（需中方控股） |
| **房地产/酒店** | 🔴 限制 | 🔴 限制 |
| **高科技/AI** | 🟡 需审批 | 🟡 需安全审查 |
| **新闻传媒** | 🔴 禁止 | 🔴 禁止 |

## 典型场景示例

### 场景1：境内企业→境外（绿地投资）

```
【资产定位】境内资金→境外某国
【投资方】境内民营企业
【投资内容】东南亚某国设立制造工厂
【投资金额】5000万美元

【Step 0判定】✅ 境内→境外，适用ODI备案框架

【合规判定】🟢 基本开放

【审批路径】
发改委备案 → 商务部备案 → 外汇登记 → ODI专户 → 资金汇出

【材料清单】
□ 境外投资备案表 □ 董事会决议 □ 可行性研究报告
□ 股权穿透图 □ 审计报告 □ 资金来源证明

【时间周期】25-45工作日
【成本估算】2-5万元
```

### 场景2：境外企业→境内（外资入华）

```
【资产定位】境外资金→中国境内
【投资方】某境外企业
【投资内容】收购中国某省矿产资源开发项目
【投资金额】1亿美元

【Step 0判定】✅ 境外→境内，适用外商投资框架

【合规判定】🟡 限制类+需安全审查

【核心障碍】
1. 矿产开发属限制类，需中方控股
2. 涉及战略资源，触发安全审查
3. 多部门联合审批

【审批路径】
商务部门准入 → 矿业权申请 → 安全审查 → 环评 → 项目核准 → 企业登记

【材料清单】
□ 外商投资准入申请 □ 注册文件公证认证
□ 资金来源说明 □ 可行性报告 □ 环评报告

【时间周期】12-24个月
【建议】寻求中方合资伙伴
```

### 场景3：境内个人→境外（不动产）

```
【资产定位】境内个人资金→境外
【投资方】境内居民个人
【投资内容】境外某城市购置住宅房产
【投资金额】200万美元

【Step 0判定】✅ 境内→境外，个人购汇渠道

【合规判定】🔴 禁止个人购汇投资房产

【政策依据】个人购汇禁止用途含境外买房

【替代方案】
| 方案 | 可行性 |
|------|--------|
| 境外已有资金使用 | 🟢 可行 |
| 企业ODI渠道 | 🟡 复杂 |
| 目的地国贷款 | 🟢 可行 |

【风险】地下钱庄/分拆购汇为🔴违法行为
```

### 场景4：境外资产→港澳（港股上市）

```
【资产定位】境外资产→香港
【投资方】境外注册公司（实际控制人为境内居民）
【投资内容】境外科技公司拟在港交所上市
【投资金额】IPO估值10亿美元

【Step 0判定】✅ 境外→港澳，适用港交所规则+资产所在地审批

【合规判定】🟢 政策支持（港股通至2027年税收优惠）

【审批路径】
目标地证监会审批 → 港交所上市申请 → 中国证监会备案（如需）

【关键合规点】
1. 37号文登记（实际控制人）
2. 目标地证券法规
3. CRS合规

【建议】聘请境内外律师团队协同
```

## 边界说明

### 本Skill擅长

✅ 投资方向判定（先问清楚"钱在哪""要去哪"）  
✅ 境内→境外合规路径（ODI备案全流程）  
✅ 境外→境内合规路径（外商投资准入）  
✅ 审批流程梳理  
✅ 材料清单/时间/成本估算  

### 建议转介专业律师

⚠️ 复杂离岸架构（多层BVI→开曼→香港）  
⚠️ 涉及安全审查的跨境并购  
⚠️ 37号文历史遗漏登记  
⚠️ 境外资产港股/美股上市  
⚠️ 目的地国深度合规（CFIUS等）  

### 局限性

- ⚠️ 政策基于公开文件，不含未公开规定
- ⚠️ 不提供法律意见，以主管部门意见为准
- ⚠️ 政策变化可能滞后，建议核实最新规定
