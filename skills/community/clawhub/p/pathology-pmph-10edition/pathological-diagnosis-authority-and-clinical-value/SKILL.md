# 病理学诊断的权威性与临床价值

## 何时使用
- 需要确诊肿瘤或其他疾病的组织学类型；
- 制定个体化治疗方案（如用药、手术范围、放疗剂量）；
- 术中需实时判断病变边界或性质（依赖冷冻切片）；
- 动态评估治疗效果或疾病进展；
- 死亡病例需明确根本死因或验证临床诊断。

## 执行步骤
1. **确认样本类型**：识别当前可用的诊断材料，包括：
   - 手术切除组织
   - 活体穿刺或内镜活检样本
   - 脱落细胞、刮片或体液（如胸腹水、脑脊液）
   - 尸体剖验标本

2. **选择诊断形式**：根据样本类型和临床需求，采用以下一种或多种方法：
   - 组织学诊断（基于HE染色等形态学分析）
   - 细胞学诊断（适用于液体或脱落细胞）
   - 分子病理学诊断（如免疫组化、基因突变检测、FISH等）

3. **评估诊断可靠性**：
   - 活检组织诊断可靠性高于细胞学检查；
   - 尸体剖验为“最终诊断”（final diagnosis），具有最高验证效力；
   - 注意我国尸检率低可能限制诊断闭环验证。

4. **指导临床决策**：
   - 将病理报告作为制定治疗方案的核心依据；
   - 术中冷冻切片结果应即时反馈给外科团队；
   - 动态活检用于监测疗效、耐药或复发。

5. **发挥尸检价值**（如适用）：
   - 明确死因，验证生前诊断；
   - 总结诊疗经验，支持医疗质量改进；
   - 积累教学与科研资源；
   - 为医疗纠纷或事故鉴定提供法律依据。

> **注意**：所有病理诊断必须基于规范处理的样本（固定、切片、染色、分子检测等），否则结果不可靠。
