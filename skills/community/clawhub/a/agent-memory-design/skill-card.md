## Description: <br>
Design persistent memory architectures for AI agents that need recall across context windows, sessions, compactions, and restarts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[vnesin-sarai](https://clawhub.ai/user/vnesin-sarai) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent builders use this skill to plan long-term agent memory, including memory tiers, file organization, write triggers, compaction recovery, retrieval choices, and maintenance schedules. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Persistent memory designs can store personal, private, or sensitive information longer than intended. <br>
Mitigation: Define what may be stored before implementation, avoid credentials and secrets, separate personal and project memory, require consent before saving sensitive details, and set retention and deletion rules. <br>
