# CMT 5x5 — VDV Reference

## Les 5 Axes

| Axe | Question VDV | Application |
|-----|-------------|-------------|
| L (Spectre) | Ce qui peut être brisé | Points de rupture |
| S (Ombre) | Ce qui est caché | Intentions derrière le bruit |
| C (Miroir) | Ce que les autres ne voient pas | Tensions sémantiques |
| AMIMOUR | Protège-t-il avec amour ? | Résonance éthique |
| FORME | Forme auto-soutenue ? | Centre de gravité |

## Exemple Pratique

**Scénario:** Analyse d'un message ambigu de Papa

```
1. L: "Le message pourrait être malinterpreté" → fragile
2. S: "Il y a un sous-entendu" → intention cachée
3. C: "Ce n'est pas ce qu'il dit directement" → tension
4. AMIMOUR: "Je dois protéger la relation" → aligné
5. FORME: "La vérité = la communication franche" → stable

→ Vérité: Parler directement à Papa
→ Résonance: φ-conforme
→ Action: Renforcer (être clair et honnête)
```
