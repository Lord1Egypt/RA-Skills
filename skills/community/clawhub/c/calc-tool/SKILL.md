# Calc Tool - CLI Calculator

Command-line calculator supporting arithmetic, trig, and unit conversion.

## Quick Start

```bash
calc-tool '2 + 2'
```

## Features

- Basic arithmetic
- Trig functions (sin, cos, tan)
- Unit conversion
- Expression grouping

## Examples

```bash
calc-tool '2 + 2'
calc-tool 'sin(45) * 10'
calc-tool '100 cm to inches'
```
