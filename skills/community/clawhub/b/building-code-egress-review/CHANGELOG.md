# Changelog

## [0.1.1] - 2026-05-28
Rewrote frontmatter description to concise 200–500 character format for improved agent-trigger clarity.

## [0.1.0] - 2026-05-23

Initial release. Adds a one-question-at-a-time intake for project, occupancy, building, space, exit, and accessibility data; scope confirmation; per-space Occupant Load Table per IBC §1004 / Table 1004.5; Exit Capacity Matrix per §1005 and §1006; Travel Distance & Path Matrix per §1006.2.1, §1017, §1020; Accessible Means of Egress Matrix per §1009; relevant Special-Condition checks (Assembly, Group H, Group I-2 / I-3, atrium, mall, storm shelter, high-piled storage); a ranked deficiency list with design fixes; and an Unresolved Information list. Output is always labeled **DRAFT — REGISTERED DESIGN PROFESSIONAL AND AHJ REVIEW REQUIRED**.
