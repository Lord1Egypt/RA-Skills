# 行业信息速查模板

## 互联网/科技行业

### 行业趋势（2024-2025）
- **AI驱动变革**：AIGC重塑内容生产，AI产品经理需求激增
- **出海加速**：东南亚、中东成为新增长点
- **降本增效**：精细化运营，数据驱动决策
- **产业互联网**：To B业务成为新增长引擎
- **合规趋严**：数据安全、隐私保护成为重点

### 热门岗位

| 岗位 | 核心要求 | 应届薪资范围 |
|------|---------|-------------|
| 产品经理 | 用户洞察、需求分析、原型设计、数据思维 | 15-30K |
| 算法工程师 | 编程基础、机器学习、论文复现、竞赛经历 | 25-50K |
| 前端开发 | HTML/CSS/JS、React/Vue、工程化 | 15-30K |
| 后端开发 | 编程语言、数据库、系统设计 | 18-35K |
| 数据分析 | SQL、Python、统计学、业务理解 | 12-25K |
| 用户运营 | 数据分析、活动策划、用户分层 | 10-18K |
| 内容运营 | 文案能力、热点敏感、数据分析 | 8-15K |

### 发展路径

**产品经理：**
```
产品助理 → 产品经理 → 高级产品经理 → 产品总监 → 产品VP/首席产品官
              ↓
         专业路线：某领域专家（如AI产品专家）
```

**技术路线：**
```
初级工程师 → 中级工程师 → 高级工程师 → 技术专家/架构师
                                  ↓
                           技术管理路线：Team Leader → 技术总监 → CTO
```

**运营路线：**
```
运营专员 → 运营经理 → 运营总监 → COO
     ↓
专业路线：用户增长专家/内容专家/社群专家
```

### 入门学习路径

**产品经理：**
1. 理论：《人人都是产品经理》《启示录》《俞军产品方法论》
2. 工具：Axure/Figma原型、Xmind脑图、SQL基础
3. 实践：拆解产品、写PRD、做项目
4. 社区：人人都是产品经理、PMCAFF

**数据分析：**
1. 基础：Excel（透视表、函数）、统计学基础
2. SQL：多表查询、聚合函数、窗口函数
3. Python：Pandas、Matplotlib、Scikit-learn
4. 可视化：Tableau/Power BI
5. 业务：指标体系、分析方法论
6. 证书：CDA数据分析师

**前端开发：**
1. 基础：HTML/CSS/JavaScript
2. 框架：React或Vue
3. 工程化：Webpack、Git、npm
4. 进阶：TypeScript、Node.js基础
5. 实践：做项目、看源码、参与开源

## 金融行业

### 行业趋势
- **金融科技**：银行数字化转型，金融科技公司崛起
- **财富管理**：居民财富增长，投顾需求增加
- **绿色金融**：ESG投资成为热点
- **监管科技**：合规需求驱动技术应用

### 热门岗位

| 岗位 | 核心要求 | 应届薪资范围 |
|------|---------|-------------|
| 投行分析师 | 财务建模、PPT、Excel、高抗压 | 20-40K（奖金另计） |
| 行研分析师 | 财务分析、行业研究、报告撰写 | 15-30K |
| 风控专员 | 数据分析、风险评估、模型应用 | 12-25K |
| 量化分析师 | 编程、数学建模、金融知识 | 30-60K |
| 产品经理（金融科技） | 金融知识、产品设计、数据分析 | 15-30K |

### 发展路径

**投行：**
```
分析师(A) → 经理(A/VP) → 副总裁(VP) → 总监(D) → 执行董事(ED) → 董事总经理(MD)
```

**研究：**
```
研究助理 → 分析师 → 高级分析师 → 首席分析师 → 研究总监
```

### 入门学习路径

**投行/行研：**
1. 证书：CPA、CFA（优先级高）
2. 技能：财务分析、估值建模、Excel建模
3. 书籍：《投行笔记》《估值》《华尔街日报》
4. 实习：四大、券商、投行实习
