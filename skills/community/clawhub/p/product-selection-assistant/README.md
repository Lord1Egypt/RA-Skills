# 选品助手 (Product Selection Assistant)

AI驱动的电商选品决策工具。运营人员输入产品名称/描述，自动联网搜索市场数据，六维综合评分，给出明确的上架建议，生成交互式HTML可视化报告。

## 快速开始

```
选品分析：露营折叠椅
这个产品好不好卖？
帮我看看 XXX 值不值得上架
```

## 六维评分模型

| 维度 | 权重 | 说明 |
|------|------|------|
| 市场需求 | 25% | 搜索趋势、社媒热度、用户需求强度 |
| 竞争强度 | 20% | 竞品数量、品牌集中度（越低越好） |
| 利润空间 | 20% | 采购成本vs售价、预估毛利率 |
| 季节性风险 | 10% | 需求波动（越低越好） |
| 入场难度 | 10% | 认证、供应链、资金门槛（越低越好） |
| 增长趋势 | 15% | 品类增速、新兴机会 |

## 推荐阈值

- 🔥 **80+** 强烈推荐上架
- ✅ **65-79** 建议上架
- ⚠️ **50-64** 谨慎上架
- ❌ **<50** 不建议上架

## HTML报告功能

- 综合评分仪表盘 + 明确建议
- Chart.js 交互式雷达图
- 六维进度条评分卡
- 5张深度分析卡片（市场/竞争/利润/风险/机会）
- P0/P1/P2 分级行动清单
- 响应式设计，桌面+移动端
