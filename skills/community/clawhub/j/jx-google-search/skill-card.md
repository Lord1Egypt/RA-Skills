## Description: <br>
Search the web using SkillBoss API Hub for live information, documentation, or topic research when the built-in web_search tool is unavailable. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kirkraman](https://clawhub.ai/user/kirkraman) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agents use this skill to run web searches through SkillBoss API Hub and retrieve current search results for documentation lookup, research, and live information gathering. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Search terms are sent to an external SkillBoss/HeyBossAI API using a user-provided API key. <br>
Mitigation: Use a dedicated, revocable API key and avoid searching for secrets, private documents, customer data, or confidential business information. <br>
