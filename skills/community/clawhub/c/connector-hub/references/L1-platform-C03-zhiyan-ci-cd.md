# C03 - 智研构建部署

## 基本信息

| 字段 | 值 |
|------|-----|
| ID | C03 |
| 连接器名 | zhiyan-ci-cd |
| 显示名 | 智研构建部署 |
| 层级 | L1 平台级 |
| 可替代 | ❌ 不可替代 |
| 分类 | CI/CD |

## 能力说明

智研构建部署提供完整的 CI/CD 流水线：

| 能力 | 说明 |
|------|------|
| 代码构建 | 自动编译、打包、测试 |
| 制品管理 | 镜像仓库、包管理 |
| 自动部署 | 多环境部署（开发/测试/生产） |
| 流水线 | 可视化流水线编排 |
| 质量门禁 | 代码质量检查、安全扫描 |

## 为什么不可替代

**需要平台构建环境**：
- 构建需要平台的编译环境和依赖缓存
- 制品存储需要平台的镜像仓库
- 部署需要平台的目标环境（K8s/VM）
- 流水线需要平台的调度和日志系统

**Skill 无法提供**：
- ❌ 无法提供构建服务器
- ❌ 无法提供制品仓库
- ❌ 无法提供部署目标环境
- ❌ 无法提供流水线调度

## 使用场景

| 场景 | 说明 |
|------|------|
| 自动构建 | 代码提交自动触发构建 |
| 持续集成 | 自动测试、代码检查 |
| 持续部署 | 自动部署到各环境 |
| 制品管理 | Docker 镜像、NPM 包管理 |

## 配置方式

在 WorkBuddy 连接器管理页面启用 `zhiyan-ci-cd`，配置：
- API Token
- 项目 ID
- 流水线 ID

## 替代方案（非 Skill）

可使用其他 CI/CD 平台：
- Jenkins
- GitLab CI
- GitHub Actions
- CircleCI

但这些都需要平台部署和配置，不是 Skill 替代。
