# 安全生产监管法规依据

本文档汇总企业安全生产监管相关的法律法规和检查要点。

## 法律法规依据

### 国家法律
1. **《中华人民共和国安全生产法》**（2021年修订）
   - 生产经营单位的安全生产保障
   - 从业人员的安全生产权利义务
   - 安全生产的监督管理
   - 生产安全事故的应急救援与调查处理
   - 法律责任

2. **《中华人民共和国劳动法》**
   - 劳动安全卫生设施必须符合国家规定标准
   - 用人单位必须建立、健全劳动安全卫生制度

3. **《中华人民共和国职业病防治法》**
   - 用人单位应当为劳动者创造符合国家职业卫生标准的工作环境

### 行政法规
1. **《安全生产许可证条例》**
   - 矿山、建筑施工、危险化学品、烟花爆竹、民用爆破器材生产企业需要取得安全生产许可证

2. **《生产安全事故应急条例》**
   - 生产经营单位应当针对本单位可能发生的生产安全事故的特点和危害，进行风险辨识和评估，制定相应的生产安全事故应急救援预案

3. **《危险化学品安全管理条例》**
   - 危险化学品生产、储存、使用、经营和运输的安全管理

### 部门规章
1. **《企业安全生产费用提取和使用管理办法》**
   - 企业应当提取安全生产费用，专门用于完善和改进企业安全生产条件

2. **《生产安全事故应急预案管理办法》**
   - 生产经营单位应当编制应急预案，并定期组织演练

3. **《安全生产培训管理办法》**
   - 生产经营单位应当进行安全生产培训，确保从业人员具备必要的安全生产知识

## 安全生产检查要点

### 一、安全生产责任制
- [ ] 是否建立全员安全生产责任制
- [ ] 是否明确各岗位的责任人员、责任范围和考核标准
- [ ] 是否对安全生产责任制落实情况进行考核

**法规依据**：《安全生产法》第二十二条

### 二、安全管理机构与人员
- [ ] 是否设置安全生产管理机构或者配备专职安全生产管理人员
- [ ] 矿山、金属冶炼、建筑施工、道路运输单位和危险物品的生产、经营、储存单位，是否设置安全生产管理机构或者配备专职安全生产管理人员
- [ ] 安全生产管理人员是否具备与本单位所从事的生产经营活动相应的安全生产知识和管理能力

**法规依据**：《安全生产法》第二十四条

### 三、安全生产资金投入
- [ ] 是否保证本单位安全生产条件所必需的资金投入
- [ ] 是否提取和使用安全生产费用
- [ ] 是否依法参加工伤保险，为从业人员缴纳保险费

**法规依据**：《安全生产法》第二十三条、第四十八条

### 四、安全生产教育培训
- [ ] 是否对从业人员进行安全生产教育和培训
- [ ] 是否建立安全生产教育和培训档案
- [ ] 特种作业人员是否取得相应资格，方可上岗作业
- [ ] 是否对被派遣劳动者进行岗位安全操作规程和安全操作技能的教育和培训

**法规依据**：《安全生产法》第二十五条、第二十八条

### 五、安全设施与设备
- [ ] 建设项目安全设施是否与主体工程同时设计、同时施工、同时投入生产和使用（三同时）
- [ ] 安全设备的设计、制造、安装、使用、检测、维修、改造和报废是否符合国家标准或者行业标准
- [ ] 是否对安全设备进行经常性维护、保养，并定期检测
- [ ] 是否为从业人员提供符合国家标准或者行业标准的劳动防护用品

**法规依据**：《安全生产法》第二十八条、第三十三条、第三十四条、第四十二条

### 六、重大危险源管理
- [ ] 是否对重大危险源登记建档
- [ ] 是否进行定期检测、评估、监控
- [ ] 是否制定应急预案并告知从业人员和相关人员在紧急情况下应当采取的应急措施
- [ ] 是否将重大危险源及有关安全措施、应急措施报有关地方人民政府安全生产监督管理部门和有关部门备案

**法规依据**：《安全生产法》第三十七条

### 七、隐患排查治理
- [ ] 是否建立健全生产安全事故隐患排查治理制度
- [ ] 是否采取技术、管理措施，及时发现并消除事故隐患
- [ ] 是否如实记录事故隐患排查治理情况并向从业人员通报

**法规依据**：《安全生产法》第三十八条

### 八、应急预案与演练
- [ ] 是否制定本单位生产安全事故应急救援预案
- [ ] 是否定期组织演练
- [ ] 是否配备必要的应急救援器材、设备和物资
- [ ] 是否建立应急值班制度

**法规依据**：《安全生产法》第七十八条、第七十九条

### 九、危险作业管理
- [ ] 进行爆破、吊装、动火、临时用电以及国务院应急管理部门会同有关部门规定的其他危险作业，是否安排专门人员进行现场安全管理
- [ ] 是否严格执行操作规程

**法规依据**：《安全生产法》第四十三条

### 十、劳动防护用品
- [ ] 是否为从业人员提供符合国家标准或者行业标准的劳动防护用品
- [ ] 是否监督、教育从业人员按照使用规则佩戴、使用
- [ ] 是否教育从业人员正确佩戴和使用劳动防护用品

**法规依据**：《安全生产法》第四十二条、第四十四条

### 十一、事故报告与处理
- [ ] 是否建立生产安全事故报告制度
- [ ] 发生生产安全事故后，是否立即如实报告
- [ ] 是否配合事故调查处理

**法规依据**：《安全生产法》第八十条、第八十三条

### 十二、相关方安全管理
- [ ] 生产经营项目、场所发包或者出租给其他单位的，是否与承包单位、承租单位签订专门的安全生产管理协议
- [ ] 是否对承包单位、承租单位的安全生产工作统一协调、管理

**法规依据**：《安全生产法》第四十六条

## 常见隐患与整改建议

### 隐患1：安全生产责任制不健全
**表现**：
- 未建立全员安全生产责任制
- 责任制内容不具体，责任不明确
- 未对责任制落实情况进行考核

**整改建议**：
1. 建立全员安全生产责任制，明确各岗位的责任人员、责任范围和考核标准
2. 责任制内容要具体、可操作、可考核
3. 定期对责任制落实情况进行考核，奖惩分明

### 隐患2：安全管理人员配备不足
**表现**：
- 未设置安全生产管理机构或者配备专职安全生产管理人员
- 安全管理人员不具备相应能力和资质
- 安全管理人员配备数量不足

**整改建议**：
1. 按照规定设置安全生产管理机构或者配备专职安全生产管理人员
2. 安全管理人员应当具备与本单位所从事的生产经营活动相应的安全生产知识和管理能力
3. 定期检查安全管理人员配备情况，确保满足需要

### 隐患3：安全生产教育培训不到位
**表现**：
- 未对从业人员进行安全生产教育和培训
- 培训内容不全面，流于形式
- 未建立安全生产教育和培训档案
- 特种作业人员未取得相应资格上岗作业

**整改建议**：
1. 制定安全生产教育培训计划，定期开展培训
2. 培训内容要全面，包括安全生产法律法规、规章制度、操作规程、应急处置等
3. 建立安全生产教育和培训档案，如实记录培训时间、内容、参加人员以及考核结果
4. 特种作业人员必须经专门的安全作业培训，取得相应资格，方可上岗作业

### 隐患4：安全设备维护保养不及时
**表现**：
- 安全设备未按规定进行维护、保养、检测
- 维护、保养、检测记录不完整
- 安全设备带病运行

**整改建议**：
1. 建立安全设备管理制度，明确维护、保养、检测周期和责任人
2. 定期对安全设备进行维护、保养、检测，确保正常运转
3. 如实记录维护、保养、检测情况，并由有关人员签字
4. 发现安全设备存在隐患，立即停用，及时修复

### 隐患5：隐患排查治理不彻底
**表现**：
- 未建立生产安全事故隐患排查治理制度
- 隐患排查不全面、不深入
- 发现隐患未及时整改
- 未如实记录事故隐患排查治理情况

**整改建议**：
1. 建立生产安全事故隐患排查治理制度
2. 定期组织隐患排查，确保全面、深入
3. 发现隐患立即整改，不能立即整改的，应当制定整改计划，明确整改措施、责任人、完成时限
4. 如实记录事故隐患排查治理情况，并向从业人员通报

### 隐患6：应急预案不实用、演练不充分
**表现**：
- 应急预案内容不实用，可操作性差
- 未定期组织应急演练
- 应急演练流于形式，未达到预期效果
- 未配备必要的应急救援器材、设备和物资

**整改建议**：
1. 制定符合本单位实际的生产安全事故应急救援预案
2. 定期组织应急演练，确保从业人员具备必要的应急知识和应急能力
3. 应急演练要真实、实用，避免形式主义
4. 配备必要的应急救援器材、设备和物资，并定期维护、保养，确保完好可用

### 隐患7：劳动防护用品配备不足或使用不当
**表现**：
- 未为从业人员提供符合国家标准或者行业标准的劳动防护用品
- 提供的劳动防护用品质量不合格
- 未监督、教育从业人员按照使用规则佩戴、使用
- 从业人员未正确佩戴和使用劳动防护用品

**整改建议**：
1. 为从业人员提供符合国家标准或者行业标准的劳动防护用品
2. 确保劳动防护用品质量合格，功能完好
3. 监督、教育从业人员按照使用规则佩戴、使用
4. 教育从业人员正确佩戴和使用劳动防护用品，提高自我保护意识

## 监管报告撰写指南

### 报告结构
1. **企业基本信息**：企业名称、地址、法定代表人、联系方式、行业类型、规模等
2. **检查情况**：检查时间、检查人员、检查范围、检查方法等
3. **安全生产现状**：安全生产管理体系建设情况、安全设施设备运行情况、隐患排查治理情况等
4. **存在问题和风险点**：列出检查发现的问题和潜在风险，按严重程度排序
5. **整改建议**：针对每个问题提出具体、可操作的整改建议，明确整改时限和责任人
6. **监管意见**：对企业安全生产管理工作的总体评价，下一步监管重点等
7. **附件**：检查记录、照片、相关文件等

### 撰写要求
1. **客观真实**：如实记录检查情况，不夸大、不隐瞒
2. **条理清晰**：按照逻辑顺序组织内容，层次分明
3. **用语规范**：使用专业术语，避免口语化表达
4. **建议具体**：整改建议要具体、可操作、可检查
5. **及时提交**：检查结束后及时撰写报告，按时提交
