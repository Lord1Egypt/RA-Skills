# 市场调研方法论

> **用途**: strategy / brand / data Agent 共享的市场调研知识库
> **版本**: v1.0 | **更新**: 2026-06-06
> **适用**: U&A 研究 / 概念测试 / 包装测试 / 焦点小组 / 深访 / 量化调研

---

## 一、调研方法分类

### 1.1 按研究目的

| 类型 | 目的 | 方法 | 产出 |
|------|------|------|------|
| **探索性** | 发现问题/假设 | 深访 / 焦点小组 / 民族志 | 定性洞察 |
| **描述性** | 描述市场现状 | U&A / 满意度 / Brand Tracking | 量化数据 |
| **因果性** | 验证因果关系 | A/B 测试 / 概念测试 | 因果推断 |
| **预测性** | 预测未来表现 | conjoint / TURF / 销量预测 | 预测模型 |

### 1.2 按数据来源

| 类型 | 描述 | 成本 | 周期 | 适用 |
|------|------|:---:|:---:|------|
| **二手研究** | 公开报告 / 行业数据 | 低 | 1-3 天 | 初步判断 |
| **社交媒体聆听** | 爬虫 / API | 低 | 1-7 天 | 趋势识别 |
| **定性研究** | 深访 / 焦点小组 | 中 | 2-4 周 | 洞察挖掘 |
| **量化研究** | 问卷调查 | 中-高 | 3-8 周 | 统计推断 |
| **混合研究** | qual + quant | 高 | 8-16 周 | 决策支撑 |

## 二、U&A 研究(Usage & Attitude)

### 2.1 U&A 核心模块

```
1. 市场规模与渗透率
    ↓
2. 用户画像与分层
    ↓
3. 使用场景与频率
    ↓
4. 品牌认知与偏好
    ↓
5. 购买决策路径
    ↓
6. 满意度与痛点
    ↓
7. 未满足需求
```

### 2.2 问卷结构模板(30-40 题)

#### Section 1: 筛选(2-3 题)
- S1: 过去 3 个月是否购买过 [品类]? (Y/N, N 终止)
- S2: 年龄? (配额控制)

#### Section 2: 使用行为(5-7 题)
- Q1: 过去 1 个月使用频率? (每天/每周/每月)
- Q2: 典型使用场景? (多选)
- Q3: 单次使用量? (量级)
- Q4: 主要使用时段? (早晨/中午/晚上)
- Q5: 与什么搭配使用? (多选)

#### Section 3: 品牌认知(5-8 题)
- Q6: 提到 [品类], 你知道哪些品牌? (开放)
- Q7: 过去 3 个月购买过哪些品牌? (多选)
- Q8: 最常购买的品牌? (单选)
- Q9: 为什么选择 [最常购买品牌]? (开放 / 多选)
- Q10: 各品牌评分(10 分制, 5-6 个维度)

#### Section 4: 购买决策(4-6 题)
- Q11: 购买前会考虑哪些因素? (排序)
- Q12: 信息来源? (多选)
- Q13: 决策时长? (即时 / 1 天 / 1 周)
- Q14: 购买渠道? (多选)
- Q15: 客单价区间?

#### Section 5: 满意度与痛点(4-6 题)
- Q16: 整体满意度? (1-10 分)
- Q17: NPS(0-10 分)
- Q18: 最不满意的地方? (开放 / 多选)
- Q19: 期望的改进? (开放)

#### Section 6: 未满足需求(3-5 题)
- Q20: 理想中的 [品类] 是什么样的? (开放)
- Q21: 是否有未被满足的需求? (Y/N)
- Q22: 如果有, 是什么? (开放)

#### Section 7: 背景信息(3-4 题)
- D1: 性别 / 年龄 / 城市 / 收入
- D2: 家庭结构
- D3: 职业

### 2.3 样本量计算

| 研究类型 | 推荐样本 | 置信水平 | 抽样误差 |
|---------|:------:|:------:|:------:|
| **快速洞察** | 200-300 | 95% | ±5-7% |
| **标准 U&A** | 500-1000 | 95% | ±3-5% |
| **高质量 U&A** | 1500-3000 | 95% | ±2-3% |
| **细分市场研究** | 3000+ | 95% | ±1-2% |

## 三、定性研究方法

### 3.1 深度访谈(In-depth Interview)

#### 适用场景
- 用户决策路径还原
- 敏感话题(医美 / 金融 / 健康)
- B2B 受访者(高管难聚齐)

#### 访谈大纲结构(60-90 分钟)

```
1. 开场与暖场(5min)
   ↓
2. 背景了解(10min)
   - 个人/家庭/工作
   - 品类使用历史
   ↓
3. 核心议题深挖(40-60min)
   - 决策过程: "请详细描述上次购买的过程"
   - 情感连接: "用 [品类] 时你的感受?"
   - 痛点挖掘: "最让你困扰的一次经历?"
   - 投射技术: "如果 [品牌] 是一个人, ta 是什么样的?"
   ↓
4. 未来展望(10min)
   - 理想产品想象
   - 未满足需求
   ↓
5. 结束与感谢(5min)
```

#### 访谈技巧

| 技巧 | 描述 | 示例 |
|------|------|------|
| **5 Why** | 连续追问 5 次 | "为什么?" → "因为..." → "为什么..." |
| **沉默** | 留白让受访者补充 | (停顿 5 秒) |
| **具体化** | 引用具体事件 | "能举个例子吗?" |
| **对比** | 通过对比揭示偏好 | "A 和 B 有什么不同?" |
| **投射** | 用第三人称 / 隐喻 | "你的朋友会怎么选?" |

### 3.2 焦点小组(Focus Group)

#### 适用场景
- 概念测试 / 包装测试 / 广告测试
- 探索群体共识与分歧
- 快速收集多元观点

#### 配置

| 要素 | 建议 | 注意 |
|------|------|------|
| 人数 | 6-8 人 | <5 难讨论 / >10 难控场 |
| 时长 | 90-120 分钟 | 不超过 2 小时 |
| 主持 | 资深主持人 | 中立不引导 |
| 场次 | 同一人群 2-3 组 | 避免单组偏差 |
| 观察 | 单向玻璃 / 视频 | 客户可观摩 |

#### 小组大纲(120 分钟示例)

```
1. 暖场(15min)
   - 自我介绍 + 破冰游戏
   
2. 主题讨论(30min)
   - 品类使用经验
   - 现有品牌评价
   
3. 概念/产品测试(45min)
   - 概念 A 展示 + 讨论
   - 概念 B 展示 + 讨论
   - A vs B 选择 + 理由
   
4. 共创(20min)
   - 理想产品想象
   - 改进建议
   
5. 结束(10min)
   - 总结 + 感谢
```

### 3.3 民族志(Ethnography)

**目的**: 在真实场景中观察用户行为(而非"说"的行为)。

| 形式 | 时长 | 成本 | 适用 |
|------|:---:|:---:|------|
| **陪同购物** | 2-4 小时 | 中 | 购买决策 |
| **入户访谈** | 2-3 小时 | 中-高 | 使用场景 |
| **日记研究** | 1-4 周 | 中 | 长期行为 |
| **影子观察** | 1-7 天 | 高 | B2B 决策 |

## 四、概念测试(Concept Testing)

### 4.1 概念测试流程

```
概念卡片撰写 → 定性预测试 → 量化测试 → 数据分析 → 决策
```

### 4.2 概念卡片模板

```markdown
## [产品/概念名]

[一句话描述]

### 核心利益
- 利益 1: ___
- 利益 2: ___
- 利益 3: ___

### RTB(Reason to Believe)
- 支撑 1: ___
- 支撑 2: ___

### 价格区间: ___
```

### 4.3 量化测试指标(5 维度)

| 维度 | 问题 | 评分方式 | 通过阈值 |
|------|------|---------|:------:|
| **理解度** | 这个概念描述清楚吗? | 1-5 分 | ≥4 |
| **可信度** | 你相信这些承诺吗? | 1-5 分 | ≥3.5 |
| **购买意愿** | 你会购买吗? | 1-5 分 | ≥3.5 |
| **独特性** | 与现有产品有何不同? | 1-5 分 | ≥3.5 |
| **价格接受度** | 这个价格合理吗? | 1-5 分 | ≥3.5 |

**综合得分 = 5 维度加权平均(等权重)**

### 4.4 TURF 分析(Reach & Frequency)

**目的**: 在多个产品变体中, 找到覆盖最多用户的组合。

```
示例: 10 种口味, 选 5 种最大化覆盖
Reach = 至少喜欢其中一种的用户比例
Frequency = 平均喜欢数量
```

## 五、包装测试(Package Testing)

### 5.1 测试维度

| 维度 | 测试方法 | 关键指标 |
|------|---------|---------|
| **可见性** | 货架模拟 / 眼动追踪 | 注视率 / 第一眼 |
| **辨识度** | 5 秒印象 | 品牌识别准确率 |
| **吸引力** | 偏好测试 | 选择率 |
| **信息传达** | 信息回忆 | 关键信息回忆率 |
| **货架冲击力** | 模拟货架 | 脱颖而出率 |

### 5.2 货架测试设计

```
A 方案: 现有包装
B 方案: 新包装设计
C 方案: 竞品包装(对照)

模拟货架: 3 层 × 5 列, 混合摆放
眼动追踪: 30 秒自由观察
回忆测试: 看到哪些品牌?
偏好测试: A vs B 选哪个?
```

## 八、调研项目管理

### 8.1 项目时间线(标准 U&A)

```
Week 1: 需求确认 + 问卷设计
Week 2: 问卷测试 + 修订
Week 3-4: 数据收集
Week 5: 数据分析
Week 6: 报告撰写
```

### 8.2 质量控制要点

| 环节 | 质量点 | 控制方法 |
|------|--------|---------|
| **抽样** | 样本代表性 | 配额 + 加权 |
| **问卷** | 题目有效 | 预测试 30 份 |
| **数据收集** | 作答质量 | 陷阱题 + 时长筛选 |
| **数据清洗** | 异常值 | 标准 3σ / IQR |
| **分析** | 显著性 | p<0.05 / CI |

### 8.3 调研报告质量标准

| 章节 | 必备要素 |
|------|---------|
| 执行摘要 | 1 页, 3-5 个关键发现 |
| 方法论 | 样本量 / 抽样 / 时间 / 方法 |
| 详细发现 | 数据表 + 图表 + 解读 |
| 洞察与建议 | 业务含义 + 行动建议 |
| 附录 | 完整问卷 / 详细数据 |

## 十、AI 时代的调研演进

### 10.1 AI 增强调研

| 传统方法 | AI 增强 | 优势 |
|---------|---------|------|
| 问卷调研 | AI 自适应问卷 | 根据回答动态调整 |
| 焦点小组 | AI 主持人辅助 | 实时分析 + 追问建议 |
| 社交聆听 | NLP 情感分析 | 大规模 + 实时 |
| 报告撰写 | AI 自动洞察 | 速度 + 规模 |

### 10.2 AI 调研伦理

- ✅ AI 辅助分析: 人工审核结论
- ✅ AI 主持人: 标注 AI 参与
- ❌ AI 替代真人受访者: 数据不可用
- ❌ AI 生成样本: 违反代表性原则
