## Known Risks and Mitigations: <br>
Risk: Suggested npm installs, tsconfig/package.json changes, or bulk file renames may alter an important project. <br>
Mitigation: Review proposed commands and file changes before applying them, then run the project's type check and test suite. <br>
Risk: TypeScript migration guidance can introduce build or runtime regressions if applied too broadly. <br>
Mitigation: Apply migrations incrementally, inspect generated type changes, and keep rollback points for converted files. <br>
