# 网织红细胞计数用于骨髓造血功能评估

## 适用场景
- 评估贫血患者的骨髓红系造血活性
- 监测缺铁性贫血、巨幼细胞性贫血等治疗后的疗效
- 鉴别再生障碍性贫血与溶血性贫血
- 排除近期输血对骨髓功能判断的干扰

## 执行步骤
1. **确认样本与前提条件**：
   - 使用未输血的外周血样本
   - 排除严重溶血急性期（可能导致假性升高）
   - 确保已采用煌焦油蓝染色并完成显微镜观察

2. **获取网织红细胞百分比**：
   - 在油镜下计数1000个红细胞中的网织红细胞数量
   - 计算百分比：(网织红细胞数 / 1000) × 100%

3. **结合患者临床状态解读结果**：
   - **正常范围**：0.5%～1.5%（成人）；新生儿可达2%～6%
   - **低于0.5%**：提示骨髓红系造血功能低下（如再生障碍性贫血）
   - **高于2.5%**：提示骨髓活跃造血，见于溶血性贫血、急性失血或贫血治疗有效

4. **输出判断结论**：
   - 明确指出骨髓功能状态（抑制/正常/活跃）
   - 关联可能的临床诊断或治疗反应

> 注意：若患者近期接受过输血，需谨慎解读结果，因输入的成熟红细胞会稀释网织红细胞比例。
