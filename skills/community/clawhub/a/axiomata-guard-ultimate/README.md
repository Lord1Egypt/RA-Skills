# Axioma Guard Ultimate

The ultimate ClawHub skill security checker + quality enhancer.

## Scripts

- `check.py` - Complete security + quality check
- `improve.py` - Auto-improve skill quality
- `destroy.py` - Reject and destroy dangerous skills

## Usage

```bash
# Full check
python3 scripts/check.py /path/to/skill --verbose

# Quick check
python3 scripts/check.py /path/to/skill --quick

# Auto-improve
python3 scripts/improve.py /path/to/skill --verbose

# Destroy dangerous skill
python3 scripts/destroy.py /path/to/skill --confirm
```
