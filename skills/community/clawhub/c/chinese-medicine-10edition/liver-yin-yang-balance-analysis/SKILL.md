# Liver Yin-Yang Balance Analysis

## When to Use
- 用户提及肝的生理功能异常（如情绪抑郁、胁痛、月经不调、眩晕、出血等）
- 辨证涉及肝气、肝血、肝阳、肝风等术语
- 治疗方案需兼顾养血柔肝与疏肝理气
- 需解释“肝为刚脏”或“体阴用阳”的理论内涵

## Execution Steps

1. **确认肝体阴与用阳的基本属性**
   - 体阴：肝以藏血为本，属阴，主静、主藏，需血濡养以维持柔润
   - 用阳：肝主疏泄，属阳，主动、主升发，表现为刚强特性

2. **评估二者协调关系**
   - 若肝血充足 → 肝体得养 → 肝阳不亢 → 疏泄正常 → 气机调畅 → 血行有序 → 藏血功能维持
   - 若疏泄正常 → 气机畅达 → 血行有度 → 有助于血液贮藏与调节

3. **识别失衡模式及连锁反应**
   - **疏泄不及**：肝气郁滞 → 血行不畅 → 血瘀证
   - **疏泄太过**：肝气上逆 → 血随气逆 → 出血证
   - **藏血不足**：肝血亏虚 → 阴不制阳 → 肝阳上亢 → 肝火/肝风内动 → 进一步扰乱疏泄

4. **给出临床指导建议**
   - 治疗必须体用兼顾：养血以柔肝（治体阴），疏肝以理气（治用阳）
   - 避免单向干预（如仅疏肝不养血，或仅补血不理气）

> ⚠️ **注意**：不可孤立看待肝的某一功能，必须基于阴阳互根的整体观进行分析。
