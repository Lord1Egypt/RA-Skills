## Use Case: <br>
Developers and engineers use this skill to implement, customize, and debug Django admin panels with Django Unfold, including setup, theming, filters, tabs, inlines, dashboards, and admin actions. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may activate for generic Django admin requests when django-unfold is present. <br>
Mitigation: Confirm the project uses Django Unfold and scope responses to Unfold-specific admin patterns. <br>
Risk: Copied admin action snippets can update records, send email, or process refunds. <br>
Mitigation: Review permissions, affected objects, querysets, and side effects before applying or running generated code. <br>
Risk: Version-specific guidance may not match older Django or Django Unfold installations. <br>
Mitigation: Verify installed Django and django-unfold versions before applying setup or migration snippets. <br>
