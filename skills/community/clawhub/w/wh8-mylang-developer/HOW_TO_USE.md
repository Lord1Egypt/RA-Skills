# WT8麦语言策略编写通用框架

> 💡 **核心发现**：双均线策略揭示了一个**万能策略编写公式**，适用于所有策略！

---

## 万能公式：6步编写任何策略

```
步骤1：定义参数
   ↓
步骤2：计算指标
   ↓
步骤3：构建条件
   ↓
步骤4：开仓信号
   ↓
步骤5：平仓/止损信号
   ↓
步骤6：AUTOFILTER
```

---

## 步骤详解

### 步骤1：定义参数（赋值用 :=）
```mql
// 所有策略都需要参数
止损 := 15;      // 止损跳数
N1 := 10;        // 短期周期
N2 := 20;        // 长期周期
```

### 步骤2：计算指标（赋值用 :=）
```mql
// 均线、MACD、布林带...
MA_S := MA(C, N1);
MA_L := MA(C, N2);
DIF := EMA(C,12) - EMA(C,26);
```

### 步骤3：构建条件（赋值用 :=）
```mql
// 交叉、背离、突破...
金叉 := CROSSUP(MA_S, MA_L);
多头 := DIF > DEA;
突破 := CLOSE > HHV(HIGH,20);
```

### 步骤4：开仓信号（条件,信号;）
```mql
// BK=买开，SK=卖开
金叉, BK;
突破, SK;
```

### 步骤5：平仓/止损信号（条件,信号;）
```mql
// SP=卖平，BP=买平
死叉, SP;
止损条件, SP;

// 止损写法
CLOSE <= BKPRICE - 止损*MINPRICE, SP;
CLOSE >= SKPRICE + 止损*MINPRICE, BP;

// 权益止损
MONEYTOT <= INITMONEY*0.9, SP;
```

### 步骤6：信号过滤（必须写！）
```mql
AUTOFILTER;
```

## 通用模板

```mql
// ============================================
// [策略名称]
// ============================================

// 步骤1：参数
参数1 := 值1;
参数2 := 值2;

// 步骤2：指标
指标A := 函数(参数);
指标B := 函数(参数);

// 步骤3：条件
条件A := 指标A > 指标B;           // 比较
条件B := CROSSUP(指标A, 指标B);  // 交叉
条件C := CLOSE > HHV(HIGH,20);   // 突破

// 步骤4：开仓
条件A, BK;   // 买开
条件B, SK;   // 卖开

// 步骤5：平仓/止损
条件B, SP;   // 平空
条件A, BP;   // 平多
止损条件, SP;
止损条件, BP;

// 步骤6：过滤
AUTOFILTER;
```

## 信号类型速查表

### 开仓信号
| 信号 | 含义 | 何时用 |
|------|------|--------|
| BK | 买开 | 做多开仓 |
| SK | 卖开 | 做空开仓 |

### 平仓信号
| 信号 | 含义 | 何时用 |
|------|------|--------|
| SP | 卖平 | 平多仓/止损 |
| BP | 买平 | 平空仓/止损 |

### 反手信号
| 信号 | 含义 | 等价于 |
|------|------|--------|
| BPK | 反手买 | BP + BK |
| SPK | 反手卖 | SP + SK |

### 加减仓指令
| 指令 | 含义 | 注意 |
|------|------|------|
| ADD_LONG(N) | 多头加仓 | 必须带手数 |
| LOWER_LONG(N) | 多头减仓 | 必须带手数 |

## 常见策略类型

### 1. 趋势策略（均线、MACD）
```mql
// 突破类
CLOSE > MA(C,20), BK;
CLOSE < MA(C,20), SK;

// 交叉类
CROSSUP(DIF,DEA), BK;
CROSSDOWN(DIF,DEA), SK;
```

### 2. 突破策略（高低点）
```mql
// 布林带
CLOSE > TOP, BK;
CLOSE < BOTTOM, SK;

// 区间突破
CLOSE > HHV(HIGH,20), BK;
CLOSE < LLV(LOW,20), SK;
```

### 3. 震荡策略（RSI、布林中轨）
```mql
// 超买超卖
RSI < 30, BK;
RSI > 70, SK;

// 布林中轨
CLOSE > MID, BK;
CLOSE < MID, SK;
```

### 4. 止损类型
```mql
// 固定止损
CLOSE <= BKPRICE-15*MINPRICE, SP;

// 百分比止损
CLOSE <= BKPRICE*0.95, SP;

// 权益止损
MONEYTOT <= INITMONEY*0.9, SP;
```

## 语法检查清单

写完代码后对照检查：

- [ ] 参数定义用 `:=`？ (`N := 10;` 不是 `N = 10;`)
- [ ] 指标计算用 `:=`？ (`MA_S := MA(C,10);`)
- [ ] 条件构建用 `:=`？ (`金叉 := CROSSUP(A,B);`)
- [ ] 信号格式 `条件, 信号;`？ (`金叉, BK;`)
- [ ] 加减仓带手数？ (`ADD_LONG(2);`)
- [ ] 写了 `AUTOFILTER;`？
- [ ] 没有 `{}` 大括号？
- [ ] 没有 `IF ()` 语句？

## 关键原则

### 1. 赋值用 := ，比较用 =
```mql
// ✅ 赋值
N := 10;
MA10 := MA(C,N);

// ✅ 比较
CLOSE > MA10
```

### 2. 条件在前，信号在后
```mql
// ✅ 正确
CLOSE > MA10, BK;

// ❌ 错误
BK, CLOSE > MA10;
```

### 3. 信号必须有条件
```mql
// ✅ 正确
金叉, BK;

// ❌ 错误
BK;  // 不能单独用！
```

### 4. 一开必须一平
```mql
// ✅ 完整
金叉, BK;   // 开多
死叉, SP;   // 平多

// ❌ 不完整
金叉, BK;   // 只有开仓
```
