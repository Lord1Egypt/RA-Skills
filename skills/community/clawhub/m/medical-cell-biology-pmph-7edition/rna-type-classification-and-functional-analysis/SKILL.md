# RNA类型分类与功能分析

## 适用对象
- 真核细胞（包括动物细胞）
- 基因表达调控系统

## 使用前提
- 细胞具备细胞核与细胞质分隔
- 存在完整的转录与翻译机制

## 执行流程

1. **初步分类**：将待分析RNA划分为两大类：
   - **编码RNA**：仅包括mRNA
   - **非编码RNA (ncRNA)**：其余所有RNA类型

2. **若为mRNA**：
   - 确认其作为蛋白质合成模板的功能
   - 检查是否具有真核特征：5'端m⁷G帽子、3'端poly(A)尾、单顺反子结构、含5'和3'非翻译区（UTR）

3. **若为非编码RNA**，进一步判断其属于持家性还是调节性ncRNA：
   - **持家性ncRNA**（维持基础细胞功能）：
     - rRNA：构成核糖体，占RNA总量80%～90%
     - tRNA：携带氨基酸，具三叶草结构，3'端为CCA
     - snRNA：参与剪接体组装，富含尿嘧啶（U）
     - snoRNA：指导rRNA修饰与加工
   - **调节性ncRNA**（调控基因表达）：
     - miRNA：通过RISC复合物靶向mRNA 3'UTR，抑制翻译或促降解
     - siRNA：介导外源dsRNA引发的RNA干扰（RNAi）
     - piRNA：主要在雄性生殖细胞中调控转座子与精子发育
     - lncRNA：>200 nt，可调控染色质、转录或蛋白活性
     - circRNA：环状结构，可作为miRNA“海绵”或编码短肽
     - 核酶（ribozyme）：具有催化活性，可自我剪接

4. **特殊注意事项**：
   - 原核细胞缺乏snRNA、lncRNA等部分ncRNA类型
   - 线粒体RNA功能特殊，需单独评估
   - 部分tRNA可作为逆转录引物；某些ncRNA具有潜在编码能力

## 输出目标
明确RNA类型及其在基因表达通路中的功能角色，为实验设计（如敲低、过表达、定位）或机制解析提供依据。
