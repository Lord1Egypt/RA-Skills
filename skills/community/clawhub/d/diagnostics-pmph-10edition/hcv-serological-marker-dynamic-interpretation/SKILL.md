# HCV血清学标志物动态解读

## 适用对象
- 疑似或确诊丙型肝炎患者
- 正在接受干扰素等抗HCV治疗的患者

## 前提条件
- 已完成HCV-RNA（RT-PCR）和抗-HCV（IgM/IgG）检测
- 拥有至少两个时间点的连续检测数据

## 解读步骤

### 1. 分析HCV-RNA结果
- 若HCV-RNA ≥ 50 IU/ml：提示病毒复制活跃，具有传染性
- 若HCV-RNA由阳性转为阴性（<50 IU/ml）：提示病毒复制被抑制，治疗有效
- 连续监测HCV-RNA变化用于评估抗病毒治疗疗效

### 2. 分析抗-HCV IgM结果
- 急性期通常在发病第1天可检出，2～4天常见，7～15天达高峰
- 持续阳性超过3个月提示可能转为慢性肝炎或病毒持续复制

### 3. 分析抗-HCV IgG结果
- 阳性仅表明曾感染HCV，不能区分现症与既往感染
- 在输血后肝炎中80%～90%呈阳性
- 长期阳性需警惕慢性化、肝硬化或肝癌风险

### 4. 联合动态判断规则
- **急性HCV感染**：HCV-RNA阳性 + 抗-HCV IgM阳性
- **慢性进展**：HCV-RNA持续阳性 + 抗-HCV IgM持续阳性
- **治疗有效**：HCV-RNA由阳转阴 + 抗-HCV IgM逐渐消失
- **既往感染已清除**：HCV-RNA阴性 + 抗-HCV IgG阳性

## 注意事项
- 单次HCV-RNA阴性不能排除感染，需结合窗口期和临床背景
- 抗-HCV IgG阳性不能作为现症感染依据
- 所有判断应基于连续时间点数据，避免单次快照误判
