# FactorHub A股量化数据

提供 5500+ 只 A 股的量化因子数据和财经指标，通过 MCP 协议接入 AI Agent。

## 能力

- 因子数据：查询因子列表、因子评分指标、因子净值曲线
- 市场行情：个股日线、指数日线、估值指标（PE/PB/PS/股息率）
- 股票信息：个股基本面、股票筛选
- 交易日历：查询交易日
- 策略回测：涨停首板、低估值、动量、均值回归等内置策略

## 安装

```bash
clawhub install factorhub
```
