# Step 7: Post-Rehearsal Revision

## Goal

根据模拟答辩中暴露的问题，更新问答库和 PPT 备份页。

## Actions

1. Record questions the user could not answer.
2. Classify failure type:
   - forgot paper evidence
   - unclear concept
   - missing code evidence
   - missing training evidence
   - overclaimed
   - answer too long
   - no backup slide
3. Update:
   - `defense_qa_bank_cn.md`
   - `defense_qa_bank_cn.json`
   - `backup_slide_plan_cn.md`
   - `evidence_gap_triage_cn.md`
4. Produce final 10-minute memo.

## Final memo format

```markdown
## 最后 10 分钟速记卡

### 必背 5 句话
...

### 最危险 5 个问题
...

### 不能说的 5 句话
...

### 缺证据时的 5 个恢复句
...
```
