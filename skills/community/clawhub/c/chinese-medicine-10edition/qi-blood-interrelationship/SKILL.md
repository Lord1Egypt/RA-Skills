# Qi Blood Interrelationship

## When to Use
- 血虚患者伴乏力、气短
- 大出血后出现冷汗、脉微欲绝
- 血瘀证需配合理气药
- 出血证需健脾补气摄血

## Execution Steps
1. **应用“气为血之帅”**：
   - **气能生血**：血虚者配补气药（如黄芪+当归）；
   - **气能行血**：血瘀者调理气机（气滞则理气，气虚则补气）；
   - **气能摄血**：出血者健脾益气（防“气随血脱”）。
2. **应用“血为气之母”**：
   - **血能载气**：大出血时急补气固脱；
   - **血能养气**：血虚常兼气虚，需气血双补。
3. **指导用药**：
   - 血虚 → 补气生血；
   - 出血 → 补气摄血；
   - 血瘀 → 调气行血。

> **注意**：必须结合脏腑功能（心、脾、肝、肺）整体分析，不可孤立看待气血。
