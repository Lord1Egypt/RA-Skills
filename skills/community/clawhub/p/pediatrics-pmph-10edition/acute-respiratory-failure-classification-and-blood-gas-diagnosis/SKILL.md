# 急性呼吸衰竭分型与血气诊断标准

## 适用对象
- 疑似急性呼吸衰竭的儿童患者

## 使用前提
- 动脉血气分析必须在以下条件下进行：海平面、吸入室内空气（FiO₂=21%）、静息状态、已排除发绀性先天性心脏病。

## 执行步骤
1. **获取标准条件下的动脉血气结果**（PaO₂、PaCO₂）。
2. **根据阈值进行分型**：
   - **Ⅰ型呼吸衰竭（低氧性）**：PaO₂ < 60 mmHg，且 PaCO₂ 正常（35–45 mmHg）或降低（<35 mmHg）
   - **Ⅱ型呼吸衰竭（高碳酸性）**：PaO₂ < 60 mmHg，且 PaCO₂ > 50 mmHg
3. **若患儿已吸氧**，不能直接使用PaO₂判断低氧程度，应采用以下校正指标之一评估真实氧合障碍：
   - 计算 **PaO₂/FiO₂ 比值**：≤300 提示存在呼吸功能障碍
   - 使用 **SpO₂/FiO₂ 比值**（无创替代）：<315 提示氧合障碍
   - 计算 **A-aDO₂（肺泡-动脉氧分压差）**：正常<10 mmHg，差值越大提示肺病变越重
4. **临床优先原则**：即使未获得血气结果，若患儿有明显呼吸困难并影响重要脏器功能，应视为严重呼吸衰竭并立即干预。
5. **注意排除干扰因素**：
   - 存在右向左分流时，PaO₂降低可能非肺源性
   - 慢性碱中毒时PaCO₂代偿性升高不等于呼吸衰竭
