# Competitive Intelligence Agent 🕵️📊

**Know what your competitors are doing — before it matters.**

## What It Does

Stop falling behind on competitor moves. The Competitive Intelligence Agent monitors your competitors 24/7 — scraping pricing pages, tracking social media, aggregating reviews, and watching for product launches. Every Monday morning (or your preferred time), you get a concise intelligence briefing that tells you exactly what changed and what to do about it.

## Why Buy This

- **No more manual monitoring** — stop tab-hopping through 20 competitor pages every day
- **Early warning system** — get alerted the moment a competitor drops prices, launches a feature, or runs a campaign
- **Pricing intel** — always know when it's time to adjust your pricing
- **Review insights** — see what customers complain about at competitors (their weakness is your opportunity)
- **Weekly briefing** — read a 2-minute report instead of hours of research
- **SA focus** — monitors South African review platforms like Hellopeter and CaCx

## Key Features

| Feature | Details |
|---|---|
| 💰 **Pricing Monitoring** | Daily price checks across all competitors. Alerts on changes > 10% |
| 📱 **Social Tracking** | Follows Twitter, LinkedIn, Facebook for posts, sentiment, and follower shifts |
| ⭐ **Review Aggregation** | Pulls ratings from Hellopeter, G2, Trustpilot, Google Reviews, CaCx |
| 🚀 **Product Detection** | Monitors website and product changes — new features, pages, or positioning |
| 📰 **News Monitoring** | Tracks hiring, funding rounds, leadership changes |
| 📊 **Weekly Briefing** | Concise, actionable report delivered to Slack, email, or Telegram |
| ⚡ **Real-time Alerts** | Pings you immediately on price drops, new launches, or reputation changes |
| 🏢 **Unlimited Competitors** | Monitor as many as you need — no per-competitor limits |

## Quick Start

```bash
# Install
openclaw skill install competitive-intelligence

# Add your competitors
nano config/targets.yaml

# Test collectors
scripts/test-collectors.sh

# Run your first full sweep
openclaw skill run competitive-intelligence --full-sweep

# Get your first weekly report
openclaw skill run competitive-intelligence --report weekly
```
