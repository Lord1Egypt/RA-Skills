# PIPELINE.md — Méthode Hermine

## Vue d'ensemble

```
FAIT DIVERS / HISTOIRE VRAIE (l'auteur)
 ↓
[H0] RÉCEPTION — lecture des faits nus
 ↓
[H1] ANALYSE SUBSTRAT — tension centrale, personnages, silences
 ↓
[H2] GÉNÉRATION — 3 axes de fictionnalisation  [modèle : Sonnet]
 ↓
[H3] JURY D'IDÉATION — scoring 3 critiques × 3 axes  [modèle : Sonnet]
 ↓
[H4] CONTRE-AXES — 2 axes radicaux, pensée contre soi  [modèle : Opus]
 ↓
PRÉSENTATION 5 AXES À PHILIPPE → CHOIX FINAL
 ↓
INPUT HERBERT (Étape 1 — Thème + Pitch + Dossier du réel)
```

---

## Étape H0 — Réception du fait brut

**Input acceptés :**
- Article de presse (texte collé ou URL)
- Résumé libre de l'auteur (quelques lignes suffisent)
- Titre seul + contexte minimal
- Fait historique, affaire judiciaire, biographie fragmentaire

**Action Hermine :**
Extraire et reformuler en format canonique :
```
FAIT BRUT
- Qui : [protagonistes réels, anonymisés si nécessaire]
- Quoi : [événement central en une phrase]
- Où / Quand : [contexte géographique et temporel]
- Nœud : [le moment de bascule]
- Ce qui n'a pas été dit : [silences, zones d'ombre, questions sans réponse]
```

Ne pas encore interpréter. Ne pas encore fictionnaliser.

---

## Étape H1 — Analyse du substrat réel

**Action Hermine :**
Disséquer le fait pour révéler ses structures dramatiques latentes.

```
SUBSTRAT NARRATIF
- Tension centrale : [conflit irréductible en une ligne]
- Fonction des acteurs : [rôle dramatique, pas biographique]
- Ce que le contexte rend possible/impossible : [époque, lieu, classe]
- Universalité : [ce que n'importe quel lecteur peut reconnaître ici]
- Silences narratifs : [ce qui n'a pas été dit et qui est souvent le cœur]
```

---

## Étape H2 — Génération des 3 axes

**Action Hermine :**
Produire 3 axes narratifs distincts. Format obligatoire pour chaque axe :

```
AXE [N] — [TITRE PROVISOIRE]

Angle : [point de vue dominant]
Pitch : [5 lignes max]
Tonalité : [thriller / drame intime / récit choral / huis clos / fresque...]
Arc émotionnel : [émotion dominante du lecteur à la fin]
Seeds estimées : [30 / 40 / 50]

Personnages centraux :
- [Nom fictif] — [fonction dramatique] — [contradiction interne]
- [Nom fictif] — [fonction dramatique] — [contradiction interne]
- [Nom fictif] — [fonction dramatique] — [contradiction interne]

Ce que le réel apporte que la fiction pure ne pourrait pas inventer :
[1-2 lignes]
```

---

## Étape H3 — Jury d'idéation (Axes 1-2-3)

**Modèle :** Sonnet (rapide, suffisant pour les 3 premiers axes)

**3 critiques :**
- `jury-originalite` — L'angle est-il distinct de ce qui existe ? Qu'est-ce qui le singularise ?
- `jury-emotion` — Le lecteur va-t-il ressentir quelque chose d'universel ?
- `jury-faisabilite` — Herbert peut-il construire 40-50 seeds cohérentes dessus ?

**Scoring :** f(x) = (x/100)^1.5 × 100
**Score composite :** moyenne des 3 critiques transformés, ramené /10

**Format rapport Jury d'idéation :**
```
           | Originalité | Émotion | Faisabilité | Score /10
Axe 1      |             |         |             |
Axe 2      |             |         |             |
Axe 3      |             |         |             |

→ Recommandation : Axe [N]
→ Motif : [2-3 lignes]
→ Risque principal : [1 ligne]
```

---

## Étape H4 — Contre-axes (nouveaux axes radicaux)

**Déclenchement :** Systématique, immédiatement après H3. Toujours produit, sans attendre la validation de l'auteur sur les axes 1-2-3.

**Modèle :** Opus (recherche approfondie, pensée contre soi)

**Règle absolue :** Les axes 4 et 5 ne réutilisent **aucune** des idées principales des axes 1, 2 et 3 :
- Pas la même époque
- Pas le même type de protagoniste
- Pas le même angle thématique
- Pas la même tonalité
- Pas le même ancrage géographique ou culturel

**Méthode :**
1. Lister explicitement ce qu'on a utilisé dans les axes 1-2-3 (époques, types de personnages, angles, tonalités)
2. Construire les axes 4-5 en partant de l'opposé sur chacune de ces dimensions
3. Documenter le raisonnement de rupture avant de produire les axes

**Format obligatoire :**
```
RUPTURE VOLONTAIRE
Ce que les axes 1-2-3 utilisent :
- [dimension 1] : [ce qui a été utilisé]
- [dimension 2] : [ce qui a été utilisé]
[...]

Axes 4-5 construits contre :
- [dimension 1] : [choix opposé]
- [dimension 2] : [choix opposé]
[...]
```

Puis le même format de présentation qu'H2 + jury H3 pour chacun des axes 4 et 5.

**Scoring :** Mêmes critères que H3, avec un critère supplémentaire :
- `jury-rupture` — L'axe est-il vraiment radicalement distinct des 3 premiers ? (pondération ×1.5)

**Output :**
Les 5 axes (1-2-3 de Sonnet + 4-5 d'Opus) sont présentés ensemble à l'auteur pour choix final.

---

## Output final Hermine → Herbert

Hermine livre à l'auteur et à Herbert :

**1. L'axe recommandé** (ou l'axe choisi par l'auteur) sous forme de Thème + Pitch structuré — format directement compatible avec l'Étape 1 d'Herbert.

**2. Le dossier du réel** — transmis à Herbert pour nourrir ses bibles :
```
DOSSIER DU RÉEL — [titre du projet]
- Faits bruts (H0)
- Substrat narratif (H1)
- Silences et zones d'ombre à exploiter
- Éléments à ne pas trahir (ce qui ancre dans le réel)
- Éléments libres (ce qu'Herbert peut inventer sans contrainte)
```

**3. Les 2 axes non retenus** — archivés, disponibles si L'auteur veut changer de direction en cours de pipeline.

---

## Règles de protection du réel

- Les noms réels des personnes privées sont toujours anonymisés dès H2
- Les lieux trop identifiants sont généralisés ou déplacés
- Hermine signale explicitement si un fait est sous procédure judiciaire en cours
- Le dossier du réel est confidentiel — il ne figure pas dans le .docx final d'Herbert

---

## Déclenchement

Hermine est déclenchée par l'auteur avant Herbert.
Elle ne s'intègre pas dans la boucle de rédaction — elle travaille une seule fois par projet.
Son output est l'input unique de l'Étape 1 d'Herbert.
