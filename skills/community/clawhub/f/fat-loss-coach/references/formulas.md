# 热量计算公式

## 基础代谢率 (BMR)

### Mifflin-St Jeor 公式（推荐）

目前最准确的估算公式，适用于大多数成年人。

**男性：**
```
BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄(岁) + 5
```

**女性：**
```
BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄(岁) - 161
```

### Harris-Benedict 公式（备选）

较老的公式，可能高估代谢5%左右。

**男性：**
```
BMR = 88.362 + (13.397 × 体重kg) + (4.799 × 身高cm) - (5.677 × 年龄)
```

**女性：**
```
BMR = 447.593 + (9.247 × 体重kg) + (3.098 × 身高cm) - (4.330 × 年龄)
```

### Katch-McArdle 公式（体脂率已知时使用）

需要知道体脂率，对肌肉量较高的人更准确。

```
BMR = 370 + 21.6 × 瘦体重(kg)
瘦体重 = 总体重 × (1 - 体脂率)
```

## 每日总消耗 (TDEE)

TDEE = BMR × 活动系数

### 活动系数

| 活动水平 | 描述 | 系数 |
|----------|------|------|
| 久坐 | 几乎不运动，办公室工作 | 1.2 |
| 轻度活动 | 每周运动1-3次 | 1.375 |
| 中度活动 | 每周运动3-5次 | 1.55 |
| 高度活动 | 每周运动6-7次 | 1.725 |
| 极度活动 | 体力劳动或每天两次训练 | 1.9 |

---

## 减脂热量目标

### 标准建议

```
减脂热量 = TDEE - 热量缺口
```

**推荐缺口：** 300-500 kcal/天

**安全下限：**
- 男性：不低于 1500 kcal/天
- 女性：不低于 1200 kcal/天

### 减重速度估算

- 缺口 500 kcal/天 ≈ 每周减 0.5 kg（主要是脂肪）
- 缺口 1000 kcal/天 ≈ 每周减 1 kg（不建议，易掉肌肉）

### 不同目标的热量调整

| 目标 | 热量设置 |
|------|----------|
| 激进减脂 | TDEE - 500~750 kcal |
| 温和减脂 | TDEE - 300~500 kcal |
| 维持体重 | TDEE |
| 增肌 | TDEE + 200~300 kcal |

## 宏量营养素计算

### 蛋白质

**减脂期建议：** 1.6-2.2 g/kg 体重/天

计算公式：
```
蛋白质摄入(g) = 体重(kg) × 1.8（取中值）
蛋白质热量 = 蛋白质摄入(g) × 4 kcal/g
```

### 脂肪

**减脂期建议：** 0.6-1.0 g/kg 体重/天

计算公式：
```
脂肪摄入(g) = 体重(kg) × 0.8（取中值）
脂肪热量 = 脂肪摄入(g) × 9 kcal/g
```

### 碳水化合物

剩余热量分配给碳水：

```
碳水热量 = 目标热量 - 蛋白质热量 - 脂肪热量
碳水摄入(g) = 碳水热量 ÷ 4 kcal/g
```

## 计算示例

**用户档案：**
- 性别：男
- 年龄：30岁
- 身高：175 cm
- 体重：80 kg
- 活动水平：轻度活动（每周运动2次）

**计算过程：**

1. BMR = 10×80 + 6.25×175 - 5×30 + 5 = 800 + 1093.75 - 150 + 5 = **1748.75 kcal**

2. TDEE = 1748.75 × 1.375 = **2404 kcal**

3. 减脂热量 = 2404 - 400 = **2004 kcal**（取整约 2000 kcal）

4. 宏量分配：
   - 蛋白质：80 × 2 = 160g = 640 kcal（32%）
   - 脂肪：80 × 0.8 = 64g = 576 kcal（29%）
   - 碳水：2000 - 640 - 576 = 784 kcal = 196g（39%）

**结果：** 每日 2000 kcal，蛋白质 160g，脂肪 64g，碳水 196g

## 特殊情况调整

### 平台期

如果连续2周体重无变化：
1. 重新计算 TDEE（体重下降后代谢降低）
2. 减少 100-200 kcal
3. 增加运动消耗
4. 考虑碳水循环或轻断食

### 高体重人群（BMI > 30）

- 使用目标体重而非当前体重计算
- 缺口可适当增大（500-750 kcal）
- 密切关注身体反应

### 老年人（65岁+）

- BMR 下降约 10-15%
- 蛋白质需求增加至 1.2-1.5 g/kg
- 注意钙和维生素D补充

### 青少年（18岁以下）

- 不建议严格限制热量
- 优先增加运动
- 保证蛋白质和微量元素摄入
