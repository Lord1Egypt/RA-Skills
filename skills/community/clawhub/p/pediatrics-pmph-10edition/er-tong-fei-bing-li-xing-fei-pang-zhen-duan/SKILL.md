# 儿童单纯性肥胖诊断

## 适用对象
- 儿童与婴幼儿（排除病理性肥胖）

## 诊断流程

### 1. 年龄 ≥2岁儿童
- 计算BMI = 体重（kg） / [身高（m）]²
- 对照同性别、同年龄段参考值百分位：
  - 超重：P85 ≤ BMI ≤ P95
  - 肥胖：BMI > P95

### 2. 年龄 <2岁婴幼儿
- 使用“身长别体重”指标
- 对照同性别、同年龄段参考值百分位：
  - 超重：P85 ≤ 身长别体重 ≤ P97
  - 肥胖：身长别体重 > P97

## 注意事项
- 必须使用标准化生长曲线（如WHO或中国标准）。
- 需排除病理性肥胖（如内分泌疾病、遗传综合征）。
- 肥胖高发年龄为婴儿期、5～6岁和青春期。

## 输出结果
- 返回“正常”、“超重”或“肥胖”分类
- 建议是否需进一步评估并发症（如脂肪肝、高血压、糖耐量异常）
