# Verdict Rewrite Patterns / 判词重写规则

用途：生成或修复首屏判词。读取 `life-archetype-library.md` 和 `archetype-tension-axes.md` 后，用本文件把普通总结改成能命名隐秘遗憾的判词。

## 判词四件套

强判词必须同时包含：

1. 承载物：城市、海外、房子、体制名额、大厂、旧公司、AI 窗口、创业念头。
2. 时间或场景：2016、2020-2022、2023、疫情、签证、月供、平台会议、旧项目、收藏夹。
3. 隐秘交换：把什么换成了什么。
4. 今日回收方向：作品、反馈、英文案例、工作流、现金流边界、外部网络。

公式：

```text
{{承载物}}在你心里，像{{隐喻}}。你反复想起它，指向{{隐秘交换}}：你怀疑自己把{{放下的训练/机会/身份}}，换成了{{保住的安全/熟悉/低摩擦}}。今天能回收的，是{{可回收资产}}。
```

## 替换测试

判词写完后做三步：

1. 删掉承载物，读起来还能套到别的题，重写。
2. 删掉年份或具体场景，仍然成立，补事件锚点。
3. 看不到“换出”和“换入”，补隐秘交换。

## 命中感三问

- 这句话有没有说出用户不愿承认的交换？
- 这句话有没有把遗憾落到一个具体年份、地方、物件或场景？
- 这句话有没有让用户知道今天还能回收哪一部分？

三问缺一项，Magic Score 最高 16 / 20。

## 弱判词与强判词

| 场景 | 弱判词 | 强判词 |
| --- | --- | --- |
| 去北京 | 你放不下北京，因为北京机会更多。 | 北京在你心里，像 2015-2019 平台增长期那张更早上牌桌的入场券。你怀疑自己把平台会议、同行比较和外部可见度，换成了深圳更稳的现金流和低摩擦生活。 |
| 海外未竟 | 你纠结自己有没有错过国际化。 | 2016 年后的英国在你心里，像一份没写完的第二身份。你担心自己把签证、英文职场和孤独训练，换成了深圳中文环境里更熟悉的发展。 |
| AI 窗口 | 你错过了 AI 机会，但今天还能补。 | 2023 年的 AI 在你心里，像一扇打开过却没走进去的门。真正让你难受的点在于，你已经看见浪潮，却把作品窗口和工作流资产，换成了收藏、试用和旁观。 |
| 买房绑定 | 房子给了你安全感，也限制了你。 | 2018 年那套房像一块把生活按住的锚。它给了家人能理解的安全感，也拿走了换城、换行业和承受空窗的弹性。 |
| 旧公司 | 你离开旧公司后有点后悔。 | 旧公司像疫情后那条熟悉又发闷的走廊。你回头看它，指向你把低沟通成本、熟人支持和稳定节奏，换成了新环境里的重新定价。 |

## 失败判词

看到以下情况，直接重写：

- 只说“你放不下的是过去的自己”。
- 只说“你需要看清机会和代价”。
- 只说“这段经历塑造了你”。
- 只说“现在开始也可以”。
- 没有时间、地点、身份、物件或事件。
- 没有说清换出与换入。

## 回写动作

低分判词按这个顺序修：

1. 从 `archetype-tension-axes.md` 找主张力轴。
2. 从 `life-archetype-library.md` 找主原型和副原型。
3. 把承载物写进第一句。
4. 把时间或事件写进第一句或第二句。
5. 用“换”写出隐秘交换。
6. 在第三句写今天能回收的资产。

## 输出长度

- 首屏判词建议 55-95 字。
- 过短会像标题。
- 过长会像解释段。
- HTML 首屏只放判词，不放评分和依据。
