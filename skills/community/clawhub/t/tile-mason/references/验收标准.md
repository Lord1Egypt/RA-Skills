# 瓦匠工程验收标准参考

## 瓷砖铺贴验收标准

| 检查项 | 合格标准 | 检测工具 |
|--------|---------|---------|
| 空鼓率 | 单块砖空鼓面积 ≤ 15%，单空间空鼓率 ≤ 5%（地砖），≤ 10%（墙砖） | 空鼓锤（金属棒） |
| 平整度 | 2m 靠尺检查，误差 ≤ 2mm | 2m 靠尺、塞尺 |
| 垂直度 | 2m 垂直检测尺，误差 ≤ 2mm | 垂直检测尺 |
| 阴阳角方正 | 90° ± 2mm | 阴阳角尺/直角尺 |
| 接缝高低差 | 瓷砖接缝高低差 ≤ 0.5mm | 钢尺、塞尺 |
| 接缝宽度 | 统一留缝，误差 ≤ ±0.5mm | 钢尺/卡尺 |
| 坡度（涉水区） | 地漏处排水坡度 ≥ 1%（1cm/m） | 水平尺、拉线 |
| 切割边 | 切口平整、无崩边崩角 | 目测 |
| 十字缝 | 十字交叉处平整、对齐 | 目测 |

### 地砖 vs 墙砖验收差异

- **地砖**: 空鼓率更严格（≤ 5%），必须做排水坡度测试
- **墙砖**: 重点关注垂直度和十字缝对齐，墙面不得有整砖空鼓
- **大板砖（600×1200 及以上）**: 空鼓更容易产生，需用双面上浆工艺

## 卫生间/阳台防水验收标准

| 检查项 | 合格标准 | 检测工具 |
|--------|---------|---------|
| 防水涂刷高度 | 淋浴区 ≥ 1.8m，其他区域 ≥ 0.3m | 卷尺 |
| 防水层厚度 | 涂料 ≥ 1.5mm，卷材按产品要求 | 湿膜测厚仪 |
| 闭水试验 | 蓄水深度 ≥ 20mm，持续 24-48h 无渗漏 | 目测 |
| 管根处理 | 管根圆弧处理 + 附加层 | 目测 |
| 墙角处理 | 阴角抹弧 + 附加层 ≥ 100mm 宽 | 目测 |
| 门口挡水 | 门口设止水坎 | 目测/水平尺 |

## 地面找平验收标准

| 检查项 | 合格标准 | 检测工具 |
|--------|---------|---------|
| 平整度（找平层） | 2m 靠尺误差 ≤ 3mm（贴木地板 ≤ 2mm） | 2m 靠尺、塞尺 |
| 空鼓 | 无空鼓（自流平/水泥砂浆） | 敲击听声 |
| 强度 | 表面不起砂、不脱层 | 硬物划试 |
| 坡度 | 按设计要求，排水区 ≥ 1% | 水平尺 |

## 门槛石/窗台石验收

| 检查项 | 合格标准 |
|--------|---------|
| 安装牢固度 | 无松动、无空鼓感 |
| 水平度 | 水平尺检查，误差 ≤ 1mm |
| 两侧留边 | 两侧嵌入墙内 ≥ 20mm |
| 窗台石出墙 | 突出墙面 ≥ 10mm（下口开滴水槽） |
