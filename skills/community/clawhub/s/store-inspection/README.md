# 🏪 巡店管理全流程助手 (Store Inspection Manager)

连锁门店巡店管理的 AI 全流程助手，从计划制定到数据分析一站式搞定。

## 功能模块

| 模块 | 说明 |
|------|------|
| 📋 **巡店计划** | 自动排期、优先级排序、路线建议 |
| 🔍 **巡店执行** | 10维加权评分、整改记录 |
| 📈 **营销方案** | 薄弱诊断、P0/P1行动方案 |
| 📊 **数据分析** | 交互式HTML报告、雷达图、趋势分析 |

## 10维检查体系

| 维度 | 权重 | 检查项 |
|------|------|--------|
| 店面形象 | 10% | 门头、灯光、清洁、橱窗 |
| 商品陈列 | 15% | 陈列面、价签、缺货、季节性 |
| 员工服务 | 10% | 着装、问候、产品知识、态度 |
| 安全合规 | 10% | 消防、灭火器、监控、标识 |
| 库存管理 | 10% | 准确率、FIFO、过期品、补货 |
| POS系统 | 5% | 收银、支付、耗材 |
| 客户体验 | 10% | 动线、试衣间、排队、投诉 |
| 营销执行 | 10% | 促销、POP、会员、效果 |
| 防损安保 | 10% | EAS、高值品、现金、钥匙 |
| 卫生环境 | 10% | 地面、卫生间、仓库、垃圾 |

## 快速开始

```bash
# 演示模式（生成模拟数据+完整报告）
python scripts/store_inspection.py --mode demo

# 制定巡店计划
python scripts/store_inspection.py --mode plan

# 执行巡店检查
python scripts/store_inspection.py --mode execute --store "XX路店"

# 生成营销方案
python scripts/store_inspection.py --mode marketing --store "XX路店"

# 数据分析生成报告
python scripts/store_inspection.py --mode analyze
```

## 评分标准

- ✅ 通过 (100分)：完全符合标准
- ⚠️ 部分通过 (60分)：基本达标需改进
- ❌ 不通过 (0分)：严重不符合需立即整改

评级：A(90+) → B(80-89) → C(70-79) → D(60-69) → E(<60)

## 技术栈

- Python 3.8+
- Chart.js (CDN) 数据可视化
- 本地 JSON 数据存储
- 交互式 HTML 报告输出
