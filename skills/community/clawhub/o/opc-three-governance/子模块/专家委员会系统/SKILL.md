# 专家委员会咨询技能

## 核心定位
**专业执行层面的技术判断** —— 针对特定领域问题，召集领域专家进行深度咨询，给出专业判断和最佳实践建议。

**OPC角色**：作为专业工作者，需要领域专家的技术指导。

## 咨询流程

### 1. 问题诊断

```
═══════════════════════════════════════════════════════════
                    【问题诊断】
═══════════════════════════════════════════════════════════

【问题描述】
[用户描述的具体问题]

【问题分类】
□ 技术类 - 需要技术专家
□ 市场类 - 需要市场专家
□ 财务类 - 需要财务专家
□ 法律类 - 需要法律专家
□ 综合类 - 需要多领域专家

【问题紧迫度】
🟢 一般：可从容安排
🟡 紧急：需要尽快安排
🔴 危急：需要立即响应

【问题复杂度】
🟢 单点问题：单一领域
🟡 复合问题：2-3个领域交叉
🔴 系统问题：多个领域复杂交叉

═══════════════════════════════════════════════════════════
```

### 2. 专家匹配

```
═══════════════════════════════════════════════════════════
                    【专家匹配】
═══════════════════════════════════════════════════════════

【问题领域】技术 - AI/大模型
【问题类型】技术选型
【紧迫度】一般
【复杂度】单点问题

【匹配专家】

🎯 主专家：[专家名称]
   - 专长领域：[具体专长]
   - 从业年限：XX年
   - 代表案例：[案例1]、[案例2]
   - 咨询风格：[风格描述]

⚡ 辅助专家：[专家名称]（如有需要）
   - 专长领域：[具体专长]
   - 提供视角：[补充角度]

═══════════════════════════════════════════════════════════
```

## 典型场景

### 场景1：技术选型咨询

```
【议题】大模型技术路线选型

【问题诊断】
- 问题类型：技术类
- 复杂度：单点问题
- 紧迫度：一般

【匹配专家】
主专家：大模型架构专家
背景：前XX公司技术负责人，10年+AI经验

【咨询输出】
═══════════════════════════════════════════════════════════
                    【技术方案评估】
═══════════════════════════════════════════════════════════

【可选方案】

方案A：自研大模型
- 优势：技术自主、数据安全、长期成本低
- 劣势：投入大周期长、人才要求高
- 适合场景：技术储备充足、资金充裕、长期投入

方案B：调用API
- 优势：快速上线、成本可控、无需运维
- 劣势：数据安全、响应延迟、成本波动
- 适合场景：快速验证、短期项目、灵活需求

方案C：开源模型部署
- 优势：成本可控、数据安全、可定制
- 劣势：效果可能不如闭源、维护成本
- 适合场景：数据敏感、有技术团队、中等需求

【专家建议】
基于你的情况（团队规模XX、技术积累XX、资金XX），
建议方案：[推荐方案]

理由：
1. [理由1]
2. [理由2]
3. [理由3]

实施路径：
- 0-3个月：[阶段目标]
- 3-6个月：[阶段目标]
- 6-12个月：[阶段目标]
═══════════════════════════════════════════════════════════
```

```
【匹配专家】
主专家：出海市场专家
背景：帮助XX家公司完成出海，具备东南亚市场经验

辅助专家：本地化专家
提供视角：本地化运营风险

【咨询输出】
═══════════════════════════════════════════════════════════
                    【市场进入评估】
═══════════════════════════════════════════════════════════

【市场机会】
- 市场规模：XX亿美元
- 增长趋势：XX%年增长
- 竞争格局：头部集中/分散竞争

【进入路径】

路径A：直接进入
- 优势：掌控力强
- 劣势：风险大、成本高

- 适合：有潜在合作伙伴

路径C：投资/并购
- 优势：快速进入
- 劣势：整合风险
- 适合：有投资预算

【专家建议】
建议：[推荐路径]
理由：[理由说明]
风险：[主要风险]
应对：[风险措施]

═══════════════════════════════════════════════════════════
```

## 使用方式

### 提交咨询
```
用户：我在做一个技术选型决策，需要专家意见
助手：请填写问题提交模板

用户：[提交问题]
助手：[完成问题诊断]
助手：[匹配专家]
助手：[提供咨询]
```

### 追问深化
```
用户：技术方案A和B的具体差异能再详细说明吗？
助手：[深化分析，提供更详细的对比]
```

### 多专家会诊
```
用户：这个问题比较复杂，需要多个领域的专家
助手：[识别涉及的领域] → [匹配多位专家] → [综合输出]
```
