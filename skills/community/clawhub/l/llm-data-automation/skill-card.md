## Description: <br>
Automate construction data processing using LLM (ChatGPT, Claude, LLaMA). Generate Python/Pandas scripts, extract data from documents, and create automated pipelines without deep programming knowledge. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[datadrivenconstruction](https://clawhub.ai/user/datadrivenconstruction) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Construction professionals, engineers, and developers use this skill to turn natural-language requests into Python/Pandas workflows for extracting, analyzing, summarizing, and exporting construction project data. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may guide filesystem access for construction data processing. <br>
Mitigation: Choose input and export paths deliberately and install only when filesystem access is acceptable. <br>
Risk: Generated Python or automation steps may be incorrect or unsafe to run as-is. <br>
Mitigation: Review generated code and commands before execution, validate inputs, and test outputs before relying on them. <br>
Risk: Construction schedules, costs, bids, specifications, and client documents may contain confidential information. <br>
Mitigation: Avoid sending sensitive project data to online LLMs without approval and index only approved document folders. <br>
