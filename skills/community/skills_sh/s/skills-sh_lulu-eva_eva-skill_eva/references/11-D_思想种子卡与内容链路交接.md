# Eva Learn 11-D：思想种子卡与内容链路交接

适用场景：Eva Learn 已产生阶段性判断，用户想沉淀，或想把学习结果转入内容链路。

本文件只做思想种子卡和交接判断，不直接写稿。

## 生成条件

满足任一条件才生成思想种子卡：

- 用户能复述、举例、反驳和迁移。
- 用户形成清楚的阶段性判断。
- 用户明确说要沉淀。
- 一个学习小闭环已经完成。

以下情况不得生成：

- 本轮掌握状态是“未验证”。
- 用户只是看完资料，但没有自己的判断。
- 只有书摘、摘要、概念表，没有判断版本。
- 用户想跳过人群、标题或第一句话直接成稿。

用户说“帮我沉淀”不等于可以生成思想种子卡。生成前必须先过检查表。

## 生成前检查表

生成思想种子卡前，逐项检查：

- 是否已有判断版本。
- 是否已有掌握状态。
- 掌握状态是否不是“未验证”。
- 是否有支持证据。
- 是否有反对证据或不确定处。
- 是否能写出“可讲给谁”。
- 是否能保存到 `06-思想种子卡.md`。

任一项不满足，不生成思想种子卡，只说明缺口，并路由回 `11-A`、`11-B` 或 `11-C` 继续学习。

## 思想种子卡格式

```text
## 思想种子卡：{标题}

- 来源项目：
- 触发问题：
- 阶段性判断：
- 支持证据：
- 反对证据：
- 我的经验/场景：
- 可讲给谁：
- 用户可能正在问：
- 不确定处：
- 成熟度：萌芽 / 可进入内容入口 / 可进入正文前校验
- 下一步：
```

思想种子卡必须保留“不确定处”。不要把阶段性判断包装成绝对真理。

生成后追加到 `06-思想种子卡.md`，关键问答追加到 `07-学习问答原稿.md`。

## 成熟度判断

| 成熟度 | 标准 | 下一步 |
|---|---|---|
| 萌芽 | 有兴趣或概念，但判断、证据、用户场景不清 | 回到 `11-A`、`11-B` 或 `11-C` 继续学习 |
| 可进入内容入口 | 有判断和材料，但人群、用户疑问、标题或第一句话不清 | 交给 `03_eva-audience-finder` 或 `shortvideo/01_eva-title` |
| 可进入正文前校验 | 判断、证据、人群、用户疑问和入口都清楚 | 仍需先过标题/第一句话交接，再进入正文链路 |

禁止使用“可成稿”作为成熟度。

## 内容链路硬闸门

思想种子卡不是：

- 标题交接卡。
- 第一句话交接卡。
- 正文任务卡。
- 商单约束卡。

进入内容链路必须经过：

```text
思想种子卡 -> 人群/用户疑问判断 -> 标题或第一句话入口 -> 正文路线图 -> 正文撰写
```

## 内容入口预备卡

思想种子卡进入内容链路前，必须先输出内容入口预备卡。它不是标题交接卡、第一句话交接卡或正文任务卡，只负责把学习成果交给人群、标题或第一句话入口。

```markdown
## 内容入口预备卡

- 阶段性判断：
- 支持证据：
- 反对证据 / 不确定处：
- 可讲给谁：清楚 / 不清楚
- 用户可能正在问：
- 真实缺口：
- 适合平台：小红书 / 抖音 / 视频号 / 待确认
- 建议入口：先做人群判断 / 标题搜索 / 第一句话开头
- 不得直接写成稿的原因：
```

缺 `可讲给谁`、`用户可能正在问` 或 `真实缺口` 任一项时，必须先读取 `03_eva-audience-finder_话题人群识别器.md`。

路由规则：

| 当前缺口 | 读取 |
|---|---|
| 不知道讲给谁 | `03_eva-audience-finder_话题人群识别器.md` |
| 要做小红书、有标题、需要封面标题 | `shortvideo/01_eva-title_标题即选题.md` |
| 做抖音/视频号、无标题口播、前 5 秒 | `shortvideo/02_eva-script_思维流爆款内容创作.md`，再按需读 `shortvideo/03_eva-opening_开头针对性优化.md` |
| 标题交接卡或第一句话交接卡已经成立 | `shortvideo/02_eva-script_思维流爆款内容创作.md` |

## 交接前状态闸门

| 节点 | 未完成前禁止 | 唯一下一步 |
|---|---|---|
| 未生成思想种子卡 | 禁止进入内容链路 | 先执行生成前检查表 |
| 未判断成熟度 | 禁止路由到标题或正文 | 先判断萌芽 / 可进入内容入口 / 可进入正文前校验 |
| 成熟度为萌芽 | 禁止进入内容链路 | 回到 `11-A`、`11-B` 或 `11-C` 继续学习 |
| 未输出内容入口预备卡 | 禁止路由到标题、第一句话或正文 | 先输出内容入口预备卡 |
| 缺少人群或用户疑问 | 禁止进入标题和正文 | 读取 `03_eva-audience-finder_话题人群识别器.md` |

## 禁止

- 不把思想种子卡直接改成完整稿。
- 不绕过人群、标题或第一句话入口。
- 不把未验证学习结果包装成确定观点。
- 不编造用户经历、书中案例、引文或页码。
