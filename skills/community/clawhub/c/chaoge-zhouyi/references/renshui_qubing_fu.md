# 壬水祛病符修持指南（王兵定制版）

## 概述

**壬水祛病符**，专为**八字火旺、用神壬水被克**者设计。适用于**火旺之极、水火不济、调候失效**等症状。

**起源**： based on 滴天髓理论，结合八字用神原理，定制符箓。

**核心作用**：
1. 补水调候
2. 制火润局
3. 水火既济
4. 祛病延年

|----------|------|----------|

| 用神被克 | 壬水用神不显，被火旺克制 | 壬水祛病符 |

| 火气血热 | 伤口难愈、术后并发症 | 壬水祛病符 + 补土和中符 |

### 二、王兵案例适配

**王兵八字**：
- 年柱：丁酉 (火鸡)
- 月柱：乙巳 (木蛇)
- 日柱：丙寅 (火虎)  ← 日主丙火
- 时柱：癸巳 (水蛇)

1. **火旺之极**：丙火身旺，得令得地得势
2. **用神被克**：壬水用神不显，被火旺克制
3. **缺土中虚**：八字无土，胃切除后中气更虚
4. **金弱卫疏**：金弱被火克，卫外不固

**适用符法**：**壬水祛病符**（首选） + **安魂符**（辅助） + **补土和中符**（配合）

## 书符方法

1. **天德日**：天德星值日（黄道吉日）

**忌择日**：
1. **丙午日**：火旺之日（凶）
2. **丁巳日**：火旺之日（凶）
3. **破日**：日月相冲（凶）
4. **害日**：日月相害（凶）

**查询方法**：
- 查《黄历》或《通胜》
- 选"宜祈福、宜祭祀"之日
- 优先选**壬子日**或**癸亥日**

### 二、择时

**宜择时**：
1. **子时（23-1点）**：水时，补水制火（最宜）
2. **亥时（21-23点）**：水时，补水制火（次宜）
3. **申时（15-17点）**：金生水，用神得助（宜）
4. **酉时（17-19点）**：金时，补金卫外（宜）

**忌择时**：
1. **午时（11-13点）**：火时，火旺（凶）
2. **巳时（9-11点）**：火时，火旺（凶）
3. **寅时（3-5点）**：木生火，火旺（凶）

### 三、择方

**宜择方**：
1. **北方**：水方，补水制火（最宜）
2. **西北方**：金水之方，补水补金（次宜）
3. **西方**：金方，补金卫外（宜）

**忌择方**：
1. **南方**：火方，火旺（凶）
2. **东方**：木方，木生火（凶）

### 四、斋戒

**斋戒要求**：
1. **前三天**：戒荤腥（不吃肉、不吃葱蒜韭菜）
2. **前三天**：戒房事（不与妻妾同房）
3. **前三天**：戒饮酒（不喝酒）
4. **前一日**：沐浴更衣（洗澡换干净衣服）

**斋戒目的**：
1. 清净身心
2. 增强灵力
3. 感通神灵

### 五、净身

**净身步骤**：
1. **沐浴**：洗澡（洗净身体）
2. **更衣**：换干净衣服（最好黑色或蓝色，水色）
3. **净口**：刷牙漱口（不说脏话）
4. **净手**：洗手（不摸脏物）
5. **净心**：静坐片刻（平心静气）

### 六、设坛

**设坛要求**：
1. **设坛位置**：安静、干净、向阳的房间
2. **神位设置**：供**水官大帝**神位（或画像）
3. **供品准备**：清水一杯、香三支、蜡烛一对
4. **符纸准备**：黑纸（或蓝纸）三张
5. **笔墨准备**：墨水（或蓝墨）、新笔
