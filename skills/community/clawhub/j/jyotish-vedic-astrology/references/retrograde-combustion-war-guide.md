## 一、逆行行星（Vakri Graha）

### 1.1 逆行频率与持续时间

| 行星 | 逆行频率 | 持续时间 | 太阳/月亮 |
|------|---------|---------|----------|
| 水星 | 每年3-4次 | ~21天 | — |
| 金星 | 每18个月 | ~42天 | — |
| 火星 | 每26个月 | ~72天 | — |
| 木星 | 每年一次 | ~120天 | — |
| 土星 | 每年一次 | ~140天 | — |
| Rahu/Ketu | **永远逆行** | 持续 | — |
| 太阳/月亮 | **永不逆行** | — | — |

### 1.2 逆行核心含义

> 逆行 ≠ 坏事。逆行 = **回溯、复习、重新处理未完成的业力**。

| 维度 | 正行 | 逆行 |
|------|------|------|
| 方向 | 向前推进 | 回顾、内省 |
| 能量 | 外向表达 | 内向深化 |
| 事件 | 新事物 | 旧事重现 |
| 业力 | 新业力 | 旧业力清算 |

### 1.3 各行星逆行精确含义

#### 水星逆行
- **沟通**：误解、延迟回复 → 重读合同、确认细节
- **商业**：旧客户回归、旧项目重启 → 适合"回头客"
- **学习**：复习效率高 → 不宜开始新课程
- ⚠️ 水逆影响被严重夸大，适合回顾/修订/重新联系

#### 金星逆行
- **感情**：前任回归、旧感情模式重现 → 审视需求、不匆忙决定
- **财务**：旧债务浮现、旧投资回报 → 清理财务
- **深层**：重新审视自己的价值观和关系需求
- 本命金星逆行 → 一生多次"关系回顾"主题

#### 火星逆行 ⚠️ 最具破坏性
- **行动力**：执行延迟、计划反复 → 策略调整期
- **冲突**：旧冲突重现 → 正面对话
- **绝对避免**：开始诉讼、发动冲突、高风险运动
- 能量被"内爆"，外在推进力下降

#### 木星逆行
- **学习**：旧知识体系重新整合 → 深度复习
- **信仰**：信念系统受挑战 → 开放心态接受新视角
- **秘密**：Shadbala的Chesta Bala给满分（逆行=力量增强）
- 木星逆行是**深化学识和智慧的时期**，不是坏事

#### 土星逆行
- **责任**：旧责任重新浮现 → 正视未完成义务
- **本命土星逆行**：一生中对自己要求极为严格（内化的父亲形象）

### 1.4 "三过"现象

逆行行星会**三次经过同一个度数**（正行→逆行→正行）：

| 经过次数 | 含义 | 影响深度 |
|---------|------|---------|
| 第一次（正行） | 预告信号 | 轻微 |
| 第二次（逆行） | **影响最深** | 最强 |
| 第三次（正行回归） | 最终确认 | 中等 |

### 1.5 本命逆行行星通用原则

> 本命逆行行星 = 该领域有"前世未完成的课题"，今生需要"重新处理"。

#### 水星逆行在12宫位

| 宫位 | 含义 | 人生表现 |
|------|------|---------|
| 1宫 | 自我表达反复修正 | 沟通风格先内向后外向 |
| 2宫 | 财务思维需回顾 | 财务计划反复调整 |
| 3宫 | 沟通方式需打磨 | 写作/说话多次修改才满意 |
| 4宫 | 家庭沟通有旧课题 | 对话反复出现同一主题 |
| 5宫 | 创意需酝酿 | 创作反复，最终精品 |
| 6宫 | 工作细节需复核 | 适合审查/编辑/质检 |
| 7宫 | 关系沟通需重建 | 伴侣对话模式反复调整 |
| 8宫 | 深层思维反复探索 | 研究/调查天赋 |
| 9宫 | 高等教育有回顾期 | 学术方向可能变更 |
| 10宫 | 职业沟通反复打磨 | 职业形象多次重塑 |
| 11宫 | 社交网络有回流 | 旧友/旧圈子反复出现 |
| 12宫 | 灵性沟通内在化 | 直觉强、内在对话丰富 |

## 二、燃烧行星（Combust/AST）

### 2.1 燃烧度数表

| 行星 | 顺行燃烧度数 | 逆行燃烧度数 |
|------|------------|------------|
| 月亮 | 12° | — |
| 火星 | 17° | 10° |
| 水星 | 14° | 12° |
| 木星 | 11° | 10° |
| 金星 | 10° | 8° |
| 土星 | 15° | 10° |

### 2.2 燃烧核心含义

> 燃烧 = 行星的独立力量被太阳的"自我"覆盖。不是"毁灭"，而是"被同化"。

| 维度 | 正常行星 | 燃烧行星 |
|------|---------|---------|
| 表现力 | 独立自主 | 被太阳意志覆盖 |
| 心理层面 | 独立判断 | 受"自我/权威"过度影响 |
| 事件层面 | 正常发展 | 被"权威/父亲/自我"主导 |

### 2.3 各行星燃烧精确含义

#### 月亮燃烧 ⚠️ 最严重
- 情绪被"自我意识"覆盖，难以识别真实感受
- 与母亲关系紧张或母亲被权威压制
- 安全感不稳定，过度依赖外部认可

#### 水星燃烧 ⚠️ 最常见
- 沟通被"自我"主导，容易主观臆断
- 学习方向受父亲/权威影响
- 商业判断容易被"自信"蒙蔽

#### 金星燃烧
- 感情需求被"自我"覆盖，难以"看见"对方
- 审美被"自我表达"主导
- 消费决策容易被"面子"驱动

#### 火星燃烧
- 行动方向被"自我意志"主导
- 愤怒管理困难（太阳放大火星攻击性）
- 勇气变成鲁莽

#### 木星燃烧
- 智慧被"自我"限制，难接受更智慧的导师
- 信仰体系受权威深刻影响

#### 土星燃烧
- 自律与"自我意志"冲突
- 太阳急切+土星延迟=内心矛盾

### 2.4 燃烧缓解条件

| 条件 | 缓解程度 |
|------|---------|
| 行星入庙/旺相 | **强缓解** |
| 行星在角宫 | 中等缓解 |
| 行星在友星星座 | 轻微缓解 |
| 行星逆行 | **无缓解**（甚至加重） |
| 吉星相位 | 中等缓解 |

### 2.5 燃烧行星在Dasha中的表现

| Dasha情况 | 效果 |
|-----------|------|
| 燃烧行星的Dasha | 该领域被"自我/权威"主导 |
| 太阳Dasha+燃烧行星Antardasha | 燃烧效果**加倍** |
| 非燃烧星Dasha+燃烧行星Pratyantar | 短期影响，可控 |

## 三、行星战争（Graha Yuddha）

### 3.1 战争条件

**两颗行星在同一星座内且度数差≤1°**。

- 太阳和月亮**不参与**
- Rahu和Ketu**不参与**
- 只有**火星、水星、木星、金星、土星**之间发生

### 3.2 胜负判定

**最常用规则**：**度数更大者胜**。

示例：火星在白羊座15°30'，金星在白羊座15°45' → 度数差0°15' < 1° → 金星胜

### 3.3 战争效果

| 维度 | 胜者 | 败者 |
|------|------|------|
| Shadbala | 增加 | 减少 |
| 征象 | 顺利推进 | 受阻/延迟 |
| 宫位事务 | 顺利 | 困难 |
| 心理 | 自信 | 不安全 |

### 3.4 具体行星对战争

| 战争对 | 胜者含义 | 败者含义 | 备注 |
|--------|---------|---------|------|
| **火星 vs 金星** | 行动主导感情 vs 感情主导行动 | 感情/财务/合作 | |
| **火星 vs 水星** | 冲动压过理性 vs 理性控制冲动 | 沟通/商业/法律 | |
| **木星 vs 土星** | 乐观扩展 vs 保守限制 | 事业/投资/人生规划 | |
| **火星 vs 土星** | 行动力突破限制 vs 纪律压制冲动 | 事业/健康/冲突 | **最激烈** |
| **金星 vs 木星** | 感情/物质主导 vs 智慧/信仰主导 | 关系/财务/教育 | 两吉互战，非坏事 |

### 3.5 战争的特殊规则

1. **战争在角宫/三方宫** → 影响更大（人生核心领域）
2. **战争在6/8/12宫** → 影响减弱（像"秘密冲突"）
3. **战争行星同时逆行** → 影响"内化"，表现为内心冲突
4. **Transit经过战争度数** → 战争主题被重新激活

### 3.6 行星战争在Dasha中的表现

| Dasha情况 | 效果 |
|-----------|------|
| 胜者的Dasha | 战争"胜利果实"显现 |
| 败者的Dasha | 战争"创伤"显现，征象受阻 |
| Transit经过战争点 | 战争主题重新激活 |

## 四、三重叠加

| 叠加条件 | 综合效果 | 建议 |
|---------|---------|------|
| 逆行+燃烧 | 行星"内化"且"被自我覆盖" | Dasha期间需深度反思 |
| 逆行+战争败者 | 内在冲突严重 | 需补救措施 |
| 燃烧+战争败者 | 被双重压制 | 特别关注该领域 |
| 逆行+燃烧+战争败者 | **最极端** | 几乎无法独立运作 |

**一楠案例检查**：
- Mercury在白羊座9宫（与太阳同宫）
- 如果Mercury-太阳度数差<14° → Mercury燃烧
- Mercury同时落陷但有Neechabhanga
- 落陷+燃烧+但Neechabhanga取消落陷 = 需精确度数才能判断叠加效果

## 六、速查卡

**逆行**：回顾>前进，内向>外向，旧事>新事。木星逆行=好事，火星逆行=小心。
**燃烧**：独立力被太阳吞噬。月亮最严重，水星最常见。入庙可缓解。
**战争**：同星座<1°，度数大者胜。胜者强化，败者压制。火土战最激烈。
