# 子宫内膜周期性变化分析

## 适用对象
- 育龄期女性
- 子宫内膜组织

## 前提条件
- 卵巢功能正常
- 下丘脑-垂体-性腺轴完整

## 禁忌场景
- 绝经后女性
- 青春期前女性

## 执行流程

### 1. 判断当前月经周期阶段
根据用户提供的月经周期天数，确定所处阶段：
- **月经期**：第1–4天
- **增生期**：第5–14天（又称卵泡期）
- **分泌期**：第15–28天（又称黄体期）

### 2. 描述对应阶段的激素环境
- **月经期**：雌激素和孕激素水平骤降（因黄体退化）
- **增生期**：卵巢次级卵泡分泌雌激素为主
- **分泌期**：黄体分泌孕激素和雌激素

### 3. 分析子宫内膜组织学特征
按阶段说明以下内容：
- 内膜厚度变化
- 腺体形态与分泌状态
- 螺旋动脉行为
- 基质细胞变化

### 4. 输出结果
综合上述信息，输出当前阶段的内膜状态描述，包括厚度范围、关键组织学标志及激素驱动机制。

## 触发条件
当用户提供月经周期天数、内膜厚度、组织学描述或激素水平，并希望解释其生理意义时，调用本技能。
