# Xia Fa Classification and Application

## When to Use

在应用下法前，必须确认以下条件全部满足：

1. **病邪已入里**，无表证或半表半里证；
2. **属实证**，具备里实证的典型表现（如痞、满、燥、实四证）；
3. **患者体质尚可**，能耐受攻伐（非高龄津枯、素体虚弱、新产后、经期、妊娠期或脾胃极度虚弱者）。

> 若表里同病，应采用“表里双解法”，不可单用下法。

## Execution Steps

### 第一步：辨识里实证类型

根据临床表现判断里实证的具体性质：

- **热结**：阳明腑实，见大便不通、腹胀拒按、潮热谵语、苔黄脉实；
- **寒积**：胃肠冷积，见便秘、手足不温、腹痛喜温；
- **水饮壅盛**：胸腹积水、喘满不能平卧；
- **肠燥津亏**：大便干结难下、口干舌燥、舌红少津；
- **蓄血/瘀血内结**：少腹硬满、如狂发狂、大便色黑；
- **痰滞壅阻**：喉中痰鸣、胸膈痞塞、呕吐痰涎；
- **虫积**：腹痛时作、吐蛔或便蛔、面黄肌瘦。

### 第二步：选择下法子类

对应证型选择下法类别：

| 证型             | 下法子类 | 治疗目标               |
|------------------|----------|------------------------|
| 热结             | 寒下     | 清热通腑               |
| 寒积             | 温下     | 温阳攻积               |
| 水饮壅盛         | 逐水     | 攻逐水饮               |
| 肠燥津亏         | 润下     | 滋阴润肠               |
| 蓄血/瘀血        | 通瘀/攻瘀| 活血化瘀               |
| 痰滞             | 攻痰     | 涤痰开结               |
| 虫积             | 驱虫     | 驱杀肠道寄生虫         |

> 注意：通瘀、攻痰、驱虫等法虽属下法范畴，但下药常为佐使，主药需针对病因。

### 第三步：判断缓急程度

- **峻下**：仅用于病势急迫（如高热神昏、腹满痛剧）且体质强盛者；
- **缓下**：用于病势轻缓或体质偏弱者，避免伤正。

### 第四步：实施与停药原则

- 服药后若大便已通，应“**中病即止**”，不可过量或久用；
- 特别是峻下剂（如大承气汤），强调“大便已下，余药勿服”；
- 观察是否出现正气受损征象（如乏力、纳差、汗出肢冷），及时调整治疗。

## Precautions

以下情况禁用或慎用下法：

- 邪在表或半表半里未解；
- 阳明腑未实（无燥屎内结）；
- 高龄津枯、素体虚弱、新产后、经期、妊娠期、脾胃虚弱者——慎用峻下，必要时改用缓下或配伍扶正药。

> 本技能适用于中医临床辨证施治场景，尤其在处理便秘、腹胀、积滞、水饮、瘀血等里实证时提供系统性决策支持。
