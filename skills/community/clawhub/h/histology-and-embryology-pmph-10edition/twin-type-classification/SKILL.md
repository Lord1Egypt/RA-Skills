# 双胎类型判断

## 适用对象
- 人类胚胎
- 新生儿

## 触发条件
当观察到一次妊娠产出两个胎儿时，应用本技能判断其类型及发育来源。

## 执行步骤

1. **判断受精卵数量**：
   - 若来自**两个独立受精卵**，则为**双卵双胎（dizygotic twins）**：
     - 各自有胎膜与胎盘
     - 性别可相同或不同
     - 相貌和生理特征差异如同普通兄弟姐妹，仅年龄相同
   - 若来自**同一个受精卵**，则为**单卵双胎（monozygotic twins）**：
     - 遗传基因完全一致，性别相同，相貌、体态、生理特征极相似

2. **对单卵双胎，进一步判断分裂时机以确定胎盘与羊膜腔配置**：
   - **情况 a**：受精卵在胚泡形成前分裂 → 两个独立胚泡植入子宫
     - 结果：各自拥有**独立羊膜腔和独立胎盘**（双羊膜双绒毛膜）
   - **情况 b**：胚泡内出现两个内细胞群 → 分别发育为胚胎
     - 结果：各自位于**独立羊膜腔**，但**共享一个胎盘**（双羊膜单绒毛膜）
   - **情况 c**：胚盘上出现两个原条与脊索 → 形成两个神经管
     - 结果：**共处同一羊膜腔**，**共享一个胎盘**（单羊膜单绒毛膜）

3. **输出结果**：返回双胎类型（双卵/单卵）、胎盘数量、羊膜腔数量及遗传一致性状态。

## 注意事项
- 本技能不适用于非哺乳动物或多细胞生物的无性繁殖。
- 胎盘和羊膜腔结构可通过超声检查在孕早期评估。
- 单羊膜单绒毛膜双胎具有较高围产期风险，需特别关注。
