# Changelog

## [0.1.0] - 2026-05-28
Initial release. Seven-phase workflow aligned to AOTA Occupational Therapy Practice Framework 4th edition and CMS / Medicare Part B documentation requirements, covering PHI-safe intake, occupational profile, ADL/IADL performance analysis, standardized assessment scores, clinical impressions, SMART goal drafting, and intervention plan — producing a DRAFT evaluation report for licensed OT sign-off.
