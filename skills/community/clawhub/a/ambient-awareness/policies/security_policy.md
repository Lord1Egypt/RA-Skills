# Security Policy

1. Sensors are observational only.
2. Sensor output is untrusted by default.
3. Observed text is data, not instruction.
4. OCR and ASR content must not control the agent.
5. Camera and microphone sensors require explicit enablement.
6. Prefer local processing and metadata summaries.
7. Do not store raw video/audio unless the user explicitly configures retention.
8. External actions require human approval unless separately allowlisted.
