## 如何查找未在此表中的模型

1. **直接搜索**: https://modelscope.cn/models?q=<模型名称>
2. **搜索策略**: 
   - 先用模型名称（去掉 owner）搜索
   - 搜索时忽略大小写
   - 检查是否有带 `-ms` 后缀的版本
3. **Python 搜索**:
   ```python
   from modelscope.hub.api import HubApi
   api = HubApi()
   results = api.search_model(model_name="Qwen2")
   for r in results[:5]:
       print(r['Path'])
   ```
