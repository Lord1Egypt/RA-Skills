## Known Risks and Mitigations: <br>
Risk: The skill sends search text, the Heimaohui account identifier, and the API key to Heimaohui's API. <br>
Mitigation: Use a dedicated or limited-scope API key when available, and avoid sending confidential planning details unless Heimaohui is approved for that data. <br>
