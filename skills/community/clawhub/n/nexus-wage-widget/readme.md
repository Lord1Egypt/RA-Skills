# 工作日薪看板 Pro

一个单文件 HTML+CSS+JS 桌面 widget，集成实时工资计算与番茄钟专注时刻，采用玻璃质感设计，支持 7 种视觉主题与响应式布局。

## 功能特性

### 实时工资计算
- 根据月薪、上下班时间、工作天数（21.75 日/月）自动计算今日已赚金额
- 每秒刷新，显示精确到分的实时累积
- 进度条直观展示当日工作时长完成比例
- 发薪日倒计时

### 番茄钟专注时刻
- 25 分钟倒计时，支持开始/暂停
- 到期时触发浏览器桌面通知 + 标题栏闪烁提醒
- 宽屏版与手机版独立布局

### 7 种视觉主题
| 主题 key | 名称 | 风格 |
|----------|------|------|
| dark | 极客暗夜 | 深紫渐变，默认主题 |
| hermes | 爱马仕橙 | 橙金暖色，轻奢质感 |
| business | 总裁黑金 | 黑金搭配，商务沉稳 |
| girly | 烂漫少女 | 粉红基调，柔和活泼 |
| zen | 东方木漏 | 蓝绿禅意，清新自然 |
| light | 极简白昼 | 浅色背景，干净简约 |
| cyber | 赛博霓虹 | 荧光色彩，未来感 |

### 响应式布局
- **宽屏版**（>900px）：3 列 2 行网格布局，max-width 740px
- **方形版**（<=900px）：单列竖排，max-width 340px，适合小屏或浮窗

### 玻璃质感设计
- 圆角玻璃卡片，支持自定义透明度与模糊度
- 多层 box-shadow 模拟环境光 + 高光线
- 伪元素顶部边缘高光
- 面板悬停动画（上移、边框显色、阴影加深）
- 环境光粒子（3 个 ambient-light 元素）缓慢呼吸

### 设置面板
- 点击齿轮按钮打开模态框
- 可配置项：布局模式、视觉主题、月薪、发薪日、上下班时间、玻璃透明度、玻璃模糊度
- 切换主题即时预览
- 所有设置通过 localStorage 持久化（键名前缀 `widget_`）

## 使用方法

1. 在浏览器中直接打开 `references/daily_wage.html`
2. 默认加载极客暗夜主题，自动开始计时
3. 点击「专注时刻」面板启动番茄钟
4. 点击右下角齿轮按钮进入设置，调整参数

### 配置参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| 展示组件 | 宽版 / 方形布局 | 宽版 |
| 视觉主题 | 7 种可选 | 极客暗夜 |
| 月薪 | 税前月薪（元） | 10000 |
| 发薪日 | 每月几号发薪 | 10 |
| 上班时间 | 格式 HH:MM | 09:00 |
| 下班时间 | 格式 HH:MM | 18:00 |
| 玻璃透明度 | 0.1 ~ 0.9 | 0.4 |
| 玻璃模糊度 | 4px ~ 40px | 24px |

## 视觉规范

- 中文：Noto Sans SC（fonts.loli.net CDN）
- 等宽：Source Code Pro
- 所有 font-size 必须使用偶数 px 值
- 玻璃卡片圆角 28px，面板圆角 20px
- 进度条高度 6px，圆角 3px

## 技术说明

- 纯前端，无外部依赖（除 Google Fonts 国内镜像 CDN）
- 数据存储：localStorage
- 实时更新：requestAnimationFrame 循环
- 番茄钟状态机：IIFE 闭包模式（不使用 class/this）
- 尊重 `prefers-reduced-motion: reduce` 可访问性设置
- JS 使用 IIFE 闭包模式，不使用 class / this
