## 免费模型配置清单与依赖拓扑

免费模型配置按 IPO 执行流程组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：MF1 → MF2 → MF3 → MF4 → MF5

---

### MF1 需求分析域

| ID | 任务类型 | 说明 | 依赖 | 元操作映射 |
|----|---------|------|------|-----------|
| MF1-01 | 模型类型识别 | 识别用户需要的模型类型（文本生成/图像生成/视频生成等） | 无（入口） | S |
| MF1-02 | 使用场景分析 | 分析用户的使用场景和需求 | MF1-01 | S→C |
| MF1-03 | 需求提取 | 从用户需求中提取关键信息 | MF1-02 | S |

---

### MF2 模型选择域

| ID | 任务类型 | 说明 | 依赖 | 元操作映射 |
|----|---------|------|------|-----------|
| MF2-01 | 平台选择 | 根据需求选择合适的平台（Agnes AI、智谱、商汤等） | MF1-01 | C |
| MF2-02 | 能力匹配 | 匹配模型能力与用户需求 | MF1-02, MF2-01 | C |
| MF2-03 | 配置生成 | 生成相应的配置信息 | MF2-02 | C→A |
| MF2-04 | 参数设置 | 设置配置参数 | MF2-03 | C |

---

| MF3-04 | WorkBuddy 模板 | 生成 WorkBuddy 配置模板 | MF3-03 | A |
| MF3-05 | API 配置规范 | 配置 API 规范参数 | MF3-03 | A |

### 依赖拓扑摘要

**需求分析链路**: MF1-01 → MF1-02 → MF1-03
**模型选择链路**: MF2-01 → MF2-02 → MF2-03 → MF2-04
**配置生成链路**: MF3-01 → MF3-02 → MF3-03
**配置应用链路**: MF4-01 → MF4-02 → MF4-03
**使用指导链路**: MF5-01 → MF5-02 → MF5-03
**多模态生成链路**: MF6-01 → MF6-02 → MF6-03 → MF6-04 → MF6-05 → MF6-06

**完整处理链路**:
MF1-01 → MF1-02 → MF1-03 → MF2-01 → MF2-02 → MF2-03 → MF2-04 → MF3-01 → MF3-02 → MF3-03 → MF4-01 → MF4-02 → MF4-03 → MF5-01 → MF5-02 → MF5-03 → MF6-01 → MF6-02 → MF6-03 → MF6-04 → MF6-05 → MF6-06

**守护链路**: MF4-03 贯穿 MF4 全程（配置维护）

更多组合根据具体任务动态推导。
