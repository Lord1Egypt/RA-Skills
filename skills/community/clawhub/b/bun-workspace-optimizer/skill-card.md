## Use Case: <br>
Developers and engineering teams use this skill to audit Bun workspace configuration, dependency layout, build scripts, test setup, CI, Docker usage, and migration paths from npm, Yarn, or pnpm. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Suggested dependency, lockfile, CI, Docker, or build-tool changes could affect project behavior. <br>
Mitigation: Review proposed changes in version control and test them before merging or deployment. <br>
Risk: Performance estimates may not match the user's project, hardware, cache state, or CI environment. <br>
Mitigation: Benchmark install, build, and test changes in the target workspace before relying on speedup claims. <br>
