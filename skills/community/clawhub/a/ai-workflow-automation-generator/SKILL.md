## What This Skill Does

Generates a complete AI automation playbook for any business — identifying where AI can replace manual work, which tools to use, how to implement them, and what the ROI looks like. Outputs a structured roadmap any business owner or ops manager can execute without hiring a consultant.

## The Problem

Most business owners know AI can save them time and money. They just don't know where to start:
- Which workflows are actually automatable?
- Which AI tools are worth paying for?
- How do you integrate AI without breaking what already works?
- What's the actual ROI and time-to-value?

Hiring an AI consultant costs $150–$400/hr. Taking online courses takes weeks. This skill generates a custom playbook in minutes.

## The 4 Prompts

### Prompt 1: Workflow Audit & AI Opportunity Map (FREE TIER)
**Input:** Business type, team size, main revenue activities, biggest time drains  
**Output:**
- List of all core workflows with current time cost per week
- AI automation score (1–10) for each workflow
- Top 5 highest-ROI automation opportunities ranked
- Estimated time savings per week if automated
- Quick wins (under 2 hours to implement) vs. strategic wins (2–4 weeks)

### Prompt 2: AI Tool Stack Recommendation
**Input:** Automation opportunities from Prompt 1, budget range, tech comfort level  
**Output:**
- Recommended AI tool for each automation opportunity
- Tool comparison (price, ease of use, integration requirements)
- Total monthly tool cost estimate
- Implementation sequence (what to set up first)
- Tools to avoid (overpriced or overcomplicated for the use case)
- Free alternatives where available

### Prompt 3: SOP with AI Integration Points
**Input:** Specific workflow to automate, current manual process steps  
**Output:**
- Step-by-step SOP rewritten with AI touchpoints
- Exact prompts to use in each AI step
- Human checkpoints (where human review is required)
- Error handling: what to do when AI output is off
- Time-per-step breakdown (before vs. after AI)
- Training notes for team members

### Prompt 4: Implementation Roadmap & ROI Projection
**Input:** Full automation plan from Prompts 1–3, hourly labor cost  
**Output:**
- 90-day implementation calendar (Week 1–2: quick wins, Week 3–8: core automations, Week 9–12: advanced)
- Total setup time estimate per automation
- Monthly time saved (in hours)
- Monthly dollar savings (at provided labor rate)
- Payback period calculation
- KPIs to track automation effectiveness
- 30/60/90-day check-in questions

## Example: Desert Star Hospitality Group (Las Vegas)

**Business:** 3-property boutique hotel group, 45 staff, $4.2M annual revenue  
**Biggest time drains:** Guest communications (12 hrs/week), scheduling (8 hrs/week), review responses (4 hrs/week), event inquiries (6 hrs/week)

**Prompt 1 Output (excerpt):**
- Guest pre-arrival emails: AI Score 9/10 — 10 hrs/week → 1 hr/week with automation
- Review responses: AI Score 10/10 — 4 hrs/week → 20 min/week
- Staff scheduling: AI Score 6/10 — 8 hrs/week → 4 hrs/week
- Event inquiry responses: AI Score 8/10 — 6 hrs/week → 1 hr/week
- **Total automatable: ~28 hrs/week across team**

**Prompt 2 Output (excerpt):**
- Guest comms: Claude API + existing PMS integration ($0–29/month)
- Review responses: Yelp/Google API + Claude ($39/month)
- Scheduling: Sling AI tier ($70/month, replaces $150/month manual tool)
- Event inquiries: Claude + Typeform ($29/month)
- **Total tool cost: $138/month**

**Prompt 3 Output (excerpt — Review Response SOP):**
1. Pull new reviews daily at 9AM via API
2. AI generates draft response (tone: warm, professional, on-brand)
3. Prompt used: "You are the guest relations manager at Desert Star Hotels. Write a 3-sentence response to this [star rating] review that thanks the guest by name, acknowledges their specific feedback, and invites them to return. Never be defensive. Review: [review text]"
4. Human reviews draft — approve or edit (target: 2 min per review)
5. Post response via API
6. Log to CRM for pattern tracking

**Prompt 4 Output (excerpt):**
- Week 1–2: Review response automation (3 hrs setup, ROI in Week 3)
- Week 3–4: Guest pre-arrival email sequence (6 hrs setup)
- Week 5–8: Event inquiry automation (8 hrs setup)
- Week 9–12: Scheduling optimization (12 hrs setup)
- **Total monthly savings: 110 hrs/month × $28/hr avg labor = $3,080/month saved**
- **Total tool cost: $138/month**
- **Net monthly benefit: $2,942/month**
- **Payback on $29 skill purchase: Day 1**
