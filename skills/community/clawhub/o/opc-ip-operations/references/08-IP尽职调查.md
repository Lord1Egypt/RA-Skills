# IP尽职调查

## 概述

本文件定义知识产权尽职调查的量化评估体系，包含**四维20项指标评分卡**。所有指标均给出量化评分标准，权重来源于国知局指标体系和学术研究。

> **数据实时验证要求**：尽职调查前必须通过CNIPA pss-system、WIPO PATENTSCOPE、USPTO API等渠道实时验证专利法律状态和权属信息。

---

## 一、评分卡总览

### 四维权重分配

| 维度 | 权重 | 包含指标数 | 最高分 | 来源依据 |
|------|------|------------|--------|----------|
| **权属清晰度** | 30% | 5项 | 50分 | 国知局法律维度+AHP层次分析 |
| **有效性评估** | 25% | 5项 | 50分 | 国知局法律维度+行业实践 |
| **侵权风险** | 25% | 5项 | 50分 | 国知局法律维度+诉讼统计 |
| **价值评估** | 20% | 5项 | 50分 | 国知局经济维度+Park Y研究 |

### 综合评分计算

```
综合评分 = 权属清晰度得分 + 有效性评估得分 + 侵权风险得分 + 价值评估得分

其中：
权属清晰度得分 = Σ(指标i得分 × 指标i权重) / Σ(指标i权重) × 50
其他维度同理
```

### 风险等级划分

| 等级 | 综合评分 | 建议 | 说明 |
|------|----------|------|------|
| A级 | ≥80分 | 优先推进 | 可直接进入成果转化 |
| B级 | 60-79分 | 积极推进 | 需补充1-2项尽调 |
| C级 | 40-59分 | 补强后推进 | 需解决关键风险点 |
| D级 | <40分 | 暂缓 | 建议完善IP布局后再评估 |

## 二、维度1：权属清晰度（权重30%）

### 指标1.1 权属链条完整性（权重8%）

- **定义**：专利权属从申请到当前的完整性和连续性
- **评估方法**：
  1. 核查原始申请文件（申请人/发明人）
  2. 追踪所有转让记录
  3. 验证转让登记的完整性
  4. 检查是否存在共有协议
- **数据来源**：[CNIPA pss-system] | [查询时间] / [USPTO Assignment API] | [查询时间]
- **评分标准**（1-10分）：

| 分数 | 条件 | 说明 |
|------|------|------|
| 10分 | 权属链条完整，无任何断层，有完整转让登记 | 完全清晰 |
| 8分 | 权属链条基本完整，有1次转让但登记完整 | 基本清晰 |
| 6分 | 权属链条有轻微不清晰，但可追溯 | 基本可追溯 |
| 4分 | 权属链条存在不清晰，需补充证明 | 需补充 |
| 2分 | 权属链条断裂，存在权属争议 | 存在争议 |
| 1分 | 权属完全不清，存在重大法律障碍 | 无法确认 |

### 指标1.2 发明人身份确认（权重5%）

- **定义**：发明人身份的合法性和确认程度
- **评估方法**：
  1. 核查发明人声明文件
  2. 验证发明人与申请人的劳动合同关系
  3. 检查是否存在职务发明争议
  4. 确认是否存在共同发明人
- **数据来源**：[CNIPA pss-system] | [查询时间]
- **评分标准**（1-10分）：

| 分数 | 条件 |
|------|------|
| 10分 | 发明人身份明确，无任何争议，劳动关系清晰 |
| 8分 | 发明人身份明确，有1-2项小瑕疵但不影响权属 |
| 6分 | 发明人身份基本明确，有一定不确定性 |
| 4分 | 发明人身份存在争议或不确定 |
| 2分 | 发明人身份存在重大争议 |
| 1分 | 发明人身份完全无法确认 |

### 指标1.3 职务发明认定（权重5%）

- **定义**：专利是否属于职务发明，以及权属约定的清晰程度
- **评估方法**：
  1. 核查发明人与单位的劳动关系
  2. 分析发明内容与工作任务的关联性
  3. 检查劳动合同/协议中的IP约定
  4. 评估职务发明补偿情况
- **数据来源**：[CNIPA pss-system] | [查询时间] / [劳动合同/协议]
- **评分标准**（1-10分）：

| 分数 | 条件 |
|------|------|
| 10分 | 明确属于个人发明，或职务发明归属已清晰约定 |
| 8分 | 属于职务发明，但单位已明确放弃权属 |
| 6分 | 属于职务发明，权属约定不明确但无争议 |
| 4分 | 属于职务发明，权属约定存在争议 |
| 2分 | 职务发明认定存在重大争议 |
| 1分 | 无法判断是否属于职务发明 |

### 指标1.4 共有协议完整性（权重5%）

- **定义**：共有专利权利行使协议的完整程度
- **评估方法**：
  1. 核查是否存在共有协议
  2. 评估协议内容的完整性
  3. 检查收益分配约定
  4. 验证处置权约定
- **适用条件**：仅对共有专利评分，独家持有则默认10分
- **数据来源**：[共有协议文件] / [CNIPA pss-system]
- **评分标准**（1-10分）：

| 分数 | 条件 |
|------|------|
| 10分 | 无共有（独家持有），或共有协议非常完整 |
| 8分 | 共有协议完整，内容基本齐全 |
| 6分 | 共有协议存在，但内容有遗漏 |
| 4分 | 共有协议不完整，关键条款缺失 |
| 2分 | 存在共有但无协议 |
| 1分 | 共有协议存在重大争议 |

### 指标1.5 质押/许可负担（权重7%）

- **定义**：专利上是否存在质押、许可等他项权利负担
- **评估方法**：
  1. 检索CNIPA质押登记
  2. 核查许可备案记录
  3. 评估负担对交易的影响
  4. 计算清除负担的成本
- **数据来源**：[CNIPA pss-system] | [查询时间]
- **评分标准**（1-10分）：

| 分数 | 条件 |
|------|------|
| 10分 | 无任何质押或许可负担 |
| 8分 | 有非独占许可，但不影响交易 |
| 6分 | 有独占许可，但许可方同意配合转让 |
| 4分 | 有质押登记，但可解除 |
| 2分 | 有独占许可，解除难度大 |
| 1分 | 存在多重负担，难以清除 |

## 三、维度2：有效性评估（权重25%）

### 指标2.1 授权状态（权重5%）

- **定义**：专利当前的法律状态
- **评估方法**：通过CNIPA pss-system实时查询
- **数据来源**：[CNIPA pss-system] | [查询时间]
- **评分标准**（1-10分）：

| 分数 | 状态 | 说明 |
|------|------|------|
| 10分 | 已授权 | 发明专利已授权且有效 |
| 8分 | 已授权 | 实用新型/外观设计已授权且有效 |
| 6分 | 实质审查中 | 发明专利已进入实审，授权概率高 |
| 4分 | 实质审查中 | 发明专利已进入实审，授权概率不确定 |
| 2分 | 初审阶段 | 尚未进入实审 |
| 0分 | 已驳回/无效/撤回 | 法律程序终止 |

### 指标2.2 缴费状态（权重5%）

- **定义**：专利年费等费用的缴纳情况
- **评估方法**：
  1. 查询CNIPA缴费记录
  2. 确认下一缴费日
  3. 计算欠缴费金额
  4. 评估滞纳金情况
- **数据来源**：[CNIPA pss-system] | [查询时间]
- **评分标准**（1-10分）：

| 分数 | 条件 |
|------|------|
| 10分 | 全部费用已缴清，无任何欠缴 |
| 8分 | 正常按时缴费，下一期未到期 |
| 6分 | 有轻微逾期但已补缴，无滞纳金 |
| 4分 | 有逾期但已补缴，产生滞纳金 |
| 2分 | 存在欠缴，可能面临权利终止 |
| 1分 | 已因欠缴而权利终止 |

### 指标2.3 无效宣告风险（权重5%）

- **定义**：专利被无效宣告的成功概率
- **评估方法**：
  1. 分析权利要求的稳定性
  2. 检索现有技术的新颖性/创造性
  3. 评估说明书充分性问题
  4. 参考专利无效宣告统计
- **数据来源**：[CNIPA pss-system] | [WIPO PATENTSCOPE] | [查询时间]
- **评分标准**（1-10分，10=极低风险）：

| 分数 | 无效率 | 说明 |
|------|--------|------|
| 10分 | <10% | 权利要求稳定，难以无效 |
| 8分 | 10%-20% | 稳定性较高 |
| 6分 | 20%-30% | 稳定性一般 |
| 4分 | 30%-50% | 存在一定风险 |
| 2分 | 50%-70% | 风险较高 |
| 1分 | >70% | 极易被无效 |

### 指标2.4 保护范围宽度（权重5%）

- **定义**：权利要求所覆盖的技术方案范围
- **评估方法**：
  1. 分析独立权利要求的覆盖范围
  2. 评估等同原则的适用空间
  3. 检查是否存在宽范围的权利要求
  4. 分析与审查档案的一致性
- **数据来源**：[CNIPA pss-system] | [审查档案]
- **评分标准**（1-10分）：

| 分数 | 条件 |
|------|------|
| 10分 | 保护范围宽，权利要求层次合理 |
| 8分 | 保护范围较宽，有多个层次保护 |
| 6分 | 保护范围适中 |
| 4分 | 保护范围较窄 |
| 2分 | 保护范围很窄，易被规避 |

### 指标2.5 等同原则适用性（权重5%）

- **定义**：等同原则对侵权认定的支持程度
- **评估方法**：
  1. 分析权利要求与被诉产品的技术特征对比
  2. 评估是否属于本领域普通技术人员不需要创造劳动即可联想到的特征
  3. 检查是否存在禁止反悔情形
  4. 评估技术效果是否相同或相似
- **数据来源**：[CNIPA pss-system] | [专利审查档案]
- **评分标准**（1-10分）：

| 分数 | 条件 |
|------|------|
| 10分 | 等同特征明显，适用等同原则概率高 |
| 8分 | 大部分特征等同，适用可能性较大 |
| 6分 | 部分特征等同 |
| 4分 | 等同特征不明显 |
| 2分 | 几乎无等同特征 |

## 四、维度3：侵权风险（权重25%）

### 指标3.1 FTO自由实施分析（权重7%）

- **定义**：专利技术自由实施的可能性，即不侵犯他人专利权的程度
- **评估方法**：
  1. 检索相关技术领域的专利布局
  2. 进行权利要求分解和技术特征比对
  3. 评估规避设计可能性
  4. 分析专利稳定性
- **数据来源**：[CNIPA pss-system] | [智慧芽] | [WIPO PATENTSCOPE] | [查询时间]
- **评分标准**（1-10分，10=完全自由）：

| 分数 | 条件 |
|------|------|
| 10分 | 无他人有效专利覆盖，可自由实施 |
| 8分 | 有部分覆盖，但可通过规避设计避免 |
| 6分 | 有较多覆盖，但整体风险可控 |
| 4分 | 存在重大专利障碍，需要许可 |
| 2分 | 存在难以规避的核心专利 |
| 1分 | 存在无法规避的基础专利 |

### 指标3.2 侵权概率（权重6%）

- **定义**：专利被侵权的概率
- **评估方法**：
  1. 分析市场覆盖度和竞争对手
  2. 评估专利技术的可替代性
  3. 统计该领域专利侵权诉讼案例
  4. 调研行业侵权普遍性
- **数据来源**：[CNIPA pss-system] | [行业报告] | [诉讼数据库] | [查询时间]
- **评分标准**（1-10分，10=0%侵权概率）：

| 分数 | 侵权概率 | 说明 |
|------|----------|------|
| 10分 | 0%-5% | 几乎无侵权风险 |
| 8分 | 5%-15% | 侵权风险很低 |
| 6分 | 15%-30% | 存在一定侵权风险 |
| 4分 | 30%-50% | 侵权风险较高 |
| 2分 | 50%-70% | 侵权风险很高 |
| 1分 | >70% | 几乎必然被侵权 |

### 指标3.3 潜在侵权方数量（权重4%）

- **定义**：可能构成侵权的潜在主体数量
- **评估方法**：
  1. 识别使用相关技术的企业
  2. 评估技术在该行业的普及程度
  3. 分析潜在侵权方的规模
  4. 统计市场主要参与者
- **数据来源**：[行业报告] | [市场调研] | [CNIPA申请数据]
- **评分标准**（1-10分，10=潜在侵权方少=保护价值高）：

| 分数 | 潜在侵权方 | 说明 |
|------|------------|------|
| 10分 | 1-2家 | 市场集中度高 |
| 8分 | 3-5家 | 市场相对集中 |
| 6分 | 6-10家 | 市场适度分散 |
| 4分 | 11-20家 | 市场分散 |
| 2分 | >20家 | 市场高度分散 |

### 指标3.4 规避设计难度（权重4%）

- **定义**：竞争对手规避专利保护范围的难度
- **评估方法**：
  1. 分析权利要求的技术特征
  2. 评估是否存在替代技术路线
  3. 检查技术原理是否容易被绕过
  4. 分析产品设计自由度
- **数据来源**：[专利文件分析] | [技术调研]
- **评分标准**（1-10分，10=难以规避=保护强）：

| 分数 | 难度 | 说明 |
|------|------|------|
| 10分 | 极高 | 几乎无法规避 |
| 8分 | 较高 | 规避难度大 |
| 6分 | 中等 | 需要一定研发投入 |
| 4分 | 较低 | 较容易规避 |
| 2分 | 极低 | 很容易规避 |

### 指标3.5 诉讼成本预估（权重4%）

- **定义**：维权诉讼所需的成本和资源
- **评估方法**：
  1. 评估侵权证据获取难度
  2. 分析诉讼周期
  3. 估算律师费和鉴定费
  4. 评估执行难度
- **数据来源**：[行业统计] | [律所报价] | [历史案例]
- **评分标准**（1-10分，10=低成本）：

| 分数 | 成本预估 | 说明 |
|------|----------|------|
| 10分 | <5万 | 维权成本极低 |
| 8分 | 5-20万 | 维权成本较低 |
| 6分 | 20-50万 | 维权成本适中 |
| 4分 | 50-100万 | 维权成本较高 |
| 2分 | 100-200万 | 维权成本很高 |
| 1分 | >200万 | 维权成本极高 |

## 五、维度4：价值评估（权重20%）

### 指标4.1 技术成熟度TRL（权重5%）

- **定义**：技术从概念到实际应用的成熟程度
- **评估方法**：采用NASA TRL标准（9级）
- **数据来源**：[技术评估报告] / [CNIPA.AI技术评估]
- **评分标准**：

| TRL等级 | 定义 | 映射评分 |
|---------|------|----------|
| TRL 9 | 实际系统通过任务运行验证 | 10分 |
| TRL 8 | 系统完整并通过鉴定 | 9分 |
| TRL 7 | 系统原型在运行环境中演示 | 8分 |
| TRL 6 | 系统/子系统模型在相关环境中演示 | 7分 |
| TRL 5 | 组件/模型在相关环境中验证 | 6分 |
| TRL 4 | 组件/模型在实验室环境中验证 | 5分 |
| TRL 3 | 关键功能分析或应用预测性分析完成 | 4分 |
| TRL 2 | 形成技术概念或应用预测 | 3分 |
| TRL 1 | 观察到基本原理 | 2分 |

### 指标4.2 市场覆盖度（权重5%）

- **定义**：专利技术的市场应用范围和普及程度
- **评估方法**：
  1. 分析专利家族覆盖的国家/地区
  2. 评估目标市场规模
  3. 统计已实施的许可协议
  4. 分析商业化案例
- **数据来源**：[CNIPA pss-system] | [行业报告] | [WIPO PATENTSCOPE]
- **评分标准**（1-10分）：

| 分数 | 市场覆盖 | 说明 |
|------|----------|------|
| 10分 | 全球覆盖 | 主要市场均有专利保护 |
| 8分 | 多国覆盖 | 覆盖3个以上主要市场 |
| 6分 | 区域覆盖 | 覆盖特定区域市场 |
| 4分 | 仅国内 | 仅在中国有保护 |
| 2分 | 初审阶段 | 尚未获得授权 |

### 指标4.3 替代技术数量（权重4%）

- **定义**：可替代本专利技术的替代方案数量
- **评估方法**：
  1. 检索相关技术领域的替代技术
  2. 评估替代技术的成熟度
  3. 分析替代技术的成本效益
  4. 评估替代技术的发展趋势
- **数据来源**：[CNIPA pss-system] | [行业报告] | [技术调研]
- **评分标准**（1-10分，10=无替代=价值高）：

| 分数 | 替代技术 | 说明 |
|------|----------|------|
| 10分 | 无替代技术 | 不可替代 |
| 8分 | 1-2种替代 | 替代性较弱 |
| 6分 | 3-5种替代 | 存在一定替代 |
| 4分 | 6-10种替代 | 替代性较强 |
| 2分 | >10种替代 | 替代性很强 |

### 指标4.4 剩余保护期（权重3%）

- **定义**：专利剩余的法律保护期限
- **评估方法**：
  1. 计算申请日到届满日的剩余时间
  2. 评估是否可申请延期（如药品专利）
  3. 分析剩余期限的商业价值
- **数据来源**：[CNIPA pss-system] | [查询时间]
- **评分标准**（按年数线性映射，10分制）：

| 剩余保护期 | 评分 |
|------------|------|
| >15年 | 10分 |
| 12-15年 | 9分 |
| 10-12年 | 8分 |
