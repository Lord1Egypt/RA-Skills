# D04 职业领域任务目录

## 领域定义

职业领域(D04)关注个体职业发展、工作执行和职业成功的全过程。核心驱动需求：N3归属（职业身份）、N4自我实现（职业成就）。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人职业规划 | 企业人力资源管理 |
| 求职策略与执行 | 招聘流程管理 |
| 工作执行与效率 | 组织运营管理 |
| 绩效发展与晋升 | 企业绩效考核体系 |
| 创业管理与融资 | 风险投资管理 |
| 副业与自由职业 | 平台运营管理 |
| 职场关系处理 | 企业政治斗争 |
