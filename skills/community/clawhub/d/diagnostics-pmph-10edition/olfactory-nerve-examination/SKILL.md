# 嗅神经功能检查

## 适用条件
- 患者意识清楚且能配合检查
- 鼻孔通畅
- 无鼻黏膜病变

## 禁忌与限制
- 不可用于昏迷或无法配合的患者
- **严禁使用**酒精、氨水、甲醛等可刺激三叉神经末梢的挥发性物质

## 操作流程
1. 检查前确认患者鼻孔通畅，并排除鼻部器质性病变。
2. 嘱患者闭目，避免视觉干扰。
3. 压住一侧鼻孔，将一种患者熟悉的、**无刺激性气味**的物品（如杏仁、松节油、肉桂油、牙膏、香烟或香皂）置于另一侧鼻孔下方。
4. 要求患者准确辨别所嗅到的气味。
5. 更换对侧鼻孔，重复上述步骤，进行双侧嗅觉对比。
6. 记录结果：正常 / 减退 / 丧失。
7. 结合鼻部状况判断是否为中枢性嗅觉障碍——若排除鼻部病变后出现单侧或双侧嗅觉丧失，提示同侧嗅神经受损，常见于嗅沟占位性病变压迫嗅球或嗅束。

## 注意事项
- 所用测试物质必须为患者熟悉且无三叉神经刺激性的气味物。
- 检查过程需双侧分别测试并对比，避免误判。
- 若患者无法识别常见气味，应首先排除鼻源性因素。
