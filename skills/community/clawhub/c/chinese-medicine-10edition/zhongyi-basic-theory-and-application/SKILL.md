# Zhongyi Basic Theory and Application

## When to Use
- 用户询问健康或疾病问题，并明确希望从中医角度理解
- 需要解释中医术语（如“证候”“正气”）或诊疗逻辑
- 涉及养生、预防或非药物疗法（如针灸、食疗）的建议场景
- 西医诊断已知，但需补充中医辨证分型思路

## Execution Steps
1. **确认整体观念前提**：将人体视为脏腑、经络、气血津液相互关联的有机整体，避免孤立看待症状。
2. **应用核心理论**：结合阴阳学说、五行学说、藏象学说、经络学说等分析病因病机。
3. **执行辨证论治**：综合症状、体征、舌象、脉象等信息，辨别具体证候类型（如“风寒感冒”“肝郁气滞”），而非仅对应西医病名。
4. **选择干预手段**：根据证型选用中药、针灸、推拿、拔罐、食疗或导引（如太极拳）等方法，目标是调和阴阳、扶助正气。
5. **贯彻“治未病”原则**：在无病或病初阶段，强调顺应四时、调摄情志、饮食有节、起居有常等预防措施。

> 注意：本技能不适用于需分子机制或循证医学证据支持的现代生物医学解释场景。
