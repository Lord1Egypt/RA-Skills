# CHARGE综合征的诊断标准应用

## 何时使用
- 患儿存在两个及以上系统发育异常
- 已完成眼科、耳鼻喉、心脏、神经发育等评估
- 需决定是否进行CHD7基因检测或解释阴性结果

## 执行步骤
1. **评估主要标准（至少满足2项）**：
   - 眼部缺损（虹膜/视网膜/脉络膜/视盘缺损）
   - 后鼻孔闭锁或唇腭裂
   - 耳部畸形（外/中/内耳结构异常或听力障碍）
   - 致病性CHD7基因变异（分子检测证实）

2. **评估次要标准（至少满足1项）**：
   - 脑神经功能障碍（如感音性耳聋）
   - 吞咽/喂养困难或食管畸形
   - 心脏畸形（圆锥动脉干异常等）
   - 大脑结构异常（嗅脑、胼胝体、小脑等）
   - 发育迟缓或孤独症样行为
   - 下丘脑-垂体功能不全或性腺异常
   - 肾脏、骨骼或肢体发育不良

3. **做出诊断判断**：
   - 若 ≥2 主要 + ≥1 次要 → 符合典型CHARGE综合征
   - 若临床高度怀疑但不符合标准 → 仍建议CHD7基因检测

> 注意：基因检测阴性不能完全排除诊断；临床表现高度异质，需多学科综合评估。
