# 列表样式规范

## 列表类型

### 无序列表
- 符号：•
- 适用：并列项
- 缩进：0.74cm

### 有序列表
1. 第一点
2. 第二点
- 符号：1. 2. 3.
- 适用：顺序/优先级
- 缩进：0.74cm

### 检查清单
- [ ] 待办项
- [x] 已完成
- 适用：任务列表

## 样式定义

| 属性 | 值 |
|------|-----|
| 字体 | 宋体 |
| 字号 | 12pt |
| 行距 | 1.5 倍 |
| 项间距 | 6pt |

## 内容规范

### 避免
