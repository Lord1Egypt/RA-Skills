## 四、七项检查清单

每次输出理赔报告前，必须检查以下7项：

```markdown
## ✅ 七项检查清单

- [ ] 1. 被保险人身份信息是否完整（姓名/身份证/保单号）
- [ ] 2. 票据识别关键字段是否齐全（金额/日期/医院/诊断）
- [ ] 3. 责任免除条款是否已核查（等待期/既往症/医院级别）
- [ ] 4. 赔付计算公式是否正确（免赔额/赔付比例/给付限额）
- [ ] 5. 反欺诈风险评估是否已执行（综合风险评分）
- [ ] 6. 金额大小写是否一致（阿拉伯数字与中文大写）
- [ ] 7. 法律合规条款是否已标注（异议期/客服电话/监管依据）
```

## 七、FAQ回复模板

### Q1：理赔需要多长时间？

> 根据《保险法》第23条规定，保险人收到理赔申请后，应当及时作出核定；情形复杂的，应当在30日内作出核定。
> 本公司承诺：简单案件（3000元以下）当日完成；普通案件（3000-20000元）3个工作日内完成；复杂案件30日内完成。

### Q2：为什么我的理赔被拒付了？

> 拒付原因通常包括：①等待期内出险；②既往症；③非保障范围；④医院不符合要求；⑤免赔额未达。
> 详细拒付原因请查看《理赔拒付通知书》，如有异议可申请复议。

### Q3：理赔款什么时候到账？

> 理赔款在理赔决定作出后10日内支付（法律规定）。本公司实际到账时间：自动理赔案件实时到账；人工审核案件3个工作日内到账。

### Q4：我对理赔决定不满意怎么办？

> 您有以下途径：
> 1. 向本公司申请复议（60日内）
> 2. 向国家金融监督管理总局投诉
> 3. 依法提起诉讼

### Q5：医疗发票OCR识别不准确怎么办？

> 您可以提供更清晰的发票照片重新识别，或选择人工审核通道。OCR识别结果仅供参考，最终以人工审核为准。

## 八、输出格式规范总结

| 格式类型 | 适用场景 | 输出方式 |
|---------|---------|---------|
| 完整报告 | 综合理赔分析 | Markdown 表格 + 分段 |
| 核定通知书 | 赔付决定 | 正式文书格式（含印章位置） |
| 拒付通知书 | 拒付决定 | 正式文书格式 + 法律条款引用 |
| 补充材料通知 | 材料不齐 | 清单格式 + 补充方式说明 |
| 欺诈风险报告 | 高风险案件 | 评分表格 + 风险因子列表 |
| 计算书 | 车险/财产险 | 定损明细表格 + 计算公式 |
