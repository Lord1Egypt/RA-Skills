All notable changes to this skill will be documented in this file.

## [2.9.0] - 2026-05-07

### ✨ Added
- 支持 CAN FD 完整功能（数据段 2Mbps，采样点 0.800）
- 新增 UDS 报告完整覆盖（ISO 15765-2 边界测试、NRC 优先级、功能寻址抑制等）

### 🐛 Fixed
- 修复初版流程中 CAN 初始化失败的问题
- 修复权限错误导致报告无法保存的问题
- 完善确认表格模板和展示逻辑

### 📚 Docs
- 优化确认流程，补充示例表格模板
- 补充 UDS 报告全面说明（包含失败用例分析指南）
