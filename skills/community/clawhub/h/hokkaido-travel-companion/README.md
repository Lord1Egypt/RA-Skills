# Hokkaido Travel Companion

A Hokkaido-specific travel companion protocol for OpenClaw.

This is not a hardcoded travel database. It is a reusable method for collecting, verifying, storing, and applying travel knowledge for Sapporo/Otaru/Yoichi/Chitose trips.

It combines:
- a local travel vault as prior memory,
- live search for restaurant/hours/transport verification,
- GO/NOGO budget checks,
- route output with Japanese/Korean/English station names.

## Privacy note
This public package intentionally ships with empty vault folders. Add your own notes under `references/vault/` after installation.
