## Description: <br>
Generate diagrams from descriptions with Mermaid, PlantUML, or ASCII for architecture, flows, sequences, and data models. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ivangdavila](https://clawhub.ai/user/ivangdavila) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers, architects, and technical writers use this skill to turn system descriptions, workflows, API interactions, and data models into concise Mermaid, PlantUML, ASCII, or optional image-rendered diagrams. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Optional PNG or SVG rendering may run Mermaid CLI through npm. <br>
Mitigation: Review the generated command before running it, use a trusted environment, and pin package versions when supply-chain control matters. <br>
