## Description: <br>
Generates modern SaaS product landing pages in several styles, with deployable HTML, CSS, React, Vue, or Next.js output. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Sunshine-del-ux](https://clawhub.ai/user/Sunshine-del-ux) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers, founders, and marketing teams use this skill to generate deployable SaaS landing page starter code from a product name, description, and selected design options. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill creates files and directories in the selected output path. <br>
Mitigation: Run it in a clean project directory or explicitly set an output directory before using generated files. <br>
Risk: Generated pages may include product names, descriptions, and public CDN-loaded frontend libraries. <br>
Mitigation: Review generated HTML and framework files, sanitize untrusted text, and pin or self-host frontend dependencies before production deployment. <br>
