# 个人地图生成功能说明

## 功能概述

基于高德 `maps_schema_personal_map` API 创建个人地图，支持批量导入所有打卡点，生成高德可识别的个人地图，并输出二维码，用户扫码直达高德App查看全量打卡点。

## 核心特性

### 1. 批量点位导入

- 支持一次性导入多个POI点位
- 自动提取POI坐标、名称、地址等信息
- 支持所有八大场景的点位

### 2. 二维码生成

- 自动生成高德个人地图二维码
- 用户扫码直达高德App查看地图
- 支持一键导航到任意点位
- 可分享给好友或保存到相册

### 3. 智能分批处理

- 单次最多创建包含50个点位的地图
- 超过50个点位时自动分批创建
- 每批生成独立的二维码和链接
- 保持完整的点位信息

### 4. 坐标验证

- 自动验证经纬度有效性
- 过滤超出中国范围的无效坐标
- 确保坐标格式正确（经度,纬度）

## 技术实现

### API调用

```javascript
// 调用高德个人地图创建API
const result = await request('maps_schema_personal_map', {
  key: 'YOUR_AMAP_API_KEY',
  sceneType: '2',  // 仅创建资源点（打卡点位标记模式）
  points: JSON.stringify(points)
});
```

```javascript
// 验证坐标有效性
function isValidCoordinate(lng, lat) {
  return !isNaN(lng) && !isNaN(lat) &&
```

## 使用流程

### 1. 生成全场景地图

```
用户输入: 帮我生成北京市全能游玩地图

系统处理:
1. 解析意图 -> full_map
2. 获取北京市行政区划
3. 搜索八大场景POI
4. 数据加工（去重、排序、补全详情）
5. 统计总POI数量
6. 如果 > 50，分批创建地图
7. 如果 <= 50，单次创建地图
8. 格式化输出结果
```

### 2. 生成单场景地图

```
用户输入: 帮我做杭州西湖区溜娃地图

系统处理:
1. 解析意图 -> scene_map (parenting)
2. 获取杭州市行政区划
3. 搜索溜娃场景POI
4. 补全POI详情
5. 过滤和排序
6. 创建个人地图
7. 格式化输出结果
```

### 3. 生成旅行地图

```
用户输入: 开启旅行模式规划成都3日自驾游

系统处理:
1. 解析意图 -> travel
2. 获取成都市行政区划
3. 搜索所有场景POI
4. 获取天气和路况信息
5. 数据加工
6. 根据天气路况优化路线
7. 创建个人地图
8. 格式化输出结果
```

## 注意事项

1. **API密钥配置**
   - 需要配置环境变量 `AMAP_API_KEY`
   - 确保密钥有效且有足够调用额度

2. **点位限制**
   - 单次最多50个点位
   - 超过限制自动分批处理

3. **坐标格式**
   - 经纬度用逗号分隔
   - 确保坐标在中国范围内

4. **错误处理**
   - 网络错误自动重试1次
   - API错误返回友好提示

5. **性能优化**
   - 缓存相同点位集合的地图结果
   - 避免重复创建相同地图
