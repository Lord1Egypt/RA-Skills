# 失败处理路径

## 4种失败场景

### 1. API不可用
- **处理**：本地规则替代
- **降级策略**：使用内置知识库回答

### 2. 输入无效
- **处理**：返回报错 + 模板
- **提示**："输入格式不正确，请尝试..."

### 3. 输出校验失败
- **处理**：重试 → 降级
- **流程**：validate_output() → 重试1次 → 仍失败走Failure路径

### 4. 资本情商危机
- **处理**：Capital EQ指导 → 建议休息 → 专业帮助
- **触发**：检测到严重情绪危机信号

## 降级策略

所有失败场景均有明确的降级路径，确保用户始终获得有用回复。
