# IP申请操作手册

## 一、软件著作权申请操作手册

### 1.1 申请前准备

**账号注册与实名认证**

1. 访问：中国版权保护中心官网 https://www.ccopyright.com.cn/
2. 点击"用户注册" → 选择"机构用户"
3. 填写信息：机构名称、统一社会信用代码、联系人等
4. 实名认证：上传营业执照+经办人身份证
5. 审核周期：1-3个工作日

**材料预审清单**

| 材料 | 要求 | 检查项 |
|------|------|--------|
| 源代码 | .java/.py/.js等格式，≥3000行 | □前后各1500行<br>□注释完整<br>□无他人代码 |
| 用户手册 | PDF格式，≥20页 | □功能描述完整<br>□图文并茂<br>□操作步骤清晰 |
| 设计说明书 | PDF格式，≥20页 | □系统架构图<br>□模块说明<br>□流程图 |
| 承诺书 | 手写签字原件扫描 | □声明原创<br>□未使用AI生成<br>□经办人签字 |
| 经办人材料 | 身份证正反面+实名认证截图 | □在有效期<br>□清晰可辨 |

### 1.2 软著申请流程（2026年新规适配）

**步骤1：创建项目档案**

在本地整理以下文件夹结构：
```
查新检索Skill_软著申请/
├── 1_源代码/
│   ├── main.java（前1500行）
│   └── main.java（后1500行）
├── 2_用户手册.pdf
├── 3_设计说明书.pdf
├── 4_承诺书.pdf
└── 5_申请表.docx
```

**步骤2：在线填报**

1. 登录中国版权保护中心官网
2. 进入"软件登记" → "计算机软件著作权登记申请"
3. 填写信息：
   - 软件名称：[主体名]V[版本号]（如：查新检索Skill V1.0）
   - 分类号：参照《计算机软件分类表》
   - 版本号：V1.0
   - 开发完成日期：YYYY-MM-DD
   - 发表状态：已发表/未发表
   - 开发方式：独立开发/合作开发/委托开发

**步骤3：四重匹配校验（新规重点）**

2026年新规要求代码、文档、说明三方面内容相互匹配：
1. **代码注释**与**设计说明书**描述一致
2. **功能描述**与**用户手册**截图一致
3. **开发完成时间**与代码时间戳一致
4. **代码行数**与说明书描述一致

建议：先完成文档，再对应整理代码，确保三者一致。

**步骤4：承诺书签署（新规重点）**

下载中心模板，手写签署以下内容：
```
本人/本单位郑重承诺：
1. 本次申请登记的软件系独立开发完成，不存在抄袭、复制他人软件的情况；
2. 本次提交的材料中，未使用人工智能（以下简称AI）自动生成；
3. 本申请及材料中涉及的全部内容真实、合法、有效；
4. 如有虚假，愿承担相应法律责任。

承诺人：[签名]
身份证号：[身份证号]
日期：[手写日期]
```

⚠️ 注意：2026年新规要求手写签字，电子签名或打印签名无效。

**步骤5：材料上传**

1. 源代码：PDF格式，分别上传前1500行和后1500行
2. 文档：用户手册+设计说明书（建议合并为一个PDF）
3. 承诺书：手写签字后的扫描件
4. 经办人材料：身份证+实名认证截图

**步骤6：在线提交**

1. 检查所有材料完整性
2. 点击"确认提交"
3. 获得登记号和受理号
4. 等待审查（30个工作日）

**步骤7：补正处理（新规限制）**

⚠️ 2026年新规：补正机会仅有2次！

收到补正通知后：
1. 认真阅读补正意见
2. 一次性完成所有补正内容
3. 在规定时间内提交（通常10个工作日）
4. 避免重复提交造成浪费

### 1.3 常见问题处理

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 代码不足3000行 | 功能较简单 | 补充注释、空行、辅助类代码 |
| 文档与代码不匹配 | 多版本迭代 | 以最终版本为准，修改文档 |
| 补正理由模糊 | 审查员理解差异 | 仔细对照说明书逐项说明 |
| 承诺书格式错误 | 使用了旧模板 | 下载最新模板，重新签署 |

## 二、商标申请操作手册

### 2.1 申请前检索

**检索入口**：国家知识产权局商标局 中国商标网 https://wcjs.sbj.cnipa.gov.cn/

**检索步骤**：
1. 点击"商标近似查询"
2. 选择查询方式：汉字查询/拼音查询/图形查询
3. 输入查询内容：如"AI5S"或"5S"
4. 选择类别：如第42类
5. 查看检索结果，关注相同/近似商标

**判断标准**：
- 音、形、义任一近似即可能驳回
- 考虑公众认知程度
- 同类别+关联类别均需检索

### 2.2 申请材料准备

**以个人名义申请**：
- 身份证正反面扫描件
- 个体工商户营业执照（如有）
- 商标图样（jpg，指定尺寸）
- 委托书（如委托代理）

**以企业名义申请**：
- 营业执照副本扫描件（加盖公章）
- 商标图样（jpg，指定尺寸）
- 委托书（如委托代理）

**商标图样要求**：
- 尺寸：不大于10×10cm，不小于5×5cm
- 颜色：黑白或彩色（黑白保护范围更广）
- 格式：jpg/png，300dpi以上

### 2.3 类别选择策略

**核心类别（必选）**：
| 类别 | 内容 | 适用业务 |
|------|------|----------|
| 第42类 | 技术研究、软件开发、计算机编程 | 核心AI服务 |
| 第38类 | 通讯服务、信息传送 | 在线平台、即时通讯 |

**防御类别（建议）**：
| 类别 | 内容 | 适用场景 |
|------|------|----------|
| 第9类 | 科学仪器、软件 | App、工具类 |
| 第35类 | 广告、商业管理 | 营销推广 |
| 第41类 | 教育娱乐 | 培训课程 |
| 第45类 | 在线社交网络 | 社区服务 |

### 2.4 申请流程

**步骤1：用户注册**
1. 访问商标局网上服务系统
2. 注册账号并进行身份认证

**步骤2：填写申请信息**
- 商标名称/图样
- 类别选择（可一标多类）
- 商品/服务项目（参照尼斯分类）
- 申请人信息

**步骤3：缴纳费用**
- 官费：300元/类/件（电子申请）
- 纸质申请：270元/类/件
- 委托代理：另加代理费500-1500元/类

**步骤4：提交与受理**
1. 提交申请 → 获得申请号
2. 形式审查（1个月）→ 受理通知书
3. 实质审查（4-6个月）

**步骤5：审查结果**
- 通过 → 初审公告（3个月）
- 驳回 → 收到驳回通知书，可申请复审

**步骤6：注册公告与发证**
- 公告期满无异议 → 注册公告
- 发放注册证

**总周期**：约8-12个月

### 2.5 驳回复审处理

**收到驳回通知后**：
1. **查询驳回理由**：登录系统查看具体驳回原因
2. **评估复审必要性**：
   - 绝对理由（违反禁用条款）→ 复审难度大
   - 相对理由（与在先商标近似）→ 可通过设计改动或复审争取
3. **决定处理方式**：
   - 复审：收到通知15日内向商标评审委员会申请
   - 调整后重新申请：修改图样或名称，重新提交

## 三、专利申请操作手册

### 3.1 技术交底书撰写

**模板结构**：

```markdown
# 技术交底书

## 一、技术领域
本发明涉及[具体领域]，特别涉及[具体技术问题]。

## 二、背景技术
### 现有技术现状
[描述现有技术的解决方案]
### 存在不足
[指出现有技术存在的问题]

## 三、发明内容
### 发明目的
[本发明要解决的技术问题]
### 技术方案
[详细描述发明的技术方案，可结合附图]
### 有益效果
[描述发明带来的技术效果]

## 四、具体实施方式
### 实施例1
[具体实施例描述]
### 实施例2
[变形例描述]

## 五、关键技术点
1. [关键技术点1]
2. [关键技术点2]
3. [关键技术点3]
```

### 3.2 专利检索

**检索入口**：
- 国家知识产权局：https://patent.cnipa.gov.cn/
- 专利检索及分析系统：https://pss-system.cponline.cnipa.gov.cn/
- 智慧芽（商业）：https://www.zhihuiya.com/

**检索策略**：
1. 确定关键词：技术领域相关术语
2. 确定分类号：根据《国际专利分类表》确定IPC分类号
3. 组合检索：关键词 AND/OR 分类号
4. 扩展检索：同义词、上位概念、下位概念

**新颖性评估**：
- 如果检出1篇完全覆盖的技术方案 → 缺乏新颖性
- 如果检出多篇相关方案 → 评估创造性

### 3.3 申请类型选择

| 类型 | 保护对象 | 要求 | 周期 | 费用 |
|------|----------|------|------|------|
| 发明专利 | 产品/方法 | 新颖性+创造性+实用性 | 2-3年 | 3450-5600元 |
| 实用新型 | 产品结构 | 新颖性+实用性 | 6-8月 | 500-1000元 |
| 外观设计 | 产品外观 | 新颖性+显著性 | 4-6月 | 500-1000元 |

**AI相关发明的特殊考虑**：
- 算法本身：可能被认为属于智力规则，不授予专利权
- 技术方案：结合具体应用场景，可申请专利
- 建议：强调技术效果，而非算法本身

### 3.4 委托代理机构

**选择标准**：
1. 具有专利代理资质
2. 有AI/计算机领域代理经验
3. 代理费用透明
4. 服务响应及时

**代理流程**：
1. 签订委托合同
2. 提供技术交底书
3. 代理人撰写申请文件
4. 申请人审核确认
5. 提交申请
6. 答复审查意见
7. 获得授权

## 四、数据知识产权登记操作手册

### 4.1 登记条件

**数据要求**：
1. 依法依规获取
2. 经过必要数据处理（清洗、标注、结构化等）
3. 具有实质性加工和创新性贡献
4. 尚未进入公有领域
5. 尚未被他人登记

**地区限制**：
目前仅限试点地区：江苏(南京/苏州/无锡)、浙江(杭州/宁波)、广东(广州/深圳/东莞)

### 4.2 申请材料

| 材料 | 说明 |
|------|------|
| 数据知识产权登记表 | 在线填报，打印盖章 |
| 数据集说明文档 | 来源、规模、更新频率、处理过程 |
| 数据加工说明 | 清洗、标注、结构化方法 |
| 数据样本 | 脱敏后的数据样例 |
| 权属证明 | 数据获取合法性证明 |
| 其他 | 各地可能有差异 |

### 4.3 登记流程

**步骤1：确认资格**
- 确认所在地区是否属于试点范围
- 确认数据是否符合保护条件

**步骤2：准备材料**
- 整理数据集元信息
- 编写数据加工说明
- 准备数据样本

**步骤3：在线申请**
1. 访问当地知识产权保护中心官网
2. 注册账号并完成认证
3. 填写数据知识产权登记表
4. 上传材料

**步骤4：审查**
- 形式审查 + 实质审查
- 可能要求补充材料

**步骤5：登记发证**
- 通过审查后发放登记证书
- 可获得数据权益凭证

### 4.4 案例：IMA知识库数据IP登记

**数据集信息**：
- 名称：OPC运营官知识库数据集
- 规模：11个文档
- 内容：OPC运营相关知识、问答、案例

**登记建议**：
1. 整理数据集元信息文档
2. 编写数据清洗和结构化加工说明
3. 准备数据样本（脱敏后）
4. 向江苏/浙江/广东任一试点地区申请登记
