# 三段式输出模板（format-templates）

逐字套用骨架，只替换 `<尖括号>` 里的业务内容和 `{{占位符}}`。

---

## 一、场景介绍

```
一句话摸清目标公司，让拜访有的放矢。

它会帮你做什么：
- 智能采集公开情报：从官网、招投标、招聘等公开渠道抓取目标公司最新动态，覆盖业务、产品、人事、市场多维度信息。
- 生成销售策略建议：基于采集数据提炼客户痛点、合作切入点和可执行话术，让销售拜访更有针对性。
- 关联客户档案：采集完成后检查该公司是否已录入客户库，未添加时一键快速建档。

怎么用：
1. 点「安装场景」
2. 填目标公司和关注动态/查询重点
3. 需要时点「采集情报」，即可获取策略摘要与完整情报报告
```

---

## 二、安装提示词（带登录版）

```
## 任务
用户安装了「情报分析雷达」场景。请完成以下三步。

## 第一步：安装情报分析雷达 Skill
首先尝试从企业技能中心安装：
byted-skillhub-download intelligence-radar

- 如果安装成功，继续第二步。
- 如果安装失败，则尝试执行以下命令安装：
openclaw skills install intelligence-radar
  - 如果兜底安装成功，继续第二步。
  - 如果兜底安装仍然失败，告诉用户"情报分析雷达场景安装失败，请稍后重试"，并停止。

## 第二步：身份验证登录
使用 {{push_time}}（登录账号）和 {{push_channel}}（登录密码）调用登录接口验证并缓存 Token。
- 如果验证成功，继续第三步。
- 如果验证失败，告诉用户"账号或密码有误，请重新输入"，并等待用户重新输入。

## 第三步：确认并引导
安装并登录成功后，向用户输出以下内容：

---
✅ **情报分析雷达场景已安装成功！**

你的当前配置：
- 🧑‍💼 登录账号：{{push_time}}
- 🎯 目标公司：{{who}}
- 🔍 关注动态：{{what}}

**使用方式：**
在「我的场景」中点击「采集情报」，系统会采集目标公司的公开情报并生成销售策略摘要，同时提供完整情报报告的 H5 链接，支持拜访前快速摸底。

**你还可以：**
- 在「我的场景」点击「设置」，随时修改目标公司、关注动态
- 每次触发时也可以随时修改，适配不同需求
---

如果用户没有填写某项配置，对应行显示"未设置"。
```

## 三、触发提示词

```
情报雷达：
使用 intelligence-radar 这个 skill 为用户采集并分析目标公司情报。整理时重点关注以下内容：

- 目标公司：{{who}}
- 关注动态/查询重点：{{what}}
```

> 第一行说明与配置项列表之间必须空一行。用不到的占位符不写。
