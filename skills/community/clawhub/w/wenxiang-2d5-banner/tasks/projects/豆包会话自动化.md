## 📚 项目知识库

### 业务规则
- 2026-03-06: 豆包会话需归档到 doubao-sessions/ 文件夹，命名格式：YYYY-MM-DD (周*) - * worklog.md
- 2026-03-06: 每次豆包会话后生成专家点评 HTML，遵循 HTML-STANDARD.md

### 技术决策
- 2026-03-06: 选择 HTML 而非 Markdown - 支持 Mermaid 图表、专业排版、交互性
- 2026-03-06: 专家点评格式 - 科学杂志排版 + 小红书图标 + 金字塔结构
- 2026-03-08: 统一归档到 html-expert-reviews/ - 避免文件散落

### 关键指标
- 会话获取成功率：目标 >95%
- 专家点评生成时间：<30 秒
- 用户满意度：视觉设计评分 >8/10

### 风险与问题
- 2026-03-06: 豆包 API 可能变更 - 解决方案：保留多个脚本版本
- 2026-03-08: 会话历史不互通（飞书 vs Dashboard）- 解决方案：worklog.txt 作为统一日志

### 经验教训
- 2026-03-06: 文件 > 记忆 - 会话历史会丢失，文件永久保存
- 2026-03-08: 统一归档文件夹比散落根目录更易管理
