# 車位租約特別事項

## 台灣車位類型說明

| 車位類型 | 權狀 | 可獨立買賣 | 可個別出租 | 說明 |
|----------|:----:|:----------:|:----------:|------|
| 法定停車位 | ❌ 含在公設 | ❌ 須隨主建物 | ✅ 可出租 | 建物按法規應設之車位 |
| 增設停車位 | ✅ 有獨立權狀 | ✅ 可 | ✅ 可 | 建商額外增設之車位 |
| 獎勵停車位 | ✅ 有獨立權狀 | ✅ 可（有限制） | ✅ 可 | 政府獎勵增設，供公眾使用 |
| 機械車位 | 依類型 | 依類型 | ✅ 可 | 需注意保養責任 |

## 車位租約關鍵審查點

### 1. 車位標示
- [ ] 車位編號或號碼（需與圖面一致）
- [ ] 車位尺寸（長×寬×高）
  - 標準車位：2.5m×5.5m×1.8m（多數轎車適用）
  - 機械車位：需確認乘載限制及高度
- [ ] 車位位置平面圖（含車道動線）

### 2. 使用規範
- [ ] 使用時間：全日/特定時段
- [ ] 停車類型：汽車/機車
- [ ] 是否可停放大型休旅車（SUV/MPV）
- [ ] 是否可停放電動車（有無充電樁）
- [ ] 禁止堆放雜物

### 3. 管理費
- [ ] 管理費金額及由誰負擔
- [ ] 清潔費（如有額外）
- [ ] 機械車位保養費分擔

### 4. 充電設備（重要！）
- [ ] 是否可安裝電動車充電樁
- [ ] 安裝費用由誰負擔
- [ ] 用電費率計算方式
- [ ] 管委會充電設備管理辦法

### 5. 責任歸屬
- [ ] 車位內車輛毀損責任
- [ ] 機械車位故障導致車輛損壞
- [ ] 第三人使用車位造成事故
- [ ] 火災/淹水等天災

### 6. 其他
- [ ] 是否隨主建物一併出租或獨立出租
- [ ] 管委會停車管理辦法與租約之關係
- [ ] 是否可使用訪客車位（如有）

## 車位租約常見爭議

| 爭議類型 | 說明 | 處理建議 |
|----------|------|----------|
| 車位尺寸不足 | 標準車位2.5×5.5m，實際不符 | 可主張民法第423條請求減少租金或解約 |
| 機械車位故障 | 機械車位故障導致無法使用 | 出租人應負責修繕，期間租金可減免 |
| 充電樁爭議 | 管委會禁止安裝 | 依2022年「公寓大廈管理條例」修正，住戶有權裝設（除非有安全疑慮） |
| 車位被占用 | 他人長期占用租賃車位 | 出租人應履行交付義務，由出租人排除占用 |
| 管理費爭議 | 管委會對非住戶車位租戶收取不同費率 | 合理差別定價可接受，但需明訂 |

## 機械車位特別注意

- 載重限制：通常 **1,500~2,200kg**（需確認車重）
- 尺寸限制：通常 **4.8m×1.8m×1.55m**（需確認）
- 每月保養費：通常 **500~2,000元**（由誰負擔）
- 保養維護責任：
  - 出租人（車位所有權人）：提供安全可用之設備
  - 承租人：正常使用，超重或操作不當由承租人負責
- 事故風險：機械故障風險較高，建議確認保險覆蓋
