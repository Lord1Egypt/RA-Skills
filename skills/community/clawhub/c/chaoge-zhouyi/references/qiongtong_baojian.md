# 穷通宝鉴

## 核心内容

### 一、调候总论

**核心观点**：
> 寒暖失调，虽用神得地，亦不吉。调候为急。

**原理**：
1. 天道有寒暖，地道有燥湿
2. 命局有寒暖，岁运有燥湿
3. 寒暖失调，则命局不吉
4. 调候得宜，则命局大贵
