# TikTok Keywords -- Quality Checklist

Validate every keyword research report against this checklist before delivery or implementation. Each item should be confirmed as passing. Flag any failures for revision before finalizing the report.

---

## Category 1: Research Scope and Completeness

- [ ] 1.1 -- A buyer intent statement is defined with demographic, need, and price range
- [ ] 1.2 -- At least 2 adjacent product categories were explored for cross-category keyword opportunities
- [ ] 1.3 -- The raw keyword candidate list contains a minimum of 40 terms before filtering
- [ ] 1.4 -- Keywords were harvested from at least 3 of the 4 required sources (TikTok autocomplete, TikTok Shop search, hashtag research, competitor mining)
- [ ] 1.5 -- TikTok Shop autocomplete was checked specifically, not only general TikTok search autocomplete
- [ ] 1.6 -- Creator video captions and voiceover language were mined for keyword candidates
- [ ] 1.7 -- The research date is recorded on the report for trend validation context

## Category 2: Intent Classification Accuracy

- [ ] 2.1 -- Every keyword in the final report has an assigned intent tier (Browse, Research, or Purchase-ready)
- [ ] 2.2 -- Intent tier assignments are justified by signal words, not assumed from keyword length or specificity alone
- [ ] 2.3 -- Purchase-ready keywords contain at least one explicit purchase modifier (price, "buy," "free shipping," size/variant specifics)
- [ ] 2.4 -- Browse keywords are excluded from product title and description placements
- [ ] 2.5 -- The intent distribution falls within target ranges: Purchase-ready 25-35%, Research 40-50%, Browse 15-25%
- [ ] 2.6 -- Ambiguous keywords (straddling two tiers) are resolved using the framework's four disambiguation rules

## Category 3: Scoring and Ranking Validity

- [ ] 3.1 -- Every keyword is scored across all four dimensions (Volume, Competition, Trend, Intent)
- [ ] 3.2 -- Volume scores are grounded in TikTok-native signals (autocomplete position, related searches), not external tool data alone
- [ ] 3.3 -- Competition density scores reflect exact-match listing counts in TikTok Shop, not Google or Amazon competition data
- [ ] 3.4 -- Trend momentum scores reference specific hashtag view counts and their week-over-week direction
- [ ] 3.5 -- Composite scores are calculated correctly (Volume x Competition x Trend x Intent)
- [ ] 3.6 -- Primary keywords all have composite scores of 200 or above
- [ ] 3.7 -- Keywords scoring below 100 are deprioritized or removed unless a specific niche rationale is documented

## Category 4: Placement Mapping and Copy Quality

- [ ] 4.1 -- The highest composite-score keyword appears within the first 34 characters of the product title
- [ ] 4.2 -- The product title reads naturally when spoken aloud (suitable for a video-first platform)
- [ ] 4.3 -- The product title does not exceed 255 characters total
- [ ] 4.4 -- The product description opening (first 100 characters) contains at least 2 target keywords woven into benefit statements
- [ ] 4.5 -- The product description is not keyword-stuffed -- every keyword appears within a natural sentence communicating product value
- [ ] 4.6 -- Backend search terms include synonyms, common misspellings, and variations not present in the title or description
- [ ] 4.7 -- The video caption draft is under 150 characters and includes 1 primary keyword plus 1 trending phrase
- [ ] 4.8 -- The video caption reads as natural spoken language, not as a keyword string
- [ ] 4.9 -- A hashtag set of 3-5 tags is provided with view counts and trend direction for each

## Category 5: Platform Specificity

- [ ] 5.1 -- No keywords use Amazon-specific conventions ("premium quality," "satisfaction guaranteed," "A+ content" phrasing)
- [ ] 5.2 -- No keywords use Google SEO conventions ("near me," "official site," "[year] guide")
- [ ] 5.3 -- Keywords reflect TikTok-native conversational phrasing (articles and prepositions included where natural)
- [ ] 5.4 -- TikTok-native aesthetic and subculture modifiers are included where relevant ("aesthetic," "clean girl," "minimalist," "cottagecore")
- [ ] 5.5 -- Price-point keywords use round-number thresholds common on TikTok ("under $10," "under $20," "under $30")
- [ ] 5.6 -- Hashtag strategy and search strategy are documented separately with distinct keyword sets and goals
- [ ] 5.7 -- The report distinguishes between general TikTok search results and TikTok Shop search results

## Category 6: Competitive and Trend Validation

- [ ] 6.1 -- The top 5 competitor listings in TikTok Shop have been reviewed for keyword patterns
- [ ] 6.2 -- The competition snapshot includes exact-match listing counts for primary keywords
- [ ] 6.3 -- Competitor price ranges are documented for primary keyword searches
- [ ] 6.4 -- Hashtag view counts are recorded with their trend direction (growing, stable, declining)
- [ ] 6.5 -- No primary keyword relies solely on declining hashtag activity as its trend signal
- [ ] 6.6 -- Seasonal keywords are timed to TikTok's trend cycle (6-8 weeks before traditional retail peak)
- [ ] 6.7 -- Competitive gaps or underserved niches are identified and documented

## Category 7: Deliverable Completeness and Usability

- [ ] 7.1 -- The report follows the output template structure with all sections completed
- [ ] 7.2 -- Keyword clusters are organized thematically, not just listed alphabetically or by score
- [ ] 7.3 -- Each keyword has a designated primary placement zone (Title, Description, Backend, Caption, Hashtag, Affiliate Brief)
- [ ] 7.4 -- Draft copy is provided for product title, description opening, video caption, and hashtag set
- [ ] 7.5 -- An affiliate brief section is included with creator-facing keyword guidance and suggested hooks
- [ ] 7.6 -- A monitoring plan is defined with check frequencies and action triggers for each metric
- [ ] 7.7 -- The next refresh date is set (30 days maximum, or sooner for seasonal products)
- [ ] 7.8 -- No keyword in the final list conflicts with another product listing in the seller's own catalog

---

## How to Use This Checklist

**Before delivery:** Run through every item. Any unchecked item represents a gap that could reduce the effectiveness of the keyword research or introduce errors into listing optimization.

**Minimum passing threshold:** All items in Categories 1, 2, and 4 must pass. No more than 2 items in Categories 3, 5, 6, or 7 can be flagged as incomplete, and those must be documented with a rationale for the gap.

**For team use:** When multiple analysts produce keyword research, use this checklist as a peer review tool. The reviewer checks every item independently and returns the report with flagged items for revision before the report is applied to live listings.

**For ongoing monitoring:** Items 6.4, 6.5, and 7.7 are recurring checks. Revisit these at each scheduled keyword refresh to ensure the report remains current.
