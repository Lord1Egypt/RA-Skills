# Alibaba Java Development Guidelines Skill (嵩山版)

在编写或修改 Java 代码时强制执行阿里巴巴 Java 开发规范（嵩山版 v1.7.0）。

## 功能

- **代码生成时自动遵守规范**: 命名风格、代码格式、OOP 规约、集合处理、并发处理等
- **代码审查时逐条检查**: 对照【强制】规则逐项检查已有代码
- **数据库/SQL 规范**: 建表、索引、SQL 语句、ORM 映射
- **安全规约**: SQL 注入防护、CSRF、参数校验、敏感数据脱敏
- **工程结构**: 应用分层、二方库依赖、服务器配置
- **设计规约**: 用例图、状态图、时序图、SOLID 原则

## 使用

在任何 Java 代码编写场景下，本 skill 会自动激活。触发场景包括：

- "写一个 Java 类"
- "修改这个方法的逻辑"
- "为这个接口生成实现"
- "优化这段代码"
- "检查代码规范"
- "Code Review"

## 规范来源

基于 `D:\ai\Java开发手册（嵩山版）.md`，涵盖全部 7 大维度、300+ 条规约。
