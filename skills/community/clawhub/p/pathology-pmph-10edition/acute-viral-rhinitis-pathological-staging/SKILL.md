# Acute Viral Rhinitis Pathological Staging

## 适用对象
- 鼻黏膜病变患者
- 婴幼儿患者
- 免疫力低下个体

## 触发条件
当患者满足以下条件时启动本技能：
- 出现鼻塞、浆液性鼻分泌物
- 有近期受凉、过劳、慢性病或鼻中隔偏曲等诱因

## 执行步骤

1. **确认潜伏期**：询问发病时间，若为感染后1～3天内，视为潜伏期。

2. **识别初期阶段**：观察是否存在鼻黏膜充血、水肿及浆液性渗出（“浆液性卡他”）。

3. **判断是否进入继发细菌感染阶段**：若分泌物由清亮转为黏稠或脓性，提示链球菌、葡萄球菌等继发感染，炎症性质已转为黏液化脓性。

4. **评估组织损伤与修复进程**：
   - 若病程已达2～3天，注意上皮纤毛是否黏结、细胞是否脱落；
   - 若病程接近2周，应评估是否进入修复期。

5. **筛查高风险并发症（仅限婴幼儿或免疫低下者）**：
   - 若为婴幼儿且症状加重（如发热不退、呼吸急促、耳痛等），立即排查鼻窦炎、中耳炎、肺炎或急性心肌炎。

## 输出建议
- 根据当前病程阶段判断炎症性质；
- 若存在继发感染证据，建议考虑抗菌治疗；
- 若为婴幼儿且病情进展迅速，必须转诊排查系统性并发症。
