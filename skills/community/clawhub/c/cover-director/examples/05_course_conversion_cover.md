# 示例 05：课程转化封面

## 输入

- 标题：AI 自媒体工作流，从选题到变现一次搭好
- 平台：课程页 / 公众号 / 视频号
- 比例：16:9
- 目标：信任 + 转化
- 风格：高级商业、专业、不要太营销号

## 输出摘要

### 标题策略

- 类型：转化型 + 结果型
- 最大关键词：工作流
- 辅助信息：选题到变现
- 人物情绪：专业、自信、可信
- 手势：抱臂或自然指向流程图

### 构图

左侧人物，右侧标题和流程图。背景深蓝或暖灰，加入抽象流程线条，不用夸张爆炸形。

### 无字底图提示词

```text
生成一张 16:9 高级商业课程封面无字底图。左侧是真实课程讲师半身形象，表情自信、专业、可信，姿态自然，轻微抱臂或指向右侧。右侧预留标题区，并有简洁的流程图视觉占位，用抽象节点和线条表达从选题到发布到转化的流程，但不包含文字。背景为深蓝到黑蓝渐变，柔和轮廓光，高级商业摄影质感，干净、有信任感、高清锐利。不要文字、水印、logo、二维码、杂乱元素。
```

### 排版

- 工作流：最大词，白字 + 金色轻描边。
- AI 自媒体：上方标签。
- 从选题到变现：辅助说明。

### 质检预估

- 评分：89/100
- 风险：过度科技感会降低亲和力，需要保持人像真实。
