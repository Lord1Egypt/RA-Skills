# 基金诊断案例

**案例**：易方达蓝筹精选混合 (000001) 深度分析

---

## 用户输入

```
@ant 分析易方达蓝筹精选，我持有 10 万元，亏损 15%，要不要继续持有？
```

---

## ant 执行流程

```
1. 识别意图：基金分析 + 持仓诊断
2. 调用模块：
   - 模块 1：单一基金分析
   - 模块 4：持仓诊断（针对用户持仓）
   - 模块 5：基金经理分析
3. 数据获取：
   - 天天基金 API：基础数据 + 业绩 + 费率
   - 且慢 MCP：持仓明细 + 收益归因
4. 生成报告
```
