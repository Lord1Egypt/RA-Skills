# 分账结算与纠纷处理

> 本文档详细说明分账触发条件、7步分账流程、纠纷调解SOP等分账结算相关内容。

---

## 一、分账触发条件

### 1.1 分账前置条件

| 条件类型 | 具体要求 | 验证方式 |
|----------|----------|----------|
| **验收完成** | 客户验收通过 | 验收签字单 |
| **款项到账** | 项目款项已到账 | 银行流水 |
| **无争议** | 无未解决的纠纷 | 纠纷台账 |
| **发票齐全** | 成本发票已提交 | 发票清单 |

### 1.2 分账启动条件

| 触发类型 | 触发条件 | 说明 |
|----------|----------|------|
| **里程碑分账** | 里程碑验收通过 | 按里程碑分账 |
| **项目分账** | 项目全部验收通过 | 按项目分账 |
| **尾款分账** | 尾款到账 | 按约定比例 |
| **提前分账** | 特殊情况申请 | 需三长老批准 |

## 二、分账流程

### 2.1 七步分账流程

```
┌─────────────────────────────────────────────────────────────┐
│                    分账七步流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【第一步：回款到账】                                         │
│  ├── 客户付款至共管账户                                      │
│  ├── 确认到账金额                                            │
│  └── 开具收据给客户                                          │
│                                                              │
│  【第二步：开票给客户】                                       │
│  ├── 牵头方开全额发票                                        │
│  ├── 确认发票内容正确                                        │
│  └── 发票交付客户                                            │
│                                                              │
│  【第三步：成本确认与开票】                                   │
│  ├── 各方提交成本发票                                        │
│  ├── 审核成本发票合法性                                      │
│  └── 成本汇总确认                                            │
│                                                              │
│  【第四步：税费核算】                                         │
│  ├── 计算应缴税费                                            │
│  ├── 预留税费资金                                            │
│  └── 确认税务处理方案                                        │
│                                                              │
│  【第五步：分账执行】                                         │
│  ├── 计算可分配金额                                           │
│  ├── 计算各方应得金额                                         │
│  ├── 生成《分账确认书》                                      │
│  └── 各方确认分账方案                                        │
│                                                              │
│  【第六步：打款发放】                                         │
│  ├── 牵头方分别打款                                           │
│  ├── 确认各方到账                                            │
│  └── 记录打款凭证                                            │
│                                                              │
│  【第七步：完税凭证】                                         │
│  ├── 各方自行申报纳税                                        │
│  ├── 保留完税凭证                                            │
│  └── 分账档案归档                                            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 分账计算公式

```
【公式1】可分配金额
可分配金额 = 到账金额 - 预留税费 - 平台服务费 - 实际成本

【公式2】各方分账金额
各方分账金额 = 可分配金额 × 该方分账比例 × 该方信用系数

【公式3】信用系数
S级 = 1.2
A级 = 1.0
B级 = 0.9
C级 = 0.7
D级 = 0.5
```

### 2.3 分账示例

**示例项目**：
- 项目总金额：100,000元
- 到账金额：100,000元
- 预留税费比例：10%
- 平台服务费比例：5%
- 实际成本：20,000元

**计算过程**：
```
步骤1：预留税费 = 100,000 × 10% = 10,000元
步骤2：扣除平台服务费 = 100,000 × 5% = 5,000元
步骤3：扣除实际成本 = 20,000元
步骤4：可分配金额 = 100,000 - 10,000 - 5,000 - 20,000 = 65,000元

步骤5：各方分账（假设比例：甲40%，乙35%，丙25%）
- 甲方（信用A级，系数1.0）= 65,000 × 40% × 1.0 = 26,000元
- 乙方（信用B级，系数0.9）= 65,000 × 35% × 0.9 = 20,475元
- 丙方（信用A级，系数1.0）= 65,000 × 25% × 1.0 = 16,250元
```

## 三、付款规则

### 3.1 付款节点

| 项目类型 | 付款节点 | 比例 | 说明 |
|----------|----------|------|------|
| **常规项目** | 预付款 | 30% | 合同签署后 |
| | 进度款 | 40% | 里程碑验收后 |
| | 尾款 | 30% | 最终验收后 |
| **小额项目** | 预付款 | 50% | 合同签署后 |
| | 尾款 | 50% | 最终验收后 |
| **长期项目** | 按月付款 | - | 每月按进度结算 |

### 3.2 付款时限

| 付款类型 | 付款时限 | 超时处理 |
|----------|----------|----------|
| **分账发起** | 验收通过后3个工作日内 | 催促/升级 |
| **打款执行** | 分账确认后5个工作日内 | 扣除违约金 |
| **发票开具** | 收款后7个工作日内 | 影响后续分账 |

### 3.3 付款方式

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| **银行转账** | 常规方式，推荐使用 | 全部场景 |
| **第三方支付** | 如支付宝、微信 | 小额付款 |
| **票据** | 如支票、汇票 | 大额付款 |

## 四、税务处理

### 4.1 税务责任

| 责任方 | 税务责任 |
|--------|----------|
| **牵头方** | 负责整体税务协调，代扣代缴（可选） |
| **各参与方** | 自行申报纳税，保留完税凭证 |

### 4.2 发票流转

| 开票方 | 发票类型 | 开票给 | 税率参考 |
|--------|----------|--------|----------|
| 牵头方（对外） | 增值税专用发票 | 客户 | 6-13% |
| 各参与方 | 增值税普通发票/劳务发票 | 牵头方 | 1-6% |

### 4.3 税费预留

| 税费类型 | 预留比例 | 说明 |
|----------|----------|------|
| **增值税** | 6-13% | 按实际开票税率 |
| **个人所得税** | 1-3% | 劳务报酬预扣 |
| **企业所得税** | 2-5% | 如为企业主体 |

## 五、纠纷处理

### 5.1 纠纷类型

| 纠纷类型 | 典型情形 | 处理优先级 |
|----------|----------|-----------|
| **质量纠纷** | 交付物质量不达标 | P1 |
| **进度纠纷** | 进度延误责任 | P1 |
| **费用纠纷** | 分账金额争议 | P1 |
| **违约纠纷** | 违反协议约定 | P2 |
| **知识产权纠纷** | 归属争议 | P2 |

### 5.2 纠纷调解SOP

```
┌─────────────────────────────────────────────────────────────┐
│                    纠纷调解SOP                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【一级：当事人协商】                                          │
│  ├── 时限：48小时内                                          │
│  ├── 方式：双方直接协商                                      │
│  └── 结果：达成一致/进入二级                                  │
│                                                              │
│  【二级：挂帅人调解】                                         │
│  ├── 时限：48小时内                                          │
│  ├── 方式：挂帅人主持协调                                    │
│  └── 结果：达成一致/进入三级                                  │
│                                                              │
│  【三级：三长老调解】                                         │
│  ├── 时限：7天内                                             │
│  ├── 方式：三长老会议裁决                                    │
│  └── 结果：形成书面裁决/进入四级                             │
│                                                              │
│  【四级：仲裁/诉讼】                                          │
│  ├── 时限：依程序                                            │
│  ├── 方式：提交仲裁/法院诉讼                                 │
│  └── 结果：仲裁裁决/法院判决                                 │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 5.3 常见纠纷处理

| 纠纷类型 | 处理原则 | 处理方式 |
|----------|----------|----------|
| **质量不达标** | 按验收标准判定 | 返工/部分扣款/解除合同 |
| **进度延误** | 区分责任归属 | 不可抗力不追责；己方责任扣款 |
| **分账争议** | 按协议约定执行 | 数据公开、账目透明 |
| **需求变更** | 书面确认变更 | 评估影响、调整计划 |

### 5.4 纠纷档案

| 要素 | 内容要求 |
|------|----------|
| **基本信息** | 项目名称、纠纷编号、发生时间 |
| **当事方** | 纠纷各方信息 |
| **纠纷内容** | 纠纷事实描述 |
| **处理过程** | 各层级处理记录 |
| **处理结果** | 最终解决方案 |
| **证据材料** | 相关证据清单 |
| **经验教训** | 纠纷反思与改进 |

## 七、快速检查清单

分账前，确认以下事项：

```
□ 验收已完成并签字确认
□ 款项已到账
□ 无未解决的纠纷
□ 成本发票已提交齐全
□ 税费已核算正确
□ 分账比例已确认
□ 各方账户信息已确认
□ 分账确认书已生成
```
