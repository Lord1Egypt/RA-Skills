# 📖 使用指南

## 快速上手

### 1. 记录积分

直接说完成了什么任务：

```
今天完成了汉字抄写 2 课，口算题卡 2 篇全对
```

系统会自动：
- ✅ 理解语义
- ✅ 记录积分
- ✅ 更新余额

### 2. 记录消费

```
积分消费 买零食花了 20 分
```

### 3. 查询余额

```
现在多少分？
```

---

## 语意识别

系统能理解各种说法：

| 你说 | 系统理解 |
|------|---------|
| "写了 2 课生字" | 汉字抄写 |
| "口算做了 2 篇" | 口算题卡 |
| "ABC Reading 读了" | ABC Reading |
| "忘带书包" | 积分消费 |

---

## 数据管理

### 查看余额文件

```bash
cat kids-points/balance.md
```

### 查看输入日志

```bash
cat kids-points/logs/input.log
```

### 手动修正

直接编辑 `balance.md`：

```markdown
| 2026-03-25 | 收入 | +2 | 98.8 | 汉字抄写 |
```

---

## 防重复

同一天说相同的内容会提示：

```
⚠️ 这条记录已经存在了
```
