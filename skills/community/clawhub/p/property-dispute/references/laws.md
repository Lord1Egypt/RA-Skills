# 物业管理法律法规速查数据库

## 一、《民法典》物权编 — 业主的建筑物区分所有权（第271-287条）

### 核心权利

| 条款 | 内容概要 | 维权要点 |
|------|----------|----------|
| 第271条 | 业主对专有部分享所有权，对共有部分享共有和共同管理权 | 电梯、走廊、屋顶、外墙等属共有部分 |
| 第272条 | 业主对专有部分可占有、使用、收益、处分 | 不得危及建筑安全、损害他人权益 |
| 第273条 | 业主对共有部分享有权利、承担义务 | 不得以放弃权利为由不履行义务 |
| 第274条 | 道路、绿地、公共场所、公用设施、物业服务用房属业主共有 | 物业无权擅自处分共有部分 |
| 第278条 | 业主共同决定事项及表决规则 | 解聘物业需2/3以上参与+参与面积/人数过半数同意 |
| 第281条 | 维修资金属业主共有，用于共有部分维修更新改造 | 紧急情况可直接申请使用维修资金 |
| 第282条 | 利用共有部分的收入，扣除合理成本后属业主共有 | 电梯广告、停车费等公共收益归业主 |
| 第284条 | 业主有权依法更换建设单位聘请的物业 | 可依法解聘前期物业 |
| 第285条 | 物业接受业主监督，及时答复询问 | 业主享有监督权和询问权 |
| 第287条 | 业主对侵害合法权益的行为可请求承担民事责任 | 可起诉物业不作为 |

### 维修资金使用（第281条详解）

- 维修资金属业主共有
- 用于电梯、屋顶、外墙、无障碍设施等共有部分维修、更新、改造
- 筹集、使用情况应**定期公布**
- **紧急情况下**，业主大会或业委会可依法申请使用（如电梯停运、屋顶严重漏水、外墙脱落危险等）

## 二、《民法典》合同编 — 物业服务合同（第937-950条）

| 条款 | 内容概要 | 维权要点 |
|------|----------|----------|
| 第937条 | 物业合同定义：物业提供服务，业主支付物业费 | 合同性质是服务合同，非管理合同 |
| 第938条 | 合同内容包括服务事项、质量、费用、维修资金使用等 | 需书面形式，口头承诺构成合同组成部分 |
| 第942条 | 物业有妥善维修、养护、清洁、绿化等义务 | 物业不作为属违约 |
| 第943条 | **物业应定期公开服务事项、收费项目/标准、维修资金使用、共有部分经营与收益** | 业主查账权的法律基础 |
| 第944条 | 业主支付物业费义务；**物业不得断水断电逼交物业费** | 断水断电=违法 |
| 第946条 | **业主可依法定程序共同决定解聘物业，提前60日书面通知** | 解聘权是业主核心权利 |
| 第949条 | 合同终止后物业须移交资料、退出服务区域 | 拒不退出可索赔 |

## 六、物业费涨价相关法律规定

### 涨价必须满足的条件（《民法典》第278条+《物业服务收费管理办法》）：

1. **物业提出调价方案** — 须公示调价理由、成本测算、新收费标准
2. **业主大会表决** — 需"双过半"（参与专有面积过半+参与人数过半）同意
3. **公示并备案** — 调价结果需向价格主管部门备案
4. **若不满足上述条件** — 涨价无效，业主可按原标准支付

### 业主拒绝涨价的合法理由：

- 未召开业主大会
- 未公示成本构成
- 涨幅明显超过CPI/同区域水平
- 服务质量未达标（可主张调价前提不成立）

## 七、业主知情权与查账权

### 《民法典》第943条（核心条款）
物业应**定期**以**合理方式**向业主公开：
- 服务事项、负责人员、质量要求
- 收费项目、收费标准、履行情况
- 维修资金使用情况
- 共有部分经营与收益情况

### 《物业服务收费管理办法》
- 物业每半年至少公示一次共用部位收益及使用情况

### 查账五步法：
1. 联合≥5%业主签名发起书面查询
2. 以EMS寄送查询函（取证用）
3. 物业应在15日内答复
4. 拒绝则向街道办投诉
5. 仍不配合则提起知情权诉讼

### 可查账目范围：
- 物业费收支明细（含保洁/保安/绿化等人工成本）
- 电梯广告/道闸广告收益及合同
- 地面停车费分配
- 公共设施维修支出（报价单+验收报告）
- 代收代缴水电费差额
- 公共收益银行流水

## 八、物业违规处罚标准

| 违规行为 | 法律依据 | 处罚标准 |
|----------|----------|----------|
| 违规涨价 | 价格法 | 退还多收款，处违法所得1-5倍罚款 |
| 财务不公开 | 民法典第943条 | 败诉担责 |
| 偷税漏税 | 税收征管法第63条 | 追缴税款+滞纳金+0.5-5倍罚款；构成犯罪追刑责 |
| 消防通道堵塞 | 消防法第60条 | 单位罚5千-5万 |
| 断水断电逼费 | 民法典第944条 | 违法，可起诉赔偿 |
| 挪用公共收益 | 民法典第282条 | 返还+赔偿 |
| 无资质经营 | 物业管理条例 | 吊销营业执照 |
