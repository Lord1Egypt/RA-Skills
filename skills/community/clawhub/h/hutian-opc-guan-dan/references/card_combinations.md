# 牌型组合速查表

## 一、基础牌型

### 1.1 单张
| 项目 | 说明 |
|------|------|
| 定义 | 任意一张牌 |
| 比较 | 点数大的胜 |
| 包含 | 王、级牌、A、K...2 |

### 1.2 对子
| 项目 | 说明 |
|------|------|
| 定义 | 两张点数相同的牌 |
| 比较 | 找最大的一张比较 |
| 示例 | 33、AA、77 |

### 1.3 三张
| 项目 | 说明 |
|------|------|
| 定义 | 三张点数相同的牌 |
| 比较 | 直接比较点数 |
| 示例 | 333、AAA、777 |

### 1.4 三带二
| 项目 | 说明 |
|------|------|
| 定义 | 三张 + 一对 |
| 比较 | 比较三张的点数 |
| 包含 | 三张点数必须相同 |
| 示例 | 33355、AAA77 |

## 二、顺类牌型

### 2.1 顺子
| 项目 | 说明 |
|------|------|
| 定义 | 5张+连续点数的单牌 |
| 限制 | 2和王不能入顺 |
| 比较 | 比较最大张 |
| 示例 | 34567、789TJ、10JQKA |

### 2.2 连对（钢板）
| 项目 | 说明 |
|------|------|
| 定义 | 3对+连续点数的对子 |
| 限制 | 2和王不能入连对 |
| 比较 | 比较最大对 |
| 示例 | 334455、778899 |

## 三、飞机类牌型

### 3.1 飞机
| 项目 | 说明 |
|------|------|
| 定义 | 2组+连续点数的三张 |
| 比较 | 比较最大的三张组 |
| 示例 | JJJQQQ、666777 |

### 3.2 飞机带翅膀
| 项目 | 说明 |
|------|------|
| 定义 | 飞机 + 每个三张带单牌/对子 |
| 翅膀 | 必须配套（3机带3翼） |
| 比较 | 比较飞机部分 |
| 示例 | JJJQQQ + 3单 + 3对 |

## 四、炸弹类牌型

### 4.1 炸弹优先级
| 炸弹类型 | 张数 | 优先级 |
|----------|------|--------|
| 七炸 | 7张 | 最高 |
| 六炸 | 6张 | ↓ |
| 五炸 | 5张 | ↓ |
| 四炸 | 4张 | 最低 |

### 4.2 炸弹大小比较
```
规则1：张数多的炸弹 > 张数少的炸弹
规则2：张数相同，点数大的 > 点数小的
规则3：七炸 > 六炸 > 五炸 > 四炸
```

### 4.3 特殊炸弹
| 类型 | 组成 | 说明 |
|------|------|------|
| 天王炸弹 | 大王+大王+小王+小王 | 最大炸弹 |
| 同花顺 | 5张+同花色顺子 | 大于普通炸弹 |

## 五、同花顺

### 5.1 定义
- 5张或更多同花色连续单牌
- 花色：♠♠♥♣♦
- 点数：连续

### 5.2 比较规则
```
规则1：张数多的同花顺 > 张数少的同花顺
规则2：张数相同，比较最大张
规则3：同花顺 > 所有普通炸弹
规则4：同花顺 < 天王炸弹
```

### 5.3 示例
| 同花顺 | 说明 |
|--------|------|
| ♠34567 | 5张同花顺 |
| ♥789TJ | 5张同花顺 |
| ♣10JQKA | 6张同花顺 |

## 六、特殊牌：逢人配

### 6.1 定义
- 6张红桃级牌
- 可替代任何牌

### 6.2 使用场景
| 场景 | 示例 |
|------|------|
| 单张 | 红桃A当A使用 |
| 对子 | 红桃A+普通A = AA |
| 三张 | 红桃A+AA = AAA |
| 顺子 | 567 + 红桃8 = 5678 |
| 炸弹 | 8888 + 红桃8 = 5555（假）或正常 |

### 6.3 同花顺中的使用
- 只能配对应花色的同花顺
- 例如：红桃8只能在红桃同花顺中使用

## 七、牌型大小速查

### 7.1 点数大小（单张）
```
大王 > 小王 > 级牌 > A > K > Q > J > 10 > 9 > 8 > 7 > 6 > 5 > 4 > 3 > 2
```

### 7.2 牌型压制关系
```
同花顺 > 天王炸弹 > 七炸 > 六炸 > 五炸 > 四炸 > 其他牌型
                            ↓
                   顺子 > 连对 > 三带二 > 三张 > 对子 > 单张
```

### 7.3 压制规则
- 必须出相同牌型
- 必须张数相同（除炸弹外）
- 必须点数/张数更大

## 九、出牌优先级

### 9.1 单张出牌
| 优先级 | 牌型 | 理由 |
|--------|------|------|
| 1 | 大王/小王 | 最大 |
| 2 | 级牌A | 控制 |
| 3 | AK | 回收容易 |
| 4 | 中间牌 | 风险小 |
| 5 | 小牌 | 被动 |

### 9.2 对子出牌
| 优先级 | 牌型 | 理由 |
|--------|------|------|
| 1 | AA | 最大 |
| 2 | KK | 较大 |
| 3 | 对子A | 可回收 |
| 4 | 中对 | 风险可控 |
| 5 | 小对 | 被动 |

### 9.3 顺子出牌
| 优先级 | 牌型 | 理由 |
|--------|------|------|
| 1 | 短顺（5张） | 灵活 |
| 2 | 中顺（6-7张） | 适中 |
| 3 | 长顺（8+张） | 风险大 |
