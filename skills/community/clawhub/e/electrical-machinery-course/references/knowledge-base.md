# 电机学课程知识体系

## 一、直流电机

### 1.1 直流电机的工作原理
- 直流电机的基本结构：定子（主磁极、换向极、机座、电刷装置）、转子（电枢铁心、电枢绕组、换向器、转轴）
- 电枢绕组：叠绕组、波绕组的连接规律与特点
- 直流发电机原理：导体在磁场中运动产生感应电动势，换向器将交流电动势转换为直流
- 直流电动机原理：载流导体在磁场中受电磁力产生转矩
- 感应电动势公式：E = CeΦn（Ce 为电动势常数，Φ 为每极磁通，n 为转速）
- 电磁转矩公式：T = CTΦIa（CT 为转矩常数，Ia 为电枢电流）

### 1.2 直流电机的运行特性
- **发电机**
  - 他励发电机：空载特性、外特性、调节特性
  - 并励发电机：自励条件（剩磁、励磁绕组接线正确、励磁回路电阻小于临界值）
  - 复励发电机：积复励与差复励
- **电动机**
  - 他励/并励电动机：机械特性 n = U/(CeΦ) - Ra·T/(CeCTΦ²)，硬特性
  - 串励电动机：软特性，转速随负载增大而迅速下降
  - 复励电动机：介于并励与串励之间

### 1.3 直流电动机的起动与调速
- 起动：直接起动（仅限小容量）、电枢串电阻起动、降压起动
- 调速方法：
  - 电枢串电阻调速（向下调速，效率低）
  - 改变电枢电压调速（向下调速，效率高，平滑）
  - 改变励磁电流调速（向上调速，弱磁调速）
- 制动：能耗制动、反接制动、回馈制动

### 1.4 换向
- 换向的物理过程
- 换向火花产生的原因：电磁原因（电抗电动势）、机械原因
- 改善换向的方法：装设换向极、移动电刷位置、选用合适的电刷
- 环火与补偿绕组

## 二、变压器

### 2.1 变压器的工作原理与结构
- 基本结构：铁心（心式/壳式）、绕组（高压绕组/低压绕组）
- 工作原理：电磁感应，电压比等于匝数比 U₁/U₂ ≈ E₁/E₂ = N₁/N₂ = k
- 额定值：额定容量 SN、额定电压 U₁N/U₂N、额定电流 I₁N/I₂N

### 2.2 变压器的运行分析
- 空载运行：空载电流（励磁分量 + 铁损分量）、空载损耗（主要为铁损）
- 负载运行：磁动势平衡 F₁ + F₂ = F₀
- 基本方程式、等效电路（T 形等效电路、Γ 形近似等效电路）
- 相量图

### 2.3 变压器的参数测定
- 空载试验：测铁损 pFe 和励磁参数 Rm、Xm
- 短路试验：测铜损 pCu 和短路参数 Rk、Xk

### 2.4 变压器的运行特性
- 电压变化率：ΔU% = (U₂₀ - U₂)/U₂N × 100%
- 效率：η = P₂/(P₂ + pFe + pCu) × 100%
- 效率最大条件：可变损耗 = 不变损耗，即 pCu = pFe

### 2.5 三相变压器
- 三相变压器的磁路系统：组式变压器、心式变压器
- 连接组别：Y,y、Y,d、D,y、D,d 及钟面表示法
- 绕组连接方式对空载电动势波形的影响

### 2.6 变压器的并联运行
- 并联运行条件：变比相同、连接组别相同、短路阻抗标幺值相同
- 变比不等时的环流

### 2.7 特殊变压器
- 自耦变压器：结构特点、容量关系（电磁容量 + 传导容量）
- 仪用互感器：电流互感器（二次侧严禁开路）、电压互感器（二次侧严禁短路）

## 三、交流电机理论的共同问题

### 3.1 交流绕组
- 绕组术语：极距 τ、节距 y、每极每相槽数 q、槽距角 α
- 单层绕组：等距式、同心式、链式
- 双层绕组：叠绕组、波绕组
- 绕组因数：短距因数 kp1 = sin(yα/2)、分布因数 kd1 = sin(qα/2)/(q·sin(α/2))

### 3.2 交流绕组的感应电动势
- 相电动势：Eφ1 = 4.44fNΦ₁kN1
- kN1 = kp1 × kd1 为绕组因数
- 谐波电动势及其削弱方法

### 3.3 交流绕组的磁动势
- 单相绕组磁动势：脉振磁动势 Fφ1 = (4/π)×(N·kw1/2p)×I
- 三相绕组磁动势：旋转磁动势，幅值 F1 = (3/2)Fφ1
- 旋转磁动势的转速（同步转速）：n₁ = 60f/p
- 旋转方向：从超前相转向滞后相

## 四、异步电机（感应电机）

### 4.1 三相异步电动机的工作原理
- 旋转磁场切割转子导体产生感应电流
- 转子电流与磁场相互作用产生电磁转矩
- 转差率：s = (n₁ - n)/n₁
- 额定转差率通常为 0.01~0.05

### 4.2 三相异步电动机的运行分析
- 转子侧频率归算：f₂ = sf₁
- 频率归算与绕组归算
- 基本方程式、T 形等效电路
- 简化等效电路

### 4.3 异步电动机的功率与转矩
- 功率流程：P₁ → pCu1 → Pem → pCu2 → Pmec → pmec+pad → P₂
- 电磁功率：Pem = m₂I₂'²·R₂'/s
- 转子铜损：pCu2 = sPem（重要关系）
- 机械功率：Pmec = (1-s)Pem
- 电磁转矩：T = Pem/Ω₁ = Pmec/Ω

### 4.4 异步电动机的机械特性
- 机械特性表达式（参数表达式、实用表达式、简化表达式）
- 额定转矩 TN、最大转矩 Tmax（临界转差率 sm）、起动转矩 Tst
- 过载能力 λ = Tmax/TN
- 机械特性曲线分析

### 4.5 异步电动机的起动
- 直接起动：起动电流大（5~7 倍额定电流），起动转矩不大
- 降压起动：Y-Δ 起动（起动电流/转矩降为 1/3）、自耦变压器起动
- 绕线型异步电动机：转子串电阻起动（既减小起动电流又增大起动转矩）
- 深槽式与双笼型电动机：利用集肤效应改善起动性能

### 4.6 异步电动机的调速
- 变极调速：有级调速，Y/YY、Δ/YY 接法
- 变频调速：平滑调速，V/f 恒定控制
- 改变转差率调速：转子串电阻（绕线型）、串级调速

### 4.7 单相异步电动机
- 脉振磁场分解为两个旋转磁场
- 起动方法：分相起动（电容起动、电阻起动）、罩极起动

## 五、同步电机

### 5.1 同步电机的工作原理与结构
- 同步转速：n = 60f/p
- 结构：隐极式（汽轮发电机）、凸极式（水轮发电机）
- 励磁方式：直流励磁机励磁、静止半导体励磁、旋转半导体励磁

### 5.2 同步发电机的运行分析
- 隐极同步发电机：
  - 电动势方程式：Ė₀ = Ú + İRa + jİXd
  - 相量图、等效电路
- 凸极同步发电机（双反应理论）：
  - 电动势方程式：Ė₀ = Ú + İRa + jİdXd + jİqXq
  - 内功率因数角 ψ 的确定方法

### 5.3 同步发电机的运行特性
- 空载特性、短路特性、零功率因数负载特性
- 用空载与短路特性求 Xd（不饱和值）
- 用空载与零功率因数特性求漏抗 Xσ 和电枢反应磁动势 Fa

### 5.4 同步发电机的并联运行
- 并联条件：相序相同、频率相等、电压相等
- 并联方法：准确同步法、自同步法
- 有功功率调节：调节原动机转矩
- 无功功率调节：调节励磁电流（V 形曲线）
- 功角特性：Pem = mEU₀sinδ/Xd（隐极）

### 5.5 同步电动机
- 同步电动机的起动：异步起动法（阻尼绕组）
- 同步电动机的功率因数调节：过励→发出无功（容性），欠励→吸收无功（感性）
- 同步补偿机：专门调节无功功率的同步电动机
