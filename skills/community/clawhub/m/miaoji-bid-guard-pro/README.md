# 秒技·广告护城河Pro版

## 快速开始

### 基本使用

1. 打开技能
2. 输入品类和目标ACoS
3. 选择季节节点
4. 获取90天广告战役规划

### 输入示例

```
品类：无线蓝牙耳机
阶段：新品推广期
目标ACoS：25%
可用预算：$2000/月
季节节点：Prime Day
```

## 功能概览

### ROI预测模型
获取：
- 90天ROI变化曲线
- 关键里程碑设定
- 阶段化策略调整

### 多活动协同策略
获取：
- 品牌广告+效果广告+展示广告联动方案
- 预算分配建议
- 协同节奏说明

### 季节性出价调整
获取：
- Prime Day出价系数表
- Q4旺季出价策略
- 新年/开年出价建议

### 关键词攻防矩阵
获取：
- 进攻/防御/撤退决策建议
- 12种场景应对方案
- 触发条件说明
