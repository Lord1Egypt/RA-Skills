## The 7 Compliance Moats

| # | Moat | What It Blocks |
|---|------|----------------|
| 1 | NBHSB Dual-License Gate | Missing license numbers; NBC formats used for barber services |
| 2 | Straight Razor / BBP Gate | Blade reuse implication; no sanitation disclosure; OSHA ECP gap |
| 3 | Master Barber Credential Gate | "Master Barber" title without NBHSB-MB designation |
| 4 | Instrument Sanitation Gate | "Sterile," "autoclave," "hospital-grade" false claims |
| 5 | Hair/Beard Product Drug Claim Gate | "Grows beard," "stops hair loss," structure/function claims |
| 6 | FTC 2023 Before/After Gate | Best-work-as-typical transformation posts |
| 7 | Booth Rental / IC Advertising Gate | "Our barbers" language for IC booth renters |
