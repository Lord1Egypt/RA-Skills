# CBIRC 合规检查清单

## 保险许可证合规
- [ ] 保险许可证在有效期内
- [ ] 业务范围包含投标险种
- [ ] 许可证复印件加盖公章
- [ ] 分支机构经营许可（如需）

## 偿付能力合规（偿二代 C-ROSS）
- [ ] 核心偿付能力充足率 ≥ 50%
- [ ] 综合偿付能力充足率 ≥ 100%
- [ ] 风险综合评级 ≥ B类
- [ ] 最近季度偿付能力报告已披露
- [ ] 偿付能力压力测试达标

## 再保险合规
- [ ] 危险单位划分符合监管规定（GB/T XXXXX）
- [ ] 财产险自留额 ≤ 实收资本的10%
- [ ] 每危险单位自留额 ≤ 实收资本的5%
- [ ] 预约/合约分保合同已签订
- [ ] 再保险接受人资质合规

## 条款费率合规
- [ ] 保险条款已向监管报备
- [ ] 保险费率已向监管报备
- [ ] 条款中无禁止性规定（格式条款）
- [ ] 免责条款已尽提示说明义务
- [ ] 不得存在"免除保险人依法应承担的义务"条款

## 投标行为合规
- [ ] 不得串通投标、抬高或压低标价
- [ ] 不得向招标人或者评标委员会成员行贿
- [ ] 投标保证金已按要求缴纳
- [ ] 投标文件无虚假信息（如虚假偿付能力数据）
- [ ] 授权委托书有效（加盖公章、法人签字）

## 团险特别规定
- [ ] 团体保险合同载明被保险人组成范围
- [ ] 不得向未参加团险的个人出具保险单
- [ ] 团险业务统计报表按监管要求报送
- [ ] 团险费率符合监管定价原则

## 引用法规
1. 《中华人民共和国保险法》（2015年修订）
2. 《保险公司偿付能力监管规则第1-20号》（C-ROSS）
3. 《再保险业务管理规定》（保监会令〔2015〕3号）
4. 《保险条款和保险费率管理办法》（保监会令〔2010〕3号）
5. 《保险经纪人监管规定》（保监会令〔2018〕3号）
6. 《保险专业代理机构监管规定》（保监会令〔2018〕5号）
