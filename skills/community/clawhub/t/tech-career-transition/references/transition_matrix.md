## 方向匹配度计算公式

```
匹配度 = 0
FOR each 维度:
  gap = MAX(0, 最低要求 - 用户得分)
  match_rate = 1 - (gap / 4)  // 缺口越大匹配度越低
  IF 用户得分 >= 理想要求: match_rate = 1.0
  match_rate = MIN(match_rate, 1.0)
  weight = 方向维度权重(方向, 维度)
  匹配度 += match_rate * weight * 100
```

## 转行建议话术模板

### 匹配度 ≥ 80%："强烈推荐"
> 你的能力画像与该方向高度匹配，核心能力基本就绪。建议在X个月内完成Y补充，即可启动转型。

### 匹配度 60-79%："推荐"
> 该方向与你有较好的匹配度，但存在X方面的能力缺口。建议通过Y方式补齐，预计Z个月可达到转型条件。

### 匹配度 40-59%："可考虑"
> 该方向有一定可行性，但需要较大投入。核心短板在X方面，建议先通过Y打基础，再根据进展决定是否继续。

### 匹配度 < 40%："暂不推荐"
> 该方向目前与你的能力画像差距较大，转型成本很高。建议优先考虑其他匹配度更高的方向，或先花1-2年系统性提升核心能力。
