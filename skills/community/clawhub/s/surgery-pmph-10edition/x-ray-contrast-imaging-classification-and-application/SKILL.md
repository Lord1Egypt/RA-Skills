# X线造影检查方法分类与应用

## 触发条件
当临床需评估以下情况时使用本技能：
- 腔道（如消化道、泌尿道）的形态异常
- 充盈缺损或管腔狭窄
- 肿瘤的血供分布或范围
- 胆胰管结构异常

## 执行流程

### 1. 普通造影
- **对比剂选择**：
  - 消化道：使用钡剂（如上消化道钡餐、结肠灌肠）
  - 泌尿系统及其他非消化道腔道：使用碘剂（如泛影葡胺、碘苯酯）
- **增强显像**：
  - 必要时加用发泡剂实施气钡双重对比，以提高黏膜细节显示
  - 可联合低张药物（如山莨菪碱）松弛平滑肌，便于发现微小病变
- **观察指标**：充盈缺损、组织破坏、管腔狭窄等形态学改变

### 2. 插管造影
- 通过特殊器械直接插管注入对比剂，例如：
  - 逆行输尿管插管行肾盂造影
  - 在纤维十二指肠镜引导下进行内镜逆行胆胰管造影（ERCP）

### 3. 排泄性造影
- 静脉注射碘剂后，利用器官自身排泄功能成像，如静脉肾盂造影（IVP）

### 4. 血管造影
- 经周围动脉（如股动脉）插管，选择性进入目标动脉（肝动脉、颈动脉、腹腔干、肠系膜上/下动脉）
- 注射碘剂显示肿瘤供血血管
- **推荐技术**：采用数字减影血管造影（DSA）以获得高清晰度血管图像

## 禁忌与注意事项
- **碘过敏者**：禁用碘剂造影
- **严重肾功能不全者**：慎用静脉造影
- **空腔脏器穿孔**：为钡剂造影绝对禁忌

## 输出结果
生成高对比度的腔道或血管影像，用于判断肿瘤位置、范围、血供及形态异常。
