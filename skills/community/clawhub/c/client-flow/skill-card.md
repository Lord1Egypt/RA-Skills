## Description: <br>
Automates client onboarding and project lifecycle setup by generating project folders, briefs, welcome emails, kickoff meetings, task boards, reminders, and a client registry from a single intake message. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ariktulcha](https://clawhub.ai/user/ariktulcha) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Freelancers, agencies, and small businesses use this skill to set up and track client projects from intake through closure. It coordinates drafts and actions across email, calendar, file storage, task management, reminders, and client status reporting. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: A single prompt can modify several connected business tools and persist client data. <br>
Mitigation: Confirm the exact email, calendar, storage, and task-manager accounts before use, and require previews before any send, invite, folder, task, reminder, archive, or registry change. <br>
Risk: Client contact, budget, deadline, and project status data may be stored in folders, memory, task systems, or registries without clear retention boundaries. <br>
Mitigation: Choose where client data is stored before execution, define how it can be removed, and avoid adding sensitive client details unless the chosen storage location is appropriate. <br>
