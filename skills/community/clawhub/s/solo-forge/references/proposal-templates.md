# 提案场景模板

生成提案、开发信、案例包装、服务介绍时参考以下模板。

## 提案

> **三档报价**：>5,000 元的项目使用三档模板（基础版/推荐版/旗舰版）。≤5,000 元的项目直接单档报价，可选附带增值服务清单。

```markdown
# {项目名称} — 项目提案

## 致：{客户公司名 / 联系人}

### 概要
基于贵方{需求描述}，提供以下解决方案，预计{时间线}完成。

### 一、需求理解
> 我理解您的核心需求是：{一句话概括}
- **现状**：{用客户语言描述痛点}
- **目标**：{业务目标，可量化}
- **成功标准**：{怎样算成功}

### 二、解决方案
| 阶段 | 内容 | 交付物 | 周期 |
|:---|:---|:---|:---|

### 三、方案选择
| | 基础版 | ⭐ 推荐版 | 旗舰版 |
|:---|:---|:---|:---|
| **价格** | **{金额}元** | **{金额}元** 🔥 | **{金额}元** |
| {交付物} | ✅ | ✅ | ✅ |
| {增值项} | ❌ | ✅ | ✅ |
| {高级项} | ❌ | ❌ | ✅ |
| **适合** | 预算有限 | **性价比最优** | 追求极致 |

### 四、为什么选择我
- **{差异化}**：{具体说明+案例}

### 五、下一步
确认 → 签合同 → 付首付 → 启动
📧 {联系方式} | ⏰ 报价有效期至 {日期}
```

## 开发信（每次生成2个版本A/B测试）

**版本A（痛点切入）**：主题≤15字直击痛点 → 提及客户公司具体事实 → 类似客户案例+量化成果 → 15分钟演示？选择式提问
**版本B（成果导向）**：帮{行业}实现{量化结果} → 上月案例 → 核心方法一句话 → 15分钟演示？

**铁律**：标题≤15字 / 正文≤150字 / 选择式提问 / 不放附件链接 / 不用"亲爱的"

## 案例包装

背景（用数字）→ 问题（精确数据）→ 诊断/策略/执行 → 成果表格（≥3指标）→ 客户原话

## 服务介绍

痛点列表（"你可能遇到的这些问题"）→ 三档服务包 → 差异化优势 → 合作流程

## 商业计划书

执行摘要(200字) → 问题与机会(真实故事+TAM/SAM/SOM) → 解决方案(核心价值+护城河) → 商业模式 → 运营计划 → 财务预测 → 风险
