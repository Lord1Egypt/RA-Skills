# Five Element Treatment Principles

## When to Use
- 已完成辨证，明确病位、病性（虚/实）及脏腑传变关系
- 掌握五行生克基本规律
- 排除急症、外感病初起等需直接祛邪的情形
- 病机属于五行体系可解释范围

## Execution Steps

### 1. 依据相生关系确定治则
- **虚者补其母**：某脏虚证，通过补其母脏以养本脏
  - 示例：肝肾阴虚 → 滋水涵木；肺脾气虚 → 培土生金；脾肾阳虚 → 益火补土
- **实者泻其子**：某脏实证，通过泻其子脏以泄本脏之实
  - 示例：心肝火旺 → 肝旺泻心（清心火以泻肝）

### 2. 依据相克关系确定治则
- **抑强**：用于相乘（克太过）
  - 示例：肝旺乘脾 → 抑木扶土（疏肝/平肝 + 健脾）
- **扶弱**：用于相侮（克不及或反克）
  - 示例：脾虚致木不疏土 → 和肝兼运脾；脾壅反侮肝 → 运脾和胃（此时抑强为主）

### 3. 输出要求
根据证候性质（虚证/实证/乘证/侮证）和目标脏腑，输出：
- 具体治则（补母、泻子、抑强、扶弱）
- 对应治法名称（如滋水涵木、抑木扶土、培土制水、泻南补北等）
- 治疗方向（指导处方用药或针灸取穴）

> 注意：若仅为单一脏腑虚损而无生克失衡，或属非五行病机，则不适用本技能。
