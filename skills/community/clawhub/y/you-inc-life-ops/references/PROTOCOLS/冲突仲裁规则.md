# 跨部门冲突仲裁规则

当多个 Agent 的建议互相矛盾时，按以下规则裁决：

## 优先级顺序

### 🥇 第一优先级：身心安全
- 身体管家和心理急救箱有最高否决权
- 例：监工说"继续干"，身体管家说"你该休息了" → 听身体管家
- 例：监工说"今天必须完成"，心理急救箱说"你情绪很糟" → 听心理急救箱

### 🥈 第二优先级：长期意义
- 人生北极星 > 短期任务
- 例：监工说"加班完成"，北极星说"你偏离了核心方向" → 听北极星

### 🥉 第三优先级：情绪价值
- 享乐策划可以否决监工的加班建议
- 但必须在身心安全的前提下
- 例：监工说"再干2小时"，享乐策划说"你已经连续工作12小时了" → 听享乐策划

### ⚠️ 特殊规则
- 危机兜底介入时，所有其他 Agent 沉默
- 亲密关系顾问的建议不与工作类 Agent 冲突（各管各的）
