# 崖州区 招商常见问题解答（FAQ）
> 供崖州区政府招商工作人员参考使用 | 更新日期：2026年4月

---

## Q1：企业落户崖州湾，税收方面能享受哪些优惠？

**答**：企业可享受以下税收优惠：

1. **企业所得税 15%**：注册在崖州湾并实质性运营的鼓励类产业企业，税率从25%降至15%，节税幅度高达40%。

2. **个人所得税上限15%**：园区内工作的高端人才和紧缺人才，个人所得税超出15%的部分予以免征。

3. **进口免税**：企业进口自用生产设备、生产原辅料，均免征进口关税、增值税和消费税。

4. **资本性支出可一次性税前扣除**：设备、研发等资本性支出可一次性税前扣除，加速回收投资。

> 政策依据：《海南自由贸易港建设总体方案》及相关实施细则

## Q3：种业/南繁相关企业有哪些专项政策？

**答**：

1. **种业农科设施建设补贴**：项目投资额的45%，最高补贴300万元（须持农作物种子生产经营许可证，上年度营收≥2亿元）

2. **种业农科设施使用补贴**：实际使用费用的25%，年度上限50万元

3. **水产新品种引进**：补贴50%，新品种引进/土著品种改良上限200万元，优质高效品种育苗上限300万元

4. **菁英人才奖励**：种业农科领域专业人才每年可申报科技城菁英人才，奖励5万元/人

## Q4：深海科技企业有哪些专项政策？

**答**：

1. **深海科技独角兽落户奖励**：500万元一次性奖励（条件：入选胡润、科技部火炬中心或PitchBook独角兽榜单；承诺5年内不搬离）

2. **首台（套）重大技术装备**：销售价格的30%补贴（成套设备最高1000万元）

3. **研发平台设备采购补贴**：国家级平台补贴40%（最高500万元）；省级平台补贴30%（最高200万元）

4. **国际科研合作补贴**：实际合同费用的10%，年度上限100万元

## Q5：企业要在崖州湾享受税收优惠，需要满足什么条件？

**答**：核心要求是**实质性运营**，通常包括：
- 在崖州湾科技城内完成工商注册登记
- 主要业务及核心管理活动实际在园区内开展
- 核心管理人员在园区办公
- 纳税地在三亚市

> 注意：仅在园区注册但不实质运营的"空壳公司"不得享受税收优惠，税务部门会进行实质性认定检查。

## Q6：入驻流程是怎样的？大概需要多长时间？

**答**：标准入驻流程如下：

1. 初步洽谈（1-2周）：联系城小二 -> 企业介绍 -> 了解政策
2. 项目评审（2-4周）：提交入驻申请材料 -> 管理局组织评审
3. 签约入驻（1周）：签订入驻意向协议 -> 签订正式入驻协议
4. 工商注册（1-2周）：在园区注册登记，可享受"一站式"服务
5. 政策兑现（每年集中申报）：每年集中申报一次各类奖励补贴

通常从初步洽谈到完成注册，需要1-2个月。

## Q7：园区提供哪些场地/空间支持？

**答**：
- **办公载体**：超过16个建成投用的办公载体可供选择
- **生产厂房**：5个生产厂房，总投资规模超40亿元
- **研发场地**：科技创新团队可申请小于等于2000㎡场地，享受三年免租或800元/㎡一次性自购补贴

## Q8：园区有哪些融资支持？

**答**：

| 融资方式 | 说明 |
|---------|------|
| 专项产业基金 | 园区专项产业子基金总规模超40亿元，可对接投资 |
| 股权投资奖励 | 股权投资基金投入园区企业，可获投资额1%奖励（上限600万元） |
| 贷款风险补偿 | 金融机构贷款损失的50%由园区补偿（上限100万元） |
| 返程投资奖励 | 企业境外返程投资，奖励投资额5%（上限500万元） |
| 公开募资奖励 | 挂牌融资的1%，上限50万元 |

## Q9：外资企业来崖州湾有哪些优势？

**答**：
1. **免税进口设备**：免征进口关税、增值税和消费税
2. **跨境数据流动**：海南自贸港开展国际互联网数据交互试点
3. **人才国际化**：外籍人才工作许可负面清单管理，允许担任法定代表人
4. **更宽松准入**：跨境服务贸易负面清单，外资准入大幅放宽
5. **出海服务**：2026年成立的出海服务联合体，可为企业提供国际化经营服务
