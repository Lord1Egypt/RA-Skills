# 合规文档模板参考

本文档提供出海合规相关的标准化文档模板参考，帮助快速生成符合目标市场要求的合规文档。

---

## 1. 安全数据表（SDS）模板结构

### 基本信息区

| 字段 | 说明 | 示例 |
|------|------|------|
| 产品名称 | 产品/物质的明确标识 | AI智能分析系统 V2.0 |
| 版本号 | 文档版本 | Version 2.1 |
| 更新日期 | 文档编制/更新日期 | 2024-06-20 |
| 编制单位 | 负责编制的企业名称 | 云旅智能体超市 |

### 成分组成信息

| 字段 | 说明 | 示例 |
|------|------|------|
| 物质名称 | 化学物质或混合物名称 | 水性聚合物基质 |
| CAS号 | 化学文摘社登记号 | 7732-18-5 |
| 浓度范围 | 成分浓度 | 5-15% |
| 危害分类 | 符合CLP法规的分类 | 无危害物质 |

### 危险性概述

```
危险性类别：
- 根据CLP法规进行分类
- 包含信号词、危害说明、防范说明

紧急情况概述：
- 事故应对措施
- 主要症状描述
- 必要的急救措施
```

### 安全措施建议

| 措施类型 | 内容 | 优先级 |
|----------|------|--------|
| 预防措施 | 操作建议、个人防护 | P201, P280 |
| 响应措施 | 泄漏、火灾、急救 | P302+P352, P305 |
| 储存措施 | 储存条件、禁忌物 | P403+P233 |
| 处置措施 | 废弃方法、容器处理 | P501 |

## 2. CLP标签要素清单

### 标签组成要素

| 要素 | 要求 | 适用市场 |
|------|------|----------|
| 产品标识符 | 物质/混合物的明确名称 | EU |
| 信号词 | "危险"或"警告" | EU |
| 危害象形图 | GHS规定的图形符号 | EU, Global |
| 危害说明 | H开头的标准描述 | EU |
| 防范说明 | P开头的标准措施 | EU |
| 供应商信息 | 制造商/进口商信息 | EU |
| 补充信息 | EUH开头的补充条款 | EU |

### 常用危害说明（H代码）

| 代码 | 描述 | 适用场景 |
|------|------|----------|
| H301 | 吞咽有毒 | 化学物质 |
| H314 | 造成严重皮肤灼伤和眼损伤 | 强酸/强碱 |
| H317 | 可能引起皮肤过敏反应 | 过敏原 |
| H318 | 造成严重眼损伤 | 刺激性物质 |
| H351 | 疑似致癌 | 可疑致癌物 |
| H361 | 疑似对生育能力或胎儿造成伤害 | 生殖毒性 |

### 常用防范说明（P代码）

| 代码 | 类型 | 描述 |
|------|------|------|
| P201 | 预防 | 使用前获取特别指示说明 |
| P202 | 预防 | 在阅读并理解所有安全措施前切勿操作 |
| P261 | 预防 | 避免吸入粉尘/烟气/气体/烟雾/蒸气/喷雾 |
| P280 | 预防 | 佩戴防护手套/防护服/防护眼镜/防护面罩 |
| P302+P352 | 响应 | 如皮肤沾染：用大量水冲洗 |
| P305+P351+P338 | 响应 | 如进入眼睛：谨慎冲洗数分钟 |
