# 全国各地区劳动仲裁规则差异参考

## 一、仲裁时效规则差异

### 1.1 一般仲裁时效
- 全国统一：自知道或应当知道权利被侵害之日起 **1年**（《调解仲裁法》第27条）

### 1.2 劳动报酬特殊时效
- **全国统一：** 劳动关系存续期间因拖欠劳动报酬发生争议的，不受1年时效限制；劳动关系终止的，自终止之日起1年内提出

### 1.3 二倍工资时效差异（重要！）

| 地区 | 时效起算规则 | 依据 |
|------|-------------|------|
| 北京 | 自未签合同满一年（即入职满两年）次日起算1年 | 京高法发〔2017〕48号 |
| 上海 | 自二倍工资支付义务产生之月起按月分别计算 | 沪高法〔2009〕73号 |
| 广东 | 自二倍工资支付义务产生之日起按月计算 | 粤高法发〔2018〕2号 |
| 江苏 | 自二倍工资支付义务产生之日起按月计算 | 苏高法审委〔2011〕14号 |
| 浙江 | 自未签合同满一年次日起算1年 | 浙高法民一庭解答 |
| 深圳 | 自二倍工资支付义务产生之月起按月计算 | 深中院裁判指引 |
| 成都/四川 | 自权利被侵害之日起按月计算 | 川高法指导意见 |

> **研判要点：** 二倍工资时效是最容易因地区差异导致败诉的事项，务必查询当地最新裁判口径。

## 二、调解前置程序差异

| 地区 | 是否要求调解前置 | 说明 |
|------|----------------|------|
| 北京 | 否 | 可直接申请仲裁 |
| 上海 | 部分要求 | 部分区仲裁委要求先调解 |
| 广东 | 否 | 可直接申请仲裁 |
| 深圳 | 是（鼓励） | 设有先行调解程序 |
| 浙江 | 是 | 浙江推行调解前置 |
| 江苏 | 否 | 可直接申请仲裁 |
| 四川 | 否 | 可直接申请仲裁 |
| 山东 | 部分 | 集体争议要求调解前置 |
| 河南 | 否 | 可直接申请仲裁 |
| 重庆 | 否 | 可直接申请仲裁 |

> **提示：** 即使不要求调解前置，仲裁委收到申请后通常也会先行调解。

## 三、管辖规则差异

### 3.1 地域管辖
- **全国统一：** 用人单位所在地或劳动合同履行地的仲裁委管辖（《调解仲裁法》第21条）
- 劳动者可选择其中之一

### 3.2 级别管辖差异

| 地区 | 区/县级仲裁委管辖范围 | 市级仲裁委管辖范围 |
|------|---------------------|-------------------|
| 北京 | 一般案件 | 中央/市属单位、外资企业等 |
| 上海 | 一般案件 | 部分大型企业案件 |
| 深圳 | 一般案件 | 市属单位案件 |
| 广州 | 一般案件 | 省属/市属单位案件 |
| 杭州 | 一般案件 | 市属单位案件 |
| 成都 | 一般案件 | 省属单位案件 |

### 3.3 管辖争议处理
- 双方分别向不同仲裁委申请的，由劳动合同履行地仲裁委管辖
- 管辖异议应在答辩期内提出

## 四、立案材料要求差异

### 4.1 仲裁申请书份数

| 地区 | 份数要求 |
|------|---------|
| 多数地区 | 被申请人数量 + 1份 |
| 北京 | 被申请人数量 + 2份（多一份留存） |
| 深圳 | 被申请人数量 + 1份 |
| 上海 | 被申请人数量 + 1份 |

### 4.2 身份证明要求

| 地区 | 要求 |
|------|------|
| 多数地区 | 身份证复印件2份 |
| 部分地区 | 需提供居住证/暂住证 |
| 涉外 | 护照及翻译件 |

### 4.3 用人单位信息证明
- 多数地区要求提供用人单位工商注册信息
- 可从"国家企业信用信息公示系统"免费下载

## 五、举证规则差异

### 5.1 举证期限

| 地区 | 举证期限 | 说明 |
|------|---------|------|
| 北京 | 开庭前 | 逾期可能不被采纳 |
| 上海 | 庭审中 | 庭审时提交 |
| 广东 | 庭审中 | 庭审时提交 |
| 浙江 | 开庭前3日 | 逾期需说明理由 |
| 江苏 | 庭审中 | 庭审时提交 |

### 5.2 举证责任倒置

**全国统一规则：**
- 用人单位作出的开除、除名、辞退、解除劳动合同、减少劳动报酬、计算劳动者工作年限等决定而发生的劳动争议，**用人单位负举证责任**（《调解仲裁法》第6条、《最高法司法解释》）

**地区特色：**
- 深圳：加班事实的举证责任，如用人单位掌握考勤记录但不提供，推定劳动者主张成立
- 北京：工资支付记录由用人单位保存，用人单位应提供2年内的工资支付凭证

## 六、特定请求的地区裁判口径差异

### 6.1 经济补偿金/赔偿金计算基数

| 地区 | 计算基数 | 说明 |
|------|---------|------|
| 全国统一 | 离职前12个月平均工资（税前） | 含奖金、津贴、补贴 |
| 北京 | 含年终奖（按月折算） | 离职当年度年终奖折算计入 |
| 上海 | 离职前12个月正常出勤工资 | 不含加班费（部分案件） |
| 广东 | 离职前12个月全部收入 | 含加班费 |
| 深圳 | 离职前12个月全部收入 | 含加班费 |

### 6.2 三倍封顶规则
- **全国统一：** 劳动者月工资超过当地上年度职工月平均工资三倍的，按三倍支付，年限最高12年
- 各地"上年度职工月平均工资"不同，每年更新

### 6.3 未休年休假工资

| 地区 | 裁判口径 |
|------|---------|
| 北京 | 支持未休年休假300%工资（含已发正常工资，额外付200%） |
| 上海 | 同上 |
| 广东 | 同上 |
| 多数地区 | 用人单位安排休假但劳动者未休的，不支持额外工资 |

### 6.4 加班费

| 地区 | 特殊规则 |
|------|---------|
| 北京 | 用人单位保存2年工资支付记录，2年内加班费举证责任在单位 |
| 上海 | 劳动者需初步证明加班事实 |
| 广东 | 如有考勤记录由单位保管，单位不提供则推定劳动者主张成立 |
| 深圳 | 约定"包薪制"如不低于最低工资标准可能被认可 |
| 江苏 | 不定时工作制需经审批，未经审批的按标准工时计算加班费 |

## 七、法律援助差异

| 地区 | 劳动争议法律援助条件 |
|------|---------------------|
| 全国 | 追索劳动报酬案件可申请法律援助，经济困难标准较宽 |
| 北京 | 劳动报酬争议免经济审查 |
| 上海 | 劳动报酬争议免经济审查 |
| 深圳 | 劳动争议法律援助"绿色通道" |
| 浙江 | 劳动争议可在线申请法律援助 |
| 多数地区 | 可拨打12348咨询法律援助 |

---

## 八、查询当地最新规定的渠道

1. **12333热线** — 人社政策咨询热线（全国统一）
2. **当地人社局官网** — 查看最新仲裁规则和通知
3. **中国裁判文书网** — 检索当地同类案件裁判口径
4. **当地12348公共法律服务热线** — 法律援助咨询
5. **当地工会法律服务** — 免费法律咨询和援助

> **重要提醒：** 各地裁判口径和规则会更新，使用本手册时应结合案件发生时的最新规定进行研判。建议在研判阶段通过12333或当地仲裁委确认关键规则。
