# 常用心理学量表清单与获取方式

> 注：以下为常见心理学研究工具，版权情况各异。使用前请确认授权情况。

---

## 1. 概念与使用前提

**量表获取渠道：**
- 中文版公开发表论文（通常附录或联系作者）
- 心理学期刊官网（如《心理学报》）
- 授权邮箱（邮件联系作者）
- 部分商业平台（需付费购买）

**信效度要求：**
- 克隆巴赫 α ≥ .70（可接受）
- 克隆巴赫 α ≥ .80（较好）
- 验证性因素分析（CFI/TLI/RMSEA）达到标准

**量表使用前检查：**
- [ ] 是否有中文版？
- [ ] 是否在目标群体中做过信效度检验？
- [ ] 版权是否已获取/授权？
- [ ] 计分方式是否清楚？（哪些题反向？维度如何计算？）

## 3. 量表数据处理注意事项

### 反向计分题目
收集数据前，必须标注所有反向计分题目：

| 量表 | 反向题编号 | 处理方式 |
|------|-----------|---------|
| STAI（部分版本） | 21, 26, 27, 30, 33, 35, 37, 39, 40 | 7 - 原始分 |
| SDS | 2, 5, 6, 11, 12, 14, 15, 17, 18, 20 | 4 - 原始分 |
| CES-D | 4, 8, 12, 16 | 4 - 原始分 |
| PANAS（正性情感） | 无 | - |
| ERQ | 4, 7, 10 | 7 - 原始分 |

**常见错误：** 漏做反向计分处理导致结果虚假相关/差异。

### 缺失值处理规则

**规则建议：**
- 单题缺失：量表总分用有效题目的均值替代（前提：缺失 < 20%）
- 整题量表缺失：该被试删除
- 超过 20% 题目缺失：该被试删除

**操作：**
```python
# Python 示例
# 若某被试在某量表上缺失题目超过 20%，则删除
threshold = len(scale_items) * 0.2
missing_count = df[scale_items].isnull().sum(axis=1)
df = df[missing_count < threshold]
```

## 4. 常见陷阱

| 陷阱 | 问题 | 规避方法 |
|------|------|---------|
| 直接用英文量表 | 中文被试理解偏差 | 使用已有中文版 |
| 只看总分不看维度 | 忽略量表结构 | 同时报告总分+维度 |
| 漏做反向计分 | 数据分析方向错误 | 数据清洗第一步处理 |
| 多维度量表只用一个维度 | 浪费数据价值 | 预设分析计划 |
| 不知道量表版权 | 论文发表有风险 | 提前联系作者获取授权 |
