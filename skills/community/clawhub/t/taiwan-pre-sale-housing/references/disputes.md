# 常見預售屋糾紛與處理

## 主要糾紛類型

| 類型 | 發生階段 | 頻率 |
|------|----------|:----:|
| 工程延宕/逾期 | 施工期間 | ⭐⭐⭐ |
| 建材設備不符 | 驗屋階段 | ⭐⭐⭐ |
| 房屋瑕疵/漏水 | 驗屋/交屋 | ⭐⭐⭐ |
| 面積誤差/坪數不足 | 交屋前 | ⭐⭐ |
| 貸款成數不足 | 對保階段 | ⭐⭐⭐ |
| 公設與廣告不符 | 交屋後 | ⭐⭐ |
| 社區管理爭議 | 交屋後 | ⭐ |

---

## 1. 工程延宕／逾期交屋

### 買方權利
- 依契約：每日按**已繳價款萬分之五**計算違約金
- 逾期**3個月**：買方可解除契約（依內政部版）
- 建商應返還已繳價款+違約金

### 處理流程
```
催告（存證信函）→ 協商 → 調解 → 訴訟
```

### 實務建議
- 先確認延宕原因（天災/工安/缺工）
- 可接受者：協商延長期限+違約金
- 不可接受者：解約要求返還全額+違約金

---

## 2. 建材設備不符

### 常見不符情形
| 項目 | 合約載明 | 實際裝設 |
|------|----------|----------|
| 衛浴 | TOTO | 和成 |
| 地板 | 80×80拋光磚 | 60×60石英磚 |
| 廚具 | 櫻花 | 豪山 |
| 門窗 | 大同氣密窗 | 一般鋁窗 |
| 電梯 | 三菱 | 永大 |

### 處理方式
1. 比較合約條文（注意是否有「同等品」字樣）
2. 「同等品」需功能/材質/品牌等級相當
3. 明顯降級 → 要求更換或折價賠償
4. 協商不成 → 消保官調解

---

## 3. 房屋瑕疵

### 常見瑕疵
- 漏水（最常見！）
- 地磚空心/隆起
- 牆面裂縫
- 排水不通
- 窗框滲水

### 處理SOP
```
發現瑕疵 → 書面通知建商（附照片）→ 修繕 → 複驗
→ 未改善 → 再通知 → 消保官 → 扣留交屋款
```

### 重要權利
- **交屋保留款**：至少總價**5%**，修繕完成前絕不支付
- 結構保固：15年
- 防水/設備保固：1年

---

## 4. 面積誤差

### 處理原則（依內政部版）
- 誤差**<1%**：無息找補（多退少補）
- 誤差**1%~3%**：按比例找補
- 誤差**≥3%**：買方**可解除契約**

### 面積計算
```
總面積誤差率 = (實際面積 − 合約面積) ÷ 合約面積
```

---

## 5. 貸款成數不足

### 三種情況

| 原因 | 責任歸屬 | 處理 |
|------|:--------:|------|
| 買方信用不良 | 買方負責 | 補足差額或由保證人貸 |
| 銀行鑑價低於成交價 | 市場因素 | 契約通常約定由買方補足 |
| 銀行政策緊縮 | 不可歸責 | 可爭取解約退還全額 |

### 預防
- 簽約前先確認聯徵信用分數
- 預留貸款短差之自備款（1~2成空間）
- 簽約時注意「貸款不足處理」條款

---

## 6. 公設與廣告不符

### 法規保護
- 廣告文宣、樣品屋、平面圖均具有**契約效力**
- 依公平交易法及消費者保護法

### 常見爭議
- 公設項目減少（三溫暖變花園）
- 公設品質降低
- 公設空間被變更用途
- 中庭/景觀與廣告不符

### 處理
- 保留所有廣告文宣、型錄、樣品屋照片
- 請求減少價金或損害賠償
- 情節重大：可解除契約

---

## 爭議處理管道

| 管道 | 時機 | 優點 | 缺點 |
|------|------|------|------|
| 建商客服 | 第一線 | 最快 | 可能無效 |
| 消保官申訴 | 協商破裂 | 免費 | 約1~2個月 |
| 消費者保護團體 | 上述無效 | 專業協助 | 需時間 |
| 縣市調解委員會 | 訴訟前 | 免費/有法律效力 | 雙方需同意 |
| 訴訟（簡易/通常） | 最終手段 | 有強制力 | 時間/費用高 |

## 避險清單（簽約前）

- [ ] 查建商過往案例（網路評價/消保案件）
- [ ] 查建造執照及銷售許可
- [ ] 合約對照內政部版定型化契約
- [ ] 確認貸款不足條款公平
- [ ] 確認公設比合理（<35%）
- [ ] 確認建材設備明確非僅供參考
- [ ] 確認保固條款符合規範
- [ ] 保留所有廣告與型錄
