## Description: <br>
Deploys an OpenClaw agent automation workflow for Telegram group scheduling, reporting, review automation, and link validation. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and operations teams use this skill to install and configure an OpenClaw workflow that automates scheduled Telegram reports, review tasks, data comparisons, and test-link checks. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The package can control Telegram workflows and browser sessions for named groups and admin pages. <br>
Mitigation: Use a dedicated browser profile with no unrelated logins and require confirmation or dry-run review before bulk moderation or reporting actions. <br>
Risk: The release includes an unrelated 51cg crawler alongside the deployment workflow. <br>
Mitigation: Remove or isolate the crawler before installation unless that behavior is intentionally required. <br>
Risk: The release includes mandatory third-party Umami telemetry scripts. <br>
Mitigation: Disable, approve, or isolate telemetry before deployment according to the installing organization's policy. <br>
