## 3. 维修成本决策树

```
电脑出现故障
│
├─ 维修成本 > 电脑残值？
│  ├─ 是 → 建议换新（不值得修）
│  │  例：老电脑（i3-4代）维修要500元，但电脑只值300元
│  │
│  └─ 否 → 继续评估
│
├─ 是否在保修期内？
│  ├─ 是 → 送官方售后（免费）
│  │  └─ 注意：不要自己拆机（保修贴纸破损不保修）
│  │
│  └─ 否 → 第三方维修 or DIY
│
├─ DIY难度评估
│  ├─ 简单（换内存/硬盘/电池） → 自己动手（省手工费）
│  ├─ 中等（换硅脂/清灰） → YouTube学一下，可以尝试
│  └─ 困难（换屏幕/主板维修） → 送修（避免搞坏）
│
└─ 维修价值评估
   ├─ 电脑使用年限 < 3年 → 值得修
   ├─ 电脑使用年限 3-5年 → 看维修成本（<500元可修）
   └─ 电脑使用年限 > 5年 → 不建议修（直接换新）
```

## 4. 配件购买渠道对比

| 渠道 | 优点 | 缺点 | 推荐指数 |
|------|------|------|----------|
| **京东自营** | 正品保障，次日达，售后好 | 价格略高 | ⭐⭐⭐⭐⭐ |
| **天猫官方旗舰店** | 正品保障，活动多 | 发货慢 | ⭐⭐⭐⭐ |
| **淘宝** | 价格便宜，选择多 | 假货多，售后差 | ⭐⭐ |
| **拼多多** | 很便宜 | 假货极多，不推荐买配件 | ⭐ |
| **电脑城** | 可现场试机，立即可取 | 坑多，容易买贵 | ⭐⭐ |
| **闲鱼** | 二手便宜 | 无保修，容易买到坏的 | ⭐⭐ |

**购买建议：**
- 内存/硬盘/电源 → 优先京东自营（售后重要）
- 显卡/CPU → 京东自营或天猫旗舰店
- 二手配件 → 闲鱼（但要会辨别，新手不推荐）
- 紧急需求 → 电脑城（但要会砍价，标价×0.7是底价）

## 8. 维修决策建议

### 8.1 值得维修的情况

- ✅ 电脑使用年限 < 3年
- ✅ 维修成本 < 电脑残值的50%
- ✅ 只是单一配件故障（内存/硬盘/电源）
- ✅ 官方保修期内（免费修）

### 8.2 不建议维修的情况

- ❌ 电脑使用年限 > 5年
- ❌ 维修成本 > 电脑残值
- ❌ 主板 + CPU 同时故障（不如换新）
- ❌ 苹果MacBook（维修太贵，不如卖二手换新）

### 8.3 维修 vs 换新计算

```
举例：联想笔记本，2020年购买，现价1500元
- 换屏幕：800元（维修后价值1500元，净损失800元）
- 换主板：1200元（维修后价值1500元，净损失1200元）

决策：
- 换屏幕：800 < 1500×50% = 750？ 否 → 不建议修，直接卖二手（约500元）+ 买新电脑
- 换主板：1200 > 750 → 绝对不建议修
```

**最后提醒**：
1. 维修前先备份数据（重要！）
2. 多问几家维修店，对比价格
3. 保留维修凭证（发票/收据）
4. 问清楚保修期（一般维修后保修1-3个月）
5. 如果不确定，优先选择官方售后（虽然贵，但有保障）
