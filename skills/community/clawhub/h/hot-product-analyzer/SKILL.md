# 电商爆款拆解分析助手

## 概述

输入商品名称或类目（如"蓝牙耳机"、"扫地机器人"、"防晒霜"），自动从多个主流电商平台搜索热门/爆款商品，拆解共性特征，总结爆款规律，生成交互式HTML可视化分析报告。

## 工作流

### Phase 1: 信息收集 — 多平台爆款搜索

对用户输入的商品/类目名称，在以下平台进行平行搜索：

1. **淘宝/天猫** — 搜索 `{产品名} 热销 爆款 排行榜`
2. **京东** — 搜索 `{产品名} 销量排行 热卖`
3. **拼多多** — 搜索 `{产品名} 爆款 热卖`
4. **抖音电商** — 搜索 `{产品名} 热销 爆款 带货`
5. **综合选品分析** — 搜索 `{产品名} 选品分析 市场分析 趋势`

使用 `WebSearch` 工具对每个平台进行搜索。每条搜索需提取：
- 商品名称（含核心关键词）
- 价格（实际售价，排除虚高标价）
- 销量数据（月销/累计）
- 评分/好评率
- 核心卖点/宣传语
- 主图/视频风格描述

**搜索策略**：
- 每个平台搜索至少获取 3-5 个爆款商品数据
- 优先采集有具体销量数字的商品
- 注意区分广告位和自然排名商品
- 对于无法直接获取的数据，根据搜索片段合理推断

### Phase 2: 特征拆解

对收集到的爆款商品进行多维度拆解：

#### 2.1 标题关键词分析
- 提取所有爆款标题中的高频关键词
- 分析关键词组合模式（功效词+场景词+人群词+品牌词）
- 统计关键词出现频次，排序生成Top10关键词

#### 2.2 价格带分析
- 统计爆款商品的价格分布
- 识别"黄金价格带"（销量最高的价格区间）
- 分析价格与销量的关系

#### 2.3 卖点分析
- 提取爆款商品的核心卖点/宣传语
- 分类：功能型卖点、情感型卖点、性价比型卖点、社交型卖点
- 总结该类目最有效的卖点类型

#### 2.4 视觉风格分析
- 分析爆款商品的视觉呈现风格（简约/科技/温馨/奢华等）
- 识别主图设计模式（产品特写/场景使用/对比展示/促销信息）
- 总结该类目主流视觉策略

#### 2.5 评价/口碑分析
- 提取爆款商品的用户评价高频词
- 识别好评关键词和差评关键词
- 总结消费者最看重的产品属性

### Phase 3: 规律总结

基于 Phase 2 的拆解数据，总结该类目的爆款规律：

1. **定价规律** — 什么价格区间最受欢迎？低价冲量还是中高价做利润？
2. **标题规律** — 什么样的标题结构点击率最高？
3. **卖点规律** — 消费者最关心什么？什么卖点最打动人？
4. **视觉规律** — 什么风格的主图最容易吸引点击？
5. **竞争格局** — 品牌集中度如何？新品牌还有机会吗？
6. **选品建议** — 如果要进入这个类目，应该选什么定位的产品？

### Phase 4: 生成HTML报告

基于 `assets/report_template.html` 模板生成最终报告。

**报告结构**：
1. **头部** — 类目名称、分析时间、数据来源说明
2. **类目概览卡片** — 市场规模、价格带、品牌数、竞争度
3. **爆款商品对比表** — 跨平台爆款商品详情对比
4. **价格分布图** — 柱状图展示价格带分布
5. **标题关键词排行** — Top关键词展示
6. **核心卖点分析** — 分类展示卖点及效果评估
7. **视觉风格分析** — 展示主流主图风格
8. **爆款规律总结** — 6大维度的规律总结
9. **选品行动指南** — 可操作的建议

**技术要求**：
- 报告使用 Chart.js CDN 生成图表
- 响应式设计，支持桌面端和移动端
- 所有样式内联，单文件交付
- 颜色方案使用电商行业专业配色

## 注意事项

1. **数据时效性**：搜索到的价格和销量数据为当前时刻的快照，可能随时间变化
2. **平台差异**：不同平台的销量统计口径不同（淘宝月销、京东评价数、拼多多已拼数）
3. **反爬限制**：电商平台可能有反爬机制，如无法直接读取页面，通过搜索片段提取关键信息
4. **数据精度**：价格和销量数据可能存在误差，报告中体现"估算"标注
5. **平台覆盖**：至少覆盖3个平台的数据，如搜索不到则不强制展示
6. **报告语言**：中文为主，保留电商行业常用的英文术语（如 SKU、ROI、CTR）
