# 香港上市公司并购制度

## 一、《公司收购及合并守则》（Takeovers Code）

### 1.1 强制全面要约触发规则

根据香港《公司收购、合并及股份回购守则》（2023年9月29日修订），任何人（单独或与一致行动人）取得上市公司投票权达到**35%**时，须向该公司所有股东发出全面要约。

#### 触发条件详解

| 要素 | 说明 |
|------|------|
| 持股比例 | 取得35%以上投票权 |
| 计算基数 | 投票权股份（具有表决权的普通股） |
| 一致行动人 | 需合并计算持股 |

#### 2023年修订要点

| 修订项 | 新规内容 |
|--------|----------|
| 近亲定义扩大 | 祖父母/外祖父母/孙子女/外孙子女；兄弟姊妹及其配偶/子女 |
| 投票权定义 | 修订为"可在股东大会上行使的投票权"，不论是否受限制 |
| 豁免情形 | 继承、法院命令等非自愿取得可申请豁免 |

### 1.2 要约收购核心规则

#### 要约价格要求
- 应支付予受要约公司同一类别股份持有人的价格须相等
- 不得低于要约人在提示性公告发布日前6个月内为取得该类股份所支付的最高价格

#### 要约期间
- 要约有效期不得少于**60天**
- 要约人可在要约文件发出后**第21天**或之前发出修订要约
- 超过4个月后不得维持要约

#### 接纳与撤回
- 接纳：股东可在要约结束前任何时间接受
- 撤回：接纳人在要约结束时仍未宣布撤回，接受即为最终

## 二、《上市规则》第14章：须予公布的交易

### 2.1 交易分类标准

| 交易类型 | 规模测试标准 | 股东批准 | 公告要求 |
|----------|--------------|----------|----------|
| 股份交易 | 任何百分比率5%-25% | 一般不需 | 通知 |
| 须予披露的交易 | 任何百分比率25%-100% | 般不需（创业板规则不同） | 通知 |
| 主要交易 | 任何百分比率≥100% | 需要 | 通函+股东批准 |
| 非常重大的收购事项 | 资产/利润比率≥100%且涉及发行股份 | 需要 | 通函+股东批准 |
| 反收购行动 | 构成RTO | 需独立股东批准 | 上市文件 |

### 2.2 规模测试的计算

百分比率计算：
```
资产比率 = 所收购资产 ÷ 上市公司资产
收益比率 = 所收购资产应占收益 ÷ 上市公司收益
代价比率 = 代价 ÷ 上市公司市值
股本比率 = 所发行代价股份 ÷ 上市公司已发行股本
```

## 三、港交所并购审核流程

### 3.1 与A股的主要差异

| 对比维度 | 香港 | A股 |
|----------|------|-----|
| 审核主体 | 港交所+证监会分工 | 交易所+证监会 |
| 审核周期 | 通常4-8周 | 3-6个月 |
| 披露时点 | 需及时披露 | 停牌+分阶段披露 |
| 股东投票 | 独立股东投票机制 | 股东大会+分类表决 |
| 要约制度 | 强制要约规则 | 自愿要约为主 |

### 3.2 香港证监会与港交所分工

| 机构 | 职责 |
|------|------|
| 香港证监会 | 收购守则执行、强制要约审批、独立财务顾问监管 |
| 港交所 | 上市规则执行、交易类审批、信息披露监督 |

### 3.3 披露要求

根据《上市规则》第13-14章：
- 须予公布的交易需在合理可行情况下尽快公布
- 需发布新闻稿及在港交所网站刊登公告
- 主要交易/非常重大收购需发出股东通函

## 四、借壳上市与反收购行动规则

### 4.1 反收购行动的定义（上市规则第14.06B条）

港交所将拟进行反收购行动的上市发行人**当作新上市申请人处理**。

#### 明确测试（2019年修订）

| 情形 | 标准 |
|------|------|
| 同时发生 | 非常重大收购+控制权变动 |
| 24个月内（2024年改为36个月） | 控制权变动后36个月内向控股股东进行非常重大收购 |
| 出售限制 | 控制权变动时或其后36个月内不得出售原有业务 |

#### 原则为本测试

1. 交易规模相对发行人规模
2. 目标资产质量
3. 发行人业务性质及规模
4. 主营业务是否出现根本转变
5. 控制权或实际控制权变动
6. 一连串的交易及/或安排（36个月内）

### 4.2 极端交易（2019年新增）

适用条件：
- 发行人须有规模庞大的主要业务
- 或发行人长时间（≥36个月）受同一人控制，且交易不导致控制权变动

### 4.3 借壳新规（2019年10月1日生效）

**新增规则**：
- 第14.06D条：禁止大规模发行证券换取现金规避新上市规定
- 第13.24条修订：发行人须有足够业务运作**且**拥有相当价值资产
- 12个月过渡期给不符合规定的发行人

### 4.4 反收购行动合规要求

1. 收购目标须符合新上市条件（第8.04、8.05条）
2. 须在股东大会上获股东批准
3. 需发出上市文件
4. 需缴付首次上市费

## 五、港股私有化规则

### 5.1 私有化方式对比

| 方式 | 通过条件 | 特点 |
|------|----------|------|
| **协议安排** | 75%以上独立股东赞成；反对票≤10% | 可强制排除异议股东 |
| **全面要约** | 90%以上独立股东接受 | 完成后可强制收购剩余股份 |
| 吸收合并 | 75%以上独立股东赞成；反对票≤10% | 需债权人同意 |
| 股份回购 | 75%以上独立股东赞成；反对票≤10% | 适合A+H股注销H股 |

### 5.2 私有化核心流程

```
┌──────────────────┐
│   准备阶段        │  尽职调查、制定方案、聘请财务顾问
└────────┬─────────┘
         ↓
┌──────────────────┐
│   申请清洗豁免   │  （如需）向证监会申请清洗交易豁免
└────────┬─────────┘
         ↓
┌──────────────────┐
│   公布要约意向   │  发布公告、发送通函
└────────┬─────────┘
         ↓
┌──────────────────┐
│   法院批准       │  协议安排需法院批准
└────────┬─────────┘
         ↓
┌──────────────────┐
│   完成交割       │  撤销上市地位
└──────────────────┘
```

### 5.3 2025年私有化特点

- **活跃度提升**：2025年以来已有19家提出私有化申请
- **溢价提高**：成功案例平均溢价率30-50%，部分超100%
- **失败率增加**：中小股东维权意识增强，金利来、复宏汉霖等失败

### 5.4 典型案例：恒生银行私有化（2025年）

| 项目 | 内容 |
|------|------|
| 要约人 | 汇丰亚太 |
| 方式 | 协议安排 |
| 价格 | 155港元/股（较收盘价溢价30%） |
| 总估值 | 约2903亿港元 |
| 现金对价 | 约1062亿港元 |

## 六、要约收购案例分析

### 6.1 沿海绿色家园强制要约（2025年）

| 项目 | 内容 |
|------|------|
| 触发规则 | Rule 26.1（取得36.93%触发强制要约） |
| 交易结构 | 出售153,126,197股（占36.93%）+销售贷款 |
| 要约类型 | 强制性有条件现金要约 |
| 披露义务 | Rule 3.7（可能强制要约月度更新） |

### 6.2 亿腾医药借壳嘉和生物（2025年）

- **方式**：换股合并
- **结果**：认定为反向收购，按新上市程序审批并最终成功
- **意义**：18A领域借壳成功案例，显示只要具备产业协同可获批

## 七、2025年港股并购新规要点

### 7.1 强制要约门槛调整（2023年修订版）

| 项目 | 修订前 | 修订后 |
|------|--------|--------|
| 触发线 | 30% | **35%** |
| 近亲范围 | 配偶、子女 | 扩展至祖父母、兄弟姊妹等 |
| 一致行动推定 | 一般推定 | 扩大至关联基金 |

### 7.2 库藏股规则（2025年4月生效）

- 上市公司可合法持有回购股份作为库藏股
- 为并购换股提供更多灵活性
- 可用库藏股满足公众持股量要求

### 7.3 协议收购新规

- 75%无利害关系股东批准要求不变
- 新增**电子投票**选项，缩短流程约两周

### 7.4 财务资源确认

- 要约人须确保有足够财务资源直至完成要约
- 2023年修订后确认规则更加严格

## 八、红筹股并购特殊考量

### 8.1 红筹架构概述

红筹股：中国境外注册、主要业务在境内、香港上市的公司

### 8.2 并购涉及的主要审批

| 审批事项 | 主管部门 | 适用情形 |
|----------|----------|----------|
| 境外投资备案 | 商务部/发改委 | 境外投资 |
| 返程投资登记 | 外管局 | 资金跨境流动 |
| 国家安全审查 | 网信办等 | 重要行业/数据安全 |
| 外商投资安全审查 | 发改委/商务部 | 关键领域 |

### 8.3 港股与A股联动并购

**典型模式**：
- A股公司收购H股上市公司（需考虑公众持股量）
- H股私有化同时保留A股上市地位
- 分拆上市与整体上市安排
