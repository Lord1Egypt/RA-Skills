# 交通方式对比参考

## 选择指南

### 飞机 ✈️

**适合场景：**
- 长途旅行（>800 公里）
- 时间紧迫
- 预算充足

**优点：**
- 速度最快
- 长途舒适度高

**缺点：**
- 需要提前到达机场（国内航班建议提前 2 小时）
- 受天气影响大，可能延误
- 机场通常离市区较远

**预订建议：**
- 出差：选择 7:00-21:00 之间的航班，避免红眼航班
- 旅游：可以考慮早班机或晚班机，价格更优

---

### 高铁 🚄

**适合场景：**
- 中短途旅行（200-800 公里）
- 商务出行（准点率高）
- 携带行李较多

**优点：**
- 准点率极高（>98%）
- 车站通常在市区，交通便利
- 无需提前太久到达（提前 30 分钟即可）
- 可以正常使用手机和网络

**缺点：**
- 长途旅行时间较长
- 节假日票源紧张

**预订建议：**
- 出差：优先选择 G 字头，时间选择 8:00-20:00
- 旅游：可以选择 D 字头，价格更实惠

---

### 自驾 🚗

**适合场景：**
- 短途旅行（<300 公里）
- 多人出行（分摊成本）
- 目的地交通不便

**优点：**
- 灵活自由，随时出发
- 行李空间大
- 目的地交通方便

**缺点：**
- 长途驾驶疲劳
- 停车费用高
- 限行政策限制

**预订建议：**
- 使用高德地图 `route` 命令规划路线，预估时间和过路费
- 提前查询目的地停车信息

---

### 打车/网约车 🚕

**适合场景：**
- 市内交通
- 短途跨城（<50 公里）
- 携带大件行李

**优点：**
- 门到门服务
- 无需考虑停车

**缺点：**
- 成本较高
- 长途可能难找返程车

**预订建议：**
- 使用高德地图 `taxi` 命令生成打车链接
- 长途建议提前预约

---

## 决策树

```
出行距离？
├─ <50 公里 → 打车/自驾
├─ 50-300 公里 → 高铁/自驾
├─ 300-800 公里 → 高铁（优先）/飞机
└─ >800 公里 → 飞机（优先）/高铁

出行目的？
├─ 出差 → 高铁（准点）/飞机（时间合适）
├─ 旅游 → 飞机（价格优）/高铁（沿途风景）
└─ 探亲 → 高铁（舒适）/自驾（灵活）

预算考虑？
├─ 充足 → 飞机商务舱/高铁商务座
├─ 中等 → 飞机经济舱/高铁二等座
└─ 紧张 → 高铁二等座/自驾（多人分摊）
```

---

## 实用提示

### 出差贴士
1. 优先选择可退改的票种，应对临时变化
2. 保留所有票据，方便报销
3. 选择含早餐的酒店，节省时间
4. 提前查好酒店到办事地点的路线

### 旅游贴士
1. 淡季出行价格更优
2. 提前 2-4 周预订机票酒店
3. 选择可取消的预订，保持灵活
4. 查好当地天气，准备合适衣物

### 家庭出行贴士
1. 选择亲子酒店，设施更完善
2. 避免红眼航班和深夜到达
3. 预留充足的中转时间
4. 准备常用药品和儿童用品
