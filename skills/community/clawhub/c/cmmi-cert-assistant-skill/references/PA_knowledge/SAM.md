# SAM — 供应商协议管理（CMMI V3.0 Domain-specific PA）

## PA 基本信息
- **缩写**：SAM
- **英文**：Supplier Agreement Management
- **中文**：供应商协议管理
- **所属**：Domain-specific PA（领域特定实践域）
- **适用领域**：供应链管理、采购管理
- **对应 V2.0**：SAM（供应商协议管理）

---

## CMMI V3.0 要求要点

### 核心目的
管理供应商协议，确保供应商交付满足组织需求。

### 关键实践（Specific Practices）
| 实践编号 | 实践内容 | 说明 |
|---------|---------|------|
| SAM 1.1 | 识别供应商需求 | 明确需要外部采购的产品/服务 |
| SAM 1.2 | 选择供应商 | 评估并选择合适的供应商 |
| SAM 1.3 | 建立供应商协议 | 签订正式协议（合同、SLA） |
| SAM 1.4 | 执行供应商协议 | 监督供应商履约情况 |
| SAM 2.1 | 管理供应商绩效 | 监控供应商交付质量和时效 |
| SAM 2.2 | 管理供应商变更 | 控制供应商变更的影响 |
| SAM 3.1 | 改进供应商关系 | 基于绩效数据优化供应商关系 |

### 通用实践（Generic Practices）
- GP 1.1 执行特定实践
- GP 2.1 建立组织政策
- GP 2.2 策划过程
- GP 2.3 提供资源
- GP 2.4 分配责任
- GP 2.5 培训人员
- GP 2.6 管理配置
- GP 2.7 识别并记录干系人需求和接口
- GP 2.8 监督和控制过程
- GP 2.9 客观评估符合性
- GP 2.10 高层审查

## 评估访谈常见问题（模式D）

### 针对采购人员的问题
1. "你们如何选择和评估供应商？有评估记录吗？"
2. "供应商协议包含哪些内容？有验收标准吗？"
3. "如何监督供应商履约情况？"
4. "供应商绩效如何管理？有绩效记录吗？"

### 评分要点
- **SAM 1.1-1.4**：是否有系统的供应商选择和协议建立流程
- **SAM 2.1-2.2**：供应商履约是否有监督，绩效是否有管理
- **SAM 3.1**：是否有供应商关系改进机制
