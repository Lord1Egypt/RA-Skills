# 胃癌淋巴结转移路径与分组规则

## 何时使用
- 已确诊胃癌且需评估淋巴结转移风险
- 制定胃癌根治术的淋巴结清扫范围（D1 或 D2）
- 解读病理报告中的淋巴结转移模式
- 分析肿瘤部位与区域淋巴结转移的相关性

## 执行步骤

1. **确认输入变量**：
   - 原发肿瘤部位（如贲门、胃窦、胃体等）
   - 肿瘤浸润深度（T分期）

2. **应用“原断进”转移模式判断常规转移路径**：
   - **第一阶段（胃周区域淋巴结，1～6组）**：
     - 1：贲门右区
     - 2：贲门左区
     - 3：沿胃小弯
     - 4sa：胃短血管旁
     - 4sb：胃网膜左血管旁
     - 4d：胃网膜右血管旁
     - 5：幽门上区
     - 6：幽门下区
   - **第二阶段（血管周围向心性转移，7～16组）**：
     - 7：胃左动脉旁
     - 8a/8p：肝总动脉前/后
     - 9：腹腔动脉旁
     - 10：脾门
     - 11p/11d：近端/远端脾动脉旁
     - 12a/12p/12b：肝动脉旁、门静脉后、胆总管旁
     - 13：胰头后
     - 14v/14a：肠系膜上静脉/动脉旁
     - 15：结肠中血管旁
     - 16：腹主动脉旁（细分 a1、a2、b1、b2）

3. **识别特殊转移路径**：
   - **跳跃式转移**：跳过中间淋巴结直接转移至远端组
   - **经胸导管转移**：导致锁骨上淋巴结肿大
   - **经肝圆韧带淋巴管转移**：导致脐周淋巴结肿大

4. **结合肿瘤部位预测高危淋巴结组**：
   - 贲门癌：易转移至 1、2、3、7、110 组
   - 胃窦癌：易转移至 4d、5、6 组

5. **输出结果**：
   - 列出可能受累的淋巴结组别
   - 标注转移路径类型（常规或跳跃式）
   - 提示清扫范围建议（D1 或 D2）

> 注意：早期胃癌淋巴转移率约 20%，进展期可达 70%，转移风险与浸润深度正相关。
