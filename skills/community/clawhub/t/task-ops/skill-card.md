## Description: <br>
Task Ops is a four-module task operations framework for assessing whether domains should exist, refactoring workflows into IPO primitives, structuring domain knowledge into skill files, and executing tasks through execution, content, and innovation axes. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wangjiaocheng](https://clawhub.ai/user/wangjiaocheng) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, knowledge workers, and agent operators use this skill to decompose complex tasks, evaluate domain or workflow structure, create reusable domain payload skills, and produce structured execution plans or deliverables. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can activate broadly on generic task requests and may over-structure simple work. <br>
Mitigation: Use it when a task-operations framework, workflow refactor, domain assessment, or skill-generation workflow is explicitly desired. <br>
Risk: The skill can guide agents toward reading or writing files, using tools, running code, performing web searches, or saving generated skills. <br>
Mitigation: Require user confirmation before file operations, code execution, web searches, or saving generated skills. <br>
Risk: Generated workflows or skill files may contain incorrect, misleading, or unsafe guidance if deployed without review. <br>
Mitigation: Review and scan generated outputs before deployment, and treat assessment results as scenario-dependent guidance. <br>
