## ✨ 功能亮点

### 📅 精准计算退休时间
- 基于 **2025年最新延迟退休政策**
- 男性：60岁 → 63岁（每4个月延迟1个月）
- 女性干部：55岁 → 58岁
- 女性工人：50岁 → 55岁
- 精确到**哪年哪月**可以退休

### 💰 双模式社保计算
支持两种就业类型的社保缴费计算：

| 类型 | 适用人群 | 特点 |
|------|---------|------|
| 👔 **职工社保** | 有单位代缴的上班族 | 个人缴纳比例低（养老8%） |
| 🚀 **灵活就业** | 自由职业者、个体户 | 个人缴纳比例高（养老20%） |

### 🗺️ 多城市支持
覆盖全国 **12个主要城市** 的社保缴费基数和医保比例：

北京、上海、广州、深圳、杭州、南京、成都、武汉、西安、重庆、天津、苏州

### 📊 智能对比
选择职工社保时，自动对比灵活就业的缴费差异，让你一目了然多缴多少钱！

## 📖 使用示例

### 示例 1：职工社保（有单位）

**用户信息**：
- 出生：1990年6月15日
- 性别：男
- 月薪：10,000元
- 城市：上海
- 就业类型：职工社保（单位代缴）

**计算结果**：
```
📅 退休时间：2053年6月（63岁）
⏱️ 距离退休：约 27年2个月

💰 每月社保缴费：
├── 养老保险：¥800（8%）
├── 医疗保险：¥200（2%）
├── 失业保险：¥50（0.5%）
├── 住房公积金：¥700（7%）
└── 合计：¥1,750/月

📊 退休前总缴费：约 ¥57万元
```

---

### 示例 2：灵活就业（自由职业）

**对话流程**：

助手：您在上海灵活就业，可以选择以下缴费档次：

| 档次 | 缴费基数 | 每月费用 | 退休前总计 | 适合人群 |
|-----|---------|---------|-----------|---------|
| 🥉 最低档 | ¥7,384 | ¥2,289 | ¥52.6万 | 收入不稳定 |
| 🥈 中低档 | ¥10,000 | ¥3,100 | ¥71.2万 | 收入一般 |
| 🥇 中档 | ¥15,000 | ¥4,650 | ¥107万 | 收入中等 |
| 💎 高档 | ¥25,000 | ¥7,750 | ¥178.4万 | 收入较高 |

💡 **建议**：根据您的收入情况，选择能长期负担的档次。如果收入不稳定，建议选最低档保证不断缴；如果收入还行，中档比较平衡。

**用户选择：中档（¥15,000基数）**

**计算结果**：
```
💰 每月社保缴费：
├── 养老保险：¥3,000（20%，其中¥1,200进个人账户）
├── 医疗保险：¥1,650（11%）
├── 失业保险：¥0（无）
├── 住房公积金：¥0（一般无）
└── 合计：¥4,650/月

📊 退休前总缴费：约 ¥107万元
```

**📈 与职工社保对比**（同样¥15,000基数）：
- 灵活就业：¥4,650/月，总计¥107万
- 职工社保：¥2,775/月，总计¥63.8万
- 灵活就业多缴：**¥43.2万**

> 💡 **温馨提示**：虽然灵活就业缴得多，但养老金待遇和职工是一样的（个人账户部分相同），只是没有单位帮你分摊统筹部分

## 🛠️ 技术参数

### 支持的参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `birth_date` | 出生日期 | `1990-06-15` |
| `gender` | 性别 | `male` / `female` |
| `job_type` | 身份类型（女性） | `worker` / `cadre` |
| `monthly_salary` | 月工资 | `10000` |
| `city` | 所在城市 | `上海` / `北京` |
| `employment_type` | 就业类型 | `employee` / `flexible` |
| `include_housing_fund` | 是否含公积金 | `true` / `false` |

```bash

# 灵活就业
python scripts/calculate_retirement.py 1990-06-15 male worker 10000 上海 flexible
```

## ⚠️ 重要说明

### 🔴 数据时效性
- 社保缴费基数每年 **7月左右调整**
- 当前使用 **2024年数据**，实际计算请以当地社保局最新公布为准

### 🔴 地区差异
- 各地社保政策存在细微差异
- 灵活就业医疗保险比例各地不同（上海11%，武汉6%等）
- 计算结果仅供参考

### 🔴 政策变化
- 延迟退休政策为 **2025年新实施政策**
- 如有调整，脚本可能需要更新

### 🔴 灵活就业特别提醒
- 养老保险20%中只有 **8%进个人账户**，12%进统筹账户
- 没有失业保险和住房公积金
- 建议收入不稳定者选择 **最低基数缴纳**
