# jiuyan-yidong-market 技能

自动抓取并分析韭研公社市场异动页面，生成结构化市场简报。

---

本技能已创建至：`~/.openclaw/skills/jiuyan-yidong-market/`

## 功能说明

自动访问韭研公社异动页面 (https://www.jiuyangongshe.com/action)，提取并分析：

1. **市场环境** - 整体热度、上涨动力、热点分散/集中程度
2. **热门板块** - 资金集中方向、涨停股数
3. **板块龙头** - 领涨股票、涨停天数、驱动因素
4. **核心机会** - 值得关注的概念及逻辑
5. **风险提示** - ST、退市、炒作、脱钩等风险

---

## 输出格式

根据输出渠道自动生成两种格式：

### 格式 A：IMA 笔记（Markdown 格式）
- 保留 Markdown 格式符号（`**`、`###`、`-` 等）
- 适合知识库存档、后续检索

### 格式 B：QQ 推送（纯文本格式）
- 去除所有 Markdown 符号
- 使用【】作为标题标识
- 适合即时通讯推送

**共同特点**：
- 不使用 emoji
- 使用中文编号（一、二、三、四）
- 包含独立的"总结"部分
- 附带免责声明

```markdown

**日期**: 2026-03-24（星期二）
**数据来源**: 韭研公社异动板块
**生成时间**: 2026-03-24 15:30

### 一、当前市场环境概述
市场呈现"**事件驱动**"与"**产业趋势**"交织的格局，热点分散。主要驱动力来自：
1）美伊冲突引发的能源、化工品涨价预期；
2）"算电协同"概念带动的绿色电力板块；
3）人形机器人产业链的持续热度。

### 二、主要机会方向
1. **"算电协同"与绿色电力**：算力基础设施的高能耗需求，提升了对绿电和储能的需求。
2. **人形机器人产业链**：从核心零部件（减速器、关节模组、PEEK 材料）到下游投资布局热度不减。
3. **地缘冲突涨价链**：关注油气装备（FPSO）、以及乙二醇、PVC、磷肥等可能供应紧张的化工品。

### 三、主要风险提示
1. **退市风险**：多家*ST 公司已提示可能因财务或市值指标触及终止上市条件。
2. **炒作风险**：部分个股上涨依赖未经证实的市场传闻，缺乏基本面支撑。
3. **脱钩风险**：部分公司主营业务与热点概念关联度低，属短期题材炒作。

### 四、热门板块及龙头列表

- **机器人板块**
  - **逻辑**：人形机器人产业化加速，产业链关注度提升。
  - **龙头**：**首开股份** (间接持股宇树科技)、**金发科技** (提供机器人材料 + 间接投资宇树)。

### 总结
当前市场主线围绕**能源（电、油）、科技（机器人、算力）、材料（化工）**展开。操作上应**聚焦有实质逻辑的板块龙头**，同时**严格规避**有明确退市风险及纯炒作的个股。
```

## 执行流程

1. **导航** - 访问韭研公社异动页面
2. **等待** - 等待页面完全加载（5 秒）
3. **快照** - 获取页面完整内容
4. **提取** - 提取板块、股票、涨停天数、驱动因素
5. **分析** - 综合判断市场环境、机会、风险
6. **生成** - 输出结构化报告

---

## 错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| 页面需要登录 | 提示用户"该页面需要登录，请使用已登录的浏览器会话" |
| 页面结构变化 | 提示"页面结构可能已更新，分析结果可能不完整" |
| 网络超时 | 重试 1 次，失败后提示"网络请求超时，请稍后重试" |
| 内容为空 | 提示"未能获取到有效内容，请检查网址是否正确" |

## 依赖

- `browser` 工具 - 用于页面导航和内容提取
- `web_fetch` - 备用方案（如 browser 不可用）

---

## 注意事项

1. 确保完整读取页面内容，如有截断需使用偏移量继续读取
2. 分析时结合涨跌幅、涨停时间、解析文本综合判断
3. 解析内容中的编号列表是判断上涨逻辑的核心依据
4. 默认网址可公开访问，如需登录需提前告知用户
5. 本技能仅供参考，不构成投资建议
