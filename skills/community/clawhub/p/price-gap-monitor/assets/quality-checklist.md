# Price Gap Monitor — Quality Checklist

Use this checklist before delivering any price monitoring output.

---

## Data Collection (8 items)

- [ ] Product identity confirmed — correct model, size, variant matched across all platforms
- [ ] All target platforms checked (minimum 2, ideally 3+)
- [ ] Timestamps recorded for every data point (UTC format)
- [ ] Organic vs sponsored listings distinguished
- [ ] Collection method stated (browser-collected or user-provided)
- [ ] Search terms used are documented
- [ ] Number of results reviewed is stated explicitly
- [ ] Stale data (>7 days old) flagged with freshness warning

## Price Normalization (7 items)

- [ ] All prices converted to same currency with rate stated
- [ ] Multi-pack prices divided to per-unit cost
- [ ] Shipping costs added to calculate landed cost
- [ ] Subscribe & Save prices noted separately from one-time prices
- [ ] Different sizes/weights normalized to common unit
- [ ] Coupons and promos reflected in effective price (not just listed price)
- [ ] Non-comparable listings excluded with reason stated

## Evidence Integrity (6 items)

- [ ] Evidence strength labeled correctly (snapshot / directional signal / trend)
- [ ] Number of observations and time span stated
- [ ] No fabricated price history or trend claims
- [ ] Partial coverage acknowledged with percentage estimate
- [ ] "Full market" language avoided unless >80% coverage demonstrated
- [ ] Each factual claim traceable to a specific, timestamped observation

## Competitive Context (6 items)

- [ ] Competitor rating counts included alongside prices
- [ ] Seller maturity noted (established vs new)
- [ ] Brand positioning differences acknowledged (premium vs budget vs generic)
- [ ] Active promotions flagged as temporary
- [ ] Buy Box ownership noted (for Amazon)
- [ ] At least 3 direct competitors included in comparison

## Output Quality (7 items)

- [ ] Executive summary is 5 lines or fewer
- [ ] Correct output template used (Mode A or Mode B)
- [ ] Recommendation is specific and actionable (not "monitor the market")
- [ ] Margin floor checked — no recommendation goes below stated floor without explicit warning
- [ ] Next check date suggested
- [ ] Anomalies and risks documented in dedicated section
- [ ] All tables properly formatted with no missing cells

## Safety and Honesty (6 items)

- [ ] No claims of access to private marketplace data
- [ ] No fabricated BSR, sales velocity, or conversion data
- [ ] Trend language matches actual evidence (not overstated)
- [ ] Platform-specific pricing policies noted where relevant
- [ ] Refurbished/used items not mixed with new-condition comparisons
- [ ] Competitor pricing not presented without noting data limitations

## Delivery (5 items)

- [ ] Output reviewed for formatting errors
- [ ] All monetary values have currency symbols
- [ ] Dates in consistent format throughout
- [ ] Tables render correctly
- [ ] Actionable next step clearly stated at the end

---

**Total items: 45**
**Minimum passing: 40/45 checked**
