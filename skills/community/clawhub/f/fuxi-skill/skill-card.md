## Description: <br>
Fuxi Skill generates persona-style expert cognitive operating system skills by researching public information, distilling positive professional patterns, and producing reusable Markdown skill packages with disclaimers. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[yewubin-jpg](https://clawhub.ai/user/yewubin-jpg) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and developers use this skill to create a complete skill package that distills a named public figure's positive, verifiable professional ideas into a structured thinking-advisor workflow. The generated skill is intended to state that it is not the person themself and to include limitations and disclaimers after responses. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated persona-style skills may overstate or misrepresent a target person's views. <br>
Mitigation: Require each generated skill to state that it is AI-generated and not the person themself, and review the generated limitations and disclaimer before use. <br>
Risk: Users may provide confidential, sensitive, minor-related, or proprietary materials as optional source input. <br>
Mitigation: Use only authorized material and prefer public Internet sources unless the user has explicit rights to provide the supplemental content. <br>
Risk: Generated research and skill files may contain inaccurate, incomplete, or legally sensitive claims about a person. <br>
Mitigation: Review generated files for factual accuracy, intellectual-property concerns, reputation risk, and lawful use before sharing or relying on them. <br>
