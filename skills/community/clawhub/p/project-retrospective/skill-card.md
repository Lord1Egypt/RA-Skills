## Description: <br>
Helps an agent run a project retrospective after complex work, extracting reusable workflow documentation and lessons learned into local skill and memory files. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[BeAChanger](https://clawhub.ai/user/BeAChanger) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill after complex multi-step projects to summarize decisions, reusable workflows, problems encountered, and lessons learned. It is intended to produce proposed skill and memory documentation for review before saving. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated retrospective files can preserve secrets, tokens, private URLs, customer data, internal paths, or confidential implementation details. <br>
Mitigation: Review generated skill and memory files before saving and remove sensitive or confidential content. <br>
Risk: Retrospective summaries can record incorrect or misleading lessons if the project history is incomplete or misunderstood. <br>
Mitigation: Confirm the summarized workflow, pitfalls, and lessons with the user before preserving them for reuse. <br>
