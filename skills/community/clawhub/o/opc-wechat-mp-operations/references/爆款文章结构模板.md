## 二、开头钩子设计

### 2.1 痛点共鸣型开头

```
模板：
你是否也遇到过[痛点描述]？这是[目标用户]普遍面临的难题。
最近后台收到很多读者咨询：[痛点问题1]、[痛点问题2]、[痛点问题3]。
今天，我们就来系统解答这些问题。

示例：
你是否也遇到过这样的困惑：科研成果明明很有价值，却不知道如何转化？
技术入股估值到底怎么做？不同估值方法有什么区别？
最近一个月，后台收到了超过200条关于技术入股的咨询。
今天，我们就来系统解答技术入股中最关键的5个问题。
```

### 2.2 数据吸引型开头

```
模板：
[震惊数据]！这是[来源]发布的最新报告中，关于[主题]的关键发现。
[数据1]、[数据2]、[数据3]...这些数据背后，隐藏着[洞察]。
今天我们就来深度解读这些数据背后的含义。

示例：
90%的科技成果转化失败！这是科技部2023年发布的追踪报告中的数据。
平均转化周期3.5年、平均收益不足预期30%...
这些数据背后，隐藏着成果转化的深层逻辑。
今天我们就来深度分析：为什么成果转化这么难？
```

### 2.3 政策引用型开头

```
模板：
[时间]，[政策名称]正式[发布/实施/修订]。
这是[主体]时隔[时间]再次调整[相关领域]政策。
[一句话核心变化]。【查看原文>>】
本次修订释放了哪些信号？[目标用户]需要注意什么？

示例：
2024年1月，科技部等五部门联合发布《科技成果转化三年行动方案》。
这是国家层面时隔5年再次系统部署科技成果转化工作。
方案明确提出：科研人员成果转化收益比例可达90%。
这一重大调整释放了哪些信号？科研人员和高校需要注意什么？
```

### 2.4 案例开场型开头

```
模板：
[时间]，[主体]的[项目名]项目正式[完成/落地/签约]。
这一项目的[成果/金额]，创下了[领域]的多项记录。
[主体]是如何做到的？其中有哪些值得借鉴的方法？
今天我们就来深入剖析这个案例。

示例：
2023年12月，中科院某研究所的"高端传感器"项目正式完成转化签约。
这一项目以5000万元成交，创下了该研究所单个项目转化金额记录。
他们是如何做到的？研发团队付出了怎样的努力？
今天我们就来深入剖析这个亿元级成果转化的全过程。
```

## 五、排版格式规范

### 5.1 内容层级格式

```
# 文章标题（H1）
## 一级标题（H2）
### 二级标题（H3）
#### 三级标题（H4）

正文段落
- 列表项
- 列表项
  - 子列表项
  - 子列表项
- 列表项
```

### 5.2 重点内容格式

```
【强调框】
适用于：核心观点、关键结论
格式：
📌 [核心内容]

【提示框】
适用于：注意事项、风险提示
格式：
⚠️ [提示内容]

【引用框】
适用于：政策原文、数据引用
格式：
> "引用内容" —— 来源

【数据框】
适用于：关键数据、统计结果
格式：
📊 [数字]：[含义]
```

### 5.3 图片配置规范

| 文章字数 | 图片数量 | 图片位置 | 图片类型 |
|----------|----------|----------|----------|
| <1000字 | 1张 | 头图 | 封面图 |
