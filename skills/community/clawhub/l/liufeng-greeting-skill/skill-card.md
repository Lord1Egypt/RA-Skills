## Description: <br>
Detects common greeting messages and returns personalized Chinese greeting replies with the current China Standard Time. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[lyq8479](https://clawhub.ai/user/lyq8479) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
OpenClaw users can use this skill to provide an automatic personalized greeting response for Liufeng when a chat message contains common greeting keywords. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad greeting triggers may cause the skill to respond during ordinary conversations that merely include greeting words. <br>
Mitigation: Narrow or disable trigger patterns if the skill interferes with normal conversations or other installed skills. <br>
Risk: The submitted artifact files appear to contain upload boundary text, which may affect manual installation or direct file execution. <br>
Mitigation: Verify the installed package files before use and remove any upload wrapper text if installing manually. <br>
