# 输出质量 Checklist

Skill 输出周报后，按此清单检查是否能用。

## 核心：有没有读出结论（最重要）
- [ ] 不是把数据念一遍，每个发现都有"所以呢"
- [ ] 突出了增长信号，不只看粉丝绝对值
- [ ] 互动率排名指出了哪个平台高、哪个低
- [ ] Top 内容总结了共性规律（不只列5条）

## 行动建议可落地
- [ ] 3 条建议都是具体动作（"做修复连载"），不是废话（"提升质量"）
- [ ] 建议基于本周数据，不是泛泛常识
- [ ] 建议能直接交给运营执行

## 数据准确
- [ ] 每条数据标了来源平台
- [ ] 两个来源（Socialinsider + YouTube）合并看了，没割裂
- [ ] 数字和输入对得上，没编造

## 完整性
- [ ] 五个模块齐全（体量榜/互动率/Top内容/策略洞察/行动建议）
- [ ] 覆盖了主要竞品，没只盯一两家

---

## 数据质量提醒
- Socialinsider 导出时建议含：粉丝数、粉丝周变化、互动率、Top post
- YouTube 数据建议含：标题、播放量、点赞，按播放量排序更好分析
- 数据越全，洞察越准。某竞品缺某平台数据时，Skill 会基于现有数据分析
