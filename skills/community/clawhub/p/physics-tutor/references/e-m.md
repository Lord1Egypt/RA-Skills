# 费曼电磁学：到底在说什么

## 第一性原理

电磁学研究的是**电荷**以及它们之间的**相互作用**。

不是"磁铁吸铁"，不是"摩擦起电"——这些只是现象。
本质是：**电荷周围存在电场，电荷在电场中受到电场力。**

变化的电场产生磁场，变化的磁场产生电场——这就是**电磁波**，光、热辐射、无线电全是电磁波。

## 电场（Electric Field）

### 电荷

自然界只有两种电荷：正电荷（+）和负电荷（-）。
同种相斥，异种相吸。

**重要：电荷是守恒的，既不能创造也不能消灭，只能转移。**

### 电场强度

> **E = F/q = k·Q/r²**

电荷Q在周围产生电场，任何其他电荷q放入这个电场，就受到力 F = qE。

**类比：** 电荷Q像太阳，质量M像地球产生引力场。
区别：电场力可以是吸引（异种）也可以是排斥（同种），引力总是吸引。

**电场线的意义：**
- 电场线密集处电场强
- 电场线方向 = 正电荷在该点受力的方向
- 电场线从正电荷出发，终止于负电荷

### 电势与电势能

> **电势能：** 电荷在电场中具有的做功能力（类似重力势能）
> **电势：** 单位正电荷在该点的电势能，U = φ（有些教材用V表示电压）

> **U = k·Q/r**

**类比：** 电势就像"高度"——电场线从高电势指向低电势，就像重力场中物体从高处落向低处。

**电压：** 两点间的电势差，ΔU = U₁ - U₂。
电压是电场推动电荷做功的"强度"。

## 电路基础

### 电流

> **I = Q/t** （单位时间内通过截面的电荷量）

单位：安培（A）。1A = 1C/s。

**电流的方向：** 历史上规定为正电荷运动方向。
实际金属导体中，是电子在运动（负电荷），但电流方向约定不变。

### 电压和电流的关系

**欧姆定律：**
> **I = U/R** 或 **U = IR**

R是电阻，由导体本身的性质决定：
> **R = ρL/S**

ρ是材料的电阻率，L是长度，S是横截面积。

**电阻率与温度的关系：**
金属电阻随温度升高而增大（原子振动加剧，电子散射增加）。
→ 这是热敏电阻（RTD）和保险丝工作的原理。

### 电功率

> **P = UI = I²R = U²/R**

- 纯电阻电路：用哪个公式都行
- 含电动机的电路：P = UI是总功率，I²R是发热损失功率，UI - I²R是机械功率

### 串联与并联

**串联：** 电流相同，总电阻 = 各电阻之和，分压
**并联：** 电压相同，总电阻的倒数 = 各电阻倒数之和，分流

**记住：** 串联像叠加——电阻越串越大；并联像拓宽——电阻越并越小。

## 磁场（Magnetic Field）

### 磁现象的本质

**不是所有磁现象都因为有磁铁！**
1820年奥斯特发现：电流周围存在磁场。
1831年法拉第发现：变化的磁场可以在导体中感应出电流。

**结论：磁现象和电现象是同一件事的两个面。**

### 磁感应强度

> **B = F/(IL)** （通电导线在磁场中受的力）

单位：特斯拉（T）。地球磁场约 10⁻⁴ T，MRI约1.5T。

**判断受力方向：** 左手定则（注意和右手定则区分！）

### 电流在磁场中受力

> **F = BIL·sinθ**

- B⊥I 时（θ=90°），F=BIL（最大）
- B//I 时（θ=0°），F=0

**应用：** 电动机、扬声器（音圈）、霍尔效应传感器。

### 运动电荷在磁场中受力

> **F = qvB·sinθ**

**圆周运动的原因：** 速度方向总与磁场方向垂直，所以洛伦兹力总指向圆心，不做功，只改变速度方向。

**质谱仪工作原理：** 相同速度的带电粒子在磁场中偏转半径 r = mv/(qB)，不同质量的粒子落在不同位置。

### 电磁感应

**法拉第定律：**
> **ε = -N·ΔΦ/Δt** （感应电动势 = 线圈匝数 × 磁通量变化率）

**关键：不是磁通量本身，而是磁通量的变化产生感应电动势。**

**楞次定律：**
感应电流的方向是**阻碍原磁通量变化**的方向。
→ 不是"阻碍原磁场"，是"阻碍变化"。

**理解：** 你把磁铁插入线圈，线圈产生感应电流，这个电流产生的磁场会抵抗你插入的动作——你得做功，这功转化为感应电流的能量。

### 感应电动势的两种形式

| 形式 | 条件 | 表达式 |
|------|------|--------|
| 磁通量变化 | ΔΦ/Δt ≠ 0 | ε = -NΔΦ/Δt |
| 导体切割磁感线 | v⊥B | ε = BLv |

**公式选择：** 看题目描述——是"磁通量变化"还是"导体切割磁感线"。

## 交流电（AC）

### 交流电的产生

矩形线圈在匀强磁场中匀速转动：
> **e = Eₘ·sin(ωt）** （感应电动势随时间按正弦规律变化）

- Eₘ = NBSω（最大电动势）
- ω = 2πf（角频率，中国工频50Hz）

### 交流电的有效值

交流电的电压/电流大小在不断变化。有效值让交流电和直流电做功能力等价：

> **U_有效 = Uₘ/√2，I_有效 = Iₘ/√2**

日常说"220V交流电"指有效值，最大值约311V。

### 变压器

> **U₁/U₂ = N₁/N₂**

电压比等于匝数比。
功率近似相等（P₁≈P₂），所以电流比与匝数比成反比。

**能量守恒：** 高压低电流 → 低压高电流。

## 电磁波

### 麦克斯韦方程组的预言

变化的电场产生磁场，变化的磁场产生电场——两者相互激发，在空间中传播。

**不需要介质——这是光。**

### 电磁波谱

从长波到短波：

```
无线电波 → 微波 → 红外 → 可见光 → 紫外 → X射线 → γ射线
←——————— 低频/长波/低能量 ———————→  高频/短波/高能量
```

**注意：** 所有电磁波在真空中速度相同，c = 3×10⁸ m/s。
波速 = 频率 × 波长，频率越高，波长越短，单光子能量 E = hf 越大。

## 常见误解

### ❌ 误解1："电子在电路里从一端跑到另一端"

不对。电子在导体中实际漂移速度极慢（约mm/s），但电场以光速建立，所以整个电路"同时"有电流。
把电路想成水管——水分子移动慢，但开阀门水就流出来，不是水分子从水厂跑到你家。

### ❌ 误解2："电压和电流是因果关系，先有电压再有电流"

在静电场中，这个说法勉强成立。但在电路中，电压和电流是**同时产生**的关系。
电动势推动电荷定向移动，同时形成电流——不是先积累电荷再放电。

### ❌ 误解3："感应电动势是因为磁铁破坏了原来的磁场"

不对。是磁通量变化率产生了感应电动势。磁铁不动但磁通量在变化（如线圈在变强磁场中收缩），同样产生感应电动势。

### ❌ 误解4："电磁波需要介质才能传播"

电磁波的独特之处：不需要介质。
电磁波本身就是场的振荡，可以在真空中传播——光就是证据。
其他波（机械波）都需要介质。

### ❌ 误解5："电能就是电力"

电能是电荷在电场中的势能。电力（电力系统）是把电能从发电厂输送到用户的过程中的基础设施。
电流本身不提供能量——是电场做功，把电能转化为其他形式的能量。

## 电磁学这张网的连接点

```
电磁学
├── 连接力学：洛伦兹力→带电粒子在磁场中做匀速圆周运动
├── 连接振动与波：LC振荡电路→电磁波（光是电磁波）
├── 连接能量：电场力做功=电势能减少→其他形式能量
└── 连接量子：光子能量E=hf→黑体辐射、光电效应
```

## 费曼式理解电磁学的关键问题

1. **是什么产生了这个场（电荷还是电流）？**
2. **场的作用是什么（产生力，还是产生感应电动势）？**
3. **场在空间是如何分布的（是均匀的，还是随距离衰减的）？**
4. **有没有变化（ΔΦ/Δt）？变化是理解大部分电磁现象的关键。**
