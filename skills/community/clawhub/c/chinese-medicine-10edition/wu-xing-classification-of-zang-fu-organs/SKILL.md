# Wu Xing Classification of Zang Fu Organs

## When to Use
- 解释肝为何开窍于目、主筋
- 阐述心与舌、脉的关系
- 理解肺为何主皮毛、通调水道
- 分析肾与耳、二阴、骨髓的关联
- 构建中医整体观下的脏腑-自然（季节、气候）对应模型

## Execution Steps

1. **匹配五脏与五行基本特性**：
   - **木**：曲直、生长、升发、条达 → 肝喜条达、主疏泄、藏血 → 肝属木
   - **火**：温热、炎上、蒸腾 → 心主血脉、推动血行 → 心属火
   - **土**：生化、承载、受纳 → 脾主运化水谷 → 脾属土
   - **金**：肃杀、收敛、变革 → 肺主宣发肃降 → 肺属金
   - **水**：滋润、向下、寒凉、静藏 → 肾主水、藏精 → 肾属水

2. **建立脏腑-体窍-五行对应系统**：
   - **肝（木）**：与胆相表里，主筋，开窍于目 → 胆、筋、目属木
   - **心（火）**：与小肠相表里，主脉，开窍于舌 → 小肠、脉、舌属火
   - **脾（土）**：与胃相表里，主肌肉四肢，开窍于口 → 胃、肌肉、口属土
   - **肺（金）**：与大肠相表里，主皮毛，开窍于鼻 → 大肠、皮毛、鼻属金
   - **肾（水）**：与膀胱相表里，主骨生髓，开窍于耳及二阴 → 膀胱、骨、髓、耳、二阴属水

3. **应用“同气相求”原则**：同一五行属性的事物具有相似功能与感应性，可用于推演病理传变、指导诊断与治疗。

## Precautions
- 仅适用于纳入五行系统的脏腑及附属组织
- 必须结合中医整体观使用，不可孤立归类
- 不适用于奇恒之腑（如脑、髓、女子胞）或未被五行涵盖的结构
