# 阿尔茨海默病药物治疗策略与靶点选择

## 适用对象
- 确诊为阿尔茨海默病（AD）的患者
- 轻至中重度认知障碍者

## 使用前提
- 已确诊为AD
- 已排除严重肝肾功能不全等禁忌症

## 执行步骤

1. **评估疾病阶段**：确定患者处于早期、中度还是重度AD。
2. **选择核心治疗策略**：
   - **轻至中度AD**：首选胆碱酯酶抑制剂（如多奈哌齐、利斯的明、加兰他敏、石杉碱甲），可改善认知功能并延缓进展。
   - **中至重度AD**：使用美金刚（NMDA受体非竞争性拮抗剂），可改善认知、行为及日常生活能力。
   - **早期AD且符合条件者**：可考虑联合使用仑卡奈单抗（抗Aβ寡聚体人源抗体），每2周静脉注射10mg/kg，但需密切监测ARIA（淀粉样蛋白相关影像学异常）及心房纤颤等风险。
3. **评估合并症与耐受性**：根据肝肾功能、心脏疾病等情况调整药物选择或剂量。
4. **处理精神行为症状**：若认知药物治疗后仍存在严重精神症状（如幻觉、激越、抑郁），可谨慎联用抗精神病药、抗抑郁药或苯二氮䓬类药物。
5. **制定监测计划**：对使用仑卡奈单抗者安排定期MRI监测ARIA；对所有患者定期评估认知、功能及不良反应。

> 注意：现有药物不能根治AD病因，治疗目标为延缓进展、改善症状及维持生活质量。
