# Venue Evaluation Scorecard

## Scoring Instructions
Rate each criterion on a 1-10 scale. Multiply by the weight to get the weighted score. Total weighted score out of 100 determines venue ranking. Minimum viable score: 60/100. Recommended: 75/100+.

## Evaluation Criteria

### 1. Foot Traffic Volume (Weight: 15)
| Score | Description |
|---|---|
| 9-10 | Verified 2,000+ daily passersby from reliable count data |
| 7-8 | Estimated 1,000-2,000 daily based on area reputation and nearby anchors |
| 5-6 | Moderate traffic 500-1,000 daily; may require active marketing to drive visits |
| 3-4 | Light traffic 200-500 daily; heavily dependent on your own promotion |
| 1-2 | Under 200 daily; remote or low-visibility location |

**How to verify**: Request foot traffic data from landlord, use Placer.ai or similar platform, visit location at different times on different days, count passersby for 15-minute intervals and extrapolate.

### 2. Demographic Match (Weight: 15)
| Score | Description |
|---|---|
| 9-10 | Neighborhood demographics closely match target customer on age, income, interests |
| 7-8 | General demographic alignment with some variance |
| 5-6 | Mixed demographics; your target is present but not dominant |
| 3-4 | Poor demographic match; few target customers in area |
| 1-2 | Completely mismatched demographics |

**How to verify**: US Census data for zip code, Yelp/Google reviews of nearby businesses (who shops there?), visit on a weekday and weekend to observe actual visitors.

### 3. Complementary Business Proximity (Weight: 10)
| Score | Description |
|---|---|
| 9-10 | Surrounded by complementary businesses that attract your target customer |
| 7-8 | Several complementary businesses nearby |
| 5-6 | Mixed — some complementary, some neutral |
| 3-4 | Few complementary businesses; mostly irrelevant neighbors |
| 1-2 | Adjacent to businesses that conflict with brand positioning |

### 4. Accessibility & Parking (Weight: 10)
| Score | Description |
|---|---|
| 9-10 | Public transit stop within 2 blocks + dedicated parking + ADA compliant |
| 7-8 | Public transit or parking easily available (not both) |
| 5-6 | Moderate accessibility; some walking from transit, street parking available |
| 3-4 | Limited accessibility; distant transit, difficult parking |
| 1-2 | Poor accessibility; no transit, no parking options |

### 5. Lease Flexibility (Weight: 10)
| Score | Description |
|---|---|
| 9-10 | Short-term lease (weekly/monthly), easy termination, no personal guarantee |
| 7-8 | Flexible lease with reasonable terms and exit options |
| 5-6 | Standard short-term lease with some restrictions |
| 3-4 | Rigid terms, significant deposit, limited exit options |
| 1-2 | Long minimum commitment, high deposit, difficult termination |

### 6. Build-Out Allowances (Weight: 10)
| Score | Description |
|---|---|
| 9-10 | Full freedom to modify space; holes, paint, fixtures all allowed; restore not required |
| 7-8 | Most modifications allowed with restore clause |
| 5-6 | Basic modifications allowed; no structural changes, paint must match |
| 3-4 | Limited to freestanding fixtures only; no wall modifications |
| 1-2 | No modifications; must use space exactly as-is |

### 7. Infrastructure (Weight: 10)
| Score | Description |
|---|---|
| 9-10 | WiFi, electrical, HVAC, restroom, storage room all included and functional |
| 7-8 | Most infrastructure present; minor gaps easily solved |
| 5-6 | Basic infrastructure; need to supplement WiFi or add temporary electrical |
| 3-4 | Minimal infrastructure; significant investment to make functional |
| 1-2 | Empty shell; all infrastructure must be brought in |

### 8. Natural Light & Space Quality (Weight: 8)
| Score | Description |
|---|---|
| 9-10 | Abundant natural light, high ceilings, open floor plan, street-facing windows |
| 7-8 | Good natural light and usable layout with minor limitations |
| 5-6 | Adequate light and space; may need supplemental lighting |
| 3-4 | Limited natural light; challenging layout (pillars, odd shape) |
| 1-2 | No natural light; basement, interior mall, or windowless space |

### 9. Visibility & Signage (Weight: 7)
| Score | Description |
|---|---|
| 9-10 | Street-level with large windows, prominent signage allowed, corner or high-visibility location |
| 7-8 | Street-level with good signage options |
| 5-6 | Visible but set back or limited signage allowance |
| 3-4 | Second floor, interior corridor, or restricted signage |
| 1-2 | Hidden location with no exterior signage opportunity |

### 10. Total Cost Value (Weight: 5)
| Score | Description |
|---|---|
| 9-10 | Well below market rate for the location quality; exceptional value |
| 7-8 | At or slightly below market rate; fair pricing |
| 5-6 | At market rate; no particular value advantage |
| 3-4 | Above market rate; paying premium for location |
| 1-2 | Significantly overpriced relative to comparable options |
