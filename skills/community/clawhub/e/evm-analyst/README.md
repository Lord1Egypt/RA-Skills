# evm-analyst slim package

This is a runtime-slimmed ClawHub skill package. It removes examples and scripts from the uploaded skill to reduce prompt/context injection cost.

Kept files:
- SKILL.md
- references/dune_query_registry.md
- references/topic0_dictionary.md
- _meta.json
- .clawhub/origin.json

Removed from runtime package:
- examples/
- scripts/
- long methodology/accounting/reference documents

Use the full development package separately if you need code examples or scripts.
