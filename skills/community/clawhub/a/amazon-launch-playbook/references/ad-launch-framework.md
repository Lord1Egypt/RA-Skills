# 亚马逊新品广告启动框架

新品期的广告策略决定了流量成本和推广效率。本框架提供从测词到优化的三阶段完整路径，帮助你在有限预算内快速找到有效的推广方向。

---

## 框架概述

### 核心原则

1. **先测后投**：在大量投放前，先小规模测试找出高效关键词
2. **数据驱动**：所有调整都基于广告数据报告，而非主观判断
3. **循序渐进**：从精准到广泛，从小预算到大预算
4. **及时止损**：对表现差的关键词和广告组果断暂停

### 三阶段模型

| 阶段 | 时间 | 核心目标 | 广告类型 | 预算占比 |
|------|------|----------|----------|----------|
| 测词期 | Day 1-14 | 积累数据，找到高效词 | 自动广告 + 精准长尾词 | 40% |
| 扩词期 | Day 15-45 | 扩大流量，测试关联 | 扩展词 + 同类关联 | 45% |
| 优化期 | Day 46-90 | 收缩低效，集中高效 | 品牌广告 + 再营销 | 15% |

---

## 第一阶段：测词期（Day 1-14）

### 阶段目标

- 积累足够的广告数据（每个广告组至少1000次展示）
- 找到3-5个高效关键词（CTR > 0.5%，CVR > 10%）
- 测试Listing对流量的承接能力

### 广告活动设计

**广告活动1：自动广告（测试流量来源）**

| 项目 | 设置 |
|------|------|
| 活动名称 | Auto-Testing-[产品名] |
| 预算 | $10-15/天 |
| 竞价策略 | 动态竞价 - 只降低 |
| 投放方式 | 自动投放（全默认） |

自动广告的四种匹配方式会自动测试：
- 紧密匹配：买家搜索词与你产品相关
- 宽泛匹配：买家搜索词与产品类目相关
- 短语匹配：买家搜索词包含你的关键词
- 关联匹配：买家浏览相关产品详情页

**广告活动2：手动精准广告（精准长尾词）**

| 项目 | 设置 |
|------|------|
| 活动名称 | Manual-Exact-[产品名] |
| 预算 | $5-10/天 |
| 竞价策略 | 固定竞价 |
| 匹配方式 | 精准匹配 |
| 关键词数量 | 3-5个长尾词 |

**关键词选择标准**：
- 月搜索量：1000-3000（避免大词竞争激烈）
- 竞争程度：中低（首页Bid低于$1.5）
- 相关性：高（与你的产品高度相关）

**示例关键词策略**（家居收纳类）：
- "bedroom storage organizer"
- "drawer dividers for clothes"
- "closet shelf organizer set"

### 测词期预算分配

```
每日总预算 = $20-25

分配建议：
├── 自动广告：$12/天（60%）
└── 手动精准广告：$8/天（40%）
```

### 关键指标关注

**每日关注**：
- 各广告活动的展示量（Impressions）
- 各广告活动的点击量（Clicks）
- 各广告活动的花费（Spend）

**每周分析**：
- CTR（点击率）= Clicks / Impressions
- CVR（转化率）= Orders / Clicks
- ACOS = Spend / Sales
- 找到表现最好的搜索词

### 测词期调整策略

**需要暂停的情况**：
- 关键词获得100+展示但0点击（CTR < 0.1%）
- 关键词花费$20+但0订单（ACOS > 100%）
- 关键词与产品完全不相关

**需要调整Bid的情况**：
- 高展示、高点击、低转化：降低Bid 10-20%
- 低展示、高点击、高转化：提高Bid 10-20%
- 首页位置花费过高但转化一般：降至第2-3页位置

## 第二阶段：扩词期（Day 15-45）

### 阶段目标

- 基于测词期数据扩展关键词库
- 开始测试同类关联流量
- 建立多层次的流量来源

### 广告活动扩展

**广告活动3：手动短语广告**

| 项目 | 设置 |
|------|------|
| 活动名称 | Manual-Phrase-[产品名] |
| 预算 | $8-12/天 |
| 竞价策略 | 动态竞价 - 提高和降低 |
| 匹配方式 | 短语匹配 |
| 关键词来源 | 测词期表现好的精准词 |

**广告活动4：同类关联广告**

| 项目 | 设置 |
|------|------|
| 活动名称 | ASIN-Targeting-[同行名] |
| 预算 | $5-8/天 |
| 竞价策略 | 固定竞价 |
| 投放方式 | ASIN投放 |

**ASIN选择策略**：
- 选择与你产品互补的同行（同品类但不同定位）
- 选择评论50-200的中等同行（更有机会获得展示位）
- 避开评论500+的超级头部（竞争激烈且CVR低）

### 扩词期预算再分配

```
每日总预算 = $30-40（较测词期提升50%）

分配建议：
├── 自动广告：$10/天（30%）- 维持观察
├── 手动精准广告：$10/天（30%）- 保持高效词
├── 手动短语广告：$10/天（30%）- 新扩展词
└── ASIN关联广告：$5/天（15%）- 关联流量测试
```

### 关键词库扩展方法

**从测词期数据中提取**：
1. 打开自动广告报告
2. 筛选CVR > 5%的搜索词
3. 将高效词转为手动精准/短语投放

**从同行Listing中挖掘**：
1. 分析同行标题包含的关键词
2. 分析同行五点描述中的高频词
3. 查看同行评论中的产品描述词

**从工具中扩展**：
- 亚马逊搜索框下拉建议
- 同行的"Frequently bought together"
- Helium 10、Mention等工具（选其一）

### 扩词期调整策略

**每周一次关键词清理**：
- 暂停ACOS > 60%的关键词
- 降低花费 > $30但转化 < 3次的关键词Bid
- 将高效词从广泛匹配移至精准匹配

**Bid调整参考表**：

| 数据表现 | 诊断 | 调整建议 |
|----------|------|----------|
| 高展示、低点击 | 关键词与产品相关度低 | 暂停或大幅降低Bid |
| 高点击、低转化 | 流量精准但产品承接差 | 检查Listing或价格 |
| 低展示、高转化 | Bid太低或竞争激烈 | 提高Bid 15-20% |
| 高展示、高点击、高转化 | 理想状态 | 提高Bid抢占更多流量 |

## 第三阶段：优化期（Day 46-90）

### 阶段目标

- 收缩低效广告活动
- 提升品牌影响力
- 建立用户复购闭环

### 广告活动收尾

**保留高效活动**：
- 筛选ACOS < 30%或TACOS < 15%的广告活动
- 这些是你的"盈利广告"，需要持续优化

**暂停低效活动**：
- ACOS持续 > 60%的广告活动
- 花费高但无法带来利润的广告组

**重组广告结构**：
- 将相同关键词的不同匹配方式整合
- 简化广告层级，便于管理

### 新增品牌广告

**Sponsored Brands（品牌推广）**

| 项目 | 设置 |
|------|------|
| 活动名称 | SB-Brand-[品牌名] |
| 预算 | $15-20/天 |
| 竞价策略 | 动态竞价 - 只降低 |
| 投放关键词 | 品牌词 + 核心品类词 |

**品牌广告优势**：
- 占据搜索结果顶部黄金位置
- 展示品牌Logo和3个产品
- 提升品牌认知和信任

**Sponsored Display（展示型推广）**

| 项目 | 设置 |
|------|------|
| 活动名称 | SD-Remarketing-[品牌名] |
| 预算 | $10-15/天 |
| 受众选择 | 产品详情页浏览者（看过但未购买） |

**再营销策略**：
- 触达浏览过你产品的用户
- 触达浏览过同行页面的用户
- 触达相关品类浏览者

### 优化期预算再分配

```
每日总预算 = $35-50（视产品毛利而定）

分配建议：
├── 搜索广告（精准+短语）：$15/天（40%）- 保持核心词排名
├── 自动广告：$5/天（15%）- 维持长尾探索
├── ASIN关联广告：$5/天（15%）- 保持关联流量
├── 品牌广告（SB）：$10/天（25%）- 品牌建设
└── 展示型广告（SD）：$5/天（15%）- 再营销
```

### 长期优化策略

**关键词库维护**：
- 每月添加2-3个新测试词
- 持续淘汰表现差的词
- 关注搜索趋势变化

**Bid自动化建议**：
- 高效词：逐步提高Bid，追求首页排名
- 低效词：逐步降低Bid或暂停
- 新测试词：保持中等Bid观察

**ACOS目标设定**：

| 阶段 | ACOS目标 | 说明 |
|------|----------|------|
| 新品期（0-30天） | 40-60% | 以积累数据为主 |
| 成长期（30-60天） | 30-40% | 逐步优化效率 |
| 成熟期（60天+） | 20-30% | 追求广告盈利 |
| 长期稳定 | 15-25% | 品牌溢价带来效率 |

## 常见问题与解决方案

**Q1：自动广告花费很高但转化很少？**
A：检查是否是流量太泛。尝试设置竞价策略为"动态竞价-只降低"，同时添加否定关键词。

**Q2：手动广告有展示但无点击？**
A：可能是Bid太低（排名靠后）或主图不吸引人。尝试提高Bid 10-20%，同时优化主图。

**Q3：某个关键词ACOS一直很高？**
A：如果CVR也低，可能是关键词相关性差，考虑暂停。如果是CVR高但Bid太高，可以尝试降低Bid或调整匹配方式。

**Q4：ASIN关联广告效果不好？**
A：可能是选错了ASIN。尝试选择与你产品更互补或定位更接近的ASIN。

**Q5：品牌广告是否值得开？**
A：品牌广告的转化通常低于搜索广告，但可以建立品牌认知和抢占同行流量。长期来看是值得的。

## 结语

广告策略需要持续优化而非一成不变。本框架提供的是起点而非终点，请根据实际数据灵活调整。

**记住**：
- 数据是最好的老师，每周分析报告
- 不要害怕暂停无效广告
- 追求TACOS而非仅看ACOS
- 品牌建设是长期投资
