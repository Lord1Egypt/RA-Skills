# 现实校准模式参考

## 后天权重调校框架

术数推演的先天模型给出气的理论走向，后天权重则决定理论如何落地到具体求测者。以下是影响后天权重的核心维度：

### 个体维度

| 维度 | 对推演的影响 | 权重调校方向 |
|------|------------|------------|
| 职业 | 不同职业对应不同的气之运行模式 | 创业者加重"动变"权重；公务员加重"守正"权重 |
| 地域 | 南北东西的气场差异 | 北方加重"坎水"权重；南方加重"离火"权重 |
| 年龄 | 十二长生在不同人生阶段的对应 | 青年偏"长生/帝旺"；老年偏"墓/绝" |
| 性别 | 阴阳之气的运行差异 | 男性偏阳主动，女性偏阴主承 |
| 当前状态 | 身心状态影响气的接收与响应 | 疲惫时加重"休囚"权重；精力充沛加重"旺相"权重 |

### 时代维度

| 维度 | 对推演的影响 | 校准方式 |
|------|------------|---------|
| 社会经济周期 | 旺衰判断需结合大环境 | 经济下行期，"衰"象更易应验；上行期，"旺"象更易显现 |
| 技术变革 | 新行业无传统象意对应 | 以"离火"映射数字产业，以"巽风"映射传播行业 |
| 集体情绪 | 社会焦虑/乐观影响后天人事 | 焦虑期"坎水"类象意触发概率提升 |

## 多路径推演规范

每次推演输出应包含：

```
路径A：[描述] — 置信度：高/中/低 — 现实条件：[触发此路径需要的现实条件]
路径B：[描述] — 置信度：高/中/低 — 现实条件：[触发此路径需要的现实条件]
路径C：[描述] — 置信度：高/中/低 — 现实条件：[触发此路径需要的现实条件]
```

置信度判断依据：
- **高**：多个象意指向同一方向，历史案例吻合度 > 70%
- **中**：象意有分歧但主流方向明确，历史案例吻合度 40~70%
- **低**：象意矛盾或信息不足，历史案例吻合度 < 40%，需等待更多信息

## 历史案例回溯模板

```
### 案例编号：C-YYYY-NNN

**象意**：[卦象/干支/神煞等]
**推演结论**：[当时给出的预测]
**实际结果**：[事后验证的实际情况]
**吻合度**：[高/中/低]
**偏差分析**：[如有偏差，分析原因]
**校准调整**：[基于偏差对权重或象意联想的修正]
```

## 反馈闭环流程

1. **预测时**：记录推演依据、多路径输出、置信度
2. **等待期**：标注关键时间节点，提醒求测者关注应验窗口
3. **验证时**：比对实际结果与预测路径，计算吻合度
4. **校准时**：将偏差回注到后天权重或象意联想库
5. **迭代**：用校准后的模型服务下一次推演

## 常见校准陷阱

| 陷阱 | 说明 | 规避方法 |
|------|------|---------|
| 巴纳姆效应 | 断语过于笼统，人人觉得准 | 多路径+具体条件，避免放之四海而皆准的断语 |
| 确认偏误 | 只关注吻合的案例，忽略偏差 | 系统记录所有案例，定期统计整体吻合率 |
| 事后合理化 | 结果出来后重新解读象意使其吻合 | 推演时固定记录，验证时不修改原始记录 |
| 过度拟合 | 为单个案例调整过多权重 | 每次校准仅微调1~2个参数，观察3个以上案例后再确认调整 |
