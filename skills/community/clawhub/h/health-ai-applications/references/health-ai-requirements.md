# AI赋能大健康行业应用内容要求清单

AI赋能大健康行业应用各任务类型的组件清单，用于UTOS内容轴清单法。

**标记说明**（Workflow Refactor 标记系统）：
- ✅核心：事情本身的逻辑步骤
- 🔶校准：过程中纠偏的中间产出物，保留为分步校准点
- ⚡校验：防止出错的审核步骤（事后检查）
- ❌传递：人之间传递信息的中间步骤（已由AI自动化消除）
- ❌协调：管理多人协作的步骤（已由AI自动化消除）
- ❌格式：满足组织流程的形式要求（已由AI自动化消除）

**创新模式说明**（Universal Task OS）：
- 模式一·直用：反常识质疑、框架升级、基元重组
- 模式二·改进：第一性原理、逆向思维、辩证综合、随机性驱动、涌现生成、演化迭代、系统动力学、约束驱动、故事叙述、游戏化
- 模式三·迁移：机制迁移、结构迁移、方法迁移、概念迁移
- 模式四·构建：维度矩阵强制连接

## 📥 内容创作层

### D1 新媒体文案创作

#### D1-01 合规底线审查
- **必选组件**: 《广告法》禁用词清单（绝对化用语/虚假宣传/贬低竞品类）、行业规范清单（食品/保健品/医疗器械/化妆品分类）、各平台规则清单（微信/抖音/小红书/微博内容规范）、版权与引用声明模板（自动化：NLP自动检测禁用词；规则引擎自动审核行业规范；平台API自动适配规则）
- **可选组件**: 修改建议生成、合规评分报告、敏感词灰名单预警（创新方向：直用·反常识→实时合规检测；改进·约束驱动→极限合规下的创意表达；迁移·机制迁移→免疫系统模式；构建→合规×效率×创意）
- **组装顺序**: ❌传递 法务人工审核 → ❌协调 多部门协调 → ✅核心 禁用词检查 → ✅核心 行业规范审核 → ✅核心 平台规则适配 → ✅核心 版权声明 → ⚡校验 合规确认
- **合规约束**: 符合《广告法》第9/17/28条、《医疗广告管理办法》、《食品安全法》第73条、各平台社区公约
- **格式**: 合规审查报告（Excel表格，含违规项/严重等级/修改建议三列）

#### D1-02 创意卖点挖掘
- **必选组件**: 用户痛点清单（按场景/情绪/功能三维度）、差异化价值清单（技术/体验/服务/价格四象限）、情感共鸣切入点清单（焦虑/向往/认同/恐惧四类型）（自动化：用户评论NLP分析；竞品数据自动抓取；情感分析自动识别切入点）
- **可选组件**: 竞品卖点对比矩阵、用户调研问卷数据、社交媒体话题热度分析（创新方向：直用·反常识→用户需求反推卖点；改进·第一性原理→从基本需求重建价值；迁移·概念迁移→营销理论；构建→卖点×用户×竞品）
- **组装顺序**: ❌传递 调研报告人工传递 → ❌协调 市场部协调 → ✅核心 痛点挖掘 → ✅核心 价值提炼 → ✅核心 情感切入点 → 🔶校准 卖点确认
- **合规约束**: 卖点真实可验证、有临床/实验数据支撑、不违反《广告法》第4条真实性原则
- **格式**: 卖点清单（Markdown表格，含卖点/证据/目标人群/竞品对比四列）

#### D1-03 标题钩子设计
- **必选组件**: 数字型标题模板（含具体数据/百分比/天数）、反转型标题模板（"你以为…其实…"结构）、悬念型标题模板（疑问/省略/反转结尾）（自动化：LLM自动生成标题；A/B测试自动优化）
- **可选组件**: A/B测试方案（含样本量计算/显著性阈值/测试周期）、标题情感评分工具、热搜关键词嵌入策略（创新方向：直用·反常识→反转用户预期；改进·游戏化→挑战式标题；迁移·结构迁移→新闻标题理论；构建→标题×情绪×点击率）
- **组装顺序**: ❌传递 标题人工筛选 → ❌协调 创意部协调 → ✅核心 卖点输入 → ✅核心 标题类型选择 → ✅核心 标题撰写 → 🔶校准 吸引力评估
- **合规约束**: 标题真实不误导、不含《广告法》禁用绝对化用语、不制造健康焦虑
- **格式**: 标题清单（Excel表格，每种类型5个标题，含预估点击率/情感得分）

#### D1-04 故事化场景化表达
- **必选组件**: 开篇场景设计（痛点场景/使用场景/效果场景三选一）、中段展开结构（问题→方案→证据→效果四步法）、结尾升华设计（行动号召/情感共鸣/品牌记忆点三选一）（自动化：LLM自动生成故事；情感分析自动优化）
- **可选组件**: 用户见证故事（含真实案例授权书）、专家背书故事、品牌创始故事（创新方向：直用·反常识→用户故事UGC；改进·故事叙述→脱口秀结构；迁移·结构迁移→电影叙事理论；构建→故事×情感×记忆）
- **组装顺序**: ❌传递 故事人工创作 → ❌协调 文案部协调 → ✅核心 场景选择 → ✅核心 开篇设计 → ✅核心 中段展开 → ✅核心 结尾升华
- **合规约束**: 故事真实不虚构、用户见证需有授权、不暗示医疗效果
- **格式**: 故事化文案（Word文档，含场景描写/对话/数据引用，800-1500字）

#### D1-05 目标受众画像
- **必选组件**: 人口统计特征（年龄/性别/地域/收入/职业）、消费行为特征（购买频次/客单价/决策因素/渠道偏好）、平台使用偏好（微信/抖音/小红书/B站/知乎使用时长与内容偏好）（自动化：CRM数据分析；用户评论NLP分析；平台数据自动抓取）
- **可选组件**: 用户调研问卷数据、竞品用户画像对比、用户旅程地图（创新方向：直用·反常识→用户需求反推画像；改进·系统动力学→画像→内容→转化循环；迁移·概念迁移→用户画像理论；构建→画像×内容×转化）
- **组装顺序**: ❌传递 调研报告人工传递 → ❌协调 市场部协调 → ✅核心 特征分析 → ✅核心 习惯研究 → ✅核心 偏好分析 → 🔶校准 画像确认
- **合规约束**: 数据来源合法、符合《个人信息保护法》、用户隐私脱敏处理
- **格式**: 受众画像表（Excel多Sheet，含基础画像/行为画像/平台画像三个维度）

#### D1-06 平台特性适配
- **必选组件**: 公众号适配方案（图文排版/标题字数/封面图尺寸）、小红书适配方案（笔记格式/标签策略/封面比例）、抖音适配方案（短视频脚本/字幕规范/BGM选择）、微博适配方案（话题标签/转发策略/超话运营）（自动化：平台API自动获取规范；内容自动适配格式）
- **可选组件**: 快手/B站/知乎/视频号等扩展平台适配、私域社群适配方案（创新方向：直用·反常识→跨平台内容复用；改进·涌现生成→从数据中发现最优平台；迁移·机制迁移→媒体组合理论；构建→平台×内容×转化）
- **组装顺序**: ❌传递 平台规范人工收集 → ❌格式 格式人工调整 → ✅核心 受众画像输入 → ✅核心 平台选择 → ✅核心 内容适配 → ✅核心 格式调整
- **合规约束**: 符合各平台内容审核规范、《互联网广告管理办法》第9条
- **格式**: 多平台适配方案（Word文档，每个平台独立章节，含格式规范/发布时间/互动策略）

#### D1-07 发布策略制定
- **必选组件**: 发布时间规划（各平台最佳时段/时区考虑/节假日排期）、频率规划（日更/周更/月更节奏表）、互动引导设计（评论区话术/投票/抽奖/UGC征集）、转化路径设计（关注→阅读→点击→下单四步漏斗）（自动化：排期工具自动规划；平台API自动定时发布）
- **可选组件**: 热点结合策略（热搜监控/节日营销/突发事件响应）、KOL合作排期、付费推广排期（创新方向：直用·反常识→实时热点自动生成；改进·涌现生成→从数据中发现最优时间；迁移·概念迁移→媒体排期理论；构建→排期×热点×转化）
- **组装顺序**: ❌传递 排期人工协调 → ❌协调 多平台协调 → ✅核心 时间规划 → ✅核心 频率规划 → ✅核心 互动设计 → ✅核心 转化路径 → 🔶校准 策略确认
- **合规约束**: 发布时间符合用户活跃规律、互动引导不违反平台规则、转化路径合规
- **格式**: 发布策略表（Excel甘特图，含日期/平台/内容类型/负责人/状态列）

### D2 品宣图片海报生成

#### D2-01 品牌视觉规范制定
- **必选组件**: 主色调与辅助色规范（HEX/RGB/CMYK色值、使用比例6:3:1）、Logo使用规范（最小尺寸/安全区域/禁止变形规则）、字体层级规范（标题/正文/注释字体字号行距）、品牌IP形象使用规范（自动化：色彩提取工具自动分析品牌色；字体管理工具统一字体）
- **可选组件**: 品牌VI手册、品牌色彩情绪板、品牌字体库扩展（创新方向：直用·反常识→品牌视觉DNA；改进·演化迭代→风格迭代进化；迁移·概念迁移→基因工程理论；构建→风格×一致性×效率）
- **组装顺序**: ❌传递 品牌部人工传递 → ❌协调 设计部协调 → ✅核心 色彩确定 → ✅核心 Logo规则 → ✅核心 字体规范 → 🔶校准 规范确认
- **合规约束**: 符合品牌VI规范、色彩对比度符合无障碍标准（WCAG AA级）、字体版权合规
- **格式**: 视觉规范文档（PDF，含色卡/字体样本/Logo使用示例/禁止示例）

#### D2-02 图形元素库建设
- **必选组件**: 图标库（线性/面性/多风格，SVG格式）、纹理库（背景纹理/渐变/图案，PNG透明底）、装饰线条库（分割线/边框/角标）、留白与安全区域标准（边距/出血/裁切线规范）（自动化：AI生图自动生成元素；批量处理工具统一风格）
- **可选组件**: 品牌IP形象素材、Lottie动画元素、3D图标素材（创新方向：直用·反常识→AI生成无限元素；改进·涌现生成→从元素中涌现新风格；迁移·机制迁移→设计系统理论；构建→元素×风格×效率）
- **组装顺序**: ❌传递 设计师人工设计 → ❌格式 设计规范人工执行 → ✅核心 图标设计 → ✅核心 纹理设计 → ✅核心 装饰设计 → ✅核心 留白标准
- **合规约束**: 风格统一品牌一致、元素分辨率≥300dpi、素材版权合规（自有/CC0/商用授权）
- **格式**: 元素库文件（按类型分文件夹，SVG/PNG/AI多格式，含预览缩略图）

#### D2-03 文案内容模块设计
- **必选组件**: 固定标语模块（品牌slogan/产品核心卖点句）、可变促销模块（限时优惠/满减/赠品文案模板）、卖点列表模块（3-5条核心卖点+数据支撑）、CTA按钮文案（立即购买/了解更多/免费试用/扫码咨询）（自动化：LLM自动生成文案；模板引擎自动组合）
- **可选组件**: 法律免责声明模块、个性化变量模块（用户姓名/地域/偏好）、二维码嵌入模块（创新方向：直用·反常识→AI自我审核进化；改进·辩证综合→专业内容娱乐化；迁移·概念迁移→文案理论；构建→文案×情感×转化）
- **组装顺序**: ❌传递 文案人工编写 → ❌协调 市场部协调 → ✅核心 标语设计 → ✅核心 促销模块 → ✅核心 卖点模块 → ✅核心 CTA设计
- **合规约束**: 文案真实符合《广告法》、促销信息标注有效期、免责声明字体≥8pt
- **格式**: 文案模块库（JSON配置文件，含模块名/文案内容/变量占位符/适用场景）

#### D2-04 模板库建设
- **必选组件**: 多尺寸模板（公众号封面900×383/朋友圈1080×1080/小红书1080×1440/抖音1080×1920）、变量占位符系统（{{产品名}}/{{价格}}/{{日期}}等）、风格统一检查清单（自动化：Canva API自动生成模板；批量处理工具统一风格）
- **可选组件**: 动态模板（含GIF/视频模板）、交互模板（H5/小程序海报）、节日主题模板库（创新方向：直用·反常识→AI生成无限模板；改进·涌现生成→从数据中发现最优模板；迁移·机制迁移→设计系统理论；构建→模板×效率×转化）
- **组装顺序**: ❌传递 设计师人工设计 → ❌格式 尺寸格式人工调整 → ✅核心 尺寸规划 → ✅核心 模板设计 → ✅核心 变量设置 → ✅核心 风格检查
- **合规约束**: 风格统一品牌一致、模板分辨率≥72dpi（屏幕）/≥300dpi（印刷）、文件大小≤5MB
- **格式**: 模板库文件（PSD/Sketch/Figma源文件+PNG预览图，按平台分文件夹）

#### D2-05 批量生成参数配置
- **必选组件**: 尺寸规格表（各平台尺寸/分辨率/色彩模式）、输出格式配置（JPG质量/PNG透明度/PDF印刷设置）、命名规则模板（{品牌}_{平台}_{日期}_{序号}）、计划排期表（自动化：配置引擎自动设置；批量处理工具自动执行）
- **可选组件**: 自动化规则（条件触发生成/定时生成）、质量检查规则（分辨率/色彩/文字可读性自动检测）、多语言版本配置（创新方向：直用·反常识→智能参数推荐；改进·系统动力学→参数→效率→质量循环；迁移·机制迁移→配置管理理论；构建→参数×效率×质量）
- **组装顺序**: ❌传递 参数人工配置 → ❌协调 技术部协调 → ✅核心 尺寸配置 → ✅核心 格式配置 → ✅核心 命名配置 → ✅核心 排期配置
- **合规约束**: 参数合理无冲突、输出文件大小≤10MB、色彩模式匹配用途（屏幕RGB/印刷CMYK）
- **格式**: 配置文件（YAML格式，含dimensions/formats/naming/schedule四个配置块）

#### D2-06 自动化工具对接
- **必选组件**: API接口对接（Canva/Figma/Photoshop API密钥与端点配置）、设计软件脚本（PS动作脚本/Sketch插件/Figma插件）、批量生成测试用例（10张以上测试海报验证）（自动化：API自动对接；脚本自动执行）
- **可选组件**: 云渲染服务对接、生成结果监控报警、版本回滚机制（创新方向：直用·反常识→零代码自动化；改进·演化迭代→工具迭代进化；迁移·机制迁移→DevOps理论；构建→工具×效率×质量）
- **组装顺序**: ❌传递 开发人工对接 → ❌协调 技术部协调 → ✅核心 API对接 → ✅核心 脚本编写 → ✅核心 批量测试 → 🔶校准 功能确认
- **合规约束**: API密钥安全存储（环境变量/密钥管理服务）、接口调用频率符合服务商限制、数据传输HTTPS加密
- **格式**: 对接文档（Markdown，含API文档链接/配置参数/调用示例/故障排查）

### D3 品宣/科普视频生成

#### D3-01 视频类型与核心信息确定
- **必选组件**: 视频类型选择矩阵（品宣片/科普片/产品片/用户证言/教程片各适用场景）、核心信息点提炼（3秒法则/15秒法则/60秒法则信息密度控制）、目标受众与平台匹配（自动化：竞品数据自动分析；用户需求自动挖掘）
- **可选组件**: 竞品视频分析报告、用户偏好调研数据、平台热门视频趋势分析（创新方向：直用·反常识→用户需求反推类型；改进·第一性原理→从基本需求重建内容；迁移·概念迁移→内容营销理论；构建→类型×信息×转化）
- **组装顺序**: ❌传递 策划人工讨论 → ❌协调 市场部协调 → ✅核心 类型选择 → ✅核心 信息提炼 → 🔶校准 信息确认
- **合规约束**: 信息真实不夸大、符合《广告法》第17条（保健品/医疗器械禁用疗效承诺）、不制造健康焦虑
