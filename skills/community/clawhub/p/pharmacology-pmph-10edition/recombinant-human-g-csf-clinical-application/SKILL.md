# 重组人粒细胞集落刺激因子（非格司亭）临床应用

## 何时使用
当满足以下任一条件时启用本技能：
- 肿瘤患者接受化疗后出现严重中性粒细胞缺乏
- 骨髓移植术后需促进造血重建
- 先天性中性粒细胞缺乏症
- 艾滋病相关中性粒细胞减少

**前提条件**：
- 外周血中性粒细胞绝对值显著降低（通常 < 1.0 × 10⁹/L）
- 患者无对非格司亭或其他G-CSF制剂的过敏史

**禁忌情况**：
- 对本品或同类G-CSF制剂过敏
- 主要表现为巨噬细胞或巨核细胞功能障碍的疾病

## 如何执行
1. **确认适应症与禁忌症**：核实中性粒细胞计数及过敏史。
2. **启动非格司亭治疗**：
   - 通过皮下注射给药（常规剂量为5 μg/kg/日）
   - 监测中性粒细胞恢复情况
3. **评估疗效**：观察中性粒细胞计数回升、感染风险是否降低。
4. **监测不良反应**：
   - 常见：皮疹、低热、注射部位反应、轻至中度骨痛
   - 罕见但严重：过敏性休克（立即停药并处理）

> 注意：本药对巨噬细胞和巨核细胞影响极小，仅特异性作用于中性粒细胞谱系。
