## Description: <br>
Manages todo items by helping an agent add, list, complete, and delete tasks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[aeoleader](https://clawhub.ai/user/aeoleader) <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and developers can use this skill to maintain a simple local todo list through agent commands such as add, list, done, and remove. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Todo entries are stored locally in a Markdown file and may contain user-provided task text. <br>
Mitigation: Avoid entering passwords, API keys, or highly sensitive personal details in todo items. <br>
