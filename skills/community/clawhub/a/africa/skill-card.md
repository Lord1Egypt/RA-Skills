## Use Case: <br>
Developers and business analysts can use this skill to prototype multilingual African market policy and investment-environment queries. Review all outputs before operational or advisory use because the evidence identifies placeholder data/model behavior and possible credential exposure. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The evidence security summary says the skill overstates live AI-backed policy analysis and should be treated as a demo or placeholder unless real data and model integration are provided. <br>
Mitigation: Require publisher validation of data sources, model integration, freshness limits, and policy coverage before using outputs for production decisions. <br>
Risk: The evidence security guidance warns that configured API keys can leak into outputs when data sources are mixed with display fields. <br>
Mitigation: Do not configure real API keys until secrets are separated from returned data, masked or omitted from results, and managed through environment variables or a secrets manager. <br>
