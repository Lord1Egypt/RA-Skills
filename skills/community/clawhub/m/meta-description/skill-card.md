## Description: <br>
Guides optimization of the meta description tag for search engines and SERP display. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kostja94](https://clawhub.ai/user/kostja94) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users, marketers, SEO practitioners, and developers use this skill to draft or refine page-level meta descriptions that fit search-result display constraints and align with page intent, keywords, language, and calls to action. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Project context files may contain brand, audience, keyword, or other page details that the agent will use while drafting SEO copy. <br>
Mitigation: Review `.claude/project-context.md` and `.cursor/project-context.md` before installing or using the skill, and keep only context you are comfortable sharing with the agent. <br>
Risk: Generated meta descriptions can be misleading if the page type, keyword, language, or call to action is incorrect. <br>
Mitigation: Confirm the page context and review the recommended description before publishing it to a site. <br>
