# 鸿蒙 SDK 接入流程

## 1. 前置条件

- 已拥有华为开发者账号，并完成实名认证
- 已在 DataNexus 平台注册应用并获取 `user_action_set_id` 和 `secret_key`
- DevEco Studio 开发环境已就绪
- SDK 要求最低系统版本：**HarmonyOS 5.0.0（API 12）** 及以上

## 2. 引入 SDK

### 方式一：ohpm 包管理器（推荐）

在项目根目录执行：

```bash
ohpm install @dn-sdk/harmony
```

### 方式二：手动集成

1. 下载 SDK 的 `.har` 包
2. 将 `.har` 文件放入 `entry/libs/` 目录
3. 在 `oh-package.json5` 中添加依赖：

```json5
{
  "dependencies": {
    "@dn-sdk/harmony": "file:./libs/dn-sdk-harmony.har"
  }
}
```

## 3. 添加 SDK 权限

在 `module.json5` 中声明所需权限：

```json5
{
  "module": {
    "requestPermissions": [
      {
        "name": "ohos.permission.INTERNET",
        "reason": "SDK 数据上报需要网络通信"
      },
      {
        "name": "ohos.permission.GET_NETWORK_INFO",
        "reason": "检测网络状态"
      }
    ]
  }
}
```

> **可选权限：** 如需获取 OAID 以提升归因效果，可额外声明 `ohos.permission.APP_TRACKING_CONSENT` 权限。

## 4. 初始化 SDK

在 `EntryAbility` 的 `onCreate` 方法中调用初始化，并**紧接着调用 `start()`** 启动数据采集：

```typescript
import dnSDK from '@dn-sdk/harmony';

export default class EntryAbility extends UIAbility {
  onCreate(want: Want, launchParam: AbilityConstant.LaunchParam): void {
    dnSDK.init({
      context: this.context,
      user_action_set_id: YOUR_USER_ACTION_SET_ID,
      secret_key: 'YOUR_SECRET_KEY',
      // show_log: true,  // 可选：开启调试日志
    });
    dnSDK.start();
  }
}
```

> **重要：** `start()` 必须在 `init()` 之后尽快调用。延迟调用会影响设备 ID 填充率和归因成功率。

### 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `context` | `Context` | 是 | Ability 上下文 |
| `user_action_set_id` | `Number` | 是 | DataNexus 平台获取的数据源 ID |
| `secret_key` | `String` | 是 | DataNexus 平台获取的数据源密钥 |
| `user_unique_id` | `String` | 否 | 自定义用户唯一标识 |
| `show_log` | `Boolean` | 否 | 是否开启调试日志，默认 `false` |

## 5. 自动采集事件

SDK 初始化并调用 `start()` 后，以下事件会被**自动采集**：

| 事件 | 说明 |
|------|------|
| `START_APP` | 应用启动 |
| `TICKET` | 票据相关 |
| `ENTER_FOREGROUND` | 应用进入前台 |
| `ENTER_BACKGROUND` | 应用进入后台 |
| `APP_QUIT` | 应用退出 |

> **注意：** `START_APP` 为自动采集事件，请勿手动上报。

## 6. 上报自定义行为数据

SDK **仅提供一个通用上报方法** `logAction`：

```typescript
dnSDK.logAction(actionType: String, actionParam?: Object)
```

### actionParam 约束

- **Key：** 仅支持字母、数字、下划线，必须以字母开头，最长 **64** 个字符
- **Value：** 支持 String / Number / Boolean / JSONArray

### 示例

```typescript
dnSDK.logAction('REGISTER', { method: 'huawei_account', is_success: true });
dnSDK.logAction('PURCHASE', { value: 9900, currency: 'CNY' });
```

## 7. 设置自定义用户标识

```typescript
dnSDK.setUserUniqueId('your_user_id');
```

## 8. 验证接入

1. 初始化时设置 `show_log: true`
2. 过滤 TAG：`[@dn-sdk/harmony v1.x.x]`
3. 上报成功：`logAction success xxxx`
4. 上报失败：`logAction failed xxxx`
