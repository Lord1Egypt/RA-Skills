## M2-内容生成域范本

### M2-01 宣传单生成范本

**任务目标**：生成产品宣传单、活动宣传单、促销宣传单

**执行步骤**：

1. **确定宣传主题**
   - 产品宣传：突出产品特点
   - 活动宣传：突出活动亮点
   - 促销宣传：突出优惠力度

2. **编写宣传内容**
   - 标题：吸引眼球、引发兴趣
   - 正文：清晰、有条理
   - 行动号召：明确指引

3. **选择宣传图片**
   - 主图片：与主题相关
   - 辅助图片：补充说明
   - 品牌元素：logo、品牌色

4. **设计宣传布局**
   - 信息层次：主次分明
   - 视觉引导：引导阅读
   - 呼吁行动：明确指引

**输出示例**：

```markdown
## 宣传单设计方案

### 宣传主题
- 主题：新产品上市
- 目标：吸引用户关注

### 宣传内容
- 标题：新品上市，限时优惠
- 正文：产品特点、使用方法、用户评价
- 行动号召：立即购买，享受优惠

### 宣传图片
- 主图片：产品效果图
- 辅助图片：用户使用场景
- 品牌元素：品牌logo、品牌色

### 宣传布局
- 信息层次：标题→产品特点→用户评价→行动号召
- 视觉引导：从上到下，从左到右
- 呼吁行动：立即购买按钮+二维码
```

### M2-02 宣传册生成范本

**任务目标**：生成企业宣传册、产品宣传册、品牌宣传册

**执行步骤**：

1. **确定宣传册主题**
   - 企业宣传册：展示企业实力
   - 产品宣传册：展示产品特点
   - 品牌宣传册：传达品牌形象

2. **编写宣传册内容**
   - 封面：品牌logo+主题
   - 目录：内容概览
   - 正文：详细内容
   - 封底：联系方式

3. **设计宣传册页面**
   - 页面风格：与品牌一致
   - 页面布局：清晰、美观
   - 页面元素：图片、文字、图标

4. **统一宣传册风格**
   - 色彩统一：品牌色为主
   - 字体统一：品牌字体为主
   - 元素统一：品牌元素为主

**输出示例**：

```markdown
## 宣传册设计方案

### 宣传册主题
- 主题：企业宣传册
- 目标：展示企业实力

### 宣传册内容
- 封面：品牌logo+企业名称
- 目录：公司简介、产品介绍、团队介绍、联系方式
- 正文：详细内容
- 封底：联系方式+二维码

### 宣传册页面
- 页面风格：现代、专业
- 页面布局：图文并茂、清晰美观
- 页面元素：高质量图片、简洁文字、品牌图标

### 宣传册风格
- 色彩统一：蓝色品牌色为主
- 字体统一：现代简约字体
- 元素统一：品牌logo、品牌图标
```

### M2-04 宣传视频生成范本

**任务目标**：生成宣传片、广告片、产品介绍视频

**执行步骤**：

1. **确定视频主题**
   - 宣传片：展示品牌实力
   - 广告片：推广产品
   - 产品介绍：展示产品特点

2. **编写视频脚本**
   - 开头：吸引注意力
   - 中间：展示内容
   - 结尾：总结升华

3. **设计视频画面**
   - 画面风格：与品牌一致
   - 画面节奏：有张有弛
   - 画面效果：有视觉冲击力

4. **选择视频音效**
   - 背景音乐：与品牌调性一致
   - 音效：增强画面效果
   - 旁白：专业、有感染力

**输出示例**：

```markdown
## 宣传视频设计方案

### 视频主题
- 主题：企业宣传片
- 目标：展示企业实力

### 视频脚本
- 开头：企业logo+品牌口号
- 中间：企业介绍+产品展示+团队介绍
- 结尾：企业愿景+联系方式

### 视频画面
- 画面风格：现代、专业
- 画面节奏：快慢结合
- 画面效果：流畅、有视觉冲击力

### 视频音效
- 背景音乐：企业主题曲
- 音效：转场音效、强调音效
- 旁白：专业、有感染力
```
