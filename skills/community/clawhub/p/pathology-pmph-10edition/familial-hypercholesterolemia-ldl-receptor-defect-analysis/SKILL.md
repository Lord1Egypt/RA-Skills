# 家族性高胆固醇血症LDL受体缺陷机制分析

## 触发条件
IF 患者血浆LDL胆固醇极度升高且伴有黄色瘤、角膜弓或早发心肌梗死 THEN 启动本技能。

## 执行步骤

1. **确认适用对象**：
   - 高胆固醇血症患者
   - 早发动脉粥样硬化个体
   - 排除继发性原因（如肾病综合征、甲状腺功能减退）

2. **评估关键前提**：
   - 血浆LDL-C显著升高（成人通常 >190 mg/dL，儿童更高）
   - 存在黄色瘤或角膜弓等特征性体征

3. **分析遗传与分子机制**：
   - 判断是否符合常染色体显性遗传模式
   - 识别LDL受体基因突变导致的受体功能缺陷
   - 理解LDL清除障碍引发的胆固醇合成反馈失调

4. **识别典型临床表现**：
   - 极高LDL-C水平
   - 幼年即可出现冠状动脉粥样硬化
   - 角膜弓（角膜周围胆固醇沉积）
   - 黄色瘤（常见于手肘、膝部、指间关节、肌腱）

5. **形成诊断建议**：
   - 建议进行LDL受体（LDLR）基因检测
   - 启动降脂治疗（如他汀类药物）以预防早发心血管事件

> 注意：他汀类药物通过抑制胆固醇合成，部分恢复细胞内反馈调节，因此对此病有效。
