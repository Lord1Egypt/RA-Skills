## 平台合规

### 极限词（多平台都禁）
- 最 / 第一 / 唯一 / 顶级 / 极致 / 国家级 / 世界级
- 100% / 绝对 / 永久 / 终身
- 神器 / 王炸 / 救命

### 医疗/功效（化妆品、护肤、食品要特别小心）
- 治疗 / 治愈 / 疗效
- 抗衰老 / 抗皱（除非有备案）
- 美白（限定使用，看品类）
- 减肥 / 燃脂（除非药品资质）

### 金融
- 保本 / 稳赚 / 无风险
- 暴富 / 财富自由

## 品牌禁忌

> 不是合规问题，是"我们品牌就不该说这些"。

### 调性掉价的词
> 引用 [[brand/brand-voice]] 的"慎用词" + 这里更严格的"禁用词"
- _例：网红 / yyds / 绝绝子 / 必入 / 求求了_
- _例：性价比（如果你做高端，这词慎用）_

### 不做的承诺
- _例：不承诺"穿了显瘦 X 斤"，只说"显瘦"_
- _例：不承诺"皮肤变好"，只说"上脸感受"_
