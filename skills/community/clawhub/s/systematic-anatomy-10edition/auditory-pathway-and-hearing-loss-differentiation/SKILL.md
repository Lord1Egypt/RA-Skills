# 听觉传导路径与耳聋类型鉴别

## 何时使用
- 患者主诉听力下降，需明确耳聋类型以指导进一步检查或治疗；
- 临床怀疑存在外耳、中耳、内耳、蜗神经或听觉中枢的器质性病变；
- 需解释Rinne试验、Weber试验等听力检查结果。

## 执行步骤

1. **理解听觉传导路径**：
   - **空气传导（主要路径）**：声波 → 外耳道 → 鼓膜振动 → 听小骨链运动 → 镫骨底振动前庭窗 → 前庭阶外淋巴波动 → 经前庭膜传至内淋巴 → 刺激基底膜螺旋器 → 神经冲动 → 蜗神经 → 听觉中枢。
   - **骨传导（次要路径）**：声波 → 颅骨 → 骨迷路 → 外淋巴与内淋巴波动 → 刺激螺旋器 → 产生较弱听觉。

2. **判断耳聋类型**：
   - **传导性耳聋**：
     - 病变部位：外耳或中耳（如鼓膜穿孔、听小骨粘连、中耳炎）；
     - 特征：空气传导受损，骨传导相对保留；
     - 听力损失通常为部分性，不会导致完全耳聋。
   - **神经性耳聋**：
     - 病变部位：内耳（如螺旋器损伤）、蜗神经、听觉传导通路或听觉中枢；
     - 特征：空气传导与骨传导均无法有效产生听觉；
     - 可导致完全性耳聋。

3. **注意特殊情况**：
   - 鼓膜穿孔时，声波可直接振动鼓室内空气，经第二鼓膜引起鼓阶外淋巴波动，形成替代传导路径，产生部分听觉。

4. **临床鉴别方法**：
   - 比较空气传导与骨传导的相对强度；
   - 结合Rinne试验（AC > BC 为正常或神经性耳聋；BC > AC 提示传导性耳聋）和Weber试验（声音偏患侧提示传导性耳聋，偏健侧提示神经性耳聋）进行判断。

5. **输出结果**：
   - 明确耳聋类型（传导性或神经性）；
   - 指导下一步检查（如纯音测听、声导抗、MRI等）或转诊建议。
