## Description: <br>
Performs a high-level security audit checklist for AI agents, covering manipulation vulnerabilities, prompt injection, privilege escalation, data exfiltration, MCP server exposure, plugin risks, and compliance concerns. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[engsathiago](https://clawhub.ai/user/engsathiago) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, security reviewers, and teams preparing AI agents for deployment use this skill to structure an agent security review, identify common risk areas, and produce prioritized remediation and compliance notes. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The checklist could be mistaken for verified professional audit coverage or a complete security assessment. <br>
Mitigation: Use it as review guidance, define the exact agent, workspace, tools, MCP servers, and data sources in scope, and independently validate findings before relying on them. <br>
Risk: High-level audit prompts can produce incomplete or misleading results when the agent environment is underspecified. <br>
Mitigation: Provide concrete system, tool, permission, and data-flow details before running the review, then have a qualified reviewer confirm prioritized remediation steps. <br>
