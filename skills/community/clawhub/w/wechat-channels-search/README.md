# 视频号作品搜索 / wechat-channels-search

---

## 简介

输入关键词就能搜到视频号上的热门作品，能按热度或最新发布排序、翻页浏览更多内容，搜出来的都是平台刚更新的新鲜数据，不会拿旧缓存糊弄你。

**核心价值**

- **内容新鲜不滞后**：每次搜索都能直接拉取视频号上刚发布的作品，只给最新的内容，不用老旧数据。
- **排序方式灵活**：默认按热度排序看推荐，也能切到最新发布追踪新内容，怎么方便怎么来。
- **翻页浏览不漏数据**：海量结果可以分页查看，想看多少看多少，不用担心遗漏。
- **每日自动推送**：订阅感兴趣的关键词，每天 9:00 自动把最新作品送到你面前，不用手动重复查。

**适用对象**

- 📝 **内容创作者** — 追踪同赛道热门内容，为选题与创作提供参考。
- 🏢 **MCN / 品牌运营** — 快速了解特定品类在视频号的热度与头部内容表现。
- 📊 **市场团队** — 按关键词洞察内容趋势，辅助传播与投放决策。

---

## 功能特性

### 核心功能

- **关键词搜索**：输入关键词搜索视频号热门作品，精准发现优质内容。
- **排序选择**：默认按热度排序，也可选择按最新发布排序，满足不同查看需求。
- **分页浏览**：支持翻页获取更多结果，逐页浏览不遗漏。
- **订阅推送**：支持关键词每日推送，每天 9:00 自动获取最新作品。

### 特色亮点

- **一键订阅**：无需额外操作，查完即订，每天自动推送最新数据。
- **数据直出**：搜索结果以表格形式直接呈现，标题、点赞数、作者、时长、发布时间一览无余。

> 🔔 受视频号平台规则限制，无法提供作品链接，您可复制作品标题前往视频号搜索查看。

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=clawhub) (`https://redfox.hk`) 提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言说出你想查的内容方向即可，无需记忆固定命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 搜索某关键词 | 「查一下瑞幸咖啡」 | 按热度排序，展示搜索结果 |
| 按最新排序 | 「看看最新的占豪」 | 按最新发布排序 |
| 翻页查看 | 「下一页」「翻页」 | 查看更多搜索结果 |
| 订阅每日推送 | 查询后回复「确认订阅」 | 每天 9:00 自动推送该关键词最新作品 |

|---|---------|--------|------|------|----------|
| 1 | 瑞幸咖啡今日上新：弗朗明戈·跳跳冰茶... | 1189 | 新之助1- | 00:46 | 06-15 16:18 |
| 2 | C罗球迷冲瑞幸，梅西死忠选库迪！... | 20 | 大象财富 | 04:21 | 06-24 21:00 |

> 🔔 受视频号平台规则限制，无法提供作品链接，您可复制作品标题前往视频号搜索查看。
