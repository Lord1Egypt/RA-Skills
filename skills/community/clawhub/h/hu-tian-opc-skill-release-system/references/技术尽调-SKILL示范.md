# 技术尽调

## 一、Skill概述

技术尽调是OPC导师矩阵中的核心评估Skill，通过原理验证、参数对比、风险评估四层金字塔框架，帮助投资人和决策者判断项目/团队的技术真实性和商业潜力。

### 适用人群
- 投资人（VC/PE/产业资本）
- 并购方技术团队
- 科技成果转化机构
- 高校科研院所

### 核心价值
1. **风险识别**：发现技术谎言、专利陷阱、团队疑点
2. **价值判断**：评估技术先进性和商业化潜力
3. **决策支持**：提供结构化的技术评估报告

---

## 二、四层金字塔评估框架

```
                    ┌─────────────┐
                    │   风险结论   │  ← 顶层：最终判断
                    └──────┬──────┘
                           │
              ┌────────────┼────────────┐
              │            │            │
        ┌─────▼─────┐ ┌────▼────┐ ┌────▼────┐
        │ 原理验证  │ │ 参数对比 │ │ 风险评估 │
        │  (30%)   │ │  (40%)  │ │  (30%)  │
        └──────────┘ └─────────┘ └─────────┘
```

### 第一层：原理验证（30%）
验证核心技术是否基于真实科学原理

**检查项**：
- [ ] 技术原理是否符合已知物理/化学/生物定律
- [ ] 是否有已发表论文/专利支撑
- [ ] 原理验证实验是否可复现

**判断标准**：
| 等级 | 描述 | 风险 |
|------|------|------|
| A+ | 原理清晰，有多项第三方验证 | 低 |
| A | 原理清晰，有初步验证 | 中低 |
| B | 原理基本合理，验证不足 | 中 |
| C | 原理存在疑问，需要深入验证 | 高 |
| D | 原理违背基本科学定律 | 极高 |

### 第二层：参数对比（40%）
对比核心技术参数与行业水平

**对比维度**：
| 参数类型 | 示例 | 评估方法 |
|----------|------|----------|
| 性能参数 | 效率/速度/精度 | 与行业标杆对比 |
| 成本参数 | 单件成本/边际成本 | 与竞品对比 |
| 可靠性参数 | MTBF/寿命 | 与标准对比 |
| 规模化参数 | 产能/良品率 | 与理论值对比 |

**输出示例**：
```markdown
| 参数 | 被评估项目 | 行业最佳 | 行业平均 | 评估 |
|------|-----------|----------|----------|------|
| 转换效率 | 85% | 92% | 75% | 中上 |
| 成本/度 | 0.12元 | 0.08元 | 0.18元 | 中等 |
```

### 第三层：风险评估（30%）
识别技术风险和潜在陷阱

**风险清单**：
1. **IP风险**：专利归属不清、存在侵权风险
2. **团队风险**：核心人员依赖、资质造假
3. **技术风险**：技术路线被替代、工艺瓶颈
4. **量产风险**：小试≠量产，良品率存疑

### 第四层：结论输出
综合三层评估，输出结构化结论

## 三、使用流程

### Step 1: 收集材料
用户提供以下材料：
- 技术/产品介绍资料
- 相关专利证书
- 检测报告/认证文件
- 团队背景信息（可选）

### Step 2: 原理验证
AI自动进行原理层面的验证分析

### Step 3: 参数对比
与已知行业数据对比，输出对比表格

### Step 4: 风险识别
根据四层框架识别潜在风险点

### Step 5: 报告生成
输出完整的技术尽调报告

## 六、使用示例

### 示例输入
```
帮我看看这个"磁性超级电池"技术靠不靠谱：
- 原理：利用磁性材料在磁场作用下的特性实现充放电
- 宣称参数：充电5分钟，续航1000公里
- 专利：已申请中国发明专利
- 团队：创始人自称曾在特斯拉工作
```

### 示例输出（部分）
```markdown
## 技术尽调报告：磁性超级电池

### 原理验证
**原理分析**：该技术声称的原理存在重大疑点...

**原理评分：D（极高风险）**
- 充电5分钟续航1000公里需要能量密度约 400Wh/kg
- 当前锂电池能量密度上限约 300Wh/kg
- 宣称的参数违背能量守恒基本定律

### 参数对比
| 参数 | 宣称值 | 理论极限 | 可信度 |
|------|--------|----------|--------|
| 能量密度 | 400Wh/kg | 300Wh/kg | ⚠️ 存疑 |
| 充电速度 | 5分钟 | 受限于电化学反应 | ⚠️ 存疑 |

### 风险评估
- 🔴 **IP风险**：专利原理存疑，保护范围不明确
- 🔴 **技术风险**：违背基本物理定律，技术路线存疑
- 🟡 **团队风险**：创始人背景无法独立验证
- 🔴 **量产风险**：小试到量产存在巨大鸿沟

### 综合结论
**技术评级：D（不推荐投资）**

**核心问题**：
1. 原理违背能量守恒定律
2. 核心参数超出物理极限
3. 团队背景无法验证

**建议**：不建议跟进，如坚持需要求项目方提供第三方权威验证
```

## 七、注意事项

1. **数据验证**：所有数据需交叉验证，单一来源不可信
2. **原理优先**：先判断原理是否成立，再评估参数
3. **风险提示**：高风险不等于不可投，需结合投资策略
4. **持续跟踪**：技术评估结果有时效性，需定期更新
