# Prediction Market Alert Format

## 推送格式

⚠️ 必须严格遵循此格式，不得修改符号或结构。

---

## 1. 热门事件变化

```
🔥 预测市场热门事件变化 | {YYYY.MM.DD HH:mm}

📈 新进入 Top 10：
{事件标题}
当前概率：{概率}% | 交易量：${金额}
🔗 {Polymarket 链接}

📉 排名下降：
{事件标题}
排名变化：#{旧排名} → #{新排名}
当前概率：{概率}%
🔗 {Polymarket 链接}
```

**示例**:
```
🔥 预测市场热门事件变化 | 2026.03.18 14:30

📈 新进入 Top 10：
Will Trump win the 2024 US Presidential Election?
当前概率：62% | 交易量：$2,450,000
🔗 https://polymarket.com/event/trump-2024

📉 排名下降：
Will Bitcoin reach $100K by end of 2024?
排名变化：#3 → #7
当前概率：45%
🔗 https://polymarket.com/event/btc-100k
```

## 2. 大额资金买入

```
💰 预测市场大额交易提醒 | {YYYY.MM.DD HH:mm}

{事件标题}
交易金额：${金额} | 方向：{Yes/No}
交易后概率：{旧概率}% → {新概率}%
影响：概率变化 {变化幅度}%
🔗 {Polymarket 链接}
```

**示例**:
```
💰 预测市场大额交易提醒 | 2026.03.18 14:35

Will OpenAI release GPT-5 in Q1 2024?
交易金额：$250,000 | 方向：Yes
交易后概率：38% → 42%
影响：概率变化 +4%
🔗 https://polymarket.com/event/gpt5-q1
```

## 3. 概率快速波动

```
📊 预测市场概率波动提醒 | {YYYY.MM.DD HH:mm}

{事件标题}
{时间范围} 变化：{旧概率}% → {新概率}% ({变化幅度}%)

Will the Fed cut rates in March 2024?
1h 变化：25% → 38% (+13%)

🔗 https://polymarket.com/event/fed-march-cut
```

## 4. 潜在交易机会

```
💡 预测市场交易机会 | {YYYY.MM.DD HH:mm}

{事件标题}
当前价格：{概率}% | 历史均值：{均值}%
偏离度：{偏离幅度}% ({高估/低估})

**示例**:

Will Ethereum merge to PoS in 2024?

风险提示：技术延期风险，建议小仓位试探
🔗 https://polymarket.com/event/eth-merge
```

## 5. 每日摘要（可选）

```
📋 预测市场每日摘要 | {YYYY.MM.DD}

🔥 今日热门：
1️⃣ {事件标题} - {概率}%
2️⃣ {事件标题} - {概率}%
3️⃣ {事件标题} - {概率}%

💰 大额交易：{笔数} 笔，总计 ${总金额}
📊 波动事件：{数量} 个
💡 交易机会：{数量} 个

详情查看：🔗 https://polymarket.com
```

**示例**:
```
📋 预测市场每日摘要 | 2026.03.18

🔥 今日热门：
1️⃣ Trump wins 2024 election - 62%
2️⃣ Bitcoin reaches $100K - 45%
3️⃣ GPT-5 released in Q1 - 42%

💰 大额交易：8 笔，总计 $1,250,000
📊 波动事件：5 个
💡 交易机会：3 个

详情查看：🔗 https://polymarket.com
```

---

## 格式关键要求

1. **时间戳**：所有提醒必须包含时间戳（YYYY.MM.DD HH:mm）
2. **Emoji 使用**：
   - 🔥 热门事件
   - 💰 大额交易
   - 📊 概率波动
   - 💡 交易机会
   - 📋 每日摘要
3. **数据精度**：
   - 概率：整数百分比（如 62%）
   - 金额：美元格式，千位分隔符（如 $1,250,000）
   - 时间：24 小时制（如 14:30）
4. **链接**：必须包含 🔗 emoji 和完整 Polymarket 链接
5. **风险提示**：交易机会必须包含风险说明
6. **简洁性**：每条提醒控制在 5 行以内（不含链接）
