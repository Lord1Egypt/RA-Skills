## Description: <br>
将 MySQL SQL（尤其是建表 DDL）转换为 Oracle SQL。用户提到“转为oracle”或“转换oracle”，或明确要求把 MySQL 语句改成 Oracle 语句时必须使用此技能。转换时强制执行类型、默认值和主键约束命名规则。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and database engineers use this skill to convert MySQL DDL, especially CREATE TABLE statements, into Oracle-compatible SQL with strict type, default value, comment, and primary-key constraint rules. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated SQL may be incorrect or over-eagerly converted for a specific schema or Oracle environment. <br>
Mitigation: Review and test generated SQL before applying it to any database, especially in production. <br>
