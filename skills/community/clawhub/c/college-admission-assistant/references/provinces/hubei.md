## 一、高考模式

**3+1+2模式**（2021年首届新高考）
- 总分：750分
- 语文150 + 数学150 + 外语150 + 物理/历史原始分100 + 4选2等级赋分各100

## 四、湖北高考特点

1. **高等教育强省**：2所985，5所211
2. **综合性大学密集**：武大、华科覆盖几乎所有学科门类
3. **武汉高校聚集效应**：百万大学生城市，招生体量大
4. **武汉地方专项计划**：面向农村和脱贫地区

---

## 五、志愿填报规则

1. **批次**：提前批 → 本科批 → 专科批
2. **志愿单位**：院校专业组
3. **本科批**：45个院校专业组平行志愿
4. **同分排序**：依次比较语数总分 → 语数单科 → 外语
