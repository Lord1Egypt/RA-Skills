# 常见证候与舌象对照

> ⚠️ 注意：舌诊是望诊的一部分，需结合其他三诊（闻诊、问诊、切诊）综合分析。
> 以下为典型舌象组合，实际临床变化复杂，不可单纯凭舌象定论。

## 一、单一特征组合判断

### 舌色 + 舌形组合
| 舌色 | 舌形 | 常见证候 |
|------|------|---------|
| 淡白 | 胖大 + 齿痕 | 脾肾阳虚，水湿内停 |
| 淡白 | 瘦薄 | 气血两虚 |
| 淡白 | 裂纹 | 血虚 |
| 红 | 胖大 | 里热证 |
| 红 | 瘦薄 + 少苔 | 阴虚火旺 |
| 红 | 芒刺 | 脏腑阳热亢盛 |
| 绛 | 干枯 | 热入营血，阴液耗损 |
| 青紫 | 有瘀斑 | 血瘀证 |
| 青紫 | 湿润 | 寒凝血瘀 |

### 舌苔 + 舌色组合
| 舌色 | 舌苔特征 | 常见证候 |
|------|---------|---------|
| 淡红 | 薄白苔 | 正常人或表证初起 |
| 淡红 | 黄苔 | 里热证 |
| 淡白 | 白腻苔 | 寒湿内阻 |
| 淡白 | 白滑苔 | 阳虚水泛 |
| 红 | 黄厚燥苔 | 阳明腑实热证 |
| 红 | 黄腻苔 | 湿热内蕴 |
| 红 | 薄黄苔 | 气分热盛 |
| 绛 | 无苔/剥苔 | 热入营血，阴亏 |
| 青紫 | 白腻苔 | 寒凝血瘀兼湿 |
| 青紫 | 黄腻苔 | 痰热瘀结 |

## 二、常见证候速查表

### 虚证类
| 证候 | 典型舌象 | 伴随症状要点 |
|------|---------|------------|
| 气血两虚 | 舌淡白，瘦薄或嫩，苔薄白 | 面色苍白或萎黄，头晕目眩，心悸气短，神疲乏力 |
| 阴虚证 | 舌红少苔或无苔，或裂纹舌 | 形体消瘦，口燥咽干，潮热颧红，五心烦热 |
| 阳虚证 | 舌淡白胖大，苔白滑 | 畏寒肢冷，神疲乏力，小便清长，大便溏薄 |
| 脾虚湿盛 | 舌淡胖，边有齿痕，苔白腻 | 食少纳呆，腹胀便溏，肢体困重 |
| 胃阴不足 | 舌红少苔或剥苔，或镜面舌 | 口干咽燥，胃脘隐痛，饥不欲食 |

### 实证类
| 证候 | 典型舌象 | 伴随症状要点 |
|------|---------|------------|
| 热入营血 | 舌绛、干燥少苔 | 身热夜甚，心烦不寐，神昏谵语 |
| 血瘀证 | 舌紫暗或有瘀斑，舌下络脉增粗 | 疼痛固定不移，肌肤甲错 |
| 湿热内蕴 | 舌红苔黄腻 | 脘腹痞闷，呕恶纳呆，身热不扬 |
| 寒湿内阻 | 舌淡苔白腻或白滑 | 脘腹胀满，口淡不渴，大便稀溏 |
| 食积胃肠 | 苔厚腻或腐苔 | 脘腹胀满，嗳腐吞酸，厌食 |
| 痰浊内蕴 | 苔腻（白腻或黄腻） | 胸闷咳嗽，眩晕恶心 |
| 肝风内动 | 舌颤动，或歪斜 | 眩晕欲仆，肢体麻木，甚至抽搐 |

### 虚实夹杂类
| 证候 | 典型舌象 | 伴随症状要点 |
|------|---------|------------|
| 气虚血瘀 | 舌淡暗或有瘀斑 | 神疲乏力，疼痛固定，面色晦暗 |
| 阴虚火旺 | 舌红绛瘦薄，少苔 | 潮热盗汗，心烦失眠，口燥咽干 |
| 阳虚水泛 | 舌淡白胖大，苔白滑 | 水肿，畏寒，小便不利 |
| 痰热腑实 | 舌红，苔黄厚腻或焦黄 | 大便不通，腹胀，烦躁 |

## 三、预后判断要点

| 舌象特征 | 预后判断 |
|---------|---------|
| 舌色红活鲜明 | 气血充足，预后良好（得神/荣舌） |
| 舌色晦暗无光 | 气血枯竭，脏腑衰败（失神/枯舌） |
| 镜面舌 | 胃气阴枯竭，病情危重 |
| 舌苔有根 | 有胃气，尚可治 |
| 舌苔无根 | 胃气大伤，预后不良 |
| 舌色由绛转红、由红转淡红 | 病情向好 |
| 舌色由淡红转红绛 | 病邪深入，病情加重 |
