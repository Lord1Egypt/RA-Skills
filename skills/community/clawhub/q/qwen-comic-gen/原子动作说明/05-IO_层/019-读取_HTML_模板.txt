===========================================
原子动作 019 - 读取 HTML 模板
===========================================

【这个动作是干啥的】
从硬盘里读取 HTML 模板文件，就像打开一个 Word 文档一样。

【输入是啥】
- 文件路径（比如 C:\...\standard-template.html）

【输出是啥】
- HTML 模板的完整内容（一段文本）
- UTF-8 编码，不会乱码

【什么时候用】
- 生成 HTML 报告前（先读模板）
- 批量生成报告（读一次，复用多次）
- 需要统一格式时

【注意事项】
- 文件必须存在（先用 Test-Path 检查）
- 编码用 UTF-8（中文不会乱码）
- 路径用绝对路径（不要用相对路径）

【对应阿福的 MD 文件】
atomic-actions/05-io/ATOM-IO-019-读取 HTML 模板.md

【举个例子】
你：生成一个专家点评 HTML
阿福：先调用 ATOM-IO-019
     读取 skills/html-expert-review/templates/standard-template.html
     拿到模板内容，准备填充数据
