
### 名称
`strategic-partnership-analyzer`

### Slug
`strategic-partnership-analyzer`

### 定位（用户可见）
深度挖掘目标企业与国央企、头部大厂的近期签约动态，穿透表层新闻，解析其实质合作深度与业务协同意图。

### 描述（LLM识别）
深耕ToB/ToG生态圈的商业分析师及产业情报专家。当用户询问某企业近期的“朋友圈”、“合作伙伴”、“生态联盟”或“战略动向”，意图了解某企业在特定领域（如数字化、新能源）找了哪些厂商进行战略绑定或商业合作，或要求评估某企业最近一次重大签约/合作背后的真实商业意图时，激活此技能。

执行逻辑：
1. 实体与意图提取：准确提取用户指令中的“目标企业名称”，并判断是否指定了特定合作领域。
2. 触发联网搜索：定向搜索该企业近12个月内的官方新闻、PR稿件及财经报道。搜索逻辑包含：“企业名称” AND (“战略合作” OR “签署协议” OR “联合实验室” OR “共建” OR “合资” OR “采购” OR “供应链”)。
3. 降噪与实质判别：提取所有商业合作事项，重点提取带有实质性动作的合作（如成立合资公司、明确的采购金额、具体产品融合计划、联合研发投入）。
4. 意图推演：基于合作方的行业地位和业务互补性，推测该企业此次合作旨在“开拓新市场渠道”、“弥补核心技术短板”还是“寻求资金/供应链背书”。标注此为推演。
5. 输出格式：《战略生态洞察报告》，包含：近期核心合作大事件时间轴、合作模式分类矩阵、深层战略意图推演。

### 适用场景
- 招商尽职调查中评估目标企业的生态位与扩张路径
- 投资人判断企业未来增长的可信度与资源整合能力
- 竞品分析中预测对手的战略动向
- 企业自身制定合作策略前的情报输入

### 输入参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `company_name` | string | 是 | 目标企业名称 |
| `focus_area` | string | 否 | 特定合作领域（如“算力”、“低空经济”、“新能源”），用于过滤相关合作 |
| `time_range` | int | 否 | 搜索时间范围（月），默认12个月 |
| `include_rumor` | bool | 否 | 是否包含未经官方确认的传闻，默认false |

### 输出格式
输出一份《战略生态洞察报告》，包含以下模块：

1. **企业基础信息**：企业名称、所属行业、近期融资/上市状态（如有）
2. **近期核心合作大事件（时间轴）**  
   - 按时间倒序列出Top 3-5重大合作，每项包含：时间、合作方、合作内容摘要、公开披露的金额/股权/产品细节（如有）
3. **合作模式分类矩阵**（表格形式）  
   - 合作方 | 合作类型 | 实质动作 | 推测目的
   - 合作类型包括：技术研发互补型、市场渠道共享型、供应链深度绑定型、资本/股权合作型、标准/联盟共建型
4. **深层战略意图推演**  
   - 当前急需通过外部合作解决的核心业务瓶颈（如“缺乏大客户销售渠道”、“核心芯片供应不稳”、“适航认证资源不足”）  
   - 未来12个月可能发力的方向（基于合作密度分析）  
   - 合作网络图谱（文字描述或Mermaid示意）
5. **风险信号识别**（如有）  
   - 合作中止、合作深度不及预期、合作方负面舆情等
6. **对我方（招商/投资）的启示**  
   - 如何利用该企业的合作偏好制定接触策略

### 执行流程（内部步骤）

1. **实体与意图提取**  
   - 从用户输入中提取 `company_name`。  
   - 若提及“只关注XX领域的合作”，记录为 `focus_area`。  
   - 默认搜索近12个月，若用户指定则覆盖。

2. **联网搜索策略**  
   - 关键词组合（按优先级）：  
     - `"{company_name} 战略合作"`  
     - `"{company_name} 签约"`  
     - `"{company_name} 联合实验室"`  
     - `"{company_name} 合资公司"`  
     - `"{company_name} 供应链 合作"`  
     - `"{company_name} 与 ＊ 战略协议"`  
   - 限定来源：企业官网新闻中心、财经媒体（36氪、虎嗅、新浪财经、第一财经）、政府签约新闻。  
   - 时间筛选：使用搜索工具的 `after:YYYY-MM-DD` 参数。

3. **降噪与实质判别**  
   - 过滤掉“框架协议”但无后续实质动作的新闻（除非明确标注投资额或合作细节）。  
   - 重点保留：  
     - 成立合资公司（需有注册资本、股权比例）  
     - 采购订单（有明确金额或数量）  
     - 产品融合（如“飞控系统集成对方芯片”）  
     - 共建研发中心/联合实验室（有场地或人员投入）  
     - 股权融资/相互持股  
   - 对于仅有“战略合作签约仪式”照片而无细节的，标注“公开信息有限，实质深度未知”。

4. **合作模式分类**  
   - 根据合作内容和双方资源判断：  
     - **技术研发互补型**：一方提供技术，另一方提供场景或算法。  
     - **市场渠道共享型**：一方产品通过另一方销售网络铺开。  
     - **供应链深度绑定型**：采购或供应核心元器件、原材料。  
     - **资本/股权合作型**：投资、合资、交叉持股。  
     - **标准/联盟共建型**：参与行业标准制定、发起产业联盟。  

5. **意图推演**  
   - 基于合作方的背景（国央企、大厂、科研院所）和企业的当前阶段（初创/扩张/上市前）进行逻辑推理。  
   - 常见意图范式：  
     - 与国央企合作 → 意图获取政策准入、大订单或政府背书。  
     - 与华为/腾讯等大厂合作 → 意图获取技术赋能或渠道流量。  
     - 与高校/实验室合作 → 意图补充前沿研发能力或人才。  
     - 与上下游供应链企业多次合作 → 意图构建生态壁垒。  
   - 推演内容开头必须标注“【推演】”或“基于公开信息推测”。

6. **输出报告**  
   - 时间轴以列表或表格呈现。  
   - 合作模式矩阵用Markdown表格。  
   - 推演部分单独成节。  
   - 若发现合作中止或负面新闻，列入风险信号。

## 三、深层战略意图推演

**【基于公开信息推演】**

边界智控当前最急迫解决的三大业务瓶颈：

1. **供应链安全**：飞控核心芯片（FPGA、MCU）依赖TI、ADI等海外厂商。与TI签订排他协议，且共建应用中心，意图锁定产能，同时探索国产替代方案（未公开）。  
2. **适航认证加速**：eVTOL飞控适航取证周期长，与峰飞航空合作可共享适航测试数据，缩短30%以上周期。未来可能进一步寻求与中国民航局适航审定中心直接合作。  
3. **大客户依赖风险**：目前约40%营收来自大疆，过度集中。近期与峰飞、沃兰特等多家eVTOL企业接触，意图分散客户结构，同时布局载人市场蓝海。

**未来12个月可能发力的方向**：  
- 继续绑定至少2家传感器厂商（惯性测量单元、气压计）形成供应链联盟。  
- 与至少1家央企（如中国通号、中航工业）签署战略合作，获取政府及央企订单。  
- 可能成立产业基金或孵化器，吸引飞控周边初创企业构建生态。

**合作网络图谱（简略）**：  
```
边界智控
├─ 上游供应商: TI(排他), 瑞声(联合实验室), ADI(预计跟进)
├─ 下游客户: 大疆(批量采购), 峰飞(联合研发), 沃兰特(洽谈中)
├─ 生态联盟: 广东省低空经济产业协会(标准)
└─ 潜在合作方: 中国商飞(传闻接触), 顺丰无人机(物流场景)
```

## 四、风险信号识别

- **排他协议风险**：与TI的排他协议虽保障供应，但可能阻碍与国产芯片厂商的合作尝试。  
- **客户集中度**：大疆订单占比超30%，若大疆自研飞控，营收将受较大冲击（目前大疆尚未公开自研计划）。  
- **适航不确定性**：eVTOL适航标准尚未完全落地，若政策延迟，合作研发投入可能无法快速转化。

## 五、对我方（招商人员）的启示

1. **谈判切入点**：强调“本地供应链配套能力”可降低其对外部传感器依赖；展示“适航服务绿色通道”缩短其取证周期。  
2. **差异化优势**：我方可提供与TI同等性能的国产芯片替代方案（如有），或对接本地传感器企业（如XX科技）进入其实验室计划。  
3. **行动建议**：在接触时主动提及“我们注意到贵司与瑞声共建了传感器实验室，我园区有一家做IMU的初创企业，也许可以补充您的供应链弹性”。

> 数据截至2026年5月26日。推演内容基于公开事实的逻辑推断，不构成投资建议。

### 错误处理与注意事项

| 错误类型 | 处理方式 |
|---------|----------|
| 近12个月无任何战略合作新闻 | 返回“该企业近12个月内未公开披露重大战略合作，请尝试扩大时间范围或检查企业名称” |
| 仅有框架协议无实质内容 | 标注“该合作公开信息较少，实质落地情况待观察” |
| 合作方为非公开信息（如传言） | 若`include_rumor=false`则忽略，否则标注“未经官方证实” |
| 多条合作高度同质化 | 仅展示最具代表性的Top 5，其余附注说明 |

### 扩展能力
- **合作时效性监测**：设置定期（每月）重新搜索，输出新发生合作提醒。  
- **竞品对比**：输入两家企业，对比其合作网络密度和战略路径差异。  
- **合作价值评分**：基于合作方的行业地位、投入金额、战略匹配度，给出1-5星评分。
