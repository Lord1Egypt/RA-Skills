## Description: <br>
OpenClaw Task Reminder helps OpenClaw agents create, list, prioritize, complete, and clear locally stored to-do items. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and OpenClaw users use this skill to manage local task reminders from the command line, including adding prioritized tasks, listing them, marking tasks done, and clearing completed work. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Task descriptions are saved in a local JSON file and may persist sensitive information. <br>
Mitigation: Do not enter passwords, tokens, or sensitive private information in task descriptions; review or delete saved tasks when needed. <br>
Risk: Registry metadata does not include independent source provenance for this release. <br>
Mitigation: Review the ClawHub publisher profile, installed files, and security summary before relying on the skill. <br>
