# 机票全网比价 Skill

自动对比飞猪、携程、去哪儿等平台的机票价格，帮您找到全网最低价！

---

## 🚀 快速开始

### 前置准备

1. **配置账号信息** - 在 `TOOLS.md` 中添加各平台账号密码
2. **确保浏览器可用** - 使用内置 `openclaw` profile
3. **提供查询信息** - 出发地、目的地、日期、单程/往返

### 使用示例

```
# 简单查询
帮我查一下 3 月 20 日北京到上海的机票

# 往返查询
查 3 月 20 日北京飞三亚，3 月 25 日返回的机票

# 指定平台
用携程和飞猪对比一下明天广州到成都的机票价格

# 多人查询
查一下 4 月 1 日深圳到东京，2 大 1 小的机票
```

---

## 📋 查询参数

| 参数 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 出发地 | ✅ | 城市名或机场代码 | 北京/PEK |
| 目的地 | ✅ | 城市名或机场代码 | 上海/SHA |
| 出发日期 | ✅ | 日期格式 | 2026-03-20/明天/下周一 |
| 返回日期 | ❌ | 往返时需要 | 2026-03-25 |
| 舱位等级 | ❌ | 经济舱/商务舱/头等舱 | 经济舱（默认） |
| 乘客信息 | ❌ | 成人/儿童/婴儿数量 | 1 成人（默认） |

---

## 🔧 脚本使用说明

### 基本查询

```bash
python scripts/search_flights.py --from 北京 --to 上海 --date 2026-03-20
```

### 往返查询

```bash
python scripts/search_flights.py --from 北京 --to 三亚 --date 2026-03-20 --return-date 2026-03-25
```

### 指定平台

```bash
python scripts/search_flights.py --from 广州 --to 成都 --date 明天 --platforms 携程，飞猪
```

### 多人查询

```bash
python scripts/search_flights.py --from 深圳 --to 东京 --date 4 月 1 日 --adults 2 --children 1
```

## 🎯 高级功能

### 价格监控

设置 cron 任务监控特定航线价格变化：

```
每天 9:00 监控北京 - 三亚航线，低于 1000 元时通知我
```

### 灵活日期比价

对比前后 3 天的价格，找最便宜的出行日期：

```
查北京到上海，3 月 20 日前后 3 天哪天最便宜
```

### 多城市对比

同时查询多个目的地的价格：

```
查 3 月 20 日北京出发，到三亚/厦门/昆明，哪里最便宜
```

## ⚠️ 注意事项

1. **价格时效性** - 机票价格实时变动，查询结果仅供参考
2. **登录状态** - 首次使用需要登录，后续会保持会话
3. **验证码处理** - 遇到验证码需要人工介入
4. **会员价格** - 各平台会员价格可能不同，建议登录后查看
5. **退改签政策** - 低价票可能退改签限制多，请注意查看

## 🔧 故障排查

### 常见问题

**Q1: 无法登录平台**
- 检查账号密码是否正确
- 可能需要扫码登录
- 清除浏览器缓存后重试

**Q2: 查询结果为空**
- 检查日期是否已过期
- 检查航线是否存在
- 尝试其他日期

**Q3: 价格对比不准确**
- 确保各平台查询条件一致
- 注意是否包含机建燃油费
- 检查是否有平台优惠券
