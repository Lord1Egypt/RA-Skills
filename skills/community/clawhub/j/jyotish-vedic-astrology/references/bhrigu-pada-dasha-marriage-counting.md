## 第一部分：Bhrigu Pada Dasha（行星推进法）

### 1. 概念总览

**Bhrigu Pada Dasha** 是 Bhrigu 体系下的一种 Dasha 系统，不同于常见的 Vimshottari Dasha，也不同于之前已收录的 BCP（Bhrigu Chakra Paddhati 自然周期法）。

**核心原理**：
- 基于**行星推进**（Progression）的概念——与西方占星的 Secondary Progression 类似
- 在 Nadi 占星体系中发扬光大
- 主要用于**婚姻时机预测**，也可用于其他人生重大事件
- 与 BCP 互补：BCP 是固定 12 年周期，Pada Dasha 是基于命盘动态计算

### 2. 与其他 Dasha 的区别

| 特征 | Bhrigu Pada Dasha | BCP (自然周期法) | Vimshottari Dasha |
|------|------------------|-----------------|-------------------|
| **基础** | 行星推进法 | 9 个固定 12 年周期 | 月亮 Nakshatra |
| **计算** | 动态，基于命盘 | 固定，基于年龄 | 动态，基于月亮 |
| **主要用途** | 婚姻时机 | 人生阶段划分 | 所有事件时机 |
| **精确度** | 中-高（需配合其他） | 中（宏观） | 高（标准） |
| **适用盘** | D1 + D9 | D1 为主 | D1 + 分盘 |

### 3. Pada Dasha 的计算要点

Pada Dasha 的具体计算较为复杂，核心步骤涉及：

1. **确定起始点**：基于命盘中特定行星的 Pada（足迹/投射点）
2. **推进速率**：每年推进一定度数（具体数值因 Bhrigu 子流派而异）
3. **星座序列**：按特定顺序（不同于 Vimshottari 的 Nakshatra 顺序）
4. **D9 验证**：Pada Dasha 的结果需在 D9 中验证

> **重要**：Pada Dasha 的精确计算公式在不同 Bhrigu 子流派中有差异。建议在实战中与 Vimshottari/Chara Dasha 交叉验证。

### 4. Narayana Dasha 在 D9 分析中的变体

文章还提到 **Narayana Dasha** 的 D9 变体用于婚姻分析：

- **Narayana Dasha** 是基于星座的 Dasha 系统（非行星）
- 在 D9 上应用 Narayana Dasha 可以得到**婚姻专用的时间线**
- 与标准 Narayana Dasha 的区别：起始星座的确定规则不同
- 可与 Bhrigu Pada Dasha 交叉验证婚姻时机

### 5. 与其他 Bhrigu 方法的配合

## 第二部分：婚姻计数法（Marriage Counting Method）

### 6. 概念

这是 Sanjay Rath 传授的一个**传统秘密方法**，用于判断一个人一生中会有多少次婚姻（或持续一年以上的认真关系）。

### 7. 计算步骤

#### Step 1：确定 D1 第 7 宫主

找到本命盘（D1）中第 7 宫的主星。

#### Step 2：找 7 宫主在 D1 中的星座（A）

确定 7 宫主在 D1 中落入哪个星座 → 记为 **A**。

#### Step 3：找 7 宫主在 D9（Navamsa）中的星座（B）

确定同一颗 7 宫主在 D9 中落入哪个星座 → 记为 **B**。

#### Step 4：计数

**从 A 到 B 的距离 = 婚姻数量**（包含 A 和 B）

计数规则：
- 从 A 开始数（A = 1）
- 按星座顺序（白羊→金牛→双子→...→双鱼→白羊...）
- 数到 B 为止
- 得到的数字 = 婚姻/认真关系的数量

### 8. 计算示例

**示例1**：
- 7 宫主 = Venus
- Venus 在 D1 中 = Gemini（A）
- Venus 在 D9 中 = Virgo（B）
- 从 Gemini 数到 Virgo：Gemini(1)→Cancer(2)→Leo(3)→Virgo(4)
- **结果：4 次婚姻/认真关系**

**示例2**：
- 7 宫主 = Mars
- Mars 在 D1 中 = Capricorn（A）——入庙
- Mars 在 D9 中 = Capricorn（B）
- A = B → 距离 = 1
- **结果：1 次婚姻**（D1 和 D9 同一星座 = 忠诚）

### 9. Parivartana（行星交换）的特殊处理

如果 7 宫主与其落入星座的主星有**交换**（Parivartana）：

1. **原方法失效**——不能直接用 7 宫主的位置计数
2. **替代方法**：使用交换后的星座重新计算
   - 7 宫主在 D1 中实际落入的星座的主星 = X
   - X 的 D1 位置取代原 7 宫主的 D1 位置作为 A
   - 同理处理 D9
3. **重新计数**

**示例**：
- 7 宫主 = Venus，Venus 在 D1 中 = Scorpio（火星的星座）
- Mars 在 D1 中 = Libra（金星的星座）
- 这是 Parivartana！
- 新 A = Mars 在 D1 中的星座 = Libra
- 需要重新检查 D9 并重新计数

### 10. "婚姻"的定义

> **重要澄清**：此处的"婚姻"（marriage）指的是**持续一年以上的认真关系**，不一定是法律婚姻。

包括：
- 法律婚姻
- 长期同居关系（>1年）
- 认真的恋爱关系（>1年，有承诺性质）

不包括：
- 短暂恋爱（<1年）
- 一夜情/随意约会
- 未确认的暧昧关系

### 11. 计数结果的解读

| 计数结果 | 解读 |
|---------|------|
| 1 | 一次终身婚姻，忠诚度高 |
| 2 | 两次重要关系，可能再婚 |
| 3 | 多段关系，关系模式复杂 |
| 4+ | 关系频繁变化，需检视关系模式 |
| A = B（同星座）| D1/D9 一致 = 关系忠诚 |
| A 与 B 对冲 | 内在矛盾 = 关系中的拉锯 |

## 第三部分：整合应用

### 12. 婚姻时机与数量的综合分析

将 Bhrigu Pada Dasha（时机）与婚姻计数法（数量）结合：

1. 先用婚姻计数法确定**关系数量**（框架）
2. 再用 Bhrigu Pada Dasha 确定**每次关系的时机**
3. 用 Vimshottari/Chara Dasha 交叉验证
4. 用 D9 Navamsa 确认每段关系的**质量**

### 13. 解盘检查清单

1. [ ] 确定 D1 第 7 宫主
2. [ ] 记录 7 宫主在 D1 中的星座（A）
3. [ ] 记录 7 宫主在 D9 中的星座（B）
4. [ ] 检查是否有 Parivartana（需特殊处理）
5. [ ] 从 A 数到 B → 得出关系数量
6. [ ] 检查 A 是否 = B（忠诚度指标）
7. [ ] 应用 Bhrigu Pada Dasha 确定具体时机
8. [ ] 与 Vimshottari/Chara Dasha 交叉验证
9. [ ] 检查 DK（Darakaraka）状态作为辅助
10. [ ] 整合到婚姻分析的综合报告中

### 14. 注意事项

1. **Parivartana 必须重新计算**——这是最常见的错误
2. **计数包含起止点**——A 和 B 都算
3. **此法不是精确的时间预测工具**——它给出的是"数量"，不是"时间"
4. **与文化背景相关**——在传统文化中"婚姻"=法律婚姻，在现代社会需灵活解读
5. **与 Dasha 收敛结合**——婚姻计数法的结果应在多 Dasha 收敛分析中被验证
