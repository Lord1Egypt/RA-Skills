# 重型腹泻并发代谢性酸中毒与电解质紊乱处理

## 触发条件
- 腹泻患儿出现脱水且伴有精神不振、唇红、呼吸深大、呼出气有丙酮味 → 警惕代谢性酸中毒
- 纠正脱水后排尿后出现无力、腹胀、心律失常 → 警惕低钾血症
- 纠正脱水酸中毒后出现手足搐搦或惊厥 → 考虑低钙或低镁血症

## 执行流程

### 1. 识别代谢性酸中毒机制与表现
- **机制**：
  - 腹泻丢失大量碱性物质（如碳酸氢盐）
  - 热能不足致脂肪分解增加，产生酮体
  - 脱水致组织缺氧，乳酸堆积
  - 肾排酸功能下降，酸性代谢产物滞留
- **典型表现**：精神不振、唇红、呼吸深大、呼出气凉而有丙酮味（小婴儿可能不典型）

### 2. 判断低钾血症发生时机与处理
- **关键转折点**：脱水与酸中毒被纠正后，因排尿增加、继续失钾、钾向细胞内转移，血清钾迅速下降
- **处理原则**：
  - **必须在有尿后开始补钾**
  - 密切监测血清钾水平和心电图变化

### 3. 识别与处理低钙/低镁血症
- **特点**：脱水酸中毒时因离子钙增多，通常无症状；**纠正后才出现抽搐**
- **处理顺序**：
  - 出现惊厥先静脉补钙
  - 若补钙无效，考虑低镁血症，给予补镁

### 4. 整体监测与干预
- 在纠正脱水和酸中毒过程中，动态监测血气分析、电解质（尤其血钾、血钙、血镁）
- 根据尿量、酸中毒纠正状态和临床表现调整补液及电解质补充方案

> **注意**：避免在无尿时补钾；警惕纠正过程中电解质的“延迟性紊乱”。
