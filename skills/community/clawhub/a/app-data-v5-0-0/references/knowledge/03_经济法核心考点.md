# 经济法 · 核心考点总览

## 科目特点
- 内容杂，法条多，需要大量记忆
- 三门中相对简单，但容易混淆
- 重点章节：公司法、合同法、物权法、税法
- 考试特点：覆盖面广，重点突出，需要准确记忆

## 各章核心考点速查

### 第20章 公司法律制度（超级重点）
**核心考点：**
1. 公司的概念和种类：母公司与子公司、总公司与分公司
2. 公司法人财产权：对外投资、对外担保的限制
3. 有限责任公司：
   - 设立条件（股东人数、注册资本、公司章程）
   - 组织机构：股东会、董事会、监事会
   - 股权转让：内部转让、外部转让、异议股东回购请求权
   - 一人有限责任公司的特别规定
   - 国有独资公司的特别规定
4. 股份有限公司：
   - 设立条件和设立程序
   - 组织机构：股东大会、董事会、监事会
   - 上市公司组织机构的特别规定
5. 公司董事、监事、高级管理人员的资格和义务
6. 公司股票和公司债券：发行、转让、上市
7. 公司财务会计：公积金制度、利润分配
8. 公司合并、分立、增资、减资
9. 公司解散和清算
10. 违反公司法的法律责任

**易考点：**
- 有限责任公司股东会特别决议事项
- 股份有限公司股东大会的召开和决议
- 董事会的职权和议事规则
- 股权转让的限制条件
- 股东诉讼（直接诉讼、代表诉讼）

### 第21章 合同法律制度（超级重点）
**核心考点：**
1. 合同的概念和分类
2. 合同的订立：
   - 要约与承诺
   - 合同成立的时间和地点
   - 格式条款
   - 缔约过失责任
3. 合同的效力：
   - 有效合同
   - 无效合同
   - 可撤销合同
   - 效力待定合同
4. 合同的履行：
   - 履行规则
   - 抗辩权（同时履行、先履行、不安）
   - 保全措施（代位权、撤销权）
5. 合同的担保：
   - 保证
   - 抵押
   - 质押
   - 留置
   - 定金
6. 合同的变更和转让
7. 合同的权利义务终止
8. 违约责任：
   - 承担违约责任的形式
   - 违约责任的免除
9. 主要合同：
   - 买卖合同
   - 借款合同
   - 租赁合同
   - 融资租赁合同
   - 承揽合同
   - 建设工程合同

**易考点：**
- 要约与要约邀请的区别
- 合同效力的判断
- 三大抗辩权的适用条件
- 合同保全的代位权和撤销权
- 各种担保方式的特点和设立
- 违约责任的承担方式
- 买卖合同中标的物所有权转移和风险承担

### 第22章 物权法律制度（重点）
**核心考点：**
1. 物权法概述：
   - 物权的概念和种类
   - 物权法的基本原则
   - 物权的变动
2. 所有权：
   - 所有权的概念和权能
   - 业主的建筑物区分所有权
   - 相邻关系
   - 共有（按份共有、共同共有）
   - 所有权取得的特别规定（善意取得、拾得遗失物等）
3. 用益物权：
   - 用益物权的概念和种类
   - 土地承包经营权
   - 建设用地使用权
   - 宅基地使用权
   - 居住权
   - 地役权
4. 担保物权：
   - 担保物权的概念和种类
   - 抵押权（一般抵押权、最高额抵押权）
   - 质权（动产质权、权利质权）
   - 留置权
5. 占有：
   - 占有的概念和种类
   - 占有的保护

**易考点：**
- 物权变动的公示方式（登记、交付）
- 善意取得的构成要件和法律效果
- 抵押权的设立和实现
- 不同担保物权的优先顺序
- 留置权的适用条件

   - 保险法的基本原则
   - 保险合同
   - 人身保险合同
   - 财产保险合同
4. 信托法律制度：
   - 信托的概念和种类
   - 信托的设立
   - 信托财产
   - 信托当事人的权利和义务

**易考点：**

### 第24章 增值税与企业所得税（超级重点+计算量大）
**核心考点：**
1. 增值税法律制度：
   - 增值税的纳税人和征税范围
   - 增值税的税率和征收率
   - 增值税应纳税额的计算（一般计税方法、简易计税方法）
   - 增值税的税收优惠
   - 增值税的征收管理
   - 出口货物、劳务和跨境应税行为增值税的规定
2. 企业所得税法律制度：
   - 企业所得税的纳税人和征税对象
   - 企业所得税的税率
   - 应纳税所得额的计算
   - 收入总额
   - 税前扣除项目（成本、费用、税金、损失等）
   - 不得扣除项目
   - 资产的税务处理（固定资产、无形资产、长期待摊费用等）
   - 企业所得税的税收优惠
   - 企业所得税的征收管理

**重要税率/征收率：**
- 增值税基本税率：13%
- 增值税低税率：9%（交通运输、邮政、建筑、不动产等）
- 增值税低税率：6%（现代服务、生活服务、金融服务等）
- 增值税征收率：3%（小规模纳税人及一般纳税人简易计税）
- 企业所得税基本税率：25%
- 小型微利企业：20%（有减免）
- 高新技术企业：15%

**重要计算：**
- 增值税一般纳税人应纳税额 = 当期销项税额 - 当期进项税额
- 销项税额 = 不含税销售额 × 税率
- 不含税销售额 = 含税销售额 ÷ (1 + 税率)
- 企业所得税应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 以前年度亏损
- 企业所得税应纳税额 = 应纳税所得额 × 适用税率
