# EAI Awesome Work 写作模板

## 栏目格式

```
标题: EAI Awesome Work | [本期主题浓缩，6-12字]
副标题: 具身智能前沿工作速递 · Vol.[N] · YYYY.MM.DD
```

## 开头模板

开头是专栏最重要的部分，必须有"钩子"。以下是几种可选的写作策略：

### 策略A：范式变迁型
```
[领域]正在经历一场安静的范式地震。
去年[时间]，大家还在[老做法]。而今天，[谁]用[什么]给出了一个令人[形容词]的回答：[核心结论]。
与此同时，[第二条线索]这个[描述]的概念，本周被[什么事件]正式按下了确认键。
更微妙的变化发生在[第三条线索]……
```

### 策略B：对比冲突型
```
如果说上周的热词是"[上周主题]"，本周的主题就是**"[本周主题]"**。[谁]用[什么][做了什么]，而[谁]率先[做到了什么]。
```

### 策略C：直接冲击型
```
[一句话抛出最震撼的发现/数字]。
这就[结论/影响]。
本周我们精选了 N 篇论文，按[主题1]、[主题2]、[主题3]、[主题4]四个方向拆解。这不是一篇新闻简报，而是一次对"[领域]正在往哪里走"的认真回答。
```

**开头铁律**：
- 第一段不超过 3 句话
- 必须有至少一个具体数字或事实
- 必须点出本期 2-3 个核心信号
- 结尾必须有一句"为什么你要读"

## 板块正文模板

### 板块标题
```
## 🧠/🛡️/🦾/🏃/🤝/🔧 [板块英文名]：中文概括（8-12字）
```

### 板块导语
```
[板块名]方向的密度/变化/信号本周[描述]。以下 N 篇文章[共同特征]，[可串联的逻辑]。
```

### 单篇论文条目

🔴 = 必读（重大突破、行业级影响）
🟡 = 推荐（solid work、值得关注）

```
### 🔴/🟡 [中文论文标题/核心概括]

**[链接](https://arxiv.org/abs/XXXX.XXXXX)**

**机构 | 作者**

> **一句话**：[核心思路的口语化表达]

**摘要翻译**：
[流畅的中文翻译，保留技术细节，分行分段，不堆砌]

**结果/亮点**：
| 指标 | 结果 |
|------|------|
| [指标1] | [数字] |
| [指标2] | [数字] |

> 🏷️ **研发团队**：[机构]（[组/作者]）
> 📎 **代码/项目页**：[链接] 或 未公开
```

### 板块小结
```
### 📊 [板块]板块小结
[2-4 句话串联：这些文章之间的逻辑关系 + 该方向的整体趋势判断]
[可选：用简单 ASCII 图示意逻辑链]
```
