# 病毒性心肌炎诊断标准应用

## 何时使用
- 患儿在1–3周内有明确病毒感染史（如呼吸道或消化道感染）
- 出现以下任一表现：心功能不全、心源性休克、心脑综合征、心脏扩大、心肌酶升高伴动态变化、显著心电图异常
- 需与其他心肌损伤病因（如冠心病、中毒性心肌炎、先天性心脏病等）鉴别

## 执行步骤

### 第一步：确认排除标准
排除以下疾病后方可应用本诊断标准：
- 冠状动脉疾病
- 中毒性心肌炎
- 先天性心脏病
- 高原性心脏病
- 风湿性疾病
- 代谢性疾病（如甲亢）
- 原发性心肌病
- 心内膜弹力纤维增生症
- 先天性传导阻滞
- 自主神经功能异常
- β受体功能亢进
- 药物性心电图改变

### 第二步：收集主要临床诊断依据（每项计1分）
检查是否满足以下任一条件：
1. 心功能不全、心源性休克或心脑综合征
2. 影像学证实的心脏扩大
3. cTnI/cTnT 或 CK-MB 升高且呈动态变化
4. 显著心电图改变（持续≥4天且动态演变），包括：
   - ≥2个以R波为主导联（I、II、aVF、V5）的ST-T改变
   - 新近出现的窦房/房室传导阻滞、完全性束支阻滞、窦性停搏
   - 成对/多形性期前收缩、非AVNRT/AVRT的室上速、房颤/房扑、室扑/室颤
   - QRS低电压（新生儿除外）或异常Q波
5. 心脏磁共振（CMR）显示典型心肌炎症（满足以下≥2项）：
   - T2加权像高信号（水肿）
   - T1早期钆增强（充血/渗漏）
   - 非缺血区晚期延迟钆增强（坏死/纤维化）

### 第三步：收集次要临床诊断依据（每项计1分）
检查是否满足以下任一条件：
1. 前驱感染史（1–3周内）
2. ≥2项症状：胸闷、胸痛、心悸、乏力、头晕、面色苍白/发灰、腹痛；小婴儿表现为拒乳、发绀、四肢凉
3. LDH、α-HBDH 或 AST 升高
4. 心电图轻度ST-T异常（未达主要标准）
5. 抗心肌抗体阳性

### 第四步：应用确诊规则
- **临床诊断心肌炎**：主要依据≥3条，或主要依据2条 + 次要依据≥3条，且已排除其他疾病
- **病毒性确诊心肌炎**：在临床诊断基础上，加以下任一病原学确诊证据：
  - 心肌/心包活检分离到病毒、检出病毒核酸或特异性抗体
- **病毒性临床诊断心肌炎**：临床诊断 + 病原学参考依据（满足以下任一）：
  - 粪便/咽拭子分离到病毒 + 恢复期抗体滴度升高4倍
  - IgM阳性
  - 血中检出病毒核酸

### 第五步：输出诊断结论
根据上述评估，输出以下结论之一：
- “可临床诊断心肌炎”
- “可确诊病毒性心肌炎”
- “可临床诊断病毒性心肌炎”
- “需随访观察”
- “排除心肌炎”

> **注意**：不得仅凭单一指标（如CK-MB升高）作出诊断；必须结合动态心电图变化及其他综合证据。
