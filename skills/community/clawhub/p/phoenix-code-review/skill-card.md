## Description: <br>
Reviews Phoenix code for controller patterns, context boundaries, routing, and plugs. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to review Phoenix applications for controller structure, context boundaries, routing, plugs, JSON API behavior, and evidence-backed findings. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may produce incorrect or misleading review findings if it reports issues without checking related router, controller, plug, and context handling. <br>
Mitigation: Require anchored file and line evidence, a handled-elsewhere sweep, and the referenced review verification protocol before reporting each finding. <br>
Risk: The release declares broad capability tags that are not explained by the documentation-only artifact. <br>
Mitigation: Install only for Phoenix code-review guidance and do not grant financial, wallet, purchase, credential, or broad automation authority unless a future version explains those permissions. <br>
