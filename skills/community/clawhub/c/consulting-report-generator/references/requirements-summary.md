# 通用报告PPT生成技能 — 完整需求规格（v6 素材增强版）

> 核心升级：新增PPT素材智能搜集引擎 + 模板风格双因子选择器
> 生成PPT前先搜集素材、选好风格，再进入设计生成阶段

---

## 一、定位

### 核心能力
**输入任意内容 → 自动识别 → 自适应模板 → 生成专业报告 → 自进化学习**

### 自进化能力
- **技能发现**：每日扫描126+本地技能，智能推荐可集成的新能力
- **使用追踪**：记录每次执行的布局模式、内容类型和结果
- **错误学习**：记录每次错误与修复方案，避免重复问题
- **模板优化**：根据使用频率自动推荐最佳模板和布局

### 自修复能力（S13 新增）
- **问题自动收集**：每次生成后收集所有错误/警告/用户反馈
- **P0/P1自动修复**：致命/严重问题自动写入SKILL.md的「常见错误与修复」
- **函数规范更新**：函数实现缺陷自动更新助手函数表
- **交付清单扩展**：新发现的检查项自动追加
- **版本号管理**：P0级别修复自动升级小版本号
- **行业图片搜索**：基于内容关键词自动搜索高质量行业配图
- **装饰图标搜集**：搜索并组织装饰性图标资源
- **参考PPT分析**：搜索同行业PPT设计，提取配色/布局/字体参考
- **配色方案识别**：从参考源提取配色方案
- **素材本地缓存**：下载并保存素材至本地缓存目录

### 风格选择能力（v6 新增）
- **双因子推荐算法**：内容类型 × 用户偏好 → 最优风格
- **4种风格模板**：deepblue(咨询专业) / tech(科技现代) / business(沉稳专业) / vibrant(热情鲜明)
- **用户确认机制**：展示素材预览+风格推荐，用户可调整

### 架构 — 九层设计
```
┌─ 自进化层: 技能发现引擎 + 使用分析 + 错误学习 + 模板优化
├─ 识别层: 内容类型自动检测 + 自适应模板选择
├─ 设计层: mck-ppt-design（70种布局模式）
├─ 素材层: PPT素材搜集 + 模板风格选择引擎（v6新增）
├─ 研究层: 深度研究引擎 + 多源交叉验证（v5.1新增）
├─ 内容层: 联网搜索（免API）+ 内容扩增引擎
├─ 提取层: OCR识别 + 文档解析 + 图片文字提取
├─ 分析层: 通用分析（所有模式）| 制造专家（仅制造领域）
└─ 输出层: python-pptx + 微软雅黑 + humanizer-zh + full_cleanup + AUDIT
```

## 二、自进化系统（v5 新增）

### 技能发现引擎
- 每次技能加载时自动扫描 `~/.workbuddy/skills/`
- 评估新技能与报告生成的相关性（关键词匹配）
- 发现的技能自动加入技能生态矩阵
- 新技能信息存储在 `self_evolution/known_skills.json`

### 进化数据存储
```
self_evolution/
├── known_skills.json       # 已知技能列表
├── usage_log.json          # 使用记录
├── error_log.json          # 错误日志
├── improvements.json       # 改进建议
├── template_stats.json     # 模板使用统计
└── evolution_summary.md    # 进化摘要
```

### 自进化执行流程
```
技能调用 → 加载进化数据 → 扫描新技能 → 执行生成 → 记录使用 → 更新日志
```

### 工作流（S0-S15）
新增 S0.5: 自进化检查（与内容分析并行）
- 加载 known_skills / usage_log
- 扫描新技能
- 检查上次错误是否已修复
- 加载高频使用的布局模式偏好

新增 S6.5: PPT素材搜集与模板风格选择（v6 核心新增）
- 基于内容类型和关键词自动搜索行业图片/图标/参考PPT/配色方案
- 双因子推荐算法选择最佳模板风格（内容因子60%×用户偏好40%）
- 素材预览与风格确认（用户可选调整）
- 素材引用信息嵌入生成脚本

新增 S13: 技能自修复
- 每次生成后自动收集所有问题/错误/用户反馈
- P0/P1级问题自动写入SKILL.md的常见错误章节
- 函数实现缺陷自动更新助手函数表
- 新发现的检查项自动追加到交付清单

新增 S14: 文件交付与微信推送
- PPT生成后通过 deliver_attachments 推送到微信小程序
- 提示用户开启「产物回传到小程序」开关
- 写入交付日志 tracking

## 三、质量门禁

### 自审（AUDIT）— 质量优先，页数不限
| 检查项 | 合格线 |
|:-------|:------:|
| 图形密度 | ≥ 10 图/页 |
| 微软雅黑覆盖 | ≥ 80% |
| 无 BLOCK_ARC / DIAMOND | 零容忍 |

### 交付检查清单（18项）
- [ ] 微软雅黑 + 英文Arial
- [ ] 无 BLOCK_ARC / 无 DIAMOND
- [ ] 使用 mck-ppt-design 布局模式
- [ ] 内容充分展开：每个模块按内容深度自然分页，无页数压缩
- [ ] 用户指定页数时：已按比例压缩但保证完整性和连贯性
- [ ] 数据呈现：每个数据点获得充分展示空间
- [ ] 来源标注完整（三级标注）
- [ ] 执行 full_cleanup()
- [ ] AUDIT() 通过（图形密度≥10，无BLOCK_ARC）
- [ ] 去AI味自检通过
- [ ] 用户资料内容已扩增
- [ ] 交付 .pptx 文件
- [ ] 自进化日志已写入
- [ ] 技能发现扫描已执行
- [ ] 【v6新增】素材搜集已执行（≥3种类型，≥5个条目）
- [ ] 【v6新增】模板风格已确认
- [ ] 【v6新增】素材引用已嵌入生成脚本
- [ ] 【v6新增】配色主题应用正确
