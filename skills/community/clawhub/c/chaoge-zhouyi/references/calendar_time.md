## 第二章：时间起卦法（梅花易数）

### 一、年月日时起卦

**公式**：
```
上卦 = (年支数 + 月数 + 日数) ÷ 8，取余数
下卦 = (年支数 + 月数 + 日数 + 时支数) ÷ 8，取余数
动爻 = (年支数 + 月数 + 日数 + 时支数) ÷ 6，取余数
```

**说明**：
- 余数=0时，取8
- 动爻=0时，取6

### 二、数字对应

#### 1. 地支数（用于起卦）

| 地支 | 数 | 地支 | 数 |
|------|------|------|------|
| 子 | 1 | 午 | 7 |
| 丑 | 2 | 未 | 8 |
| 寅 | 3 | 申 | 9（取1） |
| 卯 | 4 | 酉 | 10（取2） |
| 辰 | 5 | 戌 | 11（取3） |
| 巳 | 6 | 亥 | 12（取4） |

**说明**：大于8的数，除以8取余数。

#### 2. 先天八卦数

| 卦 | 数 | 卦 | 数 |
|------|------|------|------|
| 乾 | 1 | 巽 | 5 |
| 兑 | 2 | 坎 | 6 |
| 离 | 3 | 艮 | 7 |
| 震 | 4 | 坤 | 8 |

### 三、起卦示例

**时间**：2026年6月4日 12:22（午时）

**步骤**：
1. **年支数**：午=7
2. **月数**：五月=5
3. **日数**：初九=9
4. **时支数**：午=7

**计算**：
```
上卦 = (7 + 5 + 9) ÷ 8 = 21 ÷ 8 = 2...5 → 余数5 = 巽卦
下卦 = (7 + 5 + 9 + 7) ÷ 8 = 28 ÷ 8 = 3...4 → 余数4 = 震卦
动爻 = (7 + 5 + 9 + 7) ÷ 6 = 28 ÷ 6 = 4...4 → 余数4 = 四爻动
```

**结果**：
```
本卦：风雷益䷩（巽上震下）
动爻：四爻
变卦：天雷无妄䷘（乾上震下）
```

## 第八章：Python计算代码

### 一、获取当前时间

```python
from datetime import datetime

now = datetime.now()
print(f"当前时间：{now.strftime('%Y-%m-%d %H:%M:%S')}")
```

### 二、计算农历

```python
# 需要安装 lunarcalendar 库
# pip install lunarcalendar

from lunarcalendar import Converter

solar = datetime(2026, 6, 4)
lunar = Converter.Solar2Lunar(solar)

print(f"农历：{lunar.year}年 {lunar.month}月 {lunar.day}日")
```

### 三、计算干支

```python
# 需要安装 lunarcalendar 库

from lunarcalendar import Converter

solar = datetime(2026, 6, 4)
lunar = Converter.Solar2Lunar(solar)

print(f"年柱：{lunar.year_ganzhi}")
print(f"月柱：{lunar.month_ganzhi}")
print(f"日柱：{lunar.day_ganzhi}")
```

### 四、计算时辰

```python
from datetime import datetime

now = datetime.now()

if 23 <= hour < 1:
```

## 第十章：注意事项

### 一、时间精度

1. **公历时间**：精确到分钟
2. **农历时间**：精确到日
3. **干支时间**：精确到时辰
4. **节气**：精确到日

### 二、时区问题

1. **北京时间**：UTC+8
2. **真太阳时**：需根据经度修正
3. **起卦时间**：用北京时间即可

### 三、节气交接

1. **节气交接时刻**：需精确计算
2. **交接前后**：可能用不同月柱
3. **奇门局数**：交接前后可能不同

### 四、闰月问题

1. **农历闰月**：需特别标注
2. **月柱计算**：闰月用不同月柱
3. **起卦计算**：用实际月数
