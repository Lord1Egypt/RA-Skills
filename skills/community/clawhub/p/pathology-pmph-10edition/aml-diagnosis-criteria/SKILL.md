# 急性髓系白血病（AML）诊断标准应用

## 何时使用
- 患者出现贫血、出血、感染、骨痛等疑似白血病症状
- 外周血或骨髓涂片发现原始细胞增多
- 需要依据WHO最新分类标准确认是否为AML
- 排除急性淋巴细胞白血病等非髓系来源白血病

## 执行步骤

1. **评估原始细胞比例**
   - 获取骨髓穿刺液涂片或外周血涂片
   - 计算原始髓系细胞占全部有核细胞的百分比
   - 若 ≥20%，直接诊断为AML

2. **检查特定遗传学异常（即使原始细胞 <20%）**
   - 若存在以下任一异常，仍应诊断为AML：
     - PML::RARA [t(15;17)(q24;q21)]
     - RUNX1::RUNX1T1 [t(8;21)(q22;q22)]
     - CBFB::MYH11 [t(16;16)(p13;q22) 或 inv(16)(p13;q22)]
     - DEK::NUP214 [t(6;9)(p22;q34)]
     - RBM15::MRTFA [t(1;22)(p13;q13)]
     - KMT2A、MECOM 或 NUP98 基因重排
     - NPM1 或 CEBPA 基因突变

3. **处理特殊临床情况**
   - 对于非白血性白血病（外周血原始细胞极少或无），必须依赖骨髓活检确诊
   - 若存在髓外浸润（如髓系肉瘤），需通过病理活检结合免疫组化（如MPO阳性）确认髓系来源

4. **输出诊断结论**
   - 明确输出“符合AML诊断”或“不符合AML诊断”
   - 注明依据：形态学标准（原始细胞≥20%）或分子遗传学异常

> 注意：本标准不适用于非髓系来源的急性白血病（如急性淋巴细胞白血病）。
