# Earnings Analysis — 财报深度分析 Skill

个股财报深度分析工具，覆盖营收、盈利、现金流、机构观点全维度。

## 功能

- 📈 核心数据：近4季度营收/净利润及增速、ROE、毛利率、净利率
- 💰 质量分析：经营现金流/净利润比值、盈利趋势、PEG估值匹配
- 🏦 机构观点：近30日评级调整、目标价共识
- 📅 财报日历：下一财报发布日期预测

## 触发方式

- 说 "财报分析" / "看XX财报" / "earnings analysis"
- 提到具体股票代码+财报相关词

## 依赖

- `neodata-financial-search` skill

## 输出

Markdown 财报分析报告，保存至 `earnings-analysis/reports/YYYY-MM-DD-{代码}-earnings.md`
