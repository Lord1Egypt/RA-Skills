# Aky TCM Prescription — Formula Analysis & Pattern Correspondence (法证对应/理法方药/君臣佐使)

Core analysis framework integrating 法证对应 (pattern-principle correspondence) and 理法方药 (principle-method-prescription-medicine) learning methods.

---

## 1. 法证对应 Framework (Pattern-Principle Correspondence)

**Core principle:** "方是死的，法是活的" — Formulas are fixed, principles are alive.
**Advanced clinical thinking:** Move beyond "pattern→formula" direct mapping to "pattern elements→treatment principles→herb selection."

### Workflow

```
Step 1: Decompose Pattern Elements (拆证素)
    Break the pattern into fundamental pathological units.
    病性 elements: 气虚, 血虚, 阴虚, 阳虚, 气滞, 血瘀, 痰湿, 水饮, 寒凝, 热(火), 郁
    病位 elements: 表/里, 上/中/下焦, specific zang-fu organs, 六经, 卫气营血

Step 2: Establish Treatment Principles (立治法)
    Each pattern element → corresponding treatment principle.
    ┌────────────────┬──────────────────────┐
    │ Element        │ Treatment Principle   │
    ├────────────────┼──────────────────────┤
    │ 气虚           │ 益气 (supplement qi)  │
    │ 血虚           │ 养血 (nourish blood)  │
    │ 阴虚           │ 滋阴 (nourish yin)    │
    │ 阳虚           │ 温阳 (warm yang)      │
    │ 气滞           │ 行气/疏肝/降逆      │
    │ 血瘀           │ 活血化瘀             │
    │ 痰湿           │ 燥湿/化湿/利湿      │
    │ 寒凝           │ 温散                 │
    │ 热(火)         │ 清热/泻火            │
    │ 水饮           │ 温阳利水/逐水       │
    └────────────────┴──────────────────────┘

Step 3: Select Properties & Meridians (定性味归经)
    Bridge from principle to herbs:
    - 益气 → 甘温之品 (sweet-warm substances)
    - 养阴 → 甘寒之品 (sweet-cool substances)
    - 活血 → 辛温或苦温 (pungent-warm or bitter-warm)
    - 清热 → 苦寒之品 (bitter-cold substances)
    - 化痰 → 辛苦温燥 (pungent-bitter-warm-drying)

Step 4: Select Herbs (选药物)
    Apply herb selection based on:
    - 性味归经 matching
    - Upper/middle/lower body targeting
    - Herb comparison (see Section 4)
    - Compatibility considerations

Step 5: Explain Modifications (明加减)
    Principle unchanged, herbs can vary:
    - "法不变而药可变" — demonstrate how the same principle is applied with different herbs
```

### Example: 半夏泻心汤 Under 法证对应

```
【证素拆解】
- 寒(脾寒): 下利, 腹满, 喜温
- 热(胃热): 心下痞, 恶心, 口苦
- 虚(中虚): 纳差, 乏力, 病程久
- 痞(气机痞塞): 心下痞满, 按之不痛

【治法确立】
- 辛开散寒(升清): 针对脾寒
- 苦降清热(降浊): 针对胃热
- 甘温补虚(调和中州): 针对中虚
- 辛开苦降甘补复法 → 恢复中焦升降

【药性需求 → 药物选择】
- 辛温散寒: 半夏(辛温燥湿)、干姜(辛温守中)
- 苦寒清热: 黄芩(苦寒清胃热)、黄连(苦寒清心胃之火)
- 甘温补虚: 人参(甘温益气)、炙甘草、大枣(甘温补脾)
- 配伍结构: 辛+苦+甘 = 开+降+补 = 恢复升降

【法不变而药可变】
- 湿气重: 加苍术、厚朴、茯苓、薏苡仁
- 虚寒明显: 减芩连, 重用干姜、人参, 加炮姜、桂枝
- 兼食积: 加山楂、神曲、麦芽
- 兼气滞: 加枳实、厚朴
- 上热重: 加栀子、竹茹
```

## 2. 理法方药 Formula Learning Method

Systematic formula study: 病机 → 治法 → 性味归经 → 方药配伍 → 方歌

### Step-by-Step

**Step 1: Establish Formula Baseline**
- 方名, 出处, 组成, 主治, 功用, 证候表现, 舌脉

**Step 2: Reconstruct Pathomechanism (还原病机)**
Analyze by four dimensions:
- 病因 (Etiology): external contraction, internal damage, prolonged illness, mistreatment
- 病位 (Location): zang-fu, channels, qi-blood-fluids, triple burner, wei-qi-ying-xue, six meridians
- 病性 (Nature): cold/heat, deficiency/excess, dampness/dryness, phlegm/stasis
- 病势 (Dynamics): ascending/descending abnormalities, transmission tendency

**Step 3: Derive Treatment Principle (推导治法)**
Format: "Because [mechanism], therefore [principle]."
- Main principle, auxiliary principle, risk management

**Step 4: Properties & Meridians (性味归经)**
Before naming specific herbs, determine what properties are needed:
- 四气 (cold/hot/warm/cool), 五味 (pungent/sweet/sour/bitter/salty)
- 升降浮沉 (ascending/descending/floating/sinking)
- 归经 (meridian entry)

**Step 5: Herb Selection & Roles (方药与君臣佐使)**
For each herb, explain:
- Which mechanism layer it addresses
- Which treatment principle it actualizes
- Its property-meridian matching
- Its role: 君(monarch), 臣(minister), 佐(assistant), 使(messenger)
- Its compatibility structure with other herbs

**Step 6: Analyze Compatibility Structure (配伍结构)**
Key relationships:
- 相须 (mutual reinforcement): similar herbs with enhanced effect
- 相使 (mutual assistance): one herb assists another's primary effect
- 相畏/相杀 (mutual restraint): one reduces another's toxicity
- 一补一敛 (supplement + astringe): e.g., Huangqi + Fangfeng
- 一散一收 (disperse + collect): e.g., Guizhi + Baishao
- 一升一降 (ascend + descend): e.g., Chaihu + Shengma + Dahuang
- 一寒一热 (cold + heat): e.g., Ganjiang + Huanglian
- 一气一血 (qi + blood): e.g., Danggui + Chuanxiong
- 一燥一润 (dry + moisten): e.g., Cangzhu + Maidong
- 一攻一守 (attack + guard): purgative + supplementing

**Step 7: Generate Memory Formula Song (方歌)**
Compress into a singable verse: mechanism keywords + herbs + roles.

## 3. 君臣佐使 Analysis (Monarch-Minister-Assistant-Messenger)

| Role | Function | Characteristics |
|:----:|----------|----------------|
| 君 (Monarch) | Addresses primary pattern/disease | Largest dose, most essential action |
| 臣 (Minister) | Supports monarch; addresses secondary patterns | Reinforces or broadens therapeutic effect |
| 佐 (Assistant) | Restricts toxicity; treats concurrent symptoms; opposite guidance | 佐助(assist), 佐制(restrain), 反佐(opposite) |
| 使 (Messenger) | Harmonizes formula; directs to meridian | 引经(meridian guide), 调和(harmonizer) such as Gancao |

**Example: 麻黄汤**

```
君: 麻黄 — pungent-warm, releases exterior cold, opens lung, stops asthma
臣: 桂枝 — reinforces sweating effect, warms channels, relieves pain
佐: 杏仁 — descends lung qi, treats dyspnea (concurrent symptom)
使: 炙甘草 — harmonizes, moderates Mahuang's intensity
```

## 5. Key Clinical Formulas with 配伍结构 & 方歌 (Selections from formula-database.md)

### 桂枝汤 (Guizhi Tang)
- **病机:** 风邪袭表, 卫强营弱
- **治法:** 解肌发表, 调和营卫
- **配伍结构:** 桂枝+白芍 = 散中有收, 散不伤正, 收不留邪 (one disperses, one astringes — perfect balance)
- **方歌:** 桂枝汤治太阳风，芍药甘草姜枣同，解肌发表调营卫，表虚自汗此为宗

### 小柴胡汤 (Xiao Chaihu Tang)
- **病机:** 邪犯少阳, 正邪分争, 枢机不利
- **治法:** 和解少阳, 调达枢机
- **配伍结构:** 柴胡+黄芩 = 一散一清 (disperse + clear); 人参+甘草+大枣 = 扶正祛邪 (support + expel); 半夏+生姜 = 降逆和胃 (descend + harmonize)
- **方歌:** 小柴胡汤和解功，半夏人参甘草从，更加黄芩生姜枣，少阳百病此为宗

### 麻黄汤 (Mahuang Tang)
- **病机:** 风寒束表, 卫阳被遏, 肺气失宣
- **治法:** 发汗解表, 宣肺平喘
- **配伍结构:** 麻黄+桂枝 = 相须为用, 发汗力强; 麻黄+杏仁 = 宣降相因 (open + descend); 甘草 = 调和缓峻
- **方歌:** 麻黄汤中用桂枝，杏仁甘草四般施，发热恶寒头项痛，喘而无汗服之宜

### 理中丸 (Lizhong Wan)
- **病机:** 中焦虚寒, 运化失职
- **治法:** 温中祛寒, 补气健脾
- **配伍结构:** 干姜温中+人参补气 = 一温一补 (warm + supplement), 脾阳得温, 脾气得补
- **方歌:** 理中丸主理中乡，甘草人参术干姜，呕利腹痛阴寒盛，或加附子总扶阳

### 四逆汤 (Sini Tang)
- **病机:** 少阴寒化, 心肾阳衰, 阴寒内盛
- **治法:** 回阳救逆
- **配伍结构:** 生附子+干姜 = 相须, 大辛大热力挽沉疴; 炙甘草 = 制约毒性, 缓急留药 (restrict toxin + retain medicine)
- **方歌:** 四逆汤中附草姜，四肢厥冷急煎尝，腹痛吐泻脉微细，急投此方可回阳

### 六味地黄丸 (Liuwei Dihuang Wan)
- **病机:** 肾阴亏虚, 水不涵木, 虚火上炎
- **治法:** 滋阴补肾
- **配伍结构:** "三补三泻" — 熟地+山茱萸+山药 = 补 (supplement), 泽泻+丹皮+茯苓 = 泻 (drain); 补中兼泻, 寓泻于补, 使滋补而不留邪, 降泄而不伤正
- **方歌:** 六味地黄益肝肾，山药丹泽苓三泻，三补三泻经典法，阴虚火旺此方安

### 血府逐瘀汤 (Xuefu Zhuyu Tang)
- **病机:** 瘀血内停胸胁, 气机阻滞
- **治法:** 活血化瘀, 行气止痛
- **配伍结构:** 桃仁+红花+赤芍+川芎+牛膝 = 活血 (blood-moving), 柴胡+枳壳 = 行气 (qi-moving), 生地+当归 = 养血 (blood-nourishing), 桔梗载药上浮 (guide upward), 牛膝引血下行 (guide downward) — 一升一降一身气血通
- **方歌:** 血府当归生地桃，红花甘草壳赤芍，柴胡芎桔牛膝等，血化下行不作劳

### 补中益气汤 (Buzhong Yiqi Tang)
- **病机:** 脾胃气虚, 中气下陷
- **治法:** 补中益气, 升阳举陷
- **配伍结构:** 黄芪+人参+白术+甘草 = 补气; 升麻+柴胡 = 升举清阳; 当归 = 补血 (配伍: 补气生血); 陈皮 = 理气 (配伍: 补而不滞)
- **方歌:** 补中益气芪术陈，升柴参草当归身，劳倦内伤功独擅，气虚下陷亦堪珍

### 小青龙汤 (Xiao Qinglong Tang)
- **病机:** 外寒内饮, 寒饮射肺
- **治法:** 解表散寒, 温肺化饮
- **配伍结构:** 麻+桂 = 解表散寒; 姜+辛+夏 = 温化水饮; 五味子+白芍 = 收敛固护 (prevent over-dispersing, protect yin/qi); 甘草 = 调和
- **配伍精要:** 散中有收, 开中有合 — 这是小青龙配伍的精华所在
- **方歌:** 小青龙汤麻桂芍，细辛干姜五味草，半夏温肺化寒饮，外寒内饮此方好
