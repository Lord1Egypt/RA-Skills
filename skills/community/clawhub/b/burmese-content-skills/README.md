# Burmese Content Skills

A Burmese writing skill for Claude Code, OpenCode and AI writing tools.

This skill rewrites AI-generated Burmese text into natural Burmese writing that sounds like real people talking.

Instead of formal Burmese writing like:

- "ဖြစ်သည်"
- "၏"
- "ဆောင်ရွက်သည်"
- "အရေးကြီးသော"

The skill prefers natural Burmese conversational styles like:

- "ဖြစ်တယ်"
- "လုပ်တာ"
- "သိလား"
- "ခင်များတို့"
- "ကျွန်တော်ကတော့"
- "ကျွန်မကတော့"
- "တကယ်တော့"
- "ဆိုတော့က"
- "အဲ့တာက"
- "ဘယ်လိုလဲ ဆိုရင်"

The goal is to make Burmese content feel:

- Human
- Natural
- Emotional
- Social-media friendly
- Reader-friendly
- Burmese-native sounding

## Writing Style

This skill avoids robotic Burmese writing and uses real conversational Burmese.

Example:

### Before

"ဤနည်းလမ်းသည် လူတို့၏လုပ်ငန်းဆောင်ရွက်မှုကို တိုးတက်စေသည်"

### After

"ဒီနည်းလမ်းက လူတွေအလုပ်လုပ်တာ ပိုမြန်သွားစေတယ်"
