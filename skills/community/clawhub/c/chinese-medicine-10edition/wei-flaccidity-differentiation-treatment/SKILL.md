# Wei Flaccidity Differentiation Treatment

## When to Use
当患者出现以下任一情况时启动本技能：
- 肢体软弱无力、不能随意运动
- 肌肉萎缩、筋脉弛缓
- 伴有起病缓急信息、系统伴随症状及舌脉特征

## Execution Steps

### 1. 辨脏腑病位
根据症状组合判断病位所在：
- **肺**：起病急，伴发热、咳嗽咽痛，或热病后突发痿弱，皮肤干燥，舌红苔黄，脉细数 → 肺热津伤
- **脾胃**：四肢痿软伴乏力、食少便溏、腹胀纳呆、面色萎黄，舌淡脉细弱 → 脾胃虚弱
- **肝肾**：起病缓，腰膝酸软，腿胫肌肉萎缩，伴眩晕耳鸣、遗精或月经不调，舌红少苔，脉细数 → 肝肾亏损
- **湿热**：身体困重，足胫热气上腾，小便短赤，舌红苔黄腻，脉濡数 → 湿热浸淫
- **瘀阻**：有外伤史，四肢痿软日久，青筋显露，舌有瘀斑，脉涩 → 脉络瘀阻

### 2. 辨标本虚实
- **实证**：外感温毒或湿热浸淫，起病急、进展快
- **虚证**：内伤久病，起病慢、病程长
- **虚实夹杂**：长期病程中兼有实邪表现

### 3. 确定治疗总原则
- 遵循“治痿者，独取阳明”，重视调理脾胃
- **虚证**：扶正补虚（滋阴、益气、补肝肾）
- **实证**：祛邪和络（清热、利湿、活血）
- **虚实夹杂**：攻补兼施

### 4. 推荐具体治法与主方
- 肺热津伤：清热润燥，养阴生津 → 清燥救肺汤
- 湿热浸淫：清热利湿，通利经脉 → 加味二妙丸
- 脾胃虚弱：补中益气，健脾升清 → 参苓白术散
- 肝肾亏损：补益肝肾，滋阴清热 → 虎潜丸
- 脉络瘀阻：益气养营，活血化瘀 → 补阳还五汤

### 5. 注意事项
- 严重痿证需警惕并发症（如呼吸困难、压疮、关节挛缩）
- 治疗全程应评估脾胃功能状态，必要时加强健脾益气

## Output
输出内容必须包含：证型名称、病位脏腑、虚实属性、核心治法、推荐主方、是否需强化脾胃调理。
