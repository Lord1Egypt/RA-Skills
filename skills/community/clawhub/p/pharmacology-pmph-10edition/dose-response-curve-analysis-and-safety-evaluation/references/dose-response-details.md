# 量-效关系详细参数与计算方法

## 1. 量反应曲线数学表达
典型Hill方程：
E = Emax × [C]^n / (EC50^n + [C]^n)
其中：
- E：观测效应
- C：药物浓度
- n：Hill系数（反映曲线陡峭程度）

## 2. 质反应数据处理
- 使用Probit或Logit回归将S形曲线线性化
- ED50/LD50通过插值法或软件拟合获得

## 3. 安全性指标扩展
- **安全范围**（Margin of Safety）：LD1 / ED99
- **治疗窗**（Therapeutic Window）：有效浓度范围与毒性浓度范围之间的区间
