# 知行合一实践模板

## 使用场景
- 学习新知识后立即实践
- 设定目标并执行
- 检验是否真正理解

## 步骤

### 1. 知：明确认知
**问题：** 我要学/理解什么？
- 概念/知识点：____________________
- 核心原理：____________________
- 预期结果：____________________

### 2. 行：立即行动
**问题：** 我现在就能做什么？

| 行动项 | 完成标准 | 截止时间 |
|--------|----------|----------|
| | | |
| | | |
| | | |

### 3. 检验：知行对照
**问题：** 我做到了吗？做不到说明还没真懂。

- [ ] 我能解释清楚这个概念吗？
- [ ] 我能在实际中应用吗？
- [ ] 我能教别人吗？

### 4. 反思：深化理解
**问题：** 行动中我学到了什么？

- 新的认知：____________________
- 需要调整的：____________________
- 下一步：____________________

## 知行合一自检清单

- [ ] 学到即行动，不拖延
- [ ] 行动中检验理解
- [ ] 失败后反思，不是放弃
- [ ] 边做边学，循环提升

## 示例

**学习项目管理：**
1. **知：** 学习甘特图、里程碑、风险管理的概念
2. **行：** 立即用新方法管理当前项目
3. **检验：** 项目是否按时交付？风险管理有效吗？
4. **反思：** 哪些方法有用？哪些需要调整？
