# 测试用例说明

本目录包含马尔代夫选岛助手的测试用例，用于验证 skill 的核心功能。

## 测试覆盖场景

| 用例 | 场景 | 验证点 |
|------|------|--------|
| eval-1 | 蜜月情侣，需求明确 | 信息完整时直接搜索，报告包含 3 岛、商品卡片、预订链接 |
| eval-2 | 亲子家庭，需求明确 | 匹配亲子诉求，推荐适合儿童的岛屿 |
| eval-3 | 用户无明确意向 | 需求引导对话能力，输出热门岛屿速查矩阵 |

## 通过标准

- ✅ 步骤 1：正确提取或引导收集 4 维度信息（旅行类型、诉求、交通、预算）
- ✅ 步骤 2：并行执行 3 轮搜索，fetch 1-2 个关键链接
- ✅ 步骤 3：成功查询 flyai 商品（如可用），返回真实价格和链接
- ✅ 步骤 4：报告包含 3 个岛屿，每岛有 3 条推荐理由、2 条真实反馈、商品卡片
- ✅ 报告结构完整：需求画像、推荐岛屿、横向对比、选岛建议、注意事项
