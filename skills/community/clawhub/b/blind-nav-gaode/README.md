# 盲人出行导航助手 (blind-nav-gaode)

基于高德地图开放平台的 AI 盲人无障碍出行 Skill。

## 功能特性

- 📍 精准定位与逆地理编码
- 🗺️ 步行/公交路线规划（优先无障碍）
- 🔍 周边设施语音搜索
- ♿ 无障碍设施专项查询
- 🚨 一键紧急求助定位

## 配置

设置环境变量：

```bash
export GAODE_API_KEY="你的高德API密钥"
```

## 使用示例

```
用户：帮我导航到最近的医院
用户：附近有没有无障碍厕所
用户：我迷路了（触发SOS）
```

## 权限说明

- `location`：获取用户位置
- `network`：调用高德 API
