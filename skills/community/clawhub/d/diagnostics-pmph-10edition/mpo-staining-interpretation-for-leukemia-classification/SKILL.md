# MPO染色判读用于急性白血病分型

## 适用对象
- 骨髓原始细胞
- 急性白血病细胞

## 前提条件
- 已制备骨髓涂片并完成MPO细胞化学染色

## 判读步骤

1. **识别正常细胞的MPO反应模式**：
   - 中性粒细胞、嗜酸性粒细胞：阳性至强阳性（颗粒大而密集）
   - 单核细胞：弱阳性（颗粒细小、稀疏）
   - 淋巴细胞、幼红细胞、浆细胞：阴性

2. **评估白血病细胞的染色结果**：
   - 若细胞呈**阳性或强阳性**，提示急性粒细胞白血病（AML-M1/M2）
   - 若细胞呈**弱阳性或阴性**，结合形态考虑急性单核细胞白血病（AML-M5）
   - 若同时存在**阳性与弱阳性/阴性细胞群**，提示急性粒-单核细胞白血病（AML-M4）
   - 若所有原始细胞均**阴性**，支持急性淋巴细胞白血病（ALL）诊断

3. **判定阳性标准**：
   - **阳性**：胞质内出现紫红色或棕黑色沉淀颗粒
   - **强阳性**：颗粒粗大、密集、遍布胞质
   - **弱阳性**：少量细小颗粒，分布稀疏

4. **注意事项**：
   - 极早期分化阻滞的AML可能呈假阴性，需结合AS-D NCE、α-NAE等其他细胞化学染色综合判断
   - 本方法不适用于非造血细胞或完全分化的成熟淋巴细胞

## 输出
生成基于MPO染色强度的白血病亚型提示，指导进一步免疫表型或分子学分型。
