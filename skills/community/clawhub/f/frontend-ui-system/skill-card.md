## Description: <br>
Diseña e implementa interfaces de alta calidad para apps, sitios web y dashboards con dirección visual clara, design system, jerarquía fuerte, componentes reutilizables, mobile-first, consistencia UI y revisión iterativa. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and product teams use this skill to guide frontend UI design and implementation for apps, websites, dashboards, landing pages, and visual redesigns. It helps agents choose a visual direction, define a mini design system, structure screens, build reusable components, and review responsive UI quality before delivery. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may activate on broad or vague UI polish requests where the user intended only a narrow code change. <br>
Mitigation: Use clear task boundaries for backend-only, debugging-only, or non-visual work, or choose a more specific skill for those tasks. <br>
Risk: Generated UI guidance can over-prioritize visual polish if the product context and primary user action are unclear. <br>
Mitigation: Confirm the product type, audience, main user action, and responsive requirements before implementation. <br>
