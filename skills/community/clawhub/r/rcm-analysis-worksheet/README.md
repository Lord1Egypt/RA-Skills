# rcm-analysis-worksheet

Guides reliability engineers and maintenance teams through a structured Reliability-Centered Maintenance (RCM) analysis — from function identification through maintenance task selection — producing a DRAFT worksheet ready for engineering review before any maintenance program change.

## Who It's For

- Reliability engineers at industrial facilities (refining, power generation, utilities, mining, aerospace MRO)
- Maintenance planners and CMMS administrators
- Asset integrity leads and plant engineers
- RCM facilitators conducting team-based analyses

## What It Produces

A DRAFT RCM analysis worksheet including:

- Function and functional failure table
- Failure Mode and Effects Criticality Analysis (FMECA) with RPN scoring and HIGH PRIORITY flags
- Maintenance Significant Item (MSI) classification
- Decision logic tree outputs with task type justification
- Maintenance task list (task, type, frequency, trade, CMMS action)
- Redesign flags and information gaps list
- Reliability Engineer and Maintenance Manager Review Block

## What It Does NOT Do

- Implement changes in any CMMS or maintenance management system
- Certify intervals for safety-critical equipment without licensed engineer review
- Replace a facilitated RCM team analysis for Tier 1 critical assets

## Methodology Reference

Aligned to SAE JA1011 (Evaluation Criteria for RCM Processes), Nowlan & Heap failure pattern categories, and MSG-3 task selection principles.
