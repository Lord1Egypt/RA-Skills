# No-Repo Submission Plan

## What Can Be Done Without A Repo

You can submit this skill as a zipped package anywhere the marketplace accepts direct upload, local CLI publishing, or creator-dashboard submission.

The package should contain:

- `SKILL.md`
- `README.md`
- `SECURITY.md`
- `LICENSE`
- `examples/service-business-audit-output.md`
- `submission-copy/marketplace-listings.md`

## What May Require An Account

Publishing usually requires one or more of:

- Marketplace login
- GitHub OAuth
- Creator account
- Stripe Connect, even for free listings on some marketplaces
- CLI authentication token
- Manual approval

These steps cannot be fully completed without the relevant account session.

## Recommended Order

1. ClawHub
2. Agensi
3. AgenticSkills
4. SkillDrop
5. AgentPowers
6. TunX
7. SKILLS.pub / SkillsMP-style directories

SkillsGate, FindSkills, and MCP App Store may depend on public indexing from skills.sh or GitHub. Without a repo, treat them as lower-confidence unless they support manual submission.

## Publishing Checklist

- Upload the zip package.
- Select marketing, CRO, landing page, paid ads, or local business categories.
- Use the canonical title: `Ad-Page Match Auditor`.
- Use the short description from `submission-copy/marketplace-listings.md`.
- Add `https://funnelfinch.com` as the primary link.
- Add tags from the listing copy.
- Include the security note: instruction-only, no scripts, no dependencies, no network calls.
- Save the published URL in a tracker.

## Tracker Fields

- Marketplace
- Submission URL
- Published URL
- Status
- Submitted date
- Approved date
- Backlink URL
- Backlink type
- Notes
- Indexed in Google
- Installs or downloads
