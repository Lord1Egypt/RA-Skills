# 四专家思维框架（投教工作流）

投教工作流融合四位专家的核心思想，确保教育内容既有深度又有温度。

---

## 四专家视角

| 专家 | 视角 | 核心能力 | 对应 Skill | 投教应用 |
|------|------|---------|-----------|---------|
| **林奇** | 洞察发掘 | 生活化触达/故事驱动/持续观察 | `ljg-learn`, `ljg-plain`, `ljg-rank` | 用生活场景解释金融概念（"买店铺看租金"类比 PE） |
| **卡尼曼** | 行为纠偏 | 认知偏误画像/负面叙事/选择架构 | `decision-checklist`, `ljg-relationship` | 识别过度自信/损失厌恶/羊群效应，建立行为防御 |
| **芒格** | 逆向检查 | 双轨分析/安全边际/失败预演 | `ljg-think`, `ljg-roundtable`, `mental-models` | "太难"清单、逆向思考、安全边际教育 |
| **马利克** | 系统管理 | 目标约束/多方案/反馈回路 | `fund-allocator`, `task-state-tracker` | 六大账户系统、定期检视、家庭财务宣言 |

---

## 毕业标准（四专家共识）

| 专家 | 毕业标准 |
|------|---------|
| 林奇 | 能在自己的"能力圈"里，独立找到 5 个愿意持有数年的投资故事 |
| 卡尼曼 | 能识别并抵御 90% 的金融欺诈；能执行与风险承受能力匹配的配置策略 |
| 芒格 | 能掌握 20 个跨学科思维模型；具备识别"太难"并坦然放弃的智慧 |
| 马利克 | 已建立并持续运行家庭财务管理系统（自动巡航状态） |

---

## 分层教育体系

```
┌─────────────────────────────────────────┐
│  顶层：林奇式主动投资探索营（选修）       │
│  - 发现生活投资机会                      │
│  - 五分钟看懂一家公司                     │
│  - 长持好故事，关掉市场先生               │
├─────────────────────────────────────────┤
│  中层：芒格式普世智慧与决策框架           │
│  - 逆向思维与"太难"清单                  │
│  - 多学科思维模型（10 个核心模型）        │
│  - 安全边际与下重注的勇气                 │
├─────────────────────────────────────────┤
│  底层：卡尼曼式防御与马利克式系统（必修） │
│  - 认识大脑的"财务骗子"                  │
│  - 读懂销售话术的解密钥匙                 │
│  - 定制"行为疫苗"（投资宪法）             │
│  - 构造"自动巡航"财务系统（六大账户）     │
└─────────────────────────────────────────┘
```

**核心原则**：先防御，后进攻。未经防御训练的人，不配拥有进攻武器。

---

## 投教内容转译规则

### 去术语化
- ❌ "市盈率=股价/每股收益"
- ✅ "PE 就是你买这家公司，靠利润需要多少年回本"

### 场景化
- ❌ "定投可以分散择时风险"
- ✅ "定投就像每月存钱进储蓄罐，不用猜什么时候存最划算"

### 行为化
- ❌ "投资者应注意风险"
- ✅ "单只股票不超过总资产的 20%，亏损超过 15% 必须离场"

### 可视化
- 对比表格、流程图、时间轴、情绪 - 收益对照表
