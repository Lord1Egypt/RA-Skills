# Profit Calculator — Quality Checklist

Validate every analysis against this checklist before presenting. Each item should be checked or explicitly marked N/A with reasoning.

---

## COGS & Landed Cost (8 items)

- [ ] Landed cost includes factory cost, not just purchase price
- [ ] Inbound freight cost calculated per unit (total freight ÷ units in shipment)
- [ ] Duty/tariff rate verified against current HTS code schedule
- [ ] Packaging cost per unit included (branded box, poly bag, inserts, labels)
- [ ] Any prep or kitting costs included (FBA prep, bundle assembly)
- [ ] Inspection or quality control fees amortized per unit
- [ ] COGS updated to reflect most recent purchase order pricing
- [ ] Currency conversion applied if sourcing internationally (with date of rate noted)

## Platform Fees (9 items)

- [ ] Every fee type mapped per channel (not a single blended percentage)
- [ ] Referral/commission fees verified against current platform schedule
- [ ] Payment processing fees captured (percentage + fixed per-transaction)
- [ ] Fulfillment fees matched to correct product size tier and weight
- [ ] Storage fees included with seasonal variation noted (Q4 vs. standard)
- [ ] Subscription costs amortized per unit based on current monthly volume
- [ ] App and software costs amortized per unit (for DTC channels)
- [ ] Affiliate commissions captured separately from platform commission (TikTok Shop)
- [ ] Fee schedule verified against platform's current published rates

## Shipping & Fulfillment (7 items)

- [ ] Pick-and-pack cost sourced from actual 3PL rate card or internal labor calculation
- [ ] Outbound shipping cost based on actual weight and average zone mix
- [ ] Packaging materials cost included (box, dunnage, tape, shipping label)
- [ ] Multi-item order adjustment applied (total fulfillment ÷ avg units per order)
- [ ] FBA fulfillment correctly treated as inclusive of shipping (no double-counting)
- [ ] Free shipping cost captured as merchant expense, not ignored
- [ ] Shipping cost differences between channels documented (DTC vs. marketplace)

## Ad Spend Allocation (7 items)

- [ ] Allocation method explicitly stated (blended MER vs. channel ROAS vs. SKU ROAS)
- [ ] Ad spend and revenue figures from the same time period
- [ ] Attribution window documented per channel
- [ ] If using blended MER, acknowledged that it treats all SKUs equally
- [ ] If using channel ROAS, each channel has its own ad cost calculation
- [ ] Ad spend includes all paid media (Meta, Google, TikTok, Amazon Ads, affiliates)
- [ ] Organic revenue separated from paid-attributed revenue if possible

## Return Cost Modeling (8 items)

- [ ] Return rate sourced per channel (not a single blended rate)
- [ ] Reverse logistics cost included (return shipping label or restocking fee)
- [ ] Product disposition rates estimated (full-price resellable, liquidation, write-off)
- [ ] Product loss dollar amount calculated from disposition rates and COGS
- [ ] Restocking labor cost included (inspect, repackage, relabel)
- [ ] Original outbound shipping cost noted as unrecoverable
- [ ] Return cost calculated per unit SOLD (rate × cost per return), not per return
- [ ] Category-specific return rates used (apparel vs. beauty vs. electronics)

## Overhead Allocation (5 items)

- [ ] All fixed cost categories listed (rent, salaries, software, insurance, professional services, depreciation)
- [ ] Allocation method stated (revenue-weighted, unit-weighted, or activity-based)
- [ ] Overhead allocated per channel based on chosen method
- [ ] Per-unit overhead calculated using current monthly volume
- [ ] Overhead shown as a separate waterfall line (not mixed into other costs)

## Profit Waterfall (6 items)

- [ ] Waterfall follows standard 7-line structure (price → fees → COGS → shipping → ads → returns → overhead)
- [ ] All dollar amounts are per-unit, not monthly totals
- [ ] Each line item clearly shows subtraction (running total visible at each step)
- [ ] Net contribution margin calculated both as dollars and as percentage of retail price
- [ ] Waterfall built for each SKU on each channel (not just blended)
- [ ] Decision flag assigned based on margin range (scale / maintain / optimize / reprice / kill)

## Decision & Recommendations (5 items)

- [ ] Top and bottom SKUs ranked by net contribution margin
- [ ] Channel comparison for same SKU presented side-by-side
- [ ] Sensitivity analysis included for at least price and COGS variables
- [ ] Break-even volume or break-even price calculated
- [ ] Specific action items listed with expected margin impact quantified
