## Description: <br>
Test design and strategy skill for designing test cases, planning testing strategy, writing unit, integration, and end-to-end tests, and improving test coverage. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[sydpz](https://clawhub.ai/user/sydpz) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to design maintainable test suites, choose unit, integration, and end-to-end coverage, structure test cases, and select mocking and test data strategies. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Example commands may start services or run broad test suites in a real project. <br>
Mitigation: Review and adapt generated tests and commands before execution, especially commands that start dependencies or run broad suites. <br>
