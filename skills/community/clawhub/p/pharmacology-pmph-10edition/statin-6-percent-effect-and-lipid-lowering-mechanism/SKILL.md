# 他汀类药物的“6%效应”及调脂机制

## 何时使用
- 患者需进一步降低LDL-C但当前他汀单药治疗已达常规剂量上限；
- 评估是否因“6%效应”导致剂量加倍收益有限，需转向联合治疗；
- 判断患者是否适合继续使用他汀类药物（需满足肝肾功能基本正常、无妊娠/哺乳、无活动性肝病等前提）。

## 执行步骤
1. **确认适用对象与前提条件**：
   - 确保患者属于高胆固醇血症、ASCVD高危或家族性高脂蛋白血症人群；
   - 核实无禁忌证（如失代偿期肝硬化、急性肝衰竭）；
   - 确认转氨酶 ≤ 3倍正常上限。

2. **理解核心调脂机制**：
   - 他汀竞争性抑制HMG-CoA还原酶，阻断内源性胆固醇合成；
   - 肝细胞胆固醇减少→上调LDL受体（LDLR）表达→加速血浆LDL清除→显著降低LDL-C；
   - 同时轻度降低TG、略升高HDL-C。

3. **应用“6%效应”原则**：
   - 若当前他汀剂量已较高，剂量加倍仅额外降低LDL-C约6%，提示疗效接近平台；
   - 此时应优先考虑联合不同机制药物（如依折麦布、PCSK9抑制剂），而非继续增加他汀剂量。

4. **评估多效性作用（必要时）**：
   - 包括改善内皮功能、抗炎、稳定斑块、抗血栓等，可作为长期心血管保护的附加依据。

5. **启动安全性监控**：
   - 用药前及用药后定期检测ALT、AST、CK；
   - 警惕肌痛、肌炎、横纹肌溶解及新发糖尿病风险。

6. **决策输出**：
   - 若“6%效应”显著限制单药疗效，推荐联合治疗方案；
   - 若存在禁忌或不良反应，考虑换用非他汀类降脂药。
