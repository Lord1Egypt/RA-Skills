## Description: <br>
Implement fuzzy search in JavaScript/TypeScript projects using Fuse.js for client-side search, fuzzy matching, highlighting, weighted multi-field search, tokenized search, and Web Worker parallel search. <br>

This skill is ready for commercial/non-commercial use. <br>

## Known Risks and Mitigations: <br>
Risk: Sensitive datasets may be exposed if client-side search indexes include data that should not be available in the browser. <br>
Mitigation: Index only data intended for client-side access, and keep private or restricted records on the server. <br>
Risk: Using the beta FuseWorker package can introduce compatibility or stability risk when stable Fuse.js features are sufficient. <br>
Mitigation: Prefer the stable fuse.js package unless Web Worker parallel search is specifically required and tested. <br>
