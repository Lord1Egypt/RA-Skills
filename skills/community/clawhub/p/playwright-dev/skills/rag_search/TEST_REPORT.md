## 🐛 已知问题

### 问题 1：相对路径问题

**现象：** 从不同目录运行脚本时，Chroma 路径解析错误

**原因：** .env 文件使用相对路径

**解决方案：** ✅ 已修复
- 在 search.py 中使用硬编码绝对路径
- 建议：.env 文件也使用绝对路径

### 问题 2：API 批量限制

**现象：** 阿里云 Embedding API 限制 batch size <= 10

**解决方案：** ✅ 已修复
- indexer.py 中分批调用（10 个/批）
- 添加错误处理和重试逻辑

### 问题 3：编码问题

**现象：** PowerShell 输出中文乱码

**解决方案：** ✅ 已修复
- Python 脚本中使用 UTF-8 编码
- 避免在 f-string 中使用反斜杠


### 示例 2：自动触发

```python
from skills.rag_search.src.auto_trigger import auto_search

# 用户消息
user_message = "我之前说过瀑布的事情吗？"

# 自动搜索
result = auto_search(user_message, k=3)

if result:
    print(f"虾虾找到了 {len(result['results'])} 条相关记忆～")
    print(result['results'])
```
