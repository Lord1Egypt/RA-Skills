## Description: <br>
Discord Toolkit is a command-line reference skill that prints concise Discord devtools guidance for intro, quickstart, patterns, debugging, performance, security, migration, and cheatsheet topics. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
Developers and engineers use this skill to retrieve static Discord Toolkit reference guidance while working through devtools workflows, troubleshooting, performance, security, and migration topics. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Static reference guidance can become incomplete or outdated for a specific Discord implementation or operational environment. <br>
Mitigation: Review the generated guidance against current project requirements and authoritative Discord documentation before applying it. <br>
Risk: Security and migration guidance is generic and may not cover organization-specific controls or rollback requirements. <br>
Mitigation: Have a qualified reviewer validate security, credential, backup, and rollback steps before using the guidance in production workflows. <br>
