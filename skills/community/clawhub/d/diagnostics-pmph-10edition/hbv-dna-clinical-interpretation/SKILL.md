# HBV-DNA临床解读

## 适用场景
- 乙型肝炎患者确诊后评估病毒复制与传染性
- 接受抗病毒治疗者监测疗效与耐药风险
- HBsAg阳性母亲所生新生儿完成免疫后评估母婴垂直传播阻断效果

## 执行步骤

1. **确认HBV-DNA阳性结果**：
   - 表明乙型肝炎病毒正在体内复制
   - 患者具有传染性
   - 可作为乙型肝炎的确诊依据之一

2. **抗病毒治疗监测**：
   - 定期检测HBV-DNA水平
   - 将HBV-DNA下降或转阴作为核心疗效指标

3. **耐药性初步判断**：
   - 若治疗中HBV-DNA未持续下降或出现反弹，应怀疑耐药
   - 建议结合YMDD变异等分子检测进一步确认

4. **疫苗阻断效果评估**：
   - 针对HBsAg阳性母亲所生婴儿，在完成标准免疫程序后
   - 若婴儿HBV-DNA仍为阳性，提示垂直传播阻断失败

5. **综合评估注意事项**：
   - HBV-DNA定量值反映病毒载量，与传染性强弱正相关
   - 不直接反映肝损伤程度，必须结合ALT、肝功能及其他乙肝标志物（如HBsAg、HBeAg）综合判断

> **重要限制**：本技能不能单独用于区分急性与慢性乙肝感染，必须与其他临床和实验室信息联合使用。
