# 液体活检临床应用

## 何时使用
当满足以下任一条件时启用本技能：
- 需对难以获取组织的肿瘤（如肺、胰腺）进行早期筛查；
- 需在治疗过程中动态监测疗效或耐药情况；
- 需评估患者预后（如通过CTC计数）；
- 需检测EGFR、KRAS等驱动基因突变以指导靶向治疗。

## 执行步骤
1. **确认适用性**：
   - 确保患者存在实体瘤或疑似肿瘤；
   - 确认具备PCR、NGS等分子检测平台；
   - 排除惰性肿瘤（无CTC/ctDNA释放）或极早期微量释放高风险假阴性情况。

2. **选择样本类型**：
   - 优先选择外周血；
   - 根据临床情况可选用胸腔积液、腹水、尿液或脑脊液。

3. **确定检测目标**：
   - **CTCs**：适用于预后评估和转移风险判断；
   - **ctDNA**：适用于突变检测、疗效动态监测；
   - **ctRNA（主要为miRNA）**：适用于特定调控通路分析。

4. **执行检测与解读**：
   - 根据检测目的（早期诊断/疗效监测/预后评估/个体化治疗）选择对应分析流程；
   - 结合影像学与临床表现综合判断结果，避免单独依赖液体活检。

5. **输出行动建议**：
   - 若ctDNA水平显著下降，维持当前治疗；
   - 若ctDNA升高或检出新突变，考虑疾病进展或耐药，安排影像复查或调整治疗方案；
   - 若检出可靶向突变（如EGFR L858R），启动相应靶向治疗。

> 注意：液体活检不能完全替代组织活检，尤其在需要组织学分型或免疫组化时。
