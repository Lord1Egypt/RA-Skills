## 📖 使用方式

### 保存内容
```
直接发送链接：
https://www.bilibili.com/video/BV1xx411c7mD

或指定分类：
保存到工作：https://zhuanlan.zhihu.com/p/xxx
```

### 查询内容
```
/kb search Python          # 搜索
/kb list 学习              # 列出分类
/kb view kb-123            # 查看详情
/kb stats                  # 统计信息
/kb recent                 # 最近添加
```

### 管理命令
```
/kb delete kb-123          # 删除
/kb export                 # 导出
/kb help                   # 帮助
```
