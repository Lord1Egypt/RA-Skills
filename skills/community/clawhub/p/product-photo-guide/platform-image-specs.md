# Platform Image Technical Specifications

## Amazon

### Main Image (MAIN)
- **Background:** Pure white (#FFFFFF only) — no exceptions
- **Minimum resolution:** 1000px on longest side (2000px+ recommended for zoom)
- **Maximum file size:** 10MB
- **File types:** JPEG, PNG, GIF, TIFF
- **Product fill:** 85%+ of frame
- **Restrictions:** No text, logos (except on product itself), watermarks, graphics, mannequin
- **Restrictions (apparel):** No hangers; mannequin allowed if ghost-mannequin effect applied

### Additional Images (PT01–PT08)
- Same resolution and file size requirements
- Text overlays allowed
- Lifestyle shots allowed
- Maximum 9 images total (main + 8 additional)
- A+ Content adds additional image gallery slots

### Amazon A+ Content
- Module-specific dimensions — varies by module type
- Hero image modules: 970×600px or 1464×600px
- Standard comparison chart module: 1500×600px
- JPEGs under 2MB per image for fastest loading

## Shopify

### Product Images
- **Recommended resolution:** 2048×2048px (Shopify uses this for zoom)
- **Minimum resolution:** 800×800px
- **Maximum file size:** 20MB
- **File types:** JPEG, PNG, GIF, WebP
- **Aspect ratio:** 1:1 (square) recommended; Shopify themes can handle 3:4 portrait
- **Background:** No restriction — matches brand aesthetic

### Collections / Category Images
- **Recommended:** 2048×2048px or wider (3000px width for banners)
- **Aspect ratio varies by theme** — check your specific theme documentation

### Technical Notes
- Shopify auto-compresses on upload — original should be highest quality available
- WebP format provides smallest file size at equivalent quality; Shopify converts automatically
- ALT text on all product images required for SEO and accessibility

## TikTok Shop

### Product Images
- **Minimum resolution:** 500×500px (800×800px+ recommended)
- **Maximum file size:** 5MB per image
- **File types:** JPEG, PNG
- **Maximum images:** 9 per listing
- **First image:** White background or visually clean background preferred by algorithm
- **Recommended:** Multiple lifestyle images over studio-only sets

### Cover Image (Main Thumbnail)
- **Displays at:** Approximately 200×200px in feed; 600×600px on product page
- **Test at:** 200px width — must be instantly readable and visually striking
- **High contrast recommended:** Product must stand out at small sizes

### Video Requirements
- **Format:** MP4, MOV
- **Aspect ratio:** 9:16 (vertical) strongly preferred; 1:1 acceptable
- **Minimum resolution:** 720×1280px
- **Duration:** 10–60 seconds (15–30 seconds performs best for product listings)
- **File size:** Max 500MB
- **Note:** First 3 seconds determine watch-through rate — hook must be immediate

## Shopee

### Product Images
- **Minimum resolution:** 500×500px
- **Recommended:** 1024×1024px+
- **Maximum file size:** 2MB per image
- **File types:** JPEG, PNG
- **Background:** White background required for first image in most categories
- **Maximum images:** 9 per listing
- **Text overlays:** Widely used and accepted on Shopee (unlike Amazon)
- **Infographic style:** Heavy use of text callouts and color blocks performs well in SEA market

### Video
- **Format:** MP4
- **Maximum duration:** 60 seconds
- **Maximum size:** 30MB
- **Aspect ratio:** 1:1 preferred

## Lazada

### Product Images
- **Minimum resolution:** 500×500px
- **Recommended:** 2000×2000px
- **First image:** White background required
- **File types:** JPEG, PNG
- **Maximum images:** 8 per listing
- **Text overlays:** Permitted on supplementary images

## Instagram Shopping

### Product Images
- **Format:** JPEG, PNG
- **Minimum resolution:** 500×500px
- **Recommended:** 1080×1080px (square) or 1080×1350px (portrait)
- **Max file size:** 8MB
- **Note:** Images should work as standalone social content — both product-focused and lifestyle

## General Image Optimization Notes

### Resolution Planning
- Always shoot at maximum camera resolution
- Edit at full resolution; export compressed for web
- Keep master files in RAW or TIFF for future re-export to different specs

### Color Space
- Export all product images in **sRGB** color space
- Amazon and Shopify both render sRGB correctly
- Adobe RGB images can appear desaturated on some displays/platforms

### File Naming Convention
- Include SKU, variant, and shot number: `SKU-001_BLACK_HERO.jpg`
- Consistent naming enables automated uploads and reduces errors across platform accounts

### Compression Settings
- JPEG quality: 85–90% for web upload (imperceptible quality loss, significant size reduction)
- WebP (Shopify): Shopify auto-converts; no action needed
- PNG: Use only for images requiring transparency (product cutouts)
