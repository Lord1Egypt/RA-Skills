# 服务商模式 - 接入质量检查清单

> 服务商模式下与商户模式不同的检查项

## 服务商特有检查项

| 检查项 | 说明 |
|--------|------|
| 子商户进件状态 | 子商户必须审核通过 |
| 子商户绑定 | 子商户AppID和mchid必须已绑定 |
| 服务商权限 | 服务商需申请相关接口权限 |
| 分账配置 | 如需分账，需配置分账规则 |

## 子商户进件检查

| 检查项 | 说明 |
|--------|------|
| 进件状态 | SUBMITTED → AUDITING → FINISHED |
| 资质审核 | 营业执照、法人身份证等 |
| 账户验证 | 对公账户打款验证 |
| 费率确认 | 确认商户服务费率 |

## 子商户绑定检查

| 检查项 | 说明 |
|--------|------|
| AppID绑定 | 子商户AppID必须在商户平台绑定 |
| 授权函 | 部分场景需要补充授权函 |
| 权限配置 | 服务商后台配置子商户权限 |

## 分账检查（如使用）

| 检查项 | 说明 |
|--------|------|
| 分账比例 | 服务商与子商户分账比例配置 |
| 分账接收方 | 子商户必须已添加为分账接收方 |
| 分账时机 | 支付成功后发起分账 |
| 资金冻结 | 不分账时需解冻 |

## 合单支付检查（服务商）

| 检查项 | 说明 |
|--------|------|
| 子单数量 | 最多50个（普通商户最多10个） |
| 子单商户绑定 | 所有子商户需绑定同一AppID |
| 子单金额 | 每笔子单金额不超过1000万 |
| 关单限制 | 只能关闭整个合单，不能关闭单个子单 |

## 服务商模式上线检查清单

- [ ] 子商户进件审核通过
- [ ] 子商户AppID与mchid绑定完成
- [ ] 服务商权限申请完成
- [ ] 回调地址配置正确
- [ ] 分账规则配置（如需分账）
- [ ] 测试环境验证通过
- [ ] 子商户测试支付验证
