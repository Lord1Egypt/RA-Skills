## 按舞弊手法索引

- **过早确认收入**：微策略、冠群电脑、绿山咖啡
- **虚假收入**：汉能薄膜、新泰辉煌、康美药业、保千里
- **费用资本化**：世通、美国在线、乐视网
- **现金流操纵**：安然、康得新
- **不计提减值**：威帝斯半导体、英特尔
- **存货/生物资产舞弊**：辉山乳业、万福生科
- **关联交易操纵**：乐视网、汉能薄膜、神雾环保
