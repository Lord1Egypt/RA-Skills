

### 名称
`industry_development_barometer`

### 定位（用户可见）
基于股权、高管任职及业务关联，生成多维企业关系网；同时自动将本地企业精准“挂载”到产业链的对应环节，帮助快速看清区域家底。并融合政策、市场、技术三大维度的历史演进分析，输出产业趋势预判。

### 描述（LLM识别）
1. 用户输入特定产业节点（如“低空飞控系统”、“固态电池”等）；
2. LLM检索并汇总过去 **5年** 该节点在用户所属区域及关注产业中的政策导向、市场规模、技术迭代上的演进时间线、特征事件及趋势预判；
3. 可选功能：基于企业库（本地企业名单），自动将企业匹配到产业链各环节，生成“区域产业链挂载图谱”。

### 触发条件
当用户请求满足以下任一条件时，激活此技能：
- 询问特定产业节点的历史发展过程或演变阶段；
- 要求分析某产业环节的市场规模或技术演进趋势；
- 需要了解某产业相关的政策变迁或关键历史事件；
- 要求对某特定产业环节的未来发展方向进行预测；
- 要求将本地企业按产业链环节进行分类、挂载。

### 输入参数
| 参数名                 | 类型     | 必填  | 说明                                        |
| ------------------- | ------ | --- | ----------------------------------------- |
| `industry_node`     | string | 是   | 产业节点名称（如“eVTOL飞控系统”、“氢燃料电池电堆”）            |
| `region`            | string | 是   | 用户所属区域（如“江门市蓬江区”），用于筛选本地政策与本地企业           |
| `focus_industry`    | string | 否   | 关注的产业领域（如“低空经济”），默认与`industry_node`所在产业一致 |
| `local_enterprises` | array  | 否   | 本地企业列表（名称+主营业务），用于生成企业挂载图谱；若不提供则仅输出趋势分析   |
| `years_back`        | int    | 否   | 回溯年数，默认5                                  |

### 输出结构
生成一份标准化报告，包含以下模块（根据输入参数可选）：

#### 模块一：产业节点基础画像
- 节点名称、所属产业链环节（上游/中游/下游）、关键产品/技术。

#### 模块二：近5年核心演进时间轴（Y年至今）
- 以表格或时间线形式展示每年最具标志性的 **1-2个事件**（政策发布、市场规模里程碑、技术突破）。

#### 模块三：多维度特征事件盘点
- **政策导向维度**：列出近5年国家/省/市/区级与该节点直接相关的政策文件、补贴计划、试点项目等。
- **市场规模变迁**：引用可获取的产值、出货量、企业数量等数据（如无公开数据，标注“无权威数据”），描述增长趋势或波动。
- **核心技术迭代**：标注关键技术的演进路径（如从单余度→三余度飞控，从实验室→量产），以及标志性突破年份。

#### 模块四：未来趋势预判
- 基于历史归纳，输出3-5条趋势判断（如技术方向、政策重点、竞争格局变化）。

#### 模块五（可选）：本地企业产业链挂载图谱
- 仅当输入了 `local_enterprises` 时生成。
- 将每家企业匹配到产业节点的上下游环节（如飞控系统：上游芯片/传感器 → 中游飞控模块 → 下游整机集成）。
- 输出形式：按环节分组的企业列表，标注每家企业的核心产品或角色。
- 同时指出“空白环节”（即区域内缺失的产业链环节）。

### 执行步骤（LLM工作流）

#### 步骤1：解析输入与知识检索
- 提取 `industry_node`、`region`、`focus_industry`。
- 内部检索（基于LLM自身知识库 + 可调用的外部知识API，若有）：
  - 政策：国务院、发改委、工信部、地方政府的公开文件。
  - 市场：行业白皮书、券商研报、联盟统计（尽量引用年份和数据来源）。
  - 技术：行业综述、龙头企业的技术发布会信息。

#### 步骤2：构建时间轴与特征事件
- 按年份组织（当前年份往前推5年，如2022-2026年）。
- 每年筛选1-2件对产业节点影响最大的事件。
- 区分三大维度，但时间轴中可以混合展示。

#### 步骤3：趋势预判
- 基于技术成熟度曲线、政策连续性、资本流向等进行推理。
- 输出具体、可验证的预判（例如：“预计2027年xx技术将进入适航取证高峰”）。

#### 步骤4：企业挂载（可选）
- 若 `local_enterprises` 非空，则：
  - 根据每家企业的主营业务、产品描述，匹配到产业节点的某一环节。
  - 预先定义该产业节点的典型环节划分（若无标准划分，按通用上中下游划分）。
  - 输出挂载结果并标注空白环节。

### 使用示例

#### 示例1：仅趋势分析
```yaml
输入:
  industry_node: "eVTOL飞控系统"
  region: "深圳市南山区"
  focus_industry: "低空经济"

输出: （略，见格式）
```

#### 示例2：趋势+企业挂载
```yaml
输入:
  industry_node: "无人机飞控系统"
  region: "江门市蓬江区"
  focus_industry: "低空经济"
  local_enterprises:
    - 企业名称: "擎动千机"
      主营业务: "工业无人机整机制造"
    - 企业名称: "保卫鹰"
      主营业务: "无人机巡检服务"
    - 企业名称: "边界智控（深圳）"
      主营业务: "飞控系统研发与生产"

输出: 包含挂载图谱，并将“边界智控”标记为外部企业（非本地），提示可招引。
```

### 注意事项
1. **数据真实性**：所有政策、市场数据应注明来源（如“根据工信部《2025年低空经济发展报告》”）。若无确切数据，写明“该环节未公开市场规模”。
2. **区域敏感性**：政策部分优先提取 `region` 对应的省市区级政策，再补充国家层面。
3. **企业挂载粒度**：若企业业务横跨多个环节，选择其主要环节挂载，并在备注中说明。
4. **时效性**：回溯期默认5年，但若产业节点非常新兴（如2年内才出现），则回溯自产业形成之初，并在报告中说明。

### 调用方式（openclaw）
```
/industry_development_barometer industry_node="固态电池" region="宜宾市" focus_industry="新能源"
```

或附带企业列表：
```
/industry_development_barometer industry_node="飞控系统" region="江门市蓬江区" local_enterprises=[{"name":"擎动千机","business":"..."}]
```
