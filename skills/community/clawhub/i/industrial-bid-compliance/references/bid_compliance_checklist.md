# 逐条合规检查清单模板

## 使用说明

本模板用于审查工业技术方案时的逐条对标检查。以下为通用模板，可根据具体项目类型添加专项清单。

## 按项目类型的专项清单

> 在进行具体项目审查时，从下方选取对应类型的专项检查项追加到通用清单中。

### EQUIP — 设备采购专项
- [ ] 设备选型是否满足产能要求
- [ ] 设备材质是否满足介质/环境要求
- [ ] 设备接口/安装尺寸是否与现场匹配
- [ ] 备品备件清单是否完整
- [ ] 设备寿命周期成本分析
- [ ] 进口设备是否有国产替代方案

### ENG — 工程施工专项
- [ ] 施工总平面布置是否合理
- [ ] 关键施工工艺是否可行
- [ ] 雨季/冬季施工措施是否完备
- [ ] 工期计划是否合理可行
- [ ] 交叉作业协调机制是否明确
- [ ] 临时用电/用水方案是否合规
- [ ] 材料进场检验计划

### SVC — 技术服务专项
- [ ] 服务响应时间是否满足SLA要求
- [ ] 服务人员技术能力是否达标
- [ ] 服务工具/软件是否合规
- [ ] 数据安全保护措施是否到位
- [ ] 服务级别协议(SLA)是否明确
- [ ] 灾备与业务连续性方案
