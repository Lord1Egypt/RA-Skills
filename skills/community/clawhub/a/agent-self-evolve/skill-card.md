## Description: <br>
Agent Self-Evolve helps OpenClaw agents maintain learning records, queue skill and code improvements, and run daily or weekly review cycles through memory files, prompts, and setup commands. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent operators use this skill to add continuous learning workflows to OpenClaw agents, including mistake tracking, experience consolidation, improvement queues, cron prompts, and metrics records. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Scheduled self-evolution can modify code, skills, memory, and configuration without clear per-change approval. <br>
Mitigation: Require human review before applying code, skill, AGENTS.md, SOUL.md, or configuration changes generated by cron or heartbeat workflows. <br>
Risk: Persistent learning records can capture operational history, mistakes, preferences, or other sensitive context. <br>
Mitigation: Limit writable paths to an explicit allowlist and review retained memory files before sharing or deploying the workspace. <br>
