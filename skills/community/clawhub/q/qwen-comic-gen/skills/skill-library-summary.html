🔥 核心技能（5 个）
1. doubao-expert-review - 豆包会话专家点评 核心
自动处理豆包会话内容，生成专家级点评 HTML 网页，包含知识架构图、深度洞察和行动建议。
🎯 触发 Prompt
  • 豆包 [粘贴豆包会话内容...]
  • 分析这个豆包会话
  • 生成豆包会话专家点评
2. html-expert-review - HTML 专家点评生成 核心
从豆包会话生成专家级点评 HTML 网页，包含知识架构图、深度洞察、对比分析、行动建议，采用科学杂志排版 + 小红书视觉风格。
🎯 触发 Prompt
  • 生成专家点评
  • 创建 HTML 报告
  • 生成专家级点评 HTML
3. weekly-report-generator - 周报自动生成 核心
从 worklog.txt 自动生成正式周报，包含本周完成、进行中、待办事项，支持 Markdown 和 HTML 格式。
🎯 触发 Prompt
  • 生成周报
  • 生成本周周报
  • 从 worklog 生成周报
  • 周三了，生成周报
4. cron-task-manager - Cron 定时任务管理 核心
创建、管理、删除 Cron 定时任务，支持每小时/每天/每周定时执行，用于周报生成、优先级提醒等周期性任务。
🎯 触发 Prompt
  • 创建 Cron 任务
  • 设置定时任务
  • 每周三 15:00 生成周报
  • 每小时提醒我
  • 查看 Cron 任务列表
5. feishu-document-generation - 飞书文档生成 核心
创建、更新、删除飞书文档，支持原子动作文档同步，使用飞书 API 进行文档操作。
🎯 触发 Prompt
  • 创建飞书文档
  • 更新飞书文档
  • 同步到飞书文档
  • 在飞书创建文档
