# Introduction to Cyberbrick

CyberBrick is a modular smart toy ecosystem created by Bambu Lab. It integrates three core components:

1.  **Programmable Hardware**: A set of wireless microcontrollers and solderless electronic modules (motors, sensors, etc.).
2.  **3D-Printable Structures**: An open framework that allows users to design and 3D print their own mechanical parts and enclosures. Bambu Lab provides reference designs and full specifications.
3.  **Dual-Level Coding Environments**: It supports both a simple, app-based configuration for beginners and a more advanced MicroPython environment for complex programming.

The goal of Cyberbrick is to empower creators, students, and makers to build a wide range of custom projects, from simple R/C toys to more complex productivity tools, by combining the worlds of 3D printing, electronics, and coding in a user-friendly, LEGO-like approach.

## Key Resources

The ecosystem is supported by a rich set of resources to help creators get started and share their work:

-   **MakerWorld**: The central platform for finding and sharing Cyberbrick models. Each model includes a Bill of Materials (BOM) and assembly instructions.
-   **Maker's Supply**: The official online store for purchasing Cyberbrick hardware components.
-   **Official Wiki**: The primary source for all guides and generic information.
-   **GitHub Repository**: Contains open-source parts of the codebase for programming reference. ([https://github.com/CyberBrick-Official](https://github.com/CyberBrick-Official))
-   **API Documentation**: Provides detailed information on all custom functions available in the MicroPython environment. ([https://makerworld.com/en/cyberbrick/api-doc](https://makerworld.com/en/cyberbrick/api-doc))
-   **CAD Models**: Reference CAD files for all hardware modules are available to facilitate the design of custom parts. These can be found on the individual product pages in the Maker's Supply store.
