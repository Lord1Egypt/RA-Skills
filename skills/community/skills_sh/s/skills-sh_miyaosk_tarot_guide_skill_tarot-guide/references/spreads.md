# 牌阵定义 Spreads

---

## 一、单牌占卜 Single Card（1 张）

### 模式 1：每日指引（daily）- 默认

| 位置 | 含义 |
|------|------|
| 今日指引 | 此刻最需要关注的心理主题或能量 |

**适用场景**：每日抽牌、快速提问、任何简单直接的问题。

**解读要点**：聚焦于"此刻"，将牌面视为潜意识发出的一条核心信息。不需要宏大叙事，重在精准捕捉当下的心理状态与需要。

### 模式 2：是否判断（yesno）

| 位置 | 含义 |
|------|------|
| 是或否 | 对封闭式问题的直觉回应 |

**适用场景**：用户问了一个"是不是""要不要""该不该"的封闭式问题。

**解读要点**：正位倾向"是/顺利"，逆位倾向"否/需谨慎"。但塔罗不做绝对判断——在给出倾向后，需补充"为什么是这个倾向"的心理分析，避免简单的二元回答。

## 二、三牌占卜 Three-Card Spread（3 张）

### 模式 1：时间线（time）- 默认

| 位置 | 含义 |
|------|------|
| 过去 | 影响当前处境的根源、过去的模式或经验 |
| 现在 | 当下的核心状态、正在经历的心理主题 |
| 未来 | 如果沿当前轨迹发展，可能呈现的趋势与潜力 |

**适用场景**：事业运势、感情走向、任何"从过去到未来"的问题。

**解读要点**：三张牌构成一条叙事弧线。重点在于"过去如何塑造了现在，现在的选择如何影响未来的走向"。未来不是命定，而是当前能量的自然延伸。

### 模式 2：问题导向（problem）

| 位置 | 含义 |
|------|------|
| 处境 | 当前所处的客观情境，问题的核心面貌 |
| 挑战 | 需要面对的困难、阻碍或内在障碍 |
| 建议 | 潜意识提供的应对方向与智慧 |

**适用场景**：遇到困难、感到迷茫、需要具体建议的问题。

**解读要点**：从"是什么"到"难在哪"再到"怎么办"的递进结构。挑战位置尤其重要——它往往揭示了问题的心理根源而非表面现象。

### 模式 3：心理探索（psyche）

| 位置 | 含义 |
|------|------|
| 意识 | 你已经意识到的、头脑中明确知道的部分 |
| 潜意识 | 你尚未觉察但在深层影响你的力量 |
| 指引 | 整合意识与潜意识后，内在智慧给出的方向 |

**适用场景**：自我探索、内心矛盾、"为什么我总是这样"的困惑。

**解读要点**：最具荣格心理学色彩的牌阵。第二张牌（潜意识）是核心——它揭示了被压抑或忽视的内在真相。第三张牌代表意识与潜意识对话后的整合智慧。

### 模式 4：身心灵（holistic）

| 位置 | 含义 |
|------|------|
| 身体 | 身体层面的状态——健康、精力、行动力 |
| 心理 | 情绪与心理层面——内心感受、压力、期待 |
| 灵性 | 灵性层面——生命课题、灵魂成长方向 |

**适用场景**：整体状态检视、健康相关问题、身心平衡议题。

**解读要点**：三个层次由外而内。关注三张牌之间是否存在矛盾（如身体疲惫但灵性活跃），矛盾本身就是关键信息。整合解读时强调身心灵的协调与失衡。

### 模式 5：关系分析（relationship）

| 位置 | 含义 |
|------|------|
| 你 | 你在这段关系中的状态、角色与能量 |
| 对方 | 对方在关系中的状态、态度与能量 |
| 关系走向 | 两人之间的动态与关系的发展方向 |

**适用场景**：感情关系、朋友关系、职场人际关系问题。

**解读要点**：前两张牌不做道德评判，只呈现双方各自的心理状态。第三张牌是两股能量交汇后的趋势，而非"结局"。强调：改变任何一方的能量都可能改变走向。

### 模式 6：二选一（choice）

| 位置 | 含义 |
|------|------|
| 选项A | 选择 A 的能量、潜在的体验与影响 |
| 选项B | 选择 B 的能量、潜在的体验与影响 |
| 建议 | 更深层的指引——也许答案不在 A 或 B |

**适用场景**：面临两难选择、需要在两个方向中做决策。

**解读要点**：A 和 B 不是简单的"好/坏"判断。解读时呈现各自路径的体验与代价。第三张"建议"牌至关重要——它可能揭示一个超越二元选择的视角，或指出做决策时真正需要考虑的核心因素。

### 模式 7：成长功课（growth）

| 位置 | 含义 |
|------|------|
| 该保留的 | 当前阶段需要继续坚持和珍惜的部分 |
| 该放下的 | 不再服务于成长的模式、执念或习惯 |
| 该学习的 | 当前阶段的核心生命课题与成长方向 |

**适用场景**：人生转折期、自我成长、"接下来我该怎么做"的问题。

**解读要点**：积极的成长导向牌阵。"该放下的"不意味着否定，而是已经完成使命的部分。"该学习的"往往是目前不舒适但最有价值的方向。整体解读呈现一个"断舍离到蜕变"的弧线。

## 三、五牌占卜 Five-Card Spread（5 张）

### 模式 1：时间之流（timeline）- 默认

| 位置 | 含义 |
|------|------|
| 远过去 | 深远的根源——童年印记、家族模式或早期经历的影响 |
| 近过去 | 近期事件或最近阶段的心理主题 |
| 现在 | 此刻的核心状态与关键能量 |
| 近未来 | 即将面对的主题、正在酝酿的变化 |
| 远未来 | 长期趋势——如果持续当前的心理成长路径 |

**适用场景**：人生阶段审视、长期方向规划、重大人生议题。

**解读要点**：五张牌构成一条完整的心理编年史。解读时注重：

- 从远过去到现在，是否存在重复的模式或主题
- 现在是"转折点"还是"延续"
- 近未来与远未来之间的关系——短期挑战是否服务于长期成长
- 整条时间线是否呈现出某种心理发展的弧线（英雄之旅、死亡与重生等）

### 模式 2：十字展开（cross）

| 位置 | 含义 |
|------|------|
| 主题 | 问题的核心——当前最需要面对的主题 |
| 障碍 | 横亘在前的阻碍——可能是外在也可能是内在 |
| 过去影响 | 过去的经历如何影响了当前的局面 |
| 未来趋势 | 事态的自然发展方向 |
| 核心建议 | 整合所有信息后，最关键的行动指引 |

**适用场景**：复杂问题的深度分析、重要决策的全面审视。

**解读要点**：简化版凯尔特十字。中心"主题+障碍"构成核心张力，"过去+未来"提供时间纵深，"核心建议"是收束与指引。重点关注"障碍"牌与"主题"牌的互动关系——障碍往往是主题的阴影面。

### 模式 3：元素平衡（element）

| 位置 | 含义 |
|------|------|
| 火·行动 | 行动力、热情、意志力的状态 |
| 水·情感 | 情感、直觉、关系的状态 |
| 风·思维 | 思维、沟通、认知的状态 |
| 土·物质 | 物质、健康、现实事务的状态 |
| 灵·指引 | 整体灵性指引——四元素的平衡建议 |

**适用场景**：整体生活状态检视、寻找失衡的根源、追求全面发展。

**解读要点**：四元素对应生活的四个维度。解读时先逐一审视各元素的状态，然后关注哪些元素特别强或特别弱——失衡本身就是重要信息。第五张"灵·指引"牌帮助整合四元素，提出平衡之道。
