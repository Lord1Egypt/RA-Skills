# 卦际关系：模型格栅

> **30秒应用**：分析完主卦后，至少拉三个关联视角——错卦（反面看）、综卦（倒过来/对方角度看）、卦序（上一步和下一步）。这三个视角构成最基本的格栅交叉验证。如果三个视角得出矛盾结论→说明有盲区。

## 核心概念

64卦不是64个孤立模型。卦与卦之间存在多种关系，构成了交叉验证网络——
这正是芒格"格栅理论"的体现。面对一个情境，不只用一个卦分析，还要用关联卦交叉验证。

## 四种卦际关系

### 1. 错卦（反面视角）
**定义**：把一个卦的所有爻阴阳互换（阳变阴，阴变阳），得到错卦。
错卦代表了**同一情境的反面**或**对立视角**。

**用法**：分析完主卦后，看它的错卦——"反过来想呢？"
- 乾䷀ ↔ 坤䷁：领导视角 vs 执行视角
- 泰䷊ ↔ 否䷋：顺境逻辑 vs 逆境逻辑
- 离䷝ ↔ 坎䷜：平台思维 vs 风险思维

### 2. 综卦（镜像视角）
**定义**：把一个卦上下颠倒（旋转180度），得到综卦。
综卦代表了**从对方角度/反转角度**看同一件事。

**用法**：分析完主卦后，看它的综卦——"对方怎么看？倒过来看呢？"
- 师䷆ ↔ 比䷇：竞争 vs 合作（同一局势的两种策略）
- 临䷒ ↔ 观䷓：深入介入 vs 退后观察
- 损䷨ ↔ 益䷩：减少 vs 增加（同一经营动作的两个方向）

### 3. 互卦（机制视角）
**定义**：取卦的二三四爻为下卦、三四五爻为上卦，形成互卦。
互卦代表了**内部机制/深层结构**——"表象之下在发生什么？"

**用法**：分析主卦时，同时看互卦，找出隐含的深层问题。
- 屯䷂ 的互卦是 剥䷖：创业初期的内部隐患是资源流失
- 蛊䷑ 的互卦是 归妹䷵：组织腐败的内部机制往往是裙带关系

### 4. 卦序关系（演化视角）
**定义**：64卦的排列顺序不是随机的，相邻卦象之间有因果/演化关系。

**用法**："这个情境的上一步是什么？下一步会是什么？"
- 屯→蒙→需：创业难 → 需要培养人 → 等待时机成熟
- 泰→否：顺境中埋着衰退的种子
- 剥→复：衰退到极致就会反弹

## 格栅交叉验证模板

分析一个商业情境时，用以下矩阵交叉验证：

```
          视角1（主卦）    视角2（错卦）    视角3（综卦）    视角4（互卦）
核心矛盾    [主卦的冲突]   [对立的冲突]    [反转的冲突]     [内部的冲突]
建议策略    [主卦的策略]   [反面的考量]    [反转的策略]     [内部的策略]
关键风险    [主卦的警示]   [对立的警示]    [反转的警示]     [内部的警示]
下一步      [卦序中的下一卦]
```
