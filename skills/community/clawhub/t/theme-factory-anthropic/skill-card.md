## Description: <br>
Theme Factory Anthropic helps agents apply preset or custom font and color themes to slides, documents, reports, and HTML artifacts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[pupuking723](https://clawhub.ai/user/pupuking723) <br>

### License/Terms of Use: <br>
Apache 2.0 <br>


## Use Case: <br>
Employees, external creators, and developers use this skill to choose and apply consistent visual themes to user-selected slides, documents, reports, and HTML pages. It can also draft a custom theme when the preset theme set does not fit the artifact. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The referenced visual showcase PDF is missing from the artifact, so theme selection may rely on Markdown theme descriptions instead of a visual preview. <br>
Mitigation: Use the included theme Markdown files for selection and ask the user to confirm the chosen theme before applying it. <br>
Risk: Applying a theme to the wrong artifact could unexpectedly change user-facing styling. <br>
Mitigation: Confirm which artifact should be edited before applying a selected or custom theme. <br>
