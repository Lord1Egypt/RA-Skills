# 追本溯源 · essence-trace

> **定位**：对任何观点/现象/问题，**纵向深钻到底**——直到物理/逻辑层面不可再分。
> **方法论**：5 Whys（丰田）+ 第一性原理（马斯克）+ 善人「投资思维」杠杆解。
> **版本**：v1.0.0 | 2026-06-26

---

## 🎯 技能目标

**核心价值**：把"对一个问题的浅层理解"提升到"看见其本质结构"。

**核心能力**：
1. **5 Whys 追问**：连续追问 5 次"为什么"，逼近根因
2. **第一性原理**：剥离所有假设/类比/惯例，回到物理/逻辑起点
3. **识别不可分节点**：找到"物理/逻辑层面无法再分"的节点
4. **杠杆点识别**：在追问链中找到 2-3 个高杠杆点

**与"普通思考"的核心差异**：
- ❌ 普通思考："房价会涨是因为政策支持"
- ✅ essence-trace："房价会涨 → 政策支持 → 政策目标是稳增长 → 稳增长靠什么 → 投资驱动 → 投资需要融资 → 融资靠土地信用 → 土地信用是房价的锚"

## 🧠 认知框架：5 Whys × 第一性原理

> **核心命题**：任何"看起来复杂"的问题，**必有可识别的底层结构**。把这个底层结构挖出来 = 看透本质。

### 5 Whys（追根究底）

**原理**：对一个现象，连续问 5 次"为什么"，每次都基于上一步的答案。

**示例**（汽车抛锚）：
```
Q1: 为什么汽车抛锚？→ 发动机停了
Q2: 为什么发动机停了？→ 没油了
Q3: 为什么没油了？→ 油表坏了没发现
Q4: 为什么油表坏了没发现？→ 仪表盘没报警
Q5: 为什么仪表盘没报警？→ 传感器设计缺陷
→ 本质：单一传感器设计，缺乏冗余
```

**注意**：不是机械地问 5 次，而是问**到根因**。有时 3 次就够，有时要 7-8 次。

### 第一性原理（First Principles）

**原理**：剥离所有假设/类比/惯例，回到物理/逻辑/数学的**不可分节点**。

**步骤**：
1. 识别问题中的所有假设（"通常是这样"）
2. 把假设逐个剥离
3. 剥离后剩下的就是"第一性原理"

**示例**（电池成本）：
```
假设 1：电池很贵 → 真的吗？
  剥离：电池 = 碳 + 镍 + 铝 + 一些聚合物
  市场上这些材料：碳 $X/kg + 镍 $Y/kg + ...
  计算：$80/kWh
  → 实际售价：$1000/kWh
  → 巨额差价来自什么？→ 供应链整合/规模不经济/工艺壁垒
  → 第一性原理：原材料成本极低，溢价来自过程
```

### 善人融合：杠杆点识别

**原理**：在追问链中，识别出**2-3 个高杠杆点**——小改变带来大影响的位置。

**杠杆点判定**：
- 这点改变 → 上游/下游多个节点都受影响？
- 这点改变 → 长期/短期都有效？
- 这点改变 → 系统整体效率提升？

## 🔄 处理流程（6 步）

详细流程见 [`PROCESS.md`](./PROCESS.md)。简要步骤：

```
Step 1: 初始问题陈述 → 用 1 句话清晰陈述
Step 2: 第 1 层追问 → 前置问题是什么？
Step 3: 连续追问 → 5 Whys（不限于 5 次）
Step 4: 识别不可分节点 → 物理/逻辑终点
Step 5: 回溯链条 → 画出从问题到本质的链
Step 6: 杠杆点识别 → 2-3 个高杠杆节点
        ↓
Final: 一句话本质 = "这个问题的本质是 ______"
```

## 🛑 终止条件：什么时候不能再拆了

**核心问题**：深钻到什么程度停？

### 终止判定标准

**1. 物理/数学不可分**
- 抵达物理定律（牛顿三定律/热力学/相对论）
- 抵达数学公理（1+1=2 / 集合论基础）
- 抵达物质基本单位（原子/基本粒子）

**2. 逻辑/概念不可分**
- 抵达不可定义的基本概念（存在/意识/时间——本身需要被定义）
- 抵达公理化体系起点（欧氏几何 5 公设）
- 抵达"自明之理"（axiom）

**3. 经验/实证终止**
- 抵达"实验事实"（水的沸点 100°C / 光速 30 万 km/s）
- 抵达"统计规律"（大数定律 / 中心极限定理）

**4. 选择/价值终止**
- 抵达"个人选择"（本质上是偏好问题，无对错）
- 抵达"价值判断"（善恶/美丑/好坏——无法再追问"为什么"）

### 终止的标志

当追问遇到以下情况时停：
- "它就是这样"（事实/定律）
- "这是定义"（概念基础）
- "这是选择"（价值判断）
- "我想要"（个人偏好）
- "循环论证"（已经回到起点）

**注意**：**没有绝对的"不可分"**——任何"不可分"都是当前认知框架下的边界。
哲学上"为什么存在"等终极问题没有终点，但在**实用层面**，上面的标准足够用。

## 🚧 边界

| 不做 | 原因 |
|------|------|
| 哲学终极问题（"为什么存在"） | 实用层面无终点，转向 philosophy 类技能 |
| 纯事实查询（"地球半径多少"） | 不需要溯源，直接查 |
| 涉及情感/价值（"什么是幸福"） | 溯源到价值判断会终止，但可能缺乏实用价值 |
| 不可证伪的命题（"上帝是否存在"） | 缺乏终止条件，转向 belief 类讨论 |
