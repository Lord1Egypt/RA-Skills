## Description: <br>
Create, audit, or update an OpenClaw agent persona through a structured multi-round calibration workflow. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[dengyh](https://clawhub.ai/user/dengyh) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
OpenClaw users and developers use this skill to calibrate an agent persona through staged interviews, then translate approved preferences into durable persona, identity, user, and memory file updates. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Persona or memory updates can persist inaccurate, overly broad, or sensitive preferences in long-lived agent files. <br>
Mitigation: Review proposed changes before applying them, keep long-term memory limited to stable preferences, and avoid storing sensitive personal details unless explicitly intended. <br>
