# 核保风控体系与异常标记规则

> 本文件为 `insurance-anti-fraud` Skill 配套参考

## 一、核保风险分级标准

### 1.1 寿险核保风险等级

| 风险等级 | 评估条件 | 核保结论 | 附加条件 |
|---------|---------|---------|---------|
| **标准体（Preferred）** | 完全健康告知、无异常、无家族风险 | 标准承保 | 无 |
| **次标准体A** | 轻微异常（超重10%内、脂肪肝早期） | 加费承保 | 保费+10%-30% |
| **次标准体B** | 中等异常（高血压控制良好、糖尿病早期） | 有条件承保 | 除外责任+加费 |
| **延期体** | 病情不稳定、待确诊 | 延期观察 | 6-12个月后重核 |
| **拒保体** | 重大既往症、高风险职业 | 拒保 | 不予承保 |

### 1.2 常见异常核保处理

| 异常情况 | 评估方法 | 核保结论 |
|---------|---------|---------|
| BMI 28-30 | 轻度肥胖，体检正常 | 加费10%-20% |
| BMI >30 | 中重度肥胖 | 加费20%-30%或拒保 |
| 轻度高血压（<160/100） | 无并发症，控制良好 | 加费15%-25% |
| 重度高血压（≥160/100） | 有并发症风险 | 拒保或延期 |
| 乙肝病毒携带 | 肝功能正常 | 除外承保或加费 |
| 甲状腺结节 | TI-RADS分级1-2级 | 除外甲状腺癌 |
| 乳腺结节 | BI-RADS分级1-2级 | 除外乳腺癌 |
| 既往拒保记录 | 详细调查问卷 | 综合评估 |

### 1.3 高风险职业分级（意外险）

| 职业类别 | 典型职业 | 附加保费 |
|---------|---------|---------|
| 1类（最低风险） | 办公室职员、老师、公务员 | 0 |
| 2类（低风险） | 销售员、服务员 | +20% |
| 3类（中等风险） | 司机、建筑工人 | +50% |
| 4类（高风险） | 电工、焊工 | +100% |
| 5类（极高风险） | 矿工、悬崖作业 | +200%或拒保 |
| 6类（不可保） | 试飞员、爆破工人 | 拒保 |

## 二、异常标记规则（自动核保引擎）

### 2.1 自动标记规则表

| 异常信号 | 标记级别 | 自动动作 | 人工复核 |
|---------|---------|---------|---------|
| 同一天多张保单（多家公司） | 🔴HIGH | 进入反欺诈协查 | 必须 |
| 高保额低净值（保额/收入>20倍） | 🔴HIGH | 要求财务证明 | 必须 |
| 既往拒保/延期记录 | 🔴HIGH | 详细调查问卷 | 必须 |
| 等待期≤30天+高保额 | 🔴HIGH | 深度核保 | 必须 |
| 短期内多次投保健康险 | 🟡MEDIUM | 体检要求 | 建议 |
| 高风险职业 | 🟡MEDIUM | 职业核保专项 | 建议 |
| 既往理赔次数>5次 | 🟡MEDIUM | 历史分析报告 | 建议 |
| 年龄>65岁首次投保高额寿险 | 🟡MEDIUM | 财务+健康双核 | 必须 |
| 受益人关系异常 | 🟡MEDIUM | 受益人关系核实 | 必须 |
| 正常告知 | 🟢LOW | 自动通过 | 无需 |

### 2.2 核保决定时效标准

| 案件类型 | 核保时效 | 说明 |
|---------|---------|------|
| 标准体 | 1个工作日 | 自动通过 |
| 次标准体 | 3个工作日 | 需核保人员评估 |
| 高保额件（>100万） | 5个工作日 | 体检+财务核查 |
| 高风险案件 | 7个工作日 | 深度调查 |
| 拒保/延期 | 3个工作日 | 需高级核保审批 |

## 三、保险金额与财务核验标准

| 年缴保费/家庭年收入 | 可承受保额上限参考 | 说明 |
|------------------|-----------------|------|
| <5% | 保额的10倍 | 国际标准 |
| 5%-10% | 保额的15倍 | 合理范围 |
| 10%-20% | 保额的20倍 | 上限 |
| >20% | 需要财务证明 | 核查真实需求 |

**常见财务核验指标**：
- 年收入与累计保费比例
- 职业收入合理性
- 企业主资产核实（工商数据）
- 关联人保费累加（配偶/子女）
