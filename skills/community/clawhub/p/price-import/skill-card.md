## Description: <br>
Price Import helps agents import price data from images, text, or Excel by standardizing input, matching product records, and producing a confirmation report before API import. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[young0529](https://clawhub.ai/user/young0529) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users or operators use this skill to process supplier price lists, review matched and unmatched products, and confirm controlled imports into a product price system. It also supports rule review and requirements feedback workflows around the import process. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can affect price import and product/catalog workflows through controlled API actions. <br>
Mitigation: Install only with scoped API authority, require user confirmation before importing or creating products, and review match reports and import logs before accepting results. <br>
Risk: The broader /fix workflow can generate requirements that go beyond routine price import. <br>
Mitigation: Clarify whether requirements-management behavior is allowed before use, and review generated requirement documents before acting on them. <br>
