# AI提示词变现工作流

## 工作流概述
将AI提示词产品化，通过多平台分发实现变现的完整流程。

## 核心流程

```
[创意设计] → [提示词编写] → [质量审查] → [合规检查] → [产品包装] → [多平台发布] → [私域引流]
```

## 阶段详解

### 1. 创意设计
- **来源**：用户痛点、行业需求、平台热点
- **工具**：热点调研、竞品分析
- **输出**：提示词选题清单

### 2. 提示词编写
- **原则**：
  - 角色定位清晰
  - 任务描述具体
  - 输出格式规范
  - 约束条件明确
- **模板**：参考 `./templates/` 目录
- **输出**：原始提示词文本

### 3. 质量审查
- **维度**：
  - 功能完整性（是否能解决需求）
  - 输出稳定性（多次执行结果一致性）
  - 可用性（是否易于修改和使用）
- **标准**：详见 `./quality-checklist.md`

### 4. 合规检查
- **工具**：content-compliance-checker
- **检查项**：
  - 无敏感政治内容
  - 无违规违法内容
  - 无虚假夸大宣传
- **输出**：合规报告

### 5. 产品包装
- **要素**：
  - 产品名称（吸引眼球）
  - 使用场景描述
  - 效果展示案例
  - 使用说明
  - 常见问题FAQ
- **输出**：产品详情页

### 6. 多平台发布
- **平台策略**：
  - 闲鱼：直接销售
  - 小红书：种草引流
  - 知识星球：付费社群
- **发布规范**：详见 `./publish-guide.md`

- **路径**：平台内容 → 引导私域 → 付费转化
- **钩子**：免费提示词/使用教程
- **工具**：微信生态、邮件
