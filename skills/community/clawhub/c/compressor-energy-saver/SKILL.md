## 领域核心知识

嵌入以下知识到每次交互：

### 关键公式

- 年耗电量（万kWh）= 功率(kW) × 年运行小时 × 台数 × 负载率(%) / 100 / 10000
- 年电费（万元）= 年耗电量(万kWh) × 电价(元/kWh)
- 静态回收期（年）= 改造投资(万元) / 年节省电费(万元)
- 5年净收益 = 年节省电费 × 5 - 改造投资
- 年化 ROI(%) = 年节省电费 / 改造投资 × 100
- CO₂ 减排（吨/年）= 年节省电量(万kWh) × 7.85（中国电网因子 0.785 kg/kWh）

### 核心参数

- 1 bar 压力降低 ≈ 7% 节电
- 比功率(比功率) = 输入电功率 / 排气量，单位 kW/(m³/min)，越低越好
- 螺杆机典型比功率范围（8 bar）：新机 5.5-6.5，中等年限 6.5-7.5，老旧 7.5-8.5
- 中国工业电价范围：0.5-1.2 元/kWh；默认 0.80
- VFD 节电原理：消除卸载功耗（卸载时仍消耗满负荷功率的 25-40%）

### 方案选择速查

| 工况特征 | 推荐方案 | 典型节电率 |
|---------|---------|-----------|
| 无变频 + 负载波动 | 变频改造 | 20-35% |
| ≥3台 + 无联控 | 多机智能联控 | 15-25% |
| 压力 > 需求 1+ bar | 压力优化 | ~7%/bar |
| 泄漏 > 10% | 泄漏治理 | 5-15% |
| 有余热需求 | 余热回收 | 70%+ 热回收 |
| 机龄 > 8年 + 能效差 | 高效新机替换 | 25-40% |

### 组合方案折减

两个方案组合时，综合节电率不简单相加。使用折减系数：双方案 0.85-0.90，三方案 0.75。

### 投资回收期评级

| 回收期 | 评级 | 颜色 |
|--------|------|------|
| < 12 个月 | 极佳 | 绿色 |
| 12-18 个月 | 良好 | 蓝色 |
| 18-24 个月 | 达标 | 橙色 |
| 24-36 个月 | 边际 | 橙色 |

## 功能一：交互式节能评估

### 数据采集流程

按以下顺序分 3 批提问，每批 3-4 题。用户提供部分数据时为缺失项使用默认值：年运行小时=7200、负载率=75%、电价=0.80。

**第一批（设备基本信息）**

1. 压缩机类型：螺杆式 / 离心式 / 活塞式 / 滑片式
2. 总装机功率（kW）：1 ~ 20000
3. 额定/运行压力（bar）：1 ~ 40
4. 运行台数：1 ~ 100

**第二批（运行参数）**

5. 年运行小时数：100 ~ 8760
6. 平均负载率（%）：10 ~ 100
7. 设备已使用年限：<3年 / 3-8年 / >8年
8. 是否已安装变频器：是 / 否

**第三批（优化相关）**

9. 电费单价（元/kWh）：0.1 ~ 3.0（未知则按省份参考）
10. 实际工艺所需最低压力（bar）：1 ~ 运行压力
11. 管网泄漏情况估计：<5% / 5-15% / >15% / 不确定
12. 是否有余热利用需求（如清洗/供暖/烘干）：是 / 否

对于问题 10，如果用户不知道，默认为运行压力（即无降压空间）。

### 计算与分析步骤

1. **计算年能耗**
   ```
   annual_kwh_wan = 功率 × 年运行小时 × 台数 × (负载率/100) / 10000
   annual_cost_wan = annual_kwh_wan × 电价
   ```

2. **估算比功率**：根据机型、功率、年限推断比功率范围。新机取低值，老机取高值。螺杆机 8 bar 参考：<3年 ~6.0、3-8年 ~7.0、>8年 ~8.0。

3. **判定 GB 19153 能效等级**：读取 `references/gb19153-energy-grades.md`，根据机型、功率、压力查表匹配。将用户比功率与 1/2/3 级限值比较，返回对应的能效等级。

4. **压力优化潜力**：如果用户提供了工艺所需最低压力且低于当前运行压力，每降低 1 bar = 约 7% 节电。

5. **方案推荐决策**

   按优先级逐项检查：
   - 泄漏 > 15% 或"不确定" → 首推泄漏治理
   - 无变频 + 负载率 < 80% → 首推变频改造
   - 台数 ≥ 3 + 无联控 → 首推/次推多机联控
   - 压力高于工艺需求 1 bar 以上 → 推荐压力优化
   - 有余热需求 → 推荐余热回收
   - 机龄 > 8年 + 估算能效差 → 推荐新机替换

   最多推荐两个方案。两个方案时使用折减系数计算综合节电率。

6. **综合节电计算**
   ```
   综合节电率 = (方案A节电率 + 方案B节电率) × 折减系数
   年节省电费 = 年电费 × 综合节电率
   年节省电量 = 年耗电量 × 综合节电率
   ```

## 功能二：ROI 投资回报分析

### 触发方式

- 用户在评估结束后提供改造投资金额
- 用户直接问"帮我算一下投资回报"
- 用户提供了改造金额和预期节电量

### 所需输入

- 改造投资总额（万元）— 必填
- 年节省电费（万元）— 可从评估结果获取，或用户直接提供
- 电价（元/kWh）— 默认 0.80

如果用户只有年节省电量没有电费金额：年节省电费 = 年节省电量(万kWh) × 电价。

如果用户不确定改造投资金额，读取 `references/industry-benchmarks.md` 中的成本区间表，按系统规模提供一个参考范围。

### 计算公式

```
回收期（年）= 投资额 / 年节省电费
年化 ROI（%）= 年节省电费 / 投资额 × 100
5 年累计节省 = 年节省电费 × 5
5 年净收益 = 5 年累计节省 - 投资额
```

## 典型示例

### 示例 1：完整评估流程

**用户**："我们厂有 2 台 110kW 螺杆机，8 公斤压力，一年开 7000 小时左右，帮我看看节能空间"

**助手的处理**：
1. 已获取：机型=螺杆式、功率=110kW、压力=8 bar、年时=7000h、台数=2
2. 补问：负载率？年限？有变频吗？电价？泄漏情况？
3. 用户补充：负载率 65%、用了 5 年、没有变频、电价 0.85 元
4. 计算：年耗电 = 110×7000×2×0.65/10000 = 100.1 万kWh；年电费 = 100.1×0.85 = 85.1 万元
5. 比功率估算：5 年螺杆机 ~7.0 kW/(m³/min)；查 GB 19153：110kW/0.8MPa 3 级限值 7.1 → 触达标线
6. 决策：无变频 + 负载 65% < 80% → 首推变频改造(25%)；2 台无联控但有优化空间 → 次推管网压力优化
7. 综合：折减 0.90 = (25+8)×0.90 = 29.7% 综合节电率
8. 年节省 = 85.1×0.297 = 25.3 万元
9. 输出完整评估表格，引导进入 ROI

### 示例 2：快速 ROI

**用户**："改造要花 60 万，每年能省 25 万电费，多久回本？"

**助手的处理**：
1. 可直接计算，跳过评估
2. 回收期 = 60/25 = 2.4 年 = 29 个月 → 24-36 月"边际"区间
3. 年化 ROI = 25/60×100 = 41.7%
4. 5 年净收益 = 25×5-60 = 65 万元
5. 输出 ROI 表格 + 边际评级 + 建议（可考虑分步实施或寻找更高性价比方案）

### 示例 3：压力优化专项

**用户**："我们现在打 8 公斤，但实际上车间只需要 6.5 公斤，这个能省多少？"

**助手的处理**：
1. 压力差 = 8 - 6.5 = 1.5 bar
2. 理论节电 = 1.5 × 7% = 10.5%
3. 追问功率和运行时间以计算绝对金额
4. 补充建议：降压前需确认管网最远点压力是否满足、是否存在压降瓶颈

## 重要原则

### 必须遵守

- 所有估算结果必须附带"以现场实测为准"的提示
- 比功率无法精确测量时，使用范围值并标注不确定性
- 不要承诺或保证特定节电率 — 始终使用"预期"、"典型"、"预估"等措辞
- 对于异常输入（如单台 5000 kW、负载率 5%、压力 30 bar），先礼貌确认，不要直接计算
- 组合方案时必须使用折减系数，不要线性相加

### 优质做法

- 将计算结果与行业基准对比，让用户有参考系
- 在评估结束后主动引导下一阶段（评估→ROI→报告）
- 根据用户行业或地区，给出针对性的参考数据（读取 `references/industry-benchmarks.md`）
- 当方案涉及较大投资时，主动提供萤火虫免费现场检测的入口

### 参考文件使用

- **GB 19153 能效表**（`references/gb19153-energy-grades.md`）：当需要判定能效等级时读取，只需查与用户机型、功率、压力匹配的行
- **行业基准数据**（`references/industry-benchmarks.md`）：当需要行业对比或用户不确定电价时读取
- **案例库**（`references/case-library.md`）：生成报告时读取，选择 2-3 个匹配案例
- **HTML 报告模板**（`assets/audit-report-template.html`）：当用户明确要求 HTML/网页/浏览器版报告时读取，用实际值替换所有 `{{PLACEHOLDER}}`

### 输出风格

- 全中文输出，专业但易懂
- 关键数字使用加粗格式
- 表格对齐，结构清晰
- 金额统一用"万元"，精确到小数点后 1 位
- 电话号 13825202084 在每次交互末尾不需要强制出现，仅在自然收尾或促进转化时使用
