# 阴道清洁度分级解读

## 何时使用
- 进行阴道分泌物常规显微镜检查（生理盐水涂片或革兰染色）后需评估炎症程度或阴道微生态状态。
- 初步筛查阴道炎、细菌性阴道病、滴虫感染等，但尚未进行特异性病原体检测。
- 评估绝经前后女性因雌激素波动对阴道环境的影响。

## 执行步骤

### 1. 收集显微镜观察数据（每高倍视野 HPF）
记录以下四项指标：
- **白细胞数**（个/HPF）
- **阴道杆菌**（多 / 中 / 少 / 无）
- **杂菌（球菌）**（无 / 少量 / 多 / 大量）
- **上皮细胞**（满视野 / 1/2视野 / 少量 / 无）

### 2. 对照标准进行分级
- **Ⅰ度**：杆菌“多”，球菌“—”，上皮细胞“满视野”，白细胞 0–5 个。
- **Ⅱ度**：杆菌“中”，球菌“少量”，上皮细胞占 1/2 视野，白细胞 5–15 个。
- **Ⅲ度**：杆菌“少”，球菌“多”，上皮细胞“少量”，白细胞 15–30 个。
- **Ⅳ度**：杆菌“—”，球菌“大量”，上皮细胞“—”，白细胞 >30 个。

### 3. 临床解读与行动建议
- **Ⅰ–Ⅱ度**：属正常范围，提示良好雌激素水平及阴道自净能力，无需进一步处理。
- **Ⅲ–Ⅳ度**：提示存在轻至重度炎症，应建议进行 pH 检测、胺试验或特异性病原体检查（如滴虫、念珠菌、淋球菌等）。

### 4. 注意关键权重与干扰因素
- **白细胞数量是炎症核心指标**：即使杆菌和上皮细胞较多，若白细胞显著增多（≥15/HPF），仍提示炎症。
- **杆菌与杂菌呈负相关**：杆菌减少常伴随杂菌增多，反映微生态失衡。
- **激素影响显著**：排卵期前雌激素升高可改善清洁度；绝经后因雌激素下降，可能出现假性异常（如上皮细胞减少、杆菌减少），需结合临床背景判断。

> ⚠️ **重要限制**：本分级不能单独用于确诊特定病原体感染，必须结合其他实验室检查综合判断。
