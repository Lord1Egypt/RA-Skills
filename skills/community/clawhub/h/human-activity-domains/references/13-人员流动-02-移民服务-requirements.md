## 移民服务生成要求

移民服务每种任务类型的"零件清单"——必选/可选组件、组装顺序、领域约束。按清单逐项产出。

---

### 1 移民评估

#### T1-1 资格评估

- **必选组件**: 评估标准、评估内容、评估结果、评估报告
- **可选组件**: 评估工具、评估问卷、评估访谈
- **组装顺序**: 确定评估标准 → 收集评估内容 → 执行评估 → 生成报告
- **约束**: 标准必须明确；内容必须真实；评估必须客观
- **格式**: 资格评估报告（Markdown）

#### T1-2 条件分析

- **必选组件**: 目标国政策、申请要求、申请难度、分析报告
- **可选组件**: 政策调研、要求对比、难度评估
- **组装顺序**: 调研目标国政策 → 分析申请要求 → 评估申请难度 → 生成报告
- **约束**: 政策必须准确；要求必须完整；难度必须客观
- **格式**: 条件分析报告（Markdown）

#### T1-3 方案设计

- **必选组件**: 移民类别、申请路径、时间规划、方案说明
- **可选组件**: 方案对比、方案优化、方案评审
- **组装顺序**: 确定移民类别 → 设计申请路径 → 规划时间 → 撰写说明
- **约束**: 类别必须匹配；路径必须可行；时间必须合理
- **格式**: 移民方案（Markdown）

#### T1-4 风险评估

- **必选组件**: 风险类型、风险识别、风险评估、风险报告
- **可选组件**: 风险工具、风险模型、风险案例
- **组装顺序**: 确定风险类型 → 识别风险 → 评估风险 → 生成报告
- **约束**: 类型必须全面；识别必须准确；评估必须客观
- **格式**: 风险评估报告（Markdown）

### 2 移民申请

#### T2-1 材料准备

- **必选组件**: 材料清单、材料收集、材料审核、材料整理
- **可选组件**: 材料模板、材料指导、材料翻译
- **组装顺序**: 列出材料清单 → 收集材料 → 审核材料 → 整理材料
- **约束**: 清单必须完整；收集必须齐全；审核必须严格
- **格式**: 材料准备清单（Markdown）

#### T2-2 申请提交

- **必选组件**: 申请方式、申请材料、申请确认、申请记录
- **可选组件**: 申请平台、申请代理、申请指导
- **组装顺序**: 选择申请方式 → 准备申请材料 → 提交申请 → 记录申请
- **约束**: 方式必须正确；材料必须完整；确认必须及时
- **格式**: 申请提交记录（Markdown）

#### T2-3 面试辅导

- **必选组件**: 辅导内容、辅导方式、辅导材料、辅导效果
- **可选组件**: 辅导老师、辅导课程、辅导模拟
- **组装顺序**: 准备辅导内容 → 选择辅导方式 → 准备辅导材料 → 评估辅导效果
- **约束**: 内容必须实用；方式必须有效；材料必须完整
- **格式**: 面试辅导方案（Markdown）

#### T2-4 申请跟进

- **必选组件**: 进度查询、补料处理、结果通知、跟进记录
- **可选组件**: 跟进工具、跟进系统、跟进提醒
- **组装顺序**: 查询进度 → 处理补料 → 通知结果 → 记录跟进
- **约束**: 查询必须及时；处理必须迅速；通知必须准确
- **格式**: 申请跟进记录（Markdown）
