# 梅毒血清学检测分类与应用

## 触发条件
当患者具有以下任一情况时启动本流程：
- 有不安全性行为史；
- 出现硬下疳、皮疹等典型梅毒临床表现。

## 执行步骤

### 1. 初筛检测（非梅毒螺旋体抗原血清试验）
- **适用方法**：RPR、TRUST（常规筛查）；VDRL（仅用于脑脊液检测，尤其怀疑神经梅毒时）。
- **用途**：评估病情活动性及治疗效果，滴度变化反映疾病状态。
- **操作要求**：阳性结果必须进入确证流程。

### 2. 确证检测（梅毒螺旋体抗原血清试验）
- **适用方法**：TPHA、FTA-ABS、ELISA 或化学发光法。
- **用途**：确认是否曾感染梅毒螺旋体；阳性结果通常终身存在，不可用于疗效监测。
- **操作要求**：仅在初筛阳性后使用。

### 3. 结果综合解读
- **活动性梅毒**：确证试验阳性 + 非特异性试验（RPR/TRUST/VDRL）阳性。
- **既往感染或已治愈**：确证试验阳性 + 非特异性试验阴性。
- **早期梅毒初筛阴性**：若临床高度怀疑但血清学阴性，应进行暗视野显微镜检查溃疡渗出物；必要时采用PCR检测梅毒螺旋体核酸。

### 4. 特殊情形处理
- **神经梅毒疑似**：必须使用VDRL检测脑脊液。
- **疗效随访**：定期复查RPR或TRUST滴度，观察是否下降≥4倍（即2个稀释度）以判断治疗有效。

> 注意：不得单独依赖单一检测结果诊断梅毒；必须结合临床表现、流行病学史及检测组合结果综合判断。
