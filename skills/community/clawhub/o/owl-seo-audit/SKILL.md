# SEO Audit Skill

## Purpose
Perform a complete SEO audit of any website, identifying issues and opportunities for improvement.

## Audit Checklist

### Technical SEO
- [ ] Page speed (target: <3s load time)
- [ ] Mobile responsiveness
- [ ] SSL certificate
- [ ] XML sitemap
- [ ] Robots.txt
- [ ] Structured data markup
- [ ] Canonical tags
- [ ] 404 errors
- [ ] Redirect chains
- [ ] Core Web Vitals

### On-Page SEO
- [ ] Title tags (50-60 chars)
- [ ] Meta descriptions (150-160 chars)
- [ ] Header hierarchy (H1-H6)
- [ ] Keyword optimization
- [ ] Internal linking
- [ ] Image alt text
- [ ] Content quality and length

### Off-Page SEO
- [ ] Backlink profile
- [ ] Domain authority
- [ ] Competitor backlinks
- [ ] Brand mentions
- [ ] Social signals

## Output Format
Provide a prioritized list of issues (Critical, High, Medium, Low) with specific recommendations for each.
