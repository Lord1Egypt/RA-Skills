## pywencai(同花顺问财) 使用指南

### 安装
```bash
pip install pywencai
```

### 在投研系统中使用

```python
from data_fetcher import DataFetcher

fetcher = DataFetcher()

# 1. 信号检测
df = fetcher.get_pywencai_signal_stocks('dual_golden')  # 双金叉共振
df = fetcher.get_pywencai_signal_stocks('macd_golden')   # MACD金叉
df = fetcher.get_pywencai_signal_stocks('breakout')      # 突破

# 2. 资金流向
df = fetcher.get_pywencai_fund_flow('今日')   # 今日资金流
df = fetcher.get_pywencai_fund_flow('3日')    # 3日资金流
df = fetcher.get_pywencai_fund_flow('10日')   # 10日资金流

# 3. 行业/概念资金流
df = fetcher.get_pywencai_industry_flow('今日')

# 4. 北向资金
df = fetcher.get_pywencai_north_bound()

# 5. 高股息/高ROE
df = fetcher.get_pywencai_high_dividend()

# 6. 自定义查询（自然语言）
df = fetcher.pywencai_query('市盈率小于20并且ROE大于15%', sort_key='总市值')
df = fetcher.pywencai_query('连续5天上涨', sort_key='涨跌幅')
```

### 常用问财语句

| 查询目的 | 问财语句 |
|---------|---------|
| MACD金叉 | `MACD金叉` |
| KDJ金叉 | `KDJ金叉` |
| 双金叉共振 | `MACD金叉并且KDJ金叉` |
| 涨停股 | `今日涨停` |
| 创新高 | `60日新高` |
| 放量突破 | `放量突破` |
| 超跌反弹 | `RSI超卖` |
| 高股息 | `股息率大于4%` |
| 低PE | `市盈率小于15` |
| 高ROE | `ROE大于20%` |
| 营收高增长 | `营收增长率大于30%` |
| 北向增持 | `北向资金增持` |
| 基金加仓 | `最近一季基金加仓` |

### 注意事项
- 免费版每页最多返回100条数据
- 请求频繁可能被限流，建议加 `sleep` 参数
- 查询语句越精确，返回结果越准确
- 部分查询可能需要同花顺账号登录

## 配置方法

### Tushare Pro — 需要配置

**步骤：**
1. 访问 https://tushare.pro 注册账号
2. 登录后进入个人主页获取 Token
3. 编辑 `tushare_config.py`，填写你的 Token：

```python
TUSHARE_TOKEN = 'your_token_here'
TUSHARE_AUTH = True  # 改为True启用
```
