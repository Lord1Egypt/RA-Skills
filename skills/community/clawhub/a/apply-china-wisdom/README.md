# 📜 中国古智慧运用 V3.0 (The Wisdom Engine)

欢迎使用 **中国古智慧运用 V3.0**。这不是一个简单的古籍检索工具，而是一个将【古典文本】转化为【现代可执行方案】的智慧转化引擎。

## 🌟 核心能力
本技能将古代智慧分为四个核心执行层，能够处理从深层认知到具体博弈的多种需求：

### 1. 🧠 深度分析模式 (Deep Analysis)
**适用场景**：面对人生困境、认知迷茫、组织管理或寻求哲学解答。
- **能级分析**：结合时位与禀赋分析你的现状。
- **溯源定性**：从经典中寻找最契合的底层逻辑。
- **现代转化**：将抽象道理转化为具体的 `1, 2, 3` 实操步骤。
- **辩证对冲**：从对立视角审视风险，确保方案稳健。

### 2. ⚔️ 博弈与竞争执行层 (Tactical Execution)
**适用场景**：商业竞争、职场博弈、资源争夺、快速破局。
- **战略定调**：基于《孙子兵法》判定态势。
- **战术选型**：从《三十六计》匹配具体计谋。
- **动作拆解**：将计谋转化为现代语境下的等价行为。

### 3. 🎨 创意资产生成 (Creative Assets)
**适用场景**：创作、续写、生成 AI Prompt 或构建具有中国文化底蕴的世界观。
- **诗词重构**：提供格律解析与意象重构。
- **视觉解码**：调用美学框架生成高质量的图像 Prompt。
- **叙事骨架**：用古典结构搭建具有哲学反转的故事。

### 4. 🧘 心法训练模式 (Mindset Training)
**适用场景**：希望提升认知维度，而非简单获取答案。
- **引导式推演**：通过镜像投射与共创推演，引导你自行顿悟。
- **认知内化**：将文化直觉转化为个人的决策能力。

### 5. 🌌 天演模式 (The Great Simulation)
**适用场景**：复杂的人生/商业路径预测，或对未来发展方向的概率推演。
- **多 Agent 博弈**：激活 10 个人格模拟器，进行多轮对抗推演。
- **概率坍缩**：输出【最高概率路径】与【潜在黑马路径】。

## 🛠️ 如何使用？

你无需学习复杂的指令，只需在对话中直接描述你的场景。

### 💡 推荐的输入方式：

- **认知需求**：*"我目前在公司处于[某种处境]，感觉[某种矛盾]，请用中国古智慧帮我分析并给出实操建议。"*
- **博弈需求**：*"我现在面临[具体的竞争对手/冲突]，我想在不破坏关系的前提下达成[某个目标]，请为我制定战术。"*
- **创作需求**：*"我想创作一个关于[某个主题]的场景，请结合古典美学为我提供视觉 Prompt。"*
- **推演需求**：*"基于我之前提供给你的[历史背景/数据]，请启动天演模式，推演未来三个月可能出现的三种发展路径。"*
