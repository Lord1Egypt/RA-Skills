## Description: <br>
Creates professional Mermaid.js diagrams for software architecture, domain modeling, process visualization, database schemas, and system interactions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, architects, and technical writers use this skill to draft Mermaid diagrams and rendering guidance for architecture, data models, workflows, API sequences, and related technical documentation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Relaxed Mermaid rendering settings can be unsafe when diagrams come from users or other untrusted sources. <br>
Mitigation: Keep Mermaid's safer security defaults for untrusted diagrams and do not enable securityLevel: 'loose' unless the diagram content is trusted. <br>
Risk: CDN imports and global CLI installation can introduce supply-chain or version drift risk. <br>
Mitigation: Pin Mermaid CDN or package versions where practical and verify the Mermaid CLI package before installing it globally. <br>
