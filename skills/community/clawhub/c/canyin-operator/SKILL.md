# 小餐饮创业运营助手 🍜

专为小餐饮创业者打造的全流程 Skill，覆盖创业→开业→运营→盈利完整链路。

## 核心工作流

### 1. 选址评估模型

```
小餐饮选址评估（满分100分）：

客流质量（30分）：
- 周边人群职业构成（白领=30，学生=25，社区=20）
- 人流动线（必经之路=30，顺路=20，需绕行=5）
- 高峰时段人流（>200人/小时=30，100-200=20，<100=10）

竞争环境（25分）：
- 同品类竞品数量（0=25，1-2=15，3-5=8，>5=0）
- 竞品月销量（某团月销>5000=0，1000-5000=10，<1000=20）
- 差异化空间评估

成本结构（25分）：
- 房租占预估月收入（<15%=25，15-25%=15，25-35%=8，>35%=0）
- 免租期长度（>30天=5，15-30天=3，<15天=0）
- 转让费评估（0=5，有转让费按比例扣分）

物业条件（20分）：
- 上下水/排油烟（齐全=20，需改造=5，不能装=0）
- 电力容量（>20kW=10，10-20kW=5，<10kW=0）
- 位置可见性（街角=20，主街=15，小巷=5）

评分建议：
≥80分：强烈推荐
60-79分：可考虑，注意控制成本
40-59分：谨慎，需要差异化
<40分：不建议
```

### 2. 品类选择与定位

```
小餐饮盈利品类排行（按回本速度）：

| 品类 | 毛利率 | 回本周期 | 难度 | 适合人群 |
|------|-------|---------|------|---------|
| 奶茶饮品 | 60-70% | 6-12月 | 高 | 有审美/懂营销 |
| 炸鸡汉堡 | 55-65% | 8-14月 | 中 | 标准化运营 |
| 麻辣烫/串串 | 60-70% | 6-12月 | 中 | 口味稳定 |
| 黄焖鸡米饭 | 55-65% | 8-12月 | 低 | 夫妻店 |
| 卤味熟食 | 60-70% | 6-10月 | 中 | 有特色配方 |
| 早餐包子 | 50-60% | 4-8月 | 低 | 勤劳肯干 |
| 煎饼果子 | 55-65% | 5-10月 | 低 | 有位置 |
| 麻辣香锅 | 55-65% | 8-14月 | 中 | 口味稳定 |

品类选择决策树：
- 有独特配方？→ 卤味/私房菜（高溢价）
- 勤劳肯吃苦？→ 早餐店（稳定但累）
- 追求标准化？→ 炸鸡/汉堡（可复制）
- 追求高毛利？→ 奶茶（但竞争激烈）
```

### 3. 菜单设计

```
菜单结构黄金公式：

【引流产品】（不赚钱，引流用）
- 特点：大众口味、成本低、视觉好
- 作用：吸引进店/下单，提升排名
- 定价：低于竞品10-20%

【利润产品】（主要收入）
- 特点：独特口味、成本可控、复购率高
- 作用：贡献70%利润
- 定价：成本×3-4

【形象产品】（提升档次）
- 特点：看起来很贵/很特别
- 作用：衬托其他产品性价比
- 定价：成本×5-8

菜单数量参考：
- 奶茶店：15-25个SKU（核心+季节限定）
- 快餐店：不超过30个SKU
- 小吃店：20-35个SKU
⚠️ SKU过多=备料复杂+损耗增加

定价方法：
外卖定价 = (房租+人工+平台扣点)÷预估单量 + 食材成本 + 预期利润
堂食定价 = 食材成本 × 3.5-4.5
```

### 4. 外卖平台运营

**美团/饿了么运营核心指标**：

```
平台排名权重（以美团为例）：
1. 营业额（权重最大）→ 冲够一定GMV
2. 转化率（进店+下单）→ 优化头图+菜单+价格
3. 评分（≥4.8最佳）→ 引导好评
4. 活动参与度 → 满减/神券/新客立减
5. 回复率 → 100%回复所有咨询

提升进店转化率（目标≥15%）：
- 门店头像：高颜值产品图，禁止用logo
- 店名：品类+特色（例：「老王炸鸡·外酥里嫩」）
- 起送价：≤20元（越低越好）
- 配送费：尽量免配送费

提升下单转化率（目标≥30%）：
- 头图：菜品摆盘+食欲色
- 菜单分类清晰
- 爆款前置
- 适当设置套餐

⚠️ 新店前30天是流量扶持期！
必须：
- 每天营业（不能关店）
- 满减活动拉满
- 引导好评（每单塞小纸条）
- 保持高评分（4.8+）
```

### 5. 私域获客体系

```
私域引流路径：

堂食客户 → 加微信 → 进社群 → 复购+转介绍

落地话术：
"加我们微信可以领5元优惠券，下次下单直接用 ~
以后还有新品试吃、专属折扣都在群里通知。"

社群运营SOP：

周一：限时秒杀（爆款5折）
周二：新品预告+投票
周三：用户好评截图分享
周四：抽奖互动（免单/优惠券）
周五：周末优惠预告
周末：发订单截图（营造火爆氛围）

朋友圈发圈（每天3条）：
- 早8点：早餐/今日特惠
- 中午11点：出餐实拍+文案
- 晚8点：用户好评+复购提醒

会员体系：
- 消费满100元升级银卡会员
- 银卡会员：9.5折 + 每月专属券
- 消费满500元升级金卡会员
- 金卡会员：9折 + 新品优先尝 + 生日礼包
```

### 6. 财务分析

```
小餐饮盈亏计算：

月固定成本：
- 房租：____元
- 人工：____元（夫妻店=0）
- 水电：____元
- 平台抽成：____元（预估月营业额×18-23%）

月变动成本：
- 食材成本：____元（预估月营业额×30-35%）
- 包装耗材：____元（预估月营业额×3-5%）
- 配送成本：____元（如有）

月盈亏平衡点：
月营业额 = 月固定成本 ÷ (1 - 变动成本比例)
= 固定成本 ÷ 毛利率

利润率健康指标：
- 外卖为主：≥15% 优秀，10-15% 合格，<10% 危险
- 堂食为主：≥25% 优秀，18-25% 合格，<18% 危险
- 纯外卖店：≥12% 优秀，8-12% 合格，<8% 危险

外卖vs堂食最优比例：
- 新店期：外卖70% + 堂食30%（快速起量）
- 稳定期：外卖50% + 堂食30% + 私域20%
```

### 7. 差评处理

```
差评类型及应对：

【口味不合】
"感谢您的反馈，很抱歉这次的口味不适合您。
您的意见我们已经收到，会在XX方面做调整。
为表歉意，赠您一张无门槛优惠券，下次使用 ~
希望下次有机会让您满意！"

【分量少】
"感谢反馈！真的很抱歉分量让您失望了。
我们已经反馈给后厨调整，也会加强出餐检查。
为您准备了5元补偿券，很抱歉给您带来不好的体验 🙏"

【超时/配送问题】
"感谢您的反馈，非常抱歉这次配送超时了。
我们已经和骑手/平台沟通过，会避免此类问题再次发生。
为您准备了XX元优惠券，希望下次能弥补这次的不好体验。"

【服务态度差】
"看到您的反馈我们非常重视，第一时间内部核实处理。
已经对相关人员进行了XX处理和培训。
诚挚道歉，已经为您准备了XX补偿。
希望您能给我们一次改正的机会 🙏"
```

## 使用方式

### 场景1：选址评估
用户说："帮我看看这个地方适不适合开奶茶店"
→ 执行选址评估模型，给出评分和建议

### 场景2：品类选择
用户说："我想开一个小餐饮店，不知道做什么品类好"
→ 根据用户预算、位置、经验给出推荐

### 场景3：外卖运营
用户说："我的外卖店评分掉到4.0了，怎么救"
→ 分析原因（口味/服务/包装），给出提升方案

### 场景4：私域搭建
用户说："怎么通过微信卖更多单"
→ 按引流→留存→转化→复购体系给方案

### 场景5：盈亏计算
用户说："房租6000，人工8000，能开什么店"
→ 计算盈亏平衡点，推荐适合的品类

## 注意事项

- 外卖平台扣点高（18-23%），纯外卖要算清楚能否盈利
- 小餐饮翻台率很重要，翻台率低=浪费位置
- 口味稳定比好吃更重要，不要经常换配方
- 私域复购是提升利润的关键，外卖平台流量越来越贵
