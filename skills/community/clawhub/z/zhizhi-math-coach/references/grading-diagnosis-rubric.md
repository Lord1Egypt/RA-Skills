# Grading And Diagnosis Rubric

## Accepted Inputs

Accept any of these:

- Full completed worksheet or photo.
- Teacher-graded paper.
- Parent-copied wrong questions.
- Direct text: `题目 / 孩子答案 / 正确答案` or `题目 / 孩子答案`.

If the correct answer is missing, solve the problem and mark it as inferred. If handwriting or photo evidence is unclear, mark the item `需确认`.

## Item Diagnosis Fields

For each wrong or uncertain item, record:

| Field | Meaning |
| --- | --- |
| `学期` | Semester bucket for review, such as 一年级下学期 |
| `学期时间段` | Date range used for that semester |
| `错题类型` | The task layer that failed |
| `可能原因` | Why the mistake likely happened |
| `历史状态` | Prior status for the same weak point, if known |
| `复发判断` | Relapse, transfer failure, forgetting, or one-off slip |
| `证据` | Written work or parent observation supporting the diagnosis |
| `置信度` | `高` / `中` / `低` |
| `补救动作` | Explanation or practice targeting the cause |

## Error Type Labels

Use concise labels. Multiple labels are allowed; put the primary label first.

- `阅读理解`: missed keywords, ignored conditions, confused wording, skipped units.
- `题型判断`: chose the wrong structure or formula.
- `概念理解`: does not understand the underlying meaning.
- `计算技能`: arithmetic error, counting error, borrowing/carrying error.
- `步骤遗漏`: missed conversion, comparison, intermediate quantity, or final step.
- `审题习惯`: copied numbers wrong, answered the wrong question, forgot unit.
- `表达书写`: answer sentence, unit, equation, or explanation incomplete.
- `粗心偶发`: evidence points to one isolated slip.
- `问法未识别`: solved a related quantity but not the final question.
- `条件误用`: used irrelevant information or ignored required information.
- `干扰条件干扰`: extra information pulled the child away from the model.
- `中间量缺失`: skipped the intermediate value needed for the final answer.
- `两步逻辑漏步`: wrote only one equation when two or more were needed.
- `式子对但计算错`: model was right, arithmetic caused the wrong answer.

## Likely Causes

Prefer causes that change the next practice plan:

- `关键词不敏感`
- `算式模型不稳`
- `逆向位置转换不稳`
- `进退位过程不熟`
- `数位意识不稳`
- `口算事实不熟`
- `检查习惯不足`
- `题目文字负担过高`
- `条件取舍不稳`
- `中间量意识不稳`
- `问法回看不足`

## Confidence Rules

- `高`: written work directly shows the cause, or the same pattern repeats.
- `中`: final answer strongly suggests the cause, but work is incomplete.
- `低`: only the final answer is known, or multiple causes are plausible.

For low confidence, generate 1 to 2 verification questions before changing a major weak-point status.

## Correction Output

For each high-priority cause:

1. Give a short parent-facing explanation.
2. Show the correct solving path.
3. Explain how to check the answer.
4. Provide 3 to 6 validation questions.
5. State what result means weak / consolidating / mastered.

Group repeated errors by cause and teach the cause once.

## Word Problem Layers

For word-problem mistakes, separate these layers before assigning practice:

1. `读题`: Did the student identify the final question and keywords?
2. `条件`: Did the student use required conditions and ignore distractors?
3. `建模`: Did the student choose the right operation or relation?
4. `步骤`: Did the student compute intermediate quantities first?
5. `计算`: Were facts, carrying, or borrowing correct?
6. `答句/检查`: Does the final sentence match the question and unit?
