## Description: <br>
Formats SQL queries with uppercase keywords, consistent indentation, aligned clauses, and preserved string literals for improved readability. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and data teams use this skill to reformat raw SQL for readability, team style consistency, and review prep without parsing, validating, or executing SQL. <br>

### Deployment Geography for Use: <br>
Global <br>
