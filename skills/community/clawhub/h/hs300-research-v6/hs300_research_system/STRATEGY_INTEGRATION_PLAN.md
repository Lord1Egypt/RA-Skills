### 2.2 共享因子计算库 (core_factors.py)

所有策略共用的底层因子处理能力：
- **MAD去极值** — 3倍MAD阈值
- **Z-Score标准化** — 传统/稳健两种
- **Rank标准化** — 百分位排名
- **市值/行业中性化** — 横截面回归残差法
- **对称正交化** — SVD分解消除共线性
- **IC/ICIR计算** — 滚动窗口因子有效性
- **ICIR加权** — 2025年实证最优方式
- **等权合成** — 简洁稳健

### 2.3 策略基类 (base_strategy.py)

统一策略接口：
```python
class BaseStrategy:
    def get_stock_pool(self, date) -> List[str]          # 获取股票池
    def calculate_factors(self, stocks, date) -> pd.DataFrame  # 因子计算
    def composite_score(self, factors_df) -> pd.Series   # 综合得分
    def build_portfolio(self, scores) -> pd.Series       # 组合构建
    def generate_report(self, portfolio) -> str          # 报告生成
```
