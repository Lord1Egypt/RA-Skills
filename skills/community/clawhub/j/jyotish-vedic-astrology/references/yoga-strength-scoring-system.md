# Yoga格局强度评分系统

## 一、评分系统概述

### 1.1 目的

量化Yoga格局的强度，避免主观判断，提高预测准确性。

### 1.2 评分维度

五大维度：
1. 参与行星强度（0-40分）
2. 宫位位置（0-25分）
3. 分盘确认（0-15分）
4. 大运时机（0-15分）
5. 取消因素（-20-0分）

总分范围：-20 - 100分

强度等级：
- 90-100分：极强（必然显现）
- 70-89分：强（很可能显现）
- 50-69分：中等（可能显现）
- 30-49分：弱（可能不显现）
- 0-29分：极弱（几乎不显现）
- < 0分：取消（不会显现）

> **来源标签**: 【现代演绎·Skill整合】 — Yoga强度评分系统（Skill原创量化框架）
