# 输卵管妊娠病因分析

## 触发条件
当患者满足以下任一情况时启动本分析：
- 有盆腔感染史（如淋病奈瑟球菌、沙眼衣原体感染）
- 既往输卵管手术史或输卵管妊娠史
- 正在接受或曾接受辅助生殖技术
- 使用宫内节育器或紧急避孕药后妊娠

## 执行步骤
1. **评估输卵管炎症类型**：
   - 若为**黏膜炎**，判断是否由淋病奈瑟球菌或沙眼衣原体引起，评估纤毛功能受损或管腔狭窄程度。
   - 若为**周围炎**，确认是否继发于流产或分娩后感染，检查是否存在输卵管扭曲、粘连或蠕动减弱。
2. **识别特殊感染**：排查结核分枝杆菌感染导致的结节性输卵管峡部炎，注意肌壁结节性增生和近端肌层肥厚表现。
3. **筛查其他高危因素**：
   - 既往输卵管妊娠次数（1次再发风险≈10%，≥2次>25%）
   - 输卵管发育异常（如过长、憩室、副伞、肌层发育差、纤毛缺乏）
   - 激素失调或精神因素引起的输卵管痉挛
   - 子宫肌瘤、卵巢肿瘤压迫或子宫内膜异位症累及输卵管
4. **输出临床警示**：综合上述因素，提示临床高度警惕异位妊娠可能，尤其在早孕评估中需结合超声和血hCG动态监测。

> 注意：本技能不适用于宫内妊娠或无输卵管患者。
