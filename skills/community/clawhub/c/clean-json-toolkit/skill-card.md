## Description: <br>
Local JSON / JSONL inspection and manipulation toolkit for inspecting nested structures, querying with a jq-style path language, flattening and unflattening data, merging files, applying JSON Patch operations, and validating pragmatic schemas with Python 3 standard-library scripts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agents use this skill to inspect, query, transform, merge, patch, and validate local JSON or JSONL files without installing third-party dependencies. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Merge, flatten, query output, and patch operations can create or overwrite local JSON or JSONL files. <br>
Mitigation: Run the tools only on files intended for inspection or modification, review output paths before execution, and prefer dry-run or stdout modes when checking changes. <br>
