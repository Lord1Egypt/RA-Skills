## Description: <br>
Converts HTML and Markdown for web clipping, clean notes, and batch content migration across local files, raw content, URLs, directories, and URL lists. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wangziiiiii](https://clawhub.ai/user/wangziiiiii) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, technical writers, and content migration teams use this skill to convert web pages, HTML files, Markdown files, and batches of documents into cleaner Markdown or standalone HTML. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Fetching user-provided URLs may expose or process private intranet, localhost, metadata-service, or credential-bearing pages. <br>
Mitigation: Use only URLs intended for conversion, avoid private or sensitive endpoints, and review generated Markdown before sharing it. <br>
