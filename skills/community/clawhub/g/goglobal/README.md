# 出海通 GoGlobal

**零基础搭建出海通道。粘贴一行信息，等几分钟，手机扫码就能用。**

不需要懂技术，不需要碰命令行，不需要装SSH工具。

---

## 安装

在 OpenClaw 里跟 AI 说一句话：

```
帮我安装出海通
```

或者用命令安装：

```bash
openclaw skills install clawhub:scientificinternet/GoGlobal
```

装完之后跟 AI 说"出海通开始"就启动流程了。

## 使用流程

整个过程大约 10 分钟。你只需要做三件事：打开链接、复制粘贴、跟着点几下。

```
第一步：买一台服务器（搬瓦工，月付$20）
第二步：登录搬瓦工，复制一行服务器信息给 AI
第三步：AI 自动帮你装好管理面板
第四步：在面板里点几下创建节点
第五步：手机装个 App，扫码连上
第六步：打开 google.com 验证成功
```

AI 会一步一步带你走，每步都等你确认才继续。中间断了随时回来说"继续"，从断的地方接着走。

## 常见问题

### 打不开 AI 给的链接？

1. 先确认登录了 bwh8l.net/login
2. 换个浏览器试试
3. 用手机流量试（别用WiFi）

### 安装面板时间太长？

正常需要2-3分钟。AI 会自动帮你检查进度。如果超过3分钟，AI 会查日志帮你定位问题。

### 面板打不开？

1. 确认地址末尾有 `/`（斜杠）
2. AI 会帮你检查面板是否在运行
3. 如果端口被挡了，AI 会帮你放行

### 连上了但速度慢？

跟 AI 说"速度慢"，它会帮你开启 BBR 加速。

### 突然连不上了？

跟 AI 说"连不上了"，它会帮你一步步排查：服务器状态、IP是否被封、客户端配置。

### 想加更多设备？

同一个节点可以在多个设备上用。在新设备上扫同一个二维码就行。

### 想分享给家人？

跟 AI 说"我想给家人也用"，它会教你在面板里给每个人建独立账号。

## 这个技能做了什么

出海通不教你网络知识，它帮你把事情做完：

- 通过搬瓦工 API 远程操作你的服务器（你不碰SSH）
- 自动安装 3x-ui 管理面板
- 引导你在面板里创建节点
- 引导你在手机/电脑上安装客户端
- 最后验证 Google、ChatGPT、Claude 是否真的能用

整个过程是状态机驱动的：AI 知道你当前在哪一步，出了问题只解决当前这一个，你中断了回来从断的地方继续。

## 安全说明

- 在你提供服务器控制Key之前，AI会明确告知风险（Key能做什么、链接不要截图分享、完成后重置Key），等你确认才继续
- 你提供的服务器信息（CSV）只包含你自己VPS的控制Key，不涉及支付、账户密码或其他服务
- 所有破坏性操作（重装系统）会提前警告
- 面板安装后会提醒你修改默认密码
- 安装来源是 GitHub 开源项目（MHSanaei/3x-ui）
- 面板默认走HTTP明文，完成后会提醒你可以升级HTTPS
- 完成后会引导你重置API Key，废掉对话中出现过的旧Key
- 首次生成含Key的链接时会提醒你不要截图分享
