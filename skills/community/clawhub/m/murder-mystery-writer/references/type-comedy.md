# 欢乐本创作指南

## 欢乐本的定义

欢乐本的核心体验是"快乐"——让玩家在轻松的氛围中享受社交乐趣。

## 欢乐本的类型

### 社交欢乐本

**特点**：以社交互动为主，推理为辅。

**设计要点**：
- 角色性格鲜明，容易扮演
- 有大量对话和互动机会
- 笑点来自角色之间的碰撞
- 推理难度低，重点在体验

### 机制欢乐本

**特点**：游戏机制驱动，有明确的胜负条件。

**设计要点**：
- 有特殊游戏机制（投票、交易、对抗）
- 胜负不完全取决于推理
- 有策略性和随机性
- 适合桌游玩家

### 谐趣推理本

**特点**：推理为主，但风格轻松幽默。

**设计要点**：
- 推理有难度，但氛围不沉重
- 角色设定有趣（如"倒霉侦探"、"话痨证人"）
- 笑点来自推理过程中的意外
- 适合喜欢推理但不想太累的玩家

## 欢乐本设计要点

### 笑点设计

**笑点类型**：

| 类型 | 示例 |
|------|------|
| 角色反差 | 凶手是个话痨，总是不小心说漏嘴 |
| 情境反差 | 严肃的审讯场景，但证人是个吃货 |
| 语言幽默 | 角色的台词自带笑点 |
| 肢体幽默 | 特定的表演要求 |
| 误会 | 角色之间的误解引发笑料 |

**设计原则**：
- 笑点要自然，不能强行搞笑
- 笑点不能伤害玩家（避免冒犯性笑点）
- 笑点要服务于剧情，不是为了搞笑而搞笑

### 节奏控制

欢乐本的节奏要快，不能冷场。

**节奏安排**：
```
开场破冰 → 第一波笑点 → 推进剧情 → 第二波笑点 → 高潮 → 轻松结局
```

### 角色设计

欢乐本的角色要"好玩"，容易扮演。

**设计原则**：
- 性格标签明显（话痨、胆小、自恋、吃货）
- 有独特的口头禅或行为习惯
- 角色之间的互动点多
- 不需要太深的情感投入

## 欢乐本结构模板

### 标准结构（2-3小时）

```
第一幕：破冰（20-30分钟）
  - 角色介绍，建立人设
  - 第一波互动笑点
  - 轻松的氛围

第二幕：欢乐推进（60-80分钟）
  - 剧情推进，笑点密集
  - 有小的推理环节
  - 角色互动升级

第三幕：欢乐高潮（30-40分钟）
  - 最大的笑点/反转
  - 轻松的投票环节
  - 欢乐结局
```

## 欢乐本示例：7人欢乐阵营本

### 《谁是卧底？不，谁是凶手！》（示例）

**类型**：欢乐阵营本
**人数**：7人（不限男女）
**时长**：3-4小时
**背景**：现代·综艺节目录制现场

**故事梗概**：
七个综艺咖参加一档真人秀节目，录制地点在一栋别墅里。节目进行到一半，导演被发现死在控制室。节目组决定继续录制，让嘉宾们自己找出凶手。在互相指控、爆料、甩锅的过程中，真相逐渐浮出水面。

**欢乐元素**：
- 每个角色都有搞笑的人设（如"戏精"、"社恐"、"杠精"）
- 有特殊的"爆料"机制，可以揭露其他人的黑历史
- 投票环节有"甩锅"和"互撕"的欢乐氛围
- 结局有反转，但不沉重

**阵营元素**：
- 有"真凶"和"好人"阵营
- 真凶需要隐瞒，好人需要找出真凶
- 有"搅屎棍"角色，任务是制造混乱
