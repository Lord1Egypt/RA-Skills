## 案例 1: 新消费美妆品牌从 0 到 1 全案

### 背景
- **品牌**: 某国货护肤品牌(虚拟，基于行业经验合成)
- **阶段**: 初创(成立 6 个月)
- **品类**: 抗老精华
- **预算**: 300 万(6 个月)
- **目标**: 月销从 0 做到 500 万，建立品牌认知

### 涉及 Agent

| 顺序 | Agent | 交付物 |
|:---:|------|------|
| 1 | strategy | 品牌定位报告 |
| 2 | brand | VI 系统 + Logo Brief |
| 3 | creative | 上市 Campaign Big Idea |
| 4 | media | 媒介计划 |
| 5 | kol | KOL 种草方案 |
| 6 | data | 投放效果复盘 |

### 关键产出节选

#### 1.1 品牌定位(strategy)
```markdown
- 定位陈述: "XX 为 28-38 岁都市精英女性提供科技护肤方案，
            用成分党的严谨对抗初老焦虑。"
- 核心价值: 科学/自信/从容
- 差异化: 同等成分 1/3 价格(对标雅诗兰黛小棕瓶)
- 方法论引用: STP + Brand Wheel
```

#### 1.2 Campaign Big Idea(creative)
```markdown
- Insight: "她们买的不是抗老，是'我还在努力'的自我确认"
- Big Idea: "成分透明化运动" — 把成分表印在包装最显眼处
- Headline: "看清楚每一滴，才知道值不值"
- 方法论引用: Insight 挖掘 + SCAMPER(反转) + AISAS
```

#### 1.3 KOL 投放(kol)
```markdown
- 金字塔: 3 头部 + 15 腰部 + 80 KOC + 500 UGC
- 平台: 小红书 60% + 抖音 30% + B站 10%
- 预算: 90 万(30% 总预算)
- 预估: 总曝光 5000 万+，等效 CPC 0.8 元
```

### 结果(6 个月后)
- 月销: 480 万(达成 96%)
- 品牌搜索指数: +180%
- 复购率: 22%
- ROAS: 3.2x

### 复盘要点
- ✅ 成功: 成分党定位精准，小红书 SEO 长尾效应明显
- ⚠️ 不足: 抖音信息流素材衰退快(7 天需轮换)
- 💡 改进: 下季度加大直播比重，提升复购

## 案例 2: 双 11 Campaign 媒介+KOL 整合

### 背景
- **品牌**: 中型服饰电商
- **阶段: 成熟期
- **预算**: 500 万(双 11 期间 1 个月)
- **目标**: GMV 同比 +50%，ROAS ≥ 4

### 涉及 Agent

| Agent | 交付物 |
|------|------|
| media | 双 11 媒介计划 |
| kol | 达人种草+直播 |
| data | 实时监控+优化 |

### 关键产出节选

#### 2.1 媒介计划(media)
```markdown
- 总预算分配:
  · 信息流(抖音+小红书): 45%(225 万)
  · 电商内投(直通车+引力魔方): 30%(150 万)
  · KOL 种草: 15%(75 万)
  · 直播投流: 10%(50 万)

- 节奏:
  · 预热期(D-15~D-7): 15% 预算，种草+加购
  · 爆发期(D-1~D-3): 60% 预算，放量+收割
  · 返场期(D+4~D+7): 25% 预算，长尾+复购

- KPI 预估:
  · 总曝光: 3 亿+
  · CTR: 2.5%(信息流)
  · CVR: 4.5%(电商内)
  · ROAS: 4.2x
```

#### 2.2 数据监控(data)
```markdown
- 实时看板:
  · 分渠道消耗/ROI
  · 素材效率(CTR 趋势)
  · 库存预警

- 优化触发器:
  · ROAS < 3x → 暂停低效素材
  · CTR 下降 >30% → 素材轮换
  · 库存 <20% → 切换主推品
```

### 结果
- GMV: 同比 +62%(超目标)
- ROAS: 4.8x
- 新客占比: 65%

### 复盘要点
- ✅ 成功: 预算节奏分配合理，爆发期集中放量
- ✅ KOL 种草+搜索协同效应明显
- ⚠️ 不足: 部分爆款断货，损失约 8% GMV

## 案例 3: 危机公关 P1 级响应

### 背景
- **品牌**: 连锁餐饮(50+ 门店)
- **事件**: 顾客投诉菜品异物，微博 3 小时转发过万
- **等级**: P1(严重)
- **响应窗口**: <4 小时

### 涉及 Agent

| Agent | 交付物 |
|------|------|
| crisis | 评估+声明+Q&A+修复方案 |

### 关键产出节选

#### 3.1 六维评估
```markdown
| 维度 | 得分(1-5) |
|------|:--------:|
| 传播广度 | 4 |
| 传播速度 | 5 |
| 情绪烈度 | 4 |
| 事实严重性 | 3 |
| 利益相关方 | 4 |
| 品牌资产风险 | 4 |
| **总分** | **24/30 (80%)** → **P1** |
```

#### 3.2 首份声明(4A 原则)
```markdown
## 关于 [日期] 某门店菜品异物事件的声明

我们已注意到社交媒体上关于 [门店] 菜品中出现异物的反馈。

**Acknowledge**: 我们对所有受影响的顾客表示诚挚的歉意。
**Apologize**: 对于此次事件造成的困扰，我们深感抱歉。
**Action**: 
  1. 已对涉事门店进行停业整顿
  2. 已邀请第三方检测机构介入
  3. 已成立专项调查组，48 小时内公布结果
**Assurance**: 我们将全面排查所有门店，加强食品安全管理。

[品牌名]
[日期]
```

#### 3.3 升级建议
```markdown
- ✅ 已启动: P1 级响应，4 小时内发声
- ⚠️ 建议引入: 法律顾问(评估赔偿责任)
- ⚠️ 建议引入: 食品安全第三方认证机构
- 📞 监管预沟通: 建议主动联系市场监管局
```

### 结果(30 天后)
- 负面声量下降 85%
- 门店客流恢复 92%
- NPS 恢复至危机前 80%
- 后续食品安全认证升级

### 复盘要点
- ✅ 成功: 4 小时内快速发声，态度诚恳
- ✅ 第三方介入增强公信力
- ⚠️ 教训: 事后建立门店监控预警机制

## 案例 4: SaaS 产品增长(ABM)

### 背景
- **品牌**: B2B SaaS 工具
- **阶段: 成长期(ARR 500 万)
- **目标**: MQL → SQL 转化率从 8% → 15%
- **预算**: 月 5 万

### 涉及 Agent

| Agent | 交付物 |
|------|------|
| strategy | 增长策略 |
| media | 精准获客方案 |
| data | 漏斗分析 |

### 关键产出节选

#### 4.1 增长策略
```markdown
- 核心挑战: 考虑问题(用户知道但不选我们)
- 差异化: "一站式替代 5 个工具，年省 30 万"
- 方法论: STP + 5A
- ICP: 100-500 人企业的市场/运营负责人
```

#### 4.2 精准获客
```markdown
- 渠道: 视频号(40%) + 公众号(30%) + 知乎(20%) + 行业社群(10%)
- 内容: 行业洞察 > 产品介绍
- 私域: 企微+社群运营 SOP
- CAC 目标: <2000 元/SQL
```

### 结果(3 个月后)
- MQL → SQL 转化率: 14.5%(接近目标)
- CAC: 1800 元
- LTV/CAC: 4.2:1

## 案例 5: 抖音直播从 0 到 1

### 背景
- **品牌**: 美妆品牌新号
- **阶段**: 冷启动
- **目标**: 月 GMV 50 万

### 涉及 Agent

| Agent | 交付物 |
|------|------|
| livestream | 完整直播运营方案 |

### 关键产出节选

#### 5.1 选品排品
```markdown
| 角色 | 数量 | 价格区间 | 占比 |
|------|:---:|:------:|:---:|
| 引流款 | 3 | ¥9.9-29.9 | 10% GMV |
| 爆款 | 8 | ¥59-129 | 50% GMV |
| 利润款 | 5 | ¥199-399 | 30% GMV |
| 搭配款 | 4 | ¥29-79 | 10% GMV |
```

#### 5.2 流量策略
```markdown
- 自然流(60%): 预热短视频 + 互动留人
- 付费流(40%): 千川 PC，日预算 2000 元，ROI 目标 3x
- 预估:
  · 平均在线: 200 人
  · GPM: 2500 元
  · 月 GMV: 54 万
```

### 结果(首月)
- 月 GMV: 47 万(达成 94%)
- GPM: 2300
- ROI: 2.8x
- 粉丝增长: 8000+

### 复盘要点
- ✅ 选品结构合理，引流款有效拉停留
- ⚠️ 主播话术需优化(逼单环节转化率偏低)
- 💡 下月重点: 加强主播培训 + 优化逼单话术
