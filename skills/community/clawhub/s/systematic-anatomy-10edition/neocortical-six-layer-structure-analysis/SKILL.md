# 新皮质六层结构功能解析

## 何时使用
- 解读大脑皮质组织切片
- 理解感觉/运动/联合皮质的功能基础
- 支持Brodmann分区判断
- 分析皮质发育异常或病变对分层的影响

## 执行步骤
1. **确认适用范围**：仅适用于发育成熟的人类新皮质；排除海马（原皮质）和嗅脑（旧皮质）。
2. **按深度顺序识别六层结构**：
   - **Ⅰ层**（分子层）：树突与平行轴突为主，少量水平/星形细胞，占皮质约10%。
   - **Ⅱ层**（外颗粒层）：密集小锥体细胞与颗粒细胞，髓纤维极少，染色浅。
   - **Ⅲ层**（外锥体细胞层）：含中小型锥体细胞，垂直纤维为主，实现皮质内联络。
   - **Ⅳ层**（内颗粒层）：密集小星形细胞，含致密横行传入纤维丛；在感觉皮质更厚。
   - **Ⅴ层**（内锥体细胞层）：含中大型锥体细胞，特大贝兹（Betz）细胞轴突构成锥体束。
   - **Ⅵ层**（多形细胞层）：含梭形、星形及马丁诺蒂细胞，发出投射与联络纤维。
3. **按功能三分法归类**：
   - **粒上层**（Ⅰ–Ⅲ层）：皮质内联络，发展最晚。
   - **内粒层**（Ⅳ层）：接收间脑特异性传入。
   - **粒下层**（Ⅴ、Ⅵ层）：向皮质下结构传出，控制躯体与内脏运动。
4. **结合区域差异**：不同功能区各层厚度与细胞密度不同，可辅助Brodmann 52区分区。

> 注意：皮质发育不良等病理状态可能破坏典型分层。
