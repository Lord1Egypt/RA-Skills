## 科普类推文

### 风格特点
- 清晰易读：重点突出信息传递效率
- 逻辑严谨：结构清晰，层次分明
- 专业可信：体现专业性和权威性
- 简洁大方：避免过度装饰

### 字体配置
```
标题：18-20px，加粗，微软雅黑或思源黑体
正文：14-15px，常规，微软雅黑
引用：13-14px，斜体，微软雅黑
注释：12px，灰色，微软雅黑
```

### 配色建议
```
主色：#2E7D32（深绿 - 专业感）
辅助色：#66BB6A（浅绿 - 活力感）
背景色：#FFFFFF（纯白）
强调色：#FF6F00（橙色 - 重点标注）
文字色：#333333（深灰）
次级文字：#666666（中灰）
```

### 间距设置
```
行间距：1.75-2.0
段间距：20-25px
字间距：0-2px
章节间距：40-50px
```

### 布局结构
- 标题使用加粗，居中或左对齐
- 正文段落清晰分段，每段不超过150字
- 重点内容使用引用框或底色块
- 图表放在相关段落下方
- 使用编号列表或项目符号列举要点
- 每个章节之间使用分隔线或空行

### 排版示例
```
【标题】AI如何改变我们的生活

【正文】人工智能（AI）正在深刻改变我们的生活方式...

【重点】AI技术的三大核心应用领域：
1. 智能助手
2. 自动驾驶
3. 医疗诊断

【引用】"AI不是要取代人类，而是要增强人类能力。"
```

## 情感类推文

### 风格特点
- 温暖共鸣：注重情感传递和共鸣
- 轻柔舒适：阅读体验流畅舒适
- 文艺清新：适当的文艺感和美感
- 亲切自然：拉近距离感

### 字体配置
```
标题：16-18px，加粗，微软雅黑或方正黑体
正文：14-15px，常规，微软雅黑
引用：13-14px，斜体，楷体或宋体
诗句/格言：14px，居中，楷体
```

### 配色建议
```
主色：#E91E63（粉红 - 温暖）
辅助色：#F48FB1（浅粉 - 柔和）
背景色：#FFF9F9（浅粉白）
强调色：#9C27B0（紫色 - 深情）
文字色：#333333（深灰）
次级文字：#757575（中灰）
```

### 间距设置
```
行间距：1.8-2.0
段间距：25-30px
字间距：1-3px
章节间距：50-60px
```

### 局局结构
- 标题可使用特殊样式（如下划线、边框）
- 正文段落较短，营造呼吸感
- 引用诗句或格言使用居中样式
- 图片使用圆角或边框装饰
- 使用表情符号适度增强情感表达
- 结尾可添加暖心语句或互动引导

### 排版示例
```
【标题】有些话，只想说给你听

【正文】夜深了，你是否也在想念某个人...

【引用】"相遇是一种缘分，相知是一种幸运。"

【诗句】
  此情可待成追忆
  只是当时已惘然

【结尾】愿你我都能遇见那个懂你的人。
```

## 营销类推文

### 风格特点
- 吸引眼球：强烈的视觉冲击力
- 重点突出：核心信息一目了然
- 引导转化：明确的行动引导
- 紧迫感：制造稀缺和紧迫感

### 字体配置
```
标题：18-20px，加粗，微软雅黑
副标题：16px，加粗，微软雅黑
正文：14-15px，常规，微软雅黑
价格/优惠：20-24px，加粗，红色
行动按钮：16-18px，加粗，白色
```

### 配色建议
```
主色：#FF5722（橙红 - 激情）
辅助色：#FF9800（橙色 - 活力）
背景色：#FFF3E0（浅橙）
强调色：#E91E63（粉红 - 吸引）
文字色：#333333（深灰）
价格色：#D32F2F（深红）
按钮色：#FF5722（橙红）
```

### 间距设置
```
行间距：1.6-1.8
段间距：15-20px
字间距：0-1px
重点间距：30-40px
```

### 局局结构
- 标题使用大字号和醒目颜色
- 价格、优惠信息使用大字号和红色
- 重点内容使用底色块或边框
- 优惠券、限时信息使用倒计时样式
- 行动按钮使用醒目的颜色和样式
- 使用箭头、图标等引导视线
- 产品图片使用高清大图

### 排版示例
```
【标题】🔥 限时特惠！原价999，现价199！

【副标题】仅限今日，错过再等一年！

【正文】品质保证，七天无理由退换...

【优惠】💰 省800元！
      ⏰ 仅剩23小时59分

【行动按钮】立即抢购 >

【信任】⭐⭐⭐⭐⭐ 10000+用户好评
```

## 资讯类推文

### 风格特点
- 专业严谨：体现权威性和可信度
- 结构清晰：信息组织有序
- 客观中立：避免过度情绪化
- 时效性强：突出时效性和重要性

### 字体配置
```
标题：18-20px，加粗，微软雅黑
副标题：16px，常规，微软雅黑
正文：14-15px，常规，微软雅黑
来源/时间：12px，灰色，微软雅黑
摘要：13-14px，斜体，微软雅黑
```

### 配色建议
```
主色：#1976D2（深蓝 - 专业）
辅助色：#42A5F5（浅蓝 - 清新）
背景色：#F5F5F5（浅灰）
强调色：#D32F2F（深红 - 紧急）
文字色：#333333（深灰）
来源色：#9E9E9E（浅灰）
链接色：#1976D2（深蓝）
```

### 间距设置
```
行间距：1.7-1.9
段间距：18-22px
字间距：0-1px
资讯间距：25-30px
```

### 局局结构
- 标题左对齐，可添加发布时间
- 正文段落清晰，信息密集但易读
- 重要信息使用加粗或引用
- 来源和发布时间使用灰色小字
- 使用编号或要点列举关键信息
- 相关资讯使用链接或底部推荐
- 可添加摘要或导语

### 排版示例
```
【标题】央行宣布下调存款准备金率0.5个百分点

【来源】新华社 | 2024-01-15 09:30

【摘要】为保持银行体系流动性合理充裕...

【正文】中国人民银行今日宣布，决定下调...

【要点】
• 下调幅度：0.5个百分点
• 生效时间：2024年1月25日
• 预期影响：释放长期资金约1万亿元

【相关】
→ 央行行长解读政策意图
→ 专家分析对经济的影响
```

## 通用排版原则

### 字体使用原则
- 优先使用系统默认字体（微软雅黑、思源黑体、苹方等）
- 标题字号不超过20px，避免过大
- 正文字号不低于14px，确保可读性
- 限制字体种类，建议不超过2-3种

### 配色使用原则
- 确保文字与背景对比度≥4.5:1（无障碍标准）
- 主色调控制在1-2种，避免色彩杂乱
- 重点色使用面积不超过10%
- 避免使用过于鲜艳或刺眼的颜色

### 间距使用原则
- 行间距建议1.5-2.0，提升可读性
- 段间距至少20px，明确段落划分
- 左右页边距15-20px，适配手机端
- 保持间距一致性，避免忽大忽小

### 布局使用原则
- 重点内容放在首屏或黄金位置
- 每屏信息量适中，避免过度拥挤
- 使用空行和分隔线进行视觉分割
- 图文比例建议1:3或1:4

### 交互设计原则
- 行动按钮使用醒目颜色和样式
- 提供明确的引导文案
- 重要信息重复出现2-3次
- 结尾添加互动引导（点赞、评论、分享）
