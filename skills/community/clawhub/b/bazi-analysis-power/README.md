# Bazi Analysis Skill — 八字命理分析引擎

基于标准化推理流程（SOP）与结构化规则的 AI 八字分析 Skill，使 AI 能够稳定、可重复地进行命理结构解析与运势推演。

## 核心目标

- 保证命理分析的 **逻辑一致性**，拒绝随意套用断语
- 通过 **逐步推理流程** 得出结论，每一步可追溯
- SOP 各步骤内置 reference 钩子，确保关键规则不遗漏
- 严格区分 **完整分析 / 单点问答 / 排盘确认 / 信息不足** 四种输出模式，不强行展开长篇

---

## 快速开始

### 激活方式

当你在 DeepSeek TUI 中发送符合以下条件的消息时，Skill 会自动触发：

- 提供出生日期（年月日时），并明确要求分析八字
- 提供已排好的四柱八字，要求解读命局
- 询问大运走势、流年吉凶、事业财运婚姻等具体问题
- 要求确认排盘结果

你也可以直接使用 `/bazi-analysis` 或 `$bazi-analysis` 显式调用。

### 输入信息要求

最精准的分析需要提供以下信息：

| 必需项 | 说明 |
| --- | --- |
| 出生年月日时 | 精确到时辰（如"早上8点"即可） |
| 性别 | 乾造（男）/ 坤造（女） |
| 推荐提供（提升精度） | 说明 |
| ---------------------- | ------ |
| 出生地 | 精确到市，狭长地形建议精确到区县（用于真太阳时校准） |
| 四柱八字 | 如果你已用专业软件排盘，直接提供干支 |
| 大运排盘 | 起运年份 + 各步大运干支 |
| 当前流年 | 你想要问的具体年份 |

> 若只提供出生时间而未提供排好的八字，Skill 会**自动调用**排盘 API 生成四柱与大运，但会提醒你 API 排盘未考虑真太阳时，建议用专业排盘软件交叉确认。

## 四种输出模式

Skill 会先判断你的意图，再选择对应的输出形式：

### 1. 完整分析模式

**触发条件**：要求"全面分析""详细看八字""系统解读命局""看大运流年整体走势"时。

输出结构：

1. 基础排盘确认
2. 原局结构分析（五行分布、旺衰、月令气势）
3. 气象分析（寒暖湿燥、调候判断）
4. 寻根分析（日主根气位置与力量）
5. 做功结构（核心作用关系、合冲刑害）
6. 大运分析（有利/不利阶段）
7. 流年应期（关键年份与事件触发点）
8. 综合判断（格局、优势、风险）
9. 建议与策略 + 小白也能看懂的总结

### 2. 单点问答模式

**触发条件**：只问一个具体问题，如"财运如何""适不适合创业""某年能不能换工作"。

输出结构：

1. 结论（直接回答）
2. 结构依据（只解释相关的原局/大运/流年依据）
3. 风险与前提
4. 可执行建议

### 3. 排盘确认模式

**触发条件**：只提供出生信息，尚未要求深入分析，或只想确认四柱与大运。

输出内容：基础排盘 + 起运信息 + 当前大运，并询问是否继续深入。

### 4. 信息不足模式

**触发条件**：缺少关键信息（如出生时分不清、未提供起运信息但要断流年）。

Skill 会先提出最少量的澄清问题，不强行补全或捏造。

## 使用示例

### 示例 1：完整分析

```
请按完整分析模式分析这个八字：
乾造 丁丑 癸丑 丙寅 甲午
起运：2005 乙酉年 12 月，第一步大运壬子，当前大运庚戌，当前流年丙午
```

### 示例 2：单点问答

```
帮我看看这个八字财运如何：
乙酉 戊寅 丙寅 己丑，乾造
当前庚子大运，流年丙午
```

### 示例 3：排盘确认

```
帮我排一下八字：1997年1月20日早上8点，男
```

### 示例 4：流年测算

```
帮我看看接下来三年哪年适合换工作
```

## Skill 内部推理流程（SOP）

以下为 AI 内部的完整推理顺序，用户无需关心，但可了解其严谨性：

1. **数据校验与排盘成像** — 校验信息完整性，定日主、排十神、确认空亡
2. **宫位与宾主关系** — 划定"家里"（日柱时柱）与"家外"（年柱月柱），界定资源归属
3. **物理属性与气象辨析** — 寒暖湿燥、燥湿土区别、四墓库专判、调候参考
4. **寻根溯源与权属判定** — 比劫法则、能量传递、印星归属、共根判定
5. **做功方式与效率** — 冲合刑穿、作用优先级（贪合忘冲、三合 > 六合 > 冲 > 刑害破）
6. **大运与流年引入** — 大运环境更替、流年吉凶应期、墓库引动
7. **综合判断与策略** — 事业/财运/健康建议与风险提醒

## 注意事项

- **不替代专业命理师**：本 Skill 为 AI 辅助分析工具，结论仅供参考。
- **真太阳时**：API 排盘未考虑出生地真太阳时，若出生地经度与北京时间差异较大，建议用专业软件排盘后提供干支。
- **信息越完整，分析越精准**：推荐同时提供八字干支、起运年份和大运序列。
- **直言不讳**：Skill 以"防患于未然"为原则，即使结果显示凶险或压力，也会如实提醒。
