# 配色合集

配色工具合集：调色板生成器、渐变库、品牌色卡、色盲友好与地域传统色。更多顶选见 [SKILL.md](../SKILL.md)。

## 调色板生成器

- [ColorSpace](https://mycolor.space/) — Generate palettes from a single color
- [Palette List](https://www.palettelist.com/) — Pick 2 colors, generate thousands of palettes
- [ColorBox](https://www.colorbox.io) — Free tool to produce color sets
- [UI Colors](https://uicolors.app/create) — Tailwind CSS color palette generator

## 精选调色板

- [Happy Hues](https://www.happyhues.co/) — Palette inspiration with real-world usage
- [Colors & Fonts](https://www.colorsandfonts.com/) — Curated colors, fonts and resources
- [Nippon Colors](http://nipponcolors.com/) — Japanese traditional colors

## 渐变

- [WebGradients](https://webgradients.com/) — 180 free linear gradients for backdrops
- [Gradient Hunt](https://gradienthunt.com/) — Thousands of gradient palettes
- [CSS Gradient](https://cssgradient.io/) — Custom CSS gradients with examples
- [UI Gradients](https://uigradients.com/) — UI gradients color generator

## 专用

- [Color Brewer 2](https://colorbrewer2.org/) — Palette generator with color-blindness support
- [Brand Colors](https://brandcolors.net/) — Official brand color codes collection
