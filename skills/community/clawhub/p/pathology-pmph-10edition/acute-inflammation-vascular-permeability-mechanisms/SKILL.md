# 急性炎症中的血管通透性增加机制

当组织发生急性炎症并出现富含蛋白质的液体外渗时，使用本技能识别主导的血管通透性增加机制，以指导病理分析或治疗干预。

## 执行步骤

1. **确认前提条件**：
   - 存在明确的致炎因子（如组胺、缓激肽、烧伤、化脓菌感染）
   - 处于急性炎症阶段（非慢性炎症）
   - 观察到液体和蛋白外渗现象

2. **根据致炎因子和病程阶段，判断可能激活的机制**：
   - **内皮细胞收缩**：若由组胺、缓激肽或白细胞三烯介导，主要影响毛细血管后小静脉，起效快（数秒至分钟），持续<30分钟。
   - **内皮细胞直接损伤**：若由严重物理/化学刺激（如烧伤）或强毒性病原体引起，表现为内皮坏死脱落，通透性显著且持久（数小时至数天）。
   - **穿胞作用增强**：若存在血管内皮生长因子（VEGF）上调，通过内皮细胞内囊泡通道转运大分子蛋白。
   - **新生毛细血管高通透性**：若处于炎症修复期，新生血管因连接不全及VEGF作用而持续高通透。

3. **综合判断主导机制**：
   - 多种机制可共存（如烧伤同时引发收缩、直接损伤及白细胞介导损伤）
   - 结合血管类型（如小静脉 vs 毛细血管）、渗出液性质（富含蛋白 vs 漏出液）及时间进程进行鉴别

4. **输出应用建议**：
   - 预测主要机制后，提出针对性干预策略（如抗组胺药用于内皮收缩，抗氧化剂用于白细胞介导损伤，抗VEGF用于穿胞或新生血管机制）

> 注意：本技能不适用于慢性炎症相关的血管重塑或流体静力学性水肿（如心衰所致漏出液）。
