# 書き直し品質自己確認リスト

納品前にすべての項目を確認します。

## カテゴリー1 — AIっぽさの除去確認

- [ ] 「最高の」「革新的な」「卓越した」などのAIっぽい形容詞が削除されていますか？
- [ ] 「お客様の大切な〜のために」類の過度な敬語表現がありませんか？
- [ ] すべての段落が均一な長さで構成されていませんか？
- [ ] 「まず、次に、最後に」式の機械的な列挙がありませんか？
- [ ] 「必ず」「すぐに」「絶対に」などの根拠のない断定表現がありませんか？

## カテゴリー2 — セールスポイントの保持確認

- [ ] 元のコピーの核心セールスポイントがすべて書き直しバージョンに含まれていますか？
- [ ] 核心的な強みが薄められたり弱くなっていませんか？
- [ ] 商品固有の差別化ポイントが明確に伝わりますか？
- [ ] ターゲット顧客の問題と商品の解決策の繋がりが明確ですか？

## カテゴリー3 — プラットフォーム適合性確認

- [ ] 該当プラットフォームの文体と形式に合っていますか？
- [ ] 文字数制限があるチャネルで制限内に収まっていますか？
- [ ] モバイルで読んだときに自然にスキャン/読めますか？
- [ ] CTAが該当プラットフォームで許可されている形式ですか？

## カテゴリー4 — 日本語の自然さ確認

- [ ] 実際の日本のECセラーや運営チームが書きそうな文体ですか？
- [ ] 翻訳調の表現（「〜を通じて」「〜に伴い」「〜において」）がありませんか？
- [ ] 不必要なカタカナ英語の混用がありませんか？
- [ ] 文章が自然に読めますか？
- [ ] 口語体チャネルで文語体が混ざっていませんか？

## カテゴリー5 — コンバージョン効率確認

- [ ] 最初の文がターゲット顧客の関心をすぐに引きますか？
- [ ] CTAが明確でひとつですか？
- [ ] CTAが強要的でなく行動を促していますか？
- [ ] 消費者が購買決定に必要な情報が十分に含まれていますか？

## カテゴリー6 — コンプライアンス確認

- [ ] 証明できない絶対的表現がありませんか？
- [ ] 医薬品・医療機器の効能を示唆する表現がありませんか？
- [ ] 競合他社を批判する表現がありませんか？
- [ ] 虚偽の在庫不足や偽りの緊急性の演出がありませんか？

## カテゴリー7 — 最終確認

- [ ] 出力形式がoutput-template.mdに合わせて構成されていますか？
- [ ] メインバージョン＋代替バージョンが提供されていますか？
- [ ] リスク項目がある場合、代替表現と理由が明示されていますか？

**すべての項目がチェックされたら納品準備完了です。**
