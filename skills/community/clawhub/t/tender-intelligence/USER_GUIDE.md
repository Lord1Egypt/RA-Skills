# 招标情报采集 — 用户操作手册

> 本手册面向安防工程商、系统集成商、信息化服务商，帮助您快速上手招标情报工具。

---

## 🚀 快速开始

### 安装
本工具为 OpenClaw Skill，安装命令：
```
openclaw skills add tender-intelligence
```

### 核心能力一览

| 场景 | 命令 | 说明 |
|------|------|------|
| 生成完整日报 | `node index.js --report` | X省安防市场完整日报 |
| 生成周报 | `node index.js --report 7` | 近7天X省安防周报 |
| 生成月报 | `node index.js --report 30` | 近30天X省安防月报 |
| 搜索招标 | `node index.js` | 输出昨日X省招标摘要 |
| 搜索特定地区/关键词 | `node index.js 北京 监控` | 全国搜北京监控项目 |
| 关键词管理 | `node index.js --keywords` | 查看当前关键词 |
| 数据健康度 | `node index.js --quality` | 评估昨日数据质量 |

## 🔍 手动搜索

### 全国省份+关键词搜索

```
node index.js 北京 监控 30
```
→ 搜北京监控项目，近30天，最多20条

```
node index.js 网络安全
```
→ 全国搜网络安全关键词，默认近7天

**支持省份**：全国34省均可（X省、青海、陕西、四川、广东、上海……）

### 数据健康度检查
```
node index.js --quality
```
→ 评估昨日数据采集是否正常，识别采集异常

### 自定义关键词
```
node index.js --set-keyword 智慧灯杆
```
→ 添加"智慧灯杆"到个人关键词列表

## ⚠️ 重要说明

### 数据时效
- **每日凌晨自动采集**，飞书推送时间约为 08:00-09:00
- 您看到的是**昨日全天数据**，非实时
- 周末数据量低属正常（政府/企业周末不发布公告）

### 数据保留
- 数据保留 **30天**（自动清理超过30天的历史数据）
- 超过购买窗口的历史数据价值低，清理后可提升检索性能

### 四大数据源
- **数据源A**：全国招标信息网站
- **数据源B**：全国招标信息网站
- **数据源C**：全国招标信息网站
- **数据源D**：当地省份政采网

### 金额说明
- 约50%项目金额可解析（甲方原文披露），其余显示"未知"
- 点击链接可查看原始招标公告

## ❓ 常见问题

**Q: 为什么昨天数据是0条？**
A: 可能是周末（政府/企业不在周末发布公告）。运行 `node index.js --quality` 检查数据健康度。

**Q: 支持哪些省份？**
A: 全国34个省级行政区，包括X省、陕西、四川、广东、上海、北京等。

**Q: 搜索结果为0？**
A: 试试：扩大天数（`近30天`）或更换关键词。也可以联系技术支持。

**Q: 如何获得全国数据？**
A: 当前版本以X省为核心深度覆盖，全国数据源正在持续扩展中。
