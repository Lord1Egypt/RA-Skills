# 吸入麻醉四期分期评估与临床干预

## 触发条件
当患者接受吸入性全身麻醉，且需要实时评估麻醉深度、防止麻醉过深或意外事件时，应用本技能。

## 执行步骤

1. **识别第一期（镇痛期）**
   - 起始：吸入麻醉药开始给药；终止：意识完全消失。
   - 特征：出现镇痛、健忘，因大脑皮质和网状结构上行激活系统受抑制。
   - 注意：此期不宜进行任何手术操作。

2. **识别第二期（兴奋期）**
   - 起始：意识消失；终止：进入外科麻醉期。
   - 特征：患者躁动、呼吸不规则、血压波动，为皮质下中枢脱抑制表现。
   - 风险：易发生喉痉挛、心搏骤停等严重并发症。
   - 注意：第一、二期合称“麻醉诱导期”，现代复合麻醉常快速越过此阶段。

3. **识别第三期（外科麻醉期）**
   - 起始标志：患者恢复安静，呼吸与血压趋于平稳。
   - 麻醉加深顺序：皮质下中枢（间脑→中脑→脑桥）自上而下抑制，脊髓由下而上抑制。
   - 分级管理：
     - 一般手术维持在第三期第1–2级；
     - 若进入第3–4级，出现明显呼吸抑制、发绀、血压下降，提示延髓受累，应立即减量或停药。

4. **识别第四期（延髓麻醉期）**
   - 特征：呼吸停止、血压剧降。
   - 紧急处理：立即停用吸入麻醉药，启动人工通气、胸外按压等高级生命支持措施。

## 输出与警报
- 实时判断患者所处麻醉期别。
- 若进入第三期末（第3–4级）或第四期，立即生成高优先级警报，并建议具体干预措施。

## 使用限制
- 仅适用于吸入性全身麻醉场景，不适用于单纯静脉麻醉。
- 现代复合麻醉虽常跳过第一、二期，但第三、四期的识别仍至关重要。
