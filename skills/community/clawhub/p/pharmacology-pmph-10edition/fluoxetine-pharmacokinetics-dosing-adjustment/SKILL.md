# 氟西汀药代动力学与个体化给药调整

## 适用条件
- 患者接受**口服氟西汀**（商品名：百忧解）治疗
- 已知患者的**肝功能和肾功能状态**
- **不适用于静脉给药或其他SSRI类药物**

## 执行步骤

1. **确认吸收与分布特性**：氟西汀口服吸收良好，血药浓度达峰时间（Tmax）为6～8小时；血浆蛋白结合率为80%～95%，提示游离药物比例低。

2. **评估消除半衰期**：母体药物半衰期为48～72小时；其活性代谢物去甲氟西汀半衰期更长，导致药效持续时间延长。

3. **识别代谢途径**：氟西汀主要经肝脏CYP2D6酶代谢生成去甲氟西汀，该代谢物具有相同药理活性。

4. **根据肝功能调整给药**：若患者肝功能受损，因代谢能力下降，应考虑采用**隔日给药**以避免药物蓄积中毒。

5. **根据肾功能调整给药**：若患者存在肾功能不全且需长期用药，应**减少剂量并延长给药间隔**，因部分代谢产物可能经肾排泄。

6. **输出给药建议**：综合肝肾功能状态，明确推荐剂量与给药频率。

> **注意**：本技能仅适用于氟西汀，不可直接外推至其他选择性5-羟色胺再摄取抑制剂（SSRIs）。
