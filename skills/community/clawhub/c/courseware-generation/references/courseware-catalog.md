## 课件生成清单与依赖拓扑

课件生成按使用流程组织的任务清单，附任务间关系和执行映射提示。

**域间逻辑流**：W0-需求分析域 → W1-设计规范域 → W2-内容生成域/W3-格式处理域 → W4-输出保存域

---

### W0-需求分析域

| ID | 任务类型 | 说明 | 依赖 | 执行映射 |
|----|---------|------|------|----------|
| W0-01 | 类型识别 | 识别用户需要的类型 | 无（入口） | S |
| W0-02 | 目标受众分析 | 分析目标受众 | W0-01 | C |
| W0-03 | 使用场景分析 | 分析使用场景 | W0-02 | C |
| W0-04 | 输出格式选择 | 根据需求选择输出格式 | W0-03 | C→A |

| W1-02 | 字体系统应用 | 应用标题/正文/辅助文字规范 | W1-01 | A |
| W1-03 | 布局规则应用 | 应用页面尺寸、页边距、间距规范 | W1-02 | A |
| W1-04 | 样式系统应用 | 应用预设样式 | W1-03 | A |

---

### 依赖拓扑摘要

以下为课件生成任务的主要依赖链路：

#### 一、域内链路

**需求分析链路**: W0-01 → W0-02 → W0-03 → W0-04
**设计规范链路**: W1-01 → W1-02 → W1-03 → W1-04
**内容生成链路**: W1-04 → W2-01/W2-02/W2-03/W2-04 → W3-01
**格式处理链路**: W3-01 → W3-02 → W3-03 → W3-04 → W4-01

#### 二、跨域协同链路

**完整生成链路**: W0-01 → W1-01 → W2-01 → W3-01 → W4-01

**格式选择链路**: W0-04 → W2-01
**规范应用链路**: W1-04 → W2-01
**输出保存链路**: W4-01/W4-02/W4-03

#### 三、设计规范依赖

**颜色系统**：W1-01 → 所有生成任务
**字体系统**：W1-02 → 所有生成任务
**布局规则**：W1-03 → 所有生成任务
**样式系统**：W1-04 → 所有生成任务
