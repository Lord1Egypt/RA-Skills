# Tapping Manipulation Protocol

## When to Use
- 患者患有风湿酸痛、局部感觉迟钝或肌肉痉挛
- 施术部位为肩背、腰臀或下肢部
- 患者皮肤完整，无破损
- 患者非年老体弱或心血管疾病患者

## Execution Steps
1. **准备姿势**：术者沉肩、垂肘，确保腕部完全放松。
2. **手法形成**：以前臂带动腕部甩动，使掌指关节微屈成虚掌，五指并拢。
3. **拍打动作**：
   - 动作平稳而有节奏
   - 拍打后迅速提起，避免滞留
   - 部位准确一致，不可偏移或滑动
4. **操作方式**：可单手操作，亦可双手交替或同时操作。
5. **类型区分**：
   - 使用手掌拍打称为“掌拍法”
   - 使用特制拍子称为“拍打法”
6. **力度调整**：根据患者耐受程度动态调整力度，防止损伤。

> 注意：全程需密切观察患者反应，如有不适立即停止。
