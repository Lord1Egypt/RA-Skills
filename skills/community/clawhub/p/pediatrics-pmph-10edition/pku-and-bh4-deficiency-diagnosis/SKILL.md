# 苯丙酮尿症（PKU）与BH4缺乏症的病因诊断

## 何时使用
- 患儿血苯丙氨酸（Phe）浓度 >120 μmol/L
- 伴有发育迟缓、癫痫、肌张力异常等神经系统症状
- 需鉴别经典PKU与BH4缺乏症以指导治疗

## 执行步骤
1. **确认苯丙氨酸代谢路径异常**：
   - 正常情况下，Phe在苯丙氨酸羟化酶（PAH）和辅酶四氢生物蝶呤（BH4）作用下转化为酪氨酸
   - PAH基因突变导致酶活性下降，Phe堆积并经旁路代谢生成苯丙酮酸等神经毒性物质

2. **评估BH4合成通路**：
   - BH4由GTP-CH、PTPS、DHPR等酶参与合成
   - 任一酶缺陷均可导致BH4缺乏，进而影响PAH、酪氨酸羟化酶及色氨酸羟化酶活性
   - 导致多巴胺、5-羟色胺等神经递质合成障碍

3. **结合流行病学判断**：
   - 中国人群中约85%~90%为经典PKU，10%~15%为BH4缺乏症
   - PTPS缺乏是BH4缺乏中最常见类型

4. **输出诊断建议**：
   - 若血Phe >120 μmol/L且排除BH4缺乏，诊断为PKU
   - 若伴神经递质合成障碍表现（如肌张力低下、吞咽困难、体温不稳），需考虑BH4缺乏症并行特异性检测

> 注意：本技能不适用于非遗传性高苯丙氨酸血症或继发性代谢紊乱。
