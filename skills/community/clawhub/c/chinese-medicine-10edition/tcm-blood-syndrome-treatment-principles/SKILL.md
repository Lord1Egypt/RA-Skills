# TCM Blood Syndrome Treatment Principles

## When to Use
当患者出现以下任一出血证候时启动本技能：
- 咯血、衄血、吐血、便血、尿血、崩漏、紫癜（非外伤性）
- 外伤出血（仅限止血药选型部分）

> **注意**：若为急性大出血伴休克，应先救脱固脱；若为恶性肿瘤相关出血，需结合现代医学处理，本技能不适用。

## Execution Steps

### 第一步：辨识出血病机与性质
1. **观察出血特征**：
   - 血色鲜红、口干便秘、舌红脉数 → 考虑 **热伤血络**（实火或虚火）
   - 血色淡红、神疲乏力、面色苍白、舌淡脉细 → 考虑 **气不摄血**
   - 血色紫暗、夹血块、刺痛拒按 → 考虑 **瘀血阻络**
2. **明确主导病机**：判断是“热”、“气虚”还是“瘀血”为主导因素。
3. **辨别虚实**：
   - 实证：由气火亢盛或瘀血内阻所致
   - 虚证：由阴虚火旺或脾气亏虚所致

### 第二步：确定治则方向
根据病机选择三大治则之一或组合：
- **治火**：
  - 实火 → 清热泻火（如龙胆泻肝汤、玉女煎）
  - 虚火 → 滋阴降火（如知柏地黄丸）
- **治气**：
  - 实证 → 清气降气（如黛蛤散合泻白散）
  - 虚证 → 补气益气（如归脾汤）
- **治血**：结合止血药类型（见第三步）

### 第三步：选择止血药类型
根据病机与出血阶段选择四类止血药之一：
- **凉血止血药**：用于血热妄行（如咳血鲜红、舌红脉数）
- **温经止血药**：用于气不摄血或阴虚火旺兼寒象者
- **化瘀止血药**：用于出血兼瘀滞（血紫暗、有块、刺痛）
- **收敛止血药**：用于外伤出血或慢性渗血、创面不敛

### 第四步：注意事项核查
执行前必须确认：
- 瘀血未清者禁用收敛止血药（防“留瘀”）
- 凉血止血药性寒，脾胃虚寒者慎用
- 温经止血药性温，热盛出血者禁用
- 初起出血忌大量寒凉止血药，以防瘀血内停
- “急则治标”：暴急量多出血应先止血，甚则益气救脱

## Output
输出包含：
1. 核心病机（热伤血络 / 气不摄血 / 瘀血阻络）
2. 虚实属性
3. 推荐治则方向（治火 / 治气 / 治血）
4. 止血药类型及代表药物建议
