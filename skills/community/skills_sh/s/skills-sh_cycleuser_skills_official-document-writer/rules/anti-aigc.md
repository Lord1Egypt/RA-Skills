# 公文写作反AIGC检测规则

公文是正式行政文书，有严格的格式规范（GB/T 9704-2012），但公文的AIGC问题和一般文本不同——不是太口语化而是太"模板化"。AI生成的公文读起来像填空题，虽然格式正确但内容空洞、缺乏针对性。

## 核心原则

公文反AIGC的要点不是写得口语化（公文本就该正式），而是写得有针对性——每一句话都传达具体信息，不做空洞的口号式表述。好的公文读起来是"领导的意思很明确"，AI公文读起来是"这段放在哪个通知里都行"。

## 公文AIGC高发模式

### 1. 空洞套话（权重35%）

```
❌ AI公文典型套路：
"进一步推动XX工作高质量发展"
"不断提升XX能力和水平"
"切实加强XX工作的组织领导"
"持续深化XX体制机制改革"
"着力解决XX领域的突出问题"

✅ 有针对性的公文措辞：
"6月底前完成全市527处地质灾害隐患点排查"（具体任务+数字）
"对排查发现的18处重大隐患，实行'一点一策'整治，5月底前完成3处"（量化目标）
"各区县防汛指挥部4月15日前完成组建，指挥长由区县长担任"（时限+责任）
```

### 2. 机械排列（权重25%）

```
❌ AI公文的机械结构：
"一、加强组织领导。各区县要..."
"二、强化责任落实。各部门要..."
"三、做好应急准备。各单位要..."
"四、严格督导检查。各级要..."

每段开头都是"加强/强化/做好/严格"，句式高度雷同。

✅ 自然的公文措辞：
"一、排查要到位。6月底前完成527处隐患点排查，发现重大隐患立即整改。"
"二、物资要备齐。防汛物资5月1日前到位，储备点从3个增至8个。"
"三、人要到岗。防汛队伍4月20日前完成组建和演练，关键时刻拉得出用得上。"
```

### 3. 万能模板式内容（权重20%）

```
❌ 哪个公文都能用的万能段落：
"要高度重视，精心组织，周密安排"
"要明确责任，落实到人"
"要加强督导，确保实效"
"要做好宣传，营造氛围"

✅ 有具体内容的工作部署：
"市教育局牵头，5月底前完成全市中小学防汛排查，排查结果6月5日前报市防汛办"
"每个区县储备编织袋5万条、铅丝10吨，5月1日前完成验收"
"应急队伍4月20日前完成组建，每支不少于30人，含1名医疗人员"
```

### 4. 过度使用"进一步""持续""深入"（权重20%）

```
❌ AI公文高频词：
进一步、持续、深入、全面、切实、着力、大力、积极推进、

出现频率超过3次/千字的必须删除替换。

✅ 替换方案：
"进一步推动" → "推动"或直接写具体行动
"持续深化" → 具体深化到什么程度
"深入开展" → 写明开展到什么范围
"切实加强" → 写明加强什么、怎么加强
"着力解决" → 写明解决什么、什么时候解决完
```

## 公文专业表达反AI策略

### 策略1：数据驱动

每个部署必有数字、时限、责任人。

```
AI: "加强防汛物资储备"
专业: "5月1日前完成防汛物资储备，其中编织袋50万条、铅丝200吨、救生衣5000件，由市应急局负责验收"
```

### 策略2：时限明确

```
AI: "尽快完成排查工作"
专业: "4月30日前完成全市地质灾害排查，5月15日前完成整改"
```

### 策略3：责任到人

```
AI: "各相关部门要密切配合"
专业: "市应急局牵头，市水利局、自然资源局配合，各区县政府具体实施"
```

### 策略4：措施具体

```
AI: "做好应急准备"
专业: "每个区县组织不少于3支应急队伍，每支30人以上，4月20日前完成1次实战演练"
```

### 策略5：打破段首句式对称

不要每段都以"加强/强化/做好/严格"开头。用不同句式表达不同层次的工作要求。

## AIGC率检测清单

### 内容检测
- [ ] 每个工作部署是否有时限？→无时限=AI特征
- [ ] 是否有具体数字（金额、人数、百分比）？→无数字=AI特征
- [ ] 是否有明确责任人？→笼统"各部门"=AI特征
- [ ] 删掉空洞套话后，剩余内容是否仍有实质信息？→无实质=AI特征

### 语言检测
- [ ] "进一步/持续/深入/切实"出现频率是否超过3次/千字？→超过=AI特征
- [ ] 是否存在万能模板式段落（换个主题就能用）？→存在=AI特征
- [ ] 段首句式是否雷同？→雷同=AI特征
- [ ] 是否有针对性措辞（本地区、本部门具体状况）？→无针对性=AI特征

### 格式检测（注意：格式合规≠高质量）
- [ ] 格式符合GB/T 9704-2012？→不符合需要修正，但符合≠不AI
- [ ] 层次序号是否正确（一、（一）、1.、（1））？→必须正确
- [ ] 字体字号是否规范？→必须规范
- [ ] 日期格式是否正确（2024年3月15日）？→必须正确

### AIGC率评分标准（公文专用）

- 0-15%：高质量公文，每句话有实质内容，数据具体时限明确
- 16-30%：合格公文，有少量空洞可优化
- 31-50%：需要大改，AI痕迹明显——像万能模板
- 51-70%：需重写，内容不可执行
- 71-100%：完全套话，换个地名就能发别处
