# 栓子运行途径与栓塞部位预测

## 何时使用
- 用户提供栓子类型（如血栓、脂肪滴等）、来源（如体静脉、左心腔等）、大小（μm）及是否存在心脏结构异常（如房间隔缺损）
- 需要推断栓塞可能发生的位置（如肺、脑、肾等）及潜在临床后果
- 分析猝死、器官梗死或反常栓塞等病理机制

## 执行步骤

1. **确定栓子来源并匹配主要运行路径**：
   - 若来源于体静脉系统或右心腔 → 预测为肺动脉主干或其分支栓塞（肺栓塞）
   - 若来源于主动脉系统或左心腔 → 预测为体循环动脉栓塞（常见于脑、脾、肾、四肢指趾小动脉）
   - 若来源于门静脉系统（如肠系膜静脉） → 预测为肝内门静脉分支栓塞

2. **评估特殊路径可能性**：
   - 若栓子直径 < 20μm 且具弹性（如脂肪栓子），考虑其可通过肺毛细血管进入体循环
   - 若存在房间隔缺损等心脏结构异常且右心压力升高，考虑交叉性（反常性）栓塞：右心栓子→左心→体循环
   - 若有胸腹压骤升病史（如剧烈咳嗽），极罕见情况下考虑逆行性栓塞（如下腔静脉血栓逆流至肝、肾静脉）

3. **综合判断栓塞后果**：
   - 结合栓子大小与数量（如骑跨性肺动脉栓塞可致猝死）
   - 考虑局部侧支循环是否丰富
   - 评估靶组织对缺血的耐受性（如脑组织敏感，骨骼肌耐受性强）

4. **输出结果**：
   - 明确预测栓塞部位（如“左大脑中动脉”、“右肺下叶动脉”）
   - 简述可能临床后果（如“大面积脑梗死”、“急性右心衰竭”）
