# 肝外胆管结石临床诊断与检查策略

## 适用条件
- 患者出现上腹绞痛、黄疸、发热等症状
- 怀疑肝外胆管结石或胆道梗阻
- 不适用于单纯胆囊结石或肝内胆管结石（除非合并肝外病变）

## 执行步骤

1. **识别典型临床表现**
   - **腹痛**：剑突下或右上腹阵发性绞痛，可放射至右肩背部，伴恶心呕吐
   - **寒战高热**：弛张热，体温39–40℃，提示胆管感染
   - **黄疸**：
     - 部分梗阻：轻度黄疸
     - 完全梗阻：进行性加深黄疸，伴陶土样便、深色尿、皮肤瘙痒
   - 判断是否出现Charcot三联征（腹痛+寒战高热+黄疸）

2. **体格检查评估**
   - 无急性发作：仅右上腹或剑突下深压痛
   - 合并胆管炎：右上腹腹膜炎体征，可能触及肿大胆囊伴触痛

3. **安排辅助检查**
   - **实验室检查**：
     - 血清总胆红素（Tbil）和直接胆红素（Dbil）升高
     - ALT、AST、ALP升高
     - 白细胞及中性粒细胞升高（提示感染）
     - 尿胆红素阳性，尿胆原减少或消失
   - **影像学检查（按优先级选择）**：
     - **超声**：首选初筛，评估胆管扩张及结石，但远端易受肠气干扰
     - **EUS（超声内镜）**：对远端胆总管结石敏感性高，不受肠气影响
     - **MRCP（磁共振胰胆管成像）**：无创，可明确梗阻部位，但小结石可能漏诊
     - **ERCP/PTC**：有创检查，用于诊断兼治疗，但有诱发胰腺炎、出血等风险

4. **初步鉴别诊断**
   - 排除右肾绞痛（伴血尿、肾区叩痛）
   - 排除肠梗阻（腹胀、停止排气排便）
   - 区分壶腹癌或胰头癌（无发热、进行性无痛性黄疸、可触及无痛性胆囊）

5. **输出决策建议**
   - 若确诊肝外胆管结石合并感染：启动抗感染、解痉等非手术治疗
   - 若存在完全梗阻或病情进展：安排ERCP取石或外科手术评估
