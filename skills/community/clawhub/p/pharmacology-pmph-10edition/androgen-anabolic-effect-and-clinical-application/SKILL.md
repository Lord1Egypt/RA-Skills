# androgen-anabolic-effect-and-clinical-application

## 触发条件
当患者确诊为睾丸功能不足、再生障碍性贫血、消耗性疾病或晚期乳腺癌，且排除前列腺癌、孕妇等禁忌证时，考虑使用雄激素类药物。

## 执行步骤
1. **确认适应症与禁忌**：
   - 适应症包括：替代疗法（睾丸功能不足）、功能性子宫出血、晚期乳腺癌、贫血、虚弱状态。
   - 禁忌：前列腺癌、孕妇；慎用于肝病、高血压、心衰患者。
2. **理解药理作用**：
   - **同化作用**：促进蛋白质合成、减少分解，形成正氮平衡，促进肌肉增长、体重增加。
   - **造血作用**：刺激促红细胞生成素分泌及骨髓造血。
   - **免疫调节**：增强免疫球蛋白合成与巨噬细胞功能。
   - **心血管调节**：降低胆固醇、舒张血管、抑制代谢综合征。
3. **选择给药方案**：
   - 睾酮口服易被肝破坏 → 多用肌注油溶液或皮下植入。
   - 酯类化合物吸收慢、作用久。
   - 甲睾酮可口服或舌下给药。
4. **监测不良反应**：
   - 女性：男性化（痤疮、多毛、声音变粗、闭经）。
   - 男性：部分女性化（因外周转化为雌激素）。
   - 17α-烷基取代物可致肝损、黄疸 → 发现即停药。

## 输出结果
改善体质、纠正贫血、控制出血或缓解乳腺癌症状；需持续监测肝功能、性征变化及心血管状态。
