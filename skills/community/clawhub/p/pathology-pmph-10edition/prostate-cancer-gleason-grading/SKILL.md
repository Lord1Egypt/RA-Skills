# 前列腺癌Gleason评分分级分组系统

## 适用条件
- 病理诊断已确认为**前列腺腺癌**
- 标本来源为**前列腺穿刺活检或根治性切除术**
- 显微镜下可识别至少两种占比≥5%的腺体生长模式
- **不适用于**神经内分泌癌等非腺癌类型

## 执行步骤
1. **识别主要与次要Gleason模式**：
   - 主要模式：占比最多的腺体结构（评分1–5）
   - 次要模式：占比第二多的腺体结构（评分1–5，且占比≥5%）
2. **计算Gleason总评分**：将主要模式评分与次要模式评分相加（范围2–10）
3. **映射至ISUP分级分组**：
   - 分组1：Gleason ≤6（如3+3）
   - 分组2：3+4=7
   - 分组3：4+3=7
   - 分组4：Gleason =8（如4+4, 3+5, 5+3）
   - 分组5：Gleason 9–10（如4+5, 5+4, 5+5）
4. **输出结果**：返回ISUP分级分组（1–5），用于临床风险评估

## 注意事项
- 必须由经验丰富的病理医师判读
- 需通过免疫组化（如p63或34βE12）确认基底细胞缺失以确诊腺癌
- 高级别特征包括单细胞浸润、融合腺、筛状腺等结构
