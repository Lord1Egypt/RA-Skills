# 中文小说写作质量门

交付任何章节草稿或实质修订前使用。质量门是实用检查，不是形式：P0 未处理时，不能把章节说成已完成，除非用户明确要求看粗稿。

## P0 - 交付前必须修

- 章节编号错误、正式章节编号重复，或误把 `旧稿/` 当作当前连续性。
- 与已确认主线事实、时间线、伤病状态、身份、持有物、承诺或秘密边界矛盾。
- 人物知道了自己尚未得知的信息。
- 明显 AI 腔或泛化套话，例如“命运的齿轮”“她终于明白”“说不清的情绪”“复杂的情绪”。
- 感悟式结尾：以中心思想、顿悟、总结性心理收束，而不是动作、物象、选择或未解钩子。
- 重复填充结构，尤其是 `没有……没有……也没有……`，除非用户明确要求这种节奏。
- 使用过期记忆：从已废弃章节、旧稿或被用户否定的偏好继续写。
- 触碰本书明确雷点，例如用户写入 `用户偏好.md`、`连续性罗盘.md` 或背景设定中的禁忌。

## P1 - 建议修

- 开篇钩子弱，或场景结尾没有形成压力。
- 场景没有变化：不推进关系、剧情、秘密压力、选择、承诺或伏笔。
- 情绪解释过多，本可由动作、物件、触碰、停顿或对白承担。
- 狠话空泛、故作深沉，没说清具体威胁、利益、代价或筹码。
- 同一情绪节点中堆叠相近意象，例如连续花、雪、灰、雾等同功能比喻。
- 对白把潜台词说得过直，尤其在古风权谋、拉扯、试探场景中。
- 设定像讲义一样倾倒，没有通过行动、冲突和后果显影。
- 五层次写作原则明显失手：摄像头式叙事、情绪单一、细节悬浮、句式平板、动词无力。
- 情感戏与剧情线脱节：只有暧昧、心疼或拉扯，没有信息、选择、风险或关系边界变化。
- 剧情推进没有人物压力：事件发生了，但称呼、站位、信任、欠债、退路或秘密边界都没变。
- 肢体接触只作装饰亲密，没有救治、试探、拒绝、控制、交换或危险感。
- 权谋/群像人物缺少自利逻辑，只按正邪标签或作者需要行动。
- 章末没有动作入口，只停在情绪或抽象判断，下一章无法自然接上。

## P2 - 可选润色

- 句子呼吸：短促冲击句与较密的感官/权谋句是否有张弛。
- 古风密度：优先用称谓、礼法、器物和权力关系建立古风，不靠生僻词堆砌。
- 对白压缩：删寒暄、铺垫和解释，把话锋留给试探、交换和选择。
- 感官准确性：触感、重量、温度、气味、光线、布料、纸张、金属、血、药、尘、雨只在改变情绪或行动时加入。
- 章节配比：情感、权谋、动作、悬疑或日常压力是否符合用户本章要求。

## 五角色评审

每个角色 0-100 分。最终分取最低角色分；有 P0 时无论分数如何都阻断交付。

| 角色 | 检查重点 |
|---|---|
| 阅读者 | 开篇吸引力、节奏、画面感、是否想继续读。 |
| 编审 | 错别字、标点、措辞准确、连续性、章节编号。 |
| 故事家 | 因果、剧情推进、伏笔、场景必要性。 |
| 文学顾问 | 声音、层次、细节、呼吸、活性。 |
| 毒舌读者 | AI味、水字数、套路、假深刻、读者雷点。 |

## 交付摘要

交付前后使用短摘要：

```markdown
## 交付前自检
- 编号/连续性：通过 / 已修正 / 阻塞
- P0：无 / 已修正 / 用户要求保留
- P1：无重大问题 / 已处理主要问题
- 风格命中：五层次命中点 + 本书偏好命中点
- 交付文件：正文/卷X/NN 第NN章：标题.md
```

如果仍有 P0，不要说“已完成”。说明阻塞点，并问用户需要怎样取舍。
