## 宣传物料生成清单与依赖拓扑

宣传物料生成按使用流程组织的任务清单，附任务间关系和执行映射提示。

**域间逻辑流**：M0-需求分析域 → M1-设计规范域 → M2-内容生成域/M3-格式处理域 → M4-输出保存域

---

| ID | 任务类型 | 说明 | 依赖 | 执行映射 |
|----|---------|------|------|----------|
| M0-01 | 类型识别 | 识别用户需要的类型 | 无（入口） | S |
| M0-02 | 目标受众分析 | 分析目标受众 | M0-01 | C |
| M0-03 | 使用场景分析 | 分析使用场景 | M0-02 | C |
| M0-04 | 输出格式选择 | 根据需求选择输出格式 | M0-03 | C→A |

### 依赖拓扑摘要

以下为宣传物料生成任务的主要依赖链路：

#### 一、域内链路

**需求分析链路**: M0-01 → M0-02 → M0-03 → M0-04
**设计规范链路**: M1-01 → M1-02 → M1-03 → M1-04
**内容生成链路**: M1-04 → M2-01/M2-02/M2-03/M2-04 → M3-01
**格式处理链路**: M3-01 → M3-02 → M3-03 → M3-04 → M4-01

#### 二、跨域协同链路

**完整生成链路**: M0-01 → M1-01 → M2-01 → M3-01 → M4-01

**格式选择链路**: M0-04 → M2-01
**规范应用链路**: M1-04 → M2-01
**输出保存链路**: M4-01/M4-02/M4-03

#### 三、设计规范依赖

**颜色系统**：M1-01 → 所有生成任务
**字体系统**：M1-02 → 所有生成任务
**布局规则**：M1-03 → 所有生成任务
**样式系统**：M1-04 → 所有生成任务
