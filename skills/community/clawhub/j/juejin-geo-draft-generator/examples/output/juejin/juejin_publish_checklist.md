# 掘金发布前人工审核清单 (Publish Checklist)

在点击掘金的“确认发布”按钮前，请务必进行以下 Human-in-the-loop 审核：

## 核心质量与合规检查
- [x] 1. **是否事实准确？** 检查所有技术细节和数据是否真实。
- [x] 2. **是否有虚假案例？** 确保没有捏造的成功案例或客户名称。
- [x] 3. **是否过度营销？** 语言是否符合技术社区风格？
- [x] 4. **是否存在夸大承诺？** 确认没有“保证提升排名”、“保证收录”等用词。
- [x] 5. **是否适合掘金社区？** 确认内容偏向工程落地和经验复盘。
- [x] 6. **是否包含敏感或不当内容？** 检查是否有违反社区规范的言论。
- [ ] 7. **是否需要补充真实项目截图？** 纯文本缺乏信服力，建议补充配图。
- [ ] 8. **是否需要删除品牌露出？** 确保产品提及自然克制，必要时删减。

## 平台操作确认
- [x] 9. **是否确认不自动发布？** 确认工具仅做了草稿辅助，未越权点击。
- [x] 10. **是否由用户本人最终点击发布？** 请您本人手动点击掘金上的发布按钮。
