## Description: <br>
Manages AI_README.md files so AI agents remember your project conventions across every session. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[draco-cheng](https://clawhub.ai/user/draco-cheng) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and coding agents use this skill to consult, initialize, validate, compress, and update AI_README.md convention files during code editing, creation, and review tasks. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can prompt an agent to initialize or update AI_README.md files in a repository. <br>
Mitigation: Review proposed AI_README changes before accepting them, especially in sensitive repositories. <br>
Risk: The skill depends on registering and running the ai-readme-mcp MCP package. <br>
Mitigation: Review the ai-readme-mcp package before registration and confirm the MCP configuration matches the intended version. <br>
