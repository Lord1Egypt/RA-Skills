# 先天性心脏病胚胎发生分析

## 何时使用
- 用户提及“法洛四联症”“室间隔缺损”“主动脉骑跨”等具体畸形
- 用户询问胎儿心脏发育第4–8周期间可能出现的结构异常
- 需要解释紫绀型先心病与胚胎分隔过程的关系
- 排除其他类型先心病（如大动脉转位）时需明确发育机制差异

## 执行步骤

### 1. 确认适用对象
仅适用于以下结构：
- 胚胎/胎儿心脏
- 心球、动脉干、心内膜垫、房间隔、室间隔

### 2. 判断发育阶段
若胚胎处于第4–8周，进入心脏分隔关键期，应用完整分隔流程；若聚焦第5–7周且存在主动脉肺动脉隔偏移，则重点分析法洛四联症成因。

### 3. 分析法洛四联症成因（当触发条件满足）
**触发条件**：主动脉肺动脉隔偏于肺动脉侧

执行以下分析：
- 确认主动脉肺动脉隔未按正常螺旋方式走行
- 推导四种并存畸形：
  1. 肺动脉狭窄（隔偏移致通道受限）
  2. 主动脉骑跨（主动脉根部跨过室缺）
  3. 室间隔缺损（心球嵴、肌部、心内膜垫融合失败）
  4. 右心室肥大（继发于肺动脉狭窄）
- 提示血流动力学后果：右心室血液经室缺入体循环，导致严重紫绀

### 4. 应用通用心脏分隔知识（当无特定畸形指征）
按顺序解析：
- 房室管分隔（心内膜垫融合）
- 心房分隔（第一/第二房间隔、卵圆孔形成）
- 心室分隔（肌部向上生长 + 膜部闭合）
- 心球与动脉干螺旋分隔（形成主动脉与肺动脉）
- 静脉窦重塑（左右角命运差异）

### 5. 输出结论
- 若符合法洛四联症机制：提示“存在法洛四联症风险，需心脏超声确认四联畸形”
- 若为正常分隔过程：说明“成功完成四腔分隔，胎儿期保留功能性通道（如卵圆孔）”

> **注意**：本技能不适用于出生后心脏或非哺乳类脊椎动物心脏发育分析。
