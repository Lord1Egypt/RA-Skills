# API 接口

## 基础 URL

```
http://localhost:5000
```

## 页面路由

| 路径 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 首页（登录/档案/建议/记录） |
| `/login` | POST | 登录/自动注册 |
| `/logout` | GET | 退出 |
| `/save_profile` | POST | 保存或更新患者档案 |
| `/add_log` | POST | 添加每日记录 |

## 核心 AI 决策引擎

### 输入参数

| 参数 | 来源 | 说明 |
|------|------|------|
| 年龄 | 档案 | 用于计算最大心率 (208-0.7×年龄) |
| EF | 档案 | 左室射血分数 (%) |
| 心脏病诊断 | 档案 | 心肌梗死后/心衰/不稳定心绞痛等 |
| 合并症 | 档案 | 高血压/糖尿病/心衰/慢阻肺 |
| 静息心率/血压 | 档案 | 晨间静息值 |
| 药物 | 档案 | 检测β阻滞剂、抗血小板、他汀 |
| 症状 | 每日记录 | 检测红色/橙色预警症状 |
| 运动时长 | 每日记录 | 用于安全评估 |

### 输出

| 输出 | 说明 |
|------|------|
| 风险等级 | 高/中/低 |
| 运动强度 | 对应 RPE 值 |
| 靶心率范围 | 基于年龄和β阻滞剂调整 |
| 运动处方 | 每周频率、时长、类型 |
| 药物提醒 | 抗血小板/他汀依从性 |
| 安全警告 | 紧急时阻断运动 |
