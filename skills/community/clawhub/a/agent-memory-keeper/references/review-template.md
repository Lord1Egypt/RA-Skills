# 对话复盘模板

## 复盘清单

### 1. 决策追踪
- [ ] 有没有新的业务/方向决策？
- [ ] 有没有技术选型变化？
- [ ] 有没有确认/取消的项目？

### 2. 信息提取
- [ ] 有没有新的术语/概念？
- [ ] 有没有值得建知识节点的新领域？
- [ ] 有没有用户偏好或习惯？

### 3. 反馈处理
- [ ] 有没有收到批评？ → 提炼为规则
- [ ] 有没有新要求？ → 更新待办
- [ ] 有没有方向调整？ → 更新计划

### 4. 知识裂变
- [ ] 有新知识能推导出什么？
- [ ] 跟已有知识有什么关系？
- [ ] 能生成什么新价值？

### 5. 记忆归档
- [ ] 写了当日日志？
- [ ] MEMORY.md 需要更新？
- [ ] 有没有需要清理的过期信息？

## 复盘输出格式

```markdown
# 对话复盘 YYYY-MM-DD

## 关键决策
- ...

## 新知识点
- ...

## 新规则
- ...

## 待办
- [ ] ...
```
