## Use Case: <br>
Developers and engineers use this skill to create, compile, test, prove, and verify Noir programs while selecting an appropriate proving backend. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated or modified Noir files and proof-related outputs could be incorrect for the intended circuit or proving backend. <br>
Mitigation: Review generated files and proof-related outputs, run `nargo test`, and confirm the proving backend before relying on results. <br>
Risk: Installing Barretenberg tooling from untrusted sources or using unpinned tool versions can reduce build reproducibility. <br>
Mitigation: Install `bbup` and backend tooling only from official sources and pin compatible tool versions for reproducible builds. <br>
