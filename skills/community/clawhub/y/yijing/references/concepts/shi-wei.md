# 时与位：时机与阶段的判断框架

> **30秒应用**：先判断大环境（时）：上升/平台/下行/混沌？再判断自己（位）：在六个阶段的哪一步？如果"位"需要你冲（比如二爻成长→三爻突破），但"时"是下行→调整节奏，别硬冲。

## 核心概念

易经最重要的两个维度是**"时"（时机）**和**"位"（位置/阶段）**。
同样的行动，在正确的时间、正确的阶段做，就是对的；在错误的时间、错误的阶段做，就是错的。

## 时（时机）

### 含义
"时"指整个大环境的周期——市场的周期、行业的周期、公司生命周期的阶段。

### 商业中的"时"

| 时 | 特征 | 适合的策略 |
|----|------|-----------|
| 上升期 | 需求增长、资本充裕、人才涌入 | 积极扩张、大胆创新 |
| 平台期 | 需求稳定、竞争格局清晰 | 优化效率、建立壁垒 |
| 下行期 | 需求萎缩、资本收紧、裁员潮 | 收缩防守、保留核心 |
| 混沌期 | 方向不明、旧规则打破 | 小步试错、灵活转向 |

### 核心问题
- 现在是这个行业的什么"时"？
- 这个"时"会在多久之后变化？
- 你能影响这个"时"吗？还是只能顺应？

## 位（位置/阶段）

### 六爻：任何商业进程的六个阶段

每个卦有六爻（六个横线），从下到上代表事物发展的六个阶段。在商业分析中：

```
上爻 (Stage 6) → 巅峰/过气：功成身退 OR 盛极而衰
五爻 (Stage 5) → 领导/主导：掌握全局，做正确决策
四爻 (Stage 4) → 平台/过渡：上有压力下有支撑
三爻 (Stage 3) → 转折/危险：进退两难，最危险的阶段
二爻 (Stage 2) → 成长/积累：开始有基础，但仍脆弱
初爻 (Stage 1) → 萌芽/起步：看不清方向，需要试探
```

### 各阶段的核心任务

| 阶段 | 核心任务 | 常见错误 |
|------|---------|---------|
| 初爻 | 试探、学习、找方向 | 操之过急，过早定方向 |
| 二爻 | 扎根、建立核心能力 | 分散精力，不够聚焦 |
| 三爻 | 突破瓶颈、做艰难选择 | 犹豫不决，进退失据 |
| 四爻 | 建平台、搭班子、规模化 | 脱离一线、官僚化 |
| 五爻 | 定战略、建文化、做生态 | 刚愎自用、脱离现实 |
| 上爻 | 功成身退/断臂转型 | 恋栈不去、不肯放手 |

## 时与位的匹配

这是易经分析的核心逻辑：**当前的"位"要求的策略，是否匹配当前的"时"？**

- 位在扩张（二爻→三爻），时在下行 → 需要调整节奏
- 位在收缩（四爻），时在上升 → 可能过于保守
- 位在三爻（转折），时在混沌 → 最需要灵活试错

## 在分析中的用法

1. 先判断"时"：行业/市场环境处于什么周期？
2. 再判断"位"：你在这件事上处于什么阶段？
3. 检查匹配度：你做的事是否适应这个时和位？
4. 预判下一步：按照六爻的演进逻辑，下一个阶段会是什么？
