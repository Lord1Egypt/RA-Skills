# 肾小球滤过屏障结构与功能分析

## 何时使用
- 用户分析蛋白尿的病理机制
- 评估肾小球疾病（如糖尿病肾病、膜性肾病）对滤过功能的影响
- 解释为何特定分子（如白蛋白）在正常情况下不被滤过
- 区分肾小球滤过障碍与肾小管重吸收异常

## 如何执行
1. **确认适用对象**：仅适用于肾小球、血管球毛细血管及尿液生成初始阶段；不适用于肾小管重吸收或其他非肾小球滤过结构。
2. **识别三层结构**：
   - 第一层：有孔毛细血管内皮（孔径70–90 nm），阻挡血细胞
   - 第二层：肾小球基膜（GBM），含Ⅳ型胶原和带负电蛋白聚糖，阻挡中大分子蛋白
   - 第三层：足细胞裂孔膜（含nephrin蛋白，厚4–6 nm），构成最终屏障
3. **评估选择性机制**：
   - **分子大小选择性**：分子量 <70 kDa 的物质（如葡萄糖）可自由滤过；>70 kDa 者通常被阻挡
   - **电荷选择性**：因各层带负电，带负电蛋白质（如白蛋白）被排斥，中性或正电分子更易通过
4. **关联病理变化**：若存在GBM增厚或足细胞损伤（如糖尿病肾病），则屏障完整性破坏，导致蛋白尿。
