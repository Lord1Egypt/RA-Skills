## Full feedback summary

### Quick wins

- Sharpen positioning: name the audiences (lawyers, contract managers, QS, claims consultants, PMs)
- Add concrete sample outputs for each tool
- Stronger legal disclaimer (educational, not legal advice; verify time-bars; local law overrides)
- Clear "Supported / Use caution" boundaries
- Make wizard the hero feature — surface it earlier
- Distinguish intelligence modes: template gen vs date math vs clause analysis vs comparison
- Keyword-rich description for discoverability
- Add "Best use cases" + "Bad use cases" sections

1. What this skill does

### Future product-quality ideas (bigger lift)
- **Matter intake mode** — guided flow (form, edition, governing law, project type, issue, dates, notices served, amendments, relief sought) → outputs issue summary, missing facts, deadline checklist
- **Amendment sensitivity** — checklist warning system for bespoke amendments that change notice periods, EOT entitlement, design responsibility, dispute steps
- **Confidence labels** on each output (High: dates/comparisons; Med: standard-form clause; Low: bespoke interpretation)
- **Document ingestion** — upload contract → detect form/edition → identify amendments → auto-build registers (biggest differentiator)

- Reduce deadline risk with notice calendars and SOP timelines
