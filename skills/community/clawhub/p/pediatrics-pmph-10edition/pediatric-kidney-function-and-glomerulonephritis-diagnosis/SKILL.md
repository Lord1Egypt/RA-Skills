# 儿童肾功能评估与肾小球疾病诊断

## 适用场景
- 需评估肾功能、调整药物剂量或液体管理
- 出现血尿、蛋白尿、水肿、高血压、少尿
- 怀疑电解质紊乱或溶血性贫血伴急性肾损伤

## 执行步骤
1. **参考GFR发育规律**：
   - 出生时GFR为成人1/4，2岁接近成人水平；
   - 指导输液速度及肾毒性药物使用。
2. **诊断急性链球菌感染后肾炎**（APSGN）：
   - 临床表现 + 前驱感染史 + C3降低 + ASO/DNase B升高。
3. **诊断巴特综合征**：
   - 低钾、低氯、代谢性碱中毒 + 高肾素、高醛固酮 + 血压正常。
4. **诊断溶血尿毒症综合征**（HUS）：
   - 微血管病性溶血性贫血 + 血小板减少 + 急性肾损伤；
   - 分型：典型（腹泻前驱）vs 非典型（无腹泻，补体异常）。
5. **泌尿道感染处理**：
   - 上尿路感染：静脉头孢曲松/噻肟，10–14天；
   - 下尿路感染：口服阿莫西林/克拉维酸钾，2–14天。

## 输出结果
根据临床与实验室数据，输出具体诊断（如APSGN、巴特综合征、HUS分型）及相应治疗建议。
