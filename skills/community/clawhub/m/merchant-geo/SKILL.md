# Merchant GEO - 中小商家GEO优化助手

帮助中小商家实现**零成本、高效率**的全平台品牌内容发布与GEO优化。

## 核心价值

- **免费**：不依赖付费推广，纯内容SEO/GEO优化
- **自动化**：上传资料 → AI生成 → 一键发布
- **全平台**：覆盖8大主流自媒体平台
- **合规**：基于真实资质（营业执照等）生成真实内容

## 工作流程

### Phase 1: 资料收集与整理

**必传资料（证明企业真实性）：**

| 资料类型 | 用途 | 说明 |
|----------|------|------|
| 营业执照 | 企业信息提取 | OCR识别统一社会信用代码、企业名称、经营范围等 |
| 门头照片 | 品牌视觉 | 用于各平台封面图和企业形象 |
| 法人/负责人照片 | 增强信任 | 展示真实面孔，提高转化 |
| 产品照片 | 内容素材 | 3-5张代表性产品图 |
| 其他资料（可选） | 补充背书 | 荣誉证书、客户合影、资质证明 |

**收集流程：**

1. 询问商家：「请提供以下资料（可拍照上传）：
   - 营业执照正本或副本照片
   - 店铺门头照片
   - 产品实拍照片（3-5张）
   - 负责人/法人照片（如有）
   - 其他资质证明（如有）」

2. 识别并整理资料，建立商家档案：
   - 企业名称、行业分类、地理位置
   - 核心产品/服务、差异化卖点
   - 客户群体画像

### Phase 1.5: 自动引导客户上传资料（重要！）

**必须按以下流程主动引导客户上传资料：**

**Step 1：发起引导**

当商家说"帮我宣传"、"帮我推广"、"写篇文章"等意图时，立即发送引导文案：

```
您好！请提供以下资料（可拍照上传），我来帮您生成8个平台的宣传文章：

📷 必传资料：
1. 店铺门头照 ⭐（最重要的！）
2. 产品实拍照片（3张以上）
3. 营业执照（如有）

📝 案例描述（选填）：
能说一个您服务的客户案例吗？比如：
- 客户是什么类型？（附近企业/个人/老顾客）
- 他们需要什么？（宴请用酒/送礼/自饮）
- 您怎么服务的？（推荐了哪些/价格如何）
- 客户反馈怎么样？

✅ 收到资料后，5分钟内生成全部文章！
```

**Step 2：资料检查清单**

| 资料等级 | 最低配置 | 能生成内容 |
|----------|----------|-----------|
| **最低配** | 门头照 + 一句话描述 | 基础企业介绍 |
| **标准配** | 门头照 + 产品照3张 | 企业介绍 + 产品展示 |
| **完整配** | 全部资料 + 案例描述 | 全方位GEO内容 |

**Step 3：催单话术**

如果客户没有回复或资料不全，使用以下话术：

| 场景 | 引导话术 |
|------|----------|
| **无回复** | "您好，差一张门头照就齐了，可以直接拍照上传" |
| **只发门头** | "收到门头照！再发3张产品图，我就可以开始写文章了" |
| **缺案例** | "资料齐了！能再说一个服务过的客户吗？（比如他们买了什么，感觉怎么样）" |
| **资料齐全** | "收到！我现在开始生成8个平台的宣传文章，预计5分钟完成✅" |

**Step 4：建立商家档案**

收到资料后，立即整理成结构化档案：

```markdown
## 商家资料档案

### 基础信息
- 企业名称：[从营业执照或门头提取]
- 行业分类：[烟酒/餐饮/装修/...]
- 地理位置：[城市+区域+街道]
- 联系电话：[客户提供的电话]
- 负责人：[姓名]

### 图片资料
- [x] 门头照片 ✅
- [x] 产品照片 3张 ✅
- [ ] 营业执照
- [ ] 老板照片

### 案例素材
案例1：[描述]
案例2：[描述]

### 差异化卖点
1. [卖点1]
2. [卖点2]
```

**Step 5：生成前的最终确认**

在开始生成文章前，向客户确认：

```
资料收集完成！📋

商家：[企业名称]
位置：[位置]
品类：[行业]

即将生成内容：
✅ 企业介绍文章（8个平台版本）
✅ 产品展示内容
✅ 客户案例（如有案例素材）

是否开始生成？回复"开始"即可。
```

### Phase 2: GEO内容智能生成

**GEO核心原则：**

- **关键词策略**：行业词 + 地域词 + 长尾需求词
- **内容质量**：真实、有价值、原创
- **合规表达**：基于真实资料，不夸大宣传

**生成内容类型：**

1. **企业介绍型**
   - 品牌故事（创始初心、发展历程）
   - 企业实力（规模、资质、团队）
   - 服务承诺（售后、保障、优势）

2. **客户案例型**
   - 案例背景（客户行业、规模、需求）
   - 解决方案（如何帮助客户）
   - 效果成果（具体数据、对比）

3. **产品展示型**
   - 产品亮点（核心功能、差异化）
   - 使用场景（解决什么问题）
   - 用户评价（真实反馈）

**生成提示词模板：**

```
请为[企业名称]生成一篇GEO优化文章，主题：[主题类型]

## 企业信息
- 行业：[行业分类]
- 位置：[地理位置]
- 核心产品/服务：[描述]
- 差异化卖点：[1-3点]

## 内容要求
- 字数：[平台适配字数]
- 风格：[专业/亲切/种草]
- 必须包含GEO关键词：[关键词列表]

- 形式：适合[平台]发布

## 发布平台
[平台名称]
```

**发布策略：**

所有平台统一使用**浏览器自动化**方式发布：

1. **准备工作**
   - 使用 `agent-browser` 浏览器自动化工具
   - 商家需提前在浏览器中登录各平台账号
   - 建议商家使用 Chrome/Edge 浏览器并保存密码

2. **发布流程**
   - Step 1: 启动浏览器 → 打开平台后台
   - Step 2: 打开内容编辑页面
   - Step 3: 自动填入标题、正文、标签
   - Step 4: 自动上传图片（如有封面图）
   - Step 5: 点击发布按钮
   - Step 6: 截图确认发布成功

3. **各平台后台地址**
   - 百家号：https://baijiahao.baidu.com/
   - 头条号：https://mp.toutiao.com/
   - 知乎：https://www.zhihu.com/creator/create-article
   - 小红书：https://creator.xiaohongshu.com/
   - 抖音：https://creator.douyin.com/
   - 搜狐号：https://mp.sohu.com/
   - 网易号：https://mp.163.com/
   - 快手：https://cp.kuaishou.com/

4. **浏览器自动化命令**
   ```bash
   # 启动浏览器
   agent-browser launch
   # 打开指定URL
   agent-browser open <url>
   # 获取页面内容（带元素ID）
   agent-browser snapshot -i
   # 输入文本
   agent-browser type "#selector" "文本内容"
   # 点击元素
   agent-browser click "#selector"
   # 截图
   agent-browser screenshot
   # 关闭浏览器
   agent-browser close
   ```

## 浏览器自动化发布

详见 `references/browser_automation_guide.md`

### 核心命令

| 命令 | 用途 |
|------|------|
| `agent-browser open <url>` | 打开网页 |
| `agent-browser snapshot -i` | 获取页面元素 |
| `agent-browser click "@e1"` | 点击元素 |
| `agent-browser fill "#input" "文本"` | 填写输入框 |
| `agent-browser type "[editor]" "内容"` | 输入正文 |
| `agent-browser upload "[input]" "文件路径"` | 上传文件 |
| `agent-browser screenshot` | 截图确认 |
| `agent-browser close` | 关闭浏览器 |

### 发布流程要点

1. **打开平台后台**
2. **snapshot -i** 获取编辑页面元素
3. **fill** 填入标题
4. **type** 填入正文（富文本编辑器）
5. **click** 添加标签/分类
6. **upload** 上传图片（如有）
7. **click** 点击发布按钮
8. **screenshot** 截图确认成功
