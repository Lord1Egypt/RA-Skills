# Deep Researcher — Citation & Reference Guide
## APA 7th / MLA 9th / Chicago Notes

---

## DEFAULT STYLE: APA 7th Edition

Use (Author, Year) for in-text citations. Include full reference in References section alphabetically by author.

---

## IN-TEXT CITATION FORMATS

### One author
```
(Gomez, 2025)
Gomez (2025) argued that...
```

### Two authors
```
(Smith & Chen, 2026)
Smith and Chen (2026) demonstrated...
```

### Three or more authors
```
(Patel et al., 2025)
Patel et al. (2025) found that...
```

### Multiple works same author, same year
```
(Brown, 2024a)
(Brown, 2024b)
```
List as "a", "b", "c" in References as well.

### Multiple works in one citation
```
(Harris, 2023; Kim, 2024; Nguyen, 2025)
```

### Direct quote (include page number)
```
(Jones, 2024, p. 47)
Jones (2024) stated, "the results were conclusive" (p. 47).
```

## REFERENCE LIST — APA 7th

### Journal Article
```
Author, A. A., & Author, B. B. (Year). Title of article. Journal Name, Volume(Issue), Page-Range. https://doi.org/xxxxx
```

### Book
```
Author, A. A. (Year). Title of book (Edition ed.). Publisher.
```

### Book Chapter
```
Author, A. A. (Year). Title of chapter. In B. B. Editor (Ed.), Title of book (pp. xx-xx). Publisher.
```

### Conference Paper
```
Author, A. A. (Year, Month). Title of paper. Conference Name, Location. https://url
```

### Technical/Research Report
```
Author, A. A. (Year). Title of report (Report No. XXX). Organization. https://url
```

### Web Page
```
Author, A. A. (Year, Month Day). Title of page. Website Name. https://url
```

### GitHub / Software
```
Author, A. A. (Year). Title of repository [Computer software]. GitHub. https://github.com/xxx
```

### arXiv Preprint
```
Author, A. A. (Year). Title of paper. arXiv. https://arxiv.org/abs/xxxxx
```

---

## REFERENCE LIST — MLA 9th

**In-text**: (Author Page) — always include page number

**Reference entry**:
```
Author Last, First. Title of Work. Publisher, Year.
```

**Journal article**:
```
Author Last, First, and Second Author. "Title of Article." Journal Name, vol. X, no. X, Year, pp. X-X.
```

---

## REFERENCE LIST — Chicago Notes/BIBLIOGRAPHY

**In-text**: Use footnote numbers¹

**Bibliography entry**:
```
Author Last, First. Title of Book. Place: Publisher, Year.
```

---

## COMMON ERRORS TO AVOID

| Error | Fix |
|-------|-----|
| Missing author initial formatting | Initials lowercase, surnames uppercase: `Smith, J. D.` |
| Missing DOI | Always include DOI or URL for online sources |
| Inconsistent capitalization | Book titles: Title Case. Journal articles: Sentence case |
| Missing access date | Add access date for web sources without publication date |
| Duplicate entries | Deduplicate — same work cited only once |
| Wrong ordering | Sort alphabetically A-Z by first author's surname |
