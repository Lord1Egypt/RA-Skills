## Description: <br>
Helps users discover and install agent skills when they ask questions like "how do I do X", "find a skill for X", "is there a skill that can...", or express interest in extending capabilities. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[yuanfa247](https://clawhub.ai/user/yuanfa247) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill to find task-specific skills in the open agent skills ecosystem, present relevant options, and provide install commands when the user chooses to proceed. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can steer broad help requests toward installing third-party skills globally with confirmation prompts skipped. <br>
Mitigation: Review the suggested skill and publisher, confirm the exact source before installation, avoid skipped confirmations where possible, and prefer reversible or non-global installation when available. <br>
