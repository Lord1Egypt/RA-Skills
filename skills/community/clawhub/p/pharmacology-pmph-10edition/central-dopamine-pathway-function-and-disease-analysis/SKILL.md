# central-dopamine-pathway-function-and-disease-analysis

## 适用场景
- 用户提及运动障碍（如动作迟缓、不自主运动）
- 用户描述精神症状（如幻觉、妄想、注意力缺陷、情绪失调）
- 需要根据症状推断潜在的多巴胺通路异常
- 需要推荐基于多巴胺机制的药物治疗方向

## 执行步骤

1. **识别涉及的多巴胺通路**：
   - 黑质-纹状体通路：主要调控运动功能
   - 中脑-边缘通路：主要调控情绪与奖赏行为
   - 中脑-皮质通路：主要调控认知、推理与高级思维

2. **判断通路功能状态**：
   - 功能减弱：表现为运动迟缓、动机缺乏等
   - 功能亢进：表现为不自主运动、幻觉、妄想或多动

3. **关联疾病与干预策略**：
   - 若黑质-纹状体通路功能减弱 → 考虑帕金森病 → 推荐补充多巴胺（如左旋多巴）或使用多巴胺受体激动剂（如普拉克索）
   - 若黑质-纹状体通路功能亢进 → 考虑亨廷顿病或多动症
   - 若中脑-边缘/皮质通路功能亢进 → 考虑精神分裂症阳性症状 → 推荐多巴胺受体阻断药（如氯丙嗪、氟哌啶醇、舒必利）

4. **排除不适用情况**：
   - 不适用于外周多巴胺系统相关问题（如胃肠道反应）
   - 不适用于非多巴胺介导的神经精神疾病（如抑郁症主要与5-HT相关）

> 注意：本技能仅用于初步机制分析与治疗方向建议，不能替代临床诊断。
