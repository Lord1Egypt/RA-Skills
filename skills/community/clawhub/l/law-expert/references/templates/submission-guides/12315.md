# 12315消费投诉提交全流程指引

## 适用场景
消费者为生活消费需要购买、使用商品或者接受服务，与经营者发生消费者权益争议，向市场监督管理部门投诉举报。

## 管辖机构
| 项目 | 内容 |
|------|------|
| 机构名称 | 国家市场监督管理总局 / 各地市场监督管理局 |
| 统一热线 | 12315（全国统一） |
| 官方网站 | www.12315.cn（全国12315平台） |
| 微信小程序 | 搜索"全国12315平台" |
| 工作时间 | 热线24小时受理；人工座席一般为工作日9:00-17:00 |

**管辖原则**：
- 被投诉商家所在地的市场监督管理局
- 平台内经营者：也可向平台所在地市监局投诉
- 网络交易：可向经营者住所地或第三方交易平台住所地市监局投诉

## 所需材料

| 序号 | 材料 | 份数 | 要求 | 模板 |
|------|------|------|------|------|
| 1 | 投诉人身份证明 | 1份 | 本人身份证正反面复印件；单位投诉提供营业执照复印件+法定代表人身份证明+授权委托书 | 无 |
| 2 | 投诉书/投诉登记表 | 1份 | 载明：投诉人姓名/电话/地址、被投诉人名称/地址、投诉请求、事实经过与理由、投诉日期 | 有（平台在线填写） |
| 3 | 消费凭证 | 1份 | 发票、收据、订单截图、支付记录等能证明消费关系存在的材料 | 无 |
| 4 | 证据材料 | 1份 | 合同/协议、聊天记录截图、广告宣传截图、商品照片/视频、鉴定报告等 | 无 |
| 5 | 被投诉人信息 | 1份 | 商家准确名称、地址（可从营业执照或平台商家信息页获取） | 无 |

## 提交流程

### 线上投诉（推荐）
1. 打开全国12315平台网站（www.12315.cn）或微信小程序搜索"全国12315平台"
2. 实名注册并登录
3. 点击"我要投诉"，阅读《投诉须知》并同意
4. 搜索/选择被投诉商家（输入企业名称或统一社会信用代码）
5. 填写投诉信息：商品/服务类别、品牌、金额、问题描述
6. 上传证据材料及身份证明
7. 提交投诉，获取案件编号
8. 在"我的投诉"中跟踪处理进度

### 电话投诉
1. 拨打12315热线
2. 向接线员说明投诉事项
3. 根据要求补充书面材料
4. 记录受理编号以备查询

### 现场投诉
1. 携带全部材料前往商家所在地市场监督管理局
2. 填写书面投诉登记表
3. 提交材料，领取受理回执

## 费用
- **完全免费**，不收取任何费用

## 时间预估
| 阶段 | 预计时间 |
|------|----------|
| 受理决定 | 自收到投诉之日起 **7个工作日** 内作出受理或不予受理决定 |
| 调解处理 | 受理后一般 **30-60日** 内完成调解 |
| 终止调解 | 调解不成的终止调解，告知可通过其他途径解决 |

## 注意事项
1. **管辖要准确**：投诉应提交至被投诉商家所在地的市监局，否则可能被移转或不予受理
2. **实名投诉**：线上平台需实名认证；匿名投诉受理优先级较低
3. **被投诉方信息完整**：商家名称必须准确，最好提供统一社会信用代码（可从营业执照获取）
4. **诉求明确**：退款、换货、赔偿等诉求需具体化，金额明确
5. **证据留存**：提交前自行备份所有材料；聊天记录建议录屏保存
6. **时效注意**：一般消费纠纷建议在知道权益受损后 **3年内** 提出
7. **不属于受理范围的情形**：
   - 经营者之间的购销纠纷
   - 已由其他行政机关或法院处理的
   - 购买后超过法定/约定期限的
   - 无法提供被投诉人准确信息的
8. **与法院诉讼并行**：12315调解不影响同时提起诉讼的权利，但调解达成协议后一般不再受理同一事项
