# 单幕完整示例

## 示例：《钟楼谜案》第一幕

以下是6人本格推理本《钟楼谜案》第一幕的完整写作示范，展示一个剧本杀的第一幕应该怎么写。

---

### 公共信息（所有玩家可见）

```markdown
# 《钟楼谜案》

## 故事背景

1937年上海，法租界。

钟楼酒店是上海最豪华的酒店之一，今晚举行一年一度的慈善晚宴。上海滩的名流们齐聚一堂，觥筹交错间，暗流涌动。

晚宴进行到一半，酒店老板林德昌被发现死在钟楼顶层书房。书房门从内反锁，窗户紧闭，形成密室。

所有嫌疑人都在晚宴现场，但每个人都有不可告人的秘密。

## 角色简介

1. **林景行**，男，28岁，林德昌之子。留洋归来，外表光鲜，实则在调查父亲的秘密。
2. **张佩珊**，女，25岁，林景行未婚妻。军阀之女，温柔外表下藏着冷酷。
3. **陈翰文**，男，35岁，林家法律顾问。出身寒门，忠于职责，知晓林家财务秘密。
4. **林月如**，女，22岁，林德昌之女。自幼嫉妒哥哥受宠，情绪不稳定。
5. **王福**，男，55岁，林家管家。服务林家30年，知晓所有秘密。
6. **顾晓曼**，女，30岁，记者。调查林家黑幕多年，今晚混入晚宴。

## 区域

- 一楼：大厅、厨房、花园
- 二楼：走廊、洗手间
- 三楼：书房（案发现场）、钟楼
```

---

### 角色剧本示例：林景行第一幕

```markdown
# 《钟楼谜案》
## 林景行 · 第一幕

### 你的故事

你是林德昌的独子，三年前被父亲送去英国留学。表面上是深造，实际是被流放——因为你发现了父亲公司的秘密。

在英国的三年，你一直在暗中调查。你发现林氏集团涉及大规模走私，而父亲就是幕后主使。你无法接受这个事实，决定回国当面质问父亲。

今晚的慈善晚宴，是你回国后第一次与父亲正面交锋的机会。

### 你的秘密

1. 你在调查父亲时，发现他与日本商人有秘密交易。你拿到了一份交易记录，但还没来得及看。
2. 你昨晚偷偷进入了父亲的书房，打开了保险柜。保险柜里有一封信和一份文件。你只看了信的开头——"如果你看到这封信，说明我已经死了"——就被门外的脚步声吓走了。你匆忙关上保险柜，但不确定是否锁好了。

### 你的时间线

- **18:30**：你到达酒店，在大堂与张佩珊聊天。她今晚看起来心事重重。
- **19:00**：晚宴开始，你在大厅敬酒。你看到父亲在主桌，脸色不太好。
- **19:15**：你去洗手间，在走廊遇到陈律师。他从书房方向走来，表情严肃。你问他去做什么，他说"处理一些文件"。
- **19:30**：你决定去找父亲摊牌。你来到书房门口，发现门锁着。你敲门，没人应。你贴耳听，隐约听到里面有声音，但不确定是什么。
- **19:45**：你放弃敲门，回到大厅。钟楼敲响了报时钟声。
- **20:00**：管家王福宣布发现你父亲死亡。

### 你的任务

1. **主线**：找出杀死父亲的凶手
2. **支线**：隐藏你昨晚进入书房的事（如果被发现，你会被怀疑）
3. **支线**：保护张佩珊（你怀疑她可能被卷入）
4. **支线**：找到你昨晚没看完的那封信

### 你知道的线索

- 父亲书房的保险柜密码是你的生日（0115）
- 你看到陈律师在19:15从书房方向走来
- 张小姐最近一直想退婚，但父亲不同意
- 管家王福服务林家30年，知道很多秘密
- 顾晓曼是今晚唯一你不认识的人

### 你可以问其他人

- 问陈律师："你19:15在书房做什么？"
- 问张佩珊："你今晚为什么心事重重？"
- 问王福："父亲最近有没有异常？"
- 问顾晓曼："你是谁？怎么混进来的？"
```

### 角色剧本示例：凶手（陈翰文）第一幕

```markdown
# 《钟楼谜案》
## 陈翰文 · 第一幕

### 你的故事

你是林家的法律顾问，出身寒门，靠勤奋考入法政学堂，后被林德昌一手提拔。你对林家忠心耿耿，因为你欠林老爷一条命。

十年前，你母亲重病无钱医治，是林老爷出钱救了她。从那时起，你就发誓效忠林家。

但今晚，你做了一件背叛林家的事。

### 你的秘密（重要：你是凶手）

1. 你在19:10杀死了林德昌。
   - 你用书房里的茶壶，在茶里下了毒（氰化物）。
   - 你看着他喝下毒茶，在19:10左右死亡。
   - 你利用钟楼的齿轮机关，制造了延时密室：
     - 你在门锁上安装了一个小机关，连接到钟楼的齿轮
     - 机关设定在19:30触发，自动锁上门
     - 你在19:15离开书房，此时门还没锁
     - 机关在19:30自动触发，门从内锁上
     - 机关在触发后自动脱落，无法被发现
   - 你离开时，在走廊遇到了林景行。你说"处理一些文件"。

2. 你杀林德昌的原因：
   - 林德昌发现你知道了走私的秘密，威胁要杀你灭口
   - 你母亲还在林家的控制下，你别无选择
   - 你是被迫杀人以自保

### 你的时间线（对外宣称版本）

- **18:30**：你在大厅准备晚宴文件
- **19:00**：晚宴开始，你在主桌附近
- **19:15**：你去书房取文件（在走廊遇到林景行）
- **19:30**：你回到大厅
- **19:45**：你在大厅，和几位客人聊天
- **20:00**：管家宣布发现林老爷死亡

### 你的时间线（真实版本）

- **19:05**：你借口去书房取文件
- **19:08**：你进入书房，林老爷正在喝茶
- **19:09**：你在茶里下毒
- **19:10**：林老爷喝下毒茶，开始不适
- **19:12**：林老爷死亡
- **19:13**：你安装机关
- **19:15**：你离开书房，在走廊遇到林景行
- **19:30**：机关触发，书房门自动锁上

### 你的任务

1. **主线**：隐藏你是凶手的事实
2. **支线**：引导怀疑到其他人身上（张佩珊或顾晓曼是好的目标）
3. **支线**：确保你母亲的安全
4. **支线**：找到那份走私交易记录（如果被发现，你的动机就暴露了）

### 你的自辩策略

如果被问到19:15在书房做什么：
- "我去取一份法律文件，林老爷让我拿的。"
- "我在书房待了大约5分钟，拿完文件就走了。"
- "我离开时林老爷还活着，他正在喝茶。"

如果被问到动机：
- "我和林老爷关系很好，他提拔了我，我没有理由杀他。"
- "我欠林家一条命，怎么会杀他？"

### 你知道的线索

- 林老爷最近在清理"知道太多的人"
- 张佩珊的父亲和林老爷有生意纠纷
- 顾晓曼的真实身份可能是记者
- 林景行昨晚偷偷回国，没有告诉任何人
```

### 线索卡示例

```markdown
# 线索卡 - L001

**类型**：实物线索
**地点**：书房 - 死者手中
**是否强制公开**：是

**内容**：
死者手中紧握着一块瑞士手表，表盘碎裂，时间停在19:47。

**指向**：死亡时间在19:47左右

---

# 线索卡 - L002

**类型**：实物线索
**地点**：书房 - 保险柜
**是否强制公开**：否（可以选择不公开）

**内容**：
保险柜被打开，里面有：
- 一封信，开头写着"如果你看到这封信，说明我已经死了"
- 一份文件，封面写着"林氏集团·日本分部·绝密"

**指向**：死者生前有预感，留下了遗言
```

---

### 这个示例展示了什么

1. **公共信息**：所有玩家都需要知道的基础信息
2. **角色剧本**：每个角色有独立的故事、秘密、时间线、任务
3. **凶手剧本**：有完整的真实时间线和自辩策略
4. **线索卡**：有明确的类型、地点、内容、指向
5. **信息分配**：每个角色掌握不同的信息，需要交流才能还原真相
