# 飞书推送配置说明

## 方式一：使用飞书机器人Webhook（推荐）

### 步骤

1. **创建飞书机器人**
   - 在飞书群聊中，点击群设置 → 群机器人 → 添加机器人
   - 选择"自定义机器人"
   - 设置机器人名称和头像
   - 复制生成的Webhook URL

2. **配置Webhook URL**
   ```bash
   # 方式1：设置环境变量
   export FEISHU_WEBHOOK_URL="https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxxxxxxxxxxxxxxxxx"

   # 方式2：在代码中直接设置
   from scripts.feishu_notifier import FeishuNotifier
   notifier = FeishuNotifier(webhook_url="https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxxxxxxxxxxxxxxxxx")
   ```

3. **测试发送**
   ```python
   from scripts.feishu_notifier import FeishuNotifier
   notifier = FeishuNotifier()
   notifier.send_text_message("测试消息")
   ```

### 优缺点

**优点**：
- 配置简单，只需Webhook URL
- 不需要申请飞书开放平台应用
- 适合单群聊推送

**缺点**：
- 仅支持文本和Markdown消息
- 不支持发送文件（PDF等）
- 每分钟发送次数有限制

## 方式二：使用飞书开放平台API（高级）

### 步骤

1. **创建飞书应用**
   - 访问 https://open.feishu.cn/
   - 创建企业自建应用
   - 获取App ID和App Secret
   - 配置应用权限：
     - 获取群聊信息
     - 发送消息
     - 上传文件

2. **配置应用凭证**
   ```bash
   export FEISHU_APP_ID="cli_xxxxxxxxxxxxxxxx"
   export FEISHU_APP_SECRET="xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
   ```

3. **获取群聊ID**
   ```python
   from scripts.feishu_notifier import FeishuOpenAPI

   api = FeishuOpenAPI()
   api.get_access_token()

   # 获取群聊列表（需要先添加机器人到群聊）
   # 或者直接使用已知的群聊ID
   chat_id = "oc_xxxxxxxxxxxxxxxx"
   ```

4. **发送文件消息**
   ```python
   from scripts.feishu_notifier import FeishuOpenAPI

   api = FeishuOpenAPI()

   # 上传文件
   file_key = api.upload_file("./招投标数据/daily/招投标信息日报_2026-04-12.pdf")

   # 发送文件消息
   api.send_file_message(chat_id, file_key)
   ```

### 优缺点

**优点**：
- 支持发送文件（PDF等）
- 支持更多消息类型
- 发送次数限制更宽松

**缺点**：
- 配置复杂，需要申请应用和权限
- 需要获取群聊ID
- 需要定期刷新访问令牌

## 使用示例

### 示例1：每日自动推送

```python
from scripts.scheduler import TaskScheduler

# 创建调度器
scheduler = TaskScheduler()

# 添加每日9:00定时任务
scheduler.add_daily_job(cron_expression="0 9 * * *")

# 启动调度器
scheduler.start()
```

### 示例2：手动推送

```python
from scripts.pdf_generator import PDFGenerator
from scripts.feishu_notifier import FeishuNotifier
from scripts.crawler import DatabaseManager
from datetime import datetime

# 生成PDF
generator = PDFGenerator()
today = datetime.now().strftime('%Y-%m-%d')
pdf_file = generator.generate_daily_report(today)

# 发送通知
notifier = FeishuNotifier()
db = DatabaseManager()

# 查询数据
projects = db.get_new_projects(today)
notifier.send_daily_report(today, projects, pdf_file)
```

### 示例3：命令行推送

```bash
# 方式1：立即执行一次
python scripts/scheduler.py --mode once

# 方式2：定时执行（每天9:00）
python scripts/scheduler.py --mode schedule --cron "0 9 * * *"

# 方式3：自定义时间（每天18:00）
python scripts/scheduler.py --mode schedule --cron "0 18 * * *"
```

## 安全建议

1. **保护凭证**
   - 不要将Webhook URL、App ID、App Secret提交到代码仓库
   - 使用环境变量或配置文件存储敏感信息
   - 定期更换Webhook和App Secret

2. **权限控制**
   - 仅授予应用必要的权限
   - 限制机器人可以操作的群聊
   - 定期审计应用权限使用情况

3. **错误处理**
   - 添加重试机制
   - 记录详细的错误日志
   - 设置告警阈值

## 常见问题

### Q1: Webhook URL失效怎么办？
**A**: Webhook URL有时效性，如果失效需要重新生成。

### Q2: 飞书机器人消息发送失败？
**A**: 检查以下几点：
- Webhook URL是否正确
- 网络连接是否正常
- 消息格式是否符合要求
- 是否超过发送频率限制

### Q3: 如何发送PDF文件？
**A**: 需要使用飞书开放平台API（方式二），Webhook不支持发送文件。

### Q4: 如何获取群聊ID？
**A**:
1. 将机器人添加到群聊
2. 调用API获取群聊列表
3. 从列表中找到目标群聊ID

### Q5: 定时任务不执行？
**A**: 检查以下几点：
- Cron表达式是否正确
- 系统时间是否准确
- 进程是否正常运行
- 日志中是否有错误信息
