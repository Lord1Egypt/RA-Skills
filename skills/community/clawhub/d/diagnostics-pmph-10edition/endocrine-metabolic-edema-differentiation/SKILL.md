# 内分泌代谢性水肿鉴别

## 适用场景
- 患者出现水肿，但已排除心源性、肝源性、肾源性等常见器质性疾病
- 水肿为全身性或特定分布（如面部、下肢、上肢）
- 存在内分泌或代谢异常病史或可疑表现

## 执行步骤

1. **评估水肿特征**：
   - 判断是否为凹陷性或非凹陷性水肿
   - 观察皮肤表现（如增厚、粗糙、苍白、温度降低）
   - 记录水肿部位（面部、下肢、上肢等）

2. **结合临床特征初步判断病因**：
   - **非凹陷性水肿 + 皮肤增厚粗糙苍白 + 体温低** → 提示甲状腺功能减退症（黏液性水肿）
   - **凹陷性水肿 + 局限性黏液性水肿** → 考虑甲状腺功能亢进症（可能因低蛋白血症及黏多糖沉积）
   - **下肢及面部轻度水肿** → 考虑原发性醛固酮增多症或库欣综合征（均因钠水潴留）
   - **面部黏液性水肿伴上肢水肿** → 提示腺垂体功能减退症
   - **糖尿病患者早期出现水肿（无心肾并发症）** → 可能为糖尿病相关代谢紊乱所致

3. **识别伴随症状辅助诊断**：
   - 心动过缓、低血压支持甲状腺功能减退
   - 高血压、低血钾提示原发性醛固酮增多症
   - 向心性肥胖、紫纹提示库欣综合征

4. **排除不适用情况**：
   - 局部性水肿（如炎症、静脉回流障碍）
   - 营养不良性或妊娠性水肿

5. **建议下一步检查**：
   - 甲状腺功能（TSH、FT3、FT4）
   - 血浆醛固酮、肾素活性
   - 24小时尿游离皮质醇或地塞米松抑制试验
   - 垂体激素全套（如ACTH、TSH、FSH/LH等）

> 注意：部分与激素周期相关的水肿（如经前期紧张综合征、特发性水肿）多见于女性，需结合病史综合判断。
