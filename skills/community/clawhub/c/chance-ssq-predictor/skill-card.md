## Description: <br>
双色球彩票数据分析与号码选择辅助工具。综合统计分析和趋势分析，提供遗漏值、频率、和值、AC值、跨度、奇偶比、大小比、区间分布等14个维度的数据，帮助生成号码组合。仅供娱乐参考。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users can ask for entertainment-oriented SSQ lottery number analysis and generated number combinations based on local statistical heuristics. Outputs should be treated as recreational guidance only, not as a way to improve lottery odds or as purchasing advice. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Users may mistake lottery number recommendations for improved winning odds. <br>
Mitigation: Present outputs as entertainment-only analysis and repeat that lottery drawings are random independent events. <br>
Risk: The included blue-ball analysis has a verified bug. <br>
Mitigation: Do not rely on blue-ball recommendations for decisions; review or fix the script before using that analysis. <br>
Risk: The package includes a manual Python script even though artifact wording says there is no executable code. <br>
Mitigation: Review the script before execution and run it only in an appropriate local environment. <br>
