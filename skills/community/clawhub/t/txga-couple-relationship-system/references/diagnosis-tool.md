## 能量状态诊断

### 核心问题：你现在处于哪种状态？

**选项A：充电器模式**
- 回家感觉放松、期待
- 可以从伴侣那里获得支持和能量
- 愿意分享心事和想法

**选项B：耗电模式**
- 回家感觉紧张、有压力
- 与伴侣交流需要消耗能量
- 倾向于回避或减少交流

**选项C：中性模式**
- 回家没什么特别感觉
- 关系平淡，但也没有冲突
- 可有可无的状态

### 评分标准

| 状态 | 分值 | 说明 |
|-----|-----|-----|
| 充电器模式 | 8-10 | 关系提供正向能量 |
| 中性偏正 | 5-7 | 关系不消耗但也不充能 |
| 中性偏负 | 3-4 | 有一定消耗但可承受 |
| 耗电模式 | 0-2 | 关系严重消耗能量 |

## 断点类型诊断

### 诊断问题组1：情绪账户透支

回答"是"得1分，"否"得0分：

1. 最近是否经常为小事发脾气？
2. 是否发现自己对伴侣的容忍度明显下降？
3. 是否开始翻旧账，把过去的问题拿出来说？
4. 是否感觉对方做什么都不对？
5. 是否感觉付出和回报严重不对等？

**得分 ≥ 3**：你可能处于情绪账户透支状态

### 诊断问题组2：身份否定

回答"是"得1分，"否"得0分：

1. 对方是否经常说"你怎么这么没用"类似的话？
2. 是否感觉在伴侣眼中自己一无是处？
3. 是否经常感到自我价值被否定？
4. 对方是否把你的付出视为理所当然？
5. 是否感觉被贬低而不是被欣赏？

**得分 ≥ 3**：你可能正在经历身份否定攻击

### 诊断问题组3：能量断供

回答"是"得1分，"否"得0分：

1. 你们是否已经很久没有深入交流？
2. 是否出现无性状态或明显减少？
3. 是否感觉双方都在各过各的？
4. 是否对这段关系不再有期待？
5. 是否感觉分开和在一起没什么区别？

**得分 ≥ 3**：你的关系可能已进入能量断供状态

## 闭环完整性检测

### 能量闭环完整性

| 检测项 | 状态 |
|-------|-----|
| 每天是否有至少一次正面互动？ | 有/偶尔/无 |
| 每周是否有固定的高质量相处时间？ | 有/偶尔/无 |
| 正向互动是否能形成正向循环？ | 是/不确定/否 |

**3项都是"是"**：能量闭环完整
**有2项以上"否"**：能量闭环断裂

### 价值闭环完整性

| 检测项 | 状态 |
|-------|-----|
| 双方是否清楚各自的核心需求？ | 是/不确定/否 |
| 是否存在明显的供给错位？ | 无/偶尔/经常 |
| 双方是否都觉得被满足？ | 是/部分/否 |

**3项都是正向**：价值闭环完整
**有2项以上负向**：价值闭环断裂

### 成长闭环完整性

| 检测项 | 状态 |
|-----|-----|
| 是否有共同的目标或项目？ | 有/曾经有/无 |
| 是否经常一起面对挑战？ | 是/偶尔/否 |
| 关系是否让你们双方都成长？ | 是/不确定/否 |

**3项都是正向**：成长闭环完整
**有2项以上负向**：成长闭环断裂

### 稳定闭环完整性

| 检测项 | 状态 |
|-----|-----|
| 是否有多重绑定（孩子/资产/社交圈）？ | 多/少/无 |
| 双方是否都认可这段关系的价值？ | 是/不确定/否 |
| 退出这段关系的成本是否足够高？ | 高/不确定/低 |

**3项都是正向**：稳定闭环完整
**有2项以上负向**：稳定闭环断裂

## 综合诊断结论

### 诊断框架

1. **能量状态**：充电器/耗电/中性
2. **主要断点**：透支/否定/断供
3. **链路健康度**：哪几条链路受损
4. **闭环完整性**：哪几个闭环断裂

### 修复优先级

| 优先级 | 情况 | 建议 |
|-------|-----|-----|
| P0 | 能量断供 | 必须先解决"是否继续"的问题 |
| P1 | 身份否定 | 优先修复自我价值系统 |
| P2 | 情绪透支 | 重建正向互动循环 |
| P3 | 链路受损 | 逐条修复能量链路 |
| P4 | 闭环断裂 | 设计和重建闭环结构 |
