## 一、核心演进路径

### 1.1 路径设计原则

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         演进路径设计原则                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  ✅ 精简层级：SPV → 实体公司 → SPAC，三阶段直达                              │
│  ✅ 无中间Spark阶段，避免不必要的架构复杂度                                  │
│  ✅ 每阶段有明确触发条件，不是人为划定的时间节点                             │
│  ✅ 权益递进：贡献权重 → 工商股权 → 上市股份                                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 三阶段总览

| 阶段 | 名称 | 形态 | 核心特征 | 触发条件 |
|------|------|------|----------|----------|
| **阶段1** | SPV虚拟共同体 | 虚拟 | 技能交换、贡献权重分配 | 当前阶段 |
| **阶段2** | 实体公司（双轨制） | 混合 | 工商注册、股权折算 | 连续3月盈利 + 年营收>500万 |
| **阶段3** | SPAC上市 | 上市 | 壳公司并购/直接上市 | 5个以上成熟项目 + 年营收>5000万 |

## 二、阶段1：SPV虚拟共同体

### 2.1 核心特征

| 特征维度 | 具体描述 |
|----------|----------|
| **组织形态** | 无工商注册，纯协议约束 |
| **合作模式** | 技能交换、贡献权重分配 |
| **权益载体** | 贡献权重（非股权） |
| **灵活性** | 高度灵活，随时加入退出 |
| **合规要求** | 低，主要依赖自愿遵守 |
| **资金归集** | 共管账户，不形成资金池 |

### 2.2 演进触发条件

**必须同时满足**：
- 连续3个月盈利
- 年营收突破500万
- 核心3人愿意全职投入
- 商业模式经过至少1个项目验证

### 2.3 阶段目标

| 目标类型 | 具体目标 | 验收标准 |
|----------|----------|----------|
| 商业验证 | 跑通至少1个商业模式 | 连续3个月盈利 |
| 团队磨合 | 建立协作机制和信任关系 | 核心成员稳定合作6个月+ |
| 案例积累 | 完成3-5个交付案例 | 有完整案例文档 |
| 合规准备 | 建立基础财务和合同体系 | 收支记录完整 |

## 三、阶段2：实体公司（双轨制）

### 3.1 核心特征

| 特征维度 | 具体描述 |
|----------|----------|
| **组织形态** | 成立有限合伙企业或有限责任公司 |
| **工商注册** | 正式法律实体，可开具发票 |
| **权益折算** | SPV贡献权重折算为工商股权 |
| **治理结构** | 股东会+董事会/执行事务合伙人 |
| **资本运作** | 可正式引入外部投资 |

### 3.2 演进触发条件

**必须同时满足**：
- 5个以上成熟SPV项目
- 年营收>5000万
- 研发投入占比≥15%
- 具备独立运营能力

### 3.3 转型关键动作

```
┌─────────────────────────────────────────────────────────────┐
│                  双轨制转型关键动作                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ 【Step 1：评估启动】                                        │
│    │ 连续3个月盈利                                         │
│    │ 年营收突破500万                                       │
│    │ 核心3人愿意全职投入                                   │
│    ↓                                                       │
│ 【Step 2：法律架构设计】                                    │
│    │ 选择公司类型（有限合伙/有限责任）                    │
│    │ 设计股权架构                                         │
│    │ 预留期权池（10-20%）                                 │
│    ↓                                                       │
│ 【Step 3：权重折算】                                        │
│    │ 历史贡献评估                                         │
│    │ 权重→股权折算比例确定                               │
│    │ 签署折算协议                                         │
│    ↓                                                       │
│ 【Step 4：工商注册】                                        │
│    │ 公司注册                                             │
│    │ 股权登记                                             │
│    │ 开立对公账户                                         │
│    ↓                                                       │
│ 【Step 5：运营切换】                                        │
│    │ 合同主体变更为公司                                   │
│    │ 发票开具变更为公司                                   │
│    │ 收益分配改为股权分红                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.4 股权架构建议

| 持股方 | 持股比例 | 性质 | 说明 |
|--------|----------|------|------|
| 核心创始团队 | 60-70% | 直接持股 | 按历史贡献折算 |
| 期权池 | 15-20% | 预留 | 激励未来加入者 |
| 战略投资者 | 10-20% | 引入 | 产业资源方 |

## 四、阶段3：SPAC上市

### 4.1 三种上市路径

| 路径 | 适用场景 | 核心要求 | 优缺点 |
|------|----------|----------|--------|
| **路径A：18C章特专科技公司** | 无盈利但有潜力的科技公司 | 商业化已落地、研发投入≥15% | ✅门槛低 ✅政策支持 ❌流动性较弱 |
| **路径B：SPAC并购上市** | 快速上市、减少审批风险 | 找到合适的SPAC发起人 | ✅周期短 ✅确定性高 ❌稀释较大 |
| **路径C：传统主板上市** | 盈利达标、规模较大 | 最近一年利润≥3500万港元 | ✅流动性好 ✅品牌效应 ❌门槛高周期长 |

### 4.2 政策对接方向

| 政策方向 | 对接目标 | 适用场景 |
|----------|----------|----------|
| **北交所** | 专精特新企业 | 中小创新型企业融资 |
| **港交所18C** | 特专科技公司 | 无盈利但有潜力的科技公司 |
| **科创板** | 硬科技企业 | 核心技术驱动的企业 |

### 4.3 演进触发条件

**必须同时满足**：
- 实体公司运营满2个完整财年
- 最近一年营收>5亿港元（或等值人民币）
- 具备核心技术或独特商业模式
- 有知名的机构投资者背书

## 五、权益递进链条

### 5.1 折算关系总览

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        权益递进折算链条                                      │
│                                                                             │
│   贡献权重（SPV层）                                                         │
│        │                                                                    │
│        │ ← 历史贡献评估                                                     │
│        │ ← 持续投入评估                                                     │
│        │ ← 未来价值评估                                                     │
│        ↓                                                                    │
│   工商股权（实体公司层）                                                     │
│        │                                                                    │
│        │ ← 稀释（引入投资）                                                 │
│        │ ← 增发（期权激励）                                                 │
│        │ ← 合并（装入母公司）                                               │
│        ↓                                                                    │
│   上市股份（SPAC层）                                                         │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 5.2 权重折算股权原则

| 评估维度 | 权重 | 评估内容 |
|----------|------|----------|
| **历史贡献** | 40% | 资金投入、时间投入、资源投入 |
| **持续投入** | 30% | 全职投入程度、不可替代性 |
| **未来价值** | 30% | 核心能力、稀缺资源、增长潜力 |

### 5.3 股权折算上市股份原则

| 阶段 | 折算比例 | 说明 |
|------|----------|------|
| 实体公司→SPAC | 1:1（通常） | 上市公司承继原股权结构 |
| 稀释影响 | 根据投资条款 | pre-money估值决定稀释比例 |
| 期权池 | 上市前确定 | 上市后行权 |

## 六、演进风险与应对

| 阶段转换 | 主要风险 | 应对策略 |
|----------|----------|----------|
| SPV→实体公司 | 税务风险、股权争议 | 提前6个月准备、专业律师介入 |
| 实体公司→SPAC | 估值争议、合规整改 | 提前2年规划、引入FA |
| 全程 | 政策变化 | 保持灵活性、备选方案 |
