# xhs-data-api 详细参考 v1.3.0

## 一、小红书CES评分算法深度解读

### 1.1 CES评分公式
```
CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
```

**权重解读：**
- 关注权重最高(8分)：平台最重视"内容能否带来粉丝"
- 评论/转发次之(4分)：深度互动比浅层互动更重要
- 点赞/收藏基础(1分)：基础互动指标
- **策略启示**：引导评论和关注比单纯要赞更有效

### 1.2 流量池递推机制
```
第1层：初始流量池（200-500曝光）
├── 互动率>5% → 进入第2层
├── 互动率2-5% → 观察期
└── 互动率<2% → 停止推荐

第2层：扩展流量池（1000-3000曝光）
├── 互动率>8% → 进入第3层
├── 互动率5-8% → 维持第2层
└── 互动率<5% → 逐渐减少推荐

第3层：热门推荐池（5000-10000曝光）
├── 互动率>10% → 进入爆款池
└── 互动率<8% → 维持第3层

第4层：爆款池（10000+曝光）
└── 持续监控互动率，下降则退出
```

### 1.3 影响推荐的其他因素
| 因素 | 权重 | 说明 |
|------|------|------|
| 账号权重 | 高 | 活跃度/垂直度/历史表现/违规记录 |
| 内容质量 | 高 | 原创度/图片质量/信息密度 |
| 用户停留 | 中高 | 阅读时长/完播率/反复查看 |
| 时效性 | 中 | 新内容优先/热点内容加权 |
| 社交关系 | 中 | 粉丝优先看到/互关加权 |
| 地域 | 低 | 同城内容有一定加权 |
| 搜索匹配 | 低 | 标题/标签与搜索词匹配度 |

## 二、爆款标题模式详解

### 2.1 高点击率标题公式
| 类型 | 公式 | 示例 | 点击率参考 |
|------|------|------|-----------|
| 数字型 | N个/第N/TopN | "5个让你皮肤变好的习惯" | +35% |
| 对比型 | 之前vs之后 | "减肥前vs后，同事都没认出来" | +50% |
| 痛点型 | 终于/不再XX | "终于找到了不脱妆的方法" | +40% |
| 清单型 | XX必看/收藏 | "去日本旅行必备清单" | +25% |
| 疑问型 | 怎么/为什么 | "为什么你的皮肤总是暗沉？" | +30% |
| 身份型 | XX人必看 | "油皮星人必看！这些水乳别踩雷" | +20% |
| 权威型 | 专家/XX年 | "皮肤科医生推荐的护肤步骤" | +30% |
| 紧迫型 | 赶紧/马上 | "入秋前一定要买的5件单品" | +25% |
| 揭秘型 | 真相/内幕 | "护肤品柜台不会告诉你的真相" | +45% |
| 省钱型 | 平价/学生党 | "学生党平价护肤，月花200搞定" | +35% |

### 2.2 标题优化规则
- **字数**：18-25字最佳（太短信息不足，太长被截断）
- **关键词前置**：核心信息放前10个字（搜索权重更高）
- **emoji点缀**：1-2个emoji提升视觉吸引力（不要过多）
- **避免违禁词**：不使用绝对化用语/医疗声称/虚假承诺
- **A/B测试**：同一内容换标题发2次对比数据

## 三、封面设计数据规律

### 3.1 高点击封面特征
| 风格 | 适合品类 | 点击率参考 | 注意事项 |
|------|---------|-----------|---------|
| 真人出镜 | 美妆/穿搭/生活 | 高 | 表情自然+光线好 |
| 对比图 | 减肥/护肤/家居 | 很高 | 前后对比要明显 |
| 清单/合集 | 好物/旅行/美食 | 中高 | 排版清晰+文字可读 |
| 场景图 | 旅行/探店/家居 | 中 | 构图讲究+色调统一 |
| 文字封面 | 干货/教程 | 中 | 字体大+配色好 |
| 精修美图 | 穿搭/美妆 | 中 | 不要过度失真 |

### 3.2 封面尺寸与比例
- **推荐比例**：3:4竖版（占屏面积最大，点击率最高）
- **次选**：1:1正方形（适合拼图/对比图）
- **避免**：16:9横版（占屏小，不推荐）
- **分辨率**：至少1080×1440px

## 四、标签策略框架

### 4.1 标签组合策略
```
黄金标签组合（每篇5-10个标签）：
├── 热门标签×2-3（10万+浏览量，获取流量）
├── 精准标签×2-3（1-10万，精准匹配）
├── 长尾标签×2-3（1万以下，低竞争高转化）
└── 品牌/IP标签×1（如有合作或自有品牌）
```

### 4.2 标签选择原则
- **热度+相关**：热度高但不相关=无效流量
- **竞争分析**：看标签下笔记数量，>10万太卷
- **趋势追踪**：关注近期新起的热门标签
- **品类细分**：大品类+细分品类组合覆盖
- **地域标签**：本地生活类必加城市标签

### 4.3 各品类热门标签参考（2026年）
| 品类 | 热门标签 | 精准标签 | 长尾标签 |
|------|---------|---------|---------|

| 穿搭 | 每日穿搭/ootd | 通勤穿搭/小个子穿搭 | 155小个子通勤搭配 |

## 五、竞品账号分析方法

### 5.1 选择对标账号标准
- **同领域**：内容品类相同或高度重叠
- **粉丝量**：比我方高1-3个量级（可学习不可望）
- **活跃度**：近30天有持续更新（至少周更）
- **有爆款**：近3个月至少1篇互动>1000的笔记
- **可复制**：内容制作门槛在我方能力范围内

### 5.2 竞品分析维度
```
竞品账号分析清单：
├── 基础数据：粉丝数/笔记数/获赞收藏总数
├── 更新策略：频率/时间段/内容类型比例
├── 爆款分析：近3个月爆款标题/封面/话题/标签
├── 失败分析：数据差的笔记有什么共性
├── 粉丝互动：评论区活跃内容/粉丝关注话题
├── 变现模式：是否有广告/带货/引流/知识付费
└── 差异化：与同品类其他账号的区别是什么
```

## 六、内容生命周期分析

### 6.1 笔记数据曲线
```
典型爆款笔记生命周期：
发布 → 2h（初始流量池200-500曝光）
     → 6h（如互动好，进入1000+曝光）
     → 24h（爆发期，5000-10000+曝光）
     → 3-7天（长尾期，搜索流量持续进入）
     → 30天+（搜索长尾，优质内容可持续获流量）

普通笔记生命周期：
发布 → 2h（初始200曝光）
     → 6h（互动不足，停止推荐）
     → 24h（基本停止推荐）
     → 7天+（仅搜索流量）
```

### 6.2 数据复盘时间点
| 时间点 | 复盘内容 | 动作 |
|--------|---------|------|
| 2小时后 | 初始互动率/点击率 | 判断是否进入扩展池 |
| 24小时后 | 总互动数/流量来源 | 判断是否成为爆款候选 |
| 7天后 | 完整数据表现 | 分析成功/失败因素 |
| 30天后 | 长尾表现/搜索排名 | 总结可复制的经验 |

## 八、涨粉转化漏斗

```
曝光 → 点击 → 阅读 → 互动 → 关注
100%   8-15%   60-80%  5-10%  1-3%

各环节优化重点：
曝光→点击：标题+封面（最关键！）
点击→阅读：开头Hook（前3秒/前2行）
阅读→互动：内容价值+CTA引导
互动→关注：账号价值感+系列化内容+个人IP
```

### 涨粉效率提升策略
1. **系列化内容**：创建系列专栏，粉丝为了看后续会关注
2. **个人IP强化**：统一封面风格/固定开头/标志性结尾
3. **互动引导**："关注我，下期告诉你XX"
4. **评论区运营**：回复评论提升CES+建立粉丝关系
5. **主页优化**：简介+置顶笔记展示账号价值
