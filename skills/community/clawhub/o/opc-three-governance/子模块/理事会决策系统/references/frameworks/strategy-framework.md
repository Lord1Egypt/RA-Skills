# 战略治理框架

## 战略理事角色定义

**核心职责**：审视战略方向、竞争格局、长期发展

**决策权重**：+1（具有否决权）

---

## 战略评估维度

### 1. 战略一致性

```
评估问题：
□ 该决策是否与公司使命、愿景一致？
□ 是否符合中期战略规划？
□ 是否强化核心竞争力？
□ 是否偏离战略边界？

评估标准：
✅ 一致：完全符合战略方向
⚠️ 部分一致：需要调整以匹配战略
❌ 不一致：违背战略方向
```

### 2. 竞争格局分析

```
评估问题：
□ 对竞争格局的影响？
□ 是否会引发竞争对手反应？
□ 竞争优势是否得到加强？
□ 竞争壁垒是否提升？

波特五力分析框架：
- 行业内现有竞争者
- 潜在进入者威胁
- 替代品威胁
- 供应商议价能力
- 客户议价能力
```

### 3. 战略时机评估

```
评估问题：
□ 时机是否成熟？
□ 是否错过最佳窗口？
□ 是否需要等待？
□ 宏观环境是否有利？

时机判断：
🟢 最佳时机：条件成熟，可以行动
🟡 准备时机：需要更多准备
🔴 时机不对：需要等待或放弃
```

### 4. 战略风险识别

```
风险类型：
1. 执行风险：能否有效执行？
2. 市场风险：市场是否接受？
3. 技术风险：技术是否可行？
4. 资源风险：资源是否充足？

风险评估矩阵：
         影响大 | 影响小
可能性高 | 高风险 | 中风险
可能性低 | 中风险 | 低风险
```

## 战略理事议事规则

### 审议标准

```
【通过标准】
1. 与公司战略一致
2. 竞争风险可控
3. 时机合适
4. 资源可匹配

【否决条件】
1. 严重偏离战略方向
2. 重大竞争风险
3. 时机明显不对
4. 资源严重不足

【条件通过】
1. 在某些条件下可以通过
2. 需要补充某些信息
3. 需要调整某些内容
4. 需要制定风险预案
```

### 战略否决权使用

当战略理事行使否决权时：

```
【否决理由】
必须明确说明：
1. 哪一点战略不一致？
2. 什么竞争风险？
3. 时机问题在哪里？
4. 资源缺口有多大？

【建议措施】
1. 如何调整可以满足条件？
2. 需要补充什么信息？
3. 需要做哪些准备？
```

## 战略理事检查清单

- [ ] **战略一致性**：与公司战略方向一致？
- [ ] **竞争影响**：竞争风险可控？
- [ ] **时机判断**：行动时机合适？
- [ ] **资源匹配**：战略资源可支撑？
- [ ] **风险识别**：重大风险已识别？
- [ ] **退出机制**：有清晰的退出标准？
- [ ] **战略否决**：是否需要行使否决权？
