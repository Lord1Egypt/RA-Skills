## 1. 安全产品

### 1.1 终端安全（杀毒软件）
- **产品形态**：PC版终端安全客户端 + 服务器版安全客户端
- **核心能力**：病毒查杀、终端管控、统一安全管理平台、实时告警
- **适用场景**：企业终端安全防护、公安整改要求、内网安全加固
- **推荐信号**：客户IP被攻击、公安发函要求整改、终端失陷、无终端安全防护
- **卖点**：统一平台可视化管理、快速部署、电信级安全运营

### 1.2 防火墙
- **产品形态**：下一代防火墙（NGFW）
- **核心能力**：访问控制、入侵防御、应用识别、VPN
- **适用场景**：网络出口安全防护、内外网隔离、合规要求
- **推荐信号**：客户无出口防护、网络边界不安全、需等保合规
- **卖点**：电信级防护能力、7×24安全运营支持

### 1.3 上网行为管理
- **产品形态**：上网行为管理设备/云服务
- **核心能力**：员工上网行为审计与管控、流量管理、内容过滤
- **适用场景**：企业网络管理、员工行为合规、带宽优化
- **推荐信号**：客户需管控员工上网行为、网络带宽被占用、合规审计要求

### 1.4 等保测评与整改
- **产品形态**：一站式等保合规服务
- **核心能力**：等保咨询、差距分析、整改实施、测评协助
- **适用场景**：政府、金融、医疗、教育等需等保合规的行业
- **推荐信号**：客户有合规要求、需要等保二级/三级、新系统上线

### 1.5 云安全
- **产品形态**：DDoS防护、WAF、主机安全、安全态势感知
- **核心能力**：云上安全防护体系、7×24小时安全运营
- **适用场景**：已上云或有上云计划的企业
- **卖点**：电信级安全运营、与天翼云深度整合

### 1.6 SD-WAN（安全增强版）
- **产品形态**：SD-WAN设备 + 安全服务
- **核心能力**：智能选路、加密传输、安全防护、可视化管理
- **适用场景**：多分支机构组网、跨境合规访问、远程办公
- **推荐信号**：客户有多地办公、跨境业务需求、传统专线成本高
- **卖点**：行业数据显示可降本30%-50%、快速部署、支持合规跨境

## 2. 基础通信产品

### 2.1 V网（虚拟专用移动网）
- **定义**：企业内部短号互拨免费通话
- **适用场景**：员工沟通频繁的中大型企业、制造业工厂、物流团队
- **核心卖点**：短号拨打、网内免费、统一管理、降低通信成本
- **推荐信号**：客户有50人以上团队、跨部门沟通多、通信费用占运营成本较高

### 2.2 专线
- **类型**：互联网专线、数据专线、MSTP专线
- **适用场景**：金融机构、政府机关、跨国企业、数据中心互联
- **核心卖点**：独享带宽、低时延、高可靠、SLA保障
- **推荐信号**：客户有远程办公需求、多分支机构互联、对网络稳定性要求高

### 2.3 企业宽带
- **适用场景**：中小企业办公、商铺、连锁门店
- **核心卖点**：上下行对称、固定公网IP、专属客服
- **推荐信号**：客户为新成立或搬迁企业、现有宽带体验差

### 2.4 融合通信
- **产品形态**：企业固话、云总机、融合通信解决方案
- **适用场景**：客服中心、前台接待、多分支机构统一号码
- **核心卖点**：统一号码管理、呼叫转移、录音质检

## 3. 云与数据中心产品

### 3.1 天翼云
- **核心服务**：云主机（ECS）、云存储（OBS）、云数据库（RDS）、CDN、容器服务
- **适用场景**：企业IT上云、业务系统部署、数据备份与灾备
- **核心卖点**：国有云安全合规、全国资源布局、电信级运维保障、云网融合
- **推荐信号**：客户有上云计划、自建机房成本高、需满足等保合规要求

### 3.2 云灾备
- **产品形态**：云灾备服务、异地容灾方案
- **适用场景**：有数据保护需求的企业、需灾备合规的行业
- **推荐信号**：客户关注数据安全、有灾备需求、合规要求
- **卖点**：分钟级RTO、自动化备份、成本远低于自建灾备中心

### 3.3 IDC托管
- **适用场景**：金融、互联网、政务等对机房等级要求高的行业
- **核心卖点**：Tier3+/Tier4机房等级、双路供电、电信级运维
- **推荐信号**：客户自建机房面临扩容、合规、成本压力

## 4. AI与智能化产品

### 4.1 大模型API聚合平台
- **定义**：大模型API聚合服务平台，兼容OpenAI接口规范
- **能力**：国内外主流大模型API统一接入（doubao/qwen/kimi/glm/deepseek等）
- **适用场景**：企业AI开发、研发团队编程提效、智能体搭建
- **核心卖点**：成本更低、安全合规、统一管控、服务保障
- **推荐信号**：客户有AI开发需求、研发团队想用AI编程工具、需多模型切换、使用TRAE/Cursor等编程工具

### 4.2 AI编程提效
- **产品形态**：AI编程工具 + 大模型API平台
- **适用场景**：研发团队日常开发、代码审查、技术文档生成
- **推荐信号**：客户有研发团队、使用或想用AI编程工具（TRAE/Cursor/Copilot等）、关注开发效率

### 4.3 AI行业应用
- **子方向**：AI客服、AI质检、AI风控、AI巡检
- **适用场景**：客服中心、制造业、金融、政务

- **适用场景**：学校、教育局、培训机构

- **适用场景**：医院、卫健委、医药企业

### 5.5 智慧商业
