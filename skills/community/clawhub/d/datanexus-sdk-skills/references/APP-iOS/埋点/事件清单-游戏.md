# iOS 游戏 — 必报与选报事件

> 来源：腾讯广告 iOS SDK 开发指引
>
> iOS 端必报事件与 Android 端基本一致，差异点已标注。

## 必报事件

| action_type | 中文名 | 说明 |
|------------|--------|------|
| `START_APP` | 启动应用 | 用于广告归因、广告效果优化。**必须手动上报**（冷启+热启+Deeplink 三种场景均需上报） |
| `REGISTER` | 注册 | 用于广告效果优化。用户完成账号注册时上报 |
| `LOGIN` | 登录 | 用于广告效果优化。用户每次完成账号登录时上报 |
| `PURCHASE` | 付费 | 用于广告归因、广告效果优化。**必须传 `value`（金额，单位：分）**，完整上报所有用户所有支付方式的全部付费行为 |

## 条件必报事件（无该行为可不报）

| action_type | 中文名 | 说明 |
|------------|--------|------|
| `CREATE_ROLE` | 创建角色 | 用于广告归因、广告效果优化。无该行为可不报 |
| `TUTORIAL_FINISH` | 完成新手教程 | 用于广告归因、广告效果优化。无该行为可不报 |
| `VIEW_CONTENT` | 浏览商城/活动 | 用于广告效果优化。无该行为可不报 |
| `UPDATE_LEVEL` | 游戏等级提升 | 用于广告效果优化。无该行为可不报 |

## 选报事件

> 与 Android 端一致，根据业务场景选择上报。

| action_type | 中文名 | 专用方法 |
|------------|--------|---------|
| `BINDACCOUNT` | 绑定社交账号 | `reportBindAccountAction(method, isSuccess)` |
| `QUEST_FINISH` | 完成任务 | `reportQuestFinishAction(id, type, name, number, desc, isSuccess)` |
| `SHARE` | 分享 | `reportShareAction(channel, isSuccess)` |
| `RATE` | 评分 | `reportRateAction(value)` |
| `ADD_TO_CART` | 加入购物车 | `reportAddToCartAction(type, name, id, number, isSuccess)` |
| `CHECKOUT` | 提交订单 | `reportCheckoutAction(type, name, id, number, isVirtual, virtualType, currency, isSuccess)` |
| `ADD_PAYMENT_CHANNEL` | 添加支付渠道 | `reportAddPaymentChannelAction(channel, isSuccess)` |

## 专用上报方法（GDTAction+convenience）

| 方法 | 对应 action_type |
|------|-----------------|
| `reportRegisterAction(method, isSuccess)` | REGISTER |
| `reportLoginAction(method, isSuccess)` | LOGIN |
| `reportPurchaseAction(type, name, id, number, channel, currency, amount, isSuccess)` | PURCHASE |
| `reportCreateRoleAction(roleName)` | CREATE_ROLE |
| `reportUpdateLevelAction(level)` | UPDATE_LEVEL |
| `reportViewContentAction(type, name, id)` | VIEW_CONTENT |
| `reportShareAction(channel, isSuccess)` | SHARE |

> 建议优先使用专用方法。
>
> iOS 端 `START_APP` 使用 `[GDTAction logAction:@"START_APP"]` 上报。
