## Description: <br>
Darts Forecasting guides agents through time-series forecasting for multi-market financial data, while the current artifact also contains ZVT quant-trading and backtesting workflows that require review before use. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and quantitative analysts use this skill to get guidance, code, shell commands, and configuration steps for financial time-series forecasting, data preparation, backtesting, and related market-data workflows. Users should review the scope because the artifact mixes Darts forecasting material with ZVT trading behavior. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The release is labeled as Darts forecasting, but the security evidence reports conflicting ZVT quant-trading, backtesting, broker/provider, and unrelated documentation-tooling instructions. <br>
Mitigation: Install only when the intended use includes ZVT or quant-trading assistance, and review generated workflows before acting on them. <br>
Risk: The artifact may lead users toward data collection, package installation, generated skill saving, or trading-related actions. <br>
Mitigation: Keep workflows in dry-run or backtest mode by default and require explicit confirmation before any data collection, package install, saved generated skill, or trading-related action. <br>
Risk: Broker or paid-provider credentials may be requested for some workflows. <br>
Mitigation: Avoid entering broker or paid-provider credentials unless the permissions and account impact are understood. <br>
