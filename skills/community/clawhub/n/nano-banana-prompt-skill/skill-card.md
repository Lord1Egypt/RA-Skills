## Description: <br>
Converts Chinese image descriptions into structured English JSON prompts and generation parameters for Nano Banana-style image generation workflows. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[xk103295870-alt](https://clawhub.ai/user/xk103295870-alt) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Creators, marketers, and developers use this skill to turn Chinese image requests into structured JSON prompts for product images, avatars, logos, posters, and social media visuals. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad activation wording may cause the skill to handle ordinary image-design requests and return Nano Banana-style JSON unexpectedly. <br>
Mitigation: Use explicit wording when invoking the skill, and review generated prompts plus safety and person-generation parameters before sending them to an image provider. <br>
