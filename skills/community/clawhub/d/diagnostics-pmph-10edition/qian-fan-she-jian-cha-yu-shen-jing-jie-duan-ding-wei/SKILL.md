# 浅反射检查与神经节段定位

## 适用对象
- 腹部皮肤
- 股内侧皮肤
- 足底
- 肛周皮肤

## 检查前提
1. 病人仰卧并充分放松。
2. 腹壁松弛（可令下肢稍屈曲）。
3. 排除局部皮肤病变（如皮疹、瘢痕、水肿等）。

## 操作流程

### 1. 腹壁反射
- **方法**：用钝头竹签由外向内轻划腹壁皮肤。
  - 上腹（肋缘下）→ 对应胸髓7–8节
  - 中腹（脐平）→ 对应胸髓9–10节
  - 下腹（腹股沟上）→ 对应胸髓11–12节
- **正常反应**：相应区域腹肌收缩。
- **异常解读**：
  - 单一水平消失 → 对应胸髓节段损害
  - 同侧上中下均消失 → 同侧锥体束损害
  - 双侧全消失 → 昏迷、急性腹膜炎或广泛锥体束损害

### 2. 提睾反射
- **方法**：用竹签由下向上轻划股内侧上方皮肤。
- **正常反应**：同侧睾丸上提。
- **异常解读**：
  - 双侧消失 → 腰髓1–2节损害
  - 一侧减弱或消失 → 锥体束损害
  - 局部病变（如疝、水肿）也可导致假性异常

### 3. 跖反射
- **方法**：沿足底外侧，从足跟向前至小趾跖关节处，再转向内侧划动。
- **正常反应**：足趾跖屈（Babinski征阴性）。
- **异常解读**：反射消失提示骶髓1–2节损害。

### 4. 肛门反射
- **方法**：用大头针轻划肛周皮肤。
- **正常反应**：肛门外括约肌收缩。
- **异常解读**：反射障碍提示骶髓4–5节、锥体束或肛尾神经损害。

## 注意事项
- 划动必须轻柔，避免引起疼痛或防御反应。
- 所有反射均需双侧对比观察。
- 肥胖、老年、经产妇可能出现腹壁反射生理性减弱，昏迷者浅反射普遍消失，需结合临床背景综合判断。

## 输出结果
记录各浅反射是否存在，并依据消失模式判断可能的脊髓节段或锥体束损害部位。
