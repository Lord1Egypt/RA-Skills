# 胚胎致畸敏感期分期规则

## 适用对象
- 人类胚胎
- 胎儿发育阶段

## 前提条件
- 已知受精时间或孕周

## 使用限制
- 不适用于非人类物种
- 不适用于出生后发育异常

## 执行步骤

1. **确定当前孕周**（自受精日起计算的整数周数）。

2. **按以下三阶段判断致畸敏感性**：
   - **受精后前2周（胚胎期）**：
     - 强致畸因子 → 胚胎死亡（“全或无”现象）
     - 弱致畸作用 → 细胞代偿修复 → 很少发生畸形
   - **受精后第3～8周末（胚期）**：
     - 细胞增生分化活跃，器官原基形成
     - 是主要致畸敏感期，最易发生器官形态畸形
   - **受精后第9周至出生（胎期）**：
     - 器官进行组织与功能分化
     - 致畸因子可导致组织结构异常或功能缺陷
     - 一般不引起器官形态畸形 → 不属于典型致畸敏感期

3. **结合具体致畸因子类型，评估特异性敏感窗口**：
   - 若暴露于**风疹病毒**：高风险期为孕第1个月（50%畸形率），次之为第2个月（22%），第3个月风险显著降低（6%～8%）
   - 若暴露于**沙利度胺（反应停）**：高风险窗口为受精后第21～40天

4. **输出风险提示**：
   - 明确当前是否处于高风险致畸敏感期
   - 提示相应防护或医学随访建议
