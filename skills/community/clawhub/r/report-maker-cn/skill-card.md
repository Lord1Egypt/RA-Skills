## Description: <br>
Generates local daily or weekly work reports in Markdown from optional JSON task data, including completed work, in-progress items, plans, metrics, and risks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[freedompixels](https://clawhub.ai/user/freedompixels) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Employees and teams use this skill to turn structured work updates into readable daily or weekly Markdown reports. It helps summarize completed items, active work, plans, metrics, and risks before sharing a report. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated reports may include sensitive work details if the input JSON contains them. <br>
Mitigation: Provide only the work data intended for the report and review the Markdown before sharing. <br>
Risk: Users may assume the skill automatically sends reports because some trigger text mentions automatic reporting. <br>
Mitigation: Treat the skill as a local report generator; send or publish the generated Markdown only after manual review. <br>
