# 右心室结构与功能分区

## 适用场景
- 分析右心室血流路径与瓣膜功能
- 规划或执行右心室手术（如切口定位）
- 评估三尖瓣反流或右心功能障碍的解剖基础

## 执行步骤

1. **识别室上嵴（supraventricular crest）**：定位该弓形肌性隆起，将其作为分界标志，将右心室分为：
   - **流入道（固有心腔）**：位于后下方
   - **流出道（动脉圆锥/漏斗部）**：位于前上方

2. **评估流入道结构与功能**：
   - 检查内壁是否存在肉柱（trabeculae carneae），确认其凹凸不平特征
   - 定位三群乳头肌（前、后、隔侧），特别注意前乳头肌发出的腱索连接三尖瓣前尖与后尖
   - 识别隔缘肉柱（节制索）：从室间隔下部横跨至前乳头肌根部，内含右束支及血管——**手术中必须避免损伤，以防右束支传导阻滞**
   - 确认三尖瓣环位于右房室口周围，三尖瓣由前尖、后尖、隔侧尖组成，与腱索、乳头肌共同构成三尖瓣复合体

3. **理解三尖瓣功能机制**：
   - 心室收缩时，瓣环缩小 + 血流推动使瓣叶闭合
   - 乳头肌同步收缩并通过腱索牵拉，防止瓣叶翻入右心房，确保血液单向流动

4. **评估流出道（动脉圆锥）特征**：
   - 内壁光滑无肉柱，呈圆锥状
   - 上端为肺动脉口，周缘附着三个半月形肺动脉瓣，游离缘朝向肺动脉干
   - 舒张期返流血液充盈肺动脉窦，促使瓣膜靠拢关闭，防止血液反流入右心室

5. **确定手术安全区域**：
   - 右心室前壁较薄（约为左心室厚度的1/3）且血管较少，是常用手术切口部位

> **注意**：本技能不适用于先天性右心室发育异常的特殊病例。
