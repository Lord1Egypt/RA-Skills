# 房地产

## 核心财务特征

- 高杠杆运营（行业均值资产负债率 60-80%）
- 预收账款/合同负债规模大（预售制）
- 存货特殊（开发成本+开发产品，区别于制造业存货）
- 利润对融资成本高度敏感
- 现金流波动极大（拿地→开发→预售→交付周期长）

## 关键分析指标

| 指标 | 健康值 | 警戒值 | 说明 |
|------|--------|--------|------|
| 净负债率 | <100% | >150% | （有息负债-现金）/净资产 |
| 现金短债比 | >1x | <0.5x | 货币资金/短期有息负债 |
| 剔除预收款的资产负债率 | <70% | >85% | 三道红线核心指标 |
| 融资成本 | <5% | >8% | 央企<4%，民企6-12% |
| 预收账款/营业收入 | >1x | <0.5x | 未来业绩锁定度 |
| 存货周转率 | 0.2-0.5x | <0.1x | 去化速度 |

## 风险评估重点

- 融资能力（三道红线达标情况）
- 土储质量（一二线vs三四线占比）
- 表外负债（联合营公司、明股实债）
- 交付风险（竣工面积vs合同销售面积）
- 多元化风险（物业/商业/长租等业务亏损拖累）

## 舞弊高发区

- 提前确认收入（未竣工即确认收入）
- 表外负债（SPV、联合营公司藏债）
- 存货减值不足（房价下跌仍不减值）
- 利息资本化过度（资本化率 > 90%）
