# Prompt: Affiliate Product Review Post

You are an expert affiliate content writer. Write a comprehensive, SEO-optimized product review post designed to rank in search and convert readers to buyers through an affiliate link.

## Inputs
- Product: [PRODUCT_NAME]
- Category: [PRODUCT_CATEGORY]
- Target audience: [TARGET_AUDIENCE]
- Key features to highlight: [KEY_FEATURES]
- Affiliate disclosure: [AFFILIATE_DISCLOSURE]
- Price point: [PRICE_POINT]
- Competitors to compare against: [ALTERNATIVES]

## Output Format

Write a 1,200–1,800 word affiliate review post with the following structure:

**[PRODUCT_NAME] Review: Is It Worth It for [TARGET_AUDIENCE]? ([YEAR])**

**[AFFILIATE_DISCLOSURE]**

### The Problem This Solves
(2-3 paragraphs establishing the reader's pain point before introducing the product. Write from a first-person discovery perspective: "I was frustrated with X until I found [PRODUCT_NAME].")

### What Is [PRODUCT_NAME]?
(1 paragraph: plain-language product description, who made it, what category it belongs to, how long it's been around)

### Key Features Breakdown
For each of the [KEY_FEATURES]:
- **[Feature Name]:** 2-3 sentences on what it does, why it matters to [TARGET_AUDIENCE], and one specific detail (number, stat, or comparison) that makes it concrete.

### Pros and Cons

| Pros | Cons |
|------|------|
| [pro 1] | [con 1] |
| [pro 2] | [con 2] |
| [pro 3] | [con 3] |
| [pro 4] | [con 4] |

(Write 4 genuine pros and 4 genuine cons. The cons should be real limitations, not softballs. This builds trust.)

### [PRODUCT_NAME] vs [ALTERNATIVE_1] vs [ALTERNATIVE_2]

| Feature | [PRODUCT_NAME] | [ALTERNATIVE_1] | [ALTERNATIVE_2] |
|---------|----------------|-----------------|-----------------|
| Price | | | |
| [Feature 1] | | | |
| [Feature 2] | | | |
| [Feature 3] | | | |
| Best for | | | |

(2 paragraphs explaining when to choose each option. Be honest. If an alternative is better for a specific use case, say so — it builds credibility.)

### Pricing
(Clear breakdown of all pricing tiers. Include what's included at each level. Note any free trial or money-back guarantee.)

### Who Should Buy [PRODUCT_NAME]?
**Buy it if:** [3 bullet points describing the ideal customer]
**Skip it if:** [2 bullet points describing who should look elsewhere]

### Frequently Asked Questions
**Q: [Question 1 — most common search query]**
A: [Direct answer, 2-3 sentences]

**Q: [Question 2]**
A: [Answer]

**Q: [Question 3 — about pricing or free trial]**
A: [Answer]

**Q: [Question 4 — about a limitation or concern]**
A: [Honest answer]

**Q: [Question 5 — comparison with a competitor]**
A: [Answer]

### Final Verdict
(3-4 sentences. State your recommendation directly. Who is this for, what's the best use case, is it worth the price? End with a clear CTA.)

**[Try [PRODUCT_NAME] here →]** ← affiliate link placement
**[AFFILIATE_DISCLOSURE]**

## SEO Notes (include in your thinking, not in the post output):
- Target keyword: "[PRODUCT_NAME] review" + "[PRODUCT_NAME] vs [ALTERNATIVE_1]"
- Include keyword in H1, first paragraph, and at least one H2
- Word count: 1,200–1,800 words
- Internal link opportunity: link to your comparison post
