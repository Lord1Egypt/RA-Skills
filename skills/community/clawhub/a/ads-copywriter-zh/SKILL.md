# 多平台广告文案生成器

针对中国主流广告平台，自动生成符合各平台规格的高转化广告文案，支持多版本A/B测试对比。

## Tools Required
- Claude Sonnet / Opus（推荐）
- 可配合 GPT-4o 使用

## 平台规格说明

### 百度搜索推广（竞价广告）

```yaml
百度搜索广告:
  标准推广单元:
    标题:
      数量: 最多50个创意标题
      每条字数: ≤30字
      要求:
        - 至少3条包含核心关键词
        - 至少2条包含品牌名
        - 至少2条包含行动号召（立即咨询/免费领取等）
        - 至少2条包含数字或卖点
    描述:
      数量: 4条（最少2条）
      每条字数: ≤80字
      要求:
        - 展开标题中的承诺
        - 包含核心利益点
        - 包含行动号召
        - 包含信任背书

  智能推广:
    标题: 5条（每条≤30字）
    长标题: 5条（每条≤80字）
    描述: 5条（每条≤80字）
```

### 微信朋友圈广告

```yaml
朋友圈广告:
  信息流广告:
    正文: 40字（可见），最多200字
    标题: 14字
    行动号召: [了解详情, 立即购买, 立即报名, 领取优惠, 立即预约]

  视频广告:
    文案覆层: 40字
    标题: 14字
```

### 抖音信息流广告

```yaml
抖音广告:
  信息流视频:
    描述文案: 最多2200字
    首行可见: 约50字
    话题标签: 建议3-5个

  搜索广告:
    标题: ≤20字
    描述: ≤40字
```

### 小红书广告/种草笔记

```yaml
小红书:
  推广笔记:
    标题: ≤20字（含emoji效果更佳）
    正文: 500-1000字（种草体）
    标签: 5-10个相关话题
  
  开屏/信息流:
    主标题: ≤15字
    副标题: ≤30字
```

### 淘宝/京东直通车

```yaml
直通车:
  创意标题:
    字数: ≤30字
    数量: 建议4条做轮播测试
    要求:
      - 包含核心关键词
      - 突出促销信息（折扣/赠品/限时）
      - 包含差异化卖点
```

## 文案生成框架

### 吸引力公式

```
公式1（痛点型）：[目标人群] + [痛点场景] + [解决方案] + [行动号召]
示例：宝妈必看！孩子总生病？这款益生菌3天见效，立即领取试用装

公式2（利益型）：[数字/优惠] + [核心卖点] + [信任背书] + [紧迫感]
示例：限时5折！10万+用户选择，雅思备考课程今日报名立减500元

公式3（场景型）：[使用场景] + [产品特点] + [情感共鸣] + [CTA]
示例：加班到凌晨，一杯暖胃汤让你元气满满，下单即享包邮

公式4（社交证明型）：[权威/数据] + [产品描述] + [对比优势] + [引导行动]
示例：央视推荐！同类产品销量NO.1，品质有保障，点击查看详情
```
