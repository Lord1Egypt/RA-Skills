# 核仁结构与功能分析

## 适用场景
- 核仁形态与功能关联分析
- rRNA加工效率评估
- 核糖体生物合成研究
- 癌症或衰老相关核仁变化解读

## 执行步骤
1. **确认适用条件**：
   - 细胞为真核且处于间期
   - 存在核仁组织区（NOR）

2. **映射结构-功能关系**：
   - 纤维中心（FC）：含rDNA，为18S/5.8S/28S rRNA转录位点
   - 致密纤维组分（DFC）：含正在转录的rRNA
   - 颗粒组分（GC）：含核糖体前体颗粒

3. **追踪rRNA加工路径**：
   - 45S前体 → 剪切为18S、28S+5.8S
   - 5S rRNA由Pol III在核内转录，后转运至核仁参与大亚基组装

4. **输出评估结果**：
   - 核仁结构完整性及rRNA加工效率可反映细胞蛋白质合成活性
