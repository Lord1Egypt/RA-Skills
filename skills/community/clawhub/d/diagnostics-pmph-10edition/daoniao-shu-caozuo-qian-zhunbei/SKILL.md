# 导尿术操作前准备

## 何时使用
- 需要对患者实施导尿术前
- 患者已签署知情同意书
- 操作者需确认是否满足所有前提条件和约束条件

## 执行步骤

### 1. 环境准备
- 选择安静的操作环境
- 关闭门窗或使用屏风遮挡，劝离无关人员
- **特别规定**：男医师为女性患者导尿时，必须有女性医护人员或家属在场，以保护患者隐私

### 2. 患者准备
- 确保患者及家属已知晓导尿的目的及注意事项
- 患者已自行初步清洗外阴

### 3. 物品准备
- **核心物品**（一次性无菌导尿包内含）：
  - 治疗巾1块
  - 消毒外阴用手套1副
  - 治疗盘1个
  - 治疗碗1个
  - 小药杯1个
  - 持物钳3把
  - 液体石蜡棉球1包
  - 纱布2块
  - 内含20ml生理盐水的20ml注射器1支
  - 0.5%碘伏棉球2包 **或** 0.1%苯扎溴铵无菌棉球1包
  - 洞巾1块
  - 导尿管1根
  - 导尿用手套1副
  - 引流袋1个
- **酌情准备物品**：弯盘、中单、橡胶单、留标本用试管及试管架、胶布、别针、便盆等

### 4. 操作者准备
- 核对患者身份，评估病情，特别注意是否存在导尿禁忌证
- 严格按照七步洗手法洗手
- 戴好帽子和医用外科口罩

## 关键约束
- 存在导尿禁忌证时不得执行
- 未满足隐私保护条件（如男医师为女性患者操作无第三方在场）时不得进行

## 输出结果
完成全部四项准备后方可进入导尿操作流程；任一环节缺失即暂停操作。
