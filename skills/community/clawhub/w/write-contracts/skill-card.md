## Description: <br>
Generates Aptos Move V2 smart contracts with Object model, Digital Asset integration, security patterns, and storage type guidance. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[iskysun96](https://clawhub.ai/user/iskysun96) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to draft Aptos Move V2 modules for NFTs, marketplaces, minting, DAOs, staking, and related smart-contract workflows. It also helps choose storage patterns and apply access-control, event, initialization, and arithmetic-safety guidance. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated contract drafts may contain code-quality issues, including NFT ownership or percentage-overflow mistakes noted by security guidance. <br>
Mitigation: Use the output as drafting assistance, compare patterns against current Aptos documentation, fix identified issues, and get an independent audit before deploying contracts that manage real assets. <br>
Risk: Publish or run commands could be executed against the wrong Aptos profile or network. <br>
Mitigation: Confirm the intended profile and network before running CLI commands. <br>
