# Market Intel 📊

金融市场情报聚合系统 — 对标 investing.com 核心功能，全免费公开数据源。

> ⚡ 专为 OpenClaw 设计，微信发一条消息即可获取浓缩市场情报。

## 模块

| 触发词 | 模块 | 数据源 |
|--------|------|--------|
| `恐慌指数` | 恐惧贪婪指数 | CNN + Alternative.me |
| `行情` `大盘` | 指数行情 | 东方财富 |
| `要闻` `快讯` | 市场要闻 | CNBC + MarketWatch |
| `日历` `财经日历` | 宏观事件日历 | 内置日程 + RSS |
| `BTC大盘` `走势` `走势分析` | 加密货币技术分析 | OKX + Alternative.me |

### 加密货币分析 (v1.1 新增)

七维分析框架 (`references/crypto-analysis.md`)：

1. **量能分析** — 24h成交量 vs 7日均值，判断放量/缩量
2. **多周期背离检测** — 15m/4H/1D 的 RSI+MACD 顶背离/底背离
3. **关键位置** — 支撑/阻力位（前高前低、EMA、布林带）
4. **波动率评估** — 24h波动率 vs 7日均值
5. **合约市场** — 资金费率 + 持仓量(OI)方向
6. **消息面** — 跨引用要闻快讯
7. **浓缩信号** — 3-5条后续关注重点 + 方向预判

## 设计原则

- **零 API Key**: 全部使用免费公开数据源
- **手机安全**: 纯 ASCII 排版，5星评级替代进度条，不折行不错位
- **即问即答**: 一句话触发，无需翻网页
