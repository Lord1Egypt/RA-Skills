# 小红书爆款封面生成大师

需要执行**完整流程**时：**必须先读取**与本技能同目录下的 `references/core_workflow.md`，并**完整遵循**其中的触发规则、术语规范、数据来源约束、操作步骤、自检清单与注意事项。生成 **HTML 报告**时另需按该文档要求读取 `references/xhs_trend_data_format.md`。脚本路径（相对技能目录）：`scripts/fetch_explosive_covers.py`。

## 简介

面向小红书创作者的**爆款封面**设计辅助：依托每日持续收录的爆款笔记数据（如 2000+ 条量级，以实际脚本与数据源为准），用同赛道高互动样本与视觉规律提炼，减少凭感觉试错，把「能点的封面」落到可执行的结构与提示词级方案。

通过本 Skill，你可以：

- **🎯 对标有据**：结合收录的爆款笔记与封面相关数据，缩小同赛道视觉参考范围
- **📐 规律结构化**：把版式、元素、调性等拆成可复用的设计要点
- **✅ 可交付**：输出贴合笔记主题与风格偏好的落地方案（细节与命令以 `core_workflow.md` 为准）

适用于个人博主、品牌/代运营、MCN 内容组等需要稳定提升封面点击率与一致性的角色。环境需可运行 Python 及 `requests>=2.28.0`（与 workflow 中脚本说明一致）。

## 功能特性

### 核心功能

- **🔍 意图与关键词结构化**：从用户主题、类型、风格中提炼搜索与方案所需字段（执行细则见 workflow）
- **📊 同赛道样本获取**：通过 `scripts/fetch_explosive_covers.py` 按关键词等参数查询数据（时间窗与命令见 workflow）
- **🧩 方案生成**：结合用户需求与数据侧特征，输出可落地的封面设计思路与配套说明（含报告场景时的格式约束）

### 特色亮点

- **📚 流程外置**：复杂步骤、术语禁忌、失败处理一律以 `references/core_workflow.md` 为单一事实来源，避免 Skill 正文与执行脱节
- **🤖 主 Agent 专属**：保证上下文与工具链完整，不拆给子 Agent

## 使用指南

### 基础路径

#### 第 1 步：确认是否进入「执行流程」

用户需给出**明确的创作主题/赛道**、**明确的封面设计或分析诉求**，或提供**可分析的文案/图片**。若仅问「能做什么」「怎么用」，只介绍能力，不跑数据、不编造案例（与 workflow 触发规则一致）。

> 用户：帮我做一篇职场穿搭笔记的封面，要专业一点、适合点击
> 助手：先按 workflow 做意图解析与关键词；再执行数据查询与方案输出；术语统一为「爆款封面」等规范表述。

若用户只说「做个封面」：**追问主题、赛道、风格或参考方向**，勿自行假定品类。

#### 第 2 步：执行完整流程前必读

打开并遵循 `references/core_workflow.md`（含脚本命令、数据范围、自检清单）。HTML 报告需同时满足 `references/xhs_trend_data_format.md`。

#### 第 3 步：数据与输出

按 workflow 调用 `scripts/fetch_explosive_covers.py`；不可用时**说明原因**，**不得虚构**互动数据或封面来源。

### 常用命令速查

| 命令示例                                                         | 功能                                                                |
| ---------------------------------------------------------------- | ------------------------------------------------------------------- |
| `python scripts/fetch_explosive_covers.py --keyword <k1,k2,...>` | 按关键词查询爆款封面相关数据（完整参数与示例见 `core_workflow.md`） |

**失败或无法执行时**：说明是缺参数、数据不可用、环境限制或触发条件未满足；禁止编造数据与截图链接。

## 注意事项

- **数据**：来源与时间范围以 `core_workflow.md` 为准；接口或脚本失败时如实告知，不填充假数据。
- **合规与表述**：遵守平台与社区规范；对外统一使用 workflow 规定的术语（如「爆款封面」），禁用已列出的替代表述。
- **边界**：未满足触发条件时不执行拉数与方案生成；子 Agent 不承接本技能完整流程。
