## Use Case: <br>
Developers and engineers use this skill to configure Acorn project paths, verify and reverify .ac proof files, generate training data, and produce Acorn library documentation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Incorrect or overly broad ACORN_LIB and ACORN_PROJECT paths could cause Acorn commands to run over unintended local directories and expose command output from those projects. <br>
Mitigation: Validate both paths before setup, keep them scoped to the intended Acorn library and project, and review command output before sharing it. <br>
