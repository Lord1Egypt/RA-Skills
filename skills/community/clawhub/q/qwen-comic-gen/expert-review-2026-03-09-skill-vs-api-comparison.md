### 方案 B：直接调用 API（推荐开发者）

**流程：**
```
1. 注册阿里云百炼（5 分钟）
   ↓
2. 创建 API Key（2 分钟）
   ↓
3. 阅读 API 文档（30 分钟）
   ↓
4. 编写代码（1-2 小时）
   - HTTP 请求构建
   - 签名认证
   - 异步任务轮询
   - 错误处理
   - 图片下载
   ↓
5. 测试调试（30 分钟）
   ↓
6. 部署运行
```

**示例代码（Python）：**
```python
import requests
import time

# 1. 创建任务
response = requests.post(
    "https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation",
    headers={
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    },
    json={
        "model": "wan2.6-t2i",
        "input": {"prompt": "可爱的小龙虾超级英雄，日式漫画封面"},
        "parameters": {"size": "1280*1280", "n": 1}
    }
)

task_id = response.json()["output"]["task_id"]

# 2. 轮询状态（每 5 秒检查一次）
while True:
    response = requests.get(
        f"https://dashscope.aliyuncs.com/api/v1/tasks/{task_id}",
        headers={"Authorization": f"Bearer {API_KEY}"}
    )
    
    status = response.json()["output"]["task_status"]
    
    if status == "SUCCEEDED":
        image_url = response.json()["output"]["results"][0]["url"]
        break
    elif status == "FAILED":
        print("任务失败")
        break
    
    time.sleep(5)

# 3. 下载图片
image_data = requests.get(image_url).content
with open("output.png", "wb") as f:
    f.write(image_data)
```

**优势：**
- ✅ **完全自定义** - 想怎么写就怎么写
- ✅ **透明** - 清楚每一步发生什么
- ✅ **灵活集成** - 可以集成到自己的系统
- ✅ **不依赖第三方** - 技能停了也能用

**劣势：**
- ❌ **代码量大** - 100+ 行代码
- ❌ **学习成本高** - 需要读文档、调试
- ❌ **错误处理复杂** - 各种异常要自己处理
- ❌ **轮询逻辑** - 异步任务要自己等待
- ❌ **参数优化** - 要自己摸索最佳实践

**适合人群：**
- ✅ 开发者
- ✅ 需要深度定制
- ✅ 集成到生产系统
