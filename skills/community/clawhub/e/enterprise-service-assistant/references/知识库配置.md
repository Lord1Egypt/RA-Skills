## 部署方式选择

### 方式A — 本地文件（推荐个人/单人使用）

1. 将你的 Excel 台账文件放到 workspace 的 `knowledge-base/` 目录下
2. 或在 `references/` 模板文件中填入真实数据
3. 确认工作表名称与上表一致

### 方式B — 腾讯文档（推荐团队/多人协作）

1. 在腾讯文档创建多个智能表格，工作表名与上表一致
2. 参考 `references/` 下的字段映射文件确认列结构
3. 将文档ID填入下方「腾讯文档链接」表格
4. 团队共同维护，Agent 实时读取最新数据
5. 支持灵活拆分：可以一张表放全部数据，也可以按工作表拆成多张表

### 方式C — IMA知识库（推荐长期积累）

1. 在 IMA 知识库中创建对应知识空间
2. 将企业档案、费用标准、合同模板等数据上传至 IMA
3. Agent 通过 IMA MCP 连接读取知识库内容
4. 支持持续更新：日常文件直接丢给专家，自动格式化后上传 IMA 沉淀

> 💡 IMA 知识库特别适合「日常更新、长期积累」的数据场景，比如走访记录、企业档案、政策文件等。

## 🔒 数据安全说明

| 安全原则 | 说明 |
|----------|------|
| **数据不外发** | 所有知识库数据存储在本地，不会发送到任何服务器 |
| **腾讯文档仅读取** | Agent 仅读取你提供的文档链接，不获取任何其他信息 |
| **workspace 优先** | 真实数据建议放在 `workspace/knowledge-base/` 目录，与 Skill 包物理隔离 |
| **模板不含数据** | 模板文件不包含任何真实业务数据，可安全分享 |

> ⚠️ 重要提醒：安装本 Skill 时，请勿将含真实数据的 `workspace/knowledge-base/` 目录分享给他人。
> 如需迁移，重新在新环境填入数据即可。

## 首次配置引导（新用户必读）

欢迎使用企服助手！使用前需要完成以下两步：

**第1步：准备数据文件**
- 准备好包含以上7个工作表的 Excel 文件
- 确保工作表名称与「数据源说明」表格一致
- 直接发送给企服助手，AI 会自动格式化

**第2步：选择数据源方式**
- 单人使用 → 将 Excel 文件放到 workspace 的 `knowledge-base/` 目录
- 团队使用 → 在腾讯文档创建智能表格，链接填入「腾讯文档链接」表格
- 长期积累 → 上传至 IMA 知识库，填入「IMA知识库链接」表格

**第3步：确认配置**
- 发送：`@企服助手 检查配置`
- Agent 会验证数据是否就位
