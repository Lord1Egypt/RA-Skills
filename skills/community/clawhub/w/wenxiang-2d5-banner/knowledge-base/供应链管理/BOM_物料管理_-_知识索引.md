## 🎯 核心知识点

### 物料分类（7 种）

| 代码 | 名称 | 类型 | 物权 | 毛需求 | 核心特征 |
|------|------|------|------|--------|----------|
| B | 外购总成 | 采购级 | 主机厂 | ✅ | 直接采购，直接装配 |
| HC | 委外总成 | 采购级 | 主机厂 | ✅ | 委外装配，总成回厂 |
| V | VAA 双经销 | 采购级 | 主机厂（经手） | ✅（归入 B） | 主机厂先买后卖 |
| C | CSMT 零件 | 委外子件 | 供应商 | ❌ | Consignment，供应商物权 |
| D | DB 件 | 指定子件 | 供应商 | ❌ | 主机厂指定，上级采购 |
| S | 散件 | 普通子件 | 供应商 | ❌ | 自由选择，只看结构 |
| M | 自制件 | 内部生产 | 主机厂 | ❌ | 工厂自产 |

### 核心公式

```
最全总成件毛需求量 = B（外购总成） + HC（委外总成）
```

**解释**:
- C/D/S 都是子零件，价值已包含在 HC 中，重复计算会导致库存积压
- V 虽经手但本质是采购级总成，归入 B 类
- M 走内部生产计划，不计入外部采购需求

### 判断决策树

```
新零件
  ↓
是否主机厂生产？
  ├─ 是 → M 自制件
  └─ 否 → 是否采购级？
      ├─ 否 → 是否主机厂指定？
      │   ├─ 是 → D 指定子件
      │   └─ 否 → 是否工程管控？
      │       ├─ 是 → S 散件
      │       └─ 否 → C 委外子件
      └─ 是 → 是否总成？
          ├─ 否 → V 双经销
          └─ 是 → 是否委外装配？
              ├─ 否 → B 外购总成
              └─ 是 → HC 委外总成
```

## 💡 专家洞察

### 洞察 1：为什么 C 不算毛需求？
**物权视角**: C 的物权从头到尾都在供应商手中，主机厂从未拥有过 C  
**业务视角**: 主机厂要的是 HC 总成（装配完成的成果），不是 C 零件（中间原材料）  
**财务视角**: 如果 C 和 HC 都算，就重复计算了——C 的价值已经包含在 HC 里了

### 洞察 2：VAA 双经销的本质是什么？
**商务过桥**: 当专利/商务壁垒导致总成商无法直接采购零件 A 时，主机厂充当"中间商"  
**风险隔离**: 通过主机厂经手，规避了总成商与零件 A 供应商之间的直接交易障碍  
**财务处理**: 虽然 V 要经手，但最终归入 B 类需求（因为 V 本身就是采购级总成）

### 洞察 3：DB 件的"指定"意味着什么？
**权力博弈**: 主机厂通过指定供应商/定价，保留对关键子零件的控制权，但不承担采购执行责任  
**供应链管理**: 防止总成商为了降低成本而选用劣质子零件，确保产品质量  
**责任划分**: 虽然主机厂指定，但采购合同是总成商与 DB 供应商签的，出了问题找总成商
