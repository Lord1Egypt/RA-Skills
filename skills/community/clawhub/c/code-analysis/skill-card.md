## Description: <br>
Perform multi-dimensional advanced code analysis including knowledge graph generation, code quality evaluation, performance analysis, security review, architecture review, and test coverage analysis, outputting structured reports with actionable recommendations. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[openlark](https://clawhub.ai/user/openlark) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to review user-provided code for quality, security, performance, architecture, knowledge-graph relationships, and test coverage, then receive prioritized recommendations and an improvement roadmap. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: User-provided code may contain secrets or proprietary material. <br>
Mitigation: Point the agent at specific files or directories and exclude secrets or sensitive proprietary code unless that content is approved for review. <br>
Risk: Generated findings, risk assessments, or code examples may be incomplete or incorrect. <br>
Mitigation: Review the report before relying on it, especially before applying security, performance, or architecture changes. <br>
