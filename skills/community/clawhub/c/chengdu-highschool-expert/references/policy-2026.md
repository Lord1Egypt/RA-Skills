# 2026年成都中考政策解读

## 使用说明
本文档解读2026年成都市中考政策，供初升高分析参考。
**重要提醒**：政策每年可能调整，请务必核实最新官方文件。

**数据更新时间**：2025年12月（基于2025年政策推测2026年）
**官方信息来源**：成都市教育局官网 (http://edu.chengdu.gov.cn)

---

## 一、录取批次

成都中考录取分为5个批次：

### 第一批：指标到校生
- **市级指标**：顶尖高中（七中、树德、石室等）分配到各初中的名额
- **区域指标**：各区属重点高中分配到本区初中的名额
- **录取方式**：不需参加中考，但需通过初中学校的指标选拔
- **优势**：提前锁定高中学位，减少中考压力

### 第二批：普通高中（重点高中）
- 第一梯队、第二梯队高中主要在此批次录取
- 可填报1-3个志愿（统招+调剂）
- **统招**：户籍学籍均在本区，优先级高
- **调剂**：户籍或学籍不在本区，名额较少

### 第三批：普通高中（一般高中）
- 第三梯队、第四梯队高中
- 录取线相对较低

### 第四批：职普融通班、中职学校
- 职业教育方向
- 部分学校提供"职普融通"通道（可转普高）

### 第五批：补录
- 前四批录取结束后，未招满的学校进行补录
- 滑档考生的最后机会

## 二、户籍与学籍政策

### 户籍学籍一致
- **定义**：户籍和学籍在同一区
- **优势**：可报考本区所有高中（统招优先级）
- **示例**：户籍在高新区，学籍也在高新区 → 可报考高新区所有高中（统招）

### 户籍学籍不一致
- **定义**：户籍和学籍不在同一区
- **政策**：
  - 可报考学籍所在区的高中（统招或调剂）
  - 可报考面向全市招生的高中（如七中、树德等）
  - 通常不能报考户籍所在区的高中（除非有特殊政策）
- **示例**：户籍在锦江区，学籍在高新区 → 可报考高新区高中 + 全市招生高中，但不能报考锦江区高中（统招）

### 随迁子女
- **定义**：父母在成都工作但无成都户籍的初三学生
- **政策**：
  - 需提供居住证、社保缴纳证明等材料
  - 通常按学籍所在区报考（调剂生身份）
  - 部分区域有特殊政策，需咨询教育局
- **重要**：随迁子女一般不能享受指标到校政策

### 回城生
- **定义**：户籍在成都，但学籍在外地的初三学生
- **政策**：
  - 需回户籍所在区报名参加中考
  - 通常不享受指标到校政策（部分区有例外）
  - 需提前办理报名手续（通常3月前）
- **建议**：提前咨询户籍所在区教育局

## 三、指标到校政策

### 市级指标
- **分配学校**：七中、树德、石室等顶尖高中
- **分配对象**：各初中学校（按初中教学质量分配名额）
- **选拔方式**：初中学校内部选拔（通常看一诊/二诊成绩）
- **优势**：
  - 不需参加中考，提前锁定学位
  - 进入顶尖高中的最佳途径
- **风险**：一旦放弃指标，不能参加中考录取（部分区有例外）

### 区域指标
- **分配学校**：各区属重点高中（如川大附中、列五中学等）
- **分配对象**：本区初中学校
- **选拔方式**：初中学校内部选拔
- **优势**：提前锁定学位，减少中考压力
- **风险**：同市级指标

### 指标到校 vs 统招对比

| 维度 | 指标到校 | 统招 |
|------|---------|------|
| 录取方式 | 初中选拔，不需中考 | 凭中考成绩录取 |
| 压力 | 小（提前锁定） | 大（一考定乾坤） |
| 学校选择 | 受限（只能选分配的学校） | 自由（可报所有符合政策的学校） |
| 风险 | 放弃后不能中考（部分区） | 滑档风险 |
| 适合人群 | 成绩稳定，有把握获得指标 | 成绩波动大，或想冲更好学校 |

**建议**：
- 如果能获得七中/树德/石室的指标 → 建议接受
- 如果只能获得一般高中的指标，但一诊/二诊成绩较好 → 可考虑放弃指标，冲统招

## 四、特殊政策

### 加分政策
- **少数民族**：通常加5-10分（需提供证明）
- **军人子女**：根据父母军衔和服役地区，加10-20分
- **公安英模子女**：通常加10-20分
- **抗疫一线医务人员子女**：2020-2022年有政策，2026年可能取消

**注意**：加分政策每年可能调整，需查看当年官方文件。

### 特长生政策
- **类别**：艺术、体育、科技
- **录取方式**：
  - 需参加特长生专业测试
  - 中考成绩达到最低控制线即可录取
  - 名额较少，竞争激烈
- **风险**：一旦被特长生录取，通常不能转普通班

## 五、志愿填报策略

### "冲-稳-保"原则
- **冲刺志愿**：比一诊/二诊成绩高10-20分的学校（录取概率20-30%）
- **稳妥志愿**：与一诊/二诊成绩相当的学校（录取概率60-80%）
- **保底志愿**：比一诊/二诊成绩低15-30分的学校（录取概率>90%）

### 常见误区
1. **只报顶尖高中** → 滑档风险极高
2. **不报保底学校** → 一旦滑档，只能参加补录
3. **忽略调剂风险** → 若为调剂生，录取概率降低
4. **盲目跟风** → 别人报什么就报什么，不考虑自身情况

## 六、2026年政策变化预测

**可能的变化**（基于近年趋势推测）：
1. **指标到校比例可能增加** → 统招名额相对减少
2. **中考题目可能更注重能力考察** → 死记硬背得分难度增加
3. **职业教育地位提升** → 职普融通班可能更受欢迎
4. **随迁子女政策可能收紧** → 需提前准备材料

**重要**：以上为推测，务必以2026年官方文件为准！
