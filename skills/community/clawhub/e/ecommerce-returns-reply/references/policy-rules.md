# 电商平台退换货规则参考

## 一、退货规则

| 条件 | 规则 |
|---|---|
| 七天无理由退货 | 自签收次日起7天内可申请，商品需保持原状（未使用、未洗涤、吊牌未剪、包装完整） |
| 质量问题退货 | 不受7天限制，需提供照片/视频凭证，运费由卖家承担 |
| 定制类商品 | 不支持七天无理由退货，质量问题除外 |
| 生鲜/食品类 | 签收后不支持无理由退货，变质/破损可凭凭证退货 |
| 虚拟商品 | 不支持退货（充值、会员、数字内容等） |

## 二、换货规则

| 条件 | 规则 |
|---|---|
| 尺码/颜色不合 | 签收7天内可申请换货，运费买家承担（除非卖家承诺包邮换） |
| 发错货/漏发 | 签收7天内申请，运费卖家承担 |
| 质量问题换货 | 不受7天限制，运费卖家承担 |
| 无库存换货 | 引导退款或等待补货，优先退款 |

## 三、退款时效

| 阶段 | 时效 |
|---|---|
| 卖家审核 | 提交后48小时内 |
| 买家退货发货 | 审核通过后7天内寄回并填写物流单号 |
| 卖家确认收货 | 签收退货后48小时内 |
| 退款到账 | 确认后1-3个工作日（原路退回） |

## 四、运费承担原则

- **卖家原因**（质量、发错、描述不符）→ 卖家承担运费
- **买家原因**（不喜欢、尺码不合）→ 买家承担运费
- **运费险** → 按保险规则理赔，先行垫付后理赔到账

## 五、常见拒绝情形

1. 超过7天无理由退货期限且非质量问题
2. 商品已使用/损坏/影响二次销售
3. 缺少配件、赠品或原包装
4. 买家人为损坏
5. 特殊商品类别（定制、生鲜、虚拟）
