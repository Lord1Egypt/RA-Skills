# 费曼热学：到底在说什么

## 第一性原理

热学研究的是**能量在微观层面的行为**。

- 温度：微观粒子运动剧烈程度的度量
- 热量：能量传递的方式
- 内能：物体内部所有分子动能 + 分子势能的总和

**热力学第零定律：** 热平衡是传递热量时温度相等的状态。

## 温度与分子动理论

### 温度的本质

> **温度是系统平均动能的度量。**

对理想气体：T ∝ 平均动能
> **Ēk = (3/2)kT**

k是玻尔兹曼常数，T是热力学温度。
温度不是"冷热程度"的主观感觉，而是**分子平均动能**的客观量度。

### 分子动理论三条基本假设

1. 物质由大量分子组成，分子间有间隙
2. 分子永不停息做无规则运动（布朗运动是其宏观表现）
3. 分子间存在相互作用力（引力和斥力，同时存在）

### 分子动能

**平均动能只由温度决定。**
温度越高，分子平均动能越大——无论是什么物质。

**分子平均动能 ≠ 物体的动能。**
是所有分子动能的总和，再除以分子数。

## 热量与热传递

### 热量（Q）

热量是能量传递的量，不是物体拥有的能量。

**比热容：** 单位质量物质升高（或降低）1℃吸收（或放出）的热量。
> **Q = cmΔT**

- c：比热容（J/(kg·℃)）
- m：质量
- ΔT：温度变化

**水的比热容最大（4.2×10³ J/(kg·℃)）。**
这是为什么水能调节温度——同样吸热，水温变化最小。

### 三种热传递方式

| 方式 | 发生条件 | 传热介质 |
|------|---------|---------|
| 热传导 | 直接接触 | 固体（金属最显著） |
| 热对流 | 流体流动 | 液体和气体 |
| 热辐射 | 任何温度 | 不需要介质（电磁波） |

**热传导的微观本质：** 高温物体分子动能大，通过分子碰撞把能量传给低温物体。

**热对流的本质：** 温度不均导致流体密度不均，密度差产生浮力驱动流动。

**热辐射：** 任何物体都在向外辐射电磁波，同时吸收辐射。温度越高，辐射越强，辐射峰值频率越高（维恩位移定律）。

## 内能

### 内能的组成

> **U = E分子动能 + E分子势能**

- 分子动能：由温度决定
- 分子势能：由分子间距决定（跟体积有关）

**理想气体的内能：**
理想气体假设分子无相互作用（无势能）→ U只由温度决定。
> **U = n·(3/2)RT**

对实际气体，内能还和体积有关。

### 改变内能的两种方式

1. **热传递**（Q）：能量从高温物体流向低温物体
2. **做功**（W）：机械功转化为内能

**热力学第一定律（能量守恒）：**
> **ΔU = Q + W**

内能变化 = 吸收热量 + 外界对物体做功。

**符号约定（中学常用）：**
- Q：吸收为正，放出为负
- W：外界对物体做功为正，物体对外做功为负
- ΔU：内能增加为正，减少为负

### 做功和热传递的区别

- **做功：** 宏观机械运动转化为内能（用力推动、摩擦、压缩）
- **热传递：** 微观粒子碰撞传递能量，不需要宏观运动

**等效但本质不同：** 两者改变内能的效果相同，但途径不同。

## 热力学第一定律的应用

### 等容过程（V = 常数）

体积不变 → W = 0
> **ΔU = Q**

吸收的热量全部用于增加内能。

### 等压过程（P = 常数）

体积改变 → 有做功
> **ΔU = Q - PΔV**

吸收的热量一部分用来对外做功，一部分增加内能。

### 等温过程（T = 常数）

理想气体内能不变 → ΔU = 0
> **Q = W**

吸收的热量全部用于对外做功。

### 绝热过程（Q = 0）

没有热交换
> **ΔU = W**

外界对物体做功，全部转化为内能。

## 热力学第二定律

### 为什么需要第二定律

第一定律说能量守恒，但没说不可能发生什么。
第二定律规定了**能量转化的方向**。

### 两种表述（等价）

**克劳修斯表述（热传导方向）：**
> 热量不能自发地从低温物体传到高温物体。

**开尔文表述（热功转化方向）：**
> 不可能从单一热源吸热，使之完全变为有用的功而不产生其他影响。

### 第二定律的本质

**熵增加原理：**
> 孤立系统的熵（无序度）永不减少。

热量自发从高温→低温，是因为这个方向熵增加。
热机不可能100%效率，是因为总有热量散逸到低温环境，熵增加。

### 热机效率

> **η = W/Q₁ = (Q₁ - Q₂)/Q₁ = 1 - Q₂/Q₁**

- Q₁：吸热（高温热源）
- Q₂：放热（低温热源）
- W = Q₁ - Q₂：对外做功

**卡诺热机（最理想的热机）：**
> **η = 1 - T₂/T₁**

效率取决于两个热源的温度比。T₂→T₁时，效率→1，但不可能达到1（第二类永动机不可能）。

## 物态变化与潜热

### 相变

- 熔化（固→液）：吸热，内能增加，分子势能增加
- 凝固（液→固）：放热
- 气化（液→气）：吸热
- 液化（气→液）：放热
- 升华（固→气）：吸热
- 凝华（气→固）：放热

### 潜热

> **Q = Lm**

L是潜热（单位质量相变时吸收/放出的热量）。
**注意：** 相变过程中温度不变，但内能在变（分子势能在变）。

## 常见误解

### ❌ 误解1："温度高就是热量多"

不对。温度是"激烈程度"，热量是"传递的能量"。
一杯100℃的水和一桶100℃的水，内能差很多——但温度一样。

### ❌ 误解2："热量是物体含有的能量"

热量不是物体拥有的，是**传递过程中的量**。
说"物体含有多少热量"是错的。物体含有的是**内能**。

### ❌ 误解3："第二类永动机不可能是因为能量不守恒"

不对。第二类永动机违反的是第二定律，不是第一定律。
它承认能量守恒，只是想从单一热源100%转化为功——这被第二定律禁止。

### ❌ 误解4："热传导和热对流是同一种现象"

不同。热传导需要介质（固体内部），热对流是流体整体流动（需要介质运动）。

### ❌ 误解5："制冷机把热量从低温搬到高温违反了第二定律"

不违反。第二定律说热量不能**自发地**从低温传到高温。
制冷机通过外界做功（压缩机耗电），实现了热量从低温→高温——这不是自发，所以不违反。

## 热学这张网的连接点

```
热学
├── 连接力学：热功当量→功和热是能量传递的两种方式
├── 连接电磁学：热辐射是电磁波
├── 连接量子：黑体辐射→普朗克量子假设→量子论诞生
└── 连接统计物理：熵是系统混乱度的统计量度
```

## 费曼式理解热学的关键问题

1. **内能变化了多少？是吸热、放热、还是做功导致的？**
2. **过程是等容、等压、等温还是绝热？**
3. **能量转化的方向是什么？第二定律允许这个方向吗？**
4. **有没有相变？相变时温度不变，内能变了没有？**
