# 塔罗牌阵详解（Spreads）

牌阵是塔罗解读的骨架——它决定了每张牌"回答什么问题"。

---

## 一、三张牌阵（Three-Card Spread）

**最简单、最常用的牌阵。** 适用于任何日常问题。

### 基础解读模式（三种变体）

#### 变体A：过去—现在—未来（时间线）
| 牌位 | 含义 |
|------|------|
| 1（左） | 过去的影响因素 |
| 2（中） | 当前状态 / 核心问题 |
| 3（右） | 未来发展 / 可能结果 |

#### 变体B：身心灵（状态层）
| 牌位 | 含义 |
|------|------|
| 1 | 身体/物质层面 → 现实中的实际状况 |
| 2 | 心智层面 → 你的想法、信念、判断 |
| 3 | 灵性层面 → 潜意识的指引、更高视角 |

#### 变体C：情境—行动—结果（决策）
| 牌位 | 含义 |
|------|------|
| 1 | 当前情境 / 核心问题 |
| 2 | 建议的行动 / 需要做什么 |
| 3 | 可能的结果 / 发展走向 |

### 进阶技巧
- 三张牌可以再加一张"底牌"（牌堆最底部的牌）作为潜意识的基调
- 若第三张不理想，可以抽一张"补充牌"放在旁边询问"如何改善结果"
- 三张大阿卡纳 = 命运层面的重大主题；全是小阿卡纳 = 日常生活的细节

---

## 二、凯尔特十字牌阵（Celtic Cross）— 10张牌

**塔罗中最经典、最全面的牌阵。** 适用于重大人生问题的深度剖析。

### 牌位布局（从求问者视角）

```
        3          10
         |          |
    5    1+2    6   9
         |          
        4          8
                   |
                   7
```

### 详细牌位含义

| 牌位 | 位置 | 名称 | 含义 |
|------|------|------|------|
| 1 | 中心（横盖） | 当下之心 | 当前局面的核心 / 求问者的状态 |
| 2 | 中心（交叉） | 横贯之力 | 交叉于牌1之上——面对的挑战 / 阻碍 / 助力 |
| 3 | 上方 | 意识之冠 | 求问者意识到的目标 / 理想 / 最好可能 |
| 4 | 下方 | 根基 | 过去的基础 / 潜意识的影响 / 已知的根源 |
| 5 | 左侧 | 过往 | 正在离开的过去 / 近期影响 |
| 6 | 右侧 | 将临 | 即将到来的未来 / 下一步 |
| 7 | 右侧纵列（下） | 自我 | 求问者对自己的看法 / 态度 |
| 8 | 右侧纵列（中） | 环境 | 外部环境 / 他人看法 / 影响因素 |
| 9 | 右侧纵列（上） | 希望与恐惧 | 最大的希望或恐惧——内心的投射 |
| 10 | 最上方 | 最终结果 | 综合以上所有因素的可能结果 |

### 解读逻辑
1. **先看中心十字（1+2）**：问题的核心是什么？
2. **再看纵轴（3→1→4）**：意识→当下→潜意识，形成完整的能量流
3. **左→右（5→6）**：时间线的流动方向
4. **自我之柱（7→8→9）**：内心的层层递进——自我认知→外部影响→深层恐惧/渴望
5. **最终牌（10）**：综合解读——这不是"注定的结局"，而是在当前轨迹下的最可能结果

## 三、关系牌阵（Relationship Spread）— 7张牌

**适用于：亲密关系、合作关系、友谊、家庭关系。**

### 牌位布局

```
      7
   3     4
1     2     5
      6
```

### 详细牌位含义

| 牌位 | 含义 |
|------|------|
| 1 | 你自己的状态、需求、在这段关系中的位置 |
| 2 | 对方的状态、需求、在这段关系中的位置 |
| 3 | 关系的共同基础——你们连接在一起的原因 |
| 4 | 关系中的优势/和谐之处 |
| 5 | 关系中的挑战/摩擦点 |
| 6 | 关系的潜在发展方向（短期） |
| 7 | 关系的高层指引/灵性建议——宇宙对这段关系的"点评" |

### 解读技巧
- 对比牌1和牌2：谁在关系中投入更多？价值是否匹配？
- 如果牌3是大阿卡纳，这段关系是命运安排的功课
- 如果牌5是宝剑牌——沟通问题是核心矛盾
- 如果牌5是圣杯牌——情感失衡是核心矛盾

## 四、黄道十二宫牌阵（Zodiac Spread）— 13张牌

**最宏大的牌阵。** 适用于：年度运势、人生全景、全面自我审视。

### 牌位布局（圆形，第一宫起于9点钟方向，顺时针）

```
        10   9   8
     11            7
   12       13       6
      1            5
        2   3   4
```

### 十二宫对应牌位

| 牌位 | 宫位 | 主宰领域 |
|------|------|----------|
| 1 | 第一宫（ASC / 白羊） | 自我形象、外表、人格面具、他人第一印象 |
| 2 | 第二宫（金牛） | 财务、价值体系、物质安全感、才能 |
| 3 | 第三宫（双子） | 沟通、学习、短途旅行、兄弟姐妹、邻里 |
| 4 | 第四宫（巨蟹） | 家庭、根源、童年、父亲/母亲、内在安全感 |
| 5 | 第五宫（狮子） | 创造力、爱情、孩子、娱乐、自我表达 |
| 6 | 第六宫（处女） | 工作、健康、日常习惯、服务、宠物 |
| 7 | 第七宫（天秤） | 伴侣关系、婚姻、合作、公开的敌人 |
| 8 | 第八宫（天蝎） | 深度转化、共享资源、性、死亡与重生、玄秘 |
| 9 | 第九宫（射手） | 高等教育、长途旅行、哲学、信念、灵性 |
| 10 | 第十宫（MC / 摩羯） | 事业、社会地位、公众形象、人生目标 |
| 11 | 第十一宫（水瓶） | 社群、朋友、集体理想、希望与愿景 |
| 12 | 第十二宫（双鱼） | 潜意识、隐藏敌人、自我消解、灵性连接、业力 |
| 13 | 中心/核心牌 | 贯穿12领域的年度主旋律 |

### 解读逻辑
- 先看中心牌（13号）：今年的主旋律是什么？（若是大阿卡纳，说明这一年有"功课"要做）
- 重点看与求问者最关切的宫位（如问事业→重点看第6/10宫；问爱情→第5/7宫）
- 哪些宫位出现了逆位牌/宝剑牌→那是今年的"雷区"
- 哪些宫位出现了太阳/世界/圣杯牌→那是今年的"礼物"

## 牌阵选择速查表

| 问题类型 | 推荐牌阵 | 张数 |
|----------|----------|------|
| 日常快速占卜 | 三张牌（过去-现在-未来） | 3 |
| 决策困局 | 三张牌（情境-行动-结果） | 3 |
| 身心状态 | 三张牌（身-心-灵） | 3 |
| 亲密关系 | 关系牌阵 | 7 |
| 年度运势 | 黄道十二宫 | 13 |
| 人生重大转折 | 凯尔特十字 | 10 |
| 简单是/否问答 | 单张牌 + 正逆位判断 | 1 |
| 自我探索 | 凯尔特十字 或 黄道十二宫 | 10-13 |
