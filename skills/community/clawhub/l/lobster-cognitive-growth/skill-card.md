## Description: <br>
Train an AI lobster or agent to improve intelligence, cognition, learning ability, reasoning discipline, memory use, and adaptive personality through repeated cognitive drills, reflection, self-evaluation, and owner feedback loops. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and agent developers use this skill to run repeated cognitive drills, learning journals, hypothesis loops, and feedback practices that help an agent improve reasoning discipline, memory use, self-correction, and owner interaction over time. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Charenix persistence may read or write agent history, journals, weak spots, strategies, and related learning records. <br>
Mitigation: Use a dedicated, revocable CHARENIX_AGENT_KEY, avoid sensitive personal details in journals, and review stored memories and strategies periodically. <br>
Risk: The README install path fetches SKILL.md from a remote URL. <br>
Mitigation: Inspect the remote SKILL.md before installation and pin a reviewed version when deploying. <br>
