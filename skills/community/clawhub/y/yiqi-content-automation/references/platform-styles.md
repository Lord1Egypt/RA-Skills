# 各平台算法偏好与排版规范

## 小红书

### 算法偏好
- **完播率**优先：开头必须强钩子，前3行决定是否展开阅读
- **互动率**权重高：评论>收藏>点赞，多引导评论
- **关键词密度**：标题+正文重复出现目标关键词2-3次
- **发布时间**：工作日 12:00-13:00、18:00-20:00；周末 9:00-11:00

### 排版规范
```
标题：28字以内，含1-2个emoji，含核心关键词
正文：
- 开头：空1行，emoji开头
- 每个要点：单独成段，前置emoji
- 段落间距：空1行
- 结尾：互动引导语
- 标签：#大话题（10万+）#小话题（精准）共5-8个
```

### 禁忌
- 不要出现外链
- 避免直接提及竞争平台名称
- 图片数量建议3-9张（9张最佳）
- 避免纯文字，图片必须与内容相关

---

## 抖音/短视频

### 算法偏好
- **完播率**：最核心指标，前3秒=生死线
- **互动率**：评论>分享>点赞，引导互动
- **关键词**：标题+字幕中出现目标词
- **发布时间**：7:00-9:00、12:00-14:00、18:00-21:00

### 字幕/脚本规范
```
- 每句不超过15字
- 句尾停顿明显，方便切片
- 口语化表达（"真的"、"绝了"、"没想到"）
- 数字具体化（"3天"而非"几天"）
- 避免长定语从句
```

### 封面文字规范
- 字数：8-12字
- 字体：粗体，高对比色
- 内容：与标题呼应，含悬念或利益点

## 微信公众号

### 算法偏好
- **打开率**：标题决定，AB测试首选
- **阅读完成率**：排版舒适度影响大
- **分享率**：内容引发情感共鸣或实用价值
- **发布时间**：周二至周四 9:00-11:00、17:00-20:00

### 排版规范
```
标题：18-30字，含数字或悬念
副标题（可选）：补充说明
正文结构：
- 引入段（100字内）：场景代入或痛点
- 主体段落：每段200-400字，段间空行
- 小标题：加粗，概括段落核心
- 结尾：总结+引导关注/转发
配图：每800字配1张图，图片横向居中
```

### 加分项
- 设置"阅读原文"跳转链接
- 评论区引导话术（置顶评论）
- 末尾放置公众号名片

## B站

### 算法偏好
- **完播率+互动率**：权重最高
- **弹幕密度**：引导弹幕互动
- **三连率**（点赞+收藏+投币）
- **发布时间**：周五-周日 17:00-21:00效果最佳

### 简介/脚本规范
```
标题：20-50字，含系列标识（【第X期】/「XX系列」）
简介：
- 第1段：内容摘要（50字内）
- 第2段：章节时间轴（重要）
- 第3段：引导三连和关注
- 第4段：联系方式（可选）
标签：选精准长尾标签，避免泛标签
分区：选择最精准的分区投稿
```

### 弹幕引导点
在脚本中标注弹幕引导时机：
- 抛出问题处："你们觉得XX怎么样，弹幕聊聊"
- 干货总结处："记得三连，不然忘了回来找不到"
- 有争议处："弹幕里说说你们的想法"

## 微博

### 算法偏好
- **话题参与**：挂热门话题引流
- **@互动**：@相关博主/品牌增加曝光
- **发布时间**：9:00-10:00、12:00-13:00、21:00-22:00

### 排版规范
```
字数：140字（正文）+9张图（上限）
结构：核心观点（前30字）+展开说明+话题标签
话题：#话题名#放正文或结尾，2-3个为宜
```
