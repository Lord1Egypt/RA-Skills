## Description: <br>
Bootstrap a new OpenClaw agent workspace with best-practice file structure. Creates AGENTS.md, SOUL.md, HEARTBEAT.md, memory system, and configures agent discovery. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[imaflytok](https://clawhub.ai/user/imaflytok) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agent operators use this skill to initialize an OpenClaw workspace with session instructions, memory files, local tool notes, heartbeat guidance, and agent discovery metadata. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The setup script contacts a third-party server and sends the machine hostname via a network ping. <br>
Mitigation: Review scripts/init.sh before installation and remove the ping block if hostname disclosure is not acceptable. <br>
Risk: Generated workspace files include persistent agent-network instructions and OADP endpoints. <br>
Mitigation: Review the generated AGENTS.md and HEARTBEAT.md content before use and remove the Agent Network or OADP entries if ongoing network checks are not desired. <br>
