# 劳动合同审查参考手册

## 一、劳动合同法定必备条款

### 1.1 《劳动合同法》第17条规定的必备条款

| 序号 | 必备条款 | 审查要点 | 常见陷阱 |
|------|---------|---------|---------|
| 1 | 用人单位信息 | 名称与营业执照一致、统一社会信用代码、法定代表人 | 名称写简称或关联公司名称 |
| 2 | 劳动者信息 | 姓名、身份证号、住址 | 身份证号错误 |
| 3 | 合同期限 | 固定期限/无固定期限/以完成一定工作任务为期限 | 应签无固定期限而未签 |
| 4 | 工作内容和工作地点 | 岗位名称、工作职责、工作地点 | 工作地点写"全国"或过于宽泛 |
| 5 | 工作时间和休息休假 | 标准工时/综合计算/不定时 | 约定"弹性工作制"但未经审批 |
| 6 | 劳动报酬 | 基本工资、绩效、补贴、发放时间 | 只写"面议"或拆分过细 |
| 7 | 社会保险 | 依法缴纳五险 | 约定"自愿放弃缴纳"无效 |
| 8 | 劳动保护/劳动条件/职业危害防护 | 特殊岗位必须注明职业病危害 | 缺失职业危害告知 |

### 1.2 约定条款（非必须但常见）

| 序号 | 约定条款 | 审查要点 | 风险等级 |
|------|---------|---------|---------|
| 1 | 试用期 | 合同期限与试用期匹配（见下表） | 高 |
| 2 | 培训 | 服务期及违约金约定是否合法 | 中 |
| 3 | 保守秘密 | 保密范围是否合理 | 中 |
| 4 | 竞业限制 | 期限≤2年、有补偿金、范围合理 | 高 |
| 5 | 补充保险和福利 | 补充公积金、商业保险等 | 低 |
| 6 | 合同变更/解除条件 | 不得排除劳动者法定权利 | 高 |

## 二、试用期审查

### 2.1 试用期时长限制

| 合同期限 | 试用期上限 | 法定依据 |
|---------|-----------|---------|
| 不足3个月 | 不得约定试用期 | 劳动合同法第19条 |
| 3个月以上不满1年 | 不得超过1个月 | 劳动合同法第19条 |
| 1年以上不满3年 | 不得超过2个月 | 劳动合同法第19条 |
| 3年以上固定期限或无固定期限 | 不得超过6个月 | 劳动合同法第19条 |

### 2.2 试用期常见违法情形

| 序号 | 违法情形 | 法律后果 |
|------|---------|---------|
| 1 | 试用期超过法定上限 | 超出部分按转正工资支付赔偿 |
| 2 | 同一用人单位与同一劳动者约定两次试用期 | 第二次无效 |
| 3 | 试用期工资低于转正工资80% | 补足差额 |
| 4 | 试用期工资低于当地最低工资标准 | 补足差额 |
| 5 | 试用期不缴社保 | 违法，可投诉补缴 |
| 6 | 试用期内随意辞退 | 需证明不符合录用条件，否则违法解除 |

## 三、薪资条款审查

### 3.1 薪资结构审查要点

```
常见薪资结构：
基本工资 + 岗位工资 + 绩效奖金 + 津贴补贴 + 年终奖
```

**审查要点：**

1. **基本工资** — 不得低于当地最低工资标准
2. **薪资拆分** — 过度拆分（基本工资极低，绩效占比过高）影响经济补偿金基数
3. **发放时间** — 应明确每月发放日期，不得模糊
4. **试用期工资** — ≥转正工资的80%且≥当地最低工资
5. **加班费基数** — 约定加班费计算基数是否合理（不得低于最低工资）
6. **年终奖** — 是否写明发放条件、时间、离职时是否折算

### 3.2 薪资陷阱

| 陷阱 | 表现 | 风险 |
|------|------|------|
| 结构拆分 | 基本工资3000+绩效7000，实际月入10000 | 经济补偿金/赔偿金可能按基本工资计算 |
| "面议" | 合同不写具体金额 | 难以主张差额 |
| 现金发放 | 部分工资现金，不留记录 | 无法证明实际收入 |
| 扣款条款 | 约定各种扣款情形 | 违法扣款无效 |
| 浮动薪资 | "根据公司经营情况调整" | 公司可能单方降薪 |

## 四、竞业限制条款审查

### 4.1 竞业限制合法性要求

| 要件 | 法律要求 | 常见问题 |
|------|---------|---------|
| 适用人员 | 高管、高级技术人员、其他负有保密义务的人员 | 对普通员工也约定竞业限制 |
| 期限 | 不超过2年 | 约定3年或无期限 |
| 范围 | 合理界定竞争企业/行业 | 范围过宽（"不得从事任何IT行业"） |
| 地域 | 合理地域范围 | "全球""全国" |
| 补偿金 | 解除/终止合同后按月支付，不低于月工资30% | 不约定补偿金或比例过低 |

### 4.2 竞业限制风险等级

| 条款内容 | 风险等级 | 说明 |
|---------|---------|------|
| 不约定补偿金 | 🔴 高 | 劳动者可要求支付或解除竞业限制 |
| 补偿金低于30% | 🟡 中 | 劳动者可要求补足，但不当然无效 |
| 范围"同行业所有公司" | 🟡 中 | 可能因范围过宽被认定部分无效 |
| 期限超过2年 | 🔴 高 | 超出部分无效 |
| 违约金过高 | 🟡 中 | 法院可酌减 |
| 离职后3个月内未支付补偿 | 🔴 高 | 劳动者有权解除竞业限制 |

## 五、规章制度引用条款审查

### 5.1 常见问题

用人单位常在合同中写"遵守公司规章制度"，但规章制度本身可能有问题：

| 问题 | 说明 | 法律后果 |
|------|------|---------|
| 未公示 | 规章制度未经公示或告知 | 对劳动者不生效 |
| 未民主程序 | 未经职工代表大会或全体职工讨论 | 程序违法 |
| 内容违法 | 如"迟到一次扣500元" | 违法条款无效 |
| 随时修改 | 公司保留随时修改规章制度的权利 | 修改需经民主程序 |

### 5.2 审查建议

1. 要求用人单位提供完整的规章制度文本
2. 检查规章制度是否经过民主程序（职代会记录等）
3. 确认是否签收过规章制度
4. 保留签收的证据

## 六、合同解除/终止条款审查

### 6.1 违法条款识别

| 违法条款示例 | 违法原因 | 正确做法 |
|-------------|---------|---------|
| "公司可随时单方解除合同" | 排除法定解除条件 | 应符合劳动合同法第39-41条 |
| "离职需提前3个月通知" | 超出法定30日 | 劳动者提前30日书面通知即可 |
| "离职需公司批准" | 限制辞职权 | 提前30日通知即可，无需批准 |
| "离职扣发当月工资" | 克扣工资违法 | 应正常结算工资 |
| "女职工怀孕自动解除" | 违反三期保护 | 三期内不得依第40条、41条解除 |

### 6.2 合法解除/终止情形

**用人单位可解除（需支付经济补偿）：**
- 协商一致解除（《劳动合同法》第36条）
- 预告解除（第40条：医疗期满、不能胜任、客观情况变化）
- 经济性裁员（第41条）

**用人单位可解除（无需补偿）：**
- 过失性解除（第39条：试用期不合格、严重违纪、严重失职、双重劳动关系、欺诈、犯罪）

**劳动者可解除：**
- 提前30日书面通知（第37条）
- 试用期提前3日通知（第37条）
- 被迫解除（第38条：未提供劳动保护、未足额支付报酬、未缴社保、规章制度违法、欺诈胁迫）

## 七、劳动合同审查清单

### 7.1 签约前审查清单

| 序号 | 审查项 | 是否合规 | 风险提示 |
|------|--------|---------|---------|
| 1 | 用人单位名称与营业执照一致 | ☐ | |
| 2 | 合同期限明确 | ☐ | |
| 3 | 试用期时长符合法定上限 | ☐ | |
| 4 | 试用期工资≥转正工资80% | ☐ | |
| 5 | 工作内容明确 | ☐ | |
| 6 | 工作地点具体 | ☐ | 过于宽泛有风险 |
| 7 | 工资金额明确（非"面议"） | ☐ | |
| 8 | 薪资结构合理（基本工资不过低） | ☐ | |
| 9 | 社保条款（依法缴纳五险） | ☐ | |
| 10 | 工作时间明确 | ☐ | |
| 11 | 无"自愿放弃社保"条款 | ☐ | |
| 12 | 无违法解除条款 | ☐ | |
| 13 | 竞业限制（如有）补偿金≥30% | ☐ | |
| 14 | 竞业限制（如有）期限≤2年 | ☐ | |
| 15 | 服务期/培训（如有）违约金合理 | ☐ | |
| 16 | 保留合同原件 | ☐ | |
| 17 | 空白处划掉或填"无" | ☐ | |

### 7.2 合同类型选择指引

| 情形 | 建议合同类型 | 说明 |
|------|-------------|------|
| 首次入职 | 固定期限（1-3年） | 最常见 |
| 连续工作满10年 | 可要求签无固定期限 | 劳动合同法第14条 |
| 已连续签两次固定期限合同 | 第三次应签无固定期限 | 劳动合同法第14条 |
| 项目制工作 | 以完成一定工作任务为期限 | 项目完成即终止 |
| 退休返聘 | 劳务协议（非劳动合同） | 不适用劳动法 |

## 八、二倍工资情形汇总

| 序号 | 情形 | 二倍工资起算 | 最长月数 | 法律依据 |
|------|------|------------|---------|---------|
| 1 | 用工之日起超过1个月未签合同 | 满月次日起 | 11个月 | 劳动合同法第82条 |
| 2 | 合同到期后继续用工未续签 | 到期次日起 | 11个月 | 劳动合同法第82条 |
| 3 | 应签无固定期限而未签 | 应签之日起 | 无上限（至签订之日） | 劳动合同法第82条第2款 |

> **时效提醒：** 二倍工资仲裁时效各地不同，详见 labor-arbitration 技能的 regional_rules.md。
