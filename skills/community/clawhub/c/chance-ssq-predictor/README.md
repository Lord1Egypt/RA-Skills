# SSQ Predictor — 双色球彩票分析预测助手

> 🎲 基于统计学与数据分析的双色球选号辅助工具

## 功能说明

本 Skill 为中国福利彩票双色球（SSQ）提供专业的数据分析与号码推荐功能，帮助用户进行更科学的选号决策。

### 核心功能

- **遗漏值分析**：分析各号码的冷热状态
- **频率统计**：统计近期号码出现频率
- **和值分析**：识别黄金和值区间
- **AC 值分析**：评估号码组合复杂度
- **奇偶比/大小比分析**：形态分布统计
- **区间分析**：三区号码分布分析
- **连号分析**：高概率连号组合识别
- **012 路分析**：号码除3余数分类
- **蓝球分析**：蓝球冷热遗漏分析
- **综合推荐**：基于多指标加权生成推荐号码

## 使用示例

### 示例 1：生成推荐号码

```
用户：帮我选 5 注双色球
助手：执行分析并输出 5 注推荐号码组合
```

### 示例 2：遗漏值分析

```
用户：分析一下最近 30 期双色球哪些号码遗漏值较高
助手：输出遗漏值排名表，标注冷/温/热号
```

### 示例 3：和值分析

```
用户：最近双色球和值一般是多少
助手：计算最近 N 期和值分布，指出黄金和值带
```

### 示例 4：综合分析

```
用户：帮我分析双色球
助手：输出遗漏值、频率、形态统计 + 5 注推荐号码
```

## 脚本命令

如需在终端直接运行分析脚本：

```bash
# 综合分析 + 推荐号码
python scripts/ssq_predictor.py --analysis --recent 50 --generate 5

# 遗漏值分析
python scripts/ssq_predictor.py --missing

# 频率统计
python scripts/ssq_predictor.py --frequency --recent 30

# 蓝球分析
python scripts/ssq_predictor.py --blue

# 生成推荐
python scripts/ssq_predictor.py --generate 10
```
