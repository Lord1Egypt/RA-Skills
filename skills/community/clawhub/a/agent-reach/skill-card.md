## Description: <br>
Agent Reach helps an AI agent search, read, and interact with web sources across social platforms, GitHub, RSS, Exa web search, and arbitrary web pages. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent users use this skill when they need an agent to retrieve information from supported web platforms, inspect shared URLs, search online, or configure platform-specific channels. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Authenticated platform access and posting commands can act through user accounts. <br>
Mitigation: Use secondary accounts or isolated browser profiles, provide only the minimum cookies needed, and require explicit approval before posts, comments, uploads, or other account actions. <br>
Risk: Setup may rely on a remote install guide and anti-bot or proxy automation. <br>
Mitigation: Inspect the remote install guide before running it, prefer safe or dry-run setup, and require approval before proxy setup, scheduled tasks, or anti-bot automation. <br>
Risk: Broad web research can return incomplete, stale, or misleading external content. <br>
Mitigation: Review retrieved content before relying on it and cross-check important claims against authoritative sources. <br>
