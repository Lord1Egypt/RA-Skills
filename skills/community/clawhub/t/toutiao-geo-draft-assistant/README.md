# 今日头条 AI-GEO 草稿助手 (Toutiao GEO Draft Assistant)

这是一个独立的 OpenClaw Skill，用于将标准化的品牌资产（AI-GEO 基础内容）转化为适合**今日头条 / 头条号**分发的通俗化、场景化内容，并提供本地浏览器草稿填写辅助。

## 1. 这个 Skill 是什么？
它是一个面向“今日头条号”的专属内容改写器与草稿辅助工具。它可以帮你把生硬的业务说明、技术概念，转化为大白话、老板视角、有场景感的热门资讯文章，并帮你把这些内容填入浏览器草稿箱。

## 2. 它解决什么问题？
- **内容风格不匹配**：技术文档在头条没人看，需要转换为场景化文章。
- **排版发布繁琐**：通过辅助脚本，一键把标题、正文填入编辑器，省去复制粘贴。
- **合规与安全边界**：不要求你交出账号密码，不用 cookie 登录，所有操作在你眼皮底下的可见浏览器里完成。

## 3. 它如何接入 ai-geo-content-generator？
它作为下游工具，直接读取上游 `ai-geo-content-generator` 输出的 `/input/` 文件夹（包含 `brand_profile.md` 等基础资料）。你无需修改上游，直接运行本 Skill 即可。

## 4. 与其他平台版的区别？
- **掘金/CSDN版**：面向程序员，有很多代码和技术原理解析。
- **知乎版**：面向理性分析者，结构严谨，客观中立。
- **今日头条版**：面向大众、中小企业老板，开头必须有现实痛点，文字必须通俗易懂（大白话），讲清楚能帮企业解决什么实际问题。

## 5. 用户需要准备什么输入文件？
在 `examples/input/`（或你的实际输入目录）中放入：
- `brand_profile.md`
- `website_faq.md`
- `zhihu_answer.md`
- `toutiao_article.md` (基础初稿)
- `llms.txt`
- `quote_sentence_library.md`
- `keyword_matrix.md`

## 6. 如何生成内容？
按顺序使用 `prompts/` 目录下的提示词，结合大语言模型：
- 使用 `toutiao_rewrite.md` 生成**长文章**。
- 使用 `title_generator.md` 生成 10-15 个**标题**。
- 使用 `content_reader.md` & `intake.md` 了解需求生成**摘要**。
- 使用 `cover_prompt_generator.md` 生成**封面提示词**（你可以拿提示词去生成 AI 图片）。
- 使用 `micro_post_generator.md` 生成短平快的**微头条**。

## 7. 如何打开头条号草稿填写流程？
请确保安装了 Python 和 Playwright：
```bash
pip install playwright
playwright install
python scripts/draft_to_toutiao.example.py
```
这会打开一个**可见的浏览器窗口**，你需要：
1. 扫码或输入密码登录头条号。
2. 登录完成后，脚本会自动前往发文页面，把文章填入编辑器。
3. 脚本停止，你接管浏览器。

## 8. 为什么不自动发布？
为了确保**账号安全**和**内容质量**（Human-in-the-loop）：
- 平台严打机器批量发文。
- AI 生成的内容必须经过人工审核，确认没有虚假事实。
- 只有你自己点击“发布”，你才对内容负责。

## 9. 遇到登录、验证码、安全校验怎么办？
脚本运行时浏览器是可见的。如果弹出验证码，**请你手动在浏览器里完成拼图或验证**。脚本会在后台等待，如果超时，请重新运行。本 Skill 绝对不会尝试绕过任何平台风控。

## 10. 如何人工审核、保存草稿与发布？
结合生成的 `toutiao_publish_checklist.md`：
1. 检查文章是否太像硬广？
2. 检查是否有夸张的承诺（如“保证涨粉 10 万”）？
3. 调整排版，手动上传封面图。
4. 自己点击界面底部的“保存草稿”或“发布”。

## 11. 常见问题
**Q: 会泄露我的账号吗？**
A: 不会。脚本不保存密码，不读取 cookie，只在你本地运行控制本地浏览器。

**Q: 保证能上头条推荐吗？**
A: **绝不保证**。本工具只负责将内容改写成“更适合推荐”的格式，最终流量取决于平台算法和你提供的内容本身价值。
