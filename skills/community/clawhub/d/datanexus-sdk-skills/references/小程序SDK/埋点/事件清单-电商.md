# 电商小程序 — 必报与选报事件

> 来源：[腾讯广告小程序 SDK 开发指引](https://doc.weixin.qq.com/doc/w3_AJUA1gbWACgBSVyDeMgQ3irunfYyt) — 电商行为章节

## 必报事件

| action_type | 中文名 | 说明 |
|------------|--------|------|
| `START_APP` | 启动应用 | 用于广告归因、广告效果优化。开启 SDK 自动采集后可不再手动上报 |
| `PRODUCT_VIEW` | 商品详情页浏览 | 用于广告归因、广告效果优化。用户进入商品详情页时上报 |
| `ADD_TO_CART` | 加入购物车 | 用于广告归因、广告效果优化。用户添加商品到购物车时上报 |
| `COMPLETE_ORDER` | 下单 | 用于广告归因、广告效果优化。用户提交订单时上报。在 `wx.requestPayment` 成功回调中上报 |
| `PURCHASE` | 付费 | 用于广告归因、广告效果优化。付费成功时上报，**必须传 `value`（单位：分）** |
| `RE_FUND` | 退款 | 用于广告归因、广告效果优化。用户发起退款申请时上报 |

## 专用上报方法

| 方法 | 参数 | 对应 action_type |
|------|------|-----------------|
| `sdk.onAppStart()` | 无 | START_APP |
| `sdk.onAppQuit()` | 无 | APP_QUIT |
| `sdk.onPageShow()` | 无 | PAGE_VIEW |
| `sdk.onPageHide()` | 无 | PAGE_LEAVE |

> 小程序 SDK 专用方法较少，大部分行为需使用通用方法 `sdk.track(action_type, action_param)` 上报。
