## 能源转换任务清单

能源转换按子域组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：转换规划 → 转换设备 → 转换运营 → 转换优化

---

| ID | 任务类型 | 说明 | 依赖 | 元操作映射 |
|----|----------|------|------|----------|
| T1-1 | 需求分析 | 分析能源转换需求，包括输入能源、输出能源、转换效率 | 无（入口） | S→C |
| T1-2 | 技术选型 | 选择转换技术，包括热力转换、动力转换、光电转换、电化学转换 | T1-1 | C |

## 依赖拓扑摘要

### 一、域内链路

**转换规划链路**: T1-1 → T1-2 → T1-3 → T1-4
**转换设备链路**: T2-1 → T2-2 → T2-3 → T2-4
**转换运营链路**: T3-1 → T3-2 → T3-3 → T3-4
**转换优化链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整能源转换链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T3-1 → T3-2 → T3-3 → T3-4 → T4-1 → T4-2 → T4-3 → T4-4

**反馈改进链路**: T4-4 → T1-2（技术升级反馈到技术选型）

**效率驱动链路**: T4-1 → T1-4（效率提升驱动效率评估）

更多组合根据具体任务动态推导。
