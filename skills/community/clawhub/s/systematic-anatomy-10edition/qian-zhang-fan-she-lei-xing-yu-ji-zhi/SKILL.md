# 牵张反射的类型与机制

## 何时使用本技能
- 当需要解释骨骼肌受牵拉后产生反射性收缩的生理机制时
- 在临床检查中分析膝反射、跟腱反射等深腱反射结果时
- 理解肌紧张如何维持身体姿势或评估γ-环路对肌梭敏感性的调节作用时
- 排除去神经肌肉或脊髓休克期等不适用情况时

## 执行步骤

1. **确认前提条件**：
   - 肌肉必须有完整神经支配
   - 脊髓反射弧必须完整
   - 肌梭功能正常

2. **识别牵张反射触发条件**：
   - IF 骨骼肌受到外力牵拉伸长 → THEN 触发牵张反射

3. **区分反射类型并应用对应机制**：
   - **腱反射（单突触反射）**：
     - 由快速牵拉肌腱引发（如叩击髌腱）
     - 反射弧仅含一个传入神经元和一个α-运动神经元，仅一次突触
     - 常见实例：膝反射、跟腱反射、肱二头肌反射
   - **肌紧张（多突触反射）**：
     - 由缓慢持续牵拉肌肉引发
     - 表现为受牵拉肌肉的持续性轻度收缩
     - 涉及中间神经元，属多突触反射
     - 是维持躯体姿势最基本的反射活动

4. **评估γ-环路的作用**：
   - γ-运动神经元兴奋 → 核内肌纤维收缩 → 肌梭敏感性提高 → 增强牵张反射 → 维持肌张力

5. **排除不适用情形**：
   - 不适用于去神经支配的肌肉
   - 不适用于脊髓休克期患者

## 输出结果
肌肉收缩以抵抗牵拉，表现为快速反射动作（腱反射）或持续性轻度收缩（肌紧张），从而维持姿势或保护关节。
