# thyroid-hormone-synthesis-regulation-and-iodine-application

## 触发条件
- 需评估甲状腺功能状态（如甲亢/甲减诊断）；或
- 需快速抑制甲状腺激素释放（如甲亢术前2周或甲状腺危象）。

## 执行步骤
### A. 甲状腺激素合成与调节
1. **碘摄取**：滤泡细胞主动摄碘，浓度可达血浆25倍（甲亢时250倍）。
2. **碘活化与酪氨酸碘化**：过氧化物酶催化I⁻氧化，与TG中酪氨酸结合生成MIT/DIT。
3. **偶联**：DIT+DIT→T₄；DIT+MIT→T₃；缺碘时T₃比例升高。
4. **释放**：蛋白水解酶水解TG释放T₄/T₃；T₄占90%，外周转化为活性更强的T₃。
5. **调节轴**：TRH（下丘脑）→ TSH（垂体）→ 甲状腺激素；T₃/T₄负反馈抑制TRH/TSH。

### B. 大剂量碘剂应用
1. **作用机制**：
   - 抑制TG水解 → 阻止T₃/T₄释放（通过减少GSH）；
   - 拮抗TSH作用；
   - 抑制过氧化物酶活性（短期）。
2. **时间效应**：1–2天起效，10–15天达峰；因Wolff-Chaikoff效应，长期无效。
3. **临床应用**：
   - **术前准备**：复方碘溶液（卢戈液）术前2周，使腺体缩小变韧；
   - **甲状腺危象**：静脉或口服碘剂，**必须联合硫脲类**（碘抑释放，硫脲抑合成）。
4. **禁忌**：碘过敏者禁用；孕妇及哺乳期慎用；不可单独用于长期内科治疗。

## 输出结果
- 用于解读TSH/T₃/T₄水平以诊断甲状腺疾病；
- 指导大剂量碘剂在围手术期或急症中的安全短期使用。
