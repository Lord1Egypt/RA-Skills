# 广告投放优化最佳实践

## 一、账户搭建策略

### 账户结构金字塔
```
账户 (Account)
├── 推广计划 (Campaign) — 按推广目标分组
│   ├── 广告组 (Ad Group) — 按定向/出价分组  
│   │   ├── 创意 (Creative) — 素材+文案组合
│   │   └── 创意
│   └── 广告组
└── 推广计划
```

### 搭建原则
1. **目标分层**：不同目标（拉新/促活/转化）分开建计划
2. **预算隔离**：主力计划和测试计划分开预算池
3. **人群差异化**：不同人群包分广告组，避免内部竞争
4. **素材AB测试**：每个广告组至少2-3套素材异步测试

## 二、出价策略

### 冷启动期（0-3天）
- 出价 = 预期CPA × 1.3 ~ 1.5
- 预算 = 预期CPA × 20（至少保证20个转化/天）
- 目标：快速积累转化数据，让模型学习

### 稳定期（3-7天）
- 出价 = 实际CPA × 1.0 ~ 1.1
- 按ROAS目标微调
- 关注：消耗稳定性、转化量波动

### 衰退期
- 素材疲劳 → 换新素材，不要加价硬拉
- 人群耗尽 → 放宽定向或新人群包
- 大盘波动 → 检查行业节点（618、双11）

## 三、预算分配策略

### 532分配法（推荐）
- 50% 预算 → 已验证的优质计划（稳定产出）
- 30% 预算 → 中等表现计划（有潜力，需观察）
- 20% 预算 → 新测试计划（探索新方向）

### 动态调整规则
- ROAS > 3.0 → 可加预算，单次+20%
- ROAS 1.5~3.0 → 维持，观察趋势
- ROAS 1.0~1.5 → 谨慎，小预算维持
- ROAS < 1.0 → 关停或大幅调整

## 四、素材优化策略

### 素材生命周期
```
上新 → 学习期(1-2天) → 成长期(3-7天) → 稳定期(7-14天) → 衰退期(14天+)
```

### 疲劳度信号
- CTR持续3天下滑
- CPM持续上涨但转化未涨
- 频次 > 5（信息流场景）

### 素材轮换节奏
- 主力素材：1-2周更新一次
- 测试素材池：每周新增3-5套
- 视频 > 图片（当前趋势，CTR高20-40%）

### 素材ABC测试法
- A版（主力）：已验证方向，微调元素
- B版（探索）：新卖点/新风格
- C版（大胆）：完全不同的切入角度

## 五、定向策略

### 定向由窄到宽
```
第一阶段(冷启动)：精准人群包 + 核心兴趣
第二阶段(放量期)：兴趣标签放宽 + 相似人群
第三阶段(通投期)：系统智能定向 + 排除已转化
```

### 人群分层
- 拉新层：行业兴趣 + 相似人群
- 种草层：互动人群（浏览/加购未下单）
- 转化层：高意向人群 + 已购复购

## 六、数据复盘节奏

### 日复盘（每日）
- 消耗、曝光、点击、转化、CPA
- 与昨日/上周同日对比
- 异常计划标记和处理

### 周复盘（每周）
- ROAS/GMV趋势
- 素材表现TOP/BOTTOM
- 计划生命周期评估
- 下周预算和测试计划

### 月复盘（每月）
- ROI综合评估
- 渠道效率对比
- 素材资产盘点
- 下月策略调整

## 七、常见避坑指南

1. **不要频繁调价**：一天调价不超过2次，给模型学习时间
2. **不要同时大量上新**：单日新素材不超过5套，分散权重
3. **不要关停重启**：关闭的计划不要立刻重启，等24小时
4. **不要忽视落地页**：落地页加载速度>3秒，流失40%用户
5. **不要忽略归因窗口**：不同窗口ROAS差异大，统一口径
6. **不要只追ROAS**：关注GMV绝对值，ROAS高但量小意义不大
7. **不要忽视素材合规**：平台审核趋严，素材违规影响账户权重
