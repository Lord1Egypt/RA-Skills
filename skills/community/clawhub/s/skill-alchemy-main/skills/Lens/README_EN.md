## What It Does

When you describe a task, you naturally leave things out: perspectives you haven't considered, pitfalls insiders know to avoid, quality standards you only realize you missed once it's too late. Lens rounds out the problem space before you start building. No questions asked, no judgment, no preaching. Use it standalone or let an orchestrator call it.

## Quick Start

```
npx skills add agentsope/SkillAlchemy/skills/Lens
```

Trigger with natural language:

```
What am I not seeing in this project?
Analyze this requirement
Generate a skill from this
```

## How It Works

Four steps run internally, no interaction:

1. **Characterize** — What are you doing? For whom? What's the deliverable?
2. **Expand** — Abstract upward, decompose subject/method, ask "what does *good* mean here?"
3. **Verify** — Name-swap, terminology, differentiation tests. Fire targeted search if any fail.
4. **Converge** — Match ambition level, keep only the 5-10 highest-impact dimensions.

Two output modes: **Mode A** — hidden dimensions checklist; **Mode B** — enhanced description, ready to feed downstream.
