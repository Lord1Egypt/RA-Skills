# 拼多多百亿补贴

提供拼多多百亿补贴频道好货推荐工具。百亿补贴是平台核心补贴频道，全量仅568件商品入选——门槛高、数量少、精选稀缺好货。采用4维好货评分模型智能排序，支持关键词搜索、价格筛选、品牌店筛选和翻页浏览。

## 能做什么

- **好货浏览**：分页浏览百亿补贴精选好货，每页30条，按好货评分排序
- **关键词搜索**：按品类或商品关键词筛选，如"手机""男装""海淘"
- **价格筛选**：设置到手价区间，锁定预算范围
- **品牌店筛选**：仅看品牌店商品，品质更有保障
- **多种排序**：好货评分（默认）、价格升降序、销量优先
- **品类导览**：始终展示全量品类分布，引导用户按品类逛
- **购买链接**：每条商品附带短链接，点击即可跳转下单

## 不能做什么

- 不能下单购买，只提供商品信息和购买链接
- 不能查询历史价格走势
- 不能设置降价提醒通知
- 不能获取单品详情页的完整信息

## 使用示例

1. "百亿补贴有什么好货" → 返回好货Top30+品类分布引导
2. "百亿补贴有手机吗" → 返回手机品类匹配商品
3. "百亿补贴50元以内的" → 按到手价筛选
4. "百亿补贴品牌店看看" → 仅看品牌店商品
5. "下一页" → 翻页查看更多
6. "最便宜的百亿补贴" → 按价格升序排列

## 注意事项

- 百亿补贴全量仅568件商品，数量稀缺，热门品类竞争激烈
- 商品价格实时变动，5分钟缓存刷新，每次查询可能看到不同商品
- 购买链接为短链接，点击后自动跳转到拼多多商品页面
- 优惠券有使用门槛，以实际页面显示为准
- 看到合适的好货建议尽快下单，库存有限
- **数据流向说明**：查询通过云端代理转发到拼多多官方API，代理不存储用户数据

## 使用提示

- 不传关键词时默认返回好货Top30，适合"逛"的模式
- 带品类关键词时结果通常较少，说明该品类入选商品稀缺属正常
- sort=price_asc找最便宜的好货，sort=sales看大家都在买什么
- brand_only=true筛选品牌店，品质更有保障
- 每次返回结果都包含品类分布，引导按品类深入浏览
- 每条商品包含商品图片URL（image_url），请用 ![商品名](image_url) 格式渲染图片，图片下方放购买链接
