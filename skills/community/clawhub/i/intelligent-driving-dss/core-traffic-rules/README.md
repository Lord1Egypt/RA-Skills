# Core Traffic Rules (核心交通法规)

## 用途
本模块存储《中华人民共和国道路交通安全法》（2021 年第三次修正版）的核心条文和关键概念定义。

**法律依据：** 中华人民共和国道路交通安全法  
**生效日期：** 2021-04-29（现行有效）  
**最新修订：** 2026 年度立法工作计划中已列入修订草案审议（预计即将更新）

## 核心文件
- `总则.md` - 第一章：法律宗旨、适用范围和基本原则
- `车辆和驾驶人.md` - 第二章：机动车/非机动车登记检验及驾驶人资质要求
- `道路通行条件.md` - 第三章：交通信号、安全设施等基础设施规定
- `道路通行规定.md` - 第四章：机动车/非机动车/行人通行规则（含高速公路特别规定）
- `交通事故处理.md` - 第五章：事故报警、救助和责任认定
- `执法监督.md` - 第六章：交警队伍建设和执法规范
- `法律责任.md` - 第七章：各类违法行为的行政处罚和刑事责任
- `附则.md` - 第八章：定义解释和施行日期

## 使用方式
```javascript
// 加载法规模块
const rules = require('intelligent-driving-dss/core-traffic-rules');

// 获取特定条款
rules.getChapter('总则');
rules.getArticle(76); // 机动车与非机动车/行人事故责任划分

// 查询核心概念
rules.getConcepts(); // 返回所有关键术语和定义
```
