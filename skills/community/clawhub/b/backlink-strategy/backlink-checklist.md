# Backlink Strategy Quality Checklist

Verify every item before delivering the final strategy to ensure completeness and quality.

## Link Profile Audit (8 items)
- [ ] Current DA/DR score documented with source tool and date
- [ ] Total referring domains counted and compared to 6-month trend
- [ ] Anchor text distribution analyzed and compared to healthy benchmarks
- [ ] Toxic/spammy links identified with specific disavow recommendations
- [ ] Top 3-5 competitors identified with DA and referring domain counts
- [ ] Link gap quantified (number of referring domains to close)
- [ ] Link velocity calculated (average new links per month)
- [ ] Top 10 existing backlinks identified and categorized by type

## Competitive Analysis (6 items)
- [ ] Competitor backlink profiles exported and analyzed
- [ ] Common linking domains identified (sites linking to 2+ competitors)
- [ ] Unique competitor link sources cataloged for replication
- [ ] Competitor content assets identified that earn the most links
- [ ] Competitor link-building tactics inferred from profile patterns
- [ ] Specific link opportunities from competitor analysis documented

## Strategy Design (7 items)
- [ ] Monthly link acquisition targets set based on competitive gap
- [ ] Tactics selected match niche characteristics and budget constraints
- [ ] Each tactic has specific prospect criteria defined
- [ ] Expected response rates cited with realistic benchmarks
- [ ] Effort-versus-impact assessment completed for each tactic
- [ ] Budget allocation mapped across tactics
- [ ] Timeline aligns with content production capacity

## Linkable Assets (6 items)
- [ ] Minimum 3 linkable asset concepts proposed with rationale
- [ ] Each asset has a defined target audience for link earning
- [ ] Production timeline and cost estimated for each asset
- [ ] Distribution and promotion plan outlined for each asset
- [ ] Asset topics validated against competitor link-earning patterns
- [ ] Content briefs include SEO requirements and link-earning hooks

## Outreach Preparation (7 items)
- [ ] Prospect lists built for each tactic with minimum 15 prospects per list
- [ ] Each prospect scored on DA, relevance, traffic, and feasibility
- [ ] Priority tiers assigned (A/B/C) with clear outreach sequencing
- [ ] Outreach templates created for each tactic type
- [ ] Templates include personalization framework beyond just name insertion
- [ ] Follow-up sequences defined with appropriate timing (3-5 day gaps)
- [ ] Template tone is human and conversational, not robotic

## Campaign Planning (6 items)
- [ ] 90-day campaign calendar created with weekly milestones
- [ ] Quick wins prioritized for months 1-2 to build momentum
- [ ] Higher-effort tactics scheduled with sufficient content lead time
- [ ] Weekly outreach volume targets are sustainable for team size
- [ ] Resource allocation plan matches available budget and personnel
- [ ] Contingency approaches identified if primary tactics underperform

## Risk Management (5 items)
- [ ] Anchor text strategy ensures diversity (no over-optimization)
- [ ] Link velocity targets maintain natural growth pattern
- [ ] No tactics recommended that violate Google Webmaster Guidelines
- [ ] Toxic link disavow file prepared if existing issues found
- [ ] Monitoring plan includes alerts for negative SEO or link decay

## Tracking and Reporting (5 items)
- [ ] KPIs defined with specific monthly targets
- [ ] Tracking dashboard or spreadsheet template provided
- [ ] Monthly reporting template includes all key metrics
- [ ] Tactic performance comparison framework established
- [ ] Iteration triggers defined (when to double down vs. pivot)
