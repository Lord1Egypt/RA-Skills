## Description: <br>
Analyze the top 100 S&P 500 companies by market capitalization using CANSLIM-style signals and return a ranked shortlist in Markdown. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[lkmsteven](https://clawhub.ai/user/lkmsteven) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users, developers, and analysts use this skill to run a local CANSLIM-style screen over the top 100 S&P 500 companies by market capitalization and summarize the strongest candidates. The output is a stock screen, not investment advice. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Python dependencies and public market-data responses can change over time, which can affect reproducibility and package risk. <br>
Mitigation: Run the skill in a dedicated Python virtual environment and review or pin dependency versions before installation. <br>
Risk: The analyzer contacts public market-data sources and writes canslim_results.json locally. <br>
Mitigation: Run it in a dedicated workspace and review generated files before sharing or relying on the results. <br>
Risk: The ranked output could be mistaken for investment advice. <br>
Mitigation: Present it as a screening aid only, state that it is not investment advice, and verify market data independently before acting on it. <br>
