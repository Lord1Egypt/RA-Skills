# 红细胞形态学诊断：球形与裂红细胞识别

## 适用场景
- 外周血Wright-Giemsa染色涂片中发现异常红细胞形态
- 怀疑溶血性贫血但需通过形态学初步分型
- 需区分遗传性球形红细胞增多症、微血管病性溶血（如TTP/DIC）或机械性溶血

## 执行步骤

### 一、球形红细胞识别
1. **测量尺寸**：直径 <6μm 且厚度 >2.9μm 提示球形变。
2. **观察染色特征**：
   - 细胞呈均匀圆球形（非双凹盘状）
   - Wright-Giemsa染色后着色深
   - 中央淡染区完全消失
3. **定量评估**：
   - 若球形红细胞占比 >25%，高度提示遗传性球形红细胞增多症
   - 若 <25%，考虑自身免疫性溶血性贫血或酶缺陷等其他病因

### 二、裂红细胞识别
1. **识别碎片形态**：包括但不限于盔形、新月形、梨形、三角形、哑铃形、逗点形或长圆形。
2. **确认非人为损伤**：排除制片过程中机械破坏导致的假阳性。
3. **关联危重病因**：
   - 微血管病性溶血性贫血（TTP、HUS）
   - 弥散性血管内凝血（DIC）
   - 人工心脏瓣膜相关溶血
   - 严重烧伤
4. **发出临床警示**：一旦确认裂红细胞存在，应视为急症信号，立即启动溶血及凝血功能评估。

## 注意事项
- 仅适用于Wright-Giemsa染色的人源外周血涂片
- 球形红细胞属于高色素性红细胞；裂红细胞属于红细胞形态不整（poikilocytosis）
- 形态学发现必须结合临床和实验室指标综合判断
