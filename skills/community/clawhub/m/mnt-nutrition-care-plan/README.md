# mnt-nutrition-care-plan

Drafts a Medical Nutrition Therapy (MNT) documentation note in ADIME format using the Nutrition Care Process (NCP), aligned to AND NCPT and CMS MNT benefit requirements.

## Overview

This skill guides a Registered Dietitian (RD/RDN) or dietetic intern through five structured phases — referral intake, ABCDE nutrition assessment, PES-statement diagnosis, SMART MNT intervention goals, and monitoring/evaluation parameters — assembling a complete DRAFT ADIME note ready for licensed RD sign-off before medical record entry or payer submission.

## Use When

- An RD/RDN or dietetic intern has completed a nutrition assessment and needs to document findings as a structured MNT note
- The note must meet CMS Part B MNT documentation requirements for diabetes or non-dialysis CKD
- A nutrition support team needs to document enteral or parenteral nutrition prescription and monitoring plan
- A preceptor is reviewing an intern's ADIME note draft

## What It Produces

- A structured DRAFT ADIME note with:
  - Referral and setting context (no direct patient identifiers)
  - Assessment: ABCDE framework — anthropometrics, biochemical labs, clinical/physical findings, dietary intake, environmental/social factors
  - Diagnosis: PES statement(s) using AND NCPT-recognized nutrition diagnosis terminology
  - Intervention: estimated energy/protein/fluid requirements (method stated), SMART MNT goals, intervention strategy (education, counseling, coordination, EN/PN prescription)
  - Monitoring and Evaluation: parameter table with targets, measurement methods, and reassessment timeframes; billing code noted
  - RD attestation placeholder
  - DRAFT FLAGS checklist of unresolved items

## Scope and Boundaries

- Covers outpatient, acute care, long-term care, home health, dialysis, and community nutrition settings
- Applicable diagnoses: diabetes, CKD, malnutrition, cardiovascular disease, oncology, GI disorders, wound/pressure injury, eating disorders, pediatric growth concerns
- **Does not** produce a signed clinical document — requires licensed RD/RDN review and signature before medical record entry or payer submission
- **Does not** prescribe medications or perform the clinical assessment — the RD supplies all clinical data
- PES statement construction follows IDNT rules: P is never a medical diagnosis
