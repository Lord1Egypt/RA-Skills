# 彩色宝石鉴定与选购指南

## 一、红宝石 (Ruby)

### 核心评价标准
| 维度 | 说明 | 顶级标准 |
|------|------|----------|
| 颜色 | 鸽血红(Pigeon's Blood)为最高 | 鲜艳纯正红色，无褐/紫色调 |
| 净度 | 天然红宝石常有内含物 | 肉眼干净即为上品 |
| 切工 | 影响火彩和颜色表现 | 对称、比例好、不漏底 |
| 克拉 | 1ct以上具有收藏价值 | 每增大1ct价格指数级增长 |
| 产地 | 缅甸(抹谷)最负盛名 | 缅甸 > 莫桑比克 > 泰国/坦桑尼亚 |
| 处理 | 有烧/无烧影响巨大 | 无烧(Unheated)溢价3-10倍 |

### 价格参考 (2025-2026)
| 规格 | 有烧(Heated) RMB/ct | 无烧(Unheated) RMB/ct |
|------|---------------------|------------------------|
| 1ct 鸽血红 | 15,000-30,000 | 50,000-150,000 |
| 2ct 鸽血红 | 30,000-60,000 | 120,000-350,000 |
| 3ct 鸽血红 | 60,000-120,000 | 300,000-800,000 |
| 5ct+ 鸽血红 | 150,000+ | 80万-数千万(拍卖) |

### 选购建议
- 日常佩戴：1-2ct 莫桑比克有烧，性价比高
- 收藏投资：3ct+ 缅甸无烧鸽血红
- 避坑：合成红宝石(焰熔法/水热法)泛滥，注意区别

## 二、蓝宝石 (Sapphire)

### 核心评价标准
| 维度 | 说明 | 顶级标准 |
|------|------|----------|
| 颜色 | 皇家蓝(Royal Blue)、矢车菊蓝(Cornflower) | 浓郁均匀蓝色 |
| 净度 | 比红宝石净度高 | 肉眼干净 |
| 切工 | 影响颜色均匀度 | 对称饱满 |
| 产地 | 克什米尔(绝矿) > 缅甸 > 斯里兰卡 > 马达加斯加 |
| 处理 | 有烧/无烧 | 无烧溢价3-5倍 |

### 价格参考 (2025-2026)
| 规格 | 有烧 RMB/ct | 无烧 RMB/ct |
|------|------------|------------|
| 1ct 皇家蓝 | 3,000-8,000 | 10,000-30,000 |
| 2ct 皇家蓝 | 8,000-18,000 | 30,000-80,000 |
| 3ct 皇家蓝 | 18,000-40,000 | 80,000-200,000 |
| 5ct+ 皇家蓝 | 50,000+ | 25万+ |

**2025年趋势**：斯里兰卡蓝宝石价格半年涨20%，是近年最活跃的彩宝品类。

### 彩色蓝宝石
- **帕帕拉恰(Padparadscha)**：粉橙色，斯里兰卡产，日落色为佳，1ct 无烧 20,000-80,000/ct
- **粉色蓝宝石**：近年热款，1ct 3,000-15,000
- **黄色蓝宝石**：性价比之选，1ct 1,000-5,000

## 六、彩宝通用选购法则

### 必看四要素（按优先级）
1. **颜色** > 2. **净度** > 3. **切工** > 4. **克拉重量**

### 处理方式对比
| 处理 | 说明 | 对价值影响 | 接受度 |
|------|------|-----------|--------|
| 无处理 | 完全天然 | 最高 | ⭐⭐⭐⭐⭐ |
| 传统加热(有烧) | 仅加热改善颜色 | 降低50-80% | ⭐⭐⭐⭐ |
| 扩散处理 | 表面加色 | 大幅贬值 | ⭐⭐ |
| 充填处理 | 裂隙填充 | 大幅贬值 | ⭐ |
| 染色处理 | 人工染色 | 几乎无价值 | ❌ |

### 证书优先级
| 宝石品类 | 首选证书 | 备选 |
|----------|---------|------|
| 红宝石/蓝宝石 | GRS / SSEF / Gübelin | AIGS / GIC |
| 祖母绿 | GRS / SSEF / Gübelin | AIGS / GIC |
| 帕拉伊巴 | GRS / Gübelin | AIGS |
| 尖晶石 | GRS / AIGS | GIC |

## 七、常见陷阱
1. **合成宝石冒充天然**：焰熔法合成红/蓝宝石泛滥，注意证书标注"Natural"
2. **"新烧"充"老烧"**：扩散处理冒充传统加热
3. **产地虚标**：莫桑比克红宝标缅甸，马达加斯加蓝宝标斯里兰卡
4. **注油量虚标**：中油标微油，注意核对证书
5. **拼合石**：二层/三层拼合冒充天然宝石
