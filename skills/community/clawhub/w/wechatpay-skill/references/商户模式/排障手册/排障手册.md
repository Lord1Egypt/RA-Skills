# 排障手册

> 基于微信支付官方排障手册精简

## 错误码TOP20速查

| 错误码 | 含义 | 快速解决方案 |
|--------|------|--------------|
| 268512771 | 订单不存在 | 检查out_trade_no是否正确，确认订单是否已创建 |
| 268447840 | 请求频率超限 | 降低请求频率，使用指数退避重试 |
| 268435461 | 参数非法 | 检查请求参数格式、必填字段 |
| 268449538 | 订单已支付 | 已支付订单不能重新下单 |
| 268443715 | 商户状态不可用 | 检查商户号状态，是否被注销 |
| 268443699 | 签名错误 | 检查签名算法、私钥、证书序列号 |
| 268501013 | JSAPI必须传openid | 下单时需传openid参数 |
| 268445527 | appid和mchid不匹配 | 检查AppID与商户号是否绑定 |

## 常见问题排查

### 1. 下单失败

**检查清单**：
- [ ] 商户号和AppID是否匹配
- [ ] openid是否正确获取
- [ ] 签名是否正确
- [ ] notify_url是否可达
- [ ] amount.total是否为整数（分）

### 2. 调起支付失败

**JSAPI/小程序**：
- 检查WeixinJSBridge/wx.requestPayment参数
- 检查调起签名是否正确
- 确认prepay_id未过期

**APP支付**：
- 检查OpenSDK是否正确初始化
- 检查调起参数
- 确认包名与AppID匹配

### 3. 收不到回调

**排查步骤**：
1. 确认notify_url配置正确
2. 确认回调地址公网可达
3. 检查服务器日志是否有POST请求
4. 确认返回200状态码
5. 10秒内未返回200会重试（最多24小时）

### 4. 回调验签失败

**常见原因**：
- 用了错误的公钥
- 签名串构造格式错误
- 微信支付公钥未更新（证书续期后）

### 5. 退款失败

**检查清单**：
- 订单是否已支付
- 退款金额是否超限
- 退款次数是否超限（最多50次）
- 退款申请单号是否复用

## 常见踩坑速查表

| 坑点 | 表现 | 解决方案 |
|------|------|----------|
| prepay_id过期 | 调起报签名错误 | 重新下单获取新prepay_id |
| appId填错 | 调起失败 | JSAPI用公众号AppID，小程序用小程序AppID |
| openid获取方式错 | INVALID_REQUEST | 公众号用网页授权，小程序用wx.login |
| 金额用浮点数 | 支付金额不对 | 金额必须用整数（分） |
| 回调未验签 | 资金被盗刷 | 收到回调必须先验签 |
| 重复处理回调 | 重复发货/加款 | 必须实现幂等处理 |
| h5_url直接打开 | 调起失败 | h5_url需在手机浏览器打开 |
