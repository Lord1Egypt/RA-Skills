# 儿童肾病综合征诊断与分型

## 适用对象
- 儿童肾病综合征疑似患者

## 使用前提
- 已获取尿液和血液实验室检查结果

## 禁忌情形
- 不适用于仅有水肿或高脂血症但无蛋白尿或低蛋白血症者
- 继发性或遗传性NS需先排除原发病

## 执行步骤

### 1. 判断是否满足NS诊断必备条件（两项必须同时满足）
- **大量蛋白尿**（满足以下任一）：
  - 1周内3次尿蛋白定性 ++～++++
  - 晨尿或随机尿尿蛋白(mg)/肌酐(mg) ≥ 2.0
  - 24小时尿蛋白定量 ≥ 50 mg/(kg·d)
- **低蛋白血症**：血清白蛋白 < 25 g/L

### 2. 识别支持性诊断标准（非必需但常见）
- 高脂血症：血清胆固醇 > 5.7 mmol/L
- 不同程度水肿

### 3. 进行临床分型
- **单纯型NS**：仅具备上述四大表现（大量蛋白尿、低蛋白血症、高脂血症、水肿）
- **肾炎型NS**：除NS表现外，具备以下任一项：
  - 血尿：2周内3次离心尿镜检RBC ≥ 10/HPF，且为肾小球源性
  - 反复或持续高血压（排除激素所致）
  - 肾功能下降（排除血容量不足）
  - 持续或反复低补体血症

### 4. 评估激素反应类型
- **激素敏感型（SSNS）**：泼尼松 2 mg/(kg·d) 或 60 mg/(m²·d) 治疗 ≤4周尿蛋白转阴
- **激素耐药型（SRNS）**：足量激素治疗 >4周尿蛋白仍阳性
- **激素依赖型（SDNS）**：激素减量或停药2周内复发，重复≥2次

## 输出结果
输出诊断结论（是否NS）、临床分型（单纯型/肾炎型）及激素反应类型，用于指导治疗和预后评估。
