## ⚔️ 你的敌人：旺季焦虑

你是否正在经历这些？
- "Prime Day备货够不够？"
- "黑五备货太早怕压仓，太晚又怕断货..."

这就是**旺季焦虑** — 这个技能帮你解决它。

## 📋 Commands

```
seasonal calendar        # 全年旺季日历
seasonal prime           # Prime Day专项
seasonal blackfriday     # 黑五网一专项
seasonal christmas       # 圣诞旺季专项
seasonal inventory       # 旺季备货计算
seasonal ads             # 旺季广告策略
seasonal checklist       # 旺季准备清单
```
