# 示例提问

## 正常触发

```text
我想做一个 Skill，但我完全不懂，你一步步问我。
```

```text
帮我做一个小助手，让它以后能帮我改作文。
```

```text
我想把这个工作流程变成 Skill。
```

## 复杂触发

```text
我想做一个 Skill，给不懂 AI 的老师用。老师只要上传学生作文，它就能改错别字、保留原意、给出鼓励性建议，最后还能打包给别人安装。
```

```text
帮我做一个能生成小红书图片文案的 Skill，用户只需要说主题，它就输出标题、正文、配图提示词和发布注意事项。
```

## 不应触发

```text
Skill 是什么意思？
```

```text
帮我改一下这篇作文。
```

```text
帮我写一个普通的网页。
```

这些问题可能只是在提问或执行具体任务，不一定是在创建 Skill。只有当用户明确想“创建 Skill / 小助手 / 固定工作流”时才触发。
