# 技能名称：miaoji-video-script-pro
# 显示名：秒技·视频脚本Pro版

---

## 定位升维

**敌人**：只拍"能看完的视频"、不追求"看完就想下单"的完播率思维

**升维打击**：从"写个视频脚本"升级到"设计完整视频内容矩阵"

免费版给你一个30秒脚本，Pro版给你15秒/30秒/60秒/3分钟四种规格，加上平台专属Hook和A/B测试方案。

---

## 核心定位

视频脚本Pro版是免费版的战略升级版，提供多时长视频脚本、平台专属开头设计、A/B测试脚本方案和缩略图文案。帮助卖家建立完整的视频内容矩阵，覆盖从吸引到转化的全链路。

**与免费版的本质区别**：
- 免费版：单次脚本生成 → 给你一个视频的脚本
- Pro版：视频内容矩阵 → 给你4种时长×3个角度×N个平台的完整方案

### 2. 平台专属Hook设计

Amazon Shorts/TikTok/YouTube/独立站开头3秒差异化：

```
【平台专属Hook设计指南】
━━━━━━━━━━━━━━━━━━━━

📱 Amazon Shorts Hook（0-3秒）

特点：竖版全屏，静音为主，文字驱动
目标：滑动停止率 > 40%

Hook类型：
1. 痛点前置型
   "[画面开头直接展示用户痛点场景]"
   示例：宝宝哭闹/衣服染色/头发毛躁
   
2. 数据冲击型
   "[字幕：大数字/极端场景]"
   示例："93%的人不知道这个技巧"/"被问了1000次的..."

3. 反常识型
   "[颠覆认知的开场]"
   示例："用这个方法，洗碗不用弯腰"

4. 情绪共鸣型
   "[强烈的情感画面]"
   示例：妈妈的笑容/用户的惊喜表情

⚠️ Amazon Shorts禁忌：
- 不要硬广感太强
- 不要上来就展示产品
- 不要使用引导购买的夸张言语


📱 TikTok Hook（0-3秒）

特点：竖版，音效重要，娱乐性强
目标：完播率 > 50%，互动率 > 5%

Hook类型：
1. 剧情悬念型
   "[制造悬念的开场]"
   示例："用了这个方法之后，我再也不..."

2. 热点嫁接型
   "[蹭热门话题的开场]"
   示例：引用热门BGM或挑战

3. 身份认同型
   "[直接对目标人群喊话]"
   示例："如果你也在找..."

4. 反差萌型
   "[出人意料的转折]"
   示例：严肃话题+搞笑结尾
   
5. 教程干货型
   "[直接给价值的开头]"
   示例："今天教你3招..."


📺 YouTube Hook（0-10秒）

特点：横版，有声为主，可深度内容
目标：留存率 > 50%

Hook类型：
1. 价值预告型
   "在这个视频里，我会教你..."

2. 结果展示型
   "这是使用X方法X周后的效果..."

3. 故事引入型
   "三个月前，我差点放弃..."

4. 专家背书型
   "作为XX领域的专家，我要告诉你..."


💻 独立站视频 Hook（0-5秒）

特点：品牌调性，品质感强
目标：观看完整率 > 60%

Hook类型：
1. 品牌美学型
   "[高品质画面开场]"
   配合简约文案
   
2. 用户证言型
   "[真实用户的开场]"
   "我用了X产品X个月，这是我的体验..."
   
3. 问题解答型
   "[直接回答用户问题]"
   "你一直在找的X产品，这是最好的选择..."
```

### 3. A/B测试脚本方案

同一产品3种不同角度：

```
【A/B测试脚本方案】
━━━━━━━━━━━━━━━━━━━━

🎯 测试目标：验证不同角度的视频内容对转化率的影响

📊 脚本方案矩阵：

📹 脚本A：痛点型
角度：聚焦用户痛点，强调"解决问题"
目标人群：理性消费者，关心效果
预期CTR：参考值 +12-18%
预期转化率：参考值 +8-15%

[脚本结构]
开场：展示痛点场景（不舒适/麻烦/无效产品）
过程：产品解决方案
结尾：痛点解决，用户满意

适用品类：工具类、家电类、健康类产品


📹 脚本B：功能型
角度：聚焦产品功能，强调"专业品质"
目标人群：参数控，喜欢对比数据
预期CTR：参考值 +8-12%
预期转化率：参考值 +5-10%

[脚本结构]
开场：产品特写+核心参数
过程：功能演示+技术亮点
结尾：专业认证+品质背书

适用品类：3C电子、户外运动、宠物用品


📹 脚本C：情感型
角度：聚焦情感体验，强调"生活方式"
目标人群：感性消费者，重视体验
预期CTR：参考值 +15-22%
预期转化率：参考值 +10-18%

[脚本结构]
开场：目标用户理想生活场景
过程：产品融入生活场景
结尾：美好体验+情感满足

适用品类：家居、服饰、美妆、食品


📋 测试执行计划：

Week 1-2：
├─ 脚本A上线：曝光量目标 5000+
├─ 脚本B上线：曝光量目标 5000+
└─ 脚本C上线：曝光量目标 5000+

Week 3-4：
├─ 数据收集：CTR、完播率、转化率
├─ 中期分析：初步判断优势角度
└─ 优化调整：细节微调

Week 5-6：
├─ 冠军脚本扩量
└─ 劣势脚本下线或大改
```
