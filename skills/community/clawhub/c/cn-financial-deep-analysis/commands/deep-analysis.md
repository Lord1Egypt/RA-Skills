# /deep — 深度分析

对指定A股公司执行宏观-中观-微观三层深度分析，输出完整财务分析报告。

## 执行步骤

1. 拉取近5年三表数据、全部财务指标、行业数据、宏观数据、新闻公告
2. 宏观层：按 `references/core/three-layer-framework.md` 执行行业生命周期判断、产业链分析、波特五力、八维战略框架
3. 中观层：三表联动分析 → 200+指标五维度展开（盈利能力/营运能力/偿债能力/成长能力/现金流）→ 杜邦分析三因子拆解
4. 微观层：七大盈余操纵模式筛查 → 舞弊恒等式验证 → 资产负债表舞弊痕迹
5. 价值评估：DCF + PE/PB/PS/EV-EBITDA行业对比 + 安全边际判断

## 输入格式

- 股票代码 或 公司名称

## 输出要求

包含以下章节的完整分析报告：
- 三层分析结果
- 杜邦ROE归因分析
- 价值评估区间
- 投资建议
- 核心风险提示
