# llm-deploy-helper 🚀

**一键检测硬件 + 推荐部署方案 + 生成 Docker Compose / systemd 配置** 的 LLM 本地部署助手。

## 安装

```bash
pip install llm-deploy-helper
```

可选：安装 `psutil` 以获得更精确的系统检测（不安装也能工作）：

```bash
pip install llm-deploy-helper[full]
```

### 2. 生成 Docker Compose 配置

```bash
# Ollama (自动使用 GPU)
$ llm-deploy-helper generate --engine ollama --model llama3.1:8b

# vLLM (GPU 必需)
$ llm-deploy-helper generate --engine vllm --model meta-llama/Llama-3.1-8B-Instruct

# llama.cpp (CPU/GPU 均支持)
$ llm-deploy-helper generate --engine llama.cpp --model llama3.1-8b-Q4_K_M.gguf --no-gpu --port 9090
```

生成的 `docker-compose.yml` 示例（Ollama）：

```yaml
# Generated by llm-deploy-helper v1.0.0
# Engine: ollama
# Model: llama3.1:8b
# GPU: enabled

services:
  ollama:
    image: ollama/ollama:latest
    container_name: ollama-server
    ports:
      - "11434:11434"
    volumes:
      - ollama_models:/root/.ollama
    restart: unless-stopped
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: 1
              capabilities: [gpu]
    environment:
      - OLLAMA_KEEP_ALIVE=24h

volumes:
  ollama_models:
```

然后启动：

```bash
docker compose up -d
```

### 3. 生成 systemd 服务文件

```bash
$ llm-deploy-helper systemd --engine ollama --model llama3.1:8b
```

输出 `ollama-llm.service`，安装方法：

```bash
sudo cp ollama-llm.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable --now ollama-llm
```

## 支持的三款引擎

| 引擎 | 镜像 | 默认端口 | 特点 |
|------|------|----------|------|
| **Ollama** | `ollama/ollama:latest` | 11434 | 简单易用，CPU/GPU 自适应 |
| **vLLM** | `vllm/vllm-openai:latest` | 8000 | 高性能 GPU 推理，吞吐量最佳 |
| **llama.cpp** | `ghcr.io/ggerganov/llama.cpp:server` | 8080 | 轻量级，CPU 友好，支持量化模型 |

## 系统要求参考

| 模型大小 | 最低 VRAM | 最低 RAM | 推荐引擎 |
|----------|-----------|----------|----------|
| 7-8B (4-bit) | 无 (CPU) | 8 GB | llama.cpp, Ollama |
| 7-8B (FP16) | 16 GB | 22 GB | vLLM, Ollama |
| 13-16B (4-bit) | 无 (CPU) | 16 GB | llama.cpp, Ollama |
| 20B+ (FP16) | 24 GB+ | 32 GB+ | vLLM |

## 命令总览

| 命令 | 说明 |
|------|------|
| `llm-deploy-helper check` | 检测系统硬件，显示推荐引擎 |
| `llm-deploy-helper generate` | 生成 docker-compose.yml |
| `llm-deploy-helper systemd` | 生成 systemd 服务文件 |
| `llm-deploy-helper models` | 列出推荐模型及资源要求 |
