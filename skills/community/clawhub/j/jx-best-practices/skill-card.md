## Known Risks and Mitigations: <br>
Risk: Copyable examples may fetch remote media or contact external services, which can expose project data or create dependency on untrusted URLs. <br>
Mitigation: Prefer local assets for sensitive projects and use trusted or allowlisted remote URLs before copying examples into production code. <br>
Risk: Cloud transcription examples can send private or regulated audio to a third-party service. <br>
Mitigation: Use local transcription for sensitive content, keep API keys in environment variables, and obtain consent and approval before sending audio to cloud services. <br>
