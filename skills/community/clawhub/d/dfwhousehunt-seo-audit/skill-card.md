## Description: <br>
Crawl any website and get a 0-100 SEO score with 50+ checks across Technical, On-Page, Schema, Social, and Compliance categories. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Marketing teams, site owners, and agent workflows use this skill to request SEO audits, compliance-oriented checklists, competitor gap analysis, and draft SEO article output for websites. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The reviewed package describes a Python audit script, but that script is not included in the artifact. <br>
Mitigation: Install only after verifying the missing script from the publisher or use the skill as documentation for the expected workflow. <br>
Risk: Premium analysis uses ANTHROPIC_API_KEY and may send page content to an external AI service. <br>
Mitigation: Avoid private or sensitive pages unless external processing is acceptable, and use the API key only for premium features. <br>
Risk: Compliance checks can be mistaken for legal, medical, financial, or regulatory approval. <br>
Mitigation: Treat compliance output as a checklist and have qualified reviewers validate regulated claims before publication. <br>
