# Local Install Notes

This package is prepared for local installation into the OpenClaw workspace.

## Suggested Steps
1. Verify the zip file exists in `/workspace/downloads`
2. Inspect package contents before extracting
3. Copy or extract into `/workspace/skills/requirements-analysis`
4. Confirm `SKILL.md` and `_meta.json` are present

## Validation
- Package name should be `requirements-analysis`
- Version should be `1.0.0`
- Templates should exist under `templates/`
