## Description: <br>
Comprehensive AI security posture assessment across 14 enterprise security domains including identity, data protection, prompt injection defense, and compliance mapping. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Security leaders, security architects, and AI governance teams use this skill to assess enterprise AI controls, identify security gaps, and generate remediation-oriented findings across AI security and compliance domains. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Assessment requests may contain sensitive internal security posture details. <br>
Mitigation: Use pseudonymous session IDs, omit userId when possible, and submit redacted or synthetic assessment data unless the organization has approved sending real security posture information to this provider. <br>
