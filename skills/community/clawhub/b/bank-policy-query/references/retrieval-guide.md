# 检索与输出规则

## 数据读取

用 `feishu_sheet` 工具读取电子表格的「全国政策」工作表：
- action: `read`
- url 或 spreadsheet_token: `ZIfoscVEJhvsHttAwYocTCd2n1b`
- range（可选）: 不指定以读取全部数据

## 查询匹配规则

1. **城市匹配**：用户问的城市在「活动城市」列中，做包含匹配（如用户问"北京"，匹配活动城市含"北京"的行）
2. **区域聚合匹配**：如果用户问省份（如"河北省"），匹配该省份下所有城市
3. **模糊匹配**：用户可能用简称（如"上海"搜"上海市"），做模糊包含匹配
4. **无匹配处理**：如果没有匹配记录，明确告知用户暂无该地区银行合作活动

## 输出排序规则

在最终输出中，按以下优先级排序：

1. **活动类型**：通道优先于绑卡
2. **政策类型**：对私优先于对公，然后是连锁/不限等
3. **若有多条同类型**：按补贴费率从高到低排序，费率相同按免考核月数从多到少排序

## 每条活动的输出要素

每条活动必须包含以下信息：

- **银行**：银行名称
- **活动类型**：通道 / 绑卡
- **活动时间**：如 "2025/1/1-2026/12/31"
- **补贴费率**：如 "0.3%" 或 "≤0.38%"
- **政策类型**：对私 / 对公 / 个体法人 / 连锁
- **新客条款**：
  - 免考核期几个月（如有）
  - 免考核期封顶值
  - 或考核存款要求（如有）
- **存客考核期条款**：
  - 起步存款额
  - 各档次补贴封顶
- **特殊备注**（如有）："暂停新增"、"已终止"、"仅适用中投通道" 等

## 特殊备注处理规则

- 「暂停新增」「已终止」「暂停」→ 活动已不可新增，在输出时标注 ⚠️ **（暂停新增）**
- 「仅支持新客提报」「新客仅适用中投通道」→ 注明适用范围
- 「存客不予生效」→ 存客不推荐该活动
- 「仅适用中投通道商户」→ 活动仅限中投通道提报

## 精准输出要求

- **不得编造条款**：如果数据中没有明确的条款内容，如实说"暂无详细条款"
- **不得修改数值**：存款门槛、封顶值、费率等数值逐字引用原始数据
- **不得省略重要条件**：存款门槛、考核要求、时间限制等必须完整呈现
