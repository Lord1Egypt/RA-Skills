## Description: <br>
Design beautiful interfaces using 16+ design systems including Material You, Fluent Design, Apple HIG, Ant Design, Carbon Design, Shopify Polaris, Minimalism, Glassmorphism, Neo-Brutalism, Neumorphism, Skeuomorphism, Claymorphism, Swiss Design, and Atlassian Design, with expertise in Tailwind CSS, color harmonics, component theming, and accessibility (WCAG). <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and designers use this skill to plan and implement user interfaces across established design systems, including palette generation, component theming, Tailwind-oriented structure, and accessibility review. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The helper script permanently appends UI-design instructions to the project's .cursorrules. <br>
Mitigation: Run the script only in projects where updating AI instruction files is acceptable, then review .cursorrules after use. <br>
Risk: Added design guidance may conflict with controlled shared AI instructions. <br>
Mitigation: Avoid running the helper in projects with tightly managed instruction files unless the change has been approved. <br>
