# Emby 电视剧媒体库整理助手

## 简介

本技能用于整理电视剧文件夹，按照 Emby 媒体库标准命名规范重命名并重新组织文件结构。

**核心理念**：先预览、后执行 — 在你确认整理方案之前，绝不会移动或重命名任何文件。

---

## 目录结构规范

整理后的标准结构如下：

```
电视剧名称 (年份)/
├── Season 01/
│   ├── 电视剧名称 (年份) - S01E01.mkv
│   └── 电视剧名称 (年份) - S01E01.zh-CN.srt
├── Season 02/
│   ├── 电视剧名称 (年份) - S02E01.mkv
│   └── 电视剧名称 (年份) - S02E02.mkv
└── Season 00/          ← 特别篇/SP/OVA/番外等
    ├── 特别篇原文件名.mkv   ← 文件名保持原样
    └── SP原文件名.mkv
```

## 工作流程

### 第一步：扫描分析

只读扫描，不修改任何文件。技能会：

- 扫描文件夹下所有文件（视频、字幕、NFO、图片等）
- 识别电视剧名称（自动清理分类标签、画质信息、季数说明等干扰字符）
- 识别年份（从文件夹名或 NFO 文件读取）
- 识别季号和集号（支持多种格式）
- 判断是否包含特别篇

### 第二步：输出整理方案

扫描完成后，我会输出完整的整理方案，包含：

1. **识别结果摘要** — 电视剧名称、年份、季数、是否含特别篇等
2. **建议目录结构** — 树形展示整理后的完整结构
3. **重命名前后对照表** — 原始路径 → 整理后路径
4. **需要人工确认的问题** — 无法自动判断的条目
5. **Excel 导出信息** — 整理完成后 Excel 对照表的保存路径

### 第三步：等待确认

方案展示完毕后，**技能会暂停等待你的明确确认**。

你可以在这一步：

- 审阅对照表，检查是否有错误
- 提出修改意见
- 确认无误后说「确认执行」或「开始整理」
- 也可以说「取消」放弃本次整理

### 第四步：执行整理

你确认后，才开始实际执行：

- 按对照表逐文件移动和重命名
- 如果目标文件已存在，**不覆盖**，标记为「冲突」
- 全部完成后生成 Excel 对照表

### 第五步：生成 Excel

整理完成后，自动生成 Excel 文件，包含：

| 字段 | 说明 |
|------|------|
| 原始路径 | 整理前的完整路径 |
| 原始文件名 | 整理前的文件名 |
| 新路径 | 整理后的完整路径 |
| 新文件名 | 整理后的文件名 |
| 季号 | S01、S02、S00 等 |
| 集号 | E01、E02；特别篇填「特别篇」 |
| 处理状态 | 已整理 / 待确认 / 跳过 / 失败 |
| 说明 | 附加说明 |

保存位置：`/vol2/1000/SyncFile/OpenClawSync/电视剧名称_yyyyMMdd_HHmmss.xlsx`

## 路径清理规则

识别剧名时，会自动移除以下干扰字符：

| 类型 | 示例 | 清理结果 |
|------|------|----------|
| 分类标签 | `[动漫]` `[电视剧]` `[美剧]` `[日剧]` `[韩剧]` | 去除中括号及内容 |
| 季数说明 | `1-3季` `第一季` `第二季` `全集` `完结` | 去除 |
| 画质信息 | `1080p` `2160p` `4K` `720p` | 去除 |
| 编码信息 | `x264` `x265` `HEVC` `WEB-DL` `BluRay` | 去除 |
| 音频信息 | `AAC` `DTS` `Atmos` | 去除 |
| 色彩信息 | `HDR` `DV` `Dolby Vision` `10bit` | 去除 |
| 其他 | 发布组名称、多余空格/点号/横线 | 清理 |

**示例：**

```
[动漫]布鲁伊 1-3季 1080p Web-DL AAC → 布鲁伊
```

## 集数识别规则

支持识别以下所有格式：

| 原始格式 | 示例 | 转换结果 |
|----------|------|----------|
| S 格式 | `S01E01` `s01e01` | S01E01 |
| 短 S 格式 | `S1E1` | S01E01 |
| X 格式 | `1x01` | S01E01 |
| EP 格式 | `EP01` `Ep.01` `E01` | E01（季号另行判断） |
| 中文集 | `第01集` `第1集` | E01 |
| 日文话 | `01话` `第01话` | E01 |
| English | `Episode 01` | E01 |
| 纯数字 | `01` | E01（季号另行判断） |

**季号判断优先级：**

1. 文件名中含明确季号（`S01`、`第一季`等）→ 直接使用
2. 文件位于明确季文件夹（`Season 02`、`第二季`）→ 使用该季号
3. 无法判断时 → 标注「需要人工确认」

**多集合并：** 多集文件命名为 `电视剧名称 (年份) - S01E01-E02.mkv`

## 文件类型分类

| 文件类型 | 扩展名 |
|----------|--------|
| 视频 | `.mkv` `.mp4` `.avi` `.ts` `.m2ts` |
| 字幕 | `.srt` `.ass` `.ssa` `.sub` `.idx` |
| NFO | `.nfo` |
| 图片 | `.jpg` `.jpeg` `.png` `.webp` `.tbn` |
| 其他 | 以上之外的所有文件 |

**字幕命名规则：**

```
布鲁伊 (2018) - S01E01.mkv         ← 视频
布鲁伊 (2018) - S01E01.zh-CN.srt   ← 中文字幕
布鲁伊 (2018) - S01E01.en.srt      ← 英文字幕
```

特别篇（Season 00）的字幕保持原文件名风格，不强制改名。

## 安全约束

- ✅ **先预览后执行**：不确认绝不动文件
- ✅ **不覆盖**：目标文件已存在时，标记「冲突」并跳过
- ✅ **失败不中断**：单个文件失败不影响其他文件处理
- ✅ **信息不确定时标注**：不强行猜测，标注「需要人工确认」

## 常见问题

**Q: 整理后的文件夹应该放在哪里？**

A: 默认建议放在 Emby 媒体库的电视剧根目录。例如：`/vol3/1000/Movies/TVPlay/布鲁伊 (2018)`

**Q: 已有字幕文件名不规范怎么办？**

A: 技能会自动将字幕与同名视频匹配，并按规范命名。如果字幕无法匹配到视频，会在「需要人工确认」中列出。

**Q: 整理出错怎么办？**

A: 任何步骤失败都会记录在 Excel 的「说明」列中，不会中断其他文件的处理。你可以审阅 Excel 后手动处理异常文件。

**Q: 可以只整理部分文件吗？**

A: 可以在确认方案后告诉我「只整理第 X 到第 Y 条」或「跳过第 X 条」。
