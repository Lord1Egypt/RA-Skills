# 秒技·视频脚本Pro版

## 快速开始

### 基本使用

1. 打开技能
2. 输入产品和目标人群
3. 选择时长规格和平台
4. 获取完整视频内容矩阵

### 输入示例

```
产品名称：XXX无线蓝牙耳机
产品品类：3C电子
核心卖点：降噪、长续航、舒适佩戴
目标人群：25-35岁通勤人群
视频需求：全规格（15秒/30秒/60秒/3分钟）
目标平台：Amazon Shorts/TikTok/YouTube/独立站
```

## 功能概览

### 多时长脚本
获取：
- 15秒冲击型脚本
- 30秒标准型脚本
- 60秒深度型脚本
- 3分钟品牌型脚本

### 平台专属Hook
获取：
- Amazon Shorts开头3秒设计
- TikTok开头3秒设计
- YouTube开头设计
- 独立站视频设计

### A/B测试方案
获取：
- 3种角度脚本（痛点/功能/情感）
- 测试执行计划
- 效果评估标准

### 缩略图文案
获取：
- 封面标题+副标题
- CTA设计建议
- 多平台适配版本
