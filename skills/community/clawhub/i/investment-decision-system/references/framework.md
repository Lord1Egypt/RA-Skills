# INVEST 决策框架参考文档

## 框架概述

INVEST 是一个六维度投资决策框架，旨在将分散的信息转化为结构化的决策流程。它不做预测，只做评估——帮你把"我知道什么"转化为"我该怎么做"。

## 六维度详解

### I - Intent (意图) — 权重 15%

**核心问题：你为什么做这笔交易？**

- 你是在投资还是在赌博？
- 这笔交易是否符合你的整体投资目标和风险承受能力？
- 你的预期回报和持有时间是否合理？
- 如果亏损 50%，你的生活会受影响吗？

**评估标准：**
| 分数 | 描述 |
|------|------|
| 0-3 | 目标模糊，纯跟风，FOMO 或情绪驱动 |
| 4-6 | 有大致方向，但缺乏具体的投资计划 |
| 7-8 | 目标明确，与整体策略一致，有 exit plan |
| 9-10 | 目标极其清晰，完整的投资论点和退出策略 |

**常见陷阱：**
- "再不买就来不及了" — FOMO 不是投资意图
- "大家都说好" — 从众不是独立决策
- "我就想赚点快钱" — 投机心态，不是投资

---

### N - Numbers (数字) — 权重 20%

**核心问题：数据支持你的判断吗？**

- 估值水平：PE/PB/PS 与历史区间和同行对比
- 盈利能力：ROE/ROA/毛利率及其变化趋势
- 成长性：营收/利润增长率，自由现金流
- 技术面：趋势、成交量、支撑阻力位

**评估标准：**
| 分数 | 描述 |
|------|------|
| 0-3 | 缺乏关键数据，凭"感觉"决策 |
| 4-6 | 看过一些数据，但分析不深，未做交叉验证 |
| 7-8 | 多维度数据交叉验证，逻辑自洽 |
| 9-10 | 全面深入的数据分析，有量化模型支撑 |

**关键指标速查：**
- A股：PE < 行业均值*0.7 可能是价值洼地；PE > 行业均值*2 需警惕
- ROE > 15% 且持续5年以上是优质公司的共同特征
- 营收增速 > 15% 对应高成长，< 5% 可能是成熟期甚至衰退

---

### V - Value (价值) — 权重 20%

**核心问题：现在买，值不值？**

- 你估计的内在价值是多少？用什么方法？
- 当前价格与内在价值的差距（安全边际）是多少？
- 公司的护城河是什么？可持续吗？
- 最差情况下，这个资产值多少钱？

**估值方法选择指南：**
| 公司类型 | 推荐方法 | 示例 |
|---------|---------|------|
| 稳定盈利 | DCF / PE / PEG | 茅台、银行 |
| 高成长 | PEG / PS / DCF | 新能源、科技 |
| 周期股 | PB / 重置成本 | 钢铁、化工 |
| 金融 | PB / ROE-PB | 银行、保险 |
| 亏损/初创 | PS / 用户价值 | 创新药、SaaS |

**评估标准：**
| 分数 | 描述 |
|------|------|
| 0-3 | 没有估值概念，只看价格涨跌 |
| 4-6 | 有粗略估值，但安全边际不足（<20%） |
| 7-8 | 估值合理，安全边际充裕（>30%） |
| 9-10 | 深度低估（>50%安全边际），护城河清晰可验证 |

---

### E - Edge (优势) — 权重 10%

**核心问题：你凭什么比别人判断得更准？**

- 你对这个行业/公司有什么独特了解？
- 你的信息优势来源？（行业经验、专业背景、独特数据）
- 市场共识与你的判断差异在哪？为什么你是对的？
- 如果你的信息已经公开，价格已经反映了吗？

**常见的信息优势类型：**
1. **行业洞察**：你在该行业工作，了解前瞻性趋势
2. **专业分析**：你有独特的分析框架或数据挖掘能力
3. **时间维度**：你愿意持有比市场平均更长时间
4. **逆向思维**：市场恐慌时你冷静，市场狂热时你警惕

**⚠️ 诚实评估：** 大多数人没有 genuine edge。给这个维度打高分需要极强的理由。

**评估标准：**
| 分数 | 描述 |
|------|------|
| 0-3 | 没有信息优势，认知与大众完全一致 |
| 4-6 | 有一些行业了解，但 Information Edge 不显著 |
| 7-8 | 有明显的认知差，信息不对称优势 |
| 9-10 | 深度行业洞察，大众尚未认知到关键价值 |

---

### S - Safety (安全) — 权重 20%

**核心问题：如果判断错了，你会损失多少？**

- 这笔交易的最大可能亏损是多少？你能承受吗？
- 止损设在哪里？基于技术位还是资金管理？
- 这笔交易占总资产的比例是否合理？
- 投资组合是否足够分散？有没有集中度风险？
- 黑天鹅场景下，你的应对方案是什么？

**仓位管理规则：**
| 参数 | 保守 | 适中 | 激进 |
|------|------|------|------|
| 单票最大仓位 | 5-10% | 15-20% | 25-30% |
| 单行业最大仓位 | 20% | 40% | 50% |
| 单笔最大亏损（占资产） | 1% | 2% | 3-5% |
| 硬止损 | 5-8% | 8-10% | 10-15% |
| 最低现金 | 20% | 10% | 5% |

**评估标准：**
| 分数 | 描述 |
|------|------|
| 0-3 | 没有止损计划，仓位过重或过于集中 |
| 4-6 | 有止损意识但缺乏系统性执行 |
| 7-8 | 止损明确，仓位控制规范，组合合理分散 |
| 9-10 | 完善的风控体系，压力测试充分，极端情景有预案 |

---

### T - Timing (时机) — 权重 15%

**核心问题：为什么是现在？**

- 当前市场处于什么周期阶段？（牛/熊/震荡/转折）
- 技术面是否有明确的入场/出场信号？
- 近期是否有重大事件影响？（财报、政策、行业动态）
- 分批建仓还是一次性买入？各有利弊是什么？
- "如果错过这个时机，你会后悔吗？" — 如果答案是"不会"，就再等等。

**市场周期位置判断：**
| 阶段 | 特征 | 策略 |
|------|------|------|
| 熊市末期 | 极度悲观、成交量萎缩、估值历史低位 | 左侧分批建仓 |
| 牛市初期 | 信心恢复、突破关键阻力、成交量放大 | 右侧确认后加仓 |
| 牛市中期 | 普涨、情绪乐观、新股民入场 | 持有为主，谨慎追高 |
| 牛市末期 | 过度乐观、估值泡沫、利空不跌转利空真跌 | 分批减仓 |
| 熊市初中期 | 恐慌、踩踏、政策救市失效 | 观望或对冲 |

**评估标准：**
| 分数 | 描述 |
|------|------|
| 0-3 | 追涨杀跌，FOMO 或恐慌驱动 |
| 4-6 | 粗略判断时机，缺乏系统分析 |
| 7-8 | 时机选择有逻辑支撑，分批执行 |
| 9-10 | 精准把握周期节点，策略清晰可执行 |

---

## 综合评分与决策

### 加权计算
```
总分 = Intent×15% + Numbers×20% + Value×20% + Edge×10% + Safety×20% + Timing×15%
```

### 决策阈值

| 总分 | 建议 | 行动 |
|------|------|------|
| ≥ 7.0 | 强烈推荐 | 按仓位计算器执行，严格止损 |
| 5.0-6.9 | 谨慎推荐 | 可以执行，但降低仓位到标准的50% |
| 3.0-4.9 | 建议等待 | 暂不行动，重新评估薄弱维度 |
| < 3.0 | 应回避 | 放弃此交易，记录在案 |

### 仓位计算公式

使用**固定比例法**（Fixed Fractional）：

```
可买股数 = (总资产 × 单笔风险%) ÷ |入场价 - 止损价|
仓位占比 = (可买股数 × 入场价) ÷ 总资产 × 100%
```

如果计算出的仓位超过单票上限，按上限截断。

## 决策后复盘清单

每笔交易结束后（无论盈亏），回答以下问题：

1. **决策执行度**：实际执行是否完全按照决策计划？（0-10分）
2. **信息充分度**：决策时掌握的信息是否足够？遗漏了什么？
3. **情绪影响度**：情绪（贪婪/恐惧/FOMO）在多大程度上影响了决策？（0-10分，越低越好）
4. **运气成分**：结果是能力还是运气？如果重来一次，会做出同样的决策吗？
5. **改进点**：这次经历最值得改进的一个方面是什么？

---

## 长期迭代

- **每周**：更新持仓市值，检查风控合规
- **每月**：回顾所有决策记录，统计胜率和盈亏比
- **每季度**：重新评估投资画像（目标、风险承受能力是否变化）
- **每年**：全面复盘，调整 INVEST 各维度权重
