# ⭐ 质量评分标准

## 🎯 评分体系说明

采用**五维度评分法**，每个维度20分，总分100分。

| 等级 | 分数区间 | 处理方式 |
|------|---------|---------|
| 优秀 | 90-100分 | 直接通过，可作为范本 |
| 良好 | 80-89分 | 按建议优化后通过 |
| 合格 | 70-79分 | 必须优化，重新评审 |
| 不合格 | < 70分 | 打回重写 |

---

## 📊 各维度评分细则

### 维度1：文笔流畅度（20分）

#### 评分标准
| 得分 | 标准 |
|------|------|
| 18-20 | 语言生动自然，完全没有AI痕迹，有个人风格，代入感强 |
| 15-17 | 语言流畅，描写到位，基本没有AI痕迹，阅读体验好 |
| 12-14 | 语言通顺，没有明显语病，但略显平淡，有轻微AI痕迹 |
| 8-11 | 语言基本通顺，但比较生硬，有明显AI套话 |
| < 8 | 语病多，逻辑混乱，完全是AI模板化输出 |

#### 扣分点
- 出现明显的AI套话，每处扣1分
- 出现语病，每处扣1分
- 重复啰嗦，每处扣1-2分

#### 加分点
- 有精彩的金句，每句加1-2分
- 氛围感营造特别好，加2-3分

---

### 维度2：人物塑造（20分）

#### 评分标准
| 得分 | 标准 |
|------|------|
| 18-20 | 人物立体鲜活，性格鲜明，行为完全符合设定，让人印象深刻 |
| 15-17 | 人物性格明确，行为符合逻辑，有辨识度 |
| 12-14 | 有人物设定，但不够立体，基本符合设定 |
| 8-11 | 人物扁平，像工具人，偶尔有OOC（不符合性格）行为 |
| < 8 | 人物完全OOC，没有性格，纯工具人 |

#### 扣分点
- 人物OOC，每处扣2-3分
- 人物行为动机不明确，每处扣2分
- 对话不符合人物身份，每处扣1分

#### 加分点
- 人物有让人印象深刻的细节，加2-3分
- 人物成长弧线清晰自然，加3-5分

---

### 维度3：情节节奏（20分）

#### 评分标准
| 得分 | 标准 |
|------|------|
| 18-20 | 节奏完美，张弛有度，爽点密集，钩子有力，让人欲罢不能 |
| 15-17 | 节奏合理，有爽点有悬念，阅读体验好 |
| 12-14 | 节奏基本合格，但略显平淡，爽点不够突出 |
| 8-11 | 节奏混乱，要么太拖沓，要么太跳跃，没有爽点 |
| < 8 | 情节崩坏，逻辑混乱，完全不知所云 |

#### 扣分点
- 情节注水，扣3-5分
- 节奏拖沓，扣2-3分
- 没有结尾钩子，扣3分
- 爽点铺垫不足，扣2-3分

#### 加分点
- 结尾钩子特别精彩，加2-3分
- 爽点设计巧妙，让人意想不到，加3-5分
- 伏笔回收设计精妙，加2-3分

---

### 维度4：逻辑严谨性（20分）

#### 评分标准
| 得分 | 标准 |
|------|------|
| 18-20 | 逻辑严密，无任何矛盾，因果关系清晰，完全符合世界观设定 |
| 15-17 | 逻辑通顺，没有明显矛盾，基本符合设定 |
| 12-14 | 有小bug，但不影响整体阅读，基本逻辑自洽 |
| 8-11 | 有明显逻辑矛盾，需要脑补才能理解 |
| < 8 | 逻辑崩坏，矛盾百出，完全无法理解 |

#### 扣分点
- 逻辑矛盾，每处扣3-5分
- 不符合世界观设定，每处扣3-5分
- 因果关系不成立，每处扣2-3分

#### 加分点
- 逻辑设计特别精妙，加3-5分
- 前后呼应巧妙，加2-3分

---

### 维度5：创意新颖度（20分）

#### 评分标准
| 得分 | 标准 |
|------|------|
| 18-20 | 创意十足，完全不套路，让人眼前一亮，有记忆点 |
| 15-17 | 有新意，虽然有套路元素，但有自己的创新 |
| 12-14 | 中规中矩，不算新颖，但也不算老套 |
| 8-11 | 比较套路，基本是常见桥段的组合 |
| < 8 | 完全是老套桥段的堆砌，没有任何新意 |

#### 扣分点
- 明显的套路化桥段，每处扣1-2分
- 抄袭或过度借鉴，扣5-10分

#### 加分点
- 有让人眼前一亮的创意，加3-5分
- 反套路设计成功，加3-5分

---

## 🎯 通过标准

### 单章通过标准
1. **总分 ≥ 70分**
2. **没有红线内容违规**
3. **字数符合要求**
4. **没有严重的逻辑矛盾**

### 批量通过标准（如10章一起评审）
1. **平均分 ≥ 75分**
2. **单章最低分 ≥ 70分**
3. **没有红线内容违规**
4. **整体连贯性良好**

---

## 📝 评审流程

### 第一步：红线检查
- 检查是否违反内容审查红线
- 检查字数是否符合要求
- 任一不通过 → 直接打回，不评分

### 第二步：五维度评分
- 按上述五个维度分别评分
- 计算总分

### 第三步：详细评价
- 指出优点
- 指出问题和不足
- 给出具体的修改建议

### 第四步：结论
- 90分以上 → 直接通过
- 70-89分 → 按建议优化后通过
- < 70分 → 打回重写

---

## ⚠️ 注意事项

1. **评分要客观公正**，不能带个人偏好
2. **修改建议要具体可执行**，不能只说"写得不好"
3. **优点也要指出**，不能只批评不表扬
4. **红线问题零容忍**，只要有红线问题，不管写得多好都打回

---

**生效时间**：永久生效
**适用范围**：所有小说项目的质量评审
**最终解释权**：总策划
