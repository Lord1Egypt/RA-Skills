## Description: <br>
Depth Memory provides a workspace-local long-term knowledge layer for storing, indexing, and retrieving structured Markdown memories by keyword. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[qshfu](https://clawhub.ai/user/qshfu) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent operators use this skill to maintain a persistent, searchable knowledge base for technical concepts, tool usage notes, skill experience, and long-term project knowledge. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Persistent Markdown memory files may retain sensitive, private, regulated, or confidential information if users store it in the workspace. <br>
Mitigation: Use the skill only for information intended to be retained, reviewed, and possibly synced or shared with the workspace; avoid storing credentials, tokens, private personal data, regulated data, and confidential documents. <br>
