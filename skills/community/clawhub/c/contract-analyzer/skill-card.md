## Description: <br>
Analyze and review contract documents for key clauses, high-risk terms, contract differences, summaries, and review checklists. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[pressbytesagain](https://clawhub.ai/user/pressbytesagain) <br>

## Use Case: <br>
External users and agents use this skill to run local contract review helpers over user-selected text files, including clause extraction, risk checks, document comparison, executive summaries, and due diligence checklists. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Contract files can contain confidential agreement terms that may be printed to terminal output or logs. <br>
Mitigation: Run only on contract files the user explicitly selects, avoid shared terminals, CI logs, recorded sessions, and chat transcripts for confidential agreements, and redact sensitive content when needed. <br>
Risk: The artifact runs local shell and Python scripts against files on the user's machine. <br>
Mitigation: Review the script before execution and run it in a controlled workspace with only the intended contract text files available. <br>
Risk: Server evidence lists crypto and purchase capability tags, but the security guidance says those tags appear unrelated to the reviewed artifact behavior. <br>
Mitigation: Do not treat the skill as crypto or purchase automation, and have the publisher correct the capability tags before relying on automated policy routing. <br>
