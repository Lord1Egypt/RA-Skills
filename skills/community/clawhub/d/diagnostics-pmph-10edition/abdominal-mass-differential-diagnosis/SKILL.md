# 腹部肿块七要素鉴别诊断

## 适用场景
当在腹部触到非正常结构的肿块，且已排除腹直肌、横结肠、乙状结肠粪块等正常可触及结构时，使用本技能进行系统评估。

## 执行步骤

依次评估以下七个要素：

1. **部位**：
   - 上腹中部 → 胃、胰腺；
   - 右肋下 → 肝、胆；
   - 两侧腹部 → 结肠、肾；
   - 脐周/右下腹不规则压痛肿块 → 肠结核粘连；
   - 下腹两侧类圆形可活动压痛肿块 → 腹腔淋巴结；
   - 深在坚硬不规则 → 腹膜后肿瘤；
   - 腹股沟韧带上方 → 盆腔器官（如卵巢）。

2. **大小**：
   - 测量三径（纵、横、深），可用实物比喻（如鸡蛋、拳头）；
   - 巨大肿块多见于卵巢、肾、肝、胰、子宫；
   - 胃肠道肿物通常较小（因早现梗阻）；
   - 大小可变或自行消失 → 考虑肠痉挛或充气肠袢。

3. **形态**：
   - 圆形光滑 → 良性（囊肿、淋巴结）；
   - 不规则凸凹 → 恶性或炎性；
   - 条索状多变 → 蛔虫团或肠套叠；
   - 右上腹卵圆形 → 胆囊积液；
   - 左上腹有切迹 → 脾脏。

4. **质地**：
   - 实质性（柔韧至坚硬）→ 肿瘤、炎性浸润；
   - 囊性（柔软）→ 囊肿、脓肿。

5. **压痛**：
   - 明显压痛 → 炎性（如阑尾脓肿、肠结核）；
   - 肿瘤压痛轻重不一。

6. **搏动**：
   - 腹中线膨胀性搏动 → 腹主动脉瘤（正常腹主动脉宽度 ≤3.5cm）；
   - 可伴震颤。

7. **移动度**：
   - 随呼吸上下移动 → 肝、脾、胃、肾及其肿物；
   - 可手推动 → 胃、肠或肠系膜来源；
   - 移动度大 → 带蒂肿物或游走脏器；
   - 固定不动 → 炎性肿块、脓肿、腹膜后肿瘤。

## 附加检查
区分肿块是否源于腹壁或腹腔内：让患者抬头或抬肩，观察肿块是否随腹肌紧张而更明显。

## 输出建议
综合七要素判断肿块性质，决定是否需影像学检查或转诊专科。

> ⚠️ 注意：肥胖或腹壁紧张会降低触诊准确性；部分肿块必须依赖影像学确诊。
