### 1.2 AI写作工具使用指南

#### 文心一言（重点推荐）

**使用技巧**：
```
角色设定：你是技术成果转化领域的资深政策分析师
背景补充：我的公众号受众是企业管理者、科研人员、政府官员
风格要求：权威、有条理、有温度，避免过于学术化
输出格式：先列大纲，再展开内容
```

**高效提示词模板**：
```
请帮我撰写一篇关于[主题]的公众号文章，要求：
1. 标题吸引人，含[核心关键词]
2. 结构清晰，包含[要点1]、[要点2]、[要点3]
3. 开头300字要有吸引力，可从[痛点/数据/政策]切入
4. 包含2-3个具体案例
5. 结尾有互动引导
6. 字数1500-2000字
```

#### 通义千问

**使用技巧**：
- 适合处理长文档，可直接上传政策原文让AI解读
- 「效率工具」中的「长文创作」功能适合深度内容
- 支持多语言，可中英对照翻译政策

**高效提示词模板**：
```
请阅读以下政策文件，[粘贴政策全文]
然后帮我：
1. 提炼3-5个核心要点（每点100字内）
2. 分析对[企业/高校/园区]的影响
3. 列出申报条件/注意事项
4. 撰写一篇1500字的公众号推文
```

#### Kimi

**使用技巧**：
- 适合处理超长文本，可上传PDF政策文件
- 「探索助手」适合做行业研究
- 支持追问，可深度挖掘某个观点

### 1.3 AI辅助写作工作流

```
Step 1 - 选题确认：AI生成3-5个标题候选 → 人工选定
Step 2 - 大纲生成：AI列出文章结构 → 人工调整完善
Step 3 - 内容撰写：AI初稿 → 人工润色（加入行业洞察）
Step 4 - 标题优化：AI生成10个标题 → 人工选择+微调
Step 5 - 质量检查：AI模拟用户反馈 → 人工调整
```

**适用场景**：

- 团队协作编辑

### 2.3 推荐排版规范（科技政策类公众号）

**整体风格**：
```

辅助色：白色背景 + 灰色正文 + 蓝色强调
字体：正文微软雅黑/思源黑体
风格：简洁专业、留白充足、信息层次清晰
```

**排版参数建议**：
| 元素 | 参数设置 |
|------|----------|
| 正文字号 | 15-16px |
| 标题字号 | H2=18-20px / H3=16-17px |
| 行间距 | 1.75-2.0 |
| 字间距 | 1px |
| 段间距 | 16px |
| 页边距 | 12-15px |
| 图片宽度 | 100%（全宽）或 70%（居中） |

### 3.2 各工具使用指南

#### 新榜

**核心功能**：
- 搜索领域公众号排行榜
- 查看竞品公众号的热门文章
- 监控竞品更新动态
- 行业热词趋势

**使用技巧**：
```
1. 搜索"技术成果转化"、"科技政策"等关键词
2. 关注领域内TOP20公众号
3. 查看他们的热门文章，分析选题方向
4. 设置更新提醒，第一时间获取竞品动态
```

#### 微信指数

**使用入口**：
- 微信搜一搜 → 搜索框输入「微信指数」
- 或小程序搜索「微信指数」

**使用场景**：
```
验证选题热度：
输入「科技成果转化」，查看近期热度趋势
热度上升时 → 抓紧发布相关选题
热度下降时 → 考虑调整选题方向
```

#### 百度指数

**使用技巧**：
```
1. 搜索需求图谱，了解用户关联搜索词
2. 对比近30天/近90天趋势变化
3. 地域分布帮助判断目标受众
4. 人群画像辅助内容定位
```
