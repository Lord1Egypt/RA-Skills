## Description: <br>
Pine Script Developer helps agents write, modify, debug, and optimize TradingView Pine Script indicators and strategies. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[lufei888-yi](https://clawhub.ai/user/lufei888-yi) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, traders, and agent users use this skill to create and review TradingView Pine Script v5 indicators, strategies, alerts, backtests, and chart utilities. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The optional JavaScript helper can create or overwrite local project files, and project names are not path-sanitized. <br>
Mitigation: Run the helper only in a workspace you control, avoid untrusted or path-like project names, and review generated files before use. <br>
Risk: Generated trading strategy code may be incorrect or unsuitable for live trading. <br>
Mitigation: Review and test all Pine Script output in TradingView before using it for trading decisions. <br>
