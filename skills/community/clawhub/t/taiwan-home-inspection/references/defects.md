# 常見瑕疵與嚴重程度分級

## 分級標準

| 分級 | 定義 | 影響 | 處理建議 |
|------|------|------|----------|
| 🔴 大缺失 | 影響結構安全、公共安全或居住安全 | 居住風險、法律責任 | 必須修復，建議專業廠商 |
| 🟡 中缺失 | 影響正常功能，但不涉及安全 | 生活不便、潛在惡化 | 建議修復或交屋時折價 |
| 🟢 小缺失 | 美觀問題或輕微功能異常 | 不影響居住 | 記錄告知、自行決定 |
| ℹ️ 注意事項 | 正常使用損耗或潛在風險 | 未來可能需要關注 | 知會買方即可 |

---

## 🔴 大缺失列表

### 結構類
| 項目 | 判斷標準 |
|------|----------|
| 樑柱45度剪力裂縫 | 裂縫寬度>0.3mm，長度貫穿構件 |
| 鋼筋裸露鏽蝕 | 混凝土保護層剝落，鋼筋已生鏽 |
| 柱體膨脹(爆柱) | 柱表面出現水平裂縫、混凝土剝落 |
| 樓板裂縫貫穿 | 從樓下可見裂縫方向與樓板同向 |
| 承重牆被打除 | 非經結構技師簽證的牆體變更 |
| 建築明顯傾斜 | 鉛垂儀量測偏差>1/200 |
| 氯離子超標(海砂屋) | 檢測值>0.3kg/m³（依現行標準） |
| 輻射鋼筋 | 檢測值>0.5μSv/h |

### 水電類
| 項目 | 判斷標準 |
|------|----------|
| 配電箱過熱燒焦 | 箱體有明顯燒焦變色痕跡 |
| 未裝漏電斷路器 | 總開關無漏電保護功能 |
| 電線老化裸露 | 電線絕緣層脆化、破損 |
| 水管破裂漏水 | 持續性滲漏無法止漏 |
| 瓦斯漏氣 | 泡沫水測試有氣泡冒出 |

### 衛浴類
| 項目 | 判斷標準 |
|------|----------|
| 樓上漏水至衛浴天花 | 持續滴水、天花板發黑 |
| 馬桶排糞管滲漏 | 底座周圍有廢水滲出 |
| 熱水器安裝通風不良 | 室內型熱水器無強制排氣 |

---

## 🟡 中缺失列表

| 項目 | 判斷標準 |
|------|----------|
| 窗框滲水 | 颱風/大雨後窗框周圍有水漬，但非持續性 |
| 地磚隆起/空心磚 | >3片連續空心或已隆起 |
| 衛浴防水失效 | 淋浴區外牆面有滲水潮濕 |
| 排水管堵塞 | 排水速度慢、易回堵 |
| 天花板水漬(已乾) | 漏水已修復但留痕跡 |
| 插座無接地線 | 三孔插座實測僅兩孔通電 |
| 紗窗破損 | 無法有效防蚊蟲 |
| 氣密窗氣密失效 | 窗戶關閉後仍有明顯風聲 |
| 壁癌 | 牆面有白華結晶，面積>0.5m² |
| 抽油煙機排煙不良 | 運轉但油煙無法有效排出 |
| 蓄水池滲漏 | 水塔或蓄水池有滲水情況 |
| 廚下管線滲漏 | 排水管接頭處有輕微滲漏 |

---

## 🟢 小缺失列表

| 項目 | 判斷標準 |
|------|----------|
| 牆面油漆剝落 | 小面積油漆脫落，不影響結構 |
| 磁磚縫隙缺填 | 部分磁磚縫隙未填實 |
| 門片開關雜音 | 門鉸鏈需上油或微調 |
| 插座鬆動 | 插頭插入後易鬆脫 |
| 水龍頭滴水 | 關閉後偶有滴漏，更換墊片即可 |
| 燈具不亮 | 燈泡/燈管更換即可 |
| 櫥櫃門片不正 | 鉸鏈調整即可對齊 |
| 窗軌卡澀 | 窗軌需清潔或潤滑 |
| 牆面小裂紋(非結構) | 寬度<0.2mm，表面漆層裂縫 |
| 把手鬆脫 | 需重新鎖緊 |

---

## ℹ️ 注意事項列表

| 項目 | 說明 |
|------|------|
| 管線過於老舊 | 屋齡>20年，建議預防性更換 |
| 外牆防水層年限 | 屋齡>15年，建議注意外牆防水 |
| 窗戶氣密條老化 | 建議更換以達節能效果 |
| 熱水器使用年限 | 一般8-10年建議更換 |
| 配電線路容量 | 老舊建築可能無法負荷現代用電 |
| 冷媒管預留孔 | 未預留日後裝冷氣需處理 |
| 頂樓加蓋漏水風險 | 既存違建需注意防水維護 |
| 管線間隔音 | 老舊建築樓板隔音效果有限 |
| 電梯保養記錄 | 大樓電梯需定期保養檢查 |
