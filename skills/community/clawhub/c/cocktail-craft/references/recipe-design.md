# 鸡尾酒配方设计方法论

## Cocktail Codex 六根配方框架

所有鸡尾酒从 6 种根配方演化而来。设计新酒时先定位框架，再替换材料。

### 1. Old Fashioned 家族
- 公式：Spirit + Sugar + Bitters
- 核心：基酒为主角，糖和苦精是配角，冰和水的稀释是隐形维度
- 变体方向：换基酒（龙舌兰→Oaxaca OF）、换糖（枫糖/龙舌兰糖浆）、换苦精、加少量利口酒（1/4 oz 以内）

### 2. Martini / Manhattan 家族
- 公式：Spirit + Fortified Wine (Vermouth, Sherry, etc.)
- 核心：基酒与加强酒的博弈，比例从 2:1 到 15:1
- 变体方向：换味美思类型（dry/sweet/blanc）、换基酒、换苦精、换修饰（橄榄/柠檬皮/樱桃）

### 3. Daiquiri 家族
- 公式：Spirit + Citrus + Sugar
- 核心：酸甜平衡，酒体轻盈，shake 后清爽
- 变体方向：换基酒（Gimlet=Gin、Whiskey Sour=Whiskey）、加蛋清（Boston Sour）、加薄荷、加苦精

### 4. Sidecar / Margarita 家族
- 公式：Spirit + Citrus + Liqueur
- 核心：利口酒既提供甜度也提供复杂度，通常比 Daiquiri 家族更浓烈
- 变体方向：换基酒、换柑橘（柠檬→青柠→葡萄柚）、换利口酒（Triple Sec→Cointreau→Maraschino→Chartreuse）

### 5. Highball / Collins 家族
- 公式：Spirit + Bubbles/Water + Modifier
- 核心：气泡延长风味、降低 ABV、增加饮用量
- 变体方向：换气泡类型（苏打→姜汁啤酒→汤力水→香槟）、加柑橘、加糖浆、加苦精

### 6. Flip / Egg 家族
- 公式：Spirit + Egg/Dairy + Sugar
- 核心：蛋或奶油提供质地和厚度，对技法要求最高（干摇乳化）
- 变体方向：全蛋→蛋清→蛋黄、牛奶→奶油→椰奶、加香料（肉豆蔻/肉桂）

## 风味平衡公式

### 酸甜平衡 (Sweet-Sour Balance)
- 基本比例：酸（柠檬/青柠汁）：糖（1:1 简单糖浆）= 3:2 至 1:1
- 柠檬汁：糖浆 = 22.5ml : 15ml（3:2）是标准起点
- 偏酸：可加糖、换更甜的利口酒
- 偏甜：可加酸、减糖浆、加苦精对冲

### 酒精度控制
- 短饮 (Short Drink)：60-90ml 总液量，通常 Stir/Shake + Strain，不加冰
- 长饮 (Long Drink)：120-240ml 总液量，加冰或气泡
- ABV 控制：
  - Short: 20-30% ABV（不加冰饮用）
  - Long: 8-15% ABV（冰融化会继续稀释）

### 稀释率 (Dilution)
- Stir：约 20-25% 水量增加（45 秒 stirring）
- Shake：约 25-30% 水量增加（10-15 秒 shaking）
- 这是配方设计中的隐形因素——最终饮用的 ABV 比配方计算的低

### 温度对感知的影响
- 低温抑制甜味感知：冰饮需要比常温饮更多糖才能达到相同甜度感
- 低温增强苦味感知：冰镇的 Negroni 比常温更苦
- 碳酸增强酸感：Highball 家族的气泡使酸味更突出

## 风味搭配理论

### 互补搭配 (Complementary)
- 相似风味家族配对：柑橘+柑橘、草本+草本
- 例子：Gin（杜松子）+ 接骨木花利口酒 → 花草共鸣

### 对比搭配 (Contrast)
- 甜+酸、甜+苦、咸+甜
- 例子：Negroni（甜味美思+苦 Campari）、Margarita（甜橙酒+酸青柠）

### 桥梁搭配 (Bridge)
- 用一个中间风味连接两个不认识的风味
- 例子：威士忌+咖啡→巧克力苦精是桥梁；朗姆+菠萝→椰子奶油是桥梁

### 跨界搭配 (Cross-Category)
- 从烹饪借风味组合：甜点风味（提拉米苏马天尼）、沙拉风味（番茄金酒）
- 从地域借风味：泰式（香茅+椰奶+辣椒）、日式（抹茶+柚子+清酒）

## 配方开发流程

### Step 1: 确定概念
- 这杯酒要讲什么故事？（季节/地点/情绪/食材）
- 这是什么类型的酒？（短饮/长饮/开胃/餐后/派对？）
- 目标风味方向？（清爽/浓烈/甜/苦/酸/咸/鲜？）

### Step 2: 定位根配方
- 符合哪个家族？
- 先按标准模板写配方，再替换个别材料

### Step 3: 平衡草案
- 画出 4 个维度的草图：基酒/甜/酸/苦
- 先用便宜材料试，等方向确定再用好酒

### Step 4: 迭代
- 第一轮：调酸甜比例
- 第二轮：调基酒比例（不同品牌差异大）
- 第三轮：细节微调（苦精几滴、装饰选择）
- 记录每一次试饮的用量和结果

### Step 5: 技法选择
- Shake（含果汁/蛋清/奶油 → 需要快速冷却+乳化）
- Stir（纯烈酒+少量修饰 → 需要精准稀释不破坏质感）
- Build（长饮+气泡 → 直接在杯里组装）
- Throw/Roll（想要搅拌的稀释但不想太清澈 → 两杯之间反复倒）
- Blend（含冰沙/固体水果 → 打碎混合）

## 常见错误检查清单

- 柠檬汁 + 奶油/牛奶 → 絮凝（酸使蛋白质变性，除非走奶洗路线）
- 过甜的利口酒（比如某些 Crème de Cassis）+ 简单糖浆 → 双重甜度
- 气泡饮料 + Shake → 炸罐
- 高 ABV (>50%) + 慢煮 >55°C → 酒精挥发
- 低温 Stir 不够 → 稀释不足、口感过烈
- 蛋清没有干摇直接加冰 → 泡沫不充分
- 奶洗用低脂/脱脂牛奶 → 酪蛋白不足、澄清效果差
- 浸泡时间过长（尤其是茶叶和苦味草药）→ 过度萃取变苦
