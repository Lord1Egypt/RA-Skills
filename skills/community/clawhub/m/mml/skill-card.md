## Description: <br>
Build 3D scenes and interactive experiences using MML (Metaverse Markup Language) for the Otherside metaverse and other MML-compatible environments. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[honeybee1130](https://clawhub.ai/user/honeybee1130) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and world builders use this skill to create MML scenes, objects, animations, media, interactions, and reference markup for MML-compatible metaverse environments. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated MML can include scripts, event handlers, links, frames, and remote media assets that affect shared worlds. <br>
Mitigation: Review generated MML before publishing, especially scripts, event handlers, links, frames, and remote asset URLs. <br>
Risk: Position probes and chat probes can collect participant location or chat data in a shared scene. <br>
Mitigation: Clearly disclose chat or position tracking to participants before using those features. <br>
