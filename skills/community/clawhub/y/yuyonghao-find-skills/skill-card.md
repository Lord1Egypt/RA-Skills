## Description: <br>
Helps users discover and install agent skills from ClawHub when they are looking for specialized functionality that may exist as an installable skill. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[yuyonghao-123](https://clawhub.ai/user/yuyonghao-123) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill to find relevant installable skills for specialized tasks, then receive search guidance, install commands, and reference links before deciding whether to add a skill. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can steer broad help requests toward persistent third-party skill installation, including global installs that skip confirmation. <br>
Mitigation: Use it only when skill discovery is intended; review the suggested skill source, publisher, page, and install command before installing, and avoid global confirmation-skipping installs unless the package is trusted. <br>
Risk: Search results or suggested skills may not match the user's task or security expectations. <br>
Mitigation: Treat results as options to evaluate, not automatic recommendations, and continue with general assistance when no suitable trusted skill is found. <br>
