# 常見糾紛與處理方式

## 糾紛處理原則

1. **先溝通，再存證，最後走法律**
2. **保留完整證據鏈**（對話記錄、照片、單據）
3. **租金補貼與協商＞訴訟**（訴訟成本高）
4. **第三方調解＞法院判決**

---

## 糾紛類型與處理

### 1. 欠繳租金

| 階段 | 處理方式 | 法律依據 |
|------|----------|----------|
| 遲繳3天內 | Line提醒（溫和） | — |
| 遲繳5-7天 | 正式Line訊息催告 | 租約約定 |
| 遲繳15天 | 存證信函催告 | 民法第440條 |
| 遲繳≥2個月 | 終止租約+請求返還房屋 | 民法第440條第2項 |

**實務流程：**
```
Line提醒 → Line正式催告 → 存證信函催告30日 
→ 期滿未繳 → 終止租約 → 調解 → 支付命令 → 強制執行
```

**總時間：約3~6個月**（從欠租到收回房屋）

### 2. 房屋損壞

| 情況 | 責任歸屬 | 處理 |
|------|:--------:|------|
| 自然耗損（壁癌、管線老化） | 房東負擔 | 房東維修 |
| 房客使用不當（窗簾扯破） | 房客負擔 | 照價賠償或修復 |
| 天災（颱風漏水） | 保險理賠 | 有火險/地震險理賠 |
| 第三人（樓上漏水） | 樓上屋主 | 協調或訴訟 |

### 3. 噪音糾紛

| 類型 | 對策 |
|------|------|
| 鄰居反映房客太吵 | 先溝通→管委會協調→報警 |
| 房客反映鄰居太吵 | 協助溝通→建議報警 |
| 樓板隔音問題 | 管委會協調鋪設隔音墊 |

### 4. 押金返還爭議

| 狀況 | 處理原則 |
|------|----------|
| 正常返還 | 30日內全額退還（匯款+拍照存證） |
| 有欠租 | 扣抵欠租+違約金後退還（附明細） |
| 有損壞 | 檢附修繕單據，扣抵合理金額 |
| 正常耗損 | 不可扣抵（油漆褪色、壁紙黃化等） |

### 5. 違規事項

| 違規類型 | 處理方式 |
|----------|----------|
| 室內抽菸（禁菸約定） | 口頭勸導→書面警告→終止租約 |
| 飼養寵物（禁寵約定） | 同上 |
| 轉租（未經同意） | 依民法第443條終止租約 |
| 非法使用（吸毒/賭博） | 立即終止租約+報警 |

### 6. 房客自殺/凶宅

**定義：** 房屋內發生非自然死亡事件

**處理流程：**
1. 保持現場，報警處理
2. 通知家屬
3. 保險理賠（若有投保兇宅險）
4. 房屋淨化/法事（民俗處理）
5. 房屋價值減損求償（向繼承人請求）

## 法律工具使用時機

| 工具 | 時機 | 時間 | 費用 |
|------|------|:----:|:----:|
| 存證信函 | 正式催告、終止租約 | 1天 | 約100元 |
| 調解委員會 | 家事類糾紛第一站 | 1-2個月 | 免費 |
| 支付命令 | 對方不抗辯的事實明確案件 | 1-2個月 | 約500元 |
| 簡易訴訟 | 標的<50萬 | 3-6個月 | 約3,000元起 |
| 通常訴訟 | 標的>50萬 | 6個月-1年 | 約5,000元起 |
| 強制執行 | 判決確定後 | 1-3個月 | 執行費 |

## 糾紛預防（最好就是不發生）

| 預防措施 | 效果 |
|----------|------|
| 嚴格審核房客 | 從源頭減少問題 |
| 明確的書面合約 | 減少各說各話 |
| 定期檢查 | 及早發現問題 |
| 鄰居關係 | 多一個資訊來源 |
| 設備定期保養 | 減少緊急維修 |
| 保險規劃 | 降低意外損失 |
