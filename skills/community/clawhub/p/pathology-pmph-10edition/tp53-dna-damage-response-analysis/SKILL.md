# TP53基因在DNA损伤应答中的多重功能分析

## 何时使用
- 用户提供关于电离辐射、化学诱变等DNA损伤场景的信息；
- 需要判断细胞是否能正常启动G1期停滞、修复、衰老或凋亡程序；
- 分析肿瘤样本中TP53突变对治疗反应或预后的影响；
- 解释为何某些细胞在DNA损伤后仍持续增殖。

## 执行步骤
1. **确认前提条件**：
   - 存在DNA损伤（如电离辐射、化学诱变剂暴露）；
   - TP53至少一个等位基因功能正常（非完全缺失或热点失活突变）。

2. **评估p53状态**：
   - 若为野生型 → 进入下游通路激活；
   - 若为热点突变（如Arg175、Arg248、Arg249、Arg273）、缺失、被病毒蛋白结合或MDM2过度降解 → 判定为功能失活，跳至第5步。

3. **激活p53下游靶基因**（仅当p53功能正常）：
   - **CDKN1A（p21）**：抑制CDK活性，诱导G1期停滞；
   - **GADD45**：参与核苷酸切除修复；
   - **BAX、PUMA**：启动线粒体凋亡通路。

4. **根据DNA损伤程度决定细胞命运**：
   - **轻微/可修复损伤** → 维持G1停滞并促进修复；
   - **严重/不可逆损伤** → 诱导细胞衰老或凋亡，防止突变传递。

5. **若TP53失活**：
   - 细胞无法停滞于G1期；
   - 损伤DNA继续复制；
   - 提示高肿瘤转化风险，并建议进一步分子诊断（如TP53测序、p53蛋白表达检测）。

## 注意事项
- 不适用于TP53完全缺失或已知热点突变失活的细胞系或临床样本；
- 病毒相关肿瘤（如HPV阳性宫颈癌）需特别考虑E6介导的p53降解；
- MDM2扩增可导致功能性p53缺失，即使基因序列正常。
