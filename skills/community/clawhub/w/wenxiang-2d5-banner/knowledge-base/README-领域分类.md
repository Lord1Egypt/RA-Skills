## 📋 文件命名规范

### 豆包会话记录
```
格式：doubao-history-YYYY-MM-DD.md
位置：对应领域目录/
示例：供应链管理/doubao-history-2026-03-05.md
```

### 专家评点
```
格式：[主题]-专家评点.html
位置：对应领域目录/
示例：供应链管理/bom-物料管理 - 专家评点.html
```

### 知识架构图
```
格式：[主题]-知识架构.html
位置：对应领域目录/
示例：供应链管理/bom-知识架构.html
```

### 模板文件
```
格式：[模板名称]-template.md/html
位置：_模板/
示例：_模板/专家评点框架-template.html
```

## 🔄 知识入库流程

### 标准流程（豆包会话后）

```
1. 导出会话 → 对应领域/doubao-history-日期.md
2. 专家评点 → 对应领域/[主题]-专家评点.html
3. 架构图 → 对应领域/[主题]-知识架构.html
4. 更新索引 → 运行 update-knowledge-index.ps1
5. 记录记忆 → memory/日期.md
```

### 快速流程（普通知识）

```
1. 整理内容 → 对应领域/[主题].md
2. 更新索引 → 运行 update-knowledge-index.ps1
3. 记录记忆 → memory/日期.md
```

## 💡 使用建议

### 查找知识
