## How to Display Certifications Correctly

### General Rules
1. **Only display certifications you currently hold** — expired certifications must be removed
2. **Follow the certification body's brand guidelines** — each has specific rules for logo size, placement, and color
3. **Don't modify certification logos** — no resizing disproportionately, color changes, or adding text
4. **Include your license/certificate number** where required
5. **Apply only to certified products** — don't display a certification across your entire line if only some products are certified
6. **Keep digital assets updated** — use the latest version of certification logos

### Common Display Errors
- Using the logo without a valid license (even if the product is certified, you need the logo license)
- Applying a product-level cert (like GOTS) to non-certified products on the same page
- Displaying a cert on the brand's About page implying all products are certified
- Using outdated logo versions
- Modifying the logo (adding background, changing colors, skewing proportions)

### Recommended Placement
| Channel | Placement | Notes |
|---|---|---|
| Product page | Near product details, below price | Only on certified products |
| Packaging | Back panel or bottom | Per certification guidelines |
| Sustainability page | Dedicated certifications section | All certs with explanations |
| Marketing email | Footer or dedicated section | With link to certification details |
| Social media | Within content, not in bio/header (unless all products certified) | Reference in caption |
