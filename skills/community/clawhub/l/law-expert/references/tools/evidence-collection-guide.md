# 证据收集与保全指南

## 用途
按证据类型分类，提供面向诉讼/仲裁场景的证据收集、获取与保全操作指引。

## 使用场景
- 已发生或可预见的诉讼/仲裁案件准备
- 劳动纠纷中的证据固定
- 商业违约/侵权事件发生后的第一时间取证
- 日常经营中的证据意识培养与合规留痕

## 核心内容

### 证据类型操作指南

| 证据类型 | 什么场景需要 | 怎么获取 | 怎么保全 | 常见错误 |
|---------|------------|---------|---------|---------|
| **书证** | 合同纠纷、债权债务、股权争议 | 合同原件/发票/收据/对账单/往来函件/内部审批单 | 原件妥善保管；复印件需与原件核对无误后公证；关键页拍照存云盘 | 只保留复印件丢失原件；文件上随意批注破坏证据效力；未保存完整的合同附件 |
| **物证** | 产品质量纠纷、交付争议、侵权 | 保存争议产品实物/样品/包装；固定仓储/物流状态 | 封存拍照并注明时间地点；委托公证处现场封存；避免改变原始状态 | 擅自检测/破坏物证原状；封存不规范导致证据链断裂；未及时固定导致物证灭失 |
| **电子数据** | 邮件争议、微信/钉钉沟通、网页侵权、源代码纠纷 | 邮件导出原始.eml格式；微信聊天记录完整截屏+录屏；网页公证/哈希存证 | 使用可信时间戳（TSA）/区块链存证平台；公证处网页保全；保持原始设备不删除 | 仅截图不保存原始文件；清空聊天记录/更换手机导致数据丢失；录屏不完整缺少上下文 |
| **证人证言** | 事实无书面凭证的劳动/人身损害/合同口头变更纠纷 | 找到直接目击者/参与人；排除利害关系风险（在职员工慎用） | 及时做询问笔录并签字；考虑公证证人证言；诉讼中申请证人出庭 | 证人陈述含主观判断（应只陈述事实）；证人身份不适格（需具备作证能力）；未考虑证人可能翻供 |
| **视听资料** | 录音录像、监控、会议记录 | 合法录音（不得侵犯他人隐私/窃听）；调取公共场所监控；导出会议录像 | 保存原始文件不剪辑；附文字稿进行公证；说明录制时间/地点/参与人/设备 | 私自窃听取得（非法证据排除）；剪辑后提交（真实性存疑）；录音中未表明身份导致关联性不足 |
| **鉴定意见** | 笔迹/印章真伪、工程造价、产品质量、医疗损害、知识产权技术比对 | 委托有资质的第三方鉴定机构；法院指定的司法鉴定 | 保存送检样本来源记录；获取鉴定机构资质证明；对鉴定意见有异议可申请重新鉴定 | 单方委托鉴定（对方可能不认可）；送检样本来源不清；超鉴定范围委托 |

### 证据三性检查清单

| 属性 | 检查要点 | 不合格后果 |
|------|---------|-----------|
| **真实性** | 原件/原始载体/未经篡改 | 不被采信 |
| **合法性** | 取得方式不侵犯他人权益/不违反禁止性规定 | 非法证据排除 |
| **关联性** | 与待证事实有逻辑联系 | 无证明力 |

### 紧急场景取证时间线

```
事发 → 0-24小时：固定现场/截图/录屏/通知相关方
     → 24-72小时：委托公证/申请诉前证据保全
     → 3-7天：整理证据清单/缺失证据补充
     → 7-15天：律师审核证据链/决定是否起诉
```

## 使用说明

1. **时间第一**：证据存在"易失性"，事发后第一时间取证，不要依赖"回头再找"
2. **原件为王**：任何证据优先保存原件/原始载体，复印/截图只是辅助
3. **公证赋能**：关键电子证据第一时间公证或时间戳存证，费用远低于证据不被采信的代价
4. **链条思维**：单一证据可能不够，需要多个证据形成完整的时间线和逻辑链
5. **合法底线**：不要通过窃听、偷拍私密空间、侵犯他人隐私等方式取证——会被排除
6. **专业介入**：重大案件证据收集应咨询律师，特别是鉴定意见和证据保全程序
