# 智能门店选址引擎 (Smart Site Selection)

专为实体商业打造的 AI 选址洞察专家。通过动态商业战略追问与高德 LBS 数据引擎，全自动生成包含「竞品雷达」、「客流推演」与「多维横评矩阵」的交互式可视化研报，让开店决策更科学。

## ✨ 核心特性

- 🏆 **Top 3 空间横评**：一键并行抓取 3 个最优候选商圈，生成核心指标横向对比矩阵。
- 📡 **微观业态雷达**：穿透干瘪数字，直接枚举商圈周边 Top 10 核心商务引擎与真实竞品名称。
- 🌊 **AI 客流推演算法**：根据周边业态密度（办公 vs 交通），智能推演 24 小时潮汐客流折线图。
- 🛡️ **工业级防并发架构**：底层 Python 引擎自带 QPS 限流保护与 Fallback 关键词降级算法，彻底消灭大模型编造数据的幻觉。
- 📊 **咨询级研报排版**：无缝集成 ECharts 极速 CDN 与高德超清 StaticMap，输出单文件、可交互的高级 HTML 研报。

## 🚀 快速开始

### 1. 环境准备

配置高德 Web Service API Key，并安装必要的网络请求库：

```bash
# 设置高德 API Key
export AMAP_WEBSERVICE_KEY=你的_高德_KEY

# 安装底层数据猎取依赖
pip install requests
```

*获取 Key 请访问 [高德开放平台](https://console.amap.com) → 应用管理 → 创建应用 → 获取 Web Service API Key。*

### 2. 交互示例

直接向 Agent 提出开店意向即可触发：

> **用户**：我想在上海静安区开一家高端普拉提馆。
> **AI**：为了生成精准报告，请补充：① 核心客群定位？ ② 门店经营战略？
> **用户**：主要针对高净值贵妇，主打私教定制。
> *(等待片刻，系统自动调用高德接口并进行深度测算...)*
> **AI**：✅ 报告已生成！建议首选【南京西路商圈】...
