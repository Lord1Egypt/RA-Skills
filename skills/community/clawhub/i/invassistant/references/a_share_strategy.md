# A 股策略（v2.1 重制版）

## 三条件建仓系统

三条件是**交叉验证过滤器**，全部满足才允许建仓：

| 条件 | 指标 | 标准 |
|------|------|------|
| ① 引擎评分 | 选股引擎核心池评分 | ≥80 分 + 连续 3 日在榜 |
| ② 价格到位 | 当前价 ≤ 动态目标价（含地板） | `max(静态×0.85, min(静态, MA20×0.95))` |
| ③ MA20 企稳 | 20 日均线走平或拐头 | MA20 delta ≥ -0.05 |

## 弹性窗口（v1.4 继承）

- 2/3 条件满足 + 第三个偏差 ≤10% → 半仓试探
- 不强行追求 25-35% A 股比例（v2.1 明确：流动性约束下不强行加仓）

## 时间止损 + 引擎有效性

- 精选层最长观察期 6 个月，到期强制评估；逻辑成立可延 3 个月
- 降级标的 3 个月冷却期
- 季度看引擎 top10 相对沪深 300 超额收益，连续 2 季跑输 >5% → 启动 review

## A 股组合结构（理想态）

| 层级 | 占比建议 | 说明 |
|------|----------|------|
| 底仓 ETF | ~40% | 宽基 + 红利 |
| 精选观察 | ~30% 预算 | 三条件验证后建仓 |
| 存量持仓 | 视情况 | 历史持仓的退出/减仓管理 |
| 机动资金 | ~5% | 特大错杀机会专用 |
| 永久现金 | ~20% | 不动 |

## v1.4 六条规则（继承）

1. **弹性窗口**：2/3 条件满足+第三个偏差≤10% → 半仓试探
2. **动态目标价**：`min(静态, MA20×0.95)` 每周更新
3. **韧性降级**：VIX>25+A股不跟跌 → 连续2日确认后降防守
4. **动量双轨**：78-80 分区间叠加 20 日动量判断
5. **财报季规则**：发布前 3 日冻结+四档决策树
6. **周五风险清单**：地缘/政策/外围/财报/仓位暴露度

## 调仓纪律

**触发调仓**：
- 连续 2 周跌出核心池（<75 分） → 研究替换
- 新风险标记 → 查财报确认
- 季报低于预期 → 按财报决策树执行
- 单只止损 -8% → 强制

**不触发**：排名小波动 / 新标的进池只观察 / 分数波动 1-2 分
