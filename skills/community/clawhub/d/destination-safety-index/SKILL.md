# 目的地安全指数

查询旅行目的地安全评级，5维度评分（犯罪安全/恐怖袭击/自然灾害/健康卫生/交通安全），含风险提示和安全建议，帮你评估目的地是否安全。

## 能做什么

- **安全检查**：输入目的地，给出综合安全评分和5维度分项评分
- **风险提示**：列出目的地的主要安全风险，如恐怖袭击、犯罪、自然灾害等
- **安全建议**：提供实用的安全出行建议和当地紧急电话
- **排名查询**：查看全球/区域安全指数排名
- **对比功能**：同时对比多个目的地的安全指数，帮你在犹豫时做决策

## 不能做什么

- 不能替代外交部/使领馆的官方旅行提醒，出行前务必查阅
- 安全指数基于公开数据综合评估，不代表实时安全状况
- 不能预测突发事件（恐袭/政变等）
- 部分小众目的地可能未收录

## 使用示例

1. "泰国旅游安全吗"
2. "东南亚哪个国家最安全"
3. "法国和意大利哪个更安全"
4. "去印度旅游要注意什么"
5. "欧洲安全指数排名"

## 注意事项

- 安全指数基于Global Peace Index、US DOS、FCDO、Numbeo等公开数据综合评估
- 局势可能快速变化，出行前请查阅最新外交部旅行提醒
- 评分1-10分，10分最安全；🟢安全/🟡较安全/🟠注意安全/🔴不建议前往
- **数据流向**：所有数据为本地内置，不发送任何外部请求，不收集用户数据

## 使用提示

- 🟢极安全(8.5+)：日本、新加坡、瑞士、新西兰，正常旅行即可
- 🟡较安全(6-8.5)：泰国、法国、意大利，注意防盗和特定区域
- 🟠注意安全(5-6)：菲律宾、印度、土耳其，需提前了解风险区
- 女性旅行者特别关注印度/埃及等国家的安全建议
- 紧急电话务必提前保存，不同国家号码不同
- 东南亚国家交通安全普遍偏低（5分以下），租摩托车需格外小心
