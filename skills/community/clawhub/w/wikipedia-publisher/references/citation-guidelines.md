# Citation Guidelines

Prefer citations with:
- clear title
- publication/work name
- date
- author when available
- archive URL for brittle pages

Use primary sources only for routine facts.
Use independent secondary coverage for notability and evaluative claims.

## Weak citation patterns

- subject website used to prove importance
- press release wires for significance claims
- directories/databases cited as narrative sources
- bare URLs with no title/work/date

## Better default

A shorter article with 5 clean independent sources beats a longer article padded with weak references.
