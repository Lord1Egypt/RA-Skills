# Amazon品牌注册完整 Checklist

本Checklist提供从商标申请到Brand Registry权益激活的完整步骤指引。请根据实际情况选择适用的阶段和任务。

---

## 第一阶段：商标申请准备

### 1.1 品牌命名与检索

- [ ] 确定品牌名称（建议2-3音节，简短易记）
- [ ] 在目标市场商标局进行名称检索，确认无近似商标
  - 美国：https://tmsearch.uspto.gov/
  - 欧盟：https://euipo.europa.eu/eSearch/
  - 英国：https://trademarks.ipo.gov.uk/
- [ ] 检查域名和社媒账号的可注册性
- [ ] 确认名称在目标类别可注册（避免过于通用的词汇）

### 1.2 商标类别规划

- [ ] 明确主营产品对应的尼斯分类（Nice Classification）
  - 3类：化妆品、洗漱用品
  - 9类：电子产品、科学仪器
  - 21类：厨房器具、家用器皿
  - 35类：广告、商业经营（服务类）
- [ ] 评估是否需要跨类别保护
- [ ] 考虑防御性注册的类别（防止他人在相关类别抢注）

### 1.3 申请材料准备

- [ ] 收集商标持有人信息（个人身份证或公司营业执照）
- [ ] 准备品牌Logo高清图片（建议PNG透明背景，不少于1000x1000像素）
- [ ] 确定商标申请的商品/服务描述清单
- [ ] 准备使用证据（如已有销售记录）或意向使用声明

---

## 第二阶段：商标申请提交

### 2.1 美国 USPTO 申请

- [ ] 选择申请类型：TEAS Plus / TEAS Standard / TEAS RF
- [ ] 创建USPTO账户（https://uspto.gov）
- [ ] 填写商标申请表格
  - 商标样式（标准文字 / 设计 / 文字+设计）
  - 真实使用声明或意向使用声明
  - 商品/服务描述
  - 申请人信息
- [ ] 上传Logo图片
- [ ] 支付申请费用（按类别计费）
- [ ] 保存申请确认函和序列号

### 2.2 欧盟 EUIPO 申请

- [ ] 创建EUIPO账户（https://euipo.europa.eu）
- [ ] 进行相似商标检索
- [ ] 填写申请表格
  - 商标类型选择
  - 商品/服务描述（使用EUIPO数据库的标准描述）
- [ ] 上传Logo图片
- [ ] 支付申请费用
- [ ] 确认申请回执

### 2.3 英国 UKIPO 申请

- [ ] 创建UKIPO账户（https://gov.uk/topic Intellectual-property）
- [ ] 进行商标检索确认可注册性
- [ ] 填写在线申请表
- [ ] 选择商品/服务类别
- [ ] 支付申请费用
- [ ] 保存申请确认

---

## 第三阶段：商标审核与维护

### 3.1 审核阶段跟进

- [ ] 定期登录商标局账户查看审核状态
- [ ] 如收到审查意见（Office Action），在规定时限内回复
- [ ] 如需要补充使用证据，准备真实的产品图、销售记录等
- [ ] 监控公告期是否有他人提出异议

### 3.2 商标注册成功后续

- [ ] 下载并保存商标注册证书
- [ ] 在亚马逊商标局官网验证入口保存商标链接
- [ ] 设置商标续展提醒（商标有效期10年，需定期续展）

### 3.3 商标维护任务

- [ ] 建立商标使用证据档案（产品图、包装图、销售记录）
- [ ] 定期检索近似商标，发现抢注及时异议
- [ ] 记录商标许可使用情况（如有授权第三方使用）

---

## 第四阶段：Amazon Brand Registry 申请

### 4.1 申请前准备

- [ ] 确认商标已在对应市场完成注册
- [ ] 准备商标注册号/申请号
- [ ] 准备高清品牌Logo文件
- [ ] 确认拥有亚马逊卖家账户或Vendor账户
- [ ] 准备支持性文件（商标注册证书电子版）

### 4.2 Brand Registry 后台操作

- [ ] 访问 Brand Registry 后台：brandregistry.amazon.com
- [ ] 登录亚马逊卖家/供应商账户
- [ ] 点击"注册新品牌"或"Register a New Brand"
- [ ] 选择卖家或供应商身份
- [ ] 填写品牌信息：
  - 品牌名称
  - 商标注册号
  - 商标局选择
  - 商标类别
- [ ] 上传品牌Logo
- [ ] 选择品牌关联的产品分类
- [ ] 添加其他品牌权益持有者信息（如适用）
- [ ] 提交申请

### 4.3 申请后跟进

- [ ] 等待审核确认邮件（通常1-3个工作日）
- [ ] 如收到补充材料请求，及时提供
- [ ] 审核通过后，确认品牌权限已正确分配
- [ ] 记录Brand Registry案例编号

---

## 第五阶段：品牌权益激活

### 5.1 Transparency 透明计划

- [ ] 登录Brand Registry后台，进入Transparency
- [ ] 阅读计划要求和服务条款
- [ ] 申请加入计划
- [ ] 审核通过后获取Transparency代码
- [ ] 为产品申请Transparency标签
- [ ] 在Production Partner Network注册（如需要）

### 5.2 A+ Content 激活

- [ ] 登录卖家后台，进入A+ Content管理页面
- [ ] 确认品牌资质已激活
- [ ] 开始创建A+ Content
- [ ] 提交内容等待审核
- [ ] 审核通过后关联到相关ASIN

### 5.3 品牌旗舰店设置

- [ ] 登录Brand Registry或卖家后台的品牌旗舰店入口
- [ ] 开始创建旗舰店
- [ ] 设计首页布局（Hero Banner + 导航）
- [ ] 创建产品分类页面
- [ ] 添加品牌故事模块
- [ ] 发布旗舰店并进行预览检查

### 5.4 Vine 计划注册

- [ ] 在卖家后台找到Vine计划入口
- [ ] 选择要注册的新品ASIN
- [ ] 支付Vine计划费用（通常$200/ASIN）
- [ ] 确认注册成功
- [ ] 等待亚马逊分配评审员
- [ ] 监控评价收到情况

### 5.5 品牌广告权限

- [ ] 在广告平台确认Sponsored Brands权限已开通
- [ ] 确认Brand Video广告权限
- [ ] 设置品牌广告活动的第一步测试

---

## 第六阶段：品牌保护配置

### 6.1 举报违规权限设置

- [ ] 在Brand Registry后台熟悉举报违规工具
- [ ] 准备侵权举报的模板和证据材料
- [ ] 建立举报违规的工作流程

### 6.2 Project Zero 申请（如符合条件）

- [ ] 确认符合Project Zero加入条件
- [ ] 在Brand Registry后台申请加入Project Zero
- [ ] 审核通过后配置自助移除权限

### 6.3 跟卖防护准备

- [ ] 了解跟卖的识别方法和监测方式
- [ ] 准备Test Buy操作流程
- [ ] 制定跟卖应急响应预案
- [ ] 考虑加入Transparency计划的长期防护

---

- [ ] 已完成任务打勾标记

- [ ] 完成后保存此Checklist作为品牌档案
