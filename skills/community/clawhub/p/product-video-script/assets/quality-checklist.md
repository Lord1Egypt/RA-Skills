# Script Quality Checklist

Run before delivering any script. If an item fails, fix it before sending.

## Hook (0–2s)
- [ ] Tension, payoff, or curiosity is promised within the first second.
- [ ] Hook works as a spoken line AND as on-screen text.
- [ ] No brand intro, logo, or "hi guys" opener.
- [ ] Hook is tied to a real audience pain or desire, not generic hype.
- [ ] Three hook variants provided for A/B testing.

## Framework & structure
- [ ] Framework was chosen from product type + awareness stage (not by default).
- [ ] The body delivers on the promise the hook made.
- [ ] A new visual or idea appears every 2–3 seconds.
- [ ] At least one open loop or pattern interrupt holds retention.
- [ ] Beat timing matches the target length (see platform playbook).

## Value & proof
- [ ] Each selling point is framed as a buyer benefit, not a spec.
- [ ] Every claim has a matching proof visual (demo, result, number, before/after).
- [ ] The single most impressive capability is the centerpiece, not buried.
- [ ] No unsubstantiated superlatives ("#1", "best") without evidence.

## CTA
- [ ] Exactly one primary action.
- [ ] CTA is matched to the platform's buy path (cart / link sticker / bio / description).
- [ ] Any urgency or scarcity is real and honest.
- [ ] CTA appears both spoken and on screen.

## Platform fit
- [ ] Length is within the platform's converting range.
- [ ] Pacing and sell intensity match the platform (TikTok hard, Reels save-led, Shorts education-led).
- [ ] CTA path is the one this platform rewards.
- [ ] Caption and hashtags suit the platform (count + mix).

## On-screen text & accessibility
- [ ] Text reinforces rather than duplicates the audio.
- [ ] Text stays on screen at least 2 seconds and is readable.
- [ ] Script is fully captioned for sound-off viewing.

## Compliance & brand
- [ ] Claims needing substantiation are flagged (health, earnings, "results may vary").
- [ ] No fabricated reviews, fake countdowns, or misleading before/after.
- [ ] Tone matches the brand voice.
- [ ] Output delivered in the shot-by-shot template with header, table, caption, and production notes.
