# 十维 — 价值研判评分卡模板

## 使用说明

每个模块分析完成后，使用以下模板进行统一评分。先逐模块下结论，最后汇总得出综合评级。

## 评分卡

| 模块 | 核心判断标准 | 合格/不合格 | 简要理由 |
|------|------------|------------|---------|
| 1. 营收+净利润 | 近5年整体正增长，增速≥行业平均 | | |
| 2. ROE | 成长股≥15%，白马股≥10%，稳定 | | |
| 3. 资产负债表 | 现金≥有息负债，商誉合理，应收≤15% | | |
| 4. 现金流量表 | 净现比≥1（优质≥1.2） | | |
| 5. 股东回报 | 连续5年以上分红，支付率稳定/提升 | | |
| 6. 赛道分析 | 行业增速≥10%，需求驱动长期存在 | | |
| 7. 同业对标 | 毛利率/净利率/ROE高于行业平均 | | |
| 8. 股权变动 | 无大股东大幅减持，有回购更优 | | |
| 9. 人均持股 | 人均持股金额≥20万为佳，<15万需警惕 | | |
| 10. 估值分析 | PB/PE处于历史低位，或双腰斩已落地 | | |

## 综合评级

| 合格数 | 评级 | 操作建议 |
|--------|------|---------|
| 8-10 | ⭐⭐⭐ 优质 | 建议重点关注，可考虑配置 |
| 6-7 | ⭐⭐ 中性 | 可观察，存在不确定性，等待更优时机 |
| 0-5 | ⭐ 不合格 | 直接排除 |

## 参考案例：爱尔眼科

| 模块 | 评判 | 合格与否 |
|------|------|---------|
| 1. 营收+净利润 | 营收20年正增长，净利润短期回落后回升 | ✅ 合格 |
| 2. ROE | 近5年稳定在15%以上 | ✅ 合格 |
| 3. 资产负债表 | 现金覆盖负债，商誉逻辑合理，合同负债增长 | ✅ 合格 |
| 4. 现金流量表 | 净现比1.57倍，利润真实 | ✅ 合格 |
| 5. 股东回报 | 连续17年分红，支付率达50% | ✅ 合格 |
| 6. 赛道分析 | 眼科赛道高增速，刚需属性 | ✅ 合格 |
| 7. 同业对标 | 三率全面领先，龙头地位稳固 | ✅ 合格 |
| 8. 股权变动 | 无减持，有回购 | ✅ 合格 |
| 9. 人均持股 | 人均持股金额12.04万，散户扎堆 | ❌ 不合格 |
| 10. 估值分析 | 双腰斩落地，估值10年低位 | ✅ 合格 |

**综合结论**：10个模块中9个合格，仅人均持股模块不足。判断为底部反转，而非短期反弹。核心风险：散户扎堆带来的短期震荡。
