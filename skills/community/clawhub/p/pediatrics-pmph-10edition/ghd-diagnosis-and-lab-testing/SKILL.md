# 生长激素缺乏症（GHD）诊断与实验室检查

## 适用对象
疑似生长激素缺乏的矮身材儿童，需满足以下全部前提条件：
- 身高低于同年龄同性别P3或均值减2SD
- 年生长速率<5 cm
- 骨龄落后≥2年
- 智力正常

> **注意**：不适用于家族性矮身材、体质性青春期延迟等非病理性矮小，且需排除其他慢性疾病。

## 执行流程

### 1. 初步筛查
- 测定血清IGF-1和IGFBP-3水平（适用于5岁至青春期前儿童），结果需按年龄、性别校正参考值。
- 若IGF-1低下但GH刺激试验正常，应考虑GH分子变异或受体缺陷，进入进一步鉴别流程。

### 2. 生长激素刺激试验
必须进行至少两种不同机制的药物试验，常用组合包括：胰岛素 + 可乐定 或 胰岛素 + 左旋多巴。

**试验准备**：
- 禁食、卧床休息
- 上午8–10时进行
- 提前30分钟置入留置针

**给药方法**：
- **胰岛素**：0.05–0.1 U/kg 静注，于0、15、30、60、90 min测血糖和GH
- **精氨酸**：0.5 g/kg 配成5%–10%溶液，30 min内静滴完，于0、30、60、90、120 min测GH
- **可乐定**：0.004 mg/kg 口服
- **左旋多巴**：10 mg/kg 口服

**结果判读**：
- GH峰值 <5 μg/L → 完全缺乏
- GH峰值 5–10 μg/L → 部分缺乏
- GH峰值 ≥10 μg/L → 排除GHD

### 3. 免做刺激试验的特殊情况（依据国外共识）
同时满足以下三点可直接确诊：
- 符合生长学标准（身高<P3或<-2SD，年增速<5 cm）
- 存在下丘脑-垂体结构异常（如垂体后叶异位、垂体发育不全、垂体柄中断）或有颅内肿瘤/放疗史
- 合并一种以上其他垂体激素缺乏（如TSH、ACTH、Gn缺乏）

### 4. 辅助检查
- **骨龄X线**：拍摄左手腕掌指骨片，确认骨龄落后≥2年
- **头颅MRI**：排查下丘脑-垂体发育异常或肿瘤
- **其他内分泌评估**：TSH/T4、GnRH刺激试验、染色体核型（尤其女性）、必要时行GH1、PROP1、GHR等基因检测

## 输出
若满足全部诊断条件，则确诊为生长激素缺乏症，应启动重组人生长激素（rhGH）治疗并安排长期随访。
