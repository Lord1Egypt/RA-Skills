## Description: <br>
Generates Chinese-language PowerPoint outlines with slide structures, page-level bullet points, design suggestions, and optional Markdown or PPTX export. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[freedompixels](https://clawhub.ai/user/freedompixels) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Employees, creators, trainers, and business users can generate Chinese presentation outlines for reports, product launches, training materials, business plans, and proposals. The skill is useful when a user needs a structured deck draft that can be reviewed, edited, and exported. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The script can write generated output to a user-provided file path. <br>
Mitigation: Review the output path before export and run the skill only in a workspace where writing generated Markdown or PPTX files is expected. <br>
