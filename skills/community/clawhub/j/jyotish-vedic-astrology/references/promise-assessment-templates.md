# 承诺判定模板（Promise Assessment Templates）

> **版本**：v1.0 | **日期**：2026-04-24
> **来源**：Astrokarak "Chart Promise vs Daśā Activation" + Astrosutras "Static Promises and Dynamic Framework"
> **核心原则**：星盘必须先承诺，大运才能交付。大运是钥匙，承诺是锁。

> **来源标签**: 【现代演绎·Skill整合】 — 承诺评估模板
---

## 使用方法

1. 确定用户询问的领域
2. 找到对应的领域模板
3. 逐层检查5项指标
4. 统计正面项数，判定承诺等级
5. 再进入激活评估（见 prediction-boundary-protocol.md）

**承诺等级判定**：
- 4-5项正面 → **承诺完整**（高概率兑现）
- 2-3项正面 → **承诺部分**（可能打折实现）
- 0-1项正面 → **承诺缺失或极弱**（Dasha无法凭空创造）

## 1. 婚姻/伴侣关系

| 层级 | 指标 | 检查内容 |
|------|------|---------|
| ①宫位 | 7宫（Sthira Bhava） | 宫内星？宫体强度？有无受克（6/8/12宫主入住）？Bhava Bala？ |
| ②宫主星 | 7宫主 | 位置？入庙/旺相/友宫/落陷？Shadbala？Ashtakavarga？与Lagna/Lagna主的关系？ |
| ③Karaka | 女=Jupiter，男=Venus | 状态？力量？受克程度？逆行？燃烧？行星战争？ |
| ④Jaimini | DK（Darakaraga）+ UL（Upapada Lagna） | DK位置和力量？UL位置和UL主星状态？UL是否Vargottama？ |
| ⑤D9分盘 | Navamsa第7宫 | D9 7宫内星？D9 7宫主？D9 Lagna与7宫关系？Venus/Jupiter在D9的状态？ |

**负面信号**：
- 7宫主落陷且无Neechabhanga
- Venus/Jupiter严重受克（6/8/12宫+凶星相位）
- D9 7宫主弱或受克

- Kala Sarpa Yoga影响7宫轴线
