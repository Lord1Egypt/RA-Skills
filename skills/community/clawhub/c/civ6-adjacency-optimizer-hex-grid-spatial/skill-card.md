## Description: <br>

## Use Case: <br>
Developers and engineers use this skill to add odd-r offset hex-grid calculations for map tooling, game-adjacent planning, and spatial queries such as neighbor lookup, adjacency checks, distance, and tiles in range. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Using a different hex coordinate convention or row origin can produce incorrect neighbor, distance, or range results. <br>
Mitigation: Confirm the project uses odd-r offset coordinates and test representative map positions before relying on the helpers for game logic. <br>
