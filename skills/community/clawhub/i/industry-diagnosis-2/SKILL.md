# 行业战略诊断系统

## 概述

基于《连锁品牌：让复制不变形的5力模型》战略力章节提炼的四维诊断模型，为想进入某个行业的人提供"能不能做+怎么做"的决策依据。

不是行业分析报告，是战略决策工具。

## 工作流程

用户输入"诊断XX行业" → 按以下5步执行：

### 第1步：数据采集（并行搜索）

用 `web_search` 并行发起以下搜索（4组，每组2条）：

```
# 市场容量
"{行业}市场规模 亿元"
"{行业}增长率 增速"

# 头部品牌
"{行业}品牌排名 TOP10 门店"
"{行业}头部品牌 市占率"

# 竞争格局
"{行业}竞争格局 CR3 集中度"
"{行业}倒闭 关店 闭店率"

# 差异化
"{行业}差异化竞争 定位"
"{行业}产品创新 趋势"
```

必要时追加搜索补充关键数据缺口。

### 第2步：数据整合与校验

- 同一数据至少2个来源佐证（多源验证）
- 优先近6个月的数据（时效性）
- 无法验证的数据标注"待验证"（存疑标注）
- **数据口径分离**：区分"核心战力"（重管控加盟/直营）与"生态网络"（轻合作网点），不可混用

### 第3步：四维评分

按以下标准量化评分：

**容量引力**：>1,000亿=90分 / 100-1,000亿=60-80分 / <100亿=30-50分

**趋势动力**：增速>15%=90分 / 5-15%=60-75分 / 0-5%=30-45分 / <0%=10-20分

**竞争友好度**：CR3<20%=70-90分 / 20-40%=50-70分 / 40-60%=30-50分 />60%=20-30分

**差异化引力**：三层都有突破口=80-90分 / 1-2层=60-70分 / 仅定位=40-50分 / 三层占满=10-30分

**综合评分** = 容量×0.25 + 趋势×0.25 + 竞争×0.25 + 差异化×0.25

### 第4步：生成MD诊断报告

按8大模块输出，写入文件 `{行业}_诊断报告_{日期}.md`。

### 第5步：报告质量自检

生成后必须通过以下检查：

| 检查项 | 标准 |
|--------|------|
| 结论前置 | 每个模块开头必须有"结论先行"段落 |
| 去icon | 全文零emoji/icon，用纯文字标注 |
| 数据口径 | 竞争格局表格必须标注"核心战力"vs"生态网络" |
| 闭环校验 | 每条策略必须显式引用前文诊断数据 |
| 战略手记 | 风险警示章节必须包含"操盘手战略手记" |

## 报告模板

按以下结构生成，每个模块**先写结论，再写分析**：

```markdown
# {行业} 行业诊断报告

> 生成时间: {日期}
> 基于《五力模型》战略力框架分析

---

## 一、市场容量分析

**结论：{1-2句核心结论+关键数字}**

- 全国总规模、年增长率、赛道等级
- 细分市场构成（表格）
- 容量评分：XX / 100

---

## 二、行业趋势分析

**结论：{1-2句核心结论}**

- 趋势四指标（表格：指标/数值/判断）
- 趋势交叉信号
- 关键数据点（3-4条）
- 趋势评分：XX / 100

---

## 三、竞争格局分析

**结论：{1-2句核心结论}**

- 头部品牌排名（表格：排名/品牌/类型/规模/数据口径）
- 竞争集中度CR3（核心战力口径）
- 生存现实（关键数据3-5条）
- 数据口径说明（如有）
- 竞争友好度：XX / 100

---

## 四、差异化机会分析

**结论：{1-2句核心结论+突破口}**

- 差异化三层次（表格：层次/机会/护城河深度）
- 差异化三道检验（表格）
- 具体突破口（2-3条）
- 差异化评分：XX / 100

---

## 五、综合评估

**结论：{综合评分+风险等级+核心判断}**

- 四维评分（文字柱状图）
- 风险评级表

---

## 六、切入策略建议

**结论：{推荐策略+核心逻辑}**

- 策略一：切弱点/找空白/做擅长（表格：维度/分析）
- 策略二（如有）：同上
- 扩张节奏：慢→稳→快

---

## 七、风险警示

**结论：{闭合规则核心表述}**

- 扩张开关闭合红线（表格：指标/合格线/警戒线/状态）
- 闭合规则
- 操盘手战略手记（固定模板）
- 关键风险（2-4条）

---

## 八、发展机遇

**结论：{最紧迫窗口期+最大空间方向}**

- 机遇清单（表格：机遇/窗口期/潜力）
```

## 核心规则

### 1. 结论前置

每个模块开头必须是"结论先行"段落，用1-2句话给出核心结论+关键数字。分析过程放在结论之后。读者只看每段第一句话就能抓住全篇核心。

### 2. 去icon

报告正文不使用任何emoji、icon符号。风险等级用纯文字（"低风险"/"中等风险"/"较高风险"/"高风险"），状态用文字（"是"/"否"/"待验证"）。

### 3. 数据口径分离

竞争格局中区分"核心战力"与"生态网络"：
- **核心战力**：重管控加盟/直营店 → 战略对标的真实战力
- **生态网络**：轻合作/安装网点 → 品牌影响力但不等于管控力

CR3计算优先使用核心战力口径。品牌数据需双标注，如："3,000+加盟店/万级合作网络"。

### 4. 诊断-策略闭环校验

策略建议必须显式引用前文的具体诊断发现，形成"信号→策略→执行"完整链条。每条策略至少引用一个前文数据点。诊断中的"危"必须转化为策略中的"机"。

### 5. 操盘手战略手记

风险警示章节闭合规则之后，必须包含以下固定模板：

> **操盘手战略手记**：很多企业不是死在没有战略，而是死在战略盲目。四项红线指标只要有一项跌破警戒线，必须无条件暂停扩张。战略力的最高境界，是把"踩刹车"的权力交给客观数据，而不是老板的欲望。

### 6. 趋势交叉信号判断

| 信号组合 | 含义 | 建议 |
|---------|------|------|
| 增速高+开店多+资本进 | 黄金赛道 | 快速进入 |
| 增速降+开店多+资本进 | 红海前夜 | 谨慎进入，打差异化 |
| 增速降+关店多+资本退 | 行业出清 | 不进入，或等出清结束后进入 |
| 增速稳+开店稳+资本观望 | 成熟期 | 只做差异化细分 |

### 7. 定位三策匹配

| 综合评分 | 推荐策略 |
|---------|---------|
| >=80 | 找空白 |
| 60-79 | 切弱点 |
| 40-59 | 做擅长 |
| <40 | 不进入 |
