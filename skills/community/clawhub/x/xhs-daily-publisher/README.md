# xhs-auto-publisher

**小红书自动化内容生产与发布助手** · User-level Skill for WorkBuddy

## 一句话介绍

对话式配置账号定位 → 每天自动完成「热点选题 + 9 图文案 + 视觉渲染 + 浏览器自动发布」，你只需扫码 1 次、每天点一下「发布」。

## 典型使用场景

- AI 资讯 / 工具速报 / 科技财经类账号起号
- 想做小红书作品集证明 AI 工程化能力
- 已有日更计划但每天手工耗时 2-3 小时，想降到 30 秒

## 调用方式

在任意对话里说以下任一句，Skill 会被自动触发：

- "帮我做一个小红书自动化账号"
- "我要用 AI 每天自动发小红书"
- "/skill xhs-auto-publisher"

## 交互流程

1. **第一轮**：Skill 问你 6 个字段（领域/昵称/受众/发布时间/视觉风格/数据源），能默认就默认
2. **第二轮**：展示配置摘要，确认后一次性打通全流程
3. **之后**：每天指定时间浏览器自动弹出发布页，你点一下「发布」
4. **随时**：说"换个风格"、"看下数据"、"改一下文案语气"即可微调

## 可选视觉风格

| 预设 | 调性 | 示意 |
|---|---|---|
| `research` | 研究笔记风（黑底白卡+宋体+红竖线+黄高亮+灰引用） | 深度/理性/高知 |
| `punchy` | 爽文资讯风（纯色大色块+黑体大字+强反差） | 快餐/流量/情绪 |
| `geek` | 极客简约风（白底+等宽字体+单色+代码块引用） | 工程师/工具党 |
| `custom` | 自定义 | 你给描述，Skill 出 |

## 输出结构（用户项目目录）

```
project/
├─ output/
│   └─ YYYY-MM-DD/
│       ├─ digest.md    # 选题+文案
│       ├─ slides.html  # 9 图源
│       └─ images/      # 9 张 2160×2880 jpg
├─ scripts/             # 从 skill 复制过来
├─ .auth/               # 登录态 + profile（gitignore）
└─ README.md
```

## 核心技术细节

- **Puppeteer + 持久化 profile**：权限永久记住、反检测参数齐全
- **HTML + CSS 渲染 9 图**：一次写模板，每天只换文字
- **WorkBuddy automation**：调度每日定时任务
- **半自动发布**：最后一步人工点发布，防封号（新号前 7 天硬规则）

## 合规要求

- 用户必须提供自己的小红书账号（不代表用户做任何不合规发布）
- cookies 和 profile 只保存在用户本机项目目录
- 不抄袭具体博主风格，仅定义"风格类别"
