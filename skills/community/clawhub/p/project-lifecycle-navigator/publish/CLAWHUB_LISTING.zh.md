# 项目生命周期导航助手

## 一句话介绍

一个面向非技术用户和 AI Coding Agent 工作流的中英双语项目导航 Skill：自动识别用户处于新项目启动、中期跑偏校准，还是代码审查升级阶段，并进入对应的结构化提问与执行计划流程。

## 适合场景

- 你想做一个新软件、AI 工具、内部系统，但不知道怎么描述需求。
- 项目做到一半感觉跑偏、功能太多、MVP 失控，需要重新校准方向。
- 已经有代码，希望审查安全漏洞、冗余代码、性能瓶颈、架构问题和测试缺口。
- 希望把需求和复盘结果整理成可以交给 AI Coding Agent 或工程师执行的计划。

## 核心能力

1. **新项目需求访谈**  
   通过分轮提问帮用户明确目标用户、核心场景、MVP 范围、数据来源、AI 能力、技术方案和开发计划。

2. **项目中期方向校准**  
   当项目偏离原始目标时，帮助用户识别目标漂移、功能膨胀、技术过度设计、MVP 失控和沉没成本，并决定继续、收缩、转向、归档或重启。

3. **代码审查与升级优化**  
   对已有代码进行安全、冗余、逻辑、性能、架构、测试、依赖和配置审查，并输出可执行、可验证、可回滚的修复计划。

## 触发示例

```text
我想做一个新项目，但我不懂技术，你先问我问题。
```

```text
项目做到一半感觉跑偏了，功能越来越多，帮我重新校准方向。
```

```text
我已经有代码了，帮我审查漏洞、冗余代码、架构问题，并生成升级优化计划。
```

## 安全说明

这是一个纯文本指令型 Skill：

- 不运行代码
- 不需要 API Key
- 不需要环境变量
- 不安装第三方包
- 不需要网络权限
- 不包含脚本执行逻辑
