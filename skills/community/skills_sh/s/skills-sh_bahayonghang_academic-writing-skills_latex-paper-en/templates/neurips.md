# NeurIPS (LaTeX)

> Venue-specific snapshot extracted from `references/venues/catalog.md`. NeurIPS and
> ICML share most expectations; this file mirrors the joint section, with the
> NeurIPS-specific numbers highlighted at the bottom. For ICML-specific
> deltas (page limit, Impact Statement), open `icml.md`. Load this file
> directly when the user names NeurIPS as the target venue, instead of
> reading the full venue catalog.

## Style

- Concise writing essential
- Mathematical notation consistent

## Format

- 9-page main paper (NeurIPS 2025/2026); references and appendix do not count
- Blind review requirements

## Special

- Mandatory paper checklist (missing = desk reject)
- Reproducibility checklist
- Code submission encouraged
- A lay summary is required only for the Position track, not main-track papers

## NeurIPS-Specific Quick Reference

> `references/venues/catalog.md` is authoritative for these numbers.

- **Page limit (NeurIPS 2025/2026)**: 9 pages (+1 camera-ready = 10)
- **Key requirement**: Mandatory paper checklist
- **Template**: `neurips_2025.sty` / `neurips_2026.sty` (named per year)
