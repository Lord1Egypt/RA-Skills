## Description: <br>
Create and troubleshoot Excel reporting workbooks with charts, dashboards, conditional formatting, print areas, PDF/image export, openpyxl or Office automation, and recurring data refresh. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kyro-ma](https://clawhub.ai/user/kyro-ma) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Analysts, finance teams, operations teams, and developers use this skill to build or repair recurring Excel report workbooks that combine data, formulas, charts, dashboards, conditional formatting, print setup, and export-ready presentation output. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad implicit invocation can cause the skill to be used for Excel report tasks when the user expected narrower manual activation. <br>
Mitigation: Review invocation settings and trigger phrases before deployment if manual activation is preferred. <br>
Risk: Workbook automation guidance can produce reports with stale formulas, chart ranges, formatting, print areas, or exports if results are not checked. <br>
Mitigation: Validate formulas, chart references, conditional formatting ranges, page setup, and exported PDFs or images before using the workbook as a presentation artifact. <br>
