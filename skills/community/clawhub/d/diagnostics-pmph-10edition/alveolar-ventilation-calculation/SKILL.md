# 肺泡通气量计算

## 何时使用
- 需要评估患者每分钟实际参与气体交换的有效通气量
- 分析潮气容积（VT）、生理无效腔（VD）和呼吸频率（RR）对通气效率的综合影响
- 解释为何浅速呼吸效率低于深缓呼吸
- 支持肺功能检查或呼吸生理评估中的定量判断

> **注意**：仅适用于静息状态；不适用于剧烈运动或严重通气/血流比例失调未校正的情况。

## 执行步骤
1. 确认已知参数：潮气容积（VT，单位 ml）、生理无效腔（VD，单位 ml）、呼吸频率（RR，单位 次/分）。
2. 使用以下任一公式计算肺泡通气量（VA）：
   - `VA = (VT - VD) × RR`
   - `VA = VT × (1 - VD/VT) × RR`
3. 将结果转换为升/分钟（L/min）。
4. 结合正常参考值（约 5.25 L/min）及 VD/VT 比值（正常 0.3～0.4）进行临床解读。
5. 若 VD/VT 显著升高（如 ≥0.6），即使 VT 和 RR 正常，也提示有效通气量下降，需警惕通气效率低下。

## 输出说明
返回单位为 L/min 的肺泡通气量数值，并附简要临床意义说明（如“低于正常，提示通气不足或呼吸模式异常”）。
