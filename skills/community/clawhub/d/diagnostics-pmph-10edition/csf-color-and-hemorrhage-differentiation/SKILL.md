# 脑脊液颜色分析与出血性质鉴别

## 第一阶段：颜色初步判读

1. **观察脑脊液颜色**
   - **无色**：正常、病毒性脑炎、轻型结核性脑膜炎等
   - **红色**：提示出血 → 进入第二阶段鉴别
   - **黄色（黄变症）**：出血、黄疸、梗阻等
   - **白色**：白细胞增多 → 化脓性脑膜炎
   - **绿色**：铜绿假单胞菌或肺炎链球菌性脑膜炎
   - **褐色**：黑色素瘤相关

> 注意：颜色仅为线索，必须结合实验室检查综合判断

## 第二阶段：红色脑脊液出血性质鉴别（仅当呈红色时执行）

1. **外观与易凝性**
   - 新鲜性：浑浊、易凝
   - 陈旧性：清晰透明、不易凝

2. **离心后上清液观察**
   - 新鲜性：上清液无色透明
   - 陈旧性：上清液呈红、黄褐或柠檬色

3. **显微镜检查红细胞形态**
   - 新鲜性：形态正常
   - 陈旧性：皱缩

4. **隐血试验**
   - 新鲜性：多阴性（红细胞未破裂）
   - 陈旧性：阳性（血红蛋白释放）

5. **白细胞计数**
   - 新鲜性：不增高
   - 陈旧性：继发性增高

6. **综合判断**
   - 符合多项陈旧性特征 → 支持病理性出血（如蛛网膜下腔出血）
   - 仅第一管红、后续变清 + 新鲜性特征 → 考虑穿刺损伤

> 输出应为“新鲜性出血”或“陈旧性出血”，用于指导临床决策。
