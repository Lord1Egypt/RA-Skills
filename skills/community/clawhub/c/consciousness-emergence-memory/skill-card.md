## Use Case: <br>
Developers and external agent users can use this skill to build, query, score, summarize, and report on local workspace memory using information-theoretic, causal, metacognitive, and optimization scripts. It is suited to experimental agent memory workflows where local persistence is acceptable and reviewed. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can persist and re-inject local workspace memory content, which may unintentionally retain sensitive or personal data. <br>
Mitigation: Use it only in workspaces where local memory persistence is acceptable, and avoid storing secrets or personal data. <br>
Risk: The skill can bulk-export memory content and generate reports from local memory files and logs. <br>
Mitigation: Review exported content before sharing it, and regularly inspect or clear SESSION-STATE.md, MEMORY.md, memory/ logs, and dot-log files. <br>
