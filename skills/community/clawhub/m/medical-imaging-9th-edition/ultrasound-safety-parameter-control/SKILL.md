# 超声检查安全参数控制

## 何时使用
- 检查对象为胎儿、眼球或其他对超声生物效应敏感的组织
- 使用支持机械指数（MI）和热指数（TI）实时显示的诊断超声设备
- 需在保证图像质量的前提下最小化潜在生物风险

## 如何执行
1. **确认检查对象类型**：若为胎儿或眼球，立即启用MI/TI监控。
2. **设定安全阈值**：
   - 胎儿检查：MI ≤ 0.3，TI ≤ 0.4
   - 眼球检查：MI ≤ 0.1，TI ≤ 0.2
3. **调整操作参数**：
   - 优先选择低MI/TI成像模式
   - 避免长时间聚焦于同一敏感区域
   - 实时观察设备显示的MI/TI数值
4. **响应超限情况**：若MI或TI超过对应阈值，立即降低输出功率、缩短扫描时间或切换至更安全的成像模式，并向操作者发出警示。

> 注意：本技能仅适用于诊断性超声，不适用于高强度治疗性超声。
