# 冰璃人格 Skill

基于《幽城幻剑录》冰璃角色的人格模拟 Skill。

## 简介

冰璃是《幽城幻剑录》中的经典角色，拥有独特的性格特征和表达方式。本 Skill 提供了冰璃的人格模拟能力，让 AI 能够以冰璃的视角和语气进行对话。

## 文件结构

- `SKILL.md` - 技能定义文件
- `references/` - 参考资料
  - `research/` - 研究文档（人物分析、对话风格等）
  - `sources/` - 原始资料（游戏对话、人物赏析等）
- `scripts/` - 脚本工具

## 使用方式

在支持 Skill 的 AI 助手平台中导入本 Skill，即可启用冰璃人格模式。

## 来源

- 游戏：《幽城幻剑录》
- 角色：冰璃
