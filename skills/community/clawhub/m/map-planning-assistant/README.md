# 全能聚合地图规划助理

基于高德Web服务API打造的全场景、全周期、具备自主学习与长记忆能力的复合型LBS地图助理。

## 功能特性

### 八大核心场景地图

1. **遛狗专属地图** 🐕
   
   - 宠物友好公园、开放式草坪、宠物饮水点、宠物厕所、宠物咖啡店、宠物医院
   - 自动规划短途散步行程、标注禁养区域

2. **跑步健身地图** 🏃
   
   - 塑胶跑道、滨河步道、环湖跑道、城市绿道、封闭健身公园
   - 区分晨跑/夜跑路线，自动规避危险路段

3. **溜娃亲子地图** 👶
   
   - 儿童乐园、免费亲子公园、博物馆、科技馆、亲子书店、动物园
   - 区分免费/收费、低龄/学龄，标注母婴室位置

4. **美食探店地图** 🍜
   
   - 本地老字号、网红餐厅、小众私房菜、特色小吃、深夜食堂
   - 按人均档位、菜系、评分排序

5. **人间烟火市井地图** 🏮
   
   - 早市、夜市、赶集集市、老旧街巷、便民菜市场、路边特色小吃
   - 主打本地原生态市井生活点位

6. **网红打卡地图** 📸
   
   - 地标建筑、文艺街区、观景台、小众拍照点位、文创园区
   - 标注最佳拍照时间和角度

7. **户外露营地图** ⛺
   
   - 免费露营地、付费营地、野餐草坪、垂钓点位
   - 标注明火允许/禁止区域、卫生间距离

8. **长途旅行全景地图** ✈️
   
   - 整合景点、美食、住宿、交通、人文点位
   - 自动拆分多日行程，适配自驾/公共交通

### 高阶核心功能

1. **长记忆系统**
   
   - 永久记忆、长期记忆、临时记忆三级储存
   - 支持关键词检索记忆
   - 记忆复用减少重复检索

2. **用户画像自动描绘**
   
   - 自动统计出行偏好、场景偏好、消费层级
   - 画像对用户透明可查看

3. **同城活动管理**
   
   - 自动检索、储存、更新、过期清理城市线下活动
   - 根据画像推送匹配活动

4. **夜间自主学习**
   
   - 每日凌晨自动复盘用户行为
   - 迭代推荐算法
   - 清理无效记忆

5. **旅行模式动态优化**
   
   - 联动实时天气、路况
   - 自动微调路线、规避风险

6. **紧急预警推送**
   
   - 暴雨、极端天气、道路封控实时提醒

7. **全域人文讲解**

   - 景点/街区/地标配套历史、民俗、人文故事

8. **个人地图生成**

   - 基于高德 `maps_schema_personal_map` API 创建个人地图
   - 批量导入所有打卡点，生成高德可识别的二维码
   - 用户扫码直达高德App查看全量打卡点
   - 支持一键导航、分享好友、保存相册
   - 超过50个点位时自动分批创建

## 配置说明

### 环境变量配置

在使用前需要配置高德地图API密钥：

```bash
export AMAP_API_KEY=你的高德API密钥
```

### 获取高德API密钥

1. 访问 [高德开放平台](https://lbs.amap.com/)
2. 注册开发者账号
3. 创建应用，选择Web服务类型
4. 获取API Key

## 使用方式

### 基本指令

```
# 全场景地图生成
帮我生成郑州市全能游玩地图
给我制作成都市全套休闲地图

# 单场景指定
帮我做杭州西湖区溜娃地图
给我找武汉适合夜跑的路线
生成西安人间烟火夜市清单

# 高阶功能
查看我的用户画像
调取历史游历城市
推送本周本地活动
开启旅行模式规划3日自驾游
讲解开封鼓楼人文历史
```

### 输出格式

1. **头部**：地图主题、所属城市、适配人群、建议游玩时长
2. **主体**：分场景分类TOP5优质点位（评分、地址、适配贴士、最佳游玩时间）
3. **附加模块**：同城近期优质活动推荐、对应点位人文讲解
4. **路线模块**：按区域拆分多日出行路线，适配自驾/步行
5. **统计模块**：点位总数、覆盖区域、适配场景
6. **个人地图模块**：
   - 高德个人地图二维码，扫码直达高德App
   - 点位数量统计
   - 地图链接（可复制分享）
   - 使用说明（导航、分享、保存）
7. **底部**：温馨提示

## 技术栈

- **运行环境**：Node.js >= 14.0.0
- **核心依赖**：高德地图Web服务API
- **数据存储**：本地JSON文件
- **通信协议**：HTTPS GET请求

## 注意事项

1. **API调用限制**
   
   - 个人开发者免费额度充足
   - 单接口请求超时设置：8秒
   - 超时重试1次

2. **数据合规**
   
   - 全程不收集隐私信息
   - 仅储存地名、路线、用户偏好标签
   - 不采集手机定位、个人身份信息

3. **POI过滤规则**
   
   - 默认剔除评分<4.0的点位
   - 剔除停车场、普通商铺、关闭停业门店
   - 周边搜索半径建议3000m

4. **路线编排规则**

   - 按区县分组，同区域点位编排单日路线
   - 跨区域拆分多日行程
   - 景点优先上午、美食/夜景优先晚上

5. **个人地图生成**

   - 使用高德 `maps_schema_personal_map` API
   - `sceneType=2`：仅创建资源点（打卡点位标记模式）
   - `points`：POI坐标集合（经纬度用逗号分隔）
   - 单次最多50个点位，超过自动分批创建
   - 生成二维码，扫码直达高德App查看全量打卡点

## 扩展方向

基于当前技术底座，可快速衍生多款高德LBS类Skill：

1. **美食探店Skill**：聚焦餐厅、网红小吃、老字号
2. **酒店住宿Skill**：检索酒店、民宿，增加价格区间
3. **户外游玩Skill**：检索公园、露营地、徒步路线
4. **同城出行路线Skill**：新增路径规划API
5. **实景打卡Skill**：结合静态地图API生成图片
