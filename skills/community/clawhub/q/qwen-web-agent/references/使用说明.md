# Qwen 自动化脚本使用说明

## 概述
`qwen_agent.py` 是一个基于 Playwright 的浏览器自动化脚本（Headed 模式），用于与通义千问网页版（qianwen.com）进行交互。脚本启动 Chromium 实例，自动输入问题、等待回复并将其保存到本地 Markdown 文件中。

## 环境要求
- **Python 3.x**
- **Playwright**: `pip install playwright`
- **Playwright 浏览器驱动**: `playwright install chromium`

## 功能特性
- **持久化登录**: 使用 `~/.qwen_session/` 目录存储浏览器上下文。**首次运行时会弹出浏览器窗口，需手动完成登录**，后续运行将自动保持登录状态。
- **流式输出支持**: 脚本会持续监控页面内容，直到检测到回复内容不再变化（即生成结束）。
- **灵活输入**: 支持命令行参数或通过管道 (stdin) 输入问题。
- **结果持久化**: 每次运行的最新回复都会覆盖写入到 `last_output.md`。

## 使用方法

### 1. 首次运行（手动登录）
由于 qianwen.com 需要阿里云账号登录或手机验证码，**首次运行时会弹出浏览器窗口，需手动完成登录**。

```bash
python qwen_agent.py "你好，请介绍一下你自己。"
```

**注意**：运行后会弹出一个 Chromium 浏览器窗口，自动导航到 `https://www.qianwen.com/`。如果需要登录，请在窗口中完成所有验证流程。一旦看到聊天界面加载完成，脚本会自动继续执行。

### 2. 命令行参数调用
直接在命令后添加问题：
```bash
python qwen_agent.py "什么是量子计算？"
```

### 3. 管道调用
通过 `echo` 或其他命令传递问题：
```bash
echo "写一首关于秋天的诗" | python qwen_agent.py
```

### 4. 设置超时时间
如果问题非常复杂，可能需要更长的等待时间，可以使用 `--timeout` 参数（单位：秒）：
```bash
python qwen_agent.py "请分析一下当前的全球经济趋势" --timeout 300
```

## 输出文件
- **结果文件**: `/home/derek/文档/Derek_Obsidian_DB/AI应用对接/Qwen模块/last_output.md`
- **文件格式**: 包含 YAML Frontmatter（查询内容、时间戳、来源 `qianwen.com`）和 Markdown 正文。

## 故障排除
- **登录失败**: 如果脚本卡在等待输入框阶段，请检查浏览器窗口是否需要处理验证码或登录。
- **选择器失效**: 如果 qianwen.com 更新了 DOM 结构，可能需要修改脚本中的 `input_selectors` 或 `message_selectors` 列表。
- **权限问题**: 确保运行脚本的用户对 `~/.qwen_session/` 和输出目录有读写权限。
