# 全能地图规划助理 - 项目总结

## 项目概述

本项目是一个基于高德Web服务API打造的全场景、全周期、具备自主学习与长记忆能力的复合型LBS地图助理。整合遛狗、跑步、溜娃、美食、烟火市井、旅游出行、人文科普七大核心场景，一站式完成点位检索、路线规划、智能优化、活动推送、记忆沉淀、智能预警。

## 核心功能实现

### 1. 八大场景地图生成 ✅

| 场景 | 状态 | 功能说明 |
|------|------|----------|
| 遛狗地图 | ✅ | 宠物友好公园、饮水点、宠物医院等 |
| 跑步地图 | ✅ | 塑胶跑道、滨河步道、区分晨跑/夜跑 |
| 溜娃地图 | ✅ | 儿童乐园、博物馆、亲子书店等 |
| 美食地图 | ✅ | 老字号、网红餐厅、特色小吃等 |
| 市井地图 | ✅ | 早市、夜市、菜市场、老街巷等 |
| 打卡地图 | ✅ | 地标建筑、文艺街区、观景台等 |
| 露营地图 | ✅ | 露营地、野餐草坪、垂钓点等 |
| 旅行地图 | ✅ | 多日行程规划、自驾/公共交通适配 |

### 2. 长记忆系统 ✅

- **三级记忆存储**：永久记忆、长期记忆、临时记忆
- **记忆索引**：支持关键词检索历史记忆
- **记忆复用**：重复访问相同城市/场景时优先调取历史数据
- **自动清理**：临时记忆7天后自动过期

### 3. 用户画像系统 ✅

- **自动描绘**：从历史交互中提取用户偏好标签
- **场景偏好**：统计各场景使用频率
- **时间偏好**：分析用户活跃时间段
- **城市偏好**：记录常去城市
- **标签管理**：支持查看和重置画像

### 4. 同城活动管理 ✅

- **活动检索**：自动检索城市展会、市集、音乐节等
- **智能推荐**：根据用户画像推荐匹配活动
- **活动分类**：支持按类型、时间筛选
- **过期清理**：自动清理过期活动数据

### 5. 路线规划系统 ✅

- **单场景路线**：为每个场景规划最优路线
- **多日行程**：自动拆分多日行程安排
- **时间安排**：景点上午、美食中午、夜景晚上
- **距离计算**：自动计算总距离和预计用时

### 6. 天气路况联动 ✅

- **天气适配**：雨天替换室内点位、高温优先早晚出行
- **路况适配**：实时规避拥堵路段
- **紧急预警**：暴雨、极端天气实时提醒

### 7. 人文知识库 ✅

- **城市概览**：历史沿革、文化特色
- **景点讲解**：历史典故、游玩贴士
- **最佳拍照点**：推荐拍照位置和角度

### 8. 个人地图生成 ✅

- **批量导入**：一次性传入多个POI坐标
- **二维码生成**：生成高德个人地图二维码
- **一键导航**：扫码直达高德App查看点位

## 技术架构

### 模块结构

```
src/
├── index.js              # 主入口：意图解析、流程调度
├── utils/
│   └── amap.js           # 高德API封装：行政区划、POI检索、天气、路况
├── config/
│   └── scenes.js         # 八大场景配置：POI类型、关键词、过滤规则
├── memory/
│   └── index.js          # 记忆系统：三级存储、索引、清理
├── profile/
│   └── index.js          # 用户画像：偏好统计、标签管理
├── activity/
│   └── index.js          # 活动管理：检索、推荐、过期清理
├── route/
│   └── index.js          # 路线规划：单场景、多日行程、天气联动
├── output/
│   └── formatter.js      # 输出格式化：标准模板渲染
└── culture/
    └── index.js          # 人文知识库：城市、景点文化讲解
```

### 数据流

```
用户输入 → 意图解析 → 场景识别 → POI检索 → 数据加工 → 路线规划 → 格式化输出
                ↓
        记忆加载 ← 用户画像
                ↓
        记录交互 → 更新画像
```

### API调用

| API | 用途 | 调用场景 |
|-----|------|----------|
| 行政区划查询 | 获取城市区县列表 | 全场景地图 |
| 地理编码 | 地址转坐标 | 区域搜索 |
| POI关键字搜索 | 按关键词搜索POI | 所有场景 |
| POI周边搜索 | 按坐标范围搜索 | 指定区域 |
| POI详情 | 获取详细信息 | 数据补全 |
| 天气查询 | 获取天气预报 | 旅行模式 |
| 路况查询 | 获取实时路况 | 路线优化 |
| 个人地图创建 | 生成地图二维码 | 输出阶段 |

## 部署说明

### 环境要求

- Node.js >= 14.0.0
- 高德地图API密钥
- 网络连接

### 配置步骤

1. 获取高德API密钥
2. 设置环境变量 `AMAP_API_KEY`
3. 运行 `npm install`（可选）
4. 运行 `npm test` 验证配置

### 使用方式

```javascript
const { main } = require('./src/index');

const result = await main({
  userMessage: '帮我生成北京市全能游玩地图',
  userId: 'user_123'
});

console.log(result.content);
```
