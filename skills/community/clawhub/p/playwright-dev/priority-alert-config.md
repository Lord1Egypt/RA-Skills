# 优先级提醒 - 静默模式配置

**创建时间：** 2026-03-11 01:49  
**用途：** 配置优先级提醒的静默模式规则

---

## 🌙 **静默时间段**

**夜间静默：** 23:00 - 08:00
- 不播放语音
- 只发送文字消息

---

## 😴 **息屏检测**

**检测逻辑：**
1. 检测最后用户输入时间
2. 如果空闲 > 5 分钟 → 判定为离开/息屏
3. 跳过语音播报

**PowerShell 检测代码：**
```powershell
# 获取最后输入时间
$lastInput = Get-LastInputInfo
$idleTime = (Get-Date) - $lastInput

# 如果空闲 > 5 分钟
if ($idleTime.TotalMinutes -gt 5) {
    return $false  # 不播放语音
}
return $true  # 播放语音
```

---

## 📋 **播报规则**

| 时间段 | 息屏状态 | 播报方式 |
|--------|---------|---------|
| 08:00-23:00 | 亮屏 | TTS + 文字 |
| 08:00-23:00 | 息屏 | 只文字 |
| 23:00-08:00 | 任意 | 只文字 |
