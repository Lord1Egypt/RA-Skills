# 掼蛋助手使用指南 v2.1

## 重要更新：规则对比说明

### 为什么有规则差异？
掼蛋起源于江苏淮安，后来扩散到全国，不同地方发展出了不同的玩法变种。不同比赛的规则也不一样。

### 三大主流规则体系对比

| 规则项 | ✅ **正式比赛标准（江苏掼蛋协会）** | ⚠️ 地方变种1（苏北/安徽） | ⚠️ 地方变种2（苏南/浙江） |
|--------|-----------------------------------|-------------------------|-------------------------|
| **连对** | 最多三连对（3对连续牌） | 最多五连对 | 可出六连对 |
| **三带二** | 必须带两张（33344） | 可带一张或两张 | 必须带两张 |
| **顺子** | 最少5张，最多6张 | 最少5张，最多8张 | 最少5张，最多6张 |
| **钢板** | 两个连续的三张（333444） | 允许三钢板（333444555） | 只允许双钢板 |
| **进贡规则** | 双下两人都进贡，上游先挑 | 只有末游进贡 | 抗贡需要两张大王 |
| **红桃配** | 可配任何牌（除了王炸） | 可配任何牌（包括王炸） | 可配任何牌（除了王炸） |
| **炸弹大小** | 天王炸 > 炸数多 > 点数大 | 同左 | 8炸 > 天王炸 |

### 我们的陪练模式默认采用
**【正式比赛标准规则】**（江苏掼蛋协会）
- 连对最多三连对
- 顺子最多6张
- 三带二必须带两张
- 钢板只能双钢板
- 红桃2不能配大王小王

### 规则切换指令
| 指令 | 功能 |
|------|------|
| `规则 正式` | 切换到正式比赛标准规则（默认） |
| `规则 地方1` | 切换到苏北/安徽变种规则 |
| `规则 地方2` | 切换到苏南/浙江变种规则 |
| `规则 查看` | 查看当前使用的规则 |

## 一、实战模式（现场辅助）

### 核心功能

#### 1. 手牌识别
- **操作**：拍照/发送手牌图片
- **识别内容**：54种牌各2张 = 108张（含大小王、红桃级牌标记）
- **输出**：结构化牌型清单 + 可组牌型分析

#### 2. 记牌器
- **操作**：输入各位置出牌，如 `2 对5`、`3 顺子34567`、`4 炸弹8888`
- **追踪**：已出牌、剩余牌、关键牌位置推断
- **查询**：`记牌` 或 `还剩什么`

#### 3. 出牌建议
- **触发**：`建议` 或 `怎么打`
- **输出**：2-3个方案 + 推荐理由 + 风险提示
- **考虑因素**：剩余牌数、局面控制、炸弹时机、对家配合

### 交互格式

#### 输入出牌
```
2 对5          # 2号位出了对5
3 顺子34567   # 3号位出了顺子
4 炸弹8888    # 4号位出了炸弹
1 单张A        # 你出了单张A
```

#### 快捷指令
| 指令 | 功能 |
|------|------|
| `建议` / `怎么打` | 获取出牌建议 |
| `记牌` / `还剩什么` | 查看剩余牌追踪 |
| `摸 ♥3 ♠7` | 更新手牌（摸牌） |
| `打 方案1` | 确认出牌方案 |
| `新局` | 开始新一局 |
| `级牌 8` | 设定当前级牌 |

## 二、陪练模式（AI陪练）

### 核心特点
- **AI发牌**：随机洗牌发牌，用户只看到自己的手牌
- **AI对手**：2号位（右手对手）、3号位（对家）、4号位（左手对手）均由AI扮演
- **信息隔离**：每个AI只知道自己的牌 + 公共出牌信息，不偷看其他人的牌
- **实时反馈**：每手出牌AI都会评分和点评

### AI身份性格差异

| 位置 | 性格 | 策略特点 |
|------|------|----------|
| 2号位 | 激进型 | 喜欢抢出牌权，爱用炸弹，进攻意识强 |
| 3号位 | 配合型 | 优先支持1号位，帮挡帮送，默契配合 |
| 4号位 | 保守型 | 擅长防守记牌，后发制人，稳健经营 |

### AI难度可调

| 难度 | AI表现 |
|------|--------|
| 初级 | 判断力弱，容易犯错，帮用户建立信心 |
| 中级 | 势均力敌，正常水平 |
| 高级 | 记牌精准，配合默契，很难赢 |

### 陪练流程

1. 用户说 `开始陪练` 或 `新局 陪练`
2. AI发牌，只显示1号位（用户）的手牌
3. 按出牌顺序 1→2→3→4 循环
4. 轮到用户时，AI给出出牌建议（2-3个方案）
5. 轮到AI时，AI独立决策出牌（只能看到自己的牌+公共信息）
6. 每轮更新公共信息
7. 局结束自动进入复盘

### 陪练指令
| 指令 | 功能 |
|------|------|
| `陪练` / `开始陪练` | 开启陪练模式 |
| `新局` | 开始新一局 |
| `难度 初级/中级/高级` | 调整AI难度 |
| `出牌 [牌型]` | 用户出牌 |
| `建议` | 获取当前出牌建议 |
| `接管 X号位` | 接管AI位置 |

## 四、复盘系统（最核心的学习功能）

### 复盘目标
让用户逐步提升，最终"毕业"不再需要AI辅助

### 复盘内容

#### 1. 全局复盘
- 本局结果（谁先跑、升级情况）
- 关键转折点（第几轮、出了什么牌、局势逆转）

#### 2. 逐手评分
- 用户的每一次出牌，AI打分（1-10分）
- 10分 = 最优解，8-9分 = 合理选择，6-7分 = 可接受但不是最优，5分以下 = 失误
- 每手牌标注：你出了什么、推荐出什么、为什么推荐那个更好

#### 3. 失误分析
- 标注所有5分以下的出牌
- 分析失误原因：时机不对/牌型选择错误/没考虑对家/炸弹用早了
- 给出同类场景的正确打法口诀

#### 4. 亮点表扬
- 标注9分以上的好牌
- 说明为什么打得好

#### 5. 学习进度追踪
- 统计胜率变化趋势
- 常见失误类型分布（配合同步/炸弹时机/顺子拆分/进贡策略...）
- 薄弱环节识别 + 针对性练习建议
- "毕业指标"：连续5局胜率>60%且无重大失误

#### 6. 口诀沉淀
- 复盘中提炼实用口诀（如"逢配必留、炸弹压尾、对家先跑我断后"）
- 逐步积累用户的个人打法手册

### 复盘输出示例
```
=== 第X局复盘 ===
结果：我方[胜/负]，升级至X
关键转折：第7轮，2号位炸弹压制，我方失去出牌权

【逐手评分】
第1手：你出 ♠34567顺子 → 8分（合理开局释放小顺子）
第5手：你出 ♥AA对子 → 4分（应保留对A压制，出小对探路）❌
  → 推荐：出♣44，保留AA作为后手控制
第12手：你出 炸弹♠KKKK → 9分（关键时刻夺回出牌权）✅

【失误TOP3】
1. 第5手：对A出早了 → 口诀：大对留后手，小对先探路
2. 第8手：没帮3号位挡牌 → 口诀：对家在冲刺，我方必挡牌
3. 第15手：炸弹拆了打顺子 → 口诀：炸弹不拆牌，整手压到底

【学习进度】
累计对局：8局 | 胜率：50% ↑ | 平均得分：6.8 ↑
薄弱环节：炸弹使用时机（失误率40%）
毕业进度：■■■□□□□□□□ 30%
```

## 五、毕业机制

### 毕业指标
- 连续5局胜率 > 60%
- 无重大失误（每局失误≤1次）
- 对家配合默契度达标

### 学习阶段
| 阶段 | 目标 | 重点练习 |
|------|------|----------|
| 新手期 | 胜率30%+ | 基本规则、单张对子打法 |
| 成长期 | 胜率45%+ | 牌型组合、炸弹时机 |
| 进阶期 | 胜率55%+ | 对家配合、控局技巧 |
| 毕业期 | 胜率60%+ | 全面实战、摆脱辅助 |

### 毕业奖励
- 解锁高级技巧教学模式
- 获取专属"掼蛋达人"称号
- 生成个人打法手册PDF
