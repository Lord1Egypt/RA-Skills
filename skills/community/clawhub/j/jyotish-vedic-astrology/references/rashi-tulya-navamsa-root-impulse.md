## 第一部分：Rashi Tulya Navamsa（星座等位 Navamsa）

### 1. 概念定义

**Rashi Tulya Navamsa**（缩写 RTN）是一种将 D1（本命盘）行星位置投射到 D9（Navamsa 盘）上进行分析的技法。

**核心操作**：保持行星在 D1 中的星座位置（度数映射到该星座的 0°-30°），然后查看这个位置在 D9 中对应什么星座/行星。

> 简单说：把 D1 的行星"放"到 D9 的坐标系里看，但保留 D1 的星座位置不变。

### 2. 计算方法

1. 取行星在 D1 中的**星座内度数**（degree in sign，0°-30°）
2. 将此度数视为 D9 中同一位置（从白羊 0° 开始）
3. 确定该位置在 D9 中落在哪个 Navamsa 星座
4. 分析该 Navamsa 星座的主星对该行星的"动机激励"

**示例**：
- 金星在 D1 中 = Aries 15°
- 15° 在 Aries 中属于第 2 个 Navamsa（Taurus，由 Venus 主宰）
- RTN：金星在 D1 的位置投射到 D9 → 金星落入 Taurus Navamsa
- 解读：金星的深层动机受 Taurus（稳定、物质、感官）影响

### 3. 行星的 Navamsa 主星 = "动机激励者"

RTN 揭示的是：**行星受到其 Navamsa 主星的"动机"驱动**。Navamsa 主星给行星一个"root impulse"（根源冲动），决定该行星在深层面为什么而行动。

## 第四部分：实战应用

### 6. 经典组合解读

#### 组合1：金星在火星 Navamsa（无法满足的性欲）

- **RTN 分析**：金星在 D1 中位置投射到 D9，落入火星主宰的 Navamsa
- **深层含义**：金星（欲望）受到火星（战斗/激情）驱动 → 性欲永不满足
- **婚姻影响**：容易婚外情、多段关系、对伴侣的身体要求高
- **缓解条件**：如果火星状态好（入庙/耀升）→ 升华为体育/创造力

#### 组合2：金星在土星 Navamsa（孤独/资源匮乏）

- **RTN 分析**：金星在 D1 中位置投射到 D9，落入土星主宰的 Navamsa
- **深层含义**：金星（关系/美）受到土星（限制/延迟）驱动 → 关系中的孤独
- **性取向影响**：部分学者认为此位置指示同性关系倾向
- **正面解读**：土星也代表深度和持久 → 如果土星状态好，关系虽然来得晚但很深

#### 组合3：木星在水象 Navamsa（智慧受阻）

- **RTN 分析**：木星在 D1 位置投射到 D9，落入水象 Navamsa
- **深层含义**：智慧追求被情感干扰 → 判断力被情绪影响
- **正面解读**：如果木星本身强→成为情感导师/咨询师
