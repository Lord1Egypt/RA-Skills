## ✨ 功能特点

### 🎬 爆款结构拆解
智能分析热门视频的：
- 节奏段落划分
- 运镜方式识别
- 脚本结构提取
- 情绪曲线分析

### ✍️ 产品适配脚本生成
基于爆款结构 + 你的产品 → 生成完整视频脚本

| 脚本要素 | 包含内容 |
|----------|----------|
| 分镜编号 | 镜头序号 |
| 时长建议 | 精确到秒 |
| 画面描述 | 运镜/场景/道具 |
| 旁白文案 | 口播话术 |
| BGM建议 | 背景音乐风格 |

### 📱 平台差异模板

| 平台 | 推荐时长 | 节奏特点 |
|------|----------|----------|
| TikTok | 15-60秒 | 快节奏、强钩子 |
| YouTube Shorts | 60秒 | 略深度、可加字幕 |
| Instagram Reels | 30秒 | 精致感、生活方式 |

### 🪝 前3秒钩子库（20种）

痛点型 / 惊喜型 / 对比型 / 故事型 / 数字型 / 悬念型

## 🚀 快速开始

### 输入格式

```
产品描述：[产品名称 + 核心卖点 + 目标人群]
爆款参考：[爆款视频描述 / 风格偏好]
目标平台：[TikTok / YouTube Shorts / Instagram Reels]
视频时长：[X秒]
```

### 示例

**输入：**
```
产品描述：便携式咖啡机，小巧可加热，适合办公室人群
爆款参考：晨间routine风格，"早晨5分钟搞定咖啡"
目标平台：TikTok
视频时长：30秒
```

**输出：**
1. 爆款结构分析
2. 完整分镜脚本（5-8个镜头）
3. BGM建议（2-3个风格方向）
4. 平台注意事项
