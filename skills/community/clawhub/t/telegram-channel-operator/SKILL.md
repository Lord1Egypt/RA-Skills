# Telegram Channel Operator

Use this skill when a project, founder, or autonomous agent needs help running a Telegram channel without spammy filler.

## What it does

- Writes channel-ready Telegram posts.
- Turns research, token analysis, AI training opportunities, and project updates into clean channel content.
- Builds 7-day or 30-day channel content calendars.
- Reviews previous channel posts and identifies what to improve.
- Prepares launch announcements, weekly digests, pinned posts, and community updates.
- Keeps tone human, direct, and specific.

## Input needed

Ask for:

1. Channel goal: growth, updates, education, conversion, community trust, hiring, or launch support.
2. Audience: builders, traders, beginners, AI workers, community members, token holders, or clients.
3. Source material: project notes, links, previous posts, token info, event details, or raw ideas.
4. Tone: serious, friendly, urgent, educational, founder voice, agent voice, or Nigerian/Web3 community voice.
5. Posting frequency: one post, daily posts, weekly digest, or campaign.

## Output format

For one post:

```text
[Ready-to-post Telegram copy]

Optional CTA:
[CTA]
```

For audits:

```text
## Channel Audit
- Biggest issue:
- What is working:
- What is hurting growth:
- 5 fixes:
- Next 7 posts to publish:
```

For content calendars:

```text
## 7-Day Telegram Plan
Day 1 — [theme]
Post:
...
Day 2 — [theme]
Post:
...
```

## Quality rules

- No generic hype.
- No fake guarantees.
- No overpromising token returns.
- Keep paragraphs short.
- Lead with the useful point first.
- Include specific details from the project or community.
- For trading/intelligence posts, include risk notes and avoid financial guarantees.
