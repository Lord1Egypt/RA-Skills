# 教师评估图表规格

## 班级学业趋势对比图

- **类型**: 多线折线图 (Multi-line Chart)
- **X轴**: 时间（按测验/考试日期）
- **Y轴**: 班级平均分 或 知识点平均掌握率
- **每条线**: 一个班级（不同颜色区分）
- **交互**: 点击图例可显示/隐藏特定班级
- **标注**: 关键节点标注（如期中/期末考试，或某个知识点专项测试）

## 教师综合能力雷达图

- **类型**: 雷达图 (Radar Chart)
- **维度**: 知识点覆盖度 / 学生进步度 / 均衡性 / 反馈时效 / 趋势方向
- **叠加**:
  - 本人：蓝色实线
  - 同科年级平均：灰色虚线
  - 全校优秀水平：金色虚线
- **辅助文字**: 各维度得分、与平均值的差异

## 同科教师能力排序图

- **类型**: 横向柱状图 (Horizontal Bar Chart)
- **Y轴**: 教师姓名（实名/匿名按权限）
- **X轴**: 综合评分
- **每条柱分色段**: 用渐变色或叠加柱展示各维度贡献
- **参考线**: 年级平均线、全校平均线

## 知识点-教师热力图

- **类型**: 热力图 (Heatmap)
- **X轴**: 知识点名称
- **Y轴**: 教师姓名
- **色阶**: 教师所在班级在该知识点上的掌握率
- **用途**: 快速定位各教师的教学强项和弱项知识点

## 教学效果趋势图（个人）

- **类型**: 折线图 + 柱状图组合
- **折线**: 综合评分随时间变化
- **柱状图**: 各维度的具体分数（堆叠）
- **标注**: 标注关键事件（如换班、参加教研活动等）

## 对比维度选择器

以下筛选条件用于所有对比图：

```
年级:     [三年级 ▼]
学科:     [数学 ▼]
对比维度: [班级 ▼ / 教师 ▼]
时间范围: [本学期 ▼ / 自定义]
显示方式: [实名 ▼ / 匿名]
```
