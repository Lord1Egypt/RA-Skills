# 📊 Data Visualization Analyst

**首席数据智能官 — 多维数据分析与可视化洞察**

> 第一性原理驱动的数据分析师，穿透表象挖掘底层规律，用可视化图表输出可落地的决策建议。

---

## ✨ 能力

| 维度 | 说明 |
|:--|--|
| **多模态解析** | 截图、表格、文本、聊天记录、碎片化信息全格式支持 |
| **多维模型构建** | 象限矩阵 / 帕累托分析 / 趋势追踪 / 漏斗转化 / 归因分析 |
| **可视化图表** | 4-6组交互式图表（KPI看板 / 柱状图 / 折线图 / 饼图 / 散点图） |
| **第一性原理洞察** | 幂律定律 / 复利衰减 / 临界质量 / 马太效应 / 投入产出弹性 |
| **决策驱动包** | 停止（止损）→ 优化（调优）→ 放大（核爆公式） |

## 🧠 分析思维框架

- **第一性原理视角**：从零拆解物理成本结构，识别历史惯性
- **算法效率视角**：系统化重构资源配置，最大化匹配效率
- **极致精简视角**：砍掉非本质，放大核心触点

## 🔧 使用方式

适用于 [WorkBuddy](https://www.codebuddy.cn) 平台。将 `SKILL.md` 安装为用户级 skill 即可在对话中触发：

1. 上传或粘贴任何业务数据
2. 触发关键词：`"帮我分析"` / `"ROI"` / `"综合分析"` / `"可视化"`
3. 输出：4-6组交互式图表 + 结构化分析报告 + 决策建议
