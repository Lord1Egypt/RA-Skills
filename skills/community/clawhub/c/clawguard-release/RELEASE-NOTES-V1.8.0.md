# 🦞 ClawGuard V1.8.0 - 发布说明

**发布日期**: 2026-03-17  
**版本**: V1.8.0  
**核心功能**: 数字水印 + 区块链确权

---

## 🎉 新增功能

### 1. 数字水印功能 💧

**功能描述**:
- 在图片右下角添加可见水印
- 水印内容：`🦞 证明 ID | © 权属人`
- 自动根据图片大小缩放字体
- 半透明黑色背景 + 白色文字

**支持格式**:
- JPG/JPEG
- PNG
- BMP
- GIF
- WEBP

**使用方式**:
1. 批量确权界面勾选"在图片中添加水印"
2. 确权时自动为图片添加水印
3. 水印永久嵌入图片

**依赖**:
- Pillow 库（`pip install Pillow`）
- 很多系统已预装

---

### 2. 灵活证明保存 📁

**三种保存方式**:
1. ✅ 生成.clawguard 证明文件夹（推荐）
2. ✅ 保存到全局索引 (bh_data/)
3. ✅ 导出证书到指定位置

**用户可自由选择任意组合**

---

### 3. 证书优化 📄

**单个文件证书**:
- INDEX.txt（人类可读）
- INDEX.json（机器可读）
- certificates/*.txt（单个证书）
- README.txt（使用说明）

**批量证书**:
- 文件清单（前 10 个）
- 统计信息（总数/成功/失败）
- 保存位置说明

## 📦 安装说明

### 基础安装（推荐）
直接使用，无需额外依赖。

### 完整安装（包含水印）
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 按需安装
```bash
# 只安装水印功能
pip install Pillow -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 🎯 使用场景

### 场景 1: 摄影师保护作品
1. 批量选择照片
2. 勾选"添加数字水印"
3. 确权后每张照片都有证明 ID

4. 文件夹随项目一起归档

1. 为设计稿添加水印
2. 生成证书文件夹
3. 连同设计稿一起交付
4. 客户可随时验真

## 🐛 已知问题

### 问题 1: 盲水印不兼容
- **原因**: blind-watermark 库 API 变化 + OpenCV 路径问题
- **解决**: 改用可见水印（Pillow）
- **状态**: ✅ 已解决

### 问题 2: 水印不支持自动提取
- **原因**: 可见水印需要 OCR 识别
- **解决**: 用户手动查看水印文字
- **状态**: ⚠️ 待优化（未来版本）
