## Description: <br>
super-dev-factory guides agents through living-spec-driven, atomized, parallel full-stack development with integrated regression testing. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineering teams use this skill to break new projects, features, fixes, or changes into small implementation units, coordinate parallel work, update living specifications, and converge on integrated browser-based regression testing. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The workflow can propose broad code, test, documentation, and database migration changes across a project. <br>
Mitigation: Review proposed code and migrations before applying them, and run the workflow on a branch or non-production environment. <br>
Risk: The skill makes strong quality and testing claims that may not eliminate defects in a real project. <br>
Mitigation: Continue normal code review, security review, and regression testing before release. <br>
