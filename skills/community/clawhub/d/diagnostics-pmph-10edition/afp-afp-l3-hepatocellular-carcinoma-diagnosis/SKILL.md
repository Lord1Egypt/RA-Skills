# AFP与AFP-L3联合判读用于肝癌诊断

## 触发条件
- 血清AFP升高 **或**
- 需鉴别肝癌与良性肝病（如肝炎、肝硬化）

## 前提条件
- 已排除妊娠、活动性肝病、生殖系胚胎源性肿瘤
- 已完成血清AFP及AFP-L3检测

## 执行步骤

### 1. 判读AFP基础水平
- 参考区间：≤7.0 ng/ml（CLIA法）
- 若满足以下任一标准 **且** 已排除干扰因素，则高度怀疑肝癌：
  - AFP ≥ 400 ng/ml 持续 >1个月，**或**
  - AFP ≥ 200 ng/ml 持续 ≥2个月
- 此时应立即安排B超、CT/MRI及活检以确诊

### 2. 结合AFP-L3%辅助判断
- 计算 AFP-L3% = (AFP-L3 / 总AFP) × 100%
- 判读标准：
  - **AFP-L3% ≥ 10%**：高度怀疑肝癌，需紧急评估
  - **AFP-L3% < 10%**：不能排除肝癌，应联合影像学或PIVKA-Ⅱ等其他检查

### 3. 动态监测建议
- 对已确诊肝癌患者，定期监测AFP用于术后/移植后复发监测及预后评估

### 4. 注意事项
- 约30%~40%肝癌患者AFP为阴性，不可仅凭AFP阴性排除肝癌
- 良性肝病（如急慢性肝炎、肝硬化）可致AFP轻度升高（20~200 ng/ml），通常2个月内随病情好转下降

## 输出行动
若满足AFP升高标准且排除干扰因素，应启动影像学与病理检查；若AFP-L3%≥10%，强烈提示肝癌，需紧急评估。
