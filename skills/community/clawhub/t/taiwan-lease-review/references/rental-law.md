# 租賃專法重點摘要

## 租賃住宅市場發展及管理條例（租賃專法）

### 適用範圍
- **住宅租賃**：整層、分租、套房、雅房（不含學生宿舍、社福機構）
- **不適用**：店面、辦公室、停車位獨立出租
- 生效日期：2018年6月27日

### 房客保護核心規定

| 項目 | 規定內容 |
|------|----------|
| 押金上限 | 不得超過**二個月租金總額** |
| 電費計算 | 不得超過台電夏季/非夏季最高級距單價 |
| 提前終止違約金 | 承租人最多一個月租金 |
| 修繕責任 | 房屋結構/管線/設備由出租人負擔 |
| 終止租約催告 | 欠租須達2個月總額，經催告30日仍未繳納 |
| 不得禁止設籍 | 出租人不得拒絕承租人辦理戶籍登記 |
| 不得禁止申報補貼 | 出租人不得拒絕承租人申請租金補貼 |
| 出租人進入限制 | 未經同意不得任意進入租賃住宅 |

### 懲罰性賠償

出租人違反以下事項，房客可主張**1倍違約金**：
- 強制或禁止房客設籍
- 強制或禁止房客申請租金補貼
- 電費超收
- 未依規定返還押金

## 租金補貼方案（2023年起）

### 中央擴大租金補貼
- 申請條件擴張（18歲以上即可，無所得門檻）
- 補貼金額：2,000~8,000元/月（依縣市/家庭類型）
- 房東稅務優惠：每月1.5萬元免稅額（符合公益出租人）

### 房東不得拒絕
- 依租賃專法，出租人不得以任何理由拒絕承租人申請租金補貼
- 違反者：可處1~5萬元罰鍰，承租人得終止租約

## 訴訟實務重點

### 收回房屋程序（出租人）
1. 欠租 → 先書面催告（存證信函）
2. 經催告30日仍未繳 → 終止租約
3. 終止租約後 → 調解/支付命令/訴訟

### 減免租金（承租人）
- 租賃物因天災/不可抗力致無法使用 → 可請求減免
- 房屋有瑕疵影響居住品質 → 可請求減少租金

### 租約到期拒絕搬遷
- 得聲請法院強制執行
- 需有確定判決或調解成立
- 通常需3-6個月程序
