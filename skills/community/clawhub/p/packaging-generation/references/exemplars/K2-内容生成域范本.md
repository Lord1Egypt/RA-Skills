## K2-内容生成域范本

### K2-01 产品包装生成范本

**任务目标**：生成产品外包装、内包装、标签设计

**执行步骤**：

1. **整理产品信息**
   - 产品名称：简洁、易记
   - 产品成分：完整、准确
   - 使用方法：清晰、易懂
   - 保质期：明确标注

2. **设计包装结构**
   - 外包装：保护产品、展示品牌
   - 内包装：保护产品、方便使用
   - 标签：信息完整、易于识别

3. **设计包装外观**
   - 品牌元素：logo、品牌色、品牌字体
   - 产品图片：高质量、展示产品
   - 信息布局：清晰、易读

4. **选择包装材质**
   - 材质类型：纸质、塑料、金属等
   - 材质质量：环保、耐用
   - 材质成本：控制成本

**输出示例**：

```markdown
## 产品包装设计方案

### 产品信息
- 产品名称：天然洗发水
- 产品成分：天然植物提取物
- 使用方法：取适量洗发水，揉搓起泡，按摩头皮
- 保质期：3年

### 包装结构
- 外包装：纸盒包装，保护产品
- 内包装：塑料瓶，方便使用
- 标签：产品信息标签

### 包装外观
- 品牌元素：品牌logo、绿色品牌色
- 产品图片：产品效果图
- 信息布局：产品名称→成分→使用方法→保质期

### 包装材质
- 材质类型：外包装纸质，内包装塑料
- 材质质量：环保材料
- 材质成本：适中
```

### K2-02 礼品包装生成范本

**任务目标**：生成礼盒、礼品袋、礼品包装设计

**执行步骤**：

1. **整理礼品信息**
   - 礼品名称：精美、有档次
   - 赠送对象：朋友、家人、客户
   - 赠送场合：生日、节日、商务

2. **设计包装形式**
   - 礼盒：高档、有质感
   - 礼品袋：便携、有设计感
   - 包装纸：美观、有创意

3. **设计包装外观**
   - 色彩搭配：符合场合
   - 图案设计：精美、有品味
   - 装饰元素：丝带、贺卡、标签

4. **搭配包装装饰**
   - 丝带：增加档次感
   - 贺卡：表达祝福
   - 标签：注明赠送人

**输出示例**：

```markdown
## 礼品包装设计方案

### 礼品信息
- 礼品名称：高档茶叶礼盒
- 赠送对象：商务客户
- 赠送场合：商务拜访

### 包装形式
- 包装形式：礼盒
- 包装档次：高档、有质感
- 包装尺寸：30cm×20cm×10cm

### 包装外观
- 色彩搭配：深蓝色+金色
- 图案设计：中国传统纹样
- 装饰元素：金色丝带、烫金logo

### 包装装饰
- 丝带：金色丝带，增加档次感
- 贺卡：商务贺卡，表达感谢
- 标签：品牌标签，注明品牌
```

### K2-04 化妆品包装生成范本

**任务目标**：生成化妆品包装、护肤品包装、香水包装设计

**执行步骤**：

1. **整理化妆品信息**
   - 产品名称：高端、有品牌感
   - 产品成分：完整、准确
   - 使用方法：清晰、易懂
   - 功效：明确说明

2. **设计包装结构**
   - 密封性：保证产品质量
   - 防氧化：延长产品寿命
   - 便携性：方便使用

3. **设计包装外观**
   - 品牌元素：logo、品牌色、品牌字体
   - 产品图片：高端、有质感
   - 信息布局：清晰、易读

4. **提升包装质感**
   - 材质质感：有档次
   - 印刷工艺：烫金、UV、压纹
   - 细节处理：精致、有品味

**输出示例**：

```markdown
## 化妆品包装设计方案

### 化妆品信息
- 产品名称：抗衰老精华液
- 产品成分：玻尿酸、胶原蛋白、维生素E
- 使用方法：洁面后，取适量涂抹于面部
- 功效：抗衰老、保湿、修复

### 包装结构
- 密封性：真空包装，保证产品新鲜
- 防氧化：避光包装，延长产品寿命
- 便携性：按压瓶设计，方便使用

### 包装外观
- 品牌元素：品牌logo、玫瑰金品牌色
- 产品图片：高端产品效果图
- 信息布局：产品名称→成分→使用方法→功效

### 包装质感
- 材质质感：玻璃瓶，有档次
- 印刷工艺：烫金logo、UV印刷
- 细节处理：精致瓶盖、细腻手感
```
