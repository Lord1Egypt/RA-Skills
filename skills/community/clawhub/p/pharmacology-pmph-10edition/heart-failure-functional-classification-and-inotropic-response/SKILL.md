# heart-failure-functional-classification-and-inotropic-response

## 适用条件
- 患者已确诊为心力衰竭
- 已完成心功能评估（包括射血分数、心输出量等指标）
- 排除非心源性循环障碍及急性心肌梗死早期未稳定状态

## 执行步骤

1. **判断心力衰竭的功能类型**：
   - **收缩性心力衰竭**：心肌收缩力减弱，心输出量减少，左心室射血分数（LVEF）明显下降，组织灌注不足；此类患者**对正性肌力药物反应良好**。
   - **舒张性心力衰竭**：心室充盈异常、舒张受限、顺应性降低，舒张末期压升高，但射血分数下降不明显甚至正常；此类患者**对正性肌力药物反应差**。
   - **高输出量心力衰竭**：由贫血、甲状腺功能亢进、动静脉瘘等引起，心输出量不减少甚至增高；常规抗心衰药物（包括正性肌力药）**难以奏效**。

2. **根据分型指导药物选择**：
   - 收缩性心衰：可考虑使用正性肌力药物。
   - 舒张性心衰：应避免依赖正性肌力药物，优先处理容量负荷和舒张功能。
   - 高输出量心衰：重点治疗原发病，常规抗心衰药物效果有限。

## 关键变量
- **射血分数（LVEF）**：数值型，用于区分收缩与舒张功能障碍。
- **心输出量**：数值型，用于识别高输出量型心衰。
- **病因**：分类变量（如贫血、甲亢、动静脉瘘），用于确认高输出量型诊断。

## 输出结果
明确心衰功能类型，并给出是否使用正性肌力药物及预期疗效的建议。
