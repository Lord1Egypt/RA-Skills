## Description: <br>

This skill is for research and development only. <br>

## Known Risks and Mitigations: <br>
Risk: The crawler fetches external web content and can be configured for different target URLs. <br>
Mitigation: Run it only against URLs you intend to crawl, respect the target site's crawling policy, and use an isolated Python environment. <br>
Risk: Generated summaries and links are derived from web content and may contain untrusted or inaccurate information. <br>
Mitigation: Review generated JSON and Markdown before relying on the content or following links. <br>
Risk: Dependency versions are specified as minimum ranges, which can affect reproducibility. <br>
Mitigation: Review or pin dependency versions when reproducible runs are required. <br>
