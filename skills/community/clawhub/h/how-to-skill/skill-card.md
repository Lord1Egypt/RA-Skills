### Deployment Geography for Use: <br>

## Known Risks and Mitigations: <br>
Risk: Instruction-only guidance may still lead an agent to draft or publish low-quality or unintended skills if followed without review. <br>
Mitigation: Install this as readable guidance for creating skills. Review any generated skill before publishing, and only run the example publish command when you explicitly intend to publish under your own account. <br>
