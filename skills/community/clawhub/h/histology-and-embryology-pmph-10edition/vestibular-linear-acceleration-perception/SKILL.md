# 前庭系统直线加速度感知机制

## 何时使用
- 分析人体平衡觉中对直线加速度（如启动、刹车、升降）的感知机制
- 解释重力方向感知的组织学基础
- 区分前庭系统中直线运动与旋转运动的不同感受器

## 执行步骤
1. **确认适用结构**：仅适用于膜迷路中的椭圆囊与球囊，其内含位觉斑（椭圆囊斑和球囊斑）。
2. **识别位觉斑空间取向**：
   - 椭圆囊斑位于椭圆囊底壁，长轴水平，主要感知水平方向直线加速度；
   - 球囊斑位于球囊前壁，长轴垂直，主要感知垂直方向直线加速度与重力；
   - 二者相互垂直，确保全方向覆盖。
3. **描述关键组成**：
   - 位觉斑由支持细胞和毛细胞构成；
   - 支持细胞分泌糖蛋白形成耳石膜，内含碳酸钙结晶（耳石）；
   - 毛细胞顶部有静纤毛（40～80根）和一根动纤毛，均嵌入耳石膜。
4. **解释感知机制**：
   - 耳石比重远大于内淋巴，在直线变速或重力作用下产生惯性位移；
   - 耳石膜相对毛细胞胞体移动，导致纤毛弯曲；
   - 纤毛向动纤毛方向弯曲时兴奋，反向则抑制；
   - 信号经突触传递至前庭神经，最终传入中枢神经系统。
5. **强调功能限制**：本机制仅响应直线加速度和静态重力，不参与角加速度（旋转）感知。

> 注意：若问题涉及半规管或旋转运动感知，请勿使用本技能。
