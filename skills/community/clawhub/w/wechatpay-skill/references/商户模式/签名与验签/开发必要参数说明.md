# 开发必要参数说明

## 商户模式必填参数

| 参数 | 来源 | 说明 |
|------|------|------|
| **mchid** | 商户平台 | 商户号，15位数字 |
| **serial_no** | 商户平台 → API安全 | 商户API证书序列号 |
| **private_key** | 商户平台 → API安全 → 下载证书 | 商户API私钥（PKCS8 PEM） |
| **appid** | 公众平台/小程序平台 | AppID |
| **api_v3_key** | 商户平台 → API安全 | APIv3密钥（32字节） |
| **notify_url** | 商户自己提供 | 支付结果回调地址 |

## 获取步骤

### 1. 商户号（mchid）
- 登录微信商户平台
- 账户中心 → 商户信息 → 商户号

### 2. API证书和序列号
1. 登录微信商户平台
2. 账户中心 → API安全 → 申请证书
3. 下载证书压缩包，包含：
   - `apiclient_cert.p12`（V2用，可选）
   - `apiclient_cert.pem`（V3用公钥）
   - `apiclient_key.pem`（V3用私钥）
4. 证书序列号在商户平台查看

### 3. APIv3密钥
1. 登录微信商户平台
2. 账户中心 → API安全 → 设置密钥
3. 设置32位APIv3密钥（自己生成）

### 4. AppID
- **公众号**：公众平台 → 设置与开发 → 基本配置
- **小程序**：小程序管理后台 → 开发 → 开发管理

### 5. 支付授权目录（JSAPI/H5）
- 登录公众平台
- 微信支付 → 支付设置 → 支付授权目录
- 添加格式：`https://yourdomain.com/pay/`

## 服务商模式额外参数

| 参数 | 说明 |
|------|------|
| **sp_mchid** | 服务商商户号 |
| **sp_serial_no** | 服务商证书序列号 |
| **sp_appid** | 服务商AppID（公众号/小程序） |
| **sub_mchid** | 子商户商户号 |
| **sub_appid** | 子商户AppID |
| **sub_openid** | 子用户openid |

## 参数配置示例

### Python配置示例

```python
# wechatpay_config.py
WECHATPAY_CONFIG = {
    "mchid": "1234567890",           # 商户号
    "serial_no": "XXXXXXXXXXXXXXXX", # 证书序列号
    "private_key": """-----BEGIN PRIVATE KEY-----
    ...（私钥内容）...
    -----END PRIVATE KEY-----""",
    "appid": "wxa5f5c1d6e8f9a2b3",    # AppID
    "api_v3_key": "0123456789abcdef", # APIv3密钥（32字节）
    "notify_url": "https://yourdomain.com/pay/notify",
}
```

### Node.js配置示例

```javascript
// wechatpay_config.js
module.exports = {
    mchid: "1234567890",              // 商户号
    serialNo: "XXXXXXXXXXXXXXXX",    // 证书序列号
    privateKey: `-----BEGIN PRIVATE KEY-----
...（私钥内容）...
-----END PRIVATE KEY-----`,
    appid: "wxa5f5c1d6e8f9a2b3",      // AppID
    apiV3Key: "0123456789abcdef",    // APIv3密钥
    notifyUrl: "https://yourdomain.com/pay/notify",
};
```

## 常见错误

| 错误 | 原因 |
|------|------|
| 商户号与AppID不匹配 | appid和mchid未在商户平台绑定 |
| 证书序列号不存在 | 使用了错误的序列号或证书过期 |
| 私钥格式错误 | 未使用PKCS8格式，或有空格/换行问题 |
| 签名错误 | 私钥与证书不匹配 |
