## 二、股权架构合规

### 2.1 股权结构审查清单

```
□ 股东身份是否清晰（自然人/法人/特殊主体）
□ 持股比例是否符合法规要求
□ 股权代持是否已清理或规范化
□ 员工期权池是否已设立
□ 国有股东是否完成审批
□ 外资股东是否合规
□ 关联方持股是否披露
```

### 2.2 股权代持处理

| 情形 | 处理方式 | 时间要求 |
|------|----------|----------|
| **可以保留** | 签署代持协议+公证 | T-1年完成 |
| **必须还原** | 股权转让还原 | T-1.5年完成 |
| **特殊情况** | 出具法律意见说明 | T-1年完成 |

### 2.3 期权池设立

| 项目 | 具体要求 |
|------|----------|
| **持股平台** | 有限合伙企业或资管计划 |
| **持股比例** | 上市前预留10-20% |
| **行权条件** | 服务期限+业绩考核 |
| **行权价格** | 公允价值或折价 |
| **管理机制** | 明确的授予、成熟、行权流程 |

### 2.4 国有股东处理

| 情形 | 处理方式 |
|------|----------|
| **国有股转持** | 按规定履行转持义务 |
| **国有股增值** | 需进行资产评估备案 |
| **国有股转让** | 需通过产权交易所 |

## 三、知识产权合规

### 3.1 IP梳理清单

```
□ 商标：自有商标、许可使用商标、傍名牌排查
□ 专利：自有专利、许可专利、侵权风险
□ 著作权：软件著作权、作品著作权、授权链条
□ 技术秘密：保密措施、竞业限制
□ 域名：自有域名、到期风险
□ 软件：正版化、授权链
```

### 3.2 核心IP要求

| IP类型 | 要求 | 证明文件 |
|--------|------|----------|
| **自有商标** | 注册在发行人名下 | 商标注册证 |
| **核心专利** | 与主营业务相关 | 专利证书 |
| **软件著作权** | 登记在发行人名下 | 著作权登记证书 |
| **授权IP** | 完整授权链 | 授权协议 |

### 3.3 IP风险排查

| 风险类型 | 排查内容 | 应对措施 |
|----------|----------|----------|
| **侵权风险** | 是否侵犯他人IP | 规避设计或购买许可 |
| **无效风险** | 自有IP是否稳固 | 无效宣告应对 |
| **许可风险** | 授权是否完整 | 补签授权协议 |
| **到期风险** | IP是否到期 | 续展或替代 |

## 四、业务合规证明

### 4.1 业务资质清单

| 行业 | 常见资质 | 主管部门 |
|------|----------|----------|
| **科技类** | 高新技术企业、软件企业 | 科技局 |
| **增值电信** | ICP许可证 | 通信管理局 |
| **出版类** | 网络出版许可证 | 宣传部 |
| **教育类** | 办学许可证 | 教育局 |
| **食品类** | 食品经营许可证 | 食药监局 |

### 4.2 合规证明获取

```
□ 营业执照：合法有效
□ 行业许可证：齐全有效
□ 排污许可证：合规排放
□ 安全生产证：高危行业必备
□ 消防验收：合规通过
□ 员工社保：合规缴纳
□ 税务合规：无重大违规
```

### 4.3 劳动合规检查

| 检查项 | 标准 | 风险 |
|--------|------|------|
| **劳动合同** | 100%签署 | 未签合同双倍工资 |
| **社保缴纳** | 全员缴纳 | 补缴+罚款 |
| **公积金** | 按规定缴纳 | 补缴+罚款 |
| **加班** | 依法支付加班费 | 劳动仲裁 |
| **解除** | 依法解除 | 违法解除赔偿 |

## 五、VIE架构设计

### 5.1 VIE适用场景

| 业务类型 | 是否需要VIE | 说明 |
|----------|-------------|------|
| **纯内资业务** | 不需要 | 直接持股 |
| **外资受限业务** | 需要 | 典型VIE |
| **混合业务** | 部分需要 | 区分处理 |

### 5.2 VIE架构设计原则

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         典型VIE架构                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│                        ┌─────────────────┐                                  │
│                        │  上市公司主体   │                                  │
│                        │ (HK/BVI Listed) │                                  │
│                        └────────┬────────┘                                  │
│                                 │ 100%                                        │
│                        ┌────────┴────────┐                                  │
│                        │  香港子公司     │                                  │
│                        │  (HK Subsidiary)│                                  │
│                        └────────┬────────┘                                  │
│                                 │ WVIE协议                                   │
│                        ┌────────┴────────┐                                  │
│                        │  境内WFOE       │                                  │
│                        │ (协议控制主体)   │                                  │
│                        └────────┬────────┘                                  │
│                                 │ 协议控制                                   │
│                        ┌────────┴────────┐                                  │
│                        │  运营实体        │                                  │
│                        │ (内资公司+许可)  │                                  │
│                        └─────────────────┘                                  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 5.3 VIE协议体系

| 协议类型 | 主要内容 | 作用 |
|----------|----------|------|
| **股权质押协议** | 创始人股权质押给WFOE | 控制权保障 |
| **独家购买权协议** | WFOE独家购买运营实体股权 | 随时转为直接持股 |
| **独家服务协议** | WFOE向运营实体提供服务 | 利润转移 |
| **投票权委托协议** | 创始人投票权委托WFOE | 控制权保障 |
| **借款协议** | WFOE向创始人提供借款 | 股权质押前提 |

## 六、诉讼与处罚清理

### 6.1 诉讼排查清单

```
□ 未了结的民事诉讼
□ 潜在的诉讼风险
□ 仲裁案件
□ 行政处罚案件
□ 刑事风险
```

### 6.2 清理要求

| 情形 | 处理要求 |
|------|----------|
| **重大诉讼** | 必须在上市前解决或取得法律意见 |
| **一般诉讼** | 如实披露，评估影响 |
| **行政处罚** | 必须缴纳罚款、整改完成 |
| **刑事风险** | 必须在上市前消除 |

### 6.3 合规证明获取时间表

| 证明类型 | 获取时间 | 有效期 |
|----------|----------|--------|
| **工商无违规证明** | T-3个月 | 1个月 |
| **税务无违规证明** | T-3个月 | 1个月 |
| **社保合规证明** | T-3个月 | 1个月 |
| **环保合规证明** | T-6个月 | 3个月 |
| **无行贿受贿证明** | T-3个月 | 1个月 |
