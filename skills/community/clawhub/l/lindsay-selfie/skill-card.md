## Known Risks and Mitigations: <br>
Risk: The skill generates identity-based selfie images from local reference photos, and broad triggers could create or store images when the user only intended casual conversation. <br>
Mitigation: Use only where the identity reference images are authorized, verify the referenced files and gallery path, and require explicit confirmation before creating or storing identity-based images. <br>
