# 鼻咽癌组织学分类规则

## 适用条件
仅适用于以下情况：
- 病理标本来源于鼻咽部
- 已确诊为原发性癌（上皮源性）
- 排除淋巴瘤、肉瘤及转移性癌

## 执行步骤

1. **判断是否为鳞状细胞癌**（占绝大多数）：
   - **若是**，按分化程度进一步分类：
     - **分化性鳞癌**：
       - **角化型**：高分化，可见细胞间桥、细胞内角化或角化珠。
       - **非角化型**：低分化，无角化、无细胞间桥，细胞形态多样；最常见类型，通常与EB病毒相关。
     - **未分化性鳞癌**：
       - **泡状核细胞癌**：癌细胞呈片状或不规则巢状，胞质丰富呈合体状，核大空泡状伴明显核仁，核分裂象少见，癌巢间有大量淋巴细胞浸润；约占10%，对放疗高度敏感。
       - **小细胞未分化癌**：癌细胞小、胞质少，呈小圆形或短梭形，弥漫分布，无巢状结构；需与恶性淋巴瘤、未分化横纹肌肉瘤、神经母细胞瘤鉴别。
   - **若否**，考虑**腺癌**（少见）：
     - 起源于柱状上皮或小腺体。
     - **高分化型**：如柱状细胞腺癌、乳头状腺癌。
     - **低分化型**：腺样结构不明显，癌细胞较小。
     - **极少数**为黏液腺癌。

2. **必要时辅以辅助检查**：
   - 对于小细胞未分化癌等疑难病例，必须结合免疫组化（如CK、LCA、desmin、NF）或电镜检查进行鉴别诊断。

3. **输出结果**：
   - 明确给出组织学类型（例如：“非角化型低分化鳞状细胞癌”或“泡状核细胞癌”）。
   - 注明是否与EB病毒相关（尤其在非角化型中）。
   - 提示治疗意义（如放疗敏感性）。

> 注意：因鼻咽部储备细胞具有多向分化潜能，部分病例组织构象复杂，当前尚无完全统一的分类标准，需结合临床与多模态病理证据综合判断。
