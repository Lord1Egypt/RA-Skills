# 亚马逊新品上架检查清单

本清单覆盖从选品决策到FBA上架的完整流程，帮助你确保每个关键步骤都已完成，避免因遗漏细节导致不必要的损失。

---

## 第一阶段：上架前决策检查

### 1.1 市场与品类分析

- [ ] 核心关键词月搜索量 > 10,000
- [ ] Top 10同行平均评论数 < 500（存在机会窗口）
- [ ] 头部产品价格区间存在差异化空间
- [ ] 品类没有严重的垄断或专利壁垒
- [ ] 产品不属于亚马逊限制类目或需要审核类目
- [ ] 季节性分析已完成，确认全年平均需求稳定
- [ ] 目标市场的节假日/大促节点已规划

### 1.2 同行分析

- [ ] 已找到10个对标同行（优先选择近6个月上架的新品）
- [ ] 对标同行的标题结构已拆解分析
- [ ] 对标同行的五点描述策略已记录
- [ ] 对标同行的定价和促销策略已记录
- [ ] 对标同行的评论已分析（好评关键词、差评痛点）
- [ ] 对标同行的QA重点已记录
- [ ] 同行的差异化机会已识别

### 1.3 差异化定位

- [ ] 已确定差异化方向（价格/功能/视觉/服务）
- [ ] 产品卖点清单已整理（不超过5个核心卖点）
- [ ] 产品与同行的对比图已完成设计
- [ ] 品牌名称已确定（已在亚马逊完成品牌备案或准备备案）
- [ ] 品牌故事/品牌宣言已撰写

### 1.4 成本与利润测算

- [ ] 产品成本已确认（含包装）
- [ ] 头程物流方式已确定（海运/空运/快递）及费用已测算
- [ ] FBA费用已计算（配送费+月租费+仓储费）
- [ ] 亚马逊佣金已计算（通常15%）
- [ ] 预期退货率已评估（3C产品5-10%，家居用品3-5%）
- [ ] 预期广告ACOS已设定（新品期目标30-50%）
- [ ] 单品净利润率计算完成（目标 > 25%）
- [ ] 首批备货量已确定（建议覆盖30-45天销售）

---

## 第二阶段：产品准备检查

### 2.1 产品合规

- [ ] 产品符合目标市场的安全标准（UL/CE/FCC等）
- [ ] 产品标签符合亚马逊要求（含FNSKU、Made in China等）
- [ ] 包装符合亚马逊FBA要求（尺寸、重量、稳定性）
- [ ] 产品不属于亚马逊禁售或限制类目
- [ ] 如有需要，已完成类目审核/产品认证

### 2.2 产品图片

- [ ] 主图已完成（白底、产品占85%以上、无水印、1000×1000像素）
- [ ] 场景图已完成（1-2张，展示真实使用场景）
- [ ] 功能图已完成（展示核心功能和使用细节）
- [ ] 对比图或尺寸图已完成
- [ ] 包装图已完成（展示包装内容和配件）
- [ ] 所有图片格式和分辨率符合要求（PNG/JPG、1000像素以上）

### 2.3 产品视频（可选但推荐）

- [ ] 主图视频已完成（时长60-90秒，展示产品核心卖点）
- [ ] 视频分辨率符合要求（1920×1080或以上）
- [ ] 视频不含任何违规内容（无对比性声明、无夸大效果）

---

## 第三阶段：Listing创建检查

### 3.1 基础信息

- [ ] 产品类型（Category/Department）选择正确
- [ ] 搜索词（Search Terms）已优化（不超过249字节）
- [ ] 制造商信息已填写
- [ ] 商品编码（UPC/EAN）已准备
- [ ] 物品备注（Item notes）已填写（如适用）

### 3.2 标题优化

- [ ] 标题包含核心关键词（前50字符最重要）
- [ ] 标题公式已应用：品牌 + 核心词 + 产品类型 + 特性 + 规格
- [ ] 标题长度控制在200字符以内
- [ ] 标题无拼写错误
- [ ] 标题不包含促销信息（"Sale"、"Hot"等）
- [ ] 标题不包含特殊字符（仅用空格分隔）

### 3.3 五点描述

- [ ] 五个要点都已填写
- [ ] 每点首字母大写
- [ ] 每点控制在200字符以内
- [ ] 结构遵循：痛点 → 卖点 → 场景 → 优势 → 保障
- [ ] 核心关键词已自然融入
- [ ] 每点强调benefits而非features

### 3.4 产品描述

- [ ] 品牌故事已撰写（500字符以内）
- [ ] 产品规格参数已列出
- [ ] 使用说明/注意事项已包含
- [ ] 保修信息已说明
- [ ] A+内容已上传（如有品牌备案）

### 3.5 变体设置（如适用）

- [ ] 父ASIN已创建
- [ ] 子ASIN变体关系已设置（颜色/尺寸等）
- [ ] 变体主题选择正确
- [ ] 每个子ASIN的库存和价格已分别设置

---

## 第四阶段：FBA物流检查

### 4.1 FBA创建

- [ ] 已在卖家后台创建FBA货件
- [ ] FNSKU标签已生成并打印
- [ ] 产品外箱已贴FNSKU和FBA标签
- [ ] 外箱符合亚马逊尺寸要求
- [ ] 发货计划已确认（海运/空运/快递）

### 4.2 库存管理

- [ ] 库存数量已准确填写
- [ ] 补货提醒已设置（建议库存 < 14天时激活）
- [ ] 头程物流跟踪号已记录
- [ ] 预计到仓时间已确认

### 4.3 备货清单

- [ ] 首批货件数量已确认（覆盖30-45天销售）
- [ ] 备用库存已预留（应对突发情况）
- [ ] 产品说明书/保修卡已准备
- [ ] 品牌感谢卡已准备（如适用）

---

## 第五阶段：上线前最后检查

### 5.1 定价策略

- [ ] 产品定价已确定（参考成本、同行、利润目标）
- [ ] Coupon已设置（建议5-10%）
- [ ] 促销价格已规划（如Prime Day、Black Friday）
- [ ] Vine计划已注册（如适用）

### 5.2 广告设置

- [ ] 自动广告已创建（预算$10-15/天）
- [ ] 首个手动广告组已创建（精准长尾词，预算$5-10/天）
- [ ] 否定关键词清单已准备
- [ ] 每日广告预算上限已设置

### 5.3 合规与账户

- [ ] 卖家绩效指标正常（订单缺陷率 < 1%）
- [ ] 信用卡/收款账户信息已更新
- [ ] 税务信息已确认
- [ ] 账户健康状态已检查

---

## 第六阶段：上架后行动检查

### 6.1 Day 1-3 行动

- [ ] 确认FBA库存已上线
- [ ] Listing页面显示正常
- [ ] 自动广告已开启
- [ ] 手动广告已开启
- [ ] 开始记录BSR排名

### 6.2 Day 4-7 行动

- [ ] 分析第一天的自然流量数据
- [ ] 检查是否有无效流量来源
- [ ] 关注Vine计划进展
- [ ] 记录第一周销量和转化率

### 6.3 Week 2 行动

- [ ] 下载自动广告报告
- [ ] 筛选高效关键词
- [ ] 添加新的手动广告组
- [ ] 更新否定关键词列表
- [ ] 评估是否需要调整Bid

---

## 检查清单使用建议

**上架前**：按照第一至第五阶段的顺序，逐项检查并打勾。每遗漏一项，都可能为未来的运营埋下隐患。

**上架后**：按照第六阶段的节奏执行，每周复盘一次。数据是最好的老师，及时发现问题才能及时调整。

**提醒**：这份清单是通用模板，每个品类可能有特殊要求。请根据你的具体产品情况进行适当调整。
