## Description: <br>
Tracks installed Claude Code skill usage over a selected time range and presents the results as a readable statistics report. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[akira82-ai](https://clawhub.ai/user/akira82-ai) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and Claude Code users use this skill to review how often installed skills have been invoked across recent or all available local history. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill reads local Claude history and project session logs, which may contain secrets, client data, or proprietary prompts. <br>
Mitigation: Use it only on systems where those local logs are approved for analysis, and review the script before running it in sensitive workspaces. <br>
