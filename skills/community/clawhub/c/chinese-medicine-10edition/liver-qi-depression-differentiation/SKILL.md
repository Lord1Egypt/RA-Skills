# Liver Qi Depression Differentiation

执行以下步骤以完成肝气郁结证的辨证：

1. **确认核心情志症状**：检查是否存在精神抑郁、烦躁易怒、善太息（频繁叹气以缓解胸闷）。
2. **定位气滞部位**：
   - 观察是否出现胸胁或少腹胀痛（肝经循行区域）；
   - 询问是否有咽喉哽塞感（“梅核气”表现）。
3. **评估妇科表现（仅限女性）**：检查是否伴有乳房胀痛、痛经、月经不调或闭经。
4. **识别久病入血征象**：查看是否出现胁下痞块、舌质紫暗或边有瘀斑、脉沉弦涩，提示气滞已兼血瘀。
5. **追溯病因**：确认是否存在情志不遂诱因，导致肝失疏泄、气机郁滞。
6. **梳理病机链条**：
   - 情志不遂 → 肝失条达 → 气机郁结 → 经脉不利 → 胀痛；
   - 气郁生痰 → 痰随气逆 → 咽中梗阻；
   - 气滞血瘀 → 痛经、闭经、癥瘕。
7. **鉴别排除化火证**：若见口苦、舌红苔黄、脉弦数等热象，应排除单纯肝气郁结证，考虑肝火上炎证。

**输出结果**：若符合上述全部条件且无化火表现，诊断为肝气郁结证，治宜疏肝解郁、理气畅中，方选柴胡疏肝散或逍遥散加减。
