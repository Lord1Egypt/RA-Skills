## 价格核算公式

### FOB价格计算
```
FOB = 出厂价 + 包装费 + 验货费 + 拖车费 + 报关费
```

### CIF价格计算
```
CIF = FOB + 海运费 + 保险费
     = FOB + 海运费 + FOB × 1.1 × 0.3%
```

### DDP价格计算
```
DDP = CIF + 关税 + 增值税 + 目的港费用
```

### 利润率计算
```
利润率 = (售价 - 成本) / 售价 × 100%
```

## 多语种报价常用语

### 英语
- "Thank you for your inquiry"
- "Please find attached our quotation"
- "We look forward to your order"

### 阿拉伯语
- "شكرا على استفسارك" (感谢您的询盘)
- "مرفق عرض السعر الخاص بنا" (请查收我们的报价)
- "نتطلع إلى طلبك" (期待您的订单)

### 俄语
- "Спасибо за ваш запрос" (感谢您的询盘)
- "Пожалуйста, найдите наше коммерческое предложение" (请查收我们的报价)
- "Мы надеемся на ваш заказ" (期待您的订单)
