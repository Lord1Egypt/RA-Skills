# 论文章节结构与写作要点

## 一、学位论文标准结构（人文社科）

### 硕士论文

```
摘要（中文）
摘要（英文）
目次
圖表目次
緒論
  1.1 研究背景與動機
  1.2 研究問題
  1.3 文獻回顧
  1.4 研究方法
  1.5 章節安排
第 2 章 核心理論/背景
第 3 章 主要論證（一）
第 4 章 主要論證（二）
第 5 章 延伸討論/對話
結論
  總結發現
  研究貢獻
  研究限制
  未來展望
參考文獻
附錄
```

### 本科毕业论文

```
摘要
緒論
第 1 章 背景與方法
第 2 章 文本分析/個案研究
第 3 章 討論
結論
參考文獻
```

## 二、各章節寫作要點

### 緒論（佔全文 8-12%）

**研究背景與動機**：
- 從宏觀問題切入，逐步聚焦到你的具體議題
- 說明這個問題為什麼重要（學術價值 + 現實意義）
- 避免「古今中外」式泛泛而談，聚焦核心脈絡

**研究問題**：
- 明確提出 1-3 個具體的研究問題
- 問題要有可回答性，避免過大或過模糊
- 格式示例：「本研究旨在探討...」、「本文關注的核心問題是...」

**文獻回顧（緒論版，簡述）**：
- 概述已有研究的幾條主要線索
- 指出這些研究未能解決的問題（gap）
- 你的研究如何填補這個 gap

**研究方法**：
- 具體說明你的研究路徑（文本細讀/歷史考證/比較分析/思想史方法等）
- 材料範圍（你分析了哪些文本/使用了哪些史料）
- 論證策略（你是如何從材料推導出論點的）

### 正文章節（佔全文 70-80%）

**每章標準結構**：
```
章標題
  開門見山：本章核心論點（1-2 段）
  節 1：背景鋪墊或理論框架
  節 2-3：核心論證（分層推進）
  節 4：小結/過渡到下一章
```

**論證技巧**：
- 每段一個論點，段首主題句明確
- 論點 → 證據（引文/史料/數據） → 分析 → 小結
- 引文不要堆砌，每條引文後要有你的詮釋
- 轉折處使用過渡句：「然而...」、「值得注意的是...」、「相較之下...」

**引文使用原則**：
- 直接引用：保留原文，加引號，注明出處
- 間接引述：用自己的話概括，仍需注明出處
- 概述觀點：多位學者的共通看法，可合併引用
- 引文長度：一般不要超過 100 字，過長引文單獨成段

### 結論（佔全文 8-10%）

**總結發現**：
- 逐條回答緒論中提出的研究問題
- 概括各章的核心發現，整合為整體論點

**研究貢獻**：
- 理論層面：推進了什麼認識、修正了什麼觀點
- 方法層面：開闢了什麼新路徑
- 材料層面：發掘了什麼新史料/新文本

**研究限制**：
- 坦誠說明本研究未能解決的問題
- 解釋原因（材料限制/篇幅限制/方法局限）
- 避免過度謙虛，但也不要迴避不足

**未來展望**：
- 指出本研究可延伸的方向
- 提示後續研究者可關注的問題

## 三、語言風格

### 學術中文要點

**應該做的**：
- 使用書面語：「探討」、「梳理」、「揭示」、「指出」
- 邏輯連接清晰：「首先...其次...最後...」、「一方面...另一方面...」
- 適度使用長句，但避免過度歐化
- 術語一致：同一概念全文統一譯名/表述

**避免的**：
- 口語詞：「我覺得」、「其實」、「蠻」、「挺」
- 網路用語：「yyds」、「絕絕子」、「破防」
- 主觀情緒詞：「令人震驚的是」、「不可思議的是」
- 模糊量詞：「很多」、「一些」、「大概」→ 改用具體數據或明確範圍

### 常用學術表達

| 場景 | 推薦表達 |
|------|----------|
| 提出觀點 | 本文認為...、筆者主張...、本研究提出... |
| 引述他人 | 如余英時所言...、據王汎森的研究...、有學者指出... |
| 表示轉折 | 然而、相對而言、值得注意的是、與此形成對照的是 |
| 表示補充 | 此外、與此同時、不僅如此、進一步而言 |
| 表示因果 | 由此可見、因此、基於此、鑒於... |
| 表示總結 | 綜上所述、總括而言、歸結起來 |
| 表示限定 | 在某種程度上、就...而言、至少就本文所論 |

## 四、常見問題

**問題 1：章節之間缺乏銜接**
- 對策：每章結尾寫 1-2 段過渡，說明本章發現如何引出下一章的問題

**問題 2：引文過多，自己的聲音被淹沒**
- 對策：遵循「引文 + 詮釋」原則，每條引文後必須有你自己的分析

**問題 3：論點過大，論證撐不住**
- 對策：把大論題拆解為 2-3 個可操作的子問題，分章論證

**問題 4：文獻回顧像書目羅列**
- 對策：按主題/論爭/脈絡組織，每組文獻要說明它們之間的關係

**問題 5：結論變成摘要重複**
- 對策：結論要「昇華」，把具體發現提升到方法論/理論層面
