Compatible with: Lobster AI
Version: 3.0

【Installation Steps】
1. Copy Kai's Horse.skill
2. Navigate to Lobster root directory → skills folder
3. Paste the file and restart Lobster
4. Deployment complete when load prompt appears

【All Interactive Commands】
/kh help          Open help panel
/kh list          View complete list of 48 thought experiments
/kh random        Random experiment selection with interpretation
/kh ask [number]  Basic concept + logic breakdown
/kh ai [number]   Modern AI context reinterpretation
/kh story [number] Literary narrative · atmospheric storytelling
/kh compare A+B   Compare core differences between two experiments
/kh search [keyword] Fuzzy search for related propositions
/kh all           View complete system setup

[Additional Features]
1. Silent background operation, does not interfere with daily chat
2. Auto-detects philosophy, AI, ethics, consciousness topics
3. Consistently cool and speculative writing style, unified KAI persona
4. Supports unlimited follow-up questions and deep speculative discussion
