# 血清-胸腔积液白蛋白梯度（SPFAG）分析

## 何时使用
- 患者确诊或疑似存在胸腔积液
- 需鉴别积液性质为漏出液还是渗出液
- Light标准判断结果不明确，或临床怀疑为充血性心力衰竭、肝硬化、肾病综合征等导致的漏出液

## 执行步骤
1. **同步采集样本**：同时获取患者的静脉血清和胸腔穿刺所得的胸腔积液样本。
2. **测定白蛋白浓度**：使用标准化生化方法测定血清白蛋白和胸腔积液白蛋白浓度，确保单位一致（通常为g/dL）。
3. **计算SPFAG**：应用公式 SPFAG = 血清白蛋白浓度 - 胸腔积液白蛋白浓度。
4. **临床解读**：
   - 若 SPFAG ≥ 1.2 g/dL，提示为**漏出液**
   - 若 SPFAG < 1.2 g/dL，提示为**渗出液**
5. **结合临床综合判断**：特别注意患者是否正在接受利尿剂治疗或其他可能影响结果的因素。

> 注意：本技能仅适用于胸腔积液，不适用于腹腔或其他体腔积液；若积液为血性或乳糜性，需谨慎解读结果。

详细公式、判读阈值依据及与其他诊断标准的比较见参考文档。
