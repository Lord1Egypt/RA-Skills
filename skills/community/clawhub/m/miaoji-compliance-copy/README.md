# 秒技·合规文案重写 (miaoji-compliance-copy)

> 🛡️ 粗糙文案 → 秒级清洗 → Rufus可读 + 合规安全

## 核心价值

| 痛点 | 解决方案 |
|------|----------|
| AI文案像机器人 | 自然场景化改写 |
| 含违规词被下架 | 智能检测+替换建议 |
| Rufus搜索排名低 | 植入意图词+场景词 |

## 功能概览

### 1. 合规性检查
- Amazon平台禁止词扫描
- 过度营销词检测
- 替换建议生成

### 2. Rufus适配改写
- 用户意图词植入
- 场景化描述优化
- 主观属性词增强

### 3. 多版本输出
- **版本A**：标准合规版
- **版本B**：场景强化版（Rufus优化）
- **版本C**：转化导向版

## 快速开始

### 输入格式
```
产品品类：[产品品类名称]

原始标题：
[粘贴标题]

五点描述：
1. [第一条]
2. [第二条]
3. [第三条]
4. [第四条]
5. [第五条]
```

### 输出内容
1. 合规扫描报告
2. 3个版本的重写文案
3. 搜索词补充建议

## 示例

**输入**：
```
产品品类：护膝

原始标题：
Best护膝|保证有效|顶级材质|膝盖疼痛专用

五点描述：
1. 采用顶级材质
2. 保证快速缓解疼痛
3. 适合中老年人
4. 销量第一
5. 免费 shipping
```

**输出（示例）**：
```
🔍 合规扫描结果
❌ 发现问题词：Best护膝、保证有效、顶级材质、快速缓解、销量第一、免费 shipping
建议替换：人气之选、用户反馈良好、优质面料、舒缓不适、深受喜爱、包邮

📝 版本A：标准合规版
【优化标题】
护膝运动款 | 舒适透气 | 膝盖保暖 | 适合跑步爬山

【五点描述】
• 高弹透气面料，长时间佩戴也舒适
• 缓震支撑设计，减轻膝盖压力
• 适合跑步、爬山、骑行等多种运动场景
• 男女通用，多尺寸可选
• 专注膝盖护理，让运动更轻松
```
