# PPT设计流程

## 简介

交互式PPT设计全流程管控，基于create-ppt技能，增加"需求确认→风格预览→正式生成"三阶段，确保交付即满意。 1. **先问后做，先看后全做。** 不允许接到任务直接开干，必须经过需求确认和风格预览两个关卡。 2. **文字和图像分离。** AI生图模型是"画"字而非"写"字，中文极易出现乱码、缺笔画、虚构字。解决方案：**图像负责"好看"，HTML负责"正确"**。
