## 三个常用预设（按资本强度）

- **软件/SaaS 型**（如中控 TPT、Palantir）：低 $5M / 中 $50M / 高 $500M
- **半导体/硬科技型**（如 Corning、Cerebras）：低 $50M / 中 $1B / 高 $10B
- **重资产基建型**（如 Oracle OCI、数据中心）：低 $500M / 中 $10B / 高 $50B+

这些只是起点，正式分析时按目标公司实际资本强度调整并标注为假设。
