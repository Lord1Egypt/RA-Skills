# 技术指标使用指南

技术面分析是股票分析的第二个维度（基本面 + 技术面 + 资金面）。

## 核心指标

### 1. 均线系统（MA）

**常用周期：**
- MA5：短期趋势（1 周）
- MA10：中短期趋势（2 周）
- MA20：中期趋势（1 个月）
- MA60：长期趋势（1 季度）

**买卖信号：**
- **金叉**：短期均线上穿长期均线 → 买入信号
- **死叉**：短期均线下穿长期均线 → 卖出信号
- **多头排列**：MA5 > MA10 > MA20 > MA60 → 强势上涨
- **空头排列**：MA5 < MA10 < MA20 < MA60 → 弱势下跌

**实战应用：**
- 股价站上 MA60 → 中期趋势转强
- 回调至 MA20 获得支撑 → 加仓机会
- 跌破 MA60 → 趋势转弱，考虑减仓

### 2. MACD

**组成：**
- DIF线：快线（12日EMA - 26日EMA）
- DEA线：慢线（DIF的9日EMA）
- 柱状图：DIF - DEA

**买卖信号：**
- **金叉**：DIF 上穿 DEA → 买入信号
- **死叉**：DIF 下穿 DEA → 卖出信号
- **底背离**：股价创新低，MACD 不创新低 → 反转信号
- **顶背离**：股价创新高，MACD 不创新高 → 反转信号

**实战应用：**
- MACD 在 0 轴上方 → 多头市场
- MACD 在 0 轴下方 → 空头市场
- 柱状图由负转正 → 买入信号

### 3. RSI（相对强弱指标）

**参数：**
- RSI6：超短期（6日）
- RSI12：短期（12日）
- RSI24：中期（24日）

**买卖信号：**
- **RSI > 70**：超买区，考虑卖出
- **RSI < 30**：超卖区，考虑买入
- **RSI 背离**：股价创新高/新低，RSI 不创新高/新低 → 反转信号

**实战应用：**
- RSI 跌破 30 后回升 → 买入信号
- RSI 突破 70 后回落 → 卖出信号
- RSI 在 50 附近横盘 → 观望

### 4. 成交量（Volume）

**量价关系：**
- **放量上涨**：成交量放大 + 股价上涨 → 强势
- **缩量上涨**：成交量缩小 + 股价上涨 → 弱势（可能见顶）
- **放量下跌**：成交量放大 + 股价下跌 → 恐慌抛售
- **缩量下跌**：成交量缩小 + 股价下跌 → 惜售（可能见底）

**实战应用：**
- 突破关键阻力位时，必须放量确认
- 上涨过程中，成交量逐步萎缩 → 警惕见顶
- 下跌过程中，成交量逐步萎缩 → 可能见底

### 5. KDJ

**组成：**
- K线：快线（9日RSV的3日EMA）
- D线：慢线（K线的3日EMA）
- J线：3 × K - 2 × D

**买卖信号：**
- **金叉**：K 上穿 D → 买入信号
- **死叉**：K 下穿 D → 卖出信号
- **超买区**：K、D > 80 → 考虑卖出
- **超卖区**：K、D < 20 → 考虑买入

**实战应用：**
- KDJ 在低位金叉 → 短线买入信号
- KDJ 在高位死叉 → 短线卖出信号
- J 值 > 100 → 严重超买
- J 值 < 0 → 严重超卖

## 综合应用

### 多头确认（买入信号）

**三大条件：**
1. **均线多头排列**：MA5 > MA10 > MA20 > MA60
2. **MACD 金叉**：DIF 上穿 DEA，且在 0 轴上方
3. **放量突破**：成交量放大，突破关键阻力位

**案例：**
- 茅台 2019 年 1 月：均线多头排列 + MACD 金叉 + 放量突破 MA60 → 买入信号
- 结果：2019-2021 年涨幅 300%+

### 空头确认（卖出信号）

**三大条件：**
1. **均线空头排列**：MA5 < MA10 < MA20 < MA60
2. **MACD 死叉**：DIF 下穿 DEA，且在 0 轴下方
3. **放量下跌**：成交量放大，跌破关键支撑位

**案例：**
- 平安 2021 年 3 月：均线空头排列 + MACD 死叉 + 放量跌破 MA60 → 卖出信号
- 结果：2021-2023 年跌幅 50%+

### 震荡市（观望）

**特征：**
- 均线纠缠（MA5/10/20 交织）
- MACD 在 0 轴附近横盘
- 成交量萎缩

**策略：**
- 短线交易者：高抛低吸（RSI > 70 卖，RSI < 30 买）
- 中长线投资者：观望，等待方向明确

## 注意事项

### 1. 指标滞后性

- 均线是滞后指标（基于历史价格）
- MACD 也是滞后指标
- **不要盲目追涨杀跌**

### 2. 指标钝化

- 强势上涨中，RSI 可能长期 > 70（超买钝化）
- 强势下跌中，RSI 可能长期 < 30（超卖钝化）
- **结合趋势判断**

### 3. 假突破

- 突破关键阻力位，但成交量未放大 → 假突破
- 跌破关键支撑位，但成交量未放大 → 假跌破
- **必须放量确认**

### 4. 多周期共振

- **日线 + 周线 + 月线** 同向 → 强信号
- 日线看涨，但周线看跌 → 谨慎（可能只是反弹）
- 日线看跌，但月线看涨 → 可能是回调（逢低买入）

### 技术面分析模板

```markdown
## 技术面分析

### 趋势分析
- 均线系统：MA5=1800, MA10=1750, MA20=1700, MA60=1650
- 当前价格：1820 元
- 趋势判断：多头排列，中期趋势向上 ✅

### 动量分析
- MACD：DIF=12.5, DEA=10.3, 柱状图=+2.2
- 信号：金叉，且在 0 轴上方 ✅
- RSI：RSI6=65, RSI12=58, RSI24=52
- 信号：未超买，仍有上涨空间 ✅

### 量价分析
- 成交量：5日均量 200 万手，今日 280 万手
- 量价关系：放量上涨 ✅
- 资金流向：主力资金净流入 5 亿元 ✅

### 综合判断
技术面：**看涨** 📈
关键支撑位：1800 元（MA20）
关键阻力位：1900 元（前高）
```

## 总结

技术面分析的核心：

1. **趋势为王**：均线系统判断趋势方向
2. **动量确认**：MACD、RSI 确认趋势强度
3. **量价配合**：成交量验证价格突破
4. **多周期共振**：日线、周线、月线同向才强

**适用场景：**
- ✅ 短线交易（1-5 天）
- ✅ 波段交易（1-4 周）
- ⚠️ 中长线投资（需结合基本面）

**局限：**
- ❌ 无法判断公司价值（需结合基本面）
- ❌ 假突破、假跌破常见（需结合资金面）
- ❌ 指标滞后（需结合实时盘面）
