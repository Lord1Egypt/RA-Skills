# 跨境出口认证与清关查询（Logistics Compliance Check）

> 货被扣了、被退回了、平台下架了——90% 都是合规问题。输入产品 + 目的国，一分钟看清认证/标签/包装/测试全套要求。

## 核心能力

输入：产品类型 + 目的地国家 → 输出：五大类合规要求 + 海关扣货应对清单

```
"蓝牙耳机出口德国被海关扣了，说要 WEEE 注册号"
→ 输出：WEEE 是什么 / 怎么注册 / 多少钱 / 多久能拿到
→ 同时列出：德国还需要 CE + RoHS + 包装法 VerpackG
```

## 8 大目标市场

| 代码 | 国家/地区 | 核心认证体系 | 特别注意事项 |
|------|----------|------------|------------|
| `US` | 🇺🇸 美国 | FCC / FDA / UL / CPSC / FTC | 儿童产品强制 CPSC + CPC |
| `GB` | 🇬🇧 英国 | CE(过渡) / UKCA / BSI | UKCA 标志自 2023 年强制 |
| `DE` | 🇩🇪 德国 | CE / TÜV / GS / REACH | **WEEE + 包装法强制注册** |
| `EE` | 🇪🇪 爱沙尼亚 | CE / EU 统一标准 | REACH + WEEE + EPR 包装法 |
| `SG` | 🇸🇬 新加坡 | PSB / IMDA / SAFETY Mark | 33 类管制商品需认证 |
| `JP` | 🇯🇵 日本 | PSE / PSC / JIS / JET / MIC | 电气产品强制 PSE |
| `AU` | 🇦🇺 澳大利亚 | SAA / RCM / AS/NZS | 电气产品强制 RCM |
| `HK` | 🇭🇰 香港 | 一般免认证 | 大部分商品免税，四类除外 |

## 边界说明

### ✅ 能做的
- 8 国认证/标签/包装/测试/限制五大类要求速查
- 强制/条件/自愿性区分
- 海关扣货应急处理指南
- 多国横向对比
- 关税概览和自由贸易协定提示

### ⚠️ 不能替代的
- **不构成清关建议**：实际清关以当地海关判定为准
- **关税需 HS 编码**：精确关税需用户提供 HS Code 查询
- **法规会更新**：标注最后验证日期，超过 12 个月需核实
- **不覆盖特殊品类**：药品/医疗器械/危险品需专业报关

---

## 反模式

| ❌ 错误用法 | ✅ 正确用法 |
|------------|-----------|
| 认为所有 EU 国家要求一样 | 德国额外要求 WEEE+包装法，法国有 Triman 标签，意大利有 RAEE |
| 有 CE 就够了 | CE 是基础，还需 WEEE 注册号、包装法注册、REACH 合规声明 |
| 亚马逊 listing 上没要求就不用 | 平台不要求 ≠ 海关不要求。海关按法规执行 |
| 样品/小批量不需要合规 | 海关不分批量大小，法规一视同仁 |

### 示例1：认证查询
**用户输入**：「蓝牙耳机出口德国被海关扣了，说要 WEEE 注册号」

**Skill 输出**：
```
## 🚨 德国海关扣货应对

### 常见扣货原因
| 原因 | 占比 | 解决周期 |
|------|:---:|:---:|
| 缺少认证文件 | 45% | 1-4 周 |
| 标签不合规 | 25% | 3-7 天 |
| HS 编码错误 | 15% | 1-3 天 |

### 你该做什么
1. **确认扣货原因** — 物流商/海关通知单上会写 WEEE 注册号缺失
2. **注册 WEEE** — 申请 EAR 编号，周期 2-4 周，费用约 €50-200
3. **补办 CE 认证** — 电子产品必须，联系 TÜV 或 BSI
4. **注册包装法** — 德国 VerpackG，必须在 LUCID 系统注册

### ⚠️ 紧急处理
- 食品/药品/化妆品被扣 → 不要尝试自己清关，找 FDA 注册代理
- 货值 < $800 → 美国 de minimis 已取消（2025年8月起），不再适用

## 📋 电子产品出口德国全套要求

| 认证 | 是否强制 | 说明 |
|------|:---:|------|
| CE | 🟢 强制 | 产品安全基础要求 |
| WEEE | 🟢 强制 | 电子废弃物回收，电商平台 listing 必须显示注册号 |
| EMC 测试 | 🟢 强制 | 电磁兼容性 |
| LVD 测试 | 🟢 强制 | 低电压安全 |
| REACH | 🟢 强制 | 化学物质注册 |
| 包装法 VerpackG | 🟢 强制 | 必须在 LUCID 系统注册 |
```

### 示例2：多国对比
**用户输入**：「玩具出口日本和美国，两个国家要求有什么不同？」

**Skill 输出**：
```
## 🧸 玩具出口合规对比：日本 vs 美国

| 维度 | 🇺🇸 美国 | 🇯🇵 日本 |
|------|:---:|:---:|
| 核心认证 | CPSC + CPC + ASTM F963 | PSE（自愿/菱形）+ 玩具安全标准 |
| 测试要求 | 铅/邻苯二甲酸盐/小零件 | JIS 或 ASTM F963 等效 |
| 标签 | 年龄警告 + 追踪标签 | 日语标签 + 原产地 |
| 注册 | CPSC 强制测试 | 无强制注册，但 PSE 菱形增强信任 |
| 电商平台 | Amazon CPC 必须 | Amazon JP 推荐 PSE |
| 关税 | HTS 分类，约 0-12% | 约 0-5%（协定税率） |

### 🎯 建议
- **同时进入**：分别满足 CPSC 和 PSE 要求，测试报告可互认部分
- **成本考量**：美国测试更严格（CPSC 强制），日本可先不做 PSE 菱形
- **平台策略**：Amazon JP 玩具类目建议获取 PSE 标志以增强转化
```
