# 网织红细胞生成指数（RPI）计算与临床解读

## 何时使用
- 患者确诊为贫血（血细胞比容 HCT 低于正常参考值）
- 需评估骨髓红系造血活性或代偿能力
- 鉴别溶血性贫血、失血性贫血与骨髓造血功能低下

## 执行步骤
1. **收集必要数据**：获取患者的网织红细胞百分比（Ret%）、血细胞比容（HCT）和性别。
2. **确定网织红细胞成熟所需天数**（依据 Finch 方法）：
   - HCT ≥ 0.35 → 1.0 天
   - 0.25 ≤ HCT < 0.35 → 1.5 天
   - 0.15 < HCT < 0.25 → 2.0 天
   - HCT ≤ 0.15 → 2.5 天
3. **设定健康人 HCT 参考值**：男性为 0.45，女性为 0.40。
4. **代入公式计算 RPI**：
   ```
   RPI = (Ret% / 成熟所需天数) × (患者 HCT / 健康人 HCT)
   ```
5. **解读结果**：
   - RPI > 2：骨髓造血代偿有效，红系增生活跃
   - RPI > 3：强烈提示溶血性贫血或急性失血性贫血
   - RPI < 2：提示骨髓增生低下或红细胞成熟障碍（如再生障碍性贫血）

> 注意：本技能仅适用于贫血患者；非贫血状态下不应使用 RPI 解释网织红细胞水平。
