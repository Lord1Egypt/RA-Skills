# 外阴硬化性苔藓活检指征判断

## 何时使用
- 患者临床表现不典型或诊断不确定；
- 存在可疑恶性征象（如溃疡、硬结、快速进展）；
- 青春期前患者出现非典型病损需谨慎评估；
- 典型VLS但治疗无效或病情恶化。

## 执行步骤
1. **评估是否需要活检**：
   - 若具备典型VLS体征（对称性白色萎缩性病变、瘙痒、烧灼感）且无恶性风险，则**不常规活检**，尤其青春期前患者应避免。

2. **必须活检的指征**（满足任一即执行）：
   - 持续性溃疡、糜烂或瘤样隆起；
   - 规范药物治疗无效或进行性恶化；
   - 合并子宫颈或阴道高级别鳞状上皮内病变（HSIL）。

3. **活检操作要点**：
   - 取材部位选择色素减退区、粗糙、硬结、皲裂或溃疡处；
   - 注意：早期或治疗后病变可能缺乏典型病理特征，非特异性结果不能排除VLS。

4. **特殊人群处理**：
   - 青春期前无症状或仅轻微花斑样改变者可观察；
   - 锁孔状病变或伴排尿不适者需个体化评估。
