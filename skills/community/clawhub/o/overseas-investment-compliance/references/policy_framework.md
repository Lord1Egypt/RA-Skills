## 关键法规要点速记

### 国令第837号核心条款（2026年7月1日施行）

| 条款 | 内容要点 |
|------|----------|
| **第一条** | 立法目的：推进高水平对外开放，促进对外投资高质量发展 |
| **第二条** | 适用范围：**投资者包括企业、其他组织和居民个人** |
| **第十条** | 管理原则：分类分级实施全过程监管 |
| **第十一条** | 政策导向：明确鼓励、限制、禁止的对外投资方向 |
| **第十二条** | 备案义务：核准备案、信息报告、跨境资金登记 |
| **第十三条** | 技术出口管制：不得出口禁止/限制技术 |
| **第十五条** | 安全审查：健全境外投资安全审查制度 |
| **第二十七条** | 处罚：未备案处投资额1‰-5‰罚款 |
| **第三十二条** | 港澳台：参照执行，另有规定从其规定 |

### 37号文核心要点

| 问题 | 答案 |
|------|------|
| 什么情况要办？ | 境内居民以投融资为目的设立SPV |
| 怎么办？ | 向户籍所在地或注册地外汇局申请 |
| 什么时候办？ | SPV设立后30日内 |
| 不办后果？ | 资金无法合规汇出/汇回，最高罚款涉案金额30% |
