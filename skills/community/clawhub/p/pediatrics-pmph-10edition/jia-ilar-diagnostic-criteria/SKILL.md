# 幼年型特发性关节炎（JIA）的ILAR诊断标准

## 何时使用
- 患者为16岁以下儿童；
- 存在一个或多个关节的肿胀或疼痛，持续时间≥6周；
- 经初步评估无法明确病因（即“特发性”）；
- 需排除感染、肿瘤及其他结缔组织病等鉴别诊断。

## 执行步骤

### 1. 确认基础诊断条件
- **年龄**：必须 <16 岁；
- **关节症状**：至少一个关节出现肿胀或疼痛；
- **持续时间**：症状持续 ≥6 周；
- **病因不明**：无明确感染、创伤或其他系统性疾病证据。

### 2. 排除其他疾病（必须全部排除）
- 全身感染（如脓毒症、结核、病毒感染）；
- 肿瘤性疾病（如白血病、淋巴瘤）；
- 其他风湿免疫病（如系统性红斑狼疮、川崎病、过敏性紫癜）；
- 化脓性、结核性或创伤性关节炎；
- 非炎症性病因（如脊髓肿瘤、先天性髋关节发育不良）。

### 3. 进行JIA亚型分类
在满足基础条件并完成排除后，根据以下特征确定亚型：
- **受累关节数量**：<5个为少关节型，≥5个为多关节型；
- **全身症状**：如有弛张热、皮疹、肝脾大等，归为全身型；
- **类风湿因子（RF）**：间隔≥3个月两次阳性，定义为RF阳性多关节型；
- **银屑病相关表现**：本人有银屑病或一级亲属有银屑病史；
- **HLA-B27状态及附着点炎**：结合家族史（一级亲属有HLA-B27相关疾病）判断是否为附着点炎相关关节炎。

### 4. 辅助检查参考（用于支持评估，非确诊依据）
- 血沉（ESR）、C反应蛋白、IL-1/IL-6升高提示炎症活动；
- RF、抗CCP抗体、ANA阳性提示特定亚型或预后不良风险；
- 影像学（X线/MRI/超声）用于评估关节结构损害。

> 注意：本技能仅用于临床诊断辅助，最终诊断需由专业医师结合完整病史、体征及实验室结果综合判断。
