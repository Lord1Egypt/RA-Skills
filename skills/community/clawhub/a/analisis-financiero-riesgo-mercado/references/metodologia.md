# Metodología

## Alcance

Esta skill cubre análisis de:

- Inversiones y portafolios.
- Dólar, divisas y USD/CLP.
- Criptomonedas.
- Mercados accionarios, incluyendo Bolsa chilena/IPSA y S&P 500.
- Macro chilena y global.
- Noticias financieras.

## Fuentes

Usar cuando sean relevantes y accesibles:

- `https://mindicador.cl/api` para contexto macro chileno rápido: UF, IVP, dólar observado, euro, IPC, UTM, IMACEC, TPM, cobre, desempleo y bitcoin.
- Banco Central de Chile / SI3 para series oficiales. Código clave USD/CLP observado: `F073.TCO.PRE.Z.D`.
- SII para dólar observado tributario.
- Financial Times, Wall Street Journal, Bloomberg, Diario Financiero y El Mercurio Inversiones para noticias financieras, sin saltar paywalls.
- Bolsa chilena/IPSA y S&P 500 para contexto de mercado local/global.
- Eventos políticos y naturales relevantes: elecciones, reformas, presupuesto/fiscal, decisiones de bancos centrales, tensiones geopolíticas, terremotos, incendios, inundaciones, sequías u otros shocks.

## Señales de Inversión

No declarar “mejor momento” como certeza. Usar lenguaje de escenarios:

- Comprar parcial: condiciones atractivas pero con riesgo vivo.
- Esperar: precio medio/alto, incertidumbre o momentum desfavorable.
- Alerta riesgo: evento o contexto puede deteriorar significativamente el setup.

Para USD/CLP, incluir siempre el valor observado del día o último disponible si se emite señal.
