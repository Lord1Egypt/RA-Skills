## [0.1.0] - 2026-05-24
Initial release. Drafts an ICH E2D / E2B(R3) / GVP Module VI–aligned Individual Case Safety Report (ICSR) narrative for a single adverse-event case — confirms the four ICH valid-case minimum criteria, computes Day 0 and the Day 7 / Day 15 / non-expedited reportability clock, performs PHI-safe intake, captures suspect and concomitant products with batch / dose / route / action-taken, builds the event table with MedDRA LLT / PT proposals, ICH seriousness criteria, dechallenge / rechallenge, expectedness vs Reference Safety Information, and separate reporter and company causality, drafts the chronological Introduction → Body → Conclusion narrative, runs an ICH E2D quality self-check, and emits a follow-up question list for safety-physician / QPPV review before transmission.
