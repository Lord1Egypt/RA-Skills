# 坦洛新对α₁A受体的选择性及其在良性前列腺增生中的应用

## 何时使用
- 患者为男性，确诊由良性前列腺增生（BPH）引起的下尿路症状（如排尿困难、尿流变细、尿不尽感等）；
- 患者无Q-T间期延长或严重肝肾功能障碍；
- 治疗目标是改善排尿症状，而非控制高血压或治疗血管痉挛性疾病；
- 优先考虑心血管安全性，尤其适用于老年患者。

## 如何执行
1. **确认α₁受体亚型分布差异**：
   - α₁A受体主要位于前列腺、膀胱颈和尿道的平滑肌，调控排尿阻力；
   - α₁B受体主要分布于血管平滑肌，参与血压调节。

2. **评估坦洛新的选择性机制**：
   - 坦洛新对α₁A受体的亲和力显著高于α₁B受体，实现选择性阻断；
   - 因此可有效松弛前列腺和膀胱底部平滑肌，改善尿流率和排尿症状。

3. **对比其他α受体阻断药**：
   - 酚苄明（非选择性）：显著降低血压，易致心悸；
   - 哌唑嗪（α₁选择性但无亚型区分）：可能引起直立性低血压；
   - 坦洛新：对血管α₁B受体影响极小，心血管副作用少。

4. **推荐用法**：
   - 剂量：0.2–0.4 mg，每日一次，口服；
   - 药代动力学：生物利用度高，半衰期9–15小时，支持每日一次给药。

5. **注意事项**：
   - 虽心血管副作用罕见，仍需警惕头晕或逆行射精；
   - 避免用于严重肝功能不全患者（因经肝代谢）；
   - 若出现明显低血压，应重新评估是否误用其他非选择性α阻断药。

## 预期结果
使用坦洛新后，预期国际前列腺症状评分（IPSS）下降，排尿症状改善，同时血压和心率保持稳定。
