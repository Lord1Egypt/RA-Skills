# 银行活动政策分析规则

## 评价标准

分析银行活动新旧政策变化时，按以下4条硬性标准判定：

1. **补贴费率变高** → ✅变好
2. **新客/存客补贴额度降低** → ⚠️变差
3. **存客存款门槛变高** → ⚠️变差
4. **新客期免考核优于考核存款**
   - 免考核变考核 → ⚠️变差
   - 考核变免考核 → ✅变好
   - 同时有变好和变差 → ⚖️中性

## 免考核期判定规则

- **有考核金额 = 无免考核期**：如"前2个活动月需存入活期资金X元并保持资金留存"，属于考核存款，不算免考核期
- **有免考核期**：只有明确写明"无存款考核"、"免考核"、"无考核要求"字样的才算
- **简单说**：要求存钱=有考核，不要求存钱=无考核=免考核

## 政策变动整理规则

### 整体排列顺序

按 **政策调整 → 新增活动 → 活动暂停/到期** 排列。

### 政策调整维度（逐条输出）

每条只反馈有变化的部分，**无变化的不体现**：

1. **减免费率是否变化** — ⚠️变差 / ✅变好
2. **免考核期是否变化** — 有变化时说明调整了哪些，标注⚠️变差/✅变好
3. **存客期是否变化** — 有变化时只反馈两件事：
   - 考核存款门槛：⚠️提高 / ✅降低 / — 不变
   - 最高补贴额度：✅提高 / ⚠️降低 / — 不变

### 新增活动维度

每条展示：政策名称、发生时间、活动城市、关键条款（新客期是否考核存款及补贴额度、存客期考核存款起步门槛及最高补贴）。

### 活动暂停/到期维度

每条只展示：政策名称、发生时间。
