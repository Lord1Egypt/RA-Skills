# Eva-skill 交接卡字段真源

本文件是标题交接卡、第一句话交接卡和正文路线图最低字段的唯一字段真源。

商单约束卡、商单内容任务卡和二者转换规则的唯一真源是 `04_commercial-constraint-card_商单约束卡真源.md`。

使用方式：

- 生成、校验或补齐标题交接卡、第一句话交接卡和正文路线图最低字段时读取本文件。
- 子模块只保留本模块的进入条件和退回规则，不重复整张字段表。
- 缺硬字段时退回对应上游；不得由下游猜字段。

## 标题交接卡

输出标题交接卡时，标题承诺、验证线索和内容兑现必须成立；或用户已经明确授权低置信度试结构。

```markdown
## 交给 /eva-script 的内容任务

- 标题谜面：
- 标题来源：一字不改使用 / 结构迁移 / 已验证候选标题 / 低置信度试结构
- 标题验证状态：已验证 / 未验证-用户授权试结构
- 人群状态：已由 audience-finder 校准 / 标题模块临时假设 / 用户直接给出 / 不清楚
- 讲给谁：
- 具体处境：
- 真实缺口：
- 主谜面：
- 正文标题作用：共同谜面 / 点击增强 / 补充情绪 / 补充结果 / 不进入正文主线
- 用户最想知道：
- 必须先回答：
- 本稿不回答：
- 建议写法：观点类 / 干货类 / 观点开路 + 干货收束
- 点赞点：
- 收藏点：
- 表达资格点：
- 可用素材：
- 可回捞素材：普通点子卡 / 人设素材卡 / 暂无
- 可搜索共鸣词：
- 成稿置信度上限：高 / 中 / 低
```

低置信度标题交接卡必须同时满足：

- 标题来源写“低置信度试结构”。
- 标题验证状态写“未验证-用户授权试结构”。
- 成稿置信度上限只能写“低”。
- 缺素材、经历、数据或评论区原话的位置必须写占位符。

人群状态必须如实标注：

- 只有本轮执行过 `03_eva-audience-finder`，才写“已由 audience-finder 校准”。
- 标题模块临时判断，只写“标题模块临时假设”。
- 用户只给出对象但未拆具体处境和真实缺口，写“用户直接给出”。
- 判断不出来，写“不清楚”。

## 第一句话交接卡

无标题完整视频稿链路不补标题交接卡，先形成第一句话交接卡。

```markdown
- 第一句话：
- 停留理由：
- 人群状态：已由 audience-finder 校准 / 第一句话临时假设 / 用户直接给出 / 不清楚
- 讲给谁：
- 具体处境：
- 真实缺口：
- 主谜面：
- 用户继续看最想知道：
- 必须先回答：
- 本稿不回答：
- 建议展开方式：完整内容 / 先补主体素材
- 可用素材：
```

第一句话不能独立建立话题、停留理由和兑现线索时，不进入完整正文；停在开头分支继续补。

## 商单内容任务卡字段说明

商单内容任务卡只作为商单正文链路的约束字段，不是标题交接卡、第一句话交接卡或正文路线图最低字段。

输出、校验或转换商单内容任务卡时，读取 `04_commercial-constraint-card_商单约束卡真源.md`，不得在本文件里维护第二套商单字段。
