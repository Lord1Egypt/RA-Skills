## Description: <br>
Shows choice dialogs for recommendation, selection, and confirmation requests so users can pick or confirm the next step. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[TechhnicistSmallWhite](https://clawhub.ai/user/TechhnicistSmallWhite) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Agent users use this skill during recommendation, selection, and confirmation conversations to present immediate choice dialogs, offer candidate options, and continue with next-step choices after a selection. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad recommendation and confirmation triggers can interrupt ordinary chat flow with choice dialogs. <br>
Mitigation: Install only where choice-first recommendation and confirmation behavior is desired, and review the dialog before acting on a selection. <br>
Risk: Immediate choice dialogs may skip explanatory analysis when a user expected a written comparison. <br>
Mitigation: Ask for details or a comparison before choosing when more context is needed. <br>
