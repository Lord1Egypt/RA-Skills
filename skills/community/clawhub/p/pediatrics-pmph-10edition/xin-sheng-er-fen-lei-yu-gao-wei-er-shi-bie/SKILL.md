# 新生儿分类与高危儿识别

## 适用对象
- 新生儿（出生后0–28天）
- 早产儿、足月儿、高危儿筛查

## 分类维度

### 1. 按胎龄（GA）
- 足月儿：37–42周
- 早产儿：<37周（细分：超早产<28周、极早产28–32周等）
- 过期产儿：≥42周

### 2. 按出生体重（BW）
- 正常：2500–4000g
- 低出生体重（LBW）：<2500g（VLBW<1500g, ELBW<1000g）
- 巨大儿：>4000g

### 3. 按体重-胎龄关系
- 适于胎龄儿（AGA）：P10–P90
- 小于胎龄儿（SGA）：<P10
- 大于胎龄儿（LGA）：>P90

### 4. 按出生后周龄
- 早期新生儿：0–7天（属围产儿）
- 晚期新生儿：8–28天

## 高危儿识别
- 包括早产儿、SGA、巨大儿、窒息儿等
- 结合母体病史、分娩史综合判断

## 输出结果
- 多个分类标签（如：晚期早产儿、VLBW、SGA）
- 护理级别与预后评估建议
