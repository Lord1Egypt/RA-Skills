# 无痛get十年工作经验

> 新媒体运营、HR、行政专员、市场营销、产品经理 —— 十年老兵的经验沉淀，一次全带走。

## 这是什么？

一个 WorkBuddy Skill，结构化记录了五类常见岗位的十年工作经验全貌。无论你是正在找工作的应届生、想转行的职场人，还是想系统了解某个岗位的在校生，这个 Skill 都能帮你：

- 🔍 **一键查询**任意岗位的工作全貌（职责/流程/对接角色/注意事项）
- 📝 **用STAR法则写简历**——五类岗位各有一套模板
- 🎤 **准备面试**——高频问题清单 + 回答框架 + 模拟面试
- 📚 **补充行业知识**——配合大模型解释术语、完善案例、对比岗位

## 覆盖岗位

| 岗位 | 模块数 | 核心视角 |
|------|--------|----------|
| 新媒体运营 | 8大模块 | 内容→流量→变现全链路 |
| HR（人力资源） | 6大模块 + OD + 数据分析 | 选育用留 + 合规底线 |
| 行政专员 | 8大模块 | 办公环境为中枢的全面支撑体系 |
| 市场营销 | 8大模块 | 品牌×增长双引擎 + 数据归因闭环 |
| 产品经理 | 6大模块 | 用户价值驱动的六阶段迭代 |

## 使用方法

在 WorkBuddy 对话中直接提问即可自动触发，例如：

- "新媒体运营每天都在做什么？"
- "帮我写一份产品经理的简历"
- "HR面试一般问什么问题？"
- "帮我准备市场营销的模拟面试"
- "行政专员和HR的工作有什么区别？"
