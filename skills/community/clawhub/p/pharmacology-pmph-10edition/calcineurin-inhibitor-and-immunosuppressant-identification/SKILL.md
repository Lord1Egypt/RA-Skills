# calcineurin-inhibitor-and-immunosuppressant-identification

## 触发条件
- 需抑制T细胞活化（如器官移植抗排斥、自身免疫病治疗）；或
- 需确认某药物是否为经典免疫抑制剂。

## 执行步骤
### A. 免疫抑制剂识别
1. **检查药物名称**：若为以下之一，即为经典免疫抑制剂：
   - 吗替麦考酚酯、硫唑嘌呤、来氟米特、雷帕霉素（西罗莫司）、他克莫司。
2. **机制归类**：
   - 吗替麦考酚酯：抑制IMP脱氢酶 → 阻断T/B细胞增殖；
   - 硫唑嘌呤：嘌呤类似物 → 干扰DNA合成；
   - 来氟米特：抑制DHODH → 阻断嘧啶合成；
   - 雷帕霉素/西罗莫司：抑制mTOR通路；
   - 他克莫司：抑制钙调磷酸酶。

### B. 钙调磷酸酶抑制药使用
1. **作用机制**：
   - 环孢素 + 亲环蛋白 → 抑制钙调磷酸酶；
   - 他克莫司 + FKBP → 抑制钙调磷酸酶；
   - 导致NFATc无法去磷酸化入核 → 抑制IL-2等细胞因子转录。
2. **免疫效应**：选择性抑制T细胞活化、增殖及CTL生成。
3. **监测要求**：因治疗窗窄，**必须进行血药浓度监测**。
4. **禁忌**：非T细胞介导的免疫反应无效；肾功能严重受损者慎用。

## 输出结果
- 成功识别免疫抑制剂并提示适用场景；
- 指导钙调磷酸酶抑制药的安全使用与监测。
