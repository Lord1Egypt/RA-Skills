# 期刊格式规范参考

本文档汇总主要地学期刊的格式要求，供论文撰写和投稿参考。

---

## 一、Nature 系列

### Nature (主刊)

| 项目 | 要求 |
|-----|------|
| 标题 | 简洁，≤120 characters |
| 摘要 | 150词，无结构分段 |
| 正文 | 约3000词 |
| 图表 | ≤6 figures + tables |
| 参考文献 | 顺序编号 |

**特色结构：**
- 无独立"引言/方法/结果/讨论"标题
- 逻辑连贯的散文形式
- 审稿周期较短

### Nature Communications

| 项目 | 要求 |
|-----|------|
| 标题 | 简洁，≤90 characters |
| 摘要 | 150词 |
| 正文 | 无严格限制 |
| 图表 | 适量 |

---

## 二、Science

| 项目 | 要求 |
|-----|------|
| 标题 | 简洁，反映核心发现 |
| 摘要 | ~150词，分段 |
| 正文 | 约3000词 |
| 图表 | 通常5-6个 |

**结构特点：**
- 与Nature类似，散文化的完整叙述
- 审稿严格，竞争激烈

---

## 三、PNAS (Proceedings of the National Academy of Sciences)

| 项目 | 要求 |
|-----|------|
| 标题 | 简洁 |
| 摘要 | 250词以内 |
| 正文 | 无严格限制 |
| **Significance** | 必须在投稿时填写，简洁描述研究意义 |

**格式特点：**
- 强制填写"Significance Statement"
- 建议添加"Plain Language Summary"

---

## 四、GRL (Geophysical Research Letters)

| 项目 | 要求 |
|-----|------|
| 标题 | 简洁 |
| 摘要 | ≤1500字符（约200-250词） |
| 正文 | 无严格字数限制，鼓励简洁 |
| **KeyPoints** | 必须，3-5个要点，每个≤85字符 |
| **Plain Language Summary** | 建议添加 |

**格式特点：**
- 快报型期刊，简洁为首
- 图表精炼
- 结论不能过度推断

---

## 五、EPSL (Earth and Planetary Science Letters)

| 项目 | 要求 |
|-----|------|
| 标题 | 描述性标题 |
| 摘要 | 200-300词 |
| 正文 | 标准结构（引言/方法/结果/讨论/结论） |
| Keywords | 5-8个 |
| 参考文献 | 统一格式 |

**格式特点：**
- 完整的研究论文格式
- 方法部分要求详尽
- 讨论深入

---

## 六、GCA (Geochimica et Cosmochimica Acta)

| 项目 | 要求 |
|-----|------|
| 标题 | 描述性，避免缩写 |
| 摘要 | 200-300词 |
| 正文 | 标准五段式 |
| Keywords | 5-8个 |
| 致谢 | 详细的项目和基金说明 |

**格式特点：**
- 地球化学专业期刊
- 注重分析方法和数据质量
- Supporting Information 重要

---

## 七、Icarus

| 项目 | 要求 |
|-----|------|
| 标题 | 描述性 |
| 摘要 | 200-300词 |
| 正文 | 标准结构 |
| Keywords | 5-8个 |
| 图表 | 自明性强 |

**格式特点：**
- 行星科学专业期刊
- 注重观测和实验数据
- 方法描述需详尽

---

## 八、Annual Review 系列

| 项目 | 要求 |
|-----|------|
| 摘要 | 300-500词，可分段 |
| 正文 | 综述性，无固定字数 |
| 结构 | 分节标题，粒度细 |
| 引用 | 广泛，篇幅长 |

**格式特点：**
- 综述类文章
- 适合领域概览和前沿总结
- 需有独到见解

---

## 九、中文期刊格式

### 岩石学报 (Acta Petrologica Sinica)

| 项目 | 要求 |
|-----|------|
| 标题 | 中英文双语 |
| 摘要 | 中英文双语，中文200-300字，英文可稍长 |
| 关键词 | 中英文各5-8个 |
| 作者单位 | 中英文双语 |
| 正文 | 标准五段式 |
| 致谢 | 中英文双语 |

**格式特点：**
- 完全双语化
- 图表需中英文标题
- 基金项目标注清楚

### 地学前缘 (Earth Science Frontiers)

| 项目 | 要求 |
|-----|------|
| 标题 | 中英文 |
| 摘要 | 中英文 |
| 关键词 | 中英文 |
| 其他 | 参照岩石学报格式 |

### 中国科学 (Science China)

| 项目 | 要求 |
|-----|------|
| 标题 | 中英文 |
| 摘要 | 中英文 |
| 关键词 | 中英文 |
| 其他 | 高标准学术规范 |

## 十、投稿检查清单

### 投稿前必查

- [ ] 摘要字数符合期刊要求
- [ ] 标题简洁，反映核心创新
- [ ] Keywords 数量和质量符合
- [ ] 图表分辨率达标
- [ ] 参考文献格式统一
- [ ] 作者信息完整
- [ ] 基金标注完整
- [ ] 英文语法检查
- [ ] 查重（避免自我抄袭）
- [ ] 审稿人推荐合理

### 格式检查

- [ ] 页面边距
- [ ] 行号（如要求）
- [ ] 双倍行距（如要求）
- [ ] 图表位置正确
- [ ] 缩写首次定义

### 伦理检查

- [ ] 所有作者知情同意
- [ ] 无一稿多投
- [ ] 利益冲突声明
- [ ] 数据可用性说明
- [ ] 基金致谢完整
