## 三、P1级漏洞修正方案

### 漏洞2：Mangal Dosha判断标准不统一

**问题描述**：
- 不同占星师对Mangal Dosha的判断标准不同
- 有的只看火星位置，有的综合考虑土星、罗睺、计都、太阳
- 多图表分析（Lagna、Navamsa、Chandra、Venus）增加了复杂性
- Dosha强度量化方法缺乏统一标准

**修正方案**：建立标准化Mangal Dosha判断流程

#### 标准化判断流程（六步法）

**第一步：主图表判断（Lagna Chart）**
```
主图表判断规则：
- 火星在1、2、4、7、8、12宫 → 有Dosha
- 火星在3、5、6、9、10、11宫 → 无Dosha
```

**第二步：强度评估（量化）**
```
Dosha强度量化（总分100分）：
- 火星在1宫：100分（最强）
- 火星在2宫：75分
- 火星在4宫：75分
- 火星在7宫：100分（最强）
- 火星在8宫：100分（最强）
- 火星在12宫：75分
```

**第三步：强度调整因素**
```
调整因素：
1. 火星落陷（Debilitated）→ 强度减半
2. 火星入庙（Exalted）→ 强度加倍
3. 火星受克（Aspected by malefics）→ 强度增加20%
4. 火星有吉相位（Aspected by benefics）→ 强度减少20%
```

**第四步：Dosha取消检查**
```
Dosha取消规则（优先级从高到低）：
1. 火星在白羊座（入庙）→ Dosha取消
2. 火星在摩羯座（耀升）→ Dosha取消
3. 火星在巨蟹座（落陷）→ Dosha部分取消（强度减半）
4. 火星与木星合相 → Dosha取消
5. 火星与月亮合相 → Dosha取消
6. 火星在Kendra宫（1、4、7、10）且有吉相位 → Dosha部分取消
7. 火星在Navamsa图表中入庙或耀升 → Dosha取消
```

**第五步：多图表验证（可选）**
```
多图表验证（用于高精度分析）：
1. Lagna Chart（主图表）
2. Chandra Chart（月亮图表）
3. Venus Chart（金星图表）
4. Navamsa Chart（九分图表）

验证规则：
- 如果在2个及以上图表中都有Dosha → Dosha确认
- 如果只在1个图表中有Dosha → Dosha弱
- 如果在所有图表中都无Dosha → 无Dosha
```

**第六步：最终判断**
```
最终判断输出：
- has_dosha: True/False
- strength: 0-100分
- cancellation: True/False/Partial
- cancellation_reasons: 取消原因列表
- message: 最终判断信息
```

**修正效果预期**：
- 判断标准统一：所有占星师使用相同流程
- 可重复性提高：不同占星师得出相同结论
- 准确率提升：从约60%提升到80%+
