# Wu Xing Pathological Transmission

## When to Use
- 肝病患者出现脾虚症状（如纳差、腹胀）
- 肾精亏虚伴肝血不足（“水不涵木”）
- 肺虚患者突发咳血（疑“木火刑金”）

## Execution Steps
1. **确认五脏五行归属**：确保已明确肝属木、心属火、脾属土、肺属金、肾属水。
2. **判断传变类型**：
   - **相生传变**：
     - 母病及子（顺传）：如肾（水）病致肝（木）虚；
     - 子病及母（逆传）：如肝阳上亢耗伤肾阴（子盗母气）。
   - **相克传变**：
     - 相乘（克太过）：如肝气横逆犯脾（肝木乘土）；
     - 相侮（反克）：如肝旺反侮肺金（木火刑金）。
3. **结合四诊验证**：排除非内伤杂病（如外伤、中毒），确保符合临床实际。
4. **输出预警**：提示潜在受累脏腑及传变路径，支持“见肝之病，知肝传脾，当先实脾”的预防策略。

> **注意**：此为理论模型，必须结合具体证候综合判断，不可机械套用。
