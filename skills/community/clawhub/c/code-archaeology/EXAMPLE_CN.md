# 代码考古学 - 使用示例

## 基本调用
```
分析这个遗留的PHP代码库，查找安全漏洞和技术债务。
```

## 商业智能提取
```
执行深度代码考古，从这个遗留系统中提取业务规则、生成PRD文档并映射业务工作流。
```

## 针对性分析  
```
专门对合同管理模块执行代码考古分析，重点关注业务规则提取和工作流文档化。
```

## 恢复之前的分析
```
从上次停止的地方继续代码考古分析。
```

## 预期输出结构
当代码考古学完成分析后，您将收到涵盖以下内容的综合报告：

### 商业智能交付物
- **业务规则目录**：提取的业务规则及实现细节
- **PRD风格文档**：功能规格、用户故事和验收标准
- **流程图**：端到端业务工作流和状态转换
- **实体关系图**：数据模型和实体关系

### 技术评估报告
- **执行摘要**：关键发现和整体风险评估
- **业务域图**：业务能力和关系的组织视图
- **安全评估**：带严重性评级的详细漏洞分析
- **性能概况**：瓶颈识别和优化机会
- **技术债务清单**：维护挑战和现代化建议
- **行动计划**：带实施指导的优先级后续步骤

分析结果存储在您工作区中的 `.code-archaeology/` 目录中，供将来参考和增量更新使用。
