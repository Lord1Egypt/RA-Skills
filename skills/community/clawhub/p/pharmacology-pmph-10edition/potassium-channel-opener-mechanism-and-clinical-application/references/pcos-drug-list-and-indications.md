# 钾通道开放药（PCOs）详细药物清单与适应症

## 常用药物
| 药物名称 | 主要适应症 | 血脑屏障通透性 | 特殊药理特点 |
|----------|------------|------------------|----------------|
| 米诺地尔 | 重度高血压 | 否 | 强效血管扩张剂，常致多毛症 |
| 尼可地尔 | 心绞痛 | 有限 | 兼具K_ATP开放与NO供体双重机制 |
| 二氮嗪 | 高血压危象、胰岛素瘤 | 否 | 可升高血糖，慎用于糖尿病 |
| 吡那地尔 | 实验性高血压治疗 | 否 | 曾用于欧洲，现少用 |
| 克罗卡林 | 实验性神经保护 | 是 | 在阿尔茨海默病模型中显示神经保护作用 |
| 阿普卡林 | 实验性脑缺血保护 | 是 | 在帕金森病模型中减少多巴胺能神经元损伤 |

## 神经退行性疾病中的研究证据
- **阿尔茨海默病**：PCOs通过减轻线粒体功能障碍和氧化应激，改善认知功能（动物模型）
- **帕金森病**：K_ATP通道开放可抑制兴奋性毒性，保护黑质多巴胺神经元
- **脑缺血/再灌注损伤**：预处理或后处理给予PCOs可缩小梗死体积

## 禁忌与限制
- 心力衰竭患者慎用（可能加重液体潴留）
- K_ATP通道基因缺失或突变者无效
- 正常血压者使用可能仅引起轻微血管扩张，无显著降压效果
