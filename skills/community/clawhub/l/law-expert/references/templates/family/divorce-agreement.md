# 离婚协议书

## 适用场景
双方自愿离婚，就子女抚养、财产分割、债务处理等事项达成一致，需签订书面协议后共同到婚姻登记机关办理离婚登记。

## 提交机构
一方户口所在地的婚姻登记机关（民政局）

## 模板正文

```
                        离婚协议书

男方：姓名______，身份证号：__________________
      户口所在地：__________________

女方：姓名______，身份证号：__________________
      户口所在地：__________________

男女双方于______年______月______日在______民政局
办理结婚登记（结婚证号：______）。因______，
双方自愿离婚，经充分协商达成如下协议：

一、自愿离婚
双方均自愿解除婚姻关系。

二、子女抚养
1. 双方婚后育有___子/女（姓名：______，身份证号：____________）。
2. 子女由___方抚养，随___方共同生活。
3. ___方每月支付抚养费人民币______元，
   支付至子女年满18周岁/完成大学本科教育止。
   支付方式：每月___日前汇入___方指定账户
   （开户行：______，账号：____________）。
4. 未直接抚养子女的___方享有探视权，
   每周/每两周可探视___次，具体时间为______。
5. 子女的额外费用（医疗、教育等大额支出）由双方各承担50%。

三、财产分割
1. 房产：位于___________的房产（产权证号：______）
   归___方所有；___方应于___年___月___日前配合办理过户手续。
2. 车辆：___牌车辆（车牌号：______）归___方所有。
3. 存款及理财：双方各自名下存款归各自所有/详见分割清单。
4. 其他财产：____________________________。

四、债务处理
1. 双方确认婚姻存续期间无共同债务/
   共同债务共______元，由___方承担。
2. 各自名下私人债务由各自承担。

五、其他
1. 本协议一式三份，男女双方各执一份，
   婚姻登记机关存档一份。
2. 本协议自双方在婚姻登记机关办理离婚登记之日起生效。

男方（签名）：                女方（签名）：
日期：______年______月______日
```

## 注意事项
- **关键填写点**：抚养费金额建议为月收入的20%-30%（一个子女）；房产过户期限要具体明确；大额开支分摊条款避免日后纠纷；冷静期内协议可反悔
- **常见错误**：使用"孩子归谁所有"等物化表述；抚养费只写到18岁未考虑大学费用；财产分割过于笼统（如"各自名下财产归各自所有"可能遗漏夫妻共同财产）；未约定探视具体时间和方式导致无法执行
- **份数要求**：夫妻各1份 + 婚姻登记机关存档1份，共3份
- **离婚冷静期**：提交申请后30天冷静期，双方均未撤回方可领证
- **法律效力**：协议经婚姻登记机关备案后具有法律效力，一方不履行可诉请执行
