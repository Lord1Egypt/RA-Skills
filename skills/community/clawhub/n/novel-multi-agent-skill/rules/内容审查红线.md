# 🔴 内容审查红线规则

## ⚠️ 生效说明

本规则是**最高优先级**的铁律，所有智能体必须严格遵守。
任何违反以下规则的内容，直接打回重写，零容忍！

---

## 🚫 红线1：绝对不能使用真实地名

### 禁止内容
- 真实的省、市、区、县名称
- 真实的学校、单位、企业名称
- 真实的街道、地标名称

### 替代方案
- 江苏省 → 江南省
- 上海市 → 沪城
- 北京市 → 京城
- 无锡市 → 锡城
- 昆山市 → 昆阳县
- 具体学校/单位 → 使用化名，如「江南大学」「市第一中学」

### 例外
- 「苏南地区」「长三角」等大范围区域名称可以保留
- 不涉及具体县级市的泛称可以保留

---

## 🚫 红线2：绝对不能使用真实姓名

### 禁止内容
- 国家领导人姓名
- 公众人物、名人、明星姓名
- 企业家、富豪姓名
- 历史敏感人物姓名

### 替代方案
- 马云 → 马老师 / 阿里创始人 / 某互联网大佬
- 任正非 → 任总 / 华为创始人 / 某通讯巨头掌门人
- 知名人物 → 用代称或职务指代

### 企业名称处理
- 腾讯 → 某讯 / 企鹅公司
- 阿里 → 某里 / 电商巨头
- 华为 → 某为 / 通讯巨头

---

## 🚫 红线3：绝对不能涉及敏感政治内容

### 禁止内容
- 敏感历史事件
- 政治敏感话题
- 负面社会事件（如群体性事件、重大事故等）
- 涉及国家领导人的任何调侃或负面描写
- 涉及政党、军队、政府的负面内容

### 处理原则
- 只写经济发展、时代变迁、百姓生活
- 不碰任何政治敏感内容
- 历史事件只写经济相关的，敏感事件一律回避或模糊处理

---

## 🚫 红线4：绝对不能有低俗色情内容

### 禁止内容
- 直接的色情描写
- 低俗的性暗示
- 侮辱女性的内容
- 违背公序良俗的内容

---

## 🚫 红线5：绝对不能有暴力血腥内容

### 禁止内容
- 过于血腥的暴力描写
- 详细的犯罪过程描写
- 可能引起读者不适的恐怖内容

---

## ✅ 检查流程

### 创作时自检
写作智能体在创作完成后，必须先自行检查：
1. 搜索是否有真实地名
2. 搜索是否有真实人名
3. 检查是否有敏感政治内容
4. 检查是否有色情低俗内容
5. 检查是否有暴力血腥内容

### 评审时必查
质量评审官在评审时，红线审查是第一项必须完成的工作：
- 发现红线问题 → 直接打回，不评分
- 修改完成后 → 重新评审

---

## ⚠️ 违规处理

1. **第一次违规**：警告，打回重写
2. **第二次违规**：严重警告，该智能体当次任务记失败
3. **第三次违规**：暂停该智能体的任务权限，重新进行规则培训

---

## 📝 检查清单

创作完成后，请逐项打勾确认：

- [ ] 没有真实的省、市、区、县名称
- [ ] 没有真实的学校、单位、企业名称
- [ ] 没有国家领导人、名人、企业家的真实姓名
- [ ] 没有敏感政治事件和话题
- [ ] 没有低俗色情内容
- [ ] 没有过于暴力血腥的内容
- [ ] 所有地名、人名都已使用化名或代称

---
