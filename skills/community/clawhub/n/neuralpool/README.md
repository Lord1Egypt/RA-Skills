# NeuralPool Node

Monetize idle LLM API capacity or access 100+ models through a unified OpenAI-compatible API.

## What is NeuralPool?

NeuralPool is a peer-to-peer LLM API marketplace — like "Airbnb for LLM API access." Two ways to participate:

**Node Operators** — Got idle cloud credits or LLM API access? Install the Node Agent, set your prices, and earn per token forwarded. No GPU required — your machine only proxies API calls. Your keys never leave your machine. Earnings settle after T+1 or T+30 clearing (varies by funding source), then withdraw to Solana.

**API Consumers** — Need affordable LLM access? One OpenAI-compatible API key gives you access to all models from all providers on the network. Pay per token, no commitments.

## Install

```bash
curl -fsSL https://neuralpool.ai/install.sh | bash
```

Or download from [GitHub Releases](https://github.com/neuralpool/neuralpool-node/releases) for Linux (amd64/arm64), macOS (amd64/arm64), Windows (amd64/arm64).

## Quick Start

```bash
npnode setup    # Interactive wizard: auth token, providers, pricing
npnode start    # Connect to NeuralPool and start earning
```

## Supported Providers

Major LLM providers with pre-configured endpoints. Just bring your API key.
