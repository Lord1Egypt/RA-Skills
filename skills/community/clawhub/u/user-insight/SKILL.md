# User Insight

> 通过对话逐渐了解用户，构建动态用户画像以优化沟通。支持被动收集（从对话中提取兴趣、偏好、习惯）和主动探索（像"刷短视频"一样尝试不同话题，发现用户潜在兴趣点）。在需要个性化回应、寻找聊天话题或了解用户背景时读取 ~/.openclaw/workspace/memory/user-profile.json。
