# 举证责任分配

## 通用原则
- 劳动者：证明自己权利被侵害（先举证）
- 用人单位：证明自己的决定合法（举证倒置）

## 劳动者必举的证据

### 第一层：证明劳动关系存在
| 证据类型 | 获取方式 | 优先级 |
|---|---|---|
| 劳动合同 | 签字版本（手里有就交） | ⭐⭐⭐ |
| 工资银行流水 | 银行APP导出（证明实际履行） | ⭐⭐⭐ |
| 社保缴费记录 | 支付宝/微信→城市服务→社保查询 | ⭐⭐⭐ |
| 个税记录 | 个人所得税APP→收入汇总 | ⭐⭐ |
| 微信/邮件工作往来 | 截图+录屏公证 | ⭐⭐ |
| 工牌/门禁卡 | 实物证据+照片 | ⭐ |

> **原则**：劳动合同+工资流水+社保记录，三选二就能证明劳动关系。

### 第二层：证明权利被侵害
根据你要告的内容准备对应证据（见下方证据清单）

## 用人单位必须举的证据（举证倒置）
- 开除/除名/辞退/解除劳动合同的决定
- 减少劳动报酬的决定
- 计算工作年限的决定

## 加班费举证特别规则
1. 劳动者先证明加班事实存在（考勤记录/加班申请/工作记录）
2. 如果能证明公司掌握加班证据 → 举证责任转移到公司
3. 公司不提供 → 承担不利后果

## 证据清单

| 争议类型 | 劳动者准备什么 | 用人单位准备什么 |
|---|---|---|
| 违法解除 | 劳动合同/工资流水/社保记录 + 解除通知 | 解除依据的事实和法律理由 |
| 拖欠工资 | 劳动合同/工资约定/银行流水 | 工资支付凭证/考勤记录 |
| 加班费 | 考勤记录/加班申请/邮件/聊天记录 | 考勤记录/加班审批表（公司掌握） |
| 未缴社保 | 劳动合同/工资流水/社保查询记录 | 社保缴费记录/申报材料 |
| 年假工资 | 劳动合同/工龄证明/请假记录 | 已安排年假的证明 |

## 执行步骤

1. **列清单** — 先确定你要告什么（解除/工资/加班费/社保…），查上面的证据清单
2. **收集自己证据** — 准备属于你自己掌握的证据（劳动合同、工资流水、邮件、聊天记录等）
3. **要求公司提供** — 向公司发函要求提供你无法自行获取的证据（如考勤记录）
4. **公司拒绝则触发举证倒置** — 公司无正当理由不提供，由公司承担不利后果

## 证据优先级

**证明力从高到低：**
1. 书面文件（劳动合同/书面通知/盖章文件）
2. 电子数据（邮件/聊天记录/系统截图）
3. 录音录像
4. 证人证言

> 原则：书面 > 电子 > 证人。证据越正式，证明力越强。
