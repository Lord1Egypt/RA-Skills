# 技能掌握度评分 (score)

## 概述

展示牛仔的综合技能掌握度评分，帮助商家了解牛仔当前的能力水平和提升方向。

## 入参

无（直接执行即可）

## TPP 接口

`POST /api/CowboyScore/1.0.0`

请求体：
```json
{}
```

返回字段：
```json
{
  "success": true,
  "data": {
    "totalScore": 72,
    "dimensions": [
      {
        "name": "知识库完善度",
        "score": 65,
        "description": "知识库覆盖了65%的常见问题"
      },
      {
        "name": "技能流程覆盖",
        "score": 80,
```

## 评分维度说明

| 维度 | 计算规则 |
|------|----------|
| 知识库完善度 | 已有答案的问题数 / 所有出现过的问题数 × 100 |
| 技能流程覆盖 | 已创建技能涵盖的业务场景数 / 店铺实际出现的场景数 × 100 |
| 报价准确率 | 近7天报价确认通过数 / 报价总数 × 100 |
| 独立接待率 | 近7天无转人工对话数 / 总对话数 × 100 |

## 输出格式

Markdown 表格展示各维度分数 + 文字列出提升建议。

## 使用时机

- 商家询问"牛仔表现怎么样"
- 商家询问"评分"、"掌握度"、"完善度"
- 日报中综合评价的补充详情
