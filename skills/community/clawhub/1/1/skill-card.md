## Description: <br>
Personal knowledge base powered by Ensue for capturing and retrieving understanding. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Nastrology](https://clawhub.ai/user/Nastrology) <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and developers use this skill to save, search, update, and delete structured personal knowledge through an Ensue-backed second brain. It supports recurring workflows such as saving concepts, recalling previous notes, managing toolbox entries, and connecting new learning to existing understanding. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Selected knowledge is stored remotely through Ensue-backed API operations. <br>
Mitigation: Install only if the user trusts Ensue and is comfortable storing selected knowledge remotely. <br>
Risk: Create, update, and delete operations can change or remove user knowledge entries. <br>
Mitigation: Review drafts before saving and confirm updates or deletes before execution. <br>
Risk: The skill requires an Ensue API key. <br>
Mitigation: Use a dedicated or revocable Ensue API key and avoid storing secrets, credentials, private paths, or system details in entries. <br>
