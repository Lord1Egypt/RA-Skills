# 特殊情况处理指南

## 全省统一命题

### 背景
部分省份从某年开始实行全省统一命题，此时"XX省中考"和"XX市中考"是同一份试卷。

### 已知情况
| 省份 | 开始年份 | 说明 |
|------|----------|------|
| 湖南 | 2024 | 长沙市中考使用湖南省统一命题试卷 |
| 浙江 | 2024 | 部分科目统考 |
| 江西 | 2024 | 全省统一命题 |

### 搜索策略调整
- 2024年起搜索"湖南省中考英语"而非"长沙市中考英语"
- 中考网可能只提供图片版（因统考试卷管控更严）
- 同省其他城市的试卷页面可能包含同一份试题

## 只有图片版试卷

### 判断方法
- 中考网页面底部无"下载完整版"按钮
- 页面显示"图片版"标签

### 处理流程
1. 获取各分页的图片URL
2. 逐张下载图片：`curl -L -A "Mozilla/5.0" -o "page_N.png" "{图片URL}"`
3. 整理命名：`{年份}年{城市}中考{科目}_第{N}页.png`
4. 上传知识库时逐个上传（IMA不支持ZIP）

### 图片URL获取
- 方法1: 用web_fetch获取页面HTML，提取`<img>`标签中files.eduuu.com的src
- 方法2: 用浏览器自动化（playwright）获取动态加载的图片
- 方法3: 从搜索结果中直接获取夸克网盘预览链接

## RAR文件解压

### macOS处理
```bash
# 安装unar (推荐)
brew install unar

# 解压RAR
unar -o /output/dir "file.rar"

# 或用Python + rarfile (需要unrar二进制)
pip install rarfile
python3 -c "import rarfile; rarfile.RarFile('file.rar').extractall('.')"
```

### 已知问题
- macOS系统默认无unrar，需安装
- Python rarfile模块需要unrar二进制支持
- 推荐使用 `unar`，Homebrew一键安装

## IMA知识库上传注意事项

### 支持的格式
| 格式 | media_type | 最大大小 |
|------|-----------|----------|
| PDF | 1 | 200MB |
| DOCX | 3 | 200MB |
| PPTX | 4 | 200MB |
| XLSX | 5 | 10MB |
| Markdown | 7 | 10MB |
| PNG/JPG | 9 | 30MB |
| TXT | 13 | 10MB |

### 关键限制
- ❌ **不支持ZIP格式**：上传ZIP会报"Unrecognized file extension"
- ❌ **无删除API**：IMA OpenAPI不提供delete_knowledge接口，上传后需在客户端手动删除
- ✅ 批量上传：可逐个文件上传，使用check_repeated_names检查重名

### 上传流程
使用ima-skill的脚本：
1. `preflight-check.cjs` — 检查文件类型和大小
2. `create_media` API — 获取media_id和COS凭证
3. `cos-upload.cjs` — 上传文件到COS
4. `add_knowledge` API — 关联到知识库

## 文件命名规范

### 统一格式
- 原卷版: `{年份}年{省份}{城市}中考{科目}真题.{扩展名}`
- 含答案: `{年份}年{省份}{城市}中考{科目}真题(含答案).{扩展名}`
- 图片版: `{年份}年{省份}{城市}中考{科目}_第{N}页.png`

### 示例
- `2021年湖南长沙中考英语真题.docx`
- `2025年湖南长沙中考英语真题(含答案).docx`
- `2023年长沙中考英语_第1页.png`

## 质量红线（禁止事项）

### ❌ 绝对禁止
1. **不可用搜索片段拼凑试卷** — 从不同来源拼凑的题目可能张冠李戴
2. **不可上传未经验证的文件** — 必须通过verify.sh验证后才能上传
3. **不可用AI编造/补全题目内容** — 缺失的部分标注"原文缺失"，不要自行补充
4. **不可跳过验证步骤** — 即使来源可信，也要做基本的大小和格式检查

### ⚠️ 警告
1. 中考网图片版可能清晰度不够 — 告知用户
2. 扫描嵌入版docx文件较大 — 正常现象，不要误判
3. 不同来源的同一年试卷可能有微小差异（含/不含答案、排版）— 以原卷版为准
