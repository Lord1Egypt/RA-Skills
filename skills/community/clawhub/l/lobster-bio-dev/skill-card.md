This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill when working on the Lobster AI codebase to understand architecture, add or modify agents and services, run tests, and follow contribution patterns. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Install or upgrade commands may change the active Lobster AI development environment. <br>
Mitigation: Run package commands only in a trusted repository and isolated environment after reviewing the exact command. <br>
Risk: Provider credentials may be used by workflows that call configured LLM services. <br>
Mitigation: Use least-privileged OpenAI, Anthropic, or AWS credentials and avoid committing local configuration or secrets. <br>
Risk: Lobster workspaces may retain sessions, provenance, cache files, and exported notebooks on disk. <br>
Mitigation: Review workspace contents before sharing or archiving projects, especially when working with sensitive data. <br>
