# 文书校验清单 (Litigation Validation Checklist)

> 使用说明：文书生成完成后，使用本清单逐项校验，确保文书完整合规。

---

## 一、通用校验项（12项 — 所有文书共用）

| # | 校验项 | 通过 | 备注 |
|---|-------|------|------|
| 1 | 输出格式符合法定要求 | ☐ | 标题/结构/落款完整 |
| 2 | 主体信息完整 | ☐ | 自然人6项（姓名/性别/民族/出生日期/住址/身份证号/电话）；法人5项（名称/住所地/法定代表人/统一社会信用代码/电话） |
| 3 | 事实与理由格式正确 | ☐ | 段落分明/时间顺序/逻辑清晰 |
| 4 | 法规引用带时效标注 | ☐ | ✅现行 / ⚠️修订 / ❌废止 |
| 5 | 费用承担完整 | ☐ | 含律师费/鉴定费/评估费等维权费用 |
| 6 | 日期格式正确 | ☐ | YYYY年MM月DD日 |
| 7 | 当事人信息一致性 | ☐ | 前后不矛盾 |
| 8 | 诉讼请求与事实理由对应 | ☐ | 每项请求都有事实支撑 |
| 9 | 法条引用有原文匹配 | ☐ | 准确引用条款号+原文 |
| 10 | 管辖法院标注正确 | ☐ | 级别+地域均正确 |
| 11 | 签名/盖章位置正确 | ☐ | 当事人签名在前，日期在后 |
| 12 | 附件清单完整 | ☐ | 证据目录/身份证明/委托书等 |

## 常见问题速查

| 常见错误 | 正确做法 |
|---------|---------|
| 原被告姓名/名称前后不一致 | 首次出现时用全称并注明简称（以下简称"XX"） |
| 诉讼请求金额只有"元"，未精确到分 | 金额须精确到分，如"人民币50,000.00元" |
| 引用法条未标注时效 | 标注 ✅/⚠️/❌ 及最新修订日期 |
| 证据目录列入身份证复印件 | 身份证明另行提交，不入证据目录 |
| 日期写成"2025.5.26" | 必须为"2025年05月26日" |
| 答辩状填写案由 | 答辩状不填案由字段 |
| 上诉状仅写"一审判决错误" | 应具体说明事实/法律/程序哪个方面错误 |
