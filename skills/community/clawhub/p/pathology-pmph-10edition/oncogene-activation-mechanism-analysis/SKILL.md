# 原癌基因活化机制分析

## 适用场景
- 已通过测序、FISH等分子检测手段确认特定原癌基因存在异常
- 需要解释该异常如何导致蛋白功能持续激活
- 需将机制与临床靶向治疗策略关联（如HER2扩增对应曲妥珠单抗）

## 执行步骤
1. **确认原癌基因名称**（如KRAS、HER2、CMYC）
2. **识别活化机制类型**：
   - **点突变**：单碱基替换导致氨基酸改变，使蛋白丧失调控能力（如RAS G12V）
   - **基因扩增**：拷贝数增加导致蛋白过表达（如HER2在乳腺癌中的扩增）
   - **染色体重排**：包括易位或倒位，可导致：
     - 原癌基因置于强启动子下游而高表达（如CMYC-IgH易位）
     - 形成融合基因编码组成性活性蛋白（如BCR::ABL1）
3. **描述功能后果**：明确蛋白是否持续激活、过表达或获得新功能
4. **关联肿瘤类型**：指出该机制常见于哪些人类肿瘤
5. **提示治疗意义**：简要说明对应靶向药物（如伊马替尼用于BCR::ABL1）

> 注意：本技能不适用于肿瘤抑制基因失活机制，也不涵盖非基因突变类机制（如纯表观遗传沉默）。
