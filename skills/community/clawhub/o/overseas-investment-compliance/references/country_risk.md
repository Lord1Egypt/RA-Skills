# 目的地国审查机制与地缘风险分级

## 一、主要投资目的地审查机制

### 美国：CFIUS

**法律基础**：FIRRMA(2018) → OIR规则(2025年1月生效) → COINS法案(2025年12月)

#### 强制申报门槛（OIR规则，针对中国）

| 领域 | 禁止类 | 需申报类 |
|------|--------|----------|
| **半导体** | 先进制程（≤14nm）代工、封测 | 传统制程 |
| **人工智能** | 军事/监控用途；训练算力>10^25次 | 其他AI系统 |
| **量子计算** | 全量子计算技术 | 特定量子技术 |
| **网络安全** | 特定网络防御技术 | 一般网络安全 |
| **数据安全** | 涉及美国人数据超过100万 | 敏感个人数据 |

#### 审查周期与成本

| 阶段 | 时间 | 费用 |
|------|------|------|
| 申报审查 | 30天（标准） | 免费（自愿申报） |
| 调查期 | 45天 | — |
| 总统决定 | 15天 | — |
| **总计** | **最长90天** | **律师费另计** |

### 欧盟：FDI审查框架

#### 成员国审查机制差异

| 国家 | 审查机制 | 关键行业 | 申报门槛 |
|------|----------|----------|----------|
| 德国 | 贸易法典第55条 | 关键技术、基础设施 | 10%股权 |
| 法国 | 2019年L.151-3条 | 国防、5G、能源 | 25%股权 |
| 英国 | NSIA法案2021 | 17个敏感行业 | 触发即申报 |
| 意大利 | "黄金权力"法案 | 5G、能源、金融 | 任意股权 |
| 西班牙 | RD 571/2019 | 国防、公共卫生 | 5%股权（特定行业） |

#### 2026年英国NSIA扩项预测
- 新增：AI基础设施、量子技术
- 强化：国防供应链、医疗设备

---

### 澳大利亚：FIRB

#### 核心特点

| 特点 | 说明 |
|------|------|
| **零门槛申报** | 国家安全业务一律申报 |
| **费用** | 申请费按投资金额计算 |
| **审查期** | 30天（可延长至90天） |
| **事后审查权** | 10年内可追溯审查 |
| **禁止令** | 可要求撤销已完成的交易 |

#### 高风险行业
- 农业用地/农业企业
- 电信/5G
- 能源/电力
- 港口/机场
- 国防相关

---

| 类别 | 行业 | 申报要求 |

| **航天** | 航天器、火箭 | 事前申报 |

| **尖端材料** | 集成电路、先进制造 | 事前申报 |

## 二、地缘风险分级

### 一级风险（🔴 高风险）

| 国家/地区 | 风险因素 | 建议 |
|------------|----------|------|
| **台湾** | 中美博弈焦点 | 极高谨慎，政治风险突出 |
| **美国** | 科技战、CFIUS审查 | 半导体/AI领域高度警惕 |
| **印度** | 边境摩擦、营商环境恶化 | 制造业转移需审慎评估 |

### 二级风险（🟡 中风险）

| 国家/地区 | 风险因素 | 建议 |
|------------|----------|------|
| **澳大利亚** | FIRB审查、数据安全法 | 农业/能源需预审 |
| **英国** | NSIA审查、技术出口管制 | 金融/科技需关注 |
| **欧盟主要国家** | FDI审查、GDPR合规 | 基础设施需预判 |
| **韩国** | 产业链敏感、中美夹缝 | 制造业相对安全 |
| **日本** | 技术出口管制 | 半导体需预审 |
| **一带一路部分国家** | 政治稳定性、法律环境 | 逐国评估 |

### 三级风险（🟢 低风险）

| 国家/地区 | 风险因素 | 建议 |
|------------|----------|------|
| **香港** | 政策稳定、CEPA支持 | 金融、专业服务首选 |
| **澳门** | 政策稳定 | 博彩/旅游需特许 |
| **东盟（新加坡、马来西亚等）** | 政策友好 | 制造业转移优先 |
| **中东（阿联酋、沙特）** | 开放政策 | 能源/基建机会 |

## 三、主要国家审查对中国企业的影响

### 美国对华投资限制领域（2024-2026年）

| 限制类型 | 具体内容 | 生效时间 |
|----------|----------|----------|
| **投资限制** | 禁止对华投资半导体、AI、量子领域 | 2024年生效 |
| **出口管制** | 先进芯片及制造设备出口限制 | 持续升级 |
| **实体清单** | 大量中国企业/个人列入清单 | 持续更新 |
| **OIR规则** | CFIUS强制申报门槛大幅降低 | 2025年1月 |

### 中国反制措施

| 措施类型 | 具体内容 |
|----------|----------|
| **不可靠实体清单** | 限制对华企业/个人交易 |
| **出口管制** | 关键矿物（镓、锗等）出口管制 |
| **安全审查** | 外国投资法安全审查 |
| **反制措施** | 《外商投资法》第35条 |

## 五、港澳台特殊定位

### 香港/澳门

| 维度 | 定位 | 说明 |
|------|------|------|
| **国令第837号** | 参照执行 | 第三十二条明确 |
| **外商投资** | CEPA特殊安排 | 零关税、服务贸易高开放度 |
| **投资便利** | 港资港法、港资港仲裁 | 2025年3月新增 |
| **内地企业赴港** | 需ODI备案 | 但通道相对成熟 |

### 台湾地区

| 维度 | 定位 | 说明 |
|------|------|------|
| **法律依据** | 《台湾同胞投资保护法》 | 2020年修订 |
| **参照适用** | 外商投资法 | 保护法未规定事项 |
| **安全审查** | 参照执行 | 《外商投资安全审查办法》第21条 |
| **投资形式** | 举办企业、房产、股票、土地开发等 | 全面保护 |

## 六、地缘风险应对策略

### 投资前风险评估清单

| 评估维度 | 检查项 |
|----------|--------|
| **目的地政策** | 是否与中国有重大摩擦？ |
| **行业敏感性** | 是否属于目的地敏感行业？ |
| **竞争格局** | 是否有本土竞争对手游说限制？ |
| **媒体舆论** | 是否有负面报道影响审批？ |
| **合作伙伴** | 是否有可靠的本地合资方？ |

### 风险缓释措施

| 风险类型 | 缓释措施 |
|----------|----------|
| 政治风险 | 购买政治风险保险（MAE） |
| 审查风险 | 聘请当地专业顾问提前沟通 |
| 合规风险 | 建立本地合规团队或聘请合规顾问 |
| 退出风险 | 设计合理的退出条款（对赌、回购） |
