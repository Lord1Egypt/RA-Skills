# Phlegm-Rheum Pathology Analysis

## When to Use
当出现以下情况之一时启用本技能：
- 患者主诉眩晕、头重、精神不振、咳吐痰涎、喉中痰鸣、胸闷气喘、恶心呕吐、肢体麻木、半身不遂或癫狂等症状；
- 存在外感六淫、情志失调或饮食不节等病因背景；
- 临床怀疑“怪病多痰”“百病多由痰作祟”等中医痰饮致病模式。

## Execution Steps

### 1. 判断痰饮形成机制
- 确认是否存在诱因：外感六淫、七情内伤、饮食不节；
- 评估肺、脾、肾、肝等脏腑气化功能是否失调；
- 推断津液代谢障碍 → 水液停聚 → 形成痰饮。

### 2. 辨识痰饮类型
- **有形之痰**：可见、可闻或可触（如咳痰、痰核、喉中痰鸣）；
- **无形之痰**：仅通过症状推断（如眩晕、癫狂、神昏）；
- **按部位分类饮证**：
  - 痰饮：停于胃肠；
  - 悬饮：停于胁下；
  - 溢饮：停于四肢肌表；
  - 支饮：停于肺。

### 3. 分析致病特点
- **阻滞气血运行**：观察是否出现经络或脏腑气机受阻表现（如麻木、胸闷、呕吐）；
- **影响津液代谢**：评估是否加重水湿不运、宣降失职；
- **扰乱神明**：判断是否有神昏、痴呆、癫狂、头昏等神志症状；
- **病势缠绵、变化多端**：注意病情反复、病程长、兼夹风火燥寒等复杂演变。

### 4. 输出结果
根据痰饮类型与停留部位，明确其致病表现，并为后续辨证论治提供方向（如化痰、蠲饮、调理脏腑等）。

> 注意：本技能不适用于津液代谢完全正常的健康状态。
