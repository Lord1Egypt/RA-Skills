# 反空洞词典 (Anti-Slop Lexicon)

> 注意力是预算。空洞词不仅丑，还贵——浪费模型有限的条件影响。
> 参考 `model-mechanics.md` 机制 1：注意力是预算。

---

## 核心原则

**删除所有不增加视觉信息的词。** 每个形容词必须指向一个可见的像素变化；如果不能，删除。

**替代方案**：用具体、可观察的视觉描述替代抽象质量判断。

## 保留的质量词 ✅

以下词在**特定上下文**中可以保留（因为它们指向可验证的技术标准）：

- **写实** / **realistic** — 当描述具体写实元素时
- **电影感** / **cinematic** — 当配合具体电影技术时（"电影感，85mm 浅景深"）
- **细腻** / **delicate** — 当描述具体细节时（"皮肤纹理细腻"）
- **精致** / **refined** — 当描述具体工艺时（"金属雕刻精致"）
- **厚重** / **weighty** — 当描述具体物理感时（"动作厚重，有惯性"）

**规则**：这些词后面必须跟具体视觉描述，不能单独使用。

## 反空洞检查流程

### 生成提示词后，执行以下检查：

1. **删除所有空洞质量词**（masterpiece, stunning, amazing 等）
2. **替换为具体视觉描述**
3. **检查每个形容词**：是否指向可见像素变化？如果不是，删除
4. **检查否定句**：能否改为正向描述？
5. **压缩长度**：目标 60-100 词（中文 200-400 字）

### 示例

**❌ 空洞版本**：
```
masterpiece, best quality, ultra-detailed, stunning landscape, 
amazing lighting, cinematic atmosphere, perfect composition
```

**✅ 具体版本**：
```
晨雾中的山谷，远山轮廓模糊，近景松树剪影。黄金时刻侧光，
暖橙色逆光勾勒人物轮廓。35mm 胶片颗粒，高光溢出，暗部不死黑。
85mm T1.5 浅景深，背景虚化为椭圆光斑。三分法构图，人物位于右下三分之一。
```

## 核心洞察

**空洞词是懒惰的遮羞布。** 如果你不能描述具体看到什么，就不要用抽象判断填充提示词。

**好的提示词像分镜脚本**：每个词都指向可见的像素变化。

**坏的提示词像广告文案**：堆砌空洞形容词，希望模型"理解"你的意图。
