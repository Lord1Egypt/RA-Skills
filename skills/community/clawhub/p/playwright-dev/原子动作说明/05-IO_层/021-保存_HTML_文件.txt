===========================================
原子动作 021 - 保存 HTML 文件
===========================================

【这个动作是干啥的】
把填充好的 HTML 内容保存到硬盘，生成一个 .html 文件。

【输入是啥】
- HTML 内容（从 020 动作来）
- 保存路径（比如 C:\...\expert-review.html）

【输出是啥】
- 硬盘上的 HTML 文件
- 返回文件路径和大小

【什么时候用】
- 写入内容后（020 动作完成后）
- 需要永久保存时
- 要用浏览器打开前

【注意事项】
- 目录不存在会自动创建
- 编码用 UTF-8（中文不会乱码）
- 文件名规范：expert-review-日期 - 主题.html

【对应阿福的 MD 文件】
atomic-actions/05-io/ATOM-IO-021-保存 HTML 文件.md

【举个例子】
阿福：生成 expert-review-2026-03-07-感知与行动中心.html
     保存到 C:\Users\Xiabi\.openclaw\workspace\
     大小：9.42 KB
     然后可以调用 009 动作用 Chrome 打开
