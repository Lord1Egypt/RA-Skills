# 对赌协议框架

> ⚠️ **法律声明**：本框架仅供参考，实际对赌条款需根据具体交易情况设计，建议在专业律师指导下完成。对赌协议效力在司法实践中已得到确认，但条款设计需符合公平原则。

---

## 对赌协议概述

### 对赌协议的法律性质

```
【定义】
对赌协议（Valuation Adjustment Mechanism，VAM），是投资方与融资方在达成协议时，
对未来不确定情况进行的一种约定。如果约定的条件出现，投资方可行使一种权利；
如果约定的条件不出现，融资方则可行使另一种权利。

【法律效力】
- 2019年《全国法院民商事审判工作会议纪要》（九民纪要）明确：
- 与目标公司股东/实际控制人对赌：有效
- 与目标公司本身对赌：需符合公司法关于利润分配的强制性规定
```

### 对赌协议的核心要素

| 要素 | 说明 | 关键考量 |
|------|------|---------|
| 标的 | 赌什么 | 净利润、收入、市场份额 |
| 期限 | 赌多久 | 1-5年常见 |
| 门槛 | 赌多少 | 承诺值设定 |
| 补偿方式 | 怎么赌 | 现金/股权/双向 |
| 触发条件 | 何时触发 | 低于门槛 |
| 豁免情形 | 哪些不算 | 不可抗力等 |

## 业绩承诺设计

### 承诺指标选择

#### 常用指标对比

| 指标类型 | 优点 | 缺点 | 适用场景 |
|---------|------|------|---------|
| 净利润 | 最直观，易核实 | 受会计政策影响 | 成熟企业 |
| 营业收入 | 核实简单 | 不反映盈利能力 | 早期/高增长 |
| 毛利率 | 反映竞争力 | 核实复杂 | 产品型企业 |
| 用户数 | 互联网常用 | 可能注水 | 互联网企业 |
| 市场份额 | 战略意义强 | 难以核实 | 竞争激烈行业 |
| 复合指标 | 综合评估 | 设计复杂 | 复杂交易 |

#### OPC建议：技术方指标选择

```
【优先选择】
1. 净利润（最常用）
   - 建议增加"扣除非经常性损益"限定
   - 建议明确"审计口径"

2. 营业收入
   - 适用于早期高增长企业
   - 需明确收入确认标准

3. 复合指标
   - 主指标：净利润（权重60%）
   - 辅指标：收入增长率（权重30%）
   - 辅指标：重点客户留存（权重10%）

【谨慎选择】
❌ 过于复杂的指标（如需主观判断）
❌ 无法核实的指标（如"品牌价值"）
❌ 单一指标过大（如仅选净利润）
```

### 承诺期限设计

```
【行业参考】

| 行业 | 常见期限 | 特点 |
|------|---------|------|
| 传统行业 | 3年 | 业绩稳定，易预测 |
| 科技行业 | 2-3年 | 增长快，周期短 |
| 制造业 | 3年 | 周期长，稳定 |
| 互联网 | 1-2年 | 变化快 |
| 生物医药 | 3-5年 | 研发周期长 |

【OPC建议】

1. 期限设置
   □ 建议2-3年，避免过长
   □ 跨年度递增（首年低，后续提高）

2. 分段对赌
   第1年：承诺800万净利润
   第2年：承诺1000万净利润
   第3年：承诺1200万净利润

3. 豁免情形
   □ 不可抗力（自然灾害、疫情等）
   □ 政策重大变化（行业政策调整）
   □ 市场重大变化（非双方原因）
```

## 补偿计算方式

### 方式一：现金补偿

```
【公式】
补偿金额 = (承诺利润 - 实际利润) × 补偿系数

【示例】
- 承诺利润：1000万元
- 实际利润：800万元
- 差额：200万元
- 补偿系数：1
- 应补偿金额：200万元

【变体公式】
补偿金额 = 投资额 × (承诺利润/实际利润 - 1)

【优缺点】
✓ 计算简单直观
✓ 卖方风险可控
✗ 不影响股权结构
✗ 通货膨胀影响
```

### 方式二：股权补偿

```
【公式】
补偿股数 = (承诺利润 - 实际利润) / 每股价格

【示例】
- 承诺利润：1000万元
- 实际利润：800万元
- 差额：200万元
- 每股净资产：10元
- 应补偿股数：20万股

【优缺点】
✓ 直接影响股权结构
✓ 激励效果明显
✗ 计算较复杂
✗ 需办理工商变更
```

### 方式三：双向对赌

```
【结构】
├── 下行对赌（业绩未达标）
│   └── 卖方补偿买方
└── 上行对赌（业绩超标）
    └── 买方奖励卖方

【双向对赌示例】
若净利润<承诺×80%：卖方补偿差额
若净利润>承诺×120%：买方奖励卖方（超额部分×50%）

【优缺点】
✓ 激励约束对等
✓ 激发团队动力
✗ 设计复杂
✗ 需双方协商一致
```

## 对赌条款模板

### 基础条款模板

```
【业绩承诺条款模板】

第X条 业绩承诺

X.1 承诺期间
各方确认，本协议项下的业绩承诺期间为[X]年，
自[Y年]年[1]月[1]日起至[Y+n年]年[12]月[31]日止。

X.2 承诺指标
承诺期间内，目标公司各年度的承诺指标如下：

| 年度 | 承诺净利润 | 承诺营业收入 | 备注 |
|------|-----------|-------------|------|
| 第1年 | [X]万元 | [X]万元 | - |
| 第2年 | [X]万元 | [X]万元 | - |
| 第3年 | [X]万元 | [X]万元 | - |

X.3 指标定义
"净利润"指目标公司经审计的合并报表归属于母公司股东的净利润，
扣除非经常性损益。

X.4 审计机构
各方同意，由[审计机构名称]担任本协议的审计机构。
审计报告应于每个承诺年度结束后[X]个月内出具。
```

### 补偿条款模板

```
【补偿条款模板】

第X条 补偿安排

X.1 补偿触发条件
若承诺期间内任一年度的实际利润低于当年承诺利润的[X]%（以下简称"触发线"），
则触发本条项下的补偿义务。

X.2 现金补偿方式
补偿金额 = (承诺利润 - 实际利润) × 补偿系数

其中：
- 承诺利润：当年承诺净利润
- 实际利润：当年审计确认的净利润
- 补偿系数：1（可根据谈判调整）

X.3 股权补偿方式（可选）
补偿股数 = (承诺利润 - 实际利润) / 每股价格

其中：
- 每股价格：[投资时的每股价格]元

X.4 补偿上限
卖方向买方补偿的总额不超过本次交易对价的[X]%。

X.5 补偿期限
补偿义务应在相应年度审计报告出具后[X]个工作日内履行。
```

### 豁免条款模板

```
【豁免条款模板】

第X条 豁免情形

X.1 不可抗力豁免
因以下情形导致业绩承诺无法完成的，卖方无需承担补偿责任：
（一）地震、台风、洪水等自然灾害
（二）战争、暴乱等社会异常事件
（三）瘟疫、传染病等公共卫生事件
（四）法律法规或政策重大变化

X.2 市场因素豁免
因以下情形导致业绩承诺无法完成的，卖方可申请豁免：
（一）目标公司所处行业整体下行超过[X]%
（二）目标公司主要客户/供应商发生重大不利变化
（三）目标公司核心技术被重大技术替代

X.3 豁免程序
（一）触发豁免情形后[X]日内，卖方应书面通知买方
（二）双方协商确定豁免范围
（三）协商不成的，提交[仲裁机构]裁定

X.4 部分豁免
若豁免情形仅影响部分业绩，经协商可按比例豁免。
```

## 对赌谈判要点

### 卖方（技术方）谈判策略

```
【争取条款】

1. 设置合理的承诺值
   □ 基于历史业绩合理预测
   □ 留有一定安全边际
   □ 分段递增设计

2. 争取豁免条款
   □ 不可抗力豁免
   □ 政策变化豁免
   □ 市场变化豁免

3. 争取调整机制
   □ 善意调整（非卖方原因）
   □ 口径调整（会计准则变化）
   □ 范围调整（收购剥离）

4. 争取分段补偿
   □ 避免一次性大额补偿
   □ 争取分年度补偿

5. 争取超额奖励
   □ 争取双向对赌
   □ 激发团队动力

【拒绝条款】

❌ 过于激进的承诺值
❌ 无豁免情形
❌ 一次性大额补偿
❌ 补偿无上限
❌ 过长的承诺期限
```

### 买方谈判策略

```
【争取条款】

1. 设置合理的触发线
   □ 建议80-90%的触发线
   □ 避免全额完成才不算触发

2. 争取完整的补偿机制
   □ 现金+股权双重补偿
   □ 补偿比例较高

3. 争取较长的承诺期限
   □ 建议3年
   □ 观察长期表现

4. 争取加速到期条款
   □ 核心人员离职触发
   □ 重大资产处置触发

【注意条款】

⚠️ 避免过高承诺值导致无法完成
⚠️ 避免过于复杂的指标导致争议
⚠️ 确保计算方式明确无争议
```

## 对赌风险分析

### 卖方（技术方）主要风险

| 风险类型 | 风险描述 | 应对措施 |
|---------|---------|---------|
| 承诺值过高 | 无法完成触发补偿 | 设置合理预测 |
| 外部因素 | 市场/政策变化 | 增加豁免条款 |
| 会计政策 | 口径不一致 | 明确计算标准 |
| 审计风险 | 审计调整 | 保留核查权 |

### 买方主要风险

| 风险类型 | 风险描述 | 应对措施 |
|---------|---------|---------|
| 数据造假 | 虚假业绩 | 加强尽调 |
| 会计处理 | 利润操纵 | 明确口径 |
| 人员流失 | 核心人员离开 | 绑定条款 |
| 执行难度 | 补偿无法落实 | 担保措施 |

## 案例参考

### 经典对赌案例

```
【案例1：PE对赌经典案例】
某PE投资某科技公司：
- 承诺3年净利润分别达到3000万、4000万、5000万
- 未达标部分由实控人现金补偿
- 结果：第2年未达标，实控人补偿2000万元

【案例2：双向对赌案例】
某上市公司收购某游戏公司：
- 设置双向对赌
- 净利润<承诺：卖方补偿
- 净利润>承诺×150%：买方奖励卖方
- 结果：超额完成，双方均获益

【案例3：技术方谈判成功案例】
某技术公司融资：
- 原始方案：承诺3年净利润5000万、6000万、7000万
- 谈判后：改为分段递增4000万、5000万、6000万
- 增加：不可抗力豁免条款
- 增加：超额奖励条款
```

## OPC特别注意事项

```
【技术方对赌策略建议】

1. 保守承诺
   □ 不要被高估值冲昏头脑
   □ 承诺值留20-30%安全边际
   □ 宁可估值低一些，也要承诺可完成

2. 增加保护条款
   □ 豁免条款必须争取
   □ 明确会计口径
   □ 保留数据核查权

3. 设计缓冲机制
   □ 分段对赌（每年独立考核）
   □ 累计对赌（3年累计）
   □ 复合指标（多维度）

4. 争取激励条款
   □ 超额完成奖励
   □ 长期绑定奖励
   □ 团队激励参与

5. 设置退出机制
   □ 提前还款可解除对赌
   □ 业绩反转可恢复
   □ 股权回购安排
```
