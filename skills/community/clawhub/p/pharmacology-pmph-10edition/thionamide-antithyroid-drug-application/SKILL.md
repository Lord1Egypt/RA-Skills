# 硫脲类抗甲状腺药的作用机制与临床应用

## 何时使用
- 患者确诊为甲状腺功能亢进（甲亢）
- 无粒细胞缺乏病史
- 非结节性甲状腺肿合并甲亢或甲状腺癌
- 妊娠期需谨慎评估后选择药物（避免妊娠晚期使用）
- 适用于轻症甲亢、儿童/青少年患者、术后复发或合并心肝肾疾病者
- 用于甲亢术前准备或甲状腺危象的紧急处理

## 如何执行

### 1. 抑制甲状腺激素合成
- 硫脲类药物通过作为过氧化物酶底物被氧化，干扰酪氨酸碘化及MIT/DIT偶联，从而减少T₃和T₄合成。
- 注意：对已合成的甲状腺激素无效，症状改善通常需2～3周，基础代谢率恢复正常需1～2个月。

### 2. 抑制外周T₄向T₃转化
- **丙硫氧嘧啶（PTU）**：显著抑制T₄转T₃，适用于重症甲亢或甲状腺危象，列为首选。
- **甲巯咪唑**：此作用较弱，不推荐用于危象。

### 3. 免疫调节作用
- 轻度抑制免疫球蛋白生成，降低甲状腺刺激性免疫球蛋白（TSI）水平，对甲亢病因有一定治疗作用。

### 4. 临床应用场景操作流程

#### 内科治疗
- 初始剂量控制症状（通常1～2月内）。
- 基础代谢率接近正常后逐步减量至维持量（甲巯咪唑5～10 mg/日）。
- 疗程持续1～2年。
- 使用T₃抑制试验或TRH兴奋试验监测疗效；结果正常后再停药可降低复发率（治愈率40%～70%）。

#### 术前准备
- 先用硫脲类使甲状腺功能恢复正常。
- 因硫脲类可致TSH升高、腺体充血，术前2周加用大剂量碘剂使腺体缩小变韧，便于手术。

#### 甲状腺危象
- 立即使用大剂量硫脲类（常选PTU，剂量为常规治疗量的2倍）。
- 同时合用碘剂以抑制已合成激素的释放。
- 疗程一般不超过1周。

### 5. 监测与安全
- 定期监测血常规（警惕粒细胞缺乏）。
- 监测甲状腺功能（FT₃、FT₄、TSH）。
- 妊娠期优先选择PTU（尤其孕早期），但需权衡风险。

> **注意**：禁用于结节性甲状腺肿合并甲亢及甲状腺癌患者；对已合成激素无效，不可期望立即起效。
