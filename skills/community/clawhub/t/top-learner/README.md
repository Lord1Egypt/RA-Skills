## ❓ 为什么你学了 10 遍还是忘？

- 看网课时感觉「懂了」，做题时却一片空白
- 笔记记了三大本，考前发现根本来不及复习
- 刷了 100 道题，换一道新题型还是不会
- 花 3 小时「看书」，大脑实际只工作了 20 分钟

**问题不在你的智商，而在方法。**

认知科学早就证明：专家的脑子并不比新手大，他们只是**把知识组织成了更优的结构**。这个 Skill 就是把这套「专家级知识组织法」和「刻意练习系统」装进 AI，让你拥有一个 24 小时在线的认知科学教练。

## ✨ 三大核心能力

### 1️⃣ 深度教学 —— 讲清「为什么」，而不只是「是什么」

- **直觉先于形式**：先让你在脑中「看见」概念在发生什么，再进入公式和符号
- **迷思概念预警**：提前告诉你「90% 的学生在这里想错」，避免踩坑
- **边界条件剖析**：不仅讲「怎么用」，更讲「什么时候失效」
- **知识网络连接**：每个新概念都告诉你——它从哪来？和什么相连？通向哪里？

### 2️⃣ 结构诊断 —— 检验你的知识「组织形态」

不只是问「这道题对不对」，而是诊断你的大脑把知识存成了「新手形态」还是「专家形态」：

| 诊断维度 | 检测内容 |
|---------|---------|
| 🔍 提取检验 | 合上书，你能独立写出定义和推导吗？ |
| 🧩 结构检验 | 给你一堆题，你按「表面特征」还是「深层原理」分类？ |
| 🎯 迁移检验 | 条件变了、题型交叉了，你还能解吗？ |
| 🧠 元认知检验 | 你知道自己哪里不懂吗？校准度如何？ |

### 3️⃣ 刻意练习编排 —— 从「知道」到「做到」

**📝 考试型路径**：限时训练 → 模拟考试工程 → 题型交叉编排 → 错题归因 → 时间压力适应

**🔧 应用型路径**：端到端项目 → 真实数据/真实约束 → 开放问题 → 错误调试 → 输出物导向

## 🚀 快速开始

1. **安装本 Skill** 到你的 AI 客户端
2. **上传学习材料**（讲义 / 课件 / 课本章节 / 笔记）或描述你想学的主题
3. **告诉目标**：考试？应用？有多少时间？
4. **跟随教练流程**：材料解析 → 深度教学 → 结构检验 → 刻意实践 → 总结追踪

## 🔬 科学依据

- **Barbara Oakley**《学习之道》— 组块化、专注与发散模式
- **Make It Stick**（《认知天性》）— 提取练习、间隔重复、多样化练习
- **Michele Chi** 知识结构研究 — 专家与新手的组织形态差异
- **MIT / Stanford 主动学习** — 概念检验、同伴教学、即时反馈

核心原则：**Learning is a skill, not a talent.**
