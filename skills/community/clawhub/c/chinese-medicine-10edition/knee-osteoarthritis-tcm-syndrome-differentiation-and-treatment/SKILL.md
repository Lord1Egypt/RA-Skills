# Knee Osteoarthritis TCM Syndrome Differentiation and Treatment

## When to Use
- 膝痹患者
- 膝关节退行性病变者

## When to Use
- 存在膝关节疼痛、屈伸不利、肿胀或变形
- 已排除急性感染或肿瘤

## Execution Steps

### 1. 辨识病机总纲
确认“本虚标实”核心病机：
- **本虚**：肝肾亏损（肝血不养筋，肾精不充骨）
- **标实**：风寒、湿热、痰湿、瘀血闭阻经络

### 2. 判断证型并选择内服方药
根据临床表现辨为以下五型之一，并对应选方：
- **湿热痹阻**：膝红肿热痛、苔黄腻、脉濡数 → 清热通络、祛风除湿 → 四妙散加减
- **寒湿痹阻**：疼痛重着、遇冷加重、苔白腻 → 温阳散寒、缓急止痛 → 蠲痹汤加减
- **气滞血瘀**：刺痛定处、舌紫暗、脉沉涩 → 活血化瘀、通络止痛 → 血府逐瘀汤加减
- **肝肾亏虚**：隐痛酸软、遇劳加重、舌红少苔 → 补益肝肾、舒筋止痛 → 左归丸加减
- **气血亏虚**：酸痛伴心悸气短、面色少华 → 益气养血、和营通络 → 八珍汤加减

### 3. 配合外治法
- **中药外用**：桃红四物汤或海桐皮汤熏洗；三黄油膏外敷（急性红肿热痛期慎用温热外治）
- **手法治疗**：理筋整复，手法轻柔深透，避免暴力
- **针灸疗法**：取穴膝眼、犊鼻、血海、阳陵泉等，辨证施针或针灸并用

### 4. 指导运动疗法
- 推荐非负重膝关节屈伸训练
- 强化股四头肌锻炼
- 练习太极拳、八段锦或易筋经，防止肌肉萎缩

### 5. 注意事项
- 严重关节畸形或功能障碍者需考虑手术干预
- 治疗目标为缓解疼痛、延缓关节损伤、维持功能，强调内外合治、标本兼顾

> 提示：若症状处于急性红肿热痛期，避免使用温热性质的外治方法。
