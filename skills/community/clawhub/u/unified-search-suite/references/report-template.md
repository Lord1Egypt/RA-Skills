# Unified Search Report Template

Use this format after running the script:

1. **结论**
   - 1-3 bullet points with direct answer.

2. **证据（按可信度）**
   - Source + key quote/claim + link.
   - Mark overlaps across engines.

3. **冲突点/不确定性**
   - What disagrees and why.

4. **建议下一步**
   - Exact command/query refinement if needed.

## Query Refinement Tips

- Language bias: add Chinese keywords + Chinese site constraints in query.
- Freshness: switch to `--topic news --days 7`.
- Recall: increase `--num` from 5 to 8-10.
- Precision: narrow by domain keywords (policy/API/docs/version).
