# GB 19153-2019 容积式空气压缩机能效限定值及能效等级

## 适用范围

本标准适用于：
- 螺杆空压机（喷油/无油，空气冷却/水冷却）
- 离心空压机
- 活塞空压机
- 功率范围：2.2 kW ~ 630 kW
- 压力范围：0.7 MPa ~ 1.25 MPa（即 7 bar ~ 12.5 bar）

## 能效等级定义

| 等级 | 含义 |
|------|------|
| **1级** | 最高能效，达到国际先进水平 |
| **2级** | 节能水平，高于市场准入要求 |
| **3级** | 市场准入最低能效限值（低于此级不允许生产销售） |
| **未达标** | 不满足 3 级限值，属于淘汰设备 |

## 使用方法

1. 确定压缩机类型（螺杆/离心/活塞）
2. 确定冷却方式（风冷/水冷，以上为风冷值；水冷降低约 0.2-0.4 kW/(m³/min)）
3. 确定额定排气压力（MPa），选择对应表格
4. 根据驱动电机功率（kW）找到对应行
5. 将用户的比功率（kW/(m³/min)）与三个等级限值比较：
   - 比功率 ≤ 1级限值 → 1级能效
   - 1级限值 < 比功率 ≤ 2级限值 → 2级能效
   - 2级限值 < 比功率 ≤ 3级限值 → 3级能效
   - 比功率 > 3级限值 → 未达标（淘汰设备）

## 注意事项

- 以上数据基于 GB 19153-2019，实际判定以正式标准文本为准
- 变频机组的能效评价方式与工频机组不同，涉及加权比功率（需多个负载点的比功率加权平均）
- 本参考表仅用于快速估算，精确判定需现场测量：实际排气量、实际输入功率、实际排气压力
- 如果用户无法提供实测比功率，用典型范围估算：
  - 新螺杆机（<3年）：5.8-6.5 kW/(m³/min) @ 8 bar
  - 中等年限螺杆机（3-8年）：6.5-7.5 kW/(m³/min) @ 8 bar
  - 老旧螺杆机（>8年）：7.5-8.5 kW/(m³/min) @ 8 bar
