## 环境准备

1. 安装金蝶云星空（本地开发环境或私有云）
2. 打开浏览器访问 `http://IP/K3Cloud` 进入星空前端
3. 使用 administrator 登录管理中心（默认密码 888888 或 kingdee@123）
4. 创建账套、注册许可

## BOS IDE 登录

1. 前端 → 应用菜单 →「BOS设计平台」
2. 或直接访问 `http://IP/K3Cloud/BOSIDE`
3. 登录后左侧为「业务对象模型树」，按模块组织

### 常见登录问题

| 问题 | 解决 |
|------|------|
| BOS平台打开报错 | https://vip.kingdee.com/article/471624933962111488 |
| 页面加载慢 | 清理浏览器缓存，使用 Chrome |
| 无法看到业务对象 | 检查用户权限，确保有开发权限 |

## 单据设计

### 新建单据

1. 右键目标模块 →「新增业务对象」→「单据」
2. 填写标识（如 `MY_Order`）、名称
3. 系统自动创建单据头 + 单据体
4. 添加字段到单据头和单据体
5. 保存 → 发布到主控台

### 扩展系统自带单据

**重要原则：系统自带单据不能直接修改，必须先「扩展」！**

1. 右键目标单据 →「扩展」
2. 系统自动创建扩展副本（标识带 `_Extension` 后缀）
3. 在扩展副本上添加字段、修改布局
4. 修改前必须「签出」
5. 修改后「保存」→「签入」→「发布到主控台」

### 签出/签入规则

| 操作 | 说明 |
|------|------|
| 签出 | 锁定对象，独占编辑权 |
| 保存 | 保存当前修改 |
| 签入 | 释放锁定，其他人可编辑 |
| **顺序** | 签出 → 修改 → 保存 → 签入 → 发布 |

⚠️ **必须先保存再签入！未保存直接签入会丢失修改。**

## 字段操作

### 查看单据所有字段

1. 打开单据设计器
2. 工具栏点击「编辑」→「批量编辑单据字段」
3. 可查看/修改所有字段的属性

### 常用字段类型

| 字段类型 | 对应控件 | 数据库类型 | 示例 |
|----------|----------|------------|------|
| 文本 | TextBox | nvarchar | FBillNo |
| 整数 | Integer | int | FQty |
| 小数 | Decimal | decimal | FAmount |
| 日期 | Date | datetime | FDate |
| 布尔 | CheckBox | bit | FIsChecked |
| 基础资料 | BaseData | bigint(FK) | FMaterialId |
| 下拉列表 | Combo | int | FBillStatus |
| 多语言文本 | Locale | nvarchar | FName |

### 添加自定义字段

1. 签出单据
2. 在单据设计器中选择单据头/单据体区域
3. 右键 →「新增字段」
4. 设置字段属性：
   - **标识**：F + 业务含义（如 FCustRemark）
   - **名称**：中文名
   - **类型**：选择对应类型
   - **必录**：是否必填
5. 保存 → 签入

### 字段绑定到界面

1. 在设计器中拖拽「标签+控件」到布局区域
2. 绑定字段标识
3. 设置控件属性（可见性、启用、只读等）

### 隐藏/只读字段（代码控制）

在表单插件的 `AfterBindData` 中：
``csharp
this.View.GetControl("FFieldName").Visible = false;   // 隐藏
this.View.GetControl("FFieldName").Enabled = false;   // 禁用（只读）
``

## 菜单发布

单据/报表创建后必须发布到主控台才能在前台看到。

### 发布步骤

1. 工具栏 →「发布」→「发布到主控台」
2. 选择发布位置（模块 → 子功能 → 业务场景）
3. 选择发布对象（单据对象）
4. 设置权限
5. 勾选「显示前台类型」（列表/明细/新增/修改等）
6. 确定

### 取消发布

1. 工具栏 →「发布」→「取消发布」
2. 选择要取消的发布记录

## 单据转换规则

配置单据之间的下推关系（如销售订单 → 发货通知单 → 销售出库单）。

### 配置步骤

1. BOS IDE →「转换规则」节点
2. 新建转换规则
3. 设置源单和目标单
4. 配置字段映射（自动映射 + 手动调整）
5. 配置下推条件
6. 保存并发布

### 转换插件

在转换规则上注册转换插件，干预转换逻辑：
- 修改字段映射
- 过滤源单数据
- 自动填充目标字段

## 动态表单

动态表单是不绑定单据对象的表单，用于自定义对话框、查询界面等。

### 创建动态表单

1. 右键模块 →「新增业务对象」→「动态表单」
2. 设计布局（不包含单据头/体结构）
3. 添加字段和按钮
4. 注册表单插件
5. 发布

### 打开动态表单

``csharp
var showParam = new DynamicFormShowParameter();
showParam.FormId = "MY_CustomDialog";
showParam.OpenStyle = FormOpenStyle.Modal;  // 模态窗口
this.View.ShowForm(showParam, (formResult) =>
{
    // 处理返回结果
    if (formResult.ReturnData != null)
    {
        // ...
    }
});
``

## 基础资料扩展

系统自带基础资料（物料、客户、供应商等）可以扩展字段。

1. 找到基础资料对象（如 `BD_Material`）
2. 右键 →「扩展」
3. 添加自定义字段
4. 在基础资料表单中显示自定义字段
5. 发布

## 权限配置

### 功能权限

1. 系统前台 →「基础管理」→「权限管理」
2. 创建角色或选择已有角色
3. 分配功能权限（菜单级）
4. 分配字段权限（字段级，可选）

### 字段级权限

1. 在角色权限设置中 →「字段权限」
2. 选择单据
3. 设置每个字段的「可见/编辑/不可见」

### 用户权限同步

⚠️ **修改用户权限后，必须在用户列表页点击「同步注册用户」按钮！**
