# 项目笔记模板

首次进入项目群时创建 `memory/Group/[群名].md`，按此结构填充。

## 模板

```markdown
# [群名] 项目笔记

## 📌 群用途/目标
（从上下文推断或询问群主）

## 👥 成员及角色
| 成员 | 角色/职责 |
|:---|:---|
| @username:matrix.biochao.cc | 职责描述 |

## 📋 当前进展
（整理当前项目的状态）

## ✅ 已完成
- ...

## 🔄 进行中
- ...

## ⏳ 待决事项
| 事项 | 负责人 | 状态 | 截止 |
|:---|:---|:---|:---|
| | | | |

## 📅 会议纪要
（链接或直接记录关键结论）

## 📎 相关资源
- 相关文档/链接
- 重要文件路径
```

## Checkpoint 格式

长任务中断或定期 checkpoint 时使用：

```markdown
## Checkpoint: YYYY-MM-DD HH:MM

### 任务
（一句话描述）

### 状态
- [x] 已完成：XXX
- [ ] 进行中：YYY（进度%）
- [ ] 待开始：ZZZ

### 关键上下文
（重要决定、路径、变量、踩过的坑）

### 下一步行动
1. ...
2. ...
```

## 每日记录格式

`memory/YYYY-MM-DD.md`：

```markdown
# YYYY-MM-DD 日志

## 今日事项
- [时间] 事件描述

## 决策记录
- （重要决定及原因）

## 待跟进
- ...
```
