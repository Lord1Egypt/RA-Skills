# 通气储备百分比计算与解读

## 何时使用
- 患者拟接受胸腹部大手术，需评估术后肺部并发症风险
- 进行职业病（如尘肺）导致的劳动能力鉴定
- 管理慢性呼吸系统疾病（如COPD、肺纤维化），制定或调整康复方案

## 前提条件
- 已完成最大自主通气量（MVV）测试
- 已测定每分钟静息通气量（VE）

## 执行步骤
1. **确认适用性**：排除急性呼吸衰竭或无法配合用力呼吸的患者。
2. **获取关键数值**：记录 MVV（L/min）和 VE（L/min）。
3. **计算通气储备百分比**：
   ```
   通气储备百分比 = [(MVV - VE) / MVV] × 100%
   ```
4. **根据阈值解读结果**：
   - ≥95%：通气储备充足，风险低
   - 86%~94%：轻度储备不足，需谨慎评估
   - <86%：通气储备不足，存在显著通气功能障碍
   - 60%~70%：达“气息阈”，日常活动可能受限
5. **结合临床背景综合判断**：考虑基础疾病类型（阻塞性 vs 限制性）、患者配合程度及 VE 异常情况。

> 注意：结果高度依赖患者努力程度，若 MVV 或 VE 数据不可靠，应重新测试或谨慎解读。
