## 第一部分：动物咬伤

### 一、狗/猫咬伤

#### 狂犬病风险

> ⚠️ **狂犬病一旦发病，死亡率接近100%。所有哺乳动物咬伤都要考虑狂犬病风险！**

#### 伤口处理三步法

| 步骤 | 操作 | 要点 |
|------|------|------|
| **1. 冲洗** | 肥皂水（或清水）+ 流动水交替冲洗 | **至少15分钟**，这是最关键的一步！ |
| **2. 消毒** | 碘伏或75%酒精消毒伤口 | 不要包扎（除非出血不止） |
| **3. 就医** | 尽快前往犬伤门诊 | 接种狂犬疫苗 ± 免疫球蛋白 |

#### 狂犬疫苗决策

| 暴露等级 | 接触方式 | 处理 |
|----------|----------|------|
| I级 | 接触/喂养动物，皮肤完好被舔 | 无需处理 |
| II级 | 裸露皮肤被轻咬，无出血的轻微抓伤/擦伤 | **接种狂犬疫苗** |
| III级 | 单处/多处贯穿皮肤咬伤，黏膜被污染 | **狂犬疫苗 + 狂犬免疫球蛋白** |

**关键信息**：
- 狂犬疫苗最佳接种时间：**被咬后24小时内**
- 即使超过24小时，也**必须接种**（只要未发病）
- 中国常用"5针法"（0、3、7、14、28天）或"2-1-1"法

#### 破伤风预防
- 检查破伤风疫苗接种史
- 如距离上次接种 > 5年，需打破伤风针

### 二、蛇咬伤

#### 毒蛇 vs 无毒蛇咬痕

|  | 毒蛇 | 无毒蛇 |
|------|------|------|
| 牙痕 | 1-2个深大牙痕（毒牙） | 多排细小牙痕 |
| 疼痛 | 剧痛，迅速肿胀 | 轻微疼痛 |
| 全身症状 | 恶心、视力模糊、呼吸困难、出血 | 无 |

#### 毒蛇咬伤急救

| ✅ 该做 | ❌ 不该做 |
|---------|----------|
| 保持镇静，减少活动（减缓毒液扩散） | **不要用嘴吸毒！**（口内有伤口会中毒） |
| 取下患肢的戒指/手镯/手表 | **不要切开伤口** |
| 用夹板/绷带固定患肢，保持低于心脏 | **不要冰敷**（加重组织损伤） |
| 记住蛇的外观特征（颜色、花纹、头形） | **不要饮酒** |
| 尽快送医（使用抗蛇毒血清） | **不要使用止血带**（现在的共识） |
| 如可能，拍蛇的照片（安全前提下） | **不要追赶/杀死蛇**（可能再次被咬） |

#### 压力固定法（Pressure Immobilization）

适用于神经毒蛇咬伤（如金环蛇、银环蛇）：
1. 用弹性绷带从咬伤处向上包扎整个肢体
2. 松紧度：能轻松塞入一根手指
3. 用夹板固定患肢
4. 不要移动患肢

### 三、蜂蜇伤

| 蜂种 | 特点 | 处理 |
|------|------|------|
| 蜜蜂 | 蜂针留在皮肤内，蜜蜂死亡 | 用卡片刮除蜂针（不要用镊子夹，会挤出更多毒液） |
| 胡蜂/马蜂 | 可反复蜇，蜂针不残留 | 清洗伤口 |

#### 处理步骤

1. 蜜蜂蛰：卡片/指甲**刮除**蜂针（不要夹/拔）
2. 肥皂水清洗伤口
3. 冷敷减轻肿痛
4. 抗组胺药（如氯雷他定）缓解瘙痒

#### ⚠️ 过敏性休克

**速发型过敏反应**（蜂蛰后数分钟-30分钟出现）：
- 全身荨麻疹/红肿
- 呼吸困难/喉头水肿
- 低血压/意识丧失

这是**致命急症**！处理：
1. 立即拨打120
2. 如有肾上腺素自动注射笔（EpiPen），立即大腿外侧注射
3. 平卧，抬高双腿
4. 如呼吸停止 → 开始CPR

### 四、蜱虫叮咬

> ⚠️ 蜱虫可传播莱姆病、森林脑炎、发热伴血小板减少综合征（SFTS）

#### 正确取蜱方法

1. 用**尖头镊子**夹住蜱虫头部（尽可能贴近皮肤）
2. **垂直向上**，平稳、缓慢地拔出
3. **不要旋转、不要挤压腹部**（可能将病原体挤入咬伤处）
4. 取出后用碘伏/酒精消毒
5. 蜱虫放入密封袋保存（如之后出现症状供医生鉴定）
6. 用肥皂水洗手

#### ❌ 错误取蜱方法

- 不要用烟头烫
- 不要涂指甲油/酒精/凡士林
- 不要用手指捏/挤爆

#### 后续观察

被蜱虫叮咬后 **1个月内** 观察有无：
- 游走性红斑（靶形皮疹 — 莱姆病特征）
- 发热、头痛、乏力
- 关节疼痛

出现任何症状 → 立即就医并告知医生有蜱虫叮咬史。

### 五、海洋生物蜇伤

| 生物 | 处理 |
|------|------|
| 水母 | 海水冲洗（不要用淡水！会触发刺细胞释放更多毒素）→ 醋浸泡30秒 → 刮除触须 |
| 海胆 | 热水浸泡（45℃）30-90分钟破坏毒素，不要强行拔刺 |
| 蓝环章鱼 | ⚠️ 剧毒！可致呼吸麻痹，需立即CPR+人工呼吸+送医 |
| 石头鱼/狮子鱼 | 热水浸泡45℃ 30-90分钟 |

## 第二部分：户外急救

### 一、户外急救通用原则

1. **安全第一**：先评估环境，保护自己
2. **系统检查**：ABC（气道→呼吸→循环）→ 全身从头到脚检查
3. **不要移动脊柱损伤患者**（除非环境危险）
4. **保暖**：户外伤者极易出现低体温
5. **记录**：受伤机制、时间、已做处理 → 交给救援人员

### 二、高海拔疾病

| 类型 | 海拔 | 症状 | 处理 |
|------|------|------|------|
| AMS（急性高山病） | >2500m | 头痛+至少一个：恶心/头晕/疲劳/失眠 | 停止上升、休息、布洛芬、乙酰唑胺 |
| HACE（高原脑水肿） | >3000m | 步态不稳、意识改变、严重头痛 | ⚠️**立即下撤！** 地塞米松 |
| HAPE（高原肺水肿） | >3000m | 静息时呼吸困难、干咳→粉红色泡沫痰 | ⚠️**立即下撤！** 硝苯地平 |

**高山病黄金法则**：任何在高海拔出现的症状都是高山病，直到证明不是！

### 三、雷电防护

**30/30原则**：看到闪电后30秒内听到雷声 → 危险！最后一声雷后等30分钟再出去。

**户外避雷**：
- 远离孤立大树、山顶、开阔水边
- 不要在空旷地站立（蹲下、双脚并拢、双手抱膝）
- 远离金属物体（登山杖、铁丝网、铁轨）
- 汽车内相对安全（法拉第笼效应）

### 四、野外失联/求救

**国际求救信号**：
- 声音/光：3短-3长-3短（SOS，莫尔斯码 ··· --- ···）
- 每分钟重复一次
- 地面标志：用石头/树枝摆出 **SOS** 或 **X**（需要救援）
- 火堆：三堆火成三角形（国际求救信号）
