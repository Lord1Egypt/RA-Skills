# 培训中心 - 本地知识库

## 页面定位

商家可添加本地文件夹路径，牛仔自动解析其中文档并同步到云端知识库（OSS），支持 PDF、Word、Excel、图片。

## 页面内容

### 文件夹路径列表

每条路径展示：

| 字段 | 说明 |
|------|------|
| 文件夹名称 | 用户自定义的名称 |
| 完整路径 | 本地文件夹的绝对路径 |
| 文件数量 | 该文件夹下的文档数 |
| 上次同步时间 | 最后一次同步完成的时间 |
| 同步状态 | 「已同步」（绿色）/ 「同步中」（蓝色旋转图标）|

### 操作按钮

| 操作 | 说明 |
|------|------|
| 打开 | 在系统文件管理器中打开对应文件夹路径 |
| 移除 | 二次确认弹窗 → 确认后从列表移除，牛仔停止同步该目录 |
| 添加文件夹路径 | 展开表单，填写名称和路径后确认添加，添加后自动开启同步云端 |

### 支持的文件格式
- PDF、Word(.doc/.docx)、Excel(.xls/.xlsx)、图片(jpg/png)、TXT

## Agent 引导要点

当用户说"添加知识库"、"同步文件"时，Agent 应：
1. 引导文案："您可以指定本地文件夹，牛仔会自动解析文档并同步到云端。"
2. 跳转到该页面
3. 跳转后话术："已打开本地知识库管理页面。您可以添加文件夹路径，牛仔会自动解析同步。"
