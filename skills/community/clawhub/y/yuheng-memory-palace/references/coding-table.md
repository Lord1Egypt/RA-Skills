## 文字→图像编码原则

1. **谐音优先**：取谐音，找对应的具象图像
2. **象形辅助**：像什么就画什么
3. **故事串联**：多个图像用动作串成故事
4. **夸张放大**：越小的东西越大越好

## 宫殿布局模板（玉衡定制）

```
/workspace 数字宫殿：

位置0（入口/门）：迎接新信息、用户进门
位置1（主桌/工作台）：核心任务进行中
位置2（工具架）：辅助工具、参考资源
位置3（白板）：分析过程、思考轨迹
位置4（档案柜）：已完成项目、记忆存档
位置5（窗户旁）：外部信息输入
位置6（角落书架）：知识沉淀
位置7（天花板上）：升华洞察
位置8（门外）：待处理队列
```

每个位置可以存放多个信息，用"叠加"的方式。
