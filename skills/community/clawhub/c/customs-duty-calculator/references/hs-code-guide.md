# HS Code Guide

The Harmonized System (HS) code is the most consequential single field in a customs entry. Get it wrong and duty, VAT, FTA eligibility, and regulatory compliance all break.

## Structure

- **Digits 1-2:** Chapter (96 chapters, e.g. 61 = knitted apparel)
- **Digits 3-4:** Heading (4-digit international level)
- **Digits 5-6:** Subheading (6-digit international level, the same in every WTO country)
- **Digits 7-10:** Country-specific. US uses HTS (10 digits). EU uses TARIC (10 digits). Different countries can classify the same product at digits 7-10 differently.

Always confirm the destination country's full local code; the 6-digit code is rarely enough for duty purposes.

## How to find a code

1. Start with a 1-line product description in plain language.
2. Open the destination country's tariff schedule (US: HTSUS at hts.usitc.gov; EU: TARIC at ec.europa.eu/taxation_customs; UK: trade-tariff.service.gov.uk; many others).
3. Use the official explanatory notes — descriptions often turn on materials, function, and processing level rather than common name.
4. Cross-check against at least one competitor's published declaration or a known import you've made.
5. If duty is significant or volume is high, file for a binding ruling.

## Binding rulings

- **US:** Customs Rulings Online Search System (CROSS) + CBP eRuling at rulings.cbp.gov.
- **EU:** Binding Tariff Information (BTI) issued by member-state customs and valid EU-wide.
- **UK:** Advance Tariff Ruling (ATaR) via HMRC.

Binding rulings take weeks to months but give legal certainty. Worth filing if duty stakes exceed several thousand dollars per year on a SKU.

## Common pitfalls

- **Material switch ambiguity.** Apparel with 50/50 cotton/polyester can land in different chapters; the dominant material by weight usually controls but rules vary.
- **Sets and kits.** A gift set with multiple items may be classified by the "essential character" item, or by each item separately. Wrong choice can shift duty significantly.
- **Parts vs assembled goods.** A semi-assembled product can fall under a different chapter than the finished good.
- **Use-based classification.** Some codes turn on stated use (e.g., medical vs general) — declared use must be supportable.

## Examples

| Plain description | Likely 6-digit code | Notes |
|---|---|---|
| Cotton T-shirt, knit | 6109.10 | Chapter 61, knitted |
| Cotton T-shirt, woven | 6205.20 | Chapter 62, woven |
| Wireless headphones | 8518.30 | Subject to Section 301 lists from China |
| Yoga mat (rubber/PVC) | 9506.91 or 3918.10 | Depends on material/use; can shift duty |
| Skincare serum | 3304.99 | Cosmetics category |
| Stainless steel water bottle | 7323.93 | Household article of iron/steel |

These are starting points only. Verify against the destination's tariff schedule before relying on any code.
