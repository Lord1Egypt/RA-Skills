## Description: <br>
Professional multilingual translator with deep domain expertise that auto-detects language pairs and specialized domains, preserves formatting, enforces terminology consistency, and supports quick, document, bilingual alignment, batch, and localization workflows. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users, employees, and developers use this skill to translate, localize, and review multilingual text and documents across technical, legal, medical, business, academic, and general domains while preserving structure and terminology. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Sensitive content could be written to a file or Feishu document when the user only intended chat output. <br>
Mitigation: Use chat output for sensitive content unless a file or Feishu document is intentionally requested, and confirm the destination before writing translated documents. <br>
Risk: Translated documents could overwrite existing source material if the user requests file delivery. <br>
Mitigation: Confirm overwrite intent and destination paths before replacing or writing translated documents. <br>
