# cn-number-formatter

数字格式化工具。支持千分位、科学计数法、百分比等格式。

## 使用方法

```bash
python3 scripts/format_num.py --number 1234567 --thousands
python3 scripts/format_num.py --number 0.123 --percent
python3 scripts/format_num.py --number 1000000 --scientific
```

纯Python标准库。
