# Polymarket Prompt Examples

## Market Discovery

```text
Search Polymarket for active crypto regulation markets.
Show me the highest-volume election markets on Polymarket.
List active sports prediction markets with the most liquidity.
Browse Polymarket events tagged for crypto and macro.
```

## Event Analysis

```text
Get the current odds, liquidity, and volume for this Polymarket event.
Explain this market's end date and resolution rules before I trade it.
Analyze whether the expiration mechanics favor one side.
Summarize the main risks and hidden resolution edge cases for this market.
```

## Positions And Market Orders

```text
Show my open Polymarket positions.
Prepare a market order summary for YES on this event, but do not execute.
Before placing a sell order, show me the exact token ID from my current position.
Redeem my winning Polymarket position and explain what will happen first.
```

## Important Constraints

```text
If this market does not support limit orders, explain the limitation clearly.
Do not place anything until you re-check the event details and token IDs.
Use Polymarket tools instead of web search for the actual market data.
```
