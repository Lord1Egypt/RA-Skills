# 🎬 video-script-writer

> 中文短视频脚本写作助手 — 让 AI 帮你写出高完播率的短视频脚本

## ✨ 功能

- **脚本生成**：根据主题/平台/时长，生成完整短视频脚本（分镜+台词+字幕+BGM建议）
- **爆款开头**：5种 Hook 公式（数据冲击/反常识/悬念/共鸣/痛点）
- **选题推荐**：基于垂直领域的选题建议
- **脚本优化**：分析已有脚本的完播率问题并改写
- **长切短**：将长文/长视频切成多个短视频脚本
- **8种脚本模板**：Vlog / 教程 / 探店测评 / 摩旅骑行 / 种草推荐 / 挑战实验 / 知识科普 / 通用
- **4大平台适配**：抖音 / 小红书 / B站 / 快手

## 📦 安装

将 `video-script-writer` 目录复制到 OpenClaw 的 skills 目录下即可使用。

## 🎯 使用方式

### 生成脚本
```
写一个抖音短视频脚本，主题是"新手第一次骑摩托车"，60秒
```

### 爆款开头
```
帮我写 5 个小红书视频的开头，主题是"北京周边骑行路线"
```

### 脚本优化
```
优化这个脚本的完播率：[贴入你的脚本]
```

### 长内容切片
```
把这篇文章切成 3 个抖音短视频脚本：[贴入内容]
```

## 📊 平台差异一览

| 维度 | 抖音 | 小红书 | B站 | 快手 |
