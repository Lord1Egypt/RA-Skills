Risk: Prompts and uploaded business datasets may be sent to the SkillBoss/HeyBoss API. <br>
Mitigation: Use a dedicated API key where possible and redact confidential fields before sending data. <br>
Risk: Generated HTML and JavaScript can contain incorrect logic or unsafe behavior. <br>
Mitigation: Review generated code before opening, sharing, or deploying it. <br>
