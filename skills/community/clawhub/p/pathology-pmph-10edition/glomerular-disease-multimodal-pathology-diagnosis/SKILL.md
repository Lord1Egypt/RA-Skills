# 肾小球疾病多模态病理诊断流程

## 适用条件
- 已获取肾穿刺活检组织
- 临床怀疑肾小球疾病（如血尿、蛋白尿、肾功能异常）
- **不适用于**：单纯肾小管或间质疾病无肾小球受累者；急性出血或凝血障碍患者（肾穿刺禁忌）

## 执行步骤

### 1. 光镜检查（必须包含以下染色）
- **HE染色**：评估肾小球基本结构、细胞增生、坏死、玻璃样变
- **PAS染色**：清晰显示肾小球基膜和系膜基质
- **PASM染色**：更精确评估基膜厚度、断裂及钉突形成
- **Masson三色染色**：识别胶原纤维沉积及特殊蛋白性物质（如免疫复合物）

### 2. 免疫荧光检查
- 检测IgG、IgM、IgA及补体C3、C1q、C4
- 判读荧光模式：
  - **颗粒状沉积** → 提示免疫复合物介导肾炎（如急性弥漫性增生性肾炎、Ⅱ型RPGN）
  - **连续线性沉积** → 提示抗GBM抗体病（如Ⅰ型RPGN、Goodpasture综合征）
  - **阴性** → 考虑pauci-immune型病变（如Ⅲ型RPGN）

### 3. 透射电镜检查
- 观察足突融合、基膜断裂、系膜插入等超微结构改变
- 定位电子致密沉积物：
  - **上皮下**（subepithelial）：如膜性肾病、“驼峰状”沉积（急性肾炎）
  - **内皮下**（subendothelial）：如狼疮性肾炎
  - **系膜区**：如IgA肾病、系膜增生性肾炎
- 记录沉积物形态（块状、驼峰状等）及分布特征

### 4. 综合诊断
- 整合光镜、免疫荧光、电镜三方面结果
- 结合临床信息（如年龄、感染史、补体水平）进行最终病理分类
- 输出格式：完整病理报告，包含各模态发现及疾病命名（例如：“急性弥漫性增生性肾小球肾炎，免疫复合物介导”）
