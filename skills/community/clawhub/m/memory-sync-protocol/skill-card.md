## Description: <br>
Synchronize durable behavior or preference changes across TOOLS.md, MEMORY.md, AGENTS.md, and memory/YYYY-MM-DD.md with one consistent workflow. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent operators use this skill to keep durable memory, routing, governance, and daily log files synchronized when user preferences or operating rules change. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can cause durable repository changes to memory and governance files when invoked. <br>
Mitigation: Invoke it only for intended long-term preferences or operating rules, review changes to TOOLS.md, MEMORY.md, AGENTS.md, and memory logs, and avoid storing secrets or temporary preferences. <br>
