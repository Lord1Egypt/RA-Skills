# 空腹血糖与OGTT糖尿病诊断分级

## 何时使用
- 已获得空腹血糖（FBG）检测结果，需判断是否为正常、IFG、高血糖或低血糖；
- 患者符合OGTT适应证（如疑似糖尿病、有家族史、妊娠期妇女、不明原因肾病/视网膜病变等），且已完成标准75g OGTT；
- 需结合《中国2型糖尿病防治指南（2020年版）》对血糖状态进行临床分级和病因提示。

## 执行步骤

### 第一步：评估空腹血糖（FBG）
1. **确认前提条件**：
   - 患者处于空腹状态（≥8小时未进食）；
   - 使用静脉血浆或血清样本；
   - 采用标准化检测方法（如葡萄糖氧化酶法或己糖激酶法）。
2. **判断血糖状态**：
   - **高血糖症**：FBG ≥ 7.0 mmol/L
     - 轻度：7.0 < FBG ≤ 8.4 mmol/L
     - 中度：8.4 < FBG ≤ 10.1 mmol/L
     - 重度：FBG > 10.1 mmol/L
   - **空腹血糖受损（IFG）**：FBG 6.1～<7.0 mmol/L（静脉血浆），且未达糖尿病诊断标准。
   - **低血糖症**：
     - 非糖尿病人群：FBG < 2.8 mmol/L；
     - 接受降糖治疗的糖尿病患者：FBG < 3.9 mmol/L，进一步分级：
       - 1级：3.0 ≤ 血糖 < 3.9 mmol/L
       - 2级：血糖 < 3.0 mmol/L
       - 3级：出现意识或躯体改变，需他人帮助的严重事件。

### 第二步：执行并解读OGTT（如适用）
1. **确认OGTT前提条件**：
   - 空腹8–14小时；
   - 无急性疾病或应激状态；
   - 已停用影响糖代谢药物≥3天；
   - 口服75g无水葡萄糖（儿童1.75g/kg，最大75g）于5分钟内服完。
2. **采血时间点**：空腹、服糖后0.5h、1h、2h、3h。
3. **诊断标准（静脉血浆）**：
   - **糖尿病**（满足任一）：
     - 典型症状 + FPG ≥ 7.0 mmol/L；
     - 典型症状 + 随机血糖 ≥ 11.1 mmol/L；
     - OGTT 2h PG ≥ 11.1 mmol/L（症状不典型者需另一天重复确认）。
   - **糖耐量受损（IGT）**：FPG < 7.0 mmol/L 且 2h PG 为 7.8～<11.1 mmol/L。
   - **正常反应**：2h PG < 7.8 mmol/L，3h恢复至空腹水平。
4. **识别特殊OGTT曲线类型**：
   - **平坦型**：血糖上升不明显，2h仍低 → 胰岛β细胞瘤、肾上腺皮质功能减退；
   - **储存延迟型**：服糖后血糖急剧升高（>11.1 mmol/L），2h反低于空腹 → 胃切除术后、严重肝损伤；
   - **功能性低血糖**：2–3h后低血糖 → 特发性低血糖；
   - **肝源性低血糖**：FPG低、高峰提前且高、2h PG仍高 → 广泛肝损伤。

### 第三步：综合判断与输出
- 结合FBG与OGTT结果，输出最终诊断分类；
- 根据血糖模式提示可能的生理性或病理性病因；
- 注意样本类型（静脉血浆/全血/毛细血管）对阈值的影响（详见参考文档）。

> **注意**：本技能不适用于非空腹采样、毛细血管全血未经校正结果、已确诊典型糖尿病患者或急性应激状态下的检测。
