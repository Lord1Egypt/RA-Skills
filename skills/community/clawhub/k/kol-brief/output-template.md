# Output Template — KOL Brief

Deliver the brief in this exact structure. Target length: 1–2 pages. Every section header stays; remove nothing, add nothing.

## 1. Campaign Summary (3–4 lines)

```
Brand / Product: [name + one-line description]
Creator: [@handle, niche, follower count]
Campaign goal: [GMV target / launch / awareness / LIVE]
Campaign window: [start–end dates]
```

## 2. Positioning Angle (1 sentence)

The single angle this creator should communicate, written so it could be read aloud. Example: "The FDA-cleared LED mask that fits a real nighttime routine — no cords, no salon appointments."

## 3. Top 3 Talking Points (claim + proof each)

```
1. [Claim] — Proof: [demo moment / stat / comparison the creator can show or cite]
2. [Claim] — Proof: [...]
3. [Claim] — Proof: [...]
```

## 4. Deliverables

| Item | Spec |
|---|---|
| Content type & count | [e.g., 2 short videos ≥30s] |
| Posting window | [exact date range] |
| Required tags | [#hashtags, @mentions] |
| Link placement | [TikTok Shop anchor / bio link / LIVE pin] |
| Draft review | [draft due date, review turnaround] |
| Revision policy | [e.g., 1 revision round within 5 days] |

## 5. Dos (max 7)

Conversion levers: first-3-seconds product visibility, on-camera hero-feature demo, verbal discount code, etc. Each one actionable on camera.

## 6. Don'ts (max 7)

Compliance and brand protection: banned claims verbatim, competitor mentions, disclosure requirements, price-promise limits.

## 7. Compensation & Rights

| Item | Terms |
|---|---|
| Structure | [flat / commission / hybrid with numbers] |
| Payment trigger & schedule | [e.g., 50% on confirmation, 50% net-7 after posting] |
| Usage rights | [ad re-use allowed? duration? whitelisting?] |
| Exclusivity | [category exclusivity window, if any] |

## 8. Timeline

```
[Date] Product ships → [Date] Draft due → [Date] Feedback returned → [Date window] Posting → [Date] Payment
```

## 9. Contact & FAQ pointer

One named contact with response-time expectation, plus the FAQ block for sampling, approval, and link setup questions.

## Quality gates before delivery

- One positioning angle only, matched to this creator's audience
- Exactly 3 talking points, each with attached proof
- All deliverable specs filled with dates and numbers — no "TBD"
- Dos ≤7, don'ts ≤7
- Compensation table complete including usage rights
- Total length ≤2 pages, no internal notes left in
