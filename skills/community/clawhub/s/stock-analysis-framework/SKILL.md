## 二、基本面分析（第①维）

### 核心指标

| 维度 | 指标 | 计算公式 | 判断标准 |
|------|------|----------|----------|
| 盈利能力 | 毛利率 | (营收-成本)/营收 | >行业均值=好 |
| 盈利能力 | 净利率 | 净利润/营收 | 持续提升=优 |
| 盈利能力 | ROE | 净利润/净资产 | >15%=优秀 |
| 成长性 | 营收增速 | (本期-上期)/上期 | >20%=高增长 |
| 成长性 | 利润增速 | (本期-上期)/上期 | >营收增速=好 |
| 估值 | PE | 股价/每股收益 | 对比行业/历史分位 |
| 估值 | PB | 股价/每股净资产 | <1=破净 |
| 财务健康 | 资产负债率 | 总负债/总资产 | <50%=健康 |
| 财务健康 | 经营现金流 | 正/负 | 长期为正=健康 |

### 分析流程

1. 查最新季报/年报（营收、利润、毛利率）
2. 看同比增速（同比去年同期的变化）
3. 对比行业平均（在行业中的位置）
4. 算估值水平（PE/PB在历史分位）
5. 看财务风险（负债、应收账款、现金流）

## 三、技术面分析（第②维）

### 核心技术指标

| 指标 | 参数 | 用法 | 信号 |
|------|------|------|------|
| MA均线 | MA5/10/20/60 | 多头=上涨，空头=下跌 | 金叉/死叉 |
| RSI | 周期14 | >70超买，<30超卖 | 背离=反转 |
| MACD | 12/26/9 | 0轴上下 | 金叉/死叉+柱体 |
| KDJ | 9/3/3 | K/D交叉 | 超80/超20 |
| BOLL | 20/2 | 价格在轨道内 | 碰上下轨=反转 |
| 成交量 | 配合价格 | 放量突破=有效 | 缩量=动能不足 |

### K线形态速查

#### 看涨形态
- **W底**：二次探底不破前低 → 突破颈线买入
- **头肩底**：左肩<头<右肩 → 突破颈线买入
- **多方炮**：阳阴阳 → 看涨
- **启明星**：长阴+小线+长阳 → 反转
- **看涨吞没**：阳线吞没前阴 → 强势反转

#### 看跌形态
- **M顶**：两次冲高不过 → 跌破颈线卖出
- **头肩顶**：左肩>头>右肩 → 跌破颈线卖出
- **空方炮**：阴阳阴 → 看跌
- **黄昏星**：长阳+小线+长阴 → 反转下跌
- **看跌吞没**：阴线吞没前阳 → 强势下跌

### 量价关系

| 价格 | 成交量 | 信号 |
|:----:|:------:|:----|
| 上涨 | 放大 | 🟢 健康上涨 |
| 上涨 | 缩小 | ⚠️ 动能不足 |
| 下跌 | 放大 | 🔴 恐慌下跌 |
| 下跌 | 缩小 | 🟡 下跌趋缓 |

## 四、消息面分析（第③维）

### 消息来源分类

| 类型 | 来源 | 优先级 |
|------|------|:------:|
| 公司公告 | 巨潮资讯网 | 🚨 最高 |
| 季报/年报 | 交易所披露 | 🚨 最高 |
| 行业政策 | 政府官网/新闻 | 🔴 高 |
| 媒体报道 | 财联社/证券时报 | 🟡 中 |
| 股吧/雪球 | 舆情情绪 | 🟢 参考 |

### 消息影响评估

```
利好程度：重大利好 > 一般利好 > 中性 > 一般利空 > 重大利空

利好出尽 = 高位出利好 → 警惕兑现
利空出尽 = 低位出利空 → 可能见底
```

### 消息面与股价关系

```
突发利好 → 已涨停买不进 → 等开板
突发利空 → 赶紧看持仓 → 决定去留
报表超预期 → 仔细看细节 → 营收or利润？
报表不及预期 → 看是暂时的还是趋势性的
```

## 五、资金面分析（第④维）

### 核心数据

| 数据 | 含义 | 获取来源 |
|------|------|----------|
| 主力净流入 | 超大单+大单合计 | 东方财富/同花顺 |
| 北向资金 | 外资每天买卖 | 沪港通/深港通 |
| 龙虎榜 | 涨停跌停的买卖席位 | 交易所 |
| 融资余额 | 杠杆资金 | 交易所 |
| 换手率 | 交易活跃度 | 行情软件 |

### 资金流向判断

```
主力流入 + 股价上涨 = 🟢🟢 强烈看多
主力流入 + 股价下跌 = 🟡 主力在吸筹
主力流出 + 股价上涨 = 🔶 拉高出货
主力流出 + 股价下跌 = 🔴🔴 强烈看空

北向持续流入 = 中长期看好
龙虎榜机构买入 = 机构看好
```

```
别人贪婪我恐惧 → 涨停潮时减仓
别人恐惧我贪婪 → 跌停潮时找机会
但需配合基本面和资金面，不可盲目逆向
```

## 九、操作策略框架

### 根据趋势
```
上涨趋势 → 持股为主，回踩MA20加仓
下跌趋势 → 减仓，不轻易抄底，等右侧信号
震荡市   → 高抛低吸，支撑位买压力位卖
```

### 根据资金
```
主力持续流入 + 上涨 = 持有
主力持续流出 + 下跌 = 离场
低位放量 = 启动 🟢
高位放量 = 出货 🔴
```

### 根据消息
```
低位利好 = 真涨 ✅
高位利好 = 利好出尽 ⚠️
突发利空 = 先走为敬 🏃
预期内的利空 = 利空出尽 ✅
```

### 止损铁律
| 条件 | 操作 |
|:----|:-----|
| 亏损达-8% | 无条件止损 |
| 跌破MA60 | 趋势破位，清仓 |
| 大盘系统性风险 | 仓位降至5成以下 |
