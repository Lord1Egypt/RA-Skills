# 复方磺胺甲唑（SMZco）的协同抗菌机制与应用

## 何时使用
- 需治疗由以下敏感病原体引起的感染：
  - 泌尿道感染（大肠埃希菌、变形杆菌、克雷伯菌）
  - 呼吸道感染（肺炎链球菌、流感嗜血杆菌）
  - 肠道感染（志贺菌、霍乱弧菌、伤寒沙门菌）
  - 机会性感染（卡氏肺孢子虫肺炎、弓形虫病预防）
  - 皮肤软组织感染（MRSA）
- 细菌对磺胺甲噁唑（SMZ）和甲氧苄啶（TMP）均敏感
- 患者无新生儿期、妊娠期、严重肝肾功能障碍或G6PD缺乏症

## 如何执行
1. **确认复方组成**：SMZco由SMZ与TMP按5:1比例制成。
2. **理解协同机制**：
   - SMZ抑制细菌二氢蝶酸合酶，阻断PABA转化为二氢蝶酸
   - TMP抑制细菌二氢叶酸还原酶，阻断二氢叶酸还原为四氢叶酸
   - 二者实现“双重阻断”，彻底中断细菌四氢叶酸合成通路
3. **验证给药比例合理性**：
   - 体外最佳抗菌浓度比为TMP:SMZ = 1:20
   - 因TMP脂溶性和分布容积大于SMZ，按1:5给药后血药峰浓度比可达1:20～1:30，符合最佳比例
4. **评估临床适用性**：
   - 即使对单用磺胺耐药的大肠埃希菌、伤寒沙门菌或志贺菌属，SMZco仍可能有效
   - 抗菌活性为单药等量应用时的数倍至数十倍，部分情况下呈杀菌作用
5. **监测安全性**：
   - 不良反应包括过敏、血液系统抑制、肝肾损害等
   - 治疗期间需定期监测血常规及肝肾功能

> 注意：禁用于新生儿和孕妇；对已对TMP或SMZ单药高度耐药的菌株可能无效。
