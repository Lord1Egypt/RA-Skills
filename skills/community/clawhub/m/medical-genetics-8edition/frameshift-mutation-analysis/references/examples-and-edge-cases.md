# 移码突变示例与边界情况

## 典型示例

### 正常序列
```
...UAC-AGU-CCU-ACA-GAA-AAC-GCU...
→ Tyr-Ser-Pro-Thr-Glu-Asn-Ala...
```

### 插入1个A（在AGU后）
```
...UAC-AGA-AUC-CUA-CAG-AAA-CGC-U...
→ Tyr-Arg-Ile-Leu-Gln-Lys-Arg...
```
所有后续氨基酸改变，且可能提前终止。

### 缺失1个G（在AGU中）
```
原：...UAC-AGU-CCU...
突变后：...UAC-GUC-CUA...
→ Tyr-Val-Leu...
```
同样导致读码框偏移。

## 整码突变（非移码）
- **插入3个碱基**位于两个密码子之间：
  ```
  ...UAC|AGU... → ...UAC-XXX-AGU...
  ```
  仅增加一个氨基酸，读码框不变。

- **插入3个碱基**跨密码子内部：可能影响两个相邻密码子，但不引起全局移码。

## 复合突变
- 先插入1个碱基，再缺失1个碱基（位置不同）：
  - 仅两突变点之间的序列读码框改变。
  - 突变点之后序列恢复正常读码框。

## 致病性关联
- 移码突变常见于遗传病（如囊性纤维化、杜氏肌营养不良）。
- 提前终止常触发无义介导的mRNA降解（NMD），导致蛋白表达缺失。
