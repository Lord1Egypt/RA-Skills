# 游戏机制设计指南

## 搜证机制

### 基础搜证：行动点（AP）系统

每轮玩家获得固定AP，消耗AP获取线索。

**设计要点**：

| 要素 | 建议 |
|------|------|
| 每轮AP数量 | 3-5点 |
| 普通线索消耗 | 1 AP |
| 关键线索消耗 | 2 AP |
| 特殊行动消耗 | 1-3 AP |

**AP分配表模板**：

```markdown
## 搜证规则

每轮每位玩家获得 **3点AP**。

### 可搜索地点与线索
| 地点 | 可用AP | 可获得线索 |
|------|--------|-----------|
| 书房 | 1AP | L001、L002 |
| 厨房 | 1AP | L003 |
| 卧室 | 1AP | L004、L005 |
| 阁楼 | 2AP | L006（关键线索） |

### 特殊行动
| 行动 | 消耗 | 效果 |
|------|------|------|
| 搜身 | 2AP | 查看某人持有的线索 |
| 偷听 | 1AP | 听到一段私聊内容 |
| 质问 | 1AP | 强制某人回答一个问题 |
```

### 进阶搜证：条件解锁

某些线索需要满足条件才能获取。

**条件类型**：
- **道具条件**：需要先获得钥匙/密码才能进入
- **信息条件**：需要先知道某个信息才能发现
- **合作条件**：需要两人同时在场才能打开

```markdown
## 条件搜证

### 阁楼（需要钥匙）
- 前置条件：在书房找到阁楼钥匙（L002）
- 消耗：2AP
- 可获得：L006、L007

### 保险柜（需要密码）
- 前置条件：获得密码提示（L005），推理出密码
- 消耗：1AP
- 可获得：L008（核心证据）

### 双人搜证（需要两人合作）
- 前置条件：两名玩家同时选择搜证花园
- 消耗：每人1AP
- 可获得：L009（只有合作才能发现的线索）
```

## 投票机制

### 标准投票：指认凶手

**时间点**：通常在最后一轮讨论结束后

**投票方式**：

```markdown
## 投票规则

### 投票时间
第三轮讨论结束后，进入投票环节。

### 投票方式
- 每人一票，匿名投票
- 不能投自己
- 票数最多者被指认为凶手

### 平票处理
- 如出现平票，平票者各有一分钟自辩
- 自辩后重新投票
- 如仍平票，则凶手逃脱
```

### 多重投票：抉择投票

用于情感本或有多重结局的剧本。

```markdown
## 抉择投票

### 投票内容
除了指认凶手外，还需对以下问题投票：

1. 是否公开死者的真实身份？
   - 是 / 否

2. 是否原谅凶手？
   - 原谅 / 不原谅

3. 选择哪个结局？
   - 结局A：所有人和解
   - 结局B：真相被永远埋藏

### 结局触发
根据投票结果，触发不同结局。
```

## 私聊机制

### 私聊时间分配

**标准配置**：

```markdown
## 私聊规则

### 私聊时间
- 第一轮讨论后：私聊时间10分钟
- 第二轮讨论后：私聊时间10分钟
- 每人最多进行2次私聊
- 每次私聊最多与1人进行

### 私聊规则
- 私聊内容可以撒谎
- 私聊中获得的信息可以公开，也可以隐瞒
- 不能在私聊中使用搜证行动
```

### 私聊引导

给玩家一些私聊的引导问题：

```markdown
## 私聊建议

### 你可以问对方
- "案发时你在哪里？"
- "你和死者是什么关系？"
- "你知道死者的什么秘密？"
- "你觉得谁最可疑？"

### 你可以选择
- 诚实地分享你掌握的信息
- 撒谎隐瞒你的秘密
- 试探对方是否在撒谎
- 与对方结盟
```

## 公聊机制

### 公聊时间分配

```markdown
## 公聊规则

### 公聊轮次
- 第一轮：自我介绍 + 案发前情况（15分钟）
- 第二轮：分享线索 + 质疑（20分钟）
- 第三轮：最终推理 + 指控（15分钟）

### 公聊规则
- 每人每轮至少发言一次
- 可以选择隐瞒信息，但不能说"我什么都不知道"
- 被直接质问时，可以选择回答或拒绝（但拒绝会被视为可疑）
```

## 特殊机制

### 举报机制

```markdown
## 举报规则

- 玩家可以举报其他玩家"说谎"
- 举报需要消耗2AP
- 如果举报成功（对方确实在关键信息上说谎），获得额外线索
- 如果举报失败，自己失去下一轮的1AP
```

### 保护机制

```markdown
## 保护规则

- 某些角色有"保护"技能
- 可以选择保护一名玩家不被投票出局
- 每局只能使用一次
- 保护成功后，被保护者免疫一次投票
```

### 交换机制

```markdown
## 交换规则

- 玩家之间可以交换线索卡
- 交换是双向的，双方各出一张线索卡
- 交换后不能反悔
- 可以用假线索交换（但被发现有惩罚）
```
