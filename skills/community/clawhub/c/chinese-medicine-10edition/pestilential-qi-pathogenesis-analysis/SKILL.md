# Pestilential Qi Pathogenesis Analysis

## When to Use
- 疾病起病急骤、进展迅速、死亡率高；
- 多人出现高度相似症状，呈现群体性发病；
- 怀疑为烈性传染病（如霍乱、白喉、疫痢、天花等）；
- 需区分六淫外感与疳气致病。

## Execution Steps

### 第一步：确认疳气致病特征
1. **评估发病特点**：
   - 是否潜伏期极短，甚至“触之即病”？
   - 是否病情危重、发展迅猛？
2. **判断传染性**：
   - 是否通过空气、饮食、接触或虫媒传播？
   - 是否出现家庭或社区聚集性感染？
3. **验证“一气一病”原则**：
   - 同一疫病患者症状是否基本一致？
   - 是否符合《素问·刺法论》“五疫之至，皆相染易，无问大小，病状相似”？

### 第二步：识别影响疫病流行的四大因素
1. **气候因素**：检查是否存在久旱、酷热、淫雨、洪涝、湿雾或山岚瘴气等反常气候。
2. **环境与饮食因素**：
   - 水源或空气是否污染？
   - 是否存在动物尸体未掩埋、秽物腐败或食物不洁？
3. **预防措施因素**：
   - 是否已隔离患者？
   - 易感人群是否服用预防药物？
   - 是否注重调摄起居、保养正气？
4. **社会因素**：
   - 是否处于战乱、天灾、贫困或社会动荡时期？
   - 公共卫生体系是否健全？

### 第三步：采取应对措施
- 立即隔离患者，阻断传播途径；
- 对易感人群实施预防性干预；
- 结合气候与社会背景制定防控策略；
- 强调“正气存内，邪不可干”，提升个体抵抗力。

> **注意**：疳气“非风、非寒、非暑、非湿”，纯属外感烈性病邪，无内生形式，切勿与六淫混淆。
