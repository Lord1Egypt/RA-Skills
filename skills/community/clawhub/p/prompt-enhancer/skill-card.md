## Description: <br>
Automatically rewrites rough user inputs into optimized, structured prompts for dramatically better AI responses. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[gyaan-dub](https://clawhub.ai/user/gyaan-dub) <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and developers use this opt-in skill to turn rough requests prefixed with p: or prompt: into clearer prompts before the agent answers. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The enhanced prompt may add assumptions that are useful but not exactly what the user intended. <br>
Mitigation: Review the displayed enhanced prompt before relying on the response and ask the agent to revise it if it changes the intended request. <br>
