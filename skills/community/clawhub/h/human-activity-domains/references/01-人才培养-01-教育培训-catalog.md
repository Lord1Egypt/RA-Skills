## 教育培训任务清单

教育培训按子域组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：课程设计 → 教学实施 → 学员管理 → 效果评估

---

### 1 课程设计

| ID | 任务类型 | 说明 | 依赖 | 元操作映射 |
|----|----------|------|------|----------|
| T1-1 | 需求调研 | 调研学员培训需求、岗位能力要求、市场人才需求 | 无（入口） | S→C |
| T1-2 | 目标设定 | 基于需求分析设定培训目标、能力指标、考核标准 | T1-1 | C |
| T1-3 | 内容开发 | 设计课程大纲、知识点体系、案例素材、练习题目 | T1-2 | C→A |
| T1-4 | 教材编制 | 编写教材、课件、实操手册、参考资料 | T1-3 | A→O |

## 依赖拓扑摘要

### 一、域内链路

**课程设计链路**: T1-1 → T1-2 → T1-3 → T1-4
**教学实施链路**: T2-1 → T2-2 → T2-3 → T2-4
**学员管理链路**: T3-1 → T3-2 → T3-3 → T3-4
**效果评估链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整培训链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T4-1 → T4-2 → T4-3 → T4-4

**学员管理并行链路**: T3-1 ‖ T1-1（招生与需求调研可并行）

**反馈改进链路**: T4-4 → T1-3（课程改进反馈到内容开发）

更多组合根据具体任务动态推导。
