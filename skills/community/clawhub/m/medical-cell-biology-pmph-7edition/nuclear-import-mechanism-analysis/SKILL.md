# 核定位信号介导入核机制分析

## 适用场景
- 亲核蛋白定位验证
- 外源蛋白核靶向设计
- NLS功能突变分析
- 核孔转运机制研究

## 执行步骤
1. **识别NLS特征**：
   - 搜索富含Lys/Arg的4–8个氨基酸序列（如SV40 T抗原127–132位）
   - 确认关键残基（如Lys-128）是否保守

2. **验证NLS功能**：
   - 截短实验：删除候选序列，观察是否滞留胞质
   - 融合实验：将候选序列连接胞质蛋白，观察是否入核
   - 点突变：关键Lys突变为Thr/Asn，验证功能丧失

3. **评估转运条件**：
   - 是否存在输入蛋白α/β和Ran-GTP系统？
   - 蛋白尺寸是否 >9nm（需主动运输）？

4. **输出结论**：
   - 确认某序列为功能性NLS，并预测其在其他蛋白中的引导能力
   - 确保亲核蛋白精准入核，维持基因表达调控秩序
