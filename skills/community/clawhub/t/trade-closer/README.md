# Trade Closer - 开发信技能

> ✉️ **外贸开发信生成** | 3种模板 + A/B测试 + 多语言支持

## 功能介绍

Trade Closer 是外贸获客闭环的第三步，为筛选出的优质客户生成高回复率开发信。

### 核心能力

- ✍️ **3种邮件模板** - 直接型/故事型/价值型
- 🧪 **A/B主题行测试** - 智能评分优化
- 📅 **7天触达计划** - 科学间隔递增
- 🌐 **多语言支持** - EN/ES/FR/DE等
- 📊 **回复率预测** - 发送前效果预估

## 邮件模板类型

| 模板类型 | 适用场景 | 回复率基准 |
|----------|----------|------------|
| **直接型** | 已了解竞品的客户 | 18-22% |
| **故事型** | 决策链长的客户 | 22-28% |
| **价值型** | 注重ROI的采购者 | 25-30% |

## 7天触达计划

```
Day 1: 首次邮件 - 引起注意
Day 2: 价值内容 - 提供干货
Day 4: 案例分享 - 社会证明
Day 6: 限时优惠 - 紧迫感
Day 7: 最后提醒 - 促使行动
```

## 技术规范

- **平台**: Coze / Caw
- **AI模型**: GPT-4o
- **输出格式**: HTML邮件 + 纯文本版本
