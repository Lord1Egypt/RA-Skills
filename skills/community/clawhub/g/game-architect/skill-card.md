## Description: <br>
Game Architect supports game system architecture planning for combat, skills, AI, UI, multiplayer, narrative, and scene systems with paradigm selection guides, system-specific references, and mixing strategies. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Yuki001](https://clawhub.ai/user/Yuki001) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and game teams use this skill to choose architecture paradigms and plan game systems across requirements, technical design, and implementation planning. It can serve as a domain reference alongside workflow skills or as a standalone planning workflow. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The standalone workflow can produce or update architect/*.md planning files that may be used as implementation direction. <br>
Mitigation: Review generated planning documents before using them to guide implementation. <br>
