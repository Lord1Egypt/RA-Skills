# 散户决胜法 · 案例库

## 案例 1：Polymarket

**评估：** 🟢 可以

```
赛道：预测市场 ✅（VC四大收窄方向之一）
链上可验证性：5/5 — 交易量、用户数、市场数量全部链上公开
VC筹码结构：已TGE，解锁接近完成，筹码分散
散户优势位：⭐⭐⭐ 用产品就是做研究

结论：
1️⃣ 参与吗：可以，但以用户身份参与而非投机
2️⃣ 怎么参与：注册账号，用 USDC 参与预测。培养信息定价能力
3️⃣ 赢在哪：每一次预测都是认知训练；输在：高频预测被手续费吃掉
```

## 案例 2：Trace Finance

**评估：** 🔴 回避

```
赛道：稳定币/支付 ✅
链上可验证性：1/5 — B2B API 基础设施，无链上数据可追踪
VC筹码结构：❌ 没有 Token — 纯 equity 公司
散户优势位：无

结论：
1️⃣ 参与吗：不能。没有 Token、没有空投、没有链上数据
2️⃣ 怎么参与：没有入口
3️⃣ 赢在哪：无参与通道；输在：好公司≠好投资
```

## 案例 3：Uniswap

**评估：** 🟢 可以

```
赛道：DEX/DeFi ✅（成熟赛道）
链上可验证性：5/5 — TVL、交易量、费用收入全部链上
VC筹码结构：已完全解锁，UNI 流通量 100%
散户优势位：⭐⭐⭐ 数据完全透明 + 可以用产品

结论：
1️⃣ 参与吗：适合作为长期仓位配置
2️⃣ 怎么参与：现货买入 UNI。挂单做市赚手续费。使用 App 了解市场
3️⃣ 赢在哪：全网最透明的协议之一；输在：监管不确定性，治理分歧
```

## 案例 4：某未 TGE 的 L2 项目

**评估：** 🟡 观望

```
赛道：L2 基础设施 ✅
链上可验证性：3/5 — TVL 和桥数据公开，但收入模型不清晰
VC筹码结构：⚠️ 大量 VC 筹码尚在锁仓，解锁压力未知
散户优势位：🟡 可能有空投，但确定性低

结论：
1️⃣ 参与吗：暂不参与。VC 解锁时间表不透明
2️⃣ 怎么参与：交叉链桥使用（如果未来有空投）；等 TGE 后 6 个月看情况
3️⃣ 赢在哪：L2 赛道长期看好；输在：解锁砸盘可能很惨
```
