# 肾损伤影像学检查适应证判断

## 何时使用
当患者满足以下任一临床情形时，应使用本技能评估是否需行肾影像学检查：
- 腹部、侧腹、下胸部或背部遭受穿透性或钝性创伤；
- 出现肉眼血尿或镜下血尿（尿中红细胞 >5 个/高倍视野）；
- 儿科患者出现镜下血尿；
- 创伤后出现低血压（收缩压 <90 mmHg）伴镜下血尿。

> **注意**：仅适用于血流动力学相对稳定的患者。若患者血流动力学不稳定，应优先抢救，暂缓影像检查。

## 执行步骤

1. **确认五大适应证之一是否满足**（满足任一即应安排影像学检查）：
   - 穿透性创伤（腹部、侧腹或下胸部入口/出口伤），且患者血流动力学稳定；
   - 钝性创伤伴明显加速性/减速性机制（如高速机动车事故急停、高处坠落）；
   - 钝性创伤伴肉眼血尿；
   - 钝性创伤伴镜下血尿（红细胞 >5 个/高倍视野）且在评估或复苏期间出现低血压（收缩压 <90 mmHg）；
   - 儿科患者镜下血尿（红细胞 >5 个/高倍视野）。

2. **选择首选检查方法**：
   - **对比剂增强CT（CT平扫+增强）** 为金标准，可清晰显示肾实质裂伤、尿外渗、血肿范围及肾血供。

3. **根据临床需要选择辅助检查**：
   - **CT尿路成像（CTU）**：用于评估对比剂排泄及尿外渗；
   - **CT血管成像（CTA）**：用于检测肾动脉损伤、动静脉瘘或假性动脉瘤；
   - **超声**：用于快速筛查血肿、尿外渗及对侧肾情况（但无法可靠评估肾蒂血管）；
   - **MRI**：对血肿特征显示更优，但急诊场景较少使用。

4. **注意事项**：
   - 即使症状轻微（如仅有轻度腰痛或微量血尿），只要存在上述创伤机制，仍需影像学排查——因临床症状与实际损伤严重程度常不一致。

## 输出结果
明确输出：
- 是否符合影像学检查指征（是/否）；
- 推荐的首选检查方式（通常为增强CT）。
