# ADC作用机制与旁观者效应分析

## 何时使用
- 肿瘤表达ADC靶向的特定表面抗原（如HER2、CD30、TROP-2等）
- 考虑使用含可裂解连接子（如Val-Cit）的ADC治疗抗原表达异质性的实体瘤
- 需评估ADC是否可能通过旁观者效应杀伤邻近抗原阴性肿瘤细胞
- 需预测ADC相关器官特异性毒性（如血液、肝、肺、神经毒性）

## 执行步骤

1. **确认ADC组成**：
   - 抗体：识别所用单克隆抗体及其靶向抗原（如抗HER2）
   - 有效载荷：确定细胞毒性药物类型（如MMAE、DXd、刺孢霉素、SN-38）
   - 连接子：判断是否为可裂解型（如Val-Cit、二肽类）或不可裂解型（如硫醚）

2. **验证适用前提**：
   - 肿瘤细胞必须表达ADC所靶向的抗原
   - 若期望旁观者效应，连接子必须为可裂解型

3. **分析作用机制**：
   a) 抗体与肿瘤细胞表面抗原特异性结合
   b) ADC-抗原复合物经内吞进入细胞
   c) 在溶酶体内，可裂解连接子被组织蛋白酶B切割或在酸性环境中水解
   d) 活性细胞毒性药物释放，破坏微管或DNA，诱导细胞死亡

4. **评估旁观者效应可能性**：
   a) 可裂解连接子ADC可在肿瘤微环境（低pH、高蛋白酶活性）中提前释放药物
   b) 游离药物通过被动扩散进入邻近抗原阴性肿瘤细胞
   c) 此效应对抗原表达异质性肿瘤（如HER2低表达乳腺癌）尤为关键

5. **识别毒性风险**：
   - 根据有效载荷类型预测器官特异性不良反应（如MMAE→周围神经病变；DXd→间质性肺病）
   - 注意血液循环中连接子不稳定可能导致脱靶毒性

> **注意**：不适用于抗原完全阴性且无旁观者效应机制的肿瘤。
