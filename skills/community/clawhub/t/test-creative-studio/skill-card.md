## Description: <br>
Headless creative production studio for AI agents that generates and edits images, creates video and audio assets, and assembles polished Remotion output. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[cohnen](https://clawhub.ai/user/cohnen) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, creative operators, and agents use this skill to plan and produce marketing, explainer, product, social, image, video, voiceover, music, and sound-effect assets through provider APIs and local Remotion rendering. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may send selected prompts or media to external AI and media providers. <br>
Mitigation: Use only provider services approved for the project and avoid confidential or regulated media unless the chosen provider is approved. <br>
Risk: Provider API usage may spend credits or incur cost. <br>
Mitigation: Set only the API keys intended for the run, use dry-run or provider overrides when control is needed, and review planned actions before execution. <br>
Risk: Generated assets and local render outputs may affect files in the working project. <br>
Mitigation: Run the skill in a separate project folder and review generated media, storyboard JSON, and Remotion output before publication. <br>
