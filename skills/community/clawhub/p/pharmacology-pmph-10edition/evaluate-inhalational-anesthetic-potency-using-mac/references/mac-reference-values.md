# 吸入麻醉药MAC参考值（常压、成人、常温）

| 麻醉药       | MAC (vol%) |
|--------------|------------|
| 七氟烷       | ≈2.0       |
| 异氟烷       | ≈1.2       |
| 地氟烷       | ≈6.0       |
| 恩氟烷       | ≈1.7       |
| 氧化亚氮     | ≈105       |
| 氟烷         | ≈0.75      |

## 影响MAC的因素
- **降低MAC**：高龄、妊娠、低温、低氧血症、酸中毒、合用阿片类/镇静药
- **升高MAC**：长期酗酒、高钠血症、中枢兴奋状态

## 临床应用要点
- 1.0 MAC = 50%患者对切皮无体动反应
- 1.3 MAC ≈ 95%患者无反应（ED95）
- 脑内药物浓度在稳态下≈肺泡浓度，故MAC可间接反映脑内有效浓度
- MAC不适用于评估静脉麻醉药或非吸入给药途径
