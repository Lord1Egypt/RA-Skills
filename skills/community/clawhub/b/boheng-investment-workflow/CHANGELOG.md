# 更新日志

## v1.5.5 (2026-05-19)

### ✨ 新增功能
- **动态ROE评估系统** (`roe_evaluator.py`)
  - 根据行业基准动态调整ROE评估标准（不再固定15%）
  - 考虑财报季度因素：Q1=25%, Q2=50%, Q3=75%, Q4=100%
  - 支持历史趋势对比
  - **新增季度对比表**：显示每个季度的行业基准、动态阈值、公司ROE、差距

### 🔧 优化
- `analysts.py` 集成动态ROE评估
- `financial_data_baostock.py` 财报期间改为结构化数据
- `quick_analysis.py` 传递财报季度信息给分析师

### 📚 文档更新
- SKILL.md 新增"动态ROE评估说明"章节（含季度对比表示例）
- 文件结构新增 `roe_evaluator.py`
