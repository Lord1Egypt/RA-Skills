# AI Resume Builder

Professional resume and cover letter creation with ATS optimization, industry-specific templates, and career narrative crafting for all experience levels.

## Core Capabilities

### 1. Resume Structure
- Format selection: chronological, functional, hybrid based on experience
- Section optimization: summary, experience, education, skills, projects
- Bullet point crafting: action verb + task + result formula
- Quantifiable achievement translation, metrics integration
- ATS keyword optimization, formatting for parsing

### 2. Industry Templates
- Tech/Engineering: skills matrix, project portfolio, GitHub links
- Finance/Consulting: deal experience, revenue impact, client engagement
- Creative: portfolio integration, visual layout, brand alignment
- Healthcare: licensure, clinical rotations, patient outcomes
- Academic: publications, grants, teaching portfolio

### 3. Cover Letters
- Opening hook strategies, company research integration
- Value proposition articulation: 3 key selling points
- Culture fit demonstration, mission alignment
- Closing call to action, follow-up integration

### 4. LinkedIn Optimization
- Headline and summary optimization for search
- Skills endorsement strategy, recommendation requests
- Content and engagement for visibility
- Network building: connection requests, alumni outreach

### 5. Career Transitions
- Transferable skill identification and translation
- Career gap explanation strategies
- Industry pivot narrative, passion to profession
- Military to civilian, academia to industry transition
