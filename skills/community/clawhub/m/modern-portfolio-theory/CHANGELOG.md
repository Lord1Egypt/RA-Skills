# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-23

### Added
- Initial release of Modern Portfolio Theory Portfolio Optimizer.
- Support for generating `config.yaml` using structured user questioning.
- Integration with CLI tools for optimization and backtesting.
- Markdown report generation including performance metrics and asset weights.
- Automated daily rebalancing and performance tracking via cron scripts.
- Portfolios and reports management system.
