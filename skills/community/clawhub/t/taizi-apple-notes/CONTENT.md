class="markdown"><h1>Apple Notes CLI</h1>
<p>Use <code>memo notes</code> to manage Apple Notes directly from the terminal. Create, view, edit, delete, search, move notes between folders, and export to HTML/Markdown.</p>
<p>Setup</p>
<ul>
<li>Install (Homebrew): <code>brew tap antoniorodr/memo &amp;&amp; brew install antoniorodr/memo/memo</code></li>
<li>Manual (pip): <code>pip install .</code> (after cloning the repo)</li>
<li>macOS-only; if prompted, grant Automation access to Notes.app.</li>
</ul>
<p>View Notes</p>
<ul>
<li>List all notes: <code>memo notes</code></li>
<li>Filter by folder: <code>memo notes -f &quot;Folder Name&quot;</code></li>
<li>Search notes (fuzzy): <code>memo notes -s &quot;query&quot;</code></li>
</ul>
<p>Create Notes</p>
<ul>
<li>Add a new note: <code>memo notes -a</code>
<ul>
<li>Opens an interactive editor to compose the note.</li>
</ul>
</li>
<li>Quick add with title: <code>memo notes -a &quot;Note Title&quot;</code></li>
</ul>
<p>Edit Notes</p>
<ul>
<li>Edit existing note: <code>memo notes -e</code>
<ul>
<li>Interactive selection of note to edit.</li>
</ul>
</li>
</ul>
<p>Delete Notes</p>
<ul>
<li>Delete a note: <code>memo notes -d</code>
<ul>
<li>Interactive selection of note to delete.</li>
</ul>
</li>
</ul>
<p>Move Notes</p>
<ul>
<li>Move note to folder: <code>memo notes -m</code>
<ul>
<li>Interactive selection of note and destination folder.</li>
</ul>
</li>
</ul>
<p>Export Notes</p>
<ul>
<li>Export to HTML/Markdown: <code>memo notes -ex</code>
<ul>
<li>Exports selected note; uses Mistune for markdown processing.</li>
</ul>
</li>
</ul>
<p>Limitations</p>
<ul>
<li>Cannot edit notes containing images or attachments.</li>
<li>Interactive prompts may require terminal access.</li>
</ul>
<p>Notes</p>
<ul>
<li>macOS-only.</li>
<li>Requires Apple Notes.app to be accessible.</li>
<li>For automation, grant permissions in System Settings &gt; Privacy &amp; Security &gt; Automation.</li>
