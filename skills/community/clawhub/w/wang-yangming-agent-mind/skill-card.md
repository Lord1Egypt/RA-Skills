## Description: <br>
A philosophical skill for AI agents based on Wang Yangming's Heart-Mind doctrine that supports decision-making, task execution, multi-step workflows, ethical judgment, error recovery, and self-correction. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent builders use this skill to apply Wang Yangming's Heart-Mind principles as an operational framework for planning, intent routing, alignment checks, tool orchestration, execution monitoring, and self-correction. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad activation and a required philosophy note can influence normal agent planning, self-correction, and strict response formats. <br>
Mitigation: Use narrower activation rules in tightly controlled environments and make the philosophy note optional where fixed output formats are required. <br>
