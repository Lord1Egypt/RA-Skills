# /report — 完整研报

对指定A股公司执行深度分析 + 行业分析，整合输出为标准化五章研报。

## 执行步骤

1. 执行深度分析（等同于 `/deep` 工作流）
2. 执行行业分析（等同于 `/sector` 工作流）
3. 整合为标准化五章研报结构

## 研报结构

```
第一章：公司概况
  - 基本信息、主营业务、股权结构
  - 管理层评估（薪酬、履历、持股稳定性）
  - 商业模式与核心竞争力

第二章：财务分析
  - 三大报表联动分析（5年趋势）
  - 五维度财务指标（盈利/营运/偿债/成长/现金流）
  - 杜邦分析（ROE三因子归因）
  - 核心利润体系分析

第三章：行业分析
  - 行业地位与市场份额
  - 竞争格局与主要竞争对手对比
  - 产业链位置与议价能力
  - 行业发展趋势

第四章：估值分析
  - DCF估值（FCFE/FCFF，含敏感性分析）
  - 相对估值（PE/PB/PS/EV-EBITDA行业对比）
  - 合理价值区间判断
  - 催化剂与下行风险

第五章：风险提示与投资建议
  - 财务风险（舞弊信号、流动性、杠杆）
  - 行业风险（周期、政策、竞争）
  - 治理风险（股权质押、关联交易、管理层）
  - 投资建议（买入/增持/中性/减持/回避）
  - 目标价位与持仓建议
```

## 输入格式

- 股票代码 或 公司名称

## 输出格式

- Markdown 结构化研报（默认）
- 如已安装 Python 输出插件，可选择生成 PNG 图表 + Word 文档
