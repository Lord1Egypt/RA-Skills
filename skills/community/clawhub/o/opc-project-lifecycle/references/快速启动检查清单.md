# 快速启动检查清单

> 配套文件：胡田-OPC导师-项目全流程管理.Skill v1.0
> 用途：启动新项目时必须逐项勾选

---

## 项目启动前检查

```
□ 项目编号已生成（格式：OPC-年份-序号-缩写）
□ 立项申请（ONE-PAGER）已填写
□ 私董会已完成评审
□ 项目档案目录已创建
□ README.md已初始化
□ 项目总览看板已更新
```

## 每个阶段推进前检查

```
□ 前置阶段已完成（状态为✅）
□ 前置输出数据已存到项目档案
□ 本阶段对应的Skill已就绪
□ 本阶段输入数据已就绪
□ 执行人/角色已明确
□ 预期交付物清单已确认
```

## 每个阶段完成后检查

```
□ 产出物已保存到项目档案
□ 阶段性考评已填写并保存
□ 成就解锁条件已检查
□ README.md状态已更新
□ 总览看板已同步
□ 下阶段触发条件已检查
□ （可选）语音回放已生成
```

## 项目终止/休息前检查

```
□ 终止/休眠原因已记录
□ 未完成阶段的产出已打包保存
□ 可复用资产已标记
□ 项目总览看板已更新
```
