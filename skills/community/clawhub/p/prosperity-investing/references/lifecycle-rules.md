# 生命周期法则

> **核心**：行业和企业有生命周期，不同阶段的投资策略完全不同。景气投资的本质是找到「成长期」的标的，在「稳定期」持有，在「衰退期」退出。

---

## 一、行业生命周期四阶段

| 阶段 | 特征 | 估值特征 | 投资策略 | 典型A股案例 |
|------|------|---------|---------|------------|
| **初创期** | 技术刚突破，市场未验证，厂商少，亏损 | 高PE甚至无PE，纯概念估值 | **不参与或极小仓位博弈题材浪** | 早期AI大模型、量子计算 |
| **成长期** | 需求爆发，厂商增多，价格上升，利润快速增长 | PE较高但可被增速消化（PEG<1.5） | **主战场，预期浪布局，业绩浪持有** | 2023光模块、2021磷酸铁锂 |
| **稳定期** | 格局固化，增速放缓至10-20%，龙头市占率高 | PE回归中枢（15-25倍） | GARP策略，赚业绩+分红钱 | 白酒2018前、医药CXO |
| **衰退期** | 需求下滑，产能过剩，价格战，利润萎缩 | PE看似低实则陷阱（价值陷阱） | **坚决回避，不可抄底** | 2023光伏、2024锂电 |

---

## 二、企业生命周期与投资匹配

```
企业生命周期 → 投资策略映射

初创期（0-3年）
    └── 主题投资/风险投资逻辑
    └── 适合游资和VC，不适合二级市场重仓

成长期（3-10年）
    └── 景气投资核心战场
    └── 预期浪买入 → 业绩浪持有 → 泡沫浪退出
    └── 允许较高估值，关键是增速能否维持

稳定期（10-20年）
    └── 价值投资/GARP策略
    └── 看ROE、现金流、分红率
    └── 增速>15% + 估值<25PE = 可持有

衰退期（20年+或技术被颠覆）
    └── 只考虑困境反转/并购重组
    └── 不参与长期持有
```

## 三、景气股的生命周期判断（实操版）

### 3.1 初创期识别（不参与）

| 信号 | 说明 |
|------|------|
| 营收<1亿或无常营收 | 纯研发阶段 |
| 无核心客户验证 | 产品未通过大客户认证 |
| 技术路线不确定 | 多条路线并存， winner未出现 |
| 市场教育成本高 | 下游接受度低 |

**例外**：如果题材浪有机会，可做短线，但不作为景气投资标的。

### 3.2 成长期识别（主战场）

**必要条件（必须全部满足）**：
- [ ] 营收增速 > 30%（连续2个季度）
- [ ] 有明确的大客户或订单验证
- [ ] 技术路线已确定，公司处于领先位置
- [ ] 行业渗透率 < 50%（仍有巨大空间）
- [ ] 竞争格局未恶化（CR3在提升，新进入者少）

**成长期的四个子阶段**：

| 子阶段 | 特征 | 投资节奏 |
|--------|------|---------|
| **导入期** | 产品刚上市，小批量出货 | 预期浪早期，可布局 |
| **爆发期** | 订单/产能双爆发，营收翻倍 | 预期浪→业绩浪，持有 |
| **扩张期** | 产能持续扩张，但增速开始放缓 | 业绩浪持有，不追 |
| **成熟期** | 增速降至20-30%，格局固化 | 泡沫浪考虑退出 |

### 3.3 稳定期识别（GARP）

| 信号 | 说明 |
|------|------|
| 营收增速 10-20% | 增速稳定但缺乏弹性 |
| 毛利率稳定 | 竞争格局固化，无价格战 |
| 高ROE（>15%） | 龙头壁垒深厚 |
| 稳定分红 | 进入现金奶牛阶段 |

**操作**：低PE（<20）时买入，赚业绩钱+分红钱。

### 3.4 衰退期识别（坚决回避）

| 信号 | 说明 |
|------|------|
| 营收连续2季度下滑 | 需求见顶 |
| 全行业扩产后产能过剩 | 价格战开始 |
| 技术路线被颠覆 | 老技术被新技术替代 |
| 龙头开始多元化 | 主业增长乏力，被迫转型 |

**生命法庭原话**：
> "赛道周期化，新能源海控化。"——所有产业最终都会进入周期，没有永恒的成长期。
> "宁德时代成了老大，股价却提前见顶，业绩降速成了价值股。"

## 四、生命周期法则的操作纪律

### 4.1 不同阶段的估值容忍度

| 阶段 | 可接受PE | 核心指标 |
|------|---------|---------|
| 成长期导入期 | 50-100倍 | 营收增速、订单、渗透率 |
| 成长期爆发期 | 30-60倍 | 净利润增速、产能利用率 |
| 成长期扩张期 | 20-40倍 | 毛利率、净利率、ROE |
| 稳定期 | 15-25倍 | 分红率、现金流、ROE稳定性 |
| 衰退期 | **不看PE** | **回避** |

### 4.2 生命周期切换的预警信号

**成长期 → 稳定期/衰退期**：
- 行业增速从50%降至20%以下
- 新进入者大幅增加，竞争加剧
- 龙头市占率不再提升
- 产品价格开始下降

**稳定期 → 衰退期**：
- 营收连续下滑
- 技术路线出现颠覆者
- 下游需求结构性转移

### 4.3 持股周期与生命周期的匹配

| 投资目标 | 匹配的生命周期阶段 | 预期持股周期 |
|----------|-------------------|-------------|
| 10倍股 | 成长期导入期→爆发期 | 1-2年 |
| 3-5倍股 | 成长期爆发期→扩张期 | 6-12个月 |
| 50%收益 | 稳定期低估值买入 | 3-6个月 |
| 股息收入 | 稳定期高分红龙头 | 长期持有 |

## 五、经典案例：生命周期错配的血泪史

### 案例1：光伏（2020-2024）

| 年份 | 生命周期阶段 | 市场表现 |
|------|------------|---------|
| 2020 | 成长期导入期 | 隆基股份暴涨 |
| 2021 | 成长期爆发期 | 全产业链翻倍 |
| 2022 | 成长期扩张期 | 硅料见顶，组件开始内卷 |
| 2023-2024 | **衰退期** | 产能过剩，价格战，全行业亏损 |

**教训**：2022年增速放缓时就应该意识到周期顶部，而非按成长股长期持有。

### 案例2：磷酸铁锂（2021-2023）

| 年份 | 生命周期阶段 | 市场表现 |
|------|------------|---------|
| 2021 | 成长期爆发期 | 磷三杰5-10倍 |
| 2022 | 成熟期/衰退期 | 碳酸锂从60万跌到10万，全产业链A杀 |

**教训**：当供给端（锂矿/磷矿）疯狂扩产、新进入者涌入时，成长期结束。

### 案例3：光模块（2023-2026）

| 年份 | 生命周期阶段 | 投资节奏 |
|------|------------|---------|
| 2023 | 成长期导入期 | 800G预期浪布局 |
| 2024 | 成长期爆发期 | 业绩兑现，中际旭创新高 |
| 2025 | 成长期扩张期 | 1.6T导入，硅光博弈 |
| 2026 | **成熟期？** | 需验证：1.6T放量+AI需求持续性 |

**关键验证**：2026年光模块是否进入成熟期，取决于AI算力需求增速是否放缓。

## 六、快速诊断表

拿到一只股票，30秒内判断生命周期：

| 问题 | 答案 | 判断 |
|------|------|------|
| 营收增速连续2季度>30%？ | 是 → 成长期；否 → 看其他 |
| 行业渗透率<50%？ | 是 → 成长期还有空间 |
| 新进入者大量涌入？ | 是 → 成长期末期/衰退期开始 |
| 产品价格趋势？ | 涨 → 成长期；平 → 稳定期；跌 → 衰退期 |
| 龙头市占率在提升？ | 是 → 成长期；否 → 稳定期/衰退期 |
| 技术路线是否被颠覆？ | 是 → 衰退期风险 |

## 七、与其他法则的配合

| 法则 | 作用 | 配合方式 |
|------|------|---------|
| **四浪理论** | 判断股价处于哪个阶段 | 成长期标的多半在预期浪/业绩浪 |
| **一季度法则** | 验证景气持续性 | 一季度增速是全年方向标 |
| **选股标准** | 买入/卖出纪律 | 生命周期决定估值容忍度和持股周期 |

**实战组合**：
- 先用**生命周期法则** → 判断行业处于哪个阶段，是否值得参与
- 再用**四浪理论** → 判断个股处于哪一浪，该不该买
- 最后用**一季度法则** → 季度业绩验证景气是否持续
