## Description: <br>
Skill Scaffold is an AI agent skill scaffolding CLI for creating OpenClaw, Moltbot, MCP, and generic skill structures. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[NextFrontierBuilds](https://clawhub.ai/user/NextFrontierBuilds) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and engineers use this skill to generate starter folders and documentation for agent skills, MCP servers, and simple skill packages. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated skill documentation can include generic descriptions, trigger phrases, and author-supplied text that may be inaccurate for the final published skill. <br>
Mitigation: Review and edit generated SKILL.md and README.md content before publishing or letting another agent rely on the generated skill. <br>
