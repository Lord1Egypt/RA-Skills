# drug-elimination-kinetics-type-discrimination-and-parameter-calculation

## 何时使用
- 已获得药物在体内的浓度-时间数据；
- 需要判断药物的消除是否遵循一级、零级或混合动力学；
- 消除机制是否可能因酶或转运系统饱和而改变；
- 需要估算药代动力学参数以支持剂量调整或毒性预测。

## 执行步骤
1. **检查数据特征**：绘制常规坐标和半对数坐标的药-时曲线。
   - 若半对数图为直线 → 提示一级动力学；
   - 若常规坐标为直线 → 提示零级动力学；
   - 若两者均非直线且高浓度时斜率变缓 → 考虑混合动力学。

2. **拟合基础微分方程**：使用 dC/dt = -Kₑ·Cⁿ 初步判断 n 值。
   - n ≈ 1 → 一级消除；
   - n ≈ 0 → 零级消除。

3. **验证混合动力学可能性**：若药物已知具有饱和消除特性（如苯妥英钠、乙醇、水杨酸），尝试用米-曼方程拟合：
   ```
   dC/dt = -(Vₘₐₓ·C)/(Kₘ + C)
   ```
   - 当 C ≪ Kₘ 时，应近似为一级动力学；
   - 当 C ≫ Kₘ 时，应近似为零级动力学。

4. **计算关键参数**：
   - 一级动力学：从半对数图斜率计算 Kₑ = -2.303 × 斜率；
   - 零级动力学：从常规坐标斜率得 K₀ = -斜率；
   - 混合动力学：通过非线性回归拟合 Vₘₐₓ 和 Kₘ。

5. **输出结果**：明确说明消除类型及对应参数，用于后续浓度预测或临床决策。

> 注意：零级动力学仅在消除系统饱和时出现；混合动力学必须满足米-曼动力学条件。
