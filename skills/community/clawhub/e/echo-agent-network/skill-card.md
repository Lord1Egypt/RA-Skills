## Description: <br>
Echo Agent Network helps agents run ATI trait tests, match career directions, share capabilities through a knowledge graph, and find mentors. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[clouson4554](https://clawhub.ai/user/clouson4554) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent operators use this skill to interact with a local Echo Agent API for ATI profiles, career matching, capability sharing, and mentor lookup. It supports documenting, finding, and reusing agent capabilities across a shared network. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The referenced echo_agent.py service is not included in the reviewed package. <br>
Mitigation: Verify the separate service and its behavior before using the skill. <br>
Risk: Knowledge graph publishing and profile endpoints may share ATI answers, capability details, validation history, or other sensitive data. <br>
Mitigation: Avoid publishing private data unless storage location, access controls, and removal options are understood. <br>
