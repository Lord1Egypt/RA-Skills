# cn-join-lines

文本行合并工具。将多行文本合并为单行，支持自定义分隔符。

## 使用方法

```bash
python3 scripts/join_lines.py --text "第一行\n第二行\n第三行"
python3 scripts/join_lines.py --text "第一行\n第二行" --separator ","
```

纯Python标准库。
