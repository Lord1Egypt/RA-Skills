# 冲稳保评分算法

## 核心数学模型

### 输入参数
```
R = 考生省排名（位次，数值越小越好）
P = 往年录取最低位次（该校该专业）
招生计划变动 = 今年计划/去年计划（1=不变，1.2=扩招20%）
```

### 录取概率计算

```
基础概率 = 1 - (P - R) / P

当 P < R (冲):
  位次差距 = (R - P) / P
  概率 = max(0, 0.5 - 位次差距 * 0.8)
  * 扩招加分：每扩招10%，概率+3%
  
当 P ≈ R (稳):
  概率 = 0.6-0.8 (取决于缩扩招)
  * 位次相同 = 70%
  * 位次前5% = 80%
  * 位次后5% = 60%
  
当 P > R (保):
  概率 = min(0.95, 0.8 + 位次优势 * 0.3)
```

### 梯度划分

| 梯度 | 概率区间 | 数量建议 | 策略说明 |
|:----:|:--------:|:--------:|:--------:|
| 冲高 | 10-25% | 2-3个 | 万一爆冷或不退档（很少真能上） |
| 冲刺 | 25-40% | 3-4个 | 正常发挥+专业可调剂可冲 |
| 稳妥 | 55-75% | 3-4个 | 大概率进的学校，主攻 |
| 保底 | 85-95% | 2-3个 | 绝对能进，只是防滑档 |

### 专业匹配评分（总分100）

```
总分 = 兴趣得分(30) + 就业得分(35) + 学科得分(25) + 资源得分(10)

兴趣得分(0-30):
  强烈兴趣: 30
  一般兴趣: 15
  无所谓: 5
  排斥: -20 (排除)

就业得分(0-35):
  高薪热门(计算机/AI/芯片): 30-35
  稳定中薪(电气/师范/医学): 25-30
  中等(自动化/机械): 15-25
  偏低(文科/基础科学): 5-15

学科得分(0-25):
  A+学科: 25
  A: 22
  A-: 19
  B+: 16
  B: 13
  未参评: 8-10

资源得分(0-10):
  家庭行业支持: 8-10
  城市校友网络: 5-7
  无特殊资源: 2
```

## 城市偏好加权

| 城市分级 | 加分系数 | 代表城市 |
|:--------:|:--------:|:--------:|
| 一线超强 | +0% | 北京、上海 |
| 新一线强势 | +0-5% | 杭州、深圳、广州、南京、成都 |
| 省会/经济强市 | - | 武汉、西安、长沙、苏州、重庆 |
| 普通城市 | -5-10% | 其他城市 |

## 2026年特殊因素

1. **扩招趋势**：2026年多数双一流高校扩招5-10%，利好考生
2. **专业热度**：AI/芯片相关专业分数线可能继续上涨5-10分
3. **地域偏好**：经济发达城市分数继续走高
4. **编制光环**：公务员录用率高的专业（法学、计算机、财会）持续热门
