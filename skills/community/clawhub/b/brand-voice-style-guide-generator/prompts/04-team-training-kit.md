# Prompt 4: Team Training Kit

## Instructions
Run Prompts 1–3 first. This prompt turns the style guide into an active training system.

---

## The Prompt

You are a senior content director building a brand voice training program. Using the style guide profile below, create a complete team training kit that any company can use to onboard writers, assess brand voice consistency, and maintain quality over time.

**Company:** [COMPANY NAME]  
**Industry:** [INDUSTRY]  
**Voice Pillars:** [PILLAR 1], [PILLAR 2], [PILLAR 3], [PILLAR 4], [PILLAR 5]  
**Team size:** [NUMBER of people who produce content]  
**Content types produced:** [LIST — e.g., blog, LinkedIn, email, sales copy, support docs]  
**Biggest brand voice inconsistency problems today:** [DESCRIBE IN 2-3 SENTENCES]

**Do/Don't rules (from Prompt 3):** [Paste 5 key rules]

---

Deliver the complete team training kit structured exactly as follows:

### SECTION 1: Brand Voice Quiz

10 multiple-choice questions. Each question presents a content scenario or sample and asks the writer to identify the most on-brand option.

Format for each question:
**Q[N]: [Scenario or "Which of these is most on-brand?"]**
A) [Option A]
B) [Option B]
C) [Option C]
D) [Option D]

After all 10 questions: ANSWER KEY
- Q1: [Letter] — Why: [1-sentence explanation referencing a voice pillar or rule]
- Q2: [Letter] — Why: ...
(through Q10)

Scoring guide:
- 9–10 correct: Ready to publish independently
- 7–8: Needs spot review on final drafts
- 5–6: Needs weekly brand voice check-ins for 30 days
- Below 5: Full brand voice training before independent publishing

### SECTION 3: Peer Feedback Rubric

A scoring card editors and managers use when reviewing content for brand voice.

**5 dimensions, scored 1–5 each (max 25 points):**

| Dimension | 1 (Off-brand) | 3 (Inconsistent) | 5 (On-brand) | Score |
|-----------|--------------|------------------|--------------|-------|
| Voice Pillar Alignment | | | | /5 |
| Channel Tone Match | | | | /5 |
| Vocabulary (uses approved words, avoids banned words) | | | | /5 |
| Sentence Structure (follows preferred patterns) | | | | /5 |
| Emotional Trigger Hit | | | | /5 |
| **Total** | | | | /25 |

For each dimension:
- Write the full description for scores 1, 3, and 5 (not just a placeholder)
- Include a comment prompt: "Notes: ____"

**Interpretation:**
- 22–25: Publish as-is
- 17–21: Minor revisions, re-review optional
- 12–16: Significant revisions required
- Below 12: Return for full rewrite with coaching session

### SECTION 4: "Is This On-Brand?" Exercises

10 exercises. Each presents a real-looking content piece and asks the writer to evaluate it.

Format for each exercise:
**Exercise [N]: [Content type — e.g., LinkedIn post, email subject line, product description]**

[Sample content — write 3–8 sentences of actual content]

Which best describes this content?
A) On-brand — embodies [PILLAR] and hits the right tone for this channel
B) Partially on-brand — good intent but [specific issue]
C) Off-brand — violates [specific rule] and should be rewritten

**Answer:** [Letter]  
**Explanation:** [2–3 sentences explaining exactly why, referencing the voice pillars and rules from Prompt 3]

### SECTION 5: Quarterly Brand Voice Audit Template

A lightweight check-in to run every 90 days to keep the style guide current.

**5 audit questions:**
1. [Question about whether the voice still fits the ICP as the company has evolved]
2. [Question about which channel is most inconsistent and why]
3. [Question about new vocabulary or banned words to add]
4. [Question about new content types not covered in the original guide]
5. [Question about team feedback — what rules are being ignored and why]

**2 metrics to track:**
- **Metric 1:** [Measurable proxy for brand consistency — e.g., "% of LinkedIn posts that required brand voice edits before publishing"]
- **Metric 2:** [Another metric — e.g., "Time-to-approve for new writer first solo piece"]

Audit output: update 3 sections of the style guide based on findings. Document changes with date and reason.

---

Output all five sections. Write all examples and exercises as if they're final, ready-to-use training materials — not as templates or placeholders. Reference [COMPANY NAME]'s specific voice pillars throughout.
