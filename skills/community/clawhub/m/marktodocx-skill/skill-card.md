## Description: <br>
Convert Markdown into a DOCX file using the shared marktodocx Node runtime when a user explicitly wants Markdown content or a Markdown file converted to .docx. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[zhao-kun](https://clawhub.ai/user/zhao-kun) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users invoke this skill when they need Markdown from a file or inline prompt converted into a Word .docx document. It supports document styling options and explicit output paths for repository or workspace workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill writes DOCX files to local paths. <br>
Mitigation: Invoke it explicitly and review the requested output path before running conversion. <br>
Risk: Local images referenced by the Markdown can be embedded into the generated DOCX. <br>
Mitigation: Review the Markdown input and base directory so only intended local assets are available for embedding. <br>
Risk: Mermaid-enabled builds may require Chromium or Puppeteer components and compatible host libraries. <br>
Mitigation: Use the standard ClawHub release for non-Mermaid conversion, or provision and test a Mermaid-enabled export on the target host. <br>
