# 儿童1型糖尿病长期胰岛素治疗方案

## 适用前提
- 患儿已确诊1型糖尿病
- 已完成急性期（如糖尿病酮症酸中毒）处理
- 能正常进食

## 禁忌与约束
- 不适用于2型糖尿病或继发性糖尿病
- 必须配合饮食与运动管理

## 执行步骤

### 1. 胰岛素制剂选择
根据药代动力学特性选择合适类型：
- **速效类似物**：起效0.25小时，峰值1–2小时，持续4–6小时
- **短效胰岛素（RI）**：起效0.5小时，峰值3–4小时，持续6–8小时
- **中效胰岛素（NPH）**：起效1.5–2小时，峰值4–12小时，持续18–24小时
- **长效类似物**：甘精胰岛素（无峰，持续24小时）；地特胰岛素（峰值6–12小时，持续20–24小时）
- **预混胰岛素**：双峰释放，持续16–24小时

### 2. 选择治疗方案
从以下四种常用方案中选择一种：

**① 基础-餐时大剂量方案**
- 三餐前注射短效或速效胰岛素
- 睡前注射中效或长效胰岛素类似物
- 夜间基础胰岛素占全日总量30%–50%（若使用速效类似物，比例宜更高）

**② 持续皮下胰岛素输注（CSII）**
- 使用短效或速效类似物
- 全日总量分为基础量与餐前追加量，比例约为1:1
- 日间（07:00–21:00）与夜间（21:00–次日07:00）基础量之比为2:1
- 餐前追加量三餐平均分配
- 根据指尖血糖或连续葡萄糖监测（CGM）动态调整剂量

**③ 每天3次注射方案**
- 早餐前：短效 + 中效混合
- 午餐前：单用短效
- 晚餐或睡前：短效 + 中效混合

**④ 每天2次注射方案（现已少用）**
- 早餐前与晚餐前各注射一次短效+中效混合剂
- 短效:中效 ≈ 1:2
- 早餐前用量占全日2/3，晚餐前占1/3

### 3. 剂量调整原则
- **婴儿**：剂量偏小；**年长儿**：剂量偏大
- **新诊断轻症**：0.5–1.0 U/kg/天
- **青春期前**：0.75–1.0 U/kg/天
- **青春期**：通常 >1.0 U/kg/天
- 每2–3天根据血糖或尿糖结果调整一次，目标为尿糖 ≤ ++

### 4. 注射技术与注意事项
- 优先使用胰岛素笔或泵以减少痛苦
- 注射部位：大腿、上臂、腹壁，轮换使用
- 同一部位1个月内不重复注射，针距 ≥2.0 cm
- 运动前避免在运动肢体注射，以防吸收过快导致低血糖
- 胰岛素泵需定期更换输注部位并保持清洁，预防感染

### 5. 特殊现象识别与初步处理
- **Somogyi现象**（胰岛素过量）：午夜低血糖→清晨高血糖；若胰岛素用量 >1.5 U/kg/天且血糖波动大，应测凌晨1–3点血糖确认
- **黎明现象**（胰岛素不足）：清晨5–9点高血糖；可加大晚间剂量或将NPH注射时间后移
- **胰岛素耐药**：无DKA情况下 >2 U/kg/天仍无法控糖，排除Somogyi后可考虑换用基因重组人胰岛素
