# 推荐度评价规则

报告中的推荐度评价必须拆成两条，顺序可以按场景调整，但两条都必须出现，且不能合并成平均分。

## 个性化推荐度评价（学生/家庭适配）

判断问题：这个项目是否适合当前学生和家庭。

必须使用或按情景补全以下维度：

- 学生分数/位次与本年预估区间的关系：冲、稳、保、不适合；必须写位次安全边际或差距。
- 家庭预算：四年总投入、出国阶段费用、外方大学所在城市生活成本、汇率波动、延毕和语言成本；必须写年度预算差额。
- 出国偏好：愿意出国、可接受、不愿出国；连接证书和留服认证边界。
- 未来路径：保研/考研、海外硕士、就业、考公考编、国企事业单位、外企科技企业。
- 学习承受度：英语、GPA、全英文/双语、挂科和延毕压力。
- 城市/校区与生活条件：是否接受异地校区、住宿安排和资源共享差异。
- 风险承受度：新项目、首年招生、无毕业生、留服路径不清、证据链薄弱。

等级：高度适合 / 适合 / 边界适合 / 不太适合 / 不适合。

## 项目综合实力推荐度评价（项目综合实力角度）

判断问题：撇开当前家庭条件，这个项目本身质量和风险如何。

必须使用以下维度：

- 监管身份：CRS、批准/备案、有效期、计划内/计划外、招生省份。
- 双方实力：中方层次、外方院校与专业实力；QS 等排名必须使用运行时打开的最新官方榜单。
- 专业竞争力：专业方向、行业前景、课程结构、师资和培养方案。
- 证书与留服：国内学籍、国内毕业证/学位证、外方学位、CSCSE/留服认证路径与边界。
- 招生稳定性：招生计划、专业组、选科、学费、校区是否稳定。
- 毕业成果：项目级优先；无项目级时标注学院/机构级或学校级背景，不写姓名。
- 风险：费用、延毕、转专业、调剂、校区资源、公开质疑和未核到材料。

等级：强 / 较强 / 中等 / 偏弱 / 高风险。

## 冲突处理

必须允许并解释以下情况：

- 项目综合实力强，但因预算紧张、不愿出国、英语弱、考公优先或位次不匹配，对该学生并不适合。
- 项目综合实力一般，但因费用低、证书路径清楚、位次安全或地域便利，对某个家庭可作为备选。
- 两项都弱时，不要中性表达，直接给观察、慎选、暂不建议或仅关注/待核验。


## 用户输入优先级

用户明确提供预算、位次、出国偏好或未来路径时，个性化推荐度必须把这些信息放在结论和候选卡中反复使用，而不是只在画像里复述一次：

- 位次：与本年参考区间比较，输出安全边际、差距和冲/稳/保动作。
- 年度预算：与学费、住宿、国内生活费、境外城市生活成本和汇率缓冲比较，输出年度预算差额。
- 出国偏好：连接到外方学位、CSCSE/留服、海外生活成本、海外硕士和就业路径。
- 不匹配时要明确降级：项目强但预算或位次不匹配，可判为边界适合、不太适合或慎选。
