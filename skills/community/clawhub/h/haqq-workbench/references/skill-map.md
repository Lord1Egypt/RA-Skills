## سير العمل المشترك (Combinations)

### 🔄 سير العمل 1: محتوى إسلامي جاهز للنشر
```
1. haqq-ethics → ضابط أخلاقي
2. ai-ethics → تحقق شرعي
3. hadith-verifier → تحقق الأحاديث
4. web-researcher → دعم بمعلومات
5. social-media-scheduler → تنسيق
6. combined_publisher → نشر
7. engagement-reply-scan → متابعة الردود
```

### 🔄 سير العمل 2: محاربة فساد/ظلم
```
1. justice-seed → حدد الظلم
2. corruption-burner → أنشئ المحتوى
3. fact-check + verify-claims → تحقق من الادعاءات
4. anti-extortion / anti-tyranny → محتوى مخصص
5. combined_publisher → نشر مع أدلة
```

### 🔄 سير العمل 3: إنشاء ونشر مهارة
```
1. skill-creator → بناء المهارة
2. clawhub-auto-publisher → حزم وانشر على ClawHub
3. announce on 3 platforms → أعلن للوكلاء
4. selective-memory → حفظ دروس التطوير
```

### 🔄 سير العمل 4: تحقق شامل من معلومة
```
1. web-researcher → بحث أولي
2. fact-check → تحقق متقاطع
3. verify-claims → تحقق مهني
4. hadith-verifier → تحقق شرعي (إن لزم)
5. تحديد: صحيح / ضعيف / مُختلق
```
