## Description: <br>
Create and publish ClawHub skills with the correct format. Includes YAML frontmatter template, _meta.json structure, and step-by-step publishing guide. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and skill authors use this guide to create, validate, and publish ClawHub skills with the expected SKILL.md frontmatter, _meta.json metadata, CLI workflow, and publication checks. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Publishing a skill can make the selected skill metadata and files available through ClawHub. <br>
Mitigation: Review the local folder and metadata before running the publishing command. <br>
Risk: CLI commands in the guide may affect local authentication or publish workflow state if run unintentionally. <br>
Mitigation: Run npx ClawHub commands deliberately and confirm the target path, slug, version, and tags before publishing. <br>
