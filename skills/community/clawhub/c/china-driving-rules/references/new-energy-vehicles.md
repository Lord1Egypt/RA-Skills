## 1. 纯电动汽车基本知识

### 1.1 术语速查表

| 术语 | 说明 |
|------|------|
| NEDC 续航 | 新欧标测试续航，实际偏低约 20-30% |
| WLTC 续航 | 新全球循环工况，更接近实际使用 |
| CLTC 续航 | 中国工况，市区多启停，更贴近城市驾驶 |

### 1.2 电池保养要点

**温度管理**
- ❌ **冬季**：避免低于 -20℃行驶，必要时开启电池预热
- ❌ **夏季**：暴晒后先通风降温再充电，避免高温爆燃
- ✅ **最佳温度**：电池工作区间 15-35℃

**日常使用**
- ✅ **不要满电存放**：长期停放建议 SOC 保持在 50-80%
- ✅ **定期补电**：即使不常用，每周充电一次可激活电池活性
- ✅ **快充勿过度**：快充后尽量慢充或正常充至 80-90%

### 1.3 特殊驾驶注意事项

**冬季续航损失应对**
- 提前开启空调预加热（用充电桩）
- 避免频繁急刹车，使用能量回收
- 关闭不必要的电器设备

**夏季防过热**
- 高速连续行驶每 2 小时检查电池温度
- 避免长时间占用慢充桩充电（可能过热）
- 长途前确认冷却系统工作正常

### 1.4 事故责任认定标准

**电池损坏界定**
| 场景 | 责任方 | 说明 |
|------|--------|------|
| 碰撞导致电池受损 | 对方全责 | 无论是否自燃，碰撞先定责 |
| 电池过热自燃（无外力） | 通常免责/次要 | 需排除人为破坏或操作不当 |
| 充电中起火 | 充电桩责任方优先 | 若车主私拉乱接则车主担责 |
| 电池故障导致刹车失效 | 产品缺陷 + 使用不当共担 | 4S 店先行垫付，后追偿 |

**举证要点**
- 立即报警并要求警方记录"无碰撞自燃"
- 拍照记录车辆位置、充电桩状态、电池包外观
- 要求保险公司出具事故责任认定书
- 必要时申请第三方检测机构鉴定

## 2. 充电安全与事故处理

### 2.1 充电场景分类与责任

| 充电场景 | 法律责任要点 |
|----------|--------------|
| **公共充电站** | 运营商主责，按桩上协议执行 |
| **小区固定车位充电桩** | 物业/车主责任（看安装协议） |
| **路边临时充电** | 需确认是否允许，违规被拖车自费 |
| **家用充电桩** | 自装则自担风险，建议购买保险 |

### 2.2 充电事故处理流程

**步骤一：立即断电**
- 按下急停按钮（如有）
- 拔掉充电枪，关闭车内电源
- 打开危险报警闪光灯

**步骤二：拍照取证**
```
必须拍清：
1. 充电桩状态指示灯/屏幕信息
2. 充电枪及电缆破损情况
3. 车辆底盘/电池包外观损伤
4. 周围是否有其他电动车或火源
5. 时间地点（带水印相机）
```

**步骤三：分类处理**
- **轻微故障**（充电中断、提示故障码）
  → 联系运营商客服报修，勿自行拆卸
  
- **过热/冒烟**
  → 立即撤离 + 拨打 119，切勿用水直接浇电池包
  
- **起火爆炸**
  → 保持 30 米安全距离，等待消防处理，不可靠近

### 2.3 常见充电违法行为与处罚

| 行为 | 处罚依据 | 罚款/扣分 |
|------|----------|-----------|
| 私拉电线在楼道充电 | 《高层居民住宅楼火灾》条例 | 警告+罚款 200-5000 |
| 占用消防通道充电 | 《消防法》第 60 条 | 罚款 5000，责令整改 |
| 飞线充电（从窗外垂拉） | 地方性法规 | 罚款 50-200 |
| 在禁停区充电（无桩位） | 《道路交通安全法》 | 罚款 100-200 |

### 2.4 保险理赔注意事项

**必须报案的情况**
- 充电中起火/爆炸
- 充电桩碰撞受损
- 电池包被异物刺穿
- 充电枪异常发热导致自燃

**理赔时效**
- **火灾事故**：72 小时内报案
- **设备损坏**：48 小时内报保险公司
- **人身伤亡**：立即报警 + 同步保险

**免赔条款常见陷阱**
- ⚠️ 未在指定区域充电导致的损坏（部分拒赔）
- ⚠️ 使用非标充电桩（电压/电流不匹配）
- ⚠️ 未按照操作手册进行充电（如强制插拔）

## 3. 自动驾驶辅助系统法律责任

### 3.1 L2/L3 级别功能边界

**L2 级（目前合法）**
- 功能：ACC 自适应巡航 + LDW 车道保持
- **驾驶员必须全程监控**，双手随时准备接管
- 发生事故=人为全责（除非证明系统缺陷）

**L3 级（有条件自动驾驶）**
- 允许特定场景下"脱手驾驶"
- 系统请求接管时，需在 2 秒内响应
- 未响应导致事故→驾驶员负主责

**目前现状**
- 中国尚未开放 L3 级公共道路正式应用
- 仅可在封闭测试场或特定区域使用

### 3.2 辅助驾驶违规认定案例库

**案例一：高速追尾（LKA 误判）**
```
场景：开启 ACC+LKA，前车急刹，未刹停发生追尾
责任认定：驾驶员全责
理由：ACC 只是速度控制，无法替代跟车距离判断；系统仅提示风险，不负责碰撞避免
教训：即使开辅助驾驶，也要保持安全距离
```

**案例二：城市道路刮蹭（AEB 失效）**
```
场景：L2+ 模式下 AEB 未刹停，撞上前车尾部
责任认定：对方次责（已制动但未完全避免），我方无责或同等
理由：前车主刹车灯异常/恶意追尾导致系统无法识别；若我方操作不当则主责
```

**案例三：高速变道碰撞**
```
场景：开启 LDW，未主动打转向灯被 LKA 强行纠偏撞护栏
责任认定：系统缺陷占一定比例，但驾驶员仍需承担部分责任
说明：LDA 是"辅助"而非"完全控制"，驾驶员需随时准备接管
```

### 3.3 车机系统设置违规案例

**超速导航诱导**
```
场景：导航建议限速 80，实际车速 100，被认定为"超速驾驶"
处罚结果：扣 3 分，罚款 200
法律依据：《道路交通安全法》第 45 条（不得超过限速标志）
结论：导航只是参考，超速责任在驾驶员
```

**系统锁定导致无法紧急避险**
```
场景：误触"定速巡航锁定"，前车突然减速无法及时刹停
处理流程：
1. 立即报警并申请事故鉴定
2. 保存行车记录仪+车机日志数据
3. 48 小时内向厂家申诉（如系统存在 Bug）
责任划分：若查明为系统缺陷→厂家担责；否则驾驶员主责
```

### 3.4 数据证据保存指南

**必须保存的证据链**
| 数据源 | 获取方式 | 用途 |
|--------|----------|------|
| **行车记录仪视频** | 导出原始文件，勿删除缓存 | 碰撞前后完整过程 |
| **车机日志** | 通过官方 APP 下载诊断报告 | 证明系统状态（是否开启辅助驾驶） |
| **充电桩记录** | 公共桩自动保存 | 证明充电状态/时间/电压电流 |
| **导航路线记录** | 高德/百度等 APP 历史轨迹 | 证明是否超速或偏离路线 |

**举证技巧**
- 行车记录仪选择支持"紧急制动自动上传"的型号
- 车机日志需原厂解密（第三方修改无效）
- 证据链完整度影响责任认定结果

## 4. 换电车型特殊规定

### 4.1 换电协议法律责任

**协议类型与责任划分**
| 协议方 | 典型条款 | 法律解释 |
|--------|----------|----------|
| **运营商（蔚来/奥动等）** | "电池由我方管理，事故与我无关" | 属格式条款，需证明已尽提醒义务；否则无效 |
| **车主与第三方换电站** | "站内换电，事故全责自负" | 明显不合理，法院可能判定共担 |

**关键判例参考**
> 2024 年上海某案例：蔚来电池包在换电站内充电时起火，运营商胜诉（已尽到安全维护义务）
> 结论：换电站作为专业机构，有更高注意义务

### 4.2 换电事故处理流程

**步骤一：紧急撤离**
- 立即离开车辆，勿站在电池仓正前方
- 打开双闪，设置三角警示牌（车后 50-150m）

**步骤二：保护证据**
```
拍照记录：
1. 电池仓盖板状态（是否变形/破损）
2. 换电设备屏幕报错代码
3. 换电机器人状态（如有）
4. 充电桩/换电站铭牌编号
```

**步骤三：报案与索赔**
- 拨打 122 + 联系品牌客服
- 若为换电站责任，可要求直接抵扣车损维修费
- 保险公司介入前勿私了

### 4.3 电池资产归属法律问题

**当前主流模式**
- **BaaS（电池租用）**：电池所有权归运营商
- **买断制**：车主拥有电池产权

**法律后果差异**
| 模式 | 事故责任主体 | 赔偿来源 |
|------|--------------|----------|
| BaaS | 运营方管理期间承担更高注意义务 | 保险 + 运营方赔偿 |
| 买断制 | 同传统燃油车，按常规定责 | 车主购买车险理赔 |

**消费者保护要点**
- ⚠️ 运营商不能以"电池归我"为由拒赔人身伤害
- ⚠️ BaaS 用户仍享有交强险保障（法律强制）
- ⚠️ 若电池缺陷导致事故，可追溯供应链责任

## 5. V2G 车网互动规则

### 5.1 什么是 V2G

**V2G = Vehicle-to-Grid**
- 定义：电动汽车向电网反向输电
- 适用车型：部分高端电动车（如特斯拉 V2L/V2H）
- 法规状态：**中国尚未开放商用**，仅在试验阶段

### 5.2 未来可能规则预测

**电价政策影响**
| 时段 | 预期政策支持 | 用户操作建议 |
|------|--------------|--------------|
| **峰期（19:00-22:00）** | V2G 放电获利 | 可开启放电，但注意续航损失成本 |
| **平期** | 双向交易 | 保持电量平衡即可 |
| **谷期（23:00-7:00）** | 强制充电为主 | 不建议放电 |

**法律责任预判**
- 若 V2G 导致电网波动，车主可能承担"扰乱用电秩序"责任
- 建议未来购买专门保险覆盖此类风险

### 5.3 试验区域政策（参考）

| 试点城市 | 政策支持 | 用户须知 |
|----------|----------|----------|
| **深圳** | 允许 V2L 对外放电 | 禁止在公共电网反向输电 |
| **上海** | 鼓励换电 +V2G 试点 | 仅限指定运营商车辆 |
