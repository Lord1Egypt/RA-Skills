## Description: <br>
Cocktail Craft helps agents answer cocktail recipe, technique, history, bar-style research, recipe-development, homemade-ingredient, glassware, garnish, and ice-handling questions using bundled cocktail references. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[makishimatouri](https://clawhub.ai/user/makishimatouri) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and agent developers use this skill to look up cocktail recipes, validate drink proportions, research cocktail history, and draft new recipes with precise measurements, technique notes, glassware, garnish, and source-aware constraints. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Installation examples may run shell commands that write persistent skill files locally. <br>
Mitigation: Review the source archive, destination path, and exact install commands before running them. <br>
Risk: Cocktail guidance can involve alcohol, high-proof spirits, homemade infusions, potent botanicals, or raw egg recipes. <br>
Mitigation: Apply separate food-safety and responsible-adult-use checks before preparing or serving recipes. <br>
