# 三层分析框架

## 概述

基于融合8本财务经典的三层递进分析方法论，从宏观战略到微观科目逐层穿透。

## 宏观层：战略与环境

### 行业生命周期判断

| 阶段 | 营收增速 | 利润特征 | 竞争格局 | 代表行业 |
|------|---------|---------|---------|---------|
| 导入期 | >30% | 普遍亏损 | 分散，技术路线未定 | 量子计算、固态电池 |
| 成长期 | 15-30% | 龙头盈利、尾部亏损 | 快速集中 | 光伏、新能源车 |
| 成熟期 | 5-15% | 龙头稳定盈利 | 寡头格局 | 白酒、家电、银行 |
| 衰退期 | <5%或负 | 利润下滑 | 存量博弈 | 传统纸媒、功能手机 |

### 产业链位置分析

分析方法：
1. 画出产业链图谱（上游→中游→下游→终端）
2. 计算各环节毛利率分布（产业链利润池分析）
3. 判断目标公司在链条中的位置和议价能力
4. 识别纵向一体化或上下游挤压风险

### 波特五力分析

| 力量 | 评估维度 | 高威胁信号 |
|------|---------|-----------|
| 供应商议价能力 | 供应商集中度、替代成本 | 前五大供应商占比>50% |
| 购买者议价能力 | 客户集中度、转换成本 | 前五大客户占比>50% |
| 新进入者威胁 | 壁垒高度、资本需求 | 毛利率>50%且壁垒低 |
| 替代品威胁 | 性价比、转换成本 | 替代品价格年降>10% |
| 现有竞争者 | 集中度、差异化 | CR5<30%且增速放缓 |

### 八维战略分析框架

对公司在以下八维逐一评估，每维给予 1-5 分：
1. **战略**：战略清晰度、差异化、执行力
2. **资产**：资产质量、无形资产壁垒、表外资源
3. **利润**：利润来源可持续性、核心利润率趋势
4. **现金流**：造血能力、融资弹性
5. **成长**：增长驱动因素、天花板、第二曲线
6. **风险**：财务风险、经营风险、治理风险
7. **价值**：当前估值合理性、安全边际
8. **未来**：行业趋势、技术变革、政策方向

## 中观层：报表与指标

### 核心利润分析体系

```
核心利润 = 营业收入 - 营业成本 - 税金及附加 - 四项费用
（四项费用：销售费用 + 管理费用 + 研发费用 + 财务费用）

核心利润获现率 = 经营活动现金流量净额 / 核心利润
- 理想值：1.2-1.5
- 警戒线：< 0.7（利润含金量低）
- 危险线：< 0（利润无现金支撑）
```

### 杜邦分析

```
ROE = 净利率 × 资产周转率 × 权益乘数

逐层归因：
1. 净利率变化 → 是毛利率下降还是费用率上升？
2. 周转率变化 → 是存货慢还是应收慢？
3. 杠杆变化 → 是主动加杠杆还是被动（净资产缩水）？
```

## 微观层：科目与交易

### 舞弊恒等式

```
虚增利润 = 虚增资产 / 虚减负债 / 虚减权益
```

凡是利润异常增长，必然在资产负债表留下痕迹。检查方向：
- 虚增资产：应收高企、存货异常、商誉虚高、在建工程迟迟不转固
- 虚减负债：少计应付、隐瞒担保、表外负债
- 虚减权益：通过复杂交易藏匿亏损

### 七大盈余操纵模式

1. **过早确认收入**：发货即确认、完工百分比操纵、长期合同一次性确认
2. **确认虚假收入**：关联交易循环、对敲交易、虚构客户
3. **费用资本化**：研发全部资本化、利息资本化过度、营销费用资本化
4. **不计提或少计提减值**：坏账准备不足、存货跌价不计提、商誉不减值
5. **非经常性收益充利润**：卖资产保利润、政府补贴美化、投资收益掩盖主业亏损
6. **表外负债**：SPE合并、经营租赁不计、担保不披露
7. **现金流操纵**：将筹资/投资现金流入包装为经营现金流
