# 肾小体滤过屏障分析

## 适用对象
- 肾小体
- 血管球
- 足细胞
- 血浆成分

## 前提条件
- 血液流经肾小体血管球
- 毛细血管内血压较高

## 使用限制
- 不适用于滤过膜已受损的状态（如肾小球肾炎）
- 不适用于大分子蛋白质在正常情况下的滤过判断

## 执行步骤

1. **识别滤过屏障的三层结构**：
   - 有孔型毛细血管内皮（孔径50～100nm，多无隔膜）
   - 基膜（约330nm厚，含Ⅳ型胶原蛋白、层粘连蛋白及带负电荷的硫酸肝素蛋白聚糖）
   - 足细胞裂孔膜（次级突起间宽约25nm的裂孔，覆盖4～6nm厚的裂孔膜）

2. **评估物质通透性**：
   - 若分子量 < 70kDa 且 直径 < 4nm，则可滤过
   - 若物质带正电荷，则更易通过（因滤过屏障富含负电荷，排斥带负电物质）

3. **判断结果**：
   - 可滤过物质包括：葡萄糖、多肽、尿素、电解质和水
   - 正常情况下，大分子蛋白质无法通过，原尿不含蛋白质
   - 若出现蛋白尿或血尿，提示滤过膜可能受损

4. **说明修复机制**：
   - 系膜细胞可清除基膜沉积物
   - 内皮细胞与足细胞协同重建新基膜，恢复滤过功能
