## Known Risks and Mitigations: <br>
Risk: Cloud API calls can create or update moderation resources in the wrong Alibaba Cloud account, region, or resource scope. <br>
Mitigation: Use least-privilege Alibaba Cloud credentials and confirm region, resource IDs, and target operations before running create, update, modify, or set actions. <br>
Risk: Saved API metadata or response summaries may contain operational details from the user's environment. <br>
Mitigation: Review artifacts written under output/aliyun-green-moderation/ before sharing or publishing them. <br>
