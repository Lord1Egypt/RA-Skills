# BlogBurst — Proof-Led Growth Operator

BlogBurst helps solo and bootstrapped founders turn real buyer questions into tracked growth experiments.

It is not a generic AI writing tool. It connects:

- real buyer intent and product evidence
- grounded post generation
- quality gates before publishing
- short-link attribution
- learning from clicks, signups, and revenue

## Install

```bash
npx clawhub@latest install blogburst
```

## What To Ask

```text
Find buyer questions for my product and turn them into proof-led posts.
Generate a Bluesky post from a real buyer pain point.
Show me which posts created clicks or signups.
Run a brand audit and tell me what AI search engines understand about us.
Draft a human-reviewed Reddit reply, but do not auto-post it.
```

## What Makes BlogBurst Different

Most tools start with "write me a post."

BlogBurst starts with "what real buyer question should this post answer, and can we track whether it worked?"

That is the product memory users should keep:

> buyer question → grounded post → quality gate → tracked click/signup → next experiment

## Safety

Do not paste or print `BLOGBURST_API_KEY`.

Do not use BlogBurst for automated Reddit or Hacker News promotion. Use it to draft useful replies, then review and post manually.
