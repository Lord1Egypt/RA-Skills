# 车辆年检沿途路线规划器

## 用途

上班路上顺路做车辆年检，从一批候选检测站中找出绕行最少、耗时最短的方案。输出带时间轴的系统性出行方案，包含每段的出发时间、到达时间、里程、耗时和逐条导航指引。

## 适用场景

- 车辆年检（也可用于任何"途中办事"的路线规划）
- 有固定的出发地和目的地（如家→公司）
- 有多个候选中途点（如检测站列表）
- 需要实时路况预估时间

## 输入参数

| 参数 | 说明 | 示例 |
|---|---|---|
| `home_address` | 出发地地址 | 浦东新区XX路XX弄 |
| `company_address` | 目的地地址 | 浦东新区XX路XX弄 |
| `depart_time` | 计划出发时间（24h制） | 07:30 |
| `inspect_duration` | 办事预计耗时（分钟） | 45 |
| `stations` | 候选站点列表（名称+地址），可从交管平台获取 | 见下方数据源 |
| `amap_api_key` | 用户的**高德Web服务API Key**（需用户自行申请，切勿硬编码） | 用户自行提供 |

> ⚠️ **安全提示**：API Key 属于敏感信息，必须由用户运行时提供，**绝不能硬编码在脚本或文档中**。

## 数据源

检测站列表可从以下渠道获取：
- [交通安全综合服务管理平台](https://122.gov.cn/) — 选择所在城市 → 业务办理 → 机动车检测站查询
- [交管12123 APP](https://122.gov.cn/#/app) — 检测站预约页面

## 工作流程

### 第1步：地理编码

对家庭地址、公司地址、每个候选检测站地址调用高德地理编码 API，获取经纬度坐标。

```
GET https://restapi.amap.com/v3/geocode/geo?key={API_KEY}&address={地址}&city={城市}
```

> 若地理编码失败（地址不规范），改用 POI 搜索接口：
> ```
> GET https://restapi.amap.com/v3/place/text?key={API_KEY}&keywords={地址关键词}&city={城市}&citylimit=true
> ```

### 第2步：计算直达基线

计算出发地→目的地的直达路线，作为对比基线。

```
GET https://restapi.amap.com/v3/direction/driving?key={API_KEY}&origin={起点}&destination={终点}&strategy=5&extensions=all
```

> `strategy=5` = 躲避拥堵且距离较短，含实时路况。

### 第3步：批量计算绕行路线

对每个候选站，分别计算「出发地→检测站」和「检测站→目的地」两段路线：

```
GET https://restapi.amap.com/v3/direction/driving?key={API_KEY}&origin={出发地}&destination={检测站}&strategy=5&extensions=all
GET https://restapi.amap.com/v3/direction/driving?key={API_KEY}&origin={检测站}&destination={目的地}&strategy=5&extensions=all
```

记录每段的距离、耗时和导航步骤。

### 第4步：排名与筛选

按总驾驶耗时（leg1 + leg2）排序，取前 N 名。同时标注：
- **最快方案**：总耗时最短
- **离家最近**：leg1 距离最短
- **里程最省**：总距离最短

### 第5步：生成时间轴方案

以用户指定的出发时间为基准，叠加办事耗时，生成完整时间轴：

```
出发时间 T0
  → 驾车 leg1（耗时 t1）→ 到达检测站 T1 = T0 + t1
  → 办事（耗时 inspect_duration）→ 离开检测站 T2 = T1 + inspect_duration
  → 驾车 leg2（耗时 t2）→ 到达目的地 T3 = T2 + t2
```

### 第6步：输出报告

输出 Markdown 格式报告，包含：
1. **Top N 总览表**（排名、站名、地址、各段里程/耗时、到达时间）
2. **每个方案的详细页面**：
   - 站点信息（名称、地址、具体位置、导航搜索词）
   - 时间轴表格
   - 逐段导航明细（路口级指引）
3. **注意事项**（预约、材料、违章处理等）

### 时间轴

  2. ...
```

## 注意事项

1. **API Key 安全**：高德 API Key 由用户运行时提供，不得硬编码或写入任何持久化文件。
2. **路况时效性**：高德返回的是查询时刻的实时路况。工作日早高峰与周末差异大，建议在工作日当天查询以获得准确预估。
3. **地理编码精度**：部分检测站地址可能无法直接地理编码，需用 POI 搜索补充。建议用检测站全称作为 POI 搜索关键词。
4. **年检预约**：建议提前在「交管12123」APP 预约，避免现场排队。需携带行驶证、交强险保单、身份证、三角警示牌，且提前处理交通违法。
