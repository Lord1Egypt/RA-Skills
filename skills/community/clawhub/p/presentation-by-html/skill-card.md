## Description: <br>
Use when the user needs to create a technical sharing, presentation, or talk material. Triggered by requests like "prepare a sharing", "make slides", "build a presentation", or any scenario requiring structured visual storytelling for an audience. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, engineers, and technical presenters use this skill to plan a talk through audience and narrative questions, then build an offline HTML slide deck with separated HTML, CSS, JavaScript, and interactive detail-on-demand components. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated presentations may include private screenshots, internal metrics, demo videos, or company-specific details. <br>
Mitigation: Review generated files and media before sharing, and remove or redact sensitive content. <br>
Risk: Interactive HTML slide decks can contain incorrect narrative framing or broken interactions. <br>
Mitigation: Confirm the presentation framework with the user, then preview and test navigation, tabs, and expandable sections before presenting. <br>
