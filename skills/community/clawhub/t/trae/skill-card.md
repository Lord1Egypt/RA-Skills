## Description: <br>
Guides agents in explaining and designing TRAE IDE skills, rules, context usage, and workflow automation patterns. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[DuanC-Chao](https://clawhub.ai/user/DuanC-Chao) <br>

## Use Case: <br>
Developers and teams use this skill to understand TRAE's skill architecture, create reusable agent workflows, and decide when to use skills, rules, context, sub-agents, and MCP integrations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Skills, scripts, MCP servers, or organization-wide rules created from this guidance can change agent behavior or access local files, project secrets, or external services. <br>
Mitigation: Review generated or third-party skill contents, permissions, and any embedded scripts before installation or use. <br>
