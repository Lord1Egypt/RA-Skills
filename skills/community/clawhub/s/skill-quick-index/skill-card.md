## Description: <br>
Build a wide-trigger, precise-match index for local OpenClaw skills (L1-L3), then quickly route by intent/category/keywords. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[halfmoon82](https://clawhub.ai/user/halfmoon82) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and agent users use this skill to query a maintained local OpenClaw skill index and identify candidate skills by natural-language intent, category, or keyword. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad keyword matching can return imprecise skill recommendations, including for higher-impact actions such as messaging, browser automation, configuration, or agent management. <br>
Mitigation: Use recommendations as a discovery aid only, and review the selected skill before running it. <br>
