# 结核病基本病理变化与免疫状态的关系分析

## 适用场景
- 患者已感染结核分枝杆菌
- 出现肺或其他器官的组织病理学改变
- 需根据免疫与病原特征判断病变主导类型

> **注意**：本技能不适用于非结核性肉芽肿性疾病。

## 执行步骤

1. **收集关键变量信息**：
   - 免疫力水平（低 / 较强）
   - 超敏反应强度（较弱 / 较强 / 强）
   - 菌量（少 / 多）
   - 毒力（较低 / 强）

2. **对照以下三类病理变化的判定条件，确定主导类型**：

   ### 渗出为主
   - **条件**：免疫力低、超敏反应较强、菌量多、毒力强
   - **病理特征**：浆液性或浆液纤维蛋白性炎，早期中性粒细胞浸润，后由巨噬细胞取代
   - **好发部位**：肺、浆膜、滑膜、脑膜

   ### 增生为主
   - **条件**：菌量少、毒力较低、免疫力较强、超敏反应较弱
   - **病理特征**：形成结核结节（结核性肉芽肿），含上皮样细胞、朗汉斯巨细胞、淋巴细胞及少量成纤维细胞，中央常有干酪样坏死
   - **诊断价值**：具有特异性诊断意义

   ### 坏死为主
   - **条件**：菌量多、毒力强、免疫力低或超敏反应强
   - **病理特征**：干酪样坏死，呈淡黄色、均匀细腻、质地较实，镜下为红染无结构颗粒状物
   - **临床意义**：坏死物中常含活菌，可致病情恶化或播散

3. **综合判断**：
   - 三种变化常共存，但以一种为主
   - 同一患者不同器官或同一器官不同时期病变可能转化
   - 输出主导病理类型及其对应的免疫-病原状态组合

4. **输出结果格式**：
   ```
   主导病理类型：[渗出/增生/坏死]
   对应状态：免疫力[低/较强]，超敏反应[较弱/较强/强]，菌量[少/多]，毒力[较低/强]
   ```
