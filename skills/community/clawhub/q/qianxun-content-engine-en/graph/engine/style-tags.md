## Conventions

1. Synonyms collapse to one canonical word ("premium-feel" = "premium" → use "premium")
2. Cross-genre tags: join with `+` — e.g., `Heritage-modern + Literary + Craft-driven`
3. New tag workflow: search dictionary first → if not present, add → mark "first source"
