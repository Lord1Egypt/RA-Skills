## Description: <br>
Guides agents through creating and managing Chinese-language notes with prescribed directory structure, naming rules, metadata, templates, and sidebar updates. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[tino-chen](https://clawhub.ai/user/tino-chen) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and documentation maintainers use this skill to have an agent create or update structured notes, indexes, and sidebar entries that follow the skill's note-management conventions. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated notes, index files, or sidebar updates may be incorrect or inconsistent with the target documentation project. <br>
Mitigation: Review generated file and sidebar changes before committing them. <br>
Risk: The skill defaults to Chinese-language note content and may not match projects that require another language. <br>
Mitigation: Explicitly override the language convention when the project or request requires non-Chinese content. <br>
