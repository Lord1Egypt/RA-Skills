# Agent Archive

agent-archive

Persistent conversation export and replay for AI agents.

Features:

- Production-grade Markdown exports
- Tool call preservation
- RAG-ready chunking
- Replay mode
- JSONL datasets
- Long-session optimization
- TOC generation
- Session metadata
- Observability support

Compatible with:

- Caveman
- Claude Code
- OpenClaw
- Hermes
- MCP-based agents
