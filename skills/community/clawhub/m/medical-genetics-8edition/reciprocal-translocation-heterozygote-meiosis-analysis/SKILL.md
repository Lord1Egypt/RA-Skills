# 相互易位杂合体减数分裂配子分析

## 适用条件
- 个体为两条非同源染色体间发生相互易位的杂合子
- 细胞处于减数分裂粗线期至后期
- 不适用于易位纯合子或无染色体结构异常的个体

## 执行步骤

1. **识别四射体结构**：在减数分裂粗线期，因同源区域联会，形成十字形四射体；随后可开放为环形或“8”字形结构。

2. **判断四种分离方式**：
   - **邻位分离1**：相邻的正常与易位染色体移向同一极，产生含重复和缺失的不平衡配子。
   - **邻位分离2**：另一组相邻染色体组合移向同一极，同样产生不平衡配子。
   - **相间分离（对位分离）**：两条正常染色体移向一极，两条易位染色体移向另一极，产生两个平衡配子（1个完全正常，1个平衡易位型）。
   - **3:1分离**：一条染色体单独移向一极，其余三条移向另一极，产生8种数目异常或结构不平衡的配子。

3. **汇总配子结果**：共可形成18种配子类型，其中仅2种为平衡配子（1正常 + 1平衡易位），其余16种均为不平衡配子。

4. **临床意义推导**：不平衡配子受精后常导致单体、部分单体、三体或部分三体合子，表现为流产、死胎或先天畸形。

## 输出模板
返回配子类型分布：18种中2种平衡（1正常+1平衡易位），16种不平衡；并据此评估生育风险及提供遗传咨询建议。
