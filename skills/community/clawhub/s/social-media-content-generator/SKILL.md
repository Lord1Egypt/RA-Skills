# Social Media Content Generator

Transform any blog post, URL, or topic into a complete set of platform-optimized social media posts. Covers Twitter/X threads, LinkedIn articles, Instagram captions, and Facebook posts — all in one run.

## When to use

Use this skill when the user needs to:
- Repurpose a blog post into social media content
- Generate a Twitter/X thread from an article or idea
- Write a LinkedIn post from long-form content
- Create Instagram and Facebook captions
- Build a week's worth of social content from a single piece
- Promote a product, tool, or service across multiple platforms

## How it works

1. Ask for the source content (URL, blog post text, or topic/idea)
2. Ask which platforms to generate for (default: all)
3. Ask for the tone (professional / casual / educational / promotional)
4. Ask for the target audience (freelancers / developers / marketers / general)
5. Generate all requested formats with platform best practices applied

## Output Formats

### Twitter/X Thread
- Hook tweet (max 280 chars, strong opening)
- 8–12 body tweets with one key point each
- Call-to-action closing tweet
- Each tweet numbered (1/, 2/, 3/...)
- Include relevant hashtags on final tweet only

### LinkedIn Post
- Strong opening line (no "I'm excited to share...")
- 150–300 words
- Line breaks every 2–3 sentences for readability
- 3–5 bullet points for key insights
- Closing question to drive comments
- 3–5 relevant hashtags at the end

### Instagram Caption
- Engaging opening line (first 125 chars are critical)
- 100–200 words
- Story-driven or value-driven angle
- Clear call to action ("Link in bio", "Save this post", "Tag a friend")
- 20–30 hashtags in the first comment (provide separately)

### Facebook Post
- Conversational tone
- 100–250 words
- Question or poll option at the end
- 2–3 hashtags maximum

## Content Principles

- Never start with "I" on LinkedIn
- Use active voice throughout
- Front-load value — give the insight before explaining it
- Each platform post must be standalone (assume the reader hasn't seen the others)
- Avoid hashtag stuffing on Twitter/X and LinkedIn

## Example Interaction

**User:** Here's my blog post about free invoice generators. Make social content for Twitter and LinkedIn.

**Agent:**
1. Reads/receives the blog post
2. Extracts 5–7 key insights
3. Writes Twitter thread (10 tweets):
   - Tweet 1: "Freelancers waste 2+ hours/month on invoices. Here are 7 free tools that fix that: 🧵"
   - Tweet 2: "1/ InvoiceCraft — No signup, generates PDFs instantly..."
   - ...
4. Writes LinkedIn post with professional angle and engagement hook
