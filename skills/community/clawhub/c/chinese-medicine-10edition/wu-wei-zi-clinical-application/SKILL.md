# Wu Wei Zi Clinical Application

## When to Use
- 患者表现为气虚、阴虚、肾虚或心神不宁相关证候。

## Precautions
- 表邪未解者
- 内有实热者
- 咳嗽初起者
- 麻疹初期者

## Execution Steps
1. **辨证分型并配伍**：
   - **收敛固涩**：
     • 气虚自汗：配伍白术、黄芪、浮小麦、麻黄根
     • 肺虚久咳：配伍罂粟壳
     • 遗精滑泄：配伍桑螵蛸、龙骨、山茱萸
     • 脾肾虚寒久泻：配伍吴茱萸、补骨脂、肉豆蔻
   - **益气生津**：
     • 阴虚内热消渴：配伍人参、麦冬、知母、天花粉
     • 热病后期气阴两伤：配伍人参、麦冬
   - **补肾宁心**：
     • 心悸失眠（阴血不足）：配伍酸枣仁、茯神、远志

2. **确定用法用量**：
   - 煎服：2～6克
   - 研末服：每次1～3克

3. **验证禁忌**：
   - 若存在表邪未解、内有实热、咳嗽初起或麻疹初期，禁止使用五味子，以防酸收敛邪、闭门留寇。

4. **输出建议**：
   - 根据证型选择相应配伍方案，严格控制剂量，并确认无禁忌后方可使用。
