# Tongue Observation Clinical Interpretation

## When to Use
- 中医临床辨证；
- 疾病预后评估。

## When to Use
- 已完成舌质与舌苔的独立观察；
- 必须结合问诊、闻诊、切诊等其他三诊信息（四诊合参）。

## Execution Steps

### 1. 明确舌质与舌苔分工
- **舌质**反映脏腑虚实、气血盛衰（如舌淡白=气血亏虚，舌红润=气血旺盛）；
- **舌苔**反映病邪寒热、深浅及邪正消长（如苔薄白=病在表，苔厚=邪入里）。

### 2. 判断正气盛衰
- 舌质红润 + 苔薄白而润 → 正气内存，胃气旺盛；
- 舌质淡白 → 气血亏虚；
- 舌光无苔（镜面舌）→ 胃之气阴衰败。

### 3. 辨病位深浅
- 苔薄白 → 病在表；
- 苔厚 → 邪入里；
- 舌质绛 → 热入营血，病情危重。

### 4. 区别病邪性质
- 白苔 → 寒证；黄苔 → 热证；腐腻苔 → 食积、痰浊；青紫舌/瘀斑 → 瘀血。

### 5. 推断病势进退
- 苔由白→黄→灰黑 → 病邪由表入里，病情加重；
- 苔由润→燥 → 热盛伤津；
- 苔由厚→薄、燥→润 → 邪退正复。

### 6. 预测预后
- 舌胖瘦适中、活动自如、淡红润泽、有苔 → 预后佳；
- 舌质枯晦、苔骤剥、舌强/偏歪 → 正气亏损，胃气衰败，预后不良。

### 7. 注意事项
- 若舌质与舌苔变化不一致（如舌淡白但苔黄厚），必须结合全身症状及其他三诊信息综合分析，不可孤立依赖舌象；
- 严格遵循“四诊合参”原则，避免片面结论。

## Output
指导临床辨证方向、治疗策略调整及预后告知。
