# 高渗性脱水的补液原则与速度控制

## 何时使用
当患者符合以下任一情况时应用：
- 诊断为高渗性脱水（血清Na⁺ >150 mmol/L 或 血浆渗透压 >310 mOsm/(kg·H₂O)）且无休克；
- 高渗性脱水合并休克（需先快速扩容再缓慢纠水）。

**禁忌/慎用**：严重心功能不全者需调整补液速度；若存在休克，必须优先处理循环不稳定。

## 执行步骤
1. **分阶段处理**：
   - **第一阶段（如有休克）**：快速静脉输注等渗液体（如0.9%氯化钠）恢复血容量；
   - **第二阶段（循环稳定后）**：使用5%葡萄糖溶液静脉输注（能口服者优先口服）逐步纠正水缺乏。

2. **估算补液总量**：
   - 每丧失体重1%，补液400–500 ml；
   - 总量 = 已丧失量 + 不显性失水 + 尿量 + 胃肠道持续丢失量。

3. **严格控制纠正速度**：
   - 血钠下降速率不得超过0.5–1.0 mmol/(L·h)；
   - 过快可致脑水肿、癫痫甚至死亡。

4. **监测与调整**：
   - 密切观察意识、血压、全身状况；
   - 定期检测血钠浓度，据此调整后续补液方案。

> **特别提醒**：高渗性脱水患者体内总钠实际减少，补水同时应适当补钠，防止转为低钠血症。
