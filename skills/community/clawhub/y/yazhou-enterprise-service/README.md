# 崖州区企业服务技能 (yazhou-enterprise-service)

崖州区企业服务工作人员专用技能，提供一站式企业服务，包括企业信息查询、优惠政策推荐、安全生产监管、企业发展建议等。

## 功能特性

- 🏢 **企业信息查询**：通过企业名称或统一社会信用代码查询工商信息、经营状态、资质等
- 📋 **优惠政策推荐**：根据企业类型/行业/规模，精准匹配适用优惠政策
- 🔒 **安全生产监管**：生成安全生产监管清单和监管报告
- 📈 **企业发展建议**：结合崖州区产业发展情况，为企业提供发展建议
- 📊 **上市公司财报查询**：支持查询上市公司财务数据（通过 neodata-financial-search 技能）
- 📝 **多企业报告生成**：支持同时为多家企业生成服务报告（Word 文档输出）

## 使用方法

1. 安装本技能后，直接向 AI 助手提出企业服务需求
2. 示例：
   - "查询卓世科技的基本情况"
   - "推荐适合科技型中小企业的政策"
   - "为海南迈思科技生成企业服务报告"
   - "崖州区产业发展重点是什么？"

## 输出格式

所有服务报告均以 Word 文档（.docx）输出，支持多企业批量报告生成。

## 参考资料

本技能包含以下参考资料（位于 `references/` 目录）：

- `policies.md` - 崖州区及上级优惠政策汇总（221 条企业相关政策）
- `safety-regulations.md` - 安全生产监管法规依据
- `enterprise-types.md` - 企业类型与政策匹配指南
- `industrial-development.md` - 崖州区产业发展情况与规划

## 依赖技能

本技能需要以下技能配合使用：

- `docx` - 生成 Word 文档报告
- `neodata-financial-search` - 查询上市公司财务数据（可选）
