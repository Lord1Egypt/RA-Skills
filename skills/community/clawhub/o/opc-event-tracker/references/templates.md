# 赛事记录模板

## 赛事记录文件格式

文件路径：`公司经营/06-客户管理/赛事活动追踪/赛事记录.md`

```markdown
# 赛事活动记录

> 最后更新：YYYY-MM-DD

## 进行中 / 即将开始

| 赛事名称 | 类型 | 报名截止 | 比赛时间 | 奖品 | 链接 | 发现日期 |
|---------|------|---------|---------|------|------|---------|
| 示例赛事 | AI黑客松 | 2026-07-01 | 2026-08-15 | $5000 | https://... | 2026-06-22 |

## 已结束（归档）

| 赛事名称 | 类型 | 比赛时间 | 奖品 | 链接 |
|---------|------|---------|------|------|
```

---

## 输出格式（给用户看的摘要）

当有新赛事时，输出以下格式：

```markdown
🏆 发现 X 个新赛事

1. **[赛事名称]**
   - 类型：OPC / AI黑客松 / 创业大赛
   - 报名截止：YYYY-MM-DD（⚠️ 已临近 / 还有X天）
   - 比赛时间：YYYY-MM-DD
   - 奖品：[奖金/资源/荣誉]
   - 链接：[URL]

2. ...
```

## 定时任务 Prompt 模板

用于创建 automation 时的 prompt：

```
帮我追踪 OPC/AI/独立开发者相关赛事活动。
执行步骤：
1. 用 WebSearch 搜索以下关键词（当前年份）：
   - "OPC 独立先锋挑战赛 2026"
   - "Kaggle competitions 2026 open"
   - "Devpost hackathon 2026"
   - "独立开发者大赛 2026"
   - "Indie Hackers Awards 2026"
2. 解析搜索结果，提取赛事名称、报名时间、比赛时间、奖品、链接
3. 读取本地记录文件（如存在）：`公司经营/06-客户管理/赛事活动追踪/赛事记录.md`
4. 比对，找出新增赛事（本地没有的）
5. 如有新增赛事，更新记录文件，并输出新增赛事摘要提醒用户
6. 如无新增赛事，输出"暂无新赛事"
```
