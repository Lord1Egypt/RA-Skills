# 内科急症识别与急救：癫痫、心梗、卒中

> **⚠️ 免责声明**：本文仅供参考，不能替代专业急救培训。出现疑似心梗/卒中请立即拨打120，时间就是生命。

---

## 第一部分：癫痫急救

### 一、识别癫痫大发作

- 突然意识丧失、倒地
- 全身强直（僵硬）→ 阵挛（抽搐）
- 口吐白沫/血沫（可能咬伤舌头）
- 双眼上翻
- 大小便失禁
- 通常持续 **1-3分钟** 自行停止

### 二、癫痫急救 — "五要五不要"

#### ✅ 五要

| 要做 | 怎么做 |
|------|--------|
| 保持镇定 | 癫痫发作通常自行停止 |
| 移开危险物 | 桌椅、尖锐物、热源 |
| 头部保护 | 垫软物（衣服/包）防头部撞地 |
| 计时 | 记住发作开始时间 |
| 侧卧位 | 发作停止后转为侧卧（恢复体位） |

#### ❌ 五不要

| 不要做 | 为什么 |
|--------|--------|
| **不要往嘴里塞任何东西！** | 不会"吞舌头"！塞物可致牙齿断裂/窒息 |
| **不要按压约束肢体** | 可导致骨折/脱臼 |
| **不要掐人中** | 无效 |
| **不要人工呼吸** | 发作期呼吸暂停是暂时的 |
| **不要喂水喂药** | 窒息风险 |

### 三、需要拨打120的情况

- 发作持续 **> 5分钟**（癫痫持续状态）
- 一次发作停止后未恢复意识又再次发作
- 发作在水中发生
- 患者是孕妇/糖尿病患者
- 发作后有外伤
- 这是患者**第一次**发作

## 第二部分：急性心肌梗死

### 一、心梗典型症状

| 典型症状 | 描述 |
|----------|------|
| 胸痛 | 胸骨后压榨性、闷痛，持续>15分钟 |
| 放射痛 | 向左肩/左臂/下颌/背部放射 |
| 伴随症状 | 大汗（冷汗）、恶心呕吐、濒死感、呼吸困难 |

### 二、非典型症状（更易被忽视！）

- **女性**：疲劳、气短、上腹痛、恶心（比男性更不典型）
- **老年人**：突发呼吸困难、意识模糊、无力
- **糖尿病患者**：无痛性心梗（神经病变导致痛觉减退）

### 三、心梗急救 — "静卧-服药-CPR-AED"

| 步骤 | 操作 | 要点 |
|------|------|------|
| **1. 静卧** | 立即停止一切活动，半卧位（上身抬高） | 减少心肌耗氧 |
| **2. 拨打120** | 说清地址+症状+已做处理 | **不要自己开车去医院！** |
| **3. 服药** | 硝酸甘油0.5mg舌下含服（如患者有处方） | **确认无禁忌症**：低血压（<90/60）、24小时内用过伟哥类药物禁止使用硝酸甘油！ |
| **4. 服药** | 阿司匹林300mg嚼碎吞服（如无禁忌） | 阿司匹林过敏/活动性出血禁用 |
| **5. CPR+AED** | 如患者意识丧失、无正常呼吸 → 立即CPR+AED | 心梗最易发生室颤，AED是关键！ |

### 四、心梗黄金120分钟

- 从发病到血管开通（PCI/溶栓）**< 120分钟**最佳
- 每延误30分钟，死亡率增加7.5%
- 所以：**拨打120，不要自己开车/打车！** 救护车上有急救设备和心电图

### 五、心绞痛 vs 心梗

| | 心绞痛 | 心梗 |
|------|--------|------|
| 诱因 | 劳累、情绪激动 | 安静时也可发生 |
| 持续时间 | <15分钟 | >15分钟 |
| 休息/服药缓解 | 通常缓解 | **不缓解** |
| 心肌坏死 | 无 | **有** |

## 第三部分：脑卒中（中风）

### 一、快速识别 — FAST原则

| 字母 | 检查 | 操作 |
|------|------|------|
| **F**ace（面部） | 面部不对称？ | 让患者**微笑**，观察一侧口角是否歪斜 |
| **A**rm（手臂） | 一侧无力？ | 让患者**闭眼双臂平举10秒**，一侧掉落为阳性 |
| **S**peech（语言） | 说话含糊？ | 让患者**重复一句简单的话** |
| **T**ime（时间） | **马上打120！** | 溶栓黄金时间窗：**发病4.5小时内** |

### 二、中风其他警示信号（BEFAST扩展）

- **B**alance（平衡）：突发行走不稳、眩晕
- **E**yes（眼睛）：突发单眼或双眼视物模糊/视野缺损

### 三、中风急救流程

1. **拨打120**，告知"疑似中风"
2. 平卧，头部抬高约30°
3. 头偏向一侧（防止呕吐窒息）
4. 解开领扣领带，保持呼吸道通畅
5. 记录**发病时间**（最后正常时间）——这对溶栓决策至关重要！
6. **不要给任何口服药物/食物/水**（可能有吞咽困难导致窒息）

### 四、中风急救禁忌

| ❌ | ✅ |
|----|-----|
| 摇晃患者"唤醒" | 保持安静，减少搬动 |
| 喂安宫牛黄丸等药物 | 禁食禁水！ |
| 掐人中 | 无效且可能造成伤害 |
| 等待症状自行缓解 | 时间就是大脑，立即120 |
| 自行开车送医 | 救护车可提前通知医院准备卒中绿色通道 |

### 五、出血性 vs 缺血性中风

| | 缺血性（脑梗） | 出血性（脑出血） |
|------|-----------|------------|
| 占比 | ~70% | ~30% |
| 起病 | 安静时/睡眠中多发 | 活动/情绪激动时多发 |
| 头痛 | 较少 | 剧烈头痛 |
| 呕吐 | 较少 | 多见（喷射状） |
| 意识 | 较轻 | 较重，可迅速昏迷 |

**现场无法区分，处理原则相同：禁食水、保持气道、马上120。**
