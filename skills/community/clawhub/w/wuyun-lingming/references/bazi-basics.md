## 一、八字的组成

八字由**四柱**组成，每柱包含一个天干和一个地支，共八个字：

| 四柱 | 代表 | 时间跨度 |
|------|------|----------|
| **年柱** | 祖父辈、家庭环境、童年 | 出生年份 |
| **月柱** | 父母、社会关系、职业 | 出生月份 |
| **日柱** | 自身、本质、婚姻 | 出生日期 |
| **时柱** | 子女、晚年、成就 | 出生时辰 |

### 天干（10个）
```
甲、乙、丙、丁、戊、己、庚、辛、壬、癸
```
- **阳干**：甲、丙、戊、庚、壬
- **阴干**：乙、丁、己、辛、癸

### 地支（12个）
```
子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥
```
- **阳支**：子、寅、辰、午、申、戌
- **阴支**：丑、卯、巳、未、酉、亥

## 三、排八字方法

### 1. 排年柱
- 以**立春**为分界（不是正月初一）
- 例：2026年立春后为**丙午**年

### 2. 排月柱
- 地支固定：正月寅、二月卯、三月辰...十二月丑
- 天干由年干决定，记住口诀**「五虎遁」**：

```
甲己之年丙作首（甲年/己年 → 正月丙寅）
乙庚之岁戊为头（乙年/庚年 → 正月戊寅）
丙辛必定寻庚上（丙年/辛年 → 正月庚寅）
丁壬壬寅顺水流（丁年/壬年 → 正月壬寅）
若问戊癸何处起（戊年/癸年 → 正月甲寅）
```

### 3. 排日柱
- **最复杂**，需查万年历
- 日干支60天一循环，大小月、平闰年不同
- 子时（23:00）为日分界

### 4. 排时柱
- 地支固定：
  ```
  子时 23-1点 | 丑时 1-3点 | 寅时 3-5点 | 卯时 5-7点
  辰时 7-9点  | 巳时 9-11点| 午时 11-13点| 未时 13-15点
  申时 15-17点| 酉时 17-19点| 戌时 19-21点| 亥时 21-23点
  ```
- 天干由日干决定，记住口诀**「五鼠遁」**：

```
甲己还加甲（甲日/己日 → 子时甲子）
乙庚丙作初（乙日/庚日 → 子时丙子）
丙辛从戊起（丙日/辛日 → 子时戊子）
丁壬庚子居（丁日/壬日 → 子时庚子）
戊癸推壬子（戊日/癸日 → 子时壬子）
```

## 四、日主（五行属性）

### 日主定义
- **日主 = 日柱的天干**，代表命主自身
- 日干的五行就是命主的五行属性

### 日主五行与性格特征

| 日主 | 五行 | 性格特点 | 喜忌 |
|------|------|----------|------|
| **甲木** | 阳木 | 正直、进取、领导力强、易固执 | 喜水火调候，忌金过克 |
| **乙木** | 阴木 | 细腻、灵活、善交际、易依赖 | 喜丙癸，忌土重 |
| **丙火** | 阳火 | 热情、慷慨、有感染力、易急躁 | 喜壬水，忌土多 |
| **丁火** | 阴火 | 敏锐、专注、注重细节、易多虑 | 喜甲木，忌癸水 |
| **戊土** | 阳土 | 务实、包容、守信、易保守 | 喜甲木壬水，忌土重 |
| **己土** | 阴土 | 温和、隐忍、善协调、易优柔 | 喜丙火癸水，忌木过 |
| **庚金** | 阳金 | 果决、坚韧、重义气、易过露 | 喜火炼金，忌木多 |
| **辛金** | 阴金 | 聪慧、敏感、追求完美、易计较 | 喜壬水，忌土厚 |
| **壬水** | 阳水 | 豁达、机智、适应力强、易散漫 | 喜戊土，忌火炎 |
| **癸水** | 阴水 | 内敛、洞察力强、善谋略、易感伤 | 喜辛金，忌土重 |

### 日主强弱判断三要素

1. **得令**：日主五行与月令一致或得生扶（如甲木生寅月）
2. **得地**：地支中有同类本气根（如甲木见寅）
3. **得势**：天干见比肩/劫财助阵（如甲木见甲乙）

## 五、五行生克关系

### 相生
```
金生水 → 水生木 → 木生火 → 火生土 → 土生金
```

### 相克
```
金克木 → 木克土 → 土克水 → 水克火 → 火克金
```

### 地支六冲
```
子午冲 | 丑未冲 | 寅申冲 | 卯酉冲 | 辰戌冲 | 巳亥冲
```

### 地支六合
```
子丑合 | 寅亥合 | 卯戌合 | 辰酉合 | 巳申合 | 午未合
```

## 六、八字五行分布统计

分析一个八字时，需要统计：
- 各五行（木火土金水）各有几个天干地支
- 日主强弱（身强/身弱）
- 五行是否齐全（缺什么五行）

### 示例
假设八字为：甲子年、丙寅月、丁丑日、戊申时

| 五行 | 天干 | 地支 | 合计 |
|------|------|------|------|
| 木 | 甲(1) | 寅(1) | 2 |
| 火 | 丙(1)、丁(1) | 午(1) | 3 |
| 土 | 戊(1)、己(0) | 辰丑(2) | 3 |
| 金 | 庚(0) | 申(1) | 1 |
| 水 | 癸(0) | 子(1) | 1 |
