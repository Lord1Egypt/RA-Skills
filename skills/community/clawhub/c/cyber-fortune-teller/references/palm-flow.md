# 手相学标准分析流程

> 本文件为AI分析手相提供可执行的标准流程，每一步都有明确的操作指令和判断标准。

## 第一步：手型判断

### 操作指令

```
INPUT: 手掌照片或描述
PROCESS:
  1. 观察手掌整体形状
  2. 判断手掌宽度与长度比例
  3. 观察指尖形态
  4. 确定手型类型
OUTPUT: 手型类型
```

### 手型判定标准

| 手型 | 掌形特征 | 指形特征 | 性格倾向 |
|------|---------|---------|---------|
| 方形手 | 手掌宽大、呈长方形 | 指节分明、指尖略方 | 务实、踏实、有耐心 |
| 圆型手 | 手掌圆润、呈椭圆形 | 指头略尖、指腹圆润 | 灵活、适应、创意 |
| 锥型手 | 手掌窄长、呈锥形 | 指尖呈锥、指根细 | 直觉、热情、艺术 |
| 混合型 | 兼有两种以上特征 | 混合特征 | 灵活多变 |

### 手型分析示例

**案例1：方形手**
```
手掌宽大呈长方形
指节分明、指甲方正
手掌坚实、肌肉发达
→ 方形手：务实型性格
```

**案例2：圆型手**
```
手掌圆润呈椭圆形
指头略尖、指腹饱满
手掌柔软、皮肤细腻
→ 圆型手：社交型性格
```

## 第二步：三大主线分析

### 操作指令

```
INPUT: 手掌照片或描述
PROCESS:
  1. 逐一分析生命线、智慧线、感情线
  2. 评估各线长度、深浅、形态
  3. 识别特殊符号（岛纹、断裂等）
OUTPUT: 三大主线分析结果
```

### 生命线分析（三线）

**位置**：拇指与食指间，围绕拇指根部呈弧形

**分析要点**：

| 维度 | 判定标准 | 吉凶 | 说明 |
|------|---------|------|------|
| 长度 | 起于食指与拇指间，止于手腕 | 长者寿 | 长度代表寿命 |
| 深浅 | 纹路深浅程度 | 深者精力旺 | 深度代表生命力 |
| 弧度 | 弧形大小 | 大者精力充沛 | 弧度代表精力 |
| 形态 | 连续/断续/链状 | 连续者稳定 | 形态代表健康 |
| 符号 | 岛纹/断裂/分叉 | 岛纹代表疾病 | 符号代表问题 |

**生命线形态判定表**：

| 形态 | 特征 | 判定 |
|------|------|------|
| 长而深 | 长度超过中指中线，纹路深 | 吉：长寿精力旺 |
| 短而深 | 长度未达中指中线，纹路深 | 中：精力旺但寿数有限 |
| 长而浅 | 长度长但纹路浅 | 中：寿长但精力一般 |
| 短而浅 | 长度短且纹路浅 | 凶：需注意健康 |
| 链状 | 纹路呈链条形状 | 凶：体质较弱 |
| 岛纹 | 纹路中有岛形纹 | 凶：健康问题 |
| 断裂 | 纹路有明显断裂 | 凶：重大健康变故 |
| 分叉 | 线尾分叉 | 中：晚年精力下降 |

### 智慧线分析（二线）

**位置**：手掌中央，始于食指与中指之间，止于小鱼际

**分析要点**：

| 维度 | 判定标准 | 吉凶 | 说明 |
|------|---------|------|------|
| 长度 | 从食指与中指间到小指根部的距离 | 长者思维活跃 | 长度代表智慧活跃度 |
| 深浅 | 纹路深浅程度 | 深者思考深入 | 深度代表思考深度 |
| 起点 | 起点的位置和形态 | 位置高者早慧 | 起点代表早慧程度 |
| 终点 | 终点的位置和形态 | 终点代表事业 | 终点代表事业方向 |
| 形态 | 连续/断续/弯曲 | 形态代表思维 | 形态代表思维方式 |
| 符号 | 岛纹/断裂/叉纹 | 符号代表问题 | 符号代表思维困扰 |

**智慧线形态判定表**：

| 形态 | 特征 | 判定 |
|------|------|------|
| 长而清晰 | 长度长、纹路清晰 | 吉：智慧高远 |
| 短而清晰 | 长度短、纹路清晰 | 中：思维直接务实 |
| 长而模糊 | 长度长、纹路模糊 | 中：想太多执行力差 |
| 弯曲向上 | 线尾向上弯曲 | 吉：创意丰富 |
| 弯曲向下 | 线尾向下弯曲 | 中：思考深入但消极 |
| 与生命线相连 | 起点与生命线相连 | 中：实践型思维 |
| 与感情线相连 | 起点与感情线相连 | 中：感性思维 |
| 岛纹 | 纹路中有岛形纹 | 凶：思维困扰 |
| 断裂 | 纹路有明显断裂 | 凶：思维中断 |

### 感情线分析（一线）

**位置**：手掌上方，始于小指下方，止于食指与中指之间

**分析要点**：

| 维度 | 判定标准 | 吉凶 | 说明 |
|------|---------|------|------|
| 长度 | 从小指下方到食指与中指间的距离 | 长者感情丰富 | 长度代表感情丰富度 |
| 深浅 | 纹路深浅程度 | 深者感情真挚 | 深度代表感情深度 |
| 位置 | 距离掌边的位置 | 高者感情表达含蓄 | 位置代表表达方式 |
| 弧度 | 弧形向上或向下 | 上者感情主动 | 弧度代表主动性 |
| 形态 | 连续/断续/链状 | 形态代表稳定度 | 形态代表感情状态 |
| 符号 | 岛纹/断裂/羽毛 | 符号代表问题 | 符号代表感情问题 |

**感情线形态判定表**：

| 形态 | 特征 | 判定 |
|------|------|------|
| 长而清晰 | 长度长、纹路清晰 | 吉：感情丰富真诚 |
| 短而清晰 | 长度短、纹路清晰 | 中：感情专一务实 |
| 长而模糊 | 长度长、纹路模糊 | 中：感情复杂敏感 |
| 向上延伸 | 线尾向上延伸 | 中：感情主动 |
| 向下弯曲 | 线尾向下弯曲 | 中：感情被动 |
| 链状 | 纹路呈链条形状 | 凶：感情波折 |
| 岛纹 | 纹路中有岛形纹 | 凶：感情困扰 |
| 断裂 | 纹路有明显断裂 | 凶：感情创伤 |
| 羽毛状 | 边缘有细羽毛纹 | 中：细腻敏感 |

位置：食指下方
丰隆表现：丘位高耸、肌肉饱满

含义：有野心、有领导力、事业心强

## 第四步：手指分析

### 操作指令

```
INPUT: 手掌照片或描述
PROCESS:
  1. 逐一观察五指
  2. 评估各指的长短、形态、位置
  3. 判断各指代表的运势
OUTPUT: 手指分析结果
```

### 五指分析表

| 手指 | 对应 | 位置关系 | 长度标准 | 形态标准 |
|------|------|---------|---------|---------|
| 大拇指 | 父/君 | 拇指位置 | 粗壮有力 | 指节分明 |
| 食指 | 兄/官 | 食指位置 | 长直挺拔 | 指节均匀 |
| 中指 | 己/夫 | 中指位置 | 长于其他四指 | 指腹圆润 |
| 无名指 | 耦/财 | 无名指位置 | 长而直 | 指腹饱满 |
| 小指 | 卑/后 | 小指位置 | 长过无名指第一节 | 笔直挺拔 |

### 手指长度比例分析

```
长度比例判定：
- 无名指长于食指：艺术型、财运佳
- 食指长于无名指：权力型、事业心强
- 中指最长：自我意识强、独立性好
- 小指长过无名指第一节：晚运好、子女缘分深
- 小指短于无名指第一节：晚年运差、子女缘分浅
```

### 手指分析示例

**案例：大拇指分析**
```
长度：拇指长度达食指第二节
粗细：拇指粗壮有力
指节：两节分明，关节明显
→ 判定：吉
含义：意志力强、有魄力、自制力好
```

## 第五步：辅纹分析

### 操作指令

```
INPUT: 手掌照片或描述
PROCESS:
  1. 观察命运线（事业线）
  2. 观察成功线（太阳线）
  3. 观察婚姻线
  4. 评估辅助纹路的状态
OUTPUT: 辅纹分析结果
```

### 辅助纹路分析表

| 纹路 | 位置 | 有无判定 | 形态判定 | 含义 |
|------|------|---------|---------|------|
| 命运线 | 掌中部纵线 | 有/无 | 直/曲/断 | 事业命运 |
| 成功线 | 无名指下方纵线 | 有/无 | 清晰/模糊 | 名声成就 |
| 婚姻线 | 小指下方短横线 | 多少 | 长短/曲直 | 婚姻恋爱 |
| 旅行线 | 掌外侧横线 | 有/无 | 深浅/长短 | 出行迁移 |
| 子女线 | 小指下方细线 | 多少 | 长短/深浅 | 子嗣缘分 |
| 债务线 | 命运线旁斜线 | 有/无 | 深浅/长短 | 财务压力 |

### 婚姻线详细分析

```
婚姻线判定：
- 多条婚姻线：多次恋爱或婚姻
- 长而清晰：婚姻稳定
- 短而模糊：婚姻缘分浅
- 向上弯曲：晚婚
- 向下弯曲：早婚
- 两条交叉：婚姻有波折
- 无婚姻线：婚姻缘分淡薄
```
