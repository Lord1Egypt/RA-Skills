## Description: <br>
Tracks cryptocurrency portfolios across TRON, Ethereum, and BSC with balance checks, holdings analysis, yield calculations, gas-fee tracking, and optional price alerts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and agents use this skill to review crypto wallet balances and portfolio activity across TRON, Ethereum, and BSC. It supports read-only portfolio analysis workflows such as token balance checks, holdings summaries, yield review, gas-fee tracking, and swap-analysis context. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Wallet addresses can expose holdings and transaction history. <br>
Mitigation: Treat wallet addresses as sensitive and use only trusted read-only blockchain tools or data providers. <br>
Risk: Requests for seed phrases, private keys, or signing permissions could compromise assets. <br>
Mitigation: Never provide seed phrases, private keys, or signing permissions when using this skill. <br>
