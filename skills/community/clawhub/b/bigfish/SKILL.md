# 🎣 BigFish · 钓鱼AI助手

为钓鱼爱好者（野钓/路亚/台钓）提供 AI 分析支持的 ClawHub Skill。

## 📋 功能概览

BigFish 是一款 **AI 钓鱼助手**，结合水面分析、天气数据、鱼种知识，推荐最佳钓点和钓法。

**核心功能：**
- **📸 水面分析** — 拍水面照片，AI 分析水质、水草、鱼情
- **🌤️ 天气融合** — 结合当地气温、气压、风向、潮汐推荐最佳出钓时间
- **🎣 鱼种推荐** — 根据季节、地域、天气推荐目标鱼种
- **🪝 钓法建议** — 路亚/台钓/野钓/海钓，饵料搭配和装备推荐
- **📝 出钓报告** — 自动记录渔获数据，积累个人钓鱼日志

## 核心功能

### 1. 🔬 钓点分析

用户发钓点照片（水面/环境），可选提供城市/位置。

**分析流程：**

```
解析图片 → 识别环境特征（水草/水质/流速/障碍物/岸边地形）
   ↓
获取用户位置（问城市或直接提供）
   ↓
weather skill 查当日天气/气压/风向/温度/季节
   ↓
输出综合推荐：
  · 适合鱼种 + 概率评分
  · 建议钓法（路亚/台钓/海杆/矶钓）
  · 建议饵料（拟饵/活饵/商品饵）
  · 推荐下杆位置（草边/回流/深浅交接/障碍物旁）
  · 出钓时机建议（现在去还是等等）
```

**⚠️ 原则：** 不做"AI透视眼看穿鱼在哪"，做"环境+天气+知识库的有依据推荐"，类似 BigA 评分体系。

### 2. 📝 钓鱼记录

帮用户记录每次出钓，格式示例：

```
🎣 出钓记录
📅 日期：2026-05-27
📍 钓点：深圳西丽水库
🌡 天气：多云 28°C 东南风3级
⏰ 时段：06:00-10:00
🎯 目标：翘嘴
🐟 鱼获：翘嘴×3（最大45cm）/ 白条×5
🎣 装备：ML调路亚竿 + 10g铁板
💬 备注：早口不错，9点后停口
```

### 3. ⚡ 出钓时机

用户问"今天适合钓鱼吗"时：

```
查 weather → 当前天气+未来6h预报
分析：气压变化趋势 / 风力风向 / 温差 / 降水概率
输出：⭐⭐⭐⭐⭐ 评分 + 建议
```

### 4. 📊 鱼情报告

积累用户的出钓记录后，可以给：月度报告/个人最佳/钓点排行榜。

## 钓鱼常识知识库（参考要点）

### 鱼种活跃条件

| 鱼种 | 最佳水温 | 适宜气压 | 活跃时段 |
|:----|:--------|:--------|:--------|
| 翘嘴 | 18-28°C | 1005-1020hPa 稳定 | 清晨/黄昏/夜 |
| 鳜鱼 | 18-26°C | 稳定或缓慢上升 | 清晨/阴天 |
| 鲈鱼 | 15-25°C | 1005-1020hPa | 早晚 |
| 草鱼 | 20-30°C | 稳定 | 全天（夏季早晚） |
| 鲤鱼 | 18-28°C | 稳定或微降 | 清晨/傍晚 |
| 鲫鱼 | 15-25°C | 稳定 | 全天 |
| 罗非 | 22-32°C | 不限 | 全天（天热活跃） |
| 黑鱼 | 20-30°C | 不限 | 夏季全天 |

### 天气对钓鱼的影响

| 天气 | 影响 | 建议 |
|:----|:----|:-----|
| 气压骤降 | 鱼上浮不开口 | 不宜出钓 |
| 气压稳定/微升 | 鱼活跃 | 好时机 |
| 东南风/南风 | 水中含氧低 | 鱼情一般 |
| 北风/东北风 | 含氧高 | 好时机 |
| 大雨前 | 气压低 | 不宜 |
| 小雨/阵雨 | 含氧高+安全感 | 好时机 |
| 连续阴天 | 全天都可钓 | 好时机 |
