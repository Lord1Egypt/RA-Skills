# Enterprise Solution Title-Bar Template

Use this reference when starting a new PPT PNG production task or when enforcing title consistency across generated slides.

## Asset

- `assets/titlebar-template/enterprise-solution-titlebar-4k.png`
- Size: `3840x190`
- Format: RGB PNG
- Intended for `3840x2160` 16:9 PPT pages and as a style reference for 2K pages.

## Visual Style

The title bar is a compact pale blue-white enterprise header band:

- Left blue slanted emblem block
- Two white slanted strokes inside the emblem
- Navy bold Chinese main title
- Subtle wave-line ornament on the right
- Light dotted technology texture on the right
- No page number, logo, watermark, or industry-specific wording

## Usage Rules

- Confirm this title-bar style in the first sample round.
- Once confirmed, reuse the same main title-bar style on every normal content page.
- Cover, directory, and section divider pages may vary, but normal content pages must stay consistent.
- Replace only the main title text per page.
- The title bar must contain exactly one line of title text.
- Never wrap the title into two or more lines.
- The single-line title must be vertically centered in the title bar.
- If the title is too long, shorten the wording, reduce font size within the confirmed style, or ask the user for a shorter title; do not make a two-line title bar.
- Do not copy any domain words from the source project that originally provided the asset.
- If the user asks for a different title style, create and confirm a new title-bar reference before bulk generation.
