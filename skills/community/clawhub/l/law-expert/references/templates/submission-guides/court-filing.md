# 法院立案提交指引（民事一审）

## 适用场景
公民、法人或其他组织因民事权利义务争议向有管辖权的人民法院提起民事诉讼一审，包括但不限于：合同纠纷、侵权纠纷、婚姻家庭纠纷、继承纠纷、物权纠纷、劳动争议（仲裁后起诉）、知识产权纠纷等。

## 管辖机构
| 项目 | 内容 |
|------|------|
| 机构名称 | XX省/市/区（县）人民法院 |
| 管辖层级 | 一般案件由 **基层人民法院** 管辖；重大涉外、在本辖区有重大影响的由中级人民法院管辖 |
| 地域管辖 | 一般原则为 **被告住所地**；合同纠纷可由合同履行地；侵权纠纷可由侵权行为地或结果发生地 |

**常见管辖规则速查**：
| 案件类型 | 管辖法院 |
|----------|----------|
| 一般民事纠纷 | 被告住所地法院 |
| 合同纠纷 | 被告住所地 或 合同履行地法院 |
| 侵权纠纷 | 侵权行为地 或 被告住所地法院 |
| 不动产纠纷 | 不动产所在地法院（专属管辖） |
| 离婚纠纷 | 被告住所地；被告不在国内/下落不明的→原告住所地 |
| 劳动争议 | 仲裁委所在地 或 用人单位所在地法院 |

## 所需材料

### 原告为自然人

| 序号 | 材料 | 份数 | 要求 | 模板 |
|------|------|------|------|------|
| 1 | 民事起诉状 | 被告人数+1份 | 载明：<br>- 原告姓名/性别/出生日期/民族/住址/联系电话<br>- 被告姓名/住址/联系电话（单位则名称+住所+法定代表人）<br>- 诉讼请求（明确具体）<br>- 事实与理由<br>- 落款签名/日期 | 有 |
| 2 | 原告身份证明 | 1份 | 身份证正反面复印件，现场核对原件 | 无 |
| 3 | 被告身份信息 | 1份 | 被告为个人：身份证号或住址信息<br>被告为单位：工商登记信息（全国企业信用信息公示系统打印） | 无 |
| 4 | 证据材料 | 被告人数+1份 | 所有支持诉讼请求的证据复印件，编页码，附证据目录 | 有（证据清单模板） |
| 5 | 授权委托书 | 1份（如有代理人） | 律师代理：授权委托书+律师证复印件+律所函<br>公民代理：授权委托书+代理人身份证复印件+推荐证明/近亲属证明 | 有 |
| 6 | 送达地址确认书 | 1份 | 填写收件地址和联系方式（部分法院要求） | 法院提供 |

### 原告为单位

| 序号 | 材料 | 份数 | 要求 | 模板 |
|------|------|------|------|------|
| 1 | 民事起诉状（盖章） | 被告人数+1份 | 加盖单位公章 | 有 |
| 2 | 营业执照副本复印件 | 1份 | 加盖公章 | 无 |
| 3 | 法定代表人身份证明书 | 1份 | 载明法定代表人姓名/职务，加盖公章 | 有 |
| 4 | 法定代表人身份证复印件 | 1份 | — | 无 |
| 5 | 授权委托书 | 1份（非法定代表人亲自办理时） | 加盖公章 | 有 |

## 提交流程

### 现场立案
1. **确定管辖法院**：确认有管辖权的法院（被告住所地/合同履行地等）
2. **准备起诉材料**：按上述清单准备，复印件份数为被告人数+法院留存1份
3. **前往立案窗口**：携带身份证原件+全部材料至法院诉讼服务中心
4. **递交材料**：立案人员审查材料是否齐全、是否符合受理条件
5. **获取回执**：
   - 材料齐全且符合条件 → 当场登记立案，出具《案件受理通知书》
   - 材料不齐 → 收到《补正材料通知书》，7日内补齐
   - 不符合条件 → 收到《不予受理裁定书》（可上诉）
6. **缴纳诉讼费**：收到《交纳诉讼费用通知书》后7日内缴纳
7. **领取开庭传票**：等待法院排期通知

### 网上立案（推荐）
1. 登录"人民法院在线服务"微信小程序或各省法院诉讼服务网
2. 实名注册认证
3. 选择"我要立案"→"审判立案"
4. 选择管辖法院
5. 上传起诉状（PDF格式）、身份证明、证据材料扫描件
6. 提交审核
7. 审核通过后按要求邮寄或现场递交纸质材料

## 费用

| 费用类型 | 说明 |
|----------|------|
| 案件受理费 | 按诉讼标的额比例计算（详见《诉讼费用交纳办法》），一般由原告预交 |
| 申请费 | 申请财产保全/证据保全/执行等需另交 |
| 减免情形 | 追索赡养费/扶养费/抚育费/抚恤金/劳动报酬等可申请缓交/减交/免交 |
| 简易程序 | 减半收取受理费 |

**诉讼费速算参考**（财产案件）：
- 不超过1万元：50元
- 1万-10万元：标的额×2.5% - 200元
- 10万-20万元：标的额×2% + 300元
- 20万-50万元：标的额×1.5% + 1300元

## 时间预估

| 阶段 | 预计时间 | 法律依据 |
|------|----------|----------|
| 立案审查 | 7日内（符合条件当场立案） | 《民事诉讼法》第126条 |
| 简易程序审限 | 立案之日起 **3个月** | 需双方同意适用 |
| 普通程序审限 | 立案之日起 **6个月** | 可延长 |
| 整体周期 | 一般3个月-1年 | 视案情复杂程度及是否上诉 |

## 注意事项

1. **诉讼时效**：一般民事权利保护时效为 **3年**（《民法典》第188条），从知道或应当知道权利受损及义务人之日起算。超过时效可能丧失胜诉权

2. **管辖务必准确**：提交到无管辖权法院会被裁定驳回/移送，浪费时间。合同纠纷、侵权纠纷有多个管辖选择时，选择对自己最便利的

3. **被告信息要准确**：被告姓名/名称、地址必须准确，否则无法送达，需公告送达（公告期30日，极大拖延时间）

4. **诉讼请求要明确具体**：
   - 金钱给付：写明具体金额和计算依据
   - 行为请求：写明具体行为内容和履行期限
   - 确认之诉：写明确认的法律关系
   - 多项请求应逐项列出并编号

5. **证据编目**：按"证据目录"格式整理，每组证据标明名称、来源、页数、证明目的

6. **诉讼费缴纳**：收到交费通知后务必在7日内缴纳，否则按撤诉处理

7. **财产保全**：如果担心被告转移财产，可在起诉同时申请财产保全（需提供担保）

8. **小额诉讼程序**：标的额在各省上年度就业人员年平均工资50%以下的简单案件，一审终审
