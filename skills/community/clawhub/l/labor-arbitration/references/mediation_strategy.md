# 劳动仲裁调解策略参考手册

## 一、调解概述

### 1.1 调解在劳动仲裁中的地位
- 调解贯穿劳动仲裁全过程：庭前调解、庭中调解、庭后调解
- 多数地区仲裁委在开庭前会组织调解
- 调解书与裁决书具有同等法律效力
- 调解达成后不可就同一事项再次申请仲裁或起诉

### 1.2 调解 vs 裁决的利弊

| 维度 | 调解 | 裁决 |
|------|------|------|
| 时间 | 快（1个月内） | 慢（45-60天，复杂可延长） |
| 金额 | 通常需让步 | 全额支持（如证据充分） |
| 执行 | 可约定分期/一次性 | 需待裁决生效后执行 |
| 关系 | 相对缓和 | 对抗性强 |
| 上诉 | 一调终局，不可上诉 | 部分可起诉至法院 |
| 风险 | 双方各让一步 | 视证据而定，存在败诉风险 |

## 二、调解前的准备

### 2.1 确定底线金额

**计算方法：**
1. 计算仲裁请求总金额（法定应得金额）
2. 确定可接受的最低金额（底线）
3. 确定期望金额（理想结果）

**底线金额参考建议：**
- 证据充分的案件：底线 = 法定金额的 70%-80%
- 证据一般的案件：底线 = 法定金额的 50%-60%
- 证据不足的案件：底线 = 法定金额的 30%-40%

**影响底线的因素：**
- 时间成本（裁决可能拖2-3个月，二审更长）
- 执行风险（公司可能注销、转移资产）
- 举证风险（部分证据可能在质证中被排除）
- 精力成本（多次开庭、准备材料）

### 2.2 分析对方立场

**预判对方诉求：**
1. 用人单位通常希望：降低赔偿金额、分期支付、避免裁决记录
2. 用人单位可能抗辩点：
   - 否认劳动关系（无合同情况下）
   - 主张合法解除（规章制度依据）
   - 主张已足额支付工资
   - 主张超过仲裁时效
   - 降低工资基数（以实发工资主张）

3. 评估对方底线：
   - 大型企业：通常有预算和法务审核，底线较高
   - 中小企业：资金紧张，可能只接受较低金额
   - 面临倒闭的企业：需关注其支付能力

### 2.3 准备谈判筹码

**有利筹码：**
- 证据充分的请求 → 高开价
- 用人单位明显违法的行为 → 增加谈判筹码
- 可能影响企业声誉的情节 → 如用人单位害怕投诉
- 二倍工资、赔偿金等惩罚性赔偿 → 作为让步空间

**不利因素（需提前准备应对）：**
- 证据不足的请求 → 降低预期
- 时效争议 → 准备时效中断证据
- 程序瑕疵 → 准备解释说明

## 三、调解谈判策略

### 3.1 开局策略

**策略一：高开低走**
- 初始报价高于期望金额15%-20%
- 为后续让步预留空间
- 示例：期望50000元，初始报价58000元

**策略二：锚定法定金额**
- 坚持法定应得金额为起点
- 强调法律依据充分
- 让步时从"免除部分争议请求"开始

**策略三：分项谈判**
- 将多个仲裁请求拆分谈判
- 先谈争议小、金额低的请求，建立共识
- 争议大的请求最后谈

### 3.2 中盘策略

**让步技巧：**
1. **等额让步递减法：**
   - 第一轮让步5000元
   - 第二轮让步3000元
   - 第三轮让步1000元
   - 传递"已接近底线"信号

2. **条件交换法：**
   - "金额可以降，但要求一次性支付"
   - "金额可以降，但要求当庭支付"
   - "可以放弃加班费请求，但经济补偿金不能少"

3. **期限施压法：**
   - "如果今天能签调解协议，金额可以让步到X"
   - "如果不能当庭支付，将不再接受调解"

4. **外力借势法：**
   - 引导仲裁员介入调解
   - 提示对方败诉风险
   - 提及后续可能的行政投诉（社保、税务）

### 3.3 收尾策略

**接近成交时：**
1. 确认所有条款无歧义
2. 明确支付方式和期限
3. 确认违约条款（如逾期支付的后果）
4. 确认调解书效力（一调终局）

**调解协议关键条款清单：**
- [ ] 支付金额（大小写）
- [ ] 支付方式（银行转账）
- [ ] 支付期限（具体日期）
- [ ] 收款账户信息
- [ ] 违约责任（逾期支付利息/恢复全额请求）
- [ ] 放弃其他请求的声明
- [ ] 保密条款（如双方同意）
- [ ] 调解书送达方式

## 四、常见调解场景应对

### 场景一：对方要求大幅降价

**应对话术：**
- "我理解贵方的困难，但法定金额是有明确依据的。我愿意在X项请求上做出让步，但核心请求不能降太多。"
- "如果金额降到X，需要贵方当庭支付/3日内支付，我可以接受分期方案转为一次性。"

### 场景二：对方要求分期支付

**应对要点：**
- 确认分期期数和每期金额
- 要求提供担保或增加违约条款
- "分期可以，但如果任何一期逾期，全部金额立即到期，并按X%支付违约金"
- 争取首期支付比例不低于40%

### 场景三：对方否认部分事实

**应对要点：**
- 出示关键证据原件
- "证据X已经清楚证明这一事实，如果调解不成，仲裁庭也会据此裁决"
- 区分"可以退让的事实"和"必须坚持的事实"

### 场景四：对方以公司经营困难为由压价

**应对要点：**
- 了解公司实际经营状况（工商信息、社保参保人数）
- "经营困难不是拖欠劳动报酬的合法理由"
- "如果公司确实困难，可以接受适当分期，但金额不能低于法定标准的X%"
- 警惕公司可能注销/破产的风险，争取尽快支付

### 场景五：仲裁员施压调解

**应对要点：**
- 尊重仲裁员但不盲从
- "感谢仲裁员的调解工作，我愿意配合调解，但底线是X元"
- 如仲裁员建议金额低于底线，可要求时间考虑
- 不要在压力下仓促签字

## 五、调解协议注意事项

### 5.1 必须包含的条款
1. 双方当事人信息
2. 争议事项概述
3. 协商结果（金额、支付方式、期限）
4. 权利义务了结声明
5. 违约责任
6. 签署日期和签名

### 5.2 常见陷阱
- ❌ "一次性了结所有争议" → 需明确范围，避免放弃未主张权利
- ❌ 模糊支付期限（"尽快支付"）→ 必须写明具体日期
- ❌ 无违约条款 → 对方拖延无后果
- ❌ 口头承诺未写入协议 → 无法律效力
- ❌ 调解金额含税含税不明确 → 需写明税前/税后

### 5.3 调解书不履行的救济
- 调解书生效后对方不履行的，可向法院申请强制执行
- 申请执行期限：调解书规定的履行期限届满后2年内
- 执行申请向对方住所地或财产所在地法院提出

## 六、调解 vs 裁决决策矩阵

| 情形 | 建议策略 | 理由 |
|------|---------|------|
| 证据充分，金额明确，对方有支付能力 | 可坚持裁决 | 证据充分裁决胜诉概率高 |
| 证据充分，但对方可能转移资产 | 接受调解，争取快速支付 | 调解可快速拿到款项 |
| 证据一般，金额有争议 | 积极调解 | 降低败诉风险 |
| 证据不足，胜诉概率低 | 尽量调解 | 争取部分金额优于全败 |
| 对方恶意拖延，无意调解 | 不再调解，要求裁决 | 调解无效，浪费时间 |
| 金额较小（1万以下） | 优先调解 | 时间成本高于差额 |
| 金额较大（5万以上） | 谨慎调解 | 让步金额绝对值大 |
| 涉及恢复劳动关系 | 通常裁决 | 调解难以执行 |
