## Description: <br>
Helps Chinese-speaking OpenClaw users install and configure ClawHub, set a mirror, discover skills, and manage skill installation, updates, enabling, and disabling. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[binbin](https://clawhub.ai/user/binbin) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and OpenClaw users use this skill as a Chinese-language ClawHub setup and skill-management guide. It provides commands and workflow guidance for installing ClawHub, configuring mirrors, finding skills, and managing installed skills. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Suggested global package installation, bulk skill installation, or update-all commands can change the local agent environment. <br>
Mitigation: Inspect commands and skills before running them, and avoid bulk installs or update-all commands on stable environments unless behavior changes are acceptable. <br>
Risk: Mirror configuration changes where ClawHub retrieves packages or skill assets. <br>
Mitigation: Confirm that the mirror is trusted and appropriate for the environment before setting it in global configuration. <br>
