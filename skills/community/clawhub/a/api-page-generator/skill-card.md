## Description: <br>
Helps create, optimize, or audit API introduction and overview pages for developer-facing API offerings. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, marketers, and product teams use this skill to draft or assess an API overview page that explains the API's value, target use cases, documentation path, calls to action, and developer-search metadata. <br>

### Deployment Geography for Use: <br>
Global <br>
