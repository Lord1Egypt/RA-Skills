## Description: <br>
Finds guitar tab or sheet sources from a song title or link, ranks the best matches, and produces a compact practice note in markdown, JSON, or optional Obsidian format. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Robinnnnn](https://clawhub.ai/user/Robinnnnn) <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and agents use this skill to locate likely guitar tabs, chords, sheet sources, or fingerstyle arrangements for a song and turn the results into a concise practice packet. It can return portable markdown or JSON, and can format Obsidian notes only when explicitly requested. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Optional Obsidian or local-file mode can create or update files in a user-selected location. <br>
Mitigation: Use local-file mode only after explicit user request and with a specific folder the user is comfortable modifying. <br>
Risk: Tab search results may include uncertain, mismatched, or paywalled sources. <br>
Mitigation: Return ranked links with confidence notes and clearly call out uncertainty or likely paywalls. <br>
