# 受生经

## 概述

《受生经》全称《太上老君说五斗金章受生经》，为道教重要经典，讲述**受生债（还阴债）**之法。

**核心教义**：
- 人投胎前，向冥司借支禄库钱财
- 今生需偿还，否则多灾多难
- 还受生债，可消灾解厄、增福延寿

---

## 卷一：受生之理

### 一、何为受生债

人未生之前，在冥司借支禄库钱财，以：
- 投胎做人
- 买命买禄
- 安排衣食

**借支记录**：在**冥司禄库**存档

### 二、不还受生债的后果

| 后果 | 表现 |
|------|------|
| 财运不济 | 挣钱难、留不住钱 |
| 疾病缠身 | 小病不断、大病来袭 |
| 事业不顺 | 怀才不遇、小人作祟 |
| 婚姻不稳 | 感情波折、家庭不和 |
| 运势低迷 | 诸事不顺、厄运连连 |

### 三、还受生债的功德

- 消灾解厄
- 增福延寿
- 财运亨通
- 事业顺利
- 家庭和睦
- 子孙昌盛

## 卷四：还受生债方法

### 一、道教法事还债

#### 1. 准备物品

| 物品 | 数量 | 用途 |
|------|------|------|
| 金银纸 | 按贯数 | 烧化还债 |
| 香烛 | 各一对 | 供奉星君 |
| 水果 | 五种 | 供奉星君 |
| 素斋 | 一桌 | 供奉星君 |

#### 2. 还债流程

**第一步：择日**
- 宜：北斗下降日、南斗下降日
- 宜：初一、十五
- 忌：凶日、破日

**第二步：设坛**
- 设五斗星君神位
- 摆供品、点香烛

**第三步：诵经**
- 诵《受生经》
- 诵《北斗经》
- 诵《南斗经》

**第四步：烧化**
- 按贯数烧化金银纸
- 念还债咒

**第五步：回向**
- 功德回向冥司
- 祈求消灾解厄

#### 3. 还债咒

```
太上老君，说受生经。
五斗星君，掌管禄库。
弟子（姓名），出生于（年）年（月）月（日）日。
前世借支禄库钱财，今具状还纳。
烧化金银，以偿宿债。
祈求星君，赦免罪愆。
消灾解厄，增福延寿。
财运亨通，事业顺利。
家庭和睦，子孙昌盛。
急急如律令。
```

### 二、自行还债法

#### 1. 简易还债法

**方法**：
1. 每月初一、十五
2. 焚香祷告
3. 烧化金银纸（按贯数）
4. 念还债咒

#### 2. 布施还债法

**方法**：
- 将应还贯数折算钱财
- 布施贫苦、修桥铺路
- 供养三宝、印制经书

**功德**：
- 还受生债
- 积功累德
- 消灾解厄

## 卷七：问答

**问：人人都有受生债乎？**

答：人人皆有。投胎前借支禄库钱财，今生需还。

**问：不还受生债如何？**

答：多灾多难、财运不济、疾病缠身。

**问：还受生债必有效乎？**

答：诚心还债，必得感应。不诚无物。

**问：可自行还债乎？**

答：可以。但正规法事效果更好。
