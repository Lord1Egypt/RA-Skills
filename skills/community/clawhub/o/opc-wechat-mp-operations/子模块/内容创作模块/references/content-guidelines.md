## 文章结构标准

### 标准结构

**导语**（3-5句）
- 开场白：使用"说真的"、"你知道吗"等拟人化表达
- 引出问题：描述一个普遍现象或痛点
- 承诺价值：告诉读者读完能获得什么
- 过渡下文：自然引出正文

**正文**（3-5个部分）
- 每个部分一个小标题
- 每部分3-5段，每段2-4句
- 逻辑递进：问题-原因-解决方案
- 使用加粗强调重点

**结尾**（3-5句）
- 总结核心观点
- 行动号召
- 温暖鼓励或价值升华

### 导语示例

**主题：睡眠质量**

```
说真的，你是不是每天都感觉：明明睡够了8小时，起床还是累得要死？

我太懂这种感觉了！那种"睡了个寂寞"的无力感，简直让人崩溃。

今天这篇文章，我花了3天时间整理，只为你解决这个困扰。
```

### 小标题示例

**主题：睡眠质量**

```
一、为什么睡眠质量更重要？

二、这些食物让你睡不着

三、睡前运动有讲究
```

### 结尾示例

**主题：睡眠质量**

```

---

## 💙 写在最后

好的睡眠，是高效生活的基础...

如果这篇文章对你有帮助，记得点个"在看"，分享给身边睡不好的朋友～
```

## 语言风格指导

### 基本原则

**1. 引子开场**
- 文章开头需要一个故事/事件引子
- 制造代入感和共鸣
- 自然引出文章主题

**引子类型**：
| 类型 | 适用场景 | 示例开头 |
|------|---------|---------|
| 网友私信 | 知识科普、问题解答 | "最近收到很多私信，都在问..." |
| 自身经历 | 经验分享、感悟类 | "前几天我XXX，发现..." |
| 身边故事 | 实用技巧、案例类 | "邻居王阿姨最近总说..." |
| 引用对话 | 观点表达、讨论类 | "'你怎么看起来这么累？'朋友问我..." |
| 热点事件 | 时事评论、趋势分析 | "这两天朋友圈都在讨论..." |

**2. 拟人化表达**
- 使用"我"作为主语，建立个人化视角
- 适当加入"咱们"、"你知道吗"等口语化表达
- 模拟真实对话场景，增强亲和力

**拟人化技巧**：
- 用"我观察到"、"我发现"替代"数据显示"、"研究表明"
- 用"你有没有"替代"很多人会"
- 用"真的很XX"替代"非常XX"

**3. 情绪共鸣**
- 描述读者的真实感受和痛点
- 用"我太懂了"、"说真的"等表达共情
- 分享个人经历，增加真实感

**共鸣句式**：
- "你是不是也..."
- "我太懂这种感觉了"
- "说真的，这件事困扰我好久了"

**4. 简洁明了**
- 每句不超过30字
- 一段不超过5句
- 避免复杂从句和专业术语

**简洁技巧**：
- 将长句拆成短句
- 删除冗余修饰词
- 用具体例子替代抽象概念

### 语言禁忌

**避免的表达**：
- ❌ "本文将介绍..."（太正式）
- ❌ "根据研究显示..."（太学术）
- ❌ "综上所述..."（像论文）
- ❌ "通过以上分析..."（太理性）
- ❌ "希望大家..."（太官方）
- ❌ "感谢阅读..."（太套路）

**推荐的替代**：
- ✅ "今天来聊聊..."
- ✅ "我发现一个有趣的现象..."
- ✅ "最后想说..."
- ✅ "写到这里..."
- ✅ "如果你也..."
- ✅ "如果这篇文章对你有帮助..."

### 语言风格对比

| 场景 | 不佳示例 | 优化示例 |
|------|---------|---------|
| 开场白 | 本文将介绍睡眠的重要性 | 说真的，睡眠这件事比你想的更重要 |
| 引用数据 | 研究表明60%的人失眠 | 我查了下数据，超过一半的人都睡不好 |
| 提出建议 | 建议用户保持规律作息 | 别小看按时睡觉这件事，规律作息很重要 |
| 结尾总结 | 综上所述，睡眠对健康很重要 | 写到这里，希望你能重视睡眠这件事 |

### 专业术语处理

**原则**：能用通俗语言就不用专业术语

**转换技巧**：
| 专业术语 | 通俗表达 |
|---------|---------|
| 生物节律 | 身体的"闹钟" |
| 快速眼动期 | 大脑"做梦"的阶段 |
| 褪黑素 | 睡眠激素 |
| 昼夜节律 | 睡眠-觉醒周期 |
| 入睡潜伏期 | 入睡需要的时间 |

**无法避免术语时**：
- 第一次出现时用引号标注
- 紧接着用通俗语言解释
- 后续直接使用简化说法

**示例**：
```
研究发现，快速眼动期（REM，也就是"做梦"的阶段）对记忆巩固很重要。

所以做梦不是坏事，反而说明大脑在"工作"。
```

### 情绪价值设计

**提供3种情绪**：

**1. 理解与共情**
```
我太懂这种感觉了！那种想睡却睡不着的煎熬，简直让人崩溃。

你不是一个人，很多人都和你一样。
```

**2. 安慰与鼓励**
```
别焦虑，慢慢来。

给自己一点时间，身体会慢慢恢复的。

相信自己，你能做到的。
```

**3. 希望与温暖**
```

---

## 💙 写在最后

好的睡眠，是高效生活的基础。

如果这篇文章对你有帮助，记得点个"在看"，分享给身边睡不好的朋友～

明天见，晚安~
```

### 小标题设计原则

**类型选择**：
- **数字式**：一、二、三 / 1. 2. 3.
- **疑问式**：为什么...？/ 如何...？
- **陈述式**：睡眠质量更重要 / 影响睡眠的因素

**设计要点**：
- 长度控制：3-8字最佳
- 逻辑连贯：小标题之间有递进或并列关系
- 吸引力：使用疑问句或数字增加阅读欲望

**示例对比**：

| 不佳示例 | 优化示例 |
|---------|---------|
| 第一部分 | 一、为什么睡眠质量更重要？ |
| 关于饮食 | 二、这些食物让你睡不着 |
| 运动建议 | 三、睡前运动有讲究 |

## 人设写作规范

### 男编辑版人设特点

**核心特质**：
- **理性分析**：数据驱动，逻辑清晰
- **专业权威**：行业洞察，观点独到
- **简洁有力**：直击要害，不绕弯子
- **深度解析**：透过现象看本质

**语言风格**：
- 使用"我认为"、"据我观察"、"数据显示"
- 引用行业报告、专业数据
- 句式偏短，节奏明快
- 避免情绪化表达，保持客观

**开头模板**：
```
最近有个数据引起我注意...

根据XX机构的最新报告，我发现...

从业X年，我发现一个现象...

今天跟大家聊聊我的观察...
```

**结尾模板**：
```
以上是我的一些观察，希望对你有帮助。

数据不会说谎，关键是如何运用。

有问题欢迎评论区讨论，我们一起分析。
```

**示例片段**：
```
据国家统计局数据，2024年年轻人失眠率已达68%。

我分析了一下，主要有三个原因。

第一，工作压力大。数据显示，互联网行业从业者平均睡眠时长不足6.5小时。

第二，手机依赖严重。我的团队做过调研，睡前刷手机超过2小时的人，入睡时间延迟40%。

第三，作息不规律。这个最致命，生物钟一旦乱了，恢复至少需要2周。
```

---

### 女编辑版人设特点

**核心特质**：
- **情感共鸣**：温暖治愈，像朋友聊天
- **生动比喻**：用生活场景类比，通俗易懂
- **细节丰富**：从个人经历切入，真实感强
- **情绪价值**：让读者感到被理解和被关心

**语言风格**：
- 使用"我最近"、"我朋友"、"你知道吗"
- 大量使用表情符号和语气词（~、呢、呀）
- 句式有起伏，语气温柔
- 分享个人经历和感受

**开头模板**：
```
最近闺蜜跟我说...

昨天我刷到一条消息，瞬间破防了...

说真的，这件事困扰我好久...

今天想跟大家掏心窝子聊聊...
```

**结尾模板**：
```
写到这里，心里暖暖的。

如果你也有类似的经历，欢迎在评论区告诉我。

咱们一起变好呀~

对了，别忘了点个"在看"，让更多姐妹看到~
```

**示例片段**：
```
说真的，最近我闺蜜一直在跟我吐槽睡不好。

昨天凌晨两点，她还在给我发消息："你说我是不是废了？躺床上翻来覆去，脑子里像放电影一样。"

我太懂这种感觉了！那种想睡却睡不着，越急越清醒的煎熬，简直让人崩溃。

后来我帮她调整了一些习惯，效果立竿见影。今天把我的"睡眠秘籍"分享给你们。

首先，睡前必须戒掉手机！真的，这东西是睡眠杀手。我之前也是，刷抖音一刷就两小时，结果第二天顶着熊猫眼上班。现在规定自己10点后手机关机，放在客厅，一周下来睡眠质量明显提升。

然后，试试这个方法：睡前泡脚15分钟。我奶奶教的，超管用！温水+生姜，泡到微微出汗，整个身体都放松了，睡觉特别香。
```

### 人设选择标准

**选择男编辑版**的情况：
- 主题需要专业数据支撑
- 目标读者偏理性（如职场人士、管理者）
- 内容偏分析、趋势、行业报告
- 需要权威感和专业度

**选择女编辑版**的情况：
- 主题偏向生活方式、情感共鸣
- 目标读者偏感性（如年轻女性、宝妈）
- 内容偏经验分享、情感疏导
- 需要温暖感和陪伴感

## 爆款文章要素

### 文章钩子设计（1-5秒抓住读者）

**核心目标**：让读者在1-5秒内决定继续阅读，提高点击率和完读率

**钩子设计原则**：
- **即时价值**：开头必须传达"这篇文章对我有用"
- **情绪触发**：激发好奇、焦虑、期待等情绪
- **身份认同**：让读者觉得"这就是在说我"
- **悬念设置**：制造疑问，引出下文

**钩子类型与模板**：

#### 1. 痛点共鸣型钩子

**适用场景**：普遍存在的困扰、未被解决的问题

**男编辑版模板**：
```
【数据引入】
据XXX报告显示，XX%的XX人群正面临XX问题。

【痛点描述】
具体表现为：症状1、症状2、症状3。

【价值承诺】
今天基于数据分析，给出3个解决方案。
```

**女编辑版模板**：
```
【故事开场】
昨天闺蜜跟我吐槽："我每天XX，结果还是XX..."

【情绪共鸣】
我太懂这种感觉了！那种XX的煎熬，简直让人崩溃。

【价值承诺】
今天把我的"XX秘籍"分享给你，亲测有效~
```

**示例**：
```
男编辑版：
据国家统计局数据，2024年年轻人失眠率已达68%，较去年增长15%。
很多人睡够了8小时，起床还是跟没睡一样。
今天基于数据分析，聊聊睡眠的核心逻辑。

女编辑版：
昨天凌晨两点，闺蜜给我发消息："躺床上翻来覆去，脑子里像放电影一样。"
我太懂这种感觉了！那种想睡却睡不着的煎熬，简直让人崩溃。
今天把我的"睡眠秘籍"分享给你，亲测有效~
```

#### 2. 反常识型钩子

**适用场景**：颠覆认知的观点、反直觉的发现

**模板**：
```
【反常识陈述】
你知道吗？XX竟然是错的！

【证据支撑】
据XX研究显示，XX的人反而更XX。

【悬念设置】
今天来聊聊这个颠覆认知的话题。
```

**示例**：
```
你知道吗？睡够8小时，反而可能让你更累。

据睡眠专家研究，深度睡眠比例高的人，即使睡6小时也比睡8小时但浅睡的人精神好。

今天来聊聊这个颠覆认知的话题。
```

#### 3. 数据冲击型钩子

**适用场景**：需要权威性、可信度的话题

**模板**：
```
【冲击性数据】
一个数字让你震惊：XX%的人正在XX！

【数据解读】
这意味着什么？意味着XX、XX、XX。

【提问引导】
你，是不是其中之一？
```

**示例**：
```
一个数字让你震惊：超过68%的年轻人都有失眠问题！

这意味着什么？意味着每3个人中就有2个人睡不好。

你，是不是其中之一？
```

#### 4. 故事代入型钩子

**适用场景**：需要情感共鸣、生动表达的话题

**模板**：
```
【故事开场】
昨天我遇到一件事，让我重新思考XX。

【冲突描述】
这个人明明XX，结果却XX。

【反思提问】
为什么？今天来聊聊背后的逻辑。
```

**示例**：
```
昨天我遇到一个程序员，连续加班半个月，每天只睡4小时，结果昨天在公司突然晕倒了。

明明他那么努力，为什么身体却垮了？

今天来聊聊这个背后的逻辑。
```

#### 5. 热点借势型钩子

**适用场景**：结合当前热点、热搜话题

**模板**：
```
【热点引入】
最近XX（热点事件/热搜）刷屏了。

【热点关联】
很多人都在讨论XX，但很少有人注意到XX。

【价值点明】
今天结合这个热点，聊聊XX这个被忽视的问题。
```

**示例**：
```

很多人都在讨论游戏难度，但很少有人注意到：熬夜打游戏不仅影响第二天状态，长期还会伤身。

今天结合这个热点，聊聊如何平衡爱好和健康。
```

### 钩子设计技巧

**技巧1：数字化表达**
- 用具体数字替代模糊表述
- 示例："68%的人"替代"很多人"，"3个方法"替代"一些方法"

**技巧2：场景化描述**
- 描述具体场景，让读者产生画面感
- 示例："凌晨两点还在刷手机"替代"熬夜"

**技巧3：身份标签化**
- 使用目标读者熟悉的身份标签
- 示例："90后"、"程序员"、"宝妈"

**技巧4：疑问式结尾**
- 钩子结尾设置疑问，引导继续阅读
- 示例："为什么？"、"怎么办？"、"你知道吗？"

**技巧5：对比反差**
- 制造对比或反差，增强冲击力
- 示例："睡得越久，反而越累"、"越努力，越焦虑"

### 钩子设计避坑指南

**❌ 错误示例**：
- "今天我们来聊聊XX的话题"（太平淡，无吸引力）
- "XX很重要，大家要注意"（说教语气，无情绪价值）
- "最近我发现了一个现象"（无具体内容，无信息量）
- "XX是什么？怎么解决？"（无吸引力，无情绪触发）

**✅ 正确示例**：
- "睡够8小时还是累？问题可能在这里"（痛点+疑问）
- "68%的人都在做这件事，但80%都做错了"（数据+反差）
- "昨天一个朋友跟我吐槽，说他每天XX，结果XX"（故事+共鸣）
- "你知道吗？XX竟然是错的！"（反常识+悬念）

## 算法优化技巧

### NLP关键词布局

**核心原则**：
- 关键词在标题、开头、正文、结尾的合理分布
- 避免关键词堆砌，保持自然流畅
- 关键词密度建议：2-5%

**黄金位置布局**：

**1. 标题（权重最高）**
- 必须包含核心关键词
- 优先放在标题前半部分
- 示例：`✨ 睡眠质量比时长更重要：3个方法改善失眠`

**2. 导语（前200字）**
- 2-3次自然提及核心关键词
- 使用长尾关键词
- 示例：`你是不是也有这种感觉？明明睡够了8小时，起床还是跟没睡一样？这可能是睡眠质量问题。`

**3. 小标题**
- 每个小标题包含相关关键词
- 避免重复，使用近义词
- 示例：`一、为什么睡眠质量更重要？`、`二、改善睡眠的3个方法`

**4. 正文段落**
- 每100-150字自然提及1次关键词
- 使用同义词增强多样性
- 保持自然流畅，避免堆砌

**5. 结尾段**
- 总结核心关键词
- 引导互动
- 示例：`睡眠质量比时长更重要，从今晚开始，改善你的睡眠吧。`

### 关键词分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 核心关键词 | 文章的核心主题 | 睡眠质量、失眠、健康管理 |
| 次级关键词 | 核心主题的细分 | 深度睡眠、入睡困难、睡眠障碍 |
| 长尾关键词 | 具体问题或解决方案 | 年轻人失眠原因、改善睡眠质量的方法 |
| 情感关键词 | 情绪相关的词汇 | 焦虑、疲惫、困扰、改善 |

### 内容标签化

**为什么需要标签化**：
- NLP更容易识别结构化信息
- 提升算法理解度
- 优化推荐匹配

**标签化方法**：

**1. 数字化表达**
- 使用数字列表：`1. 2. 3.`、`第一、第二、第三`
- 数字化标题：`3个方法`、`5个迹象`、`7个习惯`

**2. 引用标注**
- 数据引用：`据国家统计局数据显示...`
- 专家引用：`睡眠专家指出...`
- 研究引用：`《中国睡眠健康白皮书》指出...`

**3. 强调标注**
- 使用加粗：`**核心观点**`
- 使用引用块：`> 重要提示`
- 使用颜色框：`<div class="highlight">重点内容</div>`

**4. 问答式表达**
- 使用疑问句：`为什么...？`、`如何...？`
- 自问自答：`原因是什么？主要有三个。`
- 增强可读性和算法理解度

### 互动数据优化

**1. 提升点击率**
- 优化标题吸引力
- 使用高质量封面图
- 明确价值承诺

**2. 提升完读率**
- 优化文章结构

## 排版要求

### 基本原则

**只使用三种排版元素**：
1. 小标题（## 或 ###）
2. 换行（段落之间空一行）
3. 加粗（**文字**）

**禁止使用**：
- 复杂表格（如需表格，简化为列表）
- 多级标题（仅使用二级标题）
- 特殊符号（如 ※ ★ ● ◆）
- 颜色字体（纯文本无法呈现）

### 换行规则

**段落之间**：
- 一个段落结束后，空一行
- 小标题前后，各空一行

**句内换行**：
- 每句话独立成行
- 句子之间不空行

**示例**：
```
【导语】
你是不是每天睡够8小时，却还是觉得累？

问题可能不在时长，而在睡眠质量。

今天我们来聊聊那些被忽视的睡眠细节。

一、睡眠质量比时长更重要
（此处空一行）
很多人觉得，只要睡够时间就行。

但研究发现，**睡眠质量比时长更重要**。

睡得久不代表睡得好，关键在于深度睡眠的比例。
```

### 加粗使用规范

**加粗场景**：
1. 核心观点
2. 重要数据
3. 关键结论
4. 需要强调的内容

**使用原则**：
- 一段话最多加粗1-2处
- 加粗内容不超过整段文字的30%
- 不连续加粗整句话

**示例对比**：

| 不佳示例 | 优化示例 |
|---------|---------|
| **睡眠质量比时长更重要**，研究表明深度睡眠比例高的人，即使睡6小时也比睡8小时但浅睡的人精神好。 | 睡眠质量比时长更重要。研究表明，**深度睡眠比例高的人**，即使睡6小时也比睡8小时但浅睡的人精神好。 |
| **很多人都有睡眠问题，据统计超过60%的成年人存在不同程度的睡眠障碍，这个数字还在逐年上升。** | 很多人都有睡眠问题。据统计，**超过60%的成年人**存在不同程度的睡眠障碍，这个数字还在逐年上升。 |

### 列表使用规范

**可以使用简单列表**：
- 无序列表：- 或 *
- 有序列表：1. 2. 3.

**列表要求**：
- 列表项不超过10条
- 每条不超过2行
- 列表前后各空一行
