# Changelog

## v1.0.0

Initial release of the GitHub Projects v2 skill.

### Features

- List backlog items with filtering (status, assignee, open/closed)
- Set project fields (status, iteration, priority, story points, dates)
- Clear field values
- Add comments to issues (inline or from file)
- Create issues and add them to a project
- Create sub-issues via `gh-sub-issue` extension with GraphQL fallback
- Auth prerequisite guidance (`project` scope)
