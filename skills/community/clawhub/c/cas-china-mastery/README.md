# CAS China Mastery — 中国企业会计准则全面实务体系

> 5本财政部权威著作2021-2025版，覆盖1项基本准则+40项具体准则

## 概述

融合5部中国企业会计准则权威著作（共2,116页、2,302,676字符）：
1. **《企业会计准则案例讲解》**（2021年版）— 669页，40项准则全覆盖案例
2. **《企业会计准则实施问答汇编》**（2025年4月更新版）— 41页，财政部官方问答
3. **《企业会计准则详解与实务》**（2025年版）— 802页，条文解读+实务应用+案例讲解
4. **《企业会计准则应用案例汇编》**（2025年4月更新版）— 93页，收入/股份支付/金融工具/PPP实务案例
5. **《企业会计准则应用指南》**（2021年版）— 511页，40项准则应用操作指南

## 核心内容

- 基本准则：4项基本假设 + 权责发生制 + 8项会计信息质量要求
- 40项具体准则实务体系（通用业务/特殊业务/报告披露三大层次）
- 收入五步法模型详解
- 金融工具三分类+预期信用损失法
- 新租赁准则承租人使用权资产模型
- 所得税会计资产负债表债务法
- 长期股权投资成本法/权益法
- 2025年4月最新实施问答精选
- PPP项目会计处理三种模式
- CAS与IFRS持续趋同分析
- 会计准则解释第14-17号核心内容
- 数据资源会计处理暂行规定
