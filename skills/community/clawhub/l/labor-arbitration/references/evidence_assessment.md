# 劳动仲裁资料研判参考手册

## 一、研判总则

资料研判是劳动仲裁准备阶段的核心环节，目的在于：
- 确认现有证据能否支撑仲裁请求
- 发现证据链缺口，指导补充收集
- 评估对方可能提出的抗辩及己方反证能力
- 为仲裁请求金额计算提供依据支撑

---

## 二、证据分类体系

### 2.1 劳动关系证据（证明"存在劳动关系"）

| 优先级 | 证据类型 | 证明力 | 说明 |
|--------|---------|--------|------|
| ★★★ | 劳动合同 | 极强 | 最直接证明，约定薪资、岗位、期限等 |
| ★★★ | 社保缴纳记录 | 极强 | 可从社保局或"掌上12333"App下载 |
| ★★★ | 公积金缴存记录 | 强 | 可从公积金管理中心获取 |
| ★★☆ | 工资银行流水 | 强 | 标注"工资"字样的转账记录 |
| ★★☆ | 工资条/工资明细 | 强 | 需有公司盖章或电子签章 |
| ★★☆ | 工牌/门禁卡/工作证 | 中 | 需有公司名称和照片 |
| ★★☆ | 考勤记录 | 中 | 打卡记录、签到表、钉钉/企业微信截图 |
| ★☆☆ | 工作邮件/内部通讯 | 中 | 需体现工作内容和汇报关系 |
| ★☆☆ | 同事证言 | 弱 | 需出庭作证才具较强效力 |
| ★☆☆ | 工服/名片/工位照片 | 弱 | 辅助证据 |

> **关键提示：** 未签劳动合同的情况下，需依据劳社部发〔2005〕12号文件，用工资记录、社保、工牌、考勤等组合证据证明事实劳动关系。

### 2.2 薪资待遇证据（证明"欠薪/欠补偿金额"）

| 证据类型 | 用途 | 获取方式 |
|---------|------|---------|
| 劳动合同薪资条款 | 证明约定工资 | 原件留存 |
| 银行工资流水（近12个月） | 证明实际发放金额和规律 | 银行App/柜台打印 |
| 工资条/薪酬明细 | 证明薪资结构 | HR系统截图/纸质件 |
| 个税APP纳税记录 | 交叉验证实际收入 | 个人所得税App→收入纳税明细 |
| 加班审批单/通知 | 证明加班事实 | 邮件/系统截图 |
| 欠薪承诺书/对账单 | 证明欠薪金额和承认 | 对方书面确认 |
| 微信/邮件催薪记录 | 证明主张过权利（时效中断） | 聊天记录导出 |

### 2.3 解除/终止劳动关系证据

| 证据类型 | 适用场景 |
|---------|---------|
| 解除通知书/辞退通知书 | 用人单位单方解除 |
| 离职证明/离职交接单 | 协商解除或劳动者离职 |
| 限期返岗通知 | 用人单位主张旷工解除 |
| 微信/邮件沟通记录 | 协商解除过程或口头辞退的佐证 |
| 录音录像 | 口头辞退、协商过程 |
| 交接清单 | 证明最后工作日和交接情况 |

### 2.4 程序性证据

| 证据类型 | 用途 |
|---------|------|
| 劳动监察投诉受理回执 | 证明已寻求行政救济，时效中断 |
| 调解申请书及回执 | 证明经过调解前置程序（部分地区要求） |
| 送达回证/快递签收 | 证明文件已送达对方 |
| 仲裁时效中断证据 | 催款记录、投诉记录等 |

## 三、研判工作流程

### 步骤一：梳理仲裁请求

逐项列出劳动者的仲裁请求，常见请求包括：
1. 确认劳动关系存续期间
2. 支付拖欠/克扣工资
3. 支付未签劳动合同二倍工资差额
4. 支付解除/终止劳动合同经济补偿金
5. 支付违法解除劳动合同赔偿金（2N）
6. 支付加班费
7. 支付未休年休假工资
8. 补缴社会保险/公积金
9. 出具离职证明/办理档案转移

### 步骤二：逐项匹配证据

对每一项仲裁请求，按以下框架匹配证据：

```
仲裁请求 → 法律依据 → 所需证据 → 现有证据 → 缺口分析
```

**示例：**
- 请求：支付违法解除赔偿金 50000元
- 法律依据：《劳动合同法》第48条、第87条
- 所需证据：①劳动关系证明 ②解除事实 ③解除违法性 ④月工资标准（计算基数）
- 现有证据：劳动合同✓、辞退通知书✓、银行流水✓
- 缺口分析：需确认辞退理由是否属于法定解除情形，收集解除理由不成立的证据

### 步骤三：证据效力评估

对每份证据从三个维度评分：

1. **真实性** — 是否为原件？电子证据是否完整未被篡改？
2. **合法性** — 获取方式是否合法？录音是否侵犯隐私？
3. **关联性** — 是否与仲裁请求直接相关？

**评分标准：**
- 🟢 充分：三性俱全，可独立支撑请求
- 🟡 基本充分：部分属性需补强，需组合其他证据
- 🔴 不足：关键属性缺失，需补充或换用其他证据

### 步骤四：形成研判报告

研判报告应包含：
1. 仲裁请求清单
2. 每项请求的证据匹配表
3. 证据缺口及补充建议
4. 证据风险点及应对方案
5. 预估胜诉概率（高/中/低）
6. 下一步行动建议

## 四、常见证据风险点

### 4.1 时效风险
- 仲裁时效：自知道或应当知道权利被侵害之日起 **1年**
- 时效中断：有催款记录、投诉记录等可中断时效
- 二倍工资时效：部分地区从未签合同满一年起算，部分地区逐月计算
- **研判要点：** 逐项核对每项请求的时效起算日和截止日

### 4.2 证据灭失风险
- 电子证据（钉钉、企业微信）离职后可能被清退权限
- **研判要点：** 在职时提前截图保存、导出聊天记录
- 建议：所有电子证据截图时需包含时间戳、对方账号信息

### 4.3 录音证据风险
- 需在公共场所或工作场所录制，不得侵入私人空间
- 不能使用窃听设备
- 录音需完整不得剪辑
- **研判要点：** 评估录音获取场景是否合法，内容是否清晰

### 4.4 金额计算风险
- 经济补偿金基数：离职前12个月平均工资（含奖金、补贴）
- 三倍封顶：当地上年度职工月平均工资三倍
- 年限计算：六个月以上不满一年按一年，不满六个月按半个月
- **研判要点：** 核对基数计算是否准确，是否遗漏应计入项目

## 五、研判输出模板

研判完成后，应输出结构化报告，格式参见以下模板：

```
【劳动仲裁资料研判报告】

一、当事人信息
  申请人：XXX  被申请人：XXX公司

二、仲裁请求
  1. 请求事项：...
     金额：¥XXX
  2. ...

三、证据清单
  序号 | 证据名称 | 证据类型 | 证明目的 | 效力评估 | 备注

四、证据缺口
  1. 缺失证据：...
     补充建议：...
  2. ...

五、风险评估
  时效风险：高/中/低
  证据风险：高/中/低
  胜诉概率：高/中/低

六、下一步建议
  1. ...
  2. ...
```
