## Known Risks and Mitigations: <br>
Risk: The guide includes examples for teaching broad remembered tool triggers, which could cause an agent to call tools in unintended situations. <br>
Mitigation: Use specific trigger phrases, avoid persistent auto-run rules for destructive or account-changing actions, and review any proposed tool behavior before relying on it. <br>
Risk: The documentation includes package installation and script examples that may be copied into a local environment. <br>
Mitigation: Review example scripts and install Python packages only from trusted sources before execution. <br>
