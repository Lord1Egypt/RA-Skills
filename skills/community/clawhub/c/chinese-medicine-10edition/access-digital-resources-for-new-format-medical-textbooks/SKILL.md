# Access Digital Resources for Medical Textbooks

## When to Use
- 您拥有第十轮全国高等学校五年制本科临床医学专业规划教材（例如《中医学》第10版）的实体纸质书
- 教材封底贴有红色标签（红标）和其下方覆盖的绿色激活码（绿标）
- 您具备智能手机或移动设备及网络连接

> **注意**：本流程仅适用于配备红标/绿标的新形态教材，旧版无此标识的教材不适用。

## Execution Steps

1. **扫描红标二维码**：使用手机相机或微信扫描教材封底红色标签上的二维码，下载并阅读“使用说明”文档。
2. **揭开红标**：小心撕开红标覆盖层，露出下方的绿色标签激活码。
3. **扫描绿标激活码**：再次使用手机扫描该绿标激活码，系统将跳转至人卫账号注册或登录页面。
4. **完成账号绑定**：注册新账号或登录已有账号，系统将自动为您绑定该教材的数字资源访问权限。
5. **访问数字资源**：绑定成功后，您可随时通过以下任一方式访问内容：
   - 扫描书中任意章节旁的二维码
   - 再次扫描绿标激活码
   - 登录 [zengzhi.ipmph.com](https://zengzhi.ipmph.com)
   - 下载并使用“人卫智网”App
6. **使用功能**：电子教材内容与纸质版完全一致，支持多终端（手机、平板、电脑）浏览，并提供目录导航、全文检索、虚拟仿真、AI互动、思维导图等增强学习功能。

## Output
成功绑定教材数字资源权限，实现纸数一体化学习体验，随时随地访问配套电子内容与互动资源。
