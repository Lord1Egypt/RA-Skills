 (辅助)    (辅助)     (辅助)     (辅助)    (增强)

## 二、LLM 辅助创意

### 2.1 概念测试(Concept Testing with LLM)

**方法**: 让 LLM 扮演目标受众, 对多个概念进行评估。

```
Prompt 模板:
"你是 [人群描述] 的典型用户。
请对以下 [N] 个产品概念评分(1-10), 并说明:
1. 你最看重的 3 个利益点是什么?
2. 每个概念在这 3 个利益点上的表现?
3. 你会购买哪个? 为什么?
4. 哪个概念最让你'心动'? 为什么?

概念 A: [描述]
概念 B: [描述]
..."
```

**注意**: LLM 概念测试**不能替代**真实用户测试, 仅用于:
- ✅ 早期快速筛选(从 10 个方向筛到 3 个)
- ✅ 发现明显盲点(如文化敏感)
- ❌ 不能作为最终决策依据

### 2.2 文案测试

| 测试类型 | LLM 任务 | 输出 |
|---------|---------|------|
| **标题测试** | 扮演用户评估吸引力 | 排序 + 理由 |
| **Slogan 测试** | 评估记忆度 / 独特性 | 评分 |
| **CTA 测试** | 评估点击意愿 | 排序 |
| **场景化文案** | 不同场景下的文案适配度 | 多版本生成 |

### 2.3 创意方向拓展

**SCAMPER + LLM 组合**:

```
Prompt:
"基于以下产品 [描述], 用 SCAMPER 方法生成 10 个创意方向:
- S (Substitute): 替换什么?
- C (Combine): 组合什么?
- A (Adapt): 借鉴什么?
- M (Modify/Magnify): 放大什么?
- P (Put to other use): 其他用途?
- E (Eliminate): 删除什么?
- R (Reverse): 反转什么?

每个方向输出:
- 一句话概念
- 目标人群
- 潜在 Insight
- 可能的执行方向"
```

## 三、扩散模型视觉探索

### 3.1 Moodboard 生成(Midjourney / DALL-E / SD)

**适用**: brand / creative Agent 的视觉风格探索

```
Prompt 模板(Moodboard):
"[品牌类型] 品牌 moodboard, 包含以下元素:
- 色彩: [主色 + 辅色]
- 风格: [极简 / 复古 / 未来感 / 手绘]
- 情绪: [温暖 / 科技感 / 高级 / 亲切]
- 参考: [类似品牌风格]
- 排除: [不想出现的元素]

输出 6-8 张图, 不同变体"
```

### 3.2 Logo 方向探索

```
Logo Prompt 结构:
"[品牌名] logo, [行业], [风格关键词],
[颜色], [图形元素], [排除项],
white background, vector style, minimal, --no text"
```

**注意**: AI 生成的 Logo 仅作为**方向参考**, 最终需人类设计师基于 Logo Brief 执行。

### 3.3 场景视觉化

| 用途 | Prompt 要点 |
|------|-----------|
| **产品场景** | 产品 + 使用场景 + 光线 + 构图 |
| **KV 主视觉** | 大气 / 张力 / 留白 / 文字位 |
| **包装效果** | 真实质感 / 阴影 / 反光 |
| **广告场景** | 人物 + 互动 + 情绪 |

## 四、AI 视频生成(2025-2026)

### 4.1 主流工具

| 工具 | 类型 | 适用 | 时长 |
|------|------|------|:---:|
| **Sora / Veo / Seedance** | 文生视频 | 短视频 / 创意原型 | 5-60s |
| **Runway Gen-3** | 文/图生视频 | 创意视频 / 特效 | 5-30s |
| **Pika / Kling** | 文/图生视频 | 短视频 | 3-15s |
| **HeyGen / Synthesia** | 数字人 | 产品讲解 / 教程 | 不限 |
| **CapCut / 剪映** | AI 剪辑 | 短视频批量生产 | 不限 |

### 4.2 AI 视频在创意中的应用

| 场景 | AI 价值 | 局限 |
|------|--------|------|
| **创意原型** | 快速可视化 Big Idea | 不能替代拍摄 |
| **Mood 视频** | 传达调性 / 节奏 | 真实感不足 |
| **产品演示** | 标准化讲解 | 复杂动作有限 |
| **批量素材** | 多尺寸 / 多文案适配 | 需人工终审 |
| **数字人** | 节省出镜成本 | 表演上限有限 |

## 五、AI 创意质量控制

### 5.1 AI 生成内容的 5 大风险

| 风险 | 描述 | 防范 |
|------|------|------|
| **版权问题** | 训练数据可能侵权 | 商用前确认授权 |
| **品牌一致性** | AI 输出可能偏离品牌 | 终审 + Brand Guideline |
| **文化敏感** | AI 可能不识别本土敏感点 | 人工审核 |
| **法务合规** | 广告法 / 平台规则 | 法务过审 |
| **真实感不足** | AI 生成可能不自然 | 真实场景验证 |

### 5.2 AI 创意审核清单

- [ ] **品牌一致性**: 色彩 / 字体 / 风格是否符合 VI?
- [ ] **文案准确性**: 是否有夸大 / 绝对化 / 违禁词?
- [ ] **文化敏感**: 是否有种族 / 性别 / 地域歧视?
- [ ] **视觉真实**: 是否过度美化导致失真?
- [ ] **版权清晰**: 训练素材是否授权?
- [ ] **平台适配**: 不同平台尺寸 / 时长是否覆盖?
- [ ] **A/B 预测**: 是否经过 LLM 概念测试?

## 六、AI 辅助创意工作流(推荐)

### 6.1 标准 AI 增强创意流程

```
Phase 1: Insight 挖掘(LLM)
    - 用户评论 NLP 分析
    - 趋势识别
    - 洞察生成
    ↓
Phase 2: 概念发想(LLM + SCAMPER)
    - 10+ 方向快速生成
    - LLM 概念测试筛选到 3 个
    ↓
Phase 3: 视觉探索(扩散模型)
    - Moodboard 生成
    - Logo 方向(如需要)
    - KV 概念图
    ↓
Phase 4: 素材制作(AI 工具)
    - 批量生成多尺寸
    - 多文案适配
    ↓
Phase 5: 效果测试(A/B)
    - 小流量测试
    - 数据驱动优化
```

### 6.2 效率提升预估

| 环节 | 传统耗时 | AI 增强 | 提升倍数 |
|------|:------:|:------:|:------:|
| Insight 挖掘 | 3-5 天 | 1 天 | 3-5x |
| 概念发想 | 2-3 天 | 0.5 天 | 4-6x |
| Moodboard | 2-3 天 | 0.5 天 | 4-6x |
| 素材制作 | 5-10 天 | 1-2 天 | 5-10x |
| **整体** | 12-21 天 | 3-5 天 | **4-5x** |
