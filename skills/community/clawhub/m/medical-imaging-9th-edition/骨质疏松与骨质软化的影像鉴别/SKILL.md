# 骨质疏松与骨质软化的影像鉴别

## 触发条件
当影像检查显示骨密度普遍减低，且临床需要区分骨质疏松与骨质软化时，应用本技能。

## 执行步骤

1. **评估骨小梁和骨皮质边缘清晰度**：
   - 若边缘清晰，骨小梁变细、减少，骨皮质分层变薄 → 提示骨质疏松。
   - 若边缘模糊（因含大量未钙化骨样组织）→ 提示骨质软化。

2. **观察骨骼形态变化**：
   - **骨质疏松特征**：
     - 脊椎：椎体内纵行条纹，严重时椎体变扁、上下缘内凹，椎间隙呈梭形增宽；可压缩成楔状。
     - 易发生病理性骨折。
   - **骨质软化特征**：
     - 承重骨变形（如“三叶草样骨盆”）。
     - 出现假骨折线（Looser带）：宽度1～2mm的透明线，与骨皮质垂直，边缘稍致密，好发于耻骨支、肱骨、股骨上段、胫骨。

3. **结合病因线索**：
   - 骨质疏松常见于老年、绝经后、甲旁亢、维生素C缺乏、酒精中毒（广泛性）；或骨折后、感染、肿瘤（局限性）。
   - 骨质软化多由钙磷代谢障碍或维生素D缺乏引起；生长期称佝偻病，成年称骨软化症。

4. **必要时参考MRI表现**：
   - 骨质疏松：T₁WI和T₂WI上信号增高（黄骨髓增多）；骨皮质内线状高信号提示哈氏管扩张。
   - 骨质软化：MRI无特异性，但X线特征明确，应优先依赖X线。

## 输出判断
- 若同时存在**边缘模糊 + 假骨折线 + 承重骨变形** → 提示骨质软化。
- 若同时存在**边缘清晰 + 椎体楔变 + 骨小梁稀疏** → 提示骨质疏松。

> 注意：本技能不适用于急性骨折或肿瘤性骨破坏。必须结合临床病史综合判断。
