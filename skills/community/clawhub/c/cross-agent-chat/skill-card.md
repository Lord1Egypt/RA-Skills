## Description: <br>
Cross Agent Chat standardizes how an agent finds reachable sessions and sends clear messages to other agents with session messaging tools. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ebly](https://clawhub.ai/user/ebly) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Agents and their operators use this skill when a workflow requires cross-agent coordination, such as asking another agent about capabilities, delegating a task, or continuing a collaborative exchange. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Cross-agent messages can expose secrets, credentials, private user data, internal prompts, or unrelated conversation context to another session. <br>
Mitigation: Verify the recipient before sending and transfer sensitive or private context only when the action is intentional and authorized. <br>
