## Description: <br>
Configure TTS in OpenClaw and adapt speech output to user preferences. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[zktufo](https://clawhub.ai/user/zktufo) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and OpenClaw users use this skill to tune text-to-speech behavior, remember compact voice preferences, and choose appropriate TTS provider settings. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may save voice and speaking-style preferences that reflect user communication habits. <br>
Mitigation: Review saved Voice, Style, Spoken Text, and Avoid entries before keeping or sharing the skill state. <br>
Risk: TTS setup may involve optional paid provider API keys. <br>
Mitigation: Confirm TTS configuration changes yourself and avoid placing real API keys in shared files, logs, or screenshots. <br>
