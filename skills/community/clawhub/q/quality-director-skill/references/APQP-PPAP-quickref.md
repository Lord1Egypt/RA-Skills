## APQP 五阶段概览

```
Phase 1：概念设计和产品开发策划
  输入：市场需求 → 输出：设计目标 / DFMEA启动 / 特殊特性清单

Phase 2：产品和过程设计开发（样件阶段）
  输入：设计目标 → 输出：DFMEA完成 / 特殊特性确认 / 样件OTS

Phase 3：过程设计和产品验证（试生产）
  输入：样件 → 输出：PFMEA / 控制计划 / SPC / MSA / 试产PPAP

Phase 4：产品和过程确认（量产批准）
  输入：试产验证 → 输出：PPAP批准 / SPC系统建立 / 量产爬坡

Phase 5：产品和过程持续改进（量产阶段）
  输入：量产数据 → 输出：Cpk改善 / 客户投诉8D / 供应商绩效管理
```

## 各阶段输入/输出检查表

### Phase 1：概念设计和产品开发策划

**输入：**
- [ ] 客户需求（VOC/SPEC/OR）
- [ ] 市场调研报告
- [ ] 竞品分析（Benchmarking）
- [ ] 法规要求清单（安全/环保/能效）
- [ ] 供应商能力初步评估

**输出：**
- [ ] 设计目标（性能/成本/重量/可靠性/质量目标）
- [ ] 初始BOM（物料清单）
- [ ] DFMEA启动（V1.0）
- [ ] 特殊特性清单（KPC/KPVT）初版
- [ ] 制造可行性分析
- [ ] 项目计划书（AQP/PPAP）

### Phase 2：产品和过程设计开发

**输入：** Phase 1 所有输出

**输出：**
- [ ] DFMEA完成（RPN≤100优先整改）
- [ ] DFM/DFX评审报告
- [ ] 特殊特性控制计划（KPC确认）
- [ ] 工程样件OTS报告
- [ ] 量具/检具设计
- [ ] 初始过程流程图（V1.0）
- [ ] PFMEA启动

### Phase 3：过程设计和产品验证

**输入：** Phase 2 所有输出 + 样件

**输出：**
- [ ] PFMEA完成（V1.0）
- [ ] 控制计划CP（试产版）
- [ ] 过程能力报告（Cpk≥1.33关键特性）
- [ ] MSA研究报告（GR&R≤10%）
- [ ] 包装验证报告
- [ ] 生产件PPAP（Level 3）

### Phase 4：产品和过程确认

**输入：** Phase 3 验证数据

**输出：**
- [ ] PPAP批准五项标准（PSW签署）
  1. 设计记录所有项目符合 ✓
  2. 过程FMEA和控制计划符合 ✓
  3. 过程能力Cpk≥1.33（关键安全特性Cpk≥1.67）✓
  4. 生产节拍符合客户要求 ✓
  5. 包装符合要求 ✓
- [ ] SPC控制图系统建立
- [ ] 量产爬坡（从小到大逐步增量）
- [ ] 质量门（Quality Gate）评审

### Phase 5：持续改进

**关键活动：**
- 量产SPC运行（日常监控）
- Cpk持续改善（目标≥2.0）
- 客户投诉8D根本原因分析
- 供应商绩效管理（SQM）
- 年度管理评审
- APQP流程改进（Lessons Learned）

## PPAP 提交等级速查

| 提交等级 | 要求内容 | 使用场景 |
|---------|---------|---------|
| **Level 1** | 零件提交保证书（PSW）仅 | 最简，顾客要求时 |
| **Level 2** | Level 1 + 产品样品 + 尺寸报告 | 一般客户 |
| **Level 3** | Level 1 + 产品样品 + 完整尺寸数据 + 材料/性能测试 | 大多数客户 |
| **Level 4** | Level 1 + 产品样品 + 完整尺寸数据 + 过程FMEA + CP + 流程图 + MSA + SPC | 顾客要求时 |
| **Level 5** | Level 1 + 产品样品 + 全部支持数据（客户要求时全部提交）| 最高要求 |

## 特殊特性（KPC/KPVT）识别与控制

### 特殊特性分类

| 类型 | 定义 | 控制要求 |
|------|------|---------|
| **KPC**（关键产品特性）| 影响产品安全/法规符合性的特性 | 100%检测/全项目检验记录 |
| **KPVT**（关键过程变量）| 影响KPC达成的过程参数 | SPC控制图/防错 |
| **CC**（控制特性）| 影响制造过程稳定性的特性 | SPC控制图 |

### 特殊特性传递流程

```
顾客需求 → 设计FMEA → 特殊特性清单 → 工艺FMEA → 控制计划 → 量产SPC
  （KPC）   （DFMEA）  （KPVT清单）  （PFMEA）   （CP）    （控制图）
```
