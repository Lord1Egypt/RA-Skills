## Description: <br>
Obsidian Master Assistant helps users configure Obsidian with folder architecture guidance, plugin recommendations, note templates, Bases database setup, AI integration guidance, and a four-week learning path. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and developers use this skill to plan and configure an Obsidian knowledge-management vault, including templates, plugins, databases, and optional AI-assisted workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Optional chat import, AI, sync, or memory features may store sensitive personal content in an Obsidian vault or memory directory. <br>
Mitigation: Review optional integrations before enabling them, connect only intended chat accounts, and periodically delete or protect sensitive imported notes. <br>
Risk: Optional Claude or Google AI Studio integrations may require API credentials. <br>
Mitigation: Keep API keys secure and avoid committing or sharing local configuration that contains credentials. <br>
Risk: Cloud synchronization may replicate vault or memory contents beyond the local device. <br>
Mitigation: Confirm whether the Obsidian vault or memory directory syncs to cloud storage and apply appropriate access controls. <br>
