# 肛周脓肿的病因与病理机制

## 何时使用
当患者出现以下情况时启用本技能：
- 肛门局部红、肿、热、痛
- 伴或不伴发热等全身感染症状
- 需明确脓肿是否源于肛腺感染以制定手术策略

## 执行步骤
1. **确认解剖基础**：识别肛窦（位于齿状线附近、开口向上的隐窝）及其与肛腺的连接关系。
2. **追溯感染起源**：判断是否存在粪渣存留于肛窦，引发肛窦炎，进而导致肛腺感染。
3. **追踪感染扩散路径**：
   - 感染首先累及位于内、外括约肌之间的肛腺，形成括约肌间感染；
   - 随后沿疏松结缔组织间隙扩散，形成不同类型脓肿：
     - 向上 → 高位肌间脓肿或骨盆直肠间隙脓肿
     - 向下 → 肛周皮下脓肿
     - 向外 → 坐骨直肠窝脓肿
     - 向后 → 肛管后间隙或直肠后间隙脓肿
4. **评估继发因素**：排查是否由肛裂、内痔注射治疗、克罗恩病、糖尿病等基础疾病诱发。
5. **排除非适用情况**：若脓肿由放射性损伤等非感染性炎症引起，则本机制不适用。

## 输出目标
明确原发感染源为肛腺，为手术中定位内口和预防肛瘘提供病理依据。
