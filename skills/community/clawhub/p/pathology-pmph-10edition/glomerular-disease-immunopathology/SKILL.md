# 肾小球疾病免疫病理机制与分类

## 适用场景
- 蛋白尿、血尿、肾病综合征
- 快速进展至少尿/无尿
- 肾活检显示沉积物或新月体

## 执行步骤

### 一、确定免疫机制类型
1. **循环免疫复合物沉积**：
   - 抗原为外源性或非肾小球性
   - 免疫荧光呈“颗粒状”
   - 沉积部位受复合物电荷影响（阳离子→上皮下；阴离子→内皮下）

2. **原位免疫复合物沉积**：
   - 抗体直接攻击肾小球固有抗原
   - 包括：
     - 抗GBM抗体型（线性荧光）
     - Heymann肾炎模型（足细胞抗原）
     - 植入抗原型

### 二、急进性肾炎（RPGN）三型鉴别
前提：>50%肾小球有新月体 + 快速肾衰
- **Ⅰ型（抗GBM型）**：
  - 线性IgG沉积，抗GBM抗体阳性，可伴肺出血
- **Ⅱ型（免疫复合物型）**：
  - 颗粒状沉积，电镜见电子致密物，继发于SLE/链球菌感染等
- **Ⅲ型（寡免疫型）**：
  - 免疫荧光阴性，ANCA阳性，关联血管炎

### 三、膜性肾小球病机制
- 原发性占85%，为自身免疫病
- 自身抗体结合上皮细胞膜抗原 → 原位复合物 → MAC激活 → 蛋白漏出

> **关键操作**：必须结合免疫荧光、电镜及血清学标志物（抗GBM、ANCA）综合判断。
