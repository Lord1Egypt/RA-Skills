# 信源分级标准

> 用于 s3 原始结果评级和 s6 证据整理

## 分级体系

### T1 — 一手权威（P0）

| 类型 | 示例 |
|------|------|
| 官方政策文件 | 国务院发文、部委通知、地方政策 |
| 法律/法规原文 | 法条、司法解释、行业标准 |
| 行业报告原文 | 艾瑞、赛迪、IDC、Gartner 发布的正式报告 |
| 官方统计数据 | 国家统计局、行业监管机构公布的统计数据 |
| 企业官方公告 | 上市公司年报、招股书、正式新闻稿 |

**特征**：可追溯至原始文件，有文号/发布机构/发布日期

### T2 — 权威二手（P1）

| 类型 | 示例 |
|------|------|
| 学术论文 | 核心期刊、会议论文、硕博论文 |
| 权威研究机构报告 | 北大产研院、中国信通院、McKinsey/BCG/Bain |
| 知名智库报告 | 国务院发展研究中心、社科院 |

**特征**：有研究方法论说明，有署名作者/机构

### T3 — 专业媒体（P2）

| 类型 | 示例 |
|------|------|
| 知名科技媒体 | 36氪、虎嗅、机器之心、TechCrunch、The Verge |
| 财经媒体 | 财新、第一财经、华尔街日报、Bloomberg |
| 垂直专业媒体 | 雷锋网、量子位、InfoQ |

**特征**：有编辑审核流程，但不等同于一手数据

### T4 — 社区/自媒体（P3）

| 类型 | 示例 |
|------|------|
| 社区讨论 | 知乎（高赞/专业回答除外）、Reddit、V2EX |
| 自媒体 | 个人公众号、个人博客、头条号 |
| 行业自媒体 | 未验证的行业分析文 |

**特征**：无编辑审核，内容质量不可控

### T5 — 不可引用（P4）

| 类型 | 示例 |
|------|------|
| AI 生成内容 | 未经验证的 AI 撰文 |
| 匿名来源 | 无署名、无日期的网络文章 |
| 明显软文/广告 | 以"推荐""排行"为名的商业推广 |

## 使用规则

- **T1-T2**：可作为核心论据，必须注明具体文档/编号/条款
- **T3**：可作为辅助佐证，不可作为唯一来源支撑核心结论
- **T4**：仅作背景参考和线索发现，出现于报告中必须标注"⚠️ 未经充分印证"
- **T5**：不可引用

## 三源印证规则

核心结论（报告中加粗的判断句）必须满足：
- 至少有 3 个独立来源支撑，其中至少 1 个为 T1-T2
- 如果只有 2 个来源，标注"⚠️ 仅双源印证"
- 如果只有 1 个来源，标注"⚠️ 单一来源，置信度低"
- AI 生成内容不计入印证来源数
