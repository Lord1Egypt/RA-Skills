# 化学突触信号转导级联反应

## 何时使用
当需要解释以下任一情况时应用本技能：
- 神经冲动如何跨突触传递
- 神经递质释放的分子机制
- 药物如何通过影响突触传递发挥作用
- 突触后兴奋性或抑制性电位的产生原理
- 突触整合与动作电位触发的阈值逻辑

## 执行步骤

1. **确认结构基础**：验证分析对象是否为化学突触，包含突触前成分（轴突终末/突触小体）、15–30 nm 的突触间隙和突触后成分（树突或胞体膜）。

2. **追踪信号传递流程**：
   - 步骤1：神经冲动到达轴突终末 → 激活电压门控 Ca²⁺ 通道。
   - 步骤2：Ca²⁺ 内流 → 在 ATP 参与下使突触素（synapsin）磷酸化。
   - 步骤3：磷酸化突触素降低与突触小泡亲和力 → 小泡脱离细胞骨架并移向突触前膜。
   - 步骤4：小泡与前膜融合 → 出胞释放神经递质至突触间隙。
   - 步骤5：递质结合突触后膜特异性受体 → 打开离子通道 → 产生兴奋性突触后电位（EPSP）或抑制性突触后电位（IPSP）。

3. **判断突触效应类型**：根据神经递质种类及其受体类型，确定输出为兴奋性或抑制性。

4. **评估突触整合结果**：若单个神经元接收的兴奋性输入总和超过抑制性输入且达到阈值，则在轴突起始段触发动作电位。

5. **关联药理或病理意义**：将上述机制用于解释药物作用（如受体激动剂/拮抗剂）或突触功能障碍（如递质合成/释放异常）。

> 注意：本技能仅适用于化学突触，排除电突触（缝隙连接）及非神经元间连接。
