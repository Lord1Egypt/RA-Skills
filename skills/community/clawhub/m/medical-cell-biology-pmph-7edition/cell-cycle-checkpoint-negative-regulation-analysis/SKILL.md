# 细胞周期检查点负调控机制分析

## 何时使用
- 用户询问细胞如何响应DNA损伤、未完成复制、纺锤体组装失败或染色体分离错误
- 需要解释ATM、ATR、p53、Mad2等蛋白在检查点中的角色
- 需判断细胞周期是否应阻滞或启动凋亡
- 分析癌细胞中检查点失效的潜在机制

## 执行步骤

### 1. 确认触发条件
检查是否存在以下异常状态之一：
- DNA未完全复制
- DNA损伤（如双链断裂）
- 纺锤体未正确组装
- 染色体未正确分离至两极

若满足任一条件，则对应检查点被激活。

### 2. 应用通用三步机制
对任意检查点，按以下逻辑执行：
- **感受（Sensor）**：识别异常的特定蛋白结合异常结构（如ATR结合停滞复制叉，Mad2结合未连接动粒）。
- **转导（Conductor）**：通过激酶级联放大信号（如ATR→Chk1；ATM→Chk2）。
- **效应（Effector）**：抑制Cdk活性或阻止APC/C活化，导致细胞周期阻滞。

### 3. 匹配具体检查点路径
根据异常类型选择对应通路：
- **DNA复制检查点**：ATR → Chk1 → Cdc25失活 → cyclin A/B-Cdk1抑制 → 阻止进入M期
- **DNA损伤检查点**：ATM/ATR → Chk1/2 → Cdc25降解 + p53稳定 → p21表达 → 抑制cyclin E/A-Cdk2 → G₁/S阻滞
- **纺锤体组装检查点**：Mad2抑制Cdc20 → APC/C不活化 → securin不降解 → 姐妹染色单体不分离
- **染色体分离检查点**：监测子代染色体位置 → 调控Cdc14活化 → 控制M期cyclin降解 → 决定是否进入末期

### 4. 判断最终命运
- 若异常可修复：解除检查点，周期继续
- 若损伤不可逆（如严重DNA双链断裂）：启动凋亡程序

> 注意：本机制仅适用于具有完整检查点通路的真核细胞；癌细胞常因关键蛋白突变而失效。
