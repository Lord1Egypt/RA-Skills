## Use Case: <br>
Developers, product teams, and marketers use this skill to create A/B test variants, record metrics, inspect results, and decide which content or experience performs better. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Crafted test names can cause local JSON files outside the intended test folder to be read or overwritten. <br>
Mitigation: Use trusted, simple test names, avoid storing sensitive analytics without proper privacy controls, and prefer a fixed version that validates names or confines resolved paths to the intended test folder. <br>
