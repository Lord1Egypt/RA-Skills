# 行业研究报告技能

## 适用场景

当用户需要以下任务时加载本技能：

- 研究某个行业的整体情况
- 为投资或并购项目做行业尽调
- 快速了解一个陌生行业的竞争格局与机会
- 向董事会或合作方呈现行业分析

## 分析框架总览

本技能采用四维经典框架，按顺序执行：

1. **PEST 宏观环境分析** — 政治/法规、经济、社会/人口、技术趋势
2. **波特五力竞争分析** — 现有竞争、新进入者威胁、替代品威胁、供应商议价力、买方议价力
3. **价值链分析** — 行业上中下游结构、利润池分布、关键环节
4. **市场规模测算** — TAM（总可寻址市场）/ SAM（可服务市场）/ SOM（可获取市场），附增速预测

## 执行步骤

### 第一步：明确研究标的

如果用户未说明以下信息，须先询问：
- 具体行业名称（如：中国新能源汽车行业、全球云计算SaaS行业）
- 地理范围（全球 / 中国 / 特定区域）
- 报告截止时间基准（默认当前年份）
- 是否有特定关注点（如某细分赛道、某竞争对手）

### 第二步：并行信息收集

使用 `search_web` 工具，按以下维度并行搜索最新数据，每个维度至少2条查询：

**PEST信息**
- `[行业名] 政策法规 最新 [年份]`
- `[行业名] 宏观经济驱动因素`
- `[行业名] 消费者趋势 人口结构`
- `[行业名] 技术创新趋势`

**竞争格局**
- `[行业名] 主要竞争对手 市场份额 [年份]`
- `[行业名] 行业集中度 CR3 CR5`
- `[行业名] 新进入者 进入壁垒`
- `[行业名] 替代品威胁`

**价值链**
- `[行业名] 产业链上下游结构`
- `[行业名] 利润率分布 毛利率`
- `[行业名] 供应链关键节点`

**市场规模**
- `[行业名] 市场规模 [年份] 亿元 或 billion`
- `[行业名] 市场增长率 CAGR 预测`
- `[行业名] TAM SAM 市场空间`

### 第三步：撰写报告结构

报告必须包含以下章节，按顺序输出：

---

#### 封面信息
- 报告标题：《[行业名称] 行业深度研究报告》
- 研究日期
- 免责声明（仅供内部参考，不构成投资建议）

#### 执行摘要（Executive Summary）
- 行业定义与核心结论（3-5条要点）
- 投资吸引力评级（高/中/低）及理由
- 主要风险提示

#### 一、PEST 宏观环境分析
每个维度输出：
- 关键因素（2-4条）
- 对行业的影响（正面/负面/中性）
- 数据支撑与信息来源

| 维度 | 关键因素 | 影响方向 | 核心数据/来源 |
|------|---------|---------|-------------|
| 政治（P） | ... | ↑正面 | ... |
| 经济（E） | ... | ... | ... |
| 社会（S） | ... | ... | ... |
| 技术（T） | ... | ... | ... |

#### 二、波特五力竞争分析
对每个"力"进行评分（强/中/弱）并说明理由：

1. **行业内部竞争**：集中度、价格战程度、主要玩家
2. **新进入者威胁**：资本壁垒、牌照/资质、品牌壁垒
3. **替代品威胁**：现有或潜在替代方案
4. **供应商议价力**：关键供应商集中度、可替代性
5. **买方议价力**：客户集中度、转换成本、价格敏感度

输出五力雷达评分表：

| 竞争力量 | 强度评级 | 核心判断依据 |
|---------|---------|------------|
| 行业内竞争 | 强/中/弱 | ... |
| 新进入者威胁 | ... | ... |
| 替代品威胁 | ... | ... |
| 供应商议价力 | ... | ... |
| 买方议价力 | ... | ... |

**综合竞争强度结论**：[高/中/低竞争烈度，对盈利能力的影响]

#### 三、价值链分析
- 绘制上中下游结构（文字描述或ASCII示意图）
- 各环节毛利率对比
- 利润池集中在哪个环节
- 关键价值创造节点与战略卡位建议

| 环节 | 代表企业类型 | 估算毛利率 | 竞争强度 | 战略价值 |
|------|------------|---------|---------|---------|
| 上游（原材料/供应） | ... | ...% | ... | ... |
| 中游（生产/平台） | ... | ...% | ... | ... |
| 下游（渠道/服务） | ... | ...% | ... | ... |

#### 四、市场规模测算
- **TAM**（总可寻址市场）：定义、规模、数据来源
- **SAM**（可服务市场）：基于地域/细分的缩窄口径
- **SOM**（可获取市场）：实际可竞争份额估算

市场增速预测表（未来3-5年）：

| 年份 | 市场规模（亿元） | 同比增速 | 关键假设 |
|------|--------------|---------|--------|
| [当前年] | ... | ...% | ... |
| [+1年] | ... | ...% | ... |
| [+2年] | ... | ...% | ... |
| [+3年] | ... | ...% | ... |

#### 五、综合投资价值评估
- **吸引力得分**：从市场空间、竞争格局、增长确定性三维度打分（各1-5分）
- **关键成功要素**（KSF）：进入该行业必须具备的核心能力
- **主要风险因素**：政策风险、市场风险、竞争风险，每类至少2条
- **战略建议**：是否推荐进入？若进入，建议切入点与时机

#### 附录
- 主要数据来源列表（含URL）
- 关键术语释义
- 分析局限性说明

---

### 第四步：生成 PDF 报告

加载 `office/pdf` 技能，将上述报告内容转换为正式 PDF 文件。

PDF 格式要求：
- 封面页独立
- 各章节有清晰标题与页码
- 表格样式清晰，数据来源标注在表格下方
- 页脚注明"仅供内部参考，不构成投资建议"
- 字体专业（中文宋体/黑体，英文 Arial/Times New Roman）

### 第五步：交付确认

报告完成后，告知用户：
1. PDF 文件已生成，可下载
2. 报告所用核心数据来源列表
3. 提示用户可以要求深挖某一章节或更新某类数据

## 质量控制要求

- 每个数据点必须注明来源（网址或机构名称）
- 不得凭空捏造市场规模数字，若数据无法确认，须标注"估算"并说明假设
- PEST 分析须区分当前现状与未来趋势
- 波特五力评级须有具体事实支撑，避免主观判断
- 市场规模须区分自上而下（Top-down）和自下而上（Bottom-up）两种测算逻辑，至少采用其中一种并说明方法

## 示例对话

**用户**：帮我研究一下中国宠物食品行业，我们在考虑投资一家宠物粮企业。

**技能响应**：
1. 确认研究范围：中国宠物食品行业，重点关注宠物粮细分
2. 并行搜索 PEST/竞争/价值链/市场规模信息
3. 按框架撰写完整报告
4. 生成 PDF 并交付
