## Known Risks and Mitigations: <br>
Risk: The skill includes dated model price examples and ROI estimates that may become inaccurate. <br>
Mitigation: Verify current provider pricing and local exchange rates before using the estimates for budgeting or purchasing decisions. <br>
Risk: Configuration snippets use placeholder API keys and could encourage unsafe handling of real credentials. <br>
Mitigation: Store real API keys in secure configuration or secret storage, and do not paste secrets into prompts or shared documents. <br>
