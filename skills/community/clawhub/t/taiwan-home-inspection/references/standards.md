# 台灣建築法規與標準參考

## 建築法規

### 建築技術規則（主要條文）
| 項目 | 法規要求 |
|------|----------|
| 樓板厚度（RC構造） | 一般≧12cm，屋頂板≧15cm |
| 樑深 | 一般為跨距之1/12~1/15 |
| 柱斷面最小尺寸 | 不得小於30cm（未裝修） |
| 鋼筋保護層厚度 | 樓板：2cm，樑/柱：3cm以上 |
| 混凝土強度 | 一般結構f'c≧210kgf/cm² |

### 海砂屋相關
| 項目 | 標準 |
|------|------|
| CNS 3090 氯離子含量標準 | 新拌混凝土≦0.15kg/m³ |
| 高氯離子混凝土建築物(海砂屋) | 檢測值>0.3kg/m³ (部分縣市認定標準) |
| 建議檢測時機 | 屋齡>15年、靠近沿海地區 |

### 輻射鋼筋
| 項目 | 標準 |
|------|------|
| 年有效劑量≤1mSv | 對應劑量率約0.2μSv/h |
| 建議檢測時機 | 屋齡>20年、特定時期建築(1982~1984) |

### 耐震相關
| 項目 | 要點 |
|------|------|
| 89年以前建築 | 舊耐震設計標準，建議補強評估 |
| 921地震後(1999) | 耐震規範大幅提升 |
| TVBS(2005年起) | 實施鋼筋綁紮配管監造制度 |

## 重點提醒

1. **非破壞檢測優先**：先目視→工具檢測→必要時才破壞性取樣
2. **需專業判斷的情況**：結構裂縫→應由結構技師判斷
3. **檢測時機**：建議在晴天進行，但檢查滲水最好在大雨後
4. **權責區分**：
   - 屋主負責：專有部分（室內、陽台、附屬建物）
   - 管委會負責：共用部分（屋頂、外牆、公共管線）
   - 建商負責：保固期內（一般結構保固15年，防水5年）
