# Student Profile Template

Create a local profile in `memory/long-term.md`. Keep personally identifying information out of open-source repositories.

```markdown
# Long-Term Memory

## Student

- Display name: <student nickname or generic label>
- Education system: 中国大陆小学
- School entry year: <YYYY>
- Region/school calendar notes: <optional>
- Current grade rule:
  - September 1 to January 31: first semester of the school year.
  - February 1 to August 31: second semester of the school year.
- Current textbook/curriculum: <optional>
- Textbook edition: <人教版 / 北师大版 / 苏教版 / other>
- Textbook volume: <一年级上册 / 一年级下册 / etc.>

## Stable Parent Preferences

- Parent-facing explanations should be short and practical.
- Worksheets should be printable PDF/HTML, with PDF as the default delivery artifact when Chrome/Chromium is available.
- Child-facing worksheets should not contain answers.
- Answer keys and grading rules stay in `answer-key.md`.
- Preferred reminder channel: <optional>

## Stable Learning Rules

- Separate reading/modeling errors from calculation errors.
- Use short validation practice after correction.
- Use spaced review for repaired weak points.
- Keep school mistakes and system-generated worksheet mistakes in separate files.
- Do not generate out-of-scope textbook content unless preview was requested.
- Scheduled tasks may suggest reminders but should not change mastery status automatically.
```
