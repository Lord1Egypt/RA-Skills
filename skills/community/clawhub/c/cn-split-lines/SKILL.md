# cn-split-lines#

文本拆分工具。按照指定分隔符将文本拆分成多行。

## 使用方法#

```bash
python3 scripts/split_lines.py --text "a,b,c" --delimiter ","
python3 scripts/split_lines.py --file input.txt --delimiter "|"
```

纯Python标准库，无需外部依赖。
