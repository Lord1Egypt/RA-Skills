# 乙酰胆碱内皮依赖性血管舒张机制分析

## 何时使用
- 分析乙酰胆碱对特定血管（动脉、静脉、冠状血管、肺血管）的药理作用
- 解释为何在某些病理状态下（如内皮损伤）ACh引起血管收缩而非舒张
- 设计或解读涉及内皮功能的血管反应实验

## 执行步骤
1. **确认血管类型**：明确目标血管属于动脉、静脉、冠状血管或肺血管之一。
2. **评估内皮完整性**：检查血管内皮是否完整（是/否）。
3. **判断M₃受体状态**：确认内皮细胞表达功能性M₃胆碱受体。
4. **应用乙酰胆碱**：模拟或观察ACh与内皮M₃受体结合。
5. **推导下游效应**：
   - 若内皮完整 → ACh激活M₃受体 → 释放EDRF/NO → 平滑肌cGMP升高 → **血管舒张**
   - 若内皮受损 → ACh直接作用于平滑肌M受体或暴露α受体 → **血管收缩**
6. **输出结论**：根据内皮状态，明确ACh引起的血管反应方向（舒张或收缩）。

> **注意**：此机制严格依赖内皮完整性。内皮损伤是导致ACh效应反转的关键条件。
