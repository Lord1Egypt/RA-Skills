## P2-内容生成域范本

### P2-01 广告海报生成范本

**任务目标**：生成产品广告、品牌广告、促销广告海报

**执行步骤**：

1. **确定广告主题**
   - 产品特点：突出产品核心优势
   - 品牌理念：传达品牌价值观
   - 促销信息：突出优惠力度

2. **创作广告文案**
   - 标题：简短有力、吸引注意力
   - 正文：突出卖点、有说服力
   - 口号：易记、有号召力

3. **选择广告图片**
   - 产品图片：高质量、展示产品细节
   - 场景图片：与产品相关、有代入感
   - 背景图片：与主题一致、有视觉冲击力

4. **设计广告布局**
   - 主次分明：标题突出、正文清晰
   - 层次清晰：引导视觉流程
   - 呼吁行动：明确的行动号召

**输出示例**：

```markdown
## 广告海报设计方案

### 广告主题
- 主题：产品核心优势
- 目标：吸引目标用户关注

### 广告文案
- 标题：这款产品真的太好用了！
- 正文：采用先进技术，让你的生活更简单
- 口号：立即体验，改变你的生活

### 广告图片
- 产品图片：高清产品展示图
- 场景图片：用户使用场景图
- 背景图片：简约渐变背景

### 广告布局
- 标题位置：顶部居中，大字体
- 产品图片：中央突出展示
- 正文位置：产品图片下方
- 行动号召：底部居中，按钮样式
```

### P2-02 宣传海报生成范本

**任务目标**：生成活动宣传、品牌宣传、产品宣传海报

**执行步骤**：

1. **确定宣传主题**
   - 活动宣传：突出活动亮点
   - 品牌宣传：传达品牌形象
   - 产品宣传：展示产品特点

2. **编写宣传内容**
   - 主标题：吸引眼球、引发兴趣
   - 副标题：补充说明、增强说服力
   - 详细内容：清晰、有条理

3. **选择宣传图片**
   - 主图片：与主题相关、有视觉冲击力
   - 辅助图片：补充说明、增强效果
   - 品牌元素：logo、品牌色、品牌字体

4. **设计宣传布局**
   - 视觉焦点：突出主题
   - 信息层次：主次分明
   - 品牌一致性：保持品牌形象

**输出示例**：

```markdown
## 宣传海报设计方案

### 宣传主题
- 主题：品牌宣传活动
- 目标：提升品牌知名度

### 宣传内容
- 主标题：探索无限可能
- 副标题：与我们一起开启新征程
- 详细内容：品牌介绍、活动详情

### 宣传图片
- 主图片：品牌logo+产品展示
- 辅助图片：用户使用场景
- 品牌元素：品牌色、品牌字体

### 宣传布局
- 视觉焦点：品牌logo居中突出
- 信息层次：标题→副标题→详细内容
- 品牌一致性：统一品牌色和字体
```

### P2-03 活动海报生成范本

**任务目标**：生成会议、展览、演出、赛事活动海报

**执行步骤**：

1. **确定活动信息**
   - 活动名称：简洁、有吸引力
   - 活动时间：具体日期和时间
   - 活动地点：详细地址
   - 活动内容：活动亮点和流程

2. **设计活动主题**
   - 主视觉：与活动主题相关
   - 色彩搭配：符合活动氛围
   - 字体选择：与活动风格一致

3. **整理活动信息**
   - 时间信息：突出显示
   - 地点信息：清晰易读
   - 报名信息：明确指引

4. **设计活动布局**
   - 信息层次：主次分明
   - 视觉引导：引导阅读顺序
   - 行动号召：明确报名方式

**输出示例**：

```markdown
## 活动海报设计方案

### 活动信息
- 活动名称：2026年产品发布会
- 活动时间：2026年7月15日 14:00
- 活动地点：北京国际会议中心
- 活动内容：新品发布、技术分享、互动体验

### 活动主题
- 主视觉：产品发布会主题海报
- 色彩搭配：科技蓝+白色
- 字体选择：现代简约字体

### 活动信息整理
- 时间信息：大字体突出显示
- 地点信息：清晰易读
- 报名信息：二维码+网址

### 活动布局
- 信息层次：活动名称→时间→地点→内容
- 视觉引导：从上到下，从左到右
- 行动号召：立即报名，扫码参与
```

### P2-04 产品海报生成范本

**任务目标**：生成产品介绍、产品特点、产品优势海报

**执行步骤**：

1. **整理产品信息**
   - 产品名称：简洁、易记
   - 产品特点：核心优势
   - 产品价格：如有促销信息
   - 产品使用场景：展示产品价值

2. **设计产品展示**
   - 产品图片：高质量、多角度
   - 产品细节：突出产品特点
   - 产品使用场景：展示产品价值

3. **提炼产品卖点**
   - 核心卖点：最突出的优势
   - 辅助卖点：补充说明
   - 用户评价：增强说服力

4. **设计产品布局**
   - 产品突出：产品图片为核心
   - 信息清晰：卖点一目了然
   - 购买引导：明确购买方式

**输出示例**：

```markdown
## 产品海报设计方案

### 产品信息
- 产品名称：智能手表Pro
- 产品特点：健康监测、运动追踪、智能提醒
- 产品价格：¥1299
- 使用场景：日常佩戴、运动健身、商务办公

### 产品展示
- 产品图片：高清产品图+佩戴效果图
- 产品细节：表盘、表带、功能界面
- 使用场景：不同场景下的使用效果

### 产品卖点
- 核心卖点：24小时健康监测
- 辅助卖点：7天超长续航、50米防水
- 用户评价：用户好评截图

### 产品布局
- 产品突出：产品图片居中放大
- 信息清晰：卖点分点展示
- 购买引导：立即购买按钮+二维码
```
