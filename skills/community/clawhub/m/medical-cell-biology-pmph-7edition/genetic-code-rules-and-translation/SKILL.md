# 遗传密码规则与蛋白质翻译

## 何时使用
- 将mRNA编码区序列翻译为氨基酸序列
- 识别起始密码子或终止密码子
- 分析遗传密码的简并性或通用性
- 验证翻译过程是否符合方向性和连续性

## 执行步骤
1. 确认输入为mRNA的编码区（排除5'和3'非翻译区）。
2. 从5'端开始，按每3个连续核苷酸划分为一个密码子。
3. 应用标准遗传密码表进行翻译：
   - 起始密码子通常为AUG（编码甲硫氨酸）
   - 终止密码子为UAA、UAG或UGA，标志翻译终止
   - 其余61个密码子对应20种氨基酸
4. 按5'→3'方向连续读取，直至遇到终止密码子。
5. 输出对应的氨基酸序列或终止信号。

> 注意：线粒体和叶绿体中的遗传密码存在例外，本技能默认使用通用密码表。
