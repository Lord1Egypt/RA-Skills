# 标题规则

## 不可违反

1. 不得自拟标题。
2. 不得改写原标题的核心表述。
3. 不得为了好看擅自截短标题。

## 三条规则

- **中文标题且与内容相符**：编译文档标题 = 原标题
- **英文标题**：编译文档标题 = 英文原标题 + 中文副标题
- **中文标题但与内容不符**：编译文档标题 = 原标题 + 中文副标题

## 反例

- 把原标题概括成一个新名字
- 去掉原标题中的限定词、范围、时间
- 只保留原标题前半句

## 检查点

- 中转站文件名 = H1 标题
- `compile_check.sh` 必须能验证 H1 与文件名一致
- 如果触发副标题，副标题只补充解释，不替换原标题
