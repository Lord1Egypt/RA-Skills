# 运动负荷试验阳性判断

## 适用场景
- 疑似冠心病患者的筛查
- 不明原因胸痛的鉴别诊断
- 冠心病治疗后的疗效评估

## 前提条件
- 已完成规范的平板或踏车运动负荷试验
- 患者达到亚极量或极量心率（通常为目标心率的85%~90%）
- 同步记录12导联心电图

## 执行步骤
1. **检查是否满足任一阳性标准**：
   - 出现典型心绞痛症状
   - ST段呈水平型或下斜型下移 ≥ 0.1 mV（1 mm），且持续 ≥ 2分钟
     - 若基线已有ST段下移，则需在原有基础上再下移 ≥ 0.1 mV
   - 运动中收缩压下降 >10 mmHg
   - 特殊情况：运动中ST段抬高 ≥ 0.1 mV（需结合基线心电图判断意义）

2. **若未满足阳性标准，检查是否符合阴性标准**：
   - 已达目标心率
   - 心电图无ST段下移，或下移幅度 < 0.1 mV

3. **综合判断注意事项**：
   - 阳性结果不能单独确诊冠心病，必须结合临床症状、病史及其他检查
   - 注意假阳性（如女性、服用β受体阻滞剂等）和假阴性（如运动量不足、单支病变）的可能

## 输出格式
返回“阳性”或“阴性”，并附具体依据，包括：
- ST段变化的导联、幅度、形态及持续时间
- 是否伴随典型心绞痛
- 血压变化情况
- 特殊心电图表现（如ST段抬高）及其临床提示

> **注意**：本技能不适用于预激综合征、左束支传导阻滞或起搏心律患者。
