## D1 设计规范域范本

### D1-01 颜色系统应用范本

**任务目标**：选择合适的色系和层级，应用到文档

**执行步骤**：

1. **分析文档类型**
   - 正式文档：使用蓝色、灰色系
   - 非正式文档：使用绿色、橙色系
   - 学术文档：使用蓝色、灰色系

2. **选择色系**
   - 主色调：blue-800 (#0C447C)
   - 辅助色：gray-600 (#5F5E5A)
   - 强调色：blue-600 (#185FA5)

3. **确定层级**
   - 标题：800层级
   - 副标题：600层级
   - 正文：900层级

4. **应用颜色**
   - 标题颜色：blue-800
   - 副标题颜色：blue-600
   - 正文颜色：gray-900

**输出示例**：

```markdown
## 颜色应用方案

### 色系选择
- 主色调：blue
- 辅助色：gray
- 强调色：blue

### 层级选择
- 标题：800层级
- 副标题：600层级
- 正文：900层级

### 颜色应用
- 标题颜色：#0C447C (blue-800)
- 副标题颜色：#185FA5 (blue-600)
- 正文颜色：#2C2C2A (gray-900)
- 强调内容：#185FA5 (blue-600)
```

### D1-02 字体系统应用范本

**任务目标**：应用标题/正文/辅助文字规范

**执行步骤**：

1. **确定字体**
   - 标题字体：黑体
   - 正文字体：宋体
   - 辅助文字：宋体

2. **设置大小**
   - 标题1：24px (小二号)
   - 标题2：18px (小三号)
   - 标题3：14px (四号)
   - 正文：12px (小四号)
   - 辅助文字：10px (五号)

3. **设置字重**
   - 标题：500 (粗体)
   - 正文：400 (常规)

4. **设置行高**
   - 标题：1.4倍
   - 正文：1.6倍

**输出示例**：

```markdown
## 字体应用方案

### 字体选择
- 标题字体：黑体
- 正文字体：宋体
- 辅助文字：宋体

### 大小设置
- 标题1：24px (小二号)
- 标题2：18px (小三号)
- 标题3：14px (四号)
- 正文：12px (小四号)
- 辅助文字：10px (五号)

### 字重设置
- 标题：500 (粗体)
- 正文：400 (常规)

### 行高设置
- 标题：1.4倍
- 正文：1.6倍
```

### D1-03 布局规则应用范本

**任务目标**：应用页边距、段落间距、行距规范

**执行步骤**：

1. **设置页面尺寸**
   - 纸张大小：A4 (210mm × 297mm)
   - 方向：纵向

2. **设置页边距**
   - 上边距：2.54cm
   - 下边距：2.54cm
   - 左边距：3.17cm
   - 右边距：3.17cm

3. **设置段落间距**
   - 段前：0.5行
   - 段后：0.5行

4. **设置行距**
   - 行距：1.5倍

5. **设置页眉页脚**
   - 页眉距离：1.5cm
   - 页脚距离：1.75cm

**输出示例**：

```markdown
## 布局方案

### 页面尺寸
- 纸张大小：A4 (210mm × 297mm)
- 方向：纵向

### 页边距
- 上边距：2.54cm
- 下边距：2.54cm
- 左边距：3.17cm
- 右边距：3.17cm

### 段落间距
- 段前：0.5行
- 段后：0.5行

### 行距
- 行距：1.5倍

### 页眉页脚
- 页眉距离：1.5cm
- 页脚距离：1.75cm
```

### D1-04 样式系统应用范本

**任务目标**：应用预设样式（标题/正文/列表/表格/图片等）

**执行步骤**：

1. **确定样式类型**
   - 标题样式：用于章节标题
   - 正文样式：用于正文内容
   - 列表样式：用于列表内容
   - 表格样式：用于表格内容
   - 图片样式：用于图片说明

2. **应用样式类**
   - 标题1：黑体，24px，居中
   - 标题2：黑体，18px，左对齐
   - 标题3：黑体，14px，左对齐
   - 正文：宋体，12px，两端对齐
   - 列表：宋体，12px，左对齐
   - 表格：宋体，10px，居中
   - 图片：宋体，10px，居中

3. **自定义样式补充**
   - 根据需要调整样式
   - 保持一致性

**输出示例**：

```markdown
## 组件样式方案

### 样式类型
- 标题样式：用于章节标题
- 正文样式：用于正文内容
- 列表样式：用于列表内容
- 表格样式：用于表格内容
- 图片样式：用于图片说明

### 样式类应用
- 标题1：黑体，24px，居中，填充50层级
- 标题2：黑体，18px，左对齐，填充100层级
- 标题3：黑体，14px，左对齐，填充200层级
- 正文：宋体，12px，两端对齐，首行缩进2字符
- 列表：宋体，12px，左对齐，项目符号或编号
- 表格：宋体，10px，居中，边框1像素
- 图片：宋体，10px，居中，图下方

### 自定义样式
- 无自定义样式，使用标准样式
```
