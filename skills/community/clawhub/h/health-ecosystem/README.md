# 健康闭环系统

基于华为运动健康的智能个人健康管理方案。

## 快速开始

1. 安装依赖技能：
```bash
npx clawhub install xiaoyi-health
npx clawhub install obsidian-git-vault
```

2. 初始化系统：
对我说「初始化健康闭环系统」

3. 开始使用：
- 「我今天身体状态怎么样？」
- 「这周健康情况怎么样？」
- 「帮我设置每天走 8000 步的目标」
