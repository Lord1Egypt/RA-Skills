## 1. 行业概览

### 1.3 竞争格局特征

- **国货 vs 国际**：国货在 200-500 价格带强势，国际品牌在 800+ 高端线守稳
- **成分透明化**：用户主动研究成分，"烟酰胺浓度""视黄醇工艺"成为内容题材
- **KOL 即渠道**：李佳琦 / 董宇辉等头部主播对 GMV 影响巨大
- **私域+复购**：高复购品类，私域沉淀价值高

## 2. 消费者决策路径

### 2.1 典型决策周期

| 品类 | 决策周期 | 关键特征 |
|------|:-------:|---------|
| 护肤（高端线） | 2-4 周 | 多方比价、看测评 |
| 护肤（大众线） | 3-7 天 | 易冲动消费 |
| 彩妆 | 1-3 天 | 视觉驱动、KOL 种草即下单 |
| 香水 | 1-2 周 | 试香+口碑 |
| 美容仪器 | 1-3 个月 | 高客单、深度比较 |

### 2.2 信息渠道权重

| 渠道 | 种草权重 | 转化权重 | 适合内容 |
|------|:------:|:------:|---------|
| 小红书 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | 成分测评、教程、Before/After |
| 抖音 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 30 秒挑战、剧情植入、直播间 |
| 微博 | ⭐⭐ | ⭐ | 话题炒作、明星同款 |
| B站 | ⭐⭐⭐⭐ | ⭐⭐ | 深度测评、成分解析（中长视频） |
| 微信私域 | ⭐⭐ | ⭐⭐⭐⭐⭐ | 老客复购、新品内购 |
| 知乎 | ⭐⭐⭐ | ⭐ | 成分科普、专业回答 |

### 2.3 典型转化漏斗

```
曝光 → 点击: 2-4%（小红书）/ 1.5-3%（抖音）
点击 → 加购: 8-15%
加购 → 下单: 70-85%
下单 → 支付: 80-90%
整体：曝光 → 支付: 1-3%
```

## 5. 合规与监管

### 5.1 广告法专属限制

| 限制类型 | 禁用/限制 | 罚则 |
|---------|---------|------|
| 极限词 | "最""第一""国家级""顶级""极致""永久""彻底" | 20-100 万 |
| 绝对化承诺 | "100% 有效""永不脱妆""祛斑立竿见影" | 20-100 万 |
| 医疗用语 | "治疗""治愈""处方""药用"（非特殊化妆品） | 撤下 + 罚款 |
| 比较广告 | 不能直接贬低竞品 | 民事赔偿 |

### 5.2 功效宣称边界（**重点**）

| 化妆品类别 | 宣称边界 | 必备资质 |
|----------|---------|---------|
| 普通化妆品 | 清洁、保湿、香氛、修饰 | 备案 |
| 祛斑美白 | 必须为"特殊化妆品" | 注册证（"国妆特字"） |
| 防晒 | 必须为"特殊化妆品" | 注册证 + SPF/PA 测试 |
| 防脱发 | 必须为"特殊化妆品" | 注册证 |
| 染发/烫发 | 必须为"特殊化妆品" | 注册证 |
| 新原料 | 3 年监测期 | 单独备案 |

> ⚠️ **AI 方案禁用**：不得为非特殊化妆品设计"美白""祛斑""防脱""抗衰"等宣称。

### 5.3 平台审核红线

| 平台 | 重点红线 |
|------|---------|
| 抖音 | "Before/After"对比过度、医疗美容暗示、低俗擦边 |
| 小红书 | 软广未声明、夸大功效、医美项目推荐 |
| 微信 | 朋友圈过度营销、诱导分享、多级分销 |
| 微博 | 数据造假、营销号炒作、负面舆情压制 |

### 5.4 行业特殊资质

| 业务类型 | 必备资质 |
|---------|---------|
| 自营电商 | 营业执照 + 化妆品经营许可证 |
| 进口化妆品 | 海关报关 + 中文标签备案 |
| KOL 推广 | 广告代言资质（部分品类） |

## 6. 典型 Campaign 节奏

### 6.1 新品上市（90 天）

```
T-30 ~ T-15：种草期
- 小红书 KOC 铺量 100-300 篇
- 抖音达人测评 5-15 位
- 内容形式：成分解析、教程、对比测评

T-14 ~ T-7：蓄水期
- 头部 KOL 集中发声（小红书 + 抖音）
- 抖音话题挑战赛
- 微博明星同款话题

T-7 ~ T0：爆发期
- 李佳琦/头部主播直播首发
- 抖音信息流 + 千川放大
- 天猫小黑盒 / 超级品牌日

T+1 ~ T+30：长尾期
- UGC 激励（晒单返现/抽奖）
- 私域沉淀（包裹卡 + 企微）
- 复购推送（28 天周期）
```

```
T-30：策略定调 + Brief
T-21：素材制作 + KOL 预约

T-3：流量冲刺 + 直播预告
T0：爆发 + 全员直播
```

## 7. 常见雷区（Adversarial 攻击参考）

| 雷区 | 典型表现 | 严重度 | 规避方法 |
|------|---------|:-----:|---------|
| 功效宣称越界 | 普通化妆品宣"美白""抗衰" | 🔴 致命 | 必须查验注册证 |
| Before/After 过度 | 美妆仪器过度 P 图 | 🔴 高 | 标注"效果因人而异" |
| KOL 翻车 | 头部达人负面舆情 | 🔴 高 | 备选 + 解约条款 |
| 成分虚假宣传 | 浓度虚标、工艺造假 | 🔴 致命 | 第三方检测报告 |
| 季节性错配 | 夏季推滋润型 / 冬季推清爽型 | 🟡 中 | 节气日历检查 |
| 同质化创意 | "成分党"已成套路 | 🟡 中 | 引用 creative-diversity-protocol.md |
| 私域过度营销 | 朋友圈刷屏被屏蔽 | 🟡 中 | 频次控制 + 内容轮替 |

## 8. 推荐方法论组合

| 方法论 | 适用场景 | 备注 |
|--------|---------|------|
| **STP + 品牌原型** | 品牌定位 | 用原型（魔术师/智者/情人）匹配品类调性 |
| **AISAS** | 消费者旅程 | 美妆典型决策路径 |
| **AIPL** | 阿里系运营 | 适用于天猫旗舰店 |
| **Binet & Field 60:40** | 长期品牌 vs 短期效果 | 高复购品类适用 |
| **FAST / GROW** | 阿里人群资产 | 适用于大促 |
| **KOL 金字塔** | 达人矩阵 | 头部+腰部+KOC 标配 |

**不推荐**：
- 单纯的 AIDA 漏斗（过于线性，不符合美妆"种草-拔草-复种草"循环）
- 价格战策略（损害品牌资产，长期不可持续）
