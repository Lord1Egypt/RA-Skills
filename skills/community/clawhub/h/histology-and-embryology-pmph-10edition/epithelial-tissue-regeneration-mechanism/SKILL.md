# 上皮组织更新与再生机制

## 触发条件
当出现以下情况之一时应用本技能：
- 上皮细胞因自然衰老而脱落
- 发生创伤、烧伤或炎症导致上皮损伤

## 执行步骤

### 1. 判断再生类型
- **生理性再生（日常更新）**：用于维持组织稳态，由上皮内未分化干细胞持续增殖补充。
- **病理性再生（损伤修复）**：由周围或深层存活的上皮细胞增殖迁移以覆盖创面。

### 2. 确定适用对象及更新周期
根据目标上皮类型参考典型更新时间：
- 小肠绒毛上皮：2～6天
- 表皮：1～2个月
- 胰腺上皮：约50天

### 3. 评估再生可行性
检查以下关键条件：
- 是否存在未分化的干细胞储备
- 基底膜是否完整（严重破坏将阻碍再生）
- 损伤是否处于正常或可修复的炎症/创伤状态

### 4. 预测修复过程
- **皮肤表皮损伤**：伤口边缘基底细胞或附属腺导管上皮细胞分裂 → 向裸露区迁移形成单层扁平覆盖 → 逐步恢复多层结构。
- **消化管上皮损伤**：邻近正常上皮或腺体颈部细胞增殖 → 初为立方形细胞 → 成熟为单层柱状上皮。

## 输出
基于上述分析，指导组织修复评估或提出再生医学干预策略（如干细胞移植、基底膜保护等）。
