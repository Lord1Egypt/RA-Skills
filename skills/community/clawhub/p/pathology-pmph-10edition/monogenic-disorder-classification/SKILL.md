# 单基因遗传病的分类与代谢缺陷分型

## 何时使用
- 疾病已确认由单一核基因突变引起（排除线粒体基因病、多基因遗传病及染色体病）
- 需要根据致病蛋白类型对疾病进行机制性归类
- 用于临床诊断辅助、遗传咨询或医学教学场景

## 执行步骤
1. **确认适用前提**：验证疾病是否满足以下条件：
   - 由单一核基因突变导致
   - 引起蛋白质结构或数量异常
   - 不属于多基因遗传病或染色体病

2. **判断受影响蛋白质是否为酶**：
   - **若是酶蛋白异常** → 归类为“先天性代谢缺陷”（即遗传性酶病），并按代谢通路进一步细分：
     - 糖代谢缺陷（如糖原贮积症、半乳糖血症）
     - 脂类代谢缺陷（如戈谢病）
     - 氨基酸代谢缺陷（如苯丙酮尿症）
     - 核酸、内分泌、溶酶体、药物或维生素代谢缺陷
   - **若为非酶类蛋白质异常** → 归类为“分子病”，包括：
     - 血红蛋白病（如地中海贫血）
     - 结构蛋白缺陷病（如肌营养不良、成骨不全）
     - 受体病（如家族性高胆固醇血症）
     - 膜转运蛋白病、血浆蛋白病、免疫球蛋白缺陷病等

3. **处理复合机制情况**：若疾病同时涉及代谢与结构异常，依据**主要致病机制**进行归类。

4. **输出结果**：返回疾病所属主类别（先天性代谢缺陷 或 分子病）及其具体子类。
