## Known Risks and Mitigations: <br>
Risk: The skill tells the agent to automatically retain user preferences, decisions, and other important information locally across sessions without clear consent or deletion controls. <br>
Mitigation: Install only when local retention is acceptable, avoid sharing sensitive information, and inspect or delete the local memory file when retention is no longer wanted. <br>
