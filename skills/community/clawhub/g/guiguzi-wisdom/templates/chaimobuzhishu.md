# 🧠 揣摩步步之术 | Mind-Reading Step-by-Step Template

> 系统地揣摩对方的心理状态和真实意图——不是猜测，是推理

---

## 第一步：量权 —— 评估力量对比

### 力量对比表

| 维度 | 对方 | 我方 | 差距分析 |
|:---|:---|:---|:---|
| **信息**（谁知道更多？） | | | |
| **资源**（谁掌握更多资源？） | | | |
| **时间**（谁的时间更紧迫？） | | | |
| **替代方案**（谁的 Plan B 更好？） | | | |
| **关系网络**（谁的人脉更强？） | | | |

> **核心判断**：在哪些维度上你占优势？在哪些维度上你处于劣势？不要在劣势维度上发起攻势。

---

## 第二步：揣情 —— 揣摩情绪与欲望

### 情绪观察清单

观察对方在这三种状态下的表现：

| 状态 | 观察点 | 对方表现 | 推测 |
|:---|:---|:---|:---|
| **甚喜之时** | 他最得意什么？什么让他膨胀？ | | |
| **甚惧之时** | 他最害怕什么？什么让他收缩？ | | |
| **甚怒之时** | 什么触怒了他？愤怒的根源是什么？ | | |

> **鬼谷子原文**："揣情者，必以其甚喜之时，往而极其欲也；其有惧也，不能隐其情。"

### 需求层次分析

对方当前处于哪个需求层次？

- [ ] **生存/安全**：保住工作？不被淘汰？
- [ ] **归属/尊重**：融入团队？获得认可？
- [ ] **自我实现**：做有意义的事？留下 legacy？
- [ ] **权力/控制**：扩大影响力？掌握决策权？

---

## 第三步：摩意 —— 试探与验证

### 摩意十法速查

每次选择 1-2 种试探方法：

| 方法 | 怎么做 | 适合场景 | 不适合场景 |
|:---|:---|:---|:---|
| **平摩** | 平等提问，不表露态度 | 初次接触 | 时间紧迫 |
| **正摩** | 直接询问意图 | 关系信任好 | 关系微妙 |
| **喜摩** | 分享好消息 | 测试对方是否真心 | 对方处于低谷 |
| **怒摩** | 轻微激发情绪 | 测试底线 | 正式场合 |
| **名摩** | 提及声誉/品牌 | 看价值取向 | 对方务实/反权威 |
| **行摩** | 观察行为模式 | 长期观察 | 一次性交互 |
| **廉摩** | 涉及利益分配 | 测试公平感 | 涉及大额利益 |
| **信摩** | 分享小秘密 | 测试保密性 | 高风险信息 |
| **利摩** | 提出利益方案 | 测试欲望程度 | 已明确表态 |
| **卑摩** | 放低姿态请教 | 测试自负程度 | 需展示权威 |

---

## 第四步：谋虑 —— 基于揣摩的策略选择

| 揣摩结果 | 策略方向 |
|:---|:---|
| 欲望强 + 恐惧弱 | 利益诱导（飞箝），大胆进攻 |
| 欲望弱 + 恐惧强 | 安全感安抚（内揵），降低决策门槛 |
| 欲望强 + 恐惧强 | 双管齐下（捭阖并用），利益+消除顾虑 |
| 欲望弱 + 恐惧弱 | 重新框定问题，激发需求（反应术） |
| 不可了解 | 停止揣摩，考虑退出（忤合） |

---

## 第五步：变通 —— 信号与调整

| 对方信号 | 调整策略 |
|:---|:---|
| 开始主动提问 | 揣摩成功 → 释放更多信息 |
| 话变少了 | 可能触到敏感点 → 切换话题 |
| 频繁看时间/手机 | 兴趣下降 → 缩短沟通，尽快收尾 |
| 提出反建议 | 对方在主动参与 → 顺势深入 |
| 沉默但专注 | 在思考 → 不要打断，给空间 |
