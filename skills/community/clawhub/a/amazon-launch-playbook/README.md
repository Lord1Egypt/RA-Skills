# Amazon Launch Playbook

**亚马逊新品上架实战手册 — 从0到爆款的90天行动方案**

---

## What It Does

Amazon Launch Playbook 是一款面向中国亚马逊卖家的新品上架策略工具。它基于用户提供的品类和产品描述，帮助生成完整的90天上架实战方案，涵盖选品决策、Listing基建、物流库存、广告推广和数据优化的全流程。

**核心价值**：
- 提供经过验证的上架框架和方法论
- 避免新品期的常见错误和资源浪费
- 帮助卖家建立系统化的运营思维

---

## Who It's For

- **新手卖家**：第一次在亚马逊上架新品，不了解完整流程
- **转型卖家**：从其他平台转到亚马逊，需要针对性指导
- **迷茫卖家**：有产品但上架后销量不佳，缺乏系统方法
- **效率导向卖家**：希望快速获得可执行的上架方案

**前提条件**：
- 已完成选品，有明确的产品方向
- 准备或已在亚马逊开店
- 愿意投入时间和资源进行新品推广

---

## Quick Start

1. **提供基本信息**：
   - 产品品类（如：家居收纳、运动户外、美妆工具）
   - 产品一句话描述（产品是什么、解决什么问题）

2. **获得完整方案**：
   - 90天分阶段行动计划
   - Listing优化指南
   - 广告推广框架
   - 数据关注模板

3. **参考配套资源**：
   - 上架检查清单（references/launch-checklist.md）
   - 广告启动框架（references/ad-launch-framework.md）
   - KPI关注看板（references/kpi-dashboard.md）
