# 推品执行手册 (launch_execution.md)

## 概述

本文件定义推品执行（Step 10）的输出格式和模板。选品、找人、寄样都做完了，推品当天最怕手忙脚乱。本节提供推品日的完整作战时间表和执行清单。

---

## A. 最佳发布时间策略

### 类目 × 时间矩阵

不同类目在不同平台的最佳发布窗口不同：

#### 抖音最佳发布时间

| 类目 | 最佳日期 | 最佳时段 | 理由 |
|------|---------|---------|------|
| 美妆护肤 | 周五/周六 | 19:00-21:00 | 周末前种草，周末消费 |
| 食品零食 | 周三/周四 | 12:00-14:00 / 20:00-22:00 | 午餐和晚间零食高峰 |
| 母婴用品 | 周一/周二 | 10:00-12:00 / 14:00-16:00 | 宝妈孩子上学后的碎片时间 |
| 家居日用 | 周六/周日 | 10:00-12:00 / 15:00-17:00 | 周末居家场景消费 |
| 数码3C | 周四/周五 | 20:00-22:00 | 周末前决策型消费 |
| 服装鞋包 | 周四/周五/周六 | 19:00-22:00 | 晚间浏览高峰期 |
| 保健品 | 周一/周三 | 10:00-12:00 | 养生类内容上午流量好 |
| 宠物用品 | 全天 | 12:00-14:00 / 19:00-21:00 | 宠物内容曝光较平均 |

#### 快手最佳发布时间

快手的推荐分布更偏下沉市场，时间策略略有不同：

| 类目 | 最佳日期 | 最佳时段 | 理由 |
|------|---------|---------|------|
| 全品类通用 | 工作日 | 12:00-13:00 / 18:00-21:00 | 快手用户午休和下班后集中刷 |
| 白牌/清仓品 | 周六/周日 | 10:00-12:00 | 周末冲动消费转化好 |
| 品牌品 | 周四/周五 | 19:00-21:00 | 和抖音同步推 |
| 食品 | 周三 | 20:00-22:00 | 快手食品带货周三流量峰值 |

### 一周发布节奏建议

```
周一：确认本周推品清单 + 预热素材准备
周二：达人沟通 + 确认推品时间
周三：第一波推品（食品/日用等快消品）
周四：持续推品 + 社群预热
周五：第二波推品（美妆/数码等决策型品类）
周六：社群追单 + 朋友圈战报
周日：数据汇总 + 下周推品预告
```

## B. 发布渠道优先级排序

### 输出格式

基于商品特征和策略方向，生成渠道优先级排序表：

```
┌────┬──────────┬──────────┬──────────┬────────────────────┐
│ 优先级│ 渠道    │ 内容形式  │ 首发/转发│ 理由               │
├────┼──────────┼──────────┼──────────┼────────────────────┤
│ 1  │ 抖音短视频│ 种草视频  │ 首发     │ 流量最大，曝光覆盖广 │
│ 2  │ 社群/团长群│ 图文+链接 │ 首发     │ 精准转化，即时下单   │
│ 3  │ 快手短视频│ 种草视频  │ 转发     │ 抖音内容复用，效率高 │
│ 4  │ 朋友圈   │ 战报+图   │ 转发     │ 信任背书，长尾转化   │
│ 5  │ 小红书   │ 图文笔记  │ 首发     │ 搜索流量，长尾种草   │
└────┴──────────┴──────────┴──────────┴────────────────────┘
```

### 渠道选择决策树

```
佣金率高 + 白牌 → 优先社群/团长群，先做精准转化
品牌力强 + 佣金低 → 优先短视频种草，长线品牌建设
新品类 + 需教育 → 优先知识型短视频 + 小红书
客单价低 + 冲动型 → 优先直播带货 + 促销社群
客单价高 + 决策型 → 优先测评短视频 + 社群深度种草
```

## C. 直播脚本大纲模板

### 结构框架（5-8分钟单品讲解）

使用"钩子-痛点-展示-价格-逼单"五段式结构：

```
┌──────────────────────────────────────────────────────────────┐
│ 阶段           │ 时长   │ 内容             │ 情绪曲线          │
├──────────────────────────────────────────────────────────────┤
│ ① 开场钩子      │ 0:30  │ 制造好奇/痛点共鸣  │ 🔥 兴奋/好奇      │
│ ② 痛点引入      │ 1:00  │ 描述用户痛点       │ 😣 共情           │
│ ③ 产品展示      │ 2:30  │ 产品+效果+成分     │ 😲 惊讶/信服      │
│ ④ 价格锚定      │ 1:00  │ 对比+福利+限时     │ 🤩 惊喜           │
│ ⑤ 逼单收尾      │ 1:00  │ 催促下单+提醒福利   │ ⏰ 紧迫           │
├──────────────────────────────────────────────────────────────┤
│ 总计            │ 6:00  │                    │                  │
└──────────────────────────────────────────────────────────────┘
```

### 各阶段脚本要点

#### ① 开场钩子（30秒）

```
模板类型（选一）：

A. 悬念型："今天这个洗发水，你用完会回来感谢我的"
B. 痛点型："有没有姐妹跟我一样，洗完头第二天就油到不行？"
C. 对比型："左边是XX洗发水洗的，右边是这个洗的——你们自己看"
D. 数据型："我测了30天，头皮油脂分泌降了60%——就是这个"
E. 反常识型："99%的人洗头方法都是错的，先别急着反驳，看下去"
```

#### ② 痛点引入（1分钟）

```
结构：
1. 描述用户的真实痛点（1-2句）
2. 分享自己/用户的经历（1-2句）
3. 引出"问题出在洗发水上"（1句）

示例：
"有没有姐妹跟我一样，洗完头第一天蓬松，第二天就贴头皮的？
我用了三年各种控油洗发水，越洗越油，头皮还开始长小疙瘩
后来一个配方师朋友告诉我：问题不在你，在你的洗发水成分"
```

#### ③ 产品展示（2分30秒）

```
结构：
- 外观展示（15秒）：产品包装、质地特写
- 成分解析（45秒）：核心成分+技术亮点，只说人话
- 使用演示（45秒）：泡沫质地、使用过程、洗干净后的对比
- 效果对比（45秒）：Before/After对比，真实不做作

关键技巧：
- "你可以闻到"——多感官描述
- "你看这个泡沫"——引导视线
- "你感受一下"——邀请想象
```

#### ④ 价格锚定（1分钟）

```
结构：
1. 先对比同类竞品价格（往高说）"市面上氨基酸洗发水基本129起步"
2. 再报本品价格（往低说）"今天直播间只要79"
3. 加福利（赠品/满减/限时）"还送价值39的旅行装"
4. 算账（把价格拆到每天）"500ml用3个月，一天不到9毛钱"

锚定技巧：
- 先对比高端品牌（200+）→ 显得本品性价比高
- 再对比超市款（20-30）→ 拉开品质差距
- 锁定本品（79）→ 既不太贵也不太便宜，刚好在"试一下"区间
```

#### ⑤ 逼单收尾（1分钟）

```
结构：
1. 限时限量："前100单送旅行装，现在不到20单了"
2. 降低决策门槛："不满意的30天无理由退"
3. 二次提醒福利："下单备注"主播推荐"额外减5块"
4. 营造紧迫："3-2-1，准备上链接！"

注意：
- 不要虚假紧迫感（"最后3单"说了一小时）
- 不要过度施压（让观众觉得被绑架）
- 紧迫感来源于真实限量/限时福利
```

## D. 预热文案模板

### 推品前1天：悬念式预热

```
模板：
"明天给你们看个好东西 👀"
+ [模糊的产品暗示]
+ [为什么值得期待]
+ [行动暗示：点赞/评论预告]

示例：
"明天给你们看个好东西 👀
我最近换了一款洗发水，三天没洗头居然不油
明天视频里告诉你是哪一款，先点赞蹲着"
```

### 推品前1小时：倒计时预热

```
模板：
"还有1小时 [emoji]"
+ [产品亮点一句话]
+ [福利预告]
+ [行动号召]

示例：
"还有1小时！
氨基酸洗发水，洗一次就知道不一样
前100单送旅行装，蹲住的评论区打个1让我看看有多少人"
```

### 推品前10分钟：上线通知

```
模板：
"上了上了！[链接/指路]"
+ [一句话卖点]
+ [福利提醒]

示例：
"上了上了！评论区看链接 👇
79块氨基酸洗发水，买1送1旅行装，限量100"
```
