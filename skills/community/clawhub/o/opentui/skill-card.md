## Description: <br>
OpenTUI is a Zig-native terminal UI framework guide covering installation, renderer setup, core components, declarative constructs, Flexbox layout, and React and Solid bindings. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill when building terminal TUI applications with OpenTUI, including renderer lifecycle, interactive components, layout, and framework bindings. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Installing the referenced Bun packages can add third-party dependencies to a project. <br>
Mitigation: Review the packages and lockfile before adding them to a project, as recommended by the security guidance. <br>
Risk: Framework API guidance may become stale as OpenTUI evolves. <br>
Mitigation: Confirm examples against the intended OpenTUI package version and test generated terminal UI code in a disposable project before deployment. <br>
