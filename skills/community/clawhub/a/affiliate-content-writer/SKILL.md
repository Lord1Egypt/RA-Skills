## Content Formulas for Affiliate Conversion

### 1. Product Review (深度测评)

```markdown
📊 Title Formula:
[品牌型号] 深度测评｜用了[X]天后，说点大实话

📝 Structure:
1. 开头: 为什么买/为什么测 (80-120字)
   "最近[品类]市场很火，[品牌]出了新品，我第一时间入手了..."

2. 开箱/外观 (150-200字)
   - 包装、配件、第一印象
   - 配图：开箱图、产品图

3. 核心体验 (300-400字)
   - 分3-4个维度详细说明
   - 优点+缺点都要说（增加可信度）
   - 配图：使用场景图

4. 横向对比 (200-300字)
   - 和2-3款竞品对比
   - 对比表格：价格、功能、评分
   - 配图：对比表格

5. 购买建议 (100-150字)
   - 适合谁？不适合谁？
   - 什么时候买最划算？
   - [插入京东/淘宝联盟链接] 👈 关键！

6. 结尾互动 (50字)
   "你们买过这款吗？评论区说说感受"
```

### 2. Top N Listicle (好物排行榜)

```markdown
📊 Title Formula:
2026年最值得买的[N]款[品类]，第[X]款我囤了[Y]次

📝 Structure:
1. 开头引子 (80-100字)
   "整理了今年用过最香的[X]款[品类]，从入门到旗舰都有..."

2. 单品推荐 ×5-10 (每个150-200字)
   每个产品格式:
   🏷️ [产品名] - ¥[价格]
   ✅ 优点: [3点，简洁]
   ⚠️ 缺点: [1-2点，增加可信度]
   🎯 适合: [目标人群]
   🔗 [联盟链接] 👈

3. 总结推荐表 (50字)
   一句话总结哪个最好

4. CTA (50字)
   "全部亲测，放心跟买。有问题评论区问👇"
```

### 3. Shopping Guide (选购攻略)

```markdown
📊 Title Formula:
[品类]怎么选？[X]分钟看完这篇就够了

📝 Structure:
1. 痛点引入 (80字)
   "[品类]的水太深了，很多人花冤枉钱..."

2. 选购要点 (300字)
   - 核心参数科普（用通俗语言）
   - 避坑指南
   - 不同预算推荐

3. 分级推荐 (400字)
   💰 入门级 (¥XX - ¥XX): 推荐2-3款
   💰 进阶级 (¥XX - ¥XX): 推荐2-3款
   💰 旗舰级 (¥XX - ¥XX): 推荐2款
   - 每款配图+联盟链接

4. 总结（50字）
```

### 4. Seasonal Guide (节日/季节专题)

```markdown
📊 Examples:
```

## SEO Optimization for Affiliate Content

### Title Keywords
```
高转化标题关键词:
- "测评" "推荐" "值得买" "必入"
- "测评对比" "性价比" "好用吗"
- "怎么选" "选购攻略" "避坑"

搜索量大的词:
- "[品类]推荐" "[品类]测评"
- "[品类]哪个牌子好" "[品类]怎么选"
```

### Content SEO
- First 100 characters must include target keyword
- Use H2/H3 subheadings with keywords
- Internal linking (link to your other articles)
- Image alt text with product names
- Article length: 1500-3000 words optimal

## Affiliate Link Best Practices

```markdown
✅ DO:
- 插入在购买建议和产品推荐部分
- 使用短链接（联盟后台生成）
- 注明是推广链接（合规要求）
- 定期检查链接有效性

❌ DON'T:
- 不要堆砌链接
- 不要虚假宣传
- 不要隐藏推广标识
- 不要复制他人的测评内容
```
