# Localization Plan Quality Checklist

Each "no" is a blocker until resolved.

## Hypothesis
- [ ] Value proposition in this market is one paragraph
- [ ] Target buyer profile is concrete (age, income, channel)
- [ ] Demand signal that triggered consideration is documented
- [ ] Top 3 local competitors named with one strength and one gap each

## Listing surface
- [ ] All copy translated by native speaker (not MT-only)
- [ ] Tone matches market (formality, idiom)
- [ ] Imagery selected/reshot for local cues
- [ ] Sizing/units localized
- [ ] Care labels and ingredient lists localized
- [ ] All certifications stated and verified

## Payments and pricing
- [ ] Top 3 local payment methods integrated
- [ ] Local currency display with local conventions
- [ ] Installment options (where culturally expected) live
- [ ] Test orders placed with each payment method in production

## Logistics and returns
- [ ] Carrier selected and contracted
- [ ] Lead-time visible to buyer
- [ ] Return policy meets legal minimum
- [ ] Return label cost and process defined
- [ ] Customs / duties handled (DDP vs DDU declared to buyer)

## Compliance
- [ ] Country-specific certifications mapped per category
- [ ] Lead time for each cert documented
- [ ] Labeling reviewed by in-market reviewer
- [ ] Packaging-waste registration done where required (e.g. VerpackG, EPR)
- [ ] Restricted ingredients/substances reviewed

## Calendar
- [ ] Next 12 months of local events captured
- [ ] Creative lead time built in for each peak
- [ ] Inventory pre-positioning aligned to peaks
- [ ] Ad-budget aligned to peaks

## Launch sequence
- [ ] Phase 1 SKU list locked
- [ ] Phase 2 trigger criteria defined
- [ ] Phase 3 trigger criteria defined
- [ ] Kill criteria documented per phase

## KPIs
- [ ] Baseline metrics from home market frozen
- [ ] Target metrics per phase set
- [ ] Reporting dashboard live before launch
- [ ] Weekly review owner assigned

## Sign-off
- [ ] In-market reviewer signs off on listing surface
- [ ] Legal signs off on compliance items
- [ ] Finance signs off on phase budgets
- [ ] Customer-care lead briefed on local expectations
