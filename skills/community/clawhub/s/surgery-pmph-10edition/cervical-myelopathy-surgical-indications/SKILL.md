# 脊髓型颈椎病与OPLL手术决策技能

## 适用场景
当满足以下任一条件时启用本技能：
- 患者确诊为脊髓型颈椎病
- 患者确诊或高度怀疑颈椎后纵韧带骨化症（OPLL），年龄50–60岁，有四肢感觉运动障碍或行走不稳
- 影像学（MRI/CT）证实脊髓受压

## 执行步骤

1. **立即执行禁忌措施**
   - **严禁按摩治疗**：任何脊髓型颈椎病患者均禁止按摩，以防瘫痪加重

2. **评估手术适应证**
   - **脊髓型颈椎病**：
     - 神经功能障碍（肌力下降、感觉异常、反射改变）
     - 保守治疗≥6个月无效且影响生活/工作
     - 应尽早手术，避免不可逆脊髓损伤
   - **OPLL**：
     - 有明显脊髓压迫症状（行走不稳、肌张力增高、Hoffmann/Babinski征阳性、大小便障碍）
     - 轻度麻木或疼痛但不影响生活者可非手术治疗

3. **选择手术入路**
   - **前路手术**：
     - 适用于单节段或短节段前方压迫（如局灶型OPLL、单节段脊髓型颈椎病）
     - 方式：椎间盘切除/椎体次全切 + 植骨融合
   - **后路手术**：
     - 适用于多节段后方压迫（如连续型/混合型OPLL、多节段脊髓型颈椎病）
     - 方式：椎板开门成形术（保留活动度）或后路减压+内固定融合
   - **前后路联合**：用于广泛骨化伴颈椎不稳者

4. **输出结果**
   - 明确是否安排手术
   - 推荐手术入路及方式
   - 排除禁忌疗法（如按摩）

> 注意：本技能不适用于神经根型、椎动脉型颈椎病或其他非脊髓压迫性疾病。

## 参考文档

### OPLL影像分型细节

基于CT/X线侧位片的四种类型：
1. **连续型**：骨化影跨越≥2个椎体节段
2. **局灶型**：局限于单个椎间隙后方
3. **间断型**：多个椎节出现不连续骨化灶
4. **混合型**：同时存在两种及以上类型

MRI特征：脊髓受压变扁；CT可见椎体后方致密骨化影，与脑脊液信号不同，可区别于脊髓空洞症。
