## Description: <br>
Generate logos, icons, illustrations and other vector graphics from natural language descriptions with style directives for flat, gradient, stroke, rounded, minimal, and tech designs. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[openlark](https://clawhub.ai/user/openlark) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Designers, developers, and other external users can use this skill to turn natural-language design requests into standalone SVG markup for logos, icons, illustrations, and UI graphics. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated SVG may be embedded in a website or app before review. <br>
Mitigation: Review the generated SVG before use and keep it standalone, especially before adding scripts, external links, fonts, or images. <br>
Risk: Generated vector code may contain syntax, sizing, or color issues that affect rendering. <br>
Mitigation: Validate SVG syntax, viewBox dimensions, tag closure, and color balance before saving or embedding the output. <br>
