# Product Photo Guide Quality Checklist

## Pre-Shoot Planning
- [ ] Competitive audit completed (top 5 listings' shot lists analyzed)
- [ ] Purchase objections mapped to visual solutions (return data, reviews reviewed)
- [ ] Platform-specific requirements noted for all target platforms
- [ ] Shot list created with: shot #, name, description, props, model Y/N, platform target
- [ ] Every top 3 purchase objection has at least one visual solution in the shot list
- [ ] Scale/size reference included in shot list (minimum 2 shots)
- [ ] Infographic briefs written before shoot day (callouts decided in advance)
- [ ] Photography brief shared with photographer at least 3 days before shoot
- [ ] Props and model confirmed

## Hero Image Requirements
- [ ] White background confirmed for Amazon listings (#FFFFFF)
- [ ] Product fills 85%+ of frame in hero shot
- [ ] Minimum resolution target: 2000×2000px
- [ ] No text overlays planned for Amazon hero image
- [ ] No mannequin (or ghost mannequin applied in post) for Amazon apparel
- [ ] Hero shows complete product in primary state (not a detail or crop)

## Shot Coverage
- [ ] Hero shot (clean background, full product)
- [ ] 3/4 or alternative angle shot
- [ ] Scale/size reference shot (hand, ruler, common object)
- [ ] Lifestyle in-use shot
- [ ] Feature callout / infographic (at least 1)
- [ ] Close-up / texture / material detail shot
- [ ] Box contents or flat lay
- [ ] All color/size variants have minimum hero shot
- [ ] Video thumbnail shot (if video planned)

## Infographic Quality
- [ ] Maximum 3 callouts per infographic
- [ ] Numbers used over vague claims ("320 GSM" not "premium quality")
- [ ] Each callout addresses a specific purchase concern or objection
- [ ] Text is readable at 200px thumbnail width (test this)
- [ ] Brand colors and fonts consistent
- [ ] No infographic that merely replicates the bullet points in visual form

## Post-Production
- [ ] All images exported in sRGB color space
- [ ] File sizes within platform limits (Amazon: 10MB; TikTok: 5MB; Shopee: 2MB)
- [ ] File naming convention applied (SKU_COLOR_SHOT#.jpg)
- [ ] Resolution meets platform requirements for each target platform
- [ ] Master files retained at full resolution for future re-export

## Platform Compliance
- [ ] Amazon: white background hero confirmed
- [ ] Amazon: no text/graphics/watermarks on hero image
- [ ] TikTok: first image high-contrast and readable at 200px
- [ ] TikTok: video in 9:16 vertical format
- [ ] Shopee/Lazada: white background first image
- [ ] All images reviewed against platform-specific restriction lists

## Performance Tracking Setup
- [ ] Pre-launch conversion rate baseline recorded
- [ ] Return rate "not as described" baseline recorded
- [ ] A/B test plan for hero image (if platform supports)
- [ ] Review alert set for any new 1-star review mentioning photo expectations
- [ ] 90-day review date scheduled to assess photo performance vs. return rate and CTR
