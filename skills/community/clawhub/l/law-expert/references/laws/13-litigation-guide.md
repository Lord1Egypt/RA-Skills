# 13: 民事诉讼全流程工作指引

## 一、诉讼时效速查表

### 普通诉讼时效
| 时效类型 | 期限 | 起算点 | 法律依据 |
|---------|------|--------|---------|
| 普通诉讼时效 | 3年 | 知道或应当知道权利受损+义务人之日 | 民法典第188条 |
| 最长保护期限 | 20年 | 权利被侵害之日 | 民法典第188条 |

### 特殊诉讼时效
| 纠纷类型 | 期限 | 法律依据 |
|---------|------|---------|
| 国际货物买卖合同 | 4年 | 民法典第594条 |
| 国际技术进出口合同 | 4年 | 民法典第594条 |
| 环境侵权 | 3年 | 民法典相关规定 |
| 产品责任（造成损害） | 2年（自交付起10年） | 产品质量法第45条 |
| 海上货物运输 | 1年 | 海商法第257条 |
| 共同海损分摊 | 1年 | 海商法第263条 |
| 船舶碰撞 | 2年 | 海商法第261条 |
| 航空运输 | 2年 | 民用航空法第135条 |
| 票据权利 | 6个月-2年（视情况） | 票据法第17条 |
| 劳动仲裁时效 | 1年 | 劳动争议调解仲裁法第27条 |

### 诉讼时效中断（重新计算）
- 权利人向义务人提出履行请求
- 义务人同意履行义务
- 权利人提起诉讼或申请仲裁
- 与提起诉讼或申请仲裁具有同等效力的其他情形

### 诉讼时效中止（暂停计算）
- 不可抗力
- 无民事行为能力人/限制民事行为能力人无法定代理人
- 继承开始后未确定继承人/遗产管理人
- 权利人被义务人控制
- 其他导致权利人不能行使请求权的障碍

> 中止事由消除后，无论剩余时效多少，一律补足6个月。

## 二、管辖法院确定规则

### 级别管辖速查
| 法院级别 | 管辖范围 |
|---------|---------|
| 基层人民法院 | 大多数第一审民事案件 |
| 中级人民法院 | 重大涉外案件、在本辖区有重大影响的案件、最高人民法院确定由中院管辖的案件（如专利纠纷、海事海商） |
| 高级人民法院 | 在本辖区有重大影响的案件 |
| 最高人民法院 | 在全国有重大影响的案件、认为应当由本院审理的案件 |

### 地域管辖速查

#### 一般原则
| 管辖类型 | 规则 | 法律依据 |
|---------|------|---------|
| 一般地域管辖 | 被告住所地法院（原告就被告） | 民诉法第22条 |
| 例外（原告住所地） | 对不在境内的人提起身份关系诉讼/对下落不明人/被采取强制教育措施人/被监禁人 | 民诉法第23条 |

#### 专属管辖（不可协议变更）
| 纠纷类型 | 管辖法院 |
|---------|---------|
| 不动产纠纷 | 不动产所在地法院 |
| 港口作业纠纷 | 港口所在地法院 |
| 继承遗产纠纷 | 被继承人死亡时住所地/主要遗产所在地法院 |

#### 特殊地域管辖
| 纠纷类型 | 可选管辖法院 |
|---------|-------------|
| 合同纠纷 | 被告住所地 或 合同履行地 |
| 侵权纠纷 | 被告住所地 或 侵权行为地（含行为发生地+结果地） |
| 票据纠纷 | 票据支付地 或 被告住所地 |
| 运输合同纠纷 | 运输始发地/目的地 或 被告住所地 |
| 交通事故 | 事故发生地/车辆最先到达地/航空器最先降落地 或 被告住所地 |

#### 协议管辖
- 合同或其他财产权益纠纷的当事人可以书面协议选择
- 可选范围：被告住所地/合同履行地/合同签订地/原告住所地/标的物所在地
- 不得违反级别管辖和专属管辖的规定
- 选择需明确、唯一

## 四、案由规定简介

### 先纵后横法（《民事案件案由规定》应用核心）

**纵向检查顺序（10级优先级）：**
1. **非讼程序案件** — 宣告失踪/死亡、认定财产无主、实现担保物权等
2. **特殊诉讼程序案件** — 公益诉讼、第三人撤销之诉、执行异议之诉
3. **特殊法律关系** — 劳动争议、公司纠纷、证券纠纷、知识产权纠纷、海事海商
4. **合同纠纷** — 买卖合同、借款合同、租赁合同、建设工程合同等
5. **无因管理纠纷**
6. **物权纠纷** — 所有权、用益物权、担保物权、占有保护
7. **不当得利纠纷**
8. **特殊侵权** — 产品责任、机动车事故、医疗损害、环境污染、高度危险等
9. **人格权纠纷** — 名誉权、肖像权、隐私权、个人信息保护等
10. **一般侵权责任纠纷**

**横向选择原则：**
- 同一类目下，优先选择最具体的四级案由
- 没有对应四级案由，选三级；再没有选二级
- 确实没有对应案由，使用上级更抽象案由（如"合同纠纷"兜底）

**核心原则：**
- 穿透形式看实质 — 不因合同名称或被表面关系迷惑
- 特殊优于一般 — 有特殊规定的优先适用
- 防止案由竞合 — 一个案件选一个核心案由

## 五、起诉 / 答辩 / 上诉核心区别表

| 维度 | 起诉状 | 答辩状 | 上诉状 |
|------|--------|--------|--------|
| **适用阶段** | 一审立案 | 一审应诉 | 二审 |
| **提交主体** | 原告 | 被告 | 上诉人（一审当事人） |
| **核心功能** | 启动诉讼程序 | 回应原告诉求 | 质疑一审判决 |
| **标题** | 民事起诉状 | 民事答辩状 | 民事上诉状 |
| **案由字段** | ✅ 必须填写 | ❌ 不填 | ❌ 不填 |
| **核心内容** | 诉讼请求 + 事实理由 | 答辩请求 + 答辩意见 | 上诉请求 + 上诉理由 |
| **证据目录** | ✅ 始终生成 | ⚠️ 条件生成 | ⚠️ 条件生成 |
| **法定时限** | 诉讼时效内（一般3年） | 收到起诉状副本后15日内 | 判决书送达后15日内 / 裁定书送达后10日内 |
| **费用** | 预交受理费 | 不单独交费 | 按一审标准预交上诉费 |
| **提交法院** | 有管辖权的一审法院 | 受理起诉的法院 | 原审法院的上一级法院 |
| **共同校验项** | 通用12项 + 起诉状6项 | 通用12项 + 答辩状6项 | 通用12项 + 上诉状6项 |

## 六、诉讼流程时间节点速览
| 节点 | 时限 |
|------|------|
| 起诉 → 立案审查 | 7日内 |
| 立案 → 发送起诉状副本给被告 | 5日内 |
| 被告答辩 | 收到副本后15日内 |
| 管辖权异议 | 提交答辩状期间 |
| 一审普通程序审限 | 立案后6个月（可延长） |
| 一审简易程序审限 | 立案后3个月（不得延长） |
| 上诉期（判决） | 判决书送达后15日内 |
| 上诉期（裁定） | 裁定书送达后10日内 |
| 二审审限（判决） | 立案后3个月（可延长） |
| 二审审限（裁定） | 立案后30日内 |
| 再审申请 | 判决生效后6个月内 |
