# 秒技·AI广告护栏

> 设定利润边界 → 秒级方案生成 → 熔断建议

## 产品定位

**升维打击**：告别ACoS分析工具的被动思维，升级为「AI广告护栏」的主动防御体系。

**核心敌人**：亚马逊官方AI为花预算乱出价，卖家利润被持续吞噬。

**解决方案**：用户只需说"ACoS不能超25%"，系统秒级计算每个展示机会的最优出价，触碰红线立即生成降出价或暂停建议。

## 核心功能

### 1. 利润边界设定
基于目标ACoS/TACoS，计算关键护栏指标：
- 单次点击最高出价
- 每日预算建议上限
- 关键词出价天花板

### 2. 出价计算方案
基于广告阶段和竞争环境，生成关键词出价建议：
- 新品冷启期（0-30天）
- 爬坡期（30-90天）
- 稳定期（90-180天）
- 爆款期（180天+）

### 3. 熔断策略建议
三级熔断机制：
- **词级熔断**：单词ACoS触红线 → 建议降价至最低或暂停
- **时段熔断**：连续亏损时段 → 建议降预算
- **活动级熔断**：整活动TACoS超标 → 建议预算再分配

### 4. 飞轮出价联动
自然排名与广告投放的协同策略：
- 自然排名7-20位：强攻策略
- 自然排名前5位：防御降本

### 5. 视频广告脚本
30秒痛点驱动型视频脚本框架：
- 0-3秒：痛点引入
- 3-18秒：方案展示
- 18-25秒：社会证明
- 25-30秒：行动号召

## 极简输入

只需提供三要素：
1. **产品描述**：产品名称、类目、核心卖点
2. **广告阶段**：新品冷启 / 爬坡期 / 稳定期 / 爆款期
3. **ACoS上限**：目标ACoS百分比

## 使用示例

```
用户：我要推一款瑜伽垫，客单价35美元，转化率预估10%，ACoS控制在25%以内

AI：生成完整利润护栏报告 + 冷启期出价方案 + 熔断阈值
```
