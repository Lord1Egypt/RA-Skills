| 品类 | 目标ACOS | 优秀ACOS | 说明 |

| 美妆 | 15-30% | <15% | 复购重要 |
| 家居 | 15-25% | <12% | 客单价高 |
| 玩具 | 20-35% | <18% | 季节性强 |
| 宠物 | 15-25% | <12% | 复购高 |
| 运动 | 18-28% | <15% | 品牌重要 |
| 食品 | 12-22% | <10% | 复购率高 |

### 4.2 按产品阶段ACOS目标

| 阶段 | ACOS目标 | 说明 |
|------|----------|------|
| 新品期 | 30-50% | 冲排名优先 |
| 成长期 | 20-35% | 平衡排名和利润 |
| 稳定期 | 12-25% | 优化ROAS |
| 旺季 | 25-45% | 接受高ACOS冲量 |
| 清仓期 | 无所谓 | 清货优先 |

### 4.3 ACOS与利润率关系

```
如果：
- 毛利率 = 40%
- ACOS = 25%

那么：
- 广告贡献 = 售价 × 25%
- 广告后利润 = 售价 × 40% - 售价 × 25% = 售价 × 15%

如果：
- 毛利率 = 60%
- ACOS = 40%

那么：
- 广告贡献 = 售价 × 40%
- 广告后利润 = 售价 × 60% - 售价 × 40% = 售价 × 20%

结论：毛利率越高，可接受的ACOS越高
```

| 母婴 | 5-12% | 婴儿推车 | 纸尿裤 |

### 5.2 退货成本计算

```
退货成本 = 退款金额 + 退货处理费 + 物流费 + 商品损耗

示例（服装，$30售价）：
- 退款金额：$30
- 退货处理费：$3（约10%）
- 来回物流：$6
- 商品损耗（可能二手）：$0-15

总退货成本：$39-54

如果退货率20%：
每件退货成本 = $39 × 20% = $7.8

建议：将退货成本计入定价
调整毛利率 = 毛利率 - (退货率 × 退货成本/售价)
```

## 六、价格区间参考

### 6.1 低价区（<$20）

适合：
- 引流款
- 配件耗材
- 冲动消费品
- 新店铺起步

特点：
- 竞争激烈
- 利润率低（15-25%）
- 容易出单
- 评价积累快

### 6.2 中价区（$20-75）

适合：
- 大多数品类
- 有一定差异化
- 稳定经营

特点：
- 市场主流
- 利润率适中（25-40%）
- 需要持续优化
- 平衡排名和利润

### 6.3 高价区（>$75）

适合：
- 强差异化
- 专业工具
- 高端品牌
- 礼品类

特点：
- 竞争相对少
- 利润率最高（40%+）
- 转化率低但精准
- 客户要求高
