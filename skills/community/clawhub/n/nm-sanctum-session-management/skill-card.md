## Description: <br>
Manages Claude Code sessions with naming, checkpointing, and resume strategies. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[athola](https://clawhub.ai/user/athola) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to organize complex Claude Code work with named sessions, checkpoints, and resume patterns. It is most useful for debugging, feature development, PR review, and investigations that span multiple sessions. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Session names and resumed context may expose project details or stale assumptions from prior work. <br>
Mitigation: Use descriptive but non-sensitive session names and review resumed context before acting on it. <br>
Risk: The security evidence recommends limiting installation to ClawHub staff or development contexts. <br>
Mitigation: Install only in staff or development environments and keep admin credentials limited. <br>
