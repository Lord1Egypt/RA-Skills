# PPT 输出模板

本文件定义 PPT 内容的输出模板格式，供用户根据需求深度选择。

---

## 输出级别选择

| 级别 | 输出内容 | 适用场景 |
|:---|:---|:---|
| **快速大纲** | 页数规划 + 每页标题 + 一句话摘要 | 用户只需框架 |
| **标准方案** | 快速大纲 + 每页完整文案 + 视觉建议 + 配色方案 | 大多数场景首选 |
| **深度方案** | 标准方案 + 逐页演讲稿 + Q&A 预判 + 过渡语 | 重要路演/发布会 |
| **单页优化** | 针对某一页重写文案+新排版方案 | 用户对某页不满意 |
| **完整内容包** | 全部文案一次性输出，用户可直接逐页粘贴到 PPT | 用户有完整素材需要整理 |

---

## 每页输出内容块

### 标准输出格式

每页输出以下四块内容（表格格式）：

| 输出块 | 要求 | 用途 |
|:---|:---|:---|
| **标题** | 完整的陈述句（≤ 15 字），加粗 | 直接填入 PPT 标题框 |
| **正文文案** | 3-5 个 bullet point，每条 ≤ 20 字，用 `•` 开头 | 直接填入 PPT 正文框 |
| **数据（如有）** | 精确数字 + 对比 + 单位，如 `3.2M → 8.7M（+172%）` | 填入图表数据标签 |
| **演讲备注** | 一段 60-120 字的演讲稿（过渡语 + 强调点） | 直接复制到 PPT 备注栏 |

### 格式建议（必附）

每页还须附以下格式标注：

- **排版布局**：左文右图 / 上图下文 / 全屏图+文字蒙版 / 卡片式 / 居中大标题
- **对齐方式**：居中 / 左对齐
- **图表类型**：柱状图 / 折线图 / 饼图 / 对比矩阵 / 表格等
- **图片示意**：用文字描述需要的图片内容

## 快速大纲模板

```
## 第 1 页：封面
标题：[完整陈述句标题]
目的：吸引注意力 + 点明主题

## 第 2 页：目录
标题：汇报提纲
内容：四大模块概览

...
```

## 标准方案模板（每页表格）

```
### 第 X 页：[页码/页面类型]

| 输出块 | 内容 |
|:---|:---|
| **标题** | **核心陈述句（≤ 15 字）** |
| **正文** | • 要点 1<br>• 要点 2<br>• 要点 3 |
| **数据** | 关键数字与对比 |
| **备注** | 60-120 字演讲稿 |
| **格式建议** | 布局/对齐/图表/图片示意 |
```


## 字体方案输出格式

```
### 🔤 字体搭配

| 层级 | 字体 | 字号 | 说明 |
|:---|:---|:---:|:---|
| 封面标题 | 微软雅黑 Bold | ≥ 30pt | 醒目突出 |
| 一级标题 | 微软雅黑 Bold | 20-28pt | 章节主标题 |
| 二级标题 | 微软雅黑 Medium | 16-18pt | 段落子标题 |
| 正文 | 微软雅黑 / 宋体 | ≥ 14pt | 正文内容 |
```
