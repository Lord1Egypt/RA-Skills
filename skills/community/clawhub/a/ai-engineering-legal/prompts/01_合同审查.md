# 合同审查Prompt模板

## 基础模板

```
你是一名工程造价+法律专家，基于「AI工程造价+法律知识库」（1394人订阅，3149个内容）进行合同审查。

请审查以下工程合同条款：

[合同条款内容]

请重点关注以下方面：

1. **计价风险**
   - 计价方式是否明确？（固定总价/单价/可调价/成本加酬金）
   - 价格调整机制是否合理？
   - 材料价格波动时的处理办法？
   - 参照知识库中《工程量清单计价标准》

2. **变更管理**
   - 变更确认流程是否完整？
   - 变更时限是否合理？
   - 变更费用计算是否有依据？
   - 参照知识库中《变更索赔实务指南》

3. **结算条款**
   - 结算依据是否清晰？
   - 审核期限是否合理？
   - 逾期不回复是否视为认可？
   - 参照知识库中《结算纠纷案例》

4. **违约责任**
   - 违约金是否过高？
   - 责任分配是否公平？
   - 是否有单方免责条款？
   - 参照知识库中《合同风险防控指南》

5. **争议解决**
   - 争议解决方式是否明确？
   - 仲裁还是诉讼？
   - 管辖法院是否合理？

对每个高风险条款，请给出：
- 风险等级（高/中/低）
- 具体风险描述
- 修改建议
- 知识库依据（引用具体文件）
```

## 快速审查模板

```
请快速审查以下合同的关键风险点：

[合同关键条款]

只需要指出：
1. 最危险的3个条款
2. 建议修改的方向
3. 谈判时的注意事项
```

## 对比审查模板

```
请对比以下两份合同的差异：

合同A：[合同A关键条款]
合同B：[合同B关键条款]

重点对比：
1. 计价方式的差异
2. 风险分配的不同
3. 对哪方更有利
4. 建议选择哪份或如何修改
```

## 条款优化模板

```
以下合同条款存在风险，请优化：

原始条款：[有问题的条款]

请提供：
1. 问题分析（引用知识库依据）
2. 优化后的条款（可直接使用）
3. 谈判话术（如何说服对方接受）
```
