## 🛒 Prime Day作战指南

### Prime Day时间与规模

**历年时间**：
| 年份 | Prime Day日期 | 时长 |
|------|---------------|------|
| 2024 | 7月16-17日 | 48小时 |
| 2023 | 7月11-12日 | 48小时 |
| 2022 | 7月12-13日 | 48小时 |

**预期销量**：
- 整体：平日2-5倍
- 热门品类：平日5-10倍
- 做好准备：平日10-20倍

### Prime Day备战时间线

**T-4个月（3月）**：
```
□ 复盘去年Prime Day数据
□ 选定今年主打产品
□ 开始计算备货量
□ 准备Prime Day专属优惠券
```

**T-3个月（4月）**：
```
□ 下单生产/采购
□ 准备Prime Day素材
□ 优化listing（标题/图片/评价）
□ 开始FBA发货
```

**T-2个月（5月）**：
```
□ 完成首批FBA入仓
□ 开启广告预热（手动+自动）
□ 激活Prime专享折扣
□ 设置Prime DayDeal/LD
```

**T-1个月（6月）**：
```
□ 入仓截止检查
□ 调整广告预算
□ 准备客服排班
□ 确认库存充足
□ 提前预热（社媒/邮件）
```

**T-7天（Prime Day前一周）**：
```
□ 最后库存检查
□ 广告预算加满
□ 确认所有Deal状态
□ 准备紧急预案
```

**Prime Day当天**：
```
□ 实时监控销售
□ 调整广告竞价
□ 关注库存
□ 快速响应问题
□ 记录实时数据
```

### Prime Day定价策略

**定价公式**：
```
Prime Day价格 = 平日价 × (1 - 折扣率)

折扣率建议：
- 15-20%：利润保护型
- 25-30%：中等促销
- 30-50%：引流冲量型（谨慎！）

⚠️ 注意：Prime Day价格不能低于历史最低价！
```

**各类产品策略**：
| 产品类型 | 策略 | 折扣建议 |
|----------|------|----------|
| 引流款 | 吸引流量，带动店铺 | 25-40% |
| 利润款 | 稳定利润 | 15-20% |
| 新品 | 测试市场 | 20-30% |
| 滞销品 | 清库存 | 30-50% |

### Prime Day广告策略

**分阶段打法**：

**阶段1：预热期（Prime Day前2-4周）**：
```
目标：积累权重，提升排名
策略：
- 提升广告预算20-30%
- 降低竞价，积累点击
- 优化listing关键词
```

**阶段2：冲刺期（Prime Day前1周）**：
```
目标：冲排名，抢流量
策略：
- 广告预算翻倍
- 提高竞价抢首屏
- 开启更多关键词
```

**阶段3：旺日（Prime Day当天）**：
```
目标：最大化转化
策略：
- 保持高预算
- 实时调整竞价
- 关注ACOS，但不要过度保守
```

**阶段4：延续期（Prime Day后1-2周）**：
```
目标：维护排名，清理库存
策略：
- 逐步降低预算
- 清理Prime Day没卖完的库存
- 分析数据做复盘
```

## 📦 旺季备货计算

### 备货量公式

**基础公式**：
```
备货量 = 旺季预期日销量 × 旺季天数 × 安全系数 + 预留库存

旺季天数：
- Prime Day：7天（包含前后预热）
- 黑五网一：10天
- 圣诞：30天

安全系数：
- 1.2-1.3（保守）
- 1.1-1.2（常规）
- 1.0（激进）
```

**计算示例**：
```
产品A：
- 平日日销量：20单
- Prime Day预期倍数：5倍
- Prime Day备货天数：7天
- 安全系数：1.2

备货量 = 20 × 5 × 7 × 1.2 = 840单
```

### 分批发货计划

**建议分3批发货**：
| 批次 | 时间 | 数量占比 | 目的 |
|------|------|----------|------|
| 第一批 | T-90天 | 40% | 建立基础库存 |
| 第二批 | T-60天 | 40% | 补充库存 |
| 第三批 | T-30天 | 20% | 应急/补货 |

### 旺季库存管理

**库存预警机制**：
```
绿区（>14天库存）：安全
黄区（7-14天库存）：注意
橙区（3-7天库存）：预警
红区（<3天库存）：紧急补货
```

**断货应对**：
1. 提高售价（减少需求）
2. 暂停广告（减少流量）
3. 通知客户（管理预期）
4. 加快补货（但不能催FBA）

## ⚠️ 旺季避坑清单

### 常见旺季坑

| 坑 | 问题 | 解决方案 |
|----|------|----------|
| 备货太多 | 旺季卖不完，仓储费爆炸 | 安全系数≤1.2 |
| 备货太少 | 断货，错过旺季 | 早备货，分批发 |
| 入仓太晚 | FBA来不及上架 | 提前3-4周入仓 |
| 广告太保守 | 流量被抢 | 预算充足 |
| 定价失误 | 亏本或没流量 | 参考同行+利润底线 |
| 客服跟不上 | 差评爆炸 | 提前准备客服 |

### 旺季后清仓策略

**清仓时机**：
- 节后第一天立即开始
- 不要等，不要犹豫

**清仓方式**：
```
□ 设置Outlet Deal
□ 站外促销（FB群组/Deal站）
□ 大额Coupon
□ 主动降价
□ 考虑销毁（长期仓储费太贵）
```
