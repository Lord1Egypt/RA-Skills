# 微信小程序 SDK 接入流程

## 1. 前置条件

- 已完整阅读本 SDK 接入流程文档
- 在微信公众平台「开发 → 开发管理 → 开发设置 → 服务器域名」中，将以下域名添加至 **request 合法域名**：
  ```
  https://api.datanexus.qq.com
  ```

## 2. 下载 SDK

### 方式一：npm 安装（推荐）

```bash
npm i @dn-sdk/miniprogram
```

安装后在微信开发者工具中点击「工具 → 构建 npm」。

引用方式：

```javascript
import { SDK } from '@dn-sdk/miniprogram';
```

### 方式二：链接下载

从 DataNexus 文档站下载 SDK 文件（含 index.js 和类型声明），放入项目目录直接引用。

## 3. 引用 SDK 并初始化

根据开发框架不同，引用方式略有差异（详见[框架适配](./框架适配.md)）。以下为通用初始化示例：

```javascript
import { SDK } from '@dn-sdk/miniprogram';

const sdk = new SDK({
  user_action_set_id: '您的数据源ID',
  secret_key: '您的密钥',
  appid: 'wx1234567890abcdef',
});
```

### 初始化参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `user_action_set_id` | number | 是 | 数据源 ID |
| `secret_key` | string | 是 | 数据源密钥 |
| `appid` | string | 是 | 当前小程序的 AppID（wx 开头） |
| `openid` | string | 否 | 微信 OpenId |
| `unionid` | string | 否 | 微信 UnionId |
| `user_unique_id` | string | 否 | 业务自定义用户 ID |
| `auto_track` | boolean | 否 | 自动采集开关，默认 `true` |

> ⚠️ openid 和 unionid 二选一即可，openid 优先。

### 设置用户 ID

```javascript
sdk.setOpenId('用户的openid');
sdk.setUnionId('用户的unionid');
```

> ⚠️ 未设置 openid 或 unionid 时，SDK 会暂停数据上传。

## 4. 事件上报

### 4.1 通用方法

```javascript
sdk.track('PURCHASE', { value: 9900 });
sdk.track('REGISTER');
```

### 4.2 专用方法

| 方法 | 对应 action_type | 说明 |
|------|-----------------|------|
| `sdk.onAppStart()` | START_APP | 自动采集 |
| `sdk.onAppQuit()` | APP_QUIT | 退出应用 |
| `sdk.onEnterForeground()` | ENTER_FOREGROUND | 自动采集 |
| `sdk.onEnterBackground()` | ENTER_BACKGROUND | 自动采集 |
| `sdk.onPageShow()` | PAGE_VIEW | 自动采集 |
| `sdk.onPageHide()` | PAGE_LEAVE | 自动采集 |

## 5. 数据对账

DataNexus 平台 → 日志查询，Network 面板监控返回 `code: 0` 即成功。

## 6. 高级功能

- 多数据源上报（最多 4 个）
- `SDK.setRequestConcurrency(5)`
- 自动采集：auto_track: true
- 调试模式：`SDK.setDebug(true)`，TAG `[@dn-sdk/miniprogram v1.x.x]`
