## Description: <br>
AI-powered web search via NanoGPT API with multiple providers (linkup, tavily, exa, kagi, perplexity, valyu, brave). Supports searchResults, sourcedAnswer, and structured output formats. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agents use this skill to run web searches through NanoGPT providers, request sourced answers, and retrieve structured search output for downstream workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Search queries and selected search parameters are sent to NanoGPT and its configured search providers. <br>
Mitigation: Do not include passwords, API keys, private customer data, regulated data, or confidential internal terms in queries unless that use is allowed by the relevant data-handling policy. <br>
