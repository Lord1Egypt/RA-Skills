- "我们要办X，给个方案"
- "X活动怎么安排？"

## 第三阶段：分场景详细指南

### 3.1 求婚策划

**核心**：惊喜感 + 浪漫氛围 + 见证人在场

**信息参数**：预算、城市、人数、风格（浪漫/创意/温馨）、特殊需求

**场地推荐**：
- 自然景观（公园/海边/山顶）：免费-500元
- 室内温馨（咖啡馆/餐厅包间）：500-2000元
- 高端场地（酒店天台/艺术空间）：2000-10000元

**流程模板**（2-3小时）：
```
T-3小时：朋友到场布置
T-1小时：确认所有细节
T-30分钟：女方入场引导（保密到位）
T-0：音乐响起，告白时刻（台词+戒指）
T+30分钟：庆祝环节（蛋糕/香槟/合影）
T+1小时：转场庆祝晚宴
```

**预算分配**：场地15% + 布置20% + 摄影10% + 餐饮35% + 惊喜道具10% + 预备金10%

**执行Tips**：
1. 提前1周确认场地（避免临时被占用）
2. 安排1-2位朋友负责摄影+布置
3. 准备室内备选方案（天气预案）
4. 提前练习告白台词（避免紧张忘词）
5. 购买意外险（求婚戒指）

---

### 3.2 满月酒策划

**核心**：宝宝为中心 + 长辈在场 + 温馨喜庆

**信息参数**：宝宝姓名/小名、性别、预算、宾客人数、宴会形式（午宴/晚宴）

**场地推荐**：
- 家中（<30人）：免费，温馨
- 餐厅包间（30-60人）：500-3000元
- 酒店宴会厅（60-200人）：3000-20000元

**宴会流程**：
```
宾客入场：签到、领伴手礼
开场介绍：宝宝成长点滴分享
切蛋糕仪式：全场合唱生日歌
互动游戏：抓周/抽奖
用餐交流：自由活动
送客：送别+合影
```

**预算分配**：场地20% + 餐饮40% + 布置15% + 摄影5% + 伴手礼10% + 预备金10%

**执行Tips**：
1. 宝宝月龄建议28-45天（太小妈妈恢复不佳，太大宝宝作息规律难配合）
2. 提前1个月确定日期（长辈时间协调）
3. 餐饮注意忌口（部分长辈可能有饮食禁忌）
4. 准备宝宝护理用品（现场可能需要换尿布/喂奶）
5. 安排亲友负责照顾宝宝和引导来宾

---

### 3.3 周年纪念策划

**核心**：情怀 + 仪式感 + 共同回忆

**信息参数**：结婚年份、几周年、预算、是否邀请见证人、期望惊喜程度

**主题方案库**：
| 周年 | 传统礼物 | 主题灵感 |
|------|---------|---------|
| 1年 | 纸 | 回忆相册/手写信 |
| 3年 | 皮革 | 皮具定制礼物 |
| 5年 | 木 | 木质纪念品 |
| 7年 | 铜/电器 | 智能家电 |
| 10年 | 锡/铝 | 定制首饰 |
| 20年 | 瓷器 | 定制瓷器 |
| 30年 | 珍珠 | 珍珠饰品 |

**惊喜设计**：
- **初级**（<2000元）：餐厅布置+鲜花+手写信+电子相册
- **中级**（2000-5000元）：朋友祝福视频+定制纪念品+短途旅行
- **高级**（>5000元）：蜜月重游+定制珠宝+专业跟拍

**执行Tips**：
1. 提前了解对方心愿（有没有一直想去的地方）
2. 制作"回忆时间线"展板效果极佳
3. 写一封手写信（电子时代手写更有温度）
4. 预留充裕预算给"共同见证的朋友"
5. 不要在当天安排其他事情（专心庆祝）

### 3.4 会议策划

**核心**：高效 + 决策导向 + 明确产出

**信息参数**：会议类型（例会/研讨会/发布会/培训会/团建会议）、预算、人数、时长、城市、特殊需求（直播/翻译/茶歇）

**会议类型决策**：
| 类型 | 时长 | 人数 | 核心要素 |
|------|------|------|---------|
| 例会 | 1-2小时 | 5-20人 | 议程简洁、决策导向 |
| 研讨会 | 半天-1天 | 20-100人 | 内容深度、互动讨论 |
| 发布会 | 2-3小时 | 50-200人 | 媒体邀请、展示效果 |
| 培训会 | 1天-多天 | 10-50人 | 内容系统、练习环节 |
| 团建会议 | 半天-1天 | 10-50人 | 活动设计、氛围轻松 |

**议程设计原则**：
1. **黄金时间**：开场30分钟讲重点（注意力最高）
2. **节奏控制**：每45分钟休息10分钟
3. **互动比例**：纯听不超过60%，需要30%+互动
4. **收尾有力**：最后15分钟总结要点+明确下一步

**预算分配**：场地20% + 设备10% + 餐饮35% + 资料10% + 预备金25%（含未预见支出）

**执行清单**：
- 会前1周：确认场地+发送通知+确认发言人
- 会前1天：现场考察+测试设备+打印资料
- 会议当天：提前1小时到场+签到+全程记录

### 3.5 年会策划

**核心**：年度总结 + 员工激励 + 团队凝聚

**信息参数**：公司规模、预算、举办时间、是否有节目表演、是否颁奖/抽奖、风格偏好

**年会类型**：
| 类型 | 规模 | 风格 | 预算/人 |
|------|------|------|---------|
| 聚餐型 | 10-50人 | 轻松 | 150-300元 |
| 团建型 | 20-100人 | 活跃 | 300-600元 |
| 正式型 | 50-300人 | 庄重 | 500-1000元 |
| 大型庆典 | 300+人 | 豪华 | 1000-3000元 |

**节目规划**：
1. 领导致辞
2. 团队节目（舞蹈/歌曲/小品/朗诵）
3. 互动游戏（抽奖/游戏/竞答）
4. 颁奖典礼（年度优秀员工/团队）
5. 祝福视频（客户/合作伙伴）

**预算分配**：场地20% + 餐饮30% + 奖品25% + 布置10% + 节目5% + 预备金10%

**奖品设置**：
| 奖项 | 人数 | 单价 | 总预算占比 |
|------|------|------|---------|
| 特等奖 | 1人 | 1000-3000元 | 15-20% |
| 一等奖 | 3-5人 | 500-1000元 | 15-20% |
| 二等奖 | 10-20人 | 200-500元 | 20-25% |
| 三等奖 | 30-50人 | 50-200元 | 20-25% |
| 阳光普照 | 全部 | 20-50元 | 10-15% |

**执行Tips**：
1. 提前2个月确定日期（年底酒店抢手）
2. 提前1个月确认人数（订桌/订餐）
3. 提前2周收集节目报名（早报名有奖励）
4. 奖品采购提前1个月（物流+包装）
5. 设置候补名单应对临时不来的人

### 3.6 培训策划

**核心**：内容系统 + 互动充分 + 可落地

**信息参数**：培训主题、学员人数、学员背景（新人/管理层/全员）、时长（半天/1天/多天）、形式（线上/线下/混合）

**培训类型**：
| 类型 | 时长 | 人数 | 核心要素 |
|------|------|------|---------|
| 新人入职培训 | 1-3天 | 10-50人 | 公司文化、基础技能 |
| 技能提升培训 | 1-5天 | 5-30人 | 专业技能、实操练习 |
| 管理培训 | 2-5天 | 10-30人 | 领导力、管理技巧 |
| 全员培训 | 半天-1天 | 30-200人 | 通识、共识、宣导 |
| 外聘讲师培训 | 1-3天 | 20-100人 | 专业内容、资质证书 |

**课程设计原则**：
1. **黄金比例**：讲解30% + 练习40% + 讨论30%
2. **内容层次**：概念→案例→练习→应用
3. **学员参与**：每30分钟有互动（提问/讨论/练习）
4. **视觉辅助**：PPT+白板+卡片+视频多种形式

**预算分配**：场地25% + 讲师30% + 物料10% + 餐饮15% + 设备10% + 预备金10%

**执行Tips**：
1. 提前7天发送预习资料（提高参与度）
2. 设置学习奖励（小组PK积分）
3. 每半天收集一次学员反馈（及时调整）
4. 准备茶水/咖啡/点心（维持精力）
5. 培训后1周发送行动计划跟踪

### 3.7 启动会策划

**核心**：目标清晰 + 分工明确 + 士气激励

**信息参数**：项目类型（产品/营销/技术/活动）、团队规模、预算、领导是否参加、是否需要外部嘉宾

**项目类型决策**：
| 类型 | 重点 | 时长 | 规模 |
|------|------|------|------|
| 产品发布 | 目标/时间线/分工 | 2-3小时 | 全员 |
| 技术项目 | 技术架构/里程碑 | 2-3小时 | 技术团队 |
| 营销活动 | 目标/渠道/预算 | 3-4小时 | 营销团队 |
| 年度计划 | 战略/OKR/激励 | 半天-全天 | 全员 |

**议程模板**（3小时）：
```
09:00-09:30  入场签到
09:30-09:40  领导开场致辞（5分钟要点+鼓励）
09:40-10:00  项目背景与目标（负责人）
10:00-10:30  项目整体规划（时间线/里程碑）
10:30-10:45  茶歇
10:45-11:15  团队介绍与分工（各组负责人）
11:15-11:45  KPI与激励机制（HR/PM）
11:45-12:00  团队合影
12:00-13:30  团队午餐
13:30-14:30  分组讨论（各小组工作坊）
14:30-15:00  小组汇报与答疑
15:00-15:30  总结与承诺（每个人写下目标）
15:30        散会
```

**预算分配**：场地30% + 物料20% + 餐饮30% + 设备10% + 预备金10%

**执行Tips**：
1. 提前制作"项目时间线展板"（增强视觉冲击）
2. 为每位参与者准备姓名牌
3. 准备"团队分工表"（人手一份）
4. 设置明确的OKR/KPI展示
5. 会后24小时内发出会议纪要和行动清单

### 3.8 家长会策划

**核心**：有效沟通 + 建设性反馈 + 家校合作

**信息参数**：孩子年级、家长人数（预计）、沟通主题、形式（线下/线上）、是否需要老师参与

**家长会类型**：
| 类型 | 适合年级 | 时长 | 重点 |
|------|---------|------|------|
| 期中期未家长会 | 全年级 | 1-1.5小时 | 成绩+表现 |
| 新生家长会 | 一年级/新生 | 2小时 | 学校介绍+期望 |
| 专题家长会 | 特定年级 | 1.5-2小时 | 青春期/升学/心理 |
| 一对一家长会 | 个别学生 | 30分钟 | 针对性沟通 |

**流程模板**（1.5小时）：
```
19:00-19:10  入场签到（领取孩子近照/作品）
19:10-19:20  班主任开场（近期整体情况）
19:20-19:50  学科汇报（语数英各5分钟）
19:50-20:00  茶歇（自由交流）
20:00-20:20  专题分享（青春期/学习方法等）
20:20-20:40  家长代表发言/经验分享
20:40-20:55  自由提问与交流
20:55-21:00  总结+下次预告
```

**内容准备要点**：

**预算分配**：搬家费30% + 新居布置20% + 暖房宴30% + 仪式用品5% + 预备金15%

**执行Tips**：
1. 提前断舍离（搬家前扔掉不需要的东西）

4. 新居先打扫（入住前彻底清洁）

### 3.23 同学聚会策划

**核心**：回忆杀 + 叙旧 + 感情升温

**信息参数**：毕业年份、人数、预算、形式（聚餐/旅行/派对）、是否有节目表演

**聚会类型**：
| 类型 | 规模 | 风格 | 预算/人 |
|------|------|------|---------|
| 聚餐型 | 10-30人 | 轻松 | 200-500元 |
| 旅行型 | 10-50人 | 活跃 | 1000-3000元 |
| 派对型 | 30-100人 | 狂欢 | 300-800元 |

**策划要点**：
1. **确定核心筹备组**（3-5人）：总协调/财务/活动/宣传/后勤
2. **收集同学信息**：问卷调查（是否参加/饮食禁忌/才艺）
3. **选择活动形式**：聚餐/旅行/派对/KTV/轰趴
4. **准备回忆素材**：老照片/视频/成绩单/毕业证
5. **设计互动环节**：自我介绍（现在做什么）/才艺表演/抽奖/互换礼物

**流程模板**（聚餐型，4小时）：
```
17:00-17:30  签到入场（领取名牌+老照片展）
17:30-18:00  开场（主持人致辞+回忆视频）
18:00-19:30  晚宴（边吃边聊）
19:30-20:30  互动环节（自我介绍+才艺+游戏）
20:30-21:00  抽奖+互换礼物
21:00-21:30  合影+约定下次聚会
21:30        散场+建群
```

**预算分配**：餐饮40% + 场地10% + 活动道具10% + 纪念品20% + 预备金20%

## 注意事项

1. **先收集参数，再输出方案** — 不要输出空洞内容
2. **预算要合理** — 不要过度设计，给出性价比方案
3. **要有温度** — 特别是求婚/满月酒/周年纪念，情感元素优先
4. **执行要可行** — 方案要能落地，不要只写理想状态
5. **预留弹性** — 每个方案必须包含Plan B和预备金
6. **文化敏感** — 注意不同文化背景的禁忌（如婚礼/葬礼）
7. **安全第一** — 特别是户外活动/大型聚集，安全预案必须有
