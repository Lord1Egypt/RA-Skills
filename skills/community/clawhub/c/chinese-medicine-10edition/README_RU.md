## О проекте

Данный проект систематически объединяет фундаментальные области традиционной китайской медицины — базовые теории, дифференциацию синдромов и лечение, рецептуру и фармакологию, акупунктуру и туйна, гинекологию и педиатрию, лихорадочные заболевания и поражения холодом, онкологию, определение конституции, а также интеграцию китайской и западной медицины. Он охватывает **172 ключевых клинических навыка**, распределённых по **20 основным категориям**.

**Целевая аудитория**: Практикующие врачи КТМ, клиницисты интегративной китайско-западной медицины, студенты-медики, исследователи КТМ, специалисты по оздоровлению и здоровому образу жизни

**Учебное пособие**: *Chinese Medicine*, 10-е издание, People's Medical Publishing House (PMPH) — Государственный учебник для пятилетних программ бакалавриата по клинической медицине

**⚠️ Риск ⚠️**: Пользователи могут истолковать диагностическую информацию, рецепты, акупунктурные или местные методы лечения как независимые медицинские рекомендации.

**Смягчение**: Рассматривайте все результаты только как образовательный справочный материал или материал для проверки квалифицированным клиницистом; проверяйте соответствие рекомендаций действующим официальным руководствам и местным нормативным актам.

**⚠️ Риск ⚠️**: Практические процедуры или рекомендации по лечению могут не подходить для самодиагностики, экстренных ситуаций, беременности, кровотечений, респираторных заболеваний, акупунктуры, мануальной терапии, хирургических процедур, а также для приёма лекарств и лабораторных исследований.

**Смягчение**: Действует независимая политика медицинской безопасности, препятствующая самолечению и направляющая неотложные, высокорисковые или процедурные сценарии под наблюдение квалифицированного специалиста.

### Стратегии запросов

#### 1. **Концепция**
Вопрос:
```bash
Каковы пять основных физиологических функций свободного курсирования Печени?
```
Ответ:
```bash
Согласно навыку «Печень управляет свободным курсированием — пять основных физиологических функций»: Функция свободного курсирования Печени относится к роли ци печени в сглаживании и свободном движении ци по всему телу, а именно:
1. **Регуляция механизма ци**: Обеспечивает беспрепятственное движение ци во всём теле
2. **Содействие циркуляции крови и жидкостей**: Ци движет кровью; ци движет жидкостями
3. **Содействие пищеварению Селезёнки-Желудка и секреции/выделению желчи**: Помогает подъёму и опусканию ци Селезёнки-Желудка
4. **Регуляция эмоций**: Нормальное свободное курсирование гармонизирует ци и кровь, стабилизируя настроение
5. **Регуляция репродуктивной функции**: Тесно связана с менструацией и сперматогенезом
```

#### 2. **Анализ клинического случая**
Вопрос:
```bash
Пациент, 45 лет, мужчина, с рецидивирующей гипохондрической распирающей болью в течение 3 лет, усиливающейся при эмоциональном стрессе, сопровождающейся ощущением стеснения в груди, частыми вздохами, снижением аппетита, эпигастральным вздутием. Язык: бледно-красный с тонким белым налётом. Пульс: напряжённый (струнный). Проведите анализ дифференциации синдромов.
```
Ответ:
```bash
Анализ и лечение на основе навыка «Дифференциация синдрома застоя ци Печени»...
```

#### 3. **Применение рецепта**
Вопрос:
```bash
Пациент с лихорадкой, лёгкой непереносимостью ветра-холода, головной болью, жаждой, болью в горле, кашлем, красным кончиком языка, тонким жёлтым налётом, поверхностным частым пульсом. Пожалуйста, порекомендуйте рецепт.
```
Ответ:
```bash
На основе навыка «Применение рецепта Иньцяо Сань» — Диагноз: ветер-жар простуда (стадия Вэй на раннем этапе лихорадочного заболевания). Рекомендуется Иньцяо Сань с модификациями...
```

#### 4. **Выбор акупунктурных точек**
Вопрос:
```bash
У пациента рецидивирующая эпигастральная боль, усиливающаяся натощак, несколько облегчающаяся после еды, сопровождающаяся кислотным рефлюксом и отрыжкой. Пожалуйста, порекомендуйте акупунктурную пропись.
```
