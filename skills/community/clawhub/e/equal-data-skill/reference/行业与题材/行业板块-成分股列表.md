# 行业板块-成分股列表

> 分类: 行业与题材 | 目录: `行业与题材` | 索引见 `SKILL.md`

<!-- generated_at: 2026-04-30T09:29:07.184-1777512547186 -->


### 描述

查询指定行业板块下的成分股列表，按涨幅从高到低排序，返回股票代码、名称、涨跌幅等。适用于分析行业内个股表现差异

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "F5" | 接口ID |
| `tradeDate` | `str` | 否 | - | 交易日期，示例：2024-03-20 |
| `industryName` | `str` | 否 | - | 板块名称（来自行业板块列表） |
| `pageNum` | `int` | 否 | `0` | 分页偏移量，从 0 开始（第 1 页为 0，第 2 页为 1） 示例：0（第一页） |
| `pageSize` | `int` | 否 | `10` | 每页条数 示例：10（每页10条） |
| `industryLevel` | `int` | 否 | - | 行业级别（1：申万一级行业，2：申万二级行业，3：申万三级行业, 默认） 示例：3 |
| `sortDirection` | `int` | 否 | - | 排序方向,按照涨跌幅排序，支持定义排序方向（1：升序, 2：降序, 默认） |

### 返回参数

ResultData.data 为 List，元素为热门行业板块-股票列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `nameLevelOne` | `str` | 否 | 板块/概念名称 |
| `nameLevelTwo` | `str` | 否 | 所属板块/概念二级分类 |
| `nameLevelThree` | `str` | 否 | 所属板块/概念三级分类 |
| `industryChgPct` | `str` | 否 | 板块/概念涨跌幅(%) |
| `stockName` | `str` | 否 | 股票名称 |
| `stockCode` | `str` | 否 | 股票代码 |
| `latestChgPct` | `str` | 否 | 最新涨跌幅(%) |
| `themeIntroduction` | `str` | 否 | 概念板块简介 |
