# SLE组织损伤机制分析

## 触发条件
当患者血清中检测到抗DNA抗体、抗红细胞/血小板/淋巴细胞抗体等自身抗体时，启动本分析流程。

## 执行步骤

### 1. 判断主导损伤机制类型
- **若存在免疫复合物沉积证据**（如肾小球“白金耳样”改变、血管炎），按Ⅲ型超敏反应处理。
- **若表现为溶血、血小板减少或淋巴细胞减少**，且检出相应抗血细胞抗体，按Ⅱ型超敏反应处理。

### 2. Ⅲ型超敏反应（免疫复合物介导）分析
- 确认DNA-抗DNA免疫复合物在血管壁或肾小球沉积。
- 评估补体激活及中性粒细胞浸润程度。
- 关联临床表现：重点排查狼疮性肾炎（尤其弥漫增生型）和系统性血管炎。

### 3. Ⅱ型超敏反应（抗体介导细胞毒）分析
- 确认抗血细胞抗体结合靶细胞表面抗原。
- 判断损伤途径：补体介导溶解 或 抗体依赖细胞毒作用（ADCC）。
- 关联临床表现：全血细胞减少（溶血性贫血、免疫性血小板减少、淋巴细胞减少）。

### 4. 辅助诊断标志识别
- 检查外周血或骨髓涂片是否存在LE细胞（吞噬狼疮小体的中性粒细胞或巨噬细胞）。
- 皮肤活检确认真皮-表皮交界处IgG/IgM/C3线性沉积（狼疮带试验阳性）。

### 5. 输出治疗导向建议
- 若以Ⅲ型为主：考虑强化免疫抑制（如环磷酰胺、霉酚酸酯）及补体通路干预。
- 若以Ⅱ型为主：优先使用糖皮质激素、利妥昔单抗等靶向B细胞或抗体清除策略。

> 注意：抗核抗体本身无直接细胞毒性，仅结合已受损或变性细胞核；本技能不适用于无自身抗体阳性的非SLE个体。
