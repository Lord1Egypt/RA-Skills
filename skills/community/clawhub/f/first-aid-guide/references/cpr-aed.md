# 心肺复苏 (CPR) 与 AED 使用指南

> **⚠️ 免责声明**：本文仅供参考，不能替代专业急救培训。施救前请拨打120。急救操作可能因地区指南（AHA/ERC）不同而有差异。

---

## 一、生存链 (Chain of Survival)

```
识别心跳骤停 → 立即CPR → 快速除颤(AED) → 高级生命支持 → 综合救治
```

**黄金4分钟**：心脏骤停后4分钟内开始CPR，存活率可提高2-3倍。

---

## 二、成人CPR标准流程 (AHA 2020指南)

### 1. 确保现场安全
- 确认施救者和患者所处环境安全，避免二次伤害

### 2. 判断意识与呼吸
- **拍打双肩 + 大声呼喊**："你还好吗？"
- 同时观察胸部起伏（5-10秒），判断有无正常呼吸
- **喘息样呼吸 ≠ 正常呼吸**，视为心脏骤停

### 3. 启动应急系统
- 立即叫人拨打 **120**
- 叫人取来 **AED**（自动体外除颤器）
- 如独自一人：先用手机免提拨打120，再开始CPR

### 4. 胸外按压 (Chest Compression)

| 参数 | 标准 |
|------|------|
| **位置** | 两乳头连线中点（胸骨中下1/3处） |
| **深度** | 5-6 厘米 |
| **频率** | 100-120 次/分钟 |
| **回弹** | 每次按压后让胸廓完全回弹 |
| **中断** | 中断时间 < 10 秒 |
| **姿势** | 双手掌根重叠，十指相扣，手臂伸直，垂直下压 |

**节奏参考**：可默唱《Stayin' Alive》(Bee Gees)，节奏约104 BPM，恰好匹配。

### 5. 人工呼吸 (如经过培训并愿意执行)

- **按压:呼吸 = 30:2**
- 开放气道：仰头抬颏法 (Head Tilt-Chin Lift)
- 捏住鼻翼，口对口吹气，每次1秒，看到胸廓隆起为有效
- 如不愿或不能做人工呼吸，可仅做**纯胸外按压 (Hands-Only CPR)**

### 6. 持续CPR直到
- AED到达准备分析/除颤
- 专业急救人员接手
- 患者出现明显生命迹象（睁眼、自主呼吸、活动）
- 施救者精疲力竭

## 三、AED 使用流程

### AED 操作口诀："开-贴-插-电"

1. **开机 (ON)**：打开AED电源，跟随语音提示
2. **贴电极片 (PAD)**：
   - 一片贴右上胸（锁骨下方）
   - 一片贴左胸外侧（腋中线第5-6肋间）
   - 确保电极片紧贴皮肤，如有毛发需剃除
3. **插入插头 (PLUG)**：将电极片导线插头插入AED
4. **分析心律 (ANALYZE)**：AED自动分析，**所有人离开患者**
5. **电击 (SHOCK)**：
   - 如AED提示"建议电击"，确认所有人离开后按电击按钮
   - 电击后**立即恢复CPR**，从胸外按压开始
   - 如"不建议电击"，立即恢复CPR

### AED 特殊情况处理

| 情况 | 处理 |
|------|------|
| 胸部有汗/水 | 快速擦干，电极片需贴于干爽皮肤 |
| 有膏药贴 | 撕掉膏药，擦净残留药物 |
| 体内起搏器 | 电极片离起搏器至少2.5厘米 |
| 有胸毛 | 用AED包内剃刀剃除贴片区域 |
| 儿童 (1-8岁) | 使用儿童电极片或儿童模式 |
| 孕妇 | 与成人操作相同，AED对胎儿安全 |

## 四、婴儿 CPR（< 1岁）

| 项目 | 方法 |
|------|------|
| 按压位置 | 两乳头连线中点稍下方 |
| 按压方式 | **双指法**（食指+中指）或双拇指环抱法 |
| 按压深度 | 约4厘米（胸廓前后径1/3） |
| 按压频率 | 100-120 次/分钟 |
| 按压:呼吸 | 30:2（单人）/ 15:2（双人） |
| 人工呼吸 | 口对口鼻，轻吹气，见胸廓微隆即可 |

---

## 五、儿童 CPR（1岁-青春期）

| 项目 | 方法 |
|------|------|
| 按压方式 | 单手或双手掌根（视体格） |
| 按压深度 | 约5厘米 |
| 其余参数 | 同成人标准 |

---

## 六、高质量CPR关键指标

- ✅ 按压分数 (CCF) > 60%（CPR总时间中按压时间占比）
- ✅ 按压中断每次 < 10秒
- ✅ 避免过度通气（成人每次500-600ml潮气量）
- ✅ 每2分钟轮换按压者（避免疲劳导致按压质量下降）

---

## 七、何时停止CPR

**绝不轻易停止。** 仅在以下情况停止：
1. 专业急救人员到场接手
2. 患者恢复自主呼吸和意识
3. 环境变得不安全
4. 施救者体力耗尽无法继续
5. 医生宣布死亡

**注意**：即使患者看似无望，也要持续CPR直到AED建议不除颤且无生命迹象持续20分钟以上。
