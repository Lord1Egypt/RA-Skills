# 全域内容数据分析 📊

## 触发条件
- 分析小红书/抖音/B站/微博/视频号内容数据
- 了解爆款规律和特征
- 对标竞品内容策略
- 优化发布时间/标题/标签
- 了解平台算法推荐机制
- 复盘已发布内容
- 跨平台数据对比

## 核心流程（8 Steps）

### Step 1: 数据采集与账号诊断
- 粉丝/内容数/获赞收藏/爆款率/平均互动率/涨粉效率
- 识别账号当前阶段（冷启动/成长期/成熟期/瓶颈期）

### Step 2: 爆款特征解构
- 标题模式（10种高点击公式）
- 封面风格（对比型/数据型/情绪型/悬念型）
- 内容结构（钩子→痛点→方案→证据→CTA）
- 互动分析：赞藏比>0.3为优，评论/收藏深度分析
- 时间分析：发布时段×互动率热力图

### Step 3: 算法解读
**小红书CES**：
- CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
- 流量池递推：200→1000→5000→10000+

**抖音推荐机制**：
- 完播率×点赞率×评论率×转发率→综合评分
- 200-500初始池→1000-5000→1万-10万→10万+

**B站推荐**：
- 三连率（点赞+投币+收藏）×弹幕密度×完播率
- 频道权重+用户画像匹配

### Step 4: 竞品对标
- 选对标账号（同领域/粉丝略高/有爆款）
- 对比：更新频率/爆款率/内容类型/互动数据/变现模式
- 竞品近期策略变化追踪

### Step 5: 跨平台对标（新增）
- 同一内容在不同平台的表现差异分析
- 各平台算法偏好对比（小红书重收藏/B站重三连/抖音重完播）
- 内容改编策略：一鱼多吃，核心内容×4平台适配

### Step 6: 机会识别
- 低竞争话题 / 内容空白 / 可复制爆款模式
- 季节性选题 / 标签策略
- 跨平台流量洼地识别

### Step 7: 运营优化建议
- 内容方向 / 发布节奏 / 标题优化 / 互动策略
- 30天提升计划（周维度拆解）
- A/B测试方案（标题/封面/发布时间）

### Step 8: 数据复盘与迭代
- 每篇内容发布48h后自动复盘
- 关键指标 vs 预期对比
- 下篇调整建议（选题/标题/封面/时间）
- 建立个人内容数据飞轮

## 核心指标基准
| 指标 | 优秀 | 良好 | 需改进 |
|------|------|------|--------|
| 互动率 | >8% | 5-8% | <2% |
| 赞藏比 | >0.5 | 0.3-0.5 | <0.15 |
| 爆款率 | >20% | 10-20% | <5% |
| 完播率(视频) | >60% | 40-60% | <25% |
| 粉丝转化率 | >3% | 1-3% | <0.5% |

## 10种高点击标题公式
1. **数字冲击**："3天涨粉1万的秘密"
2. **对比反差**："月薪3千 vs 月薪3万的区别"
3. **痛点直击**："为什么你的内容没人看？"
4. **悬念钩子**："90%的人不知道的XX技巧"
5. **权威背书**："XX大V都在用的方法"
6. **时间限定**："2026年最新/本月必看"
7. **身份认同**："做XX的人一定要看"
8. **利益承诺**看完这篇省下XX元"
9. **反常识**："XX其实是错的"
10. **情绪共鸣** "被XX气到了/终于有人说了"

## 边界约束
1. 仅分析公开数据
2. 算法持续变化，月度更新基准
3. 不同品类基准不同（美妆/科技/母婴差异大）
4. 内容质量始终是根本
5. 跨平台对比需注意用户画像差异

## Anti-rationalization 反借口表
| 常见借口 | 应对 |
|---------|------|
| "没有数据就不分析了" | 用公开搜索数据+竞品估算代替 |
| "算法变了基准没用了" | 每月更新基准，标注有效期 |
| "跨平台对比太麻烦" | 先做同平台深度分析，再横向对比 |
