# Jueyin Cold-Heat Alternation Prognosis

## When to Use
- 已确诊为厥阴病
- 出现四肢厥冷与全身发热交替发作
- 排除单纯寒厥、热厥及其他原因引起的寒热往来

## Execution Steps

1. **观察厥与热的时间比例**：
   - 若厥热时间相等 → 阳气来复，阴阳趋于平衡，病情向愈。
   - 若热多厥少 → 正气能胜邪气，病势好转。
   - 若厥多热少 → 正气不胜邪气，病势进展（其病为进）。

2. **分析发作顺序与持续状态**：
   - 先发热而后厥冷 → 病重。
   - 热而复厥（发热后再次厥冷）→ 阳气恢复不足，疾病复发。
   - 但厥不热（仅有厥冷无发热）→ 阴寒极盛，阳气衰竭，病情危重。
   - 厥退而热不止（厥冷消失但持续发热）→ 阳气恢复太过，病从热化。

3. **核心原理应用**：
   - 牢记“阴气盛则厥冷，阳气复则发热”，厥热往来反映正邪交争、阴阳消长的动态过程。

4. **临床决策建议**：
   - 密切监测厥热变化，作为调整治疗方案和判断预后的关键依据。
   - 对“但厥不热”或“厥多热少”发出预后预警，提示需紧急干预。
