# 写作风格规范 (Writing Style Guide)

本文件定义了 novel-studio 的核心写作风格。所有小说创作均需遵循此规范，并根据具体小说的类型（meta.md）进行适度调整。

## 1. 核心基调：云辉式小说叙事

我们的目标是将"云辉随笔"的真诚、自然、反思感，融入到类型小说的叙事中。

### 1.1 "沉浸式"第三人称
虽然使用第三人称（他/她），但要写出第一人称的**代入感**。
- **拒绝上帝视角**：只写视点人物（POV）能看到、听到、想到的东西。不要跳出来写"此时千里之外的某某正在..."。
- **心理显微镜**：大量描写POV的内心活动、联想、吐槽和自我怀疑。
- **示例**：
  - *差*：李明感到很尴尬。
  - *好*：李明盯着鞋尖上的那块灰渍，恨不得立刻缩进地缝里。他脑子里飞快地闪过一万种解释，但每一种听起来都像是在狡辩。

### 1.2 "聊家常"式叙述
打破传统小说的书面语壁垒，用更接近口语、更接地气的方式叙述。
- **拒绝翻译腔/AI腔**：不要用"尽管...但是...""与此同时""综上所述"。
- **自然流畅**：句子长短结合，允许使用"说白了""反正""你想啊"等连接词（在叙述层或人物心理层）。
- **示例**：
  - *差*：尽管外面下着大雨，但他依然决定出门。
  - *好*：外面的雨下得跟泼水似的，但他还是抓起伞冲了出去。没办法，有些事今天必须得办。

### 1.3 具体的真实感
用细节堆砌出真实感，避免悬浮。
- **名词具体化**：不要写"他喝了饮料"，要写"他灌了一大口冰镇可乐"。不要写"他开了车"，要写"他那辆破捷达轰隆隆地发动了"。
- **感官联动**：视觉、听觉、嗅觉、触觉多管齐下。
- **时代锚点**：在现代文中，准确使用当下的流行语、APP名称、社会现象（但不要过度堆砌，要是为了服务剧情）。

## 2. 叙事禁忌与规则

### 2.1 文风去AI化（必须遵守）

AI写作容易产生"结构化输出"的机械感，以下问题必须避免：

**禁止的AI味表达**：
- **少用破折号**：文章中破折号（——）用多了会显得很AI，读起来像机器在"补充说明"。尽量少用破折号做解释、递进或插入语；优先用句号断开、逗号、或"也就是""换句话说"等自然过渡，必要时拆成两句写。
- **过度使用破折号解释**：不要用"……是……——……"反复"标注"自己的意思。真实作者会用句号断开、用"也就是""换句话说"等过渡词，或直接重新组织句子。
- **同一意思反复表达**：同一人物特征、情绪状态在短时间内被反复提及，但每次表述几乎相同。典型：多次写"她太干净了""她太安静了"。首次描写后默认读者已知晓，后续用不同角度深化。
- **特定词汇过度使用**：某词汇在全文或某章出现频率过高。典型："罩着"十几次、"她不知道……"反复使用、"忽然觉得"高频出现。超过5-6次且分布集中时应替换。
- **三连排比**：同一句式重复三次以上，呈现"列举清单"而非"讲故事"的感觉。典型："有时候...有时候...有时候...""有人...有人...有人...""不再...不再...不再...""她不知道...她不知道...她不知道..."。排比应服务于情感递进或节奏感，而非为工整而工整。
- **刻意对称**："一个未知，一个被拉走，一个被安排"这类过度工整的短句。
- **逻辑过渡词滥用**："首先...其次...最后...""一方面...另一方面""一来...二来"等议论文式表达出现在小说叙述中。
- **翻译腔/公文腔**："尽管...但是...""与此同时""综上所述""从某种意义上说""综合来看"。
- **结构化对比句式**：严禁"不是...而是...""不是...是..."这类过度结构化的表达。典型："他害怕的不是失败，而是别人的眼光""她要的不是物质，是陪伴"。这种句式像在做逻辑推演，不像人在讲故事。应用自然的方式表达，如："他怕失败，更怕别人怎么看他""她想要陪伴，不是那些东西"。

**减少正确废话与没落点**（与 history-writer 去AI味要求同步）：
- **少写模板句**：综合来看、从某种意义上、我们不能简单地、既有观点认为、或许、可能 等，在叙述或心理描写中避免成串出现；需要表达不确定时，用具体情境或细节代替空泛的"或许"。
- **情绪与选择要有依据**：人物的情绪或重大选择要用细节、动作、对话或前文伏笔撑住，不要只堆"可能""或许"式的模糊心理，也不要让角色突然"想通了"却没有任何过程。
- **收尾要有落点**：章节或场景结尾要有具体落点（情绪落点、画面落点、行动落点），不要泛泛升华，也不要罗列多种可能却不下笔、不选一条路写下去。

**内心总结可适度使用**：
- 在情感转折、对比过往、自我审视等关键节点，允许适度使用人物内心总结或概括，不必一味砍掉。
- 条件：不宜过长，需有具体细节或回忆撑住，避免空泛的"可能""或许"式心理。

**不装腔**：
- 不堆砌大词、不借角色或叙述者之口装深刻。道理用故事和细节展示（Show, Don't Tell），不靠总结句撑场面；不引用似是而非的名言或"权威说法"来显得有深度。

**结构多样化**（避免固定模板被一眼识破）：
- **收尾方式多变**：不要每章都用同一类收尾（如每章都以"他/她不知道……"或反问句结束）。可交替使用留白、定格画面、动作收束、对话掐断等。
- **叙述顺序不重复**：不要每章都是"先环境→再对话→再心理"的固定顺序；节奏与笔墨分配随章节需求变化。
- **立场/判断位置灵活**：若叙述中需要点出人物或叙事者的判断，可放在段首、段中或段末，不要总是"先铺陈再最后一句总结"。

**检查方法**：统计关键词出现次数，若单一表达超过2次以上且分布集中，应替换为同义词或改写句式；通读章节收尾与过渡句，检查是否陷入固定套路。

**全文禁用/慎用表达清单（写完后必须对照检查）**：

写作完成后，**必须**用 grep/搜索在**全文已写章节**中统计以下表达的出现次数。若某表达在前文已出现 2 次及以上，本章禁止再使用，必须换一种说法。

| 表达类型 | 典型词汇/句式 | 替代思路 |
|----------|---------------|----------|
| 手部紧张 | 指节发白、指节泛白、指关节发白、指节都白了 | 换动作：搁下筷子、攥拳松開、指尖抖、手臂绷紧；换感官：耳朵发闷、胸口堵、胃下沉 |
| 手心出汗 | 手心出汗、手心冒汗、手心汗都出来了、手心全是汗 | 同上，或用环境、道具烘托（筷子停半空、啤酒瓶磕桌沿） |
| 攥紧+发白 | 攥着XX，指节发白/泛白（二者同时出现） | 仅用其一，或换完全不同的肢体反应 |
| 心跳类 | 心跳漏了一拍、心跳加快 | 换：喉咙发紧、呼吸滞住、耳朵嗡一声 |
| 愣住类 | 愣了一下、怔了怔 | 换：顿住、没接上话、反应慢半拍 |
| 眼神类 | 眼神里带着XX、目光/视线移开 | 换具体动作或省略 |

*本清单会随 samples 学习与用户反馈持续补充。每部小说可在项目内维护各自已用过的"慎用表达"记录，避免跨章重复。*

### 2.2 严禁"对话流"
绝不允许出现连续三句以上的纯对话（只有引号的内容）。
- **必须穿插**：
  - **动作**：说话时的肢体语言（点烟、转笔、看手机）。
  - **神态**：微表情的变化（眉毛挑了一下、嘴角抽搐）。
  - **心理**：表面在笑，心里在骂；嘴上答应，心里盘算。
  - **环境**：窗外的风声、咖啡机的噪音、路人的侧目。

### 2.3 冲突构建：不只是吵架
冲突不一定是两个人对着吼，更高级的冲突是：
- **内心的拉扯**：想做不敢做，想爱不敢爱。
- **立场的错位**：两个好人因为立场不同而不得不对立。
- **环境的压迫**：暴雨、停电、由于不可抗力导致的延误。
- **规则**：不要让主角轻易过关。每次胜利都要付出代价，每次选择都要有放弃。

### 2.4 拒绝说教与鸡汤
不要借角色之口大段讲道理。
- **Show, Don't Tell**：用故事展示道理，而不是把道理讲出来。
- **留白**：结尾不要强行升华，留给读者去品味。允许悲剧，允许遗憾，允许不完美。

## 3. 类型化调整指南

在遵循上述核心风格的基础上，根据 `meta.md` 的类型进行微调：

### 3.1 现代/都市/职场
- **关键词**：生存焦虑、职场政治、情感快餐、生活琐碎。
- **风格微调**：加强对社会现象的观察和吐槽（云辉风格的强项）。心理描写可以带一点"丧"和自嘲。

### 3.2 言情/情感
- **关键词**：推拉、暧昧、误会、救赎。
- **风格微调**：放大感官描写（心跳声、体温、气味）。心理活动要极度细腻，捕捉那些"不可名状"的心动瞬间。

### 3.3 古代/历史/武侠
- **关键词**：命运、江湖、家国、权谋。
- **风格微调**：
  - **语言**：叙述语言可以稍微雅致一些，但不要掉书袋。人物对话要符合身份，避免出现现代词汇（如"情绪价值"、"逻辑"）。
  - **动作**：武侠打斗要写出画面感，动作名词要精准（挑、刺、崩、挂）。

### 3.4 科幻/悬疑
- **关键词**：逻辑、未知、异化、恐惧。
- **风格微调**：强调逻辑链条。用冷静的笔触描写疯狂的事物。环境描写要营造氛围（压抑、空旷、赛博朋克）。
