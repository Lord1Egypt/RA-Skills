## ✨ 核心亮点

* 🎯 **飞书生态深度优化**
  完美适配飞书单/多 Agent 场景。智能识别 `user_id` 与 `chat_id`，确保私聊、群聊精准触达。
* 🛡️ **告别拥堵与不稳定**
  强制引入 **防并发打散** (`stagger`) 与 **隔离运行** (`isolated`) 机制。彻底告别任务冲突，保障每一条指令稳定执行。
* 🧠 **时区与格式免操心**
  自动处理 UTC 时间换算与标准 Cron 表达式生成。彻底告别手写 `jobs.json` 带来的格式灾难。
* 🔒 **坚如磐石的底层规范** *(New)*
  全面接管 CLI 指令创建流程，自动规避系统 `crontab` 与延时脚本 (`sleep`) 带来的不可控风险。

## 🚀 快速开始与避坑指南

为获得最完美的使用体验，请务必遵循以下 **“三步走”** 黄金法则：

### 1️⃣ 唤醒与初始化
安装或更新此技能后，请执行：
* **重启系统**，并输入 `/new` 开启全新对话，确保技能完全生效。
> ⚠️ **建议**：在 `TOOLS.md` 中增加定时任务指定用当前 skill 来完成的相关语句；同时首次使用前，先让虾仔细阅读一次本地的 `TOOLS.md`，确保它不使用系统 crontab、延时脚本等非 openclaw 官方方式。

### 2️⃣ 标准工作流：创建与确认
无需了解复杂的命令，直接用自然语言下达指令即可。
* **🗣️ 话术示例**：*“工作日每天上午 10 点，提醒群里开站会。”*
> ✅ **强制核对（避坑核心）**：每次让助手创建完任务后，**建议让它主动列出一下「当前的定时任务配置列表」**。花 10 秒钟确认一下**时间、渠道、时区**是否完全符合预期。
