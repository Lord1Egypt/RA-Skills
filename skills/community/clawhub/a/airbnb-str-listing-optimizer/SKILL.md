## What This Skill Does

Turns raw property details into a fully optimized short-term rental listing that ranks higher in Airbnb and VRBO search, converts browsers into bookers, and automates guest communication — all without hiring a professional copywriter.

**Four-prompt system:**

| # | Prompt | Output |
|---|--------|--------|
| 1 | **Listing Headline + Description** (FREE) | 10-word hook title + 400–500 word SEO description with neighborhood story, amenity highlights, and STR-specific keywords |
| 2 | **Photo Caption Bank** | 15–20 captions keyed to room/feature, each with an emotional hook and conversion line |
| 3 | **House Rules + Guest FAQ** | Platform-formatted house rules (Airbnb/VRBO compliant) + 10-question FAQ that pre-empts 90% of guest questions |
| 4 | **Automated Guest Message Templates** | 6-message sequence: booking confirmation, pre-arrival, check-in day, mid-stay check-in, checkout reminder, post-stay review request |

## Inputs Required

**For Prompt 1 (Headline + Description):**
- Property type (condo, house, cabin, villa, etc.)
- Number of bedrooms/bathrooms/max guests
- Location (city, neighborhood, distance to key landmarks)
- Top 5 amenities (pool, hot tub, fast wifi, EV charger, etc.)
- Unique selling point (views, walkability, theme, design style)
- Target guest (families, couples, business travelers, groups)

**For Prompt 2 (Photo Captions):**
- List of 15–20 photo subjects (master bedroom, kitchen, pool, etc.)
- Property name or vibe (e.g., "The Desert Oasis")
- Primary guest emotion to evoke (relaxation, luxury, adventure)

**For Prompt 3 (House Rules + FAQ):**
- Check-in/checkout times
- Pet policy, smoking policy, party/event policy
- Parking details, noise restrictions (quiet hours)
- Any property-specific rules (pool hours, trash day, etc.)

**For Prompt 4 (Guest Messages):**
- Property name and host first name
- Check-in method (lockbox code, smart lock, keypad)
- WiFi name and password
- Parking instructions
- Checkout instructions (key return, trash, etc.)
- Any local recommendations (restaurants, attractions)
