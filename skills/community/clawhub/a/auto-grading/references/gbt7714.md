# GB/T 7714-2015 参考文献格式速查

## 常见类型格式模板

| 类型 | 格式 |
|------|------|
| 期刊 [J] | `作者. 标题[J]. 期刊名, 年, 卷(期): 页码.` |
| 专著 [M] | `作者. 书名[M]. 出版地: 出版社, 年.` |
| 会议 [C] | `作者. 标题[C]//编者. 论文集名. 出版地: 出版社, 年: 页码.` |
| 学位论文 [D] | `作者. 标题[D]. 出版地: 学校, 年.` |
| 网络资源 [EB/OL] | `作者. 标题[EB/OL]. (发布日期)[引用日期]. URL.` |
| 报告 [R] | `机构. 标题[R]. 出版地: 出版社, 年.` |
| 其他 [Z] | `作者. 标题[Z]. 出版地: 出版者, 年.` |

## 格式规则强制项

1. **期号不补零**：`(1)` 非 `(01)`
2. **英文作者**：姓全大写，名缩写。`STOKES P` 非 `Stokes, P.`
3. **分隔符**：中文文献用全角标点，英文用半角；出版地后接 `:`
4. **会议论文**：必须用 `[C]//` 格式，禁止用 APA 的 `In... (Eds.)`
5. **[EB/OL]**：必须含 `(发布日期)[引用日期]. URL.`，三个要素缺一不可
6. **英文标题**：统一 sentence case（仅首词和专有名词大写）

## 常见错误检测

- `(01)` → 应为 `(1)`
- `Stokes, P.` → 应为 `STOKES P`
- `上海：上海交通大学` → 应为 `上海: 上海交通大学`
- `In A. Moreman (Ed.),` → 应为 `[C]//MOREMAN A.`
- `[EB/OL]. 2024-04-20.` → 缺少 URL 和引用日期
