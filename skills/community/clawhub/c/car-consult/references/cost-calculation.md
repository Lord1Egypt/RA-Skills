# 用车成本计算

## 一、购车成本

### 落地价构成
```
落地价 = 裸车价 + 购置税 + 保险 + 上牌费 - 优惠
```

### 购置税
| 车型 | 税率 | 计算方式 |
|------|------|----------|
| 燃油车 | 10% | 裸车价 ÷ 11.3 |
| 新能源（≤30万） | 0% | 免征 |
| 新能源（>30万） | 10% | 裸车价 ÷ 11.3 |

### 保险费用（第一年）

| 车价 | 交强险 | 商业险 | 合计 |
|------|--------|--------|------|
| 10万 | 950 | 4000-5000 | ~5500 |
| 15万 | 950 | 5000-6000 | ~6500 |
| 20万 | 950 | 6000-8000 | ~7500 |
| 30万 | 950 | 8000-10000 | ~10000 |

**新能源车保费**：通常比同级燃油车高15-20%

### 上牌费
- 4S店代办：500-2000元
- 自己办：200-300元

## 二、能源成本

### 燃油车
```
年油费 = 年里程 ÷ 100 × 油耗 × 油价
```

示例（年2万公里，8L/100km，8元/L）：
```
20000 ÷ 100 × 8 × 8 = 12,800元/年
```

### 新能源车

**纯电：**
```
年电费 = 年里程 ÷ 100 × 电耗 × 电价
```

示例（年2万公里，15kWh/100km，家充0.5元/kWh）：
```
20000 ÷ 100 × 15 × 0.5 = 1,500元/年
```

**插混/增程：**
- 纯电模式：同上
- 亏电油耗：参考燃油车计算

### 能源成本对比（年2万公里）

| 类型 | 能耗 | 年成本 | vs 燃油车 |
|------|------|--------|-----------|
| 燃油车 | 8L/100km | 12,800元 | 基准 |
| 纯电（家充） | 15kWh/100km | 1,500元 | 省11,300元 |
| 纯电（快充） | 15kWh/100km | 4,500元 | 省8,300元 |
| 插混（50%纯电） | 4L/100km | 6,400元 | 省6,400元 |

## 三、保险成本

### 年度保费递减

| 年份 | 折扣 | 估算保费 |
|------|------|----------|
| 第1年 | 无 | 6000-8000 |
| 第2年 | 9折 | 5400-7200 |
| 第3年+ | 7折 | 4200-5600 |

**新能源车**：保费普遍高15-20%，且涨价风险更高（电池理赔）

## 四、保养成本

### 燃油车
| 项目 | 周期 | 费用 |
|------|------|------|
| 小保养 | 5000-10000km | 400-800元 |
| 大保养 | 40000km | 2000-4000元 |
| 年均 | - | 1500-3000元 |

### 新能源车
| 项目 | 周期 | 费用 |
|------|------|------|
| 常规检查 | 10000-20000km | 200-400元 |
| 更换空调滤芯 | 1年 | 100-200元 |
| 刹车油/冷却液 | 2-3年 | 500-800元 |
| 年均 | - | 500-1000元 |

**优势**：新能源保养便宜约50%

## 五、折旧成本

### 3年保值率参考

| 类型 | 保值率 | 说明 |
|------|--------|------|
| 日系燃油车 | 65-75% | 保值标杆 |
| 德系燃油车 | 60-70% | 次之 |
| 国产燃油车 | 50-60% | 一般 |
| 特斯拉 | 55-65% | 新能源标杆 |
| 蔚来/理想/问界 | 50-60% | 较好 |
| 其他新能源 | 40-55% | 差异大 |

### 折旧计算
```
年折旧 = 裸车价 × (1 - 保值率) ÷ 年数
```

示例（20万车，3年保值60%）：
```
20万 × 40% ÷ 3 = 26,666元/年
```

## 六、综合成本对比

### 年用车成本（2万公里/年）

| 成本项 | 燃油车 | 纯电 | 插混 |
|--------|--------|------|------|
| 能源 | 12,800 | 1,500 | 6,400 |
| 保险 | 5,000 | 6,000 | 5,500 |
| 保养 | 2,000 | 800 | 1,200 |
| 折旧 | 25,000 | 30,000 | 25,000 |
| **合计** | **44,800** | **38,300** | **38,100** |

**结论**：新能源综合成本略低，但折旧风险更大
