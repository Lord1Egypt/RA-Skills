# 报告输出模板

## 类型一：快评（~500字）

**适用场景**：突发政策、财报速评、重大事件  
**时间目标**：15分钟内出品  
**交付结构**：

```
1句核心判断（最精炼的结论，不含修饰）

支撑1：关键数据/逻辑（含数据来源）
支撑2：关键数据/逻辑
支撑3：关键数据/逻辑

1句影响预判（对行业/公司的实质影响 + 概率/幅度）
```

**示例**：

> 乖宝宠物2025年财报："增收降利"是表面，品牌投入的边际效率才是核心问题。
>
> 营收67.69亿（+29%），但利润增速仅7.75%，差距21.25pp被销售费用吞噬。销售费用率22.56%，同比+3.8pp，弗列加特+80%增长的代价。好消息是自主品牌占比升至73.4%、经营现金流8.42亿覆盖在建工程绰绰有余。
>
> 关键变量：销售费用率何时触顶？参照人用保健品电商转型经验（2015-2019），费用率高企期通常持续2-3年后面临拐点。如果2026H2费用率不再增长，估值框架可切换。

## 类型二：深度报告（3000-5000字）

**适用场景**：行业首次覆盖、深度分析请求  
**交付结构**：

```
执行摘要（500字）
  - 核心判断（1-2句）
  - 关键数据摘要（3-5个）
  - 投资主线/结论

第一章：行业概况
  规模/增速/CAGR | 生命周期阶段 | 产业链结构 | 核心驱动因素 | 全球对标

第二章：竞争格局
  市场集中度 | 波特五力 | 品牌梯队（至少分3层） | 竞争壁垒 | 全球对标

第三章：品牌深度（若有足够信源）
  逐品牌拆解（定位/品类/财务/渠道/战略） | 战略原型分类 | 竞争热力图

第四章：商业模式与渠道
  盈利模式 | 成本结构 | 渠道利润分配 | 护城河分析

第五章：财务深度（若有上市公司对标）
  核心指标横向对比表 | 增收降利原因分析 | ROIC估算 | 估值框架

第六章：监管与政策
  监管框架 | 准入要求 | 政策稳定性 | 跨国对比

第七章：风险与趋势
  风险矩阵（概率×影响×触发条件） | 确定性趋势 | 大概率方向 | 关键变量

结论与投资启示
  - 核心结论（回扣执行摘要）
  - 值得关注标的及逻辑
  - 应警惕类型
  - 品牌终局预判

附录：信息来源（Tier 1/2/3分级）
```

**必须要求**：
- 每章配 [图表建议]：图表类型 | 标题 | 数据序列 | X/Y轴
- 风险章节不走形式，必须概率×影响实质评估
- 区分短期/中期/长期风险，配触发条件
- 每个判断有数据或逻辑链支撑
- 不确定处明确说明

## 类型三：数据简报（表格为主）

**适用场景**：数据对比、行业指标追踪  
**时间目标**：30分钟出品  
**交付结构**：

```
核心结论（一行字，不超过50字）

数据表格
  - 表头清晰、单位统一
  - 来源标注每列下方
  - 关键数字加粗

简要解读（3-5条，每条不超过2行）
```

**表格标准**：
- 第一列为对比维度
- 后续列为各公司/各期数据
- 最后一行可加"排名/趋势"
- 来源标注：Tier 1/2/3

**示例**：

```
核心结论：四家宠物上市公司全部出现"增收降利"，但原因各不相同。

| 指标 | 乖宝宠物 | 中宠股份 | 佩蒂股份 | 路斯股份 |
|------|---------|---------|---------|---------|
| 营收 | 67.69亿 | 52.21亿 | 14.49亿 | ~7.8亿 |
| 营收增速 | +29% | +17% | -13% | +11% |
| 利润增速 | +8% | -7% | -37% | -12% |
| 毛利率 | 42.5% | 境内39%/境外24% | 31.9% | 20.9% |
| 销售费用率 | 22.6% | 13.5% | 5.7% | — |

来源：2025年年报 (Tier 1)

解读：
1. 乖宝：规模扩张甜蜜负担，销售费用率22.6%吞噬利润
2. 中宠：境内高歌猛进（+32%）vs 境外承压（毛利率仅24%）
3. 佩蒂：被关税精准打击，拐点待确认
4. 路斯：小而美性价比路线，毛利率约束天花板
```

## 类型四：对标分析（2000-3000字）

**适用场景**：公司vs竞争对手、中国vs海外  
**交付结构**：

```
横向对比表（核心指标一览）
  - 规模 | 增速 | 盈利能力 | 估值 | 战略定位

差距解读（3-5条，每条深挖两层"为什么"）
  - 差距不是结论，差距的来源才是

差距来源分析
  - 技术？品牌？供应链？制度？时机？
  - 哪些是可追赶的？哪些是结构性的？

追赶可能性判断
  - 时间窗口：多久可以追赶？窗口期有多长？
  - 必要条件：需要什么资源/条件？
  - 概率评估：高/中/低 + 依据
```

**对标维度建议**：

| 维度 | 中国市场 | 对标市场 | 差距 | 可追性 |

| 市场规模 | | | | |
| 渗透率 | | | | |
| CR3 | | | | |
| 渠道结构 | | | | |
| 监管框架 | | | | |
| 定价水平 | | | | |
