# 商事仲裁申请书

## 适用场景
合同/协议中约定了仲裁条款，发生商事纠纷后，直接向约定的仲裁委员会申请仲裁解决。

## 提交机构
合同约定的仲裁委员会（常见：中国国际经济贸易仲裁委员会CIETAC、北京仲裁委员会、上海国际仲裁中心等）

## 模板正文

```
                    仲裁申请书

申请人：______公司
        统一社会信用代码：________________
        住所地：____________________________
        法定代表人：______，职务：______
        联系电话：________________________
        指定送达地址：____________________

被申请人：______公司
          统一社会信用代码：________________
          住所地：____________________________
          法定代表人：______，职务：______
          联系电话：________________________

仲裁依据：
申请人依据双方于______年______月______日签订的《______合同》
第______条约定的仲裁条款，向贵会申请仲裁。

仲裁请求：
1. 裁决被申请人支付货款/违约金______元；
2. 裁决被申请人赔偿损失______元
   （计算方式：________________________）；
3. 裁决被申请人承担本案全部仲裁费用；
4. （如有）裁决被申请人承担申请人为本案支出的律师费______元。

事实与理由：
申请人与被申请人于______年______月______日签订了
《______合同》（编号：______），约定：________________________。

合同履行过程中，被申请人于______年______月______日___（陈述违约事实）。
经申请人多次催告，被申请人至今未_______________。

被申请人的行为已构成根本违约，根据合同第______条及
《中华人民共和国民法典》第______条规定，应承担相应违约责任。

综上所述，恳请贵会依法裁决，支持申请人的全部仲裁请求。

附：
1. 仲裁申请书副本______份；
2. 仲裁协议/仲裁条款（合同复印件）；
3. 证据清单及证据材料______份；

                                    ______年______月______日
```

## 注意事项
- **关键填写点**：写明仲裁依据（合同中仲裁条款的具体条款号）；仲裁请求金额需附详细计算方式和合同依据；证据材料需编号并按争议焦点分类
- **常见错误**：合同中无仲裁条款或仲裁条款无效仍申请仲裁；仲裁请求遗漏律师费（以合同有约定为前提）；被申请人信息不完整（尤其是有效送达地址）
- **份数要求**：正本1份 + 按仲裁员人数和对方当事人人数提交副本
- **时效提示**：一般诉讼时效3年，注意查看合同特别约定
- **一裁终局**：仲裁裁决作出后即生效，不得上诉（特殊情况可申请撤销）
