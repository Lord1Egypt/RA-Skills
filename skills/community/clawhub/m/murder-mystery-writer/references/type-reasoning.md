# 推理本创作指南

## 推理本的定义

推理本是剧本杀的原始形态——"谋杀之谜"。核心体验是通过逻辑推理破解案件，找出真凶。

## 本格 vs 变格

### 本格推理

**定义**：注重纯粹逻辑推理，所有超自然元素都不出现。

**核心要素**：
- 物理可行的犯罪手法
- 公平的线索展示
- 逻辑链完整，无漏洞

**适合人群**：推理小说爱好者、逻辑思维强的玩家

**代表作品**：《豪门惊情系列》、《紫藤夫人》

### 变格推理

**定义**：允许超自然元素，世界观可以是架空的。

**核心要素**：
- 需要详细解释世界观规则
- 诡计可以超越物理限制，但必须自洽
- 更考验想象力

**适合人群**：喜欢新奇设定、脑洞大开的玩家

**代表作品**：《年轮》、《漓川怪谈簿》、《死者在幻夜中醒来》

## 推理本结构模板

### 标准结构（4-5小时）

```
第一幕：阅读剧本 + 初步介绍（30-40分钟）
  - 每人阅读个人剧本
  - 自我介绍，建立初步关系

第二幕：第一轮讨论（40-50分钟）
  - 分享案发前的信息
  - 建立初步时间线
  - 第一轮搜证（发放表面线索）

第三幕：第二轮讨论（40-50分钟）
  - 分享新获取的线索
  - 修正时间线
  - 第二轮搜证（发中层线索）
  - 开始深入推理

第四幕：第三轮讨论（30-40分钟）
  - 关键线索公开
  - 最终推理
  - 投票指认凶手

第五幕：复盘（15-20分钟）
  - DM公布真相
  - 揭示所有隐藏信息
  - 讨论推理过程
```

### 硬核推理结构（6-8小时）

```
第一幕：案件呈现（30分钟）
  - 多个独立案件同时呈现
  - 每个案件有独立的嫌疑人

第二幕：独立案件推理（120-180分钟）
  - 逐个破解每个案件
  - 每个案件有独立的线索链

第三幕：案件关联（60分钟）
  - 发现案件之间的联系
  - 所有案件指向同一个真相

第四幕：核心推理（60分钟）
  - 破解核心诡计
  - 找出幕后黑手

第五幕：复盘（30分钟）
  - 完整真相揭示
  - 所有角色的完整故事
```

## 推理本设计要点

### 时间线设计

时间线是推理本的骨架。

**设计原则**：
1. 每个角色每分钟的行踪都要有记录
2. 关键时间点必须有交叉（多人在同一时间不同地点）
3. 时间线不能有漏洞（玩家会找到每一个矛盾）

**时间线表格模板**：

```markdown
| 时间 | 死者 | 角色A | 角色B | 角色C | 角色D | 角色E |
|------|------|-------|-------|-------|-------|-------|
| 19:00 | 大厅 | 大厅 | 房间 | 厨房 | 花园 | 书房 |
| 19:15 | 书房 | 大厅 | 走廊 | 厨房 | 花园 | 书房 |
| 19:30 | 书房 | 书房 | 大厅 | 大厅 | 走廊 | 大厅 |
| 19:45 | 死亡 | 走廊 | 大厅 | 大厅 | 大厅 | 大厅 |
| 20:00 | - | 大厅 | 大厅 | 大厅 | 大厅 | 大厅 |
```

### 动线设计

每个角色在案发时间段内的移动路线。

**设计原则**：
1. 动线必须有理由（不能无缘无故出现在某地）
2. 动线之间要有交叉点（制造证人/嫌疑）
3. 凶手的动线必须有破绽（但不明显）

### 嫌疑分配

**原则**：每个角色都有"看起来像凶手"的理由。

| 角色 | 表面嫌疑 | 真实情况 |
|------|----------|----------|
| A | 有动机+无不在场证明 | 有动机但没杀人 |
| B | 无动机+有不在场证明 | 不在场证明是假的 |
| C | 有动机+有不在场证明 | 不在场证明是真的 |
| D | 无动机+无不在场证明 | 实际是凶手 |
| E | 有动机+有不在场证明 | 动机是假的 |

## 推理本示例：6人本格推理本

### 《钟楼谜案》（示例）

**类型**：本格推理
**人数**：6人（3男3女）
**时长**：4-5小时
**背景**：民国·上海·钟楼酒店

**故事梗概**：
1937年上海，钟楼酒店举行慈善晚宴。晚宴期间，酒店老板被发现死在钟楼顶层，房门从内反锁，窗户紧闭，形成完美密室。所有嫌疑人都在晚宴现场，但每个人都有不可告人的秘密。

**角色设计**：
1. 林少爷（男）- 死者之子，纨绔子弟，实际在调查父亲的秘密
2. 张小姐（女）- 林少爷未婚妻，军阀之女，有退婚打算
3. 陈律师（男）- 林家法律顾问，知晓财务秘密
4. 林小姐（女）- 死者之女，嫉妒哥哥受宠
5. 王管家（男）- 服务林家30年，知晓所有秘密
6. 顾记者（女）- 调查林家黑幕，晚宴混入

**核诡**：利用钟楼的齿轮机关制造延时密室
**关键线索**：
- 钟楼齿轮有被改动的痕迹
- 案发前夜有人订购了特殊零件
- 死者手表停在10:47
- 电闸被人关闭过3分钟

**结局**：根据玩家推理结果，有凶手逃脱/被抓获两种结局
