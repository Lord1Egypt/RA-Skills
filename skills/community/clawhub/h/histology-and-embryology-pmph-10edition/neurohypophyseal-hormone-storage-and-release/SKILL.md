# 神经垂体激素贮存与释放机制

## 适用场景
- 分析抗利尿激素（ADH）或催产素的来源与释放路径
- 区分神经垂体与腺垂体的功能差异
- 解释尿崩症等与ADH分泌异常相关的病理机制
- 理解下丘脑-神经垂体轴的组织学基础

## 执行步骤

1. **确认激素合成部位**：下丘脑视上核和室旁核的神经内分泌细胞负责合成两种激素：
   - **血管升压素（抗利尿激素，ADH）**：调节水重吸收与血压
   - **催产素**：促进子宫收缩与乳汁排出

2. **包装与轴突运输**：激素在胞体内包装成分泌颗粒，沿轴突向神经垂体运输。

3. **识别赫林体**：运输过程中，分泌颗粒聚集成团，在光镜下呈弱嗜酸性串珠状膨大结构，称为赫林体（Herring body）。

4. **贮存于神经垂体**：激素最终贮存于神经垂体（神经部），而非在此合成。

5. **释放入血**：在生理刺激下，激素从神经末梢释放入有孔毛细血管，进入全身循环，作用于靶器官。

6. **注意支持结构**：神经垂体中的垂体细胞（pituicyte）为胶质细胞，起支持与营养作用，不参与激素合成。

## 关键约束
- 本机制**不适用于**腺垂体激素的合成与调控
- **不涉及**垂体门脉系统（属腺垂体通路）
- 前提条件：下丘脑神经内分泌细胞功能正常，且轴突运输通路完整

> 提示：若问题涉及激素合成、门脉调控或腺垂体功能，请勿使用本技能。
