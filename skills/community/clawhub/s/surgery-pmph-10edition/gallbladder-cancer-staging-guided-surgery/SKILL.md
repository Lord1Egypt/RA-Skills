# 胆囊癌分期导向手术治疗方案

## 适用条件
- 患者已通过病理确诊为胆囊癌
- AJCC TNM分期已明确
- 患者具备手术耐受能力

## 手术方案决策流程

### 1. 0期和I期
- **肿瘤特征**：癌局限于黏膜层或固有层，未侵犯肌层；常为意外发现（UGC）
- **手术方式**：单纯胆囊切除术，无需追加手术

### 2. ⅡA、ⅡB、ⅢA期
- **肿瘤特征**：癌已侵犯肌层或浆膜，可能累及胆囊床
- **手术方式**：胆囊癌根治性切除术，包括：
  - 胆囊切除
  - 肝Ⅳb段（方叶）和Ⅴ段切除 或 亚肝段切除
  - 胆囊引流区域淋巴结清扫（范围：胆囊淋巴结 → 胆总管周围 → 胰上 → 胰头后 → 肠系膜上动脉 → 肝动脉周围 → 腹主动脉旁）

### 3. ⅢB、ⅣA、ⅣB期
- **肿瘤特征**：癌侵犯邻近器官（如肝、十二指肠、胃、横结肠）或存在远处转移
- **手术方式**：胆囊癌扩大根治术，包括：
  - 肝右三叶切除
  - 必要时联合胰十二指肠切除

### 4. 无法切除的晚期病例（任何分期）
- **手术目标**：缓解症状（如黄疸、梗阻）
- **姑息性手术选项**：
  - 肝管空肠Roux-en-Y吻合内引流术（解除黄疸）
  - 经皮经肝或内镜胆管支架置入
  - 胃空肠吻合术（解除十二指肠梗阻）

## 辅助治疗建议
- 晚期病例推荐以吉西他滨为基础的化疗，常联合铂类药物
- 近年可考虑联合免疫治疗与靶向治疗

## 关键解剖基础
- 胆囊床与肝脏之间仅隔一层肌膜，早期即可直接侵犯肝脏
- 可经胆囊静脉发生肝内转移
