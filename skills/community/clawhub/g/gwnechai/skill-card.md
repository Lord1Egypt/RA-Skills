## Description: <br>
Gwenchai is a mixed ClawHub skill bundle advertised for solopreneur automation workflows, with packaged materials spanning browser automation, speech-to-text, memory workflows, search, messaging integrations, and setup scripts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and developers may use the release as a source of automation workflow guidance and agent setup materials. Because server security evidence flags the package as a mixed, high-authority bundle, review the included capabilities and credentials before installation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The package is advertised as solopreneur automation guidance but server security evidence says it contains several unrelated high-authority capabilities. <br>
Mitigation: Install only after manually reviewing the full artifact and removing unrelated scripts or capabilities. <br>
Risk: The bundle may touch local agent configuration, external APIs, workspace files, git history, scheduled jobs, long-term memory, and Discord or Telegram integrations. <br>
Mitigation: Use an isolated environment, grant least-privilege credentials, and confirm each file or integration boundary before execution. <br>
Risk: Server capability tags indicate crypto, purchase, and OAuth-token-related capability exposure. <br>
Mitigation: Avoid loading production credentials and disable purchase, crypto, and OAuth-dependent flows unless explicitly needed and approved. <br>
