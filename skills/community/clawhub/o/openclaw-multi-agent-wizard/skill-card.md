## Description: <br>
Guide beginners through OpenClaw multi-agent setup with a step-by-step wizard for creating agents, connecting Feishu bots, configuring routing, and verifying the result. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[majiabin2020](https://clawhub.ai/user/majiabin2020) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and OpenClaw users use this skill to set up beginner-friendly multi-agent OpenClaw workflows, including multi-bot and single-bot Feishu routing, starter agent profiles, and setup verification. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The security summary reports that the wizard asks users to paste Feishu app secrets into chat without adequate safety warnings. <br>
Mitigation: Do not paste Feishu App Secret into chat; enter it only through a local secure OpenClaw prompt or protected config path, redact logs before sharing, and back up existing agent workspaces before running starter profile scripts. <br>
