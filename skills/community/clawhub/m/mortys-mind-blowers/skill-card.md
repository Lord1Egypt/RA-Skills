## Description: <br>
Find an unusual, funny, bleak, uncanny, technical, or deep-cut memory entry and retell it as a short story in the style of Rick and Morty's "Morty's Mind Blowers." <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[inertia186](https://clawhub.ai/user/inertia186) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Agents use this skill when a user asks for a playful narrative recap of an unusual, funny, bleak, uncanny, technical, or deep-cut memory instead of a dry summary. It selects from memory files, transcripts, or logs and turns one supported entry into a short story with a traceable source note when practical. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may read memory files, transcripts, or logs that contain secrets or highly sensitive personal information. <br>
Mitigation: Use it only with memory stores appropriate for playful recap, avoid highly sensitive sources, and summarize rather than reveal sensitive details. <br>
Risk: Stylized storytelling can make a memory sound more certain or dramatic than the source supports. <br>
Mitigation: Keep the recap faithful to the memory file, distinguish user recollection from file evidence, quote only tiny supported fragments, and include a source note when practical. <br>
