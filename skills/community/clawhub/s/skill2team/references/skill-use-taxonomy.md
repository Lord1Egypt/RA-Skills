# Skill Use Taxonomy

Use this taxonomy to classify skills before assigning them to agents.

| Category | Examples |
|---|---|
| Orchestration and architecture | team design, state manager, work queue |
| Research and knowledge work | web research, literature review, market research |
| Data collection and extraction | PDF extraction, spreadsheet parsing, API pull, database query |
| Evidence verification and fact governance | fact checking, source ranking, claim ledger, conflict resolution |
| Analysis and decision support | risk analysis, comparison, prioritization, recommendation |
| Software engineering | coding, tests, debugging, DevOps, security review |
| Data science and analytics | statistics, visualization, ML, dashboarding |
| Documentation and content creation | reports, emails, PDFs, slides, editing |
| Creative media and design | brand, UX, image prompts, scripts |
| Education and coaching | lesson plans, quizzes, learning paths, tutoring |
| Business operations | meeting notes, SOPs, project plans, process automation |
| Marketing, growth, and sales | SEO, ads, social media, sales research |
| Customer support and communication | FAQ, ticket triage, complaint escalation |
| Lifestyle and entertainment | travel, food, fitness, games, recommendations |
| High-impact domains | legal, finance, medical, education, hiring, safety |
| Tool/API execution | sending messages, updating records, deploying, scheduling |
| QA, review, and acceptance | final review, quality gates, risk sign-off |
