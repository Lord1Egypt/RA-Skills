# Return Reducer — Implementation Checklist

Use to track the status of all return-reduction actions. Each section should be reviewed before declaring the optimization complete.

---

## 1. Data & Measurement Setup (7 items)
- [ ] 90-day return data exported with reason tags
- [ ] Return reason taxonomy updated to 8–12 specific categories
- [ ] All existing return records re-classified into new taxonomy
- [ ] Return rate calculated by SKU (not just overall)
- [ ] Return rate calculated by return reason
- [ ] Full cost per return calculated (see return-cost-calculator.md)
- [ ] Baseline metrics documented (before any changes)

## 2. High-Return SKU Identification (4 items)
- [ ] Top 5 SKUs by return rate identified
- [ ] Top 5 SKUs by total return cost (return count × cost/return) identified
- [ ] Root cause confirmed for each high-return SKU (not assumed)
- [ ] Expected return rate reduction estimated for each action

## 3. Listing Updates — Photos (6 items)
- [ ] Scale reference photo added to all high-return SKUs
- [ ] Lifestyle photo added (product in use, realistic context)
- [ ] Texture/material close-up added
- [ ] Model measurements stated in caption for all apparel (height, weight, size worn)
- [ ] All color variants have distinct, accurate photos
- [ ] Before/after screenshot of listing taken for A/B tracking

## 4. Listing Updates — Copy (7 items)
- [ ] Exact product dimensions added (L × W × H in inches/cm)
- [ ] Real-world size comparison added ("about the size of...")
- [ ] Material composition listed explicitly (fabric %, material type)
- [ ] "What's in the box" section added
- [ ] Compatibility statement added (works with / does NOT work with)
- [ ] Color accuracy disclaimer added where relevant
- [ ] Care and maintenance instructions added

## 5. Size Guidance (Apparel/Footwear only) (5 items)
- [ ] Size chart updated with actual body measurements (not just size letters)
- [ ] "How to measure" guide created and linked
- [ ] Fit descriptor added ("Runs true to size / Runs small — size up")
- [ ] Customer review sizing quotes pulled and added to listing
- [ ] Model measurements updated to current model/photo set

## 6. Post-Purchase Email Sequence (5 items)
- [ ] Day-1 email created: order confirmation + product tips
- [ ] Day-7 email created: usage tips, care guide, social proof
- [ ] Day-14 email created: check-in, proactive support, FAQ
- [ ] All 3 emails activated in ESP (Klaviyo / Mailchimp / etc.)
- [ ] Return rate tracking set up for orders with vs. without email sequence

## 7. Packaging Improvements (4 items, if applicable)
- [ ] Damaged return products photographed and root cause identified
- [ ] Packaging drop test conducted on revised packaging
- [ ] Packaging change costed and ROI calculated
- [ ] New packaging ordered and in production

## 8. Return Policy Optimization (5 items)
- [ ] Return reason photo requirement added to return portal
- [ ] Exchange incentive activated (credit amount + positioning)
- [ ] Return window reviewed vs. actual return timing data
- [ ] Return portal tested end-to-end as a customer would experience it
- [ ] Policy changes communicated to customer service team

## 9. Measurement & Review (5 items)
- [ ] Tracking tags added to measure impact of each change (UTM, SKU tags)
- [ ] 30-day post-change return rate review scheduled
- [ ] 90-day full audit review scheduled
- [ ] Return rate targets set and approved by relevant stakeholders
- [ ] Findings from this audit documented and shared with product development team

---

**Total items: 48**
**Minimum to report results: Section 1 complete (baseline established)**
**Minimum to launch return-reduction campaigns: Sections 1–5 complete**
**Full optimization: All sections complete**
