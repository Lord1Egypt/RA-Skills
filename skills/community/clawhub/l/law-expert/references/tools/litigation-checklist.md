# 诉讼准备自检清单 (Litigation Checklist)

> 使用说明：在启动诉讼程序前，逐项确认以下问题。如任一项为「否」，需慎重考虑是否继续推进诉讼。

---

## 一、诉前准备（6项）

| # | 检查事项 | 状态 |
|---|---------|------|
| 1 | 是否已尝试协商/调解？ | ☐ 是 / ☐ 否 |
| 2 | 是否已向对方发送过律师函/催告函？ | ☐ 是 / ☐ 否 |
| 3 | 是否已咨询过律师或法律专业人士？ | ☐ 是 / ☐ 否 |
| 4 | 对方是否有执行能力？（名下有财产可供执行） | ☐ 是 / ☐ 否 / ☐ 不确定 |
| 5 | 是否有对方准确的身份信息/工商信息？ | ☐ 是 / ☐ 否 |
| 6 | 是否了解对方的实际经营/居住地址？ | ☐ 是 / ☐ 否 |

---

## 二、时效与主体（5项）

| # | 检查事项 | 状态 |
|---|---------|------|
| 7 | 诉讼时效是否未过？ | ☐ 是（未过） / ☐ 否（已过/临近） |
| 8 | 如时效已过，是否存在中断/中止情形？ | ☐ 是 / ☐ 否 |
| 9 | 原告主体是否适格？（与本案有直接利害关系） | ☐ 是 / ☐ 否 |
| 10 | 被告主体是否明确？（具体个人或单位） | ☐ 是 / ☐ 否 |
| 11 | 是否遗漏必要共同诉讼人？ | ☐ 否 / ☐ 是（需补充：___） |

---

## 三、证据状况（5项）

| # | 检查事项 | 状态 |
|---|---------|------|
| 12 | 核心证据是否充分？（能证明基本事实） | ☐ 是 / ☐ 否 |
| 13 | 证据原件是否在手？（避免仅有复印件） | ☐ 是 / ☐ 否 |
| 14 | 证据是否合法取得？（非非法手段获取） | ☐ 是 / ☐ 否 |
| 15 | 是否有书面合同/协议？ | ☐ 是 / ☐ 否（仅有口头） |
| 16 | 是否有聊天记录/转账记录/录音录像等辅助证据？ | ☐ 是 / ☐ 否 |

---

## 四、管辖与程序（4项）

| # | 检查事项 | 状态 |
|---|---------|------|
| 17 | 管辖法院是否正确？（级别+地域） | ☐ 是 / ☐ 否 |
| 18 | 是否有协议管辖约定？ | ☐ 是 / ☐ 否 |
| 19 | 是否有仲裁协议？（可能排除法院管辖） | ☐ 否 / ☐ 是（应走仲裁） |
| 20 | 是否已确定正确的案由？ | ☐ 是 / ☐ 否 |

---

## 五、成本与策略（5项）

| # | 检查事项 | 状态 |
|---|---------|------|
| 21 | 诉讼费预算是否清楚？ | ☐ 是 / ☐ 否 |
| 22 | 是否需要申请财产保全？ | ☐ 是 / ☐ 否 |
| 23 | 是否需要申请证据保全？ | ☐ 是 / ☐ 否 |
| 24 | 胜诉可能性是否较高？（初步判断 >50%） | ☐ 是 / ☐ 否 |
| 25 | 是否有明确的诉讼目标？（要达成什么结果） | ☐ 是 / ☐ 否 |

---

## 评估结论

| 通过项数 | 结论 | 建议 |
|---------|------|------|
| 20-25项 ✅ | 🟢 准备充分 | 可启动诉讼程序 |
| 15-19项 ✅ | 🟡 基本准备 | 补充短板后启动 |
| 10-14项 ✅ | 🟠 准备不足 | 建议先收集证据/咨询律师 |
| <10项 ✅ | 🔴 严重不足 | 不建议立即起诉，优先补充准备 |

---

> **特别提醒**: 本清单为诉前自查参考，不替代专业律师判断。如有疑问，建议咨询执业律师。
