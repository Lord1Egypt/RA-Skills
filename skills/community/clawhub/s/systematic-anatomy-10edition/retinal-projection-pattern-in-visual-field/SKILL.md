# 视野成像的视网膜投射规律

## 适用条件
- 仅适用于单眼静态平视状态
- 不考虑眼球运动或双眼融合效应
- 用户已了解眼球屈光系统及视网膜方位命名（鼻侧/颞侧、上/下）

## 执行步骤

1. **确认视野与视网膜的对应关系**：由于角膜和晶状体的折射作用，外界物体在视网膜上形成倒置且左右翻转的实像。应用以下投射规律：
   - 鼻侧半视野 → 投射到颞侧半视网膜
   - 颞侧半视野 → 投射到鼻侧半视网膜
   - 上半视野 → 投射到下半视网膜
   - 下半视野 → 投射到上半视网膜

2. **定位视野缺损对应的视网膜区域**：根据患者报告或检查发现的视野缺损位置（如“左眼颞侧视野缺失”），利用上述规律确定受损的视网膜区域（如“左眼鼻侧视网膜”）。

3. **推断视觉通路损伤部位**：结合视觉通路解剖知识（如鼻侧视网膜纤维在视交叉处交叉），将视网膜损伤区域映射至可能的中枢损伤位点（如视交叉、视束等）。

4. **验证临床一致性**：确保推断结果与典型综合征（如视交叉损伤导致双眼颞侧偏盲）相符。

> **注意**：此技能仅提供基础投射逻辑，复杂病例需结合其他神经眼科评估方法。
