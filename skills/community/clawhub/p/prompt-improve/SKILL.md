# improve your prompt

> 将模糊的用户请求转化为基于四要素框架的详细、可操作提示。该技能直接根据用户的输入生成结构化提示，无需交互式澄清。技能仅生成用于执行任务的提示文本，不执行实际任务或探索性操作。基于角色设定、任务描述、上下文信息和指令约束四个核心要素构建优秀prompt。
