# 政府公文输出模板

本文件定义各类政府公文的输出格式模板，AI 应按对应模板生成符合公文规范的完整内容。

---

## 一、模板使用原则

| 原则 | 说明 |
|:---|:---|
| **文种匹配** | 根据 `doc_type` 选择对应的标准模板结构 |
| **格式合规** | 严格遵循《党政机关公文格式》（GB/T 9704-2012）的要素规范 |
| **层级分明** | 使用规范的层级序号：一、（一）1.（1） |
| **用语规范** | 使用法言法语和公文固定表述，避免口语化 |

---

## 二、公文通用版式要素

```
密级和保密期限（涉密时标注）

发文机关标志（红头）+ 发文字号

┌─────────────────────────────────┐
│          标  题（二号小标宋体）          │
```

## 四、格式规范速查

| 要素 | 规范要求 |
|:---|:---|
| 标题 | 二号小标宋体，居中 |
| 主送机关 | 三号仿宋体，顶格，冒号后换行 |
| 正文 | 三号仿宋体，首行缩进 2 字符 |
| 一级标题（一、） | 三号黑体 |
| 二级标题（（一）） | 三号楷体 |
| 三级标题（1.） | 三号仿宋体，可加粗 |
| 层级序号 | 一、（一）1.（1）① —— 逐级使用，不可跳级 |
| 成文日期 | 用阿拉伯数字标全年、月、日 |
| 印章 | 发文机关印章须盖在成文日期上方（骑年盖月） |
| 页码 | 4 号半角宋体，居中（- 1 -） |
| 装订 | 左侧装订，骑马订或平订 |
| 数字用法 | 公历世纪、年代、年、月、日、时刻用阿拉伯数字 |
| 计量单位 | 使用国家法定计量单位 |
