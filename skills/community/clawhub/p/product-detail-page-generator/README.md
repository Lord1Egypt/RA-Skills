# 幻妆秀商品详情页生成器 — 技术文档

## 平台信息
- 地址：https://show.3diy.world
- 账号体系：3diy/123456（演示账号，实际使用需客户自有账号）

## 操作流程详解

### Step 1: 新建作品
1. 导航到 `https://show.3diy.world/#/workbench/promotion`
2. 点击「新增+」按钮
3. 输入作品标题（如"XX品牌产品名"）
4. 点击「创建设计」

### Step 2: AI生成
1. 点击编辑器右上角「AI」按钮，打开AI面板
2. 点击「上传图片」上传1-3张产品图
3. 在提示词输入框填写产品描述
4. 点击「开始分析」

### Step 3: 确认方案
1. AI自动完成：图片识别 → 章节方案 → JSON配置
2. 弹出章节方案确认面板（5个章节）
3. 可编辑每个章节的标题、卖点、生图提示
4. 点击「确认方案并继续生成」

### Step 4: 等待生成
1. AI自动生成5张章节图片（RunningHub引擎）
2. 自动校验图片质量
3. 应用到画布
4. 耗时约3-5分钟

### Step 5: 保存发布
1. 点击「保存」
2. 点击「发布」
3. 获取分享链接和二维码

## 校验结果说明
- **通过**：图片质量合格，可直接使用
- **占位图**：RunningHub超时，使用上传的产品图作为占位，可后续替换
- **校验不通过**：图片与方案偏差较大，建议重新生成该章节

## 注意事项
- 图片上传路径必须为 `/tmp/openclaw/uploads/` 目录下的文件
- browser wait最大30秒，生图阶段需多次snapshot轮询
- 如遇"Agent 连接中断"提示，需点击重新启动AI生成流程
- 章节方案确认后不可回退，需谨慎确认

## 错误处理
| 错误 | 原因 | 解决方案 |
|------|------|----------|
| RunningHub超时 | 服务器负载高 | 使用占位图继续，后续手动替换 |
| Agent连接中断 | WebSocket断开 | 点击重新启动 |
| 图片上传失败 | 路径不存在 | 确保文件在/tmp/openclaw/uploads/ |
| 发布失败 | 未保存 | 先保存再发布 |
