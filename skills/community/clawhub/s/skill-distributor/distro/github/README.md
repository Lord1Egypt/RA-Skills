# Skill-Distributor

> 你来创作 Skill，分发交给我

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/GMF515/skill-distributor)
[![Platform](https://img.shields.io/badge/platform-WorkBuddy%20%7C%20ClawHub-green.svg)](https://clawhub.ai/skills/skill-distributor)

## 🎯 是什么

Skill-Distributor 是一款 AI 技能分发工具，可以将你的 Skill **一键分发到 10+ 平台**。

## ✨ 核心价值

| 指标 | 传统方式 | Skill-Distributor |
|------|----------|------------------|
| 耗时 | 2 小时+ | **5 分钟** |
| 平台 | 逐个适配 | **一键生成** |
| 效率 | 手动检查 | **自动校验** |

**效率提升 95%+**

## 🚀 快速开始

### 安装

在 WorkBuddy / ClawHub 中搜索 `skill-distributor` 安装，或下载 ZIP 后手动安装。

### 使用

```markdown
帮我分发这个 Skill：/path/to/your-skill/
```

## 🔧 核心功能

- ✅ **智能校验** - 发布前自动检测 SKILL.md 字段完整性
- ✅ **平台兼容** - 自动识别各平台字段差异，屏蔽不兼容参数
- ✅ **品牌统一** - 全平台文案标准化，统一人设与风格
- ✅ **一键导出** - 生成的 distro/ 目录可直接复制使用

## 📁 输出结构

```
distro/
├── github/                 ← GitHub 仓库用
├── platforms/              ← 各技能市场上架用
└── social/                 ← 社交媒体推广用
```
