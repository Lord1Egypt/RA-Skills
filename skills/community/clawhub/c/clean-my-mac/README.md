### Overview

- **Multi-platform support**: Native scanning targets for macOS, Windows, and Linux.
- **AI Agent Integration**: Connects as a tool/skill to LLMs (Claude/Gemini/GPT) for smart folder interpretations.
- **CleanMyMac Style**: Premium Apple-style interactive dashboard with dark/light themes and bilingual EN/ZH switches.
- **Safety Centered**: Local web server deletion protected by validation tokens and path boundaries.
