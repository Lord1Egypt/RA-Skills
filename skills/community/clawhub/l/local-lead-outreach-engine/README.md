# Local Lead Outreach Engine

Claude Code skill for building qualified B2B lead shortlists and practical cold outreach sequences.

## What It Does

- Defines the best-fit ICP for an offer
- Scores pasted, spreadsheet, CRM, or researched leads
- Selects a small send batch
- Writes first emails and follow-ups
- Flags weak leads, risky claims, and missing evidence
- Adds a send-readiness checklist before outreach

## Best For

- agencies
- freelancers
- consultants
- SaaS founders
- local service providers
- B2B service businesses
