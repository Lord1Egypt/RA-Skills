# 知识点-教材版本映射方案

## 问题背景

中国中小学教材存在多个版本（人教版、北师大版、苏教版、沪教版、浙教版等），同一知识点在不同版本中的编排顺序、深度、呈现方式各不相同。知识点掌握评估必须与**学校实际使用的教材版本**对齐。

## 映射架构

```
国家课程标准（统一）
       ↕ 映射
教材版本知识图谱
       ↕ 映射
学校教学进度（实际）
       ↕ 映射
作业题目 → 知识点
```

### 第一层：课程标准层级

中国教育部发布的**义务教育课程标准**是所有教材版本的共同基础。每个学科的知识点最终可以上溯到课标中的"内容标准"条目。

**示例（数学·三年级）：**

| 课标编号 | 内容标准 | 说明 |
|----------|----------|------|
| M3-1-1 | 数的认识 | 认识万以内的数 |
| M3-1-2 | 数的运算 | 两位数加减法、多位数乘除法 |
| M3-1-3 | 常见的量 | 认识年月日、24时计时法 |
| M3-2-1 | 图形的认识 | 长方形、正方形、平行四边形 |
| M3-2-2 | 测量 | 面积的意义及单位 |
| M3-3-1 | 数据的收集与整理 | 简单数据统计 |

### 第二层：教材版本知识图谱

各教材版本对课标知识点进行编排，形成版本特有的知识图谱：

```
人教版（人教版数学）
├── 三年级上册
│   ├── 第一单元 时、分、秒 (M3-1-3)
│   ├── 第二单元 万以内的加法和减法 (M3-1-2)
│   ├── 第三单元 测量 (M3-2-2)
│   └── ...
├── 三年级下册
│   ├── 第一单元 位置与方向 (M3-2-1)
│   └── ...
└── ...

北师大版（北师大版数学）
├── 三年级上册
│   ├── 第一单元 混合运算 (M3-1-2)
│   ├── 第二单元 观察物体 (M3-2-1)
│   └── ...
└── ...
```

## 数据存储方案

### 方案一：JSON 知识图谱（推荐起步）

```json
{
  "subject": "math",
  "gradeLevel": 3,
  "textbookVersion": "人教版",
  "units": [
    {
      "unitId": "math-g3-rj-1",
      "unitName": "时、分、秒",
      "curriculumRef": "M3-1-3",
      "knowledgePoints": [
        {
          "kpId": "math-g3-rj-1-01",
          "kpName": "秒的认识",
          "weight": 0.3,
          "questionPatterns": ["认识秒针", "秒与分的换算"]
        },
        {
          "kpId": "math-g3-rj-1-02",
          "kpName": "时间的计算",
          "weight": 0.4,
          "questionPatterns": ["经过时间计算", "开始/结束时间推算"]
        }
      ]
    }
  ]
}
```

### 方案二：关系型数据库

适用于多学校、多版本管理：

```
表: textbook_versions
  version_id (PK) | subject | grade | publisher | edition_year | is_active

表: knowledge_points
  kp_id (PK) | curriculum_ref (FK) | name | description | level

表: textbook_kp_mapping
  map_id (PK) | version_id (FK) | kp_id (FK) | unit | chapter | sequence | weight

表: question_kp_mapping
  map_id (PK) | question_id (FK) | kp_id (FK) | weight | difficulty
```

## 学校适配流程

```
1. 学校注册 → 选择教材版本（人教/北师大/苏教...）→ 选择年级
2. 系统加载对应版本的知识图谱
3. 教师确认/微调进度（例如：本校教学进度比标准快或慢两周）
4. 教师按课时/单元布置作业 → 题目自动关联知识点
5. 评估报告基于版本知识图谱生成
```

## 版本间知识点映射

同一课标知识点在不同教材版本间可建立自动映射：

```
人教版 M3-1-2     →   北师大版 M3-1-2
"万以内加减法"     ↕    "混合运算中的加减法"
  权重: 0.4             权重: 0.35
```

映射准确性依赖于：
1. **课标编号**作为统一标识（最可靠）
2. **知识点名称 + 描述**的语义匹配（辅助）
3. **题目模式匹配**（根据题目内容自动关联）

## 多版本管理策略

| 策略 | 适用场景 | 复杂度 | 推荐 |
|------|----------|--------|------|
| 单一版本（全校统一教材） | 单校部署 | 低 | MVP首选 |
| 多版本预置（维护主流版本图谱） | 区域部署，有限版本 | 中 | 推广阶段 |
| 动态映射（AI自动匹配题目→知识点） | 多版本混合 | 高 | 成熟阶段 |
| 开放图谱（教师自定义知识点） | 校本课程/特色教学 | 中高 | 高阶定制 |
