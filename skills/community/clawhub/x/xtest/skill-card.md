## Description: <br>
xtest provides a pre-delivery quality gate for code, script, and configuration changes by classifying validation depth, planning tests, running checks, and looping through limited fixes before pass or reject. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill as a code delivery gate to decide validation depth, create and run test plans, and report pass or reject outcomes for code, script, and configuration changes. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Local validation commands or suggested fixes may produce misleading results or unwanted behavior if accepted without review. <br>
