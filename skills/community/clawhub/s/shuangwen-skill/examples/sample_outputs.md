# 输出示例

```text
1B，2B，3B，4B，5A
```

含义：

- 题材：医学神医
- 主角表面身份：规培医生 / 实习医生
- 场景：急诊抢救 / 专家会诊
- 反转方式：一眼看穿关键问题
- 文风：番茄网文风，中爽

正文结尾示例：

```markdown
## 本章爽点
- 初始误判：所有人都以为主角只是普通规培医生。
- 深度羞辱：副主任当众否定主角资格，并威胁影响规培考核。
- 反派加码：副主任坚持错误方向，要求主角离开抢救区。
- 第一层打脸：主角指出被忽略的关键异常。
- 第二层打脸：远程专家确认主角思路。
- 章末钩子：专家认出主角曾参与过某个绝密疑难病项目。
```
