## Description: <br>
Enforce Markdown output formatting policy whenever the user asks for full Markdown source code, raw Markdown, or copyable Markdown documents. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Jeff-Tian](https://clawhub.ai/user/Jeff-Tian) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agents use this skill when they need copyable Markdown source output with consistent outer fencing and link spacing rules. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill intentionally changes raw Markdown output style, including triple-tilde outer fences and added spaces in links. <br>
Mitigation: Use it only for workflows that accept these Markdown conventions, and review generated Markdown before publishing or feeding it into strict parsers. <br>
