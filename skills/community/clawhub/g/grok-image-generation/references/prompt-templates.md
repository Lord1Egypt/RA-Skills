# Prompt Templates

Use these as starting points, then adapt to the user’s subject, style, platform, and constraints.

## 1. Retro poster

```text
Create a retro movie-poster illustration of {subject}. Use bold composition, dramatic lighting, limited vintage color palette, textured print look, and legible title treatment. Make it feel like a restored mid-century poster, not a modern glossy ad.
```

## 2. Product mockup / promo art

```text
Create clean product marketing art for {product}. Show the product clearly as the hero, use realistic lighting, uncluttered composition, ecommerce-friendly styling, and a background that supports the product without distracting from it.
```

## 3. T-shirt design concept

```text
Create a print-friendly T-shirt graphic for {theme}. Keep the silhouette strong, central composition readable at thumbnail size, and avoid tiny fragile details. Use high contrast and a limited palette suitable for apparel printing.
```

## 4. Character portrait

```text
Create a polished portrait of {character}. Preserve recognizable facial features, use expressive lighting, coherent costume details, and a background that supports the character’s story without overpowering the portrait.
```

## 5. Image edit: style transfer

```text
Transform this image into {target style} while preserving the main subject, pose, framing, and overall readability. Improve polish and consistency, but do not replace the core identity of the scene.
```

## 6. Image edit: cleanup

```text
Clean up this image. Remove distractions, improve composition clarity, fix awkward artifacts, and keep the subject natural. Preserve the original intent instead of radically redesigning it.
```

## 7. Image edit: background replacement

```text
Keep the main subject intact and replace the background with {new background}. Match perspective, lighting, and mood so the result feels naturally photographed or illustrated in one scene.
```

## 8. Variation prompt

```text
Create {n} distinct variations of {subject}. Keep the same core concept, but vary composition, camera angle, lighting, and emphasis so the options are meaningfully different.
```

## Heuristics

- Ask for **composition**, **lighting**, **palette**, and **texture** explicitly.
- For product work, say what must remain **accurate**.
- For edits, say what must be **preserved** before saying what should change.
- For T-shirt or print work, ask for **readability at small size** and **strong silhouette**.
- Avoid contradictory prompts like “minimal but full of intricate tiny details.”
