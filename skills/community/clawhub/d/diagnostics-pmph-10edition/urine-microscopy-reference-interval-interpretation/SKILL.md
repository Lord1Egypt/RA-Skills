# 尿液显微镜检查参考区间判定

## 何时使用
- 用户提供尿液显微镜检查数据（包括细胞/管型计数及检测方法）
- 需判断结果是否在正常参考区间内
- 用于泌尿系统疾病初步筛查或异常结果复核

## 执行步骤
1. **确认前提条件**：
   - 样本为新鲜离心尿液
   - 使用标准显微镜检查方法（玻片法或定量法）
   - 排除污染样本或非标准检测方法

2. **按成分判定是否异常**：
   - **红细胞**：
     - 玻片法：>2个/HPF（高倍视野）视为镜下血尿
     - 定量法：>5个/μl 视为异常
   - **白细胞**：
     - 男性玻片法：>3个/HPF；女性：>5个/HPF
     - 定量法：>10个/μl 视为镜下脓尿
   - **上皮细胞**：
     - 肾小管上皮细胞：任何检出均异常
     - 移行上皮细胞：多于“偶见”视为异常
     - 鳞状上皮细胞：男性多于“偶见”，女性 >5个/HPF 视为可能污染或异常
   - **管型**：
     - 透明管型 >1个/HPF 视为异常；其他类型管型检出即异常

3. **综合判断**：
   - 若任一指标超出参考区间，提示可能存在泌尿系统病变
   - 必须结合患者性别、年龄、标本类型（如晨尿）、临床症状等综合分析
   - 显微镜观察应在×400放大倍数（标准HPF）下进行

4. **输出结论**：
   - 明确指出哪些指标异常
   - 建议“需结合临床进一步评估”

> 注意：本技能不适用于非标准方法、污染样本或无临床背景的数据解读。
