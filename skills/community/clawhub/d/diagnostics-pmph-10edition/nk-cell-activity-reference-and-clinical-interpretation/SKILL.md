# NK细胞活性参考区间及临床意义解读

## 何时使用
- 已完成NK细胞活性测定（使用⁵¹Cr释放法、酶释放法或流式细胞术）
- 需评估患者抗肿瘤或抗病毒免疫能力
- 怀疑免疫缺陷、血液系统肿瘤、实体瘤、艾滋病或移植物抗宿主反应

## 执行步骤
1. **根据检测方法选择对应参考区间**：
   - **⁵¹Cr释放法**：
     - 自然释放率：<10%~15%
     - 自然杀伤率：47.6%~76.8%
     - ⁵¹Cr利用率：6.5%~47.8%
   - **酶释放法**：细胞毒指数 27.5%~52.5%
   - **流式细胞术**：13.8% ± 5.9%

2. **临床意义判断**：
   - **活性减低**：见于血液系统肿瘤、实体瘤、免疫缺陷病、艾滋病、某些病毒感染
   - **活性升高**：见于宿主抗移植物反应（如移植排斥）

3. **注意事项**：
   - 不同检测方法结果不可直接比较
   - 应使用本实验室建立的参考区间
   - 结果需结合临床及其他免疫指标综合判断
   - 不适用于非免疫相关疾病的初筛
