# essence-trace 示例 1：为什么房价会涨

> **模式**：普通模式 | **复杂度**：⭐⭐⭐⭐ | **领域**：经济/社会

```
🎯 essence-trace: 为什么中国大城市房价会持续上涨
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
初始问题：为什么中国大城市房价会持续上涨？
输入类型：现象

追问链：
Q1: 为什么房价会涨？
  → 答：供不应求（需求 > 供给）
       追问：为什么供不应求？

Q2: 为什么大城市住房供不应求？
  → 答：人口持续流入 + 土地供应有限
       追问：为什么土地供应有限？

Q3: 为什么土地供应有限？
  → 答：政府垄断土地一级市场 + 18 亿亩耕地红线
       追问：政府为什么要限制土地供应？

Q4: 政府为什么要限制土地供应？
  → 答：土地财政依赖（卖地收入占地方财政 30-50%）
       追问：政府为什么依赖土地财政？

Q5: 政府为什么依赖土地财政？
  → 答：1994 年分税制改革后，地方政府事权 > 财权
       追问：为什么会形成这种"事权-财权"错配？

Q6: 为什么会形成事权-财权错配？
  → 答：中央集权 + 地方支出责任大 + 税收上缴
       追问：更深层的原因是什么？

Q7: 更深层的原因是什么？
  → 答：中国发展模式 = 投资驱动型 + 城市化 + 工业化
       → 投资需要融资 → 融资工具 = 土地信用
       → 土地成为"第二货币" → 房价上涨 = 信用扩张的镜像

Q8: 为什么必须靠投资驱动？
  → 答：[不可分节点] 发展阶段选择 + 比较优势 + 国际竞争压力
       终止类型：经济规律 + 价值选择
       原因：这是国家发展模式的战略选择，已触及"路径依赖"边界

关键杠杆点：
① 货币政策（杠杆层级 L5）— 放水/收紧直接影响信用扩张
   影响力：影响所有资产价格
   可操作性：短期可做（跟随政策）

② 土地政策（杠杆层级 L5）— 土地供应增减影响供需
   影响力：直接影响新房供给
   可操作性：政府主导，个人无法改变

③ 人口流动（杠杆层级 L6）— 大城市化的方向
   影响力：影响需求侧
   可操作性：长期趋势，短期难改变

④ 个人选择（杠杆层级 L2）— 在哪个城市买、买什么房
   影响力：决定个人财富
   可操作性：个人可立即做

💡 本质：房价 = 土地财政 + 货币信用 + 供需失衡的三角共振
```

---

## 解析

**关键洞察**：
- 房价不是单一因素，而是多个机制的**共振**
- 杠杆点层级越高（思维/规则层），影响越大但越难改变
- 善人可操作的是 **L2 杠杆（个人选择）**

**与"普通思考"的差异**：
- ❌ 普通思考："房价会涨是因为政策支持、刚需大"
- ✅ essence-trace：从"现象"追溯到"国家发展模式"的战略选择

**可衍生行动**：
- 跟随货币政策（M2、社融数据）
- 关注土地政策变化（集中供地等）
- 跟随人口流动方向选城市
- 在杠杆点中选最适合自己的
