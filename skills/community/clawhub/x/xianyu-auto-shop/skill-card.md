## Known Risks and Mitigations: <br>
Risk: The skill requests powerful device automation and notification access for a marketplace seller account. <br>
Mitigation: Install only after reviewing permissions, and run it in an account context where automated marketplace actions are acceptable. <br>
Risk: Automated replies and listing or title changes can affect buyer communication and marketplace visibility without documented review steps. <br>
Mitigation: Keep reply text and title changes supervised, review outputs regularly, and stop the process if behavior no longer matches seller intent. <br>
Risk: The security summary notes too little user control is described for unattended marketplace actions. <br>
Mitigation: Use conservative schedules and limits, monitor the process while active, and avoid granting access on accounts that cannot tolerate automated actions. <br>
