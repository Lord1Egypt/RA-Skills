## 1. 技术指标 (Technical Indicators)

### 1.1 移动平均线 (Moving Average, MA)

**定义**：一定周期内价格的算术平均值

**计算公式**：
```
MA(n) = (P₁ + P₂ + ... + Pₙ) / n
```

**常用参数**：
- MA5: 短期趋势（1周）
- MA10: 短期趋势（2周）
- MA20: 短期趋势（1月）
- MA60: 季度线（3月）
- MA120: 半年线（6月）
- MA250: 年线（1年）

**应用场景**：
- 判断趋势方向（价格在均线之上为多头，之下为空头）
- 支撑和压力位（均线往往成为支撑/压力）
- 金叉/死叉信号（短期均线上穿/下穿长期均线）

**解读标准**：
- **多头排列**：MA5 > MA10 > MA20 > MA60 → 强势上涨
- **空头排列**：MA5 < MA10 < MA20 < MA60 → 弱势下跌
- **均线粘合**：各周期均线靠近 → 方向选择前夕

---

### 1.2 指数移动平均线 (Exponential Moving Average, EMA)

**定义**：对近期价格赋予更高权重的移动平均

**计算公式**：
```
EMA(t) = α × P(t) + (1 - α) × EMA(t-1)
其中：α = 2 / (n + 1)
```

**优势**：对价格变化反应更快，减少滞后性

---

### 1.3 MACD (Moving Average Convergence Divergence)

**定义**：利用两条不同速度的指数移动平均线的差值判断趋势

**计算公式**：
```
DIF = EMA(12) - EMA(26)
DEA = EMA(DIF, 9)
MACD柱 = 2 × (DIF - DEA)
```

**应用场景**：
1. **金叉/死叉**：
   - 金叉：DIF上穿DEA → 买入信号
   - 死叉：DIF下穿DEA → 卖出信号

2. **背离**：
   - 顶背离：价格创新高，MACD不创新高 → 见顶信号
   - 底背离：价格创新低，MACD不创新低 → 见底信号

3. **零轴穿越**：
   - DIF上穿零轴 → 多头趋势确立
   - DIF下穿零轴 → 空头趋势确立

**解读标准**：
- MACD柱 > 0 且增长 → 强势上涨
- MACD柱 > 0 但缩短 → 上涨动能减弱
- MACD柱 < 0 且扩大 → 强势下跌
- MACD柱 < 0 但缩短 → 下跌动能减弱

---

### 1.4 RSI (Relative Strength Index)

**定义**：衡量价格涨跌速度和幅度的动量指标

**计算公式**：
```
RS = 平均涨幅 / 平均跌幅（14天）
RSI = 100 - (100 / (1 + RS))
```

**应用场景**：
- 超买/超卖判断
- 背离信号识别

**解读标准**：
- **RSI > 70**：超买区域，警惕回调
- **RSI < 30**：超卖区域，关注反弹
- **RSI 50-70**：健康上涨
- **RSI 30-50**：健康回调
- **顶背离**：价格新高，RSI不创新高 → 卖出信号
- **底背离**：价格新低，RSI不创新低 → 买入信号

---

### 1.5 布林带 (Bollinger Bands, BOLL)

**定义**：由中轨、上轨、下轨组成的波动区间

**计算公式**：
```
中轨 (MB) = MA(20)
上轨 (UP) = MB + 2 × σ
下轨 (DN) = MB - 2 × σ
其中：σ = 标准差（20天）
```

**应用场景**：
1. **突破信号**：
   - 价格突破上轨 → 强势，可能继续上涨
   - 价格跌破下轨 → 弱势，可能继续下跌

2. **回归信号**：
   - 价格触及上轨后回落 → 短期见顶
   - 价格触及下轨后反弹 → 短期见底

3. **缩口与开口**：
   - 布林带收窄（缩口）→ 波动率降低，蓄势待发
   - 布林带扩张（开口）→ 波动率增加，趋势启动

**解读标准**：
- 价格在中轨之上 → 多头强势
- 价格在中轨之下 → 空头强势
- 价格沿上轨运行 → 超强势，可持有
- 价格沿下轨运行 → 超弱势，避免抄底

---

### 1.6 KDJ 指标

**定义**：结合动量、强弱和移动平均的随机指标

**计算公式**：
```
RSV(n) = (收盘价 - n日最低价) / (n日最高价 - n日最低价) × 100
K = 2/3 × K(前一日) + 1/3 × RSV
D = 2/3 × D(前一日) + 1/3 × K
J = 3 × K - 2 × D
```

**应用场景**：
- 超买超卖判断
- 短期买卖点

**解读标准**：
- **K > 80, D > 80**：超买，卖出信号
- **K < 20, D < 20**：超卖，买入信号
- **K上穿D（金叉）**：买入信号
- **K下穿D（死叉）**：卖出信号
- **J > 100**：严重超买
- **J < 0**：严重超卖

---

### 1.7 成交量 (Volume)

**应用场景**：
- 验证价格趋势的有效性

**解读标准**：
- **放量上涨**：多头力量强，趋势健康 ✅
- **缩量上涨**：上涨乏力，警惕回调 ⚠️
- **放量下跌**：空头力量强，趋势破坏 ❌
- **缩量下跌**：下跌动能减弱，可能见底 ⚠️
- **天量天价**：成交量极大，往往是顶部信号
- **地量地价**：成交量极小，往往是底部信号

**量价关系**：
- 价涨量增 → 健康上涨 ✅
- 价涨量减 → 上涨乏力 ⚠️
- 价跌量增 → 恐慌杀跌 ❌
- 价跌量减 → 下跌减弱 🔄

## 2. 财务指标 (Financial Ratios)

### 2.1 盈利能力指标

#### 2.1.1 毛利率 (Gross Profit Margin)

**计算公式**：
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```

**解读标准**：
- **> 50%**：极高毛利，如软件、医药
- **30-50%**：较高毛利，如品牌消费
- **15-30%**：中等毛利，如一般制造业
- **< 15%**：低毛利，如零售、农业

**趋势判断**：
- 毛利率上升 → 产品竞争力增强，定价权提升 ✅
- 毛利率下降 → 竞争加剧，成本上升 ⚠️

---

#### 2.1.2 净利率 (Net Profit Margin)

**计算公式**：
```
净利率 = 净利润 / 营业收入 × 100%
```

**解读标准**：
- **> 20%**：优秀（如贵州茅台 50%+）
- **10-20%**：良好
- **5-10%**：一般
- **< 5%**：较差

---

#### 2.1.3 ROE (Return on Equity, 净资产收益率)

**计算公式**：
```
ROE = 净利润 / 平均股东权益 × 100%
```

**杜邦分析拆解**：
```
ROE = 净利率 × 总资产周转率 × 权益乘数
    = (净利润/收入) × (收入/总资产) × (总资产/股东权益)
```

**解读标准**（A股）**：
- **> 20%**：优秀（如贵州茅台 25%+）
- **15-20%**：良好
- **10-15%**：合格
- **< 10%**：较差

**重要性**：
- 巴菲特最看重的指标之一
- 反映股东投资回报率
- 连续10年ROE > 15%的公司是稀缺标的

---

#### 2.1.4 ROA (Return on Assets, 总资产收益率)

**计算公式**：
```
ROA = 净利润 / 平均总资产 × 100%
```

**解读标准**：
- **> 10%**：优秀
- **5-10%**：良好
- **< 5%**：一般

---

#### 2.1.5 ROIC (Return on Invested Capital, 投入资本回报率)

**计算公式**：
```
ROIC = 税后净营业利润 / (股东权益 + 有息负债) × 100%
```

**解读**：
- 衡量公司使用资本的效率
- **ROIC > WACC（加权平均资本成本）** → 创造价值 ✅

---

### 2.2 偿债能力指标

#### 2.2.1 资产负债率 (Debt to Asset Ratio)

**计算公式**：
```
资产负债率 = 总负债 / 总资产 × 100%
```

**解读标准**：
- **< 30%**：财务非常稳健（如茅台）
- **30-50%**：财务稳健
- **50-70%**：中等风险
- **> 70%**：高风险（除金融、地产行业外）

**行业差异**：
- 银行业：80-90%正常（存款属于负债）
- 地产业：70-80%正常（高杠杆行业）
- 制造业：< 50%为佳

---

#### 2.2.2 流动比率 (Current Ratio)

**计算公式**：
```
流动比率 = 流动资产 / 流动负债
```

**解读标准**：
- **> 2**：短期偿债能力强
- **1.5-2**：良好
- **< 1**：短期偿债压力大 ⚠️

---

#### 2.2.3 速动比率 (Quick Ratio)

**计算公式**：
```
速动比率 = (流动资产 - 存货) / 流动负债
```

**解读标准**：
- **> 1**：短期偿债能力强
- **0.8-1**：可接受
- **< 0.8**：偿债压力大 ⚠️

---

#### 2.2.4 利息保障倍数 (Interest Coverage Ratio)

**计算公式**：
```
利息保障倍数 = EBIT / 利息费用
```

**解读标准**：
- **> 5**：偿债能力强
- **3-5**：良好
- **1-3**：偿债压力较大 ⚠️
- **< 1**：利息都无法覆盖 ❌

---

### 2.3 运营能力指标

#### 2.3.1 应收账款周转率 (Receivables Turnover Ratio)

**计算公式**：
```
应收账款周转率 = 营业收入 / 平均应收账款
应收账款周转天数 = 365 / 应收账款周转率
```

**解读**：
- 周转率越高，回款越快 ✅
- 周转天数越短，现金流越好 ✅
- 周转天数 > 90天 → 警惕坏账风险 ⚠️

---

#### 2.3.2 存货周转率 (Inventory Turnover Ratio)

**计算公式**：
```
存货周转率 = 营业成本 / 平均存货
存货周转天数 = 365 / 存货周转率
```

**解读**：
- 周转率越高，存货管理越好 ✅
- 存货周转天数上升 → 警惕库存积压 ⚠️

---

#### 2.3.3 总资产周转率 (Total Asset Turnover)

**计算公式**：
```
总资产周转率 = 营业收入 / 平均总资产
```

**解读标准**：
- **> 1**：资产使用效率高（如零售业）
- **0.5-1**：中等
- **< 0.5**：资产使用效率低（如重资产行业）

---

### 2.4 成长能力指标

#### 2.4.1 营收增长率 (Revenue Growth Rate)

**计算公式**：
```
营收增长率 = (本期营收 - 上期营收) / 上期营收 × 100%
```

**解读标准**：
- **> 30%**：高速增长 🚀
- **15-30%**：快速增长 ✅
- **5-15%**：稳健增长
- **< 5%**：低速增长

---

#### 2.4.2 净利润增长率 (Net Profit Growth Rate)

**计算公式**：
```
净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润 × 100%
```

**重要判断**：
- 净利润增长率 > 营收增长率 → 利润率提升 ✅
- 净利润增长率 < 营收增长率 → 利润率下降 ⚠️

---

#### 2.4.3 EPS增长率 (EPS Growth Rate)

**计算公式**：
```
EPS增长率 = (本期EPS - 上期EPS) / 上期EPS × 100%
```

---

## 3. 估值指标 (Valuation Metrics)

### 3.1 市盈率 (PE, Price-to-Earnings Ratio)

**计算公式**：
```
静态PE = 股价 / 每股收益(EPS, 上一年度)
动态PE = 股价 / 预测EPS(未来12个月)
TTM PE = 股价 / 最近12个月EPS
```

**解读标准**（A股）**：
- **< 10**：严重低估（或业绩下滑）
- **10-20**：合理估值
- **20-30**：略高估值
- **> 30**：高估值（除非高成长）

**行业差异**：
- 银行业：PE 5-8正常（低估值行业）
- 白酒业：PE 30-40可接受（确定性溢价）
- 科技成长股：PE > 50常见（高成长溢价）

**关键判断**：
- PE过低 + 业绩增长 = 低估机会 ✅
- PE过高 + 业绩下滑 = 估值陷阱 ❌

---

### 3.2 市净率 (PB, Price-to-Book Ratio)

**计算公式**：
```
PB = 股价 / 每股净资产
```

**解读标准**：
- **< 1**：破净，可能严重低估或资不抵债
- **1-3**：合理
- **3-5**：略高
- **> 5**：高估（除非轻资产行业）

**适用场景**：
- 适合资产密集型行业（银行、地产、制造业）
- 不适合轻资产行业（互联网、咨询）

---

### 3.3 市销率 (PS, Price-to-Sales Ratio)

**计算公式**：
```
PS = 市值 / 营业收入
```

**适用场景**：
- 适合尚未盈利的成长型公司
- 适合对比同行业公司

**解读标准**：
- **< 1**：低估
- **1-3**：合理
- **> 5**：高估（除非高成长）

---

### 3.4 PEG (PE to Growth Ratio)

**计算公式**：
```
PEG = PE / 净利润增长率
```

**解读标准**：
- **PEG < 1**：低估 ✅
- **PEG = 1**：合理估值
- **PEG > 1**：高估 ⚠️
- **PEG > 2**：严重高估 ❌

**彼得·林奇法则**：
- PEG < 0.5 → 非常便宜，重点关注
- PEG在0.5-1之间 → 可投资
- PEG > 2 → 避免投资

---

### 3.5 股息率 (Dividend Yield)

**计算公式**：
```
股息率 = 每股股息 / 股价 × 100%
```

**解读标准**：
- **> 4%**：高股息，适合价值投资
- **2-4%**：中等股息
- **< 2%**：低股息

**应用**：
- 适合防御型投资者
- 熊市中高股息股抗跌
- 长期持有获取稳定现金流

---

## 4. 风险指标 (Risk Metrics)

### 4.1 波动率 (Volatility, σ)

**计算公式**：
```
日波动率 = 日收益率的标准差
年化波动率 = 日波动率 × √252
```

**解读标准**：
- **< 15%**：低波动（如银行股）
- **15-30%**：中等波动
- **> 30%**：高波动（如科技股）

---

### 4.2 最大回撤 (Maximum Drawdown, MDD)

**计算公式**：
```
MDD = (历史最高点 - 最低点) / 历史最高点 × 100%
```

**解读**：
- 衡量最坏情况下的损失
- **MDD < 10%**：低风险
- **MDD 10-20%**：中等风险
- **MDD > 30%**：高风险

---

### 4.3 夏普比率 (Sharpe Ratio)

**计算公式**：
```
Sharpe Ratio = (投资组合收益率 - 无风险收益率) / 投资组合标准差
```

**解读标准**：
- **> 2**：优秀
- **1-2**：良好
- **0.5-1**：可接受
- **< 0.5**：较差

**意义**：单位风险获得的超额收益

---

### 4.4 Beta系数 (β)

**计算公式**：
```
β = Cov(股票收益率, 市场收益率) / Var(市场收益率)
```

**解读**：
- **β = 1**：与市场同步波动
- **β > 1**：波动大于市场（进攻型）
- **β < 1**：波动小于市场（防御型）
- **β < 0**：与市场负相关（对冲资产）

---

### 4.5 VaR (Value at Risk, 风险价值)

**定义**：在给定置信水平下，一定时期内可能遭受的最大损失

**计算方法**（历史模拟法）：
```
VaR(95%) = 历史收益率分布的5%分位数
```

**示例**：
- 95% VaR = -5% → 有95%的概率，日损失不超过5%

---

### 4.6 下行风险 (Downside Risk)

**计算公式**：
```
Downside Risk = √(Σ(min(收益率 - 目标收益率, 0))² / n)
```

**意义**：只计算下跌时的波动，更关注实际风险

---

## 5. 组合指标 (Portfolio Metrics)

### 5.1 组合收益率

**计算公式**：
```
组合收益率 = Σ(权重ᵢ × 资产ᵢ收益率)
```

---

### 5.2 组合标准差（风险）

**计算公式**：
```
σₚ = √(Σᵢ Σⱼ wᵢ wⱼ Cov(rᵢ, rⱼ))
```

**简化（两资产）**：
```
σₚ = √(w₁²σ₁² + w₂²σ₂² + 2w₁w₂ρ₁₂σ₁σ₂)
```

---

### 5.3 相关系数 (Correlation Coefficient, ρ)

**计算公式**：
```
ρ = Cov(X, Y) / (σₓ × σᵧ)
```

**解读**：
- **ρ = 1**：完全正相关
- **ρ = 0**：不相关（最佳分散化）
- **ρ = -1**：完全负相关（完美对冲）

**应用**：
- 构建投资组合时，选择低相关性资产分散风险

---

### 5.4 信息比率 (Information Ratio)

**计算公式**：
```
IR = (组合收益率 - 基准收益率) / 跟踪误差
```

**解读**：
- **IR > 0.5**：优秀的主动管理能力
- **IR < 0**：跑输基准

---

## 综合应用示例

### 案例：贵州茅台（600519）综合分析

假设数据（示例，实际分析应使用最新完整年度财报，如当前为2026年则用2025年数据）：

## 6. 专业估值方法

> 详细方法论参见 `references/valuation-analysis.md`，此处仅列出核心公式速查。

### 6.1 DCF 估值核心公式

**自由现金流 (UFCF)**：
```
EBIT = 营收 × EBIT利润率
NOPAT = EBIT × (1 - 税率)
UFCF = NOPAT + D&A - CapEx - ΔNWC
```

**WACC**：
```
Ke = Rf + β × ERP
Kd = 利息费用/负债 × (1 - 税率)
WACC = Ke × E/(D+E) + Kd × D/(D+E)
```

**终值 (永续增长法)**：
```
TV = 末年FCF × (1+g) / (WACC - g)
g = 2-3% (GDP增长率水平)
```

**企业价值**：
```
EV = Σ(FCFₜ/(1+WACC)^t) + TV/(1+WACC)^n
股权价值 = EV - 净负债
每股价值 = 股权价值 / 总股本
```

### 6.2 可比公司分析 (Comps) 核心倍数

| 倍数 | 公式 | 最适用场景 |
|------|------|-----------|
| PE | 市值/净利润 | 盈利稳定的公司 |
| PB | 市值/净资产 | 重资产/金融业 |
| PS | 市值/营收 | 亏损/高增长公司 |
| EV/EBITDA | 企业价值/EBITDA | 跨资本结构对比 |
| PEG | PE/增长率 | 成长型估值 |

**统计基准**：Max / P75 / Median / P25 / Min

### 6.3 三表联动验证清单

```
✅ BS 平衡：资产 = 负债 + 权益
✅ 现金一致：期末现金 = 期初 + 三类净现金流
✅ 利润联动：净利润 - 分红 ≈ 留存利润变动

```
