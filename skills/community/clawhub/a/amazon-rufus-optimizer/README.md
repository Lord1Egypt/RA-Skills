# Amazon Rufus Optimizer 🤖

> 亚马逊Rufus对话优化工具 — 让你的产品在AI导购对话中被优先推荐

## 快速开始

**输入格式**：产品名称/ASIN + 核心卖点（3-5个关键词）

```
保温杯 + 保温持久/食品级/轻便
```

## 核心功能

### 1. Rufus对话模拟
模拟真实用户与Rufus的对话，检测产品是否会被推荐：

```
✅ "保温效果好的杯子" → 你的产品会被推荐
❌ "送女朋友的生日礼物" → 不会被推荐（缺失标签）
```

### 2. 意图标签诊断
基于四维意图体系分析产品：

| 维度 | 示例标签 |
|------|---------|
| 使用意图 | 日常使用、专业用途、旅行、运动、办公 |
| 人群意图 | 新手、专业人士、送礼、家庭、个人 |
| 场景意图 | 室内、户外、通勤、聚会、季节性 |
| 情感意图 | 安心、舒适、品质感、性价比、潮流 |

### 3. 对话式文案改写
将参数罗列式文案改写为Rufus友好的对话式描述：

| 类型 | 改写前 | 改写后 |
|------|--------|--------|
| 标题 | Stainless Steel Insulated Water Bottle | 32oz Insulated Bottle - Perfect for Hiking & Office |
| 五点 | 304不锈钢，双层真空 | 户外徒步后还能喝到热茶——12小时锁住温度 |

### 4. 跨类目关联
保温杯 → 户外装备、健身配件、通勤用品

## 输出示例

```
输入：保温杯 + 保温持久/食品级/轻便

=== Rufus可见度评分 === 58/100 🟠 建议优化

=== 对话模拟 === 推荐 3/5
=== 意图覆盖 === 3/12 (25%)

=== 优化建议 ===
1. 🔴 高优先：补充场景意图（户外、通勤）
2. 🟡 中优先：优化标题增加场景词
3. 🟢 低优先：补充人群意图（送礼）
```
