# 烧伤急救指南

> **⚠️ 免责声明**：本文仅供参考，不能替代专业急救培训。严重烧伤请立即拨打120。

---

## 一、烧伤深度分级（三度四分法）

| 分度 | 深度 | 外观 | 痛觉 | 愈合时间 |
|------|------|------|------|----------|
| **I度** | 表皮层 | 红斑、干燥、无水泡 | 剧痛 | 3-7天，不留疤 |
| **浅II度** | 真皮浅层 | 大水泡、潮红、湿润 | 剧痛 | 1-2周，一般不留疤 |
| **深II度** | 真皮深层 | 小水泡或无水泡、红白相间 | 迟钝 | 3-4周，留疤 |
| **III度** | 全层皮肤 | 蜡白/焦黄/炭化、皮革样 | 无痛 | 需植皮 |

**需要就医的指征**：
- 所有 II度及以上烧伤
- 面积大于手掌（约1%体表面积）
- 面部/手/足/会阴/关节部位烧伤
- 电烧伤、化学烧伤
- 吸入性损伤（口鼻有烟灰、声音嘶哑、呼吸困难）
- 儿童/老人/糖尿病患者

---

## 二、热烧伤（火焰、热液）急救五步法

### 口诀："冲 → 脱 → 泡 → 盖 → 送"

| 步骤 | 操作 | 时间/要点 |
|------|------|-----------|
| **1. 冲** | 流动冷水冲洗烧伤部位 | **至少15-20分钟**，水温15-25℃ |
| **2. 脱** | 脱去/剪开烧伤部位衣物 | 如衣物粘在皮肤上，**剪开**不要撕扯 |
| **3. 泡** | 冷水浸泡 | 继续15-30分钟（大面积烧伤不泡，防低体温） |
| **4. 盖** | 干净纱布/保鲜膜覆盖 | 不要涂任何东西！ |
| **5. 送** | 送医 | II度以上烧伤需专业处理 |

---

## 三、大面积烧伤特别处理

1. 停止燃烧：**停 → 倒 → 滚**（Stop, Drop, Roll）
2. 不要脱衣服
3. 不要冲冷水（可能引起低体温休克）
4. 用干净床单整体包裹
5. 抬高烧伤肢体（高于心脏）减轻水肿
6. 监测呼吸，做好CPR准备

---

## 四、化学烧伤

### 原则：立即去除化学物质！

| 化学品类型 | 处理方法 |
|------------|----------|
| **酸烧伤**（硫酸、盐酸等） | 大量流动水冲洗至少30分钟 |
| **碱烧伤**（石灰、氢氧化钠等）⚠️更危险 | 干布擦去固体粉末 → 再大量水冲洗30分钟+ |
| **磷烧伤** | 浸入水中，水中去除磷颗粒，创面湿敷，**不要暴露空气中** |
| **氢氟酸** | ⚠️立即冲洗+葡萄糖酸钙凝胶涂抹+紧急就医 |
| **眼部化学伤** | 掰开眼睑，从内眼角向外眼角冲洗至少20分钟 |

### 化学烧伤通用处理流程

1. 脱去污染衣物（戴手套保护自己）
2. 大量流动水冲洗（至少20-30分钟）
3. **不要试图用酸碱中和！**（中和反应产热加重损伤）
4. 拨打120

---

## 五、电烧伤

> ⚠️ 电烧伤表面可能很小，但深层组织损伤严重！必须就医！

### 处理步骤

1. **先断电！** 切断电源或用于非导电物体（干燥木棍）移开电线
2. **不要直接接触伤者**（可能导电）
3. 检查意识和呼吸 → 如无 → 立即CPR+AED
4. 处理入口和出口烧伤创面
5. 所有电烧伤患者**必须送医**（监测心律失常、肾损伤）

---

## 六、特殊烧伤

| 类型 | 处理要点 |
|------|----------|
| 沥青烧伤 | 冷水持续冲淋使沥青冷却凝固，**不要强行撕扯**，送医处理 |
| 口腔烫伤 | 含冷水/冰块，不要吞咽；严重需就医防气道水肿 |
| 晒伤 | 冷敷、芦荟胶涂抹、多饮水；如出现水泡/发热/头痛→就医 |

---

## 七、烧伤急救禁忌

| ❌ 绝对禁止 | ❌ 为什么 |
|-------------|-----------|
| 涂牙膏/酱油/醋/蛋清 | 引起感染、干扰医生判断、散热受阻 |
| 涂红药水/紫药水 | 染色影响深度判断 |
| 冰敷 | 造成冻伤+血管收缩加重损伤 |
| 挑破水泡 | 感染风险，水泡液含保护因子 |
| 撕掉粘在伤口上的衣物 | 撕脱皮肤加重损伤 |
| 大面积烧伤大量喝水 | 可能引起脑水肿，需静脉补液 |
