# 弥散神经内分泌系统（DNES）组成与功能

## 何时使用
- 在组织学或病理学分析中发现非传统内分泌器官内存在含致密核芯小泡的细胞；
- 遇到不明来源的多肽类或胺类激素分泌现象；
- 诊断或解释类癌综合征、异位激素分泌等临床情况；
- 需要理解局部激素调节机制（如旁分泌）在消化、呼吸或泌尿系统中的作用。

## 执行步骤
1. **确认细胞是否符合DNES基本前提**：
   - 细胞含有圆形、具致密核芯的神经内分泌小泡；
   - 能分泌胺类或多肽类物质（如胃泌素、降钙素、肾上腺素等）。

2. **排除非适用对象**：
   - 不包括甲状腺滤泡细胞等经典内分泌腺细胞；
   - 不涵盖中枢神经系统纯神经元。

3. **识别典型分布部位并关联功能**：
   - **胃肠道/胰腺/胆管**：分泌胃泌素、胆囊收缩素等，调节消化功能；
   - **呼吸道/泌尿生殖道**：分泌局部特异性激素，调节黏膜防御、平滑肌收缩等；
   - **甲状腺滤泡旁细胞**：分泌降钙素，参与钙代谢；
   - **肾上腺髓质嗜铬细胞**：分泌肾上腺素/去甲肾上腺素，介导应激反应；
   - 其他成员（如颈动脉体细胞、血管内皮细胞等）也属DNES，参与广泛生理调节。

4. **判断分泌方式**：
   - **旁分泌**：作用于邻近细胞；
   - **内分泌**：经血流作用于远处靶细胞。

5. **输出提醒**：
   - 提醒用户：在非传统内分泌组织中可能存在功能性激素分泌细胞，需考虑DNES在疾病（如类癌）或整体神经-内分泌-免疫网络调节中的作用。
