# D09 创意领域任务目录

## 领域定义

创意领域(D09)关注创意想法的产生、发展、实现和传播。核心驱动需求：N4自我实现。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人创意表达 | 大规模工业生产 |
| 艺术作品创作 | 传统手工艺制作 |
| 内容创作与传播 | 基础美术教育 |
| 创意项目策划 | 文化遗产保护 |
| 创意产业运营 | 政府文化政策 |
| 创意工具使用 | 纯技术开发 |
| 创意团队协作 | 传统媒体运营 |
