# 转场 Prompt 示例库

按叙事场景分类，提供可直接引用的转场 prompt 模板。

---

## 时间转场

从现在过渡到过去/未来，或跨越较长时间跨度。

### 当下 → 回忆（闪回）

```
[角色]望向[某物]，下一秒，画面切至[回忆场景]
```

**示例**：
```
女人轻抚婚纱照，淡出
淡入 → 十年前的婚礼现场，她笑靥如花
```

### 回忆 → 当下

```
[回忆画面]，下一秒，画面骤转 [当下场景]
```

**示例**：
```
战火中的奔跑，下一秒，画面切至老兵独坐窗前
```

### 时间流逝（季节/年代）

```
[当前季节意象] ↔ [目标季节意象]
```

**示例**：
```
飘落的秋叶 ↔ 积雪的屋檐，冬日降临
樱花纷飞 ↔ 红叶满地，岁月如梭
女人驻足窗前 ↔ 窗外白雪皑皚，又是一年
```

### 多年后

```
[场景]，[情绪]，淡出
淡入 → [同一场景/人物，若干年后]，[新状态]
```

**示例**：
```
两人并肩走在雨中，淡出
淡入 → 同一街道，她独自撑伞，身旁空无一人
```

## 空间转场

从一地切换到另一地，依赖方向性或路径逻辑。

### 室内 → 室外

```
[室内最后一个镜头]，↓划出 → 淡入 [室外场景]
```

**示例**：
```
电梯门关闭，↓划出
淡入 → 摩天大楼外景，俯瞰城市
```

### 远景 → 近景（进入）

```
[远景/环境]，↓划入 [人物/细节近景]
```

**示例**：
```
废弃工厂全景，↓划入锈迹斑斑的铁门
古镇远景，↓划入石板路上的脚印
```

### 同一空间的方向位移

```
[镜头A]，[移动描述]，←划入 [镜头B]
```

**示例**：
```
走廊尽头，镜头沿走廊向左移动，←划入病房门牌
赛车直道加速 → ↗划入终点维修站
```

### 门/窗框作为过渡

```
[穿过门/窗的动作]，下一秒，画面切至[新空间]
```

**示例**：
```
推开旧木门，吱呀作响，下一秒，画面切至阁楼内部
破窗跃出，下一秒，画面切至楼下游击队接应
```

## 情绪转场

情绪的延续、升级或转换，通过视觉意象的类比/对比实现。

### 悲伤延续

```
[悲伤意象] ↔ [同类悲伤意象]
```

**示例**：
```
泪滴滑落脸庞 ↔ 雨滴敲打窗玻璃
空荡荡的餐桌 ↔ 熄灭的烛台，孤独的晚餐
```

### 紧张升级

```
[突发事件]，下一秒，画面切至[紧张场景]
下一秒，画面切至[更紧张场景]
```

**示例**：
```
门锁崩裂，下一秒，画面切至走廊狂奔
破门而入，下一秒，画面骤转至枪口对准
```

### 释然/平静

```
[紧张释放]，淡出
淡入 → [平静意象]
```

**示例**：
```
最后一口气呼出，淡出
淡入 → 海浪轻拍沙滩，晨曦初现
```

### 希望萌芽

```
[绝望意象] ↔ [希望意象]
```

**示例**：
```
熄灭的油灯 ↔ 远处灯塔光芒，黎明将至
枯萎的花 ↔ 新芽破土，春回大地
```

## 场景转换（综合型）

复杂的叙事跳转，结合多种转场方式。

### 梦醒时分

```
[梦境/异常画面]，下一秒，画面骤转 [现实]
```

**示例**：
```
坠落深渊，下一秒，画面切至她猛然惊醒，冷汗涔涔
```

### 幻觉/现实切换

```
[现实场景] ↔ [幻觉意象]，交替叠化
```

**示例**：
```
他的脸温柔微笑 ↔ 他的脸扭曲成厉鬼，短暂交替
```

### 章节/幕间分隔

```
[本章最后一个镜头]，情绪收敛，淡出
淡入 → [下一章第一个镜头]
```

**示例**：
```
血染的婚纱，淡出
淡入 → 黑白照片，陈旧的婚姻登记处
```

### 并行叙事（双线）

```
[场景A] ↔ [场景B]，交替叠化
```

**示例**：
```
他在雨中奔跑 ↔ 她在窗边等候，时间在同一夜晚
```

## AI 短剧常用转场短语

可直接嵌入 prompt 的转场词组：

| 场景 | 转场 Prompt |
|------|-------------|
| 回忆 | `淡出 → 淡入，十年前` |
| 惊醒 | `下一秒，画面切至` |
| 时间流逝 | `↔` 叠化意象 |
| 穿越门 | `推开[门]，下一秒，画面切至` |
| 并行叙事 | `↔ 交替叠化` |
| 情绪收束 | `淡出` |
| 情绪开启 | `淡入` |
| 空间位移 | `↓划入` / `←划入` |
