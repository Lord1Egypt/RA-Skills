# Play Rediscovery Guide
帮助成年人重新发现“玩”的能力，找回创造力、连接感和真正的放松。

## What it does
- 🧒 玩乐历史回溯：追溯童年喜欢的活动，识别什么信念扼杀了玩耍
- 🎭 成人版玩乐实验：生成个性化的低门槛玩乐方案（即兴游戏、创意手工、线下社交活动）
- 📅 玩乐融入日常：将“玩乐约会”像会议一样排入日程，建立可持续的松弛感

## Example scenarios

**1. 30岁了除了刷手机不知道玩什么**
> 👤 小时候挺会玩的，现在周末除了刷抖音追剧就不知道干啥了，感觉很无聊。
> 🤖 提供'成年人重新学玩'计划：1）回顾童年最喜欢的3项活动重新尝试；2）推荐低门槛线下活动：剧本杀/密室逃脱、Switch聚会游、室内攀岩/射箭；3）建议参加豆瓣同城或即刻App的本地兴趣小组，从一个人玩变成跟大家一起玩。

**2. 中年职场人忘了怎么找乐子**
> 👤 我已经好几年没有纯粹为了开心做一件事了，所有事情都是为了一些目的。
> 🤖 引导玩乐历史回溯：童年爱做什么、什么信念扼杀了玩乐（“太幼稚”“没时间”）、哪些成人版活动可能适合现在。结果：用户重新发现童年搭建模型的爱好，承诺每周一小时“无结果要求”的创意时间。

**3. 高压管理者只会用消费解压**
> 👤 下班回家就刷手机喝酒，但从来没真正放松过，感觉很空虚。
> 🤖 区分消费型解压与创造性玩乐，建议基于限制的创意练习（30分钟无结果绘画、为过程而非成品的烹饪），引导建立每周“玩乐约会”并给予与董事会会议同等的日程尊重。
