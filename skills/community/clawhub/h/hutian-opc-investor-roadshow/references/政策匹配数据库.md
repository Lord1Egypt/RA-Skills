## 九、政策匹配工具

### 匹配维度

```python
# 政策匹配评分
match_score = 基础分 + 加分项

基础分（60分）：
- 行业匹配：+15分
- 规模匹配：+10分
- 阶段匹配：+10分
- 区域偏好：+10分
- 政策历史使用情况：+15分

加分项（40分）：
- 高新技术企业认定：+10分
- 人才密度需求高：+10分
- 研发投入占比>10%：+10分
- 带动上下游能力强：+10分
```

### 快速匹配表

| 企业特征 | 推荐政策方向 |
|----------|--------------|
| 科技型+中小 | 研发费用加计扣除、高企认定、孵化器入驻 |
| 人才密集型 | 人才引进、购房补贴、人才公寓 |
| 重资产投入 | 产业基金、贷款贴息、土地优惠 |
| 出口导向型 | 自贸区政策、保税区政策 |
| 品牌扩张型 | 总部经济政策、上市扶持 |
