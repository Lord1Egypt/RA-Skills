# 阴阳五行

## 阴阳

阴阳是根本的分类法则：

- **阳**：天、日、男、刚、动、热、明、奇数
- **阴**：地、月、女、柔、静、冷、暗、偶数

在占卜体系中：
- 天干中甲丙戊庚壬为阳干，乙丁己辛癸为阴干
- 地支中子寅辰午申戌为阳支，丑卯巳未酉亥为阴支
- 六爻中阳爻为"—"，阴爻为"- -"

## 五行基本属性

### 木
- 方位：东
- 季节：春
- 颜色：青、绿
- 五味：酸
- 五脏：肝
- 天干：甲乙
- 地支：寅卯
- 属性：生发、条达、仁

### 火
- 方位：南
- 季节：夏
- 颜色：赤、红
- 五味：苦
- 五脏：心
- 天干：丙丁
- 地支：巳午
- 属性：炎上、温热、礼

### 土
- 方位：中
- 季节：四季末
- 颜色：黄
- 五味：甘
- 五脏：脾
- 天干：戊己
- 地支：辰戌丑未
- 属性：稼穑、承载、信

### 金
- 方位：西
- 季节：秋
- 颜色：白
- 五味：辛
- 五脏：肺
- 天干：庚辛
- 地支：申酉
- 属性：从革、肃杀、义

### 水
- 方位：北
- 季节：冬
- 颜色：黑
- 五味：咸
- 五脏：肾
- 天干：壬癸
- 地支：亥子
- 属性：润下、寒凉、智

## 五行生克

### 相生（顺生）
```
木 → 火 → 土 → 金 → 水 → 木
```

- 木生火：木燃生火
- 火生土：火烬成土
- 土生金：土中出金
- 金生水：金熔成水（或金寒凝露）
- 水生木：水润生木

### 相克（顺克）
```
木 → 土 → 水 → 火 → 金 → 木
```

- 木克土：木根穿土
- 土克水：土堤挡水
- 水克火：水浇灭火
- 火克金：火熔金属
- 金克木：斧砍树木

### 反克（侮）
当被克一方力量过强时，可反过来克制对方：
- 木本克土，土重则木折
- 火本克金，金多则火熄
- 土本克水，水多则土崩
- 金本克木，木坚则金缺
- 水本克火，火炎则水干

## 五行在占卜中的常用应用

### 颜色对应（断方位、颜色线索时使用）
- 木=青绿，火=红赤，土=黄棕，金=白，水=黑蓝

### 身体部位对应（断健康时使用）
- 木=肝胆、筋骨
- 火=心脏、小肠、血脉
- 土=脾胃、肌肉
- 金=肺、大肠、皮肤
- 水=肾、膀胱、耳

### 五常对应（断人事品性）
- 木主仁、火主礼、土主信、金主义、水主智
