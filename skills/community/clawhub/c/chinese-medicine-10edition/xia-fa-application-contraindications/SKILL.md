# Xia Fa Application Contraindications

## When to Use
- 计划对确诊为里实证的患者使用下法（如大承气汤、十枣汤等）；
- 需排除表证、半表半里证或其他禁忌情况；
- 患者存在高龄、产后、妊娠、经期或素体虚弱等特殊状态。

## Execution Steps
1. **确认前提条件**：
   - 已明确诊断为里实证；
   - 表证与半表半里证已排除。

2. **评估禁忌情形**：
   - **病位禁忌**：邪在表、邪在半表半里（如少阳证）、阳明腑未实者，禁用下法；
   - **体质禁忌**：高龄津枯便秘者、素体虚弱或阳气衰微者、脾胃虚弱者，禁用峻下；
   - **特殊生理状态**：妊娠期、行经期、新产后营血不足者，禁用下法。

3. **执行核心原则**：
   - 遵循“以邪去为度”，服药后大便已通即应“中病即止”；
   - 峻下逐水剂必须严格掌握适应证与剂量，避免过量或久用。

4. **风险警示**：
   - 若误用于虚证或表证，可能导致“正气更伤”或“引邪深入”等严重后果；
   - 出现任一禁忌条件时，应禁止使用下法，并考虑替代疗法（如润下、温下或扶正祛邪并用）。

> **提示**：本技能适用于中医临床决策支持，不替代专业辨证论治。
