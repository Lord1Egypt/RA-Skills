# D04 职业领域任务要求

## 使用说明

本文件定义职业领域50个任务类型的组件要求。每个任务包含：
- **必选组件**: 完成任务必须具备的输入/资源
- **可选组件**: 增强任务质量的补充输入
- **组装顺序**: 组件的使用顺序
- **约束**: 任务执行的限制条件
- **格式**: 输出的标准格式

## C1 职业定位

### C1-01 自我认知
- **必选组件**: 兴趣爱好梳理、价值观评估、性格特质分析
- **可选组件**: 能力自评、过往经历回顾、他人反馈
- **组装顺序**: 梳理兴趣爱好→评估核心价值观→分析性格特质→识别优势劣势→形成自我认知报告
- **约束**: 需要客观真实、需要多角度评估
- **格式**: 自我认知报告(兴趣/价值观/性格/优势/劣势)

### C1-02 行业分析
- **必选组件**: 目标行业选择、行业趋势分析、市场机会评估
- **可选组件**: 竞争格局分析、技术发展影响、政策环境分析
- **组装顺序**: 确定目标行业→收集行业数据→分析行业趋势→评估市场机会→识别风险挑战
- **约束**: 数据需要及时准确、分析需要全面客观
- **格式**: 行业分析报告(行业概况/趋势/机会/风险/建议)

### C1-03 岗位研究
- **必选组件**: 目标岗位选择、岗位职责分析、任职要求研究
- **可选组件**: 薪资水平调研、发展路径分析、工作环境评估
- **组装顺序**: 确定目标岗位→收集岗位信息→分析岗位职责→研究任职要求→评估匹配程度
- **约束**: 信息需要真实可靠、需要考虑长期发展
- **格式**: 岗位研究报告(岗位概况/职责/要求/薪资/发展)

### C1-04 能力盘点
- **必选组件**: 技能清单、经验总结、知识储备评估
- **可选组件**: 证书资质、项目成果、他人评价
- **组装顺序**: 梳理专业技能→总结工作经验→评估知识储备→识别核心优势→发现能力短板
- **约束**: 需要客观真实、需要与目标岗位对比
- **格式**: 能力盘点表(技能/经验/知识/优势/短板)

### C1-05 价值观排序
- **必选组件**: 职业价值观列表、重要性评估、优先级排序
- **可选组件**: 价值观冲突分析、价值观实现路径、价值观调整建议
- **组装顺序**: 列出职业价值观→评估重要性程度→进行优先级排序→分析价值观冲突→确定核心价值观
- **约束**: 需要真实反映内心、需要考虑现实约束
- **格式**: 价值观排序表(价值观/重要性/优先级/实现路径)

### C1-06 职业锚定
- **必选组件**: 自我认知结果(C1-01)、能力盘点结果(C1-04)、职业锚定类型测试
- **可选组件**: 职业锚定理论参考、成功案例借鉴、专家建议
- **组装顺序**: 整合自我认知结果→分析能力优势→进行职业锚定测试→确定锚定类型→制定锚定策略
- **约束**: 需要综合考虑多因素、需要与实际情况匹配
- **格式**: 职业锚定报告(锚定类型/优势/发展路径/行动计划)

### C1-07 职业愿景
- **必选组件**: 职业锚定结果(C1-06)、长期目标设定、发展路径规划
- **可选组件**: 愿景可视化、阶段目标分解、风险应对策略
- **组装顺序**: 整合职业锚定结果→设定长期职业目标→规划发展路径→分解阶段目标→制定行动计划
- **约束**: 愿景需要清晰可行、需要定期回顾调整
- **格式**: 职业愿景规划(长期目标/发展路径/阶段目标/行动计划)

## C2 求职策略

### C2-01 简历撰写
- **必选组件**: 个人信息、教育背景、工作经历
- **可选组件**: 项目经历、技能证书、荣誉奖项
- **组装顺序**: 整理个人信息→梳理教育背景→总结工作经历→突出关键成就→优化格式排版
- **约束**: 需要针对目标岗位定制、需要简洁明了
- **格式**: 简历(个人信息/教育/工作/技能/项目)

### C2-02 求职信写作
- **必选组件**: 目标公司信息、岗位要求分析、个人匹配点
- **可选组件**: 公司文化了解、行业见解、个人故事
- **组装顺序**: 研究目标公司→分析岗位要求→找到匹配点→撰写求职信→优化表达
- **约束**: 需要个性化定制、需要简洁有力
- **格式**: 求职信(开头/主体/结尾)

### C2-03 LinkedIn优化
- **必选组件**: 专业头像、职业标题、工作经历
- **可选组件**: 技能背书、推荐信、文章分享
- **组装顺序**: 上传专业头像→优化职业标题→完善工作经历→添加技能背书→请求推荐信
- **约束**: 需要保持专业形象、需要定期更新
- **格式**: LinkedIn个人资料(头像/标题/经历/技能/推荐)

### C2-04 求职渠道管理
- **必选组件**: 求职渠道清单、渠道特点分析、投递策略
- **可选组件**: 渠道效果追踪、人脉推荐、猎头合作
- **组装顺序**: 梳理求职渠道→分析渠道特点→制定投递策略→执行投递计划→追踪投递效果
- **约束**: 需要多渠道并行、需要持续跟进
- **格式**: 求职渠道管理表(渠道/特点/策略/效果)

### C2-05 面试准备
- **必选组件**: 面试类型分析、常见问题准备、模拟面试练习
- **可选组件**: 公司背景研究、行业知识准备、着装礼仪准备
- **组装顺序**: 分析面试类型→准备常见问题→练习回答技巧→进行模拟面试→调整状态应试
- **约束**: 需要充分准备、需要真实模拟
- **格式**: 面试准备计划(面试/问题/练习/模拟/调整)

### C2-06 薪资谈判
- **必选组件**: 市场薪资调研、个人价值评估、谈判策略制定
- **可选组件**: 谈判技巧学习、底线设定、替代方案准备
- **组装顺序**: 调研市场薪资→评估个人价值→制定谈判策略→准备谈判话术→执行谈判
- **约束**: 需要了解市场行情、需要合理预期
- **格式**: 薪资谈判方案(市场调研/个人价值/策略/话术)

### C2-07 背景调查应对
- **必选组件**: 个人信息核实、工作经历确认、证明人准备
- **可选组件**: 背景调查公司了解、常见问题准备、应对策略
- **组装顺序**: 核实个人信息→确认工作经历→准备证明人→了解调查流程→准备应对策略
- **约束**: 信息需要真实一致、需要提前沟通证明人
- **格式**: 背景调查准备清单(信息/经历/证明人/策略)

### C2-08 入职决策
- **必选组件**: Offer分析、公司评估、个人需求匹配
- **可选组件**: 多Offer比较、长期发展评估、风险评估
- **组装顺序**: 分析Offer内容→评估公司情况→匹配个人需求→比较多个Offer→做出入职决策
- **约束**: 需要综合考虑多因素、需要考虑长期发展
- **格式**: 入职决策分析(Offer分析/公司评估/匹配度/建议)

## C3 工作执行

### C3-01 目标管理
- **必选组件**: 工作职责、目标设定方法、执行计划
- **可选组件**: 目标分解、进度追踪、调整机制
- **组装顺序**: 明确工作职责→设定工作目标→分解目标任务→制定执行计划→建立追踪机制
- **约束**: 目标需要SMART、需要定期回顾
- **格式**: 目标管理表(目标/任务/计划/进度/调整)

### C3-02 任务优先级
- **必选组件**: 任务清单、优先级评估标准、时间估算
- **可选组件**: 任务分类、委派策略、批量处理
- **组装顺序**: 列出所有任务→评估优先级→估算时间需求→安排执行顺序→执行并调整
- **约束**: 需要客观评估、需要灵活调整
- **格式**: 任务优先级表(任务/优先级/时间/状态)

### C3-03 时间管理
- **必选组件**: 时间管理方法、工具选择、时间安排
- **可选组件**: 时间记录、效率分析、优化建议
- **组装顺序**: 选择时间管理方法→配置管理工具→制定时间安排→执行并记录→分析并优化
- **约束**: 方法需要简单易用、需要持续执行
- **格式**: 时间管理方案(方法/工具/安排/记录/优化)

### C3-04 会议管理
- **必选组件**: 会议目的、会议议程、参会人员
- **可选组件**: 会议材料、会议记录、行动计划
- **组装顺序**: 明确会议目的→制定会议议程→确定参会人员→准备会议材料→主持并记录
- **约束**: 需要控制时间、需要明确产出
- **格式**: 会议管理方案(目的/议程/人员/材料/记录)

### C3-05 邮件沟通
- **必选组件**: 邮件目的、收件人分析、邮件内容
- **可选组件**: 邮件模板、跟进策略、效率工具
- **组装顺序**: 明确邮件目的→分析收件人→撰写邮件内容→优化表达→发送并跟进
- **约束**: 需要简洁明了、需要专业得体
- **格式**: 邮件沟通方案(目的/收件人/内容/跟进)

### C3-06 项目管理
- **必选组件**: 项目目标、项目计划、资源分配
- **可选组件**: 风险管理、进度追踪、沟通计划
- **组装顺序**: 明确项目目标→制定项目计划→分配资源→执行并监控→调整并收尾
- **约束**: 需要系统规划、需要灵活应对
- **格式**: 项目管理方案(目标/计划/资源/风险/进度)

### C3-07 跨部门协作
- **必选组件**: 协作目标、利益相关方、沟通机制
- **可选组件**: 协作流程、冲突解决、效果评估
- **组装顺序**: 明确协作目标→识别利益相关方→建立沟通机制→制定协作流程→执行并评估
- **约束**: 需要建立信任、需要明确责任
- **格式**: 跨部门协作方案(目标/相关方/沟通/流程/评估)

## C4 绩效发展

### C4-01 绩效目标设定
- **必选组件**: 工作职责、绩效指标、目标值
- **可选组件**: 关键结果、权重分配、评估标准
- **组装顺序**: 明确工作职责→设定绩效指标→确定目标值→设定关键结果→制定行动计划
- **约束**: 目标需要SMART、需要与上级对齐
- **格式**: 绩效目标表(指标/目标值/关键结果/权重)

### C4-02 自我评估
- **必选组件**: 绩效目标(C4-01)、实际表现、评估标准
- **可选组件**: 数据支撑、案例说明、改进计划
- **组装顺序**: 回顾绩效目标→评估实际表现→对比评估标准→分析差距原因→制定改进计划
- **约束**: 需要客观真实、需要数据支撑
- **格式**: 自我评估报告(目标/表现/差距/原因/改进)

### C4-03 绩效面谈准备
- **必选组件**: 自我评估结果(C4-02)、工作成果展示、发展需求
- **可选组件**: 反馈准备、问题清单、谈判策略
- **组装顺序**: 整理自我评估→准备工作成果→明确发展需求→准备反馈和问题→制定面谈策略
- **约束**: 需要充分准备、需要积极态度
- **格式**: 绩效面谈准备清单(评估/成果/需求/策略)

### C4-04 晋升策略
- **必选组件**: 晋升要求分析、个人能力评估、行动计划
- **可选组件**: 导师建议、成功案例、风险应对
- **组装顺序**: 分析晋升要求→评估个人能力→识别差距→制定行动计划→执行并调整
- **约束**: 需要现实可行、需要持续努力
- **格式**: 晋升策略方案(要求/能力/差距/计划/调整)

### C4-05 跳槽决策
- **必选组件**: 跳槽原因分析、机会成本评估、风险评估
- **可选组件**: 市场调研、多机会比较、长期规划
- **组装顺序**: 分析跳槽原因→评估机会成本→评估风险→调研市场机会→做出决策
- **约束**: 需要理性分析、需要考虑长期
- **格式**: 跳槽决策分析(原因/成本/风险/机会/决策)

### C4-06 转型规划
- **必选组件**: 转型目标、能力差距分析、转型路径
- **可选组件**: 学习计划、实践机会、风险应对
- **组装顺序**: 确定转型目标→分析能力差距→设计转型路径→制定学习计划→执行并调整
- **约束**: 需要现实可行、需要持续投入
- **格式**: 转型规划方案(目标/差距/路径/计划/调整)

### C4-07 职业瓶颈突破
- **必选组件**: 瓶颈识别、原因分析、突破策略
- **可选组件**: 导师建议、学习提升、机会创造
- **组装顺序**: 识别职业瓶颈→分析瓶颈原因→制定突破策略→执行行动计划→评估突破效果
- **约束**: 需要客观分析、需要积极行动
- **格式**: 瓶颈突破方案(瓶颈/原因/策略/行动/效果)

- **约束**: 需要合理估值、需要保护控制权

- **组装顺序**: 设计运营流程→分配资源→建立监控体系→执行并优化→持续改进
- **约束**: 需要高效执行、需要灵活调整
- **格式**: 运营管理方案(流程/资源/监控/优化/改进)

### C5-07 退出策略
- **必选组件**: 退出方式选择、退出时机判断、退出准备
- **可选组件**: 估值评估、买家寻找、法律准备
- **组装顺序**: 选择退出方式→判断退出时机→准备退出材料→寻找买家/投资者→执行退出
- **约束**: 需要最大化价值、需要保护利益
- **格式**: 退出策略方案(方式/时机/准备/执行/分配)

## C6 副业自由职业

- **必选组件**: 技能匹配分析、市场需求评估、时间投入评估
- **可选组件**: 收入潜力、风险评估、发展前景
- **组装顺序**: 分析技能匹配→评估市场需求→评估时间投入→预测收入潜力→做出机会判断
- **约束**: 需要与主业平衡、需要考虑风险

### C6-02 自由职业定位
