# Legal Risk Shield — 企业法律风险防控体系

> 从公司设立到市场退出的全生命周期法律风险防控 + 企业家个人全场景风控护甲

## 概述

融合三大权威来源：
1. **广东高院** — 民营企业法律风险防控手册2.0版（法院实务指引）
2. **金杜律所** — 企业家个人风控手册2025年版（62个专题+30个高频罪名）
3. **苏州检察院** — 企业常见法律咨询解答及风险防控指南2025年版（检察视角）

## 核心内容

- 公司设立→治理→融资→合同→退出 全生命周期风险防控
- 企业家个人：民商事风险+家事财富管理+刑事合规
- 30个高频犯罪罪名清单及量刑
- 企业合规治理五步法
- 企业家"护心甲"十诫
