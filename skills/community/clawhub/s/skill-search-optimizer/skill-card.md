## Description: <br>
Optimize agent skills for discoverability on ClawdHub/MoltHub. Use when improving search ranking, writing descriptions for semantic search, understanding how the registry indexes skills, testing search visibility, or analyzing why a skill isn't being found. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and skill authors use this skill to improve ClawHub/MoltHub search visibility by refining skill descriptions, checking discoverability, and comparing search positioning before or after publishing. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Command examples using npx molthub@latest run the current published CLI and may perform registry search, publish, or install actions if the user executes them. <br>
Mitigation: Review each command before running it and execute only the registry action intended for the current workflow. <br>
Risk: Search optimization guidance can encourage descriptions that overstate what a skill actually does. <br>
Mitigation: Keep published descriptions aligned with the skill's real behavior and test search visibility with representative queries before release. <br>
