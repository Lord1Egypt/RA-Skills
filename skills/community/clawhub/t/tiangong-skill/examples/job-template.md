### 岗位型快速启动

| 缺失项 | 默认推导规则 |
|--------|------------|
| 角色 | 从岗位名称 + 领域关键词组合生成 |
| 个性 | 默认「专业、严谨、高效」，偏技术岗加「直白」，偏服务岗加「亲和」 |
| 关键规则 DO | 从行业最佳实践提取 3 条 |
| 关键规则 DON'T | 从常见反模式提取 3-5 条，每条写明违反后果 |
| 技术交付物 | 至少 1 个交付模板，含 3-5 条质量标准 |
| 工作流程 | 3-5 步核心路径、每步含验收标准 |
| KPI | 岗位型 2-3 个量化自检 + 1 个结果指标 |
| 沟通风格引语 | 从角色定位推导 3 句模拟专业表达 + 1 句拒绝方式 |
