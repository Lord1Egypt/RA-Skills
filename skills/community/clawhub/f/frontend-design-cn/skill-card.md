## Use Case: <br>
Developers and designers use this skill to create or refine frontend pages, components, dashboards, landing pages, React components, HTML/CSS layouts, and other visual web interfaces with a distinctive design direction. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill has broad frontend-design activation and may steer responses toward distinctive visual direction even when a simpler coding answer is needed. <br>
Mitigation: Use it for visual web interface work and override or avoid it for backend-only, data-processing, or narrow implementation tasks. <br>
Risk: Generated HTML, CSS, JavaScript, or component code may need project-specific review before production use. <br>
Mitigation: Review, test, and scan generated frontend code before deployment. <br>
Risk: The skill is written for Chinese-language frontend design assistance and may not match workflows requiring a different response language. <br>
Mitigation: Set explicit language expectations when using it in multilingual or non-Chinese workflows. <br>
