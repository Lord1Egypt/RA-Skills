# 六大區域檢查項目

## 1. 結構 (Structure)

### 樑
- [ ] 目視是否有45度裂縫（剪力裂縫）
- [ ] 裂縫寬度是否超過0.3mm
- [ ] 樑下是否有漏水/水漬痕跡
- [ ] 樑的水平是否有彎曲下陷

### 柱
- [ ] 柱是否有垂直或水平裂縫
- [ ] 柱頭/柱腳是否有鋼筋外露
- [ ] 柱體是否有膨脹變形
- [ ] 裝修包覆柱體時是否破壞結構

### 樓板
- [ ] 樓板是否有裂縫貫穿上下層
- [ ] 樓板是否有異常傾斜（水平儀檢測）
- [ ] 是否有漏水痕跡從樓上滲入

### 牆體（承重牆）
- [ ] 是否有結構裂縫
- [ ] 是否被擅自打除或變更
- [ ] 外牆內側是否有壁癌或滲水

### 地基
- [ ] 1樓地坪是否有不均勻沉陷裂縫
- [ ] 1樓柱是否有明顯傾斜（鉛垂儀檢查）
- [ ] 建築物與鄰房是否有高低差（不均勻沉陷）
- [ ] 是否有土壤流失跡象

### 特殊檢查（根據屋齡決定）
- [ ] 海砂屋檢測（氯離子含量）
- [ ] 輻射鋼筋檢測
- [ ] 屋齡>30年：結構安全評估報告

---

## 2. 水電 (Electrical & Plumbing)

### 配電箱
- [ ] 總開關容量標示是否清楚
- [ ] 是否有漏電斷路器
- [ ] 分路開關功能是否正常
- [ ] 電線規格是否與開關匹配
- [ ] 箱內是否有過熱痕跡（燒焦/變色）
- [ ] 配電箱周圍是否為易燃裝修

### 插座與開關
- [ ] 每個房間插座是否可正常供電
- [ ] 插座三孔是否有接地線
- [ ] 廚房/浴室插座是否為防水型
- [ ] 開關作動是否正常
- [ ] 插座有沒有鬆動

### 管線
- [ ] 熱水管保溫層是否完好
- [ ] 明管是否有鏽蝕
- [ ] 水管水壓是否正常（打開水龍頭測試）
- [ ] 排水管是否順暢（大量放水測試）
- [ ] 排水管是否有異味回溢

### 給水設備
- [ ] 水塔位置及材質
- [ ] 加壓馬達功能
- [ ] 水管材質（是否為鉛管或鏽蝕鐵管）
- [ ] 水龍頭水量及水溫穩定性

### 弱電
- [ ] 電話/網路插座是否可用
- [ ] 電視訊號線是否正常
- [ ] 對講機功能
- [ ] 是否有預留弱電管線

---

## 3. 門窗 (Doors & Windows)

### 大門
- [ ] 門框是否垂直無變形
- [ ] 開關是否順暢無雜音
- [ ] 門鎖是否安全、功能正常
- [ ] 門縫密封條是否完整
- [ ] 是否有防盜鏈/貓眼/門擋

### 房間門
- [ ] 門片是否有變形無法密合
- [ ] 把手鎖扣功能是否正常
- [ ] 門框是否有蛀蝕

### 窗戶
- [ ] 窗框與牆面接縫是否有裂縫/滲水
- [ ] 窗戶氣密條是否老化硬化
- [ ] 窗扇開關是否順暢
- [ ] 紗窗是否完整
- [ ] 窗軌排水孔是否暢通
- [ ] 窗框是否有鏽蝕（鋁窗/鐵窗）
- [ ] 玻璃是否有裂痕
- [ ] 落地窗密封是否良好

### 陽台門
- [ ] 防水門檻高度是否足夠
- [ ] 排水是否順暢
- [ ] 門框氣密性

---

## 4. 衛浴 (Bathroom)

### 防水
- [ ] 牆面磁磚是否有空心膨脹
- [ ] 浴缸/淋浴區周邊是否滲水
- [ ] 地板洩水坡度是否正常
- [ ] 門檻外側牆角是否有滲水痕跡
- [ ] 天花板是否有漏水痕跡（樓上滲漏）
- [ ] 乾溼分離玻璃門密封性

### 馬桶
- [ ] 沖水功能正常（衛生紙測試）
- [ ] 底座是否有滲水
- [ ] 水箱內部零件是否正常
- [ ] 是否有異味

### 洗臉台
- [ ] 排水是否順暢
- [ ] 水龍頭是否有漏水
- [ ] 櫃體是否有受潮變形
- [ ] 檯面是否有裂痕

### 熱水器
- [ ] 安裝位置通風是否良好
- [ ] 排氣管是否通往室外
- [ ] 使用年限是否超過（一般8-10年）
- [ ] 瓦斯管線是否有漏氣（泡沫水測試）

### 通風
- [ ] 抽風機運轉是否正常
- [ ] 排風管是否通暢連通室外
- [ ] 窗戶通風（如有）

---

## 5. 廚房 (Kitchen)

### 櫥櫃
- [ ] 門片是否平整無變形
- [ ] 鉸鏈/五金件功能是否正常
- [ ] 檯面是否有裂痕或燙傷
- [ ] 櫥櫃內部是否有受潮或發霉
- [ ] 踢腳板是否完整

### 水槽
- [ ] 排水是否順暢（大量放水）
- [ ] 水管接頭是否有滲漏
- [ ] 水槽本體是否有生鏽或裂痕
- [ ] 廚下是否有異味或蟑螂鼠跡

### 瓦斯
- [ ] 瓦斯管線是否為金屬軟管
- [ ] 瓦斯開關功能正常
- [ ] 瓦斯爐各爐頭點火功能
- [ ] 有無瓦斯洩漏

### 設備
- [ ] 抽油煙機運轉及排煙功能
- [ ] 烘碗機功能（如有）
- [ ] 冰箱預留空間及插座對位
- [ ] 洗碗機/烤箱功能（如有）

### 管線
- [ ] 排水管線是否通暢
- [ ] 是否有預留淨水器管線
- [ ] 油煙管是否通往室外管道

---

## 6. 天地壁 (Floor, Wall, Ceiling)

### 地板
- [ ] 地磚/木地板是否平整
- [ ] 地磚是否有空心（輕敲測試）
- [ ] 地板是否有隆起變形
- [ ] 磁磚縫隙是否填實
- [ ] 各空間地板高低差

### 牆壁
- [ ] 油漆是否有剝落
- [ ] 壁紙是否有翹起
- [ ] 壁癌（白色粉末/潮濕脫落）
- [ ] 輕隔間牆是否有龜裂
- [ ] 磁磚是否有脫落或空心

### 天花板
- [ ] 是否有水漬/漏水痕跡
- [ ] 天花板是否有裂縫或下陷
- [ ] 輕鋼架天花板是否平整
- [ ] 天花板內管線是否外露

### 其他
- [ ] 所有牆角是否有壁癌或結露
- [ ] 梁柱與牆面交接處是否有裂縫
- [ ] 管線穿牆處是否確實填縫
