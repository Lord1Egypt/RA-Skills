# 二尖瓣狭窄分度评估

## 何时使用
- 已获得二尖瓣狭窄患者的二尖瓣口面积测量值（单位：cm²）
- 需要依据国际通用标准对狭窄程度进行分类
- 不适用于主动脉瓣或其他瓣膜病变的评估

## 执行步骤
1. 确认输入为经影像学（如超声心动图）测定的二尖瓣口面积（valve_area，单位：cm²）。
2. 按以下阈值进行分度：
   - **轻度狭窄**：瓣口面积 > 1.5 cm² 且 ≤ 2.0 cm²
   - **中度狭窄**：瓣口面积 > 1.0 cm² 且 ≤ 1.5 cm²
   - **重度狭窄**：瓣口面积 ≤ 1.0 cm²
3. 注意：正常二尖瓣口面积参考范围为 4.0～6.0 cm²，可用于对比理解狭窄程度。
4. 提醒用户：此分度基于解剖面积，临床决策应结合血流动力学参数（如跨瓣压差、肺动脉压等）综合判断。

## 输出
返回狭窄程度分类：轻度、中度或重度。
