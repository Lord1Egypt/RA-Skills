## 招聘配置任务清单

招聘配置按子域组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：招聘规划 → 招聘实施 → 录用决策 → 入职管理

---

### 1 招聘规划

| ID | 任务类型 | 说明 | 依赖 | 元操作映射 |
|----|----------|------|------|----------|
| T1-1 | 需求分析 | 分析招聘需求，明确岗位要求和人才标准 | 无（入口） | S→C |
| T1-2 | 招聘计划 | 制定招聘计划，包括招聘人数、时间节点、招聘预算 | T1-1 | C→A |
| T1-3 | 渠道选择 | 选择招聘渠道，包括内部推荐、招聘网站、猎头、校园招聘 | T1-2 | C |
| T1-4 | 预算编制 | 编制招聘预算，包括渠道费用、面试费用、入职费用 | T1-3 | C→O |

---

### 2 招聘实施

| ID | 任务类型 | 说明 | 依赖 | 元操作映射 |
|----|----------|------|------|----------|
| T2-1 | 简历筛选 | 筛选简历，评估候选人基本条件和匹配度 | T1-4 | S→C |
| T2-2 | 面试安排 | 安排面试，包括初试、复试、终试 | T2-1 | A→I |
| T2-3 | 笔试测评 | 组织笔试和测评，评估候选人专业能力和综合素质 | T2-2 | S→C→G |
| T2-4 | 背景调查 | 调查候选人背景，核实信息真实性 | T2-3 | S→G |

---

### 3 录用决策

| ID | 任务类型 | 说明 | 依赖 | 元操作映射 |
|----|----------|------|------|----------|
| T3-1 | 录用评估 | 综合评估候选人，包括能力评估、潜力评估、文化匹配 | T2-4 | C |
| T3-2 | 薪酬谈判 | 与候选人谈判薪酬，包括薪资、福利、股权 | T3-1 | I |

| T3-4 | 入职准备 | 准备入职，包括工位、设备、账号、培训计划 | T3-3 | A→O |

### 4 入职管理

|----|----------|------|------|----------|

## 依赖拓扑摘要

### 一、域内链路

**招聘规划链路**: T1-1 → T1-2 → T1-3 → T1-4
**招聘实施链路**: T2-1 → T2-2 → T2-3 → T2-4
**录用决策链路**: T3-1 → T3-2 → T3-3 → T3-4
**入职管理链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整招聘配置链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T3-1 → T3-2 → T3-3 → T3-4 → T4-1 → T4-2 → T4-3 → T4-4

**反馈改进链路**: T4-4 → T1-1（转正评估反馈到需求分析）

**渠道优化链路**: T2-1 → T1-3（简历筛选反馈到渠道选择）

更多组合根据具体任务动态推导。
