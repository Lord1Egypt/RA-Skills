## Description: <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and testers use this skill to verify basic agent invocation by echoing user input and returning a timestamp. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill repeats user-provided text back into the conversation. <br>
Mitigation: Do not enter secrets, credentials, or private information when using this echo test skill. <br>
Risk: Broad activation wording may cause the skill to respond when the user did not specifically intend to invoke it. <br>
Mitigation: Use explicit invocation wording or scope the skill to controlled testing workflows. <br>
