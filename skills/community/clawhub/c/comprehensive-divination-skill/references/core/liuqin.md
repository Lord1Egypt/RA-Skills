# 六亲与用神体系

六亲是六爻预测的核心范畴。将一个六爻卦中的各爻按其五行与卦宫五行的关系，分配六亲名称。

## 六亲定义

以卦宫五行为"我"，各爻五行为"彼"定义六亲：

| 关系 | 六亲 | 含义 |
|------|------|------|
| 生我者 | 父母 | 庇护、文书、房屋、长辈、知识 |
| 我生者 | 子孙 | 晚辈、娱乐、医药、解忧、福德 |
| 克我者 | 官鬼 | 功名、职务、疾病、灾祸、盗贼 |
| 我克者 | 妻财 | 财富、物资、妻子、饮食 |
| 同我者 | 兄弟 | 同辈、竞争、耗财、口舌 |
| 比和者 | - | （同兄弟） |

## 六亲与用神选取

根据占问事项确定用神（核心爻位）：

### 感情/婚姻
- 看世爻代表自身
- 男命看妻财爻（代表女友/妻子），女命看官鬼爻（代表男友/丈夫）
- 应爻也代表对方态度
- 兄弟爻持世对感情不利（竞争者/耗神）

### 事业/工作/求职
- 官鬼爻为用神（代表职位、上级）
- 父母爻代表单位、文书、合同
- 妻财爻代表待遇、薪资
- 官鬼旺相临青龙 = 好岗位、有发展
- 子孙持世 = 对求职升迁不利（子孙克官鬼）

### 财运/财富
- 妻财爻为用神
- 子孙爻为财源（子孙生财）
- 兄弟爻为破财（兄弟克妻财）
- 官鬼爻有时也代表求财的障碍

### 健康/疾病
- 官鬼爻为病灶
- 子孙爻为医药解神
- 世爻代表自身身体状况
- 官鬼旺而克世 = 病重
- 子孙旺而制官鬼 = 药到病除

### 出行/出行吉凶
- 世爻为自身
- 应爻为目的地
- 官鬼爻为路上的障碍/危险
- 兄弟爻为花费

## 六兽（六神）

六爻卦中，每爻各配一神，从初爻起按日干顺排：

| 神名 | 属性 | 吉凶 | 含义 |
|------|------|------|------|
| 青龙 | 木 | 大吉 | 喜事、贵人、财利、婚庆 |
| 朱雀 | 火 | 中吉 | 文书、口舌、消息、诉讼 |
| 勾陈 | 土 | 小凶 | 田土、迟滞、牵连、牢狱 |
| 螣蛇 | 土 | 凶 | 虚惊、噩梦、怪异、纠缠 |
| 白虎 | 金 | 大凶 | 血光、疾病、丧事、暴力 |
| 玄武 | 水 | 凶 | 盗贼、暗昧、私情、欺骗 |

六神排法：甲乙日起青龙，丙丁日起朱雀，戊日起勾陈，己日起螣蛇，庚辛日起白虎，壬癸日起玄武。均从初爻开始依次向上。

## 世应与六爻排列

- **世爻**：代表占问者自身，在卦中位置由卦宫所属决定
- **应爻**：代表问事对象/对方，与世爻隔两位（世在初则应在四，世在二则应在五，世在三则应在六，世在四则应在初，世在五则应在二，世在六则应在三）
