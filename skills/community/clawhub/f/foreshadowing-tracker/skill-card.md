## Description: <br>
伏笔追踪器 - 识别章节中的伏笔并追踪回收状态。当需要管理伏笔、确保前后呼应时使用，支持新增伏笔识别、待回收伏笔提醒、已回收伏笔标记。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Writers, editors, and story-development agents use this skill to identify foreshadowing, hints, suspense, and callbacks in novel chapters, then track which items remain pending or have been resolved. It can generate local JSON reports and records, with an optional LLM script for model-assisted extraction. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The LLM tracker can send full manuscript text to DashScope when used with an API key. <br>
Mitigation: Do not run scripts/track_foreshadowing_llm.py on confidential drafts unless sending that text to DashScope is intentional and approved. <br>
Risk: Foreshadowing detection may be incomplete or inaccurate because the documented local tracker is keyword-based and the security guidance recommends review before installation for private drafts. <br>
Mitigation: Review generated findings before relying on them for story continuity, and scan or inspect the skill before deployment. <br>
