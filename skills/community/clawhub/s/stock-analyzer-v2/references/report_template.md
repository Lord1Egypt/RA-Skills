## 分析维度清单

### 可选维度
- [ ] 营收/利润增速
- [ ] ROE/ROA
- [ ] 行业对比

## 输出格式建议

### Markdown 格式
- 适合：邮件正文、GitHub、文档
- 优点：可读性好、支持表格

### HTML 格式
- 适合：网页展示、邮件富文本
- 优点：排版灵活、支持图表

### PDF 格式
- 适合：正式报告、存档
- 优点：格式固定、可打印
