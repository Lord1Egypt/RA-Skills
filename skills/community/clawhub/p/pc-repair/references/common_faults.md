# 常见电脑故障诊断流程图

本文档提供常见电脑故障的诊断决策树，帮助快速定位问题。

## 1. 无法开机故障诊断树

```
开始：按下电源键无反应
│
├─ 检查电源指示灯
│  ├─ 不亮 → 电源问题
│  │  ├─ 检查插座（插手机充电器测试）
│  │  ├─ 检查电源线（重新插拔）
│  │  └─ 更换电源线测试
│  │
│  └─ 亮但黑屏 → 转"有反应但黑屏"分支
│
└─ 风扇转但无显示 → 转"有反应但黑屏"分支
```

## 2. 有反应但黑屏诊断树

```
症状：风扇转、键盘灯可亮，但屏幕无显示
│
├─ 听蜂鸣器声音
│  ├─ "滴-滴"长鸣 → 内存故障
│  │  └─ 解决方案：拔插内存，用橡皮擦金手指
│  │
│  ├─ 无声音 → 转下一步
│  └─ 其他报警声 → 查主板说明书（不同BIOS报警声含义不同）
│
├─ 清除CMOS
│  ├─ 拔掉电源，取下主板电池，等待5分钟
│  └─ 装回电池，尝试开机
│
└─ 最小系统法
   ├─ 只连接：CPU + 内存 + 电源 + 显卡（如有核显则拔掉独显）
   ├─ 逐一添加配件测试
   └─ 找到故障配件
```

## 3. 蓝屏故障诊断树

```
出现蓝屏
│
├─ 记录错误代码（拍照或记下来）
│
├─ 最近是否安装新硬件/软件？
│  ├─ 是 → 进安全模式卸载
│  │  ├─ 硬件：拔掉新硬件，测试稳定性
│  │  └─ 软件：系统还原到安装前的时间点
│  │
│  └─ 否 → 转下一步
│
├─ 使用BlueScreenView分析dump文件
│  ├─ 找到导致崩溃的驱动文件（.sys）
│  └─ 更新或回滚该驱动
│
└─ 硬件压力测试
   ├─ 内存：运行MemTest86（4小时无错误才稳定）
   ├─ 硬盘：运行CrystalDiskInfo（查看健康度）
   └─ 温度：运行HWiNFO64（查看CPU/GPU温度）
```

## 4. 电脑卡顿诊断树

```
电脑运行缓慢
│
├─ 开机慢（>1分钟）
│  ├─ 硬盘是机械硬盘？ → 升级SSD（效果最明显）
│  └─ 开机启动项太多？ → 任务管理器→启动→禁用不必要项
│
├─ 运行软件卡
│  ├─ 内存占用率>80%？ → 加内存（8G→16G）
│  ├─ CPU占用率>80%？ → 查任务管理器（哪个进程占用高）
│  │  ├─ 系统进程 → 可能中病毒，杀毒
│  │  └─ 某个软件 → 卸载重装或找替代品
│  └─ 磁盘占用率100%？ → 硬盘有坏道，更换SSD
│
└─ 上网卡（只有浏览器卡）
   ├─ 测速测试（speedtest.cn）
   ├─ 带宽正常但网页打开慢 → 清浏览器缓存/换Chrome
   └─ 带宽本身慢 → 联系运营商升级
```

## 5. 网络故障诊断树

```
无法上网
│
├─ 检查物理连接
│  ├─ 网线是否插好？（网卡指示灯是否亮）
│  └─ WiFi是否连接？（是否输错密码）
│
├─ 检查IP配置
│  ├─ Win+R → cmd → ipconfig
│  ├─ 有IP地址（192.168.x.x）？ → 转DNS检查
│  └─ 没有IP/169.254.x.x → 重启路由器或手动设置IP
│
├─ 检查DNS
│  ├─ ping 8.8.8.8 （通 → DNS问题）
│  └─ ping www.baidu.com （不通 → 改DNS为114.114.114.114）
│
└─ 重置网络
   ├─ Win+R → cmd（管理员） → netsh winsock reset
   └─ 重启电脑
```

## 6. 硬盘故障诊断树

```
怀疑硬盘有问题
│
├─ 运行CrystalDiskInfo
│  ├─ 健康状态"良好" → 硬盘没问题，查其他原因
│  ├─ 健康状态"警告" → 立即备份数据，准备换硬盘
│  └─ 健康状态"不良" → 硬盘即将报废，立即更换
│
├─ 听到"咔咔"异响 → 机械故障，立即断电
│  └─ 数据重要 → 找专业数据恢复（>500元）
│
└─ 无法读取数据
   ├─ 尝试DiskGenius修复分区表
   └─ 尝试Recuva恢复删除文件
```

## 7. 过热故障诊断树

```
电脑发热严重/风扇狂转/频繁重启
│
├─ 检查散热口是否被堵住
│  └─ 清理散热口灰尘（用吹风机冷风吹）
│
├─ 检查CPU温度（HWiNFO64）
│  ├─ 待机>60°C / 满载>85°C → 散热不良
│  │  ├─ 台式机：重新涂硅脂，检查风扇是否转动
│  │  └─ 笔记本：拆机清灰，更换硅脂（建议找专业人士）
│  │
│  └─ 温度正常但风扇响 → 风扇轴承磨损，更换风扇
│
└─ 环境温度过高？
   └─ 空调房使用，或降低环境温度
```

## 快速诊断命令

以下命令可在CMD（命令提示符）中运行，快速诊断系统问题：

```bash
# 查看系统信息
systeminfo

# 查看内存使用情况
systeminfo | find "Total Physical Memory"

# 查看硬盘健康（需要管理员权限）
wmic diskdrive get status

# 查看蓝屏记录
eventvwr.msc → Windows日志 → 系统 → 查找"BugCheck"事件

# 内存诊断（需要重启）
mdsched.exe

# 系统文件检查
sfc /scannow

# 磁盘检查（需要重启）
chkdsk C: /f

# 查看开机时间（判断是否有异常重启）
systeminfo | find "System Boot Time"

# 查看CPU信息
wmic cpu get name,numberofcores,numberoflogicalprocessors

# 查看内存信息
wmic memorychip get capacity,speed,manufacturer

# 查看硬盘信息
wmic diskdrive get model,size,status
```

## 故障代码速查表

| 故障现象 | 可能原因 | 快速解决方案 |
|----------|----------|--------------|
| 开机1长1短报警 | 内存错误 | 拔插内存，用橡皮擦金手指 |
| 开机1长2短报警 | 显卡错误 | 拔插显卡，检查供电 |
| 开机1长3短报警 | 键盘错误 | 重新插拔键盘 |
| 开机不断重启 | 电源/主板/CPU过热 | 检查散热，清除CMOS |
| 蓝屏0x7B | 硬盘模式错误 | 进BIOS改SATA模式为AHCI |
| 蓝屏0x50 | 内存错误 | 内存加压或降低频率 |
| 蓝屏0x116 | 显卡驱动错误 | 安全模式卸载显卡驱动 |
| 黑屏但主机运行 | 内存/显卡/显示器 | 拔插内存，换显示器测试 |
| 卡LOGO界面 | 硬盘/外接设备 | 拔掉所有USB设备，检查硬盘 |
| 无法识别硬盘 | 硬盘/数据线/供电 | 更换SATA线，检查供电 |
