## Description: <br>
Miro CLI helps agents guide command-line workflows for querying Miro boards, exporting board data, viewing teams and organizations, and automating Miro tasks from the terminal. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and operations teams use this skill to work with Miro board, team, organization, export, content log, and audit log workflows through the external mirocli command-line tool. It is suited for inventory, reporting, export, and automation tasks that need shell commands or helper scripts rather than manual Miro UI work. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The external mirocli npm package handles Miro OAuth client secrets and tokens. <br>
Mitigation: Install only if you trust the package, verify the npm package before use, and test with a non-sensitive Miro account before using production credentials. <br>
Risk: Miro board exports, inventory CSVs, audit logs, and content logs may contain sensitive workspace data. <br>
Mitigation: Store exported files and reports only in approved, access-controlled locations and avoid sharing them in chats or logs. <br>
Risk: Authentication token output can expose active Miro session credentials. <br>
Mitigation: Avoid printing session tokens unless necessary, and do not paste token output into shared terminals, tickets, chats, or logs. <br>
