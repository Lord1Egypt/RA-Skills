# 秒技·场景工作室Pro版

## 快速开始

### 基本使用

1. 打开技能
2. 输入产品和品类
3. 选择目标平台
4. 获取完整视觉策略

### 输入示例

```
产品品类：空气炸锅
产品名称：XXX空气炸锅
目标平台：全平台
当前季节：秋季
同类产品ASIN：B09YYYYY, B07ZZZZZ
```

## 功能概览

### A/B测试方案
获取：
- 3套不同风格构图方案
- 测试执行计划
- 效果评估标准

### 平台适配规格
获取：
- Amazon/TikTok/Shopify尺寸表
- 各平台构图差异说明
- 文件规格要求

### 季节性场景库
获取：
- 春夏秋冬场景建议
- 节日场景模板
- 关键节点提醒

### 同类视觉分析
获取：
- TOP10同类产品评分
- 机会点识别
- 差异化建议
