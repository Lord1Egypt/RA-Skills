# 病毒载体选择用于基因治疗

## 触发条件
当需为基因治疗选择病毒载体时，根据以下三个关键因素匹配最适病毒类型：
- 目的基因大小（插入容量）
- 靶细胞是否处于分裂状态
- 是否需要长期稳定表达

## 执行步骤

1. **评估前提条件**：
   - 明确目的基因大小（kb）
   - 确定靶细胞类型（分裂 / 非分裂）
   - 评估是否需要长期表达

2. **根据决策逻辑选择载体**：
   - **若需长期整合表达且靶细胞为分裂细胞** → 选择 γ 反转录病毒
   - **若靶细胞为非分裂细胞（如神经元）** → 选择慢病毒或腺相关病毒（AAV）
   - **若需短期高表达且避免基因组整合风险** → 选择腺病毒
   - **若优先考虑安全性并用于体内治疗** → 选择 AAV

3. **验证约束条件**：
   - γ 反转录病毒不能感染非分裂细胞
   - 腺病毒不整合至宿主基因组，表达短暂
   - 所有载体均有插入容量限制，需确保目的基因适配

4. **输出结果**：
   推荐最适病毒载体类型，并说明其适用依据，平衡递送效率、安全性和表达持久性。
