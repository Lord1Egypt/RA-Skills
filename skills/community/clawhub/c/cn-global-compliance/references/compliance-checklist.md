# 出海合规快速自检清单

## 产品上线前必查项（通用）

### 数据隐私
- [ ] 隐私政策已翻译成目标市场语言
- [ ] 隐私政策URL可公开访问
- [ ] 用户注册时有明确的同意机制（不是默认勾选）
- [ ] 敏感数据（位置、健康、生物识别）有额外同意
- [ ] 用户可删除账号和数据
- [ ] 用户可导出个人数据
- [ ] 数据泄露72小时内可通知监管机构（EU要求）

### 支付
- [ ] 目标市场支付牌照要求已确认
- [ ] 不在App内引导用户到外部支付（Apple要求）
- [ ] 退款政策符合当地法律
- [ ] 价格显示含税（EU/日本要求）

### 内容
- [ ] 用户生成内容有举报/屏蔽机制
- [ ] 违法内容24小时内可删除（德国NetzDG）
- [ ] 无仇恨言论/歧视性内容
- [ ] 儿童安全措施到位（如适用）

### AI/算法
- [ ] AI决策有人工复核选项
- [ ] 算法推荐有退出机制（EU DSA）
- [ ] AI生成内容有标注（EU AI Act）
- [ ] 无基于种族/性别的歧视性算法

## 各市场特殊要求

### 🇪🇺 EU额外
- [ ] 指定DPO（数据保护官）
- [ ] 完成DPIA（数据保护影响评估）
- [ ] Cookie横幅+同意机制
- [ ] 与欧盟外数据接收方签SCC

### 🇺🇸 US额外
- [ ] CCPA"Do Not Sell"链接
- [ ] COPPA年龄门控（如适用）
- [ ] 州级AI透明度要求（CO/IL/TX）
- [ ] CFIUS审查（如涉及投资/收购）

### 🇯🇵 日本额外
- [ ] 日语隐私政策
- [ ] 跨境数据传输评估
- [ ] 支付服务法登记（如涉及支付）
- [ ] 特定商取引法披露

### 🇻🇳 越南额外
- [ ] 特定数据本地存储
- [ ] 违法内容24小时删除能力
- [ ] 在越南设立代表处或本地代理

### 🇸🇦 沙特额外
- [ ] 特定行业数据本地化
- [ ] 阿拉伯语界面
- [ ] PDPL合规评估

## 中国数据出境

### 必须通过网信办安全评估的情况
- CIIO（关键信息基础设施运营者）
- 处理100万人以上个人信息
- 累计向境外提供10万人以上个人信息
- 累计向境外提供1万人以上敏感个人信息
- 属于重要数据

### 可选标准合同的情况
- 不满足上述任何条件
- 需签署《个人信息出境标准合同》
- 需完成个人信息保护影响评估

## 常见踩坑

1. **App Store支付**：中国App常见的"VIP会员"外部支付链接，在iOS上会被拒
2. **实名认证**：海外无统一实名体系，手机号验证≠实名
3. **社交评分**：任何"信用分"/"社交分"功能在EU/US面临严格审查
4. **推送通知**：EU要求明确同意才能发送营销推送
5. **Cookie**：中国网站常见的"继续浏览即同意"在EU不合规
6. **用户协议**：中文用户协议直接翻译在海外可能无效（管辖权/适用法律问题）
7. **截图/录屏**：未经同意的屏幕录制功能违反多国隐私法
8. **儿童数据**：即使不是儿童App，如果"可能吸引儿童"也受COPPA约束
