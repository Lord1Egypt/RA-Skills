# 传出神经系统药物作用机制分类

## 何时使用
- 对未知传出神经药物进行机制归类
- 解释某药物为何产生拟交感或抗胆碱效应
- 指导联合用药或预测药物相互作用

## 决策流程

### 第一步：判断作用靶点
- **若药物直接结合受体** → 归为“直接作用于受体”
- **若药物不直接结合受体** → 归为“影响递质”

### 第二步：细化机制

#### A. 直接作用于受体
- **激动药（agonist）**：激活受体，模拟内源性递质效应（如毛果芸香碱、肾上腺素）
- **阻断药（antagonist）**：占据受体但不激活，阻断递质结合（如阿托品、普萘洛尔）

#### B. 影响递质（四种子机制）
1. **影响生物合成**：如α-甲基酪氨酸抑制NA合成
2. **影响释放**：
   - 促进：麻黄碱（促进NA释放）
   - 抑制：可乐定（抑制外周NA释放）
3. **影响转运与贮存**：
   - 利血平：耗竭囊泡内NA
   - 可卡因：抑制摄取-1，增加突触间隙NA
4. **影响生物转化**：
   - 胆碱酯酶抑制剂（如新斯的明）：抑制ACh水解，导致ACh堆积

> 所有传出神经系统药物均可归入上述两类之一或组合。
