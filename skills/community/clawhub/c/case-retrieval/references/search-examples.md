# 案例检索场景示例库

## 场景1：关键词检索

**用户输入**：`查一下借款合同纠纷的案例`

**处理流程**：
1. 意图识别：关键词检索
2. 要素抽取：关键词=借款合同纠纷
3. 执行脚本：`python3 scripts/search_cases.py "借款合同纠纷"`

**输出示例**：

```plaintext
- **文书类型：** 判决书
- **判决日期：** 2023-06-15
- **法官：** 张三、李四

### 2. (2023)京01民初678号 - 借款合同纠纷
- **案由：** 借款合同纠纷
- **审理法院：** 北京市第一中级人民法院
- **审理程序：** 一审
- **文书类型：** 判决书
- **判决日期：** 2023-08-20
```

## 场景5：长文本语义匹配

**用户输入**：用户上传了一份借款合同纠纷的案件材料，要求"帮我找一下类似的案例"

**处理流程**：
1. 意图识别：类案匹配
2. 要素抽取：从案件材料中提取关键事实（当事人关系、合同类型、争议焦点等）
3. 执行脚本：`python3 scripts/search_cases.py --long-text "原告与被告于2023年3月签订借款合同，借款金额120万元，约定年利率10%，借款期限12个月。被告到期未还款，原告起诉要求偿还本金及利息..."`

**输出示例**：
```plaintext
基于案件材料语义匹配，共找到 **15** 条相似案例：

### 1. (2023)粤03民终8901号 - 民间借贷纠纷
- **案由：** 民间借贷纠纷
- **审理法院：** 广东省深圳市中级人民法院
- **相似度较高**的案件
- **内容摘要：** 本案同样涉及个人之间的借款合同纠纷，借款金额100万元，约定年利率10%...
- **裁判要点：** 法院支持了原告要求偿还本金及约定利息的诉讼请求，认为年利率10%未超过法律保护上限...

💡 **提示**：以上案例按与您案件材料的相似度排序，建议重点关注裁判要点与您案件事实相近的案例。
```

## 场景6：翻页

**用户输入**：先检索了"借款纠纷"，看完第一页后说"再看一些"

**处理流程**：
1. 保持之前的检索条件不变
2. 将 `--page` 参数从 1 递增为 2
3. 执行脚本：`python3 scripts/search_cases.py "借款纠纷" --page 2`

## 场景7：API Key 未配置

**用户输入**：`查一下劳动争议的案例`

**处理流程**：
1. 脚本执行时报错：config.json 中的 apikey 为占位值
2. 直接告知用户，不执行后续检索

**输出示例**：
```plaintext
⚠️ config.json 中的 apikey 尚未配置。请按以下步骤操作：

1. 前往 https://platform.ailawyers.cn 注册/登录
2. 创建应用并获取 API Key
3. 将 API Key 填入技能目录下的 config.json 文件中的 "apikey" 字段
```
