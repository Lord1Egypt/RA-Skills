# 皮肤弹性检查与临床评估

## 适用对象
- 患者
- 体检对象

## 前提条件
- 可进行触诊
- 检查部位无局部皮肤破损或水肿

## 禁忌与限制
- 不适用于局部水肿、瘢痕或皮肤病区域
- 老年人生理性弹性减退需与病理性弹性减弱相鉴别

## 执行步骤
1. **选择检查部位**：首选手背或上臂内侧皮肤，因其受年龄和局部病变影响相对较小。
2. **操作手法**：用拇指和示指将皮肤轻轻提起，形成皱褶，然后迅速松开。
3. **观察结果**：
   - **弹性正常**：皮肤皱褶在松手后立即（迅速）平复。
   - **弹性减弱**：皮肤皱褶平复缓慢，提示可能存在严重脱水或慢性消耗性疾病。
4. **记录结论**：输出为“弹性正常”或“弹性减弱”，用于辅助临床判断。

## 触发场景
- 怀疑患者存在脱水（如呕吐、腹泻、高热后）
- 评估慢性消耗性疾病（如肿瘤、结核、重度营养不良）的体征
- 常规体格检查中筛查皮肤及营养状态

> 注意：发热时因血管充盈可能暂时增强皮肤弹性，需结合其他体征综合判断。
