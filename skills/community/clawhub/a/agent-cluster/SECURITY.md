# 外贸硅基军团 - 安全声明

## 安全架构

### 三层安全网

**第1层：LLM层检查**
- 输入验证（防Prompt注入）
- 输出过滤（防幻觉）
- 意图识别异常检测

**第2层：规则层验证**
- 金额阈值审批（>1000需人工确认）
- 高风险操作拦截（批量删除/修改）
- 黑名单关键词过滤

**第3层：人工审批**
- 采购订单>5000需经理审批
- 支付操作强制人工确认
- 审计日志完整记录

## 敏感数据保护

- **API密钥**：存储于环境变量，不硬编码
- **用户凭证**：不持久化明文密码
- **日志脱敏**：日志中自动过滤敏感字段（手机/邮箱/银行卡）

## 安全最佳实践

1. **输入验证**：所有用户输入经过严格类型检查
2. **输出验证**：AI响应经过规则引擎二次确认
3. **错误处理**：敏感错误信息不上报到日志
4. **权限最小化**：Agent只能访问授权范围内的CMS资源

## 依赖安全

- 所有Python依赖使用精确版本号锁定（`==`）
- 定期执行 `pip-audit` 检查已知漏洞
- 禁止使用 `requests`、`pickle` 等高风险依赖
