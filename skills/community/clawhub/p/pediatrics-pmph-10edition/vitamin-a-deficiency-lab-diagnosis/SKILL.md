# 维生素A缺乏实验室诊断

## 适用场景
- 疑似维生素A缺乏患儿的确诊
- 营养缺乏流行病学调查
- 指导治疗强度的实验室依据

## 执行步骤
1. **确认前提条件**：
   - 空腹采血
   - 排除急性感染或炎症（避免RBP假性降低）
   - 结合临床表现（如夜盲、干眼症）
2. **选择诊断方法**：
   - **首选**：血浆视黄醇测定
   - **评估肝储备**：相对剂量反应试验（RDR）
   - **辅助方法**：RBP测定、尿脱落细胞检查、暗适应测试
3. **解读结果并分型**：
   - 血清维生素A < 0.35 μmol/L → 临床型缺乏
   - 0.35–0.70 μmol/L → 亚临床型缺乏
   - RDR > 20% → 肝储备不足（亚临床型）
4. **排除干扰因素**：
   - 若检测RBP，同步测CRP以排除感染影响
   - 尿检需排除泌尿系感染

> **注意**：单一指标不足以确诊，需结合临床与多方法交叉验证。
