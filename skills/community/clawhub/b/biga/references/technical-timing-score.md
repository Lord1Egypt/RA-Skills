# BigA 技术面择时分框架
## 4维度评分(-2~+2/维度) + 极端修正,总分-10~+10

## ⚠️ 硬性前置条件：必须拥有日K线数据
**所有4个维度均依赖日K线数据（近5日收盘价、60日高低点、20日均量等）。仅靠实时行情快照无法计算择时分。**

**规则：**
1. 必须先从Sina或腾讯K线API获取至少20个交易日的日K线数据
2. **若K线获取失败 → 不计算择时分，输出中不包含择时字段**
3. 严禁在无K线数据的情况下用其他数据伪造/估算
4. K线数据优先存缓存文件`memory/biga-technical-data.md`，避免重复拉取

### 数据源
- K线(主): Sina日K线 http://money.finance.sina.com.cn/...?symbol=<代码>&scale=240&ma=5&datalen=60
- K线(备): 腾讯日K线 http://web.svip.ifzq.gtimg.cn/appstock/app/fqkline/get?param=<代码>,day,,,1
- 实时行情: 腾讯API(已有)
- **主备均失败 → 标记kline_unavailable，跳过择时分**

### 维度1:短期涨跌动量(近5日涨幅)
| 涨跌幅 | 分 | 信号 |
|--------|----|------|
| <-15% | +2 | 严重超跌 |
| -8~-15% | +1 | 显著回调 |
| -3~+5% | 0 | 横盘 |
| +5~+15% | -1 | 已有涨幅 |
| +15~+25% | -2 | 涨幅过大 |
| >+25% | -2 | 严重超买 |

### 维度2:日内量价特征(当日K线+换手率)
| 特征 | 分 |
|------|----|
| 低开高走+振幅>5% | +2 |
| 高开低走+振幅>5%+放量 | -2 |
| 缩量回调(跌幅<3%+换手<3%) | +1 |
| 放量滞涨(高换手+收盘≈开盘) | -2 |
| 窄幅震荡(振幅<2%) | 0 |

### 维度3:成交量异常(换手率 vs 近20日均值)
| 特征 | 分 |
|------|----|
| 地量(<均值×0.5) | +1 |
| 放量下跌(>均值×2) | +2 |
| 放量上涨(>均值×2) | -1 |
| 天量(>均值×3+滞涨) | -2 |

### 维度4:相对强度(个股vs大盘/板块)
| 特征 | 分 |
|------|----|
| 个股大跌+板块小跌(基本面未变) | +2 |
| 跑输板块>3% | +1 |
| 同步(幅度差<3%) | 0 |
| 跑赢板块>3% | -1 |
| 跟风(板块大涨+个股涨幅<一半) | -2 |

### 极端修正(-2~+2)
- 连续3涨停开板:-2 | 连续3跌停开板:+2
- 创60日新高:-1 | 创60日新低:+1

### 总分判定
| 分数 | 操作 |
|------|------|
| ≥+6 | 🔥 强烈买入 |
| +3~+5 | ✅ 买入 |
| -2~+2 | 👀 持有观望 |
| -5~-3 | ⚠️ 仅持有不新增 |
| ≤-6 | 🛑 考虑止盈 |

### 组合决策(BigA × 择时分)
择时≥+5 + BigA≥70→重仓买入, 择时≤-4 + BigA≥70→持有不追
择时≥+3 + BigA<50→短线反弹, 择时≤-3 + BigA<50→跳过
