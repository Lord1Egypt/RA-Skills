# 反保险欺诈工作办法解读与各险种欺诈特征

> 本文件为 `insurance-anti-fraud` Skill 配套参考

## 一、《反保险欺诈工作办法》核心内容（2024年）

**发布机构**：国家金融监督管理总局
**实施时间**：2024年7月22日

### 1.1 四位一体工作体系

```
┌─────────────────────────────────────────────────────┐
│                   四位一体体系                        │
├─────────────────────────────────────────────────────┤
│ ① 行业协作：银保信平台 + 精算师协会 + 保险行业协会   │
│ ② 公司主体：各保险公司建立反欺诈组织和内控制度       │
│ ③ 司法联动：涉嫌犯罪的案件移送公安/检察机关          │
│ ④ 监管约束：日常监管 + 专项检查 + 行政处罚           │
└─────────────────────────────────────────────────────┘
```

### 1.2 保险公司主体责任清单

| 责任 | 具体要求 |
|------|---------|
| 组织架构 | 建立反欺诈部门或岗位，指定高管负责 |
| 制度建设 | 制定反欺诈管理制度和操作规程 |
| 系统建设 | 建立欺诈风险识别和预警系统 |
| 人员培训 | 定期开展反欺诈培训和考核 |
| 信息报送 | 定期向监管部门报送欺诈数据 |
| 案件处置 | 配合司法调查，主动报案 |

### 1.3 欺诈渗漏率基准

- 行业平均欺诈渗漏率：**15%-20%**
- 先进水平（优秀公司）：**<5%**
- 差距来源：技术投入不足、数据孤岛、调查能力弱

## 二、各险种欺诈特征详解

### 2.1 重疾险欺诈十大特征

| 特征 | 描述 | 核查方法 |
|------|------|---------|
| 带病投保 | 健康告知全为"否"，理赔时发现既往症 | 调取体检记录、医保数据 |
| 虚假诊断 | 伪造重大疾病诊断证明 | 医院核查、专家会诊 |
| 夸大病情 | 轻症夸大成重疾 | 病理报告复查 |
| 等待期出险 | 等待期刚过即集中报案 | 核查投保时间与发病时间 |
| 团伙作案 | 多人同时投保同家公司同种重疾 | 关联案件分析 |
| 重复索赔 | 多家保险公司重复申请同种重疾 | 银保信数据查询 |
| 高额保额 | 低收入人群购买超高保额 | 财务证明核查 |
| 指定受益人异常 | 受益人为陌生人或存在债务关系 | 受益人关系调查 |
| 病历造假 | 病历签名笔迹不一致 | 笔迹鉴定 |
| 恶意投保 | 明知已患重疾后投保 | 全行业投保记录协查 |

### 2.2 医疗险欺诈特征

| 欺诈类型 | 具体手法 | 识别信号 |
|---------|---------|---------|
| 虚增费用 | 医疗费用虚高，超出正常水平 | 费用合理性分析，对标同类医院 |
| 挂床住院 | 办理住院但实际未住院 | 住院日志核查、病房查房记录 |
| 过度医疗 | 不必要的检查和治疗 | 临床路径核查 |
| 冒名顶替 | 非被保险人就医 | 核验身份信息 |
| 虚假病历 | 伪造住院记录、诊断证明 | 医院直接核查 |
| 机构合谋 | 医院与患者联手造假 | 异常医院黑名单库 |
| 倒卖保单 | 收购他人保单申请理赔 | 频繁更换受益人 |

### 2.3 意外险欺诈特征

| 欺诈类型 | 具体手法 | 识别信号 |
|---------|---------|---------|
| 伪造事故 | 虚构意外事故经过 | 事故现场勘查、监控调取 |
| 制造意外 | 故意制造意外事故 | 事故合理性分析 |
| 酒后顶包 | 酒驾后找人顶替报案 | 笔录矛盾点 |
| 高危职业虚假 | 投保时隐瞒高危职业 | 职业核实 |
| 死亡骗保 | 故意杀害被保险人 | 刑侦介入调查 |
| 自残骗保 | 故意自残获取意外赔偿 | 伤情与事故不符 |

### 2.4 车险欺诈特征（补充参考）

| 欺诈类型 | 描述 |
|---------|------|
| 故意碰撞 | 利用修车厂实施有预谋的车辆碰撞 |
| 扩大损失 | 小刮蹭夸大维修范围 |
| 虚假三者 | 编造不存在的第三方受害者 |
| 重复索赔 | 已赔付案件再次索赔 |
| 酒驾顶包 | 酒驾事故找人顶替 |
| 套牌骗赔 | 使用假车牌或套牌车索赔 |

## 三、欺诈风险评分卡

### 3.1 健康险欺诈评分（0-100分）

| 评分项 | 分值范围 | 评分规则 |
|--------|---------|---------|
| 等待期 | 0-30分 | 等待期≤30天=30分，31-90天=15分，>90天=0分 |
| 保额收入比 | 0-25分 | >20倍=25分，10-20倍=15分，<10倍=0分 |
| 医院欺诈历史 | 0-20分 | 该医院有欺诈记录=20分，核查难度高=10分，正常=0分 |
| 既往索赔记录 | 0-15分 | 有拒赔/欺诈记录=15分，多次索赔=8分，首次=0分 |
| 同案关联 | 0-10分 | 与疑似欺诈案件关联=10分，正常=0分 |

**风险等级**：
- ≥60分：**HIGH_RISK** → 立即启动深度调查，必要时报案
- 30-59分：**MEDIUM_RISK** → 重点审核，补充调查材料
- <30分：**LOW_RISK** → 正常理赔流程

## 四、大数据反欺诈技术路线

| 技术 | 应用场景 | 预期效果 |
|------|---------|---------|
| 机器学习评分 | 理赔欺诈评分 | 识别准确率提升30%-50% |
| 关系图谱 | 团伙欺诈识别 | 发现隐藏关联关系 |
| 自然语言处理 | 病历真实性分析 | 识别虚假病历 |
| 声纹/OCR | 身份核实 | 防冒名顶替 |
| 知识图谱 | 反向骗保识别 | 发现异常投保行为 |
