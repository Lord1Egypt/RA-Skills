# 里程碑叙事模板

## 适用场景
签约、达产、上市、业绩公布、产能突破、重大合作

## 事件级别
L2 标准 或 L3 重要

## 结构骨架

```
成果锚点 → 历程钩沉 → 意义放大 → 未来指向
```

## 填写表

| 段落 | 写什么 | 字数 | 铁律 |
|------|--------|------|------|
| 成果锚点 | 一句话说清，**必须带1个数字** | 30-50字 | 不要铺垫，第一句亮结果 |
| 历程钩沉 | 2-3个关键转折点 | 80-120字 | 写"难"不写"苦"，有克制 |
| 意义放大 | 对客户/行业的价值，**从对方视角写** | 60-100字 | 不自我感动 |
| 未来指向 | 下一步具体行动 | 30-50字 | 可验证，不说"继续努力" |

## 变量槽

| 变量 | 说明 | 示例 |
|------|------|------|
| {{MILESTONE}} | 里程碑名称 | 签约额突破1.2亿 |
| {{KEY_NUMBER}} | 关键数据 | 1.2亿 |
| {{TIMELINE}} | 关键时间节点 | 14个月、7次技术交流 |
| {{BENEFICIARY}} | 受益方 | 客户、行业 |
| {{BENEFIT}} | 具体价值 | 交付周期从45天缩至30天 |
| {{NEXT_STEP}} | 下一步 | Q3完成项目组进驻 |

## 自检

- [ ] 成果锚点是否第一句就亮出了数字？
- [ ] 历程钩沉是写"过程"还是写"不容易"？（要前者）
- [ ] 意义放大是从对方视角写的吗？
- [ ] 未来指向可以验证吗？（三个月后回来看能不能检查）
- [ ] 禁用词检查通过？

## 常见问题

| 问题 | 修正 |
|------|------|
| "在各级领导关怀下..."开头 | 直接亮数字和结果 |
| "大幅提升""显著改善" | 给具体数字 |
| "我们将继续努力，再创辉煌" | "下一步，XX将在9月完成" |

## 示例

> **成果锚点**：签约额1.2亿。从第一次接洽到今天落笔，中间隔着14个月。
>
> **历程钩沉**：7次技术交流、3轮方案迭代。客户换了两次对接人，但每次交接，方案都在桌上。
>
> **意义放大**：对客户来说，这意味着华南区域的供应链响应时间从45天缩短到30天。每快一天，终端的缺货率就降一点。
>
> **未来指向**：下个月，联合项目组正式进驻客户华南仓。
