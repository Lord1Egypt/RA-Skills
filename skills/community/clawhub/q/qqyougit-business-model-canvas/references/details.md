# 商业模式画布 详细参考

## 九大模块详解

### 1. 客户细分（Customer Segments）
**核心问题：** 你在为谁创造价值？
- 主要客户：最核心的付费群体
- 次要客户：延伸用户群
- 付费决策者 vs 使用者（ToB尤其重要）

**客户细分策略：**
| 策略 | 说明 | 示例 |
|------|------|------|
| 大众市场 | 不区分细分 | 微信/抖音 |
| 利基市场 | 专注小众需求 | 专业工具/垂直SaaS |
| 多边平台 | 服务多类群体 | 淘宝(买家+卖家) |
| 细分化 | 差异化服务 | 航空(经济舱/商务舱) |

### 2. 价值主张（Value Propositions）
**核心问题：** 你为客户解决什么问题？
- 功能价值：解决了什么具体问题
- 情感价值：带来什么感受（安全感/优越感/归属感）
- 社会价值：社会认同/身份标签

**价值主张画布：**
```
客户任务 → 客户痛点 → 你的止痛方案
客户任务 → 客户收益 → 你的创益方案
```

**常见价值主张类型：**
- 新颖性（全新体验）
- 性能提升（更快/更强）
- 定制化（个性化）
- 设计（美学/体验）
- 品牌/身份认同
- 成本降低（省钱）
- 风险降低（保障）
- 便利性（省时间/省精力）
- 可达性（让原来用不了的人能用）

### 3. 渠道通路（Channels）
**核心问题：** 客户怎么找到你？

**渠道阶段：** 认知→评估→购买→交付→售后

**渠道类型及成本：**
| 渠道 | 获客成本 | 适用阶段 | 示例 |
|------|----------|----------|------|
| SEO/内容营销 | 低 | 中长期 | 博客/教程/社区 |
| 社交媒体 | 中低 | 全阶段 | 小红书/抖音/公众号 |
| SEM/信息流 | 中高 | 有预算后 | 百度/Google广告 |
| 口碑/推荐 | 极低 | 有用户后 | 邀请奖励/社区 |
| 线下活动 | 高 | ToB早期 | 展会/沙龙 |
| BD合作 | 中 | 有资源后 | 渠道商/集成商 |
| PLG产品驱动 | 低 | 产品成熟 | Freemium→付费 |

### 4. 客户关系（Customer Relationships）
**核心问题：** 如何获取、留存、增长客户？

| 关系类型 | 说明 | 适用场景 |
|----------|------|----------|
| 专属服务 | 1对1人工服务 | 大客户/高端 |
| 自助服务 | 用户自己搞定 | SaaS/工具 |
| 自动化服务 | 系统智能推荐 | 电商/内容 |
| 社区 | 用户互助 | 开源/开发者 |
| 共同创造 | 用户参与设计 | UGC平台 |

**留存策略框架：**
- 获取（Acquisition）：首次体验要惊艳
- 激活（Activation）：Aha moment要快
- 留存（Retention）：习惯养成循环
- 收入（Revenue）：付费转化时机
- 推荐（Referral）：让用户帮你传播

### 5. 收入来源（Revenue Streams）
**核心问题：** 怎么赚钱？

**定价策略对比：**
| 策略 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 固定月费 | 收入可预测 | 门槛高 | 成熟产品 |
| 按量计费 | 低门槛 | 收入波动 | API/云服务 |
| Freemium | 快速增长 | 转化率低 | 工具/协作 |
| 交易抽佣 | 平台效应 | 需规模 | 市场/平台 |
| 广告 | 用户免费 | 需流量 | 内容/社交 |
| 授权/许可 | 一次卖断 | 无持续收入 | 软件/IP |
| 订阅制 | 持续收入 | 需持续价值 | 内容/会员 |

### 6-9. 基础设施层

**核心资源类型：**
- 实体资源：设备/办公/库存
- 知识资源：专利/数据/品牌
- 人力资源：技术/运营/销售团队
- 财务资源：现金/信贷/股权

**关键业务类型：**
- 生产型：设计/制造/交付
- 平台型：维护/优化/扩展
- 问题解决型：咨询/服务/定制

**重要伙伴类型：**
- 供应商：上游资源
- 渠道商：分销网络
- 战略联盟：互补合作
- 投资方：资金支持

**成本结构类型：**
- 成本驱动：极致压缩成本（廉价航空）
- 价值驱动：关注品质而非成本（奢侈品）
- 固定成本：租金/薪资/设备
- 可变成本：获客/云服务/支付手续费

## 行业商业模式模板

### SaaS产品画布模板
- 客户：中小企业/个人用户
- 价值：提升效率/降低成本
- 渠道：PLG+内容营销+口碑
- 收入：Freemium + 订阅（月/年）
- 资源：产品团队+云服务
- 关键指标：MRR/Churn Rate/NDR/LTV/CAC

### 平台型画布模板
- 客户：双边市场（供需两端）
- 价值：匹配效率+信任保障
- 渠道：补贴获客→自然增长
- 收入：交易抽佣+增值服务
- 资源：技术平台+运营团队
- 关键指标：GMV/Take Rate/MAU

### 内容付费画布模板
- 客户：知识焦虑人群/专业学习者
- 价值：系统化知识+圈层社交
- 渠道：KOL+社群+公开课引流
- 收入：课程+会员+咨询
- 资源：内容生产能力+IP
- 关键指标：完课率/续费率/NPS

## 商业模式验证方法

**精益验证四步法：**
1. 假设识别：列出商业模式中的所有关键假设
2. 风险排序：按"致命程度×不确定程度"排序
3. 实验设计：为Top 3假设设计最小验证实验
4. 决策标准：定义"通过/不通过"的量化标准

**验证实验类型：**
| 假设类型 | 验证方法 | 成本 | 时间 |
|----------|----------|------|------|
| 需求存在 | 用户访谈/问卷 | 低 | 1-2周 |
| 付费意愿 | 预售/Landing Page | 低 | 1-2周 |
| 获客渠道 | 小规模投放测试 | 中 | 2-4周 |
| 产品方案 | 原型测试/MVP | 中 | 4-8周 |
| 规模化 | 增长实验 | 高 | 持续 |

## 常见商业模式陷阱

1. **伪需求**：用户说"好棒"但不付费 → 用预售验证
2. **获客成本过高**：CAC > LTV → 算清单位经济模型
3. **留存黑洞**：用户来了就走 → 关注Aha moment和习惯循环
4. **规模不经济**：越大越亏 → 验证边际成本递减
5. **护城河幻觉**：以为技术壁垒实际可替代 → 验证切换成本
6. **现金流断裂**：收入慢成本高 → 先跑通现金流再扩张
