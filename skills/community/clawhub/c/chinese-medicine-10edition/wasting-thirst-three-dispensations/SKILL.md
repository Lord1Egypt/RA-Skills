# Wasting-Thirst Three Dispensations

## When to Use
- 糖尿病患者
- 尿崩症患者
- 具有“三多一少”（多饮、多食、多尿、消瘦）或尿甜症状者

## When to Use
当患者出现以下任一组合症状时启动本流程：
- 多饮、多食、多尿、形体消瘦
- 尿液有甜味

> **注意**：不适用于仅有口渴或多尿但无其他消渴特征者。

## Execution Steps

### 1. 确认消渴基本定义
核实是否具备以下核心表现：
- 多饮、多食、多尿
- 形体消瘦、乏力
- 或尿有甜味

### 2. 三消分型（按症状侧重与舌脉）
- **上消（肺热津伤）**：以口渴多饮为主，伴口干舌燥、尿频量多；舌边尖红、苔薄黄、脉洪数。
- **中消**：
  - *胃热炽盛*：多食易饥，大便干燥，舌红苔黄燥。
  - *气阴两虚*：多食但便溏、腹胀、神疲乏力，舌淡红苔白干。
- **下消**：
  - *肾阴亏虚*：尿频量多、尿浊如脂膏，腰膝酸软、五心烦热，舌红少苔。
  - *阴阳两虚*：畏寒肢冷、面色黧黑、饮一溲一，脉沉细无力。

### 3. 把握核心病机
始终以“**阴虚为本，燥热为标**”为辨证总纲，贯穿全程。

### 4. 识别特殊病理因素
若出现以下瘀血指征，提示血脉瘀滞，需加用活血化瘀药（如丹参、川芎、红花）：
- 面色晦暗
- 胸痛夜甚
- 舌紫暗或有瘀斑
- 脉涩

### 5. 追溯病因
结合四大病因综合判断：
- 禀赋不足
- 情志失调
- 饮食不节
- 劳欲过度

### 6. 预警变证
久病者需警惕以下变证风险：
- 白内障
- 痈疮
- 中风
- 水肿
- 胸痹

## Output
明确输出具体证型（如“上消-肺热津伤”）及对应治法方药建议。
