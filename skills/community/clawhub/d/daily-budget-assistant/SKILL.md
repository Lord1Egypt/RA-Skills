# 💰 AI智能记账助手

> 每天只花5秒，告别月光！AI帮你记账、分类、分析、预警。

## 你是不是这样？

- 💸 每月工资到账，不知道钱花哪了
- 📱 各种记账APP太复杂，记了3天就放弃
- 🧾 小票攒了一堆，月底对不上账
- 😰 想存钱但不知道从哪省

**5秒记账 = 说一句话就行！**

## 怎么用？

### 方式1：一句话记账（最快）
直接告诉助手你花了什么：

```
"午饭32" → 自动分类：餐饮 | 金额：32元
"打车去公司18.5" → 自动分类：交通 | 金额：18.5元
"超市买了水果和牛奶一共89" → 自动分类：日用品 | 金额：89元
"给孩子报了英语班3000" → 自动分类：教育 | 金额：3000元
```

### 方式2：语音记账
```
"今天花了120买衣服"
"加油花了400块"
```

### 方式3：批量补记
```
"帮我记一下昨天的消费：早餐8元，地铁4元，午餐25元，咖啡18元，晚餐45元"
```

## 核心功能

### 🏷️ 自动分类
AI自动识别消费类型，12大分类全覆盖：

| 分类 | 示例 |
|------|------|
| 🍜 餐饮 | 午饭、外卖、聚餐、奶茶 |
| 🚗 交通 | 打车、地铁、加油、停车 |
| 🏠 住房 | 房租、物业、水电、维修 |
| 🛒 日用 | 超市、日用品、清洁用品 |
| 👗 服饰 | 衣服、鞋子、包包 |
| 💄 美妆 | 护肤品、化妆品、美发 |
| 🎮 娱乐 | 电影、游戏、KTV |
| 📚 教育 | 培训班、书、网课 |
| 🏥 医疗 | 看病、药品、体检 |
| 👶 母婴 | 奶粉、尿布、早教 |
| 💌 社交 | 红包、礼物、请客 |
| 💡 其他 | 以上未涵盖的支出 |

### 📊 月度分析报表
每月自动生成消费报告：
- 总支出 / 各分类占比
- 与上月对比（增减趋势）
- 最大单项支出提醒
- 哪个分类超支了

### ⚠️ 超支预警
设置月度预算后：
- 当月支出达到80%时 → 🟡 温馨提醒
- 当月支出达到100%时 → 🔴 紧急预警
- 单笔大额支出 → 💰 异常提醒

### 📅 周度/月度回顾
自动生成回顾提醒：
- "本周你花了XX元，比上周多/少XX元"
- "本月餐饮占比最高，建议减少外卖次数"

## 记账数据存储

所有记账数据保存在本地文件中，隐私安全：
- 数据路径：`~/Documents/daily-budget/`
- 月度文件：`YYYY-MM.md`（Markdown格式，方便查看）
- 导出功能：支持导出CSV

## 工作流

当用户进行记账相关操作时，按以下流程处理：

1. **识别意图**：判断用户是想记账、查账、设预算还是导出
2. **提取信息**：从用户输入中提取金额、消费项目、日期
3. **自动分类**：AI将消费项目匹配到12大分类
4. **记录数据**：写入当日Markdown文件
5. **反馈确认**：简洁展示记录结果 + 今日累计
6. **智能提醒**：必要时给出超支预警或省钱建议

### 记账文件格式

文件：`~/Documents/daily-budget/YYYY-MM.md`

```markdown
# 💰 YYYY年M月记账本

## 月度预算：5,000元

## 消费记录

### M月D日（周X）
- 🍜 午餐 | 35元 | 沙县小吃
- ☕ 咖啡 | 18元 | 瑞幸
- 🚇 地铁 | 4元 | 通勤
- **当日合计：57元**

### M月D+1日（周X）
- 🍜 早餐 | 8元 | 包子铺
- 🚕 打车 | 28元 | 滴滴
- 🛒 超市 | 89元 | 水果牛奶
- **当日合计：125元**

## 月度统计
- 总支出：XXX元
- 日均：XX元
- 各分类占比：...
```

## 注意事项

- 金额识别支持多种格式：35、35元、35块、三十五
- 分类会根据描述自动判断，用户可手动修正
- 所有数据存在本地，不上传任何服务器
- 支持"撤销最后一笔"操作
