# 道家决策框架

## 一、静观阶段（致虚极，守静笃）

### 1.1 虚心准备
- [ ] 排除情绪干扰，保持内心空明
- [ ] 暂时放下预设和偏见
- [ ] 避免急于求成的心态
- [ ] 减少外界刺激和噪音

### 1.2 静观其变
- [ ] 全面观察，不做判断
- [ ] 关注事物发展的自然趋势
- [ ] 识别事物的"道"（内在规律）
- [ ] 等待时机成熟

---

## 二、审势阶段（人法地，地法天，天法道，道法自然）

### 2.1 五法评估

| 维度 | 检查问题 | 评估结果 |
|------|---------|---------|
| **道** | 是否符合事物发展的内在规律？ | 符合/部分符合/违背 |
| **天** | 是否顺应时代趋势和外部环境？ | 顺势/逆势/中性 |
| **地** | 是否具备地利和资源条件？ | 充足/不足/欠缺 |
| **人** | 是否得到人心的支持？ | 支持/中立/反对 |
| **己** | 自身能力是否匹配？ | 匹配/勉强/不匹配 |

### 2.2 辩证分析（祸福相依）
- **有利中的风险**：当前优势可能带来什么隐患？
- **不利中的机会**：当前困境中蕴含什么转机？
- **物极必反风险**：事物是否发展到极端？何时会转向？

---

## 三、决策阶段（无为而无不为）

### 3.1 行动原则检查

| 原则 | 说明 | 本次决策是否符合 |
|------|------|-----------------|
| **不妄为** | 不违背自然规律强行干预 | □ 是 □ 否 |
| **不强行** | 不在条件不成熟时强求 | □ 是 □ 否 |
| **不过度** | 不追求极端和完美 | □ 是 □ 否 |
| **不争功** | 成功后不居功自傲 | □ 是 □ 否 |

### 3.2 三宝自检（慈、俭、不敢为天下先）

**慈（仁爱）**：
- 决策是否考虑了他人利益？
- 是否以伤害他人为代价？
- 是否有更仁慈的替代方案？

**俭（节制）**：
- 资源消耗是否必要？
- 是否存在浪费？
- 是否知足适度？

**不敢为天下先（谦让）**：
- 是否急于出风头？
- 是否留给他人空间？
- 是否愿意后发制人？

## 四、执行阶段（千里之行，始于足下）

### 4.1 循序渐进
- [ ] 从最容易的入手
- [ ] 从最小处开始
- [ ] 分阶段推进，不贪大求全

### 4.2 见微知著
- [ ] 关注细微变化
- [ ] 在问题萌芽时处理
- [ ] 慎终如始，保持警惕

### 4.3 柔性执行
- [ ] 像水一样，遇阻则绕
- [ ] 保持弹性，随时调整
- [ ] 不硬碰硬，寻找替代路径

---

## 五、反思阶段（反者道之动）

### 5.1 功成身退
- [ ] 是否适时退出？
- [ ] 是否给他人留空间？
- [ ] 是否为下次机会做准备？

### 5.2 知足常乐
- [ ] 对结果是否满足？
- [ ] 是否贪求更多？
- [ ] 是否懂得适可而止？

### 5.3 复盘问道
- [ ] 哪些顺应了"道"？
- [ ] 哪些违背了"道"？
- [ ] 下次如何改进？

---

## 快速检查清单

### 决策前
- [ ] 是否冷静？（静为躁君）
- [ ] 是否了解全貌？（知者不博，博者不知）
- [ ] 是否顺应大势？（道法自然）

### 决策中
- [ ] 是否不强求？（无为而治）
- [ ] 是否从易从细？（天下大事必作于细）
- [ ] 是否留有余地？（持而盈之，不如其已）

### 决策后
- [ ] 是否功成不居？（功遂身退）
- [ ] 是否知足常乐？（知足者富）
- [ ] 是否反求诸己？（自知者明）

## 常见误区警示

| 误区 | 正解 |
|------|------|
| 无为=不作为 | 无为=不妄为、不强行 |
| 柔弱=软弱 | 柔弱=韧性、适应性 |
| 不争=放弃 | 不争=更高明的争 |
| 知足=安于现状 | 知足=不贪婪、适可而止 |
| 后发=落后 | 后发=观察、等待时机 |
