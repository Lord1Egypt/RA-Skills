# Yin Qiao San Formula Application

## When to Use
当满足以下全部条件时启用本技能：
- 证属风热犯卫或温病初起卫分证
- 主症包括发热、微恶风寒、无汗或有汗不畅、头痛口渴、咳嗽咽痛、舌尖红、苔薄白或薄黄、脉浮数
- 排除风寒感冒及脾胃虚寒体质（慎用大量寒凉药）

## Execution Steps
1. **确认方剂组成**：金银花、连翘、淡豆豉、牛蒡子、薄荷、荆芥穗、桔梗、甘草、竹叶、芦根。
2. **理解配伍结构**：
   - 君药：金银花、连翘（清热解毒，疏散风热）
   - 臣药：薄荷、牛蒡子（清利头目，利咽）；荆芥穗、淡豆豉（辛散透表）
   - 佐使：桔梗、甘草（宣肺利咽）；竹叶、芦根（清热生津）
3. **掌握煎服要点**：
   - 薄荷、荆芥穗后下
   - “香气大出，即取服，勿过煮”，以保辛凉透表之效
4. **指导临床加减**（简要）：
   - 咽痛甚可考虑加马勃、玄参
   - 项肿可考虑加板蓝根
   - 热重可考虑加黄芩、石膏

> 注意：本方为辛凉平剂，强调轻清宣透，久煎则失效；风寒感冒禁用。
