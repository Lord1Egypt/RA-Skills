# 劳动仲裁全流程辅助技能

> **覆盖全国31省市 · 从资料研判到一键生成仲裁申请书**

## 技能简介

本技能为普通劳动者提供劳动仲裁全流程辅助，帮助劳动者在面临拖欠工资、违法解除、未签合同二倍工资、加班费等劳动争议时，系统化地完成资料研判、证据整理、文书生成和线上提交。

## 核心功能

| 功能模块 | 说明 |
|---------|------|
| 资料研判 | 评估现有证据充分性，识别证据缺口，评估时效风险 |
| 资料整理 | 按7种仲裁请求分类生成个性化证据清单 |
| 文档生成 | 一键生成仲裁申请书、证据清单、金额计算表 |
| 提交指引 | 全国31省市线上提交平台和操作步骤 |
| 调解策略 | 底线金额计算、谈判技巧、调解协议避坑 |
| 地区规则 | 全国各地区时效、管辖、举证、裁判口径差异 |

## 适用场景

- 公司拖欠/克扣工资
- 违法解除劳动合同（要求2N赔偿金）
- 未签劳动合同（要求二倍工资差额）
- 单方降薪/调岗调地
- 未支付加班费
- 未休年休假工资
- 确认劳动关系

## 使用方式

### 方式一：对话触发

在 WorkBuddy 中直接描述情况，技能会自动触发：

> "公司群通知取消底薪只有提成，我有合同没签变更协议，北京地区，怎么仲裁？"

> "入职2年被违法辞退，没给经济补偿金，深圳，帮我整理材料"

### 方式二：一键生成文档

收集完信息后，调用脚本生成仲裁申请书：

```bash
python scripts/generate_arbitration_docs.py --config case_info.json --output-dir ./output
```

生成文件：
- 仲裁申请书.md
- 证据清单.md
- 金额计算表.md

## 金额计算示例

以北京某案件为例（月薪8000元，工作1.5年，违法解除）：

| 请求事项 | 计算方式 | 金额 |
|---------|---------|------|
| 拖欠工资 | 2个月 × 8000 | ¥16,000 |
| 违法解除赔偿金 | 8000 × 1.5月 × 2倍 | ¥24,000 |
| 未休年休假工资 | 367.82 × 200% × 5天 | ¥3,678 |
| **合计** | | **¥43,678** |
