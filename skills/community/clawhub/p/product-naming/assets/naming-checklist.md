# Product Naming Quality Checklist

## Search Fit (6 checks)
- [ ] Primary keyword identified and confirmed via search volume data
- [ ] Product name includes primary keyword or strong synonym
- [ ] 34-char TikTok visible title includes primary keyword
- [ ] Full 255-char TikTok search title includes 3+ keyword variations
- [ ] Amazon title includes primary keyword in first 3 words (after brand)
- [ ] Shopify SEO title is within 70-character Google SERP limit

## Hook Strength (5 checks)
- [ ] Opening words (first 3) create curiosity, desire, or identity — not just describe
- [ ] Name is distinct from top 5 competitor names in the same category
- [ ] A creator could say the name naturally in a 30-second video
- [ ] Buyers can recall the name after reading once (10-second memory test)
- [ ] Name evokes a positive emotion: aspiration, relief, joy, or confidence

## Compliance (8 checks)
- [ ] No superlatives: "best", "#1", "top", "world's", "only", "greatest"
- [ ] No unsubstantiated health claims: "cures", "treats", "eliminates [disease]"
- [ ] No before/after claims in name (reserved for images, not title text)
- [ ] No competitor brand names embedded
- [ ] Trademark search completed in all target markets (USPTO/EUIPO/WIPO)
- [ ] Name tested for cultural sensitivity in all target market languages
- [ ] No pricing or promotional language ("free", "discount", "sale")
- [ ] No HTML or special characters (Amazon compliance)

## Character & Format (4 checks)
- [ ] TikTok visible field: 28–34 characters (fill the space)
- [ ] Amazon title: within category character limit
- [ ] Shopify SEO title: under 70 characters
- [ ] Brand name placement correct per platform convention

## Brand Fit (3 checks)
- [ ] Name consistent with brand voice (playful / clinical / premium / etc.)
- [ ] Name aligns with naming conventions of other products in the range
- [ ] Name does not create confusion with other SKUs in the same catalog

## Pre-Launch Final Check (4 checks)
- [ ] Recommended name scored 4.0+ in the scoring matrix
- [ ] At least 2 alternative names documented and scored (for future reference)
- [ ] Name approved by all relevant stakeholders
- [ ] Platform titles (TikTok / Amazon / Shopify versions) finalized and entered into seller dashboards
