## Description: <br>
Guides agents in using MiniMax Hailuo AI for text-to-video, image-to-video, subject reference, multi-shot video prompts, and API examples. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Creators, marketers, and developers use this skill to plan Hailuo AI video prompts, compare video-generation capabilities, and adapt MiniMax API examples for generated video workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: API examples can lead users to place real MiniMax API keys in prompts or copied code. <br>
Mitigation: Store real API keys outside prompts and source snippets, and review code before execution. <br>
Risk: Prompts or reference images may contain private or sensitive content sent to MiniMax. <br>
Mitigation: Avoid submitting private prompts or images unless the user intends to share them with MiniMax. <br>
