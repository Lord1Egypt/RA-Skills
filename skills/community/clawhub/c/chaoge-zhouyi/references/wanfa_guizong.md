# 万法归宗

## 概述

《万法归宗》为道教法术总集，收录各类符咒、法术、科仪，号称"万法归一，道法自然"。

---

## 卷一：道法总纲

### 道法本源

道法者，以道为本，以法为用。

**三宝**：
- 道：宇宙本源
- 经：圣真经典  
- 师：传承明师

**三乘**：
- 上乘：内丹修炼
- 中乘：符咒法术
- 下乘：科仪斋醮

### 修法须知

学法先修德，德高法自灵。

**五戒**：
1. 不杀生
2. 不偷盗
3. 不邪淫
4. 不妄语
5. 不饮酒

## 卷二：符咒篇

### 符之构成

**符三要素**：
- **符头**：请神敕令
- **符腹**：核心内容
- **符胆**：灵力所聚

### 书符须知

1. **时辰**：宜子午卯酉
2. **方位**：面北向南
3. **心境**：诚心正念
4. **材料**：黄纸朱砂

### 常用符咒

#### 一、平安符

**用途**：保平安、避凶险

**画法**：
```
符头：三清敕令
符腹：太上老君急急如律令
符胆：敕令平安
```

#### 二、招财符

**用途**：招财进宝

**画法**：
```
符头：赵公明元帅令
符腹：五路财神招财进宝
符胆：敕令招财
```

#### 三、驱邪符

**用途**：驱邪避煞

**画法**：
```
符头：太上老君敕令
符腹：天圆地方律令九章
符胆：敕令驱邪
```

#### 四、姻缘符

**用途**：催姻缘、和合

**画法**：
```
符头：月老仙师令
符腹：千里姻缘一线牵
符胆：敕令和合
```

### 核心咒语

#### 一、净心神咒

```
太上台星，应变无停。
驱邪缚魅，保命护身。
智慧明净，心神安宁。
三魂永久，魄无丧倾。
急急如律令。
```

#### 二、净口神咒

```
丹朱口神，吐秽除氛。
舌神正伦，通命养神。
罗千齿神，却邪卫真。
喉神虎喷，气神引津。
心神丹元，令我通真。
思神炼液，道气长存。
急急如律令。
```

#### 三、净身神咒

```
灵宝天尊，安慰身形。
弟子魂魄，五脏玄冥。
青龙白虎，队仗纷纭。
朱雀玄武，侍卫我真。
急急如律令。
```

#### 四、安土地咒

```
元始安镇，普告万灵。
岳渎真官，土地祗灵。
左社右稷，不得妄惊。
回向正道，内外澄清。
各安方位，备守家庭。
太上有命，搜捕邪精。
护法安人，各各归依。
弟子皈命，道法得成。
急急如律令。
```

#### 五、金光神咒

```
天地玄宗，万气本根。
广修亿劫，证吾神通。
三界内外，惟道独尊。
体有金光，覆映吾身。
视之不见，听之不闻。
包罗天地，养育群生。
受持万遍，身有光明。
三界侍卫，五帝司迎。
万神朝礼，役使雷霆。
鬼妖丧胆，精怪亡形。
内有霹雳，雷神隐名。
洞慧交彻，五气腾腾。
金光速现，覆护真人。
急急如律令。
```

## 卷八：问答篇

**问：符咒灵验否？**

答：符本无灵，灵在人心。心诚则灵，心乱则废。

**问：人人可学法乎？**

答：人人可学，非人人可成。德行为本，根基为要。

**问：法术可伤人乎？**

答：法不伤人，人自伤之。恶用反噬，善用积德。
