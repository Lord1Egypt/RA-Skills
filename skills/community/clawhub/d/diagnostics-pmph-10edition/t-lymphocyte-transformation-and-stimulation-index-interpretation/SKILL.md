# T淋巴细胞转化率与刺激指数判读

## 何时使用
- 已完成T淋巴细胞转化实验（使用形态学计数法或³H-TdR掺入法）
- 需评估患者细胞免疫功能（如怀疑免疫缺陷、自身免疫病、肿瘤或感染）
- 需解释唐氏综合征等特殊人群的异常结果

## 执行步骤
1. **确认实验方法**：
   - 若为**形态学计数法**：
     - 正常参考区间：60.1% ± 7.6%
     - 转化率 < 50% → 判定为“T淋巴细胞转化低下”
   - 若为**³H-TdR掺入法**：
     - 计算刺激指数（SI）= 实验组放射性计数 / 对照组放射性计数
     - SI > 2 → 视为“有生物学意义的T细胞反应”
     - SI < 2 → 判定为“淋巴细胞转化低下”

2. **临床意义解读**：
   - 该试验反映体外T细胞功能，用于评估细胞免疫水平
   - 可辅助判断疾病疗效与预后
   - **注意例外**：唐氏综合征患者T细胞转化率常明显增高

3. **注意事项**：
   - 结果必须结合临床表现及其他免疫指标综合判断
   - 不同实验室方法存在系统偏差，应优先采用本实验室建立的参考区间
   - 本技能不适用于B细胞功能评估
