## Use Case: <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Security evidence flags a bundled autoreview helper that may run nested review with broad local access and approval bypass. <br>
Mitigation: Install only from a trusted publisher and keep sandbox and approval controls enabled; set AUTOREVIEW_YOLO=0 or pass --no-yolo before using that helper. <br>
