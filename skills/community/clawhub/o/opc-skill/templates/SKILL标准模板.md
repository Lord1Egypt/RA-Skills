## 二、模板使用指南

### 2.1 YAML Frontmatter必填项

| 字段 | 格式 | 说明 |
|------|------|------|
| name | 小写连字符 | Skill唯一标识 |
| description | 50字以内 | 一句话描述 |
| license | MIT/Proprietary | 许可证类型 |
| compatibility | 平台列表 | 支持的平台 |
| author | 字符串 | 作者信息 |
| version | semver格式 | 版本号 |
| pricing | free/premium_XX | 定价策略 |

### 2.2 平台兼容性说明

| 平台 | 标识 | 要求 |
|------|------|------|
| Coze商店 | coze | 符合Coze规范 |
| 虾小宝 | xiaopeng | 提供中英双语 |
| ClawHub | clawhub | 完整README |

### 2.3 定价策略

| 定价 | 格式 | 说明 |
|------|------|------|
| 免费 | free | 完全免费使用 |
| 低价 | premium_99 | ¥99/次 |
| 中价 | premium_149 | ¥149/次 |
| 高价 | premium_199 | ¥199/次 |

## 三、目录结构规范

```
[Skill名称]/
├── SKILL.md                    # 主文件（必须）
│   ├── YAML frontmatter
│   ├── 内容主体
│   ├── Follow-up模块
│   ├── 文档推荐模块
│   └── 版权声明
├── scripts/                    # 脚本目录
│   ├── recommend.py           # 推荐引擎
│   └── [其他工具脚本].py
├── references/                 # 参考资料
│   └── [参考文档].md
└── templates/                  # 模板目录
    └── [输出模板].md
```
