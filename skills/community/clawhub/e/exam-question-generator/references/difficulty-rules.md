# 三套试卷难度规则

## 总原则

三套试卷构成递进学习路径：**基础巩固 → 综合能力 → 冲刺模拟**
每套完全独立，可单独使用，但三套合用效果最佳。

---

## 套1 · 基础巩固

**目标**：覆盖所有考点，确认基础理解，消灭记忆盲区。

### 出题规则

**单选题（mc）**
- 题干：直接考查概念定义或公式，无歧义，无场景包装
- 干扰项：明显错误，如"完全相反的陈述"或"常见笔误"
- 示例题干模式："以下关于X的说法，正确的是"

**判断题（tf）**
- 陈述基本事实或常见误解，让学生判断对错
- 错误陈述选取该考点最高频的学生误区
- 解析必须指出错误所在，不能只写"错误"

**简答题（sa）**
- 题干模式："什么是X""列举Y的主要特点""简述Z的步骤"
- 答案结构：2–4个要点，每点50字以内
- 无需推导或综合，直接提取知识点即可作答

### 考点优先级
1. is_weak_point = true → 最先出题，确保覆盖
2. importance 降序排列
3. 每题 knowledge_ref 对应单一考点 id

---

## 套2 · 综合能力

**目标**：检验理解与应用，识别知识点之间的关联。

### 出题规则

**单选题（mc）**
- 题干：给出具体场景或例子，需判断适用哪个概念/规律
- 干扰项：含"似对实错"选项，需仔细辨别
- 示例题干模式："在X情况下，Y的结果是"

**多选题（mma）**
- 从多个维度考查同一考点，或考查相关联的多个考点
- 正确答案设计为2–3个，避免全选或全错
- 每题 knowledge_ref 可关联 1–2 个考点 id

**判断题（tf）**
- 考查概念之间的区别与联系，错误陈述为近义混淆
- 示例："A与B的区别在于…（判断）"

**简答题（sa）**
- 题干模式："分析X的原因""比较A与B的异同""说明为什么Z"
- 答案需体现逻辑推理，而非单纯列举

### 考点优先级
- importance ≥ 0.7 的考点优先出现在套2
- is_weak_point = true 的考点：单选或多选中设置有针对性的陷阱选项

---

## 套3 · 冲刺模拟

**目标**：仿真考试场景，检验综合应变能力，暴露认知盲区。

### 出题规则

**单选题（mc）**
- 含语言陷阱，相似概念容易混淆，需精确辨别
- 题干可含"以下说法错误的是"反向题型
- 干扰项：设计"看起来对但有一处细节错误"的选项

**多选题（mma）**
- 复杂场景，选项中含"似对实错"和"似错实对"各至少一个
- 设计漏选陷阱：学生容易遗漏的正确选项
- 可跨多个考点设计综合题

**判断题（tf）**
- 考查细节差异，如充分条件 vs 充要条件、定义中的边界条件
- 陈述方式接近正确，但存在关键词偷换

**简答题（sa）**
- 题干模式："综合分析…""结合X和Y，论述Z""从多角度说明…"
- 答案要求串联 2–3 个相关考点，有逻辑结构
- 关键词提示：给分参考点包含跨考点的综合表述

### 考点优先级
- 仅覆盖 importance ≥ 0.8 的高优先级考点
- 题目风格尽量贴近目标考试的真实题型表述习惯

## 难度梯度示例（以"极值定理"为例）

**套1（基础）**
> 若函数 f(x) 在 x₀ 处的导数 f'(x₀) = 0，则 x₀ 是 f(x) 的极值点。（判断）
>
> 答：**错误**。f'(x₀) = 0 是极值点的必要条件而非充分条件。例如 f(x) = x³ 在 x = 0 处导数为0，但 x = 0 不是极值点。

**套2（综合）**
> 设 f(x) = x³，下列说法正确的是（单选）：
> A. x = 0 是极大值点　B. x = 0 是极小值点　C. x = 0 既不是极大也不是极小值点　D. 无法判断
>
> 答：**C**。f'(0) = 0，f''(0) = 0，二阶导数判别法失效。由 f'(x) = 3x² ≥ 0，x = 0 两侧导数同号，故 x = 0 非极值点。

**套3（冲刺）**
> 若函数 f(x) 在 x₀ 处满足 f'(x₀) = 0 且 f''(x₀) = 0，则以下判断正确的有（多选）：
> A. x₀ 一定是极大值点　B. x₀ 一定是极小值点　C. x₀ 可能是极值点　D. x₀ 可能是拐点　E. x₀ 一定是拐点
>
> 答：**CD**。二阶导为0时无法通过二阶判别法下结论，需用一阶导变号判别极值，用凹凸性变化判别拐点，两种情况均有可能。

## 答案解析格式规范（三套统一）

简答题 answer 字段格式：
```
1. [要点一内容]
2. [要点二内容]
3. [要点三内容（如有）]
```

explanation 字段：
- 单选/多选：说明各选项对错理由，重点解释干扰项为何错
- 判断：给出完整判断逻辑链
- 简答：列出关键词和给分参考点
