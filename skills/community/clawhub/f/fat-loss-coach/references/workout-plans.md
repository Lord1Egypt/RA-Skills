# 运动方案

## 新手方案（第1-4周）

### 第1-2周：适应期

**目标：** 建立运动习惯，适应身体

**每周安排：**
| 天 | 内容 | 时长 | 强度 |
|----|------|------|------|
| 1 | 快走 | 30分钟 | 轻松 |
| 2 | 快走 | 30分钟 | 轻松 |
| 3 | 休息 | - | - |
| 4 | 快走 | 30分钟 | 轻松 |
| 5 | 快走 | 30分钟 | 轻松 |
| 6 | 休息 | - | - |
| 7 | 快走或休息 | 30分钟 | 轻松 |

**要点：**
- 快走心率：100-120次/分（能正常说话）
- 选择平坦路面或跑步机
- 穿舒适的运动鞋

### 第3-4周：过渡期

**目标：** 加入简单力量训练，提高强度

**每周安排：**
| 天 | 内容 | 时长 |
|----|------|------|
| 1 | 快走 40分钟 + 深蹲 | 50分钟 |
| 2 | 快走 30分钟 | 30分钟 |
| 3 | 休息 | - |
| 4 | 快走 40分钟 + 俯卧撑 | 50分钟 |
| 5 | 快走 30分钟 | 30分钟 |
| 6 | 休息 | - |
| 7 | 可选轻松活动 | - |

**力量训练入门：**

**深蹲：**
- 3组，每组10-15次
- 动作要点：脚尖略朝外，下蹲时膝盖不超过脚尖
- 可从椅子辅助深蹲开始

**俯卧撑：**
- 3组，每组5-10次
- 女性可从跪姿俯卧撑开始
- 动作要点：身体保持一条直线

**平板支撑：**
- 3组，每组20-30秒
- 核心收紧，不塌腰

3. 弓步蹲：3组 x 10次（每侧）
4. 臀桥：3组 x 15次
5. 小腿提踵：3组 x 20次

**核心日：**

1. 平板支撑：3组 x 60秒

3. 死虫式：3组 x 12次
4. 仰卧抬腿：3组 x 15次

## 高级方案（第9周+）

### 有氧训练

**选择1：稳态有氧**
- 频率：每周3次
- 时长：45-60分钟
- 心率：最大心率的60-70%
- 类型：跑步、游泳、骑车、椭圆机

**选择2：HIIT**
- 频率：每周1-2次（不连续）
- 格式：30秒冲刺 + 30秒休息，循环15-20分钟
- 类型：冲刺跑、跳绳、波比跳

### 力量训练

**频率：** 每周3-4次

**分化方案：**

**方案A：全身训练（每周3次）**
- 深蹲、硬拉、卧推、划船、推举

**方案B：上/下肢分化（每周4次）**
- 上肢日：卧推、划船、推举、弯举
- 下肢日：深蹲、硬拉、弓步蹲、臀桥

**方案C：推/拉/腿分化（每周3-6次）**
- 推日：卧推、推举、俯卧撑、三角肌
- 拉日：硬拉、划船、下拉、弯举
- 腿日：深蹲、弓步蹲、臀桥、小腿

## 运动强度指南

### 有氧强度划分

| 等级 | 心率范围 | 体感 | 用途 |
|------|----------|------|------|
| 低强度 | 最大心率50-60% | 轻松，能正常说话 | 热身、恢复 |
| 中强度 | 最大心率60-70% | 微喘，能说话但不能唱歌 | 燃脂区 |
| 高强度 | 最大心率70-85% | 呼吸急促，说几个词就喘 | 提升心肺 |
| 极高强度 | 最大心率85%+ | 无法说话 | HIIT冲刺 |

**最大心率估算：** 220 - 年龄

### 力量训练重量选择

**增肌/力量：**
- 重量：能完成8-12次的重量
- 最后2次应该感到吃力

**耐力：**
- 重量：能完成15-20次
- 较轻重量，多次数

**午休：**

- 快走回家30分钟

## 运动前后

### 热身（5-10分钟）

**目的：** 升高体温，激活肌肉，预防受伤

**通用热身：**
- 慢走/慢跑 3分钟
- 关节活动：肩、髋、膝、踝各转动10次
- 动态拉伸：摆腿、转体

### 冷身拉伸（5-10分钟）

**目的：** 降低心率，放松肌肉，促进恢复

**拉伸重点：**
- 大腿前/后侧拉伸：各30秒
- 髋部拉伸：30秒
- 胸部拉伸：30秒
- 背部拉伸：30秒

## 常见问题

### 运动后腿酸痛怎么办？

- 轻度酸痛：正常现象，继续运动但减量
- 中度酸痛：增加拉伸，温水泡脚
- 重度酸痛：休息1-2天，下次减量

### 没时间运动怎么办？

- 利用碎片时间：爬楼梯、走路通勤
- 高强度短时训练：HIIT 15-20分钟
- 把运动融入日常：边看电视边做力量训练

### 运动后反而更饿了？

- 正常现象：运动消耗增加食欲
- 解决：运动后30分钟内补充蛋白质
- 避免：运动后大吃高热量食物

### 女性担心练出大肌肉？

- 不容易：女性睾酮水平低，很难练出大块肌肉
- 力量训练的好处：提高代谢、塑形、预防骨质疏松
- 想要紧致线条：必须力量训练
