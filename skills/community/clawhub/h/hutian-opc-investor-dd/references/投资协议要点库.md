# 投资者背景调查技能 (Investor Due Diligence)

## 1. 技能介绍

### 技能名称
投资者背景调查

### 适用场景
- 融资前对潜在投资者进行尽职调查
- 评估投资机构的靠谱程度与合作风险
- 识别"毒药钱"（估值陷阱/条款剥削/资源诈欺）
- 制定针对性的融资对接策略

### 核心价值
- **风险规避**：避免与劣迹机构合作，防止资金链断裂
- **策略优化**：根据机构风格定制化BP与谈判策略
- **资源匹配**：找到真正能带来产业赋能的投资者
- **谈判优势**：掌握机构底牌，在条款谈判中占据主动

### 使用限制
- 本技能不涉及法律建议，重大条款请咨询专业律师
- 信息来源为公开数据，准确性依赖数据源质量
- 建议多渠道交叉验证，避免单一信息依赖

## 2. 调查框架

### 2.1 机构基本信息

| 调查维度 | 具体内容 | 数据来源 |
|---------|---------|---------|
| **管理规模** | 在管基金总规模、累计投资项目数 | 中基协、官网、LP报告 |
| **成立时间** | 成立年份、存续期（通常7-10年） | 工商信息、中基协 |
| **团队背景** | 核心合伙人履历、专业背景、投资经验 | 官网、LinkedIn、新闻报道 |
| **股权结构** | 国资/民营/混合所有制、LP构成 | 工商信息、企业公告 |
| **合规资质** | 基金业协会备案、是否有处罚记录 | 中基协官网 |

### 2.2 投资偏好深度画像

#### 2.2.1 细分赛道偏好（细化分析）

| 偏好层级 | 示例（以医疗为例） | 评估方法 |
|---------|------------------|---------|
| **宏观赛道** | 医疗健康、生物医药、医疗器械 | 官网定位、投资组合分布 |
| **中观领域** | 创新药、辅助生殖、医疗AI | IT桔子分类、新闻报道 |
| **细分方向** | 核医学、BNCT、CAR-T、基因编辑 | 投资项目关键词、专利布局 |
| **技术路线** | 小分子、大分子、细胞治疗、放射治疗 | 学术论文、临床试验数据 |

**评估标准：**
- 高度匹配：机构有明确且持续的投资记录
- 中度匹配：偶有涉及但非核心
- 低度匹配：几乎没有相关投资

#### 2.2.2 投资风格画像

| 风格类型 | 特征描述 | 识别信号 |
|---------|---------|---------|
| **激进型** | 追求高风险高回报、敢于押注早期、接受高估值 | 偏好早期轮次、出手快、愿意给高估值 |
| **稳健型** | 平衡风险与回报、偏好成长期、追求合理估值 | 偏好B-C轮、有明确财务指标要求 |
| **保守型** | 追求确定性、偏好成熟期、要求严格对赌 | 要求营收规模、偏好后期轮次 |

**判断方法：**
```
激进型信号：
- 投资组合中有大量天使/种子轮
- 接受没有收入的早期项目
- 投资估值明显高于市场水平
- 愿意快速决策、快速打款

稳健型信号：
- 投资组合以A-B轮为主
- 有明确的财务/业务指标要求
- 尽调周期适中（2-3个月）
- 条款设置平衡

保守型信号：
- 偏好C轮以后项目
- 要求收入/利润规模
- 决策周期长（3-6个月）
- 条款严苛、要求担保
```

#### 2.2.3 估值偏好分析

| 评估维度 | 具体内容 | 数据来源 |
|---------|---------|---------|
| **估值区间** | 可接受估值范围（百万-千万级） | 历史投资案例估值 |
| **估值锚定** | 按营收/用户/临床进度估值 | 行业惯例对比 |
| **估值调整机制** | 是否接受平权估值、是否有对赌 | 条款分析 |
| **稀释比例** | 单轮释放股权比例偏好 | 投资协议 |

**估值评估模板：**
```markdown
机构估值偏好评估：
- 种子轮典型估值：___万-___万（___倍PS/___倍PE）
- 天使轮典型估值：___万-___万（___倍PS/___倍PE）
- A轮典型估值：___万-___万（___倍PS/___倍PE）
- 单轮稀释比例：___%-___%
- 接受最高估值：___万（超出则需特殊理由）
```

#### 2.2.4 条款偏好分析

| 条款类型 | 激进型偏好 | 稳健型偏好 | 保守型偏好 |
|---------|-----------|-----------|-----------|
| **对赌条款** | 宽松业绩承诺 | 中等对赌条件 | 严格对赌+担保 |
| **回购条款** | 延期回购、低息 | 中等回购条件 | 即时触发、高比例 |
| **反稀释** | 豁免或加权平均 | 有限加权平均 | 完全加权平均 |
| **董事会席位** | 观察员即可 | 1席观察员 | 要求董事会多数 |
| **否决权** | 仅核心事项 | 限定事项否决 | 广泛否决权 |
| **竞业禁止** | 宽松范围 | 中等限制 | 严格限制创始人 |
| **随售权** | 较低比例（10-20%） | 中等比例（20-30%） | 较高比例（30%+） |
| **拖售权** | 无或豁免 | 特定条件触发 | 主动权在投资方 |

#### 2.2.5 退出预期

| 评估维度 | 具体内容 | 评估重点 |
|---------|---------|---------|
| **预期回报倍数** | IRR目标（通常15%-30%） | 与市场水平对比 |
| **预期持有周期** | 3年/5年/7年/更长 | 与基金存续期匹配 |
| **退出方式偏好** | IPO/并购/S基金/清算 | 灵活性评估 |
| **退出时机要求** | 上市时间表要求 | 是否合理可行 |
| **接盘能力** | 能否对接并购方/下轮融资 | 产业资源 |

### 2.4 基金运营数据

> 参考：`references/基金运营数据模板.md`

#### 2.4.1 核心指标定义

| 指标 | 定义 | 行业基准 | 评估标准 |
|-----|------|---------|---------|
| **AUM** | 管理资产规模（AuM） | 各机构差异大 | 与投资阶段匹配 |
| **IRR** | 内部收益率 | 15-25%优秀 | 越高越好 |
| **DPI** | 已分配收益倍数 | >1为盈利 | 现金回报能力 |
| **TVPI** | 总价值倍数 | 1.5-3倍 | 含未实现收益 |
| **MOIC** | 投入资本倍数 | 1.5-3倍 | 绝对回报 |
| **J曲线** | 早期负收益后期正收益曲线 | 正常现象 | 需结合年限分析 |

#### 2.5.2 决策链分析维度

| 决策层级 | 关键人物 | 影响力 | 接触策略 |
|---------|---------|-------|---------|
| **最终拍板人** | 创始合伙人/管理合伙人 | 100%（一票通过/否决） | 通过深度关系引荐 |
| **投决会成员** | 核心合伙人 | 投票权重高 | 参与路演、建立信任 |
| **MD/ED** | 执行董事/董事总经理 | 60-80%（主导推进） | 直接对接，维护关系 |
| **VP** | 投资副总监/高级经理 | 30-50%（撰写报告） | 专业沟通，保持联系 |
| **投资经理** | Associate/Analyst | 10-20%（初步筛选） | 礼貌尊重，及时响应 |

#### 2.5.3 决策流程时间表

| 决策阶段 | 正常周期 | 过长信号 | 风险提示 |
|---------|---------|---------|---------|
| **初次接触→内核/立项** | 1-2周 | >3周 | 兴趣不足或流程复杂 |
| **立项→尽调启动** | 1-2周 | >1个月 | 需要补充材料或内部讨论 |
| **尽调执行** | 2-4周 | >2个月 | 问题较多或团队不专业 |
| **尽调→投决** | 1-2周 | >1个月 | 内部争议或需要补充尽调 |
| **投决→TS** | 1周内 | >2周 | 条款谈判困难 |
| **TS→SPA→打款** | 2-4周 | >1个月 | 法律/合规问题或LP审批 |

#### 2.5.4 否决原因分析

**高频否决原因：**
```
市场规模问题：
- TAM（总可寻址市场）太小
- 细分赛道天花板低

团队问题：
- 创始人背景不够亮眼
- 团队不完整或缺乏关键人才
- 股权结构不合理

商业模式问题：
- 盈利模式不清晰
- 单位经济模型不健康
- 获客成本过高

竞争问题：
- 赛道已有领先者
- 技术护城河不足
- 差异化不明显

估值问题：
- 估值过高
- 融资额与阶段不匹配

其他问题：
- 利益冲突（已投竞品）
- 合规/法律问题
- 关键人物否决
```

### Step 2: 投资案例分析（20分钟）

3. **退出案例研究**

## 三、对赌条款（Valuation Adjustment Mechanism, VAM）

### 3.1 对赌类型

| 类型 | 定义 | 适用场景 | 风险等级 |
|-----|------|---------|---------|
| **业绩对赌** | 未达业绩目标触发估值调整 | 成长期企业 | 中 |
| **里程碑对赌** | 未达关键里程碑触发调整 | 早期/研发型企业 | 中低 |
| **上市对赌** | 未在约定时间上市触发回购 | Pre-IPO轮次 | 高 |
| **复合对赌** | 多重条件组合 | 复杂交易 | 视情况 |

### 3.2 业绩对赌要素

| 要素 | 常见设置 | 评估要点 |
|-----|---------|---------|
| **对赌指标** | 营收/利润/GMV/用户数 | 是否合理可达 |
| **对赌期限** | 1-3年 | 是否合理 |
| **调整方式** | 估值调整/股权补偿/现金补偿 | 影响大小 |
| **调整比例** | 单次调整上限 | 是否有保护 |

**计算示例：**
```
原估值：4000万（1000万投资，占20%）
承诺业绩：年营收5000万

情形A（达成）：
- 维持原估值和股权比例

情形B（未达成70%）：
- 估值调整为：4000万 × 70% = 2800万
- 投资方应占股权：1000/2800 = 35.7%
- 创始团队需补偿：35.7% - 20% = 15.7%股权
```

### 3.3 谈判建议

```
✅ 可接受的对赌：
- 里程碑对赌（而非纯财务指标）
- 弹性目标区间（80-100%均可接受）
- 股权补偿而非现金补偿
- 宽限期（3-6个月补救期）

❌ 需避免的对赌：
- 严格业绩承诺（无弹性）
- 现金补偿（可能影响运营）
- 严苛的回购触发条件
- 无上限的股权补偿
```

## 四、回购条款（Redemption Rights）

### 4.1 回购触发条件

| 触发条件 | 触发时间 | 触发主体 | 风险评估 |
|---------|---------|---------|---------|
| **IPO触发** | 约定时间内未上市 | 投资人选择 | 高 |
| **融资触发** | 后续融资估值低于本轮 | 投资人选择 | 中 |
| **创始人触发** | 创始人离职/违约 | 投资人选择 | 高 |
| **事件触发** | 特定负面事件 | 自动触发 | 视情况 |

### 4.2 回购价格

| 类型 | 计算方式 | 市场惯例 | 评估 |
|-----|---------|---------|-----|
| **本金+利息** | 投资本金 × (1 + 年利率 × 年数) | 年利率6-12% | 合理 |
| **固定收益** | 本金 + 固定收益（8-15%） | 视情况 | 可接受 |
| **市价回购** | 按届时估值 | 退出时确定 | 风险较高 |

**计算示例：**
```
投资金额：1000万
投资期限：3年
年利率：8%

回购价格 = 1000万 × (1 + 8% × 3) = 1240万
```

### 4.3 谈判建议

```
✅ 友好条款：
- 延期触发（投资2年后才可触发）
- 低利率（≤8%）
- 创始人个人不连带
- 仅在IPO失败时触发

❌ 严苛条款：
- 立即触发
- 高利率（≥12%）
- 创始人个人连带责任
- 主动权在投资方
```

## 五、反稀释条款（Anti-dilution）

### 5.1 反稀释类型

| 类型 | 计算方式 | 对创始人影响 | 市场偏好 |
|-----|---------|-------------|---------|
| **完全加权平均** | 全额调整，考虑所有后续融资 | 较大保护投资人 | 最保护投资人 |
| **狭义加权平均** | 仅考虑本轮和后续融资 | 中等保护 | 中等 |
| **无反稀释** | 不调整 | 最保护创始人 | 较少见 |

### 5.2 加权平均计算公式

```
完全加权平均转换价格 = (A + B) / (C + D)

其中：
A = 转换前投资人持有的股权价值
B = 后续融资中收到的金额
C = 完全稀释后的总股本（转换前）
D = 后续融资增加的股本

转换价格 = min(原始价格, 加权平均价格)
```

### 5.3 谈判建议

```
✅ 友好条款：
- 豁免例外：员工期权、战略合作等不触发
- 有限加权平均（而非完全加权）
- 仅适用于"严重降价融资"（>20-30%降幅）
- 有最长保护期限

❌ 严苛条款：
- 完全加权平均无豁免
- 无门槛触发
- 无限期保护
```

## 六、优先清算权（Liquidation Preference）

### 6.1 清偿顺序

```
1. 债权人（银行贷款等）
   ↓
2. 优先股股东（投资人）
   ↓
3. 普通股股东（创始团队）
```

### 6.2 优先清算类型

| 类型 | 说明 | 创始人影响 | 市场惯例 |
|-----|------|-----------|---------|
| **1倍非参与** | 优先收回本金，不参与分配 | 较小 | 最常见 |
| **1倍参与** | 优先收回后，继续按持股比例分配 | 中等 | 较常见 |
| **2-3倍优先** | 按2-3倍本金优先收回 | 较大 | 较少见 |

### 6.3 参与分配上限

| 类型 | 说明 | 评估 |
|-----|------|-----|
| **无上限** | 可同时获得优先权和分配 | 最不利 |
| **有上限（1-2倍）** | 最高获得投资额1-2倍 | 较合理 |
| **无参与权** | 仅获优先分配，不参与剩余 | 最友好 |

### 6.4 谈判建议

```
✅ 友好条款：
- 1倍非参与型优先清算
- 或有上限参与（≤1.5倍）
- 触发条件限定（仅"重大出售"）

❌ 严苛条款：
- 2倍以上优先
- 无上限参与
- 触发条件宽泛
```

## 七、随售权/拖售权

### 7.1 随售权（Tag-along Rights）

| 要点 | 说明 |
|-----|------|
| **定义** | 创始人出售股权时，投资人有权按同等条件出售 |
| **触发** | 创始人/早期股东向第三方转让 |
| **比例** | 通常10-30%按比例参与 |
| **优先购买权** | 随售前，投资方可优先购买 |

**谈判要点：**
```
✅ 合理范围：
- 随售比例≤20%
- 超过一定比例才触发（如>5%转让）
- 大股东间转让豁免

❌ 过于严格：
- 随售比例>30%
- 任何转让都触发
- 无豁免情形
```

### 7.2 拖售权（Drag-along Rights）

| 要点 | 说明 |
|-----|------|
| **定义** | 特定条件下，投资人可强制其他股东一起出售 |
| **触发条件** | 第三方收购、投资方同意、价格门槛 |
| **时间要求** | 通常需给其他股东提前通知（30-60天） |
| **评估** | 对创始人影响较大 |

**谈判建议：**
```
✅ 友好条款：
- 触发条件严格（需超过一定估值）
- 需大多数股东同意
- 给创始人合理通知期
- 创始人有权优先购买

❌ 严苛条款：
- 投资方单方决定
- 极低门槛触发
- 无优先购买权
```

## 八、信息披露与知情权

### 8.1 定期报告

| 报告类型 | 频率 | 内容 | 评估 |
|---------|-----|------|-----|
| **财务报告** | 季度/年度 | 资产负债表、损益表、现金流量表 | 标准要求 |
| **运营报告** | 月度/季度 | 业务进展、用户数据 | 视要求合理性 |
| **重大事项** | 及时 | 诉讼、人员变动、重要合同 | 必须 |
| **预算计划** | 年度 | 年度预算、资金计划 | 合理 |

### 8.2 审计要求

| 类型 | 要求 | 评估 |
|-----|------|-----|
| **年度审计** | 四大会计师事务所 | 成本较高，视规模 |
| **季度审阅** | 外部会计师审阅 | 可接受 |
| **内部审计** | 投资人内部团队 | 需控制频率 |

### 8.3 知情权范围

```
✅ 合理知情权：
- 财务报表、重大合同
- 诉讼/仲裁信息
- 股权变动
- 关联交易

❌ 过度知情权：
- 所有员工合同详情
- 日常经营决策
- 竞争对手信息
- 核心技术细节
```

## 九、创始人限制条款

### 9.1 竞业禁止

| 要素 | 合理范围 | 过于严格 |
|-----|---------|---------|
| **时间** | 任职期间+离职后12-18个月 | 超过24个月 |
| **范围** | 同行业/直接竞争业务 | 泛化到"相关行业" |
| **地域** | 公司业务覆盖区域 | 全球/无限地域 |
| **补偿** | 任职期间无需额外补偿 | 要求额外补偿 |

### 9.2 股权锁定

| 要点 | 说明 | 评估 |
|-----|------|-----|
| **锁定期** | 任职期间或约定年限 | 通常合理 |
| **提前终止** | 允许一定比例提前成熟 | 需争取 |
| **回购权** | 离职时回购未成熟股权 | 需明确条件 |

### 9.3 禁止挖角

| 范围 | 合理 | 过于严格 |
|-----|------|---------|
| 禁止挖核心员工 | 合理 | - |
| 禁止任何员工 | 过于宽泛 | ❌ |
| 明确"核心"定义 | 合理 | - |
| 无限期禁止 | 不合理 | ❌ |

## 十、条款可谈判性评估

### 10.1 核心条款（底线）

| 条款 | 底线建议 | 说明 |
|-----|---------|-----|
| 董事会多数 | 必须争取 | 保持团队控制权 |
| 重大事项否决权 | 严格限定范围 | 避免干预日常经营 |
| 回购主体 | 仅公司，非创始人个人 | 保护创始人 |
| 竞业禁止 | 任职期间合理 | 避免过度限制 |
| 随售权比例 | ≤20% | 避免套牢 |

### 10.2 可谈判条款

| 条款 | 建议 | 谈判空间 |
|-----|------|---------|
| 估值 | 争取合理估值 | 视市场情况 |
| 对赌指标 | 争取弹性区间 | 通常可谈 |
| 反稀释类型 | 争取豁免例外 | 可争取 |
| 回购利率 | 争取≤8% | 通常可谈 |
| 锁定期 | 争取分期成熟 | 可谈 |

### 10.3 友好机构特征

```
✅ 机构风格友好标志：
- 跟投为主，不追求控制
- 条款模板偏向创始人友好
- 历史案例多为中早期项目
- 强调"赋能"而非"管控"
- 投后管理轻量化
- 退出方式灵活（不强制IPO）
- 愿意接受共同条款（非最惠国待遇）

⚠️ 需警惕的风格：
- 领投为主，追求控制
- 条款模板严苛
- 投资组合涉及大量竞品
- 投后管理强势
- 强制要求IPO退出
- 坚持最惠国待遇
```
