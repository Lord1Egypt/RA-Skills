# 达人匹配规则 (influencer_matching.md)

## 概述

本文件定义达人匹配（Step 8）的匹配逻辑、达人分类体系、筛选标准和邀约话术模板。基于商品特征和策略方向，智能匹配最优达人类型。

---

## 达人类型分类体系

### 五大达人类型

| 类型 | 特点 | 适用品类 | 转化特点 |
|------|------|---------|----------|
| **测评型** | 专业评测、数据说话、信任度高 | 美妆、数码、家电、食品 | 转化率高但受众窄 |
| **好物分享型** | 日常种草、自然推荐、亲和力强 | 日用、零食、家居、母婴 | 曝光量大但转化率中等 |
| **剧情植入型** | 剧情中自然带货、记忆度高 | 全品类 | 品牌认知强但即时转化弱 |
| **知识科普型** | 专业讲解、成分/技术解析 | 护肤、保健品、母婴、宠物 | 高客单价品转化好 |
| **促销带货型** | 价格驱动、节奏快、冲动消费 | 白牌、清仓、大促品 | 转化高但复购低 |

## 匹配规则：从商品特征 → 达人类型

### 规则一：按商品类目匹配

| 商品类目 | 首选达人类型 | 次选 | 不建议 |
|---------|------------|------|--------|
| 美妆护肤 | 测评型/知识科普型 | 好物分享型 | 促销带货型（伤品牌） |
| 食品零食 | 好物分享型 | 促销带货型 | 剧情植入型（转化路径长） |
| 数码3C | 测评型 | 知识科普型 | 好物分享型（受众不精准） |
| 家居日用 | 好物分享型 | 促销带货型 | 知识科普型（品类属性弱） |
| 母婴用品 | 知识科普型 | 测评型 | 促销带货型（品类敏感） |
| 服装鞋包 | 好物分享型 | 剧情植入型 | 测评型（展示效率低） |
| 保健品 | 知识科普型 | 测评型 | 促销带货型（合规风险高） |
| 宠物用品 | 好物分享型 | 知识科普型 | — |
| 白牌/贴牌 | 促销带货型 | 好物分享型 | 测评型（经不起测） |

### 规则二：按策略方向修正

在类目匹配基础上，策略方向会修正达人选择：

| 策略方向 | 达人偏好修正 |
|---------|------------|
| 利益驱动（高佣金） | 优先促销带货型，话术突出佣金和 ROI |
| 品牌溢价（强品牌） | 优先测评型/知识科普型，话术突出品牌背书 |
| 价格锚定（竞争激烈） | 优先好物分享型+促销带货型混合策略 |
| 认知教育（新品类） | 优先知识科普型，话术以教育内容打底 |
| 信任建设（新品牌） | 优先测评型，以第三方评测数据建立信任 |

### 规则三：按客单价修正

| 客单价 | 达人策略修正 |
|--------|------------|
| < 50元 | 促销带货型/好物分享型，冲动消费友好 |
| 50-200元 | 好物分享型/测评型，需要一定信任门槛 |
| 200-500元 | 测评型/知识科普型，需要建立完整信任 |
| > 500元 | 知识科普型/测评型，长决策周期 |

## 达人筛选标准模版

### 输出格式

对每个推荐的达人类型，输出一套筛选标准：

```
达人类型：[类型名称] — 优先级：⭐⭐⭐

粉丝量建议：[区间] 万粉
带货分要求：≥ [分数]
内容风格关键词：[词1]、[词2]、[词3]
近期带货品类匹配：[匹配度要求]
达人筛选提示：[一句话建议，如"优先找最近一个月带过同品类且数据好的达人"]
```

### 示例

```
达人类型：测评型美妆达人 — 优先级：⭐⭐⭐

粉丝量建议：10-50 万粉（腰尾部精准测评粉，性价比最高）
带货分要求：≥ 4.5 分
内容风格关键词：成分党、实测、红黑榜、空瓶记
近期带货品类匹配：近30天带过洗发/护发类目
达人筛选提示：优先找做过"洗发水红黑榜"或"发质实测"内容的达人，转化率明显高于纯好物种草
```

## 达人邀约话术模板

### 分层策略

按沟通阶段提供 3 套话术，从初次触达到合作确认。

### 模板一：初次触达话术

```
结构：
1. 破冰句（1句，自然不油腻）
2. 品介绍（2-3句，突出品的特点和匹配度）
3. 佣金/利益点（1句，直接但不过分）
4. 行动邀请（1句，降低回应门槛）

示例：
"宝子，看到你上期做的洗发水测评数据很好啊！[破冰]
我这边有个氨基酸洗发水，跟你之前测的不是一个路线，成分很抗打，客单79佣金30% [品介绍+利益点]
感觉挺适合你出个深度测评的，感兴趣的话我寄个样品你试试？[低门槛行动邀请]"
```

### 模板二：跟进话术（三种子场景）

#### 子场景 A：已读不回（24小时后）

```
结构：
1. 不指责（1句）
2. 补信息（1-2句，给个对方可能关心的新信息）
3. 退一步（1句，给对方台阶）

示例：
"没事哈可能比较忙~ 对了这个品最近有3个跟你同量级的达人在测了，数据都不错，想着先给你看一眼免得被抢先了 🤔 你先看着，有想法随时找我"
```

#### 子场景 B：犹豫中（对方说"我再看看"）

```
结构：
1. 理解（1句）
2. 补信心（1-2句，给数据/案例/优势）
3. 降低风险（1句，先寄样不设期限）

示例：
"理解理解，确实要看清楚~ 不过可以跟你透个底，同客单价这个利润确实不好找，而且品牌方这次给的素材包很全 [补信心]
这样，我可以先给你寄个样品，你先用着感受一下，什么时间推你自己定，不设期限 [降风险]"
```

#### 子场景 C：讨价还价（要求更高佣金）

```
结构：
1. 确认底线（1句，明确自己不能给什么）
2. 补其他利益（1-2句，给替代价值）
3. 设截止（1句，制造稀缺感）

示例：
"佣金这块确实已经是上限了，品牌方卡得比较死 [确认底线]
不过我可以额外给你安排品牌方直接对接、首发权益、还有这次活动的独家头图 [补利益]
但我只能保留到今天下午5点，后面还有两个达人在排队了 [设截止]"
```

### 模板三：合作确认话术

```
结构：
1. 庆祝（1句）
2. 同步流程（3-5句，明确接下来的步骤）
3. 素材清单（列表形式）
4. 沟通方式（1句）

示例：
"好嘞，合作愉快！那我同步一下接下来的流程：
1. 今天发样品，顺丰到付，预计2天到你那边
2. 收到后你先试用，有问题随时找我
3. 我把素材包发你，里面包含：主图、视频B-roll、卖点清单、违禁词避雷表
4. 推品时间你来定，提前一天跟我说，我配合做预热和转发素材
咱们先加个微信吧，方便实时沟通 [联系方式]"
```

## 达人匹配度打分卡

在输出达人推荐时，为每个推荐的达人类型附上匹配度打分：

| 评估维度 | 权重 | 说明 |
|---------|------|------|
| 品类匹配 | 30% | 达人近期带货品类与本品匹配度 |
| 受众匹配 | 25% | 达人粉丝画像与本品目标人群重合度 |
| 内容匹配 | 25% | 达人内容风格与本品策略方向的一致性 |
| 性价比 | 20% | 佣金成本与预期转化的投入产出比 |

总分 80+：强烈推荐 | 60-79：推荐 | 40-59：可考虑 | <40：不推荐

## 场景适配

### 场景一：用户已知达人信息

如果用户说"我有个10万粉的穿搭达人，这个洗发水适合她吗"，则跳过达人类型推荐，直接评估匹配度并生成针对性话术。

### 场景二：批量达人管理

如果用户管理多个达人，生成达人管理面板，含每个达人对应的品、沟通阶段、下次跟进时间。

### 场景三：达人资源不足

如果用户表示"找不到合适的达人"，给出达人挖掘渠道建议（精选联盟搜索、竞品评论区、同类达人关注列表等）。
