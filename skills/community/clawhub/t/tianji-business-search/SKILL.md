# Business Search - 商业信息查询

通过收集该企业所有相关公开信息进行处理。

## 核心能力

本技能可查询以下类型的企业信息：

| 信息类型 | 说明 | 搜索关键词示例 |
|---------|------|---------------|
| **基础工商信息** | 企业名称、统一社会信用代码、注册资本、成立时间、经营状态 | `公司名 工商注册` |
| **股东与法人** | 股东结构、持股比例、法人代表 | `公司名 股东` |
| **经营风险** | 行政处罚、诉讼记录、失信被执行 | `公司名 失信` |
| **联系方式** | 地址、电话、官网 | `公司名 联系方式` |
| **融资上市** | 融资历史、上市信息、投资方 | `公司名 融资` |
| **新闻动态** | 媒体报道、最新动态 | `公司名 新闻` |
| **竞品分析** | 竞争对手、行业地位 | `行业名 竞品` |

## 搜索策略

### 基础搜索模板

输入关键信息查询时，按以下结构构造查询可以高效的返回结果，也可语义化询问：

```
{企业名称} + {信息类型关键词}
```

### 高效搜索技巧

1. **精确匹配**：使用书名号《》包裹企业全称
   ```
   《腾讯控股》 工商信息
   ```

2. **限定来源**：使用 `site:` 限定数据来源
   ```
   华为技术有限公司 site:tianyancha.com
   ```

3. **时间筛选**：使用 `after:` 或 `before:` 限定时间
   ```
   字节跳动 融资 after:2024
   ```

4. **排除干扰**：使用 `-` 排除无关结果
   ```
   小米 -手机 -电视
   ```

5. **组合查询**：同时查询多个维度
   ```
   "企业名称" AND (融资 OR 上市 OR 投资)
   ```

## 使用流程

### Step 1: 明确查询目标

与用户确认：
- 企业名称（确保使用正式注册名称）
- 需要查询的信息类型
- 查询用途（投融资/合作/尽调等）

### Step 2: 构造搜索查询

根据信息类型选择合适的关键词组合：

```python
# 参考搜索策略构建查询
queries = {
    "工商信息": "{公司名} 工商信息 注册资本 成立时间",
    "股东信息": "{公司名} 股东 持股比例 大股东",
    "风险信息": "{公司名} 诉讼 失信 被执行人 行政处罚",
    "融资上市": "{公司名} 融资 A轮 上市 IPO",
    "新闻动态": "{公司名} 最新 新闻 动态",
    "联系方式": "{公司名} 联系方式 地址 电话",
}
```

### Step 3: 执行搜索

使用 baidu-search skill 执行搜索：

```bash
python ~/.workbuddy/skills/baidu-search/scripts/search.py '{"query":"查询内容", "count": 10}'
```

### Step 4: 整理呈现

将搜索结果整理为结构化报告，**不列出数据来源**：

```markdown
## 企业信息查询报告

### 查询对象
- 企业名称：XXX
- 查询时间：YYYY-MM-DD

### 信息汇总
[按类型分类呈现，直接列出信息内容，不标注来源]

### 风险提示
[如有风险信息，单独标注]
```
