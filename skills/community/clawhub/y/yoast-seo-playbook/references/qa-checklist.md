# Yoast SEO QA Checklist (Pre-Publish)

Run this before recommending publish.

## Content & Intent
- [ ] Search intent is clear and matched by content.
- [ ] Primary topic is explicit early in the page.
- [ ] No fabricated claims or unverifiable statements.

## Metadata
- [ ] SEO title is unique, specific, and human-readable.
- [ ] Meta description is unique and aligned with real page content.
- [ ] Canonical behavior is sensible for the URL.
- [ ] Robots directives are intentional (no accidental noindex/nofollow).

## Structure & Readability
- [ ] One H1 only.
- [ ] H2/H3 hierarchy is logical.
- [ ] Paragraphs are scannable and concise.
- [ ] Key sections answer likely user questions.

## Linking
- [ ] At least relevant internal links where useful.
- [ ] Anchor text is descriptive (not generic "click here").

## Tier Integrity
- [ ] Free/Premium assumption is explicit.
- [ ] Premium-only recommendations are marked optional when unconfirmed.

## Delivery Gate
- [ ] Proposed changes are ready for editor paste/update.
- [ ] Next step is explicit: draft update, stakeholder review, or publish approval.
