# 胃癌TNM第八版病理分期

## 适用条件
- 患者已通过手术或活检获得明确的胃癌组织学确诊
- 病理报告包含原发肿瘤浸润范围、区域淋巴结转移情况和远处转移评估
- 仅适用于上皮源性胃癌（不适用于GIST、淋巴瘤等非上皮源性胃肿瘤）

## 执行步骤
1. **确定T分期**：根据肿瘤侵犯深度判断：
   - Tx：无法评估
   - T0：无原发肿瘤
   - Tis：原位癌
   - T1a/T1b：侵犯黏膜层/黏膜下层
   - T2：侵犯固有肌层
   - T3：穿透浆膜下层但未达脏层腹膜
   - T4a/T4b：侵犯浆膜或邻近结构

2. **确定N分期**：依据转移的区域淋巴结数量（限胃周23组淋巴结）：
   - N0：无转移
   - N1：1–2个
   - N2：3–6个
   - N3a：7–15个
   - N3b：≥16个

3. **确定M分期**：
   - M0：无远处转移
   - M1：存在远处转移（如肝、肺、骨等）

4. **综合分期**：将T、N、M组合，参照AJCC/UICC官方表格确定最终I–IV期。

> 注意：锁骨上或脐周淋巴结转移视为M1，不属于区域淋巴结。

**区域淋巴结定义**
- 胃周23组淋巴结编号：1–16, 110–112
- 其他部位淋巴结转移归为M1

**常见误区**
- 将腹腔干旁淋巴结（第9组）误判为远处转移（实际属于区域淋巴结）
- 将网膜或肠系膜种植灶误判为T4b（应视为M1）
- 未区分黏膜肌层（T1a）与黏膜下层（T1b）

## 输出格式
返回完整的TNM分期代码（例如：T2N1M0）及对应的临床病理分期（如IIA期）。
