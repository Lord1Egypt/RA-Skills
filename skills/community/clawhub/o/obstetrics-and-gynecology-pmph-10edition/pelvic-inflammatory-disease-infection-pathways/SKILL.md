# 盆腔炎性疾病三大感染途径分析

## 适用条件
- 患者为女性，出现下腹痛、异常阴道分泌物或发热等症状
- 存在性活跃史、近期宫腔操作（如人工流产、放置宫内节育器）或产褥期状态
- 怀疑由淋病奈瑟球菌、沙眼衣原体、结核分枝杆菌等病原体引起的感染性盆腔炎症

## 执行步骤

### 1. 判断是否符合触发条件
确认患者满足以下任一组合：
- 症状（下腹痛/异常分泌物/发热） + 性活跃史
- 症状 + 近期宫腔操作或产褥期

若不符合，不适用本技能。

### 2. 收集关键变量
- **pathway_type**（感染途径类型）：初步推测可能的传播方式
- **pathogen**（病原体种类）：根据流行病学、实验室检查或经验判断
- **clinical_context**（临床背景）：如“产褥期”、“流产后”、“性活跃期”等

### 3. 匹配感染途径
根据收集的信息，按以下逻辑匹配最可能的感染途径：

#### (1) 沿生殖道黏膜上行蔓延
- **典型场景**：非妊娠期、非产褥期的性传播感染
- **常见病原体**：淋病奈瑟球菌、沙眼衣原体、葡萄球菌
- **扩散路径**：外阴/阴道 → 宫颈黏膜 → 子宫内膜 → 输卵管 → 卵巢/腹腔

#### (2) 经淋巴系统蔓延
- **典型场景**：产褥感染、流产后感染、宫内节育器放置后感染
- **常见病原体**：链球菌、大肠埃希菌、厌氧菌
- **扩散路径**：外阴/阴道/宫颈创伤处 → 淋巴管 → 盆腔结缔组织及内生殖器

#### (3) 经血液循环传播
- **典型场景**：全身性感染（如肺结核）后播散至生殖系统
- **常见病原体**：结核分枝杆菌
- **累及顺序**：输卵管（最常见）→ 子宫内膜 → 卵巢 →（较少）宫颈/阴道/外阴

### 4. 输出结论
综合判断最可能的感染途径，并据此建议：
- 针对性病原体检测（如核酸扩增试验、结核菌素试验）
- 抗生素选择（覆盖相应病原体谱）
- 是否需排查其他系统感染源（如肺部）

> 注意：本技能仅适用于感染性盆腔炎，不适用于非感染性炎症或男性生殖系统疾病。
