# 微信小程序 SDK 代码示例索引

## 初始化

| 接口 | 说明 |
|------|------|
| `new SDK(config)` | 构造并初始化 SDK 实例 |

## 用户标识

| 接口 | 说明 |
|------|------|
| `sdk.setOpenId(openid)` | 设置微信 OpenId（优先使用） |
| `sdk.setUnionId(unionid)` | 设置微信 UnionId |

## 通用上报

| 接口 | 说明 |
|------|------|
| `sdk.track(action_type, action_param?)` | 通用行为上报 |

## 专用上报

onAppStart / onAppQuit / onEnterForeground / onEnterBackground / onPageShow / onPageHide

## 全局配置

| 接口 | 说明 |
|------|------|
| `SDK.setDebug(true/false)` | 调试模式 |
| `SDK.setRequestConcurrency(n)` | 并发数（最小 1，默认 4） |

## 自动采集事件

TICKET / START_APP / ENTER_FOREGROUND / ENTER_BACKGROUND / PAGE_VIEW / PAGE_LEAVE / SHARE / START_PAY / FINISH_PAY / ADD_TO_WISHLIST
