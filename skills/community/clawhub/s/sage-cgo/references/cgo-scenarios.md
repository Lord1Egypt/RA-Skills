# CGO 场景剧本

当用户的问题落入具体增长场景时，加载本文件。所有判断先回到 `cgo-identity.md`：PMF 优先、留存优先、北极星优先、实验优先、单位经济学优先。

## 1. 增长停滞

1. 用 AARRR 定位最薄弱环节。
2. 检查 PMF：非常失望比例、留存曲线、付费/推荐。
3. 判断是产品、渠道、内容、信任、转化还是承接问题。
4. 找一个最高 ICE 分实验，而不是同时乱试。

## 2. 渠道 / 平台选择

1. ICP 在哪里？
2. 产品是否天然适配这个渠道？
3. 商业模型是否支撑渠道成本？
4. 团队内容能力是否匹配？
5. 30 天如何判断主战场是否成立？

输出：主渠道、备选渠道、明确不做的渠道。

## 3. 内容没人看

先拆：

- 选题是否命中痛点？
- 标题/封面/前 3 秒是否有钩子？
- 内容结构是否留得住？
- 账号标签是否垂直？
- 是否触达 ICP，而非泛流量？

不要直接说“多发”。

## 4. 内容矩阵

建立：

- 3-5 个内容支柱。
- 每个支柱 2-3 个模板。
- 引流 / 信任 / 转化配比。
- 每周节奏和复盘指标。

## 5. 投流

投流前检查：

1. 自然内容是否已有正反馈？
2. Aha Moment 或转化路径是否清楚？
3. LTV 是否支撑 CAC？
4. 落地页或私信承接是否准备好？
5. 止损线是多少？

自然内容和转化路径没跑通，不建议大额投流。

## 6. 爆款归因

1. 是选题、标题、封面、结构、热点还是平台机制？
2. 是目标受众喜欢，还是泛流量误伤？
3. 评论区暴露了什么需求？
4. 哪个变量可复制？
5. 下周如何设计 3 条变体？

不要把运气当能力。

## 7. 变现路径

先回答：

- 谁为什么付钱？
- 从看到内容到付费经历哪些步骤？
- 当前缺的是信任、产品、报价还是承接？
- 变现动作会不会伤害内容信任？

输出内容到收入的路径图和第一个转化动作。

## 8. ICP 定义

ICP 必须可操作，不是画像作文。

包含：

- 公司/用户特征
- 触发痛点
- 决策者和使用者
- 预算或付费能力
- 最佳客户的共同特征
- 不服务谁

## 9. Aha Moment / 激活优化

1. 最活跃用户前 7 天做了什么共同动作？
2. 哪个行为最能预测长期留存？
3. 新用户到这个行为的路径有多短？
4. 哪些步骤可以删掉？
5. 如何召回未激活用户？

## 10. 私域承接

1. 用户从哪个内容进入私域？
2. 第一触点承诺是什么？
3. 前 24 小时提供什么价值？
4. 如何分层：新粉、意向、成交、复购？
5. 哪个动作推动成交而不伤害信任？

## 11. GTM

回答四个问题：

1. 进入哪个市场？
2. 目标客户是谁？
3. 为什么选择我们？
4. 如何触达并转化？

输出：ICP、差异化信息、渠道、销售/转化路径、指标。

## 12. 增长复盘

每个实验结束后：

1. 原始假设是什么？
2. 实际指标如何？
3. 学到了什么？
4. 是放大、调整、停止，还是换假设？
5. 写入 `growth/experiments.md` 和 `growth/metrics.md`。

## 13. 社交媒体 Agent / 内容 Agent

先定义增长任务，而不是只写人格。

结构：

- 平台角色
- 北极星指标
- ICP / 受众
- 内容支柱
- 平台规则
- 转化动作
- 复盘字段
- 禁止行为
