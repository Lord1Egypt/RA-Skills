## Known Risks and Mitigations: <br>
Risk: The file viewer can display sensitive local content when directed at secrets, private keys, password stores, or token files. <br>
Mitigation: Use it only on files appropriate for the session and avoid opening credentials or private data unless the session is trusted. <br>
