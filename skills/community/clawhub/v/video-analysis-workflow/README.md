# 视频案例分析助手 🎬

> Video Analysis Workflow · 一键拆解参考视频 · 自动生成分镜/台词/结构分析 · 帮你沉淀长期可复用的视频案例库

---

## 📌 Skill 介绍

**视频案例分析助手** 可以把你的 📽️ **本地视频素材**，或者 🔗 **抖音 / B站在线视频**，自动整理成一套可复用的视频案例资产。

它会帮你回答：

```text
分镜怎么排？
节奏怎么走？
台词怎么写？
结构怎么搭？
亮点怎么复用？
怎么套用到自己的项目？
```

最终输出四份核心报告：

- **案例分析**
- **抽帧分镜**
- **台词转写**
- **脚本模版**

支持输出 **Markdown** 或按需生成 **HTML**，并可按照统一规范沉淀到 Obsidian 案例库，适合长期积累团队创意资产。

---

## 💡 功能特性

- 🎞️ **全流程视频拆解**：提取分镜、画面节奏、旁白台词、叙事结构、脚本句式。
- 🔗 **在线视频链路支持**：支持本地视频，也支持抖音 / B站链接先下载再分析。
- 🖼️ **自动生成时间轴分镜**：直观看画面设计，可作为 Prompt 生图和视频策划参考。
- 🎙️ **本地转写字幕与旁白**：通过 Whisper 提取视频台词、字幕和时间码。
- 📚 **自动生成案例报告**：沉淀 `案例分析`、`抽帧分镜`、`台词转写`、`脚本模版` 四类文档。
- 🧩 **生成视频模板 + Prompt**：把案例转成可复用的脚本、镜头语言、节奏公式和 AI Prompt。
- 🗂️ **打造团队视频案例库**：按照“一个视频 = 一个文件夹”的规范归档，方便检索和调用。
- 📝 **输出 Markdown / HTML 报告**：默认输出 Obsidian 友好的 Markdown；需要分享时可额外生成 HTML。

## ⚠️ 实测踩坑与处理规则

### 抖音短链不一定能直接下载

抖音短链可能无法直接被下载器解析。推荐策略：

~~~text
短链下载失败
→ 解析真实视频页
→ 提取 video_id
→ 用 douyin-video-fetch 下载 video_id
~~~

### 下载器推荐顺序

1. `douyin-video-fetch`：抖音优先，video_id 下载更稳。
2. `yt-dlp`：通用兜底，B站优先；抖音可能需要 cookies。
3. 本地视频：如果下载链路失败，用户可以直接提供视频文件继续分析。

### cookies 授权

如果下载器提示需要 cookies，助手会先询问：

> 是否允许读取浏览器 cookies 重试下载？

没有授权时，不会读取浏览器 cookies。

### 没有旁白也能分析

有些活动视频主要靠大字卡和发布文案传递信息。Whisper 可能只识别到片尾署名或空文本。这种情况下，分析会改用画面字卡、发布文案、奖励/道具文字和抽帧画面信息。

### 中文路径写入规范

案例库通常包含中文路径和中文文件名。为避免乱码，写入 Markdown 时必须使用稳定 UTF-8 写入方式，不使用容易丢编码的 Python 管道写入。
