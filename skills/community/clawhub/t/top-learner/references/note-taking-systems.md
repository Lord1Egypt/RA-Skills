# 笔记与阅读系统

## 系统选择指南

| 场景 | 推荐系统 |
|------|---------|
| 听课/讲座 | 康奈尔笔记 |
| 阅读教材/论文 | SQ3R / PQ4R |
| 长期知识库 | Zettelkasten |
| 快速浏览网页 | 渐进式摘要 |
| 复习前整合 | 思维导图 |
| 写作/报告准备 | 大纲法 |

## 核心要点

- **康奈尔笔记**：页面三分（左侧关键词/问题栏、右侧笔记栏、底部总结栏）。流程：记录→简化→背诵→思考→复习。
- **SQ3R**：浏览(Survey)→提问(Question)→阅读(Read)→复述(Recite)→复习(Review)。Survey建立认知框架使后续效率提升40%。
- **PQ4R**：SQ3R + Reflect（这与我已知什么相关？有何矛盾？）。Reflect触发精细编码。
- **渐进式摘要**：L1抓取→L2加粗→L3高亮→L4摘要→L5创见。按需提炼，高频笔记自然爬升。
- **Zettelkasten**：fleeting（临时）→literature（阅读理解）→permanent（原子化+上下文自足+双向链接）。用自己的话，绝不复制。
- **思维导图**：中心图像、曲线分支、单关键词、颜色区分。适合发散联想和复习盲区检测。
- **大纲法**：层级≤4层，平行原则，MECE（不重叠不遗漏）。
