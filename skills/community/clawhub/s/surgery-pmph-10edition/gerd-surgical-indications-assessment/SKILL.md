# GERD手术适应证评估

## 何时使用
在以下任一情况出现时使用本技能：
- 患者已完成规范内科治疗（如质子泵抑制剂），但症状或并发症持续存在；
- 患者虽药物控制有效，但因生活质量、经济负担或不愿终身服药而主动考虑手术；
- 存在重度反流性食管炎、Barrett食管或食管炎性狭窄等严重并发症；
- 合并明确为反流或梗阻病因的食管裂孔疝；
- 出现慢性咳嗽、哮喘、反流性咽喉炎、喉咙痉挛或肺纤维化等慢性食管外表现。

## 如何执行
1. **确认适用对象**：仅适用于已确诊GERD的成年患者，且已完成充分药物试验。不适用于未经药物治疗的初诊患者或未谨慎评估的儿童。
2. **评估手术适应证**：检查是否满足以下任一条件：
   - **药物治疗失败**：症状控制不足、抑酸药无法控制反流，或因副作用无法继续用药；
   - **患者主动选择手术**：对当前生活质量不满意、不愿终身服药，或药物经济负担过重；
   - **存在严重并发症**：如重度反流性食管炎、Barrett食管、食管炎性狭窄；
   - **合并食管裂孔疝**：疝被确认为反流或急性梗阻的病因；
   - **慢性食管外表现**：包括慢性咳嗽、哮喘、反流性咽喉炎、喉咙痉挛、肺纤维化。
3. **推荐处理**：若满足任一适应证，建议转诊至外科评估腹腔镜抗反流手术（首选腹腔镜食管裂孔疝修补术加胃底折叠术）的可行性。
4. **说明手术目的**：缓解症状、提高生活质量、治愈或预防并发症、摆脱长期用药依赖。

> 注意：约30%~35%的GERD患者因药物控制不佳、并发症或合并食管裂孔疝而需考虑手术。
