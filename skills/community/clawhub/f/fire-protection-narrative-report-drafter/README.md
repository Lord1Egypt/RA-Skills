# Fire Protection Engineering Narrative Report Drafter

**Platforms:** Claude · Openclaw · Codex
**Domain:** Fire Protection Engineering / Permit Submittal / NFPA / IBC

## Purpose

Converts a project's jurisdiction and adopted code editions, AHJ identification, occupancy classification (IBC Chapter 3), construction type (IBC Chapter 6), hazard classification (NFPA 13 / 30 / 30B / 33 / 34 / 45 / 72 / 101 / 484 / 5000), floor area / building height / fire-area takeoffs, water-supply data (most recent AHJ-stamped flow test), system selections (NFPA 13 / 13R / 13D sprinkler; NFPA 14 standpipe; NFPA 20 fire pump; NFPA 22 water tank; NFPA 24 underground; NFPA 72 fire alarm + mass notification; NFPA 25 ITM; NFPA 80 / 105 opening protectives; NFPA 92 smoke control; NFPA 110 / 111 emergency power; NFPA 101 / IBC Ch. 10 means of egress), and AHJ-specific overlay (state and city amendments, deferred-submittal posture) into a DRAFT Fire Protection Engineering narrative report for permit submittal — with hydraulic-calculation summary, water-supply flow-test evidence, seismic design category, monitoring arrangement, AMME (alternative materials, design, and methods of construction) requests, equivalency packets per IBC §104 / NFPA 1 §1.4, deferred-submittals list, ITM plan, an unsigned PE-of-record sign-off block, an open-questions list, and an evidence index — for the FPE, AHJ plans-examiner, owner, design team, and contractor review.

## When to Use

- A Fire Protection Engineer (PE / FPE; SFPE-member) needs a narrative report for permit submittal of a new building, addition, alteration, change-of-occupancy, or tenant fit-out
- A NICET-III / NICET-IV designer or contractor-side engineering manager needs the narrative to accompany sprinkler / standpipe / alarm shop drawings (where the AHJ requires a PE-sealed narrative rather than a designer-of-record certificate)
- A code consultant needs the narrative to anchor an IBC §104 / NFPA 1 §1.4 equivalency request or AMME submittal
- A high-rise, atrium, mall, ambulatory-care, I-1 / I-2 / I-3 institutional, R-1 / R-2 residential, or battery-energy-storage-system (BESS, NFPA 855) project needs a coordinated narrative across sprinkler, standpipe, alarm, smoke-control, and emergency-power disciplines
- A plan-review-comment-cycle response is due and the FPE needs the narrative restructured to address AHJ comments without scope creep
- An owner / GC needs a deferred-submittals list and ITM plan ahead of certificate-of-occupancy

## What It Does

**Phase 1: Project Intake and Code-Edition Lock-in**
1. Captures the project (address, project ID, owner, GC, FPE firm and PE-of-record name + state license #), the **AHJ** (city fire-prevention bureau, state fire marshal, county, port authority, federal occupant, healthcare licensing), the **adopted code editions** (IBC, IFC, NFPA 1, NFPA 101, NFPA 13, NFPA 14, NFPA 20, NFPA 22, NFPA 24, NFPA 25, NFPA 72, NFPA 80, NFPA 92, NFPA 96, NFPA 110, NFPA 111, NFPA 855, others as applicable — including state amendments and city amendments), and the deferred-submittal posture (which packets are deferred per IBC §107.3.4.1)
2. Confirms the project posture (new building, addition, alteration, change of occupancy, tenant fit-out, retroactive compliance) and submittal type (full permit / phased / foundation-only / shell / fit-out / deferred)

**Phase 2: Occupancy / Construction / Hazard Triad and Allowable Area / Height**
3. Locks in occupancy classification per IBC Chapter 3 (A-1 through A-5; B; E; F-1 / F-2; H-1 through H-5 with control-area analysis; I-1 / I-2 / I-3 / I-4; M; R-1 / R-2 / R-3 / R-4; S-1 / S-2; U; with mixed-use accessory / non-separated / separated determination per IBC §508), construction type per IBC Chapter 6 (I-A / I-B / II-A / II-B / III-A / III-B / IV-A / IV-B / IV-C / IV-HT / V-A / V-B; with the new mass-timber Types IV-A / IV-B / IV-C where adopted), and hazard classification per the discipline standard (NFPA 13 Light / Ordinary 1 / Ordinary 2 / Extra 1 / Extra 2 / Storage; NFPA 30 flammable-liquid Class IA / IB / IC / II / IIIA / IIIB; NFPA 30B aerosols; NFPA 33 / 34 spraying; NFPA 45 lab unit Class A / B / C; NFPA 484 combustible metals; NFPA 855 BESS UL 9540A test data; NFPA 96 commercial cooking)
4. Runs allowable-area / height / story analysis per IBC §504 / §506 (frontage increase, sprinkler increase, story-and-height tables), confirms fire-area boundaries per IBC §707 / NFPA 1 / NFPA 101, and confirms separation between mixed occupancies (separated vs non-separated; rated assembly hours)

**Phase 3: Water Supply Evidence and Hydraulic Demand**
5. Captures the most recent AHJ-stamped flow test (date, conducted-by entity and certification, hydrant locations static / residual / flow, pitot pressure, residual pressure, flow GPM, derated curve), with the date and the validity per the AHJ rule (often within 12 months of submittal) and the seasonal / time-of-day note where the AHJ requires
6. Summarizes the hydraulic demand for each system: sprinkler demand (density × area-of-application + hose stream, with the most demanding remote area calculation), standpipe demand per NFPA 14 (Class I / II / III, with 100 psi at the topmost outlet and the demand-curve overlay onto the water-supply curve), fire pump per NFPA 20 (rated flow, churn pressure, 150% flow capacity, pressure-relief / pressure-maintenance arrangement), and the combined demand vs available water supply (graphical and numerical, with a safety-margin statement)

**Phase 4: System Narrative per Discipline**
7. Drafts the per-discipline narrative — automatic sprinkler (NFPA 13 / 13R / 13D — applicability per IBC §903.2, including the 2026 NFPA 25 proposed updates the user must verify against the adopted edition; CPVC and steel piping discipline; seismic bracing per NFPA 13 Ch. 18 and the project's Seismic Design Category; freezing-protection methodology; in-rack and ESFR storage applicability; antifreeze restrictions), standpipe (NFPA 14 — Class I / II / III, pressure-regulating valve discipline, hose-valve locations, manual / wet / dry distinction, IBC §905 interior-exit-stairway hose connection from fire-service-access elevator lobbies), fire pump (NFPA 20 — listed components, controllers, alternate power per IBC §913.2, jockey pump, pressure relief), water storage (NFPA 22 — capacity, refill, freeze protection), underground (NFPA 24 — thrust blocks / restraint, depth of bury, post-indicator valves, FDC location), fire alarm and mass notification (NFPA 72 — initiation, notification, occupant notification per Annex A of the adopted edition, voice-evacuation in A-2 / A-3 / B / E / M / R-1 / R-2 occupancies, mass-notification system per NFPA 72 Ch. 24 where required, off-premises supervising-station signal path), smoke control (NFPA 92 — atria, high-rise, mall, ambulatory-care, smokeproof enclosures), emergency power (NFPA 110 / 111 — Level 1 / 2, Type / Class), opening protectives (NFPA 80 / 105 — fire-rated assemblies, smoke-and-draft control), commercial cooking (NFPA 96 — Type I hood, listed extinguishing system, gas-shutoff, ventilation), special hazards (clean-agent NFPA 2001, gaseous oxidant NFPA 67, BESS NFPA 855 with UL 9540A and explosion-control), means of egress (IBC Ch. 10 / NFPA 101 — number of exits, common path of egress travel, exit access travel distance, dead-end corridor, capacity, doors, panic / fire-exit hardware), and accessibility (ICC A117.1 areas of refuge / two-way communication where required)

**Phase 5: ITM, AMME, Equivalency, and Deferred Submittals**
8. Drafts the ITM plan (NFPA 25 for water-based; NFPA 72 Ch. 14 for alarm; NFPA 96 for kitchen; NFPA 12 / 12A / 17 / 17A / 2001 for special hazards), the AMME / equivalency packet (IBC §104 / NFPA 1 §1.4) where a code path is not strictly compliant and an alternative is proposed with equivalent or greater protection, and the deferred-submittals list (per IBC §107.3.4.1) with a stated submittal-by-trade owner and target date

**Phase 6: AHJ-Comment Anticipation, Assembly, Sign-off, Open Questions**
9. Self-checks against common AHJ plan-review comments (water-supply data older than the AHJ-permitted window; missing seismic Cp / Sds; FDC siamese location not on the address-frontage side; fire-department-connection signage; high-rise stair-pressurization arrangement; smoke-control rational-analysis report; alarm voltage-drop / battery-calc; mass-notification interface; backup-power transfer-time per the occupancy; BESS deflagration-vent calculation; specialty / R-2 attic-sprinklers per local amendment)
10. Assembles the narrative with the executive summary, code-edition lock-in, occupancy / construction / hazard triad and allowable-area analysis, water-supply summary, per-discipline narrative, ITM plan, AMME / equivalency / deferred-submittal packets, references-and-standards table, an unsigned PE-of-record sign-off block (with the PE seal slot left blank for wet/digital signature), an open-questions list, and an evidence index that cross-references the flow-test certificate, hydraulic calculations, alarm voltage-drop / battery-calc, smoke-control rational analysis, BESS UL 9540A test report, and prior-permit references

## Output

A draft fire-protection-narrative packet consisting of (a) an executive summary; (b) the code-edition lock-in section; (c) the occupancy / construction / hazard / allowable-area analysis; (d) the water-supply summary with flow-test evidence; (e) the per-discipline narrative (sprinkler, standpipe, fire pump, water storage, underground, alarm + mass notification, smoke control, emergency power, opening protectives, commercial cooking, special hazards, means of egress, accessibility); (f) the ITM plan; (g) AMME / equivalency packets; (h) the deferred-submittals list; (i) the references-and-standards table; (j) an unsigned PE-of-record sign-off block; (k) a numbered evidence index; (l) an open-questions list — all marked **DRAFT — FOR FPE, AHJ PLANS-EXAMINER, OWNER, DESIGN-TEAM, AND CONTRACTOR REVIEW**.

## Notes

This skill **drafts** a Fire Protection Engineering narrative report to support — never replace — the licensed Professional Engineer with the Fire Protection branch designation (PE / FPE) who will stamp and seal the report under their professional responsibility. The skill does not stamp or seal, does not perform the hydraulic calculation (only summarizes the calculation submitted with the package), does not certify compliance, does not represent the AHJ, does not waive any AHJ-amendment or condition, does not concede or waive a code-path option on behalf of the owner / design team / contractor, and does not opine on the merits of an AHJ comment beyond restructuring the narrative to address the comment. The skill follows the **adopted** code editions in the project's jurisdiction — the user must confirm those editions (state amendments, city amendments) before drafting; the skill will flag and stop where the user cannot confirm.
