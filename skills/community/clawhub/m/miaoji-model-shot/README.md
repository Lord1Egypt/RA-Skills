## 📦 核心功能

### 1️⃣ 模特展示提示词生成器
- 输入服装品类 + 款式特征 + 目标风格
- 输出专业级AI绘图提示词（可直接用于Midjourney/Stable Diffusion）

### 2️⃣ 场景适配模板
- **日常休闲风** - 城市街景、咖啡厅、公园
- **商务通勤风** - 写字楼、会议室、城市建筑
- **运动户外风** - 健身房、跑道、山野、沙滩
- **约会浪漫风** - 海边、日落、餐厅、花园

### 3️⃣ 多角度构图方案
单款产品输出 **4-6张** 完整方案：
| 序号 | 角度 | 用途 |
|-----|------|------|
| 1 | 正面全身 | 主图/详情页 |
| 2 | 侧面全身 | 详情页/搭配 |
| 3 | 背面全身 | 展示背部设计 |
| 4 | 场景中景 | 社交媒体 |
| 5 | 细节特写 | 详情页卖点 |
| 6 | 搭配展示 | 穿搭推荐 |

### 4️⃣ 测款加速规划
- 单次规划：1-5款精细方案
- 快速测款：6-20款批量方案
- 批量上新：21-100款模板化方案

## 🚀 快速开始

### 最小输入示例
```
品类：女士连衣裙
风格：约会浪漫
```

### 完整输入示例
```
品类：女士连衣裙
款式：碎花吊带，收腰设计
材质：雪纺面料，轻薄透气
颜色：浅粉色底+白色碎花
风格：约会浪漫
目标场景：海边日落
```

## 📖 使用指南

### 输入规范

| 参数 | 必填 | 说明 |
|-----|-----|------|
| 品类 | ✅ | 如：女士T恤、男士衬衫、童装等 |
| 款式 | ○ | 如：圆领、修身版、宽松等 |
| 材质 | ○ | 如：纯棉、丝绸、牛仔等 |
| 颜色 | ○ | 如：深蓝色、黑白条纹等 |
| 风格 | ✅ | 日常休闲/商务通勤/运动户外/约会浪漫 |
| 场景 | ○ | 指定具体场景需求 |

### 输出内容

1. **AI绘图提示词** - 可直接用于MJ/SD等AI绘图工具
2. **完整拍摄方案** - 包含场景、光线、构图、道具清单
3. **批次规划表** - （如需批量规划）

## ⚠️ 重要说明

> **定位声明**：本技能专注于**产品展示方案设计**，输出内容可作为拍摄指导或AI绘图参考提示词。
> 
> **核心不是"生成模特图"**，而是**"秒级生成模特展示构图方案+AI绘图提示词"**。
