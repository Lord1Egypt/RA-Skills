# Touchpoint Metrics Reference

KPIs per journey stage, benchmark conversion rates, channel-specific metrics, attribution models, and measurement tools for ecommerce journey mapping.

---

## KPIs by Journey Stage

### Awareness
| Metric | Description |
|---|---|
| Impressions / Reach | Total and unique exposures |
| Brand search volume | Searches for your brand name over time |
| Click-through rate (CTR) | Clicks / impressions |
| Cost per mille (CPM) | Cost per 1,000 impressions |
| New visitor rate | First-time sessions as % of total |
| Video view rate | % watched to 25/50/75/100% |

### Consideration
| Metric | Description |
|---|---|
| Pages per session | Depth of brand exploration |
| Session duration | Time spent on site |
| Product page view rate | % of visitors viewing a product page |
| Email signup rate | Lead capture efficiency |
| Return visitor rate | Returning sessions as % of total |
| Wishlist/save rate | High-intent signal |

### Evaluation
| Metric | Description |
|---|---|
| Review read rate | % who scroll to or click reviews |
| Average star rating | Mean review score |
| FAQ/help page visit rate | Pre-purchase info seeking |
| Pre-purchase support contacts | Chat, email, or phone before buying |
| Exit rate from product pages | Leaving the site from PDP |

### Purchase
| Metric | Description |
|---|---|
| Add-to-cart rate | Sessions with ATC / total sessions |
| Cart abandonment rate | Carts not purchased / total carts |
| Checkout completion rate | Purchases / checkout initiations |
| Average order value (AOV) | Revenue / orders |
| Payment failure rate | Failed payments / total attempts |
| Guest checkout rate | Orders without account creation |

### Onboarding
| Metric | Description |
|---|---|
| Average delivery time | Days from order to delivery |
| On-time delivery rate | Delivered by promised date |
| Welcome email sequence completion | % opening all emails in flow |
| Support tickets within 7 days | Post-delivery friction indicator |
| Return rate within 30 days | Product-expectation mismatch |

### Retention
| Metric | Description |
|---|---|
| Repeat purchase rate | Customers who buy again / total |
| Repurchase cycle (days) | Average time between purchases |
| Customer lifetime value (CLV) | Total revenue per customer |
| Loyalty program enrollment/active rate | Adoption and engagement |
| Subscription renewal rate | Recurring revenue health |
| Win-back campaign conversion | Lapsed customers reactivated |

### Advocacy
| Metric | Description |
|---|---|
| Review submission rate | Reviews / review requests |
| Net Promoter Score (NPS) | Promoters minus detractors |
| Referral program participation | % sharing referral codes |
| UGC volume (monthly) | Customer posts featuring brand |
| Social media mention volume | Brand mentions across platforms |

## Attribution Models

| Model | How It Works | Best For | Limitation |
|---|---|---|---|
| Last click | 100% to last touch | Bottom-funnel optimization | Ignores awareness |
| First click | 100% to first touch | Awareness evaluation | Ignores conversion |
| Linear | Equal credit to all | Balanced channel view | Oversimplifies |
| Time decay | More credit near purchase | Short purchase cycles | Undervalues awareness |
| Position-based | 40/20/40 first/middle/last | Balanced discovery + conversion | Arbitrary weighting |
| Data-driven | Algorithm-assigned credit | Mature businesses with data | Needs large volume |

**Recommended approach for journey mapping:**
- Awareness analysis: use first-click attribution
- Purchase analysis: use last-click attribution
- Full-journey analysis: use position-based or data-driven
- Investment decisions: use incrementality testing when possible

**Key challenges:** Cross-device tracking gaps, Amazon attribution limitations, iOS privacy restrictions reducing Meta tracking, view-through attribution debates, influencer attribution difficulty (use unique codes/landing pages).
