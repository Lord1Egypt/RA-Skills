# 渣谱系库（11 款）

每款：行为模式 → 话术指纹 → 危险等级。匹配时看**反复出现的模式**，不凭单条定罪。

## 🌊 海王 / 海后（广撒网，雨露均沾）
- **行为模式**：同时吊着多人，给所有人发相似的暧昧，但绝不给任何人确定关系。
- **话术指纹**："宝""在干嘛呀"群发式开场 · 称呼很泛（叫"你"而非你名字）· 约会总在 ta 方便的碎片时间 · 朋友圈从不公开你 · "我们慢慢来""先这样处着看看" · 节日/深夜突然冒泡（鱼塘巡逻）。
- **危险等级**：🔴

## 🍵 绿茶 / 抹茶（装清纯，带节奏拉踩）
- **行为模式**：用"无辜/可怜/我什么都不懂"获取关注，暗中制造竞争、引人争抢。
- **话术指纹**："哥哥你好厉害教教我嘛" · "是不是我不够好呀"（钓安慰）· "他们都对我好好，可我只想要你懂我"（暗示有备胎 + 抬你）· 当众示弱私下另一套 · 总有"别的男生/女生"在帮 ta。
- **危险等级**：🔴

## 🎭 PUA / 煤气灯（打压你自尊，精神操控）
- **行为模式**：先捧后贬，让你自我怀疑、离不开 ta。**最危险的一款。**
- **话术指纹**："也就我能受得了你这脾气" · "你太敏感了/你想多了"（否定你的感受）· "为我你都不愿意？" · 忽冷忽热制造焦虑 · 孤立你（"你那些朋友没一个好东西"）· 把错全归你。
- **危险等级**：🔴🔴（涉及精神操控，报告里要建议远离 + 找现实支持）

## 🎣 回避型钓鱼（若即若离，吊着你）
- **行为模式**：你热他冷、你退他追，永远保持"够不着但不放手"的距离。含 ghosting（突然消失再突然回来）、藕断丝连前任纠缠。

- **危险等级**：🟡→🔴（看是真回避依恋还是装的；长期消耗即避雷）

## 🍬 玩玩型（只要暧昧不要承诺）

## 🟢 绿灯对照：这些不是渣（防误伤，判定前必须排除）

- **慢热但稳定**：回复慢但有来有回、说到做到、进度慢但方向一致 → 不是渣，是性格。
- **直男钝感**：不解风情但真诚、不撒谎、愿意改 → 钝 ≠ 渣。
- **社恐 / i 人**：线上少话但线下用心、愿意见你的圈子 → 真心的笨拙。

**判定铁律**：渣的核心是 **"言行不一 + 不给确定性 + 消耗你"**。缺这个内核的，再不完美也只是性格差异，只给🟡不扣🔴。

## ⚪ 不在本库范围（避免误伤与跑偏）

凤凰男 / 妈宝 / 扶弟魔等属于**价值观与家庭议题**，对方可能很真心，不等于"渣"。本工具不据此扣🔴，至多在报告里提一句"这更像价值观差异，另当别论"。
