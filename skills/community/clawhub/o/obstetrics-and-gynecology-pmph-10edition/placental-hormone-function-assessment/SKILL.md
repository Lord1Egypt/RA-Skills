# 胎盘激素功能评估与临床解读

## 何时使用本技能
- 需要解释妊娠期激素水平动态变化（如hCG峰值时间、雌三醇显著升高）；
- 产前筛查显示hCG或雌三醇异常，需判断是否提示胎盘功能障碍；
- 临床怀疑胎盘功能不良（如死胎、子痫前期、胎儿生长受限/FGR），需关联激素水平变化；
- 分析妊娠期母体代谢或免疫适应机制（如葡萄糖分配、免疫耐受）。

## 执行步骤

### 1. 评估人绒毛膜促性腺激素（hCG）
- **检测时机**：着床后1日可测，妊娠8～10周达峰，产后2周消失。
- **核心功能**：
  - 维持妊娠黄体，促进甾体激素分泌；
  - 促进雄激素芳香化为雌激素，刺激孕酮形成；
  - 抑制母体淋巴细胞对胚胎的免疫攻击；
  - 刺激男胎睾丸分泌睾酮，促进性分化；
  - 激活母体甲状腺（通过TSH受体交叉反应）。
- **临床提示**：hCG异常升高或延迟下降可能提示滋养细胞疾病或多胎妊娠；显著降低可能反映胎盘功能不良。

### 2. 评估人胎盘催乳素（hPL）
- **检测时机**：妊娠5周可测，持续上升至分娩，产后7小时内消失。
- **核心功能**：
  - 促进乳腺发育及乳蛋白合成；
  - 增强胰岛素生成；
  - 促进脂解，提高游离脂肪酸，抑制母体葡萄糖摄取，保障胎儿能量供应；
  - 抑制母体排斥反应。
- **临床提示**：hPL水平低下常提示胎盘功能减退。

### 3. 分析雌激素动态
- **来源转换**：妊娠10周前由卵巢黄体产生，之后依赖胎儿-胎盘单位。
- **关键数据**：妊娠末期雌三醇达非孕期1000倍，雌二醇/雌酮达100倍。
- **合成路径**：母体胆固醇 → 胎盘孕烯醇酮 → 胎儿肾上腺DHEAS → 胎儿肝16α-羟化 → 胎盘去硫酸根 → 芳香化 → 雌三醇。
- **临床提示**：雌三醇显著下降是胎儿-胎盘单位功能障碍的重要标志。

### 4. 评估孕激素来源与作用
- **来源转换**：8～10周前由妊娠黄体分泌，之后由胎盘合体滋养细胞主导。
- **功能**：与雌激素协同调节子宫内膜、肌层松弛、乳腺发育及其他系统适应。

### 5. 检查辅助胎盘标志物
- **缩宫素酶**：灭活缩宫素以维持妊娠；胎盘功能不良时水平降低。
- **耐热性碱性磷酸酶（HSAP）**：妊娠16～20周可测，产后3～6日消失；动态监测可用于评价胎盘功能。
- **细胞因子（EGF、IGF、TNF-α、IL-1/2/6/8等）**：参与胎儿营养供给与免疫保护。

## 注意事项
- 所有激素评估必须结合妊娠周数；
- 胎盘功能不良（如子痫前期、FGR、死胎）常导致多种激素同步异常；
- 合体滋养细胞功能正常是激素合成的前提。
