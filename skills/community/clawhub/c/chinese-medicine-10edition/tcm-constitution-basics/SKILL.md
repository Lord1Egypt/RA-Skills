# TCM Constitution Basics

## When to Use
- 需要从中医整体观出发理解个体在形态、功能和心理上的综合特质
- 进行中医健康评估、体质辨识或个性化调养方案制定
- 区分中医体质概念与西医单一指标诊断的适用边界

## Execution Steps

### 1. 确认适用对象
确保对象为：
- 人体个体
- 健康评估对象
- 中医诊疗患者

排除非生命体或仅依赖单一生理指标的西医诊断场景。

### 2. 核验前提条件
确认个体满足以下条件：
- 具有先天禀赋（由父母生殖之精决定的遗传基础）
- 经历后天调养（包括饮食、起居、情志、劳逸、地理等环境与行为因素）
- 处于生长发育或衰老过程

### 3. 应用体质定义
根据以下结构判断是否符合中医体质特征：
- **“体”**：指形体、身体，引申为躯体和生理结构
- **“质”**：指特质、性质
- **核心内涵**：个体在“禀受于先天，调养于后天”基础上，在生长、发育和衰老过程中形成的，在形态结构、生理功能和心理状态三方面综合的、相对稳定的固有特质

### 4. 验证理论基础
确认该体质观体现中医学“形神合一”原则：
- **“形”**：包括脏腑、经络、精气血津液
- **“神”**：包括魂、神、意、魄、志及七情
- 强调形与神的统一性

### 5. 判断健康理想状态
理想体质应满足：
- 充分发挥先天遗传潜力
- 经后天积极培育
- 形、神、环境适应力均处于良好状态，即“形神统一”

## Output
提供判断依据，用于后续体质辨识、养生指导或疾病防治策略制定。
