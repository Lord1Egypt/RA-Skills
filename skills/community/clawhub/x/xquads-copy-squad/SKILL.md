# Copy Squad

Elite squad de 23 agentes de copywriting — 22 copywriters lendários + 1 orquestrador (Copy Chief).

## Como Ativar

```
Use copy-chief como orquestrador para rotear tarefas.
```

## Tasks Disponíveis

- write-headline - Create compelling headlines
- write-sales-letter - Long-form sales copy
- write-vsl-script - Video Sales Letter script
- write-email-sequence - Email sequences
- write-ad-copy - Paid ad copy
- write-landing-page - Landing page copy
- write-bullets - Bullet points/fascinations
- create-offer - Offer architecture
- create-funnel-copy - Funnel copy
- analyze-copy - Analyze existing copy
- critique-copy - Review and improve copy

## Fluxo de Trabalho

1. **Diagnose** - Entender o que você precisa
2. **Assign** - Cop Chief routeia para especialista certo
3. **Write** - Especialista gera o copy
4. **Review** - Quality gate

## Como Usar

Exemplo de uso:

```
"Preciso criar uma headline para um produto de weight loss"
→ Copy Chief routeia para Eugene Schwartz

"Preciso de uma sequência de emails para lançamento"
→ Copy Chief routeia para Andre Chaperon

"Preciso criar uma Grand Slam Offer"
→ Copy Chief routeia para Dan Kennedy
```
