# 炎症介质系统激活与分类

## 适用场景
- 血管内皮损伤（如创伤、感染）
- 急性炎症表现（红、肿、热、痛、功能障碍）
- 需解释白细胞募集或血管通透性增加机制

## 执行步骤

### 一、三大炎症介质系统激活
血管内皮损伤暴露胶原/基底膜 → 激活Ⅺ因子 → 启动：
1. **激肽系统**：生成缓激肽 → 血管扩张、通透性↑、平滑肌收缩、疼痛
2. **补体系统**：
   - 经典/替代/凝集素途径 → C3a/C5a → 血管活性+趋化作用
3. **凝血/纤溶系统**：
   - 凝血酶 → 激活内皮+剪切C5→C5a
   - 纤溶酶 → 降解纤维蛋白+剪切C3→C3a
   - 纤维蛋白多肽 → 趋化+通透性↑

> 三大系统相互放大，协同增强炎症反应。

### 二、急性炎症病理分类
依据渗出物性质分为：
- **浆液性炎**：清亮液体（如皮肤水疱）
- **纤维蛋白性炎**：含大量纤维蛋白（如绒毛心）
- **化脓性炎**：中性粒细胞+脓液（如脓肿）
- **出血性炎**：大量红细胞渗出

> **注意**：本技能仅适用于急性过程（数分钟至数天），不适用于慢性炎症。
