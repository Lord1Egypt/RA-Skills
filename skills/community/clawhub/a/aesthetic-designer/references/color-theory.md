# 配色理论速查

## 配色基本原则

1. **60-30-10法则**：60%主色（背景+大面积）、30%辅色（次要元素）、10%强调色（点缀）
2. **不超过5色**：超出5个即可识别颜色会让视觉疲劳
3. **色调统一**：所有颜色共享一个底调（暖或冷）避免视觉冲突
4. **对比度足够**：文本/关键图形需足够对比（WCAG AA = 4.5:1）

## 跨流派融合配色

**技术**：取流派A的主色 + 流派B的强调色 + 流派C的辅色

**示例：巴洛克 × 赛博朋克**
- 巴洛克主色：深绀 #0D0221
- 赛博朋克强调色：霓虹粉 #FF2D6E
- 巴洛克金色：D4A017
- 赛博朋克电蓝：00E5FF
→ 结果是暗黑华丽未来感
