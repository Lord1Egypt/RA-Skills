# EmblemAI Prompt Examples

Standalone skill for curated non-developer EmblemAI prompts and phrasing guidance across wallet operations, portfolio review, market research, trading, transfers, cross-chain orders, Bitcoin assets, NFTs, prediction markets, Emblem Vault actions, and assistant workflows.

## Install

```bash
npx skills add EmblemCompany/Agent-skills --skill emblem-ai-prompt-examples
```
