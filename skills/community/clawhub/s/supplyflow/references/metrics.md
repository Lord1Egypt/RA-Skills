# 供应链指标与公式

## 库存管理指标

### 安全库存 (Safety Stock)
```
SS = Z × σ_d × √L
```
- Z: 服务水平对应的标准正态分位数（95% → 1.65, 99% → 2.33）
- σ_d: 日需求标准差
- L: 提前期(天)

### 经济批量 (EOQ)
```
EOQ = √(2DS / H)
```
- D: 年需求量
- S: 单次订货成本
- H: 单位年持有成本 = 单价 × 持有成本率

### 再订货点 (Reorder Point)
```
ROP = d × L + SS
```
- d: 日均需求
- L: 提前期
- SS: 安全库存

### 库存周转率
```
ITR = 年销售成本 / 平均库存价值
```

### 库存天数 (DOI)
```
DOI = 365 / ITR
```

## 供应商指标

### 准时交付率 (OTD)
```
OTD = 按时交付批次 / 总交付批次 × 100%
```

### 质量合格率
```
Yield = 合格数量 / 总交付数量 × 100%
```

### 供应商综合评分
```
Score = OTD×40% + Quality×35% + Response×25%
```

## 需求预测指标

### MAPE (平均绝对百分比误差)
```
MAPE = Σ|实际-预测|/实际 × 100% / n
```

### 移动平均
```
MA(n) = Σ(最近n期需求) / n
```

### 指数平滑
```
S_t = α × D_t + (1-α) × S_{t-1}
```

## 供应链风险评分

| 维度 | 权重 | 关键因素 |
|------|------|----------|
| 单一来源 | 25% | 有无备选供应商 |
| 地域集中 | 20% | 贸易摩擦、地缘政治 |
| 交期风险 | 15% | 交期长度、波动性 |
| 质量风险 | 20% | 历史不良率 |
| 财务风险 | 10% | 供应商财务状况 |
| 依赖度 | 10% | 采购金额占比 |
