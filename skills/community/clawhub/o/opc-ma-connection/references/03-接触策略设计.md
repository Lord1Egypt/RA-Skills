# 接触策略设计

> 本文档详细说明针对不同关键人的接触策略，包括冷启动、温启动、热启动三种路径。

---

## 一、策略设计原则

### 1.1 核心原则

| 原则 | 说明 | 应用要点 |
|------|------|----------|
| **循序渐进** | 从易到难，逐步深入 | 不要一开始就直接找实控人 |
| **角色匹配** | 以对方接受的身份切入 | 选择对方愿意接受的身份 |
| **价值导向** | 让对方感到值得花时间 | 提供有价值的信息或合作点 |
| **信任优先** | 先建立信任，再谈合作 | 不要急于推销 |
| **合规底线** | 所有行为必须合法合规 | 不使用灰色手段 |

### 1.2 策略选择矩阵

| 关系基础 | 接触难度 | 推荐策略 |
|----------|----------|----------|
| 无关系 | 高 | 冷启动：公开渠道+活动接触 |
| 一度人脉 | 中 | 温启动：中间人引荐 |
| 直接引荐 | 低 | 热启动：直接推进 |

## 二、冷启动路径

### 2.1 适用场景

- 零关系基础
- 无法找到中间人
- 需要先建立信息渠道

### 2.2 三种冷启动方式

#### 方式一：公开渠道触达

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 获取公开邮箱/电话 | 通过年报、官网、互动易 |
| 2 | 发送专业邮件 | 以行业研究者身份 |
| 3 | 电话跟进 | 确认邮件收到 |
| 4 | 等待回复 | 如未回复，按周期跟进 |

**邮件要点**：
- 标题明确但不提"并购"
- 内容专业，体现行业理解
- 不附带推销意图
- 提供有价值的信息

#### 方式二：活动接触

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 锁定目标活动 | 行业峰会、投资论坛等 |
| 2 | 了解参会嘉宾 | 提前研究目标人物 |
| 3 | 设计接触机会 | 选择合适的提问/交流时机 |
| 4 | 现场建立联系 | 交换联系方式 |
| 5 | 活动后跟进 | 24小时内发送感谢+后续 |

**活动接触话术**：
```
"XX总好，我是XX，刚才听您关于XX的观点深受启发。我是做XX研究的，正好有个问题想请教..."
```

#### 方式三：信息影响

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 产出专业内容 | 行业分析报告、技术文章 |
| 2 | 通过正规渠道发布 | 行业媒体、学术平台 |
| 3 | 引起目标关注 | 让对方主动来找你 |
| 4 | 建立对话基础 | 以回应你的内容切入 |

## 三、温启动路径

### 3.1 适用场景

- 有一度人脉（共同认识的人）
- 可以找到中间人引荐
- 需要提高接触效率

### 3.2 三种温启动方式

#### 方式一：中间人引荐

**操作流程**：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        中间人引荐流程                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  Step 1：确认中间人关系                                                 │
│  └─ 确认中间人与目标的关系和信任程度                                     │
│                                                                          │
│  Step 2：准备背景材料                                                   │
│  └─ 给中间人足够的背景信息，便于引荐                                     │
│                                                                          │
│  Step 3：请求引荐                                                       │
│  └─ 简洁说明意图，不要给中间人压力                                       │
│                                                                          │
│  Step 4：等待回复                                                       │
│  └─ 给中间人思考和安排的时间                                            │
│                                                                          │
│  Step 5：安排会面                                                       │
│  └─ 配合中间人安排的时间和方式                                          │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

**请求中间人引荐话术**：
```
"XX总，有件事想请您帮忙。我这边有个[项目/技术]，和XX公司可能有合作空间。
不知道您是否方便帮忙引荐一下？以下是简要情况...[发送背景资料]
当然，如果觉得不合适，完全理解，不给您添麻烦。"
```

**给中间人的背景材料要素**：
- 项目/技术是什么
- 为什么适合XX公司
- 能给XX公司带来什么价值
- 你的身份和背书

#### 方式二：借势搭车

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 参加中间人组织的活动 | 论坛、沙龙、饭局 |
| 2 | 让中间人知道你的目的 | 提前沟通 |
| 3 | 自然接触目标 | 通过活动自然建立联系 |
| 4 | 不要让中间人尴尬 | 给中间人面子 |

#### 方式三：价值交换

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 识别对方需求 | 了解对方关心什么 |
| 2 | 提供价值 | 分享信息、引荐资源、提供帮助 |
| 3 | 建立互惠 | 让对方感到值得认识你 |
| 4 | 顺势提出需求 | 在信任建立后提出合作想法 |

## 四、热启动路径

### 4.1 适用场景

- 有强力背书人直接引荐
- 可以直接进入正题
- 目标已经知道你的情况

### 4.2 三种热启动方式

#### 方式一：背书人直接带见

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 背书人充分了解情况 | 提前沟通项目详情 |
| 2 | 背书人直接引荐 | 背书人到场或电话介绍 |
| 3 | 直接进入主题 | 不需要再铺垫背景 |
| 4 | 快速推进 | 利用背书人信任背书 |

**背书人引荐话术**：
```
"XX总，这是我的朋友/合作伙伴XX，他有个[项目/技术]我觉得很不错，
和你们公司的[方向]可能有关联，今天想介绍你们认识，听听他的介绍。"
```

#### 方式二：背书人提前沟通

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 背书人先与目标沟通 | 降低目标心理门槛 |
| 2 | 目标同意见面 | 背书人事先铺垫 |
| 3 | 安排正式会议 | 以项目交流为由 |
| 4 | 带上完整材料 | 准备充分的项目材料 |

#### 方式三：高层对话

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 己方高层出面 | 对等接待 |
| 2 | 双方高层直接交流 | 体现诚意 |
| 3 | 快速进入正题 | 时间宝贵 |
| 4 | 明确后续推进 | 当场约定下一步 |

## 五、分角色接触策略

### 5.1 董秘接触策略

| 策略维度 | 具体内容 |
|----------|----------|
| **最佳身份** | 机构投资者、行业研究员、潜在业务合作方 |
| **切入话题** | 行业趋势、公司战略、业务合作可能性 |
| **材料形式** | 行业分析报告、合作建议书（非并购直接提案） |
| **接触频率** | 每季度1-2次，避免过于频繁 |
| **注意事项** | 不能直接提并购，会被过滤；需要"绕道" |
| **成功标志** | 获得后续沟通机会，愿意继续交流 |

**董秘接触话术示例**：
```
"XX总，您好！我是XX机构的行业研究员[姓名]，近期在研究[行业]领域，
注意到贵司在[具体业务]上有不错的布局，想请教几个行业问题，
同时也分享一些我们观察到行业趋势，供您参考。"
```

### 5.2 高管层接触策略

| 策略维度 | 具体内容 |
|----------|----------|
| **最佳身份** | 战略合作伙伴、技术提供方、行业专家 |
| **切入话题** | 产业协同、技术价值、战略意义 |
| **材料形式** | 商业计划书、合作方案、案例分析 |
| **接触频率** | 根据反馈调整，保持适度联系 |
| **注意事项** | 需要有说服力的商业逻辑，强调协同价值 |
| **成功标志** | 愿意安排后续会议，进入深度交流 |

**高管接触话术示例**：
```
"XX总，您好！我是[姓名]，经[中间人/渠道]介绍了解到贵司。
我们团队在[技术/领域]深耕多年，有一个[项目/产品]与贵司的[方向]
可能有协同，想请教一下贵司在[相关领域]的战略规划，
看看是否有合作空间。"
```

### 5.3 实控人/大股东接触策略

| 策略维度 | 具体内容 |
|----------|----------|
| **最佳身份** | 必须通过强力中间人引荐 |
| **切入话题** | 简洁有力的价值主张，核心利益点 |
| **材料形式** | 一页纸摘要，口头汇报优先 |
| **接触频率** | 精准出击，不频繁打扰 |
| **注意事项** | 时间极宝贵，话不在多，在于精 |
| **成功标志** | 直接拍板或指定对接人 |

**实控人接触话术示例**：
```
"XX总，您好！冒昧打扰。我是[姓名]，经[重量级中间人]介绍，
有个[技术/项目]可能与贵司的战略方向契合，能给我5分钟做个简要汇报吗？"
```

### 5.4 独立董事接触策略

| 策略维度 | 具体内容 |
|----------|----------|
| **最佳身份** | 学术同行、产业专家、专业顾问 |
| **切入话题** | 技术前景、行业趋势、专业判断 |
| **材料形式** | 技术评估报告、行业白皮书、学术论文 |
| **接触频率** | 专业场合交流，不商业化 |
| **注意事项** | 保持专业客观，不施加压力 |
| **成功标志** | 在董事会发表专业意见支持 |

### 5.5 外部推动者接触策略

| 策略维度 | 具体内容 |
|----------|----------|
| **最佳身份** | 项目方、合作方 |
| **切入话题** | 合作机会、资源互补 |
| **材料形式** | 项目简介、合作方案 |
| **接触频率** | 根据合作潜力调整 |
| **注意事项** | 评估其利益立场和排他要求 |
| **成功标志** | 获得引荐或合作承诺 |

## 七、策略调整清单

### 7.1 常见情况应对

| 情况 | 策略调整 |
|------|----------|
| 对方不回复 | 等待3-5天后再次联系，语气更简洁 |
| 对方表示没兴趣 | 问清原因，调整方向或换标的 |
| 对方态度冷淡 | 降低接触频率，提供更有价值的信息 |
| 对方很忙 | 简短预约，尊重对方时间 |
| 被推荐给其他人 | 感谢并立即跟进新联系人 |

### 7.2 风险控制

| 风险类型 | 预防措施 |
|----------|----------|
| **暴露意图过早** | 先建立关系，再逐步深入 |
| **过度打扰** | 控制接触频率，尊重对方反馈 |
| **信息泄露** | 注意保密，重要信息先签NDA |
| **撞车风险** | 同一公司避免多人同时联系 |
