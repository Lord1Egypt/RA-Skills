# Session Archiver Pro — Output Formats

## Format Catalog

| Format | Use Case | File Extension |
|--------|----------|----------------|
| `markdown` | Human-readable reports, sharing | `.md` |
| `json` | Programmatic consumption, API integration | `.json` |
| `obsidian` | Note-taking in Obsidian vault | `.md` |
| `memory-inject` | Direct agent long-term memory injection | `.json` |

## Format Examples

### Markdown
Full structured report with headers, tables, and tagged sections.

### JSON
Machine-parseable output with session-by-session breakdown.

### Obsidian
Wikilink-format compatible with `[[backlinks]]` for vault navigation.

### Memory-Inject
Agents can read this JSON to populate their long-term memory context.
