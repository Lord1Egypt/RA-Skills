# 儿童传染病诊断与防治

## 适用对象
- 疑似或确诊传染病患儿
- HIV暴露儿童

## 疾病处理要点

### 1. 风湿热
- **Jones标准**：链球菌感染证据 + (2项主要或1主2次)
- **主要表现**：心脏炎、多发性关节炎、舞蹈病、环形红斑、皮下小结
- **预防**：苄星青霉素每3–4周肌注，至少5年

### 2. 麻疹
- **典型四期**：潜伏期→前驱期（Koplik斑）→出疹期（自上而下）→恢复期（色素沉着+脱屑）
- **非典型**：轻型（症状轻）、重型（黑麻疹）、异型（灭活疫苗接种史）

### 3. 流脑
- **普通型**：前驱→败血症期（瘀点瘀斑）→脑膜炎期
- **暴发型**：休克型、脑膜脑炎型、混合型

### 4. 流感
- **抗病毒**：奥司他韦（≤8月：6mg/kg/d；≥1岁按体重分层）
- **重症神经受累**：甘露醇降颅压（0.5–1.0g/kg q4–8h）+ IVIG（2g/kg）

### 5. 结核性脑膜炎
- **预后因素**：治疗早晚、年龄、病期病型、耐药性、治疗规范性

### 6. 鹅口疮
- **诊断**：口腔白色乳凝块样斑膜，不易擦除
- **治疗**：2%碳酸氢钠清洁 + 制霉菌素局部涂抹

### 7. HIV感染
- **<18月**：2次HIV RNA阳性确诊
- **≥18月**：抗体阳性或HIV RNA阳性确诊

## 输出结果
- 疾病诊断结论（含分型/分期）
- 具体治疗方案（药物、剂量、疗程）
- 预防与监测建议
