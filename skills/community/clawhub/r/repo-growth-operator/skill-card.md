## Description: <br>
Recommend small, practical growth actions for a public GitHub repository based on releases, skills, docs, and current positioning. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[guoma970](https://clawhub.ai/user/guoma970) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External developers and maintainers use this skill to plan short, practical growth and maintenance actions for public GitHub repositories after a release or visible project milestone. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Recommendations may be misleading if the user provides private context, sensitive repository details, or asks the skill to infer traction that is not publicly visible. <br>
Mitigation: Use the skill for public repositories and explicit public evidence; avoid private or sensitive repos unless that context is intended to be analyzed. <br>
Risk: Growth advice can overstate community activity or repository momentum if not tied to visible assets. <br>
Mitigation: Keep recommendations grounded in releases, documentation, examples, issues, and other visible repository signals. <br>
