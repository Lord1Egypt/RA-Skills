# 用户指南

## 快速开始

### 1. 创建账本

首次使用时，Agent 会帮你创建账本文件。只需告诉 Agent：
- 你常用的支付账户（如微信钱包、支付宝、银行卡）
- 各账户的当前余额

或者从示例账本开始：
```bash
cp data/example.csv ~/.openclaw/workspace/data/ledger/default.csv
```

### 2. 开始记账

最简单的方式，直接说话：

```
今天中午吃饭花了50块
昨天打车35
工资到账8000
买了一件外套299，用支付宝付的
```

Agent 会自动解析并写入账本。

> ⚠️ **注意：** 所有写入操作（记账、修改、删除）都是**持久化、直接修改本地文件**的，不可自动撤销。
> 建议定期备份 `~/.openclaw/workspace/data/ledger/` 目录。

### 3. 查询消费

```
这个月花了多少？
上个月餐饮多少钱？
查一下我的余额
```

### 4. 生成月度报告

```
给我做个5月的月度报告
```

## FAQ

**Q: 账本存在哪里？**
A: 默认 `~/.openclaw/workspace/data/ledger/default.csv`，可以用任何文本编辑器打开。

**Q: 分类不准怎么办？**
A: 可以手动指定：`花了50，分类是餐饮`。Agent 会记住你的偏好。

**Q: 记账错了怎么改？**
A: 直接告诉 Agent：「删掉5月15日那笔餐饮」或「把刚才那笔改成30块」。

**Q: 需要网络吗？**
A: 不需要。所有功能都基于本地文件，纯 Python 标准库，无网络依赖。

**Q: 数据安全吗？**
A: 数据存在你本地的文件里，没有任何外部服务。你可以随时备份、移动或加密。

**Q: 支持多账户吗？**
A: 支持。微信钱包、支付宝、银行卡、现金等都可以记录，也可以互相转账。

**Q: 可以导出数据吗？**
A: 直接用文本编辑器打开 CSV 文件即可，格式是标准 CSV，任何表格软件都能打开。

## 分类推荐

建议保持简单，优先用一级分类：
- 餐饮、交通、购物、居住、娱乐、通讯、医疗、教育、社交、其他
- 收入用：工资、奖金、理财收益、兼职、其他

分类太细会增加记录负担，失去随手记的意义。
