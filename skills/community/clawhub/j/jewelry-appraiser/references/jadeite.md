# 翡翠鉴定与选购指南

## 一、翡翠 ABC 货定义

| 级别 | 定义 | 处理方式 | 价值 | 证书标注 |
|------|------|---------|------|----------|
| **A 货** | 天然翡翠，仅物理打磨抛光 | 无化学处理 | 100%（基准） | 翡翠(A货) / 天然翡翠 |
| **B 货** | 酸洗注胶翡翠 | 强酸浸泡去杂质 → 注胶填充 | 1-5% | 翡翠(处理) / B货翡翠 |
| **B+C 货** | 酸洗 + 染色 | 酸洗注胶 + 人工染色 | <1% | 翡翠(处理) |
| **C 货** | 仅染色处理 | 人工染料着色 | <1% | 翡翠(处理) |
| **D 货** | 仿冒品 | 玻璃/石英岩/玉髓冒充 | 无价值 | 非翡翠 |

## 二、翡翠品质评价：种水色工

### 种（质地细腻度）

| 种 | 特征 | 透明度 | 价值排序 |
|----|------|--------|----------|
| **玻璃种** | 完全透明，玻璃光泽 | 全透明 | ⭐⭐⭐⭐⭐ |
| **冰种** | 半透明，冰感 | 高 | ⭐⭐⭐⭐ |
| **糯冰种** | 微透明，糯米汤感 | 中 | ⭐⭐⭐ |
| **糯种** | 不透明，细腻 | 低 | ⭐⭐ |
| **豆种** | 颗粒感明显 | 不透明 | ⭐ |

### 水（透明度）
- 水头足 = 透明度高 = 价值高
- 常见说法："三分水"指 9mm 厚度可见背面字迹

### 色（颜色）

翡翠颜色等级（价值排序）：

| 颜色 | 代表品种 | 价值特征 |
|------|---------|----------|
| **绿色** | 帝王绿、阳绿、苹果绿 | 最高 |
| **紫色** | 紫罗兰(春色) | 高（纯浓为贵） |
| **红色/黄色** | 翡色（红翡、黄翡） | 中高 |
| **无色** | 玻璃种/冰种无色 | 中 |
| **白色** | 白底青 | 中低 |
| **黑色** | 墨翠 | 中 |
| **蓝色** | 蓝水 | 中 |

### 绿色等级细分
| 等级 | 描述 | 参考价(手镯) |
|------|------|-------------|
| 帝王绿 | 浓郁纯正翠绿，色调均匀 | 百万级-千万 |
| 正阳绿 | 鲜艳翠绿，略逊于帝王绿 | 十万-百万级 |
| 苹果绿 | 清新明亮绿色 | 数万-数十万 |
| 豆绿 | 偏黄绿色调 | 数千-数万 |
| 油青绿 | 灰绿色调，偏暗 | 数百-数千 |

### 工（雕工）
- **素面 > 雕花**：光面手镯、蛋面价值最高
- 好料配好工，差料靠雕工遮瑕
- 大师工溢价 2-5 倍

## 三、A/B 货鉴别方法

### 肉眼鉴定法
| 方法 | A 货特征 | B 货特征 |
|------|---------|----------|
| 光泽 | 强玻璃光泽 | 树脂/蜡状光泽 |
| 颜色 | 色根明显，分布不均 | 颜色漂浮，网纹状 |
| 结构 | 粒状/纤维交织结构 | 结构松散，酸蚀网纹 |
| 声音 | 清脆悦耳(手镯敲击) | 沉闷沙哑 |
| 荧光 | 紫外灯下无或弱荧光 | 紫外灯下强蓝白荧光(注胶特征) |
| 比重感 | 手感沉甸（密度3.33） | 偏轻 |

## 五、选购黄金法则

### 预算分配建议
| 预算 | 推荐方案 |
|------|---------|
| 1,000-5,000 | 糯种小挂件、A货蛋面 |
| 5,000-20,000 | 糯冰种挂件/小摆件、冰种蛋面 |
| 20,000-100,000 | 冰种飘花手镯、冰糯阳绿挂件 |
| 100,000-500,000 | 冰种正阳绿手镯、玻璃种挂件 |
| 500,000+ | 玻璃种阳绿手镯、顶级蛋面 |

### 必备原则
1. **先看种，再看色**：种差的翡翠颜色再好也缺乏灵气
2. **必须要有鉴定证书**：首选 NGTC 证书
3. **灯光下不买翡翠**：灯光美化颜色和透明度，必须在自然光下看
4. **新手先买小件**：先从蛋面、挂件入门，再上镯子
5. **警惕"捡漏"心态**：翡翠市场没有捡漏，便宜必有猫腻

## 六、常见陷阱

1. **B+C 充当 A 货**：最常见骗局，染色充胶翡翠冒充天然
2. **危料充缅料**：危地马拉翡翠冒充缅甸翡翠（蓝水/墨翠重灾区）
3. **永楚料充冰种**：危地马拉永楚料外观似冰种但结构不同
4. **抛光粉残留**：紫色/绿色抛光粉残留冒充天然色
5. **合成/人造翡翠**：GE合成翡翠已在市场出现
6. **央视/明星背书骗局**：旅游区购物点惯用套路
7. **直播套路**：直播间灯光+滤镜美化，实物完全两回事
