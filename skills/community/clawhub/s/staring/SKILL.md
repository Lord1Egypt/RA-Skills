### 🥇 达人匹配 — 10秒出名单，不是AI瞎猜

五维加权评分 → S/A/B/C 四级，每一档都有数据依据：

| 维度 | 权重 | 说明 |
|---------|:---:|------|
| 类目契合度 | 30% | 类目不对，后面全浪费 |
| 历史带货力 | 25% | 近30天场均GMV |
| 粉丝量级 | 20% | 粉丝基数 × 画像匹配 |
| 佣金率匹配 | 15% | 筛掉开口就要50%的 |
| 近期活跃度 | 10% | 近14天开播频次 |

```

```

⏰ 逼单     → 限时限量+价格锚点+社交证明
💬 互动控场 → 评论引导+高频QA预制+冷场救急

> 不是让你念稿，是把说服逻辑排好。彩排两遍比即兴稳10倍。

## 📦 5分钟上手

```bash
# 1. 安装
easyclaw skills install https://clawhub.ai/myc1020/staring

# 2. 匹配达人
python cli.py influencer match --category "美妆"

# 3. 生成脚本
python cli.py script fabe --product "你的品名" --price 你的价格

# 4. 宣传文案
python cli.py promo wechat --style story --product "你的品名"
```

两种用法：**对话式**（说人话就行）｜ **命令行**（批量自动化）
