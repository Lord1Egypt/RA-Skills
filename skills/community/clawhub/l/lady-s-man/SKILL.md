# Lady's Man / 陆小凤

全面的浪漫沟通和恋爱话术技能，帮助你在各种场景下与女生愉快交流。

## 快速开始

当用户请求聊天/浪漫/撩妹建议时：

1. 识别**场景**（开场、调情、表白、日常聊天、关怀、邀约、应对拒绝、幽默互动）
2. 匹配下方的**参考文件**
3. 选择合适的话术或根据用户具体情况改编模式
4. 提供3-5个选项，按大胆程度排序（保守 → 适中 → 大胆）
5. 包含使用技巧和时机建议

## 黄金法则

1. **真诚 > 技巧** — 话术只是载体，真心才是核心
2. **频率很重要** — 每次对话最多1-2个套路，配合正常聊天
3. **察言观色** — 如果她反应不好，切换到真诚对话
4. **改编而非照搬** — 个性化话术以适应你的风格和她的性格
5. **自信就是一切** — 同样的话术因表达方式而成功或失败
6. **懂得适可而止** — 识别真正的冷淡 vs 顽皮的抵抗
7. **行动胜言语** — "多喝热水"不如"粥给你点好了"，说得到做得到

## 风险等级

为每条话术建议标记风险等级：
- **绿色** — 安全，适用于大多数情况
- **黄色** — 需要已建立融洽关系或特定语境
- **红色** — 高风险高回报，仅在确认相互吸引后使用

## 安全指南

- 永远不要鼓励骚扰或在明确拒绝后继续纠缠
- 强调尊重边界和同意
- 建议不要在专业场合使用话术
- 提醒用户没有话术对所有人都有效 — 关键在于合适，而非公式
- 区分"欲擒故纵"（仍在互动）和真正的冷淡（不再互动）
- 所有话术都应服务于真诚的情感表达
- **"不"就是"不"**，不存在"她其实是想要的"这种解读
