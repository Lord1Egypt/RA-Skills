## Known Risks and Mitigations: <br>
Risk: The security evidence notes powerful workflow helpers may grant an agent operational authority over ClawHub or Convex work. <br>
Mitigation: Install only from trusted sources and confirm the exact target and expected effect before running moderation, deployment, migration, proof-publishing, or autoreview commands. <br>
Risk: Nested autoreview can run with broad local access when enabled. <br>
Mitigation: Use the autoreview --no-yolo option when broad nested review access is not intended. <br>
