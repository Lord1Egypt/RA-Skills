## 质量控制词

### 必加（结尾追加）
```
photorealistic / illustration style, ultra-detailed, sharp focus,
no watermark, no text (unless requested), high resolution
```

### 皮肤（人像必加）
```
realistic skin texture, visible pores, natural imperfection
no plastic skin, no airbrushing
```

### 光影（电影感必加）
```
volumetric light rays, cinematic rim lighting
cinematic color grading, film grain
```

## 禁忌清单

| 不要写 | 原因 | 替代 |
|--------|------|------|
| beautiful | 空洞模糊 | specific feature: sharp jawline, clear eyes |
| good lighting | 抽象 | soft diffused, dramatic rim light |
| nice background | 模糊 | bokeh city lights, wooden walls |
| cool | 主观 | leather jacket, sunglasses, confident pose |
| modern | 模糊 | minimalist design, sleek lines, white tones |
| pretty | 空洞 | elegant, refined, sophisticated |
| cute | 空洞 | specific cute element: big eyes, small nose |

## 反馈调整速查

| 问题 | 快速修复 |
|------|---------|
| 太假 | + "realistic skin texture, no plastic skin" |
| 太暗 | + "bright, well-lit, natural lighting" |
| 太亮 | + "soft lighting, muted tones" |

- 「上手度」评分最高
- 最适合目标使用场景
