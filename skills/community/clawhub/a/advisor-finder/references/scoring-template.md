# Scoring Template

Use this as a soft rubric, not fake precision.

## Dimensions
- Direction match: 0-5
- Recent activity: 0-5
- Evidence strength: 0-5
- User-background fit: 0-5
- Contact value: 0-5

## Meaning

### Direction match
- 5: target direction is clearly a core current line
- 3: target direction appears repeatedly but not as the only core line
- 1: only weak or occasional overlap

### Recent activity
- 5: multiple recent papers or visible current projects in 2-3 years
- 3: some recent activity but not very dense
- 1: little recent visible evidence

### Evidence strength
- 5: official page + personal page + recent papers line up
- 3: two of those line up
- 1: mostly inferred from noisy sources

### User-background fit
- 5: strong overlap in methods or topics
- 3: some overlap, user can plausibly bridge the gap
- 1: likely hard to make a convincing contact case

### Contact value
- 5: strong candidate worth early outreach
- 3: reasonable candidate
- 1: backup only

## Ranking bands
- Priority A: mostly 4-5 with no major unresolved contradiction
- Priority B: mixed 3-4 with one notable uncertainty
- Priority C: weak evidence or weak fit
