# 女性生殖器官胚胎发育关键路径

## 触发条件
当需解释女性内生殖器解剖结构起源或先天畸形机制时，依据米勒管与尿生殖窦发育路径进行分析。

## 执行步骤

1. **确认适用场景**：
   - 仅适用于胚胎第3–12周的发育阶段分析。
   - 不适用于出生后病理变化或青春期激素诱导的发育。

2. **分析性腺发育**：
   - 胚胎第3–4周：原始生殖细胞出现在卵黄囊内胚层。
   - 第5周：尿生殖嵴形成，内侧为生殖腺嵴。
   - 第7周前：性腺未分化，男女相同。
   - 第10周后（XX染色体）：初级性索退化，次级性索形成卵巢皮质，表面间充质形成白膜。

3. **追踪内生殖管道发育**：
   - 在无雄激素和抗米勒管激素（AMH）环境下，米勒管持续发育。
   - 第7周：米勒管头段保持管状 → 发育为输卵管（伞端开口）。
   - 中段与下段融合 → 形成子宫体、子宫颈及阴道上1/3。
   - 融合初期存在纵隔，通常于胎儿20周吸收，形成单一宫腔。

4. **解析下生殖道发育**：
   - 阴道下2/3来源于尿生殖窦盆腔部分的远端。
   - 外生殖器发育：
     - 阴唇阴囊隆起 → 大阴唇
     - 尿生殖褶 → 小阴唇
     - 生殖结节 → 阴蒂

5. **识别关键分化时间点**：
   - 遗传性别：受精时确定（XX）。
   - 生殖腺性别：第7周可辨认。
   - 外生殖器性别：第12周基本完成分化。

6. **判断调控因素**：
   - 女性发育为默认路径，前提是缺乏Y染色体SRY基因、睾丸决定因子（TDF）、雄激素及AMH。

## 输出用途
用于解释正常解剖结构形成机制，或诊断苗勒管发育不全、阴道闭锁等先天畸形。
