This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and skill authors use this agent skill to create and validate Claude Code skills through test-driven authoring, progressive disclosure, description tuning, and anti-rationalization guidance. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Strong persuasion and anti-rationalization language may influence agent behavior more forcefully than some workflows expect. <br>
Mitigation: Review the tone before deployment and adjust local activation or acceptance criteria if the organization needs softer guidance. <br>
Risk: Broad triggers such as authoring, skills, writing, best-practices, and validation may activate the skill for ordinary writing or validation tasks. <br>
Mitigation: Narrow triggers or usage policy when deploying if activation should be limited to skill-authoring work. <br>
