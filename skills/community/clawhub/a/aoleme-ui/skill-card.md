## Use Case: <br>
Developers and designers use this skill to restyle React, Tailwind, or CSS interfaces as dark neon cyber-xianxia dashboards, gamified pages, and glassmorphism-heavy app surfaces while preserving the host app structure. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Blindly importing the bundled global CSS or Tailwind resources can override host layout, scrolling, text selection, or existing design tokens. <br>
Mitigation: Merge only the needed tokens and utility classes, then review the changed interface against the host application's structure and interaction patterns. <br>
Risk: Heavy blur, glow, and animation effects can reduce performance, contrast, or mobile usability. <br>
Mitigation: Verify contrast, responsive layout, touch targets, scroll behavior, and animation cost after applying the style. <br>
