## Description: <br>
Build web3 applications, scripts, CLIs, bots, mobile apps, and desktop apps that need blockchain data via the Blockscout PRO API, a single HTTP API spanning 100+ EVM chains. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to build software that reads blockchain data through the Blockscout PRO API, including transactions, blocks, addresses, tokens, NFTs, logs, contract context, internal transactions, and account abstraction data. It guides API key handling, endpoint selection, request construction, and code patterns for direct HTTP integrations. <br>
