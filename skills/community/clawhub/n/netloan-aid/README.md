# 网贷还款援助助手 (NetLoan Aid)

> ⚠️ 免责声明：本工具由 AI 生成，结合公开法律法规和网贷资料提供参考建议。所有内容**仅供参考，不构成正式法律意见或金融建议**。

## 功能

- 🔍 **利率合规分析**：判断利率是否在法定范围内
- 💰 **债务明细计算**：本金 + 利息 + 罚息
- 💡 **多种还款方案**：展期/分期/减免/法律维权等
- 📈 **可行性评估**：每个方案的成功概率
- 💬 **协商话术模板**：可直接使用

## 使用

```bash
# 交互模式
node netloan-aid.js

# 参数模式
node netloan-aid.js --platform "借呗" --amount 50000 --rate 36 --overdue_days 90 --monthly_income 8000

# 帮助
node netloan-aid.js --help
```

## 法规依据

- 《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》
- 年利率 24% 以内：合法
- 年利率 24%~36%：自然债务
- 年利率超过 36%：超过部分无效
