# 《中國文哲研究集刊》撰稿格式規範

## 一、總則

本文獻引用格式適用於中文學術論著，涵蓋腳註與文末參考文獻兩大系統。

## 二、腳註格式

### 2.1 首次引用（完整格式）

**專書**：
```
作者名，《書名》（出版地：出版社，年份），頁碼。
```
例：余英時，《朱熹的歷史世界：宋代士大夫政治文化的研究》（臺北：允晨文化，2003），頁 25。

**外文專書**：
```
Author Name, Book Title (Place: Publisher, Year), pp. XX-XX.
```
例：Mark Edward Lewis, *Writing and Authority in Early China* (Albany: State University of New York Press, 1999), pp. 5-10.

**期刊論文**：
```
作者名，〈篇名〉，《期刊名》卷.期（年月），頁碼。
```
例：王汎森，〈思想史中的「思想」〉，《中央研究院歷史語言研究所集刊》75.1（2004.3），頁 1-28。

**外文期刊**：
```
Author Name, "Article Title," *Journal Name* Volume.Issue (Month Year): page range.
```
例：Joshua A. Fogel, "'Shanghai-Japan': The Japanese Residents' Association of Shanghai," *Journal of Asian Studies* 59.4 (Nov. 2000): 927-950.

**論文集（書中篇章）**：
```
作者名，〈篇名〉，收入編者名編，《書名》（出版地：出版社，年份），頁碼。
```
例：John C. Y. Wang, "Early Chinese Narrative: The Tso-chuan as Example," in *Chinese Narrative: Critical and Theoretical Essays*, ed. Andrew H. Plaks (Princeton: Princeton University Press, 1977), pp. 3-20.

**學位論文**：
```
作者名，《論文題目》（Ph.D. diss. / M.A. thesis, 學校名, 年份），頁碼。
```
例：Hwang Ming-chorng, "Mingtang: Cosmology, Political Order and Monuments in Early China" (Ph.D. diss., Harvard University, 1996), p. 20.

### 2.2 第二次引用（縮寫格式）

- 同一文獻第二次引用時，使用「同上」。
- 若頁碼與前注不同，標註頁碼：「同上，頁 XX。」
- 若頁碼與前注相同，只寫「同上。」

例：
```
同上，頁 33。
```

### 2.3 間隔引用（同註）

- 非連續引用同一文獻時，使用「同註[編號]，頁 XX。」
- 若頁碼與該注相同，可省略頁碼。

例：
```
同註[3]，頁 45。
```

### 2.4 多作者引用

**兩位作者**：
```
作者甲、作者乙，《書名》...
```

**三位及以上作者**：
```
作者甲等，《書名》... 或 作者甲et al., Book Title...
```

### 2.5 重複引用同一位作者多篇著作

第二次引用同一作者不同著作時：
```
作者名，《書名》（見註[編號]），頁 XX。
```
或簡寫：
```
作者名，《書名》，頁 XX。
```

### 2.6 報紙引用

```
作者名，〈篇名〉，《報紙名》[出版地] 日期，版次，頁碼。
```
例：Michael A. Lev, "Nativity Signals Deep Roots for Christianity in China," *Chicago Tribune* [Chicago] 18 March 2008, Sec. 1, p. 4.

## 三、文末參考文獻格式

### 3.1 排序規則

- 按作者姓氏拼音（外文）或筆畫（中文）排序。
- 無作者者按書名首字排序。
- 同一作者按出版年份排序。

### 3.2 格式示例

**中文專書**：
```
余英時：《朱熹的歷史世界：宋代士大夫政治文化的研究》。臺北：允晨文化，2003。
```

**外文專書**：
```
Lewis, Mark Edward. Writing and Authority in Early China. Albany: State University of New York Press, 1999.
```
（注意：外文參考文獻採用姓名倒置，標點改為句點。）

**期刊論文**：
```
王汎森：〈思想史中的「思想」〉。《中央研究院歷史語言研究所集刊》75.1（2004.3）：1-28。
```

**外文期刊**：
```
Fogel, Joshua A. "'Shanghai-Japan': The Japanese Residents' Association of Shanghai." Journal of Asian Studies 59.4 (Nov. 2000): 927-950.
```

### 3.3 外文參考文獻特殊規則

- 作者姓名倒置：Last Name, First Name.
- 書名改為斜體（以 * 標記）。
- 刊名改為斜體。
- 篇名加引號。
- 標點符號從中文格式（頓號、書名號）改為英文格式（句點、逗號、引號）。

## 四、常見錯誤與修正

| 錯誤 | 正確 |
|------|------|
| 同上，頁 25-30（首次引用用同上） | 首次引用必須用完整格式 |
| 余英時，《書名》，頁 25（缺少出版資訊） | 余英時，《書名》（臺北：允晨文化，2003），頁 25。 |
| 同註 3（缺少頁碼且頁碼不同時） | 同註[3]，頁 45。 |
| Lewis, Mark Edward, Book Title...（外文參考文獻未倒置姓名） | Lewis, Mark Edward. Book Title... |
| 書名未用書名號《》 | 中文專書必須用《》，期刊用〈〉 |
| 頁碼寫成 "p. 25"（中文文獻） | 中文用「頁 25」，外文用 "p. 25" 或 "pp. 25-30" |

## 五、中英混排規則

- 中文正文夾雜外文時，外文詞組、書名、專有名詞保持原文，不翻譯。
- 腳註中外文文獻完全按外文格式處理，不混入中文標點（除整句結構外）。
- 參考文獻區分明確分類：中文文獻在前（按筆畫），外文文獻在後（按字母）。

## 六、輸出模板

### 腳註模板

```
[首次引用]
作者名，《書名》（出版地：出版社，年份），頁碼。

[第二次引用]
同上，頁碼。

[間隔引用]
同註[編號]，頁碼。
```

### 參考文獻模板

```
[中文]
作者名：《書名》。出版地：出版社，年份。

[外文]
Author Last Name, First Name. *Book Title*. Place: Publisher, Year.
```
