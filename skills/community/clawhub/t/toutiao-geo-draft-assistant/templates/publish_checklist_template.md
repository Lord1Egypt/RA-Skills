# 今日头条发布前人工审核清单 (Publish Checklist)

在您点击头条号的“发布”按钮前，请务必人工核对以下事项：

- [ ] **内容准确性**：是否事实准确？是否**不存在**虚假案例、虚假新闻感或伪造数据？
- [ ] **平台风格**：文章是否足够通俗？是否具有现实场景感？是否适合今日头条用户阅读（而非晦涩的技术文档）？
- [ ] **标题与封面**：标题是否具有吸引力但**绝不标题党**？封面图是否合规、无侵权元素且非虚假截图？
- [ ] **营销克制**：是否存在过度营销？品牌露出是否自然？是否需要减少硬广成分？
- [ ] **合规红线**：是否**不含有**制造焦虑的表达？是否**不含有**夸大承诺（如“保证收录”、“一定带来流量”、“颠覆行业”）？
- [ ] **边界清晰**：文章中是否明确说明了该技术或方案的“适用边界”（即它不能做什么）？
- [ ] **本地检查**：是否确认没有配置任何自动发布的脚本？是否确认将由**您本人最终手动点击发布**？
