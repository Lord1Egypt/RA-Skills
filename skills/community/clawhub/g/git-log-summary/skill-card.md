## Description: <br>
Generates Markdown Git commit summary reports with project details, branch information, commit totals, author statistics, recent commits, and commit-type statistics. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wj-chitang](https://clawhub.ai/user/wj-chitang) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to generate shareable Markdown reports summarizing local or remote Git repository history, contributors, branches, and commit categories. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The remote repository helper can run unintended shell commands from crafted arguments. <br>
Mitigation: Use the remote helper only with trusted repository URLs and argument values until the eval usage is fixed. <br>
Risk: Generated reports can disclose local paths, remote URLs, author names, branch names, and commit messages. <br>
Mitigation: Inspect and redact generated Markdown reports before sharing them outside the intended audience. <br>
Risk: Output paths can overwrite existing files. <br>
Mitigation: Choose report output paths deliberately and avoid pointing the scripts at important existing files. <br>
