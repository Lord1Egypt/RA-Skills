# SEO Guidelines for Product Descriptions

Keyword and search optimization best practices for ecommerce product descriptions across major platforms. Use this reference alongside the Description Frameworks and Output Template.

---

## Keyword Research Fundamentals

### Keyword types for product descriptions

**Primary keyword:** The single most important search term for this product. Appears in the title, first bullet, and first paragraph of the long description. One per product — never split focus.

**Secondary keywords:** 3-5 supporting terms that capture related searches. Distributed across bullets and body copy. Should complement, not duplicate, the primary keyword.

**Long-tail keywords:** Specific 3-5 word phrases with lower volume but higher purchase intent. Examples: "stainless steel garlic press dishwasher safe" vs "garlic press." These convert better and face less competition.

**Semantic variations:** Synonyms and related terms that search engines associate with the primary keyword. "Running shoes" → "jogging sneakers," "athletic footwear," "trail runners." Include naturally — don't force.

### Keyword placement priority

1. **Product title** — Highest SEO weight on every platform. Primary keyword must appear here.
2. **First bullet point** — Second-highest weight. Primary keyword restated in a benefit context.
3. **First paragraph of long description** — Reinforces relevance for both platform search and Google indexing.
4. **Subsequent bullets** — One secondary keyword per bullet, naturally integrated.
5. **Meta description** — For Shopify/DTC sites. Primary keyword within first 70 characters.
6. **Image alt text** — Often overlooked. Include primary keyword + descriptive context.
7. **Backend search terms** — Amazon-specific. Use remaining keywords not in visible copy.

### Keyword density guidelines

- **Title:** 1-2 keyword instances maximum. More looks spammy.
- **Bullets (combined):** Primary keyword 1-2 times across all bullets. Each secondary keyword once.
- **Long description:** Primary keyword 2-3 times per 250 words. Secondary keywords 1-2 times each.
- **Overall density target:** 1-2% for primary keyword. Below 1% = missed opportunity. Above 3% = keyword stuffing risk.
- **Rule of thumb:** If the keyword placement feels forced when read aloud, remove one instance.

## Platform-Specific SEO

### Amazon Search (A9 / COSMO Algorithm)

Amazon's search algorithm ranks products based on relevance and performance.

**Relevance factors (what you control in copy):**
- Exact keyword match in title carries highest weight
- Backend search terms fill gaps without cluttering visible copy
- Bullet points are indexed — keywords here count
- Product description is indexed but weighted lower than title and bullets
- Category node selection affects which searches surface the product

**Performance factors (influenced by copy quality):**
- Click-through rate from search results — compelling titles and main images
- Conversion rate on the product page — persuasive bullets and descriptions
- Sales velocity — better copy → more sales → higher ranking (virtuous cycle)
- Review count and rating — not directly copy-related but affected by setting accurate expectations

**Amazon-specific SEO rules:**
- Do NOT repeat the brand name in backend search terms (it's already indexed from the brand field)
- Do NOT use commas in backend search terms — spaces are sufficient separators
- Backend search terms have a 250-byte limit — use every byte
- Include common misspellings in backend terms (not in visible copy)
- Include Spanish translations of key terms if selling in US market
- Avoid subjective claims ("best," "top-rated") — Amazon may suppress listings
- Do NOT include competitor brand names in any field — policy violation

**Title optimization for Amazon:**
```
[Brand] [Product Name] [Key Feature] [Size/Count] [Color/Variant]
```
- Primary keyword within first 80 characters (mobile truncation point)
- No promotional language ("Sale," "Free Shipping," "Best Seller")
- No special characters except hyphens and commas
- Capitalize first letter of each word (Title Case)

### Shopify / Direct-to-Consumer SEO

Shopify product pages are indexed by Google. Optimize for Google's algorithm.

**On-page SEO elements:**
- **Page title (H1):** Auto-set from product name. Keep ≤60 characters for full Google display.
- **Meta description:** Set manually in Shopify admin. ≤155 characters. Include primary keyword and a value proposition or CTA.
- **URL handle:** Auto-generated from title. Edit to be short and keyword-rich: `/products/merino-wool-hiking-socks` not `/products/ultra-premium-merino-wool-blend-hiking-socks-3-pack-grey`
- **Image alt text:** Describe the image AND include the primary keyword. "Merino wool hiking socks in charcoal grey — ankle height" not "product-image-001"
- **Structured data:** Shopify auto-generates Product schema. Verify it includes price, availability, and review data using Google's Rich Results Test.

**Content structure for Google:**
- Use H2 and H3 tags in the product description (Shopify supports full HTML)
- First 160 characters of the description may appear as Google's snippet — front-load value
- Internal links to related products help Google understand site structure
- Unique descriptions for every product — duplicate content hurts rankings

**Technical SEO checklist:**
- Page load speed under 3 seconds (compress images, minimize scripts)
- Mobile-responsive layout (Google uses mobile-first indexing)
- Canonical URLs set correctly for variant pages
- No blocked resources in robots.txt that prevent Google from rendering the page

### Walmart SEO

Walmart's search algorithm (similar to Amazon's) weighs:
- **Title relevance:** Primary keyword in title is essential
- **Content quality score:** Walmart rates listings on completeness. Fill all available fields.
- **Rich media:** Listings with video and enhanced content rank higher
- **Price competitiveness:** Walmart factors price into ranking more heavily than Amazon

**Walmart-specific rules:**
- Shorter titles preferred (≤75 characters ideal)
- Key features limited to 80 characters each — be concise
- No promotional language, pricing info, or competitor mentions
- Walmart Content Quality Score visible in Seller Center — aim for 95%+

### Etsy SEO

Etsy search uses a different model focused on relevance, listing quality, and recency.

**Etsy-specific optimization:**
- **Tags:** 13 tags available, each ≤20 characters. Use all 13. Multi-word phrases preferred over single words.
- **Title:** First 40 characters matter most. Front-load primary keyword.
- **Categories and attributes:** Selecting the most specific category improves relevance.
- **Listing renewal:** New and renewed listings get a temporary search boost.
- **Shop sections:** Organizing products into sections helps Etsy understand your catalog.

**Tag strategy:**
- Use long-tail phrases: "personalized leather wallet" not "leather" + "wallet" + "personalized" as separate tags
- Include gift-related tags if applicable: "gift for him," "anniversary gift"
- Mix broad and specific: "kitchen decor" (broad) + "ceramic spoon rest blue" (specific)
- Check Etsy's search bar autocomplete for real buyer search terms

## Keyword Cannibalization Prevention

When writing descriptions for multiple products in the same catalog:

- **Each product gets a unique primary keyword.** No two products should target the exact same primary keyword.
- **Map keywords before writing.** Create a keyword-to-product assignment table. Review for overlaps.
- **Differentiate by modifier:** If two products are similar, differentiate keywords by size, color, use case, or audience. "Women's running shoes" vs "Men's trail running shoes."
- **Use the Keyword Integration Map** (from the output template) to document placement and prevent accidental overlap.
- **Check existing catalog first.** Before assigning keywords to a new product, verify no existing product already targets that term.

## Content Quality Signals

Search algorithms increasingly evaluate content quality beyond keyword matching.

### Readability
- Write at a 6th-8th grade reading level for general consumer products
- Shorter sentences (≤20 words) perform better in bullets
- Avoid jargon unless the target audience expects it (e.g., technical products for professionals)
- Use active voice: "Cuts cooking time in half" not "Cooking time is reduced by half"

### Specificity over generality
- "Holds 24 oz" outperforms "Large capacity"
- "Machine washable at 140°F" outperforms "Easy to clean"
- Numbers, measurements, and concrete details signal quality to both algorithms and buyers

### Freshness
- Update product descriptions quarterly or when significant changes occur
- Refresh seasonal keywords before relevant shopping periods
- Update competitor-facing language when market positioning shifts

### User-generated content integration
- Reference review themes in descriptions: "Rated 4.8 stars for durability by 2,000+ customers"
- Address common review complaints proactively in the description
- Use the language buyers use in reviews — it's natural keyword research

## Common SEO Mistakes in Product Descriptions

1. **Keyword stuffing** — Repeating the same keyword 8+ times in a 200-word description. Hurts readability and may trigger penalties.
2. **Duplicate descriptions** — Using the same description across multiple products or platforms. Each listing needs unique copy.
3. **Ignoring backend fields** — Amazon backend search terms left empty means lost keyword coverage.
4. **Title keyword at the end** — Placing the primary keyword at position 150+ in a 200-character title. It gets truncated on mobile and loses SEO weight.
5. **Generic alt text** — "Product image" instead of "Stainless steel French press coffee maker 34oz." Lost image search traffic.
6. **Optimizing for the wrong keyword** — Targeting a high-volume keyword that doesn't match purchase intent. "Coffee" has volume; "French press coffee maker 34 oz stainless steel" has buyers.
7. **Neglecting mobile formatting** — Long paragraphs that work on desktop become walls of text on mobile. Use short paragraphs and bullets.
8. **Static keyword strategy** — Setting keywords once and never updating. Search trends shift seasonally and competitively.
