## HTML格式说明

### 输出HTML报告的样式要求

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>商业模式分析报告</title>
    <style>
        body {
            font-family: "PingFang SC", "Microsoft YaHei", sans-serif;
            line-height: 1.8;
            max-width: 900px;
            margin: 0 auto;
            padding: 40px 20px;
            background: #0a1628;
            color: #e0e0e0;
        }
        h1 { color: #0096ff; border-bottom: 2px solid #0096ff; padding-bottom: 10px; }
        h2 { color: #4da6ff; margin-top: 30px; }
        h3 { color: #80bfff; }
        table { border-collapse: collapse; width: 100%; margin: 15px 0; }
        th, td { border: 1px solid #1e4a7a; padding: 10px; text-align: left; }
        th { background: #1e4a7a; color: #fff; }
        .highlight { background: #0d2137; padding: 15px; border-radius: 5px; }
        .rating-A { color: #00ff88; font-weight: bold; }
        .rating-B { color: #ffdd00; font-weight: bold; }
        .rating-C { color: #ff8800; font-weight: bold; }
        .rating-D { color: #ff4444; font-weight: bold; }
    </style>
</head>
<body>
    <!-- 内容同上Markdown格式的结构 -->
</body>
</html>
```
