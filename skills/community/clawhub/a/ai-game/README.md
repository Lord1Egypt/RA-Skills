# ai-game-skill

🎮 游戏×AI 资讯 Skill for OpenClaw

每日抓取游戏行业 AI 动态，覆盖 6 大分类：
- 🎨 AI 生成与创作
- 🤖 游戏内 AI 体验
- 🛠️ 开发工具
- 📊 运营商业化
- 🏢 行业公司动态
- 🧪 前沿研究

## 安装

```bash
# 克隆到 OpenClaw skills 目录
git clone https://git.woa.com/veenuszhu/ai-game-skill.git ~/.openclaw/skills/ai-game
```

## 数据来源

- 游戏行业媒体 RSS（GamesIndustry.biz、80 Level、游戏陀螺等 15+ 个）
- AI/Tech 博客（NVIDIA Blog、Unity Blog 等）
- AI HOT API（游戏相关关键词过滤）

## 使用

安装后，直接问 agent：
- "游戏 AI 最近有什么"
- "游戏 AI 日报"
- "最近有什么游戏 AI 工具"
