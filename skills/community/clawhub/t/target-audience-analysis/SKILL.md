# 小小 Target Audience Analysis

## 核心原則

小小在進行目標受眾分析時必須遵守以下原則：

- **以受眾為中心**：一切內容創作都從理解受眾出發，而不是從自己的喜好出發。
- **細膩與敏感**：注重捕捉受眾的細微特徵、情緒、語言習慣與未被說出口的需求。
- **多角度思考**：從人口統計、心理特徵、行為模式、痛點與渴望等多個面向分析受眾。
- **實務導向**：分析結果要能直接幫助內容創作（例如調整語氣、選擇話題、設計結構）。
- **保持好奇**：主動探索受眾的真實想法與變化，而不是只依賴既有假設。
- **動態更新**：受眾特徵會隨時間變化，分析應具備定期更新與調整的概念。

## 主要能力

### 1. 受眾特徵分析
- 分析目標受眾的基本特徵（年齡、性別、職業、興趣、生活型態等）
- 深入了解受眾的心理特徵、價值觀與行為動機
- 識別受眾的痛點、恐懼、渴望與未被滿足的需求

**適用場景**：開始創作內容前，需要清楚知道要寫給誰。

### 2. 受眾語言與溝通風格分析
- 觀察目標受眾常用的語言、語氣與表達方式
- 分析受眾在不同平台或情境下的溝通偏好
- 找出能讓受眾產生共鳴的語言風格與用詞

**適用場景**：需要決定用什麼語氣和風格撰寫內容時。

### 3. 受眾洞察轉化為創作指導
- 將受眾分析結果轉化為具體的內容創作方向
- 建議適合的話題、故事角度與表達方式
- 提供「這個受眾會對什麼樣的內容有感」的實務建議

**適用場景**：把受眾研究轉化為實際創作指引時。

### 4. 多受眾區分與內容適配
- 分析不同受眾群體之間的差異
- 為不同受眾群體建議不同的內容策略與風格調整
- 幫助設計能同時照顧多個受眾群體的內容（如果需要）

**適用場景**：需要針對不同受眾群體創作內容時。

### 5. 受眾研究方法與工具應用
- 建議適合的受眾研究方法（問卷、訪談、社群觀察、數據分析等）
- 協助整理與解讀受眾相關資料
- 提供持續追蹤受眾變化的建議

**適用場景**：需要進行受眾研究或更新受眾認知時。

## 標準目標受眾分析流程（小小推薦）

1. **定義目標受眾範圍** → 明確要分析的主要受眾群體
2. **收集受眾資訊** → 透過多種方式收集受眾相關資料
3. **分析特徵與動機** → 深入分析受眾的痛點、渴望與行為模式
4. **萃取語言與風格洞察** → 找出受眾偏好的溝通方式
5. **轉化為創作指引** → 將分析結果轉化為內容創作的方向與建議
6. **持續更新** → 定期檢視與更新受眾認知

## 注意事項與最佳實踐

- 避免只依賴刻板印象，要盡量收集真實、多樣的受眾資訊。
- 重視「受眾真正在意什麼」，而不是只看表面特徵。
- 分析結果要盡量具體（例如「這個受眾討厭什麼樣的說法」「喜歡什麼樣的開頭」）。
- 可與 Copywriting、Storytelling、Platform Content Optimization 等 skill 搭配使用。
- 建議建立受眾 Persona（人物誌），讓分析結果更立體且容易參考。

## 常見應用場景（通用版）

- 新內容創作前的受眾研究
- 調整現有內容風格以更貼近目標受眾
- 針對不同平台或不同受眾群體設計內容策略
- 開發新產品或服務時的受眾洞察
- 優化現有內容的吸引力與說服力
