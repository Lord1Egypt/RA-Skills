# 感染性心内膜炎病理形成机制评估

## 何时使用
- 患者有先天性心脏病（如室间隔缺损、动脉导管未闭）或获得性心瓣膜病
- 近期接受拔牙、扁桃体摘除术等口腔/上呼吸道侵入性操作
- 存在口腔感染或其他可能导致菌血症的情况
- 需判断赘生物形成风险及预防性抗生素使用指征

## 执行步骤
1. **确认定植菌来源**：识别是否存在口腔或上呼吸道常见定植菌（如甲型溶血性链球菌、金黄色葡萄球菌）。
2. **评估防御功能状态**：判断患者是否处于机体防御低下状态（如急性感染、术后恢复期）。
3. **核实心内膜基础病变**：确认是否存在心瓣膜、心内膜或大血管内膜的病理改变或先天性缺损。
4. **分析血流动力学**：评估心脏或大血管两侧是否存在显著压力差，导致高速湍流冲击内膜。
5. **定位潜在赘生物部位**：根据心脏缺陷类型确定压力较低侧的易损区域：
   - 室间隔缺损 → 缺损右缘、三尖瓣隔叶、肺动脉瓣
   - 动脉导管未闭 → 肺动脉侧
   - 主动脉瓣关闭不全 → 左心室侧
6. **综合判断IE启动风险**：若同时满足基础病变、菌血症诱因和内膜损伤机制，则提示高风险，应考虑预防性抗生素干预。

> **输出建议**：对高风险患者（尤其先天性心脏病患儿），在口腔或侵入性操作前应预防性使用抗生素，以阻断感染性心内膜炎病理进程。
