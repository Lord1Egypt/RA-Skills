# 儿科学的核心特点与儿科医师必备素质

## 何时使用本技能
- 对0–18岁儿童进行病史采集、体格检查或制定诊疗方案时
- 面临将成人医学逻辑套用于儿童的倾向时
- 需向医学生或初级医生传授儿科临床思维时
- 评估儿科照护质量或医师胜任力时

## 如何执行

### 第一步：识别儿童区别于成人的六大核心特点
1. **器官功能**：注意年龄依赖性变化（如新生儿右心室重于左心室，2周后逆转；心率随年龄下降）。
2. **免疫能力**：识别正常免疫指标范围（如新生儿白细胞及中性粒比例高于成人；婴儿胸腺未退化，胸片可见胸腺影）。
3. **疾病反应**：区分生理性阳性体征与病理表现（如克尼格征在2–3月内可阳性，巴宾斯基征2岁前可阳性）。
4. **药物代谢**：严格按年龄、体重或体表面积调整剂量与给药方式。
5. **心智与运动发育**：理解影像学正常变异（如婴儿MRI显示脑外间隙增宽、脑沟回较浅属正常）。
6. **情绪反应方式**：认识到儿童表达受限，高度依赖照护者提供信息与情感支持。

### 第二步：应用儿科特有疾病谱知识
- **呼吸系统**：优先考虑支气管肺炎而非大叶性肺炎。
- **心血管系统**：关注先天性心脏病而非冠心病。
- **其他**：警惕先天畸形、遗传代谢病等在儿童中的高发性。

### 第三步：践行儿科医师三大必备素质
1. **专业能力**：结合最新循证指南、临床经验与个体发育阶段，制定个体化诊疗方案。
2. **实践能力**：
   - 通过照护者有效采集病史
   - 执行适合年龄的规范体格检查
   - 熟练掌握儿科操作技术
   - 快速识别危重症征象
3. **职业精神**：以患儿及家庭利益为中心，展现同情心、关爱、奉献精神，并敏感体察家属情绪需求。

> **关键提醒**：切勿将儿童视为“小大人”，所有诊疗决策必须基于其发育阶段特征。
