# A+模块参考表

## 按类目匹配模块组合

### 服装类 (Apparel)

**推荐模块组合**：
1. 多图轮播模块 (Image Carousel)
2. 尺寸指南模块 (Size Guide)
3. 穿搭场景模块 (Lifestyle Grid)
4. 材质说明模块 (Material Highlights)
5. 颜色选择模块 (Color Swatches)

**文案风格**：
- 强调时尚感和搭配建议
- 使用场景化的描述
- 突出舒适度和材质特点

**合规要点**：
- 尺寸数据需准确
- 避免真人对比图暗示身材改变
- 颜色名称使用标准色号

---

### 电子类 (Electronics)

**推荐模块组合**：
1. 对比图表模块 (Comparison Chart)
2. 功能拆解模块 (Feature Highlights)
3. 视频嵌入位 (Video Placement)
4. 规格参数模块 (Technical Specs)
5. 使用场景模块 (Usage Scenarios)

**文案风格**：
- 技术细节丰富
- 数据驱动
- 强调创新点和差异化

**合规要点**：
- 技术参数需准确
- 避免绝对化用词（如"最好"、"第一"）
- 比较数据需标注来源

---

### 家居类 (Home & Kitchen)

**推荐模块组合**：
1. 场景展示模块 (Lifestyle Hero)
2. 对比图表模块 (Before/After Comparison)
3. 安装指南模块 (Installation Guide)
4. 尺寸图解模块 (Dimension Diagram)
5. 材质说明模块 (Material Quality)

**文案风格**：
- 强调生活品质提升
- 突出实用性和便利性
- 温馨家庭氛围

**合规要点**：
- 场景图需真实
- 尺寸数据需精确
- 避免暗示产品具有医疗效果

---

### 美妆类 (Beauty)

**推荐模块组合**：
1. 成分展示模块 (Ingredients Display)
2. 使用步骤模块 (How to Use)
3. Before/After对比模块
4. 认证标志模块 (Certifications)
5. 用户反馈模块 (Customer Highlights)

**文案风格**：
- 强调自然成分
- 突出使用体验
- 温和不刺激的承诺

**合规要点**：
- 不能声称FDA批准（除非已获批）
- 不能使用医疗术语
- Before/After需真实无修图

---

4. 尺寸指南模块 (Size Selection)

**文案风格**：
- 强调宠物安全和健康

## Premium A+专属模块

### 可交互轮播 (Shoppable Carousel)

**适用场景**：
- 服装多色展示
- 套装内含物品展示
- 产品角度细节

**规范要求**：
- 图片尺寸一致
- 热点标注清晰
- 跳转链接有效

### 视频嵌入位 (Video Placement)

**适用场景**：
- 品牌故事
- 产品演示
- 使用教程

**规范要求**：
- 视频时长15-30秒
- 无音视频同步问题
- 品牌元素可展示

### 对比图表 (Comparison Chart)

**适用场景**：
- 新旧版本对比
- 与普通产品对比
- 规格参数对比

**规范要求**：
- 数据来源标注
- 禁止虚假对比
- 格式清晰易读

### 问答摘要 (Q&A Summary)

**适用场景**：
- 整合用户常见问题
- 解答购买疑虑
- 补充产品信息

**规范要求**：
- 问题来源真实
- 回答准确专业
- 更新及时

---

## 多语言本地化要点

### 德语 (German - DE)

- 使用"Lebensmittelecht"而非直接翻译
- 强调认证：LFGB > FDA（欧洲市场）
- 技术术语准确
- 避免口语化表达

### 日语 (Japanese - JP)

- 使用敬语和礼貌体
- 强调"使いやすさ"（易用性）
- 细节描述详尽
- 包装和礼盒概念重要

### 法语 (French - FR)

- 优雅浪漫的表达
- 生活方式导向
- 强调设计感和美感
- 避免直接对比

### 西班牙语 (Spanish - ES)

- 热情实用的表达
- 家庭场景强调
- 性价比突出
- 文化活动关联

### 意大利语 (Italian - IT)

- 设计与品质并重
- 美学表达丰富
- 手工艺传统
- 强调独特品味

## 合规检查清单

### 图片规格
- [ ] 分辨率：300 DPI
- [ ] 最小尺寸：1000 x 1000 像素
- [ ] 文件格式：PNG 或 JPEG
- [ ] 无拉伸变形

### 文字规范
- [ ] 无绝对化用词（最好、第一、唯一）
- [ ] 无未经授权的Logo或商标
- [ ] 无误导性描述
- [ ] 无虚假对比数据

### 模块规范
- [ ] 符合Premium A+模块要求
- [ ] 模块组合符合类目推荐
- [ ] 跳转链接有效
- [ ] 无违规元素叠加

### 描述准确性
- [ ] 技术参数与产品一致
- [ ] 认证信息真实有效
- [ ] 价格信息未出现
- [ ] 库存状态未提及
