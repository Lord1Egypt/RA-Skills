# 阿昔洛韦类抗疱疹病毒药的选择性激活机制

## 何时使用
- 用户询问阿昔洛韦为何对疱疹病毒具有选择性毒性
- 需要解释为何该药对CMV无效或对TK缺陷型病毒耐药
- 临床场景涉及HSV脑炎、生殖器疱疹或带状疱疹的药物作用原理

## 执行步骤
1. 确认目标病毒是否为HSV-1、HSV-2、VZV或部分有效的EBV；若为CMV，则本机制不适用（应使用更昔洛韦）。
2. 验证感染细胞是否表达病毒胸苷激酶（TK）；若TK缺失或突变，则阿昔洛韦无法被有效激活。
3. 药物（阿昔洛韦或伐昔洛韦）进入体内后分布至靶组织。
4. 在感染细胞内，病毒TK将阿昔洛韦单磷酸化为ACV-MP。
5. 宿主细胞激酶进一步将其转化为活性形式三磷酸阿昔洛韦（ACV-TP）。
6. ACV-TP通过以下方式抑制病毒：
   - 竞争性抑制病毒DNA聚合酶
   - 掺入病毒DNA链并因缺乏3'-OH而终止链延伸
7. 若出现耐药（如TK突变），考虑改用无需TK活化的替代药物（如膦甲酸）。

> 注意：口服阿昔洛韦生物利用度低，临床常选用前药伐昔洛韦以提高吸收。
