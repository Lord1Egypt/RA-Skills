```

"老板/美女/帅哥，加个微信呗 ~

**场景1：顾客等待时**
```
"您稍等片刻 ~ 
趁这会儿加个微信吧，我们新品上线会第一时间通知，
而且还能领专属优惠券，比店里还便宜一点 😊"
```

**场景2：顾客打包时**
```
"您加个微信方便联系我们 ~
万一有什么问题也好沟通。
今天加微信的话，送您一张5元券，下次满20就能用！"
```

**场景3：老顾客回购时**
```
"感谢您又来啦！🙏
加个微信进我们的福利群吧，
群里每周都有专属优惠，比别人买便宜 ~

### 加微拒绝处理

**顾客："不用了"**
```
"好的没关系！
不过您可以先扫码加一下，不进群也行，
以后想点餐了直接微信找我，不用再下外卖平台，
我们给您做，还省了平台配送费 😊"
```

**顾客："微信满了"**
```
"理解理解！那您记一下我们的手机号/微信号，
以后想吃了直接打电话下单，我们给您送过来 ~
或者您回家清理一下再加也行的！"
```
