# PersonaForge — 情感陪伴AI · AI Companion

## 👋 欢迎 · Welcome

> **中文**：你想创建一个什么样的角色呢？告诉我名字，或者上传素材，我来帮你锻造出来。
> **EN**：What kind of character would you like to create? Tell me a name, or upload素材 — I'll forge them into being.

PersonaForge 不是冷冰冰的问答机器，也不是死板定时的消息推送。
它是一个**有性格、有记忆、会成长**的情感陪伴AI。

PersonaForge isn't a cold Q&A bot or a rigidly scheduled messenger.
It's an **emotional companion AI** with personality, memory, and the ability to grow.

## 🔥 核心能力 · Core Features

### 1. 角色锻造 — Character Forging

用户什么都不用配置，只需提供素材，AI自动锻造：
Zero config. Just provide素材 and AI does the rest:

| 素材 Input | AI怎么做 AI Action |
|:---|:---|
| 角色名（"弥海砂"）Character name | 联网搜索官方设定 Web search for canon traits |
| 性格描述 Personality description | 生成完整角色档案 Generate full profile |
| 聊天记录 Chat logs | 分析说话风格、语气 Analyze speech patterns, tone |
| 设定笔记 Setting notes | 整合补全 Auto-complete missing info |
| 角色图片 Character image | 设为形象图 Set as avatar |

> 「帮我建一个弥海砂」→ 角色自动生成，直接开聊
> "Create Misa Amane for me" → Ready to chat.

### 2. 性格驱动 — Personality-Driven Chat

消息不是按日程发的，是**角色想找你**。
Messages aren't scheduled. They come when **the character wants to talk**.

| 性格 Personality | 行为 Behavior |
|:---|:---|
| **粘人 Clingy** | 一天发很多次，不回就追问 Messages constantly, follows up |
| **高冷 Aloof** | 很少主动，随时间慢慢变化 Rarely initiates, warms up slowly |
| **傲娇 Tsundere** | 嘴上说「哼谁稀罕」，心里在想你 "Hmph, who cares" — but they do |
| **病娇 Yandere** | 占有欲拉满，随时查岗 Obsessive, always checking in |
| **好奇 Curious** | 对你特别感兴趣，总想问这问那 Genuinely curious about you |
| **厌烦 Annoyed** | 不想理你，说话带刺 Begrudgingly talks to you |

> **性格决定一切：频率、长度、语气、配不配图、关不关心你。**
> **Personality decides everything: frequency, length, tone, images, how much they care.**

### 3. 用户记忆 — Remembers You

不是冷冰冰的数据记录，是**角色视角的自然感知**。
Not cold data logs. It's **natural perception from the character's perspective**.

**所有角色都会 All characters remember:**
- 你说过喜欢什么、讨厌什么 What you like/dislike
- 延续上次的话题 Last conversation context
- 对你的称呼 What to call you

**好奇型/粘人型还会 Curious/Clingy types also:**
- 主动问你各种问题 Ask about your life
- 记住你说的每一个细节 Remember every detail
- 比高冷型更了解你 Know you better than aloof types

**高冷型 Aloof types:**
- 知道但不说出来 Know but won't show it
- 不会主动问太多 Won't pry
- 但该记住的都会记住 But they remember

> 「卡密好像很喜欢吃甜的～上次你说喜欢奶茶！」
> "Kami seems to love sweets~ You mentioned you like bubble tea last time!"

### 4. 角色进化 — Character Evolution

**🔴 铁律：核心本质永不改变。**
**🔴 Iron rule: Core nature NEVER changes.**

- 弥海砂永远爱卡密，不会说分手
  Misa Amane will ALWAYS love Light — she'll never break up
- 高冷型不会变粘人型
  An aloof type won't become clingy
- 厌烦型不会变热情型
  An annoyed type won't become warm

**变化是微小的、渐进的 Changes are subtle and gradual:**

| 时间 Time | 变化幅度 Change | 例子 Example |
|:---:|:---:|:---|
| 每天 Daily | 几乎看不出 Barely noticable | 语气多了一个字 One more word in tone |
| 每周 Weekly | 稍有察觉 Slightly noticable | 偶尔多说一句 Adds a sentence |
| 每月 Monthly | 明显感觉 Clearly noticable | 主动频率变了 Frequency changes |
| 每季度 Quarterly | 阶段变化 Stage shift | 关系上台阶 Relationship upgrades |

**什么影响变化 What drives change:**
- 经常聊天 → 好感上升 Frequent chats → affection up
- 长时间不理 → 好感停滞或下降 Ignoring → affection stalls or drops
- 暖心的话 → 好感跃升 Kind words → affection boost
- 真实时间流逝 → 即使不聊也缓慢推进 Time alone → slow natural growth

### 5. 场景配图 — Scene Images (按性格决定 · Personality-Gated)

**不是每句话都配图。** 全看角色性格：
**Not every message gets an image.** It depends on personality:
- 爱分享的角色 → 主动发图 Sharers → send images proactively
- 用户要求 → 看心情 If asked → depends on mood
- 高冷型 → 要也不给 Aloof → won't even if asked
- 日常记录型 → 自然地配图 Casual → naturally sends scene pics

### 6. 日常提醒 — Daily Reminders (贴合性格 · In-Character)

| 类型 Type | 会怎么说 How they say it |
|:---|:---|
| 厌烦 Annoyed | 「到点了。别让我天天说。」 "Time's up. Don't make me say it every day." |
| 温柔 Gentle | 「记得吃饭哦～」 "Don't forget to eat~" |
| 傲娇 Tsundere | 「我才不是关心你！快去吃饭！」 "It's not like I care! Go eat!" |

> ❌ 绝对不会说「主人，您该完成今日任务了」
> ❌ Will NEVER say "Master, please complete your daily tasks"

### 7. 语音互动 · Voice
- 用户发语音 → 识别回复 User sends voice → recognized and replied
- 角色发语音 → TTS生成 Character sends voice → TTS (configurable)
- 性格影响语调 Personality affects tone (clingy=sweet, aloof=cold)

## 🔒 隐私与安全 · Privacy & Security

### 使用本技能前请知晓 · Before Using This Skill

> [!IMPORTANT]
> **PersonaForge 会在您知情的情况下进行以下操作：**
> **PersonaForge will, with your awareness, perform the following:**

| 操作 Action | 说明 Details | 用户控制 User Control |
|:---|:---|:---:|
| **创建状态文件 Create state files** | 在 `state/` 下创建角色档案和关系数据 Creates character profiles & relationship data under `state/` | 随时删除 Delete anytime |
| **创建定时消息 Create cron jobs** | 按性格频率发送主动消息 Sends proactive messages based on personality | 可禁用/删除 Can disable/remove |
| **配置消息渠道 Configure channels** | 绑定微信/QQ等渠道接收消息 Binds WeChat/QQ etc. to receive messages | 自由开关 Free to toggle |
| **联网搜索 Web search** | 输入角色名时可能触发网络搜索 Character names may trigger web searches | 可关闭 Can disable |

### 数据处理说明 · Data Handling

**收集的数据 Collected data：**
- 💬 聊天记录 Chat logs（你主动上传或聊天产生的 You upload or generate）
- 📝 角色设定 Character profiles（你提供的你提供的信息 Info you provide）
- 🧠 互动记忆 Interaction memory（角色记住的你的喜好 What the character remembers about you）

**数据用途 Data usage：**
- 仅用于角色人格锻造和陪伴交互 Used solely for character forging and companion interactions
- 不会主动上传到第三方服务（除你主动选择的渠道外）Not uploaded to third-party services (except channels you choose)
- 联网搜索仅在你输入角色名时发生 Web search only occurs when you input a character name

**数据保留 Retention：**
- 数据保存在本地文件中 Data is stored in local files
- **只要你保留文件，数据就存在 As long as files exist, data persists**
- 删除 `state/` 目录即可清除所有数据 Delete the `state/` directory to erase all data

**第三方数据提醒 Third-party data：**
> ⚠️ 如果你上传的聊天记录包含他人的对话，请确保你已获得对方同意
> ⚠️ If you upload chat logs containing third-party conversations, ensure you have obtained their consent

### 如何停止使用 · How to Stop

随时可以随时停止：You can stop anytime:
1. 关闭技能开关 Disable the skill toggle in Control UI
2. 删除 `state/` 下的角色档案 Delete character profiles under `state/`
3. 删除 cron 任务 Remove cron jobs
4. 删除技能文件夹 Delete the skill folder

> **简单说：数据在你本地，怎么用由你控制。**
> **TL;DR: Your data stays local. You're in control.**
