# Product Detail Page Generator

## Description
AI-powered H5 product detail page generator. Upload product images, get a complete mobile-optimized detail page in 5-10 minutes — auto image recognition, chapter planning, copywriting, AI-generated scenes, and layout.

## Features
- 🖼️ AI Image Recognition — Auto-detect category, industry, and key selling points
- 📋 Smart Chapter Planning — 3-5 structured sections covering materials/design/scenes/customization/warranty
- ✍️ Professional Copywriting — Headlines, subheadings, selling points with industry-specific terminology
- 🎨 AI-Generated Scenes — High-quality product scene images for each chapter
- 📱 Mobile Optimized — 750px width long page, perfect for vertical scrolling
- 🔗 Instant Delivery — Share link + QR code upon publishing
