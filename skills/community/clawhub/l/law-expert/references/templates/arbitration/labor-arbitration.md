# 劳动仲裁申请书

## 适用场景
与用人单位发生劳动争议（拖欠工资、违法辞退、未缴社保、工伤待遇等），协商无果，申请劳动仲裁前置程序。

## 提交机构
用人单位所在地或劳动合同履行地的劳动人事争议仲裁委员会

## 模板正文

```
                    劳动仲裁申请书

申请人：姓名______，性别______，身份证号：__________________
        住址：__________________，联系电话：____________

被申请人：______公司/单位
          统一社会信用代码：________________
          住所地：____________________________
          法定代表人：______，职务：______
          联系电话：________________________

仲裁请求：
1. 请求裁决被申请人支付拖欠工资______元
   （______年______月至______年______月）；
2. 请求裁决被申请人支付违法解除劳动合同赔偿金______元
   （计算方式：______个月×月工资______元×2）；
3. 请求裁决被申请人补缴______年______月至______年______月
   的社会保险/支付未缴社保造成的损失______元；
4. 其他：____________________________。

事实与理由：
申请人于______年______月______日入职被申请人处，
担任______岗位，约定月工资______元。

______年______月______日，被申请人以______为由
单方面解除劳动合同/拖欠工资/未缴纳社保……（详细陈述事实）。

被申请人的行为违反《中华人民共和国劳动合同法》第______条
以及____________等相关规定。

为此，特依法申请劳动仲裁。

此致
____________劳动人事争议仲裁委员会

附：
1. 本申请书副本______份；
2. 身份证复印件、劳动合同、工资流水等证据材料；
3. 证据清单。

                                    申请人（签名）：
                                    ______年______月______日
```

## 注意事项
- **关键填写点**：准确写明入职时间、岗位、月工资（以银行流水为准）；区分拖欠工资、违法解除赔偿金（2N）、经济补偿金（N）的计算；证据包括劳动合同、工资单、银行流水、微信聊天记录等
- **常见错误**：未经劳动仲裁直接向法院起诉（劳动争议仲裁前置）；仲裁请求金额未附计算过程；遗漏工时记录、加班费请求；超过1年仲裁时效
- **份数要求**：正本1份 + 被申请人人数副本 + 自留1份
- **时效提示**：劳动仲裁时效为知道或应当知道权利被侵害之日起1年内
- **免费用**：劳动仲裁不收取费用
