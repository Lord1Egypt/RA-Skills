# 更新日志

## v3.4 — 2026-06-25（含当天所有更新）

### CDP 直连方案 + 每步自检重试 + 飞书融合 + 行索引修复

#### publisher.py
- **标题匹配行索引** — 不再固定使用 `trs[0]`（第1行），改为通过商品标题匹配找到正确的行
- **弹窗Vue异步等待** — 配置发货弹窗 body 最长等 15 秒确认渲染完成
- **表格加载重试** — 发布后跳转商品列表，改为 5 次 × 2 秒重试
- **状态显示优化** — 最终验证用 ✅/❌ 图标
- **CDP 直连方案** — 使用 Python websockets 直连 Edge CDP，绕过 Playwright 内置 ws 库兼容性问题，适配 Edge 149+
- **每步自检重试** — 新增 `_step_with_retry()` 机制，标题、图片、描述、城市、店铺、单规格、售价、库存、包邮每步操作后自检，失败自动重试3次
- **Vue 响应式兼容** — 新增 `_set_input_vue_compat()` 使用 `nativeInputValueSetter` + 多种事件触发，解决 Element UI 输入框 placeholder 为空时的查找问题
- **双模式输入框查找** — `_check_field_value()` 先按 placeholder 查找，找不到再按 `el-form-item__label` 文本查找
- **表单校验** — `_check_form_errors()` 发布前检查表单错误，`_check_publish_success()` 发布后等待跳转（最长30秒）
- **发布按钮重试** — 最多重试3次，发现售价/库存错误自动重填
- **自动发货二次触发** — 保存后开关未变蓝 → 自动再点开关 + 重新保存（最多3次）
- **售罄上架二次触发** — 同理，没变蓝就再点开关 + 重新确定（最多3次）

#### 问题记录
1. **is_blue 检测行索引错误** — 修复：标题匹配行索引
2. **弹窗Vue异步加载** — 修复：增加 body 长度轮询等待

## v3.3 — 2026-06-22

### 整合沛神优化

#### batch_parser.py
- **新增新版格式支持** — 自动识别「一、发布时间 / 二、商品文案 / 三、店铺 / 四、网盘发货文案」格式
- **已发布标记跳过** — 解析时遇到含「已发布」的商品块自动跳过并打印 [SKIP]
- **日期解析** — 提取发布时间字段（M.D格式）
- **商品级店铺** — 每个商品块可独立指定店铺（三、店铺），自动解析店铺名→店铺号
- **旧版向后兼容** — 原有「数字.」格式继续支持，自动检测格式
- **按日期分组拆分** — 新增 `--by-date` 参数，按日期分组到day目录
- **店铺映射表** — 新增 `SHOP_MAP` 字典，支持多种店铺名称写法
- **统计输出** — 拆分完成后打印各店铺商品数量统计

#### auto_publish.py
- **日期过滤** — 读取系统当前日期，只处理发布时间=今天的商品（ENABLE_DATE_FILTER开关）
- **已上架记录** — 写入 `已上架记录.txt`，检查「日期+标题+店铺」三要素防重复（ENABLE_RECORD_DEDUP开关）
- **已发布标记写回** — 上架成功后自动在总上货.txt中追加「已发布」标记（ENABLE_PUBLISHED_MARK开关）
- **高级功能开关** — 脚本顶部三个布尔开关（ENABLE_DATE_FILTER / ENABLE_PUBLISHED_MARK / ENABLE_RECORD_DEDUP）
- **dry-run模式** — 新增 `--dry-run` 参数预览不上架
- **跳过统计** — 汇总时单独统计跳过数量

#### publisher.py
- **dispatchEvent选店铺** — 使用 `MouseEvent('click')` 绕过 Element UI teleport 不可见问题
- **getComputedStyle检测弹窗** — 替代 `offsetParent` 检测，对 fixed 定位弹窗更可靠
- **2人小刀二级兜底** — 常规弹窗检测失败时，先 force click 再 Vue emit 兜底
- **自动发货弹窗检测** — 先用 getComputedStyle 确认弹窗已打开再操作
