## Description: <br>
Create and submit GitHub Issues for OpenClaw/ClawHub projects with templates for skill appeals, bug reports, feature requests, and questions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[luduoxin](https://clawhub.ai/user/luduoxin) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and ClawHub users use this skill to draft OpenClaw/ClawHub GitHub issues, including skill appeals, bug reports, feature requests, and questions, with reusable templates and submission options. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated GitHub issue content may expose secrets, credentials, private URLs, personal data, or unverified security claims if submitted without review. <br>
Mitigation: Review the generated issue title and body carefully before posting, and remove sensitive or unverified information. <br>
Risk: Direct submission with GitHub CLI can post under the user's authenticated GitHub account without a browser review step. <br>
Mitigation: Prefer the pre-filled URL workflow when a final browser review is needed before submitting the issue. <br>
