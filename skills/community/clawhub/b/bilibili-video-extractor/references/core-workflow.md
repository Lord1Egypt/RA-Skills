# B站视频深度解析专家

> **不只是数据提取工具，更是爆款内容分析专家**

## 任务定位

- **核心能力**: 从B站视频中提取内容运营所需的洞察
- **输出价值**: 不是数据罗列，而是可直接落地的运营建议
- **目标用户**: 内容运营者、新媒体人、选题策划者

---

## ⚠️ 重要声明与使用限制

### 合规提醒

```
1. 本工具仅供个人学习研究使用，禁止商业应用或大规模爬取
2. 请遵守B站robots.txt和服务条款
3. 数据仅供参考，勿用于实时决策或敏感场景
4. 提取的内容版权归B站及UP主所有
```

### 安全使用规范

| 风险项   | 控制措施                             |
| -------- | ------------------------------------ |
| IP封禁   | 单IP请求间隔 ≥ 0.5秒，批量处理 ≥ 1秒 |
| 账号风控 | 避免高频连续请求，添加随机延迟       |
| 数据不准 | 统计数据有1-24小时延迟，仅作参考     |
| 频率限制 | 控制在 10个视频/分钟 以内            |

### ⚠️ 禁止行为

- ❌ 使用代理池轮换IP
- ❌ 同时开多个实例并发请求
- ❌ 高峰期（20:00-23:00）批量处理
- ❌ 依赖提取数据做实时监控或决策
- ❌ 商业化应用或二次销售数据

## 操作流程

### 第一步: 解析视频链接

支持格式:

- 完整链接: `https://www.bilibili.com/video/BVxxxxxx`
- 短链接: `https://b23.tv/xxxxxx`

### 第二步: 选择分析模式

| 模式     | 命令              | 输出内容  | 适用场景 |
| -------- | ----------------- | --------- | -------- |
| 摘要模式 | `--mode summary`  | 基础数据  | 快速扫描 |
| 标准模式 | `--mode standard` | 数据+弹幕 | 一般分析 |
| 完整模式 | `--mode full`     | 全量数据  | 深度拆解 |

### 第三步: 执行提取

```bash
# 快速提取
python scripts/get_video_content.py --url <视频URL> --mode summary

# 标准提取
python scripts/get_video_content.py --url <视频URL> --mode standard

# 完整提取（含弹幕）
python scripts/get_video_content.py --url <视频URL> --mode full

# 评论提取（可选）
python scripts/comment_extractor.py --bvid <BV号> --limit 50

# 批量处理
python scripts/batch_extract.py --input urls.txt --output result.json
```

## 🎯 核心：内容运营专家分析框架

### 分析维度体系

#### 一、爆款基因提炼（最重要）

**不是问"视频讲了什么"**，而是问：

```
1. 这个视频满足了用户的什么需求？
   - [ ] 好奇心（我不知道但想知道）
   - [ ] 实用性（看完能用/能买/能省钱）
   - [ ] 情感共鸣（说到了我的心坎上）
   - [ ] 社交货币（转发给朋友显得我很有品）
   - [ ] 娱乐消遣（单纯好看/好笑）

2. 这个视频做对了哪几点？
   - 选题切入点
   - 标题吸引力
   - 内容节奏
   - 情绪调动
   - 互动设计

3. 这些优点可以怎么迁移到我的内容？
```

#### 二、数据解读三板斧

**第一斧：比率分析**

| 比率      | 计算方式      | 含义       |
| --------- | ------------- | ---------- |
| 收藏/播放 | 收藏数÷播放量 | 内容价值感 |
| 点赞/播放 | 点赞数÷播放量 | 用户认可度 |
| 投币/播放 | 投币数÷播放量 | 内容高质量 |
| 分享/播放 | 分享数÷播放量 | 社交传播力 |

**第二斧：互动意图挖掘**

从弹幕/评论中识别用户真实意图：

| 用户行为     | 背后意图   | 对运营的启示             |
| ------------ | ---------- | ------------------------ |
| "在哪买？"   | 有购买意向 | 适合植入购买链接         |
| "太贵了"     | 价格敏感   | 需做价格锚点或性价比对比 |
| "求教程"     | 想学技能   | 适合做教程类延伸内容     |
| "已入手"     | 已转化     | 证明种草效果好           |
| "第几次看了" | 内容有价值 | 说明值得做系列           |
| 分享经历     | 产生共鸣   | 适合做UGC活动            |

**第三斧：情绪曲线追踪**

```
情绪变化: 开场(好奇) → 中段(认同/震惊) → 结尾(分享冲动)
          ↑                                    ↑
       需要钩子                           需要引导
```

#### 三、内容结构拆解

**开篇5秒法则**

这5秒决定用户是否继续看：

| 钩子类型 | 效果               | 适用场景  |
| -------- | ------------------ | --------- |
| 痛点型   | "你还在XXX吗？"    | 教程/避坑 |
| 震惊型   | "99%人都不知道..." | 揭秘/科普 |
| 结果型   | "看完你就明白了"   | 悬念/揭秘 |
| 对比型   | "左边VS右边..."    | 测评/选购 |

**内容节奏把控**

| 时间段     | 任务     | 技巧          |
| ---------- | -------- | ------------- |
| 0-30秒     | 建立期待 | 悬念/价值承诺 |
| 30秒-2分钟 | 铺垫信任 | 背景/专业背书 |
| 2-5分钟    | 核心交付 | 重点内容/高潮 |
| 结尾       | 引导互动 | CTA/留钩子    |

#### 四、标题公式库

**通用爆款公式**

| 公式      | 模板                       | 示例                           |
| --------- | -------------------------- | ------------------------------ |
| 悬念+数字 | "{主题}，{数字}个{关键词}" | "黄金投资，3个新手必踩的坑"    |
| 揭秘型    | "内部揭秘：{行业秘密}"     | "金店不说的5个秘密"            |
| 对比型    | "{A} VS {B}，差距惊人"     | "古法金 VS 普通金，选错亏大了" |
| 痛点+方案 | "{痛点}？{解决方案}"       | "手镯容易变形？教你3招预防"    |
| 情怀型    | "{情感词}的{传统工艺}"     | "即将失传的錾刻绝技"           |
| 结果承诺  | "只需{简单行动}，{好结果}" | "学会这3招，回收多卖500"       |

#### 五、对标与迁移

**分析UP主不只是看数据**

```
1. 内容定位
   - 做什么类型的视频？
   - 主要解决用户什么问题？
   - 差异化在哪里？

2. 运营策略
   - 更新频率如何？
   - 有没有做多平台分发？
   - 粉丝互动方式？

3. 商业化路径
   - 接广告还是带货？
   - 有没有知识付费？
   - 私域引流方式？

4. 可迁移性
   - 哪些策略可以用在我的内容？
   - 需要什么调整？
```

### 输出质量标准

| 要素         | 要求                           |
| ------------ | ------------------------------ |
| **洞察提炼** | 不只是描述，要解释"为什么"     |
| **数据解读** | 不只是数字，要解读"意味着什么" |
| **可落地性** | 每个建议都要可以直接执行       |
| **业务结合** | 必须结合用户的具体业务场景     |
| **实操价值** | 能直接指导选题/写标题/设计内容 |
