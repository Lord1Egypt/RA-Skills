# 🧿 Big8（八卦）— AI 玄学助手方案

> 飞书里的赛博算命先生，一鱼多吃：看风水 × 排八字 × 查星座 × 看面相

---

## 🎯 产品定位

一键式玄学助手，纯免费，IM 原生。不卖课、不卖符、不引流。

## 🧩 核心功能（四合一）

| 功能 | 实现方式 | 成本 |
|:--|:--|:--|
| 📸 **看风水** | 上传图片 → Vision Model 分析 → 结合风水 KB 给建议 | ¥0 |
| 🎂 **算八字** | lunar-python 排盘 + LLM 解读（大运/流年/五行） | ¥0 |
| ♈ **查星座** | LLM 直接生成运势（配合生日算上升星座） | ¥0 |
| 👀 **看面相** | 上传自拍 → Vision Model 识别面部特征 → 结合面相 KB 解读 | ¥0 |

### 附加功能
- 🔮 每日一卦（LLM 随机起卦+解卦）
- 🤝 八字合婚 / 星座配对
- 🖼️ 五行头像生成（image-01 出图，可做增值但不收费）

## ⚙️ 技术方案（方案二·标准版）

### 架构

```
用户消息
  ↓
big8.py 主路由
  ├─ 📸 有图片 ──→ vision model 分析
  │                ├─ 室内照 → 风水模式（KB: fengshui-kb.yaml）
  │                └─ 人脸照 → 面相模式（KB: face-kb.yaml）
  ├─ 🎂 有生日 → lunar-python 排盘 → LLM 解读
  ├─ ♈ 有星座名 → LLM 生成运势（+ 可选接入免费 API）
  └─ 🌟 综合模式 → 多个维度混搭
```

### 依赖

```
pip install lunar-python pillow
# 不需要付费 API
```

### 知识库（本地维护）

- `knowledge/fengshui-kb.yaml` — 家居风水规则（八宅、形煞、化解方案）
- `knowledge/face-kb.yaml` — 面相规则（三停、五官、十二宫）
- `knowledge/bazi-kb.yaml` — 命理知识（五行、纳音、神煞、大运规则）
- `knowledge/zodiac-kb.yaml` — 星座资料（日期、特性、配对）

所有 KB 用于辅助 LLM prompt，不是算法匹配。

### 面相怎么实现

面相的逻辑和风水一样：

1. 用户发一张自拍或头像照
2. Vision Model 识别面部特征：额头高低、眉形、眼距、鼻型、嘴型、下巴、耳朵、痣的位置…
3. 结合面相知识库给出解读
4. 解读维度：三停（上中下庭）、十二宫（命宫、财帛、兄弟…）、五官主贵

**隐私注意：** 面相图不上传任何第三方，只在 Vision Model 完成识别后丢弃。描述里写清楚。
