## 二、十二宫位（Bhava）深度含义

### 1宫（Lagna/Ascendant）—— 自我宫
**梵文：** Thanu Bhava（身体之宫）
**掌管：** 外貌、性格、体质、生命力、自我认知、童年早期、自我表达方式
**天然卡拉卡（Karaka）：** 太阳
**重要性：** 最重要的宫位，决定整个命盘的基调

**分析要点：**
- 上升星座特质 → 外貌与整体命运基调
- 上升主星位置 → 自我发展的方向
- 1宫内的行星 → 直接影响个人形象与体质
- 上升主星与月亮的关系 → 外在与内在的协调度

---

### 2宫（Dhana Bhava）—— 财富宫
**掌管：** 积蓄/财富、家族/家庭、语言/口才、饮食/面部、眼睛（右眼）
**天然卡拉卡：** 木星
**特殊：** 2宫主是 Maraka（死亡指示星）之一

**财富分析：**
- 2宫主状态 → 财富积累方式
- 木星与2宫的关系 → 财富的多寡
- 凶星在2宫 → 财务损耗或说话刻薄

---

### 3宫（Sahaja Bhava）—— 兄弟宫
**掌管：** 兄弟姐妹、勇气、短途旅行、沟通技能、写作、媒体、手臂/肩膀
**天然卡拉卡：** 火星
**分组：** Upachaya宫（随时间增强）

**分析要点：**
- 3宫主落处 → 兄弟关系、沟通发展方向
- 强壮的3宫 → 勇气、行动力、与人连接的能力
- 火星在3宫 → 勇气十足，好胜

---

### 4宫（Sukha Bhava）—— 幸福宫
**掌管：** 母亲、家庭/房产/土地、车辆、教育（基础）、内心平静、故乡
**天然卡拉卡：** 月亮
**分组：** Kendra宫（最强角宫之一）

**分析要点：**
- 4宫及4宫主状态 → 家庭幸福度、与母亲关系
- 月亮在4宫 → 情感依恋于家庭，或感情细腻
- 土星/Rahu在4宫 → 与母亲的关系复杂，离家倾向

---

### 5宫（Putra Bhava）—— 子女宫
**掌管：** 子女、智慧/直觉、创造力、爱情/恋爱、投机/赌博、过去世积德、教育（高等）
**天然卡拉卡：** 木星
**分组：** Trikona宫（三角幸运宫）

**子女分析（查悦占星体系）：**
- 5宫主状态 + 木星状态 → 子女缘分
- 5宫主落入6/8/12宫 → 子女缘较弱或延迟
- 凶星占据5宫 → 子女方面有挑战
- 木星过境5宫 → 常为生育时机
- D-7（Saptamsha分盘）→ 深入子女分析

**生育时机：**
木星大运、5宫主大运，配合木星过境5宫或9宫

---

### 6宫（Ripu Bhava）—— 敌人宫
**掌管：** 健康/疾病、债务、敌人/竞争者、服务/日常工作、诉讼
**天然卡拉卡：** 火星、土星
**分组：** Dusthana（凶宫）、Upachaya（随时间增强）

**分析要点：**
- 6宫强 → 战胜疾病和敌人的能力强
- 6宫主落入3宫 → 兄弟可能成为竞争者
- 6宫主与上升主星的关系 → 健康的整体状况

---

### 7宫（Kalatra Bhava）—— 婚姻宫
**掌管：** 配偶/婚姻、合伙关系、外交、商业伙伴、欲望
**天然卡拉卡：** 金星（男命）/ 木星（女命）
**分组：** Kendra宫、Maraka宫

**婚姻分析（查悦占星五条思路）：**
1. **7宫分析**：7宫内行星、7宫主位置与状态
2. **金星（男命）/ 木星（女命）**：伴侣的基本性质
3. **D-9（Navamsa）7宫**：婚姻的深层质量
4. **Karaka 行星状态**：金星或木星在本命盘的强弱
5. **Dasha时机**：金星/木星大运，或7宫主大运期间最常出现婚姻

**婚姻障碍信号：**
- 土星相位7宫或7宫主 → 婚姻延迟
- 火星在1/2/4/7/8/12宫（Manglik） → 婚姻冲击
- 7宫主落入6/8/12宫 → 婚姻有挑战
- Rahu在7宫 → 非传统婚姻，伴侣来自不同背景

---

### 8宫（Mrityu/Randhra Bhava）—— 转化宫
**掌管：** 寿命、深刻转变/危机、神秘/神秘学、配偶财产/遗产、慢性病、秘密
**天然卡拉卡：** 土星
**分组：** Dusthana（凶宫）

**分析要点：**
- 8宫强（上升/三角宫主在此） → 神秘学才能，能从危机中再生
- 8宫主落入上升 → 体质不稳，可能有秘密
- 吉星在8宫 → 长寿，可能有突发之财（遗产）

---

### 9宫（Dharma/Bhagya Bhava）—— 命运宫
**掌管：** 父亲、宗教/信仰、哲学/高等教育、命运/幸运、长途旅行、上师/贵人
**天然卡拉卡：** 木星、太阳
**分组：** Trikona宫（最吉祥的三角宫）

**分析要点：**
- 9宫强 → 命运顺遂，宗教信仰深厚，好运连连
- 9宫主 = 命运主星：其状态决定人生整体幸运程度
- 木星在9宫 → 最吉配置之一，智慧与命运兼得

---

### 10宫（Karma/Rajya Bhava）—— 事业宫
**掌管：** 职业/事业、社会地位/名誉、权威、政府、行动/成就
**天然卡拉卡：** 太阳、土星、木星、火星
**分组：** Kendra宫（最强角宫）、Upachaya

**事业分析：**
- 10宫主状态 → 职业发展方向和高度
- 太阳 → 领导力和权威
- 土星 → 职业的稳定性和持久性
- D-10（Dashamsha分盘）→ 深入事业分析

**事业高峰时机：**
10宫主大运、太阳大运，配合木星过境10宫

---

### 11宫（Labha/Ayaya Bhava）—— 收益宫
**掌管：** 收入/财务收益、愿望实现、朋友/社交圈、长子（次于5宫）
**天然卡拉卡：** 木星
**分组：** Upachaya（随时间增强）

**分析要点：**
- 11宫强 → 愿望易实现，财运和人脉好
- 11宫主落入角宫 → 收益稳定
- 11宫主与2宫主联系 → 财富双重加强

---

### 12宫（Vyaya Bhava）—— 损耗宫
**掌管：** 支出/损失、隐居/隔离、外国/海外、灵性解脱、睡眠/床事、秘密敌人
**天然卡拉卡：** 土星、Ketu
**分组：** Dusthana（凶宫）

**分析要点：**
- 12宫强（上升主/三角宫主在此）→ 灵性倾向，可能在海外生活
- 12宫主落入1宫 → 灵性与现实之间的张力
- Ketu在12宫 → 灵性解脱的强烈倾向

---

## 三、宫位分组系统

### 角宫（Kendra）—— 最强
1宫、4宫、7宫、10宫
→ 行星在角宫力量增强，影响力最大

### 三角宫（Trikona）—— 最吉
1宫、5宫、9宫
→ 幸运宫，行星在此带来祝福，Raja Yoga 的关键

### 上升宫（Upachaya）—— 随时间增强
3宫、6宫、10宫、11宫
→ 凶星在此宫反而好：越来越能转化挑战

### 凶宫（Dusthana）—— 挑战宫
6宫、8宫、12宫
→ 宫主落入凶宫通常不利，吉星在此力量受压

### Maraka 宫（死亡指示宫）
2宫、7宫
→ 2宫主和7宫主是 Maraka 行星，在Dasha分析中与寿命和健康危机相关

## 四、Bhavat Bhavam（宫之宫）

某宫在另一宫的扩展含义：第X宫的第Y宫 = 本命盘的第(X+Y-1)宫

| 原宫 | 含义扩展 |
|------|----------|
| 5宫的5宫 = 9宫 | 5宫主题的深化（孙辈、深层智慧） |
| 7宫的7宫 = 1宫 | 伴侣反映自我 |
| 4宫的4宫 = 7宫 | 家庭母亲与婚姻的关联 |
| 9宫的9宫 = 5宫 | 命运与智慧的互动 |

## 五、宫主星飞入各宫速查

### 1宫主飞入各宫
- **→1宫**：强健体质，自我意识强
- **→2宫**：财富靠自身努力，家庭意识强
- **→3宫**：沟通能力强，勇于冒险
- **→4宫**：家庭幸福，内心平静，有房产
- **→5宫**：聪明，有子女缘，创造力强
- **→6宫**：健康挑战，但有克服困难的毅力
- **→7宫**：婚姻重要，合伙吉，自我可能妥协
- **→8宫**：神秘色彩，寿命议题，有转化经历
- **→9宫**：幸运，宗教倾向，父亲关系重要
- **→10宫**：事业成就突出，名誉好
- **→11宫**：愿望易实现，财运好，人脉广
- **→12宫**：灵性倾向，可能在外国生活，需防损耗

## 六、行星相位（Graha Drishti）

**通用相位（所有行星）：** 第7宫（对冲）

**特殊额外相位：**
| 行星 | 额外相位 |
|------|----------|
| 木星 | 第5宫、第9宫 |
| 土星 | 第3宫、第10宫 |
| 火星 | 第4宫、第8宫 |
| Rahu/Ketu | 第5宫、第9宫（部分传统认可） |

**Jaimini 相位体系（星座相位）：**
- 动星座（白羊/巨蟹/天秤/摩羯）相位所有固定星座，除了毗邻的那个
- 固定星座（金牛/狮子/天蝎/水瓶）相位所有动星座，除了毗邻的那个
- 变星座（双子/室女/射手/双鱼）相位所有其他变星座

### Gandanta（业力结节）位置
水象/火象星座交界处各3°26'：
- 巨蟹 26°40' → 狮子 3°20'（Ashlesha → Magha）
- 天蝎 26°40' → 射手 3°20'（Jyeshtha → Mula）
- 双鱼 26°40' → 白羊 3°20'（Revati → Ashwini）

上升或月亮落于 Gandanta 区域，代表强烈业力节点，需专门处理（通常通过 Nakshatra Shanti 仪式）。
