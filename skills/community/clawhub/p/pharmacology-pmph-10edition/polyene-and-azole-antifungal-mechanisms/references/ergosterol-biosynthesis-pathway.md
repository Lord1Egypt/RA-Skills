# 麦角固醇生物合成通路详解

麦角固醇是真菌细胞膜的关键成分，其合成路径如下：

1. **起始物**：乙酰辅酶A → 甲羟戊酸 → 异戊烯焦磷酸 → 角鲨烯
2. **环化**：角鲨烯在角鲨烯环氧酶作用下生成环氧角鲨烯，再经羊毛固醇合酶催化形成羊毛固醇
3. **脱甲基化**：羊毛固醇 → 24-甲烯二氢羊毛固醇 → 在CYP51（14α-脱甲基酶）催化下脱去14位甲基
4. **后续修饰**：经C-5脱氢、C-24甲基化等步骤最终生成麦角固醇

唑类药物主要抑制第3步中的CYP51酶。两性霉素B则直接结合终产物麦角固醇。

## 耐药机制
- CYP51基因突变（唑类耐药）
- 麦角固醇合成通路下调或替代固醇积累
- 药物外排泵过表达

## 人源CYP450交叉反应
- 咪唑类（如酮康唑）对人CYP3A4、CYP2C9亲和力高，易致肝毒性和药物相互作用
- 三唑类（如伏立康唑、泊沙康唑）选择性更高，但仍需监测相互作用
