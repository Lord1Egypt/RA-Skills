## Description: <br>
Provides comprehensive US stock analysis using current market research for fundamentals, technical indicators, peer comparisons, and investment-style report generation. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[litiao1224](https://clawhub.ai/user/litiao1224) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and developers use this skill to analyze US stock tickers, compare public companies, and produce structured markdown reports covering financial metrics, valuation, technical signals, risks, and current market context. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated ratings, target prices, stop-losses, and position sizing may be wrong, stale, or unsuitable for a user's financial situation. <br>
Mitigation: Treat outputs as informational research, verify market data independently, and consult a licensed financial professional before making investment decisions. <br>
Risk: The skill may use a Tavily API key or web search to gather market research. <br>
Mitigation: Confirm the user is comfortable using external search services and protect any required API keys before installation or execution. <br>
