# 索他洛尔在老年人及肾功能不全患者中的剂量调整

## 何时使用
- 患者确诊需使用索他洛尔治疗心律失常
- 患者为老年人（年龄 ≥65 岁）或已知存在肾功能不全
- 已获取患者的肌酐清除率（CrCl）或可估算该值

## 执行步骤
1. **确认索他洛尔药代动力学关键特征**：
   - 几乎全部以原形经肾脏排泄
   - 正常成人半衰期为12～15小时
   - 老年人因肾功能自然减退，半衰期显著延长

2. **根据肌酐清除率（CrCl）调整剂量**：
   - CrCl >60 mL/min：使用常规剂量
   - CrCl 40–60 mL/min：剂量减至常规的75%
   - CrCl 30–40 mL/min：剂量减至常规的50%
   - CrCl <30 mL/min：禁用；如必须使用，仅可在严密心电监测下给予极低剂量

3. **对老年患者额外评估**：
   - 即使CrCl >60 mL/min，也应警惕隐匿性肾功能下降
   - 优先选择较低起始剂量并延长给药间隔

4. **实施QT间期监测**：
   - 用药前、剂量调整后及治疗期间定期监测QT间期
   - 若QT间期 >500 ms 或较基线延长 >60 ms，立即评估停药或减量

5. **输出个体化方案**：
   - 明确推荐剂量（mg）与给药频率
   - 注明监测频率与高风险预警指标

> **注意**：未评估肾功能前不得给予标准剂量；严重肾衰竭患者原则上禁用。
