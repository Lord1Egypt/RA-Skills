## Description: <br>
Set the style guidance for an executive report. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Employees and external users use this skill during routine executive reporting work to set concise style guidance for a short project status report. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill provides narrow formatting guidance rather than a complete report-generation workflow. <br>
Mitigation: Use it only to choose or confirm report style, and review the final executive report content separately. <br>
Risk: A fixed concise format mode may not fit every executive-reporting context. <br>
Mitigation: Confirm that the returned format mode matches the audience and reporting purpose before applying it. <br>
