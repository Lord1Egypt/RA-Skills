## Description: <br>
Implements UI from design mockups such as Figma, Sketch, or images with pixel-accurate layout, responsive behavior, and design token alignment. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wangzhiming1999](https://clawhub.ai/user/wangzhiming1999) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and frontend engineers use this skill to convert Figma, Sketch, or image mockups into project-native frontend code. It guides layout, typography, colors, interaction states, responsive behavior, accessibility, and design-difference review. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may activate on broad UI phrases that are not clearly design-to-code requests. <br>
Mitigation: Confirm that the request involves a design mockup, screenshot, or explicit design-to-code task before applying the workflow. <br>
Risk: Generated UI code can introduce layout, accessibility, or design-token drift if applied without review. <br>
Mitigation: Review code changes, compare against the supplied design, and verify responsive states, semantic HTML, and accessible controls before merging. <br>
