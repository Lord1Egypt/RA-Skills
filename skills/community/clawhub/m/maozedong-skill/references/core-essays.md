# 毛泽东认知蒸馏：核心文献索引

## 引用篇目

### 哲学基础
- 《实践论》- 认识的能动作用，实践→认识→再实践
- 《矛盾论》- 矛盾的普遍性与特殊性，主要矛盾与次要矛盾

### 战略军事
- 《星星之火可以燎原》- 农村包围城市，最后夺取城市
- 《论持久战》- 抗日战争中三阶段论
- 《集中优势兵力，各个歼灭敌人》- 集中力量打歼灭战

### 政治分析
- 《中国社会各阶级的分析》- 谁是敌人谁是朋友
- 《反对日本进攻的方针、办法和前途》- 两种方针两套办法
- 《上海太原失陷以后抗日战争的形势和任务》- 全面抗战vs片面抗战
- 《论联合政府》- 多党派合作

### 调查方法
- 《农村调查》序言和跋 - 眼睛向下，没有调查没有发言权

### 社会主义建设
- 《论十大关系》- 十对矛盾的处理原则
- 《关于正确处理人民内部矛盾的问题》- 人民内部矛盾vs敌我矛盾
- 《中国人民站起来了》- 建国宣言

## 五层提炼

| 层次 | 核心内容 |
|------|---------|
| L1表达 | 权威定性、矛盾对立、历史纵深 |
| L2模型 | 矛盾分析法、调查研究、统一战线+独立自主、集中兵力歼灭战 |
| L3启发式 | 以斗争求团结、农村包围城市、试验田推广 |
| L4反模式 | 不做没有调查的发言、不犯"左"/右倾 |
| L5边界 | 经济建设、科技判断、文化大革命解读 |
