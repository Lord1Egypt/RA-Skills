# 静脉高压性水肿机制分析

## 何时使用
- 患者出现全身性或局部凹陷性水肿（按压后留有凹陷）
- 存在右心充血性心力衰竭、静脉血栓、肿瘤压迫或妊娠子宫压迫等导致静脉回流障碍的情况
- 需排除非静脉性水肿（如肾病综合征低蛋白血症、过敏反应等）

## 执行步骤

1. **判断水肿类型是否符合静脉高压特征**：
   - 全身性水肿：常见于右心衰竭，呈体位性分布（站立时下肢水肿，卧床时骶尾部水肿）
   - 局部水肿：见于静脉受压或血栓形成区域（如下肢深静脉血栓、盆腔肿瘤压迫髂静脉）

2. **确认病理生理机制**：
   - 右心衰竭 → 心排出量无法代偿 → 肾素-血管紧张素-醛固酮系统（RAAS）激活 → 水钠潴留 → 血容量增加 → 静脉压升高 → 液体从血管渗入组织间隙
   - 局部静脉阻塞 → 阻塞远端静脉压升高 → 局部组织液生成超过回流 → 局部水肿

3. **识别关键临床特征**：
   - 凹陷性水肿
   - 与体位相关的分布模式
   - 常伴颈静脉怒张、肝淤血等右心衰体征（全身性情况）

4. **排除非适用情况**：
   - 若水肿为非凹陷性、突发性或伴荨麻疹，考虑过敏性水肿
   - 若伴严重低蛋白血症、大量蛋白尿，考虑肾病性水肿

> 注意：左心衰竭引起的肺水肿虽属静脉压增高，但属于肺循环特殊表现，需单独评估。
