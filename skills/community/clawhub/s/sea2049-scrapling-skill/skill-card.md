## Description: <br>
Scrapling Skill helps agents create authorized Python web-scraping workflows with Scrapling, including setup, fetcher selection, extraction code, crawling patterns, and fallback guidance. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agents use this skill to plan and implement authorized web scraping with Scrapling. It guides setup, fetcher choice, structured extraction, crawl scaffolding, output schema design, and reliability fallbacks. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Web scraping can be misused against sites where scraping is not authorized or may collect private, paywalled, login-required, or personal data. <br>
Mitigation: Use the skill only for lawful, authorized targets, avoid restricted or personal-data sources, respect site terms, and keep request rates reasonable. <br>
Risk: Browser-based scraping setup may install and run Scrapling fetcher dependencies that should be reviewed for the target environment. <br>
Mitigation: Review or pin the Scrapling dependency before running setup commands and install it in a controlled virtual environment. <br>
