## Description: <br>
Call real humans to test your product (URL or app). Get structured usability feedback with screen recordings, NPS scores, and AI-aggregated findings. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[avivahe326](https://clawhub.ai/user/avivahe326) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Product teams, UX researchers, developers, and AI agents use this skill to request human usability testing for a URL or app and receive structured findings that guide product fixes. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Product-testing recordings and reports may contain sensitive product, user, or account information. <br>
Mitigation: Use test accounts and non-sensitive environments, and install only if you are comfortable sending recordings and reports to the hosted service or your self-hosted instance. <br>
Risk: Providing repository details and GitHub access can enable automated code analysis and possible pull request creation. <br>
Mitigation: Provide repoUrl and GitHub write access only when intentional; prefer read-only access and review all generated diffs before merging. <br>
Risk: Self-hosting requires installing the npm package, running a server, and supplying AI provider API keys. <br>
Mitigation: Review and pin the npm package, constrain the server environment, and provide only the AI API keys intended for this workflow. <br>
