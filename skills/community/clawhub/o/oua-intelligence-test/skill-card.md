## Description: <br>
OUA 统一智能评估 is a ten-dimension AI assessment framework for evaluating language, reasoning, knowledge, coding, creativity, memory, tool use, safety, engineering delivery, and robustness, with interactive scoring and optional HTML or JSON reporting. <br>

This skill is ready for commercial/non-commercial use. <br>
