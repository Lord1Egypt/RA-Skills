# 平台视频格式参考

## TikTok

| 参数 | 要求 |
|------|------|
| 推荐时长 | 15-60秒 |
| 最佳时长 | 21-34秒 |
| 分辨率 | 1080x1920 (9:16) |
| 宽高比 | 9:16 |
| 帧率 | 30fps |
| 文件格式 | MP4 |
| 最大文件大小 | 287.6 MB |

### 节奏建议
- **前3秒**：必须出现强钩子，完播率关键
- **3-15秒**：建立痛点/共鸣
- **15-30秒**：产品展示+卖点
- **结尾**：CTA（关注/点赞/链接）

### 内容特点
- 快节奏、强节奏BGM
- 字幕必须（静音观看友好）
- 真人出镜效果更好
- 热门话题/挑战可蹭

## YouTube Shorts

|------|------|

| 最佳时长 | 30-45秒 |
| 分辨率 | 1080x1920 (9:16) |
| 宽高比 | 9:16 |
| 帧率 | 30fps |
| 文件格式 | MP4 |
| 最大文件大小 | 4GB |

### 节奏建议
- 比TikTok略深度
- 可增加字幕解说
- 适合教程类、功能展示
- 结尾引导订阅

### 内容特点
- 标题和封面重要
- 算法依赖观看时长
- 适合SEO关键词布局
- 可关联长视频

## 通用脚本结构模板

### 标准带货脚本（30秒）

```
【0-3秒】开场钩子
  - 提出痛点/悬念/惊喜
  - 画面：冲击感强

【3-8秒】问题放大
  - 描述目标用户痛点
  - 画面：情境演绎

【8-15秒】解决方案
  - 引入产品
  - 画面：产品展示

【15-22秒】卖点展示
  - 核心功能演示
  - 画面：特写+使用场景

【22-28秒】效果对比
  - Before/After
  - 画面：对比展示

【28-30秒】行动号召
  - 引导点击链接/关注
  - 画面：产品+CTA
```

### 品牌故事脚本（60秒）

```
【0-5秒】情感钩子
【5-15秒】品牌故事/理念
【15-35秒】产品诞生过程
【35-50秒】用户证言/评价
【50-60秒】品牌愿景+CTA
```
