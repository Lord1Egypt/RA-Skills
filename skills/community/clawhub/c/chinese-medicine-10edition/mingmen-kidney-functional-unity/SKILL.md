# Mingmen and Kidney Functional Unity

## When to Use
- 讨论命门的生理功能或解剖定位时
- 分析“命门之火”“命门之水”等术语的实质含义时
- 辨证论治中遇到“命门虚损”类证候需转化为肾阴阳辨证时
- 阅读《难经》《素问》等经典中关于命门的论述需结合后世共识理解时

## Execution Steps
1. **确认命门的理论起源**：命门作为内脏概念始见于《难经》，意为“生命之门”。
2. **梳理命门部位的三种主流观点**：
   - 右肾为命门（《难经·三十六难》）
   - 两肾总称为命门（元代滑寿）
   - 两肾之间为命门（明代赵献可）
3. **归纳命门功能的四种认识**：
   - 机体生命活动的原动力
   - 与生殖功能密切相关
   - 全身阳气的根本
   - 水火之宅，内含肾阴肾阳
4. **统一核心共识**：
   - 命门与肾在生理上“息息相通”
   - 同为“元气之本”“五脏阴阳之本”
   - “命门之火” = 肾阳，“命门之水” = 肾阴
   - 肾阴肾阳 = 真阴真阳 = 元阴元阳
5. **临床转化应用**：将“命门虚损”直接理解为肾阳不足或肾阴亏虚，纳入肾系辨证体系。

## Precautions
- 不适用于将命门视为现代解剖学意义上的独立实体器官
- 不适用于脱离中医藏象理论框架的纯结构定位讨论
