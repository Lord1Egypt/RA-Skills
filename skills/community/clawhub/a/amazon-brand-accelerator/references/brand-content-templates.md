# 品牌内容模板集

本模板集为亚马逊品牌卖家提供A+ Content、旗舰店、Brand Story和视频内容的结构化框架。直接参考使用，按需调整具体内容。

---

## Part 1：A+ Content 模板集

### 模板一：功能性产品标准A+模板

```
【模块1：产品核心卖点Hero图】
- 左侧：产品白底主图（占据60%宽度）
- 右侧：
  - 标题：产品核心卖点一句话（18字以内）
  - 副标题：补充说明（可选，12字以内）
  - 三点核心优势图标+简短描述

【模块2：产品特性对比图】
- 四宫格或五宫格布局
- 每格：图标 + 特性标题 + 简短描述
- 特性示例：材质/容量/功率/安全性

【模块3：使用场景图】
- 大幅场景图（户外/家居/办公等真实使用场景）
- 叠加文字标注关键使用环节

【模块4：产品规格详解】
- 左侧：产品规格参数表
- 右侧：产品尺寸图（带数据标注）

【模块5：品牌故事/质量保证】
- 品牌Logo + 品牌宣言
- 质量认证图标（CE/FCC/ROHS等）
- 售后服务承诺（退换货、保修期等）
```

### 模板二：高端产品A+模板（强调品质与设计）

```
【模块1：品牌Hero展示】
- 全宽幅高品质产品图
- 叠加品牌名称和核心价值主张
- 品牌色彩为主色调

【模块2：设计理念阐述】
- 左侧：细节特写图
- 右侧：设计理念文字（150字以内）
- 设计亮点标注

【模块3：材料与工艺】
- 材料特写图（高清细节）
- 材料优势说明
- 工艺流程简述

【模块4：用户体验场景】
- 3-4张不同场景的生活方式图
- 每张配简短用户利益描述

【模块5：品牌承诺】
- 品牌故事片段（100字）
- 质保服务说明
- 品牌联系方式
```

### 模板三：多SKU产品线A+模板

```
【模块1：产品线概览Hero】
- 产品线全家福图
- 产品线名称和定位

【模块2：产品对比表】
- 表格形式：型号/规格/适用场景/价格区间
- 配产品图

【模块3：各型号详细介绍】
- 每个SKU一个模块（可循环）
- 产品图 + 核心卖点 + 适用人群

【模块4：选购指南】
- 决策树或流程图形式
- 帮助用户根据需求选择合适产品

【模块5：品牌统一收尾】
- 品牌Logo
- 跨产品线的品牌承诺
```

## Part 3：Brand Story 框架

### Brand Story 基础框架（5步法）

```
【第一步：引人入胜的开头】
- 用一个令人印象深刻的画面或数据开头
- 示例："每天，全球有超过XX万家庭选择XX品牌，守护他们的XX。"

【第二步：痛点洞察】
- 描述你发现的问题或用户面临的困境
- 引起目标用户的共鸣
- 示例："在XX领域，用户常常面临XX困扰..."

【第三步：解决方案诞生】
- 描述品牌的解决方案是如何诞生的
- 可以加入创始人故事或研发历程
- 体现品牌的独特视角

【第四步：核心价值传递】
- 用简洁有力的语言传递品牌核心价值
- 避免空洞口号，用具体细节支撑
- 示例："我们相信XX，所以XX产品采用XX材料和XX工艺..."

【第五步：情感收尾与行动召唤】
- 用一句感性的语言收尾
- 引导用户成为品牌故事的一部分
- 示例："选择XX，不仅是选择一款产品，更是选择一种XX的生活方式。"
```

### Brand Story 内容要素清单

```
□ 品牌名称和Logo
□ 品牌创立年份
□ 创始初衷或灵感来源
□ 品牌使命（为什么存在）
□ 品牌愿景（未来图景）
□ 核心产品特点
□ 材料/工艺亮点
□ 设计理念
□ 用户群体描述
□ 品牌价值观（3个关键词）
□ 社会责任或公益行动（如有）
□ 品牌承诺（品质/服务/环保等）
□ 一句品牌Slogan
□ CTA引导语
```

## Part 4：视频脚本结构模板

### 产品视频脚本模板（60-90秒）

```
【开场 Hook（0-5秒）】
- 视觉冲击：产品效果对比/使用场景/痛点呈现
- 或情感共鸣：一个让人有代入感的场景

【问题陈述（5-15秒）】
- 明确指出目标用户的痛点或需求
- "你是否遇到过..."
- 建立与观众的连接

【解决方案引入（15-25秒）】
- 介绍产品名称
- 展示产品外观
- 点出产品的核心定位

【核心卖点演示（25-45秒）】
- 按重要程度依次展示3-4个核心卖点
- 每个卖点：特点说明 + 效果演示/对比
- 配合字幕和旁白

【使用场景展示（45-55秒）】
- 展示产品在实际场景中的使用
- 体现生活方式和情感价值

【品牌收尾（55-60秒）】
- 品牌名称/Logo
- 品牌Slogan或核心价值
- 屏幕文字：品牌名 + CTA（如"立即选购"）

【可选：详细参数（60-90秒区间）】
- 如果需要展示更多规格参数
- 使用平铺字幕或分屏展示
```

### 品牌视频脚本模板（2-3分钟）

```
【开场（0-10秒）】
- 情感基调画面 + 品牌名
- 或创始人简短引言

【问题与愿景（10-45秒）】
- 行业现状和用户痛点
- 品牌的初心和愿景
- 建立情感连接

【品牌故事（45-90秒）】
- 创始故事（可选）
- 品牌理念和发展历程
- 核心价值观阐述

【产品与创新（90-180秒）】
- 产品线概览
- 设计理念和创新点
- 材料/工艺亮点

【用户证言（180-210秒）】
- 真实用户的使用反馈
- 或用户使用场景展示

【品牌承诺（210-240秒）】
- 品质承诺
- 服务承诺
- 品牌愿景重申

【收尾（最后10秒）】
- 品牌Slogan
- 品牌Logo
- 社媒账号提示
```

### 用户证言视频脚本模板（1-2分钟）

```
【开场（0-5秒）】
- 真实用户形象（素人感，不要过度包装）
- 用户简介："我是XX，我是XX（用户身份/职业）"

【问题引入（5-20秒）】
- "在使用XX品牌之前，我遇到的问题是..."
- 描述使用前的痛点或困扰

【转变过程（20-40秒）】
- "我开始使用XX品牌是因为..."
- 初次使用的印象或感受

【使用体验（40-90秒）】
- 详细描述使用后的感受和效果
- 可以展示产品使用过程
- 具体的改变或收益（数字化的描述更好）

【推荐理由（90-120秒）】
- "我推荐XX品牌给..."
- 最打动的一点是什么
- 一句话总结推荐理由

【结尾（最后10秒）】
- 用户姓名（可用化名）
- "我是XX，这是我的真实使用体验。"
```

## Part 5：内容创作质量自检清单

使用以下清单检查内容是否符合亚马逊品牌内容标准：

### A+ Content自检
- [ ] 图片清晰度高，无模糊或拉伸变形
- [ ] 文字清晰可读，与背景有足够对比度
- [ ] 品牌色彩一致
- [ ] 核心信息在首屏呈现
- [ ] 没有夸大宣传或绝对化用词（如"第一""最好"等）
- [ ] 规格数据准确无误
- [ ] 符合亚马逊A+ Content政策要求
- [ ] 所有文字已翻译为目标市场的语言

### 品牌旗舰店自检
- [ ] 首页加载流畅
- [ ] 导航清晰易用
- [ ] 产品分类逻辑合理
- [ ] 品牌故事完整且真实
- [ ] CTA按钮明显易点击
- [ ] 移动端显示效果正常
- [ ] 社交媒体链接有效

### 视频内容自检
- [ ] 视频画质清晰（1080P以上推荐）
- [ ] 音频清晰无噪音
- [ ] 字幕与语音同步
- [ ] 内容逻辑清晰
- [ ] 时长适中（前30秒有关键信息）
- [ ] 品牌标识出现
- [ ] 没有误导性或虚假宣传内容
