# 营养不良与营养过度诊断

## 适用对象
- 成人患者
- 营养评估对象

## 前提条件
- 已知患者的身高和体重，可计算 BMI
- 可获取或估算标准体重

## 使用时机
当患者存在以下情况之一时应用本技能：
- 体重显著下降或增加
- 临床怀疑营养状态异常（如乏力、食欲改变、体型明显变化）
- 需进行营养风险筛查或干预前评估

## 执行步骤
1. **计算 BMI**：使用公式 BMI = 体重(kg) / 身高(m)²。
2. **评估体重百分比**：将实际体重与标准体重比较，计算偏差百分比。
3. **判定营养状态**：
   - 若 BMI < 18.5 kg/m² 或体重低于标准体重 10%，判定为**消瘦**；若极度消瘦，考虑恶病质。
   - 若 BMI ≥ 28 kg/m²（中国标准）或 ≥ 30 kg/m²（WHO 标准），或体重超过标准体重 20%，判定为**肥胖**。
4. **初步病因分析**：根据临床背景，区分营养不良或肥胖的可能原因（如摄食障碍、内分泌疾病等）。
5. **输出结果**：明确标注“消瘦”、“正常”或“肥胖”，并注明是否提示恶病质或特定病因类型。

> **注意**：本技能不适用于儿童、孕妇或肌肉发达者（如运动员）。诊断需结合完整病史和病因分析，不可仅依赖 BMI 或体重百分比。
