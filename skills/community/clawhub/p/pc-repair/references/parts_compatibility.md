# 硬件兼容性检查表

本文档提供详细的硬件兼容性检查指南，帮助用户正确升级电脑硬件。

## 1. 内存兼容性检查

### 1.1 台式机内存兼容性

| 检查项目 | 如何检查 | 购买建议 |
|----------|----------|----------|
| **内存类型** | CPU-Z → SPD → 查看"类型" | DDR3/DDR4/DDR5，必须匹配 |
| **内存频率** | CPU-Z → Memory → 查看"DRAM Frequency" | 买相同或更高频率（会降频运行） |
| **插槽数量** | 主板说明书或CPU-Z → SPD | 一般2或4个插槽 |
| **最大支持容量** | 主板官网查参数 | 一般64GB或128GB |
| **双通道建议** | 优先买2条同款（套装） | 性能比单条提升10-15% |

**台式机内存购买决策树：**
```
查看现有内存
├─ 有空闲插槽？
│  ├─ 是 → 买同品牌同型号同频率（完美兼容）
│  └─ 否 → 拔掉旧内存，买2条新的（套装）
│
└─ 不确定？
   └─ 用CPU-Z查看 → 买相同参数的内存
```

### 1.2 笔记本内存兼容性（更复杂）

| 检查项目 | 如何检查 | 注意事项 |
|----------|----------|----------|
| **是否有空闲插槽** | AIDA64 → 主板 → 内存 | 很多新笔记本板载内存，无法升级 |
| **内存类型** | CPU-Z → SPD | 注意：笔记本内存比台式机短 |
| **最大支持容量** | 官网查型号参数 | 一般最大32GB |
| **频率限制** | 必须严格匹配 | 买错频率可能无法开机 |
| **电压** | 1.2V/1.35V | 必须匹配，否则不稳定 |

**笔记本内存升级建议：**
1. **优先查官网**：在品牌官网输入型号，查"用户手册"→"内存规格"
2. **慎重购买**：很多新款笔记本（尤其是超薄本）内存焊死在主板上，无法升级
3. **推荐工具**：AIDA64（可查看详细内存信息）

### 1.3 内存购买避坑指南

❌ **不要买：**
- 杂牌内存（金士顿假冒最多，建议京东自营）
- 频率过高的内存（如果主板只支持3200MHz，买3600MHz也跑不到）
- 台式机和笔记本内存搞混（金手指缺口位置不同）

✅ **推荐买：**
- 品牌：金士顿、威刚、海盗船、芝奇
- 渠道：京东自营、天猫官方旗舰店
- 型号：买套装（2条同款），兼容性最好

## 2. SSD兼容性检查

### 2.1 SSD接口类型速查

| 接口类型 | 外观 | 速度 | 适用场景 |
|----------|------|------|----------|
| **SATA 3.0** | 和普通硬盘一样（2.5英寸） | 500MB/s | 老电脑升级（有SATA接口就能用） |
| **M.2 SATA** | M.2接口但走SATA协议 | 500MB/s | 少数笔记本使用（已淘汰） |
| **M.2 NVMe** | M.2接口走PCIe协议 | 2000-7000MB/s | 主流选择（需要主板支持） |
| **PCIe 4.0** | M.2接口（支持PCIe 4.0） | 5000-7000MB/s | 新平台（AMD 5000系/Intel 11代+） |
| **PCIe 5.0** | M.2接口（支持PCIe 5.0） | 10000MB/s+ | 最新平台（AMD 7000系/Intel 13代+） |

### 2.2 台式机SSD升级检查清单

```
检查主板是否有M.2插槽
├─ 有M.2插槽？
│  ├─ 是 → 买M.2 NVMe SSD（速度快）
│  │  └─ 检查：是否支持NVMe（有的老主板M.2只支持SATA）
│  └─ 否 → 买2.5英寸SATA SSD（通用性强）
│
└─ 不确定？
   └─ 查主板型号（CPU-Z → 主板） → 官网查参数
```

### 2.3 笔记本SSD升级检查清单

```
拆机或查官网
├─ 是否有M.2插槽？
│  ├─ 有 → 买M.2 NVMe SSD
│  └─ 无 → 只能换原有的2.5英寸硬盘
│
├─ 是否有2242/2280长度限制？
│  └─ 有些超薄本只能用2242短卡（买错装不上）
│
└─ 是否支持PCIe 4.0？
   └─ 不支持的话，买PCIe 4.0 SSD也跑不到满速（但可以用，向下兼容）
```

### 2.4 SSD购买建议（2024年）

| 用途 | 推荐型号 | 容量 | 价格区间 |
|------|----------|------|----------|
| **日常办公** | 致钛PC005/西数SN570 | 512GB | 200-300元 |
| **游戏/设计** | 三星980/致钛TiPlus7100 | 1TB | 400-600元 |
| **高性能需求** | 三星990 Pro/西数SN850X | 1TB-2TB | 600-1200元 |

**避坑指南：**
- 不要买无缓存的入门级SSD做系统盘（寿命短）
- 笔记本升级注意厚度（7mm和9.5mm，买错装不上）
- M.2 SSD注意长度（2280是标准，有些本子只能用2242）

## 3. 显卡兼容性检查

### 3.1 台式机显卡升级检查

| 检查项目 | 如何检查 | 最低要求 |
|----------|----------|----------|
| **电源功率** | 看电源标签上的"额定功率" | GTX 1660: 450W+ / RTX 4060: 500W+ |
| **电源接口** | 看电源是否有6/8pin显卡供电线 | 必须要有（老电源可能没有） |
| **主板接口** | 必须是PCIe x16插槽（蓝色插槽） | 所有现代主板都有 |
| **机箱空间** | 测量机箱可用长度 | 显卡长度（300mm+的卡需要大机箱） |
| **CPU瓶颈** | CPU-Z查看CPU型号 | 老CPU会拖累新显卡（如i3配RTX 4060） |

### 3.2 显卡性能匹配表（避免CPU瓶颈）

| CPU档次 | 推荐显卡 | 备注 |
|---------|----------|------|
| i3/R3 | GTX 1650 / RTX 3050 | 入门级，1080p中画质 |
| i5/R5 | RTX 4060 / RX 7600 | 主流级，1080p高画质 |
| i7/R7 | RTX 4070 / RX 7800 XT | 高端级，2K高画质 |
| i9/R9 | RTX 4090 / RX 7900 XTX | 旗舰级，4K极致画质 |

### 3.3 笔记本显卡说明

⚠️ **重要**：笔记本显卡**无法升级**（焊死在主板上）

如果需要更强图形性能：
1. 外接显卡坞（雷电3/4接口，成本高，性能损失20%）
2. 换电脑（唯一实际解决方案）

## 4. 电源兼容性检查

### 4.1 电源功率计算

```
估算整机功耗
├─ CPU功耗（TDP）
│  ├─ i3/R3: 65W
│  ├─ i5/R5: 65-95W
│  └─ i7/R7: 95-125W
│
├─ 显卡功耗
│  ├─ GTX 1650: 75W
│  ├─ RTX 3060: 170W
│  ├─ RTX 4060: 115W
│  └─ RTX 4090: 450W
│
├─ 其他配件（估算50W）
│
└─ 总功耗 = (CPU+显卡+50W) × 1.5（预留余量）
   例：i5-12400F (65W) + RTX 4060 (115W) = 180W × 1.5 = 270W
   推荐：买450W或500W电源（留余量）
```

### 4.2 电源购买建议

| 整机功耗 | 推荐电源功率 | 推荐型号 |
|----------|--------------|----------|
| <200W | 400-450W | 航嘉JUMPER 450B |
| 200-300W | 500-550W | 海韵Focus GX550 |
| 300-400W | 650-750W | 海韵Focus GX750 |
| >400W | 850W+ | 海韵Prime TX-850 |

**电源避坑指南：**
- 必须买80Plus认证（铜牌/金牌/白金）
- 必须买知名品牌（海韵/振华/航嘉/长城）
- 不要买杂牌电源（容易烧硬件）
- 额定功率 ≠ 峰值功率（看清楚标签）

### 5.1 主板CPU接口对照表（Intel）

| 接口类型 | 支持CPU | 年代 |
|----------|---------|------|
| LGA 1151 | 6代/7代酷睿（i3-6100/i7-7700） | 2015-2017 |
| LGA 1151 v2 | 8代/9代酷睿（i3-8100/i9-9900K） | 2017-2019 |
| LGA 1200 | 10代/11代酷睿（i5-10400/i7-11700K） | 2020-2021 |
| LGA 1700 | 12代/13代/14代酷睿（i5-12400/i9-14900K） | 2021-2024 |
| LGA 1851 | 新一代酷睿（Ultra 200S） | 2024+ |

### 5.2 主板CPU接口对照表（AMD）

| 接口类型 | 支持CPU | 年代 |
|----------|---------|------|
| AM3+ | FX系列（已淘汰） | 2011-2015 |
| AM4 | Ryzen 1000-5000系列 | 2017-2022 |
| AM5 | Ryzen 7000/8000/9000系列 | 2022+ |

### 5.3 CPU升级建议

**升级前必查：**
1. 主板支持的CPU列表（官网查CPU支持列表）
2. BIOS版本（老主板可能需要先升级BIOS才能支持新CPU）
3. 散热能力（新CPU功耗更高，散热器要跟上）

**性价比升级路线（2024年）：**
- 老平台（AM4）：Ryzen 5 5600（600元，性价比之王）
- 新平台（AM5）：Ryzen 5 7600（1500元，支持DDR5）
- Intel平台：i5-12400F（800元，性价比高）或 i5-13400F（1200元）

## 7. 兼容性检查清单（升级前必看）

### 内存升级检查清单
- [ ] 确认内存类型（DDR3/DDR4/DDR5）
- [ ] 确认内存频率（用CPU-Z查看）
- [ ] 确认是否有空闲插槽
- [ ] 确认最大支持容量（官网查）
- [ ] 笔记本确认内存是否板载（无法升级）

### SSD升级检查清单
- [ ] 确认是否有M.2插槽
- [ ] 确认M.2是否支持NVMe（还是只支持SATA）
- [ ] 确认M.2长度（2242/2260/2280）
- [ ] 确认是否有空闲SATA接口（如果没有M.2）
- [ ] 确认电源是否有多余SATA供电线（台式机）

### 显卡升级检查清单
- [ ] 确认电源额定功率（≥显卡要求）
- [ ] 确认电源有显卡供电线（6/8pin）
- [ ] 确认机箱能装下显卡（测量长度）
- [ ] 确认CPU不会瓶颈（用天梯图对比）
- [ ] 确认主板PCIe插槽是否正常

### 电源升级检查清单
- [ ] 计算整机功耗（用功耗计算器）
- [ ] 确认电源尺寸（ATX/SFX，笔记本无解）
- [ ] 确认是否有足够供电线（主板24pin/CPU 8pin/显卡6+8pin）
- [ ] 确认品牌和80Plus认证（避免杂牌）
