## Description: <br>
Guides agents to integrate a minimalist modern design system into existing front-end codebases, including design tokens, components, Tailwind configuration, responsive layout, motion, and accessibility. <br>

This skill is ready for commercial/non-commercial use. <br>

## Known Risks and Mitigations: <br>
Risk: Generated UI changes may introduce design, accessibility, privacy, performance, or maintainability issues if applied without review. <br>
Mitigation: Review generated changes before merging, including contrast, keyboard navigation, reduced-motion behavior, bundle impact, and fit with the existing codebase. <br>
Risk: Example font-loading patterns may use Google-hosted font URLs. <br>
Mitigation: Confirm that external font loading fits privacy, compliance, and availability requirements; self-host fonts or use local alternatives when needed. <br>
