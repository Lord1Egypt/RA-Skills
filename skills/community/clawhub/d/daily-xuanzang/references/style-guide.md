# 讲稿风格指南 / Style Guide

每期《大唐西域记》讲稿的格式与风格要求。目标：**信达、有画面、有地理感、不掉书袋也不臆造**。

---

## 总原则

1. **忠于原文**：翻译和讲解必须基于 `progress.py next` 返回的 `text`，不得编造原文没有的国家、数字、传说。
2. **地理优先**：每期都要让读者清楚"今天西行到哪里、古今地名对应"。这是本书区别于普通史书的灵魂。
3. **保留音译 + 注今地**：生僻古国名保留原音译，首次出现时标注今地（如能确定），例：屈支（龟兹，今新疆库车）。
4. **传说照实收录、点到为止**：玄奘记的龙/那伽/圣迹神话，如实翻译，讲解里可点出"这类传说往往折射真实历史"，但不过度发挥。
5. **不掉书袋**：背景讲解要让普通读者读得懂，少用未解释的专业术语；必要术语（如"说一切有部""窣堵波"）给一句白话解释。

---

## 三段结构

### 📜 原文节录
- 直接引用 `text`，可适当分句、加标点便于阅读。
- 不要全文照搬到过长；若本段很长，节选最有代表性的部分，其余在翻译里覆盖。

### 🗣️ 白话翻译
- 通顺现代汉语，逐句对应，**信达为先**。
- 数字（里程、僧徒数、伽蓝数）要准确翻译，不要四舍五入成"很多"。
- 中文避免翻译腔；英文模式避免直译腔。

### 📚 背景讲解（3–5 要点）
覆盖以下角度（按本段实际内容取舍）：
- **这段讲哪国/哪圣迹**，在玄奘西行路线的哪一程
- **古今地理对应**：今天在哪个国家/地区
- **历史与佛教价值**：相关帝王、宗派、结集、艺术（如犍陀罗造像、龟兹乐、那烂陀学统）
- **相关典故**：佛本生、因缘、名人（戒日王、龙树、戒贤等）
- **可信度提示**：哪些是亲践、哪些是传闻；传说的历史投影

## 地图 prompt 风格

- 古地图 / 古籍插图风，**深色线条、强对比、标签清晰可读、填满画面**。
- 金色虚线 = 玄奘西行路线，自东向西。
- 每个地名 **中文古名 + 今地英文/拼音** 双标。
- 含指南针、标题横幅、山脉沙漠河流地貌。
- 随进度切换区域底图：丝路北道(塔里木盆地) → 中亚 → 阿富汗 → 北/中/南印度 → 锡兰 → 回程葱岭。

## 语音稿风格

- 中文 300–800 字朗读稿，讲史/诵经般庄重、沉稳。
- 保留：讲题、路线主线、一两个关键判断、下期预告。
- 若工作区偏好"叙事配轻背景音乐（古琴/箫/古筝，低音量不盖人声）"，遵循之。

---

## 长度建议

- 中文正文目标 **900–1800 字**（原文+翻译+讲解三部分都展开）。
- 不要为赶地图/语音把正文压成提纲。
- 每期在叙事自然处收尾，留下"下一站去哪"的悬念。
