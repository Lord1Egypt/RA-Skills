## The Compliance Moat

**ASE certification accuracy:** "ASE Master Technician" (all 8 A-series + L1) vs "ASE certified" (could be one cert) — not the same claim. This kit uses the exact credential you provide.

**EPA Section 608:** Required for anyone handling refrigerants in AC service. The kit discloses "EPA 608 Universal certified" accurately — and never says "licensed" (it's a federal certification, not a state license).

**Magnuson-Moss Warranty Act:** "Lifetime warranty" without defining lifetime's scope is a federal violation. This kit generates compliant warranty language: who provides it, what's covered, for how long, and what's excluded.

**State estimate laws:** CA (BAR), TX (DMV Rule 84), FL (DHSMV), NV (DMV) all require written estimates before work begins. This kit generates the correct disclosure language for your state.
