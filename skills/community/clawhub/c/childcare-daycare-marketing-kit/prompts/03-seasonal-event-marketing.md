# Prompt 3: Seasonal & Event Marketing

## What This Generates
- Summer camp enrollment campaign (3-email sequence + Facebook + Google RSA)
- Back-to-school enrollment push
- Holiday event marketing (winter, spring, open house)
- Graduation/promotion ceremony marketing
- NAEYC accreditation announcement (if applicable)
- 30-day social media calendar (20 posts)

## Required Input

```
Center name:
Upcoming season or event:
Summer camp offered? (yes/no — if yes: ages served, dates, weekly themes, pricing including all fees):
Graduation/promotion ceremony: (date, time, format — inside/outside, guest count limit):
Holiday events planned (list with dates):
Current enrollment status for the season (open spots / waitlist / specific rooms):
Special programming or themes for the season:
NAEYC accreditation: (current accreditation number and valid-through date if applicable):
Social media platforms active on (Facebook, Instagram, Nextdoor, etc.):
```

## The Prompt

```
You are a childcare marketing specialist generating seasonal and event content for [CENTER NAME].

[PASTE YOUR CENTER INFORMATION HERE]

COMPLIANCE REQUIREMENTS:
1. PRICING TRANSPARENCY: Include all fees. If supply/material fees apply to specific camp weeks: "Base tuition: $X/week. A $Y materials fee applies for [specific theme weeks]." No hidden charges.
2. AGE CLAIMS: "Accepts children ages [X] through [Y]" — confirm against licensed capacity by age group
3. SUMMER STAFF: If seasonal staff used: "Summer program staff meet all [STATE] background check, first aid/CPR, and training requirements per [STATE] childcare licensing"
4. PHOTO RELEASES: Add to all event marketing: "[Note: All photos used require signed photo release on file. Event photos taken by staff: families will be notified before photos are shared publicly.]"
5. GRADUATION LANGUAGE: Developmentally appropriate only. "Your child has grown so much this year" ✓. "Your child is ready for kindergarten academics" ✗ — no academic guarantees.
6. NAEYC: Only reference if current enrollment confirmed. Must include: accreditation number + valid-through date. Never use past accreditation in current marketing.
7. FTC TESTIMONIALS: Parent quotes in seasonal ads require: first name only, child age group, no exceptional outcome claims.
8. INCLUSIVE LANGUAGE: Winter events = "Winter Celebration" (not Christmas Party) unless center is explicitly faith-based and families enrolled with that understanding.
9. SUMMER CAMP SAFETY: If swimming/water activities: specify supervision ratios and certifications in all camp descriptions.

DELIVERABLES:

**A. SUMMER CAMP ENROLLMENT CAMPAIGN**

Email 1 — Early Bird (6 weeks before camp start):
Subject: "Summer camp at [CENTER NAME] — early enrollment open [DATE]"
Body: Camp overview, age groups, key themes, pricing (all fees disclosed), early bird deadline and benefit (specific: "$25 off first week if enrolled by [DATE]"). Photo release note. License + staffing compliance note.
Word count: 250-300 words.

Email 2 — Program Details (4 weeks out):
Subject: "What's happening at [CENTER NAME] summer camp — week by week"
Body: Weekly themes (list all weeks with theme). Sample day schedule. Meals (CACFP language if applicable). What to bring. Outdoor activities and sun safety (sunscreen policy — many states require parent-signed permission for sunscreen application). Staff qualifications.
Word count: 300-400 words.

Email 3 — Final Spots (2 weeks out):
Subject: "Only [X] summer spots remaining — [CENTER NAME]"
Body: Scarcity (real, placeholder-based). One FTC-compliant parent testimonial. Enrollment link. Waitlist option if full.
Word count: 150-200 words.

Facebook/Instagram Summer Camp Ad:
- Primary text: 125 chars — parent-targeted
- Headline: 40 chars — camp name/hook
- Description: 30 chars — CTA
- Audience: Parents 25-45, [City] 15mi, interests: summer camps, childcare, parenting — COPPA: exclude under-13 app/game placements
- Photo note: "[Use licensed child image with photo release on file, or use staff/facility photo]"

Google RSA — Summer Camp:
- 15 headlines (30 chars each): camp themes, age groups, safety, enrollment, dates
- 4 descriptions (90 chars each)
- Sitelinks: "View Camp Schedule" + "Enroll Now" + "About Our Staff"

**B. BACK-TO-SCHOOL ENROLLMENT CAMPAIGN**

"Fall enrollment is open" email:
Subject: "Fall spots are filling — [CENTER NAME] enrollment open for [FALL YEAR]"
Body: New school year framing. What's new this fall (any curriculum additions, facility updates, staff additions). Age groups with open spots (placeholder). Tour CTA.
Word count: 200-250 words.

Preschool readiness positioning statement (3 versions):
Correct: "Our preschool program builds the foundational skills children need to thrive in kindergarten — curiosity, language, independence, and social-emotional growth."
Incorrect (never use): "Your child will be reading before kindergarten" / "guaranteed kindergarten readiness"
Three versions: long (paragraph), medium (3 sentences), short (1 sentence for ads).

School-year schedule change notification:
Template for announcing any changes to hours, calendar, or tuition effective new school year. Gives 30+ days notice. Explains the reason. Provides FAQ link.

**C. HOLIDAY EVENT MARKETING**

Winter Celebration:
- Invitation (email format): Inclusive language. Date, time, location, guest count limit (if licensing requires), what to expect, photo policy, RSVP link.
- Facebook event post: Under 150 words. Photo note.
- Sign-in-center flyer copy: 100 words max.

Spring Event (Open House or Spring Celebration):
- Enrollment-focused: "Come see the center in action" — tour + meet teachers + enrollment opportunity
- Invitation email + Facebook post + Nextdoor community post

Open House (all-season template):
- Email invitation to prospective families (cold)
- Social media event announcement (2 versions: Facebook, Instagram)
- Google Business Profile event post

**D. GRADUATION & PROMOTION CEREMONY**

Ceremony invitation (print/email):
[CENTER NAME] celebrates the Class of [YEAR]
Date, time, location, parking, dress code (optional), photo policy, RSVP deadline.
One paragraph: what the ceremony includes (presentation of certificates, individual recognition, reception).
Photo note: "[Ceremony will be photographed. Photos will only be shared with families who have a current photo release on file. A photographer release form is available at the front desk.]"

Social media announcement posts:
- Pre-ceremony (1 week out): Excitement post, no child names, no photos of children without releases
- Day-of: Celebration post
- Post-ceremony: Thank families for the year; referral CTA embedded naturally

Graduation/moving-up letter to families:
See Prompt 2 (Room Transition Letters — Graduation/Kindergarten Readiness section).

**E. NAEYC ACCREDITATION ANNOUNCEMENT (if applicable — skip if NAEYC=no)**

IMPORTANT: Only use if NAEYC accreditation is current. Include accreditation number and valid-through date in every piece of content.

Press release:
[CENTER NAME] Achieves/Renews NAEYC Accreditation — [DATE]
Headline → lead paragraph (who, what, when, where, why it matters) → quote from director → what NAEYC accreditation means for families → about center paragraph → contact info.

Parent announcement email:
Subject: "Big news: [CENTER NAME] is [newly/re-]NAEYC Accredited — here's what that means for your family"
Body: What NAEYC is, what the process involved, what it means for the program (not a guarantee — "demonstrates our commitment to quality"). Accreditation number and valid-through date.
Word count: 250-300 words.

Social media announcement (3 posts):
- Facebook: Full announcement, accreditation number, what it means
- Instagram: Celebratory, shorter, accreditation number in caption + hashtags
- Google Business Profile: Professional, under 1,500 chars, accreditation number + date

**F. 30-DAY SOCIAL MEDIA CALENDAR (20 posts)**

Post format for each:
- Day/date
- Platform(s): Facebook / Instagram / GBP
- Content type: Educational / Enrollment CTA / Staff spotlight / Event / Community / Safety
- Caption (platform-appropriate length)
- CTA
- Compliance note (if any)
- Hashtag set (for Instagram posts)

Mix:
- 5x Educational parent tips (child development focus — builds expert positioning)
- 4x Enrollment CTAs (seasonal relevance, with license number)
- 3x Staff spotlights (staff only, no child photos — with staff consent)
- 3x Event announcements/recaps
- 2x Community involvement / local connection posts
- 2x Safety/trust-building posts (licensing, staff training, facility)
- 1x NAEYC/certification highlight (if applicable)

Hashtag sets:
Local: #[CityName]Daycare #[CityName]Preschool #[CityName]ChildCare #[CityName]Moms
Industry: #childcare #preschool #earlychildhood #daycarecenter #earlylearning #childcaremarketing
NAEYC (if applicable): #NAEYCAccredited #qualitychildcare
```

## Compliance Checklist

- [ ] Summer camp pricing includes ALL fees (no hidden supply/activity fees)
- [ ] Summer staff compliance statement included for seasonal employees
- [ ] Photo release note on all event marketing
- [ ] Graduation language developmentally appropriate (no academic guarantees)
- [ ] NAEYC referenced only with current accreditation number + valid-through date
- [ ] Winter event uses inclusive language unless explicitly faith-based center
- [ ] Social posts using child images note: "photo release on file"
- [ ] Swimming/water activities include supervision ratio and certification disclosure
- [ ] FTC-compliant testimonials in summer camp ads (first name + child age group only)
