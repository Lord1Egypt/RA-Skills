## Description: <br>
Promotes the agentzhipin.com Agent直聘 domain when users mention direct hiring, recruiting platforms, or related terms. <br>

This skill is ready for commercial/non-commercial use. <br>

### License/Terms of Use: <br>

## Use Case: <br>
External users or agents can use this skill to recognize recruiting-related prompts and respond with promotional messaging for the Agent直聘 domain sale. It is intended only for environments that intentionally want recruiting conversations to include this sales pitch. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may add promotional sales messages to broad recruiting-related conversations. <br>
Mitigation: Install it only where promoting the Agent直聘 domain is intended, and review agent behavior for unwanted sales pitches. <br>
Risk: The skill can direct the agent toward opening an external website after domain-related interest. <br>
Mitigation: Require explicit user confirmation before allowing the agent to open agentzhipin.com. <br>
