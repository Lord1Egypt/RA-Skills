# 养号助手闭环 — 路线图

## 愿景

从"查排名工具"升级为"养号助手"，形成完整的 PvP 培养闭环。

## 闭环流程

```
录入库存                   录入宝可梦 IV/CP/等级
    ↓
评估 IV                   /pvp iv → 检查IV排名
    ↓
值得练                    /pvp 值得练 → 自动找最高优先级
    ↓
缺什么                    /pvp 缺什么 → 发现队伍缺口
    ↓
推荐配队                  /pvp 配队 → 自动组队 + 评分
    ↓
培养顺序                  /pvp 培养顺序 → 全局排序
    ↓
下一步投入                推荐星尘/糖果/精英学习器方向
    ↓
（培养后回库存）           更新 my_pokemon.json
```

## 命令地图

```
/pvp 培养 <宝可梦> <联盟>          → 单只宝可梦详细信息 + 联盟适配
/pvp iv <宝可梦> <联盟> [IV]       → IV 排名查询
/pvp 添加 <宝可梦> <联盟> ...      → 录入库存
/pvp 我的 [联盟]                    → 查看库存
/pvp 评估我的 <联盟>                → 库存评估（配招/IV）
/pvp 配队 <联盟> [核心]            → 配队推荐 + 评分
/pvp 值得练 [联盟]                 → 培养优先级
/pvp 缺什么 [联盟]                 → 队伍缺口分析
/pvp 培养顺序                      → 全局培养顺序
```

## 依赖关系

```
值得练 ⬅ IV排名 ⬅ gamemaster
缺什么 ⬅ 配队推荐 ⬅ 值得练
培养顺序 ⬅ 值得练 + 缺什么
联盟适配 ⬅ gamemaster + calcCP
```
