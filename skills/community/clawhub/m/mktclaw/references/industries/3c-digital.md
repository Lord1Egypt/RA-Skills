## 1. 行业概览

### 1.1 市场规模与增长

| 细分品类 | 市场规模(2025) | 年增速 | 主要趋势 |

| 笔记本/PC | ≈ 2500 亿 | 3-5% | AI PC、轻薄化、办公场景 |

| 影音设备 | ≈ 600 亿 | 5-8% | 投影、VR/AR |

### 1.3 竞争格局特征

- **头部集中度高**：手机/PC 头部 5 家占据 80% 份额
- **参数党文化**：用户主动研究参数（芯片/影像/续航）
- **长决策周期**：高客单，比价+测评+体验
- **O2O 链路**：线上种草 → 线下体验 → 线上下单（或反之）
- **生态绑定**：苹果/华为/小米的生态护城河

## 5. 合规与监管

### 5.1 广告法专属限制

| 限制类型 | 禁用/限制 |
|---------|---------|
| 极限词 | "最""第一""顶级""全球首发"（无依据时） |
| 性能虚标 | 不得虚标芯片/续航/像素等参数 |
| 与竞品对比 | 不得贬低、不得使用未经授权的对比数据 |
| 5G/卫星/智能 | 不得夸大未实际具备的功能 |

### 5.2 行业特殊资质

| 业务类型 | 必备资质 |
|---------|---------|
| 入网许可 | 工信部入网许可证 |
| 3C 强制认证 | 中国强制性产品认证 |
| 无线电发射 | SRRC 核准 |
| 广告中的参数 | 必须有第三方测试报告 |

### 5.3 平台审核红线

| 平台 | 重点红线 |
|------|---------|
| 抖音 | 虚假测评、参数造假、低俗擦边 |
| B站 | 利益相关未声明、数据无来源 |
| 京东 | 虚假好评、参数虚标 |
| 小红书 | 未声明广告、虚假"自用推荐" |

## 6. 典型 Campaign 节奏

### 6.1 新品发布（手机/PC）

```
T-45：泄露悬念（媒体爆料 + 官方海报）
T-30：发布会官宣 + 媒体邀请
T-21：发布会直播（全网同步）
T-14：B站/抖音头部 UP 主首发测评（ embargo 解除）
T-7：腰部 UP 主测评 + 场景化内容
T0：开售日 + 直播爆发
T+7：用户口碑发酵 + 长尾种草
T+30：促销节点（首降价 / 套装）
```

### 6.2 大促节奏

```
T-30：策略 + 创意 Brief
T-21：素材制作 + UP 主预约
T-14：种草启动 + 蓄水
T-7：预售 + 会员营销
T-3：流量冲刺 + 直播预告
T0：爆发 + 全员直播
T+7：复盘 + 长尾
```

## 7. 常见雷区

| 雷区 | 典型表现 | 严重度 | 规避方法 |
|------|---------|:-----:|---------|
| 参数虚标 | 续航/像素/芯片性能夸大 | 🔴 致命 | 第三方检测 |
| 与竞品对比侵权 | 未授权使用竞品 logo / 数据 | 🔴 高 | 法务审阅 |
| 发布会翻车 | 现场 demo 失败 / 数据被质疑 | 🔴 高 | 多轮预演 + Plan B |
| 头部 UP 主差评 | B站顶流测评给出负面 | 🔴 高 | 提前沟通 + 风险预案 |
| 价格倒挂 | 线上线下不同价 / 渠道冲突 | 🔴 高 | 渠道分层 |
| 季节性错配 | 学生开学后才推学生机 | 🟡 中 | 提前 30 天布局 |
| 同质化 | "AI 影像"已成套路 | 🟡 中 | 引用 creative-diversity-protocol.md |

## 8. 推荐方法论组合

| 方法论 | 适用场景 |
|--------|---------|
| **5A + 雷军方法论** | 新品发布 |
| **SCAMPER** | 产品迭代创意 |
