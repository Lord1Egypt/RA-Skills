# 中医辨证参考

> 本文档收录中医辨证体系核心内容与常见证型辨识要点。
> 检索关键词：证型名（如脾虚湿困、肝郁气滞）、症状关键词（如失眠、胃胀）、脏腑名

## 一、八纲辨证

八纲是中医辨证的总纲，一切辨证均在此基础上展开。

| 纲领 | 辨别要点 | 舌象特征 | 脉象特征 |
|------|---------|---------|---------|
| **阴阳** | 总纲：阳证—实证、热证、表证；阴证—虚证、寒证、里证 | 阳证舌红苔黄；阴证舌淡苔白 | 阳证脉数有力；阴证脉沉迟无力 |
| **表里** | 表证—病在肌表（发热恶寒、头身痛）；里证—病入脏腑（但热不寒、腹痛泄泻） | 表证舌苔薄白；里证舌苔多变化 | 表证脉浮；里证脉沉 |
| **寒热** | 寒证—怕冷、喜温、面色白、小便清长；热证—发热、喜凉、面色红、小便短赤 | 寒证舌淡苔白润；热证舌红苔黄燥 | 寒证脉迟或紧；热证脉数 |
| **虚实** | 虚证—久病、体虚、声低气短；实证—新病、体壮、声高气粗 | 虚证舌淡嫩少苔；实证舌质苍老苔厚 | 虚证脉无力；实证脉有力 |

### 八纲复合证型举例

- **表寒证**：恶寒重、发热轻、无汗、头身痛、脉浮紧
- **表热证**：发热重、恶寒轻、有汗或无汗、口微渴、脉浮数
- **里寒证**：畏寒肢冷、腹痛喜温、呕吐清水、泄泻清稀、脉沉迟
- **里热证**：发热不恶寒、口渴、烦躁、便秘尿黄、脉数
- **虚寒证**：畏寒肢冷、面色㿠白、神疲乏力、舌淡胖、脉沉迟无力
- **实热证**：壮热、口渴、烦躁、便秘、舌红苔黄燥、脉洪数有力

## 三、常见复合证型

### 1. 肝郁脾虚
- **核心表现**：情绪抑郁或急躁 + 食少腹胀、大便溏泄
- **病机**：肝失疏泄，横逆犯脾，脾失健运
- **参考方**：逍遥散
- **舌脉**：舌淡红或暗红，苔薄白，脉弦缓

### 2. 心脾两虚
- **核心表现**：心悸失眠 + 食少乏力、面色萎黄
- **病机**：脾虚气血生化不足，心血亏虚，心神失养
- **参考方**：归脾汤
- **舌脉**：舌淡嫩，苔薄白，脉细弱

### 3. 肝肾阴虚
- **核心表现**：腰膝酸软 + 头晕耳鸣、五心烦热、遗精
- **病机**：肝阴与肾阴相互资生，肾阴不足导致肝阴亦虚
- **参考方**：杞菊地黄丸
- **舌脉**：舌红少苔，脉细数

### 4. 脾肾阳虚
- **核心表现**：腰膝冷痛 + 五更泄泻、形寒肢冷、水肿
- **病机**：脾阳虚日久累及肾阳，命门火衰不能温煦脾土
- **参考方**：附子理中丸、四神丸
- **舌脉**：舌淡胖有齿痕，苔白滑，脉沉迟无力

### 5. 痰湿内蕴
- **核心表现**：体形肥胖、头身困重、脘腹痞闷、痰多
- **病机**：脾虚运化失常，水湿停聚成痰
- **参考方**：二陈汤、温胆汤
- **舌脉**：舌淡胖苔白腻，脉滑

### 6. 湿热蕴结
- **核心表现**：脘腹痞闷 + 口苦口黏、身热不扬、小便短赤
- **病机**：湿邪与热邪相合，困遏气机
- **参考方**：三仁汤、龙胆泻肝汤
- **舌脉**：舌红苔黄腻，脉滑数
- **古籍参考**：《湿热病篇》薛生白湿温辨证

### 7. 气滞血瘀
- **核心表现**：胸胁胀痛或刺痛 + 痛有定处、面色晦暗
- **病机**：气机郁滞导致血行不畅，日久成瘀
- **参考方**：血府逐瘀汤
- **舌脉**：舌暗红或有瘀斑瘀点，脉弦涩

### 8. 气阴两虚
- **核心表现**：神疲乏力 + 口干咽燥、手足心热
- **病机**：久病耗气伤阴，气虚与阴虚并存
- **参考方**：生脉散
- **舌脉**：舌红少苔，脉细弱或细数

## 四、舌象辨证速查

### 4.1 舌质

| 舌质表现 | 主证 | 辨证意义 |
|---------|------|---------|
| 淡白舌 | 虚证、寒证 | 气血不足或阳虚 |
| 红舌 | 热证 | 实热或阴虚内热 |
| 绛舌 | 热盛 | 热入营血或阴虚火旺 |
| 紫舌 | 瘀证、寒证 | 气血瘀滞或寒凝血瘀 |
| 胖大有齿痕 | 脾虚、湿盛 | 脾阳虚或脾虚湿困 |
| 瘦薄舌 | 阴虚、血虚 | 阴津不足或气血亏虚 |

### 4.2 舌苔

| 舌苔表现 | 主证 | 辨证意义 |
|---------|------|---------|
| 白苔 | 表证、寒证 | 正常或风寒初起 |
| 黄苔 | 热证、里证 | 实热或湿热 |
| 灰黑苔 | 寒极或热极 | 寒盛（苔灰黑润）或热极（苔灰黑燥） |
| 腻苔 | 湿证、痰证 | 湿浊内蕴或痰饮 |
| 剥苔/无苔 | 阴虚、胃阴不足 | 阴津亏损 |
| 黄腻苔 | 湿热 | 湿热蕴结 |

### 4.3 常见舌象组合

| 舌象 | 常见证型 |
|------|---------|
| 舌淡胖有齿痕 + 苔白腻 | 脾虚湿困 / 脾肾阳虚 |
| 舌红少苔 | 阴虚（肝肾阴虚 / 胃阴虚） |
| 舌尖红 | 心火亢盛 |
| 舌两边红 | 肝胆有热 |
| 舌暗红有瘀斑 | 气滞血瘀 |
| 舌红苔黄腻 | 湿热蕴结 |
| 舌淡 + 苔薄白 | 气虚 / 血虚 |
| 舌红绛无苔 | 热入营血 / 阴虚火旺极 |

## 五、脉象辨证基础

| 脉象 | 特征 | 主证 |
|------|------|------|
| 浮脉 | 轻取即得，重按减弱 | 表证 |
| 沉脉 | 轻取不应，重按始得 | 里证 |
| 迟脉 | 一息不足四至 | 寒证 |
| 数脉 | 一息五至以上 | 热证 |
| 虚脉 | 三部脉举按皆无力 | 虚证 |
| 实脉 | 三部脉举按皆有力 | 实证 |
| 滑脉 | 往来流利，如珠走盘 | 痰湿、食积、妊娠 |
| 涩脉 | 往来艰涩，如轻刀刮竹 | 瘀血、精亏、血虚 |
| 弦脉 | 端直以长，如按琴弦 | 肝胆病、疼痛、痰饮 |
| 细脉 | 脉细如线，应指明显 | 气血两虚、湿邪 |

## 九、黄元御中气升降理论（《四圣心源》）

黄元御以"中气"为核心，构建了独特的气机升降理论：

- **中气**：脾胃为气机升降之枢纽。脾主升清，胃主降浊
- **左升**：肝随脾升，心血随之温煦上行
- **右降**：肺随胃降，肾水随之润泽下行
- **核心病机**：中气虚衰 → 升降失常 → 百病丛生
- **辨证要点**：
  - 脾虚不升 → 清阳下陷（腹泻、脱肛、内脏下垂）
  - 胃虚不降 → 浊阴上逆（呕吐、嗳气、痞满）
  - 肝郁不升 → 木克土（胁痛、腹胀、情志异常）
  - 肺失肃降 → 金不制木（咳嗽、气逆）

此理论对脾胃病、肝脾不调、气机升降失常类病症有重要参考价值。

## 十、现代体质辨识参考

基于王琦九种体质学说，结合现代人常见体质：

| 体质 | 核心特征 | 调养原则 |
|------|---------|---------|
| 平和质 | 体态适中、面色润泽、精力充沛、睡眠好 | 维持现状，顺应四时 |
| 气虚质 | 易疲乏、气短懒言、易感冒、出汗多 | 益气健脾，避免过劳 |
| 阳虚质 | 怕冷、手脚凉、面色苍白、喜热饮 | 温阳散寒，避寒保暖 |
| 阴虚质 | 口干、手足心热、潮热盗汗、大便干 | 滋阴降火，少食辛辣 |
| 痰湿质 | 体形肥胖、腹部松软、口黏腻、嗜睡 | 健脾利湿，少食肥甘 |
| 湿热质 | 面垢油光、口苦口干、身重困倦、痤疮 | 清热利湿，忌辛辣油腻 |
| 血瘀质 | 面色晦暗、易有瘀斑、唇色暗 | 活血化瘀，适度运动 |
| 气郁质 | 情志抑郁、多愁善感、胸胁胀满、叹气 | 疏肝解郁，调理情志 |
| 特禀质 | 易过敏、打喷嚏、皮肤敏感 | 益气固表，避免过敏原 |
