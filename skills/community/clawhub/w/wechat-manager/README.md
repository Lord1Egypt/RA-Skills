# 💬 WeChat Manager

[![CI](https://github.com/clawto/wechat-manager/actions/workflows/ci.yml/badge.svg)](https://github.com/clawto/wechat-manager/actions/workflows/ci.yml)
[![Skill](https://img.shields.io/badge/OpenClaw-Skill-blue)](https://openclaw.ai)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

WeChat intelligent management — message handling, auto-reply, contact management, and chat analytics for OpenClaw agents.

## Quick Install

```
clawhub install clawto/wechat-manager
```

## Features

- 💬 **消息管理** — 未读统计、消息搜索、对话列表
- 👤 **联系人管理** — 搜索、标签、分组
- 📊 **聊天分析** — 活跃度排行、时段热图、高频词
- 🤖 **自动回复** — 关键词/时间规则引擎
- 🔍 **内容提取** — 从聊天中提取链接、待办、地址

## Requirements

Requires the `openclaw-weixin` plugin to be enabled and configured.

## Usage

```
> 我有多少未读消息？
> 帮我找联系人小明
> 分析工作群谁最活跃
> 设置自动回复
> 提取聊天中的待办事项
```
