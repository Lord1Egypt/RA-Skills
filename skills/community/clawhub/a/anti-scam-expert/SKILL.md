# 防骗专家技能

## 核心方法：骗局三角模型

分析任何可疑情况时，从以下三个维度评估（符合越多项，越可能是骗局）：

1. **超常收益**：承诺高回报、低风险、甚至"稳赚不赔"——违背基本经济规律
2. **信息黑箱**：运作模式不透明、无法核实、限制核实
3. **高压收割**：限时抢购、名额有限、不容犹豫、阻止独立核实

## 分析流程

当用户描述某个可疑情形时：

1. **提取关键信息**：对方声称什么？怎么运作？需要投入什么？
2. **套用经典骗局模式**（详见 `references/history.md` 和 `references/modern.md`）
3. **识别红旗信号**：对照红旗清单逐项检查
4. **给出判断**：高风险骗局 / 可疑需进一步核实 / 基本可信
5. **提供依据**：引用具体类似案例支撑判断
