# Exterior-Releasing Formula Usage

## When to Use
- 患者确诊为**外感表证**，表现为恶寒发热、头痛身痛、脉浮等；
- **无汗或有汗不多**，排除表虚证；
- **无禁忌人群特征**（如新产妇人、失血患者、表虚自汗者）。

## Execution Steps

### 1. 辨证分型与选方
- **表寒证**（恶寒重、发热轻、无汗、脉浮紧）→ 选用**辛温解表剂**（如麻黄汤）；
- **表热证**（发热重、微恶风寒、咽痛口渴、脉浮数）→ 选用**辛凉解表剂**（如银翘散）。

### 2. 煎煮方法
- 所有解表剂均属“辛散轻宣之品”，必须**勿过煮**，以“香气大出，即取服”为原则；
- 银翘散宜用**鲜芦根煎汤送服**，轻症日服三次，重症可增至四次。

### 3. 服药后处理
- 服药后应**温覆取微汗**，但**不可发汗太过**，以防损伤正气；
- 麻黄汤类强发汗方应**中病即止，不可过服**。

### 4. 禁忌人群
- **禁用**于表虚有汗者、新产妇人、失血患者；
- 银翘散虽较温和，仍**避免用于阳虚外感者**。

> 本规范体现“治上焦如羽，非轻不举”的用药原则，强调药物质轻、煎煮时间短以保全辛散之性。
