# 螺旋器结构与听觉机制

## 适用场景
- 研究听觉换能过程
- 解释高频/低频听力障碍的解剖基础
- 教学耳蜗组织学与听觉生理学

## 执行步骤
1. **识别螺旋器组成**：
   - **支持细胞**：柱细胞（围成内隧道）、指细胞（托住毛细胞）
   - **毛细胞**：
     - 内毛细胞（1列，U形静纤毛）
     - 外毛细胞（3–4列，V/W形静纤毛）

2. **分析关键结构关系**：
   - 静纤毛呈阶梯状排列，外侧高于内侧
   - 外毛细胞长静纤毛插入**盖膜**胶质中
   - 毛细胞底部与耳蜗神经节树突形成突触

3. **追踪听觉传导路径**：
   - 声波→鼓膜→听骨链→卵圆窗→前庭阶外淋巴→膜蜗管内淋巴→基底膜振动
   - 基底膜振动使静纤毛弯曲→毛细胞兴奋→神经信号传入中枢

4. **应用频率定位原理**：
   - **蜗底**：基底膜窄、听弦短 → 共振**高频**
   - **蜗顶**：基底膜宽、听弦长 → 共振**低频**
   - 损伤部位决定听力障碍类型（高音/低音）

> 注意：本技能仅适用于耳蜗听觉功能，不涉及前庭平衡机制。
