# 离婚财产分割类案件

案号：最高人民检察院发布第四批6件检察机关贯彻实施民法典典型案例之一：刘某珍与康某卿离婚纠纷再审检察建议案
概括：最高检发布典型案例，针对刘某珍与康某卿的离婚纠纷，检察机关通过提出再审检察建议，监督纠正原审错误，以贯彻实施民法典。

案号：(2023)黑01民终13154号
概括：郭某改与李某考离婚后，就车辆、银行存款等未分割的夫妻共同财产产生纠纷，法院对双方有争议且证据充分的财产进行了分割，对涉及案外人或证据不足的主张不予处理。

案号：(2024)苏09民终2396号
概括：王某飞与徐某华离婚时约定将房屋赠与婚生女，后房屋被征收，王某飞主张撤销赠与，法院认为离婚协议中的赠与条款不可单方撤销，征收补偿款应归女儿所有。

案号：(2023)渝05认港2号
概括：涂某翠向内地法院申请认可香港特别行政区法院作出的离婚判决，内地法院经审查，依据相关司法协助安排，裁定认可了香港离婚判决的法律效力。

案号：(2024)粤14民终99号
概括：梁某玲与温某雄离婚后，因财产分割纠纷的管辖问题产生争议，法院认定双方在《离婚协议书》中约定的管辖条款有效，应据此确定管辖法院。

案号：最高人民检察院发布10起民事再审检察建议典型案例之九：曹某与田某离婚后财产纠纷再审检察建议案
概括：最高检发布典型案例，针对曹某与田某的离婚后财产纠纷，检察机关通过提出再审检察建议，对原审裁判进行法律监督。

案号：司法部发布4个法律援助工作指导案例之三：黑龙江省齐齐哈尔市龙江县法律援助中心对受家庭暴力妇女马某提供法律援助案
概括：马某因长期遭受家暴诉请离婚并分割财产，法律援助律师协助收集证据并获得检察机关支持起诉，最终通过法院调解成功离婚并分得财产折价款。

案号：(2023)京03民终2580号
概括：胡某在协议离婚一年多后，发现前夫刘某在婚内存在重大过错（离婚后迅速再婚生子），诉请离婚损害赔偿，法院适用《民法典》新规支持了其诉求。

案号：(2022)甘0702民初6284号
概括：王娜与蒋铁军离婚后，就婚姻期间与蒋铁军父母共同购置的房产等家庭共有财产提起分割诉讼，法院考虑蒋铁军存在过错，判决王娜多分财产。

案号：(2021)津03民终4835号
概括：王某1与张某离婚后，就离婚诉讼期间及之后新发现的银行存款、保险、收入等未分割的夫妻共同财产产生纠纷，二审法院重新核算后改判了分割数额。
