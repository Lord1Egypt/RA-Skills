# 知识产权侵权投诉函

## 适用场景
发现他人未经许可使用你的知识产权（商标/著作权/专利），向侵权方或平台发送正式投诉。

## 模板正文

```
知识产权侵权投诉函

致：[侵权方名称/平台名称]
地址：[地址]

投诉人：[姓名/公司名称]
联系人：[姓名]
联系电话：[电话]
电子邮箱：[邮箱]

一、权利证明
投诉人系以下知识产权的权利人：
[ ] 商标：注册号[    ]，核定使用商品/服务第[  ]类
[ ] 著作权：作品名称《    》，登记号[    ]
[ ] 专利：专利号[    ]，专利名称[    ]

二、侵权事实
[具体描述侵权情况]
- 侵权商品/服务名称：
- 侵权链接/位置：
- 侵权时间：
- 侵权行为描述：

三、侵权分析
1. 对方商品/服务/内容与投诉人权利保护范围的对比
2. 对方使用行为不具有合法授权
3. 该行为构成对投诉人[商标权/著作权/专利权]的侵害

四、法律依据
根据《中华人民共和国[商标法/著作权法/专利法]》第X条...

五、要求
请贵方于收到本函后[X]日内：
1. 立即停止侵权行为（下架/删除/停止使用）
2. 书面回复处理结果
3. 保留追究法律责任的权利

如逾期未处理，投诉人将采取进一步法律行动（包括但不限于向行政主管机关投诉、向法院提起诉讼）。

投诉人（签名/盖章）：
日期：    年   月   日

附件：
1. 权利证明文件复印件
2. 侵权证据材料
3. 投诉人身份证明
```

## 注意事项
- 先做证据保全（截图/录屏/公证），再发函
- 电商平台有专门的知识产权投诉通道（如：阿里巴巴知识产权保护平台）
- 发函不等于对方必须响应。无回复或拒绝 → 向行政机关投诉或起诉
- 不要虚张声势：声称要「追究法律责任」但永远不行动，反而削弱威慑力
