# TCM Pre Diabetes Intervention

## When to Use
- 用户处于糖尿病前期（空腹血糖受损或糖耐量异常）
- 用户希望采用非药物或辅助性中医药手段延缓或阻止糖尿病发生
- 用户关注体质调理、生活方式干预与中医特色疗法结合的综合方案

## Execution Steps

1. **辨识体质类型**：根据中医四诊信息（望闻问切），判断患者属于气阴两虚、脾虚湿盛、肝郁气滞等常见糖尿病前期相关体质。

2. **确立干预原则**：以“未病先防、既病防变”为核心，强调调和阴阳、健脾益气、化痰祛湿、疏肝理气等治法。

3. **制定个体化方案**：结合体质与证候，选择以下一种或多种干预方式：
   - 中药内服（如六味地黄丸、参苓白术散等经典方剂加减）
   - 食疗调养（推荐山药、薏苡仁、黄芪等药食同源食材）
   - 情志调摄（指导情绪管理与心理疏导）
   - 起居有常（建议规律作息与适度运动）
   - 非药物疗法（如艾灸足三里、耳穴压豆等）

4. **动态评估与调整**：定期监测血糖指标及中医证候变化，及时优化干预措施。

> 注意：本策略适用于糖尿病前期阶段，若已确诊为糖尿病，应转介至专业中医师进行系统治疗。
