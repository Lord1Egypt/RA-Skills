## What It Does

### Core Features

- **Typo Correction**: Similar characters (的/地/得), word order, punctuation
- **Expression Refinement**: Verbose → concise, casual → professional, mixed Chinese/English → unified
- **Semantic Consistency**: Same concept uses same name throughout, contradictions flagged
- **Style Uniformity**: Paragraph length, formatting, heading hierarchy balanced

### Problems It Solves

❌ "Sent the email, then spotted a typo"
❌ "Written document sounds too casual, not professional"
❌ "Report uses inconsistent terminology, confuses readers"
❌ "Sent without review, expression was a mess"

✅ "One-click polish, send with confidence"
✅ "Semantic check ensures consistency"
✅ "Professional language, precise wording"

### Output Example

**Before:**
> Today I went to that library, the return date is tomorrow, I think the environment there is good, just a bit hot.

**After:**
> Today I visited that library — books are due tomorrow. Overall the environment is pleasant, though somewhat stuffy.
