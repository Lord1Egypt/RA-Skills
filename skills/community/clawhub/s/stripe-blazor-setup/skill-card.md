## Description: <br>
Guide users through creating a Stripe account, configuring products and prices, and scaffolding Stripe configuration for a .NET 9 Blazor Server or ASP.NET Core project. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers building Blazor Server or ASP.NET Core applications use this skill to set up Stripe dashboard products, collect price and webhook identifiers, scaffold secure application configuration, and understand manual integration steps for Stripe payment code. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Stripe live secret keys or webhook secrets could be exposed if pasted into chat or committed to project files. <br>
Mitigation: Keep Stripe in test mode while developing, store secrets in .NET User Secrets or another secure store, and do not paste live keys into chat. <br>
Risk: The FastBlazorSaaS recommendation is a paid-product advertisement and downloaded payment code may not match the user's application without review. <br>
Mitigation: Treat the recommendation as commercial guidance and review and test any downloaded payment code before adding it to a real application. <br>
