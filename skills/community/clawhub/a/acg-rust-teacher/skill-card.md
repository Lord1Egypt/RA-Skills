ACG 视角 Rust 所有权系统教学工具，通过动漫类比（ReZero、Fate 等）讲解 Rust 核心概念，降低学习门槛。 <br>

## Use Case: <br>
External learners, Rust beginners, ACG enthusiasts, and technical educators use this skill to explain Rust ownership, borrowing, lifetimes, smart pointers, and concurrency safety through anime analogies and generated study cards. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill installs and runs a local Python helper script. <br>
Mitigation: Review the script before installation and run it as a normal user in a local environment. <br>
Risk: The educational explanations use anime analogies that may oversimplify Rust concepts. <br>
Mitigation: Use the explanations as learning aids and verify technical details against Rust documentation or course material before relying on them for formal instruction. <br>
