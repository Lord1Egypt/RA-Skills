# AI Content Repurposer 🔄✨

**Stop creating content once. One article → Twitter thread, LinkedIn post, newsletter, and video script in under 60 seconds.**

## What It Does

Write one long-form piece — a blog post, podcast transcript, YouTube video, or newsletter. The AI Content Repurposer intelligently extracts the core narrative and creates platform-optimised versions for every channel you care about. No more staring at a blank LinkedIn compose window.

## Why Buy This

- **10x your output** — create once, post 4+ times
- **Platform-native formats** — each output is built for *that* platform's audience, not copy-pasted
- **No more writer's block** — the prompting system handles structure, you just review
- **Brand voice preserved** — configure tone once, every output stays on-brand
- **Quality guaranteed** — automated checks catch length, tone, quote accuracy before you publish
- **Batch processing** — run through 20 articles while you sleep

## Quick Start

```bash
# Install
openclaw skill install ai-content-repurposer

# One command — repurpose into everything
openclaw skill run ai-content-repurposer --input https://myblog.com/post --output all

# Outputs land in:
#   output/twitter/thread-2025-01-15.md
#   output/linkedin/post-2025-01-15.md
#   output/newsletter/edition-2025-01-15.md
#   output/video/script-2025-01-15.md
```
