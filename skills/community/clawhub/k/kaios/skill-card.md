## Use Case: <br>
Developers and agent operators use this skill to set up AI assistant identity, user context, persistent memory, heartbeat-style task planning, and multi-agent collaboration files. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can encourage persistent memory, scheduled behavior, and autonomous task execution without strong scoping or retention safeguards. <br>
Mitigation: Use it in a controlled workspace, review proposed file writes, keep memory paths narrow, and confirm any scheduled jobs or kept agent sessions before enabling them. <br>
Risk: Security evidence notes possible gateway configuration changes and Discord channel management when the workflow is applied broadly. <br>
Mitigation: Grant Discord and gateway permissions narrowly, inspect configuration changes before applying them, and avoid broad workspace or server-level access. <br>
