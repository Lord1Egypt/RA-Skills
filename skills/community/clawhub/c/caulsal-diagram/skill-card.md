## Description: <br>
A causal diagram and DAG helper that generates PNG diagrams and Mermaid code for visualizing relationships between variables. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[lyhue1991](https://clawhub.ai/user/lyhue1991) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, analysts, and researchers use this skill to turn stated causal relationships into causal diagrams, DAG visualizations, Mermaid snippets, and PNG assets for documents or presentations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The PNG generator writes to the output path the user provides and may overwrite an existing file. <br>
Mitigation: Choose a new or disposable output path and review the command before running it. <br>
Risk: Generated diagrams can make incorrect user-provided causal assumptions look authoritative. <br>
Mitigation: Review nodes, edges, and causal direction with domain knowledge before using diagrams for decisions. <br>
