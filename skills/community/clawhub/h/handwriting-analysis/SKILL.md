# 刑侦笔迹分析技能

## 概述

本技能基于笔迹学（Graphology）原理，通过分析手写笔迹的视觉特征，推断书写者的性格特质、心理状态和行为倾向。适用于刑侦调查、人员评估、心理分析等场景。

## 核心分析维度

笔迹分析从以下 **8 大维度** 进行系统评估：

| 维度 | 观察要点 | 关联性格 |
|------|---------|---------|
| **字形大小** | 字体大小、一致性 | 自信、自我认知、空间需求 |
| **倾斜角度** | 向右/向左/垂直 | 情感表达、社交倾向、理性程度 |
| **书写压力** | 笔压轻重、均匀度 | 意志力、情绪强度、能量水平 |
| **字间距** | 字与字之间距离 | 社交边界、独立性、亲密需求 |
| **行间距** | 行与行之间距离 | 思维条理、计划性、边界感 |
| **基线走向** | 水平/上扬/下倾 | 情绪稳定性、目标导向、悲观倾向 |
| **连笔程度** | 字母/汉字连接 | 思维速度、整体性思维、创造力 |
| **签名特征** | 签名与正文关系 | 自我认同、社交面具、真实性 |

## 分析流程

### 第一步：图像预处理

用户提供笔迹图像后，需要：
1. 确认图像清晰度（建议 300 DPI 以上）
2. 识别书写工具（钢笔/圆珠笔/铅笔/毛笔）
3. 判断书写载体（白纸/格纸/横线纸）
4. 记录书写环境因素（站立/坐着、情绪状态）

### 第二步：特征提取

对笔迹图像进行逐项特征提取，填写《笔迹特征记录表》：

```
【笔迹特征记录表】

一、字形大小
□ 大字（>5mm）：自信、外向、需要关注
□ 中字（3-5mm）：平衡、适应性强
□ 小字（<3mm）：专注、内向、注重细节
□ 大小不一：情绪波动、适应性差

二、倾斜角度
□ 右倾（>5°）：情感外露、善于表达、冲动
□ 左倾（< -5°）：内敛、自我保护、批判性
□ 垂直：理性、自控、客观

三、书写压力
□ 重压：意志坚定、情绪强烈、可能固执
□ 轻压：敏感、灵活、适应性强
□ 压力不均：情绪不稳定

四、字间距
□ 宽间距：需要独立空间、边界感强
□ 窄间距：需要亲密关系、边界感弱
□ 间距不均：人际关系不稳定

五、行间距
□ 宽行距：思维清晰、有条理
□ 窄行距：思维密集、可能焦虑
□ 行距不均：计划性差

六、基线走向
□ 上扬：乐观、目标导向
□ 下倾：悲观、消极倾向
□ 波动：情绪波动大
□ 水平：情绪稳定、自律

七、连笔程度
□ 高连笔：思维敏捷、整体思维
□ 低连笔：谨慎、分析型思维
□ 适中：平衡

八、签名特征
□ 签名>正文：自我展示欲强
□ 签名<正文：低调、可能隐藏自我
□ 签名风格迥异：社交面具明显
```

### 第三步：综合分析

根据特征组合进行综合研判，参考 `references/personality-traits.md` 中的特征-性格映射表。

### 第四步：生成报告

输出结构化分析报告，包含：
- 各维度特征描述
- 性格特质推断（置信度）
- 心理状态评估
- 行为倾向预测
- 注意事项与局限性说明

## 使用方法

### 基本用法

用户上传笔迹图片后，直接描述分析需求：

```
请分析这份笔迹，推断书写者的性格特征
```

### 高级用法

指定分析维度或场景：

```
请从刑侦角度分析这份笔迹，重点关注签名特征和压力变化
这份笔迹是否存在伪装书写的迹象？
对比这两份笔迹，判断是否为同一人书写
```

## 重要声明

⚠️ **笔迹分析的局限性**

1. 笔迹分析属于投射技术，具有一定主观性
2. 结论应作为参考线索，不作为唯一判断依据
3. 书写环境、工具、身体状况会影响笔迹
4. 专业鉴定需结合其他证据综合判断
5. 存在伪装书写的可能，需识别异常特征
