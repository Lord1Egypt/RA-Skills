# Case Study Generator

Turn any customer success story into a complete, publication-ready case study package — in minutes, not days.

## What You Get

Run 4 prompts. Get everything:

1. **Full narrative** (700–1,000 words) — the human story that makes buyers say "that's me"
2. **Results & metrics section** — ROI table, before/after comparisons, data viz recommendations
3. **5 pull quotes** — one for each stage of the B2B buying journey, plus a testimonial and ad one-liners
4. **Multi-format package** — blog post, sales one-pager, LinkedIn story, email snippet

## Who This Is For

B2B companies, agencies, SaaS founders, consultants, and freelancers who need case studies that close deals — not generic "we helped a client" summaries.

## Quick Start

1. Gather your inputs (see SKILL.md for the full list — takes about 5 minutes)
2. Run Prompt 1 → paste result into Prompt 2 → repeat through Prompt 4
3. Review output, get customer approval on quotes, publish
