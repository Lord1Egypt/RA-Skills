## 一、期刊筛选策略

### 期刊层次体系

| 层次 | 说明 | 投稿难度 | 审稿周期 |
|:---:|------|:-------:|:-------:|
| Nature/Science/Cell | 顶刊（跨学科） | ⭐⭐⭐⭐⭐ | 3-6月 |
| 学科顶刊 | 各学科TOP期刊 | ⭐⭐⭐⭐ | 2-4月 |
| 中科院1区/2区 | 高水平 | ⭐⭐⭐ | 2-4月 |
| 中科院3区 | 中等水平 | ⭐⭐ | 2-6月 |
| 中科院4区/ESCI | 发表门槛 | ⭐ | 1-4月 |
| 中文核心（北大/CSSCI） | 国内高水平 | ⭐⭐⭐ | 3-8月 |
| 普通中文期刊 | 发表门槛最低 | ⭐ | 1-3月 |

### 期刊选择标准
1. **主题匹配度**：期刊scope是否覆盖该研究方向
2. **期刊水平**：影响因子、分区、声誉
3. **审稿周期**：官方公布的平均审稿周期
4. **录用率**：公开的录稿比例（一般在10-30%）
5. **是否OA**：开放获取的版面费（APC）通常在$1000-5000
6. **是否有版面费补贴**：很多机构对OA文章有补贴

## 二、参考文献格式规范

### 常用参考文献格式

| 格式 | 主要适用领域 | 特点 |
|:---:|------------|------|
| **APA 7th** | 心理学、教育学、社会科学 | 作者-日期制，括号引用 |
| **MLA 9th** | 人文学科（语言、文学） | 作者-页码制 |
| **IEEE** | 工程、计算机、电子 | 序号制 [1] |
| **Chicago** | 历史、艺术、人文 | 脚注/作者-日期两种 |
| **GB/T 7714** | 中国国家标准 | 中国期刊常用 |
| **Vancouver** | 医学、生物 | 序号制，AMA类似 |

### 常见格式示例

```
APA 7th（期刊论文）：
Zhang, L., Wang, Y., & Li, X. (2025). Deep learning in medical imaging: A comprehensive review. 
Nature Reviews Bioengineering, 3(2), 115-132. https://doi.org/10.1038/s44286-025-00001-x

IEEE（期刊论文）：
[1] L. Zhang, Y. Wang, and X. Li, "Deep learning in medical imaging: A comprehensive review,"
Nature Reviews Bioengineering, vol. 3, no. 2, pp. 115-132, 2025.

GB/T 7714（期刊论文）：
张三, 李四, 王五. 基于深度学习的心电图分析[J]. 中国生物医学工程学报, 2025, 44(2): 123-135.

APA 7th（会议论文）：
Chen, J., & Liu, H. (2025). A novel approach for protein structure prediction. 
Proceedings of the 2025 International Conference on Bioinformatics (pp. 45-52). ACM.

APA 7th（图书）：
Smith, J. A. (2024). Machine Learning for Healthcare (3rd ed.). MIT Press.
```

## 三、投稿流程

```
1. 投稿前准备
   ├── 确认期刊格式要求（下载作者指南Author Guidelines）
   ├── 按要求排版论文
   ├── 准备投稿信（Cover Letter）
   └── 准备补充材料（Supplementary Materials）

2. 在线投稿系统
   ├── 创建账号或登录
   ├── 选择文章类型（Research Article/Review/Case Report等）
   ├── 填写作者信息、关键词、推荐审稿人
   ├── 上传稿件和相关文件
   └── 确认提交

3. 编辑初审
   ├── 格式检查（不符合要求可能直接退回）
   ├── 主题匹配度检查
   ├── 学术水平初步判断（Desk Reject Rate约20-40%）
   └── 通过后送外审

4. 同行评审
   ├── 一般2-4位审稿人
   ├── 审稿周期：2周-3个月
   ├── 审稿结果：Major Revision / Minor Revision / Reject / Accept
   └── 大修通常给30-60天

5. 修改和重新提交
   ├── 逐条回复审稿意见（Response to Reviewers）
   ├── 标注修改处（使用修订模式或高亮）
   ├── 修改说明不要顶撞审稿人
   └── 一般需要1-2轮修改

6. 录用和发表
   ├── 收到录用通知（Acceptance Letter）
   ├── 签署版权转让协议
   ├── 支付版面费（如适用）
   └── 在线发表（Online First）→ 纸质发表
```

## 四、投稿信（Cover Letter）模板结构

```
[日期]

[主编姓名]
[期刊名称]
[期刊地址]

请考虑我们的手稿"[论文标题]"发表在[期刊名称]上。

敬礼

[通讯作者姓名]
[所属机构]
[邮箱]
[电话]
```

## 五、审稿意见回复策略

### 基本原则
1. **态度谦逊**：感谢审稿人的时间和建议
2. **逐条回复**：每条意见都要回复
3. **标注修改**：告诉审稿人做了什么修改、在哪一页
4. **有理有据**：不同意时要给出理由，而不是直接反驳
5. **时间尊重**：如果要多花时间，可以申请延期

### 回复结构

```
=== 致审稿人 ===
感谢审稿人的宝贵意见，这些建议对提升文章质量非常有帮助。
我们已按您的建议进行了修改，具体回复如下：

=== 审稿人意见 1 ===
审稿人: "建议补充XX方法的对比实验..."
作者回复: 同意审稿人的意见。我们补充了与XX和YY方法的对比实验
（详见修改稿第5页表2）。新增结果表明本方法在XX指标上提升约5%。

[以此类推逐条回复...]
```

## 六、学术道德和出版伦理

### 必须避免的行为
1. **抄袭/剽窃**：直接复制他人文字和数据
2. **一稿多投**：同一稿件同时投多个期刊
3. **重复发表**：已发表内容再次以"新"文章发表
4. **数据造假**：编造或篡改数据
5. **作者争议**：添加没有实质贡献的人为作者
6. **未披露利益冲突**：如受资助企业有利益关系未声明

### 合理使用AI工具
- 多数期刊要求声明使用了AI辅助工具
- AI不能列为作者
- AI生成的内容由作者承担学术责任

## 七、OA与订阅期刊对比

| 维度 | OA期刊 | 订阅期刊 |
|:----:|:------:|:--------:|
| 读者免费访问 | ✅ 是 | ❌ 需付费/机构订阅 |
| 作者付费 | ✅ APC（$1000-5000） | ❌ 一般免费 |
| 引用优势 | 略有优势（OA优势） | 传统模式 |
| 出版速度 | 通常较快 | 通常较慢 |
| 期刊质量 | 差异大（有掠夺性OA） | 普遍较高 |
| 版权 | 作者保留（CC协议） | 转让给出版社 |
