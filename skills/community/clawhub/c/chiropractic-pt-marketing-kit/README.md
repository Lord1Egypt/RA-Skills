## Why This Beats Generic AI Tools

Generic AI tools write healthcare marketing copy that can get you in trouble:
- HIPAA violations in email subject lines (naming conditions or treatments)
- FTC violations in testimonials ("I was cured in 3 visits!" — not compliant)
- State board violations (unsubstantiated "best" and "leading" claims)
- ACA/APTA ethics violations (comparing yourself favorably to MDs or other providers)

This skill enforces all of it at generation time. Every prompt has a compliance checklist built in.
