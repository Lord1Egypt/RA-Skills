# 质检流水线 — 参考资料

## 一、去AI味写作规则（核心蒸馏）

### 真人学者写作的"不完美感"
真人写作有以下特征，AI写作通常没有：

1. **句子长度波动大**：有长有短，不会每句都是"主谓宾+修饰语"的均匀结构
2. **偶尔的非标准衔接**：用"值得注意的是""更值得关注的是"而非"此外""另外"
3. **观点性表述**：敢于用"笔者认为""从实践来看""经验表明"，而非纯客观描述
4. **数据嵌入自然**：不会每段都"数据显示xxx"，而是融入叙述中
5. **段落间有跳跃感**：不会每段都严格"主题句→论证→小结"
6. **偶尔的口语化学术词**："说白了就是""简单来讲""换个角度看"（但不是纯口语）
7. **引用的节奏**：不是密集堆砌[1][2][3]，而是该引才引
8. **不确定的表述**：用"可能""或许""在一定程度上"表示学术审慎

### 禁止清单（AI味重灾区）
- ❌ "首先...其次...最后..."（太整齐）
- ❌ "综上所述""总而言之"（AI高频词）
- ❌ "值得注意的是"（出现超过2次就假）
- ❌ 每段结尾都小结
- ❌ 所有段落等长（±10%）
- ❌ 完全没有个人观点
- ❌ 过度使用"显著""有效""积极"等正面形容词
- ❌ 连续3个以上排比句
- ❌ "随着...的发展/进步"开头

### 参考范文特征（从6篇核心期刊论文蒸馏）

**王佳2026《公立互联网医院患者满意度》**：
- 散文式引言，不急于切入主题
- 文献综述用自己的话重述，不是罗列
- 数据讨论有个人解读

**梁爽2025《患者在线就诊体验》**：
- 因果链推导清晰但不刻板
- 偶尔用反问句引发思考
- 结果讨论有"出乎意料"的表述

**孙丽君2025《成都公立医院智慧化服务》**：
- 实践导向，语言接地气
- 政策引用自然融入
- 建议部分务实，不说空话

## 二、五层质检标准

### L1 存在性检查（豆包/快速模型）
- [ ] 所有承诺的文件是否生成
- [ ] 文件命名是否规范
- [ ] 字数是否达标
- [ ] 格式文件（docx/pdf/pptx）是否可打开

### L2 真实性检查（DeepSeek/中等模型）
- [ ] 有无乱码/占位符（xxx/TODO/placeholder）
- [ ] 有无跑题内容
- [ ] 数据是否自洽（前后矛盾）
- [ ] 参考文献是否真实存在
- [ ] 有无编造的统计数据

### L3 准确性检查（另一agent交叉）
- [ ] 格式是否符合要求（字体/字号/行距/页边距）
- [ ] 文件命名是否按规范
- [ ] 引用编号是否连续无遗漏
- [ ] 表格/图表编号是否正确
- [ ] 语言风格是否统一

### L4 专业审稿（editor-in-chief + academic-reviewer）
- [ ] 逻辑链条是否完整
- [ ] 论点是否有充分支撑
- [ ] 文献综述是否精简（砍30-40%）
- [ ] 讨论部分是否有深度
- [ ] 结论是否回答了研究问题

### L5 CEO最终把关（main）
- [ ] 综合L1-L4的打分
- [ ] 判断是否达到交付标准
- [ ] 决定是否退回修改

## 三、学术写作质量评分卡

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 选题价值 | 15% | 是否填补空白、是否有实践意义 |
| 文献综述 | 15% | 是否精简、是否有批判性、是否聚焦 |
| 方法严谨性 | 20% | 量表选择、样本设计、分析流程 |
| 结果呈现 | 20% | 表格规范、统计正确、解读到位 |
| 讨论深度 | 15% | 是否与已有研究对话、是否有新见 |
| 写作风格 | 15% | 去AI味、学术规范、可读性 |

### 评分等级
- **90+**：可直接投稿
- **80-89**：小修后可投
- **70-79**：需重点修改
- **60-69**：需大改或重写
- **<60**：不合格，退回

## 四、Word文档格式规范（中文核心期刊）

| 元素 | 格式 |
|------|------|
| 标题 | 小二号黑体，居中 |
| 作者 | 小四号楷体，居中 |
| 摘要 | 五号楷体 |
| 关键词 | 五号楷体 |
| 一级标题 | 小三号黑体 |
| 二级标题 | 四号黑体 |
| 三级标题 | 小四号黑体 |
| 正文 | 小四号宋体 |

| 图题 | 五号宋体，图下方 |

| 纸张 | A4 |

## 五、常见问题速查

### 参考文献GB/T 7714格式
- 期刊：作者.题名[J].刊名,年,卷(期):起止页码.
- 专著：作者.书名[M].出版地:出版社,年.
- 学位论文：作者.题名[D].城市:学校,年.
- 网络文献：作者.题名[EB/OL].URL,访问日期.

### 三线表规则
- 只有顶线、栏目线、底线三条横线
- 无竖线
- 表注在底线以下
- 表题在表上方
