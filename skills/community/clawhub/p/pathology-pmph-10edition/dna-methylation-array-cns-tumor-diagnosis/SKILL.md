# DNA甲基化芯片在中枢神经系统肿瘤诊断中的应用与局限

## 何时使用
- 需对中枢神经系统肿瘤进行分子分型；
- 病理形态学特征不典型或诊断存在不确定性；
- 已获取满足质量要求的肿瘤组织DNA样本（肿瘤细胞比例建议 >70%）；
- 具备标准化甲基化芯片平台及经验证的参考数据库。

## 如何执行
1. **确认适用性**：
   - 检查样本中肿瘤细胞比例是否足够高（通常 >70%）；
   - 排除已知甲基化谱重叠的肿瘤类型（如PLNTY与节细胞胶质瘤）。

2. **执行检测**：
   - 使用亚硫酸氢盐处理DNA样本；
   - 利用特异性探针与甲基化CpG位点杂交，通过荧光信号强度判定甲基化状态；
   - 将获得的全基因组甲基化谱输入标准化生物信息学流程。

3. **结果解读**：
   - 将未知样本与参考数据库进行聚类比对，输出分子分型及置信度评分；
   - 若置信度低、结果与形态学矛盾，或聚类结果模糊，必须结合其他分子检测（如IDH突变、1p/19q共缺失）或临床信息综合判断。

4. **注意事项**：
   - 避免在低肿瘤纯度样本中单独依赖甲基化芯片结果；
   - 注意不同聚类算法或参考数据集可能导致结果差异，优先使用WHO推荐或经临床验证的分析流程。

> **输出形式**：肿瘤分子分型结果 + 置信度评分。若不可靠，触发警报并建议多模态整合诊断。
