# AIOS 数据处理规则

只用于 `aios-apps-invoke-cli` 的真实返回结果。

## 何时用 Python

出现以下任一需求就用 Python：

- 统计、求和、计数、平均、中位数
- 分组、聚合、排序、排名、去重
- 跨多条记录筛选
- 比率、百分比、环比、同比
- 时间窗口分析
- 任意派生计算

仅在“读取单条记录的单个简单字段且不做计算/比较”时可直接读值。

## 最低流程

1. 读取 JSON。
2. 校验结构和必需字段。
3. 处理 `null`、空字符串、缺失值、异常值。
4. 数值先校验类型再计算。
5. 日期数值按 OADate 转换后再分析。

OADate：

- `datetime = 1899-12-30 + OADate`

## 输出必须说明

- 用了哪些字段
- 做了哪些过滤、分组或计算
- 实际处理了多少条记录
- 是否跳过无效数据
- 是否存在不确定性
