## Known Risks and Mitigations: <br>
Risk: The skill can propose Python, pip, and uv commands that install or execute local code. <br>
Mitigation: Use an isolated environment, consider pinning the duckduckgo-search package version, and review proposed commands before running them. <br>
Risk: Search queries may expose sensitive private content to an external search service. <br>
Mitigation: Do not search for credentials, secrets, or sensitive private content. <br>
