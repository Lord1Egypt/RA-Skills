# 图片文件清单

## 需要添加的图片

请将以下 8 张 PNG 图片放入 `assets/` 文件夹：

1. 白字黑边.png - 模板 1：白字黑边
2. 黑字白边.png - 模板 2：黑字白边
3. 黄字黑边.png - 模板 3：黄字黑边
4. 红字白边.png - 模板 4：红字白边
5. 黑底白字.png - 模板 5：黑底白字
6. 白底黑字.png - 模板 6：白底黑字
7. 黄底黑字.png - 模板 7：黄底黑字
8. 红底白字.png - 模板 8：红底白字

## 图片要求

- 格式：PNG
- 推荐宽度：800-1200px
- 每张图片建议小于 500KB
- 总大小控制在 5MB 以内

## 文档中的引用

subtitle-templates.md 中已添加图片引用，格式如下：

```markdown
## 模板1：白字黑边

### 样式预览
![模板1：白字黑边](assets/白字黑边.png)

### 描述
...
```

## 注意事项

- 确保文件名与上述列表完全一致（区分大小写）
- 图片必须是 PNG 格式
- 文件名使用中文，与文档中的引用一致
