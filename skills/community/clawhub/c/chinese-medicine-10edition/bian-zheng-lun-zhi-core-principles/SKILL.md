# Bian Zheng Lun Zhi Core Principles

## When to Use
- 患者已提供完整的望、闻、问、切四诊信息
- 需要区分“症”“病”“证”三者关系
- 目标是制定反映当前病理本质的个体化治疗方案
- 不适用于仅依据单一症状或西医诊断直接套用固定方药的情形

## Execution Steps

1. **收集患者整体信息**：通过四诊合参，系统获取症状（如头痛、发热）、舌象、脉象、体质、情志、环境等数据。

2. **明确“症”“病”“证”的区别**：
   - “症”：异常表现（如咳嗽、乏力），可由多种病因导致；
   - “病”：具有特定病因与演变规律的完整疾病过程（如消渴、黄疸）；
   - “证”：对疾病当前阶段的病理概括，包含病因、病位、病性、邪正关系及发展趋势。

3. **构建证候判断**：综合四诊信息，提炼出能反映疾病现阶段本质的证型（如肝郁化火、肺脾气虚、中气下陷等）。

4. **确定治疗法则**：遵循“证同治亦同，证异治亦异”原则，选择相应治法（如疏肝解郁、益气健脾、升提中气等）。

5. **动态调整治疗**：根据病情变化、季节、地域、个体差异，重新辨证并优化理、法、方、药。

6. **坚持核心理念**：中医治疗的核心在于把握“病机”而非仅关注“病名”，“证”已内含病机，是立法处方的根本依据。

> **输出结果**：针对当前“证”的个体化治疗方案，包括治则、治法、方剂建议及药物组成。
