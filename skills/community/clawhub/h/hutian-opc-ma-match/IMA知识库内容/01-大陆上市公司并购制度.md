# 大陆上市公司并购制度

## 一、《上市公司重大资产重组管理办法》核心框架

### 1.1 重大资产重组标准
根据《上市公司重大资产重组管理办法》（2025年修订版，证监会令第230号），满足以下任一标准即构成重大资产重组：

| 指标 | 比例标准 | 金额门槛 |
|------|----------|----------|
| 资产总额 | ≥50% | - |
| 营业收入 | ≥50% | - |
| 资产净额 | ≥50% | ≥5000万元 |

**计算规则**：
- 股权类资产：成交价与账面值较高者
- 非股权类资产：成交价与买入价较高者
- 12个月内连续交易的同类资产需累计计算

### 1.2 审核流程（注册制下）

```
受理 → 审核问询 → 重组委审议 → 证监会注册
  ↓        ↓           ↓           ↓
  2工作日  交易所5工作日  15工作日   5-15工作日
```

**2025年新设简易审核程序**：
- 适用条件：上市公司之间吸收合并；或市值≥100亿元、信息披露质量评价连续两年为A的优质公司发行股份购买资产（不构成重大资产重组）
- 审核时效："2+5+5"机制（2工作日受理、5工作日审核、5工作日注册）

### 1.3 发行股份购买资产核心要求
（依据第43-46条）

1. **财务报告要求**：最近一年及一期被出具**无保留意见**审计报告
2. **合规要求**：上市公司及现任董事、高级管理人员不存在正被立案调查情形
3. **定价规则**：发行价格不低于市场参考价的**90%**
4. **锁定期安排**：
   - 一般情形：12个月
   - 特定对象（控股股东、实控人控制的关联人、取得实际控制权、持续拥有不足12个月）：36个月

### 1.4 2025年修订要点（证监会令第230号）

| 修订内容 | 要点说明 |
|----------|----------|
| 分期支付机制 | 分期发行股份购买资产的注册决定有效期延长至48个月 |
| 简易审核程序 | 首设"2+5+5"快速通道 |
| 私募基金反向挂钩 | 投资期限满48个月，第三方交易锁定期由12个月缩短为6个月 |
| 上市公司间吸收合并 | 被吸并方控股股东设置6个月锁定期（构成收购则18个月） |
| 独立董事职责强化 | 重大资产重组构成关联交易的，需经全体独立董事过半数同意 |

**来源**：中国证监会《关于修改<上市公司重大资产重组管理办法>的决定》（2025年5月16日发布，自公布之日起施行）

## 二、《上市公司收购管理办法》

### 2.1 收购方式分类

| 收购方式 | 定义 | 特点 |
|----------|------|------|
| 协议收购 | 通过协议方式受让股份 | 灵活性高，需遵守权益变动披露规则 |
| 要约收购 | 向全体股东发出收购要约 | 全面要约/部分要约，定价需遵循规定 |
| 间接收购 | 通过控制关系变更实现 | 穿透认定实控人 |

### 2.2 权益变动披露规则

| 触发条件 | 披露要求 | 暂停交易 |
|----------|----------|----------|
| 5% | 3日内编制权益变动报告书 | 可申请豁免 |
| 5%-20% | 简式权益变动报告书 | 可申请豁免 |
| 20%-30% | 详式权益变动报告书 | 需暂停 |
| 30% | 触发要约收购义务 | 需申请豁免或进行全面要约 |

### 2.3 要约收购核心规则

1. **要约价格**：
   - 相同条件：不得低于提示性公告日前6个月内收购人取得该种股票所支付的最高价格
   - 流通股东：不低于提示性公告日前30个交易日该种股票的每日加权平均价格的算术平均值

2. **支付方式**：可以采用现金、证券、现金与证券相结合等合法方式支付

3. **豁免情形**：
   - 国有资产行政划转
   - 股份在同一实际控制人控制的不同主体之间转让
   - 继承、赠与

## 三、再融资与并购配套融资

### 3.1 2025年再融资市场概况

| 工具 | 数量 | 规模 | 特点 |
|------|------|------|------|
| 定向增发 | 123家 | 7141亿元（含银行5200亿） | 占绝对主导地位 |
| 可转债 | 44家 | 631亿元 | 含4家定向可转债 |
| 配股 | 0家 | 0 | - |

### 3.2 并购配套融资规则

1. **发行股份购买资产可同时募集配套资金**，定价按现行相关规定
2. **2025年并购六条**：允许配套融资用于补充流动资金、偿还债务（比例放宽）
3. **定向可转债**：可作为并购支付工具（如华海诚科收购衡所华威案例）

### 3.3 定价机制改革

- **底价规则**：定增发行价格不低于定价基准日前20个交易日均价的80%
- **2025年趋势**：发行折扣收窄，平均约86%，反映投资者对优质标的信心

## 四、科创板/创业板特殊规则

### 4.1 科创板并购特殊规定

**定位要求**（依据《上海证券交易所科创板上市公司重大资产重组审核规则》）：

- 标的资产应符合科创板定位
- 所属行业应与上市公司处于同行业或上下游
- 与上市公司主营业务具有**协同效应**

### 4.2 重组上市标准差异

| 板块 | 净利润要求 | 营业收入要求 |
|------|------------|--------------|
| 主板 | 最近3年净利润累计≥2亿元，且最近1年≥1亿元 | 或营业收入累计≥15亿元 |
| 科创板 | 最近2年净利润均为正且累计≥5000万元 | 或最近1年≥3亿元且现金流≥1亿元 |

### 4.3 未盈利资产并购（科创板八条）

- **允许收购未盈利资产**：符合硬科技定位的标的可以豁免盈利要求
- **审核案例**：芯联集成收购芯联越州（首单"亏收亏"交易）
- **估值包容**：可采用市场法评估，设置差异化定价安排

### 4.4 创业板"三创四新"定位

- 支持符合"创新、创造、创意"及"新技术、新产业、新业态、新模式"的跨界并购
- 监管审查趋严，需充分论证协同效应

## 五、证监会并购重组审核流程

### 5.1 标准审核流程

```
┌─────────────┐
│   停牌/公告  │  董事会决议→公告预案
└──────┬──────┘
       ↓
┌─────────────┐
│   交易所受理  │  提交申请文件（5工作日内受理）
└──────┬──────┘
       ↓
┌─────────────┐
│   审核问询   │  首轮问询通常20个工作日内
└──────┬──────┘
       ↓ (需回复)
┌─────────────┐
│   重组委审议  │  重组委会议，形成审议意见
└──────┬──────┘
       ↓
┌─────────────┐
│   证监会注册  │  15个工作日内决定
└──────┬──────┘
       ↓
┌─────────────┐
│   实施完成   │  资产交付/过户后持续督导
└─────────────┘
```

### 5.2 审核时限

| 阶段 | 时限 |
|------|------|
| 交易所受理到审核 | 无明确固定期限（通常1-3个月） |
| 证监会注册 | 15个工作日（简易程序5个工作日） |
| 注册决定有效期 | 12个月（分期发行48个月） |

### 5.3 2025年"并购六条"审核优化

- 建立重组"2+5+5"快速审核机制
- 加大申报前咨询沟通服务力度
- 提高对优质未盈利资产的包容度
- 新增收购未盈利资产、增加多种评估方法等披露和核查要求

## 六、信息披露与内幕信息管理

### 6.1 停牌规则

| 情形 | 停牌时间 | 可豁免情形 |
|------|----------|------------|
| 重大资产重组 | 原则上不超过10个交易日 | 延期需说明理由 |
| 要约收购 | 公告日至要约期结束 | - |

### 6.2 分阶段信息披露

1. **提示性公告**：筹划阶段及时披露进展
2. **重组预案**：董事会决议后次一工作日披露
3. **重组报告书**：股东大会通知前公告
4. **持续披露**：实施进展每30日公告一次

### 6.3 内幕信息管理

- **知情人登记**：内幕信息知情人的范围、交易情况需登记备查
- **窗口期限制**：董监高在定期报告、业绩预告等敏感期不得买卖
- **违规处罚**：内幕交易可处以1-10倍罚款

## 七、反垄断审查

### 7.1 经营者集中申报标准

| 指标 | 申报标准 |
|------|----------|
| 营业者合计年营业额 | 全球≥100亿元且中国大陆≥20亿元 |
| 单独经营者 | 中国大陆≥20亿元 |
| 且至少两个经营者 | 中国大陆≥4亿元 |

### 7.2 审查流程

```
申报 → 立案审查（30日） → 进一步审查（90日） → 决定
         ↓                    ↓
      简易程序              延长期（60日）
```

### 7.3 免于申报情形

- 参与集中的经营者为金融机构等特定行业
- 同一控制权下的企业合并
- 股权/资产购买但未导致控制权变更

## 八、外资并购特殊规定

### 8.1 外国投资者并购境内企业

**安全审查**：关系国家安全的重要行业（国防、关键技术等）需进行安全审查

**行业准入**：
- 鼓励类：享受优惠政策
- 限制类：需中方控股或合资
- 禁止类：不得并购

### 8.2 QFII/RQFII相关规定

- 合格境外机构投资者可投资境内上市公司
- 持股比例限制：单个境外投资者持有上市公司股份不超过10%
- 战略投资者：可参与定向增发，锁定期36个月
