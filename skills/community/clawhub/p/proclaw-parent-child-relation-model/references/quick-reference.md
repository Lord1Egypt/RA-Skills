## 核心公式

### 主体性公式

```
主体性 = 决策自主 × 解释内化 × 责任承担

高主体性：三者都高
低主体性：任一者缺失
```

### 能量公式

```
能量 = 注意力 + 兴趣 + 探索欲 + 行动冲动

能量枯竭：任一要素被严重剥夺
能量充沛：四要素完整且循环
```

### 内核公式

```
内核 = 心力（底座）× 能量（燃料）× 关系（放大器）× 心智（引擎）

任一层受损：整体效能降低
四层协同：超级个体
```

### 自驱公式

```
自驱 = 自主感 × 胜任感 × 联结感 × 意义感

四大支柱：全部满足时自驱最强
任一缺失：自驱打折
```

### 熵边界公式

```
健康熵边界 = 足够的探索空间 + 清晰的安全底线

熵过低：控制过度，主体死亡
熵过高：无序混乱，安全感缺失
```

### 亲子关系公式

```
关系质量 = 安全感 × 沟通质量 × 边界清晰度

三者都好：优质关系
任一缺失：关系紧张
```

## 关键判断标准

### 四轴快速判断

| 轴线 | 问题信号 | 正常信号 |
|------|----------|----------|
| **控制权** | 孩子等指令做事 | 孩子主动做选择 |
| **解释权** | 孩子问"为什么" | 孩子能说"因为..." |
| **能量流** | 孩子被动、没兴趣 | 孩子主动、好奇 |
| **责任归属** | 孩子推卸责任 | 孩子接受后果 |

### 主体性快速判断

**高主体性孩子**：
- 能自己做决定（小的开始）
- 能说明为什么
- 接受结果
- 从结果学习

**低主体性孩子**：
- 等父母决定
- 不说原因
- 推卸责任
- 不反思

### 能量快速判断

**能量充沛**：
- 有活力
- 愿意尝试
- 持续行动
- 对未来有期待

**能量枯竭**：
- 没精神
- 不愿尝试
- 需要催促
- 什么都"没意思"

### 三重剥夺快速判断

| 剥夺类型 | 核心问题 | 表现 |
|----------|----------|------|
| **熵剥夺** | 不允许探索 | 不敢尝试、害怕未知 |
| **反馈剥夺** | 不形成结果 | 行为无变化、无法学习 |
| **责任剥夺** | 不归因自己 | 总是怪别人、不反思 |

### 熵结构快速判断

| 类型 | 特征 | 主体性 |
|------|------|--------|
| **高压（低熵）** | 规则多、控制强 | 低 |
| **混乱（无边界）** | 无规则、无反馈 | 低 |
| **健康（可调）** | 有边界+有空间 | 高 |

## 三线快速诊断

### 家庭赋能线问题

**检查点**：
- [ ] 父母是否过度控制？
- [ ] 是否替代孩子做选择？
- [ ] 是否评判多于理解？
- [ ] 是否提供选择机会？

**诊断**：
- 过度控制 → 家庭赋能线问题
- 选择替代 → 家庭赋能线问题
- 评判为主 → 家庭赋能线问题

### 亲子关系线问题

**检查点**：
- [ ] 孩子愿意分享吗？
- [ ] 沟通是否双向？
- [ ] 冲突后能修复吗？
- [ ] 有情感联结吗？

**诊断**：
- 不愿分享 → 关系线问题
- 单向沟通 → 关系线问题
- 无法修复 → 关系线问题

### 内核能量线问题

**检查点**：
- [ ] 孩子有活力吗？
- [ ] 愿意尝试吗？
- [ ] 有自驱力吗？
- [ ] 能承受挫折吗？

**诊断**：
- 没活力 → 能量线问题
- 不愿尝试 → 能量线问题
- 无自驱 → 能量线问题

## 干预优先级

### 紧急干预（立即处理）

```
1. 孩子有自伤/自杀风险 → 立即转介
2. 家庭暴力 → 立即报警
3. 严重心理问题 → 立即转介
```

### 高优先级（1-2周）

```
1. 联结严重断裂 → 先修复关系
2. 能量完全枯竭 → 先恢复能量
3. 依恋严重不安全 → 专业支持
```

### 中优先级（1个月）

```
1. 控制权严重失衡 → 重构控制层级
2. 解释权完全外置 → 帮助内化
3. 自驱力严重缺失 → 重建自驱
```

### 常规优先级（持续进行）

```
1. 边界优化 → 持续调整
2. 沟通提升 → 持续练习
3. 心力培养 → 长期建设
4. 探索力激发 → 持续支持
```

### 优先级原则

```
安全 > 关系 > 能量 > 能力 > 发展

先确保安全
再修复关系
然后恢复能量
接着培养能力
最后追求发展
```

## 话术急救包

### 当孩子不听话时

**错误**："你怎么这么不听话！"

**正确**：
- "我看到你不想做...能告诉我为什么吗？"
- "我们一起来想想怎么做更好？"
- "你的感受是什么？"

### 当孩子失败时

**错误**："你怎么这么笨！"

**正确**：
- "这次没做好，你感觉怎么样？"
- "我们来想想哪里可以改进？"
- "我看到你努力了..."

### 当孩子没动力时

**错误**："你怎么这么懒！"

**正确**：
- "你现在感觉怎么样？"
- "有什么我能帮忙的吗？"
- "想不想休息一下再开始？"

### 当亲子冲突时

**错误**："我是你妈/爸，你必须听我的！"

**正确**：
- "我理解你的感受..."
- "我也有些担心..."
- "我们一起想办法？"

### 当孩子求助时

**错误**："这都不会，自己想办法！"

**正确**：
- "让我看看是什么困难..."
- "你试过什么方法？"
- "我可以这样支持你..."

### 当孩子表达情绪时

**错误**："有什么好哭的！"

**正确**：
- "你看起来很难过..."
- "愿意说说吗？"
- "我在这里陪着你..."

## 快速决策树

### 孩子不合作时

```
孩子不合作
    ↓
判断类型
    ↓
┌─────────┬─────────┬─────────┐
↓         ↓         ↓
能力问题  意愿问题  边界问题
    ↓         ↓         ↓
支持+鼓励  探询+连接  明确+坚定
```

### 亲子冲突时

```
亲子冲突
    ↓
暂停（双方冷静）
    ↓
父母先道歉（如果父母有责任）
    ↓
倾听孩子感受
    ↓
表达父母感受
    ↓
共同寻找解决方案
```

### 决定介入程度时

```
问题严重程度
    ↓
┌─────────┬─────────┬─────────┐
↓         ↓         ↓
轻微      中等      严重
    ↓         ↓         ↓
观察+支持  介入+引导  专业转介
```
