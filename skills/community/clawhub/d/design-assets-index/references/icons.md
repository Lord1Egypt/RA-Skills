# 图标合集

图标库合集：单体 SVG 图标库、品牌图标、3D 图标、图标搜索器与 React/SVG 组件库。更多顶选见 [SKILL.md](../SKILL.md)。

## 通用 UI 图标库

- [Phosphor Icons](https://phosphoricons.com) — Flexible icon family for interfaces
- [Heroicons](https://heroicons.dev/) — Free open-source icons by the Tailwind creators
- [Remix Icon](https://remixicon.com/) — Simply delightful icon system
- [Ionicons](https://ionicons.com) — Open-source icons for web/iOS/Android/desktop
- [Feather Icons](https://feathericons.com/) — Beautiful, customizable open-source icons
- [Eva Icons](https://akveo.github.io/eva-icons/) — 480+ open-source icons for UIs
- [Radix Icons](https://www.radix-ui.com/icons) — Crisp 15×15 icons as React/SVG

## 品牌与多格式

- [Simple Icons](https://simpleicons.org/) — Free SVG icons for popular brands
- [Material Design Icons](https://materialdesignicons.com/) — Icons in any format, size or color
- [Bootstrap Icons](https://icons.getbootstrap.com/) — Free icons for Bootstrap and any project
- [The Noun Project](https://thenounproject.com/) — 2M+ curated icons by a global community
- [iconmonstr](https://iconmonstr.com/) — 4496+ free icons in 313 collections
- [Iconscout](https://iconscout.com/) — Icons, illustrations and stock photos in one place
- [3DICONS](https://3dicons.co) — Beautifully crafted open-source 3D icons

## 图标搜索器

- [Icones](https://icones.js.org/) — Icon explorer with instant search (Iconify-powered)
