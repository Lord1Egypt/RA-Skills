# 风湿病病理三期演变

## 适用场景
- 心脏、关节、皮肤等结缔组织受累
- 病理切片见黏液样变性或纤维蛋白样坏死
- 需评估病变活动性与慢性化程度

## 执行步骤

### 1. 变质渗出期（约1个月）
- **特征**：基质黏液样变性 + 胶原纤维蛋白样坏死
- **伴随**：浆液/纤维蛋白渗出 + 少量淋巴细胞、浆细胞浸润

### 2. 增生期/肉芽肿期（2–3个月）
- **标志性结构**：风湿小体（Aschoff小体）
  - 位于心肌间质小血管旁
  - 含Aschoff细胞（巨噬细胞来源，核呈“枭眼状”或“毛虫状”）
  - 伴淋巴细胞、浆细胞

### 3. 纤维化期/硬化期（2–3个月）
- **演变**：坏死吸收 → Aschoff细胞转为成纤维细胞 → 小体纤维化 → 梭形瘢痕

> **关键提示**：整个病程4–6个月；因风湿病易复发，新旧病变常并存，需全面评估。
