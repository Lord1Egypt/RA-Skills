# Plum Blossom Divination Methods (梅花易数起卦法)

Based on Shao Yong's (邵雍) Plum Blossom Yishu (梅花易数) system.

---

## 1. Divination Methods Overview

| Method | Input Data | Best For |
|:------:|:----------:|:---------|
| **数字起卦** | Two or three numbers | Any question — most common method |
| **时间起卦** | Current date/time | Spontaneous inquiries |
| **方位起卦** | Direction or location | Travel, relocation, directional questions |
| **文字起卦** | Chinese character stroke count | Name, message, written question |

## 2. Core Methodology — Numerical Casting (数字起卦法)

The standard three-number method:

### Step 1: Get Numbers
Take three numbers (any source — from user, random thoughts, or time):
- **First number (上卦/Upper):** → determines upper trigram
- **Second number (下卦/Lower):** → determines lower trigram  
- **Third number (动爻/Changing line):** → determines changing line

### Step 2: Convert to Trigrams
Divide each number by 8 and use the remainder:
- Remainder 1 = ☰ 乾 (Qián/Heaven)
- Remainder 2 = ☱ 兑 (Duì/Lake)
- Remainder 3 = ☲ 离 (Lí/Fire)
- Remainder 4 = ☳ 震 (Zhèn/Thunder)
- Remainder 5 = ☴ 巽 (Xùn/Wind)
- Remainder 6 = ☵ 坎 (Kǎn/Water)
- Remainder 7 = ☶ 艮 (Gèn/Mountain)
- Remainder 0 (or 8) = ☷ 坤 (Kūn/Earth)

**Upper trigram:** First number ÷ 8 = remainder → trigram
**Lower trigram:** Second number ÷ 8 = remainder → trigram

### Step 3: Determine Changing Line
Third number ÷ 6 = remainder:
- Remainder 0 or 6 = 6th line (top)
- Remainder 1 = 1st line (bottom)
- Remainder 2-5 = 2nd-5th line respectively

### Step 4: Derive Hexagrams

| Hexagram | How to Get |
|----------|------------|
| **本卦 (Original)** | Upper trigram (from #1) + Lower trigram (from #2) |
| **变卦 (Changing)** | Apply change to the changing line — yang→yin or yin→yang; then form new hexagram |
| **互卦 (Interplay)** | Use lines 2,3,4 of original as lower and 3,4,5 as upper |

### Step 5: Determine Body-Use (体用生克)

| Role | Which Part | Represents |
|:----:|:----------:|:-----------|
| **体 (Body)** | Lower trigram (下卦) of 本卦 | The person asking |
| **用 (Function)** | Upper trigram (上卦) of 本卦 | The matter, the other party, the question |

**Analysis:**
- 体用比和 (same element) → very favorable
- 用生体 (function generates body) → favorable
- 体生用 (body generates function) → effort, sacrifice, draining
- 体克用 (body controls function) → achievable with effort
- 用克体 (function controls body) → unfavorable

## 3. Time-Based Casting (时间起卦法)

### Based on the lunar calendar (农历):

**Upper trigram:** (Year + Month + Day) ÷ 8 = remainder

**Lower trigram:** (Year + Month + Day + Hour) ÷ 8 = remainder

**Moving line:** (Year + Month + Day + Hour) ÷ 6 = remainder

**Lunar year numbers:** Use the 地支 (Earthly Branch) number:
- 子=1, 丑=2, 寅=3, 卯=4, 辰=5, 巳=6, 午=7, 未=8, 申=9, 酉=10, 戌=11, 亥=12

**Lunar month:** Plain month number (正月=1, 二月=2, etc.)

**Lunar day:** Day number of the lunar month (初一到三十)

**Hour (时辰):** Use the 地支 number of the current two-hour period:
- 子时 23-1=1, 丑时 1-3=2, 寅时 3-5=3, 卯时 5-7=4, 辰时 7-9=5, 巳时 9-11=6
- 午时 11-13=7, 未时 13-15=8, 申时 15-17=9, 酉时 17-19=10, 戌时 19-21=11, 亥时 21-23=12

### Quick time method (when lunar calendar is unavailable):
Use current Gregorian time (not ideal but common in modern practice):

**Upper:** (Y+M+D)÷8; **Lower:** (Y+M+D+H)÷8; **Line:** (Y+M+D+H)÷6

## 4. Object-Based Methods

### 方位起卦
Use the observed direction + object property:
- Upper trigram = direction trigram
- Lower trigram = object trigram
- Moving line = sum ÷ 6

### 声音起卦
Count the number of sounds heard:
- First group = upper trigram
- Second group = lower trigram  
- Total ÷ 6 = moving line

### 文字起卦 (Character Stroke Method)
For a single character:
- Upper: 左边/上半 stroke count ÷ 8
- Lower: 右边/下半 stroke count ÷ 8
- Line: total strokes ÷ 6

For two characters:
- Upper: first char strokes ÷ 8
- Lower: second char strokes ÷ 8
- Line: both ÷ 6

For three or more characters:
- Upper: first word strokes ÷ 8
- Lower: last word strokes ÷ 8 (or sum of remaining ÷ 8)
- Line: total strokes ÷ 6

## 5. Interpretation Framework (断卦)

### Step-by-step:
1. **Identify the 本卦 (original hexagram)** — the current situation
2. **Identify the 变卦 (changing hexagram)** — the outcome/trend
3. **Identify the 互卦 (interplay hexagram)** — the process/hidden factors
4. **Analyze 体用生克** — relationship between subject and object
5. **Read the 卦辞 (judgment)** — general meaning of the hexagram → See references/iching-core.md
6. **Read the 爻辞 (line statement)** — specific guidance from the changing line
7. **Consider the five elements** — what element generates, what element controls
8. **Synthesize** — integrate all factors into a coherent answer

### When reading hexagrams:

| Factor | What It Indicates |
|--------|-------------------|
| **本卦 (Original)** | Current state, root of the matter |
| **变卦 (Changing)** | Future outcome, where it's headed |
| **互卦 (Interplay)** | Process, hidden influences, intermediate stage |
| **体用生克** | Subject-object relationship, odds of success |
| **动爻 (Moving line)** | The focus of change — read the line text |

### Time frame estimation:
- Use the moving line number: line 1-2 = near future, 3-4 = medium, 5-6 = distant
- Consider the hexagram sequence: first part of upper canon = faster, later = slower
- 互卦 can indicate timing within the process
