# DMD/BMD糖皮质激素治疗规范

## 何时使用
- 患儿已明确诊断为DMD或BMD
- 需制定初始泼尼松治疗方案
- 需调整维持剂量或处理长期治疗相关并发症

## 执行步骤

### 1. 药物与起始剂量
- **首选药物**：泼尼松（prednisone）
- **起始剂量**：0.75 mg/(kg·d)，口服，每日一次

### 2. 疗效评估
- **早期反应**：用药10天后观察肌力是否改善
- **峰值疗效**：持续用药约3个月达最大效果

### 3. 维持剂量调整
- **常规维持范围**：0.5～0.6 mg/(kg·d)
- **最低有效剂量**：不低于0.3 mg/(kg·d)
- **调整频率**：最快每3～6个月根据临床反应和副作用评估调整一次

### 4. 多系统监测与管理
- **定期评估项目**：
  - 生长发育指标（身高、体重、BMI）
  - 骨密度（预防骨质疏松）
  - 血糖与血压（筛查类固醇性糖尿病和高血压）
- **内分泌管理**：警惕肾上腺皮质功能抑制，尤其在应激或停药时
- **药物禁忌**：避免联用抗胆碱能药和神经节阻断药

### 5. 辅助支持措施
- **康复干预**：鼓励主动与被动运动，防止废用性萎缩
- **矫形支持**：使用支具维持关节功能位，延缓挛缩
- **呼吸管理**：定期进行肺功能检查
- **心脏监测**：
  - 10岁前：每2年1次心电图 + 超声心动图
  - 10岁后：每年1次心电图 + 超声心动图

> **注意**：本方案需在神经肌肉疾病专科团队指导下实施，结合个体化评估动态调整。
