## 第一档：核心同构（熵 / 交叉熵直接对应）

### 第 5 章 沟通（最直接的同构）

| 概念 | 熵语言对应 |
|---|---|
| **人际隔阂 (interpersonal gap)** | $H(p_{\text{意图}}, q_{\text{理解}})$ —— Shannon 通信模型在亲密关系的直接对应 |
| **自我表露 (self-disclosure)** | 主动降低对方对自己的认知熵 |
| **社会渗透理论** | 自我表露的广度 + 深度 = 多维度降熵 |
| **应答性 (responsiveness)** | 高效降低对方"我重要吗"的熵 |
| **积极倾听 + 复述确认** | 让 q 显式化、$p$ 拥有者校对 → 降 KL |
| **共享密码本（衍生）** | 长期伴侣的共同预测先验 |
| **沉默 / 抑制表达** | $p$ 不传输 → 接收者只能用旧 $q$ 推测 → KL 散度爆炸 |
| **非言语沟通** | 多信道并行传输 → 信号冗余降噪 |
| **微表情** | 主信道压抑时的次信道泄露 |

### 第 6 章 相互依赖（满意度即交叉熵）

| 概念 | 熵语言对应 |
|---|---|
| **CL (Comparison Level)** | $q_{CL}$：你对关系的期望分布 |
| **关系满意度** | $\approx -H(p_{\text{实际}}, q_{CL})$ —— 现实与期望的交叉熵 |
| **CLalt** | 备选关系的期望分布；未真采样时为幻想 q |
| **投入 (investment)** | 锁住低熵态的势能屏障 |
| **承诺 (commitment)** | 自愿降低未来选择空间的熵 |
| **公平 (equity)** | 双方"投入/收益"分布的对称性 |
| **共有关系 vs 交换关系** | 放弃精确熵记账 vs 维持精确记账 |
| **Rusbult 投入模型** | 关系熵态稳定性的三因素：满意度 + CLalt + 投入 |

### 第 1 章 依恋（IWM 即 q）

| 概念 | 熵语言对应 |
|---|---|
| **内部工作模型 (IWM)** | $q_{IWM}$：童年形成的"关系应当如何"的先验分布 |
| **安全依恋** | $q$ 与多数现实 $p$ 匹配良好 → 低交叉熵 |
| **痴迷依恋** | $q$ 高估抛弃概率 → 多数 $p$ 触发误警报 → 持续主观高熵 |
| **回避依恋** | $q$ 把亲密信号编码为危险 → 主动维持高熵 |
| **恐惧依恋** | $q$ 内部矛盾 → 任何 $p$ 都不匹配 |
| **earned security 习得的安全** | 通过良好关系 $p$ 数据流，长期重训练 $q$ |
| **Sociometer 自尊** | 社会接纳概率分布的实时读数 |
| **归属需要** | 降低社会熵的内驱力 |

### 第 4 章 社会认知（认知偏差即降熵捷径）

| 概念 | 熵语言对应 |
|---|---|
| **首因效应** | 初始高熵时刻信息权重最大 |
| **验证性偏差** | 维持低熵 $q$ 比重训练省力 → 忽略反证 |
| **过度自信** | 主观熵 < 真实熵 |
| **积极错觉** | 主动塑造低熵正面 $q$（健康剂量是有益的） |
| **归因** | 给行为分配解释 → 降意义熵 |
| **行动者/观察者效应** | 对自己（低熵）vs 对他人（高熵）的不同归因路径 |
| **自我服务偏差** | 维持低熵自我模型 $q_{\text{自我}}$ |
| **重构性记忆** | 当下低熵叙事重写过去观测 |
| **自我实现的预言** | 预测改变现实 → 主动让 $p$ 匹配 $q$ |
| **印象管理** | 控制对方关于你的 $q$ |
| **自我证实** | 选择 $H(p_{\text{他人}}, q_{\text{自我}})$ 低的关系 |

### 第 10 章 压力与紧张（信任与背叛的 q 动力学）

| 概念 | 熵语言对应 |
|---|---|
| **信任** | 累积低交叉熵关系：$q$ 与 $p$ 长期高度匹配 |
| **背叛** | 单次极端 $p$ 偏离 $q$ → 强制 $q$ 重写 + 历史回溯重写 |
| **过度自信代价** | $q(\text{背叛}) \to 0$ → $-\log q \to \infty$ → 一次足以摧毁 |
| **怀疑性嫉妒** | 无 $p$ 数据校准的 $q$ 内部循环放大 |
| **反应性嫉妒** | 基于真实 $p$ 信号的高熵反应 |
| **真相偏差** | 默认低熵的信任先验 |
| **欺骗者猜疑** | 把自己 $q(\text{他人撒谎})$ 投射到他人 |
| **宽恕** | 接受更高熵的新 $q'$（含背叛可能性），而非回到原低熵 $q$ |
| **关系贬值** | "我重要吗"分布的负向更新 |
| **伴侣排斥** | 极端 $p$ 信号剥夺 → 主观熵爆炸 |

## 第二档：良好同构

### 第 3 章 吸引力

| 概念 | 熵语言对应 |
|---|---|
| **临近 (proximity)** | 接触机会 → 降熵机会 |
| **曝光效应 (mere exposure)** | 重复曝光 → $q$ 更接近 $p$ → 偏好上升 |
| **相似性 (similarity)** | 已知一半 → 降熵成本低 |
| **互补性** | 表面例外但其实在功能层面降熵 |
| **匹配 (matching)** | 选择被拒绝熵低的目标 |
| **礼尚往来** | 喜欢信号 → 接纳熵下降 |
| **心理抗拒 (reactance)** | 阻断 $p$ → 人为提高熵 → 反弹 |
| **Romeo-Juliet 效应** | 信息隔离 → 自我幻想的低交叉熵假象 |
| **适配价值 (mate value)** | $-D_{KL}(p_{\text{候选}} \| q_{\text{理想}})$ |

### 第 11 章 冲突

|---|---|

| **激发事件** | 熵的突发输入 |
| **归因式冲突** | 双方意义熵分布不一致 |
| **负面情感相互作用** | 熵的级联放大（正反馈） |
| **要求/退避** | 一方想降熵 vs 一方想维持高熵 |
| **voice/loyalty/exit/neglect** | 四种熵处理策略 |
| **顺应 (accommodation)** | 主动打断熵爆炸级联 |
| **Gottman 四类型** | 四种熵剖面 + 阻尼组合（见 §下方） |

### 第 14 章 维持与修复

| 维持机制 | 熵语言对应 |
|---|---|
| **认知相互依赖** | 自我模型部分重合 → 共享低熵 |
| **积极错觉** | 主动维持低熵正面 $q$ |

### 第 13 章 解体

| 概念 | 熵语言对应 |
|---|---|
| **VSA 模型 (Karney & Bradbury)** | 脆弱性（系统敏感性）× 应激（熵输入）× 适应（降熵能力） |
| **障碍模型 (Levinger)** | 离开成本的熵壁垒 |
| **持续动力** | 结构性高脆弱性 |
| **突现危难** | 后期熵冲击 |
| **幻灭 (disillusionment)** | 积极错觉的低熵 $q$ 接触真实 $p$ → KL 增 |
| **不懈的间接性** | 主动维持高意义熵以避免正面冲突 |
| **分手脚本** | 共同文化提供的降意义熵工具 |

| **浪漫之爱** | 高熵态：降熵速度快 → 兴奋 |
| **相伴之爱** | 低熵态：稳定预测 |
| **同情之爱** | 为对方降熵付出 |

| **柯立芝效应** | 熵适应导致激发衰退 |

## 第三档：部分对应

### 第 2 章 研究方法

- **相关研究** = 降低 "X 与 Y 是否同变" 的熵
- **实验** = 通过随机分配把混淆变量熵"平均掉"
- **自我报告** = 有噪声的信道
- **元分析** = 多源信息熵合并

### 第 7 章 友谊

- **尊重** = 对对方价值的低熵评估
- **信任** = 对未来行为的低熵预测
- **资本化** = 双重降熵（事件 + 关系）
- **无形支持** = 避免熵增副作用
- **关系规则** = 降低互动熵的契约
- **二元退缩** = 熵管理预算重新分配
- **社会情绪选择** = 老年人偏好低熵关系

### 第 9 章 性

- **性双重标准** = 性别熵规则不对称
- **社会性性取向** = 对短期高熵关系的开放度
- **多元无知** = 群体熵估计偏差
- **性满足** = 可预测性 vs 新奇的平衡

### 第 12 章 权力与暴力

- **权力** = 改变对方分布的能力
- **较小利益原则** = CLalt 更低者权力小
- **命运控制 / 行为控制** = 对未来熵 / 当下熵的控制
- **亲密恐吓** = 全维度熵剥夺
- **I³ 模型** = 抑制 = 维持低熵行为分布的能力

## Gottman 四类型 × 熵剖面 速查

| 类型 | 三熵剖面 | 阻尼/动力学 | 稳定性 |
|---|---|---|---|
| Volatile 多变型 | 行为熵高，认知/意义熵低 | 高 Q 振荡 | 稳定（双方都耐受） |
| Validator 确认型 | 三熵都低 | 适度阻尼 | 稳定 |
| Avoider 逃避型 | 认知熵中-高，行为/意义熵低 | 强阻尼 | 稳定（但脆弱） |
| Hostile 敌对型 | 三熵全高 | 发散振荡 | **不稳定** |

**核心发现（Gottman）**：前三种都能稳定——稳定关键不是冲突有无，而是**正负互动比例 ≥ 5:1**。从熵的视角，这相当于 **q 更新的方向偏正**——每次更新都让双方对彼此的 q 更接近正面 p。

## 依恋四类型 × q 特征 速查

| 类型 | $q$ 自我 | $q$ 他人 | 关键 $H(p, q)$ 模式 |
|---|---|---|---|
| 安全 | 正面 | 正面 | 与多数现实匹配 |
| 痴迷 | 负面 | 正面 | 高估抛弃，过度降熵尝试 |
| 回避 | 正面 | 负面 | 主动不降熵 |
| 恐惧 | 负面 | 负面 | $q$ 自相矛盾 |

## VSA 模型 (Karney & Bradbury) × 熵语言

VSA 模型本身就是"系统响应论"的口语版：

| 概念 | 熵语言 |
|---|---|
| 持久脆弱性 (V) | 系统对熵扰动的敏感性 |
| 应激事件 (S) | 外部熵输入 |
| 适应过程 (A) | $q$ 更新能力 / 降 KL 能力 |
| 关系满意度 = f(V, S, A) | 三因素决定 $H(p, q)$ 的稳定性 |

低 V + 低 S + 高 A → 稳定低熵
高 V + 高 S + 低 A → 必然崩溃

## I³ Model (Finkel) × 熵语言

暴力发生 = Instigation × Impellance > Inhibition

- Instigation = 熵扰动输入
- Impellance = 系统脆弱性
- Inhibition = $q$ 控制能力（自我控制 = 主动维持低熵行为分布）

抑制低于扰动 × 脆弱性 → 行为分布失控 → 极端结果。
