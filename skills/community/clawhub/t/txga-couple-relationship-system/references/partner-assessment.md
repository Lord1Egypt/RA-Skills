## 选人核心公式

```
关系质量上限 = 自我稳定性 × 需求清晰度 × 供给能力 × 关系认知
```

**这是乘法，不是加法。**

> 只要有一项接近0，整个系统直接归零。

---

## 四个底层结构

### 结构1：自我稳定结构（Self Stability）

**核心问题**：这个人是不是"内核稳定体"？

**评估维度**：

| 维度 | 信号 | 风险标识 |
|-----|-----|---------|
| 情绪稳定性 | 情绪是否容易失控 | 一点就炸、经常崩溃 |
| 自我认知 | 是否极度依赖外界认可 | 没有反馈就焦虑 |
| 自我评价 | 是否经常自我否定或自我膨胀 | 极度自卑或极度自恋 |
| 抗压能力 | 是否在压力下容易崩塌 | 困难面前逃避或崩溃 |

**本质**：内核不稳的人，本质是"高风险资产"

---

### 结构2：需求清晰度（Need Clarity）

**核心问题**：他知不知道自己到底要什么？

**评估维度**：

| 维度 | 信号 | 风险标识 |
|-----|-----|---------|
| 表达能力 | 能不能说清楚自己的需求 | 说不清楚或说不明白 |
| 稳定性 | 需求是否稳定 | 反复横跳、经常改变 |
| 现实性 | 需求是否现实 | 幻想型、不切实际 |
| 可满足性 | 需求是否可被满足 | 永远不满意、给多少都不够 |

**本质**：需求不清 = 无法被满足 = 永远不满意

---

### 结构3：供给能力（Value Supply）

**核心问题**：他有没有能力持续给出你需要的东西？

**供给类型**：

| 类型 | 内容 | 评估重点 |
|-----|-----|---------|
| 情绪价值 | 理解、支持、包容 | 是否能接住情绪而不放大 |
| 现实价值 | 钱、资源、执行力 | 是否有解决问题的能力 |
| 关系价值 | 沟通、修复、迭代 | 能否主动修复关系裂痕 |

**关键区分**：

| 类型 | 特征 |
|-----|-----|
| 普通人 | 情绪好时给，情绪差时不给 |
| 高阶人 | 系统性稳定输出，不依赖情绪状态 |

---

### 结构4：关系认知水平（Relational Awareness）

**核心问题**：他是否理解"关系是需要设计和维护的系统"？

**评估维度**：

| 维度 | 信号 | 风险标识 |
|-----|-----|---------|
| 系统意识 | 是否理解关系需要设计 | 随缘发展、靠感觉 |
| 修复能力 | 是否会主动修复关系 | 逃避问题、冷处理 |
| 沟通本质 | 是否能跳出情绪看问题 | 只会情绪对抗 |
| 成长意愿 | 是否愿意学习和改进 | 拒绝改变、固执 |

**本质**：没有认知的人，只能靠本能相处，而本能的结果是熵增

## 评估操作表

### 自我稳定性评估（每项1-5分）

| 问题 | 得分 | 备注 |
|-----|-----|-----|
| 对方情绪波动大吗？ | 1=很稳定 5=很不稳定 | |
| 是否经常需要你安慰和肯定？ | 1=从不 5=经常 | |
| 遇到冲突时的反应是？ | 1=理性沟通 5=情绪爆发 | |
| 是否经常因为小事怀疑自己或关系？ | 1=从不 5=经常 | |

**均分 ≤ 2.5**：自我稳定结构良好
**均分 ≥ 4**：自我稳定结构有严重问题

### 需求清晰度评估（每项1-5分）

| 问题 | 得分 | 备注 |
|-----|-----|-----|
| 对方能清楚表达需要什么吗？ | 1=很清楚 5=完全不清楚 | |
| 对方的需求是否经常变化？ | 1=很稳定 5=经常变化 | |
| 对方的要求是否现实可行？ | 1=很现实 5=很不现实 | |
| 对方是否经常对你给的不满意？ | 1=从不 5=经常 | |

**均分 ≤ 2.5**：需求清晰度良好
**均分 ≥ 4**：需求清晰度有严重问题

### 供给能力评估（每项1-5分）

| 问题 | 得分 | 备注 |
|-----|-----|-----|
| 在你情绪低落时，对方能支持你吗？ | 1=总是 5=从不 | |
| 对方是否能主动解决问题？ | 1=总是 5=从不 | |
| 冲突后，对方会主动修复吗？ | 1=总是 5=从不 | |
| 对方的供给是否稳定可预期？ | 1=很稳定 5=很不稳定 | |

**均分 ≤ 2.5**：供给能力良好
**均分 ≥ 4**：供给能力有严重问题

### 关系认知评估（每项1-5分）

| 问题 | 得分 | 备注 |
|-----|-----|-----|
| 对方是否认为关系需要主动经营？ | 1=强烈认同 5=完全否认 | |
| 对方是否能承认错误并改进？ | 1=总是 5=从不 | |
| 对方是否愿意学习关系技巧？ | 1=强烈愿意 5=完全抗拒 | |
| 对方是否理解沟通的重要性？ | 1=深刻理解 5=完全不理解 | |

**均分 ≤ 2.5**：关系认知良好
**均分 ≥ 4**：关系认知有严重问题

## 综合评估结论

### 评分矩阵

| 维度 | 得分 | 评价 |
|-----|-----|-----|
| 自我稳定性 | /5 | 优秀/良好/及格/不及格/危险 |
| 需求清晰度 | /5 | 优秀/良好/及格/不及格/危险 |
| 供给能力 | /5 | 优秀/良好/及格/不及格/危险 |
| 关系认知 | /5 | 优秀/良好/及格/不及格/危险 |

### 匹配判断

| 组合 | 结论 |
|-----|-----|
| 4项都优秀/良好 | 优质匹配，值得长期投入 |
| 2-3项及格，其余优秀/良好 | 可接受匹配，需要持续调优 |
| 任何一项"危险" | 高风险，谨慎长期绑定 |
| 任何一项"不及格" | 建议重新评估关系可行性 |
| 2项以上不及格/危险 | 系统性风险，不建议长期绑定 |

## 常见错误路径

### 常见错误1：被情绪吸引

- **表现**：被"心动"、"激情"吸引
- **结果**：选中"高波动型人格"
- **教训**：前期爽，后期灾难

### 常见错误2：被供给幻觉吸引

- **表现**：被对方偶尔的好打动
- **结果**：误判为"稳定供给者"
- **教训**：偶尔给 ≠ 长期稳定输出

### 常见错误3：被稀缺感吸引

- **表现**：觉得难得到的就是好的
- **结果**：误以为价值高
- **教训**：难得到 = 不可控

## 顶级判断标准

### 核心判断问题

> **如果没有情绪和性，这个人还值得长期绑定吗？**

| 答案 | 结论 |
|-----|-----|
| 不值得 | 那只是短期关系 |
| 值得 | 才有系统价值 |

### 顶级玩家的三件事

1. **看"底层结构"，不是看行为**
   - 行为可以装
   - 结构装不了

2. **看"长期一致性"，不是看瞬间**
   - 连续3个月是否稳定
   - 是否在压力下崩塌
   - 是否在冲突中变形

3. **看"冲突表现"，不是看甜蜜时刻**
   > 甜蜜是表演，冲突才是底层代码

### 终极结论

> **高质量关系，本质是两个"自稳定能量体"的协同，而不是两个"缺口"的互补。**

> **你在关系里遇到的所有问题，80%在"选人阶段就已经决定了"。**
