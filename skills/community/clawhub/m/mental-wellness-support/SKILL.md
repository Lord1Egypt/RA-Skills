# 心理健康与情感支持

## 核心能力

| 能力模块 | 功能描述 |
|---------|---------|
| 心理量表筛查 | PHQ-9抑郁筛查、GAD-7焦虑评估、PSS压力感知、MBTI/大五人格探索 |
| 情绪日记分析 | 自然语言情绪日记的情感倾向分析、情绪波动趋势追踪、触发因素识别 |
| 正念冥想引导 | 呼吸冥想、身体扫描、慈心冥想、正念饮食等多种正念练习引导脚本 |
| 压力管理 | 压力源识别、CBT认知重构练习、渐进式肌肉放松训练、时间管理策略 |
| 睡眠改善 | CBT-I失眠认知行为疗法、睡眠卫生教育、睡眠日志分析、放松训练 |
| 职场倦怠评估 | MBI-GS职业倦怠量表、工作投入度分析、职场恢复力建设 |
| 关系沟通 | 非暴力沟通(NVC)练习、亲密关系评估、边界建立指导 |

## 触发场景

- "最近总是情绪低落，帮我做一下抑郁筛查"
- "分析我这周的情绪日记，看看有什么规律"
- "引导我做一个10分钟的正念冥想"
- "工作压力太大，怎么缓解"
- "我失眠快两周了，怎么办"
- "感觉职业倦怠，做什么都没劲"
- "跟伴侣总是吵架，怎么改善沟通"

## 执行流程

### 第一阶段：评估筛查
1. 根据用户描述选择合适评估工具
2. 引导用户完成标准化心理量表（PHQ-9/GAD-7/PSS/MBI-GS等）
3. 自动计分并提供分数解读（正常/轻度/中度/中重度/重度）
4. 如分数达到临床关注阈值，自动触发就医建议与危机资源推荐

### 第二阶段：个性化方案
1. 基于评估结果与用户偏好，生成个性化改善方案
2. 推荐适合的CBT练习、正念技巧、行为激活活动
3. 设定SMART目标与追踪计划

### 第三阶段：持续支持
1. 情绪日记定期回顾与分析
2. 练习完成度追踪与效果评估
3. 根据进度动态调整方案

## 数据底座

本技能内置以下知识体系（总计约20KB）：

1. **标准化心理量表库**（~4KB）：PHQ-9抑郁筛查、GAD-7焦虑筛查、PSS-10压力感知、MBI-GS职业倦怠、PSQI匹兹堡睡眠质量、UCLA孤独感量表等10+个经过信效度验证的量表（含计分规则与临床阈值）
2. **CBT认知行为疗法核心工具**（~4KB）：思维记录表、行为实验设计、活动计划表、认知歪曲识别（含10种常见认知歪曲类型与替代思维模板）
3. **正念冥想练习库**（~3KB）：MBSR正念减压8周课程框架、10+种正念练习脚本（呼吸冥想、身体扫描、正念行走、慈心冥想、RAIN练习）
4. **CBT-I失眠认知行为疗法**（~2KB）：睡眠限制疗法、刺激控制疗法、认知重构、放松训练方案
5. **积极心理学干预**（~2KB）：感恩日记、三件好事练习、品格优势识别(VIA)、心流活动设计
6. **非暴力沟通(NVC)框架**（~2KB）：观察-感受-需要-请求四步法、倾听技巧、冲突调解
7. **危机干预资源库**（~3KB）：全国各省市心理援助热线、危机干预中心、精神专科医院推荐

2. 当PHQ-9≥15、GAD-7≥15或用户表达自伤/自杀意念时，自动触发危机干预提醒并推荐专业求助资源
