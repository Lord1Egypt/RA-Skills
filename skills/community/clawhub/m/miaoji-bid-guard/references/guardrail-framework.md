# 熔断护栏框架详细文档

> 本文档提供三级熔断机制的完整框架和决策逻辑

## 一、熔断机制概述

熔断机制是AI广告护栏的核心防线，其目的是在广告表现偏离预期时，及时生成调整建议，防止利润持续被侵蚀。

### 熔断三原则

1. **快速响应**：发现异常后秒级生成建议
2. **渐进调整**：避免大幅波动，每次调整有据可依
3. **数据驱动**：所有建议基于实际数据而非主观判断

---

## 二、词级熔断

### 生效条件

| 等级 | ACoS超标倍数 | 状态 | 建议操作 |
|------|--------------|------|----------|
| 正常 | < 100% | 绿色 | 维持现状 |
| 关注 | 100%-120% | 黄色 | 持续关注 |
| 警告 | 120%-150% | 橙色 | 建议降价 |
| 熔断 | > 150% | 红色 | 立即行动 |

### 熔断决策流程

```
开始词级评估
    ↓
评估该词ACoS指标
    ↓
ACoS > 上限×150%？
    ├─ Yes → 立即降价50%，进入观察期
    └─ No
        ↓
    ACoS > 上限×120%？
        ├─ Yes → 降价20%，3天后复盘
        └─ No
            ↓
        ACoS > 上限×100%？
            ├─ Yes → 标记为关注，保持关注
            └─ No → 标记为健康
                    ↓
                ACoS < 上限×80%？
                    ├─ Yes → 可小幅度加价测试
                    └─ No → 维持现状
```

### 观察期管理

**观察期时长**：3天
**观察指标**：ACoS变化趋势、点击量、转化率
**提前结束条件**：ACoS恢复到上限以内

### 降价阶梯

| 熔断等级 | 当前出价 | 建议降价幅度 | 降价后出价 |
|----------|----------|--------------|------------|
| 警告 | $1.00 | 20% | $0.80 |
| 熔断 | $1.00 | 50% | $0.50 |

## 三、时段熔断

### 生效条件

- 连续3个时段（每时段4小时）亏损率超过50%
- 或单个时段ACoS超过上限的200%

### 时段划分

| 时段 | 时间范围（示例） | 特点 |
|------|------------------|------|
| 凌晨 | 00:00-06:00 | 流量低，竞争小 |
| 上午 | 06:00-10:00 | 流量上升 |
| 上午高峰 | 10:00-14:00 | 流量高峰期 |
| 下午 | 14:00-18:00 | 流量平稳 |
| 晚间高峰 | 18:00-22:00 | 最高流量期 |
| 深夜 | 22:00-24:00 | 流量下降 |

### 时段熔断决策

```
开始时段评估
    ↓
评估各时段ACoS指标
    ↓
识别连续亏损时段
    ↓
亏损时段数 ≥ 3？
    ├─ Yes → 降低该时段预算50%
    └─ No
        ↓
    单时段ACoS > 上限×200%？
        ├─ Yes → 建议暂停该时段投放
        └─ No → 降低该时段预算20%
                    ↓
                重分配预算至高转化时段
```

## 四、活动级熔断

### 生效条件

- 整活动TACoS超过目标值的120%
- 或连续7天ACoS呈上升趋势

### TACoS计算

```
TACoS = （广告花费 / 总销售额）× 100%
总销售额 = 自然销售 + 广告销售
```

### 活动级熔断决策树

```
开始活动级评估
    ↓
计算活动TACoS
    ↓
TACoS > 目标×120%？
    ├─ Yes → 进入深度诊断
    └─ No → 活动健康
            ↓
深度诊断：
├─ 识别TACoS最高ASIN
├─ 识别亏损最严重的关键词
└─ 分析时段表现差异
        ↓
诊断结果应用：
├─ 暂停/降低亏损ASIN预算
├─ 重新分配预算至高效ASIN
├─ 调整关键词匹配策略
└─ 优化投放时段
```

## 六、熔断恢复机制

### 恢复条件

1. **词级恢复**：连续3天ACoS低于上限
2. **时段恢复**：连续3天该时段盈利
3. **活动恢复**：TACoS连续7天呈下降趋势

### 恢复操作

| 原状态 | 恢复操作 | 恢复后状态 |
|--------|----------|------------|
| 词级熔断 | 维持降价后出价 | 持续观察 |
| 时段熔断 | 逐步恢复预算 | 关注 |
| 活动熔断 | 小幅增加高效ASIN预算 | 持续观察 |

### 恢复时间表

```
第1天：检测到恢复条件
第2-3天：保持当前设置，持续观察
第4-7天：小幅度加价测试（每次10%）
第8天后：恢复正常出价（如数据支持）
```

## 七、熔断策略库

### 常见场景应对

| 场景 | 原因分析 | 建议策略 |
|------|----------|----------|
| 新品ACoS飙升 | 权重积累期正常现象 | 放宽上限，聚焦点击量 |
| 旺季ACoS上升 | 竞争加剧 | 动态调整上限 |
| 同类产品降价导致ACoS上升 | 外部冲击 | 评估是否跟价 |
| 转化率下降 | listing权重下降或选品问题 | 优先优化listing |

## 八、配置建议

### 阶段化阈值

| 广告阶段 | ACoS上限调整系数 | 熔断阈值调整系数 |
|----------|------------------|------------------|
| 新品冷启 | 1.3x | 1.3x |
| 爬坡期 | 1.1x | 1.2x |
| 稳定期 | 1.0x | 1.0x |
| 爆款期 | 0.9x | 1.0x |

### 自定义参数

用户可根据实际情况调整：
- ACoS上限
- 熔断生效阈值
- 观察期时长
- 降价阶梯幅度
