# 定价策略指南

## 一、定价策略类型

### 1. 成本加成定价
```
公式: 售价 = 成本 × (1 + 加成率)

适用场景:
- 产品标准化程度高
- 市场竞争激烈
- 成本可准确核算

建议加成率:
| 产品类型 | 加成率 |
|---------|-------|
| 大宗产品 | 10-15% |
| 标准产品 | 15-25% |
| 定制产品 | 25-40% |
| 高端产品 | 40%+ |
```

### 2. 市场导向定价
```
方法: 参考竞品价格，制定具有竞争力的价格

步骤:
1. 调研竞品价格区间
2. 确定我方定位（高/中/低）
3. 基于定位制定价格
4. 根据自身优势调整

定价区间参考:
- 高端定位: 竞品均价 × 1.1-1.3
- 中端定位: 竞品均价 × 0.9-1.1
- 低端定位: 竞品均价 × 0.7-0.9
```

### 3. 价值导向定价
```
核心: 基于客户感知价值定价

评估因素:
- 认证价值（认证越多，客户成本越低）
- 交期价值（快速交期溢价5-10%）
- 质保价值（长质保溢价5-10%）
- 服务价值（配套服务溢价10-20%）

公式: 价格 = 成本 + 认证溢价 + 服务溢价 + 品牌溢价
```

## 二、动态定价策略

### 淡旺季定价
| 时期 | 策略 | 调整幅度 |
|-----|------|---------|
| 旺季（Q4） | 价格上涨 | +5-10% |
| 淡季（Q1-Q2） | 促销定价 | -5-10% |
| 展会季 | 优惠吸引 | 保持稳定 |

### 批量折扣策略
```
阶梯定价:
| 数量 | 折扣 | 说明 |
|-----|------|------|
| MOQ - 100pcs | 0% | 标准价 |
| 101-500pcs | 3% | 小量优惠 |
| 501-1000pcs | 5% | 中量优惠 |
| 1001-5000pcs | 8% | 大量优惠 |
| 5000pcs+ | 10%+ | 特殊议价 |
```

## 三、竞品应对策略

### 竞品降价应对
| 情况 | 策略 | 说明 |
|-----|------|------|
| 竞品低于我方10%内 | 强化差异 | 强调质量/服务 |
| 竞品低于我方10-20% | 成本优化 | 评估降本空间 |
| 竞品低于我方20%+ | 价值重构 | 考虑产品升级 |

### 竞品涨价应对
| 情况 | 策略 | 说明 |
|-----|------|------|
| 短期优势窗口 | 加速开发 | 快速抢占市场 |
| 宣传强调 | 稳定价格 | 强化品牌信任 |
| 客户锁定 | 签订长单 | 锁定长期合作 |
