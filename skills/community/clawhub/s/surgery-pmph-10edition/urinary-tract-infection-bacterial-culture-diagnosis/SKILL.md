# 尿路感染的细菌培养诊断标准

## 何时使用
当患者出现尿频、尿急、尿痛等膀胱刺激症状，并已正确采集合格尿标本（清洁中段尿、导尿或耻骨上膀胱穿刺）时，使用本技能解读细菌培养结果。

## 执行步骤

1. **确认标本采集与处理合规**：
   - 首选清洁中段尿或导尿标本；新生儿或截瘫患者可采用耻骨上膀胱穿刺（最可靠）；
   - 标本必须在采集后2小时内处理，避免污染或杂菌繁殖。

2. **根据菌落计数判读结果**：
   - ≥10⁵ CFU/ml：确诊尿路感染；
   - ≤10⁴ CFU/ml：视为污染（尤其在无症状者中）；
   - 10⁴～10⁵ CFU/ml：可疑感染，需重复培养确认。

3. **结合临床综合判断**：
   - 仅适用于未使用抗菌药物的急性尿路感染患者；
   - 若患者已用药或为慢性感染，菌落数可能被抑制，应同时评估脓尿（>5 WBC/HPF）、尿沉渣涂片及临床症状；
   - 若存在无菌性脓尿，需考虑结核、间质性膀胱炎等非细菌性病因。

4. **输出决策**：
   - 明确是否启动抗菌治疗，或安排重复检查以确认诊断。
