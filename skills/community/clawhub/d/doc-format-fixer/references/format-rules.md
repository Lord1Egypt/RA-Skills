| 一级标题 | 黑体/小标宋 | Arial | 二号/三号 | 居中 | 无 |
| 二级标题 | 黑体/楷体 | Arial | 三号/四号 | 左对齐 | 无 |
| 三级标题 | 楷体/仿宋 | Arial | 四号/小四 | 左对齐 | 无 |

| 注释/脚注 | 仿宋/宋体 | Times New Roman | 小五/六号 | 左对齐 | 视情况 |

### 常见排版故障
1. **多余空格**：段落末尾、文字间不可见空格
2. **空行错乱**：连续多个空行、标题前后缺空行
3. **标题孤行**：标题在页末、正文在下一页
4. **字体不统一**：同类内容使用不同字体
5. **缩进不一致**：同类段落首行缩进值不同
6. **行距不统一**：同类段落行距设置不同
7. **页码异常**：页码缺失、编号不连续
8. **表格溢出**：表格宽度超出页面边界
