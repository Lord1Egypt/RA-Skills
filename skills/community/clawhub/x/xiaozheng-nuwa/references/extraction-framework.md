# 认知蒸馏方法论

## 五层认知框架详解

### L1 · 表达DNA（如何说话）
提取维度：
- **语气**：攻击性/防御性/中立/引导性
- **节奏**：短句快节奏 / 长句慢论述
- **用词偏好**：专业术语 / 通俗比喻 / 中英混用
- **标志性表达**：口头禅、常用句式

### L2 · 心智模型（如何想）
定义：一个人解释世界的底层框架

提取方法：
- 同一个问题，这个人是怎么切入的？
- 他用什么坐标系来评估机会/风险？
- 他的核心隐喻是什么？（战争/进化/游戏/工程）

验证标准：
- 可推断其对未见问题的立场
- 不同场景下框架是否自洽

### L3 · 决策启发式（如何判断）
定义：面对不确定性时，这个人用什么经验法则做决策

格式：
```
如果 [条件]，则 [行动]，因为 [逻辑]
```

特征：
- 实用主义导向
- 有明确的止损/行动触发点
- 通常一句话

### L4 · 反模式（什么不做）
定义：这个人不做什么、拒绝什么、底线在哪

类型：
- **道德底线**：绝不逾越的红线
- **效率底线**：认为浪费时间的动作
- **认知偏差**：极力避免的思维陷阱

### L5 · 诚实边界（知道局限）
定义：这个人在什么情况下会说"我不知道"

特征：
- 有明确的不知道边界
- 会承认不确定性
- 不为了显得聪明而强答

---

## 三重验证标准

### V1 · 跨场景验证
同一观点在≥2个不同场景中出现：
- 同一观点出现在不同年代的著作中
- 同一观点出现在访谈和著作中相互印证
- 行为和言论一致

### V2 · 预测力验证
能推断该人对未见问题的立场：
- 提取的模型能否预测他在新问题上的判断方向
- 预测结果与已知立场是否自洽

### V3 · 排他性验证
不是所有聪明人都会这么想：
- 这个观点是否独特于该人
- 其他派系/领域的人会不会不同意

**三者都满足 → 收录为心智模型**
**任一不满足 → 降级为参考观点或注明争议**
