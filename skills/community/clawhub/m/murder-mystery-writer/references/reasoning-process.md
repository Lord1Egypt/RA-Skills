# 推凶过程写法指南

## 为什么推凶过程这么重要

百变大侦探审核特别强调：**不能只有上帝视角的真相复盘，必须有玩家视角的推理流程。**

区别在于：
- **上帝视角**："凶手在18:52进入书房，用毒药杀死了死者，然后从后门离开。"
- **玩家视角**："玩家通过线索C003发现死者手表停在19:47，结合线索C007毒药需要30分钟生效，可推断投毒时间在19:17左右。此时角色A声称自己在花园（线索C012），但角色B在19:15看到走廊有人影（线索C015），两者矛盾。进一步调查线索C018发现花园后门的锁有被撬痕迹，说明有人可以从花园绕回书房……"

## 推凶过程的结构

### 第一层：确定死亡时间

**写法示例**：
```
根据线索L001（死者手表停在19:47）和线索L002（法医初步判断死亡时间在19:30-20:00），
可以确定死亡时间在19:30-19:47之间。
```

### 第二层：缩小嫌疑人范围

**写法示例**：
```
在19:30-19:47这个时间段内：
- 角色A：声称在花园，但无人证明（嫌疑）
- 角色B：在大厅与角色C聊天，C可证明（排除）
- 角色D：声称在洗手间，但E在19:35看到洗手间空着（嫌疑）
- 角色E：在大厅，多人可证明（排除）
- 角色F：声称在书房，但书房是案发现场（高度嫌疑）

因此嫌疑人范围缩小到A、D、F三人。
```

### 第三层：分析作案手法

**写法示例**：
```
线索L005显示死者死于氰化物中毒，线索L008显示茶杯中有氰化物残留。
线索L012显示厨房有氰化物的空瓶，线索L015显示只有角色F有厨房钥匙。

但角色A声称19:20去过厨房泡茶（线索L018），如果属实，A也有接触毒药的机会。
然而线索L020显示厨房门锁在19:00后被换过，A的钥匙已失效。

因此，只有F能在19:00后进入厨房。
```

### 第四层：破解不在场证明

**写法示例**：
```
角色F声称19:30-20:00一直在书房看书。
但线索L025显示书房的灯在19:45才打开（之前是灭的），
说明F在19:45之前并不在书房。

F的不在场证明不成立。
```

### 第五层：锁定凶手

**写法示例**：
```
综合以上推理：
1. 死亡时间在19:30-19:47
2. 只有F能在19:00后进入厨房获取毒药
3. F的不在场证明被推翻
4. 线索L030显示F与死者有财务纠纷（动机）

因此，凶手是角色F。
```

## 写作要点

### 必须做到

1. **每一步推理都要引用具体线索编号**
2. **说明线索之间的逻辑关系**
3. **解释为什么某些角色可以被排除**
4. **说明凶手可能的反驳以及如何应对**

### 不能做的

1. 不能直接说"凶手是X"然后解释为什么
2. 不能跳过中间推理步骤
3. 不能用"众所周知"、"显然"等词跳过论证
4. 不能只用一条线索就锁定凶手
