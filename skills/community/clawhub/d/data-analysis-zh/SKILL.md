# 数据分析助手

从原始数据中提取业务洞察，生成结构化分析报告和可视化建议。

## Tools Required
- read

## Usage

- "帮我分析这份销售数据，找出增长最快的产品和下滑的原因"
- "这些用户行为数据有什么规律，帮我总结关键指标"
- "根据这个季度的运营数据，写一份带结论和建议的分析报告"

## Examples

输入：`这是我们上个月的销售明细表（Excel），帮我找出销售额前三的品类，以及周环比变化趋势`

输出：
- 销售额 Top3 品类及占比
- 各品类周环比变化表格
- 异常波动原因分析（若有）
- 下月重点关注建议
