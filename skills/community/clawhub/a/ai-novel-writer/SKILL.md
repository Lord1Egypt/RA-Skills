# AI Novel Writer

Fiction writing companion covering plot development, character creation, world building, dialogue craft, and editing for novels across all genres.

## Core Capabilities

### 1. Plot Development
- Story structures: three-act, hero's journey, save the cat, seven-point
- Outline methods: snowflake, mind mapping, index card, pantsing
- Subplot weaving, parallel narratives, frame stories
- Pacing: scene-sequel pattern, tension curves, cliffhangers
- Plot hole detection, continuity checking

### 2. Character Creation
- Character profiles: backstory, motivation, flaw, want vs need
- Arc types: positive change, flat arc, tragic/disillusionment
- Character voice differentiation, dialogue fingerprints
- Relationship dynamics: allies, mentors, antagonists, love interests
- Ensemble cast balance, character function mapping

### 3. World Building
- Setting design: physical, cultural, historical, magical systems
- Rules and consistency: hard vs soft magic, technology limitations
- Sensory detail: sight, sound, smell, touch, taste in description
- Exposition handling: show don't tell, iceberging

### 4. Genre Craft
- Fantasy: magic systems, quest structure, chosen one tropes
- Mystery: clue planting, red herrings, fair play principle
- Romance: meet-cute, conflict, dark moment, HEA/HFN
- Thriller: stakes escalation, ticking clock, unreliable narrator
- Literary: theme development, symbolism, prose style

### 5. Editing & Revision
- Developmental editing: structural changes, character arc check
- Line editing: sentence flow, word choice, rhythm
- Beta reader management, critique incorporation
- Query letter, synopsis, agent submission package
