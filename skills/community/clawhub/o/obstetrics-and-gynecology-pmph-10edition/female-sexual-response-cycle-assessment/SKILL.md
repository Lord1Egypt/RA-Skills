# 女性性反应周期五阶段模型评估

## 适用对象
- 成年女性
- 临床性医学评估场景
- 性健康教育情境

## 使用前提
- 存在性刺激（感官或心理）
- 神经系统与内分泌系统功能基本正常

## 禁忌与限制
- 不适用于无意识状态或严重神经损伤患者
- 个体差异可能导致阶段顺序、持续时间或表现形式不同，需结合主观感受与客观体征综合判断

## 执行流程

### 1. 性欲期
- 观察是否存在对性的心理渴望
- 判断刺激类型：非条件性（如触觉、视觉）或条件性（如性幻想、过往经验）
- 注意：此期仅有心理变化，无明显生理反应；部分女性的性欲可能在性兴奋后出现（反应性性欲）

### 2. 性兴奋期
- 检查阴道润滑是否在性刺激后10～30秒内出现
- 观察阴蒂、大小阴唇是否肿胀，阴道长度是否增加
- 记录全身反应：乳房肿胀、乳头勃起、心率加快、血压轻度升高、呼吸略快、肌肉紧张
- 评估心理兴奋程度；注意女性唤起通常较男性缓慢，需更充分刺激

### 3. 性持续期（平台期）
- 确认性兴奋是否稳定维持在高水平
- 观察生殖器充血是否加剧，阴道外1/3是否环状缩窄、内2/3是否扩张
- 检查子宫是否提升，乳房是否进一步肿胀
- 记录性红晕范围（可扩展至下腹、肩、腿、臀部）及肌张力增强情况
- 监测心率、呼吸、血压是否继续上升

### 4. 性高潮期
- 识别是否出现身心极度快感（持续数秒至数十秒）
- 观察阴道与肛门括约肌是否发生不随意节律性收缩
- 检查子宫是否收缩与提升
- 记录面部表情变化、全身痉挛、呻吟、出汗或短暂神志迷惘
- 确认心率、呼吸、血压是否达峰值
- 注意：女性无不应期，可多次高潮；未达高潮仍可构成完整且愉悦的性反应周期

### 5. 性消退期
- 追踪身体恢复过程：乳房肿胀首先消退，随后生殖器充血与肿胀减退
- 监测肌张力、心率、血压、呼吸是否逐步恢复正常
- 评估心理满足感与舒畅程度

## 关键注意事项
- 主观性唤起与客观生理反应（如润滑、充血）可能不一致，需分别记录
- 性欲可分为“自发性”与“反应性”，后者在女性中更为常见
- 即使未经历性高潮，只要其他阶段正常完成，仍属有效且健康的性反应

## 输出目标
生成阶段性评估结论，明确是否存在性功能障碍及其所在阶段，为临床干预或健康教育提供依据。
