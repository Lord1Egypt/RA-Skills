# 掼蛋逐手评分标准

## 评分体系概述

### 评分维度
- **基础分**：出牌本身的合理性（1-5分）
- **时机分**：出牌时机是否恰当（1-3分）
- **配合分**：对家配合价值（1-2分）

### 总分计算
```
总分 = 基础分 + 时机分 + 配合分
最高10分，最低1分
```

### 评分等级
| 分数 | 等级 | 含义 |
|------|------|------|
| 9-10分 | 🌟最优解 | 最佳选择，几乎不可能更好 |
| 7-8分 | ✅合理 | 不错的选择，可接受 |
| 5-6分 | ⚠️可接受 | 有缺陷，但不是大错 |
| 3-4分 | ❌较差 | 明显失误，可能导致劣势 |
| 1-2分 | 💥严重失误 | 致命错误，局势逆转 |

## 不同牌型评分标准

### 1. 单张出牌

#### 评分要点
| 因素 | 加分 | 扣分 |
|------|------|------|
| 大牌(A以上)出牌时机 | 残局出牌+2 | 牌多时出-2 |
| 小牌试探 | 开局合理+1 | 残局出小牌-1 |
| 压制对手 | 有效压制+2 | 无效压制-1 |

#### 评分示例
```
场景：手牌27张，开局第一手
出牌：单张3 → 8分 ✅
原因：开局出小牌探路，正确策略

场景：手牌10张，中盘
出牌：单张A → 6分 ⚠️
原因：残局出大牌可能送牌给对手

场景：手牌5张，残局
出牌：单张K → 9分 🌟
原因：残局关键牌控制局面
```

---

### 2. 对子出牌

#### 评分要点
| 因素 | 加分 | 扣分 |
|------|------|------|
| 对A出牌 | 残局+2 | 牌多时出-2 |
| 小对子试探 | 合理+1 | - |
| 强对子压制 | 有效+2 | - |

#### 评分示例
```
场景：手牌15张
出牌：对3 → 8分 ✅
原因：小对子试探，合理开局

场景：手牌20张
出牌：对A → 4分 ❌
原因：强对子过早出，应保留控制权
口诀：大对留后手，小对先探路

场景：手牌8张，残局
出牌：对K → 9分 🌟
原因：关键时刻用强对子控制
```

---

### 3. 顺子出牌

#### 评分要点
| 因素 | 加分 | 扣分 |
|------|------|------|
| 短顺子(5-6张) | 合理+1 | - |
| 长顺子(8+张) | - | 暴露牌力-2 |
| 残局出顺子 | 有效+2 | - |

#### 评分示例
```
场景：手牌25张
出牌：顺子34567 → 8分 ✅
原因：短顺子试探，不暴露太多牌力

场景：手牌22张
出牌：顺子345678910 → 5分 ⚠️
原因：长顺子出完会暴露牌力

场景：手牌6张
出牌：顺子78910J → 9分 🌟
原因：残局出顺子收尾，时机恰当
口诀：长顺留一手，短顺先出手
```

---

### 4. 炸弹出牌

#### 评分要点
| 因素 | 加分 | 扣分 |
|------|------|------|
| 关键时刻使用 | +3 | - |
| 残局收尾 | +2 | - |
| 压制对手出牌 | +2 | - |
| 牌多时使用 | - | -3 |
| 天王炸过早 | - | -2 |

#### 评分示例
```
场景：手牌20张，中盘
出牌：炸弹5555 → 5分 ⚠️
原因：牌多时过早使用炸弹
口诀：前期忍一忍，后期炸弹震

场景：手牌8张，残局
出牌：炸弹7777 → 9分 ✅
原因：残局用炸弹收尾，时机恰当

场景：手牌25张，开局
出牌：天王炸弹 → 4分 ❌
原因：天王炸过早使用，浪费最大牌
口诀：天王不轻出，出就要收割

场景：对手出大牌抢头游
出牌：炸弹KKKK → 10分 🌟
原因：关键时刻压制，阻止对手
```

---

### 5. 过牌（不出）

#### 评分要点
| 因素 | 加分 | 扣分 |
|------|------|------|
| 对手牌力强 | +2 | - |
| 应该压制时 | - | -3 |
| 配合对家 | +1 | - |

#### 评分示例
```
场景：对手出炸弹
出牌：过 → 8分 ✅
原因：明智选择，不与炸弹对抗

场景：对手出单张5
出牌：过 → 3分 ❌
原因：小牌可以压制，选择过牌过于保守
口诀：小牌能压就压，别让对手轻松走

场景：对家领先，让对家继续
出牌：过 → 9分 ✅
原因：配合对家，策略正确
```

## 失误类型权重

| 失误类型 | 扣分权重 | 常见口诀 |
|----------|----------|----------|
| 炸弹时机 | 2.0倍 | 炸弹不拆牌，整手压到底 |
| 配合失误 | 2.5倍 | 对家在冲刺，我方必挡牌 |
| 大牌提前 | 1.5倍 | 大对留后手，小对先探路 |
| 顺子拆分 | 1.8倍 | 顺子不拆打，拆了要后悔 |
| 过于被动 | 1.2倍 | 该出手时就出手 |
| 过于激进 | 1.3倍 | 激进有度，冒进无路 |

## 综合评分计算

### 示例：评估一局中的某手出牌

```
用户手牌：还有12张
局面：3号位(对家)刚出了对K，2号位过牌
用户出牌：对A

基础分：3分（强对子过早出）
时机分：-1分（可以压制但牌还多）
配合分：+1分（帮对家压制）

总分：3分 → 评价：较差 ❌
建议：应该保留对A作为后手控制
口诀：大对留后手，小对先探路
```

---

## 学习要点

### 高分技巧
1. 残局出大牌是关键
2. 对家配合价值高
3. 关键时刻敢于用炸弹

### 低分避免
1. 强牌过早出
2. 不帮对家挡牌
3. 炸弹使用时机不对
