# Live Session Script Framework

## Pre-Live Checklist (30 min before)
- [ ] All products staged and within arm's reach in reveal order
- [ ] Stock counts confirmed for each product
- [ ] Cart links / pins tested on a second device
- [ ] Lighting checked (ring light on, no harsh shadows)
- [ ] Internet connection tested (speed test >10 Mbps upload)
- [ ] Script and lineup printed or on second screen
- [ ] Backup products staged in case a product runs out early
- [ ] Timer or segment clock visible to host

## Opening Script (0:00–3:00)
```
"Hey hey hey — we are LIVE! [wave to camera]

For anyone who's new here — I'm [Name], and [Store Name] is your go-to for [category].

Tonight I have [NUMBER] products for you — and one of them I've literally never shown on live before. You're going to want to stay for that.

To everyone who's joining — drop your location in the comments, let me see where you're watching from!

Okay — first product. I need to get the cart link pinned. [Pause to pin link.]

Are you ready? Let's GO."
```

## Segment Transition Script
```
"[Acknowledge comments for 15 seconds]

Okay so that [PREVIOUS PRODUCT] — [STOCK REMAINING] units left, link is still pinned if you want to grab it.

Now. [Build pause.] I told you I had something special tonight. This next one — I've been getting so many DMs asking when I was going to feature this. It's finally here."
```

## Close Mechanic Script (Countdown Version)
```
"[PRODUCT NAME] — [PRICE] — link is pinned right now.

We have [STOCK COUNT] left. That's it. When these are gone, they're gone — I'm not re-listing tonight.

I'm going to count down from 10 — if you're adding to cart, go now.

10... 9... 8... [pause, read comments] — I see orders coming in! — 7... 6... 5..."

[Check stock] "We're at [REMAINING]. If you're on the fence — this is the moment. 4... 3... 2... 1."

"Okay — link stays up for 2 more minutes while I move to the next product."
```

## Scarcity Re-Engage Script
```
"For anyone who just joined — we are down to [STOCK] units of [PRODUCT]. The people who grabbed it in the first 5 minutes got [BONUS]. I'm keeping the link live for a few more minutes — after that I'm pulling it."
```

## Closing Script (Final 5 min)
```
"Okay chat — we are almost at the end of tonight's live.

Before I go — I'm putting up one final thing. This is only available right now while we're live — [BUNDLE NAME] — everything you need to [OUTCOME], together for [BUNDLE PRICE].

I'm only doing [QUANTITY] of these tonight.

Link is pinning now.

And to everyone who bought tonight — thank you. I genuinely appreciate every single one of you. Notifications on so you know when I go live next — I'll be back [DAY/TIME].

Okay — [QUANTITY] of the bundle left — going... going... [count stock] — see you next time!"
```
