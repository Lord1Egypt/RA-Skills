# 项目初始化模板

## 目录结构模板

每个新项目运行 `scripts/init-project.py` 后生成以下结构：

```
[项目名]/
├── 00_总览.md
├── 01_资产库/
│   ├── 角色/
│   ├── 场景/
│   └── 道具/
├── 02_分镜/
├── 03_参考图索引/
│   ├── CHAR_001/
│   ├── SCENE_001/
│   ├── TPL_001/
│   └── SHOT_001/
├── 04_生成记录/
├── 05_一致性检查/
└── 06_返工记录/
```

## 模板文件清单

| 模板路径 | 用途 |
|---------|------|
| `01_资产库/角色/00_角色资产卡_模板.md` | 新建角色时复制填写 |
| `01_资产库/场景/00_场景资产卡_模板.md` | 新建场景时复制填写 |
| `01_资产库/道具/00_道具资产卡_模板.md` | 新建道具时复制填写 |
| `02_分镜/00_分镜表_模板.md` | 分镜总表 |
| `02_分镜/00_镜头卡_模板.md` | 单镜头详细卡 |
| `03_参考图索引/00_参考图索引_模板.md` | 参考图索引 |
| `04_生成记录/00_生成记录_模板.md` | 生成记录归档 |
| `05_一致性检查/00_一致性检查表_模板.md` | 一致性检查表 |
| `06_返工记录/00_返工记录_模板.md` | 返工记录 |

## 模板复制规则

1. 从 `AI视频制作工作流/99_模板/` 目录复制
2. 复制后重命名为 `资产类型_ID_名字.md`
3. 立即填写 frontmatter 中的 `assetId` 和 `title`
4. 状态默认为草稿

## ID 编号规则

- 角色：`CHAR_001` 起，最大 `CHAR_999`
- 场景：`SCENE_001` 起
- 道具：`PROP_001` 起
- 镜头：`SHOT_场景ID_镜头序号`（如 `SHOT_001_01`）
- 模板：`TPL_001` 起
- 生成记录：`GEN_001` 起
- 一致性检查：`CHECK_001` 起
- 返工记录：`RW_001` 起

## 新项目最小启动包

新项目至少需要：
- 1 个角色资产卡（已锁定）
- 1 个场景资产卡（已锁定）
- 1 个参考图目录（有内容）
