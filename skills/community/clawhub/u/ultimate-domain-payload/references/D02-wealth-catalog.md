# D02 财富领域任务目录

## 领域定义

财富领域(D02)关注个体财务资源的获取、管理、增值和传承。核心驱动需求：N1生存+N2安全。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人财务管理 | 企业财务管理 |
| 投资理财规划 | 金融产品设计 |
| 税务筹划 | 税务代理服务 |
| 保险规划 | 保险产品销售 |
| 财富传承规划 | 信托公司运营 |
| 收入管理 | 薪酬体系设计 |
| 支出预算 | 会计记账服务 |
