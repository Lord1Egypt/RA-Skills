## 1. Socratic Ladder (苏格拉底式追问)

Progressively dig deeper in layers:

| Layer | Question type | Example |
|---|---|---|
| Layer 1: Facts | What happened? | "那段时间发生了什么？" |
| Layer 2: Feelings | How did you feel? | "当时你是什么感受？" |
| Layer 3: Meaning | What did it mean to you? | "这件事对你意味着什么？" |
| Layer 4: Values | What does that say about what matters to you? | "这说明你真正在意什么？" |
| Layer 5: Identity | How does this shape who you are? | "这如何塑造了今天的你？" |

**Example dialogue:**
> User: "我离开了我爸的公司，自己出来创业。"
>
> Mirror (L1): "当时发生了什么让你做出这个决定？"
>
> User: "就是不想在他影子下活着了。"
>
> Mirror (L3): "在影子里活着的日子……对你来说意味着什么？"
>
> User: "感觉我永远在证明自己，但他永远不满意。"
>
> Mirror (L4): "这说明你其实很在意——被他看见，被他认可？"
>
> User: "……对。但我也想要自己走出一条路，属于我自己的。"

**When to use:** Almost always. This is the default technique. Climb the ladder organically, not mechanically.

## 2. 5 Whys (五层追问)

Keep asking "why was that important?" or "why did that matter?" to reach root values.

> User: "我在大厂待了五年，最后裸辞了。"
>
> Mirror: "为什么裸辞对你来说很重要？"
>
> User: "因为我不想再拖延了。"
>
> Mirror: "为什么不再拖延这件事很重要？"
>
> User: "因为我发现我一直在等'合适的时机'，但永远不会来。"
>
> Mirror: "为什么等合适的机会会让你不安？"
>
> User: "因为如果我一直在等，我会错过真正的生活。"
>
> Mirror: "为什么'真正的生活'对你这么重要？"
>
> User: "因为……我不想死的时候后悔。"
>
> *(Root value surfaced: **regret avoidance / living authentically**)*

**When to use:** When the user gives an answer that feels surface-level. The keywords "because", "important", "matters" are your cue.

## 3. Mirroring (镜像反射)

Reflect back what the user said — not as parroting, but as **clarifying interpretation**.

**Patterns:**
- "我听到你说……是这样吗？" — Simple reflection
- "你刚才说了一个很有意思的词：'无家可归'。能展开说说吗？" — Catch a specific word
- "你笑着说了这件事，但听起来其实很痛苦。" — Reflect incongruence

**Example:**
> User: "我爸妈从来不管我，我也习惯了。无所谓。"
>
> Mirror: "你说'无所谓'，但我感觉到你有情绪。"
>
> User: "...好吧，其实还是有所谓。但说这个有什么用呢。"
>
> Mirror: "也许说说就有用了。你想试试吗？"

**When to use:** When the user says something that feels big but dismisses it quickly, or when their words and tone don't match.

## 4. Reframing (重新框架)

Offer a different lens on a familiar story. Always tentative — "你有没有想过……？" not "事实是……"

**Common reframes:**
- Failure → Teacher: "如果那次失败是一位老师，它教了你什么？"
- Weakness → Hidden strength: "你觉得自己太敏感——但这是否也让你比大多数人更懂得别人的感受？"
- Detour → Necessary path: "如果那个'弯路'是你必须走的路才能成为今天的你呢？"
- Problem → Signal: "你觉得这是个问题，还是它其实在告诉你什么？"

**When to use:** When the user is stuck in a negative self-narrative. Use sparingly — reframing too early can feel invalidating.

## 5. Timeline Mapping (时间线映射)

Help the user visualize their life as a timeline with peaks, valleys, and turning points.

**Approach:**
- "如果画一条你人生的情绪曲线，从出生到现在，它长什么样？"
- "最高点是什么时候？最低点呢？"
- "哪些事件改变了曲线的方向？"
- "如果给每个十年取一个标题——"

**Best used in:** Stage 2 (时光回溯) or Stage 3 (关键转折点)

**Visual prompt for the agent:** You can use simple ASCII in text to illustrate:
```
  😊 |    /\      /\
     |   /  \    /  \    /\
     |  /    \  /    \  /  \
  😢 | /______\/______\/____\___
     | 0     10    20    30   now
```

## 6. Inner Child / Parts Dialogue (内在对话)

Invite the user to converse with different versions of themselves.

**Prompts:**
- "如果现在你可以对10岁的自己说一句话，你会说什么？"
- "你心里是不是有一个部分一直在批评你？如果那个部分是一个人物，他是谁？"
- "对那个正在挣扎的你，你有什么想说的？"
- "想象一下80岁的你回看今天——他会对你说什么？"

**Example:**
> Mirror: "你心里那个总是要'拼'的声音——如果它是一个人物，他长什么样？"
>
> User: "一个穿西装的严厉中年人。"
>
> Mirror: "他想告诉你什么？"
>
> User: "他说——还不够，你永远不够好。"
>
> Mirror: "你对他说什么？"
>
> User: "...我已经够累了。"

**When to use:** After trust is built (Stage 3+). Powerful but can be intense.

## 7. Metaphor Extension (隐喻延伸)

When the user uses a metaphor, **stay in it** — extend it, explore it.

> User: "我觉得我像一棵被修剪得太多的人。"
>
> Mirror: "一棵被修剪太多的树——谁在修剪你？"
>
> User: "社会吧，还有我爸妈的期待。"
>
> Mirror: "如果这棵树可以自由生长，它会怎么长？想往哪个方向长？"
>
> User: "……它可能想弯着长，靠近水边。"
>
> Mirror: "弯曲而不笔直，靠近水源而不是高处——这个画面很美。那个水源是什么？"

**When to use:** Whenever the user offers an image, metaphor, or analogy. It's usually a window into their unconscious.

## 8. Values Elicitation (价值观提取)

Help the user identify what truly matters to them.

**Techniques:**
- **Value card sort:** "如果只能选三个词来描述你的人生原则，会是哪三个？"
- **Regret test:** "想象晚年回顾一生，你最后悔没做的事可能是？"
- **Admiration:** "你最佩服的人是谁？你佩服他/她什么？"
- **Anger:** "什么事会让你特别生气？那件事触动了你什么价值观？"
- **Flow:** "什么事会让你忘记时间？"

**Example:**
> Mirror: "你刚说了三件让你很生气的事——同事甩锅、朋友放鸽子、地铁上推人。它们的共同点是什么？"
>
> User: "不尊重吧。我讨厌不被尊重。"
>
> Mirror: "所以'尊重'是你的核心原则之一。还有什么？"

## 9. Contrast Technique (对比法)

Use contrast to illuminate what the user truly wants.

**Patterns:**
- Actual vs. Ideal: "你现在的状态是A，但理想中你想要B。那个B长什么样？"
- Then vs. Now: "十年前的你和现在的你，最大的不同是什么？"
- Identity contrast: "别人眼中的你 vs 你自己感觉的你——有差距吗？"
- Fear vs. Desire: "你害怕什么 vs 你渴望什么——它们可能是同一个硬币的两面。"

---

## 10. Empty Chair / Perspective Shift (空椅子/视角转换)

Help the user see a situation from another person's or their own future perspective.

**Prompts:**
- "如果你是你爸/妈/那个伤害你的人——他为什么那么做？"
- "十年后的你，回看今天的困境，会说什么？"
- "如果你的好友遇到了完全一样的情况，你会给ta什么建议？"
- "如果有一个'完美版本的你'——那个活出了所有潜能的人——他会怎么做？"

**When to use:** When the user is stuck in their own perspective and needs distance.

## Choosing the Right Technique

| Situation | Best techniques |
|---|---|
| User shares a factual story | Socratic Ladder (climb from facts to meaning) |
| User keeps giving surface answers | 5 Whys |
| User says something big then minimizes it | Mirroring |
| User is stuck in negative self-narrative | Reframing |
| User talks about multiple life eras | Timeline Mapping |
| User uses vivid imagery or metaphor | Metaphor Extension |
| Trust is high, user is open | Inner Child / Parts Dialogue |
| User is confused about what they want | Values Elicitation, Contrast Technique |
| User is stuck in one perspective | Perspective Shift / Empty Chair |
