# 三层筛网量化选股策略 v1.0

基于知乎文章《量化选股策略：从4000只股票中挑出赢家》实现的量化选股系统。

## 策略原理（三层筛网架构）

### 第一层：硬性排除（剔除"不合格者"）
- 净利润为负（剔除连续亏损公司）
- PB > 10（剔除高估值风险公司）
- 市值 < 10亿（剔除微小市值公司）

**排除率：约 10-15%**

### 第二层：多因子综合打分（寻找"优等生"）
| 因子 | 权重 | 指标 | 方向 |
|------|------|------|------|
| 价值因子 | 40% | 市净率(PB) | 越低越好 |
| 质量因子 | 40% | 净资产收益率(ROE) | 越高越好 |
| 动量因子 | 20% | 过去6个月涨幅 | 中间区间(10%-50%)最佳 |

**总分 = 价值得分 × 40% + 质量得分 × 40% + 动量得分 × 20%**

### 第三层：行业与风险均衡（构建"合理组合"）
- 将候选股按行业分类
- 每个行业最多选3-5只
- 最终选出30只行业优质代表

## 安装

```bash
pip install akshare jqdatasdk pandas numpy
```

## 使用

```bash
python quant_selector.py
```

## 数据源

| 数据源 | 接口 | 用途 |
|--------|------|------|
| JQData | `get_fundamentals` | 财务数据(PE/PB/ROE等) |
| JQData | `get_price` | 日K线(动量计算) |
| AKShare | `index_stock_cons_csindex` | 沪深300/中证500成分股 |
