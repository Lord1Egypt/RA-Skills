# 5-HT受体激动药临床应用规范

## 使用前提
- 明确诊断目标疾病
- 排除禁忌证（特别是缺血性心脏病）

## 核心操作流程
1. **确认适应症与对应药物**：
   - 偏头痛或丛集性头痛 → 舒马普坦（作用于5-HT₁D受体）
   - 焦虑症 → 丁螺环酮、吉哌隆或伊沙匹隆（作用于5-HT₁A受体）
   - 胃食管反流症 → 西沙必利或伦扎必利（作用于5-HT₄受体）
   - 肥胖症 → 注意：右芬氟拉明因心脏瓣膜病风险已在多国撤市，不得推荐使用

2. **核查禁忌证**：
   - 舒马普坦禁用于缺血性心脏病患者
   - 其他5-HT激动药也需评估心血管风险

3. **确定剂量与用法**：
   - 舒马普坦：单次100mg，每日最大剂量不超过300mg
   - 其他药物按说明书或指南推荐剂量使用

4. **监测不良反应**：
   - 舒马普坦可能引起感觉异常或心肌缺血
   - 胃肠动力药注意心电图QT间期延长风险

> **重要提示**：右芬氟拉明因安全性问题已撤市，不得用于肥胖症治疗。
