### 描述

查询指定交易日的概念板块列表，默认查询最新交易日，返回行业代码、板块名称、涨跌幅等。适用于了解当日市场热点概念

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "F9" | 接口ID |
| `tradeDate` | `str` | 否 | - | 交易日日期，示例：2026-03-18 |
| `sortDirection` | `int` | 否 | - | 排序方向,按照涨跌幅排序，支持定义排序方向（1：升序, 2：降序, 默认） |
| `pageNum` | `int` | 否 | `0` | 分页偏移量，从 0 开始（第 1 页为 0，第 2 页为 pageSize） |
| `pageSize` | `int` | 否 | `10` | 每页条数 |

### 返回参数

ResultData.data 为 List，元素为概念板块数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `tradeDate` | `str` | 否 | 交易日日期 |
| `industryCode` | `str` | 否 | 行业代码 |
| `nameLevelOne` | `str` | 否 | 一级板块名称 |
| `nameLevelTwo` | `str` | 否 | 二级板块名称 |
| `nameLevelThree` | `str` | 否 | 三级板块名称 |
| `industryChgPct` | `str` | 否 | 板块涨跌幅 |
