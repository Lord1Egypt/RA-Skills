# 境外城市生活成本与家庭预算适配

## 适用场景

只要项目存在或可能存在境外学习阶段，或用户表达明确出国/可接受出国，就必须把外方大学所在城市生活成本纳入费用分析。不能只看国内学费，也不能凭经验估计境外生活费。

## 必搜信息

1. 外方大学官方 international student cost-of-living / student budget / accommodation 页面。
2. 外方大学所在国家或地区的政府、移民、签证或学生资金要求页面。
3. 城市维度生活成本：住宿/租金、餐饮、交通、保险/医疗、教材材料、通信、当地通勤和基本个人开销。
4. 当前汇率来源，用于换算成人民币；说明汇率波动会改变预算结论。
5. 项目培养方案或招生章程中明确的境外学习时长、是否必须出国、境外学费是否另计。

## 输出格式

```markdown
### 境外城市生活成本与家庭预算适配

- 外方大学所在城市：...
- 境外学习时长：...
- 年度生活成本构成：住宿约...、餐饮约...、交通约...、保险/医疗约...、教材材料约...、其他约...
- 年度境外总支出：境外学费 + 生活成本 + 保险/签证/交通 + 汇率缓冲 = 约...
- 年度预算差额：家庭年度预算约... - 年度总支出约... = 约...
- 对个性化推荐度的影响：...
- 人工确认：请打开链接核对城市/校区、学年、币种、费用是否含住宿、是否含保险、是否是最低要求还是建议预算。
- 质量等级：A / B / C / D。
```

## 判断规则

- 年度预算明显覆盖总支出且留有汇率/延毕缓冲：费用压力可判为低或中。
- 年度预算只覆盖学费但覆盖不了城市生活成本：个性化推荐度必须降级。
- 境外城市生活成本未核到官方或权威资料：费用置信度降级，不能把项目判为高度适合。
- 若项目可全程国内完成但用户想出国，分别给“国内完成”和“出国路径”两套预算结论。
- 若项目必须出国但家庭预算紧张，应明确给出慎选或不太适合，除非学校有官方奖学金、费用减免或家庭能接受额外预算。

## 禁止项

- 禁止只写“国外一年大概几十万”而没有城市、构成和来源。
- 禁止把外方国家平均生活费直接当作大学所在城市成本。
- 禁止用公开讨论或机构宣传作为唯一生活成本来源。
- 禁止忽略汇率、保险、签证、住宿押金和假期住宿成本。
