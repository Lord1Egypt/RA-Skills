# Assignment Requirements

[把你的作业要求粘贴在这里]

## 例子格式:

### Course: CS5123 - Machine Learning
### Assignment: Final Project
### Deadline: Week 13

**Task**: Build a machine learning pipeline that...

**Requirements**:
1. Data preprocessing
2. Model training
3. Evaluation metrics
4. Visualization of results

**Deliverables**:
- Source code (.py files)
- Report (max 10 pages, .docx format)
- Presentation slides (10-15 minutes)

**Grading Criteria**:
- Technical correctness: 40%
- Innovation: 20%
- Report quality: 20%
- Presentation: 20%
