## 🛡️ 安全原则

1. **最小权限** - 只请求必要的权限
2. **隔离** - 敏感操作在独立进程
3. **超时** - 所有外部调用设置超时
4. **审计** - 关键操作记录日志
5. **用户控制** - 敏感操作需确认
