## 二、Physics/Action · 物理动作（30%）

### 力量传递链
```
施力姿态 → 力量发出 → 介质传递 → 接触点 → 受力反应 → 重心转移
```

**示例：拳击**
```
右拳后拉至肩侧，肱三头肌紧绷 → 腰部发力驱动肩胛 → 右臂前冲，拳面加速 → 拳峰接触对方面部鼻梁 → 对手头部后仰15度 → 重心前倾失衡
```

### 武器碰撞物理
```
剑锋斜向下45度 → 接触对方刀身前段 → 力沿刀身传导 → 对方虎口震麻，刀身偏移30度 → 对方身体借力后撤
```

### 地面反作用力
| 地面 | 效果 |
|------|------|
| 硬石地 | 声音脆，脚步稳，无下陷 |
| 泥地 | 下陷3-5cm，拔脚时有黏滞感 |
| 木板 | 弹性，脚步有轻响，隔音差 |
| 沙地 | 脚印深，拔脚扬尘 |
| 草地 | 脚步无声，草叶倒伏 |

## 力量/反作用力循环实例

### 实例1：剑与刀碰撞
**A的攻击（同时=B的受力+B的反击起点）：**
> "A双手握刀，刀身自右上向左下斜劈；刀刃接触B剑身前段，金属撞击火花飞溅，B虎口震动刀身偏转15度，B借惯性身体左旋反手一剑刺向A右肋"

### 实例2：重拳击打
**A的攻击 = B的受力：**
> "A右拳内旋直击，拳峰正中B腹部，B胃部凹陷，身体前屈，口腔内分泌增加，双腿微曲降低重心"

**B的反击 = A的受力：**
> "B借前屈姿态发力，头槌撞向A胸口，A胸骨受击后仰，脚步踉跄后退两步"

### 实例3：法术对轰
**能量碰撞物理：**
> "红色火球与蓝色寒流在两人之间碰撞，接触点爆发白色闪光，周围空气被能量波挤压向两侧，形成以碰撞点为圆心的扩散气流"
