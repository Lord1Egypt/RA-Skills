# 叉车选型决策指南

> 一份按决策顺序的选型流程,搭配车型/品牌推荐。
> 选型三要素:**工况 → 吨位 → 动力类型**,三者定了再挑品牌和配置。

## 一、选型流程图

```
[Step 1] 室内 vs 室外?
   │
   ├── 室内为主 → [Step 2A] 电动路径
   └── 室外为主 → [Step 3] 看工况细节
       │
       ├── 室外重载/港口/无充电条件 → 内燃
       └── 半室外/混合 → [Step 4] 锂电 vs 铅酸

[Step 2A] 电动路径
   │
   ├── 货架高度?
   │   ├── < 4m → Class I 平衡重式(座驾)或 Class III 步行式
   │   ├── 4-8m → Class I 高起升 / Class II 前移式
   │   └── > 8m → Class II 前移式 / Class II 三向堆垛
   │
   ├── 通道宽度?
   │   ├── > 3.5m → 普通平衡重
   │   ├── 2.5-3.5m → 前移式 / 窄通道
   │   └── < 2.5m → 三向堆垛 / VNA
   │
   └── 班次?
       ├── 单班 → 铅酸可考虑
       └── 双班/多班 → 锂电(快充 + 免维护)

[Step 4] 锂电 vs 铅酸(已选电动的)
   │
   ├── 预算紧 / 单班 / 有充电窗口 → 铅酸
   ├── 双班 / 多班 / 需快充 → 锂电(LFP)
   ├── 极寒(冷库 -20℃ 以下)→ 锂电带加热 OR 钠电(比亚迪 2026 量产)
   └── 高温 / 防爆 → 锂电(LFP 安全性好,带 BMS)

[Step 5] 品牌 + 配置
   │
   ├── 预算紧 → 中力、诺力、宝骊、如意
   ├── 主流工业 → 合力、杭叉、柳工、龙工、丰田、海斯特
   ├── 高端 / 港口 → 林德、永恒力、科朗、卡尔玛
   ├── 锂电专业 → 比亚迪、杭叉 XE、合力 K、丰田 8FBE
   └── 智能物流 → 合力 + 华为、杭叉 AGV、海康/极智嘉
```

## 二、按行业场景推荐(2025-2026)

| 行业 | 工况特征 | 推荐类型 | 品牌建议 |
|------|------|------|------|
| **冷链/冷库** | -25℃, 卫生要求 | 冷库型锂电(带电池加热)/ 钠电 | 比亚迪钠电(2026)、合力冷库型 |
| **食品饮料** | 卫生、低噪、室内 | I 类座驾式锂电 + 不锈钢属具 | 比亚迪、杭叉 XE、合力 K |
| **医药** | GMP、低噪、零排放 | I 类锂电 + 食品级液压油 | 比亚迪、林德、杭叉 XE |
| **3C 电子** | 静电敏感、洁净 | 防静电(ESD)锂电 + 离子风机 | 比亚迪、林德、海斯特 J 系列 |
| **汽车制造** | 重载、连续作业、产线配送 | I 类锂电 / 大吨位内燃 | 合力 K、杭叉、林德 |
| **港口/集装箱** | 室外、25-46 吨 | V 类柴油 / 集装箱搬运车 | 卡尔玛、杭叉、柳工、龙工 |
| **电商/快递** | 高频、标准化通道 | III 类步行式 + AGV | 中力、诺力、海康 AGV、极智嘉 |
| **建材/钢材** | 重载、室外 | 5-10 吨 IV/V 类柴油 | 柳工、龙工、杭叉 |
| **化工/制药** | 防爆要求 | 防爆锂电/防爆柴油(限用) | T/QGCML 4100-2024 合规产品 |
| **制造业车间** | 多班、混合工况 | 锂电 I 类 2-3.5 吨 | 合力 K、杭叉 XE、比亚迪 |
| **批发市场** | 室外、装卸货 | 3-5 吨柴油 / LPG | 合力、杭叉、柳工 |
| **物流仓储** | 高位货架 | II 类前移式 / 三向堆垛 | 林德、永恒力、科朗 |

## 三、吨位与载荷选型

**核心原则:实际负载 = 货叉货物重量 + 属具自重,留 20% 安全裕度**

| 标称吨位 | 典型应用 | 推荐货物重量 |
|------|------|------|
| 1.0-1.6 吨 | 电商、装卸、超市 | < 1.3 吨 |
| 2.0-2.5 吨 | 通用制造业、车间 | < 2 吨 |
| 3.0-3.5 吨 | 主流工业 | < 2.8 吨 |
| 5.0 吨 | 重载车间、批发市场 | < 4 吨 |
| 7.0-10 吨 | 钢材、建材 | < 8 吨 |
| 16-25 吨 | 港口、集装箱 | < 22 吨 |

**载荷曲线衰减(GB/T 10827)**:
- 起升高度每增加 1 米,载荷能力下降约 1-2%
- 重心距每超出标准 100mm,载荷能力下降 10-15%
- **配置属具后必须用属具的实际重心和重量重算载荷**

## 四、配置选型 Checklist

### 4.1 必备配置
- [ ] 货叉长度(标准 1070mm,加长 1220/1500/1800mm)
- [ ] 货叉架(2 级或 3 级门架)
- [ ] 起升高度(根据货架/作业需求定)
- [ ] 蓄电池容量(kWh)或 内燃机型号
- [ ] 充电桩(锂电需专用,内燃免)
- [ ] 护顶架(FOPS)
- [ ] 安全带、倒车蜂鸣、灯光

### 4.2 常用选配
- [ ] 侧移器(±100mm 侧移,装卸对位)
- [ ] 调距叉(可变货叉间距)
- [ ] 属具(纸卷夹、油桶夹、推出器)
- [ ] 冷库型改装(防锈、电池加热、低温液压油)
- [ ] 食品级改装(不锈钢货叉、H1 液压油)
- [ ] 防爆改装(防爆电机、本安电路)
- [ ] 数字称重 + 货叉(称重 + 仓储管理)
- [ ] 摄像头 + 显示器(高位堆垛安全)
- [ ] 智能调度(车联网、车队管理)

### 4.3 锂电选配要点
- [ ] 电池容量(常用 80V/300-500Ah,折合 24-40 kWh)
- [ ] 充电桩功率(常规 30kW / 快充 80kW+)
- [ ] 电池品牌(比亚迪、宁德、亿纬、国轩)
- [ ] 加热膜(低温环境)
- [ ] BMS(电池管理系统,主流标配)
- [ ] 充电策略(夜间慢充 + 白天快充补电)

## 五、采购合同关键条款

- 主机 + 电池 + 充电桩 + 属具分项报价
- 质保期:**整车 2 年/2000 工作小时**,电池单独质保(主流 5 年/10000 小时)
- 上牌/检测协助(厂内特种设备)
- 操作员培训(免费基础 + 付费专项)
- 备件供应承诺(关键件 24-48 小时到货)
- 上门服务半径(80-150 公里)
- 二手置换 / 残值回购政策

## 六、总拥有成本(TCO)计算框架

5 年 TCO 公式(可让用户套数字):

```
TCO = 采购 + 能源 + 维护 + 培训 + 保险 + 残值(-)

采购:主机 + 属具 + 充电桩 + 安装
能源:电费(电动)= 工时 × kWh/h × 单价;柴油(内燃)= 工时 × L/h × 单价
维护:含备件 + 人工(电动约 0.4 元/工时,内燃约 0.9 元/工时)
培训:操作员初训 + 复训
保险:特种设备责任险 + 财产险
残值:5 年后电池梯次利用 / 整车二手
```

**经验值**(2025-2026 行情,3 吨双班):
- 锂电 TCO: 25-28 万/5 年
- 内燃 TCO: 30-35 万/5 年

## 七、常见选型决策失误

1. **吨位选大**:多花了 5-8 万,载荷能力浪费
2. **吨位选小**:长期超载,门架/液压早期损坏,GB 10827 也禁止
3. **盲信"锂电更省钱"**:单班工况下铅酸 TCO 反而更低
4. **忽视充电条件**:室内无充电桩位 + 大容量锂电,等效于"买了手机没充电器"
5. **忽视属具成本**:属具 1-5 万,远超"小配置"心理预期
6. **忽视服务半径**:偏远地区买了进口品牌,故障停机 1 周,损失远超采购差价
7. **忽视培训**:操作不当是 70% 早期故障的根因

## 八、决策建议模板(可直接发给用户参考)

> **场景**:某汽车零部件厂,2 班工作,主要搬运铸件(单件 1.2 吨),起升 3 米,室内车间 + 短距离室外
>
> **建议**:
> - 吨位:2.5 吨(I 类平衡重)
> - 动力:锂电(双班,需快充)
> - 门架:2 级标准 + 3000mm 起升
> - 属具:标配货叉 + 侧移器
> - 备选:比亚迪 2.5T 锂电 / 合力 K2 系列 / 杭叉 XE25
> - 预算:18-25 万主机 + 5-8 万属具 + 2-3 万充电桩
