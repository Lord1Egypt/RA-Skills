# 阵营本创作指南

## 阵营本的定义

阵营本的核心体验是"对抗"——玩家分成不同阵营，通过策略和博弈争取胜利。

## 阵营类型

### 明阵营

**特点**：玩家知道自己属于哪个阵营。

**设计要点**：
- 阵营目标明确
- 阵营内部有合作，阵营之间有对抗
- 胜利条件清晰

### 暗阵营

**特点**：玩家不知道谁是队友。

**设计要点**：
- 需要通过信息判断谁是队友
- 信任是稀缺资源
- 可能有"叛徒"机制

### 混合阵营

**特点**：部分明阵营，部分暗阵营。

**设计要点**：
- 阵营身份逐步揭露
- 动态变化的阵营关系
- 更复杂的策略空间

## 阵营本设计要点

### 阵营平衡

**平衡原则**：
- 阵营人数不能差距太大
- 阵营能力要相对均衡
- 有翻盘机制（弱者有机会逆转）

### 胜利条件

**常见胜利条件**：
- 阵营目标：完成特定任务
- 投票胜利：投票选出对方阵营的人
- 生存胜利：活到最后
- 积分胜利：获得最多积分

### 私聊机制

阵营本的核心是私聊——玩家通过私聊结盟、交换信息、背叛。

**私聊设计**：
- 私聊时间要充足（每人至少10-15分钟）
- 私聊内容可以撒谎
- 有"偷听"机制（增加策略性）

## 阵营本结构模板

### 标准结构（4-5小时）

```
第一幕：阵营确认（30-40分钟）
  - 阅读剧本，确认阵营身份
  - 第一轮公共讨论
  - 初步建立阵营关系

第二幕：策略博弈（90-120分钟）
  - 多轮私聊
  - 信息交换与欺骗
  - 执行阵营任务
  - 中间投票/对抗环节

第三幕：阵营对决（60-80分钟）
  - 真相揭露
  - 最终阵营对抗
  - 胜负判定

第四幕：复盘（15-20分钟）
  - 揭示所有隐藏信息
  - 讨论策略得失
```

## 阵营本示例：8人阵营本

### 《谍影重重》（示例）

**类型**：阵营本
**人数**：8人（4男4女）
**时长**：4-5小时
**背景**：民国·上海·1941

**故事梗概**：
1941年上海，日本宪兵队举办一场舞会。舞会表面是社交，实际是各方势力的情报战场。有日本军官、汪伪政府官员、军统特工、中共地下党。每个人都有自己的任务，而任务的完成将决定历史的走向。

**阵营设计**：
- 日本阵营（2人）：保护情报，找出间谍
- 汪伪阵营（2人）：两面讨好，保全自己
- 军统阵营（2人）：获取情报，刺杀目标
- 中共阵营（2人）：传递情报，保护同志

**胜利条件**：
- 日本阵营：找出所有间谍
- 汪伪阵营：活到最后
- 军统阵营：完成刺杀任务
- 中共阵营：成功传递情报

**特殊机制**：
- 私聊舞会：每轮有固定的私聊时间
- 情报交易：可以用情报换取利益
- 背叛机制：可以叛变到其他阵营
