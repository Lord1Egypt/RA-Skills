# 示例：研究生毕业答辩

## 输入

- 论文主题
- 研究背景
- 方法
- 实验结果
- 总结

## 共享主线收口

### story_units

- cover
- problem
- contribution
- method
- experiment
- result
- ablation
- closing

### 文字层级

- kicker：页型或章节提示
- headline：本页主结论
- summary：一句话解释
- body：实验、方法说明
- metric：结果数字
- annotation：边界与补充说明

### 信息图元素

- 背景页：问题卡片
- 方法页：流程图
- 结果页：指标卡与对比条
- 分析页：消融矩阵

## adapter 选择

- 需要可编辑性
- 需要正式答辩页结构
- 最终选 `ppt`
