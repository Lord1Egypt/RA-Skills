# Pediatric Fingerprint Diagnosis Three Guan Four Dimensions

## When to Use
- 仅适用于3岁以内小儿。

## When to Use
- 在光线充足处操作；
- 医生需以拇指蘸清水，从患儿示指指端向掌心方向推擦数次，使络脉显露。

## Precautions
- 不适用于3岁以上儿童；
- 避免在光线不足或患儿哭闹状态下观察。

## Execution Steps

### 1. 解剖定位（三关）
- **风关**：示指第一节（靠近指尖）；
- **气关**：示指第二节；
- **命关**：示指第三节（靠近掌心）。

### 2. 正常指纹特征
- 色泽红黄隐隐；
- 仅显于风关之内。

### 3. 四维辨证法则

#### 浮沉分表里
- **浮显**（浅表易见）→ 表证；
- **深沉**（隐伏难见）→ 里证。

#### 色泽辨病性
- **红紫** → 热证；
- **鲜红** → 寒证；
- **青色** → 惊风或疼痛；
- **紫黑** → 血络闭郁，病情危重；
- **浅淡而白** → 虚证。

#### 淡滞定虚实
- **色浅淡、推之流畅** → 虚证；
- **色浓滞、推之不移** → 实证。

#### 三关测轻重
- **仅在风关** → 病轻；
- **达气关** → 病重；
- **显于命关** → 病危；
- **透关射甲**（直达指端）→ 临床危象，预后极差。

### 4. 标准操作步骤
1. 抱小儿面向光源；
2. 左手握其示指指端固定；
3. 右手拇指蘸清水，自指端向掌心方向推擦数次（力度适中）；
4. 观察指纹位置、颜色与形态；
5. 结合四维辨证法则综合判断病性、病位与病情等级。

## Output
- 表里属性（表证/里证）；
- 寒热虚实性质；
- 病位深浅；
- 病情轻重等级（轻/重/危）。
