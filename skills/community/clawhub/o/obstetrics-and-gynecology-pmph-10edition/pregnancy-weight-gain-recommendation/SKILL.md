# 妊娠期体重增长个体化推荐标准

## 何时使用
- 已知孕妇孕前体重和身高
- 已计算出孕前BMI
- 适用于单胎妊娠、无严重慢性疾病者
- 不适用于多胎妊娠或需特殊医学干预者

## 执行步骤
1. **计算孕前BMI**：
   - BMI = 孕前体重（kg） ÷ 身高²（m²）

2. **按BMI分类匹配推荐值**：
   - **低体重（BMI < 18.5）**：总增重11.0–16.0 kg，孕中晚期每周0.46 kg
   - **正常体重（18.5 ≤ BMI < 24.0）**：总增重8.0–14.0 kg，每周0.37 kg
   - **超重（24.0 ≤ BMI < 28.0）**：总增重7.0–11.0 kg，每周0.30 kg
   - **肥胖（BMI ≥ 28.0）**：总增重5.0–9.0 kg，每周0.22 kg

3. **实施要点**：
   - 首次产检即确定BMI并启动指导
   - 定期监测实际增重是否在推荐范围内
   - 结合饮食与运动干预

4. **风险提示**：
   - 增重过多：增加大于胎龄儿、妊娠糖尿病、难产风险
   - 增重不足：增加胎儿生长受限、早产、低体重儿风险

5. **输出结果**：
   - 输出对应BMI类别的总增重范围和每周增重目标值，用于制定个性化管理计划
