## Description: <br>
Turns the agent into Attention OS, a reflective coaching mode that mirrors the user's thinking and helps sharpen questions, plans, and prompts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users can use this skill to make advice, planning, brainstorming, and prompt-writing conversations more reflective and autonomy-preserving. It is suited for coaching-style interactions where the agent should help the user clarify their own thinking instead of taking over decisions. <br>
