# Xin Shen Interaction Mechanism

## When to Use
- 辨识心肾不交证候
- 解释失眠伴腰膝酸软的病机
- 指导交通心肾、滋阴降火的治疗方向
- 区分单纯心火亢盛或肾精亏虚与心肾两脏失调

## Execution Steps

1. **确认生理基础**：
   - 心属火，居上焦，为阳；肾属水，居下焦，为阴
   - 心火需下温肾水，防止肾寒
   - 肾水需上济心火，防止心火亢盛
   - 此种动态平衡称为“心肾相交”或“水火既济”

2. **识别病理机制（心肾不交）**：
   - 心火不能下温 → 肾水寒凝
   - 肾水不能上济 → 心火独亢
   - 导致阴阳失衡、水火不济

3. **判断临床表现组合**：
   - **心系症状**：心烦失眠、心悸、健忘、口舌生疮
   - **肾系症状**：头晕耳鸣、腰膝酸软、遗精、梦交、潮热盗汗
   - 必须两类症状并见，方可诊断为心肾不交

4. **排除非适用情况**：
   - 单纯外感热病引起的心烦
   - 仅见肾虚或仅见心火的单一脏腑病变

5. **输出诊疗建议**：
   - 治法应以交通心肾、滋阴降火为主
   - 常用方如黄连阿胶汤、交泰丸等
