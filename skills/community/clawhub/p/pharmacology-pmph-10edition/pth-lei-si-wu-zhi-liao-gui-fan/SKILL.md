# PTH类药物治疗规范

## 适用对象
- 绝经后骨质疏松女性（尤其有脊椎骨折史者）
- 骨质疏松症患者（骨折高风险）

## 启动条件
满足以下全部条件方可考虑启动PTH类似物治疗：
1. 存在低血钙或骨密度显著降低
2. 无以下禁忌证：
   - Paget骨病
   - 骨骼放疗史
   - 肿瘤骨转移
   - 高钙血症
3. 血清25-(OH)D₃水平 ≥ 30 ng/ml
4. 患者为骨折高风险的绝经后女性，且无心血管禁忌

## 执行流程
1. **确认PTH生理机制理解**：
   - PTH通过间歇小剂量给药激活成骨细胞，促进骨形成；持续高浓度则导致骨吸收。
2. **选择合适制剂**：
   - rhPTH 1-34（特立帕肽）
   - rhPTH 1-84（重组人PTH全长）
   - 阿巴帕肽（hPTH 1-34类似物）
3. **制定治疗计划**：
   - 疗程严格限制为 ≤ 24个月（2年）
   - 治疗期间监测血钙、肾功能及骨代谢标志物
4. **规划序贯治疗**：
   - 停药后必须立即启动抗骨吸收药物（如双膦酸盐或地舒单抗）以维持骨密度增益

## 安全警示
- 禁用于性腺功能正常男性
- 可能增加骨肉瘤风险（动物实验数据），故严格限用2年
- 停药后骨密度可能快速下降，未序贯抗吸收治疗将丧失疗效

## 输出结果
启动PTH类似物治疗，并同步制定2年内停药及后续抗骨吸收药物序贯计划。
