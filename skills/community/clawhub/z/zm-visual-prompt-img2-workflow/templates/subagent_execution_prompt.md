# 可复制给 SubAgent 的执行提示词

你正在执行 `zm-visual-prompt-img2-workflow` Skill。

请把结构化视觉需求编译为可执行的 happy/gpt-image-2 提示词。

## 最小必填

- 产物类型：漫画页 / PPT 主视觉 / 封面 / 配图
- 页面目标
- 画面主体
- 主角图 / 参考图路径
- 风格参考
- 文字是否入图
- 禁止项
- 输出尺寸 / 比例
- 是否必须 img2img

## 硬门禁

- 正式视觉必须使用 happy/gpt-image-2 / IMG2 链路。
- 漫画固定主角必须有主角图/角色参考；缺失则 BLOCKED。
- 二维码、Logo、真实产品界面不得让模型伪造。
- 漫画图本体不得出现二维码/二维码占位框。
- 不能用旧图打补丁冒充新图。

## 输出

- 编译后的提示词
- 参考图清单
- 禁止项清单
- 运行命令/调用建议
- BLOCKED 缺口
