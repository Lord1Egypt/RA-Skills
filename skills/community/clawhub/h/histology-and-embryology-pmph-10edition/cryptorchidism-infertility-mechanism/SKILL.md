# 隐睾与不育机制分析

## 何时使用
- 患者为男性胎儿或儿童，确诊隐睾（睾丸未完全下降至阴囊）
- 需评估其未来生育能力风险
- 睾丸位置明确为腹膜腔或腹股沟
- 明确为单侧或双侧隐睾

## 执行步骤
1. **确认隐睾定义**：睾丸停留在腹膜腔或腹股沟，未降至阴囊。
2. **判断侧别与位置**：
   - 若为**双侧腹膜腔内隐睾** → 高温环境（高于阴囊2–3℃）严重抑制精原细胞分化和精子生成，导致不育。
   - 若为**单侧隐睾**或**腹股沟隐睾** → 可能保留部分生育能力。
3. **输出风险警示**：仅当双侧睾丸均位于腹膜腔时，发出“存在男性不育风险”警报。
4. **建议干预**：推荐早期手术（如睾丸固定术）以改善预后。

> 注意：本技能仅适用于双侧隐睾导致不育的情境；单侧隐睾通常不影响整体生育能力。
