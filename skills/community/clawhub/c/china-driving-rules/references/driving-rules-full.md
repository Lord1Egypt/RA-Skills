## 1. 驾驶证分类与申领

### 1.1 准驾车型代号

| 代号 | 车型 | 准驾车辆 |
|------|------|----------|
| C1 | 小型汽车 | 小型、微型载客汽车 + 轻型、微型载货汽车 |
| C2 | 小型自动挡汽车 | 小型、微型自动挡载客汽车 + 轻型、微型自动挡载货汽车 |
| A1 | 大型客车 | 大型载客汽车 |
| A2 | 牵引车 | 重型、中型全挂、半挂汽车列车 |
| B1 | 中型客车 | 中型载客汽车 |
| B2 | 大型货车 | 重型、中型载货汽车 |

### 1.2 申领条件

| 车型 | 年龄限制 | 其他要求 |
|------|----------|----------|
| C1/C2 | 18 周岁以上 | 无限制 |
| B1/B2 | 20-60 周岁 | 需有 C 照驾龄 |
| A1 | 22-60 周岁 | 需有 B 照驾龄 2 年以上 |
| A2 | 22-60 周岁 | 需有 B 照驾龄 1 年以上 |

### 1.3 驾驶证有效期

- 初次申领：**6 年**
- 第二次换证：**10 年**
- 第三次换证：**长期有效**

### 1.4 换证时间

- 期满前 **90 日内** 申请换证
- 超过有效期 **1 年内** 可正常换证
- 超过 **1-3 年** 需重考科目一
- 超过 **3 年** 驾驶证注销

### 1.5 实习期规定

- 实习期：**12 个月**
- 实习期上高速：需有 **3 年以上** 相应或更高准驾车型驾驶人陪同
- 实习期扣分：扣满 **12 分** 注销实习车型

## 2. 交通信号灯

### 2.1 机动车信号灯

| 灯色 | 含义 |
|------|------|
| 🔴 红灯 | 禁止通行（右转箭头红灯除外，圆饼红灯可右转但需让行） |
| 🟡 黄灯 | 已越过停止线的可以继续通行 |
| 🟢 绿灯 | 准许通行 |

### 2.2 特殊信号灯

- **箭头红灯**：禁止该方向通行
- **闪烁黄灯**：瞭望确认安全后通过
- **铁路道口红灯**：停车等待，禁止抢行

### 2.3 黄灯规则

> 黄灯亮时，**已越过停止线**的车辆可以继续通行，**未越过**的应当停车

## 3. 交通标志

### 3.1 警告标志（⚠️ 黄底黑边三角形）

| 标志 | 含义 |
|------|------|
| 十字交叉路口 | 前方有十字路口 |
| 连续弯道 | 前方有连续弯道 |
| 陡坡 | 上/下陡坡 |
| 注意行人 | 前方有人行横道 |
| 注意儿童 | 学校区域 |
| 注意牲畜 | 可能有牲畜出没 |

### 3.2 禁令标志（🚫 白底红圈）

| 标志 | 含义 |
|------|------|
| 禁止通行 | 禁止一切车辆行人通行 |
| 禁止驶入 | 禁止车辆驶入 |
| 禁止停车 | 禁止停车（临时 + 长时） |
| 禁止长时停车 | 可临时停车 |
| 禁止左转/右转 | 禁止相应方向转弯 |
| 禁止掉头 | 禁止掉头 |
| 限速标志 | 最高时速限制 |
| 禁止鸣喇叭 | 禁止鸣笛 |

### 3.3 指示标志（🔵 蓝底白图案）

| 标志 | 含义 |
|------|------|
| 直行 | 只准直行 |
| 左转/右转 | 只准相应方向转弯 |
| 环岛行驶 | 进入环岛 |
| 人行横道 | 人行横道入口 |
| 机动车道 | 机动车行驶 |

### 3.4 指路标志（🟢 绿底/🔵 蓝底）

- **高速公路**：绿底白字
- **一般道路**：蓝底白字
- **旅游区**：棕底白字

### 3.5 辅助标志（⚪ 白底黑字黑框）

设在主标志下方，说明时间、车种、距离等

## 4. 交通标线

### 4.1 指示标线

| 标线 | 含义 |
|------|------|
| 白色虚线 | 同向车道分隔线，可变更车道 |
| 白色实线 | 同向车道分隔线，禁止变更车道 |
| 黄色虚线 | 对向车道分隔线，可越线超车 |
| 黄色实线 | 对向车道分隔线，禁止越线 |
| 双黄实线 | 严格禁止越线 |
| 一实一虚 | 虚线侧可越线超车 |

### 4.2 禁止标线

| 标线 | 含义 |
|------|------|
| 黄色网格线 | 禁止停车（消防通道、路口等） |
| 导流线 | 禁止压线或越线行驶 |
| 停止线 | 红灯时停车位置 |
| 人行横道线 | 礼让行人 |

### 4.3 特殊标线

- **减速让行线**：倒三角形 + 虚线，需减速让行
- **停车让行线**：双实线 + "停"字，需停车观察
- **车距确认线**：白色折线/半圆，确认安全车距

## 5. 通行规则

### 5.1 让行规则

> **口诀：转弯让直行，右转让左转，支路让主路**

| 情况 | 让行规则 |
|------|----------|
| 转弯 vs 直行 | 转弯让直行 |
| 右转 vs 左转 | 右转让左转 |
| 支路 vs 主路 | 支路让主路 |
| 环岛 | 进环岛让出环岛 |
| 坡道 | 下坡让上坡（下坡已行至中途除外） |
| 有障碍 | 有障碍方让无障碍方 |
| 人行横道 | 必须停车让行 |

### 5.2 超车规则

**禁止超车的情况：**
- ❌ 前车正在左转、掉头、超车
- ❌ 对向有来车
- ❌ 前车为执行紧急任务的警车、消防车、救护车、工程救险车
- ❌ 行经铁路道口、交叉路口、窄桥、弯道、陡坡、隧道、人行横道、市区交通流量大的路段

**正确超车：**
- 从**左侧**超车
- 提前开启**左转向灯**
- 超车后开启**右转向灯**驶回原车道

### 5.3 掉头规则

**允许掉头的情况：**
- ✅ 有掉头信号灯且为绿灯
- ✅ 无禁止掉头标志，且不影响其他车辆
- ✅ 黄色虚线侧可以掉头

**禁止掉头的情况：**
- ❌ 有禁止掉头标志
- ❌ 人行横道上
- ❌ 铁路道口、桥梁、急弯、陡坡、隧道
- ❌ 高速公路（除指定区域）

### 5.4 限速规定

| 道路类型 | 最高限速 |
|----------|----------|
| 高速公路 | 120 km/h |
| 城市快速路 | 80-100 km/h |
| 城市道路（有中心线） | 70 km/h |
| 城市道路（无中心线） | 50 km/h |
| 公路（有中心线） | 80 km/h |
| 公路（无中心线） | 60 km/h |
| 窄路、窄桥 | 30 km/h |
| 进出非机动车道 | 30 km/h |
| 铁路道口、急弯、陡坡、隧道 | 30 km/h |
| 能见度<50m | 30 km/h（高速应驶离） |

**高速公路最低限速：**
- 最高车速>100km/h 时，车距≥100m
- 最高车速<100km/h 时，车距≥50m

### 5.5 高速公路特别规定

- **最低车速**：60 km/h（最左侧车道最低 110km/h）
- **车道车速**：
  - 2 车道：左 100-120，右 60-100
  - 3 车道：左 110-120，中 90-110，右 60-90
- **禁止行为**：倒车、逆行、穿越中央分隔带掉头、停车（除服务区）
- **驶离高速**：提前 **500m** 开启右转向灯

## 7. 交通事故处理

### 7.1 事故处理流程

1. **立即停车**，开启危险报警闪光灯
2. **设置警告标志**（普通道路 50-100m，高速 150m 外）
3. **抢救伤员**，拨打 120
4. **报警**，拨打 122
5. **保护现场**，拍照取证
6. **配合调查**

### 7.2 快速处理（无伤亡）

**可以自行协商的情况：**
- 仅财产损失
- 责任明确
- 双方对事实无争议

**处理方式：**
- 拍照后移至不妨碍交通的地方
- 填写《交通事故快速处理协议书》
- 向保险公司报案

### 7.3 必须报警的情况

- 有人员伤亡
- 驾驶人无证、酒驾、毒驾
- 车辆无号牌、假牌
- 碰撞建筑物、公共设施
- 一方逃逸

### 7.4 责任认定

| 责任类型 | 含义 |
|----------|------|
| 全部责任 | 一方过错导致 |
| 主要责任 | 一方过错较大（70%） |
| 同等责任 | 双方过错相当（50%） |
| 次要责任 | 一方过错较小（30%） |
| 无责任 | 无过错 |

### 7.5 逃逸后果

- 不构成犯罪：罚款 200-2000，拘留 15 日以下，扣 12 分
- 构成犯罪：追究刑事责任，吊销驾照，终生禁驾

## 8. 安全驾驶常识

### 8.1 出车前检查

- ✅ 轮胎（气压、磨损）
- ✅ 灯光（远近光、转向、刹车）
- ✅ 制动系统
- ✅ 转向系统
- ✅ 燃油、机油、冷却液
- ✅ 后视镜调整

### 8.2 安全车距

| 车速 | 最小安全车距 |
|------|--------------|
| 60 km/h | 60m |
| 80 km/h | 80m |
| 100 km/h | 100m |
| 120 km/h | 150m |

**简易判断：** 3 秒法则（前车经过某点，3 秒后你才到达）

### 8.3 灯光使用

| 情况 | 灯光 |
|------|------|
| 夜间路灯良好 | 近光灯 |
| 夜间无路灯 | 远光灯（会车改近光） |
| 雾天 | 雾灯 + 危险报警闪光灯 |
| 超车 | 交替使用远近光灯 |
| 进入隧道 | 近光灯 |
| 临时停车 | 危险报警闪光灯 |
| 牵引故障车 | 危险报警闪光灯 |

### 8.4 会车规定

- 夜间会车：**150m 外**改用近光灯
- 窄路会车：减速靠右
- 有障碍会车：无障碍方先行

### 8.5 停车规定

**禁止停车地点：**
- ❌ 人行横道
- ❌ 网状线区域
- ❌ 施工路段
- ❌ 公共汽车站、急救站、加油站（30m 内）
- ❌ 消防栓、消防队门前（30m 内）
- ❌ 交叉路口、铁路道口、急弯、陡坡、隧道（50m 内）

### 8.6 酒驾标准

| 类型 | 血液酒精含量 | 处罚 |
|------|--------------|------|
| 饮酒驾驶 | 20-80mg/100ml | 暂扣 6 个月，罚款 1000-2000，扣 12 分 |
| 醉酒驾驶 | ≥80mg/100ml | 追究刑事责任，吊销驾照，5 年禁驾 |

### 8.7 疲劳驾驶

- 连续驾驶不超过 **4 小时**
- 停车休息不少于 **20 分钟**
- 24 小时内累计驾驶不超过 **8 小时**（营运车辆）

## 9. 特殊道路驾驶

### 9.1 高速公路

**进入高速：**
- 匝道内加速
- 开启左转向灯
- 观察后视镜
- 确认安全后并入

**行驶中：**
- 保持安全车距
- 不随意变道
- 不在车道内停车
- 错过出口继续前行，下一出口驶出

**恶劣天气：**

| 能见度 | 车速限制 | 车距 | 灯光 |
|--------|----------|------|------|
| <200m | ≤60km/h | ≥100m | 雾灯 + 近光 + 示廓灯 |
| <100m | ≤40km/h | ≥50m | + 危险报警闪光灯 |
| <50m | ≤20km/h | - | 尽快驶离高速 |

### 9.2 山区道路

- 靠右行驶
- 转弯提前减速、鸣笛
- 下坡不空挡滑行
- 会车靠山体一侧先行

### 9.3 夜间驾驶

- 降低车速
- 正确使用灯光
- 注意行人和非机动车
- 避免直视对向车灯

### 9.4 雨天驾驶

- 降低车速
- 加大车距（干燥路面的 2 倍）
- 避免急刹车
- 注意积水路段

### 9.5 雪天驾驶

- 低速慢行
- 缓打方向
- 提前减速
- 安装防滑链（必要时）

### 9.6 隧道驾驶

- 进入前开启近光灯
- 禁止超车
- 禁止停车
- 保持车距

## 10. 科目一/四考试技巧

### 10.1 考试概况

| 科目 | 题型 | 题量 | 时间 | 及格分 |
|------|------|------|------|--------|
| 科目一 | 判断 + 单选 | 100 | 45 分钟 | 90 分 |
| 科目四 | 判断 + 单选 + 多选 | 50 | 45 分钟 | 90 分 |

### 10.2 科目一重点

**必考内容：**
- 交通信号（灯、标志、标线）
- 通行规则
- 违法行为处罚
- 驾驶证管理
- 事故处理

**记忆口诀：**
> "假一吊二撤三醉五逃终生"
> - 假证：1 年不得申领
> - 吊销：2 年不得申领
> - 撤销：3 年不得申领
> - 醉驾：5 年不得申领
> - 逃逸：终生禁驾

### 10.3 科目四重点

**安全文明驾驶：**
- 恶劣天气驾驶
- 紧急情况处置
- 伤员急救
- 危险化学品运输

**多选题技巧：**
- 看清题目要求（选正确的/错误的）
- 安全选项通常全选
- 谨慎选项要分析

### 10.4 紧急情况处置

| 情况 | 处置方法 |
|------|----------|
| 爆胎 | 握紧方向盘，轻踩刹车，缓慢靠边 |
| 刹车失灵 | 降挡减速，用手刹辅助，寻找避险车道 |
| 转向失控 | 立即减速停车 |
| 车辆起火 | 停车、熄火、疏散、灭火 |
| 落水 | 解开安全带，等车内进水后开门或破窗 |
| 高速碰撞 | 紧握方向盘，保护头部 |

### 10.5 伤员急救

- **原则**：先救命后治伤，先重伤后轻伤
- **出血**：加压包扎止血
- **骨折**：固定后搬运
- **昏迷**：侧卧位，保持呼吸道通畅
- **烧伤**：冷水冲洗，不涂抹药膏
