## Description: <br>
混子哥风格知识脚本生成器，把枯燥、复杂的知识转化为生动、有趣、视觉化的知识脱口秀脚本，适合历史事件、典故、制度等有信息差和因果链的主题。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External content creators and agent users use this skill to draft humorous Chinese knowledge-video scripts with a seven-part storyboard, visual guidance, and cover/title suggestions. It is best suited to topics where the user can verify facts and build a clear cause-and-effect narrative. <br>
