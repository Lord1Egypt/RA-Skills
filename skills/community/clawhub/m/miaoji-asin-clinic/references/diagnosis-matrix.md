# ASIN五维体检评分矩阵

本文档详细描述各维度的评分规则和判断标准。

---

## 一、合规度评分矩阵

### 评分维度
| 检测项 | 分值 | 判断标准 |
|-------|-----|---------|
| 禁止词检测 | 0-40分 | 发现禁止词直接扣40分 |
| 边缘词检测 | 0-30分 | 每个边缘词扣5分 |
| Rufus适配性 | 0-30分 | 场景词密度评估 |

### 禁止词清单（示例）
- 涉及虚假宣传的绝对化用语
- 平台明确禁止的功效声明
- 违反广告法的极限词

### 边缘词清单（示例）
- 可能引起投诉的夸大表述
- 需要资质证明的功效词
- 同类产品指向性对比用语

### Rufus适配性评分标准
| 等级 | 分值 | 标准 |
|-----|-----|-----|
| 优秀 | 25-30 | 植入5个以上场景词，自然融入 |
| 良好 | 18-24 | 植入3-4个场景词 |
| 一般 | 10-17 | 植入1-2个场景词 |
| 较差 | 0-9 | 缺乏场景词，纯产品描述 |

### 合规度最终判定
| 健康等级 | 分值范围 | 建议操作 |
|---------|---------|---------|
| 🟢 优秀 | 90-100 | 保持，定期检查 |
| 🟢 良好 | 80-89 | 维持，关注边缘词 |
| 🟡 一般 | 60-79 | 优化边缘词，加强场景词 |
| 🟡 关注 | 50-59 | 必须处理边缘词 |
| 🔴 警告 | 30-49 | 存在禁止词风险 |
| 🔴 危险 | 0-29 | 立即处理禁止词 |

## 综合评分算法

### 五维权重分配
| 维度 | 权重 | 说明 |
|-----|-----|-----|
| 评论度 | 30% | 信任基础，影响转化 |
| 合规度 | 25% | 搜索排名，风险控制 |
| 内容度 | 20% | 购买决策，内容为王 |
| 广告度 | 15% | 流量成本，效率优化 |
| 视觉度 | 10% | 用户体验，视觉冲击 |

### 综合健康指数计算
```
综合得分 = 合规度×0.25 + 广告度×0.15 + 评论度×0.30 + 视觉度×0.10 + 内容度×0.20
```

| 等级 | 分值范围 | 整体建议 |
|-----|---------|---------|
| 🟢 优秀店铺 | 80-100 | 行业标杆，保持领先 |
| 🟢 良好店铺 | 70-79 | 健康状态，细微优化 |
| 🟡 潜力店铺 | 55-69 | 有提升空间，按优先级修复 |
| 🔴 问题店铺 | 35-54 | 需要系统修复 |
| 🔴 危险店铺 | 0-34 | 紧急全面整改 |
