## Description: <br>
Web Search Instant 1.1.0 provides a command-line DuckDuckGo Instant Answer API client for quick facts, definitions, calculations, conversions, summaries, abstracts, and related topics without API credentials. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, operators, and external agent users use this skill to run quick web lookups from a shell workflow when they need instant answers, definitions, conversions, summaries, or related-topic leads. It is best suited for evergreen lookup tasks rather than full web search, recent news, or real-time market and sports data. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Search queries are sent to DuckDuckGo, which can expose sensitive terms included in prompts or command arguments. <br>
Mitigation: Do not submit passwords, API keys, private customer data, confidential project names, or other sensitive material in searches. <br>
Risk: Instant Answer data can be incomplete, stale, or unavailable for some queries. <br>
Mitigation: Verify important results against authoritative sources before using them for decisions, publications, or customer-facing guidance. <br>
Risk: The skill returns instant answers rather than full ranked web results. <br>
Mitigation: Use the provided DuckDuckGo full-search link or another review process when a broader source survey is required. <br>
