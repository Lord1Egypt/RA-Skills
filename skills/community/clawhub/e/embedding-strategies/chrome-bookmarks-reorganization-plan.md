## 🔧 执行步骤

### 步骤 1: 备份当前书签
```powershell
# 备份当前书签文件
Copy-Item "C:\Users\Xiabi\AppData\Local\Google\Chrome\User Data\Default\Bookmarks" `
          "C:\Users\Xiabi\AppData\Local\Google\Chrome\User Data\Default\Bookmarks.backup.$(Get-Date -Format 'yyyyMMdd-HHmmss')"
```

### 步骤 2: 导出书签为 HTML
1. Chrome → 书签 → 书签管理器
2. 点击右上角三个点 → 导出书签
3. 保存为 `bookmarks-export-2026-03-08.html`

### 步骤 3: 手动整理 (推荐)
由于 Chrome 书签 JSON 格式复杂，建议手动整理：

1. **删除传化相关** (约 150 个)
   - 传化文件夹
   - 陆鲸测试
   - 仓配
   - 货嘀测试

2. **重新组织文件夹**
   - 按上述新分类创建文件夹
   - 拖拽书签到新位置

3. **清理重复书签**
   - 秀米 (10+ 重复)
   - 飞书文档 (多个重复)

### 步骤 4: 验证
- 检查常用书签是否都在
- 测试关键链接是否有效
- 确认书签栏布局合理
