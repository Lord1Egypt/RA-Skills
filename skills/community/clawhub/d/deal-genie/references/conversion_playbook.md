# 追单转化手册 (conversion_playbook.md)

## 概述

本文件定义追单转化（Step 11）的完整策略和话术模板。推品不是终点——推品后的二次转化、口碑裂变、差评处理，直接决定了单品的最终利润。覆盖：追单话术、限时活动、晒单裂变、差评应对四大场景。

---

## A. 追单话术（按时间节点）

### 追单节奏

```
推品当天     推品后1天     推品后3天     推品后7天
    ↓            ↓            ↓            ↓
 评论区互动   社群追单     晒单引导    复购唤醒+关联推荐
```

### 节点一：推品当天 — 评论区互动

```
时机：达人推品后2小时内
场景：达人视频/直播评论区
目标：答疑解惑，消除最后购买顾虑，引导下单

话术模板：
"感谢宝子们的关注！统一回复几个问得最多的问题：
1. 这个洗发水是氨基酸配方，pH5.5弱酸性，敏感头皮放心用
2. 孕妇可用，0硅油0硫酸盐
3. 今天下单备注[主播名]额外送旅行装
还有其他问题直接评论区问，我看到就回 🙌"

技巧：
- 用"统一回复"提高效率
- 用数字编号让信息清晰
- 用"备注送赠品"刺激下单
```

### 节点二：推品后1天 — 社群追单

```
时机：推品后24小时
场景：团长群/招商群/粉丝群
目标：提醒还没下单的人，制造"再不买就没了"的紧迫感

话术模板（三种风格）：

风格A：库存紧迫型
"昨天XX推的那个洗发水，品牌方说库存剩不到200件了 🔥
推之前备了500件，没想到一天就出了300+
要囤的趁早，这波卖完下一批要等半个月"
⚠️ 前提：库存确实紧张，不能编造

风格B：好评带动型
"群里买了洗发水的宝子们，你们用得怎么样？评论区聊一波
刚有个姐妹私信我说洗了三次头皮就不痒了，我之前也是这个感受 👏
还没下单的可以看看群里的真实反馈再决定"

风格C：场景拓展型
"昨天推洗发水的时候忘了说：这个送人超合适！
再过两周就是母亲节了，79块买不了吃亏，妈妈肯定喜欢
昨天下了两瓶的姐妹，是不是一瓶自用一瓶送妈的？哈哈"
```

### 节点三：推品后3天 — 晒单引导

```
时机：大部分用户已收到货
场景：社群/朋友圈/评论区
目标：引导晒单，制造社会证明，带动观望用户下单

话术模板：
"收到洗发水的宝子们，晒个单呗！📸
拍照不需要多专业，就你洗完头发的样子就行
带话题 #XX洗发水体验官 发到群里
我抽3个最走心的晒单，送正装一瓶！"

晒单引导技巧：
- 降低门槛："不需要多专业"
- 给模版：发一张示范图，说"像这样就行"
- 给奖励：抽奖/优惠券/免单
- 给话题词：方便二次传播和搜索
```

### 节点四：推品后7天 — 复购唤醒

```
时机：用户使用一周左右
场景：社群/私信
目标：收集使用反馈，做关联推荐，引导复购

话术模板：
"宝子们，洗发水用了一周感觉怎么样？
来说说你的感受，好的坏的都行，帮助我们改进 🙏

对了，跟洗发水同系列的护发素也上线了
搭配使用效果翻倍，洗发水+护发素组合装只要129，比单买省29
要的老规矩，群里扣1"
```

## B. 限时活动文案套件

### 四大活动主题

| 主题 | 适用场景 | 文案核心 |
|------|---------|---------|
| 库存告急型 | 真实库存紧张 | 数量+稀缺动词 |
| 时间紧迫型 | 活动即将结束 | 倒计时+损失厌恶 |
| 专属福利型 | 粉丝/团长专享 | 暗号+专属感 |
| 社交裂变型 | 引导二次传播 | 利益驱动+低门槛 |

### 模板一：库存告急型

```
结构：
[库存状态] + [已售数据] + [补货难度] + [行动号召]

示例：
"洗发水就剩最后37件了，品牌方说下一批要到月底
昨天一个团长自己要了10件，他说按这个价格下次肯定涨价
还没拿货的抓紧，我这边过了今天就不留了 🏃‍♂️"
```

### 模板二：时间紧迫型

```
结构：
[截止时间] + [即将失去的福利] + [行动号召]

示例：
"今晚24点！洗发水买1送1活动结束
过了12点恢复原价79/瓶，现在下单一瓶的钱拿两瓶
手慢真的无了 ⏰"
```

### 模板三：专属福利型

```
结构：
[专属身份] + [专属福利] + [暗号] + [限定时间]

示例：
"仅限本群的福利：下单备注暗号【团长推荐】
额外再减5块，就是74块到手氨基酸洗发水+旅行装
到今晚8点，过时不候 🔑"
```

### 模板四：社交裂变型

```
结构：
[晒单任务] + [奖励内容] + [参与规则] + [话题标签]

示例：
"活动：晒单免单！
规则：发布你的洗发水使用体验（图文/视频都可以）
      带话题 #XX洗发水体验 和 #好物分享
奖励：截止本周日，点赞最高的3条直接退款免单
      另外随机抽10个送旅行装组合
门槛很低，你洗个头的功夫就能参加，冲！ 🎁"
```

## C. 晒单引导策略

### 为什么晒单很重要

- 1条真实晒单 > 10条达人推广广告
- 晒单是免费的二次曝光
- 观望用户看到真实反馈更容易下单

### 晒单引导三步走

```
第一步：示范先行
在你自己的社群里先发一条示范晒单
格式："刚用了XX洗发水，洗完头发这样 [图片]，头皮真的不痒了"
配图：洗完头发的真实状态（不需要美颜，越真实越好）

第二步：低门槛邀请
"你们也来试试，不要求拍得多好看，就洗完头拍一张就行
我看看谁的头发最顺，最顺的送一瓶正装"

第三步：奖励兑现
晒单活动结束后，及时公布中奖名单
把中奖者的晒单截图再发一次，刺激下一波
```

### 晒单引导话术变体

```
变体A：幽默型
"姐妹们，考验你们自拍技术的时刻到了！
拍一张你洗完头发的照片发群里
我先来打个样 [附图] ——比这个好看就赢了我的人情，比这个丑就赢了我的奖品
怎么样都不亏吧？"

变体B：求助型
"宝子们帮个忙！
品牌方问我要一些真实用户反馈，我手上只有自己的照片不够
你们用过的晒一张，我拿去向品牌方申请更大福利
你们帮忙，我帮你们争取——双赢！"

变体C：PK型
"洗发水效果PK赛！
左边是你用之前的头发，右边是用一周之后的头发
发对比图到群里，变化最明显的送正装+护发素组合
给大家一周时间，下周日截止"
```

## D. 差评应对模板

### 差评分类与应对策略

| 差评类型 | 核心原则 | 错误做法 |
|---------|---------|---------|
| 品质类 | 承认+补偿+改进 | 推卸责任、删除差评 |
| 物流类 | 道歉+追踪+安抚 | 说"这是快递公司的问题" |
| 期望类 | 感谢反馈+解释+赠品 | 说"是你不懂/不会用" |
| 恶意差评 | 冷静回应+事实展示+平台申诉 | 情绪化对喷 |

### 模板一：品质类差评

```
场景：用户反馈"用了过敏/效果不好/跟描述不符"

错误回复：
"不可能啊，别人用了都没事" ← 推卸责任

正确回复：
"非常抱歉给你带来了不好的体验 [承认]
你方便跟我说一下具体是什么情况吗？过敏的话可能是某个成分不耐受 [了解详情]
我给你安排退款 + 寄一罐舒缓修复霜，算我们的歉意 [补偿]
你的反馈我们会同步给产品团队，后续配方会做优化 [改进]"

关键：先道歉，再解决，最后给信心
```

### 模板二：物流类差评

```
场景：用户抱怨"物流太慢了/包装破了/发错货了"

错误回复：
"这个是快递公司的问题，你找快递" ← 甩锅

正确回复：
"实在抱歉！我马上查一下你的快递状态 [行动]
物流这次确实慢了，我这边马上催一下站点 [追踪]
为表歉意，下次下单给你免运费 [安抚]
感谢你的耐心等待，有任何问题随时找我 🙏"

关键：别甩锅给快递，你就是责任主体
```

### 模板三：期望类差评

```
场景：用户说"没有视频里说的那么神奇/跟超市的差不多"

错误回复：
"是你不会用/是你期望太高了" ← 否定用户

正确回复：
"感谢你的真实反馈，这恰恰是我们需要的 🙏 [感谢]
洗发水效果因人而异，而且需要坚持使用才能看到明显变化 [解释]
这样，我送你一支护发素搭配使用，效果会明显很多 [赠品]
如果有任何疑问，欢迎随时找我聊聊使用心得"

关键：不要否定用户感受，要给解决方案
```

### 模板四：恶意差评

```
场景：竞品水军/无理由差评/勒索式差评

策略：
1. 冷静不回怼
2. 公开回复事实
3. 平台申诉

公开回复模板：
"感谢您的反馈。关于您提到的问题：
1. 我们的产品有XX检测报告，成分安全合规
2. 如果确实遇到品质问题，请提供购买凭证，我们负责到底
3. 对于不实的信息，我们将保留平台申诉的权利
	
我们始终欢迎真实的用户反馈，但对恶意攻击零容忍。"

注意：
- 不在评论区展开争论（越争越难看）
- 用事实和数据说话，不用情绪说话
- 私下联系平台客服处理
```
