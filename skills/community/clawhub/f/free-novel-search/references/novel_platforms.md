# Free Novel Platform Database

## Public Domain & Open Access

### Project Gutenberg
- **URL**: https://www.gutenberg.org
- **Type**: Public Domain
- **Format**: ePub, Kindle, HTML, Plain Text
- **Content**: 70,000+ classic works
- **License**: Public domain
- **Languages**: 60+
- **Notable**: First digitization project (1971), no DRM
- **API**: https://gutendex.com

### Open Library
- **URL**: https://openlibrary.org
- **Type**: Open Access + Borrowing
- **Format**: Web reader, ePub
- **Content**: 20M+ books, 1M+ free to read
- **License**: Mix of public domain and lending
- **Features**: Author profiles, book loans, book scans

### ManyBooks
- **URL**: https://manybooks.net
- **Type**: Public Domain
- **Format**: ePub, PDF, MOBI, HTML
- **Content**: 50,000+ free eBooks
- **License**: Public domain
- **Categories**: Genre-organized

### Feedbooks
- **URL**: https://www.feedbooks.com
- **Type**: Mixed (Public + Original)
- **Format**: ePub, MOBI
- **Content**: Original fiction + public domain
- **License**: Check individual titles

### Wikisource
- **URL**: https://en.wikisource.org
- **Type**: User-Contributed Public Domain
- **Format**: Web
- **Content**: 700,000+ works
- **License**: Public domain contributions

## Original Fiction Platforms

### Wattpad
- **URL**: https://www.wattpad.com
- **Type**: User-Generated (Ad-supported free)
- **Format**: Web, Mobile app
- **Content**: 400M+ story uploads
- **Free Model**: Ad-supported chapters
- **Premium**: Paid Stories subscription
- **Notable**: Discover new authors, serialized fiction

### Royal Road
- **URL**: https://www.royalroad.com
- **Type**: Original Web Fiction
- **Format**: Web
- **Content**: Fantasy, Sci-Fi, LitRPG
- **Free Model**: Ad-supported
- **Notable**: Serious original fiction community

### Kindle Vella (Amazon)
- **URL**: https://kindlevella.com
- **Type**: Original Serialized
- **Format**: Kindle app
- **Content**: Original stories by US authors
- **Free Model**: First 3 episodes free, tokens to unlock
- **Note**: US-only access

### Kobo Writing Life
- **URL**: https://www.kobowritinglife.com
- **Type**: Self-published
- **Format**: ePub
- **Content**: Self-published works
- **Free Model**: Many publishers offer free titles

## aggregator Sites

### Internet Archive
- **URL**: https://archive.org
- **Type**: Digital Library
- **Format**: Multiple
- **Content**: Books, audio, video
- **Free Model**: Free accounts, borrowing system

### LibriVox
- **URL**: https://librivox.org
- **Type**: Audio Books (Public Domain)
- **Format**: Audio (MP3)
- **Content**: 20,000+ public domain audiobooks
- **License**: Public domain recordings
