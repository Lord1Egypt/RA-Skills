This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agents use this skill to generate speech audio from text with selected Voice.ai personas, languages, streaming mode, and configurable output files. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The required Voice.ai API key may permit actions beyond speech generation. <br>
Mitigation: Use a least-privilege key when available, store it only in the VOICE_AI_API_KEY environment variable, and rotate it if exposed. <br>
Risk: The bundled SDK includes voice-management functions that can modify or permanently delete Voice.ai voices. <br>
Mitigation: Review any requested voice-management action before execution and restrict routine use to documented text-to-speech commands. <br>
Risk: Generated audio is written to a user-selected local output path. <br>
Mitigation: Confirm the output path before running the skill to avoid overwriting important files. <br>
Risk: Submitted text is sent to the Voice.ai API for synthesis. <br>
Mitigation: Do not submit secrets, regulated data, or other sensitive text for synthesis. <br>
