# competitor-monitor

竞品研究 skill for OpenClaw/SkillBoss.

## 功能

- **监控模式**：查看竞品最近 X 天的动态
- **分析模式**：四维深度分析（产品功能、定价、用户、社媒）
- **多竞品对比**：横向对比多家公司

## 使用方法

```bash
# 安装
openclaw skill install competitor-monitor

# 或手动复制到 skills 目录
```

## 触发词

### 监控模式
- "帮我看看 [公司] 最近在干嘛"
- "[公司] 过去 X 天做了什么"
- "监控一下 [公司]"

### 分析模式
- "分析一下 [公司]"
- "做 [公司] 的竞品分析"
- "研究一下 [公司]"

## 输出

生成单文件 HTML 报告，可直接用浏览器打开、分享、汇报。

## 依赖

- SkillBoss API (可选)
- 网络访问能力
