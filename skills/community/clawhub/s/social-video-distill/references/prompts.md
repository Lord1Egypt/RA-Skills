# Prompt patterns

Use short prompts. Tell the model exactly what shape you want back.

## 1. Quick summary

```text
Summarize this clip in 3 bullets:
1. what is being said
2. what is happening socially/emotionally
3. the strongest line

Transcript:
...
```

## 2. Humor distillation

```text
Distill this short clip into:
1. one-sentence joke premise
2. why it's funny in 3 bullets
3. best punchline
Keep it short.

Transcript:
...
```

## 3. Caption ideas

```text
Based on this clip, write 3 short caption options:
- one dry
- one playful
- one savage
Do not explain them.

Transcript:
...
```

## 4. Separate evidence from inference

```text
Using the transcript below, output:
- directly supported by transcript
- likely inferred context
- any ambiguity
Keep it tight.

Transcript:
...
```

## 5. NotebookLM handoff note

```text
Use these transcripts/notes as source material. Build a short synthesis covering:
- recurring themes
- strongest moments
- useful quotes
- open questions
```
