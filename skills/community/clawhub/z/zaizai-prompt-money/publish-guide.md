# 多平台发布指南

## 闲鱼平台

### 发布规范
- **标题**：痛点+效果+数字，如"小红书爆款标题生成器|3秒产出10个爆款"
- **图片**：3-5张，含效果对比图+使用说明
- **定价**：单条9.9-29.9元，模板包49.9-99.9元
- **描述结构**：
  1. 痛点共鸣（用户会遇到什么问题）
  2. 解决方案（提示词能做什么）
  3. 效果展示（真实案例截图）
  4. 使用说明（如何购买和使用）
  5. FAQ（常见问题）

### 注意事项
- 服务类商品必须手机端发布
- 及时回复咨询
- 好评后引导私域

## 小红书平台

### 发布规范
- **标题**：悬念/数字/情绪，如"这个提示词让我效率提升10倍"
- **封面**：3:4比例，高清有吸引力
- **正文结构**：
  1. 开头钩子（吸引点击）
  2. 问题导入（引发共鸣）
  3. 解决方案（介绍提示词）
  4. 使用效果（真实案例）
  5. 引导私域（获取方式）

### 标签策略
- #AI提示词 #ChatGPT #效率工具 #AI副业
- #小红书运营 #文案神器 #副业赚钱

### 注意事项
- 图片必须一张一张上传
- 避免直接提及"购买"
- 引导至私域或闲鱼

## 知识星球

### 发布规范
- **星球定位**：AI工具技巧分享
- **内容结构**：
  1. 入门指南（星球介绍）
  2. 提示词合集（持续更新）
  3. 案例库（使用效果）
  4. 互动问答

### 定价策略
- 年费制：99-299元/年
- 季费制：39-99元/季
- 单买提示词包：49-99元
