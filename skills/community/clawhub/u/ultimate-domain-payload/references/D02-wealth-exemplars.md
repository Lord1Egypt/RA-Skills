## 范本1：工薪阶层财富四阶段规划

### 题目
30岁工薪阶层的财富四阶段规划

### 背景
陈先生，30岁，互联网公司工程师，年薪30万，单身，无房无车，存款20万。希望进行系统的财富规划，实现财务自由。

### Step 1: 财务诊断阶段

#### 1.1 资产负债盘点 (W1-01)
**输入**: 资产清单，负债清单
**执行**:
1. 列出所有资产：
   - 现金及存款：20万
   - 投资：无
   - 房产：无
   - 车辆：无
2. 评估资产价值：现金20万
3. 列出所有负债：无
4. 计算负债总额：0
5. 计算净资产：20万

**输出**: 资产负债表

#### 1.2 收支分析 (W1-02)
**输入**: 收入来源，支出项目
**执行**:
1. 记录所有收入来源：
   - 工资：2.5万/月
   - 奖金：约5万/年
   - 其他：无
2. 分类记录支出：
   - 固定支出：房租5000，生活费3000
   - 可变支出：娱乐2000，交通1000
   - 意外支出：约1000/月
3. 计算收支差额：月结余约1.3万
4. 分析收支结构：支出占比56%，结余率44%
5. 识别优化空间：可优化娱乐支出

**输出**: 收支分析表

#### 1.3 财务健康评分 (W1-04)
**输入**: 资产负债，收支分析
**执行**:
1. 计算各项指标：
   - 流动性比率：应急基金/月支出=20万/1.2万=16.7个月(优秀)
   - 负债率：负债/资产=0%(优秀)
   - 储蓄率：结余/收入=44%(优秀)
   - 投资比率：投资/资产=0%(需改善)
2. 评分：流动性9分，负债10分，储蓄9分，投资2分
3. 加权综合评分：7.5分(良好)
4. 识别薄弱环节：投资比率过低
5. 给出改进建议：开始投资理财

**输出**: 财务健康报告

#### 1.4 风险承受能力评估 (W1-06)
**输入**: 年龄，收入稳定性，投资经验，投资期限
**执行**:
1. 评估客观风险能力：
   - 年龄30岁，风险承受能力高
   - 收入稳定，风险承受能力高
   - 投资期限长(30年)，风险承受能力高
2. 评估主观风险态度：
   - 无投资经验，风险态度中性
   - 心理承受能力中等
3. 综合风险承受等级：中高风险(R4)
4. 确定资产配置建议：股票70%，债券20%，现金10%
5. 定期重新评估：每年评估一次

**输出**: 风险评估报告

**校准点1**: 财务状况确认
- 确认净资产20万
- 确认月结余1.3万
- 确认风险承受等级R4

### Step 2: 收入管理阶段

#### 2.1 主动收入优化 (W2-01)
**输入**: 当前收入水平，行业薪资水平
**执行**:
1. 分析当前收入：年薪30万，处于行业中上水平
2. 对比市场水平：同岗位薪资范围25-40万
3. 识别提升空间：可通过晋升或跳槽提升
4. 制定提升策略：
   - 短期：提升技术能力，争取晋升
   - 中期：跳槽到更高薪资公司
   - 长期：转型管理或技术专家
5. 执行行动计划：参加培训，提升技能

**输出**: 收入优化方案

#### 2.2 被动收入开发 (W2-02)
**输入**: 可投资资产，被动收入类型
**执行**:
1. 评估可用资源：20万可投资资产
2. 选择被动收入类型：
   - 利息收入：货币基金
   - 股息收入：高股息股票
   - 租金收入：暂无条件
3. 制定开发计划：
   - 10万投资货币基金，年化2%
   - 10万投资高股息股票，年化4%
4. 执行投资：开设证券账户，购买基金和股票
5. 监测收益：每月查看收益

**输出**: 被动收入计划

#### 2.3 收入多元化策略 (W2-05)
**输入**: 当前收入结构，可开发收入来源
**执行**:
1. 分析当前收入结构：100%来自主动收入
2. 识别多元化机会：
   - 投资收入：利息、股息
   - 副业收入：技术咨询、在线课程
3. 评估可行性：投资收入可行，副业收入需时间投入
4. 制定多元化策略：
   - 短期：开发投资收入
   - 中期：尝试副业收入
   - 长期：建立多元收入体系
5. 逐步实施：先投资，后尝试副业

**输出**: 收入多元化方案

**校准点2**: 收入管理确认
- 确认收入优化策略
- 确认被动收入计划
- 确认多元化路径

### Step 3: 投资管理阶段（财富增值）

#### 3.1 投资目标设定 (W4-01)
**输入**: 财务目标，风险承受能力，投资期限
**执行**:
1. 明确财务目标：
   - 短期(1-3年)：积累购房首付50万
   - 中期(3-10年)：资产增值到200万
   - 长期(10年以上)：实现财务自由(资产1000万)
2. 评估风险承受能力：中高风险(R4)
3. 确定投资期限：长期投资(30年)
4. 设定收益目标：年化8-10%
5. 制定投资策略：股债混合，定期定额

**输出**: 投资目标说明书

#### 3.2 资产配置方案 (W4-02)
**输入**: 投资目标，风险等级
**执行**:
1. 确定资产类别：
   - 股票类：70%(国内股票50%，海外股票20%)
   - 债券类：20%(国内债券)
   - 现金类：10%(货币基金)
2. 设定配置比例：如上
3. 细化地域/行业配置：
   - 国内股票：沪深300指数基金40%，中证500指数基金10%
   - 海外股票：标普500指数基金15%，纳斯达克100指数基金5%
4. 考虑相关性：国内外股票相关性低，分散风险
5. 制定再平衡规则：每年再平衡一次

**输出**: 资产配置方案

#### 3.3 基金筛选与组合 (W4-04)
**输入**: 基金类型，筛选标准
**执行**:
1. 确定基金类型：指数基金为主
2. 设定筛选标准：
   - 费用率：<0.5%
   - 规模：>10亿
   - 跟踪误差：<2%
3. 初步筛选：选择各指数头部基金
4. 深入分析：比较历史业绩、基金经理、公司实力
5. 构建组合：
   - 沪深300指数基金：40%
   - 中证500指数基金：10%
   - 标普500指数基金：15%
   - 纳斯达克100指数基金：5%
   - 国内债券基金：20%
   - 货币基金：10%

**输出**: 基金组合方案

#### 3.4 投资组合再平衡 (W4-07)
**输入**: 当前组合，目标配置，再平衡规则
**执行**:
1. 评估当前组合：初始配置
2. 对比目标配置：一致
3. 确定调整需求：暂无
4. 执行再平衡：定期定额投资
5. 记录调整：记录每次投资

**输出**: 再平衡方案

**校准点3**: 投资方案确认
- 确认投资目标合理
- 确认资产配置适当
- 确认基金选择合理

### Step 4: 保险保障阶段（风险防护）

#### 4.1 保障需求分析 (W6-01)
**输入**: 家庭责任，收入水平，负债情况
**执行**:
1. 评估家庭责任：目前单身，父母赡养责任
2. 计算保障缺口：
   - 寿险：需覆盖父母赡养费用(约100万)
   - 重疾险：需覆盖治疗费用+收入损失(约100万)
   - 医疗险：需覆盖高额医疗费用(约200万)
   - 意外险：需覆盖意外风险(约100万)
3. 确定保障类型：定期寿险+重疾险+医疗险+意外险
4. 设定保额：如上
5. 制定配置计划：优先配置重疾险和医疗险

**输出**: 保障需求分析

#### 4.2 保险配置 (W6-02, W6-03, W6-04, W6-05)
**输入**: 保障需求，保费预算
**执行**:
1. 寿险规划：
   - 类型：定期寿险
   - 保额：100万
   - 期限：至60岁
   - 保费：约2000元/年
2. 重疾险选择：
   - 类型：消费型重疾险
   - 保额：50万
   - 期限：至70岁
   - 保费：约5000元/年
3. 医疗险配置：
   - 类型：百万医疗险
   - 保额：200万
   - 免赔额：1万
   - 保费：约1000元/年
4. 意外险规划：
   - 类型：综合意外险
   - 保额：100万
   - 保费：约500元/年
5. 总保费：约8500元/年，占收入2.8%

**输出**: 保险配置方案

**校准点4**: 保险保障确认
- 确认保障需求覆盖
- 确认保费预算合理
- 确认产品选择适当

### Step 5: 财富传承阶段（长期规划）

#### 5.1 传承目标设定 (W7-01)
**输入**: 传承资产，传承对象
**执行**:
1. 盘点可传承资产：目前净资产20万，未来增值潜力大
2. 确定传承对象：未来子女，父母
3. 设定传承目标：
   - 短期：为父母准备应急资金
   - 中期：为子女准备教育金
   - 长期：实现财富代际传递
4. 选择传承方式：保险+投资+遗嘱
5. 制定传承计划：逐步完善

**输出**: 传承目标说明书

#### 5.2 税务筹划 (W5-01, W5-02, W5-03)
**输入**: 收入信息，扣除信息
**执行**:
1. 应税收入分析：
   - 工资收入：30万/年
   - 投资收入：暂不考虑(长期持有)
   - 应纳税所得额：30万-6万(起征点)-专项扣除(约5万)-专项附加扣除(约3万)=16万
2. 扣除项梳理：
   - 专项扣除：五险一金约5万
   - 专项附加扣除：继续教育、住房贷款等约3万
3. 税收优惠利用：
   - 个人养老金：每年1.2万税前扣除
   - 商业健康保险：每年2400元税前扣除
4. 税务递延策略：
   - 个人养老金：投资递延纳税
   - 保险：部分收益免税
5. 年度税务申报：按时申报，享受优惠

**输出**: 税务筹划方案

**校准点5**: 财富传承确认
- 确认传承目标合理
- 确认税务筹划合法
- 确认长期规划可行

## 范本2：创业者的财务规划

### 题目
35岁创业者的财务规划

### 背景
李女士，35岁，创办了一家电商公司，年营收500万，净利润100万。已婚，有一个5岁孩子，家庭资产包括公司股权、一套房产(市值300万，贷款200万)、存款50万。收入不稳定，需要系统的财务规划。

### Step 1: 财务诊断阶段

#### 1.1 资产负债盘点 (W1-01)
**输入**: 资产清单，负债清单
**执行**:
1. 列出所有资产：
   - 公司股权：估值500万(按净利润5倍)
   - 房产：市值300万
   - 存款：50万
   - 其他：车辆20万
2. 评估资产价值：总资产870万
3. 列出所有负债：
   - 房贷：200万
   - 其他：无
4. 计算负债总额：200万
5. 计算净资产：670万

**输出**: 资产负债表

#### 1.2 收支分析 (W1-02)
**输入**: 收入来源，支出项目
**执行**:
1. 记录所有收入来源：
   - 公司分红：约80万/年
   - 工资：约20万/年
   - 其他：投资收益约5万/年
2. 分类记录支出：
   - 固定支出：房贷1.2万/月，生活费2万/月
   - 可变支出：教育1万/月，娱乐0.5万/月
   - 意外支出：约0.5万/月
3. 计算收支差额：年结余约40万
4. 分析收支结构：收入来源多元，但公司分红不稳定
5. 识别优化空间：可优化支出结构，增加储蓄

**输出**: 收支分析表

#### 1.3 财务健康评分 (W1-04)
**输入**: 资产负债，收支分析
**执行**:
1. 计算各项指标：
   - 流动性比率：50万/5万=10个月(优秀)
   - 负债率：200万/870万=23%(良好)
   - 储蓄率：40万/105万=38%(良好)
   - 投资比率：投资资产/总资产=5万/870万=0.6%(需改善)
2. 评分：流动性9分，负债8分，储蓄8分，投资2分
3. 加权综合评分：6.8分(中等)
4. 识别薄弱环节：投资比率过低，收入不稳定
5. 给出改进建议：建立应急基金，增加投资

**输出**: 财务健康报告

#### 1.4 风险承受能力评估 (W1-06)
**输入**: 年龄，收入稳定性，投资经验，投资期限
**执行**:
1. 评估客观风险能力：
   - 年龄35岁，风险承受能力中高
   - 收入不稳定，风险承受能力中等
   - 投资期限长(25年)，风险承受能力中高
2. 评估主观风险态度：
   - 创业者，风险态度积极
   - 心理承受能力高
3. 综合风险承受等级：中高风险(R4)
4. 确定资产配置建议：股票60%，债券25%，现金15%
5. 定期重新评估：每年评估一次

**输出**: 风险评估报告

**校准点1**: 财务状况确认
- 确认净资产670万
- 确认年结余40万
- 确认风险承受等级R4

### Step 2: 收入管理阶段

#### 2.1 收入多元化策略 (W2-05)
**输入**: 当前收入结构，可开发收入来源
**执行**:
1. 分析当前收入结构：
   - 公司分红：76%
   - 工资：19%
   - 投资收益：5%
2. 识别多元化机会：
   - 投资收入：股票、基金、房产
   - 被动收入：版税、租金
   - 副业收入：咨询、投资
3. 评估可行性：投资收入可行，副业需时间
4. 制定多元化策略：
   - 短期：增加投资收入
   - 中期：开发被动收入
   - 长期：建立多元收入体系
5. 逐步实施：先投资，后开发被动收入

**输出**: 收入多元化方案

#### 2.2 收入预测建模 (W2-06)
**输入**: 历史收入数据，增长假设
**执行**:
1. 收集历史数据：近3年收入数据
2. 确定增长假设：
   - 公司分红：年增长10-20%
