# 急性胰腺炎病情严重程度三级分类

## 适用条件
- 患者已确诊为急性胰腺炎
- 不适用于慢性胰腺炎或其他胰腺疾病

## 执行步骤

1. **评估是否存在器官功能衰竭**
   - 使用改良Marshall评分系统评估呼吸、心血管、肾脏等系统功能
   - 单个系统评分 ≥2 分即判定为存在器官功能衰竭

2. **判断器官功能衰竭持续时间**
   - 若器官功能衰竭在48小时内恢复 → 一过性
   - 若持续超过48小时且不能自行恢复 → 持续性

3. **检查是否存在局部或全身并发症**
   - 局部并发症：如胰腺假性囊肿、坏死积聚等
   - 全身并发症：如全身炎症反应综合征（SIRS）、脓毒症等

4. **根据以下标准进行分级**
   - **轻症急性胰腺炎（MAP）**：无器官功能衰竭，无局部或全身并发症
   - **中重症急性胰腺炎（MSAP）**：伴有一过性器官功能衰竭（≤48小时）和/或存在并发症
   - **重症急性胰腺炎（SAP）**：伴有持续性器官功能衰竭（>48小时）

5. **输出结果**
   - 返回分级结果：MAP、MSAP 或 SAP
   - 用于指导后续治疗强度、监护级别及预后评估

> 注意：若患者出现Grey-Turner征（腰肋部瘀斑）或Cullen征（脐周瘀斑）、广泛腹膜炎、休克或多脏器功能障碍，应高度怀疑SAP。
