# 二、五行生克关系

## 相生

木生火、火生土、土生金、金生水、水生木

## 相克

木克土、土克水、水克火、火克金、金克木

## 生克强度

- 天干生克力量 > 地支生克力量
- 相邻干支生克力量 > 隔位干支
- 月令当令的五行力量最强

# 三、身强身弱判断

## 判断方法

1. 看月令：日主在月令的状态（长生、沐浴、冠带、临官、帝旺为得令；死、墓、绝、胎、养为失令）
2. 看地支：日主在地支是否有根（同类五行）
3. 看天干：日主在天干是否有生扶（印星/比劫）
4. 综合加权计算（见 `analysis-guide.md` 第二章）

## 身强身弱的意义

- 身强：日主力量足，喜克泄（官杀/食伤/财星），忌生扶（印星/比劫）
- 身弱：日主力量不足，喜生扶（印星/比劫），忌克泄（官杀/食伤/财星）
- 从格：日主极弱无根，从旺势走，越旺越好

## 常见陷阱

不可机械套用「生我就是喜用」——辛金忌土就是经典反例（印多反而埋金）。详见「十日主喜忌」章节。

# 四、十神详解

## 十神定义（以日干为我）

| 十神 | 生克关系 | 阴阳 | 含义 |
|------|----------|------|------|
| 比肩 | 同我 | 同阴阳 | 兄弟朋友、竞争、独立 |
| 劫财 | 同我 | 异阴阳 | 合作、争夺、冲动 |
| 食神 | 我生 | 同阴阳 | 才华、享受、温和 |
| 伤官 | 我生 | 异阴阳 | 才华、叛逆、个性 |
| 偏财 | 我克 | 同阴阳 | 大财、父亲（男命）、投机 |
| 正财 | 我克 | 异阴阳 | 稳定财、妻子（男命） |
| 七杀 | 克我 | 同阴阳 | 压力、魄力、野心 |
| 正官 | 克我 | 异阴阳 | 约束、正直、丈夫（女命） |
| 偏印（枭神） | 生我 | 同阴阳 | 聪明、多疑、母亲（女命） |
| 正印 | 生我 | 异阴阳 | 学识、保护、母亲（男命） |

## 六亲十神对应

### 父亲
- 男命：偏财为父（同性我克）
- 女命：正财为父（异性我克）

### 母亲
- 男命：正印为母（异性生我）
- 女命：偏印为母（同性生我，又叫枭神）

### 配偶
- 男命：正财为妻（异性我克）
- 女命：正官为夫（异性克我）

### 子女
- 男命：七杀为子（同性克我），正官为女（异性克我）
- 女命：伤官为子（异性我生），食神为女（同性我生）

### 兄弟姐妹（男女互换）
- 男命：比肩=兄弟，劫财=姐妹
- 女命：比肩=姐妹，劫财=兄弟

### 祖辈（辅助参考）
- 男命：偏印为祖父，正财为祖母
- 女命：食神为祖父，偏财为祖母

### 原理

男命属阳，取「同性」星代表同性亲属、「异性」星代表异性亲属；女命属阴，反之。核心逻辑是「同阴阳为偏，异阴阳为正」。

### 常见错误

1. 不分男女一律用「偏财为父、正印为母」——这是错的
2. 不分男女一律用「正财为妻」——女命没有「妻」，女命配偶星是正官
3. 不分男女一律用「官杀为子女」——女命子女星是食伤，不是官杀
4. 必须在分析开头明确标注命主性别对应的所有六亲十神，后续全文保持一致

### 子女星特殊情况

- 男命以官杀为子女（克我者），因为古代「子女克父母」即子女让父母操心
- 女命以食伤为子女（我生者），因为母亲生育子女
- 如果命局中子女星不现（不见天干也不藏地支），说明子女缘薄或来得晚
- 子女星被克（男命食伤克官杀、女命印星克食伤）→ 生育不顺或子女身体弱

### 实战示例（以甲木日主为例）

#### 甲木男命
- 父 = 戊土（偏财）
- 母 = 癸水（正印）
- 妻 = 己土（正财）
- 子 = 庚金（七杀），女 = 辛金（正官）
- 兄弟 = 甲木（比肩），姐妹 = 乙木（劫财）

#### 甲木女命
- 父 = 己土（正财）
- 母 = 壬水（偏印）
- 夫 = 辛金（正官）
- 子 = 丁火（伤官），女 = 丙火（食神）
- 姐妹 = 甲木（比肩），兄弟 = 乙木（劫财）

#### 丁火男命
- 父 = 辛金（偏财）
- 母 = 甲木（正印）
- 妻 = 庚金（正财）
- 子 = 壬水（七杀），女 = 癸水（正官）

#### 丁火女命
- 父 = 庚金（正财）
- 母 = 乙木（偏印）
- 夫 = 壬水（正官）
- 子 = 戊土（伤官），女 = 己土（食神）

## 十神组合

### 食神制杀
- 食神温和管人，化解七杀压力
- 适合管理、教育、咨询等以柔克刚的工作
- 工作形态不局限于大机构，可能出现在个体户、自由职业等多种场景

### 伤官见官
- 伤官克正官，叛逆与约束冲突
- 容易与领导/权威冲突，但也可能带来创新突破
- 需要其他五行通关或制化

### 财星破印
- 财星克印星，物质追求与学识追求冲突
- 可能因赚钱放弃学业，或因学业影响收入
- 需要平衡取舍

### 印星护身
- 印星生身，保护日主
- 适合学术、研究、教育等需要积累的工作
- 印星太旺可能懒散、依赖性强

# 五、刑冲合害破关系

## 天干五合

甲己合（化土）、乙庚合（化金）、丙辛合（化水）、丁壬合（化木）、戊癸合（化火）

- 合 = 牵绊、合作、稳定
- 天干五合优先级最高
- 合能解冲：如果两个天干之间有合，第三方来冲时可能合住不解

## 地支六合

子丑合（化土）、寅亥合（化木）、卯戌合（化火）、辰酉合（化金）、巳申合（化水）、午未合（化土）

- 六合 = 合作、和谐、稳定
- 优先级仅次于天干五合

## 地支三合

申子辰合水局、亥卯未合木局、寅午戌合火局、巳酉丑合金局

- 三合 = 强大的五行能量汇聚
- 半合（三字中只有两字）= 能量减半
- 三合局力量 > 六合

## 地支三会

寅卯辰会木局、巳午未会火局、申酉戌会金局、亥子丑会水局

- 三会 = 最强的五行能量汇聚（比三合更强）
- 必须三字齐全

## 地支六冲

子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲

- 冲 = 冲突、变动、分离
- 冲的力量大于刑、害
- 月柱冲日主 = 父母宫与子女缘分冲突（离婚/分离信号）

## 地支相刑

- 寅巳申三刑（无恩之刑）
- 丑戌未三刑（恃势之刑）
- 子卯刑（无礼之刑）
- 辰辰、午午、酉酉、亥亥自刑

- 刑 = 内在折磨、反复、纠结
- 三刑必须三字齐全才成立
- 自刑 = 自己折磨自己

## 地支相害

子未害、丑午害、寅巳害、卯辰害、申亥害、酉戌害

- 害 = 暗中损害、小人、背后使绊子
- 力量小于冲、刑

## 地支相破

子酉破、丑辰破、寅亥破、卯午破、巳申破、未戌破

- 破 = 破坏、损耗
- 力量最小，常被忽略

## 关系优先级

天干五合 > 地支六合 > 地支三合 > 地支三会 > 地支六冲 > 地支相刑 > 地支相害 > 地支相破

## 实战应用

- 冲克原命局 → 变动/冲突年
- 合原命局 → 稳定/牵绊年
- 刑原命局 → 内在折磨、反复
- 害原命局 → 小人、暗中损害

# 七、格局判断

## 常用格局

### 正官格
- 正官当令透干
- 为人正直、适合公职
- 喜印星相生、财星相生
- 忌伤官克官、七杀混杂

### 七杀格
- 七杀当令透干
- 有魄力、有野心
- 喜食神制杀、印星化杀
- 忌无制无化、官杀混杂

### 正财格
- 正财当令透干
- 勤俭务实
- 喜官星护财、食伤生财
- 忌比劫夺财、印星克财

### 偏财格
- 偏财当令透干
- 大方慷慨、适合经商
- 喜官星护财、食伤生财
- 忌比劫夺财

### 正印格
- 正印当令透干
- 学识渊博、适合文教
- 喜官星生印
- 忌财星克印

### 偏印格（枭神夺食格）
- 偏印当令透干
- 聪明多疑
- 喜官星生印
- 忌食神被夺

### 食神格
- 食神当令透干
- 温和有福
- 喜财星泄食、官星制食
- 忌枭神夺食

### 伤官格
- 伤官当令透干
- 才华横溢、个性叛逆
- 喜财星泄伤、印星制伤
- 忌见正官（伤官见官）

## 特殊格局

### 从格
- 日主极弱无根，从旺势走
- 从财格：财星最旺，从财走
- 从杀格：七杀最旺，从杀走
- 从儿格：食伤最旺，从食伤走
- 越旺越好，忌生扶日主

### 化气格
- 天干合化成某五行
- 甲己化土、乙庚化金、丙辛化水、丁壬化木、戊癸化火
- 必须月令当令才能化

### 专旺格
- 日主极旺无制，顺其势走
- 润下格（水旺）、炎上格（火旺）、曲直格（木旺）、从革格（金旺）、稼穑格（土旺）
- 越旺越好，忌克制

## 格局与用神的关系

- 格局确定后，用神围绕格局来取
- 正格取扶抑/调候/通关
- 从格取顺势（越旺越好）

# 八、用神取法

## 四种取法

### 1. 调候法
- 冬生喜火暖局，夏生喜水润局
- 春秋看具体配置
- 调候为先，其他次之

### 2. 病药法
- 找到命中最突出的「病」（过旺/过弱/刑冲）
- 对应的「药」就是用神
- 病重药轻 = 格局低，病轻药重 = 格局高

### 3. 通关法
- 两种五行对峙时，用中间五行通关
- 如木火对峙用水通关，金木对峙用土通关

### 4. 扶抑法
- 身弱喜生扶（印星/比劫）
- 身旺喜克泄（官杀/食伤/财星）

## 常见陷阱

不可机械套用「生我就是喜用」——辛金忌土就是经典反例（印多反而埋金）。

# 九、大运流年分析

## 大运

- 每步大运 10 年
- 重点分析当前和未来 2 步大运
- 结合流年给出关键年份提醒

### 大运十神要完整列出

列出大运天干十神 + 大运地支所有藏干十神，不要遗漏。

例如丙火日主走乙巳大运：
- 乙 = 正印（天干）
- 巳藏丙 = 比肩、巳藏庚 = 偏财、巳藏戊 = 食神（地支藏干）
- 完整：正印 + 比肩 + 偏财 + 食神

❌ 常见遗漏：只写「正印 + 比肩 + 偏财」，漏掉食神。

### 大运 ≠ 原命局（最常见的幻觉）

严格区分「原命局（四柱）中有」和「大运中出现」：

- 原命局有 = 命主先天自带的特质/缘分
- 大运有 = 某个 10 年阶段才会出现的运势/机遇

❌ 「庚金偏财不见天干，仅在巳中暗藏庚金（大运乙巳中才有）」——混淆了大运和原命局。

✅ 分两句写：
1. 「庚金偏财在原命局中完全不见（天干无、地支无、四柱所有藏干均无庚金）」
2. 「仅在大运乙巳（2030-2039）中巳藏庚金才有偏财出现」

同理，正财在原命局完全不见时，不能写「暗藏」。

## 流年

### 分析方法

1. 看流年天干与原命局/大运的十神关系
2. 看流年地支与原命局/大运的刑冲合会
3. 重点关注：
   - 流年冲克原命局 → 变动/冲突年
   - 流年合原命局 → 稳定/牵绊年
   - 流年引动子女星/配偶星 → 相关事件
   - 流年与大运天克地冲 → 大运交接的重大变动

### 关键年份提醒

- 岁运并临（大运地支 = 流年地支）→ 该十神能量极强
- 天克地冲（流年与原命某柱）→ 该宫位代表的六亲/事项有重大变动
- 三刑（寅巳申/丑戌未/子卯刑等）→ 内在折磨、反复

# 十、十日主特性与喜忌

## 甲木（参天大树）

### 特性
- 正直、有担当、有领导力
- 喜向上生长、不喜欢被压制
- 外形高大、骨架宽

### 喜忌

| | 喜 | 忌 |
|---|---|---|
| 喜 | 庚金（斧斤修剪成材）、丙火（阳光温暖）、癸水（雨露滋润） | 辛金（小刀无力）、土重（埋根） |
| 特殊 | 春甲喜丙火暖局；秋甲喜庚金劈甲 | 水多木漂、火炎木焚 |

## 乙木（花草藤萝）

### 特性
- 柔韧、适应力强、善于依附
- 外形纤细、气质柔和
- 最喜甲木为靠山（藤萝系甲格）

## 通用原则

- 调候为先：冬生喜火暖局，夏生喜水润局，春秋看具体配置
- 病药法：找到命中最突出的「病」（过旺/过弱/刑冲），对应的「药」就是用神
- 不可机械套用「生我就是喜用」——辛金忌土就是经典反例
