# 尤因肉瘤分子诊断流程

## 适用场景
- 患者为儿童或青年
- X线显示溶骨性病变伴“洋葱皮样”骨膜反应
- 组织学检查提示小圆细胞肿瘤
- 需启动分子病理确诊以指导化疗方案（如VDC/IE）

## 执行步骤

### 1. 临床初筛
- 确认患者年龄（通常<30岁）
- 评估症状：局部疼痛、可触及肿块，可能伴发热或病理性骨折
- 影像学确认：界限不清的溶骨性破坏 + 多层骨膜反应（洋葱皮外观）

### 2. 病理形态确认
- 肿瘤起源：髓腔，侵犯皮质、骨膜及软组织
- 大体特征：灰白色、质软如鱼肉，常伴出血坏死
- 镜下特征：
  - 小圆细胞呈片状排列，被纤维组织分隔
  - 胞质稀少或因糖原丰富呈透明样
  - 核圆形/卵圆形，染色质细腻
  - 可见Homer-Wright菊形团（非必需）

### 3. 免疫组化验证
- 检测以下标记（至少多项阳性）：
  - CD99（膜阳性）
  - NKX2.2
  - FLI-1
  - ERG

### 4. 分子遗传学确诊（关键步骤）
- 使用荧光原位杂交（FISH）检测EWSRI::FLII融合基因
- 阳性判读：红绿信号紧邻或融合成黄色信号
- 注意：约85%尤因肉瘤存在t(11;22)(q24;q12)易位

### 5. 排除诊断
- 若EWSRI::FLII阴性，考虑以下替代诊断：
  - EWSRI-非ETS融合肉瘤
  - CIC重排肉瘤
  - BCOR改变肉瘤
- 参考WHO第5版分类进行进一步分子分型

### 6. 输出结果
- 若满足全部标准 → 输出“尤因肉瘤确诊”，启动VDC/IE化疗方案
- 若EWSRI::FLII阴性但符合小圆细胞形态 → 输出“需进一步分子分型”

> **注意**：本流程不适用于非小圆细胞形态的肉瘤。
