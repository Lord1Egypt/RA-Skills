# 授权委托书模板（律师代理/公民代理）

## 适用场景
当事人因诉讼、仲裁、行政复议或其他法律事务，委托律师、基层法律服务工作者、近亲属、单位工作人员或有关社会团体推荐的人作为代理人，需向相关机构出具书面授权文书。

## 提交机构
受理案件/事务的人民法院、仲裁委员会、行政机关等

---

## 模板正文

### 模板一：律师代理（诉讼/仲裁通用）

```
                      授 权 委 托 书

致：________人民法院 / ________仲裁委员会

    委托人：[姓名/单位名称]
    （如为自然人：身份证号：________________________）
    （如为单位：统一社会信用代码：__________________）

    受委托人：[律师姓名]，________________律师事务所律师
    执业证号：______________________
    联系电话：______________________

    委托人与 [对方当事人姓名/名称] 之间因 [案由，如买卖合同纠纷] 一案（案号：________），现委托上述受委托人作为委托人的代理人，在案件的 [一审 / 二审 / 再审 / 仲裁 / 执行 / 申诉] 阶段参加诉讼/仲裁活动。

    代理权限（请选择以下一种）：

    □ A. 一般代理（一般授权）
      代为：提起诉讼/仲裁、应诉、提交证据、参加庭审、发表代理意见、接收/签署各类法律文书等

    □ B. 特别授权（全权代理）
      在上述一般代理权限的基础上，另可代为：
      1. 承认、放弃、变更诉讼/仲裁请求
      2. 进行调解/和解
      3. 提起反诉/反请求
      4. 提起上诉/申请再审
      5. 申请执行
      6. 代为签署调解协议、和解协议
      7. 代为领取/接收执行案款

    （注：选择B的，代理权限需逐项明确勾选；未勾选的视为未授权）

    代理期限：自签署之日起至 [本审程序终结之日 / 执行程序终结之日 / 具体日期] 止。

    受委托人在上述授权范围内所实施的法律行为，委托人予以承认并承担相应的法律后果。

    委托人（签名/盖章）：________________
                                              ________年____月____日

    受委托人（签名）：__________________
                                              ________年____月____日
```

```
                      授 权 委 托 书

致：________人民法院 / ________仲裁委员会

    委托人：[姓名]，身份证号：________________________
    住址：________________________________________
    联系电话：______________________

    受委托人：[姓名]，性别：____，身份证号：________________________
    与委托人关系：________________（如：夫妻 / 父子 / 单位同事）
    工作单位：__________________  职务：________________
    联系电话：____________________

    委托人与 [对方当事人] 之间因 [案由] 一案（案号：________），现委托上述受委托人作为委托人的代理人，参加本案诉讼/仲裁活动。

    代理权限：________________（一般代理 / 特别授权，同模板一）

    代理期限：自签署之日起至 [本审程序终结之日] 止。

    委托人（签名）：__________________
                                              ________年____月____日

    受委托人（签名）：__________________
                                              ________年____月____日

【附：公民代理人身份证明材料】
□ 近亲属代理：附关系证明（结婚证/户口本/派出所证明复印件）
□ 单位工作人员代理：附劳动合同/工作证明复印件
□ 社区/单位/有关团体推荐代理：附推荐函原件
```

### 模板三：执行阶段专用

```
                      授 权 委 托 书

致：________人民法院执行局

    委托人（申请执行人）：________________
    受委托人：________________律师事务所 ________律师（执业证号：________）

    委托人因 [案由] 一案（执行依据：(____)____法民初字第____号民事判决书），
    执行案号：(____)____法执字第____号

    现委托上述受委托人担任本案执行阶段的代理人。

    代理权限：
    1. 提交执行申请及相关材料
    2. 提供被执行财产线索
    3. 申请执行措施（查封、扣押、冻结、划拨等）
    4. 参与执行和解
    5. 代为领取/接收执行款物 □
    6. 代为申请撤销/中止/恢复执行
    （注：如不授权领取执行款物，请划掉第5项）

    委托人（签名/盖章）：________________
                                              ________年____月____日

    受委托人（签名）：__________________
                                              ________年____月____日
```

## 注意事项

### 关键填写点
1. **代理权限必须明确**：特别是"得到授权"/"全权代理"这类模糊表述在法律上仅被视为一般代理——如需要特别授权（承认/放弃/变更请求、和解、上诉等），必须逐项列明
2. **区别当事人亲自签名**：委托人是单位的需加盖公章+法定代表人签字，个人需本人亲笔签名
3. **受委托人信息精确**：律师写执业证号，公民写身份证号
4. **代理阶段要写清**：一审的授权委托书不能用于二审，每个阶段需要单独提交
5. **执行阶段特别注意**：是否授权领取执行款物必须明确勾选，否则律师无法代领

### 常见错误
- "全权代理"被法院视为一般代理——必须逐项写明具体权限
- 忘了写代理期限——可能导致授权范围不明确
- 公民代理未附关系证明材料——法院可能拒绝认可代理资格
- 委托书签名日期与实际提交日期相差过远——建议签署日期与实际使用日期接近
- 非当事人本人签名——委托书须本人亲笔签名，不可代签

### 关于公民代理的限制
根据《民事诉讼法》第六十一条，公民代理限于以下人员：
1. 当事人的近亲属（配偶、父母、子女、兄弟姐妹、祖父母、外祖父母、孙子女、外孙子女）
2. 当事人的工作人员（需提供劳动关系证明）
3. 当事人所在社区、单位以及有关社会团体推荐的人
4. **范围之外的"普通朋友/熟人"不能作为代理人**，法院会拒绝其出庭

### 份数/格式要求
- 一般提交1份原件给法院/仲裁委
- 如有多个共同被告/被申请人，每人提供1份
- A4纸，宋体，小四号，1.5倍行距
- 法人委托书须加盖公章
