# 细胞自噬的定义与核心过程

## 何时使用
- 用户询问细胞自噬的基本机制或生物学意义
- 需要判断特定条件下（如饥饿、mTORC1失活）是否启动自噬
- 分析实验数据（如LC3-II水平、自噬体数量）以评估自噬活性
- 区分自噬与其他降解途径（如泛素-蛋白酶体系统）

## 执行步骤

1. **确认适用对象**：仅适用于真核细胞，不适用于原核细胞。
2. **验证前提条件**：
   - 存在自噬相关基因（ATG）
   - 存在溶酶体或晚期内体
   - 细胞处于应激状态（如营养匮乏或细胞器损伤）
3. **检查抑制条件**：若mTORC1活性高，则自噬被抑制；仅当mTORC1失活时才可启动。
4. **执行自噬核心流程**：
   - **起始阶段**：在细胞质中形成分隔膜（phagophore）。
   - **延伸与包裹**：分隔膜延伸并包裹目标物质（如受损线粒体、可溶性蛋白），闭合形成自噬体。
   - **融合阶段**：自噬体与晚期内体或初级溶酶体融合，生成自噬溶酶体。
   - **降解阶段**：酸性水解酶将内容物降解为小分子（如氨基酸、核苷酸）。
   - **回收利用**：降解产物释放回细胞质，用于合成或供能。
5. **识别关键标志**：检测LC3-I是否脂化为LC3-II（结合磷脂酰乙醇胺PE），LC3-II定位于自噬体膜，是自噬激活的可靠指标。
6. **评估结果**：通过LC3-II表达量、电镜下自噬体数量或底物降解速率判断自噬水平，反映细胞稳态维持能力。

> 注意：自噬体膜的确切来源（如内质网或高尔基体）尚存争议，不影响核心流程判断。
