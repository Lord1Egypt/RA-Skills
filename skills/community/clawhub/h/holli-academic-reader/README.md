# Academic Reader - 学术阅读助手

基于钱学森《工程控制论》方法论的 AI 学术阅读助手。

## 🎯 这个技能做什么？

帮助用户：
- **系统阅读技术书籍**：不只是总结，而是深入分析
- **批判性思考**：不只是接受，而是质疑和评估
- **个人融合**：把理论应用到自己的学习和成长
- **生成执行计划**：从"知道"到"做到"

## 📖 使用方法

安装后，直接对 AI 说：

```
帮我用工程控制论的方法分析《XXX》
```

AI 会按照以下结构回应：
1. 核心概念提取
2. 理论框架分析
3. 与个人进化的融合
4. 批判性思考
5. 执行计划
