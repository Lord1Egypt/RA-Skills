# Meta Ads Quality Checklist

Review each item before launching any Meta ad campaign. Items marked [CRITICAL] must be resolved before launch.

---

## 1. Account Structure (7 items)

- [ ] [CRITICAL] Business Manager is verified and in good standing
- [ ] [CRITICAL] Ad account has valid payment method with sufficient funds
- [ ] Campaign naming convention is consistent (e.g., Brand_Funnel_Objective_Date)
- [ ] Ad set naming includes audience identifiers (e.g., LAL_Purchasers1pct_US)
- [ ] Ad naming includes format, angle, version (e.g., Video_SocialProof_V2)
- [ ] No more than 3-5 ad sets per campaign at budgets under $10K/month
- [ ] Inactive campaigns are paused or archived

## 2. Audience Strategy (8 items)

- [ ] [CRITICAL] Exclusions configured between funnel stages (purchasers excluded from TOF, BOF excluded from MOF)
- [ ] [CRITICAL] Customer lists uploaded with match rate above 60%
- [ ] Retargeting audiences built for ViewContent, AddToCart, InitiateCheckout, Purchase events
- [ ] Retargeting segmented by recency (7/14/30-day windows)
- [ ] Lookalikes created from highest-quality seeds (purchasers, high-AOV, repeat buyers)
- [ ] Audience overlap between ad sets checked and below 30%
- [ ] Interest audiences researched and documented with rationale
- [ ] Audience sizes sufficient for budget (minimum 100K for TOF)

## 3. Creative Assets (8 items)

- [ ] [CRITICAL] Images meet minimum resolution (1080x1080 feed, 1080x1920 stories)
- [ ] [CRITICAL] Videos under 4GB in supported formats (MP4, MOV)
- [ ] Minimum 3-5 creative variants per ad set
- [ ] Creative messaging tailored to funnel stage
- [ ] Mobile-first design for all creative
- [ ] Video hooks compelling within first 3 seconds
- [ ] Videos have captions/text overlays for sound-off viewing
- [ ] Correct aspect ratios for assigned placements (1:1/4:5 feed, 9:16 stories)

## 4. Ad Copy (6 items)

- [ ] [CRITICAL] Primary text has clear value proposition in first two visible lines
- [ ] Headlines under 40 characters
- [ ] CTA button matches objective (Shop Now for sales, Learn More for consideration)
- [ ] TOF copy focuses on problem/solution or social proof hooks
- [ ] BOF copy includes urgency, offers, or cart reminders
- [ ] Copy proofread for spelling, grammar, and brand voice

## 5. Placements (5 items)

- [ ] [CRITICAL] Each placement has correctly formatted creative (no square images in Stories)
- [ ] Placement selection based on historical performance data
- [ ] If using Advantage+ Placements, creative provided for all major placements
- [ ] Placement asset customization enabled where formats differ
- [ ] Low-performing placements from prior campaigns deprioritized

## 6. Bidding and Budget (7 items)

- [ ] [CRITICAL] Bid strategy matches campaign phase (Lowest Cost for learning, Cost Cap for scaling)
- [ ] [CRITICAL] Budget supports 50+ optimization events/week per ad set
- [ ] Cost/bid caps set 20-30% above target to avoid underdelivery
- [ ] Budget allocation follows funnel ratios for business maturity
- [ ] Testing budget (10-20%) reserved for experiments
- [ ] Scaling rules documented (max 20-30% increase every 3-5 days)
- [ ] Breakeven CPA calculated and documented

## 7. Pixel and Tracking (7 items)

- [ ] [CRITICAL] Meta Pixel installed and firing on all site pages
- [ ] [CRITICAL] Standard events configured: PageView, ViewContent, AddToCart, InitiateCheckout, Purchase
- [ ] [CRITICAL] Purchase event passes correct value and currency
- [ ] Conversions API (CAPI) implemented for server-side tracking
- [ ] Event match quality score above 6.0
- [ ] Event deduplication configured between Pixel and CAPI
- [ ] UTM parameters on all ad URLs (utm_source=facebook, utm_medium=paid)

## 8. Testing Plan (5 items)

- [ ] [CRITICAL] Each test isolates a single variable
- [ ] Test duration defined: 3-7 days or 50+ conversions per variant
- [ ] Success criteria defined before launch (CPA, ROAS, or CTR threshold)
- [ ] Decision matrix documented: scale, iterate, or pause
- [ ] Creative refresh scheduled every 1-2 weeks for TOF

## 9. Compliance (5 items)

- [ ] [CRITICAL] All ads comply with Meta Advertising Standards
- [ ] [CRITICAL] Landing pages are functional, mobile-optimized, and match ad promise
- [ ] Special ad categories declared if applicable (credit, housing, employment)
- [ ] No prohibited health claims, income guarantees, or misleading before/after
- [ ] Privacy policy accessible on landing page

---

## Post-Launch Monitoring (First 7 Days)

**Daily**: Campaigns spending, no rejections, CPA/ROAS within range (allow 30-50% variance during learning), pixel events firing

**Day 3**: Identify early creative winners by CTR/CPC, verify audience delivery, check retargeting frequency (below 2.0)

**Day 7**: Evaluate learning phase exit (50+ events), compare actual vs. target KPIs, pause underperformers, plan creative refresh
