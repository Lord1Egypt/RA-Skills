## Description: <br>
Research codebases and craft professional RFC proposals for GitHub publication. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[zz0116](https://clawhub.ai/user/zz0116) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and engineers use this skill to analyze codebases, research technical problems, design evidence-based architecture proposals, and prepare RFCs for open-source project discussion or publication. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can help publish RFC content to GitHub through the user's authenticated account. <br>
Mitigation: Require explicit user approval before running gh, curl, browser-based posting, or any other publication step. <br>
Risk: Generated RFCs may include private code details, internal plans, or incorrect guidance if published without review. <br>
Mitigation: Review the final RFC, confirm the target repository and GitHub account, remove sensitive details, and verify claims before publication. <br>
