# 骨骼肌肌纤维类型分析

## 何时使用
- 制定运动员专项训练方案（如耐力 vs 爆发力项目）
- 解释个体在不同运动类型中的表现差异
- 进行运动生理学或肌肉组织学相关分析

> **注意**：本技能仅适用于人类骨骼肌，不适用于心肌、平滑肌或其他非哺乳动物物种。

## 执行步骤

1. **识别肌纤维类型**
   - 慢收缩肌纤维（I型，红肌）：较细，富含肌红蛋白、血红蛋白和线粒体
   - 快收缩肌纤维（II型，白肌）：较粗，肌红蛋白含量低，糖酵解酶活性高

2. **判断主导代谢方式**
   - I型：以有氧氧化为主 → 能量产生速率慢但效率高，抗疲劳
   - II型：以无氧酵解为主 → 能量爆发快但易疲劳

3. **关联运动功能表现**
   - 耐力型活动（如马拉松、自行车）主要依赖I型纤维
   - 爆发力型活动（如短跑、举重）主要依赖II型纤维

4. **考虑可塑性因素**
   - 肌纤维类型比例受遗传影响，但可通过长期专项训练发生适应性转变（如耐力训练可提升I型比例或IIa亚型转化）

## 输出要求
输出应明确说明目标个体或场景中主导肌纤维类型、对应代谢特征及对运动能力的影响，用于指导训练干预或生理评估。
