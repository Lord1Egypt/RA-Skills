# 肾细胞癌手术治疗选择标准

## 适用场景
- 患者已明确肾癌临床分期（T1–T4）
- 已评估肾功能（是否为孤立肾、慢性肾脏病等）
- 需确定手术方式及范围

## 执行步骤
1. **确定T分期**：
   - T1：肿瘤≤7 cm，局限于肾内
   - T2：肿瘤>7 cm，局限于肾内
   - T3/T4：侵犯肾周组织、静脉、邻近器官或超出Gerota筋膜
2. **优先选择肾部分切除术（PN）**：
   - 所有T1期患者首选PN（保留肾功能，肿瘤控制效果等同根治术）
   - 可扩展用于T2及以上患者，若满足以下任一条件：
     - 解剖性或功能性孤立肾
     - 合并慢性肾脏病（CKD）
     - 家族性肾癌（如VHL综合征）
     - 双侧肾癌
     - 能保证R0切除及安全切缘
3. **选择根治性肾切除术（RN）**：
   - T1期但肿瘤位置深在、靠近肾门血管，无法安全行PN
   - T2–T4期常规选择RN
   - 手术范围包括：患肾、肾周脂肪囊、同侧输尿管上段
4. **决定附加操作**：
   - 肿瘤侵犯同侧肾上腺 → 同时切除
   - 侵犯肾周筋膜 → 切除受累筋膜
   - 存在肾静脉/下腔静脉癌栓 → 行癌栓取出术
   - 影像或术中发现可疑淋巴结 → 行区域淋巴结清扫（否则不常规清扫）
5. **选择手术入路**：优先采用腹腔镜或机器人辅助微创手术

> ⚠️ 注意：不适用于无法保证切缘阴性的T1肿瘤或存在严重凝血功能障碍等手术禁忌者。
