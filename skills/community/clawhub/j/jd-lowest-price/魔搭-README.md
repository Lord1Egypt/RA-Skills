# 京东自营历史最低价

## 简介

本服务从京东历史最低价频道中精选高品质京东自营商品，为用户提供智能化的抄底商品推荐。服务自动筛选好评率高、降价幅度大的优质商品，帮助用户在众多最低价商品中快速找到值得入手的好货。

## 核心特色

- **品质保障**：所有推荐商品均为京东自营，好评率≥97%为硬性底线
- **智能筛选**：默认推荐好评≥97%、销量≥500的抄底好货
- **多维过滤**：支持按关键词、价格上限、好评率自定义筛选
- **灵活排序**：支持按综合评分、到手价、降价幅度排序
- **分页浏览**：每页50条，支持翻页浏览更多商品

## 工具说明

### get_lowest_picks

获取京东自营历史最低价精选商品列表。

**参数：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| keyword | string | 否 | 空 | 品类或商品关键词，如"手机"、"家电"、"美妆"。支持模糊匹配，如"家电"可匹配"家用电器" |
| max_price | number | 否 | 0 | 价格上限（元），0表示不限。如设100表示筛选到手价≤100元的商品 |
| sort | string | 否 | score | 排序方式：score=综合评分、price=到手价升序、drop=降价幅度降序 |
| min_good_comments | number | 否 | 97 | 好评率下限（%），可选97/98/99/100。指定后好评率不降级，只放宽销量门槛 |
| page | integer | 否 | 1 | 页码，每页返回50条 |

**返回内容：**

- 商品名称、图片、到手价、原价、降价幅度
- 好评率、30天销量、品牌、店铺名称
- 京东商品购买链接
- 当前筛选条件摘要和商品总数
- 热门分类引导（无关键词浏览时）

## 使用示例

### 1. 浏览推荐好货

不传任何参数，获取默认推荐的抄底好货列表（第1页50条）。

### 2. 关键词搜索

`keyword: "手机"` — 搜索手机相关的历史最低价商品。

### 3. 价格筛选

`max_price: 100` — 查看到手价100元以内的抄底好货。

### 4. 好评率筛选

`min_good_comments: 99` — 只看好评率≥99%的商品，好评率不会自动放宽。

### 5. 降价幅度排序

`sort: "drop"` — 按降价幅度从大到小排列，优先展示降幅最大的抄底好货。

### 6. 综合筛选

`keyword: "家电", max_price: 500, sort: "drop"` — 搜索500元以内的家电商品，按降价幅度从大到小排列。

## 品质说明

- 所有商品均为京东自营，品质有保障
- 好评率≥97%为硬性底线，任何情况下都不会推荐低于此标准的商品
- 默认推荐好评≥97%、销量≥500的抄底好货
- 当符合条件商品较少时，服务会自动放宽销量要求（500→100→不限），确保用户能看到更多可选商品
- 用户自定义好评率时（如设98%或100%），将严格按照指定标准筛选，不会自动放宽好评率

## 注意事项

- 历史最低价商品可能很快售罄，看到合适的建议尽快下单
- 最低价商品更新较快，可以多次查询发现新好货
- 购买链接点击后自动识别设备，手机端有京东APP则唤起APP
- 降价幅度为参考价与到手价的差额，实际以下单价格为准
