### 1.3 竞争格局特征

- **区域性强**：北方 vs 南方、一二线 vs 下沉市场，口味差异巨大

- **新消费品牌涌现**：从线上爆款切入，向线下扩张

## 5. 合规与监管

### 5.1 广告法专属限制

| 限制类型 | 禁用/限制 |
|---------|---------|
| 极限词 | "最""第一""国家级""顶级""纯天然""零添加"（无依据时） |
| 功效宣称 | 非保健食品不得宣称"保健""治疗""预防疾病" |
| 健康暗示 | 不得暗示"代餐可替代正餐""减肥"等 |
| 儿童 | 不得在少儿频道/节目中投放不健康食品广告 |

### 5.2 食品标签合规

| 必备项 | 备注 |
|-------|------|
| 配料表 | 按添加量降序 |
| 营养成分表 | 能量/蛋白质/脂肪/碳水/钠 |
| 致敏物质提示 | 牛奶/坚果/麸质等 |
| 生产日期 + 保质期 | 显著标注 |
| 贮存条件 | 常温/冷藏/冷冻 |
| 食用方法 | 速食/预制菜必备 |

### 5.3 功效宣称边界

| 食品类别 | 可宣称 | 不可宣称 |
|---------|-------|---------|
| 普通食品 | 口味、原料、工艺 | 健康功效、治疗 |
| 保健食品（蓝帽子） | 已批准的 27 项功能 | 未批准的功能 |
| 特医食品 | 特定疾病人群 | 普通消费者广告 |

> ⚠️ **AI 方案禁用**：不得为普通食品设计"提高免疫力""排毒""抗衰老"等宣称。

### 5.4 平台审核红线

| 平台 | 重点红线 |
|------|---------|
| 抖音 | 夸大功效、虚假成分、低俗吃播 |
| 小红书 | "代餐减肥"暗示、医疗用语 |
| 微信 | 多级分销、虚假宣传 |
| 直播间 | 假原产地、保质期虚假 |

## 7. 常见雷区

| 雷区 | 典型表现 | 严重度 | 规避方法 |
|------|---------|:-----:|---------|
| 功效宣称越界 | 普通食品宣"提高免疫力" | 🔴 致命 | 必须查验蓝帽子资质 |
| 虚假成分宣传 | 浓度虚标、产地造假 | 🔴 致命 | 第三方检测 |
| 节庆备货过剩 | 礼盒节后大量库存 | 🟡 中 | 预售制 + JIT |
| 季节性错配 | 冬季推冰饮 / 夏季推热饮 | 🟡 中 | 节气日历 |
| 渠道冲突 | 线上线下价格倒挂 | 🔴 高 | 渠道分层 + 区别 SKU |
| 同质化 | "零糖"已泛滥 | 🟡 中 | 引用 creative-diversity-protocol.md |
| 物流时效 | 冷链断裂、保质期不足 | 🔴 高 | 物流温控 + 仓配体系 |

## 8. 推荐方法论组合

| 方法论 | 适用场景 |
|--------|---------|
| **FAST + GROW** | 大促人群资产 |
| **AISAS** | 新品种草 |
| **私域金字塔** | 社群+订阅+复购 |
| **5A** | 人群资产 |
| **客户生命周期（CLM）** | 复购运营 |

**不推荐**：
- 单纯的"线上+线下"分层（现代食品是全渠道融合）
- 价格战（损害品牌，且食品品类替代品多）
