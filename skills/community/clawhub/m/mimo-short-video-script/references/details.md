# 短视频脚本生成器 - 详细参考

## 一、各平台算法推荐机制

### 1.1 抖音推荐算法（2026年）
```
流量池机制：
200播放 → 500播放 → 2000播放 → 1万播放 → 10万播放 → 100万播放

每一级考核指标：
├── 完播率（权重最高）：>30%进入下一级
├── 点赞率：>3%
├── 评论率：>0.5%
├── 转发率：>0.3%
├── 关注转化率：>1%
└── 停留时长：>15秒

核心策略：
- 前3秒决定完播率（Hook至关重要）
- 中间15秒一个高潮（防划走）
- 结尾设计互动点（提升评论率）
- 发布时间影响初始流量池质量
```

### 1.2 快手推荐算法
```
特点：更注重社交关系和粉丝粘性
├── 关注页权重更高（粉丝运营很重要）
├── 双列展示（封面和标题决定点击率）
├── "老铁"文化（真实>精致）
└── 直播引流效果好（短视频+直播联动）
```

### 1.3 TikTok推荐算法
```
For You Page (FYP) factors:
├── Watch time / Completion rate (highest weight)
├── Engagement (likes, comments, shares, saves)
├── Video information (captions, sounds, hashtags)
├── Device & account settings (language, country, device)
└── Trending sounds / challenges boost visibility

Key differences from Douyin:
- English content only
- Music/sound-driven (use trending sounds)
- Hashtag challenges drive massive views
- 3x longer content performs well (60-120s)
```

### 1.4 小红书视频算法
```
特点：搜索+推荐双引擎
├── 封面精致度（影响点击率）
├── 标题含关键词（搜索权重高）
├── 干货属性（收藏率是核心指标）
├── 标签精准（影响推荐分发）
└── 评论区运营（互动率影响持续推荐）
```

### 1.5 视频号推荐算法
```
特点：社交推荐+算法推荐
├── 朋友点赞（社交裂变，权重最高）
├── 转发到群/朋友圈（扩散效应）
├── 完播率
├── 适合中年受众（30-50岁内容偏好）
└── 与公众号/小程序联动效果好
```

## 三、不同内容类型的脚本结构

### 3.1 知识干货类（适合涨粉）
```
结构：痛点(5s) → 方法1(15s) → 方法2(15s) → 方法3(15s) → 总结CTA(10s)
时长：60秒
关键：每个方法有具体例子，不要空洞
互动设计："你们觉得哪个最有用？评论区告诉我"
```

### 3.2 产品种草类（适合带货）
```
结构：使用场景(5s) → 产品亮相(5s) → 效果展示(20s) → 使用感受(15s) → 购买引导(15s)
时长：60秒
关键：真实使用效果>品牌描述
互动设计："想知道在哪买的扣1"
```

### 3.3 口播观点类（适合涨粉+引流）
```
结构：观点抛出(5s) → 论据1(15s) → 论据2(15s) → 争议/反转(10s) → 升华CTA(15s)
时长：60秒
关键：观点要鲜明，不要太中庸
互动设计："你同意吗？"
```

### 3.4 剧情搞笑类（适合涨粉+品牌曝光）
```
结构：日常场景(5s) → 冲突/误会(20s) → 反转/高潮(15s) → 金句/梗(10s) → CTA(10s)
时长：60秒
关键：反转要出人意料，结尾要让人想分享
```

### 3.5 测评对比类（适合带货+信任建设）
```
结构：产品引入(5s) → 产品对比展示(25s) → 数据/效果(20s) → 推荐/排名(10s)
时长：60秒
关键：要有客观数据支撑，不能纯主观感受
```

## 五、BGM使用策略

### 5.1 BGM选择原则
- 匹配视频情绪节奏（快歌→快节奏/慢歌→情感）
- 优先使用平台热门音乐（增加推荐权重）
- 注意版权问题（TikTok有商用限制）
- 音乐音量：口播视频BGM音量控制在10-20%

### 5.2 各类型推荐BGM风格
| 内容类型 | 推荐风格 | BPM范围 |
|---------|---------|---------|
| 口播干货 | 轻快/电子 | 120-140 |
| 种草测评 | 甜美/清新 | 100-120 |
| 剧情搞笑 | 热门梗曲/反差 | 随场景变化 |
| 情感故事 | 钢琴/吉他/慢歌 | 60-90 |
| 运动健身 | 高能/鼓点强 | 130-160 |
| 美食制作 | 治愈/ASMR | 80-100 |
