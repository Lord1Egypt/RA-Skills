## Description: <br>
Edits Chinese text to identify and reduce common AI-writing patterns so drafts read more naturally while preserving the intended meaning. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[v585](https://clawhub.ai/user/v585) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users, editors, and reviewers use this skill to revise Chinese drafts that sound formulaic or AI-generated. It provides pattern checks, rewrite guidance, example transformations, and optional quality scoring. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: README installation commands reference a GitHub repository that is not server-resolved provenance for this release. <br>
Mitigation: Install from the reviewed ClawHub package or verify the referenced repository before using README commands. <br>
Risk: The skill is designed to change wording, tone, and style, which can unintentionally alter meaning or authorial voice. <br>
Mitigation: Use it only on documents intended for rewriting, and review edits before saving or publishing. <br>
