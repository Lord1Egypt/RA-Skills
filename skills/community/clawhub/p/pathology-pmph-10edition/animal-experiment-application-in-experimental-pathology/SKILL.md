# 实验病理学中动物实验的应用与局限

## 何时使用
当研究目标涉及以下情况时启用本技能：
- 需探究人类疾病的病因学、发病机制或病理转归，但无法在人体直接操作；
- 需评估潜在致癌物的致癌作用或癌变过程；
- 需分析病毒、细菌毒素等生物因子的致病机制；
- 需对药物或治疗手段进行初步安全性与有效性测试。

## 如何执行
1. **选择适宜动物模型**：根据目标人类疾病的病理生理特征，选用小鼠、大鼠、兔、犬等具有相似机制的实验动物。
2. **构建疾病模型**：通过物理、化学、生物或遗传手段在动物体内诱导出模拟人类疾病的病理状态。
3. **系统观察记录**：全程监测并记录病因、发病过程、组织病理变化及疾病转归。
4. **开展对照研究**：可对模型进行任意取样、干预，并与已知人体疾病数据进行对比分析。
5. **综合判断结果**：将动物实验结果作为初步证据，结合人体病理学研究进行验证与外推。

## 关键限制
- **不适用于存在显著种属差异的生理或病理过程**（如代谢酶系、免疫应答、器官结构等）；
- **不能替代人体临床验证**，结果不可直接外推至人类；
- 必须获得伦理审查许可，并确保有明确的人类疾病对照目标。

> 注意：本技能仅提供机制探索与初步评估支持，最终结论需依赖人体研究验证。
