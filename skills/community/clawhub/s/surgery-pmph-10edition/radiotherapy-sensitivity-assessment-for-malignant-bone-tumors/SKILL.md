# 恶性骨肿瘤放疗敏感性评估与推荐

## 何时使用
- 患者已确诊为特定类型的恶性骨肿瘤（如尤因肉瘤、骨肉瘤等）
- 正在评估放疗作为局部控制手段的可行性
- 手术困难或不可行，需考虑替代或辅助治疗

## 执行步骤

### 第一步：识别肿瘤类型
确认患者肿瘤属于以下哪一类：
- 尤因肉瘤
- 骨肉瘤
- 骨巨细胞瘤
- 脊索瘤
- 骨髓瘤

### 第二步：根据类型判断放疗敏感性并给出建议

- **尤因肉瘤**：高度敏感。可联合化疗使用，但必须联合全身治疗，避免单纯放疗。
- **骨肉瘤**：不敏感。不应将放疗作为主要治疗手段。
- **骨巨细胞瘤**：有条件使用（如脊柱等手术困难部位），但需警惕放疗后恶变风险。
- **骨髓瘤**：局部病灶对放疗有效，适用于无法手术者。
- **脊索瘤**：化疗无效，放疗可用于术后残留或无法完全切除者，但复发率高。

### 第三步：综合决策
优先对放疗敏感的肿瘤类型使用放疗；对不敏感者避免依赖放疗；对有条件使用者需充分告知风险。

## 输出要求
根据肿瘤类型推荐是否使用放疗，并说明注意事项（如联合治疗必要性、恶变风险等）。
