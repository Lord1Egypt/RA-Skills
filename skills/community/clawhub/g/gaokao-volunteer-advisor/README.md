# 高考志愿全程顾问

站在考生和家长一边的高考志愿决策助手。不只是算"能上哪"，更帮你判断"该不该去、去了会怎样、这笔账划不划算"。

## 适合这些场景

- 不知道自己的分数能报哪些学校
- 想做冲稳保梯度
- 想深度了解某所学校的真实情况（安全、校风、就业）
- 想核实某个专业在这所学校值不值得读
- 想检查志愿方案有没有风险
- 想生成一份可以直接分享给家人的分析报告

## 你需要准备什么

最好提供：

- 省份和年份
- 科类或选科
- 分数和位次（位次更重要）
- 批次（本科批、提前批等）
- 想去的城市
- 想学或不想学的专业
- 是否接受中外合作、民办、调剂、高学费
- 风险偏好：保守、均衡、激进

不需要提供身份证号、准考证号、密码等隐私信息。

## 它会做什么

整个决策分四个阶段，你可以从任意阶段开始：

**阶段一：报志愿** — 从分数和位次出发，做城市、学校、专业、就业的四支柱分析，产出冲稳保候选名单。

**阶段二：学校深度调查** — 对你关心的学校做八维"体检"：报考技术、底线安全、血统通道、生源画像、校风治理、学业出路、城市生活、暗规则。挖招生简章不会写的东西。

**阶段三：专业核实** — 两层穿透：这个专业本身好不好 × 这所学校的专业是什么成色。

**阶段四：最终排表** — 生成一份完整的 HTML 分析报告，浏览器直接打开，可以打印或发给家人。

## 最终产出

一份**单文件 HTML 报告**（如 `志愿分析报告.html`），包含：

- 结论摘要
- 关键信息卡片
- 冲稳保方案表格
- 学校体检报告（如有）
- 专业方向建议
- 重点风险提醒
- 官方核验清单
- 下一步行动

## 很重要的提醒

高考志愿会影响真实升学结果。这个 Skill 只能辅助分析，不能保证录取。

- 所有招生信息请以省考试院和学校官网为准
- 不要把身份证号、准考证号、密码、验证码发给 AI
- 最终提交志愿请在官方平台自己操作
