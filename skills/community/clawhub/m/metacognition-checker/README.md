# Metacognition Checker

Check whether a user truly understands a concept, decision, plan, or belief, or is only feeling familiar and overconfident.

## Use Cases

- Learning validation after reading or studying.
- Blind-spot detection before a decision.
- Error review after a wrong judgment.
- Confidence calibration before an exam, interview, memo, or plan.

## Core Checks

1. Explain the idea without copying source wording.
2. Apply it to a new example.
3. Name where it fails or does not apply.
4. Separate observed facts from interpretation.
