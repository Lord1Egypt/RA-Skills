# PPT 压缩与知识库上传标准 v1

## 背景

飞书知识库/文件上传存在大小限制。`AI保险行业专场_最终成品_v1.5.pptx` 原始大小约 23.1MB，超过部分上传接口限制。

## 版本区分

- `final_high_quality.pptx`：正式展示 / 现场播放。
- `final_kb_safe_q95.pptx`：知识库上传安全版。
- `final_archive_original.pptx`：原始归档。
- 低清压缩版只允许临时传输验证，不得作为正式展示或最终交付。

## 压缩策略

优先采用：

1. 不缩图片尺寸；
2. 非透明图片高质量重压缩 q95；
3. 保留原始画面尺寸；
4. 检查页数、媒体数量、预览渲染。

若仍超过限制，再逐步：q95 → q92 → q90 → 单独处理超大图。

禁止直接粗暴降采样导致锯齿、颗粒、渐变发花。

## 上传记录

必须记录：

- 上传目标知识库/节点；
- 文件名；
- file_token 或链接；
- 上传身份；
- 上传时间；
- 验证结果；
- 如果是压缩版，说明压缩参数和用途。
