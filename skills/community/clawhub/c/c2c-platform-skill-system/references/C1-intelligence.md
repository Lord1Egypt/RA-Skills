## C1-01 竞品平台动态监测

- **输入**: 
  - 竞品平台列表（必选）：58同城、闲鱼、猪八戒、美团到店、天鹅到家、好慷在家等
  - 监测维度（必选）：产品功能变动、运营活动、定价策略、品类调整、用户增长动态
  - 数据源（必选）：App版本更新记录、官方公告、应用商店评价、媒体报道
  - 监测频率（可选）：日/周/月，默认周频
- **输出**: 
  - 竞品动态周报（Markdown格式），含：平台名称、变动类型、变动详情、影响评估(1-5级)、建议响应动作
  - 异动即时预警（当检测到重大变动时）：如竞品上线新品类、佣金率大幅调整、融资事件
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C1-11(竞品功能更新追踪), C1-21(竞品运营策略拆解), C1-22(市场份额估算), C1-15(平台定位SWOT分析), C2-05(推荐策略配置), C3-08(地推方案制定), C4-57(数据指标体系搭建), C8-29(竞品功能对标分析)

## C1-02 行业政策法规追踪

- **输入**: 
  - 监管领域（必选）：ICP/EDI许可新规、支付通道监管(央行支付新规)、个保法/网安法修订、虚拟币合规(代币发行风险)、反不正当竞争(裂变分润3级上限)、劳动法(灵活用工关系)、地方性法规
  - 监管机构列表（必选）：工信部、央行、市场监管总局、网信办、各省市通信管理局
  - 追踪渠道（可选）：国务院政策文件、工信部公告、央行通知、行业法律数据库
- **输出**: 
  - 政策法规月度追踪报告，含：法规名称、发布机构、生效日期、影响范围、对C2C平台的影响等级(高/中/低)、合规建议
  - 紧急政策预警：如涉及二清整改、虚拟币定性、裂变分佣上限调整等即时推送
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C1-18(支付通道政策追踪), C1-19(监管合规风险预警), C5-05(支付合规指引编写), C6-25(用户协议与隐私政策更新), C9-01(ICP备案申请), C9-03(EDI许可证申请)

## C1-03 区域市场密度分析

- **输入**: 
  - 目标区域列表（必选）：省/市/区三级行政区划，如北上广深、新一线城市、下沉市场
  - 分析维度（必选）：服务商密度（每万人服务商数）、需求密度（每万人发单数）、品类覆盖度、竞品渗透率
  - 数据源（必选）：平台内部数据、公开行业报告、地图POI数据
  - 对标竞品（可选）：58同城区域渗透率、美团到店区域覆盖、闲鱼区域活跃度
- **输出**: 
  - 区域市场密度热力图（按城市/区县粒度），含：供需密度指数、品类饱和度、市场空白度
  - 区域分级建议：核心深耕区/稳定运营区/潜力拓展区/待开发区
  - 重点城市报告：如成都家政市场密度 vs 天鹅到家渗透率
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C1-13(区域扩张可行性评估), C1-22(市场份额估算), C3-25(区域化活动适配), C4-20(区域交易密度分析), C7-20(供需冷启动策略)

  - 需求-供给匹配建议：哪些需求存在供给缺口
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C1-14(品类进入时机判断), C1-23(新品类机会识别), C2-05(推荐策略配置), C7-12(供给不足品类预警), C7-13(需求溢出品类预警)

## C1-06 价格带分布监测

- **输入**: 
  - 价格数据（必选）：平台各品类条目价格分布、历史价格变动、竞品同品类价格
  - 品类筛选（必选）：至少指定1个品类（如家政保洁、IT开发、维修安装）
  - 对标平台（必选）：58同城同类服务价格区间、闲鱼本地服务价格、猪八戒报价范围
  - 区域维度（可选）：按城市级别（一线/新一线/二线/下沉市场）分层
- **输出**: 
  - 价格带分布图（按品类×区域），含：低价带/中价带/高价带占比、中位数价格、价格离散度

  - 异常价格预警：如某品类价格突然整体下移（价格战信号）或上移（成本推动）
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C1-07(佣金率行业对标), C4-03(订单量与客单价分析), C4-49(客单价趋势预测), C8-14(运营成本核算), C8-15(佣金收入核算)

## C1-07 佣金率行业对标

- **输入**: 
  - 我方佣金率数据（必选）：各品类抽佣比例（如家政5%/IT开发8%/维修6%）、实际佣金收取率、佣金优惠活动
  - 竞品佣金率（必选）：58同城信息服务费率、美团到店佣金率、猪八戒服务费率、闲鱼免费策略
  - 分润结构（必选）：平台抽佣比例、L1/L2/L3推荐分润比例
  - 品类维度（必选）：按品类逐一对比
- **输出**: 
  - 佣金率对标表（品类×竞品矩阵），含：我方佣金率、行业均值、行业极值、竞争力评级
  - 佣金率调整建议：哪些品类佣金率过高/过低，建议调整方向与幅度
  - 分润合规校验：当前分润层级是否超过3级上限（反传销合规要求）
- **依赖**: C1-06
- **AI自治度**: ⬛
- **组合接口**: → C1-21(竞品运营策略拆解), C4-13(佣金收入分析), C4-35(推荐分润支出分析), C6-27(传销风险防控), C8-15(佣金收入核算)

## C1-15 平台定位SWOT分析

- **输入**: 
  - C1-01竞品动态数据（必选）：竞品平台特征、优劣势
  - C1-08舆情数据（必选）：用户对平台和竞品的态度、投诉热点
  - 平台自身数据（必选）：品类覆盖、服务商规模、用户规模、技术能力
  - 分析维度（可选）：聚焦特定品类或区域的SWOT
- **输出**: 
  - SWOT分析矩阵，行业化内容如：
    - S(优势)：虚拟币体系锁定资金池、裂变分润低成本获客、完成码机制保障履约
    - W(劣势)：品牌知名度低于58同城/美团、服务商规模不及竞品、下沉市场渗透不足
    - O(机会)：灵活用工政策利好、下沉市场本地生活需求爆发、竞品佣金率过高导致服务商外溢
    - T(威胁)：二清合规风险、虚拟币政策收紧、竞品下沉挤压、3级分润上限合规压力
  - 战略方向建议：SO策略/WO策略/ST策略/WT策略
- **依赖**: C1-01, C1-08
- **AI自治度**: 🟨
- **组合接口**: → C1-24(平台生态位分析), C8-30(产品路线图制定), C8-48(OKR制定与跟踪)
