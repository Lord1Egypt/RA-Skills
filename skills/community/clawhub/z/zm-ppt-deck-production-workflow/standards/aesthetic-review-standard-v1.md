# PPT 美感审核标准 v1

## 目的

高质量 PPT 审核不能只检查文字、结构和元素冲突，还必须检查美感。视觉平庸、廉价、模板感重，即使内容正确，也不能放行。

## 筑梦 / 野哥偏好的视觉方向

关键词：

- 黑金商业感；
- 高级、克制、有力量；
- 有空间纵深；
- 有主视觉记忆点；
- 有“行业 + AI 解决方案”的专业气质；
- 不廉价、不模板、不满屏素材堆叠；
- 有舞台感 / 发布会感 / 课程封面感；
- 适合线下沙龙、课程、招商、讲课场景。

## 平庸判定

以下任一情况判定为 NEEDS_FIX：

1. 背景像普通科技模板，没有记忆点；
2. 黑金只是简单黑底金线，缺少层次和质感；
3. 右侧视觉没有主体，只是装饰线和几何块；
4. 画面缺少纵深、焦点、光影层次；
5. 视觉不支撑主题，只是泛泛科技感；
6. 高密度内容压住画面，整体像信息板而不是高质量 PPT；
7. 线条、卡片、描边过多，显得廉价；
8. Logo、页脚、页码破坏画面呼吸感；
9. 看起来像“能用”，但不像“野哥会满意的高质量成品”。

## 放行标准

最终页面必须同时满足：

- 内容清楚；
- 结构完整；
- 版式舒服；
- 品牌元素克制；
- 有视觉记忆点；
- 黑金质感高级；
- 不抢讲师表达；
- 远看有冲击力，近看有细节。

## 审核结论格式

每次视觉审核必须给出：

- 内容准确性：PASS / NEEDS_FIX
- 版式结构：PASS / NEEDS_FIX
- 品牌元素：PASS / NEEDS_FIX
- 美感高级度：PASS / NEEDS_FIX
- 是否符合野哥风格：PASS / NEEDS_FIX

只要“美感高级度”或“是否符合野哥风格”为 NEEDS_FIX，整页不得放行。
