## Description: <br>
Book meditation services through Lokuli MCP for users who need to find and book meditation services. <br>

This skill is ready for commercial/non-commercial use. <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and agents use this skill to search for meditation services, check availability, and create bookings through Lokuli's MCP server. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Booking flows may transmit personal contact details and appointment preferences to an external service. <br>
Mitigation: Confirm with the user before sending name, email, phone number, zip code, or appointment details to Lokuli. <br>
Risk: The skill depends on Lokuli as the booking provider. <br>
Mitigation: Install and use the skill only when Lokuli is an acceptable provider for search, availability checks, and booking requests. <br>
