## Description: <br>
Help users decide what to cook and how to cook it, including dish recommendations from cravings or available ingredients and practical guidance for specific dishes or cooking techniques. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[zning1994](https://clawhub.ai/user/zning1994) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and agent operators use this skill to turn cravings, available ingredients, or dish-specific cooking questions into practical home-cooking recommendations, recipes, technique notes, substitutions, and failure warnings. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill requests file editing and shell command permissions that its recipe guidance purpose does not explain. <br>
Mitigation: Review before installing; prefer a safer configuration that limits tools to reading bundled reference files unless the user explicitly directs file or command use. <br>
