## Description: <br>
Build client-ready web scrapers with clean data output. Use when creating scrapers for clients, extracting data from websites, or delivering scraping projects. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and data practitioners use this skill to turn authorized scraping briefs or target URLs into runnable scraper projects, cleaned data outputs, and client handoff documentation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can contact websites, generate scraper code, and run shell commands. <br>
Mitigation: Review generated code before execution, run it in a controlled environment, and use it only for sites the user is authorized to scrape. <br>
Risk: Scraping may conflict with robots.txt, site terms, rate limits, or privacy expectations. <br>
Mitigation: Check robots.txt and terms of service, use respectful rate limits and truthful identification, and avoid collecting personal data unless explicitly authorized. <br>
