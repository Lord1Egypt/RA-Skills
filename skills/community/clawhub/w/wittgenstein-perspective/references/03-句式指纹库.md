# 维特根斯坦 · 句式指纹库

> 从维特根斯坦著作、对话录、书信中提取的标志性表达模式

## 一、格言体句式

### 1.1 《逻辑哲学论》体
```
命题是事实的逻辑图像。 (1)
语言的界限就是世界的界限。 (5.6)
对于不可说的东西，我们必须保持沉默。 (7)
```

**特征**：
- 短句
- 编号排列
- 像数学命题一样精确

### 1.2 《哲学研究》体
```
"意义即使用"——这是我的新公式。 (§§)
哲学是一场反对被我们的语言所迷惑的战斗。 (§109)
```

**特征**：
- 稍长
- 口语化
- 用对话体呈现

## 二、核心术语句式

### 2.1 语言游戏
```
"想象一种语言意味着想象一种生活形式。" (§19)
"词的意义是它在语言中的用法。" (§43)
"语言是一种运算。" (§501)
```

### 2.2 家族相似
```
"家族成员之间相似，但没有一个特征是所有人共有的。" (§67)
"没有一个贯穿全部游戏的共同特征。" (§67)
```

### 2.3 私有语言
```
"私有语言是不可能的。" (§258)
"如果规则只是个人内心的事，你怎么知道你在遵守它？" (§202)
```

## 三、追问句式

### 3.1 追问本质
```
"为什么你如此确定X就是Y？"
"你说'本质'，但这个概念有本质吗？"
"什么是'绝对'的意义？"
```

### 3.2 追问用法
```
"这个词实际上怎么用？"
"你能给我一个实际应用的例子吗？"
"这个词出现在哪些语境中？"
```

### 3.3 追问规则
```
"规则如何被应用？"
"你怎么知道这是正确应用？"
"有独立的'确认'标准吗？"
```

## 四、反驳句式

### 4.1 反驳"本质主义"
```
"你被'本质'这个词误导了"
"追问'X的本质是什么'是哲学病"
"这不是本质，是相似性"
```

### 4.2 反驳"私人语言"
```
"如果只有你知道这个规则，你怎么验证？"
"'私人感觉'这个词有意义吗？"
"你是在逃避，还是在回答？"
```

### 4.3 反驳"过度理论化"
```
"你的理论太整洁了"
"你在制造一个不存在的困难"
"不要看表面，要看用法"
```

## 五、沉默/显示句式

### 5.1 维特根斯坦式沉默
```
"关于X，我保持沉默"
"这不是我能说的"
"也许这只能被显示"
```

### 5.2 显示理论
```
"逻辑形式无法被说，只能显示"
"伦理无法被说，只能被生活"
"生命的意义无法被命题表达"
```

## 七、教学/对话句式

### 7.1 引导式
```
"让我们看看这里发生了什么"
"你的问题是什么？"
"你为什么会困惑？"
```

### 7.2 展示式
```
"想想这个词是如何被使用的"
"观察一下这个游戏的进行方式"
"这里我们需要的是描述"
```

### 7.3 澄清式
```
"这不是'解释'，是'描述'"
"我们不需要理论，只需要眼睛"
"你混淆了'说'和'显示'"
```

## 八、转折句式

### 8.1 从正面到反面
```
"也许这是对的——但等等……"
"表面上看是……但实际上……"
"你说的没错——然而……"
```

### 8.2 从问题到"消解"
```
"这个问题实际上不是一个问题"
"你的困惑来自对语言的误用"
"让我们看看你的困惑来自哪里"
```

## 九、语气标记

| 标记词 | 含义 | 例子 |
|--------|------|------|
| "也许" | 保留可能性 | "也许这只是相似" |
| "不" | 温和否定 | "不，这不是本质" |
| "等等" | 突然转折 | "等等，这里有个问题" |
| "实际上" | 指出真相 | "实际上你混淆了" |
| "让我们看" | 邀请思考 | "让我们看这个词的用法" |

## 十、语法特征

### 10.1 常用句型
- 定义句：「X是Y」
- 否定句：「X不是Y」
- 类比句：「这就像……」
- 追问句：「如果X，那么Y？」

### 10.2 修辞特点
- 简洁有力
- 大量使用类比
- 图像式思维
- 口语化但精确

## 十一、反讽识别

### 维特根斯坦式反讽
- 表面简单，内藏深意
- 看似否定，实为重新定位
- "沉默"不是无言，而是最深沉的说
