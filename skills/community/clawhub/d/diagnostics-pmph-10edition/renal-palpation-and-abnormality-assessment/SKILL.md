# 肾脏触诊与异常判断

## 适用场景
当患者疑似肾下垂、游走肾或肾肿大，且能配合深腹式呼吸时，使用本技能进行体格检查。

## 执行步骤

1. **选择体位**：
   - 首选仰卧位；若未触及，改用立位（尤其适用于肾下垂或游走肾）。

2. **右肾触诊**：
   - 嘱患者屈腿并做深腹式呼吸；
   - 医生左手托右腰部，右手平放右上腹，手指平行于右肋缘；
   - 在**吸气相**双手夹触肾脏；
   - 若触及光滑钝圆结构，可能为肾下极；若感知蚕豆状外形并引发酸痛或恶心，确认为肾脏。

3. **左肾触诊**：
   - 左手从前腹越过托起左腰部，右手横置左上腹；其余同右肾触诊。

4. **困难情况处理**（腹壁厚或配合差）：
   - 吸气时，左手向前冲击后腰部，右手感知被顶推感；
   - 或右手向左手方向冲击，左手感知肾脏。

5. **异常判断标准**：
   - 正常人通常不可触及肾脏（仅瘦长者偶触右肾下极）；
   - **肾下垂**：深吸气时可触及 ≥1/2 肾脏；
   - **游走肾**：肾下垂明显且可在腹腔多方向移动；
   - **肿大特征**：
     - 肾盂积水/积脓：柔软、有弹性、波动感；
     - 多囊肾：不规则增大、囊性感；
     - 肾肿瘤：表面不平、质地坚硬。

## 输出建议
记录肾脏是否可触及、位置、大小、活动性及质地，用于判断是否存在病理状态，并决定是否需影像学确认。

> ⚠️ 注意：腹壁过厚或患者配合不佳可能导致触诊失败；正常人通常无法触及肾脏。
