# PNH流式细胞术诊断

## 适用对象
- 疑似PNH患者
- CD59检测阴性但临床高度怀疑PNH者

## 前提条件
- 已完成蔗糖溶血试验或Ham试验筛查
- 具备流式细胞仪检测条件

## 执行步骤

1. **红细胞CD59检测**：
   - 健康人表现为单一CD59阳性峰（Ⅰ型细胞）
   - PNH细胞分为Ⅱ型（CD59部分缺失）和Ⅲ型（CD59完全缺失）
   - 诊断临界值：Ⅱ型 + Ⅲ型细胞比例 > 5%

2. **粒细胞和单核细胞Flaer检测**：
   - Flaer特异性结合GPI锚定蛋白；PNH细胞因GPI缺失而呈Flaer阴性
   - 健康人Flaer阳性率为100%
   - PNH细胞同样分为Ⅱ型（部分缺失）和Ⅲ型（完全缺失）

3. **推荐联合检测策略**：
   - 联合使用Flaer与CD59检测以提高诊断敏感性
   - 优先选择中性粒细胞作为高敏检测目标（因其不受输血影响）
   - 单核细胞可用于验证中性粒细胞克隆比例（尽管数量较少）

4. **特殊情况处理**：
   - 对于CD59阴性但临床高度怀疑PNH者，必须加做Flaer检测以明确诊断
   - 注意：近期输血可能稀释PNH克隆，影响红细胞CD59结果；Flaer检测不受此干扰

## 输出结果
返回PNH克隆比例及细胞类型（Ⅰ/Ⅱ/Ⅲ型）。若Ⅱ型 + Ⅲ型细胞比例 > 5%，则支持PNH诊断。
