# IP证券化

## 概述

本文件定义知识产权证券化的完整方法论，包括**SPV结构**、**入池标准**、**评级模型**和**利率定价公式**。

---

## 一、IP证券化概述

### 1.1 基本概念

知识产权证券化是指将知识产权的未来现金流作为基础资产，通过结构化设计发行可交易证券的融资方式。

### 1.2 与传统ABS的区别

| 特征 | 传统ABS | IP证券化 |
|------|---------|----------|
| 基础资产 | 实物资产、债权 | 知识产权、IP收益权 |
| 价值评估 | 相对客观 | 主观性较强 |
| 现金流稳定性 | 相对稳定 | 受市场影响大 |
| 法律问题 | 较少 | 权属、效力问题多 |
| 处置难度 | 相对容易 | 处置渠道有限 |

### 1.3 主要模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **版权证券化** | 影视、音乐、文学版权 | 娱乐、内容产业 |
| **专利证券化** | 专利许可费收益权 | 科技企业 |
| **商标证券化** | 品牌许可费收益权 | 品牌消费品 |
| **组合IP证券化** | 多种IP组合 | 大型IP运营企业 |

## 三、入池标准

### 3.1 五项硬性指标

| 序号 | 指标 | 标准 | 说明 |
|------|------|------|------|
| 1 | **权属清晰** | 权属无争议、无负担 | 必须提供权属证明 |
| 2 | **法律有效** | 专利有效、版权已登记 | CNIPA验证 |
| 3 | **现金流可预测** | 历史数据≥2年 | 收入稳定 |
| 4 | **剩余期限** | ≥2年 | 保证存续期 |
| 5 | **独立性** | 可剥离、不依赖原始权益人 | 风险隔离 |

### 3.2 专利入池详细标准

| 指标 | 标准 | 权重 |
|------|------|------|
| 专利有效性 | 已授权且有效，剩余保护期≥2年 | 必须 |
| 权属清晰度 | 权属链条完整，无共有/质押 | 必须 |
| 技术领域 | 属于国家重点支持领域 | 建议 |
| 商业化程度 | 已有稳定许可费收入≥1年 | 必须 |
| 市场地位 | 属于核心专利/基础专利 | 加分项 |
| 法律稳定性 | 无效宣告风险<30% | 必须 |

### 3.3 版权入池详细标准

| 指标 | 标准 | 权重 |
|------|------|------|
| 版权登记 | 已完成版权登记 | 必须 |
| 内容审核 | 内容合法合规 | 必须 |
| 历史收入 | 历史收入记录≥2年 | 必须 |
| 预期收入 | 未来收入可合理预测 | 必须 |
| 知名度 | 具备一定市场知名度 | 加分项 |
| 续集/衍生 | 存在续集/衍生潜力 | 加分项 |

### 3.4 组合入池标准

| 指标 | 标准 |
|------|------|
| 组合数量 | 单笔资产≤资产池20% |
| 行业分散 | 同行业资产≤资产池40% |
| 地域分散 | 同地区资产≤资产池50% |
| 类型分散 | 同类型资产≤资产池60% |

## 四、信用评级模型

### 4.1 评级维度

| 维度 | 权重 | 评估内容 |
|------|------|----------|
| **资产质量** | 40% | IP价值、权属、稳定性 |
| **现金流** | 30% | 历史收入、预测可靠性 |
| **交易结构** | 20% | 增信措施、隔离效果 |
| **参与方** | 10% | 原始权益人、服务机构 |

### 4.2 评级打分卡

| 维度 | 指标 | 评分标准 | 权重 |
|------|------|----------|------|
| 资产质量 | IP有效性 | 有效10分/无效0分 | 10% |
| | 权属清晰度 | 1-10分 | 10% |
| | 技术/内容质量 | 1-10分 | 10% |
| | 法律稳定性 | 1-10分 | 10% |
| 现金流 | 历史收入稳定性 | 标准差越小分越高 | 15% |
| | 收入预测可靠性 | 1-10分 | 10% |
| | 集中度风险 | 越分散分越高 | 5% |
| 交易结构 | 信用增级程度 | 增信比例越高分越高 | 10% |
| | 风险隔离效果 | 1-10分 | 5% |
| | 流动性安排 | 1-10分 | 5% |
| 参与方 | 原始权益人资质 | 1-10分 | 5% |
| | 服务机构能力 | 1-10分 | 5% |

### 4.3 评级结果映射

| 综合得分 | 评级 | 增信要求 |
|----------|------|----------|
| ≥90分 | AAA | 极低 |
| 80-89分 | AA | 较低 |
| 70-79分 | A | 中等 |
| 60-69分 | BBB | 较高 |
| <60分 | 不予评级 | — |

## 五、利率定价公式

### 5.1 定价公式

```
证券利率 = 无风险利率 + 信用利差 + 流动性溢价 + 期限溢价

其中：
无风险利率 = 同期限国债收益率
信用利差 = 评级对应利差
流动性溢价 = 1%-3%（IP证券流动性较低）
期限溢价 = 每增加1年 + 10-30BP
```

### 5.2 各评级信用利差参考

| 评级 | 参考利差（年化） | 数据来源 |
|------|------------------|----------|
| AAA | 50-80BP | 中债资信统计 |
| AA+ | 80-120BP | 中债资信统计 |
| AA | 120-160BP | 中债资信统计 |
| A+ | 160-200BP | 中债资信统计 |
| A | 200-250BP | 中债资信统计 |

### 5.3 定价计算示例

```
情景：发行IP资产支持证券

条件：
- 评级：AA
- 期限：3年
- 无风险利率：2.5%（3年国债）
- 信用利差：140BP（AA级）
- 流动性溢价：1.5%
- 期限溢价：0.3%（3年，比1年多90BP）

证券利率 = 2.5% + 1.4% + 1.5% + 0.3% = 5.7%
```
