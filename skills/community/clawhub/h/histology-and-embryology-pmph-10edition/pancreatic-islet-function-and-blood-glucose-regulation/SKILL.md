# 胰岛细胞功能与血糖调节机制

## 何时使用
- 用户提及空腹/餐后血糖异常
- 需判断高血糖或低血糖是否由胰岛激素分泌异常引起
- 需解释1型与2型糖尿病的病理生理差异
- 排除非胰岛源性血糖异常（如肾性糖尿）后聚焦内分泌调节

## 执行步骤

### 1. 识别触发条件
确认是否存在以下任一情况：
- 血糖浓度异常升高（>7.0 mmol/L 空腹 或 >11.1 mmol/L 随机）
- 血糖浓度异常降低（<3.9 mmol/L）

### 2. 分析胰岛激素响应
- **若血糖降低**：激活A细胞 → 分泌胰高血糖素 → 促进肝糖原分解 → 血糖回升
- **若血糖升高**：激活B细胞 → 分泌胰岛素 → 促进肝细胞、脂肪细胞摄取葡萄糖并储存为糖原或脂肪 → 血糖下降

### 3. 判断病理状态
- **胰岛素绝对缺乏**（B细胞破坏）→ 1型糖尿病 → 必须外源胰岛素治疗
- **胰岛素抵抗伴相对缺乏或分泌受损** → 2型糖尿病 → 可考虑口服促泌药或增敏剂
- **胰岛素分泌过多**（如胰岛素瘤）→ 反复低血糖发作

### 4. 验证前提条件
确保满足以下前提，否则结果不可靠：
- 存在功能性胰岛A细胞和B细胞
- 肝脏具备正常糖原储备能力

> 注意：本技能不适用于肾性糖尿、应激性高血糖等非胰岛源性血糖异常。
