## 一、Jyotish能预测的维度

| 维度 | 精度 | 说明 | 正确输出方式 |
|------|------|------|------------|
| **时间窗口** | 高 | Dasha+Transit交叉可给出高低概率时间段 | "X月前后有显著变化窗口" |
| **压力/机会/变化窗口** | 高 | "什么时候有压力/机会"比"具体发生什么"更可靠 | "此期间第Y宫处于压力/扩张期" |
| **固定业力（Dridha Karma）** | 高 | 深层责任、家庭结构模式、关键转折点——反复出现的强签名 | "此领域有强结构性签名" |
| **个人运作方式** | 高 | 思维类型、风险承受力、审美智能等 | "此人倾向于[运作方式]" |
| **宫位领域活跃度** | 高 | "X宫处于高度活跃期" | "第X宫在[时间段]高度活跃" |
| **承诺vs激活判断** | 中-高 | 命盘有承诺+Dasha-Transit对齐=高概率激活 | 使用两步法输出 |

## 二、Jyotish不能预测的维度

| 维度 | 原因 | 正确做法 |
|------|------|---------|
| **具体职业头衔/行业** | 同一行星能量可对应多种职业（强水星=金融/文案/咨询/法律/数据） | 列出"可能方向"，不指定某一个 |
| **具体事件的精确形式** | 形式受现实语境过滤（Rahu大运=辞职创业，不是升职） | 描述能量性质，不描述具体形态 |
| **个人在事件中的角色** | 主动/被动/主导/配合超出星盘精度 | 只说宫位属性（个人向/协作向），不说具体角色 |
| **保证确定的结果** | 占星给出概率，不是保证 | 每条预测标注置信度，不承诺结果 |
| **主观定义的满足** | "富有""幸福"是主观定义 | 读取"繁荣周期/稳定性潜力"，不定义"足够" |
| **具体的人物身份** | 不可能从星盘推出具体是谁 | 描述人物特征倾向，不说"是某类人" |

## 三、承诺vs激活两步分析法

### 第一步：承诺评估（Promise Assessment）

**问题**：此命盘是否有[主题]的结构性承诺？

**检查清单**（以婚姻为例）：

```
□ 相关宫位（7宫）：宫内星？宫体强度？Bhava Bala评分？
□ 宫主星：位置？入庙/落陷？Shadbala？Ashtakavarga？
□ Karaka（天然象征星）：状态？力量？受克程度？
  → 男性盘：Venus（妻子征象星）
  → 女性盘：Jupiter（丈夫征象星）
□ 分盘确认：D9第7宫？D9 Lagna？D9 7宫主？
□ Jaimini特殊指标：Upapada Lagna（ULA）？Darakaraka（DK）？
□ Yoga组合：有无婚姻相关Yoga？
```

**判定标准**：
- 4+项正面 → **承诺完整存在**
- 2-3项正面 → **承诺部分存在**（可能打折实现）
- 0-1项正面 → **承诺缺失或极弱**

### 第二步：激活评估（Activation Assessment）

**问题**：当前Dasha-Transit是否对齐以激活此承诺？

**检查清单**：

```
□ 当前Mahadasha主星：是否与该宫位/宫主星/Karaka相关？
□ 当前Antardasha主星：是否进一步指向该领域？
□ 当前Pratyantar：是否精确激活？
□ Transit行星：是否正在激活该宫位？
□ 多参考点同步：Lagna+Chandra Lagna+AL+D9 Lagna是否同步？
□ Ashtakavarga Transit评分：该宫位当前AV分数是否有利？
```

**判定标准**：
- 4+项对齐 → **高概率激活**
- 2-3项对齐 → **中等概率激活**
- 0-1项对齐 → **未激活或延迟**

### 第三步：结论矩阵

| | 激活强 | 激活中 | 激活弱 |
|---|--------|--------|--------|
| **承诺完整** | 事件高概率发生 | 可能打折实现 | 潜在但未显化 |
| **承诺部分** | 可能部分实现 | 不确定 | 不太可能 |
| **承诺缺失** | 不会发生（Dasha不能创造不存在的承诺） | 不会发生 | 不会发生 |

## 四、Dridha vs Adridha Karma区分

### 固定业力（Dridha Karma）——可预测性高
- 深层家庭结构模式
- 关键健康敏感点
- 不可避开的转折点
- 核心性格特征
- 反复出现的生命主题

### 可塑业力（Adridha Karma）——预测性降低
- 职业方向的具体选择
- 关系的修复和调整
- 健康问题的预防
- 时机的主动把握

**输出规则**：
- Dridha Karma → 可用"高概率倾向"
- Adridha Karma → 最多"中等概率倾向"，且必须强调"个人选择可改变"

## 五、预测失败的常见原因

当预测与实际不符时，按以下顺序自查：

| 原因 | 概率 | 自检方法 |
|------|------|---------|
| **出生时间不准** | 最高 | 检查±5分钟是否改变Lagna或关键宫主星 |
| **过度依赖单一因素** | 高 | 检查是否只用了1个系统就下结论 |
| **承诺未先确认** | 高 | 检查是否跳过了"承诺评估"直接进入"激活评估" |
| **用真实信息锚定** | 中 | 检查是否先知道用户生活再"找"星盘依据 |
| **多重重叠影响** | 中 | 检查是否有2+个主题同时激活，模糊了单一故事线 |
| **人类自由意志** | 低-中 | 检查用户是否在窗口中做出了非典型选择 |

## 六、标准精度声明（每次完整预测必须包含）

```
⚠️ 精度声明：
- 本分析基于Jyotish推运系统，描述的是[能量场/时机窗口/宫位活跃度]
- 时间窗口精度：±[X周/月]
- 置信度：[高/中/低]
- 具体事件的形态、个人的角色、具体项目方向——超出星盘精度
- 占星给出的是概率和时机，不是保证交付
- 即使窗口完美，个人在窗口中的选择和行动决定最终结果
```
