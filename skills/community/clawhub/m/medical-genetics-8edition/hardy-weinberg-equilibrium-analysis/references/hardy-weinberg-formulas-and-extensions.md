# Hardy-Weinberg公式详解与扩展

## 基础公式推导
对于一对等位基因A和a，设其频率分别为p和q（p + q = 1）。在随机交配下，配子随机结合，子代基因型频率为：
- P(AA) = p × p = p²
- P(Aa) = p × q + q × p = 2pq
- P(aa) = q × q = q²

且 p² + 2pq + q² = (p + q)² = 1。

## 平衡稳定性证明
经一代随机交配后，子代等位基因频率仍为p和q；第二代继续随机交配，基因型频率保持不变，表明系统已达稳定平衡。

## 复等位基因情形
若有三个等位基因A₁、A₂、A₃，频率分别为p、q、r（p+q+r=1），则基因型频率为：
- A₁A₁: p²
- A₂A₂: q²
- A₃A₃: r²
- A₁A₂: 2pq
- A₁A₃: 2pr
- A₂A₃: 2qr
总和为(p+q+r)² = 1。

## X连锁基因处理
- 男性（XY）：基因型频率直接等于等位基因频率（因仅有一个X染色体）
- 女性（XX）：遵循常染色体Hardy-Weinberg公式

## 实际应用场景
1. **携带者频率估算**：若某隐性遗传病发病率（aa）为1/10,000，则q = √(1/10000) = 0.01，携带者频率2pq ≈ 2×1×0.01 = 0.02（即2%）。
2. **基因分型质控**：若观测到的杂合子频率显著低于2pq，可能提示分型错误或近交。
3. **进化研究**：显著偏离H-W平衡可能暗示选择、瓶颈效应或种群混合。

## 不适用情形警示
- 小群体（<100个体）：遗传漂变影响显著
- 近亲婚配：增加纯合子比例
- 存在选择：某些基因型适合度差异大
- 突变率高或近期迁移：破坏等位基因频率稳定性
