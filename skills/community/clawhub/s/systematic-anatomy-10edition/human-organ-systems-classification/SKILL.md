# 人体器官系统的九大分类及其功能

## 使用场景
- 学习系统解剖学或生理学基础知识
- 整理医学课程内容结构
- 理解临床学科划分依据（如内科、外科等）
- 分析某一器官在整体人体功能中的定位

## 执行步骤
1. **确认目标器官**：明确需分类的器官名称。
2. **应用九大系统框架**：根据器官的主要生理功能，将其归入以下九大系统之一：
   - **运动系统**：骨、骨连结、骨骼肌 → 支持体重、保护内脏、产生躯体运动
   - **消化系统**：消化管（口腔至肛门）、消化腺（肝、胰等） → 消化食物、吸收营养、排出代谢废物
   - **呼吸系统**：呼吸道（鼻、咽、喉、气管、支气管）、肺 → 气体交换（吸入氧气，呼出二氧化碳）
   - **泌尿系统**：肾、输尿管、膀胱、尿道 → 排出多余水分及水溶性代谢产物（如尿素、尿酸）
   - **生殖系统**：男性与女性生殖器 → 繁衍后代、分泌性激素
   - **内分泌系统**：垂体、甲状腺、肾上腺等内分泌器官 → 通过激素协调全身各系统活动
   - **脉管系统**：心血管系统（心、血管）和淋巴系统（淋巴管、淋巴器官） → 输送血液与淋巴，维持循环
   - **感觉器**：视网膜、耳蜗等感受器及其附属结构 → 感受内外环境刺激并产生兴奋
   - **神经系统**：中枢神经系统（脑、脊髓）和周围神经系统 → 调控全身各系统器官活动的协调与统一
3. **处理多系统参与器官**：若某器官参与多个系统（如胰腺兼具内分泌与外分泌功能），按其**主要功能**归类。
4. **输出分类结果**：返回器官所属系统、组成结构及核心功能。

## 注意事项
- 本分类基于功能整合原则，适用于人体，不适用于单细胞生物或非人体系统。
- 需预先掌握四种基本组织类型（上皮、结缔、肌肉、神经）。
- 临床实践中，某些器官可能因疾病表现涉及多个系统，但基础分类仍以主要生理功能为准。
