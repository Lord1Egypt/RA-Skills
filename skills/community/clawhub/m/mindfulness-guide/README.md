# Mindfulness Guide

引导短时正念练习，用于日常减压、稳定情绪、恢复注意力

## What it does
- 提供呼吸重置、身体扫描、感官接地、思维观察、睡前放松五种正念练习路径
- 根据用户状态自动选择合适的练习类型和时长（默认3分钟）
- 用短句逐步引导，语言温和非评判，不在练习后过度解读

## Example scenarios

**1. 即时焦虑缓解**
> 👤 我有点焦虑，呼吸很急，能不能带我静一下？
> 🤖 明白。先坐稳一点。慢慢吸气。再慢慢呼气。如果分心了，也没关系。把注意力轻轻带回来…现在和刚才比，有没有一点点变化？

**2. 睡前身体扫描**
> 👤 睡前放松，帮我做个身体扫描
> 🤖 从脚到头逐步引导注意力扫描全身，使用较慢的指令节奏，保持内容紧凑但不压迫，结束时分轻过渡进入休息状态

**3. 地铁通勤时做正念**
> 👤 每天坐地铁上下班各40分钟，车里又挤又吵，能不能教我在这段时间里做点正念？
> 🤖 "地铁正念"三步法：1）站立时感受脚底与地面接触，注意重心转移；2）注意力带到呼吸，吸气数1呼气数2循环1-10；3）打开感官——注意3种声音、2种气味、1种触感。可配合潮汐/小睡眠App的短时正念音频
