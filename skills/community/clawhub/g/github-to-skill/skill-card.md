## Description: <br>
Documentation-only reference guide for converting GitHub projects into OpenClaw skills. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[jijinbao-code](https://clawhub.ai/user/jijinbao-code) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and OpenClaw users use this documentation-only skill to understand how to evaluate a GitHub project, plan manual conversion steps, and package it as an OpenClaw skill. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Using the guide to package a separate GitHub project may carry risks from that external project's license, dependencies, secrets, or code. <br>
Mitigation: Review the external project license, dependency tree, secrets exposure, and source code before installing, running, or publishing the resulting skill. <br>
