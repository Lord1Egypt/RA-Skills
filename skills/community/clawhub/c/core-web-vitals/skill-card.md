## Description: <br>
Guides optimization of Core Web Vitals, including LCP, INP, CLS, page speed, and page experience issues. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, SEO practitioners, and site owners use this skill to diagnose and prioritize Core Web Vitals improvements for loading performance, responsiveness, and visual stability. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad page-speed wording can activate the skill when a user may need a different performance workflow. <br>
Mitigation: Confirm the target metric, data source, and page type before applying recommendations. <br>
Risk: The skill may read local project context files to find a site URL. <br>
Mitigation: Keep sensitive data out of project context files and review those files before using them as inputs. <br>
