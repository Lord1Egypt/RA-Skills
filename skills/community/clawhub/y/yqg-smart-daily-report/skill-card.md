## Description: <br>
Smart Daily Report generates professional daily, weekly, standup, and sprint status reports from project history, task managers, calendars, and user input. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[yqghlx](https://clawhub.ai/user/yqghlx) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Employees, developers, and project contributors use this skill to turn recent work activity into daily reports, weekly summaries, standup notes, sprint reviews, and other status updates. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may inspect project history, task data, calendars, and optional export destinations while preparing reports. <br>
Mitigation: Specify the exact date range, projects, data sources, and export destination before use. <br>
Risk: Broad workspace, home, or desktop scans may include more projects or work traces than intended. <br>
Mitigation: Limit collection to known project directories unless the user explicitly wants all discovered projects included. <br>
Risk: Generated reports can omit context or overstate progress if source data is incomplete. <br>
Mitigation: Review the generated report before sharing and supplement missing work, blockers, or meeting context manually. <br>
