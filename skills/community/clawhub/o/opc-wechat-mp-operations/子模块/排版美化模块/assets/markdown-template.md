# 微信公众号排版模板（Markdown格式）

## 使用说明

本文档提供Markdown格式的排版模板，可以转换为微信公众号编辑器格式使用。

---

## 标题区域

```markdown
# 主标题（不常用）

## 二级标题（推荐）

### 三级标题

#### 四级标题
```

---

## 正文段落

```markdown
正文内容第一段。建议每段不超过150字，保持段落清晰，便于阅读。

正文内容第二段。段落之间空一行，保持良好的阅读节奏。

正文内容第三段。继续撰写内容...
```

---

## 重点标注

### 加粗

```markdown
这是普通段落，**这是重点内容**，继续普通段落。
```

### 斜体

```markdown
这是普通段落，*这是斜体内容*，继续普通段落。
```

### 强调

```markdown
这是普通段落，**这是重点强调的内容**，继续普通段落。
```

---

## 引用

```markdown
> "这是一段引用内容，可以用于引用名言、专家观点、数据来源等。使用引用样式可以让重点内容更加突出。"
```

---

## 列表

### 无序列表

```markdown
要点列表：
- 这是第一个列表项，可以是要点或步骤说明
- 这是第二个列表项，保持格式统一
- 这是第三个列表项，简洁明了
```

### 有序列表

```markdown
步骤说明：
1. 第一步：准备工作
2. 第二步：执行操作
3. 第三步：完成验收
```

---

## 分隔线

```markdown
---
```

---

## 代码块

```markdown
```
这是代码块内容
```
```

---

## 链接

```markdown
[链接文字](https://mp.weixin.qq.com/s/xxx)
```

---

## 表格（不建议在微信中使用）

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容1 | 内容2 | 内容3 |
```

**注意**：微信公众号不完全支持表格，建议使用列表或文本代替。

---

## 完整示例

```markdown
## 文章标题

### 副标题或导语

正文内容第一段。这里可以放置文章的开头部分，建议每段不超过150字，保持段落清晰，便于阅读。

正文内容第二段。可以通过调整字体大小和行间距来优化阅读体验。

这是普通段落，**这是重点内容**，继续普通段落。

> "这是一段引用内容，可以用于引用名言、专家观点、数据来源等。"

要点列表：
- 这是第一个列表项
- 这是第二个列表项
- 这是第三个列表项

步骤说明：
1. 第一步：准备工作
2. 第二步：执行操作
3. 第三步：完成验收

### 小标题

小标题下方的正文内容，继续保持清晰的段落结构。

---

感谢您的阅读，欢迎分享和留言讨论！
```

## 转换为微信编辑器格式

### 方法1：使用Markdown编辑器

1. 使用支持Markdown的编辑器（如Typora、Mark Text等）编写内容
2. 导出为HTML格式
3. 复制HTML代码到微信编辑器的源代码模式
4. 切换回可视化模式进行微调

### 方法2：使用在线转换工具

1. 访问Markdown转HTML在线工具
2. 粘贴Markdown内容
3. 转换为HTML格式
4. 复制HTML代码到微信编辑器

### 方法3：手动转换

参考`assets/wechat-template.html`中的HTML格式，手动将Markdown元素转换为对应的HTML标签：

| Markdown | HTML |
|----------|------|
| `## 标题` | `<h2>标题</h2>` |
| `**加粗**` | `<strong>加粗</strong>` |
| `*斜体*` | `<em>斜体</em>` |
| `> 引用` | `<blockquote>引用</blockquote>` |
| `- 列表项` | `<li>列表项</li>` |
| `---` | `<hr>` |

---

## 配色方案说明

Markdown本身不包含样式，转换时需要手动添加内联样式：

### 标题样式

```html
<h2 style="color: #2E7D32; font-size: 18px; font-weight: bold;">标题</h2>
```

### 正文样式

```html
<p style="color: #333333; font-size: 14px; line-height: 1.75;">正文内容</p>
```

### 引用样式

```html
<blockquote style="background-color: #F1F8E9; padding: 15px; border-left: 3px solid #2E7D32; color: #555555;">
  引用内容
</blockquote>
```

---

## 注意事项

1. **图片处理**：Markdown的图片语法`![描述](URL)`转换为微信编辑器时，需要确保图片URL有效
2. **链接限制**：微信只支持内部链接（mp.weixin.qq.com），外部链接可能被过滤
3. **样式丢失**：Markdown转换为HTML后，需要手动添加内联样式
4. **表格支持**：微信不完全支持表格，建议使用列表或文本代替
5. **空行处理**：确保段落之间有足够的空行，保持良好的阅读节奏

---

## 推荐工作流程

1. 在Markdown编辑器中编写内容（重点在于结构和内容）
2. 转换为HTML格式
3. 在`wechat-template.html`基础上进行样式定制
4. 复制到微信编辑器进行最终调整
5. 预览效果并优化
