# 技能名称：miaoji-model-shot-pro
# 显示名：秒技·模特拍摄Pro版

---

## 定位升维

**敌人**：临时约模特、到现场再想拍什么的"拍到哪算哪"思维

**升维打击**：从"拍几张图"升级到"设计完整视觉战役"

免费版告诉你"建议找个白人女性"，Pro版告诉你"TOP3同类产品的视觉弱点是什么，你的模特应该走什么路线、从哪个角度反超"。

---

## 核心定位

模特拍摄Pro版是免费版的战略升级版，提供完整的拍摄计划、多场景组合方案和同类产品视觉反超策略。通过季节性拍摄日历和同行产品视觉分析，帮助卖家建立差异化的视觉竞争力。

**与免费版的本质区别**：
- 免费版：单次拍摄建议 → 告诉你用什么模特
- Pro版：完整视觉战役 → 告诉你怎么拍、什么时候拍、怎么超越同行

---

## Pro专属功能

### 1. 完整拍摄计划

场景→模特→道具→光线→角度全链路方案：

```
【完整拍摄计划】
━━━━━━━━━━━━━━━━━━━━

📋 拍摄项目：[产品名称]
拍摄周期：[X天]
拍摄地点：[建议]
总预算参考：[$$$]

📸 场景1：白底纯净版
- 目的：Amazon主图、Icon图
- 数量：2-3张，拍摄时长：2-3小时
- 模特：性别[男/女/不限]，年龄段[25-35岁]
- 光线：柔光箱×2 + 顶光，色温5500K
- 角度：45度侧光+正面构图

📸 场景2：生活场景版
- 目的：A+内容、社交媒体
- 数量：4-6张，拍摄时长：4-6小时
- 场景类型：[家居/户外/办公/聚会]
- 光线：窗边自然光优先，黄金时段拍摄

📸 场景3：开箱/展示版
- 目的：开箱视频、细节展示
- 数量：3-4张，拍摄时长：2-3小时
- 要点：开箱节奏、细节特写、配件全家福

📋 模特引导指南
✅ 有效引导语：
- "想象这是你收到礼物的那一刻"
- "用你最舒服的方式拿起来"
- "给我一个'这就是我想要的'表情"

❌ 避免：生硬摆拍、机械指令、过度指导
```

### 2. 季节性拍摄日历

```
【季节性拍摄日历】
━━━━━━━━━━━━━━━━━━━━

🌸 春季（3-5月）
- 踏青郊游、春季焕新、复活节场景
- 准备：春季色系道具、轻薄春装

☀️ 夏季（6-8月）
- 泳池/海边/露营/派对场景
- 注意：避开正午强光，优先黄金时段

🍂 秋季（9-11月）
- 秋景色调、万圣节、感恩节场景
- 注意：感恩节前4周为最佳拍摄期

❄️ 冬季（12-2月）
- 圣诞礼物开箱、新年派对、冬季居家
- 准备：节日道具、室内暖光

📅 关键节点提前量
| 节日 | 建议拍摄时间 | 提前量 |
|------|-------------|--------|
| 情人节 | 12月-1月初 | 6-8周 |
| 复活节 | 2月-3月初 | 8-10周 |
| Prime Day | 4月-5月 | 10-12周 |
| 黑五/网一 | 9月-10月 | 12-14周 |
```

### 4. 同行视觉反超方案

分析TOP3同行产品视觉弱点，生成差异化拍摄策略：

```
【同行产品视觉反超策略】
━━━━━━━━━━━━━━━━━━━━

📊 TOP3同行产品视觉分析

【同行产品A】[ASIN/品类排名]

视觉优势：
- [优势1，如：白人模特专业感强]
- [优势2，如：户外场景光线好]

视觉弱点（机会点）：
- [弱点1]：模特表情偏僵硬，缺乏真实感
- [弱点2]：场景单一，缺少生活化元素

反超策略：
- 方向1：使用多元文化模特
- 方向2：增加家庭场景
- 方向3：强调"真实用户感"

🎯 综合反超路线图

【模特策略】人种多元、年龄多元、身份多元
【场景策略】识别同行未覆盖的场景蓝海
【拍摄风格】光线/色调/构图的差异化

| 优先级 | 反超方向 | 具体行动 |
|-------|---------|---------|
| P0 | [最关键差异] | [行动1/2/3] |
| P1 | [次要差异] | [行动1/2] |
| P2 | [加分项] | [行动1] |
```
