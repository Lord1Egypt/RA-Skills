# 定价规则详解 / Pricing Rules Reference

## 价格体系

所有价格为**出厂含税价**（EXW, VAT included），单位：万元（人民币）。

### 主机定价逻辑

- **永磁变频系列（YDV）**：以功率为主要定价因子，约 2000-2500 元/kW
- **双级压缩系列（YDV-D）**：同功率比单级贵 30-40%，但能效更高
- **工频系列（YD）**：同功率比永磁便宜 25-30%，适合满载工况
- **激光切割系列（YDV-L）**：高压机，同功率溢价 20-25%

### 后处理定价

- 按主机排气量匹配，价格相对固定
- 推荐冷干机+三级过滤器作为标配
- 储气罐为选配件

## 折扣规则

```
折扣阶梯（按设备总价）：
  0 - 10 万    →  0%（标准报价，不打折）
 10 - 30 万    →  3%
 30 - 50 万    →  5%
 50 - 100 万   →  8%
100 万以上    → 10%

特殊折扣（需审批）：
- 年度框架协议折扣：5-15%
- 批量采购折扣（≥3台同型号）：额外 2-5%
- 以旧换新补贴：旧机评估价 2-5 万
```

## 运费计算

```
200km 内：免运费
超出部分：5 元/km → 折算为 0.0005 万元/km
示例：500km → 0.15 + (500-200)×0.0005 = 0.30 万元
偏远地区（西藏、新疆、青海）：基础运费 × 1.5
```

## 安装调试费

标准安装包含：设备就位、管道连接、电气接线、调试运行。

```
安装费 = 折后总价 × 3%，最低 0.15 万元（1500 元）
特殊工况（防爆、洁净室、高层吊装）：另议
```

## 税费

```
增值税率：13%（一般纳税人）
默认为含税报价（include_vat_default: true）
不含税价 = 含税价 ÷ 1.13
```

## 付款条件

| 阶段 | 比例 | 触发条件 |
|------|------|---------|
| 预付款 | 30% | 合同签订 |
| 发货款 | 60% | 设备出厂前 |
| 验收款 | 10% | 安装调试完成，验收合格 |

> 大额项目（>100万）：可协商 30%-30%-30%-10% 分四期。

## 质保与延保

- 标准质保：永磁变频系列 2 年，工频系列 1 年，双级压缩 3 年
- 延保：0.30 万元/年，最多延至 5 年
- 质保范围：主机机头、电机、控制器（不含易损件）

## 备件包

标准备件包（0.18 万元）：
- 空滤芯 ×2
- 油滤芯 ×2
- 油分芯 ×1
- 专用润滑油 20L
