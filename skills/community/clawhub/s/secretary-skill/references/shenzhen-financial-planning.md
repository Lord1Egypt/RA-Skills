# 深圳（南山区）财务规划背景知识

本参考文件提供中国深圳地区的社保、公积金、个税计算基础知识，供 Secretary Skill 处理攒钱/理财类目标时使用。

## 深圳五险一金比例（2025-2026 年参考）

### 养老保险
- **单位**：14% ～ 16%（深户/非深户有差异）
- **个人**：8%
- 基数：上一年度月平均工资，下限 3523 元（最低工资的 60%），上限 26421 元（社平工资 3 倍）

### 医疗保险
深圳分三档，差别主要在单位缴费比例和就医范围：

| 档次 | 个人 | 单位 | 说明 |
|------|------|------|------|
| 一档（综合医保） | 2% | 6%（含生育保险 0.5%） | 南山区通常是一档，有个人账户 |
| 二档（住院医保） | 0.5% | 0.8%（含生育保险） | 无个人账户 |
| 三档（劳务工医保） | 0.2% | 0.55%（含生育保险） | 仅限特定人群 |

### 失业保险
- **个人**：0.3%
- **单位**：0.7%
- 基数同养老保险下限

### 工伤保险
- **个人不交**，单位 0.2%～1.9%（按行业风险）

### 生育保险
- 已并入医疗保险（一档内含）

### 住房公积金
- **个人和单位各 5%～12%**（由公司决定比例，常见 5% 或 12%）
- 基数：上一年度月平均工资，上限约 3.8 万（每年调整）
- 公积金账户资金可提取用于租房、买房、装修

## 个人所得税计算（2025-2026 年参考）

### 免征额
- 每月 5000 元（每年 6 万）

### 综合所得税率表（年度）

| 应纳税所得额（年） | 税率 | 速算扣除数 |
|-------------------|------|-----------|
| 0 - 36,000 | 3% | 0 |
| 36,000 - 144,000 | 10% | 2,520 |
| 144,000 - 300,000 | 20% | 16,920 |
| 300,000 - 420,000 | 25% | 31,920 |
| 420,000 - 660,000 | 30% | 52,920 |
| 660,000 - 960,000 | 35% | 85,920 |
| 960,000+ | 45% | 181,920 |

### 专项附加扣除（每月）
- **子女教育**：每个子女 2,000 元
- **继续教育**：400 元（学历）或 3,600 元/年（职业资格）
- **大病医疗**：实际支出超过 15,000 的部分，上限 80,000/年
- **住房贷款利息**：1,000 元（首套房）
- **住房租金**：深圳属于省会城市标准，约 1,500 元/月
- **赡养老人**：3,000 元/月（独生子女）
- **3 岁以下婴幼儿照护**：每个子女 2,000 元

### 年终奖计税
2027 年前可选择单独计税（除以 12 找税率）或并入综合所得，取低者。

## 计算实用公式（月薪制）

### 税前月扣减
```
五险一金个人部分 = 养老(8%) + 医疗(2% 一档) + 失业(0.3%) + 公积金(5%~12%)

应纳税所得额 = 税前工资 - 五险一金个人部分 - 5000(免征额) - 专项附加扣除

个税 = 应纳税所得额 × 适用税率 - 速算扣除数

税后到手 = 税前工资 - 五险一金个人部分 - 个税 + 公积金单位部分（公积金可提取）
```

### 实际可支配收入估算
注意：**公积金虽然扣了但钱还是你的**（可租房提取）。所以实际 "消失" 的钱只有社保个人部分（养老 8% + 医疗 2% + 失业 0.3%）和个税。

## 常见目标拆解参考

### 三年攒 100 万
月存 ≈ 100万 / 36 = **27,778 元/月**

以深圳月薪 35,000（税前，公积金 5%）为例：
1. 五险一金个人扣减 ≈ 35,000 × (8% + 2% + 0.3% + 5%) = 5,355 元
2. 应纳税所得额 ≈ 35,000 - 5,355 - 5,000 = 24,645 元
3. 月个税 ≈ 24,645 × 20% - 1,410（月速算）= 3,519 元（约，取决于专项附加扣除）
4. 税后到手 ≈ 35,000 - 5,355 - 3,519 = 26,126 元
5. 加上公积金提取（单位+个人 = 3,500）≈ 29,626 元
6. 需存 27,778 元 → 每月生活费预算 ≈ 1,848 元（极度节俭）

**结论：仅靠工资 35K 很难三年攒 100 万。** 需要：
- 提高收入（副业/跳槽涨薪）
- 降低公积金比例不可行（公司决定）
- 或把投资/理财收益算进来

## 提醒
- 以上比例和数字为参考值，具体以深圳市人力资源保障局最新公告为准
- 公积金基数每年 7 月调整
- 个税专项附加扣除需自行在"个人所得税"APP 填报
