# 致良知工作模板

## 使用场景
- 遇到道德/价值观判断
- 做重要决策
- 自我反省

## 步骤

### 1. 静心：回到本心
**方法：** 
- 放下外部干扰
- 深呼吸3次
- 问自己：这件事触动了我什么？

### 2. 问良知：第一反应
**问题：** 我的良知如何判断这件事？

- 这个反应是否被私欲遮蔽？
- [ ] 是 → 排除私欲，回到良知
- [ ] 否 → 信任这个判断

### 4. 依良知行动
**问题：** 良知让我怎么做？

- 正确的行动：____________________
- 可能的阻力：____________________
- 如何克服阻力：____________________

### 5. 反省：事后检验
**问题：** 我是否依良知而行？

- [ ] 我按良知判断行动了吗？
- [ ] 行动后的结果如何？
- [ ] 我心安吗？

## 良知判断准则

| 良知信号 | 含义 | 应对 |
|----------|------|------|
| 心安 | 符合良知 | 继续坚持 |
| 心不安 | 警示信号 | 反思调整 |
| 内疚 | 良知谴责 | 改正错误 |
| 悔恨 | 违背良知 | 补救修正 |

## 注意事项

1. **第一反应最重要** - 良知的判断往往在第一时间出现
2. **排除私欲干扰** - 利益、情绪、偏见会遮蔽良知
3. **信任良知** - 良知是人人本有的，不需要外求
4. **勇于行动** - 知良知而不行，等于不知
