# 急性胰腺炎中MODS的临床表现与病理生理对应关系

## 何时使用本技能
- 患者诊断为重症或中重症急性胰腺炎
- 发病时间 ≥7天 或 存在持续全身炎症反应综合征（SIRS）
- 出现以下任一警示症状：低血压、呼吸困难、少尿、意识障碍、腹胀加剧、黄疸加深、皮肤瘀斑、高热不退、消化道出血等

## 执行步骤

根据患者具体症状，对照以下对应关系判断受累器官及机制：

| 症状/体征 | 受累系统 | 病理生理机制 |
|----------|--------|--------------|
| 低血压、休克 | 循环系统 | 炎性渗出、严重炎症反应、感染 |
| 呼吸困难 | 呼吸系统 | 肺间质水肿、胸腔积液、腹膜炎致膈肌受限 |
| 腹痛腹胀、肠鸣音消失、移动性浊音阳性 | 消化系统 | 肠麻痹、腹腔间隔室综合征 |
| 少尿、无尿 | 泌尿系统 | 肾前性休克、急性肾损伤 |
| 黄疸加深 | 肝胆系统 | 胆总管梗阻、肝损伤或衰竭 |
| Grey-Turner征、Cullen征 | 皮肤 | 胰腺出血坏死致血液外渗 |
| 持续高热 | 全身炎症 | 严重炎症反应或继发感染 |
| 意识障碍、精神异常 | 神经系统 | 胰性脑病 |
| 上消化道出血 | 消化道 | 应激性溃疡、左侧门静脉高压 |
| 猝死 | 心血管系统 | 严重心律失常 |

## 输出结果
- 明确指出受累器官系统
- 提示潜在病理机制
- 触发MODS预警，建议多学科干预

## 注意事项
- 本技能**不适用于轻症急性胰腺炎（MAP）患者**
- 所有判断需结合实验室与影像学证据综合评估
