AutoCAD/Autodesk 清洁卸载工具 v2.1
====================================

## 功能说明

彻底清除Windows系统中AutoCAD、Autodesk系列软件的**所有残留**，包括安装文件、用户配置、注册表、软件卸载信息等。

### 核心特性
- **跨平台兼容**：使用环境变量和相对路径，适配任意Windows电脑和任意用户名
- **智能发现**：自动扫描注册表获取实际安装路径，适配非标准安装目录（如"Autodesk, Inc."）
- **每步确认**：每个危险操作前都会列出具体内容并请求确认
- **透明清晰**：每个步骤都说明作用和必须删除的原因

## 使用场景

- 彻底卸载AutoCAD/Autodesk软件后清理残留
- 软件卸载不完整导致重装失败
- 需要释放磁盘空间（通常可释放5-10GB）
- 换电脑前彻底清除个人软件数据

## 执行前须知

### ⚠️ 危险性说明
本技能涉及以下**不可逆**操作：
- 删除文件目录（可能包含未备份的用户文件）
- 删除注册表项（可能影响系统功能）
- 终止正在运行的进程

### 前置条件
- 建议关闭所有正在使用的程序
- 如有重要CAD文件，请提前备份
- 部分操作需要管理员权限


====================================
第一步：环境扫描（无需确认）
====================================

### 作用说明
在执行清理前，首先全面扫描系统，**从注册表获取实际安装路径**（不仅限于标准Autodesk目录），列出所有待清理项。确保清理前状态透明，避免遗漏。

### 智能发现原理
- 标准目录：`C:\Autodesk`、`C:\Program Files\Autodesk` 等
- 注册表路径：从 `HKLM\Uninstall` 的 `InstallLocation` 字段读取
- 常见变体：`Autodesk, Inc.`、`AutodeskInstallation` 等


====================================
第二步：终止进程（需要确认）
====================================

### 作用说明
终止所有正在运行的Autodesk相关进程。这是清理前的必要准备，因为：
- 正在运行的程序会锁定文件，导致无法删除
- 后台服务可能自动重启或重新创建配置

### 必须删除的原因
如果进程仍在运行，删除文件时会收到"文件被占用"错误，部分文件可能残留。


====================================
第三步：停止服务（需要确认）
====================================

### 作用说明
停止Autodesk相关的Windows后台服务。这些服务包括：
- **AdskLicensing**：Autodesk许可证认证服务
- **Autodesk Desktop App**：Autodesk桌面应用服务

### 必须删除的原因
- 服务会在后台持续运行，消耗系统资源
- 即使删除了程序文件，服务条目仍会留在系统中
- 可能阻止Autodesk产品的重新安装


====================================
第四步：删除文件目录（需要确认）
====================================

### 作用说明
删除所有Autodesk软件安装和运行产生的文件目录，包括：
- 从注册表动态发现的实际安装目录
- 标准Autodesk目录
- 用户数据目录

### 必须删除的原因
| 目录类型 | 删除原因 |
|----------|----------|
| 程序安装目录 | 包含主程序文件和许可证信息 |
| 用户本地数据 | 可能包含5GB+的缓存和临时文件 |
| 用户漫游数据 | 包含个性化配置和用户偏好 |


====================================
第五步：清理注册表（需要确认）
====================================

### 作用说明
删除Windows注册表中所有与Autodesk相关的条目。

### 必须删除的原因
| 注册表项 | 删除原因 |
|----------|----------|
| `HKLM\Uninstall\*` | 软件卸载信息，残留会导致"已安装"的错误显示 |
| `HKLM\SOFTWARE\Autodesk` | Autodesk核心配置，影响许可证管理 |
| `HKCU\SOFTWARE\Autodesk` | 用户个性化配置，包含许可证和偏好设置 |
| `HKCU\Classes\.dwg/.dwt/.dxf` | 文件关联，残留会导致打开方式混乱 |
| `HKCU\Classes\AutoCAD*` | COM类注册，影响程序交互 |
| `appdatalow\Autodesk` | 低完整性数据，缓存和临时文件 |


====================================
第六步：验证清理结果
====================================

验证清理是否完成，检查所有残留是否已被清除。


====================================
一键清理脚本
====================================

一键执行：环境扫描 → 终止进程 → 停止服务 → 删除文件 → 清理注册表 → 验证结果
