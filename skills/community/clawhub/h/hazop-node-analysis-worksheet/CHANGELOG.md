## [0.1.0] - 2026-05-27
Initial release. Six-phase workflow aligned to IEC 61882:2016 and CCPS process-hazard-analysis guidance: Phase 1 study set-up (facility, P&ID and revision, study scope, chair / scribe / discipline roster, design intent for the parent unit, applicable regulatory frame — OSHA PSM 1910.119 / EPA RMP 40 CFR 68 / Seveso III / MOC trigger / re-validation cycle, risk-matrix selection, LOPA-trigger criteria); Phase 2 node definition (single node at a time — node ID, P&ID reference, line / vessel / equipment bounds, normal and design operating envelope for flow / pressure / temperature / level / composition / phase, instrumentation list, isolations, utility ties); Phase 3 design-intent statement (function and target operating envelope for the node, source-of-truth references); Phase 4 deviation analysis (full guideword × parameter matrix — No / Less / More / Reverse / As Well As / Part Of / Other Than against Flow / Pressure / Temperature / Level / Composition / Phase / Reaction / Time / Sequence — with cause, consequence to people / asset / environment / production / reputation, existing safeguards split into prevention vs. mitigation, and severity × likelihood scored on the facility risk matrix); Phase 5 recommendations and LOPA referral (action wording, single named owner, target date, LOPA-trigger flag when residual risk exceeds the corporate tolerance, recommendation type — design change / procedure / training / IPL / further study); Phase 6 node closure and worksheet assembly (DRAFT HAZOP worksheet in the IEC 61882 column layout, recommendation register, deviations-not-credible log, parking-lot list, and chair / scribe / discipline review-and-sign-off block) — for the HAZOP chair and process-safety responsible person's review. Never finalises a PHA, never signs the PSSR, never authorises start-up, and never substitutes for the LOPA / SIL or QRA studies that may follow.
