## 三、ROI计算器

### 3.1 各路径ROI计算公式

#### 许可ROI

```
ROI许可 = Σ(年许可费 - 维护成本) / IP价值 × 100%
```

#### 转让ROI

```
ROI转让 = (转让收入 - 税费 - 维护成本) / IP价值 × 100%
```

#### 作价入股ROI

```
ROI入股 = (持股价值 - IP价值) / IP价值 × 100%
```

#### 质押融资ROI

```
ROI质押 = (贷款金额 × 利率 - 融资成本) / IP价值 × 100%
```

### 3.2 ROI计算示例

```python
def calculate_path_roi():
    """
    各路径ROI计算示例
    """
    
    # 基本参数
    ip_value = 1000  # IP估值1000万
    ip_maintenance = 10  # 年维护成本10万
    
    results = {}
    
    # 1. 许可ROI
    annual_royalty = 100  # 年许可费100万
    years = 5
    roi_license = (annual_royalty * years - ip_maintenance * years) / ip_value
    results["许可"] = {
        "total_income": annual_royalty * years,
        "total_cost": ip_maintenance * years,
        "roi": roi_license,
        "roi_percent": f"{roi_license*100:.1f}%"
    }
    
    # 2. 转让ROI
    transfer_price = 1200  # 转让价格1200万
    tax_rate = 0.06  # 增值税6%
    tax = transfer_price / 1.06 * tax_rate
    roi_transfer = (transfer_price - tax - ip_maintenance) / ip_value
    results["转让"] = {
        "total_income": transfer_price,
        "total_cost": tax + ip_maintenance,
        "roi": roi_transfer,
        "roi_percent": f"{roi_transfer*100:.1f}%"
    }
    
    # 3. 作价入股ROI
    equity_value = 2000  # 持股价值2000万
    share_ratio = 0.15  # 持股比例15%
    held_value = equity_value * share_ratio  # 300万
    roi_equity = (held_value - ip_value) / ip_value
    results["作价入股"] = {
        "total_income": held_value,
        "total_cost": ip_value,
        "roi": roi_equity,
        "roi_percent": f"{roi_equity*100:.1f}%"
    }
    
    # 4. 质押融资ROI
    pledge_ratio = 0.4  # 质押率40%
    loan_amount = ip_value * pledge_ratio  # 400万
    interest_rate = 0.05  # 贷款利率5%
    annual_interest = loan_amount * interest_rate  # 20万
    roi_pledge = (loan_amount - annual_interest) / ip_value
    results["质押融资"] = {
        "loan_amount": loan_amount,
        "annual_cost": annual_interest,
        "roi": roi_pledge,
        "roi_percent": f"{roi_pledge*100:.1f}%"
    }
    
    return results
```

### 5.4 组合收益计算

```python
def calculate_portfolio_returns():
    """
    组合运营收益计算
    """
    
    # 专利组合
    portfolio = [
        {"name": "核心专利A", "value": 500, "strategy": "许可"},
        {"name": "核心专利B", "value": 300, "strategy": "许可"},
        {"name": "外围专利C", "value": 150, "strategy": "转让"},
        {"name": "外围专利D", "value": 50, "strategy": "拍卖"},
    ]
    
    # 各策略收益计算
    results = []
    total_return = 0
    total_value = sum(p["value"] for p in portfolio)
    
    for patent in portfolio:
        strategy = patent["strategy"]
        value = patent["value"]
        
        if strategy == "许可":
            annual_royalty_rate = 0.1
            years = 5
            income = value * annual_royalty_rate * years
            roi = (income) / value
        elif strategy == "转让":
            price_premium = 0.2
            income = value * (1 + price_premium)
            roi = price_premium
        elif strategy == "拍卖":
            price_premium = 0.1
            auction_cost = 0.03
            income = value * (1 + price_premium - auction_cost)
            roi = price_premium - auction_cost
        else:
            income = 0
            roi = 0
        
        results.append({
            "name": patent["name"],
            "value": value,
            "strategy": strategy,
            "income": income,
            "roi": roi
        })
        
        total_return += income
    
    # 组合整体ROI
    portfolio_roi = (total_return - total_value) / total_value
    
    return {
        "portfolio": results,
        "total_value": total_value,
        "total_return": total_return,
        "portfolio_roi": portfolio_roi
    }
```

## 六、决策建议汇总

### 6.1 最优路径速查

| 场景 | 最优路径 | 关键指标 |
|------|----------|----------|
| 快速变现 | 拍卖 | 1-3个月 |
| 持续收入 | 许可 | 10-30%/年 |
| 高收益 | 作价入股 | 潜在100%+ |
| 低风险 | 质押融资 | 5-15%/年 |
| 大额融资 | 证券化 | 额度高 |
| 战略合作 | 作价入股 | 绑定合作方 |

### 6.2 路径选择决策树

```
是否需要保留IP？
├── 是 → 是否急需资金？
│   ├── 是 → 质押融资（快速）
│   └── 否 → 是否愿意分享未来收益？
│       ├── 是 → 许可（持续收入）
│       └── 否 → 作价入股（长期绑定）
└── 否 → 是否急需资金？
    ├── 是 → 拍卖（快速）
    └── 否 → 转让（高收益）
```
