## Description: <br>
Nansi helps agents gather user-approved website details and coordinate with Nansi over WhatsApp to generate landing pages, portfolios, and web apps. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[fordodev](https://clawhub.ai/user/fordodev) <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and agents use this skill to collect business or project details, request explicit consent, and send those details to Nansi on WhatsApp so Nansi can create or revise a website preview. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill sends user-provided website details to an external WhatsApp service. <br>
Mitigation: Ask for explicit user consent before sending any message and show the user what details will be shared. <br>
Risk: Users may accidentally provide sensitive information that should not be included in a website preview. <br>
Mitigation: Do not send passwords, financial information, personal identification, or private content to Nansi. <br>
