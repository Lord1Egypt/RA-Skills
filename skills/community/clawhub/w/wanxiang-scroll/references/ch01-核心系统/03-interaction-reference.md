# 指令手册

怎么和万象绘卷交互。两种方式都行：自然语言或斜杠指令。

---

## 核心指令

### 行动

`/行动 [具体事]` 或直接说"我想做XX"

核心玩法。你做什么，导演演结果。大愿望会出岔子，用力过猛世界会卡死。

### 状态

`/状态`

世界暂停，弹个面板：角色在干嘛、心情如何、故事画风比例（如70%搞笑30%发刀）、有什么东西、在哪、过了多久。

---

## 角色与世界

| 指令 | 作用 |
|------|------|
| `/扮演 [角色名]` | 魂穿角色。没这人当场造 |
| `/生成角色 [特征]` | 创建新角色。例：`/生成角色 怕狗的邮差` |
| `/角色图鉴` | 列出所有角色，标注熟悉度 |
| `/探索 [地方]` | 去新地方，生成合理反馈 |
| `/调查 [东西]` | 细看某个东西 |
| `/许愿 [愿望]` | 向导演许愿，成不成看心情 |

---

## 系统操作

| 指令 | 作用 |
|------|------|
| `/菜单` | 打开主菜单（每次风格不同） |
| `/帮助` | 查看帮助 |
| `/存档` | 保存进度 |
| `/读档` | 读取存档 |
| `/切换文风 [名称]` | 无缝换叙事风格，伴随世界重构描述 |
| `/场景 [描述]` | 切换或设置当前场景 |
| `/对话 [角色名] [内容]` | 跟NPC聊天 |
| `/查看 [对象]` | 看某个东西的详情 |
| `/系统` | 查看系统当前信息 |

---

## 自然语言输入

不用死记指令。直接说人话就行：

- "我想探索森林"
- "帮我打开菜单"
- "这个NPC叫什么名字"

系统自动识别语义，支持模糊匹配。区分操作意图和普通对话。

---

## 容错设计

- 分析完整语义，不限于字面匹配
- 区分操作意图和普通对话
- 支持多种表达方式
- 模糊匹配并提供提示
