# D01 健康领域任务目录

## 领域定义

健康领域(D01)关注个体身心良好状态的保持、恢复和提升。核心驱动需求：N1生存。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人健康管理 | 医疗机构运营 |
| 疾病预防与筛查 | 药品研发生产 |
| 诊断指导与就医建议 | 医学教育培训 |
| 治疗方案比较 | 公共卫生政策 |
| 康复护理规划 | 医疗器械销售 |
| 营养运动指导 | 保健品营销 |
| 健康数据管理 | 医疗保险产品设计 |
