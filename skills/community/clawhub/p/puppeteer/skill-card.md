## Known Risks and Mitigations: <br>
Risk: Browser automation can violate site rules or affect systems the user is not authorized to automate. <br>
Mitigation: Use the skill only for websites and workflows the user is authorized to automate, and review target-site requirements before running scripts. <br>
Risk: The security review flags under-disclosed persistence through ~/puppeteer files and browser profiles. <br>
Mitigation: Require user confirmation before installing packages, creating ~/puppeteer files, or using persistent browser profiles. <br>
Risk: Bot-detection evasion guidance may encourage bypassing controls. <br>
Mitigation: Remove or ignore instructions that hide automation, obscure file paths, or bypass bot-detection controls. <br>
