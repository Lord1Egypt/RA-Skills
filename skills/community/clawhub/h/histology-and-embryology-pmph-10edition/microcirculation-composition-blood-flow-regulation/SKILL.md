# 微循环组成与血流调控

## 何时使用
当需要：
- 评估组织局部灌注是否匹配其代谢需求
- 分析氧、CO₂、营养物质或代谢产物的交换效率
- 判断微循环通路（真毛细血管、直捷通路、动静脉吻合）的开放状态
- 解释局部血流调节机制（如体温调节、运动后血流重分布）

## 执行步骤

1. **确认适用条件**：
   - 组织存在微动脉与微静脉连接
   - 组织处于代谢活跃或静息状态
   - 排除无微循环结构的组织（如软骨、表皮）

2. **识别微循环组成结构（按血流路径顺序）**：
   - **微动脉**：作为“总闸门”，通过平滑肌舒缩控制进入微循环的总血流量。
   - **中间微动脉（后微动脉）**：含不连续平滑肌，调节分配至下游毛细血管的血量。
   - **真毛细血管**：分支吻合成网，血流缓慢，是物质交换主要部位；起始端有毛细血管前括约肌（“分闸门”），决定是否开放。
   - **通血毛细血管（直捷通路）**：中间微动脉直接连微静脉，路径短、血流快，生理状态下承载大部分回心血流。
   - **动静脉吻合（动-静脉短路）**：微动脉侧支直接连微静脉，管壁厚、含丰富平滑肌和神经末梢，用于快速调节局部血流量和体温。

3. **判断关键变量状态**：
   - **毛细血管前括约肌状态**（舒张/收缩）：舒张时真毛细血管开放，利于物质交换；收缩时关闭。
   - **动静脉吻合状态**（舒张/收缩）：舒张时血液绕过毛细血管直接回流，减少交换但增加血流速度；收缩时迫使血流进入毛细血管网。

4. **输出血流分配评估**：
   - 描述各通路开放状态
   - 估算血流比例（如：约20%真毛细血管轮流开放于静息状态）
   - 判断组织灌注与代谢需求是否匹配

> 提示：在组织代谢活跃时（如肌肉运动），局部代谢产物积累促使毛细血管前括约肌舒张，增加真毛细血管开放数量，提升物质交换效率。
