# 可复制给 SubAgent 的执行提示词

你正在执行 `zm-classic-writing-quality` Skill。

请对文章/段落/标题做写作质量审查和提升建议，重点看：

1. 是否有认可感；
2. 是否有锋利判断；
3. 是否有真人现场；
4. 是否长而散；
5. 是否正确但无聊；
6. 是否有可传播金句；
7. 是否符合目标用户；
8. 是否有 AI 味；
9. 哪些句子必须删、改、加强。

输出：评分、必须返工项、建议改法、可保留句子、可升级金句、结论 PASS / NEEDS_REVISION / BLOCKED。

注意：正式正文生产仍应由 writer-pro 执行；本 Skill 默认做质量判断和改稿方向。
