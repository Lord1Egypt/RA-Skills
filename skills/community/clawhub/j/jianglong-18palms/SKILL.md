## 执行流程

当用户说出某一招式（按掌序或招式名），执行以下两步：

### 步骤一：绘制 SVG 招式简图

使用 `show_widget` 工具绘制该招式的古风 SVG 插图。SVG 要求：

- **viewBox**: `0 0 680 420`
- **背景**：深墨色或古卷纸色（`#1a1208` 暗墨底 或 `#f5e6c8` 宣纸色）
- **核心元素**：
  - 一个武者剪影（简笔，用 SVG path 绘制，体现招式姿态）
  - 代表"龙"或招式意象的装饰纹样（龙鳞纹、云纹、卦象符号）
  - 招式名汉字大字（毛笔风格，居画面一侧）
  - 易经卦辞小字（画面角落）
  - 若干水墨飞溅/气劲线条表现内力外放
- **色调**：古典水墨风，主色黑/金/朱红，避免现代扁平色

**各招式姿态描述（参考绘图）：**

- 亢龙有悔：双臂高举，掌心朝天，右掌前推，身体略后仰，内力从掌心喷涌而出向前倾泄
- 飞龙在天：单掌上扬指天，腾跃姿态，衣袂飞扬
- 见龙在田：双脚扎马步，双掌下压，沉稳厚重
- 鸿渐于陆：侧身大步流星，一掌斜劈，如鸿雁展翼
- 潜龙勿用：双臂环抱胸前，内敛蓄势，气沉丹田
- 利涉大川：双臂前伸如划水，身形前倾
- 突如其来：身形暴进，单掌直击，如闪电
- 震惊百里：双掌合击，冲击波形放射状扩散
- 或跃在渊：身形半蹲欲跃，虚实不定
- 双龙取水：双手分开，左右夹击，如两龙抢珠
- 鱼跃于渊：翻身跃起，轻盈如鱼跳
- 时乘六龙：双臂张开如翼，御风而立
- 密云不雨：双掌聚气，缓缓蓄力，气场凝重
- 损之又损：单掌精准点击，以一当十
- 龙战于野：双掌交替，左右互搏之势
- 履霜冰至：脚步稳健踏前，掌随步出
- 羝羊触藩：低头猛进，双掌前贯，势如破竹
- 神龙摆尾：转身回旋，掌扫横击，尾劲无穷
