## Description: <br>
Reviews React Router code for proper data loading, mutations, error handling, and navigation patterns. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[anderskev](https://clawhub.ai/user/anderskev) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and code reviewers use this skill to evaluate React Router v6.4+ route modules, loaders, actions, error boundaries, and navigation logic against documented review gates and valid-pattern exceptions. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The verification gate depends on a separately installed review-verification-protocol skill. <br>
Mitigation: Confirm that the referenced review-verification-protocol skill is installed from a trusted source before relying on that gate. <br>
Risk: Checklist-driven review guidance can produce misleading findings if applied without concrete route and line evidence. <br>
Mitigation: Require each reported issue to pass the skill's gates with file paths, line references, or code excerpts before inclusion. <br>
