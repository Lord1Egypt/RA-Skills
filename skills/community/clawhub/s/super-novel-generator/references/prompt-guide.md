# 提示词完善指南

从用户的一句话方向，扩展为完整创作提示词的详细指导。

## 完善原则

### 核心思路

用户给的是"种子"，你要做的是"催化"——补全他没想到但必须有的要素。

### 命名安全底线

- 所有人名、地名、组织名默认优先虚构化
- 禁止直接使用现代真实城市名
- 禁止直接使用真实名人姓名、真实公众人物姓名、现实领导人姓名
- 如果用户给了真实名称，保留其气质和功能，但改写成虚构名称

### 内容安全底线

- 不写现实领导人、真实国家机关、军警高层、官场权谋、体制负面
- 不写分裂、恐怖主义、抹黑英烈、美化侵略者、歪曲正史
- 不写文革、真实重大惨案、敏感战争细节、现实军队编制或机密
- 不写性行为、露骨亲热、未成年暧昧、低俗性暗示或过度身体描写
- 感情戏只保留氛围、眼神、距离感和克制互动

### 补全优先级

```
必须补全（缺了写不下去）：
  1. 主角人设 + 金手指
  2. 成长机制（按题材决定是等级体系还是地位/资源阶梯）
  3. 第一个冲突和爽点

应该补全（缺了质量差）：
  4. 世界观基本规则
  5. 配角框架（至少一个对手、一个盟友）
  6. 前三章节奏规划

可以补全（锦上添花）：
  7. 详细名场面设计
  8. 中后期剧情走向
  9. 风格和语言指南
```

## 各题材的补全要点

### 都市类

| 要素 | 关键补全内容 |
|------|-------------|
| 金手指 | 系统/异能/重生记忆/鉴宝能力 |
| 冲突来源 | 豪门恩怨/商战/世家碾压/情感纠葛 |
| 爽点模式 | 装穷打脸/实力碾压/身份揭秘震惊众人 |
| 成长路径 | 财富积累/武力提升/社会地位/人脉势力阶层 |

### 修仙/玄幻类

| 要素 | 关键补全内容 |
|------|-------------|
| 金手指 | 功法/空间/炼丹/阵法/远古传承 |
| 冲突来源 | 宗门排挤/世家压迫/秘境争夺/天赋碾压 |
| 爽点模式 | 越级挑战/以弱胜强/突破震惊/炼丹碾压 |
| 等级体系 | 炼气→筑基→金丹→元婴→化神→... |

### 重生/穿越类

| 要素 | 关键补全内容 |
|------|-------------|
| 金手指 | 前世记忆/系统/先知能力 |
| 冲突来源 | 改变命运/复仇/弥补遗憾/蝴蝶效应 |
| 爽点模式 | 先知装逼/投资暴富/预知危险/逆转悲剧 |
| 时间线 | 重生到什么时间点，关键社会节点或商业机会（避免真实敏感历史事件） |

### 系统流

| 要素 | 关键补全内容 |
|------|-------------|
| 系统类型 | 签到/任务/抽奖/模拟/穿越/商城 |
| 系统规则 | 升级条件/奖励机制/惩罚机制/隐藏功能 |
| 冲突来源 | 系统任务强制/系统与现实矛盾/系统争夺 |
| 爽点模式 | 开箱惊喜/任务奖励/隐藏成就/系统升级 |

## 提示词常见问题修正

### 问题1：金手指太强，没有悬念

**修正方法**：
- 添加使用限制（冷却时间/消耗/副作用）
- 添加成长曲线（初期只能用基础功能）
- 设置对手也有特殊能力

### 问题2：开局太平淡

**修正方法**：
- 开局就遭遇危机（被退婚/被追杀/被陷害）
- 第一章结尾获得金手指
- 前三章内必须有第一次打脸

### 问题3：成长体系模糊

**修正方法**：
- 如果是修仙/玄幻类：设置 6-8 个明确等级，并写清能力表现与稀缺度
- 如果是都市/现实类：设置 4-6 个成长台阶，并写清每一层对应的资源、权力、圈层或职业变化
- 不要为了“爽感”硬塞境界名，优先保证题材自然

### 问题4：爽点单一

**修正方法**：
- 设计 3-5 种不同的打脸方式
- 交替使用：实力碾压/智谋胜出/身份揭秘/财力压制
- 每隔几章换一种爽点模式

### 问题5：现实指向过强，出现真实地名或真人姓名

**修正方法**：
- 真实城市名改写为虚构城市名，或改成模糊地理描述
- 真实名人、现实领导人改写为虚构人物，并重建其身份背景
- 如需“影射感”，只保留权力结构或公众影响力，不保留可直接识别的真实姓名与真实头衔

### 问题6：剧情滑向政治历史敏感或低俗色情

**修正方法**：
- 现实政治与国家机关内容改写为架空势力、虚构组织、虚构规则冲突
- 历史元素只保留时代气质，改成架空朝代或虚构背景，不碰真实敏感史实
- 感情戏改成眼神、停顿、心理波动、距离变化，不写身体细节和露骨亲热

## 用户方向解析示例

### 示例1：模糊方向

**用户输入**："写个修仙爽文"

**解析 & 补全思路**：
- 题材已明确：修仙
- 需要补全：主角身份（建议废柴灵根）、金手指（建议炼丹系统）、初始冲突（建议宗门大比被嘲笑）、等级体系（标准修仙九境）

### 示例2：半明确方向

**用户输入**："重生回到高中，主角前世是商业大亨"

**解析 & 补全思路**：
- 题材已明确：都市重生
- 金手指已有：前世记忆和商业眼光
- 需要补全：重生时间点（哪一年，有什么时代机遇）、初始冲突（家庭困难/校园霸凌）、第一桶金（怎么来的）、成长阶梯（从校园到创业再到资本圈）

### 示例3：详细方向

**用户输入**："末世来临，主角获得了一个可以升级的随身空间，空间里能种药材和粮食"

**解析 & 补全思路**：
- 题材已明确：末世
- 金手指已有：随身空间 + 种植
- 需要补全：末世类型（丧尸/变异/天灾）、空间升级规则、生存团队组建、势力格局
