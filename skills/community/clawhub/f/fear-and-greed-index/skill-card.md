## Description: <br>
Access the Alternative.me Crypto Fear & Greed Index to anchor baseline crypto market sentiment analysis with the current score and classification. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and developers use this skill to fetch the Alternative.me Crypto Fear & Greed Index and ground broader crypto sentiment analysis in a daily public market sentiment signal. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The sentiment index could be over-weighted or mistaken for financial advice. <br>
Mitigation: Treat the returned index as one analysis input alongside independent market research and risk review. <br>
Risk: Using the skill requires an unauthenticated request to Alternative.me for public sentiment data. <br>
Mitigation: Use it only when external public API access is acceptable for the agent's environment. <br>
