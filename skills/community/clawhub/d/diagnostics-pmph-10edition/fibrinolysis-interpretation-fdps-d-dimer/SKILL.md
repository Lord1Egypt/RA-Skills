# 纤溶亢进的FDPs与D-二聚体联合判读

## 触发条件
当患者出现以下情况时启用本技能：
- 出血伴血栓表现
- 术后广泛或难以控制的出血
- 临床高度怀疑纤溶亢进或DIC

## 执行步骤
1. 获取患者的FDPs（纤维蛋白(原)降解产物）和D-二聚体检测结果。
2. 根据两者水平组合，按以下四类进行判读：

   **情况1：FDPs正常 + D-二聚体正常**
   - 判定：纤溶活性正常
   - 解释：出血原因很可能与纤溶系统无关；优球蛋白溶解时间（ELT）通常正常

   **情况2：FDPs升高 + D-二聚体正常**
   - 判定：理论上提示原发性纤溶（仅纤维蛋白原被降解）
   - 注意：实际多为FDPs假阳性，常见于肝病、手术出血、重型DIC早期、剧烈运动或类风湿关节炎

   **情况3：FDPs正常 + D-二聚体升高**
   - 判定：理论上提示继发性纤溶（仅纤维蛋白被降解）
   - 注意：实际多为FDPs假阴性，常见于DIC、静脉/动脉血栓或溶栓治疗

   **情况4：FDPs升高 + D-二聚体升高**
   - 判定：纤维蛋白原和纤维蛋白同时被降解，典型见于继发性纤溶
   - 最常见于DIC或溶栓治疗后；ELT通常延长

3. 结合临床背景（如创伤、感染、肿瘤、术后状态）综合判断。
4. 若提示继发性纤溶或DIC可能，建议进一步检测3P试验、可溶性纤维蛋白单体复合物（SFMC）等。

## 关键参考值
- FDPs参考区间：<5 mg/L
- D-二聚体（ELISA法）参考区间：0～0.256 mg/L

> **注意**：D-二聚体阴性可基本排除深静脉血栓（DVT）和肺栓塞（PE），但陈旧性血栓可致假阴性，故其特异度低、灵敏度高。
