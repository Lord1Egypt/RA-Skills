## Use Case: <br>

### Deployment Geography for Use: <br>

## Known Risks and Mitigations: <br>
Risk: Suggested Python execution and session-spawn actions can modify the workspace. <br>
Mitigation: Review proposed commands before running them and execute only actions that match the intended project changes. <br>
Risk: Research workflows may involve private datasets, secrets, or untrusted code. <br>
Mitigation: Avoid exposing sensitive inputs and inspect third-party code or data before using it in analysis or generated projects. <br>
