# 基于年龄的惊厥病因评估

## 何时使用
- 患儿出现惊厥发作
- 已知确切发病年龄
- 需在急诊或初诊场景下快速形成病因假设

## 执行步骤

### 第一步：按年龄分段确定常见病因
根据患儿发病年龄归入以下四类，列出该年龄段最可能的3–5种病因（按可能性排序）：

- **新生儿期（0～28天）**：
  - 产伤、窒息
  - 先天颅脑畸形
  - 低血糖症、低钙血症
  - 脓毒症、化脓性脑膜炎
  - 破伤风

- **1个月～1岁**：
  - 围产期损伤后遗症
  - 先天颅脑畸形
  - 低钙血症
  - 化脓性脑膜炎
  - 婴儿痉挛
  - （6个月后）热性惊厥逐渐增多

- **1～3岁**：
  - 热性惊厥（最常见）
  - 各种脑膜炎和脑炎
  - 中毒性脑病
  - 低血糖

- **学龄前期及学龄期（>3岁）**：
  - 中毒性脑病
  - 脑膜炎/脑炎
  - 颅内肿瘤
  - 颅脑外伤
  - 中毒
  - 高血压脑病
  - 癫痫

### 第二步：结合辅助判断要素细化鉴别
- **是否发热**：
  - 发热 → 优先考虑感染性病因（如CNS感染、热性惊厥、中毒性脑病）
  - 无热 → 警惕代谢异常、结构性病变或癫痫
- **惊厥后意识恢复速度**：
  - 快速恢复（数分钟内）→ 支持热性惊厥
  - 持续昏睡或疲乏 → 提示脑炎、脑病等
- **惊厥类型**：
  - 局灶性发作 → 提示结构性病变（如肿瘤、外伤、皮质发育异常）
  - 全面性强直-阵挛发作 → 多见于代谢紊乱或癫痫性病因

### 第三步：输出结果
生成一份按可能性排序的病因列表（3–5项），并明确建议下一步关键检查（如血糖、电解质、腰穿、头颅影像等）。

> **注意**：年龄是首要线索，但必须结合完整病史、体格检查及必要辅助检查综合判断，不可单独依赖年龄诊断。
