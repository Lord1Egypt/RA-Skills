## Description: <br>

This skill is ready for commercial/non-commercial use. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Future updates may add real Qwen or OpenAI network calls and API-key handling. <br>
Mitigation: Audit updated code before deployment and provide API keys only through environment variables after review. <br>
Risk: The current package may install the OpenAI Python client even though the script currently runs as a local prompt-generation helper. <br>
Mitigation: Review dependency installation in the target environment and confirm that no unexpected network calls are introduced. <br>
