# 示例与边界情况

## 典型示例
- **pembrolizumab（帕博利珠单抗，页码450）**：PD-1抑制剂，通过阻断免疫检查点增强T细胞抗肿瘤活性 → 属于免疫增强型单抗。
- **rituximab（利妥昔单抗，页码440, 466）**：靶向CD20的B细胞耗竭剂，用于自身免疫病和淋巴瘤 → 属于免疫抑制/调节型单抗。
- **trastuzumab（曲妥珠单抗，页码440）**：主要作用为靶向HER2信号通路，免疫调节作用次要；但在本教材编排中仍列于免疫相关章节附近，需结合正文判断其免疫影响。

## 抗体偶联药物（ADC）处理
以下ADC也应纳入考量，因其载体可能影响免疫微环境：
- **trastuzumab deruxtecan（德曲妥珠单抗，页码448）**
- **polatuzumab vedotin（维泊妥珠单抗，页码447）**

## 排除项
- 小分子靶向药（如伊马替尼）
- 疫苗（如HPV疫苗）
- 非抗体类生物制剂（如干扰素、白介素）
