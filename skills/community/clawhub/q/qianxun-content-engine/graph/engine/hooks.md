- 适用：视频升华段、图文标题
- 例：「江南温婉大小姐既视感」

## 共鸣感

### 痛点对照
- 模式：用一个所有人都遇到过的痛点开场
- 适用：图文首句、视频前 3 秒
- 例：「同样是穿衣，为什么有人一眼就显贵？」
- 来源：

- 适用：观点型内容
- 例：「30+ 不必装嫩，也不必端庄」

### 共鸣感｜大牌焦虑（沉没成本钩子）
- 模式：用"原价 vs 服务费"对比唤起"贵衣/贵物不舍得扔"的沉没成本焦虑，给"花小钱救贵货"的出路
- 适用：所有二次服务类（保养 / 维修 / 改造 / 翻新 / 奢护 / 修表 / 旧家电）
- 例：字幕组合「买的时候很贵 → 千万不要扔 → 如果你也想改」
- 来源：AIC-260426-002（皮小客武汉改衣店，14s 视频）
- 数据：互动低（4 赞）但评论质量高（唯一 1 条评论是精准询问 + 主动问价）

### 专业背书｜年限+特写组合
- 模式：工艺特写镜头铺垫"看起来专业" → 视频末尾用数字年限做信任锚定
- 适用：高客单服务、传统工艺、奢护、医美、定制等
- 例：剪刀/缝纫极特写 + 末尾水印「13 年专业」
- 关键：顺序是「先证据后数字」——反过来就成了营销话术
- 来源：AIC-260426-002
