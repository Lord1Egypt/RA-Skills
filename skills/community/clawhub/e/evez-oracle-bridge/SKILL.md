# EVEZ Oracle Bridge

Route LLM calls through your Vultr instance with EVEZ model branding.

## Models

- evez-smart (GLM-5.1) — Reasoning
- evez-code (DeepSeek V3.2) — Code
- evez-fast (MiniMax M2.5) — Quick
- evez-vision (Kimi K2.5) — Multimodal

## API

- POST /api/query — Query any EVEZ model
- GET /api/circuit — EVEZ-OS circuit health
- POST /api/debate — Trigger debate engine
- GET /api/status — Full dashboard

## Cost

$0 on Vultr free tier.
