## Description: <br>
Guides personal OpenClaw users through a minimal PowerMem long-term memory setup that reuses their existing OpenClaw model configuration. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[teingi](https://clawhub.ai/user/teingi) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External personal OpenClaw users use this skill to install PowerMem and the memory-powermem plugin, verify that the gateway loads the plugin, and test long-term memory health. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The install flow recommends running an unpinned remote shell installer with automatic approval. <br>
Mitigation: Download and inspect the installer before execution, and prefer a pinned release or verified checksum when available. <br>
Risk: Long-term memory may store sensitive information that a user did not intend to retain. <br>
Mitigation: Avoid storing sensitive information unless the user knows how to inspect, disable, or delete stored memories. <br>
