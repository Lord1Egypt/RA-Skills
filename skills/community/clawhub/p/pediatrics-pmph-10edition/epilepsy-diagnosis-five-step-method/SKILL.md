# 癫痫诊断五步法

执行以下五个步骤以完成癫痫的系统性诊断：

1. **确认发作是否为癫痫性**：排除晕厥、心因性非癫痫发作、短暂性脑缺血发作等非癫痫性事件。
2. **确定癫痫发作类型**：根据国际抗癫痫联盟（ILAE）分类标准，判断是局灶性发作、全面性发作还是未知起源发作。
3. **确定癫痫综合征**：结合年龄、发作特征、脑电图模式和影像学结果，识别是否存在特定癫痫综合征（如青少年肌阵挛癫痫、Lennox-Gastaut综合征等）。
4. **寻找病因**：通过病史、体格检查、神经影像（如MRI）和实验室检查，明确潜在病因（结构性、遗传性、代谢性、免疫性、感染性或未知）。
5. **评估共患病与功能影响**：识别认知、精神、行为等共患病，并评估癫痫对患者生活质量、学习或工作的影响。

在每一步中，整合临床信息、辅助检查结果和诊断标准，确保诊断过程逻辑严密、可追溯。若任一环节信息不足，应建议补充相应检查或转诊至癫痫专科。
