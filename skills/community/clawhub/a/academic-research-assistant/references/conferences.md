## 一、会议类型

### 按范围分类

| 类型 | 特点 | 适用范围 |
|:---:|------|---------|
| 国际会议 | 全球范围征稿，语言多为英文 | 国际交流、争取发表 |
| 国内会议 | 全国范围，语言多为中文 | 国内学术圈交流 |
| 区域性会议 | 省/市/校际范围 | 小范围交流、初学者投稿 |
| 专题研讨会 | 聚焦特定主题 | 深度交流、专家讨论 |

### 按论文处理方式分类

| 类型 | 论文出版 | 推荐度 |
|:---:|:-------:|:-----:|
| 有论文集（Proceedings） | 被会议论文集收录，部分被EI/CPCI检索 | ⭐⭐⭐⭐ |
| 仅摘要 | 仅收录摘要，不出版全文 | ⭐⭐⭐ |
| 推荐期刊发表 | 优秀论文推荐至合作期刊发表 | ⭐⭐⭐⭐⭐ |

## 二、会议论文与期刊论文对比

| 对比维度 | 会议论文 | 期刊论文 |
|:-------:|:--------:|:--------:|
| 出版周期 | 3-12月 | 6-24月 |
| 篇幅 | 4-15页 | 10-30页 |
| 深度 | 初步成果 | 完整研究 |
| 评审 | 一轮评审为主 | 多轮修改 |
| 检索 | 部分被EI检索 | SCI/SSCI/AHCI |
| 学术权重 | 计算机领域较高 | 所有领域 |

## 三、会议投稿流程

```
1. 查找合适的会议
   ├── 查找领域知名会议
   ├── 检查截稿日期（Deadline）
   ├── 确认会议地点和时间
   └── 检查往届论文质量

2. 准备会议投稿
   ├── 按会议要求排版（通常为模板格式）
   ├── 准备摘要和全文（视会议要求）
   ├── 部分会议需要匿名投稿（Blind Submission）
   └── 准备论文编号准备提交

3. 提交和审稿
   ├── 在线投稿系统提交
   ├── 通常为2-3位审稿人
   ├── 审稿周期：1-3个月
   └── 收到后准备修改或参加

4. 参会准备
   ├── 注册会议（早鸟价通常更便宜）
   ├── 准备演讲PPT
   ├── 准备海报（如Poster Session）
   ├── 办理签证（海外会议）
   └── 预订机票和住宿
```

## 四、参会实用建议

### 演讲Tips
1. **PPT简洁**：每页不超过5个要点
2. **时间控制**：一般15-20分钟的演讲，准备12-15分钟内容
3. **提前演练**：对着计时器练习至少2遍
4. **准备问答**：预判可能的3-5个问题
5. **语言选择**：国际会议使用英文，国内会议使用中文

### 社交建议
1. **提前做好功课**：查找参会的学者和同行
2. **准备好自我介绍**：30秒内的"电梯演讲"
3. **提问环节**：至少提问1次
4. **茶歇交流**：认识同行最好的时机
5. **交换联系方式**：ResearchGate/邮箱/LinkedIn

### 经济策略
1. **早鸟注册**：可节省20-30%注册费
2. **学生优惠**：研究生通常有优惠注册费
3. **会议志愿者**：部分会议志愿者免注册费
4. **申请学校资助**：很多高校有会议资助
