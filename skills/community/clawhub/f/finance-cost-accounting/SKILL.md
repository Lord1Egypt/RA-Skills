## 1. 成本核算基础

### 1.1 成本核算的目的和原则

#### 成本核算的目的
| 目的 | 说明 |
|------|------|
| 定价决策 | 为产品/服务定价提供成本依据 |
| 成本控制 | 识别浪费，挖掘降本空间 |
| 绩效考核 | 评价各部门的成本管理绩效 |
| 经营决策 | 支持产品线取舍、投资决策 |
| 财务报告 | 准确计量存货价值和利润 |

#### 成本核算的基本原则
1. **权责发生制原则**：成本费用应按其实际发生期间确认，而非款项支付期间
2. **配比原则**：收入与成本应在同一会计期间匹配
3. **一致性原则**：成本核算方法一旦选定，不得随意变更
4. **重要性原则**：对重要事项详细核算，次要事项简化处理
5. **实际成本原则**：原则上按实际成本计价，特殊情况可用标准成本

### 1.2 成本对象

#### 成本对象分类
```
成本对象
├── 产品成本（制造业/农业）
│   ├── 产成品
│   └── 在产品
├── 服务成本（服务业）
│   ├── 餐饮服务
│   └── 咨询服务
├── 项目成本（建筑/软件）
│   ├── 开发项目
│   └── 工程项目
└── 客户成本（客户关系管理）
    ├── 客户群体A
    └── 客户群体B
```

### 1.3 成本要素分类

#### 按经济用途分类
| 成本要素 | 定义 | 包括内容 | 会计科目 |
|----------|------|----------|----------|
| **直接材料** | 可直接追溯到产品/服务的材料 | 原材料、主要材料、辅助材料、包装物 | 直接材料 |
| **直接人工** | 直接从事产品制造的人员薪酬 | 生产工人工资、计件工资、社保公积金 | 直接人工 |
| **制造费用** | 间接计入产品成本的费用 | 折旧、水电费、间接人工、机物料消耗 | 制造费用 |

### 1.4 成本性态分类

#### 成本性态定义
| 类型 | 定义 | 示例 | 特点 |
|------|------|------|------|
| **固定成本** | 总额不随产量变动而变动 | 厂房租金、管理人员工资、折旧 | 单位固定成本与产量成反比 |
| **变动成本** | 总额随产量变动而成比例变动 | 直接材料、直接人工（计件） | 单位变动成本相对稳定 |
| **混合成本** | 总额随产量变动但不成比例 | 电话费（基本费+通话费）、水电费 | 需分解为固定部分和变动部分 |

#### 混合成本分解方法

**高低点法**：
```
变动成本率 = (最高点成本 - 最低点成本) / (最高点产量 - 最低点产量)
固定成本 = 最高点成本 - 变动成本率 × 最高点产量
```

## 2. 材料成本核算

### 2.1 材料采购成本构成

#### 准则依据
根据《企业会计准则第1号——存货》，存货的采购成本包括：
- 购买价款
- 相关税费（不可抵扣的增值税、消费税等）
- 运输费、装卸费、保险费
- 运输途中的合理损耗
- 入库前的挑选整理费用

#### 采购成本计算公式
```
材料采购成本 = 买价 + 运输费 + 保险费 + 装卸费 + 合理损耗 + 入库前整理挑选费
```

#### 采购成本归集示例
```
【示例】某公司采购材料，收到以下发票：
- 增值税专用发票：价款100,000元，税额13,000元
- 运费发票：运费2,000元，税额180元（9%）
- 保险费：500元
- 入库前挑选整理费：300元

材料成本计算：
买价 = 100,000元
运费（不可抵扣部分）= 2,000元
保险费 = 500元
挑选整理费 = 300元
─────────────────────
材料采购成本 = 102,800元

注：若取得增值税专用发票且可抵扣，税额不计入成本
```

### 2.2 发出计价方法

#### 方法对比表
| 方法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| 先进先出法 | 物价波动大、时效性强商品 | 反映当时成本水平 | 计算繁琐 |
| 月末一次加权平均法 | 收发不频繁 | 简化计算 | 平时无法了解成本 |
| 移动加权平均法 | 收发频繁 | 及时了解成本 | 计算工作量大 |
| 个别计价法 | 高价值、可辨认商品 | 精确 | 需要逐件辨认 |

### 2.2.1 先进先出法（FIFO）

#### 基本原理
假定先购入的材料先发出，按购入顺序确定发出成本。

#### 计算公式
```
发出材料成本 = Σ 各批发出数量 × 对应批次单位成本
```

#### 示例
```
【示例】某材料收发存情况：

期初结存：100件 @ 10元 = 1,000元
3月5日   购入：200件 @ 11元 = 2,200元
3月10日  发出：250件
3月15日  购入：150件 @ 12元 = 1,800元
3月20日  发出：100件

发出成本计算：
┌────────────────────────────────────────────────────────┐
│ 3月10日发出250件：                                        │
│ ├── 100件（期初）@ 10元 = 1,000元                         │
│ └── 150件（3月5日购入）@ 11元 = 1,650元                    │
│ 发出成本合计 = 2,650元                                    │
├────────────────────────────────────────────────────────┤
│ 3月20日发出100件：                                        │
│ └── 50件（3月5日购入剩余）@ 11元 = 550元                   │
│ 发出成本合计 = 550元                                      │
├────────────────────────────────────────────────────────┤
│ 本月发出总成本 = 2,650 + 550 = 3,200元                     │
│ 期末结存 = 200件                                          │
│   ├── 100件（3月15日购入）@ 12元 = 1,200元                │
│   └── 50件（3月15日购入）@ 12元 = 600元                    │
│ 期末结存金额 = 1,800元                                    │
└────────────────────────────────────────────────────────┘
```

### 2.2.2 加权平均法

#### 月末一次加权平均法
```
加权平均单位成本 = (期初结存金额 + 本月入库金额) / (期初结存数量 + 本月入库数量)

发出材料成本 = 发出数量 × 加权平均单位成本
期末结存金额 = 期末结存数量 × 加权平均单位成本
```

#### 移动加权平均法
```
移动加权平均单位成本 = (本次购入前结存金额 + 本次购入金额) / (本次购入前结存数量 + 本次购入数量)

每次发出后立即计算新的加权平均单位成本
```

#### 示例（月末一次加权平均法）
```
【示例】某材料收发存情况：

期初结存：100件 @ 10元 = 1,000元
3月5日   购入：200件 @ 11元 = 2,200元
3月10日  发出：250件
3月15日  购入：150件 @ 12元 = 1,800元
3月20日  发出：100件

加权平均单位成本计算：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
加权平均单位成本 = (1,000 + 2,200 + 1,800) / (100 + 200 + 150)
                 = 5,000 / 450
                 = 11.11元/件

本月发出总成本 = (250 + 100) × 11.11 = 350 × 11.11 = 3,888.50元
期末结存成本 = 100 × 11.11 = 1,111.50元

验证：3,888.50 + 1,111.50 = 5,000元 ✓
```

### 2.2.3 个别计价法

#### 适用场景
- 高价值产品（大型设备、珠宝、精密仪器）
- 可以物理辨认的产品批号
- 需要精确成本的订单

#### 核算要点
```
┌────────────────────────────────────────────┐
│  个别计价法关键控制点                         │
├────────────────────────────────────────────┤
│  1. 建立完善的批号管理制度                     │
│  2. 记录每批次的购入单价                       │
│  3. 确保发出的产品可追溯到具体批次              │
│  4. 适合存货较少但单价较高的企业               │
└────────────────────────────────────────────┘
```

### 2.2.4 计划成本法

#### 核算流程
```
┌─────────────────────────────────────────────────────────────┐
│                    计划成本法核算流程                         │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【平时】                                                      │
│  1. 按计划成本入账：                                           │
│     借：原材料（计划成本）                                      │
│         材料成本差异（超支差异）                                │
│       贷：材料采购（实际成本）                                  │
│                                                              │
│  【月末】                                                      │
│  2. 结转入库材料成本差异：                                      │
│     借：材料成本差异                                           │
│       贷：材料采购                                            │
│                                                              │
│  3. 计算差异率并分摊差异：                                      │
│     差异率 = (期初差异 + 本月入库差异) / (期初计划成本 + 本月入库计划成本) × 100%
│                                                              │
│     发出材料应负担差异 = 发出材料计划成本 × 差异率              │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

#### 计划成本法示例
```
【示例】某企业采用计划成本法核算材料：

期初：原材料（计划成本）50,000元，材料成本差异（贷方）1,000元（节约）
本月入库：材料采购实际成本99,000元，计划成本100,000元
本月发出：原材料（计划成本）80,000元

计算过程：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
差异率 = (-1,000 + (100,000 - 99,000)) / (50,000 + 100,000)
       = (-1,000 + 1,000) / 150,000
       = 0 / 150,000
       = 0%

发出材料应负担差异 = 80,000 × 0% = 0元
发出材料实际成本 = 80,000 + 0 = 80,000元
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

若差异率为 +2%（超支）：
发出材料应负担差异 = 80,000 × 2% = 1,600元（超支调增）
发出材料实际成本 = 80,000 + 1,600 = 81,600元

若差异率为 -1%（节约）：
发出材料应负担差异 = 80,000 × (-1%) = -800元（节约调减）
发出材料实际成本 = 80,000 - 800 = 79,200元
```

### 2.3 材料损耗处理

#### 合理损耗 vs 非正常损耗
| 类型 | 原因 | 会计处理 | 税务处理 |
|------|------|----------|----------|
| **合理损耗** | 自然蒸发、计量误差、正常破损 | 计入采购成本 | 可抵扣进项税额 |
| **非正常损耗** | 管理不善、自然灾害、违规操作 | 计入营业外支出 | 不得抵扣进项税额 |

#### 损耗处理示例
```
【示例】购入材料100吨，单价1,000元/吨，到货验收发现合理损耗2吨

正常处理（合理损耗）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
材料入库数量 = 98吨
材料单位成本 = 100,000 / 98 = 1,020.41元/吨
入库总成本 = 98,000元（不含税）
合理损耗2吨的成本由98吨承担

非正常处理（管理不善）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
材料入库数量 = 98吨
材料单位成本 = 100,000 / 100 = 1,000元/吨
入库总成本 = 98,000元
非正常损耗2吨 = 2,000元 → 计入"管理费用"或"营业外支出"
```

### 2.4 材料盘盈盘亏处理

#### 盘盈处理
```
借：原材料
  贷：待处理财产损溢
      ↓
借：待处理财产损溢
  贷：管理费用（定额内合理损耗）
```

#### 盘亏处理
```
借：待处理财产损溢
  贷：原材料
      应交税费——应交增值税（进项税额转出）
      ↓
借：管理费用（定额内合理损耗）
    其他应收款（责任人赔偿）
    营业外支出（自然灾害损失）
  贷：待处理财产损溢
```


## 3. 人工成本核算

### 3.1 直接人工归集

#### 计件工资制
```
应付工资 = Σ(各产品产量 × 单位产品计件单价)
```

#### 计时工资制
```
应付工资 = 实际工时 × 小时工资率
或
应付工资 = 出勤天数 × 日工资率
```

### 3.2 社保公积金计提（五险一金）

#### 2024年参考比例（以北京为例，具体以当地政策为准）
| 社保项目 | 单位缴费比例 | 个人缴费比例 | 缴费基数上下限 |

### 3.3 人工成本分配标准

#### 分配方法对比
| 方法 | 公式 | 适用场景 |
|------|------|----------|
| **工时比例法** | 某产品人工成本 = 产品工时 × 人工分配率 | 工时记录准确 |
| **产量比例法** | 某产品人工成本 = 产品产量 × 单位人工成本 | 产品单一、产量易统计 |
| **产值比例法** | 某产品人工成本 = 产品产值 × 人工分配率 | 多品种生产 |
| **定额工时法** | 某产品人工成本 = 定额工时总和 × 分配率 | 有明确工时定额 |

#### 分配率公式
```
人工分配率 = 本月归集的直接人工总额 / 各种产品实际工时（产量、产值）总和

某产品应分配人工成本 = 该产品实际工时（产量、产值）× 人工分配率
```

#### 分配示例
```
【示例】某车间生产A、B两种产品

直接人工归集总额：100,000元
A产品实际工时：2,000小时
B产品实际工时：3,000小时

人工分配率 = 100,000 / (2,000 + 3,000) = 20元/小时

A产品应分配人工成本 = 2,000 × 20 = 40,000元
B产品应分配人工成本 = 3,000 × 20 = 60,000元

验证：40,000 + 60,000 = 100,000元 ✓
```

### 5.2 约当产量法

#### 约当产量法公式
```
在产品约当产量 = 在产品数量 × 完工率（投料程度/加工程度）

单位成本 = 生产成本总额 / (完工产品产量 + 在产品约当产量)

完工产品成本 = 完工产品产量 × 单位成本
在产品成本 = 在产品约当产量 × 单位成本
```

#### 材料投料方式与完工率计算
| 投料方式 | 投料程度计算 | 说明 |
|----------|--------------|------|
| **一次投料** | 材料投料程度 = 100% | 材料在开始时全部投入 |
| **逐步投料** | 投料程度 = 某工序累计材料费用 / 该产品全部材料费用 | 材料随加工进度投入 |

#### 加工进度完工率计算
| 计算方法 | 公式 | 说明 |
|----------|------|------|
| **按工时** | 完工率 = 累计工时 / 全部工时 | 最常用 |
| **按工序** | 完工率 = (上道工序累计时间 + 本道工序时间×50%) / 总工时 | 简化计算 |

#### 约当产量法示例
```
【示例】某产品生产情况：

产品A在产品100件，完工程度60%，材料一次投入
产品B在产品80件，第一工序（50小时），第二工序（30小时，已完成20小时）
总工时定额100小时，材料在开始时一次投入

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【产品A计算】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
材料约当产量 = 100 × 100% = 100件（一次投料，材料全部投入）
加工约当产量 = 100 × 60% = 60件

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【产品B计算】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
某工序完工率 = (上道工序累计工时 + 本工序工时×50%) / 总工时

第一工序：完工率 = (0 + 50×50%) / 100 = 25%
第二工序：完工率 = (50 + 30×50%) / 100 = 65%

加工约当产量 = 80 × 65% = 52件
材料约当产量 = 80 × 100% = 80件（一次投料）
```

### 5.3 定额比例法

#### 公式
```
分配率 = 月初在产品成本 + 本月发生成本 / (完工产品定额 + 月末在产品定额)

完工产品成本 = 完工产品定额 × 分配率
在产品成本 = 在产品定额 × 分配率
```

#### 定额比例法示例
```
【示例】某产品定额资料和成本资料：

定额资料：
- 完工产品：材料500kg，人工200小时
- 月末在产品：材料200kg，人工80小时

成本资料：
- 月初在产品成本：材料10,000元，人工3,000元，制造费用1,500元
- 本月发生成本：材料50,000元，人工12,000元，制造费用6,000元
合计：材料60,000元，人工15,000元，制造费用7,500元

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
计算过程：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【材料分配率】
分配率 = 60,000 / (500 + 200) = 85.71元/kg
完工产品材料成本 = 500 × 85.71 = 42,857.14元
在产品材料成本 = 200 × 85.71 = 17,142.86元

【人工分配率】
分配率 = 15,000 / (200 + 80) = 53.57元/小时
完工产品人工成本 = 200 × 53.57 = 10,714.29元
在产品人工成本 = 80 × 53.57 = 4,285.71元

【制造费用分配率】
分配率 = 7,500 / (200 + 80) = 26.79元/小时
完工产品制造费用 = 200 × 26.79 = 5,357.14元
在产品制造费用 = 80 × 26.79 = 2,142.86元

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
完工产品总成本 = 42,857.14 + 10,714.29 + 5,357.14 = 58,928.57元
在产品总成本 = 17,142.86 + 4,285.71 + 2,142.86 = 23,571.43元
验证：58,928.57 + 23,571.43 = 82,500元 ✓
```

### 5.4 定额成本法

#### 公式
```
月末在产品成本 = 月末在产品数量 × 单位在产品定额成本
完工产品成本 = 月初在产品成本 + 本月发生成本 - 月末在产品成本
```

### 6.2 品种法

#### 适用范围
- 大量大批单步骤生产
- 发电、采掘、铸造、供水、供气等

#### 特点
- 以产品品种为成本计算对象
- 成本计算定期按月进行
- 一般不需要计算在产品成本（或按简单方法计算）

#### 核算流程
```
┌─────────────────────────────────────────────────────────────┐
│                       品种法核算流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │

│                                                              │
│  2. 归集各项成本费用                                          │
│     ├── 直接材料                                             │
│     ├── 直接人工                                             │
│     └── 制造费用                                             │
│                                                              │
│  3. 分配辅助生产费用                                          │
│                                                              │
│  4. 分配制造费用（一种方法）                                   │
│     制造费用分配率 = 制造费用总额 / 各种产品产量之和            │
│                                                              │
│  5. 计算完工产品与在产品成本                                   │
│     （在产品很少时可不计算）                                   │
│                                                              │
│  6. 结转完工产品成本                                          │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 6.3 分批法

#### 适用范围
- 单件小批生产
- 船舶、飞机、重型机器制造、专用设备、服装加工

#### 特点
- 以产品批别为成本计算对象
- 成本计算不定期，与生产周期一致
- 一般不需分配在产品成本

#### 核算流程
```
┌─────────────────────────────────────────────────────────────┐
│                       分批法核算流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1. 设置生产成本明细账（按产品批别/订单号）                     │
│                                                              │
│  2. 归集成本费用                                              │
│     ├── 直接归集：直接材料、直接人工                          │
│     └── 间接归集：制造费用（先归集后分配）                     │
│                                                              │
│  3. 分配制造费用（可选择多种方法）                             │
│                                                              │
│  4. 订单完工后结转成本                                        │

│                                                              │
│  5. 跨月完工处理                                              │
│     ├── 计划成本法：按计划成本暂估                            │
│     └── 定额成本法：按定额成本暂估                            │
│        （实际成本确定后调整差异）                              │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 6.4 分步法

#### 适用范围
- 大量大批多步骤生产
- 冶金、纺织、造纸、化工、食品、机械制造

#### 分类
| 类型 | 特点 | 适用场景 |
|------|------|----------|
| **逐步结转分步法** | 各步骤半成品成本逐步结转 | 需要计算半成品成本 |
| **平行结转分步法** | 各步骤费用平行汇总 | 不需要计算半成品成本 |

#### 逐步结转分步法
```
┌─────────────────────────────────────────────────────────────┐
```

#### 综合结转 vs 分项结转
| 方式 | 特点 | 优缺点 |
|------|------|--------|
| **综合结转** | 半成品成本综合结转 | 需成本还原，计算复杂 |
| **分项结转** | 半成品成本分项结转 | 直观反映成本结构，计算繁琐 |

#### 成本还原
```
还原分配率 = 完工产品耗用的上一步骤半成品成本 / 上一步骤完工半成品成本合计

某成本项目还原数 = 还原分配率 × 上一步骤该成本项目金额

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【示例】成本还原计算
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第二步骤完工产品成本：
- 半成品成本：50,000元
- 直接人工：10,000元
- 制造费用：8,000元
合计：68,000元

第一步骤完工半成品成本：
- 直接材料：30,000元
- 直接人工：12,000元
- 制造费用：8,000元
合计：50,000元

还原分配率 = 50,000 / 50,000 = 1

还原后第二步骤完工产品成本：
- 直接材料 = 30,000 × 1 = 30,000元
- 直接人工 = 12,000 × 1 + 10,000 = 22,000元
- 制造费用 = 8,000 × 1 + 8,000 = 16,000元
合计：68,000元 ✓
```

### 6.5 作业成本法（ABC）

#### 基本原理
```
┌─────────────────────────────────────────────────────────────┐
│                    作业成本法分配逻辑                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│    资源              作业              产品                   │
│                                                              │
│  ┌──────┐         ┌──────┐                               │
│  │ 工资 │────────→│ 生产 │──────→┌──────┐              │
│  └──────┘         │ 准备 │       │ 产品A│              │
│                   └──────┘       └──────┘              │
│  ┌──────┐         ┌──────┐       ┌──────┐              │
│  │ 折旧 │────────→│ 加工 │──────→│ 产品B│              │
│  └──────┘         └──────┘       └──────┘              │
│                                                              │
│  ┌──────┐         ┌──────┐       ┌──────┐              │
│  │ 电费 │────────→│ 检验 │──────→│ 产品C│              │
│  └──────┘         └──────┘       └──────┘              │
│                                                              │
│  【两步分配】                                                 │
│  第一步：资源 → 作业（资源动因）                               │
│  第二步：作业 → 产品（作业动因）                               │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

#### 作业成本法公式
```
资源分配率 = 资源成本总额 / 资源动因总量

某作业成本 = Σ(资源分配率 × 该作业耗用资源动因量)

作业分配率 = 作业成本总额 / 作业动因总量

某产品成本 = Σ(作业分配率 × 该产品耗用作业量)
```

#### 作业成本法示例
```
【示例】某企业生产A、B两种产品

资源成本归集：
- 设备折旧：200,000元
- 人工成本：100,000元
- 水电费：50,000元
合计：350,000元

作业中心划分：
┌──────────────┬──────────────┬──────────────┬──────────────┐
│    作业      │   资源动因   │  资源动因量  │   成本(元)   │
├──────────────┼──────────────┼──────────────┼──────────────┤
│  生产准备    │   准备次数   │     20次     │    60,000    │
│  加工制造    │   机器工时   │   500小时    │   150,000    │
│  质量检验    │   检验次数   │     50次     │    80,000    │
│  材料处理    │   材料重量   │  1,000kg     │    60,000    │
└──────────────┴──────────────┴──────────────┴──────────────┘

产品耗用作业量：
┌──────────────┬──────────────┬──────────────┬──────────────┐
│    产品      │  生产准备    │   加工制造   │   质量检验   │ 材料处理 │
├──────────────┼──────────────┼──────────────┼──────────────┼─────────┤
│    产品A     │     8次      │   200小时    │     20次     │  400kg  │
│    产品B     │    12次      │   300小时    │     30次     │  600kg  │
└──────────────┴──────────────┴──────────────┴──────────────┴─────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【第一步：计算作业分配率】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
生产准备分配率 = 60,000 / 20 = 3,000元/次
加工制造分配率 = 150,000 / 500 = 300元/小时
质量检验分配率 = 80,000 / 50 = 1,600元/次
材料处理分配率 = 60,000 / 1,000 = 60元/kg

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【第二步：计算各产品成本】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
产品A成本：
- 生产准备 = 8 × 3,000 = 24,000元
- 加工制造 = 200 × 300 = 60,000元
- 质量检验 = 20 × 1,600 = 32,000元
- 材料处理 = 400 × 60 = 24,000元
──────────────
产品A总成本 = 140,000元

产品B成本：
- 生产准备 = 12 × 3,000 = 36,000元
- 加工制造 = 300 × 300 = 90,000元
- 质量检验 = 30 × 1,600 = 48,000元
- 材料处理 = 600 × 60 = 36,000元

产品B总成本 = 210,000元
```

## 7. 成本差异分析

### 7.1 材料成本差异分析

#### 计算公式
```
总差异 = 实际材料成本 - 标准材料成本

价格差异 = （实际价格 - 标准价格）× 实际数量
用量差异 = （实际用量 - 标准用量）× 标准价格

验证：价格差异 + 用量差异 = 总差异
```

#### 示例
```
【示例】某产品材料成本差异分析

标准资料：
- 标准用量：2kg/件
- 标准价格：10元/kg

实际资料：
- 实际产量：1,000件
- 实际用量：2,100kg
- 实际采购价格：9.5元/kg

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【计算过程】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
标准成本 = 1,000 × 2 × 10 = 20,000元
实际成本 = 2,100 × 9.5 = 19,950元
总差异 = 19,950 - 20,000 = -50元（节约）

价格差异 = (9.5 - 10) × 2,100 = -1,050元（节约）
用量差异 = (2,100 - 2,000) × 10 = +1,000元（超支）

验证：-1,050 + 1,000 = -50元 ✓

【差异解读】
- 价格差异节约1,050元：采购部门功劳
- 用量差异超支1,000元：生产部门责任
```

### 7.2 人工成本差异分析

#### 计算公式
```
总差异 = 实际人工成本 - 标准人工成本

工资率差异 = （实际工资率 - 标准工资率）× 实际工时
效率差异 = （实际工时 - 标准工时）× 标准工资率

标准工时 = 实际产量 × 单位产品标准工时
```

#### 示例
```
【示例】某产品人工成本差异分析

标准资料：
- 标准工时：5小时/件
- 标准工资率：20元/小时

实际资料：
- 实际产量：100件
- 实际工时：480小时
- 实际人工成本：10,560元

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【计算过程】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
实际工资率 = 10,560 / 480 = 22元/小时
标准工时 = 100 × 5 = 500小时
标准成本 = 500 × 20 = 10,000元

总差异 = 10,560 - 10,000 = +560元（超支）

工资率差异 = (22 - 20) × 480 = +960元（超支）
效率差异 = (480 - 500) × 20 = -400元（节约）

验证：960 + (-400) = +560元 ✓

【差异解读】
- 工资率差异超支960元：员工薪资上涨或高工资员工临时调配
- 效率差异节约400元：生产效率提升，减少了20小时工时
```

### 7.3 制造费用差异分析

#### 两差异法
```
耗费差异 = 实际制造费用 - 预算制造费用

效率差异 = （实际工时 - 标准工时）× 标准分配率
```

#### 三差异法（更精细）
```
耗费差异 = 实际制造费用 - 预算制造费用

产能利用差异 = 预算制造费用 - 标准工时 × 标准分配率
效率差异 = （实际工时 - 标准工时）× 标准分配率
```

#### 示例
```
【示例】制造费用差异分析

标准资料：
- 标准工时：4小时/件
- 标准制造费用分配率：15元/小时（基于机器工时）

预算资料：
- 预算产量：1,000件
- 预算制造费用：60,000元
- 预算工时：4,000小时

实际资料：
- 实际产量：1,100件
- 实际工时：4,600小时
- 实际制造费用：64,000元

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【两差异法】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
标准分配率 = 60,000 / 4,000 = 15元/小时
标准工时 = 1,100 × 4 = 4,400小时
标准成本 = 4,400 × 15 = 66,000元
实际成本 = 64,000元

总差异 = 64,000 - 66,000 = -2,000元（节约）

耗费差异 = 64,000 - 60,000 = +4,000元（超支）
效率差异 = (4,600 - 4,400) × 15 = +3,000元（超支）

两差异合计：4,000 + 3,000 = 7,000元
但实际是节约2,000元，此方法存在误差...

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【三差异法】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
耗费差异 = 64,000 - 60,000 = +4,000元（超支）

产能利用差异 = 60,000 - 4,400 × 15 = 60,000 - 66,000 = -6,000元（节约）
（实际产量对应标准工时4,400小时，预算为4,000小时，产能未充分利用）

效率差异 = (4,600 - 4,400) × 15 = +3,000元（超支）

验证：4,000 + (-6,000) + 3,000 = +1,000元
```

### 9.3 本量利分析（CVP分析）

#### 基本公式
```
┌─────────────────────────────────────────────────────────────┐
│                    本量利分析基本公式                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  边际贡献 = 销售收入 - 变动成本                               │
│            = 销售量 × (单价 - 单位变动成本)                    │
│                                                              │
│  边际贡献率 = 边际贡献 / 销售收入                            │
│             = (单价 - 单位变动成本) / 单价                    │
│                                                              │
│  营业利润 = 边际贡献 - 固定成本                              │
│            = 销售量 × 单位边际贡献 - 固定成本                  │
│                                                              │
│  【盈亏平衡点】                                               │
│  盈亏平衡销售量 = 固定成本 / (单价 - 单位变动成本)              │
│  盈亏平衡销售额 = 固定成本 / 边际贡献率                        │
│                                                              │
│  【安全边际】                                                 │
│  安全边际量 = 实际销量 - 盈亏平衡销量                          │
│  安全边际额 = 实际销售额 - 盈亏平衡销售额                      │
│  安全边际率 = 安全边际量 / 实际销量                            │
│                                                              │
│  【安全边际率评价标准】                                        │
│  安全边际率 > 20%：安全                                       │
│  安全边际率 10%-20%：较安全                                    │
│  安全边际率 < 10%：危险                                       │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

#### 本量利分析示例
```
【示例】某公司生产产品A，相关数据如下：

- 单价：100元/件
- 单位变动成本：60元/件
- 固定成本：200,000元
- 实际销量：8,000件

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【计算过程】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
单位边际贡献 = 100 - 60 = 40元/件
边际贡献率 = 40 / 100 = 40%

盈亏平衡销售量 = 200,000 / 40 = 5,000件
盈亏平衡销售额 = 200,000 / 40% = 500,000元

实际销售额 = 8,000 × 100 = 800,000元
营业利润 = 800,000 - 8,000×60 - 200,000 = 120,000元

安全边际量 = 8,000 - 5,000 = 3,000件
安全边际额 = 800,000 - 500,000 = 300,000元
安全边际率 = 3,000 / 8,000 = 37.5%

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【结论】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 该产品边际贡献率40%，盈利能力较好
2. 盈亏平衡点5,000件，当前销量8,000件，已超过盈亏平衡点
3. 安全边际率37.5% > 20%，经营安全
4. 若要实现利润150,000元，需要销量：
   销量 = (150,000 + 200,000) / 40 = 8,750件
```

## 11. 降本增效方法

### 11.1 目标成本法

#### 基本原理
```
┌─────────────────────────────────────────────────────────────┐
│                    目标成本法流程                             │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【传统成本法】                                              │
│  产品成本 → 加利润 → 销售价格                                │
│  成本 + 利润 = 价格                                          │
│                                                              │
│  【目标成本法】                                              │
│  销售价格 → 目标利润 → 目标成本                              │
│  价格 - 目标利润 = 目标成本                                   │
│                                                              │
│  ┌────────────────────────────────────────────────────────┐ │
│  │  目标成本 = 目标售价 × (1 - 目标利润率)                  │ │
│  │                                                        │ │
│  │  或                                                    │ │
│  │                                                        │ │
│  │  目标成本 = 目标售价 - 目标利润                          │ │
│  └────────────────────────────────────────────────────────┘ │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

#### 目标成本法示例
```
【示例】某企业开发新产品

目标售价：1,000元/件
目标利润率：15%
目标成本：？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【计算过程】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
目标利润 = 1,000 × 15% = 150元/件
目标成本 = 1,000 - 150 = 850元/件

现状分析：
- 当前估算成本：950元/件
- 与目标成本差距：950 - 850 = 100元/件（需降本11.8%）

降本分解：
- 直接材料：当前400元，目标降至360元（降40元）
- 直接人工：当前250元，目标降至220元（降30元）
- 制造费用：当前300元，目标降至270元（降30元）
──────────────
合计降本：100元 ✓
```
