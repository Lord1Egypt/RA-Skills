# Qi Theory Characteristics and Movement

## When to Use
- 分析宇宙万物的生成与演化机制
- 探讨人体生命起源、健康维持或疾病发生原理
- 解释自然现象（如潮汐、共鸣）中的感应与联动关系
- 阐述有形与无形物质之间的相互转化过程

## Execution Steps
1. **确认气的本质**：将“气”视为运动着的、至精至微的物质实体，是构成宇宙万物的最基本元素，源于“云气说”，发展为“元气一元论”。

2. **识别气的存在形式**：
   - “无形”之气弥散于天地之间，可聚合为“有形”之物；
   - “有形”之物亦可离散复归为“无形”之气；
   - 引用《素问·六节藏象论》：“气合而有形”，说明形与气可相互转化。

3. **分析气的运动特性**：
   - 气永不停息地运动，表现为“升降出入”四种基本形式；
   - 引用《素问·六微旨大论》：“气之升降，天地之更用也……高下相召，升降相因，而变作矣”；
   - 强调气的运动具有普遍性：“以升降出入，无器不有”。

4. **解析气运动的内在机制**：
   - 由气自身所含“阴”与“阳”两方面相互作用驱动；
   - “阳”主升、浮、动、散、排斥；“阴”主降、沉、静、聚、吸引；
   - 阴阳相互渗透、推荡、胜负交替，决定气的运动方向与强度，无需外力推动。

5. **阐明气的中介作用**：
   - 气作为万物感应的媒介，实现“气有潜通”；
   - 举例说明：乐器共鸣、磁石吸铁、潮汐应月等现象皆因气中介；
   - 引用《庄子·知北游》：“通天下一气耳”，强调宇宙整体通过气联结。

6. **关联生命与气的关系**：
   - 人由“天地合气”化生，引用《素问·保命全形论》：“天地合气，命之曰人”；
   - 引用《庄子》：“人之生，气之聚也；聚则为生，散则为死”；
   - 说明人体元气发源于肾（含命门），由先天之精化生，赖后天之精滋养，藏于丹田，借三焦输布全身。

> 注意：本技能不适用于非物质性或超自然解释体系，也不适用于静态不变的实体模型。
