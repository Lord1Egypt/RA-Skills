# 辅助生殖技术选择

## 何时使用
- 患者被诊断为不孕症（尝试自然受孕≥1年未成功）
- 已完成基础生育力评估（包括卵巢储备、输卵管通畅性、精液分析）
- 常规治疗（如促排卵、手术）无效

## 执行步骤
1. **确认ART适应证**：
   - 明确不孕病因（如输卵管阻塞、严重少弱精症、排卵障碍等）
   - 排除可自然受孕情况
2. **选择合适ART类型**：
   - **夫精人工授精（AIH）**：适用于轻度男性因素或宫颈因素不孕
   - **供精人工授精（AID）**：适用于无精子症或遗传病携带者
   - **体外受精-胚胎移植（IVF-ET）**：适用于输卵管因素、子宫内膜异位症、不明原因不孕等
   - **卵胞质内单精子注射（ICSI）**：适用于严重男性因素不孕（如精子浓度<5×10⁶/mL）
3. **启动前准备**：
   - 完成AMH、AFC评估卵巢储备
   - 确认输卵管状态（如HSG）
   - 完善男方精液分析
4. **遵循法规与伦理**：
   - 遵守《人类辅助生殖技术管理办法》
   - 禁止非医学需要的性别选择
   - 获取患者知情同意
5. **制定个体化治疗流程**：根据所选ART类型启动相应促排卵、取卵、授精及移植程序。

> 注意：本技能仅适用于符合医学指征的不孕症患者，且必须在合法合规前提下实施。
