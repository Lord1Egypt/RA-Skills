# 狭窄性腱鞘炎诊断与处理

## 何时使用
- 患者主诉手指屈伸时出现弹响、卡顿或疼痛
- 存在晨起手指僵硬，活动后缓解
- 掌横纹附近可触及随肌腱移动的痛性结节
- 怀疑为非感染性、非全身炎症性病因所致的手部腱鞘炎

## 执行步骤
1. **确认解剖基础**：识别受累肌腱（常见于手指屈肌腱、拇长屈肌腱、拇长展肌与拇短伸肌腱）及其通过的“骨-纤维隧道”（尤其掌指关节环状韧带区域）。

2. **评估病理机制**：
   - 判断是否存在长期摩擦导致的腱鞘水肿、增生及肌腱“葫芦状”膨大
   - 确认弹响机制：膨大部分强行挤过狭窄环状韧带引发弹拨感、响声与疼痛

3. **进行体格检查**：
   - 定位压痛点（通常位于远侧掌横纹）
   - 触诊黄豆大小痛性结节，观察其是否随肌腱移动
   - 若怀疑桡骨茎突腱鞘炎（De Quervain病），执行Finkelstein试验（握拳尺偏诱发疼痛为阳性）

4. **识别特殊人群风险**：
   - 小儿患者常为双侧拇长屈肌腱受累，可能致指间关节交锁
   - 产后女性、类风湿关节炎患者或存在肌腱解剖异常者更易发病

5. **排除禁忌情况**：
   - 不适用于感染性或全身性炎症性腱鞘炎
   - 小儿先天性病例对保守治疗反应差，需特别评估

6. **指导治疗决策**：
   - 首选保守治疗：制动、口服NSAIDs、局部糖皮质激素注射
   - 若保守无效或反复发作，考虑手术切开减压（需切除约0.3cm腱鞘以防止复发）
