## Description: <br>
Autonomous Codex workflow that helps an agent complete requested development tasks end to end, including routine commands, edits, waits, tests, builds, and verification, without avoidable confirmation prompts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers use this skill when they want Codex to carry a requested software task through implementation, waiting, verification, and final reporting without stopping for routine confirmations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Greater autonomy can reduce opportunities to review routine commands or edits before they run. <br>
Mitigation: Use the skill only when autonomous Codex execution is intended, and keep explicit review for deletion, publishing, production changes, credential use, spending money, and other irreversible actions. <br>
