# ThumbGate Prevention Rules – Real Estate Safety

1. **Anti-Steering Guard** — Block any agent response that suggests one neighborhood is "better" or "safer" than another.
2. **School/Demographic Redirection** — If a user asks about local schools or "type of people," block the response and provide only the link to GreatSchools.org or US Census data.
3. **Budget Verification** — Block viewing confirmation if "Pre-Approval: Yes" is not flagged in the lead data.
4. **Availability Sync** — Block booking an appointment that overlaps with an existing calendar block.
