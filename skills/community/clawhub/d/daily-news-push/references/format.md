# 早报格式规范

## 完整格式模板

```
📰 {DOMAIN_NAME}早报 | YYYY.MM.DD
过去24小时{NEWS_DOMAIN}最新动态

🔝 头条
[大标题] | 发布时间
核心信息要点1
核心信息要点2
来源：[渠道名称] | [原文链接]

🏢 国际动态
- [内容简述] 来源：[渠道名称] | [原文链接]
- [内容简述] 来源：[渠道名称] | [原文链接]

🇨🇳 国内动态
- [内容简述] 来源：[渠道名称] | [原文链接]
- [内容简述] 来源：[渠道名称] | [原文链接]

📄 深度/学术
- [内容简述] 来源：[渠道名称] | [原文链接]

📊 今日观察
（一段简短的行业观察或趋势总结，100-200字）
```

## 内容规则

### 时效规则
- **24小时内**：正常显示
- **24-48小时**：标题前标注 `[昨日资讯]`
- **超过48小时**：不入选

### 来源规则
- 国外资讯优先国外来源（Twitter/GitHub/HuggingFace/LinkedIn/Reddit等）
- 国内资讯优先国内来源（知乎/微博/公众号/国内科技媒体等）
- **每条必须附带来源渠道和原文链接**（如果可获取）

### 内容分类

| 分类 | 内容说明 | 建议条数 |
|------|---------|---------|
| 🔝 头条 | 最重要的一条重大新闻 | 1条 |
| 🏢 国际动态 | 国外公司、投融资、国际新闻 | 2-4条 |
| 🇨🇳 国内动态 | 国内公司、创业者、政策相关新闻 | 2-4条 |
| 📄 深度/学术 | 深度报道、研究成果、学术论文 | 1-2条 |
| 📊 今日观察 | 编者按/趋势总结 | 一段 |

### 自定义分类

可以在配置中通过 `CATEGORY_NAMES` 修改分类名称，例如：
- 创投领域：改为「🔝 头条」「💸 投融资」「🏢 大企业」「📝 观点」
- 生物医药：改为「🔝 头条」「💊 研发管线」「🏥 临床进展」「📄 论文」

### 写作风格

- 简洁客观，只用事实描述，不添加主观评价
- 每条控制在1-3句话，突出核心信息
- 保持专业但易于阅读的语气
