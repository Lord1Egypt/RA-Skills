## 🎯 是什么

OPC 创意造物局是成果转化平台（OPC-SPARK）中的**创意入口 Agent**，与技术经理人事务所形成闭环，共同构建"创意 → 验证 → 转化"的完整链路。

**SPARK 双重含义**：
- **星火燎原**：创意的微小起点，通过方法论和资源支撑走向规模化
- **智慧闪光**：S-P-A-R-K 五级原子能力体系的落地实践

---

## ✨ 核心能力

| 能力层 | 说明 |
|--------|------|
| **S-Skill** | 技能层 - 专项能力调用 |
| **P-Pipeline** | 管线层 - 流程编排与执行 |
| **A-Agent** | 智能体层 - 多角色协同 |
| **R-Reasoning** | 推理层 - 逻辑分析与决策 |
| **K-Knowledge** | 知识层 - 专业领域知识调用 |

---

## 🔧 功能特性

### 1. 创意识别与评估
- 快速识别创意价值
- 五维打分体系
- 差异化方向判断

### 2. 资源匹配
- 技能（Skill）→ 管线（Pipeline）→ 智能体（Agent）联动
- 资源瓶颈识别与建议

### 3. 全局视角
- 从完整项目生命周期思考
- 阶段性目标与里程碑规划
- 风险预判与预案准备

### 4. 人机协作
- 保留人类决策节点
- 清晰的人机分工界面
- 进度可视化与反馈机制
