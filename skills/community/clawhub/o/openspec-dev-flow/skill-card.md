## Description: <br>
Provides a Chinese OpenSpec-based iterative development workflow for planning, implementing, verifying, and archiving creation tasks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[huangxiaoqian007](https://clawhub.ai/user/huangxiaoqian007) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to follow a Chinese OpenSpec workflow for skill development, feature work, refactoring, bug fixes, and other multi-step creation tasks. It guides agents through exploration, proposal, implementation, verification, and archive steps with OpenSpec-style artifacts. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can broadly steer ordinary coding and creation requests. <br>
Mitigation: Install it only when an OpenSpec workflow should apply to broad development tasks, and review or narrow the trigger rules before use. <br>
Risk: The workflow allows confirmation steps to be skipped before workspace-changing work. <br>
Mitigation: Require explicit confirmation before creating files or implementing changes in important repositories. <br>
