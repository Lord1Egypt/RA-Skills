# Supplier Scorecard Quality Checklist

Run before finalizing any scorecard or comparison. Sections 1–3 are blocking.

## 1. Evidence log
- [ ] Log covers the full review period
- [ ] Every order in the period has a row
- [ ] Promised vs. actual ship dates recorded per order
- [ ] Defects split: incoming inspection vs. customer-reported (60d)
- [ ] Spec deviations recorded with approved Y/N status
- [ ] Communication incidents logged (negative AND proactive-positive)
- [ ] Landed cost per unit computed (not quote price)
- [ ] Estimated/reconstructed cells flagged as such

## 2. Scoring
- [ ] All five dimensions scored 1–10 against the anchored rubrics
- [ ] Every score cites at least one log entry
- [ ] No impression-based scores ("they seem reliable")
- [ ] Insufficient evidence marked "provisional", not guessed
- [ ] Lead time scored on accuracy vs. promise, not raw speed
- [ ] Period scored independently of last period's score
- [ ] Weights sum to 100%; any deviation from defaults justified in writing

## 3. Banding & recommendation
- [ ] Weighted total computed correctly (Σ score × weight × 10)
- [ ] Band assigned per the 85/70/55 thresholds
- [ ] Recommendation is a decision (consolidate/monitor/improve/transition), not a summary
- [ ] Trend vs. previous period stated with the moving dimensions named
- [ ] Leading indicators (comms slippage, substitutions) flagged even when total is fine

## 4. Comparison-specific
- [ ] Same review period and rubric anchors for all suppliers compared
- [ ] Pilot-order suppliers marked as lower-confidence evidence
- [ ] Switching recommendation includes switching-cost estimate
- [ ] Score gap and its duration stated against the ≥15-points/2-quarters bar
- [ ] Cost difference translated into defect/returns impact, not just unit price

## 5. Actions
- [ ] ≤4 actions, each with owner, deadline, success measure
- [ ] At least one action shared with the supplier (scorecard transparency)
- [ ] Improvement targets attached to a concrete PO or volume commitment
- [ ] Re-score date set; ad-hoc triggers listed

## 6. Risk hygiene
- [ ] Hero SKUs checked for single-source exposure
- [ ] Dual-source split reviewed (default 70/30)
- [ ] Tooling ownership documented for molds you paid for
- [ ] Golden-sample photos on file and referenced in latest PO
- [ ] Seasonal capacity risks (CNY, holiday peak) noted where relevant
