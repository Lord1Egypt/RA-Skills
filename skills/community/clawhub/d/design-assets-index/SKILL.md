# Design Assets Index

外部设计素材导航，告诉 agent「去哪找」现成素材。每类精选 2-3 条最常用入口放于此；更多条目按分类拆进 [references/](references/) 下各文件，按需读取即可。条目均为「名称（链接）— 一句话说明」，精选自 [awesome-design](https://github.com/gztchan/awesome-design) 与 [design-resources-for-developers](https://github.com/bradtraversy/design-resources-for-developers)。

## 图库素材

- [Unsplash](https://unsplash.com/) — Free high-resolution photos, no attribution required
- [Pexels](https://www.pexels.com/) — Free stock photos & videos shared by creators
- [Pixabay](https://pixabay.com/) — Over 1M+ free stock images and videos

更多见图库素材合集 [references/stock-photos.md](references/stock-photos.md)。

## 图标

- [Tabler Icons](https://tabler-icons.io/) — 3500+ customizable open-source SVG icons
- [Lucide](https://lucide.netlify.app/) — Open-source icon library, fork of Feather Icons
- [Iconify](https://iconify.design/) — One framework, 40,000+ vector icons unified

更多见图标合集 [references/icons.md](references/icons.md)。

## 字体

- [Google Fonts](https://fonts.google.com/) — Library of ~1000 free licensed font families
- [Fontsource](https://fontsource.org/) — Self-host open-source fonts via NPM packages
- [Fontshare](https://www.fontshare.com/) — Free quality fonts from the Indian Type Foundry

更多见字体合集 [references/fonts.md](references/fonts.md)。

## 配色

- [Coolors](https://coolors.co) — Super-fast color scheme generator
- [Adobe Color](https://color.adobe.com/create) — Build palettes, extract gradients from images
- [Color Hunt](https://colorhunt.co/) — Thousands of hand-picked palettes

更多见配色合集 [references/color-palettes.md](references/color-palettes.md)。

## Mockup

- [Mockup World](https://www.mockupworld.co/) — Largest source of free photo-realistic mockups
- [Smart Mockups](https://smartmockups.com/) — Create product mockups (free & premium)
- [Screely](https://www.screely.com/) — Turn a screenshot into a mockup instantly

更多见 Mockup 合集 [references/mockups.md](references/mockups.md)。

## UI Kit

- [undraw](https://undraw.co/) — Open-source illustrations for any idea
- [uiverse](https://uiverse.io) — Hundreds of open-source UI elements (HTML & CSS)
- [Headless UI](https://headlessui.dev/) — Unstyled, accessible components for Tailwind

更多见 UI Kit 合集 [references/ui-kits.md](references/ui-kits.md)。

## 模板

- [HTML5 Up](https://html5up.net/) — Modern, unique responsive HTML5/CSS3 themes
- [Start Bootstrap](https://startbootstrap.com/) — Bootstrap starter themes
- [Colorlib](https://colorlib.com/wp/templates/) — Themes in almost any category

更多见模板合集 [references/templates.md](references/templates.md)。

更全的 awesome 设计相关总目录见 https://github.com/sindresorhus/awesome
