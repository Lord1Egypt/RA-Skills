# D10 公民领域任务目录

## 领域定义

公民领域(D10)关注个体在公共生活中的权利行使、责任履行和社会参与。核心驱动需求：N3归属。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人公民权利行使 | 政府行政管理 |
| 公共事务参与 | 政党政治活动 |
| 公益志愿服务 | 商业慈善活动 |
| 公共表达与倡导 | 媒体专业报道 |
| 社会责任履行 | 企业社会责任(CSR) |
| 社区治理参与 | 基层政权建设 |
| 数字公民素养 | 网络安全技术 |
