## 📊 输出内容说明

### 标准输出包含：

1. **📍 头部信息**
   - 地图主题
   - 所属城市
   - 适配人群
   - 建议游玩时长

2. **📍 点位推荐**
   - 分场景TOP5优质点位
   - 评分、地址、适配贴士
   - 最佳游玩时间

3. **🚗 路线规划**
   - 按区域拆分路线
   - 总距离、预计用时
   - 游览顺序建议

4. **🎉 活动推荐**
   - 同城近期活动
   - 活动时间、地点、票价

5. **📚 人文讲解**
   - 历史沿革
   - 文化特色
   - 历史典故

6. **📱 地图二维码**
   - 高德个人地图链接
   - 扫码一键导航

## ⚙️ 配置说明

### 环境变量配置

```bash
# Linux/Mac
export AMAP_API_KEY=你的高德API密钥

# Windows
set AMAP_API_KEY=你的高德API密钥
```

### 获取API密钥

1. 访问 https://lbs.amap.com/
2. 注册开发者账号
3. 创建应用，选择"Web服务"
4. 获取API Key

## 🔧 常见问题

### Q: 为什么搜索结果为空？
A: 可能原因：
- 城市名称不标准（使用"北京"而非"帝都"）
- 搜索词过于具体
- 该地区数据较少

**建议**：尝试更通用的关键词，或扩大搜索范围

### Q: 如何提高推荐质量？
A: 多使用几次后，系统会自动学习你的偏好，推荐会越来越精准

### Q: 如何查看历史记录？
A: 说"查看我的历史游历"或"调取历史记录"

### Q: 如何重置画像？
A: 说"重置我的用户画像"
