# Project Directory Index Spec

The deep-read workspace must stay auditable.

Maintain:

- `metadata/project_directory_index.json`
- `reports/project_directory_index.md`
- `metadata/project_directory_annotations.json`

The index must identify authoritative per-paper reports, machine-readable intermediates, carried upstream source records or focus-spec snapshots, delivery-bundle and stage-delivery artifacts, visual artifacts still in use, and stale probe outputs that should be ignored.
