# True-False Cold-Heat Differentiation

## When to Use
- 患者处于疾病危重阶段
- 外在症状与内在病机呈现寒热矛盾（如外寒内热或外热内寒）
- 需结合舌象、脉象、二便、口渴性质等内在表现进行综合判断

## Execution Steps

### 1. 判断是否符合触发条件
确认患者同时具备以下特征：
- 表现出寒象与热象相互矛盾的症状
- 病情处于危重阶段

### 2. 识别真热假寒（阳盛格阴）
**假象（外在）**：四肢厥冷、脉沉（似寒证）
**真象（内在）**：
- 身寒而不喜加衣被
- 脉沉而有力
- 口渴喜冷饮、咽干口臭
- 谵语、小便短赤、大便燥结

### 3. 识别真寒假热（阴盛格阳）
**假象（外在）**：身热、面红、口渴、脉大（似热证）
**真象（内在）**：
- 身热而欲加衣被
- 面红但四肢寒冷
- 口渴喜热饮且饮不多
- 脉大无力、小便清长、大便稀溏、舌淡苔白

### 4. 应用辨别原则
- **部位原则**：假象多见于四肢、皮肤、面色；真象反映于脏腑、气血、津液
- **诊断依据**：以里证、舌象、脉象为核心判断标准
- **关键细节对比**：
  - 面赤：假热为颧颊浅红娇嫩、时隐时现；真热为满面通红
  - 四肢厥冷：若胸腹灼热、反不欲近衣被，为假寒；若蜷卧喜暖、欲得衣被，为真寒

### 5. 输出警示
根据辨证结果生成治疗警示：
> 警示：当前症状为假象，真实病性为[寒/热]，需按真实病性治疗，避免误治

## Precautions
- 本技能不适用于普通寒证或热证
- 必须结合舌象、脉象及二便等内在表现，不可仅凭外在症状判断
- 若无法明确辨识，应暂缓用药并寻求高级中医师会诊
