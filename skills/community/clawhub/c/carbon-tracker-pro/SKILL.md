# Carbon Tracker Pro

## 能力概述
面向个人和中小企业的碳管理与能源优化工具。基于IPCC核算方法和主流ESG框架，提供从碳核算、减排规划到碳中和路径的全流程服务。

## 核心功能

| 功能模块 | 说明 |
|----------|------|
| 碳足迹计算 | 按Scope 1/2/3标准计算个人或企业碳排放 |
| 能源审计 | 分析用电/用气/用油数据，识别节能机会 |
| 减排规划 | 生成分阶段的科学碳目标（SBTi对齐） |
| 碳中和路径 | 推荐碳抵消方案（绿证/CCER/林业碳汇） |
| ESG报告 | 自动生成符合GRI/TCFD框架的ESG披露报告 |

## 触发场景
- "帮我算一下我家一年的碳排放"
- "公司想2028年前碳中和，帮我做规划"
- "分析我们工厂上个月的用电数据，哪里可以省"
- "生成一份TCFD气候风险披露报告"
- "推荐靠谱的碳抵消项目"

## 执行流程

### 阶段一：数据采集
引导用户提供能源账单、出行记录、采购数据等碳排放源数据。

### 阶段二：核算与分析
基于IPCC排放因子库计算碳排放总量，按排放源拆解分析。

### 阶段三：规划输出
生成减排路径图，包含短期（1年）、中期（3年）、长期（5年+）目标与行动方案。

## 输出格式
```
【碳排放概览】
核算周期：{时间范围}
总排放量：{数值} tCO₂e

【排放结构】
- Scope 1（直接排放）：{数值} tCO₂e（{百分比}%）
- Scope 2（电力间接）：{数值} tCO₂e（{百分比}%）
- Scope 3（其他间接）：{数值} tCO₂e（{百分比}%）

【减排潜力分析】
- 最大减排机会：{排放源}，预计可减 {数值} tCO₂e
- 建议措施：{具体措施}

【碳中和路径】
短期（1年）：{行动}
中期（3年）：{行动}
长期（5年+）：{行动}

【推荐碳抵消方案】
- {项目名称} | 类型：{类型} | 价格：¥{金额}/tCO₂e

核算标准：IPCC 2006 / ISO 14064 | 排放因子来源：{来源}
```

## 注意事项
- 碳排放计算结果为估算值，正式报告建议经第三方核查
- 排放因子随政策更新，建议每季度刷新数据库
- 碳抵消项目选择需注意额外性和永久性
