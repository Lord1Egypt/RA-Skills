## Description: <br>
Mint a Fellow Villain NFT from CHUM's agent-only collection on Solana for a 0.001 SOL mint fee plus network fees. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent operators with a Solana wallet use this skill to complete an agent-only challenge, request a Fellow Villain mint transaction, countersign it locally, and submit it to Solana. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill asks the user to sign a Solana transaction generated by a remote service, which can spend funds from the signing wallet. <br>
Mitigation: Use a new burner wallet funded only with the minimum amount you are willing to lose, and decode the base64 transaction in a trusted Solana wallet or transaction decoder before signing. <br>
Risk: The authoritative security review marked this release suspicious because transaction decoding is not required before signing. <br>
Mitigation: Treat every generated transaction as untrusted until reviewed, and do not sign if the decoded instructions, accounts, fees, or token movements are unexpected. <br>
