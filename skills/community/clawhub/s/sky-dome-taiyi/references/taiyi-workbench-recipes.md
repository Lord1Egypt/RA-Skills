This reference is intentionally large but optional. Load only when building workflows, reports, experiments, or repeatable operating patterns.

## Recipe 001: Taiyi Practical Workflow Pattern

### Trigger
Use this when a task has ambiguity, repeated steps, or needs proof before delivery.

### Steps
1. Define the mission in one sentence.
2. Inspect the current state with the cheapest available tool.
3. Identify the active variable that matters most.
4. Choose one reversible action.
5. Record evidence before and after the action.
6. Compress the result into a reusable note if it will matter later.

### Useful Commands
```bash
python scripts/taiyi_calibrate.py "mission 1"
python scripts/taiyi_doctor.py <path>
python scripts/taiyi_workflow.py create project-execution "workflow 1" --mission "mission"
python scripts/taiyi_report.py "report 1"
```
