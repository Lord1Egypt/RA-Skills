# Zhushui Herb Contraindications and Usage

## When to Use
当涉及以下情况时启用：
- 患者诊断为「水肿膨胀」或「胸胁停饮」
- 需评估是否适合使用逐水药（如甘遂、大戟、芫花等）
- 用户询问逐水药的安全用法、禁忌或中毒风险

## Execution Steps

1. **确认适应症**：仅适用于体内实水积聚所致的水肿、膨胀或胸胁停饮，且患者正气未衰（表现为精神尚可、脉象有力、无明显虚损症状）。

2. **排除禁忌人群**：立即禁用以下情况：
   - 年老体弱者
   - 孕妇
   - 正气已虚者（如气短乏力、面色萎黄、脉细弱等）

3. **控制用量与剂型**：
   - 内服仅限丸散剂，不可煎服（如甘遂每次0.5～1.0g）
   - 外用可适量调敷患处

4. **减毒处理**：内服前必须经醋制以降低毒性（如醋甘遂）。

5. **注意配伍禁忌**：严禁与甘草同用（“十八反”原则）。

6. **遵循使用原则**：
   - 「中病即止」：症状缓解后立即停药
   - 「不可久服」：避免损伤正气

7. **监测中毒反应**：密切观察是否出现恶心、呕吐、剧烈腹痛、腹泻加重等中毒表现，一旦发生应立即停药并处理。
