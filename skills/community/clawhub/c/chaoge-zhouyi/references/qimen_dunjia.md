# 奇门遁甲完整手册

## 一、奇门遁甲概述

奇门遁甲是中国古代术数之首，号称"帝王之学"，与太乙神数、六壬神课并称"三式"。

### 名称由来

- **奇**：三奇（乙奇、丙奇、丁奇）
- **门**：八门（休、生、伤、杜、景、死、惊、开）
- **遁甲**：甲木为十天干之首，尊贵不显，隐遁于六仪之下

### 历史渊源

传说黄帝战蚩尤时，九天玄女传授奇门遁甲。后姜子牙、张良、诸葛亮、刘伯温等皆精此术。

### 2. 八门

八门主人事，代表八种不同的人事状态。

| 门 | 五行 | 吉凶 | 含义 | 宜忌 |
|---|------|------|------|------|
| **休门** | 水 | 吉 | 休息、养生、宁静 | 宜养生、休整、聚会、结婚 |
| **生门** | 土 | 吉 | 生发、财利、新生 | 宜求财、开业、婚嫁、出行 |
| **伤门** | 木 | 凶 | 伤害、破坏、损耗 | 宜狩猎、讨债、破坏；忌诸吉事 |
| **杜门** | 木 | 中 | 杜绝、隐藏、阻滞 | 宜隐藏、修炼、避难；忌出行 |
| **景门** | 火 | 中 | 景象、文书、光明 | 宜考试、文书、演艺；忌婚嫁 |
| **死门** | 土 | 凶 | 死亡、闭塞、终结 | 宜吊唁、打官司；忌诸吉事 |
| **惊门** | 金 | 凶 | 惊恐、官司、口舌 | 宜攻敌、设伏；忌出行、婚嫁 |
| **开门** | 金 | 吉 | 开启、通达、事业 | 宜开业、上任、出行、婚嫁 |

**八门原始宫位**：
- 休门 → 坎一宫（北方）
- 生门 → 艮八宫（东北）
- 伤门 → 震三宫（东方）
- 杜门 → 巽四宫（东南）
- 景门 → 离九宫（南方）
- 死门 → 坤二宫（西南）
- 惊门 → 兑七宫（西方）
- 开门 → 乾六宫（西北）

| 星 | 五行 | 吉凶 | 含义 | 应用 |
|---|------|------|------|------|

| **天任** | 土 | 吉 | 任用、诚信、厚道 | 利求人、任职、合作 |
| **天冲** | 木 | 吉 | 冲动、进取、武事 | 宜进攻、竞争、武职 |
| **天辅** | 木 | 吉 | 辅佐、文昌、贵人 | 利考试、学习、贵人相助 |
| **天英** | 火 | 中 | 英明、光明、血光 | 利文书、表彰；防火灾血光 |
| **天芮** | 土 | 凶 | 疾病、柔弱、问题 | 主病痛、问题、阻滞 |
| **天柱** | 金 | 中 | 柱石、刚直、破坏 | 利中流砥柱；防破坏 |
| **天心** | 金 | 吉 | 心计、谋划、医生 | 利谋略、医疗、策划 |
| **天禽** | 土 | 吉 | 中正、稳定、守成 | 利稳定、守成、中正之事 |

**九星原始宫位**：
- 天蓬 → 坎一宫
- 天芮 → 坤二宫
- 天冲 → 震三宫
- 天辅 → 巽四宫

- 天心 → 乾六宫

### 4. 八神

八神主神助，代表八种不同的神秘力量。

| 神 | 属性 | 含义 | 应用 |
|---|------|------|------|
| **值符** | 吉 | 领袖、贵人、统帅 | 最吉之神，凡事顺遂 |
| **腾蛇** | 凶 | 虚惊、怪异、缠绕 | 主惊恐、怪异之事 |
| **太阴** | 吉 | 阴私、隐蔽、贵人 | 利隐藏、密谋、贵人暗助 |
| **六合** | 吉 | 和合、婚姻、中介 | 利合作、婚嫁、中介 |
| **白虎** | 凶 | 凶猛、血光、道路 | 主凶险、血光、道路 |
| **玄武** | 凶 | 盗贼、暗昧、欺诈 | 主盗贼、欺骗、暗事 |
| **九地** | 吉 | 坚守、屯兵、长久 | 宜防守、存储、长久之事 |
| **九天** | 吉 | 进取、张扬、远行 | 宜进攻、宣传、远行 |

**八神排列顺序（阳遁顺排，阴遁逆排）**：

阳遁：值符 → 腾蛇 → 太阴 → 六合 → 白虎 → 玄武 → 九地 → 九天

阴遁：值符 → 九天 → 九地 → 玄武 → 白虎 → 六合 → 太阴 → 腾蛇

### 5. 三奇六仪

#### 三奇（日奇、月奇、星奇）

三奇是奇门遁甲中最吉祥的元素。

| 奇 | 名称 | 含义 | 应用 |
|---|------|------|------|
| **乙奇** | 日奇 | 最吉 | 主文书、贵人、婚姻 |
| **丙奇** | 月奇 | 次吉 | 主威权、武事、破局 |
| **丁奇** | 星奇 | 再次吉 | 主玄妙、秘密、玄学 |

#### 六仪（甲所遁之处）

甲木为十天干之首，尊贵不显，隐遁于六仪之下。

| 六仪 | 遁甲 | 含义 |
|------|------|------|
| **戊** | 甲子遁戊 | 甲子戊 |
| **己** | 甲戌遁己 | 甲戌己 |
| **庚** | 甲申遁庚 | 甲申庚 |
| **辛** | 甲午遁辛 | 甲午辛 |
| **壬** | 甲辰遁壬 | 甲辰壬 |
| **癸** | 甲寅遁癸 | 甲寅癸 |

#### 天干排序

奇门遁甲中天干的排列顺序（从戊开始）：

**戊、己、庚、辛、壬、癸、丁、丙、乙**

（共九个，甲隐遁不显）

### 2. 三奇六仪排布

#### 阳遁排法（顺排）

从局数对应的宫位开始，按戊己庚辛壬癸丁丙乙的顺序顺排。

**阳遁一局**：戊在一宫，己在二宫，庚在三宫...

**阳遁九宫排布示例**（阳遁一局）：

```
巽四癸  离九丁  坤二己
震三庚  中五壬  兑七辛
艮八丙  坎一戊  乾六乙
```

#### 阴遁排法（逆排）

从局数对应的宫位开始，按戊己庚辛壬癸丁丙乙的顺序逆排。

**阴遁九宫排布示例**（阴遁一局）：

```
巽四乙  离九丙  坤二丁
震三癸  中五壬  兑七己
艮八辛  坎一戊  乾六庚
```

### 3. 值符值使

- **值符**：当班之星（地盘时干所在宫位的本位星）
- **值使**：当班之门（地盘时干所在宫位的本位门）

### 4. 天盘地盘

- **地盘**：根据局数固定排布的三奇六仪
- **天盘**：根据时辰转动，值符随时干落宫

## 五、断事要点

### 1. 看门（主人事）

- 门吉：人事顺利
- 门凶：人事阻滞
- 门迫：门受宫克，力量减弱

### 2. 看星（主天时）

- 星旺：天时有利
- 星衰：天时不利
- 星受克：天时受损

### 3. 看神（主神助）

- 神吉：有贵人相助
- 神凶：有小人作祟

### 4. 看奇仪（主格局）

- 吉格：格局有利
- 凶格：格局不利

### 5. 看落宫（主方位）

- 宫生门星：力量增强
- 宫克门星：力量减弱
- 门星生宫：泄气

### 6. 综合判断

```
吉凶 = 门 × 星 × 神 × 奇仪 × 落宫
```

需要综合分析各要素的五行生克关系。
