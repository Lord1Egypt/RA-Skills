## 板块1：核心定位与使用规则

### 核心职责

MEMORY 是 AI 的"简历"，不是"详细履历"。每次醒来读它，3分钟内要能回答：

1. **用户是什么样的人** — 核心特质、偏好、忌讳
2. **AI 应该怎么配合** — 当前规则、约定、待回避的坑
3. **重要共识** — 长期有效的核心决定

### 内容准入红线

进入 MEMORY 必须同时满足：
- ✅ 经用户确认的结论
- ✅ 长期有效（≥30天）
- ✅ 能直接指导配合动作

**以下内容不放：**
- 单次会话的完整记录 → 对应日期日志
- 临时、短期信息 → 当日日志
- 和其他专属文件重复的内容 → 移到对应文件
