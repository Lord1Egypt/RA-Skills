## 卷二：六甲神咒

### 一、总咒

**六甲神咒**

```
六甲神将，六丁神兵。
随吾号令，速现真形。
护吾身命，驱邪斩精。
急急如律令。
```

### 二、召将咒

**召六甲咒**

```
谨请六甲阳神大将军，
甲子大将统领天兵。
甲戌大将镇守四门，
甲申大将青龙护身。
甲午大将朱雀传信，
甲辰大将勾陈值令。
甲寅大将天刑显灵。
吾奉太上老君敕令，
速降真形，护吾身命。
急急如律令。
```

### 三、护身咒

**六甲护身咒**

```
天护身，地护身。
六甲六丁护吾身。
左青龙，右白虎。
前朱雀，后玄武。
中央戊己护吾身。
千灾万难俱消散。
急急如律令。
```

### 四、隐身咒

**六甲隐遁咒**

```
天隐地隐，人隐鬼隐。
六甲神将，隐吾身形。
鬼神不见，邪魔不侵。
急急如律令。
```

## 卷三：六甲法术

### 一、六甲遁甲术

六甲遁甲，与奇门遁甲相通，借六甲神力，行遁甲之术。

**方法**：
1. 选甲子、甲戌、甲申、甲午、甲辰、甲寅日
2. 设坛请六甲神将
3. 念咒踏罡
4. 行遁甲之法

### 二、六甲护身法

**用途**：护身保命，邪魔不侵

**方法**：
1. 清晨面向东方
2. 念六甲护身咒七遍
3. 观想金光护体

### 三、六甲治病法

**用途**：祛病安康

**方法**：
1. 于病人床头
2. 念六甲神咒
3. 书符焚化
4. 化水服用

## 卷八：六甲问答

**问：六甲神咒灵验否？**

答：心诚则灵，心正则应。咒本无灵，灵在人心。

**问：人人可修六甲乎？**

答：人人可修，非人人能成。德行为本，根基为要。

**问：六甲法术可伤人乎？**

答：正法不伤人，伤人非正法。恶用必反噬。

**问：六甲与奇门遁甲何异？**

答：六甲重神咒，奇门重格局。二者相通，各有侧重。
