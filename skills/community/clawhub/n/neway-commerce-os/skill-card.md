## Description: <br>
Generate, scaffold, build, and prepare deployment for a reusable React, Vite, and Hono commerce operating system for digital products, storefronts, subscriptions, AI-assisted sales, admin dashboards, and Stripe-style payments. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[lwter](https://clawhub.ai/user/lwter) <br>

MIT <br>


## Use Case: <br>
Developers, creators, and commerce teams use this skill to generate a runnable full-stack commerce website skeleton from a short product idea, including storefront, cart, checkout, wishlist, admin dashboard, AI concierge, and EdgeOne deployment files. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated commerce projects can be mistaken for production-ready checkout and payment security implementations. <br>
Mitigation: Treat checkout and Stripe webhook code as placeholders until real payment verification and production controls are implemented. <br>
Risk: The scaffold script can remove an existing generated output project without confirmation. <br>
Mitigation: Run it only in a prepared output directory and keep backups of valuable work before scaffolding. <br>
Risk: AI assistant integrations may expose customer, payment, or business-sensitive data if configured without privacy controls. <br>
Mitigation: Avoid sending sensitive data to AI providers unless explicit privacy controls and provider approvals are in place. <br>
