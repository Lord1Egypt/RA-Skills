## Description: <br>
Crea skills para OpenClaw con el formato correcto. YAML frontmatter, _meta.json, estructura de archivos. Publicar en ClawHub es opcional. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[yejay7](https://clawhub.ai/user/yejay7) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and OpenClaw users use this skill to create correctly structured local OpenClaw skills with valid SKILL.md frontmatter and _meta.json metadata. It also provides optional guidance for publishing a completed skill to ClawHub. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill guides local file creation and updates for OpenClaw skills. <br>
Mitigation: Confirm the local OpenClaw skills path and review generated or modified files before using the skill. <br>
Risk: The skill includes optional authentication and publishing commands for ClawHub. <br>
Mitigation: Run login or publish commands only when intentionally authenticating with ClawHub or publishing a skill. <br>
