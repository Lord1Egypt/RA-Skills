## Description: <br>
Play audio/video locally on the host. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Xejrax](https://clawhub.ai/user/Xejrax) <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and developers use this skill to ask an agent to play, pause, or stop local audio/video files and trusted remote media URLs through mpv on the host. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can ask the host media player to open local files or remote media URLs. <br>
Mitigation: Install mpv only through a trusted package manager and use the skill with trusted file paths or URLs. <br>
