## Description: <br>
Converts Markdown files to styled PDFs with Chrome headless, including support for tables, code blocks, and Chinese text. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[earl-chen](https://clawhub.ai/user/earl-chen) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agents use this skill to convert trusted Markdown reports or documents into PDF files from the command line. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Markdown content may render active HTML or JavaScript in Chrome without warning or strong isolation. <br>
Mitigation: Use the skill only with Markdown files you trust; avoid untrusted Markdown unless the maintainer adds HTML escaping or sanitization and stronger Chrome isolation. <br>
