# 儿童生长曲线临床解读

## 使用前提
确保满足以下条件再执行解读：
- 已有至少两次定期、连续的身高/体重测量数据
- 具备准确的年龄与性别信息
- 使用标准化生长曲线图（如WHO或中国标准）
- 排除急性疾病期临时体重下降或明显测量误差

## 解读步骤

### 1. 确认个体生长轨道
- 分析连续测量点形成的趋势线，而非依赖单次数据
- 正常范围为 P₃ 至 P₉₇（或均值±2个标准差），无需追求“达到均值”

### 2. 评估喂养方式影响
- 纯母乳喂养婴儿早期生长可能略低于配方奶喂养儿，属正常现象
- 避免因此过早添加辅食或不必要补充配方奶

### 3. 识别回归均值现象
- 出生时极端百分位（过高或过低）的儿童，约2/3在2～3岁前趋向 P₅₀
- 需先复核测量准确性，再判断是否为正常生理现象

### 4. 判定生长波动
- **定义**：生长曲线偏离原稳定轨道超过1条主百分位线（P₉₇、P₇₅、P₅₀、P₂₅、P₃）
- **操作**：增加监测频率，排查营养、疾病或环境因素，提供针对性指导

### 5. 判定生长异常
- **任一条件满足即视为异常**：
  a) 生长水平或体型匀称度 < P₃ 或 > P₉₇；
  b) 系列测量中偏离原轨道超过2条主百分位线
- **操作**：立即寻找病因，必要时转诊至上级医疗机构或专科

## 输出结果
根据分析输出以下三类结论之一：
- **正常**：个体维持稳定生长轨道，在 P₃–P₉₇ 范围内
- **生长波动**：偏离1条主百分位线，建议加强监测与营养指导
- **生长异常**：超出 P₃/P₉₇ 或偏离≥2条主百分位线，建议医学评估或转诊
