# Messages — Make Friends

Keep messages short, specific, and easy to accept. Early friendship is built through low-friction continuity.

## Contextual Openers

| Situation | Message shape |
|-----------|---------------|
| Repeated class or club | "Good seeing you again. Are you coming next week too?" |
| Event with shared topic | "You mentioned X. Send me that recommendation if you find it." |
| Coworking or work-adjacent | "I usually work from this place on Thursdays if you ever want to overlap." |
| Online-to-offline | "You seem fun to talk to. If you're up for it, coffee or the next meetup could be easy." |

## Follow-Up After Good Vibes

Use within 24-72 hours:

- "Fun talking today. If you're going to the next one, I probably will too."
- "Good meeting you. That thing you mentioned stuck with me."
- "I liked the conversation. If you want, we could continue it over coffee sometime."

## Low-Pressure Invites

Prefer these first:

- coffee near the shared location
- walk
- join the same class again
- group plan with a clear end time
- casual errand or market run

Avoid for early stage:

- full-day plans
- expensive dinners
- emotionally heavy hangouts
- invites that sound like obligations

## Gentle Close if Interest Is Weak

- "No worries, maybe another time."
- "All good, see you around there."
- "Sounds like your schedule is packed. If timing works later, happy to reconnect."

The exit matters. It protects dignity and keeps ambient friendliness possible.
