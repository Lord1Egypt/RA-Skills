# 小红书内容优化参考

## 爆款标题模式

### 痛点触发型
- 「苦等X年，终于…」
- 「XX终于来了，但…」
- 「为什么XX，这么简单的事没人告诉我」
- 「救命！XX竟然这么简单」

### 数字吸睛型
- 「X个技巧，XX的人不知道」
- 「X分钟学会XX」
- 「花了XX块，总结X条教训」
- 「XX%的XX都不知道」

### 情绪共鸣型
- 「谁懂啊…」
- 「崩溃了…」
- 「XX人狂喜」
- 「XX原地升天」

### 利益前置型
- 「XX之后，XX直接翻倍」
- 「用这个方法，XX效率提升X倍」
- 「免费！XX白嫖攻略」

## 封面设计原则

| 要素 | 说明 |
|------|------|
| 大字标题 | 突出核心关键词，8字以内 |
| 人脸 | 女性侧脸/表情夸张的人像效果最好 |
| 颜色 | 高对比：白底黑字/亮色背景+深色字 |
| 留白 | 不要堆太满，四周留呼吸空间 |
| 禁止 | 纯截图、模糊图、水印图 |

## 互动引导话术

结尾加以下之一，可有效提升评论率：
- 「你们遇到过吗？」
- 「你们最想要哪个功能？」
- 「评论区告诉我」
- 「你们觉得呢？」
- 「你们觉得哪个更好？」

## 发布检查清单

- [ ] 封面：大字+人脸/设计感+高对比
- [ ] 标题：痛点/数字/情绪 任选一个在前半句
- [ ] 正文前3行：利益点+数字+行动触发
- [ ] 标签：2–4个精准话题
- [ ] 互动引导：结尾有问句
- [ ] 不要在发布后2小时内编辑
