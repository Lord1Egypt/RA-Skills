## Description: <br>
Query crypto wallet portfolios, transactions, DeFi positions, NFTs, token prices, and gas fees across EVM chains and Solana using Zerion's API. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, analysts, and agents use this skill to research wallet portfolios, transactions, DeFi positions, NFT holdings, token prices, and gas fees through Zerion's remote MCP server. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Queried wallet addresses, token identifiers, and related research questions are sent to Zerion's remote service. <br>
Mitigation: Use a dedicated or revocable API key, verify the MCP URL before adding credentials, and avoid submitting sensitive targets without authorization. <br>
