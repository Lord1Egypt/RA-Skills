# 🚀 SaaS 产品辅助决策助手

> 输入SaaS产品行业与定位，自动从 **市场/用户/需求/竞品/变现/获客/推广/成本/技术** 9大维度综合分析，生成专业交互式HTML可行性决策报告。

## 快速使用

```
SaaS决策：做一个面向中小电商的AI客服SaaS
帮我评估做HR SaaS招聘模块可行吗
```

## 9维度SaaS专属评分模型

| 维度 | 权重 | SaaS专属内容 |
|------|:--:|------|
| 📊 市场需求 | 15% | TAM/SAM、市场规模、增长率 |
| 👥 用户画像 | 10% | B2B/B2C决策链、付费意愿 |
| 🎯 需求痛点 | 10% | 痛点强度矩阵、替代方案 |
| ⚔️ 竞品格局 | 15% | 头部对比表、护城河潜力 |
| 💰 变现能力 | 15% | Freemium/订阅/用量/混合、MRR/LTV |
| 📈 获客增长 | 10% | CAC、PLG飞轮、渠道效率 |
| 📢 推广营销 | 15% | 品牌定位、分阶段推广时间线 |
| 💸 成本结构 | 5% | R&D/S&M/G&A占比参考 |
| 🔧 技术可行性 | 5% | 多租户架构、技术选型决策 |

## 报告结构

1. 封面页 + 综合评分卡
2. 多维度评分矩阵（9格可视化）
3. 市场需求分析（TAM/SAM/趋势信号）
4. 用户画像分析（用户特征/决策链/付费意愿）
5. 需求痛点分析（痛点强度矩阵/替代方案/迫切度）
6. 竞品格局分析（直接/间接竞品/头部对比/差异机会）
7. 变现定价策略（4种SaaS定价模式对比/MRR预估/定价梯度）
8. 获客增长分析（CAC/6渠道效率对比/PLG评估）
9. 推广营销策略（品牌定位/内容策略/4阶段时间线）
10. 成本结构分析（云/研发/营销/三阶段占比参考）
11. 技术可行性评估（架构选型/多租户策略/合规要求）
12. 风险提示（SaaS特有PMF/流失/定价/获客风险）
13. 综合决策建议（P0-P3行动路线图）

## SaaS 专属特性

- **SaaS定价模式对比**：订阅制/用量制/Freemium/混合四模式卡片
- **MRR/ARR预估** + LTV/CAC分析
- **定价梯度建议表**：4级定价(免费→专业→企业→定制)
- **获客渠道矩阵**：6大渠道的CAC/转化率/可复制性/SaaS适配度
- **SaaS成本结构参考**：早期/成长期/规模化三阶段R&D/S&M/G&A占比
- **技术决策表**：租户隔离/认证/计费/部署/监控5大选型
- **SaaS死亡陷阱Top3**：未验证PMF/大而全/忽视客户成功

## 评分等级

- ≥80分：✅ **强烈建议做**
- 65-79分：🟡 **谨慎推进**
- 50-64分：⚠️ **暂缓观望**
- <50分：❌ **不建议做**
