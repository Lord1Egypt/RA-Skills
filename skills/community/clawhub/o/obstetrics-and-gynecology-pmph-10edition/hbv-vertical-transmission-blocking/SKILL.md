# HBV垂直传播阻断策略

## 触发条件
当确认孕妇为HBsAg阳性时，立即启动本阻断流程。

## 执行步骤

### 1. 孕期干预
- 若孕妇HBV DNA ≥ 2×10⁵ IU/mL（高病毒载量），于妊娠28～32周开始口服替诺福韦等抗病毒药物，以降低分娩时病毒载量，减少宫内及产时传播风险。

### 2. 新生儿联合免疫（出生后12小时内完成）
- 肌肉注射乙型肝炎免疫球蛋白（HBIG）：剂量≥100 IU
- 同时肌肉注射首剂乙型肝炎疫苗：0.5 mL（重组酵母疫苗）

### 3. 后续疫苗接种
- 严格按0、1、6月龄程序完成三剂乙肝疫苗接种

### 4. 注意事项
- 即使母亲HBsAg阴性，若父亲或家庭成员为HBV携带者，新生儿仍需规范全程接种疫苗
- 母乳喂养不增加HBV传播风险，应鼓励
- 未按时接种疫苗的新生儿，感染后80%以上转为慢性携带状态

## 输出结果
生成包含以下内容的执行清单：
- 孕妇是否需启动抗病毒治疗及起始孕周
- 新生儿HBIG与首剂疫苗的接种时间窗
- 完整的0-1-6月疫苗接种时间表
