## ✨ 六大核心功能

### 🎥 1. 全天候智能录制

- 后台守护进程，7×24 小时不间断录制
- 10 秒分段存储，精确到每一个时刻
- 支持多摄像头同时录制，统一管理
- 对话式操控 —— 说一句"开始录制"即可启动

### 🔍 2. 自然语言视频搜索（核心卖点）

不需要任何专业操作，**用你的日常语言描述你要找的画面**：

- "找到那个搬箱子的人"
- "有人在后门出现"
- "快递员今天来了几次？"
- "昨天晚上车库有没有异常？"

AI 自动理解语义，通过**向量相似度匹配**在数据库中精准定位相关片段。

### 🧠 3. AI 场景自动描述

每个 10 秒视频片段都会经过 AI 分析，自动生成文字摘要：

- 识别人物、动作、物品、场景
- 构建可检索的事件索引
- 让"哑巴监控"变成"会说话的安保助手"

### ⏱ 4. 智能时间范围过滤

- 支持按时间段精确筛选："今天上午"、"昨天下午3点到5点"
- 自然语言时间表达自动转换
- 跨摄像头联合搜索，无死角覆盖

### 🏃 5. 运动检测 & 智能过滤

- 自动识别静态画面，跳过无意义片段
- **节省高达 70% 的 API 调用费用**
- 可调节灵敏度，适配不同场景需求
- 所有画面仍然保存，只是静态片段不做 AI 分析

### 🗄 6. 本地事件索引 & 自动管理

- **SQLite 本地数据库**，离线可查，数据不外泄
- 自动过期清理，释放磁盘空间
- 可审计日志，每一次操作有据可查
- 灵活配置保留天数（默认 3 天）

## ⚡ 2 分钟快速上手

### 第一步：安装

安装 Skill 后，AI 助手会自动引导你完成环境配置：

- 检测 Python 3.10 环境
- 创建隔离虚拟环境（`.venv`）
- 安装所需依赖（`numpy`、`opencv-python`、`requests`）

**无需手动操作，跟着提示走即可。**

### 第二步：配置摄像头

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `STREAM_URL` | 摄像头 RTSP 地址 | —（必填） |
| `KAMI_API_KEY` | Kamivision API 密钥（[免费获取](https://kamiclaw-skill.kamihome.com)） | —（必填） |
| `DEVICE_ID` | 摄像头标签，如 `front-door` | `CAM-001` |
| `SKIP_STATIC` | 跳过静态画面，节省费用 | `true` |
| `STATIC_THRESHOLD` | 运动灵敏度，越高越不敏感 | `5.0` |
| `DATA_DIR` | 存储目录 | `stream_data` |
| `RETENTION_DAYS` | 录像保留天数 | `3` |

**支持多摄像头配置** —— 一次设置，全屋覆盖。

### 第三步：开始使用

直接用自然语言与 AI 助手对话：

```
"开始录制"
"停止录制"  
"搜索一个搬箱子的人"
"最近2小时发生了什么？"
"查看日志"
```

**零学习成本，即装即用。**
