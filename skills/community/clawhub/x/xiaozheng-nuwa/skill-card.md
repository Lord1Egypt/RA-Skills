## Description: <br>
小郑女娲 helps an agent research public sources, distill a person's thinking patterns, and generate reusable SKILL.md files with explicit evidence and uncertainty boundaries. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[boomzikazita](https://clawhub.ai/user/boomzikazita) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Agent users can use this skill to turn public information about a person into a structured cognitive-framework skill, or ask the agent to analyze a topic through that public-source-derived perspective. It is intended for generating and reviewing markdown skill files, not for treating inferred views of real people as authoritative facts. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated persona-style skills may contain inaccurate, incomplete, or overconfident claims about real people. <br>
Mitigation: Review generated SKILL.md files before use, require public-source support for claims, and preserve explicit uncertainty and limitation notes. <br>
Risk: The skill can create persistent skill files in an OpenClaw workspace. <br>
Mitigation: Inspect generated files and run the normal skill security review before installing or invoking them. <br>
