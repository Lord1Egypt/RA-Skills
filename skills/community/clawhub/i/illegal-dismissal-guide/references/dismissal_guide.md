# 违法辞退应对参考手册

## 一、解除劳动合同的合法性判断

### 1.1 用人单位合法解除的法定情形

#### 过失性解除（第39条，无需支付补偿）

| 序号 | 情形 | 举证要点 | 常见争议 |
|------|------|---------|---------|
| 1 | 试用期被证明不符合录用条件 | 需有明确的录用条件+考核记录 | 录用条件不明确 |
| 2 | 严重违反用人单位规章制度 | 需有合法有效的规章制度+违纪事实 | 制度未公示或过于笼统 |
| 3 | 严重失职，营私舞弊，造成重大损害 | 需证明失职+因果关系+损害金额 | "重大损害"标准不明确 |
| 4 | 同时与其他单位建立劳动关系，经提出拒不改正 | 需证明双重劳动关系+催告记录 | 兼职是否构成双重劳动关系 |
| 5 | 以欺诈、胁迫手段订立或变更合同 | 需证明欺诈事实 | 学历/履历造假常见 |
| 6 | 被依法追究刑事责任 | 需法院判决 | 拘留/逮捕期间不算 |

#### 非过失性解除（第40条，需支付N+1）

| 序号 | 情形 | 程序要求 |
|------|------|---------|
| 1 | 患病或非因工负伤，医疗期满后不能从事原工作也不能从事另行安排的工作 | 提前30日书面通知或额外支付1个月工资 |
| 2 | 不能胜任工作，经培训或调岗后仍不能胜任 | 需先培训/调岗，再考核 |
| 3 | 客观情况发生重大变化，合同无法履行，协商不成变更 | 需先协商变更 |

#### 经济性裁员（第41条，需支付N）

| 条件 | 说明 |
|------|------|
| 人数 | 裁减20人以上或占职工总数10%以上 |
| 程序 | 提前30日向工会或全体职工说明情况+听取意见+向人社部门报告 |
| 优先留用 | 长期合同/无固定期限/家庭无其他就业人员且有需扶养老幼 |
| 优先录用 | 6个月内重新招用优先原职工 |

### 1.2 不得解除的情形（第42条保护）

| 序号 | 保护情形 | 例外 |
|------|---------|------|
| 1 | 从事接触职业病危害作业未做离岗前职业健康检查 | 第39条过失性解除除外 |
| 2 | 疑似职业病病人在诊断或医学观察期间 | 第39条过失性解除除外 |
| 3 | 在本单位患职业病或因工负伤被确认丧失/部分丧失劳动能力 | 第39条过失性解除除外 |
| 4 | 患病或非因工负伤在规定医疗期内 | 第39条过失性解除除外 |
| 5 | 女职工在孕期、产期、哺乳期内（"三期"） | 第39条过失性解除除外 |
| 6 | 在本单位连续工作满15年且距法定退休年龄不足5年 | 第39条过失性解除除外 |

### 1.3 违法解除的认定

以下情形均构成违法解除：

| 序号 | 违法情形 | 法律后果 |
|------|---------|---------|
| 1 | 不符合法定解除条件而解除 | 继续履行或2N赔偿金 |
| 2 | 未履行法定程序（未通知工会等） | 程序违法，解除无效 |
| 3 | 在第42条保护期内解除（非第39条） | 违法解除 |
| 4 | 以"裁员"为名单方辞退未走法定程序 | 违法解除 |
| 5 | 调岗降薪逼迫员工辞职（变相辞退） | 可主张被迫解除 |

## 二、N、N+1、2N 的区分

### 2.1 经济补偿金（N）

```
N = 工作年限 × 月平均工资
```

| 工作年限 | N值 |
|---------|-----|
| 不满6个月 | 0.5 |
| 满6个月不满1年 | 1 |
| 满1年不满1.5年 | 1 |
| 满1.5年不满2年 | 1.5（部分地区取整为2） |
| 以此类推 | 每满1年+1 |

**月平均工资：** 离职前12个月平均工资（税前，含奖金、津贴、补贴）

**封顶规则：** 月工资超过当地上年度职工月平均工资3倍的，按3倍计算，年限最高12年

### 2.2 代通知金（+1）

| 适用情形 | 说明 |
|---------|------|
| 第40条第1项（医疗期满） | 可选择提前30日通知或额外支付1个月工资 |
| 第40条第2项（不能胜任） | 同上 |
| 第40条第3项（客观情况变化） | 同上 |

> **注意：** 第39条过失性解除、第41条经济性裁员、协商解除均无+1。

### 2.3 违法解除赔偿金（2N）

```
2N = 经济补偿金 × 2
```

- 用人单位违法解除劳动合同，劳动者不要求继续履行的
- 2N与N不可同时主张，需选择其一
- 2N的封顶规则同N（3倍封顶、12年封顶）

### 2.4 选择决策树

```
用人单位解除劳动合同
├── 合法解除？
│   ├── 是 → 能否主张N或N+1？
│   │   ├── 第36条协商 → N
│   │   ├── 第40条非过失性 → N+1
│   │   ├── 第41条裁员 → N
│   │   └── 第39条过失性 → 无补偿
│   └── 否（违法解除）→ 选择：
│       ├── 要求继续履行合同 → 恢复劳动关系+补发工资
│       └── 不要求继续履行 → 2N赔偿金
```

## 三、第38条被迫解除全场景

### 3.1 可主张被迫解除的情形

| 序号 | 情形 | 法条依据 | 证据要点 |
|------|------|---------|---------|
| 1 | 未提供劳动保护/劳动条件 | 第38条第1款第1项 | 工作环境恶劣、无安全设备 |
| 2 | 未及时足额支付劳动报酬 | 第38条第1款第2项 | 工资流水显示拖欠/克扣 |
| 3 | 未依法缴纳社保 | 第38条第1款第3项 | 社保查询截图显示未缴/少缴 |
| 4 | 规章制度违法损害权益 | 第38条第1款第4项 | 制度内容+损害事实 |
| 5 | 欺诈/胁迫/乘人之危致合同无效 | 第38条第1款第5项 | 欺诈/胁迫事实 |
| 6 | 其他法律规定的情形 | 第38条第1款第6项 | 兜底条款 |
| 7 | 暴力/威胁/非法限制人身自由强迫劳动 | 第38条第2款 | 无需提前通知 |
| 8 | 违章指挥/强令冒险作业危及安全 | 第38条第2款 | 无需提前通知 |

### 3.2 被迫解除操作流程

```
步骤1：确认违法事实
  → 收集证据（工资流水、社保截图、沟通记录等）

步骤2：书面催告（建议但非必须）
  → 微信/邮件向公司催告："请于X日内补发工资/补缴社保"
  → 保留催告记录

步骤3：起草《被迫解除劳动合同通知书》
  → 明确引用劳动合同法第38条
  → 列明违法事实
  → 注明解除日期

步骤4：EMS邮寄送达
  → 邮寄到用人单位注册地址或法定代表人地址
  → 邮单备注"被迫解除劳动合同通知书"
  → 保留邮寄凭证和签收回执

步骤5：申请劳动仲裁
  → 仲裁请求：
    1. 确认双方劳动关系于X日解除
    2. 支付经济补偿金N
    3. 支付拖欠工资/补缴社保（如有）
```

### 3.3 《被迫解除劳动合同通知书》模板

```
被迫解除劳动合同通知书

致：[用人单位全称]

本人[姓名]，身份证号[XXXX]，于[入职日期]入职贵公司，担任[岗位]。

因贵公司存在以下违法行为：
1. [具体违法事实，如：自2024年5月起拖欠本人工资，每月克扣底薪XXXX元]
2. [如：未依法为本人缴纳2024年1月至6月的社会保险]

上述行为已严重违反《劳动合同法》第38条之规定。本人据此被迫解除与贵公司的劳动合同，解除日期为[日期]。

请贵公司在解除后15日内：
1. 结清拖欠工资共计XXXX元
2. 支付经济补偿金共计XXXX元
3. 出具解除劳动合同证明
4. 办理社保、公积金转移手续

如贵公司未在上述期限内履行义务，本人将依法申请劳动仲裁。

通知人：[签名]
日期：[日期]
```

## 四、常见违法辞退场景分析

### 4.1 "不胜任工作"辞退

**公司常见操作：** 以"不胜任工作"直接辞退

**法律要求：**
1. 必须有考核制度且经过民主程序和公示
2. 考核结果需有证据支撑
3. 不能胜任后需先培训或调岗
4. 培训/调岗后仍不胜任才可解除
5. 需提前30日通知或额外支付1个月工资
6. 需支付经济补偿金N+1

**常见违法点：** 跳过"培训或调岗"环节直接辞退 → 违法解除（2N）

### 4.2 "组织架构调整"辞退

**公司常见操作：** 以"部门撤销/组织架构调整"为由辞退

**法律分析：**
- 如果属于客观情况发生重大变化 → 需先协商变更，协商不成才能解除（第40条第3项），需N+1
- 如果属于经济性裁员 → 需走第41条程序，需N
- 如果直接辞退未走程序 → 违法解除（2N）

### 4.3 "试用期不合格"辞退

**公司常见操作：** 试用期最后几天以"不符合录用条件"辞退

**法律要求：**
1. 必须有明确、具体的录用条件
2. 录用条件需在入职时告知并签字确认
3. 需有客观的考核记录证明不符合条件
4. 需在试用期内做出决定并通知

**常见违法点：** 录用条件不明确、无考核记录 → 违法解除（2N）

### 4.4 "调岗降薪"逼迫辞职

**公司常见操作：** 将员工调到偏远地区或降职降薪，逼迫员工主动辞职

**法律分析：**
- 调岗需协商一致（第35条）
- 单方调岗降薪违法
- 员工可拒绝调岗，如公司以此为由辞退 → 违法解除（2N）
- 员工也可依据第38条被迫解除 → 主张N

### 4.5 "末位淘汰"辞退

**法律定性：** 违法。最高人民法院指导案例已明确"末位淘汰"属于违法解除。

## 五、赔偿金计算

### 5.1 计算公式

```
违法解除赔偿金 = 经济补偿金 × 2

经济补偿金 = 工作年限 × 月平均工资（封顶3倍社平，年限封顶12年）
```

### 5.2 各地社平工资参考（2023年度，供参考）

| 城市 | 上年度职工月平均工资 | 3倍封顶 |
|------|-------------------|---------|
| 北京 | 约13,887元 | 约41,661元 |
| 上海 | 约12,849元 | 约38,547元 |
| 深圳 | 约14,568元 | 约43,704元 |
| 广州 | 约12,693元 | 约38,079元 |
| 杭州 | 约10,739元 | 约32,217元 |
| 成都 | 约8,693元 | 约26,079元 |

> **注意：** 社平工资每年更新，以上为参考值，实际以当地人社局公布数据为准。

### 5.3 计算示例

**案例：** 北京某员工月薪20,000元，工作3年8个月，被违法辞退

- 月平均工资：20,000元（未超3倍社平41,661元，无需封顶）
- 工作年限：3年8个月 → 满6个月按1年算 → 4个月
- 经济补偿金N：20,000 × 4 = 80,000元
- 违法解除赔偿金2N：80,000 × 2 = 160,000元

## 六、仲裁请求设计

### 6.1 违法解除案件的仲裁请求清单

| 序号 | 仲裁请求 | 法律依据 | 适用情形 |
|------|---------|---------|---------|
| 1 | 确认用人单位解除劳动合同违法 | 劳动合同法第48条 | 所有违法解除 |
| 2 | 支付违法解除赔偿金（2N） | 劳动合同法第48条、第87条 | 不要求继续履行 |
| 3 | 确认劳动关系继续存续 | 劳动合同法第48条 | 要求继续履行 |
| 4 | 支付争议期间工资 | 工资支付暂行规定 | 继续履行情形 |
| 5 | 支付拖欠工资 | 劳动合同法第30条 | 有拖欠工资 |
| 6 | 支付未休年休假工资 | 职工带薪年休假条例 | 有未休年假 |
| 7 | 支付加班费 | 劳动法第44条 | 有加班事实 |
| 8 | 补缴社保 | 社会保险法第60条 | 有未缴社保（部分地区不受理） |

### 6.2 策略选择：2N vs 继续履行

| 因素 | 选择2N | 选择继续履行 |
|------|--------|-------------|
| 证据充分 | 适合 | 适合 |
| 岗位还在 | 都可以 | 适合 |
| 岗位已撤销 | 适合 | 不适合（无法履行） |
| 已找到新工作 | 适合 | 不适合 |
| 想回去上班 | 不适合 | 适合 |
| 公司经营困难 | 适合（尽快拿钱） | 不适合（可能执行不了） |

## 七、证据清单

### 7.1 违法解除核心证据

| 序号 | 证据 | 证明目的 | 优先级 |
|------|------|---------|--------|
| 1 | 解除通知书 | 证明解除事实和理由 | ★★★ |
| 2 | 劳动合同 | 证明劳动关系和薪资标准 | ★★★ |
| 3 | 工资流水（近12个月） | 计算月平均工资 | ★★★ |
| 4 | 考勤/工作记录 | 证明正常提供劳动 | ★★☆ |
| 5 | 绩效考核记录 | 证明胜任工作 | ★★☆ |
| 6 | 公司规章制度 | 审查制度合法性 | ★★☆ |
| 7 | 规章制度签收记录 | 证明是否知晓 | ★☆☆ |
| 8 | 解除前的沟通录音/聊天记录 | 证明解除真实原因 | ★★☆ |
| 9 | 同事证言 | 佐证违法解除事实 | ★☆☆ |
| 10 | 社保缴费记录 | 证明社保缴纳情况 | ★★☆ |

| 序号 | 证据 | 证明目的 |
