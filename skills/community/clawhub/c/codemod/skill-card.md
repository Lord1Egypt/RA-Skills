## Description: <br>
Use Codemod CLI whenever the user wants to migrate, upgrade, update, or refactor a codebase in a repeatable way. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[mohebifar](https://clawhub.ai/user/mohebifar) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to find, dry-run, apply, and author repeatable codemods for migrations, upgrades, deprecations, and large mechanical refactors. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Codemod CLI execution can make broad code changes during migrations or refactors. <br>
Mitigation: Review dry-run results and affected files before applying changes. <br>
Risk: Publishing or authoring codemods may leave incomplete scaffolds or stale package metadata if not verified. <br>
Mitigation: Run package validation and tests before treating authored codemods as complete. <br>
