# Qi Transformation and Form Qi Conversion

## When to Use
- 需解释万物生、长、化、灭的根本机制；
- 分析人体“生长壮老已”或植物“生长化收藏”等生命节律；
- 理解病理状态下气机紊乱如何导致疾病；
- 阐述中医调气治疗的理论依据；
- 区分渐进性变化（化）与突发性转变（变）。

## Execution Steps

1. **确认适用条件**：
   - 存在气的运动；
   - 具备阴阳互动条件；
   - 有形与无形物质共存；
   - 排除无气参与的机械物理模型或非动态系统。

2. **识别气化类型**：
   - 若变化缓慢、渐进、不明显 → 判定为“化”（量变）；
   - 若变化剧烈、显著、突发 → 判定为“变”（质变）。

3. **分析形气转化形式**：
   - **气生形**：无形之气聚合为有形之物；
   - **形化气**：有形之物消散为无形之气；
   - **气与气转化**：如地气升为云，天气降为雨；
   - **形与形转化**：在气推动下发生形态转换（如冰→水→雾）；
   - **自身更新循环**：如生命过程中的周期性变化。

4. **评估气机状态**：
   - 检查气的升降出入是否有序；
   - 若气机协调 → 生理正常；
   - 若气机紊乱 → 病理状态；
   - 若气机停息 → 生命终止。

5. **输出应用指导**：
   - 用于中医诊断（判断气机失调类型）；
   - 指导治疗（如调气、理气、补气）；
   - 支持养生建议（顺应气化规律）；
   - 解释自然与生命现象的变化逻辑。
