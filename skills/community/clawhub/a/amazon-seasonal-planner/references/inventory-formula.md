# 旺季备货计算公式详解

## 一、备货基础概念

### 1.1 关键术语

| 术语 | 定义 | 说明 |
|------|------|------|
| 日均销量 | 平日每天平均销售量 | 基础数据 |
| 旺季倍数 | 旺季销量/平日销量 | 反映旺季增长 |
| 安全库存 | 防止断货的额外库存 | 缓冲 |
| 在途库存 | 已发货但未入仓 | 提前规划 |
| 库存周转 | 库存销售一轮的时间 | 影响仓储费 |
| 补货周期 | 从下单到入仓的时间 | 关键参数 |

### 1.2 补货周期构成

```
总补货周期 = 生产时间 + 国内运输 + 清关（进口） + 国际运输 + FBA入仓

典型时间（美国FBA）：
- 国内采购：3-7天
- 生产周期：15-45天
- 国内运输：3-7天
- 清关：3-7天
- 国际运输：15-30天
- FBA入仓：3-7天

总计：约45-90天

⚠️ 旺季前物流会慢，建议提前备货
```

## 二、备货量计算公式

### 2.1 基础公式

**公式1：最低备货量**
```
备货量 = 日均销量 × 补货周期 + 安全库存

说明：
- 日均销量：取近30天平均
- 补货周期：从下单到上架的时间
- 安全库存：3-7天销量
```

**公式2：旺季备货量**
```
旺季备货量 = 预期日销量 × 旺季天数 × 安全系数

预期日销量 = 平日日销量 × 旺季倍数
安全系数 = 1.1 ~ 1.3（推荐1.2）
```

### 2.2 计算示例

**示例1：Prime Day备货**
```
产品信息：
- 平日日销量：30单
- Prime Day预期倍数：5倍
- Prime Day天数：7天
- 安全系数：1.2
- 当前库存：150单
- 在途库存：100单

计算：
预期日销量 = 30 × 5 = 150单
旺季备货量 = 150 × 7 × 1.2 = 1260单

需求缺口 = 旺季备货量 - 当前库存 - 在途库存
        = 1260 - 150 - 100
        = 1010单

结论：需要补货1010单
```

**示例2：黑五备货**
```
产品信息：
- 平日日销量：20单
- 黑五预期倍数：8倍
- 黑五网一天数：10天
- 安全系数：1.2
- 当前库存：80单
- 在途库存：120单
- 补货周期：45天

计算：
预期日销量 = 20 × 8 = 160单
旺季备货量 = 160 × 10 × 1.2 = 1920单

当前可售 = 80 + 120 = 200单
断货前天数 = 200 / 20 = 10天

需求缺口 = 1920 - 200 = 1720单

结论：需要立即补货1720单
补货时间：如果45天到，需要在40天前下单
```

## 三、批次发货策略

### 3.1 为什么分批发货

| 优势 | 说明 |
|------|------|
| 降低风险 | 避免一次发太多压仓 |
| 灵活调整 | 根据销售调整后续发货 |
| 资金压力 | 分期付款，缓解资金 |
| 仓储成本 | 减少旺季前仓储费 |

### 3.2 批次分配建议

**标准3批法**：
```
批次1：40% - T-90天发货
- 目的：建立基础库存
- 优点：早入仓，早积累权重

批次2：40% - T-60天发货
- 目的：补充库存
- 优点：持续供给

批次3：20% - T-30天发货
- 目的：应急/补货
- 优点：应对突发情况
```

**特殊情况的批次调整**：

| 情况 | 建议批次 | 说明 |
|------|----------|------|
| 第一次做旺季 | 2批就够 | 保守起见 |
| 高风险品 | 4-5批 | 降低积压风险 |
| 爆款确定性高 | 3-4批 | 加大力度 |
| 清库存 | 1批或不分批 | 不需要太多 |

### 3.3 FBA发货时间表

```
T-90天（约3个月前）
□ 确定备货方案
□ 下单第一批货
□ 发第一批FBA

T-60天（约2个月前）
□ 根据第一批销量调整
□ 下单第二批货
□ 发第二批FBA

T-30天（约1个月前）
□ 评估前两批效果
□ 下单第三批（应急）
□ 发第三批FBA

T-14天（2周前）
□ 最终库存检查
□ 决定是否补发
□ 准备应急方案
```

## 四、安全库存策略

### 4.1 安全库存公式

**公式**：
```
安全库存 = Z值 × 标准差 × √(补货周期)

Z值说明：
- 95%服务级别：Z=1.65
- 90%服务级别：Z=1.28
- 85%服务级别：Z=1.04

标准差：日销量波动
```

**简化版**：
```
安全库存 = 日均销量 × 补货周期 × 安全系数

安全系数参考：
- 稳定品（波动<20%）：1.1
- 一般品（波动20-40%）：1.2
- 波动品（波动>40%）：1.3
```

### 4.2 库存预警等级

```
🔴 紧急断货：< 3天库存
   立即补货，预计7-10天到
   考虑提高售价减缓需求

🟠 预警：3-7天库存
   立即安排补货
   暂停非必要广告

🟡 注意：7-14天库存
   关注销售趋势
   准备补货计划

🟢 安全：> 14天库存
   正常销售
   按计划补货
```

### 4.3 旺季安全系数调整

**为什么要提高安全系数**：
- 旺季物流不稳定
- 销量波动大
- FBA爆仓风险
- 清关可能延误

**旺季建议安全系数**：
| 旺季 | 建议安全系数 |
|------|-------------|
| Prime Day | 1.2-1.3 |
| 黑五网一 | 1.2-1.4 |
| 圣诞 | 1.3-1.5 |

### 5.2 旺季倍数计算方法

**方法1：历史数据参考**
```
旺季倍数 = 去年旺季日销量 / 去年平日日销量

如果没有去年数据，参考品类平均值
```

**方法2：同行参考**
```
- 查看同类产品旺季销量
- 参考论坛/社媒讨论
- 咨询服务商经验
```

```
- 取品类下限
- 预留缓冲
- 根据实际情况调整
```

## 六、仓储成本计算

### 6.1 FBA仓储费用

**月度仓储费（美国）**：
| 时段 | 标准件 | 大件 |
|------|--------|------|
| 1-9月 | $0.78/立方英尺 | $0.56/立方英尺 |
| 10-12月 | $2.40/立方英尺 | $1.40/立方英尺 |

**旺季仓储费暴增**：
```
10-12月仓储费是平时的3倍！
需要计算仓储成本
```

### 6.2 仓储成本计算示例

**示例：计算旺季仓储成本**
```
产品信息：
- 产品体积：0.02立方英尺
- 产品数量：1000件
- 旺季仓储月数：2个月

计算：
总体积 = 0.02 × 1000 = 20立方英尺
旺季仓储费 = 20 × $2.40 × 2 = $96

每件仓储费 = $96 / 1000 = $0.096

如果售价$25，利润率5%
每件利润 = $25 × 0.05 = $1.25
仓储占比 = $0.096 / $1.25 = 7.7%
```

### 6.3 避免旺季仓储费陷阱

**建议**：
1. 不要备货太多，宁可断货
2. 旺季前清理老库存
3. 计算好旺季仓储成本
4. 考虑分批入仓
5. 旺季结束后立即清仓
