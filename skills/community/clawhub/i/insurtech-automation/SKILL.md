# 保险科技 (InsurTech Automation)

## 概述

为保险行业的代理人、经纪人和消费者提供智能化保险服务能力，从产品推荐到理赔跟进一站式覆盖。利用AI推理能力实现精准风险评估和个性化保险方案。

## 核心能力

### 1. 保单智能管理
- 多保单统一视图（保险公司、险种、保额、保费、到期日）
- 续保提前预警与缴费提醒
- 保单条款快速查询与解读

### 2. 理赔流程自动化
- 理赔材料清单自动生成（按险种/事故类型）
- 理赔进度追踪与催办
- 理赔金额预估（基于历史赔付数据与条款匹配）

### 3. 风险评估建模
- 个人/企业风险画像（健康、财产、责任、信用）
- 保险缺口分析（已有保障 vs 实际需求）
- 行业风险数据库查询（地区灾害、行业事故率）

### 4. 产品对比与推荐
- 同类保险产品多维对比（保障范围、价格、免责条款、理赔率）
- 基于用户画像的个性化产品推荐
- 保费试算与性价比分析

### 5. 合规与监管辅助
- 保险条款合规性审查（是否符合最新监管要求）
- 销售话术合规检查
- 监管政策变动追踪（银保监会、保协最新文件）

## 使用方式

```
分析我的保单 [保单文件路径] 是否存在保障缺口
为 [年龄/职业/健康状况] 推荐适合的重疾险
检查 [理赔材料] 是否齐全，预估理赔金额
对比 [产品A] 和 [产品B] 的优劣
```

## 输出格式

- **保单总览**：多险种保障矩阵表格
- **风险报告**：风险评分 + 缺口清单 + 建议方案
- **产品对比**：多维度对比表格 + 综合推荐
- **理赔指引**：材料清单 + 流程步骤 + 时效预估

## 适用场景

- 保险代理人提升展业效率与专业度
- 个人/家庭保险规划与管理
- 企业团险方案设计与优化
- 保险科技平台的智能核保辅助

## 注意事项

- 保险产品数据需要定期更新以保持时效性
- 理赔金额预估仅供参考，实际以保险公司核定为准
- 数据底座覆盖中国保险行业协会公开数据及主流保险公司产品库
- 涉及具体保单的操作需用户确认后执行
