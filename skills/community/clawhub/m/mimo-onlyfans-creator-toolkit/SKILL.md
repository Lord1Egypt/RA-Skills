# 订阅制创作者运营工具 💎

> 从0到1搭建订阅制运营体系。给你30天路线图+粉丝分层CRM+收入结构优化方案，不是"多发内容"的废话。

## 触发条件

### ✅ 匹配（满足任一即触发）
- 订阅制 / 创作者变现 / 粉丝经济 / 内容付费 / 订阅收入
- OnlyFans / OF / 订阅平台 / Fansly / Patreon
- DM自动化 / 粉丝管理 / 内容排期 / 内容日历
- 定价策略 / PPV / 小费策略 / 收入优化
- 粉丝分层 / 高价值粉丝 / 沉睡粉丝唤醒 / 留存
- creator monetization / subscription / fan management / PPV

### ❌ 排除
- 内容审核 / 违规申诉 → 联系平台客服
- 法律/税务问题 → 咨询专业人士
- 平台技术故障 → 联系平台支持

### 🎯 上下文条件
- "刚起步" / "新手" → 30天入门路线图
- 有收入数据 → 针对性优化分析
- "收入下降" → 诊断流失+召回策略

## 核心流程（6 Steps）

### Step 1: 账号定位与人设
细分赛道（越细越好）→ 竞品分析（定价/内容/引流）→ 差异化人设 → 受众画像 → 阶段目标 → 内容调性

### Step 2: 定价策略
订阅阶梯（新手$4.99→成长$9.99→成熟$19.99+）→ PPV定价（普通$3-15/精选$15-50/限量$50-200）→ 小费引导 → 捆绑销售 → 限时优惠

### Step 3: 内容策略
内容日历（周度排期）→ 类型配比（免费30-40%/订阅40-50%/PPV10-20%/互动5-10%）→ 批量生产工作流 → A/B测试

### Step 4: 粉丝CRM
DM自动化（新粉欢迎→首购→复购序列）→ 粉丝分层（新粉/活跃/高价值/沉睡/流失）→ 流失预警（3天/7天/14天）→ 反馈收集

### Step 5: 引流推广
Instagram/Twitter/TikTok/Reddit多平台引流 → 合作互推 → 聚合页 → 邮件列表 → 引流与付费内容边界

### Step 6: 数据+增长
MRR/ARPU/LTV/留存率追踪 → 收入结构优化（目标40%订阅+40%PPV+20%小费）→ 推荐计划 → 社群 → 定制服务 → 风险预案

## 关键约束
1. 合规运营，遵守平台TOS和法律法规，年满18岁
2. 收入估算基于行业均值，实际因个人和市场而异
3. 不使用欺骗性营销（虚假承诺/误导宣传）
4. 关注创作者心理健康，不鼓励过度劳动
5. 完整工具详解、DM流程、定价参考见 references/details.md

## Anti-rationalization 反借口规则
1. 禁止说"先发内容再说，定价以后想" — 定价策略是第一天就要确定的核心决策
2. 禁止说"DM不重要，粉丝会自己看" — DM自动化是PPV转化的第一渠道，贡献30%+收入
3. 禁止说"不用做数据分析" — 没有数据驱动的运营就是盲猜，浪费内容产能
4. 禁止说"引流太早了，先做内容" — 冷启动期引流和内容同等重要，0粉丝=0收入
5. 禁止说"粉丝分层太复杂" — 不做分层就是平均用力，高价值粉丝需要差异化维护
