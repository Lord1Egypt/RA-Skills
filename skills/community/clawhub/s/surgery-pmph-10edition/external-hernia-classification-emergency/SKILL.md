# 腹外疝临床分型与紧急处理

## 触发条件
当患者出现以下任一情况时使用本技能：
- 疝块无法回纳
- 局部疼痛加剧
- 疝内容物颜色改变或伴有全身中毒症状

## 执行流程

1. **判断疝的临床类型**：
   - **易复性疝**：疝内容物可完全回纳入腹腔，平卧或手动即可回纳，常伴“咕噜声”。
   - **难复性疝**：内容物不能完全回纳，但无血运障碍；滑动性疝属此类，常见于右侧。
   - **嵌顿疝**：疝囊颈卡住内容物，表现为肿块突然增大、紧张发硬、明显疼痛；注意特殊类型（Richter疝、Littre疝、Maydl疝）。
   - **绞窄性疝**：嵌顿未解除导致动脉血流中断，肠壁变黑坏死，疝囊渗液呈暗红或脓性。

2. **识别演变路径**：
   易复性 → 难复性 → 嵌顿性 → 绞窄性（不可逆）。

3. **紧急处理原则**：
   - 嵌顿/绞窄疝需急诊手术。
   - 手术中必须探查腹腔内肠袢，尤其警惕Maydl疝中可能被遗漏的中间坏死肠段。

> **警示**：一旦怀疑绞窄，不得延误手术评估。
