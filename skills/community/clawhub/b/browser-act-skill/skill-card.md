## Use Case: <br>
Developers and AI agent users use this skill to route browser automation requests through browser-act for rendered content extraction, authenticated workflows, form interaction, screenshots, network capture, and session-aware browsing. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill enables broad browser automation, including session, form, screenshot, and network-capture actions. <br>
Mitigation: Install only when full browser automation is needed, use isolated browser profiles, and manually review login, form submission, file upload, captcha-solving, and authenticated account actions before allowing them. <br>
Risk: Connecting to a personal or work Chrome session can expose local browser context to automation. <br>
Mitigation: Prefer isolated browser profiles and connect to personal or work Chrome sessions only when necessary and explicitly confirmed. <br>
