## Description: <br>
Processes image folders in batches for compression, resizing, watermarking, format conversion, and renaming. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[freedompixels](https://clawhub.ai/user/freedompixels) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, operators, and content teams use this skill to run local batch image processing for ecommerce images, social media assets, ID photos, and blog images. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Processed files can overwrite existing names in the chosen output folder. <br>
Mitigation: Run the tool on copies or write to a dedicated output directory before replacing originals. <br>
Risk: Compression, resizing, and format conversion can reduce image quality or change visual appearance. <br>
Mitigation: Review representative outputs and keep backups before applying results to valuable images. <br>
