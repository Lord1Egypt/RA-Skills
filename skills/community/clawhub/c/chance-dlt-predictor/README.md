# Lottery Data Analysis & Number Generator (DLT/Da Le Tou) / 大乐透预测分析师#

> **English:** AI-powered China Sports Lottery "Da Le Tou" (DLT, Super Lotto) professional analysis tool. Integrates 10+ methodologies: frequency heatmap, omission analysis, trend analysis, odd/even & range analysis, consecutive numbers, span analysis, Monte Carlo simulation, and Python data analysis. Helps lottery players select numbers scientifically and bet rationally. Probability reference only.

**Keywords:** lottery, DLT, Da Le Tou, number prediction, data analysis, lottery strategy, Chinese lottery*

## ✨ Features#

- ✅ **10 Analysis Methods** — frequency heatmap, omission value, odd/even ratio, big/small ratio, sum value, zone distribution, consecutive numbers, span, remainder (0/1/2 road), Monte Carlo
- ✅ **Monte Carlo Simulation** — generates candidate numbers with multi-dimensional filters, outputs top N recommendations
- ✅ **Three Play Types** — straight pick (direct select), group 3, group 6 full coverage
- ✅ **Quick Templates** — conservative (low risk), balanced (comprehensive), aggressive (cold number rebound)
- ✅ **Rational Betting Guide** — expected value calculation, budget control, responsible gambling reminder*
