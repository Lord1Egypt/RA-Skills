# 量子力学基础概念

## 一、量子态与叠加原理

### 1.1 量子态 (Quantum State)

量子态是描述量子系统物理状态的数学表示，通常用Dirac符号`|ψ⟩`表示。

**基本概念**
- **纯态**：可用单一波函数完整描述的量子态
- **混态**：由多个纯态以不同概率混合而成的状态
- **基态**：系统的最低能量状态
- **激发态**：高于基态的能量状态

### 1.2 叠加原理 (Superposition Principle)

量子叠加原理指出，量子系统可以同时处于多个量子态的叠加。

**数学表达**
```
|ψ⟩ = α|0⟩ + β|1⟩
```
其中α和β为复振幅，满足|α|² + |β|² = 1

**物理意义**
- 测量前，系统同时处于|0⟩和|1⟩态
- 测量时，波函数坍缩到某一确定态
- 概率由振幅的模平方决定

## 二、量子纠缠 (Quantum Entanglement)

### 2.1 定义

量子纠缠是一种量子力学现象，其中两个或多个粒子的量子态相互关联，以至于不能单独描述单个粒子的状态。

### 2.2 纠缠态类型

**Bell态（最大纠缠态）**
```
|Φ⁺⟩ = (|00⟩ + |11⟩)/√2
|Φ⁻⟩ = (|00⟩ - |11⟩)/√2
|Ψ⁺⟩ = (|01⟩ + |10⟩)/√2
|Ψ⁻⟩ = (|01⟩ - |10⟩)/√2
```

**Dicke态**
多个相同二能级系统的对称纠缠态：
```
|D,N⟩ = (N!)^(-1/2) Σ permutations |0⟩⊗(N-k) ⊗ |1⟩⊗k
```

### 2.3 纠缠在量子电池中的作用

- **量子加速**：纠缠减少Hilbert空间遍历状态数
- **集体效应**：多体纠缠增强光-物质耦合
- **功率增强**：纠缠态充电功率可超线性扩展

## 三、量子相干性与退相干

### 3.1 量子相干性 (Quantum Coherence)

量子相干性指量子态之间存在确定的相位关系，是量子干涉现象的基础。

**度量指标**
- **l₁范数相干量**：C_l₁(ρ) = Σ_{i≠j} |ρ_ij|
- **相对熵相干量**：C_rel(ρ) = S(ρ_diag) - S(ρ)
- **鲁棒性相干量**：RC(ρ)

### 3.2 退相干 (Decoherence)

退相干是量子系统与环境相互作用导致量子相干性丧失的过程。

**退相干来源**
- 热扰动
- 电磁噪声
- 机械振动
- 材料缺陷

**退相干时间 (T₂)**
- 超导量子比特：约100微秒
- 离子阱：可达秒级
- 有机分子系统：飞秒至纳秒级

### 3.3 退相干的双刃剑效应

在量子电池中，退相干具有矛盾作用：
- **有害面**：破坏量子相干性，降低量子优势
- **有益面**：抑制超辐射发射，稳定储能

## 四、能级系统与量子跃迁

### 4.1 二能级系统 (Two-Level System, TLS)

最简单的量子系统，仅有两个能量本征态。

**哈密顿量**
```
H = (ℏ/2)ωσ_z
```
其中σ_z为泡利Z矩阵

**Bloch球表示**
- |0⟩态：北极
- |1⟩态：南极
- 叠加态：球面上其他点

### 4.2 三能级系统

在量子电池中用于实现：
- 快速充电（单态激发）
- 稳定储能（三重态存储）
- 高效提取（电荷分离）

### 4.3 量子跃迁

系统在不同能级之间切换的过程。

**选择定则**
- 电偶极跃迁选择定则：Δl = ±1
- 禁戒跃迁：违背选择定则的跃迁

## 五、光-物质相互作用

### 5.1 Rabi振荡

光场驱动下量子态的周期性演化。

**Rabi频率**
```
Ω_R = μE/ℏ
```
μ：电偶极矩，E：电场强度

### 5.2 强耦合与弱耦合

**弱耦合**
- 耦合强度g < 退相干率κ, γ
- 光-物质相互作用可忽略
- 典型吸收谱：Lorentzian线形

**强耦合**
- g > κ, γ
- 形成准粒子：极化激元(Polariton)
- 光场与物质态混合
- 可观测Rabi劈裂

### 5.3 Jaynes-Cummings模型

描述二能级原子与单模光场相互作用的理论模型。

**哈密顿量**
```
H = ℏω_aσ_z/2 + ℏω_c a†a + ℏg(σ⁺a + σ⁻a†)
```

## 六、量子多体效应

### 6.1 超辐射 (Superradiance)

相干辐射过程，辐射强度与N²成正比（N为原子数）。

**Dicke超辐射**
- 1964年由Robert Dicke提出
- 发射时间尺度：τ ∝ 1/N
- 峰值功率：P ∝ N²

### 6.2 超吸收 (Superabsorption)

超辐射的逆过程，吸收速率超线性增长。

**关键公式**
- 吸收速率：Γ ∝ N（线性）或 N²（超线性）
- 充电时间：τ_c ∝ 1/√N 或 1/N
- 功率密度：P/N ∝ √N 或 N

### 6.3 集体效应 (Collective Effects)

多粒子系统中出现的协同行为。

**超扩展特性**
```
充电功率密度 ∝ N^α, 其中 α > 0
```

## 七、量子热力学基础

### 7.1 量子功与量子热量

**量子功**：幺正演化过程中传递的能量
```
W = Tr[H(t)ρ(t)] - Tr[H(0)ρ(0)]
```

**量子热量**：耗散过程中传递的能量

### 7.2 可提取能 (Ergotropy)

从量子态中可提取的最大功。

**定义**
```
ρ → σ = UρU†, 其中 σ为被动态
Ergotropy = Tr[Hρ] - min_U Tr[H UρU†]
```

### 7.3 量子资源理论

**量子资源种类**
- 量子纠缠
- 量子相干性
- 量子非定域性

**资源量化**
- 纠缠度量：形成纠缠、蒸馏纠缠
- 相干度量：l₁范数、相对熵
