# 强度分（intensityScore）推导

## 目标

给每场比赛算一个 0-100 的"该看指数"，综合：
- 两队实力（强强对话加分）
- 赛事阶段（淘汰赛 > 小组赛）
- 恩怨局加成
- 球星/流量加成（适度）

## 公式

```
strengthBase   = 100 * (1 - (rank_a + rank_b) / 200)
                 # rank_a, rank_b 来自 teams.json.fifaRank
                 # 范围 ~50 (#1+#1) 到 ~0 (#100+#100, 但实际 48 队 max~95)
                 # 实际有效范围: 50 (#1+#1) ~ 0 (#95+#95)

stageBonus     = { group: 0, R32: 3, R16: 8, QF: 15, SF: 25, F: 40, third: 8 }
                 # 阶段越靠后加权越大

rivalryBonus   = 10  if (a, b) in rivalries.rivalries else 0
                 # 命中恩怨/德比/历史决赛

starBonus      = min(8, sum of trios where trio.team in {a, b} * 1.5)
                 # 球星/流量加成，但封顶 8 避免通胀

intensityScore = min(100, round(strengthBase + stageBonus + rivalryBonus + starBonus, 1))
```

## 实测样例

| 对阵 | FIFA 排名 | 阶段 | 恩怨 | 球星 | 强度分 |
|---|---|---|---|---|---|
| 法国 vs 阿根廷 | 2 + 1 | 小组赛 | ✓ (2022 Final) | Mbappé(5)+Messi(8) | 50 + 0 + 10 + 8 = **68** |
| 英格兰 vs 克罗地亚 | 4 + 12 | 小组赛 | ✓ (2018 SF) | Bellingham(5) | 83 + 0 + 10 + 5 = **98** |
| 巴西 vs 海地 | 5 + 84 | 小组赛 | – | – | 56 + 0 + 0 + 0 = **56** |
| 南非 vs 韩国 | 60 + 22 | 小组赛 | – | – | 59 + 0 + 0 + 0 = **59** |
| 阿根廷 vs 法国 | 1 + 2 | 决赛 | ✓ | – | 99 + 40 + 10 + 8 = **100**（封顶）|
| 库拉索 vs 沙特 | 82 + 56 | 小组赛 | – | – | 31 + 0 + 0 + 0 = **31**（菜鸡候选）|
| 巴拿马 vs 加纳 | 45 + 73 | 小组赛 | – | – | 41 + 0 + 0 + 0 = **41**（无噱头候选）|

## 阈值建议

- **激情（≥70）**：跨级强强对话 + 恩怨 + 关键阶段。约 15-20 场（含淘汰赛）
- **值得看（≥55）**：大多数淘汰赛 + 强强小组赛。约 30-40 场
- **随便看看（<55）**：默认全开，约 60+ 场

## 调参点

如用户觉得：
- 强度分普遍偏高 → 把 `rivalryBonus` 降到 5
- 强度分普遍偏低 → 把 `strengthBase` 乘 1.2
- 淘汰赛想更突出 → `stageBonus` 整体 +5

修改时同步更新 SKILL.md 的公式说明。
