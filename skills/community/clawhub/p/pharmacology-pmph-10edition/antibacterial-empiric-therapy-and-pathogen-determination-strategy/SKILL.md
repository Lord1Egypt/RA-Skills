# antibacterial-empiric-therapy-and-pathogen-determination-strategy

## 何时使用本技能
当患者出现感染症状（如发热、脓毒症、局部感染征象）且满足以下条件时启用：
- 可获取感染部位样本（如血液、痰液、尿液等）；
- 病原体尚未明确；
- 不适用于已知非细菌性感染（如单纯病毒性上感）或病原已明确无需调整用药的情况。

## 执行步骤

### 1. 立即采集病原学样本
- 在开始抗菌治疗前，从感染部位（如痰、血、尿、脑脊液等）采集样本送检；
- 进行细菌培养与分离。

### 2. 启动经验性治疗（如病情危重）
若患者病情严重（如高热、休克、脓毒症），在等待病原结果期间，基于以下因素选择初始抗菌药物：
- **临床诊断推测最可能致病菌**（如社区获得性肺炎→肺炎链球菌；尿路感染→大肠埃希菌）；
- **本地或本院细菌耐药监测数据**（如大肠埃希菌对氨苄西林耐药率高）；
- **抗菌药物特性**：覆盖目标病原、良好药动学/药效学、组织穿透力。

### 3. 动态评估与调整
- 若48–72小时内临床无改善，重新评估病原可能性并调整治疗方案；
- 一旦获得药敏结果，立即转为针对性（目标）治疗，避免广谱或不必要的联合用药。

### 4. 避免常见错误
- 不在无细菌感染证据时使用抗菌药物；
- 不在病原明确后继续经验性广谱方案；
- 不忽略重复采样以提高病原检出率。

> 提示：本技能核心是“先采样、再用药、快调整”，确保抗菌治疗精准、及时、合理。
