## Description: <br>
Live Task Pulse helps agents track multi-step task progress through a Python CLI, persisted task JSON, and live chat progress notifications. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[siaslfs](https://clawhub.ai/user/siaslfs) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agent operators use this skill to coordinate multi-step work, publish live progress updates, detect stalled tasks, and preserve task state for recovery. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The local task CLI can be misused to read or modify JSON files outside the intended task directory. <br>
Mitigation: Review before installing and restrict use to task IDs generated by the create command, or patch task ID validation before use in shared or sensitive workspaces. <br>
Risk: Task names, results, progress messages, or error text may expose secrets or sensitive information in chat notifications or local task files. <br>
Mitigation: Do not include secrets, private URLs, customer data, credentials, or sensitive error details in task names, results, or progress updates. <br>
