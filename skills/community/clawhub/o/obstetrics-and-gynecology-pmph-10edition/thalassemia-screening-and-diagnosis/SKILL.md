# 地中海贫血筛查与诊断流程

## 何时使用
- 适用对象：计划妊娠女性、孕妇、地贫高发地区人群
- 触发条件：血常规显示 **Hb 正常或减少**、**MCV < 82 fl** 且 **MCH < 27 pg**
- 前提条件：处于妊娠前或妊娠早期；夫妻一方为地贫患者或携带者
- 排除条件：未排除缺铁性贫血等非遗传性小细胞低色素性贫血者不得直接启动本流程

## 执行步骤
1. **初筛判断**：确认血常规满足以下任一组合即视为筛查阳性：
   - Hb 正常或降低
   - MCV < 82 fl
   - MCH < 27 pg

2. **HbA2 检测分型**：对筛查阳性者检测 HbA2 水平以初步分型：
   - 若 HbA2 < 2.5%，提示 α 地贫
   - 若 HbA2 > 3.5%，提示 β 地贫

3. **基因检测确诊**：所有筛查阳性者必须进行地贫基因检测，以明确诊断和具体基因型。

4. **特殊情况处理**：
   - 若夫妻有重型地贫胎儿分娩史但常见地贫基因检测阴性，需排查罕见地贫基因突变
   - 对疑似重型 α 地贫胎儿，结合超声指标辅助诊断（如 NT 增厚、心胸比增大、胎盘增厚等）

5. **重要提醒**：血常规异常仅作为筛查阳性依据，**不能用于确诊**，确诊必须依赖基因检测。

## 输出结果
根据筛查与检测结果决定是否启动产前诊断，并将阳性者转诊至遗传咨询门诊。
