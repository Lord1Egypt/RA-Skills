# Wiki Queries — 问答库

> 本目录存储有价值的投资问答记录，经过整理和验证的 Q&A 条目。

## 说明

本目录用于存放从投资者交流、电话会议、业绩发布会等渠道收集并整理的有价值问答。

## 文件命名规范

```
YYYY-MM-DD-[问题关键词简要描述].md
```

## 条目结构

每个问答文件应包含：

```markdown
# [问题摘要]

> **来源:** [渠道]
> **日期:** YYYY-MM-DD
> **问:** [原始问题]
> **答:** [整理后回答]

## 补充说明

[如有背景信息或交叉引用]
```

## 与 entities/ 的区别

- **entities/**: 客观事实类实体（公司、产品、人物）页面
- **queries/**: 主观/互动类问答记录，需要来源和上下文

## 维护人

由 Ingest 流程自动写入，人工仅在特殊情况下修正。
