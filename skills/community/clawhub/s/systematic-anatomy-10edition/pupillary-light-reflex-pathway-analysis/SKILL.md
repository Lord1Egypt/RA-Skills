# 瞳孔对光反射通路分析

## 何时使用
- 在神经系统查体中发现瞳孔对光反射异常
- 评估昏迷患者是否存在脑干功能障碍
- 测试动眼神经功能完整性
- 需要区分传入（视神经）与传出（动眼神经）通路损伤

## 执行步骤
1. **确认检查条件**：排除药物性瞳孔改变等干扰因素。
2. **执行光照测试**：分别照射左、右眼，观察双侧瞳孔的直接与间接对光反射。
3. **分析反射通路**：
   - 光线刺激视网膜 → 视神经传入 → 视交叉部分交叉 → 视束 → 顶盖前区（中脑）
   - 顶盖前区投射至双侧动眼神经副核（Edinger-Westphal核）
   - 动眼神经副核发出节前纤维 → 睫状神经节换元 → 节后纤维支配瞳孔括约肌
4. **解读异常模式**：
   - **一侧视神经损伤**：光照患侧眼时双侧无反应；光照健侧眼时双侧有反应（间接反射存在）
   - **一侧动眼神经损伤**：无论光照哪侧眼，患侧瞳孔均无收缩，但健侧眼反射正常
5. **临床判断**：
   - 反射完全消失需警惕脑干功能衰竭，但必须首先排除周围神经（视神经或动眼神经）损伤
   - 结合其他脑干反射综合评估是否提示病危状态

> 注意：本技能不适用于已知存在药物性瞳孔改变的情况。
