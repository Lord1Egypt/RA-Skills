## 对标分析方法

### 1. 横向对标（同一行业同类事件）

```
当前事件：某航司机械故障
对标案例：某航司机械故障事件
对比维度：发生→爆发转换时间、高峰期覆盖媒体数、应对时效
```

### 2. 纵向对标（同一主体历史事件）

```
当前事件：某品牌第二次产品争议
对标案例：该品牌上一次产品争议事件
对比维度：回应速度提升/恶化、公众容忍度变化、品牌信用消耗
```

### 3. 跨行业参照

```
当前事件：某服务行业投诉事件
参照案例：某快递延误事件
借鉴点：赔偿方案设计、透明化沟通节奏
```

## 案例库使用原则

1. **时间维度优先**：先对齐事件阶段（发生1h vs 2h），再比较应对策略
2. **类型匹配第一**：同一类型的事件可比性最强
3. **行业差异标注**：跨行业对标时标注行业特征差异
4. **场景变量控制**：标注外部环境（政策风向、社会情绪、特殊时期）
5. **动态更新**：新案例持续入库，淘汰过时案例
