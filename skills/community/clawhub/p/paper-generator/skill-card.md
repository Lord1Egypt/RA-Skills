## Description: <br>
Generates a Markdown academic paper scaffold in English or Chinese from a selected template. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[laojun509](https://clawhub.ai/user/laojun509) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and researchers use this skill to create a starting Markdown draft for an academic paper by choosing a template, language, title, and output path. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The generator writes to a user-supplied output path and may overwrite an existing file. <br>
Mitigation: Check the output path before running and use a working directory where file creation is expected. <br>
Risk: The artifact appears less capable than advertised, including no provided PDF conversion or LaTeX-template implementation. <br>
Mitigation: Treat Markdown generation as the supported behavior and verify any PDF or LaTeX workflow separately before relying on it. <br>
