# 交底书信息收集清单

在撰写前确认以下信息（缺失则向用户追问）：

## 必填

- [ ] 发明名称（准确、能体现技术特征）
- [ ] 技术领域
- [ ] 现有技术是什么？存在什么问题？
- [ ] 本发明要解决的技术问题（一句到一段）
- [ ] 技术方案（怎么实现的？核心部件/步骤）
- [ ] 至少 1 个完整实施例
- [ ] 有益效果（相比现有技术的改进）

## 建议

- [ ] 附图或示意图（可手绘描述）
- [ ] 关键参数、材料、工艺条件
- [ ] 是否已有论文/产品/上线版本（影响新颖性）
- [ ] 目标申请类型（发明/实用新型）
- [ ] 是否需保密或加快审查

## 检索（patent-search）

- [ ] 已用关键词检索现有技术
- [ ] 已记录 3–5 篇最接近对比文献
