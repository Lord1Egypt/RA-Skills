# 京东自营实时热销榜

提供实时热销频道商品查询工具，展示当前最畅销的好货，基于京东官方数据。仅筛选京东自营商品，品质有保障。默认精选好评≥98%、近30天销量≥2000的品质商品，结果不足时自动逐级降级（先降销量到≥1000，再降到≥500，再到≥200，最后不限销量），好评率≥98%固定不降级，保证每次查询都有推荐。

## 能做什么

- **实时热销查询**：展示当前最热销的商品，已精选好评≥98%、销量≥2000的品质商品
- **品类筛选**：按关键词筛选特定品类的热销商品（支持模糊匹配，如"家电"可匹配"家用电器"）
- **多种排序**：综合评分、到手价升序、折扣力度降序
- **品质自定义**：可自定义最低好评率（98/99/100），好评率固定不降级，只放宽销量门槛
- **动态降级**：结果不足时自动放宽销量门槛（2000→1000→500→200→不限），好评≥98%不降级，保证推荐数量
- **翻页浏览**：每页50条，说"下一页"查看更多

## 不能做什么

- 不能下单购买，只提供商品信息和购买链接
- 不能设置价格变动提醒通知
- 不能查询历史价格走势

## 使用示例

1. "现在什么卖得最火" → 返回热销推荐第1页（50条）
2. "热销的家电有哪些" → 按品类筛选
3. "100元以内的热销好货" → 按价格筛选
4. "折扣最大的热销商品" → 按折扣力度排序
5. "下一页" → 查看更多商品

## 注意事项

- 热销排名实时变化，以京东页面实际数据为准
- 热销商品库存可能紧张，部分可能售罄
- 仅筛选京东自营商品，专卖店和个人店已自动剔除
- 好评率≥98%为品质标准，固定不降级
- **数据流向说明**：查询通过云端代理转发到京东官方API，代理不存储用户数据

## 使用提示

- keyword为空时返回全部热销推荐，每页50条
- 带品类关键词时结果较少属正常现象，说明符合条件的好货稀缺
- sort=discount按折扣力度排序，适合找折扣最大的好货
- sort=price按到手价升序，适合找最便宜的好货
- 商品更新频繁，建议多翻几页发现更多好货
- 每条商品包含商品图片URL（image_url），请用 ![商品名](image_url) 格式渲染图片，图片下方放购买链接
- 购买链接点击后自动识别设备，手机端有京东APP则唤起APP
