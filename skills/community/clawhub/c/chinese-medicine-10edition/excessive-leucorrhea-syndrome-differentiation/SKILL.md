# Excessive Leucorrhea Syndrome Differentiation

## When to Use
患者带下量明显增多，且色、质、气味异常，或伴局部及全身症状，已排除排卵期、妊娠期等生理性带下及阴道、宫颈恶性肿瘤。

## Execution Steps

### 第一步：确认核心病机
任脉损伤，带脉失约，湿邪为患。

### 第二步：分型辨治
- **脾虚**：带下色白、质稀如涕、无臭，伴纳少便溏、体倦乏力 → 完带汤
- **肾阳虚**：带下清稀如水，畏寒肢冷，腰背冷痛 → 内补丸
- **阴虚夹湿**：带下色黄或赤白相兼，质稠有臭，五心烦热 → 知柏地黄丸
- **湿热下注**：带下黄稠臭秽，阴部瘙痒，口苦纳呆 → 止带方
- **湿毒蕴结**：带下黄绿如脓，臭秽难闻，小腹胀痛 → 五味消毒饮

### 第三步：治疗总则
- 治脾：宜运、宜升、宜燥
- 治肾：宜补、宜固、宜涩
- 湿热热毒：宜清、宜利
- 阴虚夹湿：补清兼施

## Precautions
- 避免过用苦寒或温燥药物，以防伤正。
- 不适用于带下过少患者。
