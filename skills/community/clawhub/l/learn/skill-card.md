## Description: <br>
Structure and track learning with spaced repetition and active recall across any domain. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ivangdavila](https://clawhub.ai/user/ivangdavila) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Employees, external users, and developers use this skill to create local learning plans, practice concepts with active recall, track mastery, and schedule spaced repetition reviews across any subject. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill retains study topics, notes, quiz progress, preferences, and review schedules locally under ~/learn. <br>
Mitigation: Install only if local learning-history retention is acceptable; review or delete ~/learn when that history should no longer be retained. <br>
