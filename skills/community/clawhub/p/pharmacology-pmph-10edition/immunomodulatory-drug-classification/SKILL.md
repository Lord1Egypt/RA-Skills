# immunomodulatory-drug-classification

## 何时使用
- 药物被提及在“影响免疫功能的药物”相关章节或索引中；
- 需明确该药对免疫系统的作用方向（抑制/增强/调节）；
- 用于器官移植、自身免疫病、免疫缺陷或肿瘤免疫治疗等场景的药物评估。

## 分类标准
1. **免疫抑制剂**：抑制T/B细胞活化或增殖，用于抗排斥或自身免疫病。
   - 代表药：环孢素、他克莫司、吗替麦考酚酯、来氟米特、硫唑嘌呤、芬戈莫德、环磷酰胺。
2. **免疫增强剂**：提升细胞或体液免疫功能，用于免疫缺陷或慢性感染。
   - 代表药：胸腺素、转移因子、匹多莫德、异丙肌苷、干扰素、白介素-2、卡介苗。
3. **生物免疫调节剂**：靶向特定免疫通路的单抗或融合蛋白。
   - 代表药：利妥昔单抗、阿达木单抗、莫罗单抗-CD3、泰它西普。

## 执行步骤
1. 确认药物是否列于“第五十章 影响免疫功能的药物”或相关页码（如463–469）；
2. 根据药物名称匹配上述三类之一；
3. 结合适应症判断主要免疫作用方向；
4. 注意部分药物（如糖皮质激素）虽具免疫抑制作用，但需结合上下文确认是否归入本类。

> 提示：返回结果应包含药物类别（抑制/增强/生物调节）及参考页码。
