# 抖音爆款标题生成器

## 功能介绍

自动生成符合抖音平台算法的爆款视频标题，显著提高视频播放量和用户互动率。

## 快速开始

### 基础使用

```bash
# 生成单个标题
python main_simple.py --topic "美食制作" --style "搞笑" --count 1

# 批量生成多个标题
python main_simple.py --topic "健身减脂" --style "励志" --count 3

# 不同风格生成
python main_simple.py --topic "美妆教程" --style "实用" --count 2
```

### 支持的风格

- **搞笑**：幽默风趣，引发笑声
- **励志**：正能量，激励人心
- **实用**：干货教程，实用价值
- **情感**：情感共鸣，触动人心
