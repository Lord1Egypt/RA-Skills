# Constipation Syndrome Differentiation

## When to Use
- 新发便秘伴口干、尿黄（热结）
- 老年便秘伴乏力、气短（气虚）
- 产后便秘伴面色萎黄（血虚）
- 久病便秘伴畏寒、腰冷（阳虚）

## Execution Steps
1. **辨别虚实**：
   - 实秘：热结、气滞、寒凝；
   - 虚秘：气虚、血虚、阴虚、阳虚。
2. **匹配治法与方药**：
   - 热结便秘 → 清热润肠 → 麻子仁丸；
   - 气滞便秘 → 顺气行滞 → 六磨汤；
   - 气虚便秘 → 益气健脾 → 黄芪汤；
   - 血虚便秘 → 养血润燥 → 润肠丸；
   - 阴虚便秘 → 滋阴补肾 → 六味地黄丸；
   - 阳虚便秘 → 温阳益肾 → 济川煎。
3. **处理兼夹证**：根据主次轻重灵活配伍，标本兼顾。

> **注意**：虚秘禁用苦寒泻下，“通”指恢复大肠传导功能，非单纯泻下。
