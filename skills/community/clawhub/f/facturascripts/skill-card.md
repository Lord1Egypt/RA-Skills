## Use Case: <br>
Developers and engineers use this skill as Spanish-language FacturaScripts 2025 reference material to create plugins, work with models, controllers, views, database schemas, translations, security, and REST API integrations. ## Known Risks and Mitigations: <br>
Risk: Broad activation wording may cause the skill to influence unrelated or sensitive ERP tasks. <br>
Mitigation: Use it as FacturaScripts reference material only and require human review before applying generated changes. <br>
Risk: Generated API or MCP code may perform write-capable ERP actions such as creating invoices, changing stock or accounting data, or deleting records. <br>
Mitigation: Test generated integrations in staging first, use scoped API keys, and require explicit human approval before write or delete operations. <br>
Risk: Prompts or generated source could expose real API tokens or credentials. <br>
Mitigation: Keep real tokens out of prompts and source code, and use scoped credentials for any FacturaScripts API access. <br>
