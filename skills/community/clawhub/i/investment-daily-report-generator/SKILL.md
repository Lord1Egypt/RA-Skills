# 技能名称
招商日报生成器

## 技能定位（用户可见）
根据关注地点（城市/区县），自动生成结构化招商日报，涵盖重点政策解读、产业机会分析、任务清单及政策工具包，辅助招商团队每日决策。

## 技能描述（LLM识别）
接收地点参数（如“银川市兴庆区”）及报告偏好（目前仅支持“招商报告”），通过联网搜索获取该地区最新的产业政策、经济动态、重点项目及招商机会，基于固定模板生成日报。日报包含以下模块：
- **今日重点**：宏观摘要，提炼当日最核心的两大招商方向及价值。
- **今日头条**：最高优先级政策/事件，详细描述机会、承接基础、招商抓手及价值点。
- **今日可对接产业机会**：按落地难易度排序，每个机会包括：机会名称、动态来源、区域承接基础、具体招商抓手、推荐价值点。
- **今日招商任务清单**：具体到上午/下午/今日内的可执行任务。
- **今日可用政策工具包**：列出相关扶持政策名称、核心条款及生效时间。

所有内容必须基于实时检索的公开信息，确保数据真实（含来源、发布时间）。不依赖内部数据库，仅使用联网搜索和LLM推理。

## 输入参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `location` | string | 是 | 关注地点，如“银川市兴庆区”、“成都市高新区” |
| `report_type` | string | 是 | 报告偏好，目前仅支持 `招商报告`，未来可扩展 |
| `date` | string | 否 | 报告日期，默认当天（YYYY-MM-DD） |

## 执行流程

### 步骤1：地点产业分析（内部隐藏变量）
- 调用 `web_search` 执行以下查询（并行）：
  1. `"{location} 主导产业 重点产业"`
  2. `"{location} 2026年 政府工作报告 产业规划"`
  3. `"{location} 经济数据 GDP 产业结构"`
- 提取出该地区的 **核心重点产业**（如现代商贸、数字经济、绿色能源、装备制造等），并存储为内部变量 `key_industries`。
- 同时检索该地区近3天内发布的 **最新政策文件**、**重大活动**（如赛事、展会）、**重点项目招标** 等动态。

### 步骤2：多维度机会挖掘
根据 `key_industries` 扩展搜索，每个产业至少执行2次搜索：
- 政策机会：`"{location} {产业名} 补贴 扶持政策 2026"`
- 市场动态：`"{产业名} 市场规模 增长 2026"`
- 区域承接基础：`"{location} {产业名} 基础 优势"`
- 具体企业/项目名单（如有）：`"{location} {产业名} 重点企业 项目"`

对搜索结果进行筛选，保留发布时间在 **过去7天内** 或虽稍早但仍在有效期（如公示期）的信息。每个机会需包含：
- **机会名称及动态**：具体政策/事件名称 + 核心内容摘要 + 来源 + 发布时间
- **兴庆区承接基础**（或对应地点）：根据搜索数据描述该地区在该领域的现有资源、区位优势、已有企业等
- **具体招商抓手**：至少3条可操作的行动点（如对接哪个部门、梳理什么清单、定向推送等）
- **推荐价值点**：量化的预期收益（如可撬动投资金额、提升招商成功率等）

### 步骤3：生成日报各模块
按照模板顺序生成文本，需满足：
- **今日重点**：从所有机会中提炼最重要的两条（通常选取政策级别最高、投资额最大或时效性最强的），用简洁语言概括方向、政策支撑及落地场景。
- **今日头条**：选取优先级最高的单一政策/事件，详细展开。必须包含：发布时间、来源、我们能承接的具体机会（分点列举）、价值点。
- **今日可对接产业机会**：按落地难易度排序（易→难），每个机会使用固定二级标题格式：“X. 赛道名称：机会简称”。内容需结构化（机会名称及动态、发布时间来源、承接基础、招商抓手、推荐价值点）。
- **今日招商任务清单**：生成具体任务，按上午、下午、今日内划分。任务需可执行（如“致电XX局获取实施细则”、“走访XX商圈摸排铺位”），并与当日机会强相关。
- **今日可用政策工具包**：列出3-5条具体政策，每条格式为《政策名称》（施行日期）+ 核心补贴条款。

### 步骤4：输出格式化报告
- 以Markdown格式输出，标题为 `# {location} 招商日报 {YYYY年MM月DD日}`。
- 内部不包含任何注释或额外说明。
- 若某项搜索无结果，注明“暂未检索到相关信息”。

## 示例调用

**用户输入**：生成银川市兴庆区今天的招商日报

**Agent执行**：
1. 并行搜索兴庆区主导产业、最新政策、马拉松赛事、技改试点等。
2. 发现“银川市入选国家级制造业技改试点”、“银川马拉松5月17日开跑”、“分布式光伏推进方案”等热点。
3. 按模板生成日报，内容类似用户提供的示例但数据实时检索。
4. 输出Markdown报告。

## 注意事项
- 所有事实必须有来源标注（至少网站名称+发布时间），禁止虚构政策或数据。
- 如未找到近7天内的动态，可放宽至30天，但需在报告中注明“该信息发布于X日前”。
- 报告中的“招商抓手”应具体到部门名称（如“工信局”、“文旅局”），可通过搜索“{location} 工信局 官网”获取标准名称。
- 该技能不依赖任何内部接口，仅使用联网搜索工具（`web_search`）。

## 与定时任务系统的集成说明
外部系统（如Cron调度器）在触发时，需调用本技能并传入 `location` 和 `report_type`。技能返回日报文本后，可由外部系统保存或发送邮件/钉钉。本技能不负责定时逻辑。
