# 先天性心脏病血流动力学分类与分流风险评估

## 适用对象
- 所有已确诊先天性心脏病的患儿

## 前提条件
- 已完成初步心脏听诊和影像学筛查

## 执行流程

### 第一步：判断是否存在心内或大血管间异常分流通道
- 若无异常通道 → 归为【无分流型】，进入第三步。
- 若有异常通道 → 进入第二步。

### 第二步：判断静息状态下分流方向

#### 左向右分流型（潜伏青紫型）
- **特征**：体循环压力 > 肺循环压力，血液从左心系统流向右心系统。
- **常见病种**：房间隔缺损（ASD）、室间隔缺损（VSD）、动脉导管未闭（PDA）。
- **临床特点**：平时无青紫，但在哭闹、屏气或肺动脉高压时可出现暂时性右向左分流（“潜伏青紫”）。
- **晚期风险**：若发展为梗阻性肺动脉高压，可形成艾森门格综合征，此时分流逆转为持续右向左，出现固定青紫，手术禁忌。

#### 右向左分流型（青紫型）
- **特征**：右心静脉血直接进入体循环，导致动脉血氧饱和度降低。
- **常见病种**：法洛四联症、大动脉转位、三尖瓣闭锁。
- **临床特点**：出生后即有或早期出现持续性中央性青紫，常伴杵状指/趾、蹲踞等代偿行为。

### 第三步：确认无分流型
- **特征**：无心内或大血管间异常交通，但存在流出道梗阻或血管狭窄。
- **常见病种**：肺动脉狭窄、主动脉瓣狭窄、主动脉缩窄。
- **临床特点**：无青紫，主要表现为心力衰竭、血压异常或杂音。

## 关键预警信号
- **房间隔缺损患者**：若出现肺动脉第二心音增强、固定分裂消失、杂音缩短，提示肺动脉高压进展。
- **动脉导管未闭患者**：若下半身青紫而右上肢正常（差异性发绀），提示肺动脉压 > 主动脉压，已发生右向左分流。
- **任何左向右分流型先心病**：一旦出现持续青紫，应高度怀疑艾森门格综合征，预后差且手术禁忌。

## 输出建议
根据分类结果，明确：
1. 青紫类型与风险（无青紫 / 潜伏青紫 / 持续青紫 / 差异性发绀）
2. 肺动脉高压阶段（动力性 vs 梗阻性）
3. 手术干预窗口期与禁忌证
4. 长期随访重点指标
