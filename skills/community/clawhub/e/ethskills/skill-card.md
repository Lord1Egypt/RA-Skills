## Description: <br>
Ethereum development knowledge for AI agents, from idea to deployed dApp, that fetches current docs on gas costs, Solidity patterns, Scaffold-ETH 2, Layer 2s, DeFi composability, security, testing, and production deployment. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[austintgriffith](https://clawhub.ai/user/austintgriffith) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and engineers use this skill when building, reviewing, testing, or deploying Ethereum and EVM dApps. It routes agents to current ETHSkills documentation for Solidity, gas costs, wallets, Layer 2s, standards, security, frontend UX, and production QA. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill fetches external Ethereum development documentation during agent work. <br>
Mitigation: Keep use scoped to Ethereum development tasks and review fetched guidance before applying it to contracts, deployments, or production systems. <br>
Risk: Generated workflows may include npm or npx scaffolding commands that execute package code. <br>
Mitigation: Run scaffolding commands only when intentionally creating a project and after confirming the package source is acceptable. <br>
