# Bank Regulatory Reporting Assistant
# 银行监管报告助手

> *AI-powered banking regulatory reporting — generate CBIRC/NRTA/SAFE reports, NPA reports, AML reports, and compliance documentation from raw data.*

> AI驱动的银行监管报告工具——1104统计报送、NPA报告、反洗钱报告、合规文档，根据原始数据一键生成。

---

## Overview

This skill generates standard regulatory reports for Chinese commercial banks. It covers 1104 statistical reports, non-performing asset (NPA) reports, anti-money laundering (AML) reports, capital adequacy reports, liquidity risk reports, and CBIRC/NRA/SAFE compliance documentation.

本工具为商业银行生成标准化监管报告，涵盖1104统计报表、不良贷款(NPA)报告、反洗钱(AML)报告、资本充足率报告、流动性风险报告及银保监/外管局合规文档。

## Installation

```bash
clawhub install bank-regulatory-reporting
```

## Core Capabilities

| Capability | Description |
|------------|-------------|
| 1104 Statistical Reports | Monthly/quarterly regulatory statistics |
| NPA Report | Non-performing asset analysis and management report |
| Capital Adequacy Report | CAR calculation, RWA analysis, capital planning |
| Liquidity Risk Report | LCR, NSFR, HQLA assessment |
| AML Report | Suspicious transaction reporting, KYC compliance |
| Green Credit Report | Green loan classification and statistics |
| SME Credit Report | 普惠金融小微企业贷款统计 |
| Real Estate Finance Report | 房地产金融监测数据 |

## Report Coverage

**Primary Regulatory Frameworks:**
- CBIRC 1104非现场监管统计制度
- NRTA 非银行金融机构监管
- 人民银行统计 (PRC央行)
- SAFE 外汇业务监管
- 银保监现场检查数据要求
