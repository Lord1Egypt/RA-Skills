# 客户引导与资料收集指南

本文档为merchant-geo skill提供标准化的客户引导流程。

---

## 触发时机

当用户表达以下意图时，立即启动引导流程：

- "帮我宣传"、"帮我推广"、"写篇文章"
- "发到抖音/小红书/知乎/百家号"
- "上传门头照片"、"发一下营业执照"
- "生成推广文案"、"帮我写个介绍"
- 任何涉及店铺宣传、企业推广的请求

---

## 标准引导话术库

### 首次接触（客户主动请求）

```
您好！请提供以下资料，我来帮您生成8个平台的宣传文章：

📷 必传资料：
1. 店铺门头照 ⭐（最重要！）
2. 产品实拍照片（3张以上）
3. 营业执照（如有）

📝 案例描述（选填）：
能说一个您服务的客户案例吗？比如：
- 客户是什么类型？（附近企业/个人/老顾客）
- 他们需要什么？（宴请用酒/送礼/自饮）
- 您怎么服务的？
- 客户反馈怎么样？

✅ 收到资料后，5分钟内生成全部文章！
```

### 催单话术

| 客户状态 | 引导话术 |
|----------|----------|
| 无回复 | "您好，差一张门头照就齐了，可以直接拍照上传" |
| 只发门头 | "收到门头照！再发3张产品图，我就可以开始写文章了" |
| 只发产品图 | "收到产品图！再来一张店铺门头照就齐全了" |
| 缺案例 | "资料齐了！能再说一个服务过的客户吗？（比如他们买了什么，感觉怎么样）" |
| 资料齐全 | "收到！我现在开始生成8个平台的宣传文章，预计5分钟完成✅" |

### 资料补全话术

| 缺少资料 | 话术 |
|----------|------|
| 无门头照 | "差一个店铺门头照就齐了，可以直接用手机拍照上传" |
| 无产品图 | "再发3张最畅销的商品图，我就能写出更具体的产品介绍" |
| 无营业执照 | "有营业执照的话可以发一下，没有也没关系，我可以从门头照提取店铺信息" |
| 无案例 | "能说一个最近服务的客户吗？比如他们买了什么、感觉怎么样，说几句就行" |

## 资料检查清单

### 商家资料清单

```markdown
## 商家资料清单

### 基础信息
- [ ] 企业/店铺名称
- [ ] 详细地址
- [ ] 联系电话
- [ ] 行业/品类
- [ ] 营业时间

### 图片资料
- [ ] 门头照片（必填）⭐
- [ ] 产品照片（至少3张）
- [ ] 营业执照（可选）
- [ ] 店内环境（可选）
- [ ] 老板/员工照片（可选）

### 案例素材
- [ ] 案例1：客户类型 + 需求 + 服务 + 反馈
- [ ] 案例2：（同上）
- [ ] 特色卖点（1-3条）
- [ ] 优惠信息（如有）
```

### 资料等级评估

| 等级 | 配置要求 | 生成内容质量 |
|------|----------|--------------|
| ⭐ 最低配 | 门头照 + 一句话描述 | 基础企业介绍（60分） |
| ⭐⭐ 标准配 | 门头照 + 产品照3张 | 企业介绍 + 产品展示（80分） |
| ⭐⭐⭐ 完整配 | 全部资料 + 案例描述 | 全方位GEO内容（100分） |

## 快速收集模板

### 一句话收集法

适用于时间紧迫的客户：

```
发一张门头照 + 一句话描述您是做什么的
我来帮您写推广文章！
```

### 案例快速收集

```
随便说一个您服务的客户例子：
格式：客户类型 → 需要什么 → 您怎么服务 → 客户怎么说
例如："附近科技园企业 → 年会用酒 → 推荐茅台五粮液组合 → 说比超市便宜很多"
```

## 特殊场景处理

### 客户不愿意提供资料

```
理解！我可以先帮您写一篇通用模板文章，
您只需要提供：
- 店铺名称
- 位置
- 主要卖什么

其他内容我用通用描述，您再修改调整。
```

### 客户只有口头描述，无照片

```
没关系！您口头告诉我：
1. 店铺叫什么名字？
2. 在深圳哪个区/街道？
3. 主要卖什么（烟酒/餐饮/服装/...）?
4. 有什么特色？（便宜/老店/正品/服务好）

我可以先写文章框架，您再补充照片发布。
```

### 客户资料质量不佳（照片模糊）

```
照片收到，但有点模糊，能重新拍一下吗？
拍摄技巧：
- 门头照：站在门口正前方拍，光线充足
- 产品照：放在桌上，从正上方45度角拍
用手机普通拍照就行，不需要专业相机。
```

## 客户成功引导示例

### 场景1：烟酒店老板

```
客户：帮我宣传一下我的烟酒店

我：
您好！请提供以下资料，我来帮您生成8个平台的宣传文章：

📷 必传资料：
1. 店铺门头照 ⭐
2. 产品实拍照片（3张以上）

📝 案例描述：
能说一个您服务的客户案例吗？
例如："附近科技园企业，年会用酒，推荐茅台五粮液组合，客户说比超市便宜"

✅ 收到后5分钟内生成全部文章！

客户：[上传门头照]

我：收到门头照！产品图和案例再来一些就齐了

客户：[上传3张白酒照片]

我：收到产品图！能简单说一个服务过的客户案例吗？

客户：附近有个科技园公司，每次公司聚餐都来我这买酒

我：收到！他们一般买什么？大概消费多少？

客户：茅台五粮液，一次买三四千块钱的

我：太棒了！收到，我现在开始生成8个平台的宣传文章！
```

## 注意事项

1. **主动引导**：客户说"帮我写"时，不要直接开始写，先引导收集资料
2. **分步收集**：不要一次要求所有资料，分步引导降低客户负担
3. **正向激励**：告诉客户"资料越全，文章越生动"
4. **灵活处理**：如果客户实在不愿提供某些资料，用通用描述代替
5. **最终确认**：生成前必须让客户确认"开始"
