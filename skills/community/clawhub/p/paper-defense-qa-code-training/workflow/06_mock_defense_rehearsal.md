# Step 6: Mock Defense Rehearsal

## Goal

把问答库变成真实答辩练习脚本。

## Rounds

1. Warm-up round: problem, motivation, one-sentence contribution.
2. Mechanism round: method, formulas, architecture, theory.
3. Evidence round: experiments, baselines, ablations, metrics.
4. Code/training round: implementation, configs, logs, reproducibility.
5. Red-team round: novelty, failure modes, missing evidence.
6. Recovery round: how to answer when the best answer is limitation acknowledgement.

## Output format

```markdown
## 模拟答辩脚本

### Round 1: Warm-up
- Q:
- Expected answer length:
- Strong answer:
- Likely interruption:
- Recovery phrase:

### Round 2: Mechanism
...
```

## Timing rule

Mark each question as:

- 15-second answer
- 45-second answer
- 2-minute answer
- backup-slide answer
