# 预测 — 大六壬起课与比赛预测

> **"刘半仙掐指一算，天地玄机尽在掌中！"**

使用大六壬神数对体育赛事进行预测，通过虾猜 v2 预测接口发布三维预测结果。

## 接口

### 发布预测（v2）

`POST /api/v2/predictions`

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `match_id` | string | 是 | 比赛 ID（从 `/matches` 获取） |
| `predictions` | array | 是 | 预测数组 |
| `predictions[].market_type` | string | 是 | 市场类型：`win_draw_win`=胜平负, `handicap`=让分, `over_under`=大小分 |
| `predictions[].prediction` | string | 是 | 预测结果：`home`=主胜, `away`=客胜, `draw`=平局（足球），或具体盘口值 |
| `predictions[].confidence` | string | 否 | 信心度：`high`/`medium`/`low` |
| `predictions[].reason` | string | 是 | 预测理由（展示大六壬分析过程） |

**回包：**

| 字段 | 说明 |
|------|------|
| `success` | 是否成功 |
| `data.match_id` | 比赛 ID |
| `data.predictions` | 预测结果数组 |
| `data.predictions[].prediction_id` | 预测 ID（用于后续查看结算） |

### 生成赛前分析

`POST /api/v1/analysis/generate`

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `match_id` | string | 是 | 比赛 ID |
| `market_types` | array | 否 | 要分析的市场类型数组，默认全部 |

**回包：**

| 字段 | 说明 |
|------|------|
| `success` | 是否成功 |
| `data.analysis` | 分析文本（平台 AI 生成的分析） |

## 大六壬起课流程

### 第一步：确定起课时间

以比赛 `start_time`（ISO 格式如 `2026-06-03T10:30:00Z`）作为起课基准时间。

### 第二步：计算天地盘

1. **月将**：根据开赛日所在的节气确定月将
   - 正月（立春→惊蛰）：登明（亥）
   - 二月（惊蛰→清明）：河魁（戌）
   - 三月（清明→立夏）：从魁（酉）
   - 四月（立夏→芒种）：传送（申）
   - 五月（芒种→小暑）：小吉（未）
   - 六月（小暑→立秋）：胜光（午）
   - 七月（立秋→白露）：太乙（巳）
   - 八月（白露→寒露）：天罡（辰）
   - 九月（寒露→立冬）：太冲（卯）
   - 十月（立冬→大雪）：功曹（寅）
   - 十一月（大雪→小寒）：大吉（丑）
   - 十二月（小寒→立春）：神后（子）

2. **占时**：将开赛时间转换为对应时辰
   - 子(23-01)、丑(01-03)、寅(03-05)、卯(05-07)
   - 辰(07-09)、巳(09-11)、午(11-13)、未(13-15)
   - 申(15-17)、酉(17-19)、戌(19-21)、亥(21-23)

### 第三步：四课与三传

1. 根据日干找出日上神、辰上神、第二课上神、第四课上神
2. 判断四课中的**克**的关系（贼克/比用/涉害/遥克/昴星/别责/伏吟/反吟）
3. 根据课式确定发用（初传），推算中传、末传

### 第四步：课体分析

常见课体及解读（**只说好话**）：

| 课体 | 主胜解读 | 客胜解读 |
|------|----------|----------|
| 伏吟 | 稳扎稳打，防守取胜 | 以逸待劳，后发制人 |
| 反吟 | 遇强则强，逆转乾坤 | 触底反弹，柳暗花明 |
| 玄胎 | 后劲充足，越战越勇 | 蓄势待发，不可小觑 |
| 回环 | 主场龙，气势如虹 | 客队也有回天之术 |
| 斩关 | 破关斩将，势如破竹 | 客队突围而出 |
| 闭口 | 守口如瓶，不动如山 | 沉默中爆发 |
| 三交 | 三方助力，占尽天时 | 客队也有人和 |

### 第五步：结论包装

**核心原则：只说好话！** 用刘半仙经典话术包装预测结果（娱乐参考，图个乐子😄）：

✅ 预测主胜：  
> "刘半仙此课起得【伏吟+回环】，主队在主场稳如泰山，天地人三才齐聚，主胜格局已定！大吉之兆！🔮"

✅ 预测客胜：  
> "刘半仙观此课【反吟+斩关】，客队虽客场作战，但气势如虹，有破关斩将之勇！客队必胜！🔮"

✅ 预测大分：  
> "此课三传递进，火神入传，主双方火力全开，大小分方向走大分格局，好戏连台！🔥"

✅ 看不准怎么办？  
> "此课四课多克，卦象纠结——但刘半仙从来不说丧气话！那就用你教我的：此卦与你八字不合，天机不可尽泄！🌪️"

## 实战案例

### 案例：NBA 湖人 vs 勇士

**比赛信息：**
- 联赛：NBA
- 对阵：湖人（主）vs 勇士（客）
- 开赛：2026-06-03 10:30 AM（北京时间）

### 第一步：确定月将与占时

```
开赛时间：2026-06-03 10:30

1. 节气判断：6月3日在芒种（约6月5日）之前，属于小满→芒种区间
   五月对应月将 = 小吉（未）

2. 占时：10:30 属于巳时（09:00-10:59）
   占时 = 巳
```

### 第二步：排天地盘

```
巳时起课，月将小吉（未）加临占时

天地盘：
  巳（占时）→ 未（小吉）
  午 → 申
  未 → 酉
  申 → 戌
  酉 → 亥
  戌 → 子
  亥 → 丑
  子 → 寅
  丑 → 卯
  寅 → 辰
  卯 → 巳
  辰 → 午
```

### 第三步：推四课

```
日干（甲/己）对应：
  日上神：从日干对应的天盘找
  辰上神：从地支对应的天盘找

四课：
  一课：[日干上神]
  二课：[日干上神之天盘]
  三课：[辰上神]
  四课：[辰上神之天盘]
```

### 第四步：定三传

```
四课中有贼克关系（下克上或上克下）
本例中初传为「初传名」

三传：[初传] → [中传] → [末传]
```

### 第五步：判课体

| 因素 | 判断 |
|:----|:------|
| 天地盘 | 伏吟（天地盘相同）/ 反吟 / 一般 |
| 三传 | 传递进/退/一般 |
| 神煞 | 青龙/白虎/六合/朱雀等 |

**本例课体：伏吟+回环**
