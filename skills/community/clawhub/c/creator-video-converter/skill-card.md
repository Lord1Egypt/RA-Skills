## Description: <br>
Convert videos into more compatible formats, containers, and codec profiles for publishing, sharing, ecommerce workflows, and platform uploads. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External creators, publishing teams, ecommerce operators, and developers use this skill to plan practical video conversions when a source file is unsupported, too large, hard to preview, or unsuitable for upload. It recommends a target format, codec/container decision, compatibility rationale, quality or size tradeoffs, and an alternate option when useful. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: A later tool or command may convert the wrong file, overwrite an output, or use unsuitable conversion settings if the plan is applied without review. <br>
Mitigation: Review the proposed command, source file, output path, overwrite behavior, target format, and codec settings before executing any conversion. <br>
Risk: Format conversion can reduce quality or increase file size when the selected codec, container, or bitrate is not appropriate for the target platform. <br>
Mitigation: Confirm the target platform, compatibility priority, quality tolerance, and size constraints before accepting the recommended conversion plan. <br>
