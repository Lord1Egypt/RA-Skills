# Platform Title Formats Reference

## TikTok Shop

### Display Rules
- **Visible in feed:** First 34 characters
- **Full search index:** Up to 255 characters
- **Lead with:** Hook phrase or product benefit (NOT brand name)
- **Include:** Product name + size/variant/flavor if fits within 34 chars
- **Avoid:** Emojis in title (inconsistent rendering on different devices)
- **Tone:** Conversational; can be slightly informal

### TikTok Title Template
```
[Hook/benefit phrase] [Product type] [Key spec] [Size/variant]
```
Examples:
- "Glow Boost Vit C Serum 30ml" (27 chars) ✅
- "Cloud Sleep Silk Eye Mask" (24 chars) ✅  
- "Silent HEPA Room Purifier 500sqft" (33 chars) ✅

### TikTok Full Title (255 chars — search optimization)
```
[Hook phrase] [Full product name] [Key feature 1] [Key feature 2] [Size] [Skin type / compatibility] [Use occasion]
```

## Amazon

### Title Rules
- Max: 200 characters (category-specific; some allow 80 only)
- Format: Title Case
- Structure: Brand + Primary KW + Feature + Differentiator + Size/Pack
- NO: All-caps words (except abbreviations), promotional language, symbols

### Amazon Title Template
```
[Brand Name] [Primary Keyword] [Key Feature 1] – [Key Feature 2], [Size/Pack/Color]
```
Examples:
- "PureGlow Vitamin C Serum 20% with Hyaluronic Acid – Brightening Face Serum for Dark Spots, 1 fl oz"
- "AirSilent H13 HEPA Air Purifier for Large Rooms 500 Sq Ft – Ultra Quiet Sleep Mode, Auto Fan Speed"

## Shopify (DTC / Brand Website)

### SEO Title Rules
- Target: 50–70 characters for full Google SERP display
- Structure: Primary KW + Product Type + Brand OR Brand + Primary KW + Product Type
- Include a differentiator if character budget allows

### Shopify Title Examples
- "Vitamin C Brightening Serum 20% | PureGlow" (43 chars) ✅
- "HEPA Air Purifier for Large Rooms | AirSilent" (45 chars) ✅

---

## Shopee (Southeast Asia)

### Title Rules
- Max: 120 characters
- Include: Product name + Brand + Key attribute + Variant
- Local language preferred; bilingual (EN + local language) accepted
- Shopee search weighted toward exact keyword match — prioritize search fit over hook

### Shopee Title Template
```
[Product type] [Key feature] [Brand] [Variant] [Size/pack]
```

## Cross-Platform Naming Adaptations

When one product is sold on multiple platforms, maintain a "name family":

| Element | TikTok | Amazon | Shopify |
|---------|--------|--------|---------|
| Hook phrase | "Glow Boost" (leads) | Featured in bullet, not title | Featured in H1 / tagline |
| Brand | Optional (after hook) | Required (first in title) | In title or meta |
| Primary keyword | Required | Required (position 1-–3) | Required (position 1–2) |
| Variant/size | In 34-char visible | At end of title | In product variant selector |
