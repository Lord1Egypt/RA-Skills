# 肝素抗凝治疗的实验室监测

## 适用对象
- 正在接受普通肝素（uFH）或低分子量肝素（LMWH）治疗的患者

## 触发条件
当患者满足以下任一条件时启用本技能：
- 已开始肝素类抗凝治疗
- 存在血栓风险或已确诊血栓性疾病

## 监测流程

### 1. 普通肝素（uFH）监测
- **首选指标**：活化部分凝血活酶时间（APTT）
  - 目标范围：APTT为正常对照的1.5～2.0倍（中国人群推荐范围）
- **替代方案**：测定uFH血浆浓度
  - 目标范围：0.2～0.4 IU/ml
- **特殊场景**（如体外循环、血液透析）：使用活化凝血时间（ACT）
  - 目标范围：250～360秒（参考区间60～120秒）

### 2. 低分子量肝素（LMWH）监测
- **常规剂量**：通常无需监测
- **大剂量使用时**：采用抗因子Xa活性测定
  - 预防性用药目标：0.2～0.4 AFXa IU/ml
  - 治疗性用药目标：0.5～0.7 AFXa IU/ml

### 3. 血小板计数监测（适用于uFH和LMWH）
- 定期监测血小板计数
- 若血小板计数 < 50×10⁹/L，立即暂停肝素并排查肝素诱导的血小板减少症（HIT）等病因

### 4. 抗凝血酶（AT）活性监测
- 确保AT活性在80%～120%
- AT活性 < 70%：肝素效果减弱
- AT活性 < 50%：肝素效果明显减低
- AT活性 < 30%：肝素基本失效，应暂停用药并评估原因

## 后续行动
根据监测结果调整肝素剂量：
- 若指标超出目标范围，上调或下调剂量
- 若血小板 < 50×10⁹/L 或 AT活性 < 30%，暂停用药并评估原因
