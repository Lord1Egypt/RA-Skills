# 常染色质与异染色质的区分标准

## 何时使用
- 分析间期真核细胞核内的染色质结构
- 需要判断某区域是否具有转录活性
- 研究染色质凝缩状态与基因表达关系

## 执行步骤
1. **检查螺旋化程度**：
   - 松散 → 常染色质
   - 凝集 → 异染色质
2. **观察染色深浅**：
   - 浅而均匀 → 常染色质
   - 深染 → 异染色质
3. **定位空间分布**：
   - 核中央 → 常染色质
   - 核膜内缘 → 异染色质
4. **评估功能活性**：
   - 有转录活性、含单一序列 → 常染色质
   - 无/低转录活性、含重复序列、晚复制 → 异染色质
5. **进一步分类异染色质**：
   - **结构异染色质**：所有细胞中恒定凝缩（如着丝粒、端粒）
   - **功能异染色质**：特定细胞或发育阶段由常染色质转化而来，可逆
6. **考虑调控机制**：DNA甲基化是两者相互转化的关键表观遗传开关。

> ⚠️ 注意：本技能不适用于有丝分裂期染色体或原核生物。
