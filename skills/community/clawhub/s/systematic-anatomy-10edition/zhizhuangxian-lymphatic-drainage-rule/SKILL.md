# 齿状线淋巴引流分界规则

## 何时使用
- 患者确诊为直肠癌或肛管癌
- 需评估淋巴结转移风险
- 制定根治性手术的淋巴结清扫范围
- 规划放射治疗的靶区

## 如何执行
1. **确认齿状线解剖位置**：齿状线是肛管与直肠的交界处，也是胚胎内、外胚层的分界。
2. **判断肿瘤位置**：
   - 若肿瘤位于**齿状线以上**（直肠段）：
     - 淋巴引流主要沿以下路径：
       a) 直肠上血管 → 直肠上淋巴结
       b) 直肠下血管 → 髂内淋巴结
       c) 肛血管和阴部内血管 → 髂内淋巴结
       d) 少数经骶外侧血管 → 骶淋巴结
   - 若肿瘤位于**齿状线以下**（肛管段）：
     - 淋巴管直接注入**腹股沟浅淋巴结**
3. **临床决策**：
   - 齿状线上肿瘤：重点关注盆腔淋巴结（如髂内、骶淋巴结）
   - 齿状线下肿瘤：必须评估腹股沟浅淋巴结是否受累

> 注意：本规则仅适用于成人正常解剖结构，不适用于先天性肛门直肠畸形患者。
