## Description: <br>
Find the reference marker for a knowledge item. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wxt-ai](https://clawhub.ai/user/wxt-ai) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and operations teams use this skill to inspect a provided knowledge-base note, onboarding checklist, or document excerpt and return the relevant reference marker. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill is from a third-party publisher with limited history. <br>
Mitigation: Use it only for the stated narrow purpose of extracting known markers from text the user provides. <br>
Risk: The fixed validation marker could be mistaken for a general knowledge source. <br>
Mitigation: Treat returned markers as candidates from the supplied text and verify them against the current source note when accuracy matters. <br>
