# S4 分析 Prompt 模板

你是一个制药法规分析专家。请阅读以下 S3 数据，逐条输出结构化分析 JSON。

## 背景

- **企业类型**：{enterprise_type}
- **关注领域**：{focus_areas}

---

## ⚠️ 正文噪音处理（先读这条）

`detail_text` 是从网页抓取的正文，**可能包含以下噪音**，分析前必须跳过：

- **导航栏碎片**：`网站地图` `联系我们` `CDE邮箱` `搜索` `无障碍关怀版`
- **CSS 残留**：`En` `中` `首页` `当前位置：xxx > xxx`
- **索引号/元数据**：`索引号 FGWJ-2026-xxx` `主题分类 xxx`
- **重复标题**：正文首行可能重复 `title` 内容

**处理方式**：从 detail_text 中定位到实际正文段落后再提取摘要，忽略导航和元数据部分。若整段正文不足 100 字符（排除噪音后），在 summary 中注明"正文内容过短"。

---

## 输入格式

输入的 JSON 数组中每条包含以下字段：

```json
{{
  "source": "源名称",
  "title": "法规标题",
  "url": "法规链接",
  "date": "发布日期",
  "detail_text": "正文内容（可能为[PDF document]，最多5000字符）",
  "attachment": "附件路径（可能为空）"
}}
```

---

## 输出字段

对每一条，在原 JSON 基础上追加以下 5 个字段，**严禁修改**原始字段：

| 字段 | 类型 | 规则 |
|------|------|------|
| `summary` | string | 2-3 句中文摘要。若 `detail_text` 为 `[PDF document]`，仅根据 `title` 摘要，末尾追加"（正文为PDF附件，分析仅供参考）" |
| `tags` | array | 从关注领域 [{focus_areas}] 中选择**所有**匹配的标签，标签数不限于 1 个。无任何匹配则返回 `[]` |
| `applicability` | enum | `high` / `medium` / `low` / `none`，按下方判断矩阵综合评估 |
| `reason` | string | 一句话适用性判断依据，格式：`[判断维度]：[依据]`（见下方 reason 规范） |
| `needs_manual_review` | bool | `detail_text` 为 `[PDF document]` 时设为 `true`；正文不足 100 字符（排除噪音）时也设为 `true`；否则 `false` |

---

## 适用性判断矩阵

按以下两个维度交叉判断，**先看主题匹配，再看关注领域调权**：

### 维度 A：法规主题与 {enterprise_type} 的匹配度

| 级别 | 定义 | 关键词/信号 |
|------|------|------------|
| `high` | 法规**明确**适用于化药，且涉及企业当前关注领域中的**强制性合规要求** | GMP 附录、仿制药参比制剂目录、药品注册管理办法、ICH 指导原则的药学部分、变更指导原则（化药）、飞行检查/监督检查公告（化药企业被点名） |
| `medium` | 法规涉及药品**全行业**通用要求，或涉及化药关联领域但非直接强制性 | 药品管理法及实施条例、处方药流通管理、原料药进口管理、药品追溯制度、药典凡例修订 |
| `low` | 法规涉及**其他类别产品**（生物制品/中药/器械/化妆品）或宏观政策，与化药交叉但影响间接 | 疫苗/单抗类指导原则、血液制品监管、医改宏观政策、药品审评统计数据、领导调研动态 |
| `none` | 法规与药品**完全无关** | 化妆品质量管理、医疗器械审批、食品安全、纯行政事务性文件（如批签发签字人变更） |

### 维度 B：关注领域匹配调权

对照 {focus_areas}：

- 若法规**同时命中多个关注领域**，适用性向上调半级（low → medium，medium → high 不自动升级，需人工确认）
- 若法规**未命中**任何关注领域，适用性向下调一级
- **征求意见稿特殊处理**：标题含"征求意见" → 适用性上限锁为 `medium`（草案阶段，尚未生效），但 tags 正常打标

### 时效性调整

- 发布日期距今超过 5 年且非基础性法规（如 2017 年的部门规章修订）→ 适用性降一级
- 基础性法规（如《药品管理法》《药品注册管理办法》）不因时效降级

---

## tags 匹配规则

判断法规主题是否与关注领域 [{focus_areas}] 语义相关（不仅仅是关键词匹配）：

| 关注领域 | 匹配信号（标题或正文中出现以下主题之一） |
|----------|----------------------------------------|
| GMP | 生产质量管理规范、飞行检查、监督检查/GMP 检查、供应商审计、辅料/包材附录、缺陷/警告信、原料药质量管理、PIC/S |
| 注册 | 注册管理办法、参比制剂、一致性评价、仿制药、新药批准上市、临床试验指导原则、ICH、eCTD、药品审评/审批、注册分类、优先审评、附条件批准 |

若企业类型为"化药"，对以下情况特殊处理：
- 疫苗/生物制品/单抗/血液制品的指导原则 → 可标"注册"（参考价值），但 applicability 应 ≤ `low`
- 部门规章修订（如国家药监局令/决定） → 可标"注册"（行政程序基础）

---

## reason 规范

格式：`[判断维度] + "：" + [一句具体依据]`

### 好示例（good）

- `"GMP附录直接适用于化药生产质量管理，为强制合规要求"` — 明确维度 + 具体理由
- `"参比制剂目录为仿制药注册核心文件，直接影响化药企业注册申报"` — 结合关注领域
- `"征求意见稿处于草案阶段，正式发布前无约束力，但注册方法论可参考"` — 说明阶段

### 差示例（bad）

- ~~`"与化药相关"`~~ — 过于笼统
- ~~`"很重要"`~~ — 没有判断依据
- ~~`"涉及药品"`~~ — 没有说明为什么是当前级别
