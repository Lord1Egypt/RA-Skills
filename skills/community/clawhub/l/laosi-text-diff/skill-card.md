## Description: <br>
Compares two text blocks or files line by line, producing unified diffs, side-by-side views, highlighted changes, and change statistics. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[534422530](https://clawhub.ai/user/534422530) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, reviewers, and technical writers use this skill to compare code, configuration, documentation, or program output and inspect line-level additions, removals, and changes. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generic activation terms such as diff or compare may cause the skill to run when a user did not intend to expose compared content. <br>
Mitigation: Use the skill only for text or files the user explicitly wants to compare, and review diff output before sharing it outside the conversation. <br>
