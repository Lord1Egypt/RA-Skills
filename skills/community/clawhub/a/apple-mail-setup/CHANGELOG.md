# Changelog

All notable changes to the Apple Mail Channel Setup Skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-06-22

### Added
- Initial skill release
- Comprehensive setup and configuration documentation
- Installation guide for ClawHub and npm
- Configuration examples for single and multi-account setups
- Troubleshooting section for common issues
- Advanced configuration patterns
- API reference documentation
- Quick start templates
- Security and permissions setup guide

### Documentation
- `SKILL.md`: Complete skill documentation
- `README.md`: Skill overview and quick start
- `openclaw.skill.json`: Skill metadata and configuration schema
- Configuration examples for:
  - Basic single account
  - Multi-account (work + personal)
  - Customer support with auto-archive
  - Custom polling intervals
  - Strict sender filtering

### Features
- Step-by-step installation instructions
- Multiple ready-to-use configuration templates
- Comprehensive troubleshooting guide
- Best practices for production use
- Security considerations and allowlist setup
- HTML processing and sanitization explanation
- Session isolation documentation

### Requirements
- macOS platform
- OpenClaw >= 2026.1.0 or Hermes
- `@jehadurre/openclaw-apple-mail` plugin >= 1.0.0
- Node.js >= 18.0.0
