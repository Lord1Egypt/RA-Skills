## Description: <br>
Manage Alibaba Cloud Cloud Firewall (Cloudfw) via OpenAPI/SDK for firewall policy and resource operations, change management, status checks, and troubleshooting Cloud Firewall API workflows. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, cloud security engineers, and operators use this skill to discover Alibaba Cloud Cloud Firewall APIs, plan or perform firewall resource and policy workflows, and verify results with describe or list operations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can guide real Alibaba Cloud Cloud Firewall changes when used with live cloud credentials. <br>
Mitigation: Use a dedicated least-privilege AccessKey and confirm the account, region, target resources, and intended change before any mutation. <br>
Risk: Saved outputs may include sensitive cloud resource details or API response data. <br>
Mitigation: Review generated artifacts before sharing and avoid storing secrets in output files. <br>
