# 钻石 4C 标准与选购指南

## 一、4C 标准详解

### 1. Carat（克拉重量）

| 克拉 | 直径(约) | 价格区间(RMB) | 适用场景 |
|------|---------|--------------|----------|
| 0.30ct | 4.3mm | 3,000-8,000 | 日常佩戴、时尚款 |
| 0.50ct | 5.1mm | 8,000-25,000 | 经典婚戒入门 |
| 0.70ct | 5.7mm | 18,000-50,000 | 进阶婚戒 |
| 1.00ct | 6.5mm | 40,000-150,000 | 经典婚戒、纪念 |
| 1.50ct | 7.4mm | 80,000-300,000 | 高端婚戒 |
| 2.00ct | 8.2mm | 150,000-600,000 | 收藏级 |

**溢价阶梯**：0.30 → 0.50 → 0.70 → 0.90 → 1.00 → 1.50 → 2.00ct 存在明显的克拉溢价，跨越整数关口价格跳涨 15-30%。

### 2. Color（颜色）

按 GIA 分级标准（D-Z）：

| 等级 | 描述 | 肉眼效果 | 推荐度 |
|------|------|---------|--------|
| D | 完全无色（极稀有） | 冰透无色 | ⭐⭐⭐（溢价高） |
| E | 无色 | 冰透无色 | ⭐⭐⭐ |
| F | 无色 | 冰透无色 | ⭐⭐⭐⭐（性价比之选） |
| G | 近无色 | 正面无色 | ⭐⭐⭐⭐⭐（最佳性价比） |
| H | 近无色 | 正面无色、侧面微黄 | ⭐⭐⭐⭐ |
| I | 近无色 | 轻微暖色调 | ⭐⭐⭐（预算有限选） |
| J | 微黄 | 肉眼可见暖色调 | ⭐⭐ |
| K-Z | 明显黄色 | 肉眼明显 | ⭐（不推荐） |

**选购建议**：
- 日常佩戴：G-H 色性价比最高
- 婚戒：F-H 色搭配白色金属戒托
- K 金戒托：可放宽到 I-J 色（金色遮掩黄色调）
- 克拉钻：建议 G 色及以上

### 3. Clarity（净度）

| 等级 | 描述 | 内含物可见度 | 推荐度 |
|------|------|------------|--------|
| FL/IF | 内外无瑕 | 10x 镜下无瑕 | ⭐（溢价过高） |
| VVS1-VVS2 | 极微瑕 | 10x 镜下极难见 | ⭐⭐⭐ |
| VS1-VS2 | 微瑕 | 10x 镜下可见 | ⭐⭐⭐⭐⭐（最佳性价比） |
| SI1 | 小瑕 | 10x 镜下明显 | ⭐⭐⭐⭐ |
| SI2 | 小瑕 | 肉眼隐约可见 | ⭐⭐⭐（预算有限） |
| I1-I3 | 有瑕 | 肉眼可见 | ⭐（不推荐） |

**选购建议**：
- 日常佩戴：VS2-SI1 完全够用，肉眼干净
- 婚戒：VS1-VS2 是甜蜜点
- 克拉以上：建议 VS2 及以上

### 4. Cut（切工）

切工是钻石火彩的关键，比颜色和净度更重要。

| 等级 | 描述 | 火彩表现 | 推荐度 |
|------|------|---------|--------|
| Excellent (3EX) | 完美切工 | 璀璨闪耀 | ⭐⭐⭐⭐⭐ |
| Very Good | 很好 | 明亮 | ⭐⭐⭐⭐ |
| Good | 好 | 尚可 | ⭐⭐⭐ |
| Fair/Poor | 一般/差 | 暗淡 | ⭐（不推荐） |

**3EX 标准**：切工(Cut Grade) + 对称性(Symmetry) + 抛光(Polish) 三项均为 Excellent。

**选购铁律**：切工绝不可妥协，至少 Very Good，优先 3EX。

## 二、荧光 (Fluorescence)

| 等级 | 效果 | 影响 |
|------|------|------|
| None | 无荧光 | D-F 色钻石首选 |
| Faint | 微弱 | 基本无影响 |
| Medium | 中等 | G 以下可接受 |
| Strong | 强 | 可能使钻石发蒙 |
| Very Strong | 极强 | 不推荐（除特殊审美） |

**规则**：高色级（D-F）避免中强荧光；G 以下可有微弱荧光，反而可中和黄色调。

## 三、预算分配公式

```
总预算 = 钻石 + 戒托 + 品牌溢价
钻石预算占比：
- 1万以下：70% 钻石 + 30% 戒托
- 1-3万：75% 钻石 + 25% 戒托
- 3-5万：80% 钻石 + 20% 戒托
- 5万以上：85% 钻石 + 15% 戒托
```

**经典配置推荐**：

| 预算(RMB) | 推荐配置 | 说明 |
|-----------|---------|------|
| 5,000-8,000 | 0.30ct H VS2 3EX | 入门款，日常佩戴佳 |
| 8,000-15,000 | 0.50ct H VS2 3EX | 经典婚戒入门 |
| 15,000-25,000 | 0.50ct F VS1 3EX | 高品质半克拉 |
| 25,000-40,000 | 0.70ct G VS2 3EX | 70分完美之选 |
| 40,000-70,000 | 1.00ct H VS2 3EX | 一克拉性价比方案 |
| 70,000-120,000 | 1.00ct F VS1 3EX | 一克拉高品质 |
| 120,000-200,000 | 1.50ct G VS1 3EX | 一克拉半进阶 |

## 四、GIA 证书关键参数解读

| 字段 | 含义 | 要点 |
|------|------|------|
| Report Number | 证书编号 | 官网可查真伪 |
| Shape and Cutting Style | 形状和切工类型 | Round Brilliant 最经典 |
| Measurements | 尺寸(mm) | 检查直径是否达标 |
| Carat Weight | 克拉重量 | 核心参数 |
| Color Grade | 颜色等级 | D-Z |
| Clarity Grade | 净度等级 | FL-I3 |
| Cut Grade | 切工等级 | Excellent-Poor |
| Polish | 抛光 | Excellent-Poor |
| Symmetry | 对称性 | Excellent-Poor |
| Fluorescence | 荧光 | None-Very Strong |
| Clarity Characteristics | 内含物特征 | 关键看类型和位置 |
| Inscription(s) | 镭射编码 | 腰围刻字（GIA编号） |

## 五、常见陷阱与避坑

1. **"奶油钻"**：颜色好但雾蒙蒙，GIA证书不会标注，必须实物检查
2. **"咖钻"**：带棕色调，影响火彩
3. **克拉溢价陷阱**：0.99ct 比 1.00ct 便宜 20-30%，肉眼看不出差异
4. **切工克扣**：有些证书只标注"Very Good"但火彩差，必须看实物/视频
5. **套证**：GIA编码被复制到另一颗钻石上，注意核对镭射编码和内含物图
6. **散货充证**：无证书钻石自称"散货性价比"，风险极高
