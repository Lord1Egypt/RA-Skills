# 人身伤害类案件

案号：(2024)黑0726刑初11号
概括：王某民在制止王某选的不法侵害后，在对方已丧失加害能力的情况下再次击打对方头部致其轻伤，其二次加害行为不构成正当防卫，被认定为故意伤害罪。

案号：(2024)内0922民初27号
概括：高某某因池某某的行为遭受人身伤害，法院判决池某某赔偿相关费用共计39699.54元。

案号：(2024)新0103民初768号
概括：翟某因人身损害向范某、信某提起诉讼，法院判决范某、信某连带赔偿翟某医疗费、误工费等各项损失。

案号：(2022)辽02刑终476号
概括：刘某故意伤害案中，法院通过综合审查证据，采信了被害人车某峰左眶下壁骨折构成轻伤二级的鉴定意见，对刘某定罪量刑。对其右眶内壁骨折因证据存疑未予认定。

案号：(2023)湘刑终215号
概括：肖某因被张某凡掌掴而持匕首捅刺对方胸部致其死亡，法院认为掌掴行为不具有明显紧迫性，肖某的行为不构成防卫，应以故意伤害罪定罪处罚。肖某有自首、赔偿并获谅解等情节。

案号：(2023)粤07刑终76号
概括：赵某健因邻里纠纷持棍殴打陈某珍致其轻伤，构成故意伤害罪。法院认为被害人无过错，赵某健有前科且社会危险性较大，不适用缓刑，但考虑其自首、赔偿等情节予以从轻处罚。
