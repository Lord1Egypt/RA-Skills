## Use Case: <br>
Developers, operators, and prompt authors use this skill as a reference for designing, testing, optimizing, and managing prompts and prompt libraries for LLM and agent workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Prompt examples may be copied into agents that send messages, create tickets, access private data, or perform real-world actions without enough review. <br>
Mitigation: Review and adapt each template before using it in action-capable systems, especially where private data or external side effects are involved. <br>
Risk: Generic prompt templates can produce incorrect or misleading outputs if deployed without task-specific testing. <br>
Mitigation: Use the skill's evaluation loop, scoring rubrics, test cases, and review checklist before relying on prompts in production workflows. <br>
