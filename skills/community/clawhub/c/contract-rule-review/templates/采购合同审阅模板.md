# 采购合同审阅模板

## 一、合同双方主体
- [ ] 采购方（甲方）信息：名称、地址、联系人
- [ ] 供应商（乙方）信息：名称、地址、联系人
- [ ] 营业执照、经营许可是否有效
- [ ] 代理人授权是否合法有效

## 二、采购标的
- [ ] 物资名称、规格、型号是否清晰
- [ ] 数量是否符合采购需求
- [ ] 质量标准是否符合要求（国标/行标/企标）
- [ ] 技术参数/配置清单是否完整

## 三、价格与结算
- [ ] 单价是否明确（含税/不含税）
- [ ] 总价计算是否正确
- [ ] 付款方式（预付/货到/账期）是否合理
- [ ] 付款条件与节点是否清晰
- [ ] 发票类型（增值专用/普通）是否约定

## 四、交货条款
- [ ] 交货时间是否明确
- [ ] 交货地点/方式（运输/自提）是否清晰
- [ ] 运输费用及风险划分是否明确
- [ ] 交货验收标准是否可操作

## 五、质量保证
- [ ] 质量保证期限是否明确
- [ ] 质量标准（国家标准/合同约定）是否清晰
- [ ] 质量异议处理流程是否完善
- [ ] 退换货条款是否公平合理

## 六、违约责任
- [ ] 延期交货违约金是否过高（建议≤千分之一/日）
- [ ] 质量问题责任划分是否清晰
- [ ] 采购方逾期付款责任是否对等

## 七、争议解决
- [ ] 管辖法院/仲裁条款是否明确
- [ ] 法律适用是否约定

## 八、风险条款
- [ ] 不可抗力定义是否合理
- [ ] 保密条款是否完善
- [ ] 知识产权归属是否明确
