## Known Risks and Mitigations: <br>
Risk: The skill can control browsers, desktop applications, shell commands, cookies, screenshots, files, and process termination. <br>
Mitigation: Install and run it only in an isolated test environment or dedicated browser profile with explicit user approval for broad automation actions. <br>
Risk: Session persistence, screenshots, and file operations can expose sensitive accounts, secrets, or private screen contents. <br>
Mitigation: Avoid real accounts, secrets, sensitive screens, and broad file paths unless the user intentionally scopes the task to those resources. <br>
