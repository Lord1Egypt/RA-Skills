# 支气管舒张试验阳性判断标准

## 适用对象
- 疑似哮喘患者
- 需与慢性阻塞性肺疾病（COPD）鉴别的患者
- 需评估气道可逆性的临床场景

## 使用前提
确保满足以下条件后再执行判断：
- 患者已停用支气管舒张药物至少24小时
- 基础肺功能检查显示FEV₁或FEV₁/FVC降低
- 已完成用药前后的肺功能测定

## 执行步骤
1. **确认试验条件**：排除急性呼吸道感染期及无法配合肺功能检查的患者。
2. **计算通气改善率**：
   - 公式：`通气改善率 = [(用药后FEV₁ - 用药前FEV₁) / 用药前FEV₁] × 100%`
3. **计算FEV₁绝对增加值**：
   - `绝对增加值 = 用药后FEV₁ - 用药前FEV₁`
4. **阳性判定**：同时满足以下两个条件：
   - 通气改善率 ≥ 12%
   - FEV₁绝对增加值 ≥ 200 ml
5. **结果解释**：
   - 阳性结果支持哮喘诊断
   - 仅改善率达标但绝对增加值不足200 ml时，不视为阳性

> 注意：若任一前提条件未满足，不得应用此判断标准。
