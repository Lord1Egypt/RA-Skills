## 六、数据使用注意事项

### 6.1 信息交叉验证

```
单一来源 → ⚠️ 可信度存疑
两个独立来源 → ✅ 基本可信
三个以上来源 → ✅✅ 高度可信
```

### 6.2 信息时效性

| 数据类型 | 建议更新周期 | 说明 |
|---------|------------|------|
| 机构基本信息 | 每6个月 | 工商信息可能变更 |
| 投资案例 | 每季度 | 新投资持续更新 |
| 团队信息 | 每季度 | 人员流动 |
| 口碑评价 | 实时 | 动态变化 |
| 负面信息 | 实时 | 发现即更新 |

### 6.3 信息可信度排序

```
官方渠道（最高）
  ↓ 中基协 > 政府官网 > 交易所

第三方数据平台（中等）
  ↓ 付费平台 > 免费平台

媒体报道（参考）
  ↓ 权威媒体 > 自媒体

社交媒体（参考）
  ↓ 实名认证 > 匿名
```

### 6.4 常见数据问题

| 问题类型 | 识别方法 | 解决方案 |
|---------|---------|---------|
| **信息滞后** | 数据更新时间久远 | 多渠道交叉验证 |
| **数据缺失** | 关键信息为空 | 通过其他渠道补充 |
| **信息错误** | 与事实不符 | 以官方信息为准 |
| **刻意美化** | 过度正面描述 | 关注负面信息 |
| **恶意抹黑** | 无证据的负面 | 核实信息来源 |

## 七、高效查询建议

### 7.1 快速查询路径

**机构基础信息查询（5分钟）：**
1. 天眼查/企查查 → 工商信息
2. 中基协官网 → 备案信息
3. 官网 → 团队信息

**投资案例查询（10分钟）：**
1. IT桔子 → 投资事件
2. 天眼查 → 对外投资
3. 新闻搜索 → 关键词

**口碑调研（20分钟）：**
1. 搜索引擎 → 负面关键词
2. 裁判文书网 → 诉讼记录
3. 行业社群打听

### 7.2 关键词模板

**机构名搜索：**
```
"[机构名]" 投资
"[机构名]" 融资
"[机构名]" 案例
"[机构名]" 投诉
"[机构名]" 纠纷
```

**合伙人搜索：**
```
"[合伙人名]" 投资
"[合伙人名]" 观点
"[合伙人名]" 履历
```

**组合搜索：**
```
"[机构名]" + "[赛道]" + 投资
"[机构名]" + "[竞品名]" + 投资
"[机构名]" + 创业者 + 评价
```
