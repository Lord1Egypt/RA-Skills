# 薪酬调研方法论

## 1. 分位值定义

### 什么是分位值？

分位值是将一组数据按从小到大排序后，处于某个百分比位置的数值。在薪酬调研中，常用的分位值包括：

| 分位值 | 含义 | 使用场景 |
|--------|------|----------|
| P10 | 10% 的数据低于此值 | 市场最低水平参考 |
| P25 | 25% 的数据低于此值 | 市场较低水平，保守定位 |
| P50 | 50% 的数据低于此值 | 市场中位值，常用基准 |
| P75 | 75% 的数据低于此值 | 市场较高水平，竞争力定位 |
| P90 | 90% 的数据低于此值 | 市场最高水平参考 |

### 分位值 vs 平均值

- **平均值**：受极端值影响大，容易失真
- **分位值**：更稳健，不受极端值影响
- **推荐**：薪酬分析优先使用分位值

## 2. 统计学原理

### 样本量要求

| 样本量 | 可信度 | 建议 |
|--------|--------|------|
| < 10 | 低 | 数据不足，仅供参考 |
| 10-20 | 中 | 可以得出初步结论 |
| 20-50 | 高 | 结论可靠 |
| 50+ | 很高 | 结论非常可靠 |

### 异常值处理

1. **识别方法**：
   - IQR 方法：超过 Q3 + 1.5×IQR 或低于 Q1 - 1.5×IQR
   - 3σ 方法：超过均值 ± 3 倍标准差

2. **处理方式**：
   - 标记但不删除（保留原始数据）
   - 在计算分位值时排除
   - 记录异常原因

## 3. 数据质量评估

### 评估维度

| 维度 | 权重 | 评估标准 |
|------|------|----------|
| 完整性 | 30% | 字段填充率 |
| 时效性 | 25% | 数据收集时间 |
| 准确性 | 25% | 数据来源可信度 |
| 一致性 | 20% | 格式统一程度 |

### 质量评分

```
总分 = 完整性×30% + 时效性×25% + 准确性×25% + 一致性×20%
```

| 评分 | 等级 | 说明 |
|------|------|------|
| 90-100 | A | 优秀，可直接使用 |
| 80-89 | B | 良好，少量补充 |
| 70-79 | C | 一般，需要补充 |
| < 70 | D | 较差，重新收集 |

## 4. 加权计算方法

### 来源权重

```
加权分位值 = Σ(数据点 × 来源权重) / Σ(来源权重)
```

### 权重分配原则

1. **行业报告** (1.2)：专业机构发布，样本量大，方法论严谨
2. **财报数据** (1.1)：上市公司公开数据，可信度高
3. **招聘网站** (1.0)：实时市场数据，但可能包含水分
4. **其他来源** (0.9)：非主流渠道，需谨慎验证
5. **论坛/社区** (0.8)：个人分享，主观性强

## 5. 薪酬区间转换

### 月薪转年薪

```
年薪 = 月薪 × 12 + 年终奖（如有）
年薪 = 月薪 × 月薪数（如 15 薪 = 月薪 × 15）
```

### 区间处理

```
薪酬中点 = (下限 + 上限) / 2
薪酬区间宽度 = (上限 - 下限) / 中点 × 100%
```

## 8. 注意事项

1. **数据时效性**：优先使用近 6 个月内的数据
2. **样本代表性**：确保样本覆盖目标岗位的主要特征
3. **口径一致性**：确保所有数据使用相同的计算口径
4. **地域差异**：不同城市薪酬差异大，需单独分析
5. **行业对标**：优先选择同行业数据
6. **职级匹配**：确保职级定义一致
