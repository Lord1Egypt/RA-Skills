## 政策描述要素检查清单

每条政策描述必须包含以下全部要素：

1. **年份**
   - 格式：20XX年
   - 例如：2025年、2024年

2. **招生人数**
   - 格式：招收X人、招生X人、名额X人、无名额限制
   - 示例：招收3人、招收5人

3. **报名条件**
   - 必须包含奖项级别：市级一等奖、省级二等奖、国家级三等奖等
   - 示例：Python编程赛获市级一等奖以上

4. **招生对象**
   - 明确招收范围
   - 示例：全区应届初中毕业生、本校科技特长生

5. **录取方式**
   - 明确录取流程
   - 示例：学校测试后按中考成绩择优录取、综合素质评定

## 地区文件格式

`地区.txt` 文件格式：
- 每行一个地区名称
- 支持空行（跳过）
- 支持 # 开头的注释行
- 支持带括号的说明文字（只取括号前内容）
- 示例：
  ```
  北京市海淀区
  上海市浦东新区
  # 广东省
  广州市（→深圳罗湖区）
  ```

## CSV 输出格式

使用 UTF-8-sig 编码（Excel 可直接打开无乱码）

| 列名 | 说明 | 示例 |
|-----|------|------|
| 学段 | 小升初/初升高 | 初升高 |
| 地区 | 具体地区 | 桂林秀峰区 |
| 学校名称 | 学校全称 | 桂林中学 |
| 赛事/项目名称 | 具体赛事名 | NOC大赛Python赛项 |
| 对升学的具体帮助 | 完整描述 | 2024年桂林市科技特长生... |
