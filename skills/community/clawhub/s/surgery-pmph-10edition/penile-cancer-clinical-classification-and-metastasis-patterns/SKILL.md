# 阴茎癌的临床分型与转移特征

## 适用场景
- 患者已确诊为阴茎癌
- 具备组织病理学检查结果
- 需明确亚型、生长模式及转移风险

## 执行步骤
1. **进行组织学分类**：
   - **非HPV相关亚型**：普通鳞癌、疣状癌、乳头状癌、腺鳞癌、肉瘤样癌、混合性癌
   - **HPV相关亚型**：基底样癌、湿疣样癌、淋巴上皮瘤样癌
2. **判断生长方式**：
   - **乳头型**：向外生长为主，呈菜花样，伴溃疡及脓臭分泌物
   - **结节型**：浸润性生长，质地硬，可深入海绵体，晚期才侵犯尿道
3. **评估转移特征**：
   - **淋巴转移**：主要至腹股沟及髂血管淋巴结（是首要转移途径）
   - **血行转移**：晚期可转移至肺、肝、骨、脑等远处器官
4. **指导临床决策**：
   - HPV相关亚型通常更具侵袭性
   - 结节型比乳头型更易深层浸润
   - 腹股沟淋巴结状态是预后关键因素，必要时行淋巴结清扫

> ⚠️ 注意：本技能仅适用于上皮源性阴茎癌（主要是鳞癌），不适用于肉瘤等非上皮来源恶性肿瘤。
