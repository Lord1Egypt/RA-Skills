## 🚀 一句话介绍

**遇到复杂决策不知道怎么办？** 让 36 位顶级顾问帮你分析——从毛泽东、德鲁克、Jobs 到巴菲特，每个顾问都有独特的思维框架，15 分钟给你一份完整的决策报告。

## ⚡ 快速开始

### 安装
```bash
openclaw skills install advisor-board
```

### 使用
```
启动顾问会诊，我要分析一下是否应该辞职创业
```

### 输出示例
```
🔍 全网检索情报（先搜最新信息）
→ 2026 年创业趋势报告
→ 当前经济环境分析
→ 你所在行业的最新融资情况

📊 各顾问观点
→ 德鲁克：客户是谁？价值主张是什么？
→ 段永平：这个决策本分吗？长期主义对吗？
→ 马斯克：第一性原理是什么？为什么还没开始？

✅ 综合建议 + 下一步行动清单
```
