# 非小细胞肺癌组织学类型与基因突变关联的靶向治疗决策

## 何时使用
- 患者已确诊为非小细胞肺癌（NSCLC），占所有肺癌的85%～90%
- 已获得合格的肿瘤组织样本并完成分子病理检测
- 需要根据组织学类型和基因突变结果选择靶向药物

## 执行步骤

1. **确认组织学亚型**：明确是否为肺腺癌、鳞状细胞癌或其他NSCLC亚型。
2. **按亚型优先检测关键基因**：
   - **肺腺癌**：重点检测 EGFR、KRAS、EML4::ALK、ROS1
   - **鳞状细胞癌**：可检测 EGFR 和 BCL2 表达，但靶向治疗机会较少
3. **解读关键基因突变及其治疗意义**：
   - **EGFR 突变阳性**（外显子19缺失或L858R点突变）→ 使用酪氨酸激酶抑制剂（如吉非替尼）
   - **ALK 融合阳性** → 使用 ALK 抑制剂（如克唑替尼）
   - **ROS1 重排阳性** → 使用 ROS1 抑制剂
   - **KRAS 突变阳性**（第12或13密码子）→ 考虑 KRAS 抑制剂
4. **综合判断**：结合临床、影像和病理结果，避免假阴性/阳性导致误治。

> 注意：本流程不适用于小细胞肺癌，且需确保检测技术可靠、样本充足。
