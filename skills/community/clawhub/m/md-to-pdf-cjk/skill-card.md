## Description: <br>
Convert Markdown files to professional PDF documents with full CJK (Chinese/Japanese/Korean) support. <br>

This skill is ready for commercial/non-commercial use. <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and engineers use this skill to convert Markdown reports, documentation, invoices, proposals, contracts, and other CJK-language documents into PDF files. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: CJK characters may not render if no compatible CJK font is installed. <br>
Mitigation: Install a trusted CJK font package before converting documents that contain Chinese, Japanese, or Korean text. <br>
Risk: The documented command includes an unsupported title argument, which can cause incorrect invocation. <br>
Mitigation: Run the script with the input Markdown path and output PDF path expected by the script. <br>
Risk: Generated PDFs depend on the local ReportLab installation and available system fonts. <br>
Mitigation: Install ReportLab and font packages from trusted repositories and review generated PDFs before distribution. <br>
