# 子痫前期病理生理机制分析

## 适用对象
- 妊娠期高血压疾病患者
- 子痫前期孕妇
- 子痫患者

## 触发条件
当孕妇在妊娠20周后出现以下任一情况时触发：
- 高血压（收缩压 ≥140 mmHg 或舒张压 ≥90 mmHg）
- 蛋白尿（≥0.3 g/24h）或无蛋白尿但存在其他器官功能障碍表现
- 器官灌注减少的临床征象（如头痛、视力障碍、上腹痛、少尿、胎儿生长受限等）

## 执行步骤
1. **识别全身小动脉痉挛**：确认外周阻力增加导致血压升高。
2. **评估血管内皮损伤**：判断是否存在通透性增加、血液浓缩、血细胞比容上升及组织水肿。
3. **系统性评估各脏器灌注状态**：
   - **脑**：检查是否出现脑水肿、缺血或出血相关症状（如昏迷、抽搐、视力障碍）。
   - **肾**：检测蛋白尿、血肌酐、尿酸水平及尿量，评估肾小球滤过率（GFR）下降程度。
   - **肝**：关注右上腹痛、转氨酶升高，警惕门静脉周围出血及肝包膜下血肿。
   - **心血管**：评估心输出量、肺水肿及心衰风险，注意“低排高阻”血流动力学特征。
   - **血液系统**：监测血细胞比容变化，识别溶血或红细胞破坏迹象。
   - **子宫胎盘**：结合超声评估胎盘灌注，判断胎儿生长受限、窘迫或胎盘早剥风险。
4. **分析内分泌代谢紊乱**：评估水钠潴留、低胶体渗透压及可能的乳酸性酸中毒。

## 输出结果
生成一份结构化评估报告，说明各系统受累机制及其临床关联，用于指导进一步诊疗决策。
