# JD Shopping Skill v2.0.1

京东决策型购物助手 Skill。它不是单纯“帮你逛京东”，而是帮你判断这单值不值得在京东买、贵一点是否换来更稳的自营/时效/售后，再用浏览器把搜索、比价、看评论、选规格、加购物车做到结算前。

## 功能特性

### v2.0 - 安全浏览器辅助

- 商品搜索：关键词搜索，支持按价格、品牌、店铺类型和评价线索筛选
- 用户评价：读取评论、评分、晒单线索和重复质量问题
- 价格对比：多店铺、多规格、可见促销和优惠条件对比
- SKU 选择：确认颜色、容量、版本、服务包等规格
- 购物车准备：登录态下、经用户确认后添加商品和整理购物车
- 优惠信息：整理商品页和购物车可见优惠，结算页限定优惠交给用户核对

### v1.x - 决策支持

- 平台适用性分析
- 店铺类型指导（自营 vs 第三方）
- 价格和时机建议
- 信任和物流建议

## 安全边界

| 操作 | Agent | 用户 |
|------|-------|------|
| 搜索/浏览 | Yes | - |
| 比价/看评论 | Yes | - |
| 选规格 | Yes, with confirmation | - |
| 加购/整理购物车 | Yes, with confirmation | - |
| 登录/CAPTCHA/身份校验 | No | Yes |
| 去结算/提交订单/确认订单 | No | Yes |
| 支付/银行卡/钱包/白条 | No | Yes |

安全交接点是商品页或购物车页。Agent 不进入结算流程，不提交订单，不支付。

## 工作流程

```text
发现 -> 对比 -> 选规格 -> 加购 -> 购物车核对 -> 用户手动结算和支付
```

## 安装

```bash
clawhub install jd-shopping
```

## 使用示例

```text
用户：帮我买 Mac mini
Agent：搜索 -> 对比 -> 看评论 -> 确认规格 -> 加入购物车 -> 输出购物车就绪摘要
用户：手动登录、结算、确认订单并支付
```
