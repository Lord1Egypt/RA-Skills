## Description: <br>
Helps users discover and install agent skills when they ask for specialized capabilities or installable workflows. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[tianjin-ren](https://clawhub.ai/user/tianjin-ren) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill to identify relevant installable skills, present options, and provide commands for adding selected skills. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can steer broad help requests toward persistent global installation of third-party skills while skipping installer confirmations. <br>
Mitigation: Review the exact skill source and publisher before installation, use search-only discovery first, avoid skipped confirmations, and install globally only when the skill should affect future agent sessions. <br>
