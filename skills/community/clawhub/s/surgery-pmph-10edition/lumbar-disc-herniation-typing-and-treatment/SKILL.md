# 腰椎间盘突出症分型与治疗技能

## 适用场景
当同时满足以下条件时启用本技能：
- 患者有腰腿痛症状
- 影像学（CT/MRI）证实椎间盘突出
- 已排除无神经症状的Schmorl结节或经骨突出型

## 执行步骤

1. **识别突出类型**
   - **膨出型**：纤维环部分破裂，髓核局限隆起，表面光滑
   - **突出型**：纤维环完全破裂，髓核突入椎管，后纵韧带完整
   - **脱出型**：髓核穿破后纵韧带，呈菜花状，根部仍在椎间隙
   - **游离型**：髓核完全脱离椎间隙，游离于椎管内
   - **Schmorl结节/经骨突出型**：髓核突入椎体松质骨或向前形成骨块，**无神经压迫症状**

2. **评估紧急情况**
   - 若为**中央型突出**并出现**马尾综合征**（鞍区麻木、大小便失禁），视为**急诊手术指征**

3. **制定治疗策略**
   - **膨出型**：首选非手术治疗（卧床、牵引、NSAIDs、神经营养药）
   - **突出型**：保守无效或疼痛加重时考虑手术
   - **脱出型**：需手术治疗（如内镜下摘除）
   - **游离型**：必须手术，因游离碎片可移位造成严重压迫
   - **Schmorl结节/经骨突出型**：无需手术，仅对症处理

4. **输出结果**
   - 明确突出类型
   - 判断是否存在马尾综合征
   - 推荐非手术或手术治疗方案

> 注意：无神经症状的Schmorl结节或经骨突出型不适用手术指征，避免过度治疗。
