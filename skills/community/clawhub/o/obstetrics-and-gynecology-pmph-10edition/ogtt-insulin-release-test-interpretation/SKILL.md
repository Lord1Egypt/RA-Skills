# OGTT-胰岛素释放试验操作与结果判读

## 适用对象
- 疑似糖尿病患者
- 多囊卵巢综合征患者
- 需评估胰岛β细胞功能者

## 使用前提
确保满足以下条件后再执行：
- 患者已禁食8～12小时
- 无急性感染或应激状态
- 近期未使用影响糖代谢的药物

## 禁忌与限制
- **不适用于妊娠期糖尿病筛查**（应采用专门标准）
- **严重胃轻瘫患者结果不可靠**，应避免使用

## 执行流程
1. **术前准备**：安排患者于清晨空腹状态下进行检查。
2. **采血时间点**：
   - **0分钟（空腹）**：采集静脉血，检测空腹血糖（正常值 <5.1 mmol/L）和空腹胰岛素（参考范围 4.2～16.2 mU/L）
   - **服糖后30、60、120、180分钟**：分别采集静脉血，测定对应时间点的血糖与胰岛素水平
   - 葡萄糖负荷：口服75g无水葡萄糖溶于250～300mL温水中
3. **结果判读**：
   - **正常反应**：胰岛素峰值出现在30～60分钟，达基础值5～10倍（通常50～100 mU/L），3小时后回落至空腹水平
   - **胰岛素分泌不足**：各时点胰岛素绝对低下 → 提示β细胞功能衰竭（常见于1型糖尿病）
   - **胰岛素抵抗**：空腹及服糖后胰岛素和血糖均显著升高 → 提示外周组织对胰岛素敏感性下降
   - **胰岛素分泌延迟**：空腹胰岛素正常或偏高，但峰值延迟至>60分钟 → 典型2型糖尿病表现

> 根据胰岛素动态变化模式判断：分泌不足→警惕1型糖尿病；高峰延迟→支持2型糖尿病；持续高值→提示胰岛素抵抗，需结合临床用于多囊卵巢综合征等妇科疾病诊断。
