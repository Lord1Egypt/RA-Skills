# 结算纠纷示例

## 场景：某住宅项目结算审减争议

### 背景

某住宅项目竣工后，承包人提交结算金额为1.25亿元。发包人委托的审计机构审核后将结算金额审减至1.08亿元，审减金额1700万元。承包人对审减内容有异议。

### 争议焦点

1. **钢筋工程量争议**：审减钢筋200吨，金额约100万元
2. **混凝土单价争议**：审减C30混凝土单价，金额约300万元
3. **变更费用争议**：审减设计变更费用，金额约1200万元
4. **质量扣款争议**：因外墙渗漏问题扣款100万元

### 技能处理过程

#### 第一步：使用结算纠纷自查清单

**自查结果**（20点检查）：

| 检查项 | 是否完成 | 问题记录 |
|--------|---------|---------|
| 1. 合同结算条款明确 | ✅ | 合同约定固定单价合同 |
| 2. 招标/投标文件一致 | ✅ | 投标单价与合同单价一致 |
| 3. 补充协议 | ✅ | 无改变结算方式的补充协议 |
| 4. 结算审核办法 | ✅ | 合同约定"审计结果为准" |
| 5. 适用的计价标准 | ✅ | 使用2018版计价标准 |
| 6. 工程量计算书正确 | ❌ | 钢筋计算有争议 |
| 7. 施工图/竣工图完整 | ✅ | 图纸齐全 |
| 8. 变更工程量单独标出 | ✅ | 变更部分单独计算 |
| 9. 签证工程量经签认 | ✅ | 所有签证均有监理签认 |
| 10. 隐蔽工程记录齐全 | ✅ | 隐蔽工程验收记录完整 |
| 11. 合同单价适用正确 | ❌ | 混凝土单价有争议 |

| 13. 材料价差计算正确 | ✅ | 材料调差有据可查 |

| 16. 结算书提交时间 | ✅ | 在合同约定时限内提交 |
| 17. 审核回复时间 | ✅ | 对方在约定期限内回复 |
| 18. 关键文件有签认 | ✅ | 所有关键文件均有签认 |
| 19. 往来函件保存 | ✅ | 函件保存完整 |
| 20. 会议纪要签字确认 | ✅ | 会议纪要均有签字 |

**自查结论**：发现3个"否"，需要重点处理。

#### 第二步：争议焦点分析

**焦点一：钢筋工程量争议**

- **对方观点**：钢筋计算有误，多算200吨
- **我方证据**：
  1. 钢筋计算书（按施工图计算）
  2. 钢筋进场记录（与计算量基本一致）
  3. 监理验收记录（钢筋隐蔽验收合格）
- **知识库依据**：《工程量清单计价标准》第5.2条（工程量计算规则）
- **应对策略**：要求对方指出具体计算错误，否则应维持原计算

**焦点二：混凝土单价争议**

- **对方观点**：合同单价过高，应按市场价调整
- **我方证据**：
  1. 合同文件（明确约定C30混凝土单价）
  2. 投标文件（投标单价与合同单价一致）
  3. 中标通知书（确认中标价格）
- **知识库依据**：《建设工程施工合同纠纷司法解释》第16条（合同约定优先）
- **应对策略**：坚持合同约定，合同单价是双方真实意思表示

**焦点三：变更费用争议**

- **对方观点**：变更未经批准，费用不应计入
- **我方证据**：
  1. 设计变更通知单（有设计单位盖章）
  2. 变更实施方案（有监理审批）
  3. 费用计算书（按合同单价计算）
- **知识库依据**：《变更索赔实务指南》第3.1条（变更确认程序）
- **应对策略**：变更程序完整，费用计算有据

**焦点四：质量扣款争议**

- **对方观点**：外墙渗漏属于质量问题，应扣款
- **我方证据**：
  1. 竣工验收报告（验收合格）
  2. 质量保修承诺书（在保修范围内）
  3. 渗漏修复记录（已修复完成）
- **知识库依据**：《建设工程质量保证金管理办法》第8条（保修责任）
- **应对策略**：属于保修责任，不应在结算时扣款

#### 第三步：制定应对方案

**整体策略**：
1. **谈判优先**：争取通过协商解决
2. **证据充分**：准备完整的证据材料
3. **法律支撑**：明确法律依据
4. **专业支持**：必要时请造价咨询机构出具意见

**分步方案**：

| 争议焦点 | 谈判底线 | 可接受方案 | 谈判话术 |
|---------|---------|-----------|---------|
| 钢筋工程量 | 不审减 | 双方共同复核 | "我们愿意共同复核计算，但必须有依据" |
| 混凝土单价 | 按合同价 | 维持合同单价 | "合同单价是双方约定的，应遵守" |
| 变更费用 | 全额计入 | 按合同单价计算 | "变更程序完整，费用计算有据" |
| 质量扣款 | 不扣款 | 转为保修责任 | "属于保修范围，不应在结算时扣款" |

#### 第四步：准备争议解决材料

**材料清单**：
1. 争议焦点分析报告
2. 证据材料汇编（按争议焦点分类）
3. 法律依据汇编
4. 谈判方案
5. 专家意见（如有）

### 技能产出

**完整争议解决包**：
1. 争议焦点分析表
2. 证据链构建图
3. 法律依据清单
4. 谈判策略方案
5. 争议解决预案

### 关键要点

1. **证据为王**：每个争议点都要有充分证据
2. **法律支撑**：引用具体的法律条文和司法解释
3. **专业支持**：必要时寻求专业机构支持
4. **策略灵活**：根据对方反应调整策略
