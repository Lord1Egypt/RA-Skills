# 未休年休假计算公式大全

> 数据来源：《职工带薪年休假条例》（国务院令第514号）、《企业职工带薪年休假实施办法》（人社部令第1号）
> 最后更新：2026年4月

---

## 一、享受年休假的前提条件

**《实施办法》第三条：** 职工连续工作满12个月以上的，享受带薪年休假。

- "连续工作满12个月"不限于同一用人单位
- 跨单位工作时间可合并计算

---

## 二、年休假天数标准

**《条例》第三条：**

| 累计工作年限 | 年休假天数 |
|:---|:---:|
| 已满1年不满10年 | **5天** |
| 已满10年不满20年 | **10天** |
| 已满20年 | **15天** |

> 注：国家法定休假日、休息日不计入年休假的假期。

---

## 三、新员工入职当年折算

**《实施办法》第五条：**

### 适用场景
当年度新入职员工，当年度年休假天数按在本单位剩余日历天数折算。

### 计算公式
```
当年可休天数 = (当年度在本单位剩余日历天数 ÷ 365天) × 员工全年应休天数
```

### 特殊处理规则
- **折算后不足1整天的部分，不计入**（不享受年休假）
- 剩余日历天数含入职当日

### 计算示例
假设员工2026年3月15日入职，累计工作年限8年（全额应休5天）：
```
剩余日历天数 = 从3月15日到12月31日 = 292天
折算后天数 = (292 ÷ 365) × 5 = 4.0天 → 取整为4天
```

---

## 四、离职员工年休假折算

**《实施办法》第十二条：**

### 适用场景
用人单位与职工解除或终止劳动合同时，当年度未安排职工休满应休年休假天数。

### 计算公式
```
离职时剩余应休天数 = (当年度在本单位已过日历天数 ÷ 365天) × 员工全年应休天数 - 当年度已安排年休假天数
```

### 特殊处理规则
1. **折算后不足1整天的部分，不计入**（不支付未休年休假工资报酬）
2. **已安排年休假多于折算应休天数的，多出部分不再扣回**

### 已过日历天数计算
- 从当年度1月1日起算
- 含离职当日

### 计算示例
假设员工2026年4月27日离职，累计工作年限12年（全额应休10天），当年度已休3天：
```
已过日历天数 = 从1月1日到4月27日 = 117天
折算应休天数 = (117 ÷ 365) × 10 = 3.21天 → 取整为3天
未休天数 = max(0, 3 - 3) = 0天
```

## 五、日工资计算

**《实施办法》第十一条：**

### 计算公式
```
日工资 = 月工资 ÷ 21.75
```

### 重要说明
- 月工资是指**剔除加班工资后**的月平均工资
- 在本用人单位工作时间不满12个月的，按实际月份计算月平均工资

---

## 六、未休年休假工资报酬计算

**《条例》第五条、《实施办法》第十条：**

### 计算公式
```
未休年休假工资报酬 = 日工资 × 未休天数 × 300%
```

### 重要说明
1. **300%包含正常工作期间的工资收入**
   - 即：正常工作工资（100%）+ 额外补偿（200%）
   - 实际额外应发补偿 = 日工资 × 未休天数 × 200%

2. **职工本人原因书面提出不休的除外**
   - 用人单位安排职工休年休假，职工因本人原因且书面提出不休的
   - 单位可以只支付其正常工作期间的工资收入

### 计算示例
假设日工资为500元，未休5天：
```
总工资报酬 = 500 × 5 × 300% = 7,500元
其中正常工资 = 500 × 5 = 2,500元（已发）
额外补偿 = 500 × 5 × 200% = 5,000元（应补发）
```

## 七、不享受当年年休假的情形

**《条例》第四条：**

职工有下列情形之一的，不享受当年的年休假：
1. 职工依法享受寒暑假，其休假天数多于年休假天数的
2. 职工请事假累计20天以上且单位按照规定不扣工资的
3. 累计工作满1年不满10年的职工，请病假累计2个月以上的
4. 累计工作满10年不满20年的职工，请病假累计3个月以上的
5. 累计工作满20年以上的职工，请病假累计4个月以上的

**《实施办法》第八条补充：**
- 职工已享受当年年休假，年度内又出现上述第（二）（三）（四）（五）项情形之一的
- 不享受下一年度的年休假

---

## 八、特殊假期不计入年休假

**《实施办法》第六条：**
- 探亲假
- 婚丧假
- 产假
- 因工伤停工留薪期间

上述假期**不计入年休假假期**。

---

## 九、寒暑假与年休假的关系

**《实施办法》第七条：**
- 享受寒暑假天数**多于**年休假天数的：不享受当年年休假
- 寒暑假天数**少于**年休假天数的：用人单位应当安排补足

---

## 十、计算参数速查表

| 参数 | 取值 |
|:---|:---|
| 月计薪天数 | 21.75天 |
| 年休假倍数 | 300%（含正常工作工资100% + 额外补偿200%） |
| 折算不足1天处理 | 不计入（向下取整） |
| 离职已安排多于折算 | 多出部分不再扣回 |

## 十一、典型场景速算

### 场景1：新员工入职
| 输入 | 输出 |
|:---|:---|
| 入职日期、累计工作年限 | 当年可休天数 |

### 场景2：正常离职
| 输入 | 输出 |
|:---|:---|
| 离职日期、累计工作年限、已休天数、月工资 | 未休天数、未休补偿金额 |

### 场景3：在职结算
| 输入 | 输出 |
|:---|:---|
| 当年度应休天数、已休天数、月工资 | 未休天数、未休补偿金额 |
