# 幽门螺杆菌根除的联合用药方案

## 触发条件
当消化性溃疡患者检测出幽门螺杆菌阳性，且满足以下前提条件时，启动本方案：
- 已确诊幽门螺杆菌感染
- 无抗生素过敏史
- 肝肾功能基本正常

## 执行步骤
1. **必须采用联合用药**：单用抗生素易导致耐药且根除率低，禁止单独使用任何一种抗菌药物。
2. **选择推荐方案类型**：
   - **三联疗法**：质子泵抑制剂（PPI） + 两种抗菌药
   - **四联疗法**：PPI + 两种抗菌药 + 铋制剂
3. **选用具体药物组合**（关键抗生素不可随意替换）：
   - PPI（如奥美拉唑、兰索拉唑等） + 克拉霉素 + 阿莫西林
   - 或 PPI + 克拉霉素 + 甲硝唑（适用于青霉素过敏者）
   - 或 枸橼酸铋钾 + 四环素（或阿莫西林） + 甲硝唑
4. **遵守关键限制**：
   - 克拉霉素不可用红霉素或阿奇霉素替代
   - 阿莫西林不可用其他半合成青霉素替代
   - 四环素不可用多西环素替代
5. **疗程设定**：标准疗程为14天。
6. **理解辅助机制**：PPI通过升高胃内pH值，增强抗菌药物的稳定性和活性。
7. **预期疗效**：合理方案的根治率可达80%～90%，显著降低溃疡复发率（未经根除治疗的溃疡年复发率高达80%）。

## 输出结果
制定并执行为期14天的三联或四联疗法，以根除幽门螺杆菌并预防消化性溃疡复发。
