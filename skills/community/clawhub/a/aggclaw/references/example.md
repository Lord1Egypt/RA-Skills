# Usage Examples

## API Request Format (new conversation)

### Chinese (game mode)
```json
{"input": "分析 Zen Word 的广告创意策略", "chat_mode": 7, "language_code": "zh"}
```

### Japanese (non-game mode)
```json
{"input": "TikTokのグローバル広告クリエイティブ戦略を分析してください", "chat_mode": 8, "language_code": "ja"}
```

### English (game mode)
```json
{"input": "Analyze the ad creative strategy for Royal Match", "chat_mode": 7, "language_code": "en"}
```

### Follow-up (reuse session, omit chat_mode + language_code)
```json
{"input": "这些素材中哪个情绪钩子最有效？", "session_id": "ZcX2pafLzbx0jwGdhZ7EsQ=="}
```

## Example Input

```
/aggclaw What are the main gameplay features of high-exposure RPG game creatives in China over the last 30 days, and which anime IPs have appeared?
```
