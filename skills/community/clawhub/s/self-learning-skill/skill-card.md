## Description: <br>
Guides an agent to keep self-review notes, learn from errors, generalize patterns, and improve task execution over time. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[davidme6](https://clawhub.ai/user/davidme6) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users can use this skill to encourage structured retrospectives, error tracking, and iterative improvement across technical tasks. It is best suited for workflows where users explicitly want persistent learning habits and review records. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The security scan notes broad automatic learning and logging habits that can retain private project details. <br>
Mitigation: Set explicit boundaries before use and require approval before creating or updating persistent notes that contain sensitive project, user, or business information. <br>
Risk: The security scan highlights credential-search and external-account action habits. <br>
Mitigation: Do not allow credential or cookie searches, token handling, account actions, or broad website automation unless the user gives task-specific authorization. <br>
Risk: Artifact examples encourage broad browser automation and fallback workflows across websites. <br>
Mitigation: Constrain automation to named sites and actions, prefer manual confirmation for account-affecting steps, and stop when selectors, permissions, or user intent are unclear. <br>
