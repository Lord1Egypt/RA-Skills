# 瓦匠成本估算参考

## 瓷砖用量计算公式

### 地面砖片数
```
片数 = (房间长 × 房间宽) / (单片长 × 单片宽) × (1 + 损耗率)
```

### 墙面砖片数
```
墙面面积 = 墙周长 × 铺贴高度 - 门窗洞口面积
片数 = 墙面面积 / (单片长 × 单片宽) × (1 + 损耗率)
```

### 损耗率参考表

| 空间 | 砖规格 | 损耗率 |
|------|--------|--------|
| 客厅 | 常规地砖（800×800） | 3-5% |
| 客厅 | 大板（900×1800） | 8-12% |
| 厨房 | 常规墙砖（300×600） | 5-8% |
| 卫生间 | 小砖、异形砖 | 10-15% |
| 过道/异形 | 任何规格 | 8-12% |

### 水泥砂估算（水泥砂浆铺贴，每㎡）

| 铺层厚度 | 水泥(kg) | 砂(kg) |
|---------|---------|--------|
| 20mm | 8-10 | 25-30 |
| 30mm | 12-15 | 35-45 |
| 50mm | 20-25 | 55-65 |

**实用估算**：水泥砂浆铺贴 ≈ 水泥 10kg/㎡ + 砂 30kg/㎡（常规厚度）

### 瓷砖胶估算

| 砖规格 | 瓷砖胶用量(kg/㎡) |
|--------|-----------------|
| 300×600 及以下 | 3-5 |
| 600×600 | 5-7 |
| 600×1200 | 7-9 |
| 900×1800 | 8-12 |

> 一包20kg瓷砖胶可铺约：
> - 小砖：4-6㎡
> - 大板：1.5-2.5㎡

## 总成本构成

```
总成本 = 瓷砖物料 + 辅料 + 人工 + 美缝
```

**案例估算：10㎡ 卫生间（C级装修标准）**

| 项目 | 费用 |
|------|------|
| 墙地砖（40㎡ × 80元/㎡） | 3,200元 |
| 水泥砂/瓷砖胶 | 400元 |
| 防水材料 | 500元 |
| 人工费（铺贴+找平+防水） | 2,800元 |
| 美缝 | 600元 |
| **合计** | **7,500元** |
| **折合每平米** | **750元/㎡** |
