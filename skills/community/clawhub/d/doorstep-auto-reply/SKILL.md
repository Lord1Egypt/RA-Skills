# Auto-Reply — 多平台自动回复系统

## 概述

通用自动回复系统，支持4个主流平台的评论、私信、关键词回复。可配置关键词库+AI动态生成双模式。

## 快速开始

当用户提出自动回复需求时：

1. **确定平台** — 抖音/小红书/微信公众号/飞书
2. **确定回复场景** — 评论回复/私信回复/关键词自动回复
3. **收集话术** — 从用户处获取或使用AI生成
4. **生成配置** — 输出平台适配的回复规则

---

## 平台回复策略

### 1️⃣ 抖音

| 场景 | 回复风格 | 特点 |
|------|---------|------|
| 评论回复 | 短、口语化、带梗 | 字数30字以内，可以加表情 |
| 私信回复 | 热情、引导转化 | 可放联系方式/小程序链接 |
| 关键词自动 | 精准匹配 + 引导私信 | 评论触发后引导私信沟通 |

**评论回复示例：**
- 用户："怎么收费的？" → "私你了宝，看消息~ 💌"
- 用户："在哪里？" → "全国都能约哦，具体发我地址帮你查~"

**私信回复示例：**
- 首次咨询：热情问候 + 简单介绍 + 引导留联系方式
- 价格咨询：报价 + 优势说明 + 限时优惠话术
- 异议处理：先共情再解答

### 2️⃣ 小红书

| 场景 | 回复风格 | 特点 |
|------|---------|------|
| 评论回复 | 温柔、干货感、种草语气 | 可以放长回复 |
| 私信回复 | 专业、耐心、详细 | 注意平台规则，避免敏感词 |
| 关键词自动 | 引导关注/收藏/私信 | 用"姐妹们"等亲密称呼 |

**评论回复示例：**
- "求教程" → "姐妹，整理好了发你~先关注不然找不到我哦💕"
- "效果怎么样" → "坚持了一个月真的变化很大！你可以先试一节课感受下~"

### 3️⃣ 微信公众号

| 场景 | 回复风格 | 特点 |
|------|---------|------|
| 关注自动回复 | 欢迎语 + 引导操作 | 介绍服务 + 关键词提示 |
| 关键词回复 | 准确、详细、引导转化 | 支持多关键词匹配 |
| 留言回复 | 官方、专业 | 回复后可精选展示 |

**关键词回复示例：**
- 关键词"价格" → 发价格表链接 + 简单介绍
- 关键词"预约" → 发预约流程 + 客服二维码
- 关键词"教练" → 发教练团队介绍
- 关键词"地址" → 发服务区域说明

### 4️⃣ 飞书机器人

| 场景 | 回复风格 | 特点 |
|------|---------|------|
| 关键词回复 | 结构化、清晰 | 支持富文本、按钮卡片 |
| 客户咨询 | 专业、高效 | 可嵌入预约系统 |
| 自动问答 | 知识库匹配 | 支持多轮对话 |

## 回复模板库

### 通用话术模板（适用于所有平台）

```
【询价话术】
用户：怎么收费？
回复：我们主要有[按次/月卡/私教包月]三种模式，[XX元/次]起。
       初次体验有[XX元]的优惠价，私信我帮您安排~

【询址话术】
用户：在哪里？
回复：我们是上门服务，您在家或者在小区都可以~ 
       全国[XX]个城市已开通，发我地址帮您查附近有没有教练~

【效果话术】
用户：有效果吗？
回复：我们根据每个人的体质定制方案，大部分学员4周就能看到明显变化。
       发您一些学员案例看看？😊

【犹豫话术】
用户：考虑一下
回复：没事，您慢慢考虑~ 
       不过首体验限时优惠还剩[XX]个名额了，先体验再决定也不亏~
```

## 部署方式

### 抖音自动回复
- 使用抖音企业号后台 → 自动回复 → 关键词回复
- 或通过第三方工具（如蝉妈妈、飞瓜数据）

### 小红书自动回复
- 专业号后台 → 消息管理 → 自动回复设置
- 评论回复需人工+工具辅助

### 微信公众号自动回复
- 公众号后台 → 自动回复 → 关键词回复/被关注回复
- 高级功能：使用公众号开发接口

### 飞书自动回复
- 飞书机器人 → 自定义回复规则
- 支持关键词匹配 + 知识库问答
