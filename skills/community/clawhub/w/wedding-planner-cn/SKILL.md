# 婚庆策划运营助手 💒

专为婚庆从业者及备婚新人打造的全流程 Skill，覆盖预算→策划→执行→复盘完整链路。

## 核心工作流

### 1. 婚礼预算分配

```
总预算分配参考（根据规模调整）：

| 项目 | 占比 | 说明 |
|------|------|------|
| 婚宴酒席 | 40-50% | 最大支出项 |
| 婚礼策划+布置 | 10-15% | 含花艺、灯光、道具 |
| 摄影摄像 | 8-10% | 双机位+航拍 |
| 婚纱礼服 | 5-8% | 含新郎西装 |
| 化妆造型 | 3-5% | 含跟妆 |
| 主持人 | 2-3% | 含音响设备 |
| 婚车 | 2-3% | 主车+跟车 |
| 喜糖伴手礼 | 3-5% | 按桌/按人计算 |
| 机动预算 | 5-10% | 应急和超支 |

预算档次参考（一线城市）：
- 简约型：8-15万（10-15桌）
- 精致型：15-30万（15-25桌）
- 奢华型：30-60万（25-40桌）
- 顶配型：60万+（40桌+）
```

### 2. 场地筛选评估

```
场地评估清单（按优先级）：

位置交通（权重20%）：
- 距新娘家/酒店车程
- 停车位数量（≥桌数×0.5）
- 公共交通可达性

场地条件（权重30%）：
- 层高（≥4米适合搭建，<3.5米很受限）
- 柱子（无柱最佳，有柱需评估遮挡）
- 进场时间（前一天晚上>当天上午>当天下午）
- 退场时间（至少2小时拆场）
- 电力容量（灯光+音响≥60A）

餐饮服务（权重25%）：
- 菜品试吃评分
- 是否允许外带酒水（省50%酒水费）
- 素食/过敏餐能否定制
- 服务人员配比（1桌1人最佳）

性价比（权重25%）：
- 场地费+最低消费
- 是否含婚房
- 是否含LED屏/音响
- 服务费比例（15%服务费很常见）

⚠️ 必须现场考察！图片和实际差距很大
```

### 3. 婚礼流程策划

**中式婚礼时间线**：
```
06:00 新娘化妆开始
08:00 新郎出发接亲
08:30 堵门游戏（控制在30分钟内）
09:00 找红鞋+敬茶
09:30 出发去酒店/新家
10:00 新家敬茶+拍照
11:00 到达酒店准备
11:58 仪式开始（吉时）
12:00 主仪式（20-30分钟）
12:30 开席
13:00 换敬酒服
13:15 敬酒（按桌数，每桌1-2分钟）
14:00 婚宴结束
```

**西式婚礼时间线**：
```
15:00 First Look + 拍照
16:00 宾客签到
16:30 仪式开始
  - 主持人开场
  - 新郎入场
  - 伴娘入场
  - 新娘入场（父亲陪同）
  - 誓言交换
  - 戒指交换
  - 亲吻+退场
17:00 鸡尾酒会+合影
17:30 晚宴开始
18:00 新人入场+第一支舞
18:30 用餐
19:30 互动游戏+抽奖
20:00 切蛋糕+抛花球
20:30 晚宴结束
```

### 4. 婚庆获客与营销

**获客渠道**（按ROI排序）：

```
1. 小红书种草（ROI最高）
   - 发布真实婚礼案例（9图+长文）
   - 标题公式：城市+风格+预算（例：杭州法式婚礼|3万布置|太美了）
   - 每周3-5篇，持续3个月见效
   - 成本：0元（仅时间投入）

2. 抖音短视频（起量最快）
   - 婚礼当天快剪（15-30秒）
   - 备婚干货视频（3-5分钟）
   - 直播婚礼现场（需新人同意）
   - 成本：0-500元/月

3. 婚博会/展会（获客最精准）
   - 展位费3000-20000元
   - 现场签约率10-20%
   - 必须有现场优惠促单
   - 适合季度1-2次

4. 异业合作（成本最低）
   - 与婚纱店/化妆师/摄影师互推
   - 佣金比例10-20%
   - 建立长期合作关系

5. 大众点评（稳定流量）
   - 精修案例+客户评价
   - 投推广费500-2000元/月
   - 转化率稳定但竞争大
```

**报价话术框架**：
```
第1步：了解需求（预算/桌数/风格/档期）
第2步：展示案例（匹配度最高的2-3个）
第3步：价值塑造（强调差异化服务，不只比价格）
第4步：报价（分项报价，让客户看到每一分钱的价值）
第5步：促单（限时优惠/档期紧张/赠品加码）
```

### 5. 供应商管理

```
核心供应商清单及筛选标准：

摄影师：看完整客片（非精选），≥2场同风格案例
摄像师：看完整成片，确认双机位/航拍配置
化妆师：试妆！必须试妆！看素颜→完妆对比
主持人：看完整主持视频，确认控场风格
花艺师：看现场图（非效果图），确认花材品质
灯光师：看现场视频（非照片），确认灯光层次

合同必备条款：
- 服务时间（超时费怎么算）
- 交付时间（照片/视频多久给）
- 修改次数（通常2-3次免费修改）
- 违约责任（双方）
- 定金比例（≤30%）
```

### 6. 婚礼当天执行

```
婚礼日协调员Checklist：

提前1天：
□ 确认所有供应商到场时间
□ 确认场地进场时间
□ 检查所有道具/物料清单
□ 准备应急包（针线包/别针/创可贴/胶带/水/零食）

当天：
□ 6:00 到达场地
□ 检查音响/灯光/LED屏
□ 确认花艺布置到位
□ 跟进化妆进度
□ 协调摄影摄像机位
□ 控制堵门时间（不超过30分钟）
□ 把控仪式节奏
□ 安排敬酒路线
□ 应急处理（随时待命）

⚠️ 最常出问题的事项：
1. 供应商迟到 → 提前1天再确认
2. 麦克风没电 → 准备备用电池
3. 堵门超时 → 提前和伴娘沟通时间
4. 菜品上错 → 提前和酒店确认菜单
5. 天气变化 → Plan B永远要准备
```

## 使用方式

### 场景1：帮新人做预算
用户说："预算XX万，帮我规划婚礼"
→ 按预算分配表，根据城市和桌数给出详细方案

### 场景2：场地推荐
用户说："在XX城市找婚礼场地"
→ 按场地评估清单，询问需求后给筛选建议

### 场景3：流程策划
用户说："帮我策划婚礼流程"
→ 确认中式/西式，按时间线模板定制

### 场景4：获客推广
用户说："婚庆公司怎么获客？"
→ 按5个获客渠道给具体执行方案

### 场景5：供应商筛选
用户说："选摄影师要注意什么？"
→ 按供应商筛选标准给具体建议

## 注意事项

- 预算建议基于一般情况，需结合当地物价调整
- 场地必须实地考察，不可仅凭图片决策
- 合同条款建议请专业律师审核
- 婚礼当天必须准备应急方案
