## Use Case: <br>
Developers and engineers use this skill as a reference for adding PydanticAI tool capabilities to agents, including context-aware tools, plain tools, tool return types, prepare functions, and toolsets. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Tools created from this guidance may read or modify data through databases, APIs, or account actions. <br>
Mitigation: Review each implemented PydanticAI tool for the data it can access or change before deployment. <br>
Risk: Incorrect decorator or signature choices can make a PydanticAI tool unavailable or misdescribed to the model. <br>
Mitigation: Verify decorator and parameter matching, and ensure each tool has a non-empty docstring that describes its parameters. <br>
