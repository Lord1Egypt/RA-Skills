## 文案生成清单与依赖拓扑

文案生成按使用流程组织的任务清单，附任务间关系和执行映射提示。

**域间逻辑流**：C0 → C1 → C2/C3 → C4（按需调用）

---

### C0 需求分析域

| ID | 任务类型 | 说明 | 依赖 | 执行映射 |
|----|---------|------|------|----------|
| C0-01 | 文案类型识别 | 识别用户需要的文案类型：广告/营销/品牌/产品/社交媒体 | 无（入口） | S |
| C0-02 | 目标受众分析 | 分析目标受众：年龄、性别、兴趣、需求 | C0-01 | C |
| C0-03 | 使用场景分析 | 分析使用场景：线上/线下、正式/非正式 | C0-02 | C |
| C0-04 | 输出格式选择 | 根据需求选择输出格式：文本/文档/网页 | C0-03 | C→A |

### C1 设计规范域

| ID | 任务类型 | 说明 | 依赖 | 执行映射 |
|----|---------|------|------|----------|
| C1-01 | 语言系统应用 | 选择合适的语言风格：正式/非正式、严肃/活泼 | 无（入口） | C→A |
| C1-02 | 风格系统应用 | 应用文案风格：简洁/详细、理性/感性 | C1-01 | A |
| C1-03 | 结构系统应用 | 应用文案结构：标题/正文/口号/标语 | C1-02 | A |
| C1-04 | 情感系统应用 | 应用情感色彩：情感诉求、语气语调 | C1-03 | A |

### C2 内容生成域

| ID | 任务类型 | 说明 | 依赖 | 执行映射 |
|----|---------|------|------|----------|
| C2-01 | 广告文案生成 | 生成产品广告、品牌广告、促销广告 | C1-04 | A |
| C2-02 | 营销文案生成 | 生成营销邮件、推广文案、销售文案 | C1-04 | A |
| C2-03 | 品牌文案生成 | 生成品牌故事、品牌理念、品牌口号 | C1-04 | A |
| C2-04 | 产品文案生成 | 生成产品描述、产品卖点、产品介绍 | C1-04 | A |

### C3 格式处理域

| ID | 任务类型 | 说明 | 依赖 | 执行映射 |
|----|---------|------|------|----------|
| C3-01 | 样式应用 | 应用文案样式到内容 | C2-01/C2-02/C2-03/C2-04 | A |
| C3-02 | 格式调整 | 调整语言、风格、结构 | C3-01 | A |
| C3-03 | 排版优化 | 优化文案排版和布局 | C3-02 | A |
| C3-04 | 兼容性处理 | 处理不同输出格式的兼容性 | C3-03 | G |

### C4 输出保存域

| ID | 任务类型 | 说明 | 依赖 | 执行映射 |
|----|---------|------|------|----------|
| C4-01 | 文本文件输出 | 保存为.txt文件 | C3-04 | O |
| C4-02 | 文档文件输出 | 保存为.docx或.pdf文件 | C3-04 | O |
| C4-03 | 网页文件输出 | 保存为.html文件 | C3-04 | O |

---

### 依赖拓扑摘要

以下为文案生成任务的主要依赖链路：

#### 一、域内链路

**需求分析链路**: C0-01 → C0-02 → C0-03 → C0-04
**设计规范链路**: C1-01 → C1-02 → C1-03 → C1-04
**内容生成链路**: C1-04 → C2-01/C2-02/C2-03/C2-04 → C3-01
**格式处理链路**: C3-01 → C3-02 → C3-03 → C3-04 → C4-01

#### 二、跨域协同链路

**完整文案生成链路**: C0-01 → C1-01 → C2-01 → C3-01 → C4-01

**格式选择链路**: C0-04 → C2-01（格式决定内容生成方式）
**规范应用链路**: C1-04 → C2-01（规范应用到内容生成）
**输出保存链路**: C4-01/C4-02/C4-03（输出后验证和兼容处理）

#### 三、设计规范依赖

**语言系统**：C1-01 → 所有生成任务
**风格系统**：C1-02 → 所有生成任务
**结构系统**：C1-03 → 所有生成任务
**情感系统**：C1-04 → 所有生成任务

## 设计规范速查表

### 语言系统

| 风格 | 适用场景 | 特点 |
|------|----------|------|
| 正式 | 商务文案、官方公告 | 严谨、规范、专业 |
| 非正式 | 社交媒体、朋友交流 | 轻松、亲切、活泼 |
| 严肃 | 法律文案、医疗文案 | 严谨、准确、权威 |
| 活泼 | 年轻受众、娱乐产品 | 有趣、生动、有活力 |
| 专业 | 技术文案、行业报告 | 专业、准确、深入 |
| 通俗 | 大众产品、日常消费 | 简单、易懂、亲切 |

### 风格系统

| 风格 | 适用场景 | 特点 |
|------|----------|------|
| 简洁 | 广告标语、社交媒体 | 简短、有力、易记 |
| 详细 | 产品介绍、品牌故事 | 详细、全面、深入 |
| 理性 | 技术产品、专业服务 | 逻辑、数据、事实 |
| 感性 | 品牌故事、情感营销 | 情感、故事、共鸣 |
| 直接 | 促销广告、销售文案 | 直接、明确、行动导向 |
| 含蓄 | 品牌形象、高端产品 | 含蓄、优雅、有品味 |

### 结构系统

| 元素 | 用途 | 特点 |
|------|------|------|
| 标题 | 吸引注意力 | 简短、有力、有吸引力 |
| 正文 | 传达信息 | 清晰、有条理、有说服力 |
| 口号 | 品牌记忆点 | 简短、易记、有特色 |
| 标语 | 活动主题 | 明确、有号召力、有行动导向 |

### 情感系统

| 情感 | 适用场景 | 特点 |
|------|----------|------|
| 温馨 | 家庭产品、亲情营销 | 温暖、亲切、有爱 |
| 激情 | 运动产品、年轻人营销 | 热情、活力、有冲劲 |
| 信任 | 金融产品、专业服务 | 可靠、专业、值得信赖 |
| 时尚 | 潮流产品、年轻人营销 | 时尚、前卫、有品味 |
| 怀旧 | 经典产品、复古营销 | 怀旧、经典、有情怀 |
| 创新 | 科技产品、创新服务 | 创新、前沿、有未来感
