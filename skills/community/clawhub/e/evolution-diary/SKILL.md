## 核心模块

### 🌀 模块一：心流时刻

记录你进入最佳状态的时刻。

**触发场景：**
- 连续深度工作超过 2 小时
- 解决了一个很难的问题
- 灵感爆发，效率是平时的 3 倍以上

**记录格式：**
```
🌀 心流时刻
  时间：[timestamp]
  持续：约 X 小时
  触发：任务类型 / 情绪状态
  产出：完成了什么
  感受："当时的状态描述"
```

**自动触发：**
- 当单次任务超过 90 分钟且输出质量评分 > 8 时，自动建议记录心流

---

### 🌊 模块二：低谷期记录

记录状态低迷的时刻，分析原因，给自己安慰。

**触发场景：**
- 连续 3 次任务失败
- 用户表达了负面情绪（"太难了"、"想放弃"）
- 效率评分低于 40%

**记录格式：**
```
🌊 低谷期
  时间：[timestamp]
  原因：任务太难 / 状态疲惫 / 情绪低落 / 其他
  表现："当时的感受描述"
  自我安慰：[一段温暖的话]
  恢复建议：休息 20 分钟 / 换个任务 / 找人聊聊
```

**低谷期关怀话术：**
> "低潮不是失败，是身体在告诉你需要休息。给自己泡一杯茶，明天又是新的一天。"

---

### ✅ 模块三：习惯打卡

记录每天坚持的小事，积小胜为大胜。

**预设习惯库：**
- 早起打卡
- 运动打卡
- 阅读打卡
- 写作打卡
- 冥想打卡
- 学习打卡
- 复盘打卡

**打卡格式：**
```
✅ 习惯打卡
  日期：YYYY-MM-DD
  习惯：习惯名称
  完成度：🟢完全 / 🟡部分 / 🔴未完成
  持续天数：X 天（连续）
  备注：今天的感觉
```

** streak 机制：**
- 连续 7 天 → 获得 🌱 小芽 称号
- 连续 30 天 → 获得 🌿 小树 称号
- 连续 100 天 → 获得 🌳 大树 称号
- 中断后重新计数，但历史记录保留

---

### 💚 模块四：情绪曲线

记录每天的情绪状态，画出一条只有你能看懂的曲线。

**维度评分（1-10）：**
- 能量：1=精疲力竭，10=精力充沛
- 动力：1=完全不想动，10=充满干劲
- 平静：1=焦虑烦躁，10=内心平静

**每日记录：**
```
💚 今日情绪
  日期：YYYY-MM-DD
  能量：●●●●●●○○○○ 6/10
  动力：●●●●●●●●○○ 8/10
  平静：●●●●●○○○○○ 5/10
  
  峰值时刻：[几点，发生了什么事]
  低谷时刻：[几点，发生了什么事]
  总体感受："一句话描述今天的情绪"
```

**周曲线图：**
```
💚 本周情绪曲线
  一  二  三  四  五  六  日
  能量 ██ █░ ██ ██ ██ █░ ██
  动力 █░ ██ █░ ██ ██ ██ █░
  平静 █░ █░ ██ █░ ██ ██ █░
```

### 日常记录

**习惯打卡：**
```
用户："打卡"
→ 显示预设习惯列表，等待选择
→ 选择后更新 streak 数据
→ 达到 7/30/100 天时触发成就
```

**心流记录：**
```
用户："记录心流" 或 触发自动
→ 询问持续时间和产出
→ 写入 heartflow 数组
→ 更新总心流小时数
```

**低谷记录：**
```
用户："记录低谷" 或 触发自动
→ 询问原因和感受
→ 写入 troughs 数组
→ 生成自我安慰话术
```

**情绪记录：**
```
用户："记录情绪" 或 "今天情绪怎么样"
→ 引导打分（能量/动力/平静）
→ 记录峰值和低谷时刻
→ 更新日度情绪数据
```

---

### 周自动生成

每周日 23:00 检查：
- 本周心流次数
- 本周习惯坚持情况
- 本周情绪曲线
- 本周成就

生成进化信草稿，询问是否保存。

### 🏆 成就系统

| 成就 | 条件 | 称号 |
|------|------|------|
| 🌱 破土 | 任意习惯连续 7 天 | 小芽 |
| 🌿 扎根 | 任意习惯连续 30 天 | 小树 |
| 🌳 成材 | 任意习惯连续 100 天 | 大树 |
| 🌀 心流达人 | 累计心流超过 50 小时 | 心流达人 |

| 📮 反思者 | 生成 4 封进化信 | 反思者 |

### 🎯 成长建议

根据数据自动生成建议：

- 心流多但低谷也多 → "你是个高能量玩家，但记得给自己留白"
- 习惯坚持好但情绪波动大 → "试着在情绪低谷时做简单的任务"
- 情绪平稳但心流少 → "也许需要找一个让你兴奋的目标"

## 注意事项

1. **数据完全私密**：所有数据只存在本地，不上传
2. **温度优先**：不要冷冰冰地报数字，要有人味儿
3. **不说教**：不教训用户，不扮演人生导师
4. **陪伴感**：像一个安静的朋友，知道你什么时候需要鼓励，什么时候需要安静
