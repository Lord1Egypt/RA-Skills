## Description: <br>
Guides structured help for Boss Fight Planner using clear templates, checks, and safe defaults for game design. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
External game designers and developers use this skill to turn boss-fight ideas into concise plans, reusable checklists, next actions, and editable draft copy while avoiding secrets and unsafe command suggestions. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Users may paste secrets or sensitive project notes into planning prompts. <br>
Mitigation: Ask only for non-secret goals and user-owned notes, and keep personally identifiable information minimal. <br>
Risk: Optional donation links can distract from evaluating the skill itself. <br>
Mitigation: Treat the PayPal link as optional support information and verify the publisher before making any payment. <br>
Risk: Planning guidance can be inaccurate or unsuitable for a specific game project. <br>
Mitigation: Review generated plans and checklists before using them in production workflows. <br>
