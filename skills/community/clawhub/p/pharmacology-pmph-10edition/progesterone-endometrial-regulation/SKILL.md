# 黄体酮对子宫内膜的调节及临床应用

## 何时使用
- 患者诊断为黄体功能不足导致的功能性子宫出血
- 患者出现先兆流产症状，且确认为黄体功能不足所致
- 患者处于月经周期后期或妊娠早期，并已具备雌激素对子宫内膜的基础作用

## 执行步骤
1. **确认前提条件**：确保子宫内膜已受雌激素作用，处于增殖期晚期。
2. **给予黄体酮**：
   - 在月经后期，促使子宫内膜由增殖期转为分泌期，表现为继续增厚、充血、腺体增生并分支，为受精卵着床创造条件。
   - 在妊娠早期，降低子宫对缩宫素的敏感性，抑制子宫平滑肌收缩，发挥保胎作用。
3. **调节宫颈黏液**：黄体酮同时抑制子宫颈管腺体分泌，减少精子进入子宫（此效应在保胎场景中为辅助作用）。
4. **治疗功能性子宫出血**：
   - 使用黄体酮使子宫内膜同步转化为分泌期。
   - 待停药后，内膜可完整脱落，实现规律撤退性出血，达到止血目的。
5. **注意事项**：
   - 仅对黄体功能不足引起的先兆流产有效。
   - 对习惯性流产疗效不确切，不应常规使用。
   - 禁用19-去甲睾酮类孕激素（因其具雄激素样作用，可能导致女性胎儿男性化）。

## 关键变量
- **黄体酮剂量**：大剂量用于安胎或子宫内膜异位症；常规剂量用于调整内膜周期。
- **雌激素基础**：必须存在，否则黄体酮无法有效作用于子宫内膜。
