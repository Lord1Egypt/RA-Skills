## 一、解盘工作流总览

### 1.1 三级深度体系

| 深度级别 | 适用场景 | 输出长度 | 分析时间 |
|---------|---------|---------|---------|
| **Level 1 快速概览** | 初次咨询、快速回答 | 500-800字 | 5分钟 |
| **Level 2 标准分析** | 专项问题、单领域深挖 | 2000-4000字 | 20分钟 |
| **Level 3 完整解盘** | 全面人生蓝图、年度预测 | 8000-15000字 | 60分钟+ |

### 1.2 工作流决策树

```
收到解盘请求
    │
    ├─ 有PDF星盘？ → 是 → 提取数据 → 进入分析
    │                └─ 否 → 有文本数据？ → 是 → 结构化 → 进入分析
    │                                       └─ 否 → 询问出生信息
    │
    进入分析
    │
    ├─ 用户问了具体问题？ → 是 → 专项分析（Level 2）
    │                        └─ 否 → 用户要全面分析？ → 是 → 完整解盘（Level 3）
    │                                                  └─ 否 → 快速概览（Level 1）
    │
    专项分析路由
    │
    ├─ 事业/职业 → 10宫+D10+AmK
    ├─ 婚姻/关系 → 7宫+D9+DK
    ├─ 财富/投资 → 2/11宫+D2
    ├─ 健康 → 6宫+D6+土星分析
    ├─ 子女 → 5宫+D7+PK
    ├─ 出国 → 9/12宫+Rahu分析
    ├─ 教育 → 4/5宫+D24
    └─ 灵性 → 9/12宫+Ketu+AK
```

## 三、Level 2 — 专项分析工作流

### 3.1 专项分析通用流程

```
Step 1: 确定分析领域 → 识别相关宫位
Step 2: D1分析 → 本命征象（宫主星+宫内行星+相位）
Step 3: 分盘分析 → 相关分盘确认（D9/D10/D2等）
Step 4: 行星力量评估 → Shadbala+Ashtakavarga
Step 5: Yoga识别 → 相关Yoga格局
Step 6: Jaimini层 → Karaka分析
Step 7: KP层 → Sub-Lord确认
Step 8: Dasha分析 → 当前和未来大运
Step 9: Transit分析（⚠️ 多参考点强制，详见transit-multi-reference-guide.md）
  Step 9a: 从Lagna看过境 → [宫位][含义]
  Step 9b: 从Chandra Lagna看过境 → [宫位][含义]（⚠️ 强制）
  Step 9c: 从Arudha Lagna看过境 → [宫位][含义]（Level 3强制）
  Step 9d: 从Navamsa Lagna看过境 → [宫位][含义]（Level 3可选）
  Step 9e: 四参考点综合判断 → [结论]
Step 10: 综合判断 → 三系统交叉验证
```

## 四、Level 3 — 完整解盘工作流

### 4.1 完整解盘12步法

| 步骤 | 内容 | 输出 | 时间 |
|------|------|------|------|

| **12. 综合报告** | 三系统交叉验证 | 最终报告 | 10分钟 |
