# 倪海厦天纪体系 — 紫微斗数 + 风水 + 易经

> 蒸馏自倪海厦《天纪》系列：天机道（紫微斗数）、人间道（易经六十四卦）、地脉道（阳宅风水）
> 核心理念：命相同参，天地人三才合一

---

## 天纪总纲

倪海厦天纪三大体系：

| 道 | 内容 | 核心 |
|----|------|------|
| **天机道** | 紫微斗数命盘分析 | 知天命，识吉凶，明进退 |
| **人间道** | 易经六十四卦解人事 | 处世的智慧，君子小人辨别 |
| **地脉道** | 阳宅风水/地理布局 | 环境对人的影响，"地善则苗秀，宅吉则人荣" |

### 核心理念

1. **命相同参**：命（八字/斗数）+ 相（面相/气色）= 完整判断
2. **天地人三才**：天时（命）+ 地利（风水）+ 人和（易经处事）三者缺一不可
3. **知命不认命**：知道命是为了趋吉避凶，不是消极认命
4. **象数理**：象（现象）→ 数（数据/干支）→ 理（道理/规律）

## 二、人间道 — 易经六十四卦处事智慧

倪海厦不是纯算命的，他强调的是用易经的智慧指导人生。

### 核心思想
- 易经不是用来算卦的，是用来**明理**的
- 每一卦都是告诉你在这个位置该怎么做
- 关键概念：**君子小人辨**、**时位中正**、**进退之道**

### 几个重要卦象

| 卦 | 核心教导 |
|----|---------|
| **乾** | 自强不息，但初九"潜龙勿用"——刚开始要沉住气 |
| **坤** | 厚德载物，柔顺但不是懦弱 |
| **屯** | 始生之难，"盘桓、利居贞"——不要急于求进 |
| **蒙** | 启蒙，"匪我求童蒙，童蒙求我"——教学的真谛 |
| **泰** | 天地交泰，上下一心 |
| **否** | 天地不交，君子道消 |
| **谦** | 唯一六爻皆吉的卦——谦受益、满招损 |
| **家人** | 家有严君，各司其位 |
| **革** | 变革之道，顺势而改 |

## 三、地脉道 — 阳宅风水

倪海厦讲风水是**讲理**的，不是迷信。

### 基本格局

| 方位 | 八卦 | 家庭成员 | 五行 |
|------|------|---------|------|
| 正北 | 坎 | 中子（第二个儿子） | 水 |
| 东北 | 艮 | 少子（小儿子） | 土 |
| 正东 | 震 | 长子 | 木 |
| 东南 | 巽 | 长女 | 木 |
| 正南 | 离 | 次女（二女儿） | 火 |
| 西南 | 坤 | 母亲 | 土 |
| 正西 | 兑 | 少女（小女儿） | 金 |
| 西北 | 乾 | 父亲 | 金 |

### 阳宅核心规则（倪师精选）

**名位相等法则**：
- 父亲 → 西北角卧室（乾位），大利事业
- 母亲 → 西南角卧室（坤位），家庭和睦
- 长子 → 正东卧室（震位），学业事业顺利
- 大女儿 → 东南卧室（巽位）
- **名位不等就会生病出问题**

**厨房位置**：
- 厨房在西北角（乾位）→ "火烧天门"，男主人心脏、事业出问题，夫妻死别之象
- 厨房在西南角 → 影响母亲健康

**厕所位置**：
- 厕所（污水）在房子正中央 → 心脏病、家中不宁
- 厕所对大门 → 财来财去

**特殊宅格**：
- **无子宅**：正东方是客厅/餐厅（缺东宫），主无子或子多灾
- **夫妻死别宅**：①厕所居中（心脏病）②西北厨房（刀象）
- **泼妇宅**：母亲住正北卧室（地水师卦），女主性刚好斗

### 倪海厦风水核心理念

1. **"宅子是养人的"** — 家里有病人，先看宅，宅改则病减
2. **外局 > 内局** — 房子的朝向和位置最关键
3. **光和气** — 采光好、通风好，阳气足则百病不生
4. **气场** — 房子的气要流通，不聚死气
5. **"天地人"统一** — 命盘看出问题，风水可以调，易经教你如何处事
