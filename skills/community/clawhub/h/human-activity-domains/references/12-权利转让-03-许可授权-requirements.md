## 许可授权生成要求

许可授权每种任务类型的"零件清单"——必选/可选组件、组装顺序、领域约束。按清单逐项产出。

---

### 1 授权准备

#### T1-1 许可评估

- **必选组件**: 权利价值、市场价值、风险评估、评估报告
- **可选组件**: 评估工具、评估模型、评估机构
- **组装顺序**: 评估权利价值 → 评估市场价值 → 评估风险 → 生成报告
- **约束**: 价值必须客观；市场必须准确；风险必须识别
- **格式**: 许可评估报告（Markdown）

#### T1-2 授权方案

- **必选组件**: 授权范围、授权方式、授权期限、方案说明
- **可选组件**: 方案评审、方案优化、方案对比
- **组装顺序**: 确定授权范围 → 选择授权方式 → 设定授权期限 → 撰写说明
- **约束**: 范围必须明确；方式必须可行；期限必须合理
- **格式**: 授权方案（Markdown）

#### T1-3 授权条件

- **必选组件**: 使用限制、质量要求、监督机制、条件说明
- **可选组件**: 条件评审、条件优化、条件对比
- **组装顺序**: 设定使用限制 → 设定质量要求 → 设定监督机制 → 撰写说明
- **约束**: 限制必须合理；要求必须明确；监督必须有效
- **格式**: 授权条件文件（Markdown）

#### T1-4 授权审批

- **必选组件**: 审批流程、审批材料、审批结果、审批记录
- **可选组件**: 审批顾问、审批代理、审批工具
- **组装顺序**: 准备审批材料 → 提交审批 → 跟进审批 → 记录结果
- **约束**: 材料必须完整；流程必须合规；记录必须完整
- **格式**: 授权审批文件（Markdown）

### 3 授权执行

#### T3-1 合同签订

- **必选组件**: 合同条款、合同审核、合同签署、合同备案
- **可选组件**: 合同模板、法律顾问、合同公证
- **组装顺序**: 拟定合同条款 → 审核合同 → 签署合同 → 备案合同
- **约束**: 条款必须清晰；审核必须严格；签署必须合法
- **格式**: 授权合同（Markdown）

#### T3-2 授权登记

- **组装顺序**: 提交登记申请 → 审核登记 → 确认登记 → 获取证明
- **约束**: 申请必须完整；审核必须严格；确认必须准确
- **格式**: 授权登记文件（Markdown）

#### T3-3 授权通知

- **必选组件**: 通知对象、通知内容、通知方式、通知确认
- **可选组件**: 通知平台、通知服务、通知记录
- **组装顺序**: 确定通知对象 → 准备通知内容 → 选择通知方式 → 确认通知
- **约束**: 对象必须明确；内容必须准确；方式必须正式
- **格式**: 授权通知文件（Markdown）

#### T3-4 授权生效

- **必选组件**: 生效条件、生效时间、生效确认、生效记录
- **可选组件**: 生效证明、生效公告、生效管理
- **组装顺序**: 确定生效条件 → 设定生效时间 → 确认生效 → 记录生效
- **约束**: 条件必须满足；时间必须明确；确认必须及时
- **格式**: 授权生效文件（Markdown）
