# solar-project-feasibility-report

Drafts a pre-development solar PV project feasibility report — covering resource assessment, system sizing, financial metric highlights, interconnection path, permitting checklist, and risk flags.

## Use When

A solar developer, renewable energy analyst, project finance associate, or EPC engineer needs to produce a structured feasibility document before committing capital or advancing a project into full development. Useful at the prospecting, pre-development, or early development stage for utility-scale, commercial-industrial, and community solar projects.

## What It Produces

A DRAFT feasibility report containing:

- Project overview table (capacity, location, type, stage, POI, ITC/PTC rate)
- Resource assessment (GHI, capacity factor, annual production, assumptions)
- System design summary (DC/AC ratio, array type, land use efficiency)
- Interconnection path narrative with timeline risk flag
- Permitting matrix with Not Triggered / Triggered / Unknown status per permit type
- Financial highlights table (LCOE, IRR, payback — labeled PRELIMINARY ESTIMATES)
- Risk matrix with deal-killer flags
- Open-questions list and data-gap register
- Unsigned developer/analyst review block

## Not a Substitute For

- Investment-grade financial modeling by a project finance team
- Licensed professional engineering for system design
- Environmental and cultural resources surveys by qualified consultants
- Legal advice on land control, tax equity structures, or interconnection agreements

## Scope

Covers utility-scale, commercial-industrial, and community solar PV in any U.S. jurisdiction. References NREL PVWatts, IRA ITC/PTC, FERC Order 2023 interconnection process, and standard permitting frameworks (NHPA Section 106, ESA Section 7, CWA Section 404, NEPA).
