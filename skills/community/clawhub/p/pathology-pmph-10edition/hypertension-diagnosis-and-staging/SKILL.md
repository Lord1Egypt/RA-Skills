# 高血压的诊断标准与分期

## 何时使用
- 用户提供非同日三次静息状态下测量的血压值
- 需要判断是否符合高血压诊断标准
- 需要对确诊高血压进行分级（1级、2级、3级）或识别单纯收缩期高血压
- 排除妊娠期女性或急性应激状态下的单次测量场景

## 执行步骤
1. **确认测量条件**：确保血压数据来自非同日至少三次、使用标准血压计、在静息状态下测得。
2. **应用诊断标准**：
   - 正常血压：<120 mmHg 收缩压 且 <80 mmHg 舒张压
   - 正常高值：120～139 mmHg 收缩压 和/或 80～89 mmHg 舒张压
   - 高血压：≥140 mmHg 收缩压 和/或 ≥90 mmHg 舒张压
3. **进行高血压分级**：
   - 1级（轻度）：140～159 / 90～99 mmHg
   - 2级（中度）：160～179 / 100～109 mmHg
   - 3级（重度）：≥180 / ≥110 mmHg
4. **处理不一致情况**：若收缩压与舒张压分属不同级别，以较高的分级为准。
5. **识别特殊类型**：若收缩压 ≥140 mmHg 且 舒张压 <90 mmHg，诊断为“单纯收缩期高血压”。
6. **输出结论**：明确给出诊断结果（如“2级高血压”），用于指导后续治疗和风险评估。

> 注意：本技能不适用于妊娠期女性（应使用妊娠高血压标准），且不可基于急性应激状态下的单次测量结果做出诊断。
