# 话术库

> 本文档按场景（冷启动/温启动/热启动）和角色（董秘/高管/实控人等）提供标准化话术模板。

---

## 一、冷启动话术

### 1.1 给董秘的冷启动话术

#### 首次联系邮件（行业研究者身份）

```markdown
【邮件主题】关于[行业名称]行业发展的研究交流 - [姓名]

【邮件正文】

尊敬的[董秘姓名]（或"董秘老师"）：

您好！

我是[机构名称]的行业研究员[姓名]，长期关注[行业名称]领域的发展动态。

近期我们在研究[行业趋势/技术方向]，注意到贵司在[具体业务]方面展现出前瞻性布局。
为了更好地了解行业发展现状，我们整理了一份[行业分析报告/趋势研判]，希望能与贵司
投资者关系部门进行交流。

如蒙方便，烦请安排一次30分钟以内的沟通，我们非常愿意分享我们的研究成果。

占用您宝贵时间，深表歉意。期待您的回复！

顺颂商祺

[姓名]
[职务]
[机构名称]
手机：[____]
邮箱：[____]
```

#### 电话跟进话术

```markdown
【电话话术】

"您好，请问是[公司名称]投资者关系部吗？
我是[机构名称]的[姓名]，之前发过一封邮件给董秘老师，关于[行业]研究的交流。
想确认一下邮件是否方便查阅？如董秘老师方便，希望能有一次交流机会。
好的，打扰了，谢谢！"
```

### 1.2 给高管的冷启动话术（活动场景）

```markdown
【活动接触话术】

"您好，请问是[姓名]总吗？
我是[姓名]，来自[机构/公司]。
刚才听了您关于[演讲主题]的分享，深受启发，特别是[具体观点]这部分。
我这边也在关注[相关领域]，正好有个问题想请教您...
[简短的自我介绍+提出问题]
方便的话，可以加一下您的微信吗？后续有问题想继续向您请教。"
```

顺颂商祺

## 二、温启动话术

### 2.1 请求中间人引荐话术

```markdown
【请求中间人引荐】

XX总，您好！

有件事想请您帮忙，看看是否方便。

我们团队有个[项目/技术]，是关于[简述项目]，目前正在寻找[战略合作/被并购]的机会。
经过研究，我们认为[目标公司名称]在[相关领域]的布局与我们的方向高度契合。

不知道您是否方便帮忙引荐一下[目标公司关键人姓名/职位]？

以下是项目的简要情况：
• 项目名称：[名称]
• 核心价值：[一句话]
• 期望合作：[简述]

当然，如果觉得不太合适，完全理解，不给您添麻烦。

谢谢您的时间！

[姓名]
```

一、项目概况
[用3-5句话描述项目]

• 契合点2：[描述]

[简述期望的目标公司回应]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```markdown
【中间人代为转达】

经[中间人]介绍，我们这边有个[项目/技术]想与您交流一下。

[姓名] | [联系方式]
```

## 三、热启动话术

### 3.1 背书人直接引荐话术

```markdown
【背书人开场白】

[目标姓名]，您好！

这位是我的[朋友/合作伙伴/被投企业创始人]，[姓名]。
他们团队在[领域]深耕多年，有[技术/产品]方面的优势。

今天介绍你们认识，看看是否有合作可能。
[姓名]想向您汇报一下项目情况。

[姓名]，请开始吧。
```

### 3.2 直接进入正题话术

```markdown
【直接进入正题】

[目标姓名]，感谢您百忙之中抽时间见面！

我是[姓名]，[公司名称]的[职务]。经[背书人姓名]介绍，
我们有个[项目/技术]想与贵司探讨合作可能。

[用1-2分钟简要介绍：]
• 项目是什么：[一句话]
• 核心优势：[2-3个亮点]
• 为什么适合贵司：[1-2个协同点]
• 我们希望：[简述期望]

不知道您这边对[相关领域]有什么想法或规划？
```

### 3.3 高层对等对话话术

```markdown
【高层对话】

[双方高层在场]

[我方高层]：[目标高层姓名]，感谢您接见！

我们这边有个项目想与贵司探讨，我是[姓名]，[职务]，这位是[团队成员姓名]。

项目情况让[下属]简要汇报一下。

[下属]汇报...

[我方高层]：不知道贵司在[相关领域]有什么规划或想法？

[等待目标回应后，继续深入交流]

[结束时]：
今天交流很受启发。回去后我们会根据您提到的[要点]整理一版更针对性的方案，
届时再向您汇报。您看这样安排可以吗？
```

## 四、分角色话术

### 4.1 董秘话术（投资者关系角度）

```markdown
【联系董秘】

您好，请问是[公司名称]投资者关系部吗？
我是[机构名称]的[姓名]，我们是一家专注于[行业]的投资机构。
想了解一下贵司近期在[相关领域]的战略规划，
方便约一次与投资者关系部门的交流吗？

【如被问到具体目的】
我们关注[行业]领域有一段时间了，
注意到贵司在该领域有一定布局，想进一步了解公司的规划。
不一定马上有合作，但希望建立长期信息沟通渠道。
```

### 4.2 战略投资总监话术（项目推介角度）

```markdown
【联系战略投资总监】

[姓名]总，您好！

我是[姓名]，[公司名称]的[职务]。
我们是一家专注于[领域]的[公司类型]，经过多年研发，
在[技术/产品]方面取得了一些成果。

了解到贵司在[相关领域]有战略布局，我们的产品/技术与贵司方向有一定契合，
想向您汇报一下，看看是否有合作可能。

我们期望的合作方式包括[并购/控股投资/参股/合资]等，
具体可以根据贵司的需求灵活探讨。

不知道您什么时候方便？我们可以先去贵司拜访，当面汇报。
```

### 4.3 实控人话术（简洁有力）

```markdown
【联系实控人】

[姓名总/董事长]，您好！

冒昧打扰，我是[姓名]，经[强力中间人/顾问]介绍。
我们有个[技术/项目]，与贵司[相关领域]可能有协同。

只占用您5分钟做个简要汇报：
• 我们是做什么的：[一句话]
• 核心价值：[一句话]
• 能为贵司带来什么：[一句话]

如蒙垂青，希望能有机会当面详细汇报。
[中间人姓名]可以帮我引荐一下吗？
```

### 4.4 独立董事话术（学术/专业角度）

```markdown
【联系独立董事】

[姓名]教授/总，您好！

我是[姓名]，[机构名称]的[职务]。
了解到您担任[公司名称]的独立董事，在[专业领域]有深厚造诣。

我们团队在[技术/领域]有一些研究成果和产业化探索，
想就[技术趋势/产业方向]问题向您请教。

不知道您是否方便安排一次交流？
我们可以向您汇报一下[技术/行业]的最新进展，
也听听您对[产业发展/技术路线]的专业判断。

[如果对方感兴趣]
太好了！我加一下您的微信，把项目材料发给您看。
您方便的时候，我们再约时间。
```

## 五、场景应对话术

### 5.1 被问"你们想干什么"

```markdown
【应对话术】

感谢您的直接。我们目前在做的是[项目/技术]，
希望找到[战略合作/被并购]的机会。

简单说，我们想找一个有[资源/渠道/品牌]的产业合作伙伴，
一起把[市场/技术]做大。

当然，这只是初步想法，具体合作方式可以根据对方的需求灵活探讨。
您看这个方向是否有进一步交流的必要？
```

### 5.2 被问"你们估值多少"

```markdown
【应对话术】

感谢您的直接。关于估值，我们参考了几个维度：
• 同类交易的倍数：[__]x
• 我们的[财务/技术/市场]指标：[具体数据]
• 综合考虑，我们期望的投后估值是[__]万元。

当然，这只是我们的预期，最终还需要根据[尽职调查/双方谈判]来确定。
您这边有其他考虑吗？
```

### 5.3 被问"我们为什么需要你们"

```markdown
【应对话术】

感谢您提出这个问题。我们认为，在[领域]方面，
我们可以为贵司带来以下价值：

• [价值点1]：具体说明
• [价值点2]：具体说明
• [价值点3]：具体说明

当然，这只是我们的理解。
也希望能听听贵司在[相关领域]的规划和需求，
看看我们能如何配合。
```

### 5.4 被拒绝后的应对

```markdown
【礼貌告别】

非常感谢您抽出时间接待我们。

您的意见我们记下了，[关于XX的看法/建议]对我们很有启发。
如果未来情况有变化，希望还能有机会合作。

祝您工作顺利！再见！
```

## 六、电话话术模板

### 6.1 首次电话联系

```markdown
【电话话术框架】

1. 开场白（30秒）
"您好，请问是[公司名称]吗？
我是[姓名]，[机构/公司]的[职务]，
想与贵司[投资者关系/战略投资/业务发展]部门做个交流。"

2. 说明来意（1分钟）
"我们关注[行业/领域]有一段时间了，
注意到贵司在该领域有一定布局，
想了解一下贵司的战略规划，
看是否有合作可能。"

3. 请求会面（30秒）
"不知道贵司什么时候方便？
我们可以去贵司拜访，当面做更详细的交流。"

4. 结束语
"好的，打扰您了，谢谢！"
```

### 6.2 电话跟进

```markdown
【电话跟进话术】

"您好，请问是[公司名称]吗？
我是[姓名]，上周给您发过邮件，关于[项目/合作]的交流。
不知道邮件是否方便查阅？
...（如确认已读）
您看什么时候方便？我们非常希望能有一次当面交流的机会。
...（如表示暂无兴趣）
完全理解。可能时机还不成熟。
那我们先不打扰了，如果未来有变化，希望能再联系。
谢谢！再见！"
```

## 七、话术使用注意事项

### 7.1 基本原则

| 原则 | 说明 |
|------|------|
| **简洁** | 控制在规定时间内，不要啰嗦 |
| **真诚** | 不夸大、不欺骗 |
| **专业** | 体现专业性，不露怯 |
| **尊重** | 尊重对方时间，不要死缠烂打 |

### 7.2 常见问题应对

| 问题 | 应对要点 |
|------|----------|
| 对方很忙 | 简明扼要，尊重对方时间 |
| 对方态度冷淡 | 降低期望，寻找其他渠道 |
| 对方不回复 | 等待后再次联系，语气更谦逊 |
| 对方很忙 | 预约时间，不要强聊 |
