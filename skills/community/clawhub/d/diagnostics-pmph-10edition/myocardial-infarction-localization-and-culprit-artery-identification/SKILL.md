# 心肌梗死定位诊断与相关血管判断

## 何时使用
- 患者疑似急性心肌梗死
- 心电图已记录12导联，必要时包含V₇–V₉（后壁）和V₃R–V₄R（右胸）导联
- 存在典型心肌梗死心电图改变（如异常Q波、ST段显著偏移）
- 已排除非梗死性Q波或继发性ST-T改变

## 执行步骤

### 1. 确定定位依据
- **主要依据**：异常Q波或QS波（提示心肌坏死）
- **早期（<数小时）无Q波时**：依赖ST段抬高/压低及T波异常的导联分布

### 2. 根据导联异常组合定位梗死部位
- **前间壁梗死**：V₁～V₃导联异常Q波/QS波
- **前壁梗死**：V₃、V₄（±V₅）导联异常Q波/QS波
- **侧壁梗死**：
  - I、aVL、V₅、V₆导联异常Q波 → 侧壁
  - 仅V₅、V₆ → 前侧壁
  - 仅I、aVL → 高侧壁
- **下壁梗死**：II、III、aVF导联异常Q波/QS波
- **正后壁梗死**：V₇–V₉导联异常Q波/QS波；同时V₁–V₂导联出现R波增高、ST段压低、T波增高（对应性改变）
- **广泛前壁梗死**：V₁～V₅导联均出现异常Q波/QS波
- **右心室梗死**：V₃R–V₄R导联ST段抬高≥0.1 mV（常与下壁梗死共存）

### 3. 推断最可能闭塞的冠状动脉
- **左前降支（LAD）闭塞**：前间壁、前壁或广泛前壁梗死
- **左回旋支（LCX）或LAD闭塞**：侧壁梗死
- **右冠状动脉（RCA）闭塞（多数）或LCX（少数）**：下壁梗死
- **LCX或RCA闭塞**：正后壁梗死
- **RCA近段闭塞**：下壁合并右心室梗死

### 4. 注意事项
- 多支病变、多部位梗死（向量抵消）或梗死位于导联盲区（如孤立正后壁）可导致心电图表现不典型
- 最终血管判断应结合临床表现、心肌酶学及冠脉造影结果

> 提示：若心电图表现不典型但临床高度怀疑心肌梗死，应立即启动急诊评估流程，不可仅依赖本技能结论。
