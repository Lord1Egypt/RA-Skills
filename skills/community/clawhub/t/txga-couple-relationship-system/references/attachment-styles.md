## 四种依恋风格

### 安全型（Secure）

**核心特征**：
- 能自如地依赖他人，也能独立
- 情绪稳定，不易焦虑
- 能建立深度亲密，也能承受分离
- 信任他人，不过度担心被抛弃

**关系表现**：
- 表达需求直接自然
- 冲突时能理性沟通
- 能给予对方空间
- 关系满意度高

**占比**：约55-60%人口

---

### 焦虑型（Anxious）

**核心特征**：
- 过度担心被抛弃
- 渴望高强度的亲密
- 情绪波动大，容易过度解读
- 害怕不确定性

**关系表现**：
- 频繁确认对方是否还爱自己
- 过度关注对方行为
- 容易因为小事引发大反应
- 对方可能感到窒息

**占比**：约20%人口

---

### 回避型（Avoidant）

**核心特征**：
- 强调独立，排斥依赖
- 情感表达受限
- 保持距离作为安全策略
- 难以接受亲密需求

**关系表现**：
- 遇到冲突就退缩
- 不善于表达情感需求
- 对方可能感到被冷落
- 被称为"冰山"

**占比**：约25%人口

---

### 混乱型（Disorganized）

**核心特征**：
- 既渴望又恐惧亲密
- 行为和情绪难以预测
- 可能同时表现出焦虑和回避
- 常与创伤经历相关

**关系表现**：
- 忽冷忽热
- 难以建立稳定的关系模式
- 可能伤害自己也伤害对方
- 需要专业心理干预

**占比**：约5%人口

## 风格诊断

### 快速自测（每题1-5分）

| 问题 | 1(完全不符合) - 5(完全符合) |
|-----|----------------------------|
| 1. 我很容易担心伴侣不爱我 | |
| 2. 我担心伴侣会离开我 | |
| 3. 我需要频繁确认伴侣的感情 | |
| 4. 伴侣没有及时回复时我会焦虑 | |
| 5. 我害怕被伴侣抛弃 | |
| 6. 我喜欢有私人空间 | |
| 7. 亲密让我感到不自在 | |
| 8. 我不喜欢被依赖 | |
| 9. 我倾向于保持情感距离 | |
| 10. 表达情感让我感到困难 | |

**计算方式**：
- 焦虑得分 = Q1-Q5 总分
- 回避得分 = Q6-Q10 总分

**判断**：
- 焦虑高 + 回避低 = 焦虑型
- 焦虑低 + 回避高 = 回避型
- 焦虑高 + 回避高 = 混乱型
- 焦虑低 + 回避低 = 安全型

## 组合匹配分析

### 最佳组合

| 组合 | 匹配度 | 说明 |
|-----|-------|-----|
| 安全型 + 安全型 | ★★★★★ | 最理想的组合，易于建立健康关系 |
| 安全型 + 焦虑型 | ★★★★☆ | 安全型能稳定焦虑型 |
| 安全型 + 回避型 | ★★★★☆ | 安全型能给予空间又不失亲密 |

### 中等组合

| 组合 | 匹配度 | 说明 |

| 组合 | 匹配度 | 说明 |

| 任意型 + 混乱型 | ★★☆☆☆ | 需要专业干预 |

## 适配策略

### 如果你是安全型

**优势**：你有建立健康关系的天然能力

**与焦虑型相处**：
- 给予更多情绪确认
- 主动表达爱意
- 理解对方的"粘人"不是控制
- 保持稳定和可靠

**与回避型相处**：
- 给予足够空间
- 不逼迫情感表达
- 理解对方的"冷漠"是自我保护
- 主动但不强求

---

### 如果你是焦虑型

**自我觉察**：
- 你的不安来自内在，不是对方的行为
- 频繁确认只会让关系更紧张
- 对方没有及时回应不等于不爱你

**成长方向**：
1. 建立内部安全感
2. 学习忍受不确定性
3. 发展独立的生活重心
4. 练习"延迟反应"（等24小时再表达担忧）

**话术转换**：
- ✗ "你为什么这么久不回我？"
- ✓ "我现在有点不安，但我知道这是我的问题，你能抱抱我吗？"

---

### 如果你是回避型

**自我觉察**：
- 你的距离感是一种防御机制
- 独立是好的，但过度疏离会伤害关系
- 对方需要的是连接，不是你的"完美"

**成长方向**：
1. 练习偶尔的脆弱
2. 允许自己依赖
3. 区分"真实的需要"和"被吞噬的恐惧"
4. 主动创造小规模亲密

**话术转换**：
- ✗ "我需要空间，别烦我"
- ✓ "我需要独处一下，但我爱你，晚点找你"

---

### 如果你是混乱型

**建议**：
- 寻求专业心理咨询
- 先处理个人创伤
- 建立基本的安全感后再进入关系
- 自我进化路径见 [references/self-evolution.md](references/self-evolution.md)

## 进化路径

### 从非安全型到安全型

**焦虑型的安全化路径**：

| 阶段 | 目标 | 具体练习 |
|-----|-----|---------|
| 阶段1 | 觉察焦虑 | 识别焦虑触发点，记录反应 |
| 阶段2 | 延迟反应 | 焦虑升起时，等待24小时再行动 |
| 阶段3 | 自我安抚 | 学习不依赖他人获得安全感 |
| 阶段4 | 重新诠释 | 学会把模糊信号解读为正向 |
| 阶段5 | 安全依恋 | 建立稳定的内部工作模型 |

**回避型的安全化路径**：

| 阶段 | 目标 | 具体练习 |
|-----|-----|---------|
| 阶段1 | 觉察回避 | 识别回避触发点，记录反应 |
| 阶段2 | 允许脆弱 | 每周尝试一次情感表达 |
| 阶段3 | 小步依赖 | 主动提出一个小需求 |
| 阶段4 | 接受亲密 | 允许关系中的不确定性 |
| 阶段5 | 安全依恋 | 建立信任的内部工作模型 |

## 关系中的风格协调

### 追逃陷阱破解（焦虑+回避）

**问题循环**：
```
焦虑方：需要确认 → 追问 → 回避方：感到窒息 → 更逃避 → 焦虑方：更不安 → 更追问
```

**破解方法**：

**焦虑方要做**：
1. 识别自己的焦虑触发
2. 在追问前先做3次深呼吸
3. 把精力转移到自己的事
4. 信任对方，即使对方没有回应

**回避方要做**：
1. 识别自己在逃避什么
2. 给出最低程度的回应（一条消息、一个拥抱）
3. 表达"我需要空间"时，同时表达"我在乎你"
4. 练习延迟回应而非不回应

**共同约定**：
- 焦虑方：每天最多追问1次
- 回避方：每天至少主动回应1次
- 每周固定一次深度连接时间
