# 完工驗屋重點

## 驗屋三階段

### 第一關：初驗（使用執照取得後）
- 建築師查驗通過，取得使用執照
- 建商通知可進行驗屋
- 第一次進入實際房屋檢查

### 第二關：複驗（修繕後）
- 初驗缺失修繕完成
- 買方複驗確認
- 確認所有缺失已改善

### 第三關：最終點交
- 貸款核撥後
- 現場點交鑰匙及相關文件
- 簽署交屋確認書

## 驗屋重點清單

### 結構與外觀
- [ ] 牆面是否有裂縫（尤其樑柱接合處）
- [ ] 地板是否平整（水平儀檢查）
- [ ] 磁磚是否有空心（輕敲測試）
- [ ] 窗框有無滲水（大雨後最準）
- [ ] 陽台排水孔是否暢通

### 水電
- [ ] 所有插座通電測試（可用驗電筆）
- [ ] 開關功能是否正常
- [ ] 漏電斷路器測試
- [ ] 弱電（電話/網路/電視）預留管線是否暢通
- [ ] 總開關箱線路標示是否清楚
- [ ] 水龍頭出水測試（冷熱水）
- [ ] 排水管排水測試（大量注水）
- [ ] 馬桶沖水功能
- [ ] 熱水器管線預留位置

### 門窗
- [ ] 大門開關順暢度
- [ ] 大門門鎖功能
- [ ] 窗戶氣密性
- [ ] 紗窗安裝完整

### 衛浴
- [ ] 衛浴設備品牌與合約一致
- [ ] 馬桶品牌型號
- [ ] 洗臉台/浴櫃安裝水平
- [ ] 淋浴區防水測試（地排積水測試）
- [ ] 乾溼分離玻璃門
- [ ] 抽風機運轉正常

### 廚房
- [ ] 廚具品牌與合約一致
- [ ] 抽油煙機功能
- [ ] 瓦斯爐功能及安全裝置
- [ ] 烘碗機功能
- [ ] 廚下淨水器預留管線
- [ ] 水槽排水

### 其他
- [ ] 與合約平面圖核對尺寸
- [ ] 有無滲漏水（重點檢查）
- [ ] 天花板是否平整
- [ ] 油漆/批土是否均勻
- [ ] 樓高是否與合約一致
- [ ] 停車位位置及尺寸
- [ ] 公共設施（如有）

## 常見缺失（新建案）

| 缺失類型 | 常見情況 | 要求 |
|----------|----------|------|
| 油漆瑕疵 | 不平整、有粉塵、刷痕明顯 | 批土打磨重漆 |
| 地磚空心 | 輕敲有空洞聲 | 大面積需重鋪 |
| 排水不順 | 排水坡度不足 | 重做坡度 |
| 門窗刮傷 | 安裝過程造成 | 更換或修復 |
| 尺寸不符 | 與合約平面圖有出入 | 確認是否在合理公差 |
| 插座鬆脫 | 安裝未鎖緊 | 重新安裝 |
| 矽利康未打 | 接縫處未填補 | 補打矽利康 |
| 設備缺件 | 少遙控器/零件 | 補齊 |

## 驗屋注意事項

1. **雨天驗屋最準**：漏水問題在雨天最容易發現
2. **帶齊合約**：建材設備表、平面圖對照
3. **拍照錄影**：每個缺失都要拍照+標註位置
4. **驗屋清單**：可用 home-inspection skill 輔助
5. **修繕期限**：要求建商在**15~30天內**完成
6. **交屋保留款**：**至少保留總價5%**，修繕完成再付
7. **專業驗屋**：新成屋建議找專業驗屋公司
