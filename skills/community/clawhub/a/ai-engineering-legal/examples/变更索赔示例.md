# 变更索赔示例

## 场景：地下室防水做法变更索赔

### 背景

某商业综合体项目，施工过程中发现地下水位高于勘察报告数据，原设计的SBS防水卷材方案无法满足防水要求。设计单位发出设计变更，要求增加水泥基渗透结晶型防水层，并加厚混凝土底板。

### 变更情况

- **变更指令**：2026年3月15日，设计单位发出《设计变更通知单》（编号：BG-2026-015）
- **变更内容**：增加水泥基渗透结晶型防水层，底板厚度由600mm改为800mm
- **变更影响**：新增工序导致工期增加14天

### 技能处理过程

#### 第一步：变更确认

**检查变更的合法性**：
- ✅ 变更指令为书面形式
- ✅ 设计单位具有变更权限
- ✅ 变更内容明确具体
- ✅ 现场条件确实需要变更（地下水位高于勘察数据）

**知识库依据**：《变更索赔实务指南》要求变更指令须为书面形式

#### 第二步：方案制定

**变更实施方案**：
1. 水泥基渗透结晶型防水层：涂刷厚度2mm，分2遍施工
2. 底板加厚：增加200mm混凝土，配筋相应调整
3. 分段施工：A区→B区→C区，减少工期影响

**方案报审**：方案经监理审批通过（监理批复：2026年3月18日）

#### 第三步：费用计算

**费用计算明细**：

| 项目 | 单位 | 工程量 | 单价（元） | 合价（元） | 依据 |
|------|------|--------|-----------|-----------|------|
| 水泥基渗透结晶型防水层 | ㎡ | 4,500 | 85 | 382,500 | 合同防水单价+材料价差 |
| 底板混凝土增加 | m³ | 900 | 650 | 585,000 | 合同混凝土单价 |
| 底板钢筋增加 | t | 28 | 4,800 | 134,400 | 合同钢筋单价 |
| 模板增加 | ㎡ | 450 | 45 | 20,250 | 合同模板单价 |
| 工期延误损失 | 天 | 14 | 15,000 | 210,000 | 合同约定停工损失 |
| **合计** | | | | **1,332,150** | |

**知识库依据**：《工程量清单计价标准》第9.3条（工程变更的价款调整方法）

#### 第四步：证据收集

**证据清单**：
1. ✅ 设计变更通知单（BG-2026-015）
2. ✅ 监理审批的变更方案
3. ✅ 施工日志（3月18日-4月1日）
4. ✅ 材料采购发票（水泥基防水材料）
5. ✅ 监理日志（变更施工期间）
6. ✅ 会议纪要（3月20日变更实施协调会）
7. ✅ 现场照片（施工前、中、后对比）

#### 第五步：索赔提出

**索赔报告摘要**：
```
致：[发包人]

因地下水位高于勘察报告数据导致地下室防水方案变更，
根据合同第22条（变更管理）约定，
现提交变更索赔报告如下：

1. 变更事实：设计变更通知单（BG-2026-015）
2. 费用总额：1,332,150元（详见费用计算书）
3. 工期影响：14天（详见工期调整计算）
4. 证据材料：附后（共计7类23份文件）

请于收到后28日内予以审核确认。
```

### 技能产出

**完整索赔包**：
1. 索赔报告正文
2. 费用计算书
3. 工期调整计算
4. 证据材料汇编（按时间顺序排列）
5. 法律依据汇编

### 关键要点

1. **时效性**：变更发生后立即启动索赔流程
2. **完整性**：证据、计算、依据缺一不可
3. **逻辑性**：变更→事实→费用→索赔，逻辑清晰
4. **合规性**：所有计算和依据都出自合同约定或国家标准
