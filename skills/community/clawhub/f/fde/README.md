# FDE — 前沿部署工程师全流程助手

[![Skill](https://img.shields.io/badge/WorkBuddy-Skill-blue)](https://www.codebuddy.cn/docs/workbuddy/Overview)

AI时代最火岗位——FDE（Forward Deployed Engineer，前沿部署工程师）的技能化实现。覆盖审计（Audit）→ 评估（Evals）→ 部署（Deployment）三大阶段，融合 Palantir Echo-Delta 模式，提供从客户现场需求挖掘到产品化抽象的完整方法论。

## 技能结构

```
fde/
├── SKILL.md                      # 核心方法论（三阶段工作流）
├── references/
│   ├── fde-framework.md          # 审计清单、评估框架、部署检查表
│   ├── fde-toolkit.md            # RAG搭建、模型选型、系统集成、安全合规
│   └── fde-cases.md              # 制造/农业/金融/医疗/零售/能源 6大行业案例
├── scripts/
│   └── client_audit.py           # 7行业客户审计问卷生成工具
└── README.md
```

## 核心能力

- **Audit 审计**：利益相关者访谈、业务流程映射、数据资产盘点
- **Evals 评估**：技术方案设计、POC验证、ROI量化
- **Deployment 部署**：快速原型开发、系统集成、迭代优化
- **抽象闭环**：现场经验 → 平台能力 → 飞轮加速
