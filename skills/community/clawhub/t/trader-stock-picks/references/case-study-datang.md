### ⚠️ 免责声明：这份案例既是成功案例，也是失败案例

**成功部分：** 识别出了大唐发电经历了13个月的长周期主力建仓、三段式洗盘，这一点是对。

**失败部分：** 在三连板后，基于「融资余额下降=主力出货」的教条判断短期危险，但实际走势是**继续到7连板、市值重返千亿**。每一步都预测反了。

**这份案例必须同时保留正确和错误的分析，因为错误才是最有价值的部分。**

### 错在哪里——深度诊断

#### 错误1：没有区分庄股 vs 估值重估场景

最初分析中，我把大唐发电归类为「央企庄股」，使用五维模型评分（27/50分，结论：短期危险）。

**但大唐发电实际上是「估值重估」：**
- 催化剂：「全国首个大规模算电协同项目」—— 国家级叙事，非普通题材
- 基本面：净利润从亏损→2024年45亿→2025年74亿（连续翻倍）
- 估值：从PB<1的极度低估状态起涨，有重估空间
- 机构：全覆盖+买入评级+密集上调目标价

**修正后的评分（使用估值重估框架）：**

| 维度 | 评分 | 说明 |
|:---:|:----:|:----|
| 催化剂量级 | 10/10 | 全国首个+AI算力+东数西算，三重共振 |
| 基本面质变 | 9/10 | 利润3年增长18倍，趋势明确 |
| 估值空间 | 9/10 | 从PB<1起涨，对标同类有3-5倍空间 |
| 机构共识 | 8/10 | 全覆盖+上调 |
| 筹码状态 | 6/10 | 起涨前已充分洗盘 |
| 散户普及率 | 7/10 | 前5板时大部分散户还没反应过来 |
| **总分** | **49/60** | **估值重估行情，应看多** |

#### 错误2：融资余额信号解读错误

原分析：「5月6日涨停融资余额8.84亿→5月7日降到7.87亿，涨停中融资降=主力出货」

**事实：** 5月7日缩量涨停，融资余额下降是机械现象——因为没人买得了股票，融资买入为零。这不反映主力意图。

**正确解读：** 融资余额下降本身没信号意义。真正要看的是**连续涨停中的放量分歧日**——是否有巨量换手后还能封回。（第4-5板如果有分歧放量后回封，反而是加速确认信号。）

#### 错误3：认为三连板是出货尾声

三板出货的逻辑前提是「三板已经完成了从吸筹到派发的完整周期」。

**大唐发电在5/8时的情况：**
- 起涨前横盘13个月，涨幅几乎为零 → 没被充分炒过
- 催化剂（算电协同项目5/2投运）刚落地一周 → 故事刚开始讲
- 机构研报才开始密集发布 → 信息在扩散期
- 利润从4亿亏损到74亿的拐点刚被市场充分认知 → 重估刚开始

**符合「加速板」全部条件，完全不符合「出货板」。**

#### 错误4：没有每日动态重评

5月8日分析完后，第二天（5/9）如果股价没有回调而是继续涨，**正确的反应应该是：**
- 「我的假设被事实否定了，需要重新评估」
- 「昨天判断出手为时过早，这波行情比我想象的强」
- 「重新分析催化剂和资金面，看看是否该修正方向」

**而不是**什么也不做，等用户来告诉我错了。

### 总结：这个案例改变了什么

| 原框架结论 | 修正后 |
|:----------:|:------:|
| 三连板+融资降=出货 | **先分场景：庄股还是估值重估** |
| 融资余额连续下降=洗盘末期或主力出货 | **缩量涨停中的融资降=机械现象，无信号意义** |
| 技术面是唯一判断依据 | **催化剂量级决定一切，国家级叙事压倒技术信号** |
| 一次性打分后不再更新 | **每日重评，被事实否定后立刻修正** |
| 推荐时给确定性建议 | **无法判断时说「无法判断」，宁缺勿乱** |
