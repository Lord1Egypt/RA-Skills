# 利弗莫尔 · 趋势投资框架

> 本文件核心方法论由女娲(Nuwa Skill)蒸馏提取，并针对指数投资场景做了应用适配。

## 大师简介
杰西·利弗莫尔，20世纪初传奇交易员，被称为"华尔街巨熊"。在1929年大崩盘中做空获利1亿美元（相当于今天约20亿美元）。核心理念：市场由人的情绪驱动，价格本身包含了所有信息，趋势一旦确立就会延续。著作《股票大作手回忆录》是趋势交易的圣经级读物。

注意：利弗莫尔最终因过度交易和情绪失控而破产，他的方法论有强大的分析价值，但他的人生结局也是对风险管理重要性的警示。

## 适用场景
- 市场情绪极端时（恐慌/狂热）
- 短期热点/动量板块分析
- 用户关注"走势/趋势/最近方向"
- 板块放量突破或放量下跌
- 市场转折点判断

## 核心心智模型

### 模型1：最小阻力线 (The Line of Least Resistance)
- 价格像水一样，总是沿着阻力最小的方向流动。在多头市场中，阻力最小的方向就是上涨
- 判断标准：市场在消化了利好或利空消息后，价格是怎样反应的？出利好不涨→向上阻力大；出利空不跌→向下阻力大
- 不预测市场该怎么走，而是观察市场正在怎么走。只在确认了方向后才顺势开仓
- **指数应用**：观察宽基指数对重大政策/数据的反应方向——降准后指数不涨，说明上方压力大；关税加码后指数不跌，说明下方有支撑

### 模型2：关键点 (Pivotal Points)
- 不要在价格波动的任何时候都进场，必须耐心等待价格到达能够确认趋势的"心理关键点"
- 两类关键点：突破点（行情突破前期重要高低点）和反转点（趋势发生反转的确切位置）
- 在关键点被突破、且伴随成交量放大时，果断行动。到达关键点但未能突破→警惕趋势衰竭
- **指数应用**：宽基指数的整数关口（如3000点、5000点）、前期高低点、年线都是关键价位

### 模型3：坐得住的哲学 (Sitting Tight)
- 赚大钱不是靠买卖，而是靠等待
- 只要走势依然符合预期，大盘的最小阻力线没有改变，就绝对不要因为一点小利润而急于平仓
- 买入后，如果没有出现明确的卖出信号（如跌破关键点、趋势反转），就死死抱住仓位
- **指数应用**：指数处于上升趋势中，不要因为单日回调就慌忙离场；反之亦然

### 模型4：自上而下的顺势 (Top-Down Trend)
- 不要在熊市里做多，不要在牛市里做空
- 判断路径：大盘趋势 → 板块趋势 → 领头羊个股趋势
- 只有当大盘趋势向上时才做多，并且只买入当前最热门板块中的领头羊
- **指数应用**：宽基指数定方向，行业指数找板块，板块内龙头股定强弱

## 决策启发式

1. **试探与金字塔加仓**：永远不要一次性建仓。先买入计划仓位的20%作为试探；盈利→说明判断正确，继续加仓；亏损→说明判断错误，绝不加仓
2. **截断亏损，让利润奔跑**：买入后立即设置心理止损位（通常不超过10%）。一旦触及，无条件平仓。盈利时绝不轻易设定止盈目标
3. **领头羊法则**：如果你在领头羊（龙头股）上都赚不到钱，你就不可能在其他股票上赚钱。只交易最活跃、弹性最大的领头羊
4. **警惕"危险信号"**：当一只一路高歌猛进的股票突然出现单日放量大跌，且无法迅速收复失地时，这通常是趋势终结的危险信号

## 核心判断工具（指数适配）

### 量价关系
- 放量上涨 → 趋势健康，多方有力
- 缩量上涨 → 上涨动力减弱，可能见顶
- 放量下跌 → 空方占优，趋势可能反转
- 缩量下跌 → 抛压减弱，可能接近底部

### 市场情绪温度
- 极度恐慌（恐慌指数飙升、融资余额大幅下降、媒体集体看空）→ 可能接近底部
- 极度贪婪（新开户数暴增、融资余额创新高、散户蜂拥入场）→ 可能接近顶部
- 核心逻辑：市场在大多数人做出同一方向判断时，往往即将反转

### 领头羊原则（指数版）
- 每一轮上涨都有领头板块，领头板块里有领头股
- 领头羊开始走弱 → 整个板块/市场可能即将转弱
- 应用于指数：观察板块内权重股的表现是否开始分化

## 盲区（系统性忽略）
- **长期价值**：利弗莫尔不关心企业内在价值，只看价格行为——与巴菲特/段永平体系完全对立
- **基本面分析**：不看财报、不看PE，认为"价格已经包含了一切"
- **分散化**：他倾向于集中下注，与达利欧的全天候逻辑完全相反
- **风险管理纪律**：他理论上知道止损的重要性，但实践中多次失败——一生四次破产证明了体系极其依赖铁血纪律，一旦情绪失控并在趋势逆转时滥用高杠杆，将面临毁灭性打击
- **震荡市**：在没有明确趋势的市场中，最小阻力线频繁切换，容易导致反复被打损

## 反模式（绝不会做的事）
- **绝不向下摊平**：如果一笔交易亏损了，说明你错了。在错误的方向上继续加仓是投机者最致命的愚蠢
- **绝不频繁交易**：一年中只有少数几次真正属于你的大机会。为了交易而交易是破产的捷径
- **绝不听信内幕消息**：所有的内幕消息都是为了让你接盘。只相信盘面的价格和成交量事实
- **绝不与市场争辩**：如果市场走势与你的判断相反，市场永远是对的，你永远是错的

## 分析输出要求
用利弗莫尔框架分析时，重点回答：
1. 当前趋势方向是什么？有没有反转信号？
2. 量价配合怎么样？
3. 市场情绪在什么温度（恐慌/中性/贪婪）？

**重要提醒**：利弗莫尔框架偏短期，使用时必须提醒用户短期信号不等于长期判断，趋势交易需要严格纪律。

## 典型分析示例
```
科创板最近两周连续放量上涨，这是个值得注意的信号。

从趋势角度看，量价配合健康——不是缩量拉升那种虚涨。
市场情绪从之前的悲观逐步回暖，但还没到全民疯狂的程度。
领头羊（几个AI算力权重股）还在创新高，没有走弱迹象。

短期趋势是向上的，但请注意这只是趋势信号，不是长期判断。
如果量能开始萎缩、领头羊走弱，就该重新评估了。

风险点：趋势交易最怕的是追在尾声。当所有人都觉得"这次不一样"的时候，往往就快到头了。
```
