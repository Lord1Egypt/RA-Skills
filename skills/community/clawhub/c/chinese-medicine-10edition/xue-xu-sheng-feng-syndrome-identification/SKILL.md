# Xue Xu Sheng Feng Syndrome Identification

## When to Use
当患者符合以下触发条件时启用本技能：
> IF 患者出现手足震颤、肌肉眴动、肢体麻木、眩晕耳鸣、面色无华、爪甲不荣、舌淡苔白、脉细 THEN 考虑血虚生风证

本技能不适用于实热动风或外感风邪所致的震颤麻木。

## Execution Steps

1. **确认病因**：判断是否存在失血过多或久病导致的血虚状态。
2. **分析病机**：
   - 肝血不足，不能上荣于头面 → 眩晕耳鸣、面色无华、舌质淡；
   - 筋脉失去营血濡养 → 爪甲不荣、关节拘急不利；
   - 血虚动风 → 肢体麻木、筋挛、肌肉眴动、手足震颤；
   - 血少则脉道不充盈 → 脉细。
3. **明确核心病机**：血虚不能濡养筋脉，虚风内动。
4. **鉴别要点**：症状以虚象为主（如面色无华、舌淡、脉细），无发热、口苦、苔黄等实热征象。

## Output
确诊为血虚生风证，需以养血柔肝、熄风止痉为治则。
