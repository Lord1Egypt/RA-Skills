## 融资阶段评估框架

### 1. 融资阶段匹配

```
【企业发展阶段】

种子期（0-1年）
融资需求：50-500万
资金用途：产品开发
估值逻辑：创始人+idea

天使期（1-3年）
融资需求：500-2000万
资金用途：MVP验证
估值逻辑：用户/产品验证

Pre-A/A轮（2-4年）
融资需求：2000万-1亿
资金用途：规模化
估值逻辑：GMV/收入/用户

B轮（3-5年）
融资需求：1-5亿
资金用途：市场扩张
估值逻辑：收入/市场份额

C轮及以后（5年+）
融资需求：5亿+
资金用途：行业整合/上市
估值逻辑：收入/利润

【阶段判断】
□ 你的企业处于哪个阶段？
□ 核心指标是否达到阶段标准？
□ 下一步应该做什么？
```

### 2. 估值方法

```
【估值方法选择】

收入法（适合成熟企业）
估值 = 年收入 × 市销率(P/S)
- SaaS：8-15x
- 电商：1-3x
- 消费：3-8x

利润法（适合盈利企业）
估值 = 年利润 × 市盈率(P/E)
- 传统行业：10-20x
- 成长企业：20-40x
- 高增长：40x+

用户法（适合早期）
估值 = 用户数 × 单用户价值
- 社交：$50-200/人
- 工具：$20-50/人
- 电商：$30-100/人

GMV法（适合平台）
估值 = GMV × 币值系数
- 电商：0.3-1x
- O2O：0.2-0.5x

【估值谈判】
□ 估值不是越高越好
□ 稀释比例要控制
□ 投资条款要关注
□ 资源匹配要评估
```

### 3. 投资人选择

```
【投资人类型】

财务投资人
- 目的：财务回报
- 关注：增长/退出
- 介入：适度
- 典型：红杉/高瓴/IDG

战略投资人
- 目的：战略协同
- 关注：协同价值
- 介入：深度
- 典型：腾讯/阿里/字节

产业资本
- 目的：产业布局
- 关注：产业链完善
- 介入：非常深度
- 典型：宁德/比亚迪

【选择考量】
□ 需要钱还是需要资源？
□ 愿意付出多少控制权？
□ 未来是否可能竞争？
□ 投资人能带来什么增值？
```

## 融资流程

### 融资准备阶段

```
【材料准备】

1. BP（商业计划书）
   - 执行摘要：1页
   - 问题与方案：1-2页
   - 市场分析：1-2页
   - 商业模式：1-2页
   - 团队介绍：1页
   - 融资计划：1页

2. 数据包
   - 历史财务数据
   - 关键运营指标
   - 用户数据
   - 市场数据

3. 法务文件
   - 公司章程
   - 股权结构
   - 知识产权
   - 重大合同

【材料检查清单】
□ 数据是否自洽
□ 假设是否合理
□ 风险是否充分披露
□ 机会是否清晰呈现
```

### 路演阶段

```
【路演结构】

5分钟版本（快速介绍）
1. 我们是谁？（30秒）
2. 解决什么问题？（1分钟）
3. 怎么解决？（1分钟）
4. 为什么能成？（2分钟）
5. 需要什么？（1分钟）

15分钟版本（深度路演）
1. 背景与问题（2分钟）
2. 解决方案（3分钟）
3. 市场机会（2分钟）
4. 商业模式（3分钟）
5. 竞争分析（2分钟）
6. 团队介绍（1分钟）
7. 融资计划（2分钟）

【路演技巧】
□ 讲用户故事，不要讲技术细节
□ 数据要可视化
□ 自信但不要傲慢
□ 准备好Q&A
□ 练习到熟练
```

### 尽调与交割

```
【尽调配合】

业务尽调
- 提供完整数据
- 安排核心人员访谈
- 开放客户/供应商访谈

财务尽调
- 规范财务报表
- 准备好银行流水
- 解释异常数据

法务尽调
- 梳理法律风险
- 准备证照文件
- 解释历史问题

【交割要点】
□ 股权协议签署
□ 工商变更
□ 资金到账
□ 投资条款清单
```

## 常见问题速查

| 问题 | 建议方案 | 关键考量 |
|-----|---------|---------|
| 什么时候融资 | 账上钱够12-18个月 | 不要等钱紧了 |
| 要不要估值高 | 合适最重要 | 高估值=严条款 |
| 对赌要不要签 | 尽量避免 | 对赌风险大 |
| 多家TS怎么选 | 资源>估值>条款 | 钱不是最重要 |
| 稀释多少合理 | 15-25%一轮 | 避免过度稀释 |
