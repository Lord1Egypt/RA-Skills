# 课程PPT生成技能 - 使用指南

## 快速开始

### 基本用法

用户只需提供：
- 课程主题/标题
- 学科领域（可选）
- 课程类型（可选）

### 示例对话

```
老师：制作一份深度学习的课程PPT
AI：正在为您生成PPT，请稍候...

老师：帮我做一个数据预处理的实验指导课件
AI：好的，正在检索数据预处理实验相关内容...
```

## 课程类型说明

| 类型 | 说明 | PPT结构 |
|------|------|---------|
| 理论课 | 概念原理讲解 | 导入→讲解→示例→总结 |
| 实验课 | 动手实践操作 | 目的→环境→步骤→结果 |
| 复习课 | 知识巩固 | 框架→要点→例题→练习 |
| 新课 | 新知识学习 | 案例→新知→练习→作业 |

## 样式规范

主色：#1E3A5F（深蓝）
辅色：#FFFFFF（白色）
强调：#4A90D9（亮蓝）

### 适用场景

- 理工科课程

- 学术报告

## 注意事项

1. **代码类课程**请注明编程语言版本
2. **数学公式**建议提供LaTeX格式或手写公式图片
3. **图片素材**建议提前准备或由AI自动生成无版权配图
4. **每页内容**要充实饱满，不空洞
5. **末尾必须**添加AI生成标识

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 生成失败 | 检查npm依赖是否安装 |
| 图片缺失 | 使用免费图库自动补图 |
| 格式错乱 | 调整文字大小或减少内容 |
