# 新生儿RDS的肺表面活性物质治疗

## 何时使用
- 患儿为早产儿、RDS患儿或足月剖宫产RDS患儿；
- 已确诊或高度疑似RDS；
- 满足以下任一条件：
  - nCPAP压力 ≥6cmH₂O 且 FiO₂ >0.30（轻度进展）；
  - 出现中重度RDS临床表现（如严重低氧、呼吸费力、胸片典型改变）。

## 如何执行

### 1. 判断应用指征
- **轻度进展**：在nCPAP支持下，若压力≥6cmH₂O且FiO₂>0.30，可给予PS；
- **中重度RDS**：立即气管插管并给予PS。

### 2. 确定使用时机
- **极早产儿**（尤其母亲未接受产前糖皮质激素者）：应在产房内尽早使用；
- **已确诊RDS**：越早应用效果越好；
- **病情持续进展**（如无法脱氧、需升级呼吸支持）：可追加第二剂或第三剂PS。

### 3. 选择剂量方案
- **首剂**：推荐100–200 mg/kg（研究显示200 mg/kg疗效更优）；
- **追加剂量**：通常为100 mg/kg。

### 4. 实施给药方法
- **传统法**：经气管插管缓慢注入稀释后的PS（干粉制剂需先用生理盐水稀释）；
- **微创法**（LISA/MIST）：在持续nCPAP支持下，通过细导管将PS注入气管，避免气管插管。

### 5. 注意事项
- PS产品多为猪肺或小牛肺提取的天然型，使用前确认产品类型与储存条件；
- 给药后密切监测氧合、通气状态及血气分析变化；
- 谨慎评估感染风险，未排除感染前需权衡利弊；
- 足月儿RDS对PS反应可能不如早产儿，需个体化评估。

> 提示：联合高频振荡通气（HFOV）或一氧化氮（NO）吸入可能增强疗效，但需更多循证依据支持。
