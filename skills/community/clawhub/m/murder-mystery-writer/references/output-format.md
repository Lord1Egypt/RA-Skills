# 输出格式指南

## 剧本杀成品结构

一个完整的剧本杀成品包含以下部分：

### 1. 角色剧本（每人一份，按幕分文档）

每个玩家拿到的个人剧本，按幕分文件：

**第一幕文档（案发前）：**
```markdown
# 《剧本名称》
## 角色名 - 角色简介

### 你的故事
（200-400字的背景故事）

### 你的秘密
（不能让所有人知道的事）

### 你的时间线（案发前）
- 19:00：你在做什么
- 19:15：你看到了什么
- 19:30：你做了什么
- ...

### 你的任务
1. 主线任务
2. 支线任务

### 你知道的线索
（其他玩家不知道的信息）
```

**第二幕文档（案发时/调查阶段）：**
```markdown
# 《剧本名称》
## 角色名 - 第二幕

### 案发时的时间线
- 20:00：你在做什么
- 20:10：你看到了什么
- 20:15：停电时你在哪
- 20:20：你做了什么
- ...

### 你的新发现
（本幕新揭露的信息）

### 你的困境
（被质疑或需要解释的事情）

### 你的选择
（本幕需要做的决定）
```

**第三幕文档（如有）：**
```markdown
# 《剧本名称》
## 角色名 - 第三幕

### 你的最终抉择
### 你的结局走向
```

### 2. DM手册（主持人专用）

```markdown
# 《剧本名称》DM手册

## 剧本信息
- 类型：
- 人数：
- 时长：
- 难度：

## 真相复盘
（完整的故事真相）

## 时间线
（所有角色的完整时间线表格）

## 线索发放指南
| 轮次 | 线索编号 | 发放条件 | 内容 |
|------|---------|---------|------|
| 第一轮 | C001 | 搜证 | ... |
| 第二轮 | C002 | 对话 | ... |

## 投票指南
- 投票时间：
- 投票方式：
- 结局判定：

## 常见问题解答
（玩家可能问的问题及答案）

## 节奏控制建议
（每阶段的时间建议和注意事项）
```

### 3. 线索卡（可打印）

```markdown
# 线索卡 - C001

**类型**：实物线索
**位置**：书房抽屉

**内容**：
一封未寄出的信，内容为：
"如果你敢公布账目，必死无疑。"
落款模糊，只能看清"X先生"三个字。

**指向**：死者因财务问题被威胁
```

### 4. 公共信息卡

```markdown
# 公共信息 - 案件概况

**死者**：林老爷，55岁，林氏集团董事长
**死亡时间**：10月15日晚19:45左右
**死亡地点**：林宅书房
**死因**：中毒（氰化物）
**发现者**：管家王叔

**现场情况**：
- 书房门从内反锁
- 窗户紧闭
- 桌上有半杯未喝完的茶
- 死者手表停在19:47
```

### 5. 角色关系图（可选）

```markdown
角色关系图：

林少爷 ──── 未婚妻 ──── 张小姐
    │                      │
    │                      │
  兄妹                   仇人
    │                      │
    │                      │
林小姐 ──── 暗恋 ──── 陈律师
    │
    │
  主仆
    │
    │
王管家 ──── 知晓秘密 ──── 顾记者
```

## 输出格式示例

### 角色剧本示例

```markdown
# 《钟楼谜案》
## 林少爷 - 林氏集团继承人

### 你的故事
你是林老爷的独子，从小被寄予厚望。表面上你是纨绔子弟，实际上你在暗中调查父亲的商业秘密。你发现父亲的公司涉及非法交易，你决定在今晚的慈善晚宴上与父亲摊牌。

但你没想到，今晚会发生命案。

### 你的秘密
你在调查父亲时，发现他与日本商人有秘密交易。你怀疑这些交易涉及走私。你拿到了一份交易记录，但还没来得及看。

### 你的时间线
- 18:30：你到达酒店，在大堂与张小姐聊天
- 19:00：晚宴开始，你在大厅敬酒
- 19:15：你去洗手间，在走廊遇到陈律师
- 19:30：你去找父亲，但书房门锁着，你敲门没人应
- 19:45：你回到大厅，听到钟声
- 20:00：管家宣布发现父亲死亡

### 你的任务
1. 主线：找出杀死父亲的凶手
2. 支线：隐藏你调查父亲商业秘密的事
3. 支线：保护张小姐（你怀疑她可能被卷入）

### 你知道的线索
- 父亲书房的保险柜密码是你的生日
- 你看到陈律师在19:15从书房方向走来
- 张小姐最近一直想退婚，但父亲不同意
```

## 格式检查清单

完成输出后，检查：
- [ ] 每个角色剧本字数相近（相差不超过20%）
- [ ] 每个角色都有明确的任务
- [ ] 时间线表格完整，无遗漏
- [ ] 线索卡编号与DM手册一致
- [ ] 公共信息卡包含所有玩家需要知道的基础信息
- [ ] DM手册有完整的真相复盘
- [ ] 结局判定规则清晰
