# cytotoxic-antitumor-drug-classification-and-cycle-specific-regimen-design

## 适用场景
当需要为以下肿瘤制定化疗方案时使用本技能：
- 实体瘤（通常增长缓慢，GF低）
- 急性白血病（通常增长迅速，GF高）
- 其他增殖性肿瘤
- 涉及静止期（G₀期）肿瘤细胞的治疗策略

**不适用于**非增殖性良性肿瘤或免疫功能严重低下未评估调整的患者。

## 执行步骤

1. **判断肿瘤生长特性**
   - 明确肿瘤生长比率（GF）
   - 评估肿瘤细胞周期分布（各时相比例）
   - 确认患者耐受性与免疫状态

2. **区分药物类别**
   - **细胞周期非特异性药物（CCNSA）**：作用于G₁、S、G₂、M及G₀期细胞；代表药物包括烷化剂、抗肿瘤抗生素、铂类配合物；疗效呈**剂量依赖性**。
   - **细胞周期特异性药物（CCSA）**：仅作用于特定周期时相（如S期：抗代谢药；M期：长春碱类）；对G₀期无效；疗效呈**时间依赖性**。

3. **制定联合用药策略**
   - **低GF肿瘤（如多数实体瘤）**：先用CCNSA缩小瘤体并招募G₀期细胞进入增殖周期，再用CCSA杀灭活跃增殖细胞。
   - **高GF肿瘤（如急性白血病）**：先用CCSA清除大量增殖细胞，再用CCNSA清除其他时相细胞；待G₀期细胞进入周期后重复治疗。

4. **考虑同步化策略（可选）**
   - 使用CCSA将肿瘤细胞阻滞于某一时相（如G₁期）
   - 待药物清除后，细胞同步进入下一时相
   - 再给予作用于该时相的药物以提高杀伤效率

5. **遵循给药原则**
   - 采用**间歇给药**方式，避免持续大剂量导致免疫功能过度抑制
   - 遵循一级动力学原则：每次治疗仅能杀灭一定比例肿瘤细胞，需多周期治疗

> 注意：任何方案均需在患者毒性耐受范围内实施，并动态评估疗效与不良反应。
