## Description: <br>
Build, harden, and scale Discord servers with a practical admin playbook for security baselines, role and permission architecture, onboarding, moderation operations, engagement systems, and analytics-driven iteration. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>


## Use Case: <br>
External community managers, moderators, and Discord server owners use this skill to design new servers, audit existing setups, prioritize safety and engagement fixes, and produce execution checklists. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Role, permission, verification, and AutoMod recommendations can change member access or moderation behavior if applied without review. <br>
Mitigation: Have a trusted server owner review proposed Discord settings changes before applying them. <br>
Risk: The skill provides administrative guidance but does not directly inspect a live Discord server. <br>
Mitigation: Validate audit findings against current server settings, Discord Insights, and moderator observations before implementation. <br>
