# 🎤 阶段 3：面试评估

## 设计原则

> **结构化 BARS 评分 + 反偏见自检 + 反向论证 = 反偏见招聘**
>
> 禁止主观打分，每个分数都要有行为锚定。

---

## 3.1 评估流程总览

```
面试结束
 ↓
独立评分（24h 内提交，禁止互通）
 ↓
反偏见自检（8 项）
 ↓
反向论证（强制 3 个不推荐理由）
 ↓
提交至 Calibration 会议（见包 7）
 ↓
出综合评估报告
```

---

## 3.2 BARS 8 维度评分卡（核心）

> **BARS** = Behaviorally Anchored Rating Scale
> 每个分数都有**具体可观察的行为描述**，杜绝"远超预期"这种空话

- 评分范围：1-5 分（整数 / 0.5 增量）
- 评分必须基于面试中观察到的**具体行为证据**
- 每个评分必须附**面试原话或具体场景**
- 不允许凭"感觉"打分

---

### 维度 1：过往业绩真实性

| 分数 | 行为锚点（必须可观察） |
|------|----------------------|
| **5** | 完整回答 A/B/C/D 四维度追问，数据来源可追溯，主动承认外部因素影响 |
| **4** | A/B/C 三维度清晰，反事实有思考但不完整 |
| **3** | 能讲清楚做了什么和结果，但归因模糊，数据说不出基线 |
| **2** | 只能讲故事和结论，追问归因开始回避或转移话题 |
| **1** | 明显夸大或回避，关键数据"想不起来"，与简历描述不一致 |

**🔴 红灯**：候选人说"提升 30%"，但说不出基线 / 统计周期 / 归因占比 → 直接 ≤2 分

---

### 维度 2：问题解决能力（协作模拟题）

| 分数 | 行为锚点 |
|------|----------|
| **5** | 主动澄清问题边界和约束，提出 2+ 方案并说明 trade-off，能识别隐藏假设和风险 |
| **4** | 准确理解问题，提出 1-2 个合理方案，能说取舍但深度不够 |
| **3** | 理解题目后给出 1 个方案，逻辑通顺但缺少备选，不主动识别风险 |
| **2** | 跳过问题定义直接给方案，方案过于通用或脱离实际 |
| **1** | 答非所问 / 套用模板 / 完全脱离实际场景 |

**🔴 红灯**：不问任何澄清问题就开始给方案 → ≤2 分

---

### 维度 3：归因合理性

| 分数 | 行为锚点 |
|------|----------|
| **5** | 主动量化个人贡献占比，区分自己做什么 vs 团队做什么 vs 外部因素，承认运气/时机的作用 |
| **4** | 追问后能清晰归因，但不主动量化贡献占比 |
| **3** | 用"我们"代替"我"，追问后才区分个人贡献 |
| **2** | 持续用集体语言，无法说清自己具体做了什么 |
| **1** | 把团队成果全归功于自己 OR 把个人失误全推给外部 |

**🔴 红灯**：此项 ≤2 分时，业绩真实性自动降一档

---

### 维度 4：学习与反思

| 分数 | 行为锚点 |
|------|----------|
| **5** | 能讲出 2+ 个失败案例，每个都有具体教训，能说出"如果重做会改什么" |
| **4** | 能讲出 1 个真实失败，有反思有教训，但后续应用案例不清晰 |
| **3** | 在追问下勉强说出"失败"，但更像是"挑战"，反思流于表面 |
| **2** | 把失败包装成成功，不承认自己的责任 |
| **1** | 拒绝承认有失败 / "都是别人的问题" / 完美主义型回避 |

**🔴 红灯**：声称"没有失败"或"想不起来" → 直接 1 分

---

### 维度 5：沟通表达

| 分数 | 行为锚点 |
|------|----------|
| **5** | 回答结构化（先结论后论证），能根据对方反应调整深度，2 分钟自述能精准抓住 3 个核心亮点 |
| **4** | 表达清晰有逻辑，但偶尔过长或过简 |
| **3** | 表达基本清楚，但有冗余 / 跳跃，需要面试官引导才能聚焦 |
| **2** | 流水账式叙述，缺乏重点，听 5 分钟仍不清楚要点 |
| **1** | 答非所问 / 跑题 / 表达混乱无法继续 |

> 高级岗位对此维度要求 ≥4，表达即影响力

---

### 维度 6：动机与稳定性

| 分数 | 行为锚点 |
|------|----------|
| **5** | 离职原因具体且与发展诉求一致，对新岗位有具体期望，能说出"为什么是这家公司而不是别家" |
| **4** | 离职原因清晰，对新岗位期望明确但研究稍浅 |
| **3** | 离职原因合理但偏外部归因，对新岗位期望泛化 |
| **2** | 离职原因模糊或前后矛盾，对新岗位没有具体期望 |
| **1** | 离职原因明显回避真实因素 / 频繁跳槽且每次原因都是外部 |

**🔴 红灯**：3 年内 ≥3 次跳槽 + 每次离职原因都推给外部 → ≤2 分

---

### 维度 7：文化匹配

| 分数 | 行为锚点（需结合公司价值观调整） |
|------|--------------------------------|
| **5** | 能举出符合公司价值观的具体行为案例，并能反思自己不擅长的部分 |
| **4** | 行为案例符合价值观，但反思深度一般 |
| **3** | 表达上认同价值观，但举不出具体行为证据 |
| **2** | 价值观表述与公司有明显冲突 |
| **1** | 行为模式与公司核心价值观背离 |

---

### 维度 8：岗位理解深度

| 分数 | 行为锚点 |
|------|----------|
| **5** | 提问质量高，能问到岗位真实痛点，对岗位 90 天 / 6 个月 / 12 个月目标有自己的假设并能说出依据 |
| **4** | 对岗位有清晰理解，能问出有质量的问题，但 12 个月规划深度一般 |
| **3** | 基本理解岗位职责，提问偏福利 / 团队规模等表层 |
| **2** | 对岗位理解停留在 JD 字面，提问无实质内容 |
| **1** | 明显未做准备 / 误解岗位性质 |

## 3.5 反偏见自检清单（8 项升级版）

> 每位面试官提交评分前**必须勾选完成**

```
☐ 1. 证据基础检查：评分是否基于行为证据，而非"感觉"？
☐ 2. Rubric 一致性检查：是否对所有候选人用同一套 BARS Rubric？
☐ 3. 禁区因素排除：是否排除了姓名/学校/性别/年龄/户籍的影响？
☐ 4. 反向论证检查：是否给出了"不推荐的 3 个理由"作为反向论证？
☐ 5. 表象 vs 实质检查：是否区分了"口才好"和"真有能力"？
☐ 6. Confirmation Bias 检查：是否只找了支持第一印象的证据？
☐ 7. Recency Bias 检查：最近聊到的话题是否被过度加权？
☐ 8. Similarity Bias 检查：是否因为候选人和自己背景相似而偏好？

我确认以上 8 项均已完成，评分基于结构化证据。
面试官签字：___________ 日期：___________
```

```markdown
证据：[面试中的具体行为]
对岗位的影响：[潜在风险]

理由 2：[...]
理由 3：[...]

【反思】
列出这 3 个理由后，你对推荐决定的信心是否有变化？
☐ 增强：经过反向论证仍推荐，决定更坚定
☐ 维持：能识别风险但风险可控
☐ 减弱：风险大于优势，重新评估推荐结论
```

## 3.8 阶段 3 输出前自检 Checklist

```

☐ 是否使用了 BARS 评分卡的具体行为锚点？
☐ 每个评分是否附了面试原话 / 具体场景？
☐ 是否完成了 A/B/C/D 追问深度评估？
☐ 是否完成了 PEARL 完整度评估？
☐ 是否输出了红绿灯三色信号？
☐ 是否检查了一票否决项？
☐ 是否完成了 8 项反偏见自检？
☐ 是否完成了"3 个不推荐理由"反向论证？
☐ 是否标注了本面试官评估的局限性？

```
