# 五色主病详细病机对照表

## 青色
- **面、唇、爪甲青白** → 寒证
- **青黑晦暗** → 阳虚
- **青紫** → 阳气大衰
- **鼻头色青** → 腹中疼痛
- **面色青 + 喜热饮 + 尿清长/腹满下利** → 腹中寒痛
- **面色乍青乍白 + 腹痛泛吐清水** → 虫积腹痛
- **口唇青灰** → 心阳不振、心血瘀阻
- **小儿眉间、鼻柱、唇周青色** → 惊风

## 赤色
- **外感温热** → 面赤发热
- **实热证** → 面赤、高热、口渴、便秘
- **虚热证** → 两颧嫩红或潮红（午后明显）
- **虚损劳瘵** → 两颧潮红、午后潮热、五心烦热、盗汗

## 黄色
- **淡黄晦暗无泽（萎黄）** → 脾胃气虚
- **面目虚浮淡黄（黄肿）** → 脾虚湿盛
- **面目一身俱黄** → 黄疸：
  - **鲜明如橘皮（阳黄）** → 湿热熏蒸
  - **晦暗如烟熏（阴黄）** → 寒湿郁阻
- **小儿遍体皆黄** → 胎黄
- **小儿面色青黄或乍黄乍白** → 疳积
- **黄色渐明润** → 胃气渐复，好转
- **黄色转枯** → 胃气衰败，预后不良

## 白色
- **苍白无华** → 血虚
- **淡白少华** → 气虚
- **白无华而浮肿** → 阳虚
- **面色青白** → 寒证
- **产后面色白** → 夺血伤气
- **猝然苍白** → 气随血脱危候
- **面色苍白 + 冷汗淋漓** → 阳气暴脱

## 黑色
- **黛黑、唇甲紫暗** → 肾阳衰微、阴寒凝滞
- **面黑干焦** → 肾阴虚
- **妇人眼眶灰黑无华** → 肾虚水饮或寒湿带下
- **鼻头色黑 + 目窠微肿** → 水饮内停
- **黑色 + 肌肤甲错** → 瘀血
- **心病者额见黑色** → 逆证
- **环口黛黑** → 肾绝
