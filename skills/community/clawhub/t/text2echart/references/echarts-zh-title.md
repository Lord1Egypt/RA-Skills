# title - 图表标题
Source: https://echarts.apache.org/zh/option.html#title

标题组件，包含主标题和副标题。

示例:
```json
{"text": "销售额", "left": "center", "textStyle": {"fontSize": 18}}
```

常见属性:
- text: string - 主标题文本
- subtext: string - 副标题文本  
- left/right/top/bottom: string - 位置 (center/left/right/50%)
- textStyle: object - 标题样式 {fontSize, color, fontWeight}
