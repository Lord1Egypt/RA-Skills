# GB/T 9704-2012《党政机关公文格式》完整要素

国家标准 GB/T 9704-2012 是党政机关公文的强制格式规范。**事业单位、国企央企的正式公文也参照执行。** 企业内部公文格式相对灵活，但越正式越接近此标准。

---

## 一、用纸与版心

| 项目 | 规格 |
|---|---|
| 纸张 | A4 (210mm × 297mm)，竖版 |
| 页边距（天头）上 | 37mm ± 1mm |
| 页边距（地脚）下 | 35mm ± 1mm |
| 页边距 左 | 28mm ± 1mm |
| 页边距 右 | 26mm ± 1mm |
| 版心尺寸 | 156mm × 225mm |

## 二、文档结构三大部分

```
┌─────────────────────────────────────┐
│ 一、版头 (从份号到红色分隔线)         │
│   - 份号（涉密件用）                  │
│   - 密级 + 保密期限                   │
│   - 紧急程度                          │
│   - 发文机关标志（大红字"XXX文件"）   │
│   - 发文字号（XX发〔2026〕X号）        │
│   - 签发人（上行文必备）               │
│   ─────── 红色分隔线 ───────         │
├─────────────────────────────────────┤
│ 二、主体 (从标题到附注)               │
│   - 标题                              │
│   - 主送机关                          │
│   - 正文                              │
│   - 附件说明                          │
│   - 发文机关署名                      │
│   - 成文日期                          │
│   - 印章                              │
│   - 附注                              │
├─────────────────────────────────────┤
│ 三、版记 (从抄送机关开始)             │
│   - 抄送机关                          │
│   - 印发机关 + 印发日期               │
└─────────────────────────────────────┘
```

## 三、字体字号（最核心）

**通用规则：除特殊说明外，正文及大部分要素均为 三号字。**

| 要素 | 字体 | 字号 |
|---|---|---|
| 份号 | 黑体 / 阿拉伯数字 | 三号 |
| 密级 | 黑体 | 三号 |
| 紧急程度 | 黑体 | 三号 |
| 发文机关标志 | 小标宋体 / 黑体 | 一般大于二号 |
| 发文字号 | 仿宋_GB2312 / 阿拉伯数字 | 三号 |
| 签发人 | "签发人" 用 仿宋_GB2312，姓名用 楷体_GB2312 | 三号 |
| **标题** | **小标宋体（电子文档可用"宋体加粗"）** | **二号** |
| 主送机关 | 仿宋_GB2312 | 三号 |
| **正文** | **仿宋_GB2312** | **三号** |
| **一级标题（"一、"）** | **黑体** | **三号** |
| **二级标题（"（一）"）** | **楷体_GB2312** | **三号** |
| **三级标题（"1."）** | **仿宋_GB2312 加粗** | **三号** |
| **四级标题（"(1)"）** | **仿宋_GB2312** | **三号** |
| 发文机关署名 | 仿宋_GB2312 | 三号 |
| 成文日期 | 仿宋_GB2312 / 汉字 | 三号 |
| 附件 | 仿宋_GB2312 | 三号 |
| 抄送机关 | 仿宋_GB2312 | 四号 |
| 印发机关与印发日期 | 仿宋_GB2312 | 四号 |
| 页码 | 4 号半角 | 阿拉伯数字 |

---

## 四、行距与段落

- **行距**：固定值 **28-29 磅**（标准要求每页排 22 行，每行 28 字）
- **段落首行缩进**：2 字符
- **标题居中**，可分一行或多行排列
- **段间距**：一般无额外段间距（已通过行距控制）

---

## 五、标题构造（三要素法）

**标准结构：**

```
[发文机关] 关于 [事由] 的 [文种]
```

**示例：**
- ✅ `国务院关于深化国有企业改革的指导意见`
- ✅ `教育部办公厅关于做好2026年招生工作的通知`
- ❌ `关于深化改革的通知`（缺发文机关，缺事由具体性）
- ❌ `通知`（三要素全无）

**特殊情况：**
- 转发性通知：`XX 关于转发 YY《关于……的通知》的通知`（"通知"出现两次）
- 批转性通知：可省略"关于"，如 `国务院批转 XX 部《关于……的意见》的通知`
- 标题不加书名号（除非整体引用某个标准的文件名）
- 标题不加标点符号（除特殊情况）

---

## 六、发文字号

**结构：** `[机关代字]〔YYYY〕[序号]号`

注意：
- 使用六角括号 **〔 〕**（不是方括号 [ ]，不是圆括号 ( )）
- 年份用阿拉伯数字
- 序号不编虚位（"5 号" 不写作 "05 号"）

**示例：**
- ✅ `国发〔2026〕12号`
- ✅ `教高〔2026〕3号`
- ❌ `国发[2026]12号`（括号错）
- ❌ `国发〔2026〕第12号`（多余"第"字）
- ❌ `国发〔2026〕012号`（虚位错）

---

## 七、签发人（仅上行文）

**位置**：发文字号同行右侧。
**格式**：`签发人：张三`（"签发人"3 字仿宋，"张三" 楷体）

如有多个签发人：主签发人姓名居前，其他签发人按职务从高到低、从左到右、从上到下顺序排列。

**仅上行文需要签发人。** 下行文、平行文不写签发人。

---

## 八、主送机关

**位置**：标题下空一行，顶格写。

**写法：**
- 主送一个机关：`XXX：`
- 主送多个机关：`XXX、YYY、ZZZ：` 或 `各省、自治区、直辖市人民政府，国务院各部委、各直属机构：`
- 主送机关不能是"各位领导""有关同志"等模糊称谓

**结束符号**：冒号（"："），不是逗号、句号或顿号。

---

## 九、附件说明

如有附件，在正文下空一行，左空 2 字写："附件："

**写法：**
- 单个附件：`附件：XXXX 名录`
- 多个附件：
  ```
  附件：1. XXX 名录
        2. YYY 章程
  ```

注意：
- 附件名后不加标点（不加书名号、不加句号）
- 多附件用阿拉伯数字编号
- 附件本身另起页，左上角顶格标"附件："，附件名称居中

---

## 十、发文机关署名 + 成文日期 + 印章

**位置**：正文下方右侧。

**格式：**
```
                               XXX 部
                       二〇二六年五月十三日
                          (此处加盖印章)
```

**关键规则：**
- **成文日期用汉字数字**（"二〇二六年五月十三日"），**零写作"〇"** (不是"O" 不是 "零")
- 必须精确到日，不能写"五月" 或 "2026 春"
- 印章压成文日期（"骑年盖月" — 章的上沿压在年份上）
- 联合行文：多个发文机关并列署名，所有印章并排盖

---

## 十一、版记

**位置**：公文末页下方，与版心边线对齐。

**抄送机关：**
- 顶格写"抄送："
- 按机关性质排列：党委 → 人大 → 政府 → 政协 → 法院 → 检察院 → 军队 → 民主党派 → 群众团体 → 其他
- 句尾用句号

**印发机关与印发日期：**
- 印发机关左空 1 字
- 印发日期右空 1 字
- 用阿拉伯数字，如 `2026年5月13日印发`

---

## 十二、页码

**位置**：版心外的下方（页脚）。
**字号**：4 号半角阿拉伯数字。
**排列**：单页码居版心右侧（右对齐），双页码居版心左侧（左对齐）。**单页"-1-" 双页 "-2-"** 这种带短线的格式是规范写法。

---

## 十三、印刷与装订

- 双面印刷
- 页码用阿拉伯数字
- 公文左侧装订

---

## 十四、电子文档格式速查

| 软件 | 操作 |
|---|---|
| Microsoft Word | 字体直接选"仿宋_GB2312" / "黑体" / "楷体_GB2312" / "小标宋体" |
| WPS | 上述字体一般预装 |
| 缺少字体 | 替代方案：仿宋_GB2312 → 仿宋；楷体_GB2312 → 楷体（视觉差异小） |
| 字号 | 三号 = 16pt |
| 一号 → 七号 对照 | 一号=26pt 二号=22pt 三号=16pt 四号=14pt 小四=12pt 五号=10.5pt |

---

## 十五、企业 / 事业单位常见简化

非党政机关的公文不强制完全遵循 GB/T 9704，但越正式越接近。常见简化：

| 要素 | 党政标准 | 企业简化 |
|---|---|---|
| 发文机关标志 | 大红字"XXX文件" | 公司 logo + 名称 |
| 发文字号 | 必须有 | 可选 |
| 签发人 | 上行文必须 | 一般省略 |
| 字体 | 强制仿宋_GB2312 | 仿宋或宋体均可 |
| 印章 | 实体公章 | 公章或电子章 |
| 版记 | 完整 | 抄送和印发日期常省 |

- [ ] 主送机关以冒号结尾
