# 影视分镜专业术语 (Film Storyboarding Terminology)

## 镜头焦距 (Lens Focal Length)
- **广角镜头 (Wide Angle / 16-35mm)**: 视野宽阔，强调空间感和环境背景。适合展示场景全貌、产品使用环境。
- **标准镜头 (Standard / 35-50mm)**: 接近人眼视角，自然真实。适合 UGC 类内容、人物口播。
- **长焦镜头 (Telephoto / 85-135mm)**: 压缩空间，突出主体，背景虚化（浅景深）。适合产品特写、人物情感表达。
- **微距 (Macro)**: 拍摄极近距离细节。适合产品材质展示、护肤品质地、珠宝细节。

### 焦段 → 情绪 → 适用场景映射 (Lens-Emotion Decision Table)

| 焦段 | 视觉特性 | 情绪暗示 | 适用 Mode | 示例场景 |
|------|---------|---------|----------|---------|
| **24mm 广角** | 空间扩张、边缘畸变 | 环境压迫、史诗感、孤独 | Cinematic / One-Shot | 建立镜头、废墟全景、人物被环境吞噬 |
| **35mm 小广角** | 轻微环境感、自然透视 | 亲和、对话感、纪实 | UGC / Multi-shot | 双人对话、生活场景、街头跟拍 |
| **50mm 标准** | 最接近人眼、无畸变 | 中立、客观、真实 | UGC / Ecommerce | 口播、产品演示、中性叙事 |
| **85mm 中长焦** | 浅景深、主体分离 | 亲密、注视、情绪 isolation | Cinematic / Short Drama | 单人情绪特写、产品质感、人像 |
| **135mm 长焦** | 强压缩、极浅景深 | 窥视、距离感、戏剧性 | Cinematic | 远距离观察、悬疑感、浪漫虚化 |

**选焦段决策顺序**: 先问"想表达什么情绪关系（压迫/亲和/亲密/距离）" → 再选焦段 → 最后匹配景别。不要先选焦段再硬套情绪。

## 镜头角度 (Camera Angle)
- **平拍 (Eye-level)**: 正常视角，平实稳重。UGC 默认角度。
- **俯拍 (High Angle / Bird's Eye)**: 展现人物渺小、无助或产品全貌。电商产品俯拍常见。
- **仰拍 (Low Angle)**: 展现人物高大、威严、有力量感。电影感品牌常用。
- **倾斜镜头 (Dutch Angle)**: 表现不安、疯狂或动荡。适合悬疑/紧张氛围。
- **过肩镜头 (Over-the-shoulder)**: 展示人物视角和空间关系。适合叙事、互动场景。

## 镜头运动 (Camera Movement)
- **推 (Push In)**: 镜头逐渐靠近主体，引导观众注意力。电商产品默认运动。
- **拉 (Pull Out)**: 镜头逐渐远离，展现环境和孤独感。产品揭示开场。
- **摇 (Pan)**: 镜头左右转动，横向扫描场景。环境展示。
- **移 (Truck / Dolly)**: 镜头随主体平行移动，保持跟拍。人物行走/产品展台。
- **跟 (Follow Shot)**: 镜头始终跟随主体运动。动作场景。
- **升/降 (Crane / Jib Shot)**: 镜头垂直升起或降下，展现宏大场面。品牌片开场。
- **环绕 (Orbit)**: 镜头围绕主体旋转。产品 360° 展示（电商常见）。
- **手持 (Handheld)**: 模拟真实手持拍摄。UGC 默认风格。

## 画面构图 (Composition)
- **三分法 (Rule of Thirds)**: 主体放在画面黄金分割线上。通用构图。
- **中心构图 (Center Frame)**: 产品/人物居中。电商产品默认。
- **对称构图 (Symmetry)**: 稳定、庄严、神圣感。高端品牌。
- **对角线构图 (Diagonal)**: 动态、引导性、速度感。运动/汽车品类。
- **框架式构图 (Framing)**: 利用门框、窗户等形成画中画，增加层次。叙事场景。

## 光影氛围 (Lighting & Atmosphere)
- **三点布光 (Three-Point Lighting)**: 主光 + 辅光 + 轮廓光。商业拍摄标准。
- **蝴蝶光 (Butterfly Lighting)**: 鼻下蝴蝶形阴影。时尚/美妆。
- **伦勃朗光 (Rembrandt Lighting)**: 脸部一侧倒三角亮区。电影感人像。
- **侧光 (Side Lighting)**: 强调纹理和立体感。产品材质展示。
- **逆光 (Backlight / Rim Light)**: 主体剪影或轮廓光。氛围感镜头。
- **霓虹光 (Neon Lighting)**: 蓝紫/品红色调。赛博朋克/潮流风格。
- **金色时刻 (Golden Hour)**: 暖橙色自然光。户外品牌/生活方式。
- **黑白高反差 (Noir)**: 戏剧性阴影、高对比度。电影感/奢侈品。

## 广告工业术语 (Advertising Terminology)

### 电商 (Ecommerce)
- **PDP Loop**: 商品详情页循环播放视频。通常 5-10 秒，展示产品各角度。
- **Hero Shot**: 产品主图/英雄镜头。产品最具冲击力的单一画面。
- **Packshot**: 产品包装展示镜头。通常放在视频最后几秒。
- **Paid-Social Creative**: 付费社交投放素材。需符合平台尺寸规范。
- **360° Spin**: 产品 360° 旋转展示。电商常用。
- **Lifestyle Shot**: 生活场景中的产品使用镜头。
- **White Background**: 纯白背景。电商平台要求（如 Amazon 主图）。

### UGC
- **Talking Head**: 人物对镜头讲话。信息流广告主流格式。
- **Hook**: 开头的抓注意力片段。通常前 3 秒决定留存。
- **CTA (Call to Action)**: 行动号召。视频末尾引导点击/购买。
- **Scroll Stopper**: 滚动停止器。社交信息流中让用户停下的视觉元素。
- **Creator Pacing**: 创作者节奏。自然的说话和手势节奏，不加速。
- **Product Demo**: 产品实际使用演示。比口播更有说服力。
- **Before/After**: 使用前后对比。功效型产品常用。

### 电影感 (Cinematic)
- **Establishing Shot**: 建立场景的全景镜头。通常作为开场。
- **Motivated Camera**: 有动机的镜头运动。每次运动都有叙事理由。
- **Emotional Payoff**: 情绪 payoff。每个镜头只承载一个情感点。
- **Shot List**: 分镜头列表。现场拍摄的镜头规划。
- **Master Shot**: 主镜头。覆盖整个场景的广角镜头。
- **Coverage**: 覆盖镜头。从多角度拍摄同一场戏。
- **Blocking**: 走位。演员和摄像机的调度编排。

### 一致性与质量 (Consistency & Quality)
- **Face Drift**: 人脸漂移。AI 生成中人脸特征在不同帧间变化。
- **Geometry Preservation**: 几何保持。产品形状在运动过程中保持不变。
- **Label Readability**: 标签可读性。产品标签文字清晰可辨。
- **Artifacts**: 伪影。生成的异常视觉元素（多余手指、文字乱码等）。
- **Jitter**: 画面抖动。运动过程中不自然的帧间跳跃。
- **Identity Anchor**: 身份锚定。用参考图锁定人物/产品的视觉特征。

## 一镜到底 (One-Shot / Long Take)
- **Frame Sequence**: 帧序列。按时间顺序排列的 2-10 张关键帧图片。
- **Transition Fill**: 转场补帧。AI 在两帧之间自动生成的过渡画面。
- **Spatial Continuity**: 空间连续性。一镜到底的核心质量指标，画面空间关系持续不跳跃。
- **Spiral Transition**: 螺旋转场。镜头围绕中心点螺旋推进的转场方式。
- **Match Cut**: 匹配剪辑。利用相邻画面中的相似形状、颜色或构图元素完成无缝衔接。
- **Whip Pan**: 甩镜。快速横向转动镜头完成的动感转场。
- **Wipe Transition**: 遮挡转场。利用画面中物体移动遮挡镜头完成的自然转场。

## 爆款复刻 (Viral Replication)
- **Style Transfer**: 风格迁移。将参考视频的运镜手法/色调/节奏迁移到新视频。
- **Subject Replacement**: 主体替换。保持场景和运镜，仅替换画面中的主体（人物/产品）。
- **Viral Deconstruction**: 爆款拆解。分析参考视频的爆点要素（文案结构/剧情框架/配乐画风），作为再生基础。
- **Original Ghosting**: 原主体残留。复刻失败的常见问题，原参考视频的主体在新视频中部分残留。
- **Style Drift**: 风格漂移。生成结果与参考视频的风格逐渐偏离。

## Agent 模式 (Agent Mode)
- **Intent Parsing**: 意图解析。从用户自然语言输入中提取主体、风格、时长、情绪等创作要素。
- **Route Planning**: 路径规划。根据解析结果自动选择最优创作子模式。
- **Material Detection**: 素材检测。自动识别输入中是否包含脚本标记、图片引用或视频引用。
- **Style Detection**: 风格检测。根据关键词自动映射到预设视觉风格。

## 短剧创作 (Short Drama)
- **Script Parsing**: 剧本解析。从标准格式剧本中提取场景、人物、对白、动作标记。
- **Character Registry**: 角色注册表。全局角色管理，存储每个角色的视觉设定。
- **@Character Reference**: @角色引用。在分镜中通过 @角色名 调用角色库中的形象。
- **Cross-Episode Consistency**: 跨集一致性。确保角色在多个剧集中的外貌、服装、场景保持一致。
- **Narration Adaptation**: 旁白改编。Agent 自动将剧本中的环境描述和 OS 改为视频旁白。
- **Multi-Style Support**: 多画风支持。同一剧本可切换写实/漫画/3D/二次元等不同画风输出。

## 速度修饰词 (Speed Modifiers)

| 速度 | 关键词 | 稳定性 |
|------|--------|--------|
| 极慢 | imperceptible, barely, 几乎不动 | 最稳 |
| 慢 | slow, gentle, gradual, 缓慢 | 推荐，默认 |
| 自然 | smooth, controlled, 平稳 | 安全 |
| 动态 | dynamic, swift, 灵动 | 慎用 |
| 快 | fast, rapid, 快速 | 极易翻车，避免 |

动作越慢越顺，视频越稳。

## 音频专业术语 (Audio & Sound Design)
- **Foley**: 拟音。在录音棚中模拟并录制与画面同步的日常音效（脚步声、衣物摩擦、开关门等）。
- **ADR (Automated Dialogue Replacement)**: 自动对白替换。后期在录音棚中重新录制对白，替换现场收音质量不佳的台词。
- **混音 (Mixing)**: 将对白、音效、音乐等多轨音频按比例混合为最终声轨的过程。
- **声场 (Sound Stage / Sound Field)**: 声音在三维空间中的分布和定位，包括宽度、深度和高度感。
- **低频冲击 (Low-Frequency Impact / LFE)**: 影院级低频音效（爆炸、撞击），增强物理沉浸感。Nolan 风格标志。
- **环境音 (Ambience / Room Tone)**: 场景的背景声音层，建立空间真实感（空调声、远处的交通声等）。
- **音画同步 (Sync Point / Hit Point)**: 音效与画面关键动作精确对齐的时间点。
- **声音设计 (Sound Design)**: 创造性音频制作，包括音效合成、氛围营造和声音特效。
- **对白清晰度 (Dialogue Intelligibility)**: 对白在混音中的可辨识程度，UGC/口播类内容的核心指标。
- **BGM (Background Music)**: 背景音乐。电商模式需保持在产品之下，UGC 模式需保持在语音之下。
- **音效 (SFX / Sound Effects)**: 特定动作或事件的声音效果（倒水声、点击声、转场音效等）。
- **旁白 (Voiceover / OS)**: 画面外的叙述声音，短剧模式中 Agent 自动将环境描述改编为旁白。
- **动态范围 (Dynamic Range)**: 音频中最安静与最响部分之间的差值。电影感需要大动态范围，UGC 需要压缩动态范围。
- **频段 (Frequency Band)**: 音频的频率分区。低频(20-250Hz)营造氛围，中频(250-4kHz)承载人声，高频(4-20kHz)增加细节和空气感。
