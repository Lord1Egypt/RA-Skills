# subsession-delete

Delete a child session cleanly from disk and the session index.

Use the included script for dry-run or execute mode:

```bash
python3 skills/subsession-delete/scripts/delete_subsession.py --session-key 'agent:subtest2:subagent:EXAMPLE'
python3 skills/subsession-delete/scripts/delete_subsession.py --session-key 'agent:subtest2:subagent:EXAMPLE' --execute
```
