# 各施工阶段验收清单

> 按施工阶段顺序排列，每个阶段对应"进场验收"和"完工验收"两个节点

---

## 阶段一：拆除/基础处理

### 进场核查
- [ ] 拆除范围与设计图纸一致，无超拆（尤其承重墙）
- [ ] 已确认楼上楼下通知到位（有书面或微信截图留存）
- [ ] 垃圾袋装下楼，不占用电梯过久

### 完工验收
- [ ] 墙面基层空鼓处已铲除，铲除边界整齐
- [ ] 结构墙未被破坏，如有开槽需确认非承重
- [ ] 原有老旧电线、铅管、铸铁管已拆除确认
- [ ] 拍照记录原始墙面/地面状态（防纠纷）

---

## 阶段二：水电工程（最重要的隐蔽节点）

### 进场确认（开工前）
- [ ] 设计图（水电点位图）已确认，并与业主当面对图
- [ ] 材料（电线、PPR管）品牌型号与合同一致，拍照留存

### 开槽阶段
- [ ] 横槽长度不超过墙面总宽的1/3（防止影响结构）
- [ ] 楼板禁止竖向开槽，禁止切断受力筋

### 布管阶段
- [ ] 强弱电分管铺设，间距≥150mm，拍照留存
- [ ] 弯管处使用弯管弹簧，弯曲角度不超过90°，半径≥4倍管径
- [ ] 管内无接头（接头只在接线盒内）
- [ ] 每根管内穿线总截面积不超过管截面积的40%

### 布线阶段
- [ ] 电线颜色规范：火线（红/棕）、零线（蓝/黄绿）、地线（黄绿双色）
- [ ] 强弱电绝不共管
- [ ] 厨房、卫生间、空调电路各独立回路，贴标签标注

### 给排水阶段
- [ ] PPR热熔接头外观均匀（有均匀翻边），无渗漏痕迹
- [ ] 冷热水管用不同颜色管或贴标签区分（热水管左侧/红色）
- [ ] 排水管坡度目测正确（水往出水口方向流）
- [ ] 地漏设置正确，每个需要排水的区域均有地漏
- [ ] 立管穿楼板处做隔音棉包裹，拍照存档

### 打压测试（关键！）
- [ ] 给水管打压：0.8MPa，稳压30分钟，压降≤0.05MPa
- [ ] **必须录制视频留存**（包含压力表读数、开始时间、结束时间）
- [ ] 排水管通球测试（用乒乓球或特定球体，确认管道通畅）

### 水电完工验收
- [ ] 所有线盒固定牢固，深度与抹灰面齐平
- [ ] 管槽填充水泥砂浆，表面平整
- [ ] 所有线路、管路走向拍全景照片留存（用于日后装修或维修定位）
- [ ] 每个回路标注用途（配电箱内贴标签）

---

## 阶段三：防水工程

### 施工前确认
- [ ] 卫生间/厨房基层干燥，无浮灰
- [ ] 管道根部、阴角已用砂浆做圆弧处理

### 施工中确认
- [ ] 防水材料品牌型号与合同一致（当场拍照桶身标签）
- [ ] 墙面涂刷高度达标（淋浴区≥1800mm，普通卫生间≥300mm）
- [ ] 涂刷遍数≥2遍（等第一遍干透再刷第二遍）
- [ ] 管道根部加强处理（加铺胎布）

### 闭水试验（关键节点）
- [ ] 蓄水深度≥20mm，在水位线处用蜡笔/彩色胶带标记
- [ ] 告知楼下邻居（留通知截图或书面签字）
- [ ] 静置**48小时**后检查
- [ ] 楼下无渗漏，水位线变化≤1mm/24h 为合格
- [ ] 拍摄"试验前+试验后"水位线对比照片

---

## 阶段四：泥工（瓷砖铺贴）

### 铺贴前
- [ ] 基层检查：空鼓、起砂部位已处理
- [ ] 砖已泡水（陶质砖）或抹背胶（玻化砖/大砖）
- [ ] 排版方案已确认（切砖尺寸、非整砖位置）

### 铺贴中
- [ ] 每铺5-10块后检查平整度（2m靠尺+塞尺）
- [ ] 随手轻敲检查空鼓（清脆实音为好，沉闷空音需返工）
- [ ] 缝宽均匀，符合设计要求（通常1.5-3mm）

### 铺贴完工验收
- [ ] 平整度偏差≤2mm（2m靠尺量）
- [ ] 空鼓率：整体面积空鼓≤5%，单块空鼓面积≤单块面积5%，角部无空鼓
- [ ] 缝格顺直，高低差≤0.5mm
- [ ] 非整砖排布合理，无小于1/3整砖的切割块在显眼位置
- [ ] 地砖排水坡度正确（厨卫地面向地漏找坡）

---

## 阶段五：木工工程

- [ ] 吊顶龙骨间距符合标准（次龙骨≤400mm）
- [ ] 吊筋固定牢固，螺丝与吊筋连接无松动
- [ ] 石膏板接缝留缝3-5mm，错缝铺设
- [ ] 柜体板材E1级确认（查合格证/环保证书）
- [ ] 柜体方正，对角线误差≤3mm

---

## 阶段六：涂料工程

- [ ] 腻子批刮满2遍，打磨后触感光滑
- [ ] 阴阳角顺直（用靠尺检测）
- [ ] 底漆涂刷均匀，无漏刷
- [ ] 面漆光洁，无明显刷纹、流挂、起皮
- [ ] 涂刷遍数与合同一致（检查桶数/用量是否合理）

---

## 阶段七：竣工综合验收

### 水电复查
- [ ] 全部开关插座通电测试
- [ ] 网络、电话、有线电视通路测试
- [ ] 每个回路独立分断测试（防短路）

### 门窗
- [ ] 开关顺畅，无翘角、变形
- [ ] 门缝均匀（上缝2mm，左右缝2mm，下缝5-8mm）
- [ ] 门锁、合页、门吸安装牢固

### 地暖（如有）
- [ ] 压力测试留存记录
- [ ] 分水器各路畅通
- [ ] 覆盖前拍照留存管路走向

### 厨卫
- [ ] 下水管通水测试，无堵塞
- [ ] 热水器冷热水管正确（左热右冷）
- [ ] 坐便器冲水正常，无渗漏

### 文件移交
- [ ] 水电点位图（纸质/电子均可）
- [ ] 打压试验视频
- [ ] 闭水试验照片
- [ ] 材料合格证/检测报告（尤其防水、板材、电线）
- [ ] 保修单据（水电管道质保≥2年，防水≥5年）
