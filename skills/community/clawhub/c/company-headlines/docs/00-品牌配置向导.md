# 品牌配置向导

品牌配置向导通过 **7 个问题**，为用户公司生成专属的品牌基调卡。全程约 3 分钟。

---

## 问题 1：公司名称

**"贵公司的全称是什么？"**

用户输入公司完整名称，用于所有文案落款和标题中的正式称谓。

示例：
- 广州雄韬信息技术有限公司
- XX新能源股份有限公司

简短称呼（如"雄韬"）可同时指定，用于内部版文案。

---

## 问题 2：行业/赛道

**"贵公司属于哪个行业？"**

| 选项 | 行业包 | 默认语调 | 关键词域 |
|------|--------|----------|----------|
| A | 制造业 | 稳健、重工艺 | 产线、良率、工艺、交付、智造 |
| B | 新能源/电池 | 技术+使命驱动 | 储能、零碳、循环、安全、密度 |
| C | 财税 | 精准、合规、信任 | 合规、风控、降本、数字化、筹划 |
| D | 科技/SaaS | 快节奏、数据导向 | 赋能、效率、场景、协作、AI |
| E | 电商/消费 | 活泼、用户视角 | 体验、爆款、服务、升级、生态 |
| F | 通用/自定义 | 用户自定 | 用户填入 5-8 个高频关键词 |

选择后，系统加载对应行业包的默认值，后续问题基于默认值微调。

---

## 问题 3：品牌主调人格

**"贵公司的对外形象更接近哪一种？"**

| 选项 | 人格 | 适合 | 特征 |
|------|------|------|------|
| A | 稳健可靠 | B2B、制造、财税、能源 | 像一个有深厚功底的工程师，不喧哗，但每句话有分量 |
| B | 技术先锋 | 科技、研发驱动型 | 自信锐利，用数据和技术语言说话，节奏明快 |
| C | 温暖使命 | 社会责任型、环保、健康 | 有情怀、有温度，强调愿景和意义 |
| D | 新锐活力 | 消费品牌、创业公司 | 年轻、有趣、敢表达，可以适当轻松 |
| E | 自定义 | — | 用户用一句话描述自己的品牌人格 |

---

## 问题 4：副调切换

**"不同场景下，语调需要微调吗？"**

| 选项 | 说明 |
|------|------|
| A | 节日/祝福场景 → 更温暖；技术发布 → 更锐利；其余保持主调 |
| B | 始终统一，不切换 |
| C | 自定义映射 |

系统据此生成副调切换表，例如：
```
节日 → 温暖+20%，锐利-10%
发布 → 锐利+20%，温度保持
日常 → 主调
```

---

## 问题 5：禁用词

**"有哪些词语是贵公司绝对不用的？"**

系统展示默认禁用词列表，用户可增删：

**默认禁用**：
- 颠覆、革命、重新定义、突破性、前所未有
- 大力推进、全面赋能、深度布局、重磅
- 超级、极致、独一无二（除数据支撑）

**用户操作**：
- 勾选保留默认列表
- 增删具体条目
- 或完全自定义

---

## 问题 6：渠道清单

**"内容发布到哪些渠道？"**

| 选项 | 渠道 | 说明 |
|------|------|------|
| A | 微信公众号 | 品牌传播，面向客户和行业 |
| B | 官网新闻中心 | 正式公告，面向投资人和媒体 |
| C | 内部OA/企微 | 内部通讯，面向员工 |
| D | 抖音/小红书 | 短视频/图文平台 |
| E | 自定义添加 | 其他渠道 |

可多选。不同渠道对应不同的篇幅、语气和视觉输出规格。

---

## 问题 7：内容品类

**"日常需要写哪些类型的内容？"**

系统展示 7 大品类，用户勾选启用：

- [x] 头条/行业快讯（默认开启）
- [x] 节日/节气祝福（默认开启）
- [x] 合作/签约动态
- [x] 技术升级/产品迭代
- [x] 项目考查/客户考察
- [x] 重大新闻/发布
- [x] 人物/文化故事
- [ ] 自定义品类（用户新增）

---

## 输出：品牌基调卡

6 题完成后，系统生成品牌基调卡，保存为 `.company-headlines/brand-card.md`。

详见 `templates/brand-card-template.md`。

---

## 重新配置

任何时候输入 `/头条 配置` 即可重新运行向导，覆盖已有的品牌卡。
