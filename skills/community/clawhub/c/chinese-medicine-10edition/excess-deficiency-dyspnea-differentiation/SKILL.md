# Excess vs Deficiency Dyspnea Differentiation

## When to Use
- 患者主症为喘促（呼吸急促、困难）
- 排除非喘证引起的单纯气短或焦虑性呼吸异常

## Execution Steps

### 1. 判断是否为实喘
若患者同时符合以下全部特征，则辨为实喘：
- **呼吸特点**：呼吸深长有余，呼出为快
- **声音**：气粗声高
- **脉象**：脉数有力
- **病势**：起病骤急，多见于急性加重期
- **伴随症状**：常伴咳嗽、痰多、胸闷、发热等实邪表现

### 2. 判断是否为虚喘
若患者同时符合以下全部特征，则辨为虚喘：
- **呼吸特点**：呼吸短促难续，深吸为快
- **声音**：气怯声低
- **脉象**：脉微弱或浮大中空
- **病势**：起病徐缓，时轻时重，过劳即甚
- **伴随症状**：神疲乏力、自汗、腰膝酸软等虚损表现

### 3. 处理虚实夹杂情况
- 若症状兼有实喘与虚喘特征，需分析主次
- 急则治标（先处理实证），缓则治本（兼顾虚证）
- 病位提示：实喘主要在肺，虚喘涉及肺肾

### 4. 与现代医学对应参考
- **实喘**：常见于COPD急性加重期、喘息性支气管炎发作期
- **虚喘**：多见于慢性稳定期、肺心病后期

## Output
明确判断为“实喘”或“虚喘”，并据此推荐相应治则方向（祛邪利气 或 培补摄纳）。
