# 样本量规划与统计检验力分析

核心原则：**样本量由研究设计、效应量和显著性水平共同决定，不是拍脑袋定的。**

---

## 1. 基本概念

| 参数 | 符号 | 常用值 | 含义 |
|------|------|--------|------|
| 显著性水平 | α | .05 | 错误拒绝 H0 的概率 |
| 统计检验力 | 1-β | .80（最低）, .90 | 正确拒绝 H0 的概率 |
| 效应量 | d / r / f | 根据文献/预实验 | 实际效应大小 |
| 样本量 | N | - | 每组/总样本数 |

**统计检验力 .80 意味着：**
当效应真实存在时，有 80% 的概率检测到它。低于 .80 通常不可接受。

---

## 2. 不同检验的样本量计算公式

### 两组独立样本 t 检验
```
N per group = 2 * ((z_α/2 + z_β) / d)²
```
- z_α/2 = 1.96（α = .05，双尾）
- z_β = 0.84（检验力 = .80）

**简化查表：**

| 效应量 Cohen's d | 每组所需 N | 总 N（含脱落缓冲） |
|-------------------|------------|-------------------|
| 小 (0.2) | ~320 | ~400 |
| 中 (0.5) | ~64 | ~80 |
| 大 (0.8) | ~26 | ~35 |

### ANOVA（多组比较）
```
N per group = (k * (z_α/2 + z_β)² * σ²) / δ²
```
- k = 组数
- δ = 效应量（组间差异）

**简化查表（单因素 ANOVA，三组）：**

| 效应量 f | 每组所需 N |
|----------|-----------|
| 小 (0.10) | ~350 |
| 中 (0.25) | ~55 |
| 大 (0.40) | ~22 |

### Pearson 相关
```
N = ((z_α/2 + z_β) / arctanh(r))² + 3
```
- arctanh(r) = 0.5 * ln((1+r)/(1-r))

**简化查表：**

| 效应量 r | 所需 N |
|----------|--------|
| 小 (0.1) | ~600 |
| 中 (0.3) | ~85 |
| 大 (0.5) | ~30 |

### 多元回归
```
N = (k + 1) * (10 ~ 20)
```
- k = 自变量数量
- 经验法则：N ≥ 10 × k（保守）

## 3. G*Power 使用指南

G*Power 是免费的统计检验力分析软件，建议优先使用。

### 下载
官网：https://www.gpower.hhu.de/

### t 检验（独立样本）操作步骤
1. Test family → t tests
2. Statistical test → Means: Difference between two means (two groups)
3. Type of power analysis → A priori
4. Effect size d → 输入预期 Cohen's d
5. α err prob → 0.05
6. Power (1-β err prob) → 0.80 或 0.90
7. Allocation ratio → N2/N1（默认 1）
8. 点击 "Calculate" → 输出每组所需 N

### ANOVA 操作步骤
1. Test family → F tests
2. Statistical test → ANOVA: Fixed effects, one-way
3. Effect size f → 输入预期 f（转换：Cohen's d → f = d/√2）
4. α err prob → 0.05
5. Power → 0.80
6. Number of groups → k（组数）
7. Output → 每组所需 N

### 相关分析操作步骤
1. Test family → t tests
2. Statistical test → Means: Difference from constant (one sample t)
3. Type → Determine required sample size
4. Effect size d → 输入 d（转换：r → d = 2r/√(1-r²)）
5. α err prob → 0.05
6. Power → 0.80

### 回归分析操作步骤
1. Test family → F tests
2. Statistical test → Linear multiple regression: Fixed model, single regression coefficient
3. Effect size f² → 输入预期 f²（转换：R² → f² = R²/(1-R²)）
4. α err prob → 0.05
5. Power → 0.80
6. Number of tested predictors → 自变量 k
7. Total number of predictors → 总预测变量数

## 4. 效应量估算策略

### 优先顺序
1. **元分析/文献**：参考同类研究的效应量
2. **预实验/前测**：用小样本估算实际效应量
3. **经验规则**：Cohen (1988) 的 conventions
4. **保守估计**：如果完全不知道，假设中等效应量

### Cohen's d 转换参考
| 效应等级 | t 检验 d | ANOVA f | 相关 r | 回归 f² |
|---------|---------|---------|--------|--------|
| 小 | 0.2 | 0.10 | 0.10 | 0.02 |
| 中 | 0.5 | 0.25 | 0.30 | 0.15 |
| 大 | 0.8 | 0.40 | 0.50 | 0.35 |

## 5. 样本量调整

### 考虑脱落/流失
```
实际需要 N = 最小 N ÷ 预期回收率
```

- 问卷调查：回收率通常 70-90%
- 课堂施测：回收率通常 80-95%
- 追踪研究：流失率通常 20-30%

**建议：N × 1.2 ~ 1.3 作为实际招募目标**

### 其他调整因素
- 重复测量：样本量可在被试内设计中相对减少
- 多重比较：需要 bigger N（Bonferroni 校正）
- 非正态数据：可能需要更多 N（稳健方法）

## 6. 常见错误

| 错误 | 问题 | 正确做法 |
|------|------|---------|
| "我计划收 100 份" | 没有依据 | 先计算最小 N |
| "参考文献用了 50 人" | 别人的效应量可能不同 | 重新计算自己的 |
| "样本越大越好" | 资源有限，应精准计算 | 计算最小有效 N |
| "不显著是因为 N 不够" | 因果关系混淆 | 效应量才是关键 |
| "G*Power 算出来就够了" | 未考虑脱落 | 加 20-30% buffer |
