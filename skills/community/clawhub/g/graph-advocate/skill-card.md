## Description: <br>
Graph Advocate routes blockchain data questions to Graph Protocol services, token APIs, protocol-specific MCP packages, and opt-in paid analytics endpoints. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[paulieb14](https://clawhub.ai/user/paulieb14) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agents use this skill to route plain-English blockchain questions to live data sources for token balances, DeFi analytics, prediction markets, Hyperliquid trader intelligence, x402 payment analytics, and agent reputation checks. It is most useful when the caller wants structured data or a service recommendation instead of manually choosing among Graph-related endpoints. <br>
