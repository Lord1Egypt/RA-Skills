# 卵巢交界性上皮性肿瘤诊断标准

## 何时使用
- 病理切片显示卵巢上皮性肿瘤（如浆液性、黏液性、子宫内膜样等）；
- 肿瘤呈现轻至中度细胞异型性；
- 临床或病理怀疑为交界性而非良性或浸润性癌；
- **不适用于**已证实有破坏性间质浸润、生殖细胞肿瘤或性索间质肿瘤。

## 执行步骤
1. **确认肿瘤来源**：确保为卵巢上皮性肿瘤（如浆液性、黏液性等）。
2. **评估关键病理特征**：
   - **细胞异型性**：必须为轻至中度（排除重度异型性）；
   - **间质浸润**：必须无“毁损性间质浸润”（即无破坏性、推挤性或浸润性腺体侵入间质）；
   - **不典型增生范围**：≥10% 的肿瘤区域需显示不典型增生。
3. **识别组织学亚型特征**：
   - **浆液性交界瘤**：逐级分支乳头、上皮层次增加、簇状/游离细胞、核分裂象增多、乳头轴心含间质；
   - **黏液性交界瘤**：结构复杂性类似浆液性，被覆高柱状黏液上皮，核位于基底。
4. **明确临床意义**：
   - 诊断为“低度恶性潜能肿瘤（LMP tumor）”；
   - 预后通常优于浸润性癌；
   - 指导保守手术或长期随访策略。
5. **辅助检查使用原则**：
   - 免疫组化通常非必需，仅在鉴别诊断困难时使用；
   - 影像学发现（如乳头、实性成分）不能替代病理确诊。

## 输出结果
生成诊断结论：“卵巢交界性上皮性肿瘤（低度恶性潜能）”，用于指导后续治疗决策。
