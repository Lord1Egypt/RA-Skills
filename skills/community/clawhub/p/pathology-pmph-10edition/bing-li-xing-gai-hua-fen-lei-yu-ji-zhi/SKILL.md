# 病理性钙化的分类与机制

## 何时使用
- 在病理检查、影像学报告或临床评估中发现血管、肾、肺、间质、瘢痕、动脉粥样硬化斑块等非骨/牙组织存在钙盐沉积。
- 需要区分钙化类型以指导进一步病因诊断（如是否合并高血钙、甲状旁腺功能异常、肾衰竭等）。

## 执行步骤

1. **确认钙化位置**：确保钙化发生在骨和牙齿以外的组织（如血管、肾、肺、间质、瘢痕、血栓、心脏瓣膜等）。

2. **评估全身钙磷代谢状态**：
   - **若血钙水平正常** → 判断为**营养不良性钙化**：
     - 发生于坏死、损伤组织或异物中（如结核病灶、动脉粥样硬化斑块、陈旧性血栓）。
     - 局部机制可能包括碱性磷酸酶活性升高。
   - **若血钙水平升高**（存在钙磷代谢紊乱）→ 判断为**转移性钙化**：
     - 常见于甲状旁腺功能亢进、维生素D过量、慢性肾衰竭或骨肿瘤。
     - 钙盐沉积于原本正常的组织，尤其好发于血管、肾小管、肺泡壁和胃黏膜间质。

3. **输出结果**：
   - 明确钙化类型（营养不良性 或 转移性）。
   - 指出典型发生部位及潜在病因。
   - 提示是否需进一步检测血钙、血磷、PTH或肾功能。

4. **注意排除**：不适用于生理性骨化或牙齿形成过程中的钙沉积。
