# 受体-药物相互作用动力学与药物分类

## 何时使用
- 分析药物是否通过特异性受体发挥作用
- 解释药物的量-效曲线特征（如右移、Emax变化）
- 判断药物属于完全/部分激动药、竞争性/非竞争性拮抗药或反向激动药
- 比较不同药物的亲和力或内在活性
- 评估储备受体或偏向性信号传导的影响

## 执行步骤

### 1. 验证适用条件
- 确认药物作用依赖特异性受体
- 排除非受体机制药物（如抗酸药）和不可逆结合药物的简单模型

### 2. 应用基本结合动力学
- 使用质量作用定律描述药物（D）与受体（R）结合：D + R ⇌ DR → E
- 计算解离常数：KD = [D][R]/[DR]
- 计算受体占有率：[DR]/[RT] = [D]/(KD + [D])

### 3. 计算效应与关键参数
- 应用效应公式：E/Emax = α × [D]/(KD + [D])
- 确定内在活性 α（0 ≤ α ≤ 1）
- 计算亲和力指数：pD2 = -lgKD

### 4. 药物分类判断
- **激动药**：α > 0
  - 完全激动药：α = 1
  - 部分激动药：α < 1
- **拮抗药**：α = 0
  - 竞争性：可逆结合，量效曲线平行右移，Emax不变；计算 pA2
  - 非竞争性：不可逆或变构结合，Emax降低
- **特殊类型**：考虑反向激动药（偏好静息态受体）或偏向性激动药（选择性激活特定通路）

### 5. 考虑高级概念（如适用）
- 评估储备受体的存在对拮抗药起效的影响
- 应用二态模型解释受体构象变化
- 识别偏向性信号传导特征
