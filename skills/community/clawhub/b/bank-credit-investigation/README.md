# Bank Credit Investigation Assistant
# 银行信贷尽调助手

> *AI-powered credit due diligence for commercial banks — from borrower profiling to investigation report generation.*

> AI驱动的商业银行信贷尽调工具——从借款人画像到尽调报告生成，一站式完成。

---

## Overview

This skill provides a complete workflow for bank credit investigation officers to perform corporate lending due diligence. It covers borrower analysis, financial statement review, industry risk assessment, and structured investigation report generation.

本工具为银行信贷人员提供完整的企业信贷尽调工作流，涵盖借款人分析、财务报表审查、行业风险评估及结构化尽调报告生成。

## Installation

```bash
clawhub install bank-credit-investigation
```

## Core Capabilities

| Capability | Description |
|------------|-------------|
| Borrower Profile Analysis | Company background, ownership, management, business model |
| Financial Statement Analysis | 3-year ratio analysis, cash flow, asset quality |
| Industry Risk Assessment | Cyclicality, competition, regulatory, technology risks |
| Investigation Report Generation | Complete CBRC-style credit investigation report |
| Credit Facility Recommendation | Rating, amount, tenor, covenants, monitoring indicators |

## Use Cases

- **Pre-lending investigation** (贷前尽调调查)
- **Credit renewal review** (贷后管理审查)
- **Credit restructuring assessment** (信贷重组评估)
- **Industry risk monitoring** (行业风险监控)

## Target Users

- Commercial bank relationship managers (对公客户经理)
- Credit analysts and risk officers (信用分析师/风险管理员)
- Credit approval committees (信贷审批委员会)

## Compliance Note

Aligned with **CBIRC Guidelines for Commercial Bank Credit Risk Management** (商业银行信贷风险管理指引) and **Five-Level Classification** (贷款五级分类) standards.
