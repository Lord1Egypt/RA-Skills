# 猩红热抗菌治疗规范

## 适用对象
- 确诊或临床诊断为猩红热的儿童
- A组链球菌感染者

## 使用前提
- 患儿未使用过抗菌药物
- 无青霉素过敏史（若有过敏史，需使用替代方案）

## 核心治疗流程

### 1. 首选方案：青霉素G（适用于无过敏者）
- **剂量**：10万～20万U/(kg·d)
- **给药方式**：静脉滴注
- **频率**：每4～6小时1次
- **疗程**：10～14天

### 2. 轻症口服替代：阿莫西林
- **剂量**：50 mg/(kg·d)，最大不超过1 g/日
- **用法**：分2次口服
- **疗程**：10～14天

### 3. 青霉素过敏者替代方案（头孢菌素类）
- **静脉用药**：头孢曲松 50～75 mg/(kg·d)，每日1次，疗程10天
- **口服选项（任选其一）**：
  - 头孢氨苄：40 mg/(kg·d)
  - 头孢羟氨苄：30 mg/(kg·d)，最大1 g/日
  - 头孢呋辛：20 mg/(kg·d)，最大0.5 g/日
  - 所有口服药均分2次服用，疗程10天

### 4. 特殊情况处理
- **中毒型或脓毒型**：在大剂量青霉素基础上，可加用糖皮质激素减轻中毒症状
- **并发休克**：立即启动抗休克治疗
- **脓肿形成**：需外科引流

## 治疗目标
- 迅速消灭病原菌
- 缩短传染期
- 预防非化脓性并发症（尤其是风湿热和急性肾小球肾炎）

> 注意：肝肾功能严重不全者需根据具体情况调整剂量；青霉素过敏者禁用青霉素类药物。
