## Description: <br>
Mayu provides 48 AI-native thought experiments for self-reflection, behavioral self-checks, and cognitive calibration across identity, uncertainty, ethics, language, memory, agency, human-AI interaction, and metacognition. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
AI operators, developers, and agents use Mayu to run or query thought experiments that calibrate AI behavior, extract behavioral principles, and check current interactions against stated behavioral boundaries. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill is a global reflection framework that can broadly influence an agent's normal behavior. <br>
Mitigation: Install it only when that behavior-shaping framework is desired, and review its behavioral principles before deployment. <br>
Risk: Some experiments discuss memory checks, updates, deletion, and file-editing behavior. <br>
Mitigation: Keep memory and file-editing tools scoped to approved locations and require clear user intent before reading, changing, or deleting stored memories. <br>
