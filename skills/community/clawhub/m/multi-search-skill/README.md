# Multi Domain Search

**Search 8 knowledge domains in one agent for [OpenClaw](https://openclaw.ai) / [ClawHub](https://clawhub.ai).**

Not 8 web engines — 8 completely different databases. Live web, 250M academic papers, 150M scholarly works, arXiv preprints, 36M biomedical papers, chemical compounds, government datasets, and dictionary. All via x402 micropayments.

## Install

```bash
openclaw skills install multi-domain-search
```

Or browse on ClawHub: [clawhub.ai/plagtech/multi-domain-search](https://clawhub.ai/plagtech/multi-domain-search)

## Search Domains

| # | Domain | Coverage | Cost |
|---|--------|----------|------|
| 1 | Live Web | Tavily web search + content extraction + Q&A | $0.02–$0.03 |
| 2 | Academic Papers | 250M papers via OpenAlex | $0.002 |
| 3 | Scholarly Works | 150M works via Crossref (books, datasets, conferences) | $0.001–$0.002 |
| 4 | Preprints | arXiv preprints across all categories | $0.002 |
| 5 | Biomedical | 36M PubMed papers (medicine, biology, pharma) | $0.002 |
| 6 | Chemistry | PubChem compounds and structural similarity | $0.002 |
| 7 | Government Data | Data.gov datasets + US Census | $0.001 |
| 8 | Dictionary | Definitions, synonyms, phonetics | $0.001 |

15 endpoints total. A multi-domain search across 3-4 domains costs $0.006–$0.04.

Data sourced from Tavily, OpenAlex, Crossref, arXiv, PubMed, PubChem, Data.gov, US Census Bureau.
