# 组织学发展史中的关键里程碑

## 使用场景
- 准备组织学课程的历史背景介绍
- 撰写医学史相关论文时追溯组织学演进
- 解释细胞学说等基础理论的历史来源

## 执行步骤
1. **确认需求类型**：判断用户是否需要学科起源、技术演进或人物贡献等维度的历史信息。
2. **按时间顺序呈现关键节点**：
   - 1665年：Robert Hooke 首次使用“cell”描述细胞壁结构
   - 1677年：Antoni van Leeuwenhoek 发现多种活细胞
   - 1801年：Bichat 提出“组织”概念
   - 1838–1839年：Schleiden 与 Schwann 共同创立细胞学说
   - 19世纪中期后：显微镜与染色技术推动组织学发展
   - 20世纪初–中期：特殊显微镜（相差、荧光等）应用
   - 1940年代：电子显微镜开启亚细胞研究
   - 20世纪以来：与工程学、材料学等交叉形成组织工程等新领域
3. **补充中国贡献**：提及马文昭、鲍鉴清、王有琪等20世纪中国组织学家的推动作用。
4. **强调适用边界**：明确说明本内容仅用于历史理解，不用于实验操作指导。

> 注意：若用户询问具体实验方法、染色流程或现代技术细节，请引导其查询实验手册或技术文献，而非使用本技能。
