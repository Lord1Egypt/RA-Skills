# Blood-Activating and Stasis-Removing Herb Usage

## When to Use
确保满足以下全部条件：
- 已确诊为血瘀证
- 无活动性出血（如咯血、吐血未止）
- 患者非妊娠状态

## Execution Steps

### 1. 辨识血瘀证核心指征
确认存在以下至少两项典型表现：
- **疼痛特点**：固定不移、刺痛、夜间加重
- **出血特点**：色紫暗、夹血块
- **舌脉特征**：舌质紫暗或有瘀斑，脉涩或弦
- **其他体征**：癥瘕痞块、肌肤甲错、面色黧黑

### 2. 排除禁忌人群
若患者属于以下任一情况，禁止或慎用活血化瘀药：
- **孕妇**：绝对禁用破血逐瘀药（如土鳖虫、莪术），活血药亦需慎用
- **月经过多者**：禁用，以防加重出血
- **活动性出血期**：禁用，防止动血
- **阴虚火旺或热盛者**：慎用温性活血药（如川芎）

### 3. 按病证选择药物子类
根据瘀血部位及兼证，匹配对应类别：
- **气血瘀滞所致痛证**（如胸痹心痛、风湿痹痛）→ 选用活血止痛药（如川芎）
- **月经不调、产后腹痛** → 选用活血调经药（如丹参）
- **跌打损伤、骨折** → 选用活血疗伤药（如土鳖虫）
- **癥瘕积聚、久瘀顽症** → 选用破血消癥药（如莪术）

### 4. 注意特殊用药警示
- 川芎辛温走窜，阴虚火旺或热盛者慎用
- 丹参反藜芦，不可同用
- 土鳖虫有小毒，过敏体质者慎用

> **输出规则**：若存在任何禁忌证，应禁止使用；若符合指征，按上述子类推荐具体药物。
