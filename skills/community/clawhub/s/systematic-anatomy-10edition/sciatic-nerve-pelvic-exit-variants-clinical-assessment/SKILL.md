# 坐骨神经出盆腔解剖变异的临床评估

## 触发条件
当患者表现为坐骨神经痛，且影像学检查排除腰椎间盘突出时，应考虑梨状肌综合征可能，并启动本评估流程。

## 执行步骤

1. **确认正常解剖关系**：
   - 正常情况下（约66.3%人群），坐骨神经以单干形式经梨状肌下孔出盆腔。

2. **识别常见变异类型**（合计约占33.7%）：
   - **穿肌型**：坐骨神经单干直接穿过梨状肌——此型最易导致神经受压。
   - **分两支，一支穿肌、一支经下孔**。
   - **分两支，分别经梨状肌上孔与下孔**。

3. **评估临床后果**：
   - 穿肌型变异者常因肌肉收缩压迫神经，导致臀部疼痛、沿坐骨神经放射痛，压痛点位于坐骨结节与大转子连线中点。
   - 可通过“梨状肌紧张试验”诱发阳性体征。

4. **辅助诊断建议**：
   - 推荐行MRI或高分辨率超声检查，明确坐骨神经与梨状肌的空间关系。
   - 必须先排除腰椎源性病因。

5. **治疗指导意义**：
   - 若考虑手术松解梨状肌，术前必须明确变异类型，避免术中误伤神经。

## 注意事项
- 本技能仅适用于坐骨神经相关病变，不适用于股神经、闭孔神经等其他神经。
- 不适用于非梨状肌来源的神经压迫（如肿瘤、瘢痕粘连等）。
