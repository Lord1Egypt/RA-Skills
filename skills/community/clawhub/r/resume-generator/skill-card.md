## Description: <br>
Generates professional resume content for programmers and test engineers with multiple templates, Markdown and HTML export, and job-description keyword optimization. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[shenghoo123-png](https://clawhub.ai/user/shenghoo123-png) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, engineers, and job seekers use this skill to create structured technical resumes from personal resume data, export them as Markdown or HTML, and generate keyword suggestions for a target job description. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Resume inputs and exported files may contain personal data. <br>
Mitigation: Avoid unnecessary sensitive details and handle generated resume files according to the user's privacy and retention requirements. <br>
Risk: Generated HTML can include user-provided resume content. <br>
Mitigation: Only export or open HTML generated from content the user trusts. <br>
Risk: The documented PyPI package name may not be the exact package the user intends to trust. <br>
Mitigation: Verify the package identity before installing, or run the included source directly. <br>
