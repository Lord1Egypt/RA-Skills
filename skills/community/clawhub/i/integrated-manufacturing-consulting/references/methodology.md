# 数字化制造咨询调研方法论

## 项目总览

本方法论适用于离散制造企业数字化转型咨询项目，覆盖从企业调研到合作方案输出的完整链路。

### 项目四阶段

```
Phase 1                 Phase 2                 Phase 3                 Phase 4
  调研诊断               问题分析               改善规划               合作交付
┌──────────┐          ┌──────────┐           ┌──────────┐           ┌──────────┐
│ 部门调研  │  ──→   │ 问题梳理  │  ──→    │ 改善方案  │  ──→    │ 调研报告  │
│ 数据采集  │         │ 根因分析  │          │ 项目定义  │          │ 汇报PPT   │
│ 现场观察  │         │ 改善方向  │          │ 效益评估  │          │ 合作建议  │
└──────────┘          └──────────┘           └──────────┘           └──────────┘
```

### 调研对象（7大部门）

| 序号 | 部门 | 核心调研主题 | 关键关注点 |
|:----:|------|------------|-----------|
| 1 | 销售部 | 接单流程、预测机制、订单变更管理 | 订单全生命周期、急单插单 |
| 2 | 计划部 | MPS/MRP、排产逻辑、产能规划 | 计划颗粒度、响应速度、协同 |
| 3 | 生产部 | 现场管理、OEE、人员效率、生产节拍 | 浪费识别、瓶颈工序、异常响应 |
| 4 | 采购部 | 供应商管理、采购策略、来料异常 | 采购周期、质量控制、JIT |
| 5 | 仓储物流 | 仓库管理、物料流转、配送效率 | 库存准确率、库位管理、物流路径 |
| 6 | 质量部 | 质量体系、检验流程、质量数据 | 追溯管理、SPC、客诉处理 |
| 7 | 工艺技术部 | BOM管理、工艺路线、技术文件 | ECN流程、标准化、数据源维护 |

### 调研方法

| 方法 | 适用场景 | 输出物 |
|------|---------|--------|
| 高管访谈 | 战略、KPI、痛点、期望 | 高层访谈纪要 |
| 部门访谈 | 流程细节、痛点、系统使用 | 部门调研记录 |
| 现场观察(Gemba Walk) | 生产现场、物流路线、作业观察 | 现场观察记录 |
| 数据采集 | 产量、效率、质量、库存等 | 数据汇总表 |
| 问卷调研 | 覆盖面广、量化分析 | 调研问卷分析 |
| 流程穿行测试 | 端到端流程验证 | 流程穿行报告 |
| 系统演示 | 现有系统功能验证 | 系统功能评估 |

### 调研输出物清单

| 输出物 | 用途 | 受众 |
|--------|------|------|
| 调研检查清单 | 调研时逐项确认 | 顾问团队 |
| 部门调研记录 | 记录部门调研详情 | 项目组 |
| 问题清单 | 汇总所有发现的问题 | 项目组 |
| 改善方向矩阵 | 问题-改善-优先级映射 | 内部评审 |
| 调研报告(初稿) | 客户确认版本 | 客户方 |
| 调研汇报PPT | 正式汇报材料 | 客户高层 |
| 合作建议书 | 后续合作方案 | 客户决策层 |
