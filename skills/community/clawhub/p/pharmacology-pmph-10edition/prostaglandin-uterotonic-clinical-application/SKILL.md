# prostaglandin-uterotonic-clinical-application

## 何时使用
当出现以下任一临床指征时，可考虑使用前列腺素类子宫兴奋药：
- 需终止早期或中期妊娠
- 足月或过期妊娠需引产
- 需清除宫腔内良性葡萄胎组织
- 羊水栓塞的紧急抢救

## 如何执行
1. **确认适用对象**：确保目标为妊娠各期子宫、宫颈、蜕膜或胎盘组织，且患者无禁忌证。
2. **筛查禁忌证**：
   - 禁用于支气管哮喘患者（因可能诱发支气管痉挛）
   - 禁用于青光眼患者
   - 引产时禁忌证同缩宫素（如头盆不称、胎位异常、前置胎盘等）
3. **选择合适药物**：优先选用活性强的PGE2（地诺前列酮）或PGF2α（地诺前列素），必要时可考虑硫前列酮。
4. **理解药理作用**：
   - 兴奋子宫平滑肌，产生类似生理性阵痛的节律性收缩
   - 软化并松弛宫颈
   - 促使蜕膜和胎盘组织变性坏死
   - 刺激内源性前列腺素生成
5. **监测不良反应**：密切观察是否出现恶心、呕吐、腹痛等消化道平滑肌兴奋症状。
6. **实施干预**：根据妊娠时期、宫颈状态及临床目标，按规范给药途径（如阴道给药、静脉滴注等）执行治疗。

> 注意：分娩前子宫对前列腺素最敏感，用药剂量和方式应据此调整。
