# De Minimis and Threshold Quick Reference

A de minimis is the consignment value below which duty (and sometimes VAT) is waived. These rules change with budget cycles and trade agreements — always verify the current value before relying on any number.

| Country | Duty de minimis | VAT/GST threshold | Notes |
|---|---|---|---|
| United States | USD 800 (Section 321) | n/a (no national VAT) | Tightening enforcement; commercial imports excluded |
| Canada | CAD 20 generally; CAD 40 from US/MX via CUSMA | CAD 40 from US/MX | CUSMA gives a higher threshold |
| United Kingdom | £135 consignment | £135 | Above £135, duty applies; VAT applies at any value (collected from seller via UK VAT registration or at the border) |
| EU (general) | €150 | €0 — VAT due on all imports | IOSS scheme for consignments ≤ €150 streamlines VAT collection |
| Australia | AUD 1,000 | AUD 1,000 (low value imported goods GST applies to all imports for registered sellers) | |
| Japan | JPY 10,000 | JPY 10,000 | |
| Mexico | USD 50 | USD 50 | Higher thresholds with valid USMCA preferential treatment |
| Brazil | USD 50 (express courier; varies) | n/a single number; ICMS + IPI complex | Strict customs; ANVISA applies to many products |
| China | CNY 50 (personal) | n/a | Cross-border ecommerce comprehensive zones have separate rules |
| Switzerland | CHF 5 of duty / CHF 5 of VAT (small consignment rule) | varies | Effectively low |
| Norway | NOK 350 historically removed; VOEC for ≤NOK 3000 | VOEC seller-collects | |
| New Zealand | NZD 1,000 | n/a — GST applies via overseas vendor registration | |
| Singapore | SGD 400 GST relief; from 2023 GST applies to low-value imports | varies | |
| South Korea | USD 150 (USD 200 from US under KORUS) | n/a | |
| India | n/a meaningful (₹100 commercial; ₹5000 personal gift) | GST applies | High formality |
| UAE | AED 1,000 | AED 1,000 | Above triggers full clearance |
| Saudi Arabia | SAR 1,000 | varies | |

## Verification sources

- US: CBP / USITC HTSUS
- EU: TARIC database; member-state VAT pages
- UK: HMRC Trade Tariff
- Australia: ABF Tariff
- Other: WCO Member national customs pages

Rates and thresholds can change inside a 12-month window; pull the live source for any number that affects a pricing decision.
