# 素材工坊模板 (material_workshop.md)

## 概述

本文件定义素材工坊（Step 7）的输出格式和模板。所有模板基于策略方向和商品特征自动填充。

---

## A. 短视频分镜脚本模板

### 模板结构

分镜脚本按"黄金前3秒 + 中间展开 + 结尾转化"三段式结构组织，共 4-6 个镜头。

| 镜头 | 时长 | 画面描述 | 口播文案 | 运镜建议 | 字幕关键帧 |
|------|------|---------|---------|---------|-----------|
| 1 | 3s | 痛点场景或效果对比 | 黄金前3秒钩子 | 特写/推镜 | 核心卖点大字 |
| 2 | 5s | 产品展示/使用过程 | 产品介绍 | 中景 | 产品名+价格 |
| 3 | 5s | 效果展示/成分解析 | 差异化卖点 | 特写 | 关键数据（如"30天见效"） |
| 4 | 4s | 使用场景 | 场景化种草 | 全景 | 场景关键词 |
| 5 | 3s | 价格/福利展示 | 逼单收尾 | 手持产品 | "最后XX件" |

### 生成规则

1. **黄金前3秒**必须匹配策略方向：
   - 利益驱动："佣金30%的品，团长们看过来"
   - 品牌溢价："为什么贵妇都在用这个？"
   - 价格锚定："比XX便宜一半，效果一样！"
   - 认知教育："99%的人不知道洗发水要这样选"
   - 信任建设："实测30天，数据不会骗人"

2. 每个镜头的口播文案需标注语气标记：
   - `[惊讶]` `[神秘]` `[严肃]` `[兴奋]` `[诚恳]`

3. 字幕关键帧显示大字标题，便于后期剪辑定位

### 示例

商品：氨基酸洗发水，79元，佣金30%

| 镜头 | 时长 | 画面描述 | 口播文案 | 运镜建议 | 字幕关键帧 |
|------|------|---------|---------|---------|-----------|
| 1 | 3s | 头发干枯毛躁 vs 用完柔顺的对比特写 | `[惊讶]`你是不是还在用超市那种20块的洗发水？ | 特写推镜 | "你还在用20块的洗发水？" |
| 2 | 5s | 挤出洗发水展示泡沫绵密度 | 给你看看这个氨基酸洗发水，泡沫像奶油一样，洗一次头发直接从草变丝绸 | 中景 | "氨基酸配方 温和清洁" |
| 3 | 5s | 成分表特写 + 对比普通洗发水成分 | `[严肃]`没有SLS、没有硅油，纯氨基酸表活，pH5.5弱酸性 | 特写 | "0硅油 0硫酸盐 pH5.5" |
| 4 | 4s | 女生洗完头发飘逸的慢镜头 | 洗完以后头皮会呼吸，三天不洗头都不油 | 全景慢动作 | "三天不洗头也不油" |
| 5 | 3s | 手持产品，价格标签弹出 | `[兴奋]`现在下单只要79，还送旅行装，链接在评论区 | 手持特写 | "79元送旅行装" |

## B. 直播间贴片文案模板

### 价格牌文案

用于直播间贴片或 OBS 贴片，突出价格优势。

```
模板：[核心卖点]
      [原价] → [现价]
      [附加福利]
      [行动号召]

示例：氨基酸温和洗发
      ¥129 → ¥79
      买1送1旅行装
      点击小黄车立即抢购
```

### 卖点牌文案

3-5条核心卖点，每条 ≤8 字，按重要性排序。

```
1. [首要卖点，6-8字]
2. [次要卖点，6-8字]
3. [补充卖点，6-8字]
4. [技术卖点，6-8字]（如有）
5. [信任卖点，6-8字]（如有）
```

### 福利牌文案

```
模板：[促销类型]："[具体内容]"
      [限时/限量标识]
      [紧迫感文案]

示例：新品尝鲜
      "前100名送旅行装"
      限量100份，手慢无
```

### 信任牌文案

```
模板：[信任类型]：[具体内容]
      可选类型：检测报告 / 达人推荐 / 用户好评 / 品牌资质
```

## C. 商品详情页优化建议模板

### 标题优化

```
原标题：[用户提供的原标题]
优化后：[优化后标题，≤30字，含核心关键词+卖点]
标题公式：核心关键词 + 功效关键词 + 差异化卖点 + 目标人群
```

### 主图方向建议

| 主图序号 | 类型 | 方向描述 |
|---------|------|----------|
| 第1张 | 商品主体 | 白底正面+侧面双角度，突出产品形态 |
| 第2张 | 核心卖点 | 卖点可视化，文字不超过整个画面的20% |
| 第3张 | 使用场景 | 真人实拍使用过程，展示真实效果 |
| 第4张 | 成分/技术 | 成分表特写或技术原理图解 |
| 第5张 | 信任背书 | 检测报告/好评截图/对比效果 |

### 详情页结构

```
1. 首屏大图（800px）：商品场景图 + 核心卖点一句话
2. 痛点引入（1-2屏）：用户痛点场景 + 情绪共鸣
3. 解决方案（2-3屏）：产品如何解决痛点，图文结合
4. 成分/技术解析（2-3屏）：用通俗语言解释专业成分
5. 效果展示（1-2屏）：真人实测效果对比
6. 使用教程（1-2屏）：步骤化展示使用方法
7. 信任建设（1屏）：检测报告 + 资质 + 好评精选
8. FAQ预设（1屏）：常见购买顾虑 + 解答
9. 购买引导（1屏）：规格选择 + 优惠信息 + 立即购买
```

## D. 推广海报文案模板

### 社群转发海报

```
主标题（≤12字）：[核心卖点/促销信息]
副标题（≤20字）：[补充说明]
行动号召（≤8字）：[立即行动/马上抢/点我下单]
```

### 规格适配

| 比例 | 适用场景 | 注意事项 |
|------|---------|----------|
| 9:16 | 手机竖屏浏览 | 主标题在视觉中心上方1/3处 |
| 1:1 | 朋友圈九宫格 | 信息集中在画面中央 |

## 生成优先级

当用户只说"帮我出素材"时，默认按以下优先级生成：

1. **短视频分镜脚本**（最常用，可直接给达人/拍剪团队）
2. **直播间贴片文案**（直播带货场景刚需）
3. **推广海报文案**（社群转发高频需求）
4. **详情页优化建议**（长期优化需求，优先级较低）

用户可指定需要的素材类型，如"只要分镜脚本"或"只要直播贴片"。
