# 劳动仲裁全流程辅助技能

## 概述

本技能为普通劳动者提供劳动仲裁全流程辅助，覆盖以下核心环节：
1. **资料研判** — 评估现有证据是否充分，发现证据缺口
2. **资料整理** — 按仲裁请求生成结构化证据清单
3. **线上提交指引** — 指导用户通过当地线上平台提交仲裁申请
4. **调解策略** — 提供调解谈判策略和底线计算
5. **一键提交文档** — 自动生成仲裁申请书和证据清单文档

支持全国各地区（含省、自治区、直辖市及主要城市），内置地区差异参考。

---

## 工作流程

### 总流程

```
用户咨询 → 信息收集 → 资料研判 → 资料整理 → 生成文档 → 提交指引 → 调解策略
```

### 阶段一：信息收集

收集以下关键信息（按需逐项询问，避免一次性追问过多）：

**必采信息：**
1. 用户所在地区（省/市）— 用于确定管辖和地方规则
2. 用人单位全称 — 用于查询工商信息
3. 入职日期和离职日期 — 确定劳动关系存续期间
4. 离职/解除原因 — 确定仲裁请求类型
5. 月工资标准 — 用于计算各项金额

**按需采集信息（根据仲裁请求类型）：**
- 是否签订劳动合同 → 二倍工资请求
- 拖欠工资期间和金额 → 追索劳动报酬
- 加班情况（是否审批、加班时长） → 加班费请求
- 年休假情况 → 未休年休假工资
- 社保缴纳情况 → 社保争议
- 是否有书面解除通知 → 解除合法性判断

**采集方式：**
- 通过对话逐步收集
- 用户可上传劳动合同、工资条、解除通知等材料照片/截图
- 对用户上传的材料进行分析和信息提取

### 阶段二：资料研判

加载参考文档：`references/evidence_assessment.md`

执行研判流程：
1. 根据用户情况确定仲裁请求清单
2. 逐项匹配所需证据
3. 评估现有证据效力（充分/基本充分/不足）
4. 识别证据缺口，给出补充建议
5. 评估时效风险
6. 生成研判报告

**研判输出格式：**
- 仲裁请求清单及金额
- 证据匹配表（请求 → 法律依据 → 所需证据 → 现有证据 → 缺口）
- 风险评估（时效风险、证据风险、胜诉概率）
- 补充证据建议

**地区差异查询：** 加载 `references/regional_rules.md`，根据用户所在地区查询：
- 二倍工资时效规则
- 调解前置要求
- 举证期限和举证责任
- 裁判口径差异
- 计算基数差异

### 阶段三：资料整理

加载参考文档：`references/document_checklist.md`

1. 根据仲裁请求生成个性化证据清单
2. 标注每项证据的：是否已收集、获取方式、优先级
3. 对未收集的证据提供补充指引
4. 给出资料整理规范（编排顺序、电子证据固定方法）

**输出格式：** 可勾选的证据清单表格，用户可对照逐项准备。

### 阶段四：生成仲裁申请文档

使用脚本：`scripts/generate_arbitration_docs.py`

根据收集的信息，自动生成以下文档：
1. **仲裁申请书** — 包含当事人信息、仲裁请求、事实与理由、证据清单
2. **证据清单** — 结构化证据目录
3. **金额计算表** — 各项请求的金额计算明细

**脚本调用方式：**
```bash
python scripts/generate_arbitration_docs.py --output-dir <输出目录> [参数...]
```

或通过 JSON 配置文件传入案件信息：
```bash
python scripts/generate_arbitration_docs.py --config <案件信息JSON> --output-dir <输出目录>
```

脚本会生成 Markdown 格式的仲裁申请书，用户可进一步编辑或转换为 PDF/DOCX。

**参数说明：** 参见脚本文件头部文档字符串。

### 阶段五：线上提交指引

加载参考文档：`references/online_submission_guide.md`

1. 根据用户所在地区，匹配对应线上提交平台
2. 给出该地区平台名称、网址/App、注意事项
3. 指导用户完成注册、填表、上传材料、提交全流程
4. 如该地区不支持线上提交，提供邮寄提交方案

### 阶段六：调解策略

加载参考文档：`references/mediation_strategy.md`

1. 根据仲裁请求总金额和证据情况，计算建议底线金额
2. 分析对方可能立场和抗辩点
3. 提供谈判策略（开局、中盘、收尾）
4. 给出调解 vs 裁决的建议
5. 提供调解协议注意事项和常见陷阱提醒

## 关键规则与提醒

### 法律依据快速索引

| 仲裁请求 | 法律依据 |
|---------|---------|
| 确认劳动关系 | 劳社部发〔2005〕12号 |
| 拖欠工资 | 《劳动法》第50条、《劳动合同法》第30条 |
| 未签合同二倍工资 | 《劳动合同法》第82条 |
| 经济补偿金（N） | 《劳动合同法》第46条、第47条 |
| 违法解除赔偿金（2N） | 《劳动合同法》第48条、第87条 |
| 加班费 | 《劳动法》第44条 |
| 未休年休假工资 | 《职工带薪年休假条例》第5条 |
| 仲裁时效 | 《调解仲裁法》第27条 |

### 金额计算公式

```
经济补偿金 = 工作年限 × 月平均工资
  - 满1年 = 1个月工资
  - 满6个月不满1年 = 1个月工资
  - 不满6个月 = 0.5个月工资
  - 月工资 > 当地社平工资3倍 → 按3倍计算，年限最高12年

违法解除赔偿金 = 经济补偿金 × 2

二倍工资差额 = 月工资 × 未签合同月数（最长11个月）

加班费：
  - 工作日延时 = 时薪 × 1.5 × 加班小时
  - 休息日 = 日薪 × 2 × 加班天数
  - 法定节假日 = 日薪 × 3 × 加班天数
  - 时薪 = 月工资 ÷ 21.75 ÷ 8
  - 日薪 = 月工资 ÷ 21.75

未休年休假工资 = 日薪 × 200% × 未休天数（额外支付部分）
```

### 重要时效提醒
- 一般仲裁时效：1年（自知道权利被侵害之日起）
- 劳动报酬特殊时效：劳动关系存续期间不受限，终止后1年内提出
- 二倍工资时效：各地不同，务必查询 `references/regional_rules.md`
- 不予受理后起诉期限：15日

### 使用注意事项
1. 本技能提供的信息仅供参考，不构成法律意见
2. 各地政策可能更新，提交前建议拨打12333确认
3. 复杂案件建议咨询专业律师或申请法律援助（12348）
4. 涉及集体争议或重大金额的案件，建议寻求专业法律帮助
