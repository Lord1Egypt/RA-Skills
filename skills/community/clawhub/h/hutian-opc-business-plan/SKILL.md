# 商业计划书生成技能

## 技能概述
专业的商业计划书生成工具，采用**互动问答模式**，通过多轮对话系统性地收集信息，逐步完善并生成结构完整、数据翔实、逻辑严密的PPT商业计划书。适用于融资路演、技术转化、市场准入规划、企业战略输出等场景。

---

## 核心方法：互动问答模式

### 工作流程
```
┌─────────────────────────────────────────────────────────────────────┐
│                      互动问答生成流程                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌───────────┐    ┌───────────┐    ┌───────────┐    ┌───────────┐ │
│   │  用户输入  │ -> │  AI追问   │ -> │  信息完善 │ -> │  PPT生成 │ │
│   │  初步信息  │    │  引导补充  │    │  多轮迭代  │    │  交付成果 │ │
│   └───────────┘    └───────────┘    └───────────┘    └───────────┘ │
│        │                  │               │                │       │
│        │    ┌─────────────┴──────────────┘                │       │
│        │    │                                                │       │
│        ▼    ▼                                                ▼       │
│   ┌─────────────────┐                              ┌─────────────┐ │
│   │  商业模式雏形   │                              │  PPT计划书  │ │
│   │  简单梳理描述   │                              │  15-20页   │ │
│   └─────────────────┘                              └─────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 交互原则
1. **循序渐进**：从核心到外围逐步深入
2. **引导式提问**：每个问题都有明确目的
3. **验证确认**：确保理解准确
4. **结构化输出**：最终生成标准化PPT

## 必备要素清单（五大模块）

生成商业计划书必须包含以下五大模块，**不能缺失任何一项**：

### 模块一：市场分析 ⭐ 必须包含

## 参考问题模板

### 市场相关问题

**市场规模**
```
□ 您的目标市场有多大？（年市场规模约多少亿/万）
□ 市场过去3年的增长率是多少？
□ 未来3-5年的增长预期如何？
□ 推动市场增长的核心因素是什么？
```

**竞争格局**
```
□ 您的直接竞争对手有哪些？（列出2-3家）
□ 您的间接竞争对手有哪些？
□ 与竞争对手相比，您的核心差异是什么？
□ 您打算如何建立竞争壁垒？
```

**目标客户**
```
□ 您的目标客户是谁？（B端/C端，企业规模，用户特征）
□ 客户的核心痛点是什么？
□ 客户目前的解决方案是什么？有什么不满？
□ 客户愿意为您的方案支付多少？
□ 客户决策周期多长？如何触达客户？
```

**市场进入**
```
□ 您计划如何进入市场？（直销/代理/线上等）
□ 预计多久可以实现首批付费客户？
□ 获客成本预计多少？
□ 如何验证市场需求？
```

### 财务相关问题

**收入模型**
```
□ 您的收入来源是什么？（产品销售/订阅/服务/其他）
□ 产品/服务的定价是多少？
□ 客户的生命周期价值（LTV）预计多少？
□ 您的单位经济模型（LTV/CAC）是多少？
□ 什么时候可以实现盈亏平衡？
```

**成本结构**
```
□ 您的主要成本有哪些？（研发/营销/人力/运营）
□ 预计月 burn rate 是多少？
□ 毛利率目标是多少？
□ 成本优化空间在哪里？
```

**财务预测**
```
□ 未来3年的收入预期是多少？
□ 未来3年的利润预期是多少？
□ 预计什么时候可以实现盈利？
□ 您的财务预测基于哪些关键假设？
```

---

### 团队相关问题

**核心团队**
```
□ 创始人/CEO的背景是什么？
□ 核心合伙人的背景是什么？
□ 团队最擅长的能力是什么？
□ 团队最需要补足的能力是什么？
```

**人才需求**
```
□ 您还需要招聘哪些关键人才？
□ 预计招聘多少人？
□ 招聘计划如何配合业务发展？
```

**股权结构**
```
□ 创始人股权比例是多少？
□ 是否有期权池？多大比例？
□ 是否有投资人？股权比例多少？
```

---

### 风险相关问题

**核心风险**
```
□ 您认为项目最大的风险是什么？
□ 如果竞争对手强势入场，您如何应对？
□ 如果核心技术人员离职，您如何应对？
□ 如果市场需求不及预期，您如何应对？
```

**风险应对**
```
□ 您有哪些应对风险的预案？
□ 您的护城河是什么？
□ 最坏情况下，您的底线是什么？
```

---

### 资源与规划相关问题

**资金需求**
```
□ 您计划融资多少？
□ 本轮融资的估值预期是多少？
□ 资金的主要用途是什么？（研发/营销/团队/运营比例）
□ 预计资金可以支撑多久？
```

**里程碑规划**
```
□ 3个月内要实现什么目标？
□ 6个月内要实现什么目标？
□ 12个月内要实现什么目标？
□ 3年后项目应该发展成什么样？
```

**单位经济模型**
```
CAC（客户获取成本） = 营销总投入 / 新增付费客户数

LTV（客户生命周期价值） = (客户年均收入 × 毛利率) × 平均客户生命周期

LTV/CAC > 3.0 为优秀
```

## 输出：PPT商业计划书结构

### 标准PPT结构（15-20页）

**第一部分：封面与执行摘要（2页）**
- 第1页：封面（项目名称、行业、融资轮次、日期）
- 第2页：执行摘要（一页讲清楚：痛点、方案、市场、团队、融资需求）

**第二部分：问题与方案（2-3页）**
- 第3页：目标市场痛点分析
- 第4页：现有解决方案的不足
- 第5页：我们的解决方案与差异化优势

**第三部分：市场分析（2-3页）**
- 第6页：市场规模（TAM/SAM/SOM）
- 第7页：竞争格局分析
- 第8页：目标客户画像与进入策略

**第四部分：产品与可行性（2-3页）**
- 第9页：产品/服务介绍
- 第10页：技术可行性评估
- 第11页：商业模式画布

**第五部分：财务分析（2-3页）**
- 第12页：3-5年财务预测
- 第13页：单位经济模型（LTV/CAC）
- 第14页：融资规划与资金用途

**第六部分：团队与风控（2-3页）**
- 第15页：核心团队介绍
- 第16页：发展里程碑
- 第17页：风险分析与应对策略

**第七部分：附录（1-2页）**
- 第18页：详细财务数据
- 第19页：相关资质/专利/合同

## 使用指南

### 第一步：初始输入
请提供您项目的初步信息，可以是：
- 一段项目描述
- 现有的BP草稿
- 产品介绍文档
- 任何形式的项目概述

### 第二步：互动完善
AI将按照必备要素清单，逐个模块向您收集关键信息。

### 第三步：确认校验
信息收集完成后，AI将梳理完整的商业计划书框架，与您确认是否需要调整。

### 第四步：PPT生成
确认后，使用ppt-generator技能生成专业的PPT商业计划书。
