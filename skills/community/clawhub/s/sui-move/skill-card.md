Sui blockchain and Move smart contract development. Use when the user asks about Sui, Move language, smart contracts, objects, transactions, or blockchain development on Sui. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to answer Sui and Move questions, locate relevant documentation, draft smart contract examples, and prepare Sui CLI commands for building, testing, publishing, and calling Move packages. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Example publish and call commands can create real blockchain transactions if run against a live Sui network. <br>
Mitigation: Verify wallet, network, gas budget, package IDs, arguments, and contract code before running Sui client commands; use devnet or testnet first for experiments. <br>
Risk: Setup clones and updates public documentation repositories, so referenced material can change over time. <br>
Mitigation: Run setup intentionally and review the cloned documentation sources before relying on generated guidance for contract behavior or deployment decisions. <br>
