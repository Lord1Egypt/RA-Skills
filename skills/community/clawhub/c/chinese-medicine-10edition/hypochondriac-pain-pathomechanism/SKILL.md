# Hypochondriac Pain Pathomechanism

## When to Use
当患者出现**一侧或两侧胁肋部疼痛**，且已排除外伤性疼痛及非胁肋部位疼痛时，使用本技能进行病机辨识。需结合疼痛性质、舌脉及伴随症状综合判断。

## Execution Steps

1. **判断疼痛性质以区分病机类型**：
   - **“不通则痛”（实证）**：由气滞、血瘀、湿热等邪实阻滞肝络所致
     - **胀痛**：多属肝气郁结，疼痛游走不定，常因情志波动诱发
     - **刺痛**：多属瘀血停着，痛有定处，入夜加重，舌质紫暗或有瘀斑
     - **剧烈疼痛伴口苦、尿黄**：多属肝胆湿热
   - **“不荣则痛”（虚证）**：因肝阴不足，脉络失养所致
     - **隐痛**：绵绵不休，遇劳加重，常伴口干咽燥、头晕目眩、舌红少苔

2. **结合病因辅助判断**：
   - 实证多由情志抑郁、外感湿热或强力负伤引起
   - 虚证多因久病耗伤、劳欲过度致精血亏虚

3. **对应治疗原则**：
   - 实证：疏导祛邪以畅通（如疏肝理气、活血化瘀、清利湿热）
   - 虚证：滋养不足以荣通（如滋阴柔肝）

4. **注意虚实夹杂情况**：
   - 临床上常见气郁日久致血瘀，或阴虚兼气滞等复合病机，需综合分析，不可机械割裂。

> 提示：本技能输出为病机类型判断（实证/虚证/虚实夹杂），用于指导后续方药选择，不直接提供处方。
