> 此文件为公司科创KPI考核标准模板，用户应根据自身企业实际情况修改各项指标。

## 二、线索筛选匹配规则

### 自动匹配逻辑

```
每条采集线索 → 标题+摘要关键字匹配KPI维度 → 计算匹配度得分 → 判定优先级
```

### 关键词词库

**科技奖匹配词**: 科技奖、自然科学奖、技术发明奖、科技进步奖、专利奖、创新争先

**资质认定匹配词**: 高新技术企业、专精特新、小巨人、单项冠军、企业技术中心、重点实验室、工程研究中心、新型研发机构、科技型中小企业、瞪羚企业、独角兽、绿色工厂、工业设计中心

**项目申报匹配词**: 重点研发计划、科技重大专项、揭榜挂帅、攻关计划、创新基金、引导资金、技术改造、数字化转型、智能制造

**资金补贴匹配词**: 补贴、补助、奖励、资助、贴息、创新券、税收优惠、加计扣除

**人才匹配词**: 人才计划、领军人才、创新团队、博士后、院士工作站、海智计划

**标准匹配词**: 标准征集、标准化试点、团体标准、行业标准、国家标准、标准立项

### 无效信息过滤规则

以下信息自动过滤（低优先级）:
- 纯新闻类无申报/认定窗口的信息
- 已过截止日期的申报通知（保留30天归档）
- 与本企业行业/技术领域完全不相关的内容
- 重复信息（基于URL去重）

## 三、优先级评分算法

```
优先级得分 = KPI权重 × 时效因子 × 匹配度

时效因子:
- 截止日期<15天: 1.5（紧急）
- 截止日期15-45天: 1.0（正常）
- 截止日期>45天或无截止日: 0.7（宽松）
- 已过截止日: 0（自动过滤）

匹配度:
- 标题命中多个KPI词: 1.0
- 标题命中1个KPI词: 0.8
- 摘要命中KPI词: 0.5
- 未命中: 0.2（列入观察）
```

## 四、输出优先级判定

| 总分区间 | 优先级 | 标记 |
|---------|--------|------|
| ≥15 | ⭐⭐⭐ 紧急 | 立即处理 |
| 8-14 | ⭐⭐ 重要 | 本周内关注 |
| 3-7 | ⭐ 关注 | 月度汇总 |
| <3 | 归档 | 信息备查 |
