## Use Case: <br>
External users use this skill as a productivity coach for energy-aware task triage, calendar-aware planning, inbox capture, goal alignment, and daily review. It helps the agent produce prioritized recommendations, recovery guidance, reminder proposals, and memory updates after user consent. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may save goals, tasks, calendar details, inbox notes, preferences, and learned rules in OpenClaw memory. <br>
Mitigation: Install only if comfortable with this local persistence, grant memory-write consent deliberately, and periodically review or delete MEMORY.md and daily notes. <br>
Risk: Reminder scheduling can affect the user's calendar or notification workflow. <br>
Mitigation: Confirm each reminder before enabling it and review reminder preferences during setup or daily review. <br>
Risk: Course links and social-support prompts may be promotional. <br>
Mitigation: Treat course recommendations and social-support prompts as optional resources and ignore them unless they match the user's goals. <br>
