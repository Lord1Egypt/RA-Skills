# 客户评分模型

## MatchGPT评分体系

### 评分维度（总分10分）

#### 1. 产品匹配度 (0-3分)
| 评分 | 条件 |
|-----|------|
| 3分 | 完全匹配我方产品线，无缝对接 |
| 2分 | 主要产品匹配，部分需调整 |
| 1分 | 小部分匹配，需较大产品调整 |
| 0分 | 产品不匹配 |

#### 2. 采购规模 (0-3分)
| 评分 | 年采购额条件 |
|-----|--------------|
| 3分 | >$500,000 |
| 2分 | $100,000-$500,000 |
| 1分 | $50,000-$100,000 |
| 0分 | <$50,000 |

#### 3. 竞品依赖度 (0-2分)
| 评分 | 条件 |
|-----|------|
| 2分 | 当前供应商单一，我方有明显优势 |
| 1分 | 多供应商，我方有一定竞争力 |
| 0分 | 现有供应商稳定，我方优势不明显 |

#### 4. 进入机会 (0-2分)
| 评分 | 条件 |
|-----|------|
| 2分 | 采购周期窗口正好，客户有明确需求 |
| 1分 | 有一定机会，需培育 |
| 0分 | 时机不佳或信息不足 |

---

## 综合评分标准

| 总分 | 等级 | 建议行动 |
|-----|------|---------|
| 8-10分 | A级（高优先级）| 立即跟进，重点开发 |
| 6-7分 | B级（次优先级）| 积极跟进，保持联系 |
| 4-5分 | C级（观察）| 放入资源池，持续关注 |
| <4分 | D级（低优先级）| 暂不开发 |

---

## 评分流程

1. **数据收集**：获取客户基本信息和交易记录
2. **初步筛选**：过滤明显不匹配客户
3. **四维评分**：按上述维度逐一打分
4. **综合计算**：汇总得分，确定等级
5. **人工复核**：高风险客户需人工确认
