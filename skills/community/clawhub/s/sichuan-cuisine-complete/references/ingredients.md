# 川菜食材与调料百科

## 一、川菜灵魂调料

### 郫县豆瓣
- **地位**：川菜之魂，没有之一
- **原料**：蚕豆+二荆条辣椒+面粉+盐，自然发酵3年以上
- **推荐品牌**：鹃城牌（首选）、丹丹牌
- **使用铁律**：必须剁细、必须小火炒出红油（油温≤120°C）
- **保存**：冰箱冷藏，每次取用干净勺，表面可淋一层菜籽油

### 花椒
- **红花椒（南路椒）**：麻味浓烈，适合麻辣菜——汉源、茂县产最佳
- **藤椒（青花椒）**：清香麻，适合椒麻、藤椒味——洪雅产最佳
- **使用**：整粒用（水煮类）、磨粉用（麻婆豆腐）、炼油用（花椒油）
- **挑选**：开口、籽少、气味浓烈
- **保存**：密封避光，半年内用完

### 干辣椒
- **二荆条**：微辣浓香，红油上色——川菜最常用
- **朝天椒**：辣度高，提辣用
- **灯笼椒**：香为主，辣度低——红油增香
- **七星椒**：极辣，少量使用
- **常用配比**：二荆条:朝天椒 = 2:1 ~ 3:1

### 泡辣椒（泡海椒）
- **原料**：二荆条鲜辣椒，泡菜坛泡制
- **用途**：鱼香味核心、家常味辅助
- **使用**：去蒂去籽剁茸，炒出红油
- **关键**：必须剁茸！整只泡辣椒出不了鱼香味

### 泡姜
- **用途**：酸菜鱼、水煮系列、泡椒系列
- **作用**：去腥增香，提供酸辣底味

### 豆豉
- **推荐**：永川豆豉（首选）、太和豆豉
- **使用**：回锅肉、麻婆豆腐、盐煎肉
- **要点**：稍剁碎，与豆瓣同炒

### 甜面酱
- **推荐**：鹃城/六必居
- **使用**：回锅肉、京酱肉丝、酱爆菜
- **要点**：先用少量料酒或油澥开

### 芝麻酱
- **使用**：怪味、麻酱味、担担面
- **要点**：必须先用香油澥开！直接加汤水澥不开

### 醋
- **保宁醋（阆中）**：川菜首选，麸醋，药曲发酵，酸中带香
- **香醋（镇江）**：可替代，风味不同
- **用途**：鱼香味、酸辣味、宫保荔枝味

### 复制酱油（红酱油）
- **做法**：酱油+糖+香料熬制浓缩
- **用途**：蒜泥白肉、红油抄手等冷菜
- **地位**：川菜凉菜灵魂

### 醪糟（甜酒酿）
- **用途**：去腥、增甜、提鲜
- **用法**：取汁不用米，与其他调料同烹

## 二、食材分类

### 猪肉部位与用途
| 部位 | 川菜用途 | 代表菜 |
|------|---------|--------|
| 二刀坐臀肉 | 回锅专用 | 回锅肉、盐煎肉 |
| 五花肉 | 烧、蒸、白肉 | 蒜泥白肉、红烧肉 |
| 里脊 | 切丝/片 | 鱼香肉丝、水煮肉片 |
| 猪耳 | 卤制冷菜 | 红油耳片 |
| 猪肚 | 卤/拌/烧 | 椒麻肚丝、大蒜肚条 |
| 排骨 | 糖醋/蒜香/粉蒸 | 糖醋排骨、蒜香排骨 |

### 禽类
- **三黄鸡**：口水鸡、椒麻鸡、白切鸡（嫩，适合凉菜）
- **土鸡**：炖汤、红烧（肉质紧实）
- **鸡腿肉**：宫保鸡丁、辣子鸡

### 水产
- **草鱼**：酸菜鱼、水煮鱼（肉嫩但刺多）
- **乌鱼**：酸菜鱼、沸腾鱼（肉Q无小刺，首选）
- **鳝鱼**：水煮鳝段、大蒜烧鳝段
- **牛蛙**：水煮牛蛙、泡椒牛蛙

### 蔬菜搭配常识
- **蒜苗**：回锅肉搭档（冬季最佳，夏季可用青椒+洋葱代替）
- **莴笋**：鱼香肉丝、水煮牛肉垫底
- **芹菜**：水煮牛肉垫底、芹菜炒牛肉
- **莲白（圆白菜）**：炝炒莲白、回锅肉配菜
- **豇豆**：姜汁豇豆、干煸豆角
- **土豆**：狼牙土豆、干锅土豆片

## 三、常见失误与避免

| 问题 | 原因 | 解决 |
|------|------|------|
| 回锅肉不卷 | 肉煮太熟/太生 | 煮八成熟，筷子能插但有阻力 |
| 豆瓣炒不红 | 油温太高/没剁细 | ≤120°C小火慢炒，豆瓣必须剁 |
| 水煮鱼/肉发硬 | 煮太久 | 肉片下锅30秒即可 |
| 宫保鸡丁发苦 | 辣椒炸过头 | 棕红色即可，不能发黑 |
| 鱼香味不对 | 糖醋比例错/没剁泡辣椒 | 糖醋≈1:1，泡辣椒必须剁茸 |
| 麻婆豆腐散 | 翻炒了 | 只能用锅铲背面推，不能翻 |
| 蒜泥白肉不够香 | 蒜不是现捣 | 必须现捣！不能隔夜 |
| 红油不红 | 油温不对/没用二荆条 | 150°C泼油，二荆条为主 |

## 四、家常厨房常备清单

**必备干货**：郫县豆瓣、花椒（红）、干辣椒、永川豆豉、八角、桂皮、香叶

**必备液体**：生抽、老抽、保宁醋/香醋、料酒、芝麻油、菜籽油、藤椒油

**必备鲜料**：大蒜、生姜、小葱、泡辣椒（可买瓶装）、泡姜

**进阶常备**：甜面酱、芝麻酱、冰糖、醪糟、白胡椒粉、辣椒面（自制红油用）

**香料包**（炖卤用）：八角、桂皮、丁香、小茴香、甘草、草果、砂仁、白芷
