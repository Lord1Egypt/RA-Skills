# 政府公文写作知识库

> 本知识库提供公文写作所需的全部背景知识，供 AI 在编写政府材料时引用。

---

## 一、公文种类详解

### 1.1 法定公文（党政机关 15 种）

依据《党政机关公文处理工作条例》（中办发〔2012〕14 号）：

| 序号 | 文种 | 用途 | 行文方向 | 适用场景 |
|:---:|:---|:---|:---:|:---|
| 1 | **决议** | 经会议讨论通过的重大决策事项 | 下行 | 党委全会通过决定 |
| 2 | **决定** | 对重要事项作出决策和部署、奖惩有关单位和人员 | 下行 | 表彰先进、重大决策 |
| 3 | **命令（令）** | 公布行政法规和规章、宣布施行重大强制性措施 | 下行 | 政府令、嘉奖令 |
| 4 | **公报** | 公布重要决定或重大事项 | 下行/周知 | 会议公报、统计公报 |
| 5 | **公告** | 向国内外宣布重要事项或法定事项 | 周知 | 人事任免、重大活动 |
| 6 | **通告** | 公布应遵守或周知的事项 | 周知 | 交通管制、区域管理 |
| 7 | **意见** | 对重要问题提出见解和处理办法 | 多向 | 指导意见、实施意见 |
| 8 | **通知** | 发布、传达要求执行和有关单位周知或执行的事项 | 下行 | 会议通知、工作部署 |
| 9 | **通报** | 表彰先进、批评错误、传达重要精神和情况 | 下行 | 情况通报、表扬批评 |
| 10 | **报告** | 向上级汇报工作、反映情况、回复上级询问 | 上行 | 工作报告、专题报告 |
| 11 | **请示** | 向上级请求指示、批准 | 上行 | 申请资金、请求批复 |
| 12 | **批复** | 答复下级请示事项 | 下行 | 审批答复 |
| 13 | **议案** | 向人大提出审议事项 | 平行 | 政府向人大提出议案 |
| 14 | **函** | 不相隶属机关商洽工作、询问和答复问题 | 平行 | 商洽函、复函 |
| 15 | **纪要** | 记载会议主要情况和议定事项 | 内部 | 会议议定事项 |

> ⚠️ **关键规则**：
> - **请示必须一文一事**，不得一文多请
> - **报告不得夹带请示事项**
> - **上行文必须标注签发人**
> - **函和意见可用于多种行文方向**

### 1.2 常用非公文类政府文书

| 文种 | 用途 | 特征 |
|:---|:---|:---|
| **讲话稿** | 领导在会议、活动上的正式发言 | 有称谓、有鼓动性、段落间有过渡 |
| **政府工作报告** | 政府向人大报告工作 | 法定程序、庄重严谨、高度概括 |
| **调研报告** | 反映调研情况和成果 | 有数据有分析、对策建议务实 |
| **述职报告** | 向组织报告履职情况 | 个人化视角、实事求是 |
| **工作总结** | 总结阶段工作成效和经验 | 成绩+问题+计划 |
| **实施方案** | 对某项工作的具体安排 | 目标+任务+措施+时限+责任 |
| **信息简报** | 快速反映情况、交流经验 | 短小精悍、一事一报 |
| **督查专报** | 上级交办事项落实情况反馈 | 结果导向、时限明确 |
| **主持词** | 会议主持人的串场词 | 承上启下、简洁明快 |
| **致辞/贺信** | 重要场合的正式致辞 | 礼仪性、热情得体 |

## 二、公文格式规范

### 2.1 版面要求（GB/T 9704-2012）

| 要素 | 标准 |
|:---|:---|
| 用纸 | A4 型纸（210mm × 297mm） |
| 版心尺寸 | 156mm × 225mm（不含页码） |
| 每面行数 | 22 行 |
| 每行字数 | 28 字 |
| 行间距 | 固定值 28 磅 |
| 装订 | 左侧装订 |

### 2.2 字体字号规范

| 要素 | 字体 | 字号 | 说明 |
|:---|:---|:---:|:---|
| 标题 | 小标宋体 | 二号 | 居中，可分行 |
| 主送机关 | 仿宋体 | 三号 | 顶格，冒号后换行 |
| 正文 | 仿宋体 | 三号 | 首行缩进 2 字符 |
| 一级标题（一、） | 黑体 | 三号 | 独立成行 |
| 二级标题（（一）） | 楷体 | 三号 | 可独立成行或后接正文 |
| 三级标题（1.） | 仿宋体 | 三号 | 可加粗 |
| 附件说明 | 仿宋体 | 三号 | 正文下空一行 |
| 发文机关署名 | 仿宋体 | 三号 | 居右 |
| 成文日期 | 仿宋体 | 三号 | 用阿拉伯数字 |

### 2.3 层级序号

```
第一层：一、二、三、……（用顿号）
第二层：（一）（二）（三）……（不加标点）
第三层：1. 2. 3. ……（用下脚点）
第四层：（1）（2）（3）……（不加标点）
第五层：① ② ③ ……（用于注释）
```

> ⚠️ **规则**：层级不可跳用。如只有两层，则用一、和（一）；如只有三层，则用一、、（一）、1.

### 2.4 公文要素排序

```
份号（涉密公文）
密级和保密期限（涉密公文）
紧急程度（特急/加急）
发文机关标志（红头）
发文字号
签发人（上行文）
标题
主送机关
正文
附件说明
发文机关署名（加盖印章）
成文日期
附注（如：此件公开发布）
附件
抄送机关
印发机关和印发日期
页码
```

## 三、公文用语规范

### 3.1 公文固定表述词汇

| 场景 | 标准用语 | 说明 |
|:---|:---|:---|
| 发文依据 | "根据……""依据……""按照……" | 引出政策/法律依据 |
| 发文目的 | "为……""为了……" | 说明发文目的 |
| 请求指示 | "妥否，请批示" | 请示结尾标准用语 |
| 请求批准 | "请予审批""请批准" | 请示资金/事项 |
| 请求审议 | "请予审议" | 政府工作报告结尾 |
| 报告结束 | "特此报告""专此报告" | 报告结尾 |
| 通知执行 | "请认真贯彻执行""请结合实际抓好落实" | 通知结尾 |
| 函 | "专此函达""特此函复""盼复" | 函的开头/结尾 |
| 批复 | "现批复如下" | 批复标准开头 |
| 答复 | "经研究，现答复如下" | 答复性文书开头 |
| 时间表述 | "自……之日起施行" | 生效日期 |
| 范围 | "……，结合我市实际" | 因地制宜 |

### 3.2 18 类公文禁用/慎用词清单

**禁用词（不建议在正式公文中出现）：**

| 类别 | 禁用示例 | 推荐替换 |
|:---|:---|:---|
| 模糊程度词 | "非常""十分""极其" | 用数据或事实替代 |
| 主观评价词 | "我们认为""我觉得" | 用"经研究""会议认为" |
| 夸张修辞 | "顛覆性""革命性""划时代" | 用实际成效描述 |
| 网络用语 | "赋能（特定场景可用）""抓手（慎用）" | 用具体措施描述 |
| 空洞套话 | "在……的正确领导下"（过度使用） | 适度使用，避免每段都用 |
| 绝对化表述 | "全部""所有""无一例外" | 用"绝大多数""基本实现" |
| 含糊不清 | "大约""左右""可能"（数据场景禁用） | 用确切数字或"预计" |
| 命令式口吻 | "必须""严禁"（非处罚性文件） | 用"应当""不得" |

**慎用词（公共场合注意）：**

| 词汇 | 慎用说明 | 建议 |
|:---|:---|:---|
| 赋能 | 被过度使用，尽量具体化 | "提供技术支撑""提升能力" |
| 倒逼 | 带有强制意味 | "推动""促进" |
| 出清 | 经济领域可用，其他领域慎用 | "处置""化解" |
| 杀手锏 | 战争隐喻，不适用于政府文件 | "关键举措""有力措施" |
| 最后一公里 | 被过度使用 | "落实环节""末端执行" |

### 3.3 政治术语规范

**中央精神引用规范：**

- 习近平新时代中国特色社会主义思想 → 必须是完整表述，不可简写
- 党的二十大 → 全称为"中国共产党第二十次全国代表大会"
- 二十届三中全会 → 首次出现需写全称
- "两个确立" → 必须有引号
- "两个维护" → 必须有引号
- "四个意识" → 必须有引号
- "四个自信" → 必须有引号
- 新发展理念 → 创新、协调、绿色、开放、共享
- 高质量发展 → 是全面建设社会主义现代化国家的首要任务

**常见政治表述搭配：**

| 搭配 | 使用场景 |
|:---|:---|
| 深入学习贯彻 XXX 精神 | 所有政策文件开头 |
| 深刻领悟"两个确立"的决定性意义 | 重要会议讲话 |
| 增强"四个意识"、坚定"四个自信"、做到"两个维护" | 党建工作、重要会议 |
| 完整、准确、全面贯彻新发展理念 | 经济发展类文件 |
| 着力推动高质量发展 | 经济/发展类文件 |
| 统筹发展和安全 | 综合性工作部署 |
| 坚持以人民为中心的发展思想 | 民生类文件 |
| 全面从严治党 | 党建/廉政类文件 |

> ⚠️ **引用原则**：
> 1. 政治表述必须**原样引用**，不可自行概括
> 2. 首次出现用全称，后文可用规范简称
> 3. 引用最新会议和文件精神，时效性很重要
> 4. 同一文件引用中央精神 **2-3 次**足够，不宜过多

## 四、不同文种的核心写作规则

### 4.1 政府工作报告

| 要素 | 要求 |
|:---|:---|
| 结构 | 工作回顾 + 今年安排 + 政府自身建设（三板块） |
| 数据 | 全部用统计口径数据，每个板块至少配 3-5 个数字 |
| 问题 | 问题部分必须客观，篇幅控制在全文 10%-15% |
| 部署 | 目标+举措+责任，可量化 |
| 语气 | 庄重、负责、朴实 |
| 篇幅 | 县级 8000-10000 字，市级 12000-18000 字 |

### 4.2 领导讲话稿

| 要素 | 要求 |
|:---|:---|
| 开头 | 称谓（同志们：）+ 会议背景说明 |
| 结构 | 成绩肯定 → 形势分析 → 重点任务 → 落实要求 |
| 风格 | 有气势、有文采、有号召力 |
| 篇幅 | 大型会议 4000-6000 字，专题会 2000-3000 字 |
| 语速 | 按 200-240 字/分钟估算讲话时长 |
| 过渡 | 段落间有承上启下的过渡语 |

### 4.3 政策文件（实施意见/管理办法）

| 要素 | 要求 |
|:---|:---|
| 标题 | "关于 XXX 的实施意见/管理办法/若干措施" |
| 开头 | "为贯彻落实……文件精神，结合……实际，现提出如下意见" |
| 结构 | 总体要求 + 主要任务 + 保障措施 |
| 条款 | 每条有具体举措、责任单位、完成时限（"颗粒度"要细） |
| 表述 | 可使用"鼓励""支持""引导"等引导性措辞 |
| 量化 | 尽可能量化（目标、比例、时限） |

### 4.4 调研报告

| 要素 | 要求 |
|:---|:---|
| 开头 | 调研背景 + 调研方式 + 调研对象 |
| 结构 | 成效→问题→原因→建议（四段式） |
| 数据 | 调研数据（问卷/走访/座谈）为主要支撑 |
| 案例 | 必须有典型案例或典型做法 |
| 问题 | 实事求实，不回避、不粉饰 |
| 建议 | 必须有针对性和可操作性，标注建议落实单位 |
| 篇幅 | 3000-8000 字 |

### 4.5 请示

| 要素 | 要求 |
|:---|:---|
| 核心规则 | **一文一事** |
| 开头 | 说明请示背景和原因 |
| 理由 | 充分、合理、有政策依据 |
| 内容 | 明确请示事项（资金数额/事项范围/时限） |
| 结尾 | "妥否，请批示"或"妥否，请批复" |
| 联系人 | 文末标注联系人和电话 |
| 签发人 | 上行文必须标注签发人 |

## 五、政府层级与行文关系

### 5.1 行文关系类型

| 类型 | 说明 | 适用文种 |
|:---|:---|:---|
| **上行文** | 下级→上级 | 请示、报告、意见（上行的） |
| **下行文** | 上级→下级 | 通知、通报、批复、决定、命令 |
| **平行文** | 同级/不相隶属 | 函、意见 |
| **泛行文** | 不受行文方向限制 | 公告、通告、公报 |

### 5.2 各级政府的用词差异

| 层级 | 用词特点 | 政策站位 |
|:---|:---|:---|
| **省级及以上** | 宏观、全局性、战略性强 | 全国/全省视野，强调顶层设计 |
| **地市级** | 承上启下，既有部署也有落实 | 省+本地实际结合，强调执行 |
| **区县级** | 具体、可操作、贴合基层 | 强调落地、特色、民生 |
| **乡镇/街道级** | 极简、直接、行动导向 | 聚焦具体任务完成 |

## 六、政策引用规范

### 6.1 引用层级原则

```
层级一：中央精神（最高）
    ↓
层级二：国家法律法规/国务院文件
    ↓
层级三：省级政策/规章
    ↓
层级四：市级配套文件
    ↓
层级五：本区/本县实际情况
```

> - 层级越高越不可违背——下级文件必须与上级政策一致
> - 引用时通常从高到低，体现政策的连贯性和依据的充分性

### 6.2 常见政策引用速查

**中央通用政策：**

| 政策名称 | 核心要点 | 引用场景 |
|:---|:---|:---|
| 党的二十大报告 | 中国式现代化、高质量发展、共同富裕 | 所有重要文件开头 |
| 二十届三中全会精神（2024） | 进一步全面深化改革、推进中国式现代化 | 改革类、发展类文件 |
| 中央经济工作会议精神 | 稳中求进、以进促稳、先立后破 | 经济工作会议、发展类文件 |
| "十四五"规划纲要 | 各领域五年目标任务 | 所有规划类文件 |
| 新发展理念 | 创新、协调、绿色、开放、共享 | 发展改革类文件 |

**重点领域政策：**

| 领域 | 典型政策 |
|:---|:---|
| 经济/营商环境 | 《优化营商环境条例》、促进民营经济发展壮大意见 |
| 数字政府/数据 | 《数字中国建设整体布局规划》《关于加强数字政府建设的指导意见》 |
| 基层治理 | 《关于加强基层治理体系和治理能力现代化建设的意见》 |
| 乡村振兴 | 乡村振兴促进法、乡村建设行动实施方案 |
| 生态环境 | 生态文明体制改革总体方案、双碳目标相关文件 |
| 安全生产 | 安全生产法、安全生产十五条硬措施 |
| 教育 | 教育强国建设规划纲要、双减政策 |
| 医疗/卫生 | 健康中国 2030 规划纲要 |

### 6.3 政策引用规范

| 规则 | 说明 |
|:---|:---|
| 引最新 | 同一事项有新旧文件，引用最新的 |
| 引最高层级 | 能引中央文件不引省级，能引省级不引市级 |
| 引最相关 | 与写作主题直接相关的政策为首选 |
| 引落地文件 | 上接中央精神，下接地方法规，体现政策贯通 |
| 适度引用 | 一般文件引用 2-5 个，过多反而显得堆砌 |

> ⚠️ **不允许**自行编造政策名称或文件号。如果不知道具体文件号，用"根据国家有关政策要求"等模糊引用，**切忌**伪造发文号。

## 七、常见写作问题与避坑

### 7.1 十二类常见错误

| 序号 | 错误类型 | 错误示例 | 正确做法 |
|:---:|:---|:---|:---|
| 1 | 文种误用 | 请示中夹带报告内容 | 一文一事，严格区分 |
| 2 | 层级错乱 | 直接用 1. 而不用一、 | 按层级顺序逐级使用 |
| 3 | 数据空洞 | "取得了显著成效" | 用数据说话 |
| 4 | 空话套话 | "在各级领导的大力支持下"开头三行 | 直接切入正题 |
| 5 | 语气不当 | 下行文用"请"过多 | 下行文用"要""应当" |
| 6 | 序号混乱 | "（一）"后接"（1）"跳级 | 按一、（一）1. 顺序 |
| 7 | 标题不符 | 标题太长或太随意 | 用"关于 XXX 的 XXX"格式 |
| 8 | 政治表述错 | "二十大"写成"20 大" | 全称"党的二十大" |
| 9 | 标点错误 | 一级标题后用句号 | 用顿号"一、" |
| 10 | 排版混乱 | 层级缩进不统一 | 统一规范缩进 |
| 11 | 超越权限 | 区级文件引用"对标国际" | 按政府层级对口表述 |
| 12 | 敏感信息 | 公开文件中出现内部数据 | 区分公开/内部表述 |

### 7.2 避坑原则

1. **不编造**：不编造数据、不编造政策名称、不编造领导讲话
2. **不抄袭**：不同场景须针对性调整，不可简单套用模板
3. **不越级**：按给定政府层级选择对应站位和用词
4. **不空泛**：每条主张尽量有依据、有数据、有出处
5. **不堆砌**：政策引用适量，重点突出，不宜满篇都是"根据……"

## 八、数字使用规范

### 8.1 数字用法规则（依据 GB/T 15835-2011）

| 场景 | 规范 | 示例 |
|:---|:---|:---|

## 十、常用句式库

### 10.1 开头句式

| 文种 | 推荐句式 |
|:---|:---|
| 报告 | "受 XXX 委托，现将有关情况报告如下" |
| 请示 | "为……，现就 XXX 有关事项请示如下" |
| 通知 | "经 XXX 研究决定，现将有关事项通知如下" |
| 讲话 | "今天我们召开 XXX 会议，主要任务是……" |
| 政策文件 | "为贯彻落实……精神，结合我市实际，现提出如下意见" |
| 会议纪要 | "2025 年 X 月 X 日，XXX（职务）主持召开 XXX 会议，研究……" |

### 10.2 衔接过渡句式

| 功能 | 句式 |
|:---|:---|
| 成绩→问题 | "在肯定成绩的同时，我们也要清醒地看到……" |
| 分析→部署 | "做好今年工作，要准确把握……" |
| 部署→落实 | "目标已经明确，关键在于落实" |
| 概括 | "概括起来，就是……" |
| 强调 | "特别是要……""重中之重是……" |
| 承上启下 | "在此基础上，重点抓好以下几方面工作" |

### 10.3 结尾句式

| 文种 | 推荐结尾 |
|:---|:---|
