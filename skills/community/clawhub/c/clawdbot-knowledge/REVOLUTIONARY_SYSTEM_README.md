
- `GET /api/system/metrics` - Get performance metrics

- `POST /api/system/task` - Execute task

#### **Module Endpoints**
- `GET /api/modules/{module_id}` - Get module details
- `POST /api/modules/{module_id}/execute` - Execute module
- `GET /api/modules/{module_id}/performance` - Get module performance

#### **Task Endpoints**
- `POST /api/tasks/execute` - Execute task
- `GET /api/tasks/{task_id}` - Get task status
- `GET /api/tasks/{task_id}/result` - Get task result

### **📝 API Usage Examples**

#### **Get System Status**
