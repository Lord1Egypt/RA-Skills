# 淋巴结穿刺术操作流程

## 适用对象
- 明显肿大的淋巴结
- 疑似感染、造血系统肿瘤或转移癌的患者

## 前提条件
- 淋巴结明显肿大、易于固定、大小合适
- 穿刺部位远离大血管
- 患者处于餐前状态（避免脂质干扰检查）

## 禁忌与限制
- 穿刺部位有出血倾向者禁用
- 无法固定的小淋巴结不宜穿刺

## 操作步骤
1. **选择穿刺部位**：选取明显肿大、易于固定、大小合适且远离大血管的淋巴结。
2. **消毒**：常规消毒局部皮肤及操作者手指。
3. **穿刺**：
   - 左手拇指与示指固定淋巴结；
   - 右手持10ml干燥注射器（配18～19号针头），沿淋巴结长轴刺入（深度依淋巴结大小而定）；
   - 边拔针边用力抽吸，利用负压吸出液体和细胞成分。
4. **涂片制备**：
   - 固定注射器内栓，拔出针头；
   - 将注射器取下后充气，再将针头内抽取液喷射至载玻片上；
   - 立即制备涂片。
5. **包扎固定**：以无菌纱布覆盖穿刺点，并用胶布固定。

## 注意事项
- 若首次未获抽取液，可在原穿刺点不同方向连续穿刺数次（避免出血）；
- 观察抽取液性状：炎性为淡黄色，结核性为黄绿色或污灰色黏稠液，可见干酪样物质；
- 务必在餐前操作，以免脂质过多影响检查结果。
