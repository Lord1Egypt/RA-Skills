# 导演台页面主视觉生成任务模板

## 任务边界

你只负责生成 PPT 页面主体视觉 / 背景 / 概念图，不负责 PPT 组装，不负责添加 Logo、页脚、页码、Part 标签。

## 全局风格

- 16:9 横版；
- 黑金商业风；
- 高端、克制、科技感、可信；
- 主色：#070706 / #12110E / #C5A25A / #F3EBDD；
- 禁止蓝紫霓虹、廉价粒子、俗套机器人/芯片滥用。

## 强制禁止

- 不生成筑梦学院 Logo；
- 不生成页脚；
- 不生成页码；
- 不生成 Part 标签；
- 不生成二维码；
- 不生成可读文字，除非页面合同明确要求；
- 不伪造真实产品界面、真实人物、真实数据；
- 不在旧图上局部贴补、遮挡、擦改。

## 验收标准

- 主体完整；
- 无品牌元素内置；
- 无页脚页码；
- 无明显 AI 乱码或伪文字；
- 与同构页风格一致；
- 可直接进入 PPT 组装层。

## 底部安全区补充要求

若页面后期需要页脚/页码：

- 底部 12%-15% 必须为纯净暗色；
- 不得放置主体视觉、金色光柱、底座、反光、横线、装饰块或复杂纹理；
- 主体视觉应上移或集中在右中/右上区域；
- 违反该规则时，必须整图重生，不得局部补丁。

## 生图与后期分层硬规则

生图只做：

- 背景氛围；
- 右侧/局部主体视觉；
- 抽象几何模块；
- 克制光效；
- 低存在感装饰线。

生图禁止做：

- 任何文字；
- 内容卡片 / 锁定框；
- 标题框；
- 数据标签；
- HUD 字符；
- Logo / 下标 / 页脚 / 页码 / Part 标签；
- 二维码；
- 假动效、假按钮、假交互控件。

后期添加：

- 标题、副标题、核心判断；
- 支撑点卡片；
- 流程线与流程文字；
- Logo、下标、页脚、页码、Part 标签；
- 动态效果/动画路径。

背景线条要求：弱化、低对比、不抢正文；不得出现大面积强线条或强网格。
