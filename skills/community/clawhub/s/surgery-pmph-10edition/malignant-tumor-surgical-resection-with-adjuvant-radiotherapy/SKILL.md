# 恶性肿瘤外科切除与放疗联合治疗原则

## 何时使用
- 病变对象为恶性肿瘤、受累滑膜或弥漫性病变；
- 已实施手术切除，但因病变弥漫导致切除不彻底（如边界不清、残留病灶）；
- 需判断是否应追加术后放射治疗以降低局部复发风险。

> **注意**：本策略仅适用于手术无法彻底切除的弥漫性恶性肿瘤，不适用于良性肿瘤或非弥漫性病变。

## 执行步骤
1. **确认手术已完成**：已对肿瘤包块或受累滑膜实施外科切除。
2. **评估切除彻底性**：
   - 若病变局限、边界清晰，实现完整切除 → **无需放疗**；
   - 若病变弥漫、切除边缘有残留或无法完全清除 → 判定为“切除不彻底”。
3. **启动术后辅助放疗**：对切除不彻底的病例，必须安排放射治疗。
4. **明确放疗目的**：杀灭残留恶性细胞，防止局部复发。
5. **强调治疗顺序**：手术为首要治疗手段，放疗仅为补充，不可替代初始手术。

## 关键变量
- **病变弥漫性**（布尔值）：肿瘤侵犯范围广泛、边界不清，难以完整切除。
- **切除彻底性**（枚举值）：'彻底' 或 '不彻底'，基于术中及术后病理评估。

## 输出结果
输出临床决策：是否启动术后放疗。若满足“弥漫性 + 切除不彻底”条件，则必须安排放疗。
