## 🎯 核心能力

### 1. 五阶段方法论
```
阶段1：发生期 —— 事件触发，信号微弱
    ↓
阶段2：爆发期 —— 舆论发酵，传播加速
    ↓
阶段3：高潮期 —— 舆论峰值，各方回应
    ↓
阶段4：缓和期 —— 热度回落，新热点出现
    ↓
阶段5：降温期 —— 舆论消散，反馈闭环
```

### 2. 三度评估体系
| 维度 | 说明 |
|------|------|
| **广度** | 信息来源覆盖面（≥3个独立信源） |
| **精度** | 数据准确性与标注完整性 |
| **深度** | 分析是否触及问题本质，提供前瞻建议 |

### 3. 对标分析框架
- 同类事件对比（时长、演进路径）
- 主体应对对比（回应时机、措辞、效果）
- 跨行业参照（破局思路借鉴）

### 输出格式
输出标准五阶段分析报告（Markdown格式），包含：
- 舆情性质判定
- 五阶段定位与洞察（每阶段≥150字）
- 对标分析（同类事件对比）
- 三度评估（广度/精度/深度）
- 三条可执行建议

## ✅ 质量保障

### 自检清单
- [x] 舆情性质判定正确（重大/一般/微舆情）
- [x] 数据来源 ≥3 个独立信源
- [x] 五阶段定位的时间节点准确
- [x] 每个数据点标注了来源+时间戳
- [x] 阈值数据已标注年份+使用前需校验
- [x] 对标分析覆盖 ≥2 个维度
- [x] 每个阶段都有 ≥150 字的独立判断洞察
- [x] 整篇报告有1-2句点睛金句
- [x] 结论部分有3条可落地的建议
- [x] FAQ章节完整（至少5问）
- [x] 精度声明前置（位于SKILL.md开头）
- [x] 案例示范完整（2个全流程案例）

### 报告质量标准
| 维度 | 不合格 | 合格 | 优秀 |
|------|--------|------|------|
| **广度** | < 2 个信源 | ≥ 3 个信源 | ≥ 5 个信源，跨平台 |
| **精度** | 模糊表述 | 数字标注来源+时间 | 数字+来源+时间+年份 |
| **深度** | 复述事件 | 每阶段有独立判断 | 判断背后有底层逻辑分析 |
| **对标** | 无对标 | 有1个同类对比 | 同类+主体+跨行业三维对标 |
| **结论** | 无建议 | 有1条建议 | 3条建议，可执行、可追踪 |

## 📖 方法论来源

本技能基于 **Duncan《民航舆情管理》专著**（18.5万字，2026年4月定稿）理论体系提炼，经20+真实案例验证，形成可复用的五阶段分析框架。
