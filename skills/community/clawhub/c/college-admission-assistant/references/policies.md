# 高考政策知识库

> 数据来源：教育部及各省级教育考试院公开政策文件
> 最后更新：2026年4月

---

## 一、新高考改革进程

### 时间线

| 批次 | 启动年份 | 省份 | 模式 |
|:---:|:-------:|------|:---:|
| 第一批 | 2014 | 浙江、上海 | 3+3 |
| 第二批 | 2017 | 北京、天津、山东、海南 | 3+3 |
| 第三批 | 2018 | 江苏、河北、湖北、湖南、广东、福建、辽宁、重庆 | 3+1+2 |
| 第四批 | 2021 | 河南、四川、陕西、山西、云南、贵州、广西、甘肃、吉林、黑龙江、安徽、江西 | 3+1+2 |

### 改革要点
1. **打破文理分科**：学生可根据兴趣和特长选择科目
2. **外语一年两考**：多数省份外语有两次考试机会
3. **等级赋分制**：保障不同选考科目之间的公平性
4. **综合评价录取扩大**：更多高校采用"高考+校测+综合素质评价"的方式

## 二、强基计划

### 概述
- 2020年起实施，取代原自主招生
- 聚焦基础学科（数学、物理、化学、生物、历史、哲学、古文字学等）
- 由36所（2022年扩至39所）双一流A类高校实施

### 关键信息
| 项目 | 内容 |
|------|------|
| 招生院校 | 39所985高校 |
| 招生规模 | 每校约30-180人 |
| 录取方式 | 高考成绩85% + 校测15% |
| 培养模式 | 小班化、导师制、本硕博衔接 |
| 专业方向 | 基础学科为主，部分高校新增交叉学科 |
| 报名时间 | 每年4月 |
| 考试时间 | 高考后（6月底-7月初） |

### 强基计划 vs 原自主招生

| 对比维度 | 强基计划 | 原自主招生 |
|:-------:|:-------:|:---------:|
| 招生定位 | 选拨基础学科拔尖人才 | 选拔学科特长或创新潜质 |
| 降分幅度 | 按综合成绩录取 | 可降分20-60分 |
| 专业范围 | 基础学科为主 | 专业不限 |
| 培养方式 | 专门培养方案 | 无专门培养 |

## 三、综合评价招生

### 概述
- 部分省份和高校实行的多元录取方式
- 综合考量高考成绩、高校考核结果、综合素质评价
- 典型代表：浙江三位一体、江苏综合评价、上海综评

### 综合成绩构成（一般）
```
综合成绩 = 高考成绩 × 60% + 校测成绩 × 30% + 学业水平 × 10%
```
（比例各校不同，一般为6:3:1）

### 典型省份差异

| 省份 | 代表高校 | 综合成绩权重 |
|:---:|---------|:----------:|
| 浙江 | 浙江工业大学 | 6:3:1 |
| 江苏 | 南京邮电大学 | 7:3 |
| 上海 | 复旦大学 | 85:15（类似强基） |
| 山东 | 山东大学 | 6:3:1 |

## 四、专项计划

### 国家专项计划
- **目标**：面向脱贫地区（原国家级贫困县）
- **实施高校**：中央部门高校和各省（区、市）所属重点高校
- **录取优惠**：单独划线，一般低于普通批次

### 高校专项计划
- **目标**：面向脱贫地区、农村的优秀学生
- **实施高校**：教育部直属高校和其他自主招生高校
- **录取方式**：考生须在高考前报名，高考后单独投档
- **条件**：本人及父母户籍在实施区域且连续3年以上

### 地方专项计划
- **目标**：面向各省脱贫地区、农村地区
- **实施高校**：省属重点高校
- **录取优惠**：单独投档，分数线低于普通批次

## 五、艺术类招考

### 2024年改革要点
1. **省级统考全覆盖**：各省逐步统一艺术类考试内容
2. **校考范围缩减**：仅独立设置的艺术院校可组织校考
3. **文化课成绩提高**：艺术类文化线逐步提高
4. **综合成绩计算**：文化课占比不低于50%（艺考新政）

### 艺术类综合分公式（各省不同）
```
综合分 = 文化课成绩 × 50% + 专业课成绩 × 2.5 × 50%
```
（各省比例不同，以省教育考试院公布为准）

## 六、体育类招考

1. **体育统考**：各省统一组织体育专业测试
2. **体育单招**：面向专业运动员的单独招生
3. **高水平运动队**：高校自主招收高水平运动员
4. 综合分一般 = 文化课 × 30% + 专业 × 70%（具体各省不同）
