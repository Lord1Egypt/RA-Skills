# 胃食管反流病诊断标准应用

## 适用对象
- 成人疑似胃食管反流病患者
- 存在烧心、反流等典型或不典型症状（如慢性咳嗽、咽喉炎、哮喘样表现）
- 已记录症状出现频率与持续时间

## 禁忌情形
- 急性胃肠道感染引起的暂时性反流
- 儿童或青少年患者（本标准仅适用于成人）

## 执行步骤

1. **评估症状频率与严重程度**：
   - 若轻度症状每周出现 ≥ 2天，进入诊断流程
   - 若中度或重度症状每周出现 ≥ 1天 **且** 持续时间 ≥ 8周，进入诊断流程
   - 若均不满足，暂不支持GERD诊断，建议观察或排查其他病因

2. **检查是否存在客观并发症**：
   - 反流性食管炎（内镜下可见）
   - Barrett食管
   - 食管狭窄
   - 若存在任一并发症，即使症状频率未达标，也可确诊GERD

3. **结合辅助检查结果支持诊断**（若已进行）：
   - 胃镜发现反流性食管炎、贲门松弛或食管裂孔疝
   - 食管pH监测显示酸暴露时间异常
   - 高分辨率食管测压显示功能异常
   - 抑酸治疗后症状明显缓解（PPI试验阳性）
   - pH-阻抗监测显示反流-症状相关性阳性

4. **谨慎处理不典型症状**：
   - 咳嗽、哮喘、咽喉炎等食管外表现缺乏特异性
   - 必须结合客观检查避免误诊

5. **输出结论**：
   - 返回“符合胃食管反流病诊断标准”或“不符合”
   - 明确建议是否需进一步行胃镜、反流监测等客观检查

> 注意：本技能仅用于初步临床判断，最终诊断需由医生结合完整病史与检查结果综合判定。
