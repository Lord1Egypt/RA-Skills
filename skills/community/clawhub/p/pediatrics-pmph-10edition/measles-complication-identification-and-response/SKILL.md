# 麻疹并发症识别与应对

## 触发条件
当麻疹患儿出现以下任一情况时立即启用本技能：
- 高热持续不退
- 呼吸急促或呼吸困难
- 意识障碍或行为异常
- 有结核病史且病情突然加重

## 执行流程

### 1. 呼吸系统并发症评估
- 判断是否为肺炎（占麻疹死亡原因90%以上）
- 区分病毒性间质性肺炎（通常较轻）与继发细菌性肺炎
- 警惕脓胸、脓气胸或多病原体混合感染

### 2. 心肌炎筛查
- 轻症表现：心音低钝、心率增快、一过性心电图异常
- 重症预警：心力衰竭、心源性休克

### 3. 神经系统并发症识别
- **麻疹脑炎**：出疹后2～6天再次发热，伴脑炎样症状；病死率高，存活者常遗留神经后遗症
- **亚急性硬化性全脑炎（SSPE）**：麻疹后2～17年发病，初期为行为情绪改变，渐进至智力减退、共济失调、肌阵挛，最终昏迷死亡

### 4. 结核病恶化评估
- 麻疹导致免疫抑制，可激活潜伏结核
- 警惕粟粒性肺结核或结核性脑膜炎播散

## 输出行动
- 根据疑似并发症类型，立即安排相应检查（如胸部影像、心电图、脑脊液分析、结核相关检测）
- 启动针对性治疗（抗细菌、支持循环、抗癫痫、抗结核等）
- 对高危病例转入重症监护
