# 分活配置（管理牛仔弹窗）

## 页面定位

商家通过弹窗式配置面板，管理牛仔的接待工作范围。配置修改后实时生效。

## 页面内容

### 基本状态栏
- 牛仔头像 + 当前状态（「接待中」绿色 /「已暂停」灰色）
- 暂停/恢复接待 切换按钮

### 四维度配置面板

#### 1. 需求种类（多选）

基于近30天询盘自动统计的业务类型：

| 常见类型 | 说明 |
|----------|------|
| 采购询价 | 买家直接询问产品价格 |
| 定制需求 | 买家有定制化要求 |
| 物流查询 | 询问发货方式和费用 |
| 售后咨询 | 退换货/售后投诉 |
| 普通闲聊 | 无明确采购意向的对话 |

默认全选。取消勾选后，该类询盘不交给牛仔处理，直接进入人工队列。

#### 2. 接待时间段

| 预设选项 | 时间 |
|----------|------|
| 全天候 | 00:00 ~ 24:00（默认） |
| 工作时间 | 09:00 ~ 18:00 |
| 仅非工作时间 | 18:00 ~ 09:00 |
| 仅周末 | 周六、周日全天 |
| 自定义 | 允许多时段组合 |

非接待时间段内，牛仔不主动应答，买家消息进入人工队列。

#### 3. 买家等级筛选

| 等级 | 说明 |
|------|------|
| KA客户 | 大客户，默认不交给牛仔，由人工接待 |
| V等级高 | V3及以上，可配置是否由牛仔接待 |
| 普通买家 | 默认由牛仔接待 |
| 新买家 | 默认由牛仔接待 |

#### 4. 红线场景

以下场景一旦触发，牛仔立即转人工（默认全部勾选）：

| 红线 | 说明 |
|------|------|
| 投诉升级 | 买家明确表示要投诉 |
| 退款争议 | 涉及退款金额分歧 |
| 法律合规 | 涉及合同、法律条款 |
| 情绪激动 | 买家用词激烈 |

### 底部数据概况
- 近7天接待总量、牛仔接待占比、转人工率
- 用于帮助商家判断配置是否合理

## Agent 引导要点

当用户说"调整分活配置"、"暂停牛仔"时，Agent 应：
1. 引导文案："您可以调整牛仔的接待范围，包括需求种类、工作时段、客户等级和红线场景。"
2. 跳转到管理牛仔弹窗
3. 跳转后话术："已打开管理牛仔配置面板。您可以修改接待规则，修改后实时生效。"
