# 小说写作素材库

本文档收集常用写作素材，供 novel-writer skill 使用。

---

## 一、常见情节模板

### 1. 逆袭型

**套路**：
- 主角从最底层开始
- 被轻视、被欺负、被背叛
- 获得金手指/奇遇/高人指点
- 一步步崛起，打脸所有人
- 最终成为最强者

**适用类型**：玄幻、修仙、都市、系统流

**经典桥段**：
- 退婚流：未婚妻嫌弃主角，主角逆袭后未婚妻后悔
- 废物流：被称废物，实则天才
- 赘婿流：被女方家族看不起，最终证明自己

**变体**：
- 全家被灭门，主角独活，长大后复仇
- 破产/负债，主角从零开始东山再起
- 被未婚妻/女友抛弃，主角变强后对方后悔

---

### 2. 冒险型

**套路**：
- 主角获得某个任务/使命
- 踏上旅途，遭遇各种挑战
- 途中结识伙伴，打败最终BOSS
- 完成使命，获得成长

**适用类型**：奇幻、科幻、探险

**经典桥段**：
- 寻找宝物/神器
- 拯救世界/大陆/王国
- 寻找失踪的人（父母、爱人、导师）
- 探索未知领域（秘境、异界、外太空）

---

### 3. 悬疑型

**套路**：
- 发生神秘事件/案件
- 主角深入调查
- 逐渐发现真相
- 揭开幕后黑手
- 解决事件，但留有悬念

**适用类型**：推理、惊悚、犯罪

**经典桥段**：
- 密室杀人案
- 连续杀人案
- 消失的凶手
- 双重身份/人格
- 记忆篡改

---

### 4. 言情型

**套路**：
- 两个身份悬殊的人相遇
- 经历各种误会/冲突/分离
- 克服阻碍，最终在一起

**适用类型**：现代言情、古代言情

**经典桥段**：
- 契约恋爱/假结婚
- 豪门总裁/灰姑娘
- 破镜重圆/误会分离
- 暗恋成真/追妻火葬场
- 失忆梗/替身梗

---

### 5. 成长型

**套路**：
- 主角经历重大变故/选择
- 在痛苦中成长
- 学会新技能/获得新认知
- 最终成为更好的自己

**适用类型**：现实主义、青春文学

**经典桥段**：
- 校园成长（友情、爱情、迷茫）
- 职场新人（适应、竞争、抉择）
- 家庭变故（亲人离去、承担责任）

---

## 二、老套但有效的桥段

### 1. 开篇冲突

| 桥段 | 适用类型 | 效果 |
|------|---------|------|
| 主角被陷害/冤枉 | 玄幻、都市 | 快速建立"反派"激发读者同情 |
| 主角濒死/重伤 | 修仙、玄幻 | 建立紧迫感，引入金手指 |
| 亲人被害/失踪 | 复仇、冒险 | 提供动机 |
| 末日危机 | 科幻、生存 | 建立紧迫感 |

### 2. 关键转折

| 桥段 | 适用类型 | 效果 |
|------|---------|------|
| 主角被打脸 | 所有爽文 | 制造爽感 |
| 真相揭露 | 悬疑、言情 | 制造反转 |
| 主角绝境逆袭 | 玄幻、修仙 | 制造高潮 |
| 伙伴背叛 | 冒险、热血 | 增加冲突 |
| 身份揭秘 | 所有类型 | 制造悬念 |

### 3. 结尾高潮

| 桥段 | 适用类型 | 效果 |
|------|---------|------|
| 最终对决 | 玄幻、动作 | 制造爽感 |
| 真相大白 | 悬疑 | 满足好奇心 |
| 感情确认 | 言情 | 满足期待 |
| 牺牲/离去 | 所有类型 | 催泪 |

## 三、经典人物设定

### 1. 主角类型

| 类型 | 特点 | 适用读者 |
|------|------|---------|
| 善良型 | 心软、正义感强、帮助弱者 | 女性读者 |
| 腹黑型 | 表面温和，内心狠辣 | 喜欢复杂角色 |
| 霸气型 | 强势、独断、护短 | 喜欢爽文 |
| 成长型 | 从弱到强，不断进步 | 喜欢代入感 |
| 无赖型 | 油嘴滑舌、不按常理出牌 | 喜欢轻松幽默 |

### 2. 配角类型

| 类型 | 作用 | 常用设定 |
|------|------|---------|
| 导师/师父 | 传授技能，推动剧情 | 神龙见首不见尾、严厉但关心 |
| 好友/兄弟 | 提供帮助，增加趣味 | 嘴贱但靠谱、忠心耿耿 |
| 爱人 | 增加情感线 | 势均力敌、欢喜冤家 |
| 对手/宿敌 | 增加冲突 | 天赋型、阴谋型、复仇型 |
| 小弟/追随者 | 衬托主角 | 崇拜主角、有成长潜力 |

### 3. 反派类型

| 类型 | 动机 | 表现 |
|------|------|------|
| 权力型 | 追求权力/地位 | 工于心计、心狠手辣 |
| 复仇型 | 为主角或其家人而来 | 阴险、执着、不择手段 |
| 疯批型 | 没有明确动机就是坏 | 不可理喻、制造恐惧 |
| 可怜型 | 有苦衷的坏人 | 让读者同情，增加深度 |
| 双重型 | 表面好人，实际反派 | 反转震撼 |

## 四、世界观设定灵感

### 1. 力量体系

| 类型 | 特点 | 例子 |
|------|------|------|
| 等级制 | 明确等级，层层递进 | 斗者/斗师/大斗师... |
| 职业制 | 不同职业各有特色 | 剑士/法师/牧师... |
| 天赋制 | 天赋决定上限 | 元素亲和度/灵根属性 |
| 契约制 | 与某物契约获得力量 | 召唤兽/系统/神器 |
| 修炼制 | 功法决定修炼速度 | 内力/灵力/法力 |

### 2. 世界设定

| 类型 | 特点 | 适用类型 |
|------|------|---------|
| 仙侠世界 | 修士、飞升、渡劫 | 修仙小说 |
| 魔法世界 | 法师、魔法学院 | 西幻 |
| 未来世界 | 科技、星舰、机甲 | 科幻 |
| 异界大陆 | 种族、王国、帝国 | 奇幻 |
| 现代都市 | 都市异能、商业、娱乐圈 | 都市文 |
| 末日世界 | 丧尸、废土、生存 | 末日文 |

### 3. 地图设定灵感

| 类型 | 特点 | 经典设定 |
|------|------|---------|
| 大陆型 | 一块大陆，多个国家 | 王朝更替、帝国征战 |
| 岛屿型 | 多岛屿，海洋冒险 | 海贼、航海 |
| 多元宇宙 | 多个位面/世界 | 时空穿梭、异界入侵 |
| 层级世界 | 上下分层，层层递进 | 天界/人间/地狱 |
| 秘境型 | 散落秘境，探索冒险 | 探险、流寇 |

## 五、开头写法

### 1. 快速切入型（推荐）

```
【开头直接冲突】
主角正在做某件事，突然发生意外。
或者主角正在经历某种危机。
```

**示例**：
> 张三盯着手机上那条短信，手指微微发抖。
>
> "三天内还清五百万，否则你妹妹的命就没了。"
>
> 他深吸一口气，看了看自己空空的口袋。

---

### 2. 背景铺垫型

```
【开篇建立世界观】
先描述世界/环境的独特之处，
再引出主角。
```

**示例**：
> 这个世界没有太阳。
>
> 天空永远笼罩着灰蒙蒙的云层，阳光从未穿透那片阴霾。人们在"永夜"中生活了七百年，早已忘记阳光的温暖。
>
> 而我，就是那个即将打破永夜的人。

---

### 3. 金手指导入型

```
【开篇展示金手指】
主角刚获得某种能力或机遇，
让读者期待后续发展。
```

**示例**：
> 我盯着眼前这个弹窗，陷入沉思。
>
> 【检测到宿主满足条件】
> 【神级选择系统绑定中...】
> 【绑定成功，当前积分：0】
>
> 我深吸一口气——是的，我穿越了，还带了个系统。

---

### 4. 悬念导入型

```
【开篇制造悬念】
用一个问题或场景吸引读者，
让他们想知道后续。
```

**示例**：
> 她死了。
>
> 所有人都这么说。
>
> 但我不信。因为那天晚上，我亲眼看到她走进了那栋楼。而那栋楼第二天就成了一片废墟。
>
> 三年了，我一直在找她。

## 六、结尾写法

### 1. 悬念式结尾

```
【结尾留悬念】
在章节/卷/书的结尾制造悬念，
吸引读者继续读。
```

**示例**：
> 他正要开口，突然停住了。
>
> 身后传来脚步声。
>
> 很多脚步声。
>
> "站住，不许动！"
>
> 他没有回头，只是轻轻笑了一声。
>
> "来得还真快。"
>
> 下一章预告：神秘势力登场，主角陷入危机...

---

### 2. 高潮式结尾

```
【结尾制造高潮】
在章节末尾给出一个爽点，
让读者感到满足。
```

**示例**：
> "你说什么？"赵强以为自己听错了。
>
> 李明重复了一遍，声音不大，却足以让整个包厢的人都听见：
>
> "我说，这家公司，我收购了。"
>
> 全场寂静。
>
> 赵强脸色煞白。

---

### 3. 情感式结尾

```
【结尾制造情感共鸣】
给读者留下深刻的情感印象。
```

**示例**：
> 她站在机场，看着他走进安检通道。
>
> 他回头，笑了笑，挥了挥手。
>
> 她也挥了挥手。
>
> 当他的身影消失在人群中，她终于忍不住，眼泪流了下来。
>
> 再见，也许再也不见。

## 七、常见问题处理

### 1. 如何让主角不崩人设？

**方法**：
- 明确主角的核心性格（善良/腹黑/霸气）
- 写之前先问："这个行动符合角色性格吗？"
- 给主角设置底线（什么绝对不做）
- 让金手指服务于人设（同样能力，不同人用不同方式）

---

### 2. 如何让配角不工具人？

**方法**：
- 给配角自己的目标/动机
- 给配角自己的故事线（哪怕是支线）
- 让配角有自己的性格（不是为主角而存在）
- 让配角和主角有真实互动（不只是帮忙和感谢）

---

### 3. 如何制造爽点？

**爽点类型**：
- **打脸爽**：主角反击，让轻视他的人后悔
- **逆袭爽**：主角从弱到强，碾压对手
- **装逼爽**：主角低调，实力却超强
- **复仇爽**：主角报还前仇
- **收集爽**：主角获得宝物/伙伴/美人

**方法**：
- 先抑后扬（先让他吃亏，再让他赢）
- 反差对比（之前多惨，现在多爽）
- 观众视角（让配角/反派见证主角的厉害）
- 情绪积累（压抑到极点再爆发）

---

### 4. 如何避免水文？

**水文特征**：
- 重复场景（同样的情节反复出现）
- 无意义对话（废话连篇，没有推进）
- 无意义描写（不必要的环境/外貌描写）
- 无意义升级（能力提升但没有用）

**避免方法**：
- 每章必须有进展（情节/情感/人物）
- 删除不影响理解的描写
- 让每个场景都有目的
- 问问自己："删掉这段，故事受影响吗？"

---

### 5. 如何让对话不尴尬？

**问题对话**：
- 句句都是信息（像在念说明书）
- 对话标签太多（"他说"、"她道"）
- 太书面语（不像真人说话）
- 太口语化（太水）

**改进方法**：
- 加入动作和反应（不只是说话）
- 用省略号和语气词（"嗯...好吧"）
- 让对话有潜台词（表面一套，实际暗示另一套）
- 控制信息量（不要一句接一句倒信息）

## 八、灵感收集

### 1. 日常灵感来源

| 来源 | 可能触发的灵感 |
|------|--------------|
| 新闻事件 | 社会问题、极端情况 |
| 历史故事 | 战争谋略、人物传奇 |
| 电影/游戏 | 设定套路、情节结构 |
| 音乐 | 情绪氛围、情感主题 |
| 生活经历 | 真实细节、情感共鸣 |

### 2. 随机组合灵感

**方法**：
1. 随机选择一个类型
2. 随机选择一个设定
3. 随机选择一个冲突

**示例**：
- 古代 + 穿越 + 替身 = 穿越成被替身的王妃
- 都市 + 系统 + 渣男 = 系统要求当好人，但主角是渣男
- 末世 + 异能 + 亲情 = 末世中保护家人的故事
