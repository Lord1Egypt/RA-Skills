# 血-脑屏障结构与功能

## 何时使用
- 需要解释为何特定物质（如染料、大分子、毒素）不能进入脑或脊髓组织
- 评估某药物是否可能穿过血-脑屏障用于中枢神经系统治疗
- 分析中枢神经系统内环境稳定性的结构基础

## 如何执行
1. **确认适用对象**：仅适用于中枢神经系统毛细血管、脑组织和脊髓；不适用于外周器官毛细血管或血-脑脊液屏障。
2. **识别结构组成**：
   - 毛细血管内皮细胞为连续型，细胞间由紧密连接封闭
   - 外包基膜
   - 星形胶质细胞终足形成神经胶质膜包裹血管
3. **判断通透性机制**：
   - 内皮细胞是屏障主体，缺乏窗孔和大量质膜小泡
   - 脂溶性高、分子量小的物质更易通过
   - 营养物质（如葡萄糖、氨基酸）可通过特异性转运体进入
4. **应用实验证据**：若静脉注射台盼蓝后脑/脊髓未染色，可佐证屏障存在
5. **输出警示**：若目标物质不具备脂溶性或分子量过大，提示其可能无法有效作用于中枢神经系统

> 注意：本技能不处理血-脑脊液屏障或外周毛细血管通透性问题。
