# 品牌架构策略(Brand Architecture)

> **用途**: strategy / brand Agent 共享的品牌组合决策框架
> **版本**: v1.0 | **更新**: 2026-06-06
> **适用**: 多品牌企业 / 品牌延伸决策 / 子品牌创建

---

## 一、品牌架构 4 大模式

### 1.1 模式总览

| 模式 | 结构 | 典型案例 | 适用场景 |
|------|------|---------|---------|
| **单一品牌** (Monolithic / Branded House) | 企业品牌=产品品牌 | Apple / BMW / IBM | 强势母品牌 + 业务聚焦 |
| **主副品牌** (Endorsed) | 母品牌+子品牌 | Nestlé KitKat / Marriott Courtyard | 母品牌背书 + 子品牌差异 |
| **多品牌组合** (House of Brands) | 独立品牌互不关联 | P&G(海飞丝/汰渍/ SK-II) | 跨品类 / 跨客群 |
| **子品牌** (Sub-brand) | 母品牌+独立子品牌 | Apple iPhone / Toyota Lexus | 品类拓展 + 母子协同 |

### 1.2 4 种模式对比

| 维度 | 单一品牌 | 主副品牌 | 多品牌 | 子品牌 |
|------|:------:|:------:|:----:|:----:|
| 品牌建设成本 | 低 | 中 | 高 | 中 |
| 风险隔离 | ❌ | ⚠️ | ✅ | ⚠️ |
| 协同效应 | ✅ | ✅ | ❌ | ✅ |
| 母品牌稀释风险 | 高 | 中 | 低 | 中 |
| 适用规模 | 1-3 亿 | 3-30 亿 | 30 亿+ | 3-30 亿 |
| 管理复杂度 | 低 | 中 | 高 | 中 |

## 二、决策框架:如何选择架构模式

### 2.1 决策树

```
Q1: 现有业务和新业务是否在同一品类?
├── 是 → Q2: 目标人群是否重合?
│     ├── 是 → 单一品牌(延伸)
│     └── 否 → 子品牌(区隔人群)
└── 否 → Q3: 新品类的风险是否可能波及母品牌?
      ├── 是 → 多品牌(风险隔离)
      └── 否 → Q4: 母品牌能否为新业务背书?
            ├── 是 → 主副品牌(母背书)
            └── 否 → 多品牌(完全独立)
```

### 2.2 关键决策因素评分

| 因素 | 单一品牌(分) | 主副品牌(分) | 多品牌(分) | 子品牌(分) |
|------|:--------:|:--------:|:------:|:----:|
| 母品牌强度(1-10) | ≥8 | 6-8 | <6 | 6-8 |
| 新业务相关性(1-10) | ≥7 | 5-7 | <5 | 5-7 |
| 风险隔离需求(1-10) | <3 | 3-7 | ≥7 | 3-7 |
| 资源充足度(1-10) | 不限 | ≥5 | ≥8 | ≥6 |
| 长期战略(1-10) | 聚焦 | 协同 | 多元 | 拓展 |

## 三、单一品牌延伸策略

### 3.1 延伸方向

| 方向 | 定义 | 案例 | 风险 |
|------|------|------|------|
| **品类内延伸** | 同品类不同价位/规格 | iPhone Pro/标准/SE | 模糊定位 |
| **相关品类延伸** | 供应链/技术相关 | 小米 手机→电视→IoT | 资源分散 |
| **跨界延伸** | 跨入非相关品类 | 娃哈哈 矿泉水→童装 | 稀释品牌 |
| **向下延伸** | 推出低价位子线 | BMW 1系 | 损伤高端形象 |
| **向上延伸** | 推出高价位子线 | 丰田 → 雷克萨斯 | 难以建立信任 |

### 3.2 延伸成功 5 条法则

1. **感知契合度 ≥ 60%**: 消费者觉得"合理"(Apple 做手表 ✅, Apple 做食用油 ❌)
2. **核心价值不变**: 延伸产品仍能兑现母品牌承诺
3. **独立品类空间**: 延伸品类有足够市场容量
4. **供应链优势**: 能复用母品牌的供应链/渠道/研发
5. **试错成本低**: 先 MVP 测试再全面铺开

## 四、多品牌组合管理(Portfolio Management)

### 4.1 品牌角色定义

| 角色 | 定义 | 营收占比 | 资源投入 | 案例(P&G) |
|------|------|:------:|:------:|---------|
| **战略品牌** | 未来增长引擎 | 10-30% | 高 | SK-II |
| **现金牛品牌** | 成熟利润源 | 40-60% | 中 | 玉兰油 |
| **种子品牌** | 探索性新品牌 | 5-15% | 低-中 | 试验产品 |
| **防御品牌** | 阻击竞品 | 5-15% | 低 | 低端子品牌 |
| **退出品牌** | 拟淘汰 | <5% | 极低 | 待整合 |

### 4.2 品牌组合优化矩阵(BCG 改编)

```
            高增长
              │
  ⭐ 战略品牌  │  ❓ 种子品牌
  (投入资源)   │  (选择性投入)
  ─────────────┼─────────────
  💰 现金牛品牌 │  🚫 退出品牌
  (维持收割)   │  (逐步退出)
              │
            低增长
     高份额          低份额
```

## 五、CBBE 品牌资产模型(顾客基础)

### 5.1 CBBE 金字塔(Kevin Keller)

```
                    🔺
                   /\
                  /  \   共鸣(Resonance)
                 /____\  — 忠诚 / 依恋 / 社群
                /      \
               /  关系   \ 判断+感觉(Judgment + Feeling)
              /__________\ — 质量 / 优越性 / 个性 / 反应
             /            \
            /    表现      \ 表现+形象(Performance + Imagery)
           /________________\ — 功能 / 效率 / 个性 / 历史
          /                  \
         /      显著性         \ 显著性(Salience)
        /______________________\ — 知名度 / 类别识别
```

### 5.2 品牌资产衡量指标

| 层级 | 指标 | 衡量方式 | 健康值 |
|------|------|---------|:----:|
| 显著性 | 第一提及知名度 | "提到 [品类]，你首先想到哪个品牌？" | ≥30% |
| 表现+形象 | 品牌联想强度 | 自由联想测试 | ≥5 个正面联想 |
| 判断+感觉 | 品牌偏好度 | A/B 测试 | ≥60% 偏好 |
| 共鸣 | NPS | NPS 调研 | ≥30 |

## 六、品牌估值方法(Brand Valuation)

### 6.1 3 大估值方法

| 方法 | 原理 | 适用 | 难度 |
|------|------|------|:---:|
| **成本法** | 历史品牌投入累加 | 内部核算 | 低 |
| **市场法** | 参照类似品牌交易价格 | 并购 | 中 |
| **收益法** | 品牌带来的未来现金流折现 | 最常用 | 高 |

### 6.2 Relief-from-Royalty 法(最常用)

```
品牌价值 = Σ (品牌年收入 × 版税率 × (1-税率)) / (1+折现率)^年数

参数参考:
- 版税率: 快消 2-5%, 科技 5-10%, 奢侈品 10-15%
- 折现率: WACC + 品牌风险溢价(8-15%)
```
