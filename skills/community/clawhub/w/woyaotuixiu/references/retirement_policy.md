# 中国退休政策与社保缴费参考

## 一、延迟退休政策（2025年1月1日起实施）

### 政策概述
根据《关于实施渐进式延迟法定退休年龄的决定》，从2025年1月1日起实施渐进式延迟退休。

### 延迟规则

#### 男性职工
- **原退休年龄**：60周岁
- **目标退休年龄**：63周岁
- **延迟方式**：每4个月延迟1个月
- **过渡期**：15年（36个月延迟）

#### 女性干部
- **原退休年龄**：55周岁
- **目标退休年龄**：58周岁
- **延迟方式**：每4个月延迟1个月
- **过渡期**：15年（36个月延迟）

#### 女性工人
- **原退休年龄**：50周岁
- **目标退休年龄**：55周岁
- **延迟方式**：每2个月延迟1个月
- **过渡期**：15年（60个月延迟）

### 计算公式

```
延迟月数 = min(过渡期经过月数 / 延迟间隔, 最大延迟月数)
退休年龄 = 原退休年龄 + (延迟月数 / 12)
```

## 二、社保缴费政策

### 个人缴费比例（全国统一）

| 险种 | 个人缴费比例 | 备注 |
|------|-------------|------|
| 养老保险 | 8% | 进入个人账户 |
| 医疗保险 | 2% | 进入个人账户 |
| 失业保险 | 0.5% | - |
| 住房公积金 | 5%-12% | 各地不同，一般取7%计算 |

### 缴费基数

缴费基数 = max(当地最低基数, min(本人月工资, 当地最高基数))

## 三、养老金计算简介

### 基本养老金组成
1. **基础养老金**：与缴费年限、缴费基数、当地平均工资相关
2. **个人账户养老金**：个人账户累计储存额 ÷ 计发月数

### 计发月数参考
- 60岁退休：139个月
- 55岁退休：170个月
- 50岁退休：195个月

## 四、数据来源

- 延迟退休政策：《关于实施渐进式延迟法定退休年龄的决定》（2024年）
- 社保缴费基数：各城市人力资源和社会保障局2024年公布数据
- 缴费比例：国务院《社会保险费征缴暂行条例》
