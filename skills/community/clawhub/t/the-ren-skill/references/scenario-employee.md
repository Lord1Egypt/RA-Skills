# 场景一：识员工（招·用·育·裁）

> 覆盖招聘面试、内部提拔、团队评估、问题员工识别、离职信号预判。
>
> **⚡ 协作：** 先看 `personality-model.md` 六型画像定位，再用本文件做场景落地。
> 每个岗位都有对应的最佳人格类型，详见各岗位说明。

---

## 一、招聘面试：30秒+3分钟+15分钟识人流程

### 前30秒：静态气质初筛（无需对话）

| 观察点 | 面相/行为特征 | 心理学逻辑 | 💼 决策参考 |
|---|---|---|---|
| 鞋履洁净度 | 鞋边泥渍、鞋面灰尘、鞋带整齐度 | 细节自律性是项目管理能力的缩影 | 鞋边脏污且无清理痕迹→核心岗慎选 |
| 指甲整洁度 | 指甲缝污垢、指甲长短均匀度 | 对细节的在意程度 | 指甲缝有污垢→对客岗一票否决 |
| 初始坐姿 | 舒展vs蜷缩vs瘫坐 | 领地行为=自我认同+掌控感 | 管理岗优先舒展坐姿者 |
| 眉头状态 | 自然舒展vs习惯性皱眉 | 长期皱眉=长期思虑/抱怨 | 带团队岗慎选习惯性皱眉者 |
| 眼神基线 | 进门后自然扫视vs躲闪vs四处乱看 | 对新环境的态度：好奇vs防御vs不安 | 四处乱看但不上手碰=好奇心强（好）；上手碰东西=边界感弱（警惕） |

### 前3分钟：压力测试（1个问题触发）

**提问参考：** "请用1分钟说下你过往最失败的一个项目"

| 观察点 | 面试信号 | 心理学逻辑 | 💼 决策参考 |
|---|---|---|---|
| 语速与停顿 | 流畅不卡顿，主动说细节 | 提前复盘过，对结果负责 | ✅ |
| | 卡顿、改口、回避细节 | 认知负荷过载（隐瞒/甩锅） | ❌ |
| 眼神 | 自然平视，不躲闪也不死盯 | 心态坦诚 | ✅ |
| | 频繁眨眼、飘向天花板 | 安慰行为（压抑不安/编造） | ❌ 敏感岗一票否决 |
| 归因模式 | "我当时漏掉了XX，后来做了检查清单" | 自我反思+系统改进 | ✅ |
| | "团队没配合好/前公司制度不行" | 甩锅型归因 | ❌ 不管能力多强 |
| 被打断时 | 停下倾听，点头，再继续 | 情绪稳定性高 | ✅ |
| | 皱眉/抢话/语速突然加快 | 抗压差/固执 | ❌ 管理岗慎选 |

### 前15分钟：深度四维验证

按 `four-dimension-model.md` 的四维逐一打分，配合以下岗位特定的面试问题：

**通用深度问题：**
1. "你带过的最难带的下属是什么样的？你怎么处理的？"→测共情+领导力
2. "你觉得你的前老板/前公司最大的问题是什么？"→测格局+归因模式
3. "如果让你重新做上一个项目，你会怎么改？"→测反思能力+诚实度
4. "你和同事发生过的最大的意见冲突是什么？怎么解决的？"→测冲突处理+共情

## 二、分岗位识人清单

### 销售/商务拓展岗

**核心痛点：** 抗压能力、成就动机、社交直觉。

**最佳人格类型：** 先锋型（纯正对位）、先锋×基石型（能开拓能深耕）、成长→先锋型（内部培养）。

**慎用类型：** 谋士型（社交消耗太大，续航不足）、基石型（缺乏开拓欲）。

| 观察场景 | 面相/行为信号 ✅ | 面相/行为信号 ❌ |
|---|---|---|
| 进门握手 | 力度适中偏强，眼神直视带微笑 | 握手绵软/用力捏手，眼神飘忽或死盯 |
| 谈薪资提成 | 大方谈钱，"我想要高提成，因为我能创造更多" | 扭捏回避，"我不看重钱，看重平台" |
| 模拟被拒 | 微笑反驳，给出新价值点，不慌张 | 脸色瞬间垮掉/急于辩解/沉默放弃 |
| 眼神 | 有攻击性的温和（坚定但不凶） | 要么太软（没攻击性），要么太凶（客户被吓跑） |
| 镜像反应 | 适度模仿你（建立亲和力） | 完全没有=共情低，过度=讨好型 |
| 法令纹 | 向外上扬型（真笑多） | 向下弯（假笑多）或几乎没有（不笑） |
| 面部气色 | 红润有光泽 | 暗沉/眼袋重（自我管理差） |

**一票否决：** 被拒绝后脸色明显变差/眼神变凶→抗挫力为0，不适合销售。

---

### 技术/研发/产品岗

**核心痛点：** 专注力、逻辑闭环、诚实度（不吹牛）。

**最佳人格类型：** 谋士型（纯正对位）、谋士×基石型（能设计能落地）、将帅×谋士型（技术合伙人）。

**慎用类型：** 先锋型（"这个功能很简单"然后做不出来）、嘴炮型（必须严格过滤）。

| 观察场景 | 面相/行为信号 ✅ | 面相/行为信号 ❌ |
|---|---|---|
| 眼神 | 交流时能聚焦，思考时内视（看向某处） | 全程涣散，或过度精明（嘴炮型） |
| 谈技术难点 | 能清晰拆解，承认自己不知道的部分 | 用行话堆砌，无法解释底层逻辑 |
| 被打断时 | 停下倾听，点头，再继续（包容性） | 皱眉/语速加快/不耐烦（固执难协作） |
| 归因模式 | "这个bug是我漏了一个边界条件" | "测试没测出来/需求没写清楚" |
| 坐姿 | 舒适即可，不需要刻意舒展 | 过度舒展可能是来吹牛的，不是来写代码的 |
| 手部 | 说话时手自然配合，不强求手势 | 手势过多可能是嘴炮型 |
| 眉头 | 可以自然，不一定要求开阔 | 习惯性皱眉=可能钻牛角尖（双刃剑：对技术深度有利，对协作有害） |

**特别注意：** 技术人才的眼神可能不擅长社交对视（内向型正常），但思考问题时的眼神聚焦度必须是高的。区分"社交回避"vs"注意力涣散"。

---

### 职能/运营/行政岗

**核心痛点：** 细致度、情绪稳定性、服务意识。

**最佳人格类型：** 基石型（纯正对位）、基石×成长型（稳中有升）、谋士×基石型（系统型运营）。

**慎用类型：** 先锋型（无法忍受重复性工作）、将帅型（会觉得运营"没挑战"）。

| 观察场景 | 面相/行为信号 ✅ | 面相/行为信号 ❌ |
|---|---|---|
| 递交简历 | 纸张平整，指甲修剪干净 | 简历卷边，指甲缝有污垢 |
| 谈过往失误 | "我当时漏掉了XX，后来做了检查清单" | "当时太忙了/别人没配合好" |
| 等待面试时 | 安静等待，不翻动桌面物品 | 抖腿/手机外放/随意起身走动 |
| 嘴角 | 自然平直或微上扬 | 习惯性嘴角下垂（长期抱怨者） |
| 眉心 | 舒展 | 皱眉肌发达/悬针纹（思虑过重不适合做执行） |
| 眼神 | 柔和稳定 | 眼神太犀利/有攻击性（做执行容易跟人起冲突） |

**一票否决：** 简历卷边+指甲有污垢+嘴角习惯性下垂→对细节不care+负能量→职能岗完全不匹配。

---

### 高级管理/合伙人岗

**核心痛点：** 格局、授权能力、情绪稳定性（逆商）。

**最佳人格类型：** 将帅型（CEO/总裁）、将帅×先锋型（市场驱动CEO）、将帅×谋士型（技术驱动CEO）。

**慎用类型：** 伪将帅（金玉其外型，面试最容易漏判）、谋士型（做CEO需要配强力COO基石型，否则执行会崩）、纯先锋型（0-1强但1-10弱，转型期可能驾驭不了）。

| 观察场景 | 面相/行为信号 ✅ | 面相/行为信号 ❌ |
|---|---|---|
| 谈竞争对手 | 客观承认优势，分析值得学习之处 | 贬低、嘲讽对手 |
| 谈下属过错 | "主要是我的责任，我没给够支持" | "那帮手下太笨，执行不到位" |
| 身体姿态 | 坐姿舒展，手臂自然放置 | 全程抱胸/跷二郎腿抖动 |
| 印堂 | 开阔（即使有悬针纹，也是决策压力纹） | 印堂窄+深纹+眉头紧锁（钻牛角尖） |
| 眼神 | 坚定有力，自然不施压 | 眼神浑浊/游离/或过度犀利 |
| 嘴角 | 平直或微上扬 | 嘴角下垂（长期不满） |
| 法令纹 | 向外上扬（真笑多，有领导魅力） | 向下弯（愁苦相，团队士气杀手） |
| 被质疑时 | 身体前倾，主动解释，开放心态 | 后靠/抱胸/皱眉/抢话 |

**高级管理特有的"松弛感"测试：**
- 一个真正的高管，在放松状态下是"松弛"的。他不会刻意展示权威，因为权威是内化的。
- 如果一个高管候选人在面试中表现得"过于完美"（每句话都精准、每个姿势都标准），要警惕——这可能是高度排练的结果。
- 最好的信号：他可以在严肃讨论和轻松玩笑之间自如切换。这说明他的情绪调节能力强，不紧绷。

## 三、内部提拔识人

**和外部招聘最大的不同：** 你有历史行为数据。

### 提拔前必看的三类数据

1. **面相变化趋势：** 调出他入职时的照片，对比现在。面相在变好（眉头舒展了/眼神定了/面色清亮了）=他在正向成长。面相在变差=提拔可能加速恶化。
2. **平级口碑：** 不是上司口碑，不是下属口碑，是平级口碑。上司可能偏心，下属可能奉承，平级最真实。
3. **危机时刻的行为记忆：** 回忆过去一年他最困难的时候做了什么——是挺身而出还是躲起来了？

### 提拔不同层级的识人重点

| 层级 | 最关键维度 | 为什么 |
|---|---|---|
| 提拔到一线主管 | 情绪稳定性 | 第一次带人，最怕自己先崩 |
| 提拔到部门总监 | 共情+掌控感 | 要能理解下属+能对上争取资源 |
| 提拔到VP/合伙人 | 四维全部 | 差一个维度都会在1-2年内出问题 |

### 提拔决策的六型校验

在提拔决策中增加一步：检查这个人被提拔后，和上级/平级/下属的生克关系（详见 `personality-model.md` 第三篇）。

**提拔前必问：**
- 他的上级是什么类型？会不会产生相克？
- 他将要管理的下属中，有没有和他相克的类型？
- 如果他是成长型，他还有多少成长空间？提拔会加速还是打断他的成长？

## 三-B、团队搭配速查

**搭建团队时的生克原则（详见 `personality-model.md` 第三篇）。**

### 最小核心团队的四角色

| 角色 | 最佳类型 | 为什么 |
|---|---|---|
| **拍板的** | 将帅型 | 能做决策且愿意担后果 |
| **搞钱的** | 先锋型 | 能打开局面，带来客户和资源 |
| **做出来的** | 谋士型 | 能深度思考，把想法变成产品 |
| **守住的** | 基石型 | 能持续交付，不让后院起火 |

### 常见搭配陷阱

| 搭配 | 风险 | 补救 |
|---|---|---|
| CEO将帅 + COO将帅 | 两个都想做主，半年内必有一战 | 明确分工边界，或其中一个必须让 |
| CEO将帅 + 全队先锋 | 将帅压不住一群先锋，组织混乱 | 必须至少配一个基石型稳住运营 |
| CEO谋士 + COO谋士 | 都在思考没人执行 | 必须配一个基石型，或者CEO转型将帅×谋士 |
| 全基石团队 | 稳如泰山，永远不爆发 | 必须引入一个先锋型或成长→先锋型 |

## 四、问题员工识别

### 面相预警信号（出现2个以上要重点关注）

- 面色突然变差（之前清亮→现在暗沉）：可能在做什么让你后悔的事
- 眼神开始飘忽（之前定→现在飘）：可能有了离心
- 嘴角越来越下垂：抱怨在积累
- 眉头越来越紧：压力失控或在谋划什么
- 对你的镜像反应消失：以前会模仿你的积极姿态，现在无所谓了

### 离职前兆（面相对比）

- 之前对公司话题有反应（眼睛亮、身体前倾）→现在无动于衷
- 之前和你交流时眼神自然→现在回避对视
- 之前姿态舒展→现在开始缩（减少存在感=准备消失）
- "善能改相"的反向：加入公司后面相一直在变差

## 五、裁员决策的识人辅助

**决定裁谁时，除了绩效数据，补充面相信号：**

1. **面相在变好的人：** 绩效暂时低但面相在改善=还在成长，给时间
2. **面相在变差的人：** 绩效高但面相持续恶化=可能在燃烧自己/可能快要爆了，需要干预
3. **面相从未好过的人：** 绩效低+面相差=没有改善可能，优先裁
