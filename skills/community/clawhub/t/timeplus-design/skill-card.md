## Description: <br>
Build user interfaces that match the Timeplus Console look and feel using documented design tokens, CSS, Tailwind configuration, and React component examples. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and designers use this skill to generate, style, or review Timeplus Console-aligned web UI. It provides exact design tokens, component rules, and ready-to-adapt CSS, Tailwind, and React implementations for product interfaces. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Copied examples may introduce third-party font requests or optional npm command execution into a project. <br>
Mitigation: For privacy-sensitive products, self-host Inter or document Google Fonts use; run optional npx commands only after approving the npm package for the environment. <br>
