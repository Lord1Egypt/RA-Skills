# 行业基准值与判断阈值

> 本文件按季度更新，反映当前平台和行业的实际数据基准。
> 上次更新：2026年5月

---

## 退款率

| 区间 | 判断 | 动作 |
|---|---|---|
| <5% | 健康 | 正常运营 |
| 5-10% | 需关注 | 排查个别退款高的商品 |
| 10-15% | 需排查 | 逐商品分析退款原因 |
| 15-25% | 紧急处理 | 暂停增量投放，先堵退款漏洞 |
| >25% | 先堵漏再放量 | 停止所有增量投放，专注退款率压降 |

**特殊说明**：抖音营销期的结构性退款（冲动消费+到期自动退款）不算体验问题，需与到店后退的"体验退款"分开统计。

## 核销率

| 区间 | 判断 | 动作 |
|---|---|---|
| >70% | 优秀 | 可放量 |
| 50-70% | 正常 | 持续优化 |
| 30-50% | 预警 | 排查核销低的商品和视频 |
| <30% | 严重 | 暂停投放，分析原因 |

## 5秒完播率

| 区间 | 判断 | 备注 |
|---|---|---|
| >25% | 优秀 | 历史数据：此区间千次成交>1700元 |
| 20-25% | 良好 | 转化效率甜区 |

## 达人筛选标准

### 粉丝量层级与ROI（2026Q2数据）

| 粉丝层级 | 平均千次播放成交 | 判断 |
|---|---|---|
| 5-10万 | ~970元 | 🏆最佳ROI，主力投放 |
| 3千-1万 | ~310元 | ✅高性价比 |
| 1-5万 | ~250元 | ✅稳定输出 |
| 10万+ | ~66元 | ❌贵且转化低，仅限垂直美食号 |
| <3千 | 波动大 | ⚠️样本小，可铺量但不依赖 |

### 筛选维度

| 维度 | 标准 | 权重 |
|---|---|---|
| 粉丝量 | 5千-10万为主力 | 高 |
| 内容类型 | ≥80%美食探店，禁止泛娱乐/搞笑 | 高 |
| 近30天带货 | 有本地餐饮成交记录优先 | 高 |
| 5秒完播率 | ≥20% | 中 |
| 粉丝地域 | 本地≥60% | 中 |
| 历史退款率 | <15%可复投，>30%拉黑 | 高 |

## 商品套餐定价梯度参考

| 定位 | 折扣范围 | 说明 |
|---|---|---|
