# 假设验证所需的统计检验与数据要求

目的：**确保数据能够满足你计划进行的统计分析，避免"收完数据发现跑不了"的尴尬。**

---

## 检验与数据对应表

### 1. 差异假设（H1: X → Y）

| 假设 | 检验 | 必需数据 |
|------|------|---------|
| 两组独立 | 独立样本 t 检验 | X（分组标签）、Y（连续分数）、每组 N |
| 两组配对 | 配对样本 t 检验 | T1Y、T2Y（配对数据） |
| 多组被试间 | 单因素 ANOVA | X（分组标签，k≥3）、Y |
| 多组被试内 | 重复测量 ANOVA | 同一被试在 k 个条件下的 Y |
| 混合设计 | 混合 ANOVA | 组别 × 时间点的交叉数据 |
| 分类因变量 | 卡方检验 | X（分组）、Y（分类计数） |

**常见遗漏：**
- ⚠️ 分组标签的编码（1=实验组，2=对照组）
- ⚠️ 人口学变量（协变量）
- ⚠️ 基线分数（前测作为协变量）

---

### 2. 相关/预测假设（H1: X → Y）

| 假设 | 检验 | 必需数据 |
|------|------|---------|
| 两个变量相关 | Pearson/Spearman | X、Y（连续/等级） |
| 多元预测 | 多元回归 | X1, X2...（连续/虚拟变量）、Y |
| 层级预测 | 分层回归 | Model 1 变量 + Model 2 变量 |
| 曲线关系 | 多项式回归 | X、X²（或其他非线性项） |

**常见遗漏：**
- ⚠️ 所有变量的完整数据（不能有缺失）
- ⚠️ 控制变量的数据（若做偏相关）
- ⚠️ 多重共线性检查（VIF）

---

### 3. 中介假设（H1: X → M → Y）

| 假设 | 检验 | 必需数据 |
|------|------|---------|
| X → Y 总效应 | 简单回归 | X、Y |
| X → M（a路径） | 简单回归 | X、M |
| M → Y（b路径） | 多元回归（控制 X） | X、M、Y |
| 直接效应（c'） | 多元回归（加入 M） | X、M、Y |
| 间接效应（a×b） | Bootstrap | 以上全部 |

**数据要求：**
- X、M、Y 均为连续变量（若为分类需虚拟化）
- 样本量 N ≥ 50（Bootstrap 需要）
- 中介变量 M 必须是 X 与 Y 之间的合理机制

**常见遗漏：**
- ⚠️ Bootstrap CI 不含 0 才能说"显著中介"
- ⚠️ 需要汇报标准化还是非标准化系数
- ⚠️ 若有多个中介，需分别检验 + 联合检验

---

### 4. 调节假设（H1: X × W → Y）

| 假设 | 检验 | 必需数据 |
|------|------|---------|
| 主效应 X | 多元回归 | X、Y |
| 主效应 W | 多元回归 | W、Y |
| 交互效应 X×W | 层次回归 | X、W、X×W（中心化后的乘积） |

**数据要求：**
- 自变量中心化（X - X̄）再乘积
- 调节变量 W 可以是连续或分类
- 若 W 为连续，Johnson-Neyman 法找显著区间

**常见遗漏：**
- ⚠️ 交互项不显著不等于"没有调节"
- ⚠️ 需要简单斜率分析（simple slope）
- ⚠️ 画交互图（调节效应可视化）

---

### 5. 多因素设计

| 假设 | 检验 | 必需数据 |
|------|------|---------|
| 主效应 A | 两因素 ANOVA | A（分组）、B（分组）、Y |
| 主效应 B | 两因素 ANOVA | A、B、Y |
| A×B 交互 | 两因素 ANOVA | 完整的 A×B×Y 数据 |
| 简单主效应 | 简单主效应检验 | 固定 A 看 B 的各水平 |

**常见遗漏：**
- ⚠️ 完整交叉数据（每个被试在各条件组合下的值）
- ⚠️ 顺序效应（若为混合设计）
- ⚠️ 重复测量需检验球形性

---

### 6. 追踪/纵向设计

| 假设 | 检验 | 必需数据 |
|------|------|---------|
| 群体差异（时间） | 重复测量 ANOVA | 多次测量的 Y（T1, T2...） |
| 群体 × 时间交互 | 混合 ANOVA | 组别 × 时间点交叉 |
| 增长曲线 | 增长模型/HLM | 多次测量 Y、time、被试 ID |

**常见遗漏：**
- ⚠️ 被试唯一 ID（用于匹配多次测量）
- ⚠️ 每次测量的时间点标签
- ⚠️ 脱落被试的处理方案

---

## 数据类型匹配检查清单

在规划数据收集时，确认每个变量满足：

### 连续变量（t 检验、ANOVA、相关、回归）
- [ ] 是否为数值型？（分数、反应时等）
- [ ] 数据分布是否满足假设？（正态/近似正态）
- [ ] 是否存在异常值？（±3 SD 之外）
- [ ] 缺失值如何处理？

### 分类变量（卡方检验、逻辑回归）
- [ ] 分组定义是否清晰？（1=男，2=女）
- [ ] 每组频数是否足够？（期望频数 ≥ 5）
- [ ] 多分类变量是否需要参照组？

### 中介/调节变量
- [ ] 中介变量 M 与 X、Y 的逻辑关系是否合理？
- [ ] 调节变量 W 的类型？（连续/分类）
- [ ] 是否需要虚拟化处理？

---

## 分析前的数据质量检查

数据收集完成后、分析前，必须检查：

### 基本检查
- [ ] 变量命名是否规范？
- [ ] 数据类型是否正确？（数值 vs 文本）
- [ ] 缺失值比例？（超过 20% 需谨慎处理）

### 前提假设检查
- [ ] 正态性（Shapiro-Wilk，N < 5000）
- [ ] 方差齐性（Levene 检验）
- [ ] 球形性（重复测量，Mauchly 检验）
- [ ] 独立性（被试间是否独立）

### 作答质量检查
- [ ] 作答时间是否合理？（太快可能是乱答）
- [ ] 相同答案比例是否过高？（>80% 相同答案）
- [ ] 是否存在整题缺失？

## 常见"跑不了"的窘境与预防

| 窘境 | 原因 | 预防 |
|------|------|------|
| "两组样本量差太大" | 没有控制入组 | 记录每组实际 N |
| "自变量不是正态分布" | 分类变量没编码 | 提前规划虚拟变量 |
| "中介分析跑不出" | 变量间相关不显著 | 先做相关分析确认 |
| "重复测量缺了好几个被试" | 没有追踪 ID | 每个被试有唯一 ID |
| "调节效应检验不显著" | 交互项共线性 | 中心化后再乘积 |
| "效应量太小" | 样本量不够/效应本来小 | 事前检验力分析 |
