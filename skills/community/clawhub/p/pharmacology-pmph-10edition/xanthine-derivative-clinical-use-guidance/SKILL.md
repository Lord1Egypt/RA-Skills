# 茶碱类药物临床应用指征与限制

## 适用场景
仅在以下情况考虑使用茶碱类药物：
- 慢性支气管哮喘需预防急性发作，且β₂受体激动药控制不佳；
- COPD患者伴有喘息、右心功能不全（心源性哮喘）或肺动脉高压；
- 中枢型睡眠呼吸暂停综合征（由脑部疾病或原发性呼吸中枢病变引起）。

> **注意**：哮喘急性发作首选β₂受体激动药，茶碱不作为一线治疗。

## 执行步骤

1. **评估使用指征与禁忌**
   - 确认患者属于适用对象（支气管哮喘、COPD 或中枢型睡眠呼吸暂停）；
   - 排除禁忌：血药浓度 >20 mg/L、活动性胃溃疡、严重心律失常；
   - 确认存在前提条件之一：β₂激动药控制不佳、右心功能不全/肺动脉高压、或中枢性通气障碍。

2. **根据疾病类型选择给药方案**
   - **支气管哮喘**：
     - 急性重度或持续状态：静脉注射氨茶碱快速缓解；
     - 慢性预防：口服氨茶碱，优选缓释/控释制剂（尤其夜间哮喘）。
   - **COPD**：
     - 伴右心衰竭或肺动脉高压者优先考虑；
     - 利用其扩张肺动脉、强心、利尿及支气管扩张多重作用。
   - **中枢型睡眠呼吸暂停**：
     - 使用茶碱类中枢兴奋作用增强通气，改善低通气症状。

3. **选择合适制剂**
   - **氨茶碱**：水溶性好，可用于静脉注射，但胃肠道刺激大、碱性强；
   - **胆茶碱**：口服吸收佳，胃肠道反应少，心脏及中枢兴奋作用较弱；
   - **缓释/控释制剂**：维持稳定血药浓度，减少副作用，适用于长期维持治疗。

4. **监测与调整**
   - 定期监测血药浓度（治疗窗通常为 10–20 mg/L）；
   - 观察胃肠道、心血管及中枢神经系统不良反应；
   - 根据疗效与耐受性调整剂量或更换制剂。

> 若患者不符合指征、存在禁忌或出现毒性反应，立即停用并改用其他治疗方案。
