# 法洛四联症青紫机制与分流方向判定

## 适用对象
- 法洛四联症患者
- 新生儿先天性心脏病（发绀型）

## 前提条件
- 存在室间隔缺损
- 右心室流出道狭窄
- 主动脉骑跨

## 禁忌场景
- 不适用于无肺动脉狭窄的单纯室间隔缺损
- 不适用于非发绀型先天性心脏病

## 执行步骤

1. **判断室间隔缺损性质**：确认缺损为非限制性，使左、右心室压力基本相等。

2. **评估肺动脉狭窄程度以确定分流方向**：
   - 肺动脉狭窄较轻 → 左向右分流为主 → 患者可无明显青紫；
   - 肺动脉狭窄严重 → 右向左分流显著 → 出现明显青紫。

3. **分析青紫加重机制**：
   - 主动脉骑跨于两心室之上，直接接受右心室静脉血，输送到全身；
   - 肺动脉狭窄导致肺循环血流减少，气体交换不足，进一步加重低氧血症。

4. **评估动脉导管状态对青紫表现的影响**：
   - 动脉导管未闭时，肺循环血流减少程度较轻，青紫可不明显；
   - 动脉导管关闭后，加上漏斗部狭窄逐渐加重，青紫日益明显。

5. **识别长期缺氧后果**：持续6个月以上发绀可导致杵状指/趾，并刺激骨髓代偿性红细胞增多，血液黏稠度增高，易引发脑血栓或细菌性脑脓肿。

## 关键变量
- **肺动脉狭窄程度**（轻度/中度/重度）：决定分流方向
- **动脉导管状态**（开放/关闭）：影响早期青紫表现
- **主动脉骑跨比例**（连续变量）：影响右心室血进入体循环量

## 输出目标
根据狭窄程度和导管状态，判断青紫严重程度及分流方向，用于指导临床干预时机。
