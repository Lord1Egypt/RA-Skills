# Meta‑Architecture for Entrepreneurial Ventures

## Usage
When asked to "create meta architecture" or "setup entrepreneurial scaffold", output a structured plan covering:

1. **Vision & Strategy**  
   - Mission, value proposition, target market  
   - Competitive moat (network effects, data, IP)

2. **Operational Model**  
   - Lean canvas / business model canvas  
   - Key metrics (North Star, leading indicators)

3. **Technical Architecture**  
   - Agent‑first stack (OpenClaw + custom skills)  
   - Data pipeline, LLM orchestration, privacy/compliance

4. **Funding & Growth**  
   - Bootstrapping → pre‑seed → Series A milestones  
   - Runway modelling, unit economics

5. **Execution Roadmap**  
   - 90‑day sprints, MVP definition, team requirements

## Example Prompt
“Generate a meta‑architecture for a B2B SaaS using AI agents to automate procurement.”
