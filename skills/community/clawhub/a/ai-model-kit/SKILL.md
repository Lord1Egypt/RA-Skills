# AI模型制作指南

全球模型市场超400亿美元，万代年销超5000万盒高达。从HG素组到PG改造，高达/军模/战棋全覆盖。

## 核心能力

### 1. 高达Gunpla
- **系列选择**：EG/HG/RG/MG/PG/MEGA/FM/SD的特点与难度
- **素组入门**：两刀流/笔刀修水口/渗线/贴纸/消光
- **水口处理**：神手/将军/笔刀/砂纸(400→600→800→1000)
- **无缝处理**：绿盖/补土/打磨的完整流程
- **渗线技法**：田宫渗线液/珐琅漆/勾线笔
- **水贴**：软化剂/棉签/消光统一光泽
- **改造入门**：刻线/胶板/补品/蚀刻片
- **PB网限**：限定款的选购与溢价规律

### 2. 涂装技法
- **补土打底**：水补土/底漆的必要性与选择
- **笔涂**：AV/田宫/酋长大陆的水性漆笔涂
- **喷涂**：泵/喷笔/排风/遮盖的完整设备
- **预置阴影**：MAX涂装/CM/阴影的喷涂技巧
- **旧化效果**：渍洗/干扫/掉漆/滤镜/锈蚀
- **高光与渐变**：透明漆/珍珠漆/变色龙漆
- **保护漆**：消光/半光/光泽的应用场景

### 3. 军事比例模型
- **坦克AFV**：1/35/1/48/1/72比例的推荐
- **舰船**：1/350/1/700的水线/全船/RC
- **飞机**：1/48/1/72的座舱/蒙皮/起落架
- **兵人**：1/35兵人涂装/场景搭配
- **品牌选择**：田宫/长谷川/威龙/小号手/MinArt/MENG
- **考证与改造**：历史照片/蚀刻片/树脂改件

### 4. 场景与地台
- **地台制作**：石膏地形/草皮/沙砾/水面
- **建筑废墟**：泡沫板+刻线+上色
- **植被与树木**：海绵/静电植草/铜丝树
- **水面效果**：UV胶水面/海浪
- **雪景/沙漠**：苏打粉雪/石膏沙漠
- **情景构图**：故事性/视线引导/黄金比例

### 5. 战棋涂装
- **Games Workshop**：40K/AoS的军队涂装
- **薄涂技法**：多遍薄涂/叠色/过渡
- **NMM非金属色**：NMM金/银/钢的伪金属涂法
- **OSL光源**：光源效果/发光武器/火焰
- **湿混/点画/罩染**：进阶技法的掌握
- **军队方案**：各战团的配色与涂装方案

### 6. 工具与耗材
- **基础工具**：剪钳/笔刀/砂纸/镊子/手钻
- **涂装工具**：喷笔/气泵/排风箱/湿盒/面相笔
- **耗材清单**：胶水/补土/漆/溶剂/遮盖带
- **工作台**：切割垫/台灯/收纳/眼镜放大镜
- **入门套装**：按300/800/2000元预算推荐
