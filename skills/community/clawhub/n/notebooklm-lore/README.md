# notebooklm-lore (OpenClaw skill)

This is the **OpenClaw/agent skill wrapper** for the Lore project.

- Project repo: https://github.com/prantikmedhi/lore
- What you get here: agent instructions (no executable code)

## What this skill does

When installed into OpenClaw, it gives the agent a clear, safe, repeatable playbook to:

- clone/update the Lore repo
- install it locally with Python
- ask the user to complete NotebookLM browser login
- verify the install
- configure MCP using the repo’s `.mcp.json`

## Install

```bash
clawhub install notebooklm-lore
```
