# Image Layout — WeChat Moments Poster

## 文本一致性规则（最高优先级）

图片文字必须与 copy.md 逐字一致，**唯一例外**：
- **标题**（Zone 2）：图片中可去除 emoji（copy 中的 emoji 在标题中不适用于图片排版）
- 卖点/价格/讲师/联系方式等所有其他 zone：必须逐字匹配

## CTA Zone 逻辑

| 情况 | 图片 Zone 6 | copy.md CTA |
|------|------------|-------------|
| 有二维码 | "扫码咨询" | "私信咨询" |
| 无二维码 + 有联系方式 | 显示联系方式 | "私信咨询" |
| 无二维码 + 无联系方式 | **省略整个zone** | "私信咨询" |

## 标题区规则（重要）

- 图片标题**不得有 emoji**
- 提取 copy 第1行标题时，移除 emoji 符号
- 例如 copy: "🚀 打造懂AI的超级营销个体" → 图片使用："打造懂AI的超级营销个体"

图片中的文字必须与 copy.md 中的文字**逐字完全一致**。

生成 prompt 时，将 copy.md 的内容**逐字复制**进对应 zone：
- Zone 2 = copy 第1行（标题）
- Zone 3 = copy 讲师信息（如有）
- Zone 4 = copy 中每个卖点行（顺序和文字均不得改动）
- Zone 5 = copy 价格区块（如 copy 无价格，此 zone 整个省略）
- Zone 6 = copy CTA 行（如无，此 zone 整个省略）
- Zone 7（二维码）= 仅当 qrcode_enabled=true 时出现

**禁止**：缩写 / 改写 / 省略 / 自行补充任何 copy 中的内容。

## Prompt Writing Guide

### Zone 1 — 装饰/Banner 区

```python
# 风格映射
minimal: "abstract tech illustration - subtle circuit/geometric pattern in very faint warm gold lines on deep navy, creating visual anchor"
notion: "notion-style icon, soft shadow, geometric"
warm: "warm gradient strip, organic shape, soft"
blueprint: "blueprint grid lines, schematic pattern, technical"
editorial: "bold color block, strong contrast, magazine layout"
```

**必须**：Zone 1 不只是细线条，需要有**Banner 级别的矢量背景图**（几何/科技/抽象风格），占据顶部区域，作为视觉锚点。

### 整体结构原则

**一体化原则**：标题嵌入 Banner 区，Banner 与画布无边界融合，不单独成层。卖点与价格在视觉上属同一层级（卡片式），无独立价格 zone。

**2层层级原则**：
- Level 1：Banner 标题（主视觉）
- Level 2：卡片卖点 + 底部价格/联系方式（次视觉）

**一体化设计**：Banner 背景与画布融为一体（渐变、色块延伸），不出现"嵌入"感。卡片与背景色彩协调，有边框或阴影区分。

**禁止**：单独价格 zone / 单独 CTA zone / 标题与 Banner 割裂。

### Zone 3 — 副标题/讲师区

- **内容来源**：copy.md 副标题行（如无讲师）或讲师行
- **副标题字体 < 标题字体**（层级：标题 > 副标题）
- 字号：标题的 70-90%
- **如无讲师信息，整个 zone 省略，不留空白占位**

### Zone 4 — 卖点区（模块）

- **内容来源**：copy.md 卖点行，**逐字逐行对应** + 可添加拓展描述
- **字体层级（3层，简化为原则）**：
  - Level 1（主标题）：最大，深色
  - Level 2（卖点名称 + 价格）：中等，暖色/金色
  - Level 3（描述）：最小，浅灰色
  - 禁止增加更多层级
  - 4个模块之间视觉权重相近，简洁不拥挤
- **每个模块需有 SVG 线性图标（line art，非emoji，颜色用暖金色 #C7A27C）**
  - 图标尺寸：**迷你**（不超过模块宽度的15%），仅作视觉标记，不喧宾夺主
  - 风格：简约线条风，有质感不过度装饰
- **保留25%以上留白**：模块区域内不得拥挤
- **背景**：深色渐变（不得纯黑），可叠加几何矢量图形
- **禁止**：纯黑色块背景 / 标题字体大于副标题 / 添加copy中没有的卖点

### Zone 5 — 价格区

- **内容来源**：copy.md 价格区块，**逐字不变**
- 格式：现价大字 + 原价小字划线 + "早鸟价"pill badge/tag
- **价格区上下需有足够留白**（不拥挤）
- **如无价格区块，整个 zone 省略，不留空白**

### Zone 6 — CTA / 联系方式区

**三选一**：
1. 有二维码 → CTA "扫码咨询"
2. 有联系方式（无二维码）→ 显示联系信息（如"张先生 13912345678"），无CTA文字
3. 无二维码 + 无联系方式 → **整个zone省略，不留任何占位**

### Zone 7 — 二维码区

- 位置：右下角
- 尺寸：占图片 10-15%
- 背景：白色边框隔离
- **仅当 qrcode_setting.txt 中 `qrcode_enabled: true` 时生成**
- **如无二维码，整个 zone 省略**

## 排版原则

| 原则 | 说明 |
|------|------|
| 留白 | 至少 30% 空白区域 |
| 层次 | 字号大小建立阅读顺序 |
| 对齐 | 同级元素左对齐或居中 |
| 一致性 | 同一 zone 内风格统一 |
| 省略 | 缺失信息对应 zone 整个省略，不留空白占位 |
