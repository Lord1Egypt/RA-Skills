## Description: <br>
Automates filename checks, ECG image upload, demographic form entry, diagnosis extraction, and Excel result generation for ECG image batches. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[fanglucheng](https://clawhub.ai/user/fanglucheng) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Operators use this skill to process ECG image files whose names encode date, age, sex, and patient name, then run a local browser automation script that submits the images to an external ECG service and records diagnosis text in an Excel file. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The workflow may upload sensitive ECG images and patient demographics to a third-party website. <br>
Mitigation: Use only with explicit patient consent and when the external service is approved for the medical data involved. <br>
Risk: The required filename pattern can place personal details in local filenames and generated records. <br>
Mitigation: Avoid unnecessary identifiers, restrict folder access, and handle generated spreadsheets under the same controls as medical records. <br>
Risk: The script stores browser profile data in the image folder and uses browser automation to interact with the external site. <br>
Mitigation: Run it in an isolated working folder, review the browser profile contents, and remove local session data when processing is complete. <br>
Risk: The generated ECG diagnosis text is produced by an automated external workflow. <br>
Mitigation: Have a qualified clinician review results before using them for medical decisions. <br>
