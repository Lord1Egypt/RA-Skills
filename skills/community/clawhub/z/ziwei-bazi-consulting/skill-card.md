## Known Risks and Mitigations: <br>
Risk: The skill delegates the actual consultation work to destiny-fusion-pro, so trust and behavior depend on that target skill. <br>
Mitigation: Install and use this alias only when legacy ziwei-bazi-consulting calls are expected to run destiny-fusion-pro, and review the target skill before relying on outputs. <br>
