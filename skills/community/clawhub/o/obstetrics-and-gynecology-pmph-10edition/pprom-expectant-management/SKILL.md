# 未足月胎膜早破期待治疗方案

## 何时使用
- 孕周28–33周
- 确诊PPROM
- 无绒毛膜羊膜炎、胎盘早剥、胎儿窘迫
- GBS阴性或阳性但<34周

## 执行步骤
1. **一般监测**：
   - 避免宫颈指检
   - 监测体温、宫缩、胎心、阴道流液
   - 定期查血常规、羊水量、胎监
2. **促胎肺成熟**（<34周）：
   - 地塞米松 6mg q12h ×4次 或 倍他米松 12mg q24h ×2次
3. **预防感染**：
   - 青霉素类或大环内酯类，5–7天
   - GBS阳性首选青霉素
4. **抑制宫缩**（<34周）：
   - 硝苯地平或利托君，维持48小时（完成激素疗程+转运）
   - 感染或胎盘早剥时禁用
5. **胎儿脑保护**（<34周）：
   - 硫酸镁静脉滴注（负荷4–6g，维持1–2g/h）
6. **终止妊娠指征**：
   - 母体发热≥38℃+感染表现
   - 胎儿窘迫、胎盘早剥、脐带脱垂
   - 孕周达37周

## 输出建议
执行个体化期待治疗，目标延长孕周至34周以上，保障母胎安全。
