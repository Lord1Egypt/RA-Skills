## 能量充值话术

### 每日正向输入（必须每天说）

**感激表达**：
- "谢谢你今天做了...，让我感觉很被照顾"
- "你今天...做得特别好，我注意到并很欣赏"
- "有你在，我觉得很安心"

**欣赏表达**：
- "我觉得你...这点特别厉害"
- "你处理...的方式让我学到了"
- "和你在一起，我觉得...（具体感受）"

**具体化原则**：避免"你真好""你很棒"等模糊表达，必须具体到事件+感受

### 每周高质量连接话术

**深度交流开场**：
- "这周有没有什么事让你特别有成就感？我想听听"
- "最近有什么想法或困惑想和我聊聊吗？"
- "我们来聊聊各自的近况吧，我很想了解你"

**亲密时刻**：
- "这周我们找个时间单独待一会吧，像以前那样"
- "我想和你一起做...，你感兴趣吗？"

## 需求表达话术

### 核心原则：观察+影响+具体请求

**模板结构**：
> "当...（具体观察）的时候，我感到...（情绪），因为...（需求），我希望...（具体请求）"

### 场景化示例

**场景1：需要陪伴**
- ✗ 错误："你从来都不陪我"
- ✓ 正确："这周你每天加班到很晚（观察），我感觉有点孤单（情绪），我需要你的陪伴来感到被重视（需求），周末能不能抽出半天我们一起做点什么都行？（请求）"

**场景2：需要支持**
- ✗ 错误："你一点都不支持我"
- ✓ 正确："当我决定...（具体事），我有些不确定（观察），我需要你的意见和支持（需求），你愿意听我说说并给我一些建议吗？（请求）"

**场景3：需要空间**
- ✗ 错误："你烦不烦啊，能不能别管我"
- ✓ 正确："我今天工作特别累（观察），需要一个人静一静来恢复能量（需求），给我1-2小时独处时间，然后我再回来陪你可以吗？（请求）"

**场景4：需要情绪回应**
- ✗ 错误："你根本不懂我"
- ✓ 正确："我今天在公司遇到了...（事件），我跟你说是想让你理解我的感受（需求），而不是给建议，你能先抱抱我说'辛苦了'吗？（请求）"

## 冲突降噪话术

### 冲突前：预防性表达

**设置边界**：
- "我想和你聊聊...，但担心会变成争吵，我们能不能约定就事论事？"
- "在说之前，我想请你先听我说完，不要打断我，好吗？"

**表达意图**：
- "我不是要责怪你，我是想解决我们的问题"
- "我需要你帮我理解你的想法"

### 冲突中：控制节奏

**叫停信号**：
- "我觉得我们有点激动了，暂停5分钟好不好？"
- "我们现在说的话可能会伤害对方，我们冷静一下再谈"

**回归事实**：
- "我们现在不是在说'谁对谁错'，而是在解决'这件事怎么做'"
- "我想确认一下，你说的是...这个意思吗？"

**表达感受而非评判**：
- ✗ 错误："你就是自私"
- ✓ 正确："你没有征求我的意见就决定了，这让我感觉不被尊重"
- ✗ 错误："你从来不考虑我的感受"
- ✓ 正确："这次你没有问我的想法，我感到有点失落"

### 冲突规则（双方约定）

1. 不说"你总是..."、"你从来都..."
2. 不翻历史旧账
3. 不在冲突时提离婚/分手
4. 情绪失控时任何一方可以叫停
5. 冲突后24小时内必须有一次修复对话

## 冲突修复话术

### 冲突后24小时内必须完成

**第一步：承担责任**
- "昨天我说了...，这句话说得不合适，我不该..."
- "我承认我在...这件事上有问题，我需要改进"

**第二步：解释当时状态**
- "当时我那么激动是因为...（具体原因），不是针对你"
- "我当时感到...，所以反应过度了"

**第三步：表达歉意**
- "对不起，让你受委屈了"
- "让你有那样的感受，我很抱歉"

**第四步：说明需求**
- "我需要的是...，你能理解吗？"
- "下次遇到这种情况，我希望我们能..."

**第五步：提出方案**
- "以后遇到类似情况，我们可以..."
- "我们来约定一个信号，当...的时候，我们就暂停"

### 修复对话模板

```
"我昨天想了很多，关于我们之间的争执。
我想先为...（具体行为）道歉，我的做法...（具体问题）。
当时我是想表达...，但方式让你感觉...（对方感受）。
我理解你为什么会...（对方反应）。
以后遇到这种情况，我希望能...（改进方案）。
你愿意告诉我你当时的感受和需求吗？"
```

## 身份重建话术

### 当伴侣自我否定时

**原则：不否定感受，提供正向确认**

- "我听到你说自己很失败（复述），我能感受到你现在很沮丧（情绪确认）"
- "我想告诉你我的视角：你做的...（具体事实），这些我都看在眼里"
- "你可能现在觉得自己不够好，但在我眼里，你...（具体优点+事例）"
- "一时的挫折不代表你不行，我们来看看怎么解决这个问题"

### 当自己被否定时

**原则：温和但坚定地设立边界**

**回应模式**：
1. 确认情绪："我知道你因为...感到...（理解情绪）"
2. 设立边界："但'你很没用'这样的评价让我很难接受（边界）"
3. 引导回到具体问题："我们可以就...这件事来讨论怎么做吗？（回到事）"

**示例**：
- "我知道你对我没做成这件事很失望（理解），但'你真没用'这样的话让我感觉很受伤（边界）。我们能不能说说这件事哪里出了问题，下次怎么改进？（回到事）"

### 给予身份确认

**主动给予**：
- "你知道吗，我最欣赏你的一点是...（具体优点+事例）"
- "你今天...让我觉得嫁/娶对人了"
- "在我眼里，你是...的人（正面身份定义）"

## 能量断供重启话术

### 当关系进入"无交流"状态时

**打破僵局第一步**：
- 物理接近法：到对方身边坐下，不需要说话，先待着
- 行为传递法：默默做一件对方需要的事（倒杯水、放好洗澡水）

**开口话术**：
- "我发现我们最近好像有点疏远了，你也有这种感觉吗？"
- "我想跟你谈谈我们之间的事，不是要吵架，是想让我们的关系变好"
- "我最近在想我们的关系，我觉得我们需要改变一些什么"

**重建连接**：
- "我最近在想，我们上次一起做...是什么时候？我有点怀念那种感觉"
- "我们能不能重新开始？每周找一个时间，只属于我们两个人"
- "我知道我之前...做得不好，我想改变，你愿意给我一个机会吗？"

### 无性状态重启

**原则：不要直接谈性，从情感连接开始**

- "我发现我们最近好像缺少那种...的感觉，你注意到了吗？"
- "我有点想找回我们以前的...（具体美好的回忆），你愿意和我一起试试吗？"
- "最近工作太忙了，我们的关系好像变得很'功能性'了，我想要更多"

---

## 话术使用原则

1. **具体化**：每句话都要有具体事件，不要模糊评价
2. **表达感受**：用"我感到..."而非"你让我..."
3. **聚焦需求**：明确说出需要什么
4. **提出方案**：不仅说问题，还要给解决方向
5. **保持真诚**：话术是工具，但内核是真实
