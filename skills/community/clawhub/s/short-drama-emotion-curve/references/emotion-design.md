# 情绪设计原理 (Emotion Design Principles)

## 一、核心目标：建立代入感

短剧观众的核心诉求不是"看故事"，是**照镜子**。

> **"和我处境差不多的女人，活出我不敢活的样子。"**

这个公式是所有情绪设计的底层逻辑。角色的处境要和目标观众的生活经验有交集——她面对的困境要让观众觉得"这就是我"。

### 代入感的三个层次

| 层次 | 定义 | 在短剧中的体现 |
|------|------|--------------|
| **处境认同** | "她的处境我熟悉" | 用日常细节（催费短信、银行余额）而非戏剧性事件开场 |
| **情感认同** | "她的感受我懂" | 不让她诉苦，用行为展示内心挣扎 |
| **行动认同** | "如果是我，我也会这么做" | 她的选择要让观众能理解，即使不认同 |

### 前5秒抓人核心

**她的处境本身就是困境，不是意外发生。**

观众在第一帧就要知道：这个女人不容易。不是听她说完才明白，而是从她的眼神、动作、环境里自己感受到。

**反面做法：** 用旁白介绍"她是一个单亲妈妈" → 这是告知，不是展示。

**正面做法：** 
```
镜头：医院走廊
她坐在塑料椅上，隔壁床的小孩在哭
她看了眼手机，继续发呆
（不解释，让观众自己补全：她在这坐了一夜）
```

## 二、虐点设计 (Pain Points)

### 虐点的本质

虐点不是"让她受苦"，而是**把她的秘密/处境逼到台面上**。

关键区别：

- ❌ 外力介入制造冲突 → 观众觉得"编剧在搞事"
- ✅ 她的处境被逼到台面 → 观众觉得"这下怎么办"

### 虐点类型

| 类型 | 描述 | 示例 |
|------|------|------|
| **秘密暴露** | 最不想被知道的秘密被揭开 | 富家女在餐厅打工被同学撞见 |
| **处境临界** | 经济/情感压力到达极限 | 余额显示不足，下一秒就要付账 |
| **关系撕裂** | 亲密度高的关系出现裂缝 | 儿子说"我同学的妈妈比你穿得好" |
| **身份矛盾** | 双重身份无法调和 | 做保姆时遇见雇主是前男友 |
| **选择困境** | 两个都想要但只能选一个 | 孩子的学费 vs 自己的尊严 |

### 虐点的节奏

虐点出现太早 → 情绪还没建立，观众没有情感账户可以提取  
虐点出现太晚 → 拖沓，节奏失效

**黄金位置：集数的 60-70% 处**（即 60 秒集在 36-42 秒）

## 三、爽点设计 (Reward Points)

### 爽点的本质

爽点是情绪能量的释放。但释放的前提是**压得足够深**。

> 没有被压得喘不过气来的情绪，就没有真正爽的释放。

### 爽点类型

| 类型 | 描述 | 示例 |
|------|------|------|
| **逆袭打脸** | 之前看不起她的人被打脸 | 假阔太发现她才是真正的买家 |
| **真相揭露** | 被误解的真相终于明了 | 婆婆发现儿媳一直在偷偷寄钱回家 |
| **身份逆转** | 从被压制到掌控局面 | 她拿出证据，局势逆转 |
| **情感突破** | 压抑的情感终于表达 | 说出三年没说出口的那句话 |
| **边界建立** | 终于有底气说"不" | 她把辞职信放在桌上 |

### 爽点的节奏

- **压→弹**：爽点必须出现在充分虐点积累之后
- **不能提前**：观众还没心疼她，就不能让她反击——情绪会失重
- **集末爽点 vs 集初爽点**：  
  - 集末爽点（hook 爽）：留下悬念，让观众想看下一集  
  - 集初爽点（承接爽）：接上一集的爽点余波，维持观看动力

## 四、情绪节奏 (Emotional Pacing)

### 短剧的节奏限制

60-90 秒意味着：
- 没有时间做慢热铺垫
- 没有时间做情绪缓冲
- 每一秒都要有功能

### 节奏模型：SETUP → CONFRONTATION → CRISIS → HOOK

```
[  SETUP  ]  [ CONFRONTATION ]  [  CRISIS  ]  [  HOOK   ]
   0-15s        15-40s          40-75s      75-90s

   困境          内外冲突         临界点       悬念/
   展示          升级            情绪顶点     反转前最后一帧
```

### 时间分配建议

| 阶段 | 时长 | 核心任务 |
|------|------|---------|
| Setup | 0-15s | 用细节展示处境，建立初步代入感 |
| Confrontation | 15-40s | 冲突升级，可以有一个小虐点 |
| Crisis | 40-75s | 核心虐点，情绪到达临界点 |
| Hook | 75-90s | 画面本身制造悬念，留钩 |

### 声音节奏

- **开场音效（BGM intro）**：3 秒内建立情绪基调，不要空白进场
- **对话节奏**：短句优先，不要大段独白
- **静音的力量**：在情绪顶点前一秒切静音，观众的呼吸会停一拍

## 五、情绪弧线设计检查清单

在每一集剧本完成时，问以下问题：

- [ ] 观众在第几秒第一次产生"心疼"的情绪？
- [ ] 虐点出现的位置在集时间的 60-70% 区间吗？
- [ ] 虐点是"处境被逼台面"还是"外力制造冲突"？
- [ ] 爽点之前有足够的情绪积累吗？
- [ ] 结尾 hook 是画面悬念还是廉价旁白"下集揭晓"？
- [ ] 声音/BGM 有没有在情绪顶点前一秒制造静音？
- [ ] 这一集播完后，观众最想知道的是什么？

## 六、竖屏（9:16）情绪设计特殊考量

### 空间限制带来的情绪放大

竖屏画幅限制了环境展示，**面部特写成为情绪载体**。

- 眼神变化比台词更重要
- 手的动作（握紧手机、放下筷子）可以替代大量台词
- 纵深调度（Z轴）可以制造压迫感或希望感

### 竖屏 shot design 原则

| Shot 类型 | 情绪功能 | 竖屏使用建议 |
|---------|---------|------------|
| 特写（Close-up） | 情绪放大 | 面部细节，眼泪、眼神闪躲 |
| 中景（Medium） | 关系张力 | 两人同框，对话场景 |
| 纵深（Depth） | 情绪压迫/希望 | 走廊纵深，门框纵深 |
| 俯拍（High Angle） | 弱势感 | 从上往下拍，坐着的人 |
| 仰拍（Low Angle） | 强势感 | 从下往上拍，站着她的人 |

## 七、案例情绪拆解

### 《The Night Shift》情绪弧线

- **Setup（0-15s）**：夜班医院场景，展示护士的疲惫日常
- **Confrontation（15-40s）**：危重病人入院，她的判断和上级冲突
- **Crisis（40-75s）**：她的决定导致病人情况恶化，秘密被逼台面
- **Hook（75-90s）**：家属出现——她自己的家人

### 《SHAME》母女逆袭情绪弧线

- **Setup（0-15s）**：母女二人在狭小出租屋，各自承受压力
- **Confrontation（15-40s）**：母亲发现女儿的秘密，女儿发现母亲的秘密
- **Crisis（40-75s）**：两人秘密同时被揭开，冲突到达临界点
- **Hook（75-90s）**：母亲拿出藏了三年的存款——原来一切早有准备
