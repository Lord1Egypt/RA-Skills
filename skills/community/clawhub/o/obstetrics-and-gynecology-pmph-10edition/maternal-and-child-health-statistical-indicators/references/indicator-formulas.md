# 妇幼保健核心统计指标详细公式

## 一、妇女常见病筛查指标
- **妇女常见病筛查率** = (某年某地区实查人数 / 应查人数) × 100%
- **妇女常见病患病率** = (患病总人数 / 实查人数) × 10万/10万
- **妇女病治愈率** = (治愈例数 / 患病总例数) × 100%
- **子宫颈癌人群筛查率** = [(细胞学检查人数×3 + HPV检测人数×5 + 联合检测人数×5) / 35～64岁妇女人数] × 100%
- **乳腺癌人群筛查率** = (35～64岁接受乳腺超声BI-RADS检查人数 × 3 / 该年龄段妇女人数) × 100%

## 二、孕产期保健指标
### 工作指标
- **早孕建册率** = (早孕建册产妇数 / 产妇总数) × 100%
- **产前检查率** = (产前检查人数 / 活产总数) × 100%
- **产后访视率** = (接受访视产妇数 / 活产总数) × 100%
- **住院分娩率** = (住院分娩活产数 / 活产总数) × 100%

### 质量指标
- **高危孕产妇比例** = (高危人数 / 孕产妇总数) × 100%
- **剖宫产率** = (剖宫产活产数 / 活产总数) × 100%
- **产后出血率** = (发生产后出血产妇数 / 产妇总数) × 100%
- **产褥感染率** = (感染产妇数 / 产妇总数) × 100%
- **会阴侧切率** = (侧切产妇数 / 阴道分娩产妇总数) × 100%

### 效果指标
- **围产儿死亡率** = [(孕28周以上死胎死产 + 生后7日内新生儿死亡) / (死胎死产 + 活产)] × 1000‰
- **孕产妇死亡率** = (年内孕产妇死亡数 / 年内活产总数) × 10万/10万
- **新生儿死亡率** = (生后28日内死亡数 / 活产数) × 1000‰
- **早期新生儿死亡率** = (生后7日内死亡数 / 活产数) × 1000‰

## 三、人口与避孕节育指标
- **人口出生率** = (年出生人数 / 年平均人口) × 1000‰
- **人口死亡率** = (年死亡人数 / 年平均人口) × 1000‰
- **人口自然增长率** = (自然增长数 / 年平均人口) × 1000‰
- **出生人口性别比** = (男婴数 / 女婴数) × 100
- **出生人流比** = (人工流产例数 / 活产总数)
- **某项避孕节育服务百分比** = (该项服务例数 / 节育服务总例数) × 100%

## 关键变量说明
- **numerator**：分子数值（如筛查人数、死亡数等）
- **denominator**：分母数值（如应查人数、活产总数等）
- **age_group**：适用年龄组，如“35～64岁”
- **time_period**：统计时间段，如“年内”、“期内”
