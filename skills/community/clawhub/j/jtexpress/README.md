# J&T Express 极兔速递助手

帮你搞定极兔快递的查件、比价、估运费和寄件准备——覆盖极兔速递的日常使用场景。

## What it does
- 极兔快递单号追踪和状态查询
- 极兔 vs 中通 vs 韵达的价格和时效对比
- 全国城际运费估算（按重量和距离）
- 电商退货寄件方案建议

## Example scenarios

**1. 快递单号查不到**
> 👤 极兔快递单号 JT1234567890 怎么查不到？
> 🤖 先确认单号格式是否正确（极兔单号通常是 JT 开头 + 数字）。如果是刚寄出的，追踪信息可能要几小时才同步到系统，可以过 2 小时再查。目前最新状态是"包裹已揽收"，说明商家已经发货，正常流转中。

**2. 三大快递比谁便宜**
> 👤 极兔和中通、韵达比，哪个便宜？
> 🤖 先说结论：极兔在电商件价格上有优势，尤其拼多多订单。以 1kg 包裹为例——极兔约 5-8 元、中通约 6-10 元、韵达约 6-9 元，但具体看地区和是否有协议价。极兔覆盖广度不如中通（偏远地区可能慢），服务稳定性因地区而异。如果寄轻小件且不赶时间，极兔性价比最高。

**3. 寄大件比价**
> 👤 我想从上海寄一袋 5 公斤猫粮到长沙，用极兔大概多少钱？比中通便宜吗？
> 🤖 极兔从上海到长沙 5kg 预估 15-20 元（首重 1kg 约 8 元 + 4kg 续重约 3 元/kg），中通同类约 18-25 元。如果通过拼多多/淘宝的菜鸟裹裹下单可能更便宜。极兔在续重上有一定优势，如果是常发件可以和快递员谈协议价。提示：猫粮属食品类，确认包装密封完好再寄。
