# 🚀 部署指南

> 如何把"龙虾教研日报助手"分享给同事们使用。

---

## 📦 发布到 ClawHub（推荐方式）

### 步骤一：打包技能包

在 WorkBuddy 里运行：
```bash
openclaw skills package daily-report-assistant
```

这会在当前目录生成一个 `.zip` 文件。

### 步骤二：上传到 ClawHub

1. 访问 https://clawhub.ai
2. 登录后点击"发布技能"
3. 上传打包好的 `.zip` 文件
4. 填写技能信息：
   - **名称**：龙虾教研日报助手
   - **描述**：通过对话记录日报，月底一键生成汇总
   - **标签**：效率工具 / 办公 / 日报
   - **价格**：免费

### 步骤三：分享链接

发布成功后，把技能链接分享给同事：
```
https://clawhub.ai/skills/daily-report-assistant
```

同事安装后完成配置即可使用。

---

## 💾 手动安装方式（无需上传到市场）

如果同事不通过市场安装，可以手动复制：

### 1. 获取技能文件

把 `daily-report-assistant` 整个目录复制到对方的：
```
~/.openclaw/workspace/skills/
```

### 2. 告知对方配置

告诉对方在 WorkBuddy 里说：
```
我要配置教研日报助手
```

按引导完成授权和姓名确认即可。

---

## ⚙️ 管理员操作（可选）

如果你希望管理全团队的日报汇总：

### 1. 创建共享多维表格（可选）

团队可以共建一个"每日进展镜像表"：
- 每个人写入自己的记录
- 管理者可以看所有人的汇总

### 2. 共享表格权限设置

在腾讯文档里：
1. 打开共享多维表格
2. 点击"分享" → "添加协作者"
3. 搜索并添加团队成员
4. 设置权限为"可编辑"

### 3. 修改配置指向

管理员可修改 `references/data-schema.md` 中的表格 ID，指向团队共享的表格。

---

## 🔧 常见部署问题

**Q：同事找不到技能怎么办？**
A：确认文件放在正确目录 `~/.openclaw/workspace/skills/daily-report-assistant/`，重启 WorkBuddy。

**Q：授权需要公司账号吗？**
A：腾讯文档支持微信/QQ登录，使用公司绑定的账号扫码即可。

**Q：数据存在哪里？**
A：存在腾讯文档在线表格里，本地 WorkBuddy 只存配置和缓存。
