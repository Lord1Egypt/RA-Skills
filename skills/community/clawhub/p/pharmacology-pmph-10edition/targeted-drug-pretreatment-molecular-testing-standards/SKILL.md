# targeted-drug-pretreatment-molecular-testing-standards

## 适用对象
- 小分子靶向药物
- 大分子单克隆抗体类药物

## 使用前提
在执行本流程前，请确认以下条件已满足：
- 拟用药物具有明确的作用靶点（如EGFR、HER2、ALK、BRCA、NTRK等）
- 患者可提供肿瘤组织或血液样本用于生物标志物检测

## 执行步骤
1. **确认药物类别**：查阅药品说明书或权威临床指南（如NCCN、CSCO），判断该靶向药物是否属于“需检测分子靶点”类别。
2. **安排检测**：若属于，则必须在治疗前对患者样本进行相应生物标志物检测（如EGFR突变、HER2表达、ALK融合、BRCA突变、NTRK融合等）。
3. **验证检测方法**：确保检测采用国家或行业认可的方法（如NGS、IHC、FISH、PCR等），并由具备资质的实验室出具正式报告。
4. **评估结果**：仅当检测结果确认存在药物对应的有效靶点（如EGFR敏感突变、HER2阳性、T790M突变、NTRK融合等）时，方可启动该靶向药物治疗。
5. **禁止用药情形**：若未进行检测，或检测结果为阴性，则禁止使用该药物，以避免无效治疗及潜在毒性。

## 输出决策
- 若靶点阳性 → 允许使用该靶向药物
- 若靶点阴性或未检测 → 禁止使用该靶向药物

> 注意：本规范不适用于无明确靶点、或已获批泛实体瘤适应症且无需生物标志物检测的药物（如某些免疫检查点抑制剂）。
