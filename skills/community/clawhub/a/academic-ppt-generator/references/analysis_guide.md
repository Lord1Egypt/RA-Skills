# 文献分析指南

本文档指导如何从学术文献中提取关键信息用于PPT生成。

## 分析框架

### 1. 基本信息提取

```json
{
  "title": "论文标题",
  "authors": "作者列表",
  "venue": "期刊或会议名称",
  "year": "发表年份"
}
```

### 2. 科学问题与背景 (Part 1)

#### 研究问题识别
- 论文试图回答什么核心问题？
- 这个问题为什么重要？
- 问题属于哪个研究领域？

#### 背景信息提取
- 该领域的基本概念是什么？
- 相关的理论基础有哪些？
- 当前领域的发展状况？

#### 前人工作梳理
- 已有研究解决了什么？
- 使用了什么方法？
- 得出了什么结论？

#### 研究空白识别
- 前人工作有什么不足？
- 哪些问题还没有被回答？
- 现有理论的缺陷是什么？

#### 理论假设
- 本研究提出了什么假设？
- 假设基于什么理论？
- 假设如何可以被验证？

### 3. 研究方法 (Part 2)

#### 实验设计
- 采用什么研究设计？（实验/准实验/相关/纵向等）
- 自变量是什么？如何操纵？
- 因变量是什么？如何测量？
- 是否有控制组？

#### 被试/样本
- 样本量是多少？
- 被试的人口学特征？
- 如何招募和筛选？

#### 实验流程
- 实验的具体步骤？
- 每个阶段做什么？
- 时间线是怎样的？

#### 控制变量
- 控制了哪些混淆变量？
- 如何控制的？
- 为什么控制这些变量？

#### 方法-理论对应
- 实验设计如何验证假设？
- 每个操作的理论意义？
- 测量指标如何反映构念？

### 4. 结论与验证 (Part 3)

#### 主要发现
- 发现了什么？
- 统计结果是什么？
- 效应量有多大？

#### 证据支持
- 什么数据支持这个发现？
- 图表显示了什么？
- 统计检验结果？

#### 假设验证
- 每个假设是否被支持？
- 支持/不支持的证据？
- 部分支持的情况？

#### 结论逻辑
- 从数据到结论的逻辑链？
- 排除了哪些替代解释？
- 结论的可靠性如何？

### 5. 研究局限 (Part 4)

#### 方法局限
- 样本是否有代表性？
- 测量是否有局限？
- 实验设计是否有缺陷？

#### 结果局限
- 结果能否推广？
- 效应是否稳定？
- 是否有未控制的变量？

#### 未来方向
- 如何改进研究？
- 下一步研究建议？
- 可以扩展的方向？

## 提取技巧

### 阅读顺序
1. 先读摘要，了解整体框架
2. 读引言，理解问题和背景
3. 读方法，了解实验设计
4. 读结果，提取主要发现
5. 读讨论，理解结论和局限

### 关键句识别
- 问题陈述："本文旨在..." "本研究探讨..."
- 假设陈述："我们假设..." "基于...理论，我们预测..."
- 方法描述："采用...设计" "被试为..."
- 结果陈述："结果显示..." "发现..."
- 结论陈述："这些结果表明..." "本研究证实..."

### 信息组织原则
- 简洁优先：每页PPT不超过6个要点
- 逻辑清晰：按时间或逻辑顺序组织
- 重点突出：强调核心发现和贡献
- 图文配合：建议添加关键图表

## 输出格式

分析结果应为JSON格式，包含上述所有字段。详见示例文件。
