# 舌下络脉诊法（补充知识）

> 来源：《舌下络脉诊法图谱》（袁红霞、袁和平，2013）

## 一、舌下络脉诊法源流

- **先秦《黄帝内经》**：最早记载，称舌下络脉为"廉泉、舌下两脉"，已用于诊治疾病
- **晋代葛洪《肘后备急方》**：最早记载观察舌下两脉可预测"黄病"病情深浅
- **隋代巢元方《诸病源候论》**：首次提出运用舌下络脉诊法结合全身症状诊断疾病
- **唐代孙思邈《千金要方》**：设立专章，细致观察舌下络脉形色变化
- **宋代陈自明**：提出运用舌下络脉诊法诊断妇女难产及判断预后
- **清代周学海**：发展舌下络脉细络瘀血理论，指出舌下络脉粗张、舌质偏紫而晦暗多由瘀血阻络所致
- **1964年张赞臣**：详细论述舌下络脉与心肝两经的密切关联
- **1985年《中医年鉴》**：正式命名"舌下络脉诊法"，编入《中医诊断学》教材

## 二、舌下络脉诊法定义

通过诊察舌下络脉来判断人体脏腑内在的病理变化及气血阴阳盛衰的方法。主要从舌下络脉的颜色、形状、充重度等方面进行诊察，与传统舌诊相辅相成。

### 舌下络脉的解剖学基础
- 舌下络脉实际是见于黏膜下的舌下神经伴行静脉、舌神经伴行静脉及其属支
- 舌静脉主要由一个深静脉和四个浅静脉构成

## 三、诊察方法

### 1. 操作步骤
- **体位**：坐位或仰卧位，面向自然光线，头略扬起
- **暴露**：自然将舌体伸出并向上方翘起约45°，舌尖轻抵上颚，勿用力太过，舌体保持自然松弛
- **观察顺序**：先检查位于舌系带两侧纵行的大络脉（左右各一支或两支或多支）→ 观察颜色、形态长短粗细、有无怒张弯曲和紧束细小 → 再观察周围细小络脉

### 2. 舌下络脉与舌面分区对应
- 前上部 → 心肺
- 中央 → 脾胃
- 两侧 → 肝胆
- 后下部 → 肾

### 3. 注意事项
- **光线**：面向光亮处，使光线直射口内
- **饮食**：进食热或刺激性食物后、饮酒后舌下络脉会变化，不宜在进食或漱口后立即诊察
- **染色**：药物或食物可使舌体染色（牛奶、豆浆、橘子、核黄素、酸梅、黑褐色食品等），需询问饮食及服药情况鉴别
- **年龄与体质**：
  - 小儿：舌体鲜活娇嫩，络脉颜色多淡红
  - 老年人：气血偏虚，络脉颜色多偏暗紫
  - 青年壮实：多痰多瘀，络脉颜色多偏青紫

## 四、舌下络脉颜色辨证

### 淡红色
- 外感病初起，病情轻浅；或部分内伤杂病
- 常见证：虚劳证、久泻久痢、消渴、胃脘痛、闭经等（多属气血双亏）

### 红色（赤色）
- 脏腑俱热；色紫红瘀红者脏腑热极
- 兼舌尖生芒刺 → 心火亢盛
- 兼舌中生芒刺 → 胃火

### 淡紫色
- 正常人主色

### 紫色
- 主气滞血瘀
- 舌下络脉紫而粗长 → 气滞血瘀
- 舌下络脉紫而短细 → 血虚夹杂血瘀

### 青紫色
- 主气滞血瘀重症
- 多为寒凝、气滞、水饮所致，常见于真心痛、肝硬化等

### 蓝色
- 主气滞血瘀极重或痰湿内停
- 蓝而粗大 → 血络瘀滞
- 蓝而不粗 → 痰湿内停

### 黑色
- 主气滞血瘀危重证
- 舌下络脉黑而粗大 → 血瘀极重
- 舌下络脉黑而短细 → 气血极亏

**总体原则**：瘀则色深，虚则色淡

## 五、舌下络脉形态辨证

### 1. 长短络脉
- **正常**：不超过舌尖至舌下肉阜连线的3/5
- **长络脉**（超过3/5）：
  - 红而长 → 热证
  - 青紫粗长 → 气滞血瘀，多见于心脑血管病、肝硬化、肿瘤
- **短络脉**（不及2/5）：
  - 淡红短细 → 气血亏虚
  - 青紫短细 → 寒凝或虚中夹瘀

### 2. 粗细络脉
- **正常**：管径约2.7mm
- **粗络脉**（>2.7mm）：
  - 红粗 → 热盛
  - 青紫粗 → 血瘀、气滞
  - 与肿瘤、肝硬化、冠心病等密切相关
- **细络脉**（<2mm）：
  - 淡红细短 → 气血亏虚
  - 紫暗细 → 阳虚寒凝

### 3. 其他形态
- **怒张形络脉**：络脉怒张、迂曲、凸起如串珠状或葡萄串状
  - 主血瘀证，常见于瘀血头痛、肝硬化、真心痛
- **紧束形络脉**：络脉紧束、不舒展
  - 主寒凝气滞、血虚
- **双侧不对等络脉**：两侧粗细、颜色、长短不一致
  - 主气血运行不畅、气机失调

### 4. 舌下瘀斑、瘀点、瘀血颗粒
- 舌下黏膜出现瘀斑、瘀点或瘀血颗粒
- 主血瘀证
- 出现部位越多、颜色越深，血瘀程度越重

## 六、舌下络脉与体质的关系

| 体质类型 | 舌下络脉特征 |
|---------|------------|
| 平和质 | 淡紫色，粗细均匀，无怒张 |
| 气虚质 | 细小不均，或短缩不充盈 |
| 阳虚质 | 苍白无华 |
| 阴虚质 | 细短且舌质偏红 |
| 气郁质 | 粗长怒张 |
| 血瘀质 | 迂曲饱满怒张，可有瘀点 |
| 痰湿质 | 隆起且舌腹面淡白肿胀 |
| 湿热质 | 紫红色，可伴怒张或小囊泡 |

## 七、舌下络脉的临床辨证应用

### 辨证体系

**八纲辨证**：
- 表证 → 舌下络脉淡红
- 里热证 → 舌下络脉红紫粗长
- 里寒证 → 舌下络脉青紫短细
- 虚证 → 舌下络脉淡红细短
- 实证 → 舌下络脉青紫粗长怒张

**气血辨证**：
- 舌下络脉侧重血分（舌上舌质侧重气分）
- 舌下络脉是分析气血运行情况的重要依据
- 气血不足 → 舌下络脉短而细，舌色偏淡
- 血瘀 → 舌下络脉粗胀、青紫、有瘀点

### 常见疾病的舌下络脉表现
- **真心痛（冠心病）**：舌下络脉青紫粗长怒张
- **肝硬化**：舌下络脉青紫粗大，甚则结节状
- **肿瘤**：舌下络脉多粗长迂曲、囊泡状改变
- **糖尿病**：舌下络脉多细短
- **痛经**：舌下络脉多粗胀青紫

## 八、舌下络脉诊法与传统舌诊的关系

- 舌上舌质侧重气分，舌下络脉侧重血分
- 舌下舌质候五脏为主
- 舌苔候六腑为重
- 大多情况下舌下络脉与舌质（色）变化一致
- 不一致时：多为本虚标实、寒热错杂（如舌质虚嫩而舌下络脉反青紫粗长）

## 九、现代研究

舌下络脉异常与以下因素相关：
- 微循环障碍
- 静脉压升高
- 血液凝固性增高或纤溶性降低
- 血液流变性异常
- 血小板聚集性增高或释放功能亢进
- 红细胞比容和血浆黏度增加
- 红细胞变形性减低
- 纤维蛋白上升
- 维生素C 缺乏
