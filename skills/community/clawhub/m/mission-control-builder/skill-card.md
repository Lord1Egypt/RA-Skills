## Description: <br>
Build a personal OpenClaw dashboard with task management, memory browsing, calendar views, team tracking, and GitHub trends. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Shukiclaw](https://clawhub.ai/user/Shukiclaw) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and OpenClaw users use this skill to create a local Next.js mission control dashboard for tracking tasks, browsing memories, viewing calendar events, monitoring team status, and adding workflow widgets. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The generated dashboard can expose local memory previews through an unauthenticated web app. <br>
Mitigation: Keep the app local or private, add authentication and access controls before exposing it to the public internet, and review memory sync paths so private notes, secrets, or overly broad files are not served. <br>
