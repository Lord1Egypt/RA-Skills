# ACEI与ARB类降压药的选择与应用

## 触发条件
- **优先选用ACEI**：高血压患者合并糖尿病、左心室肥厚、心力衰竭或急性心肌梗死。
- **转换为ARB**：患者因ACEI相关无痰干咳无法耐受，需更换药物。

## 执行步骤

### 1. 评估适用性与禁忌证
- **确认适用对象**：
  - ACEI适用于卡托普利、依那普利、赖诺普利等。
  - ARB适用于氯沙坦、缬沙坦、厄贝沙坦、阿利沙坦酯等。
- **排除禁忌证**：
  - 双侧肾动脉狭窄、妊娠、高钾血症患者禁用两类药物。
  - 出现血管性水肿需立即停药。

### 2. 启动ACEI治疗（如适用）
- **机制理解**：
  - 抑制ACE活性，减少AngⅡ生成，同时升高缓激肽水平，导致血管舒张。
- **预期疗效**：
  - 单药可使60%～70%患者血压达标；联合利尿药后有效率达95%。
- **监测重点**：
  - 定期评估干咳、血钾水平及肾功能。
  - 注意长期用药无耐受性，停药无反跳。

### 3. 转换为ARB治疗（如因干咳不耐受）
- **机制优势**：
  - 直接阻断AngⅡ与AT₁受体结合，不影响缓激肽代谢，几乎不引起干咳。
- **药物选择建议**：
  - 优先考虑阿利沙坦酯（单步酯酶水解，副产物少，不良反应更少）。
  - 氯沙坦需经CYP450多步代谢生成强效活性物EXP3174。
- **用药注意**：
  - 起效较慢，通常需3～6周达最大降压效果。
  - 单药效果不佳时可联用氢氯噻嗪等利尿药。
  - **禁止**与ACEI或肾素抑制剂联用。

### 4. 器官保护作用确认
- **ACEI与ARB共有的获益**：
  - 逆转左心室肥厚、延缓糖尿病肾病进展、改善心室重构。
- **ARB潜在优势**：
  - 不阻断AT₂受体，可能增强其介导的器官保护效应。

## 输出结果
- 若符合ACEI首选指征，启动治疗并制定监测计划。
- 若因干咳需换药，推荐ARB类，并优先选择阿利沙坦酯。
