# 经济补偿金N计算公式

## 适用情形

用人单位依法解除或终止劳动合同时，应当向劳动者支付经济补偿金。常见情形包括：

| 情形 | 是否支付N |
|---|---|
| 协商解除（单位提出） | ✅ |
| 劳动者过失性解除（第39条） | ❌ |
| 劳动者非过失性解除（第40条） | ✅ |
| 经济性裁员（第41条） | ✅ |
| 合同期满（单位不续） | ✅ |
| 合同期满（劳动者不续） | ❌ |
| 单位破产/解散/吊销 | ✅ |
| 事实劳动关系满一年未签合同 | ✅ |
| 违法解除（选择2N时） | N/A（选2N） |
| 非全日制用工终止 | ❌ |

---

## 计算公式

```
N = 在本单位工作年限 × 月工资
```

---

## 年限计算规则

| 工作年限 | 折算月数 |
|---|---|
| 满1年 | 1个月 |
| 6个月以上不满1年 | 按1年计算 → 1个月 |
| 不满6个月 | 0.5个月 |

> **注意**：工作年限按在本单位连续工作的时间计算，不区分是否在试用期。

---

## 工资计算规则

**月工资** = 解除劳动合同前12个月的平均工资

### 三倍上限规则（高工资劳动者）

| 条件 | 处理方式 |
|---|---|
| 月工资 > 本地区上年度职工月平均工资 **3倍** | 按3倍计算 |
| 补偿年限 | 最高不超过 **12年** |

> **通俗解释**：工资再高，最多赔12个月，且每月基数不能超过当地社平工资的3倍。

### 保底规则

月工资不得低于当地最低工资标准。

---

## 赔偿情形一览表

| 解除类型 | 计算规则 | 上限 |
|---|---|---|
| 协商解除（单位提出） | N | 无 |
| 医疗期满不能从事原工作 | N | 无 |
| 不能胜任工作经培训/调岗仍不胜任 | N | 无 |
| 客观情况重大变化协商不成 | N | 无 |
| 经济性裁员 | N | 无 |
| 合同期满不续（单位不续） | N | 无 |
| 违法解除（选择2N） | 2×N | 无 |

## 实例计算

### 案例一：普通员工

- 工作年限：3年6个月
- 解除前12个月平均工资：8,000元
- 本地社平工资3倍：15,000元

**计算：**
- 年限：3年 + 6个月（≥6个月，按1年算）= **4个月**
- N = 4 × 8,000 = **32,000元**

---

### 案例二：高工资员工（受三倍上限约束）

- 工作年限：8年
- 月工资：30,000元
- 本地社平工资3倍：12,000元

**计算：**
- 工资30,000 > 12,000 → 按12,000算
- 年限：8年，但受12年上限约束 → **8个月**
- N = 8 × 12,000 = **96,000元**

---

### 案例三：短工龄员工

- 工作年限：5个月
- 月工资：6,000元

**计算：**
- 年限：不满6个月 → **0.5个月**
- N = 0.5 × 6,000 = **3,000元**

## 执行步骤

### 第一步：收集证据

- 劳动合同（确认工作年限起始日）
- 工资发放记录（解除前12个月银行流水/工资条）
- 社保缴纳记录（确认工作年限）
- 解除劳动合同通知书（确认解除原因）

### 第二步：计算N

1. 确认工作年限起止日期，计算完整年数和月数
2. 确认解除前12个月平均工资
3. 检查是否超过三倍社平工资上限
4. 套用公式：年限 × 月工资

### 第三步：核对是否正确

1. 对照适用情形表，确认是否属于应当支付N的情形
2. 检查年限计算是否正确（6个月以上算1年，不满6个月算0.5年）
3. 检查工资基数是否正确（是否高于3倍上限）
4. 检查支付时间：应在**办结工作交接时**支付（劳动合同法第50条）
5. 如有争议，可向劳动行政部门投诉或申请劳动仲裁
