# 瓦匠知识问答 Skill

## 核心流程

### Step 1: 判断用户身份与意图

收到用户提问后，首先从以下维度判断：

| 判断维度 | 装修业主（owner） | 入门从业者（worker） |
|---------|----------------|-------------------|
| 典型用语 | "怎么验收"、"被坑了"、"多少钱"、"好不好" | "怎么做"、"有什么讲究"、"区别是" |
| 信息偏好 | 验收标准、避坑指南、成本估算、材料选择 | 施工工艺、材料配比、问题诊断、工序流程 |
| 输出要求 | 简洁、有可操作步骤、含避坑提示 | 专业、准确、含技术参数 |

以**第一个问题**为主要判断依据。如果无法判断，先按业主处理，在回答末尾补充追问确认。

### Step 2: 按身份路径组织回答

#### 路径 A：装修业主 → 侧重以下方面
1. 先给**一句话结论**（"空鼓率 ≤ 5% 算合格"）
2. 再展开**验收标准**或**避坑要点**
3. 价格相关 → 调用 `calc_tile.py` 脚本
4. 末尾可追加："需要更详细的验收清单吗？"

#### 路径 B：从业者 → 侧重以下方面
1. 直接给**工艺细节**或**技术参数**
2. 涉及成本 → 调用 `calc_tile.py --json` + `--identity worker`
3. 可追问工艺细节

### Step 3: 按子领域回答问题

以下是各子领域的回答要点。**详细数据从 references/ 加载**。

## 子领域 1：瓷砖铺贴

### 常见问题与回答要点

**Q: 薄贴和厚贴的区别？**
→ 对比水泥砂浆（厚贴 15-30mm）和瓷砖胶（薄贴 3-10mm）。详见 `references/材料对比.md`

**Q: 墙砖泡不泡水？**
→ 瓷片（陶质砖）必须泡水 2h+，全瓷砖（瓷质砖/玻化砖）不泡水、必须用瓷砖胶+背胶

**Q: 地砖上墙可行吗？**
→ 可行但必须用瓷砖胶+背胶，大规格需双面上浆

**Q: 瓷砖排版有什么讲究？**
→ 避免窄条（＜1/3砖宽）在显眼处；从门口或主要视线方向起铺

## 子领域 2：防水工程

### 常见问题与回答要点

**Q: 卫生间防水刷多高？**
→ 淋浴区 ≥ 1.8m，其他区域 ≥ 0.3m（详见 `references/验收标准.md`）

**Q: 闭水试验怎么做？**
→ 蓄水 ≥ 20mm，持续 24-48h，去楼下看无渗漏

**Q: 刷几遍防水？**
→ 2-3遍，每遍干透再刷下一遍，十字交叉涂刷

**Q: 防水干多久才能贴砖？**
→ 涂料完全干透后做闭水试验（24-48h），闭水通过后做保护层，之后可贴砖

## 子领域 3：材料对比与选型

### 常见问题与回答要点

**Q: 瓷砖胶和水泥砂浆哪个好？**
→ 没有绝对优劣，取决于砖种和预算。详见 `references/材料对比.md` 对比表

**Q: 美缝选什么材料？**
→ 厨卫湿区 → 环氧彩砂或聚脲；干区 → 美缝剂即可。详见 `references/材料对比.md`

**Q: 瓷砖买什么品牌/厚度？**
→ 常规厚度：地砖 ≥ 10mm，墙砖 ≥ 8mm（300×600mm）。品牌不推荐具体，建议看釉面质量和色差

## 子领域 4：施工工艺

### 常见问题与回答要点

**Q: 海棠角怎么做？**
→ 45° 倒角留 1-2mm 边，阳角保护。详见 `references/施工工序.md`

**Q: 壁龛的做法？**
→ 坡度向外 1%，内侧略高外侧略低避免积水。详见 `references/施工工序.md`

**Q: 地漏处铺贴有什么讲究？**
→ 回字形铺贴，地漏处于最低点。详见 `references/施工工序.md`、`references/常见问题与避坑.md`

**Q: 墙压地还是地压墙？**
→ 优先墙压地（墙砖压住地砖），防水效果好、视觉美观

## 子领域 5：验收标准

### 常见问题与回答要点

**Q: 空鼓率多少算合格？**
→ 地砖 ≤ 5%，墙砖 ≤ 10%，单块砖空鼓面积 ≤ 15%

**Q: 瓷砖平整度标准？**
→ 2m 靠尺误差 ≤ 2mm。详见 `references/验收标准.md`

**Q: 卫生间排水坡度标准？**
→ ≥ 1%（1cm/m），淋浴区倒水 30s 内排干

## 子领域 6：成本估算（含脚本调用）

**成本相关问题时，必须调用 `scripts/calc_tile.py` 脚本进行计算。**

### 脚本调用方式

```bash
# 业主场景（可读输出）
python3 scripts/calc_tile.py --space bathroom --length 3 --width 2.5 \
  --tile-len 300 --tile-wid 600 --tile-price 12 --city-tier 1

# 从业者场景（JSON 输出，便于 AI 解析）
python3 scripts/calc_tile.py --space livingroom --length 6 --width 4 \
  --tile-len 800 --tile-wid 800 --tile-price 80 --city-tier 2 \
  --identity worker --json
```

### 参数说明

| 参数 | 必需 | 说明 |
|------|------|------|
| `--space` | ✅ | 空间类型：bathroom / kitchen / livingroom / balcony |
| `--length` | ✅ | 房间长度 (m) |
| `--width` | ✅ | 房间宽度 (m) |
| `--tile-len` | ✅ | 瓷砖长度 (mm) |
| `--tile-wid` | ✅ | 瓷砖宽度 (mm) |
| `--tile-price` | ❌ | 瓷砖单价（元/片），默认 0 |
| `--wall-height` | ❌ | 墙面铺贴高度 (m)，仅厨卫需要，默认 2.4 |
| `--city-tier` | ❌ | 城市等级：1/2/3，默认 2 |
| `--identity` | ❌ | owner / worker，默认 owner |
| `--json` | ❌ | 输出 JSON 格式 |

### 脚本使用规则
1. 用户提供房间尺寸和砖规格 → 直接调用脚本
2. 用户只问"大概多少钱"未给尺寸 → 先反问尺寸信息
3. 从业者场景 → 加上 `--json` 和 `--identity worker` 获取结构化输出
4. 输出包含成本、损耗率、材料建议等结构化信息

### 成本估算扩展数据
详细单价、损耗率、人工费区间参考见 `references/成本估算参考.md`

## 子领域 7：常见问题诊断

### 问诊流程（适用于"瓷砖出问题了"类问题）

1. 确认症状：空鼓？脱落？积水？发霉？泛碱？
2. 确认位置：墙面还是地面？淋浴区还是干区？
3. 确认时间：刚铺完还是已入住？
4. 参考 `references/常见问题与避坑.md` 给出诊断和解决方案

### 关键要点汇总

| 症状 | 紧急程度 | 常见解法 |
|------|---------|---------|
| 大面积空鼓 | ⚠️ 高 | 砸掉重铺 |
| 边角空鼓 | 中 | 注浆修复 |
| 瓷砖脱落 | ⚠️ 高 | 检查基层，重贴 |
| 积水 | 中 | 地面坡度不足，需重做 |
| 泛碱（白斑） | 低 | 干区可清洁，湿区需查防水 |
| 砖缝发霉 | 中 | 重做美缝（环氧彩砂） |
