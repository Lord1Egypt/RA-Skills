# 雷达图生成说明

## 概述

雷达图用于可视化展示商业模式画布九维度评分，直观呈现商业模式的优势和短板。

## 数据准备

### 评分数据结构
```json
{
  "dimensions": [
    "客户细分",
    "价值主张",
    "渠道通路",
    "客户关系",
    "收入来源",
    "核心资源",
    "关键业务",
    "重要合作",
    "成本结构"
  ],
  "scores": [7, 8, 6, 5, 7, 6, 7, 5, 6],
  "average": 6.33,
  "rating": "B",
  "project_name": "项目名称",
  "analysis_date": "2024-01-01"
}
```

### 评分要求
- 数值范围：0-10分
- 精度：保留1位小数
- 完整：9个维度必须全部有值

## 综合评分计算

### 计算公式
```
综合评分 = (客户细分 + 价值主张 + 渠道通路 + 客户关系 + 收入来源 + 核心资源 + 关键业务 + 重要合作 + 成本结构) / 9
```

### 评级标准
| 评级 | 分数范围 | 含义 |
|------|----------|------|
| A | 8-10分 | 商业模式成熟，具备规模化条件 |
| B | 6-7分 | 商业模式基本成立，需补短板 |
| C | 4-5分 | 商业模式存在重大缺陷，需重新设计 |
| D | 1-3分 | 商业模式不成立，建议推翻重来 |

## 雷达图解读

### 图形特征
- **图形饱满**：表示各维度均衡发展
- **图形凹陷**：表示对应维度存在短板
- **图形凸出**：表示对应维度是优势
- **整体偏小**：表示整体商业模式较弱

### 解读要点
1. 找出得分最高的2-3个维度 → 核心竞争力
2. 找出得分最低的2-3个维度 → 急需改进
3. 分析图形对称性 → 判断均衡程度
4. 结合评级 → 确定整体水平
