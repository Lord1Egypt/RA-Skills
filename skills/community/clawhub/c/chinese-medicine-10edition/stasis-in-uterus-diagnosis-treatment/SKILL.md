# Stasis in Uterus Diagnosis and Treatment

## When to Use
- 妇科不孕症患者（有婚久不孕史）
- 月经异常女性（月经周期延后、经色紫黑、夹有血块）

## When to Use
当患者同时满足以下全部特征时启用本技能：
- 婚久不孕
- 月经周期延后
- 经血色紫黑、夹有血块
- 舌质紫暗、边有瘀点
- 脉象弦涩

## Execution Steps
1. **辨识核心证候**：确认患者存在婚久不孕，月经周期延后，经量可多可少，经色紫黑夹块；常伴痛经、少腹疼痛或肛门坠胀。
2. **验证舌脉特征**：观察舌质是否紫暗、边有瘀点；切脉是否弦涩。
3. **判断病机**：瘀血内阻胞宫，冲任不通，血行不畅，导致经血紫黑有块；瘀阻胞宫致精难聚，故不孕。
4. **确立治法**：活血化瘀，止痛调经。
5. **选择基础方药**：采用《医林改错》少腹逐瘀汤为基础方。
6. **根据兼证加减**：
   - 寒象明显（如小腹冷痛）者，加艾叶；
   - 兼有热象（如口干、经血黏稠）者，加丹皮。

## Precautions
- 本技能不适用于无血瘀表现的虚寒证或气血两虚型不孕。
- 治疗期间应监测月经变化及腹痛改善情况，适时调整方药。
