# 新高考选科指导参考

> 数据来源：教育部及各省级教育考试院公开信息
> 最后更新：2026年4月

---

## 一、新高考模式概览

### 3+3模式
**适用省份**：浙江、上海、北京、天津、山东、海南
- 语数外（各150分）+ 6选3（浙江7选3）
- 选考科目等级赋分制

### 3+1+2模式  
**适用省份**：江苏、河北、湖北、湖南、广东、福建、辽宁、重庆、河南（2025新高考）、四川（2025新高考）、陕西（2025新高考）、山西（2025新高考）、云南（2025新高考）、贵州（2025新高考）、广西（2025新高考）、甘肃（2024新高考）、吉林（2024新高考）、黑龙江（2024新高考）、安徽（2024新高考）、江西（2024新高考）
- 语数外（各150分）+ 物理/历史2选1（原始分）+ 4选2（等级赋分）
- 物理类和历史类分别划线录取

## 二、选科组合分析

### 3+1+2模式下的12种组合（推荐度从高到低）

| 排名 | 组合 | 可报专业覆盖率 | 适合人群 | 难度 |
|:---:|------|:------------:|---------|:---:|
| 1 | **物理+化学+生物** | >97% | 目标理工农医类专业，学习能力强 | 高 |
| 2 | **物理+化学+地理** | >95% | 目标理工类，地理思维好 | 高 |
| 3 | **物理+化学+政治** | >96% | 目标军校/公安/政治类专业 | 高 |
| 4 | **物理+生物+地理** | ~90% | 对化学不敏感，想学医或地学 | 中等 |
| 5 | **物理+生物+政治** | ~91% | 文理兼备，目标军警或教育类 | 中等 |
| 6 | **物理+地理+政治** | ~86% | 偏文理科思维，目标地理/规划类 | 中等 |
| 7 | **历史+政治+地理** | ~52% | 传统文科生，目标文史哲法经管 | 中等 |
| 8 | **历史+政治+生物** | ~50% | 文科思维，对生物感兴趣 | 中等 |
| 9 | **历史+生物+地理** | ~49% | 偏记忆型，不想选政治 | 中等 |
| 10 | **历史+化学+地理** | ~50% | 较少见组合，文科+化学思维 | 较高 |
| 11 | **历史+化学+政治** | ~51% | 目标考古/文保/法学 | 较高 |
| 12 | **历史+化学+生物** | ~50% | 目标医学护理类（理科不够又想学医） | 较高 |

### 关键选科建议

1. **物理+化学组合**：2024年后选科要求改革，大多数理工类专业要求同时选物理和化学，"物化绑定"是大趋势
2. **医学类专业**：绝大多数要求物理+化学（临床医学通常要求物理+化学+生物）
3. **公安军事类**：通常要求选政治
4. **地理相关**：地质、测绘、城乡规划等要求选地理或物理
5. **历史类考生的专业选择面**（约52%）比物理类（约90%+）明显窄

## 三、选科决策流程

```
1. 评估学生兴趣和优势学科
   ├── 喜欢动手实验 → 偏理倾向
   ├── 喜欢阅读写作 → 偏文倾向
   └── 文理均衡 → 以专业覆盖率优先

2. 明确目标专业方向
   ├── 有明确目标 → 查目标专业的选科要求
   └── 无明确目标 → 选覆盖率最高的组合（物化生）

3. 评估学科竞争力
   ├── 弱势学科要避开 → 等级赋分下弱势学科更吃亏
   └── 赋分科目选有信心的 → 原始分学科选成绩好的

4. 查询目标院校选科要求
   └── 登录各省教育考试院官网或阳光高考网查询
```

## 四、等级赋分制说明

### 等级赋分规则（通用版）

将选考科目考生按原始分排名，划分A-E等级，对应不同赋分：

| 等级 | 比例 | 赋分区间 |
|:---:|:---:|:-------:|
| A | 15% | 86-100 |
| B | 35% | 71-85 |
| C | 35% | 56-70 |
| D | 13% | 41-55 |
| E | 2% | 30-40 |

### 赋分计算逻辑
- 原始分排名决定等级
- 同一等级内，按原始分插值计算具体赋分
- 赋分不影响考生的相对排名（排名不变）

### 选科策略提示
- 物理和历史是原始分计入（学校教学质量和考生绝对水平重要）
- 其他4科是赋分计入（竞争激烈程度重要）
- **在弱校的优势学科可能比强校的普通学科更有利**（排名竞争更温和）
