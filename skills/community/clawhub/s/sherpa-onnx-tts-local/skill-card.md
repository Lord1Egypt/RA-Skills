## Description: <br>
Local text-to-speech via sherpa-onnx (offline, no cloud). <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[deichmann181](https://clawhub.ai/user/deichmann181) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill to synthesize speech locally with sherpa-onnx instead of relying on a cloud TTS service. It supports local runtime and model configuration on macOS, Linux, and Windows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill downloads and runs native sherpa-onnx runtime binaries and model archives from upstream GitHub releases. <br>
Mitigation: Install only when the upstream k2-fsa sherpa-onnx release source is trusted, and verify downloaded archives where possible before use. <br>
Risk: Runtime, model, and PATH configuration can affect which local binaries are executed. <br>
Mitigation: Keep runtime and model paths under directories you control, review PATH changes before applying them, and confirm the expected wrapper exists before adding the skill bin directory to PATH. <br>
