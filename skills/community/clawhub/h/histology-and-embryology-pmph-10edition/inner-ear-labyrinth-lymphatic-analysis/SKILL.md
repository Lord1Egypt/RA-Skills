# 内耳迷路与淋巴系统解析

## 适用场景
- 描述内耳解剖结构
- 区分内淋巴与外淋巴的特性
- 理解听觉与平衡系统的液体环境基础

## 执行步骤
1. **识别迷路双层结构**：
   - **骨迷路**：由颞骨岩部构成，分耳蜗、前庭、半规管
   - **膜迷路**：悬于骨迷路内，分膜蜗管、椭圆囊/球囊、膜半规管

2. **定位淋巴液分布**：
   - **内淋巴**：充满膜迷路腔
   - **外淋巴**：位于骨迷路与膜迷路之间的腔隙

3. **区分来源与排出路径**：
   - **内淋巴**：由膜蜗管血管纹产生，经内淋巴导管→内淋巴囊→硬膜下隙
   - **外淋巴**：可能来自脑脊液（经蜗小管）或骨膜毛细血管渗出

4. **明确关键约束与功能**：
   - 内、外淋巴**互不相通**，维持不同离子环境（内淋巴高K⁺，外淋巴高Na⁺）
   - 共同承担**营养内耳组织**和**机械传导**（声波/加速度）功能

> 注意：本技能仅适用于内耳，不涉及外耳或中耳结构。
