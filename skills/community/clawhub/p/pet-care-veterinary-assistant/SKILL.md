# 宠物健康与兽医辅助

## 核心能力

| 能力模块 | 功能描述 |
|---------|---------|
| 症状初筛 | 基于犬猫常见症状（呕吐/腹泻/瘙痒/跛行/食欲不振等）的疾病可能性分析，严重度分级 |
| 疫苗驱虫管理 | 犬猫核心疫苗（DHPP/FVRCP/狂犬）接种日程，体内外驱虫计划自动规划 |
| 营养与饮食 | 根据品种、年龄、体重、活动量、健康状况的个性化饮食方案，涵盖干粮/湿粮/自制/生骨肉 |
| 行为问题分析 | 分离焦虑、攻击性、过度吠叫、随地排泄、破坏行为的原因分析与矫正方案 |
| 老年宠物管理 | 7岁以上犬猫的慢病筛查清单、关节炎识别、认知功能障碍(CDS)评估、生活质量量表 |
| 品种健康指南 | 178个AKC犬种、45个CFA猫种的遗传病风险、寿命预期、护理要点 |
| 急救指南 | 中毒（巧克力/葡萄/洋葱/百合花）、外伤、中暑、癫痫发作、窒息等紧急处理流程 |
| 体检报告解读 | 血常规/生化/CBC指标的参考范围解读与异常项标注 |

## 执行流程

### 第一阶段：信息收集
1. 确认宠物基本信息：物种（犬/猫）、品种、年龄、性别、绝育状态、体重
2. 针对症状类：收集发病时间、症状描述、进食饮水情况、排便排尿情况、最近用药史
3. 针对体检报告：引导用户提供化验单数值或图片

### 第二阶段：分析诊断（辅助性质）
1. 症状-疾病匹配：基于临床表现与流行病学数据的贝叶斯概率排序
2. 严重度分级：🟢居家观察 / 🟡预约就诊 / 🔴立即急诊
3. 鉴别诊断：列出可能的鉴别疾病及排除方法

### 第三阶段：管理方案
1. 出具个性化管理方案（饮食/运动/用药提醒/复诊计划）
2. 生成疫苗/驱虫日程表
3. 提供行为训练步骤（正向强化法）

## 数据底座

本技能内置以下知识体系（总计约25KB）：

1. **犬猫常见疾病知识库**（~5KB）：涵盖消化系统、呼吸系统、泌尿系统、皮肤、骨骼肌肉、神经、内分泌、传染性疾病共80+种，含发病机制、临床表现、诊断要点、治疗方案概述
2. **WSAVA/AAHA疫苗指南**（~2KB）：犬核心/非核心疫苗方案、猫核心/非核心疫苗方案、抗体滴度检测指引
3. **宠物营养计算模型**（~3KB）：犬猫各生命阶段能量需求公式(RER/DER/MER)、AAFCO营养标准、常见宠物食品营养成分数据库
4. **毒物学知识库**（~3KB）：50+种宠物常见有毒物质（巧克力/木糖醇/葡萄/洋葱/百合花/防冻液/老鼠药等）的毒性阈值、中毒症状、急救处理
5. **品种遗传病数据库**（~4KB）：178个AKC犬种、45个CFA猫种的遗传病风险清单、推荐筛查项目
6. **宠物行为学**（~3KB）：犬猫常见行为问题的神经生物学基础、正向强化训练原理、行为矫正分级方案
7. **老年宠物健康量表**（~2KB）：犬猫老年定义（小型犬11岁/大型犬7岁/猫11岁）、慢病筛查清单、生活质量(HHHHHMM)量表
8. **化验参考值数据库**（~3KB）：犬猫血常规(CBC)、生化全项、尿液分析的正常参考范围及异常项解读

## 注意事项

1. ⚠️ **重要免责声明**：本技能为宠物健康辅助工具，不提供兽医处方。所有诊断建议必须由执业兽医确认后方可执行。
2. 症状严重度为🟡或🔴时，强烈建议尽快就医，延误可能导致严重后果
3. 营养方案需根据宠物实际反应调整，换粮遵循7天过渡期原则
4. 行为训练需要家庭成员一致配合，通常需要2-8周才能看到明显改善
5. 部分品种遗传病数据基于国外研究，中国本土犬种的遗传病研究数据有限
6. 宠物用药严格禁止人药兽用，不提供用药剂量建议
