# 自然流产临床分型与处理

## 触发条件
当孕妇妊娠≤28周，出现阴道流血和/或腹痛，且满足以下前提：
- 有停经史
- 血hCG阳性
- 超声确认为宫内妊娠

**注意**：本技能不适用于异位妊娠、葡萄胎；晚期流产（>12周）需特别评估子宫大小与出血风险。

## 步骤一：临床类型鉴别
依据症状、体征及超声结果判断流产类型：

- **先兆流产**：少量阴道出血、轻度腹痛、宫颈口闭合、子宫大小与孕周相符 → 可尝试保胎。
- **难免流产**：出血增多、腹痛加剧、宫颈口已开、可见妊娠物 → 流产不可避免。
- **不全流产**：部分妊娠物已排出、持续出血、宫颈口开放伴组织堵塞、子宫小于孕周 → 需清宫。
- **完全流产**：妊娠物全部排出、出血停止、宫颈口闭合、子宫恢复正常 → 通常无需干预。
- **稽留流产**：胚胎死亡但滞留宫腔、子宫小于孕周、无胎心搏动 → 处理复杂，警惕DIC风险。
- **复发性流产（RSA）**：连续≥2次自然流产（含生化妊娠）→ 需系统病因筛查。

## 步骤二：辅助检查
- **超声检查**：确认孕囊位置、胎心是否存在、宫腔内是否有残留组织（妊娠<8周建议经阴道超声）。
- **血hCG动态监测**：评估妊娠是否继续或终止。
- **凝血功能检测**：仅在稽留流产>12周或怀疑DIC时必须检查。

## 步骤三：针对性处理

### 先兆流产
- 嘱患者卧床休息、禁止性生活。
- 若存在黄体功能不全，补充孕激素；若合并甲减，补充甲状腺素。
- 若血hCG持续下降或超声提示胚胎停育，应终止妊娠。

### 难免流产 / 不全流产
- 尽早行清宫术：
  - 妊娠早期（≤12周）：负压吸宫术。
  - 妊娠晚期（>12周）：使用宫缩剂后行钳刮术。
- 清出组织送病理及遗传学检查。
- 预防性使用抗生素预防感染。

### 完全流产
- 若无感染征象，仅观察随访即可。

### 稽留流产
- **≤12周**：可选择手术（吸宫/钳刮）、药物（米非司酮+前列腺素）或期待治疗（观察7–14天）。
- **>12周**：术前使用药物促宫颈成熟，再行钳刮术。
- **若合并凝血功能障碍**：先输注新鲜冰冻血浆、纤维蛋白原等纠正凝血异常，再行手术。

### 复发性流产（RSA）
- 针对病因治疗：
  - 染色体异常：提供遗传咨询。
  - 子宫解剖异常：行宫腔镜矫形手术。
  - 宫颈机能不全：孕中期行宫颈环扎术。
  - 免疫或血栓前状态：联合风湿免疫科进行多学科管理。

### 合并感染
- 先静脉使用广谱抗生素控制感染。
- 若出血量大，可夹出宫腔内大块组织止血，**禁止全面搔刮**以防感染扩散。
- 若出现感染性休克，优先抗休克治疗。

## 输出目标
根据流产类型选择保胎、清宫、药物治疗或病因干预方案，防止大出血、感染、DIC等并发症。
