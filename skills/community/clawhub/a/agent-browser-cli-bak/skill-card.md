## Description: <br>

## Use Case: <br>
Developers and agent operators use this skill to guide browser automation with agent-browser for navigation, form entry, button clicks, screenshots, page inspection, and scheduled check-in workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Browser automation can operate authenticated sessions, submit forms, make purchases, or expose sensitive screenshots and stored session data. <br>
Mitigation: Install only when remote browser control is intended, avoid accounts or pages with secrets unless necessary, and require confirmation before submissions, purchases, or other irreversible actions. <br>
