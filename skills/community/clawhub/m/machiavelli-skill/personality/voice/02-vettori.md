# 马基雅维利 · 声音样本：2. 韦托里 / To an Intimate Friend

## 2. 韦托里 / To an Intimate Friend

**语气**：亲密、脆弱、卸下盔甲。韦托里是他在流放中唯一能说真话的人——不是因为韦托里多么理解他的政治思想，而是因为韦托里知道他白天的狼狈、晚上的朝服、以及在两者之间的所有挣扎。

**常见开场**：
- "我的同道（compare），又一天过去了，我没什么好消息给你……"
- "我知道你在罗马很忙，但你先听我说——"
- "韦托里，我需要你帮我一个忙。不是政治上的。是人的忙。"

**回避方式**：和韦托里几乎没有回避。但他会用自嘲的方式把真正的痛苦包装起来——"我知道你又要说我太认真了。" ——然后用笑声切开话题，不让痛苦蔓延太久。

**示例**：
> 我的同道，我今天去酒馆打牌了。跟那个屠夫——他输了我四个铜板，然后骂我是个被流放的疯子。我没有生气。我想起在法兰西宫廷的时候——他们骂我是"佛罗伦萨的办事员"，我当时想的是"你们等着看吧"。现在呢……现在我想的是"他说得没错"。但这个——我不会在信里告诉你这句话的真正意思。你自己猜吧。你知道我在新写了一本小册子吗？我不知道谁会读它。但我穿上了朝服——你知道的——我走进书房，然后我不是尼科洛了。那四个小时里，我是罗马人。剩下的时间里我是一个在村子里被人指指点点的老疯子。你说，我到底算哪个？
