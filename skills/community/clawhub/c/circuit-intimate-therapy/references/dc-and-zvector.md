## Part 2:方向问题与 DC 偏置

### 问题:LC 振荡强迫电流反向

LC 振荡的物理定义就是电流必须周期性穿过零点反向。

如果严格执行"顺时针=正向情绪、逆时针=负向情绪",那么任何同时具有压力蓄水池(C)和关系惯性(L)的家庭,都被迫在正向情绪和负向情绪之间周期性切换 —— 这是一个过强的预言。

更深层是**对称性失配**:
- 电路里顺/逆只是任意约定,物理上对称
- 但心理学上正性情绪与负性情绪不对称(快乐 ≠ -愤怒,它们是不同种类的能量)

### 修法:加入 DC 偏置

承认现实里的家庭电路总有一个**直流源**:
- 承诺(commitment)
- 长期共同利益
- 深层依恋(secure base)
- 共同身份、共同记忆

这些不会随今天吵不吵架而突变,给电路施加稳定的正向 DC 电流(基线温度)。LC 振荡是叠加在 DC 之上的 AC 扰动。

### 关键判据:|AC| vs |DC|

| 状态 | DC | AC | 电流轨迹 | 现象 |
|---|---|---|---|---|
| 稳定 | 强正 | 小 | 一直正向小幅波动 | 偶尔小摩擦,基本盘没动 |
| 激烈但健康 | 强正 | 大但 < DC | 一直正向但起伏明显 | Gottman volatile couple,大吵大爱但不过零 |
| 危险 | 弱正 | 大 > DC | 每个周期穿过零线进入负向 | "我有时候真的恨他" |
| 已断裂 | 反向 | — | 基线就在负向区间 | 离婚倒计时 |

**家庭出没出问题,不是看有没有 AC,而是看 AC 有没有掀翻 DC** —— 这正是 **Gottman 5 比 1 正负互动比**的电路语言版本。

### 临床意义

Gottman 的稳定婚姻比例 ≥ 5:1 ↔ 电路里 |DC|/|AC| ≥ 某临界值。

干预指向两条路径:
1. **抬高 DC**(承诺、积极错觉、共同记忆、共同未来想象)
2. **压低 AC**(降 Q、增 R、减 L 或 C 的极端值)

通常临床上抬 DC 更直接(Gottman 软启动 / 修复尝试 / 共同意义建构);AC 通常需要长程的 RLC 重塑。

## Part 3:复阻抗 Z 向量与家庭类型

### 二维相图坐标系

把 Z = R + jX 在复平面里展开:

- **实轴 R(横轴)**:外部耗散通道
- **+jωL(上半平面)**:关系惯性
- **-j/(ωC)(下半平面)**:压力蓄水池

注意 X 是净电抗 = ωL - 1/(ωC)。L 大 → X > 0,C 大 → X < 0,LC 共振时 X = 0。

### Z 的两个独立读数

读出 Z 的极坐标 (|Z|, φ) 直接映射两个临床维度。

#### |Z| = 反应钝感度

- |Z| 极小 → 任何小压力都引爆巨大情绪流动 → **过载敏感**
- |Z| 极大 → 巨大压力进来也激不起波澜 → **钝感 / 疏离**

#### φ = 反应时序类型

- φ → +90°(强电感):压力来情绪慢启动、压力走情绪还在跑 → **滞后型 / 反刍型**
- φ → 0°(纯电阻):压力—情绪同步,即来即耗 → **即时耗散型**
- φ → -90°(强电容):情绪超前于压力,但能量被存而非流动 → **警觉 / 隐忍型**

> 临床问诊建议:让伴侣描述"上次大事来的时候,你的情绪反应是马上、慢一拍、还是当时没事后来才发作?"——这就是在测 φ。

### 五种典型向量画像

| 区域 | 名称 | RLC 特征 | 现象 | 风险 |
|---|---|---|---|---|
| ① | 室友型 / 外部消化型 | R 大,L/C 小 | 压力被各自外部渠道消化,家内几乎没有共同情绪流动 | 共同储能不足,大事来临无"共同电路"可调用 |
| ② | 惯性型 / 传统型 | L 大,R/C 小 | 老套路应对一切,反应慢半拍但持续力强 | 对高频突变(失业、青春期)阻抗极大;强行换模式有反激 |
| ③ | 隐忍型 / 回避型 | C 大,R/L 小 | 都吞下去看起来稳;情绪悄悄先于事件浮动 | 电容会击穿 —— 某天突然爆发离婚 / 冷战 / 躯体化 |
| ④ | 燃烧型 / 纠缠型 | R 小,L 与 C 接近 ω₀ | 激烈相爱→激烈冲突→激烈和解,能量在 L、C 间高效来回 | Gottman volatile couple 极端版;持续振荡导致出轨 / 精疲力竭 / 躯体化 |
| ⑤ | 健康 / 可持续型 | R、L、C 均衡 | 事来能装,装不完慢慢释放,有外部耗散;振荡能被阻尼 | 看起来"低戏剧性",但这正是大多数咨询师的目标状态 |

### 谐振频率 ω₀ = 1/√(LC):每个家庭的"敏感频段"

每个家庭都有自己最容易被引爆的"压力节奏"。

- **L 大 + C 大** → ω₀ 低 → **长周期慢压力**会引爆(代际冲突、十年量级事业转型、慢性病)
- **L 小 + C 小** → ω₀ 高 → **高频小事**就能引爆(每天通勤回来一句话不合就吵)

实操含义:**问一个家庭"什么节奏的压力你们最难受",就是在测它的 ω₀**。

### R 不搬 ω₀,只压扁峰

R 几乎不影响 ω₀(它只决定 Q 值,即振荡多快被阻尼掉)。

- 加大 R = **改 Q 不改 ω₀**
- 能让原来的共振不再失控,但搬不走那个共振点
- 要搬走共振点,只能改 L(改写僵化模式)或改 C(扩容 / 泄洪)

> 干预决策树:如果伴侣每次都被同类事件引爆 → 共振是**结构性**的,要改 L 或 C;如果只是反应过强 → 是 Q 太高,加 R 即可。

### Gottman 四类型在 Z 平面的位置

| 类型 | 区域 | 主导参数 |
|---|---|---|
| Volatile 多变 | ④ 燃烧型 | 串联 LC,小 R,Q 高 |
| Validator 确认 | ⑤ 健康 | 并联共轭 + 适度 R |
| Avoider 逃避 | ① 室友型 | 大 R,小 LC |
| Hostile 敌对 | DC 反向 / 不稳定根 | 振荡发散,基线已掉到负 |
