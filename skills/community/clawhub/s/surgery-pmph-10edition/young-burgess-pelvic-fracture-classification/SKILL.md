# Young-Burgess骨盆骨折暴力方向分类

## 适用条件
- 患者具有明确外伤史
- 影像学（如X线、CT）已确认存在骨盆骨折
- 排除非创伤性骨盆病变（如肿瘤、感染、代谢性骨病）

## 分类步骤
1. **确定暴力方向**：结合受伤机制与影像特征，判断主要外力作用方向。
2. **匹配骨折类型**：根据以下四类标准进行归类：
   - **侧方挤压型（LC）**：骨盆受侧方暴力内向挤压，典型表现为耻骨支斜行骨折，伴同侧或对侧后环损伤。
   - **前后挤压型（APC）**：骨盆受前后方向暴力导致外旋，典型表现为耻骨联合分离或耻骨支纵行骨折。
   - **垂直剪切型（VS）**：纵向暴力导致半骨盆垂直移位，常伴骶髂关节完全脱位、骶结节/骶棘韧带断裂，神经血管损伤风险高。
   - **混合暴力型（CM）**：上述类型的复合形式（如LC/VS、LC/APC）。
3. **评估严重程度**：特别关注LC/APC Ⅲ型与VS型，此类骨折稳定性差、并发症多，需优先干预。
4. **结合Tile分型**：Young-Burgess分型侧重暴力机制，应与Tile分型（基于骨盆环稳定性）联合使用以全面评估。

## 输出内容
返回以下信息：
- Young-Burgess分型（LC / APC / VS / CM）
- 典型影像学特征
- 并发症风险（尤其神经血管损伤）
- 临床严重程度评级（轻度 / 中度 / 重度）

> **注意**：本分类不替代完整临床评估，需结合血流动力学状态、合并伤及其他影像发现综合决策。
