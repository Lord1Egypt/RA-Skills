# 生态集成指南

本技能包可与以下外部能力配合，增强实时性和数据覆盖。外部能力都是可选项；如果宿主环境不支持，直接使用本仓库的静态知识库和用户提供的资料。

---

## 网页搜索

**用途**：实时搜索竞品信息、行业热点、最新价格

**典型场景**：
| 场景 | 搜索关键词示例 |
|------|--------------|
| 竞品动态 | "{竞品品牌} {品类} 发布 新品" |
| 行业热点 | "{品类} 热搜 争议 翻车" |
| 最新价格 | "{产品名} 京东 价格" |
| 评测解禁 | "{产品名} 评测 解禁" |

**注意事项**：
- 搜索结果需要经过数据纪律审核——网页数据不等于知识库数据
- 价格信息时效性强，使用时标注"[实时搜索，{日期}]"
- 搜索结果不直接写入知识库，需经数据处理流程清洗后才能入库

---

## 浏览器自动化

**用途**：爬取平台热搜、话题和评论区数据

**可用条件**：仅当宿主智能体支持浏览器自动化或 browser-use 类工具时使用；OpenClaw 等环境不支持时跳过，不影响基础能力。

**典型场景**：
| 场景 | 目标URL |
|------|---------|
| 微博热搜 | weibo.com/hot/search |
| 小红书种草 | 搜索 "{品类} {产品} 推荐" |
| 抖音话题 | 搜索 "#{话题标签}" |
| B站弹幕/评论 | 视频页评论区 |

**注意事项**：
- 爬取的数据同样需要经数据处理流程清洗
- 平台反爬策略不同，成功率不稳定
- 爬取内容不自动进入知识库

---

## 长文本摘要

**用途**：处理长评测视频字幕、长篇行业报告

**典型场景**：
- 1小时+的评测视频字幕 → 提取关键发现
- 行业报告全文 → 提取核心数据点
- 多条评论区数据 → 提取高频观点

**与数据处理流程的关系**：
- 长文本摘要负责压缩：长文本 → 短摘要
- 数据处理流程负责结构化：原始文本 → 知识库格式
- 可以串联使用：先摘要压缩，再结构化入库
