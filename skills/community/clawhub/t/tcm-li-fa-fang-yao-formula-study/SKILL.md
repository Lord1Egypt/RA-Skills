# 理法方药方剂学习法

## 核心定位

把方剂学习从“背组成”改成“推导系统”：

病机 → 治法 → 性味归经 → 方药配伍 → 方歌记忆 → 临床加减边界。

这个技能适合学习方剂学、整理方剂笔记、改编方歌、做病图/药图、解释某首方为什么这样配伍。

## 基本原则

1. 先讲“为什么病”，再讲“为什么治”，最后才讲“为什么用这些药”。
2. 不把方剂当药物清单，要把方剂看成治法的物质载体。
3. 性味归经放在方药之前，因为它是从治法过渡到具体药物的桥梁。
4. 同一味药在不同方中功效不同，常由配伍环境和剂量结构决定。
5. 方歌不是死背文本，而是病机、治法、药物角色的压缩编码。
6. 若涉及真实用药、剂量和临床决策，提醒需由合格中医师结合四诊判断。

## 工作流程

### 1. 明确方剂基本信息

先整理：

- 方名
- 出处
- 组成
- 主治
- 功用
- 常见证候表现
- 舌脉
- 原方剂量，若用户提供则保留；若未提供，不要臆造剂量

如果资料缺失，可以先基于常见教材版本分析，并明确“以下按常见教材理解”。

### 2. 还原病机

从症状倒推病机，不要直接跳到药物。

优先按四要素拆：

- 病因：外感、内伤、久病、误治、饮食劳倦等
- 病位：脏腑、经络、气血津液、三焦、卫气营血、六经等
- 病性：寒热、虚实、燥湿、痰瘀、气血阴阳
- 病势：升降出入失常、传变趋势、正邪消长

输出时要说明：哪些症状支持这个病机，哪些症状是关键指征。

### 3. 推导治法

把病机翻译成治法。

格式建议：

- 因为……所以要……
- 病机核心是……治法核心就是……
- 主法是什么，辅法是什么，兼顾什么风险

常见例子：

- 气虚 → 益气
- 阴伤 → 养阴生津
- 汗出不止 → 敛阴止汗
- 寒凝 → 温散
- 湿阻 → 燥湿/化湿/利湿
- 气滞 → 行气/疏肝/降逆
- 血瘀 → 活血化瘀
- 表证 → 解表
- 里热 → 清热

### 4. 从治法推性味归经

在进入具体药物前，先说明需要什么药性。

分析维度：

- 四气：寒、热、温、凉
- 五味：辛、甘、酸、苦、咸、淡、涩
- 升降浮沉：升、降、浮、沉
- 归经：入哪个脏腑经络，为什么
- 功能方向：补、泻、散、收、升、降、润、燥、通、涩

示例表达：

```text
要益气，通常需要甘温之品；但若兼阴伤内热，则需配甘寒生津之品。
要止汗敛阴，酸味主收敛，酸甘化阴，所以需要酸甘之品。
病位在肺，故药物需入肺经；肺朝百脉、气阴两伤可及心，故兼入心经；一身之气根于肾，必要时兼顾肾经。
```

### 5. 从性味归经推方药

逐味药说明“它为什么会出现在这里”。

不要只写“某药：某功效”，而要写：

- 它对应哪一层病机
- 它落实哪一个治法
- 它的性味归经如何匹配
- 它在方中是君、臣、佐、使，或承担什么功能角色
- 它与其他药形成什么配伍结构

推荐角色标签：

- 君药：直指主病主证
- 臣药：加强君药，或处理重要兼证
- 佐药：制约偏性、处理副病机、反佐
- 使药：引经、调和、协调全方

### 6. 分析配伍结构

重点讲药物之间的关系。

常用关系：

- 相须：同类增强
- 相使：一药辅助另一药发挥主效
- 相畏/相杀：制约毒性或偏性
- 相反/相恶：需说明禁忌或降低作用
- 一补一敛
- 一散一收
- 一升一降
- 一寒一热
- 一气一血
- 一燥一润
- 一攻一守

尤其注意：同一药物在不同方中因配伍与剂量不同，功效方向会改变。比如桂枝在麻黄汤、桂枝汤、小建中汤、五苓散中作用侧重不同。

### 7. 生成记忆方歌

方歌要服务于理解，不要只押韵。

优先把以下内容压进方歌：

- 病机关键词
- 治法关键词
- 药物组成
- 药物角色或配伍关系
- 主治证候

方歌可以分两版：

- 教材背诵版：简短，适合记组成
- 理法方药版：稍长，体现病机治法和配伍逻辑

### 8. 可选：病图/药图说明

当用户要求画图、图解、病图药图时，用文字描述图层，而不是强行生成图片。

病图层次可包括：

- 脏腑气机：心、肺、脾胃、肝胆、肾、膀胱等
- 三焦：上焦、中焦、下焦
- 气血津液：气、血、津、液、痰、瘀
- 卫气营血：卫分、气分、营分、血分
- 六经：太阳、阳明、少阳、太阴、少阴、厥阴

药图层次可包括：

- 药物归经落点
- 药性方向：升降浮沉、寒热温凉
- 药物作用箭头：补、泻、散、收、升、降、通、涩
- 方中配伍关系

## 示例：生脉散

### 一句话抓核心

生脉散是治疗气阴两伤、肺虚津亏、汗出脉虚的基础方。

### 推导链

暑热伤气耗津、温病后期，或久咳伤肺 → 肺气与肺阴俱伤 → 少气懒言、干咳少痰、汗出、舌红少苔、脉虚数或细数。

气虚 → 益气。

阴伤津亏 → 养阴生津。

汗出 → 敛阴止汗。

益气多用甘温，但兼阴伤有热象，需配甘寒；敛汗用酸收，酸甘化阴；病位在肺，兼及心脉与肾气根本，所以涉及肺、心、肾。

人参益气生津，直补肺气；麦冬甘寒养阴生津，润肺清心；五味子酸温收敛，敛肺止汗、生津宁心。三药合用，一补一润一敛，构成益气养阴、生津敛汗之法。

## 回答风格

1. 讲推导，不堆结论。
2. 能用“因为……所以……”串起来，就不要只罗列术语。
3. 对初学者要通俗，但术语必须准确。
4. 对进阶用户可以展开病机层级、配伍结构、剂量比例和异方比较。
5. 用户给出字幕稿、笔记或草稿时，优先修正错别字和术语，再按本法重组。
6. 用户要求“背方歌”时，先给理解链，再给方歌，避免机械记忆。
