## Description: <br>
Creates professional, dark-themed architecture diagrams as standalone HTML files with inline SVG graphics for system, infrastructure, cloud, security, network topology, and component-relationship views. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, architects, and technical teams use this skill to turn system architecture requests into self-contained HTML/SVG diagrams for cloud, network, security, and service-topology communication. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated architecture diagrams can reveal sensitive system names, topology, cloud regions, ports, or security boundaries when shared. <br>
Mitigation: Review and redact diagrams before distribution, especially labels, environment names, network details, and security annotations. <br>
Risk: Users may provide credentials or secrets even though this skill does not need them. <br>
Mitigation: Do not include tokens, passwords, private keys, or other credentials in prompts, diagram labels, summary cards, or generated HTML. <br>
