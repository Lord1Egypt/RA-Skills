# D12 生活运营领域任务目录

## 领域定义

生活运营领域(D12)关注个体日常事务和生活基础设施的管理。核心驱动需求：N1生存 + N2安全。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 日常事务管理 | 家政服务 |
| 时间管理 | 时间管理培训 |
| 居住管理 | 房屋装修 |
| 出行管理 | 旅行社服务 |
| 健康管理 | 医疗服务 |
| 财务管理 | 理财服务 |
| 应急管理 | 应急救援 |
