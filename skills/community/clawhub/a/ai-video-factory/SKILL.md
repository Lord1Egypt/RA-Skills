## 核心流程（9 Steps）

### Step 1: 主题输入与需求确认
- 用户输入主题 / 关键词 / 文案 / URL / 竞品视频链接
- 确认目标平台（支持多平台同时选择）
- 确认视频时长（15-300秒）
- 确认风格：知识科普 / 种草推荐 / 情感故事 / 热点资讯 / 产品评测 / 日常Vlog / 教程演示 / 品牌宣传 / 电商带货 / 探店打卡
- 确认语言：中文 / 英文 / 日语 / 韩语 / 多语言（自动翻译）
- 确认目标受众：年龄段（Z世代/青年/中年/银发）/ 性别倾向 / 兴趣标签
- 参考竞品分析：输入竞品视频链接，自动拆解爆款要素（Hook方式/节奏/文案结构）
- 输出需求确认卡片，用户确认后进入下一步

### Step 2: 爆款选题分析与定位
- 分析同主题TOP50热门视频的共同特征
- 提取高频关键词、热门Hook模式、最佳时长区间
- 识别内容差异化切入点（SWOT分析）
- 评估选题热度与竞争度（热度评分1-100 / 竞争度评分1-100）
- 输出选题策略建议：切入角度、差异化卖点、目标人群画像
- 推荐3个细分切入方向供用户选择

### Step 3: 文案生成与脚本编排
- 自动生成视频脚本（旁白词 + 画面描述 + 时间轴标注 + 情绪标注）
- 字数控制公式：60秒 ≈ 150-200字，30秒 ≈ 80-100字，15秒 ≈ 40-60字
- **Hook开场**（前3秒决定生死）：
  - 悬念型："你知道XXX最大的误区是什么吗？"
  - 数字冲击型："3个方法，让我的XXX提升了200%"
  - 反常识型："千万别再XXX了！越XXX越XXX"
  - 痛点型："你是不是也经常遇到这种情况..."
  - 结果前置型："看完这个视频，你将学会..."
  - 对比型："XXX vs XXX，差距居然这么大？"
- 中段信息密度控制（每10秒一个信息点 + 情绪起伏设计）
- 结尾CTA引导（关注/点赞/评论/收藏/转发，根据平台特性选择）
- 分镜脚本输出：每个镜头的画面描述、时长、转场方式、配音文本、情绪曲线
- A/B版本生成：同一主题生成2个不同Hook的脚本版本，供用户选择

### Step 4: 素材采集与智能匹配
- 根据脚本分镜搜索匹配素材（图片/视频片段/动态图形/数据图表）
- 支持免费素材库：Pexels / Pixabay / Unsplash / Coverr / Mixkit
- 无匹配时触发AI生成：图片（DALL-E/SD）/ 视频片段
- 素材分辨率自动校验（≥1080p，4K优先）
- 自动过滤带水印 / 低质量 / 版权不明素材
- 素材风格一致性检查（色调/风格/质量统一）
- 支持用户上传自有素材（自动增强：锐化/调色/降噪）
- 素材匹配度评分：< 60% 自动降级搜索词 / 用AI补充 / 提醒用户手动补充

### Step 5: 语音合成与配音
- TTS语音合成：多音色可选（标准男声/磁性男声/标准女声/甜美女声/童声/老年声/新闻播音/方言版）
- 语速控制：0.8x（舒缓）/ 1.0x（标准推荐）/ 1.2x（快节奏）/ 1.5x（极限快）
- 情感语调：平静 / 热情 / 严肃 / 温柔 / 激动 / 幽默
- 多语言支持：中文 / 英文 / 日语 / 韩语 / 西班牙语 / 法语 / 德语
- 支持用户自定义上传配音文件（自动对齐时间轴）
- 语音情绪自适应：根据文案情绪标注自动调整语调起伏
- 多角色配音支持：对话类视频可分配不同角色音色（最多4个角色）
- 配音预览：生成后先试听，不满意可重新生成

### Step 6: 字幕渲染与视觉增强
- 字幕模式：逐字显示 / 逐句显示 / 卡拉OK效果 / 打字机效果 / 弹跳效果
- 字幕样式：字体 / 颜色 / 描边 / 阴影 / 位置 / 动画可调
- 关键词高亮：自动识别重点词汇并变色（支持自定义高亮词库）
- 转场效果：淡入淡出 / 缩放 / 滑动 / 模糊切换 / 旋转 / 3D翻转 / 闪白 / 叠化
- 动态贴纸/表情包自动插入（根据内容情绪智能匹配）
- 片头片尾模板：品牌Logo动画 / 二维码 / 社交媒体信息 / 订阅引导
- 画中画效果：支持主讲人 + 内容画面的画中画布局
- 视觉安全区检查：确保字幕/关键内容不被平台UI遮挡

### Step 7: 背景音乐与音效
- 自动匹配情绪风格BGM：轻快 / 温馨 / 悬疑 / 史诗 / 伤感 / 科技 / 古风 / 摇滚 / 爵士 / 电子 / 乡村 / 古典
- 音量自动闪避：有人声时BGM自动降至15-25%
- 音效点缀：转场音 / 强调音 / 氛围音 / 点击音 / 通知音 / 自然音
- 支持用户上传自定义BGM
- 版权安全检查：确保BGM可商用（标注来源和授权类型）
- BGM节奏卡点：自动检测音乐节奏点并匹配画面切换（节拍对齐）
- 音乐淡入淡出：开头2秒渐入，结尾3秒渐出

### Step 8: 质量检查与预览
- **自动质量检测清单**（10项）：
  1. 画面匹配度（素材与文案的相关性评分 ≥ 7/10）
  2. 音画同步度（配音与字幕的时间偏差 ≤ 200ms）
  3. 视觉舒适度（色彩/亮度/对比度在舒适区间）
  4. 平台合规性（分辨率/时长/码率符合目标平台要求）
  5. 字幕准确性（无错别字/时间轴对齐/显示完整）
  6. 原创度检测（内容原创度 ≥ 70%）
  7. 音量标准度（整体音量符合平台推荐标准 -14 LUFS）
  8. 安全区检查（关键内容不被平台UI遮挡）
  9. 转场流畅度（无突兀跳切/闪烁/黑帧）
  10. CTA完整性（结尾引导完整且符合平台特性）
- 生成低分辨率预览版本（720p快速预览）
- 输出质量报告与逐项优化建议
- 用户确认后进入最终渲染

### Step 9: 合成输出与多平台适配
- 输出MP4文件（1080p/4K可选）
- 自动适配多平台规格（比例/分辨率/帧率/码率/编码格式）
- 附带SRT/ASS字幕文件
- 输出封面图（各平台尺寸 + 文字模板）
- 批量导出：一次生成多平台版本
- 输出视频数据卡片（标题/描述/标签/发布建议）
- 生成发布排期建议（最佳发布时间/发布频率/话题标签）
- 输出视频数据追踪模板（播放量/完播率/互动率记录表）

## 批量生产模式

### 矩阵号批量生产流程
1. 输入主题列表（支持CSV/Excel批量导入，单次≤50条）
2. 为每个主题自动生成差异化脚本（避免内容重复，原创度≥70%）
3. 批量素材匹配与AI补充（统一风格基调）
4. 批量渲染输出（支持并行渲染，效率翻倍）
5. 自动生成各平台发布排期表
6. 输出批量生产报告（完成率/平均质量分/素材复用率）

### 批量命名规范
```
{日期}_{平台}_{主题关键词}_{版本号}.mp4
示例：20250115_douyin_AI教程_v1.mp4
```

### 批量质量管控
- 逐条质量检查，低于70分的自动标记为需人工复核
- 批量原创度检测，确保账号矩阵内容不互相撞车
- 素材去重检查，避免同一素材在多条视频中重复出现

## 边界约束
1. **版权合规**：仅使用免费/授权素材，BGM需可商用，所有素材需标注来源和授权类型
2. **时长限制**：单视频≤5分钟（超出建议拆分为系列短视频）
3. **素材质量**：自动过滤低于720p或带水印素材，4K素材优先使用
4. **内容安全**：不生成涉政/涉黄/虚假宣传/医疗建议/金融投资建议内容
5. **多语言**：翻译质量基于AI，涉及专业术语建议人工校对
6. **批量限制**：单次批量不超过50条（防止资源过载）
7. **原创度**：自动检测内容原创度，确保≥70%
8. **平台规则**：遵循各平台最新社区规范，避免违规内容导致限流/封号
9. **用户素材**：用户上传素材仅用于当前任务，不做二次传播
10. **数据隐私**：不存储用户上传的原始素材和成品视频
