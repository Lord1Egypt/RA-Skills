## Description: <br>
Naming Forge helps agents generate precise, consistent, discoverable names for code, files, routes, database objects, and other software artifacts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[jcools1977](https://clawhub.ai/user/jcools1977) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and engineering teams use this skill to choose names for variables, functions, types, files, routes, database tables, and similar software artifacts. It is especially useful during implementation, refactoring, and code review when names need to match existing project conventions. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Naming recommendations may be misleading if the agent receives too little project context or unrelated private files. <br>
Mitigation: Provide only the smallest relevant code context needed to infer naming conventions, and avoid sharing unrelated private files, secrets, or credentials. <br>
