## 主光类 / Key Lights

### Key Light / 主光
场景中最强的光源，决定整体曝光和阴影方向。
- **用法：** AI 生图中说 `key light from [direction]` 确定主光方位
- **Prompt：** `strong key light, single source lighting`

### Back Light / 逆光 / 背光
位于主体背后，正对镜头方向的光。制造轮廓、分层、剪影。
- **用法：** `backlight, strong backlight from behind`
- **适用：** 分离主体与背景，制造神秘感

### Rim Light / 轮廓光 / 边缘光
背光的变体，专门照亮主体边缘，分离前景与背景。
- **Prompt：** `rim light, edge lighting, hair light, separation light`
- **效果：** 银边、金边、抽离感

### Side Light / 侧光
光源位于主体侧方（90°），制造强烈明暗对比。
- **用法：** `side light, hard side lighting, 90-degree angle`
- **适用：** 审讯、冲突、戏剧张力

### Fill Light / 补光
辅助光，填补主光造成的阴影，控制光比。
- **Prompt：** `soft fill, fill light, bounce light`
- **比例：** 通常是主光的 1/4（光比 4:1）到 1/2（光比 2:1）

## 光质类 / Light Quality

### Hard Light / 硬光
未经散射的定向光，边缘锐利的阴影。
- **特征：** 小光源、晴天正午、聚光灯
- **Prompt：** `hard light, crisp shadows, direct sunlight`
- **情绪：** 紧张、压迫、审讯

### Soft Light / 柔光
经过散射的光源，阴影边缘柔和、过渡平滑。
- **特征：** 大光源、阴天、窗户柔光
- **Prompt：** `soft light, diffused light, wraparound, feathered`
- **情绪：** 温柔、浪漫、柔和

### Chiaroscuro / 明暗对比
文艺复兴光影技法，强调亮部与暗部的强烈对比。
- **Prompt：** `chiaroscuro lighting, high contrast`
- **适用：** 黑色电影、恐怖、戏剧

## 光方向类 / Directional Terms

### Front Light / 正面光
正对镜头方向的光，主体受光均匀，阴影在身后。
- **特点：** 平面、安全感、信息传达
- **Prompt：** `front light, flat lighting, flat key`

### Three-Quarter Light / 四分之三光
主光位于主体侧面偏前方（45°–60°），经典人像光。
- **效果：** 立体感 + 安全感兼顾
- **Prompt：** `three-quarter lighting, 45-degree key`

### Under Light / 底光
从下方打上来的光，制造诡异、不自然感。
- **情绪：** 恐怖、审讯、精神控制
- **Prompt：** `under light, uplighting, low angle light`

### Top Light / 顶光
从正上方打下来的光，眼睛陷入阴影。
- **Prompt：** `overhead light, top light, high angle`

## 特殊光效类 / Special Effects

### Catchlight / 眼神光
眼睛中的光源反射点，让眼睛有生命感。
- **Prompt：** `catchlight in eyes, specular highlight, eye reflection`

### Rembrandt Light / 伦勃朗光
45°侧光，在阴影面脸颊形成三角形高光区。
- **标志：** 鼻阴影与颧骨阴影形成倒三角形
- **Prompt：** `Rembrandt lighting, triangle light on cheek`

### Butterfly Light / 蝴蝶光
主光在正上方，鼻阴影落在嘴唇正下方，形成蝴蝶形状。
- **适用：** 时尚摄影、女性肖像
- **Prompt：** `butterfly lighting, paramount lighting`

### Split Light / 分割光
光从正侧面打来（90°），一半脸亮一半脸暗。
- **情绪：** 极端、戏剧、冲突
- **Prompt：** `split lighting, half face lit`

### Silhouette / 剪影
主体严重欠曝，轮廓清晰，背景过曝。
- **Prompt：** `silhouette, backlit silhouette, rim silhouette`

### Lens Flare / 镜头光晕
光线在镜头内折射产生的光斑和条纹。
- **Prompt：** `lens flare, anamorphic flare, cinematic flare`

### Volumetric Light / 体积光
光线穿过介质（烟、雾、尘埃）时可见的光束。
- **Prompt：** `volumetric light, god rays, light beams, atmospheric, dust particles in light`

### Bokeh / 焦外虚化
背景光源虚化成圆形光斑。
- **Prompt：** `bokeh, defocused lights, background blur, lens bokeh`

## 场景类灯光 / Scene Lighting

### Motivated Lighting / 依据现场光源
模拟真实环境光源（窗户、蜡烛、屏幕）。
- **原则：** 所有光都有来源依据，不能凭空出现

### Practical Light / 实用光
场景中可见的光源（台灯、蜡烛、屏幕）。
- **用法：** `practical light, practical lamp, visible light source`

### Ambient Light / 环境光 / 散射光
整体环境的散射光，没有明确方向。
- **Prompt：** `ambient light, ambient fill, bounced light`

### Moody Lighting / 情绪光
低调光（Low Key）高反差，强调氛围而非信息。
- **Prompt：** `moody lighting, cinematic mood, atmospheric`

## 光比类 / Lighting Ratios

| 光比 | 描述 | Prompt 关键词 |
|------|------|-------------|
| 1:1 | 平调 | flat, even, balanced |
| 1:2 | 柔和 | soft, low ratio |
| 1:4 | 电影感 | cinematic, medium contrast |
| 1:8 | 戏剧 | dramatic, high contrast |
| 1:16+ | 德国表现主义 | German Expressionist, extreme shadow |

## AI 生图灯光 Prompt 公式

```
[主光类型] + [方向角度] + [光质] + [色温/颜色] + [阴影描述] + [特殊光效]
```

### 示例 1：经典电影感
```
hard side light, 90-degree angle, key light from camera left,
warm golden tone #D4A574, deep shadows on right face,
Rembrandt triangle on cheek, volumetric dust, cinematic
```

### 示例 2：悬疑低对比
```
low key lighting, single source from above,
cool blue shadow tone #4A3F6B, hard light,
high contrast 1:8 ratio, chiaroscuro,
half face in shadow, dramatic
```

### 示例 3：浪漫逆光
```
backlight, rim lighting, warm golden hour #D4A574,
soft fill from front, catchlight in eyes,
hair light halo, silhouette edge,
volumetric fog, dreamy
```

## 常见错误警告

❌ **不要：** `beautiful lighting, nice light`（模糊无效）
✅ **应该：** `single key light 45° left, hard edge shadow, golden #D4A574`

❌ **不要：** `many lights, studio lighting`（多光源混乱）
✅ **应该：** `single key light, one practical lamp visible`

❌ **不要：** `no shadows`（消灭阴影=消灭电影感）
✅ **应该：** `controlled shadow direction, nasal shadow on cheek`

---

> **记住：** 灯光不是照亮东西。是雕刻东西。光的方向决定情绪，光的颜色决定温度，光的软硬决定质感。
