# ASCII Tool - Character Codes

Work with ASCII character codes and convert between decimal/hex/binary.

## Quick Start

```bash
ascii-tool --code A
```

## Features

- Look up character codes
- Convert decimal/hex/binary
- Generate ASCII table

## Examples

```bash
ascii-tool --code A
ascii-tool --decode 41
ascii-tool --table
```
