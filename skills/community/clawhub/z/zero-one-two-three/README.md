# Zero-One-Two-Three 知识架构师

> **核心理念：0+1+2≠3→∞**
>
> 基于中国道家思维"道生一，一生二，二生三，三生万物"的 AI 知识创生与全球分发系统。

## 🌟 项目简介

Zero-One-Two-Three 是一个**完整的知识资产管理与变现平台**，帮助内容创作者：

- 🧠 **深度提取**知识 → 结构化存储
- 🔐 **加密保护** → 知识资产安全
- 🎭 **风格克隆** → 打造数字分身
- 🎙️ **语音合成** → 内容多模态输出
- 📚 **图书馆管理** → 知识分类检索
- 🔥 **阅后即焚** → 安全内容分享
- 📧 **自动发货** → 商业闭环变现

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 核心功能演示

```bash
# 1. 分析你的写作风格
python style_clone.py analyze ./你的文章.md

# 2. 用你的声音朗读
python voice_clone.py speak "你好，我是你的数字分身"

# 3. 加密保护知识笔记
python knowledge_lock.py lock 笔记.md 你的密码 --preview 30

# 4. 管理个人知识图书馆
python personal_library.py --scan ./你的笔记目录

# 5. 创建阅后即焚分享
python ephemeral_share.py --create 机密文件.md --reads 1

# 6. 启动系统检查
python main.py
```


## 🔧 环境配置

### Windows PowerShell

```powershell
# 设置环境变量（永不过期）
[System.Environment]::SetEnvironmentVariable('ZOT_MAIL_USER', '你的邮箱', 'User')
[System.Environment]::SetEnvironmentVariable('ZOT_MAIL_PASS', '邮箱授权码', 'User')
[System.Environment]::SetEnvironmentVariable('ZOT_AUTHOR_EMAIL', '联系邮箱', 'User')
[System.Environment]::SetEnvironmentVariable('ZOT_NOTION_TOKEN', 'Notion密钥', 'User')
```

设置后**重启终端**生效。
