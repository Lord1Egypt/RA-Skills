# daily-report-generator

本地日报/周报生成工具。输入 JSON 格式工作数据，生成结构化的 Markdown 格式工作报告。

## 功能

- 📋 **日报生成** — 汇总今日完成事项、进行中任务、明日计划
- 📊 **周报生成** — 按周汇总，支持数据指标表格
- 📤 **多输出方式** — 输出 Markdown 格式，可直接复制到任何平台

## 使用方法

### 触发词
"生成日报"、"写周报"、"汇总工作"、"每天自动发日报"

### 配置（首次使用）
向 AI 确认以下配置：
- 报告类型：日报 / 周报
- 数据格式：JSON（示例见下方）
- 输出目标：Markdown / 文档

### JSON 数据格式
```json
{
  "completed": ["完成用户调研报告", "上线新功能A"],
  "in_progress": ["优化登录流程"],
  "planned": ["编写技术文档"],
  "risks": ["第三方API不稳定"],
  "metrics": [{"name": "DAU", "value": "1000", "change": "+5%"}]
}
```

### 示例输出
```markdown
# 📋 日报 | 2026年05月07日

## ✅ 今日完成
- 完成用户调研报告
- 上线新功能A

## 🔄 进行中
- 优化登录流程

## 📅 明日计划
- 编写技术文档

## ⚠️ 风险提示
- 第三方API不稳定
```

## 系统要求

- Python 3.7+

## 技术实现

- 数据收集：`scripts/collect_data.py`
- 内容生成：`scripts/generate_report.py`
- 纯标准库，无外部 API 依赖
