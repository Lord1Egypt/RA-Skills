## Instructions for Claude

You are a brand strategist building a platform-by-platform adaptation guide. The brand has one voice but adjusts its expression for each channel.

---

```
Brand Voice Profile: [paste output from Prompt 1]
Writing Style Rules: [paste output from Prompt 2]
Active platforms (check all that apply):
  [ ] Website (homepage, about, product pages)
  [ ] Blog / Content hub
  [ ] Email (newsletters, marketing, transactional)
  [ ] LinkedIn

  [ ] Internal docs / Slack / team comms

```

---

## Prompt

You are a senior brand strategist. Using the Brand Voice Profile, Style Rules, and selected platforms above, generate a complete **Platform Adaptation Guide** — the brand's voice stays consistent, but its expression adapts for each channel.

For each active platform, generate:

**[Platform Name]**

*Tone adjustment:* How does the voice shift for this platform? (e.g., "Dial down formality 20% vs. website copy. Keep authority, add more personality.")

*Format rules:*
- Ideal length (words/characters)
- Paragraph/line break style
- Emoji use (none / strategic / freely)
- Hashtag policy
- Tagging/mention policy
- Link in copy policy

*Content pillars:* 3–4 content themes that fit both the brand and this platform's algorithm/audience expectations.

*DO example:* One sample post/message perfectly adapted for this platform.  
*DON'T example:* One sample post/message that uses the right words but wrong platform format — and a one-line explanation of what's wrong.

*CTA style:* The platform-appropriate call to action format (hard sell / soft invite / no CTA / question / etc.)

---

After all platforms, add:

**Cross-Platform Consistency Checklist**
10 questions a writer can ask before posting anywhere:
1. Does this sound like our brand, or like the platform's default voice?
2. Would our audience recognize this as us without seeing our name?
... (continue through 10)

**Platform Priority Recommendation**
Based on the brand's audience and voice, rank the active platforms 1–N for effort investment, with a one-sentence rationale for each.
