## 📖 阅读建议

### 首次使用（必读）
1. `用户人设.md` - 了解智灵动力是谁
2. `style.md` 前 200 行 - 掌握核心风格
3. `降AI味指南.md` - 学会识别 AI 套话
4. `QUICK_REFERENCE.md` - 记住常用命令

### 创作前（快速查阅）
1. 根据平台类型，查阅对应指南（wechat/xiaohongshu/video）
2. 查看 `反模板化指南.md` - 避免重复
3. 查看 `style.md` 中的开头模板

### 审校时（重点检查）
1. `降AI味指南.md` - 逐项检查套话
2. `style.md` 中的论证四要素 - 检查论证质量
3. `反模板化指南.md` 审校清单 - 检查原创性
