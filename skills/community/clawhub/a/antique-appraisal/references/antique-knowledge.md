# 古董鉴定知识库

## 一、主要藏品类别

### 1. 陶瓷 (Ceramics)

| 类别 | 代表品种 | 鉴定要点 |
|------|---------|---------|
| 青花瓷 | 元青花、永宣青花、康熙青花 | 钴料发色、胎质、画工风格、底足特征 |
| 粉彩 | 雍正粉彩、乾隆粉彩 | 颜料层次、釉面光泽、纹饰精细度 |
| 五彩/斗彩 | 成化斗彩、万历五彩 | 填色工艺、釉下勾勒技法 |
| 单色釉 | 汝窑、官窑、哥窑、定窑、钧窑 | 釉色纯正度、开片纹路、器型比例 |
| 紫砂 | 时大彬、陈鸣远、顾景舟 | 泥料质地、成型工艺、款识印章 |

**估价关键因素**：年代(40%) > 品相(25%) > 稀有度(15%) > 尺寸器型(10%) > 来源传承(10%)

### 2. 书画 (Paintings & Calligraphy)

| 类别 | 鉴定要点 | 常见问题 |
|------|---------|---------|
| 古代书画 | 纸绢老化程度、墨色渗透、印泥成分、装裱风格 | 揭裱/重裱痕迹、后人添款 |
| 近现代书画 | 笔墨风格、落款习惯、用纸特征 | 印刷品冒充、学生代笔 |
| 油画 | 画布材质、颜料层次、笔触肌理 | 覆盖重绘、签名伪造 |
| 书法 | 行笔韵律、墨色浓淡、纸张年代 | 摹本/临本辨别 |

**估价关键因素**：作者知名度(40%) > 作品质量(25%) > 流传有序(15%) > 题材稀有度(10%) > 品相(10%)

### 3. 玉器 (Jade)

| 类别 | 代表品种 | 鉴定要点 |
|------|---------|---------|
| 和田玉 | 羊脂白玉、青白玉、黄玉 | 质地细腻度、油润感、皮色自然度 |
| 翡翠 | 老坑玻璃种、冰种、糯种 | 种水、颜色、瑕疵、工艺 |
| 古玉 | 红山文化、良渚文化、商周玉器 | 沁色自然度、加工痕迹、纹饰时代风格 |
| 地方玉 | 独山玉、岫玉、密玉 | 与和田玉的区分、价值差异 |

**估价关键因素**：材质(35%) > 工艺(20%) > 年代(20%) > 来源(15%) > 尺寸(10%)

### 4. 青铜器/金银器

- **鉴定重点**：铜质成分、铸造工艺（范铸/失蜡法）、锈蚀状态、铭文内容
- **法律提示**：出土青铜器属于国家文物，私人交易须遵守《文物保护法》
- **估价因素**：年代(30%) > 铭文价值(25%) > 品相(20%) > 稀有度(15%) > 尺寸(10%)

### 5. 钱币 (Coins & Currency)

| 类别 | 鉴定要点 | 估价范围参考 |
|------|---------|------------|
| 古钱币 | 钱文风格、铜质锈色、直径重量 | 普通品¥50-500，珍稀品¥5万-500万 |
| 银元 | 边齿特征、银色包浆、压力感 | 普通品¥300-2000，珍稀品¥5万-200万 |
| 纸币 | 纸张质感、水印、冠字号 | 普通品¥50-500，珍稀品¥5000-50万 |

### 6. 木器/家具

- **鉴定重点**：木材鉴定（紫檀/黄花梨/红木/楠木等）、榫卯结构、年代皮壳
- **估价因素**：材质(35%) > 工艺(25%) > 年代(20%) > 品相(15%) > 尺寸(5%)

### 7. 杂项 (Miscellaneous)

涵盖：文房四宝、鼻烟壶、铜镜、印章、竹木牙角雕、钟表、古籍善本等。

鉴定时参考对应品类的专业知识库。

## 二、各朝代典型特征

### 唐以前
- 陶瓷：原始青瓷、汉代陶俑色彩古朴、造型粗犷
- 青铜：商周重器纹饰繁缛（饕餮纹/夔龙纹）、器物厚重
- 玉器：多为礼器（璧/琮/圭/璋），加工痕迹可见解玉砂痕迹

### 唐宋
- 陶瓷：唐三彩、宋五大名窑（汝官哥定钧）、器型优雅内敛
- 书画：宋画写实精微、书法以行草楷为主
- 铜镜：唐镜纹饰华丽（海兽葡萄纹）、宋镜趋于简洁

### 元明
- 陶瓷：元青花开创釉下彩新纪元、明永宣青花发色浓艳、成化斗彩精巧
- 家具：明式家具简洁流畅（圈椅/官帽椅/罗汉床）
- 书画：文人画兴起、吴门四家、董其昌

### 清
- 陶瓷：康雍乾三代巅峰、粉彩珐琅彩极尽工巧、晚期衰退
- 家具：清式家具繁缛华丽、重雕刻装饰
- 玉器：乾隆工精湛、大型山子雕流行

### 民国
- 陶瓷：景德镇珠山八友、仿古瓷盛行
- 钱币：袁大头/孙小头/船洋等机制币
- 书画：海派/京津画派、中西融合

## 三、品相评级标准

| 等级 | 描述 | 价值折扣 |
|------|------|---------|
| 全品 | 原装原套，无明显瑕疵 | 100% |
| 近全品 | 微小的岁月痕迹，无明显损伤 | 85-95% |
| 小品 | 有小瑕疵（小磕/小冲/小修），不影响整体观赏 | 60-80% |
| 中品 | 有明显瑕疵或修复痕迹 | 40-60% |
| 大残 | 严重损伤、残缺、大修 | 20-40% |

## 四、常见做旧/仿制造假手法

1. **陶瓷**：酸蚀做旧、人工打磨底足、化学染色、低温复烧
2. **书画**：染色做旧纸张、仿造虫蛀痕迹、化学熏制老化、印刷品覆墨
3. **玉器**：染色做沁、强酸腐蚀、滚筒打磨仿古、激光染色
4. **青铜器**：化学腐蚀仿锈、拼接伪造、翻砂仿铸
5. **钱币**：改刻年份、黏合拼接、电镀做旧
6. **木器**：鞋油做旧、强酸腐蚀、拼接不同家具部件

## 六、鉴定常用术语

| 术语 | 含义 |
|------|------|
| 开门 | 一眼真，特征典型明确 |
| 大开门 | 毫无疑问的真品 |
| 有一眼 | 看上去还不错，值得进一步鉴定 |
| 不开门/不对 | 赝品或仿品 |
| 到代 | 年代判断准确，就是该时期作品 |
| 后仿 | 后代仿制，非原作年代 |
| 老仿 | 有一定年头的仿品 |
| 包浆 | 器物表面经长期氧化、触摸形成的自然光泽层 |
| 皮壳 | 家具/木器表面的老旧氧化层 |
| 沁色 | 玉器入土后受环境浸染的颜色变化 |
| 冲口 | 瓷器口沿的小缺口 |
| 窑缝/窑封 | 烧制时产生的裂纹，非后期损坏 |
| 惊釉 | 釉面的细微裂纹，非撞击导致 |
