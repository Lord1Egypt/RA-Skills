# 子宫收缩过强分类与风险预警

## 适用场景
- 产妇存在规律宫缩
- 产道状态和胎位已评估
- 出现以下任一表现：宫缩过于频繁（<2分钟/次）、持续性剧烈腹痛、烦躁不安、腹部拒按、胎心异常或胎位不清

## 执行步骤

### 第一步：判断子宫收缩过强类型
1. **协调性子宫收缩过强**：宫缩节律性、对称性及极性正常，仅强度或频率异常增高。
   - 若产道无阻力且总产程＜3小时 → 警惕**急产**（多见于经产妇）
   - 若存在产道梗阻或瘢痕子宫 → 警惕**病理性缩复环**及**子宫破裂**

2. **不协调性子宫收缩过强**：宫缩失去正常节律性，分为两类：
   - **强直性子宫收缩**：宫缩呈持续性强直，无间歇期
     - 常见诱因：缩宫剂使用不当
     - 表现：持续腹痛、烦躁、胎心听不清、胎位难查
     - 风险：合并梗阻时可致先兆子宫破裂（血尿、病理性缩复环）
   - **子宫痉挛性狭窄环**：子宫局部平滑肌持续痉挛形成环状狭窄
     - 常见诱因：精神紧张、疲劳、粗暴操作或缩宫剂滥用
     - 表现：持续腹痛、胎心变异、宫颈扩张停滞、胎先露下降受阻
     - 特征：狭窄环位于胎体狭窄部或子宫上下段交界处，**不随宫缩上升**（区别于病理性缩复环）
     - 第三产程可致**胎盘嵌顿**

### 第二步：启动相应预警与干预
- 怀疑强直性收缩或狭窄环 → 立即停用缩宫剂，评估是否需剖宫产
- 出现病理性缩复环、血尿、剧烈腹痛 → 视为**先兆子宫破裂**，紧急剖宫产
- 第三产程手取胎盘时触及宫颈内口上方环状结构 → 提示痉挛性狭窄环，需解除痉挛后处理

> **注意**：本技能不适用于无宫缩或宫缩乏力情况，亦不适用于非妊娠状态。
