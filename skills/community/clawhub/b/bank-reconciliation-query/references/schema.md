## 字段匹配规则

**🛑 不要猜测字段名。** 从查询返回的 `record.fields` 对象中直接检查以下模式：

1. **月份金额**: 检查 `202601` ~ `202605` 是否存在且不为空
2. **对账完成**: 检查字段名是否包含`对账完成`的文字，对应月份的字段名可能是：
   - `1月对账完成`, `2月对账完成`（1-2月）
   - `26年3月对账完成`（3月）
   - `2604对账完成`（4月）
   - `2605对账完成`（5月）
3. **出账**: 检查 `2603出账` / `2604出账` / `2605出账` 是否为 `"已出账"`
4. **对账金额**: 检查 `1月对账金额` / `2月对账金额` / `3月对账金额` / `2604对账金额` / `2605对账金额`

## 判断逻辑

```
对账已完成 = (XX对账完成 == "完成" || XX对账完成 == "对账完成但暂不开票" || XX对账完成 == "已完成暂不开票")
对账未完成 = (2026XX 有金额且金额 != "0.0万") && (XX对账完成 字段不存在 || 为空 || 非已完成)
无交易/未到账单 = (2026XX 无金额 || 2026XX == "0.0万") && XX对账完成 字段不存在或为空
```
