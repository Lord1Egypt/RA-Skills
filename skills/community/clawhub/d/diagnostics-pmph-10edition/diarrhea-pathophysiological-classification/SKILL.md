# 腹泻的病理生理机制分类

## 触发条件
当患者符合腹泻定义（每日排便≥3次液状便，或粪量>200g且含水量>80%）时，执行本技能。

## 执行步骤

1. **收集关键临床信息**：
   - 粪便性质（水样、脓血、脂肪泻、果酱样等）
   - 是否伴有腹痛
   - 禁食后腹泻是否缓解
   - 体重变化情况（如明显消瘦）
   - 已知基础疾病（如甲亢、糖尿病、胰腺疾病等）

2. **根据机制特征判断主导类型**：

   - **分泌性腹泻**：
     - 特征：粪便量大、水样，常无腹痛
     - 典型病因：霍乱弧菌感染、胃肠道内分泌肿瘤（如VIP瘤）、炎症性肠病活动期、放射性肠炎等

   - **渗出性腹泻**：
     - 特征：粪便含脓血或黏液
     - 典型病因：细菌性痢疾、阿米巴痢疾、炎症性肠病、缺血性肠炎等

   - **渗透性腹泻**：
     - 特征：禁食后腹泻停止
     - 典型病因：乳糖酶缺乏、服用盐类泻剂（如硫酸镁）、甘露醇摄入等

   - **动力性腹泻**：
     - 特征：常伴腹痛，肠鸣音亢进
     - 典型病因：肠易激综合征、甲状腺功能亢进、糖尿病自主神经病变等

   - **吸收不良性腹泻**：
     - 特征：粪便含脂肪（脂肪泻），体重明显下降
     - 典型病因：小肠切除术后、乳糜泻、慢性胰腺炎、吸收不良综合征等

3. **综合判断**：
   - 多数腹泻病例涉及多种机制，应识别其中占主导地位的机制
   - 结合病史与检查结果，输出最可能的机制类型及对应病因

> 注意：本技能不适用于便秘或其他非腹泻性排便异常。
