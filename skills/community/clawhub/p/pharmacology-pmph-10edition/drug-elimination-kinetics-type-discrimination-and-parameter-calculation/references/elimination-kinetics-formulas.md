# 药物消除动力学详细公式与推导

## 一级消除动力学
- 微分方程：dC/dt = -Kₑ·C
- 解析解：Cₜ = C₀·e⁻ᴷᵉᵗ
- 对数形式：lg Cₜ = (-Kₑ/2.303)·t + lg C₀
- 半衰期：t₁/₂ = 0.693 / Kₑ

## 零级消除动力学
- 微分方程：dC/dt = -K₀
- 解析解：Cₜ = C₀ - K₀·t
- 浓度降至零的时间：t = C₀ / K₀

## 混合消除动力学（米-曼动力学）
- 微分方程：dC/dt = -(Vₘₐₓ·C)/(Kₘ + C)
- 特征：
  - Kₘ 是达到 50% 最大消除速率 (Vₘₐₓ) 所需的药物浓度；
  - 当 C << Kₘ：dC/dt ≈ -(Vₘₐₓ/Kₘ)·C → 表现为一级动力学，Kₑ = Vₘₐₓ/Kₘ；
  - 当 C >> Kₘ：dC/dt ≈ -Vₘₐₓ → 表现为零级动力学，K₀ = Vₘₐₓ。

## 典型药物示例
- 一级动力学：大多数药物（如地高辛、茶碱在治疗浓度）；
- 零级动力学：乙醇（高剂量时）、苯妥英钠（高浓度时）；
- 混合动力学：水杨酸、苯妥英钠、甲氨蝶呤。

## 参数单位
- Kₑ：时间⁻¹（如 h⁻¹）
- K₀：浓度·时间⁻¹（如 mg/L/h）
- Vₘₐₓ：浓度·时间⁻¹
- Kₘ：浓度（如 mg/L）
