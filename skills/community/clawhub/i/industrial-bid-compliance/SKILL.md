# 工业招投标技术方案智能审查

## Overview

本技能专注于**工业领域招投标技术方案的逐条标准合规性审查**。与通用投标分析工具不同，本技能的核心价值不是"帮你写出高分标书"，而是"帮你查出不合规之处"——将技术方案中的每一条实质性要求与适用的国家/行业强制性标准进行逐条对标，输出合规/不合规/缺项的精准判定，生成具有法律效力的合规审查报告。

**核心差异化**：
- 🎯 聚焦"合规"而非"评分"——关注是否满足标准，而非竞争力
- 📋 逐条对标国家/行业标准——GB系列、AQ安全标准、HJ环保标准、GB/Z能效标准等
- ⚠️ ★否决项精准检测——自动标记实质性条款不满足导致的废标风险
- 🏭 工业领域专项——覆盖设备采购、工程施工、技术服务三大工业招投标类型
- 📊 9维合规雷达——从技术参数到知识产权的全覆盖审查

## 触发词

当用户提到以下关键词时激活本技能：
- 工业招投标审查 / 技术方案审查 / 投标合规检查 / 标书合规

## 阶段1：文档解析与条款提取

### 输入支持
- PDF（含扫描件，需OCR提示）
- DOCX / DOC
- XLSX / XLS
- 纯文本粘贴

### 项目类型识别

根据文档内容自动识别工业项目类型：

| 类型代码 | 项目类型 | 典型特征 | 适用标准体系 |
|----------|----------|----------|-------------|
| `EQUIP` | 工业设备采购 | 含设备参数表、技术规格书 | GB/T系列产品标准 |
| `ENG` | 工业工程施工 | 含施工组织设计、安全方案 | GB 5xxxx系列施工规范 |
| `SVC` | 工业技术服务 | 含服务流程、交付标准 | GB/T 服务标准 |

### 条款提取输出

每条条款结构化输出：

```yaml
clause:
  id: "CL-001"
  section: "第三章 技术参数要求"
  content: "压缩机排气量≥100m³/min，排气压力≥0.8MPa"
  category: "技术参数"        # 技术参数/安全/环保/能效/质量/验收/资质/知产/其他
  clause_type: "实质性条款"    # 实质性条款(★)/一般条款/参考性条款
  related_standard_hint: "GB/T 13279-2015"  # AI推断可能相关的标准
  risk_if_fail: "废标"         # 废标/扣分/无影响
  source_page: 15
  related_clauses: ["CL-005"]
```

### 执行指令

1. 读取用户提供的技术方案文档，提取全文文本+表格
2. 识别项目类型（EQUIP/ENG/SVC）
3. 提取所有技术条款，标注条款类型（★实质性/一般/参考）
4. 对★标记条款高亮`risk_if_fail: "废标"`
5. 为每条条款AI推断可能适用的国家标准

### 标准匹配执行流程

1. 根据阶段1识别的项目类型，加载对应标准知识库
2. 对每条条款进行关键词分析，匹配最相关标准
3. 联网搜索补充最新标准（通过WebSearch查询 `{条款关键词} + GB标准 最新版本`）
4. 输出标准匹配映射表：条款ID → 适用标准列表

## 阶段3：逐条对标审查

### 审查判定体系

对每条条款进行三级合规判定：

| 判定等级 | 符号 | 含义 | 处理建议 |
|----------|------|------|----------|
| **合规** | 🟢 PASS | 条款完全满足适用标准要求 | 无需处理 |
| **基本合规** | 🟡 PASS* | 满足实质性要求，存在微小偏差（如格式问题） | 建议完善 |
| **不确定** | 🟠 UNCERTAIN | 需进一步信息或专业判断才能确定 | 标注待确认项 |
| **不合规** | 🔴 FAIL | 明确不满足标准要求 | 必须整改 |
| **缺项** | ⚫ MISSING | 应包含但完全缺失的条款 | 必须补充 |
| **不适用** | ⚪ N/A | 该标准条款不适用于本项目 | 标注说明 |

### 审查记录格式

```yaml
review_record:
  review_id: "REV-001"
  clause_id: "CL-001"
  applicable_standards:
    - standard: "GB/T 13279-2015"
      section: "4.2.1"
      requirement: "公称容积流量偏差不超过±5%"
    - standard: "《招标文件》第三章"
      section: "3.1.2"
      requirement: "排气量≥100m³/min"
  compliance: "PASS"          # PASS / PASS* / UNCERTAIN / FAIL / MISSING / N/A
  compliance_score: 1.0       # 1.0=PASS, 0.8=PASS*, 0.5=UNCERTAIN, 0.2=FAIL, 0.0=MISSING, -1=N/A
  finding: "条款明确要求排气量≥100m³/min，方案中标注为105m³/min，满足要求。同时符合GB/T 13279-2015关于容积流量的偏差要求。"
  standard_quote: "GB/T 13279-2015 §4.2.1： '压缩机的公称容积流量……'"
  remediation: null            # 合规则无需整改
  severity: "info"             # info/warning/critical
  reviewer_note: ""
```

### 执行指令

1. 遍历所有条款（CL-xxx），逐一匹配适用标准
2. 对每条条款执行三级判定：
   - **Level 1 文本匹配**: 条款中的数值/要求与标准原文比对
   - **Level 2 语义分析**: 对非数值类条款进行语义合规分析
   - **Level 3 完整性检查**: 检测标准要求但方案未涵盖的缺项
3. 联网补充最新标准版本信息（如标准已废止或更新）
4. 生成逐条审查记录表

## 阶段4：9维合规评分

### 9大审查维度

```
                          ┌── 1. 技术参数符合度 (20%)
                          │    设备参数/性能指标/技术规格对标GB产品标准
                          │
                          ├── 2. 安全规范符合度 (15%)
                          │    安全生产/消防/特种设备/职业健康对标AQ安全标准
                          │
                          ├── 3. 环保要求符合度 (10%)
                          │    排放标准/废弃物处理/环评要求对标HJ环保标准
                          │
                          ├── 4. 能效标准符合度 (10%)
                          │    能耗指标/电机能效/系统效率对标GB能效强制性标准
                          │
    工业技术方案 ──┤
    合规审查              ├── 5. 质量管理体系 (10%)
                          │    ISO9000族/质量计划/检测手段/质保措施
                          │
                          ├── 6. 验收标准符合度 (10%)
                          │    验收条件/检测方法/验收程序对标GB验收规范
                          │
                          ├── 7. 强制性条款响应 (15%)
                          │    ★否决项逐条检查/废标风险标红预警
                          │
                          ├── 8. 资质与人员配置 (5%)
                          │    企业资质/人员证书/业绩要求硬性门槛
                          │
                          └── 9. 知识产权与合规 (5%)
                               专利合规/软件著作权/保密协议/数据安全
```

### 评分算法

```
维度合规率: R_dim = (PASS数 + PASS*数×0.8) / (条款总数 - N/A数)
综合合规率: R_total = Σ(R_dim × W_dim)

合规等级:
  ≥95%  → 🟢 优秀 — 可直接提交
  85-95% → 🟡 良好 — 建议小修后提交
  75-85% → 🟠 待改进 — 存在若干不合规项需整改
  60-75% → 🔴 不合格 — 多项不合规，需大幅整改
  <60%   → ⚫ 严重不合规 — 建议重写技术方案
```

### 输出内容

1. **合规总览卡片**: 综合合规率、合规等级、合规/不合规/缺项计数
2. **9维雷达图**: 各维度合规率可视化
3. **★否决项清单**: 所有实质性条款的逐条审查结果（红色高亮FAIL项）
4. **不合规排行榜**: 按严重程度排序的不合规项清单
5. **风险热力图**: 按章节/维度交叉的不合规密度热力分布

### 执行指令

1. 统计各维度的条款总数和合规数
2. 按加权算法计算综合合规率和各维度合规率
3. 调用 `read_me` 加载 `chart` 模块，通过 `show_widget` 生成合规雷达图
4. 标注所有`critical`级别的FAIL项（涉及废标风险）
5. 输出风险热力图数据（用于HTML报告）

## 阶段5：差距分析与整改建议

### 不合规项分析模型

对每个不合规项（FAIL/MISSING），生成结构化的差距分析：

```yaml
gap_analysis:
  gap_id: "GAP-001"
  review_id: "REV-015"
  clause_id: "CL-023"
  gap_type: "参数不达标"      # 参数不达标/标准缺失/格式不符/资质不足/其他
  current_state: "方案标注电机能效等级为IE3"
  required_state: "GB 18613-2020强制要求≥IE4（额定功率范围内）"
  gap_description: "电机选型低于国家强制性能效标准最低要求，属实质性不合格项。"
  impact: "废标风险 — 该条款为★否决项，不满足将导致投标无效。"
  remediation:
    priority: "P0"            # P0=必须立即整改(废标项), P1=高优先级(严重扣分), P2=中优先级(扣分), P3=低优先级(建议完善)
    action: "更换为IE4及以上能效等级电机，或提供豁免证明（如不在标准适用范围内）"
    reference: "GB 18613-2020 §4.2 表1"
    estimated_effort: "中等（需更换设备选型）"
  standard_quote: "GB 18613-2020 §4.2: '符合本文件能效等级……额定输出功率在0.75kW-375kW范围内的低压三相异步电动机，其能效等级不应低于IE4。'"
```

### 整改优先级

| 优先级 | 定义 | 时间要求 |
|--------|------|----------|
| **P0** | ★否决项不合规，必须立即整改，否则废标 | 投标截止前必须完成 |
| **P1** | 可能造成严重扣分的不合规项 | 投标截止前尽量完成 |
| **P2** | 一般不合规项，影响竞争力但不致命 | 下次投标前完成 |
| **P3** | 微小偏差/建议完善项 | 视情况处理 |

### 执行指令

1. 筛选所有合规判定为`FAIL`和`MISSING`的审查记录
2. 对每条不合规项分析差距类型（参数不达标/标准缺失/格式不符/资质不足/其他）
3. 评估影响等级（废标/严重扣分/一般扣分/微小偏差）
4. 生成具体可操作的整改建议
5. 引用标准原文支撑整改依据
6. 按P0→P1→P2→P3优先级排序输出整改清单

## 阶段6：合规审查报告生成

### 报告结构

生成的HTML交互式合规审查报告包含以下模块：

```
1. 📋 项目概览卡片
   - 项目名称、编号、类型（EQUIP/ENG/SVC）
   - 适用标准体系概览（标准清单）
   - 审查时间、审查范围说明

2. 🎯 合规总览仪表盘
   - 综合合规率（大号百分比仪表）
   - 合规等级（优秀/良好/待改进/不合格/严重不合规）
   - 关键统计：合规N条 / 基本合规N条 / 不合规N条 / 缺项N条 / 不确定N条

3. 📊 9维合规雷达图
   - 交互式雷达图（hover显示各维度详情）
   - 维度权重标注
   - 行业均值对比线（如有数据）

4. ⚠️ ★否决项专项报告
   - 所有实质性条款逐条审查结果
   - 红色高亮FAIL项
   - 废标风险评估

5. 📋 逐条审查清单
   - 可排序/筛选/搜索的完整审查表
   - 每条含：条款ID/章节/内容/适用标准/判定/发现/整改建议
   - 颜色编码：🟢合规 🟡基本合规 🟠不确定 🔴不合规 ⚫缺项

6. 🔥 风险热力图
   - 按章节×维度交叉的不合规密度热力分布
   - 可视化识别高风险区域

7. 🔧 整改建议清单
   - 按P0→P1→P2→P3优先级排序
```

## 注意事项

1. **AI审查结果为辅助参考，不替代专业法务/技术审核** — 关键条款建议人工复核
2. **标准版本时效性** — 审查时标注标准版本，建议确认是否为最新有效版本
3. **工业子领域差异** — 不同工业子领域适用标准体系不同，需根据项目类型准确匹配
4. **扫描件PDF需OCR** — 识别准确率受原文件质量影响
5. **联网搜索补充** — 对知识库未覆盖的标准，自动联网搜索最新版本
6. **合规判定边界** — 语义模糊处标记为UNCERTAIN，交由人工判断
7. **法律效力** — 本工具输出不构成法律意见，正式评审以评标委员会为准
