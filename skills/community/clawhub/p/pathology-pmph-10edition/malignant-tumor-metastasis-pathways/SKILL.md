# 恶性肿瘤转移的三种主要途径

在评估恶性肿瘤扩散风险或制定分期检查策略时，请按以下流程识别和应用三种主要转移途径：

## 1. 淋巴道转移（Lymphatic Metastasis）
- **适用对象**：最常见于上皮源性恶性肿瘤（癌）。
- **执行步骤**：
  1. 判断原发肿瘤是否已侵入淋巴管。
  2. 确定区域引流淋巴结（如乳腺癌→腋窝淋巴结）。
  3. 优先评估前哨淋巴结（sentinel lymph node）状态：若无转移，可避免广泛淋巴结清扫。
  4. 注意特殊模式：警惕跳跃式转移或逆行转移的可能性。
- **临床行动**：安排区域淋巴结超声、CT 或前哨淋巴结活检。

## 2. 血行转移（Hematogenous Metastasis）
- **适用对象**：肉瘤及晚期癌常经此途径转移。
- **执行步骤**：
  1. 根据原发部位推断血行转移路径：
     - 体循环静脉 → 肺（如骨肉瘤）
     - 门静脉系统 → 肝（如胃癌、结肠癌）
     - 肺静脉回流后 → 脑、骨、肾上腺等
     - 脊椎静脉丛 → 前列腺癌直接转移至脊椎
  2. 识别典型靶器官：肺和肝为最常见受累器官。
  3. 观察转移灶特征：“癌脐”（中央坏死下陷）、多发、边界清楚、近器官表面。
- **临床行动**：常规安排胸部及腹部增强CT/MRI以筛查肺、肝转移。

## 3. 种植性转移（Transcoelomic Metastasis）
- **适用对象**：胸腹腔器官肿瘤（如胃癌、卵巢癌）。
- **执行步骤**：
  1. 确认肿瘤已侵犯浆膜层。
  2. 评估是否存在体腔积液（血性或浆液性）。
  3. 对积液行细胞学检查，寻找脱落肿瘤细胞。
  4. 警惕典型表现：如双侧卵巢Krukenberg瘤（多源自胃印戒细胞癌）。
- **临床行动**：行腹腔/胸腔穿刺+细胞学检查；影像学关注大网膜、腹膜、卵巢等种植部位。

> **注意**：良性肿瘤不发生转移；基底细胞癌等低度恶性肿瘤极少转移。仅当满足“肿瘤具有侵袭性且突破基底膜或浆膜层”时，才考虑上述转移途径。
