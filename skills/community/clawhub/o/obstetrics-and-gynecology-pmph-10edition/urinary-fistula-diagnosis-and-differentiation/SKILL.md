# 尿瘘的诊断方法与鉴别流程

## 触发条件
IF 患者出现阴道无痛性持续流液 AND 漏出液经电解质/肌酐检测确认为尿液 THEN 启动本诊断流程。

## 执行步骤

### 1. 病史采集
- 重点询问分娩史（如难产、产程延长）、盆腔手术史（如子宫切除、肿瘤根治术）；
- 记录漏尿开始时间及表现特点（如产后3–7日提示坏死型，术后即刻提示直接损伤）。

### 2. 初步确认漏出液为尿液
- 比较漏出液与血清中肌酐浓度：尿液肌酐应为血清数倍；
- 若漏出液肌酐接近尿液水平，则支持尿瘘诊断。

### 3. 妇科检查
- 大瘘孔可直接肉眼观察；
- 小瘘孔需通过触诊寻找瘢痕组织；
- 若为盆腔术后且仅见尿液自阴道穹隆一侧流出，高度怀疑输尿管阴道瘘；
- 暴露困难时，取胸膝卧位，用单叶拉钩上提阴道后壁以显露上段瘘孔。

### 4. 辅助检查

#### a) 亚甲蓝试验
- 膀胱内注入300ml稀释亚甲蓝溶液；
- 阴道内置3个棉球（顶端、中1/3、远端）；
- 取出后观察染色位置：
  - 上段染色 → 膀胱阴道瘘；
  - 自宫颈口流出 → 膀胱子宫颈瘘或膀胱子宫瘘；
  - 无色或黄染 → 可能为输尿管阴道瘘；
- 若首次阴性但临床高度怀疑，可重置棉球后让患者走动30分钟再复查。

#### b) 靛胭脂试验
- 静脉注射靛胭脂5ml；
- 5–10分钟内见蓝色液体自阴道顶端流出 → 输尿管阴道瘘。

#### c) 膀胱镜/输尿管镜检查
- 评估膀胱容量、黏膜状态、有无炎症/结石/憩室；
- 明确瘘孔位置、大小、数目及其与膀胱三角区的关系；
- 必要时插入输尿管导管或行输尿管镜定位梗阻部位。

#### d) 影像学检查
- **静脉肾盂造影（IVP）**：动态观察肾盂、输尿管、膀胱显影，评估肾功能及输尿管通畅性；
- **逆行输尿管肾盂造影**：用于IVP未发现的输尿管阴道瘘；
- **CT尿路造影（CTU）**：非侵入性，清晰显示泌尿系统全貌，推荐作为新型诊断方法；
- **肾图**：评估肾及输尿管功能。

## 输出目标
明确尿瘘的类型（如膀胱阴道瘘、输尿管阴道瘘等）、解剖位置、大小、数目及可能病因，为后续手术修补提供依据。
