# 肠外营养宏量营养素配比规范

## 适用场景
当患者满足以下任一条件且需启动肠外营养（PN）支持时：
- 无法经胃肠道摄食的住院患者
- 肠功能衰竭者
- 需全胃肠外营养（PN）支持者

## 前提条件
执行前必须确认：
- 已确定患者体重（kg）
- 已评估代谢状态（是否处于应激/分解状态）
- 无严重高脂血症或高血糖禁忌

## 执行步骤

### 1. 确定葡萄糖（碳水化合物）剂量
- **常规剂量**：3～3.5 g/(kg·d)，供能约占总热量的50–60%
- **严重应激状态**（如脓毒症、大面积烧伤）：降至2～3 g/(kg·d)
- 葡萄糖提供4 kcal/g能量

### 2. 确定氨基酸（氮源）剂量
- **推荐剂量**：1.2～1.5 g/(kg·d)；严重分解代谢时可增至2.0 g/(kg·d)
- 供能占比约为总热量的15–20%
- 必需氨基酸与非必需氨基酸比例应为1:2
- 氮（g）:热量（kcal）推荐比约为1:150

### 3. 确定脂肪乳剂剂量
- **剂量范围**：甘油三酯0.7～1.3 g/(kg·d)
- **输注速度**：1.2～1.7 mg/(kg·min)
- 供能占比约为总热量的20–30%
- **禁忌**：血甘油三酯 > 4.6 mmol/L 时应减少或停用

### 4. 配制与输注原则
- 采用全营养混合液（TNA）方式，将所有成分混合后输注
- 禁止在PN液中添加其他药物，以防沉淀或污染

### 5. 监测计划
- 定期检测血糖、甘油三酯、肝功能、电解质
- 警惕再喂养综合征（尤其长期饥饿后）

## 注意事项
- 肝肾功能不全者需调整氨基酸种类与剂量
- 严重应激状态需下调葡萄糖剂量以避免代谢并发症
- 根据患者具体情况选择脂肪乳类型（长链、中/长链、橄榄油型或鱼油型）

## 参考文档

### 宏量营养素计算公式与参考值

**能量与氮含量换算**
- 葡萄糖：4 kcal/g
- 氨基酸：约4 kcal/g，含氮量≈16%
- 脂肪乳：约9 kcal/g（以甘油三酯计）

**推荐供能比例**
| 营养素 | 常规供能占比 | 应激状态调整 |
|--------|----------------|------------------|
| 碳水化合物 | 50–60% | 下调至40–50% |
| 脂肪 | 20–30% | 可适度上调 |
| 氨基酸 | 15–20% | 维持或略增 |

**氮热比（N:Cal）**
- 推荐范围：1 g 氮 : 100–150 kcal
- 重症或高分解状态：可放宽至1:100

**脂肪乳类型比较**
| 类型 | 代谢特点 | 适用人群 |
|------|----------|----------|
| 长链脂肪乳（LCT） | 依赖肉碱转运，可能抑制免疫 | 一般患者 |
| 中/长链脂肪乳（MCT/LCT） | 不依赖肉碱，代谢更快 | 肝功能受损者 |
| 橄榄油型 | 富含单不饱和脂肪酸，氧化稳定性好 | 长期PN患者 |
| 鱼油型（含ω-3） | 抗炎作用，改善肝功能 | ICU、术后、PNALD风险者 |

**再喂养综合征预警指标**
- 血磷 < 0.8 mmol/L
- 血钾 < 3.0 mmol/L
- 血镁 < 0.7 mmol/L
- 出现心律失常、呼吸衰竭、意识障碍
