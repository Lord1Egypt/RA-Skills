# 多模态内容摘要工具

对各类内容生成结构化摘要，快速提炼核心信息。

## Tools Required
- read

## Usage

- "帮我总结这篇文章的核心观点，用3-5个要点列出来"
- "这份会议记录太长了，帮我提炼出决议事项和待办"
- "把这个研究报告压缩成一页摘要，保留数据和结论"

## Examples

输入：`（粘贴一篇 3000 字的行业分析报告）帮我生成一份执行摘要`

输出：
- **核心结论**（2-3 句话）
- **关键数据**：市场规模 / 增速 / 主要玩家
- **机会与风险**：各 2-3 条
- **行动建议**：3 条可执行建议
