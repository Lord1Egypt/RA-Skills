# D07 法律领域任务目录

## 领域定义

法律领域(D07)关注法律风险防范、权益保护、纠纷解决和合规管理。核心驱动需求：N2安全。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人/企业法律风险管理 | 法学理论研究 |
| 合同全生命周期管理 | 立法政策制定 |
| 合规制度设计与执行 | 司法审判工作 |
| 知识产权保护与管理 | 法律教育出版 |
| 公司法律事务处理 | 法律科技产品开发 |
| 法律纠纷解决 | 国际法律条约制定 |
| 财产传承规划 | 法律援助机构运营 |
