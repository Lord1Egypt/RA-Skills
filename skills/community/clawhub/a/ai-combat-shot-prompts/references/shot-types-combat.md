# 战斗镜头类型详解

## 蓄力 → 发力 → 受力 → 反馈 四段式

### 蓄力（0.3s）
```
肌肉紧绷：肱二头肌隆起，肌腱可见
重心下沉：髋关节微屈，重量分布后移
武器后拉：手臂后展，角度30-45度
呼吸屏住：胸腔固定，气息内敛
```

### 发力（0.1s）
```
力量传递：腰胯驱动肩→臂→腕
武器加速：末端速度最高，轨迹确定
空气挤压：高速产生可视化气流
能量聚焦：力量汇聚于接触点
```

### 受力（0.05s）
```
接触瞬间：能量爆发，火花/烟尘/液体飞溅
冲击传导：沿骨骼传递，产生次级运动
平衡打破：重心失稳，身体开始位移
```

### 反馈（0.5s+）
```
身体旋转：借冲击力调整姿态
重心转移：从失衡到重建
武器回收：准备下一击或防御
表情变化：痛苦/愤怒/惊讶
```

## 战斗镜头机位详解

### 低角度仰拍（强烈推荐）
```
机位：地面或低于脚面
角度：仰角15-45度
效果：人物高大威压，敌人显渺小
适用：强者登场、必杀技、威压感
```

### 越肩镜头
```
前景：出招者肩膀/手臂局部（占画面30%）
中景：武器轨迹/目标
后景：对手反应
效果：临场感，紧张，代入感强
```

### 俯冲镜头
```
从上往下拍，镜头随动作下移
适用：轻功下劈、跳杀、俯冲攻击
注意：保持运动轨迹清晰
```

### 高速跟随镜头
```
相机与武器/拳头同速移动
焦点在接触点，周围动态模糊
效果：速度感，冲击感
```

### POV主观镜头
```
从挨打者视角
模糊边缘，焦点不稳
呼吸音/心跳声强化临场感
```

## 节奏模式：快慢结合

### 标准战斗节奏
```
快 → 慢 → 快 → 慢
0.1s打 → 0.3s停 → 0.1s打 → 0.5s停
```

### 情绪节奏对照
| 节奏 | 情绪 |
|------|------|
| 快快快（连续快攻） | 愤怒、拼命、压制 |
| 快慢（攻停攻停） | 从容、掌控、压迫 |
| 慢慢快（蓄力爆发） | 隐忍、爆发、逆转 |
| 慢慢慢快（最后终结） | 高潮、终结、释放 |

## 镜头组合序列示例

### 场景：剑士斩杀

**镜头1：远景建立**
```
低角仰拍，战士全身入画，剑横于胸前
侧逆光，轮廓光突出，背景简洁
```
**镜头2：近景蓄力**
```
特写握剑的手，关节发白
手腕内旋，剑尖微沉
```
**镜头3：发力特写**
```
越肩，清晰看到剑身斜向下45度切入
金属碰撞火花飞溅
```
**镜头4：中景受力**
```
对方身体被剑势带动向右旋转
脚步踉跄，重心失控
```
**镜头5：全景终势**
```
战士站定，剑尖下垂
对手倒地或跪伏
```

## 机位与轴线

### 正反打战斗
```
A攻击B：A在左，镜头从左拍B的反应
B攻击A：B在右，镜头从右拍A的反应
保持180度轴线
```

### 多对一团战
```
主机位：围圈外侧
次机位：高处俯拍
第三机位：低角仰拍被围攻者
```
