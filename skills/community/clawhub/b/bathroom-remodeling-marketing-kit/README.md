**Skill #202 | Compliance-Moated Marketing Copy for Bathroom Remodelers**

Stop using AI tools that write marketing copy with federal violations baked in.

## 7 Compliance Gates — Why They Matter

| # | Gate | What Generic AI Writes | What This Skill Writes |
|---|------|------------------------|------------------------|
| 1 | NRS 624 license scope | "We do everything in-house — plumbing, electrical, tile" | B-2 # disclosed + C-1/C-2 sub license #s required |
| 2 | EPA RRP (40 CFR Part 745) | No mention of lead risk in pre-1978 homes | EPA RRP Firm # + Certified Renovator name required for pre-1978 targeting |
| 3 | Clark County permit gate | "No permit needed — fast turnaround" | All permit-pulled work disclosed; "no permit" language permanently blocked |
| 4 | ADA scope accuracy | "ADA compliant bathroom renovation" | ANSI A117.1 aging-in-place framing; ADA Title III = commercial only |
| 5 | Mold/moisture scope boundary | "We'll handle any mold we find" | IICRC S520 scope separation required; in-scope mold language permanently blocked |
| 6 | Structural/load-bearing wall | "We can remove any wall" | PE stamp requirement disclosed; unrestricted wall removal language blocked |
| 7 | Warranty/FTC/TCPA | "Lifetime waterproofing guarantee" | Magnuson-Moss §104 terms required; source-attributed ROI claims only |
