# DM — 数据管理（CMMI V3.0 Domain-specific PA）

## PA 基本信息
- **缩写**：DM
- **英文**：Data Management
- **中文**：数据管理
- **所属**：Domain-specific PA（领域特定实践域）
- **适用领域**：数据管理、信息服务

---

## CMMI V3.0 要求要点

### 核心目的
管理数据资产，确保数据的可用性、完整性、安全性和可访问性。

### 关键实践（Specific Practices）
| 实践编号 | 实践内容 | 说明 |
|---------|---------|------|
| DM 1.1 | 制定数据管理策略 | 定义数据治理框架、政策、标准 |
| DM 1.2 | 识别数据资产 | 数据盘点、数据分类、数据所有权 |
| DM 1.3 | 建立数据质量标准 | 定义数据质量维度（准确性、完整性等） |
| DM 1.4 | 管理数据生命周期 | 数据创建、存储、归档、销毁 |
| DM 2.1 | 实施数据管理 | 执行数据管理计划 |
| DM 2.2 | 监控数据质量 | 数据质量测量、问题识别 |
| DM 3.1 | 持续改进数据管理 | 基于反馈优化数据管理实践 |

### 通用实践（Generic Practices）
- GP 1.1 执行特定实践
- GP 2.1 建立组织政策
- GP 2.2 策划过程
- GP 2.3 提供资源
- GP 2.4 分配责任
- GP 2.5 培训人员
- GP 2.6 管理配置
- GP 2.7 识别并记录干系人需求和接口
- GP 2.8 监督和控制过程
- GP 2.9 客观评估符合性
- GP 2.10 高层审查

## 用户文档 → PA 匹配规则

### 高概率匹配（主要PA）
- 文档名含"数据管理"、"Data Management"、"数据治理" → **DM 主要匹配**
- 文档含数据资产清单、数据质量标准 → **DM 主要匹配**

### 中概率匹配（次要PA）
- 文档含数据安全策略 → 同时匹配 **ESEC**（网络安全工程）
- 文档含数据质量报告 → 同时匹配 **DQ**（数据质量）

## 评估访谈常见问题（模式D）

### 针对数据管理人员的问题
1. "你们组织有哪些数据资产？如何管理的？"
2. "数据质量如何保证？有测量方法吗？"
3. "数据生命周期是如何管理的？有归档和销毁策略吗？"
4. "数据安全问题如何处理的？"

### 评分要点
- **DM 1.1-1.4**：是否有系统的数据管理策略和标准
- **DM 2.1-2.2**：数据管理是否实施，是否有质量监控
- **DM 3.1**：是否有持续改进数据管理的机制
