# 模板与标杆规范差异报告

## 当前模板 vs 标杆规范对比

### 1. 视觉布局差异

#### 字体规范差异
- **当前模板**:
  - 标题字体大小: 72px (title.size)
  - 副标题字体大小: 56px (subtitle.size)
  - 标题权重: 700
  - 正文字体大小: 42px (body.size)

- **标杆规范 (lesson_01)**:
  - 标题字体大小: 56px
  - 副标题字体大小: 32px
  - 标题权重: 800
  - 特殊文本阴影: "0 4px 20px rgba(0,0,0,0.3)"

#### 颜色规范差异
- **当前模板**:
  - 背景: linear-gradient(135deg, #1a1a2e 0%, #16213e 100%)
  - 主要文字颜色: #ffffff
  - 次要文字颜色: #e2e8f0

- **标杆规范 (lesson_01)**:
  - 背景: linear-gradient(160deg, #0f0c29 0%, #1a1145 30%, #302b63 60%, #24243e 100%)
  - 特定副标题颜色: rgba(255,255,255,0.6)

#### 间距与布局差异
- **当前模板**:
  - 安全区域: top/bottom 120px, left/right 60px
  - 行高: body.lineHeight = 1.6

- **标杆规范 (lesson_01)**:
  - 内边距: '0 60px' (水平方向)
  - 特定边距: marginTop: 24

### 2. 动态效果差异

#### 插值动画差异
- **当前模板**:
  - 淡入动画: [0, 0.5 * fps] -> [0, 1]
  - 位移动画: [0, 0.5 * fps] -> [50, 0]

- **标杆规范 (industry_medical)**:
  - 淡入动画: [0, fps * 0.5] -> [0, 1]
  - 位移动画: [0, fps * 0.8] -> [60, 0]
  - 子标题淡入: [fps * 0.3, fps * 0.8] -> [0, 1]
  - 缩放脉冲动画: [0, fps * 2] -> [1, 1.08]

#### 特效差异
- **当前模板**:
  - 无特殊动画效果
  - 使用基础渐变背景

- **标杆规范 (industry_medical)**:
  - Bokeh背景效果 (模糊光斑动画)
  - 脉冲缩放动画
  - 多层装饰元素动画
  - 文字阴影特效 (0 4px 30px rgba(99,102,241,0.5))

### 3. 组件类型差异

#### 当前模板组件
- CoverSlide: 封面幻灯片
- EndingSlide: 结尾幻灯片
- SlideContent: 内容幻灯片
- BackCover: 背面封面
- BotText: 底部文本

#### 标杆规范组件 (lesson_01 + industry_medical)
- 覆盖多种幻灯片类型:
  - cover (封面)
  - hook (钩子)
  - content_text (文本内容)
  - content_story (故事内容)
  - content_compare (对比内容)
  - summary (总结)
  - ending (结尾)

### 4. 总结

当前模板与标杆规范的主要差异在于:
1. 字体规格不同 (大小、权重)
2. 颜色系统不同 (背景渐变、文字颜色)
3. 动画效果复杂度不同 (标杆具有更丰富的动态效果)
4. 组件类型不够丰富 (缺少多种内容类型的幻灯片)

需要重构当前模板以对齐标杆规范。
