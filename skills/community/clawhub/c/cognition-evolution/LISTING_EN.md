# Cognition Evolution System

**Auto-Review · Lesson Deposition · Principle Accumulation · Learn from Every Conversation**

---

## Elevator Pitch

Every conversation ends with an automatic debrief. Lessons learned are captured as actionable principles — so you never make the same mistake twice.

---

## What It Does

### Core Features

- **Auto-Review**: Analyzes each conversation automatically at session end
- **Lesson Deposition**: Turns "mistakes made" into "do this instead next time"
- **Open-Issue Tagging**: Flags unresolved decisions clearly for later
- **Cross-Session Evolution**: Understanding compounds over time

### Problems It Solves

❌ "Made the same mistake again — should have remembered"
❌ "2-hour discussion, conclusions scattered everywhere"
❌ "Team member left, context lost"
❌ "AI did a bunch of work but nothing stuck"

✅ "The reasoning behind each decision is preserved"
✅ "Lessons section grows: when facing XX, do YY not ZZ"

### Output Example

```markdown
## Cognition Deposit 2026-05-03

### Decision
- Chose PostgreSQL over MongoDB ← due to high transaction needs, team familiarity

### Lesson
- For tech decisions, analyze business requirements first, popularity second

### Insight
- RDBMS query performance advantages are frequently underestimated

### Unresolved
- Deployment plan TBD ← waiting for ops team server confirmation
```
