## Description: <br>
Matlab Bridge lets an agent call local MATLAB to run simulations, analyze data, execute .m scripts, exchange data, and generate plots. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[jirboy](https://clawhub.ai/user/jirboy) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, engineers, and researchers use this skill to route scientific computing, signal processing, structural analysis, data conversion, and plotting tasks through an installed local MATLAB environment. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill enables agent-driven local MATLAB code and .m script execution. <br>
Mitigation: Install only when local MATLAB automation is intended, and run trusted or reviewed MATLAB code. <br>
Risk: Generated output and workspace files may contain private or sensitive data. <br>
Mitigation: Review saved MATLAB output directories periodically and delete files that should not persist. <br>
