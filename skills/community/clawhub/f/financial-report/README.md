# 📊 企业财务报表分析与预测工具 - SkillHub 发布包

## 技能简介

专业的企业财务分析工具，支持：
- 📝 **三大报表录入**：利润表、资产负债表、现金流量表
- 📈 **自动指标计算**：毛利率、净利率、ROE、资产负债率、流动比率等
- 📉 **趋势分析**：营收/净利同比增长率
- 🎨 **可视化图表**：收入利润趋势、盈利能力、资产负债结构、现金流分析
- 📥 **示例数据**：内置贵州茅台 2021-2024 年数据
- 🔮 **业绩预测**：基于历史数据的乐观/中性/悲观三情景预测

## 发布到 SkillHub.CN

### 方式一：Web 界面上传
1. 访问 https://skillhub.cn
2. 登录账号
3. 点击"发布技能"
4. 上传 `financial-analyzer-tool.zip`
5. 填写技能信息并提交审核

### 方式二：CLI 发布
```bash
# 安装 CLI（如果还没安装）
curl -fsSL https://skillhub.cn/cli | sh

# 发布技能
skillhub publish ./financial-analyzer-tool.zip
```

## 触发场景
用户提到以下关键词时自动触发：
- 财务分析、看财报、分析某公司财务状况
- 制作财务报告、财务报表可视化
- 分析企业盈利能力、成长性、健康度
- 业绩预测、企业估值

## 不适用场景
- 股票买卖建议
- 实时行情查询
- 会计记账
