# 角色设计指南

## 核心原则：每个角色都是主角

剧本杀与小说最大的区别：没有配角。每个拿到剧本的玩家都要有完整的体验。

## 角色设计五要素

### 1. 身份标签（外在）

一句话能让其他玩家记住的身份。

| 类型 | 示例 |
|------|------|
| 职业身份 | 法医、律师、船长、管家 |
| 家族身份 | 长女、私生子、养子、童养媳 |
| 社会身份 | 嫌疑人、证人、受害者家属、侦探 |
| 隐藏身份 | 双面间谍、复仇者、真凶、卧底 |

### 2. 性格内核（内在）

用3个关键词定义，必须有矛盾点。

**好性格设计**：
- 表面温柔，内心冷酷
- 看似玩世不恭，实则重情重义
- 外表强势，内心脆弱

**坏性格设计**：
- 好人（太扁平）
- 坏人（太标签化）
- 没有性格（玩家不知道怎么演）

### 3. 核心动机（驱动）

每个角色必须有一个"必须达成的目标"。

动机类型库：

| 动机类型 | 示例 |
|----------|------|
| 复仇 | 为死去的亲人报仇 |
| 保护 | 保护爱人/孩子的秘密 |
| 贪婪 | 获取遗产/宝藏/地位 |
| 嫉妒 | 无法忍受对方拥有自己没有的东西 |
| 恐惧 | 如果秘密暴露，自己会身败名裂 |
| 爱情 | 为爱可以做任何事 |
| 正义 | 相信某人有罪，要亲自执行正义 |
| 自保 | 被迫杀人以保护自己 |

### 4. 秘密与谎言

每个角色必须有：
- **1个核心秘密**（不能让所有人知道）
- **1个谎言**（对其他玩家说的假话）
- **1个弱点**（可能被利用的把柄）

### 5. 人物关系网

每两个角色之间必须有至少一种关系。

关系类型：

| 关系 | 戏剧张力 |
|------|----------|
| 恋人/夫妻 | 高（情感纠葛） |
| 仇人 | 高（直接冲突） |
| 旧友 | 中（信任与背叛） |
| 上下级 | 中（权力不对等） |
| 陌生人 | 低（需要额外设计连接点） |
| 亲人 | 高（血缘与背叛） |
| 师徒 | 中（传承与反叛） |

## 角色平衡原则

### 戏份均等

检查方法：
1. 每个角色的剧本字数相差不超过20%
2. 每个角色至少有3个关键剧情点
3. 每个角色至少与2个其他角色有直接关系

### 嫌疑均等

凶手之外的角色也需要有秘密——否则"没有秘密的人"反而最可疑。

设计技巧：
- 给每个角色一个"看起来像凶手"的特征
- 给每个角色一个"必须隐瞒"的理由
- 让每个角色在某个时间段都没有不在场证明

### 信息均等

每个角色掌握的关键信息量应该相近。

信息分配原则：
- 凶手知道自己的手法，但不知道别人的秘密
- 普通玩家知道自己的秘密，但不知道凶手的手法
- 关键线索分散在3个以上角色手中

## 角色卡模板

```markdown
# 角色名

## 基础信息
- 姓名：
- 年龄：
- 性别：
- 职业：
- 与死者关系：

## 性格关键词
1. （外在表现）
2. （内在真实）
3. （矛盾点）

## 背景故事
（200-300字，包含关键经历和情感触发点）

## 核心动机
（你必须达成的目标）

## 核心秘密
（不能让所有人知道的事）

## 你的谎言
（如果被问到XX，你要这样说）

## 时间线
- 案发前3小时：...
- 案发前2小时：...
- 案发前1小时：...
- 案发时：...
- 案发后：...

## 你知道的线索
1. ...
2. ...

## 你的任务
1. 主线：找出真相/隐瞒秘密
2. 支线：与XX达成和解/获取XX的信任
```

## 经典角色设计案例

### 案例1：《年轮》式——情感驱动型角色

```
角色：姚波
表面身份：眼神冰冷、少言寡语的消瘦男人
真实身份：（涉及剧透）
核心矛盾：理性与情感的撕裂
设计亮点：玩家在推理过程中会逐渐理解角色的苦衷
```

### 案例2：《漓川》式——凶手角色设计

```
角色：凶手位
特殊设计：第一幕就被告知"你是所有案件的凶手"
但：玩家需要通过交流来推测自己的作案手法
设计亮点：凶手玩家也有推理任务，不是单纯隐瞒
```

### 案例3：《刀鞘》式——阵营型角色

```
角色：谍战人员
设计特点：
- 每个人都有双重身份
- 信任是稀缺资源
- 选择决定结局（多结局）
```
