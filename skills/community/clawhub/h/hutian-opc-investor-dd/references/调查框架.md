# 投资者背景调查框架（详细版）

## 一、机构基本信息调查维度

### 1.1 基础工商信息

| 调查项 | 具体内容 | 获取方式 | 评估标准 |
|-------|---------|---------|---------|
| **公司全称** | 与营业执照一致 | 工商查询 | 确认主体资质 |
| **注册资本** | 认缴/实缴金额 | 工商查询 | 评估资金实力 |
| **成立时间** | 注册日期 | 工商查询 | >5年为优 |
| **经营状态** | 在业/存续/注销 | 工商查询 | 确认存续 |
| **法定代表人** | 法人代表姓名 | 工商查询 | 核对身份 |
| **注册地址** | 实际办公地址 | 工商查询 | 确认核心区域 |
| **经营范围** | 投资/资产管理等 | 工商查询 | 确认合规 |

### 1.2 基金备案信息

| 调查项 | 具体内容 | 获取方式 | 评估标准 |
|-------|---------|---------|---------|
| **管理人登记编号** | 中基协备案编号 | 中基协官网 | 必须已备案 |
| **备案时间** | 首次备案日期 | 中基协官网 | 评估历史 |
| **管理规模** | 在管基金规模 | 中基协/LP报告 | 匹配需求 |
| **产品数量** | 在管基金数量 | 中基协官网 | 评估专业度 |
| **合规记录** | 是否有过处罚 | 中基协公告 | 无处罚为优 |
| **异常经营** | 是否被列入异常 | 中基协官网 | 确认合规 |

### 1.3 股权结构与LP构成

| 调查项 | 具体内容 | 评估重点 |
|-------|---------|---------|
| **股东背景** | 国资/民营/混合 | 资源来源 |
| **LP构成** | 政府/上市公司/高净值/家族办公室 | 资金稳定性 |
| **实控人** | 最终受益人 | 决策稳定性 |
| **关联公司** | 兄弟公司/母公司 | 资源协同 |

## 二、投资风格画像框架

### 2.1 阶段偏好评估矩阵

| 投资阶段 | 典型特征 | 适用项目 | BNCT项目适配性 |
|---------|---------|---------|---------------|
| **种子轮** | 概念验证、团队组建 | TRL 1-2 | ⭐⭐⭐⭐⭐ 极早期 |
| **天使轮** | 早期验证、核心团队 | TRL 2-3 | ⭐⭐⭐⭐⭐ |
| **Pre-A轮** | 模式验证、小规模试运营 | TRL 3-4 | ⭐⭐⭐⭐ |
| **A轮** | 规模化验证、运营体系建立 | TRL 4-5 | ⭐⭐⭐ |
| **B轮及以后** | 快速扩张、精细化运营 | TRL 6+ | ⭐⭐ |

### 2.2 行业偏好评估

| 评估维度 | 评估内容 | 判断方法 |
|---------|---------|---------|
| **专注程度** | 是否专注单一赛道 | 查看投资组合集中度 |
| **赛道深度** | 在该赛道的资源积累 | 产业链布局深度 |
| **投资主题** | 当前重点关注方向 | 近期投资项目主题 |
| **热点敏感度** | 对新兴赛道的反应速度 | 首批投资项目时间 |

### 2.3 投资行为特征

| 行为特征 | 正面信号 | 负面信号 |
|---------|---------|---------|
| **出手速度** | 快速决策、快速打款 | 拖沓、反复 |
| **投资金额** | 符合项目需求 | 过度干预条款 |
| **领投/跟投** | 偏好领投、有主导权 | 纯跟投、占小股 |
| **连续投资** | 愿意连续跟投 | 一次性投资 |
| **资源投入** | 积极投后赋能 | 纯财务投资 |

## 三、已投案例分析方法

### 3.1 案例筛选标准

**重点关注的案例类型：**
1. 与本项目赛道高度相关（核医学/BNCT/创新药）
2. 投资阶段相近（种子/天使轮）
3. 投资时间较近（近3年内）
4. 有公开信息可查（便于核实）

**排除的案例类型：**
1. 过于早期的概念性投资（无法评估）
2. 时间过于久远的案例（参考价值低）
3. 无法核实真实性的案例

### 3.3 退出案例分析

| 退出方式 | 评估重点 | 评估标准 |
|---------|---------|---------|
| **IPO退出** | 上市地点、锁定期、估值增长 | IRR通常最高 |
| **并购退出** | 并购方背景、交易估值、整合情况 | 看整合效果 |
| **股权转让** | 接盘方背景、转让估值 | 评估流动性 |
| **清算退出** | 清算比例、投资回收 | 通常损失较大 |

## 四、决策流程评估框架

### 4.1 决策周期评估

| 阶段 | 正常周期 | 过长信号 | 评估结论 |
|-----|---------|---------|---------|
| 初次接触→内核 | 1-2周 | >1个月 | ⚠️ 兴趣不足 |
| 内核→立项 | 1-2周 | >1个月 | ⚠️ 流程复杂 |
| 立项→尽调 | 2-4周 | >2个月 | ⚠️ 决策链长 |
| 尽调→投决 | 2-4周 | >2个月 | ⚠️ 风险控制过严 |
| 投决→打款 | 1-2周 | >1个月 | ⚠️ LP沟通问题 |

### 4.2 决策链分析

**关键决策人识别：**
1. **合伙人（GP）**：最终决策权
2. **MD/ED**：项目负责人，主导推进
3. **VP/Associate**：执行层，撰写报告
4. **Analyst**：初步筛选，信息收集

**决策权重分布：**
- 合伙人主导型：合伙人个人喜好影响大
- 团队决策型：MD+VP+Analyst共同评估
- 投决会主导型：委员会投票决定

### 4.3 条款风格评估

| 条款类型 | 激进风格（负面） | 温和风格（正面） |
|---------|---------------|----------------|
| **估值调整** | 严格对赌、频繁调整 | 宽松条件、阶段性调整 |
| **反稀释** | 完全加权平均 | 加权平均或豁免 |
| **回购条款** | 即时触发、高比例 | 延期触发、合理比例 |
| **董事会** | 多数席位、控制权 | 观察席位、少数席位 |
| **一票否决** | 多项事项否决 | 仅核心事项否决 |
| **竞业禁止** | 严格限制 | 宽松范围 |

## 五、行业口碑调研方法

### 5.1 公开信息调研

**搜索引擎关键词组合：**
```
[机构名称] + 投诉
[机构名称] + 纠纷
[机构名称] + 诉讼
[机构名称] + 创始人 + 争议
[机构名称] + 投资人 + 负面
[合伙人名字] + 投诉
```

**重点信息来源：**
1. 裁判文书网 - 诉讼记录
2. 财经媒体 - 负面报道
3. 社交媒体 - 创业者评价
4. 论坛社区 - 匿名爆料

### 5.2 人脉打听模板

**询问对象：**
- FA机构（了解机构在业内的口碑）
- 已投企业创始人（第一手合作体验）
- 同行投资人（了解机构风格）
- 律师/会计师（了解机构声誉）

**询问话术：**
```
我们正在和[机构名称]谈融资，想了解一下这家机构在业内
的口碑怎么样？有没有什么需要注意的地方？

如果方便的话，能说说：
1. 他们投后服务怎么样？
2. 条款上有没有什么坑？
3. 决策团队靠不靠谱？
```

### 5.3 创业者反馈收集

**正面信号：**
- ✅ 资金到账快、决策效率高
- ✅ 投后资源丰富、真正赋能
- ✅ 条款合理、沟通顺畅
- ✅ 愿意长期陪伴、支持后续融资

**负面信号：**
- ❌ 资金拖延、打款条件苛刻
- ❌ 口头承诺不兑现、资源诈欺
- ❌ 条款严苛、设置陷阱
- ❌ 干预经营、越权管理
- ❌ 过河拆桥、损害创业者

## 六、综合评估模型

### 6.1 评分体系

| 评估维度 | 权重 | 评分标准 |
|---------|-----|---------|
| **赛道匹配度** | 25% | 高度相关4-5分，中度相关2-3分，低相关0-1分 |
| **阶段匹配度** | 25% | 完全匹配4-5分，部分匹配2-3分，不匹配0-1分 |
| **资金实力** | 15% | 规模大/资金足5分，规模中3分，规模小1分 |
| **增值服务** | 15% | 资源丰富4-5分，一般2-3分，无资源0-1分 |
| **决策效率** | 10% | 快速决策4-5分，中等2-3分，拖沓0-1分 |
| **行业口碑** | 10% | 口碑好4-5分，一般2-3分，口碑差0-1分 |

### 6.2 风险等级划分

| 风险等级 | 综合评分 | 行动建议 |
|---------|---------|---------|
| **A级（推荐）** | 4.0-5.0 | 优先对接，重点跟进 |
| **B级（备选）** | 3.0-3.9 | 可作为备选，保持联系 |
| **C级（谨慎）** | 2.0-2.9 | 需进一步核实，谨慎对待 |
| **D级（规避）** | <2.0 | 建议规避，或做充分尽调 |

### 6.3 决策矩阵

| | 高口碑 | 低口碑 |
|---|-------|-------|
| **高匹配** | ✅ 重点推进 | ⚠️ 核实后推进 |
| **低匹配** | ⚠️ 备选跟进 | ❌ 建议规避 |

## 七、特殊风险识别

### 7.1 "毒药钱"特征

| 风险类型 | 特征识别 | 应对策略 |
|---------|---------|---------|
| **估值陷阱** | 故意压低估值、设置严苛调整条款 | 多家比价、拒绝单一家 |
| **条款剥削** | 反稀释、回购、一票否决过度 | 寻求法律意见 |
| **资源诈欺** | 口头承诺资源不兑现 | 要求书面承诺、现资源 |
| **过河拆桥** | 早期承诺、后期干预经营 | 设置保护条款 |
| **利益冲突** | 已投竞品、关联交易 | 核实投资组合 |
| **资金断链** | LP资金紧张、募资困难 | 核实LP背景 |

### 7.2 国资基金特殊考量

| 考量维度 | 评估要点 |
|---------|---------|
| **审批流程** | 决策链长、需多级审批 |
| **返投要求** | 地方政府基金通常要求返投 |
| **考核周期** | 国资考核周期与市场不同 |
| **灵活度** | 条款灵活性可能受限 |
| **资源优势** | 政府资源、政策支持 |

### 7.3 CVC（产业资本）特殊考量

| 考量维度 | 评估要点 |
|---------|---------|
| **战略协同** | 是否与自身业务高度协同 |
| **独立性** | 是否能保持独立发展 |
| **竞争风险** | 股东/LP是否存在竞争关系 |
| **技术泄露** | 是否存在技术/商业秘密风险 |
| **整合意图** | 是否有后续并购意图 |
