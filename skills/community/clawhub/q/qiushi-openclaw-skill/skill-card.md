## Description: <br>
求是 OpenClaw Skills 是基于毛泽东思想方法论的 AI Agent 技能集合，按照 OpenClaw 技能格式进行改造，旨在为 AI 提供系统化的思考和行动框架。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent users use this Chinese-language skill pack to guide planning, prioritization, investigation, review, and multi-skill workflow selection with a Mao-inspired methodology framework. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The broad arming-thought trigger can influence planning, prioritization, review, and skill routing at the start of new top-level conversations. <br>
Mitigation: Install only where this Chinese-language Mao-inspired reasoning framework is desired, and review the automatic trigger behavior before enabling the pack. <br>
