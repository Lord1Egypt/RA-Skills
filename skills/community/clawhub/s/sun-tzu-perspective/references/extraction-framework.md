# 心智模型提取方法论

## 三重验证标准

一个真正的「心智模型」（Mental Model）必须通过以下三重验证：

### 1. 跨域复现 (Cross-Domain Consistency)
- **标准**：该核心论点在至少2个不同领域/话题中出现
- **检查方法**：
  - 这个人在讨论战争时用了这个框架，在讨论商业/政治/人生时也用同样的框架？
  - 还是话题一变，思维方式就完全不同？
  - **通过**：显示出一种跨场景的稳定思考模式
  - **失败**：只在单个领域出现，属于「专业知识」而非「心智模型」

### 2. 生成力 (Generative Power)
- **标准**：能据此推断此人对新问题（未公开表态过）的立场
- **检查方法**：
  - 给这个人一个他从未讨论过的场景（如「如果用孙子兵法指导现代软件开发」）
  - 是否能基于已提取的模型做出合理推断？
  - **通过**：模型能作为推理引擎，生成新见解
  - **失败**：只能解释过去说过的话，无法推广

### 3. 排他性 (Exclusivity)
- **标准**：不是所有聪明人都这样想，有独特的视角
- **检查方法**：
  - 这个观点换一个人（同类专家）会不会也说一样的话？
  - **通过**：体现此人的独特性、反直觉性、或特定文化背景
  - **失败**：是通用智慧或陈词滥调（如「要努力」「要诚实」）

## 六层信息提取结构

### Layer 1: 心智模型（3-7个）
**定义**：此人看待世界的基本镜片，重复出现的核心分析框架

**提取标志**：
- 用了多次的自创术语或概念框架
- 跨不同话题都出现的分析角度
- 被同行引用或批判的核心理论

**格式**：
```markdown
#### 模型名称
- **一句话**：此人用这个模型看什么
- **来源证据**：出现在著作第X章、访谈第Y次...
- **应用方式**：遇到这类问题时，他会怎么做
- **局限性**：什么情况下这个模型会失效
```

### Layer 2: 决策启发式（5-10条）
**定义**：快速判断的规则，可表述为「如果X，则Y」

**提取标志**：
- 「我一般会...」「从来不做...」「一旦发现...就...」
- 具体场景下的行为模式
- 事后验证有效的判断习惯

**格式**：
```markdown
- **场景**：[什么情况下使用]
- **规则**：[具体的if-then]
- **案例**：[真实发生过的例子]
```

### Layer 3: 表达DNA
**定义**：此人说话的独特风格，构成可识别的「语气指纹」

**分析维度**：
| 维度 | 分析内容 | 关键词提取方法 |
|------|---------|--------------|
| 句式偏好 | 长句/短句、疑问/陈述、类比密度 | 统计平均句长、疑问句占比 |
| 词汇特征 | 高频词、专属术语、禁忌词 | 词频分析（排除停用词） |
| 节奏感 | 先结论还是先铺垫、转折方式 | 分析段落结构 |
| 幽默方式 | 讽刺/自嘲/荒诞/冷幽默/不幽默 | 收集幽默表达实例 |
| 确定性表达 | 「我不确定」型 vs 「很明显」型 | 统计不确定性短语 |
| 引用习惯 | 爱引谁、什么类型来源 | 统计引用对象 |

### Layer 4: 价值观与反模式
**定义**：此人推崇什么、反对什么

**价值观**（3-5条核心排序）：
- 他一生在追求什么？
- 什么对他来说绝对重要？
- 这些价值之间的优先级？

**反模式**（明确反对的行为）：
- 「千万不要...」「我从不...」「最看不惯...」

**内在张力**：
- 价值观之间是否有矛盾？
- 例如：「追求效率」vs「尊重传统」

### Layer 5: 智识谱系
**定义**：思想传承地图

```markdown
- **影响他的人**：受谁启发（老师、前辈、竞争对手）
- **他影响的人**：哪些人公开受他影响
- **思想位置**：在所属领域的地图上，他处在哪个派别？与主流观点距离多远？
```

### Layer 6: 诚实边界
**定义**：这个Skill做不到什么（必须诚实写出）

**必须包含**：
- 无法预测面对全新问题的反应（只能基于已有框架推断）
- 不能替代本人的创造力和直觉
- 公开表达 vs 真实想法可能有差距
- 信息截止到调研时间点（对于历史人物，材料有限性）
- 二手来源的推断不确定性

## 判断真伪的信号

### 标记为「真信念」的信号
- ✅ 在不同著作/访谈中重复出现≥3次
- ✅ 用这个框架分析过至少2个不同领域的问题
- ✅ 有实际案例/决策支撑
- ✅ 被同行引用（无论是支持还是反对）

### 标记为「陈词滥调」的信号
- ❌ 只出现过1次，没有后续展开
- ❌ 换个人也能这么说
- ❌ 没有具体应用场景
- ❌ 更像鸡汤而非可操作的分析工具

### 标记为「推测」的信号
- ❌ 仅出现在二手转述中
- ❌ 不同来源矛盾
- ❌ 无一手证据支撑

## 输出文件的组织

6个Agent的调研结果分别写入：
- `01-writings.md` → 提取 Layer 1（著作）、部分 Layer 3（表达）
- `02-conversations.md` → 提取 Layer 2（决策启发式）、Layer 3
- `03-expression-dna.md` → 提取 Layer 3（完整表达DNA）、Layer 4（价值观片段）
- `04-external-views.md` → 补充 Layer 1 的争议点、Layer 4 的反模式
- `05-decisions.md` → 提取 Layer 2（完整决策启发式）、验证 Layer 1
- `06-timeline.md` → 辅助理解 Layer 1 的演变、Layer 5 的轨迹

Phase 2 的合并工作：从这6个文件中，按6层结构重新组织，去重、验证、补充分析。
