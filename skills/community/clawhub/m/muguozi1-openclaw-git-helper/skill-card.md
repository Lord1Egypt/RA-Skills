## Known Risks and Mitigations: <br>
Risk: Mutating Git actions such as pull, push, checkout, merge, rebase, or branch deletion can change repository state or publish local work. <br>
Mitigation: Review git status and diffs first, then require explicit approval before running mutating Git commands, especially in shared or production repositories. <br>
