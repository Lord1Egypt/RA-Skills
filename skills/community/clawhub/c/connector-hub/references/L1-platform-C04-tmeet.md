# C04 - 腾讯会议

## 基本信息

| 字段 | 值 |
|------|-----|
| ID | C04 |
| 连接器名 | tmeet |
| 显示名 | 腾讯会议 |
| 层级 | L1 平台级 |
| 可替代 | ❌ 不可替代 |
| 分类 | 实时音视频 |

## 能力说明

腾讯会议提供实时音视频通信能力：

| 能力 | 说明 |
|------|------|
| 会议管理 | 创建/查询/结束会议 |
| 实时音视频 | 音频/视频/屏幕共享 |
| 会议录制 | 云端录制和回放 |
| 参会人管理 | 邀请/踢出/静音 |
| 会议控制 | 主持人权限管理 |

## 为什么不可替代

**需要平台实时通信基础设施**：
- 音视频需要平台的媒体服务器和编解码能力
- 实时通信需要平台的 WebRTC/SFU 架构
- 会议管理需要平台的状态管理和调度系统
- 录制需要平台的云端录制服务

**Skill 无法提供**：
- ❌ 无法提供音视频媒体服务器
- ❌ 无法提供实时通信协议栈
- ❌ 无法提供会议状态管理
- ❌ 无法提供云端录制服务

## 使用场景

| 场景 | 说明 |
|------|------|
| 会议创建 | 自动创建会议并生成链接 |
| 日程集成 | 日历事件自动关联会议 |
| 会议通知 | 自动发送会议提醒 |
| 会议记录 | 自动录制和生成纪要 |

## 配置方式

在 WorkBuddy 连接器管理页面启用 `tmeet`，配置：
- SDK ID / SDK Secret
- App ID（应用 ID）

## 替代方案（非 Skill）

可使用其他会议平台：
- Zoom
- Google Meet
- Microsoft Teams
- 飞书会议

但这些都需要平台账号和 SDK 集成，不是 Skill 替代。
