# 月嫂助手 - 母婴护理全流程管理

## 设计理念

> "母婴安全、产后恢复、新生儿健康" — 三大核心，全程守护。

本技能专为 **月嫂/月子中心/母婴护理团队** 设计，对齐2025年3月1日实施的《家政服务 母婴护理服务质量规范》新国标。

| 原则 | 说明 |
|------|------|
| 🤱 母婴一体 | 产妇恢复+新生儿护理，双线并行 |
| 🍲 科学月子 | 四阶段饮食（排恶露→调气血→补营养→养体质） |
| 📊 数据驱动 | 黄疸趋势/体重曲线/喂养量统计，可视化看板 |
| ⚠️ 异常预警 | 黄疸超标/体温异常/情绪低落自动提醒 |
| 📱 轻量易用 | 纯前端单文件，localStorage 存储，零部署 |

---

## 10大核心模块

### 1. 👶 新生儿护理
- **喂养记录**: 母乳/配方奶、时间、量(ml)、时长、拍嗝
- **排泄记录**: 大便（次数/颜色/性状）、小便次数
- **脐带护理**: 每日消毒记录、状态追踪（湿润→干燥→脱落）
- **沐浴抚触**: 洗澡/抚触/被动操记录
- **黄疸监测**: 每日黄疸值、趋势追踪、超标预警
- **睡眠记录**: 睡眠时段、总时长、夜醒次数

### 2. 👩 产妇护理
- **恶露观察**: 颜色（红→浆→白）、量、气味、持续时间
- **伤口护理**: 顺产侧切/剖腹产伤口清洁与恢复
- **子宫复旧**: 宫底高度变化、宫缩情况
- **乳房护理**: 开奶、通乳、乳头皲裂护理
- **体征监测**: 体温/血压/体重每日记录

### 3. 🍲 月子餐管理
- **四阶段食谱**: 排恶露(D1-7)→调理气血(D8-14)→滋补身体(D15-21)→养体质(D22-28)
- **每日三餐三点**: 早餐/加餐/午餐/午点/晚餐/晚点
- **饮食禁忌**: 回奶食物/寒凉食物/过敏源管理
- **内置食谱库**: 45+道经典月子餐，含做法
- **催乳食谱**: 通草鲫鱼汤/花生猪蹄汤/木瓜牛奶等

### 4. 📊 健康监测仪表盘
- 婴儿：体温折线图、黄疸趋势图、体重增长曲线
- 产妇：体温/血压趋势图、体重恢复曲线
- 异常值自动标红预警
- 28天全程数据可视化

### 5. 💊 用药管理
- 产妇用药计划（益母草/补铁剂/通乳药等）
- 婴儿用药计划（维生素D/益生菌/退黄疸药等）
- 每日用药打卡（已服/未服/延迟）
- 用药历史追溯

### 6. 📋 每日工作台
- 今日时间线（按24h月嫂工作流程）
- 待办提醒（喂奶/换尿布/测体温/用药等）
- 母婴关键数据一览
- 快速操作入口

### 7. 📝 护理日志
- 每日护理活动记录（喂奶/换尿布/脐带护理/抚触等）
- 时长和频次自动统计
- 异常情况备注
- 按日期筛选查看

### 8. 📖 母婴知识库
- **新生儿常见问题**: 黄疸/肠胀气/红臀/脐炎/湿疹/吐奶/肠绞痛
- **月子餐食谱**: 45+道分阶段食谱
- **产后恢复**: 盆底肌训练/腹直肌修复/凯格尔运动
- **母乳喂养**: 哺乳姿势/堵奶处理/追奶方法
- **应急指南**: 呛奶急救/发热处理/过敏反应

### 9. 📄 家属报告
- 自动生成每日母婴摘要
- 包含喂养/排泄/睡眠/体征/用药
- 每周综合报告
- 一键复制导出

### 10. 🔄 下户交接
- 28天护理数据总结
- 婴儿生长发育曲线汇总
- 产妇恢复情况汇总
- 技能传授清单（洗澡/喂养/脐护/抚触）
- 下户注意事项

## 使用方式

### 启动应用
```
说"打开月嫂助手"或"月嫂管理"
→ 自动生成交互式HTML母婴护理看板
```

### 首次使用
1. 录入母婴基本信息（姓名/分娩方式/出生体重等）
2. 开始每日记录
3. 数据自动持久化到 localStorage

### 数据存储
- 所有数据存储在浏览器 localStorage
- 支持导入/导出 JSON 数据备份
- 刷新页面数据不丢失
