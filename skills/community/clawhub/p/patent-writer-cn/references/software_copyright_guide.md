# 计算机软件著作权登记全流程指南

> 来源：中国版权保护中心官网（www.ccopyright.com）

## 一、办理步骤

```
注册账号 → 实名认证 → 在线填报 → 上传材料 → 登记机构受理 → 审查 → 取得证书
```

1. **账号注册**：进入 ccopyright.com，点击"注册"
2. **实名认证**：注册后完成认证（1-3个工作日），必须在办理登记前完成
3. **在线填报**：选择"R11 计算机软件著作权登记申请"
4. **上传材料**：签章页(PDF) + 身份证明 + 源程序 + 文档
5. **受理**：符合要求→受理通知书；不符合→补正通知（30日内）
6. **审查**：自受理日起60日内完成
7. **拿证**：电子证书在线下载，纸质证书挂号信寄送

## 二、所需材料

### 必备材料
| 序号 | 材料 | 要求 |
|------|------|------|
| 1 | 软件著作权登记申请表 | 在线打印→签章→扫描PDF |
| 2 | 源程序鉴别材料 | 前30页+后30页，每页≥50行 |
| 3 | 文档鉴别材料 | 前30页+后30页，每页≥30行 |
| 4 | 身份证明文件 | 原件照片/扫描件 |

### 身份证明（按主体）
- **企业法人**：加载统一社会信用代码的营业执照副本
- **自然人**：身份证
- **分支机构**：上级法人书面授权文件

### 其他证明（视情况）
- 委托开发→委托开发合同
- 合作开发→合作开发合同
- 修改他人软件→原著作权人许可证明

## 三、填表说明

| 栏目 | 要求 |
|------|------|
| 软件全称 | 简短明确，申请材料中保持一致 |
| 版本号 | 建议V1.0格式，页眉版本号需与申请表一致 |
| 开发完成日期 | 不能晚于申请日期 |
| 编程语言 | 全部编程语言 |
| 源程序量 | 源程序总行数 |
| 主要功能 | 精简完整地填写 |
| 签章 | 自然人签名/人名章；机构加盖公章（鲜章，不能用电子章） |

## 四、鉴别材料格式

### 源程序
- 页眉：软件名称 + 版本号
- 页码：右上角，正文第1页编到第60页
- 每页不少于50行

### 文档
- 页眉：软件名称 + 版本号
- 页码：右上角，目录开始第1页编到第60页
- 每页不少于30行
- 推荐：用户手册 / 设计说明书 / 操作手册

### 例外交存（涉密时）
1. 前、后各30页，机密部分黑色宽斜线覆盖（≤50%）
2. 前10页 + 任意50页
3. 目标程序前、后各30页 + 源程序任意20页

## 六、时限

| 环节 | 时限 |
|------|------|
| 实名认证 | 1-3个工作日 |
| 审查 | 受理日起60日内 |
| 补正 | 收到通知后30日内 |
| 纸质证书 | 额外7-15天 |
