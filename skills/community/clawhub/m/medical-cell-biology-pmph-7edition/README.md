## 项目简介

本项目系统整合医学细胞生物学的核心知识体系，涵盖 **146 项关键技能**，分为 20 大分类。内容横跨细胞结构与功能、分子机制、信号转导、细胞周期调控、干细胞生物学、细胞死亡与衰老、细胞器动态、膜运输、基因表达调控、疾病机制及前沿研究技术等多个维度，系统整合了经典理论与现代研究成果。

**适用人群**：医学生、科研人员、临床工作者、教育工作者

**参考教材**：人民卫生出版社《医学细胞生物学》第 7 版（"4B32B580-C081-460A-8470-6B3C64A4CFCB"）

**⚠️风险⚠️**：该技能涵盖细胞生物学概念、分子机制、病理分析及实验技术等内容，可能被误用为独立的医疗建议或实验指导。

缓解措施：仅将输出作为教育或科研参考资料使用，并根据当前指南和领域共识核实建议。

### 提问策略

#### 1. 概念
Question:
```bash
微管动态不稳定性（dynamic instability）的分子机制
```
Answer：
```bash
## 微管动态不稳定性机制

### 核心概念
微管动态不稳定性是指微管在**聚合**与**解聚**两种状态之间的随机转换现象，由GTP帽状态决定。

### 两个关键转换
| 转换 | 定义 | 触发条件 |
|------|------|----------|
| **救援（Rescue）** | 从解聚转变为聚合 | 游离GTP-微管蛋白浓度充足时可发生 |
| **灾难（Catastrophe）** | 从聚合转变为解聚 | GTP水解速率超过新亚基添加速率，GTP帽丢失 |

### 分子机制
- **GTP帽模型**：聚合中的微管正端带有GTP-微管蛋白"帽"，保持微管稳定
- 当聚合速率减慢 → GTP水解赶上 → GTP帽丢失 → 原纤维向外弯曲 → 快速解聚
- **微管蛋白浓度**：高于临界浓度→聚合；低于临界浓度→解聚

### 临床意义
- **紫杉醇**：稳定微管，抑制解聚，阻止有丝分裂
- **秋水仙碱**：结合游离微管蛋白，抑制聚合
- **长春新碱**：抑制微管聚合，用于抗癌治疗
```

#### 2. 病理机制分析
Question:
```bash
分析家族性高胆固醇血症（FH）的细胞生物学机制
```
Answer：
```bash
## 家族性高胆固醇血症的分子病理机制

### 病因
LDL受体（LDLR）基因突变导致肝细胞无法有效摄取血浆LDL。
- **纯合子**：LDLR活性约为正常的3.6%
- **杂合子**：LDLR活性约为正常的50%

### LDL受体缺陷的5种类型
1. **结合缺陷型** — 受体不能与LDL结合
2. **聚集缺陷型** — 受体无法定位于有被小窝
3. **内化缺陷型** — 受体-LDL复合物无法内化
4. **再循环缺陷型** — 受体和LDL在溶酶体中一起被降解
5. **信号传导缺陷型** — 无法传递调控信号

### 细胞水平后果
LDLR功能缺陷 → 细胞无法摄取LDL → 胞内胆固醇↓
→ HMG-CoA还原酶持续高活性 → 细胞过度合成胆固醇
→ 血浆LDL清除减少 → 血管壁沉积 → 动脉粥样硬化

### 实验验证
- HMG-CoA还原酶活性检测：正常细胞+LDL→酶活性下降；FH细胞无此反应
- ¹²⁵I-LDL结合实验：FH细胞几乎无LDL结合能力
```
