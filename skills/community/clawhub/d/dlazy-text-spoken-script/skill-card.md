## Description: <br>
Text Spoken Script guides agents to write short video spoken scripts with contrast hooks, story structure, audience resonance, and personal IP positioning. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[dlazyai](https://clawhub.ai/user/dlazyai) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External creators and agents use this skill to draft short video spoken scripts for persona storytelling, character viewpoint pieces, and topic-driven social content. The intended script style is conversational, structured, emotionally resonant, and grounded in concrete people, events, and details. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Security evidence says the skill unexpectedly includes terminal-based image-generation workflow instructions beyond spoken-script writing. <br>
Mitigation: Review before installing and prefer a version that removes the unrelated CLI/image-generation section or separates it into a clearly named skill. <br>
Risk: The artifact requires a dLazy API key and describes storing credentials in local CLI configuration or passing them through an environment variable. <br>
Mitigation: Use only expected accounts and rotate or revoke dLazy API keys when no longer needed. <br>
