# AI Trending News Agent Skill

This repository contains an Agent Skills-compatible skill for generating a ranked top-10 AI news briefing.

## Structure

- `SKILL.md` — main skill instructions
- `references/sources.md` — curated source policy
- `references/ranking.md` — scoring model
- `scripts/fetch_ai_news.py` — source intake and ranking helper

## Use

Install the folder as a skill directory named `ai-trending-news` and let the agent load `SKILL.md`.
