## 教育培训生成要求

教育培训每种任务类型的"零件清单"——必选/可选组件、组装顺序、领域约束。按清单逐项产出。

---

### 1 课程设计

#### T1-1 需求调研

- **必选组件**: 学员画像（年龄、学历、基础水平）、岗位能力要求清单、市场人才需求分析、竞品课程分析
- **可选组件**: 行业发展趋势、企业用人反馈、学员职业规划
- **组装顺序**: 确定调研对象 → 设计调研方案 → 执行调研（访谈/问卷/数据分析） → 汇总分析 → 形成需求报告
- **约束**: 调研样本量≥30；必须覆盖目标学员和用人单位；数据来源必须可靠
- **格式**: 需求调研报告（Markdown）

#### T1-2 目标设定

- **必选组件**: 培训目标（知识目标/技能目标/态度目标）、能力指标（可量化）、考核标准（合格线/优秀线）、目标验证方法
- **可选组件**: 分层目标（基础/进阶/高级）、个性化目标
- **组装顺序**: 分析需求报告 → 提炼核心能力要求 → 设定培训目标 → 制定考核标准 → 验证目标合理性
- **约束**: 目标必须SMART（具体、可衡量、可达成、相关性、时限性）；考核标准必须可量化
- **格式**: 培训目标文档（Markdown）

#### T1-3 内容开发

- **必选组件**: 课程大纲（模块/章节/知识点）、知识点体系（知识图谱）、案例素材（真实案例/模拟案例）、练习题目（选择题/实操题）
- **可选组件**: 扩展阅读材料、视频素材、互动游戏
- **组装顺序**: 设计课程大纲 → 构建知识点体系 → 开发案例素材 → 编制练习题目 → 整合成课程包
- **约束**: 内容必须覆盖所有培训目标；知识点必须有逻辑顺序；案例必须真实可操作
- **格式**: 课程大纲（Markdown）、知识点清单（表格）

#### T1-4 教材编制

- **必选组件**: 教材正文（章节内容）、课件（PPT/演示文稿）、实操手册（步骤指南）、参考资料（书单/链接）
- **可选组件**: 视频教程、在线资源、工具模板
- **组装顺序**: 编写教材正文 → 制作课件 → 编制实操手册 → 整理参考资料 → 排版校对
- **约束**: 教材语言必须通俗易懂；课件必须视觉清晰；实操必须步骤明确
- **格式**: 教材文档（Markdown/Word）、课件（PPT）

---

### 2 教学实施

#### T2-1 教学计划

- **必选组件**: 教学进度表（课时/日期/内容）、教学方法选择（讲授/案例/实操/讨论）、资源配置（师资/场地/设备）、应急预案
- **可选组件**: 个性化教学方案、分组计划、课外辅导安排
- **组装顺序**: 分析课程内容 → 设计教学进度 → 选择教学方法 → 配置资源 → 制定应急预案
- **约束**: 进度必须合理（不过快/不过慢）；方法必须匹配内容；资源必须充足
- **格式**: 教学计划表（Markdown表格）

#### T2-2 课堂讲授

- **必选组件**: 讲授内容（知识点讲解）、重点难点解析、案例分析、课堂互动
- **可选组件**: 多媒体素材、演示实验、小组讨论
- **组装顺序**: 导入（5min） → 知识讲授（30min） → 案例分析（15min） → 互动讨论（10min） → 总结（5min）
- **约束**: 讲授必须逻辑清晰；案例必须贴近实际；互动必须有效引导
- **格式**: 课堂讲授记录（Markdown）

#### T2-3 实操指导

- **必选组件**: 实操任务说明、操作步骤指南、常见问题解答、安全注意事项
- **可选组件**: 操作视频、工具模板、参考案例
- **组装顺序**: 布置实操任务 → 演示操作步骤 → 学员动手实践 → 巡回指导 → 成果检查
- **约束**: 任务必须明确；步骤必须可操作；安全必须强调
- **格式**: 实操指导手册（Markdown）

#### T2-4 互动答疑

- **必选组件**: 问题收集渠道、答疑时间安排、问题分类处理、答疑记录
- **可选组件**: FAQ文档、在线答疑平台、同伴互助机制
- **组装顺序**: 收集问题 → 分类整理 → 集中答疑 → 个别辅导 → 记录归档
- **约束**: 问题必须及时响应；答疑必须准确清晰；记录必须完整
- **格式**: 答疑记录（Markdown）

### 3 学员管理

#### T3-1 招生注册

- **必选组件**: 招生简章、报名表、审核标准、注册流程
- **可选组件**: 招生宣讲会、试听课程、优惠政策
- **组装顺序**: 发布招生简章 → 接受报名 → 资格审核 → 录取通知 → 注册登记
- **约束**: 招生信息必须真实；审核必须公平；注册必须完整
- **格式**: 招生注册表（Markdown表格）

#### T3-2 学籍管理

- **必选组件**: 学员信息表、学习进度记录、档案管理制度、信息安全措施
- **可选组件**: 在线学籍系统、家长沟通机制、学员画像
- **组装顺序**: 建立学员档案 → 录入信息 → 跟踪进度 → 更新档案 → 归档管理
- **约束**: 信息必须准确；进度必须及时更新；隐私必须保护
- **格式**: 学籍管理表（Markdown表格）

#### T3-3 出勤考核

- **必选组件**: 出勤记录表、请假审批流程、缺勤处理规则、课堂表现评估
- **可选组件**: 在线签到系统、出勤奖励机制
- **组装顺序**: 每日签到 → 记录出勤 → 处理请假 → 统计缺勤 → 评估表现
- **约束**: 记录必须真实；处理必须公正；评估必须客观
- **格式**: 出勤考核表（Markdown表格）

#### T3-4 成绩评定

- **必选组件**: 考核方案（平时/期中/期末）、评分标准、成绩汇总表、证书模板
- **可选组件**: 补考机制、成绩申诉流程、优秀学员评选
- **组装顺序**: 制定考核方案 → 执行考核 → 评分 → 汇总成绩 → 颁发证书
- **约束**: 考核必须覆盖所有目标；评分必须公平；证书必须规范
- **格式**: 成绩评定表（Markdown表格）

---

### 4 效果评估

#### T4-1 满意度调查

- **必选组件**: 调查问卷（课程内容/教学方法/师资水平/设施环境）、调查方案、数据分析方法
- **可选组件**: 访谈提纲、焦点小组讨论
- **组装顺序**: 设计问卷 → 确定样本 → 发放问卷 → 收集数据 → 分析结果
- **约束**: 问卷必须覆盖关键维度；样本必须有代表性；分析必须客观
- **格式**: 满意度调查报告（Markdown）

#### T4-2 效果测评

- **必选组件**: 测评方案（知识测试/技能考核/综合评估）、测评工具、评分标准、结果分析
- **可选组件**: 前后对比分析、对照组设计
- **组装顺序**: 设计测评方案 → 开发测评工具 → 执行测评 → 评分 → 分析结果
- **约束**: 测评必须覆盖所有目标；工具必须有效；分析必须深入
- **格式**: 效果测评报告（Markdown）

#### T4-3 就业跟踪

- **必选组件**: 跟踪方案、调查问卷（就业率/岗位匹配度/薪资水平）、跟踪周期、数据分析
- **可选组件**: 企业回访、校友网络、职业发展支持
- **组装顺序**: 确定跟踪对象 → 设计调查方案 → 执行调查 → 收集数据 → 分析结果
- **约束**: 跟踪必须持续（至少6个月）；数据必须真实；分析必须客观
- **格式**: 就业跟踪报告（Markdown）

#### T4-4 课程改进

- **必选组件**: 改进依据（满意度/效果测评/就业跟踪结果）、改进方案（内容/方法/教材）、改进计划、效果验证
- **可选组件**: 专家评审、同行评议
- **组装顺序**: 分析评估结果 → 识别改进点 → 制定改进方案 → 执行改进 → 验证效果
- **约束**: 改进必须基于数据；方案必须可行；效果必须可验证
- **格式**: 课程改进方案（Markdown）
