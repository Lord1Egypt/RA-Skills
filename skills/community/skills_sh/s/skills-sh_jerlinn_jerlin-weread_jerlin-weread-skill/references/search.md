# search

`weread.sh search -h`

## scope 选择

未传时默认 10（电子书）。综合搜索须显式传 0。

| scope | 触发意图 | 备注 |
|-------|---------|------|
| 0 | 泛搜索，未限定类型 | results 含多个分组 |
| 10 | 明确找书/搜书/获取 bookId；普通「小说」 | 不含网文 |
| 16 | 「网文」「网络小说」 | |
| 14 | 听书/有声书/播客/专辑 | 三者同义 |
| 6 | 「查作者 xx」 | |
| 12 | 「书里提到了 xx」/全文搜索 | 搜正文内容 |
| 13 | 书单 | |
| 2 | 公众号 | |
| 4 | 文章 | |

决策边界：明确找书意图 → 10；未限定类型 → 0。互斥。

## Gotchas

- 用户未指定数量时不传 count。
- soldout=1 标注下架。
- 翻页：末条 searchIdx → `--maxIdx`。
