# Skill 增长管家

> 发了 Skill，怎么让它跑起来

Powered by SkillManager

进来选场景，增长管家按需出建议。**不用贴 SKILL.md，说方向/数据/需求就够。**

---

## 怎么用

**说任何话进来**（"你好"/"帮我看看"/"我的 installs 不涨"），管家先出菜单，选场景后直接跑。

| # | 场景 | 提供什么 |
|:---:|:---|:---|
| 1 | **赛道评估** | 一句话描述方向 |
| 2 | **增长诊断** | 贴 installs / stars 数据 |
| 3 | **文案优化** | 贴 description |
| 4 | **上下游关联** | 说 Skill 方向 |
| 5 | **变现路径** | 说 Skill 定位 |
| 6 | **合作撮合** | 说合作需求 |
