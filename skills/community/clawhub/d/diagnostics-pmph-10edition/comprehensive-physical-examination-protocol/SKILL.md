# 全身体格检查基本流程与原则

## 触发条件
- 需完成住院病历体格检查部分
- 进行全面健康评估
- 已完成问诊

## 核心原则
- **全面系统**：覆盖所有条目，同时根据问诊结果重点深入可疑系统
- **避免机械重复**：检查内容需有针对性
- **减少患者不适**：全程体位变动控制在2–3次以内
- **时间控制**：总时长尽量≤40分钟
- **沟通闭环**：检查结束简要反馈重要发现或下一步计划

## 推荐检查顺序

### 卧位起始
1. 一般情况与生命体征
2. 头颈部
3. 前/侧胸部（心、肺）
4. （转坐位）后背部（肺、脊柱、肾区、骶部）
5. （转仰卧位）腹部
6. 上肢、下肢
7. 肛门直肠、外生殖器
8. 神经系统（站立位）

### 坐位起始
1. 一般情况与生命体征
2. 上肢
3. 头颈部
4. 后背部
5. （转仰卧位）前/侧胸部 → 腹部
6. 下肢 → 肛门直肠 → 外生殖器
7. 神经系统（站立位）

## 特殊处理
- **皮肤、淋巴结、神经系统**：分段检查，统一记录
- **不确定体征**：不得随意解释，避免造成患者焦虑

> 输出为符合住院记录要求的完整体格检查文档，并标出需重点评估的异常体征。
