# Cognitive Reframe 认知重构助手

基于 CBT 原理的日常思维自调节工具——帮你识别负面思维模式，用更平衡的视角看问题。

## What it does
- 识别 8 种常见认知扭曲（灾难化、贴标签、读心术等）
- 用证据法、替代解释等技巧引导认知重构
- 区分日常调节与需要专业帮助的高风险场景
- 每次对话结尾附安全边界声明

## Example scenarios

**1. 重要汇报前焦虑灾难化**
> 👤 明天有个重要汇报，如果搞砸了我就全完了，以后在公司没法混了。
> 🤖 先理解你的焦虑很真实。这里面有"灾难化"的思维模式——把一次汇报的结果等同于你的整个职业前途。我们可以分开看：最坏情况发生的概率其实很低，而最可能的结果是你正常发挥、有亮点也有不足。建议：今晚先把前 3 页 PPT 准备扎实，这一个小行动就能大幅降低未知感。

**2. 被裁员后自我否定**
> 👤 公司优化掉了，35 岁了觉得自己啥也不是，面试了几家都挂了，是不是我能力不行？
> 🤖 这听起来是"贴标签"——把"被裁"这个事件等同于"我这个人没用"。我们用证据法来拆一下：你过去几年的从业经验、积累的技能、完成过的项目，这些都是你"能力行"的证据。就业市场不景气是环境因素，不是你一个人的问题。建议：把"被裁"在面试中正面包装成"组织调整后我主动寻求新挑战"。

**3. 开会后猜测同事负面评价**
> 👤 今天开会我说的方案，同事肯定觉得我很蠢。
> 🤖 这属于"读心术"思维——你在假设自己知道别人怎么想，但其实你没有证据。同事们可能只是在消化信息、可能在想自己的事、甚至觉得你的方案有可取之处。一个小行动：下次找个你信任的同事，直接问"你觉得今天会上那个方案有没有什么我没考虑到的？"，用事实替代猜测。
