# Skill调用接口

> 本文档详细说明如何调用"并购对接"Skill，包括输入格式、输出格式和调用示例。

---

## 一、Skill定位与调用关系

### 1.1 在并购服务三阶段中的位置

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        OPC导师 并购服务三阶段                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  【第一阶段】              【第二阶段】              【第三阶段】           │
│  并购匹配            →    并购对接           →    并购执行               │
│       │                      │                      │                    │
│       ▼                      ▼                      ▼                    │
│  筛选目标公司            找到关键人             法律尽调                 │
│  评估匹配度              设计接触策略           交易结构                 │
│  确定优先级              制作推介材料           签约交割                 │
│       │                      │                      │                    │
│  输出：                   输出：                 输出：                   │
│  ·目标公司清单           ·关键人画像            ·尽调报告                │
│  ·匹配度评分             ·接触记录              ·交易文件                │
│  ·优先级排序             ·推进状态              ·交割确认                │
│                                                                          │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━                 │
│                                                                          │
│  本Skill（并购对接）的输入是：                                          │
│  并购匹配Skill输出的"目标公司清单+匹配度评分+优先级排序"                │
│                                                                          │
│  本Skill（并购对接）的输出是：                                          │
│  关键人画像+接触策略+推介材料+推进状态                                  │
│  ↓ 传递给并购执行Skill                                                  │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 Skill调用依赖关系

| 上游Skill | 输出内容 | 传递给本Skill |
|-----------|----------|---------------|
| 并购匹配 | 目标公司清单、匹配度评分、优先级排序 | 关键人识别输入 |
| 股权合作机制 | SPV模式、分账规则 | 交易结构参考 |

| 本Skill输出 | 下游Skill |
|-------------|-----------|
| 接触记录、推进状态、交易框架 | 并购执行 |

## 五、与其他Skill的协作

### 5.1 并购匹配 → 并购对接

当用户需要从"筛选目标公司"过渡到"对接推进"时：

```
并构匹配Skill输出：
├── 目标公司清单（Top 10）
├── 匹配度评分表
└── 优先级排序

↓
直接调用并购对接Skill：
输入：上述清单+己方项目信息
输出：每个公司的关键人画像+接触策略
```

### 5.2 并购对接 → 并购执行

当某条对接线进入"尽调阶段"时：

```
并构对接Skill输出：
├── 接触记录
├── 推进状态（S3：尽调阶段）
└── 初步交易框架

↓
调用并购执行Skill：
输入：目标公司+接触记录+初步框架
输出：尽调清单+交易文件模板
```
