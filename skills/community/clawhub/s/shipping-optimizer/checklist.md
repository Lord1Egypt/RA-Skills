# Shipping Optimizer — Audit Checklist

Use before finalizing any shipping optimization plan. All sections must be reviewed.

---

## 1. Data Collection (6 items)
- [ ] 90-day shipping invoice data exported from carrier portal(s)
- [ ] Data includes: origin, destination zip, carrier, service, actual weight, billed weight, all surcharges
- [ ] Zone for each shipment calculated (or pulled from invoice)
- [ ] Residential vs. commercial delivery percentage identified
- [ ] Average order weight (actual and DIM billed) calculated
- [ ] Current blended average shipping cost per order calculated

## 2. DIM Weight Audit (6 items)
- [ ] DIM weight calculated for every box size in current use
- [ ] % of shipments billed at DIM (vs. actual) weight identified
- [ ] Per-shipment DIM premium calculated for each box type
- [ ] Alternative packaging options evaluated (smaller box or poly mailer)
- [ ] Annual cost of current DIM weight situation quantified
- [ ] Custom box ROI calculated for any product shipping 500+/month

## 3. Carrier Analysis (7 items)
- [ ] Current carrier mix documented (% volume by carrier and service level)
- [ ] Rate comparison completed for top 3 weight/zone combinations
- [ ] USPS Ground Advantage evaluated for all packages ≤ 1 lb
- [ ] Regional carrier quotes obtained for Zone 1–3 deliveries
- [ ] UPS SurePost or FedEx Ground Economy evaluated for residential lightweight
- [ ] Multi-carrier rate shopping tool evaluated (EasyPost, ShipStation, etc.)
- [ ] Carrier selection matrix documented (which carrier for which weight/zone)

## 4. Surcharge Audit (5 items)
- [ ] All surcharges from last 90 days listed and totaled
- [ ] Address correction fees identified and root cause investigated
- [ ] Residential delivery surcharge total calculated
- [ ] Address validation at checkout evaluated (SmartyStreets, etc.)
- [ ] Fuel surcharge cap negotiation identified as target (if fuel surcharges > 10%)

## 5. Zone Distribution (4 items)
- [ ] Zone distribution of last 90 days mapped
- [ ] % of orders shipping to Zone 6–8 calculated
- [ ] Alternative fulfillment location analyzed (if Zone 6–8 > 20% of orders)
- [ ] 3PL cost vs. shipping savings modeled for additional fulfillment location

## 6. Contract & Negotiation (5 items)
- [ ] Current carrier discount level vs. list rate identified
- [ ] 12-month volume data prepared for negotiation
- [ ] Competitive quotes obtained from at least 2 carriers
- [ ] Negotiation target discounts set by weight/zone tier
- [ ] Annual carrier review scheduled

## 7. Free Shipping Strategy (4 items)
- [ ] Current free shipping threshold documented
- [ ] Average shipping cost for orders at and below current threshold calculated
- [ ] Conversion rate impact of threshold increase modeled
- [ ] New threshold or conditional free shipping (minimum spend, specific zones) evaluated

## 8. Implementation (5 items)
- [ ] Priority ranked optimization initiatives (effort vs. saving)
- [ ] Implementation timeline set with owner for each initiative
- [ ] Packaging changes approved and ordered (if applicable)
- [ ] Multi-carrier routing rules programmed in shipping software
- [ ] Post-implementation cost comparison date scheduled (60 days out)

---

**Total items: 42**
**Minimum for a basic optimization plan: Sections 1–4 complete**
**Minimum for a comprehensive plan: All sections complete**
