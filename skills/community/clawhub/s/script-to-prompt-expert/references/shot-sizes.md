# 景别知识库

根据剧本内容、叙事节奏和情感需要，从以下景别中选取最合适的：

## 一、远景系列——建置环境、交代关系

### 大远景（极远景）
- **核心作用**：主体在画面中极小，环境占据绝对主导地位，人物几乎不可辨识
- **情感/氛围**：孤独、渺小、苍凉、史诗、震撼、无力、自由、敬畏
- **使用场景**：开场建立时空背景（广袤沙漠、浩瀚海洋、无边宇宙）；表现人物在宏大环境中的孤独与无助；战争片中的两军对垒阵势；史诗级场景
- **经典案例**：《2001太空漫游》开场星空与地球；《荒野猎人》格拉斯在雪原上的微小身影；《阿拉伯的劳伦斯》沙漠地平线上的小黑点

### 远景
- **核心作用**：展示主体及其周围环境的全貌，人物清晰可辨但仍以环境为主
- **情感/氛围**：开阔、全景感、空间关系、客观、疏离
- **使用场景**：开场交代场景（城市全景、庄园外景、战场全景）；人物与环境互动；群体场面
- **经典案例**：《乱世佳人》十二橡树庄园远景；《指环王》霍比特村庄全貌

## 二、全景系列——完整呈现主体

### 全景
- **核心作用**：展现人物完整全身，从头到脚，保留一定环境背景
- **情感/氛围**：完整、客观、舞台感、动作感、身份展示
- **使用场景**：人物出场介绍（全身造型、体态、走姿）；舞蹈、武打、体育等全身动作场景；群体走位、舞台剧式调度
- **经典案例**：《泰坦尼克号》杰克和露丝船头张开双臂；《黑客帝国》尼奥后仰躲子弹全身姿态

### 中全景（膝上景）
- **核心作用**：从膝盖或小腿以上取景，介于全景和中景之间
- **情感/氛围**：自然、过渡、日常、动作导向
- **使用场景**：人物行走、跑步、站立的常规镜头；双人对话需展示部分肢体动作时；从全景切到中景的过渡镜头

## 三、中景系列——聚焦上半身

### 中景
- **核心作用**：从腰部以上取景，最常用、最中性的景别
- **情感/氛围**：中性、自然、日常、叙事、交流
- **使用场景**：普通对话场景；展现上半身动作（吃饭、写字、握手）；新闻播报、访谈节目
- **经典案例**：几乎所有电影中的常规对话镜头；《老友记》等情景喜剧固定镜头

### 中近景
- **核心作用**：从胸部以上取景，介于中景和近景之间
- **情感/氛围**：亲近、关注、强调台词、情感递进
- **使用场景**：单人讲话（演讲、倾诉）；重要对话中情绪升温时的特写准备；表现角色反应和细微表情变化
- **经典案例**：《教父》迈克尔在餐厅枪杀索洛佐时的胸部以上镜头

## 四、近景系列——强调表情

### 近景
- **核心作用**：从肩部以上取景，头部成为画面主体，表情、眼神、细微肌肉变化清晰可见
- **情感/氛围**：亲密、专注、情感流露、紧张、审视
- **使用场景**：角色情感高潮段落（哭泣、愤怒、恐惧）；重要台词或告白场景；审讯、对峙时的表情交锋
- **经典案例**：《辛德勒的名单》辛德勒说"我可以救更多人"时的近景；《霸王别姬》程蝶衣"说好了一辈子"

## 五、特写系列——放大细节

### 特写
- **核心作用**：局部放大，通常指面部特写（发际线到下巴）或重要物体细节
- **情感/氛围**：强调、冲击、私密、戏剧性、符号化
- **使用场景**：面部特写用于人物情绪爆发或转折瞬间；物体特写用于关键道具（戒指、信件、枪、钥匙）；动作细节（扣动扳机的手指、滴落的药水）
- **经典案例**：《这个杀手不太冷》门缝里玛蒂尔达的眼睛特写；《七宗罪》箱子里的惊喜特写

### 大特写
- **核心作用**：极端放大局部细节（一只眼睛、嘴唇、指纹、毛孔、针尖），超越正常视觉经验
- **情感/氛围**：极度紧张、惊悚、迷幻、疼痛、亲密、抽象
- **使用场景**：瞳孔倒映影像；血液滴落、针管刺入皮肤瞬间；子弹穿过物品的慢动作；精神崩溃或吸毒后的视觉扭曲
- **经典案例**：《发条橙》阿历克斯被治疗时眼睛被撑开；《荒野猎人》格拉斯呼吸时瞳孔与雪地的极端特写

## 六、特殊景别

### 双人景别（Two Shot）
- **核心作用**：两人同框，通常展示两人从腰部或胸部以上
- **情感/氛围**：关系感、对比、对峙、合作、亲密或疏离
- **使用场景**：双人对话需展示两人位置关系和身体语言时；情侣并肩、好友同行、敌人对峙；作为正反打镜头的基础
- **经典案例**：《爱在黎明破晓前》杰西和塞琳娜火车上相对而坐

### 过肩景别（Over the Shoulder Shot）
- **核心作用**：从一人肩膀后方拍摄另一人，通常为中近景或近景
- **情感/氛围**：主观参与感、对话关系、亲密或距离
- **使用场景**：双人对话的标准拍法，尤其是情绪交流较强的对话

### 插入镜头（Insert Shot）
- **核心作用**：在连续动作中切入一个与主体不同但相关的细节特写
- **情感/氛围**：提示、强调、悬念、解释
- **使用场景**：人物看手机时插入屏幕特写；人物写东西时插入笔尖和纸面；悬疑片中插入钟表、炸弹倒计时

## 七、景别叙事功能总结

| 功能 | 适用景别 |
|------|---------|
| **建置**（建立时空环境，交代人物与环境关系） | 大远景、远景、全景 |
| **动作**（展现人物肢体动作和运动状态） | 全景、中全景、中景 |
| **情绪**（表达人物情感和内心活动） | 中近景、近景、特写 |
| **细节**（强调关键信息、道具或象征意义） | 特写、大特写 |
| **关系**（表现人物之间的互动与关系） | 双人景别、过肩景别 |

## 八、按情绪快速选型速查表

| 目标情绪 | 推荐景别 |
|---------|---------|
| 宏伟/史诗/孤独 | 大远景、远景 |
| 客观/完整/动作 | 全景、中全景 |
| 自然/日常/对话 | 中景、中近景 |
| 亲密/情感/专注 | 近景、特写 |
| 冲击/紧张/惊悚 | 特写、大特写 |
| 关系/对峙/合作 | 双人景别、过肩景别 |
| 暗示/悬念 | 插入镜头 |

## 九、补充说明

- **景别的情感递进规律**：景别越远，情感越抽离、客观；景别越近，情感越侵入、主观。从大远景到大特写，观众的心理距离从旁观者逐渐变为参与者，最后变为窥视者。
- **景别与叙事节奏**：快速切换不同景别制造节奏感；连续使用同一景别会显得单调；由远及近的景别递进往往伴随情绪升级。
- **景别与场面调度**：景别不是孤立存在的，与摄影机距离、镜头焦距、光圈景深共同决定画面视觉重量。长焦镜头可在远处拍出近景效果，广角镜头可在近处拍出全景效果。
- **经典导演景别偏好**：
  - 侯孝贤：多用远景、固定长镜头，保持客观距离
  - 王家卫：大量特写、慢镜头，强调情绪细节
  - 韦斯·安德森：对称构图配合中景、全景，人物被框在画面中
  - 库布里克：极端景别对比（大远景突然切大特写），制造心理冲击
