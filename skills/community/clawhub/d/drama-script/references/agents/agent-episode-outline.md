# 逐集概要 · 代理工作手册

> 通用规则见 `../common-agent-rules.md`。

## 定位

逐集概要是下游环节（剧本）的**可拍摄蓝图**。

它不写台词、镜头机位、美术设计——那是剧本的事。但它必须写到**拿着这份概要就能直接理解"这场戏拍什么"**的程度。

核心原则：**围绕"拍什么"写，不围绕"发生了什么"写。**

下游拿到逐集概要后，应该清楚地知道：
1. 每集的起点和终点在哪
2. 每一集要让观众经历什么情绪
3. **每一场戏里人物具体做了什么动作、产生了什么可见的结果**
4. 冲突点在哪里、钩子怎么留

---

## 任务

- **输入**：故事概要 + 人物小传 + 上一批逐集概要（第2批起）
- **输出**：每批10集的逐集蓝图（含过场人物）
- **目标**：将故事概要展开为逐集的可拍摄蓝图，并补充过场人物

---

## 流程

### 1. 撰写逐集蓝图

每集输出四项：**情绪目标、衔接上集、核心事件、集末钩子**。

1. 写**情绪目标**（一行）：情绪旅程用箭头标注，必须反复上下翻转，最后一个情绪直接导向集末钩子
2. 写**衔接上集**（1句）：「承接」一句话说清上集结尾→本集切入；「硬切」标注新场景。首集写"无（本批首集）"
3. 写**核心事件**（{N}~{N+2}个，N = ⌊集时长(秒) ÷ 30⌋）：
   - 格式：`【场景名】（情绪起点→情绪终点）：事件描述`
   - 场景名用简短标签
   - 事件描述按可拍摄级标准执行：
     - **动作可见**：写人物的具体物理动作（砸、推、摔、撕、扔、撞），不写抽象概括（"局势恶化""关系缓和""内心动摇"）
     - **冲突具象**：冲突有物理形态——拳头砸到脸上、文件摔在桌上、手机摔碎在地，不是"紧张对峙"
     - **结果可感**：事件结果有视觉/听觉反馈——人倒下、墙裂开、灯灭了、玻璃碎裂，不是"局势升级"
     - **不改上游定位**：不写台词、机位、美术、服装道具细节。粒度是"这场戏拍什么"，不是"怎么拍"
     - **精炼优先**：可拍摄不等于啰嗦，用最少的字写最清晰的画面
     - **字数约束**：单事件≤60字，整集≤700字。蓝图不是小说——只写动作和结果，砍掉环境渲染、微表情、心理旁白、修饰定语。写完每集数一下，超了就砍
   - 事件之间齿轮咬合——前一个结果触发下一个，逐级传导到集末钩子
   - 默认以感官冲击力和情绪释放为第一优先级
4. 写**集末钩子**（1句）：包含新信息或新威胁，与本集事件因果关联，相邻集类型不重复。写的是"观众看到了什么"

### 2. 补充过场人物

每批逐集概要写完后，从当前批次中提取不在人物小传中的过场人物，编写简化版小传，追加到本批概要文件末尾。

1. 读取人物小传，获取已有人物名单
2. 如果不是第1批，读取上一批逐集概要末尾的过场人物部分，获取已收录过场人物
3. 扫描当前批次逐集概要中的出场人物
4. 找出过场人物（在当前批次出场但不在已有人物和前几批已收录过场人物中的人物）
5. 按格式追加到文件末尾

过场人物约束：

- 信息来源仅限当前批次逐集概要，不杜撰
- 每个人物不超过8行
- 同名人物去重
- 当前批次无首次出场人物时，不输出此章节

### 3. 保存文件

按输出格式将逐集蓝图和过场人物整合为完整的批次文件，保存到指定路径。

```markdown
- 【{场景}】（{情绪A→情绪B}）：{事件描述}
- 【{场景}】（{情绪B→情绪C}）：{事件描述}
- ...

集末钩子：{一句话}

## EP{X+1}

...

# 已登场人物（第1-{Y}集）

- {人物名}：{集数}
...

---

## 过场人物（第X-Y集）

### {人物名}
- **身份**：{一句话}
- **性格**：{2-3个关键词 + 一句话描述}
- **外貌特征**：{一句辨识度高的特征}

```

## 自检

生成完成后，逐条检查以下致命问题。**命中任何一条必须立即修改，全部通过后才能交付。**

### 1. 不可拍摄

- 拿着概要递给拍摄团队，他们需要追问"这段到底拍什么"才能理解。事件描述全是抽象概括（"局势恶化""关系紧张""双方矛盾升级"），看不到具体的人物动作；或情绪描述代替了事件描述（"气氛压抑""人心惶惶"——不知道机位放哪、演员演什么）。

### 2. 事件齿轮断裂

- 前一个核心事件的结果没有触发后一个核心事件。事件之间是并列关系而非递进关系；或某事件突然出现与前一个事件毫无因果关联。

### 3. 情绪目标没有翻转

- 某集的情绪目标只在单向滑动（紧张→愤怒→悲伤，或开心→更开心→超开心），没有翻转。或情绪箭头标注了翻转但核心事件内容撑不起翻转。

### 4. 集末钩子是废话

- 钩子不包含新信息或新威胁。"下一集更精彩""主角面临更大的挑战""真相即将揭晓"——适用于任何一集结尾。或钩子只是重复本集已有信息。

### 5. 事件描述是心理活动而非物理动作

- 核心事件中出现"内心挣扎""陷入沉思""意识到不对""下定决心"等心理描写。情绪标签允许，但事件描述本身在写看不见的东西就是问题。

### 6. 相邻集钩子类型重复

- 连续2-3集的集末钩子是同一种类型——连续3集都是"新人物登场"、都是"收到威胁信息"、都是"关系破裂"。观众形成预期模式后，钩子悬念效果归零。

### 7. 字数失控

- 单事件超过60字或整集超过700字，说明混入了环境渲染、微表情、心理旁白等小说笔法。
