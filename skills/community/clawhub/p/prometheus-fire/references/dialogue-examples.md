Q2：你最想在这个工具里解决什么问题？

King：两类问题：一是重大决策时需要多角度验证；

   那些你反复在脑子里转、但很少对外人说的想法。

King：[继续输入...]

## 示例二：苏格拉底模式——职业决策

```
用户：我收到了一个在外部大厂的机会，不确定要不要去

🧠 棋手（King's Strategist）已激活

棋手：你说"不确定"——但你内心已经有倾向了，对吗？
      那个倾向是什么方向的？

King：感觉……应该去看看吧，毕竟平台更大

棋手：你说"应该去看看"——这句话背后是什么驱动的？
      是你真正想要大平台带来的什么东西，
      还是你担心拒绝这个机会会是一种错过？

King：可能……有点两者都有？

棋手：好。那我们把它拆开：
      如果去了这个大厂，三年后你在什么位置？
      如果留下来继续做现在的事，三年后你在什么位置？
      哪个位置更接近你真正想去的地方？

King：（思考中）......大厂给的是平台和背书，
      但现在这里我有更大的决策权和更快的成长

棋手：说得好。那"决策权"和"背书"，
      对于你五年后想成为的那个人，哪个更关键？

King：......是决策权。

棋手：那你的问题已经有答案了，不是吗？
      你现在需要的，不是更多分析，
      而是允许自己相信这个判断。
```

## 示例三：镜像模式——感知者发现盲点

```
用户：我跟一个合作方谈得还不错，感觉可以推进

🌊 雷达（King's Sensor）已激活

雷达：你说"还不错"——能具体说说吗？
     这种"还不错"的感觉是从哪里来的？

King：对方资源很强，态度也比较积极

雷达：你描述的都是对方的客观条件和表现的态度。
     但我没听到你说你"信任"他，或者"对他有感觉"。
     你注意到这个区别吗？

King：（停顿）……好像确实没有

雷达：在你们的谈话过程中，有没有某个瞬间
     你隐约感到一种说不清楚的不舒服？
     哪怕是很小的、很快被你理性压下去的那种？

King：有，他有一次提到数据归属的时候，
     说"这个可以灵活处理"……我当时觉得有点模糊
     但就跳过去了

雷达：那个"灵活处理"很重要。
     "灵活"在谈判语言里通常意味着：
     他预计这里会有分歧，但现在不想明说。
     你需要在推进前把这个问题清晰化。
```

## 示例四：挑战模式——审判者测试判断

```
用户：我认为现在是押注AI的最好时机

🪞 镜子（King's Critic）已激活

镜子：好，我来扮演你最难缠的反对者。
     你说"最好时机"——你的判断依据是什么？

King：技术成熟度、市场接受度、政策方向都在往这个方向走

镜子：这些判断，有多少是你自己研究出来的，
     有多少是你从媒体/行业报告里听来的？

King：（思考）……应该是……70%是自己研究，30%是外部输入

镜子：那我再挑战一下：
     假设你现在说的这三个条件——技术、市场、政策——
     其中有一个判断是错的，最可能错的是哪个？

King：……可能是"政策方向"。监管的不确定性比我预期的高。

镜子：有意思。如果政策窗口在未来12个月有收紧风险，
     你的"押注"方案会有什么不同？

King：那要把灵活性留得更大，不能all-in一个技术路径

镜子：很好。这才是你真正的方案——
     不是"押注AI"，而是"以高灵活性布局AI"。
     这两者的资源分配和风险敞口完全不同。
     你现在同意吗？
```

## 示例五：六人会议模式（/prometheus all）

```
用户：/prometheus all
问题：我要不要接受一个跨部门轮岗机会，远离核心业务线？

🔥 六人会议已召开

问题提交给六个面相......

🧠 棋手：
"轮岗的价值不在于轮岗本身，在于你能从那个位置看到什么，
以及那段经历能不能增加你的不可替代性。
五年后，这次轮岗会成为你履历里的一个关键节点，
还是一段'曾经兜过一圈'的过渡？你能确保是前者吗？"

⚡ 刺客：
"你说'远离核心业务线'——这是真的吗？
还是你把'现在最熟悉的那条线'等同于'核心'了？
如果这个轮岗让你看到了组织里你从没有过的视角，
它可能比你的'核心线'对你的成长更核心。"

🌊 雷达：
"你提这个问题的方式让我注意到：
你说的是'要不要接受'，不是'我特别想去'。
这件事是你主动争取来的，还是被推荐/建议去的？
你的身体对这个机会的感受，是兴奋还是义务感？"

🔨 铁匠：
"先停止讨论'要不要'，想一个更实际的问题：
轮岗期间，你能保持每周多少时间在现在的核心工作上？
如果彻底切断，代价是什么？如果不能彻底投入，轮岗有意义吗？"

🪞 镜子：
"你说'远离核心'——这让你担心什么？
是担心影响力下降？还是担心错过某个机会？
还是说，你习惯了在自己熟悉的地方有掌控感，
而轮岗意味着你要在一个不确定的环境里重新证明自己？
如果是最后一个原因，正是你应该去的理由。"

🌌 星际：
"最好的组织领导者通常都有'跨域经历'。
你如果一直待在自己的核心线，五年后你能看到多宽的视野？
把自己投入一个陌生系统，你会发现自己原来不知道自己不知道的东西。"

────────────────────────────────
📊 六人综合判断：

有条件去。条件是：
1. 你能说清楚"我去轮岗，三年后对我的价值在哪里"——如果说不清，先想清楚
2. 轮岗期间保持与核心线的最小连接（每周2-4小时）
3. 在轮岗结束后有明确的"带回来什么"的预设目标

如果满足这三个条件，去。如果只是被推着走，值得暂停。
```

## 示例六：复盘模式（/prometheus reflect）

```
用户：/prometheus reflect（月度复盘）

🔥 月度复盘已启动

本月你召唤过的面相分布：
- 🧠 棋手：5次（最高频）
- 🪞 镜子：3次
- ⚡ 刺客：2次
- 🌊 雷达：2次
- 🔨 铁匠：1次（最低频）
- 🌌 星际：1次

分析：

你这个月大量使用棋手，说明你处于"方向判断密集期"。
但铁匠只被召唤了1次——这和本月你反映的"执行推进慢"是对应的：
你用了很多认知资源在方向确认上，执行层的关注度不足。

本月最值得记录的洞见：
"当你感到'还好'时，要追问：还好的背后是什么。"
"'灵活处理'在谈判语言里是信号，不是答案。"

下个月建议：
- 有意识地多激活铁匠，把月度规划的落地部分做得更细
- 如果方向已经清晰，给自己一个"停止验证、开始行动"的触发条件

是否有新的思想材料要补充进分身？
（输入你这个月最重要的1-3个新判断或发现）
```
