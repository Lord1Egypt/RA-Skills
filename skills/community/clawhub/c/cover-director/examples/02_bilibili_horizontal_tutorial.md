# 示例 02：B站横版教程封面

## 输入

- 标题：我把 AI 写作流程做成了自动化，一篇文章 10 分钟搞定
- 平台：B站
- 比例：16:9
- 领域：AI 自媒体
- 目标：点击 + 收藏
- 参考图：有，想参考大标题 + 人物 + 流程截图结构

## 输出摘要

### 标题策略

- 类型：结果展示型 + 教程型
- 最大关键词：10分钟搞定
- 辅助信息：AI 写作自动化
- 情绪：兴奋、自信
- 手势：指向流程截图或标题

### 构图

人物放右侧，左侧大标题，中间偏下放 3 个流程卡片：选题 → 成稿 → 发布。背景为深蓝科技桌面或抽象界面。

### 无字底图提示词

```text
生成一张 B站 16:9 横版无字封面底图。右侧是真实知识博主人像，半身，表情自信兴奋，一只手指向左侧标题区域。画面中下方有简洁的 AI 工作流卡片视觉元素，但不包含任何文字，只用图标和线条表达流程。背景为深蓝科技感渐变，带轻微界面光效，左侧留出大标题空间。商业摄影质感，高对比度，高清清晰。不要生成文字、水印、logo、乱码。
```

### 排版

- 第一行小标签：AI 写作自动化
- 第二行最大词：10分钟搞定
- 第三行辅助：一篇文章全流程

### 质检预估

- 评分：88/100
- 修改项：流程卡片不要太多，否则 B站缩略图里会乱。
