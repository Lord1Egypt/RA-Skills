## Description: <br>
Browser Site Profile is a shortcut for turning an already proven site flow into a stable, reusable external site profile. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[allens0104](https://clawhub.ai/user/allens0104) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and Copilot users use this skill to create reusable external site profiles, classify blocked or special-case sites, and validate heuristic and Gemini planning flows after a site flow is already proven. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The generated or updated site profile may encode incorrect assumptions about a website flow. <br>
Mitigation: Review any reusable profile before relying on it and validate both heuristic and Gemini planning flows. <br>
Risk: Site probing may be inappropriate on sites where the user lacks authorization. <br>
Mitigation: Use the skill only on sites the user is authorized to test. <br>
