# Taobao Shopping Skill v2.0

淘宝购物助手 Skill，支持公开信息下的搜索、比价、看评论、店铺/SKU 风险判断和下单前核对清单。登录、加购、结算、提交订单和支付留给用户控制。

## 功能特性

- 🔍 **商品搜索**：关键词搜索，支持筛选和排序
- 📖 **商品浏览**：浏览分类、推荐商品
- 📄 **商品详情**：查看规格、图片、价格、促销
- 💰 **价格对比**：多店铺/多规格比价
- 🛒 **购物车指导**：整理确认好的 SKU 和手动加购前核对项
- 🎫 **优惠券**：自动检查并应用可用优惠券
- 📋 **下单前清单**：提示用户手动核对最终到手价、地址、优惠、库存、退换和发票/保修

## 安全边界

| 操作 | Agent | 用户 |
|------|-------|------|
| 搜索/浏览 | ✅ | - |
| 比价/看评论 | ✅ | - |
| 加购/用券 | 手动指导 | ✅ |
| 地址/结算/提交订单/支付 | ❌ | ✅ |

## 工作流程

```
发现 → 对比 → 选择 → 风险核对 → 手动加购/结算/支付
```

## 店铺类型优先级

1. **天猫旗舰店** ⭐⭐⭐⭐⭐ - 品牌直营，最可靠
2. **天猫** ⭐⭐⭐⭐⭐ - 品牌授权，正品保障
3. **企业店铺** ⭐⭐⭐⭐☆ - 认证企业，较可靠
4. **个人皇冠店** ⭐⭐⭐☆☆ - 经验丰富，需谨慎
5. **普通个人店** ⭐⭐☆☆☆ - 价格优势，风险较高

## 价格计算公式

```
到手价 = 标价 - 店铺券 - 平台券 - 满减 - 淘金币 - 支付优惠
```
