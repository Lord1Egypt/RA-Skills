# 社保权益查询与维权辅助技能

## 概述

本技能为普通劳动者提供社保权益查询与维权全流程辅助，覆盖以下核心环节：
1. **违法情形研判** — 评估用人单位社保缴纳行为是否违法
2. **查询渠道指引** — 指导用户查询社保缴费记录
3. **维权途径对比** — 社保稽核/劳动监察/劳动仲裁三种途径对比
4. **材料清单生成** — 根据维权途径生成所需材料清单
5. **经济补偿计算** — 因社保问题被迫解除时可主张的经济补偿金
6. **断缴影响评估** — 评估社保断缴对各项权益的影响

支持全国各地区，内置主要城市缴费比例和补缴规则差异参考。

## 工作流程

### 总流程

```
用户咨询 → 信息收集 → 违法研判 → 查询指引 → 维权途径选择 → 材料清单 → 经济补偿计算
```

### 阶段一：信息收集

**必采信息：**
1. 用户所在地区（省/市）
2. 用人单位全称
3. 入职日期和离职日期（如已离职）
4. 实际月工资标准
5. 社保缴纳情况（是否缴纳、缴了哪些险种、缴费基数、缴费起止月份）

**按需采集：**
- 是否签订劳动合同
- 试用期是否缴纳社保
- 是否有断缴记录
- 断缴原因（离职/公司未缴/协商不缴）

### 阶段二：违法情形研判

加载参考文档：`references/social_security_guide.md`

根据用户描述判断违法类型：

| 违法类型 | 表现 | 法律依据 |
|---------|------|---------|
| 不缴社保 | 完全不缴纳 | 社保法第57、60条 |
| 少缴社保 | 按最低基数而非实际工资 | 社保法第60条 |
| 滞后缴纳 | 入职数月后才开始缴 | 社保法第58条 |
| 险种不全 | 只缴部分险种 | 社保法相关条款 |
| 试用期不缴 | 以试用期为由不缴 | 社保法第58条 |

**研判输出：**
- 违法情形定性
- 违法起止期间
- 少缴金额估算
- 可主张权利清单

### 阶段三：查询渠道指引

指导用户通过以下渠道查询社保缴费记录：

1. **掌上12333 App** — 下载并实名认证，查询缴费明细
2. **电子社保卡** — 微信/支付宝搜索"电子社保卡"
3. **当地社保局官网** — 注册登录查询
4. **12333热线** — 电话咨询

查询要点：核对参保状态、缴费起止月份、缴费基数、险种、缴费单位、欠费记录。

### 阶段四：维权途径选择

加载参考文档：`references/social_security_guide.md` 第四章

对比三种维权途径：

| 途径 | 适用场景 | 优势 | 劣势 |
|------|---------|------|------|
| 社保稽核投诉 | 未缴/少缴/断缴 | 直接补缴 | 不能主张经济补偿 |
| 劳动监察投诉 | 未缴社保 | 行政介入 | 仅2年内 |
| 劳动仲裁 | 被迫解除+经济补偿 | 可拿补偿金 | 社保补缴部分可能不受理 |

**建议策略：**
- 仅需补缴社保 → 社保稽核投诉（首选）
- 需要补缴 + 经济补偿 → 社保稽核 + 劳动仲裁并行
- 用人单位拒绝补缴 → 劳动监察投诉 + 社保稽核

### 阶段五：材料清单生成

根据选择的维权途径，生成个性化材料清单：

**社保稽核投诉必备：**
1. 身份证原件及复印件
2. 劳动合同
3. 社保缴费记录截图
4. 银行工资流水
5. 入职/离职证明

**劳动仲裁额外需：**
6. 《被迫解除劳动合同通知书》及EMS送达凭证
7. 催缴社保的沟通记录

### 阶段六：经济补偿计算

使用脚本：`scripts/calculate_social_security.py`

如用户选择被迫解除路径，计算经济补偿金：

```
经济补偿金 = 工作年限 × 月平均工资
```

同时计算社保少缴金额估算：
- 养老保险少缴 = (实际工资 - 缴费基数) × (单位比例 + 个人比例) × 月数
- 医疗保险少缴 = 同上
- 其他险种同理

## 关键规则与提醒

### 法律依据快速索引

| 问题 | 法律依据 |
|------|---------|
| 用人单位缴费义务 | 《社会保险法》第57、58、60条 |
| 社保稽核 | 《社会保险法》第63条 |
| 劳动监察 | 《劳动保障监察条例》第11条 |
| 被迫解除 | 《劳动合同法》第38条第1款第3项 |
| 经济补偿金 | 《劳动合同法》第46、47条 |

### 重要时效提醒
- 社保稽核：原则上无时效限制（各地实操不同）
- 劳动监察：2年内
- 劳动仲裁：1年（从离职之日起算）
- 社保补缴滞纳金：每日万分之五

### 使用注意事项
1. 本技能提供的信息仅供参考，不构成法律意见
2. 各地社保政策可能更新，提交前建议拨打12333确认
3. 社保补缴规则各地差异大，务必查询当地最新规定
4. 复杂案件建议咨询专业律师或申请法律援助（12348）
