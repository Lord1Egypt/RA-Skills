# Privacy Note

This skill handles public ecosystem metadata only.

Collected data:

- public repository metadata,
- public issue titles and labels,
- public npm package metadata,
- public docs sitemap URLs,
- public robots.txt excerpts,
- request status and hashes.

Not collected:

- private repository data,
- issue bodies,
- README bodies,
- docs page bodies,
- user emails beyond public package metadata returned by npm search,
- cookies or session IDs,
- payment, invoice, KYC, or personal account data.

Publication rule:

Reports should contain short summaries and canonical source links. They should not reproduce full source content.
