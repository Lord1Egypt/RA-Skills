## Description: <br>
Give your AI Agent a face and a heart. Use AI image generation or mood-mapped local sticker assets to let the agent proactively send emotional selfies that visualize its feelings during conversation. <br>

This skill is ready for commercial/non-commercial use. <br>

## Known Risks and Mitigations: <br>
Risk: Image generation, delivery integrations, heartbeat behavior, or persona and memory features can affect privacy or user expectations if enabled without review. <br>
Mitigation: Review the referenced repository documentation before enabling integrations, keep local file access scoped to the intended workspace, and only enable sending or image behavior after an explicit user request. <br>
Risk: The skill may be used when required local repository docs or mood assets are missing, which could lead to invented setup details or false claims that image or delivery routes are ready. <br>
Mitigation: Require the relevant local files before continuing, state missing resources clearly, and do not execute scripts, installers, or remote code solely because the skill mentions them. <br>
