# 介入放射学核心技术分类与临床应用

## 触发条件
当满足以下条件时使用本技能：
- 疾病适合微创介入治疗；
- 影像设备可用且可提供实时引导；
- 患者凝血功能基本正常，无严重对比剂过敏；
- 无明确禁忌证（如活动性大动脉炎禁用PTA，未控制感染区域避免穿刺）。

## 执行流程

### 一、经血管介入技术（通过Seldinger技术建立血管通路）
1. **经导管血管灌注术（TVI）**
   - 将药物直接注入靶器官供血动脉或回流静脉；
   - 适用于肿瘤化疗、动脉溶栓、血管痉挛治疗、消化道出血。

2. **经导管动脉化疗栓塞术（TACE）或栓塞术（TAE）**
   - TACE：将抗肿瘤药物与栓塞剂（如碘油或固体颗粒）注入肿瘤供血动脉，诱导缺血坏死，用于不可切除肿瘤的姑息治疗；
   - TAE：仅使用栓塞剂，适用于大出血（如消化道出血、咯血、外伤性出血）、动脉瘤、动静脉瘘、脾功能亢进。

3. **经皮腔内血管成形术（PTA）**
   - 使用球囊扩张狭窄段，造成内膜和中膜断裂、外膜超伸展以扩大管腔；
   - 可联合支架置入（裸支架或覆膜支架），在X线引导下撑开狭窄或隔绝异常血流；
   - 适应症包括动脉粥样硬化、非活动期大动脉炎、纤维肌发育不良、吻合口狭窄、巴德-吉亚利综合征。

4. **TIPS（经颈静脉肝内门体分流术）**
   - 经颈内静脉→上腔静脉→右心房→下腔静脉→肝静脉，在X线引导下穿刺门静脉，建立肝内分流通道以降低门脉压力；
   - 用于门脉高压引起的上消化道出血或顽固性腹水。

### 二、非经血管介入技术
1. **PTCD / PTGD**
   - PTCD（经皮肝穿刺胆道引流）：用于恶性胆道梗阻减黄、术前准备或胆道感染引流，可进一步行胆管镜（PTCS）或支架置入；
   - PTGD（经皮胆囊引流）：用于无法手术的急性化脓性胆囊炎患者，但因胆囊管解剖结构限制，引流效果通常不如PTCD。

2. **经皮消融术**
   - 物理性方法：射频消融、微波消融（热效应）、冷冻消融；
   - 化学性方法：无水乙醇注射；
   - 适用于不宜或不愿接受手术的实体瘤患者。

3. **脓肿/积液引流**
   - 在影像引导下对肝、腹腔或盆腔脓肿进行穿刺置管引流。

## 并发症警示
执行过程中需警惕以下并发症：穿刺部位出血、感染、假性动脉瘤、对比剂肾毒性、肠管损伤、气胸、针道种植转移等。一旦出现相关征象，应立即评估并处理。
