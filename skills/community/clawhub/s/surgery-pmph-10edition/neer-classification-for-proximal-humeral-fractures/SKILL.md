# Neer分型评估肱骨近端骨折

## 适用条件
- 患者诊断为肱骨近端骨折
- 已完成X线或CT检查
- 明确识别四个解剖部位：肱骨头、大结节、小结节、肱骨干

## 执行步骤
1. **确认解剖结构**：在影像上明确肱骨近端的四个关键解剖部位。
2. **评估移位标准**：判断任意两个相邻部位之间是否存在以下任一情况：
   - 骨折块间移位 > 1 cm，或
   - 成角畸形 > 45°。
3. **确定骨折分型**：
   - **一部分骨折**：所有部位均未达移位标准，软组织附着完整，稳定性良好。
   - **两部分骨折**：仅一个解剖部位发生移位，包括解剖颈、大结节、小结节或外科颈骨折。
   - **三部分骨折**：两个解剖部位移位，常见组合为大结节+外科颈或小结节+外科颈。
   - **四部分骨折**：四个部位全部移位，肱骨头游离，血供严重受损，预后差。
4. **记录结果**：输出骨折分型（一部分/两部分/三部分/四部分），用于后续治疗方案制定。

> **注意**：未达移位标准的骨折线不计入分型；四部分骨折提示高风险缺血性坏死，需紧急处理。
