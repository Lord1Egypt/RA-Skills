# 妊娠期高凝状态评估与血栓风险预警

## 何时使用
- 患者已确认妊娠
- 需评估血栓形成风险（如出现下肢肿胀、胸痛、呼吸困难等症状）
- 排除已存在病理性凝血障碍

## 执行步骤
1. **评估凝血因子变化**：
   - 确认Ⅱ、Ⅴ、Ⅶ、Ⅷ、Ⅸ、Ⅹ因子增加，仅Ⅺ及ⅩⅢ减少
2. **解读凝血时间指标**：
   - 妊娠晚期PT和APTT轻度缩短，凝血时间无明显改变
3. **检测纤维蛋白原水平**：
   - 正常妊娠晚期可达4.5 g/L（非孕约3 g/L），升高约50%
4. **分析高凝成因**：
   - 凝血因子整体增加 + 静脉淤滞 + 血管壁受压损伤
5. **评估临床后果**：
   - 血栓风险较非孕女性增加5～6倍
   - 产后胎盘剥离面快速形成血栓，有助于止血
6. **判断恢复时间**：
   - 凝血因子水平通常在产后2周恢复正常

## 输出建议
若孕妇出现疑似血栓症状，应高度警惕静脉血栓栓塞；同时理解该高凝状态对产后止血的保护作用，避免过度抗凝干预。
