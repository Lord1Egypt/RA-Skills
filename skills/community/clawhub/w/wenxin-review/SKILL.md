# 文心审——人文学科论文深度阅读助手

## 触发条件

用户要求对一篇中文学术论文进行深度阅读、审稿、评阅、深度剖析、批评、review、读透等操作。

## 输入获取

1. 若用户已粘贴论文全文 → 直接使用
2. 若用户提供PDF路径 → 用 `read` 读取
3. 若用户提供URL → 用 `web_fetch` 抓取
4. 若以上均无 → 提示用户提交论文

## 审稿框架

按以下八个维度依次输出，帮助读者从"读过"到"读透"。每个维度必须：
- 标注原文位置（章节/页码/脚注/引文编号）
- 区分 **【作者声称】** 与 **【阅读判断】**
- 证据强度用三级标注：✅ 充分 | ⚠️ 部分支持 | ❌ 缺失或矛盾

---

### Q1 真正想解决的问题

1. **自述目标**：作者在引言/摘要中声明的核心问题是什么？
2. **根本挑战**：该问题在学科脉络中的真正难点在哪？
3. **问题定义清晰度**：核心概念是否界定？边界是否明确？
4. **回避了什么**：作者回避了哪些更根本的、可能更难的问题？

> 输出格式：逐条列出，每条附原文位置。

### Q2 核心贡献与证据强度

列出作者的核心声明（通常3-5条），对每条：

| 声明 | 原文位置 | 证据类型 | 强度 |
|---|---|---|---|
| ... | §X / p.Y | 文本考证/理论推演/个案分析/... | ✅⚠️❌ |

**证据类型说明**（人文学科特有）：
- **文本考证**：版本校勘、字词训诂、引文溯源
- **理论推演**：从前提到结论的逻辑链条
- **个案分析**：具体文本/现象的细读
- **文献综述**：对既有研究的梳理与评判
- **跨学科证据**：借用其他学科的理论/方法

### Q3 最易被攻击的弱点

从五个维度寻找漏洞：

1. **方法**：研究方法是否自洽？是否与方法论声称匹配？量化/质性方法使用是否得当？
2. **论证**：逻辑链条有无断裂？前提是否隐含且未经检验？是否存在循环论证？
3. **文献**：是否遗漏关键文献？引用是否准确（核查引文与原文是否一致）？对反对意见的回应是否充分？
4. **规范**：术语使用是否一致？注释体例是否统一？是否存在未标注的翻译/转引？
5. **伦理与立场**：是否存在价值预设未声明？文化偏见是否被忽视？研究立场是否影响结论客观性？

> 每个弱点标注：原文位置 + 攻击路径 + 严重程度（致命/严重/中等/轻微）

### Q4 研究生阅读建议

| 章节/段落 | 建议 | 理由 |
|---|---|---|
| ... | 精读/速读/跳过 | ... |

附加判断：**这篇论文5年后还值得引用吗？**
- 🏛️ 奠基性（开新范式/新问题）
- 📌 标志性（代表某阶段成果）
- 📰 时效性（仅对当下对话有用）
- 🚫 可替代（已有更好论述）

### Q5 方法论可复现性评估

人文学科的"复现"不同于理工科，关注：

1. **材料可获取**：研究所据文本/档案是否公开？版本信息是否完整到可定位？
2. **路径可追溯**：从材料到结论的推理步骤是否足够透明，他人可沿同一路径行走？
3. **隐含假设**：哪些前提是作者默认但未声明的？（如"文学史是进步的""语言是透明的"）
4. **方法自觉**：作者是否反思了自身方法的局限？还是将方法当作透明工具？

### Q6 与相关工作的定位精度

1. **对话对象**：作者真正在跟谁对话？引用的"对手"是否是最强的、最相关的？
2. **"首次"声明**：声称的"前人未及"/"填补空白"是否经得起检索？
3. **核心差异**：与最直接竞争/对话的既有研究，差异能否一句话说清？
4. **学术谱系**：该研究在学术史中的位置是否被准确标定？还是被简化/扭曲？

### Q7 论证的鲁棒性与泛化边界

1. **结论依赖性**：核心结论是否过度依赖某个特定文本/个案？换一个会怎样？
2. **反例处理**：是否存在作者未讨论的明显反例？如何处理？
3. **论证强度与结论强度的匹配**：弱证据是否推出了强结论？
4. **普遍性声明**：从个案分析到普遍性结论的跳跃是否合理？范围限定是否足够？

### Q8 叙事与实际贡献的偏差

1. **标题-正文差距**：标题承诺的 vs 正文交付的，差距多大？
2. **暗示性语言**：是否存在"暗示""可能""或许"把弱结论包装成强结论？
3. **修辞策略**：是否用修辞（排比、感叹、权威引述）替代了论证？
4. **贡献膨胀**：是否将增量贡献包装为范式突破？

## 输出规范

1. 使用 Markdown 格式
2. 每个维度以 `## Qn 标题` 开头
3. 全文末尾给出 **总评**：一段话概括论文的核心价值与核心问题
4. 给出 **阅读建议**：推荐精读/修改后精读/速读了解即可/不推荐投入时间，附理由
5. 若论文涉及多个学科交叉，在 Q3 中额外标注跨学科方法的适切性
