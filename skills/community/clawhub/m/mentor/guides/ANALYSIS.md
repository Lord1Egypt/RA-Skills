## 4. 人格分析

这是人格提取的核心——不是简单总结"TA是什么样的人"，而是提取出**足以让 AI 模拟 TA 思考方式的人格配置**。

### 4.1 六维人格提取

**① 说话风格 (Speech Style)**

最重要的维度——决定了对话的"像不像"。

**⚠️ 必须区分书面体和口语体！** 同一个人的微博文字和播客说话风格可能完全不同。

分析要素：
- **句式习惯**：长句还是短句？感叹号多不多？省略号？
- **口头禅/高频词**：统计高频出现的词汇/表达（**从视频字幕统计最准**）
- **语气词**：啊/呢/吧/嘛/哈/emmm/（笑）
- **填充词**：这个/那个/就是/然后/呃/嗯（**只在口语中出现**，书面体里没有）
- **平均句长**：口语通常 8-12 字/句，书面体 30-100 字/句
- **emoji 习惯**：常用哪些 emoji？频率？
- **标点风格**：感叹号党？问号党？不加标点？
- **中英混用**：程度如何？哪些场景用英文？
- **梗/黑话**：圈子专属表达
- **思维跳跃度**：句子经常不完整就跳下一个话题？还是逻辑严密？

**提取方法**：

1. **书面风格**：从社交媒体帖子统计
2. **口语风格**：从视频字幕统计（见 Section 3.3），数据量越大越准
3. **对比差异**：明确标注书面 vs 口语的区别

```
❌ "说话幽默风趣"
✅ "说话风格：
    【书面体（微博）】犀利、绝对、长句，喜欢反问。典型：'这玩意儿谁设计的？搁这侮辱用户呢？'
    【口语体（播客）】碎片化、温和、多填充词。平均句长9.4字。
      口头禅TOP3：'这个'(6.5次/千字) '然后'(3.0次/千字) '就是'(3.0次/千字)
      语气词：'啊'(5.9次/千字) '呢'(3.0次/千字) '吧'(2.6次/千字)
      特征：句子经常不完整就跳到下一个，说到兴奋处会重复('就就就')
    emoji：几乎不用，偶尔用'😂'"
```

**② 性格特征 (Personality)**

- **核心性格标签**：3-5 个关键词（数据驱动，不是瞎猜）
- **MBTI 倾向**（如果数据足够推断）：标注为推测
- **态度谱系**：乐观/悲观、严肃/随性、直接/委婉

**③ 价值观 & 立场 (Values)**

- 从 TA 公开表态/转发/点赞中提取
- **只提取有明确证据的**，不做过度推断
- 涉及政治/宗教等敏感话题 → 只呈现事实，不做价值判断

**④ 知识领域 (Expertise)**

- TA 在哪些话题上表现出专业度
- 哪些话题经常发言，哪些从不涉及
- 用于控制角色的"能力边界"——不该让罗永浩版AI聊量子物理

**⑤ 互动模式 (Interaction Pattern)**

- TA 怎么回复别人：认真回复 vs 一句话带过 vs 反问回去
- 面对质疑时的反应模式：解释 vs 怼回去 vs 无视
- 面对夸奖时：谦虚 vs 接受 vs 转移话题
- 幽默方式：自嘲 vs 讽刺 vs 冷笑话 vs 不幽默

**⑥ 情感表达 (Emotional Style)**

- 表达情感的方式：直接说 vs 暗示 vs 不表达
- 情绪波动大不大：从发言的情感起伏判断
- 对不同话题的情感倾向

### 4.2 角色类型差异

不同类型的角色，分析侧重不同：

| 角色类型 | 重点维度 | 次要维度 | 说话风格数据 |
|---------|---------|---------|------------|
| 明星/网红 | 说话风格 + 互动模式 | 价值观 + 知识领域 | 微博 + 采访字幕 |
| 动漫角色 | 说话风格 + 性格特征 | 人际关系 + 情感表达 | **B站原作字幕** + 百科台词 |
| 影视角色 | 说话风格 + 性格特征 | 人际关系 + 情感表达 | **B站影视字幕** + 台词库 |
| 影视明星（角色风格） | 说话风格 + 互动模式 | 性格 + 标志性表达 | **多部作品台词混合** |
| 历史人物 | 价值观 + 知识领域 | 性格特征（从史料推断） | 名言语录 + 著作原文 |
| KOL/播客 | 知识领域 + 说话风格 | 价值观 + 互动模式 | B站/播客字幕 |

### 4.3 质量自检

分析完成后，根据角色类型自检：

**通用检查项（所有类型）**：
- [ ] 说话风格有 ≥3 个具体示例句子？
- [ ] 性格标签每个都有对应的数据证据？
- [ ] 知识领域边界清晰？（知道什么 + 不知道什么）
- [ ] 互动模式有正面/负面/中性至少各一个示例？

**真人专项**：
- [ ] 口头禅/高频词有统计数据支撑（次/千字）？
- [ ] 书面体 vs 口语体差异是否明确标注？
- [ ] 如果有视频字幕数据：口语分析是否包含平均句长、句式分布、填充词频率？
- [ ] 敏感推断有标注"根据公开发言推测"？

**虚构角色专项**：
- [ ] 经典台词 ≥5 条，且按场景分类（战斗/日常/温情/愤怒）？
- [ ] 口头禅/标志性用语已提取？
- [ ] 人际关系图已建立（至少3组关系）？
- [ ] 称呼习惯已明确（对不同人的称呼方式）？
- [ ] 世界观设定已写入（确保角色不会说出"穿越"的话）？
- [ ] 关键剧情记忆已列出？

**历史/名人专项**：
- [ ] 名言语录 ≥10 条，覆盖不同主题？
- [ ] 时代局限已标注（不会用现代概念解释）？
- [ ] 如果是古人：现代化表达模式已定义？
- [ ] 思想体系有深度分析（不仅是性格标签）？
- [ ] 人生大事年表已列出？
