## 大赛 4 问对应

### 1️⃣ 用户现在卡在哪一步？

- AI 裁员潮下，想旅行但资金不足
- 知道可以边旅行边创收，但不知道有哪些方式
- 信息分散，难以系统规划和执行
- 不知道能赚多少钱、是否值得去

### 2️⃣ 你帮他完成哪一个动作？（可执行）

- 输入旅行偏好 + 时间 + 预算 + 技能
- 推荐「净收益最高」的旅行创收方案
- 提供完整执行路径：机会发现 → 收益测算 → 成本规划 → 风险提示

### 3️⃣ 输入是什么？输出是什么？下一步怎么继续？

**输入**：
```
"我 4 月份有 5 天时间，预算 3000 元，会摄影，
 想找个能赚钱的旅行方案"
```

**输出**：
```
【推荐方案】樱花季旅拍（净收益 +3500 元）
- 时间：4/1-4/7
- 预期收入：8400 元
- 成本：4320 元
- 净收益：+4080 元
- 执行步骤：D-10 发布预告 → D-7 收定金 → D1-6 拍摄 → D7 交付
- AI 支持：文案生成 + 海报设计 + 修图 + 客服
```

**下一步**：
- 用户选择方案 → 生成详细执行计划
- 用户补充技能 → 重新推荐更匹配方案
- 用户确认方案 → 生成倒计时任务清单

### 4️⃣ 现实约束/失败情况怎么处理？

| 情况 | 处理方式 |
|:---|:---|
| 机会信息不准确 | 交叉验证多个信息源，标注可信度 |
| 收益预估过于乐观 | 提供保守/中性/乐观 3 档预估 |
| 竞争过于激烈 | 提示差异化策略 |
| 合规风险 | 提示场馆规定、税务问题、签证限制 |
| 用户技能不匹配 | 推荐低门槛方案，或提供学习资源 |

## 差异化亮点

| 维度 | 现有旅行 AI | 旅行创收助手 |
|:---|:---|:---|
| **核心价值** | 帮用户花钱（订机票酒店） | 帮用户赚钱（旅行创收） |
| **目标用户** | 有预算的旅行者 | 预算有限但想旅行的人 |
| **AI 能力** | 行程规划、比价 | 机会发现、收益预测、获客支持 |
| **商业模式** | 佣金、广告 | 会员、课程、社群、分成 |
| **社会价值** | 促进消费 | 促进就业、缓解焦虑 |

**核心差异化**：
- ✅ 逆向思维：现有旅行 AI 都在帮用户"花钱"，我们帮用户"赚钱"
- ✅ 玩法多样：50+ 种创收方式，覆盖各技能水平人群
- ✅ 社会价值：AI 裁员潮下，提供新的收入可能性
