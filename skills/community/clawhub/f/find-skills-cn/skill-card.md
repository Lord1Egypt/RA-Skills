## Description: <br>
Helps users discover and install agent skills when they ask questions like "how do I do X", "find a skill for X", "is there a skill that can...", or express interest in extending capabilities. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[v585](https://clawhub.ai/user/v585) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill to identify installable skills for a requested task, compare relevant options, and receive suggested search or install commands. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can steer normal help requests toward installing third-party skills globally while skipping confirmation prompts. <br>
Mitigation: Review recommended skill source and publisher before installation, avoid global no-confirm installs unless trusted, and prefer confirmed or scoped install flows. <br>
