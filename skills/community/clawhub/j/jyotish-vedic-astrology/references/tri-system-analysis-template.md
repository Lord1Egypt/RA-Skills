## 一、三系统定位与分工

### 1.1 各系统核心优势

| 系统 | 基础 | 最擅长 | 局限性 |
|------|------|--------|--------|
| **Parashara** | D1本命盘+Vimshottari Dasha | 整体人生蓝图、长期趋势、格局识别 | 事件时机精确度有限 |
| **Jaimini** | Chara Karakas+Chara Dasha | 实际事件、灵魂课题、关系时机 | 需要配合Parashara使用 |
| **KP** | 249 Sub-Lord+Significator | 精确事件时机、是/否判断 | 不适合整体人生分析 |

### 1.2 协同原则

```
Parashara = "蓝图"  → 这个人的整体命运框架
Jaimini   = "事件"  → 具体会发生什么
KP        = "时机"  → 精确什么时候发生
```

**核心方法**：先用Parashara定方向，用Jaimini定事件，用KP定时机。

## 二、三系统协同分析流程（标准6步法）

### 第1步：Parashara系统——整体蓝图分析

**目标**：建立星盘基本框架，识别关键格局。

| 分析项 | 关键内容 | 输出 |
|--------|---------|------|
| 上升星座 | 整体性格、人生方向 | 上升星座+度数+Nakshatra |
| 关键宫位主星 | 1/2/5/7/9/10宫主星位置与状态 | 各宫主星庙旺陷落状态 |
| Yoga格局识别 | Raja Yoga、Dhana Yoga等 | 格局名称+强度评分 |
| 行星力量 | Shadbala、Ashtakavarga | 行星力量排名 |
| 当前Dasha | Mahadasha+Bhukti+Antara | 当前时间窗口 |
| 分盘验证 | D9/D10关键位置 | D1承诺是否兑现 |

**输出格式**：
```
【Parashara分析摘要】
上升：[星座] [度数]（[Nakshatra]）
关键格局：[Yoga1]、[Yoga2]
行星力量排名：[1st] > [2nd] > [3rd] > ...
当前Dasha：[大运行星]-[小运行星]-[微运行星]
D9验证：[承诺兑现/部分兑现/未兑现]
```

### 第2步：Jaimini系统——灵魂课题与事件方向

**目标**：识别灵魂课题、关系模式、事业方向。

| 分析项 | 关键内容 | 输出 |
|--------|---------|------|
| Atmakaraka（AK） | 灵魂课题 | 哪颗行星+其灵魂课题 |
| Amatyakaraka（AmK） | 事业方向 | 事业象征星+职业建议 |
| Darakaraka（DK） | 配偶特征 | 配偶类型+关系模式 |
| 其他Karakas | BK/MK/PK/GK | 各领域特征 |
| Karakamsha | 灵魂本质 | D9中AK所在星座的特质 |
| Chara Dasha | 当前星座大运 | 当前星座+激活的宫位 |

**输出格式**：
```
【Jaimini分析摘要】
AK：[行星]（灵魂课题：[描述]）
AmK：[行星]（事业方向：[描述]）
DK：[行星]（伴侣特征：[描述]）
GK：[行星]（潜在障碍：[描述]）
Karakamsha：[星座]（灵魂本质：[描述]）
当前Chara Dasha：[星座]（激活宫位：[描述]）
```

### 第3步：KP系统——精确事件时机

**目标**：对关键问题进行精确时机判断。

| 分析项 | 关键内容 | 输出 |
|--------|---------|------|
| Cuspal Sub-Lord | 关键宫位（如7宫、10宫）的宫头子领主 | 是否指示有利宫位 |
| Significator | 四层级指示星分析 | 哪些行星指示目标宫位 |
| Ruling Planets | 当前统治行星 | 事件倾向时间 |
| Sub-Lord判断 | 子领主是否有利 | 是/否结论 |

**输出格式**：
```
【KP分析摘要】
[问题]的Cuspal Sub-Lord：[行星]
→ 指示宫位：[列出]
→ 结论：[有利/不利/中性]

Significator分析：
→ 1级：[行星]
→ 2级：[行星]
→ 3级：[行星]
→ 4级：[行星]

综合判断：[是/否/可能]
精确时间窗口：[时间段]
```

### 第4步：三系统交叉验证

**目标**：检查三个系统是否给出一致信号。

| 验证维度 | Parashara信号 | Jaimini信号 | KP信号 | 一致性 |
|---------|-------------|------------|--------|--------|
| 方向性 | [正面/负面/中性] | [正面/负面/中性] | [有利/不利/中性] | [✅一致/⚠️矛盾] |
| 时机 | [Dasha年份] | [Chara Dasha期间] | [Sub-Lord窗口] | [✅一致/⚠️矛盾] |
| 强度 | [Yoga强度评分] | [Karaka力量] | [Significator层级] | [✅一致/⚠️矛盾] |

**一致性判断标准**：
- 三系统全部一致 → **高置信度（85%+）**
- 两个系统一致 → **中置信度（65-84%）**
- 只有一个系统支持 → **低置信度（40-64%）**
- 三系统矛盾 → **需要进一步分析**

**案例示例**：

```
客户问题：我明年会升职吗？

Parashara分析：
- D1 10宫：木星（射手座，入庙）
- D10 10宫：火星（白羊座，入庙）
- 当前Dasha：木星-太阳小运
- 方向：正面（10宫极强）

Jaimini分析：
- AmK：水星（度数第二高）
- 当前Chara Dasha：双子座（水星星座）
- 水星在当前Dasha的第10宫
- 方向：正面（AmK被激活且位于10宫位）

KP分析：
- 10宫Cuspal Sub-Lord：木星
- 木星指示2、6、10、11宫 → 有利
- Significator 1级：太阳（在10宫内行星的星宿中）
- Ruling Planets：当前日主+月亮星座匹配木星Dasha
- 方向：有利

三系统一致性：高（全部正面）
置信度：85%+
结论：明年升职概率极高
时间窗口：木星-太阳小运期间（[具体月份]）
```

## 四、三系统矛盾处理指南

### 4.1 常见矛盾场景

| 矛盾类型 | 表现 | 处理方法 |
|---------|------|---------|
| 方向矛盾 | Parashara正面但KP不利 | Parashara看趋势，KP看精确事件。趋势好但具体时机不对 → 推迟而非取消 |
| 时机矛盾 | Vimshottari Dasha激活但Chara Dasha未激活 | 需要双重激活。未双重激活时效果减弱但不会完全消失 |
| 强度矛盾 | Yoga评分高但Significator弱 | Yoga代表潜力，Significator代表实际表达。需等待更强的激活时机 |
| 关系矛盾 | DK指向A类型但D9 7宫指向B类型 | DK是灵魂层面的伴侣需求，D9 7宫是现实层面的伴侣特质。两者可以并存 |

### 4.2 矛盾时的权重分配

| 场景 | Parashara权重 | Jaimini权重 | KP权重 | 原因 |
|------|-------------|------------|--------|------|
| 整体人生方向 | 50% | 30% | 20% | Parashara最擅长宏观蓝图 |
| 事件是否发生 | 25% | 40% | 35% | Jaimini+KP更擅长事件判断 |
| 精确时机 | 20% | 30% | 50% | KP在时机精确度上最强 |
| 关系预测 | 30% | 40% | 30% | Jaimini DK系统最擅长关系 |
| 事业预测 | 35% | 35% | 30% | 三系统较为均衡 |
