# Bian Zheng Treatment of Bi Syndrome

## When to Use
- 痹证患者
- 关节肌肉疼痛类疾病（排除急性感染性关节炎及非痹证类疼痛）

## When to Use
IF 患者表现为肢体关节肌肉疼痛、麻木、重着、屈伸不利等症状 THEN 启动本流程。

## Execution Steps

### 1. 辨邪气偏盛类型
- **行痹（风邪胜）**：疼痛游走不定，舌苔薄白，脉浮。
- **痛痹（寒邪胜）**：疼痛剧烈如锥刺，痛有定处，遇寒加重，得热则减，舌淡苔白，脉弦紧。
- **着痹（湿邪胜）**：疼痛重着，肌肉酸楚，关节漫肿，四肢沉重，舌淡苔白腻，脉濡缓。
- **风湿热痹（热邪胜）**：关节红肿灼热，痛不可触，得冷则减，伴发热口渴，舌苔黄燥或黄腻，脉滑数。

### 2. 辨虚实
- **实证**：新发（<3个月），以邪实为主。
- **虚证**：久病（>6个月），气血亏虚或肝肾不足。
- **虚实夹杂**：久病兼痰瘀互结与肝肾亏虚。

### 3. 确定治法
- **实证**：祛邪通络、宣痹止痛，分别采用祛风、散寒、除湿、清热、化痰、祛瘀。
- **虚证或虚实夹杂**：扶正补虚，配以补肝肾、强筋骨、益气血。

### 4. 推荐代表方药（需根据证型加减使用）
- 行痹：防风汤
- 痛痹：乌头汤
- 着痹：薏苡仁汤
- 风湿热痹：宣痹汤
- 痰瘀痹阻：双合汤
- 肝肾两虚：独活寄生汤

### 5. 辅助疗法建议
可结合膏药外敷、针灸、推拿、拔罐、药物熏洗等外治法增强疗效。

## Output
明确输出：
- 具体证型（如行痹、痛痹等）
- 治法原则
- 代表方药
- 是否推荐配合外治法
