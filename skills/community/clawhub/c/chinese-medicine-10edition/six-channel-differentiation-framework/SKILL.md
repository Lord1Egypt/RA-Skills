# Six Channel Differentiation Framework

## When to Use
- 疾病处于外感阶段（如风寒、风热等外邪侵袭）
- 具备六经传变特征（如由表入里、由阳转阴等）
- 需结合八纲辨证（表里、寒热、虚实、阴阳）共同分析

## Execution Steps

1. **确认是否适用**：判断疾病是否为外感伤寒，排除内伤杂病。

2. **识别六经归属**：根据症状判断属于哪一经：
   - **太阳病**：恶寒发热、头项强痛、脉浮（主表，外感初期）
   - **阳明病**：大热、大汗、大渴、脉洪大或便秘腹满（主里，胃肠实热）
   - **少阳病**：寒热往来、胸胁苦满、口苦咽干（主半表半里）
   - **太阴病**：腹满而吐、食不下、自利（属里虚寒）
   - **少阴病**：脉微细、但欲寐、四肢厥冷（属心肾阳虚）
   - **厥阴病**：寒热错杂、消渴、气上撞心（属阴阳交争）

3. **区分三阳与三阴**：
   - **三阳病**（太阳、阳明、少阳）：多为阳证、热证、实证，正气尚盛，以祛邪为主。
   - **三阴病**（太阴、少阴、厥阴）：多为阴证、寒证、虚证，正气不足，以扶正为主。

4. **分析病位与脏腑基础**：
   - 三阳病以六腑病变为基础，三阴病以五脏病变为基础。
   - 注意表里对应关系（如阳明与太阴互为表里）。

5. **评估传变趋势**：结合病邪性质（如风寒、湿热）、正气强弱及当前阶段，预测可能的传变路径（如太阳→阳明，或直中少阴）。

6. **输出辨证结论**：明确所属六经病证类型，并据此推荐经典方剂（如太阳病用桂枝汤或麻黄汤，阳明病用白虎汤或承气汤等）。

> **注意**：本技能必须与八纲辨证联合使用，不可孤立应用。若患者无明确外感病史或传变特征，应优先考虑其他辨证方法。
