# 开题报告/研究计划模板

## 一、开题报告标准结构

```
一、研究题目
二、研究背景与动机
三、文献回顾
四、研究问题
五、研究方法
六、章节安排
七、预期贡献
八、研究进度
九、参考文献
```

## 二、各部分写作指南

### 一、研究题目

- 准确、具体，避免过大过空
- 体现核心研究对象和方法视角
- 可包含副标题，用破折号连接

**示例**：
- 主标题：清初遗民文人的身份认同与书写策略
- 副标题：以顾炎武、黄宗羲为中心的考察

### 二、研究背景与动机

**结构**：
1. **宏观背景**（1段）：该议题在学术史或现实中的位置
2. **具体契机**（1段）：什么具体现象/文本/事件引发你的兴趣
3. **现有不足**（1-2段）：已有研究做到了什么、没做到什么
4. **你的切入点**（1段）：你打算从什么角度切入

**字数**：硕士 800-1500 字，博士 1500-2500 字

### 三、文献回顾

**不要做成书目罗列**。按以下逻辑组织：

**方法一：按主题线索**
```
现有研究主要从三个角度展开：
（一）政治史视角...
（二）思想史视角...
（三）文学史视角...
```

**方法二：按时间脉络**
```
关于本议题的研究大致经历了三个阶段：
第一阶段（1980-2000）：...
第二阶段（2000-2015）：...
第三阶段（2015至今）：...
```

**方法三：按论争焦点**
```
学界关于此问题存在几种主要观点：...
观点A：...
观点B：...
两种观点的分歧在于...
```

**关键任务**：在回顾末尾明确指出 **research gap**（研究空白）

### 四、研究问题

- 1-3 个核心问题，用问句或陈述句呈现
- 问题之间要有逻辑关联
- 每个问题都要能在正文中得到回答

**格式**：
```
基于上述文献回顾，本研究拟探讨以下核心问题：

（一）...？
（二）...？
（三）...？
```

### 五、研究方法

**需要说明**：
1. **研究路径**：文本细读 / 历史考证 / 比较分析 / 田野调查 / 数字人文等
2. **材料范围**：你计划分析哪些文本/史料/数据
3. **论证策略**：如何从材料推导结论
4. **理论框架**（如有）：使用什么理论视角

**示例**：
```
本研究采取思想史与文学史交叉的方法路径。在材料层面，以《明夷待访录》《日知录》为核心文本，辅以顾炎武、黄宗羲的书信、序跋及相关史料。在论证策略上，先梳理清初政治文化语境，再分析两位遗民文人的身份认同危机，最后考察其如何在书写中建构自我形象。
```

### 六、章节安排

用提纲形式呈现，每章说明：
- 本章解决什么问题
- 主要论点
- 与前后章的关系

**格式**：
```
本文预计分五章：

第一章 绪论
  阐述研究背景、问题意识、文献回顾、方法与章节安排。

第二章 清初遗民文人群体的形成
  考察明清易代之际文人的政治选择与身份困境，为后文分析奠定历史语境。

第三章 顾炎武的身份认同与经世书写
  通过细读《日知录》，分析顾炎武如何在学术书写中调和遗民身份与现实关怀。

第四章 黄宗羲的遗民叙事与历史书写
  ...

第五章 结论
  总结两位遗民文人书写策略的异同，讨论清初遗民书写的普遍模式与历史意义。
```

### 七、预期贡献

- **学术贡献**：填补了什么空白、修正了什么观点
- **方法贡献**：尝试了哪些新的分析路径
- **现实意义**（如适用）：对当下的启示

**注意**：避免过度夸大，保持学术谦逊

### 八、研究进度

按时间节点列出：
```
2026.03-2026.05  文献搜集与阅读
2026.06-2026.08  第一、二章初稿
2026.09-2026.11  第三、四章初稿
2026.12-2027.01  结论与修改
2027.02-2027.03  全文润色与定稿
2027.04         提交答辩
```

### 九、参考文献

- 列出开题报告中引用的所有文献
- 按学术规范格式排列（中文在前，外文在后）
- 数量参考：硕士 30-50 条，博士 50-80 条

## 三、常见错误

| 错误 | 修正 |
|------|------|
| 题目过大：「中国古代文学研究」 | 缩小范围：「清初遗民文人的身份认同——以顾炎武为中心」 |
| 文献回顾像书单 | 按主题/脉络组织，说明文献间关系 |
| 研究问题太泛 | 确保每个问题都能在正文中得到明确回答 |
| 方法描述空洞：「采用文本分析法」 | 具体说明：分析哪些文本、关注什么层面、如何操作 |
| 进度安排不切实际 | 留出修改和意外延误的时间 |
| 贡献夸大 | 具体说明填补了什么小空白，而非「填补学界空白」 |

## 四、自检清单

提交前确认：
- [ ] 题目具体、可操作
- [ ] 文献回顾有逻辑线索，不是罗列
- [ ] 明确指出了研究空白
- [ ] 研究问题 1-3 个，且互不重复
- [ ] 方法描述足够具体
- [ ] 章节安排每章都有明确任务
- [ ] 预期贡献务实，不夸大
- [ ] 进度安排留有余地
- [ ] 参考文献格式统一
- [ ] 全文无错别字
