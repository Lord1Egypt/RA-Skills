# TRIZ创新方法论

## 一、TRIZ概述

### 什么是TRIZ
TRIZ（俄语"发明问题解决理论"的音译）是前苏联发明家根里奇·阿奇舒勒（Genrich Altshuller）在1946年创立的创新方法论。通过分析全球大量专利，阿奇舒勒发现技术创新有规律可循，并总结出系统化的创新工具。

### 核心思想
- **矛盾论**：技术系统的进化是为了解决矛盾
- **理想最终解（IFR）**：始终存在理想解
- **资源利用**：系统内外资源可被充分利用
- **可预测性**：技术系统遵循可预测的进化模式

### TRIZ的价值

| 传统方法 | TRIZ方法 |
|---------|---------|
| 反复试错 | 系统化分析 |
| 依赖灵感 | 依靠逻辑和方法 |
| 效率低下 | 快速找到方向 |
| 难以复制 | 可学习传承 |

## 二、40个发明原理

TRIZ的40个发明原理是解决技术问题的核心工具。以下是完整列表和简要说明：

### 1. 分割（Segmentation）
- 将物体分成独立部分
- 使物体成为可拆卸的
- 增加物体的分割程度

**应用示例**：
- 模块化手机设计
- 可拆洗的衣物
- 组合家具

### 2. 抽取（Extraction）
- 将干扰部分或特性抽取出来
- 从物体中分离出噪声或振动源

**应用示例**：
- 空调外机消音设计
- 抽油烟机的吸烟腔独立设计
- 打印机噪声隔离舱

### 3. 局部质量（Local Quality）
- 改变物体的结构，使其各部分执行不同功能
- 使物体的每个部分处于最适宜的工作条件

**应用示例**：
- 记忆枕头的分区设计
- 手机摄像头区域的特殊处理
- 轮胎的花纹设计

### 4. 不对称（Asymmetry）
- 采用不对称形状代替对称形状
- 增加不对称程度

**应用示例**：
- 笔记本键盘的非对称设计
- USB接口的不对称设计
- 电池的防呆设计

### 5. 合并（Consolidation）
- 将同类物体在空间上合并
- 将同类操作在时间上合并

**应用示例**：
- 多功能打印机（扫描、复印、打印合一）
- 手机整合相机、MP3等功能
- 组合机床

### 6. 多功能性（Universality）
- 一个物体执行多种功能，减少功能部件

**应用示例**：
- 智能手机替代多个设备
- 安全头盔同时具备防护、通话、显示功能
- 汽车的ABS系统同时具备防滑功能

### 7. 嵌套（Dollies / Matrioshka）
- 将一个物体嵌入另一个物体
- 物体之间相互嵌套

**应用示例**：
- 嵌套桌椅
- 俄罗斯套娃
- 工具箱的嵌套设计
- 伸缩天线

### 8. 重量补偿（Counterweight）
- 用弹簧、气体等补偿重量
- 利用空气动力或磁力补偿

**应用示例**：
- 配重平衡系统
- 热气球利用浮力
- 磁悬浮列车

### 9. 预先反作用（Preliminary Anti-action）
- 预先施加反向作用
- 预先施加应力以抵消后续的高应力

**应用示例**：
- 预应力混凝土
- 双向拉伸塑料薄膜
- 安全带的预紧功能

### 10. 预先作用（Preliminary Action）
- 在需要前预先完成全部或部分操作
- 以最小代价进行预先准备

**应用示例**：
- 预制菜
- 预装软件
- 快递的预打包

### 11. 预先应急措施（Beforehand Cushioning）
- 预先准备应急补偿措施

**应用示例**：
- 安全气囊
- 救命药箱
- 备用轮胎

### 12. 等势（Equipotentiality）
- 改变工作条件，使物体不需要升高或降低
- 消除物体在重力场中的上升或下降

**应用示例**：
- 传送带运输
- 自动扶梯
- 液压升降台

### 13. 反向作用（Inversion）
- 颠倒解决问题的思路
- 上下颠倒、里外颠倒、前后颠倒

**应用示例**：
- 倒立式洗衣机
- 吸盘的设计
- 电梯的上下运行

### 14. 曲面化（Curvature）
- 用曲线代替直线
- 用球面代替平面
- 用旋转运动代替直线运动

**应用示例**：
- 拱形结构
- 圆角设计
- 螺旋楼梯

### 15. 动态化（Dynamicity）
- 使静止的物体可以自动调整
- 将物体分成可相对移动的部件

**应用示例**：
- 折叠伞
- 可调节高度的桌子
- 变形金刚玩具

### 16. 部分或超量作用（Partial or Excessive Actions）
- 使用稍微超过所需的动作
- 过度满足需求

**应用示例**：
- 加大安全系数
- 过量充电保护
- 冗余备份

### 17. 多维化（Another Dimension）
- 利用多层结构代替单层
- 将物体倾斜或侧向移动
- 利用"背面"

**应用示例**：
- 立体停车库
- 双层巴士
- 立体种植

### 18. 振动（Mechanical Vibration）
- 使用振荡
- 增加振荡频率
- 共振现象的应用

**应用示例**：
- 超声波清洗
- 振动按摩器
- 振动筛分机

### 19. 周期性动作（Periodic Action）
- 用周期性动作代替连续动作
- 改变周期性动作的频率

**应用示例**：
- 脉冲喷射打印机
- 间歇式雨刷
- 脉冲光脱毛仪

### 20. 连续有用作用（Continuity of Useful Action）
- 连续执行动作，避免空转
- 消除闲置和间歇动作

**应用示例**：
- 流水线生产
- 连续供墨系统
- 不间断电源（UPS）

### 21. 快速通过（Skipping）
- 高速完成有害操作
- 快速通过危险区域

**应用示例**：
- 高速切削
- 快速冷冻
- 瞬间高温灭菌

### 22. 变害为利（Leverage Blessings）
- 利用有害因素（环境或自身）
- 将有害因素与另一个有害因素组合

**应用示例**：
- 废热发电
- 废旧轮胎回收
- 污泥堆肥

### 23. 反馈（Feedback）
- 引入反馈
- 改变反馈的程度

**应用示例**：
- 自动调温器
- 巡航控制
- 音量自动调节

### 24. 借助中介物（Mediator）
- 使用中间物体传递或执行动作
- 将物体与另一易除去的物体暂时结合

**应用示例**：
- 模具的定位销
- 润滑剂
- 催化剂

### 25. 自服务（Self-service）
- 物体自我服务、辅助自身

**应用示例**：
- 自清洁涂层
- 自充电设备
- 自我修复材料

### 26. 复制（Copying）
- 用简单、廉价的复制品代替复杂物体
- 光学复制（图像）代替实物

**应用示例**：
- 虚拟现实
- 数字副本
- 3D打印

### 27. 一次性用品（Cheap Short-Living Objects）
- 用一次性物品替代复杂物品

**应用示例**：
- 一次性注射器
- 一次性餐盒
- 一次性口罩

### 28. 机械系统替代（Mechanics Substitution）
- 用视觉系统替代光学系统
- 用声学系统替代力学系统
- 用化学刺激替代物理刺激

**应用示例**：
- 视频会议替代出差
- 超声波测量
- 电子烟替代香烟

### 29. 气动与液压结构（Pneumatics and Hydraulics）
- 用气体或液体代替固体部件
- 用空气或水产生膨胀力

**应用示例**：
- 气垫床
- 液压传动
- 气弹簧

### 30. 柔性壳体或薄膜（Flexible Shells and Thin Films）
- 使用柔性材料和薄膜
- 用薄壳替代固体结构

**应用示例**：
- 充气床垫
- 薄膜电池
- 防水透气膜

### 31. 多孔材料（Porous Materials）
- 使物体多孔或添加多孔结构
- 填充多孔结构

**应用示例**：
- 海绵
- 蜂窝材料
- 活性炭

### 32. 颜色（Color）
- 改变物体颜色
- 改变物体周围环境的颜色

**应用示例**：
- 彩色玻璃
- 警示色
- 伪装色

### 33. 均质性（Homogeneity）
- 同类物体与同种材料制作

**应用示例**：
- 全金属工具
- 整体成型
- 同质合金

### 34. 抛弃与修复（Rejecting and Regenerating）
- 已完成工作的部分被消耗或丢失
- 立即修复消耗的部分

**应用示例**：
- 火箭助推器脱落
- 自毁式注射器
- 可降解材料

### 35. 参数变化（Parameter Changes）
- 改变物体的物理状态
- 改变浓度或密度
- 改变柔度、温度

**应用示例**：
- 形状记忆合金
- 相变储能材料
- 温控水杯

### 36. 状态变化（Phase Transition）
- 利用物质相变产生的效应

**应用示例**：
- 冰袋制冷
- 蒸发冷却
- 液态金属导热

### 37. 热膨胀（Thermal Expansion）
- 利用材料的热膨胀或收缩

**应用示例**：
- 热胀冷缩配合
- 双金属片开关
- 热熔胶

### 38. 强氧化剂（Strong Oxidants）
- 用富氧空气代替普通空气
- 用氧气代替空气

**应用示例**：
- 纯氧助燃
- 富氧燃烧
- 氧吧

### 39. 惰性环境（Inert Atmosphere）
- 用惰性环境代替普通环境

**应用示例**：
- 氮气保护焊接
- 惰性气体保存文物
- 真空包装

### 40. 复合材料（Composite Materials）
- 用复合材料代替均质材料

**应用示例**：
- 碳纤维复合材料
- 钢筋混凝土
- 玻璃钢

## 三、技术矛盾矩阵

### 矛盾矩阵是什么
矛盾矩阵是将39个通用工程参数与40个发明原理关联起来的工具，帮助发明者快速找到解决特定技术矛盾的潜在方法。

### 39个通用工程参数

**改善的参数（39个）**：
1. 运动物体的重量
2. 静止物体的重量
3. 运动物体的长度
4. 静止物体的长度
5. 运动物体的面积
6. 静止物体的面积
7. 运动物体的体积
8. 静止物体的体积
9. 速度
10. 力
11. 应力或压力
12. 形状
13. 结构稳定性
14. 强度
15. 运动物体的耐久性
16. 静止物体的耐久性
17. 温度
18. 亮度
19. 运动物体的能量消耗
20. 静止物体的能量消耗
21. 功率
22. 能量损失
23. 物质损失
24. 信息损失
25. 时间损失
26. 物质或事物的数量
27. 可靠性
28. 测试精度
29. 制造精度
30. 物体外部有害因素敏感性
31. 物体产生的有害因素
32. 可制造性
33. 可操作性
34. 可维修性
35. 适应性及多用性
36. 装置的复杂性
37. 监控与测试的困难程度
38. 自动化程度
39. 生产率

### 矛盾矩阵使用示例

**场景**：改进智能手机设计
- **问题**：希望增加屏幕尺寸（改善参数5：面积）
- **但会导致**：手机体积变大、难以握持（恶化参数8：体积）

**查找方法**：
1. 在矩阵中查找"面积（5）"行
2. 找到"体积（8）"列
3. 交叉位置显示推荐原理编号

**推荐原理**：
```
面积 vs 体积 的交叉单元格：15, 17, 34, 26
- 15：动态化（可折叠屏幕）
- 17：多维化（曲面屏、环绕屏）
- 34：抛弃与修复（模块化设计）
- 26：复制（虚拟大屏）
```

## 四、AI辅助应用TRIZ

### AI如何辅助TRIZ

| TRIZ环节 | AI辅助方式 |
|---------|-----------|
| 问题分析 | 理解技术描述，提取关键矛盾 |
| 原理推荐 | 基于问题特征，推荐适用原理 |
| 方案生成 | 生成应用原理的具体方案 |
| 方案评估 | 分析方案的可行性和创新性 |

### AI辅助应用流程

```
1. 输入技术问题
   ↓
2. AI分析并识别矛盾
   ↓
3. AI推荐适用发明原理
   ↓
4. AI生成解决方案示例
   ↓
5. 人工筛选和优化
   ↓
6. 形成最终方案
```

### 使用提示词示例

**基础问题分析**：
```
我想改进[技术方案]，目标[改善什么]，但担心[可能带来的问题]。
请帮我分析可能的技术矛盾，并推荐解决方案。
```

**创意点挖掘**：
```
请用TRIZ的[指定原理]分析[技术问题]，
生成5个具体的创新方案。
```

**专利创新点描述**：
```
基于TRIZ的[原理名称]，我设计了[技术方案]来解决[技术问题]。
```
