# Heart Blood Stasis Differentiation

## When to Use
- 患者主诉心胸憋闷或疼痛
- 伴随心悸、怔忡、舌紫暗、脉涩等瘀血征象
- 已排除无心系症状者及非心源性胸痛
- 出现以下任一诱因：情绪激动、劳累、受寒、过食肥甘或饮酒

## Execution Steps

### 1. 确认基础证候
确认是否存在以下核心表现：
- 心胸憋闷或刺痛，痛引肩背内臂，时发时止
- 伴心悸、怔忡
- 舌紫暗或有瘀点瘀斑
- 脉细涩或结代
- 重症可见暴痛欲绝、口唇青紫、肢厥神昏、脉微欲绝

### 2. 分型辨证（依据疼痛性质与伴随症状）

**a) 瘀血阻络型**
- 疼痛特点：刺痛
- 伴随症状：舌紫暗、瘀斑，脉细涩或结代

**b) 痰浊停聚型**
- 疼痛特点：闷痛
- 伴随症状：体胖痰多、身重困倦、舌苔白腻、脉沉滑

**c) 阴寒凝滞型**
- 疼痛特点：痛势剧烈、突然发作、得温痛减
- 伴随症状：畏寒肢冷、舌淡苔白、脉沉迟或沉紧

**d) 气滞心脉型**
- 疼痛特点：胀痛
- 伴随症状：发作与情绪相关，脉弦；轻者舌淡红，重者舌暗红

### 3. 分析病因与病机
- 判断是否继发于心气虚或心阳虚
- 明确诱发因素（情绪、饮食、寒凉等）
- 解释病机：心气不足或心阳虚衰 → 气血运行不畅 → 瘀血内阻或痰浊停聚 → 心脉痹阻

### 4. 输出结果
明确具体证型（如“瘀血阻络型心血瘀阻证”），并据此指导治疗方向：活血化瘀、化痰通络、温阳散寒或理气止痛。
