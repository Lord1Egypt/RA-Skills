# 第二信使与细胞通讯机制分析

## 适用场景
- GPCR信号通路解析
- 外泌体介导的细胞对话研究
- 电信号或化学信号传递机制分析
- 肿瘤微环境或免疫应答建模

## 执行步骤
1. **识别信号类型**：
   - 水溶性激素（如肾上腺素）→ 启动第二信使
   - 膜锚定信号 → 接触依赖通讯
   - 小分子扩散 → 旁分泌/内分泌
   - 外泌体释放 → RNA/蛋白介导的远程调控

2. **追踪第二信使生成**：
   - cAMP、cGMP、Ca²⁺、DAG、IP₃等浓度瞬时变化
   - 激活下游激酶（如PKA、PKC）

3. **整合通讯模式**：
   - 分泌信号：激素/神经递质
   - 接触依赖：膜结合配体-受体
   - 间隙连接：connexin通道
   - 外泌体：携带miRNA等调控受体细胞

4. **输出机制图谱**：
   - 激活下游靶蛋白，引发代谢或基因表达变化
   - 识别具体通讯类型，用于分析组织协调或病理过程
