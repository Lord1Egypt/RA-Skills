# 哺乳动物细胞自噬类型识别与机制应用

根据底物递送机制和细胞需求，哺乳动物细胞自噬分为三类。按以下决策流程确定适用类型并执行对应操作：

## 决策流程

1. **检查底物是否为可溶性蛋白且含有KFERQ样序列**
   - 若是 → 启动**分子伴侣介导的自噬（CMA）**
   - 若否 → 进入下一步

2. **评估细胞是否需要大规模降解胞质成分（如应对营养缺乏、清除受损细胞器）**
   - 若是 → 启动**巨自噬（macroautophagy）**
   - 若否 → 进入下一步

3. **观察是否存在溶酶体膜直接内陷包裹邻近胞质物质的现象**
   - 若是 → 启动**微自噬（microautophagy）**

## 执行对应机制

### 巨自噬（Macroautophagy）
- **适用场景**：营养缺乏、清除大型结构（如线粒体、脂滴）、非选择性或选择性降解
- **操作步骤**：
  1. 形成双层膜自噬泡包裹目标底物
  2. 自噬泡闭合形成自噬体
  3. 自噬体与溶酶体融合形成自噬溶酶体
  4. 溶酶体酶降解内容物
- **注意**：选择性巨自噬（如线粒体自噬）需依赖特异性识别机制（如PINK1/Parkin）

### 微自噬（Microautophagy）
- **适用场景**：局部、小规模胞质成分清除；溶酶体直接参与
- **操作步骤**：
  1. 溶酶体膜内陷或出芽
  2. 直接包裹邻近胞质底物
  3. 在溶酶体腔内降解底物
- **注意**：无需自噬体形成，不依赖ATG蛋白核心机器

### 分子伴侣介导的自噬（CMA）
- **适用场景**：精准清除含KFERQ样序列的可溶性错误折叠或调控蛋白
- **操作步骤**：
  1. Hsc70识别底物蛋白上的KFERQ样序列
  2. Hsc70-底物复合物结合溶酶体膜受体LAMP2A
  3. 底物在LAMP2A协助下解折叠并单体跨膜转运
  4. 溶酶体腔内水解底物
- **关键限制**：仅适用于特定序列的可溶性蛋白；无囊泡参与

## 输出建议
根据分析结果，明确推荐自噬类型，并说明其生物学意义（如CMA用于维持蛋白稳态，巨自噬用于应激响应）。
