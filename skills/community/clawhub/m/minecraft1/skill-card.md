## Description: <br>
提供基于截图的观察和键鼠操作，实现 Minecraft Java Edition 中的角色自动控制和交互辅助。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Minecraft Java Edition players use this skill to let an assistant observe the game through screenshots, plan actions, and operate movement, inventory, mouse, typing, and chat controls. It is intended for supervised gameplay assistance where the user can review important in-game actions. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Automated controls can perform unintended in-game actions or send unintended chat messages, especially on multiplayer servers or worlds where mistakes matter. <br>
Mitigation: Keep the Minecraft window focused while using the skill and review important in-game actions or chat messages before relying on them. <br>
Risk: Screenshot-based gameplay assistance can misread unknown states or repeat actions that are not producing the expected result. <br>
Mitigation: Pause for user review when the character dies, the next step is unclear, or three consecutive operations fail to produce the expected outcome. <br>
