# 先天性肾上腺皮质增生症（CAH）的实验室诊断与急症鉴别

## 何时使用
- 患儿有失盐表现（呕吐、脱水、低钠高钾）或外生殖器畸形
- 急诊就诊伴代谢性酸中毒，需与DKA鉴别
- 新生儿筛查17-OHP阳性或临床高度怀疑CAH

## 执行步骤
### 一、CAH实验室诊断流程
1. **尿液检测**：测定17-酮类固醇（17-KS），显著升高提示肾上腺雄激素过度分泌
2. **血液检测**：
   - 血17-羟孕酮（17-OHP）：21-羟化酶缺乏症的关键指标，可升高数十至数百倍
   - 电解质：失盐型CAH表现为低钠（↓Na⁺）、高钾（↑K⁺）；单纯男性化型通常正常
   - 其他激素：DHEA、睾酮、醛固酮、肾素活性等辅助分型
3. **新生儿筛查**：干血片17-OHP >30 nmol/L提示经典21-OHD，早产儿需谨慎解读
4. **确诊手段**：液相色谱-串联质谱分析多种类固醇代谢产物

### 二、CAH与DKA的快速鉴别
| 特征 | 失盐型CAH | 糖尿病酮症酸中毒（DKA） |
|------|----------|------------------------|
| 血糖 | 正常或偏低 | ≥11.1 mmol/L |
| 血17-OHP | 显著升高 | 正常 |
| 尿17-KS | 明显升高 | 正常或轻度升高 |
| 外生殖器 | 女性男性化 | 正常 |
| 血钾 | 升高 | 通常降低 |
| 呼气气味 | 无特殊 | 烂苹果味（丙酮） |

### 三、处理原则
- **CAH**：立即补充氢化可的松+氟氢可的松，纠正电解质
- **DKA**：胰岛素治疗、补液、纠正酸中毒，禁用糖皮质激素

> 注意：早产儿17-OHP易假阳性；DKA不可误用激素，CAH不可延误替代治疗。
