# Urticaria Syndrome Differentiation

## When to Use
- 风团色赤、遇热加重（风热）
- 风团色白、遇冷加重（风寒）
- 伴脘痞、便秘（胃肠湿热）
- 反复多年、神疲乏力（气血两虚）

## Execution Steps
1. **判断病程**：急性（起病急）或慢性（迁延不愈）。
2. **辨证选方**：
   - 风热犯表 → 疏风清热 → 消风散；
   - 风寒束表 → 祛风散寒 → 荆防败毒散合桂枝汤；
   - 胃肠湿热 → 祛风通里 → 防风通圣散；
   - 气血两虚 → 养血益气 → 当归饮合玉屏风散。
3. **外治辅助**：所有证型均可配合苦参汤温洗。

> **注意**：核心病机为风邪（寒/热）郁于肌表，久病则血虚生风；不适用于非过敏性皮肤病。
