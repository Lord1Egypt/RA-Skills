## Description: <br>
Scaffold starter projects for Next.js, React, Python CLI, FastAPI, Express, static HTML, and OpenClaw skills with generated code, README, license, CI, and git initialization. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[vanthienha199](https://clawhub.ai/user/vanthienha199) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers use Project Kickstart to create new application, API, CLI, static-site, or agent-skill repositories with minimal working boilerplate and standard project files. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill creates files and initializes git as part of project scaffolding. <br>
Mitigation: Run it in an empty or clearly chosen directory and confirm the intended stack before generation when the request is broad. <br>
Risk: Generated dependency and CI files may affect later installs, builds, or publication workflows. <br>
Mitigation: Review generated dependency, license, and CI files before running package installs or publishing the repository. <br>
