# Irregular Menstruation Regulation

## When to Use
确认以下全部前提成立后再应用本技能：
- 月经周期“时或提前、时或延后7天以上”；
- 连续≥3个月经周期出现上述波动；
- 已排除妊娠、围绝经期、哺乳等非病理性因素；
- 非偶发性周期波动；
- 若存在器质性病变（如子宫肌瘤、内分泌疾病），需结合西医诊断。

## Execution Steps
根据核心病机分为两类证型，选择对应治法：

### 1. 肝郁证
**辨证要点**：
- 经量或多或少，色暗红有血块；
- 胸胁、乳房、少腹胀痛，精神郁闷，时欲太息；
- 舌苔薄白或薄黄，脉弦。

**治法**：疏肝解郁，和血调经。
**主方**：逍遥散加减。

### 2. 肾虚证
**辨证要点**：
- 经量少，色淡暗，质稀；
- 腰膝酸软，头晕耳鸣，小便频数；
- 舌淡苔白，脉沉细。

**治法**：补益肾气，养血调经。
**主方**：固阴煎加减。

## Execution Steps
贯彻“疏肝补肾，调和冲任”的总体思路，根据证型侧重调整。

## Precautions
若患者同时出现经量明显增多或经期延长，警惕发展为崩漏，应密切观察并考虑转诊或联合干预。

## Execution Steps
1. 确认诊断标准是否满足；
2. 采集情绪状态、肾系症状等关键变量；
3. 辨证分型（肝郁或肾虚）；
4. 推荐对应方剂加减；
5. 提示崩漏风险并建议随访。
