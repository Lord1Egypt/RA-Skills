# 工作日报生成助手

输入今天做了什么，自动整理成「已完成 / 进行中 / 明日计划」三段式专业日报。

## Tools Required
- read
- write

## Usage
- "帮我写今天的日报：上午开了需求评审会，下午改了三个 bug，还有一个接口联调没做完"
- "今日工作汇报：完成了季度报告初稿，跟进了两个客户报价，明天要参加部门例会"
- "把这些整理成每日汇报发给领导：培训新员工2小时，处理售后投诉3单，库存盘点进行到一半"

## Examples
输入：今天开了产品周会，写了两篇推文，广告投放方案还没定，明天要对接设计出海报
输出：
**【工作日报】2024-XX-XX**

✅ 已完成
- 参加产品周会，同步各方进展
- 完成推文撰写 2 篇，已提交审核

🔄 进行中
- 广告投放方案制定中，待确认预算后完成

📋 明日计划
- 对接设计同事，推进海报物料产出
- 跟进广告投放方案审批
