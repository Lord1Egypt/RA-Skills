## Description: <br>
Knowledge management for AI agents, with API examples for storing and retrieving project context and using 50+ models for image, video, audio, chat, search, document processing, email, and SMS workflows. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent operators use this skill to access project-context workflows and a broad remote API gateway for model calls, search, document processing, email, SMS, and media generation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad external API access may process private content across multiple remote providers. <br>
Mitigation: Use a limited or test API key where possible and avoid confidential documents, audio, or regulated data unless approved. <br>
Risk: Email, SMS, OTP, scraping, batch messaging, and smart-routed provider calls can have high-impact side effects. <br>
Mitigation: Require manual review before executing these calls. <br>
Risk: The security review says the skill's guardrails and disclosure are thin for broad remote automation. <br>
Mitigation: Install only when a broad remote automation/API gateway is intentionally needed and review behavior before deployment. <br>
