## Description: <br>
Create Edu Video helps an agent produce educational videos from a course topic, including scriptwriting, TTS narration, visual design, Remotion code development, and video export for vertical or horizontal formats. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Educators, instructional designers, content creators, and developers use this skill to have an agent prepare educational video scripts, subtitles, narration timing, Remotion project code, previews, and final exports. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Lesson topics, scripts, or narration text may be sent to external search or TTS services during video creation. <br>
Mitigation: Avoid proprietary lesson plans, private student data, and confidential scripts unless the user accepts sharing that information with those services. <br>
Risk: The workflow can create local Remotion projects, generate files, and run media or rendering commands. <br>
Mitigation: Review generated code and shell commands before execution, install required tools manually, and run the workflow only in trusted project directories. <br>
Risk: Generated educational scripts, visuals, subtitles, or narration timing may be inaccurate or poorly aligned. <br>
Mitigation: Review the preview video, verify the instructional content, and iterate before producing the final export. <br>
