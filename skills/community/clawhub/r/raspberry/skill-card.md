## Known Risks and Mitigations: <br>
Risk: The skill includes a Docker install command that downloads and runs a remote script. <br>
Mitigation: Inspect the script or use Docker's signed package repository instructions before running it on a Raspberry Pi. <br>
Risk: Incorrect power, storage, GPIO, or SSH setup can cause instability, data loss, hardware damage, or unnecessary network exposure. <br>
Mitigation: Review the checklist against the specific Raspberry Pi model and deployment environment before applying commands or wiring hardware. <br>
