## Description: <br>
Convert Python, JavaScript, or TypeScript functions into Mermaid flowcharts to visualize code logic and structure clearly. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[HonestQiao](https://clawhub.ai/user/HonestQiao) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and engineers use this skill to convert single Python, JavaScript, or TypeScript functions or methods into flowcharts for reviewing code logic and structure. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generic flowchart requests may activate this skill when the user intended a different diagramming tool. <br>
Mitigation: Use explicit prompts that name the desired diagramming tool or output format. <br>
Risk: Generated Mermaid may simplify complex loop behavior or omit details from unsupported control flow. <br>
Mitigation: Review the generated diagram against the source code before relying on it for design, debugging, or documentation. <br>
Risk: Rendering generated diagrams in external viewers may expose sensitive source code. <br>
Mitigation: Review generated Mermaid before rendering and use trusted local or approved viewers for sensitive code. <br>
