# 子宫内膜癌分子分型

## 适用对象
- 已经通过术后病理确诊为子宫内膜癌的患者
- **必须已完成以下分子检测**：
  - POLE外切酶结构域测序
  - MMR蛋白免疫组化（MLH1、PMS2、MSH2、MSH6）或微卫星不稳定性（MSI）检测
  - p53免疫组化或TP53基因测序
  - 基因组拷贝数分析（如NGS或SNP array）

## 执行步骤
1. **按优先级顺序判断分子分型类别**：
   - **POLE超突变型（POLEultramutated）**：存在致病性POLE外切酶结构域突变 → 预后极好
   - **MSI-H/dMMR型**：MMR蛋白缺失或MSI-high，且无POLE突变 → 预后中等，可能对免疫治疗敏感
   - **CNH/p53异常型**：p53异常表达（弥漫强阳性或完全阴性）且高拷贝数，且无前两类改变 → 预后最差
   - **CNL/NSMP型**：无上述任何改变，低拷贝数 → 预后中等

2. **输出分型结果与预后等级**：
   - 分型类别：POLE、MSI-H、NSMP 或 p53abn
   - 预后排序：POLE > MSI-H ≈ NSMP > p53abn

3. **临床应用建议**：
   - 用于风险分层
   - 指导是否需要辅助化疗或放疗
   - 支持临床试验筛选（如免疫治疗适用于MSI-H/dMMR型）
   - 注意：2023年FIGO分期已开始整合分子分型，但当前仍以2009分期为主

## 禁忌与限制
- 不适用于未完成分子检测的病例
- 不适用于非上皮性子宫恶性肿瘤（如肉瘤）
