## Description: <br>
Create distinctive, production-grade frontend interfaces and web UI code with strong visual direction and avoidance of generic AI aesthetics. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and designers use this skill to guide an agent in building or improving web pages, components, dashboards, and applications with distinctive visual direction, polished interaction details, and production-grade frontend code. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill intentionally applies broad, opinionated frontend design guidance when web UI or styling tasks are in scope. <br>
Mitigation: Review generated interfaces against product brand, accessibility, and usability requirements before release. <br>
Risk: Generated frontend work may include elaborate animation or visual effects that need project-level performance and accessibility review. <br>
Mitigation: Test motion, keyboard behavior, contrast, responsive layout, and runtime performance in the target application before deployment. <br>
