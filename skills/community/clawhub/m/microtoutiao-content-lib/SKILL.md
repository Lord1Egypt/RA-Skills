# 微头条素材库

专为今日头条微头条自动化发布设计的素材生成工具，支持情感故事、知识科普、赚钱干货等多种热门内容类型。

## 核心能力

### 1. 七大热门内容类型

| 类型 | 流量特征 | 适用场景 |
|------|----------|----------|
| 情感故事 | 极高，互动强 | 婆媳、夫妻、亲子、前任 |
| 人情世故 | 高，争议强 | 随礼、借钱、邻居、同事 |
| 知识科普 | 中高，稳定 | 健康、生活妙招、职场技能 |
| 赚钱干货 | 高，粘性强 | 副业、省钱、理财、创业 |
| 热点解读 | 爆发性强 | 社会民生、娱乐、政策 |
| 生活技巧 | 高，收藏高 | 家务、收纳、烹饪 |
| 奇闻轶事 | 不稳定 | 身边奇事、见闻 |

### 2. 模板结构

每个模板包含：
- **标题公式**: 情绪钩子 + 悬念 + 数字/对比
- **正文结构**: 开头吸睛 → 内容展开 → 结尾引导
- **关键词布局**: 自然融入平台推荐关键词
- **配图建议**: 图片类型和风格建议

### 3. 热点追踪

根据节日、节气、社会热点自动匹配素材方向：
- 节日类：春节/端午/中秋/情人节
- 节气类：春夏秋冬季节性话题
- 热点类：社会事件/明星八卦/政策解读

## 输出格式

### 标题公式
```
矛盾型：[具体行为]让[身份]彻底[情绪动词]，结局没想到...
对比型：A vs B，区别真的太大了
提问型：你们遇到过这样的情况吗？
数字型：[数字]个技巧/方法/秘密，一次性告诉你
```

### 正文结构
```
[开头30字] 制造悬念/冲突/共鸣
[主体80-150字] 故事/知识/观点展开
[结尾20字] 引导评论/转发/关注
```
