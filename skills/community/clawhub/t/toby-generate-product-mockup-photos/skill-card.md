## Known Risks and Mitigations: <br>
Risk: The skill asks for broad shell and file access for a product-image workflow. <br>
Mitigation: Install and run it only in an OpenClaw runtime that constrains shell and file access. <br>
Risk: Install-source instructions may not match the server-resolved ClawHub slug and publisher. <br>
Mitigation: Verify the ClawHub page, publisher profile, and exact install target before following installation commands. <br>
Risk: Product plans, unreleased designs, or customer data could be sent to an external image-generation provider. <br>
Mitigation: Avoid confidential inputs unless that external processing is approved for the intended use. <br>
