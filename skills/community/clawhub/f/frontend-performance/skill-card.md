## Description: <br>
Analyzes frontend performance issues such as LCP, FCP, CLS, bundle size, lazy loading, and runtime efficiency, then suggests prioritized optimizations. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wangzhiming1999](https://clawhub.ai/user/wangzhiming1999) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to assess frontend loading and runtime performance, identify likely bottlenecks from Lighthouse, Performance, Network, or bundle data, and plan practical optimizations with validation steps. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Suggested npx, Lighthouse, or bundle-analyzer commands could be run in the wrong project or with untrusted tooling. <br>
Mitigation: Review proposed commands before execution and run them only in the intended project with trusted tooling. <br>
Risk: Performance recommendations can be misleading when no measurements are available. <br>
Mitigation: Base optimization plans on Lighthouse, Performance, Network, or bundle analysis data and re-test after changes. <br>
