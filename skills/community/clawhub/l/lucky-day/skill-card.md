## Use Case: <br>
External users can use this skill to rank upcoming auspicious dates for events such as weddings, business openings, moving, ceremonies, and groundbreaking. It can optionally account for the user's Chinese zodiac sign when avoiding conflicting dates. <br>

## Known Risks and Mitigations: <br>
Risk: The skill runs a bundled Python helper. <br>
Mitigation: Review the helper before installation and run it only in an environment where local script execution is acceptable. <br>
Risk: Auspicious-date recommendations are advisory and may be incomplete or culturally contextual. <br>
Mitigation: Treat the output as reference guidance and confirm important scheduling decisions with the user's own requirements and expert advice when needed. <br>
