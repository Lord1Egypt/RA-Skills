## Known Risks and Mitigations: <br>
Risk: The workflow may publish local content to a public GitHub repository without enough explicit safety checks. <br>
Mitigation: Confirm the GitHub account, repository name, visibility, branch, and exact files before upload. <br>
Risk: Uploaded files could contain tokens, keys, private notes, customer data, or proprietary code. <br>
Mitigation: Scan the directory for sensitive or proprietary content before any push or browser upload. <br>
