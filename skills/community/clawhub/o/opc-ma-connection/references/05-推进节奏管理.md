# 推进节奏管理

> 本文档详细说明并购对接的六阶段推进模型、关键动作和状态管理。

---

## 一、六阶段推进模型

### 1.1 阶段概览

| 阶段 | 目标 | 预期时长 | 成功标志 |
|------|------|----------|----------|
| **阶段1** | 建立联系 | 1-2周 | 收到回复或约定沟通 |
| **阶段2** | 建立信任 | 2-4周 | 获得深入交流机会 |
| **阶段3** | 正式推介 | 2-4周 | 对方表示兴趣 |
| **阶段4** | 尽调配合 | 4-8周 | 完成初步尽调 |
| **阶段5** | 交易谈判 | 4-12周 | 签署意向协议 |
| **阶段6** | 签约交割 | 4-12周 | 工商变更完成 |

## 二、每阶段关键动作

### 2.1 阶段1：建立联系

#### 关键动作清单

| 动作 | 说明 | 频率 | 注意事项 |
|------|------|------|----------|
| **发送邮件** | 按话术模板发送首封邮件 | 每目标1次 | 发送时间：工作日上午9-10点最佳 |
| **电话跟进** | 对重要邮件进行电话确认 | 每目标1-2次 | 通话时长控制在3分钟内 |
| **活动接触** | 通过行业活动建立联系 | 视机会 | 准备好30秒自我介绍 |
| **中间人请求** | 请共同人脉引荐 | 视情况 | 给中间人足够背景信息 |

- [ ] 收集到目标公司关键人信息

- [ ] 如未回复，按计划进行跟进
- [ ] 记录了联系结果

| **预约会议** | 争取当面汇报机会 | 即时 | 会议时长控制在1小时内 |
| **现场汇报** | 专业完整的项目介绍 | 30-60分钟 | 不要超时 |

### 2.4 阶段4：尽调配合

#### 关键动作清单

| 动作 | 说明 | 时长 | 注意事项 |
|------|------|------|----------|
| **资料准备** | 整理尽调资料清单 | 1周 | 提前准备，提高效率 |
| **安排访谈** | 协调团队成员时间 | 即时 | 高管访谈需高度重视 |
| **现场尽调** | 安排现场考察 | 1-3天 | 充分准备，展示实力 |
| **问题解答** | 及时回应尽调问题 | 即时 | 保持透明，不要隐瞒 |
| **进度跟踪** | 跟进尽调进展 | 每周 | 了解对方决策进度 |

#### 尽调阶段注意事项

| 注意事项 | 说明 |
|----------|------|
| **保密协议** | 尽调前签署NDA，保护双方利益 |
| **材料真实性** | 所有提供材料必须真实准确 |
| **快速响应** | 尽调问题需在24-48小时内响应 |
| **高层配合** | 高管层需亲自参与关键访谈 |
| **问题记录** | 记录对方关注点和问题，便于后续改进 |

### 2.5 阶段5：交易谈判

#### 关键动作清单

| 动作 | 说明 | 时长 | 注意事项 |
|------|------|------|----------|
| **估值协商** | 确定交易估值 | 2-4周 | 参考市场公允价格 |
| **条款谈判** | 交易结构、业绩对赌等 | 4-8周 | 保护双方利益 |
| **律师介入** | 法律文件起草审核 | 同步进行 | 选择有经验律师 |
| **内部审批** | 各方内部决策流程 | 视情况 | 提前沟通预期 |
| **意向协议** | 签署Term Sheet | 1-2周 | 明确核心条款 |

### 2.6 阶段6：签约交割

#### 关键动作清单

| 动作 | 说明 | 时长 | 注意事项 |
|------|------|------|----------|
| **最终协议** | 签署正式并购协议 | 2-4周 | 仔细审核每一条款 |
| **审批流程** | 必要的审批程序 | 视情况 | 上市公司需董事会、股东大会 |
| **股权交割** | 完成股权变更登记 | 1-2周 | 工商变更手续 |
| **人员整合** | 团队安排和文化融合 | 持续 | 保持沟通，稳定团队 |
| **过渡期** | 管理权交接过渡 | 3-6个月 | 平稳过渡 |

## 三、被拒绝后的应对策略

### 3.1 拒绝类型分析

| 拒绝类型 | 原因分析 | 可能性 | 应对策略 |
|----------|----------|--------|----------|
| **时机不对** | 公司近期无并购计划 | 高 | 保持联系，等待时机 |
| **方向不符** | 不在目标赛道 | 中 | 寻找其他标的 |
| **条件差距** | 估值/条件差距大 | 中 | 调整预期或换标的 |
| **已有储备** | 已有类似项目在推进 | 中 | 了解情况，等待结果 |
| **兴趣不足** | 项目吸引力不够 | 低 | 优化项目包装或换标的 |

### 3.2 跟进节奏建议

| 拒绝次数 | 建议等待时间 | 跟进方式 |
|----------|--------------|----------|
| 首次拒绝 | 1个月后 | 简短跟进邮件 |
| 二次拒绝 | 3个月后 | 更新项目进展 |
| 三次拒绝 | 半年后 | 如有重大进展可再次联系 |
| 三次以上 | 建议放弃 | 归档记录，换其他标的 |

### 3.3 拒绝后话术

```markdown
【拒绝后的标准跟进话术】

首次拒绝后一个月：

"XX总，您好！上次冒昧联系您，没能得到进一步交流的机会，非常抱歉打扰。
我们团队一直在持续推进项目，如有重大进展会第一时间与您同步。
如未来贵司有相关需求，也希望能有机会合作。祝您工作顺利！"

二次拒绝后三个月：

"XX总，您好！距离上次联系已有数月，我们项目取得了[新进展]，
特此向您汇报。如贵司战略方向有所调整，欢迎随时联系。祝商祺！"
```

## 四、多线推进管理

### 4.1 多线推进原则

| 原则 | 说明 |
|------|------|
| **优先级排序** | 按匹配度和成功率排序，集中精力 |
| **差异化策略** | 不同公司使用不同话术和材料 |
| **状态跟踪** | 记录每条线的最新状态 |
| **避免撞车** | 不要同时联系同一公司的多个部门 |
| **资源分配** | 头部标的分配更多资源 |

### 4.2 状态标记系统

| 状态代码 | 状态名称 | 说明 | 后续动作 |
|----------|----------|------|----------|
| **S0** | 待联系 | 尚未发起联系 | 按计划发起联系 |
| **S1** | 跟进中 | 已联系，等待回复 | 适时跟进 |
| **S2** | 深度沟通 | 进入深入交流阶段 | 加快推进节奏 |
| **S3** | 尽调阶段 | 已进入尽调 | 配合尽调，推进交易 |
| **S4** | 暂停 | 暂时搁置 | 定期维护，等待时机 |
| **S5** | 已放弃 | 已明确放弃 | 归档记录 |
| **S6** | 已签约 | 交易完成 | 进入整合阶段 |
