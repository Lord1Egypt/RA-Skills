# 角色四视图生成提示词模板

## 通用前缀

```
角色参考图，纯白背景，无文字，无道具，无环境元素，标准直立站姿，双臂自然下垂，双手空手，完全中性表情，嘴唇自然闭合，目光平静直视前方，眉毛自然位置，不笑不皱眉。
```

## 四视图布局

生成一张 2x2 网格图：

| 位置 | 视图 | 要求 |
|------|------|------|
| 左上 | 头部特写 | 头顶到锁骨，清晰面部特征 |
| 右上 | 正面全身 | 从头到脚完整，双臂自然下垂 |
| 左下 | 侧面全身 | 90°左侧面，从头到脚完整 |
| 右下 | 背面全身 | 180°背面，从头到脚完整 |

## 角色特定描述

- 发型发色：
- 面部特征：
- 服装：

- 画风：

## 禁止事项

- 任何文字、标签、水印
- 任何手持道具
- 任何表情或情绪
- 任何动态姿势
- 任何环境背景（必须纯白）
- 任何裁切（头部特写除外）
