# 输卵管妊娠诊断流程

## 触发条件
当满足以下条件时启动本流程：
- 患者为停经伴腹痛或阴道流血的育龄女性；
- 尿或血hCG检测阳性；
- 经阴道超声未见明确宫内孕囊。

## 执行步骤

### 1. 超声检查（首选经阴道）
- 确认宫腔内无孕囊；
- 观察宫旁是否存在异常低回声区：
  - 若含卵黄囊、胚芽或原始心管搏动，可确诊异位妊娠；
  - 若仅见混合回声区伴直肠子宫陷凹游离液性暗区，高度怀疑异位妊娠；
- 注意鉴别假孕囊（由蜕膜管型与血液形成），避免误诊为宫内妊娠；
- 直肠子宫陷凹积液不能单独作为诊断依据。

### 2. hCG测定与动态监测
- 异位妊娠患者>99% hCG阳性（陈旧性病例可能阴性）；
- 若hCG ≥3500 IU/L 且超声未见宫内孕囊，应高度怀疑异位妊娠；
- 若hCG <3500 IU/L：
  - hCG持续上升 → 48小时内复查超声；
  - hCG不上升或缓慢上升 → 考虑行诊断性刮宫。

### 3. 诊断性刮宫
- 刮出物送病理检查：
  - 见绒毛 → 支持宫内妊娠；
  - 仅见蜕膜组织无绒毛 → 支持异位妊娠诊断。

### 4. 经阴道后穹隆穿刺（适用于疑似腹腔内出血者）
- 抽出暗红色不凝血 → 提示腹腔积血；
- 阴性结果不能排除异位妊娠（因出血量少、病灶位置高或盆腔粘连）。

### 5. 腹腔镜检查
- 不再作为诊断金标准（漏诊率3%～4%）；
- 主要用于治疗而非初始诊断。

### 6. 综合判断
结合以下要素做出最终诊断：
- 症状（停经、腹痛、阴道流血）；
- 体征（宫颈举痛、附件区包块）；
- hCG水平及变化趋势；
- 超声特征；
- 后穹隆穿刺结果（如执行）。

## 输出
根据综合评估结果，决定是否确诊异位妊娠，并据此选择手术治疗、药物治疗或期待治疗方案。
