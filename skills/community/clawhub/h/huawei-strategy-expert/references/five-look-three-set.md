# 五看三定 战略方法论

## 概述

"五看三定"是华为战略规划（SP）的核心方法论，贯穿DSTE体系的战略制定阶段。"五看"解决"我们在哪"的洞察问题，"三定"解决"我们去哪"的决策问题。

## 五看 (Five-Look Analysis)

### 1. 看趋势 (Look at Trends)
- **宏观环境**：PEST（政治/经济/社会/技术）
- **技术趋势**：技术 S曲线、技术替代风险
- **产业政策**：国家战略、行业监管
- **社会变迁**：人口结构、消费习惯
- **关键输出**：3-5年趋势判断报告

### 2. 看市场/客户 (Look at Market/Customers)
- **市场规模**：TAM/SAM/SOM分析
- **市场增长**：增长率、驱动力
- **客户细分**：价值客户画像、需求层次
- **价值转移**：利润池变化趋势
- **关键输出**：市场空间地图、价值客户清单

### 3. 看竞争 (Look at Competition)
- **竞争对手全景**：直接/间接/潜在竞争者
- **竞争态势**：市场份额、技术路线、商业模式对比
- **竞争对手策略**：18个月战略预判
- **竞争差距**：优劣势对比矩阵
- **关键输出**：竞争格局图、竞对动向量化分析

### 4. 看自己 (Look at Self)
- **能力盘点**：核心技术、渠道、品牌、供应链
- **资源评估**：人才、资金、IP
- **组织能力**：决策效率、执行力、文化适配
- **商业模式健康度**：商业画布评估
- **关键输出**：能力差距分析、SWOT矩阵

### 5. 看机会 (Look at Opportunities)
- **机会识别**：未被满足的需求、技术突破点
- **机会评估**：可行性×吸引力矩阵
- **机会窗口**：时间窗口判断（先发优势vs后发优势）
- **机会优先级**：战略优先级排序
- **关键输出**：战略机会点清单、机会窗时间轴

## 三定 (Three-Determination)

### 1. 定目标 (Set Goals)
采用**双轨目标制**：
- **达标目标 (Baseline)**：基于确定性可实现的目标
- **挑战目标 (Stretch)**：基于战略雄心的突破性目标

目标层级：
- **公司级目标**：3-5年战略财务目标
- **业务线目标**：市场份额、收入、利润
- **能力目标**：技术/渠道/品牌里程碑

案例：2016年华为终端"1亿台销量"拆解为渠道、产品、供应链27项子目标

### 2. 定策略 (Set Strategy)
从目标到策略的六维度：
1. **客户策略**：选择谁、不选择谁
2. **产品策略**：产品组合、路标规划
3. **渠道策略**：区域策略（农村包围城市/城市攻坚战）
4. **定价策略**：价值定价vs竞争定价
5. **品牌策略**：品牌定位与传播
6. **生态策略**：合作伙伴体系

### 3. 定控制点 (Set Control Points)
战略控制点是护城河，按强度递进：
```
成本优势 → 功能/性能优势 → 技术专利 → 品牌/客户关系
→ 行业标准 → 价值链控制 → 绝对领导地位
```

- **初级**：10%成本优势、区域市场第一
- **中级**：核心技术专利、品牌溢价
- **高级**：行业标准制定（如华为5G极化码）、产业链控制

华为案例：5G极化码专利→麒麟芯片→鸿蒙操作系统，逐级升级控制点

## 五看三定工作流程

```
五看 → 战略机会点识别 → 机会点评估排序 → 
三定（目标→策略→控制点）→ 资源分配 → 组织对齐
```

## 应用检查清单

**五看自查**：
- [ ] 未来3-5年技术趋势判断是什么？
- [ ] 核心客户的需求正在发生什么变化？
- [ ] 最大的竞争对手会在18个月内做什么？
- [ ] 我们最大的能力短板是什么？
- [ ] 最大的战略机会窗口在什么时间段？

**三定自查**：
- [ ] 达标目标和挑战目标的差距在哪？
- [ ] 策略对目标的可支撑度是多少？
- [ ] 最高级的控制点是什么？还需要什么条件？
