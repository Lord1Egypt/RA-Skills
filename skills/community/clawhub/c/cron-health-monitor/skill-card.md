## Description: <br>
Monitors OpenClaw Agent cron task health and system resources, detects failed or stalled tasks, sends alerts, retries failures, and produces health reports. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and operators of OpenClaw Agent use this skill to monitor scheduled task health and resource thresholds, receive alerts, retry failed tasks, and review daily health status. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Security evidence marks the release suspicious because it may involve external Booking.com automation that can use live account credentials, optional 2FA automation, guest messaging, rate changes, and a persistent browser session. <br>
Mitigation: Install only after reviewing and trusting the external automation code, protect credentials and browser session data, require manual review before guest messages or rate changes, and confirm the automation is allowed for the account. <br>
