# Slidev Assist

🎨 **主题参考**
- `default` — 默认简洁主题
- `seriph` — 衬线字体风格
- `unocss` — UnoCSS 驱动，高度可定制
- 更多：`npm search slidev-theme`

📐 **布局参考**
- `cover` / `center` / `two-cols` / `image-right` / `image-left` / `section` / `statement` / `default`

🛠️ **命令行**
```bash
# 初始化项目
npm init slidev

# 启动预览
npx @slidev/cli slides.md
npx @slidev/cli slides.md --remote  # 手机可控制翻页

# 导出
npx @slidev/cli export slides.md
npx @slidev/cli export slides.md --format pptx

# 在线快速尝试
# https://sli.dev/new
```
