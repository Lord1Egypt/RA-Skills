## Compliance Moats

- FTC Green Guides: "eco-friendly" only with EPA Safer Choice or named certification
- OSHA HazCom: no "chemical-free" without SDS verification
- FTC 2023 Endorsement Guides: no conditional review requests
- Bonding vs. licensing: correctly distinguishes (cleaning is not state-licensed in most states)
- Move-out cleaning: no "deposit back guaranteed" (landlord's discretion — false promise)
- IRS worker classification: neutral "cleaning professional" language avoids misclassification exposure
