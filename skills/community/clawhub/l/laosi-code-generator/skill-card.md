## Known Risks and Mitigations: <br>
Risk: Generated scaffolding can create files in an unintended directory or affect an existing repository if paths are chosen carelessly. <br>
Mitigation: Choose the output directory deliberately and have the agent preview target paths before creating files, especially inside an existing repository. <br>
