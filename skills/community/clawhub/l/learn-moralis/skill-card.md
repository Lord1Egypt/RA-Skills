## Description: <br>
Helps users learn about Moralis Web3 data products, answer exploratory questions, and choose between the Moralis Data API and Streams skills. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[novnski](https://clawhub.ai/user/novnski) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and Web3 builders use this skill to understand Moralis capabilities, supported chains, pricing, setup requirements, and which Moralis technical skill to use for data queries or real-time event monitoring. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Users may expose Moralis API keys while following setup guidance or preparing to use downstream Moralis skills. <br>
Mitigation: Do not paste API keys into chat; keep MORALIS_API_KEY in local environment configuration and review downstream Moralis skills before granting API-key, network, webhook, or file-write access. <br>
Risk: Trading-bot and alert examples could be mistaken for complete financial automation guidance. <br>
Mitigation: Treat trading-bot examples as implementation ideas only, and require separate review, testing, and risk controls before building or running financial automation. <br>
