## [0.1.0] - 2026-05-25
Initial release. Six-phase workflow that converts a building profile (occupancy type, vintage, gross floor area, climate zone, conditioned-area breakdown, schedules, envelope, HVAC / DHW / lighting / plug-load inventory, on-site generation, control system), ≥12 months of utility data (electricity, natural gas, district steam / chilled water / hot water, fuel oil, propane, water — bill-by-bill with rate schedules and time-of-use windows), walkthrough findings, and metered / spot-measured equipment data into a DRAFT ASHRAE / ACCA Standard 211 Level 2 commercial building energy audit report — covering Phase 1 intake and ASHRAE-211 Level-conformance check (Level 1 vs Level 2 vs Level 3 scoping with explicit acknowledgement that Level 2 mandates ±15–25% accuracy and an engineering-grade ECM analysis), Phase 2 utility-data analysis (12+ months, normalized for weather using HDD / CDD from a named weather station and base-temperature method, EUI in kBtu/ft²/yr with both site EUI and source EUI, ENERGY STAR Portfolio Manager score where eligible, end-use disaggregation, demand profile, time-of-use breakdown, rate-schedule analysis, anomaly flags), Phase 3 building / site description and load profile (envelope U-values, HVAC primary / secondary inventory, DHW, interior / exterior lighting LPD, plug loads, vertical transportation, process loads, controls and BAS architecture, on-site generation), Phase 4 ECM identification and engineering-grade analysis (per-ECM scope, baseline, proposed, savings calculation method, savings in kWh / therms / gallons / kBtu, demand-savings kW, $ savings at the marginal rate, interactive effects, installed cost, simple payback, NPV, SIR, IRR, lifecycle cost, non-energy benefits, M&V Option A / B / C / D per IPMVP, implementation risk, sequencing with other ECMs, decarbonization / electrification flag), Phase 5 implementation roadmap (immediate / 1-year / 3-year / 5-year sequencing, financing options including utility rebates, IRA §179D, Investment Tax Credit, ITC bonus adders, PACE, on-bill financing, performance contracting, ESPC), Phase 6 ASHRAE-211 Annex C conformance self-check, assembly with executive summary, PE / CEA sign-off block (unsigned), open-questions list, and evidence index — for owner, utility-incentive-program, IRA §179D, NYC LL97, BERDO 2.0, NYSERDA, Title 24, and PACE-underwriter review.
