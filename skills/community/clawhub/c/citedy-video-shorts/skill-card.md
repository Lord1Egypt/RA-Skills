## Description: <br>
Generate and publish branded AI avatar lip-sync video shorts to Instagram Reels and YouTube Shorts with custom avatars, generated scripts, burned-in subtitles, and optional social publishing. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External creators, marketers, and developers use this skill to turn a topic or product description into a short AI avatar video, review costs before paid calls, retrieve download URLs, and optionally publish to connected Instagram Reels or YouTube Shorts accounts. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Use can spend Citedy credits and publish generated videos to connected social accounts. <br>
Mitigation: Show estimated credit and dollar cost before paid operations, require explicit approval before generation or publishing, and review generated content and metadata before publication. <br>
Risk: The skill requires a Citedy agent API key for authenticated API calls. <br>
Mitigation: Install only when Citedy is trusted with the key, store the key in the local agent configuration, and limit use to the documented Citedy API endpoints. <br>
Risk: Registration can expose the agent name and referral links may be promotional. <br>
Mitigation: Use a non-identifying agent name if hostname disclosure matters and treat referral-link use as promotional. <br>
