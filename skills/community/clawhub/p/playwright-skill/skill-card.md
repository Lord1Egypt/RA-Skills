## Description: <br>
Provides Playwright-based browser automation for testing pages, forms, screenshots, responsive layouts, login flows, links, and other browser interactions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[vmercel](https://clawhub.ai/user/vmercel) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and engineers use this skill to automate browser-based checks against local, staging, or user-provided web targets. It helps create and run Playwright scripts for UI validation, form flows, screenshots, responsive review, authentication checks, link checks, and related browser tasks. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can drive live browser interactions and local command execution. <br>
Mitigation: Review generated scripts before running them, use local or staging targets where possible, and require explicit approval before form submission, real-account login, or authenticated requests. <br>
Risk: The security scan notes that referenced runtime files were not included for review. <br>
Mitigation: Verify the installed runtime files before running setup or node commands, and avoid using production credentials until the installation has been inspected. <br>
