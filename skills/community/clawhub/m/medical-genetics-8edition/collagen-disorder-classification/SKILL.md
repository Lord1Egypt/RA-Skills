# 胶原蛋白病的分类与分子基础

## 何时使用
当患者表现出以下特征时启用本技能：
- 骨脆、关节过度活动、皮肤弹性异常、基膜相关病变等结缔组织症状
- 存在家族遗传史
- 已排除非遗传性或非胶原蛋白相关的结缔组织疾病

## 执行步骤

1. **确认胶原蛋白类型**
   - 识别临床表型最可能关联的胶原类型（I、II、III、IV等）
   - 注意：人体有十余种胶原，其中I、II、III为间质胶原，IV型分布于基膜

2. **分析胶原结构组成**
   - I型：2条α₁链 + 1条α₂链 → 分布于皮肤、肌腱、韧带
   - II型：3条α₁链 → 分布于透明软骨、椎间盘髓核、玻璃体
   - III型：3条α₁链 → 分布于血管壁、胎盘等高伸展性组织
   - IV型：2条α₁链 + 1条α₂链 → 形成交叉巨分子，位于基膜

3. **关联典型疾病**
   - 成骨不全 → I型胶原异常
   - 埃勒斯-当洛综合征 → 多种胶原或相关蛋白缺陷
   - 奥尔波特综合征 → IV型胶原缺陷
   - 表皮松解症 → 锚定胶原异常

4. **判断分子机制**
   - 区分“胶原量减少”与“结构变异”
   - 特别关注甘氨酸密码子突变（影响三螺旋稳定性）
   - 结合突变位置（氨基端、羧基端、螺旋区）推断功能影响

## 输出要求
返回胶原蛋白病的具体类型、对应胶原亚型、α链组成及分子缺陷机制。
