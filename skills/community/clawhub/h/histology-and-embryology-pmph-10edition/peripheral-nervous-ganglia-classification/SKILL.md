# 周围神经系统神经节分类

## 适用场景
- 观察组织切片中的神经节结构
- 区分感觉性与运动性神经节
- 教学或研究周围神经系统解剖

## 执行步骤
1. **判断神经节类型**：
   - **脊神经节**：位于脊神经背根，属感觉神经节
   - **脑神经节**：位于某些脑神经干上，结构类似脊神经节
   - **自主神经节**：包括交感神经节（脊柱旁/前）和副交感神经节（器官附近或内），属运动神经节

2. **分析节细胞特征**：
   - **感觉神经节**（脊/脑）：假单极神经元，胞体圆形，核居中，尼氏体细小分散，突起呈T形分支
   - **自主神经节**：多极运动神经元，核常偏位或双核，尼氏体呈细颗粒状均匀分布

3. **识别支持细胞差异**：
   - **感觉神经节**：胞体由卫星细胞包裹，T形分支处由施万细胞包裹
   - **自主神经节**：卫星细胞较少，包绕胞体及突起

4. **确认神经纤维类型**：
   - **脊神经节**：主要为有髓神经纤维
   - **自主神经节**：节后纤维为无髓神经纤维

> 注意：本技能不适用于中枢神经系统内的神经核团。
