## 二、综合单价构成（分部分项工程费）

### 2.1 综合单价组成公式

```
综合单价 = 人工费 + 材料费 + 机械费 + 管理费 + 利润 + 风险费

其中：
  人工费 = ∑(定额人工消耗量 × 人工单价)
  材料费 = ∑(定额材料消耗量 × 材料单价)
  机械费 = ∑(定额机械台班消耗量 × 机械台班单价)
  
  管理费 = 人工费 × 48.08%
  利润   = (人工费 + 管理费) × 29.15%
  
  风险费 = 由投标人根据工程实际情况计取（山东安装工程一般不计取）
```

### 2.2 重要区分：基价 vs 计费基础

| 概念 | 定义 | 用途 |
|------|------|------|
| **基价** | 人工费 + 材料费 + 机械费 | 定额中给出的"定额单价" |
| **计费基础** | **仅人工费** | 计算管理费、利润的基数 |
| **综合单价** | 基价 + 管理费 + 利润 | 分部分项工程的报价单价 |
| **含税造价** | 综合单价 × (1 + 9%) | 最终结算单价 |

> ⚠️ **常见错误**：用"基价 × 费率"计算管理费和利润。正确做法是用**人工费**作为基数。

### 2.3 综合单价速算公式

```
已知：定额基价 = 人工费 + 材料费 + 机械费

税前综合单价 = 基价 + 人工费 × 91.25%
              = 基价 + 人工费 × (48.08% + (1+48.08%)×29.15%)
              
含税综合单价 = 税前综合单价 × 1.09
```

## 三、措施项目费构成

### 3.1 措施项目费分类

```
措施项目费
├── 单价措施项目（可计量）
│   ├── 脚手架搭拆费
│   ├── 垂直运输费
│   ├── 超高施工增加费
│   ├── 大型机械设备进出场及安拆费
│   └── 施工排水、降水费
│
└── 总价措施项目（按项/费率计取）
    ├── 安全文明施工费（不可竞争）
    ├── 夜间施工增加费
    ├── 二次搬运费
    ├── 冬雨季施工增加费
    ├── 已完工程及设备保护费
    ├── 临时保护设施费
    └── 其他总价措施
```

### 3.2 安装工程特殊措施费

| 措施项目名称 | 计算基数 | 费率 | 说明 |
|-------------|---------|------|------|
| 安装与生产同时进行增加费 | 人工费 | 10% | 含管理费、利润 |
| 在有害身体健康环境中施工增加费 | 人工费 | 26% | 含管理费、利润 |
| 高层建筑增加费 | 人工费 | 2%~12% | 按高度/层数递增 |

> ⚠️ 以上特殊措施费已包含管理费和利润，不再另行计取管理费和利润。

## 四、其他项目费构成

| 项目名称 | 定义 | 计取方式 |
|---------|------|---------|
| **暂列金额** | 招标人在工程量清单中暂定并包括在合同价款中的一笔款项 | 按招标文件给定金额计入，结算时按实调整 |
| **暂估价** | 招标人在工程量清单中提供的用于支付必然发生但暂时不能确定价格的材料、工程设备的单价以及专业工程的金额 | 材料/设备暂估价按招标给定价计入，结算时按实调整 |
| **计日工** | 承包人完成招标人提出的施工图纸以外的零星项目或工作 | 按招标文件给定的暂定数量×综合单价 |
| **总承包服务费** | 总承包人为配合协调发包人进行的专业工程分包、采购和保管等服务所需的费用 | 按分包专业工程估算造价的1%~3%计取 |

## 五、计价程序流程图（对标广联达）

```
【Step 1】分部分项工程费计算
  └─> 工程量清单 → 套定额组价 → 综合单价 → 合价
  
【Step 2】措施项目费计算
  ├─> 单价措施：同分部分项方法（套定额）
  └─> 总价措施：按费率或给定金额计算
  
【Step 3】其他项目费计算
  └─> 按招标文件给定的暂列金额、暂估价、计日工、总包服务费
  
【Step 4】规费计算（⚠️ 2025年11月1日起已取消）
  └─> 原规费中的社保、公积金已并入人工费和管理费
  
【Step 5】税金计算
  └─> 税前造价 = 分部分项 + 措施项目 + 其他项目
      税金 = 税前造价 × 9%（一般纳税人）
      
【Step 6】工程造价汇总
  └─> 总造价 = 税前造价 + 税金
```

## 六、广联达GCCP5.0计价程序界面逻辑对照

广联达软件的费用汇总表顺序如下，cnsdoce1技能输出报表时应遵循此顺序：

| 序号 | 汇总内容 | 计算式 | 备注 |
|:----:|---------|--------|------|
| 1 | 分部分项工程费 | ∑(综合单价 × 工程量) | 含人工费、材料费、机械费、管理费、利润 |
| 2 | 措施项目费 | 单价措施 + 总价措施 | |
| 2.1 | 其中：单价措施费 | ∑(综合单价 × 工程量) | 同分部分项 |
| 2.2 | 其中：总价措施费 | ∑(按项计取) | 安全文明施工费等 |
| 3 | 其他项目费 | 暂列金额 + 暂估价 + 计日工 + 总包服务费 | |
| 4 | 规费 | **已取消** | 2025年11月1日起执行 |
| 5 | 税前造价 | 1 + 2 + 3 | |
| 6 | 税金 | 5 × 9% | |
| 7 | **工程造价** | 5 + 6 | |

## 七、与cnsdoce/references/fee_standard.md的关系

本文件与 `fee_standard.md` 是**互补关系**：

- **fee_standard.md**：聚焦"费率数值"和"取费计算步骤"（怎么做）
- **cost_composition.md**：聚焦"费用构成框架"和"计价程序流程"（是什么、为什么）

两者共同构成完整的取费规则体系，组价计算时应同时引用。
