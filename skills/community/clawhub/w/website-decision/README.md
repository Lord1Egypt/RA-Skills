# 🌐 website-decision

网站开发辅助决策系统 — 输入产品类型和名称，自动生成多维度可行性分析报告。

## 功能

- 🔍 **搜索热度分析** — 关键词搜索量、趋势判断
- ⚔️ **竞品格局分析** — 竞品数量、头部玩家、差异化机会
- 📈 **行业市场分析** — 市场规模、增长率、政策环境
- 👥 **用户画像分析** — 目标用户、需求痛点、使用场景
- 🚀 **流量潜力评估** — SEO难度、搜索量、内容营销空间
- 💰 **商业模式建议** — 会员/广告/电商模式推荐
- 💸 **成本估算** — 域名/服务器/开发/运维/推广全链路
- 📢 **推广策略** — 冷启动→增长→规模化全阶段规划
- 🔧 **技术选型指南** — 前端/后端/部署方案对比
- ⚠️ **风险提示** — 政策/竞争/技术/运营风险矩阵
- 🎯 **综合评分** — 6维加权评分 + 明确决策建议

## 安装

作为 WorkBuddy/CodeBuddy 技能使用：

```bash
# 复制到技能目录
cp -r website-decision ~/.workbuddy/skills/
```

## 使用

```
触发词：网站决策、网站可行性、网站评估、website decision
示例：
  网站决策：做一个AI写作工具网站
  帮我评估做在线教育平台网站可行吗
  分析宠物社交网站的可行性
```

## 评分模型

| 维度 | 权重 |
|------|------|
| 搜索热度 | 25% |
| 竞争格局 | 20% |
| 市场前景 | 20% |
| 变现能力 | 15% |
| 开发可行性 | 10% |
| 流量获取 | 10% |

- ≥80 分 → 强烈建议做
- 65-79 分 → 谨慎推进
- 50-64 分 → 暂缓观望
- <50 分 →  不建议做
