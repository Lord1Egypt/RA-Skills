# 量子电池项目评估清单

## 快速筛查表

### A. 参数真实性检查 ⚠️

| 检查项 | 合格 | 不合格 |
|--------|------|--------|
| 充电时间 | 飞秒~秒级 | "瞬间" / "1秒充满" |
| 能量密度 | 合理提升 | "提升1000倍" |
| 工作温度 | 室温或需冷却 | "任何温度" |
| 循环寿命 | 有具体数据 | "无限循环" |

### B. 学术可验证性

```
□ 是否有arXiv/同行评审论文？
□ 研究人员是否可查证？
□ 所属机构是否真实？
□ 技术路线是否与主流一致？
```

### C. 商业化承诺评估

| 承诺 | 风险等级 |
|------|----------|
| "已量产" | 🔴 高风险 |
| "即将量产" | 🟠 中高风险 |
| "5年内商业化" | 🟡 中等风险 |
| "15-20年" | 🟢 低风险 |

### D. 投资建议

```
□ 是否技术成熟度不足(TRL 2-3)?
□ 是否无法短期变现?
□ 是否可通过ETF/材料股间接布局?
```

## 决策树

```
收到量子电池项目信息
    │
    ├── 参数是否离谱？ ──是──→ 🔴 高风险项目
    │
    └── 参数是否合理？
            │
            ├── 有学术支撑？ ──否──→ 🟠 需进一步验证
            │
            └── 有学术支撑？ ──是──→ 🟡 可关注但谨慎投资
```
