## Description: <br>
Audit React components for performance, accessibility, hook correctness, prop design, and re-render optimization - identify memo opportunities and state management issues. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[charlie-morrison](https://clawhub.ai/user/charlie-morrison) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and frontend engineers use this skill to review React components and custom hooks for re-render issues, hook correctness, accessibility gaps, prop design problems, and state management anti-patterns. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may guide an agent to inspect React source files and produce recommendations that could be incorrect or unsuitable for the application. <br>
Mitigation: Review the audit findings and any suggested code changes before applying them. <br>
