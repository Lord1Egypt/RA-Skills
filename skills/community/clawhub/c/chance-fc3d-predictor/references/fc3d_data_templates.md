## 四、七项检查清单

在输出任何分析报告前，必须完成以下检查：

### ✅ 检查1：数据时效性
- [ ] 确认最新开奖期号
- [ ] 确认数据更新时间
- [ ] 标注数据截止日期

### ✅ 检查2：频率统计
- [ ] 百位0-9各出现次数
- [ ] 十位0-9各出现次数
- [ ] 个位0-9各出现次数
- [ ] 热温冷分类准确

### ✅ 检查3：遗漏统计
- [ ] 各位置最大遗漏值
- [ ] 当前各数字遗漏值
- [ ] 遗漏状态标注（热/温/冷）

### ✅ 检查4：和值跨度
- [ ] 和值分布统计
- [ ] 跨度分布统计
- [ ] 高频区间确认

### ✅ 检查5：推荐号码
- [ ] 提供至少3种策略
- [ ] 每种策略说明理由
- [ ] 标注号码类型（组三/组六）
- [ ] 预估覆盖注数

### ✅ 检查6：风险提示
- [ ] 包含"彩票随机"声明
- [ ] 提醒"理性投注"
- [ ] 提醒"量力而行"

### ✅ 检查7：数据来源
- [ ] 标注数据获取方式
- [ ] 说明数据可靠性
