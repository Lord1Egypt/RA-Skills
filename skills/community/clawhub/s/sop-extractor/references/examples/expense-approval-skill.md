# 费用报销审批

> 「先看标准，再看金额，最后看票据——三关都过了才放行。」

## 使用说明

**触发条件**：员工提交费用报销申请时
**涉及角色**：申请人 → 部门主管 → 财务审核 → 出纳付款
**预期输出**：审批通过/驳回 + 付款处理
**预计耗时**：3-5个工作日
**自动化概况**：共 8 步，🟢自动 1 步 🟡半自动 3 步 🔴人工 4 步

## 工作流程

### Step 1: 接收报销申请
- **做什么**：收到员工提交的报销单和发票
- **看什么**：报销单是否填写完整（金额、事由、发票号、部门）
- **产出什么**：确认收到，告知预计处理时间
- **注意什么**：报销单和发票金额必须一致，不一致的直接退回

### Step 2: 票据初审
- **做什么**：检查发票合规性
- **看什么**：
  - 发票是否在有效期内（开票日期不超过6个月）
  - 发票抬头是否为公司全称
  - 发票金额与报销单金额是否一致
  - 发票类型是否符合报销范围（增值税普票/专票/电子发票）
- **产出什么**：初审结果（通过/退回/需补充）
- **注意什么**：电子发票需要查重，防止重复报销

### Step 3: 金额分类审批
- **做什么**：根据金额走不同审批流程
- **看什么**：报销总金额
- **产出什么**：进入对应审批链
- **注意什么**：

  **决策节点：金额判断**
  - 金额 ≤ 2000元 → 部门主管审批即可（Step 4a）
  - 2000元 < 金额 ≤ 10000元 → 部门主管 + 财务经理审批（Step 4b）
  - 金额 > 10000元 → 部门主管 + 财务经理 + 分管副总审批（Step 4c）

### Step 4a: 主管审批（小额）
- **做什么**：部门主管确认费用合理性
- **看什么**：费用是否在部门预算内、事由是否合理
- **产出什么**：审批通过/驳回
- **注意什么**：主管重点关注"这笔钱花得值不值"，不重复审票据

### Step 4b: 双级审批（中额）
- **做什么**：部门主管 + 财务经理分别审批
- **看什么**：主管看合理性，财务看预算和合规
- **产出什么**：两级都通过才进入下一步
- **注意什么**：财务经理重点关注是否超预算、费用科目是否正确

### Step 4c: 三级审批（大额）
- **做什么**：部门主管 + 财务经理 + 分管副总分别审批
- **看什么**：除上述外，副总关注战略合理性
- **产出什么**：三级都通过才进入下一步
- **注意什么**：大额报销建议附上项目说明或合同

### Step 5: 出纳付款
- **做什么**：审批全部通过后，安排付款
- **看什么**：审批链是否完整、付款方式（银行转账/现金）
- **产出什么**：付款完成，通知申请人
- **注意什么**：付款必须在审批完成后3个工作日内完成

## 决策节点

### 决策点: 发票初审
- **判断条件**：发票合规性检查结果
- **如果全部合规**：→ 进入金额分类
- **如果部分不合规**：→ 退回补充，说明具体问题
- **如果严重不合规（假票/过期）**：→ 直接驳回，记录原因

### 决策点: 金额分级
- **判断条件**：报销总金额
- **≤ 2000**：→ 走主管审批
- **2000-10000**：→ 走双级审批
- **> 10000**：→ 走三级审批

## 边界条件

**不启动的情况**：
- 费用发生在公司成立之前
- 发票已用于其他报销
- 费用类型明确不在报销范围内（如个人消费）

**需要找人的情况**：
- 拿不准发票是否合规 → 找财务经理确认
- 预算不足但确有需要 → 找分管副总特批
- 跨部门费用分摊 → 找两个部门主管协商确认比例

**直接驳回的情况**：
- 发票为伪造/变造
- 同一张发票重复提交
- 报销金额明显超出标准（如差旅费超标200%以上）

## 质量标准

完成此工序后，检查以下项目：
- [ ] 发票合规性100%验证
- [ ] 审批链完整，无越级审批
- [ ] 付款金额与审批金额一致
- [ ] 付款在审批完成后3个工作日内完成
- [ ] 报销记录已归档

## 常见问题

| 问题 | 原因 | 处理方式 |
|------|------|---------|
| 发票金额和报销单差几分钱 | 四舍五入差异 | 差额≤1元可接受，标注原因 |
| 电子发票查重发现重复 | 员工误提交 | 驳回并告知员工已有记录 |
| 审批人出差无法审批 | 审批人不在岗 | 转交其代理人审批 |
| 费用科目选错 | 员工不了解科目分类 | 退回并告知正确科目 |
| 预算不足 | 部门预算已用完 | 告知员工走预算追加流程 |

## 隐性知识

> 这些是财务同事"心里知道"但通常不会写在文档里的东西。

- 月底最后两天是报销高峰期，建议提前准备
- 差旅费报销最容易出错的是餐费——只能报工作餐，不能报招待餐
- 同一个人一个月内多次报销同一类型费用，可能需要关注是否合理
- 有些部门习惯年底集中报销，但12月的报销要在12月31日前完成，否则影响年度预算

## 异常处理

| 异常情况 | 处理方式 |
|---------|---------|
| OA 系统不可用 | 降级为邮件审批，事后补录系统 |
| 发票查验接口超时 | 重试 2 次，仍失败则人工查验 |
| 审批人超过 3 天未处理 | 自动提醒审批人，同时通知申请人 |
| 凭证（OA API Key）缺失 | 引导用户配置，配置前暂停自动化环节 |
