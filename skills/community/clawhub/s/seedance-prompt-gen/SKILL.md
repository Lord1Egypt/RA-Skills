# Seedance 视频提示词工业化生成

**版本**: 1.9.2
**适用模型**: Seedance 2.0
**原始文档**: references/视频提示词工业化生成工作流SKILL.md（只读参考，禁止修改）

---

## 技能描述

将剧本/故事文本工业化转化为 Seedance 2.0 可直接使用的视频提示词。输入一段剧本，输出完整的文生图提示词（人物/场景/道具）+ 分组视频提示词 + 台词核对表 + 字段自检表。

## 触发条件

- 用户提供剧本/故事，要求生成视频提示词
- 用户要求"按 Seedance 工作流生成提示词"
- 用户提到"视频分镜""视频提示词""Seedance 提示词"等关键词

## 执行前必读

**每次执行前，必须完整阅读 `references/视频提示词工业化生成工作流SKILL.md`，确保理解全部红线与规范。**

## 执行流程

### Step 0: 红线确认

在开始任何生成之前，确认 10 条红线（详见参考文档"红线警告"章节）：
1. 台词遗漏 → 逐句提取编号统计
2. 文生图提示词缺失 → 先完成角色/场景/道具
3. 画幅标注 → 每镜头标注
4. 禁止旁白标注 → 转化为画面描述
5. 空间连续性 → 段首继承段尾位置
6. 空间站位禁止脑补 → 逐字对照原文
7. 人物移动路径写清 → 从A走到B
8. 防越轴检查表 → 逐格填写具体描述
9. 分镜字段统一 → 9+1 字段公式
10. 负面词动态生成 → 三层结构

### Step 1: 需求确认

向用户确认 2 项（P0）：
| 确认项 | 选项 |
|:---|:---|
| 画幅比例 | 9:16 / 16:9 / 21:9 |
| 视觉风格 | 写实电影感+现代都市 / 其他（用户自定义） |

默认设置（无需确认）：分组数量 14组/1000字（浮动≤5%），每组时长 12-15秒

### Step 2: 台词逐句提取（P0）

流程：标记对话 → 标记心理描写 → 标记旁白 → 提取表格 → 二次核对 → 统计字数 → 计算时长 → 超长拆分

语速标准（详见参考文档 Step 2）：
- 舒缓抒情：3.0字/秒 | 沉思独白：3.2字/秒 | 默认对白：4.4字/秒
- 剧情推进：4.6字/秒 | 客观旁白：4.8字/秒 | 紧张对峙：5.0字/秒 | 慌乱惊恐：5.5字/秒

**铁律**：镜头时长 ≥ 台词时长（字数÷语速），禁止压缩语速凑时长

### Step 3: 空间状态追踪（P0）

流程：识别场景 → 逐句标记位置 → 记录初始位置 → 标记走位 → 按段落拆分 → 建立传递链 → 填写防越轴检查表

输出空间状态追踪表（模板详见参考文档 Step 3）

### Step 4: 抽象→具象转译（P0）

核心拷问："摄影机拍得到吗？麦克风录得到吗？拍不到/录不到的，必须转译。"

转译速查表详见参考文档 Step 4。转译结果直接用于分镜生成，不单独输出。

### Step 5: 文生图提示词生成（P0）

按顺序生成：

**5.1 人物文生图提示词**（详见参考文档 §4）
- 前置格式：比例21:9，真人写实风格，右侧头部特写+左侧全身三视图，中性全局照明，纯白无缝背景，8K
- 10大系统（缺一不可）：人物基础身份、脸型骨相、核心视觉记忆点、眼部、眉毛、鼻部、唇部、发型、皮肤与生活痕迹、服装
- 人物一致性铁律：形象/五官/发型/服装/饰品/年龄/气质 统一
- 禁止：描述光影/色调/效果，纯白背景以外的背景描述

**5.2 场景文生图提示词**（详见参考文档 §4.1）
- 前置格式：比例21:9，35mm人文焦距，中性全局照明，实景，画面中禁止出现任何人物
- 描述结构：空间布局 → 关键物件 → 光线氛围 → 背景细节 → 焦距光圈
- 禁止：出现人物、纯色背景、描述光影效果、描述色调/情绪氛围词

**5.3 道具文生图提示词**（详见参考文档 §4.2）
- 前置格式：比例16:9，50mm微距焦距，f/1.4-f/1.8浅景深，纯白背景，无明显光源方向，全局均匀照明
- 描述结构：材质 → 颜色 → 形态 → 状态 → 多角度
- 禁止：描述文字内容、光影效果、非纯白背景、用途描述

### Step 6: 分组拆分与视频提示词生成

**分组公式**：总字数÷4.4÷15 ≈ 组数（浮动≤5%）

**分组优先级**：剧情闭环节点 > 站位变化节点 > 语义自然断点 > 15秒硬限制

**每组输出内容**（缺一不可）：
1. 组标题 + 空间状态
2. 全局风格与环境（核心限制 + 画质光影，含色彩主题/视觉符号）
3. 镜头描述（9+1字段，时间码组内归零）
4. 关键约束（五层级：空间/光线/运镜/画面/风格）
5. 音频规范（全局音轨 + 逐镜头设计）
6. 负面词（三层：基础 + 类型专属 + 场景专属）

**镜头 9+1 字段公式**：
```
视角/景别 | 机位 | 焦段/光圈 | 运镜 | 主体动作(含台词) | 环境细节 | 构图/取景 | 焦点/景深 | 画质标识 | (可选)光线
```

**组内模式**：
- 模式A：单镜到底（1组=1长镜头，内部按 ┃时间┃描述 分段）
- 模式B：多镜快切（1组=2-4镜头，累计≤15s）

**镜头视觉节奏**（组内台词>20字时强制启用）：
- 20-40字：≥2镜
- 40-55字：≥2镜或单镜+视觉动线
- 55字+：≥3镜或单镜+强视觉动线

**运镜衔接5模式**：同向衔接、反向衔接、静切动、动切静、同质叠加

**音频规范铁律**：AI默认加BGM，必须显式控制。无声场景写"禁止BGM"，有叙事需求逐镜头设计声场。

### Step 7: 自检（原位修正，不另起输出）

逐项核对 22 项检查清单（详见参考文档"检查清单"章节），重点：
1. 台词核对：逐句比对，确认无遗漏
2. 台词时长核对：镜头时长 ≥ 台词时长
3. 字段核对：9段或10段=合格
4. 空间核对：相邻组人物位置/朝向连续，无跳轴

### Step 8: 写入文件

**铁律**：自检通过后直接写入 .md 文件，不在对话中完整输出

**文件命名**：`{剧本名}_场{场号}_视频提示词_V{版本号}.md`

**文件内容**：文生图提示词(人物/场景/道具) + 全部分组视频提示词 + 台词核对表 + 字段自检表

**向用户报告**：仅报告文件路径、总组数、总时长

## 关键约束提醒

- 每组 ≤15秒，时间码组内归零
- 每组自包含，不依赖前后组
- 台词用英文双引号""包裹嵌入镜头
- 禁止旁白标注（旁白/心想/回忆）→ 转化为画面描述
- 构图字段必须含取景边界（裁切位置）
- 负面词三层：基础 + 类型专属 + 场景专属
- 单镜分段用 ┃时间┃描述 格式
- 层级约束五级完整，无约束的级写"无"

## 输出产物

一个 .md 文件，包含完整可提交到 Seedance 的提示词内容。
