# Output style

Default shape:

- **Position:** side to move, material, king safety, immediate threats.
- **Best candidate:** move + one-line reason.
- **Lines:** 2-3 short PVs with evals when available.
- **Human plan:** what to look for next, not just engine notation.
- **Caveats:** image/FEN assumptions, missing engine, or active-game guardrail.

Avoid walls of engine notation. If the user asks for depth, include more PV detail.
