## Description: <br>
Generate cryptographically secure random passwords. Customizable length, character types, exclude similar characters. Pure Python standard library, no API key required. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, engineers, and security-conscious users use this skill to generate strong random passwords, API keys, or tokens locally without external services. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated passwords can be exposed if copied into shared terminal logs, chat transcripts, or shell history. <br>
Mitigation: Treat generated passwords as secrets and store final values in a trusted password manager. <br>
