# IDworld Planning Framework

Use this reference when producing a full IDworld social media product plan or when the user asks for detailed Japanese X/Twitter copy.

## Brand Context

IDworld is an オリジナルグッズ production brand for the Japanese market. Core audiences include:

- 同人作者
- 插画师
- 个人创作者
- 推し活用户
- 小型社团
- 活动参展者
- 想制作原创周边的人

The social content should not merely explain products. It should make the user feel:

> “我也想用自己的插画做一个。”

## Content Principles

### 1. Attract, Explain, Convert

Correct order:

```text
有趣/好看/特别
-> 这是什么商品
-> 用户可以怎么用
-> 商品规格
-> 链接
```

Avoid opening with:

```text
商品名
-> 尺寸
-> 参数
-> 工艺说明
-> 链接
```

### 2. Feeling Before Specs

Do not open with specs such as:

- 全41種類
- 4種類の箔カラー
- 164通り
- 58mm
- 白版調整

Use sensory and emotional hooks first:

- 光の角度で、きらめきが変わる
- ただの缶バッジじゃない
- 推しやOCをもっと特別に
- 透明感を活かした仕上がりに
- 見ても楽しい、触っても楽しい

### 3. One Post, One Task

Split one product into a matrix, for example:

- 新品预告
- 正式上线
- 实拍视频
- 工艺对比
- 尺寸对比
- 设计案例
- 使用场景
- FAQ
- 活动提醒
- 下单链接

### 4. Content Tasks

Every content item must belong to one of:

- 曝光型
- 种草型
- 收藏型
- 转化型
- 互动型
- 信任型

Recommended balance:

- 70%: 种草/传播内容
- 20%: 转化内容
- 10%: 互动内容

## Copy Structures

### Normal Image Post

```text
第一行：视觉钩子
第二段：商品是什么
第三段：用户能获得什么效果
第四段：规格/特点
第五段：链接/行动引导
最后：标签
```

### Video Post

```text
第一行：视频看点
第二行：商品特点
第三行：使用场景或行动引导
标签
```

### FAQ Post

```text
第一行：用户常见问题
第二段：直接回答
第三段：注意事项
第四段：相关商品/链接
标签
```

## Quality Standard

Good content usually:

- Can be understood at first glance.
- Has a hook in the first line.
- Has a clear user scene.
- Does not only list parameters.
- Shows a clear image/video subject.
- Gives a reason to save or repost.
- Is not always a hard sell.
- Makes users feel they want to make one with their own illustration.

Weak content usually:

- Opens with specs.
- Has an overcrowded image.
- Makes the user unsure where to look.
- Reads like a product manual.
- Has no use scene.
- Has no emotional value.
- Has no repost reason.
- Puts every piece of information into one post.

## Case Logic

Two new-product posts both reached about 10,000 impressions, but behaved differently:

- Case A, 缶詰モチーフ缶バッジ: 201 likes, 51 reposts, 57 saves.
- Case B, 箔押し立体感缶バッジ: 118 likes, 13 reposts, 55 saves.

Conclusion:

- Saves were similar, so both had reference value for target users.
- Case A had more likes and reposts because it had a novel shape, was understandable at a glance, had suspense in the copy, and gave users a reason to share.
- Case B still had saves, but fewer reposts because it leaned explanatory, opened with specs, and had too much image information for a first-wave breakout post.

Therefore, the first wave of new product content should prioritize communication and propagation, not a product-manual style explanation.
