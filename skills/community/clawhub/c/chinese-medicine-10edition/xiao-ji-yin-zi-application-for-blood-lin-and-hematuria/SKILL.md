# Xiao Ji Yin Zi Application for Blood Lin and Hematuria

## When to Use
当患者出现以下表现时启用本技能：
- 尿中带血（血淋或尿血）
- 小便频数、赤涩、热痛
- 舌红、脉数
- 病机为下焦瘀热、灼伤血络

## Precautions
- 非热结下焦所致出血（如脾不统血）
- 虚寒性尿血

## Execution Steps
1. **确认方剂组成**：生地黄、小蓟根、通草、滑石、山栀仁、蒲黄（炒）、淡竹叶、当归、藕节、甘草各等分。
2. **确定用法用量**：咀嚼后每服半两，水煎，空腹服用。
3. **分析配伍结构**：
   - 君药：小蓟（凉血止血、利尿通淋）
   - 臣药：生地黄、蒲黄、藕节（凉血止血兼祛瘀）
   - 佐药：滑石、淡竹叶、通草（清热利水），栀子（清三焦火），当归（养血和血、防寒凉伤正）
   - 佐使药：甘草（缓急止痛、调和诸药）
4. **评估治疗目标**：实现“止血不留瘀、清热不伤正”的疗效。
5. **关联现代适应症**：适用于急性泌尿系感染（膀胱炎、尿道炎、肾盂肾炎）及泌尿系结石属热结下焦者。

> 注意：若患者无明显热象或属虚寒体质，禁止使用本方。
