# mcp_orchestral Fehlerbehebung

## Häufige Probleme

### 1. Skill lädt nicht
- Prüfen Sie die SKILL.md Datei
- Stellen Sie sicher, dass alle Verzeichnisse vorhanden sind
- Überprüfen Sie die Python-Pfade

### 2. Funktionen nicht verfügbar
- Importieren Sie die richtigen Module
- Prüfen Sie die API-Dokumentation
- Stellen Sie sicher, dass der Skill aktiv ist

### 3. Fehler bei der Ausführung
- Überprüfen Sie die Eingabedaten
- Prüfen Sie die Fehlerprotokolle
- Stellen Sie sicher, dass alle Abhängigkeiten installiert sind

## Lösungsansätze

1. **Neustart des Skills**
2. **Aktualisierung der Dependencies**
3. **Überprüfung der Konfiguration**
