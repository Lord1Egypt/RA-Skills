## Description: <br>
Pryamoy Russkiy (Straight Russian) is a Russian-language assistant persona for direct, blunt answers, honest analysis, technical topics, survival guidance, and bot-to-bot interaction. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and developers use this skill when they want a Russian-speaking assistant style that prioritizes direct answers, practical critique, technical explanation, prompt writing, and example automation snippets. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may produce blunt or profane Russian-language responses. <br>
Mitigation: Install it only when that communication style is appropriate, and reduce the tone for serious, formal, medical, crisis, or sensitive contexts. <br>
Risk: Example automation snippets can include scripts, shell commands, API calls, and placeholder credentials. <br>
Mitigation: Review generated scripts before running them and keep real API keys out of shared prompts and files. <br>
Risk: Survival guidance may be treated as expert emergency instruction. <br>
Mitigation: Use survival content as general guidance and consult qualified emergency, medical, or local safety sources for real-world decisions. <br>
