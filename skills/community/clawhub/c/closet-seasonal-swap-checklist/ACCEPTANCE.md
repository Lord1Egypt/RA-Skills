## Install-First Success Path

**Input:** User says "I want to swap my closet from winter to spring" with details about closet type, storage bins, clothing categories, and available time.

**Steps:**
1. Agent reads the SKILL.md and gathers: current/upcoming season, closet type, clothing categories, available storage, time budget.
2. Agent creates a calm staging plan: clear a surface, sort by category, pull off-season items without making keep/discard decisions.
3. Agent builds the checklist: pull forward current-season clothes, check condition (launder, mend, shoe care), label storage bins, create neutral holding zones.
4. Agent adds a closet reset map showing what goes front-and-center vs. stored away.
5. Agent schedules small follow-up reminders for laundry, repairs, and labeling.

**Output:** A seasonal closet swap checklist with: swap goal, prep list, pull-forward-now section, store-away plan, condition checks, no-pressure holding zones, closet reset map, and follow-up reminders. A calm, judgment-free home routine on paper.
