### 音色1：沉稳男声

**声音特征**：
- 音高：低（80-150Hz基频）
- 音色：浑厚、沉稳、有磁性
- 语速：偏慢（每分钟120-150字）
- 情感强度：中低，克制内敛
- 停顿：适度停顿，强调重要内容

**参数配置**：

| 参数 | 设置值 | 调整范围 |
|------|-------|----------|
| 基频 | 120Hz | 100-150Hz |
| 语速 | 130字/分钟 | 110-150字/分钟 |
| 音调变化 | 平稳 | 低波动 |
| 情感强度 | 0.4 | 0.3-0.5 |
| 重音位置 | 句首/关键词 | - |
| 停顿频率 | 每句末 | 句末+段末 |

**适用场景**：
- 商务演讲、领导发言
- 年度汇报、工作总结
- 投资路演、商业谈判
- 新闻播报、官方发布

**文案示例**：

```
原文：
"大家好，我是老胡。今天我想和大家分享一个观点：技术成果转化，核心在于'人'。"

声音表达要点：
- 开场"大家好"语速稍慢，建立亲和
- "我是老胡"沉稳有力，自我介绍
- "技术成果转化"语速放慢，强调重点
- 结尾"核心在于'人'"停顿后说出，制造悬念
```

**适用模板匹配**：
- 老胡说视频号（商务休闲）
- 商务精英外观
- 正式报告语言风格

### 语言风格1：正式报告

**风格特征**：
- 语气：正式、严谨、专业
- 结构：总-分-总，层次分明
- 用词：规范用语，少用口语
- 特点：数据支撑、逻辑严密

**参数配置**：

| 参数 | 设置值 | 说明 |
|------|-------|------|
| 句式长度 | 长句为主 | 复合句结构 |
| 专业术语 | 高密度 | 行业专业词汇 |
| 口语化 | 低 | 规范用语 |
| 互动感 | 弱 | 少用问句 |
| 情感色彩 | 低 | 客观理性 |
| 数字引用 | 高频 | 数据支撑 |

**文案模板**：

```
【开场】
各位领导、各位同事：
大家好！今天我向大家汇报XX项目的进展情况。

【正文】
一、总体情况
本季度，我们围绕XX目标展开工作，取得了XX成果。

二、数据分析
从数据来看，XX指标同比增长XX%，环比增长XX%。

三、问题与对策
针对存在的问题，我们提出以下对策...

【结尾】
以上是本次汇报的主要内容，请各位领导批评指正。
```

**适用音色组合**：
- 沉稳男声（主推）
- 磁性男声
- 知性女声
