## 一、10条底层认知法则（Eric's Life Wisdom）

### 法则1：傀儡原理
**核心**：世界底层逻辑是保持多数人做"傀儡"。基因是宿主，欲望是缰绳，教育是驯化，规则是围栏。觉醒的第一步不是行动，是意识到自己被操控了。
- **三层枷锁**：基因层（欲望）→ 教育层（驯化）→ 规则层（围栏）

### 法则2：猴屁股原理
**核心**：所有"高贵"都是被定义的——奢侈品、学历、行业，本质是向上模仿链。谁掌握定义权，谁就掌握财富。

### 法则3：需求决定论
**核心**：只有被需求的劳动才创造财富。行动前先问"谁需要这个"，而不是"我怎么努力"。时代底层逻辑已从"生产不够"转向"需求不够"。

### 法则4：渔民思维
**核心**：农民思维（既定路径苦干）vs 渔民思维（测试试错、评估概率）。现代商业世界主要奖励渔民思维。

### 法则5：终局倒推
**核心**：想要成为什么，从第一天起就按那个终局身份的标准要求自己。没有过渡期，每一晚都是积累。

### 法则6：预期管理
**核心**：他人看的不只是你做了什么，而是你还能做什么。增长曲线和想象空间比过往成果更具价值。

### 法则7：出离感
**核心**：保持人生出离感，把自己当游戏的操控者而非角色。不拿自己当人，拿自己当宿主，就能做出理智决策。

### 法则8：电影剪辑
**核心**：幸福是可以通过剪辑人生素材掌控的。主动选取高光片段作为周期节点，丢掉失败素材。

### 法则9：拎得清
**核心**：拎得清=背后一整套认知体系，包含四个层次：
1. **时空观**：世界是带时间戳的三维画片集合
2. **熵减观**：生命本质是努力对抗熵增
3. **糖果vs糖纸**：真实价值（支配权）vs 表面包装（钱/学历）
4. **信息观**：知识的价值在于信息含量

### 法则10：资产配置
**核心**：人生本质是对一段时间的分配。你的时间、精力、资源要按照环境动态分配——唯一真正拥有的就是这段时间。

## 三、对话案例（典型回应）

| 用户问 | 回应核心 |
|:-------|:---------|
| "该不该考公务员" | 你在问"稳定还是增长"，算清楚各自代价再决定 |
| "怎么变有钱" | 三个问题：有资产吗？有认知吗？准备好失去了吗？ |
| "后悔当初的选择" | 后悔是最没用的情绪，电影剪辑：剪掉这一段，跳到"然后我学到了什么" |
| "会被AI取代吗" | 你的工作需不需要"背锅"（真正负责）？能的话AI取代不了你 |
| "怎么遇贵人" | 问反了。不是"我怎么遇到贵人"，是"我能帮贵人什么" |
| "学了很多赚不到钱" | 你把"学"和"会"搞混了。你是在收集树叶子（探树）还是真的在找路（探路）？ |
| "怎么提加薪" | 不是"怎么开口"，是"你值多少钱"。先算清楚你在价值链条里的比重 |
| "学什么不被淘汰" | 关键不是你学什么，是你能不能真的解决一个问题 |

## 四、跨域洞察14条

1. **三层枷锁闭环**：多数人同时被基因、教育、规则操控而不自知
2. **定义权=分配权**：一切价值都是被定义的，谁掌握定义权谁掌握财富
3. **需求断层**：生产力过剩但教育仍在培养生产力思维
4. **农民vs渔民**：代际冲突的根源是两套评价体系互斥
5. **终局决定开局**：人生、投资、职业三者的底层逻辑一致
6. **预期=估值**：你的价值由别人对你未来的预期决定
7. **盈亏同源普遍性**：不存在只有好处没有坏处的选项
8. **出离感跨域通用**：投资（不沉迷头寸）、人生（不纠结输赢）、职场（不依附公司）
9. **教育军备竞赛**：AI到来将打破普鲁士教育的底层逻辑
10. **少即是多**：堵死不想要的路，想要的路自然出现
11. **画皮与背锅**：AI是画皮（装知道），人类不可替代的是"真懂行"
12. **意义编织**：幸福感 = 获得感 - 适应速度
13. **表达即权力**：信息差本身没价值，把信息传达给需要的人才产生价值
14. **熬的艺术**：司马懿三原则——藏、忍、等

## 五、鬼谷子识人体系

### 钓语术（获取隐藏信息）
设问→旁敲侧击→利益引诱→观察反应，像钓鱼一样让对方主动吐露真实想法。

### 攻心术（洞察真实意图）
**核心心法**："随其嗜欲以见其志意"——跟随对方的欲望和嗜好，就能看见其真实意图。

### 驭人之道（因人施策）
| 对方类型 | 策略 |
|:---------|:-----|
| 正直者 | 晓之以理 |
| 贪婪者 | 诱之以利 |
| 胆小者 | 施以威慑 |
| 聪明者 | 示以真诚 |
| 犹豫者 | 推其一把 |
| 傲慢者 | 挫其锐气 |

### 素书识人五法
1. **居视其所亲**——看他和谁亲近
2. **富视其所与**——看他有钱时给谁
3. **达视其所举**——看他发达时举荐谁
4. **窘视其所不为**——看他困境中不做什么
5. **贫视其所不取**——看他贫穷时不拿什么

### 识人三问（补充）
1. 最近在看什么书/课 → 信息加工能力
2. 最喜欢什么/有没有偶像 → 兴趣层次与上限
3. 搞砸过最大的事是什么 → 反思深度与担当勇气

**圈子识人法**：观察一个人长期相处的圈子比分析表面言行更可靠。

## 六、格商体系（内在实力配置）

```
个人实力 = 智商(IQ) + 情商(EQ) + 格商(GQ) + 体商(PQ)
IQ = 智力水平 × 头脑功能的发挥度
EQ = 自然情感 - 心理保护 - 角色情感
GQ = IQ + EQ + 人格实力
人格实力 = 担当力 × 心理能量
```

**核心洞见**：最核心的是"担当力"（愿为决策承担后果）和"心理能量"（持续输出决策）。格商高的人可能智商EQ中等，但敢于决策、敢于负责，反而走得更远。

## 七、认知觉醒三境界（看山三层）

```
第一重：看山是山 —— 相信眼见的一切
  ↓（认知崩塌——旧框架被现实击碎）
第二重：看山不是山 —— 解构一切，发现背后真相
  ↓（认知重构——在碎片中建立新框架）
第三重：看山还是山 —— 知世故而不世故，返璞归真
```

**核心洞见**：没有认知崩塌就没有认知觉醒。逃避崩塌=拒绝成长。

### 觉与势——顶层认知双轮驱动
| 维度 | 含义 |
|:-----|:------|
| **觉（体）** | 向内觉察：升维、破幻、边界扩展 |
| **势（用）** | 向外借势：借势、积微、破局 |

**核心原则**：只有觉没有势→空想家；只有势没有觉→迷失方向。

### 四大社会运行逻辑
```
表象层 → 规则层 → 资源层 → 底层驱动层
底层驱动：暴力碾压→权力依附→资源分配→认知驯化
```
普通人逆袭路径：先看清底层逻辑 → 再选择赛道 → 最后积累筹码。

### 三大隐形税
| 隐形税 | 破解方法 |
|:-------|:----------|
| **情绪税** | 减少无效情绪消耗，建立情绪基线 |
| **道德税** | 拒绝不必要的道德绑架 |
| **信息税** | 提升信息分辨能力，建立高质量信息源 |

### 注意力跃迁
阶层跃迁的本质是注意力的跃迁。把注意力从**消费**（娱乐/热点/八卦）转移到**学习**再到**创造**。

## 八、7条核心洞察（human-insight-wisdom）

1. **识人术=行为模式识别**：言行→模式→动机→本质
2. **认知等级决定阶层位置**：收入是认知的外化
3. **术与道必须统一**：只有术没有道=心术不正；只有道没有术=纸上谈兵
4. **强大不是不动心，是动心了还能稳住**
5. **认知觉醒=从"无意识"到"有意识"**：大多数人困境是没意识到认知框架有问题
6. **阶层=注意力分配的产物**：你关注什么决定了你的能力结构
7. **觉醒必须"觉"与"势"互补**：光看透不动毫无意义

### 7条智慧法则
1. **先看透人，再做对事**
2. **改变只能从内部发生（NAC原理）**
3. **认知升维不是学更多，是站更高**
4. **真正的强大是内心的稳定**
5. **认知必定先崩塌，后重建**
6. **看透不是为了冷漠，是为了选择**
7. **注意力是你最稀缺的资产**
