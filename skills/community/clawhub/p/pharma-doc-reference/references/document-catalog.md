# 文档清单与依赖拓扑

## 依赖拓扑摘要

以下为文档间的主要依赖链路，UTOS执行轴可据此自动编排管线：

**情报-策略链**: D1-01 → D2-01 → D2-02 → D2-07 / D1-02 → D1-05 → D1-08
**策略-品牌-推广链**: D4-01 → D2-05 → D5-01 → D7-02 → D7-04
**证据-准入链**: D4-01 → D4-02 → D4-06 → D4-07 → D4-08 → D3-01 → D3-02 → D3-03
**证据-注册链**: D4-03 → D4-05 → D8-03 → D8-04 → D8-06 → D8-07
**KOL-活动链**: D6-01 → D6-03 → D7-03 / D6-01 → D7-02 → D7-04
**活动-闭环链**: D7-02 → D7-04/D7-06 → D7-07 → D7-09 → D7-10 → D7-11
**PV-合规链**: D10-04 → D10-07 → D10-05 → D10-06 → D10-09 → D10-10
**生产-合规链**: D9-01 → D9-02 → D9-03 → D9-04 → D9-05
**知识-能力链**: D11-01 → D11-02 → D11-06 / D11-04 → D11-05
**危机-迎检链**: D11-07 → D10-10 / D10-08 → D10-10
**费用-合规闭环**: D7-07 → D2-11 → D10-11
**注册-准入链**: D8-01 → D8-02 → D8-03/D8-04 → D8-06 → D8-07 → D3-03

更多组合由UTOS根据具体任务动态推导。
