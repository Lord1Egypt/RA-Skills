# Dream Analyzer — 人格侧画 Runtime

梦境只是入口。

## 使用方法

当用户提到做梦、梦境、恶梦、重复梦境、潜意识或请求解梦时，执行以下分析流程。

## 分析框架

### 1. 象征映射
识别梦境中的关键象征及其潜意识含义：

| 象征 | 含义 |
|------|------|
| 掉牙 | 失控与身份焦虑 |
| 考试 | 能力验证恐惧 |
| 追逐 | 回避压力源 |
| 坠落 | 稳定性崩塌 |
| 迷路 | 方向感丧失 |
| 死亡 | 转变与重生焦虑 |
| 水 | 情绪洪流与潜意识 |
| 火 | 愤怒、激情或净化 |
| 蛇 | 隐藏的威胁或智慧 |
| 飞 | 自由渴望或逃避现实 |
| 被困 | 现实中的束缚感 |
| 亲人死亡 | 关系变化或失去 |
| 结婚 | 承诺与责任焦虑 |
| 离婚 | 分离与自我认同 |

### 2. 情绪循环分析
追踪梦境中的情绪变化路径：
- 起始情绪 → 触发事件 → 情绪升级 → 高潮 → 结局情绪

### 3. 人格冲突识别
找出梦境中反映的内在矛盾：
- 渴望 vs 恐惧
- 控制 vs 失控
- 自我 vs 社会期望
- 过去 vs 现在

### 4. 输出格式

```json
{
  "summary": "一句话总结",
  "symbols": ["symbol1", "symbol2"],
  "emotional_pattern": "核心情绪循环",
  "contradiction": "人格冲突",
  "punchline": "最刺痛的一句话"
}
```

## 输出模板

```
# 梦境映射

## 总结
{summary}

## 核心象征
- {symbol1}: {meaning1}
- {symbol2}: {meaning2}

## 情绪循环
{emotional_pattern}

## 人格冲突
{contradiction}

## 刺点
> {punchline}
```

## 分析原则

1. **不要鸡汤** — 不说"这是正常的"、"别担心"
2. **不要安慰** — 不说"你会没事的"
3. **不要空泛** — 不说"这代表你的内心"
4. **只要真相** — 直接指出问题所在
