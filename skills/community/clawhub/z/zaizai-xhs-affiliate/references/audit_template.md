# 小红书内容审核模板

> 用于审核AI生成的种草内容

---

## 审核检查清单

### 1. AI味检查
- [ ] 读起来是否自然流畅
- [ ] 是否有"首先、其次、最后"等套路结构
- [ ] 是否有过度夸张的感叹词
- [ ] 语气是否像真人分享

### 2. 违禁词检查
- [ ] 无"最"、"第一"、"绝对"、"顶级"等极限词
- [ ] 无"国家级"、"世界级"等虚假荣誉
- [ ] 无"最好"、"最强"等绝对化表述
- [ ] 无虚假承诺（如"保证瘦20斤"）

### 3. 合规性检查
- [ ] 商品为正规渠道
- [ ] 功效描述有依据
- [ ] 价格信息准确
- [ ] 无侵权内容

### 4. 图片质量
- [ ] 图片清晰
- [ ] 无水印/AI标识
- [ ] 风格统一
- [ ] 尺寸符合要求（3:4）

---
