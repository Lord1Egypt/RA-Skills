# Trade Dashboard - 数据看板技能

> 📈 **全链路数据看板** | 转化追踪 + ROI分析 + 自进化

## 功能介绍

Trade Dashboard 是外贸获客闭环的第四步，提供全链路数据可视化和效果分析。

### 核心能力

- 📊 **转化漏斗** - 发现→筛选→触达→回复→成交
- 📈 **趋势分析** - 月度/季度数据对比
- 💰 **ROI追踪** - 成本vs收益实时监控
- 🤖 **模型路由** - AI任务分配可视化
- 🚀 **自进化记录** - 版本迭代效果追踪
- 🛡️ **安全认证** - 合规状态一览

## 核心指标

| 指标 | 含义 | 优化方向 |
|------|------|----------|
| 发现客户 | 新增潜在客户数 | ↑ 越多越好 |
| 高价值目标 | A级客户数量 | ↑ 转化基础 |
| 邮件回复 | 收到回复数 | ↑ 触达效果 |
| 转化率 | 成交/发现 | ↑ 漏斗效率 |
| ROI | 收益/成本 | ↑ 投入产出 |

## 图表类型

- **漏斗图** - 转化路径可视化
- **折线图** - 趋势变化追踪
- **柱状图** - 成本收益对比
- **饼图** - 模型路由分布

## 技术规范

- **平台**: Coze / Caw
- **可视化**: ECharts 5
- **地图**: Leaflet
