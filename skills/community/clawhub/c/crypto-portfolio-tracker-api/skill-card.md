Risk: Queried cryptocurrency symbols and portfolio holdings may be sent to PRISM API during price lookup and valuation. <br>
Mitigation: Use the skill only with holdings data you are comfortable sharing with PRISM API, and do not provide private keys, seed phrases, wallet credentials, or unrelated financial files. <br>
Risk: The release artifacts use inconsistent package and repository names, which can make package identity harder to verify. <br>
Mitigation: Confirm the npm package, publisher profile, and expected source before installing or deploying the skill. <br>
