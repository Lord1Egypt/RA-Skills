# 继发性腹膜炎的病因与致病菌特征

## 触发条件
当患者出现空腔脏器穿孔、外伤性破裂、腹腔内炎症扩散或术后污染时，考虑继发性腹膜炎。

## 执行步骤

### 1. 常见病因
- 胃十二指肠溃疡急性穿孔 → 内容物流入腹腔 → 化学性刺激 → 化学性腹膜炎 → 进展为化脓性腹膜炎；
- 急性胆囊炎致胆囊壁坏死穿孔 → 胆汁性腹膜炎；
- 外伤致肠管或膀胱破裂 → 腹腔污染 + 经腹壁伤口细菌侵入 → 快速形成腹膜炎；
- 腹腔内脏器炎症扩散（如急性阑尾炎、女性生殖器官化脓性感染）→ 含菌渗出液在腹腔扩散 → 腹膜炎；
- 其他原因：腹部手术污染、胃肠道/胆管/胰腺吻合口渗漏、腹前/后壁严重感染。

### 2. 致病菌特征
- 最常见：大肠埃希菌、肺炎克雷伯菌；
- 其次：厌氧拟杆菌、链球菌等；
- 多为混合性感染，毒性较强，导致病情严重。

### 3. 病理生理过程
胃肠内容物和细菌进入腹腔后，腹膜充血、水肿、失去光泽，产生大量浆液性渗出液以稀释毒素；巨噬细胞、中性粒细胞浸润，与坏死组织、细菌、纤维蛋白共同形成脓液；以大肠埃希菌为主的脓液呈黄绿色，混合感染时脓液稠厚、有粪便样臭味。

## 输出结果
明确继发性腹膜炎的病因类型及致病菌谱，用于指导抗生素选择和手术决策。
