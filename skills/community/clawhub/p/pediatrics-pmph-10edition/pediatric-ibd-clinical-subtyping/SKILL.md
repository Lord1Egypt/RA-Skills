# 儿童炎症性肠病（IBD）临床分型与特征鉴别

## 触发条件
当满足以下任一情况时启用本技能：
- 儿童存在持续性腹泻、腹痛、黏液脓血便等症状；
- 已排除感染性肠炎；
- 出现生长发育迟缓或关节炎、皮疹、虹膜睫状体炎等肠外表现。

> **注意**：不适用于急性胃肠炎或食物中毒患者；确诊需依赖内镜和病理检查。

## 执行步骤

### 1. 确认IBD基本分型
- IBD包括三类：溃疡性结肠炎（UC）、克罗恩病（CD）和未定型结肠炎（IC）。
- 若发病年龄＜6岁，标记为极早发型IBD（VEO-IBD），需考虑单基因免疫缺陷（如IL10RA/IL10RB突变）。

### 2. 评估共同临床特征
- 起病方式：多为亚急性或慢性；
- 核心症状：腹胀、腹痛、腹泻；
- 大便性状：黏液稀便、黏液脓便、脓血便或血水样便；
- 可伴里急后重（UC更显著）；
- 肠外表现：关节炎、强直性脊柱炎、皮疹、虹膜睫状体炎等；
- 长期影响：营养不良、生长发育障碍。

### 3. 鉴别UC与CD的关键特征
对比以下维度进行判断：

| 维度 | 溃疡性结肠炎（UC） | 克罗恩病（CD） |
|------|------------------|---------------|
| **病变范围** | 结肠和直肠，连续性、弥漫性 | 全消化道（常见末端回肠），节段性（跳跃式） |
| **腹痛部位** | 左下腹，持续隐痛/钝痛，排便后缓解 | 右下腹，餐后绞痛或痉挛性锐痛 |
| **大便特点** | 脓血便为主，里急后重明显 | 黏液便或水样便，可便秘与腹泻交替 |
| **并发症** | 出血、中毒性巨结肠、狭窄（癌变罕见于儿童） | 肠梗阻、瘘管、肛周病变（脓肿/瘘管/肛裂）、吸收障碍 |
| **内镜/病理** | 黏膜层弥漫浅溃疡、假息肉、隐窝脓肿 | 纵行深溃疡、非干酪样肉芽肿、肠壁全层炎症 |
| **预后** | 相对较好，部分可治愈 | 难以治愈，仅可控制，治疗难度更大 |

### 4. 特殊人群注意事项
- 儿童CD极少形成腹部包块（区别于成人）；
- VEO-IBD患者对传统治疗反应差，应优先筛查原发性免疫缺陷。

### 5. 输出结果
根据上述分析，输出：
- 最可能的IBD亚型（UC、CD 或 IC）；
- 支持该诊断的关键鉴别特征；
- 是否提示VEO-IBD并建议进一步基因或免疫学检查。

## 使用限制
- 本技能仅用于临床初步分型指导，不能替代内镜、影像或病理确诊；
- 若症状不典型或诊断存疑，应归类为IC并密切随访。
