## Use Case: <br>
External developers and agents use this skill to discover Instagram profiles by location and category, then collect public profile metadata, stats, images, and engagement data for JSON or CSV export. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill targets Instagram scraping and promotes scale, stealth behavior, and residential proxy use, which may create platform, legal, account, and IP risks. <br>
Mitigation: Run it only when authorized and clearly permitted; review the scraper code first; use conservative rate and volume limits; and avoid anti-detection or proxy rotation unless explicitly allowed. <br>
Risk: Scraped profile data and downloaded thumbnails can create privacy and retention obligations. <br>
Mitigation: Collect only necessary public data, store it in controlled output directories, and delete exported data and thumbnails when they are no longer needed. <br>
Risk: Credentials, API keys, and proxy credentials may be exposed if configured carelessly. <br>
Mitigation: Use minimal credentials, keep secrets out of committed files, rotate credentials when appropriate, and restrict access to local configuration and output directories. <br>
