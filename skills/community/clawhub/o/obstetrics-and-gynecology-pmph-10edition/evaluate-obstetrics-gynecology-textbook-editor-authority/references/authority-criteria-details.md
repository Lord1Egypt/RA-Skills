# 国家级学术权威性详细评估标准

## 1. 国家级职务与荣誉
- **孔北华**：中华医学会妇科肿瘤学分会主任委员、国务院政府特殊津贴专家、国家卫健委突出贡献中青年专家、全国人大代表
- **马丁**：中国工程院院士、国家妇产疾病临床研究中心主任、何梁何利奖获得者
- **段涛**：“国家重点研发计划”首席科学家、曾任中华围产学会主任委员、上海市医学领军人才

## 2. 指南与标准制定主导权
- **孔北华**：主持制定国家卫生行业标准《子宫颈癌诊断》，主编《中国妇科肿瘤临床实践指南》第6版及2023版，制定妇科肿瘤专项诊疗指南共10部
- **马丁**：主持制定我国妇科肿瘤临床诊疗标准、指南、共识共8部
- **段涛**：主编国家“十三五”《妇产科学》（五年制）第9版，长期担任国家级产科重点专科负责人

## 3. 科研与学术产出
- **孔北华**：主持国家863计划1项、国家重点研发计划2项、国家自然科学基金7项；SCI论文148篇（第一/通信作者）；获中国高校科技进步一等奖2项（首位）
- **马丁**：以第一完成人获国家科技进步二等奖等多项大奖；在国内外妇产科与肿瘤领域具高影响力

## 4. 教材编写经验
三位主编均曾主编或副主编国家“十三五”“十四五”规划《妇产科学》教材（五年制或八年制），具备连续多轮国家级教材编写经验。
