# 包租代管完整生命週期（8階段）

## Stage 1：前置準備

### 1.1 法規確認
- [ ] 確認土地使用分區是否可出租
- [ ] 確認建物使用執照用途
- [ ] 確認有無違建（既存違建仍可出租但須告知）
- [ ] 確認有無抵押權設定（需銀行同意出租）
- [ ] 確認符合租賃專法規範
- [ ] 確認有無管委會規約限制（寵物、營業等）

### 1.2 稅務規劃
- [ ] 是否申請公益出租人（房屋稅+地價稅比照自住）
- [ ] 月租金2萬以下免稅（依所得稅法）
- [ ] 租賃所得計算方式：43%必要費用扣除率(標準) vs 核實列舉
- [ ] 二代健保補充保費（租金>2萬：2.11%）

### 1.3 保險
- [ ] 住宅火險（銀行貸款要求）
- [ ] 地震險
- [ ] 房東意外責任險（第三人責任）
- [ ] 兇宅險（強烈建議！）

### 1.4 備品準備
- [ ] 合約書（使用內政部版）
- [ ] 點交清單（設備型號/狀態/照片）
- [ ] 鑰匙/遙控器/門禁卡清冊
- [ ] 水電瓦斯帳號結清

---

## Stage 2：招租

### 2.1 租金定價
| 方法 | 說明 |
|------|------|
| 實價登錄參考 | 查591、樂屋網同區域同類型物件 |
| 投報率法 | 年租金÷房價 = 2~4% (參考) |
| 成本法 | 房貸利息+稅費+管理費+折舊+期望報酬 |

### 2.2 曝光渠道
- 591房屋交易（市佔最大）
- 樂屋網
- Facebook社團（在地社團效果佳）
- Line社群
- 房仲（一般收1個月服務費）
- 店頭布告欄/社區布告
- 包租代管業者轉介

### 2.3 看屋安排
- 開放看屋時段（集中看屋效率高）
- 備妥：物件現況說明書、附近生活機能資料
- 安全提醒：避免單獨帶看（或讓同行者知道行蹤）
- 一組一組安排，避免同時多組看屋

### 2.4 待租期間管理
- 水電保持暢通
- 定期巡視（避免水管破裂等狀況）
- 每2週調整租金策略（若超過1個月未租出）

---

## Stage 3：篩選房客

### 3.1 基本門檻
- [ ] 年收入 ≥ 12倍年租金
- [ ] 在職證明／收入證明
- [ ] 身分證正反面（核對）
- [ ] 工作名片或在職證明

### 3.2 加分項目
- [ ] 有穩定工作>1年
- [ ] 家庭承租（比單身穩定）
- [ ] 前房東推薦函
- [ ] 聯徵信用分數>600

### 3.3 紅燈（直接避免）
- ❌ 無法提供身分證明
- ❌ 收入不穩定或無法證明
- ❌ 有囤積傾向（看屋時物品異常多）
- ❌ 以前有惡意欠租紀錄
- ❌ 明顯不愛惜環境

### 3.4 實務技巧
- 見面看屋先聊，觀察談吐與態度
- 詢問前一個租屋處的狀況
- 詢問為何搬家（判斷穩定性）
- 觀察看屋時的態度（有沒有脫鞋、愛惜空間）

---

## Stage 4：簽約

### 4.1 文件準備
- 內政部版住宅租賃契約書（建議使用）
- 附屬設備清單
- 水電瓦斯繳費證明單（前一期）
- 管委會管理辦法（如有）

### 4.2 簽約要點
- [ ] 雙方親簽+蓋章
- [ ] 留存雙方身分證影本
- [ ] 押金收受（最多2個月）
- [ ] 第一個月租金
- [ ] 水電瓦斯費約定方式
- [ ] 管理費由誰負擔
- [ ] 修繕責任劃分
- [ ] 寵物／抽菸／設籍約定

### 4.3 簽約後
- 交付鑰匙+遙控器+門禁卡
- 加入Line群組（聯繫窗口）
- 告知水電瓦斯帳號（或如何繳費）
- 提供緊急聯絡電話

---

## Stage 5：點交

### 5.1 點交流程
1. 逐項核對設備清單
2. 拍照存證（每個房間、每個設備、每個角落）
3. 水電瓦斯度數記錄
4. 確認鑰匙數量
5. 雙方簽名確認

### 5.2 點交重點
| 項目 | 檢查要點 |
|------|----------|
| 牆面地板 | 既有瑕疵標註 |
| 家具家電 | 型號記錄+功能測試 |
| 水龍頭 | 出水正常、無漏水 |
| 馬桶 | 沖水功能正常 |
| 門窗 | 開關順暢、鎖頭正常 |
| 鑰匙 | 數量確認 |
| 遙控器 | 功能測試 |

---

## Stage 6：管理期間

### 6.1 每月例行
- [ ] 收取租金（確認入帳）
- [ ] 催繳（遲繳2天提醒，5天正式催告）
- [ ] 回覆房客問題

### 6.2 每季檢查
- [ ] 簡單巡視公共區域（如有）
- [ ] 確認有無異常（漏水、壁癌、蟲害）
- [ ] 清洗冷氣濾網（每年至少一次）

### 6.3 年度
- [ ] 房屋稅申報
- [ ] 地價稅申報
- [ ] 租賃所得申報（每年5月）
- [ ] 保險續保

### 6.4 修繕處理
See [references/maintenance.md](references/maintenance.md) for full details.

### 6.5 租金調整
- 調整前**3個月**書面通知
- 需有合理理由（物價指數、區域行情）
- 建議每年調幅≤5%

---

## Stage 7：續約／解約

### 續約
- [ ] 續約前**1~2個月**討論
- [ ] 確認租金是否調整
- [ ] 重新簽約（或簽署續約協議書）
- [ ] 檢查有無設備增減更新
- [ ] 續約期間建議1年起跳

### 解約點交
1. 預約點交時間（契約屆滿日或退租日）
2. 檢查房屋狀況（與照片比對）
3. 確認設備數量與功能
4. 結清水電瓦斯費
5. 返還押金（30日內）
   - ✅ 房屋正常 → 全額返還
   - ⚠️ 有損壞 → 扣抵修繕費用（需提供單據）
   - ❌ 欠租 → 扣抵欠租及違約金
