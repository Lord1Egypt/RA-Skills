# 🌍 出国游全球通版 / Global Travel & Work Planner

> 一键查询48国签证、法律、文化、安全、领事馆信息、行程规划、工作留学指南
> One-stop query for 48 countries: visa, laws, culture, safety, consulate, travel, work, study

---

## 🎯 功能介绍 / Features

| 模块 Module | 功能 Function |
|:-----------|:-------------|
| 📋 **签证** / Visa | 旅游/工作/留学签证类型、费用、有效期、办理周期 |
| ⚖️ **法律法规** / Laws | 饮酒年龄、吸烟、毒品惩罚、拍照限制、小费、驾照 |
| 🏠 **风土人情** / Culture | 见面礼仪、用餐礼仪、禁忌、着装、送礼 |
| 🛡️ **安全指南** / Safety | 安全等级、常见骗局、夜间安全、女性安全、应急用语、必备APP |
| 🆘 **领事馆救助** / Consular | 使馆电话地址、护照丢失流程、被拘留流程、紧急医疗 |
| ✈️ **行程规划** / Travel | 预算规划、住宿建议、费用估算、时区货币语言 |
| 🏢 **工作指南** / Work | 工作签证、税务建议、银行开户、保险 |
| 🎓 **留学指南** / Study | 学生签证、申请建议、语言要求、费用参考 |

## 🔧 使用方法 / How to Use

查询完整报告 / Full report:
```python
python3 global_travel_planner.py JP        # 日本
python3 global_travel_planner.py US        # 美国  
python3 global_travel_planner.py TH        # 泰国
```

指定模块 / Select module:
```python
python3 global_travel_planner.py JP safety           # 只看安全
python3 global_travel_planner.py DE laws culture     # 德国法规+文化
python3 global_travel_planner.py TH visa consular    # 泰国签证+领事馆
```
