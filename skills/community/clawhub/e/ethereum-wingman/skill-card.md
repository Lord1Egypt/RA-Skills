## Description: <br>
Ethereum development tutor and builder for Scaffold-ETH 2 projects that guides agents through smart contracts, DeFi integrations, security checks, and fork-mode testing against real protocol state. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and AI coding agents use this skill to build and review Ethereum dApps with Scaffold-ETH 2, including Solidity examples, forked-network testing steps, DeFi patterns, and security gotchas. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Suggested shell commands can create projects, install dependencies, or interact with local Ethereum fork tooling. <br>
Mitigation: Run commands only in a development directory, keep cast and anvil commands pointed at a local fork, and review generated dependencies before use. <br>
Risk: Security examples and gotcha checks are teaching aids and may not catch production vulnerabilities. <br>
Mitigation: Treat the guidance as a starting point and perform independent security review before deploying smart contracts. <br>
