# Platform Specifications Reference Card

Quick-reference for dimensions, character limits, hashtag strategies, algorithm signals, and format best practices.

---

## Instagram

### Dimensions
| Format | Ratio | Resolution |
|---|---|---|
| Feed (Square) | 1:1 | 1080 x 1080 px |
| Feed (Portrait) | 4:5 | 1080 x 1350 px — preferred, more screen space |
| Carousel | 1:1 or 4:5 | Same as above; all slides same ratio; up to 20 slides |
| Reels | 9:16 | 1080 x 1920 px; 0.5-15 min (15-45s optimal) |
| Stories | 9:16 | 1080 x 1920 px; 15s segments, 24h lifespan |
| Profile Photo | 1:1 | 320 x 320 px |

### Limits
| Element | Limit | Tip |
|---|---|---|
| Caption | 2,200 chars | Hook in first 125 chars (before truncation) |
| Hashtags | 30 max | Use 5-15 for best performance |
| Alt Text | 100 chars | Describe image for accessibility |

### Algorithm Signals (by priority)
Saves > Shares/Sends > Comments > Watch time/completion > Likes > Profile visits > Follows

### Frequency
4-7 feed/week, 3-5 Reels/week, 3-7 Stories/day

---

## TikTok

### Dimensions
| Format | Ratio | Resolution |
|---|---|---|
| Video | 9:16 | 1080 x 1920 px |
| Photo Mode | 9:16 | 1080 x 1920 px; up to 35 photos |

### Limits
| Element | Limit | Tip |
|---|---|---|
| Video length | Up to 10 min | 15-45s for max completion rate |
| Caption | 4,000 chars | First 2 lines visible without tap |
| Hashtags | No hard limit | 3-5 targeted is optimal |

### Algorithm Signals (by priority)
Completion rate > Replays > Shares > Comments > Likes > Follows > Trending sound usage

### Trend Lifecycle
- Emerging (Day 1-3): highest reward, hardest to spot
- Peak (Day 3-7): still effective with good execution
- Saturated (Day 7-14): diminishing returns
- Dead (Day 14+): skip it

### Frequency
3-7 videos/week; consistency over volume

## Facebook

### Dimensions
| Format | Ratio | Resolution |
|---|---|---|
| Feed Image | 1.91:1 | 1200 x 630 px |
| Feed Image (Square) | 1:1 | 1080 x 1080 px |
| Video | 16:9 or 1:1 | 1280 x 720 px min; under 3 min optimal |
| Reels | 9:16 | 1080 x 1920 px |
| Stories | 9:16 | 1080 x 1920 px |
| Cover Photo | 2.63:1 | 820 x 312 px desktop; center elements for mobile crop |
| Event Cover | 16:9 | 1920 x 1080 px |

### Limits
| Element | Limit | Tip |
|---|---|---|
| Post text | 63,206 chars | 40-80 chars for max engagement; truncates after ~477 |
| Hashtags | No limit | Use 0-3; minimal impact on Facebook |

### Algorithm Signals
Meaningful interactions > Content type preference > Recency > Video watch time > Click-through

### Frequency
3-5 posts/week

## Pinterest

### Dimensions
| Format | Ratio | Resolution |
|---|---|---|
| Standard Pin | 2:3 | 1000 x 1500 px — ideal format |
| Square Pin | 1:1 | 1000 x 1000 px — acceptable, less real estate |
| Long Pin | 1:2.1 | 1000 x 2100 px — max height; infographics |
| Idea Pin | 9:16 | 1080 x 1920 px; up to 20 pages |
| Video Pin | 1:1, 2:3, 9:16 | 1080 px wide min; 15-60s optimal |

### SEO and Limits
| Element | Limit | Tip |
|---|---|---|
| Pin Title | 100 chars | Front-load primary keywords |
| Pin Description | 500 chars | 2-3 keyword-rich sentences with CTA |
| Board Title | 50 chars | Keyword-focused (not clever names) |
| Board Description | 500 chars | Secondary keywords |
| Alt Text | 500 chars | Descriptive with keywords |

### SEO Strategy
- Use Pinterest search autocomplete for keyword research
- Check Pinterest Trends for seasonal volume
- Add keyword-rich text overlay to Pin images
- Enable Rich Pins for automatic product data sync
- Organize Pins into keyword-titled boards with sections

### Algorithm Signals
Pin quality > Keyword relevance > Domain quality > Saves > Clicks > Freshness > Consistency

### Frequency
5-15 Pins/day (steady, not batched); 1-3 Idea Pins/week

## X (Twitter)

### Dimensions
| Format | Ratio | Resolution |
|---|---|---|
| Single Image | 16:9 | 1200 x 675 px |
| Single Image (Square) | 1:1 | 1200 x 1200 px |
| Two Images | 7:8 each | 700 x 800 px each |
| Four Images | 16:9 each | 1200 x 675 px; 2x2 grid |
| Video | 16:9 or 1:1 | 1280 x 720 min; up to 2m 20s |
| Header | 3:1 | 1500 x 500 px |

### Limits
| Element | Limit | Tip |
|---|---|---|
| Post | 280 chars | Most effective at 71-100 chars |
| Thread | 280/post | 3-8 posts per thread optimal |
| Bio | 160 chars | Value prop, category, link |
| Hashtags | No limit | Use 1-2 max; more reduces engagement |

### Algorithm Signals
Replies > Retweets > Likes > Link clicks > Bookmarks > Time on post

### Frequency
1-3 posts/day; 5-10 replies/day; 1-2 threads/week
