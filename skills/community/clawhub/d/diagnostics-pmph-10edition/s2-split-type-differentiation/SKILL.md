# S₂心音分裂类型鉴别

## 触发条件
在肺动脉瓣区听诊到可分辨的S₂分裂（A₂与P₂成分间距 > 0.03秒）。

## 执行步骤

1. **确认S₂分裂存在**：确保两个成分时间间距大于0.03秒。
2. **观察呼吸对分裂的影响**：
   - **吸气末分裂明显，呼气时消失** → 判定为**生理性分裂**（常见于青少年）；
   - **吸气时分裂增宽，但始终存在** → 判定为**通常分裂**（见于二尖瓣狭窄伴肺动脉高压、肺动脉瓣狭窄、二尖瓣关闭不全、室间隔缺损等）；
   - **吸气与呼气时分裂间距固定不变** → 判定为**固定分裂**（典型见于房间隔缺损）；
   - **吸气时分裂变窄，呼气时变宽** → 判定为**反常分裂（逆分裂）**（提示主动脉瓣关闭晚于肺动脉瓣，见于完全性左束支传导阻滞、主动脉瓣狭窄或重度高血压）。
3. **识别高风险体征**：反常分裂为病理性体征，需高度重视并进一步评估。
4. **辅助验证**：结合心电图（如QRS波形态）和超声心动图结果验证潜在病因。

## 输出内容
输出S₂分裂类型（生理性、通常、固定、反常）及其对应的临床疾病可能性。
