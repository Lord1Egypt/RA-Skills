# UI Kit 合集

UI Kit 合集：插画库、UI 组件库（Tailwind/无样式）、背景图案与多框架模板套件。更多顶选见 [SKILL.md](../SKILL.md)。

## 插画

- [Humaaans](https://www.humaaans.com/) — Mix-and-match illustrations of people
- [Blush](https://blush.design/) — Customizable illustrations with Figma plugin
- [Drawkit](https://www.drawkit.io/) — Illustrations for designers and startups
- [Open Doodles](https://www.opendoodles.com/) — Free set of sketchy illustrations
- [IRA Design](https://iradesign.io/) — Open-source gradient illustrations

## UI 组件库

- [HyperUI](https://www.hyperui.dev/) — Free Tailwind CSS components for marketing/ecommerce
- [Flowbite](https://flowbite.com) — Open-source Tailwind CSS component library
- [daisyUI](https://daisyui.com/) — Tailwind CSS components
- [Tailwind Components](https://tailwindcomponents.com/) — Community Tailwind components
- [Creative Tim](https://www.creative-tim.com/) — UI libraries and kits (React, Vue, etc.)

## UI Kit 与素材聚合

- [UI Bundle](https://uibundle.com) — Free UI kits, mockups, illustrations, icons, fonts, 3D
- [UI Design Daily](https://uidesigndaily.com/) — UI components of all types
- [Sketch App Sources](https://www.sketchappsources.com/) — Sketch UIs, wireframes and icons

## 背景图案

- [Hero Patterns](http://www.heropatterns.com/) — Repeatable SVG background patterns
- [BEAUBUS Patterns](https://patterns.beaubus.com/) — 150+ free SVG pattern backgrounds
