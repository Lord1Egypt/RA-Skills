## 🐛 常见问题

### 问题 1：数据插入失败

**错误信息**：`column "skills_required" is of type text[] but expression is of type text`

**解决方案**：
```sql
-- 确保 skills_required 字段使用数组格式
skills_required = ARRAY['摄影', '视频制作']
-- 或
skills_required = '{摄影，视频制作}'
```

### 问题 2：前端不显示新数据

**原因**：浏览器缓存

**解决方案**：
1. 硬刷新页面（Cmd+Shift+R / Ctrl+Shift+F5）
2. 清除浏览器缓存
3. 在 1D 后台重新部署应用

### 问题 3：技能筛选不工作

**原因**：前端技能选项未更新

**解决方案**：
```
请更新 DiscoverPage.tsx 中的技能选项数组，确保包含所有 20+ 个技能。
```

### 问题 4：UUID 生成失败

**错误信息**：`function gen_random_uuid() does not exist`

**解决方案**：
```sql
-- 启用 pgcrypto 扩展
CREATE EXTENSION IF NOT EXISTS pgcrypto;
-- 或使用 uuid_generate_v4()
```
