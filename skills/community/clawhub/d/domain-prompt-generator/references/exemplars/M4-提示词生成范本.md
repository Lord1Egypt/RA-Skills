# M4 提示词生成域范本

## M4-01 提示词构建

### 输入示例
模板填充结果

### 处理过程
1. 基于模板构建提示词
2. 添加必要的说明
3. 验证提示词的完整性

### 输出示例
**构建的提示词**：
```
执行协议等待指令： 团队管理 分拆成 协调 激励 发展 要素，协调 激励 发展 要素生成完整 团队管理方案，默认只展示要素结构但用户可以要求展示完整 团队管理方案。

要素说明：
- 协调：指管理者通过沟通、协商、调整等手段，使团队成员之间的关系和谐、工作有序的过程
- 激励：指管理者通过目标设定、奖励机制、情感关怀等方式，激发团队成员内在动力的过程
- 发展：指管理者通过培训、指导、挑战性任务等方式，促进团队成员能力提升的过程

使用说明：
1. 默认只展示协调、激励、发展三个要素的结构
2. 用户可以要求展示完整的团队管理方案
```
