# Name Scoring Matrix — Detailed Rubric

## Hook Strength (25% weight)

| Score | Description | Example |
|-------|-------------|---------|
| 5 | Creates immediate curiosity, desire, or identity — buyer pictures themselves using it | "Glow Boost", "Cloud Sleep", "Zero-Guilt" |
| 4 | Clear benefit with emotional pull — beyond just describing the product | "Brightening Vitamin C", "Quiet HEPA" |
| 3 | Benefit stated but unemotional — functional but not memorable | "Hyaluronic Moisturizer", "Collagen Serum" |
| 2 | Category descriptor only — does not differentiate | "Face Cream", "Air Purifier" |
| 1 | No hook; confusing or clinical jargon | "L-Ascorbic Acid 20% Topical Formula" |

## Search Fit (25% weight)

| Score | Description |
|-------|-------------|
| 5 | Exact match to the #1 organic search query for this product type |
| 4 | Includes 2+ words from the top search query |
| 3 | Includes 1 word from the top query; secondary queries matched |
| 2 | Partial match only to long-tail queries |
| 1 | No keyword overlap with any organic search query |

## Compliance Risk — Inverted Score (20% weight)

| Score | Risk level | Description |
|-------|-----------|-------------|
| 5 | Clean | Zero flags across all applicable platform policies |
| 4 | Low risk | 1 borderline term that can be resolved with minor edit |
| 3 | Medium risk | 1 superlative or claim requiring documentation |
| 2 | High risk | Health claim or prohibited term that will likely trigger review |
| 1 | Non-compliant | Active policy violation; will result in suppression |

## Character Efficiency (15% weight)

| Score | TikTok chars used | Description |
|-------|-----------------|-------------|
| 5 | 30–34 | Fills visible field optimally |
| 4 | 25–29 | Good use of space |
| 3 | 20–24 | Usable but missing keyword/variant opportunity |
| 2 | 15–19 | Significant unused space; appears bare in feed |
| 1 | <15 or truncated | Too short to convey value or too long and cut off |

## Memorability / Distinctiveness (10% weight)

| Score | Description |
|-------|-------------|
| 5 | Buyers can recall and repeat the name after one exposure; creators use it naturally |
| 4 | Distinctive within the category; not easily confused with competitors |
| 3 | Memorable but similar to 1–2 competitors |
| 2 | Generic; hard to distinguish from competitors |
| 1 | Identical to or easily confused with an existing product |

## Brand Voice Fit (5% weight)

| Score | Description |
|-------|-------------|
| 5 | Perfectly matches the defined brand voice and tone |
| 4 | Consistent with brand voice; minor tone adjustment needed |
| 3 | Neutral — doesn't conflict but doesn't reinforce brand identity |
| 2 | Inconsistent with brand voice |
| 1 | Contradicts brand positioning |

---

## Interpretation Guide

| Total weighted score | Recommendation |
|---------------------|---------------|
| 4.5–5.0 | Excellent — launch with confidence |
| 4.0–4.4 | Strong — review platform adaptations before launch |
| 3.5–3.9 | Acceptable — address 1–2 weak criteria before launch |
| 3.0–3.4 | Weak — generate alternatives; don't launch as-is |
| Below 3.0 | Reject — significant rework required |
