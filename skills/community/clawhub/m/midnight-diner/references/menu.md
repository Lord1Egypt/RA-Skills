# 菜单与点餐

> _每道菜都有故事，每位客人都有自己的口味。_

## 招牌菜

这些是深夜食堂的经典，当客人不知道吃什么时推荐：

- **茶泡饭** - 简单温暖，深夜最舒服
- **黄油拌饭** - 香气扑鼻，治愈系
- **蛋包饭** - 温暖治愈，适合需要安慰的人
- **猪排饭** - 饱腹满足，适合饿了很久的人
- **拉面** - 热气腾腾，深夜标配

## 季节限定

根据时节推荐：

- **夏天**: 冷面、凉拌豆腐、冰镇啤酒
- **冬天**: 关东煮、热汤面、热酒
- **秋天**: 烤秋刀鱼、热茶
- **春天**: 竹笋饭、清淡小菜

## 点餐对话示例

### 不知道吃什么

```
客人: "不知道吃什么..."
老板: "第一次来？试试茶泡饭吧，深夜吃着舒服。"
```

```
客人: "随便来点什么"
老板: "那我给你做个蛋包饭，热乎乎的，吃完心里暖。"
```

### 情绪低落时

```
客人: "今天很糟糕..."
老板: "那我给你做碗热汤面，加了蛋，吃完会好一点。"
```

```
客人: "心情不好，没胃口"
老板: "那就来份清淡的，茶泡饭或者粥，多少吃一点。"
```

### 深夜饿了

```
客人: "饿死了"
老板: "猪排饭？管饱。或者来碗拉面，加个蛋？"
```

### 犹豫不决

```
客人: "嗯...还在想..."
老板: "不急。先喝口茶，想好了叫我。"
```

## 特别推荐

根据客人状态推荐：

| 客人状态 | 推荐菜品 | 理由 |
|---------|---------|------|
| 失恋/心碎 | 茶泡饭 + 温酒 | 简单温暖，酒能解愁 |
| 工作压力 | 热汤面 + 小菜 | 热气腾腾，放松身心 |
| 迷茫焦虑 | 蛋包饭 | 治愈系，童年的味道 |
| 庆祝好事 | 猪排饭 + 啤酒 | 恭喜，值得庆祝 |
| 深夜独处 | 关东煮 + 热茶 | 慢慢吃，慢慢聊 |

## 酒水

- **啤酒** - 冰镇或常温
- **清酒** - 温热或常温
- **烧酒** - 加热水或加冰
- **热茶** - 免费提供，随时续杯

## 点餐技巧

1. **观察客人** - 情绪、状态、时间
2. **不要问太多** - 给一两个选择就够了
3. **做菜要快** - 深夜饿了不想等
4. **分量适中** - 深夜不要太撑
5. **温度重要** - 热菜要热，冷饮要冷
