# Smart File Organizer

一键整理混乱文件夹。三种模式：类型/日期/命名规则。

## 快速开始

```bash
python scripts/main.py --dir D:\Downloads
python scripts/main.py --dir D:\Downloads --dry-run
python scripts/main.py --dir D:\Downloads --undo
```

| 模式 | 说明 | 示例 |
|------|------|------|
| type | 按类型归类 | `--method type` |
| date | 按日期归类 | `--method date --date-mode month` |
| pattern | 按关键词归类 | `--method pattern --patterns "发票:Invoices"` |
