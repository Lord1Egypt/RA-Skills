## Description: <br>
Memory-driven human connections — match people through deep emotional understanding, not surface interests. Powered by Echo's memory graph and emotion-aware embedding system. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and product teams use Connect to identify opt-in people matches from memory graphs based on emotional resonance, cross-user memory bridges, and latent interests. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill concept uses personal memories and emotional patterns for people matching, which can be sensitive if users do not understand or control what is matchable. <br>
Mitigation: Use only opt-in memory matching, require explicit approval before sharing matches or memory-derived context, and provide clear controls to exclude, revoke, or delete matchable memories. <br>
