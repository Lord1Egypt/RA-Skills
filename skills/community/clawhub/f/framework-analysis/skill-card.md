## Description: <br>
Framework Analysis guides agents through Qian Xuesen's structural framework method for analyzing geopolitical conflicts, international crises, and historical events through constraints, verified facts, causal deduction, and falsifiable predictions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, analysts, and agents use this skill to structure complex-event analysis into hard constraints, a sourced fact matrix, stepwise causal reasoning, a single terminal conclusion, and short-horizon falsifiable forecasts. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can automatically create local report files and send analysis through Feishu without a clear per-use confirmation step. <br>
Mitigation: Require explicit user approval before PDF generation or message sending, verify the exact destination, and avoid confidential prompts unless export paths and recipients are controlled. <br>
Risk: The workflow references an unreviewed helper script from another skill path for PDF generation. <br>
Mitigation: Review the helper script and generated files before execution, or replace the conversion step with an approved local renderer. <br>
