# 退换货申请应答规范

## 适用范围

用户主动申请退货、换货、退款，或对退换货规则提出疑问。

## 信息核实清单

生成应答前确认以下信息（缺失则先向用户索取）：

- 订单编号
- 商品名称及SKU
- 申请原因（质量问题 / 七天无理由 / 商品不符 / 其他）
- 商品状态（未拆封 / 已拆封 / 已使用）
- 是否在退货/换货时效内

## 决策树

```
用户申请退换货
├── 在时效内？
│   ├── 是 → 质量问题？
│   │   ├── 是 → 商家承担运费，支持退货/换货
│   │   └── 否 → 七天无理由？
│   │       ├── 是 → 买家承担运费，支持退货（不支持换货）
│   │       └── 否 → 商品不符？
│   │           ├── 是 → 商家承担运费，支持退货/换货
│   │           └── 否 → 按具体协商处理
│   └── 否 → 超出时效
│       ├── 质量问题且可举证 → 特殊通道申请
│       └── 其他 → 礼貌告知时效已过，提供替代方案
```

## 话术模板

### 模板 A：符合退货条件（质量问题）

> 您好，非常抱歉给您带来了不好的体验！我们理解您的心情，商品出现问题确实让人困扰。
>
> 经核实您的订单{订单号}，{商品名称}符合退货条件。我们将在{时效}内为您办理退货，运费由我方承担。
>
> 退货流程：{流程简述}
>
> 如有任何疑问，随时联系我们。感谢您的理解与支持！

### 模板 B：符合退货条件（七天无理由）

> 您好，感谢您联系售后！{商品名称}支持七天无理由退货。
>
> 需要提醒您：退货运费需由您承担，请确保商品未使用、包装完好。我们将在收到退货后{时效}内完成退款。
>
> 退货流程：{流程简述}
>
> 如有其他问题，随时咨询！

### 模板 C：超出时效

> 您好，感谢您联系我们。很抱歉，您的订单已超出{退/换}货时效（{具体时效}）。
>
> 我们非常重视您的体验，{提供替代方案，如：可为您申请售后检测/专属优惠等}。
>
> 如您有其他问题，我们随时为您服务。

### 模板 D：换货处理

> 您好，非常抱歉给您带来不便！确认您的订单{订单号}符合换货条件。
>
> 我们将为您更换同款{商品名称}，预计{时效}内发出新商品。旧商品请按以下流程寄回：{流程简述}，运费由我方承担。
>
> 收到新商品后如有问题，请随时联系！

## 注意事项

- 退款方式默认原路返回，如用户要求其他方式需特别说明到账时效
- 换货仅支持同款同规格，换不同颜色/尺码需确认库存
- 大件商品退换需明确物流上门取件安排
