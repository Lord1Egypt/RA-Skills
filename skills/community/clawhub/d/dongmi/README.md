## 简介

**上市公司董事会秘书（董秘）四大职能领域全流程专家技能**，适用于 OpenClaw、WorkBuddy、QCLAW 等龙虾（ClawHub/SkillHub）工具体系。

本技能基于中国A股上市公司董秘实战经验构建，覆盖沪深北交所全部监管规则（**含2026年最新《上市公司董事会秘书监管规则》**），提供可直接落地的工作模板、合规清单与操作流程。

---


## 四大核心职能领域

| 领域 | 核心内容 |
|------|---------|
| 🛡 合规与信息披露 | 定期报告编制、临时公告发布、内幕信息管控、合规制度建设 |
| 📊 投资者关系与市值管理 | IR体系搭建、路演/业绩说明会、市值管理工具箱、危机公关 |
| 🏛 公司治理与三会运作 | 股东大会/董事会/监事会全生命周期管理、制度体系优化 |
| 💼 资本运作全流程 | 股权融资、并购重组、股权激励、股份回购、分拆上市 |

## 安装与使用

### OpenClaw / WorkBuddy / QCLAW

```bash
# 通过 npx skills 安装（SkillHub）
npx skills install board-secretary

# 或手动安装（将 SKILL.md 放入技能目录）
cp SKILL.md ~/.workbuddy/skills/board-secretary/SKILL.md
```

### 直接使用

将 `SKILL.md` 内容粘贴到你的 AI 对话上下文，即可激活董秘专家角色。

## 核心参考法规

### 2026年重大更新⚡
- **《上市公司董事会秘书监管规则》**（证监会公告〔2026〕8号，**2026年5月24日施行**）
  - 首部专门针对董秘的部门规章
  - 明确董秘不得兼任经理、财务负责人
  - 任职资格：5年以上相关工作经验或持证+5年经验
  - 过渡期至2027年12月31日

### 法律与证监会规章
- 《中华人民共和国证券法》（2019年修订）
- 《中华人民共和国公司法》（2024年7月施行）
- 《上市公司信息披露管理办法》（证监会）
- 《上市公司独立董事管理办法》（2023年）
- 《上市公司监管指引第10号——市值管理》

### 交易所规则（2026年4月修订）
- 上交所：《股票上市规则（2026年修订）》（上证发〔2026〕42号）
- 上交所：《上市公司自律监管指引第1号——规范运作（2026年修订）》（上证发〔2026〕44号）
- 深交所：《股票上市规则（2026年修订）》（深证上〔2026〕549号）
- 北交所：《北京证券交易所股票上市规则（2026年修订）》

### 香港交易所（2025年更新）
- 《董事會及董事企業管治指引》（2025年5月更新）
- 《企業管治守則》（2025年修订）
