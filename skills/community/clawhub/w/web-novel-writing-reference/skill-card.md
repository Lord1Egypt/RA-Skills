## Description: <br>
Web Novel Writing Reference is a Chinese-language reference skill for web-novel creation that organizes 8 domains and 62 task types across materials, worldbuilding, outlines, characters, plot, scenes, pacing, revision, and commercial operations. <br>

This skill is ready for commercial/non-commercial use. <br>

### License/Terms of Use: <br>

## Use Case: <br>
External writers and agent users use this skill as a structured reference library for web-novel planning, drafting, revision, and platform-oriented publishing workflows. It can guide agents in producing Markdown outlines, task checklists, scene plans, character materials, pacing reviews, and commercial-operation guidance when its Universal Task OS dependency is available. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill directs the agent to automatically install and load Universal Task OS when activated. <br>
Mitigation: Review the dependency source and version before enabling the skill; prefer read-only reference mode unless the user explicitly opts into UTOS-backed writing workflows. <br>
Risk: Reference examples include placeholders that require user-provided source excerpts before sample-based workflows are complete. <br>
Mitigation: Use only rights-cleared, user-provided, and appropriately redacted examples when filling exemplar slots. <br>
