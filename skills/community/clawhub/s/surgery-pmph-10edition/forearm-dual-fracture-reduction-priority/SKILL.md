# 前臂双骨折手法复位优先顺序原则

## 适用条件
- 患者诊断为尺桡骨双骨折
- 骨折为闭合性，无开放伤口或神经血管损伤
- X线已明确骨折类型（横形、斜形或螺旋形）和位置（上1/3、中段或下1/3）
- 骨间膜未完全断裂

## 执行步骤

1. **判断骨折稳定性**：
   - 横形骨折 → 稳定型
   - 斜形或螺旋形骨折 → 不稳定型

2. **根据稳定性选择复位顺序**：
   - 若一骨稳定、另一骨不稳定：**先复位稳定骨折**，利用骨间膜张力带动不稳定骨折复位。

3. **若双骨折均为不稳定型，按部位决定顺序**：
   - **上1/3骨折**：先复位尺骨（位置表浅、肌肉附着少、移位轻）
   - **下1/3骨折**：先复位桡骨
   - **中段骨折**：通常先复位尺骨

4. **处理旋转移位**：
   - 若X线显示斜形骨折斜面呈“背向靠拢”，提示存在旋转移位，**须先反向旋转纠正旋转畸形**，再行复位。

5. **维持正确体位与牵引**：
   - 肩外展90°、屈肘90°
   - 根据骨折平面调整前臂旋转位（如桡骨上段骨折在旋后位牵引）

> **原理**：骨间膜纤维由尺侧下方斜向桡侧上方走行，先复位一根骨可借助其张力传递机制辅助另一骨复位，提高成功率并降低交叉愈合风险。
