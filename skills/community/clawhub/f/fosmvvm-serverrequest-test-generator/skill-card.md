## Description: <br>
Generate ServerRequest tests using VaporTesting. Covers typed request/response validation for Show, Create, Update, and Delete operations. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to generate Swift/Vapor ServerRequest integration tests that exercise typed requests, responses, errors, CRUD flows, localization, and route registration through VaporTesting. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated tests may use incorrect target names, controller registrations, localization fixtures, or sample CRUD data for the user's project. <br>
Mitigation: Review generated Swift test files before committing and confirm they run only against the intended test application. <br>
