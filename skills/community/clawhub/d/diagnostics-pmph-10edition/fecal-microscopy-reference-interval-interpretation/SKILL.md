# 粪便显微镜检查参考区间判定

## 适用场景
- 已完成规范采集的粪便标本的显微镜检查
- 使用高倍视野（HPF，400倍放大）进行观察
- 需对检查结果进行标准化判读以支持消化系统疾病诊断

## 执行步骤
1. **逐项评估以下成分的存在状态**：
   - **细胞**：应无红细胞、吞噬细胞和肿瘤细胞；白细胞仅偶见；柱状上皮细胞少见。
   - **食物残渣**：淀粉颗粒和脂肪小滴偶见；肌肉纤维、结缔组织、弹力纤维、植物细胞和植物纤维可见少量。
   - **结晶**：仅允许存在少量无临床意义的结晶（如磷酸盐、草酸钙、碳酸钙）。
   - **细菌**：球菌与杆菌比例约为1:10，细菌总量约占粪便干重的1/3，主要为正常菌群；可有人体酵母菌。
   - **寄生虫**：不得检出任何寄生虫或虫卵。

2. **判定异常**：若任一项目超出上述参考区间，则视为异常，需结合临床信息进一步分析。

3. **术语定义**：
   - “偶见”：在多个高倍视野中偶尔出现1–2个
   - “少见”：频率低于“偶见”，但仍可识别
   - “无”：在规范检查下完全不可见

## 注意事项
- 本技能**不适用于**非显微镜方法（如抗原检测、PCR）
- **不适用于**未按规范采集或处理的粪便标本
- 判读必须基于标准操作流程下的高倍视野观察结果
