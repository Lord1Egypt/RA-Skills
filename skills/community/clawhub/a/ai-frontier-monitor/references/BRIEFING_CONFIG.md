# AI 前沿情报 · 配置文件
# ai-frontier-monitor v3.0

## 推送偏好
- 推送时间：08:00（完整版）/ 12:00（增量版）
- 推送目标：Feishu（配置你的飞书 user open_id）
- 语言：中文为主，英文关键案例保留原文标题

## 数量限制
- 核心情报：≤3 条（≥4分）
- 值得关注：≤5 条（3-4分）
- 快速浏览：≤5 条（2-3分）
- arXiv 论文：≤3 篇
- GitHub Trending：≤3 个
- 36氪热榜：取前 3 条补充到快速浏览

## 信号检测阈值
- 技术趋势：≥2 条相关候选
- 产品发布：≥1 条
- 资本动向：关键词触发（funding/raised/Series/投资/融资）

## 评分维度（权重）
- 企业真实落地：40%
- 数据支撑：20%
- 可学习性：20%
- 前沿性：20%

## 触发词（扩展）
- 主触发：AI 前沿、情报汇总、每日情报
- 快速模式：今天有什么信号、看看有什么新动态
- 技术专精：arXiv 最新、论文追踪
- GitHub 热榜：GitHub Trending、AI 项目热榜

## 质量门控
- 无高质量候选（≥3分 < 2条）时回复 NO_REPLY
- 同一企业/产品去重，保留评分最高者
- 连续 3 天低于 3 条核心情报 → 触发关键词审查

## 输出格式
- 使用 emoji 作为 section header
- 严格遵循 v3.0 格式模板
- 核心情报需包含：对用户的启示（1句话）
- 总字数控制：500-800 字（不含信号模块）
