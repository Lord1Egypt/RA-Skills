# Acupoint Classification and Indications

## When to Use
- 需要为特定疾病或症状选择针灸治疗穴位时
- 分析某穴位是否适用于当前病证时
- 制定针灸处方需遵循经络理论和腧穴主治规律时

## Execution Steps

### 1. 确认腧穴分类
- **十四经穴**：位于十四经脉上，共362穴，主治所属经络及脏腑病症，为临床首选。
- **经外奇穴**：疗效确切但未纳入经脉系统，对某些病有特殊疗效。
- **阿是穴**：无固定位置，以压痛点为穴，适用于疼痛性疾病，不可标准化。

> 注意：阿是穴不可用于标准化选穴流程；奇穴虽有效，但应优先考虑经穴。

### 2. 应用主治作用规律

#### 近治作用
- 所有腧穴均可治疗其所在部位及邻近器官的病症。
- 躯干部位按节段划分主治范围：
  - 胸上背（T1–T7）：主治肺、心等上焦病
  - 上腹下背（T8–L1）：主治肝胆脾胃等中焦病
  - 下腹腰骶（L2–S4）：主治肾、肠、膀胱、前后阴等下焦病

#### 远治作用
- 四肢肘膝以下穴位可治疗本经远端脏腑或器官疾病。
- 按经脉掌握四肢穴主治：如手太阴肺经主肺、喉病；足阳明胃经主胃肠及头面病。

#### 双向调节特性
- 同一穴位可对相反病理状态产生相反调节效果，例如：
  - 天枢：泄泻时止泻，便秘时通便
  - 内关：心动过速时减缓心率，心动过缓时提升心率

### 3. 综合选穴规则
- **本经取穴**：病变在某经，则取该经腧穴
- **表里配穴**：配合表里经穴位治疗两经相关疾病
- **邻近经参考**：邻近经脉主治功能相近，可辅助选穴
- **四肢分经、头躯干分部**：四肢穴按经脉记忆主治，头躯干穴按解剖区域掌握

## Output
生成符合中医经络理论的候选腧穴列表，明确其分类、主治依据及适用性，用于后续针灸处方制定。
