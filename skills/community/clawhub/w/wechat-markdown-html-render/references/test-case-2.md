# OpenClaw（龙虾）日常维护指南｜Mac 用户专属

## 前言

前阵子帮几位朋友装好了 OpenClaw（江湖人称"龙虾"），普遍反馈：装起来不难，用起来也香，但装完之后接下来怎么办？浏览器控制台在哪？模型怎么换？技能怎么装？

今天我把**安装之后日常维护**你最可能遇到的 7 个问题整理成册，拿来即用。建议收藏 🌟

---

## 一、想干新活儿？先去 SKILL 超市逛逛

OpenClaw 本身是个能力平台，但具体到"发一篇公众号文章"这种任务，它靠的是一个个**技能（SKILL）**。

**操作步骤：**
1. 打开技能商店任一入口：
   - https://clawhub.ai/
   - https://skills.sh/
   - https://skillhub.tencent.com/
2. 搜索你需要的任务，比如"写公众号文章"
3. 找到对应 SKILL 名称，告诉你的龙虾："帮我安装 xxx"
4. 龙虾会自动完成配置

💡 **提示：** 部分 SKILL 需要第三方 API 才能运行（比如文生图），请提前注册并充值相关平台，再让龙虾开工。

---

## 二、内置控制台：浏览器里直接操作

OpenClaw 自带 Web 控制台，**无需手机，只要电脑有浏览器就能访问**。

在浏览器地址栏输入：

```
http://127.0.0.1:18789
```

首次访问需要 Token，具体地址格式：

```
http://127.0.0.1:18789/#token=你的token
```

⚠️ **重要：** 如果你修改过 OpenClaw 的网络网关设置，Token 会发生变化，请以上次设置后的新地址为准。安全起见，**其他电脑无法访问这个地址**。

---

## 三、第一时间定制你的"灵魂"

OpenClaw 有几个灵魂文件，决定了它是谁、什么风格、叫什么名字。建议装完第一时间去控制台个性化。

**路径：** 控制台左侧菜单 → 代理（Agents）→ Files

在这里你可以编辑：
- `SOUL.md` —— 定义 AI 的人格和语言风格
- `IDENTITY.md` —— 名字、头像、签名
- `USER.md` —— 你想让 AI 了解你的哪些信息

别让龙虾千篇一律，给它你的味道 🎨

---

## 四、一键切换首选大模型

OpenClaw 支持多个大模型提供商。如果你想换模型（比如从 GPT 切到 Claude），无需重新安装。

**路径：** 控制台左侧 → 代理（Agents）→ Overview

找到 `primary model` 选项，直接修改即可。

💡 **提示：** 如果选项里没有你想要的模型，请参考下一条命令。

---

## 五、添加新的模型提供商

想接新的 AI 服务商？一条命令搞定：

```bash
openclaw configure
```

之后用键盘上下选择：
1. 选 `Models`
2. 选 `local`
3. 选择你的提供商（如 `openai`、`anthropic` 等）
4. 按界面提示完成配置

---

## 六、一键查看运行状态

如果突然发现龙虾不响应了，先别慌，先查状态。

```bash
openclaw status
```

这条命令会显示：
- 飞书连接状态
- WhatsApp 连接状态
- 各模块是否正常运行

遇到问题先 run 一下，**80% 的情况能自己排查出来**。

---

## 七、警惕 Token 消耗：记忆越多，花钱越快

OpenClaw 有长期记忆能力，但大模型的 Token 是按量计费的。随着记忆文件越来越大，**每次对话的消耗也会增加**。

💡 **建议：** 关注你的套餐类型，优先选择**包月型套餐**，比按量付费划算得多。

---

## 总结

| 操作 | 命令/路径 |
|------|----------|
| 安装新技能 | 技能商店搜索 → 告知龙虾安装 |
| 访问控制台 | http://127.0.0.1:18789 |
| 编辑灵魂文件 | 控制台 → 代理 → Files |
| 切换大模型 | 控制台 → 代理 → Overview |
| 添加模型提供商 | `openclaw configure` → Models |
| 查看运行状态 | `openclaw status` |
| 控制 Token 消耗 | 优先选包月套餐 |
