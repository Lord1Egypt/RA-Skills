# 火星文转换器

将正常中文文本转换为火星文（2005-2010年代非主流网络文体）。611个汉字映射，1020条映射关系，所有替代字均经过人工审核确保可辨认。

## 效果示例

| 原文 | 火星文 |
|------|--------|
| 我爱你中国 | 莪嫒伱忠囯 |
| 你今天说话真的很不好听 | 倪衿兲説咶眞旳狠吥恏聽 |
| 今天天气很好 | 衿兲兲滊佷恏 |

## 特性

- **611个汉字映射**，覆盖常用字30+分类
- **质量可控**：所有替代字均满足同音/繁体/经典火星文/形近变体规则，拒绝生僻无关字
- **多对多随机替换**：同一个字有多个替代写法，每次结果不同
- **可复现**：支持 `--seed` 参数固定随机结果
- **概率可调**：`--prob 0.5` 保留部分原字，控制"火星程度"

## 使用方法

```bash
python3 scripts/mars_text.py "要转换的文本"
python3 scripts/mars_text.py "今天天气很好" --prob 0.7
python3 scripts/mars_text.py "测试" --seed 42
python3 scripts/mars_text.py --map 我
```

## 映射质量规则

替代字必须满足以下至少一条：
- ✅ 同音字（拼音相同，忽略声调）
- ✅ 繁体/异体字
- ✅ 经典火星文（文化公认，如 莪=我）
- ✅ 形近变体（视觉结构与原字相关）
- ✅ 符号替代（①②③等）

核心标准：**普通读者3秒内能猜出原字。**
