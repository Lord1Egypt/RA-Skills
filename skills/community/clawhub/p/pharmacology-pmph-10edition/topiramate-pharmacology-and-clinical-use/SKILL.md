# 托吡酯药理机制与临床应用

## 何时使用
- 患者确诊为局限性发作或全面性强直-阵挛发作（大发作）癫痫
- 难治性癫痫需辅助治疗
- 需进行偏头痛的预防性治疗

## 如何执行
1. **确认适用对象与前提条件**：
   - 确保患者符合癫痫类型或偏头痛预防指征
   - 排除禁忌证：孕妇（因动物实验显示致畸性）、对磺酸基单糖衍生物过敏者

2. **理解药理机制**：
   - 抑制电压依赖性Na⁺通道，减少神经元高频放电
   - 增加GABA激活GABA_A受体的频率，增强抑制性突触传递
   - 减少谷氨酸释放并抑制其受体，降低兴奋性传递
   - 弱碳酸酐酶抑制作用，减少癫痫样放电持续时间

3. **制定用药与监测计划**：
   - 口服给药，主要以原形经肾排泄
   - 监测中枢神经系统不良反应（如共济失调、嗜睡、精神错乱、头晕）
   - 育龄女性需评估妊娠风险，必要时采取避孕措施

4. **决策输出**：
   - 根据发作类型决定单药或辅助治疗
   - 同步启动不良反应与特殊人群风险监测

> 注意：本技能不适用于急性偏头痛发作治疗，仅用于预防。
