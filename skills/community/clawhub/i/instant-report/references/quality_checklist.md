# 质量核查清单

## 必检项（每份报告必须通过）

### ⭐ 来源核查
- [ ] **每个数据点都有来源标注**（来源名称 + URL）
- [ ] **第一优先级来源优先**（海关总署/国家统计局/商务部原文 > USDA/FAO > 咨询机构）
- [ ] 无 Wikipedia 引用
- [ ] 所有 URL 可正常访问（附原文链接为加分项）

### ⭐ 数据准确性
- [ ] **每个来源国比例数字 ≥ 2 个独立来源确认**
- [ ] 进口量 + 国产产量 与 总消费量 误差 < 5%（缺口须弥合）
- [ ] 数字计算正确（百分比、总计）
- [ ] **价格单位必须注明（美元/吨 vs 元/公斤，禁止混用）**
- [ ] 同比/环比计算正确
- [ ] 数据在时效范围内（≤12个月，历史分析除外）

### ⭐ 数据版本标注（新增强制项）
- [ ] 封面后有「数据版本说明」标注块
- [ ] 每类数据（进口/生产/价格/政策）注明截止日期
- [ ] 预测内容与历史数据明确区分

### ⭐ 内容完整性
- [ ] 有执行摘要（≤150字）
- [ ] 涵盖市场规模、增长率、竞争格局
- [ ] **有供需缺口量化分析（含具体缺口万吨数）**
- [ ] **有价格预测情景（保守/基准/乐观三档）**
- [ ] 有风险因素分析
- [ ] 信息来源完整列出

### ⭐ 逻辑一致性
- [ ] 全文数据相互印证，无矛盾
- [ ] 章节之间逻辑连贯
- [ ] 结论与证据匹配
- [ ] SWOT 分析有数据支撑

### ⭐ 语言规范
- [ ] 第三人称叙述
- [ ] 无主观情绪用语
- [ ] 专业术语使用准确
- [ ] 无错别字

### ⭐ 格式规范
- [ ] 标题层级清晰（章/节/小节）
- [ ] 图表有编号和标题
- [ ] 表格数据对齐
- [ ] 适当分段，无超长段落

## 评分标准

| 等级 | 要求 |
|------|------|
| ⭐⭐⭐ 高可信 | 全部必检项通过，关键数据 3+ 来源 |
| ⭐⭐ 中可信 | 全部必检项通过，主要数据 2+ 来源 |
| ⭐ 低可信 | 部分必检项未通过或数据来源单一 |

## 常见错误警示

1. **"据报道""数据显示"** — 未注明具体来源，扣分
2. **单一来源** — 关键结论必须有多个来源
3. **过时数据** — 使用超过 12 个月的数据需特别说明
4. **计算错误** — 百分比、总计必须验算
5. **矛盾数据** — 不同章节同一数据必须一致
