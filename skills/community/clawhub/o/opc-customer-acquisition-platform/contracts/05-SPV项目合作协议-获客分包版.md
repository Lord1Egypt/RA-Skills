## 【声明与定义】

本协议各方（以下统称"贡献者"）自愿组成SPV虚拟合作共同体（Special Purpose Vehicle），基于"项目协作、成果共享"的OPC合作原则，以分账比例代替传统股权，共同推进本项目。

### 关键术语定义

| 术语 | 定义 |
|------|------|
| **SPV虚拟共同体** | 不注册实体，以本协议为基础的项目合作组织 |
| **分账比例** | 各方约定的收益分配比例，代表角色分工 |
| **里程碑** | 预设的阶段性目标和验收标准 |
| **交付物** | 各方按约定提交的成果物 |
| **牵头方** | 对外签约的主体，负责项目管理和分账执行 |

## 【第四条】各方权利与义务

### 4.1 共同权利

1. **知情权**：有权了解项目的全部运营数据、财务状况
2. **监督权**：有权查阅分账记录、交付物
3. **表决权**：对重大事项享有投票权（按分账比例折算票数）
4. **退出权**：在符合本协议条件下自由退出（需承担违约责任）

### 4.2 共同义务

1. **投入义务**：按承诺时间投入项目，不可抗力除外
2. **保密义务**：对项目核心信息保密，协议终止后继续有效2年
3. **协作义务**：积极配合其他方工作，按时完成交付物
4. **报告义务**：按约定提交工作进展报告

### 4.3 各方专属义务

## 【第六条】分账管理

### 6.1 分账触发条件

| 条件 | 说明 |
|------|------|
| 验收完成 | 相关方验收签字 |
| 款项到账 | 客户款项已到账 |
| 无争议 | 无未解决的纠纷 |
| 发票齐全 | 成本发票已提交 |

### 6.2 分账流程

```
1. 验收完成 → 发起分账申请
2. 审核确认 → 各方确认分账金额
3. 税费核算 → 计算应缴税费
4. 分账执行 → 牵头方打款
5. 完税凭证 → 各方自行完税
```

### 6.3 分账计算

```
【公式】各方分账金额
各方分账金额 = 到账金额 × 该方分账比例 × 该方信用系数

【信用系数】
A级 = 1.0
B级 = 0.9（分账-10%）
C级 = 0.7（分账-30%）
D级 = 取消资格

【示例】
项目到账100,000元
甲方（A级）分账比例40% → 100,000×40%×1.0 = 40,000元
乙方（B级）分账比例35% → 100,000×35%×0.9 = 31,500元
丙方（A级）分账比例25% → 100,000×25%×1.0 = 25,000元
```

### 6.4 分账优先级

1. 税费预留（按比例计提）
2. 平台服务费（按约定）
3. 实际成本报销
4. 按分账比例分配收益

## 【第八条】违约责任

### 8.1 违约金

| 违约类型 | 违约金标准 |
|----------|------------|
| 未能完成交付物 | 赔偿已造成损失 × [20]% |
| 泄露商业秘密 | 赔偿直接损失 + 预期利益 |
| 擅自退出 | 没收已投入 + 赔偿损失 × [50]% |
| 违反保密义务 | 按保密协议处理 |

### 8.2 损害赔偿

违约方除支付违约金外，还应赔偿守约方因此遭受的全部损失。

### 9.1 退出条件

| 退出类型 | 条件 | 补偿 |
|----------|------|------|
