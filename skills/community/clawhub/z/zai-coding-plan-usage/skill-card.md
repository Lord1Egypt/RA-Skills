## Description: <br>
Query Z.ai Coding Plan usage and quota limits, including token consumption, tool usage, and subscription status. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and users with a configured Z.ai account use this skill to check Coding Plan quota, token consumption, tool usage, and subscription status from their local agent environment. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill uses a locally configured Z.ai or OpenClaw API key to query account usage and quota data. <br>
Mitigation: Install only where that account access is intended, prefer a scoped key where possible, and review the selected local OpenClaw profile when multiple accounts are configured. <br>
