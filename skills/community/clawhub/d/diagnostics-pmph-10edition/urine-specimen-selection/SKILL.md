# 尿液标本类型选择

## 何时使用
- 用户明确需要进行尿液检测，但不确定应采集哪种类型的尿液标本
- 用户询问不同检测项目对应的尿液采集要求
- 用户需向患者说明正确的尿液采集方法

## 如何执行
1. **确定检测目的**：
   - 若需检查有形成分、化学成分或早孕 → 选择「晨尿」（清晨起床后第一次尿液，浓缩、酸化，成分浓度高）
   - 若为门诊或急诊快速筛查 → 选择「随机尿」（随时采集，方便但影响因素多）
   - 若需进行白细胞排泄率等有形成分排泄率检查 → 选择「3小时尿」（上午6–9时采集）
   - 若需24小时化学成分定量 → 选择「24小时尿」（晨8时排空膀胱弃去，此后至次日晨8时全部尿液）
   - 若检查病理性尿蛋白、尿糖、尿胆原 → 选择「餐后尿」（午餐后2小时）
   - 若进行微生物培养 → 选择「清洁中段尿」（清洗外阴后，弃前、后段，采集中间段于无菌容器）

2. **注意事项**：
   - 12小时尿因结果波动大已较少应用
   - 计时尿（如3小时尿、24小时尿）需向患者详细解释采集终点排尿的重要性
   - 不得混淆不同标本的采集时间要求

输出应为具体应采集的尿液标本类型及对应采集方法说明。
