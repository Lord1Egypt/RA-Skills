# 维修工具和软件使用指南

本文档介绍电脑维修所需的硬件工具和软件工具，帮助用户准备维修环境。

## 1. 硬件工具清单

### 1.1 必备工具（基础维修）

| 工具 | 用途 | 价格 | 购买建议 |
|------|------|------|----------|
| **十字螺丝刀** | 拆机必备 | 15-30元 | 买带磁性的（螺丝不掉） |
| **一字螺丝刀** | 撬卡扣/拆外壳 | 10元 | 和十字螺丝刀买套装 |
| **橡皮擦** | 擦内存/显卡金手指 | 2元 | 普通文具橡皮即可 |
| **镊子** | 拔跳线/夹小螺丝 | 10元 | 防静电镊子 |
| **吹风机** | 清灰 | 已有 | 用冷风档 |
| **硅脂** | CPU散热 | 20-50元 | 推荐：信越7921/酷冷B-type |

**推荐套装**：
- 螺丝刀套装（十字+一字+六角）：30元
- 电脑拆机工具包（含镊子+撬棒）：50元

### 1.2 进阶工具（深度维修）

| 工具 | 用途 | 价格 |
|------|------|------|
| **防静电手环** | 防止静电击穿硬件 | 15元 |
| **万用表** | 测电压/查短路 | 50-100元 |
| **焊台** | 换电容/修主板 | 200-500元 |
| **热风枪** | 拆焊芯片 | 300-800元 |
| **放大台灯** | 看清楚小元件 | 50元 |
| **导热垫** | 显存/供电散热 | 10元 |

**建议**：进阶工具只在需要芯片级维修时才需要，普通用户不需要。

### 1.3 工具使用注意事项

⚠️ **安全警告**：
1. 拆机前必须断电（拔掉电源+扣掉主板电池）
2. 戴防静电手环（或摸一下金属水管放电）
3. 螺丝分类存放（不同长度的螺丝不要混）
4. 用力适度（螺丝滑丝比坏硬件更常见）

## 2. 软件工具清单

### 2.1 硬件检测工具

#### CrystalDiskInfo（硬盘健康检测）
- **用途**：查看硬盘健康度、温度、通电时间
- **下载**：https://crystalmark.info/
- **使用**：
  1. 打开软件，自动检测所有硬盘
  2. 看"健康状态"：良好（蓝色）/ 警告（黄色）/ 不良（红色）
  3. 查看"ID CURRENT/WORST/THRESHOLD"：有红色项目说明有问题
- **判断标准**：
  - 05项（重新分配扇区计数）>0 → 硬盘有坏道
  - C4项（重新分配事件计数）>0 → 硬盘不稳定
  - C5项（当前待映射扇区计数）>0 → 硬盘正在产生坏道

#### MemTest86（内存压力测试）
- **用途**：检测内存是否有故障
- **下载**：https://www.memtest86.com/
- **使用**：
  1. 制作启动U盘（用官方工具）
  2. 从U盘启动，自动运行测试
  3. 运行4 Pass（约4小时）无错误 → 内存稳定
- **注意**：有错误说明内存故障，需要更换

#### CPU-Z（查看硬件配置）
- **用途**：查看CPU/主板/内存/显卡详细信息
- **下载**：https://www.cpu-z.com/
- **使用**：
  - CPU标签页：查看型号/核心数/频率
  - 主板标签页：查看型号/芯片组
  - 内存标签页：查看类型/大小/通道
  - SPD标签页：查看内存详情（代次/频率/时序）

#### GPU-Z（显卡信息检测）
- **用途**：查看显卡详细信息
- **下载**：https://www.techpowerup.com/gpuz/
- **使用**：
  - 查看显卡型号/显存/带宽
  - 查看传感器（温度/风扇转速/功耗）
  - 验证显卡是否正品（对比官方参数）

#### HWiNFO64（硬件监控）
- **用途**：实时监控温度/风扇转速/电压
- **下载**：https://www.hwinfo.com/
- **使用**：
  1. 打开软件，选择"Sensors-only"
  2. 查看CPU/GPU温度（满载时是否过热）
  3. 查看风扇转速（是否正常转动）
  4. 查看电压（是否稳定）

### 2.2 系统维护工具

#### DDU（显卡驱动彻底卸载）
- **用途**：彻底卸载显卡驱动（解决驱动冲突）
- **下载**：https://www.wagnardsoft.com/
- **使用**：
  1. 进安全模式（Win+R → msconfig → 引导 → 安全引导）
  2. 运行DDU，选择显卡品牌（NVIDIA/AMD/Intel）
  3. 点击"清除并重启"
  4. 重新安装官方驱动

#### Display Driver Uninstaller
- 同上（DDU的全称）

#### SFC（系统文件检查器）
- **用途**：修复损坏的系统文件
- **使用**：
  1. Win+R → 输入"cmd" → Ctrl+Shift+Enter（管理员权限）
  2. 输入：`sfc /scannow`
  3. 等待扫描完成（约5-10分钟）
  4. 如果找到损坏文件，会自动修复

#### DISM（系统映像修复）
- **用途**：修复系统映像（SFC无法修复时使用）
- **使用**：
  ```
  DISM /Online /Cleanup-Image /RestoreHealth
  ```
  等待完成（需要联网）

#### CHKDSK（磁盘检查）
- **用途**：检查并修复硬盘错误
- **使用**：
  ```
  chkdsk C: /f /r
  ```
  需要重启（会花很长时间，耐心等待）

### 2.3 数据恢复工具

#### DiskGenius（分区管理+数据恢复）
- **用途**：恢复删除文件/修复分区表/克隆硬盘
- **下载**：https://www.diskgenius.cn/
- **使用**：
  - 文件恢复：选中分区 → 工具 → 已删除或格式化后的文件恢复
  - 分区恢复：磁盘 → 搜索已丢失分区（重建分区表）
  - 系统迁移：工具 → 系统迁移（HDD→SSD）

#### Recuva（简单数据恢复）
- **用途**：恢复误删文件（比DiskGenius简单）
- **下载**：https://www.ccleaner.com/recuva
- **使用**：
  1. 打开软件，选择文件类型
  2. 选择扫描位置（原文件所在分区）
  3. 扫描完成后，勾选要恢复的文件
  4. 保存到**其他分区**（不要保存到原分区）

#### TestDisk（免费数据恢复）
- **用途**：修复分区表/恢复删除分区
- **下载**：https://www.cgsecurity.org/
- **注意**：命令行界面，不太友好，但功能强大

### 2.4 系统安装工具

#### Rufus（启动U盘制作）
- **用途**：制作Windows/Ubuntu启动U盘
- **下载**：https://rufus.ie/
- **使用**：
  1. 插入U盘（会格式化，先备份）
  2. 选择ISO镜像（Windows.iso）
  3. 分区方案：MBR（老旧电脑）/ GPT（新电脑）
  4. 点击"开始"，等待完成

#### Ventoy（多系统启动U盘）
- **用途**：一个U盘放多个ISO，启动时选择
- **下载**：https://www.ventoy.net/
- **使用**：
  1. 安装Ventoy到U盘
  2. 把ISO文件复制到U盘
  3. 开机从U盘启动，选择ISO

#### Windows Media Creation Tool（官方启动盘制作）
- **用途**：制作Windows 10/11官方启动U盘
- **下载**：微软官网
- **优点**：官方原版，最干净
- **缺点**：只能做Windows启动盘

### 2.5 性能测试工具

#### Cinebench R23（CPU性能测试）
- **用途**：测试CPU单核/多核性能
- **下载**：https://www.maxon.net/cinebench
- **使用**：点击"Run"等待完成，对比网上分数

#### 3DMark（显卡性能测试）
- **用途**：测试显卡游戏性能
- **下载**：Steam免费版
- **使用**：运行"Time Spy"或"Fire Strike"测试

#### CrystalDiskMark（硬盘速度测试）
- **用途**：测试硬盘读写速度
- **下载**：https://crystalmark.info/
- **使用**：点击"All"，查看顺序读写/随机读写速度

#### Prime95（CPU压力测试）
- **用途**：烤机测试（测试稳定性和散热）
- **下载**：https://www.mersenne.org/download/
- **使用**：选择"Blend"（测试内存和CPU），运行1小时

#### FurMark（显卡压力测试）
- **用途**：显卡烤机（测试显卡稳定性）
- **下载**：https://geeks3d.com/furmark/
- **使用**：点击"GPU Stress Test"，运行30分钟
- **警告**：温度会很高，注意散热

## 3. 工具使用流程图

### 3.1 电脑卡顿诊断流程

```
电脑卡顿
│
├─ 运行CrystalDiskInfo → 硬盘健康度
│  └─ 不良 → 备份数据，换硬盘
│
├─ 运行MemTest86 → 内存稳定性
│  └─ 有错误 → 拔插内存或更换
│
├─ 运行HWiNFO64 → 查看温度
│  └─ CPU > 85°C → 清灰换硅脂
│
└─ 运行CrystalDiskMark → 硬盘速度
   └─ 读写速度 < 100MB/s → 硬盘老化，换SSD
```

### 3.2 蓝屏故障诊断流程

```
出现蓝屏
│
├─ 记录错误代码
│
├─ 运行BlueScreenView → 分析dump文件
│  └─ 找到导致崩溃的驱动文件（.sys）
│
├─ 进安全模式 → 卸载最近安装的驱动/软件
│
├─ 运行DDU → 彻底卸载显卡驱动
│
└─ 重装系统（最后手段）
```

### 3.3 无法开机诊断流程

```
无法开机
│
├─ 清除CMOS
│  1. 拔掉电源
│  2. 取下主板电池，等待5分钟
│  3. 装回电池，尝试开机
│
├─ 拔插内存
│  1. 断电，拔掉内存条
│  2. 用橡皮擦金手指
│  3. 重新插紧（听到"咔哒"声）
│
├─ 最小系统法
│  1. 只连接：CPU + 内存 + 电源 + 核显
│  2. 逐一添加配件（显卡/硬盘/USB设备）
│  3. 找到导致不启动的配件
│
└─ 更换电源测试（可能是电源坏了）
```

## 6. 工具使用注意事项

### 6.1 软件工具

⚠️ **安全警告**：
1. 只从官网下载（第三方下载站可能有病毒）
2. 绿色版/便携版优先（不用安装，不写注册表）
3. 杀毒软件误报（添加信任即可）

### 6.2 硬件工具

⚠️ **静电防护**：
1. 操作前摸一下金属水管（释放静电）
2. 戴防静电手环（或光脚踩地板）
3. 不要在地毯上操作（容易产生静电）

⚠️ **螺丝管理**：
1. 不同长度的螺丝分类放（用磁铁盘）
2. 拍照记录螺丝位置（方便装回）
3. 不要用磁性太强的螺丝刀（可能损坏硬盘）

## 7. 维修环境准备

### 7.1 工作台要求

- ✅ 宽敞的桌子（能铺开所有配件）
- ✅ 良好的照明（最好有台灯）
- ✅ 防静电垫（或铺一张报纸）
- ✅ 小盒子（放螺丝）
- ✅ 手机（拍照记录）

### 7.2 安全措施

- ⚠️ 断电操作（拔掉所有电源线）
- ⚠️ 不要在水边操作（防止进水）
- ⚠️ 不要用金属工具短接电路板
- ⚠️ 拆机前洗手（释放静电）

---

**最后提醒**：
1. 不确定时，不要强行操作（看视频教程再动手）
2. 备件测试法最有效（用好的配件逐一替换测试）
3. 数据无价，操作前务必备份重要数据！
4. 如果搞不定，及时送修（不要硬来，可能把小问题搞成大问题）
