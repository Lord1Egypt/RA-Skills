# 腹腔脓肿分类诊断与引流治疗

## 适用对象
- 急性腹膜炎术后患者
- 腹部手术后持续发热者
- 腹腔感染未完全控制者

## 触发条件
若患者在急性腹膜炎或腹部手术后出现以下任一表现，应考虑腹腔脓肿：
- 持续发热
- 腹痛或局部压痛
- 影像学发现腹腔内占位性病变

## 执行流程

### 1. 确认脓肿类型
根据解剖位置将腹腔脓肿分为：
- **膈下脓肿**：位于膈肌下方与横结肠系膜之间
- **盆腔脓肿**：积聚于盆腔最低处
- **肠间脓肿**：位于肠管之间

### 2. 针对性诊断
- **膈下脓肿**：
  - 临床表现：高热、肋缘下钝痛（深呼吸加重）、呃逆、胸膜反应；右侧可有肝浊音界扩大
  - 首选检查：超声或CT；X线可显示膈下液气平面（10%~25%含气）
  - 确诊方式：影像引导下穿刺抽脓并送培养

- **盆腔脓肿**：
  - 临床表现：里急后重、黏液便、尿频、排尿困难；全身中毒症状较轻
  - 诊断方法：直肠指诊（男性）或阴道检查（已婚女性）触及前壁膨隆、波动感；辅以超声/CT

### 3. 选择引流方式
- **膈下脓肿**：
  - 首选**经皮穿刺置管引流术**（适用于与体壁靠近的局限性单房脓肿）
    - 步骤：影像定位 → 局麻 → 套管针穿刺 → 抽脓送培养 → 置入多孔导管 → 定期冲洗
    - 拔管标准：症状消失、脓腔缩小、引流量 <10 ml/日
    - 成功率约80%
  - 切开引流仅用于穿刺失败或复杂脓肿

- **盆腔脓肿**：
  - 小脓肿：抗生素 + 热敷 + 温盐水灌肠 + 物理透热
  - 大脓肿：
    - 男性：经直肠切开引流
    - 女性：经阴道后穹窿切开引流
    - 术后放置橡皮引流管3~4天

### 4. 禁忌情形
- 不适用于弥漫性腹膜炎尚未局限阶段
- 脓肿未被网膜或肠管包裹者

> 注：详细操作步骤、影像判读标准及并发症处理见参考文档。

## 参考文档

### 腹腔脓肿影像学特征与介入操作细节

**超声/CT表现**
- 膈下脓肿：边界清晰的低密度/无回声区，常伴周围脂肪间隙模糊，部分可见气体影
- 盆腔脓肿：直肠或子宫前方液性暗区，壁厚，可有分隔

**经皮穿刺置管引流术详细步骤**
1. **术前准备**：确认凝血功能正常；禁食4小时；预防性使用广谱抗生素
2. **穿刺路径规划**：选择最短、避开重要脏器和血管的路径；通常采用Seldinger技术或一步套管法
3. **术后管理**：引流液每日记录量、颜色、性状；每48–72小时复查超声评估脓腔变化

**并发症及处理**
- 出血：压迫止血，必要时栓塞
- 肠穿孔：立即外科会诊
- 导管堵塞：生理盐水低压冲洗，避免高压

**拔管后随访**
- 拔管后3天内复查影像确认脓腔闭合
- 若复发，需重新评估是否为多房脓肿或存在瘘管
