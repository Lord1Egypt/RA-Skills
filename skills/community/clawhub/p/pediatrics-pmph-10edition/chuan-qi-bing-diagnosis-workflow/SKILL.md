# 川崎病诊断流程

## 使用前提
- 患儿年龄 < 5岁
- 持续发热 ≥ 5天
- 已排除其他感染或炎症性疾病
- 抗生素治疗无效

## 执行步骤

### 1. 确认发热特征
- 体温通常为39～40°C
- 热型为稽留热或弛张热
- 持续7～14天或更长

### 2. 评估六项主要临床特征（发热本身不计入）
- **a) 双侧球结膜充血**：非渗出性，不累及角膜缘
- **b) 口唇及口腔改变**：口唇干红皲裂、草莓舌、口咽黏膜弥漫充血
- **c) 皮疹**：斑丘疹、猩红热样或多形红斑样；可伴卡疤（卡介苗接种处）红肿
- **d) 四肢末梢改变**：急性期手足硬性水肿+潮红；恢复期（2～3周）甲周膜状脱皮
- **e) 非化脓性颈淋巴结肿大**：单侧，直径≥1.5cm

### 3. 完全性川崎病诊断
- 若满足 **≥4项** 上述主要临床特征 → **确诊为完全性川崎病**

### 4. 不完全性川崎病诊断（<4项主要特征）
- 必须结合以下辅助证据：
  - **实验室指标异常**：CRP↑、ESR↑、白细胞↑、第2周后血小板↑、低白蛋白、低钠、转氨酶↑等
  - **超声心动图异常**：冠状动脉扩张或动脉瘤
- 若符合美国心脏协会（AHA）不完全KD评分标准（如≥2项实验室异常 + 冠脉异常）→ **诊断为不完全性川崎病**

### 5. 关键注意事项
- 卡疤红肿是相对特异的早期表现
- 高危因素包括：2岁以下男孩、ESR显著增快、CRP和血小板明显升高
- 需与猩红热、幼年特发性关节炎、EBV感染等鉴别

> 提示：若用户提供部分临床特征但未达4项，请主动询问实验室结果和超声心动图情况以评估不完全性诊断可能性。
