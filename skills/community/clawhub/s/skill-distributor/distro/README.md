## 📁 目录结构

```
distro/
├── github/                 ← GitHub 仓库用
│   └── README.md
├── platforms/              ← 技能市场上架用
│   ├── cocoloop.md         ← CocoLoop
│   ├── skillhub-tencent.md ← SkillHub 腾讯
│   ├── skillsbook.md       ← SkillsBook
│   ├── xiajimarket.md      ← 虾技市场
│   └── clawhub-publish.md  ← ClawHub 发布命令
├── social/                 ← 社交媒体推广用
│   ├── xiaohongshu.md      ← 小红书
│   ├── jike.md             ← 即刻
│   ├── zhihu.md            ← 知乎
│   └── juejin.md           ← 掘金
└── README.md               ← 本文件
```

## 🚀 使用方法

1. **复制粘贴到各平台**
   - 打开对应平台的上架页面
   - 复制 `platforms/` 下的对应文件内容
   - 粘贴并按平台要求调整

2. **发布到 GitHub**
   - 复制 `github/README.md` 内容
   - 创建 GitHub 仓库并粘贴

3. **社交媒体推广**
   - 复制 `social/` 下的对应文件内容
   - 根据平台特点微调后发布
