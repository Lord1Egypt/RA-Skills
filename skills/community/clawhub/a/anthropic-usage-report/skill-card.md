## Description: <br>
Format Anthropic API usage reports generated by anthropic-report.py, including proxy usage, CLI versus API tracking, and OpenClaw agent cost and usage reports. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kubiczech808](https://clawhub.ai/user/kubiczech808) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to standardize Anthropic usage reports and to patch an existing anthropic-report.py formatter when they want compact token, cost, cache, and CLI/API usage summaries. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The optional patch script can modify the wrong local anthropic-report.py file if the user supplies an unintended path. <br>
Mitigation: Run the patch script with --dry-run first, confirm the target path, keep the generated backup, and test the report script after patching. <br>
Risk: The reference formatter depends on host functions and values such as datetime, CET, get_pricing, and aggregate-populated cli_potential data. <br>
Mitigation: Verify those dependencies exist in the target report script before applying the formatter or adapt the integration code accordingly. <br>
