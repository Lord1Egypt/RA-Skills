# antiepileptic-drug-core-principles

## 适用场景
- 开始抗癫痫治疗
- 调整现有抗癫痫药物方案（包括换药、加药或停药）
- 管理单药无效或混合型癫痫患者的长期用药

## 核心操作流程

### 1. 根据发作类型合理选药
- 参考权威指南（如ILAE分类及推荐）选择与癫痫发作类型匹配的药物。

### 2. 首选单药治疗
- 从小剂量开始，逐步滴定至有效维持剂量。
- 单药可控制约65%患者的发作。
- 对于单纯型癫痫，优先使用单一有效药物。

### 3. 联合用药限制
- 仅在单药治疗无效或患者为混合型癫痫时考虑联合用药。
- 联用药物种类不应超过3种。
- 必须评估药物间相互作用及叠加不良反应风险。

### 4. 更换药物采用逐渐过渡法
- 在继续使用原药的基础上加用新药。
- 待新药起效并稳定后，再逐步减量并停用原药。
- 禁止突然换药，以免诱发癫痫发作或癫痫持续状态。

### 5. 停药需极其谨慎
- 即使癫痫发作完全控制，也应维持治疗至少2～3年。
- 停药过程应缓慢，通常持续数月甚至1～2年。
- 部分高风险患者（如结构性病因、脑电图异常持续存在）可能需终身用药。

### 6. 长期用药监测
- 定期监测血常规、肝功能等指标。
- 育龄期女性及孕妇慎用，因部分抗癫痫药有致畸及死胎风险。

> **注意**：本技能不适用于癫痫持续状态的紧急处理阶段，也不适用于已知对多种抗癫痫药过敏且无替代方案的患者。
