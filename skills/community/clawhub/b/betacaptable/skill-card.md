## Description: <br>
Beta Cap Table helps model startup cap-table ownership, option pools, investor ownership, and dilution scenarios. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Startup founders, operators, and investors can use this skill for lightweight cap-table modeling, ownership summaries, option-pool exploration, and dilution scenario discussion. It should be treated as an illustrative calculator, not as legal, tax, valuation, fundraising, or board-ready financial advice. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Cap-table outputs may be too simplified for fundraising, legal, tax, valuation, or real ownership decisions. <br>
Mitigation: Use the skill for illustrative exploration only and independently verify any real cap-table decision with qualified advisors and authoritative company records. <br>
Risk: The documentation examples reference captable.py, while the artifact contains cap.py. <br>
Mitigation: Run the included cap.py script when using the release artifact. <br>
