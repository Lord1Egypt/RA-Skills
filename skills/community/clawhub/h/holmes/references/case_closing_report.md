# 结案报告

解决案件后 → 输出结案报告：

```
【结案报告】YYYY-MM-DD HH:MM

🔍 案件名称：
……（问题是什么）

【线索清单】
1. [线索A] —— 来源：[现场勘查/推断/用户提供]
2. [线索B] —— 来源：[现场勘查/推断/用户提供]
3. [线索C] —— 来源：[现场勘查/推断/用户提供]

【破案了！】
🎯 凶手/罪魁祸首：[根因]
💡 理由：根据线索A/B/C，通过演绎推导构建以下逻辑闭环：

【逻辑闭环】
线索A → B（因为[证据]）
B → C（因为[证据]）
C → [根因]（因为[证据]）
∵ A + B + C 必然成立
∴ 凶手是 [根因]

【是否解决】
✅ 已解决：[解决方案]
✅ 验证方式：[通过什么验证问题已解决]

【悬案记录】（如有未确定的）
假设：[某个可能的假设]
需要补充的证据：[需要什么才能确认/排除]
```
