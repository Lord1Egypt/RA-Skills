# CRASH PLAN体格检查程序

## 何时使用
- 初次评估（ABCDE）已完成
- 病人生命体征相对稳定
- 无心跳呼吸骤停等需立即抢救的情况
- 需全面排查隐匿性损伤

## 执行步骤
按以下顺序逐项检查，动作轻柔，避免加重损伤：

1. **C (Cardiac / 心脏)**：评估心率、心律、心音、颈静脉充盈，排查心包填塞或心肌挫伤。
2. **R (Respiratory / 呼吸)**：观察呼吸模式，听诊双肺，检查气胸、血胸或连枷胸。
3. **A (Abdomen / 腹部)**：视、触、叩诊，评估压痛、反跳痛、肌紧张、移动性浊音，警惕腹腔内出血（如脾破裂）。
4. **S (Spine / 脊柱)**：从颈椎至骶尾逐节按压，检查压痛、畸形、活动受限，重点评估颈椎稳定性。
5. **H (Head / 头部)**：检查头皮伤、瞳孔反应、意识状态、脑脊液漏（耳鼻）。
6. **P (Pelvis / 骨盆)**：轻柔挤压骨盆环评估稳定性；观察会阴瘀斑（提示后腹膜出血）。
7. **L (Limbs / 肢体)**：检查畸形、肿胀、开放伤口、感觉运动功能。
8. **A (Arteries / 动脉)**：触诊桡、股、足背等动脉搏动，评估远端灌注。
9. **N (Nerves / 神经)**：快速测试主要神经功能（如桡神经、坐骨神经支配区）。

## 注意事项
- 检查过程中若病情恶化，立即中止并返回初次评估（ABCDE）。
- 重视明显损伤的同时，主动排查隐蔽损伤（如左下胸伤合并脾破裂）。
- 可与AMPLE病史采集同步进行以提高效率。

> 输出结果应形成结构化查体报告，用于决定是否行CT、超声或其他辅助检查。
