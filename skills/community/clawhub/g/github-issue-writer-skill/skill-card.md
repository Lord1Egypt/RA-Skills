## Description: <br>
Guides structured help for GitHub Issue Writer using clear templates, checks, and safe defaults. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[abdullah944](https://clawhub.ai/user/abdullah944) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and maintainers use this skill to turn goals and owned notes into concise GitHub issue plans, checklists, next actions, and editable draft copy while avoiding secrets and unsafe commands. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Users could include secrets or sensitive personal data in issue-writing inputs. <br>
Mitigation: Provide goals and notes without passwords, OTPs, private keys, wallets, seeds, session secrets, or unnecessary personal data. <br>
Risk: The disclosed PayPal donation link could be mistaken for a requirement. <br>
Mitigation: Treat donation as optional; the skill does not require payment, GitHub credentials, local file access, or account permissions to function. <br>
