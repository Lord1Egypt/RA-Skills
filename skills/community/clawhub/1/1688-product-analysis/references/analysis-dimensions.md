# 商品分析维度说明

## 概述

本文档定义了商品分析的各个维度、指标含义和分析方法。

## 核心分析维度

### 1. 商品表现维度

| 指标 | 定义 | 单位 | 重要性 |
|------|------|------|--------|
| 浏览量(PV) | 商品详情页被访问的总次数 | 次 | ⭐⭐⭐⭐⭐ |
| 访客数(UV) | 访问商品的独立用户数 | 人 | ⭐⭐⭐⭐⭐ |
| 点击率(CTR) | 曝光到点击的转化率 | % | ⭐⭐⭐⭐ |
| 转化率(CVR) | 访客到成交的转化率 | % | ⭐⭐⭐⭐⭐ |
| 成交金额(GMV) | 商品成交总金额 | 元 | ⭐⭐⭐⭐⭐ |
| 成交笔数 | 商品成交订单数 | 笔 | ⭐⭐⭐⭐ |

### 2. 货盘维度

商品按战略定位划分为不同货盘：

| 货盘类型 | 定义 | 运营策略 |
|----------|------|----------|
| 引流款 | 低价高流量商品，用于吸引客户 | 重曝光、轻利润 |
| 利润款 | 高毛利商品，贡献主要利润 | 重转化、重客单 |
| 爆款 | 销量最高的明星商品 | 全力推广、保持竞争力 |
| 长尾款 | 销量较低但有稳定需求 | 低投入维护 |
| 新品 | 新上架商品 | 重点培育、快速测款 |

### 3. 搜索推荐维度

影响搜索排名和推荐的关键因素：

| 因素 | 影响权重 | 优化方向 |
|------|----------|----------|
| 标题相关性 | 高 | 包含核心关键词、热搜词 |
| 类目准确性 | 高 | 确保类目正确，避免错放 |
| 商品质量分 | 高 | 完善商品信息、提升转化 |
| 店铺权重 | 中 | 提升店铺整体表现 |
| 价格竞争力 | 中 | 合理定价、参与活动 |
| 服务能力 | 中 | 提升发货速度、售后服务 |

### 4. 广告投放维度

| 指标 | 定义 | 健康范围 |
|------|------|----------|
| 广告消耗 | 广告投放花费 | 根据ROI调整 |
| CPC | 单次点击成本 | 行业均值以下 |
| ROI | 投入产出比 | > 1.5 |
| 曝光量 | 广告展示次数 | 稳定增长 |
| 点击量 | 广告点击次数 | CTR > 2% |

### 5. 市场竞争维度

| 分析项 | 数据来源 | 分析目的 |
|--------|----------|----------|
| 热品对比 | 下游热品数据 | 了解市场趋势 |
| 价格对比 | 竞品价格 | 评估价格竞争力 |
| 热搜词 | 类目热搜词 | 优化标题和关键词 |
| 行业趋势 | 行业大盘数据 | 判断淡旺季 |

## 分析优先级

按以下优先级进行商品分析：

1. **商品表现数据** - 了解当前状态
2. **市场竞争分析** - 对比行业水平
3. **搜推问题诊断** - 识别流量障碍
4. **广告效果分析** - 评估投放效率
5. **策略优化建议** - 制定改进方案

## 异常判定标准

### 表现异常

| 异常类型 | 判定条件 | 严重程度 |
|----------|----------|----------|
| 流量暴跌 | 日环比下跌 > 30% | 🔴 严重 |
| 转化异常 | CVR 低于行业均值 50% | 🟠 中度 |
| 排名下滑 | 搜索排名下降 > 10 位 | 🟡 轻度 |

### 搜推异常

| 异常类型 | 影响 | 处理建议 |
|----------|------|----------|
| 类目错放 | 搜索流量受限 | 立即修正类目 |
| 标题违规 | 可能被降权 | 修改违规词汇 |
| 图片问题 | 点击率下降 | 优化主图 |
| 属性缺失 | 搜索匹配度低 | 完善商品属性 |
