## Description: <br>
Review a React/TypeScript component for correctness, hook safety, prop modeling, accessibility, state coverage, and minimal safe changes. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ritual](https://clawhub.ai/user/ritual) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to review React/TypeScript components, identify correctness, hook, typing, accessibility, and state-handling risks, and produce concise findings with minimal proposed changes. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Optional Ritual Cloud setup can connect the workflow to an external service. <br>
Mitigation: Use local mode by default and enable Ritual Cloud only when the user is comfortable installing the CLI and connecting to that service. <br>
Risk: Optional knowledge-note capture can add durable project notes to the repository. <br>
Mitigation: Write knowledge notes only after explicit user approval, keep them small, and cite the files or evidence they came from. <br>
