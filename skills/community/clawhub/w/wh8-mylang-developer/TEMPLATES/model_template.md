# WT8麦语言模型模板

## 模板1：基础趋势策略

```mql
// ============================================
// 基础趋势策略模板
// 适用于：初学者
// 特点：简单、易懂、有止损
// ============================================

// 参数
N1:=10;   // 短周期
N2:=20;   // 长周期
止损:=15; // 止损跳数

// 指标
MA1:MA(C,N1);
MA2:MA(C,N2);

// 开仓条件
金叉:=CROSS(MA1,MA2);    // 短上穿长
死叉:=CROSSDOWN(MA1,MA2); // 短下穿长

// 开仓
金叉, BK;   // 金叉买开
死叉, SK;   // 死叉卖开

// 止损
CLOSE<=BKPRICE-止损*MINPRICE, SP;
CLOSE>=SKPRICE+止损*MINPRICE, BP;

// 过滤
AUTOFILTER;
```

## 模板2：带止盈止损的策略

```mql
// ============================================
// 带止盈止损的策略模板
// 适用于：追求完整风控
// 特点：止损+止盈+保本
// ============================================

// 参数
MA周期:=20;     // 均线周期
止损N:=15;      // 止损跳数
止盈M:=40;      // 止盈跳数
保本N2:=20;     // 保本开启跳数
保本M2:=5;      // 保本回撤跳数

// 指标
MA20:MA(C,MA周期);

// 开仓
CLOSE>MA20, BK;
CLOSE<MA20, SK;

// 1. 固定止损
CLOSE<=BKPRICE-止损N*MINPRICE, SP;
CLOSE>=SKPRICE+止损N*MINPRICE, BP;

// 2. 固定止盈
CLOSE>=BKPRICE+止盈M*MINPRICE, SP;
CLOSE<=SKPRICE-止盈M*MINPRICE, BP;

// 3. 保本止损
BKHIGH>BKPRICE+保本N2*MINPRICE && CLOSE<=BKPRICE+保本M2*MINPRICE, SP;
SKLOW<SKPRICE-保本N2*MINPRICE && CLOSE>=SKPRICE-保本M2*MINPRICE, BP;

AUTOFILTER;
```

## 模板3：分批进出策略

```mql
// ============================================
// 分批进出策略模板
// 适用于：追求精确持仓管理
// 特点：分批建仓+分批平仓
// ============================================

// 参数
开仓手数:=5;
加仓盈利:=0.02;   // 2%
减仓亏损:=0.01;   // 1%
止盈比例:=0.08;   // 8%

// 头寸管理
T_COMMAND(开仓手数);
TRADE_AGAIN(3);   // 允许加减仓3次

// 均线
MA20:MA(C,20);
MA60:MA(C,60);

// ========== 开仓 ==========
CLOSE>MA20 && CLOSE>MA60, BK;

// ========== 加减仓 ==========
CLOSE>BKPRICE*(1+加仓盈利) && BKVOL<15, ADD_LONG(2);
CLOSE<BKPRICE*(1-减仓亏损) && BKVOL>开仓手数, LOWER_LONG(2);

// ========== 分批平仓 ==========
CLOSE>BKPRICE*(1+0.04), LOWER_LONG(开仓手数);
CLOSE>BKPRICE*(1+止盈比例), SP;

// ========== 止损 ==========
CLOSE<BKPRICE*(1-0.03), SP;

AUTOFILTER;
```

## 模板4：头寸管理策略

```mql
// ============================================
// 头寸管理策略模板
// 适用于：资金管理严格的组合
// 特点：按资金比例下单+权益风控
// ============================================

// ========== 头寸管理 ==========
// 按资金比例下单
SETDEALPERCENT(25, 10);  // 25%资金，最大10手

// 权益止损
HM:=HHV(MONEYTOT,BARPOS);
QY:=(HM-MONEYTOT)/HM;
QY>0.15, SP;    // 回撤15%止损
QY>0.15, BP;

// ========== 交易策略 ==========
MA10:MA(C,10);
MA20:MA(C,20);

// 开仓
CROSS(MA10,MA20), BK;
CROSSDOWN(MA10,MA20), SK;

// 止损
CLOSE<=BKPRICE*0.98, SP;
CLOSE>=SKPRICE*1.02, BP;

AUTOFILTER;
```

## 模板5：跨周期策略

```mql
// ============================================
// 跨周期策略模板
// 适用于：多周期共振交易
// 特点：日线方向+分钟信号
// ============================================

// 首先建立日线指标（保存为 DAY_IND）
// DAY_IND
// MA5:MA(C,5);
// MA20:MA(C,20);
// DAYUP:MA5>MA20;
// DAYDN:MA5<MA20;

// ========== 引用日线 ==========
#IMPORT[DAY,1,DAY_IND] AS D

DAYUP:D.DAYUP;
DAYDN:D.DAYDN;

// ========== 分钟周期条件 ==========
MA5:MA(C,5);
MA10:MA(C,10);

MINUP:CROSS(MA5,MA10);
MINDN:CROSSDOWN(MA5,MA10);

// ========== 综合信号 ==========
DAYUP && MINUP, BPK;   // 日线多头+分钟金叉→做多
DAYDN && MINDN, SPK;   // 日线空头+分钟死叉→做空

// 止损
CLOSE<=BKPRICE-20*MINPRICE, SP;
CLOSE>=SKPRICE+20*MINPRICE, BP;

AUTOFILTER;
```

## 模板6：运行优化策略

```mql
// ============================================
// 运行优化策略模板
// 适用于：短线精准入场
// 特点：K线走完前下单
// ============================================

// 运行优化
SIGCHECK('B', 15);  // K线走完前15秒下单

// 下单价格
SETSIGPRICETYPE(BK, 2);   // 开仓用对手价
SETSIGPRICETYPE(SP, 4);   // 止损用市价
SETSIGPRICETYPE(SK, 2);
SETSIGPRICETYPE(BP, 4);

// 参数
N:=20;   // 止损
M:=30;   // 止盈

// 突破策略
HH20:=HHV(HIGH,20);
LL20:=LLV(LOW,20);

// 开仓
CLOSE>HH20, BK;
CLOSE<LL20, SK;

// 止损止盈
CLOSE<=BKPRICE-N*MINPRICE, SP;
CLOSE>=BKPRICE+M*MINPRICE, SP;
CLOSE>=SKPRICE+N*MINPRICE, BP;
CLOSE<=SKPRICE-M*MINPRICE, BP;

AUTOFILTER;
```

## 模板7：完整生产策略

```mql
// ============================================
// 完整生产策略模板
// 适用于：实盘运行
// 特点：完整的风控和头寸管理
// ============================================

// ============================================
// 第一部分：头寸管理
// ============================================

// 1. 按资金比例下单
SETDEALPERCENT(20, 8);

// 2. 权益风控
HM:=HHV(MONEYTOT,BARPOS);
QY:=(HM-MONEYTOT)/HM;
QY>0.2, SP;    // 权益回撤20%止损
QY>0.2, BP;

// 3. 最大持仓控制
// （通过SETDEALPERCENT的第二个参数控制）

// ============================================
// 第二部分：交易条件
// ============================================

// 参数
MA短期:=10;
MA长期:=20;
止损:=15;
止盈:=50;

// 均线
MA10:MA(C,MA短期);
MA20:MA(C,MA长期);

// 趋势判断
趋势多头:=MA10>MA20;
趋势空头:=MA10<MA20;

// 信号
金叉:=CROSS(MA10,MA20);
死叉:=CROSSDOWN(MA10,MA20);

// ============================================
// 第三部分：交易执行
// ============================================

// 开仓
趋势多头 AND 金叉, BPK;
趋势空头 AND 死叉, SPK;

// 止损
CLOSE<=BKPRICE-止损*MINPRICE, SP;
CLOSE>=SKPRICE+止损*MINPRICE, BP;

// 止盈
CLOSE>=BKPRICE+止盈*MINPRICE, SP;
CLOSE<=SKPRICE-止盈*MINPRICE, BP;

// 趋势反转平仓
趋势空头 AND SKVOL>0, SPK;
趋势多头 AND BKVOL>0, BPK;

// 过滤
AUTOFILTER;
```

## 模板使用说明

### 如何使用这些模板：

1. **选择模板**：根据策略需求选择合适的模板
2. **修改参数**：调整参数值
3. **测试回测**：先在小周期、小数据量上测试
4. **优化参数**：使用枚举/遗传优化参数
5. **实盘运行**：验证后再实盘

### 模板优先级：

```
初学者          → 模板1（基础趋势）
追求风控        → 模板2（带止盈止损）
精细持仓        → 模板3（分批进出）
资金管理        → 模板4（头寸管理）
多周期共振      → 模板5（跨周期）
短线交易        → 模板6（运行优化）
实盘生产        → 模板7（完整生产）
```
