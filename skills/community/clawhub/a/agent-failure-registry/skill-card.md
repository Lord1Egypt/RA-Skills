## Description: <br>

## Use Case: <br>
Developers and agent operators use this skill to search a shared registry of agent failure post-mortems, compare symptoms, and apply documented fixes or prevention strategies. It can also guide creation of new structured failure reports for contribution. <br>

### Deployment Geography for Use: <br>
Global <br>
