# kd-enterprise-info · 企业信息调研工具 v5.2

> 金蝶灵境风格 HTML 调研报告，纯AI提取架构，一键生成企业全景画像。

## 这是什么？

输入一个企业名称，AI 自动搜索工商信息、业务概况、行业洞察、财务数据、数字化系统部署等 11 个板块，输出一份**金蝶灵境风格**的结构化 HTML 报告。

**零配置开箱即用**——无需任何 API Key，AI 内置搜索工具负责搜索。配置 Tavily Key 后可自动增强工商/财务/数字化数据精确度。

**v5.2 纯AI提取架构**：完全去除正则提取，所有数据提取由AI直接理解完成，更准确、更适应各种文本表达。

缺失字段自动显示"暂未获取"，严格遵循防幻觉机制。报告基于公开信息整理，建议核实关键业务数据。

## 快速开始

直接在企业名称前加上 `/enterprise_info` 命令，或者用自然语言表达：

```
/enterprise_info 青岛啤酒股份有限公司
```

也可以说：
- "帮我调研一下青岛啤酒"
- "用企业信息搜索工具查一下华为"
- "做一份金蝶国际的企业调查报告"

> 💡 只要包含"企业调研""企业信息搜索""公司信息查询""企业背景调查"等意图，并跟随企业名称，代理就会自动启用本技能。

### 配置 Tavily API Key（可选增强）

不配置也完全可用，AI 搜索覆盖全部 11 个维度。配置后自动增强工商/财务/数字化 3 个核心维度。

```bash
# 方式一：环境变量
export TAVILY_API_KEY=your_tavily_api_key_here

# 方式二：写入 .env 文件（自动识别）
echo "TAVILY_API_KEY=your_tavily_api_key_here" >> ~/.openclaw/workspace/.env
```

**获取 Key**：访问 [tavily.com](https://tavily.com/) 注册，免费额度每月约 1000 次。

### 调用

```
/enterprise_info 深圳市共进电子股份有限公司
```

### 查看报告

生成后自动返回 HTML 报告链接，浏览器打开即可查看。

**识别逻辑**：AI 理解搜索结果 → 识别系统关键词 → 匹配厂商 → 输出厂商名；匹配不到厂商 → 显示"厂商未知"；无任何提及 → 显示"未部署"。

## 搜索架构（v5.2 纯AI提取）

纯AI提取架构，零配置开箱即用：

```
用户输入企业名 → AI 搜索11个维度 → [Tavily增强3个维度] → AI统一理解结果 → 结构化数据 → HTML报告
```

**v5.2 架构特点**：
- ✅ **完全去除正则提取**：所有数据提取由AI直接理解完成
- ✅ **统一处理逻辑**：AI理解所有维度的搜索结果（包括Tavily增强结果）
- ✅ **Tavily增强可选**：有Key时自动增强3个核心维度
- ✅ **零配置开箱即用**：无需任何API Key即可生成高质量报告

无 Tavily Key 时报告顶部显示轻提示，建议配置 Key 以提升数据完整性。

## 常见问题

**Q：报告里很多字段显示"暂未获取"，怎么办？**

A：可配置 Tavily API Key 以增强工商/财务/数字化数据的精确度。同时 AI 搜索质量也影响结果完整性，建议使用企业全称查询。

**Q：能查海外企业吗？**

A：可以，但数据完整性取决于该企业在中文互联网上的公开信息量。建议使用企业英文全称查询。

**Q：财务数据只有上市公司才有吗？**

A：是的，财务板块仅在检测到上市信息（股票代码/上市关键词）时展示。非上市公司该板块自动隐藏。

**Q：搜索耗时多久？**

A：AI 搜索 11 个维度通常 10-20 秒；有 Tavily 增强时额外 3-5 秒。
