# 动脉导管未闭的连续性分流与差异性发绀机制

1. 正常情况下：主动脉收缩压和舒张压均高于肺动脉 → 血液从主动脉经未闭导管持续分流至肺动脉（连续性左向右分流）。
2. 分流后果：
   - 肺循环、左心房、左心室、升主动脉血流量显著增加（达正常2～4倍）
   - 左心负荷加重 → 左心室肥大
   - 肺血增多 → 动力性肺动脉高压
3. 慢性高肺血流 → 导管壁收缩、增厚、硬化 → 梗阻性肺动脉高压 → 右心室收缩期负荷过重 → 右心室肥厚。
4. 当肺动脉压 > 主动脉压时：
   - 左向右分流减少或停止
   - 肺动脉血逆向分流入降主动脉 → 出现“差异性发绀”（differential cyanosis）
   - 表现为：下半身青紫、左上肢轻度青紫、右上肢正常（因右锁骨下动脉在导管近端，血氧正常）。
5. 杂音特征：胸骨左缘第2肋间连续性“机器样”杂音（贯穿收缩期与舒张期），伴震颤；当肺动脉高压时，舒张期成分减弱或消失，仅剩收缩期杂音。
6. 周围血管征：因舒张压降低 → 脉压增宽 → 出现水冲脉、枪击音、毛细血管搏动等。

**变量说明**：
- `aortic_pressure`：主动脉压力（收缩压与舒张压）
- `pulmonary_artery_pressure`：肺动脉压力
- `shunt_direction`：分流方向：左向右（正常）、双向、右向左（艾森门格）
