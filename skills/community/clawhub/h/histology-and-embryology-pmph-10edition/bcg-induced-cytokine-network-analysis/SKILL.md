# BCG诱导细胞因子网络分析

## 适用场景
- 患者已完成至少一次BCG膀胱灌注治疗
- 已留取治疗后的尿液样本用于检测
- 需要评估BCG诱导的局部免疫激活程度

## 执行步骤
1. **确认样本条件**：确保尿液样本采集于BCG灌注治疗后，并记录灌注次数与采样时间点。
2. **检测关键细胞因子**：对尿液进行多重细胞因子检测，重点关注以下类别：
   - **白细胞介素家族**：IL-1、IL-2、IL-5、IL-6、IL-8、IL-10、IL-12、IL-18
   - **肿瘤坏死因子**：TNF
   - **干扰素**：IFN-γ
   - **集落刺激因子**：GM-CSF
   - **趋化因子**：MDC、MCP、IP-10
3. **识别免疫调节网络特征**：判断是否存在Th1型免疫应答主导的模式（如IFN-γ、IL-2、IL-12、IP-10升高）。
4. **排除干扰因素**：注意非特异性炎症可能导致部分因子（如IL-6、IL-8）升高，需结合临床背景综合判断。
5. **量化免疫应答强度**：将细胞因子谱作为BCG免疫激活程度的生物标志物，用于疗效监测或预后评估。

## 注意事项
- 个体间细胞因子表达存在差异，应建立患者自身基线或参考群体范围
- 单一因子升高不足以说明有效免疫应答，需关注整体网络协同性
- 若检测结果异常或与临床表现不符，建议重复检测或结合其他免疫指标验证
