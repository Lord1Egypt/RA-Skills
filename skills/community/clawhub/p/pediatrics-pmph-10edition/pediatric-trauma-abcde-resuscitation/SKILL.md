# 儿科创伤ABCDE复苏流程

## 触发条件
当患儿满足以下条件时立即启用：
- 遭受外部机械力导致的身体伤害（如交通事故、高处坠落）
- 同时存在意识障碍、呼吸异常或循环不稳定

> 注意：不适用于非创伤性急症（如中毒、溺水），且若患儿已死亡且无可逆迹象则不适用。

## 执行步骤

### A：气道（Airway）
- 判断气道是否通畅，检查口、咽、鼻有无异物
- 评估面部、下颌、喉部是否存在创伤导致气道不稳定
- 处理顺序：头部中立位 → 抬下颌/推下颌法 → 吸除分泌物或清除异物
- 若仍梗阻，依次考虑：口咽/鼻咽通气道 → 气管插管（首选）→ 喉罩 → 环甲膜穿刺 → 气管切开
- 气管插管指征包括：已发生或将发生气道梗阻、肺挫伤伴低氧血症、连枷胸、失血性休克、GCS/PGCS ≤8分
- 对可疑颈椎损伤者（如车祸、坠落、GCS≤12、颈痛、神经功能缺损等），全程必须固定颈椎

### B：呼吸（Breathing）
- 观察呼吸频率、节律及呼吸困难体征
- 警惕张力性气胸、连枷胸、开放性气胸、大量血胸
- 紧急处理：
  - 给予高浓度氧气
  - 呼吸过慢/不规则/严重困难 → 气囊面罩通气 → 气管插管+正压通气
  - 张力性气胸或大量血胸 → 立即胸腔穿刺或置管引流
  - 连枷胸 → 胸壁固定后给予正压通气
  - 开放性气胸 → 敷料封闭创口 → 放置胸腔引流管 → 正压通气
- 监测呼气末CO₂、经皮氧饱和度、血气分析，动态调整治疗

### C：循环（Circulation）
- 心动过速是低血容量最早体征（血压和血红蛋白不可靠）
- 控制出血：直接压迫 → 止血带；骨盆骨折者使用骨盆带固定；内出血在伤后3小时内可考虑氨甲环酸
- 采用损伤控制性复苏（DCR）策略：限制晶体液、早期输注加温血液制品、按1:1:1比例输注红细胞/新鲜冰冻血浆/血小板、允许性低血压
- 建立血管通路：优先两条大口径外周静脉 → 骨髓通路（困难时首选）→ 中心静脉
- 液体复苏：
  - 代偿性休克：20ml/kg生理盐水快速输注
  - 无效则输注10ml/kg浓缩红细胞
  - 失代偿性休克：直接输注红细胞10–20ml/kg
- 血管活性药物仅用于神经源性休克或作为低血容量休克的辅助治疗

### D：失能（Disability）
- 紧急指征：GCS/PGCS ≤8、瞳孔不对称/散大固定、严重神经功能缺损
- 目标：预防继发性脑损伤
- 措施：
  - 维持氧饱和度 >95%
  - 有指征者尽早气管插管
  - 低血压者快速补液联合血管活性药维持脑灌注
  - 出现脑疝表现 → 静脉给予甘露醇或高渗盐水降颅压
  - GCS ≤12请神经外科会诊；GCS ≤8考虑颅内压监测或手术干预

### E：充分暴露（Exposure）
- 完全脱去衣物进行全身系统检查，避免遗漏多发伤
- 操作中注意保暖并持续保护颈椎

### 辅助检查（同步进行）
- 基线实验室检查：血常规、生化、血气、凝血功能
- 严重创伤：立即完成血型鉴定与交叉配血
- 床旁超声（FAST）：快速筛查心包积血、气胸、腹腔积液
- X线检查：评估骨折、气胸
- 诊断性腹腔穿刺：判断腹腔内出血性质（必要时）

## 输出要求
按ABCDE顺序输出具体干预措施清单及执行优先级，确保每一步骤针对当前最危及生命的状况。
