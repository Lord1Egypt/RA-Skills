# 四、AI漫剧创作专项指南（漫剧/条漫/动态漫通用）

## 4.1 视觉风格体系

### 画风选择与受众对照

| 视觉风格 | 受众定位 | 适用题材 | AI生成提示词关键词 |
|---------|---------|---------|------------------|
| **国漫古风** | 18-35岁男女 | 仙侠、玄幻、穿越、历史 | 水墨渲染、古建筑群、飘逸服饰、东方美学 |
| **国漫现代** | 18-30岁女性为主 | 都市恋爱、校园、职场 | 精致五官、时尚穿搭、现代都市背景 |
| **日漫画风** | 16-28岁泛人群 | 热血、冒险、日常、治愈 | 动漫大眼睛、漫画速度线、Q版表情 |
| **韩漫风格** | 20-40岁女性 | 霸总、甜宠、复仇 | 精致渲染、电影感构图、暖色调 |
| **写实3D** | 25-45岁男性 | 科幻、战争、工业 | 虚幻引擎画质、影视级光影 |
| **Q版/可爱** | 12-22岁 | 搞笑、萌系、儿童向 | 圆脸、大头、小身子、亮色系 |

### 内容调性对照

| 调性 | 关键词 | 适用场景 |
|------|--------|---------|
| **燃系** | 热血、战斗、崛起、爆发 | 男频逆袭、热血运动 |
| **甜系** | 心动、宠溺、撒娇、暧昧 | 女频恋爱、甜宠日常 |
| **虐系** | 虐心、悲情、牺牲、遗憾 | 悲情女主、BE美学 |
| **爽系** | 打脸、逆袭、反转、碾压 | 赘婿翻身、穿越打脸 |
| **搞笑系** | 吐槽、反差、表情包、梗 | 喜剧短剧、轻松日常 |
| **悬疑系** | 悬念、推理、惊悚、揭秘 | 探案、恐怖、谍战 |

## 4.2 漫剧专属内容套路

### 女频热门套路

**套路1：女强+马甲**
- 女主：外表柔弱/普通，实则隐藏身份（神医/黑客/首富/战神）
- 典型开局：被退婚/被欺负/被背叛 → 身份曝光打脸
- 情绪曲线：压抑(前3集) → 小反击(5集) → 全面碾压(10集)
- 分镜技巧：女主蜕变时刻用慢镜头+特写+情绪音乐铺垫

**套路2：豪门甜宠**
- 霸道总裁+软萌女主，互动模式：壁咚/摸头/公主抱/床咚
- 典型开局：契约结婚/酒后乱性/萌宝助攻/误会上错人
- 情绪曲线：欢喜冤家(前5集) → 心动暧昧(10集) → 公开撒糖(20集)
- 分镜技巧：心动时刻用近景特写+柔光+慢镜头

**套路3：复仇虐渣**
- 女主含冤而死/被害重生/带着仇恨回归
- 典型开局：前世的悲剧在今世重演，但女主已有上帝视角
- 情绪曲线：暗黑蓄力(前5集) → 逐步反击(10集) → 复仇爽点(15集)
- 分镜技巧：复仇打脸用俯冲镜头+速度线+震屏特效

### 男频热门套路

**套路1：战神回归**
- 主角：隐藏身份的上门女婿/底层人物，实为战神/神医/首富
- 典型开局：被全家羞辱、被要求离婚、丈母娘打骂
- 情绪曲线：隐忍(前3集) → 小试牛刀(5集) → 全面曝光(8集)
- 分镜技巧：身份曝光用战场景象闪回+慢动作+沉重BGM

**套路2：系统流**
- 主角获得「神豪系统/学霸系统/战神系统」，完成任务获得奖励
- 典型开局：激活系统 → 完成新手任务 → 获得第一桶金/技能
- 情绪曲线：惊喜感(前2集) → 小成功(5集) → 连续爽点(10集)
- 分镜技巧：系统面板用漫画式UI边框+金色特效+升级音效

**套路3：穿越异世**
- 主角穿越到玄幻/修仙/历史世界，利用现代知识降维打击
- 典型开局：原身是废物少爷/乞丐/战俘，主角接管身体
- 情绪曲线：适应(前2集) → 小试身手(5集) → 惊艳全场(8集)
- 分镜技巧：降维打击用古今对比蒙太奇+主角淡定表情

### 通用爆款元素（男女频皆适用）

**装逼三连（通用打脸公式）**
1. 反派嘲讽：「就你？」→ 主角沉默
2. 旁人不解：「这人谁啊？」→ 主角不动声色
3. 真相揭晓：「竟然是...」（身份曝光）→ 全场震惊

**情绪爆点设计**
- 每3集一个小高潮（打脸/表白/反转）
- 每5集一个大爆点（身份曝光/豪礼出手/关键抉择）
- 每10集一个剧情转折（新角色/新危机/新身份）

**漫剧专属画面语言**
- 震惊表情：满头黑线+符号（💢）+速度线
- 心动感：樱花/心形气泡/背景虚化+柔光
- 战斗场面：速度线+分镜切分+震屏特效
- 悲伤时刻：雨滴/落叶/单色滤镜+慢动作

## 4.3 AI漫剧分镜规范

### 人物一致性保证（核心！）

每集开头必须包含「人物特征固定描述」，写在 `aiGenerationNote` 中：

```
林霄（男主）固定特征：
- 30岁，剑眉星目，眼神冷峻
- 永远穿黑色唐装（或其他固定装扮）
- 左眉有一道旧伤疤（重要标志）
- 说话时微微仰头，态度傲慢

李四（女主）固定特征：
- 25岁，黑长直发型
- 眼角有一颗泪痣
- 穿白色连衣裙为主
- 惊讶时喜欢捂嘴
```

### 分镜格输出标准

每集按以下结构输出分镜（8-12格）：

```
□ 分镜1（0-5秒）：[开场画面]
  - sceneDescription: 详细描述（包括人物表情、动作、场景）
  - dialogue: 对话（不超过20字）
  - cameraMovement: 运镜方式
  - videoPrompt: AI视频/图像生成Prompt

□ 分镜2-11（5-85秒）：[核心内容]
  - 每格聚焦一个小节点
  - 包含：画面+对话+运镜+特效

□ 分镜12（85-90秒）：[结尾悬念]
  - 必须留钩子，引导看下集
```

### 画风统一声明

在剧本开头必须声明以下内容：

```markdown
【画风声明】
- 视觉风格：[国漫古风/日漫画风/韩漫风格等]
- 色调基调：[暖色/冷色/低饱和/高对比]
- 主角固定形象：[3-5个核心特征描述]
- 禁止出现的画面元素：[列出避免的元素]
```

## 4.4 漫剧VS真人短剧核心差异

| 维度 | AI漫剧 | 真人短剧 |
|------|--------|---------|
| 视觉一致性 | 需固定人物描述保证AI不飘移 | 真人演员自然一致 |
| 分镜密度 | 每集8-12格，节奏更快 | 每集可更长 |
| 特效自由度 | 魔法/异能/大场面成本低 | 特效成本高 |
| 表情夸张度 | 可比真人更夸张（漫画语言） | 接近真实 |
| 场景成本 | AI生成，场景丰富多样 | 实拍受限 |
| 制作周期 | 7天/百集 | 30-60天/百集 |
| 适合题材 | 玄幻/异能/大场面 | 现实/都市/家庭 |
