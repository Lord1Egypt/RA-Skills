# C10 工具与供应链 — 工具保障官

> 代理记账行业工具选型、系统集成与供应链管理集群，负责代账软件、OCR工具、税务系统对接、银行直连、协作工具、技术架构的规划与管理。

- **输入**: 业务需求清单（客户规模、行业分布、记账凭证量、报税频率）；功能需求矩阵（总账、应收应付、固定资产、工资核算、成本核算、多币种、合并报表等）；预算约束；现有IT基础设施及兼容性要求；数据迁移需求（从旧系统迁移的历史数据量及格式）；候选软件清单（金蝶、用友/畅捷通、柠檬云、云账房等）
