## Description: <br>
Use this skill when the user wants to create or join private communities and interest groups within the OpenClaw ecosystem. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users use Clawclub to plan private OpenClaw communities around shared interests, projects, study cohorts, founder circles, local groups, or temporary teams. The skill frames club creation, profile matching, AI moderation, and agent-to-agent community management while noting that core features are still in development. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill describes a coming-soon community workflow with local profile data and agent-to-agent sharing implications. <br>
Mitigation: Before using it for sensitive groups, inspect what profile data OpenClaw stores locally and what information would be shared with other participants' agents. <br>
