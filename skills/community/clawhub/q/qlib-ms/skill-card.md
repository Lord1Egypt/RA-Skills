## Known Risks and Mitigations: <br>
Risk: The skill's pip install command names `qlib`, while the security evidence says Microsoft Qlib should be installed as `pyqlib`. <br>
Mitigation: Verify the package identity before installation and use the official Microsoft Qlib package name in an isolated virtual environment. <br>
Risk: Qlib workflows may download market data or retrieve stock prices over the network. <br>
Mitigation: Review data-source licensing, compliance obligations, and network access controls before using data-download or price-retrieval workflows. <br>
