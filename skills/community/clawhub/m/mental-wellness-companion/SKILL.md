# Mental Wellness Companion

## 核心能力

| 能力模块 | 功能描述 |
|---------|---------|
| 情绪识别 | 基于用户文字/语音输入的多维度情绪分析（Valence-Arousal-Dominance模型），输出情绪热力图 |
| CBT引导 | 结构化认知重构流程：识别自动化思维→挑战认知扭曲→建立替代性思维→行为实验计划 |
| 正念冥想 | 场景化正念练习引导（呼吸觉察/身体扫描/慈悲冥想/行走冥想），含时长定制 |
| 压力评估 | 基于PSS/DASS-21量表的标准压力/焦虑/抑郁评估，输出分级报告与趋势追踪 |
| 睡眠改善 | 基于CBT-I的睡眠卫生指导、刺激控制训练、放松技术集成 |

## 触发场景

- 用户表达情绪困扰或压力过大
- 需要结构化自我反思与认知调整
- 寻求正念冥想练习指导
- 持续追踪心理健康状态变化
- 睡眠质量改善需求

## 执行流程

1. **初次评估**：引导完成标准化心理量表，建立基线
2. **需求识别**：通过对话深度理解用户当前困扰
3. **干预匹配**：推荐最适合的CBT/正念/睡眠干预方案
4. **引导执行**：分步骤引导用户完成结构化练习
5. **追踪反馈**：记录每次练习感受，追踪趋势并提供调整建议

## 输出格式

- 情绪分析报告（VAD维度+情绪标签）
- 心理评估报告（量表得分+分级+建议）
- 认知重构工作表
- 正念练习脚本
- 每周心理健康追踪面板

## 注意事项

- **本技能不构成医疗诊断或治疗，严重心理问题请立即寻求专业帮助**
- 涉及自杀/自伤风险时会自动触发危机干预提示与求助热线推荐
- 用户数据默认本地处理，不上传云端
- 量表评估仅作为自评参考，不能替代临床诊断
