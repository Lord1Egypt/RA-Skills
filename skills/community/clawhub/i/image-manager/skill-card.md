## Description: <br>
Manage local images by category with original and thumbnail storage, enabling fast tag, date, and keyword search through a JSON index file. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[xiaobu2020](https://clawhub.ai/user/xiaobu2020) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill to save, index, browse, and search local image collections by category, tags, date, and keywords. It is useful when an agent needs reusable local media organization with preserved originals and quick thumbnail previews. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Local image files and metadata may include sensitive personal or financial information. <br>
Mitigation: Use a dedicated media folder, avoid saving sensitive images unless needed, and periodically review or delete stored media and index entries. <br>
Risk: Search results depend on user-provided categories, tags, descriptions, and retained index metadata. <br>
Mitigation: Review tags and descriptions before relying on search results, especially when receipts or personal images are indexed. <br>
