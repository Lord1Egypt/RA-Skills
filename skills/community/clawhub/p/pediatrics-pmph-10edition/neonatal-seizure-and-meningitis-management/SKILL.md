# 新生儿惊厥与细菌性脑膜炎处理

## 适用场景
- 惊厥发作持续>5分钟
- 临床高度怀疑或确诊细菌性脑膜炎

## 执行步骤
1. **新生儿惊厥止惊流程**：
   - **一线**：静脉地西泮0.3–0.5 mg/kg 或 肌注咪达唑仑0.2–0.3 mg/kg；
   - **二线**：静脉苯巴比妥钠负荷量10 mg/kg；
   - **三线**：水合氯醛0.5 ml/kg 灌肠。
2. **细菌性脑膜炎抗生素治疗**：
   - **经验性**：
     - 早期新生儿：氨苄西林 + 头孢噻肟；
     - ≥1月龄：万古霉素 + 第三代头孢；
   - **针对性**：根据病原菌（肺炎球菌、流感嗜血杆菌等）调整；
   - **疗程**：7–21天，依病原菌而定。
3. **预后评估**：
   - 高危因素：肺炎球菌感染、<6个月、惊厥>4天；
   - 病死率约10%，10–20%幸存者有后遗症（听力丧失、智力倒退等）。
4. **癫痫鉴别**：排除屏气发作、晕厥、症性发作、抽动障碍等非癫痫性疾病。

## 输出结果
- 按优先级执行止惊药物干预；
- 启动经验性抗生素方案，并根据病原菌调整；
- 若存在高危因素，标记为“高风险预后”，加强监护。
