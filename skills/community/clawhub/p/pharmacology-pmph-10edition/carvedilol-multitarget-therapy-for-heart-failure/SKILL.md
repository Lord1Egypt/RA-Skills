# 卡维地洛多靶点治疗心力衰竭与高血压

## 适用条件
- 患者诊断为以下任一情况：
  - 充血性心力衰竭（症状稳定、慢性）
  - 轻至中度高血压
  - 缺血性心脏病
- **必须满足前提**：无急性心力衰竭失代偿、可耐受β受体阻断药
- **禁忌/慎用**：急性心衰禁用；严重心动过缓或低血压患者慎用

## 执行步骤
1. **确认药理基础**：卡维地洛为α₁、β₁、β₂受体非选择性阻断剂，兼具抗氧化、抗心肌重构及潜在钙拮抗作用，适用于长期心血管保护。
2. **启动治疗**：
   - 起始剂量：3.125～6.25 mg，每日2次（bid）
   - **必须从小剂量开始**，避免诱发心衰恶化或低血压
3. **剂量滴定**：
   - 每2周评估患者耐受性（监测血压、心率、心功能症状）
   - 若耐受良好，逐步递增剂量
   - 目标维持剂量：最高可达50 mg，每日2次
4. **持续监测**：
   - 密切观察是否出现低血压、心动过缓、乏力或心衰加重迹象
   - 定期评估射血分数、生活质量及住院风险

## 预期结果
- 改善心衰症状与射血分数
- 防止并逆转心肌重构
- 降低住院率与全因死亡率
- 有效控制血压（疗效与其他β阻断药或CCB相当）

> **注意**：本方案仅适用于病情稳定的慢性患者，急性期绝对禁用。
