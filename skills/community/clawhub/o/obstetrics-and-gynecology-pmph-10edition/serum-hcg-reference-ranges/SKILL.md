# 血清hCG浓度正常参考范围（ECLIA法）

## 何时使用
- 需确认女性是否妊娠
- 怀疑异位妊娠（如hCG上升缓慢或平台）
- 评估滋养细胞疾病（如葡萄胎、绒癌）
- 排查分泌hCG的肿瘤

## 核心执行步骤

### 1. 确认检测方法与患者状态
- 必须使用**ECLIA（电化学发光免疫分析法）**
- 明确患者为：非妊娠（分绝经前后）或妊娠（需准确孕周）

### 2. 对照正常参考范围（单位：IU/L）
- **非妊娠女性（绝经前）**：0～5.3
- **非妊娠女性（绝经后）**：0～8.3
- **妊娠4周**：10.2～708
- **妊娠6周**：152～32,177
- **妊娠8周**：31,366～149,094
- **妊娠10周**：44,186～170,409

### 3. 解读结果
- **hCG < 正常下限**：排除妊娠（除非极早期）
- **hCG在范围内**：符合相应孕周正常妊娠
- **hCG显著高于上限**：警惕多胎妊娠、葡萄胎或滋养细胞肿瘤
- **hCG低于预期或上升缓慢**：高度怀疑异位妊娠或胚胎停育

> **注意**：不同检测方法结果不可比；本范围仅适用于ECLIA法。
