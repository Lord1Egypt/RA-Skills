# Scoring, de-duplication, and publishable formatting

Use this file when you need a stricter quality bar.

## Candidate scoring

Score each candidate 1–5 on these dimensions:

- **重要性**: Will this actually matter to the audience?
- **新颖性**: Is this genuinely new or just another repost?
- **可用性**: Can readers use this insight, tool, paper, or trend?
- **可信度**: Is the source official, primary, or highly reliable?
- **传播性**: Is it suitable for a compelling briefing or publishable post?

Recommended rule:
- 20+ total: strong candidate
- 16–19: maybe include if it fits the theme
- 15 or below: usually drop

## De-duplication rules

When multiple sources describe the same event:

1. Keep the best primary source
2. Keep at most one secondary source if it adds context
3. Merge repeated stories into one item
4. Never list the same product/paper/project twice under different headings

## Signal vs noise filters

Strong signal examples:
- official model/product release
- major open-source repo traction with real adoption signs
- paper with benchmark significance or unusual practical relevance
- trend that changes content strategy, product choices, or overseas expansion decisions

Weak signal examples:
- generic repost threads
- shallow media rewrites of press releases
- low-substance product launch pages
- hype with no numbers, demo, users, benchmark, or practical implication

## Publishable copy rules

### 朋友圈 / 社媒版
- 80–180 Chinese characters
- one main angle only
- readable in one screen
- emphasize "why it matters"

### 公众号版
- 300–800 Chinese characters for short publishable summary
- stronger structure: opening hook -> 3 points -> judgment
- avoid excessive jargon

### 内部汇报版
- use bullet points
- focus on impact, relevance, decision value
- less dramatic language, more implication-oriented language

## Hook styles

Useful opening angles:
- “今天最值得关注的，不是更多新工具，而是……”
- “如果你只看一条 AI 信息，建议看这个……”
- “本周真正有信号的变化，集中在 3 件事上……”
- “对内容团队最有价值的，不是热闹，而是这几个可落地方向……”

## Final check

Before sending, verify:
- top items are not repetitive
- the title matches the strongest theme
- each item contains a clear significance sentence
- the publishable copy sounds like a good editor wrote it, not a scraper
