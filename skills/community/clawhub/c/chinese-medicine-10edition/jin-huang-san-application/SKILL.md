# Jin Huang San Application

## When to Use
- 患处红肿、灼热、疼痛，尚未化脓；
- 伴随全身症状：舌红、苔黄、脉滑数；
- 病证属于中医“阳证疮疡”范畴。

**禁忌**：阴证疮疡（如漫肿无头、皮色不变、不热不痛）或已溃破流脓者禁用。

## Execution Steps
1. **确认药物组成**（按古方比例）：
   - 君药：天花粉十斤、黄柏五斤、大黄五斤、姜黄五斤、白芷五斤；
   - 臣药：厚朴二斤、陈皮二斤、苍术二斤、天南星二斤；
   - 佐药：姜黄（活血）；
   - 使药：甘草二斤（调和诸药）。

2. **制备方法**：将上述药材共研为极细粉末。

3. **调敷方法**：根据病情选择调敷介质：
   - 醋：收敛止痛；
   - 酒：增强活血通络；
   - 蜂蜜：润肤护创；
   - 植物油：缓释药效、减少刺激。
   调成糊状后外敷患处，每日1–2次。

4. **预期效果**：减轻红肿热痛，控制炎症进展，促进阳证疮疡消散。

5. **现代适用病证**（需符合阳证实证）：
   - 感染性炎症：下肢丹毒、流行性腮腺炎、急性乳腺炎、阑尾周围脓肿；
   - 损伤类疾病：骨折及软组织损伤、静脉炎、痛风急性发作；
   - 皮肤慢性病：糖尿病足溃疡、压疮、湿疹、淤滞性皮炎；
   - 其他：慢性前列腺炎、炎性外痔、毒蛇咬伤肢体肿胀等。

> 注意：使用前必须辨证为阳证，避免误用于虚寒或已溃脓阶段。
