# 报警/报案指引

## 适用场景
公民发现自身或他人人身权利、财产权利受到违法犯罪行为侵害，或发现违法犯罪线索时，向公安机关报案、控告或举报。包括但不限于：盗窃/抢劫/抢夺/诈骗（含电信网络诈骗）、故意伤害、寻衅滋事、敲诈勒索、非法拘禁、侵犯公民个人信息、职务侵占、合同诈骗、侵犯知识产权等。

## 管辖机构

### 一般刑事案件
| 项目 | 内容 |
|------|------|
| 机构名称 | 案发地/结果发生地公安派出所 或 区（县）公安（分）局刑侦部门 |
| 管辖原则 | **犯罪地**（犯罪行为发生地+犯罪结果发生地）或 **嫌疑人居住地** |
| 紧急情况 | 直接拨打 **110** |
| 非紧急 | 前往派出所报案 或 拨打当地派出所电话 |
| 工作时间 | 派出所24小时值班；刑侦部门一般为工作日 |

### 电信网络诈骗专项
| 项目 | 内容 |
|------|------|
| 紧急止付 | 发现被骗后立即拨打 **96110**（反诈专线）或 **110** |
| 全国反诈中心 | 国家反诈中心App — "我要举报"功能 |
| 冻结时效 | 转账后 **30分钟内** 止付成功率最高 |

### 经济犯罪（合同诈骗/职务侵占/非法集资等）
| 项目 | 内容 |
|------|------|
| 管辖机构 | 区（县）公安（分）局 **经侦大队** |
| 报案地点 | 一般需到经侦部门专门窗口，派出所可能不受理复杂经济案件 |

### 网络犯罪/数据安全
| 项目 | 内容 |
|------|------|
| 管辖机构 | 区（县）公安（分）局 **网安大队** |
| 举报平台 | www.cyberpolice.cn（网络违法犯罪举报网站） |

## 所需材料

### 报案人本人报案

| 序号 | 材料 | 份数 | 要求 | 模板 |
|------|------|------|------|------|
| 1 | 报案人身份证 | 1份（核原件） | 本人身份证原件+正反面复印件 | 无 |
| 2 | 报案材料/控告书 | 1份 | 载明：报案人信息、嫌疑人信息（如知悉）、案发时间/地点/经过、损失情况、诉求 | 有（可自拟） |
| 3 | 证据材料 | 1份 | 视案件类型：<br>- 诈骗：转账记录/聊天记录/APP截图<br>- 伤害：伤情照片/医院病历/鉴定报告<br>- 盗窃：失物清单/购买凭证<br>- 合同诈骗：合同+付款凭证+对方虚假材料 | 无 |
| 4 | 损失清单及凭证 | 1份 | 财产损失的金额+对应的银行流水/微信/支付宝转账截图 | 无 |
| 5 | 证人信息 | 1份（如有） | 证人姓名/联系方式 | 无 |

### 单位报案

| 序号 | 材料 | 份数 | 要求 | 模板 |
|------|------|------|------|------|
| 1 | 营业执照副本复印件 | 1份 | 加盖公章 | 无 |
| 2 | 法定代表人身份证明 | 1份 | 加盖公章 | 有 |
| 3 | 授权委托书 | 1份 | 加盖公章（非法定代表人亲自报案时） | 有 |
| 4 | 报案人身份证及工作证明 | 1份 | — | 无 |
| 5 | 报案材料（加盖公章） | 1份 | 详述案情+损失+证据 | 无 |

## 提交流程

### 紧急情况（正在发生的犯罪/人身危险）
1. 立即拨打 **110**（手机无信号时可拨打112）
2. 说明：地点、什么事情、有无人员受伤/危险、嫌疑人特征/去向
3. 保持电话畅通，等待出警
4. 出警后配合民警制作笔录，领取《接报案回执》

### 非紧急案件（去派出所报案）

1. **准备报案材料**：按上述清单整理，将证据分类并按时间线排列
2. **前往派出所**：携带身份证+全部材料前往案发地派出所
3. **陈述案情**：向值班民警说明报案事项，按时间线客观叙述事实经过
4. **制作笔录**：民警询问并制作《询问笔录》，仔细核对记录内容后签名/捺印
5. **提交证据**：将证据材料提交民警，原件留存复印件交民警
6. **领取回执**：
   - 受理的 → 领取《受案回执》/《接报案回执》（载明受案编号）
   - 当场不能决定是否受理的 → 要求出具《接受证据材料清单》
   - 不属管辖的 → 要求出具《移送案件通知书》

### 经济犯罪报案（去经侦大队）

1. **准备详细的书面报案材料**：包括当事人信息、案件来龙去脉、涉嫌罪名及法条依据、损失计算
2. **携带全部合同/凭证/账目**：按时间顺序整理
3. **前往区公安局经侦大队**：一般在区公安分局内
4. **耐心配合初查**：经济案件受理前通常有初查环节，可能需多次补充材料
5. **跟踪立案进度**：经侦案件立案周期较长，需定期跟进

### 电信诈骗紧急处置
1. 发现被骗后 **立即拨打96110** 或银行客服冻结对方账户
2. 同时拨打110报警
3. 保存所有：聊天记录（不要删）、转账凭证截图、对方账号/手机号/微信/QQ、APP下载链接、网址
4. 前往派出所制作笔录
5. 配合反诈中心进行资金追查

## 费用
- **完全免费**，公安机关不收取任何报案费用

## 时间预估

| 阶段 | 时间 | 说明 |
|------|------|------|
| 110出警 | 城区一般10分钟内 | 紧急情况 |
| 受案审查 | 一般当场或24小时内 | 行政案件 |
| 立案审查 | 一般 **7日内**（重大疑难复杂可延至30-60日） | 刑事案件 |
| 不予立案通知 | 审查期满后3日内送达 | 不服可申请复议/检察院监督 |
| 侦查期限 | 逮捕后一般2个月，可延长 | — |

## 注意事项

### 报案前
1. **先固定证据**：在报案前做好证据备份——聊天记录截图/录屏、转账记录保存、通话录音等。切勿删除任何记录
2. **安全性优先**：如涉及人身威胁，首先确保自身安全，再考虑报案
3. **区分案件性质**：
   - 行政违法 → 派出所受理（治安案件）
   - 刑事犯罪 → 派出所或刑侦/经侦/网安受理
   - 民事纠纷 → 不属公安管辖，应去法院（但派出所可调解）

### 报案时
4. **客观陈述**：只陈述已知的客观事实，不要猜测、不要夸张、不要隐瞒对自己不利的事实
5. **核对笔录**：签字前务必逐字核对笔录内容，与自己陈述不一致的要求更正
6. **一定要拿回执**：不论是否当场受理，都要索取书面回执或材料接收凭证
7. **多人受害可联合报案**：同一嫌疑人涉及多起案件，联合报案有助于并案处理

### 常见不予受理情形及应对
| 情形 | 应对方案 |
|------|----------|
| 以"民事纠纷"为由不受理 | 要求出具《不予立案通知书》→向检察院申请立案监督 |
| 以"证据不足"为由不受理 | 按民警指引补充证据后再次报案 |
| 以"金额小"为由不受理 | 多次盗窃/入户盗窃/携带凶器盗窃无金额门槛（构成犯罪） |
| 以"不属本辖区"为由不受理 | 要求移送有管辖权的公安机关并出具移送文书 |

### 报案后
8. **保留案件编号**：妥善保存受案编号/回执，这是后续查询和维权的凭证
9. **及时补充材料**：如警方要求补充证据，尽快提供
10. **立案监督途径**：公安机关应当立案而不立案的 → 向检察院申请立案监督；也可向上一级公安机关申请复议
