# bounty-hunter 详细参考

## 一、外包平台攻略

### Upwork（全球最大）

**入门要求**：需通过英语能力测试和技能测试，通过率约30%

**操作流程**：
1. 完善Profile（照片+简介+作品集+工作经历）
2. 选择3-5个专业领域
3. 购买Connects（平台虚拟货币，约$10-15/10个）
4. 搜索Jobs过滤：金额>$50、技能匹配、发布时间<24h
5. 撰写Proposal（每条$1-3 Connects）
6. 等待Client回复→视频面试→签约

**Proposal模板**：
```
Hi [Client Name],

I've reviewed your project requirements for [project name] and I'm confident I can deliver exceptional results.

**My Relevant Experience:**
- [具体项目经验1]
- [具体项目经验2]

**How I'll Approach This:**
1. [阶段1：做什么]
2. [阶段2：做什么]
3. [阶段3：做什么]

**Timeline & Budget:**
- Total: $[amount]
- Delivery: [X] business days

I'm available for an immediate start. Looking forward to discussing further!

Best,
[Your Name]
```

**收入参考**：
| 项目类型 | 平均时薪 | 月入潜力 |
|----------|----------|----------|
| 写作/翻译 | $15-30 | $1,200-2,400 |
| Web开发 | $30-60 | $2,400-4,800 |
| 移动开发 | $40-80 | $3,200-6,400 |
| 数据分析 | $25-50 | $2,000-4,000 |

### Fiverr（服务制）

**核心逻辑**：创建Gig（服务产品），Client主动购买

**创建Gig步骤**：
1. 选择核心服务类别
2. 命名：精准关键词+差异化
   - ✅ "Professional SEO Article Writing in 24h"
   - ❌ "I will write articles"
3. 设置定价层级：
   - Basic ($5-25)：基础服务
   - Standard ($50-100)：进阶服务
   - Premium ($150-500)：高端服务
4. 上传封面图（7张）：突出服务价值
5. 撰写Gig Description
6. 设置Favorites和FAQ

**Gig模板**：
```
**What You Get:**
✅ [核心交付物1]
✅ [核心交付物2]
✅ [核心交付物3]
✅ [Bonus]

**Process:**
1️⃣ Order → 2️⃣ Provide Details → 3️⃣ Draft → 4️⃣ Revisions → 5️⃣ Delivery

**Delivery:** [X] days
**Revisions:** Unlimited within scope
```

**提现**：PayPal（手续费1%）或Fiverr Revenue Card

---

### 猪八戒网（国内首选）

**入驻流程**：
1. 注册账号（手机号+实名认证）
2. 完善服务商资料
3. 缴纳诚意金（可选，缴纳后可接更高金额订单）
4. 技能认证（通过测试获得认证标识）

**订单类型**：
| 模式 | 说明 | 适合人群 |
|------|------|----------|
| 招标 | 发布需求→服务商投标→选标 | 有竞价优势 |
| 店铺 | 建立店铺→客户主动下单 | 品牌化运营 |
| 定制 | 1V1需求对接 | 高端服务 |

**报价策略**：
```
报价 = 预估工时 × 单价 - 利润空间

参考单价：
- 简单文案：50-200元
- 平面设计：200-1000元
- Web开发：1000-10000元
- APP开发：5000-50000元
```

## 三、漏洞赏金攻略

### HackerOne（国际最大平台）

**入门路线**：
```
1. 学习阶段（1-3月）
   - 刷PortSwigger Web Academy
   - 学习Burp Suite工具
   - 实战DVWA靶场
   
2. 小目标阶段（3-6月）
   - 专注子域名枚举
   - 找XSS/IDOR等中低危漏洞
   - 目标：$500-1000/月
   
3. 进阶阶段（6-12月）
   - 审计API接口
   - 逻辑漏洞挖掘
   - 目标：$1000-5000/月
   
4. 专业阶段（1年+）
   - 组合漏洞利用
   - 大厂赏金猎人
   - 目标：$5000+/月
```

**常用工具**：
```bash
# 子域名枚举
amass enum -passive -d example.com

# 端口扫描
nmap -sV -p 1-10000 target.com

# Web漏洞扫描
nikto -h https://target.com

# 被动收集
subfinder -d target.com | httpx
```

**漏洞报告模板**：
```
# [漏洞名称]

## 漏洞类型

## 严重程度
[Medium - 6.1]

## 漏洞描述
[详细描述漏洞及影响]

## 复现步骤
1. 访问 [URL]
2. 在 [参数] 输入 [Payload]
3. 观察 [结果]

## 修复建议
[提供修复方案]

## 影响范围
[影响的URL和用户数量]
```

## 四、投标与报价技巧

### 定价策略

**成本加成法**：
```
报价 = (工时 × 时薪) × 1.2(利润) × 1.1(平台费)

示例：预计20小时，时薪$50
= (20 × 50) × 1.2 × 1.1 = $1,320
```

**市场对标法**：
- 搜索类似项目历史成交价
- 报价=平均值×0.9~1.1

**阶梯报价**：
```
基础版：$XXX（核心功能）
标准版：$XXX（完整功能）
高级版：$XXX（VIP服务+维护）
```

### 投标加分项

1. **展示相关案例**：截图/链接/详细说明
2. **快速响应**：24h内回复+即时开始
3. **提供建议**：主动给客户优化建议
4. **风险承诺**：提供bug修复期
5. **视频介绍**：3分钟自我介绍+案例展示

### 拒绝的订单

- ❌ 报价低于市场50%
- ❌ 需求模糊无法评估
- ❌ 客户无评价/差评多
- ❌ 尾款比例过低（<30%）
- ❌ 要求试稿不给费用
