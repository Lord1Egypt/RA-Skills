# 详细病原体致癌机制参考

## DNA肿瘤病毒

### 人乳头瘤病毒（HPV）
- **高危型别**：HPV-16、HPV-18为主
- **E6蛋白**：结合E6AP泛素连接酶，介导p53泛素化降解 → 抑制凋亡与DNA修复
- **E7蛋白**：结合并失活Rb蛋白 → E2F转录因子释放 → S期基因异常表达 → 细胞周期失控
- **主要肿瘤**：宫颈鳞癌、肛门癌、口咽癌

### EB病毒（EBV）
- **B细胞感染**：
  - LMP1模拟CD40信号 → NF-κB持续激活 → B细胞多克隆增殖
  - 后续获得MYC重排或NRAS突变 → 单克隆转化 → Burkitt淋巴瘤
- **上皮细胞感染**（鼻咽）：
  - EBNA1维持病毒基因组；LMP1/LMP2A促进存活与迁移
  - 在中国南方，>95%鼻咽癌检出EBV基因组

### 乙型肝炎病毒（HBV）
- **HBx蛋白作用**：
  - 激活Src、PI3K/Akt、Wnt/β-catenin通路
  - 抑制p53转录活性
- **间接机制**：
  - 慢性肝炎 → 肝纤维化 → 肝硬化 → 肝细胞反复再生 → 基因组不稳定性增加
- **风险倍数**：慢性感染者肝癌风险为未感染者约200倍

## RNA肿瘤病毒

### HTLV-1
- **Tax蛋白功能**：
  - 激活CREB、NF-κB通路
  - 上调IL-2、IL-2Rα、FOS、PDGFB、CSF2等促增殖基因
  - 抑制DNA修复（干扰ATM通路）
- **临床关联**：成人T细胞白血病/淋巴瘤（ATLL），潜伏期可达数十年

## 细菌

### 幽门螺杆菌（H. pylori）
- **CagA毒素**：
  - 注入胃上皮细胞后被磷酸化 → 干扰SHP-2、PAR1/MARK激酶 → 细胞极性丧失
- **慢性炎症**：
  - IL-1β、TNF-α等促炎因子抑制胃酸分泌 → 胃内菌群失调 → 氧化应激 → DNA损伤
- **肿瘤关联**：
  - 胃MALT淋巴瘤：抗原驱动B细胞增殖，部分病例根除H. pylori后可完全缓解
  - 胃腺癌：Correa级联（慢性胃炎→萎缩→肠化→异型增生→癌）

## 寄生虫

| 寄生虫 | 感染部位 | 相关肿瘤 |
|--------|----------|----------|
| 埃及血吸虫 | 膀胱壁 | 膀胱鳞状细胞癌 |
| 华支睾吸虫 | 肝内胆管 | 胆管细胞癌 |
| 日本血吸虫 | 结肠黏膜 | 结肠腺癌 |

> 注：寄生虫致癌主要通过慢性机械刺激、炎症介质释放及亚硝胺类物质生成。
