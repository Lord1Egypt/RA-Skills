# 浸润型肺结核的演变路径与转归

## 触发条件
当满足以下条件时启用本技能：
- 患者为继发性肺结核患者或局灶型肺结核进展者；
- X线检查显示锁骨下区域边缘模糊的云絮状阴影；
- 伴有低热、疲乏、盗汗、咳嗽等结核中毒症状。

> **注意**：本技能不适用于原发性肺结核或非活动性结核。

## 执行步骤

1. **确认影像学特征**：核查X线是否显示锁骨下区域边缘模糊的云絮状阴影。
2. **评估病理特点**：确认病变为以渗出性炎症为主，中央存在干酪样坏死，周围有炎性包绕。
3. **识别临床症状**：记录是否存在低热、疲乏、盗汗、咳嗽等典型结核中毒表现。
4. **判断转归方向**：
   - **良性转归（及早合理治疗）**：
     - 渗出性病变可完全吸收；
     - 增生与坏死性病变通过纤维化、钙化愈合。
   - **恶性进展（未及时治疗）**：
     - 干酪样坏死扩大（“浸润进展”）；
     - 坏死物液化经支气管排出 → 形成急性空洞；
     - 急性空洞内含大量结核分枝杆菌 → 经支气管播散 → 引起干酪性肺炎（“溶解播散”）。
5. **评估空洞愈合方式**：
   - 洞壁肉芽组织增生 → 洞腔缩小闭合 → 瘢痕形成；
   - 或空洞塌陷 → 条索状瘢痕愈合。
6. **预警慢性转化**：若急性空洞经久不愈，需警惕进展为慢性纤维空洞型肺结核。
7. **立即启动抗结核治疗**：一旦确诊浸润型肺结核，应立即开始规范抗结核治疗，以阻止空洞形成或干酪性肺炎的发生。

## 关键变量说明
- **病变性质**：以渗出为主，区别于局灶型肺结核的增生为主。
- **播散机制**：“溶解播散”指坏死物液化后经支气管播散至其他肺段。

## 输出建议
需立即启动抗结核治疗以阻止进展为空洞或干酪性肺炎。
