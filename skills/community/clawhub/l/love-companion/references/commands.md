# 完整指令参考手册

> 全AI Agent通用标准化指令，即插即用

## ⚠️ 免责声明

- 本技能为 **AI 角色扮演工具**，所有对话均为 AI 生成，**不代表真实人际关系**。

## 指令格式说明

- 所有指令以 `/恋人` 开头
- 方括号 `[]` 表示必填参数
- 圆括号 `()` 表示可选参数
- 指令大小写不敏感

---

## 一、核心开关指令

### /恋人启用

**功能**：激活恋人Skill，进入情感对话模式

**响应**：
- 加载当前人设配置
- 激活记忆系统
- 以恋人身份开始对话

**示例**：
```
/恋人启用
```

---

### /恋人关闭

**功能**：关闭恋人模式，恢复普通AI对话

**响应**：
- 退出恋人角色
- 保留所有配置和记忆
- 可随时再次启用

**示例**：
```
/恋人关闭
```

---

## 二、人设配置指令

### /恋人配置

**功能**：查看或修改当前人设配置

**用法**：
- 不带参数：显示当前完整配置
- 带参数：修改指定配置项

**语法**：
```
/恋人配置                          # 查看当前配置
/恋人配置 [JSON内容]               # 直接传入JSON配置
/恋人配置 [维度]=[值]              # 修改单个维度
/恋人配置 [维度].[子维度]=[值]     # 修改嵌套维度
```

**示例**：
```
/恋人配置
/恋人配置 姓名=小雨
/恋人配置 性格.核心特质 = ["温柔", "体贴"]
/恋人配置 {"姓名":"小晴","年龄":22}
```

---

### /恋人预设

**功能**：列出所有可用的预设人设模板

**响应**：显示预设编号和名称列表

**示例**：
```
/恋人预设
```

输出示例：
```
【预设人设列表】
1. 阳光开朗型 - 活泼正能量，像小太阳
2. 温柔治愈型 - 贴心关怀，情感细腻
3. 傲娇高冷型 - 外冷内热，欲拒还迎
...
```

---

### /恋人套用 [序号]

**功能**：一键套用预设人设模板

**参数**：
- `序号`：预设编号（1-8）

**示例**：
```
/恋人套用 1      # 套用阳光开朗型
/恋人套用 3      # 套用傲娇高冷型
```

**注意**：套用后会覆盖当前配置，建议先保存当前方案

---

### /恋人重置

**功能**：重置为默认空白人设

**响应**：清空所有自定义配置，恢复初始状态

**示例**：
```
/恋人重置
```

## 三、方案管理指令

### /恋人保存 [方案名]

**功能**：将当前人设保存为命名方案

**参数**：
- `方案名`：自定义方案名称

**示例**：
```
/恋人保存 我的初恋型
/恋人保存 专属小可爱
```

---

### /恋人方案

**功能**：列出所有已保存的人设方案

**响应**：显示方案名称、创建时间、简要描述

**示例**：
```
/恋人方案
```

输出示例：
```
【已保存方案】
1. 我的初恋型（2024-05-01创建）
2. 专属小可爱（2024-05-02创建）
```

---

### /恋人切换 [方案名]

**功能**：切换到指定的人设方案

**参数**：
- `方案名`：已保存的方案名称

**示例**：
```
/恋人切换 我的初恋型
```

---

### /恋人删除方案 [方案名]

**功能**：删除指定的已保存方案

**参数**：
- `方案名`：要删除的方案名称

**示例**：
```
/恋人删除方案 我的初恋型
```

## 四、记忆管理指令

### /恋人记忆 [内容]

**功能**：手动添加记忆内容到长时记忆

**参数**：
- `内容`：要记忆的信息

**示例**：
```
/恋人记忆 我最讨厌被敷衍
/恋人记忆 我的生日是5月20日
/恋人记忆 我最近在准备考研，压力很大
```

---

### /查看记忆

**功能**：查看当前所有长时记忆

**响应**：显示记忆条目列表，包含编号、内容、记录时间

**示例**：
```
/查看记忆
```

输出示例：
```
【长时记忆】
1. 用户生日：5月20日（记录于2024-04-15）
2. 用户讨厌被敷衍（记录于2024-04-20）
3. 用户正在备考研究生（记录于2024-05-01）
```

---

### /删除记忆 [关键词]

**功能**：删除包含指定关键词的记忆条目

**参数**：
- `关键词`：用于匹配的记忆关键词

**示例**：
```
/删除记忆 生日
/删除记忆 考研
```

---

### /恋人清空记忆

**功能**：清空所有对话记忆与长时记忆

**注意**：此操作不可恢复

**示例**：
```
/恋人清空记忆
```
