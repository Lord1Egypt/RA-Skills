## 二、文献管理工具

### Zotero（最推荐）
- **免费**：开源免费
- **跨平台**：Win/Mac/Linux
- **核心功能**：一键抓取文献、分类管理、标签系统、PDF标注
- **插件推荐**：Zotero插件推荐、Better BibTeX（LaTeX用户必装）
- **同步**：Zotero官方提供300MB免费同步，可付费扩容

### EndNote
- **收费**：机构版或个人版
- **优势**：与Web of Science深度集成
- **适用**：大型综述研究、需要高级引用功能

### Mendeley
- **基础免费**，进阶收费
- **特色**：AI文献推荐、PDF阅读标注、学术社交

### NoteExpress
- **中国市场专用**：知网兼容性最佳
- **优势**：中文文献管理首选

## 三、论文写作工具

### Overleaf（在线LaTeX编辑器）
- **免费**版功能够用，无限制合作者
- **模板丰富**：几乎涵盖所有期刊模板
- **实时协作**：多人同时编辑
- **版本控制**：历史版本可追溯

### Word（传统选择）
- **通用性最强**：几乎所有期刊支持
- **配合工具**：Zotero/EndNote插件
- **适合**：中文论文、非技术性论文

### Markdown + Pandoc
- **轻量高效**：纯文本写作
- **转换能力强**：Pandoc可转LaTeX/Word/HTML/PDF
- **推荐工具**：Typora / Zettlr / VS Code
- **适合**：技术文档、前期草稿、快速写作

## 五、AI辅助工具（需注明使用声明）

### 学术写作辅助
- **ChatGPT/GPT-4**：润色、修改语法、改写段落
- **DeepL Write**：学术英语润色
- **Writefull**：专门为学术写作设计的AI助手
- **Paperpal**：学术论文AI编辑

### 文献分析和总结
- **Connected Papers**：文献关联图谱分析
- **Scite**：引文上下文分析
- **Elicit**：AI驱动的文献检索助手
- **Research Rabbit**：文献探索和发现

### 使用原则
1. **学术诚信声明**：多数期刊要求声明AI工具使用情况
2. **不生成核心内容**：AI辅助润色和翻译可接受，不宜生成实验设计和数据分析
3. **作者完全负责**：AI工具生成的任何内容由作者承担学术责任
4. **不引用不可靠来源**：AI可能生成虚构的参考文献
