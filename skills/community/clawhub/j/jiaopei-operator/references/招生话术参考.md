# 教培机构招生话术参考 📞

## 一、电话咨询话术

### 首次接听（30秒黄金法则）

**目标：约体验课，不卖课**

```
标准开场：
"您好，欢迎致电XX培训机构，我是课程顾问XX老师。
请问您是想给孩子咨询什么课程呢？"

确认需求：
"孩子多大了？之前有学过相关的课程吗？"

自然过渡：
"我们目前有[年龄段]的免费体验课，您方便带孩子来感受一下吗？
体验课是XX分钟，孩子可以实际参与，我们老师也会给专业评估。"

约定时间：
"您这周哪天比较方便呢？我们有XX时间段可选。"
```

### 家长犹豫时

**家长："我再考虑考虑"**
```
共情：
"完全理解，给孩子选机构确实要慎重。
那我想问一下，您主要是考虑哪方面呢？是距离、时间还是课程内容？"

针对性回应：
- 距离："我们离您大概多远呢？我可以帮您看看交通是否方便"
- 时间："您平时什么时间段比较方便？我们可以灵活安排"
- 课程："我们的课程特色是XX，您可以先体验再决定"

继续邀约：
"要不这样，您先带孩子来上一节体验课，反正不收费，
您也可以亲眼看看孩子喜不喜欢、老师专不专业。
体验完您再考虑也不迟，您说呢？"
```

**家长："你们学费多少"**
```
不直接报价：
"学费是根据课程和班型来的，方便问一下您的预算范围吗？
这样我好给您推荐最适合的课程方案。
不过我建议您可以先带孩子来体验一下，感受好了再聊具体费用，您觉得呢？"
```

## 二、微信转化话术

### 加微信后第一时间

```
"您好！我是XX培训机构的XX老师，感谢您关注我们！
孩子多大了呀？我好给您推荐适合的课程 😊

我们本周有免费的体验课，名额有限，您想预约哪个时间段？
① 周六上午10点
② 周六下午3点
③ 周日上午10点
"
```

### 朋友圈互动（不刷屏）

```
内容比例建议：
- 40%：学员作品/进步展示
- 20%：课堂日常/活动照片
- 20%：教育干货/家长指南
- 20%：优惠活动/体验邀约

发圈时间：
- 早7:30-8:00（家长送孩子上学前）
- 中午12:00-13:00（午休时间刷手机）
- 晚20:00-21:00（家长陪孩子时）
```

### 私聊跟进

**家长没回复时（3天后）**：
```
"XX妈妈，上次的体验课您带孩子来过吗？
我们新一期的体验课这周末开始，如果时间不合适，
也可以预约平时放学后的时段 😊
"
```

## 三、地推话术

### 标准话术（3步法）

```
第1步：观察 + 问候
→ "您好，放学啦！"（等孩子回应，拉近距离）

第2步：价值钩子
→ "我们是附近新开的XX培训机构，
  正在做免费体验活动，孩子多大啦？"

第3步：邀约体验
→ "要不要带孩子来上一节体验课？
  完全免费，还有小礼物送 ~"
```

### 家长直接拒绝

```
"没关系，不用有压力。
您可以加个微信了解一下，
什么时候想体验了随时找我，不收费的 😊"
```

### 家长驻足询问

```
→ 展示手机里的学员作品/课堂照片
→ "这是我们学员的作品，都是零基础开始学的，
  孩子特别喜欢，上课特别认真"

→ 介绍体验课：
"体验课是1小时，孩子能实际参与，
老师会给专业评估，看看孩子有没有天赋或者兴趣"

→ 现场约体验：
"您今天方便带孩子过去吗？
就在前面XX米，走2分钟"
```

## 四、公开课/讲座邀约话术

### 邀约信息模板

```
"【XX家长您好】

我们本周六下午3点有一个[主题]家长讲座/公开课，
由[主讲人]分享[核心内容]。

🎁 报名到场可获得：
1. [奖品1]
2. [奖品2]

📍 地点：XX机构（XX路XX号）
⏰ 时间：XX月XX日（周六）下午3点
👥 名额：限XX组家庭

报名请回复"参加"，我帮您占位 🙋‍♀️"
```

### 讲座现场转化

```
开场破冰：
"今天在座的家长，有没有孩子出现XX问题的？"
（列举常见痛点，让家长产生共鸣）

痛点共鸣：
"这个问题很普遍，很多家长都有同样的困惑。
今天我会分享一个方法，大家可以试试。"

解决方案：
"那针对这个问题，我们建议……
[植入机构课程/服务作为解决方案]"

现场促单：
"今天到场的朋友，
可以享受[限时优惠]，
现场报名再减XX元，
名额有限，我先帮孩子登记一个？"
```

## 五、体验课后转化

### 课后反馈话术

```
告知孩子表现：
"XX妈妈/爸爸，今天XX的课堂表现很棒！
[具体优点1]……[具体优点2]……"

肯定家长选择：
"看得出来孩子特别有[天赋/兴趣]，这个年龄段正是培养的好时机。"

提出建议：
"根据今天的情况，我建议……
[推荐课程/班型]"

优惠促单：
"现在报名可以享受[优惠]，而且XX班名额只剩X个了，
您看要不要今天就定下来？"
```

### 家长说要再想想

```
"完全理解，您可以再考虑一下。
不过我想跟您说一下我们的[优惠截止时间]，
如果在那之前决定，可以[优惠内容]。
我先把报名链接发您，不着急决定，先占个优惠名额 ~
"
```
