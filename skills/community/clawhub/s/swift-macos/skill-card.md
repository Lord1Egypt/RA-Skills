## Known Risks and Mitigations: <br>
Risk: Generated code or commands may request sensitive macOS permissions or use APIs for screen recording, audio capture, microphone access, login items, agents, daemons, or process monitoring. <br>
Mitigation: Review proposed code and commands before running them, grant only the required macOS permissions, and test permission-sensitive behavior in a controlled environment. <br>
Risk: Distribution workflows may involve Apple credentials, code signing identities, notarization passwords, purchases, or other sensitive account material. <br>
Mitigation: Use secure credential storage such as Keychain references, avoid pasting secrets into prompts or scripts, and confirm signing and notarization commands before execution. <br>
Risk: Installer or setup examples may execute remote commands or install development tools. <br>
Mitigation: Verify source URLs, prefer pinned versions where possible, and inspect scripts before executing them on a development machine. <br>
