# 苯二氮䓬类药物分类与临床应用

## 使用前提
- 明确诊断为失眠、焦虑障碍或癫痫持续状态
- 已评估患者肝肾功能
- 已排除药物禁忌症（如劳拉西泮禁用于急性闭角型青光眼）

## 药物分类依据
按消除半衰期（t₁/₂）及活性代谢物存在与否分为：
- **短效类**（t₁/₂ < 6h）：适用于入睡困难，但依赖风险高
- **中效类**（t₁/₂ 6–24h）：适用于夜间易醒或早醒
- **长效类**（t₁/₂ > 24h）：适用于维持睡眠，但可能有次日后遗效应

## 临床应用场景与药物选择
1. **抗焦虑**：优先选择劳拉西泮（抗焦虑作用强）、地西泮（小剂量有效）或艾司唑仑
2. **催眠**：
   - 入睡困难：氟西泮
   - 高效且无后遗效应：艾司唑仑
   - 快速诱导（慎用）：三唑仑（依赖风险高）
3. **癫痫持续状态/抗惊厥**：静脉注射地西泮为首选；氯硝西泮适用于小发作和肌阵挛
4. **肌肉松弛**：地西泮或氯硝西泮

## 特殊人群用药调整
- **老年或肝病患者**：优选奥沙西泮（不经CYP酶代谢，消除不受肝病影响）
- **需快速起效**：静脉用地西泮

## 安全注意事项
- 禁止与乙醇或其他中枢抑制药合用
- 长期使用必须逐渐减量停药，以防戒断反应（如反跳性失眠、惊厥）
- 过量中毒可使用氟马西尼（flumazenil）解救

## 输出内容
推荐具体药物名称、剂量范围、给药途径及疗程建议，基于适应症、患者年龄/肝功能状态和所需作用持续时间。
