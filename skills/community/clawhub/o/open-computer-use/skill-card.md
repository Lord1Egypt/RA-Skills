## Description: <br>
Guidance for installing, configuring, and operating Open Computer Use, an open-source Computer Use MCP service for macOS, Linux, and Windows. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ifuryst](https://clawhub.ai/user/ifuryst) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill to install and configure Open Computer Use as an MCP service, then operate and troubleshoot desktop-control tools from compatible MCP clients across macOS, Linux, and Windows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Open Computer Use can let a trusted agent operate the user's real desktop session. <br>
Mitigation: Install it only for trusted MCP clients, verify the npm package and project repository before use, and require human approval for actions that send, delete, purchase, install, change settings, or expose sensitive information. <br>
