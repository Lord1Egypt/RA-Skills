# C7 客户与KOL运营 — 客户伙伴

建立并维护统一的360度客户/KOL视图。为医学部提供科学拜访支持，为市场部提供合作策划建议，为推广部管理活动邀约与关系维护日程，基于互动数据预警客户流失风险。
