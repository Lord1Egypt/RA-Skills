## Description: <br>
Manage Alibaba Cloud AIRec (Airec) via OpenAPI/SDK for recommendation-engine resource operations in Alibaba Cloud, including list/create/update flows, status inspection, and troubleshooting AIRec configuration or runtime issues. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and cloud operators use this skill to discover Alibaba Cloud AIRec OpenAPI metadata, plan API calls, manage AIRec resources, and collect reproducible output artifacts. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can guide create, update, modify, and set operations against Alibaba Cloud AIRec resources. <br>
Mitigation: Require explicit confirmation before mutating operations and verify region, resource identifiers, and desired changes first. <br>
Risk: The skill depends on Alibaba Cloud credentials and may generate API response artifacts. <br>
Mitigation: Use least-privilege RAM credentials and review generated files before sharing or committing them. <br>
