# 前置胎盘的分类标准

## 何时使用
- 孕妇妊娠≥28周；
- 已完成阴道超声检查，确认胎盘附着于子宫下段且下缘毗邻或覆盖宫颈内口；
- 需要明确前置胎盘的具体类型以制定临床管理方案。

## 执行步骤
1. **确认适用条件**：确保孕周≥28周，且已排除胎盘早剥或其他非前置胎盘引起的出血。
2. **评估胎盘下缘与宫颈内口的关系**：基于处理前最后一次阴道超声影像结果进行判断。
3. **按以下标准分类**：
   - **完全性前置胎盘（中央性）**：胎盘组织完全覆盖宫颈内口；
   - **部分性前置胎盘**：胎盘组织覆盖部分宫颈内口；
   - **边缘性前置胎盘**：胎盘下缘达到但未超越宫颈内口；
   - **低置胎盘**：胎盘附着于子宫下段，边缘距宫颈内口＜2 cm。
4. **记录并应用结果**：将分类结果用于分娩方式选择（如剖宫产指征）和围产期管理。

## 注意事项
- 子宫下段随孕周延长而伸展，宫颈管消失或宫口扩张可能改变胎盘相对位置，因此必须依据临近分娩前的最终影像学检查结果分类；
- 妊娠中期（<28周）发现的胎盘位置低应称为“胎盘前置状态”，不可直接诊断为前置胎盘。
