# 错误码速查表

## 概述

本文档汇总 DataNexus SDK 上报时可能遇到的错误码（返回码），帮助快速定位和解决问题。

## 一、Marketing API 通用返回码

> SDK 上报底层走 Marketing API 接口，以下为接口级别的通用返回码。

| 返回码 | 含义 | 排查建议 |
|-------|------|---------|
| 0 | 成功 | 无需处理 |
| 20001 | 请求的操作已成功执行（部分成功） | 检查返回详情中哪些操作未成功 |
| 20003 | 参数值无效 | 检查 `user_action_set_id`、`secret_key` 等参数是否正确 |
| 20007 | 缺少必填参数 | 检查请求参数完整性 |
| 20008 | 参数格式错误 | 检查参数类型和格式 |

## 二、SDK 特有错误码

### 51000 — 数据无效（data not valid）

**常见子类型及解决方案：**

| 子信息 | 原因 | 解决方案 |
|--------|------|---------|
| `Action Set Not Exist` | 数据源 ID 不存在 | 确认 `user_action_set_id` 填写正确 |
| `SecretKey Error` | 密钥不匹配 | 确认 `secret_key` 与 `user_action_set_id` 配对 |
| `data not valid`（无具体子信息） | SDK 代码被混淆 | 将 SDK 文件排除在混淆/压缩范围之外 |
| 模拟器环境触发 | 在模拟器上运行 | 使用真机测试 |

### 其他常见错误

| 错误信息 | 原因 | 解决方案 |
|---------|------|---------|
| `Please do not repeatedly initialize SDK` | 同一 `user_action_set_id` 创建了多个实例 | 确保每个数据源 ID 只创建一个 SDK 实例 |
| `network error` / 网络请求失败 | 网络不通或域名未配置 | 检查安全域名配置和网络连接 |
| `appid mismatch` | AppID 不匹配 | 确认初始化时的 `appid` 是当前应用的 AppID |

## 三、返回码快速排查流程

```
收到错误码
│
├── code = 0 → 上报成功，无需处理
│
├── code = 51000
│   ├── Action Set Not Exist → 检查数据源 ID
│   ├── SecretKey Error → 检查密钥
│   ├── data not valid → 检查 SDK 是否被混淆 / 是否用了模拟器
│   └── 其他 → 联系 SDK 小助手
│
├── code = 20003 → 参数值无效，检查上报参数
│
├── code = 20007 → 缺少必填参数
│
├── code = 20008 → 参数格式错误
│
└── 其他未知错误码 → 联系 SDK 小助手
```

## 四、各端特有错误

### Android 端

| 现象 | 原因 | 解决方案 |
|------|------|---------|
| Logcat 无 SDK 日志 | 未初始化或初始化失败 | 检查 init + start 调用 |
| `LogAction failed` | 上报失败 | 查看失败日志详情 |

### iOS 端

| 现象 | 原因 | 解决方案 |
|------|------|---------|
| 升级 v2.1.0 后无数据 | 未调用 `[GDTAction start]` | 在 init 后添加 start 调用 |
| Xcode 控制台无 SDK 日志 | 未正确初始化 | 检查 init + start |

### 鸿蒙端

| 现象 | 原因 | 解决方案 |
|------|------|---------|
| DevEco 无 SDK 日志 | 未开启 `show_log` 或未调用 start | 设置 `show_log: true` 并调用 start |
| 网络请求失败 | 权限未声明 | 检查 `module.json5` 中的 `INTERNET` 和 `GET_NETWORK_INFO` 权限 |

### 小程序 / 小游戏端

| 现象 | 原因 | 解决方案 |
|------|------|---------|
| 无网络请求 | 未设置 openid/unionid | 调用 `setOpenId()` 或 `setUnionId()` |
| 安全域名报错 | 域名未配置 | 添加 `https://api.datanexus.qq.com` 到 request 合法域名 |
| `data not valid` | SDK 被构建工具混淆 | 将 SDK 排除在混淆范围外 |
