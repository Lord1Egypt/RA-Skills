# 输出模板

使用此模板生成回测报告。

---

## 报告头部

```markdown
# Polymarket 推荐回测报告

**报告时间：** {YYYY-MM-DD HH:MM:SS} (UTC)
**数据区间：** {最早推荐日期} — {今天}
**推荐总数：** {总数} | 已结算：{settled} | 未结算：{open} | 未匹配：{unmatched}

> {1-2 句话总结整体表现}
```

## 仓位明细

```markdown
---

## 仓位明细

| # | 日期 | 市场 | 链接 | 方向 | 入场价 | 现价 | 仓位 | 盈亏 | 盈亏% | 持仓天数 | 状态 |
|---|------|------|------|------|--------|------|------|------|-------|---------|------|
| 1 | {日期} | {市场名称} | [链接](https://polymarket.com/event/{slug}) | {Buy Yes/No} | {入场价}% | {现价}% | ${仓位} | ${盈亏} | {盈亏%}% | {天数} | {open/won/lost} |
| ... | | | | | | | | | | |

**说明：**
- 入场价和现价以百分比显示（如 29.0% = 每份 $0.29）
- 盈亏 = (现价 - 入场价) × 份数；份数 = 仓位 / 入场价
- 正盈亏用 + 前缀，负盈亏用 - 前缀
- 已结算仓位：退出价显示为 100%（赢）或 0%（输）
```

## 格式规范

1. 所有概率以百分比显示（如 35%，而非 0.35）
2. 所有金额以 $ 为前缀，千位分隔（如 $5,000）
3. 盈亏值：正值用 + 前缀，负值无前缀
4. 百分比保留 1 位小数（如 +12.5%，-3.2%）
5. 金额保留 2 位小数（如 $1,234.56）
6. 所有时间戳使用 UTC 时区，格式：YYYY-MM-DD HH:MM:SS
7. 仓位明细表按下注日期从早到晚排列（时间顺序）
8. 数量为 0 的类别从分类分析表中省略
9. 未匹配仓位单独列出并说明原因
