## 三、宝石疗法

### 九大行星宝石对照表

| 行星 | 主宝石 | 佩戴手指 | 金属 | 佩戴日 |
|------|--------|----------|------|--------|
| 太阳 | 红宝石 | 无名指 | 金 | 周日 |
| 月亮 | 珍珠 | 小指 | 银 | 周一 |
| 火星 | 红珊瑚 | 无名指 | 金/铜 | 周二 |
| 水星 | 祖母绿 | 小指 | 金 | 周三 |
| 木星 | 黄蓝宝石 | 食指 | 金 | 周四 |
| 金星 | 钻石 | 中指 | 铂金/银 | 周五 |
| 土星 | 蓝宝石 | 中指 | 银/铁 | 周六 |
| 罗睺 | 茶晶 | 中指 | 银 | 周六 |
| 计都 | 猫眼石 | 小指 | 银 | 周二 |

### 关键原则
- **只佩戴吉星宝石**：宝石会增强行星能量，凶星宝石可能放大负面效果
- **蓝宝石和茶晶需试戴**：这两颗宝石特别敏感，需先试戴观察反应
- **宝石需贴肤佩戴**：重量通常2-5克拉，需经过净化仪式

## 四、Mantra（咒语）疗法

### 各行星对应咒语

| 行星 | 咒语 | 功效 |
|------|------|------|
| **太阳** | Om Hram Hreem Hroum Sah Suryaya Namah | 自信、活力、权威 |
| **月亮** | Om Chandraya Namah | 平和、情绪平衡 |
| **火星** | Om Mangalaya Namah | 勇气、控制愤怒 |
| **水星** | Om Budhaya Namah | 语言、商业、智慧 |
| **木星** | Om Gurave Namah | 智慧、扩展、祝福 |
| **金星** | Om Shukraya Namah | 爱情、奢华、和谐 |
| **土星** | Om Sham Shanicharaya Namah | 耐心、业力释放 |
| **罗睺** | Om Rahave Namah | 克服幻象、抱负 |
| **计都** | Om Ketave Namah | 超脱、灵性成长 |

### 三种咒语层级

| 类型 | 说明 | 示例 |
|------|------|------|
| **种子咒** | 最浓缩的形式，单音节蕴含整个行星能量 | 太阳：Om Hraam Hreem Hraum Sah Suryaya Namah |
| **吠陀咒语** | 较长的经典祈请文，用于特定行星祭祀 | 用于正式仪式中 |
| **Gayatri变体** | 按Gayatri格律，祈求行星神灵的启迪 | 祈求光明与恩典 |

### 修行要求
- **日常修持**：每日108遍，使用相应材质的念珠
- **完整修行**：需完成特定总数（如太阳7,000遍、土星23,000遍）
- **念珠材质**：土星/火星/罗睺/计都用菩提子；木星/金星用图尔西；太阳/月亮用珊瑚；水星用水晶

## 六、Yantra（几何曼陀罗）

Yantra是刻在金属板上的神圣几何图案，作为行星能量的聚焦点。

### 特点
- **数值结构**：每个行星Yantra基于特定的数字矩阵（如土星用3×3九宫格）
- **材质对应**：太阳用金/铜、月亮用银、土星用铁/钢等
- **开光仪式**：需通过咒语诵念108或1,008次进行能量激活
- **最强大者**：Sri Yantra（吉祥曼陀罗）用于整体繁荣与灵性提升

## 七、Fasting（断食/Vrat）

断食是最易行的补救方式，无需物质花费，通过身心净化调和行星能量。

### 各行星断食日

| 日子 | 行星 | 断食方式 |
|------|------|----------|
| 周日 | 太阳 | 不吃盐，日落前用一餐 |
| 周一 | 月亮 | 只吃白色食物或全断 |
| 周二 | 火星 | 不吃盐和小麦，日落后进食 |
| 周三 | 水星 | 只吃绿色食物 |
| 周四 | 木星 | 不吃盐，吃黄色食物、香蕉 |
| 周五 | 金星 | 不吃酸味食物，吃白色食物 |
| 周六 | 土星 | 最严格，不吃盐，日落后吃黑芝麻和豆类 |

**最佳时机**：在相关行星的大运或次运期间，或困难行运期间执行，通常需连续11-21周。

## 十、补救措施选择原则

根据行星状态选择合适方法：

| 行星状态 | 推荐补救措施 |
|----------|--------------|
| **强势行星** | 咒语诵念或佩戴宝石（增强益处） |
| **弱势或受克行星** | 平安火祭 + 布施 |
| **中性行星** | 冥想与斋戒维持平衡 |
