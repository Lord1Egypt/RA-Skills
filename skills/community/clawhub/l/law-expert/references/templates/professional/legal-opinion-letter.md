# 法律意见书

## 适用场景
律师事务所或执业律师接受委托人的委托，就特定法律事实、法律行为或法律文件，依据中国法律法规进行审查分析后，出具的书面专业法律意见。常用于：投融资/并购的尽职调查、合同审查、合规分析、重大交易的合法性论证、诉讼/仲裁的可行性评估等。

## 提交机构
委托人（一般为公司/机构法务部门或决策管理层），非提交法院或仲裁机构。

---

## 模板正文

```
                         法 律 意 见 书

                                                     [年份]律意字第[  ]号

致：[委托人全称]

    本所接受贵方委托，就 [委托事项简述，例如"贵司与XX公司关于XX项目的《合作协议》的法律效力及履行风险"] 事宜，依据中华人民共和国现行有效法律、行政法规及司法解释，出具本法律意见书。

    为出具本法律意见书，本所律师审阅了以下文件材料（下称"审阅材料"）：
    1. 《[文件名称1]》（编号/日期）；
    2. 《[文件名称2]》（编号/日期）；
    3. [其他材料，逐项列明]。

    贵方已向本所承诺和保证：贵方提供的全部审阅材料均为真实、完整、有效，复印件与原件一致，且不存在任何虚假记载、误导性陈述或重大遗漏。

    【注：以上为引言部分，简要说明委托关系、审查范围和出具依据】

    基于上述审阅材料，本所根据《中华人民共和国民法典》《XX法》[相关法律法规] 等规定，出具法律意见如下：

    一、 基本事实情况

    [以时间线或分类方式客观描述经审阅材料确认的案件事实/交易背景。注意：仅陈述经材料证实的事实，不作主观判断。]

    二、 法律分析

    （一）关于 [争议焦点一] 的法律分析

    [结合具体法条（注明法律名称+条款序号+条款内容）、司法解释和司法实践进行分析。引用法条格式示例：根据《中华人民共和国民法典》第XX条规定："..."（引用原文）。]

    本所认为，[给出分析结论]。

    （二）关于 [争议焦点二] 的法律分析

    [同上格式]
    ...

    三、 法律风险评估

    [针对各分析点，分层次评估法律风险：]
    1. 高风险：[明确风险点和后果]
    2. 中风险：[明确风险点和后果]
    3. 低风险：[明确风险点和后果]

    四、 结论与建议

    （一）总体结论
    [用简明语言总结核心法律判断，如：该合同合法有效/存在XX方面的瑕疵/贵司在该争议中处于有利/不利地位等]

    （二）具体建议
    1. [第一项建议，需具体可操作]
    2. [第二项建议]
    3. ...

    五、 声明与保留

    1. 本法律意见书仅依据出具日前已公布并生效的中国大陆法律、法规及司法解释作出，不构成对任何事实或法律问题的担保。
    2. 本法律意见书基于贵方提供的审阅材料出具，如存在未披露的材料或新发生的事实，本所保留修正意见的权利。
    3. 本法律意见书仅供贵方内部决策参考使用，未经本所事先书面同意，不得向第三方披露，亦不得用于其他目的。
    4. 本法律意见书不构成对诉讼/仲裁结果的任何预测或承诺。

    [此页以下无正文，为签署页]

    （签署页）

    [律师事务所全称]（盖章）


    经办律师：________________（签名+律师执业证号）

    经办律师：________________（签名+律师执业证号）


    出具日期：________年____月____日

    联系方式：[律所地址/电话/邮箱]
```

## 注意事项

### 关键填写点
1. **文号**（律意字第XX号）：律所内部统一编号，便于归档管理
2. **引言部分**：务必写明审查范围——只对列明的材料和法律问题负责；超出范围的默认不予置评
3. **事实部分**：只写客观事实，不要夹带分析意见或主观推断
4. **法条引用**：必须精确到条/款/项，引用原文时应加引号
5. **声明与保留**：这是律师免责的关键段落，不可省略。重点声明：
   - 仅依据现有材料
   - 不构成担保
   - 未经同意不得向第三方披露
6. **两名律师签名**：根据《律师法》及律所内部规定，重要法律意见书通常需两名执业律师签署

### 常见错误
- 将"法律意见"写成"法律判决预测"——避免说"此案必然胜诉"，应说"贵司在此争议中有充分的法律依据支持其主张"
- 事实叙述超出审阅材料范围——不要猜测或主观推断
- 漏写声明与保留条款——没有声明等于没有保护
- 法条引用笼统——"根据相关法律规定"是不规范的，必须写明具体条款

### 份数/格式要求
- 律所存档1份（原件）
- 委托人至少1份（一般提供盖章原件）
- 如需多份原件，在签署时一并完成
- 格式：A4纸，标题黑体/加粗，正文宋体，字号小四，行间距1.5倍
- 多页文件应加盖骑缝章
