## Description: <br>
Access and manage a persistent knowledge graph for storing, retrieving, linking, and summarizing facts, decisions, goals, and observations across sessions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Atmsamma](https://clawhub.ai/user/Atmsamma) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Agents and their users use this skill to preserve project context across sessions, recall prior decisions or facts, and connect related memory nodes while working. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The security review warns that the skill can persist sensitive information in searchable long-term memory. <br>
Mitigation: Do not store passwords, API keys, tokens, session cookies, private keys, or other secrets; inspect, scope, and delete stored memory as needed. <br>
