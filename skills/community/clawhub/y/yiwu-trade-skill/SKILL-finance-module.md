### 1.3 成本计算工具

```
【外贸成本计算器】

输入参数：
├── 商品单价：$____ /件
├── 订单数量：____件
├── 包装方式：□彩盒 □裸装 □其他
├── 发货方式：□海运拼箱 □海运整柜 □空运 □陆运
├── 目的港：____________
└── 报价方式：□FOB □CIF □DDP

计算公式：
FOB成本 = 商品成本 + 包装 + 国内运输 + 装卸 + 验货
CIF成本 = FOB成本 + 海运费 + 附加费 + 保险
DDP成本 = CIF成本 + 关税 + 增值税 + 清关费 + 送货费

报价 = 成本 ÷ (1 - 目标毛利率)
```

```

├── 商品成本

## 三、出口退税流程

### 3.1 退税基本概念

```
【出口退税基础知识】

定义：退还出口商品在国内生产和流通环节已缴纳的增值税和消费税

核心原理：
├── 增值税：价外税，可抵扣
├── 出口零税率：出口商品适用0%税率
└── 退税：退还进项税额

计算公式：
退税额 = 出口额 × 退税率 - 进项税额

影响因素：
├── 退税率：不同商品退税率不同（0%-13%）
├── 进项发票：是否有增值税专用发票
└── 商品分类：海关商品编码对应的退税率
```

### 3.3 1039免税政策详解

```
【1039市场采购贸易免税政策】

政策依据：
├── 商务部等8部门《关于加快市场采购贸易方式试点工作的函》
├── 海关总署公告2014年第54号
└── 市场采购贸易方式适用范围

适用主体：
├── 义乌市场采购贸易经营者（个体工商户/企业）
├── 在义乌国际商贸城采购的出口商
└── 委托市场采购贸易经营者代理出口的企业

免税条件：
├── 出口商品在市场采购贸易试点范围内
├── 单票货值≤$50,000
├── 贸易真实，单证齐全
└── 通过市场采购贸易综合管理系统申报

免税范围：
├── 免征增值税
├── 不适用消费税（义乌以小商品为主）
└── 企业所得税：核定征收（通常0.5%-1%）

注意事项：
├── 不得虚假申报
├── 不得低报货值
├── 单证留存3年
└── 接受海关抽查
```

### 3.4 退税流程SOP（0110模式）

```
【一般贸易退税SOP】

Step 1: 退税资格确认
├── 确认商品退税率
├── 确认进项发票合规
├── 确认申报时限（90天内）
└── 准备申报材料

Step 2: 进项发票管理
├── 索取增值税专用发票
├── 核对发票信息准确
├── 在税务系统认证
└── 留存发票原件

Step 3: 出口申报
├── 在电子口岸系统录入
├── 选择正确的贸易方式
├── 准确申报商品编码
└── 提交出口货物明细

Step 4: 收齐单证
├── 出口货物报关单
├── 增值税专用发票
├── 出口发票
├── 收汇凭证
└── 其他证明材料

Step 5: 退税申报
├── 登录电子税务局
├── 录入退税申报数据
├── 上传电子单证
└── 提交申报

Step 6: 税局审核
├── 税局审核通常15-20个工作日
├── 可能实地核查
├── 审核通过后退税到账
└── 如有问题需补充材料
```

### 4.2 税务合规检查清单

```
【税务合规自查清单】

增值税合规：
├── □ 是否按规定申报纳税
├── □ 发票是否合规管理
├── □ 进项税额是否正确抵扣
├── □ 出口退税是否符合条件
└── □ 1039模式是否规范操作

企业所得税合规：
├── □ 成本费用是否真实合法
├── □ 收入确认是否及时准确
├── □ 税前扣除是否合规
├── □ 税收优惠是否正确享受
└── □ 年度汇算清缴是否完成

个人所得税合规：
├── □ 员工工资是否代扣代缴
├── □ 股东分红是否代扣代缴
├── □ 偶得收入是否代扣代缴
└── □ 年度汇算清缴是否完成

其他合规：
├── □ 税务登记是否有效
├── □ 账簿凭证是否完整
├── □ 银行账户是否合规使用
└── □ 海关年报是否按时申报
```

### 4.3 常见税务风险与防控

```
【税务风险防控】

风险1：发票不规范
├── 表现：进项发票不合规、虚假发票
├── 后果：无法退税、罚款
└── 防控：严格审核发票、留存备查

风险2：收入确认延迟
├── 表现：出口收入未及时确认
├── 后果：税务调整、补缴滞纳金
└── 防控：建立收入确认流程

风险3：成本费用不实
├── 表现：成本与收入不匹配
├── 后果：所得税调整
└── 防控：规范成本核算、保存凭证

风险4：1039模式滥用
├── 表现：虚构贸易、虚假申报
├── 后果：补税、罚款、刑事责任
└── 防控：真实贸易、如实申报
```

│  毛利率 = 毛利润 ÷ 营业收入 × 100%                                        │
│                                                                             │
│  减：期间费用                                                             │
│  ├── 销售费用（平台费、广告、样品）                                       │
│  ├── 管理费用（人工、办公、差旅）                                          │
│  └── 财务费用（银行手续费、利息）                                          │
│                                                                             │
│  = 营业利润                                                              │
│  营业利润率 = 营业利润 ÷ 营业收入 × 100%                                   │
│                                                                             │
│  加：营业外收入                                                           │
│  减：营业外支出                                                           │
│                                                                             │
│  = 利润总额                                                              │
│  净利润率 = 净利润 ÷ 营业收入 × 100%                                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

关键指标：
├── 毛利率：反映商品竞争力（义乌小商品建议>15%）
├── 营业利润率：反映运营效率（建议>8%）
└── 净利润率：反映最终盈利（建议>5%）
```

## 六、模块总结

### 6.1 核心公式速查

| 类型 | 公式 | 说明 |
|------|------|------|
| FOB报价 | = 成本 ÷ (1 - 毛利率) | 出厂价报价 |
| CIF报价 | = (FOB成本 + 运费 + 保险) ÷ (1 - 毛利率) | 到岸价报价 |
| DDP报价 | = (CIF成本 + 关税 + 增值税) ÷ (1 - 毛利率) | 门到门报价 |
| 毛利润 | = 收入 - 成本 | 未减费用前 |
| 毛利率 | = 毛利润 ÷ 收入 × 100% | 盈利能力 |
| 净利润 | = 毛利润 - 费用 - 税务 | 最终盈利 |
| 净利率 | = 净利润 ÷ 收入 × 100% | 最终盈利水平 |
| 汇率损失 | = 订单金额 × (预期汇率 - 实际汇率) | 汇率波动损失 |
