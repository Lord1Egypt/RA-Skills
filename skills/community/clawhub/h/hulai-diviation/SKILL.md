# 胡来给你算一卦

本skill提供趣味性的卜卦服务，涵盖三大卦象体系：

## 卦象体系

### 1. 八卦主卦象（基础模式）
八卦：乾、坤、震、巽、坎、离、艮、兑
运势等级：大吉、吉、平、半吉半凶、凶

**使用方法**：当用户询问"今日运势"、"算一卦"、"卜一卦"等基础问题时，随机从八卦中抽取一卦，再从对应运势等级中随机选取一条解读。

**运势解读优先级**：
- 乾卦 → 刚健有力，适合主动出击
- 坤卦 → 柔顺包容，适合静待时机
- 震卦 → 震动突破，适合变革创新
- 巽卦 → 随风入微，适合社交沟通
- 坎卦 → 险中有机，适合谨慎行事
- 离卦 → 光明照耀，适合展示自我
- 艮卦 → 稳重止息，适合沉淀积累
- 兑卦 → 喜悦和洽，适合社交联谊

### 2. 场景化卦象（专项模式）
当用户询问特定场景时，使用对应的场景卦象：
- 外卖相关 → 外卖超时卦
- 快递相关 → 快递失踪卦
- 周报相关 → 周报憋不出卦
- 摸鱼相关 → 周五摸鱼卦
- 脱单相关 → 脱单卦
- 考试相关 → 考试必过卦
- 加班相关 → 加班预警卦
- 减肥相关 → 减肥玄学卦

### 3. 自定义卦象（自由模式）
当用户提出自定义问题且不属于上述场景时，从自定义卦象库中随机选取一条进行解读。

## 解读流程

1. **识别模式**：根据用户问题关键词判断使用哪种模式
2. **抽取卦象**：在对应数据源中随机选取一条解读
3. **格式呈现**：按照上述响应格式返回结果
4. **添加互动**：必须展示场景卦一览表和可点击的触发示例
