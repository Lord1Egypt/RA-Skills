# 肝脏功能检测项目选择原则

## 适用场景
- 健康体检
- 疑似无黄疸性肝病（急性或慢性）
- 黄疸的诊断与鉴别
- 怀疑原发性肝癌
- 怀疑肝脏纤维化或肝硬化
- 肝病治疗后的疗效判断及病情随访

## 执行步骤

1. **健康体格检查时**：选择 ALT、AST、γ-GT、A/G 比值及肝炎病毒标志物；必要时增加 ALP、STP 及血清蛋白电泳。

2. **怀疑为无黄疸性肝病时**：
   - 急性病人：查 ALT、胆汁酸、尿液尿胆原及肝炎病毒标志物；
   - 慢性肝病患者：加查 AST、ALP、γ-GT、STP、A/G 比值及血清蛋白电泳。

3. **黄疸病人的诊断与鉴别诊断时**：应查 STB、CB、尿液尿胆原与胆红素、ALP、γ-GT、胆汁酸。

4. **怀疑为原发性肝癌时**：除查一般肝功能（ALT、AST、STB、CB）外，应加查 AFP、γ-GT 及其同工酶、ALP 及其同工酶。

5. **怀疑为肝脏纤维化或肝硬化时**：以 ALT、AST、STB、A/G、蛋白电泳为筛检检查，此外应查 MAO 及 PH 等。

6. **疗效判断及病情随访时**：
   - 急性肝炎：查 ALT、AST、前白蛋白、STB、CB、尿液尿胆原及胆红素；
   - 慢性肝病：观察 ALT、AST、STB、CB、PT、血清总蛋白、A/G 比值及蛋白电泳等，必要时查 MAO、PH；
   - 原发性肝癌：应随访 AFP、γ-GT、ALP 及其同工酶等。

7. **综合判断要求**：必须结合患者临床症状、体征，并可辅以肝脏影像学、血清肝炎病毒标志物及肝癌标志物等检测技术，对肝脏功能做出“正确而全面的评价”。

## 重要前提认知
- 肝脏再生和代偿能力强，单一检查仅反映肝功能一个侧面；
- 肝功能检查正常不能排除肝脏病变；
- 血清酶学指标（如 ALT、AST）虽对肝细胞损伤灵敏度高，但缺乏特异性；
- 异常结果需考虑肝外影响因素。
