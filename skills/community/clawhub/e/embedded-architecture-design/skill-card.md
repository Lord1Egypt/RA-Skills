## Use Case: <br>
Embedded developers and engineers use this skill to review code migration from one embedded platform to another, identify platform-specific compatibility issues, and produce a migration check report. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated migration plans may be incorrect or incomplete for a real embedded system. <br>
Mitigation: Review the migration report before applying changes, then validate the migrated code through compilation, hardware testing, and functional tests. <br>
Risk: The public listing name may not fully match the current migration-check content. <br>
Mitigation: Confirm that users expect an embedded code migration check workflow before relying on the skill for architecture-design tasks. <br>
