# 💰 Income Tracker v1.0
自动化每日收入追踪与债务管理系统

## 核心功能

### 1. 每日收入记录
- 自动记录每日收入来源和金额
- 支持多币种和汇率换算
- 收入分类：主动收入、被动收入、奖金、补贴等

### 2. 债务追踪
- 记录所有债务信息（本金、利率、还款日）
- 计算每日/每月还款进度
- 预估还清日期
- 逾期提醒

### 3. 财务可视化
- 日/周/月收入趋势图
- 债务减少曲线
- 收入来源饼图
- 里程碑进度条

### 4. 自动化报告
- 每日收入摘要
- 周度财务报告
- 月度总结与预测

## 自动化流程

### 每日任务
1. 检查是否有新收入记录
2. 更新债务还款进度
3. 生成每日摘要
4. 检查逾期风险

### 周度任务
1. 汇总本周收入
2. 分析收入趋势
3. 调整还款计划（如需要）
4. 生成周报

### 月度任务
1. 生成月度财务报告
2. 计算月均收入
3. 评估债务减少进度
4. 制定下月财务目标
