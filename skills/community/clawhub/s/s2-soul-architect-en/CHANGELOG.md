# Changelog

## [3.0.0] - 2026-05-01

### Major Updates
- **S2-Visual Integration**: Output SVG plain text directly into the `SOUL.md` profile to define the physical manifestation of Embodied Agents.
- **Sample Directory Added**: Included `sample/cat_companion_S2_Visual.svg` to demonstrate how to draw a cyber-feline companion with an animated glowing switch using pure text.
- **EAM Presets Upgraded**: Added the "Brand-Specific Companion/Pet (Cyber Feline)" preset in `skill.py`.

### Changed
- Upgraded the `SOUL.md` template to include the `[VISUAL CODE BLOCK]`, empowering LLMs with direct read/write access to their own visual representation.

## [2.1.0] - 2026-05-01
- **Non-Humanoid Support**: Integrated support for AI Companions and AI Pets in the S2-EAM module.
- Added Section 3.8 "Brand Product Sync" to the Whitepaper.
