# 动脉硬化性闭塞症的踝肱指数（ABI）诊断标准

## 何时使用
- 患者年龄 > 45 岁且出现间歇性跛行、静息痛、溃疡等慢性下肢缺血症状
- 对动脉硬化性闭塞症高危人群（如糖尿病、吸烟、高血压患者）进行筛查
- 已完成四肢动脉血压测量，且无严重肢体水肿或急性感染

## 执行步骤
1. **测量血压**：
   - 使用多普勒超声分别测量双侧踝部动脉（胫后或足背动脉）收缩压
   - 测量双侧肱动脉收缩压，取较高一侧作为分母
2. **计算 ABI**：
   - ABI = 踝部动脉收缩压 / 较高侧肱动脉收缩压
3. **初步解读结果**：
   - 正常：0.9 ~ 1.3
   - < 0.9：提示动脉缺血（轻度 0.7~0.9，中度 0.4~0.7，重度 < 0.4）
4. **评估可靠性**：
   - 若患者存在糖尿病或终末期肾病等导致血管严重钙化的风险，警惕 ABI 假性升高
   - 若 ABI > 1.3 或双侧差异 > 0.15，应结合趾肱指数（TBI）或多普勒波形分析

## 注意事项
- 急性肢体缺血期不宜单独依赖 ABI
- 同侧不同节段动脉压差 > 20~30 mmHg 提示局部狭窄
- ABI 结果用于指导是否需进一步影像学检查（如 CTA、MRA）或血管介入治疗

## 参考文档

### ABI 结果详细解读与特殊情况处理

**标准阈值与临床意义**
- **正常 ABI**：0.9–1.3，表示无明显动脉狭窄
- **轻度缺血**（0.7–0.9）：通常表现为间歇性跛行，可先尝试药物和生活方式干预
- **中度缺血**（0.4–0.7）：跛行距离显著缩短，可能需血管评估
- **重度缺血**（< 0.4）：常伴静息痛、组织坏死或溃疡，需紧急血管外科会诊

**血管钙化患者的处理**
- 在糖尿病、慢性肾病或高龄患者中，动脉壁钙化可致血管不可压缩，使踝部收缩压假性升高，ABI ≥ 1.3
- 此时 ABI 不可靠，应改用趾肱指数（TBI）：
  - TBI = 趾动脉收缩压 / 肱动脉收缩压
  - 正常 TBI > 0.7；< 0.7 提示缺血
- 或结合多普勒波形分析：钙化血管常呈"单相"高阻力波形

**其他注意事项**
- 测量前患者应静坐休息至少 5 分钟
- 避免在肢体水肿、蜂窝织炎或近期手术区域测量
- 双侧 ABI 差异 > 0.15 提示单侧髂股动脉病变
- 同侧踝部不同动脉（胫后 vs 足背）压差 > 20–30 mmHg 提示远端动脉狭窄
