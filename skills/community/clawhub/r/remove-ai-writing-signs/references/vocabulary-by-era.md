# AI vocabulary by era — full lexicon

This reference maps AI-overused words and phrases to specific model eras,
based on research from Kobak et al. (2025), Juzek & Ward (2025), Geng &
Trotta (2025), and the Wikipedia:Signs of AI writing field guide.

## How to use this file

1. Identify which era the text likely comes from (or treat all as active)
2. Flag words from the relevant era tier
3. Replace using the suggestions below — but always restructure the
   sentence, don't just swap synonyms

## Era 1: 2023 to mid-2024 (GPT-4 dominant)

The golden age of "delve". Heavy puffery, metronomic prose, blatant
positivity.

### Dead giveaways (always replace)

| AI word/phrase | What it signals | Alternatives |
|----------------|-----------------|--------------|
| delve / delve into | Fake depth | examine, look at, explore, study, (or just start discussing) |
| tapestry | Abstract decoration | (remove — almost never needed) |
| testament (to) | Significance inflation | shows, reflects, demonstrates, (or drop entirely) |
| vibrant | Promotional | busy, active, lively, (or use a specific description) |
| intricate / intricacies | Fake complexity | complex, detailed, (or describe the actual complexity) |
| meticulous / meticulously | Praise without evidence | careful, thorough, detailed |
| pivotal | Significance inflation | important, key, (or explain WHY it mattered) |
| underscore (verb) | Superficial analysis | show, reveal, make clear, (or restructure) |
| interplay | Vague relationship | relationship, tension, connection |
| garner | Pretentious "get" | receive, get, earn, attract |
| landscape (abstract) | Vague context | field, area, market, situation |
| enduring | Legacy puffery | lasting, long-standing, persistent |
| bolstered | Vague improvement | strengthened, supported, increased |
| Additionally, (sentence opener) | Mechanical transition | Also, / And / Beyond that, / (just start the sentence) |
| boasts (= "has") | Promotional | has, includes, contains |

### High-density flags (replace when clustered)

| Word | Natural replacement |
|------|---------------------|
| crucial | important, necessary, critical (sparingly) |
| valuable | useful, helpful, (be specific about the value) |
| key (adjective) | main, central, important |
| showcasing | showing, displaying, demonstrating |
| emphasizing | stressing, noting, (restructure to state directly) |

### Characteristic constructions

| AI construction | Human alternative |
|-----------------|-------------------|
| serves as | is |
| stands as | is |
| marks a / represents a | is / was |
| holds the distinction of being | is |
| boasts / features | has |
| setting the stage for | leading to / before |
| in the heart of | in central / in downtown |
| rich cultural heritage | (name the specific cultural elements) |
| evolving landscape | changing field / shifting market |
| indelible mark | lasting effect |
| deeply rooted | long-standing / traditional |

## Era 2: Mid-2024 to mid-2025 (GPT-4o dominant)

"Delve" declined. The puffery became subtler but the structural patterns
persisted. More emphasis on notability assertion.

### Dead giveaways (always replace)

| AI word/phrase | Alternatives |
|----------------|--------------|
| align with | match, fit, support, complement |
| fostering | building, encouraging, supporting, creating |
| enhance | improve, strengthen, increase, (be specific) |
| highlighting | showing, noting, (restructure — often a trailing -ing clause) |
| showcasing | showing, displaying, presenting |
| underscore | (still overused, same replacements as Era 1) |

### High-density flags

| Word | Natural replacement |
|------|---------------------|
| crucial | important, essential |
| vibrant | (still overused, same as Era 1) |
| emphasizing | (still present, decline began) |
| bolstered | strengthened, supported |
| pivotal | important, significant |

### Notability-assertion phrases (new in this era)

| AI phrase | What to do |
|-----------|------------|
| "independent coverage" | Only use if citing Wikipedia policy; remove from article prose |
| "profiled in [list of outlets]" | Replace with specific claim from one source |
| "active social media presence" | Remove or replace with follower count if relevant |
| "featured in [X], [Y], and [Z]" | Pick the most relevant source and say what it said |
| "media outlets" / "major publications" | Name them or drop the claim |

## Era 3: Mid-2025 and beyond (GPT-5+)

Vocabulary became less distinctive as AI companies trained against detection.
Structural and content patterns now matter more than individual words.

### Still overused

| Word/phrase | Notes |
|-------------|-------|
| emphasizing | Persistent across all eras |
| enhance | Still overused |
| highlighting | The trailing -ing analysis survived |
| showcasing | Slightly declined but still present |

### New patterns (vocabulary is less the tell, structure is)

The main tells in this era are:
- Heavy notability assertion with named sources
- Formulaic "challenges and future" sections
- Excessive in-text source attribution for trivial facts
- Smooth, uniform sentence rhythm
- Absence of authorial opinion or doubt

## Cross-era vocabulary (overused in all periods)

These words appear across all LLM eras and remain reliable signals:

### Tier 1: Strong signals (always flag in clusters)

comprehensive, multifaceted, nuanced, paradigm, transformative, leveraging,
synergy, holistic, robust, streamline, utilize, facilitate, elucidate,
encompassing, cornerstone, reimagine, empower, harness, navigate, realm,
poised, myriad, invaluable, bustling, nestled, groundbreaking, renowned,
profound, exemplifies, seamless, catalyst, paramount, quintessential

### Tier 2: Moderate signals (flag when 4+ appear together)

furthermore, moreover, notably, accordingly, consequently, underpinning,
spearheading, trailblazing, pioneering, cutting-edge, state-of-the-art,
actionable, curated, bespoke, tailored, ecosystem (non-biological),
stakeholder, bandwidth (non-technical), scalable, innovative, impactful,
disruptive, leverage, synergize, optimize, amplify, unpack, drill down

### Overused phrases (always replace)

| Phrase | Replacement |
|--------|-------------|
| In today's digital age | (remove — start with the actual point) |
| It is worth noting that | (remove — just state the thing) |
| plays a crucial role | matters, is important for, (be specific) |
| serves as a testament | shows, proves, demonstrates |
| in the realm of | in |
| harness the power of | use |
| embark on a journey | start, begin |
| without further ado | (remove) |
| at the end of the day | ultimately, in practice |
| a diverse array of | various, many, several |
| the evolving landscape of | changes in, shifts in |
| a beacon of | (remove — use a specific description) |
| a testament to | evidence of, shows |
| the intersection of X and Y | where X meets Y / combining X and Y |
| a game-changer | (remove or be specific about what changed) |
| move the needle | change, improve, affect |
| take it to the next level | improve, advance |
| from X to Y (false range) | (restructure to list the actual items) |
