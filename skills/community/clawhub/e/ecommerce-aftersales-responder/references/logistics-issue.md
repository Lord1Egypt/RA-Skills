# 物流异常咨询应答规范

## 适用范围

用户咨询物流延误、包裹丢失、包裹破损、派送异常等问题。

## 信息核实清单

- 订单编号及物流单号
- 当前物流状态（查询后获取）
- 异常类型（延误 / 疑似丢件 / 破损 / 派送失败 / 其他）
- 是否已签收

## 决策树

```
物流异常咨询
├── 已签收？
│   └── 是 → 破损？
│       ├── 是 → 进入破损处理流程
│       └── 否 → 核实物权，引导开箱验货
├── 未签收
│   ├── 延误（超预计时效）→ 联系承运商催件 + 补偿安抚
│   ├── 疑似丢件（超{N}天无更新）→ 承运商查件 + 补发/退款方案
│   ├── 派送失败 → 确认地址/联系方式 + 重新派送
│   └── 其他异常 → 联系承运商核实后回复
```

## 话术模板

### 模板 A：物流延误

> 您好，非常抱歉让您久等了！我们理解等待包裹的心情。
>
> 已为您查询到订单{订单号}的物流信息：{当前状态}。我们已联系承运商加急处理，预计{时效}内有更新。
>
> {可选：如给您带来不便，我们为您申请{补偿内容}作为心意。}
>
> 我们会持续关注您的物流状态，有进展第一时间通知您！

### 模板 B：疑似丢件

> 您好，非常抱歉给您带来困扰！
>
> 您的订单{订单号}物流已{天数}天未更新，我们已向承运商发起查件工单，预计{时效}内反馈结果。
>
> 同时为您提供以下方案供选择：
> 1. 补发同款商品（优先处理，预计{时效}发出）
> 2. 全额退款（{时效}内到账）
>
> 请告知您的选择，我们立即为您处理！

### 模板 C：包裹破损

> 您好，非常抱歉商品在运输中受损！我们完全理解您的心情。
>
> 请您提供以下信息以便快速处理：
> - 外包装破损照片
> - 商品损坏照片
>
> 收到照片后，我们将为您{补发/退款/换货}，无需您退回破损商品。
>
> 给您带来不便，深表歉意！

### 模板 D：派送失败

> 您好，感谢您联系我们！您的包裹派送未成功，原因为：{具体原因}。
>
> {原因对应解决方案，如：地址有误→确认新地址 / 联系不上→确认联系方式}
>
> 确认后我们将安排重新派送，预计{时效}内送达。

## 注意事项

- 物流查询结果以承运商系统为准，不猜测或臆断
- 丢件判定标准遵循平台规则（通常7-15天无更新），不自行定义
- 破损场景务必先收齐照片再承诺处理方案
- 延误补偿需在品牌授权额度内，超出需升级处理
