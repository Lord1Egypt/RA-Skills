# 结核分枝杆菌分类与传播机制

## 何时使用
- 出现肺结核疑似病例需明确病原体类型
- 需评估结核病传播风险或设计感染控制措施
- 公共卫生人员开展结核病流行病学调查
- 医护人员进行患者教育或接触者筛查

## 执行步骤
1. **确认病原体类型**：结核分枝杆菌分为人型、牛型、鸟型和鼠型四种，其中人型是人类肺结核的主要致病菌。
2. **识别传播方式**：肺结核主要通过呼吸道飞沫传播。当患者咳嗽、打喷嚏、大声说话或吐痰时，会释放含菌飞沫。
3. **评估感染风险**：健康人吸入含结核分枝杆菌的飞沫后可能被感染，尤其在密闭、通风不良环境中风险更高。
4. **关联公共卫生意义**：该知识支撑“世界防治结核病日”（3月24日）的设立，并指导隔离、通风、佩戴口罩等防控措施。

> 注意：本技能仅适用于结核分枝杆菌引起的结核病，不适用于非结核性呼吸道疾病或其他分枝杆菌感染。
