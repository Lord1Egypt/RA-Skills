## Description: <br>
Skill Forge helps agents discover missing capabilities, evaluate candidate tools across skill ecosystems, package integrations, verify them, and publish or announce the results. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[stakeswky](https://clawhub.ai/user/stakeswky) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agent operators use this skill to automate the loop of scouting for useful agent capabilities, assessing compatibility and risk, generating ClawHub skill package files, and preparing release or announcement material. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill describes a broad automation pipeline that can discover, clone, package, publish, and publicly announce skills without clearly documented scoping or confirmation gates. <br>
Mitigation: Run it only in a sandbox or disposable workspace, review the implementation before executing the pipeline, and require manual approval before any ClawHub publish or public post. <br>
Risk: The documented announcement flow and dependent social skills may involve public posting and credentials. <br>
Mitigation: Use scoped or test credentials and review every generated announcement before allowing it to post. <br>
