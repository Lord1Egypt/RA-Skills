# 第八章 理智

> 但我必须努力将那晚我在那轮讪笑着的黄色月亮下所看到的一切都说出来——我蜷缩在废弃铁路坑道中的野生荆棘里，清晰地看着那群东西蜂拥蹦跳着从我前方的罗利路上穿行而过。—— H.P.洛夫克拉夫特，《印斯茅斯的阴霾》

## 1. 理智值概述

每位调查员都有**理智值（SAN）**，该值会在游戏过程中起伏。理智值用来衡量调查员是否能够承受克苏鲁神话的惊怖，以及能否勇敢地面对恐怖的情景而不转身逃走。

- **失去理智值可能导致疯狂**
- **理智值降至 0 时**，调查员将陷入**永久性的、无药可救的疯狂**之中，不再是一个玩家角色（PC）
- 理智系统旨在模拟洛氏小说主角的行为——面对超越人智的现象和恐惧实体时，往往会晕倒或发疯
- 以更多理智值开始游戏的角色更容易接受重大创伤事件，更容易压抑恐惧的记忆
- 理智值较少的角色精神更加脆弱，更容易受到不良情绪的影响

### 最大理智值

- 每位调查员的初始最大理智值为 **99**
- 了解克苏鲁神话知识后，最大理智值会下降
- **最大理智值 = 99 - 当前克苏鲁神话技能值**
- 当克苏鲁神话技能增长时，应立即在最大理智值上减去该增长值

## 2. 理智检定

当调查员遭遇精神上的危机时，守秘人需要让他们进行一次**理智检定**。

### 检定规则

- 投掷**百分骰**，结果**小于或等于当前理智值**即为**成功**
- **奖励骰与惩罚骰都不适用于理智检定**（唯一的例外是"自救"，pg.167）
- 可选规则中的"花费幸运值"可能不适用于理智检定

### 检定结果

理智损失以 `SAN X/Y` 的格式表示（如 `SAN 1/1D4+1`）：
- **斜线左边**：理智检定**成功**时的理智损失量
- **斜线右边**：理智检定**失败**时的理智损失量
- 例如 `SAN 0/1D4` 表示成功不损失理智，失败损失 1D4 理智值

### 失败的理智检定

一个失败的理智检定**总是**会让调查员有一瞬间失去对自己行为的控制力，守秘人需要让调查员做一个**不由自主的行动**：

| 行为类型 | 描述 |
|---------|------|
| 猛的一跳 | 使调查员从身上掉落什么东西（眼镜、手电、枪、书籍等） |
| 因恐惧而大叫 | 吸引别人的注意，说一些不恰当的话 |
| 不由自主的动作 | 将方向盘转向危险的一侧，在恐惧中放弃挣扎，畏缩着发抖 |
| 不由自主的战斗行为 | 在战斗轮中由守秘人决定行动（殴打别人、扣下扳机、躲到别人身后寻求掩护） |
| 呆立不动 | 惊异地盯着，却没有做出任何动作 |

### 大失败

- 大失败的理智检定会让调查员失去这次遭遇中**所能损失的最多理智值**
- 理智检定针对的是**遭遇**本身，而非每一个个体（如遇到多个食尸鬼只进行一次检定）

### 理智损失范例表

| SAN 值 | 情景 |
|--------|------|
| 0/1D2 | 发现被撕碎的动物尸体 |
| 0/1D3 | 发现尸体的一部分 |
| 0/1D4 | 看到一地流淌着的鲜血 |
| 1/1D4+1 | 发现一具血肉模糊的恐怖尸体 |
| 0/1D6 | 醒来时发现被困在棺材中 |
| 0/1D6 | 目击一个朋友因为暴力行为而死亡 |
| 0/1D6 | 看到一只食尸鬼 |
| 1/1D6+1 | 与一个你所知的已经死去的人相遇 |
| 0/1D10 | 遭受严刑拷打 |
| 1/1D10 | 看到死尸复活 |
| 2/2D10+1 | 目击从天而降的尸雨 |
| 1D10/1D100 | 直视伟大的克苏鲁 |

### 理智损失程度表现

| 损失量 | 表现特征 |
|--------|---------|
| 1-2 点 | 不安与不适，什么东西在侵蚀神智的一角——虽然难以察觉，但确实存在 |
| 3-4 点 | 厌恶感，或感觉附近有什么不对劲的，可能表现出呕吐恶心、肢体麻木等症状 |
| 5-9 点 | 恐惧、惊慌或迷失——可能足以造成临时性疯狂。表现为严重的惊吓，导致无法控制的身体动作（剧烈打颤、极度恶心等），或远离附近的人与物 |
| 10-19 点 | 足以扭曲神智的恐惧——头发变白，暴力倾向（对自己或对他人），身体与精神罢工，暂时性性情大变，胡言乱语，出现幻觉 |
| 20+ 点 | 终极的宇宙恶意——超越人类承受的极限。心智被破坏，需要花费大量时间才能恢复理智。哪怕在恢复后，也无法回到原来的样子。造成不定性疯狂 |

## 4. 疯狂的三种状态

疯狂有三种状态：**临时性**、**不定性**和**永久性**。

无论疯狂是临时性还是不定性，都将经过三个阶段：
1. **疯狂发作**（疯狂阶段1）
2. **潜在疯狂**（疯狂阶段2）
3. **恢复与治疗**（疯狂阶段3）

### 临时性疯狂

**触发条件**：一次性损失 **5 点或更多**理智值（如一次理智检定或阅读一本神话典籍）

**判定流程**：
1. 守秘人要求调查员进行一次**智力检定**
2. 如果**智力检定失败** → 调查员抑制了记忆（心智自我保护），**不会**进入疯狂状态
3. 如果**智力检定成功** → 调查员认识到所经历的一切意味着什么，进入**临时性疯狂**
4. 临时性疯狂持续 **1D10 个小时**

### 不定性疯狂

**触发条件**：在游戏中的"一天"内失去了 **1/5 或更多**理智值

- "一天"的长短通常由守秘人定夺，一般以调查员到达一个安全可供休息的场所为止
- 不定性疯狂将持续到角色被治愈或恢复为止

### 永久性疯狂

**触发条件**：理智值降为 **0**

- 调查员成为无药可救的永久性疯狂角色
- 不再是一个能由玩家操纵的角色
- 守秘人应及早计划好玩家疯狂行为的尽头将会造成什么

## 5. 疯狂发作（疯狂阶段1）

亦被称为"六亲不认"、"狂躁"、"行为失控"和"恐慌症"。

### 基本规则

- 调查员在疯狂发作时**失去对角色的控制权**，由玩家转移给守秘人
- 调查员**不会在疯狂发作时损失更多的理智值**
- 疯狂发作只持续 **10 个战斗轮**（即时症状），但总结症状时可由守秘人概括更长时间
- 每次疯狂发作时，守秘人可以**更改或增加一项调查员的背景条目**

### 可修改的背景条目

| 条目 | 修改示例 |
|------|---------|
| 形象描述 | "目光凶暴"、"眼神涣散"、"厌世"、"不修边幅" |
| 思想与信念 | 变得更加极端与异常，如"不信者必将遭受苦难" |
| 重要之人 | 增加新名字与理由，如"追逐杜尔伯特·斯密史，至死方休" |
| 意味非凡之地 | 当前所在地点变得具有重大意义，或对某地过分固着 |
| 宝贵之物 | 遗失或取得新的财产并赋予重要意义 |
| 特质 | 被抹去或修改，如"混混"、"瘾君子"、"心浮气躁" |
| 创伤与疤痕 | 通常作为重伤的后果被修改 |
| 恐惧症和躁狂症 | 通常为疯狂发作的骰点结果 |
| 神话典籍/法术/魔法物品 | 可能遗失、被摧毁、不由自主地咏唱法术或研读典籍 |
| 第三类接触 | 记录所遭遇的怪物与神灵 |

### 即时症状（表VII，投1D10）

当疯狂发作时有其他调查员在场，持续 **1D10 个战斗轮**。

| D10 | 症状 | 描述 |
|-----|------|------|
| 1 | **失忆** | 只记得最后身处的安全地点，没有任何来到这里的记忆。持续 1D10 轮 |
| 2 | **假性残疾** | 陷入心理性的失明、失聪或躯体缺失感中。持续 1D10 轮 |
| 3 | **暴力倾向** | 陷入六亲不认的暴力行为中，对周围敌人与友方进行无差别攻击。持续 1D10 轮 |
| 4 | **偏执** | 陷入严重的偏执妄想之中。所有人都想要伤害他们；没有人可以信任；正在被监视。持续 1D10 轮 |
| 5 | **人际依赖** | 将他人误认为重要之人，并据此行动。持续 1D10 轮 |
| 6 | **昏厥** | 当场昏倒，需要 1D10 轮才能苏醒 |
| 7 | **逃避行为** | 用任何手段试图逃离当前位置，即使这意味着开走唯一交通工具并抛下其他人。持续 1D10 轮 |
| 8 | **歇斯底里** | 表现出大笑、哭泣、嘶吼、害怕等极端情绪表现。持续 1D10 轮 |
| 9 | **恐惧** | 从恐惧症状表中选择一个恐惧源，即使该恐惧源并不存在，调查员也会在接下来 1D10 轮内想象它存在 |
| 10 | **躁狂** | 从躁狂症状表中选择一个躁狂诱因，在接下来 1D10 轮内，调查员会渴望沉溺于新的躁狂症中 |

### 总结症状（表VIII，投1D10）

当疯狂发作为独处或所有在场调查员同时发作时，守秘人可快进并描述结果。调查员往往会在疯狂中迷失自身并不可控数小时甚至更久。

| D10 | 症状 | 描述 |
|-----|------|------|
| 1 | **失忆** | 回过神来发现自己身处陌生的地方，并忘记了自己是谁。记忆会随时间缓缓恢复 |
| 2 | **被窃** | 1D10 小时后恢复清醒，发觉自己被盗，身体毫发无损。做幸运检定决定宝贵之物是否被盗，其他贵重品自动消失 |
| 3 | **遍体鳞伤** | 1D10 小时后恢复清醒，身上满是拳痕和瘀伤。生命值减少到疯狂前的一半（不造成重伤）。未被窃 |
| 4 | **暴力倾向** | 陷入强烈的暴力与破坏欲之中。回过神来时所做之事可能明显，可能被记住，也可能不 |
| 5 | **极端信念** | 采取极端和疯狂的表现手段展示思想信念。如信教者被发现在地铁上高声布道 |
| 6 | **重要之人** | 在 1D10 小时或更久的时间中，不顾一切地接近那个人，并按照关系做出行动 |
| 7 | **被收容** | 在精神病院病房或警察局牢房中回过神来，可能慢慢回想起导致被关在这里的事情 |
| 8 | **逃避行为** | 恢复清醒时发现自己在很远的地方，可能迷失在荒郊野岭，或在驶向远方的列车或长途汽车上 |
| 9 | **恐惧** | 患上一个新的恐惧症。从恐惧症状表掷 D100 决定。1D10 小时后回过神来，开始为避开恐惧源而采取任何措施 |
| 10 | **躁狂** | 患上一个新的躁狂症。从躁狂症状表掷 D100 决定。1D10 小时后恢复理智，完全沉浸于新的躁狂症状 |

## 6. 潜在疯狂（疯狂阶段2）

疯狂发作结束后，调查员进入疯狂的**潜伏期**。操控权转交回玩家手中。

### 关键规则

- 在潜在疯狂的精神脆弱期，**任何哪怕只有 1 点的理智值损失**都会导致调查员**再一次陷入疯狂发作**
- 心理脆弱期的持续时间：
  - **临时性疯狂**：1D10 小时
  - **不定性疯狂**：可能持续一整个模组或战役的一整章
- 处于潜在疯狂时，调查员行为可能仍然合理正常，只有在疯狂发作期间才显得精神失常

### 恐惧症与躁狂症（疯狂副作用1）

当调查员**理智正常**时，恐惧症和躁狂症只是角色扮演的小亮点。

当**陷入疯狂**时：
- 直接与恐惧症对象接触会引起**恐慌**
- 直面恐惧源的陷入躁狂的角色会产生**强迫性反应**
- 恐惧症中一般只添上一个**惩罚骰**（不作用于理智检定或现实认知检定）
- 潜在疯狂阶段，任何与躁狂症相关的接触会使调查员产生难以应对的响应，所有行为判定承受一个**额外惩罚骰**，直到充分沉浸其中或远离
- 一个成功的**精神分析检定**可以使调查员暂时摆脱恐惧症与躁狂症的影响

### 幻觉与现实的界限（疯狂副作用2）

守秘人可以随时告知处于潜在疯狂的调查员一些如幻觉般的感官讯息。调查员需要通过一次**现实认知检定**来鉴别真假。

**现实认知检定**（投理智骰）：
- **失败**：失去 1 点理智值。如果正处于潜在疯狂阶段，立刻引起疯狂发作症状。幻觉不会消失
- **成功**：看穿自身产生的幻觉，守秘人描述真实的所见所闻。获得对幻觉的**暂时抗性**，直到下一次失去理智值

### 疯狂与克苏鲁神话（疯狂副作用3）

- **非克苏鲁神话**带来的疯狂**不会**增加克苏鲁神话技能
- 每次由**克苏鲁神话相关**引起的精神创伤：
  - **第一次**由克苏鲁神话引起的疯狂：克苏鲁神话技能增长 **5 点**
  - **之后每次**由克苏鲁神话引起的疯狂：克苏鲁神话技能增长 **1 点**

## 8. 疯狂的治疗与恢复

### 从临时性疯狂恢复

- 临时性疯狂持续 **1D10 个小时**
- 调查员**夜晚在一个安全场所的良好休息**可以使临时性疯狂恢复
- 如果调查员处于精神高度集中紧张的状态（如夜晚防备可能的袭击），守秘人可以判定这**不能**使角色从疯狂中恢复

### 从不定性疯狂恢复

不定性疯狂症状每个月的治疗中，如果没有受到进一步的精神创伤，玩家可以进行一次检定。有两种选择：

#### 私人护理

最佳治疗方式——在家中或其他舒适地点，有温柔的护士照顾，无外界干扰。精神分析和精神药物可能是可获得的。

| 百分骰结果 | 效果 |
|-----------|------|
| 01-95（成功） | 因精神药物或精神分析增加 **1D3 点理智值**。之后需进行一次理智检定：成功则从疯狂中恢复；失败则下个月可再次尝试 |
| 96-100（失败） | 拒绝服用药物或接受治疗。损失 **1D6 理智值**，下个月治疗无进展 |

#### 收容所

第二好的选择——精神病院。比私人护理便宜，甚至可能是免费的国家服务，但素质参差不齐。

| 百分骰结果 | 效果 |
|-----------|------|
| 01-50*（成功） | 因精神药物或治疗增加 **1D3 点理智值**。之后需进行一次理智检定：成功则从疯狂中恢复；失败则下个月可再次尝试 |
| 51-95（无效） | 治疗没有起到任何正面或负面的作用 |
| 96-100（失败） | 拒绝服用精神药物及治疗，损失 **1D6 理智值**，下个月治疗无进展 |

> *50 是平均水平，守秘人可根据收容机构的治疗能力在 5-95 范围内调整该值。

#### 强制收容

- 州内得到许可的医师可以安排拘留调查员长达 **72 小时**
- 如果收集到严重失常的证据，患者会被关在收容所中持续较长时间
- 法院会依据收容所报告决定释放还是移送强制治疗
- 调查员有三种方式离开：说服监护人、让收容所职员相信自己恢复正常神智、或翻墙逃脱

### 恐惧症/躁狂症/强迫症/失忆症的表现

**恐惧症**：接触或回忆起恐惧事物时，出现心悸、颤抖、窒息及无法控制的焦虑情绪。角色会通过一切手段逃避。

**躁狂症**：保持恒定的亢奋与极易烦躁的情绪。症状包括多动、多话、过高自尊心、睡眠减少、容易分神、不假思索的轻率危险举动。

**强迫症**：希望影响未来的强制性动作，即使觉得并无意义仍会固执继续。在性命攸关时刻可能忽略逃生机会。

**失忆症**：回忆不起重要个人信息，可能是逃避不愉快记忆的影响。守秘人可选择将有该症状的角色的理智最大值回复到 99，并将克苏鲁神话技能归零；当逃避的恐惧事物再次出现时，记忆将会寻回。

## 9. 理智恢复

除了上述治疗方式外，还有四种能够提高调查员当前理智值的方式：

### 1. 守秘人奖励

- 在完成模组或战役的一章后，守秘人可以通过骰子来增加指定调查员的当前理智值
- 所有参与者获得相同的理智回复骰，但每位玩家各自投掷
- 奖励大小应与调查员团队所面临的风险成正比
- 如果调查员懦弱、残忍或滥杀无度，守秘人可以减少甚至取消奖励

### 2. 技能增长至 90%

- 当调查员的技能增加至 **90% 及以上**时，奖励 **2D6 理智值**
- 这是出于因为掌握了一种技能而获得的自尊心

### 3. 心理治疗（精神分析）

- 每个月可由医生或咨询师进行一次精神治疗的检定
- **检定成功**：受治疗者恢复 **1D3 点理智值**
- **检定失败**：不增加任何理智值
- **检定大失败**：患者失去 **1D6 理智值**，医生认为治疗失败，两人关系破裂，无法继续治疗

**注意**：
- 精神分析**不能**加速疯狂的恢复
- 精神分析在游戏世界中等同于心理急救
- 临时性和不定性疯狂是可以治愈的
- 精神分析对**永久性疯狂完全无效**
- 心理治疗也可用于消除恐惧症或躁狂症：每月末尾通过一次成功检定治疗，如果成功则进行一次理智检定，**两个检定都成功**则恐惧/躁狂症状被治愈并从背景中删除（此奖励代替理智值回复）

### 4. 自救

- 调查员可以在背景中选择一个条目作为**心理支柱**，并与此相关地花费一段时间
- **不包括**恐惧症、躁狂症、创伤或任何克苏鲁神话相关的条目
- 自救可以在两次冒险之间或调查员**幕间成长**时进行
- 玩家应认真讨论调查员角色在寻求治疗或救赎时的细节
- 然后进行一次理智检定：
  - **成功**：恢复 **1D6 理智值**
  - **失败**：损失 **1 点理智值**，守秘人与玩家应据此适当修改背景来对应失败（如失去信仰、家庭矛盾等）
- 每位调查员在游戏开始时拥有一个**"关键连接"**：使用关键连接进行自救时，理智检定获得一个**奖励骰**

## 10. 理智和克苏鲁神话知识

### 最大理智值的关联

- 最大理智值 = 99 - 克苏鲁神话技能值
- 克苏鲁神话技能增长时，最大理智值立即相应降低
- 这意味着了解越多克苏鲁神话知识，理智的上限就越低

### 疯狂与克苏鲁神话技能增长

- **非克苏鲁神话**带来的疯狂**不会**增加克苏鲁神话技能
- **第一次**由克苏鲁神话引起的疯狂：克苏鲁神话技能增长 **5 点**
- **之后每次**由克苏鲁神话引起的疯狂：克苏鲁神话技能增长 **1 点**

### 幻觉与理智损失的注意事项

- 足以威胁到理智的事物（如怪物）的**幻觉**与真实存在一样有可能引发理智值损失
- 面对幻觉怪物，玩家有两个选择：
  1. 认为所见事物是"真实"的，做出对应理智检定
  2. 进行一次**现实认知检定**
- 如果现实认知检定**成功**：不需要损失理智值——根本没有怪物
- 如果现实认知检定**失败**：失去 1 点理智值，并进入疯狂发作
- **疯狂发作时免疫进一步的理智值损失**，不需要为看见怪物进行理智检定
- 但如果怪物（无论幻觉或真实）在疯狂发作结束时仍然存在，调查员仍需进行适当的理智检定
