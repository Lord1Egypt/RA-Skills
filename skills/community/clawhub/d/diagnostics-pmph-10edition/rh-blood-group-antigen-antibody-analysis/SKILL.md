# Rh血型系统抗原抗体特性与临床分类

## 何时使用
- 需对已知红细胞样本进行Rh血型鉴定
- 评估输血前的Rh相容性
- 产科中筛查孕妇与胎儿Rh不相容导致的新生儿溶血病风险

## 执行步骤
1. **确认试剂**：使用抗D血清或标准Rh抗体试剂。
2. **初步凝集试验**：将红细胞样本与抗D试剂混合，观察是否发生凝集。
   - 若凝集 → 判定为Rh阳性
   - 若无凝集 → 进入第3步
3. **排除D^n亚型**：对无凝集样本进行间接抗球蛋白试验（Coombs试验）。
   - 若抗球蛋白试验呈阳性并出现凝集 → 判定为D^n型（Rh阳性变异）
   - 若仍无反应 → 判定为Rh阴性
4. **输出结果**：
   - “Rh阳性”
   - “Rh阴性”
   - “D^n型（Rh阳性变异）”

## 注意事项
- 本方法仅适用于以D抗原为核心的临床粗分型，不适用于E、C、c、e等非D抗原的精细分型。
- Rh系统天然抗体极少，绝大多数抗体由免疫刺激产生。
- D抗原是Rh系统中抗原性最强、临床意义最关键的抗原。

> 提示：若需全面Rh表型分析（如C/c/E/e分型），应调用更高级的血型分型技能。
