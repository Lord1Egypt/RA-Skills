# Exterior-Interior Concurrent Differentiation

## When to Use
- 患者同时存在表证与里证症状
- 需区分是表里同病、表里转化，还是半表半里证
- 用于指导治疗策略（如先表后里、表里双解等）

## Execution Steps

### 1. 判断是否为表里同病
- 确认患者同时具备：
  - **表证症状**：发热、恶寒、头痛、脉浮等
  - **里证症状**：腹胀、便秘、小便黄、口渴、脉沉等
- 若符合，则进一步辨识具体类型：
  - 表寒里热
  - 表热里寒
  - 表虚里实
  - 表实里虚

### 2. 若非同病，判断是否存在表里转化
- **由表入里**：原为表证，后出现里证且表证消失 → 提示病势加重
- **由里出表**：原为里证，后出现表证且里证减轻 → 提示病势好转

### 3. 分析转化条件
- **表→里**：因正气不足、邪气过盛、失治误治或护理不当
- **里→表**：多因正气恢复，驱邪外出

### 4. 排除半表半里证
- 若症状表现为寒热往来、胸胁苦满、口苦咽干、目眩、心烦喜呕、不欲饮食、脉弦，则考虑为少阳证（半表半里），不适用本规则

## Output
输出辨证结论，明确以下之一：
- 表里同病（注明具体类型）
- 由表入里
- 由里出表
- 半表半里证（仅用于排除说明）
