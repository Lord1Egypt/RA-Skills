# Smart Investor Research Workflow
# 智能投资研究工作流

## 📋 业务场景

**场景：投资经理/分析师的每日行业/标的研究**

一位投资经理每天需要研究多个投资标的，流程包括：收集公开信息 → 判断市场情绪 → 计算风险指标 → 撰写正式报告。整个过程分散在多个工具中，耗时且难以保持格式统一。

## 😣 痛点分析

1. **信息碎片化**：研报数据来自新闻、公告、社交媒体、第三方平台，分散且格式不统一
2. **情绪判断主观**：市场情绪依赖个人经验，缺乏量化指标
3. **风险计算繁琐**：手动计算 VaR、夏普比等指标费时且容易出错
4. **报告格式不统一**：最终报告格式各异，不适合直接交付客户

### Skill 协作逻辑

| 步骤 | Skill | 输入 | 输出 |
|------|-------|------|------|
| 1 | zcx-research-assistant | 投资标的名称 | SWOT / 行业简报（含信息源） |
| 2 | zcx-sentiment-analyzer | 关键词（公司名/行业词） | 情绪评分(0-100)、多空信号 |
| 3 | zcx-portfolio-risk-manager | 标的 + 模拟权重 | VaR、夏普比、最大回撤、仓位建议 |
| 4 | zcx-pdf-report-generator | Markdown汇总内容 | A4专业PDF报告 |

## 📖 使用示例

### 完整工作流示例

```
# 用户触发
帮我完成宁德时代(300750)的投资研究，输出完整 PDF 报告

# AI 自动执行：
# Step 1 → zcx-research-assistant: 收集财报/行业/竞品数据
# Step 2 → zcx-sentiment-analyzer: 分析近期新闻/研报/社交媒体情绪
# Step 3 → zcx-portfolio-risk-manager: 计算模拟10%仓位的风险指标
# Step 4 → zcx-pdf-report-generator: 生成专业PDF报告并提供下载
```

### 单独使用任一 Skill

- **只做情绪分析**：`分析一下新能源板块近期市场情绪`
- **只做风险计算**：`计算模拟组合VaR：宁德时代30%+比亚迪20%+亿纬锂能10%`
- **只生成报告**：`把这份Markdown转成PDF投资报告`

## ✅ 适用人群

- 私募/公募基金研究员
- 投资银行分析师
- 个人投资者（做基本面研究）
- 企业战略/BD 团队（行业研究）
