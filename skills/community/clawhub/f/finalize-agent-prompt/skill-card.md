## Description: <br>
Finalize prompt file using the role of an AI agent to polish the prompt for the end user. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[jhauga](https://clawhub.ai/user/jhauga) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, prompt authors, and agent operators use this skill to refine a provided prompt file for clarity, structure, grammar, and spelling while preserving its original intent and markdown structure. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Prompt revisions could accidentally change the user's intended instructions or introduce misleading guidance. <br>
Mitigation: Review the revised prompt before use and provide only prompt files intended for revision. <br>
