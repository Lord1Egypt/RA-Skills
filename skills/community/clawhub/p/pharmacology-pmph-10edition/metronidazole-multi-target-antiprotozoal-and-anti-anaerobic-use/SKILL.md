# 甲硝唑多靶点抗原虫与抗厌氧菌应用

## 何时使用
- 患者确诊为以下任一疾病：
  - 急性阿米巴痢疾（组织型阿米巴感染）
  - 阴道毛滴虫感染
  - 厌氧菌感染（如产后盆腔炎、败血症、骨髓炎）
  - 贾第鞭毛虫病
- 患者处于非妊娠早期（妊娠3个月以上）且非哺乳期
- 患者能确保在服药期间及停药后短期内严格禁酒

## 如何执行
1. **确认感染类型**：明确病原体属于阿米巴滋养体、阴道毛滴虫、厌氧菌或贾第鞭毛虫。
2. **评估禁忌证**：
   - 若为妊娠3个月内或哺乳期，禁止使用甲硝唑。
   - 若患者无法保证禁酒，不得启用该药。
3. **按感染类型确定用药策略**：
   - **阿米巴感染**：仅用于杀灭滋养体（如肝脓肿），不可用于无症状包囊携带者；治疗后需联用二氯尼特清除包囊。
   - **滴虫感染**：作为首选药物，男女患者均口服给药，无需调整阴道菌群。
   - **厌氧菌感染**：单用或与其它抗菌药联用，适用于外科手术预防或严重感染治疗。
   - **贾第虫病**：单用甲硝唑，治愈率约90%。
4. **告知关键注意事项**：
   - 强调服药期间及停药后至少48小时内严禁饮酒，以防乙醛中毒（表现为恶心、呕吐、腹痛、头痛）。
   - 提醒可能出现金属味、胃肠道不适或神经系统症状（如头晕、肢体感觉异常）。
5. **监测与随访**：对中枢神经系统症状或严重不良反应及时处理。

> 注意：甲硝唑对阿米巴包囊无效，不可单独用于根除包囊携带状态。
