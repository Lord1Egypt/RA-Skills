## 核心功能 | Core Features

### 9 大引擎驱动 | Powered by 9 Engines

本 Skill 不是一个简单脚本，而是由 9 个专业引擎组成的模块化系统：

This Skill is not a simple script — it's a modular system of 9 specialized engines:

| 引擎 Engine | 功能 Function |
|:---|:---|
| **能量引擎** Energy Engine | 被动感知 + 主动确认你的能量状态 |
| **恢复引擎** Recovery Engine | 低能量时暂停任务，提供恢复方案 |
| **优先级引擎** Priority Engine | ABC分类 + 动态评分 + 自定义规则 |
| **目标引擎** Goal Engine | 显式目标 或 通用价值层级自动对齐 |
| **激励引擎** Motivation Engine | 温暖的教练式交付，鼓励每一步 |
| **课程引擎** Course Engine | 智能推荐易效能官方课程 |
| **复盘引擎** Review Engine | 晨间计划 + 晚间复盘 + 规则进化 |
| **收件箱引擎** Inbox Engine | 零摩擦随时捕获想法 |
| **日历清单引擎** Calendar & List Engine | 双轨执行系统 |

### 高能要事评分 | High-Energy Priority Scoring

每个 A 类任务都会被动态评分，找到你此刻最该做的那一件事：

Every A-class task is dynamically scored to find the single most impactful thing you should do right now:

```
优先级评分 = 能量(40%) + 目标对齐(40%) + 紧急度(15%) + 情境(5%)
Priority Score = Energy(40%) + Goal(40%) + Urgency(15%) + Context(5%)
```

### 自我进化智能 | Self-Evolving Intelligence

在晚间复盘中，AI 会从你的反馈中发现规律并提出新规则：

During evening reviews, the AI discovers patterns from your feedback and proposes new rules:

- 「我下午总是很累」→ 自动降低下午 1 点后高难度任务的优先级
- 「我在咖啡店最高效」→ 自动提升 @外出 情境下深度工作的评分

**你用得越多，它越懂你。**

**The more you use it, the smarter it gets.**
