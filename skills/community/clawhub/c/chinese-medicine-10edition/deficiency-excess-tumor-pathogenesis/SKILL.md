# Deficiency-Excess Tumor Pathogenesis

## When to Use
- 患者为恶性肿瘤患者、癌前病变个体或需进行中医肿瘤辨证的对象；
- 存在正气虚弱状态（如乏力、消瘦、面色萎黄、舌淡脉弱等）；
- 同时伴有局部实邪表现（如固定肿块、刺痛、灼热、痰多、苔腻等）；
- 排除单纯外伤性肿块或非肿瘤性增生。

## Execution Steps

1. **确认“本虚标实”的基本格局**
   - 判断“本虚”：评估是否存在气虚、血虚、阴虚或阳虚，重点关注脾、肾及气血状态；
   - 判断“标实”：识别气滞、血瘀、痰浊、湿聚或热毒等病理产物的临床表现。

2. **分析病机演变链条**
   - 正气不足（因禀赋薄弱、年老体衰或久病耗伤）→ 防御力下降 → 外邪（六淫、邪毒）或内伤（七情、饮食劳倦）乘虚而入 → 脏腑功能失调 → 气机不畅 → 津液停聚成痰湿，血行受阻成瘀血 → 痰瘀互结，日久化毒 → 积而成瘤。

3. **引用经典佐证支持辨证**
   - 《医宗必读》：“积之成也，正气不足，而后邪气踞之”；
   - 《灵枢》：“积之始生，得寒乃生，厥乃成积也”；
   - 《诸病源候论》：“积聚者，由阴阳不和，脏腑虚弱……所为也”。

4. **警惕恶性循环机制**
   - 肿瘤形成后进一步耗伤正气，导致“正衰邪盛”，易发生扩散转移，最终五脏俱损、气血阴阳俱衰。

## Output
判断该肿瘤证型属于“本虚标实”，为制定“扶正祛邪”治疗方案提供病机依据。
