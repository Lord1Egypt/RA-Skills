# 川菜经典菜谱库（完整版）

## 一、凉菜类（冷菜）

### 蒜泥白肉
- **味型**：蒜泥味
- **难度**：★★
- **主料**：二刀坐臀肉 300g
- **配料**：蒜泥 30g、黄瓜（垫底）
- **调料**：红油 30ml、复制酱油 20ml、香油 5ml、白糖 3g
- **做法**：
  1. 五花肉冷水下锅，加姜葱料酒，煮至筷子能轻松插入，约25分钟
  2. 捞出晾凉，片成薄片（越薄越好，约2mm），码在黄瓜片上
  3. 蒜现捣成泥，加少量冷开水调匀
  4. 复制酱油、红油、蒜泥、白糖、香油调成汁，淋在肉片上
  5. 撒葱花/芝麻点缀
- **关键**：肉片要薄、蒜要现捣、红油要香

### 口水鸡
- **味型**：红油味
- **难度**：★★★
- **主料**：三黄鸡/鸡腿 1只
- **配料**：花生碎、芝麻、葱花、香菜
- **调料**：红油 50ml、花椒油 5ml、芝麻酱 10g、酱油 15ml、醋 5ml、糖 5g、盐、姜葱蒜
- **做法**：
  1. 鸡冷水下锅，加姜葱料酒，水开后小火浸煮15分钟，关火焖10分钟
  2. 捞出浸冰水（皮脆关键），斩块装盘
  3. 芝麻酱用香油澥开，加酱油、醋、糖、盐、红油、花椒油、蒜泥调匀
  4. 淋在鸡块上，撒花生碎、芝麻、葱花
- **关键**：浸煮不沸煮、冰水过凉皮才脆

### 红油耳片
- **味型**：红油味
- **难度**：★★★
- **主料**：猪耳 2只
- **调料**：红油 30ml、酱油 15ml、醋 5ml、糖 3g、盐、花椒面、香油、蒜泥
- **做法**：
  1. 猪耳刮洗干净，冷水下锅加姜葱料酒卤煮40分钟至软糯
  2. 趁热用重物压平，冷却后片成薄片
  3. 红油等调料调成汁，拌匀装盘
- **关键**：热压冷却才能切薄片

### 姜汁豇豆
- **味型**：姜汁味
- **难度**：★
- **主料**：嫩豇豆 250g
- **调料**：姜汁/姜米 15g、醋 10ml、盐 3g、香油 5ml
- **做法**：
  1. 豇豆去筋切段，沸水加少许盐油焯至断生
  2. 捞出过凉，沥干
  3. 姜汁+醋+盐+香油调匀，与豇豆拌匀
- **关键**：姜必须新鲜，焯水时间不能长

### 椒麻鸡片
- **味型**：椒麻味
- **难度**：★★★
- **主料**：鸡脯肉 200g
- **配料**：葱叶 50g、花椒（去籽）10g
- **调料**：盐、酱油、冷鸡汤、香油
- **做法**：
  1. 鸡脯肉片成薄片，沸水滑熟，过凉
  2. 葱叶+花椒+少许盐，用刀反复铡成细茸（椒麻糊）
  3. 椒麻糊加冷鸡汤调开，加酱油、香油
  4. 淋在鸡片上
- **关键**：椒麻糊必须手工铡，不能用料理机

### 夫妻肺片
- **味型**：红油味 / 麻辣味
- **难度**：★★★★
- **主料**：牛肉 150g、牛舌 100g、牛肚 100g、牛头皮 100g
- **配料**：芹菜段、花生碎、芝麻、香菜
- **调料**：红油 80ml、复制酱油 30ml、花椒面 5g、芝麻酱 15g、蒜泥 15g、醋 5ml、糖 5g、盐
- **做法**：
  1. 牛肉、牛舌、牛肚、牛头皮分别洗净，冷水下锅加姜葱料酒焯水
  2. 换清水加八角、桂皮、草果、香叶、花椒、姜葱，分别卤至软熟（时间不同：牛头皮最长约2h）
  3. 捞出晾凉，分别片成大薄片
  4. 调汁：芝麻酱用香油澥开+复制酱油+红油+花椒面+蒜泥+醋+糖+盐
  5. 所有主料片混合，加芹菜段拌匀
  6. 淋汁，撒花生碎、芝麻、香菜
- **关键**：各部位分开卤、片要薄、汁要多

### 灯影牛肉
- **味型**：麻辣味
- **难度**：★★★★★
- **主料**：牛后腿腱子肉 500g
- **调料**：辣椒面 15g、花椒面 5g、五香粉 3g、糖 10g、盐、料酒、姜、香油、熟芝麻
- **做法**：
  1. 牛肉去筋去膜，冷冻1小时至半硬（方便切），逆纹片成极薄大片（透光为佳）
  2. 盐+料酒+姜腌制30分钟
  3. 逐片铺在竹筛上，阳光下晒干或低温（50°C）烘干至半透明
  4. 入油锅中小火炸至棕红酥脆，捞出
  5. 锅中留少量油，下辣椒面小火炒香（不煳）
  6. 下炸好的牛肉片，加五香粉、花椒面、糖翻炒
  7. 淋香油、撒熟芝麻，晾凉密封保存
- **关键**：片要薄到透光、晒要干透、炸要酥脆不焦

### 棒棒鸡
- **味型**：红油味
- **难度**：★★★
- **主料**：三黄鸡 1只（约1000g）
- **配料**：黄瓜丝
- **调料**：红油 60ml、芝麻酱 15g、酱油 20ml、醋 10ml、糖 10g、花椒面 3g、蒜泥 10g、熟芝麻、葱花
- **做法**：
  1. 鸡冷水下锅加姜葱料酒煮熟（同口水鸡浸煮法），捞出浸冰水
  2. 用擀面杖/棒槌轻敲鸡肉，使肉质松软，然后手撕成丝
  3. 黄瓜丝垫底，铺上鸡丝
  4. 芝麻酱用香油澥开+所有调料调匀
  5. 淋在鸡丝上，撒芝麻、葱花
- **关键**：棒槌敲打让肉松软、一定要手撕不能刀切

### 怪味鸡丝
- **味型**：怪味
- **难度**：★★★
- **主料**：鸡脯肉 250g
- **配料**：黄瓜丝、葱丝
- **调料**：芝麻酱 15g、酱油 15ml、醋 10ml、糖 10g、红油 30ml、花椒面 3g、香油 5ml、熟芝麻、蒜泥
- **做法**：
  1. 鸡脯肉煮熟撕成丝
  2. 芝麻酱用香油澥开，依次加入酱油、醋、糖、红油、花椒面、蒜泥调匀
  3. 鸡丝+黄瓜丝+葱丝拌匀
  4. 淋汁，撒熟芝麻
- **关键**：七味调和，任何一味不能出头——咸甜麻辣酸鲜香同等重要

### 陈皮牛肉
- **味型**：陈皮味
- **难度**：★★★★
- **主料**：牛瘦肉 500g
- **配料**：九制陈皮 30g、干辣椒段 20g、花椒 5g
- **调料**：酱油、糖 20g、醪糟汁 30ml、盐、姜片、葱段、料酒、香油
- **做法**：
  1. 牛肉去筋切条/片，盐+料酒+姜葱腌制20分钟
  2. 锅中宽油烧至六成热，下牛肉条炸干水分（约2分钟），捞出
  3. 留底油，小火炒香花椒、干辣椒
  4. 加陈皮、牛肉、醪糟汁、酱油、糖、适量水
  5. 中小火收汁至干，淋香油出锅
- **关键**：九制陈皮风味最佳、收汁要彻底但牛肉不能煳

### 泡椒凤爪
- **味型**：酸辣味
- **难度**：★★
- **主料**：鸡爪 500g
- **配料**：泡椒（含泡椒水）200g、野山椒 50g、芹菜段、胡萝卜条、洋葱片、柠檬片
- **调料**：白醋 50ml、盐 15g、糖 15g、花椒 3g、料酒、姜
- **做法**：
  1. 鸡爪剪去指甲，对半剁开，冷水下锅加姜料酒煮10分钟至断生
  2. 捞出立即浸冰水（皮脆），冲洗去胶质
  3. 泡椒水+白醋+冷开水+盐+糖+花椒调成泡汁
  4. 鸡爪+泡椒+野山椒+蔬菜+柠檬片入泡汁
  5. 密封冰箱冷藏浸泡过夜（至少6小时）
- **关键**：鸡爪煮10分钟即可不能烂、必须冰水过凉、泡制时间≥6小时

### 麻酱凤尾
- **味型**：麻酱味
- **难度**：★
- **主料**：莴笋尖（凤尾部分）300g
- **调料**：芝麻酱 25g、香油 10ml、酱油 10ml、盐 3g、糖 5g
- **做法**：
  1. 莴笋尖去老叶，洗净沥干，切整齐排列
  2. 芝麻酱用香油澥开，加酱油、盐、糖调匀
  3. 淋在莴笋尖上或蘸食
- **关键**：莴笋尖要嫩、麻酱必须用香油澥开

### 芥末鸭掌
- **味型**：芥末味
- **难度**：★★★
- **主料**：去骨鸭掌 300g
- **配料**：黄瓜丝或莴笋丝
- **调料**：芥末膏/芥末粉 15g、醋 10ml、酱油 10ml、盐 3g、香油 5ml
- **做法**：
  1. 鸭掌洗净，沸水加姜葱料酒煮熟（约15分钟），捞出过凉
  2. 芥末粉用少许温水调成糊焖10分钟激发辣味（或用芥末膏直接调）
  3. 芥末+醋+酱油+盐+香油调匀
  4. 拌入鸭掌及配菜
- **关键**：芥末粉要提前焖发、不可久放芥末味会散

### 四川泡菜
- **味型**：酸辣咸鲜
- **难度**：★★
- **主料**：白萝卜、胡萝卜、豇豆、青笋、包菜、仔姜、红皮萝卜等
- **配料**：老坛泡菜水（引子）、野山椒
- **调料**：盐（盐水比例6-8%）、花椒 3g、八角 2颗、桂皮 1小段、冰糖 10g、白酒 10ml、老姜、大蒜
- **做法**：
  1. 所有蔬菜洗净晾干水分（不能有生水）
  2. 泡菜坛洗净消毒晾干
  3. 配制盐水：凉开水+盐+花椒+八角+桂皮+冰糖，搅至盐化
  4. 蔬菜放入坛中，加姜蒜、野山椒，倒入盐水没过所有菜
  5. 加白酒封口，坛沿加水密封
  6. 常温放置3-5天即可食用（夏天快冬天慢）
- **关键**：全程无油无生水、坛沿水要勤换、取泡菜的筷子要干净

## 二、经典热菜（炒·烧·煮·蒸）

### 回锅肉
- **味型**：家常味
- **难度**：★★★
- **主料**：二刀坐臀肉 300g
- **配料**：蒜苗 150g（可用青椒+洋葱替代）
- **调料**：郫县豆瓣 25g（剁细）、甜面酱 10g、豆豉 5g、酱油 5ml、糖 3g、姜片
- **做法**：
  1. 猪肉冷水下锅，加姜葱料酒，煮至八成熟（筷子能插入略有阻力），约20分钟
  2. 捞出晾凉，切成约3mm薄片（关键：皮、肥、瘦完整）
  3. 蒜苗斜刀切段，蒜白和蒜叶分开
  4. 锅烧热少油，下肉片煸炒至卷曲（灯盏窝），吐油
  5. 把肉拨到一边，下豆瓣炒出红油
  6. 加甜面酱、豆豉、姜片翻炒，加酱油、糖
  7. 先下蒜白炒断生，再下蒜叶，翻炒均匀出锅
- **关键**：肉必须二刀肉、豆瓣必须炒出红油、煸出灯盏窝

### 盐煎肉
- **味型**：家常味
- **难度**：★★★
- **主料**：去皮五花肉/二刀肉 300g
- **配料**：蒜苗 150g（或青椒+洋葱）
- **调料**：郫县豆瓣 25g、豆豉 10g、酱油 5ml、糖 3g、盐少许
- **做法**：
  1. 猪肉去皮切薄片（约2mm，比回锅肉薄）
  2. 锅烧热不加油或极少油，下肉片煸炒至吐油微卷
  3. 下豆瓣炒出红油，加豆豉、酱油、糖
  4. 下蒜苗等配料翻炒至断生
  5. 尝味补盐（豆瓣有咸味），出锅
- **关键**：与回锅肉区别——盐煎肉生炒不先煮、更干香、更家常

### 宫保鸡丁
- **味型**：煳辣味
- **难度**：★★★★
- **主料**：鸡腿肉 300g
- **配料**：花生米（油酥）50g、干辣椒段 20g、花椒 3g、葱段、姜片、蒜片
- **调料**：盐、酱油、醋、糖、料酒、淀粉、水淀粉
- **做法**：
  1. 鸡腿肉去筋膜，剞十字花刀，切1.5cm方丁
  2. 盐+酱油+料酒+水淀粉上浆，腌10分钟
  3. 调碗芡：酱油+醋+糖+水淀粉+鲜汤（荔枝味：糖醋约1:0.8）
  4. 锅烧热宽油，三四成热滑鸡丁至变白，捞出
  5. 留底油，小火炸香花椒（捞出），再下辣椒段炸至棕红色
  6. 下姜葱蒜爆香，下鸡丁翻匀
  7. 淋碗芡，中火收匀，下花生米翻匀出锅
- **关键**：辣椒段炸至棕红色出糊香味、碗芡一次淋入、花生最后下

### 鱼香肉丝
- **味型**：鱼香味
- **难度**：★★★★
- **主料**：猪里脊/瘦肉 200g
- **配料**：水发木耳 30g、莴笋丝 50g（正宗用青笋）、泡辣椒 30g
- **调料**：姜米 10g、蒜米 15g、葱花 20g、酱油、醋、糖、盐、料酒、水淀粉
- **做法**：
  1. 肉切丝，盐+料酒+水淀粉上浆
  2. 泡辣椒去籽剁茸
  3. 调碗芡：酱油+醋+糖+水淀粉+盐（糖醋约1:1）
  4. 锅烧热宽油，三四成热滑肉丝至变色，捞出
  5. 留底油，下泡辣椒茸炒出红油
  6. 下姜蒜米爆香，下肉丝、木耳丝、莴笋丝翻炒
  7. 淋碗芡收匀，撒葱花，翻匀出锅
- **关键**：泡辣椒必须剁茸炒出红油、葱姜蒜三宝缺一不可、芡汁一次淋入

### 鱼香茄子
- **味型**：鱼香味
- **难度**：★★★
- **主料**：长茄子 400g
- **配料**：泡辣椒 25g（剁茸）、姜米 10g、蒜米 15g、葱花 20g、肉末 50g（可选）
- **调料**：酱油、醋、糖、盐、水淀粉
- **做法**：
  1. 茄子去皮/不去皮均可，切长条或滚刀块
  2. 锅中多油，六七成热下茄子炸至表面微黄变软，捞出沥油
  3. 留底油，下泡辣椒茸炒出红油，加姜蒜米、肉末炒香
  4. 加少许水，加酱油、醋、糖（糖醋约1:1）、盐
  5. 下茄子烧1分钟入味
  6. 勾芡收汁，撒葱花出锅
- **关键**：茄子先炸软再烧入味、糖醋比例1:1

### 麻婆豆腐
- **味型**：麻辣味
- **难度**：★★★
- **主料**：嫩豆腐（石膏豆腐）1块 约400g
- **配料**：牛肉末 80g（传统用牛肉）、蒜苗
- **调料**：郫县豆瓣 25g、豆豉 5g、辣椒面 5g、花椒面 3g、酱油、盐、水淀粉
- **做法**：
  1. 豆腐切2cm方块，淡盐水浸泡10分钟（去豆腥，不易碎）
  2. 沸水加盐，豆腐焯水1分钟，沥干
  3. 牛肉末用小火煸炒至酥脆（牛肉臊子），盛出
  4. 锅中放油，小火炒豆瓣出红油，加豆豉、辣椒面
  5. 加鲜汤/水，下豆腐，加酱油、盐，小火烧3分钟入味
  6. 分三次勾芡（保持豆腐嫩滑）
  7. 撒牛肉臊子、花椒面、蒜苗碎
- **关键**：豆腐不能翻炒只能推、三次勾芡、牛肉臊子要酥

### 水煮牛肉
- **味型**：麻辣味
- **难度**：★★★★
- **主料**：牛里脊/牛腿肉 300g
- **配料**：莴笋尖、芹菜段、蒜苗段、豆芽（底菜）
- **调料**：郫县豆瓣 30g、干辣椒段 20g、花椒 10g、刀口辣椒（干辣椒+花椒铡碎）、姜蒜米、酱油、料酒、淀粉、盐
- **做法**：
  1. 牛肉逆纹切薄片，加盐+料酒+水淀粉上浆

### 韭香腰花
- **味型**：咸鲜
- **难度**：★★★★
- **主料**：猪腰 2只
- **配料**：韭菜 150g、姜蒜片、干辣椒段、花椒
- **调料**：酱油、醋、盐、料酒、水淀粉
- **做法**：同火爆腰花，韭菜替代其他配料，最后大火爆炒10秒即可
- **关键**：韭菜不可久炒

### 干烧鱼
- **味型**：家常味
- **难度**：★★★★
- **主料**：鲤鱼/鲈鱼 1条（约750g）
- **配料**：半肥瘦猪肉丁 50g、泡辣椒、姜蒜末、葱花、笋丁、香菇丁
- **调料**：郫县豆瓣 25g、醪糟汁 30ml、酱油、醋、盐、料酒、糖
- **做法**：
  1. 鱼剞花刀，盐+料酒腌15分钟，下油锅炸至两面金黄定型
  2. 留底油炒肉丁至酥吐油，下豆瓣炒红油
  3. 加泡辣椒、姜蒜末、笋丁、香菇丁炒香
  4. 加开水、醪糟汁、酱油、糖
  5. 下鱼，大火烧开转中火烧15分钟左右
  6. 自然收汁（不勾芡！），汤汁收干至只留红油
  7. 撒葱花出锅
- **关键**：不勾芡纯靠自然收干、肉丁要酥

### 豆瓣鱼
- **味型**：家常味
- **难度**：★★★
- **主料**：鲤鱼/鲫鱼 1条（约750g）
- **配料**：姜蒜末、葱花
- **调料**：郫县豆瓣 40g（多！）、酱油、醋、糖、料酒、水淀粉、盐
- **做法**：
  1. 鱼剞花刀，盐+料酒腌15分钟
  2. 锅中多油煎鱼至两面金黄定型，盛出
  3. 留底油，小火炒豆瓣出红油，加姜蒜末
  4. 加适量水，加酱油、糖、料酒
  5. 下鱼中火烧10分钟，中途翻面
  6. 鱼装盘，锅中汤汁勾芡，加少许醋
  7. 芡汁淋在鱼上，撒葱花
- **关键**：豆瓣要多、汁要宽、鱼肉要嫩

### 豆豉鲫鱼
- **味型**：豉汁味
- **难度**：★★★
- **主料**：小鲫鱼 3-4条
- **配料**：永川豆豉 30g、姜蒜末、葱段
- **调料**：酱油、糖、料酒、盐
- **做法**：
  1. 鲫鱼洗净剞花刀，盐+料酒腌15分钟
  2. 锅中多油煎鱼至两面金黄
  3. 留底油，炒香豆豉+姜蒜末
  4. 加开水、酱油、糖，下鱼中火烧10分钟
  5. 收汁出锅，撒葱花
- **关键**：永川豆豉是灵魂、豆豉稍剁碎

### 番茄炒蛋（川版，加葱姜）
- **味型**：咸鲜微甜
- **难度**：★
- **主料**：鸡蛋 3个、番茄 2个
- **配料**：葱花
- **调料**：盐、糖、水、油
- **做法**：
  1. 鸡蛋加盐打散；番茄切块
  2. 热锅多油，油温高时下蛋液，快速炒成大块，盛出
  3. 锅中加少许油，下番茄块中火炒出汁
  4. 加糖、少许盐、少许水煮1分钟
  5. 鸡蛋回锅翻炒均匀，撒葱花出锅
- **关键**：炒蛋油温高蛋才蓬、番茄炒出汁

## 三、汤羹类

### 开水白菜（国宴头菜）
- **味型**：咸鲜味
- **难度**：★★★★★
- **主料**：黄秧白/娃娃菜心 4颗
- **调料**：清汤（顶汤）：老母鸡1只+老鸭半只+金华火腿+排骨+猪肘+纯净水+姜葱，小火吊8小时，用鸡茸扫汤去杂质至清澈如水
- **做法**：
  1. 菜心修整齐，沸水焯过，凉水浸凉
  2. 用刀在根部切十字，轻轻掰开
  3. 菜心放入炖盅，注入清汤（如水般清澈）
  4. 上蒸锅蒸30分钟至菜心透明
  5. 取出摆盘，重新注入滚烫的清汤
- **关键**：清汤是灵魂——清澈如水但鲜美至极、扫汤工序不可省略

### 鸡豆花（四川国宴菜）
- **味型**：咸鲜味
- **难度**：★★★★★
- **主料**：鸡脯肉 200g
- **配料**：鸡蛋清 4个、火腿末
- **调料**：盐、清汤（顶汤）
- **做法**：
  1. 鸡脯肉去筋，刀背捶成茸，过筛
  2. 加冷清汤澥开，加蛋清、盐搅打上劲
  3. 清汤烧至微沸（不滚），倒入鸡茸浆，小火养熟
  4. 鸡茸自然浮起成豆花状
  5. 盛入碗中，注入清汤，撒火腿末
- **关键**：火候最关键——汤不能滚、慢慢养熟

### 酸辣汤
- **味型**：酸辣味
- **难度**：★★
- **主料**：猪血/豆腐 100g、鸡蛋 1个
- **配料**：木耳丝、笋丝、香菇丝
- **调料**：醋 30ml、白胡椒粉 5g、酱油、盐、水淀粉、香油
- **做法**：
  1. 配料切丝，猪血切丝焯水
  2. 锅中加鲜汤，下配料丝，加酱油盐煮开
  3. 勾芡（比一般汤浓一点）
  4. 鸡蛋打散淋入，轻轻推成蛋花
  5. 碗中放醋和胡椒粉，冲入汤中，淋香油
- **关键**：醋和胡椒最后放才够味

### 连锅汤
- **味型**：咸鲜味
- **难度**：★★
- **主料**：二刀坐臀肉 300g
- **配料**：白萝卜 300g
- **调料**：花椒 5粒、姜片、盐、葱花
- **做法**：
  1. 猪肉整块冷水下锅，加姜片、花椒，煮至八成熟
  2. 捞出肉切片
  3. 原汤（煮肉汤）去浮沫，白萝卜切片下入煮至半透明
  4. 肉片回锅煮2分钟
  5. 盐调味，撒葱花
- **关键**：用原汤不换水、萝卜要煮透

### 竹荪肝膏汤
- **味型**：咸鲜味
- **难度**：★★★★★
- **主料**：猪肝 200g、竹荪 30g
- **配料**：鸡蛋清 2个
- **调料**：盐、胡椒粉、清汤
- **做法**：
  1. 猪肝去筋，刀背捶成茸，过细筛
  2. 加蛋清、清汤、盐搅匀成稀糊状
  3. 倒入碗中，上蒸锅小火蒸10分钟成"肝膏"
  4. 竹荪泡发切段
  5. 清汤烧开加竹荪煮5分钟
  6. 肝膏切块/舀入碗中，注入竹荪清汤
- **关键**：肝茸要过筛细腻、蒸要用小火防止起泡

### 酸萝卜老鸭汤
- **味型**：酸咸鲜
- **难度**：★★★
- **主料**：老鸭 1只（约1500g）
- **配料**：酸萝卜（老坛泡萝卜）300g
- **调料**：姜片、花椒、料酒、盐、胡椒粉
- **做法**：
  1. 老鸭斩块，冷水下锅加姜料酒焯水
  2. 酸萝卜切片
  3. 鸭块+酸萝卜+姜片+花椒入砂锅，加足量开水
  4. 大火烧开转小火炖2-3小时至鸭肉酥烂
  5. 加盐、胡椒粉调味
- **关键**：必须用老鸭（肥鸭腥）、酸萝卜自带咸味盐要后加

## 四、小吃类

### 红油抄手
- **味型**：红油味
- **难度**：★★
- **主料**：抄手皮 20张、猪肉馅 200g
- **馅料**：猪肉末+姜葱水+盐+酱油+蛋清+花椒面+香油
- **调料**：红油 30ml、复制酱油 20ml、蒜泥 10g、花椒面、香油、葱花
- **做法**：
  1. 调馅：肉末加姜葱水顺一个方向搅上劲
  2. 包抄手，三角形对折再捏合
  3. 碗中调汁：红油+复制酱油+蒜泥+花椒面+香油
  4. 水开下抄手，煮至浮起再煮1分钟
  5. 捞入碗中拌匀，撒葱花
- **关键**：复制酱油是灵魂（酱油+糖+香料熬制）

### 担担面
- **味型**：麻辣味
- **难度**：★★
- **主料**：细面条 150g
- **配料**：猪肉末（炒酥）、碎米芽菜、花生碎、葱花
- **调料**：芝麻酱、红油、酱油、醋、花椒面、蒜泥、香油
- **做法**：
  1. 碗中调底汁：芝麻酱用香油澥开+红油+酱油+醋+花椒面+蒜泥
  2. 面条煮熟（稍硬一点），捞入碗中
  3. 浇上肉臊、芽菜、花生碎、葱花
  4. 吃前拌匀
- **关键**：面少汤少、臊子要酥、拌开才香

### 酸辣粉
- **味型**：酸辣味
- **难度**：★★
- **主料**：红薯粉条 150g
- **配料**：炸花生、榨菜末、香菜、葱花
- **调料**：红油、醋、酱油、花椒面、盐、高汤
- **做法**：
  1. 粉条提前泡软或直接煮软
  2. 碗中调汁：红油+醋+酱油+花椒面+盐+蒜水+高汤
  3. 粉条捞入碗，撒配料
- **关键**：醋要够酸、粉条要Q弹

### 钟水饺
- **味型**：红油味 / 蒜泥味
- **难度**：★★★
- **主料**：饺子皮、猪肉馅 300g
- **馅料**：猪肉末+姜葱水+盐+酱油+花椒面+鸡蛋
- **调料**：复制酱油 30ml、红油 40ml、蒜泥 15g、香油、花椒面
- **做法**：
  1. 调馅：猪肉末+姜葱水+盐+酱油+花椒面+鸡蛋搅上劲
  2. 包成月牙形水饺
  3. 水开下饺子，煮至浮起再煮2分钟
  4. 碗中调汁：复制酱油+红油+蒜泥+花椒面+香油
  5. 饺子捞入碗，拌匀，让每只饺子都裹满红油酱汁
- **关键**：皮薄馅嫩、复制酱油核心、一定要蒜泥

### 赖汤圆
- **味型**：香甜味
- **难度**：★★★
- **主料**：糯米粉 200g
- **馅料**：黑芝麻 80g（炒熟磨碎）+白糖 60g+猪油 30g+花生碎 20g
- **做法**：
  1. 馅料混合搓成小圆球，冰箱冷冻20分钟定型
  2. 糯米粉加温水揉成光滑面团
  3. 面团分剂，包入馅料搓圆
  4. 水开放入汤圆，煮至浮起再煮2分钟
  5. 连汤盛入碗中
- **关键**：馅要冻硬才好包、糯米粉用温水和面

### 三大炮
- **味型**：香甜味
- **难度**：★★
- **主料**：糯米 500g
- **配料**：黄豆面（黄豆炒熟磨粉）、红糖浆
- **做法**：
  1. 糯米泡4小时，蒸熟成糯米饭
  2. 趁热用木槌舂成糍粑状
  3. 分成小团
  4. 案板放一叠铜盘（三大炮的工具），将糍粑团用力摔在案板上弹入黄豆面中，发出三声"炮响"
  5. 裹匀黄豆面，装入碗中，淋红糖浆
- **关键**：糍粑要趁热舂、黄豆面要现炒现磨最香

### 宜宾燃面
- **味型**：麻辣味
- **难度**：★★★
- **主料**：碱水面 150g
- **配料**：碎米芽菜 30g、猪肉末（炒酥）50g、油炸花生碎、葱花、熟芝麻
- **调料**：红油 40ml、芝麻酱 10g、酱油 15ml、花椒面 5g、香油、蒜泥
- **做法**：
  1. 面煮熟至偏硬（断生），捞出用力甩干水分
  2. 迅速拌入熟菜籽油防止粘连
  3. 碗中调底汁：芝麻酱澥开+红油+酱油+花椒面+香油+蒜泥
  4. 将面放入碗中，浇上肉臊、芽菜、花生碎、芝麻
  5. 加大量葱花，吃前充分拌匀
- **关键**：面的水一定要甩干、红油要多、花椒面要足

### 蛋烘糕
- **味型**：香甜 / 咸香（甜馅咸馅两种吃法）
- **难度**：★★★
- **主料**：面粉 100g、鸡蛋 2个、红糖 30g、白糖 15g
- **配料**（自由搭配）：肉松、碎肉炒芽菜、花生碎+白糖（甜）、果酱、奶油、豆沙
- **做法**：
  1. 面粉+鸡蛋+红糖+白糖+水调成可流动面糊，静置30分钟
  2. 蛋烘糕专用铜锅（或小不粘锅）烧热，刷薄油
  3. 倒一勺面糊，转动锅使面糊均匀铺开
  4. 小火烘至表面起蜂窝孔、边缘翘起
  5. 在面饼一半撒上馅料
  6. 对折铲出，趁热吃
- **关键**：火要小、锅要热、面糊要稀

### 冰粉
- **味型**：香甜
- **难度**：★
- **主料**：冰粉粉 10g（或冰粉籽手搓）
- **配料**：红糖浆、花生碎、葡萄干、山楂碎、芝麻、水果粒（西瓜、芒果等）
- **做法**：
  1. 冰粉粉用沸水冲开搅匀（按说明书比例）
  2. 晾凉后冷藏2小时凝固成冻
  3. 勺子舀成块入碗
  4. 加入红糖浆和各种配料
- **关键**：红糖浆是灵魂（红糖加水熬至微稠冷却）、配料自由发挥

### 凉糕
- **味型**：香甜
- **难度**：★★
- **主料**：凉糕粉 50g
- **配料**：红糖浆
- **做法**：
  1. 凉糕粉加冷水调成糊
  2. 锅中水烧开，倒入糊不断搅拌
  3. 煮至透明浓稠
  4. 分装小碗，晾凉冷藏凝固
  5. 食用时倒扣脱模，淋红糖浆
- **关键**：煮的时候要不停搅拌防煳底

### 川北凉粉
- **味型**：红油味 / 麻辣味
- **难度**：★★
- **主料**：豌豆淀粉/绿豆淀粉 100g
- **配料**：黄瓜丝、豆芽
- **调料**：红油 30ml、复制酱油 20ml、蒜泥 15g、花椒面 3g、醋 10ml、盐、香油、葱花
- **做法**：
  1. 淀粉加冷水调成水粉浆（1:6比例淀粉:水）
  2. 锅中烧水，水开转小火，倒入粉浆不停搅拌
  3. 煮至透明浓稠，倒入平底容器晾凉凝固
  4. 用凉粉刮子刮成条，或用刀切条
  5. 黄瓜丝、豆芽垫底，放上凉粉
  6. 所有调料调匀淋在凉粉上，撒葱花
- **关键**：煮粉浆要小火不停搅拌、凉粉刮子刮比切更入味

## 五、火锅/干锅/冒菜类

### 四川麻辣火锅（底料）
- **味型**：麻辣味
- **难度**：★★★★
- **主料**：牛油 500g（或牛油+菜籽油 1:1）
- **调料**：郫县豆瓣 200g（剁细）、干辣椒段 100g、花椒 30g、永川豆豉 30g、姜 50g、蒜 50g、冰糖 20g、醪糟汁 50ml、草果 2颗、八角 3颗、桂皮 1段、香叶 5片、丁香 3粒、小茴香 10g、砂仁 3颗、白芷 2片、山柰 3片
- **做法**：
  1. 干辣椒冷水煮10分钟至软，捞出剁成糍粑辣椒
  2. 牛油入锅融化，烧至五成热，下姜片、蒜瓣炸香捞出弃
  3. 油温降至三成热，下糍粑辣椒、豆瓣，小火慢炒30-40分钟至红油透亮
  4. 下花椒、豆豉、冰糖、所有香料
  5. 继续小火炒20分钟至香料出味
  6. 加醪糟汁，炒匀出锅
  7. 静置几天让香味融合更佳
- **关键**：全程小火慢炒、糍粑辣椒不可替代、底料越陈越香

### 冒菜
- **味型**：麻辣味
- **难度**：★★
- **用料**：火锅底料/冒菜底料 100g
- **主料**（自由选择）：牛肉片、毛肚、午餐肉、藕片、土豆片、豆皮、海带、木耳、金针菇、花菜、鹌鹑蛋、宽粉
- **调料**：蒜泥、香油、花椒面、葱花、香菜、花生碎
- **做法**：
  1. 锅中加水+冒菜底料烧开
  2. 依次下入耐煮食材（藕、土豆、海带等），再下易熟食材（肉片、毛肚等）
  3. 全部煮好捞入碗中，舀入适量汤汁
  4. 加蒜泥、香油、花椒面、葱花、香菜、花生碎
- **关键**：下菜按难易熟顺序、汤汁要够味

### 干锅鸡
- **味型**：麻辣味
- **难度**：★★★
- **主料**：鸡腿肉 500g
- **配料**：土豆片、藕片、洋葱块、青红椒块、干辣椒段、花椒、姜蒜
- **调料**：郫县豆瓣 40g、火锅底料 30g、酱油、料酒、盐、糖、孜然粉、熟芝麻、香菜
- **做法**：
  1. 鸡腿肉切块，盐+料酒+酱油腌制20分钟
  2. 土豆片、藕片分别油炸至金黄脆
  3. 锅中多油下鸡块煸炒至金黄（或炸至表面酥脆）
  4. 留底油，炒豆瓣+火锅底料出红油
  5. 加干辣椒段、花椒、姜蒜炒香
  6. 下鸡块翻炒，加酱油、糖调味
  7. 下炸好的土豆片、藕片、洋葱、青红椒翻炒
  8. 撒孜然粉、熟芝麻、香菜，装入干锅
- **关键**：鸡肉要煸干/炸酥、土豆藕片先炸、最后撒孜然

### 干锅排骨
- **味型**：麻辣味
- **难度**：★★★
- **主料**：猪小排 500g
- **做法**：参照干锅鸡，排骨需先卤/炖软再煸炸
- **关键**：排骨先制熟再干锅、配菜同上

## 六、川菜基本功

### 复制酱油（红酱油）
酱油 200ml + 白糖 80g + 红糖 40g + 八角 1颗 + 桂皮 1小段 + 香叶 2片 + 小茴香少许
小火熬制约15分钟至略浓稠，滤出备用（凉菜灵魂）

### 红油（辣椒油）
1. 辣椒面（二荆条+朝天椒2:1混合）100g + 白芝麻 15g + 盐 5g 放入耐热碗
2. 菜籽油 300ml 加姜葱香菜各少许，烧至六成热（约180°C），捞出料渣
3. 油温降至150°C时泼入辣椒碗，边泼边搅
4. 加盖焖一晚，颜色更红

### 刀口辣椒
干辣椒段 + 花椒粒，小火无油焙炒至酥脆，铡/捣成粗粒。水煮类菜的灵魂用料。

### 郫县豆瓣使用铁律
1. **必须剁细**——整粒豆瓣影响口感
2. **必须小火炒出红油**——油温不超过120°C，炒至油红酱酥
3. **三年陈酿最佳**——鹃城牌或丹丹牌均可

### 清汤（顶汤）
老母鸡1只 + 老鸭半只 + 金华火腿 100g + 排骨 500g + 猪肘 1只 + 纯净水 10L + 姜葱
小火吊8小时，过滤后以鸡茸/肉茸扫汤2-3次去杂质，至汤清如水

### 猪油炼制
猪板油切小丁，加少量水入锅，中火熬化出油，至油渣金黄酥脆，滤出即得洁白猪油
