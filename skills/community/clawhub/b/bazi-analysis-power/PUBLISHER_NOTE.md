# 发布者说明

## 网络访问

本 Skill 包含 `scripts/fetch_bazi.py`，该脚本会向 `https://yoebao.com/bazi/api/bazi.php` 发起**只读 HTTPS GET** 请求。

- **触发条件**：仅在用户只提供了出生日期/时间/性别、未提供已排好的八字四柱时才调用。若用户自行提供了完整八字干支，脚本不会被触发。
- **发送内容**：出生时间戳（Unix 秒数）和性别（0/1）。不包含任何个人身份信息、文件内容或系统数据。
- **返回内容**：包含四柱干支、大运序列、起运信息的 JSON 数据。
- **为何需要**：精准的八字排盘需要处理阴阳合历转换、节气分界和大运顺逆判定。将完整的农历历法引擎内嵌到 Skill 中会使其体积臃肿且难以维护。该 API 提供了一个持续维护的、带版本的计算后端。

## 文件系统

脚本及所有参考文献均为只读，操作范围不超出 Skill 自身目录（`~/.deepseek/skills/bazi-analysis/`）。
