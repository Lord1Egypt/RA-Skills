# 肺棘球蚴病的诊断标准与手术原则

## 何时使用
- 患者有西北牧区等流行区居住或旅行史；
- 有犬、羊、牛、马等家畜密切接触史；
- 影像学（X线/CT）发现肺部孤立圆形阴影，呈典型囊肿表现；
- 实验室检查提示嗜酸性粒细胞比例显著升高（>5%）或寄生虫特异性免疫试验阳性。

## 如何执行

### 第一步：确认诊断三要素
1. **流行病学史**：确认患者是否来自或曾到过流行区，并有家畜接触史。
2. **影像学表现**：
   - **单纯囊肿**：密度均匀、边界清楚、边缘整齐的圆形/椭圆形单发或多发孤立阴影；
   - **囊肿破裂后典型征象**（出现任一即支持诊断）：
     - 外囊破裂 → 顶部新月形透亮区；
     - 内外囊均破 → 液平面上方双弧形透亮带；
     - 内囊陷落漂浮 → “水上浮莲”征；
     - 内容物排空 → 囊状透亮影类似肺大疱。
3. **实验室检查**：
   - 嗜酸性粒细胞比例 ≥25%（正常<5%）；
   - Casoni皮内试验阳性（阳性率70%–90%）；
   - 棘球蚴补体结合试验阳性。

### 第二步：制定手术方案
> **注意：禁止穿刺活检！以防囊液外溢导致过敏性休克或头节播散。**

- **首选治疗**：手术是唯一有效方法（无特效药物）。
- **核心原则**：完整摘除内囊 + 严格防止囊液外溢。

#### 手术方式选择：
1. **内囊摘除术**（适用于无并发症者）：
   - 方法A：先穿刺抽出部分囊液，注入10%氯化钠溶液杀灭头节，15分钟后切开外囊取出内囊；
   - 方法B：不穿刺，沿内外囊间隙分离，术中气管加压吹气促使内囊完整逸出；
   - 术后处理：剥离切除外囊壁，缝合细小支气管开口。
2. **囊肿摘除术**（适用于较小、深部、无并发症者）：内外囊一并完整摘除，缝合肺创面。
3. **肺叶/肺段切除术**（适用于并发感染者）：切除受累肺组织。

### 第三步：禁忌与预警
- **绝对禁忌**：不得以穿刺作为诊断手段；
- **高风险人群**：过敏体质者术前需备好抗过敏及抢救措施。
