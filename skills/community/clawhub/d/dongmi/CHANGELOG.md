# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-05-04

### Added
- **2026年证监会《上市公司董事会秘书监管规则》核心内容**（2026年5月24日起施行）
  - 首部专门针对董秘的部门规章
  - 任职禁止：不得兼任经理、分管经营业务的副经理、财务负责人
  - 任职资格：5年以上相关工作经验，或取得法律/注册会计师证书+5年工作经验
  - 过渡期安排至2027年12月31日
- **沪深北交易所2026年配套规则更新**
  - 上交所：《上市公司自律监管指引第1号——规范运作（2026年修订）》（上证发〔2026〕44号）
  - 深交所：《股票上市规则（2026年修订）》（深证上〔2026〕549号）
  - 北交所：《北京证券交易所股票上市规则（2026年修订）》
  - 关键条款（第4.4.1-4.4.6条等）自2026年5月24日起施行
  - 完全合规截止日：2028年1月1日
- **香港交易所2025年最新要求**
  - 《董事會及董事企業管治指引》（2025年5月更新）
  - 公司秘书兼任限制：不得超过3家无关联上市公司
  - 2025年7月1日起IPO申请人不得有超额任职独立非执行董事
  - 新增ESG、网络安全、数据治理职责

### Changed
- 更新参考法规体系，新增2025-2026年最新监管文件
- 更新免责声明，强调关注最新监管动态
- 版本号升级至1.1.0

### Notes
- A股约2400家公司存在董秘兼任情况，需在过渡期内调整
- 董秘空缺期限不得超过9个月（新规强化）
- 沪深300/科创50样本公司须制定《市值管理制度》并披露

## [1.0.0] - 2026-05-04

### Added
- 初始版本发布
- 职能领域一：合规与信息披露（定期报告、临时公告、内幕信息管控）
- 职能领域二：投资者关系与市值管理（IR体系、路演、危机公关、市值管理工具箱）
- 职能领域三：公司治理与三会运作（股东大会/董事会/监事会全生命周期管理）
- 职能领域四：资本运作全流程（股权融资、并购重组、股权激励、股份回购、分拆上市）
- 重大事项处理快速判断清单
- 合规红线汇总
- 年度工作计划框架模板
- 参考法规体系（包含2024年新《公司法》、独董管理办法、市值管理指引第10号）

### Notes
- 适用于沪深北交所上市公司
- 支持 OpenClaw / WorkBuddy / QCLAW 等龙虾工具体系
- 基于《上市公司董事会秘书四大职能领域核心技能全解析》文档构建
