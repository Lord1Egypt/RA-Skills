# 表面麻醉实施

根据目标部位选择合适的局麻药浓度与给药方式：

## 1. 给药方式与药物选择
| 部位     | 给药方式 | 推荐药物与浓度               |
|----------|----------|------------------------------|
| 眼部     | 滴入法   | 0.5%～1%丁卡因               |
| 鼻部     | 涂敷法   | 1%～2%丁卡因 或 2%～4%利多卡因 |
| 咽喉/气管| 喷雾法   | 1%～2%丁卡因 或 2%～4%利多卡因 |
| 尿道     | 灌入法   | 2%～4%利多卡因               |
| 皮肤     | 涂敷法   | 复方局麻药乳膏（如利多卡因+丙胺卡因）|

## 2. 操作要点
- 使用**穿透力强的局麻药**，确保药物能透过黏膜/角质层阻滞神经末梢。
- 麻醉目标为**产生局部无痛觉**，非肌肉松弛或运动阻滞。
- **眼部组织柔嫩**，必须使用更低浓度（≤1%丁卡因）以防角膜损伤。

## 3. 禁忌与限制
- ❌ 不适用于**实质脏器、深层组织或无痛觉结构**（如脑组织）。
- ❌ 黏膜破损或感染区域慎用，避免全身吸收增加毒性风险。

> ✅ 成功标志：目标区域痛觉消失，可安全进行浅表操作。
