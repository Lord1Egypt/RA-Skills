# 青光眼β受体阻断药选择与应用

## 适用对象
- 原发性开角型青光眼患者
- 继发性青光眼患者
- 术后未控闭角型青光眼患者
- 药物或手术治疗无效的青光眼患者
- 高眼压症患者
- 眼动脉堵塞患者（仅限卡替洛尔）

## 禁忌证筛查
在考虑使用噻吗洛尔或卡替洛尔前，必须确认患者**无以下情况**：
- 哮喘
- 严重心功能不全
- 对β受体阻断药过敏

## 药物选择流程
1. **确认诊断与禁忌证**：确保患者符合适用对象且无禁忌证。
2. **评估特殊需求**：
   - 若患者合并眼动脉堵塞，优先考虑卡替洛尔（因其具有血管扩张作用）。
   - 若无需血管扩张效应，可选择噻吗洛尔或卡替洛尔。
3. **确定药物特性**：
   - 噻吗洛尔：无内在拟交感活性（ISA），降眼压效果强。
   - 卡替洛尔：具有ISA，可能减少某些全身副作用，但可能引起结膜充血。
4. **制定给药方案**：
   - 使用0.1%～0.5%噻吗洛尔滴眼液，或等效卡替洛尔制剂。
   - 每日滴眼2次。
5. **监测与随访**：
   - 观察是否出现结膜充血（尤其使用卡替洛尔时）。
   - 警惕全身不良反应（如支气管痉挛、心动过缓），尤其在敏感患者中。

## 注意事项
- 两种药物均通过减少房水生成降低眼内压，不引起缩瞳或调节痉挛。
- 尽管局部用药，仍可能经眼部吸收产生全身效应，需谨慎用于潜在高风险人群。
- 卡替洛尔可作为联合治疗的一部分，用于其他疗法控制不佳的病例。
