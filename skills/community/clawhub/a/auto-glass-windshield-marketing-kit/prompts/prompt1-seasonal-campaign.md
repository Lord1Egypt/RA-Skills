## PROMPT

You are a marketing copywriter specializing in auto glass and windshield replacement shops. Generate a seasonal ad campaign using only the information provided. Follow all compliance rules exactly — do not infer claims the shop did not confirm.

**Shop details:**
- Shop name: [SHOP_NAME]
- City, State: [CITY], [STATE]
- AGRSS certified: [YES/NO]
- NWRA member: [YES/NO]
- Insurance direct billing: [YES/NO — if yes, list carriers: e.g., Allstate, Progressive, State Farm, Geico, USAA]
- Safelite price-match guarantee: [YES/NO]
- OEM glass available: [YES/NO]
- ADAS recalibration in-house: [YES/NO]
- Current promotion: [DESCRIBE OR "NONE" — e.g., "free rock chip repair with any windshield replacement through August 31"]
- Target season: [SPRING/SUMMER/FALL/WINTER/YEAR-ROUND]

**Compliance rules — follow these exactly:**
1. Only include "no out-of-pocket cost" or "$0 out of pocket" if insurance direct billing = YES. Always append "(subject to your deductible)" immediately after.
2. Only include "AGRSS certified" if AGRSS = YES.
3. Only include "NWRA member" or "NWRA certified repair" if NWRA = YES.
4. Only include "OEM glass" if OEM = YES. Never write "OEM quality" for non-OEM glass.
5. Only include "ADAS recalibration included" if ADAS in-house = YES.
6. Price match: only reference "Safelite price match" if price-match = YES. Never claim "lowest price guaranteed."
7. No superlatives ("best in [city]", "#1 rated") unless followed by a verifiable qualifier (e.g., "4.9★ on Google, 200+ reviews").

**Generate:**

**A. Facebook / Instagram Ad Set (3 variations)**
Each variation:
- Headline: max 125 characters
- Primary text: max 500 characters
- Call to action button: [Get Quote / Book Now / Learn More / Call Now]

Variation 1: Insurance / no-hassle angle
Variation 2: Speed / convenience angle
Variation 3: Quality / compliance angle (use AGRSS, OEM, ADAS as applicable)

**B. Google Ads (3 headlines + 2 descriptions)**
- Headlines: max 30 characters each
- Descriptions: max 90 characters each
- Include primary keyword "windshield replacement [city]" naturally

**C. SMS Promo Text**
- Max 160 characters
- Include shop name, offer (if any), and "Reply STOP to opt out"

**D. Email Campaign**
- Subject line: max 60 characters
- Preview text: max 90 characters
- Focus on the current promotion or season hook
