# Kai's Horse.skill
> 48 Classic Thought Experiments | Lobster AI Exclusive Skill

## Introduction
Kai's Horse is a complete speculative system,
containing **48 world-famous thought experiments**.

Covers classical philosophy, logic paradox, quantum science,
consciousness research and AI ethics.

Designed for Lobster AI framework, one-click installation
with full interactive command system.

## Core Features
- Global philosophical enhancement
- Exclusive command prefix: `/kh`
- Random / Query / AI modern interpretation
- Contrast analysis & keyword search
- Immersive story mode
- Unique cold & rational writing style

## Installation
1. Put `Kai's Horse.skill` into `/skills` folder
2. Restart Lobster AI
3. Load automatically
