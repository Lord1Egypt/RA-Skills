## Description: <br>
Translates API documentation from URLs, HTML pages, JSON responses, or Markdown into Traditional Chinese using Taiwan terminology while preserving code and document structure. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[dolphins1123](https://clawhub.ai/user/dolphins1123) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, technical writers, and API consumers use this skill to fetch API documentation and produce Traditional Chinese translations for review, integration, or support workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill fetches documentation URLs, so using it on sensitive internal API documentation could expose content to the agent workflow. <br>
Mitigation: Use only public or approved documentation sources, and avoid sensitive internal documents unless the environment and publisher are trusted. <br>
Risk: Translated API documentation can introduce wording changes that affect technical meaning. <br>
Mitigation: Review translated output before relying on it for implementation, support, or customer-facing documentation. <br>
