# Beehiiv Setup Guide

## Create Your Account
1. Go to beehiiv.com → click **Get Started** → sign up (free)
2. Create your publication: choose a name, select your niche, skip monetization for now

## Find Your Publication ID
1. Log into beehiiv → go to **Settings** (gear icon, bottom left)
2. Click **Publication** → scroll to **Publication ID**
3. It looks like: `pub_xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`

## Get Your API Key
1. In Settings → click **Integrations** (left sidebar)
2. Click **API** → click **New API Key**
3. Name it (e.g. "OpenClaw") → click **Create**
4. Copy the key immediately — it won't be shown again

## Auto-Publishing (Scale/Enterprise only)
Auto-posting via API requires the **Scale plan ($39/month)** or higher.
- Free plan: API is read-only (analytics only) — use paste-ready docs instead
- Scale plan: API allows creating and publishing posts programmatically
- To check your plan: Settings → Billing

## Sitemap (one-time, do after launch)
Submit your sitemap to Google Search Console:
`https://your-slug.beehiiv.com/sitemap.xml`
This helps Google find your posts faster.
