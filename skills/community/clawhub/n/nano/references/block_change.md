# xno-skills block change

```
Usage: cli block change [options]

Build an unsigned change block

Options:
  -a, --account <address>     Nano account address
  --representative <address>  New Nano representative address
  --url <url>                 RPC URL override
  -j, --json                  Output JSON with block hex + metadata
  -h, --help                  display help for command
```
