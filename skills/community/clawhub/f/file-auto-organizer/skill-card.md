## Description: <br>
File Auto Organizer helps organize local folders by moving files into type- or date-based subfolders and reporting file statistics. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[SxLiuYu](https://clawhub.ai/user/SxLiuYu) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and agents use this skill to organize Downloads, Desktop, or similar local folders by file type or modification date, then optionally print a summary report. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Organize commands move files in the selected folder into new subfolders without a preview or confirmation prompt. <br>
Mitigation: Double-check the target path and use a backup or non-critical folder before running organization commands. <br>
