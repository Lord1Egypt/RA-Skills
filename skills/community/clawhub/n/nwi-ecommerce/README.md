# NWi Ecommerce — 跨境电商数据查询 Skill


> 由 [NWi 诺舟智数](https://nuozhoushuzhi.com/) 提供的跨境电商数据分析 Skill，让 AI Agent 直接查询 Amazon、Shopee、Lazada、TikTok Shop 等平台的电商数据。

---

## ✨ 功能概览

| 功能 | 说明 | 接口 |
|------|------|------|
| 📊 品类分布 | 各平台一级类目销量、销额占比 | A1 |
| 🏆 全站排行 | Top 品牌 / 店铺 / 商品 | A2 / A3 / A4 |
| 🔍 品类搜索 | 按关键词搜索多级品类，支持中英文 | 前置接口 |
| 📈 品类分站详情 | 某品类在各站点的销售分布 | C7 |
| 🏅 品类排行 | 某品类各站点 Top 品牌 / 店铺 | C8 / C9 |
| 🔍 品牌概况 | 品牌汇总数据（不分市场） | B5 |
| 🌍 品牌分站 | 品牌在各站点表现 + Top 商品 | B6 |

**覆盖平台：** Amazon、Shopee、Lazada、TikTok Shop

**覆盖品类：** 3C数码、美容护理、服饰鞋包、家居用品、食品/保健、母婴、运动/户外 等 20 个一级品类，支持按关键词搜索多级品类（一级/二级/三级）

---

## 🚀 快速开始

### 获取 API Key

首次使用需要 API Key，有三种方式：

1. **自动申请临时 Key** — Skill 会自动调用 `generate-normal-api-key`，权限范围有限
2. **手动提供** — 已有 NWi 账户的用户可直接输入 Key
3. **留资申请** — 提交联系信息可临时升级key，后续NWi客服会主动联系并提供完整权限 Key

### 使用示例

安装后在 OpenClaw 对话中直接用自然语言查询：

```
💬 "Shopee 印尼站某品类 Top 品牌"
💬 "XX品牌在东南亚各站点表现"
💬 "2025年食品饮料品类在东南亚6国的销售趋势"
💬 "亚马逊美国站品类分布"
```

## 📋 支持的查询场景

### 按平台维度
- Amazon：美国、英国、德国、日本、新加坡 等站点
- Shopee：印尼、泰国、马来、菲律宾、新加坡、越南
- Lazada：印尼、泰国、马来、菲律宾、新加坡、越南
- TikTok Shop：印尼、泰国、马来、菲律宾、新加坡、越南、美国

### 按品类维度
3C数码、美容护理、服饰鞋包、家居用品、家装家饰、母婴、食品/保健、运动/户外、汽车配件、珠宝/首饰、宠物用品、大家电、小家电 等
支持按关键词搜索品类（一级/二级/三级），数据查询接口支持传入任意层级品类 ID

### 按品牌维度
支持品牌模糊搜索，自动匹配品牌 ID，支持跨品类、跨站点对比

## 🏗️ 技术架构

```
nwi-ecommerce/
├── SKILL.md                      # Skill 主文件（工作流、规则）
├── README.md                     # 本文件
└── references/
    ├── api.md                    # API 接口文档
    ├── api_key.txt               # API Key 存储（自动管理）
    └── platform_ids.md           # 平台 ID 速查表
```

### API 规范

- **请求方式**：POST
- **Content-Type**：`application/json`
- **认证**：API Key 通过 body 传递
- **响应格式**：JSON（`code: 0` 成功，`code: 1` 失败）
- **时间格式**：`yyyy-MM`

## 🧠 智能特性

- **品牌模糊匹配** — 输入品牌名自动搜索，多结果时按销额量级排序供用户确认
- **品类智能匹配** — 支持中英文品类名搜索，根据权限返回多级品类（含层级路径），自动匹配最佳品类 ID
- **数据异常检测** — 自动检测量级异常、数据缺失、趋势异常，并上报 NWi 后台
- **权限自动管理** — 自动检查 API Key 有效期和查询范围，过期自动引导续期
