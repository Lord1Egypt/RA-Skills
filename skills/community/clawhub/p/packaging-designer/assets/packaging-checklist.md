# Packaging Pre-Production Checklist

Go through this before signing off on a production PO. Forty-two checks across eight categories.

## 1. Sizing & fit (6 checks)

- [ ] Box inner dimensions measured against product + dunnage, with ≤1 inch headroom
- [ ] Product does not shift when box is tilted 45° and gently shaken
- [ ] Outer dimensions align to a stocked corrugate size (avoid fully custom unless >25k units)
- [ ] Dim-weight calculated against top 3 carriers — no surprise oversize charges
- [ ] Stacks flat on a warehouse pallet (no odd overhang)
- [ ] Void fill volume <25% of box interior

## 2. Brand & print (5 checks)

- [ ] Logo is present but not covering every surface
- [ ] Colors matched to brand guide (Pantone values supplied)
- [ ] Typography matches brand — no "close enough" fonts
- [ ] Print method appropriate for substrate (flexo for kraft, litho-lam for photo)
- [ ] No photo-realistic imagery on kraft (prints muddy)

## 3. Protection (5 checks)

- [ ] Dunnage matched to primary damage mode (crush / shock / abrasion / moisture)
- [ ] Fragile surfaces (screens, labels, finish) have direct contact protection
- [ ] Drop test passed 5/5 from 30 inches each face
- [ ] Live-ship test to 20 zip codes returned ≤1% damage rate
- [ ] Dunnage is curbside-recyclable where market requires it

## 4. Unboxing (5 checks)

- [ ] First visible layer on open is a deliberate brand moment, not clutter
- [ ] Product is revealed last, protected to the last layer
- [ ] No loose paperwork floating on top
- [ ] Sequence photographed and signed off by brand lead
- [ ] Opening feels smooth — no tape-fighting, no knife required

## 5. Insert card (5 checks)

- [ ] One primary CTA (review / referral / repurchase / UGC)
- [ ] QR code ≥2.5 cm square on uncoated/matte stock
- [ ] URL printed under QR as fallback
- [ ] Discount code, if any, is trackable and has an expiry
- [ ] Copy does not explicitly request 5-star or positive reviews

## 6. Sustainability & claims (5 checks)

- [ ] Recyclable claim verified against curbside reality in target markets
- [ ] FSC or recycled-content claim backed by vendor certification
- [ ] No "compostable" claim unless home-compostable, not just industrial
- [ ] Ink water-based or soy-based, not solvent, where feasible
- [ ] Plastic content minimized, noted if any

## 7. Commercial (5 checks)

- [ ] Unit cost at 1k, 5k, 25k volumes on a signed quote
- [ ] Tooling/die-line charges known and one-time
- [ ] MOQ matches realistic 6-month demand forecast
- [ ] Lead time from PO to delivery documented
- [ ] Storage terms negotiated if vendor warehouses

## 8. Legal & logistics (6 checks)

- [ ] Country-of-origin marked where required
- [ ] Barcodes scan correctly on first pass (test with a scanner)
- [ ] Batch / lot code location specified
- [ ] Customs tariff code confirmed for international shipments
- [ ] Hazmat classification confirmed if applicable (aerosols, lithium)
- [ ] AQL inspection level set (II-1.5 or II-2.5 typical)
