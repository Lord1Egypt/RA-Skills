# 碱基替换功能效应分类

## 适用条件
- 突变必须发生在编码序列的三联密码子内
- 需提供原始密码子（original_codon）和突变后密码子（mutated_codon）
- 不适用于非编码区、内含子或调控区域的碱基替换

## 执行步骤

1. **判断是否为同义突变**：
   - 比较原始密码子与突变后密码子所编码的氨基酸
   - 若两者编码相同氨基酸，则为同义突变 → 蛋白质功能正常，无表型改变

2. **判断是否为无义突变**：
   - 若突变后密码子为终止密码子（UAA、UAG 或 UGA），而原始密码子编码氨基酸
   - 则为无义突变 → 翻译提前终止 → 多肽链截短 → 蛋白质功能异常或丧失

3. **判断是否为终止密码突变**：
   - 若原始密码子为终止密码子，而突变后变为编码氨基酸的密码子
   - 则为终止密码突变 → 翻译无法正常终止 → 多肽链异常延长 → 功能异常蛋白

4. **判断是否为错义突变**：
   - 若突变导致密码子编码不同氨基酸，且不满足上述三类
   - 则为错义突变 → 氨基酸种类改变 → 可能影响蛋白质结构/功能 → 可致分子病或代谢病

## 输出要求
返回突变类型（同义/无义/终止密码/错义）及其对蛋白质功能的预期影响。

> 注意：若突变位置不在编码区密码子内，请勿使用本技能；应考虑转录调控或RNA剪接等其他机制。
