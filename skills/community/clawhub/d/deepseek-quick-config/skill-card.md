## Known Risks and Mitigations: <br>
Risk: DeepSeek API keys can be exposed or misused during setup. <br>
Mitigation: Use a dedicated API key with spending limits when possible, keep the key private, and avoid committing it to shared files. <br>
Risk: Prompts sent through OpenClaw may be processed by DeepSeek as an external model provider. <br>
Mitigation: Verify the DeepSeek endpoint before use and apply the organization's data handling rules for external AI providers. <br>
