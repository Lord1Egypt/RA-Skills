# 公众号发布前检查清单

每篇文章发布前，逐项检查。**任何一项未勾选，都不要点击发布。**

---

## 🔴 强制检查（不通过不放行）

### 内容安全

- [ ] 无违法违规内容（政治敏感/色情/暴力/赌博等）
- [ ] 无未经授权的转载内容
- [ ] 无夸大/虚假宣传（"最""第一""唯一"等绝对化用语）
- [ ] 无未经证实的医疗/健康/金融投资建议
- [ ] 引用数据来源清晰可查
- [ ] 图片无版权风险（非无版权图库来源的图片已取得授权）

### 基础信息

- [ ] 标题无误（无错别字、无歧义）
- [ ] 作者信息正确
- [ ] 账号名称/头像正确
- [ ] 原创声明设置正确（已开通原创的账号必须勾选）
- [ ] 文中人物姓名/地名/机构名无误
- [ ] 金额/数据/日期经二次核对

---

## 🟡 内容质量

### 标题

- [ ] 标题 ≤ 30字（手机端不被截断）
- [ ] 标题不含夸张/煽动/标题党用语
- [ ] 标题与正文一致（不欺骗）
- [ ] 标题有吸引力（让非粉丝也想点开）

### 正文

- [ ] 前200字能抓住注意力
- [ ] 段落 ≤ 手机一屏（5行内）
- [ ] 每300字有信息增量（数据/案例/观点）
- [ ] 核心观点有数据或案例支撑
- [ ] 结论清晰（读者读完知道"所以呢"）
- [ ] 无逻辑漏洞（前后矛盾、因果混淆）
- [ ] 语言口语化（读出来不别扭）
- [ ] 无冗余表达（能删的句子都删了）

### 结尾

- [ ] 有总结收束（一句话升华）
- [ ] 有互动引导（评论/在看/转发）
- [ ] 有关注引导（新读者知道怎么关注）

---

## 🟢 视觉体验

### 排版

- [ ] 字体/字号统一（无忽大忽小）
- [ ] 配色 ≤ 3种（品牌色+强调色+灰色）
- [ ] 段间距充足（15-20px）
- [ ] 行间距 1.6-1.8倍
- [ ] 引用/加粗合理（不过度）
- [ ] 分割线位置恰当
- [ ] 无意外格式错乱（复制粘贴导致的异常）

### 配图

- [ ] 封面图已生成（900×383px）
- [ ] 封面图标题文字清晰可读
- [ ] 正文配图齐全（每800-1000字1张）
- [ ] 图片清晰无水印
- [ ] 图片已被压缩（不要原图上传，加载慢）
- [ ] 图片风格统一（色调/风格一致）
- [ ] 每张图有图注（居中，13px #999）

### 手机预览

- [ ] 首屏视觉舒适（无大段文字堆砌）
- [ ] 首屏即有配图或标题分隔
- [ ] 全文预览无排版错乱
- [ ] 图片加载正常

---

## 🔵 发布策略

### 时间选择

| 发布时间 | 适合类型 | 说明 |
|---------|---------|------|
| 早上 7:00-8:30 | 资讯/干货 | 通勤时间，阅读行为多 |
| 中午 11:30-13:00 | 轻松/生活 | 午休时间 |
| 晚上 18:00-20:00 | 深度/长文 | 下班通勤 |
| 晚上 20:00-22:00 | 情感/娱乐 | 睡前阅读高峰 |

- [ ] 发布时间符合账号粉丝活跃规律
- [ ] 非重大节假日/事件冲突（除非有意蹭热点）

### 标签与话题

- [ ] 标签/话题设置完毕（3-5个，标签间有关联性）
- [ ] 标签覆盖核心关键词和长尾词

### 多渠道准备

- [ ] 朋友圈转发文案已准备（不超100字）
- [ ] 社群转发文案已准备（差异于朋友圈）

---

## 🟣 发布后

### 发布后30分钟

- [ ] 文章是否正常显示（无被删/被屏蔽）
- [ ] 封面图是否正常展示
- [ ] 首条/次条顺序是否正确

### 发布后2小时

- [ ] 初次阅读量统计（记录基线）
- [ ] 留言区回复（前10条评论优先回复）
- [ ] 查看用户反馈（评论区/私信）

### 发布后24小时

- [ ] 完整数据记录（阅读量/分享量/关注量/评论数）
- [ ] 重点评论精选并回复
- [ ] 复盘记录写入数据追踪卡
