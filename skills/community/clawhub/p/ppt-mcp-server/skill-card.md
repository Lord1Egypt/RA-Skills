## Description: <br>
PPT Master MCP lets an agent create professional PowerPoint presentations by generating and running python-pptx code. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[janson1983](https://clawhub.ai/user/janson1983) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and presentation authors use this skill to have an agent draft slide content, generate python-pptx code, and save finished PowerPoint files. It also lists available templates and previously generated output files. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can run unrestricted Python code generated by an assistant on the user's machine. <br>
Mitigation: Use a sandbox, VM, or container; review generated code before execution; and restrict network and filesystem access where possible. <br>
Risk: Runtime dependencies may affect behavior or security over time. <br>
Mitigation: Pin or update dependencies before relying on the skill in a production workflow. <br>
