## 中国GB 4806体系（2025-2026最新）

### 基础标准
```
GB 4806.1-2016  食品接触材料及制品通用安全要求（基础标准，正在修订）
GB 9685-2016    食品接触材料及制品添加剂使用标准（1000+许可添加剂）
```

### 材质专项标准（现行）
```
GB 4806.2-2016   奶嘴（硅胶/橡胶）
GB 4806.3-2016   搪瓷制品（铅≤1.0mg/L；镉≤0.07mg/L）
GB 4806.4-2016   陶瓷制品（铅≤0.8mg/L；镉≤0.07mg/L）
GB 4806.5-2013   玻璃制品（铅/镉/砷/锑溶出）
GB 4806.6-2016   塑料树脂（树脂规格型号）
GB 4806.7-2016   食品接触用塑料（总迁移≤10mg/dm²；高锰酸钾≤10mg/kg）
GB 4806.8-2016   纸制品（荧光增白剂不得检出；铅/砷/甲醛/菌落总数）
GB 4806.9-2016   金属材料及制品（2023年修订：须标注304/316牌号；锰≤0.6mg/kg）
GB 4806.10-2025  涂料及涂层（2026-09-02实施，替代2016版）
GB 4806.11-2023  橡胶材料及制品（硅橡胶已移出至GB 4806.16）
GB 4806.16-2025  硅橡胶材料及制品（2026-09-02实施，新增挥发性物质限量）
```

## FCM检测项目速查表

### 塑料（GB 4806.7）
```
□ 总迁移量（4%乙酸 / 95%乙醇 / 异辛烷）
□ 高锰酸钾消耗量（≤10mg/kg）
□ 脱色试验（乙醇/浸泡液，不得脱色）
□ 重金属（以铅计）
□ 游离酚（特定材质）
□ 特定迁移单体（特定材质）
□ 荧光增白剂（不得检出）
```

### 金属（GB 4806.9）
```
□ 铅/铬/镍/镉溶出量（不锈钢特定牌号）
□ 锰溶出量（2023年新增限值0.6mg/kg）
□ 锡溶出量（镀锡板）
□ 砷溶出量（特定材质）
```

### 纸制品（GB 4806.8）
```
□ 铅/砷/甲醛
□ 荧光增白剂（不得检出）
□ 菌落总数
□ 大肠杆菌
□ 沙门氏菌
□ 荧光增白剂（特定纸杯/纸餐具）
```

### 涂层（GB 4806.10）
```
□ 总迁移量
□ 苯乙烯残留量
□ 氯乙烯残留量
□ 双酚A迁移量
□ 重金属（以铅计）
```

## 全球FCM出口合规检查清单

### 出口中国
```
□ 型式检验报告（1年内有效，全项目检测）
□ QS生产许可证（塑料/纸制品/餐具等）
□ 产品标准号（GB 4806.X）与检测报告一致
□ 标签标注"食品接触用"
□ 包装使用食品级油墨
```

### 出口欧盟
```
□ 正面清单确认（所有物质须在 (EU) 10/2011 Annex I）
□ BPA禁用确认（2025-01-20起全面禁用）
□ 总迁移量测试（EU EN 1186）
□ 特定迁移量测试（SML）
□ GMP符合性声明（EC 2023/2006）
□ 全面禁用品检测（BPA/DEHP等）
```

### 出口美国
```
□ FCN确认（须向FDA提交并获得批准）
□ 21 CFR 170-189合规确认
□ 再生塑料使用须符合FDA指南
□ 特定用途须申请TOR豁免
```

### 出口日本
```
□ 正面清单确认（合成树脂用）
□ 总迁移量测试（2026-06-01起强制）
□ 重金属/蒸发残留测试
□ JFSL附则符合性
```

### 出口韩国
```
□ 肯定列表确认
□ 总迁移量测试（2026年→40mg/L收紧）
□ 邻苯二甲酸酯特定迁移量
□ 再生PP使用条件（2026年新增）
```
