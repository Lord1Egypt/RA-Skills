# 考试试卷格式规范

## 通用结构
```
标题行（居中大号）
副标题行（居中，含考试时间和满分）
信息栏（姓名/班级/学号）
注意事项
─────────────
一、选择题（N题×M分）
二、判断题（N题×M分）
三、编程填空题/解答题（N题×M分）
─────────────
结束提示
```

## 格式参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 页面 | A4 | 标准试卷用纸 |
| 边距 | 60/60/50/50mm | 左右上下 |
| 标题字号 | 20pt | 居中 |
| 正文字号 | 11pt | 题干 |
| 选项字号 | 11pt | 缩进18pt |
| 代码字号 | 10pt | 灰色背景 |
| 解析字号 | 10pt | 灰色文字 |

## 学科配置预设

### 信息学奥赛（小学组）
- 选择题15题×4分=60分
- 判断题5题×4分=20分
- 编程填空2题×10分=20分
- 考试时间60分钟，满分100分

### 信息学奥赛（初中组）
- 选择题20题×2分=40分
- 阅读程序5题×4分=20分
- 完善程序3题×8分=24分
- 编程题2题×8分=16分
- 考试时间120分钟，满分100分

## JSON数据格式
详见种子题库 `exam-data/questions/informatics.json` 的结构。

关键字段：
- `type`: choice / judgment / programming_fill
- `difficulty`: 1(基础) / 2(应用) / 3(综合)
- `topic`: 知识点分类（见informatics.md）
- `fills`: 编程填空题的填空项数组
