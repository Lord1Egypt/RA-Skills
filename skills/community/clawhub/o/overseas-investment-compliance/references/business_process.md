# 三类主体标准业务流程

## 核心前置判断：资产流向框架

> ⚠️ 业务流程因流向不同而完全不同，请先确认资产流向。

| 资产位置 | 去向 | 业务流程章节 |
|---------|------|-------------|
| 境内 | 境外 | 见本章第一节（ODI备案） |
| 境外 | 境内 | 见本章第二节（外商投资） |
| 境外 | 港澳 | 见本章第三节（港股上市） |
| 境内 | 港澳 | 见本章第四节（CEPA渠道） |

---

## 一、境内→境外：民企ODI备案流程

### 适用条件
- 资产位置：境内
- 投资目的地：境外（除敏感国家和地区）
- 投资主体：境内民营企业
- 投资行业：非敏感行业
- 投资金额：3亿美元以下

### 标准流程图

```
Day 1-3: 材料准备
├── 营业执照、公司章程
├── 近1年审计报告
├── 董事会/股东会决议
├── 可行性研究报告
├── 股权穿透图（至最终受益人）
└── 资金来源证明（6个月银行流水）

Day 4-18: 发改委备案
├── 登录jwtz.ndrc.gov.cn
├── 填写备案申请表
├── 上传材料
└── 领取《备案通知书》+ 项目编码

Day 19-25: 商务部备案
├── 登录ecms.mofcom.gov.cn
├── 关联发改委编码
├── 上传材料
└── 领取《企业境外投资证书》

Day 26-28: 外汇登记
├── 到开户银行
├── 提交发改委+商务部批文
└── 领取《外汇业务登记凭证》

Day 29: 开立ODI专户

Day 30-180: 首笔资金出境
└── 6个月内首笔资金不低于30%
```

### 审批时限

| 部门 | 审批类型 | 法定时限 | 实际参考 |
|------|----------|----------|----------|
| 发改委 | 备案 | 7工作日 | 7-15工作日 |
| 商务部 | 备案 | 3工作日 | 3-7工作日 |
| 外汇局 | 登记 | 即时 | 1-3工作日 |

## 二、境外→境内：外商投资准入流程

### 适用条件
- 资产位置：境外
- 投资目的地：中国境内
- 投资主体：境外企业/个人

### 标准流程图

```
Step 1: 外商投资准入判断
├── 查询《外商投资准入负面清单》
├── 判断：鼓励类/限制类/禁止类
└── 限制类 → 确定外资股比要求

Step 2: 安全审查判断（如触发）
├── 是否涉及国家安全？
├── 是否涉及关键行业？
└── 如触发 → 外商投资安全审查申请

Step 3: 商务部门审批/备案
├── 鼓励类 → 商务部门备案
├── 限制类 → 商务部门核准
└── 禁止类 → 不予批准

Step 4: 行业主管部门意见
├── 矿产 → 自然资源部门
├── 制造 → 工信部门
├── 环保 → 生态环境部门
└── 其他 → 相应主管部门

Step 5: 发改委项目核准（如需）
├── 大额投资项目核准
└── 依据《企业投资项目核准和备案管理条例》

Step 6: 环评批复（如需）
└── 编制环境影响评价报告

Step 7: 市场监管登记
├── 设立新企业或股权变更
└── 领取营业执照

Step 8: 后续许可（如需）
├── 安全生产许可
├── 排污许可
└── 行业特定许可
```

### 审批时限

| 阶段 | 审批类型 | 周期 |
|------|----------|------|
| 商务准入 | 核准制 | 20-30工作日 |
| 安全审查 | 审查制 | 30-60工作日 |
| 项目核准 | 核准制 | 20工作日 |
| 环评批复 | 审批制 | 60-90工作日 |
| **总计** | — | **3-12个月** |

## 三、境外→港澳：港股上市流程

### 适用条件
- 资产位置：境外（含离岸架构）
- 投资目的地：香港
- 投资目的：在港交所上市

### 标准流程图

```
Step 1: 境内合规准备
├── 37号文登记（实际控制人）
├── 税务居民身份确认（CRS）
└── 外汇合规（资金进出通道）

Step 2: 架构搭建
├── 选定上市主体（通常为开曼）
├── 搭建上层SPV架构
├── 香港公司作为持股主体
└── 境内运营实体搭建

Step 3: 目标地法律合规
├── 开曼群岛法律尽调
├── 香港公司合规
└── 律师意见书

Step 4: 港交所上市申请
├── 递交A1申请
├── 回答监管问询
├── 上市委员会审议
└── 聆讯通过

Step 5: 中国证监会备案
├── 提交境外上市备案材料
├── 取得备案通知书
└── 向港交所提交备案

Step 6: 公开发售与上市
├── 全球发售
├── 股份在香港交易所上市
└── 完成上市

Step 7: 后续合规
├── 年度报告披露
├── CRS信息申报
└── 37号文年度申报
```

### 审批时限

| 阶段 | 周期 |
|------|------|
| 境内合规准备 | 1-3个月 |
| 架构搭建与尽调 | 3-6个月 |
| 港交所审批 | 3-6个月 |
| 中国证监会备案 | 20工作日 |
| **总计** | **6-18个月** |

## 四、境内→港澳：CEPA渠道

### 适用条件
- 资产位置：境内
- 投资目的地：香港或澳门
- 投资主体：境内企业/个人

### 特殊政策

| 政策 | 内容 |
|------|------|
| CEPA货物贸易 | 零关税待遇 |
| CEPA服务贸易 | 153个部门开放（占96%） |
| 投资便利 | "港资港法"、"港资港仲裁"（2025年新增） |

### 标准流程

```
Step 1: 确认CEPA适用性
├── 投资行业是否在CEPA开放范围内？
├── 是否符合"香港服务提供者"定义？
└── 如符合 → 享受CEPA优惠

Step 2: 参照执行备案
├── 按国令第837号参照执行
├── ODI备案流程（相对简化）
└── 与一般境外投资流程一致

Step 3: 港资港法港仲裁（如适用）
├── 投资合同适用香港法律
├── 争议在香港仲裁
└── 享受更高便利度
```

## 五、居民个人37号文登记流程（境内→境外）

### 适用条件
- 资产位置：境内
- 投资主体：中国居民个人
- 投资目的：以投融资为目的设立境外SPV

### 标准流程图

```
Step 1: 设立境外SPV公司
└── 在BVI、开曼、香港等地注册

Step 2: 准备登记材料（15-20工作日）
├── 书面申请（说明投资目的、SPV情况）
├── 37号文登记表（外汇局官网下载）
├── 身份证明（身份证+护照）
├── SPV注册文件
├── 股权结构证明
├── 资金来源证明
└── 境内权益证明（如有返程投资）

Step 3: 向外汇局提交申请
├── 户籍所在地外汇局
├── 或注册地外汇局
└── 部分地区可银行代办

Step 4: 审核（10-20工作日）

Step 5: 领取《外汇登记凭证》

Step 6: 资金合规汇出

Step 7: 年度申报（利润汇回时）

Step 8: 变更登记（变更后30日内）
```

## 六、时间周期汇总

| 流向 | 投资项目类型 | 最快 | 常规 | 复杂 |
|------|--------------|------|------|------|
| 境内→境外 | 备案制（≤3亿美元） | 25天 | 30-45天 | 60天 |
| 境内→境外 | 核准制（敏感/大额） | 90天 | 120-180天 | 180天+ |
| 境外→境内 | 鼓励类项目 | 30天 | 60-90天 | 120天 |
| 境外→境内 | 限制类+安全审查 | 90天 | 120-180天 | 240天+ |
| 境外→港澳 | 港股上市 | 6个月 | 9-12个月 | 18个月 |
| 境内→港澳 | CEPA渠道 | 30天 | 45-60天 | 90天 |
| 个人37号文 | SPV登记 | 15天 | 20-30天 | 45天 |

## 八、合规自查清单

### 境内→境外

```
□ 发改委备案通知书
□ 商务部《企业境外投资证书》
□ 外汇局登记凭证
□ 境外公司实际运营证明
□ 资金来源合法性证明
□ 年度联合年报填报
□ 存量权益登记
```

### 境外→境内

```
□ 外商投资企业备案/核准文件
□ 安全审查通过（如触发）
□ 行业主管部门意见
□ 项目核准文件（如需）
□ 环境影响评价批复（如需）
□ 营业执照
```

### 境外→港澳

```
□ 37号文登记凭证（实际控制人）
□ 港交所上市批准
□ 中国证监会备案通知书
□ CRS合规申报
□ 税务居民身份确认
```
