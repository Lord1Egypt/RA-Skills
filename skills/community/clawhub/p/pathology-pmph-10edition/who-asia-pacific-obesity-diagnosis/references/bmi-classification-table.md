# WHO亚太地区BMI分类与健康危险度标准（表7-2）

| BMI范围 (kg/m²) | 肥胖程度分类     | 健康危险度   |
|------------------|------------------|--------------|
| < 18.5           | 体重不足         | 增加         |
| 18.5 – 24.9      | 正常             | 正常         |
| 25.0 – 29.9      | 超重             | 增加         |
| 30.0 – 34.9      | 肥胖Ⅰ级          | 高           |
| 35.0 – 39.9      | 肥胖Ⅱ级          | 非常高       |
| ≥ 40.0           | 肥胖Ⅲ级          | 极度高       |

## 临床意义
- 肥胖定义为体重超过正常体重20%。
- 与多种并发症密切相关，包括：
  - 2型糖尿病
  - 动脉粥样硬化
  - 高血压
  - 脂肪肝
  - 骨关节炎
  - 胆结石
  - 血脂异常
  - 多种癌症（子宫内膜、乳腺、卵巢、前列腺、肝、胆囊、肾、结肠等）

## 注意事项
- 该标准专为亚太人群制定，因体脂分布与代谢特征不同于欧美人群，故切勿混用其他地区标准。
- 对于特殊人群（如运动员、老年人、慢性病患者），应结合腰围、体脂率等其他指标综合评估。
