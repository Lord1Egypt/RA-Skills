# VideoCraft Pro - 视频创作工坊

融合了 Video Script Creator（视频脚本）、Voice Over Script（配音脚本）、Caption/Subtitle（字幕）三大技能的优点，提供从创意到成片的完整视频创作流程。

## 为什么用 VideoCraft Pro？

- **全流程**：创意→脚本→配音稿→字幕，一条龙
- **平台适配**：抖音/快手/YouTube Shorts/B站/Reels 各有专属模板
- **钩子优化**：前3秒钩子、完播率优化、互动引导
- **字幕同步**：自动生成时间轴字幕，支持中英文

## 能力

### 1. 视频脚本生成 (Script)
- 从 `video-script-creator` 继承全部能力
- 平台专属脚本（抖音/快手/YouTube/B站）
- 分镜指导、运镜建议、BGM标注
- 前3秒钩子、完播率优化
- 系列视频规划、数据复盘

### 2. 配音脚本生成 (Voice Over)
- 从脚本自动提取配音稿
- 语气标注（热情/沉稳/幽默/专业）
- 语速标注（正常/快速/慢速）
- 断句标注（用于TTS/录音）

### 3. 字幕/标题生成 (Caption)
- 自动提取关键句作为字幕
- 标题建议（SEO优化）
- 标签/话题推荐（hashtag）
- 互动引导文案（CTA）
- 双语字幕支持

## 工作流程

```
用户需求 → 脚本生成 → 配音稿提取 → 字幕生成
                ↓
            平台适配
                ↓
       抖音/快手/YouTube/B站专属
```

## 使用方式

```
video-craft-pro> 写个抖音脚本 [主题]
  → 生成完整分镜脚本+配音稿+字幕

video-craft-pro> 这个视频配字幕 [粘贴脚本]
  → 自动提取字幕+标题+标签

video-craft-pro> 改B站风格 [脚本ID]
  → 一键切换平台风格
```
