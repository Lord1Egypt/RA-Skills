# 根据肝功能状态选择糖皮质激素

## 何时使用
- 患者需使用糖皮质激素治疗
- 存在肝功能异常，尤其是严重肝功能不全
- 药物选项包括可的松、泼尼松、氢化可的松或泼尼松龙

## 如何执行
1. **识别前药与活性药**：
   - **前药（无活性）**：可的松、泼尼松（C₁₁位为酮基）
   - **活性药**：氢化可的松、泼尼松龙（C₁₁位为羟基）

2. **评估肝功能状态**：
   - **正常或轻度异常**：可使用前药
   - **严重肝功能不全**：必须使用活性药

3. **执行选择**：
   - 若肝功能严重不全，直接选用氢化可的松或泼尼松龙
   - 避免给予可的松或泼尼松，因其无法在肝脏转化为活性形式

> 注意：治疗失败可能源于错误选择前药，尤其在肝病患者中。
