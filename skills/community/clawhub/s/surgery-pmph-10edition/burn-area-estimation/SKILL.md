# 烧伤面积估算

## 何时使用
- 需要快速评估烧伤严重程度
- 准备进行烧伤补液计算
- 判断是否符合重度烧伤标准（如成人TBSA ≥ 20%，小儿 ≥ 10%）
- 患者为成人或小儿，且烧伤部位可观察

## 执行步骤

### 1. 选择合适的方法
- **成人患者**：首选九分法
- **小儿患者**：必须使用年龄校正的九分法
- **小面积或不规则烧伤**（无论年龄）：辅以手掌法

### 2. 应用估算方法
- **九分法（成人）**：按身体区域分配百分比（头颈9%、双上肢18%、躯干27%、双下肢46%+1%会阴）
- **小儿校正**：根据年龄调整头颈和双下肢面积
- **手掌法**：单掌面 ≈ 1% TBSA；整手 ≈ 2.5% TBSA

### 3. 计算总烧伤面积
- 将各烧伤区域百分比相加，得出TBSA%
- 仅计入二度及以上烧伤面积

### 4. 注意事项
- 不适用于极小面积或难以目测的烧伤
- 手掌法禁用于手掌本身烧伤者
- 小儿不可直接套用成人九分法

## 参考文档

### 烧伤面积估算详细公式与说明

**一、九分法（成人）**

| 身体区域 | 百分比（%） | 细分 |
|----------|-------------|------|
| 头面颈部 | 9 | 发部3%、面部3%、颈部3% |
| 双上肢 | 18 | 双手5%、双前臂6%、双上臂7% |
| 躯干（含会阴） | 27 | 前躯干13%、后躯干13%、会阴1% |
| 双下肢+臀部 | 46 + 1 | 双臀（男5%/女6%）、双足（男7%/女6%）、双小腿13%、双大腿21% |

**二、小儿体表面积校正公式**

基于年龄调整：
- **头面颈部面积** = [9 + (12 - 年龄)] %
- **双下肢面积** = [46 - (12 - 年龄)] %

示例：3岁患儿，头颈 = 9 + (12 - 3) = 18%；双下肢 = 46 - (12 - 3) = 37%

**三、手掌法使用规范**
- **适用条件**：小面积、散在或不规则烧伤
- **测量方式**：患者五指并拢，掌面（不含手指背面）≈ 自身TBSA的1%
- **整手面积**：包括掌面、掌背、指缝，约等于2.5% TBSA
- **双手总面积**：约5% TBSA
- **禁忌**：手掌本身烧伤者不得使用该法

**四、临床使用原则**
1. 优先使用九分法进行快速整体评估
2. 小儿必须使用年龄校正公式，严禁直接套用成人比例
3. 手掌法仅作为补充，尤其适用于<10% TBSA的烧伤
4. 估算结果用于：烧伤严重程度分级、Parkland补液公式计算（4ml × kg × TBSA%）、决定是否需转入烧伤专科中心
