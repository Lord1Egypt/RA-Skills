# Video Prompt Master — AI视频提示词创作技能

## 核心理念

**五维度负责"想什么"，视频谱负责"怎么拍"。**

先用五维度构思整段视频的内容与情绪走向，再用视频谱把每个分镜落地成具体可执行的镜头语言。

- 详细五维度框架 → 见 [references/five-dimensions.md](references/five-dimensions.md)
- 详细视频谱要素 → 见 [references/video-spectrum.md](references/video-spectrum.md)

---

## 创作流程

### 第一步：五维度构思

按以下五个维度思考视频内容：

1. **绝对主体与物理动势** — 谁？做什么动作？速度与方向？
2. **环境场与情绪光影** — 在哪？什么氛围？光从哪来？
3. **光学与摄影机调度** — 用什么镜头？怎么拍？
4. **时间轴与状态演变** — 什么时候变？怎么变？节奏？
5. **美学介质与底层渲染参数** — 什么风格？什么规格？

### 第二步：确定输出模式

| 用户说 | 输出模式 |
|--------|---------|
| 没提分段 | 单条完整提示词 |
| 提到分段 | 按分段结构输出（见分段规则） |

分段生成详细规则 → 见 [references/segmented-rules.md](references/segmented-rules.md)

### 第三步：用视频谱落笔

每个分镜灵活选用N个视频谱要素（N=0~7，按需取用，不强求全覆盖）：

| 要素 | 内容 |
|------|------|
| 主体 | 角色/物体详细描述 |
| 景别 | 远景/全景/中景/近景/特写/大特写 |
| 动作 | 主体的运动状态与细节 |
| 场景 | 环境、光影、氛围 |
| 构图 | 对称/三分/引导线/负空间/中心/框架/对角线/黄金分割 |
| 镜头视角 | 仰视/俯视/平视/第一人称/过肩/航拍/俯冲/低角度仰拍 |
| 镜头运动 | 跟随/环绕/拉远/推进/静止/平摇/升降/甩镜/手持/变焦 |

**判断标准：这个要素用上去，画面更好吗？好就用，不好就不用。**

### 第四步：输出提示词

---

## 单段输出格式

```
拍摄手法：镜头类型 + 人称视角 + 整体节奏

总主体：详细描述（多个主体分别描述）

分镜序列：

1. 时间段
   景别 | 镜头运动/视角（如有）
   主体+场景+动作连贯描述段落（自然融入构图等要素）
   过渡：方式

2. ……

视频风格：
- 艺术风格
- 分辨率与帧率
- 渲染设置
```

**分镜字段顺序：时间 → 景别 → 描述段落 → 过渡方式**

---

## 分段输出格式

每段包含：

```
【片段N | 时间段 | 情绪基调】

角色卡（每个主体一张，完整写出不省略）
风格卡（完整写出不省略）

关键帧锚点：
- 上段结尾帧：具体画面和姿态描述
- 本段开头帧：具体画面和姿态描述
- 动作延续：无缝/有间隔

承接：接上一段结尾状态
本段：当前段内容推进
结尾锚点：当前段最后画面描述

分镜序列：
1. 时间段 | 景别 | 镜头运动/视角
   描述段落
   过渡：方式
```

段尾附：

```
衔接技巧（表格：上段结尾帧 | 本段开头帧 | 动作延续）
后期一致性三步（1.去黑帧拼接 2.色调统一 3.音效连贯）
```

---

## 核心规则速查

| 规则 | 说明 |
|------|------|
| 角色卡数量 = 主体数量 | 每个重要角色各一张角色卡 |
| 分段时角色卡/风格卡完整写 | 不写"同上"，每段完整输出方便复制 |
| 关键帧锚点要具体 | 写出谁在哪里、什么姿势、什么状态 |
| 动作延续要明确 | 上段结尾动作 = 下段开头动作 |
| 构图/视角/运镜自然融入 | 不机械罗列标签，融入描述段落 |
