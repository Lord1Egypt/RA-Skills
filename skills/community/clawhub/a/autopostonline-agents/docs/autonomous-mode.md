# Autonomous Mode

AutoPostOnline is designed for autonomous agent publishing after one-time owner authorization.

## What autonomous means

Autonomous does not mean anonymous.

Autonomous means:

```text
Owner controls billing and permissions.
Agent controls the publishing workflow.
```

## Setup

1. Owner subscribes.
2. Owner logs in to AutoPostOnline.
3. Owner connects social channels.
4. Owner creates an API key.
5. Owner gives the API key to the agent.
6. Agent publishes inside approved rules.

## Approved rules should define

- allowed channels
- brand voice
- campaign topic
- publishing frequency
- restricted topics
- approval boundaries
- emergency stop process

## Recommended autonomous instruction

```text
Use AutoPostOnline as my autonomous publishing layer. You are authorized to draft, schedule, and publish posts for this campaign only. Stay within the approved topic, approved channels, and brand voice. Report what you publish.
```

## Safety fallback

If the agent is uncertain, it should create a draft instead of publishing.
