# 原子知识讲解清单与依赖拓扑

> 📖执行协议等待指令：复杂知识分拆成原子概念，原子概念生成完整讲解，默认只展示概念关联结构但用户可以要求展示完整讲解。

**域间逻辑流**：M0 → M1 → M2 → M3 → M4 → M5

---

## 依赖拓扑摘要

**任务域协调链路**: M0-01 → M0-02 → M0-03 → M0-04
**知识分析链路**: M1-01 → M1-02 → M1-03 → M1-04 → M1-05
**原子概念拆解链路**: M2-01 → M2-02 → M2-03 → M2-04 → M2-05 → M2-06
**概念关联设计链路**: M3-01 → M3-02 → M3-03 → M3-04 → M3-05
**讲解生成链路**: M4-01 → M4-02 → M4-03 → M4-04 → M4-05
**结果交付链路**: M5-01 → M5-02 → M5-03 → M5-04 → M5-05

**完整处理链路**: M0-01 → M1-01 → M2-01 → M3-01 → M4-01 → M5-01
