# Resuscitative Herb Usage Principles

## When to Use
当患者表现为**闭证神昏**（实证），包括以下任一情况时使用：
- 热陷心包所致高热、神昏、谵语、舌红绛、脉数
- 痰蒙心窍所致神昏、喉中痰鸣、苔腻、脉滑
- 中风或中暑导致的窍闭神昏

## Execution Steps
1. **辨识闭证类型**：
   - **热闭**：选用安宫牛黄丸（含麝香、牛黄、冰片）
   - **痰闭**：选用苏合香丸
2. **选药与配伍**：
   - 麝香为开窍要药，适用于各类闭证
   - 多入丸散剂型，**不可入煎剂**（因挥发性强，久煎失效）
   - 常与牛黄、冰片、朱砂等配伍以增强开窍醒神效果
3. **用法用量**：
   - 麝香内服0.03–0.10g，仅限丸散；外用适量
4. **严格禁忌**：
   - **孕妇绝对禁用**（麝香可兴奋子宫，致流产）
   - **脱证神昏（虚证）禁用**，否则耗散正气，加重病情
   - **中病即止，不可久服**（因辛香走窜，易伤正气）

> ⚠️ 警示：必须严格区分闭证（实）与脱证（虚），误用开窍药可致病情恶化甚至危及生命。
