# 血友病凝血因子替代疗法剂量计算

## 适用对象
- 血友病A（FⅧ缺乏）患儿
- 血友病B（FIX缺乏）患儿

> **注意**：不适用于获得性血友病或血管性血友病（vWD）。

## 使用前提
1. 已确诊血友病类型及当前凝血因子活性水平（基础浓度）
2. 明确出血部位或手术需求

## 执行步骤

### 1. 确定目标浓度
根据出血严重程度或手术类型，确定所需达到的凝血因子活性水平（%）：
- 轻度出血（关节/肌肉/口腔/鼻）：目标30–50%
- 重度出血（胃肠道/腹膜后/CNS）或手术：目标40–100%

### 2. 计算首次剂量
使用对应公式计算首次输注剂量（IU）：
- **血友病A（FⅧ）**：
  ```
  首次剂量 = (目标FⅧ浓度% - 基础FⅧ浓度%) × 体重(kg) × 0.5
  ```
- **血友病B（FIX）**：
  ```
  首次剂量 = (目标FIX浓度% - 基础FIX浓度%) × 体重(kg)
  ```

### 3. 确定维持方案
- **FⅧ**：半衰期8–12小时，首剂后每8–12小时给予首剂一半量
- **FIX**：半衰期18–24小时，首剂后每12–24小时给予首剂一半量

### 4. 确定疗程
- 轻度出血：1–3天，或至出血停止
- 重度出血或手术：7–10天，或至拆线

### 5. 选择制剂
- **血友病A**：首选基因重组FⅧ或病毒灭活血浆源性FⅧ浓缩物
- **血友病B**：首选基因重组FIX、病毒灭活FIX浓缩物或凝血酶原复合物（PCC）

> 对于预防治疗，请参考 `references/prophylaxis-regimens.md`。

## 输出内容
提供以下临床可执行信息：
- 凝血因子制剂种类
- 首次剂量（IU）
- 维持剂量（IU）及给药间隔
- 总疗程（天数）
