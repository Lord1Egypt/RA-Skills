## Cara Cari Kode Wilayah Lain

1. Buka https://cuaca.bmkg.go.id/
2. Cari nama kota/desa
3. URL akan menampilkan kode adm4, contoh:
   `https://cuaca.bmkg.go.id/cuaca/{kode_adm4}`

Atau test langsung:
```bash
curl "https://api.bmkg.go.id/publik/prakiraan-cuaca?adm4=XX.XX.XX.XXXX"
```
Jika response berisi data lokasi = kode benar.
