# AI太极拳指导

全球练习者超2亿。从站桩到推手，五大流派全套指南。

## 核心能力

### 1. 太极拳理
- **太极**：无极生太极/阴阳/刚柔/虚实
- **十要**：虚灵顶劲/含胸拔背/松腰/分虚实/沉肩坠肘/用意不用力/上下相随/内外相合/相连不断/动中求静
- **松沉**：松开/松沉/松活/不僵硬
- **缠丝劲**：其根在脚/发于腿/主宰在腰/形于手指
- **意气**：以意导气/以气运身
- **太极拳≠太极操**：内劲/意/气的区别
- **健身功能**：平衡/柔韧/力量/心肺/减压

### 2. 五大流派
- **陈氏Taiji**：明·陈王廷/缠丝劲/发力/震脚/老架一路二路/新架
- **杨氏Taiji**：杨露禅/舒展大方/柔和缓慢/85式/24式简化
- **吴氏Taiji**：吴鉴泉/斜中寓正/柔化/方架/圆架
- **武氏Taiji**：武禹襄/动作紧凑/左右手各管半边
- **孙氏Taiji**：孙禄堂/融合形意八卦/开合/活步
- **流派选择**：健身→杨氏/技击→陈氏/老年人→简化24式
- **24式简化Taiji**：国家体委1956年编/入门首选

### 3. 站桩Zhan Zhuang
- **无极桩**：双脚与肩宽/微屈膝/松腰/含胸拔背/虚灵顶劲
- **混元桩**：双手抱球/意守丹田
- **三体式**：前后脚/三七分/源于形意
- **时间与效果**：从3分钟→10分钟→20分钟→30分钟
- **桩功要点**：膝不过脚尖/裆圆/尾闾中正/舌抵上腭
- **站桩反应**：腿抖(正常)/发热/发胀/出汗
- **站桩即拳**：不是热身/本身就是练拳

### 4. 套路学习
- **24式动作**：起势→野马分鬃→白鹤亮翅→搂膝拗步→手挥琵琶→倒卷肱→……→收势
- **学习步骤**：看视频→分节学(每次2-3式)→连贯→找细节
- **方向与重心**：起势正南/虚实分明/移重心不摇晃
- **呼吸**：初学自然呼吸→开吸合呼→拳式呼吸
- **速度**：24式4-6分钟/85式22-25分钟/不赶
- **常见错误**：低头/耸肩/挺胸/跪膝/摇头晃脑/身法不正
- **学拳周期**：24式(2-4个月)/陈氏老架一路(1-2年)

### 5. 陈氏Taiji专项
- **缠丝劲**：顺缠(小指领)/逆缠(拇指领)/手足缠丝
- **老架一路74式**：以柔为主/拳架较低
- **老架二路(炮捶)**：以刚为主/发力/震脚/跃步
- **发力Fajin**：松→沉→弹/全身整劲/螺旋
- **掩手肱拳**：拧腰/转胯/右拳螺旋冲出
- **金刚捣碓**：震脚/砸拳/陈氏的起势
- **单鞭**：两手一分/左手钩/全身整

### 6. 推手Tui Shou
- **单推手→双推手**：感受对方劲力/沾粘连随
- **四正手**：掤/捋/挤/按的循环
- **四隅手**：采/挒/肘/靠的补充
- **听劲**：感知对方的力/不丢不顶
- **化劲**：引进落空/四两拨千斤
- **活步推手**：加了步法的推手
- **散手San Shou**：自由应用/从套路到实战的桥梁

### 7. 太极与健康
- **太极拳与跌倒**：65岁以上练习者跌倒风险降低43%(哈佛研究)
- **平衡力**：单腿支撑/重心转换/本体感觉
- **骨密度**：太极对骨质疏松的正面影响
- **心血管**：太极的轻度有氧效果/降血压
- **慢性疼痛**：纤维肌痛/膝关节炎/下背痛的太极疗法
- **心理健康**：正念/减压/抗焦虑/睡眠改善
- **太极处方**：频率每周3-5次/每次30-60分钟

- **杨露禅偷拳**：三下陈家沟/杨无敌
