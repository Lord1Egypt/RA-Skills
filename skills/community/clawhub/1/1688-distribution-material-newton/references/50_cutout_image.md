# 抠图

## 功能说明

对商品图片进行抠图处理，生成白底图和白底裁剪图。该操作为**同步操作**，直接返回结果。

## 触发关键词

当用户意图中出现以下关键词时，**必须优先使用本 skill**，不要走图片优化流程：

> 抠图、白底图、去背景、扣图、抠背景、透明背景

## CLI 调用

```bash
python3 {baseDir}/cli.py cutout_image \
  --image_url "图片URL" \
  --prompt "用户原始query"
```

## 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| `--image_url` / `-i` | 是 | **单张**图片 URL（注意：是 `--image_url`，不是 `--image_urls`） |
| `--prompt` / `-p` | 是 | 用户原始 query，**必须原封不动传入** |

## 输入来源

- **用户直接提供了图片 URL** → 直接使用该 URL
- **用户提供了 1688 商品 ID** → 先通过商品图片查询（`references/30_product_image.md`）获取主图 URL，再传入

## 输出格式

```json
{
  "success": true,
  "markdown": "抠图完成！...",
  "data": {
    "white_img": "白底图URL（原始尺寸）",
    "white_img_cropped": "白底裁剪图URL（紧凑裁剪）",
    "mask_img": "蒙版图URL"
  }
}
```

## 输出字段说明

- `white_img`：白底图（原始尺寸）— 保留商品原始色彩和细节，背景替换为纯白色，图片尺寸与原图一致，四周留有白色空白区域。适合商品详情页展示。
- `white_img_cropped`：白底裁剪图（紧凑裁剪）— 内容与白底图一致，但做了紧凑裁剪，去掉周围多余白色空白，商品尽可能填满画面，尺寸更小更紧凑。适合做缩略图、主图或统一构图比例的场景。
- `mask_img`：蒙版图（一般无需展示给用户）

## 结果展示

向用户展示两种白底图，并说明区别：

```
抠图完成！

**白底图（原始尺寸）：**
![白底图](white_img)

**白底裁剪图（紧凑裁剪）：**
![白底裁剪图](white_img_cropped)
```

说明两者的区别，让用户根据使用场景选用：
- 白底图适合详情页展示
- 白底裁剪图适合缩略图、主图

## 异常处理

| 错误场景 | 处理方式 |
|----------|----------|
| success=false + "AK 未配置" | 触发 AK 配置流程（`references/10_ak_configure.md`） |
| 抠图结果异常 | 提示用户换一张图片重试，或检查图片是否清晰 |
