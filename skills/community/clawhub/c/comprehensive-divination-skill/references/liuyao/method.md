# 六爻推演方法

## 核心原理

六爻（火珠林）是以三枚铜钱摇卦六次，通过纳甲体系装卦，结合六亲六兽和五行生克进行断事的占卜方法。核心流程：摇卦 → 纳甲装卦 → 六亲分配 → 世应定位 → 用神选取 → 断卦。

## 摇卦流程

### 铜钱准备
使用三枚硬币/铜钱，约定正反面：
- 正面（有字/数值面）= 字
- 反面（有花/图案面）= 背

### 六次摇卦
从下往上（初爻→上爻），每次同时掷三枚铜钱：

| 背面数 | 爻象 | 符号 | 含义 |
|--------|------|------|------|
| 3个背 | 老阳 | ○ | 阳极变阴（动爻） |
| 2个背 | 少阴 | - - | 阴爻（不变） |
| 1个背 | 少阳 | — | 阳爻（不变） |
| 0个背 | 老阴 | × | 阴极变阳（动爻） |

> 传统约定：三枚均为正面 = 老阳；三枚均为反面 = 老阴。记法以"背"的数量为准。

### 记卦
从第一次（初爻）到第六次（上爻），依次记录六爻的阴阳和是否有动。

## 纳甲装卦

完成摇卦后，需要装卦（给每个爻配天干地支+六亲+世应+六神）：

### 确定卦宫
1. 对比六十四卦表，确定本卦属哪一宫
2. 记录本卦名称和变卦（如有动爻则变爻翻转）
3. 查 references/liuyao/najia.md 获取纳甲表

### 装六亲
以卦宫五行为"我"，各爻五行为"彼"，按六亲关系分配（参考 references/core/liuqin.md）

### 找世应
按世应定位表确定世爻和应爻（参考 references/liuyao/najia.md）

### 排六神
从初爻起，按日干顺排六神：甲乙起青龙，丙丁起朱雀，戊起勾陈，己起螣蛇，庚辛起白虎，壬癸起玄武。

## 断卦流程

按优先级逐层分析：
1. **选取用神**：根据占问事项确定（参考 references/core/liuqin.md）
2. **判断旺衰**：用神得生得扶为旺（吉），被克被制为衰（凶）
3. **分析动爻**：动爻对用神的生克关系，变爻的回馈
4. **综合六兽**：补充细节信息
5. **时间应期**：预估结果出现的时间窗口

详细断卦规则参考 references/liuyao/duangua.md

## 输出格式

1. **摇卦记录**：六次结果呈现在表中（爻位、背面数、爻象、标记）
2. **卦象呈现**：本卦名称 + 变卦名称（如有动爻），呈现实体卦象符号
3. **装卦表**：六爻的天干地支六亲六兽
4. **用神分析**：用神选取 + 旺衰判断
5. **综合断卦**：五行生克 + 六兽补充 + 时间应期
