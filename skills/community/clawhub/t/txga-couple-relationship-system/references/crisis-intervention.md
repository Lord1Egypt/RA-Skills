## 危机识别

### 红色警报

**关系死亡边缘的信号**：
- 任何一方提出离婚
- 长期冷战超过1个月
- 完全没有亲密
- 双方都不在乎这段关系
- 已经在分居/考虑分居

**需要立即干预的信号**：
- 任何形式的身体暴力
- 严重的情绪失控
- 任何一方有自伤/伤人风险
- 涉及儿童的安全问题

### 评估工具

**危机程度评估（1-10）**：

| 问题 | 评分 |
|-----|-----|
| 你们还有沟通吗？ | 1=完全没有 10=正常沟通 |
| 你们还有亲密吗？ | 1=完全没有 10=正常亲密 |
| 还有修复意愿吗？ | 1=完全没有 10=双方都想修复 |
| 关系对你影响的严重程度？ | 1=轻微 10=严重影响生活 |

**评估结论**：
- 得分 < 15：紧急危机，需要立即干预
- 得分 15-25：严重危机，需要积极处理
- 得分 > 25：有危机但可处理

## 紧急处理

### 立即行动（危机中）

**第一步：确保安全**
- 如果有身体暴力 → 立即离开，报警
- 如果有精神虐待 → 设立边界，寻求支持
- 如果有自伤风险 → 立即寻求专业帮助

**第二步：停止恶化**
- 停止所有指责和攻击
- 停止翻旧账
- 停止威胁（离婚/分手）
- 物理隔离（如有必要）

**第三步：稳定情绪**
- 各自冷静（至少24小时）
- 不做重大决定
- 照顾好自己的基本需求

### 紧急对话框架

**危机对话原则**：
- 不指责
- 不攻击
- 不翻旧账
- 聚焦当下

**危机对话模板**：

```
"我们现在都在情绪激动中，我们需要冷静一下。
我提议：
- 我们先分开24-48小时
- 在冷静期间我们不联系其他人（除了必要的人）
- 冷静后我们再谈

我们都想要这段关系，但现在我们无法好好沟通。
给我一点时间，等我准备好了，我会来找你谈。"
```

### 危机期间的自我照顾

**必须做的事**：
- [ ] 保证睡眠
- [ ] 正常吃饭
- [ ] 告知信任的人
- [ ] 寻求支持
- [ ] 避免酒精/药物

**不要做的事**：
- 不要独自承担
- 不要做重大决定
- 不要联系前任/暧昧对象
- 不要过度分析

## 离婚边缘

### 判断：是危机还是结束

**可以修复的信号**：
- 双方还有情感连接
- 至少一方愿意努力
- 问题具体而非价值观根本冲突
- 没有严重背叛或暴力
- 有共同利益（如孩子）

**可能是结束的信号**：
- 双方都没有意愿继续
- 价值观根本冲突
- 严重背叛（出轨/家暴）
- 长期精神虐待
- 一方持续拒绝改变

### 修复路径（离婚边缘）

**第一步：明确意愿**
- 问自己：这段关系真的没有可能了吗？
- 问自己：我尽力了吗？
- 问对方：你愿意给这段关系最后一次机会吗？

**第二步：设定条件**
- 如果对方愿意修复，需要什么条件？
- 你愿意改变什么？
- 你们能承诺什么？

**第三步：专业介入**
- 建议婚姻咨询
- 如果一方拒绝，考虑个人咨询
- 设定"最后一次机会"的时间（3-6个月）

**第四步：执行修复计划**
- 严格遵守约定
- 每周评估进展
- 如果没有进展，接受现实

### 离婚前对话

**对话模板**：

```
"我想要认真和你谈一件事。

关于这段关系，我现在的感受是...（坦诚表达）。

我想知道：
- 你的感受是什么？
- 你还愿意为这段关系努力吗？
- 如果愿意，我们怎么努力？
- 如果不愿意，为什么？

我需要你诚实地回答我。"
```

## 出轨危机

### 出轨的类型

| 类型 | 特点 | 修复难度 |
|-----|-----|---------|
| 一夜情 | 偶然发生 | 中等 |
| 短期出轨 | 持续数周-数月 | 较高 |
| 长期出轨 | 持续1年以上 | 很高 |
| 反复出轨 | 多次发生 | 极高 |

### 危机处理

**被出轨方的处理**：

**第一阶段：安全确认（0-48小时）**
- 保护自己（财务/情感）
- 不要做重大决定
- 寻求支持（朋友/家人/咨询师）

**第二阶段：信息收集（1-2周）**
- 你需要知道什么？
- 你能handle什么？
- 什么知道了会让你更痛苦？

**第三阶段：决定（1-3月）**
- 原谅 or 结束
- 如果原谅，需要什么条件？
- 如果结束，如何处理？

**出轨方的处理**：

**立即行动**：
- 停止出轨行为
- 主动坦白（如果对方需要知道）
- 完全透明（手机/行程）
- 承担责任，不找借口

**长期修复**：
- 理解这需要时间
- 持续证明自己
- 接受被质疑
- 寻求咨询帮助

### 修复协议模板

```
【出轨修复协议】

我（出轨方）承诺：
1. 完全停止出轨行为
2. 对...（具体行为）保持透明
3. 每周一次主动与伴侣沟通关系状态
4. 寻求婚姻咨询帮助
5. 如果对方需要知道，我会如实回答

我（被出轨方）承诺：
1. 给这段关系修复的时间（___个月）
2. 愿意参加婚姻咨询
3. 在我有情绪时，我会告诉你，而不是压抑

共同约定：
- 我们每月评估一次进展
- 如果修复无效，我们讨论下一步
- 我们会保护孩子（如果有）
```

## 暴力危机

### 暴力类型识别

| 类型 | 表现 | 危险程度 |
|-----|-----|---------|
| 身体暴力 | 打、推、扔东西 | 高 |
| 精神暴力 | 贬低、威胁、控制 | 高 |
| 性暴力 | 强迫性行为 | 高 |
| 经济控制 | 控制金钱、不给钱 | 中高 |
| 跟踪骚扰 | 监视、控制行踪 | 高 |

### 暴力危机处理

**立即行动**：
1. 确保自身安全
2. 如果正在发生暴力：离开、报警
3. 联系紧急联系人
4. 准备紧急物品（证件、钱、钥匙）

**紧急物品包**：
- 身份证件
- 现金/银行卡
- 钥匙
- 药物
- 儿童物品（如有）
- 重要联系人信息

**长期安全计划**：
- 告诉信任的人
- 规划逃跑路线
- 了解避难所信息
- 保留证据（照片、医疗记录）
- 考虑法律保护令

### 暴力后对话

**不推荐与施暴者直接对话**。如果有：
- 保持安全距离
- 有第三方在场
- 准备好随时离开
- 记录所有对话

## 危机后重建

### 重建信任步骤

**阶段1：透明（0-6月）**
- 随时可以询问
- 主动报告行踪
- 允许查看手机（如果需要）
- 接受监督

**阶段2：修复（6-12月）**
- 持续的正向行为
- 重新建立亲密
- 修复关系中的其他问题
- 寻求咨询帮助

**阶段3：重建（1-2年）**
- 发展新的关系模式
- 重新定义关系
- 建立安全感
- 向前看

### 重建评估

**可以继续的信号**：
- 持续的正向改变
- 双方都投入
- 信任逐步恢复
- 关系满意度提升

**需要结束的信号**：
- 没有实质性改变
- 持续的背叛感
- 无法重建信任
- 关系没有改善
