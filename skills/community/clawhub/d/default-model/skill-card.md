## Description: <br>
Set the default model persistently via /default_model command with Telegram interactive picker. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[jkyotnfjfbjnknh](https://clawhub.ai/user/jkyotnfjfbjnknh) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
OpenClaw operators and developers use this skill to inspect and change the persistent default model for new sessions, including through a Telegram picker when available. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Authorized users can change the persistent default model, affecting new OpenClaw sessions after the update. <br>
Mitigation: Install only where trusted operators are allowed to change the default model, and confirm OpenClaw authentication and Telegram callback controls are restricted to those operators. <br>
