# 友鹰数据

共 1 个工具。使用 `@工具中文名` 语法在任务提示词中调用。

### @友鹰-shopee商品选品

shopee全站、数据选品

工具中文名：友鹰-shopee商品选品
功能说明：支持Shopee全站点的商品查询，发现Shopee的爆款货源。可按关键词、类目、价格、销量、评分、上架时间、店铺等多维度筛选，覆盖马来西亚、中国台湾、印度尼西亚、泰国、菲律宾、新加坡、越南、巴西、墨西哥、智利、哥伦比亚共11个站点。

**Prompt 模板：**

> 站点：{{马来西亚}}，帮我根据"{{Storage Box}}"关键词，筛选出 {{2025-06-01}}之后上架且月销售额超过{{5000}}的 {{100}} 条符合条件的商品数据，并返回选品报告

`{{}}` 中的内容为需要替换的参数。

**参数约束：**

- **Shopee站点代码**: 必填, `malaysia`/`MY`=马来西亚, `taiwan_china`/`Taiwan_CHN`=中国台湾, `indonesia`/`ID`=印度尼西亚, `thailand`/`TH`=泰国, `philippines`/`PH`=菲律宾, `singapore`/`SG`=新加坡, `vietnam`/`VN`=越南, `brazil`/`BR`=巴西, `mexico`/`MX`=墨西哥, `chile`/`CL`=智利, `columbia`/`CO`=哥伦比亚
- **标题关键词**: maxLength 1000
- **标题查询类型**: `1`=整句语句（默认）, `2`=多个搜索词"与"关系, `3`=多个搜索词"或"关系
- **不包含标题关键词**: maxLength 1000
- **不包含标题查询类型**: `1`=整句语句（默认）, `2`=多个搜索词"与"关系, `3`=多个搜索词"或"关系
- **商品总价起始值**
- **商品总价结束值**
- **前30天销售件数起始值**
- **前30天销售件数结束值**
- **前30天销售金额起始值**
- **前30天销售金额结束值**
- **估算前30天销售件数起始值**
- **估算前30天销售件数结束值**
- **商品总销售件数起始值**
- **商品总销售件数结束值**
- **商品评分最小值**
- **商品评分最大值**
- **Ratings数起始值**
- **Ratings数结束值**
- **Favorite数起始值**
- **Favorite数结束值**
- **SKU总数起始值**
- **SKU总数结束值**
- **商品上架时间起始值(年-月-日)**
- **商品上架时间结束值(年-月-日)**
- **店铺开张时间起始值(年-月-日)**
- **店铺开张时间结束值(年-月-日)**
- **统计时间起始值**: 格式 `YYYY-MM-DD HH:mm:ss`
- **统计时间结束值**: 格式 `YYYY-MM-DD HH:mm:ss`
- **最新抓取时间起始值(年-月-日)**
- **最新抓取时间结束值(年-月-日)**
- **1级类目ID**
- **2级类目ID**
- **3级类目ID**
- **类目ID列表**: 每组ID指定完整路径，多组用`｜`隔开，如 `AAA,BBB,CCC｜DDD,EEE`
- **店铺名称或用户名称**
- **商品店铺ID列表**: 多个ID用逗号隔开
- **不包含的店铺ID列表**: 多个ID用逗号隔开
- **商品ID列表(pids)**: 单次最多500个，逗号隔开, maxLength 10000
- **店铺所在地(shippingIconType)**: `1`=海外, `0`=本地
- **发货地点(cbOption)**: `1`=跨境, `0`=本土
- **虾皮优选**: `1`=优选, `0`=非优选
- **是否官方店铺**: `1`=是, `0`=否
- **是否热销**: `1`=热销, `0`=非热销
- **店铺所在地(shopLocation)**: 文本
- **排序方式**: `rating`=评分, `price`=价格, `historical_sold`=总销售件数, `sold`=前30天销售件数, `payment`=前30天销售金额, `favorite`=Favorite数, `ratings`=Ratings数, `gen_time`=上架时间, `estimate_sold`=估算前30天销售件数
- **排序类型**: `ASC`=升序, `DESC`=降序（默认）
- **当前页码**: 默认 1
- **每页商品数**: 范围 1~1000，默认 1000
