# Box-Cox 变换

## 概述

Box-Cox变换是找到最优幂变换(λ)使数据接近正态的**系统性方法**,而非盲目尝试log/sqrt。

## 何时使用

| 条件 | 要求 |
|------|------|
| 设计类型 | 组间/组内比较、回归、ANOVA等线性模型 |
| 因变量类型 | 连续正值数据（如反应时、正确率、生理指标） |
| 样本要求 | 各组样本量≥20（λ估计需要足够数据） |
| 关键假设 | 残差正态性违反正偏态分布 |

## 公式

y(λ) = (y^λ - 1) / λ (λ≠0); log(y) (λ=0)

λ=1: 不转换; λ=0: log; λ=0.5: sqrt; λ=-1: 倒数

## R代码

```r
library(MASS)
bc <- boxcox(lm(rt ~ condition, data=data))
lambda <- bc$x[which.max(bc$y)]  # 最优λ
# 应用变换
data$rt_transformed <- (data$rt^lambda - 1) / lambda
```

## 注意事项

- 变换后效应量的解释变了(不再是原始ms尺度)
- 如果λ接近0.5或0,用熟悉的sqrt或log便于解释
- Box-Cox找到的λ可能"最优"但不好解释→有时最佳≠最实用
