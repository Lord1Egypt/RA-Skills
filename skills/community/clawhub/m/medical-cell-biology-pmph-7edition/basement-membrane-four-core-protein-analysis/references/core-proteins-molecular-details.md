# 基膜四种核心蛋白的分子特性与组装机制

## 1. Ⅳ型胶原（Type IV Collagen）
- **结构特征**：全长约400 nm，含20多处非螺旋片段（7S domain 和 NC1 domain 除外），提供可折屈性。
- **组装方式**：通过C端球状头部（NC1）的非共价相互作用形成二聚体，N端尾部（7S domain）通过共价交联形成四聚体，最终构成二维网状支架。

## 2. 层粘连蛋白（Laminin）
- **结构特征**：由α、β、γ三条不同肽链组成的异源三聚体，呈非对称十字形结构。
- **功能域**：长臂（α链）含多个结合位点，可同时结合Ⅳ型胶原、整联蛋白受体及硫酸乙酰肝素；短臂参与自我组装。
- **网络形成**：通过Ca²⁺依赖的自我聚合形成独立纤维网络，并通过巢蛋白与Ⅳ型胶原网络连接。

## 3. 巢蛋白（Entactin/Nidogen）
- **结构特征**：哑铃状，含三个球状结构域（G1、G2、G3）。
- **桥接作用**：G3区高亲和力结合层粘连蛋白γ1链，G2区结合Ⅳ型胶原α3/α4链，是两种网络的关键连接分子。
- **稳定性贡献**：虽非绝对必需（部分nidogen基因敲除小鼠仍可形成基膜），但在多数组织中对基膜稳定性和成熟至关重要。

## 4. 串珠蛋白聚糖（Perlecan）
- **结构特征**：核心蛋白约400 kDa，含5个结构域（I–V），其中域I和V富含硫酸乙酰肝素（HS）链（通常2–15条）。
- **交联能力**：可同时结合Ⅳ型胶原、层粘连蛋白、纤连蛋白及细胞表面受体（如整合素、生长因子受体）。
- **功能角色**：不仅参与结构支撑，还调控生长因子（如FGF、VEGF）的储存与释放。

## 检测建议
- **免疫荧光共定位**：使用特异性抗体标记四种蛋白，观察是否在同一基底区域共表达。
- **基因表达分析**：检测编码基因（如COL4A1、LAMA1、NID1、HSPG2）的mRNA水平。
- **功能验证**：在缺失某一蛋白的模型中评估基膜通透性、机械强度或细胞极性变化。
