# ecovacs-skills-pet-control

[English](README.md) | 中文

## 技能定位

科沃斯 **毛团（FAMIBOT）** 的 **Agent Skill**：界定助手在对话里能帮用户做什么——**查状态**、**改日常参数**、**做动作或播放内置提示音**；需配合开放平台 **AK** 使用。

本仓库描述**对话侧能力边界**，不替代开放平台完整文档。

---

## 前置条件

| 项目 | 要求 |
|------|------|
| **开放平台 AK** | 在 [open.ecovacs.cn](https://open.ecovacs.cn/)（**中国大陆**）或 [open.ecovacs.com](https://open.ecovacs.com/)（**非中国区**）「服务概览」获取，按环境注入会话。**不要**用账号密码代登录。 |
| **设备指认** | 使用 **App 展示名称中的一段话**，在一台 AK 绑定多台设备时能唯一指向某一台毛团。 |

## 怎么用（用户）

说明**哪一台毛团**（App 里的叫法）和**想做什么**即可，无需记指令名。

### 说法示例

> 「**团团**现在怎么样？心情和整体状态说一下。」  
> 「**毛团**音量多少？麦克风开着吗？」  
> 「**毛团**唤醒后要是没人理，**三分钟**就去休息。」  
> 「让 **毛团** 点一下头 / 摇尾巴 / 做个开心的样子。」  
> 「让 **毛团** **开心的叫一下**。」  
> 「让 **毛团** **开心的大叫一声**。」  
> 「让 **毛团** **跳个舞**。」（默认 `showcase_custom` 定制秀舞）  
> 「跳 **定制秀舞**。」  
> 「**毛团** **每分钟**叫一声，一共 **十次**。」（设备端定时，一次下发）  
> 「让 **毛团** **睡觉**。」 / 「把 **毛团** 上排好的定时表演**停掉**。」

对接开发见 [SKILL.md](SKILL.md) · [references/api.md](references/api.md) · `scripts/ecovacs.py`

## 当前支持的能力

实际是否可用取决于**机型、固件、部署策略与是否在线**。

### 查询（只读）

| 能力域 | 可了解的内容 |
|--------|----------------|
| **状态与性格** | 综合状态、当前情绪、角色/性格列表、性别、初始化进度 |
| **设备参数** | 麦克风、摄像头、音量、眼灯亮度、界面语言、唤醒后空闲超时 |
| **其它** | 位置/天气类信息（若开放）、最新日记摘要 |

### 设置（写入）

| 能力域 | 说明 |
|--------|------|
| **麦克风 / 摄像头 / 音量** | 开关或档位 |
| **唤醒空闲超时** | 唤醒后无人互动多久进入休息 |
| **昵称 / 唤醒称呼** | 影响大，仅在用户明确要求时修改 |
| **性别** | 宠物性别 |

性格/人设**不能通过技能修改**；需要时用上方查询了解心情与性格。

### 动作、休眠与提示音（控制）

| 能力域 | 说明 |
|--------|------|
| **单步动作** | 点头、摇头、扭头、摆尾、情绪动作等 |
| **内置提示音** | 日常说法用**情绪分类**（如 happy，网关随机选一条）；用户说**大/小/响**等强度时，技能指定 **`file`**（如 `happy-h-1`…`happy-h-5`），见 [phoenix-single-action.md §2.2](references/phoenix-single-action.md) |
| **组合表演** | 定制秀舞 `showcase_custom`（约 45s 固定脚本）— [dance-routines.md](references/dance-routines.md) |
| **休眠** | 进入睡眠 |
| **停止 / 取消** | 停止当前动作，并清除设备端**已排队**的定时任务（`reset`） |

执行可见动作前，脚本会**按需自动唤醒**（开摄像头并切回标准模式），减少「返回成功但不动」的情况。

动作编排默认：单步约 **10 秒内**、手写短编排约 **20 秒内**；预设跳舞 `showcase_custom` 约 **45 秒**（脚本自动 `user_timing`），详见 [SKILL.md](SKILL.md)。

## 异常与排障

| 现象 | 建议 |
|------|------|
| 缺少 AK / 鉴权失败 | 刷新 AK；区域与账号一致 |
| 未找到设备 | 名称与 App 对齐；待在线后重试 |
| 提示未开放 / 被拦截 | 超出本技能支持范围——见 [SKILL.md](SKILL.md) 或联系运维 |
| 动作成功但不动 | 可能仍在休眠；自动唤醒失败时在 App 手动唤醒后重试 |
| 提示音未响 | 无强度词时用 `category`；有强度词时核对 `file` 是否在 §2.2 命名表内；保留完整报错 |

详见 [references/troubleshooting.md](references/troubleshooting.md)
