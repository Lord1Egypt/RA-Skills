# 运营日历：视频号+公众号双平台协同

## 一、周内容规划总览

### 周一：全球财经热点

```
【视频号内容】
发布时间：07:00-08:00
内容类型：政策解读/热点速评
时长：1-2分钟
选题方向：
- 上周全球财经重大事件
- 宏观经济政策变化
- 资本市场动态

脚本要点：
- 3秒钩子：点名事件
- 核心影响：一句话说明
- 老胡点评：专业角度解读
- 行动建议：给观众的提醒

【公众号内容】
发布时间：10:00-11:00
内容类型：深度分析文章
字数：2500-3500字
选题方向：
- 财经事件详细分析
- 影响评估与预判
- 相关行业机会与风险
- 操作建议与注意事项

联动方式：
- 视频号发布后，文章内嵌入视频卡片
- 文章补充视频中没有的详细数据和案例
```

### 储备量标准

| 内容类型 | 储备量要求 | 说明 |
|----------|------------|------|
| 视频号 | 至少3条备用 | 确保突发情况不断更 |
| 公众号 | 至少2篇备用 | 确保内容质量稳定 |

### 发布前检查清单

```
发布前24小时：
□ 视频/文章内容审核
□ 封面图片确认
□ 标题/描述确认
□ 联动内容确认
□ 发布时间确认
□ 设备/账号正常

发布前1小时：
□ 再次预览确认
□ 评论准备（首发评论）
□ 朋友圈预热文案准备

发布后：
□ 及时回复评论
□ 转发至朋友圈/社群
□ 监测数据表现
□ 记录发布效果
```
