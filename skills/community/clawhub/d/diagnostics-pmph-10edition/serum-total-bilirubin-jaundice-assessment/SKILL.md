# 血清总胆红素黄疸分级与病因推断

## 何时使用
- 患者血清总胆红素（STB）检测值 >17.1 μmol/L
- 需对黄疸严重程度分级
- 需初步推断黄疸可能病因以指导进一步检查
- 适用对象为成人黄疸患者（新生儿不适用）

## 执行步骤
1. **黄疸分级**：根据STB浓度判断严重程度：
   - 17.1～34.2 μmol/L：隐性黄疸（亚临床黄疸）
   - >34.2～171 μmol/L：轻度黄疸
   - >171～342 μmol/L：中度黄疸
   - >342 μmol/L：重度黄疸

2. **初步病因推断**：结合STB绝对值范围推测可能类型：
   - STB通常 <85.5 μmol/L → 考虑溶血性黄疸
   - STB通常 <171 μmol/L → 考虑肝细胞性黄疸
   - STB在171～265 μmol/L之间 → 考虑不完全性梗阻性黄疸
   - STB通常 >342 μmol/L → 考虑完全性梗阻性黄疸

3. **注意事项**：
   - 重症肝炎或晚期肝硬化患者即使STB >171 μmol/L，若转氨酶显著升高，仍属肝细胞性黄疸
   - 慢性胆道梗阻可能仅表现为中度STB升高
   - 必须结合结合/非结合胆红素比例、尿胆红素、尿胆原等指标综合判断
   - 排除假性黄疸（如胡萝卜素血症）

4. **输出建议**：提示黄疸分级及可能病因，并建议进一步检测结合胆红素、尿胆红素等以确诊。
