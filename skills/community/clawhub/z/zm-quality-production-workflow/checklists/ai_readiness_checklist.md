# AI 可执行性审核表

用于判断一个 Skill / 生产包是否能让另一个 AI 一眼读懂并执行。

## 一眼看懂

- [ ] Skill 名称能表达用途
- [ ] description 说明适用场景
- [ ] 入口文件明确
- [ ] 任务类型明确
- [ ] 流程顺序明确

## 能开始做

- [ ] 有标准化输入模板
- [ ] 有最小必填字段
- [ ] 有依赖缺口模板
- [ ] 有生产计划模板
- [ ] 有可复制给 SubAgent 的执行提示词

## 能做稳

- [ ] 有样章/样页门禁
- [ ] 有必要素材清单
- [ ] 有禁止项
- [ ] 有 SubAgent 路由
- [ ] 有返工上限
- [ ] 有 BLOCKED 条件

## 能审完

- [ ] 有双 Agent 审核流程
- [ ] 有 Reviewer 报告模板
- [ ] 有 main 终审模板
- [ ] 有最终交付审核表
- [ ] 有 PASS / NEEDS_REVISION / BLOCKED 结论标准

## 能入库复用

- [ ] 有 `_meta.json`
- [ ] 有 README
- [ ] 有版本号
- [ ] 有提交记录
- [ ] 有更新记录或记忆沉淀
