# 章节生成模板

每章按此模板生成，确保结构统一、节奏可控。

---

## 标准章节模板

```markdown
# 第XX章 [章节名]

> **本章概要**：一句话概括本章核心事件
> **本章爽点**：本章主要的爽感来源（打脸/突破/获宝/逆转...）
> **情绪曲线**：[低开高走 / 层层递进 / 反转爆发 / 紧张释放]
> **涉及角色**：[本章出场的角色列表]
> **涉及地点**：[本章场景所在地]

---

（正文内容）

（开篇：承接上章 / 场景切入 / 对话开场）

（发展：矛盾升级 / 信息揭示 / 力量展示）

（高潮：本章爽点 / 冲突爆发 / 反转发生）

（收束：结果呈现 / 众人反应 / 章末钩子）

---

> **章末钩子**：[留下的悬念，引导继续阅读]
> **下章预告**：[下一章将要发生什么的简要提示]
```

## 章节生成检查清单

每章生成后自检：

### 连贯性检查
- [ ] 与上一章结尾衔接自然
- [ ] 角色状态与 `CHARACTERS.md` 一致（实力、伤势、位置、关系）
- [ ] 地点描写与 `LOCATIONS.md` 一致
- [ ] 未违反 `STORY_BIBLE.md` 中的设定规则
- [ ] 未与 `PLOT_POINTS.md` 中已有情节矛盾
- [ ] 地名、人名符合虚构化要求，未出现现代真实城市名、真实名人姓名、现实领导人姓名
- [ ] 未引入现实国家机关、现实政治历史敏感事件、军警机密或低俗色情桥段

### 质量检查
- [ ] 本章有明确的爽点或情节推进
- [ ] 对话有角色个性差异
- [ ] 章末有钩子/悬念
- [ ] 字数在 3000-4000 范围
- [ ] 主角有可感知的成长或收获
- [ ] 现实感通过细节营造，而不是直接引用真实城市名和真实公众人物姓名
- [ ] 感情互动克制，不出现性行为、露骨亲热、未成年暧昧或低俗性暗示

### 去 AI 味检查
- [ ] 本章没有高频堆叠“所有人都震惊了 / 空气仿佛凝固 / 嘴角微微上扬 / 瞳孔骤缩”这类套话
- [ ] 关键情绪主要通过动作、停顿、表情、场景反应呈现，而不是作者总结
- [ ] 对话里有偏见、遮掩、误解或嘴硬，不是人人都在准确表达中心思想
- [ ] 本章至少有一处不那么工整但更自然的节奏变化，如短句截断、反应插入、话没说完
- [ ] 本章爽点不是机械重复上一章的推进方式

### 记忆更新检查
- [ ] 新角色已记录到 `CHARACTERS.md`
- [ ] 角色状态变化已更新
- [ ] 新地点已记录到 `LOCATIONS.md`
- [ ] 关键情节已记录到 `PLOT_POINTS.md`
- [ ] 如有关键场景，已生成图解

## 特殊章节模板

### 战斗章节

```markdown
# 第XX章 [战斗章节名]

> **战斗双方**：[A] vs [B]
> **实力对比**：[A的等级/实力] vs [B的等级/实力]
> **胜负关键**：[什么因素决定了胜负]
> **爽点类型**：碾压 / 逆转 / 苦战后爆发

---

（战前：对峙、挑衅、众人不看好主角）

（战中：交手、试探、劣势、爆发）

（战后：众人震惊、对手不甘、收获战利品）

---
```

### 成长/突破章节

```markdown
# 第XX章 [成长章节名]

> **成长内容**：[从当前阶段到新阶段 / 从底层到立威 / 从弱势到掌控局面]
> **成长契机**：[战斗感悟/奇遇/项目成功/关键情报/身份翻盘]
> **外在表现**：[突破异象/舆论变化/资源到手/他人态度逆转]
> **实际提升**：[具体增强了什么能力、资源、地位或话语权]

---

（成长前：积累、铺垫、压力逼近）

（成长中：关键动作发生、局势扭转、众人反应）

（成长后：结果验证、展示新能力或新地位）

---
```

### 转折章节

```markdown
# 第XX章 [转折章节名]

> **转折类型**：身世揭秘 / 背叛 / 真相大白 / 新敌出现
> **影响范围**：[这个转折会影响后续多少章的走向]
> **情绪冲击**：[读者应该感受到什么情绪]

---

（铺垫：看似正常的日常/任务）

（异常：某个细节引起注意）

（揭示：真相浮出水面）

（冲击：角色和读者的情绪反应）

---
```
