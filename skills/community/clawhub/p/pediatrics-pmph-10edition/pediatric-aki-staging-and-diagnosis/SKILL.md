# 儿科急性肾损伤（AKI）分期与诊断

## 何时使用
当患儿出现以下情况时立即应用本技能：
- 存在AKI诱因（如呕吐、腹泻、休克、使用肾毒性药物）
- 出现少尿或血肌酐升高
- 已知基线血肌酐水平，且已排除脱水等非肾性少尿

## 执行步骤

### 1. 判断AKI是否存在（满足任一即成立）
- 血肌酐在48小时内绝对值升高 ≥ 26.5 μmol/L（0.3 mg/dl）；**或**
- 血肌酐达基线水平的1.5～1.9倍；**或**
- 尿量 < 0.5 ml/(kg·h)，持续6～12小时。

### 2. 确定AKI分期（依据KIDGO标准）
- **1期（Risk）**：血肌酐达基线1.5～1.9倍 **或** 升高 ≥26.5 μmol/L；**或** 尿量 < 0.5 ml/(kg·h) 持续6–12h。
- **2期（Injury）**：血肌酐达基线2.0～2.9倍；**或** 尿量 < 0.5 ml/(kg·h) 持续 ≥12h。
- **3期（Failure）**：满足以下任一：
  - 血肌酐达基线3倍以上，**或** 绝对值 ≥353.6 μmol/L（4 mg/dl），**或** 需肾替代治疗，**或** GFR <35 ml/(min·1.73m²)；
  - 尿量 < 0.3 ml/(kg·h) 超过24h，**或** 无尿超过12h。

### 3. 应用儿童特异性补充标准（pRIFLE）
若7天内估算肌酐清除率下降 ≥25%，**或** 尿量 <0.5 ml/(kg·h) 持续8h，也支持AKI诊断。

### 4. 进行病因鉴别
- **肾前性**：补液试验后2小时内尿量增加至6–10 ml/kg；
- **肾性**：补液及利尿剂无效；
- **肾后性**：影像学证实尿路梗阻。

### 5. 注意事项
- 避免在未评估肾功能风险前使用造影剂；
- 病因不明者可考虑肾活检。

## 输出结果
明确输出：
- AKI是否存在（是/否）
- 若存在，具体分期（1期 / 2期 / 3期）
用于指导后续治疗强度、液体管理及透析决策。
