# The Founder's Playbook

> AI-native startup lifecycle: Idea → MVP → Launch → Scale.
> Adapted from Anthropic's founder playbook for any AI agent, any platform.

## What is this?

This skill converts the full AI-native startup lifecycle framework into reusable,
stage-aware workflows. It's designed for **any AI agent** — Claude, ChatGPT,
Copilot, Gemini, or any coding assistant — to guide a founder through the
four stages of building an AI-native company.

## Quick start

```bash
# Install via ClawHub
npx clawhub install the-founders-playbook

# Or install via skills.sh (any agent)
npx skills add casvian/the-founders-playbook
```

Then just start talking naturally:

> _"I have an idea for an AI tutoring platform that adapts to each student's learning style..."_

The agent detects which stage you're in and activates the appropriate playbook.

## The four stages

| Stage | Focus | Exit Criterion |
|-------|-------|---------------|
| **Idea** | Validate before building | Problem-solution fit confirmed |
| **MVP** | Build what's necessary | Product-market fit evidence |
| **Launch** | Build the company | Repeatable revenue + operations |
| **Scale** | Build the institution | Enterprise infrastructure + moats |

## What's inside

Each stage contains:
- **Goals & exit criteria** — know when you're done
- **Key challenges** — traps the framework flags before you fall in
- **Exercises** — concrete actions to take with your AI agent
- **Stage transition detection** — automatic when criteria are met

## Claude-free

Originally published by Anthropic as "The Founder's Playbook," this adaptation
has ALL product-specific references replaced. No Claude-only features. No
platform lock-in. Works with any agent, any coding assistant.
