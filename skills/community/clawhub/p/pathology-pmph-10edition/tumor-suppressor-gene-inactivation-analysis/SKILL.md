# 肿瘤抑制基因功能丧失机制分析

## 何时使用
- 用户提供肿瘤抑制基因的突变、缺失或甲基化状态数据；
- 需评估该基因是否因双等位基因失活导致功能完全丧失；
- 解释遗传性肿瘤综合征（如Li-Fraumeni综合征）或散发性肿瘤的发病机制；
- 判断是否符合Knudson“两次打击假说”。

## 执行步骤
1. **确认适用对象**：仅适用于已知的肿瘤抑制基因（如RB、TP53、APC），且存在两个等位基因。
2. **评估等位基因状态**：检查每个等位基因是否发生以下任一事件：
   - 功能丧失性突变；
   - 纯合型缺失（homozygous loss）；
   - 启动子CpG岛高甲基化（导致转录沉默）。
3. **判断功能是否完全丧失**：若两个等位基因均失活（无论机制是否相同），则判定功能完全丧失。
4. **区分遗传性与散发性病例**：
   - 遗传性：个体携带胚系突变（一个等位基因已失活），仅需一次体细胞事件即可失活另一等位基因；
   - 散发性：需在同一体细胞内独立发生两次体细胞失活事件。
5. **输出结论**：若满足双等位基因失活条件，提示该肿瘤抑制基因功能丧失，可能驱动肿瘤发生。

## 注意事项
- 不适用于单等位基因失活但蛋白功能仍部分保留的情况；
- 不适用于原癌基因或其他非肿瘤抑制基因；
- 启动子高甲基化需通过甲基化特异性PCR或测序验证。
