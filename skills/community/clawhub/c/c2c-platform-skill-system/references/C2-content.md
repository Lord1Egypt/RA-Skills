# C2 内容运营与适配 — 内容引擎

C2C本地生活服务平台的内容引擎集群，负责条目内容生产、分类检索、推荐分发、质量治理与搜索曝光的全链路运营。核心对象为条目（Intent），涵盖PROVIDE（我提供）与SEEK（我需要）两种类型，贯穿从内容创建到用户触达的完整生命周期。

---

## C2-01 条目标题规范生成

- **输入**: 条目原始标题（必填），条目类型PROVIDE/SEEK（必填），品类分类（必填），交付形态DeliveryForm（实物/虚拟/线上/线下，必填），定价模式PricingMode（固定/面议/竞价，必填），服务商历史标题样例（可选）
- **输出**: 规范化标题（≤30字符），含核心服务词+地域/场景修饰+交付形态标注；标题质量分（0-100）；违禁词命中列表
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C2-02（描述优化需标题锚定核心语义），→ C2-04（搜索关键词从规范标题提取），→ C2-06（审核标准校验标题合规），→ C2-17（质量评分纳入标题规范度维度），→ C5-15（发布规范引用标题规则）
