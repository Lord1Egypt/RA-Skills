# 平台适配指南

## 一、Alibaba/MIC平台

### 产品描述结构
```
[SEO标题 - 包含核心关键词]

[简短描述 - 1-2句话核心卖点]

【Product Details】
- [规格表]

【Key Features】
1. [卖点1]
2. [卖点2]
3. [卖点3]

【Certification】
- [认证列表]

【Packaging & Shipping】
- [包装信息]
- [发货信息]

【FAQ】
Q: MOQ?
A: [答案]

Q: Lead time?
A: [答案]
```

### 注意事项
- 首屏300字符内包含核心信息
- 使用清晰的规格表格
- 包含FAQ减少询盘
- 图片需专业展示

---

## 二、独立站（Shopify等）

### 产品描述结构
```
[产品标题 - SEO友好]

[产品故事/价值主张]

[详细规格]
[规格表]

[应用场景描述]

[客户评价]

[相关产品推荐]

[CTA按钮：立即询价/获取报价]
```

### 注意事项
- SEO长内容（1000+字）
- 包含博客文章链接
- 强调品牌故事
- 清晰的CTA

---

## 三、Amazon B2B

### 产品描述结构
```
[产品标题 - B2B关键词]

[产品要点（5行）]

[详细描述]

[规格信息]

[Bulk报价链接]

[变体信息]
```

### 注意事项
- 遵守Amazon文案规范
- 避免极限词（#1, Best等）
- Bulk报价需清晰
- B2B价格展示

## 五、海关申报

### 产品描述规范
```
[品名] - [材质] - [用途/规格]
```

### 示例
```
LED PANEL LIGHT - ALUMINUM+PC - FOR COMMERCIAL LIGHTING
OUTDOOR FURNITURE - RATTAN - DINING SET, 6 SEATS
MECHANICAL PART - STEEL - CNC MACHINING, CUSTOMIZED
```

### 注意事项
- 使用HS编码对应品名
- 避免品牌名（需授权）
- 材质+用途描述清晰
- 符合海关规范
