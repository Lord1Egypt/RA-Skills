## Description: <br>
ClawLite Office Hours helps agents run YC Office Hours-style product discovery before coding, using Startup and Builder modes to refine problem framing, evidence, minimal entry points, metrics, and risks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[x-rayluan](https://clawhub.ai/user/x-rayluan) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, founders, and builders use this skill to clarify a product idea before implementation. It guides an agent through context gathering, targeted discovery questions, premise challenges, solution options, and a design document. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may read the current repository and git history while forming product recommendations. <br>
Mitigation: Use it in repositories where that access is acceptable and limit the agent to the intended project context. <br>
Risk: The skill may create a design or planning document in the project. <br>
Mitigation: Tell the agent where to save the document and require approval before overwriting existing files. <br>
