## 各章核心考点速查

### 第13章 财务管理基础
**核心考点：**
1. 财务管理目标：利润最大化、股东财富最大化、企业价值最大化、相关者利益最大化
2. 财务管理原则：系统性、风险权衡、现金收支平衡、成本收益、利益关系协调
3. 货币时间价值：复利终值现值、年金终值现值
4. 利率的计算：插值法、名义利率与实际利率
5. 风险与收益：资产收益率的类型、系统风险与非系统风险
6. 资本资产定价模型：R = Rf + β×(Rm - Rf)
7. 成本性态分析：固定成本、变动成本、混合成本的分解

**重要公式：**
- 复利终值：F = P×(1+i)^n
- 复利现值：P = F×(1+i)^(-n)
- 普通年金终值：F = A×[(1+i)^n - 1]/i
- 普通年金现值：P = A×[1 - (1+i)^(-n)]/i
- 资本资产定价模型：R = Rf + β×(Rm - Rf)

### 第14章 筹资管理（重点章）
**核心考点：**
1. 筹资的动机与分类
2. 债务筹资：银行借款、发行公司债券、融资租赁
3. 股权筹资：吸收直接投资、发行普通股、留存收益
4. 衍生工具筹资：可转换债券、认股权证、优先股
5. 资金需要量预测：因素分析法、销售百分比法、资金习性预测法
6. 资本成本：个别资本成本、平均资本成本、边际资本成本
7. 杠杆效应：经营杠杆、财务杠杆、总杠杆
8. 资本结构：最佳资本结构的确定（每股收益分析法、平均资本成本比较法、公司价值分析法）

**重要公式：**
- 经营杠杆系数：DOL = 息税前利润变动率 / 产销业务量变动率 = 边际贡献 / 息税前利润
- 财务杠杆系数：DFL = 普通股盈余变动率 / 息税前利润变动率 = 息税前利润 / (息税前利润 - 利息)
- 总杠杆系数：DTL = DOL × DFL
- 每股收益无差别点：(EBIT - I1)×(1-T)/N1 = (EBIT - I2)×(1-T)/N2

### 第15章 投资管理（重点章+难点）
**核心考点：**
1. 企业投资的分类和意义
2. 投资项目财务评价指标：
   - 项目现金流量（初始期、营业期、终结期）
   - 净现值（NPV）
   - 年金净流量（ANCF）
   - 现值指数（PVI）
   - 内含收益率（IRR）
   - 回收期（静态回收期、动态回收期）
3. 独立投资方案的决策
4. 互斥投资方案的决策
5. 固定资产更新决策
6. 证券投资管理：
   - 证券资产的特点和投资目的
   - 债券投资：债券价值、债券收益率
   - 股票投资：股票价值、股票收益率
   - 基金投资：基金的分类、基金业绩评价
7. 期权投资（基础概念）

**重要公式：**
- 营业现金净流量 = 营业收入 - 付现成本 - 所得税
  = 税后营业利润 + 非付现成本
  = 收入×(1-所得税税率) - 付现成本×(1-所得税税率) + 非付现成本×所得税税率
- 净现值 = 未来现金净流量现值 - 原始投资额现值
- 年金净流量 = 净现值 / 年金现值系数
- 现值指数 = 未来现金净流量现值 / 原始投资额现值
- 债券价值 = 未来利息现值 + 到期本金现值
- 股票价值（固定增长模型）：V = D0×(1+g)/(Rs - g) = D1/(Rs - g)

### 第16章 营运资金管理
**核心考点：**
1. 营运资金管理策略：流动资产投资策略、流动资产融资策略
2. 现金管理：持有现金的动机、目标现金余额的确定、现金收支日常管理
3. 应收账款管理：应收账款的功能与成本、信用政策决策
4. 存货管理：存货的成本、最优存货量的确定（经济订货基本模型、扩展模型、保险储备）
5. 流动负债管理：短期借款、短期融资券、商业信用

**重要公式：**
- 最佳现金持有量（存货模型）：C* = √(2TF/K)
- 应收账款机会成本 = 应收账款平均余额 × 变动成本率 × 资本成本
- 经济订货基本模型：EOQ = √(2KD/Kc)
- 再订货点 = 平均交货时间 × 每日平均需用量 + 保险储备
- 放弃现金折扣的信用成本率 = 折扣率/(1-折扣率) × 360/(信用期-折扣期)

### 第17章 成本管理
**核心考点：**
1. 成本管理的目标、原则和内容
2. 本量利分析与应用：
   - 成本性态分析
   - 盈亏临界点分析
   - 安全边际分析
   - 利润敏感性分析
   - 边际分析
3. 标准成本控制与分析：
   - 标准成本的制定
   - 成本差异的计算与分析（直接材料、直接人工、制造费用）
4. 作业成本与责任成本：
   - 作业成本法的原理与应用
   - 责任成本管理（成本中心、利润中心、投资中心）

**重要公式：**
- 利润 = 单价×销售量 - 单位变动成本×销售量 - 固定成本
- 单位边际贡献 = 单价 - 单位变动成本
- 边际贡献率 = 单位边际贡献/单价 = 边际贡献总额/销售收入
- 盈亏临界点销售量 = 固定成本/单位边际贡献
- 盈亏临界点销售额 = 固定成本/边际贡献率
- 安全边际量 = 实际销售量 - 盈亏临界点销售量
- 安全边际率 = 安全边际量/实际销售量
- 销售利润率 = 安全边际率 × 边际贡献率
- 直接材料成本差异 = 实际产量下实际成本 - 实际产量下标准成本
  - 价格差异 = (实际价格 - 标准价格) × 实际用量
  - 用量差异 = 标准价格 × (实际用量 - 标准用量)
- 投资报酬率 = 营业利润 / 平均营业资产
- 剩余收益 = 营业利润 - 平均营业资产 × 最低投资报酬率

   - 股利支付形式与程序

### 第19章 财务分析与评价
**核心考点：**
1. 财务分析的内容、方法和局限性
2. 基本的财务报表分析：
   - 偿债能力分析（短期：流动比率、速动比率、现金比率；长期：资产负债率、产权比率、权益乘数、利息保障倍数）
   - 营运能力分析（应收账款周转率、存货周转率、流动资产周转率、固定资产周转率、总资产周转率）
   - 盈利能力分析（营业毛利率、营业净利率、总资产净利率、净资产收益率）
   - 发展能力分析（营业收入增长率、总资产增长率、营业利润增长率、所有者权益增长率）
   - 现金流量分析
3. 上市公司财务分析：
   - 特殊指标：每股收益、每股股利、市盈率、每股净资产、市净率
   - 管理层讨论与分析
4. 财务评价与考核：
   - 杜邦分析法
   - 沃尔评分法
   - 经济增加值
   - 综合绩效评价

**重要公式：**
- 流动比率 = 流动资产 / 流动负债
- 速动比率 = 速动资产 / 流动负债
- 资产负债率 = 负债总额 / 资产总额
- 权益乘数 = 总资产 / 所有者权益 = 1/(1-资产负债率)
- 应收账款周转率 = 营业收入 / 应收账款平均余额
- 存货周转率 = 营业成本 / 存货平均余额
- 净资产收益率 = 净利润 / 平均所有者权益
- 基本每股收益 = 归属于普通股股东的净利润 / 发行在外普通股加权平均数
- 市盈率 = 每股市价 / 每股收益
- 杜邦分析核心公式：
  净资产收益率 = 营业净利率 × 总资产周转率 × 权益乘数
