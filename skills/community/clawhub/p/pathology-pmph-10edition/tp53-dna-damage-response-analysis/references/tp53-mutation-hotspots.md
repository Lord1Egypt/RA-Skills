# TP53热点突变及其功能影响

## 常见突变位点与机制
- **Arg248**：位于DNA结合域大沟，突变破坏p53与DNA特异性结合；
- **Arg273**：接触DNA小沟，突变降低转录激活能力；
- **Arg249**：结构稳定性关键残基，黄曲霉毒素B1暴露特异性诱导此位点突变（肝癌标志）；
- **Arg175**：维持核心结构折叠，突变导致构象异常和聚集。

## 失活机制分类
1. **基因突变**：约50%人类肿瘤携带TP53错义突变；
2. **病毒蛋白结合**：HPV E6、SV40大T抗原直接结合并促进降解；
3. **MDM2过表达**：泛素连接酶过度激活，加速p53蛋白周转；
4. **核输出异常**：突变导致p53滞留于细胞质，丧失转录功能。

## 临床关联
- TP53突变通常预示不良预后、化疗耐药；
- Arg249Ser突变是肝细胞癌中黄曲霉毒素暴露的分子指纹。
