## Known Risks and Mitigations: <br>
Risk: Broad trigger wording may activate this skill for generic loading, request, cache, or data-flow questions. <br>
Mitigation: Confirm the user is asking about frontend API or data-layer design before applying the skill. <br>
Risk: Generated authentication, retry, and caching code can affect user sessions, request volume, or data freshness in production apps. <br>
Mitigation: Review auth, retry, cancellation, and cache invalidation behavior against the app's backend contract before use. <br>
