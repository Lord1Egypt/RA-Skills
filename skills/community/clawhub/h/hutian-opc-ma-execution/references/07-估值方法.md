# 并购估值方法框架

> ⚠️ **法律声明**：本框架仅供参考，实际估值需根据目标公司特点、行业情况、市场环境等因素综合判断，建议聘请专业财务顾问完成。

---

## 估值方法概览

### 常用估值方法

| 方法 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| 市盈率法(P/E) | 盈利稳定企业 | 简单直观 | 受行业影响大 |
| EV/EBITDA | 重资产企业 | 排除折旧影响 | 计算复杂 |
| DCF法 | 高成长企业 | 理论严谨 | 依赖假设 |
| 市销率(P/S) | 早期/亏损企业 | 适用范围广 | 忽略成本 |
| 净资产法 | 重资产/清算 | 稳健 | 忽略未来 |
| 可比交易法 | 并购交易 | 真实市场定价 | 数据有限 |

### 方法选择建议

```
【按企业发展阶段】

| 阶段 | 推荐方法 | 备选方法 |
|------|---------|---------|
| 早期/亏损 | P/S、可比交易 | DCF（保守假设） |
| 成长期 | P/E、DCF | EV/EBITDA |
| 成熟期 | P/E、EV/EBITDA | DCF、净资产 |
| 重资产 | EV/EBITDA、净资产 | P/E |

【按行业特点】

| 行业 | 推荐方法 | 特殊考量 |
|------|---------|---------|
| 互联网 | P/S、用户价值法 | 考虑网络效应 |
| 制造业 | EV/EBITDA、P/E | 考虑资产价值 |
| 科技 | P/E、可比交易 | 考虑技术壁垒 |
| 金融 | P/B、EV | 考虑资本充足率 |

【OPC建议】
技术方通常处于成长期，建议优先使用P/E法+P/S法组合，
同时参考可比交易法进行交叉验证。
```

## 市盈率法（P/E）

### 计算公式

```
企业价值 = 净利润 × 市盈率倍数

投前估值 = 净利润 × P/E倍数
投后估值 = 投前估值 + 新增投资额
```

### 参数确定

```
【净利润选择】

1. 审计净利润
   - 优先使用经审计的合并报表净利润
   - 扣除非经常性损益
   - 考虑会计政策一致性

2. 预测净利润
   - 基于历史增长率推算
   - 参考行业增速
   - 考虑季节性因素

【市盈率倍数确定】

1. 行业平均法
   - 参考同行业上市公司平均P/E
   - 考虑规模折价（私营公司通常折价20-40%）

2. 可比公司法
   - 选择业务模式相似的可比公司
   - 调整流动性折扣

3. 谈判法
   - 基于双方商业判断
   - 考虑竞争情况

【行业参考P/E倍数】

| 行业 | 主板 | 创业板 | 科创板 |
|------|------|--------|--------|
| 制造业 | 10-15x | 15-20x | 20-30x |
| 互联网 | 20-30x | 30-50x | 50x+ |
| 医药 | 20-30x | 30-40x | 40-60x |
| 科技 | 15-25x | 25-40x | 40-60x |
| 消费 | 15-25x | 25-35x | 30-40x |
```

### 计算示例

```
【示例】
- 目标公司2024年净利润：1000万元
- 行业平均P/E倍数：15x
- 流动性折扣：30%

计算：
- 行业估值 = 1000万 × 15 = 1.5亿元
- 流动性折扣后 = 1.5亿 × (1-30%) = 1.05亿元
- 投前估值 = 约1亿元
```

## EV/EBITDA法

### 计算公式

```
企业价值（EV）= EBITDA × EV/EBITDA倍数

EV = 市值 + 净债务 + 少数股东权益 + 优先股
EBITDA = 净利润 + 所得税 + 利息支出 + 折旧 + 摊销
```

### 参数确定

```
【EBITDA计算】

净利润：1000万元
+ 所得税：250万元
+ 利息支出：150万元
+ 折旧：200万元
+ 摊销：100万元
─────────────────
EBITDA：1700万元

【倍数确定】
- 参考上市公司EV/EBITDA倍数
- 考虑增长性调整
- 考虑规模调整

【行业参考倍数】

| 行业 | EV/EBITDA倍数 |
|------|--------------|
| 制造业 | 6-10x |
| 科技 | 12-20x |
| 医药 | 10-18x |
| 消费 | 8-14x |
| 互联网 | 15-30x |
```

## DCF现金流折现法

### 计算公式

```
企业价值 = Σ [ CFt / (1+r)^t ] + TV / (1+r)^n

其中：
- CFt：第t年的自由现金流
- r：折现率（WACC）
- TV：终值
- n：预测期
```

### 参数确定

```
【自由现金流计算】
自由现金流（FCFF）= EBIT × (1-税率) + 折旧摊销 - 资本支出 - 营运资金增加

【折现率确定】
WACC = Ke × E/(D+E) + Kd × (1-T) × D/(D+E)

其中：
- Ke：权益成本（CAPM模型）
- Kd：债务成本
- E：权益市值
- D：债务市值
- T：税率

【终值计算】
- Gordon永续增长模型：TV = CFn × (1+g) / (WACC - g)
- 退出倍数法：TV = EBITn × 退出倍数

【预测期】
- 通常3-5年
- 超过预测期使用永续增长率
```

### 敏感性分析

```
【单因素敏感性分析示例】

| 永续增长率 | WACC=10% | WACC=11% | WACC=12% |
|-----------|----------|----------|----------|
| g=2% | 1.8亿 | 1.5亿 | 1.3亿 |
| g=3% | 2.0亿 | 1.7亿 | 1.5亿 |
| g=4% | 2.3亿 | 2.0亿 | 1.7亿 |

【建议】
DCF估值应结合敏感性分析，
给出估值区间而非单一数值。
```

## 市销率法（P/S）

### 计算公式

```
企业价值 = 营业收入 × 市销率倍数

投前估值 = 营业收入 × P/S倍数
```

### 适用场景

```
【适用情况】
1. 早期企业（未盈利）
2. 互联网企业
3. 高研发投入企业
4. 周期性行业

【倍数确定】
- 参考可比公司P/S
- 考虑增长率
- 考虑毛利率

【行业参考倍数】

| 行业 | P/S倍数 |
|------|--------|
| 互联网（成熟） | 5-10x |
| 互联网（早期） | 10-20x |
| SaaS | 8-15x |
| 电商 | 1-3x |
| 制造业 | 0.5-2x |
```

## 可比交易法

### 方法说明

```
【原理】
参考近期同类并购交易的估值倍数，
确定目标公司的估值水平。

【数据来源】
- 上市公司公告
- 产权交易所数据
- 并购数据库（CVSource、Wind等）
```

### 选择可比交易

```
【筛选标准】
1. 行业相同或相近
2. 交易时间接近（1-3年内）
3. 交易规模接近
4. 业务模式相似

【调整因素】
1. 规模调整（大公司通常溢价）
2. 成长性调整（高增长溢价）
3. 协同效应调整
4. 流动性调整
```

## 估值方法对比与综合

### 多方法对比表

```
【示例：某科技公司估值】

| 方法 | 估值结果 | 权重 | 加权价值 |
|------|---------|------|---------|
| P/E法 | 1.2亿元 | 40% | 4800万元 |
| EV/EBITDA法 | 1.0亿元 | 25% | 2500万元 |
| DCF法 | 1.3亿元 | 25% | 3250万元 |
| 可比交易法 | 1.1亿元 | 10% | 1100万元 |
| **综合估值** | - | - | **1.165亿元** |

建议取值区间：1.0-1.3亿元
```

### 估值调整因素

```
【正向调整因素】
□ 核心团队稳定且优秀
□ 技术壁垒高
□ 客户粘性强
□ 市场份额领先
□ 增长高于行业平均
□ 具有协同效应潜力

【负向调整因素】
□ 客户集中度高
□ 技术依赖外部
□ 行业竞争加剧
□ 盈利能力下降
□ 存在或有负债
□ 核心人员流失风险
```

## OPC视角：技术方估值要点

### 技术方估值特殊性

```
【技术方溢价因素】
1. 知识产权价值
   - 专利布局完整性
   - 技术领先程度
   - IP可替代性

2. 研发能力
   - 核心团队背景
   - 研发投入占比
   - 技术储备

3. 市场潜力
   - 技术应用场景
   - 市场空间
   - 竞争格局

4. 成长性
   - 收入增长率
   - 毛利率趋势
   - 客户拓展能力
```

### 技术方谈判策略

```
【估值谈判要点】

1. 展示核心壁垒
   - 专利组合价值
   - 技术独特性
   - 研发能力

2. 强调成长性
   - 市场空间论证
   - 客户拓展计划
   - 产品管线布局

3. 控制预期
   - 合理区间定位
   - 避免过高预期
   - 设置估值调整机制

4. 争取有利条款
   - 分期付款
   - 对赌保护
   - 激励机制
```
