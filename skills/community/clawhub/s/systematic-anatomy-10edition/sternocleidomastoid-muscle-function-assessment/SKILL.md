# 胸锁乳突肌功能评估

## 适用对象
- 胸锁乳突肌
- 头部运动控制分析
- 神经肌肉功能评估场景

## 前提条件
- 肌肉结构完整，无瘫痪
- 副神经支配正常

## 使用限制
- 不适用于肌肉断裂或副神经损伤患者
- 需排除颈椎结构性病变对头部姿势的干扰

## 执行步骤
1. **确认解剖结构**：胸锁乳突肌起自胸骨柄前面和锁骨胸骨端，止于颞骨乳突。
2. **根据收缩模式判断头部运动**：
   - **单侧收缩**：
     - 头部向同侧（患侧）倾斜（侧屈）
     - 面部（下巴）转向对侧
   - **双侧同时收缩**：
     - 头部后仰（伸展），尤其在肩胛骨固定时作用显著
3. **结合临床意义解读**：
   - 该肌由副神经（第XI对脑神经）支配，损伤可导致转头无力或典型“斜颈”
   - 与斜方肌协同完成头后仰动作
4. **注意体表特征**：该肌大部分被颈阔肌覆盖，但其轮廓通常可在颈部表面观察到。

## 输出结果
预测头部运动方向，辅助诊断副神经功能状态或解释异常头部姿势。
