## Use Case: <br>
Developers and database engineers use this skill to draft or review Neo4j Cypher query patterns for upserts, traversal, indexing, batching, parameter use, and performance pitfalls. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated Cypher could modify or delete production graph data if run without review. <br>
Mitigation: Review CREATE, MERGE, DELETE, DETACH DELETE, indexing, and batch operations before execution on a real database. <br>
Risk: Database credentials with broad privileges increase the impact of an incorrect query. <br>
Mitigation: Run generated queries with least-privilege Neo4j credentials whenever possible. <br>
