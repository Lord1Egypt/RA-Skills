# Insurance Marketing Content Assistant

**保险营销素材智能助手** — AI-powered insurance marketing content generator with built-in compliance review.

---

## What This Skill Does

Generate professional insurance marketing content (copy, images, compliance review) through natural conversation — no scripts, no technical skills required.

**Target Users**: Insurance agents, brokers, and marketing teams who need to create compliant marketing materials quickly.

---

## Core Features

| Feature | Description |
|---------|-------------|
| **🔥 Hot Topic Discovery** | Monitor policy changes, health trends, holidays, and industry news; match topics to products |
| **✍️ Copy Generation** | 5 styles: hot-topic, warm story, data-driven, urgency, professional explainer |
| **✅ Compliance Review** | Auto-check against CBIRC rules; flag P0 (must-fix) and P1 (recommended) risks |
| **🎨 Image Prompts** | Generate AI image prompts for posters, single images, or 9-grid story cards |
| **📋 Compliance Declarations** | Auto-append product-specific disclaimers (universal, participating, medical, etc.) |

---

## How to Use

**Simply describe what you need — no strict format required.**

Examples:
- "帮我生成一套保险营销素材" (Generate a complete insurance marketing package)
- "蹭一下最近的热点，推增额寿" (Use a hot topic to promote whole life insurance)
- "帮我审核这条文案" (Review this copy for compliance)
- "生成一张保险海报" (Create an insurance poster)

The assistant will guide you step by step through: **Topic → Copy → Compliance → Image → Export**

## Compliance Built-In

All generated copy automatically includes:
- Product-specific disclaimers
- CBIRC-compliant language checks
- P0 risk detection (exaggerated returns, absolute terms, competitor bashing)
- P1 risk detection (misleading comparisons, missing disclaimers, sensitive regulatory words)

# 保险营销素材智能助手

## 这是什么

通过纯对话方式生成专业保险营销素材（文案+配图+合规审核），无需任何脚本和编程技能。

**适用人群**：保险代理人、保险经纪人、保险机构运营人员

---

## 核心功能

| 功能 | 说明 |
|------|------|
| **热点发现** | 监控政策变化、健康热点、节日、行业动态，自动匹配产品 |
| **文案生成** | 5种风格可选：蹭热点型、温情故事型、数据冲击型、限时紧迫型、专业科普型 |
| **合规审核** | 自动检查CBIRC规定，P0级（必须改）和P1级（建议改）风险分级提示 |
| **配图提示词** | 生成朋友圈海报、单图、九宫格故事卡对应的AI绘图提示词 |
| **合规声明** | 根据产品类型自动追加对应免责声明 |
