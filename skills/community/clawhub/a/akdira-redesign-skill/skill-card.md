## Description: <br>
Upgrades existing websites and apps by auditing visual design, identifying generic AI patterns, and applying focused frontend polish without breaking functionality. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[akdira](https://clawhub.ai/user/akdira) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and product teams use this skill to audit and improve existing web app interfaces while preserving the current framework, styling approach, and behavior. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated frontend rewrites, visual changes, or sample content may be unsuitable for production as-is. <br>
Mitigation: Review generated diffs, placeholder imagery, invented sample content, and visual rewrites before shipping. <br>
Risk: Dependency or framework choices suggested during redesign may be unnecessary or incompatible with the existing project. <br>
Mitigation: Check the current dependency files and framework version before accepting new libraries or configuration changes. <br>
Risk: Design edits can unintentionally affect behavior, accessibility, or user flows. <br>
Mitigation: Run the project tests and manually verify key flows, focus states, image accessibility, and responsive layouts after changes. <br>
