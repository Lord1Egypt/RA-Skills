## Description: <br>
Generates and edits images through SudoCode's GPT Image 2 API, including text-to-image, image editing, multi-image fusion, size and quality controls, and local image file output. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[shengy983](https://clawhub.ai/user/shengy983) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and external users use this skill to create, edit, or merge images from natural-language prompts while controlling image size, quality, format, and output filename. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Prompts and selected reference images are sent to SudoCode for processing. <br>
Mitigation: Use the skill only with content appropriate for SudoCode processing, and avoid confidential or regulated images unless that provider handling is acceptable. <br>
Risk: API keys may be exposed if passed directly as command-line arguments. <br>
Mitigation: Prefer the SUDOCODE_API_KEY environment variable instead of passing keys on the command line. <br>
