# 效应量参考表

心理学研究中常用的效应量指标及其解释标准。

## Cohen's d（t 检验 / ANOVA 组间差异）

| d 值 | 解释 |
|------|------|
| 0.20 | 小效应 |
| 0.50 | 中等效应 |
| 0.80 | 大效应 |

**计算公式：**
```
d = (M1 - M2) / SD_pooled
SD_pooled = sqrt(((n1-1)*SD1² + (n2-1)*SD2²) / (n1+n2-2))
```

**Glass's Δ（当组间样本量差异大时）：**
```
Δ = (M1 - M2) / SD2
```

**Hedges' g（校正小样本偏差）：**
```
g = d * (1 - 3/(4*df-1))
```

## η²（偏 η²，ANOVA 效应量）

| η² 值 | 解释 |
|-------|------|
| 0.01 | 小效应 |
| 0.06 | 中等效应 |
| 0.14 | 大效应 |

**计算公式：**
```
η² = SS_effect / (SS_effect + SS_error)
```

**偏 η²（从 ANOVA 输出直接读）：**
SPSS/输出表格中直接报告

---

## ω²（Omega 平方，更保守的 ANOVA 效应量）

| ω² 值 | 解释 |
|-------|------|
| 0.01 | 小效应 |
| 0.06 | 中等效应 |
| 0.14 | 大效应 |

```
ω² = (df * (F - 1)) / (df * (F - 1) + N)
```

---

## r（相关系数 / 回归 / Cohen's d 转换）

| r 值 | 解释 |
|------|------|
| 0.10 | 小效应 |
| 0.30 | 中等效应 |
| 0.50 | 大效应 |

**d 转 r：**
```
r = d / sqrt(d² + 4)
```

**F 转 r（单因素 ANOVA）：**
```
r = sqrt(F / (F + df_error))
```

---

## R²（决定系数，回归效应量）

| R² 值 | 解释 |
|-------|------|
| 0.01 | 小效应 |
| 0.09 | 中等效应 |
| 0.25 | 大效应 |

**调整后 R²：**
```
R²_adj = 1 - (1-R²)*(N-1)/(N-k-1)
```

---

## Cohen's f（ANOVA 对方差解释程度）

| f 值 | 解释 |
|------|------|
| 0.10 | 小效应 |
| 0.25 | 中等效应 |
| 0.40 | 大效应 |

```
f = sqrt(η² / (1 - η²))
```

---

## 统计检验力 (1-β) 与样本量

研究应达到 .80 统计检验力（推荐）。

| 效应大小 | 所需 N（每组，α = .05，双尾） |
|---------|------------------------------|
| 小 (d = 0.2) | ~320 |
| 中 (d = 0.5) | ~64 |
| 大 (d = 0.8) | ~26 |

使用 G*Power 软件计算精确样本量。

---

## 中介效应效应量

常用指标：**ab 路径的标准化间接效应**

报告 Bootstrap CI（95%），不含 0 为显著。

效应量解释（标准化系数乘积）：
- 小：约 0.01
- 中：约 0.09
- 大：约 0.25

## 效应量报告检查清单

- [ ] t 检验 → Cohen's d
- [ ] ANOVA → η² 或 ω²
- [ ] 相关 → r
- [ ] 回归 → R²
- [ ] 卡方 → Cramér's V / Phi (φ)
- [ ] 中介 → Bootstrap CI + 标准化间接效应
- [ ] 始终报告置信区间
