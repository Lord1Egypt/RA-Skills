## ASCII 架构图模板库

### 模板 1：流水线对比（最常用）

```
  旧方法                      新方法
┌──────────────┐          ┌──────────────┐
│   步骤1      │          │   步骤1'     │ *
├──────────────┤          ├──────────────┤
│   步骤2      │    →     │   步骤2'     │ *
├──────────────┤          ├──────────────┤
│   步骤3      │          │   步骤3'     │
└──────────────┘          └──────────────┘
```

### 模板 2：端到端 vs 模块化

```
  旧方法（流水线）         新方法（端到端）
┌──────────────┐        ┌──────────────────┐
│  模块 A      │        │                  │
├──────────────┤        │  端到端模型      │
│  模块 B      │   →    │  (A+B+C 融合)    │
├──────────────┤        │                  │
│  模块 C      │        │                  │
└──────────────┘        └──────────────────┘
```

## 模板使用规范

1. **必含项**：4 层标题 + ASCII 图 + 一句话结构化
2. **可选项**：投资影响评估、详细评分、附录
3. **长度控制**：完整版 ≤ 150 行，简化版 ≤ 30 行
4. **格式校验**：用 `═══════════════` 边框，强化视觉锚点
5. **可读性**：每层之间空一行，关键内容用 `▎` 标注
