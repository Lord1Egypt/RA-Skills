# Platform Health Metrics Reference

## Amazon Seller Central

### Account Health Metrics

| Metric | Green | Yellow | Red (Risk Zone) |
|---|---|---|---|
| Order Defect Rate (ODR) | <0.5% | 0.5–0.8% | >1% (suspension risk) |
| Late Shipment Rate | <1% | 1–3% | >4% (suspension risk) |
| Pre-fulfillment Cancel Rate | <0.5% | 0.5–2% | >2.5% (suspension risk) |
| Valid Tracking Rate (FBM) | >99% | 96–99% | <95% |
| Invoice Defect Rate (B2B) | <0.5% | 0.5–1% | >5% |

### FBA-Specific Metrics

| Metric | Target | Warning | Action |
|---|---|---|---|
| IPI (Inventory Performance Index) | >450 | 400–450 | <400 = storage limits imposed |
| Stranded Inventory | 0% | 1–3% | >5% = immediate action |
| Aged Inventory (365+ days) | 0 units | <5% of total | >10% = disposal fee risk |

### Listing Health

| Issue | Impact | Fix |
|---|---|---|
| Suppressed listing | 0 impressions, 0 sales | Fix root cause attribute within 24–48h |
| Inactive ASIN | Hidden from search | Reactivate or resolve attribute |
| A9 algorithm demotion | Reduced organic rank | Optimize main image, title, bullets |

## TikTok Shop

### Account Health Score Components

| Component | Weight | Target |
|---|---|---|
| Order Completion Rate | High | >95% |
| Response Rate | High | >90% (95%+ for top seller) |
| Item Not as Described Rate | High | <3% |
| Logistics Performance | Medium | Dispatch within 48h |
| Shop Rating | Medium | >4.5 stars |

### Promotional Eligibility Requirements (typical)
- Account health score: ≥90/100
- No active major violations
- Minimum order history: 30+ orders
- Response rate: ≥95% for Flash Deals

## Shopee / Lazada

### Shopee Seller Metrics

| Metric | Target | Notes |
|---|---|---|
| Response Rate | >85% (Preferred: >95%) | 24h window |
| Preparation Time | <2 days | Affects listing search rank |
| Non-Fulfillment Rate | <2% | Above 5% = penalty |
| Return/Refund Rate | <5% | Category-specific thresholds |
| Shop Rating | >4.5 | Below 4.2 = demotion risk |

### Preferred and Preferred+ Requirements
- Preferred: 85%+ response rate, <2% NFR, 4.5+ rating, 30+ orders/90 days
- Preferred+: 95%+ response rate, <1% NFR, 4.8+ rating, 100+ orders/90 days
