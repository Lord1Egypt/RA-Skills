# 质量检查清单

## 内容准确性

- [ ] 是否准确识别论文核心问题？
- [ ] 是否区分了作者的结果和自己的解释？
- [ ] 是否保留关键数据和实验条件（如样本量 n、P 值、置信区间）？
- [ ] 是否避免夸大因果结论？
- [ ] 方法部分是否完整翻译（尤其 Nature/Science/Cell 等期刊）？

## 图文对应性

- [ ] 每张图是否放在对应讲解位置？
- [ ] 是否说明每张图回答的问题？
- [ ] 是否解释每个小图（subfig a–h）和结论之间的关系？
- [ ] 是否避免图片与文字错位？
- [ ] 图题和图注是否翻译为中文？
- [ ] 图内英文标签是否保留并说明原因？

## 逻辑链完整性

- [ ] 是否形成"研究问题 → 实验设计 → 图中证据 → 作者推理 → 得出结论"的因果链？
- [ ] 是否避免了孤立堆砌结果？
- [ ] 图1–图4是否串联成完整故事？

## 翻译与表达

- [ ] 专业术语是否统一？（首次出现时附英文原文）
- [ ] 中文是否通顺？无明显翻译腔？
- [ ] 英文长句是否拆分为符合中文习惯的句子？
- [ ] 图表编号是否与原论文一致（如 Fig. 1 → 图1）？
- [ ] 参考文献信息是否保留英文格式？

## 文件输出（若适用）

- [ ] Word 版是否可编辑？排版是否正常？
- [ ] PDF 版排版是否正常？图片是否清晰？
- [ ] 图片编号是否与原论文一致？

## 通俗性检查

- [ ] 是否使用通俗语言解释复杂概念？
- [ ] 是否用了比喻？比喻是否准确？
- [ ] 是否避免了"这张图证明了"的绝对化表述？
- [ ] 每个技术术语首次出现时是否解释？
