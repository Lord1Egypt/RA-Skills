# 宫颈细胞学样本采集与满意度判定

## 何时使用
- 患者需进行子宫颈癌筛查，且无急性下生殖道炎症、非月经期、24小时内未有性生活/冲洗/用药；
- 已完成宫颈细胞样本采集、制片与染色，准备进行TBS分类报告前。

## 执行步骤

### 第一部分：规范采集宫颈细胞样本
1. **确认采集条件**：确保患者符合以下前提：
   - 无急性下生殖道炎症；
   - 处于非月经期；
   - 采样前24小时内未进行性生活、阴道冲洗或局部用药。
2. **清洁宫颈**：用无菌棉签轻轻拭去宫颈表面分泌物。
3. **使用专用宫颈刷**：选用中间刷毛长于两侧的宫颈刷，将长刷毛插入宫颈管，短刷毛覆盖宫颈外口，确保覆盖整个转化区。
4. **旋转取样**：向单一方向柔和旋转至少5周，保证充分接触宫颈上皮。
5. **样本保存**（任选其一）：
   - 将细胞刷在保存液中充分洗脱；
   - 将刷头直接放入含保存液的容器（适用于LBC）；
   - 将样本均匀涂抹于载玻片并立即固定（传统涂片）。
6. **优先选择LBC技术**：因其可去除血液/黏液干扰、形成单层细胞涂片、支持自动阅片及HPV检测。

### 第二部分：TBS分类法样本满意度判定
1. **评估细胞数量**：
   - **传统涂片**：需含8000–12000个保存良好、可辨认的鳞状上皮细胞；
   - **液基涂片（LBC）**：需含≥5000个保存良好、可辨认的鳞状上皮细胞。
2. **检查例外情况**：若涂片中检出任何异常细胞（如ASC、AGC），无论细胞数量多少，均视为“满意”。
3. **评估其他质量因素**：
   - 是否含有宫颈管细胞或移行区成分（反映取材完整性）；
   - 是否存在大量血液、炎症细胞或黏液干扰。
4. **判定结果**：
   - 符合上述标准 → 标记为“满意”；
   - 不符合且无异常细胞 → 标记为“不满意”，注明原因（如细胞量不足、固定不良等）；
   - “不满意”样本不得用于排除上皮内病变，必须重新采样。

> 注意：样本满意度评估是TBS报告的第一步，必须在描述细胞学发现前完成。
