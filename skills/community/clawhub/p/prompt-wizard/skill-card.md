## Description: <br>
Generates high-quality English prompts for ChatGPT Image 2 with conversational, structured, style-anchored, and case-browsing modes. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ericarcha](https://clawhub.ai/user/ericarcha) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Creators, marketers, designers, and developers use Prompt Wizard to turn image ideas into English ChatGPT Image 2 prompts, with case-grounded style choices, multilingual conversation, and optional batch generation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Bundled prompt examples and generated outputs may include unsafe, deceptive, or non-consensual image concepts. <br>
Mitigation: Review generated prompts and case references before use; remove examples that support forged records, fake social proof, non-consensual candid imagery, or sexualized young-looking subjects. <br>
Risk: The update-library command runs a local script that clones an upstream prompt library and replaces local library files. <br>
Mitigation: Review the updater and upstream changes before execution; run it only in environments where git, jq, network access, and local file replacement are acceptable. <br>
Risk: Image prompts may describe text, logos, or branding that the target image model renders unreliably. <br>
Mitigation: Treat generated text and logo instructions as drafts; add precise typography, logos, or compliance-sensitive brand elements manually after image generation. <br>
