# 免疫相关单克隆抗体药物识别

## 何时使用
- 药物名称包含“单抗”或英文名以“-mab”结尾
- 药物出现在教材第50章附近页码（440–467）
- 上下文涉及肿瘤免疫治疗、自身免疫病治疗或生物制剂分类

## 执行步骤
1. **检查命名规则**：
   - 中文名是否以“某某单抗”结尾（如帕博利珠单抗）
   - 英文名是否以“-mab”结尾（如pembrolizumab）
2. **核对页码范围**：确认该药物在教材中的页码是否落在440–467区间
3. **综合判定**：若同时满足命名规则和页码条件，则判定为免疫相关单克隆抗体药物
4. **输出提示**：返回结果模板：“提示该药物为免疫相关单克隆抗体，建议查阅对应页码了解其免疫调节机制及临床应用”

> 注意：本技能不适用于小分子化学药物、疫苗或传统免疫增强剂。
