# 心绞痛联合用药策略

## 何时使用
- 心绞痛患者单药控制不佳
- 合并高血压或运动性心动过速
- 需减少单药剂量以降低副作用

## 联合方案选择

### 方案一：β受体阻断药 + 硝酸酯类（如普萘洛尔 + 硝酸异山梨酯）
- **协同机制**：
  - β阻断药对抗硝酸酯引起的反射性心率加快
  - 硝酸酯对抗β阻断药引起的心室容积增大
- **优势**：互相取长补短，剂量可减，副作用减少
- **给药原则**：口服，从小剂量开始，逐渐加量
- **停药注意**：β阻断药必须逐渐减停，防反跳性心绞痛

### 方案二：β受体阻断药 + 钙通道阻滞药
- **首选组合**：β阻断药 + **硝苯地平**（二氢吡啶类）
  - β阻断药消除硝苯地平引起的反射性心动过速
  - 硝苯地平对抗β阻断药的外周血管收缩
  - 适用于心绞痛合并高血压者
- **禁忌/慎用组合**：
  - **避免**β阻断药 + **维拉帕米/地尔硫䓬**（非二氢吡啶类）
  - 原因：二者均有负性肌力和负性传导作用，合用可致心衰或传导阻滞

## 监测要点
- 密切监测血压：过度降压会减少冠脉灌注，加重心绞痛
- 监测心率、心电图及心功能（尤其合用非二氢吡啶类时）

> 禁忌：严重心功能不全、支气管哮喘、显著心动过缓者禁用β阻断药。
