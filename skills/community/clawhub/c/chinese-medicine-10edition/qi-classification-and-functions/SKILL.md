# Qi Classification and Functions

在中医理论框架下，气是维持生命活动的基本物质和动力。根据来源、分布与功能差异，人体之气主要分为四类：元气、宗气、营气、卫气。执行以下步骤以正确应用该分类体系：

1. **确认适用场景**：仅用于中医辨证论治、中医生理学分析或基于脏腑经络理论的诊断推理，不适用于现代解剖生理学或非中医体系。

2. **识别气的类型及其关键特征**：
   - **元气**（原气、真气）：
     - 生成：源于肾中先天之精，赖后天水谷精气滋养。
     - 分布：通过三焦通行全身，内达脏腑，外至肌肤腠理。
     - 核心功能：推动和调节生长发育与生殖；激发脏腑、经络、形体、官窍的生理活动。
   
   - **宗气**：
     - 生成：由肺吸入的清气与脾胃化生的水谷精气结合而成。
     - 分布：积于胸中“气海”（膻中），上出喉咙贯心脉，下注丹田及足阳明气街。
     - 核心功能：司呼吸（影响语言、声音）；行气血（影响心搏、脉象）。
   
   - **营气**：
     - 生成：源自脾胃运化的水谷精微之精华部分。
     - 分布：行于脉中，循脉上下，营运周身。
     - 核心功能：化生血液；营养全身脏腑经络、形体官窍。
   
   - **卫气**：
     - 生成：来自水谷精微中的剽悍滑利部分。
     - 分布：行于脉外，布散于皮肤腠理及胸腹脏腑之间。
     - 核心功能：防御外邪；温养全身以维持体温；调控腠理开阖以调节汗液排泄。

3. **应用判断**：根据临床表现或生理现象，匹配对应的气类型及其功能异常，为辨证提供依据。例如：
   - 呼吸微弱、声低懒言 → 宗气不足
   - 自汗、易感外邪 → 卫气不固
   - 生长发育迟缓、生殖功能减退 → 元气亏虚
   - 血虚萎黄、脏腑失养 → 营气不足

4. **注意事项**：
   - 必须以中医整体观念和脏腑经络理论为基础。
   - 避免脱离语境单独使用某一类气的功能解释复杂病症。
   - 若涉及多气同病（如营卫不和），需综合分析。
