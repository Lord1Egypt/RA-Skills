# 神经内分泌轴反馈调节分析

## 何时使用
当临床或实验室数据提示以下任一神经内分泌轴存在激素水平异常时：
- 下丘脑-垂体-甲状腺轴（HPT）
- 下丘脑-垂体-肾上腺轴（HPA）
- 下丘脑-垂体-性腺轴（HPG）
- 生长轴（GH-IGF-1）

特别适用于评估原发性（靶器官病变）与继发性（下丘脑或垂体病变）内分泌功能障碍。

## 执行步骤

1. **确认轴系完整性**：验证下丘脑能分泌相应释放激素、垂体能分泌促激素、靶器官具备相应受体。

2. **追踪激素通路**：
   - 下丘脑分泌释放激素（如TRH、CRH、GnRH、GHRH/SS）→ 垂体门脉系统 → 腺垂体
   - 腺垂体分泌促激素（如TSH、ACTH、LH/FSH、GH）→ 血液循环 → 靶器官
   - 靶器官合成终末激素（如甲状腺激素、皮质醇、性激素、IGF-1）

3. **评估负反馈机制**：
   - 若终末激素升高但促激素未受抑制，提示反馈通路受损或中枢病变
   - 若终末激素降低而促激素代偿性升高，提示原发性靶器官功能减退
   - 若终末激素与促激素均降低，提示继发性（垂体或下丘脑）功能减退

4. **考虑节律与应激影响**：
   - 检查采样时间是否符合激素昼夜节律（如皮质醇清晨达峰）
   - 排除急性应激对HPA轴的非反馈性激活

5. **识别生长轴特例**：
   - GH作用于肝脏生成IGF-1，IGF-1促进长骨生长
   - GH和IGF-1共同反馈抑制GHRH并刺激生长抑素（SS）分泌

6. **输出判断**：基于激素动态关系，明确功能障碍层级（下丘脑、垂体或靶器官）及性质（原发性或继发性）。

> 注意：本技能不适用于非内分泌调节系统，或孤立激素异常而无轴系关联的情况。
