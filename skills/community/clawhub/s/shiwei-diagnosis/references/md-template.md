# 诊断报告 MD 模板

此模板定义诊断报告的标准章节结构和写作规范。最终MD文档参照此结构，内容可因企业行业和具体情况调整。

---

## 报告结构总览

```
封面
目录
前言：项目说明
第一部分：历史回顾和成功经验
  1. 发展回顾
    1.1 成长阶段和主要事件
    1.2 主要成就
  2. 关键成功要素
    2.1-2.N 各要素展开
  3. 文化归因
    3.1-3.N 文化品格
  4. 成功经验小结
    4.1 基本判断
    4.2 问卷验证（可选）
第二部分：现实问题和成长制约
  1. 问题表征
    1.1 财务指标分析
    1.2 经营压力（业绩/竞争/发展）
  2. 问题原因
    2.1 战略和运营
    2.2 组织能力
    2.3 领导体制和组织文化
  3. 问题根因
    3.1 问题的性质
    3.2 三大矛盾
第三部分：战略性课题与变革建议
  1. 外部环境系统分析（可选）
  2. 对企业的基本判断
  3. 转型变革方向
  4. 变革指导思想与路径
结语
```

## 各章节写作规范

### 前言：项目说明

**固定内容块**：

```
## 项目说明

### 资料分析
- 公司发展历程及大事记
- 公司年报/年度工作报告
- 主要领导讲话资料
- 其他相关行业文献/网站资料

### 深度访谈
- 一对一访谈中高层管理人员共计 XX 人
- 员工代表座谈会 X 场，共 XX 人
- 访问分子公司：XXX、XXX

### 问卷调研
- 根据访谈信息设计面向中高层干部的调查问卷
- 回收有效问卷 XX 份
```

如分阶段执行（诊断→战略梳理→方案落地），在第一页展示项目时间线和阶段划分。

---

### 第一部分：历史回顾和成功经验

#### 1. 发展回顾

**1.1 成长阶段和主要事件**

- 首先梳理行业发展历程（可选，对To B制造业尤为重要）
- 将企业发展分为3-5个阶段，每个阶段标注年份、特征、标志性事件
- 用时间轴图表示

**写法示例**：
```
回溯成长过程，公司经历了 X 个大的发展阶段：
- 第一阶段（Year-Year）：创业探索。特征：XXX。标志事件：XXX。
- 第二阶段（Year-Year）：XXX。特征：XXX。标志事件：XXX。
...
```

**1.2 主要成就**

按维度展开（根据行业调整）：
- 业务发展：规模/增速/行业地位
- 产品/技术：覆盖范围/领先性
- 能力建设：价值链完整度
- 市场地位：市场份额/国际化

每个成就必须有数据支撑（营收图、市场份额、批签发量等）。

#### 2. 关键成功要素

抽取5-7个关键成功要素，每个要素独立一节。

**分析方法："赢的逻辑"七维框架**

在识别关键成功要素时，使用以下七维框架逐层下钻分析，而不是简单罗列要素名称。从宏观到微观，每一层回答一个"为什么赢"：

```
赢在赛道（业务选择：为什么是这个行业？进入时机如何？）
  └→ 赢在空间（市场定位：为什么是这个客群/区域？）
       └→ 赢在价值（价值主张：给客户解决了什么？为什么买单？）
            └→ 赢在模式（商业模式：怎么赚钱？盈利逻辑是什么？）
                 └→ 赢在胜点（致胜因素：凭什么比竞争对手强？）
                      └→ 赢在节奏（战略展开：什么先做、什么后做？顺序逻辑？）
                           └→ 赢在能力（关键专长：哪些组织能力支撑了以上六点？）
```

根据行业和企业情况，7层不一定全部展开——但至少覆盖**赛道、价值、胜点、能力**四层。最终从七维分析中抽象出5-7个关键成功要素作为独立章节。

**写法模式**：
```
## 2.X 要素名称
[一句话总括]：[具体描述]

### 分论点1（如：事实陈述）
[数据/案例支撑]

### 分论点2（如：战略意义）
[分析]
```

**典型要素类型**（根据行业调整）：
- 聚焦赛道（坚守主业/战略聚焦）
- 前瞻布局（产品管线/技术路线的前瞻性）
- 技术/研发驱动
- 产业转化/能力建设
- 资源整合/合作
- 团队培养
- 规则意识/合规
- 市场定位（差异化/国产替代/利基市场）

#### 3. 文化归因

成功要素背后的文化和领导力根源。每个要素对应一种文化品格。

**写法模式**：每个文化品格一页PPT（一章节），左侧是品格名称，右侧是具体表现。

#### 4. 成功经验小结

**4.1 基本判断**

用一段有文学性的总结收束：
```
纵观 [企业名称] 的发展历程和成功经验，我们可以得出结论：
- [企业名] 是 [核心定义1] 的企业；
- [企业名] 是 [核心定义2] 的企业；
- [企业名] 是 [核心定义3] 的企业。
```

**4.2 问卷验证（可选）**

引用问卷中"成功因素排序"题目的结果，如：

> 第30题、您认为公司过去取得成功的核心关键因素是什么？[多选，选5项]
> 选项1：XX%（最高）...

### 第二部分：现实问题和成长制约

> **分析方法：问题根因三层分析法**。从财务/经营的表征出发（What），逐层深入到战略/组织/人才/文化的具体现象（How），最终归因到三大基本矛盾（Why）。这是施炜诊断体系的核心递进逻辑——不跳跃、不跳跃归因、每一层都基于前一层的事实和数据。

#### 1. 问题表征

**1.1 财务指标分析**

- 营收/利润变化趋势（3-5年）
- 毛利率/净利率对比
- 研发费用率/销售费用率对比
- 与同行对比分析
- 必须输出财务结论列表：

```
通过对公司财务状况的分析，可以得出以下基本结论：
① 结论一
② 结论二
...
```

**1.2 经营压力**

三个维度（按行业调整）：

- **业绩压力**：收入利润下滑、单一品种依赖、海外进展不及预期
- **竞争压力**：主力品种被冲击、竞品迭代更快、研发进度落后
- **发展压力**：管线不聚焦、基地空转、核心能力稀释

每项压力必须包含：访谈摘录 + 事实数据/结论。

#### 2. 问题原因

**2.1 战略和运营**

**2.1.1 战略管理**

核心诊断句式：
> 战略制定到战略落地的长周期价值循环没有打通。

具体分析维度：
- 战略规划是否真正落地（目标是否过于乐观？是否根据现实调整？）
- 资源是否有效配置到战略方向（是否聚焦？是否超出资源约束？）
- 战略管理是否形成闭环（制定→解码→执行→监控→复盘）
- 对外投资是否有管理闭环
- 战略部门职能是否完备

**2.1.2 运营效率**

核心诊断：
- 研发→注册→生产→营销各环节是否存在脱节
- 经营分析体系是否建立（定期会议、目标责任、绩效考核）
- 横向协同机制是否有效（点对点 vs 制度化沟通）
- 流程效率（长度、节点数、审批逻辑）

**每一节标准格式**：

```
#### 子问题标题
##### 访谈整理
[引用多条访谈原话，每条前面加项目符号]

##### 事实和结论
[基于访谈+问卷+资料的判断]
```

**2.2 组织能力**

核心诊断维度：

- **战略规划与组织能力不匹配**：管理/专业能力达不到战略要求、人才密度不足、后备干部断层、考核激励不健全
- **专业性职能支撑薄弱**：总部职能发育不足（"一半属于瘫痪"）、职能分得过散过细、研发管理体系不健全、质量管理体系有改进空间

**2.3 领导体制和组织文化**

- **高管团队和决策机制**：是否形成真正的高管团队？是否有民主决策机制？授权是否充分？
- **组织文化**：创业精神是否流失？是否出现官僚化？是否自满封闭？管理是否"宽松有余、压力不足"？

**问卷验证**

每个分析维度结束时，必须附问卷验证：
- 相关维度的打分表（与平均线对比）
- 分层打分（高管vs中基层）
- 开放题的选项分布
- 简要分析

#### 3. 问题根因

**3.1 问题的性质**

先对问题定性：

```
普遍性：问题属于民营企业普遍出现的问题
整体性：问题不是局部的，而是全局、结构性的
顶层性：与治理结构、领导机制有关
长期性：不是一朝一夕可以解决的
深层性：涉及深层次认知和行为习惯
```

然后给出企业发展阶段锚定（基于问卷得分）。

**3.2 三大矛盾**

```
### 矛盾一："业务——能力"的矛盾
我国民营企业的发展往往呈现螺旋式上升的态势，一方面是业务的增长，另一方面是能力的提升，往往是业务增长在前，能力成长跟随，二者往往不匹配。当机会出现，能力能支撑，企业就会高速成长；当能力不能有效支撑时，业务增长就会出现瓶颈甚至倒退。

[企业名]同样没有摆脱这一规律。能力主要由技术资源、组织资源、人力资源构成... [企业名]业务能力矛盾主要表现在...

### 矛盾二："成长——治理"的矛盾
[企业名]经过二十多年的发展...企业经营规模和经营效益都上了一个台阶...但是治理方式仍然是创业型小企业模式。缺少高管团队，缺少民主决策机制，缺少职能管理...

当企业外部市场机会失去，竞争加剧时，治理方式不进化，企业就会倒在机会成长阶段...

### 矛盾三："环境——进化"的矛盾
企业进化的本质是对环境的适应...对[企业名]领导层来讲，在价值观需提炼新的价值主张和组织原则，并达成共识；在认知（战略思维）上，需要做出以下改变（X大转向）：
① 从XX转向XX；
② 从XX转向XX；
...
```

### 第三部分：战略性课题与变革建议

#### 1. 外部环境系统分析（可选，结构完整的报告包含）

- 宏观环境挑战
- 市场机会洞察（国际化机会、产业变迁机会、产品创新机会等）
- 标杆案例研究

#### 2. 对企业的基本判断

使用"四句话"格式给出总体定性判断。这是诊断的最终结论，必须有力、有判断：

```
四句话总结：
1. [企业名] 曾经是 [核心定义1]；
2. [企业名] 基础犹在（列出核心优势/资产）；
3. [企业名] 问题严重（点出最关键的1-2个问题）；
4. [企业名] 前景可期（在什么条件下可以扭转）。
```

**示例（沃森案例）**：
> 成就辉煌，基础犹在，问题严重，前景可期。

**"灵魂之问"**（基于沃森案例的延伸范式）

在四句话后面，可抛出5-7个尖锐的"灵魂之问"，直指企业最深层的矛盾。这些问题不是用来解答的，而是用来激发管理层的深层反思：

```
灵魂之问：
① 为什么成功项目一切顺利，而新项目屡屡不顺？
② 为什么方向总体正确，但执行总是走样？
③ 为什么我们的文化宣称是XX，但实际行为却是YY？
④ 为什么总是"起个大早赶个晚集"——看到机会却抓不住？
⑤ 为什么过去成功的逻辑，在今天不再有效？
⑥ 优秀的人才为什么选择离开？
⑦ [可针对企业具体情况追加1-2个定制化问题]
```

> "灵魂之问"的原理：这些问题通常没有标准答案，但问题的提出本身已经在引导管理层的认知重构。在PPT中，灵魂之问可单独一页呈现，用加粗字体和留白制造冲击力。

后面跟上深层原因分析（为什么成功项目顺利而新项目不顺？为什么投资失败？为什么价值流失？）

#### 3. 转型变革方向

明确四个转型：
1. 从机会成长→能力成长
2. 从中小企业→中大型企业
3. 从国内为主→面对全球（如适用）
4. 从个人驱动→组织驱动

#### 4. 变革指导思想与路径

**变革指导思想**（沃森案例中的五条）：
- 强化使命驱动
- 强化战略牵引
- 提升组织力量
- 建立高效体系
- 重建心智地图

**变革路径**：以三年为期，分年度列出关键动作。

具体操作思路：以三年为期，以战略转型为起点，从组织、干部、文化、流程多个方面进行系统升级，完成公司的管理整合，构建起支撑未来成长的管理体系。

---

## 写作要则

1. **访谈原文是灵魂**：每个问题的诊断必须从访谈整理出发，再给事实和结论。
2. **数据交叉验证**：访谈观点 + 财务数据 + 问卷打分，三角交叉。
3. **结论有判断力**：不做模棱两可的表述。明确说"是"或"否"。
4. **分层分析**：高管vs中基层、总部vs分子公司、不同部门。
5. **以终为始**：所有分析最终指向三个基本矛盾和变革方向。
6. **给台阶不给面子**：说真话，但用企业的语言体系包装，不冒犯。
7. **文化归因先于问题诊断**：先说成功要素和文化根源，再说问题。
