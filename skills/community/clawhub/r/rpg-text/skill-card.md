## Description: <br>
RPG Text helps an agent run a Chinese text role-playing game as a dungeon master using D&D-inspired rules, reference data, and turn-based adventure guidance. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[beancookie](https://clawhub.ai/user/beancookie) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users can use this skill to have an agent act as a dungeon master for a turn-based Chinese text RPG. The skill supplies character creation, exploration, combat, inventory, quest, shop, time, save-state, monster, item, and spell references for narrative play. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Fictional game progress, character names, and roleplay notes may be retained or summarized as part of game state. <br>
Mitigation: Avoid entering real personal details into character names, notes, or roleplay text. <br>
Risk: Generated RPG rulings and outcomes may be inconsistent with the reference data or tabletop rules. <br>
Mitigation: Treat the agent as a game narrator and review important rulings against the bundled references during play. <br>
