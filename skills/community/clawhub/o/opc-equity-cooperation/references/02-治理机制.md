# 治理机制：三长老、议事规则、争议解决

> 本文件详细阐述OPC联盟的治理架构、三长老权责、议事规则与争议解决机制。

---

## 一、三长老治理架构

### 1.1 三权分立原则

```
┌─────────────────────────────────────────────────────────────┐
│                      三长老治理铁三角                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│     ┌─────────┐                      ┌─────────┐            │
│     │  盟主   │◄──────────────────►│合规长老│            │
│     │（执行） │      相互制衡        │（合规） │            │
│     └────┬────┘                      └────┬────┘            │
│          │                                │                  │
│          │         ┌─────────┐           │                  │
│          └────────►│ 监事长老 │◄──────────┘                  │
│                    │（监督） │                               │
│                    └─────────┘                               │
│                                                             │
│   盟主：日常决策 + 争议协调                                 │
│   合规长老：合规审查 + 风险拦截                             │
│   监事长老：账务监督 + 发起弹劾                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 核心角色详细定义

#### 联盟盟主

| 项目 | 内容 |
|------|------|
| **人选建议** | 发起人/运营导师 |
| **核心职责** | 日常运营决策、争议协调、对外代表 |
| **权力范围** | 无否决权，只有提议权和协调权 |
| **任期** | 1年，可连任 |
| **禁止兼任** | 不得兼任监事长老 |

**盟主的具体职责**：
1. 召集和主持联盟会议
2. 制定联盟发展规划和年度计划
3. 协调成员之间的合作关系
4. 对外代表联盟签署协议
5. 执行联盟决议和规章制度
6. 提议三长老候选人（监事长老除外）

#### 合规长老

| 项目 | 内容 |
|------|------|
| **人选建议** | 律师/会计师/国企代表 |
| **核心职责** | 法律合规审查、税务方案审核、敏感词监控 |
| **权力范围** | 合规问题一票否决权，发现违规可立即暂停执行 |
| **任期** | 半年轮换 |
| **合规审查** | 合规审查是所有决策的前置条件 |

**合规长老的具体职责**：
1. 对所有合作协议进行合规审查
2. 审核税务方案和发票开具
3. 监控敏感词和违规行为
4. 出具合规审查意见
5. 对违规行为发出暂停/整改通知
6. 建立和维护合规台账

**合规长老的一票否决权**：
- 发现任何合规问题，可立即暂停执行
- 暂停令自动生效，无需其他审批
- 被暂停方可在24小时内申诉
- 申诉期间，暂停令持续有效

#### 监事长老

| 项目 | 内容 |
|------|------|
| **人选建议** | 2-3位核心成员（民主选举） |
| **核心职责** | 监督盟主行为、审计账务、保护中小成员利益 |
| **权力范围** | 发起弹劾的权力，无决策权 |
| **任期** | 半年轮换，连任不得超过2届 |
| **禁止兼任** | 不得由盟主或合规长老兼任 |

**监事长老的具体职责**：
1. 随时查阅所有账务和合同数据
2. 对盟主的任何决策提出质询
3. 发现违规操作可立即暂停执行
4. 每月出具监事报告，全员公示
5. 代表中小成员利益发声
6. 发起弹劾程序

### 1.3 三长老权力边界

| 权力类型 | 盟主 | 合规长老 | 监事长老 |
|---------|------|---------|---------|
| 日常运营决策 | ✅ 决定权 | ❌ | ❌ |
| 合规审查 | ❌ | ✅ 一票否决 | ❌ |
| 暂停违规操作 | ❌ | ✅ 立即生效 | ✅ 建议暂停 |
| 账务查阅 | ❌ | ❌ | ✅ 随时查阅 |
| 发起弹劾 | ❌ | ❌ | ✅ 专属权力 |
| 对外签约 | ✅ | ❌ | ❌ |
| 争议协调 | ✅ | ❌ | ❌ |
| 账务监督 | ❌ | ❌ | ✅ 主导 |

## 二、联合磋商会议制度

### 2.1 会议类型

| 会议类型 | 频率 | 时长 | 主持人 | 参与人 |
|---------|------|------|-------|-------|
| **周例会** | 每周一 | 30分钟 | 盟主 | 全体成员 |
| **月度评审会** | 每月第一天 | 2小时 | 盟主 | 全体成员+合规长老 |
| **特别会议** | 按需 | 按议题 | 盟主 | 全体成员 |

**周例会内容**：
- 进度同步
- 问题快速协调
- 下一周任务安排

**月度评审会内容**：
- 全流程复盘
- 贡献权重调整审核
- 里程碑验收确认
- 信用分评定

**特别会议触发条件**：
- 任意2名成员可发起
- 24小时内必须召开
- 48小时内必须形成决议

### 2.2 决策机制（三重表决）

| 事项类型 | 表决方式 | 通过门槛 |
|---------|---------|---------|
| 日常运营 | 盟主决定 + 监事复核 | 盟主签字即可 |
| 贡献权重调整 | 全体成员表决 | 2/3同意 |
| 新人加入/退出 | 全体成员表决 | 全体一致同意 |
| 合规问题 | 合规长老一票 | 合规长老同意 |
| 弹劾盟主 | 监事发起 + 全体表决 | 全体一致同意 |
| 规则修订 | 全体成员表决 | 全体一致同意 |

### 2.3 微调机制

- **贡献权重**：每季度可微调，单次调整幅度≤10%
- **分账比例**：可根据实际工作量临时调整
- **调整要求**：所有调整必须记录在案、上链存证

## 三、议事规则

### 3.1 标准议事流程

```
1. 【发起】任何成员可书面（系统内）提出议题
        ↓
2. 【公示】议题公示24小时收集意见
        ↓
3. 【辩论】线上会议充分辩论，每人限时发言
        ↓
4. 【表决】按事项类型走对应表决程序
        ↓
5. 【执行】决议形成后72小时内必须执行
        ↓
6. 【存档】全过程记录上链存证
```

### 3.2 发言规则

| 规则 | 说明 |
|------|------|
| 发言顺序 | 按举手顺序发言 |
| 发言时长 | 每人每次不超过5分钟 |
| 补充发言 | 需获得主持人许可 |
| 打断规则 | 紧急事项可申请打断 |
| 记录规则 | 全程文字记录，可回溯 |

### 3.3 修正案机制

- **提出权**：任何成员可提出规则修正案
- **通过条件**：全体一致同意方可生效
- **生效方式**：生效后自动覆盖旧规则
- **历史存档**：历史版本永久存档
- **定期报告**：每季度自动生成《规则修订白皮书》

## 四、争议解决机制

### 4.1 三级争议解决通道

```
一级：快速协调（24小时内）
  ↓ 未解决
二级：盟主+监事调解（48小时内）
  ↓ 未解决  
三级：全体公投裁决
```

### 4.2 各层级详解

#### 第一级：快速协调（24小时）

**触发条件**：发生争议

**处理流程**：
1. 争议方提交书面说明
2. 盟主在12小时内回应
3. 快速协调会议（24小时内）
4. 形成协调纪要

**适用范围**：
- 任务理解分歧
- 工作配合问题
- 进度沟通不畅

#### 第二级：盟主+监事调解（48小时）

**触发条件**：快速协调未解决

**处理流程**：
1. 盟主+监事长老共同主持调解
2. 双方各陈述事实和诉求
3. 调解方提出调解方案
4. 双方确认或提出修改意见
5. 达成调解协议

**适用范围**：
- 贡献权重争议
- 分账金额争议
- 任务完成度争议

#### 第三级：全体公投裁决（72小时）

**触发条件**：调解未成功

**处理流程**：
1. 监事长老发起公投
2. 争议方各自陈述立场
3. 全体成员投票（匿名）
4. 简单多数决定结果
5. 结果立即执行，不可申诉

**适用范围**：
- 严重违约行为
- 重大利益冲突
- 规则解释分歧

### 4.3 紧急处理机制

**紧急情况**（如：资金安全威胁、重大违规）：
1. 监事长老可绕过正常流程
2. 立即发出紧急暂停令
3. 24小时内召开紧急会议
4. 会议决定为最终决定

## 五、监事会运作机制

### 5.1 监事权责明细

| 权责 | 具体内容 |
|------|---------|
| **账务查阅权** | 随时查阅所有账务和合同数据 |
| **质询权** | 对盟主的任何决策提出质询 |
| **暂停权** | 发现违规操作可立即暂停执行 |
| **报告权** | 每月出具监事报告，全员公示 |
| **弹劾权** | 发起弹劾程序的专属权力 |
| **代表权** | 代表中小成员利益发声 |

### 5.2 监事产生规则

| 规则 | 说明 |
|------|------|
| **选举方式** | 由全体成员民主选举 |
| **禁止兼任** | 不得由盟主或合规长老兼任 |
| **任期限制** | 每半年轮换，确保新鲜血液 |
| **连任限制** | 连任不得超过2届 |

### 5.3 弹劾程序

**触发条件**：
- 盟主严重违反联盟章程
- 盟主损害联盟利益
- 盟主无法正常履行职责

**弹劾流程**：
```
1. 监事长老提出弹劾议案
        ↓
2. 议案公示7天，收集意见
        ↓
3. 召开全体成员大会
        ↓
4. 全体成员表决（一致同意）
        ↓
5. 弹劾通过，盟主免职
        ↓
6. 30天内选举新盟主
```

## 六、联盟章程核心条款

### 6.1 成员权利

| 权利类型 | 内容 |
|---------|------|
| **参与权** | 参与联盟会议、投票、发表意见 |
| **知情权** | 查阅账务、合同、会议纪要 |
| **收益权** | 按贡献权重获得项目收益分配 |
| **监督权** | 监督盟主和合规长老行为 |
| **提案权** | 提出议题、建议、修正案 |
| **退出权** | 书面申请退出联盟 |

### 6.2 成员义务

| 义务类型 | 内容 |
|---------|------|
| **履约义务** | 按约定完成分配任务 |
| **保密义务** | 保护联盟商业秘密 |
| **协作义务** | 积极配合其他成员工作 |
| **诚信义务** | 如实报告贡献和工作量 |
| **合规义务** | 遵守联盟章程和决议 |
| **费用分摊** | 按比例分摊联盟运营成本 |

### 6.3 退出机制

**主动退出**：
1. 书面申请退出，说明原因
2. 盟主审核，确认无未完成义务
3. 按当前贡献打折清算历史收益
4. 退出后不享有未来收益
5. 信用分保留，可查询

**被动退出（除名）**：
1. 监事长老提出除名议案
2. 合规长老出具违规证据
3. 全体成员表决（2/3同意）
4. 除名后不享有任何收益
5. 信用分大幅降低，影响未来合作
