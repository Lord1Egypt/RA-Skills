## Known Risks and Mitigations: <br>
Risk: Batch image commands and wildcard patterns can overwrite or damage many source files quickly. <br>
Mitigation: Work from copies or preserved masters, check wildcard matches before execution, and write outputs to separate paths. <br>
Risk: Metadata changes can either leak private information or remove needed orientation, rights, timestamp, or provenance data. <br>
Mitigation: Make metadata decisions deliberately; strip public GPS or unnecessary camera data while preserving required rights or workflow metadata. <br>
Risk: Ad hoc package execution in command examples can run untrusted registry code in sensitive projects. <br>
Mitigation: Prefer trusted, pinned, or locally installed tooling before running npx-based examples. <br>
