# 商业模式画布九宫格模板

## 概述

商业模式画布（Business Model Canvas）是一种战略管理工具，用于描述、分析和设计商业模式。

## 九维度框架

```
┌─────────────────────────────────────────────────────────────────────┐
│                        商业模式画布九宫格                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌───────────────────┐                    ┌───────────────────┐   │
│   │   客户细分        │    ┌─────────┐     │   价值主张        │   │
│   │   CS              │    │ 重要合作 │     │   VP              │   │
│   │                   │    │  KP      │     │                   │   │
│   ├───────────────────┤    ├─────────┤     ├───────────────────┤   │
│   │   渠道通路        │    │ 关键业务 │     │   客户关系        │   │
│   │   CH              │    │  KA      │     │   CR              │   │
│   │                   │    │         │     │                   │   │
│   ├───────────────────┤    │ 核心资源 │     ├───────────────────┤   │
│   │                   │    │  KR      │     │   收入来源        │   │

```

## 维度详解

### 1. 客户细分（Customer Segments）

**定义**：企业想要接触和服务的不同人群或组织

**核心问题**：
- 我们为谁创造价值？
- 谁是我们最重要的客户？

**子维度**：
- 客户群体特征
- 客户画像
- 市场细分
- 客户优先级

**输出格式**：
```
客户细分：

1. 主要客户群体
   - [群体1]：[描述]
   - [群体2]：[描述]

2. 客户画像
   - [画像名称]：
     - 特征：[描述]
     - 需求：[描述]
     - 痛点：[描述]

3. 市场细分
   - B2B / B2C / B2B2C / C2C

4. 客户优先级
   - 优先级1：[群体]
   - 优先级2：[群体]
```

**评分标准**：
| 分数 | 等级 | 描述 |
|------|------|------|
| 1-3 | 不及格 | 客户定义模糊，未明确目标群体 |
| 4-6 | 及格 | 有基本客户画像，但细分不足 |
| 7-8 | 良好 | 客户细分清晰，有差异化定位 |
| 9-10 | 优秀 | 精准客户画像+可量化市场规模+验证数据 |

### 2. 价值主张（Value Propositions）

**定义**：解决客户问题或满足客户需求的系列产品和服务

**核心问题**：
- 我们给客户带来什么价值？
- 解决了客户的什么问题？
- 满足客户的哪些需求？

**价值类型**：
- 新颖性：全新体验
- 性能：提升产品性能
- 定制化：满足个性化需求
- 设计：独特外观和体验
- 品牌/身份：提升客户形象
- 价格：性价比优势
- 成本削减：降低客户成本
- 风险降低：降低客户风险
- 可获得性：方便获取
- 便利性：使用便捷

**输出格式**：
```
价值主张：

1. 核心价值
   - [价值1]：[描述]
   - [价值2]：[描述]

2. 解决的问题
   - [问题1]：[解决方案]
   - [问题2]：[解决方案]

3. 满足的需求
   - [需求1]：[满足方式]
   - [需求2]：[满足方式]

4. 差异化优势
   - [优势1]：[描述]
   - [优势2]：[描述]

5. 相比竞品
   - 竞争优势：[描述]
   - 竞争劣势：[描述]
```

**评分标准**：
| 分数 | 等级 | 描述 |
|------|------|------|
| 1-3 | 不及格 | 价值主张不清，无法回答"为什么选你" |
| 4-6 | 及格 | 有基本价值主张，但差异化不明显 |
| 7-8 | 良好 | 价值主张清晰，有明确差异化 |
| 9-10 | 优秀 | 强价值主张+验证数据+护城河 |

### 3. 渠道通路（Channels）

**定义**：如何接触、抵达客户并传递价值

**核心问题**：
- 如何触达客户？
- 如何销售产品/服务？
- 如何交付价值？
- 渠道的优先级？

**渠道类型**：
- 线上：官网、App、小程序、电商平台
- 线下：直营店、经销商、代理商
- 混合：O2O、线上预约线下服务

**输出格式**：
```
渠道通路：

1. 获客渠道
   - [渠道1]：[描述+成本]
   - [渠道2]：[描述+成本]

2. 销售渠道
   - [渠道1]：[描述]
   - [渠道2]：[描述]

3. 交付渠道
   - [渠道1]：[描述]
   - [渠道2]：[描述]

4. 服务渠道
   - [渠道1]：[描述]

5. 渠道优先级
   - 主渠道：[描述]
   - 次渠道：[描述]

6. 获客成本
   - CAC：[数值]
   - ROI：[数值]
```

**评分标准**：
| 分数 | 等级 | 描述 |
|------|------|------|
| 1-3 | 不及格 | 无明确渠道规划 |
| 4-6 | 及格 | 有基本渠道，但依赖单一渠道 |
| 7-8 | 良好 | 多渠道布局，有优先级 |
| 9-10 | 优秀 | 全渠道体系+获客成本数据+渠道验证 |

### 4. 客户关系（Customer Relationships）

**定义**：与客户建立的关系类型

**核心问题**：
- 如何建立和维护客户关系？
- 如何提升客户忠诚度？
- 如何提高客户留存？

**关系类型**：
- 个人助理：专属客服
- 自助服务：无人工干预
- 自动化服务：系统自动处理
- 社区：用户社区运营
- 共同创作：与客户共同创造

**输出格式**：
```
客户关系：

1. 关系类型
   - 获客阶段：[关系类型]
   - 留存阶段：[关系类型]
   - 增值阶段：[关系类型]

2. 客户留存策略
   - [策略1]：[描述]
   - [策略2]：[描述]

3. 客户满意度指标
   - NPS：[数值]
   - 复购率：[数值]
   - 客户满意度：[数值]

4. 客户生命周期
   - 获取成本：[CAC]
   - 生命周期价值：[LTV]
   - LTV/CAC：[比值]
```

**评分标准**：
| 分数 | 等级 | 描述 |
|------|------|------|
| 1-3 | 不及格 | 未考虑客户关系维护 |
| 4-6 | 及格 | 有基本客户服务，但无体系 |
| 7-8 | 良好 | 客户关系分层管理，有留存策略 |
| 9-10 | 优秀 | 客户生命周期管理+LTV数据+NPS指标 |

### 5. 收入来源（Revenue Streams）

**定义**：从客户群体中获得的收入

**核心问题**：
- 如何赚钱？
- 客户愿意为什么付费？
- 收入结构如何？
- 定价策略？

**收入类型**：
- 资产销售：一次性购买
- 使用费：按使用量收费
- 订阅费：持续性收费
- 租赁费：使用权出租
- 授权费：知识产权授权
- 经纪费：中介服务费
- 广告费：广告展示收费

**输出格式**：
```
收入来源：

1. 收入类型
   - [类型1]：[占比+金额]
   - [类型2]：[占比+金额]

2. 定价模式
   - [模式1]：[描述]
   - [模式2]：[描述]

3. 收入结构
   - 主力收入：[描述]
   - 补充收入：[描述]

4. 财务预测
   - 收入预测：[数据]
   - 盈利预测：[数据]

5. 单位经济模型
   - ARPU：[数值]
   - 边际贡献：[数值]
```

**评分标准**：
| 分数 | 等级 | 描述 |
|------|------|------|
| 1-3 | 不及格 | 盈利模式不清 |
| 4-6 | 及格 | 有基本收入模式，但单一 |
| 7-8 | 良好 | 多元收入来源，定价合理 |
| 9-10 | 优秀 | 收入模型验证+单位经济数据+增长飞轮 |

### 6. 核心资源（Key Resources）

**定义**：执行商业模式所需的关键资产

**核心问题**：
- 需要哪些关键资源？
- 如何获取和保有这些资源？
- 资源的稀缺性和壁垒如何？

**资源类型**：
- 实体资源：设备、设施、库存
- 知识资源：专利、商标、版权、数据
- 人力资源：人才、团队、知识
- 财务资源：资金、融资能力

**输出格式**：
```
核心资源：

1. 实体资源
   - [资源1]：[描述+壁垒]

2. 知识资源
   - [资源1]：[描述+壁垒]
   - 专利：[数量+布局]

3. 人力资源
   - [资源1]：[描述+壁垒]

4. 财务资源
   - [资源1]：[描述+壁垒]

5. 资源壁垒
   - [壁垒1]：[描述]
   - [壁垒2]：[描述]

6. 资源可持续性
   - [评估]
```

**评分标准**：
| 分数 | 等级 | 描述 |
|------|------|------|
| 1-3 | 不及格 | 核心资源未识别 |
| 4-6 | 及格 | 识别了部分资源，但不够系统 |
| 7-8 | 良好 | 核心资源清晰，有获取路径 |
| 9-10 | 优秀 | 资源壁垒+独占性+可持续性 |

### 7. 关键业务（Key Activities）

**定义**：执行商业模式所需的关键活动

**核心问题**：
- 必须做好哪些事？
- 业务优先级如何？
- 业务的可扩展性如何？

**业务类型**：
- 生产：设计、制造、交付
- 平台：平台运营、生态管理
- 问题解决：咨询服务、定制开发
- 网络：资源整合、连接服务

**输出格式**：
```
关键业务：

1. 核心业务
   - [业务1]：[描述+重要性]

2. 支持业务
   - [业务1]：[描述]

3. 业务优先级
   - 优先级1：[业务]
   - 优先级2：[业务]

4. 业务里程碑
   - [时间]：[里程碑]

5. 业务可扩展性
   - [评估]
```

**评分标准**：
| 分数 | 等级 | 描述 |
|------|------|------|
| 1-3 | 不及格 | 关键业务不明确 |
| 4-6 | 及格 | 识别了部分关键业务 |
| 7-8 | 良好 | 关键业务清晰，有优先级和里程碑 |
| 9-10 | 优秀 | 业务闭环+效率指标+可扩展性 |

### 8. 重要合作（Key Partnerships）

**定义**：让商业模式运作所需的关键合作伙伴

**核心问题**：
- 需要哪些关键伙伴？
- 合作伙伴的价值是什么？
- 合作模式和绑定机制如何？

**合作类型**：
- 战略联盟：同等地位的合作
- 竞合关系：与竞争对手合作
- 供应商关系：与供应商合作
- 经销商关系：与经销商合作
- 生态合作：平台生态合作

**输出格式**：
```
重要合作：

1. 核心合作伙伴
   - [伙伴1]：[合作内容+价值]
   - [伙伴2]：[合作内容+价值]

2. 供应商
   - [供应商1]：[内容+依赖度]

3. 渠道合作
   - [渠道1]：[合作内容]

4. 生态合作
   - [合作1]：[内容]

5. 合作模式
   - [模式1]：[描述]
   - [模式2]：[描述]

6. 绑定机制
   - [机制1]：[描述]
```

**评分标准**：
| 分数 | 等级 | 描述 |
|------|------|------|
| 1-3 | 不及格 | 未考虑合作伙伴 |
| 4-6 | 及格 | 有基本合作意识，但无具体方案 |
| 7-8 | 良好 | 合作伙伴明确，有合作框架 |
| 9-10 | 优秀 | 战略联盟+生态协同+绑定机制 |

### 9. 成本结构（Cost Structure）

**定义**：执行商业模式所产生的全部成本

**核心问题**：
- 主要成本项是什么？
- 成本结构特征？
- 如何优化成本？

**成本类型**：
- 固定成本：不随产量变化的成本
- 变动成本：随产量变化的成本
- 规模经济：规模带来的成本优势
- 范围经济：业务范围带来的成本优势

**输出格式**：
```
成本结构：

1. 主要成本项
   - [成本项1]：[金额/占比]
   - [成本项2]：[金额/占比]

2. 成本结构
   - 固定成本：[占比]
   - 变动成本：[占比]

3. 成本特征
   - [特征描述]

4. 成本优化
   - [优化方向1]
   - [优化方向2]

5. 边际成本
   - [评估]

6. 盈亏平衡点
   - [数据]
```

**评分标准**：
| 分数 | 等级 | 描述 |
|------|------|------|
| 1-3 | 不及格 | 成本结构不清 |
| 4-6 | 及格 | 有基本成本估算，但不够精细 |
| 7-8 | 良好 | 成本结构清晰，有优化空间 |
| 9-10 | 优秀 | 精细成本管理+边际成本递减+盈利拐点 |
