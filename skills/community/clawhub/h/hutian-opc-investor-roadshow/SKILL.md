# 胡田-OPC导师-招商引流工具包

## 技能概述

**功能定位**：面向OPC导师（胡田）的专业招商引流工具包，服务于园区/社区招商，帮助企业精准落地发展。

**核心价值**：
- 🔍 企业精准筛选：从候选企业池中识别高潜力标的
- 📊 四层尽调体系：材料核查→技术评估→商业逻辑→风险量化
- 📋 定制化方案：匹配政策、产业、资源，输出落地方案
- 🤝 洽谈支持：话术库+异议处理+跟进管理

**适用场景**：
| 场景 | 典型需求 | 工具组合 |
|------|----------|----------|
| 园区招商 | 引入优质企业入驻 | 筛选模型+尽调报告+政策匹配 |
| 政府招商 | 产业链招商、精准招商 | 产业链分析+企业画像+政策包 |
| OPC导师服务 | 企业资源对接、落地发展 | 全流程服务+跟进管理 |

### 模块2：企业信息收集

#### 信息收集清单

| 信息类型 | 数据来源 | 采集指标 |
|----------|----------|----------|
| 工商信息 | 天眼查/企查查 | 法人、股东、注册资本、变更记录 |
| 舆情监控 | 企查查/百度 | 新闻报道、投诉记录、资质荣誉 |
| 财务数据 | 财报/年报 | 营收、利润、资产、负债 |
| 专利分析 | 智慧芽/国家专利局 | 专利数量、类型、法律状态、技术领域 |
| 融资历史 | IT桔子/烯牛数据 | 融资轮次、金额、投资方 |

#### 采集执行流程

```
1. 企业名称确认 → 2. 天眼查工商查询 → 3. 企查查舆情检索 
       ↓
4. 专利数据库查询 → 5. 财报/年报获取 → 6. 融资信息汇总
       ↓
7. 信息整合归档 → 8. 生成信息收集报告
```

### 技术代际识别模块（新增）

#### 模块概述
技术代际识别是判断项目技术属于第几代产品的系统性方法，通过对比技术参数与各代产品标准特征，明确项目在技术演进路线中的定位。

#### 识别流程

```
┌─────────────────────────────────────────────────────────────┐
│                   技术代际识别流程                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Step 1: 检索行业技术发展史                                  │
│          → 识别技术演进路线                                  │
│          → 梳理各代际关键技术节点                            │
│                                                             │
│              ↓                                              │
│                                                             │
│  Step 2: 对比项目技术参数与各代产品标准特征                    │
│          → 提取核心性能指标                                  │
│          → 匹配代际特征库                                    │
│                                                             │
│              ↓                                              │
│                                                             │
│  Step 3: 确认项目代际定位                                    │
│          → 确定当前代际                                      │
│          → 标注代际特征符合度                                │
│                                                             │
│              ↓                                              │
│                                                             │
│  Step 4: 评估代际差距                                        │
│          → 领先/持平/落后判定                                │
│          → 量化时间差距                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 输出格式

```
技术代际识别报告
├── 技术代际定位：第X代
├── 代际特征对比表
│   ├── 代际X特征1：_____ │ 项目匹配度：_____
│   ├── 代际X特征2：_____ │ 项目匹配度：_____
│   └── 代际X特征3：_____ │ 项目匹配度：_____
├── 代际领先性评分：领先X年 / 持平 / 落后X年
└── 技术演进趋势预判
```

#### 技术代际分析案例（BNCT硼中子俘获治疗）

```
代际演进路线：
┌─────────────────────────────────────────────────────────────┐
│  第一代     第二代       第三代       第四代       第五代    │
│ （实验室） （临床前）  （临床试验） （逐步推广） （普及化）  │
│   1950s      1990s       2010s        2020s       未来      │
└─────────────────────────────────────────────────────────────┘

第四代BNCT技术标准特征：
├── 照射深度：>10cm（深部肿瘤适应）
├── 照射时间：<30分钟/次
├── 剂量精度：±5%以内
├── 配套设施：小型化加速器
└── 临床进展：多中心临床试验

项目代际判定示例：
- 中硼医疗：第四代（小型化加速器+深部治疗）
```

#### 竞品对比矩阵填写指南

**Step 1：确定竞品范围**
```
选择原则：
- 同一技术路线的主要参与者
- 同一应用场景的直接竞争者
- 不同技术路线的替代方案提供者
- 建议选取3-5家核心竞品
```

**Step 2：数据采集**
```
数据来源优先级：
1. 官方披露（招股书、年报、官网）
2. 权威机构报告（行业白皮书、研究报告）
3. 第三方数据库（智慧芽、天眼查）
4. 公开媒体报道
5. 行业专家访谈
```

**Step 3：矩阵填写**
```
评分标准：
- 优秀：★★★★★ / 优于行业平均
- 良好：★★★★☆ / 达到行业平均
- 一般：★★★☆☆ / 低于行业平均
- 较差：★★☆☆☆ / 明显落后
- 未知：☆☆☆☆☆ / 数据缺失
```

#### 雷达图生成代码模板

```python
# 竞品对比雷达图生成模板
import matplotlib.pyplot as plt
import numpy as np

# 维度标签
categories = ['技术参数领先性', '技术成熟度', '合规进度', 
              '产业化能力', '成本竞争力', '团队实力']

# 数据（示例：被评估项目 vs 竞品1）
project_data = [85, 70, 55, 60, 75, 80]  # 被评估项目
competitor1 = [75, 65, 40, 50, 80, 70]  # 竞品1

# 雷达图绘制
angles = np.linspace(0, 2*np.pi, len(categories), endpoint=False).tolist()
angles += angles[:1]

project_data += project_data[:1]
competitor1 += competitor1[:1]

fig, ax = plt.subplots(figsize=(10, 8), subplot_kw=dict(polar=True))
ax.plot(angles, project_data, 'o-', linewidth=2, label='被评估项目')
ax.fill(angles, project_data, alpha=0.25)
ax.plot(angles, competitor1, 'o-', linewidth=2, label='竞品1')
ax.fill(angles, competitor1, alpha=0.25)

ax.set_xticks(angles[:-1])
ax.set_xticklabels(categories)
ax.set_ylim(0, 100)
ax.legend(loc='upper right')
ax.set_title('竞品对比雷达图', fontsize=14, fontweight='bold')

plt.tight_layout()
plt.savefig('竞品对比雷达图.png', dpi=150, bbox_inches='tight')
plt.show()
```

```

   ├── 每个技术特征对应的专利布局
```

评估企业在赛道中的竞争位置

##### 分析框架

```
1. 赛道识别与定义
   ├── 企业所在赛道的精准定义
   ├── 赛道边界（包含什么、不包含什么）
   └── 赛道细分（主赛道/细分赛道）

2. 赛道格局分析
   ├── 赛道规模与增速
   ├── 竞争格局（垄断/寡头/分散）
   ├── 竞争要素（技术/成本/渠道/品牌）
   └── 行业生命周期（导入/成长/成熟/衰退）

3. 企业竞争位置
   ├── 市场份额/排名
   ├── 技术位置（领先/追随/落后）
   ├── 资源位置（资金/人才/渠道）
   └── 增长势能

4. 赛道机会评估
   ├── 赛道增长空间
   ├── 行业变革机会
   └── 企业突破可能性
```

#### 链条五：赛道 → 资本/资源方布局

##### 分析目标
了解资本和资源方在赛道中的布局，判断资源支持可能性

##### 分析框架

```
1. 赛道资本地图
   ├── 活跃投资机构及投资风格
   ├── 已投资企业及估值
   ├── 投资轮次分布
   └── 退出情况（IPO/并购）

2. 产业资本布局
   ├── 头部企业的战略投资方向
   ├── 产业链上下游的投资布局
   └── 潜在并购方

3. 政策资源分布
   ├── 国家/地方政策支持力度
   ├── 产业基金规模
   └── 税收/补贴政策

4. 人才资源分布
   ├── 核心人才池规模
   ├── 高校/研究机构分布
   └── 人才流动趋势

5. 企业资源获取能力
   ├── 与资本方的连接程度
   ├── 与产业方的合作关系
```

#### 链式追溯分析执行流程

```
┌─────────────────────────────────────────────────────────────────────────┐
│                      链式追溯分析执行流程                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  Step 1: 技术特征提取                                                    │
│  ├─ 梳理项目声称的所有核心技术特征                                        │
│  ├─ 分类：独创性/改进性/组合性                                            │
│  └─ 输出：技术特征清单                                                   │
│                      ↓                                                  │
│  Step 2: 知识产权映射                                                    │
│  ├─ 查询每项技术特征对应的专利                                            │
│  ├─ 分析专利类型、状态、保护范围                                          │
│  └─ 输出：技术特征-知识产权映射表                                         │
│                      ↓                                                  │
│  Step 3: 产品方案验证                                                   │
│  ├─ 核实专利是否实际应用到产品                                            │
│  ├─ 对比产品性能与专利描述                                                │
│  └─ 输出：知识产权-产品方案转化表                                         │
│                      ↓                                                  │
│  Step 4: 企业能力评估                                                   │
│  ├─ 盘点企业研发/生产/市场/资金能力                                       │
│  ├─ 对比产品方案需求                                                     │
│  └─ 输出：产品方案-企业能力匹配表                                         │
│                      ↓                                                  │
│  Step 5: 赛道定位分析                                                   │
│  ├─ 识别赛道规模、格局、周期                                              │
│  ├─ 评估企业在赛道中的位置                                                │
│  └─ 输出：企业-赛道定位分析                                               │
│                      ↓                                                  │
│  Step 6: 资本资源分析                                                   │
│  ├─ 梳理赛道资本地图                                                      │
│  ├─ 分析产业资本、政策资源、人才资源                                      │
│  └─ 输出：赛道-资本资源布局分析                                           │
│                      ↓                                                  │
│  Step 7: 综合评估输出                                                   │
│  ├─ 汇总各链条断点                                                        │
│  ├─ 输出综合评估报告                                                     │
│  └─ 给出投资/招商建议                                                    │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 模块4：招商方案生成

#### 方案框架

```
一、企业概况
   1.1 企业基本信息
   1.2 核心业务与产品
   1.3 发展历程与现状

二、选址需求分析
   2.1 战略目标与布局
   2.2 选址关键因素
   2.3 区域偏好

三、推荐园区/载体
   3.1 园区概况
   3.2 产业定位与配套
   3.3 政策优势对比

四、定制化政策包
   4.1 财税支持政策
   4.2 人才引进政策
   4.3 研发创新政策
   4.4 金融支持政策

五、服务体系
   5.1 企业落地服务
   5.2 产业资源对接
   5.3 增值服务

六、预期效益
   6.1 经济贡献预测
   6.2 产业带动效应
   6.3 社会效益分析

七、合作建议与推进计划
```

#### 政策匹配逻辑

```python
政策匹配分 = 基础分 + 加分项

基础分（60分）：
- 行业匹配：+15分
- 规模匹配：+10分
- 阶段匹配：+10分
- 区域偏好：+10分
- 政策历史使用情况：+15分

加分项（40分）：
- 高新技术企业认定：+10分
- 人才密度需求高：+10分
- 研发投入占比>10%：+10分
- 带动上下游能力强：+10分
```
