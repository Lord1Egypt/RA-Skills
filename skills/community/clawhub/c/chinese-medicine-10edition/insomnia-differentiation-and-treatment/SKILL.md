# Insomnia Differentiation and Treatment

## When to Use
当患者主诉以下任一症状时启动本技能：
- 入睡困难
- 睡眠易醒
- 彻夜不眠

且满足前提条件：
- 存在睡眠障碍
- 需进行中医辨证分型

> 注意：本技能不适用于器质性脑病导致的继发性失眠，除非结合中医辨证使用。

## Execution Steps

### 1. 明确基本病机
确认核心病机为：**阴阳不调、气血失和 → 心神不安 → 神不守舍**。

### 2. 分析四大病因
根据患者病史与症状，判断以下哪一类为主要诱因：
- **情志所伤**：七情过极 → 肝郁化火 / 心脾两伤 / 心肾不交
- **饮食失常**：暴饮暴食或嗜茶酒 → 胃气失和 → 痰热上扰
- **劳逸失调**：劳倦伤脾或过逸 → 气血生化乏源 → 心神失养
- **病后体虚**：久病血虚 / 年迈肾亏 → 阴血不足 / 水火不济

### 3. 辨证首分虚实
- **虚证（多见）**：表现为阴血不足，病位在心、脾、肝、肾 → 治宜滋补肝肾、益气养血
- **实证（少见）**：表现为肝郁化火、食滞痰浊 → 治当疏肝、消导、清火化痰

### 4. 处理特殊情况
- **虚实夹杂**：采用补泻兼顾策略
- **久病入络**：从瘀论治，加入活血化瘀法
- **所有类型**：均需配合安神镇静药物或疗法

### 5. 强调非药物干预
必须同步建议精神心理调适，缓解紧张、焦虑情绪，作为治疗的重要组成部分。

## Output
输出格式为明确的辨证结论，例如：
- “心肾不交”
- “肝火扰心”
- “心脾两虚”
- “痰热扰心”

该结论将直接用于指导后续方剂选择与治疗方案制定。
