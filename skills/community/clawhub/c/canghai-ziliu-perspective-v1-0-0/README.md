# 沧海自流 · 投资思维视角 Skill

> 用沧海老师的眼睛看市场，用他的思维框架做判断。

## 是什么

这是一个 **perspective skill**——不是工具，是一套思维操作系统。

激活后，AI 会用沧海自流的视角、语气、思维框架来回答投资问题：
- 先看产业革命大趋势，再落到具体标的
- 用第一性原理过滤噪音
- 动态估值 + 全球对标
- 承认不确定性，不说"一定涨"

## 核心内容

基于沧海自流 2026-06-13 新价值投资研究院完整分享记录（约15万字）提炼：

- **7个核心心智模型**：产业革命红利捕获、五步闭环投资体系、动态估值全球对标法、第一性原理噪音过滤器、机会成本决策框架、开卷考试思维、逆向操作勇气框架
- **12条决策启发式**：王子落难买入法、估值匹配持有法、三层筛子筛选法、动态仓位管理法等
- **完整表达DNA**：口语化、短句为主、反问式互动、轻度自嘲
- **诚实边界**：明确标注信息来源局限，不假装完美

## 安装

```bash
openclaw skill install canghai-ziliu-perspective
```

或从本仓库手动安装：将整个目录放入 `~/.qclaw/skills/`。
