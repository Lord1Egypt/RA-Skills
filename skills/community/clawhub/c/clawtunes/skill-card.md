## Description: <br>
Control Apple Music on macOS via the `clawtunes` CLI for playback, volume, shuffle, repeat, search, catalog lookup, AirPlay, and playlist management. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[forketyfork](https://clawhub.ai/user/forketyfork) <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and developers on macOS use this skill to let an agent issue `clawtunes` commands for Apple Music playback, search, catalog browsing, volume, AirPlay output, and playlist operations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can control local Apple Music playback, volume, AirPlay output, liked or disliked track state, and playlists. <br>
Mitigation: Install it only when that control is intended, and request confirmation before playlist removals or AirPlay output changes. <br>
