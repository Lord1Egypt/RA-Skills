## Description: <br>
Optimize prompts for code-related tasks following prompt-engineering best practices. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[anderskev](https://clawhub.ai/user/anderskev) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to turn rough code-related requests into clearer prompts for implementation, debugging, refactoring, code review, and testing. It audits the prompt, lists targeted improvements, and returns an optimized prompt with task-specific tips. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The optimized prompt may add specificity or verification steps that go beyond the user's original wording. <br>
Mitigation: Review the Analysis and Improvements Applied sections before using the optimized prompt. <br>
Risk: Normal use does not require credentials, API keys, or broad local access. <br>
Mitigation: Use the skill as a text-only prompt refinement workflow and do not include secrets in the prompt. <br>
