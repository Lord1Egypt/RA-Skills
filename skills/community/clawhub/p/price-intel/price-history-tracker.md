# Price History Tracker

Use this template to log weekly price snapshots for your competitive set. Consistency beats precision — a weekly check on the same day produces more actionable trends than irregular deep dives.

## How to Use

1. Pick one day per week (recommend Monday morning) to log prices
2. Check each competitor's current listed price and any visible promotions
3. Note any bundle changes, new entrants, or listing quality changes
4. Flag any change >10% for immediate review using the Price Intel workflow
