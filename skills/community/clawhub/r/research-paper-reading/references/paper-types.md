# Paper Types & Reading Strategies

## 1. Theoretical/Algorithmic Papers

**Characteristics:**
- Mathematical proofs
- Novel algorithms
- Complexity analysis

**Reading Strategy:**
1. Focus on Theorem/Proof structure
2. Understand the problem formulation
3. Trace algorithm step-by-step
4. Verify correctness claims
5. Check complexity analysis
6. Look for limitations in assumptions

## 2. Experimental Papers

**Characteristics:**
- New methodology
- Benchmark comparisons
- Ablation studies

**Reading Strategy:**
1. Understand experimental setup
2. Check baseline comparisons
3. Analyze ablation studies
4. Reproducibility details
5. Statistical significance

## 3. Applied/Systems Papers

**Characteristics:**
- System architecture
- Engineering details
- Real-world deployment

**Reading Strategy:**
1. System overview first
2. Design decisions and tradeoffs
3. Performance evaluation
4. Scalability considerations
5. Practical limitations

## 4. Survey Papers

**Characteristics:**
- Literature review
- Taxonomy creation
- Benchmark comparisons

**Reading Strategy:**
1. Check scope and completeness
2. Evaluate taxonomy organization
3. Note important references to follow
4. Identify consensus and debates
5. Future research directions

## 5. Position Papers

**Characteristics:**
- Argumentative
- Opinion-based
- Provocative

**Reading Strategy:**
1. Identify central claim
2. Evaluate evidence quality
3. Check logical coherence
4. Consider alternatives
5. Assess bias
