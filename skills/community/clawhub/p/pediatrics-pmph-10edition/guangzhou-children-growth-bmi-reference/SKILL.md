# 广州市3～7岁儿童体格发育与BMI参考值查询

## 使用条件
- 儿童年龄在3.0～7.0岁之间，且年龄已精确划分至半年区间（如3.0～3.5岁）
- 已知儿童性别（男/女）
- 评估对象为广州市常住儿童

## 操作步骤
1. 确定儿童性别（男或女）。
2. 根据实际年龄确定所属半年年龄区间（例如：3.0～3.5、3.5～4.0……6.0～7.0岁）。
3. 在参考表中查找对应性别和年龄区间的以下指标（均以“均值±标准差”形式提供）：
   - 体重（kg）
   - 身高（cm）
   - 坐高（cm）
   - 胸围（cm）
   - 头围（cm）
   - BMI（kg/m²）
4. 注意表格包含两组数据：第一组无特殊标识，第二组以“满K”开头，可能代表不同调查年份或样本批次。优先采用最新或标注明确的数据集。
5. 将儿童实测BMI与表中对应年龄和性别的均值±2个标准差（SD）范围比较：
   - 若实测值 > 均值+2SD，提示可能存在超重或肥胖
   - 若实测值 < 均值−2SD，提示可能存在消瘦或营养不良

> **注意**：该参考值仅适用于广州市儿童，不可直接用于全国或其他地区儿童的评估。
