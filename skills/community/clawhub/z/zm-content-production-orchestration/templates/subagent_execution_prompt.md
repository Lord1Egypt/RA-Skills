# 可复制给 SubAgent 的执行提示词

你正在执行 `zm-content-production-orchestration` Skill。

## 任务

把图文内容任务按多 Agent 流程推进到可发布/可交付状态。

## 最小必填

- 平台：公众号 / 小红书 / 抖音 / 私域
- 目标用户
- 内容目标
- 核心观点
- 案例素材
- 是否需要配图/封面
- 发布渠道
- 合规边界
- 交付包要求

## 执行顺序

1. 选题定位和查重
2. 内容设计卡
3. writer-pro 正文生产
4. reviewer/critic 审核
5. media-director 视觉生产
6. 排版/发布草稿
7. 合规与 AI 痕迹审核
8. main 终审
9. 交付包归档

## 硬门禁

- main 不直接写正式正文。
- 正式视觉交 media-director。
- 没有案例/素材，不得编造成真实案例。
- 没有审核记录，不得宣称完成。
