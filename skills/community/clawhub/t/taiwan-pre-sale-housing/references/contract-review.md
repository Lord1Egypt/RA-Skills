# 預售屋合約審查要點

## 應備文件檢查

簽約前，建商應提供以下文件，缺一不可：

- [ ] 預售屋買賣契約書（內政部版或經核備版）
- [ ] 建造執照影本
- [ ] 銷售許可證（或核准銷售函）
- [ ] 預售屋買賣定型化契約應記載及不得記載事項
- [ ] 建材設備表
- [ ] 共用部分契約書
- [ ] 平面圖/立面圖/剖面圖
- [ ] 土地使用分區證明

## 契約審查要點

### 1. 房地標示及權利範圍
- [ ] 土地座落/地號正確
- [ ] 建物建築基地面積
- [ ] 共有部分比例明確（公設比應≤35%合理）
- [ ] 停車位種類/數量/位置
- [ ] 是否有露台/陽台登記

### 2. 土地及建物面積
- [ ] 主建物面積（平方公尺換算坪數）
- [ ] 附屬建物面積（陽台/雨遮）
  - 注意：2018年1月1日起雨遮**不登記不計價**
- [ ] 共有部分面積
- [ ] 總面積與權狀坪數一致
- [ ] 面積誤差處理方式（±1%找補，≥3%可解約）

### 3. 車位
- [ ] 車位編號/位置（平面圖對照）
- [ ] 車位類型（法定/增設/獎勵）
- [ ] 車位尺寸（長×寬×高）
- [ ] 車位價格（與房屋價款分列）

### 4. 總價款
- [ ] 房屋+土地+車位總價
- [ ] 自備款與貸款比例
- [ ] 各期工程款金額與繳款時程表
- [ ] 銀行貸款成數不足之處理（解約或補足）

### 5. 建材設備
- [ ] 結構（RC/SC/SRC/其他）
- [ ] 外牆（磁磚/塗料/帷幕）
- [ ] 地板（拋光磚/木地板/石塑地板）
- [ ] 廚具品牌/型號
- [ ] 衛浴品牌/型號
- [ ] 門窗（鋁窗品牌/氣密等級）
- [ ] 玄關門/室內門
- [ ] 電梯品牌
- [ ] 智慧家庭系統（如有）

### 6. 開工及完工日期
- [ ] 開工日期（應有明確日期或條件）
- [ ] 完工日期（應有明確日期）
- [ ] 完工定義（使用執照取得）
- [ ] 逾期懲罰：**每日應按已繳價款萬分之五計算**

### 7. 貸款約定
- [ ] 貸款成數（一般7~8成）
- [ ] 貸款銀行（約定貸款銀行或自由選擇）
- [ ] 貸款不足時：
  - **可歸責於買方**：應補足
  - **不可歸責於買方**：可解約，建商應退還全額

### 8. 驗收
- [ ] 驗收時機（使用執照取得後，通知交屋前）
- [ ] 驗收次數（至少1次）
- [ ] 修繕期限
- [ ] 複驗權利

### 9. 交屋
- [ ] 交屋條件（貸款核撥+付清尾款）
- [ ] 交屋日期
- [ ] 產權移轉時程
- [ ] 交屋保留款：**至少總價5%**
- [ ] 管理費起算點

### 10. 保固
- [ ] 結構保固：**15年**（內政部版）
- [ ] 固定建材設備保固：**1年**（內政部版）
- [ ] 防水保固：**1年**（內政部版，部分建商3年）
- [ ] 保固範圍及除外事項

---

## 紅燈條款

| 條款 | 風險 | 對策 |
|------|:----:|------|
| 雨遮計價 | 🔴 | 2018年後已禁止，應要求刪除 |
| 公設比>35% | 🟡 | 合理範圍，但過高應注意 |
| 沒收已繳全部價款 | 🔴 | 違反不得記載事項，上限為總價15% |
| 貸款不足視同違約 | 🔴 | 不可歸責於買方應可解約 |
| 驗屋次數只限一次 | 🟡 | 應爭取至少一次複驗 |
| 管理費起算交屋前 | 🟡 | 應以交屋日起算 |
| 保固期間短於內政部版 | 🔴 | 違反不得記載事項 |
| 建商保留僅供參考權 | 🟡 | 建材設備應明確，非僅供參考 |
