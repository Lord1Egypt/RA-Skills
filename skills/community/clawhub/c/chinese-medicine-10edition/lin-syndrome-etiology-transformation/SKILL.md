# Lin Syndrome Etiology and Transformation

## When to Use
- 患者主诉小便异常，如频数、短涩、淋沥不尽、灼热刺痛、尿色黄赤等；
- 需结合舌象、脉象及全身症状进行中医病机判断；
- 排除外伤性血尿、泌尿系结石机械损伤或其他非湿热/非肾膀胱相关的泌尿系统疾病。

## Execution Steps

1. **确定病位**：确认病位主要在肾与膀胱，并评估是否涉及肝、脾功能失调。

2. **识别核心病机**：判断是否存在“湿热蕴结下焦，肾与膀胱气化不利”这一基本病机。

3. **判断病程阶段与虚实属性**：
   - **初起阶段**：多属实证，由外感湿热、饮食不节或情志失调引发，正气未虚；
   - **久病阶段**：湿热耗伤正气，累及脾肾，由实转虚；
   - **反复发作或素体虚弱者**：常表现为虚实夹杂，邪未尽而正已伤。

4. **归类四大病因**：
   - **外感湿热**：秽浊之邪从下窍侵入，或他脏热邪下移膀胱；
   - **饮食不节**：嗜食肥甘厚味或饮酒过度，致脾胃湿热内生，下注膀胱；
   - **情志失调**：肝郁化火，火移下焦，扰动膀胱；
   - **禀赋不足或劳伤久病**：房劳、多产、年老体弱等致脾肾两虚，统摄无权。

5. **明确标本关系**：强调“肾虚为本，膀胱湿热为标”，据此指导治疗侧重。

## Output
输出淋证的病机类型（实证 / 虚证 / 虚实夹杂）及核心病机描述，用于指导后续辨证分型与治法选择。
