## 行业校准

本技能通过**行业校准层**适配不同行业。首次使用时：

1. 识别用户所属行业
2. 读取 `references/industry-calibration.md` 获取校准指引
3. 将行业特定术语、法规、流程特征注入单元执行上下文
4. 后续所有单元自动适配该行业语境

**已内置校准**：12个行业，6大分组

| 分组 | 行业 |
|------|------|
| 消费与零售 | 电商、快消/零售、餐饮 |
| 平台与数字 | 互联网/平台、SaaS |
| 专业与知识 | 咨询、教育 |
| 制造与工业 | 制造、汽车 |
| 强监管 | 医药、金融 |
| 资产运营 | 房地产 |

**自定义行业**：提供行业名称和3-5个核心业务场景，自动就近匹配分组并生成校准配置
