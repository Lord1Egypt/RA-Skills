# PPD皮肤试验判读

## 何时使用
- 患者为胸部影像异常者、结核密切接触者或需鉴别诊断的疑似结核病患者；
- 已完成PPD皮内注射且处于48～72小时判读窗口期（老年人可延至72小时后）；
- 前臂内侧皮肤完整，且非急性免疫抑制状态（如使用激素、HIV感染、恶性肿瘤等需谨慎解读）。

## 判读步骤
1. **确认判读时间**：在注射后48～72小时内测量硬结直径（仅测硬结，不包括红晕）；老年人反应延迟，可适当延长至72小时后。
2. **测量硬结直径**：取横径与纵径的平均值（单位：mm）。
3. **按我国标准分级**：
   - 阴性（-）：硬结平均直径＜5 mm 或无反应；
   - 一般阳性（+）：5 mm ≤ 直径 ＜10 mm；
   - 中度阳性（++）：10 mm ≤ 直径 ＜15 mm；
   - 强阳性（+++）：直径≥15 mm，或出现双圈、水疱、坏死、淋巴管炎。
4. **结合临床背景综合判断**：
   - 若为未接种卡介苗的儿童，任何阳性结果应视为活动性结核处理；
   - 成人强阳性需高度怀疑活动性结核；
   - 阴性结果不能完全排除结核感染，尤其在免疫抑制、重症结核或感染早期（4～8周内）。
5. **考虑重复试验**：初次阴性但临床高度怀疑者，可在1～3周后重复试验（利用“助强效应”提高检出率）。

## 注意事项
- 卡介苗接种史会降低PPD特异性，阳性结果不能区分自然感染与疫苗反应；
- 急性皮肤损伤或严重免疫缺陷者结果不可靠；
- 约0.4%～20%活动性肺结核患者可呈假阴性。
