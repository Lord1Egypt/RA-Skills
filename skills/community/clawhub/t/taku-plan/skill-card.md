## Description: <br>
Turn an approved design into an executable implementation plan through scope and architecture review, optional UI design review, and plan writing. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kkenny0](https://clawhub.ai/user/kkenny0) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to review approved design documents, capture scope, architecture, and UI review findings, and produce concrete execution plans for build agents. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can modify planning files from broad review prompts with unclear write boundaries. <br>
Mitigation: Ask for a dry-run review when file changes are not desired, and inspect diffs to DESIGN.md and PLAN.md before accepting updates. <br>
