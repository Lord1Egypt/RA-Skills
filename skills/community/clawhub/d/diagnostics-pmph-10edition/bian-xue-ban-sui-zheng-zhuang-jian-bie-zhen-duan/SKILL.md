# 便血伴随症状的鉴别诊断

## 适用场景
- 患者主诉存在便血（黑便、鲜血便或暗红色血便）
- 同时伴有其他临床症状（如腹痛、里急后重、发热、皮肤黏膜出血、皮肤改变或腹部肿块）
- 需要基于症状组合进行初步病因鉴别

> **注意**：本技能不适用于无便血的单纯腹痛、发热或其他非出血性消化道症状。

## 执行步骤

1. **评估腹痛特征**：
   - 若为慢性反复上腹痛，呈周期性和节律性，且出血后疼痛减轻 → 提示消化性溃疡；
   - 若为上腹绞痛伴黄疸 → 考虑胆道出血；
   - 若腹痛时排血便或脓血便，便后腹痛减轻 → 见于细菌性痢疾、阿米巴痢疾或溃疡性结肠炎；
   - 若腹痛伴便血还见于急性出血性坏死性肠炎、肠套叠、肠系膜血管栓塞/血栓形成、膈疝等。

2. **判断是否存在里急后重**（肛门坠胀感、排便不尽感、排便频繁但量少）：
   - 存在 → 提示肛门直肠疾病，如细菌性痢疾、直肠炎、直肠癌。

3. **确认是否伴发热**：
   - 伴发热 → 常见于感染性疾病（如败血症、流行性出血热、钩端螺旋体病）或恶性肿瘤（如肠道淋巴瘤、白血病）。

4. **检查是否有全身出血倾向**（如皮肤瘀点、紫癜、牙龈出血等）：
   - 存在 → 见于重症肝炎、流行性出血热、白血病、过敏性紫癜、血友病等。

5. **观察皮肤改变**：
   - 蜘蛛痣及肝掌 → 提示肝硬化门静脉高压相关出血；
   - 皮肤黏膜毛细血管扩张 → 警惕遗传性毛细血管扩张症。

6. **触诊是否发现腹部肿块**：
   - 存在 → 应考虑结肠癌、肠结核、肠道恶性淋巴瘤、肠套叠、克罗恩病等。

## 输出要求
综合上述伴随症状的组合，输出最可能的病因诊断方向（如“提示溃疡性结肠炎”或“需警惕结肠癌”），避免绝对化结论，强调需结合进一步检查确认。
