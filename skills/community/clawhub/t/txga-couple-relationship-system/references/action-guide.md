## 每日能量充值清单

### 早安仪式（5分钟）

- [ ] 一个拥抱（超过3秒）
- [ ] 一句感谢或欣赏的话
- [ ] 询问对方今天的计划或状态

### 晚安仪式（10分钟）

- [ ] 身体接触（拥抱、牵手、抚摸头发）
- [ ] 当天一件正向反馈
  - "你今天...做得很好"
  - "谢谢你今天..."
  - "我很欣赏你..."
- [ ] 一个简单的问题
  - "今天最开心的事是什么？"
  - "有什么想和我说的吗？"
  - "明天有什么安排？"

### 日常加分行为

**随时可做（选1-2项）**：
- [ ] 主动发一条关心的消息
- [ ] 为对方做一件小事（倒水、拿东西）
- [ ] 表达想念或爱意
- [ ] 认真倾听对方说话（放下手机）
- [ ] 给对方一个意外的拥抱

## 冲突修复标准流程

### 冲突发生时的步骤

```
Step 1: 识别信号（情绪上升时）
├─ 身体信号：心跳加速、呼吸急促、声音变大
├─ 应对：立即暂停，深呼吸3次
└─ 话术："我需要冷静一下，我们等会再谈"

Step 2: 物理隔离（如果情绪失控）
├─ 时长：15-30分钟
├─ 地点：不同房间或外出
└─ 规则：不刷手机/不看电视，安静处理情绪

Step 3: 情绪归因
├─ 问自己：我在生气什么？是这件事本身还是触发了我的某个创伤？
├─ 写下来：可以帮助理清思路
└─ 准备：整理好"观察-感受-需求-请求"

Step 4: 主动开启修复对话
└─ 时限：冲突后24小时内必须开始

Step 5: 修复对话（按以下顺序）
├─ ① 承担各自责任
├─ ② 解释当时状态
├─ ③ 表达歉意
├─ ④ 理解对方感受
├─ ⑤ 明确下次如何做
└─ ⑥ 一个小动作重建连接（拥抱/牵手）
```

### 修复对话检查清单

- [ ] 双方情绪都已平稳
- [ ] 选择安静不受打扰的环境
- [ ] 用"我"而不是"你"来表达
- [ ] 不翻旧账
- [ ] 不做人身攻击
- [ ] 最后有明确的改进约定
- [ ] 结束时有一个正向连接动作

## 能量链路修复步骤

### 链路1：情绪基底 → 决策质量

**问题表现**：
- 经常因为家里事分心
- 在重要决策前犹豫不决
- 感觉脑子不清醒

**修复动作**：
1. [ ] 每天给自己15分钟独处时间（清理情绪缓存）
2. [ ] 做重大决策前和伴侣提前沟通，减少不确定性焦虑
3. [ ] 建立"决策清单"，把情绪噪音外化为清单条目
4. [ ] 每周一次"决策复盘"，评估哪些决策受情绪影响

### 链路2：自我价值感 → 外部竞争力

**问题表现**：
- 经常需要伴侣认可才能自信
- 伴侣否定时工作效率明显下降
- 自我怀疑频率增加

**修复动作**：
1. [ ] 建立独立的"成就清单"（不依赖伴侣反馈）
2. [ ] 设立只有自己能完成的里程碑
3. [ ] 主动给予自己正向反馈（每天至少一条）
4. [ ] 区分"事实"和"评价"，不被评价绑架

### 链路3：能量恢复能力 → 持续输出

**问题表现**：
- 下班比上班更累
- 周末无法恢复状态
- 长期感到疲惫

**修复动作**：
1. [ ] 识别"耗电行为"和"充电行为"
2. [ ] 建立回家后的"过渡仪式"（换衣服、洗澡、散步等）
3. [ ] 确保有至少一项纯粹的"自我恢复"活动
4. [ ] 与伴侣约定"充电时间"（不被干扰的个人时间）

### 链路4：家庭氛围 → 子女模型

**问题表现**：
- 在孩子面前经常争吵
- 孩子表现出焦虑或回避
- 亲子关系紧张

**修复动作**：
1. [ ] 约定"不在孩子面前冲突"，有冲突去另一个房间
2. [ ] 每周至少一次"家庭正向活动"
3. [ ] 观察并记录孩子的情绪反应
4. [ ] 主动和孩子谈论"我们是如何解决问题的"

### 链路5：生理系统 → 健康状态

**问题表现**：
- 因关系问题失眠
- 身体免疫力下降
- 经常感到身体不适

**修复动作**：
1. [ ] 把睡眠作为优先级（关系问题不能靠牺牲睡眠解决）
2. [ ] 建立睡前放松仪式（和关系问题隔离）
3. [ ] 适度运动（至少每周3次）
4. [ ] 必要时就医检查（排除生理因素）
