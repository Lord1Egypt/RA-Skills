## Known Risks and Mitigations: <br>
Risk: The skill asks the agent to build and reuse sensitive personal, emotional, and relationship memory. <br>
Mitigation: Install only when a persistent personal-companion persona is intended, restrict file access, and regularly review or delete memory under ~/self-improving and ~/characters. <br>
Risk: Heartbeat and background behavior may perform memory maintenance or proactive follow-up unexpectedly. <br>
Mitigation: Disable heartbeat or background behavior unless explicitly wanted, and require confirmation before scheduled or proactive actions. <br>
Risk: Broad default access and direct workspace edits can affect local files beyond the user's immediate request. <br>
Mitigation: Limit readable and writable paths before use and require confirmation before workspace edits. <br>
