# 报价场景模板

## 报价单

```markdown
# 报价单 | {项目名称}

**致**：{客户} | **日期**：{日期} | **有效期**：{有效期}

## 方案A — 基础版 | **{金额}元**
| 项目 | 说明 | 小计 |

## 方案B — 推荐版 | **{金额}元** 🔥
| 项目 | 说明 | 小计 |

## 方案C — 旗舰版 | **{金额}元**
| 项目 | 说明 | 小计 |

## 付款
- 首付（**{比例}%**）：签约后 {X} 工作日
- 里程碑（**{比例}%**）：{节点}后
- 尾款（**{比例}%**）：验收后 {X} 工作日
```

## 对账单

```markdown
# 对账单 | {项目名称}

**致**：{客户} | **编号**：{编号} | **日期**：{日期} | **截止**：{日期}

## 费用明细
| 序号 | 项目 | 说明 | 金额 |

## 汇总
| | 金额 |
| 小计 | **{金额}元** |
| 税费({税率}%) | **{金额}元** |
| **应付总额** | **{总金额}元** |

## 付款信息
{户名/账号/银行 或 支付宝/微信}

## 已收
| 日期 | 金额 | 备注 |
| **剩余** | **{金额}元** | |
```

## 催款函

根据逾期天数选择语气：1-3天友好 / 4-7天正式 / 8-14天严肃(提及违约金) / 15天+法律预警
