## Description: <br>
Records, structures, tags, saves, interprets, and searches Chinese-language dream journal entries. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[abycloudcn-jpg](https://clawhub.ai/user/abycloudcn-jpg) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External OpenClaw users use this skill to capture dream descriptions, save them as local Markdown journal entries, and request gentle interpretation or history search in Chinese. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Dream descriptions can contain sensitive personal information and are stored as readable local Markdown files. <br>
Mitigation: Install only when local storage is acceptable; avoid recording details that should not appear on shared devices, backups, synced folders, or compromised systems, and delete entries that should no longer be retained. <br>
Risk: Dream interpretation may be overread as certain psychological analysis. <br>
Mitigation: Keep interpretations gentle, tentative, and non-diagnostic, preserving the skill guidance to distinguish possibilities from certainty. <br>
