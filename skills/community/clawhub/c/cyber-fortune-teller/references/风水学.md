# 风水学基础

> 中国传统环境学，以藏风得水为要

## 风水基础概念

### 风

> 风者，气之动也。

气之流动为风。藏风者，使气不散失也。

### 水

> 水者，地之血也。

水者，财之象也。得水者，财气聚集。

### 风水

> 风水之法，得水为上，藏风次之。

生气遇风则散，遇水则止。

## 龙穴砂水向

### 龙

> 龙者，山之脉也。

龙指山脉的走向与气势。真龙须穿帐过峡，迎送分明。

龙之吉凶：
- 生龙：曲折生动
- 死龙：僵直无势
- 强龙：有力雄壮
- 弱龙：软塌无力

### 穴

> 穴者，气之所聚也。

穴为生气凝聚之处。穴有窝、钳、乳、突四种形局。

穴之吉凶：
- 藏风聚气者吉
- 风吹气散者凶

### 砂

> 砂者，卫护之山也。

穴前左右之山为砂。砂须环抱有情，秀丽端正。

砂之吉凶：
- 秀砂：清秀端正
- 凶砂：破碎歪斜

### 水

> 水者，财之源也。

水为财之象征。水须弯环顾穴，去水要有情。

水之吉凶：
- 曲水环抱者吉
- 直水冲射者凶

### 向

> 向者，气之入口也。

向为立向，决定气场入口的方向。

立向原则：
- 宜顺水势
- 宜收旺气
- 宜避凶煞

## 阳宅风水

### 阳宅四要

| 要项 | 说明 |
|---|---|
| 门 | 气之口，宜开吉方 |
| 房 | 居住之所，宜安静 |
| 灶 | 厨房位置，宜在吉方 |
| 井 | 水源所在，宜清润 |

### 阳宅禁忌

- 门对门：口舌是非
- 门对灶：财帛有损
- 屋前不宽：前程受阻
- 屋后无靠：贵人无力

## 阴宅风水

### 阴宅四要

| 要项 | 说明 |
|---|---|
| 龙真 | 龙脉有力 |
| 穴的 | 穴位准确 |
| 砂抱 | 砂山环护 |
| 水交 | 界水交合 |

### 阴宅宜忌

- 龙脉宜起伏屈曲
- 穴场宜藏风聚气
- 水口宜关拦紧密
- 忌风吹水散

## 峦头理气

### 峦头派

重在观察地形山势，以龙、穴、砂、水、向五要素论吉凶。

### 理气派

重在方位五行，以九宫飞星、八宅、玄空等方法论气运。

**核心要义**：风水以"气"为核心，讲究藏风得水。山环水抱为吉地，气散风冲为凶地。阴阳宅皆以龙真穴的、砂秀水抱、向吉为要。
