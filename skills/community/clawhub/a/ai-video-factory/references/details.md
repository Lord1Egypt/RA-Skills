# ai-video-factory 详细参考 v1.3.0

## 一、平台规格详解与最佳实践

### 1.1 抖音
- **基础规格**
  - 比例：9:16（竖屏）或 16:9（横屏）
  - 分辨率：1080×1920（推荐）/ 720×1280（最低）
  - 编码：H.264，码率6-10Mbps
  - 帧率：30fps（推荐）或 60fps
  - 时长：15秒-10分钟
  - 封面：1080×1920，关键帧截取或自定义
- **算法偏好**
  - 前3秒完播率权重最高，必须有强力Hook
  - 完播率 > 互动率 > 转发率 > 评论率
  - 发布后1小时内的数据决定是否进入更大流量池
  - 连续发布同系列内容有助于账号标签建立
- **最佳实践**
  - 竖屏优先，占满全屏
  - 字幕不能放在底部20%区域（会被文案遮挡）
  - 关键画面避开右侧点赞/评论按钮区域
  - 热门发布时间：12:00-13:00 / 18:00-19:00 / 21:00-23:00

### 1.2 快手
- **基础规格**
  - 比例：9:16（竖屏）
  - 分辨率：1080×1920
  - 编码：H.264，码率4-8Mbps
  - 帧率：30fps
  - 时长：15秒-10分钟
- **平台特性**
  - 快手用户偏好真实感，不宜过度精修
  - "老铁文化"：接地气、真实、有温度的内容更受欢迎
  - 社区互动性强，评论区运营很重要
- **最佳实践**
  - 内容风格偏朴实，避免过度"高级感"
  - 适当使用方言配音增加亲切感
  - 热门发布时间：11:00-12:00 / 17:00-18:00 / 20:00-22:00

### 1.3 TikTok
- **基础规格**
  - 比例：9:16
  - 分辨率：1080×1920
  - 编码：H.264
  - 时长：15秒-3分钟（推荐60秒内）
- **平台特性**
  - 需英文字幕或双语字幕
  - 音乐/音效是核心元素，卡点很重要
  - 挑战赛和热门话题是流量入口
- **最佳实践**
  - 前1-2秒就要抓住注意力
  - 使用Trending Sound可以蹭流量
  - 热门发布时间（UTC+8）：7:00 / 12:00 / 19:00

### 1.4 YouTube
- **基础规格**
  - 比例：16:9（横屏）
  - 分辨率：1920×1080（最低）或 3840×2160（4K）
  - 编码：H.264/H.265
  - 帧率：24/30/60fps
  - Shorts用9:16竖屏，≤60秒
- **平台特性**
  - 长视频和Shorts是完全不同的流量系统
  - 标题和缩略图决定点击率
  - SEO优化很重要（标题/描述/标签）
- **最佳实践**
  - Shorts和长视频分开运营
  - 缩略图需要单独设计（非视频截图）
  - 描述栏放时间戳（chapters）提升体验
  - 热门发布时间：14:00-15:00 / 17:00-18:00 / 20:00-21:00

### 1.5 B站
- **基础规格**
  - 比例：16:9（横屏为主）
  - 分辨率：1920×1080
  - 编码：H.264
  - 帧率：30fps
- **平台特性**
  - B站用户偏好长内容，故事性和干货密度更重要
  - 弹幕文化：设计"弹幕触发点"增加互动
  - 三连（点赞+投币+收藏）是核心指标
- **最佳实践**
  - 片头不要超过5秒（B站用户反感长片头）
  - 设计2-3个"弹幕触发点"（争议观点/共鸣点/猜下一步）
  - 热门发布时间：10:00-11:00 / 18:00-19:00 / 22:00-23:00

### 1.6 小红书
- **基础规格**
  - 比例：3:4 或 9:16
  - 分辨率：1080×1440 或 1080×1920
  - 编码：H.264
  - 时长：15秒-15分钟
- **平台特性**
  - 封面极其重要，需精美设计
  - 女性用户占比高，审美要求高
  - 种草/教程类内容最受欢迎
- **最佳实践**
  - 封面必须有文字标题（大字报风格）
  - 色调统一，偏暖色调更受欢迎
  - 热门发布时间：12:00-13:00 / 18:00-19:00 / 21:00-22:00

### 1.7 视频号
- **基础规格**
  - 比例：9:16 或 16:9
  - 分辨率：1080×1920
  - 编码：H.264
  - 时长：15秒-30分钟
- **平台特性**
  - 微信生态分享便利，注重社交传播性
  - 中年/银发用户比例高
  - 点赞即推荐给朋友，社交推荐权重大
- **最佳实践**
  - 内容偏正能量/知识/情感类
  - 设计"转发触发点"（实用/有面子/有价值）
  - 热门发布时间：12:00-13:00 / 18:00-19:00 / 21:00-22:00

### 1.8 西瓜视频 / 头条视频
- **基础规格**
  - 比例：16:9
  - 分辨率：1920×1080
  - 编码：H.264，码率6-10Mbps
- **平台特性**
  - 中年男性用户比例高
  - 知识科普/三农/历史/军事类受欢迎
  - 与头条打通，流量互通
- **最佳实践**
  - 内容偏深度、有信息量
  - 标题要有信息量（非标题党）
  - 热门发布时间：8:00-9:00 / 12:00-13:00 / 18:00-19:00

## 五、Hook开场模板库（50+模板）

### 5.1 六大Hook公式详解

#### 1. 悬念型（完播率最高）
- "你知道XXX最大的误区是什么吗？"
- "接下来你看到的，可能会颠覆你的认知..."
- "99%的人都不知道的XXX秘密"
- "这个发现，让科学家都震惊了"
- "我花了3个月才弄明白的真相"

#### 2. 数字冲击型（收藏率最高）
- "3个方法，让我的XXX提升了200%"
- "1天/7天/30天，从XXX到XXX"
- "只用这5步，XXX成本降低80%"
- "月薪3000到3万，我只做了这3件事"
- "100块 vs 10000块，差别在哪？"

#### 3. 反常识型（评论率最高）
- "千万别再XXX了！越XXX越XXX"
- "你以为的XXX，其实大错特错"
- "XXX根本不需要XXX"
- "被忽悠了这么多年，真相竟然是..."
- "专家都不会告诉你的XXX"

#### 4. 痛点型（互动率最高）
- "你是不是也经常遇到这种情况..."
- "每次XXX的时候，是不是特别崩溃？"
- "XXX太难了！直到我发现了这个方法"
- "还在为XXX发愁？看完这个就懂了"
- "如果你也有这个困扰，一定要看完"

#### 5. 结果前置型（转化率最高）
- "看完这个视频，你将学会..."
- "30秒后，你也能做到XXX"
- "学会这一招，XXX再也不是问题"
- "这是我花了XXX总结的精华，全部免费分享"
- "看完=省钱/省时/少走弯路"

#### 6. 对比型（分享率最高）
- "XXX vs XXX，差距居然这么大？"
- "同样XXX，为什么差距这么大？"
- "左边100块 vs 右边1000块，你能分清吗？"
- "使用前 vs 使用后，效果太惊艳了"
- "国内 vs 国外，XXX对比"

### 5.2 Hook时长控制
| 平台 | 推荐Hook时长 | 说明 |
|------|-------------|------|
| 抖音/TikTok | 1-3秒 | 极短注意力窗口，必须一击即中 |
| 快手 | 2-4秒 | 可稍长，但也要快速切入 |
| YouTube Shorts | 1-3秒 | 同TikTok |
| YouTube长视频 | 5-15秒 | 可设置悬念，但别太拖 |
| B站 | 3-8秒 | 观众容忍度较高，但别超过10秒 |
| 小红书 | 2-5秒 | 封面决定点击，Hook决定留存 |
| 视频号 | 3-5秒 | 中年用户需要稍长理解时间 |

## 六、字幕样式模板库

### 6.1 标准字幕
```
字体：思源黑体 Bold / 阿里巴巴普惠体 Bold
大小：视频高度的5-8%（竖屏）或 4-6%（横屏）
颜色：白色 #FFFFFF
描边：2px 黑色 #000000
阴影：2px偏移 50%透明度
位置：底部安全区上方10%（竖屏）/ 底部15%（横屏）
行间距：1.5倍
```

### 6.2 关键词高亮
```
高亮色方案A：金色 #FFD700（通用推荐）
高亮色方案B：红色 #FF4444（强调/警示）
高亮色方案C：绿色 #00FF88（正面/数据）
高亮色方案D：蓝色 #4488FF（科技/专业）
效果：放大1.2倍 + 颜色变化 + 轻微抖动
触发规则：名词/数字/核心动词/品牌名
频率：每句1-2个高亮词，不超过3个
```

### 6.3 逐字显示效果
```
速度：与语音完全同步
效果：当前字高亮（金色），已读字变白，未读字半透明
字体大小：比标准字幕大20%
适用：快节奏视频、知识科普、教程类
```

### 6.4 打字机效果
```
速度：略快于语音（提前0.5秒）
效果：逐字出现 + 打字机音效 + 光标闪烁
字体：等宽字体（如 Courier / 思源等宽）
适用：代码教程、技术类、极客风格
```

### 6.5 花字特效
```
触发：情绪高潮点/关键信息/数字
效果：放大弹出 + 颜色渐变 + 粒子效果
持续：1-2秒
频率：每分钟不超过3次（避免视觉疲劳）
```

## 七、分镜脚本模板

### 7.1 知识科普类模板（60秒）
```
[0-3s] Hook：悬念/反常识开场
  画面：冲击性画面 + 文字标题
  配音：抛出问题/制造好奇
  情绪：好奇⬆️

[3-15s] 背景铺垫
  画面：历史画面/数据图表/对比图
  配音：交代背景/现状
  情绪：平稳➡️

[15-40s] 核心内容（2-3个知识点）
  画面：动画演示/实拍/图解
  配音：逐一讲解，每个点10秒
  情绪：兴趣⬆️⬆️

[40-52s] 实际应用/案例
  画面：真实案例/前后对比
  配音：联系实际，强化记忆
  情绪：认同⬆️

[52-60s] CTA + 预告
  画面：账号信息 + 下期预告
  配音：引导关注+互动
  情绪：期待⬆️
```

### 7.2 种草推荐类模板（30秒）
```
[0-3s] Hook：价格冲击/效果对比
  画面：产品特写 + 价格标签
  配音：制造性价比感知

[3-8s] 产品1：核心卖点
  画面：产品展示 + 使用场景
  配音：1个核心卖点 + 1个数据

[8-13s] 产品2：差异化卖点
  画面：对比画面 + 评分
  配音：与竞品对比优势

[13-25s] 快速轮播（产品3-5）
  画面：快节奏切换
  配音：每个产品3秒概括

[25-30s] 总结 + CTA
  画面：排名表格
  配音：引导评论互动
```

### 7.3 热点资讯类模板（45秒）
```
[0-5s] Hook + 热点画面快剪
  画面：新闻画面/社交媒体截图
  配音：一句话概括热点

[5-25s] 事件详情（3个要点）
  画面：图文+数据+时间线
  配音：客观叙述+关键数据

[25-38s] 深度分析/影响解读
  画面：对比图/专家观点/预测
  配音：独到见解+多角度分析

[38-45s] 互动引导
  画面：投票/评论区截图
```

## 八、异常处理方案

| 异常场景 | 检测条件 | 处理策略 | 降级方案 |
|----------|----------|----------|----------|
| 素材不足 | 匹配度<60% | 自动降低匹配阈值→用AI生成补充 | 提示用户手动补充素材 |
| TTS合成失败 | 返回错误码 | 换用备选音色→降低语速重试 | 提示用户上传配音文件 |
| BGM版权冲突 | 版权检测告警 | 自动替换为同类免费BGM | 标记提醒用户确认 |
| 渲染超时 | >3分钟视频 | 降低预览分辨率→分段渲染 | 合并输出+提示等待 |
| 多平台规格冲突 | 比例不兼容 | 以最严格规格为基准→其他平台裁剪 | 保留最大画面区域 |
| 用户上传素材不合格 | 分辨率<720p/有水印 | 自动增强（锐化/调色/降噪） | 提醒用户替换素材 |
| 字幕超长溢出 | 单行>20字 | 自动拆行→缩小字号→精简文案 | 标记提醒人工调整 |
| 原创度不足 | 检测<70% | 自动改写文案→替换素材→调整结构 | 提醒用户手动修改 |
| 音量不达标 | 整体<-16 LUFS | 自动音量归一化→压缩动态范围 | 标记提醒手动调整 |
| 内容安全告警 | 触发敏感词/画面 | 自动替换/模糊处理→标记审核 | 拒绝生成并告知原因 |
