## 📋 父母觉醒自评表（5级）

**L1 无觉察**
- [ ] 我主要用奖惩来管教孩子
- [ ] 我认为孩子的"问题"是他自己的问题

**L2 觉察开始**
- [ ] 我意识到自己的反应有时过度
- [ ] 我有时会想"我小时候也是这样被对待的"

**L3 有意识（有暂停）**
- [ ] 我能在发火前感觉到身体信号
- [ ] 我有时会说"等一下，让我冷静一下"

**L4 反思行动**
- [ ] 我会为自己的失控道歉
- [ ] 我会和孩子讨论我的感受

**L5 整合连接**
- [ ] 我理解自己的模式来自哪里（代际传递）
- [ ] 我能在孩子面前承认自己的脆弱
- [ ] 我开始用"慈悲是体"来指导我和孩子的互动

| 等级 | 建议重点 |
|------|---------|
| L1 | 先从"感受"开始——注意自己什么时候会发火 |
| L2 | 从反思日记开始——记录自己的情绪触发点 |
| L3 | 练习"暂停"——愤怒按停术，6秒空间 |
| L4 | 坚持道歉——每次失控后修复，修复即成长 |
| L5 | 走向"慈悲是体"——从理解自己到理解孩子 |

## 📝 父母教养风格评估问卷：四种类型20题自评（v0.9.48 NEW）

> 数据来源：`scripts/parenting_assessment.py`（跨技能参考）

### 一、四种教养风格对照表

| 类型 | 名称 | 特征描述 |
|------|------|----------|
| authoritarian | **专制型** | 高要求、低回应。孩子服从为上，惩罚多于沟通 |
| authoritative | **权威型** | 高要求、高回应。最理想的教养方式——有规则、有温度 |
| permissive | **溺爱型** | 低要求、高回应。避免冲突，回避界限 |
| neglectful | **忽视型** | 低要求、低回应。缺乏陪伴与关注 |

---

### 二、20题自评问卷（1-5分）

> 评分标准：1=完全不符合，2=偶尔符合，3=有时符合，4=经常符合，5=完全符合

#### 【专制型】— 高要求、低回应

| 编号 | 题目 |
|------|------|
| a1 | 我认为孩子应该完全服从父母的决定 |
| a2 | 我不解释为什么制定规则，孩子遵守就是了 |
| a3 | 当孩子不听话时，我会用惩罚而不是沟通 |
| a4 | 我认为严格管教是爱孩子的表现 |
| a5 | 我不允许孩子质疑我的决定 |

#### 【权威型】— 高要求、高回应（最理想）

| 编号 | 题目 |
|------|------|
| b1 | 我会解释制定规则的原因，并倾听孩子的想法 |
| b2 | 我对孩子有明确的期望，但也会考虑他们的意见 |
| b3 | 我会在坚定原则的同时，表达对孩子的关爱 |
| b4 | 我会根据孩子的表现调整管教方式 |
| b5 | 我鼓励孩子独立思考，同时设定清晰的界限 |

#### 【溺爱型】— 低要求、高回应

| 编号 | 题目 |
|------|------|
| c1 | 我尽量避免与孩子发生冲突 |
| c2 | 我认为孩子高兴最重要，规则可以灵活 |
| c3 | 我很少对孩子说"不"或设置限制 |
| c4 | 我尽量满足孩子的所有要求 |
| c5 | 我认为孩子会自己学会规矩 |

#### 【忽视型】— 低要求、低回应

| 编号 | 题目 |
|------|------|
| d1 | 我经常忙于工作，没有时间陪伴孩子 |
| d2 | 我不了解孩子在学校的情况 |
| d3 | 我很少参与孩子的日常活动 |
| d4 | 我对孩子的情绪不太关注 |
| d5 | 我认为孩子应该自己管自己 |

---

### 三、评分解读

**各风格得分 = 该类型5题得分之和（满分25分）**

| 得分范围 | 风格倾向 | 建议 |
|----------|----------|------|
| 专制型最高 | 权威型倾向不足 | 增加沟通，解释规则原因，尊重孩子想法 |
| 权威型最高 | **最理想** | 继续保持，有规则、有回应 |
| 溺爱型最高 | 界限不足 | 适当设立明确规则和界限，帮助孩子建立规则感 |
| 忽视型最高 | 投入不足 | 增加陪伴时间，关注孩子的情感需求 |

> **注**：大多数父母是混合型，关键看哪些维度得分最高。改变从"认错+学+做"开始。

## 🛠️ 家庭教育评估工具与专业资源清单（v0.9.46 NEW）

本节整理自「父母教育研究数据库」，为家庭教育指导实践提供实用评估工具与专业资源。

### 一、家庭教育评估工具

#### 1. 亲子关系质量评估

| 工具名称 | 用途 | 适用年龄 | 来源 |
|---------|------|---------|------|
| **亲子关系质量问卷（PQ）** | 评估亲子情感联结、沟通质量、冲突解决 | 3-18岁儿童家长 | 心理学标准化量表 |
| **依恋风格自评问卷** | 识别家长的依恋模式（安全型/焦虑型/回避型） | 成人 | 基于Bowlby依恋理论 |
| **家庭环境量表（FES）** | 评估家庭凝聚力、表达性、冲突性、组织性 | 全年龄 | 中国心理学会推荐 |

**使用建议**：初筛使用自评问卷，深入评估建议寻求专业指导师支持。

#### 2. 儿童行为问题筛查

| 工具名称 | 筛查内容 | 适用年龄 | 获取渠道 |
|---------|---------|---------|---------|
| **Achenbach儿童行为量表（CBCL）** | 行为问题（内化/外化）、社交能力 | 6-18岁 | 医疗机构/学校心理室 |
| **Conners家长/教师评定量表** | 多动、注意力不集中、品行问题 | 3-17岁 | 儿童心理科 |
| **SDQ（优势与困难问卷）** | 情绪症状、行为问题、同伴关系 | 3-16岁 | WHO授权中文版免费使用 |

**预警指标**：若筛查显示异常，建议在2-4周内复查，并考虑专业评估。

#### 3. 父母自我效能量表

| 工具名称 | 测量维度 | 用途 |
|---------|---------|------|
| **父母效能感量表（PSOC）** | 父母胜任感、子女控制感 | 评估家长对自身教养能力的信心 |
| **父母压力指数量表（PSI）** | 亲子子系统压力、儿童特征压力、全家压力 | 识别过度养育压力 |

### 三、家庭教育指导师职业化

#### 1. 职业认证路径

```
入门：家庭教育指导师（基础培训）→ 中级：家庭教育顾问 → 高级：家庭教育督导师
```

| 等级 | 培训时长 | 核心能力 | 认证机构 |
|-----|---------|---------|---------|
| **初级** | 80-120课时 | 基础咨询技巧、家庭评估 | 人社部/行业协会 |
| **中级** | 200+课时 | 个案干预、家庭治疗 | 中国心理学会等 |
| **高级/督导师** | 300+课时 | 培训督导、体系建立 | 专业机构认证 |

#### 2. 专业能力框架

- **评估能力**：使用标准化工具评估亲子关系、行为问题
- **咨询能力**：积极倾听、共情沟通、冲突调解
- **干预能力**：制定个性化家庭教育方案
- **协调能力**：链接学校、社区、医疗资源

#### 3. 职业伦理要点

- 保护家庭隐私（保密原则）
- 避免利益冲突（商业推广自律）
- 持续专业发展（督导、个人体验）
- 识别转介时机（超出能力范围及时转介）

#### 4. 实践指引

1. **初筛**：使用标准问卷评估家庭需求
2. **建档**：记录家庭背景、主要诉求、评估结果
3. **制定方案**：根据评估结果制定个性化指导计划
4. **跟踪反馈**：定期评估效果、调整方案
5. **多学科协作**：复杂案例联合学校心理师、儿科医生

### 四、使用说明

- **初学者**：建议从父母效能训练（PET）和正面管教基础课程入手
- **有基础者**：可学习依恋理论、家庭系统治疗等进阶内容
- **专业从业者**：参考国际认证体系，持续接受督导
