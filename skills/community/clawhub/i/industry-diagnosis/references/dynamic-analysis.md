# 细分行业 & 新兴行业动态分析

当用户提问的行业超出申万一级分类（如下级细分行业、新兴跨界行业、境外行业），
无需恐慌——使用本方法动态生成产业链结构。

## 方法：三分钟快速推导产业链

### Step 1: 快速分类

判断该行业属于哪类，采用该类的通用分析骨架：

| 类别 | 特征 | 产业链通用模板 |
|------|------|---------------|
| **资源/周期** | 产品是标准品、价格由供需决定 | 矿产/原料 → 冶炼/加工 → 终端需求 |
| **制造/工业** | 投入原料、产出制成品 | 零部件/材料 → 组装/制造 → 下游/用户 |
| **消费/品牌** | 面对消费者、品牌营销驱动 | 原料/代工 → 品牌/渠道 → 消费者 |
| **科技/软件** | 技术驱动、研发投入大 | 技术/IP → 产品/平台 → 用户/企业 |
| **金融/服务** | 无形产品、监管壁垒高 | 资金/资源 → 产品/服务 → 客户 |
| **医药/医疗** | 研发监管双壁垒 | 研发/原料 → 产品/审批 → 医院/药店 |

### Step 2: 三段式产业链推导

```
这个行业生产什么？（中游核心价值环节）
  ├ 上游需要什么原料/技术/设备？
  └ 卖给谁、怎么用、什么驱动需求？（下游）
```

### Step 3: 关键指标推导

每个环节问三个问题：

| 问题 | 对应指标类型 |
|------|-------------|
| 这个环节的**主要成本**是什么？ | 原料价格、人力成本、技术投入 |
| 这个环节的**收入来源**是什么？ | 产品价格、销量、服务费 |
| 什么**外部因素**影响这个环节？ | 政策、技术革命、替代品 |

### Step 4: 数据获取

根据推导出的指标，用 mx-finance-data / 商品期货 / web_search 采集数据。
找不到完美匹配时用近似替代，并在分析中注明。

## 示例：动态分析细分行业

### 案例1：逆变器（电力设备下属）

```
Step 1: 属"制造/工业"类
Step 2: 产业链
  IGBT/功率器件 → 逆变器组装(阳光/锦浪/固德威) → 光伏电站/储能系统/户用
Step 3: 关键指标
  - 上游: IGBT(英飞凌价格+国产替代), 铜/铝(电感/散热器成本)
  - 中游: 逆变器出口量(海关), 户用/集中式出货结构, 单瓦售价(ASP)
  - 下游: 光伏新增装机, 欧洲(户用)/美国(大储)需求, 国内配储比例
  - 技术: 微型逆变器 vs 组串式 vs 集中式市占率变化
  - 独特: 全球逆变器出货量排名(阳光电源 vs Huawei vs SMA)
Step 4: 额外指标
  板块估值: 电力设备PE分位, 阳光电源/锦浪相对强度
  资金: 北向资金流入新能源板块
```

### 案例2：AI Agent（跨行业—计算机+传媒）

```
Step 1: 跨"科技/软件"类别
Step 2: 产业链
  大模型API调用/算力 → AI Agent平台 → 企业/个人用户
Step 3: 关键指标
  - 算力层: 英伟达GPU价格+发货周期, 国产算力(昇腾)供给
  - 模型层: 大模型API调用价格(持续下降), 开源vs闭源
  - 应用层: AI Agent产品月活/付费转化, 企业Agent渗透率
  - 政策: 中国AI监管政策(算法备案, 生成式AI管理办法)
Step 4: 额外指标
  计算机/传媒板块PE, 北向资金, 英伟达(NVDA)股价（全球AI情绪锚）
```

### 案例3：低空经济（新兴行业—国防军工+机械+交运）

```
Step 1: 属"制造/工业"+"服务"混合
Step 2: 产业链
  航空材料/动力电池/传感器 → eVTOL/无人机整机制造 → 空管系统/运营服务 → 物流/出行/巡检
Step 3: 关键指标
  - 政策端: 低空空域开放进度, 各省份低空经济政策文件数
  - 整机制造: 亿航/小鹏汇天适航证进度, 订单量
  - 产业链: 碳纤维/电机/飞控/电池—各环节国产化率
  - 基础设施: 起降场建设, 空管系统(莱斯信息)
  - 需求: 物流无人机配送量(美团/顺丰), 短途出行场景
Step 4: 额外指标
  TMT板块热度, 军工etf资金流, 政策新闻密度(web_search)
```

## 关键原则

1. **不要追求完美过拟合**：找到 3-5 个最关键的驱动指标即可，比列 20 个次要指标更有用
2. **上下游利润分配**：哪个环节最稀缺，利润就往哪流
3. **替代品威胁**：分析行业时永远问"如果有新技术替代它怎么办？"
4. **政策是超级变量**：尤其是中国行业，政策力度往往超过市场力量
5. **标注置信度**：如果是临时推导的指标，在分析中注明"该指标为参考值，建议交叉验证"
