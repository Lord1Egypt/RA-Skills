# 加密货币监控与告警

## 触发条件

- 用户说"监控XX币价"、"监控比特币"、"价格提醒"
- 用户说"涨到XX提醒我"、"跌破XX报警"、"涨跌幅告警"
- 用户说"加密货币行情"、"数字货币走势"、"币圈动态"
- 用户说"BTC/ETH/SOL价格"、"查看XX币"
- 用户说"市场日报"、"加密货币报告"、"币圈早报"

## 核心流程 (5步)

1. **意图解析**：识别请求类型（价格查询/告警设置/行情报告）
2. **数据获取**：从Binance/CoinGecko/CoinCap获取价格数据
3. **告警验证**：校验告警规则（条件、目标价格、检查间隔）
4. **告警存储**：保存告警配置，返回告警ID
5. **结果输出**：返回价格信息/告警确认/行情报告

## 输出格式

```
🪙 币种：ETH/USDT
💰 当前价格：$2,985.45
📈 24h涨跌：-1.23%
📊 数据更新时间：[时间]
```

## 关键约束

1. 只监控不下单，不提供交易执行
2. 告警目标需>0，涨幅建议0.1%-100%
3. 检查间隔60-3600秒
4. 优先使用Binance API，降级到CoinGecko/CoinCap
5. 数据仅供参考，不构成投资建议
