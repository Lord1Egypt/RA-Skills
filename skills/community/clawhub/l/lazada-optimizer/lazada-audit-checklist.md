# Lazada Listing & Seller Audit Checklist

Run per listing (sections 1–5) and per store (sections 6–8). Sections 1–2 failures are blocking.

## 1. Title
- [ ] Follows Brand + Model + Key Spec + Category Keyword
- [ ] 60–120 characters
- [ ] No emoji, brackets, or symbols (【】★)
- [ ] No promo words (HOT, SALE, BEST, MURAH, PROMO)
- [ ] No keyword repetition
- [ ] One clear primary category keyword

## 2. Attributes
- [ ] 100% of REQUIRED attributes filled
- [ ] 100% of recommended attributes filled
- [ ] Values accurate (no "OEM" placeholders where real specs exist)
- [ ] Units match market convention (cm/inch, voltage, plug type)
- [ ] Variant attributes (size/color) mapped to Lazada's standard values, not free text

## 3. Gallery & video
- [ ] 8 image slots used (minimum 5)
- [ ] All images ≥800×800 (≥1000×1000 for zoom)
- [ ] Main image: white background, product ~85% of frame, no text/watermark
- [ ] At least 2 lifestyle/context shots
- [ ] Size chart or spec infographic present (mandatory for fashion)
- [ ] 15–30s video uploaded, hero function in first 5 seconds

## 4. Description
- [ ] Structured sections/bullets, not one text wall
- [ ] Secondary keywords covered (not the title keyword again)
- [ ] Specs/dimensions table included
- [ ] Warranty and care/usage info stated
- [ ] Top 3 buyer objections pre-answered
- [ ] Not duplicated verbatim across variants/SKUs

## 5. Price & reviews
- [ ] Price within sane range of category median (not high outlier)
- [ ] 30-day lowest price tracked and recorded before any campaign plan
- [ ] Rating ≥4.0 (below: triage 1–2 star reviews first)
- [ ] Review replies present on negative reviews

## 6. Seller score
- [ ] Chat response rate ≥95% within 30 min (business hours)
- [ ] Auto-reply + ≥10 templated answers configured
- [ ] On-time dispatch: packed within 24h consistently
- [ ] Return rate audited monthly with top-3 reasons identified
- [ ] No open penalty points in Seller Center

## 7. Campaigns
- [ ] Next 60 days of campaigns mapped with registration deadlines
- [ ] Discount thresholds computed off 30-day lowest (margin verified)
- [ ] Stock commitments confirmed with supplier
- [ ] Vouchers configured where required
- [ ] No pre-campaign price inflation in the last 30 days
- [ ] Post-campaign dispatch capacity planned

## 8. Multi-market (if operating >1 country)
- [ ] Titles/keywords localized per market language
- [ ] Prices benchmarked vs. top-10 local competitors per subcategory
- [ ] Size/unit conversions applied
- [ ] Local campaign calendar mapped (incl. Ramadan/local festivals)
- [ ] Local-language chat templates ready
