# 胰岛素释放试验详细参考值与判读补充

## 胰岛素参考范围（单位：mU/L）
- 空腹：4.2～16.2
- 1小时（60分钟）：41.8～109.8
- 2小时（120分钟）：26.2～89.0
- 3小时（180分钟）：5.2～43.0

## 血糖参考范围（单位：mmol/L）
- 空腹：<5.1
- 1小时：<10.0
- 2小时：<8.5
- 3小时：接近空腹水平

## 异常模式图谱说明
- **1型糖尿病典型曲线**：胰岛素全程低平，无明显峰值
- **2型糖尿病典型曲线**：空腹胰岛素正常或升高，峰值延迟至120分钟，下降缓慢
- **胰岛素抵抗（如PCOS）**：空腹胰岛素>16.2，服糖后胰岛素显著升高（如1小时>110），但血糖可能仍在正常范围

## 注意事项
- 不同实验室检测方法（如化学发光法 vs 放免法）可能导致参考值差异，应以本机构标准为准
- 肥胖、肝病、肢端肥大症等也可导致高胰岛素血症，需结合临床鉴别
