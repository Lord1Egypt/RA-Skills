# 酶联膜受体信号转导分析

## 何时使用
- 用户询问某类配体（如生长因子、细胞因子、病原体分子）如何激活细胞信号通路
- 需要区分不同酶联膜受体的结构特征与信号机制
- 探讨靶向受体或其下游通路的药物设计原理（如抗肿瘤或抗炎治疗）
- 分析信号转导异常与疾病（如癌症、自身免疫病）的关联

## 执行步骤

### 1. 识别受体类型
- **自身含酶结构的受体**：
  - **酪氨酸激酶受体**：配体（如胰岛素、EGF）结合诱导二聚化 → 自身酪氨酸残基磷酸化 → 激活下游底物
  - **丝氨酸-苏氨酸激酶受体**：如TGF-β受体，激活后磷酸化Smad蛋白 → 调控基因转录
- **自身无酶但招募胞内酶的受体**：
  - **细胞因子受体**：激活Jak激酶 → 磷酸化STAT → 核内转录调控
  - **Toll样受体（TLR）**：识别病原体成分 → 激活IRAK → NFκB通路 → 炎症反应
  - **TNF-α受体**：三聚化后招募TRADD/RIP → 激活NFκB或凋亡通路

### 2. 追踪下游信号通路
- **酪氨酸激酶受体主要激活三条通路**：
  - PLCγ → Ca²⁺释放 → PKC激活
  - Ras → Raf → MEK → ERK（MAPK通路）→ 增殖/分化
  - PI₃K → Akt → mTOR → 细胞存活/代谢
- 所有通路最终通过调控核内基因表达产生长期生物学效应（如增殖、炎症、凋亡）

### 3. 评估药理干预点
- **酪氨酸激酶抑制剂**（如伊马替尼）：阻断受体自磷酸化或下游激酶活性，用于靶向抗肿瘤治疗
- **抗TNF-α单克隆抗体**（如英夫利昔单抗）：中和配体，抑制炎症信号，用于类风湿关节炎等

### 4. 验证关键机制
- 确认信号起始于**膜受体-配体结合**
- 检查是否存在**自身磷酸化**（autophosphorylation）以增强酶活性
- 确认最终效应是否通过**转录调控**实现

> 注意：本技能不适用于细胞内受体（如甾体激素受体）或G蛋白偶联受体（GPCR）。
