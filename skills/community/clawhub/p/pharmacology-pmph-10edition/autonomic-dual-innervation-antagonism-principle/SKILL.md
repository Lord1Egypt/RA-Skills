# 自主神经系统双重支配与效应拮抗原则

## 何时使用
- 用户描述某器官（如窦房结、虹膜、支气管、胃肠道、膀胱、血管）同时受交感与副交感神经支配
- 需预测两类神经同时激活时的净生理效应（如心率加快还是减慢）
- 分析为何某些器官表现出明显的功能拮抗，而其他则不然

> 注意：仅适用于具有双重神经支配的器官；单一支配器官（如皮肤血管）不适用。

## 执行步骤

### 1. 确认器官是否具备双重支配
适用器官包括：
- 窦房结
- 虹膜
- 支气管平滑肌
- 胃肠道平滑肌
- 膀胱壁
- 大部分血管（但注意：皮肤/内脏血管通常仅交感支配）

排除：汗腺（虽属交感但递质为ACh）、竖毛肌、多数血管等单一支配结构。

### 2. 明确两类神经的典型效应
| 器官 | 交感效应（去甲肾上腺素能） | 副交感效应（胆碱能） |
|------|--------------------------|---------------------|
| 窦房结 | 心率↑（β₁受体） | 心率↓（M₂受体） |
| 虹膜 | 瞳孔扩大（α₁） | 瞳孔缩小（M₃） |
| 支气管 | 舒张（β₂） | 收缩（M₃） |
| 胃肠平滑肌 | 蠕动↓、括约肌收缩（α/β） | 蠕动↑、括约肌舒张（M） |
| 膀胱壁 | 舒张（β₂） | 收缩（M₃） |

### 3. 判断优势神经效应
当两类神经同时兴奋时，最终效应由以下因素决定：
- **神经张力**（基础放电频率）
- **受体密度与敏感性**
- **当前生理状态**（如应激 vs 休息）

**典型示例**：
- 窦房结：副交感张力通常占优 → 净效应为心率减慢
- 支气管：交感占优于运动状态，副交感主导于静息状态

### 4. 输出最终效应
根据上述分析，明确回答器官表现出的具体生理变化（如“心率减慢”、“支气管收缩”等）。

> **注意例外**：汗腺虽由交感神经支配，但释放ACh作用于M受体，不属于典型拮抗场景。
