### Overview

- **AI-Driven Figure Generation**: Automatically convert natural language descriptions into professional academic figures.
- **SCI Journal Standards**: Built-in templates for top journals like Nature, Science, and IEEE.
- **Broad Chart Support**: Generate data charts, neural network architectures, experimental flowcharts, and mechanism diagrams.
- **MCP Integration**: Ready to use with Claude Desktop, Cursor, and other MCP-compatible tools.
