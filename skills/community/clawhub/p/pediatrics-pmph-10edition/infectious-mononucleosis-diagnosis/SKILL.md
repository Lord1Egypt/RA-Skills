# 传染性单核细胞增多症诊断

## 适用对象
疑似传染性单核细胞增多症（IM）患者，尤其是伴有以下表现者：
- 发热
- 咽峡炎
- 肝脾肿大
- 全身淋巴结肿大（特别是颈部和滑车淋巴结）

## 执行步骤

1. **评估典型临床表现**：确认是否存在发热、咽峡炎、肝脾肿大及特征性淋巴结肿大。

2. **检查外周血象**：
   - 若异型淋巴细胞比例 >10% 或绝对值 >1.0×10⁹/L，则具有诊断意义。

3. **进行血清学检测**：
   - **嗜异性凝集试验（HAT）**：效价 ≥1:64 且经豚鼠肾吸收后仍阳性可支持诊断；注意5岁以下儿童常为阴性。

4. **执行EBV特异性抗体检测（关键确诊依据）**：
   - VCA-IgM 阳性 → 提示新近EBV感染；
   - 低亲和力VCA-IgG 阳性 → 提示急性原发感染；
   - EA-IgG 一过性升高 → 提示近期感染或病毒复制活跃；
   - 急性期与恢复期双份血清VCA-IgG抗体效价呈4倍以上增高 → 最特异的确诊依据。

5. **辅助检测（如需要）**：
   - EBV-DNA（RT-PCR法）阳性提示病毒血症，可作为支持证据。

6. **综合判断**：
   - 满足典型临床表现 + 任一特异性血清学阳性（尤其是VCA-IgM或低亲和力VCA-IgG）即可确诊IM。

7. **鉴别诊断**：
   - 排除巨细胞病毒、腺病毒、支原体等引起的类似综合征，其中巨细胞病毒感染最易误诊为IM。

> **注意**：5岁以下儿童因嗜异性凝集试验常阴性，应优先依赖EBV特异性抗体检测。
