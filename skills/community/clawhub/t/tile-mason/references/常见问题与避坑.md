# 瓦匠常见问题与避坑指南

## 装修业主常见问题

### 1. 空鼓
**症状**：敲击瓷砖有空洞声
**原因**：
- 铺贴时砂浆不饱满
- 基层未清理干净
- 瓷砖胶未充分搅拌/晾置时间过长
- 全瓷砖未使用瓷砖胶（用水泥浆贴）

**应对**：
- 边角空鼓可注浆修复（灌浆法）
- 中间空鼓或大面积空鼓 → 必须砸掉重铺
- 验收时用空鼓锤每块敲击

### 2. 瓷砖不平、缝隙不均
**原因**：
- 瓷砖本身平整度差（劣质砖）
- 未使用找平器（卡子）
- 铺贴时未拉线

**应对**：
- 铺贴前检查瓷砖平整度
- 推荐使用找平器 + 十字卡子
- 缝隙误差超过 ±0.5mm 需要返工

### 3. 卫生间积水
**原因**：
- 地漏处未做回字铺贴
- 排水坡度不足
- 地面找平时未做坡度

**应对**：
- 验收时倒水测试，积水处标记返工
- 淋浴区正常水量 30 秒内排干

### 4. 墙面瓷砖脱落
**原因**：
- 全瓷砖上墙用水泥砂浆（吸不住）
- 未刷背胶
- 基层起砂/空鼓

**应对**：
- 全瓷砖必须用瓷砖胶 + 背胶（双保险）
- 大板需使用 C2 级瓷砖胶

### 5. 砖缝发黄/发霉
**原因**：
- 卫生间用了普通白水泥勾缝
- 美缝不到位
- 潮湿环境未选环氧类材料

**应对**：
- 厨卫必须用环氧彩砂或聚脲
- 美缝前确保砖缝完全干燥

### 6. 瓷砖泛碱（白斑/白粉）
**原因**：
- 水泥中的氢氧化钙随水析出
- 防水没做好
- 铺贴时水分过多

**应对**：
- 干区用专用清洁剂清理
- 湿区返工防水
- 使用低碱水泥或阻碱剂

## 从业者常见施工通病

### 瓷砖胶使用误区
- ❌ 瓷砖胶掺水泥 → 严重降低粘结力
- ❌ 晾置时间过长 → 结膜后粘不上
- ❌ 单面上浆（刷砖不刷墙）→ 大砖必须双面刮浆
- ❌ 厚贴法用瓷砖胶 → 收缩太大开裂

### 防水常见误区
- ❌ 防水刷太厚（一次成型）→ 干缩开裂
- ❌ 防水未干就刷第二遍 → 防水层失效
- ❌ 闭水前未做保护层 → 破损后漏水
- ❌ 门口未做止水坎 → 水渗到客厅

### 大板铺贴注意事项
- 必须双人以上施工
- 使用吸盘搬运
- 必须双面上浆
- 使用找平系统（调平器）
- 胶粘剂等级 ≥ C2

### 地漏铺贴要点
- 地漏要处于最低点
- 回字型铺贴（四角切砖，中间放地漏）
- 地漏四周坡度 ≥ 2%（比普通区更陡）
- 地漏边缘与瓷砖平齐

## 避坑清单（业主版）

- [ ] 瓷砖进场先检查色号批号是否一致
- [ ] 铺贴前要求排版图（避免窄条在显眼处）
- [ ] 全瓷砖要问是否用瓷砖胶 + 背胶
- [ ] 卫生间防水高度确认（淋浴区 1.8m+）
- [ ] 闭水试验自己在楼下看一遍
- [ ] 地漏处回字铺贴确认
- [ ] 海棠角还是阳角条，提前约定
- [ ] 验收带空鼓锤每块敲
- [ ] 留缝宽度确认（普通砖 2-3mm，仿古砖 3-5mm）
- [ ] 墙压地还是地压墙，提前沟通
