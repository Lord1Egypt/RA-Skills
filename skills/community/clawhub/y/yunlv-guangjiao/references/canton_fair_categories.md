## 常用采购商画像标签

### 按采购商类型
- **进口商 (Importer)**：直接进口产品的海外公司
- **批发商 (Wholesaler)**：批量采购后分销
- **分销商 (Distributor)**：有区域独家代理权的分销商
- **零售商 (Retailer)**：零售渠道（电商/实体店）
- **品牌商 (Brand Owner)**：自有品牌，寻求OEM/ODM
- **制造商 (Manufacturer)**：采购原材料/配件用于生产
- **采购代理 (Buying Agent)**：代表他人采购的中间商

### 按采购规模
- **微型企业**：年采购额 < $50,000
- **小型企业**：$50,000 - $500,000
- **中型企业**：$500,000 - $5,000,000
- **大型企业**：$5,000,000 - $50,000,000
- **跨国企业**：> $50,000,000

## 高价值采购商识别信号

### 🔥 强信号（推荐优先开发）
- [ ] 已在广交会参展超过3届（持续采购意愿）
- [ ] 采购品类与我方产品高度匹配（产品匹配度>80%）
- [ ] 年采购额超过$500,000（量级大）
- [ ] 有明确的采购周期（规律性采购）
- [ ] 有采购代理或采购办（组织化采购）

### 🌡️ 中信号（次优先）
- [ ] 在LinkedIn有采购相关动态
- [ ] 公司网站有明确的产品分类
- [ ] 参加了多国展会（国际化程度高）
- [ ] 有多个供应商（竞争切入机会）
- [ ] 最近有招聘采购相关岗位（业务扩张）

### ❄️ 弱信号（需谨慎）
- [ ] 仅有公司名称，无联系方式
- [ ] 采购量极小（MOQ匹配度低）
- [ ] 已有长期稳定供应商（替换成本高）
- [ ] 采购品类过于分散（专注度低）
