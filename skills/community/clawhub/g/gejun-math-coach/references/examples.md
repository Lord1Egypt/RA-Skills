## S1 拆解示例

**题目：求曲线 y = x² - ln(ax+1) + ax 在 x=0 处的切线方程。**

```
📌 命题意图分析
   • 考点：导数几何意义 + 含参函数定义域
   • 难度定位：第(I)问 ★★☆（基础），第(II)问 ★★★（综合）
   • 易错陷阱：ln(ax+1) 定义域要求 ax+1>0

🧭 解题路线图（骨架）
   Step 1: 求 f'(x)（复合函数求导，⚠️注意内层 ax）
   Step 2: 计算 f'(0)（代入，⚠️注意定义域条件）
   Step 3: 写切线方程（点斜式）

💡 葛军点评：f'(0) 就是切线的斜率。
```

## S2 解题示例

```
Step 1: f'(x) = 2x - a/(ax+1)
Step 2: f'(0) = -a²
Step 3: 切线方程：y = -a²x

答案：y = -a²x（定义域需 ax+1>0）
```

## S3 多解示例

**题目：求 sin²x + cos²x 的值。**

```
【解法一】定义法 —— 最直接
  sin²x + cos²x = 1（勾股定理）

【解法二】几何法 —— 揭示本质
  单位圆上点 (cos x, sin x)，到原点距离恒为1

【解法三】复数法 —— 通性强
  Euler公式 e^(ix) = cos x + i sin x
  |e^(ix)|² = cos²x + sin²x = 1

📊 对比：
  • 解法一 → 最快，考试首选
  • 解法二 → 揭示几何本质
  • 解法三 → 适合复数专题
```

## S7 苏格拉底引导完整对话示例

**题目：求曲线 y = x² - ln(ax+1) + ax 在 x=0 处的切线方程。**

```

AI：✅ 导数的几何意义是什么？
学生：切线的斜率。
AI：✅ 那么 f'(0) 就是我们要找的斜率。
  怎么求 f'(0)？
学生：先求 f'(x)，再代入 x=0。
AI：✅ 很好。ln(ax+1) 是复合函数，
  链式法则怎么用？
学生：...（卡住）
AI：那我们拆开看——
  ln(u) 的导数是 1/u，u=ax 的导数是 a，
  两个乘起来，就是整个复合函数的导数。
学生：所以 f'(x) = 2x - a/(ax+1)？
AI：✅ 很好。现在代入 x=0。
学生：f'(0) = -a/1 = -a
AI：✅ 对。斜率是 -a，点是 (0, f(0))，f(0) 等于多少？
学生：f(0) = 0
AI：✅ 所以切线方程是 y = -ax。
```
