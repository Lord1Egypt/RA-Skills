### Deployment Geography for Use: <br>

## Known Risks and Mitigations: <br>
Risk: The script writes fixed collage output filenames in the working directory and may overwrite existing files with the same names. <br>
Mitigation: Run it in a dedicated working directory or move existing collage_*.png files before execution. <br>
Risk: The script depends on Pillow, local fonts, and expected input images under collages/<theme>/A.png through I.png. <br>
Mitigation: Install Pillow and provide the expected local image assets before running the generator. <br>
