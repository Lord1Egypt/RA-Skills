## Description: <br>
Searches and navigates stored knowledge in memory palaces. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[athola](https://clawhub.ai/user/athola) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agents use this skill to retrieve concepts, cross-reference stored information, and search review knowledge across memory palaces using spatial, semantic, sensory, associative, and temporal cues. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad trigger language may cause the retrieval workflow to activate more often than expected. <br>
Mitigation: Install it only when memory-palace retrieval should participate in search or recall prompts, and use explicit prompts when invoking the workflow. <br>
