# 📦 龙虾教研日报助手 — 初始配置指南

> 首次使用本技能，必须完成以下配置。配置完成后，每天只需对话即可。

---

## 📋 配置清单

| 步骤 | 内容 | 完成标准 |
|---|---|---|
| 1 | 授权腾讯文档 | 完成 OAuth 授权 |
| 2 | 确认姓名 | 告诉小爪你的姓名 |

---

## 🔧 第一步：授权腾讯文档

### 为什么要授权？
小爪需要通过腾讯文档 API 读写你的日报数据，包括：
- 读取项目名称列表（用于自动匹配）
- 写入你的每日工作记录
- 读取历史记录（补录/汇总用）

### 如何授权

1. 在 WorkBuddy 里对小爪说：
   ```
   我要配置教研日报助手
   ```

2. 小爪会发送授权链接给你，点击链接完成授权：
   👉 **https://docs.qq.com/scenario/open-claw.html?nlc=1**

3. 授权完成后，回复小爪："授权完成"

### ⚠️ 注意事项
- 授权链接有效期约5分钟，超时需重新获取
- 请使用与腾讯文档一致的微信/QQ账号扫码
- 授权仅限读写你的工作数据，不会访问其他内容

---

## 👤 第二步：确认姓名

授权完成后，告诉小爪你的**真实姓名**（与腾讯文档人员表一致）：

```
小爪：授权已完成 ✅
     请告诉我你的姓名，以便匹配你的日报记录。
用户：我的姓名是张维
小爪：✅ 已确认！你好，张维！
     配置完成，你现在可以使用教研日报助手了。
```

### 姓名匹配说明
小爪会将你的姓名与"教研人员表"进行匹配，生成唯一的 record_id。
这个 ID 决定了你的日报记录会归属到你的名下。

---

## ✅ 配置验证

配置完成后，小爪会进行一次验证：

```
📋 配置验证
✅ 腾讯文档授权：已连接
✅ 姓名：张维（record_id: rpeiUU）
✅ 最近日报记录：2026-04-29

配置验证通过 🎉
你现在可以开始记录日报了。
```

---

## 📝 配置信息保存位置

配置信息保存在你的本地记忆文件中：
```
memory/user-config.md
```

下次使用时小爪会自动读取，无需重新配置。

---

## 🔄 重新配置

如果需要更换账号或修改姓名，对小爪说：

```
重新配置教研日报助手
```

小爪会引导你重新完成授权和姓名确认。

---

## ❓ 常见问题

**Q：授权失败了怎么办？**
A：请确认：1）扫码的账号与腾讯文档一致；2）链接未过期。重新执行授权步骤即可。

**Q：姓名填错了怎么办？**
A：告诉小爪"我的姓名是XXX"，小爪会帮你更新。

**Q：换了设备还需要重新配置吗？**
A：需要重新授权，但配置信息可以备份迁移。
