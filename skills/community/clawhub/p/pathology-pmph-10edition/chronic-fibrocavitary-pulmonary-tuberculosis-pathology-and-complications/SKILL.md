# 慢性纤维空洞型肺结核的病理特征与并发症识别

## 适用场景
- 继发性肺结核进展患者
- 长期未愈的空洞型肺结核病例

## 触发条件
当满足以下任一条件时启用本技能：
- 肺结核患者空洞持续存在超过数周且未愈合
- 影像学显示厚壁空洞（壁厚 ≥1cm）伴同侧或对侧多发新旧不一病灶
- 出现大咯血、气胸、声音嘶哑或右心功能不全等并发症征象

## 执行步骤
1. **确认空洞性病变特征**：检查是否存在一个或多个位于肺上叶的厚壁空洞（壁厚通常 ≥1cm），形状不规则。
2. **评估镜下三层结构**：
   - 内层：干酪样坏死物，含大量结核分枝杆菌；
   - 中层：结核性肉芽组织；
   - 外层：纤维结缔组织。
3. **判断支气管播散模式**：观察是否沿支气管向下播散，导致下肺出现更新鲜的病灶。
4. **识别肺结构改变**：评估是否存在广泛纤维化、肺体积缩小、胸膜增厚及胸壁粘连。
5. **筛查关键并发症**：
   - 空洞与支气管相通 → 开放性肺结核（高传染性）；
   - 侵蚀大血管 → 大咯血（警惕窒息风险）；
   - 突破胸膜 → 气胸或脓气胸；
   - 含菌痰液播散 → 喉结核或肠结核；
   - 肺动脉高压 → 肺源性心脏病。
6. **判断愈合状态**：若空洞内壁被支气管上皮覆盖且无菌，提示“开放性愈合”，虽形态存留但已无活动性感染。

## 输出要求
生成明确的风险警示，强调高传染性及多重严重并发症可能性，建议立即隔离并启动强化抗结核治疗方案。
