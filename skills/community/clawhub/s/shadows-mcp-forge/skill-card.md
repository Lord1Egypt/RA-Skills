## Description: <br>
Guides agents through creating MCP servers in Python with FastMCP or TypeScript with the MCP SDK for AI-agent integrations. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>

## Use Case: <br>
Developers and engineers use this skill to scaffold, configure, and document MCP servers that connect APIs, databases, or services to AI agents in Python or TypeScript. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated MCP server code or dependency install commands may introduce unsafe packages or incorrect behavior. <br>
Mitigation: Review generated code and package names before running installs; test generated servers in an isolated environment before publishing. <br>
Risk: MCP servers can expose credentials or sensitive service access if secrets are embedded or privileges are too broad. <br>
Mitigation: Keep secrets in environment variables and grant only the permissions each generated tool needs. <br>
