# AMPLE病史采集法则

## 何时使用
- 初次检查（ABCDE）已完成
- 病人生命体征相对稳定
- 需补充病史以指导后续检查、诊断或治疗决策
- 病人昏迷、意识模糊或无法提供完整病史

## 执行步骤
1. **A (Allergies / 过敏史)**：询问药物或食物过敏史，特别关注青霉素类抗生素、丙泊酚等麻醉药。
2. **M (Medications currently used / 当前用药)**：记录所有正在使用的药物，包括抗凝药（如华法林）、抗血小板药（如阿司匹林）、镇静药、酒精等，评估其对出血风险或意识状态的影响。
3. **P (Past illness / Pregnancy / 既往疾病与妊娠)**：了解高血压、糖尿病、肝硬化、肾功能不全、血液病等慢性病史；育龄女性需确认是否妊娠。
4. **L (Last meal / 上一餐)**：记录进食时间与内容，用于评估紧急气管插管时的误吸风险。
5. **E (Events / Environments related to the injury / 受伤事件与环境)**：详细询问受伤机制（如坠落高度、撞击方向、枪弹入口/出口、着力部位、体位等）。

## 信息来源
- 若病人无法自述，立即向现场目击者、护送人员或家属获取上述信息。
- 所有信息应结构化记录，用于指导后续体格检查、影像学安排及治疗方案。

> 注意：即使病人清醒能叙述，也不应省略关键项；仅可简化流程，不可跳过核心要素。
