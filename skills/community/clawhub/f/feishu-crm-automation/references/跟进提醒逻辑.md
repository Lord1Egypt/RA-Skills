# 跟进提醒逻辑

## 查询超期未跟进客户

```
用户问：有哪些客户超过3天没跟进？
```

### 实现步骤

1. 调用 feishu_bitable_list_records 获取所有客户记录
2. 遍历每条记录，找到"最近跟进日期"字段
3. 计算 (今天 - 最近跟进日期) > 3天
4. 返回超期客户列表

### 代码逻辑

```python
from datetime import datetime, timedelta

def get_overdue_customers(records, days=3):
    """返回超过N天未跟进的客户列表"""
    today = datetime.now()
    overdue = []
    
    for rec in records:
        fields = rec["fields"]
        followup_date = fields.get("最近跟进日期")
        
        if not followup_date:
            # 没有跟进记录，算超期
            overdue.append({
                "record_id": rec["id"],
                "客户名": fields.get("CRM客户管理"),
                "状态": "从未跟进",
                "超期天数": "未知"
            })
            continue
        
        # 飞书日期是毫秒时间戳
        last_date = datetime.fromtimestamp(followup_date / 1000)
        days_since = (today - last_date).days
        
        if days_since > days:
            overdue.append({
                "record_id": rec["id"],
                "客户名": fields.get("CRM客户管理"),
                "状态": fields.get("状态", {}),
                "超期天数": days_since,
                "最后跟进": last_date.strftime("%Y-%m-%d")
            })
    
    return overdue
```

## 生成提醒消息

```python
def format_overdue_message(overdue_list):
    """格式化超期客户为飞书消息"""
    if not overdue_list:
        return "🎉 太棒了！所有客户都有及时跟进"
    
    lines = [f"⚠️ 以下 {len(overdue_list)} 位客户超过3天未跟进：\n"]
    
    for i, c in enumerate(overdue_list, 1):
        name = c.get("客户名", "未知")
        days = c.get("超期天数", "?")
        status = c.get("状态", "")
        lines.append(f"{i}. **{name}**")
        lines.append(f"   超期 {days} 天 | 状态: {status}\n")
    
    lines.append("\n建议：尽快联系客户，保持跟进节奏 📞")
    return "\n".join(lines)
```
