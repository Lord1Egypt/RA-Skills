# Attribution

## Knowledge Representation Research

The provenance and confidence model in curated-brain is informed by:
- Buneman, P. et al. (2001). *Why and Where: A Characterization of Data Provenance.* ICDT.
- The W3C PROV specification for provenance interchange. https://www.w3.org/TR/prov-overview/

## Data Management Patterns

- Semantic versioning for schema migration inspired by Alembic and Django migrations.
- JSON-based catalog storage inspired by SQLite's JSON1 extension and jq workflows.

## Tooling

- Optional SQLite backend uses the Python standard library `sqlite3` module.

## No Training Data Claims

All code, schemas, and documentation in curated-brain were authored by humans. No proprietary model training data was used in its creation.
