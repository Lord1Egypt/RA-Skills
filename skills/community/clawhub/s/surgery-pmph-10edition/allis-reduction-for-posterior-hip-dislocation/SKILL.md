# Allis法复位髋关节后脱位Ⅰ型

## 适用条件
- 确诊为髋关节后脱位Ⅰ型（单纯脱位，无关节内骨折）
- 伤后时间 ≤ 48小时（最佳窗口为24小时内）
- 无严重合并伤
- 具备全身麻醉或椎管内麻醉条件

## 执行步骤
1. **麻醉准备**：确保患者处于充分肌肉松弛状态（必须在麻醉下操作）。
2. **体位固定**：
   - 患者仰卧于手术床；
   - 助手双手牢固按压双侧髂嵴以稳定骨盆。
3. **术者操作**：
   - 将患侧髋、膝关节均屈曲至90°；
   - 双手握住腘窝（或以前臂套住腘窝）施加持续、稳定、向上的纵向牵引；
   - 待肌肉松弛后，在维持牵引的同时轻微外旋股骨；
   - 感知“弹跳感”或“响声”提示复位成功。
4. **复位确认**：
   - 畸形消失，髋关节被动活动恢复；
   - 立即行X线复查确认股骨头位于髋臼内。
5. **后续处理**：复位成功后立即制动，并启动康复计划。

> **禁忌**：超过72小时未复位、Ⅱ～V型复杂骨折脱位、存在严重合并伤者禁用此法。
