# 乳腺癌病理分型体系

## 适用对象
- 乳腺癌组织标本
- 病理诊断报告

## 前提条件
- 已获得组织病理学检查结果
- 明确癌细胞是否突破导管或腺泡基底膜

## 使用限制
- 不适用于非上皮来源的乳腺恶性肿瘤（如肉瘤）

## 执行流程

1. **判断基底膜完整性**：
   - 若癌细胞未突破导管或腺泡基底膜 → 归为「非浸润性癌」
   - 若癌细胞已突破基底膜并向间质浸润 → 归为「浸润性癌」

2. **非浸润性癌细分**：
   - **导管原位癌（DCIS）**：癌细胞局限于扩张导管内，导管基膜完整
   - **小叶原位癌（LCIS）**：肿瘤细胞充满末梢导管和腺泡，呈实体排列，小叶结构尚存

3. **浸润性癌细分**：
   - **浸润性非特殊癌（NOS）**：包括浸润性导管癌、单纯癌、腺癌、无大量淋巴细胞浸润的髓样癌（约占70%）
   - **浸润性特殊癌**：包括乳头状癌、伴大量淋巴细胞浸润的髓样癌、小管癌（高分化腺癌）、腺样囊性癌、大汗腺样癌、鳞状细胞癌等
   - **其他罕见癌**

4. **特殊情况处理**：
   - 乳头湿疹样癌（佩吉特病）若伴发浸润性癌，则归入浸润性癌，而非非浸润性癌
   - 炎性乳腺癌属于T4期浸润性癌

## 输出要求
输出具体病理类型名称及所属大类（非浸润性/浸润性），用于临床预后评估和治疗方案制定。
