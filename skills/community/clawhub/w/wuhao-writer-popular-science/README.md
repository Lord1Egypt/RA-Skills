# 吴昊风格科普文 (Wuhao Writer - Popular Science)

> 作者：Ericwuhaowr | 版本：v1.6.0 | 许可证：MIT-0

## 简介

基于吴昊（多年上市公司总监经验、AI培训专家）写作风格的公众号科普文创作技能。帮你写出有温度、有干货、有个人风格的AI科普文章，彻底告别AI味。

## 核心能力

- **HKR选题质检**：从趣味性、信息量、共鸣感三个维度评估选题质量
- **6种结构模板**：保姆教程型、经验分享型、观点型、热点解读型、沉浸式实验型、方法论工具型
- **四层自检体系**：硬性规则检查 → 风格一致性 → 内容质量 → 活人感终审
- **口语化写作**：打AI味，让文章读起来像"有见识的普通人在认真聊一件事"
- **情绪节奏控制**：波浪式情绪曲线，让读者有记忆点

## 适用人群

- AI行业公众号写作者
- 技术博主、内容创作者
- 企业内训内容创作者
- 需要批量产出高质量科普文的团队

## 安装方式

### 通过 ClawHub 安装
```bash
clawhub install wuhao-writer-popular-science
```

### 在 WorkBuddy 中使用
在 WorkBuddy 技能市场搜索「吴昊风格科普文」即可安装使用。

## 使用示例

输入提示词示例：
```
"帮我把这段关于DeepSeek的技术资料改写成公众号风格"
"我想写一篇AI工具测评，帮我规划结构和内容"
"检查这篇草稿，有没有AI味太重的地方？"
```
