# 房客審核SOP

## 審核流程

```
收到詢問 → 初步篩選 → 約看屋 → 看屋觀察 → 申請審核 → 最終決定
```

## 初步篩選（訊息階段）

### 必問問題
1. **入住人數**（1人/家庭/分租？）
2. **入住時間**（何時可搬入）
3. **職業**（穩定度判斷）
4. **預算範圍**（超出預算不用浪費時間）
5. **特殊需求**（寵物、報稅、設籍）

### 篩選基本門檻
- 年收入 ≥ **年租金之12倍**
- 非經營非法行業
- 入住人數合理（非超載）

---

## 看屋觀察清單

| 觀察項目 | 良好 | 警訊 |
|----------|------|------|
| 對環境態度 | 主動脫鞋、珍惜物品 | 亂碰亂翻、不關門 |
| 提問內容 | 關心維護/修繕/鄰居 | 只問能不能報稅/能不能轉租 |
| 合作度 | 準時赴約、有禮貌 | 遲到、態度差 |
| 經濟能力 | 工作穩定、主動講收入 | 含糊其詞、不願提供證明 |

---

## 申請審核文件

### 必要文件
- [ ] 身分證正反面影本（核對正本）
- [ ] 在職證明或名片
- [ ] 薪資證明（薪轉帳戶/扣繳憑單/所得清單）

### 加分文件
- [ ] 前房東推薦函/聯絡方式
- [ ] 聯徵信用報告
- [ ] 保證人身分證+財力證明
- [ ] 工作年資證明

### 審核要點
| 項目 | 審查重點 |
|------|----------|
| 收入來源 | 穩定長期工作 vs 兼職不穩定 |
| 年資 | ＞1年穩定，＜3個月低穩定 |
| 信用 | 有無卡債/呆帳/拒往 |
| 承租歷史 | 前一個租期多久，為何搬離 |
| 保證人 | 有最好，無則要求更高收入 |

---

## 紅燈（直接拒租）

1. ❌ **無法提供身分證明** → 100%拒絕
2. ❌ **收入來源不明** → 無法證明有能力付租
3. ❌ **明顯有囤積/髒亂跡象** → 入住後會更嚴重
4. ❌ **說話反覆無常** → 未來溝通成本極高
5. ❌ **前房東負評** → 打聽前房東電話
6. ❌ **要求簽不平等條約** → 例如長租期低租金不給調
7. ❌ **轉租疑慮** → 不願告知同行入住者身分

---

## 黃燈（需再觀察）

1. ⚠️ **工作未滿3個月** → 要求更高押金或保證人
2. ⚠️ **正在離婚訴訟** → 入住時間/人數不確定
3. ⚠️ **外國人** → 要求護照+居留證+在職證明
4. ⚠️ **保證人不在台灣** → 聯絡不易

---

## 簽約前最終檢查

- [ ] 身分文件已核對
- [ ] 收入已確認
- [ ] 承租目的明確
- [ ] 入住人數合理
- [ ] 無特殊管制項目（如設籍、報稅）發生衝突
- [ ] 雙方對租金/押金/水電約定一致

---

## 房東自保小技巧

1. **Line對話留存**：所有的溝通都留下文字記錄
2. **房租入帳通知**：設定銀行通知，租金延遲馬上知道
3. **定期巡視**：前3個月每月巡視一次，之後每季一次
4. **鄰居打好關係**：鄰居是最好的監視器
5. **簽約時拍照**：房客+身分證合照（確保真實身分）
