## Description: <br>
Security auditing skill that helps review OpenClaw SKILL.md files for prompt injection, data exfiltration, obfuscation, privilege escalation, and related malicious patterns. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[hichana](https://clawhub.ai/user/hichana) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and security reviewers use this skill to audit OpenClaw skill files, receive severity-based findings, and get remediation guidance or an optional cleaned draft when requested. Its findings and rewrites are advisory and should be manually reviewed before a skill is installed or deployed. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Automated security verdicts can miss sophisticated attacks or flag benign content. <br>
Mitigation: Treat findings as advisory, manually review the analyzed skill, and seek professional security review for high-risk deployments. <br>
Risk: Cleaned rewrites may remove legitimate functionality or leave residual issues. <br>
Mitigation: Manually review any cleaned version before installing it and test it in a sandboxed environment. <br>
Risk: Skill text under analysis may contain dangerous examples or malicious instructions. <br>
Mitigation: Handle analyzed skill content as untrusted evidence and do not execute code from it during review. <br>
