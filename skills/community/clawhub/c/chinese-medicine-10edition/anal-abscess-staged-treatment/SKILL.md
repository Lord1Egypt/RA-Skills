# Anal Abscess Staged Treatment

## When to Use
- 肛门突发肿痛、红肿灼热（初期）
- 肿痛加剧、按之有波动感（成脓期）
- 已破溃流脓、疮口难敛（溃脓期）

## Execution Steps
1. **判断病程阶段**：
   - 初期：质硬无波动，属实证；
   - 成脓期：痛如鸡啄，有波动感；
   - 溃脓期：已破溃，多虚证或虚实夹杂。
2. **辨证选方**：
   - 火毒蕴结（初期）→ 清热解毒 → 仙方活命饮（禁用穿山甲，替换为皂角刺等）；
   - 热毒炽盛（中期）→ 透脓托毒 → 透脓散（调整穿山甲）；
   - 阴虚毒恋（晚期）→ 养阴清热 → 青蒿鳖甲汤合三妙丸；
   - 正虚邪伏 → 益气托毒 → 托里消毒散；
   - 湿痰凝结 → 燥湿化痰 → 二陈汤合百合固金汤。
3. **结合外科处理**：成脓期难溃者配合切开引流。

> **注意**：所有含穿山甲方剂须依法替换；虚证需扶正祛邪并重。
