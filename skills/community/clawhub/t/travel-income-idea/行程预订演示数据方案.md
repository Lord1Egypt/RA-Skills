# 旅行创收助手 - 行程预订演示数据方案

## 🎯 方案说明

**核心思路**：演示数据展示 + 真实平台跳转

### 实现方式
1. **前端展示**：使用 Mock 数据模拟航班/火车/酒店推荐
2. **预订按钮**：跳转到飞猪/携程/12306 的真实搜索页（带参数）
3. **用户体验**：看到推荐 → 点击预订 → 跳转到对应平台完成预订

### 优势
- ✅ 无需 API 权限
- ✅ 开发简单快速（1-2 小时完成）
- ✅ Demo 效果完整
- ✅ 真实可用（跳转后用户可继续操作）

## 🎨 前端组件设计

### BookingPanel.tsx 组件结构

```tsx
import React, { useState } from 'react';
import { mockFlights } from '../data/mockFlights';
import { mockTrains } from '../data/mockTrains';
import { mockHotels } from '../data/mockHotels';

interface BookingPanelProps {
  destination: string;
  departureCity: string;
  travelDate: string;
  budget: number;
}

export const BookingPanel: React.FC<BookingPanelProps> = ({
  destination,
  departureCity,
  travelDate,
  budget
}) => {
  const [activeTab, setActiveTab] = useState<'flight' | 'train' | 'hotel'>('flight');

  // 根据出发地和目的地获取 Mock 数据
  const flights = mockFlights[`${departureCity}-${destination}`] || [];
  const trains = mockTrains[`${departureCity}-${destination}`] || [];
  const hotels = mockHotels[destination] || [];

  return (
    <div className="booking-panel">
      {/* 标签页切换 */}
      <div className="tabs">
        <button 
          className={activeTab === 'flight' ? 'active' : ''}
          onClick={() => setActiveTab('flight')}
        >
          ✈️ 航班
        </button>
        <button 
          className={activeTab === 'train' ? 'active' : ''}
          onClick={() => setActiveTab('train')}
        >
          🚄 火车
        </button>
        <button 
          className={activeTab === 'hotel' ? 'active' : ''}
          onClick={() => setActiveTab('hotel')}
        >
          🏨 酒店
        </button>
      </div>

      {/* 航班推荐 */}
      {activeTab === 'flight' && (
        <FlightList 
          flights={flights} 
          onBook={(url) => window.open(url, '_blank')}
        />
      )}

      {/* 火车推荐 */}
      {activeTab === 'train' && (
        <TrainList 
          trains={trains}
          onBook={(url) => window.open(url, '_blank')}
        />
      )}

      {/* 酒店推荐 */}
      {activeTab === 'hotel' && (
        <HotelList 
          hotels={hotels}
          budget={budget}
          onBook={(url) => window.open(url, '_blank')}
        />
      )}
    </div>
  );
};
```

### 飞猪酒店搜索链接

```typescript
function generateFliggyHotelUrl(
  city: string,
  checkIn: string,
  checkOut: string,
  budget?: number
): string {
  const cityCodes: Record<string, string> = {
    '北京': 'BJS',
    '上海': 'SHA',
    '广州': 'CAN',
    '武汉': 'WHN',
    '成都': 'CTU',
    '杭州': 'HGH',
    // ... 更多城市
  };
  
  const cityCode = cityCodes[city] || city;
  let url = `https://www.fliggy.com/hotel/list.htm?city=${cityCode}&checkin=${checkIn}&checkout=${checkOut}`;
  
  if (budget) {
    url += `&maxPrice=${budget}`;
  }
  
  return url;
}
```
