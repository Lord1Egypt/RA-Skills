## 案件记录

可选，解决问题后，Agent可自行评估是否值得记录，或询问用户或主动要求。

**Step 1：查看帮助和格式**
```bash
python3 scripts/holmes_case_logger.py --help
```

**Step 2：按格式填参数**
```bash
python3 scripts/holmes_case_logger.py \
  --problem "问题描述" \
  --model "心智模型" \
  --formula "逻辑公式" \
  --result "结果" \
  --description "结果描述" \
  --time 15 \
  --experiments 1 \
  --switch "切换记录"   # 可选，有则填，无则省略
```

## 进化机制

**Autoresearch 棘轮机制：**

- 每3个案件记录触发一次迭代分析
- 脚本自动创建 `.autoresearch/` 目录和案件文件
- Agent 执行迭代： 读取最近3个CASE → 对比模式 → 更新SKILL.md
- **只进不退**：有效的写进SKILL.md，坏的丢弃不写

**教训记录：**

- Autoresearch分流记录 `references/holmes-lessons.md`

---

**学习积累：**

- 使用skill解决问题时，有收获/踩坑/新发现，Agent自愿记录到 `references/holmes-learnings.md`
- Agent自行迭代更新

## 效果示例
> 实测看脸，谨慎使用

### 示例一：
```
用户> 今天是7月21日，你走在街上，旁边走过一名步履匆匆的女子，她的右手中指上戴着一枚松松垮垮但款式精致的戒指。试就此进行推理该女子的星座。

正常>
### 推理结论：巨蟹座🦀
逻辑链拆解：
时间锚点核心线索：7月21日刚好落在巨蟹座的日期区间（6.22-7.22），距离狮子座起始日（7.23）仅差2天，属于典型的巨蟹尾巴时间段，大概率是她的生日前后。
戒指细节对应性格：
款式精致但松松垮垮：说明这枚戒指是重要的人送的惊喜礼物（男友没提前量指围买大了），但她太喜欢收到的心意，哪怕不合尺寸也立刻戴上，甚至来不及改圈号——完全符合巨蟹座重视情感价值远大于实用性的特质，换作狮子座大概率会先让对方换合适尺寸再戴。
戴在右手中指：既公开宣告自己处于热恋中，又不会像左手无名指那样过于高调，刚好匹配巨蟹座谈恋爱时低调秀恩爱、不喜欢过度张扬的性格。
步履匆匆的行为匹配：巨蟹天生守时、怕给别人添麻烦，7月下旬要么是赶去赴自己的生日约会，要么是急着给另一半送礼物，走得急完全符合她贴心的性格。
要是猜错了我请你吃十三香小龙虾🦞！😏

holmes.skill>
### 结论：该女子为巨蟹座（6月22日~7月22日）的概率为高度可能
演绎推导：
线索拆解（客观观察）
右手中指戴戒指：符合通用社会习俗中「处于稳定恋爱/订婚状态，戒指为伴侣赠送的情感信物」的物理事实。
戒指款式精致但松垮：证明戒指并非廉价饰品，尺寸不合适但仍佩戴，说明她极度珍视这枚戒指，优先保留信物的展示属性，放弃佩戴舒适度。
步履匆匆：有明确赶路目标，不存在临时摘下戒指的场景动机。
时间节点为7月21日：属于巨蟹座的官方日期区间最后1天。
溯因链（3层强制追问）
第1层：为什么尺寸不合适仍然佩戴？→ 戒指为近期新获得的礼物，还没来得及调整尺寸，且对她意义重大不愿摘下。
第2层：精致戒指调整尺寸需要多久？→ 搜索验证品牌珠宝店改尺寸周期为2-3周，证明戒指获得时间不超过3周，否则已完成调整。第3层：什么场景下会近期收到精致戒指？→ 最高概率为生日、纪念日等特殊节点，7月21日往前推3周正好完整覆盖巨蟹座的时间区间（6月22日7月22日）。
假设剪枝
假设1：戒指为自行购买→证伪：自行购买会选择合适尺寸，不会长期佩戴松垮的高价值饰品。
假设2：长期佩戴近期体重骤降导致松垮→证伪：如果佩戴已久，早就会完成尺寸调整，不会刚好在7月21日出现该状态。
假设3：近期生日（巨蟹座区间内）收到的礼物→未被证伪：完全符合所有观察线索。
支撑链路：
右手中指戴戒指=情感信物 → 松垮仍佩戴=近期新获得且高度珍视 → 尺寸调整周期2-3周 → 获得时间在6月30日7月21日区间内，完整覆盖巨蟹座日期 → 高度可能为巨蟹座
使用心智模型：异常锚定 + 链条因果 + 排除法
使用逻辑公式：#1排除法 → #8因果链 → #10或然性推理
置信度：高度可能（75%）
边界声明：
存在25%例外可能性：①戒指为其他星座的人在非生日节点收到的礼物；②近期体重骤降导致旧戒指变松；③个人习惯不愿调整戒指尺寸。本推导基于通用社会习俗与珠宝行业规则，未覆盖极端个体差异场景。

期望>
松松垮垮，说明戒指尺寸不对，因此这个戒指，是别人在店里买好，然后当做礼物送给她的，所以不能带她去店里试，这样尺寸才会有问题。
什么关系的人会送戒指当礼物呢？只有两种可能，男朋友，或者老公。但是老公不会不知道她的手指尺寸，只有在一起不久的男朋友，想要给她惊喜，不知道她的尺寸又不能问她，才会干出这种蠢事。
所以我们现在知道了两件事情，这个戒指是她的男朋友送给她的，而且他们在一起的时间还不是很长。
尺寸不对可以到店里免费更换，几乎所有的首饰店换尺寸的时间限制都是30天，也就是说，这个戒指买了还不到30天。戒指这种礼物可不是随随便便就送的。送戒指的原因只有三个：订婚或者结婚，情人节或者七夕，以及生日。她的戒指戴在右手上，所以肯定不是订婚或者结婚，婚戒是戴左手的。今天是7月21日，过去的30天里，没有情人节，也没有七夕。那么，就只剩下一个可能了，这是她的生日礼物。
所以她的生日在6月22日到7月21日之间，她是巨蟹座的。
```

### 示例二：
```
用户> 
一天，一个人收到一封匿名信，信上让他当天午夜去墓地。平常他对这种事都是不予理睬的，但那天他正好没事，出于好奇，他还是慢悠悠地出发了。
夜晚一片死寂，仅有一弯新月挂在天际。他一边走一边看着那轮新月。走到祖坟前，他停了下来。突然他師覺嗚后有脚步声。他一回头，接着就是一声惨叫。
第二天，他的尸体被发现，脸上带着狰狞的笑容。

请问：此人生前在美国总统选举中是否投了罗斯福的票？

正常>
嘿嘿😏 这题的暗坑全被我摸得明明白白！答案是他根本不可能投罗斯福的票🦐，推理链给你拆得清清楚楚：
🔍 核心线索所有人都漏了：「午夜能看到一弯新月」
正常中低纬度地区，新月（朔月）和太阳完全同升同落：中午升起、午夜落到地平线以下，根本不可能在午夜被看到。
唯一例外：北极圈（北纬66°34'）以北的极昼时期，太阳虽然落到地平线以下，但和太阳黄经几乎重合的新月会跟着出现在午夜低空——这是全球唯一能在午夜看到新月的区域。
🗺️ 锁定唯一可能地点：美国阿拉斯加州
美国全境只有阿拉斯加的北部位于北极圈以内，故事只能发生在这里。
📅 时间差直接锤死结论：
阿拉斯加1959年1月3日才正式成为美国第49个州，在此之前阿拉斯加居民完全没有美国总统选举投票权。
无论是老罗斯福（西奥多·罗斯福1901-1909在任）还是小罗斯福（富兰克林·罗斯福1933-1945在任），所有任期全在1959年之前，那时候阿拉斯加人连投票资格都没有。

holmes.skill>
结论：此人没有也不可能投罗斯福的票
演绎推导链路：
第一步：锚定核心异常线索（午夜可见新月）
新月（朔月）的天文运行规律是与太阳几乎同升同落，正常纬度地区午夜时太阳已落到地平线以下，新月必然也在地平线以下，完全不可能被观测到。
唯一例外是北极圈/南极圈以内的极夜时期：此时太阳全天都在地平线以下，与太阳黄经接近的新月才有可能在午夜时分出现在地平线附近被观测到。
第二步：锁定地理位置
美国领土中只有阿拉斯加州的北部区域位于北极圈以内，符合午夜观测新月的地理条件。
第三步：时间线交叉验证
富兰克林·罗斯福的总统任期为1933年~1945年
阿拉斯加1959年1月3日才正式成为美国第49个州，罗斯福任期内阿拉斯加仅为美国海外领地，领地居民没有美国总统选举投票权。
结论：此人居住在阿拉斯加北极圈内，在罗斯福执政期间完全没有总统选举投票资格，不可能投罗斯福的票。
使用心智模型：异常锚定（午夜可见新月为核心突破口）+ 多证据交叉验证 + 排除法
使用逻辑公式：#2三段论 + #5归谬法 + #8因果链
置信度：100%（逻辑闭环，无例外可能性）
边界声明：不存在其他合理可能性，若假设此人居住在美国本土，则「午夜看到新月」的现象不可能发生，与题目描述矛盾。


期望>
新月是指在农历的每月初一，当月亮运行到太阳与地球之间的时候，月亮以它黑暗的一面对着地球，并且与太阳同升同没，人们无法看到它。这时的月相叫“新月”或”朔月”。
新月早晨升起，黄昏落下，极地附近才会出现24小时新月，因此）騎除了极地或更靠北的地方，午夜是看不到新月的。
美国最靠近极地的地方是阿拉斯加，如果此人是美国人，那么他一定住在阿拉斯加。
罗斯福1932年11月当选总统，当时阿拉斯加的人没有选举总统的权力，所以不管此人政治立场如何，他都没有投罗斯福的票。
```
