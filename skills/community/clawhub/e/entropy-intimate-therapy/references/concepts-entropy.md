# Part 1: 熵的基础概念（给咨询师/学生）

> 本文档为 entropy-intimate-therapy skill 的基础层。读完后能向来访者用日常语言解释熵，并把它用作思考工具。
>
> 完整长版见 `~/entropy/ent3.md` 第一、二部分。本文档是为咨询场景压缩过的版本。

## 1. 一句话定义

**熵 = 我们对系统真实状态的"无知度"。**

无知度越高 → 熵越大。完全知道 → 熵 = 0。完全不知道 → 熵最大。

这一句话足以支撑后续所有应用。

## 2. 熵的三个面孔

| 出身 | 名字 | 一句话定义 | 直觉口诀 |
|------|------|----------|---------|
| 热力学 | Clausius 熵 | 能量"用不掉"的部分 | 烧掉的煤里有多少废热 |
| 统计力学 | Boltzmann 熵 | 一个表象对应多少种可能内里 | 桌面看着乱，有多少种乱法 |
| 信息论 | Shannon 熵 | 平均一次观察能带给你多少新信息 | 一句话听完让你"哦——"多少 |

**临床用法**：婚家咨询里几乎只用 Shannon 视角——它是关于信念、预测、不确定性的语言。

## 3. 公式（理解直觉就够，不需计算）

$$H(p) = -\sum_i p_i \log p_i$$

**三句口诀替代公式**：

1. **等可能性越多，熵越大**——抛硬币 1 比特，抛骰子 2.58 比特，100 面骰子 6.64 比特。
2. **分布越尖，熵越小**——90/10 偏置硬币 ≈ 0.47 比特，99.99% 偏置 ≈ 0.001 比特。
3. **熵可加可减**——独立系统熵相加；信息流通后总熵下降。

## 4. 四个生活场景（向来访者解释时用）

### 4.1 拆礼物
- 拆开前：高熵（一堆猜想）
- 拆开后：零熵（一个答案）
- **信息和熵是同一硬币两面**："获得 N 比特信息" = "熵下降 N 比特"

### 4.2 悬疑剧
- 第一集：高熵（10 个嫌疑人均等怀疑）
- 大结局：零熵
- 好看 = 控制好降熵速度（太快没意思，太慢观众放弃）

### 4.3 走进陌生人聚会
- 进门一刻：100% 高熵 → 大脑自动开始降熵
- 30 分钟后：30% 熵 → 你"舒服了"
- **人脑天生对低熵环境感到安全；对持续高熵感到焦虑**

### 4.4 老朋友 vs 陌生人吃饭
- 老朋友：低熵（猜得到对方反应）→ 放松
- 陌生人：高熵 → 高强度预测 → 累

**但**：完全低熵 = 无聊 = 关系窒息。这是亲密关系最深刻的矛盾。

## 5. 热力学第二定律的"亲民版"

严肃版：**孤立系统的熵不减少。**

亲民版：**你不主动收拾，房间不会自己变干净。**

更深一层：**任何有序结构，如果不持续输入能量，都会衰退。**

## 6. 关系第二定律（咨询用核心命题）

> **任何不主动维护的亲密关系，关系熵都会缓慢上升。**

这听起来反直觉——按理说越熟应该越懂啊？但实际上：

- 你以为你了解他，但他这一年变了，你没察觉 → 认知熵悄悄回升
- 你以为日程是默契的，但出现裂缝你没意识 → 行为熵悄悄回升
- 你以为"我们就是恋人啊"，但他对关系的理解已偏移 → 意义熵悄悄回升

**所有"关系突然出问题"的案例，背后几乎都是熵的"无声累积"。**

临床应用：当来访者说"我不知道我们怎么会变成这样"——几乎肯定是熵的慢性累积。诊断方向应是**找出哪个维度的熵在何时开始累积、为什么没被发现**。

## 7. 关系发展 = 互相降熵的过程

```
关系熵
  │
  │ ★ 初次见面
  │   ★
  │     ★ 几次接触
  │       ★
  │         ★ 自我表露
  │           ★
  │             ★---★---★ 稳定期（低熵平台）
  │                       ★ ← 突发事件
  │                         ★---★ 重新降熵 或
  │                                ★ 解体（熵骤增）
  └─────────────────────────────────► 时间
```

**临床读图法**：来访者来咨询时通常处在两种位置：

1. **稳定期平台后期 + 慢性熵累积** → 维持/修复议题
2. **突发事件后熵骤增** → 危机干预议题

两种位置的干预策略不同（见 `interventions.md`）。

## 8. 三种关系熵（重要！咨询常用）

把"关系熵"分成三个独立维度（详见 `three-entropies.md`）：

1. **认知熵**：我对你的了解（性格、偏好、伤痕、底线）有多模糊？
2. **行为熵**：我能多准确预测你接下来会做什么？
3. **意义熵**：我能多确定"这段关系是什么"（朋友？恋人？玩玩？终身伴侣？）

**这三个分量可独立增减**。诊断时要分别评估。

例：
- 刚谈恋爱：三种熵都高
- 谈了三年：认知熵和行为熵下降，但**意义熵**可能因对未来不确定而上升
- 老夫老妻：三种熵都低，但**新鲜感**几乎枯竭——这是另一种问题

## 9. 高熵 vs 低熵问题的辨识

| 维度 | 高熵问题 | 低熵问题 |
|------|---------|----------|
| 主观感觉 | 焦虑、混乱、失控、震惊 | 无聊、停滞、窒息、麻木 |
| 行为表现 | 反复试探、过度沟通、争吵 | 各自做各自的、回避深入 |
| 治疗目标 | 降熵（建立可预测性） | 升熵（引入新鲜、挑战、新角度） |
| 常见错误 | 试图用更多低熵规则压制 → 反弹 | 试图用更多承诺安全感 → 加重停滞 |

**临床警示**：不要默认所有问题都是"沟通不够"——有些关系问题恰恰是**沟通过多、规则过多、可预测性过强**导致的窒息。诊断要先分高低熵。

## 10. 给来访者的常用比喻

| 现象 | 用什么类比 |
|------|----------|
| 关系初期的兴奋 | "拆礼物的快感"——降熵速度快 |
| 长期默契 | "老朋友吃饭"——大脑省力 |
| 长期默契的副作用 | "餐厅菜单背熟了"——失去探索 |
| 突然爆发的冲突 | "积存的熵突然释放"——不是为了牙膏帽子 |
| 维护关系的努力 | "和宇宙对抗"——熵增是默认 |
| 修复后的关系 | "重新拆礼物"——共同经历的新发现 |
