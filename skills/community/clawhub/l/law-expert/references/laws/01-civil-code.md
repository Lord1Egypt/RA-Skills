# 民法典

## 核心法规
- 《中华人民共和国民法典》（2021年1月1日施行）
- 《最高人民法院关于适用〈中华人民共和国民法典〉合同编通则若干问题的解释》（2023年）
- 《最高人民法院关于适用〈中华人民共和国民法典〉物权编的解释（一）》（2021年）

## 关键制度
- **合同效力三层次**：有效、可撤销（欺诈/胁迫/重大误解）、无效（违反强制性规定/公序良俗），撤销权行使期限1年。
- **善意取得**：无权处分人以合理价格转让，受让人善意且完成登记或交付，可取得所有权。
- **离婚冷静期**：协议离婚须经过30日冷静期，期满后30日内双方应亲自到场申领离婚证。
- **自甘风险与好意同乘**：自愿参加文体活动受损，不得请求其他参加者赔偿（故意或重大过失除外）；无偿搭乘机动车发生事故，应减轻驾驶人赔偿责任。
- **惩罚性赔偿**：产品缺陷致人死亡或健康严重损害、故意侵害知识产权情节严重、环境污染损害，可主张惩罚性赔偿。

## 常见风险场景
1. 合同签订后被主张重大误解或显失公平要求撤销
2. 二手房买卖中出卖人违约不配合过户，买受人主张继续履行
3. 婚姻期间一方隐匿转移共同财产，离婚时另一方主张少分或不分
4. 小区物业服务纠纷（维修基金使用、共有部分收益归属）
5. 网络购物平台以格式条款免除自身责任引发消费者维权

## 法眼处理指引
- **推荐链路**：事实梳理 → 法律关系定性 → 请求权基础检索（SOP-01 民事法律分析）
- **关键提示**：民法典问题务必先确定法律关系的性质（合同/侵权/物权/婚姻），不同性质适用的规则和诉讼时效各有不同；特别注意民法典施行前后新旧法衔接。
