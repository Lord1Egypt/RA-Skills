# Resume Optimizer - 简历优化器

让你的简历通过 ATS 筛选，拿到面试机会。

## 功能

- 优化工作经历描述（STAR 法则）
- 匹配目标岗位 JD 关键词
- 生成专业求职信
- 提供面试准备建议

## 使用方法

### 优化简历
```
优化我的简历：
目标岗位：<粘贴JD>
我的经历：<粘贴现有简历>
```

### 改写单段经历
```
改写这段工作经历：<粘贴内容>
要求：突出数据成果，用STAR法则
```

### 生成求职信
```
写一封求职信：
公司：XXX
岗位：XXX
我的优势：XXX
```

## 优化原则

1. **量化成果** - 用数字说话（提升 30%，管理 5 人团队）
2. **关键词匹配** - 对齐 JD 中的技能要求
3. **STAR 法则** - 情境→任务→行动→结果
4. **简洁有力** - 每点不超过 2 行
