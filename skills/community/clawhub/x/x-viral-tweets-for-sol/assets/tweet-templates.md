# Tweet Templates

Drop-in scaffolds for fast generation. Replace bracketed slots. Never ship a template untouched — always pass it through the voice filter (lowercase, terse, no marketing language).

## Founder build update

```
shipped [feature]. [one line on what it unlocks].

[receipt: number, screenshot ref, or link]
```

## Founder vulnerability

```
[honest emotion] last [time period]. [one-line cause].

[what you're doing about it]
```

## Numeric stop-scroll

```
[number] [units] [verb] [object] in [timeframe]. here's what i learned.

[2–3 bullet lessons or single line payoff]
```

## Contrarian declaration

```
[popular belief] is [wrong / overrated / cope].

[1-line reason]
```

## Insider framing

```
while everyone's watching [popular thing], the real story is [overlooked thing].

[1-line on why]
```

## Cope-bait correction

```
you don't need [common belief] to [outcome].

you need [actual lever].
```

## Receipt-led

```
[outcome with number]. here's the [exact playbook / what we did wrong / what nobody tells you].

1. [step]
2. [step]
3. [step]
```

## Question that filters

```
[question targeting your tribe]. no [thing you want to exclude].
```

## Reply bait — list opener

```
underrated [category] on solana:

mine first: [yours]

go.
```

## Quote-tweet add-a-layer

```
[short take that extends OP's point without repeating it]
```

## One-line zinger

```
[single declarative sentence, ≤ 12 words]
```

## Founder hot take

```
unpopular: [thing most people in the space agree with reversed]

[1 line of defense]
```

## Builder flex

```
[number of [unit]] later. still [doing the thing].

[receipt]
```

## Ecosystem hot take

```
the difference between [project a] and [project b] isn't [obvious thing]. it's [non-obvious thing].
```

## Meme caption tweet (no image)

```
[setup line]
[punchline]
```

(two lines, hard return, second line carries weight)

## Narrative forecast

```
[narrative] is the next [outcome]. mostly because [non-obvious reason].

[bonus: who's already positioning]
```
