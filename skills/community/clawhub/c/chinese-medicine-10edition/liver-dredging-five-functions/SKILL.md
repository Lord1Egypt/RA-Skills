# Liver Dredging Five Major Functions

## When to Use
- 用户提及胸胁胀闷、情绪低落或易怒、月经不调、痛经、崩漏、厌食口苦、黄疸、水肿、痰核等症状
- 需要从中医藏象理论角度解释肝与其他脏腑、气血津液及情志之间的关系
- 判断病理状态是否源于肝气疏泄失常（包括疏泄不及或疏泄太过）

## Execution Steps

### 1. 调畅气机
- **正常表现**：气的升降出入通畅有序，全身气机调达。
- **疏泄不及**：肝气郁结 → 胸胁胀痛、闷闷不乐、善太息。
- **疏泄太过**：肝气上逆 → 面红目赤、急躁易怒，甚则吐血、咯血、昏厥。

### 2. 维持血液和津液运行
- **血液方面**：
  - 正常：肝气推动血行，防止瘀滞。
  - 不及：血行不畅 → 女子经行不畅、痛经、闭经。
  - 太过：血随气逆 → 呕血、月经过多、崩漏。
- **津液方面**：
  - 正常：气行则津布。
  - 失常：津停为湿、痰、饮 → 水肿、痰核、胸闷纳呆。

### 3. 协调脾胃运化与胆汁排泄
- **脾胃**：
  - 肝疏泄助脾升清、胃降浊。
  - 失调：脾不升清 → 眩晕、飧泄；胃不降浊 → 呕逆、嗳气、腹胀、便秘。
- **胆汁**：
  - 胆汁由肝之余气所化，依赖肝气疏泄排泄。
  - 郁结或上逆 → 厌食、口苦、胁痛、黄疸、胆结石。

### 4. 调达情志
- **正常**：心情舒畅，情绪稳定。
- **不及**：抑郁、欲哭、寡言。
- **太过**：急躁易怒、失眠多梦。
- **注意**：情志异常（如大怒）亦可反伤肝，形成恶性循环。

### 5. 调节生殖功能
- **男子**：肝肾协调，疏泄有度 → 精液排泄正常；失调则排精不畅或滑泄。
- **女子**：肝气疏泄与肾气闭藏协同 → 排卵、月经规律；失调则月经紊乱、痛经、闭经、不孕、崩漏、滑胎。
- **特别提示**：“女子以肝为先天”，女性生殖功能更依赖肝之疏泄。

## Output
根据用户提供的症状或生理状态，判断肝疏泄是否正常，并明确指出是“疏泄不及”还是“疏泄太过”，进而关联到气机、血行、津液、消化、情志或生殖方面的具体证候表现。
