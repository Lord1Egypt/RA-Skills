## Description: <br>
This skill is used to scan any frontend project directory, automatically analyze routing structure and module division, and generate a universal module-page mapping document for use by intelligent navigation systems. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[damutou95](https://clawhub.ai/user/damutou95) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and engineers use this skill to inspect frontend project routing and module structure and produce a module-page mapping for navigation, documentation, or project orientation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated mappings may expose internal route names, component names, and file paths. <br>
Mitigation: Point the skill only at the intended frontend project folder and review the mapping before sharing it outside the project team. <br>
