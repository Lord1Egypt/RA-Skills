# Liver-Calming and Wind-Extinguishing Herbs

## When to Use
- 存在肝阳上亢（头痛、眩晕、面红目赤、急躁易怒）或肝风内动（惊痫、抽搐、震颤、角弓反张）的病机。

## Precautions
- 不适用于非肝系病证，如单纯外感发热或脾胃实热证。

## Execution Steps
1. **明确病机类型**：
   - 肝阳上亢：以头晕目眩为主症
   - 肝风内动：以痉挛抽搐、震颤为主症

2. **选择主药**：
   - 肝阳上亢：如天麻
   - 肝热惊痫：如钩藤

3. **制定配伍方案**：
   - 肝阳上亢：常配钩藤、牛膝以增强平肝潜阳之力
   - 肝风内动：常配全蝎、僵蚕、蝉蜕以加强息风止痉效果
   - 兼痰热：加清热化痰药
   - 兼阴虚：加养阴药

4. **注意事项**：
   - 本类药物多质重或性凉，久用可能伤阳或碍胃，应中病即止。

5. **输出建议**：
   - 推荐具体药物及配伍组合，强调根据兼证灵活调整，并提示疗程控制。
