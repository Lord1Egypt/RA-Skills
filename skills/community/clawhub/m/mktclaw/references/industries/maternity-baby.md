## 5. 合规与监管

### 5.1 广告法专属限制

| 限制类型 | 禁用 |
|---------|------|
| 极限词 | 全行业通用 |
| 功效宣称 | 婴幼儿化妆品需特殊资质 |
| 比较广告 | 不得贬低母乳（奶粉品类） |
| 医疗暗示 | 不得宣称"治疗""增强免疫力" |

### 5.2 行业特殊资质

| 业务类型 | 必备资质 |
|---------|---------|
| 婴幼儿配方奶粉 | 配方注册（国家市场监管总局） |
| 婴幼儿化妆品 | 儿童化妆品标志"小金盾" |
| 玩具 | 3C 认证（电动/毛绒） |
| 纸尿裤 | 卫生用品生产许可证 |

### 5.3 平台审核红线

| 平台 | 红线 |
|------|------|
| 抖音 | 利用儿童做营销、低俗擦边 |
| 小红书 | 虚假"宝妈自用"、夸大功效 |
| 全平台 | 0-6 月婴儿配方奶粉广告（禁止） |

### 5.4 特殊合规要求

| 要求 | 说明 |
|------|------|
| 奶粉广告 | 不得暗示替代母乳、不得使用婴儿形象 |
| 玩具广告 | 不得误导儿童、不得使用"最新""最炫"等极限词 |
| 早教广告 | 不得宣称"赢在起跑线""神童" |
