# 申论备考策略

## 考试概述

- **时间**：180分钟（3小时），国考3小时，省考2.5-3小时
- **题量**：国考5题（副省/市地），省考3-5题
- **材料字数**：6000-8000字
- **答案字数**：2000-2500字

## 五大题型

### 1. 归纳概括题（15-20分）
- **核心要求**：全面、准确、简洁、有条理
- **作答步骤**：
  1. 审题：明确概括对象（问题/原因/影响/对策/做法）
  2. 找点：在材料中圈画关键词句
  3. 加工：合并同类项，去粗取精
  4. 书写：总括句+分条列点
- **技巧**：
  - 尽量用材料原词，不自我发挥
  - 使用序号"一、二、三"或"1.2.3."（看题目要求）
  - 每条要点字数均衡

### 2. 综合分析题（15-20分）
- **常见类型**：解释分析、评论分析、启示分析、关系分析
- **解释分析思路**：
  1. 解释含义（是什么）
  2. 分析原因/影响（为什么）
  3. 提出对策（怎么办）
- **技巧**：逻辑层次要全，但不必每条都展开

### 3. 提出对策题（20-25分）
- **核心要求**：针对性、可行性、可操作性
- **对策来源**：
  - 直接对策：材料中已经给出的
  - 间接对策：从问题/原因反推
  - 经验对策：从成功案例借鉴
- **对策表述公式**：
  ```
  主体 + 手段 + 内容 + 目的
  例：政府部门应通过加大财政投入，完善农村基础设施，以提升农民生活质量
  ```

### 4. 贯彻执行题（应用文，25-30分）
- **常见文种**：建议书、倡议书、讲话稿、简报、调研报告、工作方案、公开信
- **格式要素**：
  - 标题（必备）
  - 称谓（讲话稿、信函类必备）
  - 正文（开头+主体+结尾）
  - 落款（署名+日期，看情况）
- **技巧**：结构完整 > 语言优美，先把架子搭好

### 5. 文章写作（大作文，35-40分）
- **核心要求**：立意准确、结构清晰、论证充分、语言规范
- **写作五步法**：
  1. **审题立意**：吃透题目，确定中心论点（15分钟）
  2. **拟定框架**：列出分论点和论证思路（10分钟）
  3. **撰写开头**：背景引入→点明问题→亮出论点（150字）
  4. **展开论证**：每段观点句+说理论证+事例佐证+小结（3-4个分论点）
  5. **收尾升华**：总结观点→回扣主题→展望升华（150字）

## 申论高分要素

### 一、政府思维
- 站在政府角度思考问题，用"管理者"而非"评论者"视角
- 解决问题导向，不只指出问题更给出方案
- 体现"以人民为中心"的发展思想

### 二、材料为王
- **70%的答案来自材料**，不可脱离材料自我发挥
- 归纳概括、综合分析、对策题尤其要忠于材料
- 大作文可用材料案例但要转化为自己的语言

### 三、常用规范词库
| 口语化表述 | 规范表述 |
|------------|----------|
| 老百姓没钱 | 群众收入水平偏低 |
| 路不好走 | 基础设施建设滞后 |
| 没人管 | 监管缺位/管理真空 |
| 大家都不知道 | 政策宣传不到位/信息不对称 |
| 有人管但管不好 | 多头管理/协调机制不健全 |
| 办个事跑好几趟 | 办事流程繁琐/行政效能不高 |
| 城乡差距大 | 城乡发展不平衡 |
| 只顾眼前 | 缺乏长远规划/短视行为 |

### 四、大作文常用结构
```
以"全面推进乡村振兴"为例：

开头（150字）：
民族要复兴，乡村必振兴。当前，我国城乡发展不平衡依然突出...
全面推进乡村振兴是解决新时代社会主要矛盾的必然要求。

分论点一（250字）：产业兴旺是乡村振兴的基础
（观点句 + 说理 + 案例 + 小结）

分论点二（250字）：人才支撑是乡村振兴的关键
（同上结构）

分论点三（250字）：生态宜居是乡村振兴的底色
（同上结构）

结尾（150字）：
乡村振兴非一日之功。唯有以产业为基、人才为本、生态为要，
方能绘就"农业强、农村美、农民富"的壮丽画卷。
```

## 备考节奏

| 阶段 | 时间 | 内容 |
|------|------|------|
| 基础 | 2-4周 | 学习五大题型方法论，精读10篇范文 |
| 强化 | 4-6周 | 分题型专项练习，每2天一篇大作文 |
| 冲刺 | 2-3周 | 全真模拟（严格计时），查漏补缺 |
