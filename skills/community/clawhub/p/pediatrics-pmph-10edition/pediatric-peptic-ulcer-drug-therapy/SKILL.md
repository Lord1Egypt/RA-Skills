# 儿童消化性溃疡药物治疗

## 适用场景
- 确诊为消化性溃疡
- 已明确是否存在幽门螺杆菌（Hp）感染
- 已评估出血或并发症情况

## 执行步骤
1. **抑制胃酸治疗**：
   - **H₂受体拮抗剂**（如雷尼替丁3～5mg/kg/天）或**质子泵抑制剂**（如奥美拉唑0.6～0.8mg/kg/天）。
2. **胃黏膜保护剂**：
   - **硫糖铝**：按体重或年龄给药，一日4～6次；
   - **铋剂**：仅适用于≥6岁儿童，6～8mg/kg/天（按铋元素计）。
3. **抗幽门螺杆菌治疗**（Hp阳性者）：
   - **首选方案**：PPI + 克拉霉素 + 阿莫西林（克拉霉素敏感且无青霉素过敏）；
   - **青霉素过敏**：阿莫西林替换为甲硝唑；
   - **高耐药地区或未知药敏**：≥6岁采用含铋四联疗法（PPI + 阿莫西林 + 甲硝唑 + 铋剂）。
4. **疗程与监测**：抗Hp疗程14天；所有药物需监测疗效与副作用。

## 输出结果
根据患儿年龄、体重、Hp感染状态及过敏史，制定个体化药物治疗方案，并监测疗效与副作用。
