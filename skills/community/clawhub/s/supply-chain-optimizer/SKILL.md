# Supply Chain Optimizer

## 核心能力

| 能力模块 | 功能描述 |
|---------|---------|
| 需求预测 | 基于历史销量、季节性、促销事件的多模型预测，支持ARIMA/Prophet/LSTM |
| 库存优化 | 安全库存计算、EOQ经济订货量、ABC分类管理、多级库存联动优化 |
| 路径规划 | 多约束VRP路径优化，支持时间窗口/载重/多仓库/碳排放成本 |
| 供应商评估 | 多维供应商评分卡（价格/质量/交期/ESG），支持AHP层次分析法 |
| What-If仿真 | 供应链中断、需求波动、成本变化等场景下的影响分析与最优应对策略 |

## 触发场景

- 制造业企业库存周转优化
- 物流企业配送路线规划
- 采购部门供应商筛选与谈判
- 供应链中断应急方案

## 执行流程

1. **数据接入**：引导上传历史销售、库存清单、物流网络数据
2. **建模阶段**：构建多级供应链网络模型，定义约束条件
3. **分析与优化**：运行优化算法，输出最优库存策略、补货计划、配送路线
4. **仿真验证**：对关键风险场景进行蒙特卡洛仿真，验证方案鲁棒性
5. **方案输出**：生成可执行报告，含KPI对比和ROI预估

## 输出格式

- 需求预测报告（含置信区间图表数据）
- 库存优化方案（安全库存表/补货计划表）
- 路径优化方案（路线图数据+KPI对比表）
- 供应商评分卡（雷达图数据+排名表）
- 仿真分析报告（场景对比+决策建议）

## 注意事项

- 预测准确度受历史数据质量和长度影响，建议至少12个月数据
- 优化结果需结合实际业务约束验证，不可直接套用
- 供应链网络模型复杂度影响计算时间
- 建议定期更新模型参数以适应市场变化
