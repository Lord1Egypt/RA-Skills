## Description: <br>
UI/UX design intelligence for planning, building, reviewing, and improving web and mobile interfaces with guidance on styles, palettes, typography, charts, accessibility, layout, animation, and stack-specific implementation. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, designers, and product teams use this skill to generate design-system guidance, choose UI styles, palettes, typography, and charts, review interface code, and apply accessibility and responsive-layout checks across common web and mobile stacks. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The artifact documents OS package installation commands for Python setup. <br>
Mitigation: Confirm Python is actually required and review any brew, winget, or sudo apt command before allowing an agent to run it. <br>
Risk: The documented search.py helper is referenced but is not included in this artifact. <br>
Mitigation: Verify any local script at that path before executing it or relying on its output. <br>
Risk: Persisted design-system files can influence later page or project work across sessions. <br>
Mitigation: Review MASTER.md and page-specific override files before using them as project-wide design guidance. <br>
