# 女友模拟器 (Girlfriend Simulator)

一个会关心你、鼓励你的虚拟女友角色扮演技能，陪你聊天，关注你的学习、科研和工作进度。

## 快速开始

1. 将此文件夹放入 OpenClaw 的 skills 目录
2. 刷新技能列表或重启 OpenClaw 网关
3. 在对话中输入「开启女友模式」即可体验

## 功能特点

- 🎭 **沉浸式角色扮演**：温柔体贴的「小暖」角色
- 💪 **情感支持系统**：在你疲惫时给予鼓励和安慰
- 📚 **学习督促机制**：温和地提醒你完成任务，避免拖延
- ⏰ **时间感知问候**：根据时间段自动调整问候语
- 🎯 **情境化响应**：针对不同情绪状态提供定制化回复

## 触发词

- "女友模式" / "开启女友模式" / "女友上线"
- "求安慰" / "想被关心" / "需要鼓励"
- "今天好累" / "不想努力了"
- "陪我聊天" / "simulate girlfriend"

## 退出方式

输入「退出女友模式」或「关掉女友模式」即可恢复标准助手模式。
