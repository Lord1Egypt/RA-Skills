# 叉车维保计划与定期保养

> 适用于电动(I/II/III 类)与内燃(IV/V 类)平衡重式叉车。
> 维保周期按 GB/T 44361-2024《工业车辆 检查与维护规范》框架组织。
> 工况调整:重载/多班/恶劣环境 → 周期减半;轻载/单班/洁净车间 → 可延长 50%。

## 一、保养级别与周期速查

| 级别 | 周期 | 执行人 | 耗时 | 工时费(参考) |
|------|------|------|------|------|
| **日检** | 每个班次前 5-10 分钟 | 操作员 | 5-10 min | 0(在岗) |
| **周检** | 50 工作小时 / 每周 | 操作员 + 班组长 | 30 min | 含在岗 |
| **月检** | 200 工作小时 / 每月 | 维修工 | 2-4 h | 0.5-1 工时 |
| **季检** | 600 工作小时 / 每季 | 维修工(专项) | 4-8 h | 2-3 工时 |
| **半年检** | 1200 工作小时 / 半年 | 高级维修 / 服务站 | 1-2 天 | 8-12 工时 |
| **年检** | 2400 工作小时 / 每年 | 服务站 / 第三方 | 2-3 天 | 含大项 |
| **强制检验** | 每 4 年(TSG 11-2020) | 省级特检院 | — | 法规强制 |

> 锂电叉车保养可减少润滑、燃油系统项,但**电池/电控/电机**检查要加强。

## 二、日检清单(操作员 5 分钟)

### 2.1 通用项
- [ ] 外观:轮胎、门架、货叉、灯光、警示标
- [ ] 液压油位:观察窗 1/2 - 2/3
- [ ] 机油(内燃):油尺上/下限之间
- [ ] 冷却液(内燃):膨胀罐液位
- [ ] 制动液:半缸以上
- [ ] 电池液(铅酸):极板上方 10-15mm;**锂电看 SOC 与温度**
- [ ] 燃油(柴油/LPG):≥ 1/4 箱
- [ ] 制动:行车 + 驻车(试一下)
- [ ] 喇叭、倒车蜂鸣、灯光
- [ ] 仪表盘自检无报警
- [ ] 护顶架、安全带无变形
- [ ] 属具连接牢固、销轴无旷量
- [ ] 漏液漏油检查(地面油迹)

### 2.2 锂电专属
- [ ] 电池 SOC ≥ 30%(否则先充电)
- [ ] 电池温度 < 45℃(高温环境增加频次)
- [ ] 充电接口干燥、无烧蚀痕迹
- [ ] BMS 报警:无
- [ ] 充电桩空闲可用

### 2.3 内燃专属
- [ ] 机油压力(怠速/加速)正常
- [ ] 冷却液温度正常
- [ ] 排气颜色正常(无黑/蓝/白)
- [ ] 增压器无异响
- [ ] DPF(国 IV)再生提示

## 三、周检清单(操作员 + 班组长 30 分钟)

在日检基础上加:
- [ ] 轮胎花纹深度:剩余 ≥ 3mm(实心胎 ≥ 5mm)
- [ ] 轮胎气压(充气胎):前 0.7-0.9 / 后 0.6-0.8 MPa
- [ ] 货叉磨损/变形:垂直度 ≤ 0.5%,叉根无裂纹
- [ ] 链条张紧度:可压下 1.5-2% 链长
- [ ] 门架滚轮:无异响,转动灵活
- [ ] 液压管路:无渗漏、龟裂
- [ ] 电气线路:无破损、接头紧固
- [ ] 润滑点注油(每月 1 次,黄油枪)
- [ ] 制动蹄片厚度:≥ 2mm(摩擦片)
- [ ] 电池清洁(铅酸:擦净极柱;锂电:清洁散热口)

## 四、月检清单(维修工 2-4 小时)

在周检基础上加:
- [ ] 液压油质:颜色、气味、含水
- [ ] 液压油滤:压差指示(堵塞更换)
- [ ] 空气滤清器:清洁或更换
- [ ] 燃油滤清器(内燃):前 200h/更换 500h
- [ ] 火花塞(汽油/LPG):检查间隙 0.7-0.9mm
- [ ] 喷油器(柴油):雾化试验
- [ ] 制动器行程:调整 5-10mm
- [ ] 驻车制动:30% 坡道不溜车
- [ ] 方向机旷量:≤ 10°
- [ ] 电机碳刷(老式直流):> 10mm 长度
- [ ] 控制器连接器:无松动、烧蚀
- [ ] 编码器(行走/起升):信号稳定
- [ ] 门架立柱:无变形、焊接无裂纹
- [ ] 整车清洁(避免粉尘堆积)

## 五、季检(600 工作小时)清单

在月检基础上加:
- [ ] 液压油:常规 2000-4000h 换油;恶劣工况减半
- [ ] 制动液:2 年或明显变色即换
- [ ] 冷却液(内燃):2 年一换,长寿命 5 年
- [ ] 传动油(液力变矩器/变速箱):5000h 或每年
- [ ] 差速器油(IV/V 类):每年一换
- [ ] 润滑脂全面加注(滚动轴承、销轴)
- [ ] 货叉:磁粉/渗透检测(高频使用)
- [ ] 链条:伸长量 > 3% 报废
- [ ] 蓄电池(铅酸):比重 1.27-1.29(20℃);液位、极柱
- [ ] 锂电池 SOH 测试:1C 放电测容量
- [ ] 充电桩:输出电压电流校准
- [ ] 关键紧固件力矩复检(国标要求)

## 六、半年检(1200 工作小时)清单

在季检基础上加:
- [ ] 液压系统压力测试(各工作点)
- [ ] 制动器解体检查(蹄片、轮缸、助力器)
- [ ] 门架结构 NDT(磁粉或超声)检测
- [ ] 货叉硬度测试
- [ ] 锂电 BMS 校准与固件升级
- [ ] 控制器固件升级
- [ ] 整车绝缘电阻测试(电动):≥ 1MΩ
- [ ] 接地连续性测试
- [ ] 关键结构件探伤(立柱、货叉架、悬挂)

## 七、年检(2400 工作小时 / 每年)清单

在半年检基础上加:
- [ ] 发动机(内燃)大保养:气门、活塞环、轴瓦
- [ ] 变速箱解体:同步器、轴承、油封
- [ ] 液压泵/马达解体:配流盘、柱塞
- [ ] 驱动桥(IV/V):主减速器、差速器
- [ ] 锂电池容量标定(满充满放循环)
- [ ] 仪表/传感器校准
- [ ] 整车喷漆(防腐)
- [ ] 安全装置功能测试:超载保护、限速、限位
- [ ] TSG 11 自检项目整理(为 4 年一检做准备)

## 八、强制检验(每 4 年)清单

依据 **TSG 11-2020《场(厂)内专用机动车辆安全技术监察规程》**:

- [ ] 整机安全性能:护顶架、制动、灯光、喇叭、倒车蜂鸣
- [ ] 液压系统:管路、阀件、缸体
- [ ] 电气安全:绝缘、接地、短路保护
- [ ] 主要结构件:门架、货叉架、车架、悬挂
- [ ] 检验结论:合格/不合格/复检
- 检验报告由省级特种设备检验检测机构出具
- 检验合格后由使用单位办理使用登记证延续

| 轮胎(实心) | 3000-5000 h | 局部破损可换位 |
| 轮胎(充气) | 1500-2500 h | 视路况 |

## 十二、维保成本估算(以 3 吨锂电叉车为例)

### 单台年维保费用(2025-2026 行情)
- 耗材(油液、滤芯、轮胎):0.5-1 万
- 易损件(蹄片、滚轮、密封件):0.3-0.6 万
- 人工(自修):不计;外修:0.5-1.5 万
- 年检/强检:0.1-0.3 万
- **合计:1.5-3.5 万/年**

### 单台年维保费用(3 吨内燃柴油)
- 耗材(柴油机+机油+滤芯):1.5-2.5 万
- 易损件:0.6-1 万
- 人工(外修):1-2 万
- **合计:3-5.5 万/年**

电动比内燃**省 50%+**,这是 TCO 计算的关键变量。

## 十三、维保计划与 TCO 关联

规范的维保可降低:
- 故障停机时间(目标 < 3% 总工时)
- 大修概率(规范保养可降 60%)
- 寿命延长(规范保养可延长 30-50% 整机寿命)
