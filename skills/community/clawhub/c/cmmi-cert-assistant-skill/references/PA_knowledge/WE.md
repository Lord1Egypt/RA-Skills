# WE — 工作环境因素（CMMI V3.0 Domain-specific PA）

## PA 基本信息
- **缩写**：WE
- **英文**：Work Environment Factors
- **中文**：工作环境因素
- **所属**：Domain-specific PA（领域特定实践域）
- **适用领域**：人力资源管理、组织管理
- **新增说明**：V3.0新增实践域，关注工作环境对人员绩效的影响

---

## CMMI V3.0 要求要点

### 核心目的
识别、管理、改进影响人员绩效的工作环境因素。

### 关键实践（Specific Practices）
| 实践编号 | 实践内容 | 说明 |
|---------|---------|------|
| WE 1.1 | 识别工作环境因素 | 识别影响绩效的物理、文化、工具环境因素 |
| WE 1.2 | 评估工作环境因素 | 评估因素对绩效的影响程度 |
| WE 1.3 | 制定改进策略 | 制定环境因素改进计划 |
| WE 2.1 | 实施改进措施 | 执行工作环境改进 |
| WE 2.2 | 监控工作环境因素 | 持续监控环境因素变化 |
| WE 3.1 | 评估改进效果 | 测量改进措施的有效性 |

### 通用实践（Generic Practices）
- GP 1.1 执行特定实践
- GP 2.1 建立组织政策
- GP 2.2 策划过程
- GP 2.3 提供资源
- GP 2.4 分配责任
- GP 2.5 培训人员
- GP 2.6 管理配置
- GP 2.7 识别并记录干系人需求和接口
- GP 2.8 监督和控制过程
- GP 2.9 客观评估符合性
- GP 2.10 高层审查

## 用户文档 → PA 匹配规则

### 高概率匹配（主要PA）
- 文档名含"工作环境"、"Work Environment"、"办公室环境" → **WE 主要匹配**
- 文档含环境因素清单、环境影响评估 → **WE 主要匹配**

### 中概率匹配（次要PA）
- 文档含培训环境 → 同时匹配 **OT**（组织培训）
- 文档含工具环境 → 同时匹配 **TS**（技术服务）

## 差距检查清单（用于差距分析）

### WE 1.x — 环境因素识别与评估
- [ ] 是否识别了影响绩效的工作环境因素？
- [ ] 是否有环境因素分类（物理/文化/工具）？
- [ ] 是否评估了各因素对绩效的影响程度？
- [ ] 是否有改进优先级排序？

### WE 2.x — 环境因素管理
- [ ] 是否有工作环境改进计划？
- [ ] 是否实施了改进措施？
- [ ] 是否有环境因素持续监控机制？

### WE 3.x — 改进效果评估
- [ ] 是否有改进效果测量？
- [ ] 是否有后续优化计划？

## 评估访谈常见问题（模式D）

### 针对PM（项目经理）的问题
1. "你的团队工作环境如何？有哪些因素影响他们的工作效率？"
2. "你如何识别和管理工作环境因素？"
3. "你实施过哪些工作环境改进措施？"
4. "如何评估工作环境改进的效果？"

### 针对人员的问题
1. "你的工作环境有哪些需要改进的地方？"
2. "工作环境因素如何影响你的工作效率？"
3. "你参与过工作环境改进活动吗？"

### 评分要点
- **WE 1.1-1.3**：是否系统识别并评估了工作环境因素
- **WE 2.1-2.2**：是否实施了改进措施并持续监控
- **WE 3.1**：是否有改进效果评估
