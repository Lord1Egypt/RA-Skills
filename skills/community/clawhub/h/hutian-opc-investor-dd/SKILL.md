# 投资者背景调查技能 (Investor Due Diligence)

## 1. 技能介绍

### 技能名称
投资者背景调查

### 适用场景
- 融资前对潜在投资者进行尽职调查
- 评估投资机构的靠谱程度与合作风险
- 识别"毒药钱"（估值陷阱/条款剥削/资源诈欺）
- 制定针对性的融资对接策略

### 核心价值
- **风险规避**：避免与劣迹机构合作，防止资金链断裂
- **策略优化**：根据机构风格定制化BP与谈判策略
- **资源匹配**：找到真正能带来产业赋能的投资者
- **谈判优势**：掌握机构底牌，在条款谈判中占据主动

### 使用限制
- 本技能不涉及法律建议，重大条款请咨询专业律师
- 信息来源为公开数据，准确性依赖数据源质量
- 建议多渠道交叉验证，避免单一信息依赖

## 2. 调查框架

### 2.1 机构基本信息

| 调查维度 | 具体内容 | 数据来源 |
|---------|---------|---------|
| **管理规模** | 在管基金总规模、累计投资项目数 | 中基协、官网、LP报告 |
| **成立时间** | 成立年份、存续期（通常7-10年） | 工商信息、中基协 |
| **团队背景** | 核心合伙人履历、专业背景、投资经验 | 官网、LinkedIn、新闻报道 |
| **股权结构** | 国资/民营/混合所有制、LP构成 | 工商信息、企业公告 |
| **合规资质** | 基金业协会备案、是否有处罚记录 | 中基协官网 |

### 2.2 投资风格画像

| 调查维度 | 具体内容 | 评估标准 |
|---------|---------|---------|
| **阶段偏好** | 种子/天使/Pre-A/A/B/C/D轮 | 与项目阶段匹配度 |
| **行业偏好** | 专注赛道、热点布局 | 与项目方向匹配度 |
| **单笔金额** | 最小/最大/平均投资额 | 与融资需求匹配度 |
| **投资频次** | 年均投资项目数、出手速度 | 评估机构活跃度 |
| **领投/跟投** | 领投比例、独投比例 | 评估话语权 |
| **投资轮次** | 偏好轮次跨度、是否连续跟投 | 评估长期陪伴意愿 |

### 2.3 已投案例分析

| 调查维度 | 具体内容 | 评估重点 |
|---------|---------|---------|
| **同行项目** | 是否投过竞品/同赛道项目 | 利益冲突风险 |
| **退出情况** | IPO/并购/股权转让/清算 | 退出能力评估 |
| **增值服务** | 投后赋能、资源对接案例 | 机构真实实力 |
| **项目存活率** | B轮存活率、死亡项目比例 | 投资质量评估 |
| **估值增长** | 典型项目估值变化曲线 | 增值能力评估 |

### 2.4 决策流程调研

| 调查维度 | 具体内容 | 评估标准 |
|---------|---------|---------|
| **决策链长度** | 初次接触到打款周期 | 效率评估 |
| **关键决策人** | 合伙人/VP/投资经理影响力 | 对接优先级 |
| **投决会构成** | 委员背景、一票否决权 | 决策机制透明度 |
| **条款风格** | 估值调整/回购/反稀释条款偏好 | 条款风险评估 |
| **尽调深度** | 法务/财务/技术尽调要求 | 评估机构专业度 |

### 2.5 行业声誉评价

| 调查维度 | 具体内容 | 数据来源 |
|---------|---------|---------|
| **创业者反馈** | 创业者评价、合作体验 | 社群、同行打听 |
| **媒体报道** | 正面/负面新闻、争议事件 | 财经媒体搜索 |
| **诉讼风险** | 是否涉及投资纠纷诉讼 | 裁判文书网、企查查 |
| **同业口碑** | FA/其他创业者评价 | 行业人脉网络 |
| **监管记录** | 基金业协会处罚、行业警示 | 中基协公告 |

### 2.6 附加价值评估

| 调查维度 | 具体内容 | 评估标准 |
|---------|---------|---------|
| **产业资源** | 能否对接上下游客户/供应商 | 产业协同能力 |
| **政府关系** | 能否带来政策支持/园区落地 | 政策赋能 |
| **人才资源** | 高管招募、团队组建支持 | 人才赋能 |
| **后续融资** | 能否帮助对接下轮融资 | 资本赋能 |
| **退出渠道** | 能否对接并购方/上市公司 | 退出赋能 |

## 5. 执行流程

### Step 1: 基础信息收集（10分钟）

1. **获取机构名单**
   - 确认目标机构名称（全称+简称）
   - 记录官方网站、联系方式

2. **官方信息查询**
   - 登录中基协官网查询备案信息
   - 查询工商登记信息
   - 记录管理规模、成立时间、团队信息

3. **初步分类**
   - 判断机构类型（VC/CVC/政府基金/天使基金）
   - 评估阶段偏好匹配度

**输出**：机构基础信息汇总表

---

### Step 2: 投资案例分析（20分钟）

1. **投资事件查询**
   - IT桔子/天眼查查询历史投资事件
   - 重点关注近3年投资案例
   - 记录投资时间、金额、轮次

2. **赛道匹配分析**
   - 梳理已投项目中与本项目的相关性
   - 识别是否存在竞品投资（利益冲突）
   - 评估本项目在机构portfolio中的稀缺性

3. **退出案例研究**
   - 查询机构已退出项目
   - 分析退出方式（IPO/并购/转让）
   - 评估退出效率与回报

**输出**：投资案例分析报告

---

### Step 3: 行业口碑调研（30分钟）

1. **公开信息搜索**
   - 搜索机构名称+关键词（投诉/纠纷/诉讼）
   - 搜索创业者评价、媒体报道
   - 记录正面与负面信息

2. **社交媒体分析**
   - 关注合伙人LinkedIn、公众号、微博
   - 了解机构投资理念、近期动态
   - 评估机构专业度与活跃度

3. **人脉网络打听**
   - 通过FA机构了解行业口碑
   - 联系已投企业创始人
   - 通过创业社群获取真实评价

**输出**：口碑调研汇总（需标注信息来源）

---

### Step 4: 风险评估（15分钟）

1. **风险清单核对**
   - 逐项检查风险提示清单
   - 标注高/中/低风险项
   - 记录需要进一步核实的内容

2. **综合评分计算**
   - 根据六维度评分标准打分
   - 计算加权综合得分
   - 判断是否进入重点跟进名单

3. **风险应对预案**
   - 针对高风险项准备应对策略
   - 制定条款谈判底线
   - 准备备选机构清单

**输出**：风险评估报告 + 对接决策建议

---

### Step 5: 对接策略制定（20分钟）

1. **画像卡片生成**
   - 按模板填写投资者画像
   - 汇总所有调研信息
   - 提炼机构核心特征

2. **对接策略设计**
   - 制定针对性BP调整建议
   - 设计话题切入角度
   - 准备关键问题应答方案

3. **执行计划安排**
   - 确定对接优先级排序
   - 安排首次接触渠道（FA/直投/引荐）
   - 设定跟进时间表与里程碑

**输出**：完整对接策略文档

## 7. 注意事项

1. **信息时效性**：公开信息可能存在滞后，建议多渠道交叉验证
2. **保密原则**：调研信息仅用于内部决策，严禁外传
3. **合规边界**：禁止使用非法手段获取信息（如黑客攻击）
4. **法律建议**：重大决策请咨询专业律师和财务顾问
5. **动态更新**：机构情况可能变化，建议定期更新调研信息
