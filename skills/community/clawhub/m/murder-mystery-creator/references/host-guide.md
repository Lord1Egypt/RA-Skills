# 主持人手册生成模板

> 主持人是剧本杀游戏的"导演"，手册必须让主持人知道每一个节点该说什么、发什么、注意什么。
> 即使是第一次主持的新手，也能靠这份手册完整运行游戏。

---

## 主持人手册结构

### 第一部分：剧本核心信息速览

```
剧本名称：《[剧本名]》
适合人数：[X]人
建议时长：[X-X]小时
难度等级：[入门/进阶/烧脑]（根据线索密度和逻辑复杂度评定）
情感强度：[轻松/适中/高燃]（根据剧情情感深度评定）

凶手角色：[角色姓名]（主持人保密，不能剧透）
作案手法：[一句话概括，如"在11点至12点间用毒酒"]
关键核心证据：[最终锁定凶手的那条线索是什么]

故事主题一句话：[用于游戏结束后主持人做复盘点评时使用]
```

---

### 第二部分：开场词脚本（逐字稿）

```
【调试阶段（正式开场前）】
主持人应做好：
- 确认所有玩家入座，手机静音
- 发放角色卡，提醒玩家只看自己的部分
- 确认场景道具/线索卡已按位摆放

【正式开场词（逐字稿）】

"各位，欢迎来到今晚的剧本杀《[剧本名]》。

在游戏正式开始之前，我来说明一些规则：

第一，你们每个人都有一张角色卡，卡上分为公开部分和私密部分。
公开部分可以在自我介绍时读给所有人听，私密部分请绝对不要主动透露。

第二，游戏分为[X]个阶段：
[按实际阶段列出，如：自我介绍 → 第一轮搜证 → 第一轮讨论 → 第二轮搜证 → 质询 → 投票 → 真相揭秘]

第三，每位玩家在游戏过程中可以撒谎，可以结盟，可以揭发他人。
唯一的要求是：不能直接说出私密部分的内容，但可以用暗示或间接方式透露。

第四，游戏结束时，每人秘密投票选出你认为的凶手。

现在，让我来为大家介绍今晚的故事背景——"

[接续案件背景介绍，逐字稿，见下方]

---

【案件背景介绍（逐字稿）】
[在此填入从大纲第【3】节"事件触发"中提取的公共开场材料，
 用渲染氛围的叙事语言朗读，建议加入背景音乐提示]

"[具体背景描述文字...（从大纲转写）]"

"好了，大家对背景已经有了初步了解。现在，让我们开始第一个环节——自我介绍。
 请从[角色1]开始，依次向大家介绍你的公开部分。"
```

---

### 第三部分：各阶段主持流程

#### 阶段一：自我介绍（建议15-20分钟）

```
主持人操作：
1. 指定顺序，让每位玩家朗读角色卡公开部分
2. 每人介绍完后，主持人可追问1-2个开放性问题帮助活跃气氛
   推荐问题："你和死者最后一次见面是什么时候？"
             "在你看来，[某角色]是什么样的人？"
3. 所有人介绍完后，宣布："好了，现在你们已经对彼此有了初步了解。
   接下来进入第一轮搜证阶段。"

注意事项：
- 如果某玩家太紧张，鼓励他们可以参考角色卡念
- 如果某玩家说得太多透露了私密信息，委婉提醒："这个问题我们后面可以深入聊"
```

#### 阶段二：第一轮搜证（建议10-15分钟）

```
主持人操作：
1. 宣布搜证规则："现在开始第一轮搜证。大家可以在[指定场景区域]内搜索，
   找到线索卡后请拿走，但同一地点只能被搜索一次。"
2. 按照线索发放总表，在规定地点摆放好第一轮线索卡
3. 主持人主动宣读：[线索#X，主动发放类]
4. 计时，到时间提醒玩家搜证结束

注意事项：
- 如果某条线索没有被找到但很重要，可以在第一轮讨论中"有人发现了XX"方式主动引入
- 场景道具请在游戏开始前摆好，不要临时找
```

#### 阶段三：第一轮讨论（建议20-30分钟）

```
主持人操作：
1. 宣布："好的，搜证阶段结束。现在开始第一轮自由讨论。
   大家可以分享你找到的线索，也可以质问任何一位玩家。"
2. 主持人观察，不主动介入，记录"关键时刻"（准备在复盘时回顾）
3. 快到时间时，提醒："还有5分钟，大家有什么问题可以抓紧问"

如果玩家推理方向完全偏离，卡场时使用以下引导词：
  "有没有人注意到[某线索]和[某时间段]之间的关系？"
  "刚才[角色X]说的话和[角色Y]的陈述之间好像有点出入，大家怎么看？"
```

#### 阶段四：第二轮搜证（建议10分钟）

```
[同第一轮搜证，但发放第二轮线索]
```

#### 阶段五：质询环节（建议20-30分钟）

```
主持人操作：
1. 宣布："现在进入质询环节。每位玩家有权要求任何一人回答一个问题。
   被问到的玩家必须作答，可以说谎，但不能拒绝回答。"
2. 建议质询顺序（可自由化）：由可疑度高的角色先被质询
3. 如果质询太混乱，可以采用"轮流质询"制：每人问一个，轮流进行

注意事项：
- 防止某玩家被"霸凌式"集体围攻，适时分散话题
- 如果凶手要被逼出来了但还不到投票时间，引导其他嫌疑方向
```

#### 阶段六：第三轮线索（终极线索，在投票前X分钟发放）

```
主持人宣布："好的，现在到了最关键的时刻。我要发放今晚的最后一批线索。"
[发放第三轮线索卡，或者宣读终极线索内容]
"大家还有[5]分钟时间思考，然后进入投票。"
```

#### 阶段七：凶手陈述 & 投票

```
主持人操作：
1. "在投票之前，每位玩家有60秒时间做最后陈述，说明你认为谁是凶手以及理由。"
2. [特别注意：凶手玩家也要在此时发表自己的辩护词，可以提醒他们参考凶手辩护词]
3. "好，现在请大家在纸条上写下你认为的凶手姓名，折好交给我。"
4. 统计投票结果，宣布："让我们来揭晓投票结果——"
   [宣读票数]
   "获得最多票数的是[姓名]。那么真相到底是什么呢？"
```

### 第四部分：真相揭秘脚本（逐字稿）

```
【开场氛围渲染】
"现在，让我来告诉大家今晚真正发生了什么。

在揭秘之前，请大家先闭上眼睛，回想一下今晚你对每个角色的印象......

[停顿3秒]

好，睁眼。"

---

【真相揭示主体（按时间顺序讲述真相）】

"今晚的故事，要从[XX时间点]说起——

[按照大纲中"完整时间线"重新组织，用叙事语言逐步揭示真相]

[高潮处减慢语速，渲染情感]

[宣布凶手身份时，眼神直视凶手玩家]
"所以，今晚真正的凶手，是——[角色姓名]。"

---

【真相细节补充】
"让我们来看几个今晚大家可能没有注意到的细节——

[补充3-5个游戏中暗藏的彩蛋和伏笔，增加复盘惊喜感]"

---

【情感收尾（情感本/欢乐本）】
"[根据剧本主题，说一段有温度的结语。
 情感本：回扣故事主题，讲凶手的悲剧根源，触动玩家
 欢乐本：揭秘所有角色搞笑秘密的后续，带来欢笑收尾]"
```

### 第五部分：玩家卡壳应对指南

| 情况 | 引导方式 |
|------|---------|
| 玩家完全不说话 | "XX，你在游戏开始之前注意到什么了吗？" |
| 玩家在互相聊天偏题 | "这个问题很有意思，但我们还有个关键的线索没有分析..." |
| 玩家猜到凶手但提前太多 | "你的推断很有意思。但在确认之前，你们有没有想过[另一个嫌疑角色]？" |
| 玩家吵架/情绪激动 | "大家记住这是游戏，[玩家名]提出的疑点是值得思考的——" |
| 没人搜到重要线索 | "我要提醒大家，[某地点]的线索还没有人仔细看过..." |
| 游戏超时 | 跳过质询环节最后10分钟，直接进入终极线索发放 |

---

### 第六部分：结算规则

```
【投票结算】
- 猜中凶手：[X分 / 或"获得胜利"]
- 凶手成功逃脱（超过一半玩家没猜中）：[凶手获得X分 / 或"凶手胜利"]
- 隐藏目标达成：[额外X分]

【颁奖建议（增加仪式感）】
- 最佳推理奖：猜中凶手且陈述最有逻辑的玩家
- 最佳表演奖：角色代入感最强的玩家
- 最佳凶手奖：凶手玩家迷惑最多人
- 最佳反转奖：被误判可疑度最高但实际无辜的玩家

【打分记录表】
| 玩家 | 角色 | 猜中凶手 | 隐藏目标 | 总分 |

| | | | | |
```
