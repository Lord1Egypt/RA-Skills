Global <br>

## Known Risks and Mitigations: <br>
Risk: The security review found that the skill requests local shell and file access that is not necessary for generating ad creative. <br>
Mitigation: Remove or restrict Bash and Read permissions unless local file access or shell commands are intentionally required. <br>
Risk: The security review reported that install instructions point to a different source than the reviewed package. <br>
Mitigation: Verify any install command or repository against the reviewed ClawHub package before installation. <br>
Risk: Generated ad copy or imagery may contain inaccurate, noncompliant, or brand-inappropriate claims before publication. <br>
Mitigation: Review and approve generated variations against campaign, brand, platform, and legal requirements before external use. <br>
