# SWOT分析模板

## 概述

SWOT分析是一种战略规划工具，用于评估企业的内外部环境，识别优势、劣势、机会和威胁。

## 四要素定义

### S-优势（Strengths）
**定义**：内部有利因素，核心竞争力

**分类维度**：
- 技术优势：核心技术、专利壁垒、研发能力
- 资源优势：资金、渠道、品牌、客户
- 能力优势：团队能力、运营经验、管理水平
- 定位优势：市场定位、差异化、品牌认知

**分析问题**：
- 我们擅长什么？
- 竞争对手做不到什么？
- 客户认可我们什么？
- 我们的独特能力是什么？

### W-劣势（Weaknesses）
**定义**：内部不利因素，短板差距

**分类维度**：
- 资源不足：资金、人才、渠道
- 能力缺失：研发、运营、管理
- 市场劣势：品牌、份额、认知
- 成本劣势：生产、获客、运营

**分析问题**：
- 我们缺乏什么？
- 竞争对手比我们强什么？
- 客户抱怨我们什么？
- 我们的短板在哪里？

### O-机会（Opportunities）
**定义**：外部有利因素，市场窗口

**分类维度**：
- 市场机会：市场空白、需求增长、细分机会
- 政策机会：政策利好、扶持补贴、政府采购
- 技术机会：技术突破、跨界融合、趋势红利
- 合作机会：战略伙伴、生态协同、资源整合

**分析问题**：
- 市场有什么新需求？
- 技术有什么新发展？
- 政策有什么新变化？
- 生态有什么新机会？

### T-威胁（Threats）
**定义**：外部不利因素，风险挑战

**分类维度**：
- 竞争威胁：竞争对手、替代产品、替代服务
- 政策威胁：政策变化、监管收紧、标准提高
- 技术威胁：技术替代、颠覆创新、专利封锁
- 市场威胁：需求变化、进入门槛、客户流失

**分析问题**：
- 市场有什么风险？
- 竞争对手在做什么？
- 政策有什么不利变化？
- 技术有什么颠覆风险？

## SWOT交叉策略

### SO策略（增长策略）
**组合**：优势 × 机会
**逻辑**：利用内部优势抓住外部机会

**策略模板**：
```
利用[优势]抓住[机会]
- 行动1：......
- 行动2：......
- 行动3：......
```

### WO策略（改善策略）
**组合**：劣势 × 机会
**逻辑**：利用外部机会弥补内部劣势

**策略模板**：
```
通过[机会]弥补[劣势]
- 行动1：......
- 行动2：......
- 行动3：......
```

### ST策略（防御策略）
**组合**：优势 × 威胁
**逻辑**：利用内部优势规避外部威胁

**策略模板**：
```
利用[优势]应对[威胁]
- 行动1：......
- 行动2：......
- 行动3：......
```

### WT策略（避险策略）
**组合**：劣势 × 威胁
**逻辑**：减少内部劣势规避外部威胁

**策略模板**：
```
改善[劣势]规避[威胁]
- 行动1：......
- 行动2：......
- 行动3：......
```

## 评分与排序

### 重要性评级
| 级别 | 含义 | 描述 |
|------|------|------|
| 高 | 决定性因素 | 对商业模式成败有决定性影响 |
| 中 | 重要因素 | 对商业模式有显著影响 |
| 低 | 一般因素 | 对商业模式有影响但不关键 |

### 排序原则
1. 高优先级排在前面
2. 同级别按相关性排序
3. 数量控制在合理范围（每个维度3-5项）

## 商业计划书提取指南

### 优势提取关键词
- 优势、核心竞争力、独特能力、技术领先、专利、品牌、团队、经验、资源、渠道、客户、合作伙伴

### 劣势提取关键词
- 不足、缺陷、短板、缺失、薄弱、劣势、困难、挑战、风险、问题

### 机会提取关键词
- 机会、机遇、趋势、风口、市场、需求、政策、利好、红利、空白、增长

### 威胁提取关键词
- 威胁、风险、挑战、竞争、替代、威胁、风险、政策变化、技术替代、市场变化

## 注意事项

1. **客观公正**：优势不夸大，劣势不回避
2. **数据支撑**：每项判断有依据
3. **区分内外**：内部分析S/W，外部分析O/T
4. **动态视角**：考虑时间因素（当前/未来）
5. **不编造**：无法提取的信息如实标注
6. **策略落地**：交叉策略要有可操作性
