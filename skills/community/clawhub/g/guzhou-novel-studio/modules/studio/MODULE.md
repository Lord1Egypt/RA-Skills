# 长篇管理模块

> **路径**：modules/studio/
> **来源**：GitHub - novel-studio

## 模块说明

长篇管理模块负责长篇小说的项目管理和记忆系统。

## 核心功能

| 功能 | 说明 |
|------|------|
| 项目管理 | 创建和管理小说项目 |
| 章节追踪 | 进度管理和状态追踪 |
| 人物管理 | 人物档案和关系图谱 |
| 伏笔管理 | 伏笔追踪和优先级标记 |
| 记忆同步 | 章节回填和记忆压缩 |

## 项目结构

```
{project-name}/
├── project.json              # 项目配置
├── outline/                   # 大纲目录
│   ├── outline.json
│   ├── outline.md
│   ├── chapters.md
│   └── foreshadowing.md
├── characters/                # 人物档案
├── world/                    # 世界观设定
├── chapters/                 # 章节正文
├── memory/                   # 记忆系统
│   ├── style-dna.json
│   ├── memory.json
│   ├── active/
│   └── archive/
├── qa-reports/               # 质检报告
└── assets/                   # 辅助资源
```

## 工作流程

```
立项 → 项目初始化 → 大纲生成 → 风格提取 → 章节创作 → 质检 → 修改 → 章节回填 → 下一章
  │         │             │           │           │        │      │          │
  │         ▼             ▼           ▼           ▼        ▼      ▼          ▼
  │    创建项目      读取/生成    提取Style    读取记忆包  质量检测  修正内容  更新记忆  循环
  │    目录结构      大纲        -DNA       进行创作    出报告    完成创作  继续
  │
  └──→ 阶段回顾（每5-10章）→ 记忆压缩 → 漂移校准
```
