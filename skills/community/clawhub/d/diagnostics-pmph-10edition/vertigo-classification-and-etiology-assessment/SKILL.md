# 眩晕分类与病因评估

## 何时使用
- 患者明确描述“天旋地转”或“自身摇晃”等典型眩晕症状
- 已初步排除晕厥、癫痫、心因性假性晕厥等非眩晕性疾病
- 需要基于临床特征快速区分周围性与中枢性眩晕

## 执行步骤

### 第一步：按解剖部位分类眩晕
**A. 周围性眩晕（耳源性）**：病变位于内耳前庭至前庭神经颅外段
- **梅尼埃病**：发作性眩晕 + 耳鸣 + 听力下降 + 眼球震颤，持续<2周，可复发
- **迷路炎**：常继发于中耳炎，伴鼓膜穿孔、耳鸣、听力下降
- **前庭神经元炎**：上呼吸道感染后突发眩晕，无耳鸣/听力下降，可持续达6周
- **药物中毒**：链霉素、庆大霉素等致渐进性眩晕+耳鸣+听力下降；水杨酸、奎宁、氯丙嗪亦可引起
- **良性阵发性位置性眩晕（BPPV）**：特定头位诱发，通常无听力障碍
- **晕动病**：乘车船飞机时发生，伴恶心、呕吐、苍白、出汗

**B. 中枢性眩晕（脑源性）**：病变位于前庭神经颅内段、脑干、小脑或大脑
- **血管性**：椎-基底动脉供血不足、小脑/脑干出血、高血压脑病
- **占位性**：听神经瘤（进行性耳鸣+听力下降+头痛+复视）、小脑肿瘤
- **感染性**：小脑脓肿、颅后窝蛛网膜炎
- **脱髓鞘/变性**：多发性硬化（肢体无力+感觉异常+眩晕）、延髓空洞症（软腭瘫+吞咽困难+发音障碍）
- **癫痫**：颞叶或前庭皮层癫痫可表现为孤立性眩晕发作

**C. 其他原因**
- **全身性疾病**：心律失常、低血压、贫血、尿毒症、肝病、糖尿病
- **眼源性**：屈光不正、青光眼、长时间屏幕使用
- **神经精神性**：焦虑症、抑郁症，常伴失眠、心悸、情绪低落，但无客观前庭体征

### 第二步：结合关键伴随症状鉴别
- **耳鸣 + 听力下降** → 提示周围性或第八脑神经病变
- **严重恶心呕吐** → 常见于梅尼埃病、晕动病
- **共济失调、构音障碍、复视** → 强烈提示小脑/脑干中枢病变
- **眼球震颤**：方向、持续时间及是否抑制有助于定位（如垂直性/旋转性眼震多为中枢性）

### 第三步：输出建议
- **周围性倾向**：建议纯音测听、前庭功能检查、Dix-Hallpike试验
- **中枢性警示征象**（如急性共济失调、复视、吞咽困难、新发头痛）：立即安排头颅MRI，警惕脑干卒中
- **全身性线索**：完善心电图、血常规、电解质、血糖、肝肾功能

> 注意：单纯头晕（头重脚轻、昏沉感）无旋转感者不适用本技能。
