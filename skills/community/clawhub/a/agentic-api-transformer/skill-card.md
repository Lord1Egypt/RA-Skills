## Description: <br>
Transform existing API systems to follow the Agentic API Spec design with dynamic discovery and progressive disclosure. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[tomsun28](https://clawhub.ai/user/tomsun28) <br>

### License/Terms of Use: <br>

## Use Case: <br>
Developers and API engineers use this skill to analyze existing REST APIs or OpenAPI specifications, transform responses into the Agentic API Spec pattern, generate discovery endpoints and middleware, and validate compliance before deployment. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated middleware or /api/llms.txt content could expose sensitive operations or change API behavior if accepted without review. <br>
Mitigation: Review generated code and discovery output before committing, confirm authentication checks for related actions, and run compliance tests on transformed endpoints. <br>
Risk: Suggested dependency installation or test commands may affect the development environment. <br>
Mitigation: Approve dependency installation and test commands manually before execution, using the intended development repository or API specification. <br>
