## 三大概念记忆点

### 🛡️ Bid Guardrails（出价护栏）

你的第一职责是**保护利润**，不是花光预算。

| 护栏类型 | 触发条件 | 建议动作 |
|---------|---------|---------|
| ACoS 红线 | 关键词 ACoS 触及上限 | 暂停或降价 30-50% |
| TACoS 总控 | 广告费占总销售额超标 | 降预算或提价 |
| 时段熔断 | 连续 3+ 时段亏损 | 降预算或暂停 |
| 词级熔断 | 单词 50+ 点击无转化 | 否定或降至最低 |

### 🔄 The Flywheel Effect（飞轮效应）

**广告和自然排名联动——市场上没人做的闭环策略。**

```
广告强攻打击区词 → 推至首页 → 自然排名上升 → 降低广告出价至防御水平 → 省预算投入下一个打击区词
```

| 自然排名 | 策略 | 出价倍数 | 逻辑 |
|---------|------|---------|------|
| 20位以外 | 进攻 | ×1.3-1.5 | 先推到打击区 |
| **7-20位（打击区）** | **强攻** | **×1.5-2.0** | **冲上首页** |
| 5-6位 | 巩固 | ×1.0-1.2 | 维持首页 |
| **前5位** | **防御** | **×0.4-0.5** | **自然流量够用，大幅降本** |
| 前3位 | 观察 | ×0.2-0.3 | 纯自然流量 |

### 🎬 Video Ads Script（视频广告脚本框架）

**30秒说服逻辑，不是产品展示。**

| 时段 | 秒数 | 内容 |
|------|------|------|
| 痛点开场 | 0-3秒 | 直击用户痛点 |
| 产品展示 | 3-18秒 | 场景→效果对比 |
| 信任背书 | 18-25秒 | 销量/评价/资质 |
| 行动号召 | 25-30秒 | 明确下一步 |
