# 联动Skill清单

> 打通口播脚本→内容采集→视频制作→发布分发的全链路。当用户需要把口播脚本变成实际视频、抓取竞品数据、或批量分发时加载此文件。

---

## 全链路联动图谱

```
口播脚本 → 拍摄/制作 → 剪辑 → 发布 → 数据复盘
   │           │          │       │        │
   │     chanjing     vectcut   wechat   douyin
   │     wangcut      video     -qq      playwright
   │                  bundle    sender   scrape
   │
   └── 本skill（daily-100-methodology）生成口播脚本
```

---

## 一、口播脚本 → 视频成片

| Skill | 调用方式 | 适用场景 | 输入 | 输出 |
|-------|----------|----------|------|------|
| **chanjing-one-click-video-creation** | 传入口播脚本 | 一键生成完整短视频（数字人+AI画面） | 口播脚本/选题 | 视频成片 |
| **wangcut** | 传入文案创建任务 | 批量创建剪辑任务（随机7素材） | 口播文案 | 剪辑视频下载 |
| **vectcut-skill** | 传入素材+脚本 | 全能剪辑（口播/混剪/电商广告） | 脚本+素材 | 成片+字幕+特效 |

### 典型联动流程

```
用户："帮我把这20条口播全生成视频"
  ↓
1. 本skill生成20条口播脚本
2. 调用 chanjing-one-click-video-creation 逐个生成
3. 或调用 wangcut 批量创建任务
4. 返回视频成片列表
```

## 六、联动调用规范

### 显式调用（用户明确指令）
```
用户："帮我把这5条口播生成视频"
→ 本skill先生成5条口播
→ 然后Skill("chanjing-one-click-video-creation") 逐个生成
```

### 隐式调用（skill自动判断）
```
用户："看看我竞品最近发了啥"
→ 本skill自动判断：需要抓数→调用douyin-playwright-scrape
→ 需要转写→调用douyin-transcribe
→ 需要分析→用本skill方法论输出
```

### 不调用的情况
- 用户只是要方案/脚本/选题 → 纯本skill产出
- 用户没说"生成视频""抓数据""发出去" → 不主动触发联动
