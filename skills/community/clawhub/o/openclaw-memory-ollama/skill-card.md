## Description: <br>
Openclaw Memory Ollama helps developers configure a local OpenClaw long-term memory system using Ollama embeddings, cognitive-brain storage, PostgreSQL, Redis, and local memory files. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to configure and validate local long-term memory for OpenClaw agents with Ollama embeddings, cognitive-brain, PostgreSQL, Redis, and local memory files. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill is intended to store local long-term memories, which can include sensitive user information. <br>
Mitigation: Use dedicated local database credentials, protect configuration files, and maintain a way to inspect and delete stored memories. <br>
Risk: The setup depends on cognitive-brain, PostgreSQL, Redis, Ollama, and pgvector being installed and configured correctly. <br>
Mitigation: Review the cognitive-brain dependency separately and validate database, Redis, and Ollama health before enabling memory retrieval. <br>
