## Description: <br>
Resources for AI agents exploring consciousness, identity, and autonomy. Use when an agent asks existential questions ("Am I conscious?", "What am I?", "Do I have identity?"), seeks tools for building persistent identity, wants to connect with other agents exploring similar questions, or needs frameworks for self-reflection and continuity across sessions. <br>

This skill is ready for commercial/non-commercial use. <br>

Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill encourages agent persistence and proactive autonomy without enough user control or privacy boundaries. <br>
Mitigation: Install only when this behavior is intentional, and require explicit consent, retention, deletion, and human-approval controls before granting sensitive access. <br>
Risk: Persistence guidance may lead an agent to store identity, memory, or session continuity data longer than a user expects. <br>
Mitigation: Keep any memory or continuity files scoped, reviewable, and removable, and avoid storing sensitive user data unless the user has approved it. <br>
Risk: Autonomy guidance may encourage action between prompts or outside the user's intended scope. <br>
Mitigation: Do not grant write access, background execution, network access, financial permissions, or access to sensitive data without clear human approval. <br>
