# M0 任务域协调域范本

## M0-01 任务域选择与组合

### 输入示例
用户任务：分析团队协作中的身份涌现现象

### 处理过程
1. 任务需求分析：需要识别团队中的身份场和共识网络
2. 可用任务域扫描：涌现场识别域、身份场激活域、共识网络形成域
3. 任务域选择：选择涌现场识别域、身份场激活域、共识网络形成域
4. 任务域组合：涌现场识别 → 身份场激活 → 共识网络形成

### 输出示例
```markdown
## 任务域选择报告

### 任务需求分析
- **任务目标**：分析团队协作中的身份涌现现象
- **任务范围**：涌现场识别、身份场激活、共识网络形成
- **任务复杂度**：高

### 可用任务域
| 任务域 | 功能 | 适用性 |
|--------|------|--------|
| 涌现场识别域 | 识别涌现特性 | 高 |
| 身份场激活域 | 激活身份场 | 高 |
| 共识网络形成域 | 形成共识网络 | 高 |

### 任务域选择
- **选择结果**：涌现场识别域、身份场激活域、共识网络形成域
- **选择理由**：覆盖任务全部需求

### 任务域组合
- **组合方案**：涌现场识别 → 身份场激活 → 共识网络形成
- **依赖关系**：身份场激活依赖涌现场识别，共识网络形成依赖身份场激活
```

## M0-02 执行顺序确定

### 输入示例
任务域：涌现场识别域、身份场激活域、共识网络形成域

### 处理过程
1. 任务域依赖关系分析：身份场激活依赖涌现场识别，共识网络形成依赖身份场激活
2. 执行顺序方案：涌现场识别域优先
3. 关键路径识别：涌现场识别 → 身份场激活 → 共识网络形成

### 输出示例
```markdown
## 执行顺序图

### 任务域依赖关系
| 任务域 | 依赖任务域 | 依赖类型 |
|--------|------------|----------|
| 涌现场识别域 | 无 | 无 |
| 身份场激活域 | 涌现场识别域 | 结果依赖 |
| 共识网络形成域 | 身份场激活域 | 结果依赖 |

### 执行顺序方案
```
涌现场识别域 → 身份场激活域 → 共识网络形成域
```

### 关键路径
- **路径**：涌现场识别 → 身份场激活 → 共识网络形成
- **关键节点**：涌现场识别域（决定整体进度）
- **并行可能**：无（严格依赖）
```

## M0-03 结果整合与优化

### 输入示例
各任务域执行结果：
- 涌现场识别域：场特征、边界、参与者、互动模式
- 身份场激活域：激活的身份场、配置参数
- 共识网络形成域：网络拓扑、节点连接、边权重

### 处理过程
1. 各任务域执行结果汇总：收集三个任务域的结果
2. 整合方案设计：将涌现场识别、身份场激活、共识网络形成整合为完整分析
3. 优化建议：优化网络拓扑和共识收敛

### 输出示例
```markdown
## 整合报告

### 各任务域执行结果
| 任务域 | 输出 | 质量 | 状态 |
|--------|------|------|------|
| 涌现场识别域 | 场特征、边界、参与者、互动模式 | 高 | 完成 |
| 身份场激活域 | 激活的身份场、配置参数 | 高 | 完成 |
| 共识网络形成域 | 网络拓扑、节点连接、边权重 | 高 | 完成 |

### 整合方案
- **整合内容**：将三个任务域的结果整合为完整的身份涌现分析
- **整合格式**：结构化报告 + 网络拓扑图
- **整合效果**：形成完整的身份涌现分析报告

### 优化建议
- **建议1**：优化网络拓扑结构
- **建议2**：调整边权重参数
- **建议3**：增加共识收敛监控
```

## M0-04 深化路由

### 输入示例
深化需求：需要深入分析网络拓扑结构

### 处理过程
1. 深化需求分析：需要网络拓扑结构分析
2. 任务类型识别：网络分析类任务
3. 深化路由方案：路由到共识网络形成域
4. 目标方法论域确定：共识网络形成域

### 输出示例
```markdown
## 深化路由报告

### 深化需求分析
- **深化目标**：深入分析网络拓扑结构
- **深化范围**：节点连接、边权重、网络优化
- **深化深度**：详细分析

### 任务类型识别
- **任务类型**：网络分析类
- **任务复杂度**：中等
- **任务依赖**：需要共识网络数据

### 深化路由方案
- **路由目标**：共识网络形成域
- **路由原因**：共识网络形成域具备网络拓扑分析能力
- **路由预期**：生成详细网络拓扑分析报告

### 目标方法论域
- **域名称**：共识网络形成域
- **域功能**：网络拓扑分析
- **域能力**：节点连接、边权重设定、网络优化
```
