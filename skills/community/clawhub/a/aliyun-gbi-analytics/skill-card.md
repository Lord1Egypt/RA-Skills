## Known Risks and Mitigations: <br>
Risk: The skill can guide cloud-management actions for Alibaba Cloud DataAnalysisGBI resources, including mutating Create, Update, Modify, Set, or similar API calls. <br>
Mitigation: Use least-privilege Alibaba Cloud credentials and require explicit user confirmation before any mutating operation. <br>
Risk: Generated API inventories, command outputs, or response summaries may contain operational details from the user's cloud environment. <br>
Mitigation: Review files under output/aliyun-gbi-analytics/ before sharing or relying on them, and avoid storing unnecessary sensitive values. <br>
