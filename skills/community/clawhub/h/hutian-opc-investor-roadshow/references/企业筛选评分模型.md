# 企业筛选评分模型

## 评分模型说明

本评分模型用于从候选企业池中筛选出最具潜力的招商标的，评分维度覆盖团队、技术、商业模式、财务、产业契合五大领域。

---

## 综合评分表（100分制）

### 一、团队实力（20分）

#### 1.1 创始人背景（0-8分）

| 评分 | 条件 | 说明 |
|------|------|------|
| 8分 | 985/211高校+海归背景+上市公司高管 | 顶级背景 |
| 6-7分 | 985/211高校 或 海归背景 或 上市公司背景 | 优秀背景 |
| 4-5分 | 普通本科+行业经验10年+ | 良好背景 |
| 2-3分 | 普通本科+行业经验5年 | 一般背景 |
| 0-1分 | 学历或经验不足 | 较弱背景 |

#### 1.2 核心团队稳定性（0-6分）

| 评分 | 条件 |
|------|------|
| 6分 | 核心团队合作3年以上+股权激励到位 |
| 4-5分 | 核心团队合作2-3年，有股权激励 |
| 2-3分 | 核心团队合作1-2年 |
| 0-1分 | 团队组建时间短或不稳定 |

#### 1.3 团队完整性（0-6分）

| 评分 | 条件 |
|------|------|
| 6分 | 研发+市场+运营+财务齐全 |
| 4-5分 | 研发+市场+运营/财务 缺1项 |
| 2-3分 | 有核心创始人但团队不完整 |
| 0-1分 | 严重依赖创始人 |

---

### 二、技术壁垒（20分）

#### 2.1 专利数量（0-8分）

| 评分 | 发明专利数量 | 实用新型/外观专利 |
|------|-------------|-------------------|
| 8分 | 10项以上 | 20项以上 |
| 6-7分 | 5-10项 | 10-20项 |
| 4-5分 | 2-5项 | 5-10项 |
| 2-3分 | 1-2项 | 3-5项 |
| 0-1分 | 无专利 | 无专利 |

#### 2.2 技术领先性（0-6分）

| 评分 | 条件 |
|------|------|
| 6分 | 国际领先，拥有核心技术壁垒 |
| 4-5分 | 国内领先，行业前3 |
| 2-3分 | 行业平均水平，有一定技术优势 |
| 0-1分 | 技术门槛低，易被模仿 |

#### 2.3 技术TRL等级（0-6分）

| TRL等级 | 定义 | 评分 |
|---------|------|------|
| TRL 9 | 实际系统已成功运行 | 6分 |
| TRL 7-8 | 系统原型验证/系统 commissioning | 5-6分 |
| TRL 5-6 | 相关环境验证/系统模型验证 | 3-4分 |
| TRL 3-4 | 实验验证/计算模拟验证 | 1-2分 |
| TRL 1-2 | 基本原理/概念形成 | 0分 |

---

### 三、商业模式（20分）

#### 3.1 市场规模（0-6分）

| 评分 | 目标市场规模（TAM） |
|------|---------------------|
| 6分 | 100亿以上 |
| 4-5分 | 10-100亿 |
| 2-3分 | 1-10亿 |
| 0-1分 | 1亿以下 |

#### 3.2 商业模式清晰度（0-8分）

| 评分 | 条件 |
|------|------|
| 8分 | 已验证商业模式+稳定客户+清晰盈利路径 |
| 6-7分 | 商业模式清晰+有收入验证 |
| 4-5分 | 商业模式基本清晰+小规模验证 |
| 2-3分 | 商业模式探索中 |
| 0-1分 | 商业模式不清晰 |

#### 3.3 增长潜力（0-6分）

| 评分 | 年增长率 |
|------|----------|
| 6分 | 50%以上 |
| 4-5分 | 30-50% |
| 2-3分 | 10-30% |
| 0-1分 | 10%以下或负增长 |

---

### 四、财务健康（20分）

#### 4.1 营收规模（0-8分）

| 评分 | 年营收 |
|------|--------|
| 8分 | 1亿以上 |
| 6-7分 | 3000万-1亿 |
| 4-5分 | 1000-3000万 |
| 2-3分 | 500-1000万 |
| 0-1分 | 500万以下 |

#### 4.2 融资阶段（0-6分）

| 评分 | 融资阶段 |
|------|----------|
| 6分 | C轮及以上 |
| 4-5分 | B轮 |
| 2-3分 | A轮 |
| 0-1分 | 未融资或天使轮 |

#### 4.3 现金流（0-6分）

| 评分 | 现金流状况 |
|------|------------|
| 6分 | 正向现金流稳定 |
| 4-5分 | 现金流基本平衡 |
| 2-3分 | 亏损但有融资支撑 |
| 0-1分 | 现金流紧张 |

---

### 五、产业契合（20分）

#### 5.1 产业链位置（0-8分）

| 评分 | 产业链位置 |
|------|------------|
| 8分 | 链主企业/平台型企业 |
| 6-7分 | 关键配套/核心技术 |
| 4-5分 | 一般配套 |
| 2-3分 | 末端应用 |
| 0-1分 | 产业链边缘 |

#### 5.2 区域产业匹配（0-6分）

| 评分 | 匹配程度 |
|------|----------|
| 6分 | 与目标园区产业100%契合 |
| 4-5分 | 与目标园区产业80%契合 |
| 2-3分 | 与目标园区产业50%契合 |
| 0-1分 | 产业不匹配 |

#### 5.3 带动效应（0-6分）

| 评分 | 带动效应 |
|------|----------|
| 6分 | 可带动10家以上上下游 |
| 4-5分 | 可带动5-10家上下游 |
| 2-3分 | 可带动2-5家上下游 |
| 0-1分 | 单体企业，无带动 |

---

## 评分结果等级

| 等级 | 分数区间 | 招商优先级 | 建议动作 |
|------|----------|------------|----------|
| A级 | 85-100分 | 最高优先级 | 重点跟进，资源倾斜 |
| B级 | 70-84分 | 高优先级 | 积极跟进，适度资源 |
| C级 | 55-69分 | 中优先级 | 保持联系，观察发展 |
| D级 | 40-54分 | 低优先级 | 持续关注 |
| E级 | 40分以下 | 暂不考虑 | 归档观察 |

## 评分工具

```python

screener = EnterpriseScreener()

# 添加企业
screener.add_enterprise("深圳智链科技有限公司", {
    "founder_background": "海归+行业经验",
    "team_stability": 5,
    "team_completeness": 5,
    "patents": 3,
    "tech_leadership": "国内领先",
    "trl_level": 7,
    "market_size": 120,
    "biz_model": "已验证",
    "growth_rate": 40,
    "revenue": 800,
    "funding_stage": "A轮后",
    "cash_flow": "基本平衡",
    "industry_position": "关键配套",
    "region_match": 0.9,
    "chain_effect": 6
})

# 获取评分结果
results = screener.get_ranking()
```
