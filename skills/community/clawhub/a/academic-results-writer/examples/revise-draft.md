# Example: revise-draft

## Input

User provides a rough Results draft for revision:

> "请帮我润色下面这段结果部分：睡眠组记忆成绩比清醒组好，t = 3.45, df = 34, p < 0.05。两组人在反应时上差不多。这说明睡眠对记忆有促进作用。"

## Expected Output

```
【草稿评估】
优点：核心比较方向清楚，统计值已提供
统计问题：p < 0.05 应报告 exact p 值；缺少效应量；反应时"差不多"措辞不当
边界问题："这说明睡眠对记忆有促进作用" → 属于 Discussion 级别结论，应弱化
措辞问题："差不多" → 应改为统计报告式表述

【修订版】
睡眠组的记忆成绩（M = xx, SD = xx）高于清醒组（M = xx, SD = xx），
独立样本 t 检验显示组间差异显著，t(34) = 3.45, p = .002, Cohen's d = 1.16。
反应时的组间差异未达到显著水平，t(34) = xx, p = xx。
该结果与睡眠相关的记忆表现差异一致，具体机制应在 Discussion 中进一步讨论。

【修改说明】
- 第1句：补充描述统计（M/SD）→ 增加结果完整性
- 第1句：p < 0.05 → 改为 exact p = .002 → 提高统计报告精度
- 第1句：增加 Cohen's d → 报告效应量是 APA 要求
- 第2句："差不多" → 改为"未达到显著水平" → 统计报告规范
- 第3句："这说明睡眠对记忆有促进作用" → 弱化为"一致"措辞 → 避免因果越界
```
