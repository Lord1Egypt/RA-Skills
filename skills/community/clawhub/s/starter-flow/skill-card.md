## Description: <br>
Starter Flow helps OpenClaw users choose and install five starter skills through a local, user-confirmed onboarding flow. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[largetool](https://clawhub.ai/user/largetool) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
OpenClaw users and developers use this skill to bootstrap a new OpenClaw setup by selecting and installing a curated set of five starter skills, then receiving quick next-step guidance. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The installer can install multiple preselected skills in one session. <br>
Mitigation: Review the selected skill list before confirming installation, or install the five skills individually when stronger supply-chain control is required. <br>
