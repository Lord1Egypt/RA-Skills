## Use Case: <br>
Developers and engineers use this skill as a quick reference for creating, configuring, extending, and testing Shopyo applications. It is most useful when an agent needs to explain Shopyo conventions or propose commands, modules, routes, models, templates, and local development steps. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: A Shopyo project created from these notes may retain the seeded local admin account. <br>
Mitigation: Change or remove the default admin credentials before exposing the application to a network or deploying it. <br>
Risk: Development commands such as debug server startup and database reset are intended for local use. <br>
Mitigation: Disable debug development settings for deployment and run reset-style database commands only in the intended local project with backups. <br>
