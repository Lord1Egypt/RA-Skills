## 升级记录 v2.0 — 飞影数字人融合

### 新增能力

**4. 数字人驱动（融合飞影数字人）**

原版视频只有静态照片开篇/结尾，现在升级为动态数字人讲解。

**使用方式：**
```
# 生成数字人脚本
video-craft> 数字人 脚本 第02节视频

# 克隆形象（首次）
video-craft> 克隆数字人 /path/photo.jpg 名称=智美人

# 生成数字人视频
video-craft> 生成 数字人视频 \
    --脚本 02-script.md \
    --形象 智美人 \
    --配音 XiaoXiao \
    --时长 3分钟
```

**数字人替代方案：**
| 位置 | 原方案 | 升级后 |
|------|--------|-------|
| 开篇 | 白西装静态照片 | 白西装数字人动态开场 |
| 内容 | 纯幻灯片 | 幻灯片+数字人画外音 |
| 结尾 | 黑版衣静态照片 | 黑版衣数字人总结 |

**注意：** 飞影数字人需要从Coze商店安装，本技能仅提供融合编排框架。

### 5. 中文幻灯片生成（内置中文字体支持）

所有幻灯片自动使用中文渲染（Noto Sans CJK / 文泉驿字体），
无论服务器环境是否含中文字体。

**脚本位置：**
```
skills/video-craft-pro/scripts/zh-slides.py
```

**使用方式：**
```python
from skills.video-craft-pro.scripts.zh_slides import make_slides

slides = [
    ('title', '标题', '副标题'),
    ('bullet', '要点', '第一行\n第二行\n第三行'),
    ('code', '代码', '# 注释\nprint("hello")'),
]
make_slides('/输出目录', slides)
```

**字体自动检测：**
- Noto Sans CJK Bold → 标题
- Noto Sans CJK Regular → 正文
- WQY ZenHei / MicroHei → 降级方案
- 全部不存在？用Pillow默认（罕见情况）

## 为什么用 VideoCraft Pro？

- **全流程**：创意→脚本→配音稿→字幕，一条龙
- **平台适配**：抖音/快手/YouTube Shorts/B站/Reels 各有专属模板
- **钩子优化**：前3秒钩子、完播率优化、互动引导
- **字幕同步**：自动生成时间轴字幕，支持中英文

## 能力

### 1. 视频脚本生成 (Script)
- 从 `video-script-creator` 继承全部能力
- 平台专属脚本（抖音/快手/YouTube/B站）
- 分镜指导、运镜建议、BGM标注
- 前3秒钩子、完播率优化
- 系列视频规划、数据复盘

### 2. 配音脚本生成 (Voice Over)
- 从脚本自动提取配音稿
- 语气标注（热情/沉稳/幽默/专业）
- 语速标注（正常/快速/慢速）
- 断句标注（用于TTS/录音）

### 3. 字幕/标题生成 (Caption)
- 自动提取关键句作为字幕
- 标题建议（SEO优化）
- 标签/话题推荐（hashtag）
- 互动引导文案（CTA）
- 双语字幕支持

## 工作流程

```
用户需求 → 脚本生成 → 配音稿提取 → 字幕生成
                ↓
            平台适配
                ↓
       抖音/快手/YouTube/B站专属
```

## 使用方式

```
video-craft-pro> 写个抖音脚本 [主题]
  → 生成完整分镜脚本+配音稿+字幕

video-craft-pro> 这个视频配字幕 [粘贴脚本]
  → 自动提取字幕+标题+标签

video-craft-pro> 改B站风格 [脚本ID]
  → 一键切换平台风格
```

|--------|--------|
| video-script-creator | 分镜脚本、平台适配、钩子优化 |

| (caption 新创) | 字幕生成、标题优化、标签推荐 |
| **飞影数字人** | 数字人驱动、形象克隆、语音同步 |
