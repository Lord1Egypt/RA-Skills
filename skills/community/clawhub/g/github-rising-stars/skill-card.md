## Description: <br>

This skill is ready for commercial/non-commercial use. <br>

Developers, researchers, and technical scouts use this skill to find newly created or recently active GitHub repositories that are gaining stars quickly. It helps an agent run the bundled discovery script, interpret ranked repository results, and highlight candidates for further review. <br>
