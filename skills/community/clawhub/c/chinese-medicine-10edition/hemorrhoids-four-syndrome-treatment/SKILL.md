# Hemorrhoids Four Syndrome Treatment

## When to Use
- 便血鲜红、喷射状（风伤肠络）
- 肛门红肿热痛、滋水（湿热下注）
- 肿物紫暗硬结、剧痛（气滞血瘀）
- 痔核脱出难回、便血色淡（脾虚气陷）

## Execution Steps
1. **辨证分型**：
   - 风伤肠络：便血鲜红暴急，舌红苔黄 → 凉血地黄汤合槐花散；
   - 湿热下注：红肿热痛、便血量多 → 脏连丸加减；
   - 气滞血瘀：紫暗硬痛、便秘 → 止痛如神汤；
   - 脾虚气陷：脱出难回、神疲乏力 → 补中益气汤。
2. **把握病机**：外因（久坐、便秘）与内因（脾虚）共同作用，气血失调为本，瘀毒阻滞为标。
3. **治疗原则**：治本健脾益气养血，治标祛风、清热、活血。

> **注意**：仅适用于Ⅰ、Ⅱ期内痔或不宜手术者；嵌顿痔或严重感染需外科处理。
