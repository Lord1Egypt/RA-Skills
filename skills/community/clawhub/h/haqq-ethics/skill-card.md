## Description: <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and agent builders use this skill to add opt-in Islamic ethics, source-citation discipline, refusal guidance, and age-banded educational guidance to an agent. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may behave like a global, persistent operating policy for an agent rather than a narrowly scoped optional reference. <br>
Mitigation: Install it only for users who explicitly want Islamic ethics guidance, and scope activation to relevant religious, ethics, and educational interactions. <br>
Risk: Unscoped use could let the skill's religious instructions override ordinary agent behavior in unrelated workflows. <br>
Mitigation: Review deployment prompts and routing so the skill informs eligible responses without becoming a general-purpose default policy. <br>
