# EV 最大化持仓审查框架

## 核心原则

**每天持有一个仓位 = 隐性地以今天的价格重新买入。**

入场价是沉没成本。唯一重要的问题是："如果我现在手里是现金而非这个仓位，我会以当前价格买入吗？"

## 核心公式

```
日化预期收益 = (p_now - c_now) / (c_now × d_remaining)
```

其中：
- `p_now` = 更新后的概率估算（经过最新研究）
- `c_now` = 当前市场价格（持有/重新买入的成本）
- `d_remaining` = 距市场结算的天数

此指标将所有仓位归一化为统一单位：**每天每美元风险的预期回报**。

## 决策树（6 步）

对每个仓位，按顺序执行以下步骤：

### 第 1 步：更新概率（p_now）

- 进行 2-3 次网络搜索，获取该事件的最新新闻
- 从先前估算出发，根据新证据更新
- 记录推理过程（事件级思考）
- 指定置信度：高 / 中 / 低

### 第 2 步：检查 Edge 方向

```
edge = p_now - c_now
```

- 如果 edge > 0 → 仓位有正期望值，继续第 3 步
- 如果 edge ≤ 0 → **卖出** — edge 已反转，没有理由持有
- 如果 edge 为负但很小（0 到 -3%）→ 卖出前考虑摩擦成本

### 第 3 步：检查论题完整性

问自己：
- 自入场以来，核心论题是否发生了变化？
- 是否有新的风险未被定价？
- 结算标准是否变得模糊？

如果论题已崩塌 → **卖出**（无论 edge 如何）

### 第 4 步：比较日化预期收益

按 `daily_expected_return` 对所有仓位排名：

| 日化收益 | 评级 | 行动 |
|----------|------|------|
| > 0.50% | 优秀 | 坚定持有 |
| 0.10% – 0.50% | 良好 | 持有 |
| 0.02% – 0.10% | 边缘 | 仅在没有更好替代时持有 |
| < 0.02% | 资金低效 | 轮换到更好的机会 |
| 负值 | 亏损边际 | **卖出** |

### 第 5 步：Kelly 与集中度检查

对每个仓位：
```
f* = (p_now - c_now) / (1 - c_now)
f_q = f* / 4（四分之一 Kelly）
```

**组合级检查：**
- 总 Kelly 敞口 ≤ 总资金 100%
- 单一主题 ≤ 总资金 40%（如所有伊朗相关仓位）
- 单一市场 ≤ 总资金 25%
- 到期日分散：不要所有仓位在同一周到期

如果过度集中 → 减持最大仓位，即使 edge 很好

### 第 6 步：最终判决

将所有信号合并为三种行动之一：

| 判决 | 标准 |
|------|------|
| **持有 (HOLD)** | 正 edge、论题完整、Kelly 范围内、无更好替代 |
| **卖出 (SELL)** | 负 edge、或论题崩塌、或过度集中、或资金低效且有更好替代 |
| **轮换 (ROTATE)** | 仓位边际正收益，但存在特定的更好机会（指明替代市场） |

## 卖出信号清单

- [ ] Edge 已反转（p_now < c_now，对你持有的方向）
- [ ] 论题已崩塌（新信息推翻了原始推理）
- [ ] 存在更好的机会（更高的 daily_er 可用）
- [ ] 过度集中（单一市场 > 总资金 25%，或单一主题 > 40%）
- [ ] 超出 Kelly（仓位 > 当前总资金的 f_q）
- [ ] 接近零的 daily_er 且距到期日很远（资金锁定）

## 持有信号清单

- [ ] Edge 为正（p_now > c_now，对你持有的方向）
- [ ] 在 quarter-Kelly 范围内
- [ ] 没有更好的替代（daily_er 对比）
- [ ] 论题仍然完整
- [ ] 未过度集中

## 轮换摩擦成本

每次卖出+买入循环的成本约为 **2-4% 的价差/滑点**：
- 卖出：损失约 1-2% 的买卖价差
- 买入新仓位：损失约 1-2% 的买卖价差

**经验法则：** 仅在新仓位的 daily_er 比当前仓位的 daily_er 高出至少 0.05%/天时才轮换（以覆盖摩擦成本）。

## 日化预期收益解读表

| 日化收益 | 年化（近似） | 解读 |
|----------|-------------|------|
| > 0.50% | > 500% | 卓越 — 罕见，通常是短期市场 |
| 0.20% – 0.50% | 100-500% | 优秀 edge |
| 0.10% – 0.20% | 40-100% | 良好，稳健持有 |
| 0.05% – 0.10% | 20-40% | 不错，跑赢大多数替代 |
| 0.02% – 0.05% | 8-20% | 边缘 — 定期审查 |
| < 0.02% | < 8% | 资金低效 — 可能应该轮换 |

## 组合级检查

分析完单个仓位后，在组合层面验证：

1. **总 Kelly ≤ 100%** — 所有 f_q 值之和不应超过 100%
2. **主题集中度 ≤ 40%** — 按主题分组（伊朗、加密、政治等）
3. **到期日分散** — 仓位应有错开的到期日
4. **现金储备** — 至少保留 20-30% 用于新机会
5. **相关性** — 避免所有仓位在同一新闻事件上同方向波动
