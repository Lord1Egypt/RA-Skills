# 课件帮 AIPPT — 一句话生成专业 PPT

接入「[课件帮](https://kejian365.com)」AI 平台，全流程自动完成 PPT 生成：大纲 → 选模板 → 生成 → 交付链接，无需手动操作任何 API。

## 适用场景

- 商务汇报（季度总结、战略分析）
- 产品介绍（融资 Pitch、新品发布）
- 学术演示（研究报告、毕业答辩）
- 培训课件（员工培训、技能分享）

## 使用方式

直接对 AI 说：

> 「帮我做一个关于 **AI 发展趋势** 的 PPT」

AI 会自动完成以下步骤：

1. 生成结构化大纲并展示给你确认
2. 从 40+ 套模板中挑选最匹配的风格
3. 创建生成任务并每 30 秒自动汇报进度
4. 生成完成后给你发送在线预览链接

## 前提条件

需要课件帮开放平台的账号密钥（`auth-token`）。

- 注册地址：https://kejian365.com/oapi-portal

### 配置方式（任选其一）

**方式一：直接告诉 AI（最简单）**

在对话中直接发送你的密钥，AI 会在本次会话中记住它，无需重复提供。

**方式二：环境变量**

```
KEJIAN365_AUTH_TOKEN=your_token_here
```

**方式三：OpenClaw `.env` 文件**

在 `~/.openclaw/.env` 中添加一行：

```
KEJIAN365_AUTH_TOKEN=your_token_here
```
