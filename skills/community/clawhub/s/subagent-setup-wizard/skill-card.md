## Description: <br>
Subagent Setup Wizard guides OpenClaw users through subagent version checks, enablement, concurrency practices, resource monitoring, and troubleshooting. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to configure OpenClaw subagents, choose concurrency settings, monitor resource use, and troubleshoot common subagent setup problems. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: State-changing configuration, restart, or kill commands can affect the active agent environment, resource use, or API cost if run without review. <br>
Mitigation: Review commands before execution and apply them only when the user intends to change OpenClaw subagent settings or manage running subagents. <br>
