## Known Risks and Mitigations: <br>
Risk: Slidev examples may include executable or interactive code, including Monaco runner examples. <br>
Mitigation: Review code before running it and avoid executing untrusted examples. <br>
Risk: PlantUML diagrams can use an external server, which may expose private diagram content. <br>
Mitigation: Configure PlantUML to use a trusted server before processing private diagrams. <br>
Risk: Presenter remote control features can expose controls when used on untrusted networks. <br>
Mitigation: Use remote control only on trusted networks and require a strong password. <br>
Risk: Recording workflows may capture screen, camera, or presentation content unintentionally. <br>
Mitigation: Review visible screen and camera content before recording or sharing exports. <br>
