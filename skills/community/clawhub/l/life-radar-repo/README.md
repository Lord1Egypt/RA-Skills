# life-radar

A custom OpenClaw skill that turns multi-channel signals into an action-first daily brief.

## What it does

It summarizes your day into:

- **Must-do (today)**
- **Should-do (soon)**
- **Nice-to-have**
- **Suggested next actions**

## Files

- `SKILL.md`: skill definition and behavior rules

## Usage / 使用方式

### Chinese examples

- `给我每日行动清单`
- `扫一下我今天要处理的事`
- `帮我做优先级摘要`

### English examples

- `Give me a daily action digest`
- `What needs action today?`
- `Summarize my priorities for today`

## Notes

- This skill focuses on **actionability**, not raw data dump.
- It prefers fewer high-confidence items over noisy long lists.
