## Description: <br>
Turn a real photo into a custom anime-style avatar, Moments cover, or Twitter/X header. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[shuminliu026](https://clawhub.ai/user/shuminliu026) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users use this skill to transform uploaded portrait photos into anime-style social media assets, including avatars, Moments covers, and Twitter/X headers, through the mew.design image API. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill requires a photo and mew.design API key to call the mew.design image API. <br>
Mitigation: Install only if you are comfortable sending the photo and API key to mew.design; prefer a dedicated or revocable key, avoid sharing it in shared chats, and rotate it after use if concerned about chat history exposure. <br>
