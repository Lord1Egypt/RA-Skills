# 青少年特发性脊柱侧凸支具治疗指征与方案

## 触发条件
IF 患者为青少年特发性脊柱侧凸，且同时满足以下条件：
- Cobb角为25°～40°
- 骨骼未成熟（Risser征 < Ⅳ度；女孩未初潮或初潮后<2年）
- 侧凸具有柔软性（可部分矫正）
THEN 启动支具治疗。

## 执行流程

### 1. 支具选择与定制
- 根据患者身材量体定做支具。
- 常用类型包括Boston支具、Milwaukee支具等。
- 支具必须覆盖主弯节段，并采用三点压力系统以实现有效矫正。

### 2. 佩戴方案
- 每日佩戴16～23小时，仅允许在洗澡、体育课等必要短时间摘除。
- **女孩**：佩戴至月经初潮后满2年，且Risser征达到Ⅳ度。
- **男孩**：佩戴至Risser征达到Ⅴ度（髂骨骨骺完全融合）。

### 3. 随访与调整
- 每4～6个月复查站立位脊柱全长正侧位X线片。
- 根据患者生长速度及侧凸角度变化，及时调整或更换支具。

### 4. 终止标准
- 确认骨骼成熟：Risser征达Ⅳ～Ⅴ度、椎体环融合、髋臼Y形软骨闭合。
- 进入维持期后继续随访2～3年，确认Cobb角无进展方可停用支具。

### 5. 失败处理
- 若在规范支具治疗期间，Cobb角增加≥5°，视为控制无效。
- 立即发出“支具治疗失败”警报，并建议转诊至脊柱外科评估手术指征。

> **注意**：Cobb角<25°者仅需定期观察；Cobb角>45°或骨骼已成熟者不适用支具治疗；僵硬性侧凸对支具无效。
