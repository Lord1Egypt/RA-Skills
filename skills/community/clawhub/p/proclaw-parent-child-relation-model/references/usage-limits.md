## 适用边界

### 适用年龄范围

| 年龄段 | 适用程度 | 主要应用 |
|--------|----------|----------|
| 0-3岁 | 高度适用 | 安全依恋建立、基础信任、安全感 |
| 3-6岁 | 高度适用 | 自主性培养、探索支持、情绪管理 |
| 6-12岁 | 高度适用 | 能力培养、学习习惯、自我管理 |
| 12-18岁 | 中等适用 | 身份认同、分离个体化、边界协商 |
| 18岁以上 | 低度适用 | 回顾性理解、自我调整 |

**注意**：青春期（12-18岁）需要更多灵活性，框架作为参考而非严格指南。

### 适用家庭类型

**高度适用**：
- 普通发展性亲子关系问题
- 养育方式调整需求
- 发展性挑战应对
- 预防性培养

**中等适用**：
- 离婚/再婚家庭调整
- 多子女家庭平衡
- 跨文化家庭养育
- 特殊天赋儿童养育

**低度适用**：
- 严重心理创伤家庭
- 家庭暴力环境
- 成瘾家庭
- 严重精神疾病家庭

### 适用问题类型

**高度适用**：
- 孩子缺乏主动性
- 亲子沟通困难
- 学业动力不足
- 行为习惯问题
- 情绪管理挑战
- 亲子冲突频繁

**中等适用**：
- 兄弟姐妹冲突
- 青春期对抗
- 学习困难
- 社交挑战

**低度适用**：
- 严重心理疾病（需专业治疗）
- 创伤后应激
- 严重行为问题
- 家庭暴力

## 使用前提

### 必要条件

**父母自身状态**：
- 情绪相对稳定
- 有改变的意愿
- 能保持基本耐心
- 有时间投入

**家庭环境**：
- 基本安全的物理环境
- 基本的物质保障
- 无严重家庭暴力
- 父母双方或主要照顾者参与

**孩子状态**：
- 发展水平正常
- 无严重心理疾病
- 无自伤/伤他行为
- 基本的社会功能

### 使用假设

使用本体系时，假设：
1. 父母有基本的爱孩子意愿
2. 问题主要是养育方式需要调整
3. 孩子的发展潜力未被严重损害
4. 家庭有基本的改变资源

## 不适用情况

### 绝对不适用

**当存在以下情况时，本体系不适用，应立即转介专业帮助**：

| 情况 | 表现 | 处理 |
|------|------|------|
| **儿童虐待** | 身体伤害、忽视、性虐待 | 立即报警+转介儿童保护服务 |
| **家庭暴力** | 任何家庭成员间的身体暴力 | 立即报警+安全规划 |
| **严重抑郁** | 持续两周以上情绪低落、兴趣丧失、自伤意念 | 转介精神科+心理咨询 |
| **自杀风险** | 有自我伤害想法、计划或行为 | 立即转介危机干预 |
| **饮食障碍** | 厌食、暴食、严重体重变化 | 转介专业饮食障碍治疗 |
| **严重成瘾** | 网络、游戏、物质成瘾 | 转介成瘾专科 |
| **精神分裂** | 幻觉、妄想、现实检验丧失 | 转介精神科 |
| **严重创伤** | PTSD症状闪回、回避、过度警觉 | 转介创伤治疗专家 |

### 相对不适用

**当存在以下情况时，本体系效果有限，需要谨慎使用或转介**：

| 情况 | 说明 | 处理建议 |
|------|------|----------|
| **父母严重心理问题** | 父母自身未解决的创伤或疾病 | 先支持父母治疗，再使用本体系 |
| **严重依恋障碍** | 混乱型依恋、安全感严重缺失 | 需要专业依恋治疗 |
| **严重行为问题** | 对立违抗、行为障碍 | 需要行为治疗专家 |
| **学习障碍** | 特定学习障碍、注意力缺陷 | 需要专业评估+教育支持 |
| **神经发育问题** | 自闭症谱系、智力障碍 | 需要专业特教支持 |
| **重大丧失** | 丧亲、重大疾病、重大生活变故 | 需要哀伤辅导或危机干预 |

## 转介标准

### 需要转介的信号

**立即转介（24小时内）**：
```
□ 任何自杀意念、计划或行为
□ 任何自伤行为
□ 幻觉或妄想
□ 严重的解离症状
□ 家庭暴力正在进行
□ 儿童虐待证据
```

**紧急转介（1周内）**：
```
□ 持续的严重抑郁（超过2周）
□ 饮食障碍症状
□ 严重的焦虑影响日常功能
□ 物质滥用
□ 严重的对立行为（攻击、破坏）
```

**常规转介（1个月内）**：
```
□ 依恋问题（过度焦虑或回避）
□ 持续的社交困难
□ 学习障碍或注意力问题
□ 行为问题持续超过6个月
□ 情绪问题影响发展
□ 父母自身需要心理支持
```

### 转介流程

```
Step 1: 识别转介信号
   ↓
Step 2: 评估紧急程度
   ↓
Step 3: 与家庭沟通（获得同意）
   ↓
Step 4: 选择合适的专业资源
   ↓
Step 5: 提供转介信息并支持过渡
   ↓
Step 6: 必要时跟进协调
```

### 转介资源类型

| 资源类型 | 适用情况 | 获取途径 |
|----------|----------|----------|
| 精神科医生 | 诊断、药物治疗 | 医院精神科 |
| 儿童心理咨询师 | 心理治疗 | 心理咨询机构 |
| 家庭治疗师 | 家庭系统问题 | 心理机构 |
| 危机干预热线 | 自杀/自伤危机 | 全国心理援助热线 |
| 儿童保护服务 | 虐待/忽视 | 民政/公安 |
| 学习支持专家 | 学习障碍 | 学校/教育机构 |

## 使用注意事项

### 避免过度使用

**问题**：把一切都当成"问题"去诊断

**正确做法**：
- 正常发展波动不需要干预
- 孩子有自我调整能力
- 父母有时过度焦虑
- 给成长留出空间

**判断标准**：
```
是否需要干预：
□ 问题持续超过2周？
□ 问题影响日常功能？
□ 孩子有明显痛苦？
□ 父母明显困扰？
□ 回答"是"的数量 ≥ 3？
```

### 避免机械化使用

**问题**：把框架当成公式，机械套用

**正确做法**：
- 每个孩子都是独特的
- 框架是参考而非教条
- 关注孩子实际反应
- 灵活调整策略

### 避免忽视父母自身状态

**问题**：只关注孩子，不关注父母

**正确做法**：
- 父母状态影响养育能力
- 先照顾好自己
- 必要时寻求父母支持
- 父母也需要成长空间

### 避免完美主义

**问题**：追求完美父母/完美孩子

**正确做法**：
- 足够好的父母就够了
- 孩子需要真实而非完美的父母
- 错误是成长机会
- 允许自己和孩子犯错

## 免责声明

### 使用范围

本体系旨在提供养育指导和教育框架，不构成：
- 医疗诊断或治疗建议
- 心理诊断或治疗
- 法律建议
- 专业心理咨询替代

### 使用责任

使用本体系时：
- 父母对自身养育决策负责
- 本体系作为参考而非权威
- 严重问题需寻求专业帮助
- 父母需根据孩子实际情况判断

### 专业边界

**本体系不替代**：
- 精神科诊断和治疗
- 专业的心理咨询
- 医学治疗
- 特殊教育支持
- 儿童保护服务

## 使用自检清单

### 开始使用前

- [ ] 确认孩子年龄在适用范围内
- [ ] 确认问题类型属于适用范围
- [ ] 确认家庭环境基本安全
- [ ] 确认父母自身状态可支持改变
- [ ] 确认没有需要立即转介的情况

### 使用过程中

- [ ] 定期评估进展
- [ ] 关注孩子反馈
- [ ] 避免机械化套用
- [ ] 给改变足够时间
- [ ] 必要时调整策略

### 发现以下情况立即行动

- [ ] 孩子出现自伤/自杀信号
- [ ] 家庭暴力发生
- [ ] 孩子状态明显恶化
- [ ] 父母无法继续
- [ ] 三个月无任何改善
