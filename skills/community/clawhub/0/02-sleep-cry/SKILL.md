# Miki 睡眠哭声分析

**使用方式：** 输入宝宝作息规律+哭声描述 → Miki判断问题类型+给出改善方案

---

## 各月龄睡眠标准

| 月龄 | 总睡眠 | 夜间 | 白天小睡 | 最长连续夜觉 |
|------|--------|------|---------|-------------|
| 0-1月 | 16-18h | 8-9h | 不规律 | 3-4h |
| 1-2月 | 15-17h | 8-9h | 3-4次 | 4-5h |
| 2-3月 | 15-16h | 9-10h | 3次 | 5-6h |
| 3-4月 | 14-15h | 10-11h | 3次 | 6-7h |
| 4-6月 | 13-14h | 10-11h | 2-3次 | 6-8h |
| 6-9月 | 12-14h | 10-12h | 2次 | 8-10h |
| 9-12月 | 12-13h | 11-12h | 2次 | 10-12h |

## 问题类型诊断

### 类型一：日夜颠倒
**特征：** 夜间不睡，白天睡得香

**改善：**
1. 白天睡觉保持光线明亮、声音正常
2. 夜间保持黑暗安静
3. 夜间喂奶不开灯/只开小灯
4. 白天多趴多互动，减少白天睡眠总长
5. 固定夜间仪式（洗澡→抚触→喂奶→睡觉）

---

### 类型二：落地醒
**特征：** 抱着睡得好，放下就醒

**改善：**
1. 先放屁股再放头
2. 用睡袋保持体温再放
3. 放下后手掌轻压10秒
4. 不要立即离开，在旁边陪
5. 等睡沉了（约15-20分钟后）再放

---

### 类型三：夜醒频繁
**特征：** 每1-2小时醒一次

**改善步骤：**
1. 先排查：饿醒/做梦醒/不舒服
2. 饿醒：睡前喂饱（可尝试"梦中喂"）
3. 做梦醒：保持黑暗，轻拍安抚
4. 大运动期（翻身/爬/站）：白天多练习
5. 睡眠周期：保持黑暗，用白噪音

---

### 类型四：早醒（5点前醒）
**特征：** 5点前醒来不再睡

**改善：**
1. 固定起床时间（不管几点睡，7点叫醒）
2. 睡前延后15-30分钟
3. 早晨用睡袋或遮光帘
4. 早醒后不互动，保持低刺激

---

### 类型五：闹觉（困了但不睡）
**特征：** 困了却哭闹不肯睡

**改善：**
1. 建立睡眠信号：固定顺序
2. 识别睡眠信号：揉眼睛/打哈欠/眼神发直 → 立即哄睡
3. 引入睡眠联想物：白噪音、安抚巾
4. 嘘拍法 + 5S安抚法

## 哭声初步判断（家长描述）

| 哭声特点 | 可能的含义 |
|---------|-----------|
| 尖锐、突然 | 疼痛、不适 |
| 哼哼唧唧、起伏 | 困了、无聊 |
| 定时哭闹 | 肠绞痛（3个月内常见） |
| 摇头、蹭头 | 困了，想睡睡不着 |
| 尖叫、挣扎 | 过度疲劳、过度刺激 |

**重要：** AI只能辅助判断，不能替代医生诊断。
