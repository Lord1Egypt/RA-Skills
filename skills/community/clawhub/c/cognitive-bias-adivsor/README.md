# 认知偏差决策顾问 (Cognitive Bias Advisor)

一个覆盖**决策、学习、行动、沟通、影响、管理**六大人生领域的认知偏差应用 Skill。通过十问框架识别思维盲区，在每个领域提供偏差应对策略和可套用的话术模板。

## 功能

### 六大应用领域

| 领域 | 核心能力 | 覆盖偏差数 |
|------|---------|-----------|
| **1. 决策** | 十问框架自检 + 自适应模式 + 评估报告 | 10 种 |
| **2. 学习** | 打破达克效应/后见之明/知识诅咒，高效成长 | 5 种 |
| **3. 行动** | 克服现状偏误/双曲贴现/决策疲劳，即刻执行 | 5 种 |
| **4. 沟通** | 谈判/反馈/亲子/日常四场景话术 + 三步说服法 | 5 种 |
| **5. 影响** | 默认效应/社会证明/稀缺效应，设计选择环境 | 6 种 |
| **6. 管理** | 结构化面试/红队蓝队/止损触发器/匿行评估 | 6 种 |

### 决策框架

- **引导模式**：根据时间压力与决策类别灵活选择自检深度
  - **自适应精简模式**：按决策类别匹配 3-5 道核心题
  - **完整十问模式**：10 类经典认知偏差地毯式排查
  - **极速模式**：紧急情况（<2 小时）仅 3 道核心题

- **十问框架**：依次诊断 10 种经典认知偏差（确认偏误/认知失调/自利性偏误/幸存者偏误/聚焦错觉/禀赋效应/基本归因错误/从众效应/长期主义/激励偏差）
- **智能恢复**：会话中断时说"继续"即可从断点接续，无需重新开始
- **决策质量评估报告**：完成后输出偏差风险摘要和行动建议
- **FAQ 常见问题**：内置偏差概念、使用流程、报告解读、异常恢复等解答

## 安装

### 方式一：ClawHub（推荐）

```bash
clawhub skill install cognitive-bias-advisor
```

安装后即可在任意对话中触发，无需额外配置。

### 方式二：SkillHub.cn

1. 访问 [SkillHub Marketplace](https://skillhub.cn) 搜索「认知偏差决策顾问」
2. 点击安装，或通过 CLI 安装：

```bash
skillhub install cognitive-bias-advisor-pro
```

### 方式三：GitHub 手动安装

```bash
# 克隆仓库
git clone https://github.com/jacksu/cognitive-bias-advisor.git

# 将 SKILL.md 和 references/ 放入你的 Agent 的 skills 目录
```

或直接下载 [最新 Release](https://github.com/jacksu/cognitive-bias-advisor/releases) 的 ZIP 包解压即可。

### 验证安装

安装完成后，在 Agent 中输入以下任意一句测试是否触发：

> "帮我决策一件事"

若进入决策引导流程，说明安装成功。

## 使用方法

### 触发方式

在任意支持 Skill 的 AI Agent 中，直接描述你的决策问题即可自动激活。以下任一关键词都会触发：

- **决策**："决策""决定""选择""我要不要…""帮我决策""十问"
- **学习**："学习""怎么学""记不住""学不会""效率低""进步"
- **行动**："拖延""不想动""执行""行动力""开始做""坚持"
- **沟通**："说服""谈判""激励""沟通技巧""怎么推动"
- **影响**："影响别人""推动改变""塑造""引导""助推"
- **管理**："管理团队""领导""带人""招聘""绩效""考核"
- **偏差自查**："认知偏差""决策质量""判断偏差"
- **恢复指令**："继续""接着来""重来""重新开始"

**示例触发语：**

> "我想辞职创业，帮我分析一下这个决定"

> "最近学东西总是记不住，怎么提高学习效率？"

> "一直在拖延一个重要的项目，怎么开始动起来？"

> "我要去跟老板谈加薪，有什么谈判技巧？"

> "怎么让团队接受一个新的工作流程？"

### 使用流程

激活后，AI 会带你经历三个阶段的引导式自检：

**Phase 1 · 收集背景（2 轮对话）**
1. AI 先了解你的决策背景和初步倾向
2. 询问时间紧迫程度
3. 自动识别决策类别（职业/投资/人际/日常）
4. 推荐引导模式，你来确认

**Phase 2 · 逐题自检（3-10 题）**
- 每道题都是点击选项即可，无需长篇打字
- 一次只发一题，答完再下一题
- 三种模式可选：

| 模式 | 题目数 | 适用场景 |
|------|--------|---------|
| ⚡ 自适应精简 | 3-5 题 | 日常推荐，按决策类别匹配核心题 |
| 📚 完整十问 | 10 题 | 重大决策，地毯式排查 |
| ⏱️ 极速模式 | 3 题 | 时间 < 2 小时的紧急决策 |

**Phase 3 · 生成报告**
- 自动输出偏差风险评估
- 标注哪些维度有风险（⚠️）、哪些可控（✅）
- 给出 1-3 条具体可执行的行动建议

### 中断恢复

- 说"**继续**"或"**接着答题**" → 从断点接续，无需重答
- 说"**重来**" → 清空进度重新开始
- 说"**换个模式**"或"**分类不对**" → 中途切换，已答题目保留
- 卡住不想答了 → 连续不选选项会自动触发跳过机制

### 常见问题速查

| 问题 | 解决 |
|------|------|
| 答到一半不想答了 | 说"跳过"，或连续不选选项自动跳过 |
| 选错了类别/模式 | 说"换个模式"或"分类不对"，随时切换 |
| 某题不知道怎么答 | 选"✍️ 我想补充说明"，用一句话描述想法 |
| 只有 2 小时时间 | Phase 1 选"⏱️ 极速模式"，只答 3 道核心题 |
| 完整十问要多久 | 通常 5-8 分钟，点击选项即可无需打字 |
| 对某个偏差概念不懂 | 随时问"什么是确认偏误"之类的，直接获得解释 |

## 注意事项

- 本技能的目的是**辅助思考**，不是代替你做决定
- 对时间极度紧迫的决策（<2小时），可跳过部分问题
- 保持开放心态，诚实回答每个问题效果最佳
- 遇到中断或卡顿，输入"继续"即可从断点恢复；输入"重来"清空重新开始
- 对偏差概念不理解？随时问"什么是XXX"即可获得解释
