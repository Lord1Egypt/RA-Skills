## 化工生产任务清单

化工生产按子域组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：原料处理 → 化学反应 → 产品分离 → 产品精制

---

|----|----------|------|------|----------|
| T1-1 | 原料采购 | 采购化工原料，包括供应商选择、原料规格、采购合同 | 无（入口） | S→C→I |

## 依赖拓扑摘要

### 一、域内链路

**原料处理链路**: T1-1 → T1-2 → T1-3 → T1-4
**化学反应链路**: T2-1 → T2-2 → T2-3 → T2-4
**产品分离链路**: T3-1 → T3-2 → T3-3 → T3-4
**产品精制链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整化工生产链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T3-1 → T3-2 → T3-3 → T3-4 → T4-1 → T4-2 → T4-3 → T4-4

**反馈改进链路**: T4-2 → T2-4（质量检测反馈到反应优化）

**安全驱动链路**: T4-4 → T1-2（安全环保驱动原料检验）

更多组合根据具体任务动态推导。
