# pm-workbench 中文样例

这组样例用于帮助中文 PM 更快理解 `pm-workbench` 的实际输出风格。

英文 `examples/` 是完整样例库；这里覆盖 9 个核心 workflow 的中文入口，方便中文 PM 直接试用和对齐输出风格。

## 这组样例的重点

这些样例不是为了把英文模板逐字翻译成中文，而是展示中文 PM 在真实工作里更可能需要的输出方式：

- 先拆清问题，而不是直接写方案
- 只追问会改变判断的关键信息
- 明确说出当前建议和不做什么
- 输出能直接放进沟通、评审或对齐场景里的内容
