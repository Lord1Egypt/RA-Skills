# Human Review Notes

This package is intentionally text-only. It contains:

- `SKILL.md` reasoning instructions
- Markdown references under `references/`
- UI metadata under `agents/openai.yaml`

It does not contain scripts, binaries, installers, dependency manifests,
credential declarations, background services, network endpoints, or an automatic
state mechanism. The Claim Ledger is an in-session reasoning artifact that may
be shown as a copy/paste text block for the user to review manually.
