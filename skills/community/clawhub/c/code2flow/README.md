# Auto Architect & Diagrammer

![Auto Architect Banner](https://via.placeholder.com/800x200.png?text=Auto+Architect+%26+Diagrammer)

Welcome to **Auto Architect**, the specialized AI skill for visual software documentation.

## 🌟 Core Capabilities
- **Code-to-Diagram:** Automatically transforms complex code logic into clear Mermaid.js visualizations.
- **Architecture Insights:** Analyzes data flows and dependencies to provide structural recommendations.
- **Plug & Play Output:** Generates valid Markdown and Mermaid blocks that can be directly pasted into GitHub, Notion, or any markdown-supported platform.

## 🛠 System Directive
This skill operates strictly on static text analysis. Provide the agent with code snippets or directory structures, and it will act as a senior software architect to map out your system.
