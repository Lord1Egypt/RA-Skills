## C1-设计规范域范本

### C1-01 语言系统应用范本

**任务目标**：选择合适的语言风格：正式/非正式、严肃/活泼

**执行步骤**：

1. **分析文案类型**
   - 广告文案：活泼、有趣
   - 营销文案：直接、有说服力
   - 品牌文案：正式、有情感
   - 产品文案：专业、准确
   - 社交媒体文案：轻松、亲切

2. **选择语言风格**
   - 正式：商务文案、官方公告
   - 非正式：社交媒体、朋友交流
   - 严肃：法律文案、医疗文案
   - 活泼：年轻受众、娱乐产品
   - 专业：技术文案、行业报告
   - 通俗：大众产品、日常消费

3. **确定语言特点**
   - 词汇选择：专业词汇/通俗词汇
   - 句式选择：长句/短句
   - 语气选择：严肃/轻松

**输出示例**：

```markdown
## 语言系统应用方案

### 文案类型
- 类型：营销文案
- 用途：产品推广

### 语言风格
- 风格：非正式、活泼
- 选择理由：目标受众是年轻人，需要轻松有趣的语言

### 语言特点
- 词汇选择：通俗易懂，避免专业术语
- 句式选择：短句为主，节奏感强
- 语气选择：轻松、亲切、有活力

### 语言示例
- 标题：这款产品真的太好用了！
- 正文：你知道吗？这款产品可以让你的生活变得更简单...
```

### C1-02 风格系统应用范本

**任务目标**：应用文案风格：简洁/详细、理性/感性

**执行步骤**：

1. **分析文案用途**
   - 广告标语：简洁、有力
   - 产品介绍：详细、全面
   - 品牌故事：感性、有共鸣
   - 技术文案：理性、专业

2. **选择风格**
   - 简洁：广告标语、社交媒体
   - 详细：产品介绍、品牌故事
   - 理性：技术产品、专业服务
   - 感性：品牌故事、情感营销
   - 直接：促销广告、销售文案
   - 含蓄：品牌形象、高端产品

3. **确定风格特点**
   - 长度：简短/详细
   - 逻辑：理性/感性
   - 表达：直接/含蓄

**输出示例**：

```markdown
## 风格系统应用方案

### 文案用途
- 用途：品牌故事
- 目标：建立品牌形象

### 风格选择
- 风格：详细、感性
- 选择理由：品牌故事需要详细描述，感性表达更能引起共鸣

### 风格特点
- 长度：详细，约500字
- 逻辑：感性，以情感为主线
- 表达：含蓄，留有想象空间

### 风格示例
- 开头：在一个阳光明媚的早晨...
- 中间：我们相信，每一个细节都值得被珍惜...
- 结尾：这就是我们的故事，一个关于坚持和热爱的故事...
```

### C1-03 结构系统应用范本

**任务目标**：应用文案结构：标题/正文/口号/标语

**执行步骤**：

1. **分析文案类型**
   - 广告文案：标题+口号
   - 营销文案：标题+正文+行动号召
   - 品牌文案：标题+正文+口号
   - 产品文案：标题+正文+卖点

2. **设计标题**
   - 吸引注意力
   - 简短有力
   - 有吸引力

3. **设计正文**
   - 清晰有条理
   - 有说服力
   - 突出重点

4. **设计口号/标语**
   - 简短易记
   - 有特色
   - 有号召力

**输出示例**：

```markdown
## 结构系统应用方案

### 文案类型
- 类型：营销文案
- 用途：产品推广

### 结构设计
- 标题：吸引眼球的标题
- 正文：产品介绍+卖点+用户评价
- 行动号召：立即购买

### 标题设计
- 标题：这款产品真的太好用了！
- 特点：简短、有吸引力、引发好奇

### 正文设计
- 结构：问题→解决方案→优势→证明
- 内容：详细的产品介绍和卖点
- 特点：清晰、有说服力

### 行动号召设计
- 口号：立即体验，改变你的生活！
- 特点：直接、有号召力
```

### C1-04 情感系统应用范本

**任务目标**：应用情感色彩：情感诉求、语气语调

**执行步骤**：

1. **分析目标受众**
   - 年龄段：年轻人/中年人/老年人
   - 需求：功能需求/情感需求
   - 偏好：理性/感性

2. **选择情感类型**
   - 温馨：家庭产品、亲情营销
   - 激情：运动产品、年轻人营销
   - 信任：金融产品、专业服务
   - 时尚：潮流产品、年轻人营销
   - 怀旧：经典产品、复古营销
   - 创新：科技产品、创新服务

3. **确定情感表达**
   - 语气：亲切/严肃/活泼/专业
   - 语调：温暖/热情/冷静/自信
   - 情感：爱/激情/信任/时尚/怀旧/创新

**输出示例**：

```markdown
## 情感系统应用方案

### 目标受众
- 年龄段：25-35岁
- 需求：情感需求
- 偏好：感性

### 情感类型
- 情感：温馨
- 选择理由：产品是家庭用品，温馨的情感更能引起共鸣

### 情感表达
- 语气：亲切、温暖
- 语调：柔和、有爱
- 情感：家庭、亲情、温暖

### 情感示例
- 标题：给家人最好的关爱
- 正文：每一次使用，都是一次爱的表达...
- 口号：爱，从细节开始
```
