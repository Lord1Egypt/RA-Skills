# 细胞病理学样本采集与诊断流程

## 何时使用
- 患者存在可疑病变部位，且该部位可提供脱落细胞、自然分泌物/体液，或适合细针穿刺（FNA）。
- 需进行无创或微创的初步筛查（如宫颈癌筛查）。
- 不适用于无脱落细胞或无法通过穿刺获取细胞的深部组织病变。

## 执行步骤

1. **确定细胞来源类型**：
   - **直接采集**：使用刷子、刮片等工具从口腔、食管、鼻咽部、女性生殖道等黏膜表面获取脱落细胞。
   - **自然排出物**：收集痰液、胸腔积液、尿液等体液中的细胞成分。
   - **穿刺获取**：在超声或内镜引导下，对乳腺肿块、甲状腺结节或淋巴结进行细针穿刺抽吸（FNA）获取细胞。

2. **制备细胞涂片**：
   - 立即均匀涂布于载玻片上。
   - 进行固定（常用95%乙醇）以防止细胞自溶。

3. **染色处理**：
   - 根据样本类型选择合适染色方法（如巴氏染色用于脱落细胞，HE染色用于穿刺样本）。

4. **显微镜下评估**：
   - 观察细胞形态、核质比、染色质结构、核仁大小及分布。
   - 识别恶性征象：细胞异型性、核分裂象增多、核膜不规则、细胞排列紊乱等。

5. **结果判读与输出**：
   - 若为筛查用途（如宫颈细胞学）：报告“阴性”或“发现异常细胞，建议进一步检查”。
   - 若发现可疑恶性细胞：**必须发出警示**：“建议组织活检确诊”，因细胞学不能单独用于恶性肿瘤的确诊。

## 关键约束
- 本方法仅作为辅助诊断或筛查工具，**不可替代组织活检**。
- 阴性结果不能完全排除恶性病变，需结合临床综合判断。
- 确保具备基本细胞采集工具、染色条件及光学显微镜观察能力。
