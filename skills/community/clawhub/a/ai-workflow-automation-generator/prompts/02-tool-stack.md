# Prompt 2: AI Tool Stack Recommendation

**Tier:** Paid  
**Input needed:** Output from Prompt 1 (or your own list of automation opportunities)  
**Output:** Complete tool stack with pricing, implementation sequence, and tools to avoid

---

## The Prompt

```
You are an AI tools specialist helping a business owner choose the right AI automation stack.

My business context:
- Business type: [same as Prompt 1]
- Team size: [number]
- Tech comfort level: [Beginner / Intermediate / Advanced]
- Monthly budget for new tools: [$X/month or "under $X"]
- Existing tools I already pay for: [list: e.g., Mailchimp, QuickBooks, Shopify, Google Workspace, etc.]

My top automation opportunities (from Prompt 1 or my own list):
1. [Automation opportunity #1]
2. [Automation opportunity #2]
3. [Automation opportunity #3]
4. [Automation opportunity #4]
5. [Automation opportunity #5]

Please generate a complete AI Tool Stack Recommendation with:

1. RECOMMENDED TOOL FOR EACH AUTOMATION
   For each opportunity:
   - Primary tool recommendation (name, price, why it fits)
   - Alternative option (if primary is too expensive or complex)
   - Free option (if one exists that does 80% of the job)
   - Integration requirements: does it connect to my existing tools?
   - Setup complexity: 1 (paste an API key) to 5 (requires developer)

2. TOOL COMPARISON TABLE
   Create a table with columns: Tool | Use Case | Price | Ease (1-5) | Integrations | Free Tier?

3. TOOLS TO SKIP (AND WHY)
   - Name any popular tools that are overrated, overpriced, or wrong for my situation
   - One-line reason for each

4. TOTAL COST ANALYSIS
   - Monthly tool cost at minimum viable implementation
   - Monthly tool cost at full recommended stack
   - Which tools overlap with something I already pay for (potential cancellation)

5. IMPLEMENTATION SEQUENCE
   - What to set up first (highest ROI, lowest effort)
   - What to set up second
   - What to save for Month 2–3
   - Red flag: any tools that require other tools to work (dependency order)

6. HIDDEN COSTS TO BUDGET FOR
   - Any tools with usage-based pricing that could spike
   - API costs to anticipate
   - Training time for team members

Format the output as a decision document I can use to approve tool purchases and brief whoever sets them up.
```

---

## Tips for Best Results

- List every tool you already pay for — the AI may find overlaps that save you money
- Be honest about tech comfort level — a Complex setup that never gets implemented has zero ROI
- If you have a firm budget cap, state it upfront

---

## What to Do With the Output

Approve or adjust the tool stack, then move to Prompt 3 to rewrite your highest-priority workflow as an AI-integrated SOP.
