## 五、战斗规则速查

### 5.1 技能检定

| 难度 | 条件 | 成功条件 |
|------|------|---------|
| 常规 | 对手技能/属性<50，或一般情形 | 掷出 <= 技能/属性 |
| 困难 | 对手技能/属性>=50，或专业人士挑战 | 掷出 <= 技能/属性的一半 |
| 极难 | 对手技能/属性>=90，或人类可能性边界 | 掷出 <= 技能/属性的五分之一 |
| 大成功 | 掷出01 | 总是成功 |
| 大失败 | 掷出100；或技能<50时掷出96-100 | 总是失败 |

### 5.2 对抗检定

- 大成功 > 极难成功 > 困难成功 > 常规成功 > 失败/大失败
- 平手时，技能/属性更高者获胜
- 对抗检定不能孤注一掷

### 5.3 近战流程

1. **确定攻击次序**：按DEX从高到低
2. **突袭检定**：侦查/聆听/心理学，失败则突袭方自动成功或获得奖励骰
3. **行动选择**：攻击、闪避、反击、逃跑、释放法术
4. **技能检定**：
   - 反击：格斗 vs 格斗
   - 闪避：闪避 vs 格斗
5. **伤害计算**：
   - 极难成功(非反击)：贯穿武器取最大伤害+额外一次武器伤害；非贯穿武器取最大伤害
6. **寡不敌众**：已反击/闪避后，后续近战攻击获得1个奖励骰

### 5.4 射击规则

| 距离 | 难度 |
|------|------|
| 基础射程内 | 常规 |
| 远射程(2x基础) | 困难 |
| 超射程(4x基础) | 极难 |

**射击调整表**

| 调整项 | 效果 |
|--------|------|
| 瞄准一轮 | +1奖励骰 |
| 抵近射击(1/5DEX英尺) | +1奖励骰 |
| 大目标(体格4+) | +1奖励骰 |
| 寻找掩体 | +1惩罚骰 |
| 部分遮蔽 | +1惩罚骰 |
| 高速移动(MOV8+) | +1惩罚骰 |

**全自动射击**：射击技能/10(最小3)为每组弹幕子弹数；成功=一半命中，极难成功=全部命中(一半贯穿)

### 5.5 战技

- 比较发起方和目标的体格；差3或以上不能发动
- 发起方体格较小：每差1点1个惩罚骰
- 成功后可造成特殊效果（缴械、推倒、擒抱等）

## 十二、幕间成长

1. 为每个标记的技能掷D100，若 > 当前技能或掷出95，技能增加1D10
2. 擦除技能标记
3. 技能提高到90%或更高：增加2D6 SAN
4. 检查信用评级并回顾经济状况
5. 尝试恢复SAN值
6. 修改调查员背景

## 十三、创建调查员快速流程

1. **掷属性**：STR/CON/DEX/APP/POW = 3D6x5；SIZ/INT/EDU = (2D6+6)x5
2. **年龄调整**：根据年龄段调整属性和EDU增强
3. **算衍生属性**：HP = (CON+SIZ)/10；MP = POW/5；SAN = POW
4. **算伤害加值和体格**：查STR+SIZ表
5. **算移动速度**：根据DEX/STR与SIZ比较
6. **选职业**：确定本职技能和CR范围
7. **分配技能点**：本职技能点(根据职业公式) + 兴趣技能点(INTx2)
8. **写背景**：形象描述、思想/信念、重要之人、意义非凡之地、宝贵之物、特质
9. **确定资产**：根据CR查现金/资产表
10. **装备**：列出重要物品、武器
