# Osteoporosis TCM Etiology and Pathogenesis

## When to Use
当接诊对象为骨质疏松症患者，且出现以下表现时应用本技能：
- 腰膝酸软、驼背、易发生骨折
- 舌象淡、脉象虚弱
- 需结合中医四诊（尤其舌脉）进行辨证

**注意**：本技能不适用于非骨代谢性疾病或急性外伤性骨折。

## Execution Steps

1. **确认核心病机**：
   - 肾为先天之本，主骨生髓；
   - 肾虚精亏 → 骨髓化源不足 → 骨枯髓减 → 骨失所养 → 发为骨质疏松。

2. **识别四大病因**：
   - **禀赋不足**：先天肾气素虚，骨髓生化乏源；
   - **劳倦内伤**：正气虚弱，卫外不固，外邪乘虚致气血痹阻；
   - **肾精亏虚**：随“天癸”竭绝，或阴损及阳、阳损及阴，致髓减骨疏；
   - **药食不当**：饮食偏嗜（如过咸）、长期使用糖皮质激素/免疫抑制剂、嗜烟酗酒，致肾气不衡、精失所藏。

3. **明确脏腑关联**：
   - 病变在骨，根本在肾；
   - 与肝（藏血、主筋）、脾（运化水谷精微以充养肾精）密切相关。

4. **判断虚实夹杂**：
   - 虚证：脏腑气血阴阳亏虚（尤以肾精亏虚为主）；
   - 实证：多见血瘀、气滞；
   - 总体病机概括为“骨虚髓空，失于荣养”。

5. **指导辨证方向**：
   - 以补肾法为核心；
   - 根据兼证兼顾肝肾同调、脾肾同治。

## Output
提供骨质疏松症的中医病机分析结论，用于后续辨证分型（如肾阳虚、肾阴虚、脾肾两虚、瘀血阻络等）及治疗方案制定。
