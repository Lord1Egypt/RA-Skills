# Red Alarm - 小红书 AI 内容审查助手

基于小红书 AI 治理主张（2026-04-27）设计的 Agent Skill，用于对笔记/文案进行结构化合规审查，输出可解释的风险结论与可执行修改建议。

## 功能亮点

- 五维度审查：AI 标识、真实性、侵权、内容质量、账号行为
- 正向优先：先判断鼓励创作方向，再进行违规排查
- 结果可解释：输出层级推理过程、风险分数、风险等级、处置建议
- 对创作者友好：不确定情况默认标注风险点，优先给出修改建议

## 适用场景

- 发布前自检：判断内容是否可能限流、删除、封号
- 运营质检：批量审查账号内容，统一合规标准
- 团队审核：形成标准化审核报告，便于复盘与协作

## 技能触发关键词（示例）

- 帮我审查这段内容
- 检查这个帖子是否违规
- 这段内容能发吗
- 会不会被限流
- 小红书内容审查
- AI 内容检测

## 输入与输出

### 输入

- 文本内容（标题、正文、标签）
- 图片/视频的文字描述（可选）
- 账号行为信息（可选）

### 输出

- 内容审查报告（Markdown）
- 风险等级（安全/低风险/中风险/高风险/违法）
- 风险分数（0-100）
- 违规项与风险点
- 具体修改建议与合规指引

## 快速开始

1. 克隆仓库：

   ```bash
   git clone https://github.com/<your-account>/red-alarm.git
   cd red-alarm
   ```

2. 在支持 Skill 的 Agent 环境中引入 `SKILL.md`。
3. 发送待审查内容，触发关键词即可执行审查流程。
