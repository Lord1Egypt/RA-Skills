## Description: <br>
AI数据可视化大师 helps users choose charts, design dashboards and infographics, and generate or refine visualization code for Tableau, Power BI, ECharts, Matplotlib, Plotly, and D3.js. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Data analysts, BI engineers, product, operations, marketing, research, and management users use this skill to select effective visualizations, plan dashboards, improve chart narratives, and request chart code or BI-tool guidance. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated chart code, formulas, or dashboard recommendations may be incorrect or unsuitable for real data. <br>
Mitigation: Review generated code and visualization guidance before applying it to production data, reports, or dashboards. <br>
