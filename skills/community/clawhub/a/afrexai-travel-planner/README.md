# Travel Planner Mastery ⚡

The complete travel planning system — from trip research to packing lists, budgets, itineraries, and 20 travel hacks. Built by AfrexAI.

## What This Does

Your AI travel agent with:
- **Trip Brief Template** — structured YAML for any trip type
- **Budget Benchmarks** — daily costs by region and travel style
- **Itinerary Builder** — day-by-day planner with pacing rules and the 3-2-1 rule
- **Packing System** — 5-4-3-2-1 clothing formula + climate-specific lists
- **Flight & Transport Strategy** — booking timing, comparison frameworks
- **Specialized Guides** — family, solo, business, road trips
- **20 Travel Hacks** — ranked by money saved and effort required
