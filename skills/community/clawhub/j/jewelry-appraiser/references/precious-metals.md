# 贵金属鉴定与选购指南

## 一、黄金 (Gold)

### 纯度标准
| 标识 | 含金量 | 俗称 | 特点 |
|------|--------|------|------|
| 足金999 | 99.9% | 千足金 | 最纯，质地软 |
| 足金990 | 99.0% | 足金 | 国标最低足金标准 |
| 22K/916 | 91.6% | 22K金 | 硬度适中，可做镶嵌 |
| 18K/750 | 75.0% | 18K金/Au750 | 首饰最常用 |
| 14K/585 | 58.5% | 14K金 | 欧美流行 |
| 9K/375 | 37.5% | 9K金 | 英国常见 |

### 2025-2026 金价走势
- 国际金价：2025年突破 $3,000/oz，2026年站稳 $3,200+
- 国内金价：突破 ¥1,000/克（2025年6月），延续强势
- 品牌溢价：周大福/周生生/老凤祥等品牌金饰工费 30-100 元/克

### 黄金投资方式
| 方式 | 优点 | 缺点 | 适合人群 |
|------|------|------|----------|
| 实物金条 | 看得见摸得着 | 保管成本、变现折价 | 长期持有者 |
| 金饰 | 可佩戴 | 工费高(10-30%)、回购折价 | 消费+保值兼顾 |
| 纸黄金 | 交易便捷、门槛低 | 无实物 | 短期交易 |
| 黄金ETF | 流动性好 | 管理费 | 普通投资者 |
| 积存金 | 定投攒金 | 手续费 | 长期定投者 |

### 金饰选购建议
- **自己买金条打金饰**：比品牌便宜 20-40%
- **3D/5D 硬金**：同重量体积大但工费高，按"一口价"卖不划算
- **古法金**：哑光质感，工费最贵（60-150元/克），但颜值高
- **5G 黄金**：高硬度可镶嵌宝石，适合钻石/彩宝镶嵌

## 二、铂金 (Platinum)

### 纯度标准
| 标识 | 含铂量 | 说明 |
|------|--------|------|
| Pt999 | 99.9% | 千足铂 |
| Pt990 | 99.0% | 足铂 |
| Pt950 | 95.0% | 珠宝最常用 |
| Pt900 | 90.0% | 日本常用标准 |

### 铂金 vs 18K 白金对比
| 对比维度 | 铂金 Pt950 | 18K 白金(Au750) |
|----------|-----------|-----------------|
| 颜色 | 天然银白，永不变色 | 需镀铑，长期需补镀 |
| 硬度 | 较软，易刮花 | 较硬，镶嵌牢固 |
| 重量 | 重（同体积重约1.6倍） | 较轻 |
| 价格 | 低于 K 金（无金价支撑） | 随金价波动 |
| 保值 | 差（二级市场流通差） | 较好 |
| 适用 | 对戒、素金首饰 | 钻戒镶嵌、彩宝镶嵌 |

### 选购建议
- **婚戒对戒**：铂金首选（素圈无镶嵌）
- **钻戒戒托**：18K 白金（镶嵌牢固度更好）

## 三、K 金（Karat Gold）

### 18K 金颜色
| 颜色 | 合金成分 | 特点 |
|------|---------|------|
| 18K 黄金 | 75%金+12.5%银+12.5%铜 | 经典金色 |
| 18K 白金 | 75%金+25%钯/镍 | 表面镀铑，银白色 |
| 18K 玫瑰金 | 75%金+22.25%铜+2.75%银 | 粉色调，近年最火 |
| 18K 黑金 | 75%金+镀层处理 | 前卫时尚 |

### 玫瑰金选购
- 铜含量越高颜色越红（偏粉/偏红/偏玫瑰）
- 不褪色，无需镀层维护
- 适合亚洲肤色，近年婚戒热门选择

- 标识：Pd950/Pd990

## 六、贵金属鉴定方法

### 黄金真假鉴定
| 方法 | 真金 | 假金 |
|------|------|------|
| 看钢印 | 清晰"足金999"/"Au750"等 | 模糊、缺失、异常 |
| 磁铁测试 | 不吸（黄金无磁性） | 含铁镍会吸 |
| 密度测试 | 19.32 g/cm³ | 其他金属密度不同 |
| 火烧法 | 变红冷却后恢复金色 | 变黑、变色 |
| 酸试法 | 不反应 | 起泡、变色 |
| XRF检测 | 精确测金含量 | 精确测金含量 |

**重要**：唯一可靠的方法是 XRF 光谱仪检测。密度测试、磁铁法只能排除明显假货，无法区分掺铱金等高仿假金。

### 高仿假金类型
| 类型 | 特征 | 检测方法 |
|------|------|----------|
| 掺铱金 | 铱密度接近金(22.56) | XRF/火试金 |
| 掺钨金 | 钨密度19.25接近金 | XRF表面正常，需钻心检测 |
| 镀金 | 表面镀金层 | 磨损后露出底金属 |
| 包金 | 金箔包裹 | XRF/锉刀测试 |
