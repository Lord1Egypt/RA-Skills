# 腰麻麻醉平面调节

在局麻药注入蛛网膜下腔后的5～10分钟关键时间窗内，执行以下操作以实现目标麻醉平面：

## 1. 穿刺间隙选择
- **L2–3间隙**：适用于需要较高麻醉平面的胸腹部手术，因重比重药液易向头端扩散。
- **L4–5间隙**：适用于鞍区麻醉（如肛门、会阴部手术），麻醉平面通常较低。

## 2. 患者体位调控
- 注药后立即转为仰卧位。
- 使用**重比重药液**时：
  - 若上界平面过低 → 将手术台调至**头低位**使平面上升；
  - 达到目标平面后 → **立即恢复水平位**，并严密监测呼吸与循环。
- **单侧下肢手术**：穿刺时取**患侧在下侧卧位**，注药后维持该体位5～10分钟，使药液偏向患侧。

## 3. 注药速度控制
- 标准速度：**每5秒注射1ml**。
- 速度越快 → 麻醉范围越广；越慢 → 范围越局限。

## 4. 关键限制
- 所有调节必须在**注药后5～10分钟内完成**，超时则药液已与神经结合，无法有效调整。
- 若平面过低无法满足手术 → **变更麻醉方式**；若平面过高 → **警惕呼吸抑制、低血压甚至全脊麻风险**，优先处理紧急情况而非继续调节。

> ⚠️ 注意：全脊髓麻醉、呼吸抑制等危急状况发生时，应立即启动急救流程，不得尝试平面调节。
