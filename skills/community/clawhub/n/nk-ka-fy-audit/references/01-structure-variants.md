# 返佣表结构图谱（参考）

## 变体总览

所有文件命名：`KA返佣YYYYMM.xlsx`，存放在 `/workspace/data/YYYYMM/` 目录下。

### 变体A：早期结构（202307唯一）
- Sheet数：3~4个
- 总表：`汇总`
- 计算表：`常规`、`中快6月`、`拓展`
- 列名：中文，表头第3行（拓展）/ 第1行（其他）
- 拓展公式：`ROUND(I×(J-K-L)×M,2)` 每通道独立四舍五入

### 变体B：中期结构（202308~202406）
- Sheet数：3个
- 总表：`汇总`
- 计算表：`常规`、`中快X月`（12种变体，用 LIKE '中快%'）、`拓展人志华`/`志华`
- 特殊表：`问题`/`异常`（部分月份）
- 列名：中文

**中快Sheet名变体（12种）：**
```
中快6月, 中快7月, 中快8月, 中快9月, 中快10月,
中快11月, 中快12月, 中快1月, 中快2月, 中快3月,
中快4月, 中快5月, 中快, 中快M+2
```

**志华/拓展人志华Sheet结构（202308+）：**
4通道×3字段：交易金额 / 收益率 / 渠道分润比例
- 支付宝(直连/间连)、微信(直连/间连)
- T列 = 分润汇总 = ∑(每通道分润)
- 公式：`分润 = ROUND(交易金额 × 收益率 × 渠道分润比例, 2)`
- ⚠️ 渠道分润比例实地采样：所有记录均为 0.6（60%），但不硬编码

### 变体C：中期进阶结构（202408~202509）
- 计算表：`常规`、`中快M+2`、`志华`（部分月份无）
- 特殊表：`问题`/`异常`（部分月份）
- 增加 level1/level2/level3_name 层级字段

### 变体D：后期结构（202510~202603）
- 列名：英文
- 表头：第1行
- 特殊表：部分月份有 `问题`/`异常`
- 202602 有 `sql` 表（不计入汇总）

**英文列名映射：**
```
zfb_trans_amt → 支付宝交易金额
zfb_commission_rate → 支付宝返佣比例
zfb_commission_amt → 支付宝返佣金额
wx_trans_amt → 微信交易金额
wx_commission_rate → 微信返佣比例
wx_commission_amt → 微信返佣金额
bill_amt → 返佣合计
months → 交易期日
rebate_type → 返佣方式
```

### 特殊表格类型

| Sheet名 | 出现月份 | 处理方式 |
|---------|---------|---------|
| `问题` | 多个月份 | 按备注规则判定 |
| `异常` | 多个月份 | 按备注规则判定 |
| `调差` | 202309等 | 差异调整行，计入汇总 |
| `sql` | 仅202602 | 手工核算表，**不计入汇总** |
