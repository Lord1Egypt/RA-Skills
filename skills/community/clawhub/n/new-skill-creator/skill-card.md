## Description: <br>
Create a new agent skill from scratch with guided, step-by-step assistance that generates a complete SKILL.md package with frontmatter, workflow, examples, and cross-platform metadata. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[fwwdn](https://clawhub.ai/user/fwwdn) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and skill authors use this skill to turn a skill idea into a structured agent skill package. It guides requirement gathering, scaffolds a skill directory, and helps draft SKILL.md, references, examples, workflow, and validation guidance. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can scaffold local files and directories when used to create a new skill. <br>
Mitigation: Choose the output path deliberately and review generated files before publishing. <br>
Risk: Broad requests about creating or scaffolding a skill may activate this skill. <br>
Mitigation: Use the documented When Not to Use boundaries and confirm intent when the user asks for adjacent tasks such as testing, SEO review, or debugging. <br>
