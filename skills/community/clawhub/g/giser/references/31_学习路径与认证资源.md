## 五、Python GIS 库版本对照（2026年6月）

| 库 | 功能 | 安装命令 | 官方文档 |
|------|------|---------|----------|
| GeoPandas | 矢量处理 | `pip install geopandas` | https://geopandas.org/ |
| Shapely | 几何运算 | `pip install shapely` | https://shapely.readthedocs.io/ |
| Rasterio | 栅格处理 | `pip install rasterio` | https://rasterio.readthedocs.io/ |
| Pyproj | 坐标转换 | `pip install pyproj` | https://pyproj4.github.io/pyproj/ |
| Fiona | 矢量读写 | `pip install fiona` | https://fiona.readthedocs.io/ |
| Folium | 交互式地图 | `pip install folium` | https://python-visualization.github.io/folium/ |
| xarray+rioxarray | 多维栅格 | `pip install xarray rioxarray` | https://docs.xarray.dev/ |
| GDAL | 万能读写 | `pip install gdal` 或 OSGeo4W | https://gdal.org/ |
| PDAL | 点云处理 | `pip install pdal` | https://pdal.io/ |
| laspy | LiDAR读写 | `pip install laspy` | https://laspy.readthedocs.io/ |
| PySAL | 空间统计 | `pip install pysal` | https://pysal.org/ |
| earthengine-api | GEE | `pip install earthengine-api` | https://developers.google.com/earth-engine |
