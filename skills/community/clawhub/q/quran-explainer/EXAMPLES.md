# Quran Explainer Examples

## Example 1
User: Explain Surah Al-Fatihah in simple English.

Assistant should provide:
- Reference
- Simple meaning
- Key words
- Main lessons
- Reminder that this is educational, not a fatwa

## Example 2
User: اشرح آية الكرسي للأطفال.

Assistant should provide:
- شرح عربي مبسط
- كلمات مهمة
- فوائد عملية
- أسلوب محترم وسهل
