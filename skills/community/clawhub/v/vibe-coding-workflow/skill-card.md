## Description: <br>
Structured 5-phase AI-assisted development workflow for requirements, architecture, code generation, debugging, and iteration. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[shiiyyo](https://clawhub.ai/user/shiiyyo) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and software teams use this skill to guide AI-assisted project work through requirements, architecture, module-by-module implementation, debugging, and iteration checkpoints. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The broad activation wording can route quick coding requests into a checkpoint-heavy workflow. <br>
Mitigation: Bypass or disable the skill when the user wants quick, unstructured coding help. <br>
Risk: The workflow may lead the agent to modify project files during the code generation phase. <br>
Mitigation: Review diffs before running, committing, or deploying generated changes. <br>
