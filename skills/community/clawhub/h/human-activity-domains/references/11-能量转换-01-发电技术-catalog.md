## 发电技术任务清单

发电技术按子域组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：发电规划 → 发电建设 → 发电运营 → 发电优化

---

|----|----------|------|------|----------|
| T1-1 | 需求分析 | 分析电力需求，包括负荷预测、电源规划、电网需求 | 无（入口） | S→C |
| T1-2 | 技术选型 | 选择发电技术，包括火电、水电、风电、光伏、核电 | T1-1 | C |

## 依赖拓扑摘要

### 一、域内链路

**发电规划链路**: T1-1 → T1-2 → T1-3 → T1-4
**发电建设链路**: T2-1 → T2-2 → T2-3 → T2-4
**发电运营链路**: T3-1 → T3-2 → T3-3 → T3-4
**发电优化链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整发电技术链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T3-1 → T3-2 → T3-3 → T3-4 → T4-1 → T4-2 → T4-3 → T4-4

**反馈改进链路**: T4-4 → T1-2（技术升级反馈到技术选型）

**安全驱动链路**: T3-4 → T3-2（安全管理驱动设备维护）

更多组合根据具体任务动态推导。
