## Description: <br>
Learn Graph helps users systematically learn a new field by collaboratively building a concept, use, and parent-child relationship map, then identifying high-value concepts, accessible entry points, and a purpose-fit learning path. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[li-evan](https://clawhub.ai/user/li-evan) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and employees use this skill when they want to enter a new field systematically, avoid fragmented learning, or decide how much learning is enough for a specific goal. The skill guides an agent to ask targeted questions, build a knowledge graph with the user, and turn that graph into a learning path. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may activate for broad learning-path requests where a user only wants a quick explanation. <br>
Mitigation: Use it when the user wants systematic learning, a concept map, or a learning path; otherwise answer directly without invoking the skill. <br>
Risk: Learning paths can become misleading if the agent assumes the user already knows prerequisite concepts. <br>
Mitigation: Ask the user to confirm prior knowledge before treating it as known, and mark uncertain prerequisites as questions in the graph. <br>
