# 纳洛酮在阿片中毒与术后呼吸抑制中的应用规范

## 何时使用
- 患者确认或疑似阿片类药物暴露，并出现呼吸抑制或意识障碍（如昏迷）
- 术后使用芬太尼、哌替啶等阿片类镇痛药后出现残留呼吸抑制
- 需通过纳洛酮激发试验辅助诊断阿片依赖（仅限特定临床判断下）

## 如何执行
1. **评估患者状态**：区分是急性阿片中毒、术后呼吸抑制，还是阿片依赖者，以决定给药策略。
2. **选择给药途径**：优先静脉给药（起效快，约2分钟）；避免口服（首过消除显著）。
3. **初始剂量与速度**：
   - **阿片急性中毒**：可快速给予负荷剂量，必要时每2–3分钟重复，直至呼吸改善。
   - **术后呼吸抑制**：应缓慢小剂量滴定，避免完全拮抗镇痛效果。
4. **持续监测与重复给药**：因纳洛酮作用时间（30–60分钟）常短于多数阿片药物，需密切观察呼吸频率、意识水平，必要时重复给药或启动持续静脉输注。
5. **警惕不良反应**：
   - 监测是否出现急性戒断症状（如焦虑、恶心、呕吐），尤其在依赖者中
   - 注意非心源性肺水肿（多发生于给药后4小时内）
   - 肝酶诱导剂（如酒精、巴比妥类）可能缩短其半衰期，需调整给药频次

> 提示：若用于鉴别阿片依赖，肌注后诱发典型戒断反应结合尿检可辅助诊断，但阴性结果不能排除依赖。

## 输出结果
根据临床场景选择纳洛酮剂量与给药速度，密切监测呼吸恢复与戒断反应，必要时重复给药。
