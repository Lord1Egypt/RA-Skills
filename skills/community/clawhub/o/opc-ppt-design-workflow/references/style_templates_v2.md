## A. 严谨学术顶刊风

### 设计范式来源
- **参考案例**：Nature/Science期刊图表、《IEEE会议论文模板》、高校学术汇报规范
- **范式特征**：信息优先、克制装饰、高对比色盲友好、平面化表达

### 色彩体系（精确代码）

| 角色 | 色值 | 用途 |
|------|------|------|
| 主底色 | #FFFFFF | 页面背景 |
| 辅底色 | #F6F6F6 | 卡片/图表底色 |
| 标题色 | #005CA9 | 一级标题（Nature蓝） |
| 正文色 | #222222 | 正文内容 |
| 强调色1 | #E64B35 | 核心数据（Nature红） |
| 强调色2 | #00A087 | 次级强调（Nature绿） |
| 辅助色 | #3C5488 | 图表/边框 |
| 图表色1 | #4DBBD5 | 数据系列1 |
| 图表色2 | #F39B7F | 数据系列2 |

**Nature Classic配色方案（色盲友好）**：
```
#E64B35（红）| #4DBBD5（青）| #00A087（绿）| #3C5488（蓝）| #F39B7F（橙）| #8491B4（紫灰）
```

### 构图公式

- **封面**：顶部20%为深蓝标题区，底部留白60%，居中大标题+副标题组合
- **目录**：左侧1/3为序号区（浅灰底），右侧2/3为内容条目，纵向排列
- **内容页-左文右图**：左侧60%为文字区（三段式标题+要点），右侧40%为图表位
- **内容页-全幅数据**：图表占70-80%，底部10%为图例和来源注释
- **章节过渡页**：纯色背景（主色），居中章节编号+标题，无装饰
- **结尾页**：中央致谢语，底部联系方式，无多余元素

### 字体搭配

| 元素 | 字体族 | 字号范围 | 字重 |
|------|--------|---------|------|
| 封面主标题 | Arial/微软雅黑 | 36-44pt | Bold |
| 页标题 | Arial/微软雅黑 | 28-32pt | Bold |
| 卡片标题 | Arial/微软雅黑 | 18-22pt | Bold |
| 正文 | Arial/思源黑体 | 16-20pt | Regular |
| 数据数字 | Roboto Mono | 24-36pt | Medium |
| 标签/注释 | Arial | 12-14pt | Regular |
| 来源注释 | Arial | 10-12pt | Regular |

### 装饰元素库

**可用装饰**：
- 细线分隔（1px，#CCCCCC）
- 色块卡片（圆角4px，低饱和色填充）
- 基础图标（线性，2px描边，与文字同色）
- 三线表（仅上线下粗）

**禁用装饰**：
- 渐变背景
- 阴影/立体效果
- 装饰性图案/插画
- 高饱和色块

### AI生图Prompt（精确版）

**封面页prompt**：
```
生成一张学术论文风格的PPT封面页。

视觉风格：线性扁平风格，几何色块和卡片结构承载文字。主色调为深蓝#005CA9搭配白色背景。标题字号小，正文字号更小，留白充足，强调数据墨水比。整体克制、专业、无装饰感。

页面顶部为深蓝色横条（高度占页面10%），用于展示论文标题。
页面中央位置展示主标题「[研究主题]」，粗体，24-28pt。
主标题下方展示副标题（研究机构/作者信息），常规字重，16-18pt。
页面底部左侧展示日期，右侧展示期刊/会议标识。

整体极简克制，信息层级清晰，无多余装饰元素。
```

**内容页prompt**：
```
生成一张学术信息图页面。

视觉风格：线性扁平风格，以白色为底，深蓝#005CA9作为主色调，强调色为红#E64B35和青#4DBBD5。图表占据页面60-70%空间，图例清晰，来源注释完整。数据墨水比最大化，禁止装饰性元素。

标题「[研究结论句而非描述句]」位于页面顶部左对齐，粗体，深蓝色，24pt。
页面主体区域展示数据图表（柱状图/折线图/饼图），图表占据左侧60%。
右侧放置2-3个关键要点卡片，每个卡片包含序号+简短结论。
页面底部右下角小字标注数据来源，10pt灰色。

禁止渐变、阴影、3D效果。
```

### 缓冲层变量

```css
--backdrop-solid: rgba(255, 255, 255, 1);
--backdrop-mid: rgba(246, 246, 246, 0.8);
--accent-primary: #005CA9;
--accent-warn: #E64B35;
--text-primary: #222222;
--text-secondary: #666666;
--border-light: #CCCCCC;
```

### 与其他风格的差异要点
> **核心辨识度**：极简留白+单一主色+数据图表主导，无任何装饰性元素，以信息传达效率为唯一衡量标准

## B. 轻松科普风

### 设计范式来源
- **参考案例**：李永乐老师科普视频、半佛仙人公众号、B站科普UP主、知识类短视频博主
- **范式特征**：亲和力强、图形化表达、色彩活泼但不杂乱、叙事感强

### 色彩体系（精确代码）

| 角色 | 色值 | 用途 |
|------|------|------|
| 主底色 | #F8F9FA | 页面背景（浅灰白） |
| 辅底色 | #FFFFFF | 卡片底色 |
| 标题色 | #2D3748 | 主标题（深灰） |
| 正文色 | #4A5568 | 正文内容 |
| 强调色1 | #FF6B35 | 重点标记（活力橙） |
| 强调色2 | #00D9FF | 知识点高亮（科技蓝） |
| 辅助色 | #48BB78 | 成功/正确（清新绿） |
| 图表色 | #805AD5 | 图表辅助（紫色） |

**B站科普UP主常用配色**：
```
主色：#00A1D6（哔哩哔哩蓝）| 辅色：#FB7299（粉色）
通用科普：#FF9500（橙色）| #00C8FF（亮蓝）| #7ED321（绿色）
```

### 构图公式

- **封面**：顶部40%为大标题区（配简化插图），底部60%为副标题+装饰元素
- **目录**：卡片式布局，2x2或3x2网格，每个卡片包含图标+章节名
- **内容页-左文右图**：左侧55%文字（要点+比喻），右侧45%插图/信息图
- **内容页-全幅信息图**：核心概念居中，环绕放射状关键词+图标
- **章节过渡页**：大字号章节标题+一行金句，下方配相关联插图
- **结尾页**：感谢语+二维码/账号信息，留白充足

### 字体搭配

| 元素 | 字体族 | 字号范围 | 字重 |
|------|--------|---------|------|
| 封面主标题 | 思源黑体/阿里巴巴普惠体 | 40-48pt | Bold |
| 页标题 | 思源黑体 | 24-28pt | Medium |
| 卡片标题 | 思源黑体 | 18-20pt | Medium |
| 正文 | 思源黑体 | 16-18pt | Regular |
| 金句/强调 | 思源黑体 | 20-24pt | Bold |
| 注释 | 思源黑体 | 12-14pt | Regular |

### 装饰元素库

**可用装饰**：
- 扁平插画（几何化简化人物/物品）
- 线性图标（与内容匹配，2px描边）
- 圆角卡片（8px圆角，浅色填充+细边框）
- 气泡/对话框元素
- 流程箭头（彩色）

**禁用装饰**：
- 过于复杂的手绘
- 荧光高饱和色
- 拟物化图标

### AI生图Prompt（精确版）

**封面页prompt**：
```
生成一张知识科普风格的PPT封面页。

视觉风格：扁平插画风格，几何化简化插画辅助说明。整体色调活泼但不杂乱，以浅灰白#F8F9FA为底，搭配活力橙#FF6B35和科技蓝#00D9FF作为强调色。留白充足，标题醒目但不压迫。

页面顶部20%区域展示简化扁平插画（与主题相关的人或物），几何化处理，无复杂细节。
页面中央偏上展示主标题「[科普主题]」，粗体，白色或深灰，36-40pt，文字紧凑。
主标题下方展示副标题「[吸引眼球的疑问句或陈述]」，20-24pt，橙色强调。
页面底部展示UP主/账号名，小号字体。

整体风格轻松有趣，吸引年轻受众。
```

**内容页prompt**：
```
生成一张科普信息图页面。

视觉风格：扁平插画风格，以浅灰白#F8F9FA为底，活力橙#FF6B35和科技蓝#00D9FF作为强调色。标题字号中等，正文小字号，整体清爽易理解。搭配几何化简化插画说明抽象概念。

标题「[核心概念/结论]」位于页面顶部左对齐，深灰#2D3748，24pt粗体。
左侧区域展示3-4个知识要点，每个要点包含：序号圆点（橙色）+ 关键词（粗体）+ 简短解释。
右侧区域放置与内容相关的扁平插画，几何化简化风格，避免写实。
页面底部可添加「思考题」或「下期预告」卡片。

整体信息密度适中，图文比例约6:4。
```


### 与其他风格的差异要点
> **核心辨识度**：扁平插画+活泼配色+对话感文字，像一位有趣的朋友在讲解，而非教科书式的严肃说教

## C. 欧式轻奢商务风

### 设计范式来源
- **参考案例**：麦肯锡/BCG咨询PPT、爱马仕/香奈儿品牌视觉、巴宝莉发布会、McKinsey Style Guide
- **范式特征**：克制用色、大量留白、衬线体搭配、材质感转译

### 色彩体系（精确代码）

| 角色 | 色值 | 用途 |
|------|------|------|
| 主底色 | #0A0A0A | 深黑背景（奢侈品牌用） |
| 辅底色 | #F5F3EF | 米白背景（商务场合） |
| 标题色 | #1A1A1A | 主标题 |
| 正文色 | #4A4A4A | 正文内容 |
| 强调色1 | #C9A962 | 金属金（奢侈品点睛） |
| 强调色2 | #8B7355 | 驼色/焦糖色 |
| 辅助色 | #E5E5E5 | 分割线/边框 |
| 品牌色 | #1A3A5C | 藏蓝（咨询公司） |

**咨询公司配色对比**：
| 公司 | 主色 | 辅色 | 强调色 |
|------|------|------|-------|
| 麦肯锡 | #005CA9（蓝） | #E5E5E5（灰） | #FFFFFF |
| BCG | #243745（墨绿） | #CCCCCC（灰） | #F39B7F（橙） |
| 贝恩 | #B40404（红） | #F5F3EF（米白） | #000000 |
| 爱马仕橙 | #E36C09 | #000000 | #FFFFFF |

### 构图公式

- **封面**：极简黑/米白底，左上角品牌LOGO位，主标题居中靠上（大字号），底部细线分隔+日期
- **目录**：左侧序号竖排（细线连接），右侧内容横向排列，大量留白
- **内容页-左文右图**：左侧65%为文字区（观点句标题+3点支撑），右侧35%为数据图表
- **内容页-全幅数据**：单一大图表占80%，标题句置于图表上方
- **章节过渡页**：半屏色块分割，章节编号（超大字号）+标题
- **结尾页**：感谢语居中，底部联系信息，无装饰

### 字体搭配

| 元素 | 字体族 | 字号范围 | 字重 |
|------|--------|---------|------|
| 封面主标题 | Bodoni MT/Didot/思源宋体 | 44-56pt | Regular |
| 页标题 | Avenir/Gill Sans/思源黑体 | 28-32pt | Light/Medium |
| 卡片标题 | Avenir/思源黑体 | 18-22pt | Medium |
| 正文 | Georgia/宋体/思源宋体 | 16-18pt | Regular |
| 数据数字 | DIN Pro/Roboto Mono | 28-40pt | Bold |
| 标签 | Avenir | 12-14pt | Light |

**衬线体搭配原则**：标题用衬线体传达优雅，正文用无衬线体保证可读性

### 装饰元素库

**可用装饰**：
- 极细分割线（0.5-1px）
- 品牌LOGO
- 哑光金色元素（点睛用，不超过5%面积）
- 皮革/金属质感纹理（微妙）
- 极简几何图形（线条构成）

**禁用装饰**：
- 鲜艳色彩
- 卡通/插画
- 大面积渐变
- 花哨图标

### AI生图Prompt（精确版）

**封面页prompt**：
```
生成一张高端奢侈品牌风格的PPT封面页。

视觉风格：极简主义设计，以纯黑#0A0A0A或米白#F5F3EF为底色，大面积留白。金属金#C9A962作为点睛色（仅用于极小细节）。整体传递克制、永恒、高端的品牌气质。

页面底部1/4处有一条极细分割线（#333333），下方为日期或品牌名，12pt。
```

**内容页prompt**：
```

视觉风格：单色设计（深蓝#1A3A5C或藏蓝为主色），留白充足，信息层级清晰。图表简洁专业，无多余装饰。整体冷静、专业、逻辑严谨。

标题「[Action Title：以结论句形式呈现]」位于页面顶部左对齐，深蓝粗体，26-28pt。
页面左侧65%区域展示文字内容：
- 核心观点（粗体，18pt）
- 3个支撑论据（编号+短句，16pt）
页面右侧35%区域展示数据图表（柱状图/折线图），简洁专业。
图表下方标注数据来源，10pt灰色。

整体风格与麦肯锡/BCG报告一致，强调逻辑而非视觉装饰。
```

### 缓冲层变量


### 与其他风格的差异要点
> **核心辨识度**：衬线体+大量留白+金属色点睛，传达"低调的奢华"而非张扬的炫富感

## D. 科技动感二次元风

### 色彩体系（精确代码）

|------|------|------|

| 辅底色 | #12121A | 卡片/面板底色 |
| 面板色 | #1A1A2E | 浮层/弹窗 |
| 标题色 | #FFFFFF | 主标题 |
| 正文色 | #E0E0EC | 正文内容 |
| 强调色1 | #00F0FF | 霓虹青（主信号色） |
| 强调色2 | #FF0080 | 霓虹粉（次信号色） |
| 辅助色 | #00FF41 | 终端绿（成功/数据） |
| 警告色 | #FFB800 | 琥珀黄（警告） |

### 构图公式

- **封面**：深色渐变背景（#0A0A0F→#1A1A2E），主标题居中（大字号+霓虹光晕），底部产品图
- **目录**：左侧竖排霓虹序号，右侧横向卡片布局，每张卡片有微光边框
- **内容页-左文右图**：左侧50%文字+右侧50%产品图/概念图，文字有霓虹高亮

- **章节过渡页**：深色底+霓虹渐变分割线，章节编号（超大字号）+标题
- **结尾页**：纯色深底+简洁感谢语

### 字体搭配

| 元素 | 字体族 | 字号范围 | 字重 |
|------|--------|---------|------|
| 封面主标题 | Orbitron/Exo 2/Rajdhani | 48-64pt | Bold |
| 页标题 | Orbitron/思源黑体 | 28-36pt | Bold |
| 卡片标题 | Rajdhani/思源黑体 | 20-24pt | Medium |
| 正文 | Exo 2/思源黑体 | 16-18pt | Regular |
| 数据数字 | Orbitron/Roboto Mono | 32-48pt | Bold |

**科技感字体推荐**：Orbitron（未来感）、Exo 2（几何无衬线）、Rajdhani（窄体HUD）

### 装饰元素库

**可用装饰**：
- 霓虹光晕效果（描边发光）
- 电路板纹理/网格线
- 扫描线/VHS纹理
- 霓虹边框/分割线
- 六边形/几何网格背景
- 粒子/数据流动画元素

**禁用装饰**：
- 暖色调渐变
- 手绘/卡通元素
- 柔和圆角
- 纸质纹理

### AI生图Prompt（精确版）

### 设计范式来源

### 构图公式

- **封面**：深色背景（左对齐粗体标题）+ 右侧工业图/装备图，方正无圆角
- **目录**：硬边网格布局，每个选项用粗线分隔，无装饰
- **内容页-左文右图**：左侧粗体标题+要点，右侧全幅图像/图表，硬边裁切
- **内容页-全幅数据**：仪表盘/KPI矩阵布局，每个数据块方正排列
- **章节过渡页**：色块分割（50%强调色+50%深色），大号章节编号
- **结尾页**：简洁感谢语，深色背景，无多余装饰

### 字体搭配

| 元素 | 字体族 | 字号范围 | 字重 |
|------|--------|---------|------|

| 卡片标题 | Roboto/思源黑体 | 20-24pt | Bold |
| 正文 | Roboto/思源黑体 | 16-18pt | Medium |
| 数据数字 | Orbitron/DIN Pro | 36-48pt | Bold |
| 标签/注释 | Roboto | 12-14pt | Medium |

**硬朗风格字体**：Teko（粗窄无衬线）、Russo One（几何粗体）、Impact（极粗）、Orbitron（科技感）

### 装饰元素库

**可用装饰**：
- 粗体几何形状（方形/三角形）
- 粗分割线（2-4px）
- 金属质感元素（拉丝金属/钢铁）
- 锯齿/斜角边缘
- 工业图标（齿轮/螺丝/子弹）
- 粗体色块对比
- 迷彩纹理（数字化处理）

**禁用装饰**：
- 圆角（除非功能性按钮）
- 柔和渐变
- 手绘/手写元素
- 花哨装饰
- 复杂图案
