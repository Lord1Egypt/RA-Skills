# 城市.Skills 🏙️

> 一站式旅行助手，专为第一次来中国的外国游客服务

## ✨ 功能特点

- 🌍 **多语言支持**：中文、English、Español、Français、Русский、العربية
- 🛂 **出入境指南**：签证政策、通关流程、住宿登记
- 🍜 **美食推荐**：本地特色美食（不辣优先）
- 🎫 **景点攻略**：安全主流景点 + 价格参考
- 📅 **行程规划**：3天精简安全行程
- 🆘 **紧急联系**：110/120/119 + 外交部领保

## 🚀 快速使用

只需输入任意中国城市名，即可获得完整攻略：

```
用户: Beijing
系统: 🏙️ Welcome to Beijing!
      This city is very safe, don't worry.
      [完整攻略输出...]
```

8. 交通与住宿建议
9. 3天精简安全行程
