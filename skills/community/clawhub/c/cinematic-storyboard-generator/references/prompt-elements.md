# 分镜提示词规范手册

## 一、七大基础要素（每个镜头必须包含）

| # | 要素 | 说明 | 示例 |
|---|------|------|------|
| 1 | 日夜景标注 | 第一句话写明 | "日景 · 清晨 / 夜 · 凌晨2点" |
| 2 | 主体 | 人物形态/装扮/表情/姿势 | "男主张启山背对镜头，深褐短发，灰色对襟长袍，金色云纹刺绣，拂尘横握" |
| 3 | 空间坐标 | 场景+位置关系 | "立于桥头最高处，桥下船只横列，前景水面，背景远山" |
| 4 | 镜头形式 | 景别+角度+运动 | "远景 · 低角仰拍 · 固定镜头" |
| 5 | 光影设计 | 光源方向+软硬+色温 | "侧逆光 · 金色时刻阳光从右侧45度打入 · 硬光 · 暖金色#D4A574" |
| 6 | 情绪落点 | 观众感受到什么 | "肃穆 · 蓄势待发 · 战意凝聚" |
| 7 | 一致性锚点 | 资产文件路径 | "参考：节点89.png + 定妆照_天海_正面.png" |

---

## 二、资产锚定原则（第一条铁律）

**人物描述必须对照定妆照，场景描述必须锚定具体场景图。**

提示词末尾附资产路径，形成硬锚定：
```
参考资产：[场景图路径] + [定妆照路径]
```

定妆照没看过就不写。写了就是错的。

---

## 三、光影逻辑（五条硬规则）

1. **单一光源为主**：只保留一个主光源，不写多光源
2. **方向具体化**："顶窗自然光从右后侧打入"，不写"自然光"
3. **接触阴影必须写**："投影落地面，方向与顶窗光源一致"，人物不悬浮
4. **逆光/侧光优先**：不写正面柔光，正面柔光=AI感=假
5. **同一空间同一光源**：前后景人物必须处于同一物理光源体系

---

## 四、质感统一原则

| 材质 | 错误写法 | 正确写法 |
|------|---------|---------|
| 皮肤 | "细腻白皙" "磨皮质感" | "真实皮肤纹理，无磨皮无柔光" |
| 服装 | "飘逸" "贴图感" | "有重力垂坠感，真实褶皱，非贴图" |
| 场景 | "背景虚化" | "前景地面反光 → 中景人物主体 → 远景建筑轮廓，三层可信纵深" |

**质感断层是AI图最大破绽：人物像抠图贴上去的。**

---

## 五、空间纵深逻辑

- **三层纵深**：近景地面反光 → 中景人物主体 → 远景场景轮廓
- **召唤物空间**：椭圆形光幕/门洞作为空间边界，不是VFX特效圈
- **投影可信**：人物站在地面上，投影方向与光源一致

---

## 六、景别规范

| 景别 | 用途 | 竖屏占比 |
|------|------|---------|
| 远景/全景 | 建立空间，交代环境 | 人物高度约1/5画面 |
| 中景 | 叙事主体，角色关系 | 人物高度约1/3画面 |
| 近景 | 聚焦表演，表情 | 人物胸部以上 |
| 特写 | 情绪放大，关键道具 | 单一部位（手/眼/物） |

---

## 七、镜头角度与情绪对应

| 角度 | 情绪效果 |
|------|---------|
| 低角仰拍 | 威严、压迫感、权力感 |
| 高角俯拍 | 渺小、劣势、压抑 |
| 平视 | 客观、平等、对峙 |
| 侧面 | 叙事感、客观观察 |
| 背面 | 悬念、引领观众入画 |

---

## 八、质量自检清单（每镜必查）

- [ ] 日夜景标注在第一句
- [ ] 人物描述与定妆照一致
- [ ] 场景描述有具体资产锚点
- [ ] 光源方向具体（不写"自然光"）
- [ ] 人物有投影落地面（不悬浮）
- [ ] 皮肤质感无磨皮无柔光
- [ ] 服装有重力垂坠感
- [ ] 三层纵深空间可信
- [ ] 景别与情绪对应正确
- [ ] 资产路径在末尾标注
