# 利福平抗菌机制与耐药性特征

## 何时使用
- 需评估利福平的抗菌谱或作用靶点
- 设计抗结核或抗麻风联合治疗方案
- 解释为何利福平不可单独长期使用
- 分析药物协同作用（如与异烟肼、链霉素联用）
- 判断不同药物浓度下的抑菌或杀菌效应

## 执行步骤

1. **确认目标微生物类型**：检查是否属于以下适用对象：
   - 结核分枝杆菌
   - 麻风杆菌
   - G⁺球菌（如金黄色葡萄球菌、脑膜炎球菌）
   - G⁻杆菌（如大肠埃希菌、变形杆菌、流感嗜血杆菌）
   - 沙眼衣原体（高浓度时）
   - 某些病毒（高浓度时）

2. **评估药物浓度水平**：
   - 若为低浓度 → 判定为抑菌作用
   - 若为高浓度 → 判定为杀菌作用

3. **应用抗菌机制知识**：
   - 利福平特异性结合细菌DNA依赖的RNA多聚酶β亚单位
   - 阻断mRNA合成，从而抑制蛋白质合成
   - 对人和动物细胞RNA多聚酶无影响，具备选择性毒性

4. **评估耐药风险**：
   - 单独使用极易因RNA多聚酶基因突变产生耐药性
   - 与其他抗菌药无交叉耐药性
   - 必须联合用药以延缓耐药发生

5. **设计或验证联合用药方案**：
   - 推荐与异烟肼、链霉素等联用
   - 利用其协同增效作用提升疗效

> 注意：本技能不适用于评估非适用微生物或非RNA多聚酶靶向药物。
