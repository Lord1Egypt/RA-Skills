# 文本润色系统 · 使用说明

## 功能

- ✅ 错别字自动纠正（形近字、语序、标点）
- ✅ 表达润色（啰嗦→简洁、口语→专业）
- ✅ 语义一致性检查（名称统一、前后不矛盾）
- ✅ 风格统一（段落长度、格式、标题层级）

## 安装

```bash
cp -r text-polish ~/.openclaw/skills/
```

## 使用

**直接发送文本给太初，或说：**
```
润色一下这段文字
帮我校对一下
错别字纠正
改一改这段表达
检查这段文字
```

## 输出示例

**输入：**
> 今天我去了那个图书馆，还书的时间是明天，我觉的这个图书馆的环境很好，就是有点热。

**润色后：**
> 今天我去了那家图书馆，还书截止日期是明天。整体环境不错，就是有些闷热。

**修改说明：**
- "那个" → "那家"（更具体）
- "明天" → "还书截止日期是明天"（明确期限）
- "觉的" → "觉得"（错别字）
- "有点热" → "有些闷热"（更准确的环境描写）

## 不做的事

- ❌ 不改变原意
- ❌ 不添加新信息
- ❌ 不删除实质性内容
- ❌ 不做风格大改（除非明确要求）

## 适用场景

- 正式文档润色
- 邮件/消息校对
- 文章改写
- 翻译后润色
- 会议记录整理
