# 模板库 — 让龙虾替你写文档、做PPT

跟龙虾说句话，它就帮你搞定一切——选模板、排版、填充、生成文件。

## 能做什么

| 你说的 | 龙虾做的 |
|--------|---------|
| "帮我写毕业论文" | 推荐64个论文模板 → 收集内容 → 一键生成 .docx |
| "做个答辩PPT" | 搜索186个PPT模板 → 排版填充 → 输出 .pptx |
| "来份简历" | 66个简历模板随选 → 填入经历 → 导出文件 |

| "写个开题报告" | 智能推荐 → 格式化 → 生成文档 |
