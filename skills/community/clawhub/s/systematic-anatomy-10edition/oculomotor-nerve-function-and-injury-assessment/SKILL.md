# 动眼神经功能与损伤评估

## 何时使用
- 患者突发单侧眼睑下垂、眼球偏向外下方
- 瞳孔扩大且对光反射消失
- 怀疑海绵窦、眶上裂或中脑病变累及动眼神经

## 执行步骤
1. **回顾动眼神经组成**：
   - **躯体运动纤维**：支配上直肌、下直肌、内直肌、下斜肌、上睑提肌
   - **副交感纤维**（E-W核）：支配瞳孔括约肌（对光反射）和睫状肌（调节反射）

2. **追踪路径关键点**：
   - 中脑脚间窝出脑 → 海绵窦外侧壁 → 眶上裂入眶 → 分上、下两支
   - 副交感纤维经下斜肌支 → 睫状神经节短根 → 节后纤维入眼球

3. **匹配损伤表现**：
   - **上睑下垂** → 上睑提肌麻痹
   - **眼球外斜视**（瞳孔斜向外下）→ 内直肌、上/下直肌、下斜肌瘫痪
   - **瞳孔散大、对光反射消失** → 副交感纤维受损

4. **输出临床警报**：
   - “若出现上睑下垂、外斜视、瞳孔散大、对光反射消失，提示动眼神经损伤”
   - 需紧急排除后交通动脉瘤压迫（尤其伴瞳孔受累时）

> 注意：此技能仅适用于动眼神经，不适用于其他脑神经或脊髓运动通路。
