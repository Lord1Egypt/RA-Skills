## ⚠️ 主要限制

### 1️⃣ 角色一致性（最大挑战）

**问题：** AI 每次生成的角色长相会有差异

**表现：**
- 第 1 张：明日香金发双马尾
- 第 2 张：头发颜色变浅/发型微调
- 第 3 张：眼睛形状变化
- 第 5 张：完全像另一个人

**原因：**
- 扩散模型的随机性
- 没有角色 LoRA 微调
- 提示词无法精确锁定特征

**可行方案：**
```
✅ 方案 A：固定种子（seed）
  - 同一 seed 生成相似角色
  - 但动作/表情会受限

✅ 方案 B：图生图（img2img）
  - 用第 1 张作为参考图
  - 保持 70-80% 相似度

✅ 方案 C：精细提示词
  - 详细描述角色特征
  - 每次使用相同的描述词
```

---

### 2️⃣ 版权角色限制

**EVA 角色（明日香、碇真嗣）是版权内容**

**可能触发：**
- 🟡 内容过滤（生成失败）
- 🟡 风格限制（不允许精确模仿）
- 🟡 商业用途限制

**实测经验：**
- 生成"金发双马尾傲娇少女" → ✅ 通常可以
- 生成"惣流·明日香·兰格雷" → 🟡 可能触发过滤
- 生成"碇真嗣" → 🟡 可能触发过滤

**规避策略：**
```
❌ 避免：精确角色名（"惣流·明日香·兰格雷"）
✅ 使用：特征描述（"金发双马尾傲娇少女，红色战斗服"）

❌ 避免："EVA 新世纪福音战士"
✅ 使用："机甲战斗场景，巨大机器人"
```

---

### 3️⃣ 剧情连贯性

**AI 不理解剧情逻辑**

**问题表现：**
- 第 1 张：两人在学校
- 第 2 张：突然在机甲里（没有过渡）
- 第 3 张：战斗场景（敌人从哪来？）
- 第 4 张：两人受伤（怎么受伤的？）

**解决方案：**
```
✅ 方案 A：手动分镜规划
  - 先写完整剧本
  - 每格画面详细描述
  - 人工审核连贯性

✅ 方案 B：逐格生成 + 人工筛选
  - 每格生成 3-5 个版本
  - 选最符合剧情的
  - 丢弃不连贯的

✅ 方案 C：混合工作流
  - AI 生成画面
  - 人工添加对话/剧情
  - 用漫画编辑软件合成
```

---

### 4️⃣ 对话气泡与文字

**AI 文字渲染能力有限**

**问题：**
- 文字模糊/扭曲
- 中文字符错误
- 气泡位置不合理

**解决方案：**
```
✅ 方案 A：AI 生成画面 + 后期加字
  - 用通义万相生成无字画面
  - 用 Photoshop/Clip Studio 加对话
  - 质量最好，推荐

✅ 方案 B：简单文字尝试
  - 提示词指定"对话气泡：你好"
  - 接受一定失败率
  - 快速迭代

✅ 方案 C：纯画面漫画
  - 无对话，靠画面叙事
  - 类似无字漫画
  - 避开文字问题
```
