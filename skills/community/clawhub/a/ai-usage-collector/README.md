# ai-usage-collector

> 从微信群消息（文字/截图）中批量提取同事 AI 工具使用信息，输出 CSV 表格，直接粘贴到 Excel / Teams。

## 功能介绍

每周收集同事在群里分享的 AI 使用场景，自动提取结构化信息，省去手工整理的时间。

**支持提取的 AI 工具**：WorkBuddy、龙虾（OpenClaw）、ChatGPT、 Claude、小火箭等任何 AI 工具。

## 提取字段

| 字段 | 说明 |
|------|------|
| 姓名 | 同事姓名，识别不到留空 |
| AI 工具 | WorkBuddy / 龙虾 / 其他，识别不到留空 |
| 业务场景 | 在什么场景下使用 |
| 价值/操作点 | 解决了什么问题、有什么效果 |
| 备注 | 其他有价值的信息 |

## 使用方法

### 方式一：直接发给 AI（WorkBuddy / 龙虾）

```
帮我整理群消息：
- 张三：今天用 WorkBuddy 生成了周报，省了半小时！
- 李四：用龙虾做了竞品对比
- 王五：用小火箭写 PRD，快多了
```

### 方式二：发送截图

将微信群聊天截图发送给 AI，触发词：
- `帮我整理群消息`
- `提取AI使用情况`
- `整理AI工具使用记录`

### 输出格式（CSV，直接粘贴到表格）

```
张三,WorkBuddy,周报生成,节省30分钟/天,
李四,龙虾,竞品对比,,
王五,小火箭,写PRD,比之前快多了,
```

> 空列是需要人工补充的字段，粘贴到 Excel / Teams 后自动分列。

## 安装（龙虾 / OpenClaw）

将 `SKILL.md` 放入工作区的 `.workbuddy/skills/` 目录，或通过技能中心安装。

## 适用场景

- 团队 AI 推广负责人，每周收集团队 AI 使用情况
- HR / 行政，统计公司 AI 工具 adoption 情况
- 产品经理，收集用户反馈和 AI 使用案例
