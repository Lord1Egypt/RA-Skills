# WT8麦语言示例：双均线策略

## 策略逻辑

双均线交叉策略是最经典的量化交易策略之一：

| 条件 | 信号 | 含义 |
|------|------|------|
| 短期上穿长期（金叉） | BK | 买开仓 |
| 短期下穿长期（死叉） | SK | 卖开仓 |
| 金叉（有空仓） | SP | 卖平空仓 |
| 死叉（有多仓） | BP | 买平多仓 |

---

## 完整代码

```mql
// ============================================
// 双均线交叉策略
// ============================================

// 步骤1：定义参数（赋值用 :=）
N1 := 10;   // 短期均线周期
N2 := 20;   // 长期均线周期

// 步骤2：计算均线（赋值用 :=）
MA_SHORT := MA(C, N1);   // 短期均线
MA_LONG := MA(C, N2);   // 长期均线

// 步骤3：判断交叉条件
金叉 := CROSSUP(MA_SHORT, MA_LONG);   // 短上穿长（金叉）
死叉 := CROSSDOWN(MA_SHORT, MA_LONG); // 短下穿长（死叉）

// 步骤4：开仓信号（条件,信号;）
金叉, BK;      // 金叉买开
死叉, SK;      // 死叉卖开

// 步骤5：平仓信号
金叉, SP;      // 金叉平空仓
死叉, BP;      // 死叉平多仓

// 步骤6：信号过滤（必须写！）
AUTOFILTER;
```

---

## 代码说明

### 步骤1：参数定义
```mql
N1 := 10;   // 短期均线周期
N2 := 20;   // 长期均线周期
```
- 使用 `:=` 赋值
- 参数可后续优化调整

### 步骤2：计算均线
```mql
MA_SHORT := MA(C, N1);
MA_LONG := MA(C, N2);
```
- `MA(X, N)` = N周期简单移动平均
- `C` = 收盘价（必须大写）

### 步骤3：交叉条件
```mql
金叉 := CROSSUP(MA_SHORT, MA_LONG);    // 上穿
死叉 := CROSSDOWN(MA_SHORT, MA_LONG); // 下穿
```
- `CROSSUP(X, Y)` = X上穿Y
- `CROSSDOWN(X, Y)` = X下穿Y

### 步骤4-5：交易信号
```mql
金叉, BK;   // 金叉买开
死叉, SK;   // 死叉卖开
金叉, SP;   // 金叉平空
死叉, BP;   // 死叉平多
```
- 格式：`条件, 信号;`
- `AUTOFILTER` 保证一开一平

### 步骤6：信号过滤
```mql
AUTOFILTER;
```
- **必须写！**
- 过滤重复信号
- 保证一开一平机制

---

## 执行流程

```
无持仓 + 金叉 → BK（买开）→ 多头持仓
    ↓
多头持仓 + 死叉 → BP（买平）→ 无持仓
    ↓
无持仓 + 死叉 → SK（卖开）→ 空头持仓
    ↓
空头持仓 + 金叉 → SP（卖平）→ 无持仓
    ↓
循环...
```

---

## 注意事项

### ✅ 正确写法
```mql
金叉 := CROSSUP(MA_S, MA_L);
金叉, BK;    // 金叉时买开
```

### ❌ 常见错误
```mql
// 错误1：赋值用 =
MA_S = MA(C,10);  // 错！

// 错误2：信号没有条件
BK;  // 错！

// 错误3：忘记AUTOFILTER
// 漏写！
```

## 扩展：添加止损

```mql
// 双均线 + 止损
N1 := 10;
N2 := 20;
止损 := 15;  // 止损跳数

MA_S := MA(C, N1);
MA_L := MA(C, N2);

金叉 := CROSSUP(MA_S, MA_L);
死叉 := CROSSDOWN(MA_S, MA_L);

// 开仓
金叉, BK;
死叉, SK;

// 平仓
金叉, SP;
死叉, BP;

// 止损（比较用 <= 或 >=）
CLOSE <= BKPRICE - 止损*MINPRICE, SP;  // 多头止损
CLOSE >= SKPRICE + 止损*MINPRICE, BP;   // 空头止损

AUTOFILTER;
```
