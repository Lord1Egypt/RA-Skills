# GA Insights - Skill Specification

Your AI-powered Google Analytics co-pilot. Ask questions in plain English, get actionable insights.

## Setup (3 Steps)

1. **Paste GA4 Property ID** — Find it in Analytics → Admin → Property Settings
2. **Add service account email** — We'll give you an email to add in GA4 access management
3. **Connect** — Done!

No GCP console, no JSON files, no complex setup.

## Usage

Just ask questions like:
- "Why did my traffic drop yesterday?"
- "Which pages have the highest bounce rate?"
- "How many users converted this week?"
- "What's my top traffic source?"

## Commands

- `ga setup` — Start the guided setup
- `ga status` — Check connection status
- `ga traffic` — Get traffic summary
- `ga realtime` — Current active users
- `ga conversions` — Conversion metrics

## Requirements

- Google Analytics 4 (GA4) property
- Access to GA4 Admin (to add service account)
