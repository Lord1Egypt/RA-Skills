# 信号叙事模板

## 适用场景
行业政策解读、市场动态、公司短讯、战略表态

## 事件级别
L1 常规（日常快讯）或 L2 标准（有观点的行业分析）

## 结构骨架

```
事实陈述 → 行业站位 → 我方视角 → 一句话信号
```

## 填写表

| 段落 | 写什么 | 字数 | 铁律 |
|------|--------|------|------|
| 事实陈述 | 发生了什么，不带判断 | 40-60字 | **客观**，可核查 |
| 行业站位 | 这个事件在行业里意味着什么 | 50-70字 | 有判断但不偏激 |
| 我方视角 | 我们的看法或已经在做什么 | 50-70字 | 不说"早就"，说"在做" |
| 一句话信号 | 可引用的立场金句 | 20-30字 | 清晰、有力、可截图 |

## 变量槽

| 变量 | 说明 | 示例 |
|------|------|------|
| {{EVENT}} | 事件/政策/动态 | 财政部发布XX管理办法征求意见稿 |
| {{INDUSTRY_IMPACT}} | 行业影响判断 | 灰色地带将被规范 |
| {{OUR_ACTION}} | 我方在做的事 | 去年Q3开始帮客户做合规预检 |
| {{SIGNAL}} | 金句 | 合规不是成本，是护城河 |

## 自检

- [ ] 事实陈述客观吗？（不带形容词和判断）
- [ ] 行业站位有判断但不过激吗？
- [ ] 我方视角是"在做"还是"早就"？（要前者）
- [ ] 一句话信号能截图转发吗？

## 常见问题

| 问题 | 修正 |
|------|------|
| "重磅！XX政策出台！" | 客观陈述政策核心内容 |
| "我们早就布局了XX" | "我们已在XX方向投入了两年" |
| "我们将密切关注" | 有态度，有方向 |
| 信号太模糊 | 一句话能被人记住和引用 |

## 示例

> **事实陈述**：上周，财政部发布《XX管理办法》征求意见稿，明确将XX服务纳入监管框架，要求2027年前完成合规改造。
>
> **行业站位**：这意味着过去三年围绕XX的灰色操作将被正式规范。对合规企业是利好，对擦边球是终局。行业洗牌会加速。
>
> **我方视角**：雄韬从去年Q3开始，已经在帮现有客户做XX合规预检。不是因为预见了政策，是因为相信一件事——财税的底线迟早会被写进条文。
>
> **一句话信号**：合规不是成本，是护城河。
