# 统计检验决策树

## Step 1: 确定因变量类型

```
因变量是连续型（分数、反应时等）？
├── 是 → Step 2A
└── 否（分类/计数）→ Step 2B
```

## Step 2A: 连续因变量 → 确定研究目的

```
研究目的是什么？
├── 比较组间差异
│   ├── 几组？
│   │   ├── 两组
│   │   │   ├── 被试间（独立组）→ 独立样本 t 检验
│   │   │   └── 被试内（配对/重复测量）→ 配对样本 t 检验
│   │   └── 多组（≥3）
│   │       ├── 被试间 → 单因素 ANOVA
│   │       ├── 被试内 → 重复测量 ANOVA
│   │       └── 混合 → 混合 ANOVA
│   └── 协变量存在 → 协方差分析 (ANCOVA)
├── 探索变量间关系
│   ├── 两个连续变量 → Pearson/Spearman 相关
│   ├── 预测一个连续变量 → 多元线性回归
│   └── 控制混淆变量 → 偏相关
├── 中介/调节效应
│   ├── 中介效应 → Bootstrap 中介分析
│   └── 调节效应 → 层次回归/Johnson-Neyman
└── 结构方程模型 → SEM（待确认）

```

## Step 2B: 分类因变量

```
因变量是分类（是/否、通过/失败）？
├── 两组 → McNemar 检验（被试内）/ 卡方检验（被试间）
└── 多组 → Cochran's Q / 卡方检验
```

## Step 3: 检查前提假设

### 参数检验前提
1. **正态性**：Shapiro-Wilk 检验（p > 0.05 表示正态）
   - 大样本（N>50）时，ANOVA/t 检验对正态性 Robust
2. **方差齐性**：Levene 检验（p > 0.05 表示齐性）
3. **独立性**：被试间设计需确保被试间独立
4. **球形性**：重复测量 ANOVA 需做 Mauchly 检验

### 非参数检验备选
当前提假设违反时：
- 两组配对 → Wilcoxon 符号秩检验
- 两组独立 → Mann-Whitney U 检验
- 多组被试间 → Kruskal-Wallis H 检验
- 多组被试内 → Friedman 检验
