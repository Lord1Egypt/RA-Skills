# BNP的生物学基础与参考值详解

## 生物学来源与生成机制
- BNP由心肌细胞（主要在心室）合成，以前体形式proBNP（108个氨基酸）存在
- 在心肌受刺激（如容量或压力超负荷）时，被酶切为：
  - NT-proBNP（N端76个氨基酸，无生物活性）
  - BNP（C端32个氨基酸，具环状结构和生物活性）
- 两者以等摩尔比例释放入血

## BNP的生理作用
- 利钠、利尿
- 扩张血管、降低血压
- 抑制血管平滑肌细胞与成纤维细胞增殖，参与心血管重塑调节

## 标准化诊断阈值（成人）
| 情境 | BNP (pg/mL) | NT-proBNP (pg/mL) |
|------|-------------|-------------------|
| 排除急性心衰 | <100 | <300 |
| 支持急性心衰诊断 | >100 | >450 (<50岁), >900 (50–75岁), >1800 (>75岁) |
| 慢性心衰监测 | 动态变化趋势比绝对值更重要，目标通常<100–150 pg/mL |

## 影响BNP水平的非心衰因素
- **升高因素**：高龄、肾功能不全（eGFR <60）、房颤、脓毒症、肺栓塞、右心负荷增加
- **降低因素**：肥胖（BMI >30）、早期心衰、某些药物（如ARNI、ACEI）

## 临床研究证据摘要
- BNP指导治疗可降低慢性心衰住院率（PRIMA、STARS-BNP等试验）
- 在急诊呼吸困难患者中，BNP <100 pg/mL 的阴性预测值 >98%
- BNP是ACS患者院内及长期死亡的独立预测因子
