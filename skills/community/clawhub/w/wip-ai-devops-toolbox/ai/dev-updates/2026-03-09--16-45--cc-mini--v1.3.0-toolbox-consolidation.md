## What happened

Massive consolidation session. Three new tools folded into the toolbox, org profile updated, release documentation hardened.

### New tools added

1. **wip-file-guard** ... copied from standalone `wip-file-guard-private` repo. Blocks destructive edits to protected identity files (CLAUDE.md, SHARED-CONTEXT.md, SOUL.md, etc.). Works with Claude Code CLI and OpenClaw.

2. **wip-universal-installer** ... copied from standalone `wip-universal-installer-private` repo. The Universal Interface spec for agent-native software. Six interfaces: CLI, Module, MCP Server, OpenClaw Plugin, Skill, Claude Code Hook. `UNIVERSAL-INTERFACE.md` promoted to repo root.

3. **wip-repos** ... built from scratch. Repo manifest reconciler. Makes `repos-manifest.json` the single source of truth. Commands: check, sync, add, move, tree. Pure JavaScript, zero dependencies. Initial scan found 98 untracked repos and 18 stale manifest entries.
