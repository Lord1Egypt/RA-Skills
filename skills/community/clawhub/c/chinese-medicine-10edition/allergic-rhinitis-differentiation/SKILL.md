# Allergic Rhinitis Differentiation

## When to Use
当患者具备以下全部主症：
- 鼻痒
- 喷嚏频频
- 流清涕如水
- 鼻塞（可伴嗅觉减退）

且同时提供反映脏腑虚实的全身兼症及舌脉信息时，应用本技能进行辨证分型与治疗建议。

## Execution Steps

1. **确认主症**：核实患者是否具备鼻鼽四大主症。
2. **辨识证型**：根据兼症与舌脉特征判断所属证型：
   - **肺气虚寒证**：畏风怕冷、自汗、语声低怯、咳嗽痰稀、面色苍白；舌淡苔薄白，脉虚弱；鼻黏膜淡白或灰白。
   - **脾气虚弱证**：食少纳呆、腹胀便溏、倦怠乏力、少气懒言、面色萎黄；舌淡胖有齿痕，苔薄白，脉弱；鼻黏膜淡白。
   - **肾阳不足证**：形寒肢冷、腰膝酸软、小便清长或遗精早泄；舌淡苔白，脉沉细；鼻黏膜苍白肿胀。
   - **肺经伏热证**：闷热天气易发，口干烦热、咽痒咳嗽；舌红苔白或黄，脉数；鼻黏膜色红或暗红。
3. **确定治则**：
   - 肺、脾、肾三脏虚证以补益为主；
   - 肺经伏热属实热，以清宣肺气为法。
4. **推荐基础方药**：
   - 肺气虚寒：温肺止流丹加减
   - 脾气虚弱：补中益气汤加减
   - 肾阳不足：肾气丸加减
   - 肺经伏热：辛夷清肺饮加减

> 注意：本技能仅适用于中医辨证体系下的鼻鼽诊疗，不适用于无典型症状者或非中医语境。
