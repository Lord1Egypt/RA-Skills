## Description: <br>
JSON Diff Tool compares two JSON documents in a browser and highlights path-level additions, removals, and unchanged fields without uploading data. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[shenghoo123-png](https://clawhub.ai/user/shenghoo123-png) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, testers, operations engineers, and product teams use this skill to compare API responses, configuration files, and test JSON data for path-level differences. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Sensitive JSON may contain secrets, tokens, customer data, or internal configuration. <br>
Mitigation: Use the reviewed local HTML or ClawHub copy for sensitive JSON and avoid pasting secrets into externally hosted pages unless the served code is trusted. <br>
