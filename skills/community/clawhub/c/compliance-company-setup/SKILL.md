# 跨境电商注册地推荐（Company Setup Check）

> 一人公司出海第一步：在哪个国家/地区注册公司最划算？一分钟看清 8 个热门注册地的成本、税率、流程和平台兼容性。

## 核心能力

输入：目标市场 + 经营模式 → 输出：最优注册地推荐 + 多管辖区对比表

```
"我在亚马逊卖 3C 配件，月销 4 万美元，想正规化经营" 
→ 对比 Delaware LLC / Wyoming LLC / 香港有限公司 / 新加坡 Pte Ltd
→ 推荐 Wyoming LLC（年费 $52，无州所得税，穿透税制）
```

## 工作流程

### Step 1：解析用户需求

从用户描述中提取关键信息：

```
用户：「我从1688拿货，在亚马逊德国法国卖，月销4万刀，PayPal收款」
  → 经营模式：跨境电商（平台卖家）
  → 主要市场：EU（德国、法国）
  → 月销规模：$40,000
  → 收款方式：PayPal
  → 品类：暂未提及（后续可追问）
```

核心维度评估：

| 维度 | 权重 | 说明 |
|------|:---:|------|
| 目标市场匹配 | ⭐⭐⭐ | 注册地是否与销售市场一致 |
| 合规成本 | ⭐⭐⭐ | 注册费+年费+税费+银行开户 |
| 平台兼容性 | ⭐⭐ | 主流电商平台是否认可 |
| 运营便利性 | ⭐⭐ | 远程管理、银行开户难度 |
| 税务优势 | ⭐⭐ | 税率、避免双重征税协定 |

### Step 2：输出对比决策表

```
## 🏢 企业设立方案对比

| **注册成本** | ¥5,000-12,000 | **¥3,000-8,000** | ¥5,000-12,000 | ¥8,000-18,000 |

| **注册周期** | 3-5天 | 3-5天 | 5-7天 | **1-2天** |
| **公司税率** | 0%（穿透） | **0%（穿透+无州税）** | 8.25-16.5% | 17%（前20万新元部分免税） |

| **EU 销售** | ❌ 需额外EU合规 | ❌ 需额外EU合规 | ❌ 需额外EU合规 | ❌ 需额外EU合规 |
| **远程管理** | ✅ 可远程 | ✅ 可远程 | ✅ 可远程 | ❌ 需本地董事 |

### 🎯 推荐：Wyoming LLC

**理由**：
1. 年费最低（$52/年），适合月销 4 万美元的 OPC
2. 无州公司所得税 + 穿透税制，无需双重纳税
3. Amazon/Shopify 全兼容
4. 但 ⚠️ 卖到 EU 仍需注册 VAT + CE 合规（这是另一回事）

### 📊 落地可行性评分 (Feasibility Score)
- **评分**：85/100 (🟢 方案可行)
- **扣分项**：暂未开立美国银行账户，需通过第三方数字银行解决。

### ⚠️ 特别提醒
- 注册了美国公司 ≠ 自动合规 EU 销售。你需要在每个 EU 销售国注册 VAT
- 建议搭配爱沙尼亚 e-Residency OÜ 作为 EU 入口（年费€0）
```

## 边界说明

### ✅ 能做的
- 8 个热门注册地的公司类型对比（成本/税率/流程/平台兼容）
- 基于用户经营模式给出推荐和理由
- 输出步骤化注册清单和时间线
- 多市场联动方案（总部+入口组合）
- 银行开户难度和替代方案提示

### ⚠️ 不能替代的
- **不构成法律意见**：数据为知识整理，最终决策请咨询注册代理或律师
- **注册费用为估算**：实际费用因代理机构、汇率、政策变动而不同
- **银行开户情况会变**：反洗钱政策持续收紧，需以银行最新要求为准
- **不覆盖全注册地**：当前覆盖 8 个国家/地区，其他管辖区需用户自行补充

## 反模式

| ❌ 错误用法 | ✅ 正确用法 |
|------------|-----------|
| 只看年费选 Wyoming | 综合考虑目标市场匹配度——卖到 EU 选美国 LLC 不能解决 VAT 问题 |
| 以为注册公司 = 合规 | 公司注册只是第一步，还需 VAT、产品认证、包装法等多层合规 |
| 忽略银行开户难度 | 提前准备 ITIN/护照/地址证明，选择数字银行作为备选 |
| 认为税率低的就一定好 | HK 税率低但 CRS 全面信息交换，隐私性不如 Wyoming |

## 常见问题

**Q：Delaware 和 Wyoming 到底选哪个？**
A：非上市、不融资的 OPC 选 Wyoming（年费低 6 倍）。需要融资、投资人熟悉的选 Delaware。

**Q：爱沙尼亚 e-Residency 靠谱吗？**
A：靠谱，但注意：① e-Residency 不是签证，不能入境工作；② 传统银行开户难，建议用 Wise Business；③ 适合 EU 销售场景，非 EU 销售没必要。

**Q：香港公司现在开户还方便吗？**
A：反洗钱审查非常严格，个人卖家开户需亲临香港，汇丰/中银/渣打均加强 KYC。建议备选数字银行或 Payoneer。

**Q：注册公司后每年要做什么？**
A：① 年审/年报（各注册地要求不同）；② 税务申报（即使零收入）；③ BOI 申报（美国公司，首次 90 天内，之后变更 30 天内）；④ 银行账户保持活跃。
