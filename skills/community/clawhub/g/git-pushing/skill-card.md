## Description: <br>

This skill is ready for commercial/non-commercial use. <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and engineers use this skill to stage repository changes, create conventional commits, and push the current branch to a configured Git remote. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can stage every current repository change and push it to the configured Git remote without a built-in review or confirmation step. <br>
Mitigation: Review git status, the diff, the current branch, and the origin URL before using the workflow, especially in public repositories or workspaces with secrets or unrelated edits. <br>
