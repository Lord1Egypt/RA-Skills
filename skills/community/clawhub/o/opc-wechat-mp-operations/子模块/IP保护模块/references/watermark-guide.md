# 水印设计指南

> 为公众号内容创建专业、有效且美观的水印标识

---

## 一、文字水印设计

### 1.1 位置策略

#### 开头水印
**适用场景**：建立首发认知，强化品牌意识

**推荐格式**：
```
━━━━━━━━━━━━━━━━━━━━━━━━
📌 本文首发于【公众号名称】
  → 专注[定位关键词]...
━━━━━━━━━━━━━━━━━━━━━━━━
```

#### 结尾水印
**适用场景**：强化品牌，引导关注

**推荐格式**：
```
━━━━━━━━━━━━━━━━━━━━━━━━
💡 原创文章，阅读全文请关注公众号【公众号名称】
   后台回复【关键词】领取[资料名称]
━━━━━━━━━━━━━━━━━━━━━━━━
```

### 1.2 样式选择

| 样式 | 代码 | 效果 | 适用场景 |
|------|------|------|----------|
| 纯文本 | 普通文本 | 简洁直接 | 资讯、干货类 |
| 引用框 | `<blockquote>` | 突出醒目 | 重要声明、观点 |
| 分隔线 | `---` | 清晰分隔 | 通用场景 |
| 带图标 | Emoji + 文字 | 活泼亲切 | 情感、故事类 |

## 二、图片水印设计

### 2.1 核心参数

```python
{
    "position": "bottom_right",
    "opacity": 0.5,
    "font_size": 24,
    "padding": 20,
    "font_family": "Microsoft YaHei",
    "font_color": [255, 255, 255],
}
```

### 2.2 位置参数对照

```
┌─────────────────────────────────────┐
│  top_left    │         top_right    │
│              │                      │
│              │                      │
│              │                      │
│              │                      │
│  bottom_left │         bottom_right │ ← 最常用
└─────────────────────────────────────┘
```

### 2.3 透明度与大小参考

| 图片尺寸 | 推荐字体大小 | 推荐透明度 |
|----------|-------------|------------|
| < 800px  | 16-20px     | 0.6-0.7    |
| 800-1200px | 22-26px  | 0.4-0.5    |
| 1200-2000px | 28-32px | 0.3-0.4    |
| > 2000px | 36-48px     | 0.2-0.3    |

---

## 三、不同图片类型的水印策略

| 图片类型 | 水印策略 | 透明度 | 说明 |
|----------|----------|--------|------|
| 信息图/数据图 | 强水印 | 40% | 核心资产，必须标注 |
| 案例配图 | 适中水印 | 50% | 保护与美观平衡 |
| 人物/场景图 | 轻量水印 | 60-70% | 不影响画面主体 |
| 封面图 | 隐藏式水印 | 70-80% | 角落小字 |

## 四、品牌一致性

### 4.1 统一水印模板
```
主品牌色：#2D7D9A（科技蓝）
辅助色：#E67E22（观点橙）、#27AE60（提醒绿）
字体：微软雅黑 / 思源黑体
圆角：8px
```

### 4.2 水印家族
- 文字水印：用于文章
- 图片水印：用于配图
- Logo水印：用于封面
- 二维码水印：用于长图/海报
