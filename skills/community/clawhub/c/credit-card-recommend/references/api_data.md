# 信用卡 API 数据结构参考

## 接口地址

```
https://cs.citiccardcdn.citicbank.com/citiccard/cardshopcloud/eshop/appimg/cardshop/card/remain_first.json
```

## 顶层结构

| 字段名 | 数据类型 | 说明 |
|--------|----------|------|
| `cdnPath` | `string` | 卡面图片 CDN 基础路径，与卡片 `ci` 字段拼接可得到完整图片 URL |
| `datas` | `array[object]` | 信用卡产品数组，每个元素代表一个产品系列 |

## datas 数组元素结构（产品系列级别）

| 字段名 | 数据类型 | 说明 |
|--------|----------|------|
| `agg` | `string` | 聚合标识：`"0"` 非聚合（独立展示），`"1"` 聚合（子卡面合并展示） |
| `au` | `string` | 申请链接 URL（标准申请通道） |
| `bi` | `string` | 保留字段，多为空字符串 |
| `bti` | `string` | 保留字段，多为空字符串 |
| `ch` | `array[object]` | **卡面变体数组**，包含同一产品下的不同卡面版本 |
| `fl` | `string` | 搜索/分类标签，逗号分隔的关键词字符串 |
| `gi` | `string` | 产品组 ID（同一产品不同渠道可能分配不同 gi） |
| `hd` | `string` | 屏蔽的不展示的渠道，逗号分隔 |
| `pn` | `string` | **产品系列名称** |
| `pu` | `string` | **产品详情页/申请页 URL** |
| `si` | `string` | 角标图片文件名（与 cdnPath 拼接得到完整 URL） |
| `tu` | `string` | 保留字段，多为空字符串 |

## ch 数组元素结构（卡面变体级别）

| 字段名 | 数据类型 | 说明 |
|--------|----------|------|
| `ci` | `string` | **卡面图片文件名**（如 `KPIBJX.png`），拼接 cdnPath 得到完整 URL |
| `cn` | `string` | **卡片完整名称** |
| `cr` | `array[object]` | **卡片权益/特性描述数组** |
| `ct` | `string` | 卡片类型代码（如 `607`=i白金, `935`=白金, `950`=金卡） |
| `cv` | `string` | 卡面版本 ID |
| `f` | `string` | **卡等级代码**（见卡等级代码映射表） |
| `gi` | `string` | 产品组 ID（与上层 gi 对应） |
| `if` | `string` | 是否首选项：`"1"` 默认/推荐卡面，`"0"` 或缺失为非默认 |
| `l` | `string` | **卡面版本标签**（如"i白金卡"、"金卡"、"西高地"、"女王"） |
| `ncg` | `string` | **新客户礼遇标语**（新户专属权益简短描述） |
| `p` | `string` | 产品代码/产品 ID（如 `CS0083`） |

## cr 数组元素结构（权益描述级别）

| 字段名 | 数据类型 | 说明 |
|--------|----------|------|
| `d` | `string` | **权益简短描述**（一句话概要） |
| `dt` | `string` | **权益详细描述**（含完整规则说明，多为多行文本，用 `\n` 换行） |
| `i` | `string` | 权益图标图片文件名（与 cdnPath 拼接） |
| `t` | `string` | **权益标签/标题**（如"航延保险"、"年费减免"、"笔笔返现"） |
| `tp` | `string` | 权益类型分类（多为空字符串） |

## 卡等级代码映射

| 代码 | 含义 | 年费大致区间 |
|------|------|------------|
| `W3` | i白金卡 | 刷卡免年费 |
| `W2` | 白金卡 | 刷卡免年费 |
| `B3` | 精英/精逸白金卡 | 480-2000元 |
| `K4` | 精逸级白金卡 | 2000元 |
| `E4` | 尊贵级白金卡 | 2000元 |
| `C5` | 尊尚白金卡 | 3600元 |
| `F7` | 高端白金卡 | 6800元 |
| `00` | 尊贵/高端白金卡 | 2000元+ |
| `TU` | 特殊白金等级 | 视产品而定 |
| `GOLD` | 金卡 | 刷卡免年费 |

## 关键构建规则

### 卡面图片 URL

```
完整URL = cdnPath + ci
示例: https://cs.citiccardcdn.citicbank.com/citiccard/cardshopcloud/eshop/appimg/cardshop/card/KPIBJX.png
```

### 申请链接优先级

1. 产品级别 `pu`（优先）
2. 产品级别 `au`（备选）

### 去重规则

同一 `gi`（产品组 ID）可能在不同渠道的 datas 条目中出现多次，推荐时按 `gi` 去重，只保留一条。

### 聚合机制

- `agg == "1"` 时，产品包含多个卡面变体（`ch` 数组有多项），推荐时选 `if == "1"` 的默认卡面，若无默认则取第一个
- `agg == "0"` 时，产品独立展示

## 数据特征

| 特征 | 说明 |
|------|------|
| 嵌套深度 | 4 层：Root → datas → ch → cr |
| 一对多关系 | 1个产品系列可含多个卡面变体，1个卡面可含多项权益 |
| 空值处理 | 大量字段使用空字符串 `""` 而非 null 表示无值 |
| 文本格式 | `dt` 字段使用 `\n` 换行，含完整活动规则 |
| 重复数据 | 同一产品可能以不同 `gi` 出现在 datas 中（多渠道投放） |
