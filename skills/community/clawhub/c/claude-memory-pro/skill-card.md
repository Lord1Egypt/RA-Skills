Risk: Persistent cross-session memory can retain user and project context without clear consent, retention, or deletion controls. <br>
Mitigation: Use the skill only with explicit consent for memory retention, avoid storing secrets or regulated data, and regularly inspect, delete, or limit stored memory. <br>
