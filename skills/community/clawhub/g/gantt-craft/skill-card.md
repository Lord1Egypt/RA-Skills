## Use Case: <br>
Project managers, developers, and operations teams use this skill to turn task lists with dates, owners, progress, and dependencies into shareable interactive Gantt chart HTML. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Project schedules, owners, dependencies, and progress data can contain sensitive business information. <br>
Mitigation: Use the skill only in an approved Codex environment for the sensitivity of the project data being supplied. <br>
Risk: Generated timelines can misrepresent project status if source task dates, dependencies, or progress values are incomplete or stale. <br>
Mitigation: Review the generated chart against the source plan before sharing or using it for project decisions. <br>
