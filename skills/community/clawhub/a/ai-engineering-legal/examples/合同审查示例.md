# 合同审查示例

## 场景：某施工总承包合同审查

### 原始合同条款（问题条款）

```
第15条 合同价款
本工程合同价款为暂定金额8,000万元，最终以审计结果为准。
具体计价标准由双方另行协商确定。

第22条 设计变更
发包人有权对本工程进行设计变更，承包人应无条件执行。
变更费用由承包人承担，重大变更可经协商后调整。

第28条 工程结算
工程竣工验收合格后由发包人委托的审计机构进行审计，
审计结果为最终结算依据。审计时间不限。
```

### 技能审查结果

#### 风险点1：计价方式不明

- **风险等级**：🔴 高
- **风险描述**：
  - "暂定金额"不是规范的计价方式
  - "以审计结果为准"将定价权完全交给发包人
  - "计价标准另行协商"导致合同价格不确定
- **知识库依据**：《工程量清单计价标准》要求合同明确计价方式
- **修改建议**：
  ```
  本工程采用工程量清单计价方式。
  合同中已有适用或类似子目的综合单价按已有综合单价执行；
  无适用或类似子目的，按以下原则确定：
  （1）参照投标报价水平；
  （2）参照工程所在地同期信息价；
  （3）由双方协商确定。
  ```

#### 风险点2：变更费用分配不公

- **风险等级**：🔴 高
- **风险描述**：
  - "无条件执行"违背合同公平原则
  - "变更费用由承包人承担"违反合同法基本规定
  - "重大变更可经协商"的"重大"标准模糊
- **知识库依据**：《变更索赔实务指南》要求明确变更费用承担
- **修改建议**：
  ```
  发包人提出的设计变更，增加的费用由发包人承担，
  减少的费用相应扣减。变更费用的计算按第15条约定执行。
  任何变更均应在实施前以书面形式确认。
  ```

#### 风险点3：结算审计时限不明确

- **风险等级**：🔴 高
- **风险描述**：
  - "审计时间不限"可能无限期拖延结算
  - "审计结果为最终依据"剥夺了承包人的异议权
  - 未约定发包人逾期不审的后果
- **知识库依据**：《建设工程价款结算暂行办法》规定审核时限
- **修改建议**：
  ```
  承包人提交完整结算资料后，发包人应在60日内完成审核。
  逾期未完成审核的，视为认可承包人提交的结算文件。
  双方对审核结果有争议的，按争议解决条款处理。
  ```

### 审查结论

- **风险评分**：85分（高风险）
- **行动建议**：必须修改上述3个条款后方可签约
- **谈判策略**：以行业惯例和法律规定为依据，争取公平条款
