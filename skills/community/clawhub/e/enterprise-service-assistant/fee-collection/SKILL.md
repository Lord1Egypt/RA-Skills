# 费用催缴管理技能

## 概述

本技能基于《C+基础保障服务手册-企服（含产促中心）管理工作规程》（{TODO: 填写文件编号}）第十一章，实现规范的费用催缴流程。

## 核心流程

### 1. 费用收缴管理

#### 费用类型
1. **入伙费用**：首月管理费、专项维修资金及水电周转金
2. **二次装修费用**：装修保证金、垃圾清运费、出入证保证金、出入证工本费、消防水费、临时施工用水、电费等
3. **管理费、电费**：每月5日前打印收费通知书，20日前客户须付清
4. **有偿服务费**：维修、清洁、空调加时费等，每月25日前汇总

#### 收费通知流程
1. 财务人员打印《收费通知书》
2. 企服管家将通知书送达客户并签收
3. 客户在每月20日前付清费用
4. 企服管家跟进催款

### 2. 智能催缴策略

#### 三级分级体系

**第一级：1-7天（静默期）**
- **策略**：静默处理，仅内部记录
- **动作**：不主动联系客户，仅系统记录逾期状态
- **话术**：无（不发送提醒）

**第二级：8-30天（提醒期）**
- **策略**：主动提醒，推送催缴通知
- **动作**：发送催缴通知，电话提醒
- **话术模板**：
  - "尊敬的{客户名称}，您的{费用类型}（{金额}元）已逾期{逾期天数}天，请尽快缴纳。"
  - "【催缴通知】{客户名称}：请于{限缴日期}前缴纳{费用类型}{金额}元。"

**第三级：31天+（升级期）**
- **策略**：升级处理，@all通知
- **动作**：@all通知，上报负责人，考虑法律途径
- **话术模板**：
  - "@all 【紧急催缴】{客户名称} - {费用类型} 已逾期{逾期天数}天，请立即处理！"
  - "【最后通知】您的{费用类型}（{金额}元）已逾期{逾期天数}天，请于{限缴日期}前缴纳，否则将采取进一步措施。"

### 3. 个性化话术生成

#### 话术个性化维度
1. **客户类型**：
   - VIP客户：尊敬的VIP客户{客户名称}...
   - 普通客户：尊敬的{客户名称}...
   - 新客户：欢迎入驻{客户名称}，请按时缴纳...

2. **逾期历史**：
   - 首次逾期：首次逾期，请尽快处理...
   - 多次逾期：您已多次逾期，请高度重视...

3. **费用类型**：
   - 管理费：物业管理费...
   - 电费：电费...
   - 租金：租金...

4. **催缴阶段**：
   - 提醒阶段：温馨提醒...
   - 警告阶段：正式警告...
   - 最后通知：最后通知...

### 4. 能耗欠费合并计算

#### 合并逻辑
1. **按客户合并**：同一客户的所有欠费合并计算
2. **按费用类型分组**：管理费、电费、租金等分别统计
3. **计算总额**：本金 + 滞纳金（如适用）
4. **生成合并通知**：一次性通知所有欠费

#### 合并通知示例
```
【合并缴费提醒】上海XX科技有限公司：

您有以下费用待缴纳：
1. 2024年1月管理费：12,500元（逾期5天）
2. 2024年1月电费：3,200元（逾期5天）
3. 2023年4季度租金：50,000元（逾期35天）

总金额：65,700元
请尽快安排缴纳，避免产生更多滞纳金。
```

## 职责分配

### 财务部
1. 负责每月按时核算、出具管理费以及其他各项费用《收费通知书》
2. 收取相关费用

### 企服部
1. 负责将《物业收费通知书》送达客户，给客户签收
2. 跟进催款
3. 列出用户欠款情况表

### 项目负责人
1. 审查欠款情况表
2. 批示对仍欠款用户采取适当催款措施

## 使用场景

### 场景1：定时检查费用收缴状态
**触发条件**：每天10:00定时任务

**处理流程**：
1. 读取Excel中的费用数据
2. 计算逾期天数
3. 按三级分级体系分类
4. 生成催缴提醒
5. 发送通知（8-30天推送，31天+ @all）

### 场景2：手动触发催缴检查
**触发条件**：@企服助手 催缴检查

**处理流程**：
1. 立即执行费用催缴检查
2. 生成催缴报告
3. 返回逾期费用清单

### 场景3：企微群关键词触发
**触发条件**：企微群中@催缴

**处理流程**：
1. 识别关键词
2. 执行催缴检查
3. 在群中发送催缴通知

### 场景4：逾期分级处理
**触发条件**：系统自动判断逾期天数

**处理流程**：
1. 1-7天：静默处理，仅记录
2. 8-30天：推送催缴通知
3. 31天+：@all通知，升级处理

## 关键指标

### 催缴效果指标
- **催缴响应率**：收到催缴后7日内缴费的比例 ≥ 70%
- **催缴成功率**：催缴后30日内缴费的比例 ≥ 90%
- **平均催缴次数**：成功催缴前的平均提醒次数 ≤ 3次

### 时效性指标
- **催缴及时率**：逾期后8日内首次催缴的比例 = 100%
- **升级处理及时率**：逾期31日后升级处理的比例 = 100%

## 与其他技能的协作

### 与customer-management协作
- 客户缴费历史用于个性化话术生成
- 欠费客户标记为风险客户

### 与contract-renewal协作
- 欠费客户可能影响续租决策
- 续租时核对费用缴纳情况

### 与visit-management协作
- 走访时提醒客户缴费
- 催缴结果在走访记录中更新

## 注意事项

1. **尊重客户**：话术要尊重客户，避免强硬措辞
2. **准确计算**：逾期天数、滞纳金等要准确计算
3. **及时跟进**：按规定时限及时跟进
4. **记录完整**：每次催缴都要完整记录
5. **合规操作**：催缴行为要符合法律法规
