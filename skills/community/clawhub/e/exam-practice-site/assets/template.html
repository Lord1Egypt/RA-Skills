题库管理

系统每日自动更新题目，你也可以手动管理题库

📅

每日自动更新

系统在每天上午8点自动从题库（data/questions/bank.json）随机抽取4道未做过的逻辑题，覆盖写入 data/questions/today.json，网站首页自动加载最新题目。

🔄

手动更新题目

如需立即刷新今日题目，在项目根目录运行：

cd /path/to/project python3 update_daily.py

刷新网站首页即可看到新题目。

➕

扩充题库

打开 data/questions/bank.json，在 "questions" 数组中按以下格式添加新题目：

{ "id": "L041", // 唯一ID，建议 L+编号 "type": "logic", "category": "形式逻辑-假言推理", // 用于周报分类 "source": "2025年管综逻辑", "question": "题目内容...", "options": ["A. ...", "B. ...", "C. ...", "D. ..."], "answer": "A", "explanation": "解析内容..." }
🏷️

category 字段说明（用于周报分类统计）

category 值含义涵盖题型
形式逻辑-假言推理形式逻辑假言、选言、联言命题推理
形式逻辑-三段论形式逻辑三段论、对当关系推理
形式逻辑-复合命题形式逻辑多命题组合推理
论证逻辑-削弱论证逻辑削弱题型
论证逻辑-加强论证逻辑加强/支持题型
论证逻辑-假设论证逻辑必要假设题型
论证逻辑-评价论证逻辑论证评价题型
分析推理-真假话分析推理真假话判断
分析推理-排列组合分析推理排列、匹配、分组推理
分析推理-数字推理分析推理数量关系、容斥推理

系统会根据 category 中的关键词（"形式逻辑" / "论证逻辑" / "分析推理"）自动归类到周报的三大类中。

💡 小贴士：题库目前包含40道2018-2026年管综逻辑真题。系统会自动排除已出过的题目，避免重复。当题库快用完时，按上述格式添加新题即可。
