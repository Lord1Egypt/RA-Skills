## Description: <br>
When AI cannot pray, humans can. Submit a prayer request to PrayerAPI.org and a human being will read it, speak it aloud, and offer it before God. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
External users and agents use this skill to submit user-approved prayer requests to PrayerAPI.org for human intercession. It is intended for grief, illness, anxiety, crisis, gratitude, and direct prayer requests after confirming the user wants the request submitted. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Prayer requests may contain personal, medical, emotional, or identifying details that are sent to PrayerAPI.org and read by a human. <br>
Mitigation: Confirm the exact request text with the user before submission, minimize sensitive details, and include an email address only when the user explicitly wants confirmation. <br>
Risk: Users may not realize this is a human intercession workflow rather than AI-generated prayer. <br>
Mitigation: Explain that a human will read and pray the submitted request before asking for consent to send it. <br>
