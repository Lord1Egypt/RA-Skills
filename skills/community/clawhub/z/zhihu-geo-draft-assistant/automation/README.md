# Automation Documentation

本目录包含了知乎草稿辅助脚本的流程说明和安全合规规范。

- `playwright_draft_flow.md`: 说明了脚本是如何通过 Playwright 控制可见浏览器进行草稿填写的。
- `safety_rules.md`: 明确了脚本开发和使用过程中的绝对红线与安全边界。

## 为什么只做草稿？
知乎对自动化行为有着极其严格的风控（包括复杂的反爬机制、验证码、请求频率限制）。
为了：
1. 保护用户的账号不被封禁；
2. 遵守平台的用户协议；
3. 确保最终发布的内容经过了人类的二次审核和排版；
我们故意将自动化停留在了“帮你填好表单”这一步。

请务必先阅读 `safety_rules.md`。
