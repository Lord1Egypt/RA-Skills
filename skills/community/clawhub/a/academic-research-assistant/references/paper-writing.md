# 论文写作指导参考

---

## 一、论文结构模板

### IMRaD 标准结构

```
标题（Title）
  ├── 简洁明了，包含关键词
  ├── 一般不超过20个字
  └── 避免"研究""探讨"等冗余词

摘要（Abstract）
  ├── 背景（1-2句） 
  ├── 目的（1句）
  ├── 方法（2-3句）
  ├── 主要结果（2-3句）
  └── 结论与意义（1-2句）
  ├── 一般200-300字（中英文各一份）
  └── 关键词：3-8个

引言（Introduction）
  ├── 研究背景（从宽到窄的漏斗结构）
  ├── 文献综述（前人工作的梳理和评价）
  ├── 研究空白（当前研究的不足）
  └── 研究目的和意义（本文要解决的问题）

方法（Methods）
  ├── 研究对象/数据来源
  ├── 实验设计/研究方案
  ├── 数据收集过程
  ├── 分析方法（统计方法或算法）
  └── 伦理声明（如涉及人类或动物）

结果（Results）
  ├── 客观呈现，不加入讨论
  ├── 使用图表辅助说明
  ├── 图和表的规范：图标题在图下，表标题在表上
  └── 按逻辑顺序组织

讨论（Discussion）
  ├── 主要发现总结（首段）
  ├── 结果解释和意义
  ├── 与前人研究的对比
  ├── 研究的局限性
  └── 未来研究方向

结论（Conclusion）
  ├── 研究的主要贡献（1-2段）
  ├── 实践意义（可选）
  └── 不超过摘要的篇幅

参考文献（References）
  └── 按目标期刊的格式要求组织
```

## 二、标题写作技巧

### 标题类型对比

| 类型 | 特点 | 示例 |
|:---:|------|------|
| 描述型 | 直接描述研究内容，客观 | "基于深度学习的医学图像分割方法" |
| 声明型 | 突出研究发现，吸引眼球 | "深度学习在医学图像分割中达到专家级精度" |
| 提问型 | 提出研究问题 | "深度学习能否取代传统医学图像分析方法？" |

### 标题写作原则
1. **简洁**：15-20字以内
2. **准确**：包含研究对象、方法、核心变量
3. **规避**："关于""的研究""初步探讨"等多余词
4. **关键词**：包含用于数据库检索的核心术语

## 三、摘要写作指南

### 结构化摘要模板

```
背景：陈述研究背景和意义（1-2句）
目的：明确研究目的（1句）
方法：简述研究方法和设计（2-3句）
结果：列出关键发现和数据（2-3句）
结论：总结主要结论和意义（1-2句）

关键词：关键词1；关键词2；关键词3；关键词4；关键词5
```

### 注意事项
1. 摘要不包含参考文献引用
2. 不出现"首次""填补空白"等自我评价
3. 不使用缩写（除非标准缩写）
4. 中英文摘要内容一致

## 四、引言写作——漏斗法

```
第1段：大背景
  → 研究领域的普遍意义和重要性
  → 领域的发展趋势

第2段：缩小范围
  → 前人主要研究成果
  → 分类综述（按方法/时间/学派）

第3段：研究空白
  → 当前研究的不足或未解决的问题
  → 为什么这些问题重要

第4段：本文方案
  → 本文的研究目的
  → 研究方法概述
  → 主要贡献（实事求是）

注意：每段2-3句即可，不要冗长
```

### 常用句式模板
- "近年来，XX领域受到广泛关注..."
- "然而，现有研究主要关注XX，对XX关注不足..."
- "本文提出了一种XX方法/框架，旨在解决XX问题..."
- "本研究的主要贡献包括：(1)...(2)...(3)..."

## 五、文献综述写作方法

### 组织方式

| 方式 | 适用场景 | 做法 |
|:---:|---------|------|
| 时间序 | 展示领域发展脉络 | 按时间线梳理关键研究 |
| 主题法 | 多方向并存的研究领域 | 按不同研究主题分类综述 |
| 方法序 | 方法驱动的研究 | 按研究方法类别组织 |
| 理论序 | 理论框架导向 | 按理论流派梳理 |

### 文献评述句式
- "Smith等（2020）发现...，但该研究存在...的局限"
- "与本文研究最相关的是...提出的...方法"
- "目前学界对...的研究主要集中在...，对...关注较少"

## 六、结果与讨论写法

### 结果部分
- **客观呈现**：仅呈现数据和分析结果，不解释
- **图表优先**：能用图表的不用文字
- **逻辑递进**：从基本分析到高级分析

### 讨论部分
- **首段**：一句话总结核心发现
- **对比分析**：与前人研究结果对比
- **解释机制**：为什么得到这个结果
- **局限性**：坦诚研究的不足
- **展望**：下一步研究方向

---

## 七、写作注意事项

1. **避免抄袭**：所有引用必须标注出处
2. **客观表述**：避免过度主观评价
3. **逻辑连贯**：段落之间过渡自然
4. **语言简洁**：删去冗余词汇
5. **格式统一**：标题层级、图表编号、参考文献格式必须统一
6. **AI辅助工具使用**：如使用AI工具写作，须按期刊要求在文中声明
