# 世界观构建指南

## 世界观的作用

世界观是剧本杀的"容器"——它决定了角色能做什么、不能做什么，也决定了诡计的边界。

## 世界观要素清单

### 1. 时代背景

| 时代 | 优势 | 限制 |
|------|------|------|
| 古代（东方） | 武侠、宫廷、江湖元素丰富 | 科学推理受限 |
| 古代（西方） | 哥特、宗教、贵族元素 | 文化距离感 |
| 民国 | 谍战、家族、国仇家恨 | 套路化风险高 |
| 现代（都市） | 最易代入，科技手段多样 | 缺乏新鲜感 |
| 未来/科幻 | 设定自由度高 | 需要详细解释规则 |
| 架空世界 | 完全自由 | 需要自洽的规则体系 |

### 2. 场景选择

好场景的标准：
- **封闭性**：角色不能轻易离开（暴风雪山庄、孤岛、密室、列车）
- **功能性**：场景本身能成为诡计的一部分（钟楼、镜屋、迷宫）
- **氛围感**：能强化类型基调（古宅配恐怖、游轮配奢华）

经典场景库：

**封闭空间类**
- 暴风雪山庄（最经典，无法离开的理由充分）
- 孤岛（更极端的封闭，适合大型本）
- 列车/飞机（移动中的封闭，时间压力强）
- 密室逃脱主题馆（天然的机关设定）

**开放但受限类**
- 宴会/派对（社交理由充分，但需要解释为何不离开）
- 学校/医院（公共空间，但有规则限制）
- 庄园/古宅（有历史感，可设置机关暗道）

**特殊设定类**
- 游戏世界（角色被困在虚拟世界）
- 时间循环（重复同一天，信息逐步积累）
- 平行世界（同一角色的不同版本）

### 3. 规则体系

对于变格本或架空世界，必须明确规则：

```
规则文档应包含：
1. 超自然力量的来源和限制
2. 特殊能力的触发条件和代价
3. 禁止事项（什么不能做）
4. 信息获取方式（玩家如何知道这些规则）
```

**示例：时间循环规则**
```
- 每天00:00重置，所有人回到起点
- 只有主角保留记忆，其他人不保留
- 每次循环，某些细节会发生微小变化
- 循环次数有限（7次），否则永远困住
- 死亡不是终结，会直接进入下一次循环
```

### 4. 世界观自检

完成世界观设定后，检查：
- [ ] 规则是否自洽（不能前后矛盾）
- [ ] 玩家是否能快速理解（不需要长篇解释）
- [ ] 规则是否服务于剧情（不是为了炫酷而设）
- [ ] 是否有隐藏的规则漏洞（玩家可能利用）

## 示例世界观

### 示例1：民国·上海滩·1937

```
背景：抗日战争前夕，上海租界歌舞升平
场景：百乐门舞厅顶楼私人包厢
限制：日军已封锁租界出口，所有人必须留到明早
氛围：纸醉金迷下的暗流涌动
可用元素：电报、枪支、鸦片、情报、租界法律灰色地带
```

### 示例2：现代·暴风雪山庄

```
背景：一群老同学受邀参加同学会
场景：阿尔卑斯山度假别墅，暴雪封山
通讯：无信号，固定电话线被切断
限制：积雪两米，无法离开，救援三天后才到
可用元素：现代科技（但受限）、别墅机关、过往恩怨
```

### 示例3：架空·镜中世界

```
背景：所有人醒来发现自己在镜中世界
规则：
  - 镜中人是现实的倒影，性格相反
  - 镜子碎裂会导致对应的人消失
  - 每面镜子连接不同的时间点
  - 杀死镜中人 = 杀死现实中的自己
氛围：诡异、不确定、身份危机
```
