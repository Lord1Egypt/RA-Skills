| 项目 | 内容 |
|------|------|
| **专利号** | ZL201811550298.8 |
| **专利名称** | 一种用于合成气催化转化的催化剂及其制备方法和应用 |
| **申请人** | 中国科学院上海高等研究院 |
| **发明人** | 钟良枢、林铁军研究员团队 |
| **技术领域** | 新材料/化工（煤化工、烯烃合成） |
| **专利类型** | 发明专利（核心专利） |
| **技术摘要** | 本发明涉及一种用于合成气催化转化制α-烯烃的高效催化剂及其制备方法。该技术通过创新的催化剂设计，实现了高选择性、高时空产率的α-烯烃合成，与传统煤制油路线相比CO₂减排超过90%，具有绿色低碳的显著优势。线性α-烯烃是生产高端聚烯烃、PAO润滑油、洗涤剂醇等的关键原料，目前我国对外依赖严重。 |

| **转化状态** | 已完成工业单管试验，正在开展万吨级工艺包设计 |

**技术优势**：
- α-烯烃选择性和时空产率高
- 碳排放极低（CO₂减排>90%）
- 稳定性好
- 已完成工业单管试验验证

**市场背景**：
- 线性α-烯烃是"卡脖子"化学品，国内供应严重不足
- 下游高端材料产业升级迫切需要
- 市场规模超百亿美元

**技术优势**：
- 有效压窄激光器线宽
