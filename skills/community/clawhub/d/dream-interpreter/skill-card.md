## Known Risks and Mitigations: <br>
Risk: The skill provides generic dream-reference text that could be mistaken for medical or psychological advice. <br>
Mitigation: Treat the output as general reference material and verify decisions with qualified sources or professionals where appropriate. <br>
Risk: The release includes a bash script that executes local command logic. <br>
Mitigation: Review the script before deployment and run it with normal user privileges; the reviewed evidence reports no credential, network, persistence, or system-change behavior. <br>
