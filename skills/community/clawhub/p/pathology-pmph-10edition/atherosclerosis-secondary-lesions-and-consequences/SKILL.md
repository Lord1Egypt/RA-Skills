# 动脉粥样硬化继发性病变及其后果

## 何时使用
- 用户需要了解动脉粥样硬化斑块不稳定或进展后的病理变化
- 用户咨询与动脉粥样硬化相关的急性心血管事件（如心肌梗死、脑卒中）的病理基础
- 用户要求解释影像学或尸检中观察到的血管异常与动脉粥样硬化的关系

## 如何执行
1. **确认原发病变**：明确存在动脉粥样硬化的基本病变（脂纹、纤维斑块、粥样斑块）。
2. **识别继发性病变类型**：根据临床或病理信息判断是否存在以下一种或多种继发改变：
   - 斑块内出血
   - 血栓形成
   - 钙化
   - 动脉瘤形成
   - 斑块破裂或溃疡
3. **关联临床后果**：将识别出的继发性病变与可能的临床结局联系起来，例如：
   - 血栓形成 → 急性缺血事件（心梗、脑梗）
   - 动脉瘤 → 破裂出血风险
   - 严重狭窄 → 慢性器官缺血
4. **提供机制简述**：简要说明继发性病变如何从原始粥样斑块发展而来，并强调其对预后的影响。
