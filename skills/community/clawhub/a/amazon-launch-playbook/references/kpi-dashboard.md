# 亚马逊新品期KPI关注看板

新品期的90天是建立竞争优势的关键窗口。本看板帮助你聚焦最重要的指标，在繁杂的数据中快速发现问题、把握机会。

---

## 看板设计原则

1. **简洁至上**：每个阶段只关注3-5个核心指标
2. **阈值清晰**：每个指标都有明确的判断标准
3. **行动导向**：每个异常都对应具体的调整建议
4. **阶段递进**：指标优先级随阶段推进而变化

---

## 第一阶段看板：Day 1-14（基建与测词期）

### 核心关注指标

| 指标 | 理想值 | 警戒线 | 关注原因 |
|------|--------|--------|----------|
| 页面访问量 | > 500次/周 | < 200次/周 | 流量基础决定转化数量 |
| 转化率（CVR） | 10-15% | < 8% | 反映Listing承接能力 |
| ACOS | 40-60% | > 70% | 广告效率的初步判断 |
| Vine评论 | 3-5条 | 0条 | 真实评价建立信任 |
| BSR排名 | 稳步上升 | 停滞或下滑 | 市场接受度信号 |

### 阶段目标

**必须完成**：
- [ ] 积累第一波Vine评论（至少3条）
- [ ] 自动广告数据样本 > 5000次展示
- [ ] 找出2-3个高效关键词
- [ ] 确认Listing基本转化能力

**理想达成**：
- [ ] 评论数达到10条
- [ ] CVR稳定在10%以上
- [ ] BSR进入品类前1000

### 指标解读与行动

**指标1：页面访问量（Sessions）**

反映Listing获得的自然+广告流量总和。

| 访问量级别 | 诊断 | 建议行动 |
|------------|------|----------|
| < 100/周 | 流量严重不足 | 检查索引词、优化广告、考虑站外引流 |
| 100-500/周 | 流量偏低 | 增加广告预算、优化标题埋词 |
| 500-1000/周 | 流量正常 | 维持现状，专注转化优化 |
| > 1000/周 | 流量良好 | 关注转化率，准备扩量 |

**指标2：转化率（CVR）**

衡量流量转化为订单的效率。

| CVR | 诊断 | 建议行动 |
|-----|------|----------|
| < 5% | 严重问题 | 立即检查Listing、价格、评论 |
| 5-8% | 存在问题 | 优化五点描述、主图、考虑降价 |
| 8-12% | 可接受 | 持续观察，优化细节 |
| 12-20% | 优秀 | 保持现状，准备扩量 |
| > 20% | 极佳 | 增加广告预算，快速抢占流量 |

**指标3：ACOS（广告成本销售比）**

衡量广告投入与产出的效率。

| ACOS | 诊断 | 建议行动 |
|------|------|----------|
| > 70% | 严重亏损 | 暂停低效广告、优化关键词 |
| 50-70% | 亏损但可接受 | 降低Bid、优化Listing |
| 30-50% | 可接受范围 | 新品期正常，持续优化 |
| 15-30% | 良好 | 保持或略微提价 |
| < 15% | 极佳 | 可增加预算抢占流量 |

**指标4：Vine评论**

Vine评论有"Verified Purchase"标识，可信度高，对转化率影响大。

| 评论数 | 状态 | 建议行动 |
|--------|------|----------|
| 0条 | 未出评论 | 确认Vine计划已注册，耐心等待 |
| 1-2条 | 偏少 | 跟进Vine计划，确保产品已送达 |
| 3-5条 | 基础达标 | 关注评论内容，回应差评 |
| 6-10条 | 良好 | 好评可以适当引导 |
| > 10条 | 优秀 | 可考虑降低Coupon力度 |

## 第二阶段看板：Day 15-45（扩张期）

### 核心关注指标

| 指标 | 理想值 | 警戒线 | 关注原因 |
|------|--------|--------|----------|
| 订单量 | 周环比增长20%+ | 持平或下降 | 增长动能判断 |
| TACOS | < 25% | > 35% | 整体广告效率 |
| 评论增长率 | +5-10条/周 | < 3条/周 | 口碑积累速度 |
| 自然排名 | 稳步上升 | 停滞 | 算法认可度 |
| ACOS趋势 | 下降 | 上升 | 优化效果 |

### 阶段目标

**必须完成**：
- [ ] 日均订单量达到目标（基于品类均值设定）
- [ ] 评论数达到20条
- [ ] 找到3-5个高效广告关键词
- [ ] TACOS开始呈下降趋势

**理想达成**：
- [ ] 核心关键词进入搜索结果前3页
- [ ] BSR进入品类前500
- [ ] ACOS降至40%以下
- [ ] 评论数达到30条

### 指标解读与行动

**指标1：订单量增长率**

反映新品的市场接受度和推广效果。

| 周环比增长 | 诊断 | 建议行动 |
|------------|------|----------|
| > 30% | 增长强劲 | 增加广告预算，补货准备 |
| 10-30% | 正常增长 | 维持当前策略 |
| 0-10% | 增长放缓 | 检查同行动态，优化Listing |
| 持平 | 停滞 | 评估是否需要大幅调整 |
| 下降 | 危险信号 | 立即分析原因，检查库存/差评 |

**指标2：TACOS（总广告成本占销售额比）**

比ACOS更能反映整体广告效率，因为包含自然订单的分摊。

| TACOS | 诊断 | 建议行动 |
|-------|------|----------|
| > 35% | 过高 | 审视整体广告策略，考虑收缩 |
| 25-35% | 偏高 | 优化广告效率，降低低效花费 |
| 15-25% | 健康 | 维持当前策略 |
| 10-15% | 优秀 | 可考虑品牌建设投入 |
| < 10% | 极佳 | 评估是否需要增加市场份额 |

**指标3：自然订单占比**

反映Listing的自然排名能力和算法认可度。

| 自然占比 | 诊断 | 建议行动 |
|----------|------|----------|
| < 30% | 依赖广告 | 优化自然排名词，降低广告依赖 |
| 30-50% | 逐步改善 | 持续优化Listing和评论 |
| 50-70% | 健康 | 继续保持 |
| > 70% | 非常健康 | 可适当收缩广告，集中高效词 |

**指标4：评论星级分布**

反映产品质量和买家满意度。

| 平均星级 | 诊断 | 建议行动 |
|----------|------|----------|
| < 4.0 | 严重问题 | 立即排查产品质量/包装问题 |
| 4.0-4.2 | 存在问题 | 主动联系差评买家，改进产品 |
| 4.2-4.5 | 可接受 | 关注差评内容，逐步改进 |
| 4.5-4.8 | 良好 | 保持现状 |
| > 4.8 | 优秀 | 维持，谨慎处理任何差评 |

## 第三阶段看板：Day 46-90（稳定与优化期）

### 核心关注指标

| 指标 | 理想值 | 警戒线 | 关注原因 |
|------|--------|--------|----------|
| ACOS | 20-30% | > 40% | 追求广告盈利 |
| BSR排名 | 品类前100 | 品类500+ | 市场地位 |
| TACOS | < 15% | > 20% | 整体效率 |
| 自然流量占比 | > 60% | < 40% | 算法认可 |
| 利润率 | > 20% | < 15% | 商业可持续 |

### 阶段目标

**必须完成**：
- [ ] ACOS降至30%以下并稳定
- [ ] BSR进入品类前100
- [ ] 自然订单占比超过50%
- [ ] 评论数达到50条以上

**理想达成**：
- [ ] ACOS降至20-25%
- [ ] 核心关键词进入首页
- [ ] BSR进入品类前50
- [ ] 建立品牌旗舰店

### 指标解读与行动

**指标1：核心关键词排名**

关键词的自然搜索排名直接影响流量成本。

| 排名位置 | 诊断 | 建议行动 |
|----------|------|----------|
| 第1页前3名 | 顶级位置 | 保持，关注同行 |
| 第1页4-10名 | 良好位置 | 逐步提升Bid争取前3 |
| 第1页后段 | 可见位置 | 增加Review，优化转化 |
| 第2页 | 边缘位置 | 增加Review，降低价格 |
| 第3页及以后 | 危险位置 | 全面检视Listing和广告 |

**指标2：类目BSR排名**

BSR是亚马逊最核心的排名指标。

| BSR排名 | 类目定位 | 建议行动 |
|---------|----------|----------|
| 前50 | 顶级爆款 | 维护地位，防备同行 |
| 50-100 | 潜力爆款 | 争取冲击前50 |
| 100-500 | 成长中 | 优化关键词和Review |
| 500-1000 | 中游产品 | 分析差距，加大推广 |
| 1000+ | 需要努力 | 全面检视产品定位 |

**指标3：退货率**

高退货率会严重影响利润和账户健康。

| 退货率 | 诊断 | 建议行动 |
|--------|------|----------|
| > 10% | 严重问题 | 立即排查产品原因，暂停广告 |
| 5-10% | 偏高 | 联系买家了解原因，改进产品 |
| 3-5% | 正常范围 | 持续关注 |
| < 3% | 优秀 | 保持 |

## 异常提醒机制

### 红色提醒（需立即处理）

| 提醒信号 | 立即行动 |
|----------|----------|
| CVR单周下降 > 30% | 检查是否有差评、价格变动、同行促销 |
| 退货率 > 10% | 暂停广告，排查产品质量 |
| BSR排名下滑 > 50% | 检查库存是否断货、是否有差评 |
| ACOS突然上升 > 20% | 暂停低效广告，分析关键词 |

### 黄色提醒（需密切关注）

| 提醒信号 | 建议行动 |
|----------|----------|
| CVR连续2周下降 | 优化Listing五点或考虑降价 |
| 评论增长放缓 | 跟进Vine计划，考虑增加促销 |
| ACOS缓慢上升 | 优化Bid，调整关键词 |
| 库存可售天数 < 14天 | 立即启动补货 |

### 绿色信号（可以推进）

| 积极信号 | 把握机会 |
|----------|----------|
| CVR > 15% 持续2周 | 增加广告预算 |
| ACOS下降趋势明显 | 加大高效词投放 |
| BSR稳步上升 | 准备旺季备货 |
| 评论达到50+且4.5星+ | 可以提价测试 |

## 结语

数据是运营的眼睛，但数据本身不会告诉你该怎么做。本看板提供的是判断基准，具体决策需要结合产品特性、市场环境和个人资源综合判断。

**记住**：
- 数据需要结合趋势看，而非单日波动
- 不同品类有不同的基准值，参考同行数据
- 指标之间相互关联，解决问题要抓主要矛盾
- 新品期追求的是正确方向，不必苛责完美数字
