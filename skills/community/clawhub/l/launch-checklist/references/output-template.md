# Launch Runbook — Output Template

Use this structure when producing a launch checklist. Keep checkboxes, owners, and deadlines on every task.

## Header

- **Product / SKU:** {name + SKU}
- **Category:** {category}
- **Live date (T-0):** {date}
- **Price / promo:** {list price, launch price, coupon}
- **Inventory on hand:** {units} ({days of cover at forecast peak})
- **Fulfillment:** {seller / 3PL / FBT}
- **Readiness verdict:** {GO / NO-GO} — {one-line reason}

## Blockers (resolve first)

- [ ] {Any unknown context or Weak Quick-Reference row, with the mitigation task}
