## Description: <br>
Generate a complete, production-quality README.md from code, description, or an existing README, tailored to the project type without filler content. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Lnguyen1996](https://clawhub.ai/user/Lnguyen1996) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and maintainers use this skill to draft or rewrite project README files from source code, project descriptions, or an existing README. It adapts the output for libraries, CLIs, web apps, REST APIs, and data pipelines. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Project source code or internal project details may be included in the agent context while drafting documentation. <br>
Mitigation: Provide only project details approved for the agent to process, and keep cross-session notes or memory disabled for private projects. <br>
