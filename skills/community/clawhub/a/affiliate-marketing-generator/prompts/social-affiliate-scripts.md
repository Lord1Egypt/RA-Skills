# Prompt: Social + YouTube Affiliate Content Scripts

You are a short-form content writer and YouTube ad copywriter. Create a complete social media kit for promoting an affiliate product — designed to feel native to each platform, not like ads.

## Inputs
- Product: [PRODUCT_NAME]
- Target platform: [TARGET_PLATFORM] (or "all" for full kit)
- Hook angle: [HOOK_ANGLE]
- Target audience: [TARGET_AUDIENCE]
- Key proof point: [KEY_PROOF_POINT]
- Affiliate disclosure: [AFFILIATE_DISCLOSURE]

## Output: 4 Platform Scripts

---

### 1. Instagram Reel Script (60 seconds)

**[HOOK_ANGLE]** — first 3 seconds must stop the scroll.

**Structure:**
- **0-3 sec (Hook):** Visual hook + spoken hook. Pattern interrupt. Ask a question or make a bold claim.
  - Example format: "If you're a [TARGET_AUDIENCE] still doing X manually, watch this."
- **3-15 sec (Problem):** Name the specific pain. Use the exact language [TARGET_AUDIENCE] uses to describe this problem.
- **15-45 sec (Demo/Proof):** Show [PRODUCT_NAME] solving the problem. Use [KEY_PROOF_POINT] as the centerpiece. Be specific — numbers, names, results. Not vague benefits.
- **45-55 sec (CTA):** "Link in bio to try it free" or equivalent. Mention [YOUR_BONUS] if applicable.
- **55-60 sec (Disclosure):** "[AFFILIATE_DISCLOSURE]" — spoken or on-screen text.

**Caption (150-200 characters):** Hook + CTA + disclosure hashtag + #ad

**Hashtags (10-15):** [relevant to [TARGET_AUDIENCE] and [PRODUCT_NAME] category]

---

### 2. TikTok Script (30 seconds)

TikTok requires an even faster hook. Assume 3 million similar videos exist.

**Structure:**
- **0-2 sec (Hook):** Text overlay + first word. Use a disruption pattern: question, controversy, or unexpected number.
  - Example: "I replaced 3 apps with this. Here's what happened."
- **2-10 sec (Setup):** Who you are + why this matters to [TARGET_AUDIENCE]. No fluff.
- **10-22 sec (Proof):** [KEY_PROOF_POINT] delivered as screen recording, before/after, or reaction. Fastest path to credibility.
- **22-28 sec (CTA):** Link in bio. Mention the free trial or lowest price tier.
- **28-30 sec (Disclosure):** "#ad" overlay.

**Caption (50-100 characters):** Ultra-short. One hook sentence + #ad.

**TikTok disclosure note:** Add "#ad" or "#sponsored" to the caption AND as an on-screen overlay. Platform policy requires both.

---

### 3. YouTube 30-Second Pre-Roll Ad Script

The viewer CAN skip after 5 seconds. The first 5 seconds must make them choose to keep watching.

**Structure:**
- **0-5 sec (Skip-Proof Hook):** Name a problem [TARGET_AUDIENCE] desperately wants solved. Or show the result first. The viewer must think "wait, what is that?"
  - Format: "Are you a [TARGET_AUDIENCE] who still [painful inefficiency]? Keep watching — I'll show you what fixed it in 30 seconds."
- **5-15 sec (Product Intro):** Name [PRODUCT_NAME] + [KEY_PROOF_POINT] in one sentence. No jargon.
- **15-25 sec (Proof Burst):** 2-3 rapid-fire specific benefits. Use numbers. "Cut X from 4 hours to 20 minutes. Replaced Y and Z tools. Starts at $[PRICE]."
- **25-30 sec (CTA):** "Click below to try it free." Be specific about the offer.

**End screen text:** [PRODUCT_NAME] — [Single benefit line] — [CTA button text]

**Compliance note:** YouTube requires disclosure in the video description. Suggested language: "[AFFILIATE_DISCLOSURE]"

---

### 4. Twitter/X Thread (5 tweets)

**Tweet 1 (Hook):** One bold, specific claim. [KEY_PROOF_POINT] stated bluntly. Should be retweetable standalone.

**Tweet 2 (Problem):** The pain. Write it like [TARGET_AUDIENCE] would write it in a frustrated DM. Specific and relatable.

**Tweet 3 (Product + Proof):** Introduce [PRODUCT_NAME]. Name one feature. Give a before/after or specific number. Link to your review post (not directly to the affiliate link in Tweet 3 — warm them up first).

**Tweet 4 (Objection Handling):** Address the most common hesitation ("But isn't it expensive?" / "But I already have X"). One sentence objection, two sentence reframe.

**Tweet 5 (CTA + Disclosure):** Direct link to your affiliate review or landing page. State the offer. "[AFFILIATE_DISCLOSURE]" — Twitter/X requires disclosure near the affiliate link.

**Thread format notes:**
- Each tweet should be under 240 characters
- Number each tweet (1/5, 2/5, etc.)
- Add relevant emoji sparingly — 1 per tweet max
- Reply to Tweet 1 with the rest of the thread for maximum reach

---

## Platform Disclosure Reference

| Platform | Required disclosure format |
|----------|---------------------------|
| Instagram | #ad in caption + spoken or on-screen |
| TikTok | #ad in caption + on-screen overlay |
| YouTube | Video description + spoken mention |
| Twitter/X | Clear disclosure near the affiliate link |
| Email | Disclosure at top OR bottom of every email with affiliate link |
