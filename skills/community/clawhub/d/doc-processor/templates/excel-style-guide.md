# Excel 表格样式规范

## 📐 基础设置

### 列宽
- 文本列：15-25 字符
- 数字列：10-15 字符
- 日期列：12 字符
- 自动调整：双击列边界

### 行高
- 表头：25pt
- 数据行：18-22pt
- 标题行：30pt

## 🎨 单元格样式

### 表头样式
| 属性 | 值 |
|------|-----|
| 字体 | 黑体 |
| 字号 | 12pt |
| 加粗 | 是 |
| 底色 | #2E75B6 |
| 文字颜色 | 白色 |
| 对齐 | 居中 |

### 数据行样式
| 属性 | 值 |
|------|-----|
| 字体 | 宋体 |
| 字号 | 11pt |
| 底色 | 白色 |
| 文字颜色 | 黑色 |
| 边框 | 细线 |

### 强调行样式
| 属性 | 值 |
|------|-----|
| 底色 | #E8F4F8 |
| 用途 | 汇总行/重要数据 |

## 📊 数字格式

| 类型 | 格式 | 示例 |
|------|------|------|
| 整数 | 0 | 1,234 |
| 小数 | 0.00 | 1,234.56 |
| 百分比 | 0% | 85% |
| 货币 | ¥#,##0.00 | ¥1,234.56 |
| 日期 | yyyy-mm-dd | 2026-03-25 |

## 🎯 使用建议

### 表头设计
- 简洁明了
- 避免过长文字
- 可使用换行（Alt+Enter）

### 数据对齐
- 文字：左对齐
- 数字：右对齐
- 日期：居中
- 表头：居中

### 条件格式
- 慎用颜色
- 突出异常值
- 保持整体协调
