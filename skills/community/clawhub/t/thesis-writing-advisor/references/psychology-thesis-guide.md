# Psychology Undergraduate Thesis Guide

Use this reference when a user asks for a concrete psychology undergraduate thesis topic, proposal, opening report, or writing guide.

## Recommended Output Shape

If the user asks for "相关论文", "样例", "参考文献怎么用", or wants examples to imitate, first read `sample-papers.md`.

If the user wants the answer to become a draftable thesis plan, read `chapter-plans.md` and include a "毕业论文可续写章节规划" table after the topic and method guidance.

Start with a clear recommendation:

> 最推荐题目：《……》
>
> 论文类型：……
>
> 推荐理由：对象明确、变量/理论清楚、方法难度适中、适合本科毕业论文。

Then provide:

1. 选题定位
2. 可选题目
3. 研究问题或假设
4. 研究方法
5. 研究对象与材料
6. 数据分析或评价设想
7. 论文目录
8. 毕业论文可续写章节规划
9. 每章写作要点
10. 可行性、创新点、局限
11. 开题报告可直接使用的表述

## Common Paper Types

### Questionnaire Study

Use when the user can collect simple data.

Method labels:

- 问卷调查法
- 横断面研究
- 相关研究法
- 定量研究法

Simple structure:

- Population: 大学生、青少年、教师、护理学生、实习生等
- Predictor: 压力知觉、手机依赖、睡眠质量、社会支持、心理韧性、自尊、完美主义等
- Outcome: 焦虑、抑郁、学习倦怠、主观幸福感、孤独感、人际关系困扰等
- Optional mediator/moderator only if the user wants a more advanced design

Basic analyses:

- 描述统计
- 信度分析
- 相关分析
- t 检验或方差分析
- 回归分析

Template wording:

> 本研究采用问卷调查法，以……为研究对象，使用……量表和……量表收集数据，通过描述统计、信度分析、相关分析和回归分析探讨……与……之间的关系。

### Literature Review

Use when the user cannot collect data and the school allows non-empirical work.

Method labels:

- 文献研究法
- 叙事性文献综述
- 系统综述
- 范围综述

Core reminder:

- Do not merely stack summaries.
- Organize by concept, mechanism, findings, disagreements, limitations, and future directions.
- If using systematic review language, specify databases, keywords, inclusion/exclusion criteria, and screening flow.

### Qualitative Study

Use when the user can collect interviews, focus groups, observations, reflective texts, documents, or case materials, and the research purpose is to understand experience, meaning, process, context, or a bounded case.

Method labels:

- 质性研究法
- 访谈法
- 焦点小组法
- 观察法
- 文本/文件分析法
- 个案研究法
- 现象学研究
- 扎根理论，only when the user genuinely aims to build a theory from data

Research question wording:

- "……群体如何体验……？"
- "……在……过程中的主要感受和需要是什么？"
- "……案例中……问题的形成过程和应对方式如何？"
- "……文本中呈现出哪些主题和意义模式？"

Template wording:

> 本研究采用质性研究方法，以……为研究对象，通过半结构式访谈/焦点小组/观察/文本分析收集资料，并运用主题分析/编码分析/持续比较法归纳……的主要经验、意义和影响因素。

Key cautions:

- Do not write causal hypotheses as if it were a survey study.
- Do not promise statistical generalization.
- Include informed consent, confidentiality, recording permission, transcription, and anonymization.
- Include trustworthiness strategies such as triangulation, participant verification, thick description, reflexive notes, and audit trail.

For detailed qualitative method selection and wording, read `qualitative-methods.md`.

### Mixed Methods Study

Use when the research question needs both numeric trend and deep explanation.

Common patterns:

- 先问卷、后访谈：先发现变量关系，再解释原因。
- 先访谈、后问卷：先探索主题，再形成问卷或验证范围。
- 并行设计：问卷与访谈同时进行，再整合解释。

Template wording:

> 本研究采用混合研究方法，先通过问卷调查了解……的总体情况，再通过半结构式访谈进一步解释……的形成原因和具体表现，最后综合定量与质性结果提出实践建议。

Undergraduate caution:

- Mixed methods workload is high. Recommend it only when the user has enough time, participants, and analysis ability.

### Intervention-Scheme Design

Use when the user wants an applied paper but will not implement the intervention.

Method labels:

- 文献研究法
- 理论建构法
- 方案设计法

Safe thesis title forms:

- 《基于认知行为疗法的大学生考试焦虑团体辅导方案设计》
- 《正念训练在大学生压力管理中的应用方案设计》
- 《团体辅导视角下新生适应问题干预方案设计》
- 《基于积极心理学的大学生主观幸福感提升方案设计》

Avoid title forms:

- "干预效果研究" unless the user will implement and evaluate the intervention.
- "实验研究" unless there is an experimental or quasi-experimental design.

Method wording:

> 本研究采用文献研究法、理论建构法和方案设计法。首先检索并梳理……相关研究；其次基于……理论构建……的发生与维持机制；最后参考心理干预方案设计与报告规范，从干预对象、干预目标、干预内容、实施流程和评价设想等方面设计……方案。

Core sections:

- 适用对象：例如轻中度考试焦虑大学生
- 不适用对象：严重焦虑、严重抑郁、自伤风险、精神障碍急性发作或需临床治疗者
- 团体规模：8-12人
- 频次：每周1次，共6-8次
- 时长：每次60-120分钟，常用90分钟
- 实施者：心理健康教育教师、心理咨询师或受过团体辅导训练者
- 伦理：自愿参加、知情同意、隐私保护、可退出、风险转介

Evaluation wording:

> 本方案若在后续研究中实施，可采用实验组-对照组前后测设计，对干预效果进行验证。评价指标可包括……量表得分、出勤率、作业完成率、团体满意度和追踪测量结果。

Limitation wording:

> 本研究仅完成干预方案设计，未实际开展干预，因此不能直接证明方案的有效性。后续研究需通过实验或准实验设计进一步检验其可行性和干预效果。

## CBT Exam Anxiety Scheme Template

Recommended topic:

> 《基于认知行为疗法的大学生考试焦虑团体辅导方案设计》

Theory model:

> 考试情境 -> 自动化负性思维 -> 焦虑情绪和生理唤醒 -> 回避复习、拖延、反复确认、临场紧张 -> 考试体验变差 -> 强化"我不行""我会失败"等核心信念。

Common automatic thoughts:

- 我肯定考不好。
- 这次失败就完了。
- 别人都比我强。
- 如果紧张，就说明我没有能力。
- 我必须考得很好，否则就是失败。

Intervention techniques:

- 心理教育
- 自动化思维识别
- 认知重构
- 行为实验
- 腹式呼吸
- 渐进性肌肉放松
- 考试情境模拟
- 复习计划与应试策略
- 复发预防

Six-session design:

| 次数 | 主题 | 目标 | 主要内容 |
|---|---|---|---|
| 第1次 | 认识考试焦虑 | 建立团体关系，理解考试焦虑 | 破冰、团体契约、考试焦虑表现讲解、CBT模型介绍 |
| 第2次 | 识别自动化思维 | 发现焦虑背后的想法 | 考试焦虑事件回顾、自动化思维记录、区分事实与想法 |
| 第3次 | 认知重构 | 调整灾难化和绝对化认知 | 识别认知偏差、证据分析、替代性想法训练 |
| 第4次 | 放松与情绪调节 | 降低生理唤醒 | 腹式呼吸、渐进性肌肉放松、考前放松练习 |
| 第5次 | 应试行为训练 | 减少回避，提高应对感 | 制定复习计划、模拟考试情境、行为实验 |
| 第6次 | 巩固与预防复发 | 形成个人应对方案 | 总结变化、制定考前应对卡、团体反馈与结束 |

## Opening Report Outline

Use this outline for "开题报告":

1. 选题背景与研究意义
2. 国内外研究现状
3. 研究目标与研究内容
4. 研究方法与技术路线
5. 论文创新点
6. 可行性分析
7. 研究进度安排
8. 预期成果
9. 参考文献

Technical route for intervention design:

> 文献检索 -> 研究现状梳理 -> 理论模型建构 -> 干预目标确定 -> 干预活动设计 -> 实施条件说明 -> 评价设想与局限分析。

## Full Thesis Outline for Intervention Design

Use this chapter structure:

1. 绪论
   - 研究背景
   - 研究意义
   - 研究思路
2. 文献综述
   - 核心概念
   - 目标问题的表现与影响因素
   - 理论基础
   - 现有干预研究
3. 研究设计
   - 研究目标
   - 研究方法
   - 技术路线
4. 理论模型建构
   - 发生机制
   - 维持机制
   - 干预靶点
5. 干预方案设计
   - 方案名称
   - 适用对象
   - 干预目标
   - 干预原则
   - 活动流程
   - 实施条件
   - 伦理与风险处理
6. 评价设想
   - 未来研究设计
   - 评价工具
   - 过程评价
   - 追踪评价
7. 讨论
   - 方案特点
   - 实践价值
   - 局限与展望
8. 结论

## Topic Generation Patterns

When the user gives "theme + influencing factors", generate titles using one of these patterns:

- 《……与……的关系研究》
- 《……对……的影响：……的中介作用》
- 《……视角下……的影响因素研究》
- 《基于……理论的……干预方案设计》
- 《……群体……问题的心理机制及干预方案设计》
- 《……领域……研究综述》

Examples:

- 主题：大学生焦虑；影响因素：睡眠质量、手机依赖  
  推荐：《大学生睡眠质量、手机依赖与焦虑水平的关系研究》
- 主题：考试焦虑；方法限制：不收集数据  
  推荐：《基于认知行为疗法的大学生考试焦虑团体辅导方案设计》
- 主题：新生适应；影响因素：社会支持、自我效能感  
  推荐：《大学新生社会支持、自我效能感与学校适应的关系研究》

## Quality Checks

Before finalizing advice, check:

- Is the topic too broad? Narrow by population, variable, and method.
- Does the title match the actual method?
- Are conclusions overstated?
- Are scales, participants, and analyses feasible for an undergraduate?
- Does the plan address ethics and participant safety?
- If non-empirical, has the response warned that school approval is needed?
