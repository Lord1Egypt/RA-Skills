# Skill: chuwi-minibook

迷你笔记本系统监控工具 — CPU/内存/磁盘/电池/温度检测

## Functionality
- Full system info report (platform, CPU, memory, disk, battery, temperature)
- Quick overview mode for at-a-glance status
- Dedicated battery status check
- Uses psutil for cross-platform hardware detection

## Invocation
- `run("")` — full system report
- `run("quick")` — one-line quick overview
- `run("battery")` — battery status only

## Trigger phrases
- "系统信息" / "system info" / "chuwi" / "迷你笔记本" / "监控"
