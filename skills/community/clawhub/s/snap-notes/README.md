## 🛠️ 如何配置 AI (开启智能模式)

SnapNotes支持所有兼容 OpenAI 格式的接口（如 OpenAI, Gemini, Claude, Kimi, DeepSeek, Ickey Airouter 等）。

1. **定位配置文件**：
   找到 Workspace 目录下的文件：`memory/snapnotes/config.json`
2. **填写参数**：
   ```json
   {
     "api_base": "https://你的接口地址/v1",
     "api_key": "sk-你的私有密钥",
     "model": "你的选用模型名称"
   }
   ```
   *推荐使用 Gemini 2.0 Flash 或 GPT-4o-mini 以获得最佳质检性价比。*
