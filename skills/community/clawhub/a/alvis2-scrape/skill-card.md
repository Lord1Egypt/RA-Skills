## Use Case: <br>
Developers and agents use this skill to plan compliant web scraping workflows, including robots.txt checks, rate limiting, data minimization, and optional managed extraction through SkillBoss API Hub. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Scraping protected, personal, copyrighted, or terms-restricted content can create legal, privacy, or compliance exposure. <br>
Mitigation: Check robots.txt and terms before scraping, prefer official APIs, avoid login-protected content without authorization, minimize personal data collection, and keep an audit trail. <br>
Risk: Managed scraping depends on an external SkillBoss API and a sensitive API key. <br>
Mitigation: Use a dedicated or limited SkillBoss API key where possible, monitor usage and billing, and do not send confidential targets, login-protected content, or regulated personal data unless provider terms have been reviewed. <br>
