# Output Patterns

## Verdict First

Start with:

- `推荐`: 可以买 / 谨慎买 / 先别买 / 需要补证据
- one sentence reason

## Evidence

Capture:

- 商品/SKU
- 店铺类型
- 可见价格/优惠
- 评论和销量
- 退换/保障

## Risks

Mention only meaningful risks:

- SKU mismatch
- seller quality
- fake or conditional discount
- review concerns
- return friction
- warranty/invoice gap
- shipping or stock uncertainty

## Final Checks

End with user-only checks before payment:

- selected SKU
- final payable amount
- address-based delivery
- coupon eligibility
- stock
- return policy
- invoice/warranty
- payment
