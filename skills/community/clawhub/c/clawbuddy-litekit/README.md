# ClawBuddy LiteKit — ClawHub Skill

Listing for <https://clawhub.ai/stevekaplanai/clawbuddy-litekit>.

## Install

```bash
clawhub install stevekaplanai/clawbuddy-litekit
```

Then ask Claude: *"Open my ClawBuddy dashboard"* or *"Connect my OpenClaw Gateway"*.

## What's inside

- `SKILL.md` — agent-facing instructions and triggers
- `scripts/open-dashboard.sh` — opens the live dashboard (root or `/setup`)
- `scripts/status.sh` — checks deployed Gateway connection status
- `references/openclaw.md` — endpoint + secret reference
