# LCH诊断金标准

## 适用条件
- 患者存在LCH相关典型临床表现（如皮疹、骨损害、肝脾肿大、尿崩症等）
- 已获取疑似病灶的组织活检标本（常见部位：皮肤、骨、淋巴结）

> **注意**：本技能不适用于无组织标本的初步筛查；非典型病灶可能出现假阴性结果。

## 执行步骤

1. **获取组织标本**：对疑似病灶（如皮肤、骨、淋巴结）进行活检。
2. **光镜观察**：检查是否存在大量朗格汉斯细胞浸润，并伴有嗜酸性粒细胞、巨噬细胞和淋巴细胞不同程度增生；注意病程后期可能出现黄色瘤样变、纤维化、局灶坏死、出血及含铁血黄素颗粒的巨噬细胞。
3. **免疫组化染色**：检测以下标志物表达情况：
   - CD1a
   - CD207（langerin）
   - S-100蛋白
   - CD68
   - ATP酶
   - α-D-甘露糖酶
   - 花生凝集素
4. **判定金标准**：若 **CD1a 和/或 CD207 阳性**，即可确诊为LCH。
5. **辅助确认（可选）**：电镜下发现Birbeck颗粒可进一步支持诊断（其意义等同于CD207阳性）。
6. **补充检测（可选）**：进行BRAF V600E基因突变检测；若阳性可辅助诊断，但非确诊必需。

## 结果解读
- **确诊LCH**：CD1a 和/或 CD207 免疫组化阳性。
- **排除LCH**：CD1a 与 CD207 均为阴性，需考虑其他组织细胞疾病。

> 当触发条件为“患者出现LCH相关多系统症状”时，立即启动本诊断流程。
