# 公文套语库 (Language Patterns)

公文语言**庄重、简洁、规范**，大量使用固定套语。掌握套语 = 掌握公文写作 70%。

按"开头—主体—结尾"三段组织。

---

## 一、开头套语（依据 + 目的）

公文几乎都需要交代"依据"和"目的"，开头就是这两件事。

### 1.1 引述上级文件 / 会议精神

- 根据《XX法》第X条规定 / 根据……的有关规定
- 按照……的部署 / 按照……要求
- 为深入贯彻落实……精神
- 为认真贯彻……决策部署
- 遵照……指示精神
- 依据……（适用法律法规依据）

### 1.2 说明背景 / 形势

- 当前，……
- 近期，……
- 随着……（情况）的不断发展
- 鉴于……（特殊情况）
- 针对……（突出问题）
- 为进一步……（目的）

### 1.3 连接到主体（过渡）

- 现就……有关事项通知如下：（通知）
- 现将……报告如下：（报告）
- 现请示如下：（请示）
- 现批复如下：（批复）
- 现就……提出如下意见：（意见）
- 经研究，现就……函告如下：（函）

**完整开头示例（通知）：**
> 为深入贯彻落实党的二十大精神，进一步加强基层党组织建设，根据《中共中央关于…的意见》（中发〔2026〕X号）和省委有关部署，现就开展2026年度党员教育培训工作有关事项通知如下：

## 二、主体套语

### 2.1 分条列项（最常用结构）

```
一、（一级标题：黑体三号）
（一）（二级标题：楷体三号加粗）
1.（三级标题：仿宋三号加粗）
(1)（四级标题：仿宋三号）
```

**示例：**
```
一、总体要求
（一）指导思想
   坚持以……为指导……
（二）基本原则
   1. 坚持……
   2. 坚持……
二、主要任务
（一）……
（二）……
三、保障措施
（一）加强组织领导
（二）强化责任落实
（三）严格督促检查
```

### 2.2 转折与递进

- 同时，……
- 此外，……
- 与此同时，……
- 在此基础上，……
- 进一步……
- 更为重要的是，……
- 需要特别指出的是，……
- 必须看到，……（话锋一转，常引出问题）
- 但也要看到，……

### 2.3 强调与要求

- 各地各部门要……
- 各级党委（党组）要切实……
- 必须……
- 要切实做到……
- 务必……
- 严禁……
- 不得……
- 应当……
- 坚决……
- 全面……
- 深入……
- 扎实……

### 2.4 列举数据 / 现象

- ……的有 X 个，占 X%
- 截至 X 年 X 月底，……
- 比上年同期增长 / 下降 X%
- 同比 / 环比……
- 主要表现在以下几个方面：
- 主要原因有三个：一是……；二是……；三是……

## 三、结尾套语（按文种）

**严格区分。混用结尾是公文常见错误。**

### 3.1 通知

- 请认真贯彻执行。
- 请各地各部门结合实际，认真组织实施。
- 请遵照执行。
- 特此通知。
- 以上工作，请认真做好，并将落实情况于 X 月 X 日前报 XX 部门。

### 3.2 请示 ⭐

**专属结尾，不能混用：**
- 妥否，请批示。
- 妥否，请审示。
- 以上请示当否，请批复。
- 以上请示妥否，请批示。
- 当否，请批复。

**禁忌**：请示不能用"特此通知" / "特此报告" / "此致敬礼"。

### 3.3 报告 ⭐

- 特此报告。
- 专此报告。
- 以上报告，请审阅。
- 以上情况，特此报告。

**禁忌**：报告不能用"请批示" / "请批复"——那是请示的语言。如果你想让上级批，那这就是请示，不是报告。

### 3.4 批复

- 此复。
- 特此批复。
- 以上批复，请遵照执行。

### 3.5 函

- 此致 / 顺致敬意（少用）
- 特此函告。
- 特此函复。
- 望函复。
- 盼复。
- 请研复。
- 请予办理。

**禁忌**：公文之"函"不用"此致敬礼"（那是私人信件用语）。

### 3.6 意见

- 各地各部门要切实加强组织领导，认真贯彻落实。
- 以上意见，请结合本地区本部门实际，认真组织实施。

### 3.7 通报

- 特此通报。
- 希各地各部门以此为鉴 / 引以为戒……
- 希望全体同志学习 XX 同志……（表彰类）

### 3.8 纪要

纪要一般不需要结尾套语，议定事项写完即结束。少数会加：

- 以上议定事项，请会议各方认真落实。

## 四、特殊位置套语

### 4.1 批复 / 复函的开头（引述）

- 你（厅 / 局 / 委 / 公司）《关于……的请示》（X 文〔2026〕X 号）收悉。经研究，现批复如下：
- 你（厅 / 局）X 年 X 月 X 日《关于……的函》收悉。经研究，函复如下：
- 你单位 X 年 X 月 X 日报来的《关于……的请示》已收悉，现批复如下：

### 4.2 转发性通知的开头

- 现将《XX 部关于……的通知》（X 文〔2026〕X 号）转发给你们，请认真贯彻执行。
- 经研究，同意 XX 部《关于……的意见》。现转发给你们，请结合实际，参照执行。

### 4.3 附件说明的位置

```
（正文最后一段结束）

附件：1. XXX 申报表
      2. YYY 实施方案
```

## 五、关键禁忌词

公文**避免**以下表达：

| 不要 | 用什么 |
|---|---|
| "前段时间" / "最近" | 具体日期或时段（"X 月 X 日至 X 日"） |
| "有关方面" / "有的人" | 具体单位 / 具体岗位 |
| "大概" / "差不多" / "可能" | 具体数据或"约 X 万元" |
| "我觉得" / "我认为" | "本机关认为" / 删去主观词 |
| "请大家……" | "请各地各部门……" |
| 感叹号"!" | 句号或换说法 |
| 问号"？" | （仅引用时用） |
| 网络词 ("点赞" "破防" "yyds") | 标准书面语 |
| "希望大家共同努力" | "希望各级各部门切实抓好落实" |
| 太长的句子 | 拆短，每句不超过 50 字为宜 |

## 六、动词搭配（增加公文味）

公文常用动词及搭配。掌握这些"公文感"立刻有：

| 行动 | 公文用法 |
|---|---|
| "搞" / "弄" | 开展 / 组织 / 部署 / 实施 |
| "看一下" | 审阅 / 研究 / 审核 / 审议 |
| "讨论" | 研究 / 审议 / 协商 |
| "做完了" | 完成 / 收官 / 结项 |
| "重视" | 高度重视 / 切实加强 |
| "下力气" | 着力推进 / 集中力量 |
| "改" | 整改 / 完善 / 优化 / 调整 |
| "说" | 强调 / 指出 / 提出 / 要求 |
| "保证" | 确保 / 保障 / 维护 |
| "解决" | 妥善处理 / 有效化解 |
| "出问题" | 出现纰漏 / 工作失误 / 失职失责 |
| "处罚" | 严肃问责 / 依规处理 |
| "好" | 显著成效 / 良好效果 / 积极进展 |

## 七、节奏与气势（高级技巧）

党政公文讲究"四字格"和"排比" — 写出"领导讲话感":

**四字格：**
- 总揽全局、协调各方
- 守土有责、守土尽责、守土担责
- 全面发力、多点突破、纵深推进
- 拉高标杆、奋勇争先、敢于碰硬

**排比 + 递进：**
- 既要……又要……还要……
- 是……是……更是……
- 不仅……而且……还……
- 一方面……另一方面……

**对偶：**
- 大事难事看担当，逆境顺境看胸襟
- 不忘初心，方得始终

⚠️ **慎用**：用过头会显得空洞。**企业公文 / 述职报告 适量用即可；不相隶属机关的函 不要用，显得做作。**

## 八、按文种快速参考卡

```
通知   → 开头"现就……通知如下" → 结尾"请认真贯彻执行" / "特此通知"
请示   → 开头"现请示如下"     → 结尾"妥否，请批示"
报告   → 开头"现将……报告如下" → 结尾"特此报告"（不要"请批示"）
批复   → 开头"你……请示收悉"   → 结尾"此复"
函     → 开头"为……函告如下"   → 结尾"特此函告" / "盼复"
意见   → 开头"现就……提出以下意见" → 结尾"请认真贯彻落实"
纪要   → 开头"X月X日，X主持召开……会议" → 结尾"议定事项列毕即止"
```
