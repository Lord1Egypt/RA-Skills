# Output Template — Lazada Optimization Audit

Deliver the audit in this structure. One report per listing (or per store for seller-score and expansion sections).

## 1. Snapshot

```
Listing: [product name + SKU/URL]
Market: [MY/TH/ID/PH/SG/VN]
Days live: [n]
Organic traffic trend: [daily visits, 30-day direction]
LQS band: [Poor / To Improve / Good / Excellent]
Seller score components: chat [x%] · on-time ship [x%] · return rate [x%] · rating [x.x]
```

## 2. Title Rebuild

```
Current:  [existing title]
Issues:   [stuffing / promo words / missing brand-model / length]
Rebuilt:  [Brand + Model + Key Spec + Category Keyword, 60–120 chars]
Rationale: [primary keyword choice and what was removed/why]
```

## 3. Attribute Gap Table

| Attribute | Status | Value to set | Filter impact |
|---|---|---|---|
| [name] | Missing/Filled | [value] | [which filtered searches this unlocks] |

End with: "X of Y attributes filled → target 100%."

## 4. Gallery & Video Plan

| Slot | Current | Replace with |
|---|---|---|
| Main | [state] | [white-background ≥800×800 spec] |
| 2–5 | [state] | [lifestyle shots — name the scenes] |
| 6 | [state] | [size/spec infographic] |
| 7 | [state] | [comparison or packaging] |
| Video | [present?] | [15–30s concept, one sentence] |

## 5. Seller Score Actions

For each slipping component: current value → target → specific action with owner and cadence (e.g., "Chat 78% → ≥95%: enable auto-reply, add 10 templated answers, assign 9am–9pm coverage").

## 6. Campaign Eligibility Map

| Campaign | Date | Entry threshold | Eligible now? | Action + deadline |
|---|---|---|---|---|
| [9.9 / 11.11 / category day] | [date] | [discount % off 30-day lowest, stock, voucher] | [Y/N + blocker] | [price/stock/voucher action, prep deadline] |

## 7. Localization Addendum (expansion audits only)

Per target market: title/keyword translation notes, local price benchmark vs. top-10 competitors, attribute/unit conversions, local campaign calendar entry points, local-language chat readiness.

## 8. Prioritized Action List

Numbered list, highest LQS/GMV impact first, each item: action → expected effect → effort (S/M/L). Maximum 10 items.

## 9. 14-Day Measurement Plan

Metrics to track before/after: search impressions, CTR, add-to-cart rate, LQS band, campaign acceptance. State the re-audit date.

## Quality gates

- Every recommendation traces to an LQS component, seller-score component, or campaign rule
- Rebuilt title follows the four-part formula within length limits
- Attribute table covers 100% of required + recommended fields
- Campaign discounts computed against 30-day lowest price, with margin stated
- No generic advice ("improve your images") — every item is executable as written
