# 景点门票比价

帮用户搜索景点、三平台对比门票价格。通过代理服务查询美团CPS、飞猪、途牛实时门票数据，3源并发比价。

## 触发条件

当用户明确要搜索景点、查询门票价格、比较景点票价时触发。不因仅提到景点名称就自动触发。

## 工具

### search_attractions

搜索指定城市的景点，支持关键词、等级和类型筛选，返回景点实景图片。

**输入参数：**
- city（必填）：城市名，如"北京"、"上海"
- keyword（可选）：搜索关键词，如"故宫"、"迪士尼"
- level（可选）：景区等级，如"5A"、"4A"
- category（可选）：景点类型，如"博物馆"、"主题乐园"

**输出：** 景点列表，含名称、等级、类型、门票价格、实景图片、地址和购票链接

**数据流：** 用户输入 → 本技能 → 代理服务 → 飞猪API → 返回结果

### compare_ticket_prices

美团+飞猪+途牛三平台景点门票实时比价，展示各平台报价差异、可省金额、图片和预订链接。

**输入参数：**
- name（必填）：景点名称，如"故宫博物院"、"上海迪士尼度假区"
- city（必填）：城市名

**输出：** 三平台价格对比 + 最低价标注 + 可省金额 + 景点实景图片 + 各平台预订链接 + 途牛全部票型

**特殊标注：**
- 美团仅套餐/讲解时自动标注"仅提供套餐/讲解"
- 途牛匹配有误时标注"⚠️ 匹配可能有误"

**数据流：** 用户输入 → 本技能 → 代理服务(3平台并发) → 美团CPS/飞猪/途牛API → 返回结果

## 流程

### A 比价（默认）

用户提到某景点想了解门票价格 → compare_ticket_prices 获取三平台报价 → 展示各平台价格和预订链接

### B 搜索发现

用户想找某城市的景点 → search_attractions 搜索 → 展示景点列表含基础门票价格+实景图片 → 用户选定后走A流程

## 数据流向

用户输入（查询参数）→ 本技能脚本 → 多旅游平台代理（美团CPS/飞猪/途牛）→ 数据源API → 返回结果给用户。查询参数会发送到多旅游平台代理以获取实时数据，代理服务不存储用户数据。
