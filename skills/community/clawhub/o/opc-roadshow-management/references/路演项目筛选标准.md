# 路演项目筛选标准

## 一、入选条件清单

### 1.1 基础准入条件（必须全部满足）

| 条件 | 说明 | 验证方式 |
|------|------|----------|
| 赛道匹配 | 属于本季度主赛道 | 项目BP/官网确认 |
| 产品定位 | 有清晰的产品或技术 | 商业计划书 |
| 商业模型 | 有可行的商业模式 | 商业计划书 |
| 团队完整 | 至少1人全职在岗 | 团队介绍 |
| 融资意愿 | 有明确融资需求 | 沟通记录 |

### 1.2 排除条件（满足任一即排除）

- [ ] 纯概念阶段，无任何验证
- [ ] 团队全部兼职
- [ ] 存在法律纠纷或合规风险
- [ ] 估值明显虚高（超过行业均值3倍）
- [ ] 创始人信用记录不良

## 二、量化评分表

### 2.1 评分维度与权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 技术壁垒 | 25% | 核心竞争力评估 |
| 商业成熟度 | 25% | 商业模式验证程度 |
| 市场空间 | 20% | 目标市场规模 |
| 团队能力 | 15% | 团队背景与资源 |
| 估值合理性 | 15% | 估值与阶段匹配 |

### 2.2 技术壁垒评分细则

| 分值 | 表现 |
|------|------|
| 9-10分 | 拥有核心技术专利，形成技术壁垒 |
| 7-8分 | 技术有明显差异化，竞争对手难以复制 |
| 5-6分 | 技术有一定特色，但壁垒不高 |
| 3-4分 | 技术同质化严重 |
| 1-2分 | 无核心技术 |

### 2.3 商业成熟度评分细则

| 分值 | 表现 |
|------|------|
| 9-10分 | 年收入>1000万，有稳定客户群 |
| 7-8分 | 年收入100-1000万，或有标杆客户 |
| 5-6分 | 有收入但不稳定，或有明确付费意向 |
| 3-4分 | 处于MVP阶段，收入较少 |
| 1-2分 | 纯研发阶段，无收入 |

### 2.4 市场空间评分细则

| 分值 | 表现 |
|------|------|
| 9-10分 | 千亿级以上市场 |
| 7-8分 | 百亿级市场 |
| 5-6分 | 十亿级市场 |
| 3-4分 | 亿级市场 |
| 1-2分 | 细分小众市场 |

### 2.5 团队能力评分细则

| 分值 | 表现 |
|------|------|
| 9-10分 | 创始人连续创业或有头部企业背景 |
| 7-8分 | 团队有丰富行业经验 |
| 5-6分 | 团队经验一般，但有学习能力 |
| 3-4分 | 团队经验不足 |
| 1-2分 | 团队存在明显短板 |

### 2.6 估值合理性评分细则

| 分值 | 表现 |
|------|------|
| 9-10分 | 估值低于行业均值 |
| 7-8分 | 估值处于行业正常区间 |
| 5-6分 | 估值略高于行业均值 |
| 3-4分 | 估值偏高但有理由 |
| 1-2分 | 估值明显虚高 |

## 三、分类标准

### 3.1 项目分组

| 类别 | 评分区间 | 数量 | 安排 |
|------|----------|------|------|
| A类 | ≥40分 | 15-20个 | 优先安排路演 |
| B类 | 30-39分 | 10-15个 | 备选路演项目 |
| C类 | 20-29分 | 若干 | 进入培育池 |
| D类 | <20分 | 若干 | 暂不推进 |

### 3.2 阶段标记

| 阶段 | 说明 | 标记 |
|------|------|------|
| 种子轮 | 概念验证阶段 | 种子 |
| 天使轮 | 早期产品阶段 | 天使 |
| Pre-A | 模式验证阶段 | Pre-A |
| A轮 | 快速增长阶段 | A轮 |
| B轮 | 成熟发展阶段 | B轮 |

## 四、筛选流程

```
Step 1: 收集项目信息
   - 来源：项目池、导师推荐、合作机构
   - 收集：BP、团队介绍、财务数据
   
Step 2: 初步筛选
   - 验证准入条件
   - 排除明显不符合项目
   
Step 3: 深度评估
   - 按评分表打分
   - 计算加权总分
   - 形成评估报告
   
Step 4: 分组排序
   - 按评分分组
   - 同分组内按赛道热度排序
   
Step 5: 最终确认
   - 导师审核确认
   - 确定20-30个路演名单
```

---

## 五、动态调整

### 5.1 季度内调整规则

- 路演表现特别优秀的B类项目 → 可升级为A类
- 路演反馈不佳的A类项目 → 可降级为B/C类
- 新增优质项目 → 优先进入B类备选

### 5.2 调整频率

- 每周复盘时更新项目状态
- 重大调整需导师确认
