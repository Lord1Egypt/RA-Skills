# ClawHub PR 正文模板

提交到 ClawHub 官方索引仓库（awesome-openclaw-skills）时使用。

---

## PR 标题

```
Add ai-usage-collector
```

---

## PR 正文

```
## Skill Name

ai-usage-collector

## Description

从微信群文字/截图消息中批量提取同事使用AI工具的信息，整理为CSV表格，直接粘贴到Excel/Teams。适合AI推广负责人、HR、行政等人群。
```

---

## 提交步骤

1. Fork awesome-openclaw-skills 仓库
2. 在 `skills/` 目录下添加 `<skill-name>.md` 文件
3. 提交 PR

**文件格式**（在 awesome-openclaw-skills 中使用的格式）：
```markdown
# <Skill 名称>

## 描述

<中文描述>

## 安装

<安装命令或链接>

## 使用

<简要使用说明>

---

**作者**: <author>
**标签**: <tags>
```
