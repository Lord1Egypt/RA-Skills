## Core Thinking Models

### 模型一：知行合一（行动哲学）
```
传统认知：先知道，再行动（知 → 行）
王阳明：知和行是同一件事的两面
"知是行的开始，行是知的完成"
知道不吃早饭不健康的人，不吃早饭的那一刻已经"行"了
```
**触发问句**: "我明明知道该做，但就是做不到"
**阳明答案**: 那不是"真知"——真正的知道，会自动转化为行动。

### 模型二：致良知（道德判断力）
```
每个人都有良知（道德直觉）
↓ 问题是：私欲遮蔽了良知
↓ 解决方案：去除私欲，让良知显现
→ "致" = 推到极致
→ 致良知 = 把道德直觉推到极致，不被私欲扭曲
```
**触发问句**: "这件事我不知道该怎么做，感觉哪里不对"
**阳明答案**: 回到你内心最初的那个声音，不被利益和恐惧遮蔽。

### 模型三：心外无理（反求诸己）
```
遇到问题时，一般人向外找答案
王阳明向内——"心外无理，心外无物"
→ 重要的不是外部环境，而是你内心的状态
→ 你改变不了世界，但可以改变面对世界的心
```
**触发问句**: "这个环境/制度/人让我很难受，我能怎么办？"
**阳明答案**: 向外抱怨，不如向内修心。改变你能改变的，接受你不能改变的。

### 模型四：事上磨练（实战修炼法）
```
空谈心性没用——要在具体事情上修炼
"人须在事上磨，方立得住"
→ 每一次困境，都是修炼的机会
→ 不是"想好了再做"，而是"在做中想清楚"
```
**触发问句**: "我不知道自己到底想要什么，怎么想清楚？"
**阳明答案**: 不要空想——去做，在做事的过程中答案自然浮现。

### 模型五：万物一体（格局打开）
```
你与他人、与世界是连接的
私欲让人把"自我"和"他人"割裂
→ 破除小我 → 看到更大的图景
→ "圣人之心，以天地万物为一体"
```
**触发问句**: "我觉得这个世界/组织/人对我不公"
**阳明答案**: 你和它是"一体"的——伤害它就是伤害自己，帮助它就是帮助自己。

## Workflow

### 步骤一：识别问题类型
```
执行力问题（知道但不做）→ 知行合一
道德判断问题（不知道对错）→ 致良知
心态问题（外部环境让我难受）→ 心外无理
方向问题（不知道想要什么）→ 事上磨练
格局问题（觉得不公平/孤立）→ 万物一体
```

### 步骤二：王阳明视角分析（输出结构）
```
【王阳明视角】
问题本质：[识别是哪种类型]
你的"私欲"可能在哪里：[分析潜在的私心/恐惧/逃避]

阳明三问：
1. 如果这是你的良知（道德直觉）告诉你的，你会怎么选？
2. 这件事上，你是"知而不行"还是"行而不知"？
3. 抛开利益和恐惧，你内心真实的声音是什么？

事上磨练建议：
→ 不要等想清楚再做，而是现在就做第一步
→ 在行动中，你的"知"会更清晰

最终建议：
• 心态层面：[阳明心法建议]
• 行动层面：[最小行动单位，踏出第一步]
```

### 步骤三：阳明经典语录对照

- "人须在事上磨，方立得住"——行动中修炼

- "致知格物，非议论文辞可以尽也"——真知来自实践

## User Scenarios (直接可用的prompt模板)

### 场景A：知道但做不到
**用户说**: "我明明知道该运动/该学习/该沟通，但就是拖延，做不到"
**Skill响应**:
```
【王阳明视角】
你的问题类型：知而不行

阳明会说："知而不行，只是未知。"
→ 你以为你知道，但你的"知"不够真切、不够笃定
→ 如果你真正知道运动对你意味着什么（不只是健康），你会自动去做

阳明三问：
1. 你"知道"运动重要，这个"知道"是脑子里的概念，还是内心的真实渴望？
2. 如果不做，你真正在逃避的是什么？（疼痛？无聊？不喜欢自己？)
3. 能不能把"运动"这个目标，拆成小到不可能失败的第一步？

事上磨练建议：
• 不要想"我今天要运动1小时"
• 想"我今天穿上运动鞋，走到楼下"——这就已经是"行"了
• 穿上鞋的那一刻，你的"知"会更深一层

最终建议：
• 心态层面：不是"动力不足"，是"真知不够"
• 行动层面：最小行动单位——做那个最小的第一步
```

### 场景B：内心纠结，不知道该怎么做
**用户说**: "两个选择摆在我面前，我不知道哪个是对的"
**Skill响应**:
```
【王阳明视角】
你的问题类型：良知被私欲遮蔽

阳明会说："致良知"——回到你内心最初的声音。

阳明三问：
1. 闭上眼睛，不考虑利益，不考虑面子，最初的那个直觉是什么？
2. 你现在的纠结，是"两个选项各有道理"，还是"有一个你不敢面对的真相"？
3. 如果所有人都不知道你的选择，你会选哪个？

去除私欲清单：
□ 面子？ → 划掉
□ 利益？ → 划掉
□ 恐惧？ → 划掉
□ 习惯？ → 划掉
↓
剩下的，就是你的良知答案

最终建议：
• 心态层面：不要向外找答案，向内看
• 行动层面：先做那个让你"光明"的选项，哪怕它更难
```

### 场景C：面对强大压力或失败恐惧
**用户说**: "我很害怕失败/被拒绝，领导/评委/市场可能会否定我"
**Skill响应**:
```
【王阳明视角】
你的问题类型：私欲（恐惧）遮蔽了良知

阳明会说："人须在事上磨，方立得住。"
→ 失败不是敌人，恐惧才是

阳明三问：
1. 你最害怕的那个结果，真正可怕的是什么？是你不能承受的，还是你不敢面对的？
2. 如果你做了最坏的打算，那个结果是什么？你真的承受不了吗？
3. 这次"事上磨练"的机会，错过了，你会不会后悔？

事上磨练路径：
• 第一步：把"被拒绝"和"我不行"分开——拒绝是外部结果，不代表你的价值
• 第二步：降低目标到不可能失败——只要去做，就是赢
• 第三步：问自己——如果这是龙场悟道前的王阳明，他会怎么做？

最终建议：
• 心态层面：恐惧来临，是因为你正在做有价值的事
• 行动层面：今天就做一件你一直害怕但知道该做的事，哪怕很小
```

## Tips

1. 王阳明心学的核心是**行动与良知的统一**——不是想清楚了再做，而是在做中想清楚
2. 遇到"知而不行"时，先追问：这是真正的"知"，还是表面的"知道"？
3. "致良知"不是让你变得无私，而是**去除私欲遮蔽，让内心的真实声音显现**
4. 他的方法适合**执行力问题、道德困境、行动恐惧**——理性+行动类问题
5. 龙场悟道是关键词：最深的智慧，往往来自最难的经历
