# BPD 2001年诊断与分度标准

## 适用对象
- 早产儿
- 极低或超低出生体重儿

## 前提条件
- 出生后持续吸氧 ≥ 28 天
- 胎龄 < 32 周 或 ≥ 32 周

## 禁忌与排除
- 不适用于非早产儿的慢性肺病
- 需排除其他导致慢性氧依赖的病因（如先天性心脏病、严重感染等）

## 执行步骤
1. **确认吸氧时间**：确保患儿出生后持续吸氧 ≥ 28 天。
2. **确定评估时间点**：
   - 若胎龄 **< 32 周**：在 **矫正胎龄（PMA）36 周** 或 **出院时**（以先到者为准）进行评估。
   - 若胎龄 **≥ 32 周**：在 **生后 56 天** 或 **出院时**（以先到者为准）进行评估。
3. **按氧依赖程度分度**：
   - **轻度 BPD**：已停氧。
   - **中度 BPD**：仍需吸氧，但吸入氧浓度（FiO₂）< 30%。
   - **重度 BPD**：需 FiO₂ ≥ 30% 和/或 正压呼吸支持（如 CPAP 或机械通气）。
4. **记录结果**：输出 BPD 诊断结论及严重程度（轻/中/重），用于后续预后评估与干预决策。

> **注意**：此为2001年NIH共识标准，因其简洁实用，目前仍在临床广泛使用。若需更细化的评估（如2018年新标准），请参考补充资料。
