# 萎缩性阴道炎诊断

## 适用对象
- 绝经后女性
- 产后闭经女性
- 接受药物假绝经治疗者

## 诊断前提
- 存在雌激素水平下降
- 有阴道分泌物或外阴症状
- 已排除滴虫性、念珠菌性等其他类型阴道炎

## 执行步骤

### 1. 病因机制确认
- 雌激素↓ → 阴道壁萎缩、黏膜变薄 → 上皮糖原↓ → 阴道pH升高（5.0～7.0）→ 乳杆菌减少 → 需氧菌过度繁殖 → 炎症

### 2. 临床表现识别
- **症状**：外阴烧灼感、瘙痒、性交痛
- **分泌物**：稀薄、淡黄色，严重时呈脓血性
- **查体**：阴道皱襞消失、黏膜菲薄充血、散在出血点或浅表溃疡

### 3. 诊断流程
1. **病史采集**：确认绝经状态（自然/人工）、卵巢手术史、盆腔放疗史
2. **排除恶性肿瘤**：尤其对血性分泌物患者必须鉴别阴道癌
3. **分泌物镜检**：见大量白细胞，无滴虫、假丝酵母菌等特异性病原体
4. **细胞学检查**：阴道上皮脱落细胞量少，以基底旁细胞为主
5. **必要时活检**：若见肉芽组织或溃疡，行局部活组织检查

> **禁忌**：不适用于育龄期雌激素正常女性；血性分泌物未排除恶性肿瘤者禁用单纯诊断为萎缩性阴道炎。
