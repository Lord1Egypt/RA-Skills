# 四大方法论速览 (Masters Quick Reference)

## 1. Alex Hormozi — 商业×内容获客的顶尖高手

- **代表作**：《$100M Offers》《$100M Leads》
- **核心哲学**：Give so much value that it feels like theft
- **内容特点**：超长但超值，每句话都有信息量，没有废话
- **适合粉丝**：想做商业/变现的人
- **入口**：YouTube @AlexHormozi
- **对你最有用**：如何把内容变成获客机器

### 他的关键模型

**The Grandmother Test**：
> 如果你奶奶都能看懂你的东西，说明你找到了核心。

**The Value Equation**：
> Value = (Dream Outcome × Perceived Likelihood of Achievement) / (Time Delay × Effort & Sacrifice)

你的内容能帮用户：增大梦想结果 + 提高成功概率 - 缩短时间 - 降低门槛

## 2. Justin Welsh — Solo Creator 的典范

- **核心模式**：一个人运营，从不做视频到月入$300万
- **主阵地**：LinkedIn + Newsletter + Twitter
- **内容模式**：短内容引流 → 长内容沉淀 → 产品变现
- **对你最有用**：如何在有全职工作的情况下做内容

### 他的关键方法

**One-to-Many**（见 content-production.md）
**Content Pyramid**：
```
1条核心长文
  → 3条Twitter短文
  → 2条LinkedIn帖
  → 1条Newsletter
```

## 3. Dan Koe — 意义感创作的倡导者

- **核心哲学**：找到你真正在乎的东西，然后用创作让它变大
- **独特视角**：创作不是"做内容"，是"整理自己的思想"
- **对你最有用**：解决"我到底要做什么类型内容"的迷茫

### 关键启示

> "大多数人害怕的不是失败，而是成功做了一些他们不关心的事。"

## 4. MrBeast (Jimmy Donaldson) — YouTube 的炼金术士

- **成就**：2亿+订阅，最懂内容的人之一
- **核心能力**：不是烧钱，是数据和心理学驱动的节奏控制
- **对你最有用**：前5秒Hook + 每30秒检查点 + 标题/封面工程

### 关键洞察

MrBeast 说他的成功可以归结为三个问题：
1. 标题够不够 clickable？
2. 前5秒能不能让人不划走？
3. 有没有持续给观众"为什么继续看下去"的理由？
