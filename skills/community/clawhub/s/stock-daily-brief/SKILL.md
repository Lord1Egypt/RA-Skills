# Stock Briefing - 股票每日简报

每天早上 5 分钟，掌握持仓动态。

## 功能

- 持仓股票行情速览
- 重要新闻和公告汇总
- 技术面简评（支撑/压力位）
- 资金流向分析
- 今日操作建议

## 使用方法

### 添加持仓
```
添加持仓：
- 股票代码1 持仓成本 持仓数量
- 股票代码2 持仓成本 持仓数量
```

### 生成简报
```
生成今日股票简报
```

### 查看单股
```
分析 股票代码
```

## 简报内容

每日简报包含：
1. **大盘概览** - 指数涨跌、成交量
2. **持仓表现** - 每只股票的盈亏情况
3. **重要消息** - 相关新闻、公告、研报
4. **技术分析** - 关键价位、趋势判断
5. **操作建议** - 买入/卖出/持有建议

## 数据来源

- 实时行情数据
- 财经新闻聚合
- 技术指标计算
