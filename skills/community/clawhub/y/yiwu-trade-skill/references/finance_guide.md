```
【外贸利润链条】

收入端：
├── 出口货款（美元）
└── 汇率折算（人民币）

支出端：
├── 商品成本
├── 物流成本

├── 财务费用（银行手续费/利息等）
├── 运营成本（人工/办公/差旅等）
└── 税务成本（所得税/关税等）

利润 = 收入 - 支出

利润率 = 利润 ÷ 收入 × 100%
```

## 三、出口退税流程

### 3.1 退税基本概念

```
【出口退税基础知识】

定义：退还出口商品在国内生产和流通环节已缴纳的增值税和消费税

核心原理：
├── 增值税：价外税，可抵扣
├── 出口零税率：出口商品适用0%税率
└── 退税：退还进项税额

计算公式：
退税额 = 出口额 × 退税率 - 进项税额

影响因素：
├── 退税率：不同商品退税率不同（0%-13%）
├── 进项发票：是否有增值税专用发票
└── 商品分类：海关商品编码对应的退税率

常见退税率参考：
├── 13%：机电产品、高新技术产品
├── 9%：纺织品、家具、塑料制品
├── 6%：部分化工产品
└── 0%：资源性产品、农产品
```

### 3.3 1039免税政策详解

```
【1039市场采购贸易免税政策】

政策依据：
├── 商务部等8部门《关于加快市场采购贸易方式试点工作的函》
├── 海关总署公告2014年第54号
└── 市场采购贸易方式适用范围

适用主体：
├── 义乌市场采购贸易经营者（个体工商户/企业）
├── 在义乌国际商贸城采购的出口商
└── 委托市场采购贸易经营者代理出口的企业

免税条件：
├── 出口商品在市场采购贸易试点范围内
├── 单票货值≤$50,000
├── 贸易真实，单证齐全
└── 通过市场采购贸易综合管理系统申报

免税范围：
├── 免征增值税
├── 不适用消费税（义乌以小商品为主）
└── 企业所得税：核定征收（通常0.5%-1%）

注意事项：
├── 不得虚假申报
├── 不得低报货值
├── 单证留存3年
└── 接受海关抽查
```

### 3.4 退税流程SOP（0110模式）

```
【一般贸易退税SOP】

Step 1: 退税资格确认
├── 确认商品退税率
├── 确认进项发票合规
├── 确认申报时限（90天内）
└── 准备申报材料

Step 2: 进项发票管理
├── 索取增值税专用发票
├── 核对发票信息准确
├── 在税务系统认证
└── 留存发票原件

Step 3: 出口申报
├── 在电子口岸系统录入
├── 选择正确的贸易方式
├── 准确申报商品编码
└── 提交出口货物明细

Step 4: 收齐单证
├── 出口货物报关单
├── 增值税专用发票
├── 出口发票
├── 收汇凭证
└── 其他证明材料

Step 5: 退税申报
├── 登录电子税务局
├── 录入退税申报数据
├── 上传电子单证
└── 提交申报

Step 6: 税局审核
├── 税局审核通常15-20个工作日
├── 可能实地核查
├── 审核通过后退税到账
└── 如有问题需补充材料

退税计算示例：
商品出口额：$10,000
退税率：13%
进项税额：¥8,000
应退税额：$10,000 × 7.25 × 13% - ¥8,000 = ¥1,425
```

## 六、成本计算工具

### 6.1 快速成本计算公式

```
【外贸成本速算公式】

FOB成本 = 出厂价 + 包装 + 国内运输 + 装卸 + 验货

CIF成本 = FOB成本 + 运费 + 附加费 + 保险
       = FOB成本 × (1 + 运费率) + 保险

DDP成本 = CIF成本 + 关税 + 增值税 + 清关费 + 送货
       = CIF成本 × (1 + 关税率) × (1 + 增值税率) + 其他

报价公式：
FOB报价 = FOB成本 ÷ (1 - 目标毛利率)
CIF报价 = CIF成本 ÷ (1 - 目标毛利率)
DDP报价 = DDP成本 ÷ (1 - 目标毛利率)

示例：
FOB成本 = ¥10.5
目标毛利率 = 25%
FOB报价 = ¥10.5 ÷ 0.75 = ¥14.0
```

### 6.2 利润敏感性分析

```
【利润敏感性分析】

关键因素：
├── 汇率波动：±1%汇率变化 → 约±1%利润变化
├── 运费波动：±10%运费变化 → 约±1-2%利润变化
├── 原材料波动：±5%成本变化 → 约±5%利润变化
└── 平台费用：±1%费用变化 → 约±1%利润变化

义乌外贸敏感性排序：
1. 汇率（影响最大）
2. 原材料成本
3. 运费
4. 平台费用
5. 其他费用
```
