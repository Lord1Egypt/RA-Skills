# 新能源与碳中和 (Clean Energy & Carbon Neutral)

## 概述

为企业和个人提供全方位的碳中和与新能源服务能力，覆盖碳排放核算、ESG报告、新能源项目评估、能源优化等关键场景。助力企业实现双碳目标和可持续发展。

## 核心能力

### 1. 碳足迹核算
- Scope 1/2/3碳排放自动计算（符合GHG Protocol标准）
- 产品碳足迹生命周期分析（LCA）
- 行业碳排放基准对比

### 2. ESG报告与合规
- ESG信息披露框架自动生成（GRI/SASB/TCFD标准）
- 监管要求合规检查（中国CSRC、EU CSRD等）
- 评级提升建议（MSCI、中证、Sustainalytics）

### 3. 新能源项目评估
- 光伏/风电/储能项目ROI与IRR计算
- 光照资源和风速数据分析
- 并网方案与补贴政策查询

### 4. 能效优化
- 企业能耗数据分析与异常检测
- 节能改造方案推荐与投资回报分析
- 智能微电网调度建议

### 5. 碳市场与绿色金融
- 碳配额价格实时追踪（全国碳市场/欧盟ETS）
- CCER/VCS等碳信用项目评估
- 绿色债券、绿色贷款政策与方案推荐

## 使用方式

```
计算 [企业能源数据文件] 的年度碳排放
为 [制造企业] 生成ESG报告初稿
评估 [光伏项目参数] 的可行性
查询最新碳配额价格与交易策略
```

## 输出格式

- **碳排放报告**：分Scope的碳排放量统计 + 同比/环比 + 行业基准
- **ESG报告框架**：按标准分章节的报告框架 + 数据缺口清单
- **项目可行性报告**：技术参数 + 财务模型 + 风险清单
- **能效建议书**：异常点报告 + 改造方案 + 节能量预估

## 适用场景

- 上市公司ESG信息披露与合规管理
- 新能源项目开发的前期评估
- 制造企业节能减排与碳资产管理
- 碳中和咨询机构的项目交付

## 注意事项

- 碳排放计算依赖准确的活动数据，数据质量直接影响结果精度
- ESG评级提升是长期过程，技能提供方向性建议
- 碳市场价格波动较大，投资决策需审慎
- 数据底座覆盖中国碳市场公开数据及主要国际标准框架
