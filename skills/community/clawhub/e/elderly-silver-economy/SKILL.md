# 养老与银发经济技能

## 核心能力

| 能力 | 说明 |
|------|------|
| 健康评估 | ADL/IADL/MMSE等老年功能评估，生成照护等级建议 |
| 用药管理 | 慢病用药提醒、药物相互作用检查、用药计划生成 |
| 养老机构匹配 | 基于预算/地点/照护等级筛选对比养老机构 |
| 长期护理保险 | 失能评估、理赔流程指导、各地长护险政策对比 |
| 银发生活规划 | 银发旅游路线、老年教育课程、再就业机会匹配 |

## 触发场景

- "帮我评估家中老人的照护等级"
- "对比北京高端养老社区"
- "长护险怎么申请理赔"
- "推荐适合老人的旅游路线"
- "提醒老人按时吃药"

## 执行流程

### 阶段一：需求评估
1. 收集老人基础信息（年龄、慢病史、居住状态）
2. 执行ADL/IADL/MMSE标准化评估
3. 输出照护等级和照护计划建议

### 阶段二：方案匹配
1. 根据照护等级匹配养老模式（居家/社区/机构）
2. 筛选符合条件的养老机构或社区服务
3. 估算月均费用和保险覆盖范围

### 阶段三：执行支持
1. 生成用药时间表和提醒计划
2. 准备长护险申请材料清单
3. 对接社区养老服务资源
