# 淘宝天天特卖

提供淘宝天天特卖频道商品查询工具，展示每日特惠低价好物。自动筛选天猫品牌店商品，涵盖家居日用、洗护清洁、零食粮油、3C数码、内衣家居服等全品类，价格多在5-30元区间。

## 能做什么

- **商品浏览**：分页浏览约188条天天特卖精选好物，每页40条共5页
- **多维度排序**：销量优先、价格从低到高、价格从高到低、推荐热度优先4种排序
- **商品详情**：获取单条商品完整信息，含优惠券、促销标签和购买链接
- **频道概览**：查看当前商品总数、缓存状态、热门类目分布
- **品质保障**：100%天猫品牌店，非C店杂货，每件商品都有品牌认证

## 不能做什么

- 不能下单购买，只提供商品信息和购买链接
- 不能按关键词搜索特定品类（天天特卖频道不支持关键词筛选）
- 不能设置价格提醒通知
- 不能查询历史价格走势

## 使用示例

1. "天天特卖有什么好货" → 默认按销量排序返回第1页40条
2. "最便宜的天天特卖商品" → sort=price_asc，按价格从低到高
3. "看看贵一点的" → sort=price_desc，按价格从高到低
4. "下一页" → page=2，查看第2页商品
5. "这个商品详情" → 从列表中获取item_id后调用get_detail

## 注意事项

- 天天特卖商品按场次轮换，5分钟缓存刷新，每次查询可能看到不同商品
- 购买链接为淘宝短链，点击后自动跳转到商品页面
- 优惠券有使用门槛，以实际页面显示为准
- 商品库存有限，看到合适的建议尽快下单
- **数据流向说明**：查询通过云端代理转发到淘宝官方API，代理不存储用户数据

## 使用提示

- 不传参数时默认返回第1页按销量排序的40条商品
- sort=price_asc适合找最低价好物，sort=price_desc适合找品质更好的商品
- 100%天猫品牌店精选，品质有保障
- 翻页浏览说"下一页"即可，共约5页可浏览
- 每条商品包含商品图片URL（pict_url），请用 ![商品名](pict_url) 格式渲染图片，图片下方放购买链接
