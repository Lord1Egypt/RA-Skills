## 🎯 名称含义

**SkillPilot** = Skill + Pilot

- **Skill** = 技能
- **Pilot** = 领航员/飞行员

**寓意**: 引导用户选择最优技能路径，自动导航到最佳技能。

## 🎯 核心功能

1. **智能选择** - 根据场景自动选择最优技能
2. **自动降级** - 失败自动切换备用技能
3. **自适应优化** - 根据成功率自动调整优先级
4. **自动发现** - 技能增减无需修改框架
