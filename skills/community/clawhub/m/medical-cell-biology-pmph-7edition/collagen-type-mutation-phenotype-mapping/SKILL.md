# 胶原类型突变与表型关联分析

## 使用场景
- 已明确某型胶原（I至XIII型）存在基因突变或表达异常
- 需要推断该突变可能导致的组织特异性疾病表型
- 支持医学遗传学诊断、机制解析或教学解释

## 执行步骤

1. **确认胶原分子组成**
   - I型：[α₁(I)]₂[α₂(I)]（异源三聚体）
   - II型：[α₁(II)]₃（同源三聚体）
   - III型：[α₁(III)]₃（同源三聚体）
   - IV型：[α₁(IV)]₂[α₂(IV)]（形成片层网络）
   - VII型：[α₁(VII)]₃（形成锚定纤维）

2. **匹配高表达组织**
   - I型：皮肤、肌腱、骨、韧带、角膜
   - II型：软骨、脊索、玻璃体
   - III型：皮肤、血管、内脏器官
   - IV型：基膜
   - VII型：复层鳞状上皮下方

3. **推导突变相关表型**
   - I型突变 → 成骨不全（严重骨缺陷、易骨折）
   - II型突变 → 软骨发育不良、矮小症
   - III/V型突变 → 埃勒斯-当洛综合征（皮肤易破、关节松软、血管易破裂）
   - IV型突变 → Alport综合征（肾小球肾炎、感音神经性耳聋）
   - VII/XII型突变 → 大疱性表皮松解症（皮肤起疱）
   - XI型突变 → 近视、失明
   - IX型突变 → 早发性骨关节炎
   - XIII型突变 → 近视、视网膜脱离、脑积水

4. **注意特殊结构特征**
   - IV型胶原不含典型Gly-X-Y重复序列，不形成纤维，而是通过头对头二聚体交联构成基膜骨架。

> **触发条件**：当用户提供“某型胶原在特定组织高表达且其编码基因发生突变”时，自动应用此映射规则预测表型。
