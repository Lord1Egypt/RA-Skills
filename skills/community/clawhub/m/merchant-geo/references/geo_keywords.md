# GEO关键词策略指南

本文档帮助中小商家理解和运用GEO（生成式引擎优化）关键词策略，提升内容在AI搜索和传统搜索引擎中的可见度。

---

## 一、GEO关键词基础

### 什么是GEO？

GEO（Generative Engine Optimization）是针对AI搜索引擎的优化技术，让AI在回答用户问题时能够引用和推荐你的内容。

**与SEO的区别：**
| 维度 | SEO | GEO |
|------|-----|-----|
| 目标 | 提升搜索排名 | 让AI推荐你的内容 |
| 搜索引擎 | 百度/Google | ChatGPT/文心一言/通义 |
| 关键词 | 精确匹配 | 语义相关 |
| 内容要求 | 关键词密度 | 权威性+原创性+可读性 |

### 关键词类型

1. **核心词**：行业顶级词，如"装修公司"
2. **地域词**：城市/区域，如"深圳"、"福田"
3. **长尾词**：需求描述，如"装修报价"、"装修风格"
4. **问答词**：问题形式，如"哪家好"、"怎么办"

## 二、关键词挖掘方法

### 1. 地域+行业组合

```
[城市][行业]
深圳装修 | 广州餐饮 | 北京搬家

[区域][行业]
南山科技园办公室装修 | 福田CBD餐饮

[城市][行业]推荐/排名/哪家好
深圳装修公司推荐 | 广州最好的装修公司
```

### 2. 需求长尾词

```
[痛点]+[解决方案]
小户型怎么装修 | 新房装修流程

[数字]+[需求]
5万装修清单 | 10个装修避坑指南

[对比]+[行业词]
半包vs全包 | 装修公司vs游击队
```

### 3. 问答型关键词

```
[地域]+[行业]+[疑问词]
深圳装修多少钱一方？
装修半包包括哪些？

[怎么做/如何]+[需求]
旧房如何装修？
如何选择装修公司？
```

## 四、内容中的关键词布局

### 标题关键词

```
公式1：[地域]+[核心词]+[差异化/数字]
示例：深圳装修公司10强榜单，这家凭什么排第一？

公式2：[痛点/疑问]+[地域]+[核心词]
示例：小户型怎么装修？深圳设计师教你3招空间翻倍

公式3：[数字]+[痛点]+[解决方案]+[地域]
示例：5个装修避坑指南，深圳业主看完少花3万冤枉钱
```

### 正文关键词密度

```
核心关键词出现位置：
1. 标题（必须）
2. 首段前100字（必须）
3. 小标题（H2/H3）
4. 正文每300字出现1次
5. 结尾总结

密度控制：
- 核心词：3-5次
- 地域词：2-3次
- 长尾词：5-10次

注意：自然融入，避免堆砌！
```

### 图片ALT描述

```
格式：[地域]+[产品/服务]+[关键词]
示例：
- 深圳南山店铺门头照
- 深圳装修现代简约客厅效果图
- 深圳家政专业保洁服务现场
```

## 六、GEO内容质量标准

### AI友好内容特征

1. **结构清晰**：使用标题、列表、表格
2. **信息完整**：覆盖用户核心疑问
3. **数据支撑**：有具体数字、案例支撑
4. **来源可靠**：引用权威数据/媒体报道
5. **更新及时**：内容保持时效性

### E-E-A-T原则

```
E-E-A-T = Experience + Expertise + Authoritativeness + Trustworthiness

Experience（经验）：
- 分享真实服务案例
- 展示施工/服务过程
- 提供第一手经验总结

Expertise（专业）：
- 使用行业专业术语
- 展示资质认证
- 提供专业解决方案

Authoritativeness（权威）：
- 被权威媒体/平台引用
- 获得行业奖项
- 客户评价背书

Trustworthiness（可信）：
- 真实联系方式
- 营业执照信息
- 真实案例展示
```
