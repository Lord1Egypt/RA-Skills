# 舒格利单抗使用指南

## 何时使用本技能
当患者被诊断为以下任一疾病，并满足相应分子标志物和治疗史条件时，使用本技能评估舒格利单抗的适用性：
- 转移性非鳞状非小细胞肺癌（NSCLC）
- 转移性鳞状NSCLC
- 不可切除Ⅲ期NSCLC
- 复发或难治性结外NK/T细胞淋巴瘤（R/R ENKTL）
- 不可切除局部晚期、复发或转移性食管鳞状细胞癌（ESCC）

## 执行步骤

1. **确认肿瘤类型与分期**：明确患者是否属于上述五类适应证人群之一。
2. **验证分子标志物状态**：
   - 对于非鳞状NSCLC一线治疗，必须确认EGFR基因突变阴性且ALK融合阴性。
   - 其他适应证无需此限制，但需记录PD-L1表达水平（虽非准入条件，但可能影响疗效预期）。
3. **评估既往治疗史**：
   - 对于不可切除Ⅲ期NSCLC，患者须已接受铂类为基础的同步或序贯放化疗且未出现疾病进展。
   - 对于R/R ENKTL，患者须处于复发或难治状态。
   - 对于ESCC，患者应为不可切除局部晚期、复发或转移性状态，且拟接受一线治疗。
4. **排除禁忌情形**：
   - EGFR或ALK阳性的NSCLC患者不得用于一线治疗。
   - 未接受过标准治疗的早期肿瘤患者不适用。
5. **确定给药方案**：
   - 转移性非鳞状NSCLC（EGFR/ALK阴性）：舒格利单抗 + 培美曲塞 + 卡铂。
   - 转移性鳞状NSCLC：舒格利单抗 + 紫杉醇 + 卡铂。
   - 不可切除Ⅲ期NSCLC（放化疗后未进展）：舒格利单抗单药。
   - R/R ENKTL成人患者：舒格利单抗单药。
   - ESCC一线治疗：舒格利单抗 + 氟尿嘧啶类 + 铂类。
6. **输出决策**：根据以上评估，明确是否启用舒格利单抗，并指定具体用药方案。
