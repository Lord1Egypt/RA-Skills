# Accounting, CPA & Tax Firm Marketing Kit

> Generate compliant marketing campaigns for CPA firms, enrolled agents, and tax preparation businesses — with IRS Circular 230, AICPA Code § 1.400, state board disclosure requirements, and FTC outcome-claim rules baked into every output.

## The Problem

Generic AI tools generate marketing content that creates serious professional liability for accounting firms:

- **"Get your biggest refund guaranteed"** — FTC Act § 5 deceptive practice + AICPA Code 1.400.001 violation
- **Missing PTIN/CPA license #** — state board advertising rules require credential disclosure; absence triggers complaints
- **Outcome promises** ("we'll save you thousands") — IRS OPR and FTC enforcement targets exactly this language
- **Unsolicited contact with distressed taxpayers** — AICPA Rule 503 and state board solicitation prohibitions
- **"Certified" claims without credentials** — state CPA board misrepresentation violation
- **Written advice without Circular 230 §10.35-10.37 compliance** — blog posts and emails that constitute "covered opinions" without proper disclaimers

This skill generates every marketing asset with the required disclosures pre-built and prohibited language pre-filtered.

## What You Get

**4 complete prompt modules:**

1. **Firm Brand & Acquisition** (Free) — Google Business Profile, website headlines, Google RSA, Facebook/Instagram ads, email subject lines, LinkedIn posts, "Why Choose Us" copy
2. **Tax Season Campaign** — Season launch, 3-email deadline sequence, SMS reminders, Google LSA optimization, audit defense upsell, extension season campaign
3. **Advisory Services Upgrade** — Off-season nurture emails, LinkedIn thought leadership, ROI case study template, referral partner pitch, advisory one-pager, pricing page copy
4. **Referral, Retention & Reputation** — Referral program (AICPA §1.500 compliant), review request sequence (FTC 2023 Endorsement Guides compliant), review response templates, annual newsletter, re-engagement campaign

## Compliance Coverage

| Rule | Coverage |
|------|----------|
| IRS Circular 230 § 10.30 | No covered opinion violations in written marketing |
| IRS Circular 230 § 10.37 | "General information, not individualized advice" disclaimers |
| AICPA Code § 1.400.001 | No false/misleading claims; no outcome guarantees |
| AICPA Code § 1.500.001 | Referral fee disclosure requirements by state |
| AICPA § 1.700 | No client confidential info in public review responses |
| State CPA Board Rules | CPA license # in all advertising outputs |
| FTC Act § 5 | No "maximum refund" or "biggest savings" guarantees |
| FTC 2023 Endorsement Guides | No conditional review requests; no incentive-for-review language |
| TCPA | SMS opt-out included in all text message templates |
| CAN-SPAM | Unsubscribe + physical address in all email templates |
