# 原发性免疫缺陷病（PID）的实验室分层检查策略

## 触发条件
当患儿满足以下任一条件时启动本流程：
- 反复、严重或机会性感染
- 早发自身免疫性疾病
- PID阳性家族史

## 执行步骤

### 第一层：初筛试验（适用于所有疑似PID患儿）
- **B细胞缺陷筛查**：检测血清IgG、IgM、IgA；B细胞计数（CD19/CD20）；抗A/B同族凝集素；嗜异性凝集素；ASO；SIgA。
- **T细胞缺陷筛查**：外周血淋巴细胞绝对计数；T细胞亚群（CD3, CD4, CD8）；迟发皮肤过敏试验（DCH，≥5种抗原）；胸部X线胸腺影。
- **吞噬细胞缺陷筛查**：白细胞计数及形态；NBT试验；IgE水平。
- **补体缺陷筛查**：CH50活性；C3和C4水平（按年龄参考值判断）。

> 注意：婴儿<2~3岁Ig水平可生理性低下，若无反复感染，暂不视为免疫缺陷。

### 第二层：进一步检查（当初筛异常或高度怀疑时）
- **B细胞**：IgG亚类、IgD/IgE、疫苗后抗体滴度、咽部侧位X线。
- **T细胞**：丝裂原增殖反应、混合淋巴细胞培养、HLA配型。
- **吞噬细胞**：化学发光试验、吞噬/杀菌功能测定、WBC动力观察。
- **补体**：各成分定量、调理素、活化片段（C3a, C4d等）。

### 第三层：特殊或研究性试验（用于确诊或机制研究）
- 基因突变分析（单基因PID确诊及产前诊断）
- 酶测定（ADA、PNP、MPO、G-6-PD等）
- 细胞功能测定（NK/CTL活性、IL-2/IFN-γ等细胞因子）
- 活检（淋巴结、胸腺）
- 体外Ig合成、B/T细胞活化增殖功能

## 输出结果
根据检查结果层级推进诊断：初筛异常 → 进一步检查 → 特殊试验确诊，并据此指导治疗（如IVIG、造血干细胞移植等）。
