# 雄韬企业头条助手

> 让每个公司都能写出专业级的公关头条。

## 这是什么？

一个 OpenClaw Skill，帮你生成企业日常资讯的文案和海报。覆盖签约、考察、技术升级、合作、节日、行业动态等常见场景。

## 5分钟快速上手

### 1. 安装

将本目录放入 OpenClaw 的 skills 目录，或通过 OpenClaw Skills UI 安装。

### 2. 首次配置

```
/头条 配置
```

回答 6 个问题（行业、调性、禁用词、渠道等），约 3 分钟。完成后品牌卡自动保存。

### 3. 日常使用

```
/头条 客户考察
/头条 节日 端午
/头条 技术升级
/头条 合作签约
```

输入关键信息 → 文案 + PNG 海报一起出。

## 支持行业

| 制造业 | 新能源 | 财税 | 科技SaaS | 电商消费 | 通用 |
|--------|--------|------|----------|----------|------|

每个行业有专属的关键词域和文案风格。

## 输出内容

每次生成输出：
- 📝 3个候选标题（不同角度）
- 📝 公众号版完整文案
- 📝 官网版完整文案
- 📝 内部版完整文案
- 🎨 PNG 海报/封面图
- ✅ 自检清单
