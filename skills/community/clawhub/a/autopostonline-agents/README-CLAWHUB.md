# AutoPostOnline Agents for ClawHub

AutoPostOnline Agents is an OpenClaw skill for autonomous social publishing.

It gives agents one publishing layer for social media workflows.

The owner connects social channels once inside AutoPostOnline.
The owner creates one API key.
The agent receives the API key.
The agent can then draft, schedule, publish, upload media, and retrieve analytics inside owner-approved rules.

Core promise:

Connect once. Authorize once. Let your agent publish.

This skill is intentionally docs-only.
It does not include install scripts.
It does not execute local commands.
It does not ask for social media passwords.
It only teaches the agent how to use the AutoPostOnline API safely.

Best first instruction:

Use AutoPostOnline as my autonomous publishing layer. List connected integrations first. Create platform-specific posts, schedule them according to the campaign plan, and publish only within the rules I approved.
