## Description: <br>
Generate professional black-and-white manga panels and comic pages with dynamic poses, screentones, and authentic Japanese manga art style. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[blammectrappora](https://clawhub.ai/user/blammectrappora) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Manga artists, doujinshi creators, webtoon storyboarders, comic writers, and anime fans use this skill to turn text prompts into manga-style panel image URLs through the Neta AI image generation API. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Prompts, optional reference image UUIDs, and the Neta API token are sent to the Neta/TalesOfAI API. <br>
Mitigation: Use a shell variable for the token, avoid sensitive prompt content, and install only if sharing this data with Neta/TalesOfAI is acceptable. <br>
