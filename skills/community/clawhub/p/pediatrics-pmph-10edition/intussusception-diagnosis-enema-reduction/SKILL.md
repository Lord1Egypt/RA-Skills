# 肠套叠诊断与灌肠复位指征评估

## 适用对象
- 健康婴幼儿
- 疑似肠套叠患儿

## 执行步骤

1. **初步临床诊断**：若患儿突发阵发性腹痛（表现为规律性哭闹）、呕吐、便血，并在腹部触及腊肠样肿块，即可临床确诊肠套叠。

2. **直肠指检**：在尚未排出血便前，必须进行直肠指检以辅助诊断。

3. **评估灌肠复位适应证**：仅当同时满足以下条件时，方可考虑灌肠复位：
   - 发病时间 ≤ 48 小时；
   - 全身情况良好；
   - 腹部无明显腹胀；
   - 无脱水或电解质紊乱。

4. **排除禁忌证**：若存在以下任一情况，禁止灌肠复位，应转为手术治疗：
   - 病程 > 48 小时，尤其伴高热、休克、精神萎靡等中毒症状（3个月以下婴儿需高度警惕）；
   - 高度腹胀、腹膜刺激征，或X线平片显示多个液平面；
   - 套叠头已达脾曲，且肿物坚硬、张力大；
   - 小肠型肠套叠。

5. **选择灌肠方法**：可采用B超监视下水压灌肠、空气灌肠或钡剂灌肠复位。

6. **判断复位成功**：复位成功标志包括：
   - 拔出肛管后排出大量带臭味的黏液血便和黄色粪水；
   - 患儿迅速入睡，停止哭闹及呕吐；
   - 腹部变平软，原包块不可触及；
   - 口服0.5～1g活性炭后6～8小时内排出炭末。

## 决策输出
若符合指征且无禁忌证，立即实施灌肠复位；若存在禁忌证或灌肠复位失败，立即转手术治疗。
