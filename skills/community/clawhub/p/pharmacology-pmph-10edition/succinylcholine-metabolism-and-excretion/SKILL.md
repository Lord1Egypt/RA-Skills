# 琥珀胆碱的代谢与排泄机制

## 何时使用
- 使用琥珀胆碱（一种除极化型肌松药）后需预测其作用持续时间
- 患者出现肌松作用异常延长，怀疑假性胆碱酯酶遗传性缺乏
- 麻醉前评估琥珀胆碱使用的安全性

## 执行步骤
1. **确认前提条件**：确保药物已静脉给药，且体内存在假性胆碱酯酶（丁酰胆碱酯酶）。
2. **启动水解过程**：药物进入血液循环后，立即被血液和肝脏中的假性胆碱酯酶水解为琥珀酰单胆碱和胆碱，肌松作用开始减弱。
3. **完成二次水解**：琥珀酰单胆碱进一步水解为琥珀酸和胆碱，肌松作用完全消失。
4. **评估排泄途径**：约98%以代谢产物形式经尿液排出，仅约2%以原形经肾脏排泄。
5. **识别高风险情况**：若患者存在遗传性假性胆碱酯酶活性低下，水解显著延迟，可能导致严重呼吸肌麻痹。
6. **解释药效特征**：该代谢路径说明琥珀胆碱作用快而短暂（通常5～8分钟），但个体差异显著。

> **注意**：碱性环境中琥珀胆碱不稳定，应避免在碱性条件下配制或储存。
