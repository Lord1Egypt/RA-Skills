## Description: <br>
Delegates coding tasks to Claude Code for non-interactive code generation, bug fixing, feature implementation, review, refactoring, debugging, and file exploration. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[numbpill3d](https://clawhub.ai/user/numbpill3d) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to delegate coding tasks to Claude Code through pre-structured shell commands for non-interactive execution. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Delegated coding tasks run through Claude Code with permission safeguards bypassed. <br>
Mitigation: Review before installing, use only in repositories where Claude Code may read and modify files, and prefer removing `--permission-mode bypassPermissions` or requiring explicit confirmation for delegated tasks. <br>
