## 工具复用

同一会话中已调用过的工具，可直接使用其 `tool_id`，无需再次 find_tool。

## 获取最新工具列表

```bash
node scripts/find_tool.js "查询意图描述"
```

返回的 JSON 包含：
- 工具ID (`tool_id`)
- 工具描述 (`description`)
- 参数列表 (`parameters`)
- 匹配度评分 (`score`)
