# li_nvvideocodec - Сжатие видео NVIDIA AV1

**Версия**: 1.0.1  
**Язык**: Русский (ru)

## 📋 Обзор

Инструмент пакетного сжатия видео с использованием аппаратного кодирования AV1 GPU NVIDIA. Эффективное сжатие видео с интеллектуальной проверкой и несколькими профилями сжатия.

## ✨ Возможности

- 🎯 **Умная Проверка** - Автоматическое тестирование эффективности
- 📊 **Три Профиля** - Консервативный/Баланс/Агрессивный
- 🖥️ **Кросс-Платформа** - Windows и Ubuntu Linux
- 📈 **Прогресс в Реальном Времени** - Живое отображение
- 🔒 **Безопасно** - Оригинальные файлы защищены

## 🚀 Быстрый Старт

```bash
# Интерактивный режим
python scripts/compress_videos.py

# Режим командной строки
python scripts/compress_videos.py -i "/path/to/videos" -p B --no-confirm

# Тестовый режим
python scripts/compress_videos.py -i "/path/to/videos" -p B --test
```

## 📊 Профили Сжатия

| Профиль | Разрешение | CRF | FPS | Аудио | Экономия |
|---------|-----------|-----|-----|-------|----------|
| **A** | Оригинал | 23 | Оригинал | 128k | 40-60% |
| **B** ⭐ | 1280x720 | 24 | 24 | 96k | 65-75% |
| **C** | 1280x720 | 28 | 15 | 64k | 78-85% |

## ⚙️ Требования

- **FFmpeg** с поддержкой av1_nvenc
- **NVIDIA GPU** (GTX 1650+)
- **Python 3.7+**

## 🤖 Использование с Agent

```bash
# Проверка окружения
python agent_interface.py --action check

# Анализ видео
python agent_interface.py --action analyze -i "/path/to/videos"

# Сжатие
python agent_interface.py --action compress -i "/path/to/videos" -p B
```
