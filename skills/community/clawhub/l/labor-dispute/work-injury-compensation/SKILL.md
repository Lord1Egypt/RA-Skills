# 工伤/职业病赔偿计算 Skill

## 两套赔偿系统对比

| 对比维度 | 工伤保险待遇 | 民法典人身损害赔偿 |
|---|---|---|
| **法律依据** | 《工伤保险条例》+ 劳动法 | 《民法典》第1179-1192条 |
| **适用前提** | 已缴纳工伤保险 + 完成工伤认定 | 用人单位侵权（无论是否参保） |
| **责任主体** | 工伤保险基金 + 用人单位（部分项目） | 用人单位直接承担 |
| **赔偿项目** | 医疗费、住院伙食补助费、伤残津贴、一次性伤残补助金、一次性工伤医疗补助金、一次性伤残就业补助金、停工留薪期工资、护理费、丧葬补助金、供养亲属抚恤金 | 医疗费、误工费、护理费、交通费、营养费、住院伙食补助费、残疾赔偿金、死亡赔偿金、精神损害赔偿 |
| **举证责任** | 劳动者举证较轻（工伤因果关系） | 原告需证明侵权事实、损失、因果关系 |
| **程序** | 劳动仲裁前置 → 法院 | 直接起诉（仲裁非必经） |
| **伤残等级** | 1-10级（劳动能力鉴定） | 1-10级（人身损害伤残鉴定，标准不同） |
| **精神损害** | 不支持 | 第1183条支持 |

## 第一步：确认工伤认定状态

### 工伤认定三要素

1. **劳动关系**：劳动者与用人单位存在劳动关系（含事实劳动关系）
2. **工作时间**：在工作时间和工作场所内
3. **因果关系**：因工作原因遭受事故伤害 / 职业病 / 上下班途中非本人主责的交通事故

### 认定时效

- 用人单位应在事故伤害发生之日或被诊断为职业病之日起 **30日内** 向社会保险行政部门提出工伤认定申请
- 用人单位未申请的，劳动者或其近亲属可在 **1年内** 直接申请
- 工伤认定作出前，医疗费等费用由用人单位先行支付

### 注意：劳动关系是核心门槛

- **有劳动关系 + 已参保工伤保险** → 走工伤保险待遇体系
- **有劳动关系 + 未参保工伤保险** → 工伤保险基金不赔付，由用人单位全额承担
- **无劳动关系**（如劳务关系、承揽关系）→ 只能走民法典侵权赔偿，不能走工伤
- **劳务派遣**：派遣单位为用人单位（申报工伤），用工单位承担连带责任

## 第二步：列赔偿清单——工伤保险待遇

### 赔偿项目（工伤保险基金支付 unless 用人单位未参保）

| 项目 | 计算规则 | 支付方 |
|---|---|---|
| **医疗费** | 符合工伤保险诊疗项目目录、药品目录、住院服务标准，据实报销 | 工伤保险基金 |
| **住院伙食补助费** | 统筹地区政府规定，一般 30-50元/天 | 工伤保险基金 |
| **交通费** | 统筹地区政府规定，工伤职工到统筹地区以外就医所需的交通费用 | 工伤保险基金 |
| **康复费** | 签订服务协议的医疗机构进行康复的费用 | 工伤保险基金 |
| **辅助器具费** | 经劳动能力鉴定委员会确认，按国家标准配置辅助器具 | 工伤保险基金 |
| **停工留薪期工资** | 原工资福利待遇不变，一般不超过12个月；伤情严重或特殊经确认可延长，但合计不超过24个月 | 用人单位 |
| **护理费** | 生活不能自理的，在停工留薪期需要护理的，由用人单位负责 | 用人单位 |
| **一次性伤残补助金** | 1级27个月本人工资，2级25个月，3级23个月，4级21个月，5级18个月，6级16个月，7级13个月，8级11个月，9级9个月，10级7个月 | 工伤保险基金 |
| **伤残津贴**（5-6级） | 5级70%本人工资，6级60%本人工资，按月发放 | 工伤保险基金 |
| **一次性工伤医疗补助金** | 解除或终止劳动关系时，按各省标准计算（与伤残等级挂钩） | 工伤保险基金 |
| **一次性伤残就业补助金** | 解除或终止劳动关系时，按各省标准计算 | 用人单位 |
| **丧葬补助金** | 6个月统筹地区上年度职工月平均工资 | 工伤保险基金 |
| **供养亲属抚恤金** | 按职工本人工资的一定比例发给因工死亡职工生前提供主要生活来源、无劳动能力的亲属 | 工伤保险基金 |
| **一次性工亡补助金** | 上年度全国城镇居民人均可支配收入的20倍（2024年约1036420元） | 工伤保险基金 |

### 停工留薪期

- 时间由劳动能力鉴定委员会确认，一般不超过12个月
- 伤情稳定后应进行劳动能力鉴定（伤残等级）
- 护理费：生活不能自理的，由用人单位支付

## 第三步：列赔偿清单——民法典人身损害赔偿

### 赔偿范围（民法典第1179条）

| 项目 | 计算规则 | 说明 |
|---|---|---|
| **医疗费** | 据实结算（需正规医疗机构票据） | 必须与工伤因果关系相关 |
| **误工费** | 按实际减少的收入计算；无固定收入的，按最近三年平均收入或受诉法院地相同/相近行业职工年平均工资 | 需要举证实际收入减少 |
| **护理费** | 护理人员有收入的，参照误工费计算；无收入的，参照当地护工从事同等级别护理的劳务报酬 | 需要医嘱或鉴定意见 |
| **交通费** | 就医、转院实际发生的交通费用，需票据 | 合理范围内的公共交通费 |
| **营养费** | 根据受害人伤残情况参照医疗机构的意见确定 | 一般按 20-30元/天 |
| **住院伙食补助费** | 参照当地国家机关一般工作人员出差伙食补助标准 | 一般 30-50元/天 |
| **残疾赔偿金** | 根据伤残等级（1-10级），按受诉法院所在地上一年度城镇居民人均可支配收入或农村居民人均纯收入标准，自定残之日起按20年计算（60岁以上每增一岁减一年，75岁以上按5年） | 城/农标准不同，伤残系数1-10级对应100%-10% |
| **残疾辅助器具费** | 普通适用器具的合理费用 | 需鉴定确认 |
| **丧葬费** | 按受诉法院所在地上一年度职工月平均工资标准，以6个月总额计算 | |
| **死亡赔偿金** | 按受诉法院所在地上一年度城镇居民人均可支配收入或农村居民人均纯收入标准，按20年计算（60岁以上每增一岁减一年，75岁以上按5年） | "同命同价"条款（第1180条） |
| **被扶养人生活费** | 根据扶养人丧失劳动能力程度，按受诉法院所在地城镇居民人均消费支出或农村居民人均年生活消费支出标准计算（被扶养人为未成年人的，计算至18周岁；被扶养人无劳动能力又无其他生活来源的，计算20年，60岁以上每增一岁减一年，75岁以上按5年） | 计入残疾/死亡赔偿金总额（2024年有重大变化） |

### 精神损害赔偿（第1183条）

- 人身权益受侵害导致**严重精神损害**的，被侵权人有权请求精神损害赔偿
- 一般人身伤害需达到"严重"程度才支持；工伤保险不包含精神损害
- 数额由法院根据过错程度、情节、后果等综合酌定（一般5万以内，个别严重情形可更高）

### 精神损害赔偿计算参考

| 伤残等级 | 参考金额 |
|---|---|
| 10-9级 | 1-2万元 |
| 8-7级 | 2-4万元 |
| 6-5级 | 4-6万元 |
| 4-3级 | 6-8万元 |
| 2-1级 | 8-10万元或更高 |

## 第四步：分别计算两套体系

### 用人单位侵权责任（民法典第1191条）

- **责任归属**：用人单位工作人员因执行工作任务造成他人损害，用人单位承担**单独责任**
- **追偿权**：用人单位承担赔偿责任后，可以向有**故意或重大过失**的工作人员追偿
- **劳务派遣**：被派遣的工作人员因执行工作任务造成他人损害，**用工单位**承担侵权责任；派遣单位有过错的，承担**相应责任**（不是连带责任）
- **适用要点**：劳动者在执行工作任务中受伤，属工伤；劳动者在执行工作任务中造成他人损害，属用人单位对第三人的责任

### 劳动者维权路径选择

```
有劳动关系 + 已认定工伤
  → 工伤保险待遇为主张路径（程序：先仲裁再诉讼）
  → 补充主张：若工伤赔偿低于民法典计算结果，可主张差额

无劳动关系（劳务/承揽）
  → 只能走民法典侵权赔偿
  → 过错责任原则（第1192条：按双方过错分担）

用人单位未缴纳工伤保险
  → 工伤保险基金不支付，全部由用人单位承担
  → 视为工伤处理，但用人单位"自掏腰包"
```

## 执行步骤清单

### Step 1：劳动关系确认
- [ ] 确认是否签订劳动合同
- [ ] 确认是否缴纳工伤保险
- [ ] 收集工资流水、考勤记录、工作证等事实劳动关系证据

### Step 2：工伤认定申请
- [ ] 用人单位30日内未申请 → 劳动者在1年内自行申请
- [ ] 材料：劳动关系证明、工伤认定申请表、医疗诊断证明
- [ ] 工伤认定结果一般60日内作出

### Step 3：劳动能力鉴定
- [ ] 伤情稳定后申请劳动能力鉴定（伤残等级）
- [ ] 确认停工留薪期时长

### Step 4：列两套赔偿清单
- [ ] 计算工伤保险待遇各项金额
- [ ] 计算民法典侵权赔偿各项金额
- [ ] 对比两套结果，分析差额

### Step 5：选择维权路径
- [ ] 有劳动关系 + 已认定工伤 → 申请劳动仲裁（工伤保险待遇）
- [ ] 无劳动关系 / 单位侵权 → 直接向法院起诉（民法典）
- [ ] 用人单位未参保 → 仲裁/诉讼中直接将用人单位列为被申请人/被告

### Step 6：仲裁/诉讼程序
- [ ] 劳动仲裁：向劳动合同履行地或用人单位所在地的劳动争议仲裁委员会申请
- [ ] 仲裁不服 → 15日内向法院提起诉讼
- [ ] 一审 → 二审（15日上诉期）
- [ ] 证据准备：工伤认定决定书、劳动能力鉴定结论、医疗票据、收入证明

## ⚠️ 关键易错点（必看，少赔上万）

| 坑 | 正确做法 |
|---|---|
| **没劳动关系（如承揽/劳务）** | 不能走工伤，只能走民法典侵权赔偿 |
| **工伤认定超时** | 单位30天不申请，你自己在1年内申请，逾期视为放弃 |
| **伤残等级标准不同** | 工伤鉴定标准 ≠ 民法典人身损害标准，结论可能不同，拿到哪个都能用 |
| **精神损害赔偿** | 工伤保险不包含，必须走民法典单独主张 |
| **单位没缴工伤保险** | 工伤保险基金不赔，全额由用人单位按工伤标准承担，别被忽悠说不算数 |
| **停工留薪期工资** | 原工资福利不变（含奖金），不是最低工资，算清楚别吃亏 |
| **被第三人侵权** | 可同时主张工伤保险待遇 + 人身损害赔偿（双重索赔） |

## 💰 实例计算（10级工伤，北京2024年参考）

假设：月薪15000元，工作3年，住院30天，医疗费25000元，工伤认定后鉴定10级伤残。

**工伤保险待遇：**
| 项目 | 计算 | 金额 |
|---|---|---|
| 一次性伤残补助金（10级=7个月） | 7×15000 | **105,000元** |
| 停工留薪期工资（3个月） | 3×15000 | **45,000元** |
| 医疗费 | 据实 | **25,000元** |
| 住院伙食补助 | ~30元/天×30天 | **900元** |
| **小计** | | **~175,900元** |

**民法典侵权赔偿（如用人单位未参保）：**
| 项目 | 计算 | 金额 |
|---|---|---|
| 医疗费 | 据实 | **25,000元** |
| 误工费 | 3×15000 | **45,000元** |
| 残疾赔偿金 | 10级≈20%×20年×北京城镇居民收入 | **~112,000元** |
| 精神损害赔偿 | 10级 | **~15,000元** |
| **小计** | | **~197,000元** |

**结论：** 10级工伤约17-20万元（地区不同有差异）。差额主张时可优先选高者。
