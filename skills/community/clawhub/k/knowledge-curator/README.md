# Knowledge Curator - 知识库管理技能

## 📖 概述

Knowledge Curator 是一个智能知识库管理技能，能够自动抓取用户分享的链接内容，整理成结构化的知识条目，并支持后续的检索和管理。

## 🎯 核心功能

### 1. 智能内容抓取

**重要**：只有用户明确指令才会触发收藏，单纯发链接不会保存！

支持主流内容平台：
- 📕 **小红书** - 笔记内容、图片描述、标签
- 📺 **B 站** - 视频信息、简介、字幕
- 📖 **知乎** - 问答内容、回答、评论
- 📹 **YouTube** - 视频信息、描述、转录
- 🌐 **通用网页** - 标题、正文、元数据

### 2. 自动主题分类
基于内容语义自动归类到 6 大主题：
| 分类 | 涵盖内容 |
|------|----------|
| 🖥️ 科技 | AI、编程、数码、互联网、科学技术 |
| 🏠 生活 | 美食、旅行、家居、日常、情感 |
| 📚 学习 | 教育、课程、教程、知识技能 |
| 🎮 娱乐 | 影视、音乐、游戏、综艺、明星 |
| 💼 工作 | 职场、管理、效率、工具、商业 |
| 💪 健康 | 运动、饮食、医疗、心理、养生 |

### 3. 结构化输出
每个知识条目包含：
- 原始链接和来源平台
- 收藏日期和时间
- 主题分类和标签
- 内容摘要（200-500 字）
- 关键知识点列表
- 原文内容（精简版）
- 用户备注

### 4. 增量更新
- 自动检测重复链接
- 支持同一主题追加内容
- 维护全局索引文件

### 5. 智能检索
- 关键词全文搜索
- 分类筛选
- 时间范围过滤
- 语义相似度匹配

## 🚀 快速开始

### 保存内容
直接发送链接即可：
```
https://www.bilibili.com/video/BV1xx411c7mD
```

或使用明确指令：
```
保存到知识库：https://zhuanlan.zhihu.com/p/123456
把这个收藏到知识库
记下来：https://xiaohongshu.com/discovery/item/xxx
```

### 查询内容
```
/kb search Python          # 搜索关键词
/kb list 学习              # 列出分类内容
/kb list                   # 列出全部内容
/kb stats                  # 查看统计
```

### 管理命令
```
/kb delete <条目 ID>       # 删除条目
/kb export                 # 导出知识库
/kb help                   # 查看帮助
```

## 🔧 技术实现

### 内容抓取流程
```
用户发送链接
    ↓
识别平台类型
    ↓
调用对应抓取策略
    ↓
提取标题、正文、元数据
    ↓
AI 生成摘要和知识点
    ↓
自动分类
    ↓
存储到对应目录
    ↓
更新索引文件
    ↓
返回确认信息
```

### 分类算法
基于关键词匹配 + 语义分析：
1. 提取内容中的关键词
2. 匹配各分类的特征词库
3. 计算相似度得分
4. 选择最高分分类（超过阈值）
5. 不确定时询问用户

### 去重机制
- URL 标准化后比对
- 标题相似度检测
- 内容指纹比对

## ⚙️ 配置选项

在 `scripts/config.js` 中可配置：

```javascript
module.exports = {
  // 知识库根目录
  knowledgeBasePath: './knowledge-base',
  
  // 索引文件路径
  indexPath: './knowledge-base/index.json',
  
  // 分类配置
  categories: ['科技', '生活', '学习', '娱乐', '工作', '健康'],
  
  // 摘要长度限制
  summaryMaxLength: 500,
  
  // 去重检测相似度阈值
  duplicateThreshold: 0.85,
  
  // 请求延迟（避免反爬）
  requestDelay: 1000,
  
  // 是否启用语义检索
  enableSemanticSearch: true
};
```

## 🔐 注意事项

1. **反爬限制**: 部分平台有访问频率限制，建议适当延迟请求
2. **视频字幕**: YouTube/B 站字幕提取可能需要额外 API
3. **隐私内容**: 敏感内容建议手动确认后再保存
4. **定期备份**: 建议定期备份 `knowledge-base/` 目录
5. **链接失效**: 保存的是快照，原链接可能失效

## 🛠️ 故障排除

### 问题：无法抓取某平台内容
**解决**: 检查该平台是否在支持列表中，部分平台可能需要更新抓取策略

### 问题：分类不准确
**解决**: 可手动指定分类：`保存到工作：[链接]`

### 问题：搜索结果为空
**解决**: 尝试更换关键词，或检查索引文件是否损坏

### 问题：重复保存
**解决**: 检查去重机制是否正常工作，可手动删除重复条目
