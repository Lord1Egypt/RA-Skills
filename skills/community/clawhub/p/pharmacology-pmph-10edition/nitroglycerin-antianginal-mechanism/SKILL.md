# 硝酸甘油抗心绞痛作用机制

## 适用对象
- 心绞痛患者（稳定型、不稳定型）
- 急性心肌梗死患者
- 心力衰竭患者（伴心肌缺血）

## 核心机制
1. **降低心肌耗氧量**：
   - 扩张静脉 → 减少回心血量 → 降低前负荷 → 心室壁张力↓ → 耗氧↓
   - 扩张动脉 → 降低后负荷 → 心肌做功↓ → 耗氧↓
2. **增加缺血区血流**：
   - 改善心内膜下灌注压梯度
   - 选择性扩张痉挛的心外膜血管
   - 开放侧支循环，促进血液流向缺血区
3. **心肌保护与抗血小板作用**：
   - 释放NO → 生成PGI₂、CGRP → 保护心肌细胞
   - 抑制血小板黏附与聚集，尤其对不稳定型心绞痛有益

## 给药途径与起效特点
- **舌下含服**：1–2分钟起效，持续20–30分钟（首选急性发作）
- **静脉给药**：用于难治性心绞痛或急性心肌梗死
- **贴剂/软膏**：用于长期预防

## 使用前提与限制
- **前提**：存在心肌缺血或冠状动脉供血不足
- **禁用/慎用**：肥厚型梗阻性心肌病、严重主动脉瓣狭窄、颅内压增高、青光眼、右室心肌梗死、严重低血压

## 分子通路简述
- 在平滑肌细胞内释放NO → 激活sGC → cGMP↑ → Ca²⁺↓ → 平滑肌松弛
- 作用不依赖血管内皮，对内皮损伤血管仍有效
