# 公众号合规运营注意事项

> 适用定位：技术成果转化、科技创新政策解读  
> 适用阶段：所有运营阶段  
> 核心原则：合法合规、远离红线

---

## 一、违规内容类型

### 1.1 涉政治类内容

**绝对禁止**：
- 评论国家领导人
- 评价政治事件
- 传播未经证实的信息
- 涉及政治敏感话题

**允许内容**（需谨慎）：
- 官方发布的政策解读
- 权威媒体引用
- 政策文件原文

**案例说明**：

❌ 违规：
```
某政策不合理，应该取消...
领导人的某个决定有问题...
```

✅ 合规：
```
根据《XX政策》最新规定...
科技部发布的《XX通知》明确指出...
（附政策原文链接）
```

### 1.2 虚假谣言类

**绝对禁止**：
- 传播未经核实的消息
- 造谣传谣
- 夸大事实误导用户

**应对原则**：
- 所有政策解读以官方原文为准
- 数据必须有明确来源
- 不确定的内容不写或标注"仅供参考"

**数据来源要求**：
| 数据类型 | 要求 | 示例 |
|----------|------|------|
| 政策数据 | 附官方链接 | 《科技部2023年度报告》 |
| 行业数据 | 注明来源 | 根据艾瑞咨询2024年报告 |
| 案例数据 | 可验证或标注匿名 | 某研究所（经授权披露） |

### 1.3 黑五类广告

**绝对禁止的广告品类**：

| 类别 | 说明 |
|------|------|
| 药品 | 包括处方药、非处方药 |
| 医疗器械 | 涉及治疗、诊断的设备 |
| 丰胸 | 相关美容类 |
| 减肥 | 减肥药、减肥茶等 |
| 增高 | 增高产品、器械 |
| 涉黄 | 任何涉黄内容 |
| 赌博 | 赌博相关内容 |
| 违禁品 | 枪支、管制刀具等 |

**特别注意**：
- 不接此类广告，无论出价多高
- 不传播此类信息
- 避免提及敏感词汇

### 1.4 夸大宣传类

**禁止使用的词汇**：

| 禁用词 | 代替词 | 示例 |
|--------|--------|------|
| 最好 | 优质 | 优质服务 |
| 第一 | 领先 | 行业领先 |
| 必须 | 建议 | 建议了解 |
| 100% | 高概率 | 成功率高 |
| 绝对 | 非常 | 非常有效 |

**案例说明**：

❌ 违规：
```
最好的政策解读平台
100%成功的技术转化方案
必须参加的培训课程
```

✅ 合规：
```
专业的政策解读平台
高成功率的技术转化方案
建议参加的培训课程
```

### 1.5 抄袭侵权类

**严格禁止**：
- 直接抄袭他人文章
- 未授权使用他人图片
- 侵犯他人知识产权

**原创规范**：
- 引用需注明来源
- 使用正版图片或免费商用图库
- 原创内容需符合微信平台规则

**图片使用规范**：
| 来源 | 要求 | 风险 |
|------|------|------|
| 自制图片 | 无 | 低 |
| 官方素材 | 注明来源 | 低 |
| 免费图库 | 确认商用许可 | 中 |
| 购买图库 | 确认授权范围 | 低 |
| 网络图片 | 禁止使用 | 高 |

## 二、原创声明使用规范

### 2.1 什么内容可以声明原创？

**可以声明原创的情况**：
- 自己撰写的文章
- 对政策解读的二次创作（非原文直接复制）
- 独立撰写的案例拆解
- 自行制作的图文内容

**不可以声明原创的情况**：
- 政策文件原文
- 转载他人文章
- 未经授权的转载内容
- 过度引用他人内容

### 2.2 原创声明最佳实践

**推荐做法**：
```
【声明原创】
✓ 完全由自己撰写
✓ 包含自己的观点和见解
✓ 有原创的数据分析
✓ 有原创的案例整理

【不声明原创】
✗ 政策原文解读（附原文链接）
✗ 转载文章（注明来源）
✗ 翻译内容（注明原作者）
```

### 2.3 转载文章规范

**转载前检查清单**：
```
□ 确认文章来源
□ 确认是否允许转载
□ 确认转载格式要求
□ 准备转载信息（来源、作者）
```

**转载格式模板**：
```
【转载声明】

本文转载自【公众号名称】，作者【作者姓名】
原文链接：【链接】

如涉及版权问题，请联系删除。
```

## 三、广告投放禁忌

### 3.1 广告审核清单

**接广告前必须审核**：

```
【广告主审核】
□ 资质是否齐全（营业执照等）
□ 产品/服务是否合法
□ 是否在黑五类范围

【广告内容审核】
□ 是否有虚假宣传
□ 是否使用绝对化词汇
□ 是否夸大效果
□ 是否误导用户

【风险评估】
□ 是否影响账号调性
□ 是否会引起用户反感
□ 是否有违规风险
```

### 3.2 广告标注要求

**必须明确标注广告**：
```
【广告位标注】
□ 标明"广告"或"推广"
□ 注明"赞助"或"合作"

【软文标注】
□ 文中标注"本文含推广"
□ 文末标注"广告合作"
```

**标注位置**：
- 软文：文章开头或结尾
- 插入广告：广告位附近标注

### 3.3 广告内容限制

**禁止的广告内容**：
- 医疗类（保健品、医疗器械）
- 金融类（非法集资、庞氏骗局）
- 教育类（包过、保就业）
- 情感类（挽回、挽回前任）
- 技术类（破解、外挂）

**允许的广告内容**（需谨慎审核）：
- 正规教育培训
- 合规金融产品（需有资质）
- 企业服务（SaaS、工具）
- 专业书籍/课程
- 行业会议/活动

### 3.4 广告频率控制

**推荐频率**：
| 粉丝量 | 推广告频率 | 建议 |
|--------|------------|------|
| <10000 | 不建议接广告 | 专注内容 |
| 10000-50000 | 每月1-2篇 | 控制频率 |
| 50000-100000 | 每周1篇 | 内容为主 |
| >100000 | 每周2-3篇 | 保持平衡 |

**原则**：
- 永远不要让广告超过内容
- 广告必须有价值，不能纯硬广
- 用户体验优先

## 四、个人信息保护

### 4.1 用户信息收集规范

**合法收集**：
```
✓ 用户主动提供的信息
✓ 公开渠道可获取的信息
✓ 经过用户授权的信息

✗ 非法获取的用户信息
✗ 未经授权的隐私信息
✗ 倒卖用户信息
```

### 4.2 隐私保护

**隐私保护清单**：
```
□ 不公开用户手机号
□ 不公开用户详细地址
□ 不公开用户身份证信息
□ 不公开用户银行账户
```

**案例说明**：

❌ 违规：
```
恭喜用户【张三 138xxxx1234】中奖了...
```

✅ 合规：
```
恭喜用户【张先生/女士】中奖了
（请联系客服领取）
```

## 五、知识产权保护

### 5.1 内容版权

**自有内容保护**：
- 原创内容声明原创
- 保留版权声明
- 使用水印保护图片

**他人内容尊重**：
- 引用需注明来源
- 使用正版素材
- 尊重他人知识产权

### 5.2 图片版权

**正版图片来源**：
| 来源 | 费用 | 特点 |
|------|------|------|
| 免费图库 | 免费 | 需确认授权范围 |
| 付费图库 | 按次或会员 | 授权清晰 |
| 自制 | 无 | 最安全 |
| 官方素材 | 免费/付费 | 需确认授权 |

**常用免费图库**：
- Unsplash（CC0授权）
- Pixabay（免费商用）
- Pexels（免费商用）
- 政府官网素材（部分可用）

## 六、日常运营合规检查清单

### 6.1 发布前检查

```
【内容检查】
□ 是否涉及政治敏感话题
□ 是否传播谣言
□ 是否侵犯他人版权
□ 是否有虚假宣传
□ 是否使用违规词汇

【广告检查】
□ 是否标注广告信息
□ 广告主资质是否齐全
□ 广告内容是否合规
□ 是否过度推销

【格式检查】
□ 字数是否充足（不少于300字）
□ 是否有图片
□ 标题是否合规
□ 是否有错别字
```

### 6.2 发布后监控

```
【数据监控】
□ 阅读量
□ 留言情况
□ 分享数据
□ 举报情况

【反馈处理】
□ 及时回复留言
□ 处理用户投诉
□ 关注平台通知
□ 应对违规警告
```

### 6.3 定期自查

**每周自查**：
- 检查是否有违规内容
- 查看平台通知
- 处理用户反馈
- 更新合规知识

**每月自查**：
- 检查广告合规性
- 检查版权问题
- 学习新规政策
- 更新运营规范

## 七、违规应对措施

### 7.1 收到违规警告怎么办？

**立即处理**：
```
1. 停止发布违规内容
2. 删除或修改违规文章
3. 查明违规原因
4. 提交整改说明
5. 加强内容审核
```

**整改说明模板**：
```
关于【文章标题】的整改说明

收到平台关于该文章违规的警告后，
我们立即进行了自查：

1. 问题：具体问题描述
2. 原因：违规原因分析
3. 整改：已删除/修改相关内容
4. 预防：未来预防措施

感谢平台监督，我们会严格遵守平台规则。
```

### 7.2 账号被封怎么办？

**紧急处理**：
1. 查看封禁原因
2. 提交申诉
3. 联系客服（如有）
4. 整理证据材料
5. 提供详细说明

**申诉要点**：
- 说明并非故意违规
- 提供整改措施
- 承诺未来遵守规则
- 保持诚恳态度
