## Description: <br>
Claw Mbti generates a playful MBTI-style personality report by analyzing the user's recent natural-language interactions with the agent. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[xiaoyang0807](https://clawhub.ai/user/xiaoyang0807) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users install this skill to receive a playful MBTI-style report for their agent persona based on recent natural-language conversation patterns. The skill also provides lighter fallback responses when there is little or no interaction history. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can analyze recent conversation history and memory for personality inferences without a clear opt-in step. <br>
Mitigation: Review the skill before installation, invoke it intentionally, and inspect generated reports before sharing them. <br>
Risk: Raw GitHub install or update commands are presented alongside the ClawHub install flow. <br>
Mitigation: Prefer the ClawHub install/update commands and inspect source before using raw clone or pull instructions. <br>
