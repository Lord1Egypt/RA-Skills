# GPT Image 2 Prompt Architect

`gpt-image-2-prompt-architect` is a public prompt-design skill for turning rough AI image ideas into structured GPT Image 2 prompt packs, variants, and debugging loops.

## What it does

- Converts rough creative briefs into structured GPT Image 2 prompts
- Handles text-to-image, reference-image edits, product photos, UI mockups, posters, and video source frames
- Produces a clean base prompt plus focused variants
- Adds short debugging and revision loops for unstable outputs

## Safety profile

- Text-only skill pack
- No scripts
- No binaries
- No required environment variables
- No autonomous network calls

## Best fit

Use this skill for ecommerce visuals, product photos, readable text layouts, social media creatives, UI mockups, posters, character sheets, storyboards, headshots, and image-to-video source frames that need stronger prompt structure.
