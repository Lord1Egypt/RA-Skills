# 出生时间矫正高级方法论

## 核心理念

**出生时间矫正（Birth Time Rectification）是印度占星的核心能力**，因为：

1. **时间精度影响预测准确性**：
   - 差1分钟，运限系统换运时间相差3-4天
   - 差4分钟，上升点相差1度
   - 差2小时，上升星座可能改变

2. **医院记录时间不准确**：
   - 很多医院记录的是"大概时间"
   - 医生可能记录的是"处理完毕时间"而非"出生时间"
   - 时钟可能不准确

3. **通过事件反向推导**：
   - 使用已发生的重要事件验证出生时间
   - 通过多层验证提高精确度
   - 最终确定精确出生时间

## 出生时间矫正八大方法

### 方法1：运限与星曜过宫法（最常用）

**原理**：通过已发生的重要事件，验证Dasha周期和Transit是否吻合。

**步骤**：
1. 收集用户已发生的重要事件（至少10-15个）
2. 分析每个事件的Dasha周期（主运、小运、微运）
3. 分析每个事件的Transit（重要行星过境位置）
4. 检查事件是否在Dasha+Transit激活时发生
5. 如果吻合度低，调整出生时间重新验证

**验证标准**：
- ✅ **高度吻合**：80%以上事件吻合 → 出生时间准确
- ⚠️ **中度吻合**：60-80%事件吻合 → 需要微调（±15分钟）
- ❌ **低度吻合**：60%以下事件吻合 → 需要大幅调整（±1-2小时）

**示例**：
```
事件：父亲去世（2006年3月）
Dasha：Jupiter-Saturn小运
验证：Jupiter（AK，灵魂指标，父亲）+ Saturn（死亡指标）
结果：✅ 吻合（Jupiter-Saturn小运过渡期）
```

    - events: 事件列表（事件类型、时间、描述）
    - physical_traits: 外表体质信息
    
    返回：
    - rectified_time: 矫正后的出生时间
    - confidence: 置信度（0-100%）
    - verification_report: 验证报告
    """
    
    # 第一阶段：初步筛选
    lagna_range = determine_lagna_range(birth_info['mother_memory_time'])
    lagna = verify_physical_traits(physical_traits, lagna_range)
    time_range = calculate_time_range(lagna, birth_info['birth_place'])
    
    # 第二阶段：事件验证
    dasha_match_rate = verify_dasha(events, time_range)
    transit_match_rate = verify_transit(events, time_range)
    overall_match_rate = (dasha_match_rate + transit_match_rate) / 2
    
    # 第三阶段：精确校正
    d9_lagna = determine_d9_lagna(relationship_traits)
    d10_lagna = determine_d10_lagna(career_traits)
    precise_time_range = calculate_precise_time(d9_lagna, d10_lagna)
    
    # 第四阶段：最终时间确定
    rectified_time = determine_final_time(time_range, precise_time_range)
    confidence = calculate_confidence(overall_match_rate, events_count)
    
    # 生成验证报告
    verification_report = generate_report(
        birth_info, events, physical_traits,
        lagna, d9_lagna, d10_lagna,
        dasha_match_rate, transit_match_rate,
        rectified_time, confidence
    )
    
    return rectified_time, confidence, verification_report
