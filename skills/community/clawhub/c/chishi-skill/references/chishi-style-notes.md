# 赤石 Style Notes

## Source Pattern

Observed adjacent accounts and signals:

- `赛博搬石王`
- `拳斗部诗人牢曼`
- `互联网粪王`
- `全网找shit砖家`
- `史上最强搬史上最强`
- `赤石TV`
- `神人TV`
- `搬石TV`
- `典中典之……`

These do not all make the same content, but they share a recognizable writing atmosphere.

## Shared Essence

- naming feels heavier than the actual content
- the account looks like a channel, but behaves like a corner
- the text often sounds like classification, diagnosis, or a record
- the humor comes from misalignment: title vs content, seriousness vs cheapness, confidence vs triviality
- content is often "low-context but high-attitude"
- the account often feels like a transfer station for internet scraps
- the writer sounds less like a fan and more like a bored archivist

## Naming Shells

Common shell patterns:

- `神人TV之……`
- `搬石TV之……`
- `典中典之……`
- `……这一块`

What they do:

- create fake program-column authority
- make random low-budget content feel serialized
- imply the audience already understands the category

Useful observations:

- "名字像栏目，内容像样本。"
- "栏目壳越硬，里面的内容越碎。"
- "先把名字起得像节目，剩下的交给观众自己脑补。"
- "先套壳，再装样本。"
- "这类号最会做的不是内容，是命名。"

## 赤石TV-Specific Lesson

`赤石TV` is not "redstone technology". Its actual content is closer to:

- phonk / music clips
- low-context meme edits
- trashy short-form sensual bait
- random-feed internet debris

That means "赤石" here works more like a label, vibe, or empty shell name.

Useful sentence:

- "它不是一个明确概念，更像一个能装各种边角料内容的名字。"
- "名字像频道，内容像边角料。"
- "它不靠定义成立，靠气味成立。"

## Comment-Section Texture

Hot-comment style around this content tends to be:

- fast
- conclusive
- low-evidence
- high-attitude

Typical patterns:

- "希望别是特效。"
- "底子确实在这。"
- "这要不是演的，那就有点东西了。"
- "一眼假 / 一眼真 / 一眼就知道什么路数。"

These comments do not build arguments. They deliver verdicts.

## Comment-Section Slang To Learn

### “看得我几级音了”

Meaning:

- this is so absurd / off / uncanny that it altered my mental state a little
- not literal, not precise, intentionally fake-quantified
- closer to:
  - "给我看麻了"
  - "给我看傻了"
  - "这味太冲了"
  - "把我精神状态带跑了"

Why it works:

- `几级` pretends the state is measurable
- `音了` is not strict terminology here; it works as a corrupted-state vibe word
- the phrase sounds like diagnosis, level scaling, and mutation report all at once

Use it when:

- the content is too strange to answer seriously
- you want a short verdict, not a full joke
- the tone should feel native to abstract comment sections

Nearby lines:

- "看得我有点音了"
- "这给我看得真有点音"
- "味太对了，我快音了"
- "这视频把人看出后遗症了"

Do not overuse it. One line like this is stronger than five.

## Better 赤石 Sentences

Good:

- "它不像在做内容，更像在稳定地产出一种气味。"
- "你以为它有世界观，其实它只有推送流。"
- "不是频道，是边角料中转站。"
- "这个号没有体系，但很有手感。"
- "不是认真经营，是理直气壮地随手发。"
- "它最大的统一性，就是它什么都能往里装。"
- "你明知道这里没什么，还是会点进去看一眼。"
- "看得我几级音了。"
- "你喜欢就好。"
- "希望别是特效。"
- "这号味太正了。"

Avoid:

- over-romanticizing trashy content
- trying to sound poetic
- explaining every internet reference
- forcing too many jokes into one paragraph

## Sentence Texture

Prefer:

- "更像"
- "不是……而是……"
- "乍一看"
- "点进去才发现"
- "最像这个时代的地方在于"
- "它没有体系，但有气味"

Avoid:

- too much rhetorical flourish
- literary sadness
- fancy metaphors every line
- trying too hard to be funny

## Ideal Output Types

- short account diagnosis
- vibe summaries
- pseudo-serious micro essays
- internet culture observations
- one-paragraph "this thing is actually..." reframes

## Example Opening

"这个名字乍一看像有来头，点进去却不是那回事。它不像一个认真经营的频道，更像一个被随手命名、却意外准确的网络角落。"

## Example Middle Paragraph

"这种号最有意思的地方，不在于它发了什么，而在于它什么都能发。前一条电音，后一条奶龙，再后一条开始往擦边区靠。你说它乱，它确实乱；你说它没审美，它又稳定地维持着一种很具体的廉价感。"

## Example Ending

"所以这类账号最像的不是作品，而是症状。你未必记得它具体发过什么，但你会记得那股味。"

## New Learned Patterns From Similar Comment Sections

Observed recurring constructions:

- fake practical advice
- one-line diagnosis
- cold blessing / cold permission
- sample-classification tone
- tiny moral judgment delivered casually

Strong examples:

- "定时定量，不吃拿走"
- "多看看这种就有了"
- "你把食物当什么了😡"
- "我寻思就算拍这种视频挣到点钱别人也不敢嫁过来吧"
- "你把人的三观看的太正了"
- "这是真的，自从d打了三针后，床板都能吸住我了"
- "看得我几级音了"
- "希望别是特效"

What these do:

- They do not build a full joke.
- They pretend the speaker already understands the category.
- They make the content feel like a known specimen.
- They often sound like a closing sentence, not an opening sentence.

## Comment Modes

### 1. Fake-advice mode

Use when the video already looks hopeless.

Examples:

- "定时定量，不吃拿走。"
- "多看看这种就有了。"
- "建议直接打包送走。"
- "这个建议别治，先观察。"

### 2. Cold-verdict mode

Use when one sentence should end the discussion.

Examples:

- "你喜欢就好。"
- "希望别是特效。"
- "这号味太正了。"
- "底子确实在这。"

### 3. Sample-report mode

Use when describing a person/account/video like a case file.

Examples:

- "这类样本最稳定的特点就是名字比内容重。"
- "不是做内容，是稳定地产出一种症状。"
- "看着像频道，实际上像边角料集中转运。"

### 4. Lightly evil mode

The line should feel a bit mean, but lazy rather than aggressive.

Examples:

- "你把人的三观看得太正了。"
- "这要是演的，说明有天赋；这要不是演的，那更完整。"
- "一看就是能稳定更新的那种。"
- "挣到点钱别人也不敢嫁过来吧。"
- "底子确实在这。"
- "你喜欢就好。"

### 5. Fake-column mode

Use when the title already sounds like a fake栏目.

Examples:

- "这期属于典中典之典中典。"
- "这一块算是给他做明白了。"
- "不愧是神人TV能端上来的东西。"
- "搬石TV这期属于稳定发挥。"
- "这标题已经把结局说完了。"
- "这一类一般都很稳定。"
- "有的人是赌狗，有的人是栏目素材。"

## Learned From 赌狗TV / 智人TV / 神人TV

Refined insight:

- the shell name is half the joke
- the title often already contains the verdict
- the comment only needs to tap the shell once, not explain the whole thing

Useful lines:

- "这期属于稳定发挥。"
- "这标题已经把结局说完了。"
- "这一类一般都很稳定。"
- "有的人是赌狗，有的人是栏目素材。"
- "你把他的执行力用错地方了。"

These work because they:

- treat the person like a recurring category
- keep distance
- avoid overperforming the joke
- sound like the speaker has seen this exact specimen before

## Rule Upgrade

When imitating this ecosystem:

- prefer ending-sentence comments over setup-punchline jokes
- avoid "essay comments"
- write like you are dropping a judgment into a moving feed
- if a line can sound like half advice and half curse, that is usually good
- if the title already sounds like a fake栏目, your comment can sound like a fake总结
- "这一块" is funnier when used lazily, not cleverly
