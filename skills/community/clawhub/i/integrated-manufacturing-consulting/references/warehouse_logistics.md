# 仓储物流调研指南

## 调研目标
- 评估仓储管理规范化水平
- 识别物料流转效率瓶颈
- 评估仓库信息化程度

## 调研重点

### 1. 仓库管理
- 仓库布局（平面仓/立体仓/阁楼式）
- 库位管理（固定库位/随机库位/WMS管理）
- 收货流程与验收标准
- 上架策略
- 先进先出(FIFO)执行情况

### 2. 库存准确率
- 盘点制度（周期盘点/年度盘点？）
- 库存准确率（实物vs系统）
- 盘点差异处理机制
- 循环盘点执行情况

### 3. 物料配送
- 物料配送方式（领料/配送制？）
- 配送频次与时机
- 线边仓管理
- 退料/补料流程

### 4. 成品发货
- 成品入库流程
- 发货流程（拣货→打包→装车）
- 发货准确率
- 物流跟踪

### 5. 信息系统
- WMS系统使用
- 条码/RFID应用
- 与ERP数据同步

## 常见问题诊断库

| 问题现象 | 可能根因 | 典型影响 |
|---------|---------|---------|
| 库存准确率低 | 出入库扫码缺失/盘点不到位 | 计划不准/缺料/超采 |
| 找不到货 | 库位管理不规范/WMS缺失 | 效率低/盘点难 |
| 先进先出执行差 | 库位规划不合理/无系统管控 | 呆滞料/过期风险 |
| 领料时间长 | 领料制/物料不按工单配送 | 产线等待/效率损失 |
| 物料配送混乱 | 无配送标准/配送频次不合理 | 线边拥挤/缺料 |
| 仓库空间利用率低 | 布局不合理/无库位优化 | 仓储成本高/扩展难 |
| 发货出错 | 拣货无校验/打包无复核 | 客户投诉/退货成本 |
