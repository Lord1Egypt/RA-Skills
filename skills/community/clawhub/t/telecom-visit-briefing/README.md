# telecom-visit-briefing

Generate professional client visit briefings and follow-up action plans from visit records, meeting notes, handwritten notes, or chat logs. Designed for telecom enterprise account managers.

## Features

- Multi-format input: text records, handwritten notes (OCR), chat logs (WeChat/Feishu/DingTalk)
- Auto-generate structured visit briefings with key findings
- AI-powered follow-up plan with product recommendations and sales scripts
- Dual-style scripts: formal business and friendly conversational
- Word document export (optional, requires user to install docx skill separately)

## Security

This skill does not auto-install any third-party skills or dependencies. The Word export feature is optional and requires the user to manually install the docx skill. All data processing is local.

## Quick Start

```bash
clawhub install telecom-visit-briefing
```
