## Description: <br>
Provides Last.fm API reference guidance for retrieving music data and user information across artists, albums, tracks, charts, tags, libraries, and user listening statistics. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[keyfrog-21K](https://clawhub.ai/user/keyfrog-21K) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agents use this skill to look up supported Last.fm API methods, required request parameters, and example requests for music metadata and public user listening data. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Last.fm API keys may be exposed if pasted into shared chats, logs, or generated request examples. <br>
Mitigation: Use a dedicated API key when possible and avoid sharing the key in logs, chats, or saved prompts. <br>
Risk: Queries for user profiles or listening history send the requested username and parameters to Last.fm. <br>
Mitigation: Only query user profiles or listening history when the user is comfortable sending that information to Last.fm. <br>
