# IP资产台账模板

## 一、模板说明

本模板用于系统性管理AI资产的知识产权状态，建立完整的归属链，确保：
1. 每个资产有明确的IP保护状态记录
2. 证书号、保护期限等关键信息可追溯
3. 续展和年费提醒有据可查
4. 风险评估有标准可依

---

## 二、IP资产台账主表

### 使用说明
- **资产编号**：格式 IP-年份-序号，如 IP-2026-001
- **风险等级**：低（已完善保护）/ 中（部分缺失）/ 高（空白或有公开泄露）
- **更新频率**：每次IP状态变化时更新

## 六、导出与使用

### Excel格式转换建议

可将上述Markdown表格复制到Excel中，添加以下增强字段：
- 筛选功能：按类型/状态/风险等级筛选
- 条件格式：高风险=红色，中风险=黄色，低风险=绿色
- 图表：保护状态饼图、风险分布柱状图

### 定期更新频率

| 场景 | 更新时机 |
|------|----------|
| 日常维护 | 每季度末更新一次 |
| IP申请 | 申请后2日内更新 |
| IP获批 | 获批后2日内更新 |
| 风险发现 | 发现后1日内更新 |

### 备份建议

- 定期导出为Excel/CSV备份
- 建议使用云文档同步
- 关键信息可打印存档
