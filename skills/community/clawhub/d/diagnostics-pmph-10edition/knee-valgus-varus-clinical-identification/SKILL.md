# 膝外翻与膝内翻的临床识别

## 适用条件
- 患者处于直立位
- 双膝充分暴露
- 双腿自然并拢
- 排除急性创伤及非负重体位（如坐位）

## 执行步骤
1. **体位准备**：要求患者取直立位，双腿自然并拢，确保双膝关节完全暴露。
2. **形态观察**：
   - 若两胫骨内踝无法接触、踝间距明显增宽，小腿向外偏斜，整体呈“X”形 → 判定为**膝外翻**（X形腿）。
   - 若两股骨内髁间距增大、小腿向内偏斜，膝关节向内成角，整体呈“O”形 → 判定为**膝内翻**（O形腿）。
3. **病因关联**：两种畸形均常见于**佝偻病**，提示维生素D缺乏所致骨骼软化与承重变形。
4. **辅助确认**：可结合平卧位检查排除姿势性假象，但诊断应以直立负重状态为准。
5. **鉴别要点**：注意与儿童发育期生理性膝外翻区分；病理性畸形通常角度显著、对称或呈进展性加重。

## 输出结果
返回定性诊断：“膝外翻”或“膝内翻”，并提示可能关联佝偻病等潜在病因。
