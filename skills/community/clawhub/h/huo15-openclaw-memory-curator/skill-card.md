## Description: <br>
Reviews structured memories, extracts durable insights, updates MEMORY.md, and cleans up expired entries. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[zhaobod1](https://clawhub.ai/user/zhaobod1) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and OpenClaw users use this skill to periodically review structured memories, remove stale or duplicate entries, and keep MEMORY.md focused on long-lived, high-value information. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can delete stored memories or update MEMORY.md, which may remove useful context or preserve an inaccurate summary. <br>
Mitigation: Run it interactively at first, review proposed deletions and MEMORY.md changes before applying them, and back up important memory data. <br>
