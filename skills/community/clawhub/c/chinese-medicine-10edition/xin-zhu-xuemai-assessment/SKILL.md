# Xin Zhu Xue Mai Assessment

## When to Use
- 需评估患者心脏生理功能状态时
- 观察到面色、舌质、脉象或心胸部异常需归因于心主血脉功能失调时
- 进行心系疾病辨证，需区分气虚、血瘀、阳虚等证型时
- 解读“奉心化血”“心气推动血行”等机制在临床中的体现时

## Execution Steps
1. **确认心主血脉的双重内涵**：
   - 心主血：推动血液运行 + 参与血液生成（“奉心化血”）
   - 心主脉：调节心脏搏动与脉管舒缩，维持脉道通利
2. **检查功能正常的四征象**：
   - 面色红润有光泽
   - 舌质淡红
   - 脉和缓有力
   - 心胸部感觉舒畅
3. **识别功能异常的证型及表现**：
   - **心气不足/血脉空虚**：心悸怔忡、胸闷气短、面白无华、舌淡、脉细弱无力
   - **心脉瘀阻**：心胸憋闷疼痛、面舌青紫、舌有瘀斑、脉细涩或结代
   - **心阳虚衰**：兼有血液化生与运行双重障碍
4. **验证核心条件**：确保评估基于心气充沛、血液充盈、脉道通利三要素的协调状态。

## Precautions
- 不适用于脱离四诊合参的孤立判断
- 不适用于非中医理论框架下的纯解剖或生理学解释
