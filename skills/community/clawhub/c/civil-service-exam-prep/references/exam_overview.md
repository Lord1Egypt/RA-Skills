# 公务员考试概述

## 考试体系

### 国家公务员考试（国考）
- **主管单位**：国家公务员局
- **时间**：每年11月底-12月初笔试，次年1-3月面试
- **岗位层级**：中央党群机关、中央行政机关、中央参公单位、省级以下直属机构
- **招录规模**：近年约3-4万人
- **考试类别**：
  - 副省级以上：行测135题/120分钟，申论大作文立意更高
  - 市地级：行测130题/120分钟，申论侧重基层治理
  - 行政执法类：行测130题/120分钟，申论侧重行政执法实务

### 地方公务员考试（省考）
- **主管单位**：各省（市）公务员局
- **时间**：
  - 联考省份（20+省）：每年3-4月
  - 独立命题省份：北京(12月)、上海(12月)、江苏(12月)、浙江(12月-1月)、山东(12月)、广东(1-3月)、深圳(3月)
- **考试差异**：
  - 题型基本一致但不完全相同
  - 部分省份有特色题型（如江苏的"平面拼合"、上海的"科学推理"）
  - 申论材料多取材本地热点

## 报考条件

| 条件 | 要求 |
|------|------|
| 国籍 | 中华人民共和国国籍 |
| 年龄 | 18-35周岁（应届硕博放宽至40岁） |
| 学历 | 大专及以上（多数岗位要求本科及以上） |
| 政治面貌 | 部分岗位要求中共党员 |
| 基层工作经历 | 中央机关部分岗位要求2年 |

## 考试流程

```
发布公告(10月/2月) → 网上报名 → 资格审查 → 缴费确认
→ 打印准考证 → 笔试(行测+申论) → 成绩查询
→ 面试名单 → 资格复审 → 面试(结构化/无领导)
→ 体检 → 考察(政审) → 公示 → 录用
```

## 计分规则

- **笔试成绩**：行测50% + 申论50%（多数岗位）
- **综合成绩**：笔试50% + 面试50%（多数岗位）
  - 部分岗位：笔试40% + 面试60%
  - 有专业科目：行测+申论+专业科目各按比例折算

## 竞争态势（2024-2025参考）

- 平均竞争比：约60-80:1
- 热门岗位：可达1000:1以上
- 进面分数线（参考）：
  - 副省级：行测60分 + 申论55分（总分110+）
  - 市地级：行测55分 + 申论50分（总分105+）
  - 实际进面线因岗位差异大

## 国考 vs 省考选择建议

| 维度 | 国考 | 省考 |
|------|------|------|
| 难度 | 较高 | 因省而异（江浙沪难度接近国考） |
| 岗位层次 | 中央机关/垂直管理 | 省/市/县级 |
| 发展空间 | 平台高、跨地域调动 | 地方晋升机会多 |
| 薪资待遇 | 全国统一标准+地区津贴 | 经济发达省份待遇更好 |
| 考试机会 | 每年1次 | 可参加多省 |
