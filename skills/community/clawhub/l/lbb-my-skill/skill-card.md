## Description: <br>
Professional Chinese-English bidirectional translation for technical documentation, following established style guides and terminology standards. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[lbbniu](https://clawhub.ai/user/lbbniu) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers, technical writers, and documentation teams use this skill to translate technical documentation, API documentation, user guides, product descriptions, and code comments between Chinese and English while preserving technical meaning and formatting. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Translations can alter technical meaning or introduce inconsistent terminology. <br>
Mitigation: Review outputs against the bundled terminology database and style guide before publication. <br>
Risk: Code examples, identifiers, or Markdown structure can be changed during translation. <br>
Mitigation: Verify that code blocks, variable names, links, headings, and UI formatting are preserved after translation. <br>
