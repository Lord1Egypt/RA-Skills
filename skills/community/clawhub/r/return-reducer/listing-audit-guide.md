# Listing Audit Guide — Reducing Listing-Related Returns

## The Listing-Return Connection

Returns caused by "looks different than photos" and "not as described" are 100% controllable. Every one of these returns is a listing failure. The goal of a listing is not just to sell — it's to sell the right product to the right customer. A listing that drives high returns is selling to the wrong customers.

---

## Product Images Audit

For each high-return SKU, check whether you have:

### Required Images (will reduce returns)
- [ ] **Main image** on white background showing the full product clearly
- [ ] **Scale reference image** — product shown next to a human hand, ruler, or everyday object (coffee cup, iPhone, etc.). This single addition reduces "smaller than expected" returns by 20–40% in most categories.
- [ ] **Lifestyle image** — product shown in use by a real person in a realistic setting
- [ ] **Detail/texture close-up** — close-up showing material quality, stitching, finish, texture
- [ ] **Multi-angle shots** — product from front, back, side, and top (as relevant)

### Apparel-Specific Images
- [ ] **On-body photo with model measurements displayed** — "Model is 5'7", 145 lbs, wearing size Medium"
- [ ] **Multiple colorway consistency** — each color variant has its own photo (not just a recolored graphic)
- [ ] **Fabric drape photo** — shows how the material hangs and moves
- [ ] **Inside view** — lining, labels, tag (for premium goods)

### Electronics/Technical Products
- [ ] **Port/connector photo** — all connections visible, labeled
- [ ] **Compatibility chart image** — shows which devices/systems it works with
- [ ] **In-use setup photo** — shows the product assembled or connected as the customer would use it
- [ ] **Size comparison** — product next to another device of known size (e.g., a smartphone)

## Product Description Audit

### Universal Requirements
- [ ] **Exact dimensions** (Length × Width × Height or as relevant) in both metric and imperial
- [ ] **Real-world size comparison** — "About the size of a deck of cards" or "fits in your back pocket"
- [ ] **Weight** (actual product weight, not shipping weight)
- [ ] **Material composition** — for apparel: "82% polyester, 18% spandex"; for products: "food-grade 316 stainless steel"
- [ ] **Color accuracy note** — "Colors may vary slightly from screen to screen. We recommend [action if color is critical]"
- [ ] **"What's in the box" list** — every item included, with quantities
- [ ] **Compatibility statement** — what it works with AND what it doesn't (be specific)
- [ ] **Care instructions** — washing, maintenance, storage
- [ ] **Warranty information**

### Apparel-Specific Copy
- [ ] **Fit description** — "True to size," "Runs small — size up one," "Fitted through the waist"
- [ ] **Fabric feel description** — "Soft and breathable," "Structured, holds shape," "Lightweight and sheer" (set expectations)
- [ ] **Stretch description** — "Non-stretch," "2-way stretch," "4-way stretch"
- [ ] **Length note** — "Falls mid-thigh on a 5'6" model; adjust sizing for longer/shorter coverage"
- [ ] **Shrinkage note** — "May shrink up to 5% after first wash; size up if between sizes"

## Size Guide Audit (Apparel & Footwear)

A size chart with just "S = 8–10, M = 10–12" is not enough. Check:

### Size Chart Requirements
- [ ] **Actual measurements** — Size chart shows body measurements (chest, waist, hip, inseam), not just dress size
- [ ] **Measurement instructions** — "How to measure" guide with body measurement diagram
- [ ] **Garment measurements** — Flat lay measurements of the actual garment (not just body size)
- [ ] **Model reference** — Model's body measurements and the size they're wearing listed
- [ ] **Fit selector** — "For a relaxed fit, size up; for a fitted look, size as measured"
- [ ] **International conversion** — US to UK/EU/AU sizes if selling internationally

### How to Measure Guide Content
Should include:
1. Bust/chest measurement (wrap tape measure around fullest point)
2. Waist measurement (natural waist, usually 1" above the navel)
3. Hip measurement (fullest point, usually 8" below natural waist)
4. Inseam measurement (inside leg from crotch to floor)
5. Note: "Measure over undergarments only, not outerwear"

## Customer Review Mining for Listing Improvements

Pull your last 50 reviews and flag any mentions of:
- Size/fit mentions ("runs small," "true to size," "fits as expected")
- Material/quality comments
- "Not as described" or "looks different" language
- Specific measurements customers mention ("it's actually 8 inches wide")

Use positive reviews to add social proof to your listing ("96% of reviewers say it fits true to size").
Use negative reviews to fix listing gaps that are causing returns.

**Review mining example:**
If 12 of 50 reviews mention "runs small" — add this to the listing immediately: "Note: This item runs about one size small. We recommend sizing up if you're between sizes."

---

## A/B Testing Listing Changes

After making a listing change to reduce returns, measure:
- Return rate for that SKU in the 30 days before the change
- Return rate for that SKU in the 30 days after the change
- Note: expect a 2–4 week lag before returns from post-change orders start arriving

**Simple tracking method:**
- Record the date you made the change
- Pull return reason data filtered to that SKU
- Compare "looks different" / "sizing" returns: 30 days pre vs. 30 days post

Most listing improvements see 20–50% reduction in listing-related returns within 60 days.

## High-Risk Return Categories and Listing Fixes

| Return Trigger | Most Common Cause | Fix |
|--------------|-------------------|-----|
| "Smaller than expected" | No scale reference in photos | Add hand/ruler reference photo |
| "Color looks different" | Inconsistent photography lighting | Reshoot product in daylight; add color disclaimer |
| "Doesn't fit" (apparel) | No measurement guide | Add body measurement guide + size chart |
| "Material feels cheap" | No material description | Add fabric composition + texture closeup |
| "Missing piece" | No "what's in the box" | Add explicit box contents list |
| "Doesn't work with my X" | No compatibility info | Add explicit compatibility chart |
| "Not suitable for my use" | No use-case description | Add "This product is designed for..." + "Not for..." |
| "Damaged on arrival" | Packaging (not listing issue) | See packaging-related return workflow |
