# 上市公司清单模板

## 使用说明

本文档提供上市公司并购标的清单的标准化模板。可根据实际需求扩展AI赛道公司库，按行业、赛道、动力类型等维度进行分类管理。

---

## 一、模板结构说明

### 字段定义

| 字段 | 说明 | 示例 |
|------|------|------|
| 证券代码 | 交易所证券代码 | 688981.SH |
| 公司名称 | 全称 | 中芯国际 |
| 上市板块 | 上交所/深交所/港交所/北交所 | 科创板 |
| 赛道分类 | AI产业链细分赛道 | 半导体制造 |
| 动力类型 | 并购驱动因素分类 | 战略投资型 |
| 市值规模 | 最新总市值（亿元） | 4500+ |
| 营收规模 | 最新营收（亿元） | 560 |
| 净利润 | 最新净利润（亿元） | 48 |
| 并购偏好 | 对外并购的战略倾向 | 产业链纵向整合 |
| 潜在标的类型 | 可能感兴趣的并购标的 | 未上市晶圆厂 |
| 历史并购 | 是否有并购经验 | 是 |
| 国资属性 | 央国企/民营/外资 | 央企 |

## 五、扩展建议

### 5.1 定期更新机制

1. **季度更新**：每季度末更新财务数据和市值
2. **事件触发**：重大并购、新产品发布、高管变动时更新
3. **年度审视**：每年重新评估动力类型和并购偏好

### 5.2 分类维度扩展

可根据实际需要增加以下维度：

| 维度 | 说明 |
|------|------|
| 地域分布 | 公司总部所在地 |
| 出口占比 | 海外收入比例 |
| 国资占比 | 国有资产比例 |
| 外贸依存度 | 进出口占比 |
| ESG评级 | 环境、社会、治理评级 |

### 5.3 筛选功能预留

建议在Excel中预设以下筛选条件：

- 按赛道分类筛选
- 按动力类型筛选
- 按市值规模筛选
- 按历史并购筛选
- 按国资属性筛选
