# 劳动合同逐条审查风险点库

## 一、高风险条款识别库

### 1.1 薪资类高风险条款

| 条款原文示例 | 风险点 | 风险等级 | 修改建议 |
|-------------|--------|---------|---------|
| "薪资面议" | 无明确金额，难以维权 | 🔴 高 | 要求写明具体金额 |
| "基本工资2000元+绩效（浮动）" | 基本工资过低，影响补偿金基数 | 🔴 高 | 要求基本工资≥实际月收入的60% |
| "公司有权根据经营情况调整薪资" | 单方降薪权 | 🔴 高 | 改为"经双方协商一致可调整" |
| "工资中已包含加班费" | 包薪制，掩盖加班费 | 🟡 中 | 要求明确加班费另行计算 |
| "年终奖由公司自主决定" | 年终奖无保障 | 🟡 中 | 要求写明发放条件和标准 |
| "离职时不发放年终奖" | 年终奖折算权被排除 | 🟡 中 | 要求按在职月份折算 |

### 1.2 工时类高风险条款

| 条款原文示例 | 风险点 | 风险等级 | 修改建议 |
|-------------|--------|---------|---------|
| "实行不定时工作制" | 未经审批的无效 | 🟡 中 | 确认是否有人社部门审批 |
| "弹性工作制，不支付加班费" | 变相逃避加班费 | 🔴 高 | 标准工时应明确加班费 |
| "每周工作6天" | 超出法定40小时/周 | 🔴 高 | 超出部分应支付加班费 |
| "加班需审批，未审批不视为加班" | 限制加班认定 | 🟡 中 | 如实际加班有记录可主张 |

### 1.3 解除类高风险条款

| 条款原文示例 | 风险点 | 风险等级 | 修改建议 |
|-------------|--------|---------|---------|
| "公司有权随时解除本合同" | 排除法定解除条件 | 🔴 高 | 应符合第39-41条 |
| "离职需提前3个月书面通知" | 超出法定30日 | 🔴 高 | 改为提前30日 |
| "离职需经公司批准后方可离开" | 限制辞职权 | 🔴 高 | 删除此条款 |
| "离职扣发最后一个月工资" | 克扣工资 | 🔴 高 | 删除此条款 |
| "合同期内不得怀孕" | 违反三期保护 | 🔴 高 | 删除，违法无效 |
| "自动续签" | 可能形成无固定期限 | 🟡 中 | 明确续签条件和期限 |

### 1.4 竞业限制类高风险条款

| 条款原文示例 | 风险点 | 风险等级 | 修改建议 |
|-------------|--------|---------|---------|
| "离职后不得从事同行业工作" | 范围过宽 | 🟡 中 | 限定具体竞争企业 |
| "竞业限制期限5年" | 超过2年上限 | 🔴 高 | 改为不超过2年 |
| 未约定竞业限制补偿金 | 劳动者可主张 | 🔴 高 | 约定不低于月工资30% |
| "竞业限制违约金100万元" | 过高可能被酌减 | 🟡 中 | 约定合理金额 |
| 对所有员工都约定竞业限制 | 普通员工无需 | 🟡 中 | 仅限高管和涉密人员 |

### 1.5 其他高风险条款

| 条款原文示例 | 风险点 | 风险等级 | 修改建议 |
|-------------|--------|---------|---------|
| "自愿放弃缴纳社会保险" | 违法无效 | 🔴 高 | 删除，社保是法定义务 |
| "自愿放弃带薪年休假" | 可能被认可 | 🟡 中 | 不建议签署 |
| "公司规章制度另行规定" | 引用未公示的制度 | 🟡 中 | 要求附上规章制度 |
| "培训费由个人承担" | 岗前培训不应收费 | 🔴 高 | 删除，岗前培训费由单位承担 |
| "押金/保证金" | 收取押金违法 | 🔴 高 | 删除，劳动法禁止收取押金 |

## 二、条款合法性速查表

| 条款类型 | 合法 | 违法 | 法律依据 |
|---------|------|------|---------|
| 试用期时长 | 符合法定上限 | 超出法定上限 | 劳动合同法第19条 |
| 试用期工资 | ≥转正80%且≥最低工资 | 低于标准 | 劳动合同法第20条 |
| 违约金 | 仅限培训服务期和竞业限制 | 其他情形约定违约金 | 劳动合同法第25条 |
| 试用期辞退 | 证明不符合录用条件 | 随意辞退 | 劳动合同法第21条、第39条 |
| 工资扣款 | 法定代扣（社保/个税等） | 罚款式扣款 | 劳动法第50条 |
| 工作时间 | 每日≤8小时，每周≤40小时 | 超出且无加班费 | 劳动法第36条 |
| 女职工保护 | 三期内不得依第40/41条解除 | 三期内辞退 | 劳动合同法第42条 |
| 押金 | — | 任何形式的押金/保证金 | 劳动合同法第9条 |
| 身份证扣押 | — | 扣押身份证件 | 劳动合同法第9条 |
