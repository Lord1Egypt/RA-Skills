## Description: <br>
Routes TODO checklists to session tracking, local TODO files, or GitHub Issues based on scope and collaboration needs. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[drumrobot](https://clawhub.ai/user/drumrobot) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and agents use this skill to decide where TODO items should live: short-lived session tracking, persistent project files, or team-shared GitHub Issues. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Guided edits to local TODO files may record inaccurate, premature, or unwanted tasks. <br>
Mitigation: Review proposed TODO file changes before applying them and keep updates scoped to user-approved items. <br>
Risk: Creating GitHub Issues can publish task details through the user's authenticated GitHub account. <br>
Mitigation: Create issues only after explicit user confirmation and review issue titles and bodies for sensitive information. <br>
