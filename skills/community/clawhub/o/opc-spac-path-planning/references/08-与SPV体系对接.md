## 二、贡献权重层（SPV层）

### 2.1 贡献权重定义

| 要素 | 说明 |
|------|------|
| **定义** | SPV阶段约定的收益分配比例 |
| **性质** | 合同权利，非工商股权 |
| **特点** | 灵活、可调整、不涉及工商变更 |
| **存证** | 区块链存证+合同约定 |

### 2.2 贡献权重构成

| 维度 | 权重占比 | 评估内容 |
|------|----------|----------|
| **资金投入** | 30% | 出资金额、资金占用时间 |
| **时间投入** | 30% | 工作时间、专业程度 |
| **资源投入** | 20% | 客户资源、技术资源、渠道资源 |
| **机会成本** | 20% | 放弃其他机会的价值 |

### 2.3 权重记录要求

```
【贡献权重记录清单】
□ 每笔贡献的详细记录
□ 贡献类型分类（资金/时间/资源/机会）
□ 贡献价值评估
□ 权重调整记录
□ 双方确认签字
□ 区块链存证
```

## 三、权重到股权的折算

### 3.1 折算触发条件

**必须同时满足**：
- 连续3个月盈利
- 年营收突破500万
- 核心成员愿意全职投入
- 决定注册实体公司

### 3.2 折算原则

| 原则 | 说明 |
|------|------|
| **公平原则** | 综合考虑历史贡献、持续投入、未来价值 |
| **书面确认** | 所有折算必须有书面协议 |
| **专业评估** | 重大折算建议第三方评估 |
| **透明原则** | 折算依据和过程透明公开 |

### 3.3 折算公式

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         折算公式                                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   股权比例 = 历史贡献分 × 40% + 持续投入分 × 30% + 未来价值分 × 30%       │
│                                                                             │
│   其中：                                                                     │
│   • 历史贡献分 = Σ(各贡献项 × 贡献权重) / Σ总贡献                         │
│   • 持续投入分 = 当前全职程度 × 岗位重要性 × 不可替代性                     │
│   • 未来价值分 = 核心能力 × 资源稀缺性 × 增长潜力                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3.4 折算评估表

| 评估维度 | 评估项 | 分值范围 | 权重 |
|----------|--------|----------|------|
| **历史贡献** | | | **40%** |
| | 资金投入（万元） | 0-100分 | 15% |
| | 时间投入（人月） | 0-100分 | 15% |
| | 资源贡献 | 0-100分 | 10% |
| **持续投入** | | | **30%** |
| | 全职程度 | 0-100分 | 10% |
| | 岗位重要性 | 0-100分 | 10% |
| | 不可替代性 | 0-100分 | 10% |
| **未来价值** | | | **30%** |
| | 核心能力 | 0-100分 | 10% |
| | 资源稀缺性 | 0-100分 | 10% |
| | 增长潜力 | 0-100分 | 10% |

### 3.5 折算示例

```
【示例：张三的权重折算】

历史贡献评估：
├── 资金投入：10万元 → 10分
├── 时间投入：12人月 → 60分
└── 资源贡献：带来3个客户 → 50分
历史贡献分 = 10×15% + 60×15% + 50×10% = 16分

持续投入评估：
├── 全职程度：100% → 100分
├── 岗位重要性：核心创始 → 90分
└── 不可替代性：高 → 80分
持续投入分 = 100×10% + 90×10% + 80×10% = 27分

未来价值评估：
├── 核心能力：AI技术专家 → 85分
├── 资源稀缺性：难以替代 → 75分
└── 增长潜力：高 → 80分
未来价值分 = 85×10% + 75×10% + 80×10% = 24分

最终得分 = 16 + 27 + 24 = 67分

假设总股权池为100%，所有成员总得分为500分：
张三股权比例 = 67/500 = 13.4%
```

## 四、实体公司层（工商股权）

### 4.1 股权架构建议

| 持股方 | 比例 | 性质 | 说明 |
|--------|------|------|------|
| 核心创始团队 | 60-70% | 直接持股 | 按折算结果 |
| 员工期权池 | 15-20% | 预留 | 激励未来加入者 |
| 战略投资者 | 10-20% | 引入 | 产业资源方 |
| SPV历史成员 | 折算后纳入 | 协商确定 | 部分直接持股，部分保留 |

### 4.2 折算后股权确认

```
【股权确认流程】
1. 完成折算评估
2. 内部公示（7个工作日）
3. 异议处理
4. 签署折算协议
5. 工商登记
6. 出具股权证明
```

### 4.3 后续调整机制

| 调整情形 | 调整方式 | 决策机制 |
|----------|----------|----------|
| 新加入成员 | 期权池稀释 | 股东会决议 |
| 核心成员退出 | 按约定回购 | 协议约定 |
| 业绩达标奖励 | 期权授予 | 股东会决议 |
| 融资稀释 | 同比例稀释 | 投资协议 |

### 5.3 稀释计算

| 阶段 | 操作 | 稀释说明 |
|------|------|----------|
| **实体公司→股份制改造** | 净资产折股 | 通常1:1，无稀释 |
| **引入pre-A投资** | 假设融资20%，估值5000万 | 创始人团队稀释至80% |
| **引入A轮投资** | 假设融资25%，估值2亿 | 进一步稀释 |
| **引入B轮投资** | 假设融资20%，估值8亿 | 进一步稀释 |
| **上市发行** | 发行25%股份 | 所有股东同比例稀释 |

### 5.4 稀释示例

```
【示例：股权稀释过程】

初始股权（折算后）：
├── 创始人团队：70%
├── 期权池：15%
└── 战略投资者：15%

Pre-A轮融资（假设估值5000万，融资1000万）：
├── 融资后估值：6000万
├── 创始人团队：70% × (5000/6000) = 58.3%
├── 期权池：15% × (5000/6000) = 12.5%
└── 战略投资者：15% + 16.7% = 31.7%（原有+新投资）

A轮融资（假设估值2亿，融资5000万）：
├── 融资后估值：2.5亿
├── 创始人团队：58.3% × (2/2.5) = 46.6%
├── 期权池：12.5% × (2/2.5) = 10%
└── 战略投资者+新投资方：43.4%

上市（发行25%股份）：
├── 上市后创始人团队：46.6% × 75% = 34.95%
├── 期权池：10% × 75% = 7.5%
├── 战略投资者：等比例稀释
└── 公众股东：25%
```

## 七、常见问题与解答

### Q1: SPV阶段的贡献权重和公司股权是什么关系？

A: 贡献权重是合同权利，股权是工商权利。折算时，按评估体系将历史贡献转化为工商股权比例。

### Q2: 折算时如果有人反对怎么办？

A: 折算前充分沟通，折算过程透明公开，重大分歧可申请第三方评估或仲裁。

### Q3: 折算后可以退出吗？

A: 可以，但需按协议约定处理。通常有锁定期，退出价格按约定或评估确定。

### Q4: 上市后股份如何交易？

A: 上市后股份可在证券交易所自由买卖，但需遵守减持规定和锁定期。

### Q5: 期权池在上市时怎么处理？

A: 上市前期权池通常为10-15%，上市后期权可按计划行权，行权后股份可流通。
