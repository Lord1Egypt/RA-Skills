# Moistening Laxative Herb Combination

## When to Use
- 患者确诊为肠燥津枯便秘，且病因明确为以下之一：
  - 年老津枯
  - 产后血虚
  - 热病伤津
  - 失血
- 排除实热便秘、气滞便秘等实证类型

## Execution Steps
1. **辨明根本证型**：确认导致肠燥便秘的具体病机类型（如血虚、津伤、气滞、气虚）。
2. **按证型选择配伍**：
   - 若为「热盛津伤」，配伍清热养阴药（如生地、麦冬、玄参）；
   - 若为「血虚」，配伍补血药（如当归、熟地黄、白芍）；
   - 若兼有「气滞」，配伍行气药（如木香、枳壳、陈皮）；
   - 若兼有「气虚」，配伍益气药（如党参、黄芪、白术）。
3. **选用常用润下药**：如火麻仁，并与当归、桃仁、熟地黄等协同使用以增强润肠通便效果。
4. **注意事项**：润下药多为富含油脂的植物种仁，作用缓和，仅适用于虚性便秘，禁用于实证便秘。

## Output
根据辨证结果选择合适的润下药及配伍方案，实现标本兼治的治疗目标。
