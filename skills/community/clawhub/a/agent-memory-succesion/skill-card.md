## Description: <br>
Provides a conceptual framework for persistent agent memory across episodic, semantic, procedural, and relational memory types, including memory architecture, retrieval concepts, and succession planning. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, agents, and organizations use this skill to plan durable agent memory structures, understand memory retrieval concepts, and prepare knowledge transfer during agent succession. It is instruction-only and does not perform storage or retrieval itself. <br>

### Deployment Geography for Use: <br>
Global <br>
