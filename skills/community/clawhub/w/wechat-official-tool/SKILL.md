# 微信公众号运营全流程技能

## 概述

本技能提供微信公众号运营的完整工作流，从选题到发布再到数据分析，覆盖全链路。

---

## 流程图

```
选题研究 → 内容创作 → 图片生成 → 内容优化 → 排版校对 → 发布 → 数据分析
   ↓                                                          ↓
百度搜索/多引擎搜索                                    自动化定时发布
```

---

## 环节一：选题研究

> **账号定位**：情感心理、财政金融、文学艺术、随想感悟四类。选题在热门话题基础上侧重这四类方向。

### 目标
找到有流量、有共鸣、适合账号定位的话题。

### 操作步骤

1. **热点捕捉**：用 `baidu-search` 或 `multi-search-engine` 搜索：
   - "今日热点 微信公众号"
   - "微信公众号 热门话题 [你的领域]"
   - "微信指数 [关键词]"

2. **竞品分析**：
   - 搜索同领域头部公众号近期推文
   - 记录：标题风格、选题角度、阅读量（如有显示）

3. **关键词规划**：
   - 主关键词：账号核心领域词
   - 长尾词：具体问题类词（"如何..."、"为什么..."）

### 输出模板

```
【选题单】
标题方向：
核心关键词：
目标读者痛点：
竞品参考：
预计发布时间：
```

## 环节二：内容创作

### AI 写作提示词模板

```
你是一位经验丰富的微信公众号编辑，请帮我写一篇推文：

【选题】[填写话题]
【目标读者】[填写受众，如：25-35岁职场人]
【文章风格】[干货型/故事型/观点型]
【字数要求】[800-1500字]
【核心观点】[1-3个要传达的要点]

要求：
1. 标题吸引人但不标题党
2. 开头用痛点/故事引入
3. 正文分3-5个小标题，逻辑清晰
4. 结尾有互动引导（评论/在看/转发）
5. 语言自然，不堆砌辞藻
```

### 文章结构模板

```
【标题】20字以内，含关键词

【导语】100字
- 痛点/现象引入
- 点出本文价值

【正文】3-5个小节，每节200-300字
## 小标题1：现象描述
## 小标题2：原因分析
## 小标题3：解决方案
## 小标题4：案例/数据支撑

【结尾】100-150字
- 总结核心观点
- 引导互动（评论/在看）
- 预告下期内容（可选）
```

## 环节三：图片生成

### 封面图

使用 `多模态内容生成` 技能生成封面图：

```
提示词模板：
微信公众号封面图，[文章主题]，[风格描述]，
简洁大气，适合手机端显示，16:9比例，
建议尺寸 900×500px，PNG格式，主色调：[颜色]，留文字排版空间
```

> ⚠️ 封面图文件大小须 < 2MB（永久素材），如用草稿 API 直写则须 < 64KB。
> 推荐使用 `wechat-oa-draft` 技能处理封面图压缩与上传。

### 文中配图

```
文中插图，[具体场景描述]，[风格：扁平插画/写实照片/信息图]，
尺寸适配手机屏幕，清晰美观
```

### 图片注意事项
- 封面图建议 900×500px（2.35:1 或 16:9）
- 文中图片宽度建议 1080px
- 避免文字过多（手机端看不清）

## 环节四：内容优化（去除AI痕迹）

使用 `humanizer` 技能对 AI 生成的内容进行优化：

1. 去除 AI 特征词汇（此外、值得注意的是、值得注意的是...）
2. 增加个人观点和语气
3. 长短句交替，避免机械化结构
4. 加入具体细节和案例

### 自检清单

- [ ] 没有"此外"、"值得注意的是"等AI高频词
- [ ] 标题不超过20字
- [ ] 有小标题，且小标题有吸引力
- [ ] 段落不超过5行（手机阅读友好）
- [ ] 有引导互动的结尾
- [ ] 关键词自然出现3-5次

## 环节五：排版校对

### 微信公众号排版规范

| 元素 | 规范 |
|------|------|
| 正文字号 | 15-16px（微信默认16px） |
| 行间距 | 1.8-2.0 |
| 段间距 | 15-20px |
| 页边距 | 10-15px |
| 小标题 | 加粗，可加底色 |
| 重点文字 | 加粗 或 标色（公众号允许的颜色） |

### 常用排版工具
- **秀米**（xiumi.us）：模板丰富，适合新手
- **135编辑器**（135editor.com）：功能强大
- **Markdown Here**：Markdown 转公众号格式

### 排版检查清单
- [ ] 封面图已上传
- [ ] 摘要已填写（50字内，吸引点击）
- [ ] 正文无错别字
- [ ] 图片已压缩（单张<500KB）
- [ ] 有"阅读原文"链接（如需要）
- [ ] 作者信息已填写

## 环节六：发布

### 方式A：手动发布（精确控制）

1. 登录微信公众平台（mp.weixin.qq.com）
2. 新建图文消息
3. 粘贴内容，上传图片
4. 设置封面、摘要
5. 预览 → 发布

### 方式B：浏览器自动化发布（使用 agent-browser 技能）

适用于定时批量发布场景：

```bash
# 1. 打开公众平台
agent-browser open https://mp.weixin.qq.com

# 2. 扫码登录（首次需要手动扫码，之后可保存状态）
agent-browser snapshot -i

# 3. 保存登录状态
agent-browser state save wechat-mp-auth.json

# 之后使用时加载状态
agent-browser state load wechat-mp-auth.json
```

> ⚠️ 注意：微信公众平台有安全验证，自动化发布可能需要处理验证码。

### 方式C：定时发布（使用 automation-workflows 思路）

在微信公众平台内配置：
1. 编辑好图文后，点击"定时群发"
2. 设置发布时间（可设未来7天内）
3. 支持每日定时（需公众号权限）

### 方式D：API 直写草稿箱（无需第三方排版工具） ⭐ 推荐

使用 `wechat-oa-draft` 技能，直接将文章写入公众号草稿箱，跳过秀米/135编辑器。

**前置条件**：
- 拥有公众号 AppID + AppSecret（管理员后台查看）
- 调用服务器 IP `112.8.202.216` 已加入 IP 白名单
- 凭证保存在 `{workspace}/wechat_credentials.json`

**完整流程**（由 `wechat-oa-draft` 技能自动执行）：

```
1. 获取 Access Token（有效期2小时）
   GET /cgi-bin/token

2. 上传封面图（永久素材接口 ⚠️ 非临时素材）
   POST /cgi-bin/material/add_material?type=image
   图片须 < 2MB，技能自动压缩

3. 构造草稿 JSON
   使用 scripts/create_draft.py 将 .md 转微信 HTML

4. 提交草稿
   POST /cgi-bin/draft/add
```

**常见错误速查**：

| errcode | 含义 | 解决方案 |
|---------|------|---------|
| 40164 | IP不在白名单 | mp后台添加 `112.8.202.216` |
| 40006 | media 尺寸超限 | 压缩图片至 < 2MB |
| 40007 | 无效 media_id | 改用 `add_material` 永久素材 |
| 40001 | token 无效/过期 | 重新获取（7200秒有效期） |

> ⚠️ **关键坑**：草稿接口只接受**永久素材** media_id，
> 用 `media/upload?type=thumb`（临时素材）上传的图会报 40007！

## 环节七：数据分析

### 核心指标

| 指标 | 意义 | 优化方向 |
|------|------|---------|
| 阅读量 | 内容触达规模 | 优化标题、封面 |
| 分享率 | 内容传播力 | 增加共鸣点、金句 |
| 在看数 | 内容认可度 | 提升观点鲜明度 |
| 完读率 | 内容质量 | 优化开头、精简篇幅 |
| 涨粉数 | 内容转化力 | 强化账号价值主张 |

### 数据获取

在微信公众平台 → 统计 → 图文分析 中查看。

### 内容优化循环

```
发布 → 看数据 → 分析问题 → 调整内容策略 → 下一期发布
```

## 完整工作流示例

### 场景：每周发布1篇干货推文

**周一**：选题研究（30分钟）
- 用 baidu-search 搜索本周热点
- 确定选题，填写选题单

**周二**：内容创作（1小时）
- 用 AI 写作提示词生成初稿
- 用 humanizer 优化文字

**周三**：图片生成（30分钟）
- 用多模态内容生成 做封面图
- 生成文中配图

**周四**：排版校对（30分钟）
- 用秀米/135编辑器排版
- 检查排版清单

**周五**：发布
- 手动发布 或 设定时发布
- 分享到朋友圈/社群

**下周一**：数据分析（15分钟）
- 记录核心指标
- 总结本次优劣
- 反馈到下一期选题

## 自动化配置建议

使用 `automation-workflows` 思路，可以设置：

1. **每周一早上9点**：提醒选题（用 automation_update 设置定时任务）
2. **发布后2小时**：提醒查看早期数据
3. **每月1号**：提醒做上月数据总结

### 设置定时提醒示例

可以用 automation_update 工具设置周期性提醒，例如：
- 每周一提醒选题
- 每周五提醒发布

## 常见问题

**Q：没有原创保护怎么办？**
A：坚持原创，在"原创"按钮处声明。持续原创7天可申请开通"赞赏"。

**Q：阅读量低怎么提升？**
A：优化标题（含关键词）、提升封面图质量、在合适时间发布（早8点、午12点、晚8-10点）。

**Q：如何快速涨粉？**
A：内容质量第一；适当在相关社群分享；与其他号互推；设置"关注后自动回复"引导。
