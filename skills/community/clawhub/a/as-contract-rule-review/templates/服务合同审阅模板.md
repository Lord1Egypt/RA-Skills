# 服务合同审阅模板

## 一、服务双方主体
- [ ] 委托方（甲方）信息：名称、地址、联系人
- [ ] 服务方（乙方）信息：名称、地址、联系人
- [ ] 服务资质/许可证是否有效
- [ ] 代理人授权是否合法有效

## 二、服务内容
- [ ] 服务范围描述是否清晰完整
- [ ] 服务标准/交付物是否明确
- [ ] 服务期限/时间节点是否清晰
- [ ] 服务地点/方式是否明确

## 三、服务费用
- [ ] 服务费用总额是否明确
- [ ] 费用构成是否清晰（人工/材料/其他）
- [ ] 付款方式（按月/按阶段/按成果）是否合理
- [ ] 费用调整机制是否明确

## 四、服务质量
- [ ] 服务质量标准是否可量化
- [ ] 验收标准和流程是否完善
- [ ] 售后服务/质保条款是否明确
- [ ] 客户满意度评价机制是否合理

## 五、人员管理
- [ ] 服务人员资质要求是否明确
- [ ] 人员更换通知机制是否完善
- [ ] 保密义务是否明确

## 六、违约责任
- [ ] 服务不达标违约金是否合理
- [ ] 延误责任划分是否清晰
- [ ] 甲方逾期付款责任是否对等
- [ ] 擅自终止服务责任是否合理

## 七、合同变更与终止
- [ ] 合同变更流程是否完善
- [ ] 终止条件是否公平合理
- [ ] 终止后义务（交接/保密）是否明确

## 八、知识产权
- [ ] 服务成果归属是否明确
- [ ] 保密信息范围是否合理
- [ ] 竞业限制条款是否合理

## 九、争议解决
- [ ] 管辖法院/仲裁条款是否明确
- [ ] 适用法律是否约定
