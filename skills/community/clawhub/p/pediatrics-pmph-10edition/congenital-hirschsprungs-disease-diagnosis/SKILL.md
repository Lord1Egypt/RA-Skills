# 先天性巨结肠诊断与鉴别

## 适用场景
- 新生儿胎粪排出>24–48小时
- 顽固性便秘、腹胀、呕吐
- 直肠壶腹空虚

## 执行步骤
1. **初筛**：观察低位肠梗阻症状（腹胀、呕吐）或婴幼儿顽固性便秘（3～7天排便1次）。
2. **关键体征**：直肠指检发现直肠壶腹部空虚，拔指后排出大量粪便。
3. **辅助检查选择**：
   - **钡剂灌肠**（诊断率约90%）：显示痉挛段、移行段、扩张段呈“漏斗状”；
   - **直肠肛门测压**：内括约肌反射性松弛消失（<2周新生儿慎用）；
   - **直肠活检**（金标准）：HE染色显示病变段缺乏神经节细胞。
4. **鉴别诊断**：
   - 新生儿期：排除胎粪阻塞综合征、肠闭锁、坏死性小肠结肠炎；
   - 婴幼儿期：排除继发性巨结肠、功能性便秘。

## 输出结果
高度怀疑先天性巨结肠时，应尽快安排钡剂灌肠或活检以确诊，避免延误导致小肠结肠炎等致命并发症。
