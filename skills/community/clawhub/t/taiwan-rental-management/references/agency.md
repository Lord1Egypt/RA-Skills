# 包租代管合約要點

## 三種管理模式的合約比較

| 面向 | 自租自管 | 代管合約 | 包租合約 |
|------|----------|----------|----------|
| 房東收租 | ✅ 直接收 | ✅ 直接收 | ⚠️ 業者付（不管有無租出）|
| 管理責任 | 房東自己 | 業者代管 | 業者全權 |
| 空置風險 | 房東自負 | 房東自負 | 業者自負 |
| 修繕責任 | 房東自負 | 業者代叫（房東付）| 業者負責 |
| 廠商選擇 | 房東決定 | 業者建議（房東同意）| 業者決定 |
| 租金收入 | 全收 | 全收−代管費 | 業者付固定租金 |
| 適合對象 | 時間多、物件少 | 繁忙房東 | 不想管、要穩定 |

---

## 代管合約重點條款

### 服務範圍
- [ ] 代管範圍（招租、帶看、簽約、收租、修繕、催繳）
- [ ] 不包含服務（重大修繕、訴訟代理）
- [ ] 代管費率（通常為月租金**8%~15%**）

### 代管費用
- 一般代管費：月租金之**10%~12%**
- 首次招租服務費：通常**1個月租金**（一次性）
- 續約服務費：**0.5個月**或免收

### 修繕權限
- 小額修繕（<$5,000）：業者可自行決定
- 中額修繕（$5,000~$30,000）：需房東書面同意
- 大額修繕（>$30,000）：需房東書面同意

### 合約期限
- 一般代管合約：**1~3年**
- 續約自動展延條款（如雙方無異議自動續約）

---

## 包租合約重點條款

### 租金計算
- 包租租金：市場租金之**6~8折**
- 租金調整機制（每1~2年調整）
- 調幅上限（如：不超過**3~5%**）

### 業者責任
- 無論有無出租，業者須按月付租金
- 正常修繕由業者負擔（結構性問題除外）
- 房客管理由業者全權負責

### 風險分擔
| 風險 | 房東負擔 | 業者負擔 |
|------|:--------:|:--------:|
| 空置 | — | ✅ |
| 欠租 | — | ✅ |
| 小額修繕 | — | ✅ |
| 結構修繕 | ✅ | — |
| 凶宅/重大事故 | ✅ | 依約定 |

### 注意事項
- 包租模式下房東與房客無直接關係
- 業者倒閉風險→選擇有規模的業者
- 合約需載明：業者違約時房東可直接收回房屋
- 建議：合約公證

---

## 選擇管理業者的檢查清單

| 檢查項目 | 重點 |
|----------|------|
| ✅ 合法登記 | 公司登記、租賃住宅服務業許可 |
| ✅ 營業規模 | 管理多少物件？ |
| ✅ 口碑評價 | 網路評價、前房東推薦 |
| ✅ 合約透明 | 費用項目全部列出 |
| ✅ 保險 | 有無投保專業責任險 |
| ✅ 修繕團隊 | 自有團隊還是外包 |
| ✅ 糾紛處理 | 有無法務或合作律師 |
| ✅ 財務透明 | 租金代收代付是否有系統對帳 |
