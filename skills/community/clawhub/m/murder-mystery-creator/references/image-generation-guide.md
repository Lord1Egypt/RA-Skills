# 可视化素材生成指南

> 本文件是 murder-mystery-creator skill 的图片生成规范。
> 目标：剧本生成后自动配套产出可打印/可上架的完整视觉素材包。
> 适用：所有风格（古风/现代/克苏鲁/修仙等）。

---

## 一、生成原则

**自动化全流程：**
- 剧本生成完成后，**自动识别需要生成的图片类型和数量**
- 无需用户额外指令，按剧本内容自动出图
- 输出格式统一为打印友好尺寸，可直接使用

**质量要求：**
- 每张图片必须包含清晰的**中文文字标签**（角色名/线索名/场景名）
- 风格必须与剧本类型匹配
- 所有图片统一命名规范：`[剧本名]_[类型]_[序号]_[名称].png`

**版权注意：**
- 生成图片仅供**内部游戏使用**，不用于商业销售素材
- 不生成含真实人物面孔的写实风格图片（用插画/概念风格代替）

### 类型2：线索卡图片（必须生成，数量≈线索总数的30-50%）

- 核心物证（凶器/关键书证/决定性证据）

- 有视觉识别度的物品（玉佩/令牌/图腾/特殊物品）

## 三、全自动流程（Skill 集成）

### 触发时机

**剧本第六步"打包输出"完成后，自动进入第七步"可视化素材生成"。**

### 执行顺序

```
1. 分析剧本内容
   → 识别角色数量 → 生成对应数量的角色肖像
   → 识别场景数量 → 生成场景地图（主要场景）
   → 识别线索类型 → 筛选生成30-50%的线索卡图片
   → 识别关键道具 → 生成道具图（如有）
   → 生成封面/海报（1张）

2. 生成图片（并行）
   → 使用多模态生成能力，按类型分批生成
   → 严格按本指南的 Prompt 模板填充内容
   → 统一输出尺寸和命名规范

3. 打包输出
   → 生成 `素材包/` 文件夹
   → 结构：
     素材包/
     ├── 封面/[剧本名]_封面.png
     ├── 角色肖像/[剧本名]_角色_[序号]_[角色名].png
     ├── 线索卡/[剧本名]_线索_[序号]_[线索名].png
     ├── 场景地图/[剧本名]_场景_[场景名].png
     ├── 道具图/[剧本名]_道具_[道具名].png
     └── 图片素材说明.md（说明每个文件的内容）
```

### 质量自检

生成完成后，检查以下内容：

- [ ] 所有角色肖像风格统一（同一剧本内）
- [ ] 所有线索卡风格统一（同一剧本内）
- [ ] 所有图片文字清晰可读（中文标签无乱码）
- [ ] 图片尺寸符合规范
- [ ] 文件命名规范正确
- [ ] 封面包含剧本名、风格、人数、时长
