# 可视化 App 教学/咨询使用指南

## 使用方式

读取本份的典型场景:
- 用户说"我要给学生 / 来访者演示一下"、"哪个 app 能教 X 概念"
- 已经在咨询里讲到某个动力学概念(振荡 / 击穿 / 配对),需要"插演示"
- 准备一节课 / 一次工作坊,需要选**演示组合**和**讲解顺序**
- 想知道某个 app 有什么预设、哪些参数可调、用什么提问引导

不要把这一份整个倒出。**按用户当前讨论的概念匹配 1–2 个 app**,给"演示什么 + 怎么操作 + 配什么提问"。

## 通用前置说明

- 12 个 app 全部在 `~/circapps/`,都是**单页 HTML**,**浏览器双击即可**(无需服务器)

## App 01 — RLC + DC 家庭响应器

**文件**:`app01-rlc-response.html`

**演示什么**:RLC + DC 在脉冲 / 周期 / 阶跃压力下的电流响应。这是元件级的"基础体检台"。

**关键控件**:
- 滑块:R / L / C / DC 偏置 / 压力幅值 / 频率
- 输入波形:脉冲 / 周期 / 阶跃
- 实时图:情绪流动波形(蓝=正向,红=穿过零线的负向,紫虚线=DC 基线)+ 能量分布

**适用场景**:
- 教元件基础(每个滑块独立调,看哪个改了什么)
- 教 AC vs DC、过零判据(DC 偏置滑到接近零,看波形多容易翻负)
- 演示 Gottman 5:1(DC 高 vs AC 大,直接看蓝/红比例)

**咨询/课堂提问**:
- "把 DC 滑到零,看波形 —— 这是失去承诺后的家庭画像,认识吗?"
- "R 改大改小,波形怎么变?加大 R 像家里多了什么?"
- "脉冲 vs 周期输入哪个让波形更乱?为什么?"

**配套理论**:Part 1 元件,Part 2 DC/AC 判据;Miller ch01 / ch06 / ch11。

## App 02 — 谐振频率探测器

**文件**:`app02-resonance.html`

**演示什么**:扫频显示 \|Z(ω)\| 曲线,看共振谷在哪;**R 改大压扁峰但不搬位置,L/C 改才搬 ω₀**。

**关键控件**:
- 滑块:R / L / C / 扫频范围
- 时域响应面板:在选定 ω 下看波形

**适用场景**:
- 教 ω₀ = 1/√(LC) 是"敏感频段"
- 演示 R 改 Q 不改 ω₀(临床决策树关键)
- 让伴侣自己识别"什么节奏的事最能引爆我们" → 测 ω₀

**咨询/课堂提问**:
- "把 L 调大,共振点往哪跑?这家变成对什么节奏的事敏感?"
- "如果只加大 R,共振点搬走了吗?那加 R 解决了什么、没解决什么?"

**配套理论**:Part 3 ω₀ 一节;干预决策树第 5 条(R 不搬 ω₀)。

## App 03 — Z 向量家庭类型诊断仪

**文件**:`app03-zvector.html`

**演示什么**:复阻抗平面上拖动黄点,实时定位 \|Z\| / φ / 所在象限,带 **Gottman 四类型预设**。

**关键控件**:
- **预设按钮**:Volatile 多变 / Validator 确认 / Avoider 逃避 / Hostile 敌对
- 拖动黄点(Z 向量末端)
- 滑块:R / L / C(滑块和拖动是双向同步的)

**适用场景**:**这是临床定位的首选 app**。一对夫妻进来,先按四类型预设各看一遍,问他们"哪一型像你们?"再让他们调到自我评估的位置。

**咨询/课堂提问**:
- "点 Volatile,这是激烈的灵魂伴侣型,你们认识这种吗?"
- "拖一下黄点,你觉得现在你们家在这个图的哪个位置?"
- "(标定后)你们想去哪里?(往 ⑤ 健康区拖)需要哪些 R/L/C 改变?"

**配套理论**:Part 3 五种家庭画像;Part 6 Gottman 四类型;Miller ch11。

## App 04 — 依恋类型 L 行为可视化器

**文件**:`app04-attachment.html`

**演示什么**:同一刺激下,Secure / Preoccupied / Dismissing / Fearful **四路波形并列对比**。

**关键控件**:
- 刺激选择(伴侣回家晚 / 一周冷淡 / 突发争吵等)
- 四个面板各自显示该依恋类型的电路反应

**适用场景**:
- Bartholomew 四类型直观演示
- 让来访者自己识别"我的反应像哪一型?他像哪一型?"
- 解释"为什么同一件事我俩感受完全不一样"(L 不同,反应函数完全不同)

**咨询/课堂提问**:
- "选一个最近真实发生的事件,看四种反应,哪个最像你?哪个最像他?"
- "Dismissing 的入向阻断 —— 你伴侣关心你的话,有传到你心里吗?"

**配套理论**:Part 6 第一档 ch01 依恋类型表;Bartholomew & Horowitz。

## App 05 — 串联 vs 并联共轭对比仪

**文件**:`app05-conjugate.html`

**演示什么**:**同样的 R、L、C**,接成串联和并联,在 ω₀ 处行为**完全相反** —— 串联共振即焚,并联陷波过滤。

**关键控件**:
- 共享参数 R / L / C
- 左右两个面板:串联 LC 频率响应 + 时域;并联 LC 频率响应 + 时域

**适用场景**:
- 这是**最容易被误解的概念** —— 用静态文字讲很多人 get 不到"为什么串/并差这么大",这个 app 一目了然
- 教"灵魂伴侣 = 燃烧型 = 串联共振"
- 讲 differentiation / cognitive interdependence 时演示拓扑改写的目标态

**咨询/课堂提问**:
- "你看左边和右边在 ω₀ 处反应完全相反 —— 同样的元件值,只是接法不同。这就是为什么'我们一模一样'是危险信号。"
- "右边并联的两条支路 = 你和他用不同的方式接同一份压力。这种'不一样'你们有吗?"

**配套理论**:Part 4 串/并联;Miller ch03 complementarity, ch14 cognitive interdependence。

## App 06 — 阻尼调节器(临界阻尼健康设计)

**文件**:`app06-damping.html`

**演示什么**:固定 LC 调 R,看二阶系统**特征根 s = -R/(2L) ± j·ω_d 在复平面如何移动**,以及对应的扰动响应。

**关键控件**:
- R 滑块(主调)
- L、C 也可调
- 复平面上特征根位置随 R 滑动
- "触发扰动"按钮 → 一次脉冲响应

**适用场景**:
- 教"每件事能不能收尾"
- 演示过 / 临界 / 欠阻尼三种态
- 让来访者识别"我们家的事是几天 / 几周 / 几月翻篇" → 在 R 滑块上找位置

**咨询/课堂提问**:
- "上一次大冲突真正翻篇用了多久?在这个滑块上找一下对应的 R 值。"
- "把 R 拖到 0,看波形 —— 这是慢性内耗,你们认识吗?"

**配套理论**:Part 4 阻尼一节;Miller ch11 冲突收尾。

## App 07 — 二极管 I-V 与心理现象映射

**文件**:`app07-diode-iv.html`

**演示什么**:**4 种二极管的 I-V 曲线** + 点击右侧具体心理现象自动加载对应参数。

**关键控件**:
- 二极管类型选择(普通 / Zener / 桥式整流 / LED)
- 现象列表(stonewalling / triggering / forgiveness / 积极错觉…)
- 蓝点 = 当前工作点

**适用场景**:
- 教非线性 4 性质(单向 / 阈值 / Zener / 整流)
- 把心理学术语和 I-V 曲线一一对应,让学生从"曲线形状"理解"为什么这个心理现象不能用 RLC 表达"

**咨询/课堂提问**:
- "选'微表情' —— 看这是 forward 阈值之下的漏电流。display rules 想关死,但漏出来一闪而过。"
- "选'宽恕' —— 这是 Zener 反向击穿。没有这个 Zener,反向能量没地方泄。"

**配套理论**:Part 5 全部;Miller ch04 / ch05 / ch10 / ch11。

## App 08 — Demand/Withdraw 半波整流回路

**文件**:`app08-demand-withdraw.html`

**演示什么**:A 推 B 退的死循环 —— A 的二极管"提出问题"方向导通,B 的二极管"回应"方向反向截止;A 的"挫败"电容慢慢充电直到击穿。**偶尔"打开" B 的二极管(回应)可以重置回路**。

**关键控件**:
- A 推动强度
- B 退避阈值
- "B 回应一下"按钮(临时打开 B 二极管)
- 节点动画 + 挫败电容电荷曲线

**适用场景**:
- 演示 Christensen-Heavey 经典模式
- 让追-避双方各自看见"自己的二极管"
- 演示"偶尔回应一下"如何重置回路 —— 而不是要求结构改变

**咨询/课堂提问**:
- "A 你看你的箭头 —— 你推过去 B 没接,挫败一直在涨。"
- "B 你按一下'回应' —— 看挫败电容怎么了?这是你**不需要解决问题、只是接住** 的电路效果。"

**配套理论**:Part 5 二极管组合;Miller ch11 demand-withdraw;Christensen & Heavey。

## App 09 — Capitalization LED-Photodiode 配对

**文件**:`app09-capitalization.html`

**演示什么**:**分享好消息(LED 发光) → 接收方转化(photodiode 接光) → 关系满意度增益**。**只有 Active-Constructive 让回路真正点亮**。

**关键控件**:
- 分享方强度
- 接收方响应风格四选一:Active-Constructive / Active-Destructive / Passive-Constructive / Passive-Destructive
- 三阶段动画 + 四种风格总增益对比柱状图

**适用场景**:
- 教 Gable capitalization
- 让伴侣识别"我平时是哪种响应风格"
- 演示**为什么只有 AC 有用**(其他三种都让 photodiode 接收效率掉到 ≤0)

**咨询/课堂提问**:
- "你伴侣昨天告诉你升职,你的反应是哪种?(选)看回路有没有亮?"
- "AC 是唯一让回路完全点亮的 —— 它需要什么?(具体的兴奋 + 追问 + 共同想象)"

**配套理论**:Part 5 二极管组合;Miller ch07 capitalization;Gable。

## App 10 — Forgiveness Zener 演示器

**文件**:`app10-forgiveness-zener.html`

**演示什么**:背叛事件 = 反向电压尖峰。**无 Zener 的关系直接被打穿;有 Zener 的关系让反向电荷受控泄放,工作点可恢复**。Zener 阈值过低 / 过高 / 适中三种结果对比。

**关键控件**:
- 背叛冲击强度(反向脉冲幅值)
- Zener 阈值
- DC 偏置
- 三场景对比:无 Zener / 低 Zener / 高 Zener / 适中 Zener

**适用场景**:
- 教宽恕的电路本质
- 演示低 Zener(任何小事就当背叛)和高 Zener(扛但不放,后期突然崩)的双失败模式
- IPV / Reactance / 背叛恢复案例的可视化

**咨询/课堂提问**:
- "你看无 Zener 这条 —— 一次背叛 DC 直接掉到负。这是没有任何修复机制的电路。"
- "调 Zener 阈值找到甜区 —— 太低?太高?中间是什么样的关系状态?"

**配套理论**:Part 5 Zener;Miller ch10 forgiveness/betrayal;ch12 IPV;ch03 Reactance。

## App 11 — 家庭电路设计器(沙盒)

**文件**:`app11-circuit-designer.html`

**演示什么**:**五种典型病电路模板** → 通过调 RLC + Zener + DC 把"病电路"逐步修复成健康电路。每个参数变化即时反映在响应里。

**关键控件**:
- 起始模板:五种病(燃烧 / 室友 / 隐忍 / 惯性 / 击穿)
- RLC 元件 / Zener / DC 全部可调
- 健康指标综合评分(实时打分)
- Z 平面位置(目标:推到绿色健康区)

**适用场景**:
- **这是最核心的"咨询沙盒"** —— 让来访者亲手把自己家的电路改健康
- 工作坊 / 团体咨询:每对夫妻选自己最像的模板,改 30 分钟,讨论改了什么
- 给学生留作业:"把燃烧型改成 validator,要哪些 R/L/C 变化?"

**咨询/课堂提问**:
- "(选燃烧型)看你们现在在 Z 平面哪里?评分多少?"
- "你想先改哪一个?为什么?(让他们体会六手干预的优先级)"
- "你刚刚加大 R,评分涨了 —— 在你们生活里这个'加 R'具体是什么?"

**配套理论**:综合 Part 3 + Part 4 + Part 7 干预清单;Miller ch11 / ch14。

## App 12 — VSA 时间线模拟器

**文件**:`app12-vsa-timeline.html`

**演示什么**:**Karney & Bradbury VSA 模型** = RLC 参数随时间在 stressors 作用下的演化。看一段关系如何从初始状态走向稳定 / 渐弱 / 终结。**带蒙特卡洛 50 次模拟分布**。

**关键控件**:
- 初始 vulnerabilities(起步 RLC)
- Stressor 强度
- Adaptation 规则强度
- 月级别参数漂移曲线
- 关系结局分布(稳定 / 渐弱 / 终结的概率)

**适用场景**:
- 教 VSA 模型
- 给"长程关系研究"或"为什么有的婚姻 5 年崩有的 30 年"提供动力学解释
- 演示"初始状态 + 压力 + 适应能力"的概率分布,而非确定性预测

**咨询/课堂提问**:
- "调高初始 vulnerabilities,蒙特卡洛分布怎么变?"
- "调高 adaptation 但保持 stressors —— 这家从'终结'走到'稳定'用了几次?"

**配套理论**:Part 6 第一档 ch13 VSA 模型;Karney & Bradbury (1995)。

## 推荐演示插入时机

源 SKILL.md 已强调:**app 是工具不是装饰**。只在以下时机推荐:

1. 用户**明确要可视化** —— 直接给路径
2. 你正在解释一个**动力学概念**(振荡 / 谐振 / 阻尼 / 击穿 / 整流)用静态文字讲不清楚 —— 主动推
3. 用户已经在 Z 平面定位 / 拓扑判断 / 干预选择 → **下一步有具体可执行的演示** —— 主动推
4. 工作坊 / 课堂场景下 —— 给套餐组合

**不要在每次回答都推 app**。讲清楚静态映射后,再用 app 演示**动力学**(波形如何随时间演化)。app 不替代解释,是解释的"实验台"。
