# 2026-02-22 - 飞书技能和模型配置

## 飞书技能配置

### 可用的飞书技能

1. **feishu-doc** - 飞书文档操作
   - 读取飞书文档
   - 写入飞书文档
   - 追加内容到文档
   - 创建新文档
   - 管理文档块
   - **权限**: `docx:document`, `docx:document:readonly`, `docx:document.block:convert`, `drive:drive`

2. **feishu-drive** - 飞书云盘管理
   - 列出云盘文件
   - 获取文件信息
   - 创建文件夹
   - 移动文件
   - 删除文件
   - **权限**: `drive:drive` (完整访问), `drive:drive:readonly` (只读)
   - **注意**: 机器人没有自己的"我的空间"，只能访问共享给它的文件夹

3. **feishu-perm** - 飞书权限管理
   - 管理文档和文件的分享权限
   - 设置协作者权限
   - **权限**: `drive:permission`
   - **注意**: 默认禁用，需要手动启用

4. **feishu-wiki** - 飞书知识库管理
   - 查看知识空间
   - 管理知识库节点
   - 创建、移动、重命名节点
   - **权限**: `wiki:wiki` 或 `wiki:wiki:readonly`
   - **依赖**: 需要 feishu-doc 启用（wiki 页面是文档）

### 飞书应用权限

- 已授予 66 个权限
- 包括消息发送、聊天管理、文档操作等
- 无待审批权限
