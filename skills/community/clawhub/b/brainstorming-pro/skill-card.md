## Description: <br>
Facilitates structured ideation sessions for features, products, content, and strategy. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Employees, external collaborators, and product teams use this skill to run structured ideation sessions that clarify goals, generate ideas, converge on priorities, document outcomes, and plan follow-through. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may inspect project files and move toward design-document writes or git commits without a clear approval checkpoint. <br>
Mitigation: Define the repository scope before use and require explicit approval before file writes, commits, external updates, ticket creation, or recorded sessions. <br>
