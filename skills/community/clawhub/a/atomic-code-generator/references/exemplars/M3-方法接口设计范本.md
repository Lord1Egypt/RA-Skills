## M3-01 接口设计

### 输入示例
方法定义：validateUsername、validatePassword、authenticateUser

### 处理过程
1. 设计接口规范
2. 定义接口签名
3. 验证接口的合理性

### 输出示例
**接口设计**：
```typescript
interface ValidationResult {
  valid: boolean;
  error?: string;
}

interface User {
  id: string;
  username: string;
  email: string;
}

function validateUsername(username: string): ValidationResult;
function validatePassword(password: string): ValidationResult;
function authenticateUser(username: string, password: string): User|null;
```
