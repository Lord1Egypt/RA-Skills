## 核心理论

### 一、刻应

**定义**：刻应，即出生时间所对应的数。

**求法**：
1. 以出生年干支求数
2. 以出生月日求数
3. 以出生时辰求数
4. 三数相加，得刻应

**示例**：
```
出生：甲子年 正月 初一 子时
年干支数：甲子=1
月日数：正月=1，初一=1 → 1+1=2
时辰数：子时=1
刻应：1+2+1=4
```

### 二、刻应表

**格式**：
```
刻应X：
  事业：...
  财运：...
  婚姻：...
  健康：...
  ...
```

**示例**：
```
刻应4：
  事业：早年奔波，中年安定，晚年享福
  财运：财运平平，靠技艺谋生
  婚姻：婚姻迟，配老配偶
  健康：健康平，注意心肺
  ...
```
