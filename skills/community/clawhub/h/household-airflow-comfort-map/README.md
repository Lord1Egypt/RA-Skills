# Household Airflow Comfort Map

为沉闷房间创建舒适气流地图、定时通风方案和每周回顾清单

## What it does
- 按房间绘制气流舒适地图，记录舒适问题、可用通风资源与限制条件
- 制定每日定时通风方案（开关窗、风扇方向、遮阳帘使用）
- 提供舒适观察日志模板与每周回顾清单，持续优化调整

## Example scenarios

**1. 房间冷热不均**
> 👤 Map my 3-bedroom apartment. Master bedroom is always hot, living room is freezing.
> 🤖 房间逐个绘制气流图。识别主卧出风口离HVAC最远且被家具部分遮挡，建议出风口增压风扇并调整塔扇朝向

**2. 夏夜用风扇替代空调**
> 👤 It's summer. I want to use window fans instead of AC at night. Which windows should I open and which fans should face in vs. out?
> 🤖 夜间降温方案：北向窗户进风扇吸入，南向窗户排风扇排出。配合室外温度在晚上10点左右降到低于室内温度的时间表

**3. 夏天开空调关窗闷得慌**
> 👤 夏天开了空调必须关窗，但时间长了觉得浑身无力、头昏、空气不新鲜。开窗通风又浪费冷气。怎么平衡？
> 🤖 四种方案：A（有新风）开新风+空调；B（无新风）每天早晚各开窗15分钟，开空调时窗开一条缝（3-5cm）正压换气；C：空调26-27度+风扇朝窗外吹抽浊气；D：桌面空气循环扇+净化器降低CO₂浓度
