# Edu Content Factory 使用说明

### 为什么之前的结构不对？
AI 倾向于快速给出“最终成稿”。但在本系统中，**AI 被降级为“提案人”**。
如果 AI 直接给你文案，请立即打断并要求它回到 Step 1。

### 进阶用法：注入“张雪峰”灵魂
当你在 Phase 2 感到内容太温吞时，输入指令：
> “用张雪峰的视角重写逻辑，重点强调：‘普通家庭的选择容错率为零’，语气要狠。”

### 发布前必做
运行 `python3 scripts/check_logic.py`。
如果得分低于 80 分，说明 AI 又开始“偷懒”用模版写话了。
