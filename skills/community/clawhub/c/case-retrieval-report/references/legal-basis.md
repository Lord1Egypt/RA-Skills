# 类案检索制度法律依据

## 一、核心制度依据

### 1. 《最高人民法院关于统一法律适用加强类案检索的指导意见（试行）》
**发文机关**：最高人民法院  
**发布日期**：2020年7月27日  
**文号**：法〔2020〕130号

**核心条款摘要**：

- **第2条（检索范围优先级）**：
  类案检索范围依次为：（一）最高人民法院发布的指导性案例；（二）最高人民法院发布的典型案例及裁判生效案件；（三）本省（自治区、直辖市）高级人民法院发布的参考性案例及裁判生效案件；（四）上一级人民法院及本院裁判生效的案件。
  > *检索时间范围：除指导性案例外，一般以近三年的案例为主。*

- **第4条（参照与参考的区分）**：
  检索到的案例为指导性案例的，**应当**参照；检索到其他类案的，可以作为参考。

- **第5条（类案识别标准）**：
  类案是指与待决案件在**基本事实、争议焦点、法律适用问题**上具有相似性的案件。

- **第7条（报告格式要求）**：
  检索报告应当包括：检索目的、检索平台、检索条件、检索结果及结论。

### 2. 《最高人民法院关于完善统一法律适用标准工作机制的意见》
**发文机关**：最高人民法院  
**发布日期**：2020年9月14日

## 二、"参照"与"参考"的区别（实务要点）

| 维度 | 参照（适用于指导性案例） | 参考（适用于其他类案） |
|-----|----------------------|---------------------|
| **法律约束力** | 类似法律义务，法官应当遵照适用 | 裁量性参考，无硬性约束 |
| **偏离要求** | 偏离须在裁判文书中说明理由 | 参考与否由法官裁量 |
| **报告表述** | "应当参照第X号指导性案例" | "可参考（2021）最高法XX号的裁判思路" |

---

## 三、检索报告应用场景

### （一）向法院提交
- **适用情形**：诉讼代理人将类案检索报告作为辩论材料向法院提交
- **格式要求**：报告须包含检索目的、检索条件、结论及具体类案信息
- **注意事项**：报告须客观呈现有利与不利类案，不得选择性呈现

### （二）内部策略研判
- **适用情形**：律师/法务在案件受理后进行裁判风险评估
- **格式要求**：可在标准格式基础上增加策略分析内容
- **侧重点**：倾向性判断与策略建议

---

## 四、特殊注意事项

### 指导性案例的特殊效力
1. 检索到指导性案例时，应当在报告中**明确标注**
2. 在分析说明部分论证**应当参照**的理由
3. 如本案与指导性案例存在差异，**必须**详细论证差异是否足以排除参照适用
4. 不得以"存在细微差异"为由轻易排除参照

### 系列案件处理规则
- 系列案件（同一投资项目，除当事人外其他内容基本相同）**合并统计为1案**
- 在脚注中说明合并理由及合并案件数量
- 利用 API `seriesCaseId` 或 `groupId` 字段识别关联案件

### 不利类案的处理规则
- **不得**因 API 返回数据的便利性刻意回避相反案例
- 对不利类案须主动分析与待决案件的**差异性**
- 差异分析应具体、充分，而非笼统表述"本案情况不同"
