## 零、测试用例可执行性检查（所有测试类型通用）

- [ ] 测试步骤中是否给出了具体的参数值/请求体/路径值（而非描述性语言）

- [ ] 请求参数是否给出了具体的 key=value 值
- [ ] 大量数据场景（如超大请求体、超长字符串参数）是否用描述性语言 + 明确参数说明
