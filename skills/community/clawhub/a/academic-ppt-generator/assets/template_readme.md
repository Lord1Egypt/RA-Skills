# PPT模板说明

本Skill使用代码自动生成PPT，不需要预定义模板文件。

## 生成的PPT特点

### 视觉设计
- **配色方案**: 学术蓝绿色系 (#1a5f7a, #57a0d3, #f4a261)
- **版式**: 16:9 宽屏比例
- **字体**: 使用系统默认字体，确保兼容性

### 页面结构
1. **标题页** - 论文标题、作者、期刊、年份
2. **章节分隔页** - 四大部分清晰分隔
3. **内容页** - 要点式布局，支持演讲者备注

### 章节安排
- **Part 1**: 科学问题与背景 (1-2页)
- **Part 2**: 研究方法 (2-3页)
- **Part 3**: 结论与验证 (多页)
- **Part 4**: 局限与展望 (1页)
- **总结页**: 核心贡献

## 自定义建议

如需调整样式，可修改 `scripts/export_ppt.py` 中的:
- `COLORS` 字典 - 修改配色
- 字体大小参数 - 调整文字尺寸
- 布局尺寸 - 修改页面元素位置
