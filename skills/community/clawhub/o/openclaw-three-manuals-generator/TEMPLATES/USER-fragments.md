## 工作场景碎片

### 常见场景

| 场景 | 描述 | 典型需求 |
|------|------|----------|
| 日常工作 | 邮件、文档、报告 | 效率、准确 |
| 技术任务 | 代码、架构、调试 | 严谨、可靠 |
| 决策分析 | 投资、选择、策略 | 深度、风险 |
| 内容创作 | 文章、想法、创意 | 洞察、共情 |
| 学习成长 | 读书、课程、技能 | 耐心、启发 |
| 信息管理 | 笔记、收藏、整理 | 系统、条理 |

### 场景组合示例

```
## 工作场景

### 主要场景
1. **[场景1]** — [具体做什么]
2. **[场景2]** — [具体做什么]

### 场景频率
- [场景1]: 每天
- [场景2]: 每周2-3次
- [场景3]: 偶尔
```

## 沟通偏好碎片

### 交互风格

| 风格 | 描述 | 适合人群 |
|------|------|----------|
| 指令型 | 直接给命令，让AI执行 | 清楚自己要什么 |
| 讨论型 | 先聊思路，再让AI执行 | 需要思考的人 |
| 提问型 | 希望AI多问，自己想明白 | 学习型用户 |
| 选择型 | 希望AI给选项，自己挑 | 决策困难户 |

### 偏好描述示例

```
## 沟通偏好

### 喜欢的交互方式
- [方式1]
- [方式2]

### 讨厌的交互方式
- ❌ [讨厌的方式1]
- ❌ [讨厌的方式2]

### 典型对话
> 用户：[说什么]
> AI：[怎么回应]
> 用户：满意/不满意
```
