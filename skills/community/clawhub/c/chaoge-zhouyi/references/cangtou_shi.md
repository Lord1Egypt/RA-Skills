# 藏头诗

## 概述

《藏头诗》，传为唐代李淳风撰，是一部**预言后世**的谶纬之作。全诗以藏头诗形式写成，每句首字连读，可得预言。

---

## 作者简介

**李淳风**（602-670），唐代著名天文学家、数学家、预言家。与袁天罡合著《推背图》。相传撰《藏头诗》《乙巳占》等。

## 核心内容

### 一、藏头诗全文

**序**：
```
藏头诗者，藏头露尾，首字连读，可得预言。
```

**正文（节选）**：
```
唐祚绵延三百秋，
后梁继统位迁流。
周武继之国祚休，
汉业重光定九州。
...
（全文甚长，在此节选）
```

**藏头**：
取每句首字，连读得："唐后周汉..."

### 二、预言内容

**唐朝**：
1. 唐祚绵延三百秋
2. 唐朝国祚约300年

**五代**：
1. 后梁继统位迁流
2. 后梁继唐统

**周**：
1. 周武继之国祚休
2. 周武继统

**宋**：
1. 汉业重光定九州
2. 宋朝统一九州

**未来**：
1. 天下大同
2. 世界和平
3. 科技昌明

### 四、与推背图、烧饼歌、梅花诗、马前课比较

**相同点**：
1. 都是预言书
2. 都预测国运
3. 都含蓄隐晦

**不同点**：
1. 推背图：唐代，六十象，图文并茂
2. 烧饼歌：明代，歌诀体，文字含蓄
3. 梅花诗：宋代，十首诗，诗意含蓄
4. 马前课：三国，十四课，拆字含蓄
5. 藏头诗：唐代，藏头诗体，首字连读
