# D02 财富领域任务目录

## 领域定义

财富领域(D02)关注个体资源的获取、管理、增值和传承。核心驱动需求：N1生存 + N2安全。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人收入管理 | 企业财务管理 |
| 支出预算控制 | 金融产品设计 |
| 储蓄规划 | 投资银行服务 |
| 投资理财 | 证券交易操作 |
| 风险管理 | 保险精算 |
| 税务筹划 | 税务代理服务 |
| 财富传承 | 遗产执行 |
