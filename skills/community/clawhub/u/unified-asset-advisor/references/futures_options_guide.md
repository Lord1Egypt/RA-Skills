# 期货期权策略框架

> **定位说明：本文件为 fallback 参考。** 期货品种映射表用于确定查询哪些合约，策略框架用于判断方向。但**具体合约参数（最新价、成交量、持仓量）**优先通过 neodata-financial-search 实时拉取；**期权链数据**优先通过 option-strategy-advisor 实时拉取。仅当 API 查询失败时，回退到本文档的静态映射，并标注"无实时行情"。

## 策略选择：按资产周期

| 周期阶段 | 期货策略 | 期权策略 | 说明 |
|---------|---------|---------|------|
| 上升期（看多） | **做多**对应期货合约 | **买入看涨期权** / 牛市价差 / 卖出看跌期权 | 方向性看多，控制风险可选用牛市价差 |
| 下跌期（看空） | **做空**对应期货合约 | **买入看跌期权** / 熊市价差 / 卖出看涨期权 | 方向性看空，避免裸卖 |
| 震荡期（横盘） | **观望**或区间操作 | **卖出跨式/宽跨式** / 铁鹰价差 / 卖出虚值期权 | 赚取时间价值，收取权利金 |
| 不确定（剧烈波动） | **减仓观望** | **买入跨式/宽跨式** | 做多波动率 |

## 策略参数建议模板

给出操作建议时，应包含：

```
品种：XXX（期货合约代码 / 期权合约名称）
方向：做多 / 做空 / 卖出期权
推荐策略：XXX策略名称
具体参数：
  - 行权价：XXXX（仅期权）
  - 到期月：XXXX年X月
  - 建议仓位：X%
风险提示：最大亏损 / 盈亏比 / 注意事项
```
