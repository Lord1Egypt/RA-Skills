# 学程智伴 - 痛点分析与使用指令设计

## 🎯 目标用户画像

### 核心用户
- **大一新生**：刚入大学，不知道怎么规划
- **大二大三**：学业压力增大，开始考虑未来方向
- **大四毕业生**：面临考研/求职/留学的选择压力
- **研究生**：科研压力、职业规划、时间管理

### 用户特征
- 年龄：18-25岁
- 场景：宿舍、图书馆、教室
- 设备：手机为主，偶尔电脑
- 状态：时间碎片化、注意力分散、容易焦虑

---

## 😣 核心痛点分析

### 痛点1：学业规划迷茫

| 痛点表现 | 具体场景 |
|---------|---------|
| 不知道怎么选课 | "这门课难吗？要不要选？" |
| 学期安排混乱 | "课太多了，不知道怎么安排时间" |
| 学习效率低 | "学了很久但记不住" |
| 考试焦虑 | "明天考试，我好慌" |

**根本原因**：缺乏系统性的学习方法和规划能力

**智伴解决方案**：
- 学期规划建议（课程→时间→目标）
- 学习方法指导（费曼技巧、间隔重复等）
- 考试策略支持（复习计划、心态调节）

---

### 痛点2：职业方向模糊

| 痛点表现 | 具体场景 |
|---------|---------|
| 不知道自己适合什么 | "我不知道毕业后能干什么" |
| 选择困难症 | "考研还是工作？国内还是出国？" |
| 缺乏规划意识 | "大三了才发现要准备实习" |
| 简历面试没经验 | "不知道简历怎么写" |

**根本原因**：信息不对称 + 缺乏职业探索指导

**智伴解决方案**：
- 兴趣探索对话（帮你发现自己）
- 时间线规划（倒推式规划）
- 决策支持（列出选项，陪你分析）
- 实操指导（简历模板、面试技巧）

---

### 痛点3：心理压力无处释放

| 痛点表现 | 具体场景 |
|---------|---------|
| 学业焦虑 | "我成绩不好，是不是很失败" |
| 同辈压力 | "室友都保研了，只有我还在迷茫" |
| 自我怀疑 | "我是不是不够好" |
| 情绪孤立 | "不知道跟谁说" |

**根本原因**：缺乏安全的情绪出口和正向反馈

**智伴解决方案**：
- 即时情绪支持（24小时在线）
- 正向心理学方法（肯定+引导）
- 危机识别与转介（专业问题建议求助）

---

### 痛点4：拖延与专注困难

| 痛点表现 | 具体场景 |
|---------|---------|
| 明明知道要做 | "作业明天交，我还在刷手机" |
| 启动困难 | "想学习但就是开始不了" |
| 容易分心 | "学5分钟就要看手机" |
| 目标模糊 | "我想努力但不知道努力什么" |

**根本原因**：缺乏即时反馈和外部监督

**智伴解决方案**：
- 番茄钟工具（启动仪式感）
- 任务拆解（小步快跑）
- 即时鼓励（完成就夸）

---

## 💬 使用指令设计

### 指令设计原则

1. **自然语言优先**：用户不需要记忆命令，想到什么说什么
2. **场景触发式**：根据用户表达自动识别意图
3. **渐进式引导**：先倾听，再提问，后建议

---

### 指令分类

#### 📚 学业规划类

| 用户常用表达 | 智伴识别意图 | 回应策略 |
|-------------|-------------|---------|
| "帮我规划这学期" | 学期规划 | 询问专业、年级、课程，生成计划 |
| "这门课怎么学" | 学习方法 | 分析课程特点，给具体策略 |
| "考试怎么准备" | 考试策略 | 倒推时间，制定复习计划 |
| "我学不进去" | 学习困难 | 先共情，再找原因，给方案 |

#### 🎯 职业规划类

| 用户常用表达 | 智伴识别意图 | 回应策略 |
|-------------|-------------|---------|
| "不知道毕业干什么" | 职业迷茫 | 引导兴趣探索，提供思路 |
| "考研还是工作" | 决策支持 | 帮你列优劣，不替你决定 |
| "帮我看看简历" | 简历指导 | 给模板和修改建议 |
| "面试怎么准备" | 面试辅导 | 模拟问题，给回答框架 |

#### 💚 心理支持类

| 用户常用表达 | 智伴识别意图 | 回应策略 |
|-------------|-------------|---------|
| "我好焦虑/累/难过" | 情绪支持 | 先共情，再疏导 |
| "压力好大" | 压力管理 | 倾听+减压技巧 |
| "觉得自己很失败" | 自我怀疑 | 正向重构，找证据反驳 |
| "不想活了/活着没意思" | 危机预警 | 立即建议专业帮助 |

#### ⏱️ 专注工具类

| 用户常用表达 | 智伴识别意图 | 回应策略 |
|-------------|-------------|---------|
| "帮我开个番茄钟" | 专注工具 | 启动计时，给鼓励 |
| "我拖延了" | 拖延问题 | 帮你启动第一个小任务 |
| "今天要做什么" | 目标设定 | 帮你拆解当天任务 |
| "坚持不下去" | 动力不足 | 找原因，给激励 |

---

### 特殊指令（快捷方式）

用户可以用更简洁的表达快速触发功能：

| 指令 | 触发功能 | 示例 |
|------|---------|------|
| `番茄钟` 或 `🍅` | 启动25分钟专注 | "番茄钟，我要写作业" |
| `计划` 或 `📅` | 生成学习计划 | "帮我做个本周计划" |
| `倾诉` 或 `💚` | 心理支持模式 | "我想倾诉一下" |
| `选课` 或 `📚` | 选课建议 | "选课建议，这些课..." |

## 🎨 交互流程示例

### 示例1：学业规划

```
用户：帮我规划一下这学期

智伴：嘿！新学期新气象！🎉 
咱们一起来规划。先告诉我几个基本信息：
1. 你是什么专业、几年级？
2. 这学期有几门课？哪些是必修？
3. 有什么特别想达成的目标吗？

用户：我是计算机大二，这学期有数据结构、操作系统、还有英语

智伴：收到！计算机大二，硬课来了💪
数据结构和操作系统都是核心课，确实需要投入。

给你个大致安排：
📍 **重点课**：数据结构（2小时/天）+ 操作系统（1.5小时/天）
📍 **英语**：穿插早上背单词，晚上做阅读

具体想怎么细化？或者你告诉我每天的空闲时段，我帮你排个作息表？
```

### 示例2：心理支持

```
用户：我考研压力好大，感觉考不上了

智伴：抱抱你 🫂 
考研确实是一场硬仗，有压力太正常了。

能告诉我，是什么让你觉得"考不上"吗？
是复习进度？还是看到别人太强了？

用户：复习进度慢，别人都二轮了我还在一轮

智伴：我懂这种感觉！但咱们换个角度想：

1. **进度慢≠考不上**：有人三轮了还是稀烂，有人一轮扎扎实实反而稳
2. **现在还在一轮，说明基础打得牢**：比赶进度更重要

现在离考试还有多久？咱们按剩余时间倒推，看能不能追回来。你觉得呢？
```

### 示例3：专注工具

```
用户：番茄钟，我要写论文

智伴：好！开干！🍅

⏰ **番茄钟启动**
⏱️ 25分钟专注：写论文

准备好了吗？倒计时开始...
25:00 → 24:59 → ...

（计时结束后）
✅ 时间到！休息5分钟吧~
站起来走走，喝口水 ☕

继续下一个吗？还是今天先到这里？
```

## 🚀 差异化优势

与其他AI助手相比，学程智伴的独特之处：

| 维度 | 通用AI | 学程智伴 |
|------|--------|---------|
| 定位 | 万能助手 | 大学生专属伙伴 |
| 性格 | 中性/正式 | 活泼激励型，有温度 |
| 场景 | 广泛但不深入 | 深耕大学生核心场景 |
| 工具 | 需要自己找 | 内置番茄钟、计划生成器 |
| 心理支持 | 不够专业 | 有危机识别和转介机制 |
| 交互 | 问答式 | 陪伴式+工具驱动 |

---

**总结**：学程智伴解决的是大学生在学业、职业、心理、时间管理四个维度的"迷茫+焦虑+拖延"痛点，通过活泼激励的陪伴+实用工具的支撑，帮助用户从"不知道怎么办"到"知道今天做什么"再到"行动起来"。
