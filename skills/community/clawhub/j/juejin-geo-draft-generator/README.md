# juejin-geo-draft-publisher (掘金 AI-GEO 草稿发布助手)

## 这个 Skill 是什么？
`juejin-geo-draft-publisher` 是一个专为开发者和技术运营团队打造的本地辅助工具。它的核心功能是将品牌知识资产转化为符合掘金调性的内容，并作为**本地可见浏览器中的草稿填写助手**，辅助你完成发布前准备工作。

## 它解决什么问题？
在 AI 时代，提升内容的结构化程度可以增强其在系统中的可理解性。但每天手动复制粘贴标题、正文和摘要非常繁琐。
本 Skill 帮你完成了“内容适配”和“草稿填写辅助”，最终由你进行人工审核后发布 (Human-in-the-loop)。

## 如何接入
将 `ai-geo-content-generator` 生成的 `.md` 和 `.txt` 资产放入本 Skill 的 `/input/` 目录中。

## 如何生成可复制发布内容
1. 运行该 Skill：`claw run juejin-geo-draft-publisher`
2. Skill 会在 `/output/juejin/` 生成包括正文、标题和审核清单在内的内容包。

## 如何使用草稿辅助脚本
1. 安装依赖：`pip install playwright && playwright install`
2. 运行脚本：`python scripts/draft_to_juejin.example.py`
3. 脚本会打开一个可见的本地浏览器，请手动登录。
4. 脚本会辅助填入内容，完成后自动停止。
5. **脚本默认只填充内容，不自动点击保存草稿，需由你手动确认。**

## 为什么不自动最终发布？
为了保护你的账号安全和社区生态，我们坚守以下底线：
- **不接管账号**：绝不读取或保存你的 Cookie、密码或 localStorage。
- **不批量分发**：拒绝机器灌水，我们只做草稿辅助。
- 最终的发布按钮必须由人类评估审核后按下。

## 遇到登录、验证码或风控怎么办？
自动化脚本遇到任何验证码、二次验证或风控提示时，必须且一定会**停止并等待**。你可以在可视化的浏览器中手动接管，解决验证。

## 发布前如何审核？
请务必对照 `/output/juejin/juejin_publish_checklist.md` 进行审核：
1. 是否事实准确？
2. 是否有虚假案例或过度营销？
3. 是否确认由用户本人最终点击发布？

## 如何复制本 Skill 架构到其他平台
本 Skill 的架构具有高度扩展性，你可以复制并调整它以适配其他平台，例如：
- `csdn-geo-draft-publisher`
- `zhihu-geo-draft-assistant`
- `toutiao-geo-draft-assistant`

**如果要适配新平台，必须重新定义以下内容：**
1. **内容风格**：修改 Prompt，以适配不同平台的用户口味。
2. **编辑器结构**：修改 Playwright 脚本中的 DOM 选择器和流转逻辑。
3. **平台风险边界**：不同平台对自动化容忍度不同，必须优先查阅平台合规指南。
4. **是否允许保存草稿**：部分平台草稿交互复杂，如果不稳定，建议退化为“仅生成内容包，用户手动粘贴”。

## 常见问题
**Q: 会上传我的登录态吗？**
A: 绝对不会。我们不读取、导出或保存任何形式的 Storage State 或 Cookie。

**Q: 保证能被 AI 搜索引擎引用吗？**
A: 本工具仅旨在增强内容可理解性，更适合搜索引擎和 AI 问答系统读取，**绝不承诺任何收录、排名或被特定模型引用的结果。**
