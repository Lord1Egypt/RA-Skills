# D03 教育领域工作流范本

教育领域的完整工作流范本，用于样本法参考。每个范本包含完整的执行流程、校准点和跨域链接。

## 范本 1：系统化学习全流程方案

### 背景
- **用户画像**：28岁职场人士，想系统提升专业技能
- **核心需求**：建立覆盖学习规划、知识获取、技能培养的全流程学习方案
- **涉及子域**：E1学习规划 + E2知识获取 + E3技能培养 + E5学习资源

### 执行流程

```
S-01(需求分析) → C-01(学习规划) → C-02(资源准备) → A-01(学习执行) → O-01(能力评估)
```

#### Step 1: 需求分析 (S-01)
- **E1-01 学习目标设定**：设定明确、可量化的学习目标
- **E1-02 学习需求分析**：分析当前能力与目标的差距
- **E3-01 技能需求分析**：分析需要培养的技能类型和水平
- **E5-01 学习资源需求分析**：分析学习所需的资源类型

**校准点**：需求分析完整性确认

#### Step 2: 学习规划 (C-01)
- **E1-03 学习路径规划**：规划从现状到目标的学习路径
- **E1-04 学习计划制定**：制定详细的学习计划（时间/内容/方法）
- **E1-05 学习资源规划**：规划所需的学习资源和工具
- **E3-02 技能学习路径**：规划技能学习的路径和阶段

**校准点**：规划可行性确认

#### Step 3: 资源准备 (C-02)
- **E2-01 阅读方法指导**：指导高效阅读方法和技巧
- **E2-02 笔记方法指导**：指导有效的笔记方法和工具
- **E2-03 信息检索指导**：指导如何高效检索和筛选信息
- **E5-02 书籍资源推荐**：推荐相关的书籍资源
- **E5-03 在线课程推荐**：推荐相关的在线课程资源
- **E5-04 学习工具推荐**：推荐学习相关的工具和软件
- **E5-05 学习社区推荐**：推荐相关的学习社区和论坛
- **E5-06 学习资源评估**：评估学习资源的质量和适用性
- **E5-07 学习资源整合**：整合各类学习资源形成资源包

**校准点**：资源质量确认

#### Step 4: 学习执行 (A-01)
- **E1-06 学习时间管理**：管理学习时间，提高学习效率
- **E2-04 知识整理方法**：指导如何整理和组织知识
- **E2-05 深度学习策略**：指导如何进行深度学习和理解
- **E2-06 记忆增强技巧**：指导如何增强记忆和保持知识
- **E2-07 知识应用指导**：指导如何将知识应用到实践中
- **E3-03 技能练习方法**：指导技能练习的方法和技巧
- **E3-04 技能实践项目**：设计技能实践项目以巩固学习

**校准点**：学习效果确认

#### Step 5: 能力评估 (O-01)
- **E1-07 学习进度追踪**：建立学习进度追踪机制
- **E1-08 学习计划调整**：根据学习进展调整学习计划
- **E3-05 技能反馈机制**：建立技能学习的反馈和改进机制
- **E3-06 技能提升策略**：制定技能提升的策略和计划
- **E3-07 技能认证准备**：准备技能认证考试或评估
- **E3-08 技能应用指导**：指导如何在实际工作中应用技能

### 跨域链接
- **D04职业**：技能提升→职业发展
- **D08科技**：技术学习→数字化能力

## 范本 2：学习评估与终身学习方案

### 背景
- **用户画像**：35岁中层管理者，想建立终身学习习惯
- **核心需求**：建立覆盖学习评估、终身学习的完整方案
- **涉及子域**：E4学习评估 + E6终身学习

### 执行流程

```
S-01(现状评估) → C-01(问题诊断) → C-02(策略制定) → A-01(习惯养成) → O-01(效果评估)
```

#### Step 1: 现状评估 (S-01)
- **E4-01 学习效果评估**：评估学习目标的达成程度
- **E4-02 知识掌握测试**：测试知识掌握程度和理解深度
- **E4-03 技能水平评估**：评估技能掌握水平和应用能力

**校准点**：评估客观性确认

#### Step 2: 问题诊断 (C-01)
- **E4-04 学习方法评估**：评估学习方法的有效性
- **E4-05 学习障碍识别**：识别学习过程中遇到的障碍

**校准点**：问题识别准确性确认

#### Step 3: 策略制定 (C-02)
- **E4-06 改进建议生成**：基于评估结果生成改进建议
- **E6-01 学习习惯养成**：培养持续学习的习惯
- **E6-02 学习动机激发**：激发和维持学习动机
- **E6-03 学习时间规划**：规划长期的学习时间安排
- **E6-04 学习目标管理**：管理长期学习目标的设定和达成

**校准点**：策略可行性确认

#### Step 4: 习惯养成 (A-01)
- **E6-05 学习社群参与**：参与学习社群以获得支持和动力
- **E6-06 学习成果积累**：积累和整理学习成果
- **E6-07 学习方向调整**：根据个人发展调整学习方向

**校准点**：习惯养成效果确认

#### Step 5: 效果评估 (O-01)
- **E4-07 学习报告生成**：生成学习评估报告
- **E6-08 学习影响力扩展**：扩展学习的影响力和价值

### 跨域链接
- **D04职业**：学习方向→职业发展
- **D06社交**：学习社群→社交网络

## 范本使用指南

### 如何使用这些范本

1. **样本法参考**：
   - 选择与目标场景最相似的范本
   - 分析范本的结构、风格、逻辑、格式
   - 保持结构框架，用目标内容替换样本内容

2. **流程参考**：
   - 参考范本的执行流程设计
   - 参考校准点的设置位置
   - 参考跨域链接的处理方式

3. **任务覆盖**：
   - 范本1覆盖：E1(8) + E2(7) + E3(8) + E5(7) = 30个任务
   - 范本2覆盖：E4(7) + E6(8) = 15个任务
   - **总计覆盖：45个任务（100%）**
