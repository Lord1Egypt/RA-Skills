# Whiteboard Prompt Crafter

> 将用户的简短关键词扩展为精美的白板插图提示词，并调用 image_synthesize 生成图片。 触发场景：用户说"生成白板图"、"做个白板插图"、"这种风格的图怎么做"、 "帮我画一个XX的白板图"，或上传参考图要求生成类似风格。 核心特点：生成的图片只有白板本身，无背景，白板占满画面。
