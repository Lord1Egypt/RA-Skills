# 合同审查与风险标注助手

自动解析合同文本，提取关键条款并标注风险等级，为法务、采购、运营人员提供结构化的合同风险评估报告。

## Tools Required
- read
- write

## Usage
- "帮我审查这份采购合同，重点看付款周期和违约责任有没有问题"
- "这份服务协议的保密条款和争议解决方式我看不太懂，帮我分析一下风险"
- "把这份合同的关键条款都提取出来，标注一下哪些地方需要重点关注或修改"
- "我们要和供应商签框架协议，帮我做一个合同风险评估，列出高风险条款"
- "这份合同的违约金条款是否合理？有没有对我方不利的地方？"

## Examples
输入：上传一份IT系统采购合同，要求审查付款条款和违约责任
输出：
**合同审查报告 — IT系统采购合同**

**📋 关键条款提取**

| 条款类型 | 合同原文摘要 | 风险等级 |
|----------|-------------|----------|
| 付款周期 | 验收后30日内付清全款（第8.2条） | 🟡 中风险 |
| 违约责任 | 乙方延迟交付，每日按合同总价0.1%扣罚，上限5%（第12.1条） | 🟡 中风险 |
| 违约责任 | 甲方逾期付款，按同期银行贷款利率计息，无上限约定（第12.2条） | 🔴 高风险 |
| 保密条款 | 双方对项目信息保密，期限2年（第15条） | 🟢 低风险 |
| 争议解决 | 向乙方所在地法院提起诉讼（第18条） | 🔴 高风险 |

---

**🔴 高风险条款说明**

1. **第12.2条 — 逾期付款利息无上限**
   - 风险描述：甲方（我方）逾期付款利息无封顶，若遭遇资金周转问题，赔偿敞口较大。
   - 修改建议：建议补充"逾期付款违约金累计不超过合同总价的XX%"上限条款。

2. **第18条 — 争议解决地点对我方不利**
   - 风险描述：约定向乙方所在地法院诉讼，我方需异地应诉，增加维权成本。
   - 修改建议：改为"双方协商解决，协商不成提交甲方所在地仲裁委员会仲裁"。

---

**🟡 中风险条款说明**

1. **第8.2条 — 付款周期较短**
   - 风险描述：验收后30日付清全款，财务压力较大，建议争取分期或账期延长。
   - 修改建议：改为"验收合格后60日内，分两期付清尾款"。

---

**✅ 总体评估**：本合同存在2处高风险条款，建议在签署前与对方协商修改，或提交法务部门复核后再行签署。
