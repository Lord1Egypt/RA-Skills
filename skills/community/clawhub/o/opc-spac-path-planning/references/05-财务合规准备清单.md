## 二、T-3年（历史清理阶段）

### 2.1 核心任务

| 任务项 | 具体内容 | 优先级 |
|--------|----------|--------|
| 历史账务梳理 | 整理所有历史财务数据 | ⭐⭐⭐ |
| 凭证补全 | 补充缺失的原始凭证 | ⭐⭐⭐ |
| 银行对账 | 完成所有历史银行对账 | ⭐⭐⭐ |
| 往来清理 | 清理历史应收应付 | ⭐⭐ |
| 库存盘点 | 完成历史库存盘点 | ⭐⭐ |

### 2.2 详细清单

#### 2.2.1 会计核算规范

```
□ 统一会计科目体系
□ 统一会计期间划分
□ 统一收入确认政策
□ 统一成本归集方法
□ 统一费用分摊标准
```

#### 2.2.2 收入确认检查

```
□ 销售合同是否完整
□ 收入确认时点是否正确
□ 跨期收入是否正确分摊
□ 关联方交易定价是否合理
□ 折扣折让处理是否规范
```

#### 2.2.3 成本核算检查

```
□ 成本计算方法是否合理
□ 在产品/产成品成本分摊是否准确
□ 制造费用归集是否完整
□ 研发费用是否单独核算
□ 成本与收入是否配比
```

### 2.3 历史问题处理

| 问题类型 | 处理方法 | 时间要求 |
|----------|----------|----------|
| 凭证缺失 | 补充说明+承诺函 | T-3年完成 |
| 账实不符 | 查明原因+调账处理 | T-3年完成 |
| 往来长期挂账 | 催收或核销 | T-2年完成 |
| 税务瑕疵 | 补税+滞纳金 | T-2年完成 |

## 四、T-1年（规范运营阶段）

### 4.1 核心任务

| 任务项 | 具体内容 | 优先级 |
|--------|----------|--------|
| 规范运营 | 严格按照制度执行 | ⭐⭐⭐ |
| 报表优化 | 完善管理报表体系 | ⭐⭐ |
| 预算管控 | 建立全面预算管理 | ⭐⭐ |
| 税务规划 | 优化税务结构 | ⭐⭐ |

### 4.2 运营规范清单

#### 4.2.1 日常核算规范

```
□ 每笔业务及时入账
□ 原始凭证齐全规范
□ 日清月结
□ 银行对账每月完成
□ 库存盘点每月执行
```

#### 4.2.2 财务报表规范

```
□ 月报：每月结束后10日内完成
□ 季报：每季度结束后15日内完成
□ 年报：每年结束后60日内完成审计
□ 报表之间逻辑一致
□ 附注披露完整
```

### 4.3 研发费用规范（针对18C章）

| 要求项 | 具体标准 |
|--------|----------|
| **研发费用定义** | 符合会计准则的研发支出 |
| **费用比例** | 占营收≥15%（标准二要求30%） |
| **辅助账** | 建立研发费用辅助核算账 |
| **证据支持** | 项目立项、进度、成果文档 |

## 六、关键财务指标监控

### 6.1 通用指标

| 指标 | 目标值 | 监控频率 |
|------|--------|----------|
| **毛利率** | 行业平均水平+5% | 月度 |
| **净利率** | ≥10% | 季度 |
| **应收账款周转天数** | ≤90天 | 月度 |
| **存货周转天数** | ≤60天 | 月度 |
| **经营活动现金流** | 正值 | 月度 |

### 6.2 18C章专用指标

| 指标 | 目标值 | 监控频率 |
|------|--------|----------|
| **商业化营收** | ≥2.5亿港元 | 年度 |
| **研发投入占比** | ≥15%（标准二30%） | 年度 |
| **研发人员占比** | ≥30% | 半年度 |
| **核心技术人员稳定性** | ≥80% | 年度 |

### 6.3 SPAC专用指标

| 指标 | 关注点 |
|------|--------|
| **信托资金使用** | 确保资金用于并购 |
| **并购标的估值** | 合理估值，避免高估 |
| **股东回报预期** | 明确的退出路径 |

## 八、常见问题与应对

| 问题 | 应对措施 |
|------|----------|
| 历史凭证缺失 | 补充说明、承诺函、审计调整 |
| 收入确认不规范 | 重新梳理业务模式、调整确认政策 |
| 成本归集混乱 | 建立成本中心、完善归集逻辑 |
| 税务风险 | 主动与税务机关沟通、补税+滞纳金 |
| 内控缺陷 | 完善制度、强化执行、聘请内控顾问 |
