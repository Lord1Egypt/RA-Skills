# 拉贝洛尔立体异构体药理特性及临床应用

## 触发条件
IF 患者为中重度高血压、妊娠高血压、心绞痛或高血压危象患者，AND 无以下禁忌证：
- 哮喘
- 严重心功能不全
- 脑出血急性期
- 儿童静脉用药需求
THEN 可考虑拉贝洛尔口服或静脉给药。

## 执行步骤
1. **确认立体异构体组成与药理活性**：
   - 拉贝洛尔含4种立体异构体（R,R；R,S；S,R；S,S）。
   - R,R：主要阻断β受体，具内在拟交感活性，引起血管舒张。
   - S,R：α受体阻断作用最强，几乎无β阻断作用。
   - R,S 和 S,S：药理活性微弱或缺失。

2. **理解临床制剂特性**：
   - 临床使用消旋混合物，兼具α和β受体阻断作用。
   - β阻断强度约为普萘洛尔的1/2.5；α阻断强度为酚妥拉明的1/10～1/6。
   - β阻断作用强于α阻断作用5～10倍。

3. **评估血流动力学效应**：
   - 降低卧位血压和外周阻力。
   - 通常不减少心排出量。
   - 可能引起直立性低血压，需监测体位变化时的血压。
   - 因内在拟交感活性，可舒张血管并增加肾血流量。

4. **特殊人群用药决策**：
   - **妊娠高血压**：可安全用于孕妇，不影响胎儿发育。
   - **高血压危象**：可静脉注射，但禁用于儿童、孕妇及脑出血患者。

5. **规避配伍与禁忌**：
   - 静脉注射液不可与葡萄糖盐水混合。
   - 禁用于哮喘及严重心衰患者。

6. **监测与不良反应管理**：
   - 常见不良反应：眩晕、乏力、恶心。
   - 给药后密切观察是否出现体位性低血压。

## 输出建议
根据患者具体病情选择口服或静脉给药途径，确保无禁忌证，避免与葡萄糖盐水配伍，并持续监测血压变化。
