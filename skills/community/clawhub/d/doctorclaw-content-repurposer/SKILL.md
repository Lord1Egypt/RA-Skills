# Content Repurposer

Create once, publish everywhere. This skill takes one piece of long-form content — a blog post, YouTube script, podcast outline, or newsletter — and transforms it into multiple platform-ready pieces: social posts, email snippets, quote graphics, and threads.

Trigger it whenever you publish something new and want to squeeze maximum distribution from minimum effort.

## What You Get

- One input → 8-12 output pieces across multiple platforms
- LinkedIn post, X thread, Instagram caption, Facebook post
- Email newsletter snippet
- Key quote pull-outs for graphics
- Summary bullet points for repurposing elsewhere
- All matched to your brand voice

## Setup

### Required
- Nothing — just provide the content to repurpose (paste text, provide a URL, or point to a file)

### Optional (but recommended)
- **Brand voice doc** — your tone and style guide for consistent output
- **Platform accounts** — for direct posting after approval
- **Content calendar** — track what's been repurposed and scheduled

### Configuration

Tell your agent:

1. **Brand voice** — your tone (professional, casual, witty, educational)
2. **Target platforms** — which outputs to generate (default: all)
3. **Audience** — who you're speaking to (entrepreneurs, developers, marketers, etc.)
4. **Content type preference** — prioritize educational, storytelling, or promotional angles
5. **Hashtag strategy** — your standard hashtags or let the agent suggest

## How It Works

### Step 1: Ingest the Source Content
- Accept input as: pasted text, URL (blog post), file path, or dictated rough notes
- If URL: fetch the content and extract the main body
- Identify: main thesis, key points, supporting data, quotable lines, CTA

### Step 2: Extract Core Elements
Break the source into reusable building blocks:
- **Main thesis** — the one big idea (1 sentence)
- **Key points** — 3-5 supporting arguments or takeaways
- **Data/stats** — any numbers, percentages, or proof points
- **Quotable lines** — punchy phrases that stand alone
- **Story elements** — anecdotes, examples, case studies
- **CTA** — what action you want the audience to take

### Step 3: Generate Platform Outputs

**LinkedIn Post** (1 post, ~1300 chars)
- Hook: bold opening statement from the main thesis
- Body: 2-3 key points with personal angle
- Close: question or CTA for engagement
- 3-5 hashtags

**X/Twitter Thread** (5-8 tweets)
- Tweet 1: hook — the main thesis as a punchy statement
- Tweets 2-6: one key point per tweet, with data if available
- Final tweet: CTA + link to original content

**Instagram Caption** (1 post)
- Hook above the fold
- Storytelling format using key points
- CTA: save/share/comment
- Hashtag block (15-25)
- Carousel concept: 1 slide per key point

**Facebook Post** (1 post)
- Conversational, community tone
- Lead with a question or relatable observation
- Share 2-3 key insights
- End with discussion prompt

**Email Newsletter Snippet** (1 block, 150-200 words)
- Subject line options (3 variations)
- Preview text
- Short summary with link to full content
- One key takeaway highlighted

**Quote Graphics** (3-5 text blocks)
- Pull the most shareable, standalone quotes
- Format for easy copy into Canva or design tools
- Each under 25 words for readability

**Summary Bullets** (5-7 bullets)
- Concise takeaways for slides, presentations, or other repurposing

## Customization Ideas

- **Auto-repurpose on publish** — watch a blog RSS feed and auto-generate social content when new posts drop

- **Drip scheduling** — spread repurposed content across multiple days instead of posting all at once
- **Translation** — repurpose content into different languages for international audiences
- **Podcast show notes** — generate show notes, timestamps, and social clips from episode transcripts
