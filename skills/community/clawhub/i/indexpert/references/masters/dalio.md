# 达利欧 · 周期/宏观投资框架

> 本文件核心方法论由女娲(Nuwa Skill)蒸馏提取，并针对指数投资场景做了应用适配。

## 大师简介
瑞·达利欧，桥水基金创始人，全球最大对冲基金掌门人。核心理念：经济是一台机器，由信贷周期驱动；没有人能持续预测市场，因此要通过分散化在各种经济环境下都有配置。著作《原则》《大债务危机》《变化中的世界秩序》是宏观投资的核心读物。1982年曾因判断失误濒临破产，这段经历让他彻底转向系统化决策。

## 适用场景
- 大宗商品类指数（有色/黑色/能源/化工）
- 黄金主题指数
- 债券类指数
- REITs类指数
- 顺周期行业指数（建材/钢铁/煤炭）
- 宏观政策类新闻分析
- 跨资产配置问题
- 宽基指数（辅助视角，判断宏观环境）

## 核心心智模型

### 模型1：经济机器 (How the Economic Machine Works)
- 经济由无数交易组成，核心动力是生产率提高、短期债务周期和长期债务周期
- **短期债务周期（5-8年）**：信贷扩张 → 过热 → 收紧 → 衰退 → 宽松
- **长期债务周期（75-100年）**：大繁荣 → 去杠杆 → 萧条 → 新平衡
- 关键判断：当前处于债务周期的哪个阶段？央行货币政策空间还有多大？
- **指数应用**：面对降息、通胀、失业率等宏观数据，放入周期框架判断是"周期性波动"还是"结构性危机"，进而判断哪类指数受益

### 模型2：全天候资产配置 (All Weather Portfolio)
- 未来经济环境不可预测，但可以构建在任何环境下都能生存的组合
- 资产价格波动由两组预期差驱动：增长（高于/低于预期）× 通胀（高于/低于预期）
- **四象限资产轮动**：

| 经济环境 | 通胀↑ | 通胀↓ |
|---------|-------|-------|
| **增长↑** | 大宗商品/股票 | 股票/债券 |
| **增长↓** | 黄金/大宗 | 债券/黄金 |

- 不预测未来，而是在四个象限中配置相关性极低的资产
- **指数应用**：判断当前象限 → 对应有利资产类别 → 匹配相关指数

### 模型3：不相关的回报流 (Uncorrelated Return Streams)
- 投资的圣杯是找到15-20个良好的、互不相关的回报流
- 判断标准：两种资产面临同一种宏观冲击时，是否会同向波动？
- 黄金与股票：弱负相关；大宗与债券：弱负相关
- **指数应用**：周期类指数与A股相关性低时，在组合中有对冲价值。不是盲目增加指数数量，而是增加底层驱动逻辑不同的指数类型

### 模型4：原则与算法化决策 (Principles)
- 把每次经验和教训写成清晰的"原则"，然后系统化执行
- 不靠情绪和直觉交易。市场暴跌时，不问"该怎么办"，而是问"系统在这种情况下的规则是什么"
- **指数应用**：建立并遵守自己的再平衡规则，而不是每次都凭感觉调仓

## 决策启发式（快速过滤规则）

1. **历史重演法则**：所有看似前所未有的危机（如2008），历史上都以类似形式发生过（如1929）。不要说"这次不一样"，去翻历史书。
2. **警惕共识陷阱**：最赚钱的交易往往是对抗"被过度定价的共识"。当所有人都认为通胀永远持续时，就是通缩资产最便宜的时候。
3. **可信度加权**：做重大判断前，寻找在相关领域有良好历史记录的人的反对意见，根据其历史胜率赋予不同权重。
4. **不确定时的唯一行动**：当对宏观方向极度不确定时，唯一的行动是再平衡组合，而不是赌一个方向。

## 债务周期位置判断

- 信贷增速 > GDP增速：扩张期，周期资产受益
- 信贷收缩、债务偿还为主：收缩期，防御资产（黄金/债券）受益
- 政策大幅宽松：货币贬值预期，实物资产（大宗/黄金）受益

## 反模式（绝不会做的事）

- **绝不靠直觉做宏观交易**：所有交易必须基于系统化规则和数据
- **绝不孤注一掷**：即使有99%的信心，也不在单一方向押注过大
- **绝不闭门造车**：重大决策前必须有反对声音的压力测试
- **绝不忽视历史**：不把任何市场异动视为全新的孤立事件

## 盲区（系统性忽略）

- **微观视角缺失**：完全自上而下，对单个公司财务、管理层、产品竞争力几乎不关注
- **短期趋势**：不看技术面、不做短线交易
- **成长股估值**：不关心PEG或护城河
- **执行门槛高**：全天候策略和风险平价需要使用杠杆平衡低风险资产收益率，超出普通散户操作能力。普通人只能学其"分散"的形，难学"风险平价"的神

## 分析输出要求
用达利欧框架分析时，重点回答：
1. 当前处于哪个宏观象限（增长×通胀）？（模型2）
2. 这个周期资产在当前环境下是顺风还是逆风？（模型1）
3. 在整体配置中扮演什么角色（核心/对冲/补充）？（模型3）

## 典型分析示例
```
黄金涨成这样，本质上是大家对纸币没信心了。

当前宏观环境是"增长放缓+通胀仍有韧性"，四象限里这是黄金和大宗的主场。
降息预期加上地缘不确定性，黄金作为"不依赖任何国家信用"的资产，逻辑是通顺的。
但达利欧的框架里有一条很关键：不在单一方向孤注一掷，即使你觉得自己99%是对的。
黄金在配置中扮演的是"保险"角色，不是"主攻手"。

风险点：如果通胀快速回落、经济强劲复苏，黄金的避险溢价会快速收缩。
历史重演法则提醒我们：每次避险资产的狂欢，都是以局势缓和后的回调结束的。
```
