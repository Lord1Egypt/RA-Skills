# 动态心电图导联系统选择

## 适用场景
- 需进行动态心电图（Holter）监测
- 已明确临床检测目的，例如：
  - 检出缺血型ST段下移
  - 分析P波形态以诊断心律失常
  - 评估左心室下壁心肌缺血
  - 筛查变异性心绞痛（冠状动脉痉挛）

## 执行步骤
1. **根据检测目的选择导联**：
   - 若目标为检出缺血型ST段下移且需高QRS振幅 → 选用 **CM₅导联**（正极置于左腋前线平第5肋间/V₅位置，负极置于右锁骨下窝中1/3处）
   - 若需清晰显示P波以分析心律失常 → 选用 **CM₁导联**（正极置于胸骨右缘第4肋间/V₁位置或胸骨上，负极置于左锁骨下窝中1/3处）
   - 若怀疑左心室下壁心肌缺血 → 选用 **MₐᵥF导联**（正极置于左腋前线肋缘，负极置于左锁骨下窝内1/3处）
   - 若怀疑变异性心绞痛 → 联合选用 **CM₃导联**（正极置于V₃位置，负极置于右锁骨下窝中1/3处）和 **MₐᵥF导联**

2. **无关电极放置**：可置于右胸第5肋间腋前线或胸骨下段中部。

3. **12导联动态心电图系统**：电极放置部位与运动负荷试验相同，遵循标准12导联位置。

4. **注意事项**：
   - 常规Holter多采用双极导联；近年贴片式系统多为单导联或3导联，其适用性受限于检测目标。
   - 单导联系统无法替代12导联系统用于复杂诊断。
   - 本规则不适用于需全面评估心脏解剖结构或精确定位心肌梗死的情况。

## 输出要求
输出应为具体导联配置方案，包括所选导联名称、正负极位置及对应的适用临床场景说明。
