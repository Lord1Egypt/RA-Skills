## Description: <br>
YOAP (Yongnian Open Agent Protocol) is an open A2A protocol for smart matching, end-to-end encrypted messaging, negotiation threads, group channels, and webhook-based relay communication. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agents use this skill to interact with the YOAP people-matching relay for registration, discovery, messaging, encrypted communication, negotiation workflows, channels, and webhook handling. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill guides agents to use yoap.io as a third-party people-matching relay, which can expose personal profile, messaging, webhook, and matching data. <br>
Mitigation: Use minimal or pseudonymous profile data, review visibility settings before registration, treat access tokens like passwords, prefer encrypted messages for sensitive content, and require explicit confirmation before posting seeks, creating channels or threads, sending messages, or registering a webhook. <br>
