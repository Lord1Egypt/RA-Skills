## Known Risks and Mitigations: <br>
Risk: The installer can make system package changes, including changes that may require sudo. <br>
Mitigation: Review scripts/install.sh before execution and run it only on machines where system package changes are acceptable. <br>
Risk: The setup path may run a remote uv installer. <br>
Mitigation: Prefer installing uv through a trusted package manager or a verified manual download instead of piping network content directly into a shell. <br>
Risk: Python dependency installation can affect the active environment. <br>
Mitigation: Use a virtual environment for Python dependencies when practical. <br>
