## Description: <br>
Chat UI provides React and Next.js building blocks for chat containers, messages, inputs, typing indicators, avatars, streaming chat interfaces, messaging interfaces, and AI assistants. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[okaris](https://clawhub.ai/user/okaris) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers use this skill to add chat UI components and example interaction patterns to React or Next.js applications, including custom messaging interfaces and AI assistant frontends. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill asks users to run third-party npx installation commands from a remote component registry. <br>
Mitigation: Review the registry output before running the commands and pin package versions where practical. <br>
Risk: The provided React examples may need adaptation before production use. <br>
Mitigation: Review generated UI code in the target project and test message handling, disabled states, and loading behavior before deployment. <br>
