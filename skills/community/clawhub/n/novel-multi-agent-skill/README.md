# 📚 多智能体小说创作系统 v1.0

一套经过生产验证的自动化小说创作系统，由5个专业智能体协同工作。

---

## ✨ 核心特性

- 🚀 **高效**：日产能 20-30 章（4-6 万字）
- 🎯 **稳定**：质量评分稳定 90+ 分
- 🔄 **闭环**：创作 → 评审 → 优化 完整闭环
- 📊 **可控**：每一步都有明确的验收标准
- 🧩 **复用**：设定一次完成，永久复用

## 🚀 快速开始

### 1. 安装到 OpenClaw

```bash
# 复制到技能目录
cp -r novel-multi-agent-skill ~/.openclaw/skills/
```

### 2. 初始化项目

```bash
cd ~/.openclaw/skills/novel-multi-agent-skill/workflow
./init-project.sh ../my-first-novel "深夜便利店"
```

### 3. 运行完整流水线

```bash
./run-full-pipeline.sh ../my-first-novel 1-3
```

或者分步执行：
```bash
./step1-world.sh ../my-first-novel       # 世界观
./step2-character.sh ../my-first-novel   # 角色
./step3-plot.sh ../my-first-novel 1-10   # 大纲
./step4-write.sh ../my-first-novel 1-10  # 创作
./step5-review.sh ../my-first-novel 1-10 # 评审
```

详细教程请参考 [QUICKSTART.md](QUICKSTART.md)

## 🤖 智能体配置

要使用完整的自动化功能，需要配置以下5个智能体：

| 智能体ID | 角色 | 职责 |
|----------|------|------|
| `novel-world-builder` | 世界观构建师 | 设计完整世界观 |
| `novel-character-designer` | 角色设计师 | 设计立体人物 |
| `novel-plot-planner` | 情节策划师 | 设计分章大纲 |
| `novel-chapter-writer` | 章节写作智能体 | 高质量正文创作 |
| `novel-quality-reviewer` | 质量评审官 | 五维评分、修改建议 |

### 配置方法

在 OpenClaw 中创建对应的代理，配置相应的系统提示词。
系统提示词可以参考各智能体的职责描述，结合 rules/ 目录下的规则。

## ❓ 常见问题

### Q: 为什么不一次性写完整部小说？
A: 长篇小说一次性写容易出现人物OOC、逻辑矛盾、节奏失控等问题。
分批创作（每批10章）可以及时调整，保证整体质量。

### Q: 为什么必须先评审再修改？
A: 创作者容易有盲区，第三方评审更客观。而且评审意见可以帮助智能体持续改进。

### Q: 智能体之间为什么不直接通信？
A: 这是 OpenClaw 的安全机制，保证用户对所有智能体活动有完全的知情权和控制权。
