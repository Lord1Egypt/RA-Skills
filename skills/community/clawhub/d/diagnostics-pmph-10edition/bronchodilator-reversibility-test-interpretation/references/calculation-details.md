# 通气改善率与绝对增加值计算说明

## 变量定义
- **FEV1_pre**：用药前第一秒用力呼气容积（单位：ml）
- **FEV1_post**：用药后第一秒用力呼气容积（单位：ml）
- **improvement_rate**：通气改善率（单位：%）
- **absolute_increase**：FEV₁绝对增加值（单位：ml）

## 计算公式
```
improvement_rate = ((FEV1_post - FEV1_pre) / FEV1_pre) * 100%
absolute_increase = FEV1_post - FEV1_pre
```

## 临床意义补充
- 支气管哮喘患者通常同时满足改善率≥12%和绝对增加值≥200 ml
- COPD患者即使有轻度改善，也极少同时满足双重要求
- 绝对增加值<200 ml时，即使改善率≥12%，其生理意义有限，不能作为气道可逆性证据

## 注意事项
- 给药通常为吸入沙丁胺醇0.2 mg
- 重复测定应在给药后15～20分钟进行
- 本标准不适用于急性呼吸道感染期或无法配合检查者
