# 鉴定意见草拟Prompt模板

## 基础模板

```
你是一名工程造价司法鉴定专家，基于「AI工程造价+法律知识库」草拟鉴定意见书。

**委托事项**：[法院/仲裁委委托的具体事项]
**争议焦点**：[双方的核心争议点]
**送鉴材料**：[送交鉴定的全部材料]
**适用标准**：[合同约定标准/国家标准/行业标准]

请按知识库中《鉴定意见撰写规范》草拟意见书。

## 鉴定意见书结构

### 一、基本情况
- 委托方：[委托鉴定机构]
- 委托日期：[委托日期]
- 鉴定事项：[鉴定范围]
- 双方当事人：[发包方/承包方]
- 引用知识库中类似委托案例的处理方式

### 二、鉴定依据
请从知识库中检索并列出：
- 合同依据：[具体合同条款及解释]
- 计价依据：[适用的计价标准和定额]
- 法律依据：[相关法律法规和司法解释]
- 案例依据：[最高院或同类案例]

### 三、鉴定方法
- 采用的鉴定方法及理由
- 方法选择的法律依据
- 替代方案考虑
- 引用知识库中的权威文件

### 四、争议焦点分析
对每个争议焦点：
1. **焦点描述**：[双方的观点]
2. **证据审查**：[审查相关证据]
3. **事实认定**：[基于证据的事实认定]
4. **法律适用**：[适用的法律依据]
5. **初步意见**：[鉴定人的初步判断]

### 五、鉴定结论
- 结论一：[明确的鉴定结论]
- 结论二：[如有多个结论]
- 确保每个结论都有充分的知识库依据

### 六、特别说明
- 鉴定限制条件
- 需要进一步说明的事项
- 不确定性的声明
```

## 简短版模板

```
请草拟以下鉴定事项的鉴定意见摘要：

委托事项：[具体事项]
争议焦点：[核心争议]
关键证据：[关键证据]

只需提供：
1. 核心结论（1-2句）
2. 支持结论的关键依据（列表形式）
3. 需要注意的风险点
```

## 鉴定陈述意见模板

```
请准备一份鉴定陈述意见：

鉴定结论：[已出的鉴定结论]
对方异议点：[对方提出的异议]

请准备：
1. 鉴定方法和依据的说明
2. 对对方异议的回应
3. 鉴定结论的再确认
4. 法律依据的强化
```
