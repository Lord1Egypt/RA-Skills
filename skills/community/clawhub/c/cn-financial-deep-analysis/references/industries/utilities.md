# 公用事业

## 核心财务特征

- 重资产、高CAPEX、长回收期
- 现金流稳定（特许经营、刚需）
- 利润受政府定价/指导价约束
- 高分红（成熟期现金流充裕）
- 增长依赖于产能扩张或并购

## 关键分析指标

| 指标 | 健康值 | 警戒值 | 说明 |
|------|--------|--------|------|
| 资产负债率 | <65% | >80% | 重资产行业杠杆较高 |
| 经营现金流/净利润 | >1.5 | <1 | 折旧大，现金流应好于利润 |
| 自由现金流 | 正值 | 连续3年为负 | CAPEX回收能力 |
| ROE | 8-12% | <5% | 受定价约束，ROE不高 |
| 分红率 | >30% | 0 | 成熟公司应有稳定分红 |
| 单位成本 | 行业中位 | 高于行业20% | 成本控制即竞争力 |

## 风险评估重点

- 价格管制：调价滞后于成本上涨
- 环保政策：排放标准提高增加成本
- 产能过剩：利用率 < 60%
- 债务风险：短债长投期限错配
- 补贴退坡：新能源补贴、税收优惠到期

## 舞弊高发区

- 在建工程推迟转固（少提折旧虚增利润）
- 政府补贴确认提前
- 资产减值不足（机组、管网）
- 关联交易转移定价
