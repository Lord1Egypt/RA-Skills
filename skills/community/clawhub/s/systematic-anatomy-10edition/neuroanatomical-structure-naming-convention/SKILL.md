# 神经系统结构层级命名规范

## 何时使用
- 命名新发现的神经结构
- 区分“核”与“节”、“束”与“神经”
- 验证术语是否符合解剖学标准
- 编写神经科学文献或教学材料

## 执行步骤
1. **确定结构层级**：
   - **细胞级**：神经元、神经胶质细胞、少突胶质细胞
   - **微观集群**：
     - 中枢 → **神经核**（如面神经核）
     - 周围 → **神经节**（如三叉神经节）
   - **纤维集合**：
     - 中枢 → **神经束**（如皮质脊髓束）
     - 周围 → **神经**（如桡神经）
2. **判断位置属性**：
   - **中枢神经系统**（脑与脊髓内）：使用“核”、“束”、“区”
   - **周围神经系统**（脑神经、脊神经及其分支）：使用“节”、“神经”
3. **应用命名规则**：
   - 功能相似但位置不同 → 术语不同（如“孤束核” vs “蜗神经节”）
   - 避免混用“tract”与“nerve”、“nucleus”与“ganglion”
4. **验证示例**：
   - “迷走神经背核” → 中枢神经核（正确）
   - “坐骨神经” → 周围神经（正确）
   - “脊髓小脑束” → 中枢传导束（正确）

> 注意：仅适用于正常神经组织，不适用于肿瘤、瘢痕等病理增生结构。
