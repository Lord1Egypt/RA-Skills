🍃 Import to Overleaf

Follow these steps to compile your cheatsheet in Overleaf

  1. Open Overleaf and click New Project → Blank Project. Give it any name.
  2. In the new project, click on main.tex in the file tree on the left. Select all existing content (Ctrl+A) and delete it.
  3. Click the Copy button in this editor's toolbar (or open the downloaded .tex file and copy its contents). Then paste into the empty main.tex on Overleaf.
  4. Click the Menu button (top-left corner in Overleaf). Under Settings, change the Compiler from pdfLaTeX to XeLaTeX.
  5. Click Recompile (or Ctrl+Enter). Your cheatsheet PDF will render on the right!
⚠ Important: You must switch the compiler to XeLaTeX (step 4). The default pdfLaTeX does not support the fontspec package used for custom fonts. If you see a compilation error about fontspec, this is the fix.
