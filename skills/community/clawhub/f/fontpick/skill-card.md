## Use Case: <br>
Designers and developers use this skill for font pairing and typography support, including font stack, scale, and contrast guidance. Review the shipped script behavior before use because the security evidence notes that it mostly stores arbitrary local input rather than clearly matching the advertised font-pairing purpose. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The packaged script may retain user-entered text in local history files and later display, search, or export it. <br>
Mitigation: Avoid entering sensitive client, project, or personal information, and review or clear the local data directory before sharing the machine or exporting data. <br>
Risk: The shipped Bash behavior does not clearly match the advertised font-pairing purpose. <br>
Mitigation: Review the script and run it in a limited test environment before installing it into a normal agent workflow. <br>
