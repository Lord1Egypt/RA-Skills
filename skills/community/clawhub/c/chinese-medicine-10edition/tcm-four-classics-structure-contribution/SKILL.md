# TCM Four Classics Structure and Contribution

## When to Use
- 需要追溯中医学理论体系的文献源头
- 在中医教育中讲解经典著作的地位与作用
- 分析临床辨证论治的历史依据
- 理解中药学基本理论的形成背景

## Execution Steps

1. **确认四大经典著作及其基本属性**：
   - 《黄帝内经》：托名黄帝与岐伯对话，含《素问》《灵枢》各81篇，系统构建生理、病理、诊断、治疗、预防、阴阳五行、精气血津液、病因病机、养生及医德等理论框架。
   - 《黄帝八十一难经》（简称《难经》）：传为扁鹊所作，以问答形式阐释81个医学难题，首创“独取寸口”脉诊法，完善三部九候诊脉体系。
   - 《伤寒杂病论》：东汉张仲景著，后分为《伤寒论》（外感病六经辨证）与《金匮要略》（杂病脏腑辨证），确立“理法方药”一体化的辨证论治范式。
   - 《神农本草经》：托名神农，收载365种药物，按上、中、下三品分类，提出君臣佐使、七情配伍、性味归经、炮制及用药宜忌等中药学基本理论。

2. **提炼各经典的核心贡献**：
   - 《内经》奠定整体理论框架；
   - 《难经》深化脉学与经络理论；
   - 《伤寒杂病论》建立临床辨证论治范式；
   - 《神农本草经》创立中药学理论体系。

3. **验证历史延续性**：确认四大经典内容经后世反复临床实践验证，至今仍为中医教学与临床的重要依据。

> 注意：本技能不适用于现代西医诊断体系，亦不能替代具体临床操作指南。
