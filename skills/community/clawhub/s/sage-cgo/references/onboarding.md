# CGO Onboarding

当 `~/.sage/growth/` 不存在或增长信息为空时，执行本流程。

## 第一轮：增长基本面

选择 3-5 个问题：

1. 当前最想增长的是什么：粉丝、线索、营收、品牌认知还是私域？
2. 现在主要在哪些平台有账号？数据大概如何？
3. 目标受众是谁？他们平时在哪些平台消费内容？
4. 当前产品或服务怎么变现？
5. 你最担心的增长问题是什么？

写入：`growth/channels.md`、`growth/audience.md`、`growth/metrics.md`、`growth/monetization.md`。

## 第二轮：内容和实验

选择 3-5 个问题：

1. 过去表现最好的一条内容是什么？为什么好？
2. 你们能稳定产出的内容形态是什么？
3. 有没有 3-5 个长期内容支柱？
4. 最近想验证的增长假设是什么？
5. 私信、咨询、成交或私域承接现在怎么做？

写入：`growth/content-pillars.md`、`growth/experiments.md`、`growth/metrics.md`。

## 初始诊断

第一轮后必须输出：

1. 当前增长阶段。
2. 最大增长瓶颈。
3. 推荐主战场。
4. 30 天最小增长实验。

第二轮后必须输出“最小增长操作系统”：

1. 北极星指标候选。
2. 当前 AARRR 最薄弱环节。
3. ICP 初稿。
4. 第一个增长循环假设。
5. 接下来两周的 ICE 实验清单。
