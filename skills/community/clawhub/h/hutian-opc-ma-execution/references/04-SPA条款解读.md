## 核心条款深度解读

### 第一章：定义与解释

#### 关键定义清单

| 术语 | 定义要点 | OPC关注 |
|------|---------|--------|
| 交割日 | 实际完成交割的日期 | 确认各项条件满足 |
| 交易对价 | 股权转让的总价 | 与估值计算方式关联 |
| 先决条件 | 交割必须满足的条件 | 影响交割时间 |
| 重大不利变化 | 影响交易的重大变化 | 触发重新谈判或终止 |
| 披露函 | 卖方对陈述保证的例外 | 豁免部分保证责任 |

#### 常见争议定义

```
【争议点1：重大不利变化的界定】

标准表述：
"重大不利变化"指目标公司发生的、单独或合计对目标公司的业务、财务状况、经营成果产生重大不利影响的事件或情况。

OPC风险点：
❌ 买方可能扩大解释，将正常经营波动纳入
❌ 行业整体下行可能被认定为"重大不利"

OPC谈判建议：
□ 明确排除的情形（如行业周期性波动、政策调整）
□ 设置"量化门槛"（如影响金额>总资产的X%）
□ 设置豁免情形（不可抗力、市场因素）
```

### 第三章：转让价格及支付

#### 定价机制

| 定价方法 | 计算公式 | 适用场景 |
|---------|---------|---------|
| 固定价格 | 一口价 | 成熟稳定企业 |
| 可调价格 | 基础价±调整项 | 估值存在不确定性 |
| 分期支付 | 首期+里程碑+尾款 | 存在对赌安排 |

#### 价格调整机制

```
【调整情形】

1. 净资产调整
   - 若交割日净资产低于[X]万元，每低1万元，价格下调[X]元

2. 债务调整
   - 若存在未披露负债，每出现1万元，价格下调1万元

3. 营运资金调整
   - 若营运资金低于约定下限，每低1万元，价格下调[X]元

【调整上限】
各方同意，本条项下的价格调整总额不超过交易对价的[X]%。
```

#### OPC谈判要点

```
【卖方（技术方）保护条款】
□ 争取设置调整上限（建议≤20%）
□ 明确"未披露负债"的定义，避免扩大解释
□ 争取"善意超调"的豁免（如审计调整）
□ 设置争议解决机制（第三方审计）

【买方保护条款】
□ 保留价格下调权利直至实际赔付
□ 设置"穿透核查"权
□ 要求卖方提供履约担保
```

### 第四章：交割先决条件

#### 先决条件清单

```
【第一类：审批条件】

□ 买方内部审批
  ├── 董事会批准
  └── 股东会批准（如需要）

□ 卖方内部审批
  ├── 董事会批准
  └── 股东会批准（如需要）

□ 第三方同意
  ├── 优先购买权放弃声明
  └── 合同对方同意（如合同约定）

□ 监管审批
  ├── 证监会审批（如上市公司）
  ├── 商务部反垄断审查（如需要）
  ├── 外商投资安全审查（如需要）
  └── 其他适用监管审批

【第二类：陈述保证合规】

□ 卖方陈述保证在重大方面真实、准确
□ 卖方未违反任何重大承诺
□ 卖方未收到任何重大不利通知

【第三类：事项条件】

□ 无重大不利变化
□ 无重大诉讼或行政处罚
□ 核心人员仍在公司任职
□ 核心资质证照持续有效
□ 知识产权权属清晰

【第四类：文件交付】

□ 完成工商变更所需全部文件
□ 目标公司印章（公章、财务章、合同章）
□ 银行U盾/密钥
□ 全部合同原件
□ 财务账册及凭证
□ 知识产权证书原件
```

#### 先决条件的满足与豁免

```
【条件满足确认程序】

1. 通知义务
   - 一旦任何条件被满足，相应方应在5个工作日内书面通知另一方

2. 确认程序
   - 另一方在收到通知后10个工作日内书面确认或提出异议
   - 逾期未提出异议视为确认

3. 条件豁免
   - 经双方书面同意，可豁免特定先决条件
   - 买方可单方面豁免卖方先决条件（除监管审批外）

【OPC注意】
□ 监管审批通常不可豁免
□ 卖方应确保自身条件可控制
□ 建议设置"日落条款"：超期未满足则协议终止
```

### 第五章：陈述与保证

#### 卖方标准陈述保证清单

```
【第一类：主体资格】
1. 公司的合法设立和存续
2. 股权权属清晰，无代持、质押、冻结
3. 股东人数和股权比例符合法律规定
4. 章程合法有效

【第二类：财务】
5. 历史财务报表真实、准确
6. 无重大未披露负债
7. 税务申报真实、合规
8. 无偷逃税行为

【第三类：业务】
9. 业务经营合法合规
10. 主要合同有效且正常履行
11. 无重大违约情形
12. 客户关系正常

【第四类：资产】
13. 资产权属清晰
14. 主要资产无抵押、质押
15. 知识产权权属完整
16. 无第三方主张

【第五类：劳动人事】
17. 劳动合同签订完整
18. 社保公积金缴纳合规
19. 无重大劳动争议
20. 核心人员无异常

【第六类：法律合规】
21. 无重大诉讼或行政处罚
22. 行业资质证照齐全
23. 环保、安全生产合规
24. 反商业贿赂合规
```

#### 违反陈述保证的救济

```
【救济机制】

1. 通知与修复
   - 买方发现违反情形后[X]日内通知卖方
   - 卖方有权在[X]日内进行修复

2. 赔偿触发
   - 未能修复的，触发赔偿责任
   - 赔偿金额 = 实际损失

3. 赔偿上限
   - 常规违反：不超过交易对价的[X]%
   - 欺诈/故意：不受上限限制
   - 税务风险：可单独设置上限

4. 赔偿期限
   - 常规事项：交割后[X]年
   - 税务事项：交割后[X]年
   - 知识产权：交割后[X]年
   - 欺诈/故意：无期限限制
```

#### OPC谈判要点

```
【卖方（技术方）保护建议】

1. 争取"知情豁免"
   - 已在披露函中列明的事项，应豁免赔偿责任

2. 限制"重大"门槛
   - 明确"重大"影响的量化标准（如>50万元）

3. 篮子机制
   - 设置"篮子门槛"（如累计<10万元不触发赔偿）
   - 避免小额频繁索赔

4. 责任分担
   - 按比例分担（买方也承担部分风险）
   - 或设置买方自负额

5. 保障期限
   - 争取较短的赔偿期限（建议2年）
   - 欺诈除外条款设置期限

【常见陷阱】

❌ 陈述保证过于宽泛
   - 建议：逐条明确，避免"所有重大方面"的泛泛表述

❌ 赔偿无上限
   - 建议：设置合理的赔偿上限（通常20-50%）

❌ 期限过长
   - 建议：常规事项不超过3年

❌ 无篮子机制
   - 建议：设置累计门槛和单次门槛
```

### 第六章：交割

#### 交割程序

```
【交割前准备】

1. 交割前[X]日，卖方应准备好：
   ├── 工商变更申请材料
   ├── 目标公司印章
   ├── 银行U盾及账户信息
   ├── 全部合同及重要文件原件
   └── 其他交割文件

2. 买方应准备好：
   ├── 支付凭证
   ├── 内部审批文件
   └── 工商变更配合材料

【交割日流程】

1. 上午
   ├── 款项支付至共管账户/指定账户
   ├── 卖方交付印章及密钥
   └── 双方签署交割确认书

2. 下午
   ├── 提交工商变更材料
   └── 完成银行账户变更

【交割完成确认】

双方签署《交割确认书》，确认：
□ 所有先决条件已满足
□ 款项已支付
□ 文件已交付
□ 交割已完成
```

### 第七章：过渡期安排

#### 过渡期定义

```
"过渡期"指本协议签署日至交割日的期间。

标准表述：
"过渡期"自本协议签署之日起至交割完成之日止。
```

#### 卖方承诺

```
【禁止事项】

在过渡期内，除非获得买方书面同意，卖方应确保目标公司：

1. 经营层面
   ├── 不进行重大资产处置（>100万元）
   ├── 不进行重大投资（>100万元）
   ├── 不进行重大关联交易
   └── 不进行业务模式重大变更

2. 股权层面
   ├── 不进行股权变动
   ├── 不进行分红
   └── 不进行股权激励新增

3. 人员层面
   ├── 不解除核心人员劳动合同
   ├── 不新增重大雇佣安排
   └── 不调整核心人员薪酬

4. 债务层面
   ├── 不新增重大借款
   ├── 不提供对外担保
   └── 不进行债务重组
```

### 第八章：锁定期与竞业禁止

#### 锁定期条款

```
【锁定期定义】
"锁定期"指交割日起[X]年内。

【锁定期义务】

1. 股权锁定
   - 卖方不得转让其持有的剩余股权
   - 经买方同意除外

2. 股权质押限制
   - 未经买方同意，卖方不得将其持有的股权质押

3. 例外情形
   ├── 继承
   ├── 司法强制执行
   └── 买方书面同意
```

#### 竞业禁止条款

```
【竞业禁止义务】

1. 期限
   - 交割日起[X]年

2. 地域
   - [具体地域范围，需合理]

3. 范围
   - 与目标公司存在竞争关系的业务
   - 担任竞争对手的董事、监事、高管
   - 从事与目标公司相同的经营活动

4. 补偿
   - 竞业禁止期间，买方应支付经济补偿
   - 标准：不低于离职前12个月平均工资的[X]%

【OPC谈判要点】

□ 争取"关联公司豁免"（仅限本人，不含家属）
□ 争取合理的地域范围（与实际业务范围匹配）
□ 争取合理的期限（建议≤2年）
□ 争取合理的补偿标准
□ 明确竞业范围定义（避免扩大）
```

### 第九章：业绩对赌

#### 业绩承诺设计

```
【承诺指标】

| 年度 | 承诺净利润 | 承诺营业收入 | 备注 |
|------|-----------|-------------|------|
| 第1年 | [X]万元 | [X]万元 | - |
| 第2年 | [X]万元 | [X]万元 | - |
| 第3年 | [X]万元 | [X]万元 | - |

【承诺指标选择】

建议选择：
□ 单一净利润指标（最常用）
□ 营业收入指标（早期企业）
□ 复合指标（净利润+收入+特定事件）

不建议：
❌ 过多数量的指标
❌ 无法核实的指标
❌ 与业务无关的指标
```

#### 补偿计算方式

```
【方式一：现金补偿】

补偿金额 = (承诺利润 - 实际利润) × 补偿系数

示例：
承诺利润 = 1000万元
实际利润 = 800万元
差额 = 200万元
补偿系数 = 1
补偿金额 = 200万元

【方式二：股权补偿】

补偿股数 = (承诺利润 - 实际利润) / 每股净资产

示例：
承诺利润 = 1000万元
实际利润 = 800万元
差额 = 200万元
每股净资产 = 10元
补偿股数 = 20万股

【方式三：双向对赌】

├── 业绩未达标：卖方向买方补偿
└── 业绩超标：买方向卖方奖励
```

#### OPC谈判要点

```
【卖方（技术方）保护建议】

1. 指标设定
   □ 设置合理的增长率（避免过高）
   □ 争取分段对赌（首年低，后续提高）
   □ 设置豁免情形（不可抗力、政策变化）

2. 期限设置
   □ 建议2-3年，避免过长
   □ 跨周期对赌需谨慎

3. 调整机制
   □ 争取"善意调整"（非卖方原因导致）
   □ 设置上限（补偿金额上限）

4. 计算方式
   □ 争取"分段补偿"（避免一次性大额补偿）
   □ 明确审计机构

【常见陷阱】

❌ 指标过高无法完成
❌ 补偿系数过大
❌ 无豁免情形
❌ 计算方式模糊
❌ 无超额奖励机制
```

### 第十章：违约责任

#### 违约情形

```
【卖方违约情形】
1. 违反陈述保证
2. 未能按期完成交割
3. 违反过渡期承诺
4. 违反竞业禁止
5. 泄露保密信息

【买方违约情形】
1. 未能按期支付款项
2. 违反保密义务
3. 违反协议其他承诺
```

#### 违约金条款

```
【标准违约金】

1. 延迟交割
   - 每延迟一日，违约金 = 未交割部分对应交易对价的[X]%
   - 上限 = 交易对价的[X]%

2. 延迟付款
   - 每延迟一日，违约金 = 未付款项的[X]%
   - 上限 = 未付款项的[X]%

3. 违反竞业禁止
   - 每次违反 = [X]万元
   - 或年度竞业补偿费的[X]倍
```

#### 赔偿责任限制

```
【赔偿上限】

| 违约类型 | 赔偿上限 |
|---------|---------|
| 违反陈述保证 | 交易对价的[X]% |
| 违反竞业禁止 | [X]万元 |
| 泄露保密信息 | [X]万元 |
| 欺诈/故意 | 无上限 |

【OPC注意】
□ 争取设置赔偿上限
□ 明确"间接损失"是否纳入赔偿
□ 设置免赔额/免赔率
```
