# 宏观经济与事件驱动分析框架

本文件定义金融分析中宏观经济和事件驱动分析的方法论。分析个股时**必须**将公司层面数据与宏观背景、国际局势相结合。

---

## 1. 宏观经济分析框架

### 1.1 经济周期阶段识别

通过多指标综合判断当前经济所处阶段：

| 指标 | 复苏期 | 扩张期 | 过热/滞胀期 | 收缩期 |
|------|--------|--------|------------|--------|
| PMI 制造业 | 回升至50以上 | 持续>50 | 高位回落 | <50 |
| CPI | 低位温和 | 温和上升(2-3%) | 快速上升(>3%) | 回落 |
| PPI | 触底回升 | 持续上涨 | 高位 | 大幅下跌 |
| GDP 同比 | 回升 | 高位增长 | 增速放缓 | 低位 |
| 社融增速 | 加速扩张 | 稳定增长 | 开始收缩 | 低位 |
| M2增速 | 上升 | 高位 | 开始回落 | 低位或回升 |
| LPR利率 | 下调/宽松 | 稳定 | 上调/收紧 | 下调/宽松 |
| 10Y国债收益率 | 低位回升 | 持续上行 | 高位或下行 | 快速下行 |
| 10Y-2Y期限利差 | 扩大（陡峭化） | 稳定正值 | 收窄或倒挂 | 扩大（宽松预期） |

**过热/滞胀期判定条件**：
- PMI > 50 + CPI > 3% + PMI环比下降 → 过热/滞胀信号
- 特征：经济仍在扩张区间但通胀压力明显上升，政策面临"控通胀 vs 保增长"两难
- 政策方向：偏紧（控通胀为主），收紧流动性

### 1.2 周期-行业轮动图谱（A股适用）

| 周期阶段 | 超配行业 | 标配行业 | 低配行业 |
|---------|---------|---------|---------|
| 复苏期 | 科技、消费、金融、地产 | 材料、通信 | 公用事业、防御 |
| 扩张期 | 制造业、周期、金融、材料 | 消费、通信 | 防御、公用事业 |
| 过热期 | 能源、材料、医药、必选消费 | 公用事业、金融 | 科技、可选消费 |
| 收缩期 | 公用事业、医药、必选消费、高股息 | 通信、地产 | 周期、金融、材料 |

**过热/滞胀期因子轮动**：
- **超配因子**：高股息、低波动、价值因子 — 防御属性强，抗通胀能力佳
- **标配因子**：质量因子（高ROE+低杠杆）— 盈利确定性高
- **低配因子**：高Beta、小盘成长、高估值 — 流动性收紧时最先受损
- **行业逻辑**：能源/材料直接受益于商品涨价；医药/必选消费需求刚性、定价权强；科技/可选消费受高利率和需求回落双重打压

### 1.3 关键宏观指标解读

#### 利率环境

| 利率环境 | 利好行业 | 利空行业 | 投资策略 |
|---------|---------|---------|---------|
| 降息周期 | 地产、科技(长久期)、成长股 | 银行(息差收窄) | 增配成长、减配价值 |
| 加息周期 | 银行、保险、价值股 | 地产、高杠杆公司 | 增配价值、减配成长 |
| 利率触顶 | 防御转进攻过渡期 | — | 逐步增加风险敞口 |
| 收益率曲线倒挂 | 必选消费、医药、公用事业 | 周期股、金融 | 防守为主 |

**Shibor 解读**：
- Shibor隔夜（O/N）反映银行间极短期流动性状态
- Shibor 突然飙升 → 流动性紧张（年末/季末/MLF到期），短期利空风险资产
- Shibor 持续低位 → 流动性充裕，有利于股市整体估值扩张
- 关注 Shibor 与 LPR 的走势背离：Shibor 下行 + LPR 未变 → 降息预期升温

#### 通胀环境

| 通胀环境 | 利好行业 | 利空行业 |
|---------|---------|---------|
| 温和通胀(CPI 1-3%) | 消费龙头(定价权)、品牌消费 | — |
| 高通胀(CPI>3%) | 能源、材料、实物资产 | 可选消费、科技 |
| 通缩(CPI<0%) | 必选消费、高股息 | 周期股、材料 |
| 滞胀(高CPI+低PMI) | 医药、公用事业、黄金概念 | 几乎所有风险资产 |

#### PMI 解读

- **PMI > 50 且上升**：经济扩张加速，利好周期股和制造业
- **PMI > 50 但下降**：扩张放缓，关注拐点信号
- **PMI < 50 且下降**：经济收缩加深，防御为主
- **PMI < 50 但上升**：经济触底回升迹象，可适度增配
- **非制造业PMI > 55**：服务业景气高涨，利好消费/服务类公司
- **非制造业PMI < 50**：服务业收缩，关注就业和消费数据

#### 经济增长指标解读

| 指标 | 健康区间 | 偏弱信号 | 关注要点 |
|------|---------|---------|---------|
| GDP季度同比 | 5-6% | <4.5% | 经济基本面，趋势比绝对值重要 |
| 社会消费品零售总额同比 | 5-8% | <3% | 内需强弱，消费升级/降级趋势 |
| 固定资产投资累计同比 | 4-7% | <3% | 基建/制造业投资意愿 |
| 工业增加值同比 | 5-7% | <4% | 工业生产景气度 |
| 出口同比 | 正增长 | 连续负增长 | 全球需求 + 贸易政策影响 |
| 进口同比 | 正增长 | 连续负增长 | 内需 + 原材料需求 |

**GDP 解读**：
- GDP 同比连续上升 → 经济复苏确认，利好周期/消费
- GDP 同比连续下降 → 经济放缓，关注刺激政策出台
- 注意：GDP 为季度数据，滞后性较强，需结合 PMI 等月度数据交叉验证

**消费与投资解读**：
- 社零高增 + FAI低迷 → 消费驱动型复苏，利好消费股
- 社零低迷 + FAI高增 → 基建/投资驱动型，利好基建/建材
- 两者均低迷 → 经济动力不足，关注刺激政策

#### 房地产市场解读

| 指标 | 回暖信号 | 下行信号 | 关联影响 |
|------|---------|---------|---------|
| 房地产开发投资同比 | 转正或收窄 | 持续负增长加深 | 影响建材/家电/银行 |
| 商品房销售面积同比 | 转正或持续正增长 | 持续负增长 | 领先于投资1-2个季度 |

- **销售回暖 + 投资仍低迷**：行业处于筑底阶段，关注政策宽松持续性
- **销售 + 投资均回暖**：地产周期反转确认，利好地产链（建材/水泥/玻璃/家电/家居）
- **销售持续萎缩**：地产下行压力，利空银行（信贷风险）、地方财政（土地出让收入）

#### 国债收益率与期限利差解读

| 利差状态 | 市场含义 | 投资策略 |
|---------|---------|---------|
| 10Y-2Y利差 > 80bp（陡峭化） | 经济复苏预期强，长端利率上行 | 增配周期/金融，减配长久期资产 |
| 10Y-2Y利差 30-80bp（正常） | 经济平稳运行 | 均衡配置 |
| 10Y-2Y利差 < 30bp（平坦化） | 经济放缓预期，或流动性收紧 | 增配防御，关注政策转向信号 |
| 10Y-2Y利差 < 0（倒挂） | 衰退预警信号 | 全面防御，超配国债/黄金/必选消费 |

- **10Y收益率快速上行 + 利差扩大** → 通胀预期升温，利空成长股，利好银行/保险
- **10Y收益率快速下行 + 利差收窄** → 避险情绪浓厚 或 降息预期，利好长久期资产（科技/成长）
- **收益率曲线倒挂**：历史上是经济衰退的强预警信号（美国尤为显著，中国参考价值略低但仍需关注）

#### 社融与 M2

- **社融放量 + M2上升**：流动性宽松信号，有利于股市整体
- **社融缩量 + M2下降**：流动性收紧，警惕估值压缩
- **社融结构**：关注中长期贷款占比（高→实体经济需求强）vs 短期票据冲量

## 2. 国际局势与地缘政治分析

### 2.1 核心关注维度

分析个股时必须考虑以下国际因素：

#### A. 中美关系与贸易政策

| 事件类型 | 直接影响行业 | 传导机制 |
|---------|------------|---------|
| 关税加征 | 出口制造、电子、农产品 | 成本上升→利润压缩 |
| 技术封锁/实体清单 | 半导体、AI、通信设备 | 供应链断裂→国产替代受益 |
| 贸易谈判进展 | 全市场 | 风险偏好变化→估值调整 |
| 金融脱钩风险 | 中概股、港股 | 资金流出→估值压缩 |

#### B. 地缘冲突

| 事件类型 | 受益标的 | 受损标的 |
|---------|---------|---------|
| 区域军事冲突 | 军工、能源、黄金 | 航空、旅游、全球贸易 |
| 能源供应中断 | 新能源、煤炭、天然气 | 高耗能制造、化工 |
| 制裁与反制裁 | 国产替代链 | 依赖进口的行业 |

#### C. 全球央行政策

| 事件 | A股影响 | 传导路径 |
|-----|--------|---------|
| 美联储加息 | 成长股承压、资金外流 | 利率差→汇率→资金流向 |
| 美联储降息 | 利好成长、缓解汇率压力 | 利率差缩小→外资回流 |
| 欧央行/日央行政策变化 | 间接影响外资配置 | 全球资产再平衡 |

#### D. 大宗商品联动

| 商品 | 关联行业 | 传导逻辑 |
|-----|---------|---------|
| 原油上涨 | 利好：石油开采、油服；利空：航空、化工下游 | 成本传导 |
| 铜价上涨 | 利好：铜矿、有色金属；反映经济预期改善 | 经济晴雨表 |
| 黄金上涨 | 利好：黄金股；反映避险情绪、通胀预期 | 避险需求 |
| 铁矿石/钢铁 | 利好：钢铁股（价涨）；利空：建筑成本 | 供需关系 |

#### E. 国家政策分析

| 政策类型 | 关注要点 | 影响行业 |
|---------|---------|---------|
| 产业政策 | 国务院/发改委/工信部出台的行业扶持政策、补贴、发展规划 | 目标行业直接受益 |
| 监管政策 | 证监会/银保监/市场监管总局的监管动态（如反垄断、行业整顿） | 被监管行业短期承压 |
| 财政政策 | 专项债发行节奏、减税降费、基建投资方向 | 基建/建材/消费/制造 |
| 地方政策 | 各地方政府的经济刺激措施（如人才补贴、购房补贴、消费券） | 区域性影响 |
| 货币政策 | 央行的窗口指导、结构性工具（PSL/碳减排工具/科技再贷款） | 定向受益行业 |

**分析要点**：
- 政策力度判断：是常规表态还是实质性措施（有预算/时间表/量化目标）
- 政策持续性：一次性刺激还是长期制度性变化
- 政策传导时滞：从出台到效果显现需要多长时间

#### F. 银行与金融机构行为分析

| 机构类型 | 关注行为 | 市场影响 |
|---------|---------|---------|
| 央行（人民银行） | 逆回购/MLF/PSL操作、SLF、降准降息、窗口指导 | 直接影响流动性和市场利率 |
| 商业银行 | 信贷投放方向、存贷款利率调整、理财产品收益率 | 反映实体经济资金需求和信用环境 |
| 社保基金 | 季度持仓变动、增减持方向 | 长线资金风向标，偏好低估值高分红 |
| 险资（保险资金） | 举牌、增减持、资产配置方向 | 长期配置型资金，偏好稳定现金流标的 |
| 公募基金 | 新基金发行、持仓调整、行业配置变化 | 反映市场热点和机构共识 |
| 外资机构（QFII/沪深港通） | 北向资金流向、MSCI/FTSE调仓、外资行评级 | 反映国际资本对A股的信心 |
| 私募/量化 | 策略转向、高频交易影响 | 短期市场波动来源 |

**分析要点**：
- **央行操作信号解读**：逆回购量变化→流动性松紧预判；MLF利率变化→LPR引导方向；降准信号→全面宽松预期
- **机构调仓方向**：多类机构同向增持→强共识信号；机构分歧→方向不确定
- **信贷结构分析**：中长期贷款占比高→实体经济需求真实回暖；票据冲量→信贷需求疲弱

### 2.2 事件驱动分析流程

当重大事件发生时，按以下步骤分析影响：

1. **事件识别**：通过 web_search 获取最新国际新闻和地缘政治事件
2. **影响链条**：画出事件→行业→公司的传导路径
3. **量化评估**：估算对营收/利润的潜在影响百分比
4. **时间维度**：区分短期冲击（情绪面）vs 长期影响（基本面）
5. **历史对标**：寻找历史上类似事件的市场反应作为参考
6. **对冲思维**：每个风险因素都考虑可能的对冲手段

分析时必须执行以下搜索（除公司层面搜索之外）：

```markdown

### 经济周期
- 当前阶段：{复苏/扩张/过热/收缩}
- 关键信号：PMI={X}({方向})，CPI={X}%，PPI={X}%
- 政策方向：{宽松/中性/偏紧}
- 行业轮动建议：超配{行业}，低配{行业}

- 全球市场：美股{涨/跌}，影响{正面/负面/中性}

- {正面因素2}：{具体说明}
- {风险因素1}：{具体说明}
- 综合影响：{利好/利空/中性}，影响程度{大/中/小}
```

## 4. 重大事件影响评估矩阵

评估重大事件对个股影响的系统框架：

| 评估维度 | 权重 | 评分(-5到+5) | 说明 |
|---------|------|-------------|------|
| 行业直接影响 | 30% | | 事件对所属行业的直接影响 |
| 供应链影响 | 20% | | 上下游供应链受影响程度 |
| 需求端影响 | 20% | | 终端需求变化 |
| 政策联动 | 15% | | 可能引发的后续政策变化 |
| 市场情绪 | 15% | | 短期市场情绪冲击 |

**综合评分**：
- **+3 到 +5**：重大利好，考虑加仓
- **+1 到 +2**：温和利好，维持持仓
- **-1 到 +1**：影响有限，不调整
- **-2 到 -1**：温和利空，关注发展
- **-5 到 -3**：重大利空，考虑减仓/止损

| 全部仪表盘 | 综合 | `macro_analysis.py --dashboard` |
| 全球局势/政策/金融机构新闻 | web_search + AI整理 | `--global-context-json` |
| 全网时政热点 | fetch_trending.py | `--trending-json` |

### 全网时政热点获取

通过 `fetch_trending.py` 脚本直接从各平台公开接口采集热榜数据，覆盖 13 个数据源：

| 数据源类型 | 包含平台 | 获取方式 |
|-----------|---------|---------|
| 综合搜索 | 百度热搜、今日头条 | `fetch_trending.py`（基础源） |
| 社交舆情 | 微博热搜、知乎热榜 | `fetch_trending.py`（基础源） |
| 财经投资 | 雪球财经 | `fetch_trending.py`（基础源） |
| 科技商业 | 36氪 | `fetch_trending.py`（基础源） |
| 时政新闻 | 澎湃新闻 | `fetch_trending.py`（基础源） |
| 科技扩展 | IT之家、哔哩哔哩 | `fetch_trending.py`（按行业扩展） |
| 消费扩展 | 抖音、豆瓣 | `fetch_trending.py`（按行业扩展） |
| 体育扩展 | 虎扑 | `fetch_trending.py`（按行业扩展） |
| 数码扩展 | 少数派 | `fetch_trending.py`（按行业扩展） |

**使用流程**：
1. Agent 执行 `fetch_trending.py` 脚本采集各平台热榜
2. 脚本将结果保存为 JSON（`trending.json`）
3. 通过 `macro_analysis.py --dashboard --trending-json trending.json` 合并到宏观仪表盘

5. 报告脚本自动渲染时政热点区块

### 国际与事件数据获取

国际局势、地缘政治、国家政策、银行/金融机构行为、全球市场联动数据**无法通过 API 获取**，必须通过 `web_search` 实时搜索：

- 宏观搜索指令生成：`macro_analysis.py --global-queries`
- Agent 按生成的搜索指令依次执行 `web_search`
- **将搜索结果结构化写入 `global_context.json`**
- 通过 `macro_analysis.py --dashboard --global-context-json global_context.json` 合并到宏观仪表盘
- 报告脚本自动从 `macro_dashboard.json` 的 `global_context` 字段渲染新闻区块

### 搜索结果整合方法

1. 每条 web_search 结果提取**关键事实**（避免主观推测）
2. 按「事件→影响链条→个股影响」逻辑整理
3. 标注信息来源和时间
4. 区分已确认事实 vs 市场预期/猜测
5. 写入 `global_context.json`，通过 `--global-context-json` 传入 `macro_analysis.py`
6. 报告中的"国际局势、政策与金融机构动态"区块将自动渲染这些内容
