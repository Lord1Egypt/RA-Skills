## Description: <br>
Provides DolphinDB-based quant finance guidance for factor computation, strategy backtesting, market data processing, performance attribution, and portfolio optimization. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and quant finance practitioners use this skill to draft DolphinDB and Python workflows for factors, backtests, OHLC and Level-2 market data processing, attribution, optimization, and real-time analytics. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Example DolphinDB connections use the default admin/123456 credentials. <br>
Mitigation: Replace example credentials with a least-privileged account and change default DolphinDB passwords before using the skill with real systems or data. <br>
Risk: Generated DolphinDB scripts or guidance may be run against financial or production data. <br>
Mitigation: Review generated scripts before execution, validate data sources and permissions, and test changes in a non-production environment first. <br>
Risk: The skill depends on referenced DolphinDB helper skills and shell wrappers. <br>
Mitigation: Install helper skills only from trusted sources and review wrapper scripts before sourcing them in an agent environment. <br>
