## 👶 母婴（baby）

> 婴幼儿用品、奶粉、辅食、童车等

### 美国（US）

**认证要求**：

- 🔴 强制 **CPSC认证**：消费品安全委员会认证，所有儿童产品强制
- 🔴 强制 **CPC证书**：儿童产品证书，制造商/进口商必须出具
- 🟡 条件 **FDA注册（食品类）**：奶粉、辅食等需FDA注册和预先申报
- 🔴 强制 **ASTM F963测试**：玩具安全标准测试

**标签要求**：

- 🔴 **追踪标签**：需包含制造商信息、生产日期、批次号
- 🔴 **年龄警告标签**：3岁以下 choking hazard 警告
- 🔴 **原产地标签**：标注Made in China

**包装要求**：

- 🔴 **防窒息警告**：包装需标注防窒息警告
- 🔴 **儿童安全包装**：特定产品需CR（Child-Resistant）包装

**测试要求**：

- 🟡 条件 **铅含量测试**：总铅含量≤100ppm
- 🟡 条件 **邻苯二甲酸盐测试**：DEHP/DBP/BBP各≤0.1%
- 🟡 条件 **可溶性重金属测试**：8种可溶性重金属限值

**特殊限制**：
- 12岁以下产品均视为儿童产品
- 强制第三方测试
- 需在CPSC登记小批量报告

### 英国（GB）

**认证要求**：

- 🔴 强制 **CE标志**：EU/UK玩具安全指令合规

- 🔴 强制 **EN 71测试**：欧洲玩具安全标准

- 🔴 **防窒息警告**：3岁以下产品必须有
- 🔴 **EPR包装法**：需注册包装回收系统

**测试要求**：

- 🟡 条件 **EN 71-1/2/3测试**：物理/阻燃/化学迁移
- 🟡 条件 **重金属测试**：可溶性重金属限值

**特殊限制**：
- 必须有UK负责人
- 食品接触材料需FSA批准
- 婴儿食品需特殊营养标签

## 📱 电子产品（electronics）

> 手机、电脑、家电、电子配件等

### 美国（US）

**认证要求**：

- 🔴 强制 **FCC认证**：联邦通信委员会认证，电子设备强制
- 🟢 自愿 **UL认证**：美国安全实验室认证（自愿但市场期待）
- 🟡 条件 **FDA注册**：辐射类设备需FDA注册
- 🟢 自愿 **ETL认证**：Intertek电气安全认证（替代UL）

**标签要求**：

- 🔴 **FCC标签**：产品需标注FCC ID
- 🔴 **原产地标签**：Made in China标注
- 🟢 **UL标记**：如获UL认证需标注

**包装要求**：

- 🔴 **说明书**：英文使用说明和安全警告
- 🔴 **合规声明**：FCC合规声明

**测试要求**：

- 🟡 条件 **FCC Part 15B测试**：电磁兼容测试
- 🟡 条件 **FCC Part 15C测试**：无线设备测试（如适用）
- 🟢 自愿 **UL安全测试**：电气安全测试

**特殊限制**：
- 锂电池需UN38.3测试
- 无线设备需FCC SDoC或Certification
- 激光产品需FDA/CDRH报告
