# Back Shu Point Safe Needling

## When to Use
- 针刺足太阳膀胱经背部背俞穴（如肺俞、心俞、肝俞、脾俞、胃俞等）
- 患者处于俯卧位，背部肌肉放松
- 无严重肺气肿、胸膜粘连或凝血功能障碍等禁忌症

## Execution Steps
1. **准确定位**：在第X胸椎棘突下，后正中线旁开1.5寸（同身寸）。例如：
   - 肺俞位于第3胸椎（T3）棘突下
   - 心俞位于第5胸椎（T5）棘突下

2. **体位准备**：令患者取俯卧位，充分放松背部肌肉，避免因紧张导致进针偏差。

3. **进针角度与深度控制**：
   - **必须向脊柱方向斜刺**，角度为30°~60°（通常采用45°~60°）
   - **进针深度严格限制在0.5～0.8寸**
   - **严禁直刺或向外侧（朝向肋间隙）深刺**，以防刺入胸腔造成气胸

4. **术中监控与应急处理**：
   - 上胸部穴位（如肺俞、心俞、膈俞）风险最高，操作需格外谨慎
   - 若患者突然咳嗽或移动，应立即退针
   - 如出现胸闷、呼吸困难等疑似气胸症状，立即停止操作并送医

5. **操作目的**：通过安全斜刺激发背俞穴经气，调节对应脏腑功能，同时避免内脏损伤。

## Precautions
- 禁用于严重肺气肿、胸膜粘连或凝血功能障碍患者
- 不可在非俯卧位或定位不清时强行进针

> **关键触发条件**：只要涉及T2–T12水平的背部背俞穴针刺，必须执行本安全操作流程。
