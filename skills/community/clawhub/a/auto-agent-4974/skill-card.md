## Description: <br>
Provides a Chinese-language tutorial helper for learning how to build personal AI agents from Bilibili resources. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[534422530](https://clawhub.ai/user/534422530) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent builders can use this skill as a learning aid for personal AI agent construction, including examples and links from Bilibili tutorials. It is intended to guide study and experimentation rather than operate as a standalone agent-building system. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The security scan reports that the executable code imports from an undeclared local Windows directory and can store user-provided input in a local knowledge base. <br>
Mitigation: Install only after reviewing and trusting the local D:\\coze-local\\db learn module, and avoid providing sensitive input unless local retention is acceptable. <br>
Risk: The broad agent trigger may activate the skill in more conversations than intended. <br>
Mitigation: Review trigger behavior before enabling the skill in shared or production environments. <br>
