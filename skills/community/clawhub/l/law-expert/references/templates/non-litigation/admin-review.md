# 行政复议申请书

## 适用场景
认为行政机关作出的具体行政行为（行政处罚、行政不作为、信息公开拒绝等）侵犯合法权益，申请上级行政机关或同级政府进行复议审查。

## 提交机构
作出具体行政行为的行政机关的上一级主管部门，或同级人民政府

## 模板正文

```
                    行政复议申请书

申请人：姓名______，身份证号：__________________
        住址：__________________，联系电话：____________
        指定接收法律文书地址：__________________

被申请人：______局/委员会/办公室
          住所地：____________________________
          法定代表人/负责人：______，职务：______

复议请求：
1. 请求撤销/变更/确认违法 被申请人作出的______
   （文号：______）______行政行为；
2. 请求责令被申请人赔偿损失______元；
3. ____________________________。

事实与理由：
______年______月______日，被申请人以______为由，
对申请人作出______（描述具体行政行为内容）。

申请人认为该具体行政行为存在以下违法/不当之处：
一、（事实认定）________________________________；
二、（法律适用）________________________________；
三、（程序问题）________________________________。

根据《______》第______条及《中华人民共和国行政复议法》
第______条之规定，该行政行为应当予以撤销/变更。

为此，特申请行政复议，恳请复议机关依法审查并支持申请人的请求。

此致
____________（复议机关名称）

附：
1. 本申请书副本______份；
2. 被复议行政行为文书复印件______份；
3. 申请人身份证明复印件；
4. 证据材料______份。

                                    申请人（签名）：
                                    ______年______月______日
```

## 注意事项
- **关键填写点**：准确的被复议行政行为（文号、作出日期、主要内容）；复议请求要明确且在法律允许范围内；复议理由必须具体（事实/法律/程序三层面）
- **常见错误**：超过复议期限（一般60日）；复议机关选择错误（看具体行政行为主体而定）；复议和诉讼在途时间混淆
- **份数要求**：正本1份 + 副本按被申请人数提交
- **复议不停止执行**：复议期间原行政行为一般不停止执行（特殊情况可申请停止）
- **时限提示**：知道具体行政行为之日起60日内提出复议申请；复议机关应在60日内作出决定
- **后续救济**：对复议决定不服，可在收到决定书15日内向法院提起行政诉讼
