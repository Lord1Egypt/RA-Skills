# 3PL Evaluation Quality Checklist

Run before delivering any 3PL comparison or audit. Every unchecked item is fixed or disclosed in Assumptions.

## Inputs & profile
- [ ] Order volume, growth rate, and SKU count captured
- [ ] Product handling traits noted (size/weight/fragile/lot/expiry/hazmat)
- [ ] Destination/zone mix from real order data, not guessed
- [ ] Peak multiplier and peak month identified
- [ ] Returns rate included
- [ ] Current fulfillment cost baseline established (incl. shipping)

## Cost normalization
- [ ] Same order basket applied to every candidate
- [ ] All fee-schedule lines requested in writing from each candidate
- [ ] Storage billed-basis confirmed (calendar vs. daily average)
- [ ] DIM divisor confirmed per candidate
- [ ] Shipping model identified (cost-plus / discount / marked-up)
- [ ] Peak month modeled separately with surcharges
- [ ] Account minimums and tech fees amortized per order
- [ ] Sub-$5 or outlier quotes re-verified, not celebrated

## Scoring
- [ ] Weights set before scoring and disclosed
- [ ] Every 4-5 score backed by demo, contract text, or reference call
- [ ] SLA scores based on credits and actuals, not targets
- [ ] Zone-weighted delivery coverage computed, not eyeballed
- [ ] Volume-fit check done (not bottom 10% of client sizes)

## Diligence
- [ ] 2+ reference calls in same category and volume band
- [ ] Peak season performance asked about specifically
- [ ] Fee creep and invoice accuracy asked about specifically
- [ ] Exit terms (data export, inventory release, wind-down fees) reviewed
- [ ] Auto-renewal and notice windows flagged

## Recommendation quality
- [ ] Landed cost table shows regular AND peak
- [ ] Switching cost and payback period computed where relevant
- [ ] Hybrid option (FBA + 3PL) considered and addressed
- [ ] Negotiation levers with asks and fallbacks included
- [ ] Transition plan has cutover timing and rollback trigger
- [ ] First-90-days invoice audit recommended
- [ ] Red flags listed per candidate
- [ ] All estimates marked "(est.)" and assumptions numbered
