# 学习规划模板

## 周计划模板

```
周一至周五：主学习时间
  上午（高精力）：新知识学习/难点攻克
  下午（中精力）：练习/项目实践
  晚上（低精力）：复习/轻松阅读

周六：综合实践/项目日
周日：复盘 + 下一周规划
```

## 日计划模板

```

时间分配：
  🟢 09:00-11:00 | 新知识学习（番茄×4）
  🟡 14:00-16:00 | 练习/实战（番茄×4）
  🔵 19:00-20:00 | 复习昨日内容
  📝 20:00-20:30 | 记录今日收获

今日完成：
  ✅/❌ 目标1: ______
  ✅/❌ 目标2: ______
  ✅/❌ 目标3: ______

反思：______
明日重点：______
```
