## 使用建议

1. **新主持人**: 先阅读 01-quick-rules.md 了解游戏基础
2. **准备游戏**: 参考 05-keeper-guide.md 和 08-appendix.md
3. **游戏中**: 快速查阅 02-game-system.md 和 03-combat-and-chase.md
4. **特殊场景**: 根据需要查阅 04-sanity.md、06-books-spells-magic.md、07-mythos-creatures.md

本规则基于《克苏鲁的呼唤》第七版守秘人规则书（中文翻译版2002c）。
