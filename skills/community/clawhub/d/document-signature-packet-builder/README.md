# Document Signature Packet Builder

Turn document-signing instructions into a practical sign-here and logistics checklist — avoid missed signatures, dates, initials, and submission details.

## What it does
- Build sign-here checklists for multi-document packets with per-page requirements
- Track signature, date, initial, witness, notary, attachment, and copy requirements
- Generate logistics checklists for ID, appointments, payment, and submission methods

## Example scenarios

**1. 合同签约材料包整理**
> 👤 签房租/工作/合伙合同需要准备一堆材料，每次东找西找。有没有一个万能材料清单模板？
> 🤖 常见签约材料包模板——租房类：身份证复印件+收入证明+工作证明+紧急联系人+押金转账截图；工作类：身份证+学历证书+离职证明+银行卡+1寸照片；合伙类：身份证+征信报告+出资证明+合伙人协议草稿。建议所有扫描件存加密云文件夹，需要时直接打印或发PDF。

**2. Home purchase packet assembly**
> 👤 Build a packet for a home purchase: purchase agreement, lead disclosure, title affidavit. Buyer signs first, then seller.
> 🤖 Packet manifest with document order, signer slots per doc, required notation fields (initials on each page, date, witness line) flagged.

**3. Send packet to multiple signers**
> 👤 I need to send this packet to 3 signers. Generate separate envelopes for each with their pages only.
> 🤖 Three sub-packets extracted: Buyer envelope (all pages), Seller envelope (seller-signature pages only), Notary envelope (notarization pages).
