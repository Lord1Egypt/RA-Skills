## Use Case: <br>
External users use this skill to search for inspection providers, check availability, and create inspection bookings through Lokuli's MCP server. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can send personal contact details to an external Lokuli service. <br>
Mitigation: Require confirmation of the provider, service, date and time, contact details, and disclosure that personal information will be sent to Lokuli before booking. <br>
Risk: The skill can create bookings through an external service. <br>
Mitigation: Ask the user for explicit approval before calling the booking tool. <br>
