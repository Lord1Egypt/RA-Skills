## Description: <br>
EigenSkill Builder helps agents create, validate, and publish ClawHub skills by covering skill structure, metadata, reference conventions, negative boundaries, CLI workflows, and quality checks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent builders use this skill to create or validate ClawHub skills, define trigger and exclusion boundaries, organize references, and prepare publishing workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Publishing, update, or install commands can publish content under a user account or change locally installed skills. <br>
Mitigation: Review generated skill files and intentionally approve any clawhub publish, clawhub update, or clawhub install command before running it. <br>
