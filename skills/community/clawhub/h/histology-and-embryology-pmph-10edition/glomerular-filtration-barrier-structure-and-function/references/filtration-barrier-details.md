# 肾小球滤过屏障详细参数与病理关联

## 分子量阈值与电荷特性
- **可滤过分子量上限**：约70 kDa（白蛋白为69 kDa，因其带负电仍被有效阻挡）
- **电荷选择性机制**：主要由基膜中的硫酸类肝素蛋白聚糖（heparan sulfate proteoglycans）提供负电荷

## 常见疾病中的屏障损伤
- **糖尿病肾病**：早期GBM增厚，晚期足细胞脱落
- **微小病变肾病**：足细胞裂孔膜结构紊乱，nephrin表达下调
- **膜性肾病**：抗PLA2R抗体沉积于GBM外侧，激活补体损伤足细胞

## 实验与临床指标
- 尿白蛋白/肌酐比（UACR）>30 mg/g 提示肾小球滤过屏障受损
- 电子显微镜可观察GBM厚度及足突融合
