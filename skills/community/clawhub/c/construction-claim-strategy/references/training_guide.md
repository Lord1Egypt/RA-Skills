## 1. Why Strategic Planning Matters

| Problem | Consequence |
|---|---|
| Jumping straight to clause selection | Arguments lack coherence and direction |
| No scope control | Volunteering information that creates new issues |
| Wrong argument type chosen | Position looks weak even when merits are strong |
| Over-disclosing early | Giving the other side material to attack |
| No risk assessment | Caught off-guard by counter-arguments |

**Key message:** A few minutes of strategic planning saves hours of rework and significantly improves the quality of the response.

## 2. The 7-Dimension Framework

| Dimension | Focus Area |
|---|---|
| 1. Situation Assessment | What exactly are we responding to? |
| 2. Scope Control | What to address, what to avoid |
| 3. Direction Decision | Primary + fallback arguments |
| 4. Argument Strategy | Which argument type to use |
| 5. Disclosure Control | Precise / General / Defer |
| 6. Response Architecture | Which structure pattern to use |
| 7. Risk Assessment | Counter-arguments and weak points |

---

## 3. Using the Wizard Effectively

### 5 Practical Tips

1. **Be specific in Dimension 1** — Name the exact letter/instruction you're responding to.
2. **Write down what you will NOT say** in Dimension 2 — scope control is about restraint.
3. **Always try Textual Interpretation first** in Dimension 4 — only move down the ranking if needed.
4. **Use the Review step** — this is where most improvements happen.
5. **Save the strategy** for reuse if the matter is ongoing.

## 4. Turning Strategy into a Formal Response

### Step-by-Step Flow

```
Strategy Report → Response Letter Template → Draft → Review → Final
```

| Strategy Report Section | Maps to Letter Section |
|---|---|
| Situation Assessment | Introduction / Acknowledgement |
| Scope Control | Scope of Response |
| Primary Argument | Primary Position |
| Fallback Argument | Alternative Position |
| Disclosure Control | What to include / what to defer |
| Risk Assessment | Internal preparation only |

---

## 5. Common Mistakes vs Fixes

| Mistake | Fix |
|---|---|
| Citing the weakest clause first | Start with the strongest textual argument |
| Saying "the clause does not apply" | Concede the ordinary case, then distinguish |
| Answering questions that weren't asked | Strictly follow scope control |
| Giving detailed costs before records are ready | Defer quantification to formal claim report |
| Using implied terms when express text supports you | Always check textual interpretation first |
| Same structure for every reply | Choose the right response architecture pattern |

---

## 6. Best Practices Summary

1. **Plan before drafting** — always run through the 7 dimensions.
2. **Lead with your strongest argument** — Textual Interpretation whenever possible.
3. **Combine primary + fallback** — shows strength while preserving alternatives.
4. **Match disclosure to argument strength** — stronger = more precise.
5. **Apply scope control strictly** — respond only to what was asked.
6. **Revisit the strategy** if new information appears before sending.

---

## 7. Key Takeaways

- Strategic planning is not optional — it's what separates strong claims from weak ones.
- The argument ranking exists because not all arguments carry the same weight.
- Disclosure control prevents you from giving the other side ammunition.
- Scope control prevents you from creating new problems.
- A structured response architecture makes your position more persuasive.

**Remember: Plan → Strategise → Draft**

## This Skill vs Traditional Approach

| Aspect | Traditional Approach | This Skill |
|---|---|---|
| Starting point | Drafting or selecting clauses | 7-dimension strategic analysis |
| Argument selection | First argument that comes to mind | Ranked by strength |
| Scope management | Often volunteers extra info | Explicit scope control step |
| Disclosure | Too much or too little | Clear disclosure levels |
| Response structure | Standard letter format | Consciously chosen architecture |
| Risk management | Considered late or not at all | Dedicated risk assessment |
| Consistency | Depends on individual experience | Repeatable framework |

**Biggest advantages appear on medium to high-value claims**, responses that need to preserve future rights, or when dealing with difficult counterparties.
