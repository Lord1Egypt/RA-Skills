# 巴厘岛旅游资源标准化文档体系

## 一、概述

本体系覆盖巴厘岛旅游业务中的 8 类核心资源，每类资源对应独立 CSV 库，采用统一的三层标签体系和结构化字段规范。

**三层标签体系：**
- 第一层：人群标签（tags_audience）
- 第二层：风格标签（tags_style）
- 第三层：特色标签（tags_featured，按资源类型定制）

**汇率参考：**
- 1 CNY ≈ 2,100 IDR
- 1 USD ≈ 15,000 IDR

## 十、异常处理规范

| 异常类型 | 判断标准 | 处理方式 |
|---------|---------|---------|
| 资源过期 | 合同有效期 < 今天 | 🔴 标红备注"已过期"，可信度降B级 |
| 价格缺失 | 核心价格字段为空 | 🔴 暂停入库，告警人工补充 |
| 价格冲突 | 同资源价差 > 10% | 🔴 标红，暂停入库，等人工决策 |
| GPS缺失 | 坐标字段为空 | ⚠️ 备注"待补充"，不阻止入库 |
| OTA缺失 | 评分字段为空 | ⚠️ 尝试 Google 搜索补录 |
| 标签缺失 | 三层标签不完整 | ⚠️ 按资源类型默认补充基础标签 |
| 数据完整度低 | 字段完整度 < 60% | ⚠️ 可信度标C级，优先处理高完整度资源 |

## 十一、写入规范

1. **编码格式**：必须使用 `utf-8-sig`（含 BOM），否则 Excel 打开会乱码
2. **多房型资源**：每种规格单独一行，基础信息重复填写
3. **Zone ID 编码规则**：
   - `Zone-Kuta` / `Zone-Seminyak` / `Zone-NusaDua` / `Zone-Ubud`
   - `Zone-OFF` = 离岛（Gili Trawangan / Nusa Lembongan / Nusa Penida）
4. **价格单位**：统一使用 IDR（印尼盾），备注中可换算 CNY/USD
