# 三合派理论 — 一念紫微斗数

## 核心思想

三合派，也叫中州派，是紫微斗数在港台最主流的流派。

**核心理念：** 以星曜性质和亮度解读为主，配合三方四正联动。

## 三方四正

- **三方**：本宫 + ±4宫（三合关系）
- **四正**：本宫 + 对宫(±6)
- 例：命宫寅（2）→ 三方财帛宫午（6）官禄宫戌（10）→ 四正迁移宫申（8）

## 星曜亮度体系

| 亮度 | 含义 |
|---|---|
| 庙 | 星曜力量最强，最吉 |
| 旺 | 力量强，吉 |
| 得 | 力量中等，吉 |
| 利 | 偏正面 |
| 平 | 中性 |
| 陷 | 力量弱，偏凶 |

## 常用星组

- **紫府相格**（紫微+天府+天相）：贵气、稳重
- **杀破狼格**（七杀+破军+贪狼）：变动大、机遇多
- **机月同梁格**（天机+太阴+天同+天梁）：稳成、才艺
- **日月同宫格**：太阳太阴同宫，贵气

## 空宫处理

某宫无主星 → 借对宫星曜论（需标明"借"）

# 飞星派理论 — 一念紫微斗数

## 核心思想

飞星派，师承钦天门体系。

**核心理念：** 以四化飞星的脉络串联为主线，宫干飞四化为主。

## 常用技法

- **宫干飞四化**：本宫天干四化入他宫
- **自化**：本宫自飞四化（天干四化入本宫）
- **飞宫串联**：A宫飞四化入B宫 → B宫再飞入C宫
- **来因宫**：出生年干的宫位，定方向

## 体用宫论

- **体宫**：命宫、财帛宫、官禄宫（本质）
- **用宫**：其他九宫（表现）

# 占验派理论 — 一念紫微斗数

## 核心思想

占验派，由紫云大师发扬。

**核心理念：** 以太岁入卦法为核心，结合星曜互涉和特殊格局。

## 常用技法

- **太岁入卦**：年干定位特殊影响
- **星曜互涉**：星与星之间的相互作用
- **特殊格局**：如紫微百官朝拱、天府守库等
