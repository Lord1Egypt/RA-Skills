## Description: <br>
Erebos namespace for Netsnek e.U. dark theme and theming engine. Generates accessible dark mode palettes, manages theme tokens, and provides runtime theme switching for web apps. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use Erebos to add dark mode support, accessible palette generation, CSS theme tokens, and runtime theme switching to web applications. <br>

## Known Risks and Mitigations: <br>
Risk: The skill requests shell execution permission, so future script changes could do more than the current benign helper. <br>
Mitigation: Review scripts/theme-gen.sh and run the skill in a constrained workspace before executing updated releases. <br>
