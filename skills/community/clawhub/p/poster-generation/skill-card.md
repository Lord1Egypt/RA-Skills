## Description: <br>
Generates image poster content and files from user requirements using built-in design specifications, templates, and content-generation guidance. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wangjiaocheng](https://clawhub.ai/user/wangjiaocheng) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and content creators use this skill to plan, generate, format, and save advertising, promotional, event, brand, or product poster assets from a brief. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may create output files and could overwrite existing files if a path is ambiguous. <br>
Mitigation: Ask the user to confirm the output path before saving and avoid overwriting existing files unless explicitly requested. <br>
Risk: The trigger phrase is broad and may activate the skill for loosely related requests. <br>
Mitigation: Confirm that the user wants poster generation before proceeding when the request is ambiguous. <br>
