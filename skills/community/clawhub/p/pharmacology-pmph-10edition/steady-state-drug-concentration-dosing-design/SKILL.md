# 稳态血药浓度与给药方案设计

## 适用场景
- 药物需长期给药以维持疗效
- 药物治疗窗窄（如地高辛、苯妥英钠）
- 已知个体或群体药动学参数（CL、F、Vₛₛ、t₁/₂）
- 已设定明确的靶浓度范围

## 核心步骤

### 1. 判断是否适用
- 仅适用于按一级动力学消除的药物
- 若个体药动学参数变异大，必须结合血药浓度监测

### 2. 计算平均稳态浓度（C̄ₛₛ）
- 使用公式：C̄ₛₛ = (F·D) / (CL·τ)
- 或等价形式：C̄ₛₛ = AUC（单剂量）/ τ

### 3. 设计维持剂量
- 目标：使药物输入速率等于消除速率
- 维持给药速率 = (CL × 靶浓度) / F
- 根据临床剂型确定具体剂量（D）与给药间隔（τ）

### 4. 计算负荷剂量（Loading Dose）
- 公式：负荷剂量 = (靶浓度 × Vₛₛ) / F
- 特殊情形：
  - 口服给药且 τ = t₁/₂ → 首剂加倍
  - 静脉滴注 → 负荷量 = 1.44 × 第一个 t₁/₂ 的滴注量静推

### 5. 评估峰谷浓度与波动
- 计算 Cₛₛ,ₘₐₓ 和 Cₛₛ,ₘᵢₙ
- 评估波动度（DF）和累积因子（R）
- 确保 Cₛₛ,ₘₐₓ < MTC 且 Cₛₛ,ₘᵢₙ > MEC

### 6. 实施与调整
- 给药后监测血药浓度
- 根据实测浓度修正 CL 或 Vₛₛ
- 迭代优化给药方案

## 注意事项
- 负荷剂量可能引发毒性，尤其对敏感患者或长半衰期药物
- 治疗窗窄药物必须监测峰浓度与谷浓度
- 不可通过缩短给药间隔提前达到稳态，仅能改变浓度水平
