# 肛门与直肠检查的体位选择

## 适用场景
- 需进行肛门或直肠检查
- 患者可配合体位调整

## 禁忌与限制
- 不适用于无法移动的危重患者（除非采用仰卧位）
- 某些体位不适用于特定人群（如肘膝位一般不用于女性）

## 执行步骤

1. **评估患者状态与检查目的**
   - 明确患者是否年老体弱、重症、女性或其他特殊状态
   - 确定检查目标：前列腺/精囊检查、内镜、直肠脱垂观察、双合诊等

2. **选择合适体位并执行摆放**
   - **肘膝位**：适用于前列腺、精囊及内镜检查；暴露充分，但年老体弱或女性患者慎用
   - **左侧卧位**：适用于病重、年老体弱或女性患者；医生位于患者背后操作
   - **仰卧位或截石位**：适用于重症体弱患者、膀胱直肠窝检查或需直肠双合诊
   - **蹲位**：专门用于检查直肠脱出、内痔及直肠息肉等需增加腹压才能显现的病变

3. **标准化记录病变位置**
   - 所有病变必须按时钟方向标注，并注明所用体位
   - **肘膝位**：肛门后正中为12点，前正中为6点
   - **仰卧位**：后正中为6点，前正中为12点

4. **完成检查后沟通**
   - 向患者简要说明重要发现及后续计划

> 注意：体位选择直接影响检查效果与患者舒适度，务必结合临床实际灵活应用。
