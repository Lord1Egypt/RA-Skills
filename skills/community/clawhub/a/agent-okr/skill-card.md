## Description: <br>
Agent OKR helps agent team operators define monthly objectives, weekly tasks, and weekly reports with strategic and tactical review checkpoints. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[mayf3](https://clawhub.ai/user/mayf3) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent team operators use this skill to create and maintain file-based OKRs for agents, including monthly objectives, key results, weekly task breakdowns, and weekly review reports. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may create or update local OKR and report YAML files that affect agent planning records. <br>
Mitigation: Run it only in repositories where those files are expected, review generated diffs before committing, and keep human approval for planning decisions. <br>
