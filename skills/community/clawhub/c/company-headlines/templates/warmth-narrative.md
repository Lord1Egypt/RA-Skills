# 温度叙事模板

## 适用场景
节日祝福、节气海报、员工故事、文化活动

## 事件级别
L1 常规

## 结构骨架

```
情感钩子 → 故事锚点 → 价值观共振 → 祝福/行动
```

## 填写表

| 段落 | 写什么 | 字数 | 铁律 |
|------|--------|------|------|
| 情感钩子 | 节日/节气的文化意涵或一个具体的人 | 40-60字 | **真诚**，不煽情 |
| 故事锚点 | 一个真实小事或场景 | 60-100字 | **越具体越有力** |
| 价值观共振 | 和公司使命的**自然**连接 | 40-60字 | 不生硬拔高 |
| 祝福/行动 | 温暖的收尾 | 30-40字 | 有温度不油腻 |

## 变量槽

| 变量 | 说明 | 示例 |
|------|------|------|
| {{FESTIVAL}} | 节日/节气名称 | 端午 |
| {{EMOTION_HOOK}} | 情感切入点 | 外婆包粽子的记忆 |
| {{STORY_ANCHOR}} | 具象故事 | 财务顾问小陈帮客户发现多缴税款 |
| {{VALUE_CONNECTION}} | 与业务的自然连接 | 一层层的认真裹住了信任 |
| {{BLESSING}} | 祝福落脚点 | 愿你的较真都被理解 |

## 自检

- [ ] 开头是个人记忆还是百度百科？（要前者）
- [ ] 故事是具体的还是泛泛的？（要具体的人、具体的事）
- [ ] 价值观连接是自然的还是硬拔的？
- [ ] 祝福有具体落点吗？

## 常见问题

| 问题 | 修正 |
|------|------|
| 节日百科式开头 | 从个人记忆/生活场景切入 |
| 强行拔高到公司使命 | 找到自然的、轻的连接点 |
| "祝大家XX快乐，XX如意" | 祝福要落到具体的事情上 |
| 太长太煽情 | 真诚克制的温度，比煽情更有力 |

## 示例（财税公司+端午）

> **情感钩子**：小时候，端午是一只粽子。外婆把糯米、红枣裹进粽叶，说裹住了味道才不会散。
>
> **故事锚点**：今年上半年，我们的财务顾问小陈帮一家连锁超市做审计。他发现一笔被多缴的税款，追了三个月的流程帮客户退了回来。客户说谢谢，小陈说应该的。
>
> **价值观共振**：做财税和包粽子有点像——一层层地裹，一分分地较真。裹住的不是食材，是信任。
>
> **祝福/行动**：端午安康。愿你的较真都被理解，愿你的专业都有回响。
