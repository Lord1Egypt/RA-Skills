# 卷级设计模板

用于在正式写章节前，为每一卷建立稳定的中线骨架，避免长篇只剩重复升级和重复打脸。

---

## 标准卷模板

```markdown
# 第X卷 [卷名]

> **卷定位**：[崛起卷 / 立威卷 / 扩张卷 / 转折卷 / 终局卷]
> **卷目标**：[主角这一卷必须达成什么]
> **卷反派**：[本卷核心压迫源是谁]
> **卷爽点主题**：[身份翻盘 / 越级碾压 / 势力扩张 / 资源争夺 / 真相揭露]
> **卷情绪曲线**：[持续压迫 / 低开高走 / 多次反转 / 卷终爆发]

## 一、卷起点
- 主角当前实力：
- 主角当前地位：
- 主角当前关系网：
- 已知外部压力：

## 二、卷核心冲突
- 明线冲突：[表层要解决的问题]
- 暗线冲突：[更深层的秘密/误会/阴谋]
- 主角代价：[这一卷需要付出什么]

## 三、卷反派设计
- 反派目标：
- 反派优势：
- 反派压迫方式：
- 反派阶段性得手：
- 反派最终败因：

## 四、卷节点设计
1. 第1节点：[卷开局冲突]
2. 第2节点：[局势升级]
3. 第3节点：[阶段性逆转或受挫]
4. 第4节点：[卷终高潮]

## 五、卷结果
- 实力变化：
- 地位变化：
- 关系变化：
- 认知变化：

## 六、卷尾钩子
[下一卷最大的悬念或更高层级的问题]
```

---

## 卷级自检清单

- [ ] 这一卷不是单纯换地图刷怪
- [ ] 卷反派与上一卷反派的压迫方式不同
- [ ] 主角不是只涨等级，还在身份/关系/认知上发生变化
- [ ] 卷终既有胜利，也留下更大的代价或悬念
