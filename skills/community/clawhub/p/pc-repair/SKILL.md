# 电脑维修智能助手

## 概述

本技能提供电脑故障的交互式诊断和维修指导。当用户报告电脑问题时，通过结构化问诊定位故障根因，提供分层处理方案（紧急自救→DIY维修→专业维修），并生成可视化的维修决策报告。

## 触发条件

当用户提到以下关键词时，应使用本技能：
- **故障现象**：无法开机、蓝屏、黑屏、死机、重启、卡顿、风扇噪音、发热、无法联网
- **硬件问题**：硬盘损坏、内存故障、显卡问题、电池不耐用、键盘失灵、屏幕闪烁
- **系统问题**：系统崩溃、病毒感染、更新失败、驱动问题
- **维修需求**：电脑维修、硬件升级、清灰保养、数据恢复
- **英文触发**：pc repair, computer fix, blue screen, slow computer

## 核心工作流程

### 阶段1：交互式故障采集

通过结构化问诊采集关键信息，生成故障画像：

**必问维度（根据故障类型动态选择）：**

1. **设备信息**
   - 设备类型：台式机/笔记本/一体机
   - 品牌型号：联想/戴尔/惠普/华硕/苹果/自定义组装
   - 使用年限：<1年/1-3年/3-5年/>5年
   - 操作系统：Windows 10/11/macOS/Linux

2. **故障描述**
   - 故障现象：无法开机/蓝屏/黑屏/卡顿/异响/其他
   - 发生频率：首次/偶尔/频繁/持续
   - 触发条件：开机时/运行中/特定操作后/无缘无故
   - 错误代码：如有蓝屏代码/蜂鸣声/指示灯状态

3. **近期变更**
   - 硬件变更：是否加过内存/换过硬盘/清过灰
   - 软件变更：是否更新过驱动/安装新软件/系统更新
   - 环境因素：是否进过水/摔过/雷击/停电

**问诊策略：**
- 先让用户自由描述，提取关键信息
- 针对模糊描述追问细节（例："电脑很卡" → 追问是开机慢还是软件运行慢）
- 使用选择题减少用户输入负担

### 阶段2：故障诊断与分类

基于采集信息，将故障分类并给出诊断结论：

**故障分类树：**

```
电脑故障
├── 无法开机类
│   ├── 完全无反应（电源/主板/CPU问题）
│   ├── 有反应但黑屏（显卡/内存/显示器问题）
│   └── 开机卡LOGO（硬盘/BIOS问题）
├── 系统运行类
│   ├── 蓝屏死机（硬件兼容性/驱动/系统文件）
│   ├── 卡顿缓慢（硬盘/内存/CPU/软件）
│   └── 频繁重启（散热/电源/内存）
├── 硬件故障类
│   ├── 存储问题（硬盘坏道/数据恢复）
│   ├── 显示问题（花屏/闪屏/偏色）
│   ├── 声音问题（无声/杂音/风扇狂转）
│   └── 外设问题（键盘/鼠标/USB/网络）
└── 性能升级类
    ├── 内存升级（兼容性检查/安装指导）
    ├── 硬盘升级（SSD迁移/系统转移）
    └── 清灰保养（拆机指导/硅脂更换）
```

**诊断输出格式：**
- 故障定位：明确指出可能的故障部件（例：内存条接触不良）
- 置信度：高/中/低（基于症状匹配度）
- 严重程度：P0紧急（数据风险）/P1重要（影响使用）/P2一般（影响体验）

## 常见故障诊断知识库

### 1. 无法开机故障

**症状：按下电源键无任何反应**
```
可能原因：
1. 电源问题（插座没电/电源线松动/电源损坏）
2. 主板问题（主板短路/电容鼓包）
3. 前面板开关故障

诊断步骤：
1. 检查插座是否有电（插手机充电器测试）
2. 检查电源线是否插紧（重新插拔）
3. 尝试短接主板PWR_SW引脚（排除开关故障）
4. 闻一下电源是否有焦味（电源烧毁）

紧急处理：
- 台式机：尝试更换电源线/换个插座
- 笔记本：尝试拔掉电源只用电池/换个充电器
```

**症状：开机有反应但黑屏**
```
可能原因：
1. 内存接触不良（最常见）
2. 显卡故障（独立显卡松动/损坏）
3. 显示器问题（数据线松动/显示器坏了）
4. 主板BIOS损坏

诊断步骤：
1. 听声音：是否有蜂鸣器报警声（内存故障会"滴-滴"响）
2. 看指示灯：键盘Caps Lock是否能亮起（判断是否在系统）
3. 拔插内存：关机断电，拔掉内存条用橡皮擦金手指，重新插紧
4. 最小系统法：只留CPU+内存+电源，逐一添加配件测试

紧急处理：
- 拔插内存（90%的黑屏是内存接触不良）
- 清除CMOS（主板电池放电）
- 尝试集成显卡输出（如果有独显）
```

### 2. 蓝屏死机故障

**常见蓝屏代码及解决方案：**

| 错误代码 | 含义 | 可能原因 | 解决方案 |
|----------|------|----------|----------|
| 0x0000007B | 无法访问启动设备 | 硬盘模式设置错误/硬盘损坏 | 进BIOS改AHCI模式/检查硬盘连接 |
| 0x00000050 | 内存访问违规 | 内存故障/驱动冲突 | 拔插内存/进安全模式卸载最近安装的驱动 |
| 0x000000D1 | 驱动程序错误 | 显卡驱动/网卡驱动冲突 | 安全模式卸载驱动/回滚驱动版本 |
| 0x00000124 | 硬件错误 | CPU/内存/主板硬件故障 | 降低CPU频率/检查散热/内存加压 |
| CRITICAL_PROCESS_DIED | 关键进程死亡 | 系统文件损坏/硬盘坏道 | 重装系统/更换硬盘 |

**诊断工具推荐：**
- BlueScreenView（分析蓝屏dump文件）
- MemTest86（内存压力测试）
- CrystalDiskInfo（硬盘健康检测）

### 3. 电脑卡顿缓慢

**诊断流程图：**
```
电脑卡顿
├── 开机慢（>1分钟）
│   ├── 硬盘问题：用CrystalDiskInfo检查硬盘健康度
│   │   └── 解决方案：更换SSD（效果最明显）
│   └── 启动项太多：用任务管理器禁用不必要的启动项
├── 运行慢（打开软件卡）
│   ├── 内存不足：任务管理器查看内存占用率
│   │   └── 解决方案：加内存条（8G→16G）
│   ├── CPU占用高：任务管理器查看CPU占用
│   │   └── 解决方案：关闭挖矿病毒/散热清灰
│   └── 硬盘读写慢：任务管理器查看磁盘占用率
│       └── 解决方案：更换SSD/清理硬盘空间
└── 上网慢（只有浏览器卡）
    ├── 浏览器插件太多：禁用不必要插件
    ├── DNS问题：改成114.114.114.114或8.8.8.8
    └── 网络本身慢：测速网站测试（speedtest.cn）
```

**升级建议优先级：**
1. **换SSD**：开机从1分钟→10秒，效果最明显（成本200-500元）
2. **加内存**：8G→16G，多任务不卡（成本200-400元）
3. **清灰保养**：CPU降频导致卡顿（成本50-100元）

### 4. 硬件升级指导

**内存升级兼容性检查表：**

| 检查项 | 台式机 | 笔记本 |
|--------|--------|--------|
| 内存类型 | DDR3/DDR4/DDR5（看主板） | DDR3/DDR4/DDR5（看型号） |
| 内存频率 | 可向下兼容（买高不买低） | 必须匹配（买原装同款） |
| 最大容量 | 看主板支持（一般64G+） | 看型号支持（一般32G） |
| 插槽数量 | 2-4个（可用CPU-Z查看） | 1-2个（拆机或用AIDA64查看） |
| 购买建议 | 买套装（2条同品牌同型号） | 买单条（留一个插槽备用） |

**SSD升级迁移方案：**
1. **克隆迁移**（推荐）：用DiskGenius将旧硬盘整盘克隆到新SSD
   - 优点：不用重装系统，软件配置都保留
   - 缺点：新SSD容量必须≥旧硬盘已用空间
2. **全新安装**（干净）：制作Windows安装U盘，全新安装到SSD
   - 优点：系统最干净，没有历史包袱
   - 缺点：需要重装所有软件，激活可能失效

## 防坑指南

### 维修店常见套路

1. **虚报故障**：小问题说成大问题（换电容说成换主板）
   - **应对**：要求看故障配件，保留替换下来的旧件
2. **以次充好**：用二手配件当新的卖
   - **应对**：要求开发票，注明配件型号和保修期
3. **漫天要价**：看人下菜碟（外地口音/女性/老人报价更高）
   - **应对**：多问几家，用本技能估算价格
4. **偷换配件**：修的时候把你好的配件换成坏的
   - **应对**：送修前拍照记录序列号

- [ ] 询问保修政策（是否保修/保修多久）
