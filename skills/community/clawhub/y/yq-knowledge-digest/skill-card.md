## Description: <br>
Converts learning resources such as PDFs, textbooks, and topics into personalized study materials, including handwritten-style notes, interactive quizzes, slides, mind maps, and audio lessons. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[tianheihei002](https://clawhub.ai/user/tianheihei002) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External learners, educators, and study-support agents use this skill to transform source material or an approved topic search into grade-adapted learning assets. It is intended for study-material generation across primary, junior, senior, university, and professional levels. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated educational content may contain inaccurate explanations, quiz answers, or grade-level mismatches. <br>
Mitigation: Review generated materials against the source document and intended education level before using them for study or instruction. <br>
Risk: Topic-based generation may rely on web sources when the user has no uploaded material and explicitly approves search. <br>
Mitigation: Confirm whether web sources should be used and prefer authoritative sources before generating the learning assets. <br>
Risk: The skill can create multiple file types, including PDFs, HTML, PPTX, images, Mermaid content, and audio. <br>
Mitigation: Confirm requested output formats, language, and file location before running the workflow. <br>
