# 坏死性凋亡分子机制分析

## 何时使用
- 细胞呈现坏死样形态（如肿胀、质膜破裂），但死亡过程受调控；
- 已确认caspase未被激活（即非caspase依赖性死亡）；
- 存在TNF配体结合、病毒感染或已知RIP1/RIP3通路激活条件；
- 需区分坏死性凋亡与意外坏死或经典凋亡。

## 执行步骤

1. **确认启动条件**：
   - 检查是否存在肿瘤坏死因子受体1（TNFR1）配体结合，或RNA/DNA病毒蛋白刺激；
   - 确认caspase活性缺失（如通过caspase活性检测或Western blot验证caspase未切割）。

2. **评估核心信号通路**：
   - 检测RIP1与RIP3是否形成necrosome复合体（可通过免疫共沉淀或磷酸化RIP3水平判断）；
   - 观察下游效应：线粒体ATP生成减少、活性氧（ROS）显著升高、溶酶体膜通透性改变。

3. **识别形态与生化特征**：
   - 显微镜下确认早期细胞膜破裂（区别于凋亡的膜完整性）；
   - 检测线粒体膜电势丧失（如JC-1染色）；
   - 注意是否伴随自噬现象及细胞内容物外泄引发的炎症反应。

4. **验证调控特性**：
   - 使用坏死抑素-1（necrostatin-1, Nec-1）处理，观察细胞死亡是否被特异性抑制；
   - 确认凋亡抑制剂（如Z-VAD-FMK）无效，以排除凋亡干扰。

5. **得出结论**：
   - 若上述特征一致，提示存在程序性坏死机制，建议进一步检测RIP1/RIP3表达水平或开展Nec-1干预实验以确认坏死性凋亡。

> **注意**：本技能不适用于caspase依赖的经典凋亡或物理/化学导致的意外坏死。
