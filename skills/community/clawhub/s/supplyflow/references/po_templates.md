# 采购订单模板

## 标准采购订单 (PO)

包含字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| PO编号 | 唯一标识 | PO-20250315-0001 |
| 日期 | 下单日期 | 2025-03-15 |
| 供应商 | 供应商全称 | 上海宏达五金有限公司 |
| 采购方 | 买方名称 | XX制造有限公司 |
| 行项目 | 物料明细 | 见下方 |
| 付款方式 | 结算方式 | 月结30天 |
| 交货地址 | 收货地址 | XX省XX市XX区XX路XX号 |
| 备注 | 特殊要求 | 需附材质证明 |

## 行项目字段

| 字段 | 说明 |
|------|------|
| 行号 | 1, 2, 3... |
| 物料编码 | 内部SKU编号 |
| 物料名称 | 品名规格 |
| 规格型号 | 技术参数 |
| 数量 | 订购数量 |
| 单位 | 个/套/kg/米 |
| 单价 | 含税/不含税 |
| 金额 | 数量×单价 |

## 特殊条款模板

### 质量条款
```
供方保证所供货物符合国家标准及双方约定的技术规格。
如质量不合格，供方应在收到通知后7个工作日内无条件退换，
并承担由此产生的全部费用。
```

### 交期条款
```
供方应在收到订单后 [X] 个工作日内交货至指定地点。
逾期交货每延误一日，供方应支付当批货款千分之三的违约金。
```

### 付款条款
```
验收合格后30天内支付当期货款。
供方需提供合法有效的增值税专用发票。
```

## 使用 supplyflow 脚本生成

```bash
python3 scripts/purchase_order.py \
  --supplier "宏达五金" \
  --items '[{"name":"螺丝M6","qty":10000,"price":0.12,"spec":"GB/T 5783"}]' \
  --buyer "XX制造有限公司" \
  --date "2025-03-15"
```
