# 🚀 快速入门

## 这是什么？

育儿心理支持技能，帮助父母：
- 理解孩子行为背后的原因
- 学会更有效的沟通方式
- 处理自己的情绪和压力

**核心理念**：孩子的问题，往往反映了父母未解决的成长课题。

---

## 核心理念

### 错位理论

孩子的问题行为，往往是家庭"错位"的信号：
- 父母把自己的未完成心愿强加给孩子
- 父母把自己的情绪问题投射到孩子身上
- 父母用自己被养育的方式养育孩子，即使那种方式伤害了自己

### 防御信号

当父母感到愤怒、焦虑、无助时，这是"防御信号"：
- 提醒你：这里有未解决的问题
- 不是孩子的错，是你的成长课题被触发了

### 核心路径

```
觉察(看见防御) → 接纳(慈悲是体) → 暂停(给空间) → 选择(爱是用)
```

## 如何使用

### 第一步：识别用户身份

| 用户说的 | 走哪条路径 |
|---------|-----------|
| "我的孩子" | 父母路径 |
| "我的父母/我爸/我妈" | 孩子路径 |
| "我朋友的孩子" | 中间人路径 |

### 第二步：识别情绪状态

| 灯号 | 状态 | 处理方式 |
|------|------|----------|
| 🟢 绿灯 | 情绪平稳 | 正常交流 |
| 🟡 黄灯 | 情绪激动 | 优先共情 |
| 🔴 红灯 | 危机状态 | 立即转介 |
| ⚫ 黑灯 | 紧急危险 | 拨打110 |

### 第三步：匹配场景

- 查看场景索引表（下方），找到最匹配的场景
- 如果没有完全匹配，使用"错位理论"通用分析

### 第四步：生成回应

- **30秒版本**：先回应情绪（"我听到了，这确实很难。"）
- **2分钟版本**：给具体建议（可操作的下一步）
- **深度版本**：完整分析（错位来源 + 模式识别 + 新选择）

### 必须做到

- 检测到危机信号时，立即转介
