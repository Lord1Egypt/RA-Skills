# Referral Engine Quality Checklist

## Program Economics
- [ ] AOV and gross margin % confirmed before setting reward values
- [ ] Reward cost per acquisition calculated and below 25% of GM per order
- [ ] Target referral CAC is at least 40% lower than current paid CAC
- [ ] Double-sided reward structure designed (both referrer and referee benefit)
- [ ] Reward value feels meaningful relative to AOV (not insulting)

## Sharing Experience
- [ ] Unique referral links generated automatically per customer
- [ ] Pre-written share message drafted (not left blank for customer to write)
- [ ] WhatsApp and SMS share buttons enabled (highest conversion channels)
- [ ] Share message names the specific offer the friend receives
- [ ] Visual referral card created showing the offer clearly
- [ ] 1-click sharing from multiple entry points

## Program Visibility
- [ ] Referral CTA on order confirmation page
- [ ] Day 7–14 post-delivery email trigger configured
- [ ] My Account → Referrals page exists and is in navigation
- [ ] Announcement email drafted for launch

## Technical Setup
- [ ] Referral tracking tool integrated with store
- [ ] UTM parameters configured for referral traffic
- [ ] Reward issuance automated (no manual steps after referred order ships)
- [ ] Test referral completed end-to-end (link → order → reward confirmation)
- [ ] Email notification sent to referrer when friend orders

## Fraud Prevention
- [ ] Reward activates on shipped order only (not sign-up or cart)
- [ ] Self-referral email domain check enabled
- [ ] IP deduplication active
- [ ] Minimum order threshold set ($25–$50)
- [ ] Payout delay matches return window
- [ ] Disposable email domain blocklist added

## Measurement
- [ ] Share rate baseline target set (aim for >5% of eligible customers sharing)
- [ ] Referee conversion rate target set (aim for >15%)
- [ ] Referral CAC tracking configured
- [ ] Referred customer cohort tagged for CLV comparison
- [ ] Monthly review cadence scheduled

## Launch Readiness
- [ ] Soft launch to 100–200 top customers completed and mechanics validated
- [ ] Reward delivery confirmed (credit arrives in account as expected)
- [ ] Customer support briefed on program mechanics and FAQ
- [ ] Fraud controls tested (self-referral attempt blocked)
- [ ] Full launch email scheduled
