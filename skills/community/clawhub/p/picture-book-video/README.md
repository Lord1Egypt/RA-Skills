# 绘本故事视频 (Picture Book Video)

将绘本故事脚本自动生成为中英文双语视频。

## 输入
- 合集名称（可选）
- 合集描述（可选）
- 序列号（可选，如 S02E01）
- 故事脚本（必需）

## 输出
- 视频 mp4（中文 + 英文两个版本）
- 抖音发布描述（标题/话题/简介）

## 管线
1. 脚本解析 → 分镜方案确认
2. ComfyUI 生成分镜图片
3. 静态画面 + 封面（含琪琪角色）
4. 串联 + 溶解过渡
5. TTS 旁白（中英文，-15% 语速）
6. ASS 字幕（80pt，自动换行）
7. 最终合成

## 依赖
- ComfyUI + Flux（文生图）
- Edge TTS（旁白）
- ffmpeg（视频合成）
- 琪琪角色图（`references/qiqi_default.png`）
