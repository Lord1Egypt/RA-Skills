# 儿科腹部MRI适用范围与执行指南

## 何时使用
- 需要无放射性、高软组织对比度的影像评估儿童腹部器官（肝脏、胰腺、胆管、小肠）
- 怀疑以下病变：肝脏肿瘤（如血管瘤、囊肿）、局限性脂肪浸润、胰腺囊性纤维化伴脂肪沉积或囊肿、胆胰管异常、肠壁或肠周炎症/纤维化
- 患儿可配合保持静止，或可接受镇静
- 需重复影像随访（因MRI无电离辐射，适合儿童多次检查）

## 如何执行
1. **确认适用前提**：
   - 排除体内金属植入物
   - 确认患儿能配合检查体位；若为婴幼儿且无法静止，需安排镇静

2. **根据临床问题选择MRI技术**：
   - **常规MRI**：用于评估肝脏、胰腺实质病变（如肿瘤、脂肪浸润）
   - **MRI小肠造影（MR enterography, MRE）**：用于评估小肠腔内及肠壁外病变（如克罗恩病、狭窄、瘘管）
   - **MRI胆胰管造影（MRCP）**：用于无创显示胆管和胰管结构（如胆道闭锁、胰胆管扩张）
   - **磁共振血管造影（MRA）**：用于清晰显示肝脏血管解剖及病变（如门静脉畸形、肝动脉异常）

3. **注意局限性**：
   - MRI对钙化灶和气体显示不如CT，若临床高度怀疑钙化（如某些肿瘤或感染后钙化），应考虑补充CT
   - 检查时间较长，需提前做好患儿准备或镇静安排

4. **输出结果用途**：
   - 提供高分辨率软组织影像，明确病变性质、范围及其与周围结构关系
   - 支持诊断决策与治疗规划（如手术、介入或药物治疗）
