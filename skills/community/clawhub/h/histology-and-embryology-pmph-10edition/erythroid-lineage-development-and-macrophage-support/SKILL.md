# 红细胞系发育时序与巨噬细胞支持机制

## 何时使用
- 研究红细胞生成（erythropoiesis）全过程
- 评估骨髓红系造血功能是否正常
- 分析红细胞岛（erythroblastic island）中巨噬细胞的作用

## 执行步骤
1. **确认前提条件**：
   - 存在完整的骨髓红系造血微环境
   - 巨噬细胞功能正常
2. **追踪发育阶段与时程**：
   - 原红 → 早幼红 → 中幼红 → 晚幼红：全程约3–4天
   - 晚幼红脱核形成网织红细胞，入血后1–2天成熟为红细胞
3. **识别关键形态标志**：
   - 血红蛋白表达：早幼红“开始出现” → 中幼红“增多” → 晚幼红“大量”
   - 分裂能力：原红和早幼红有 → 中幼红弱 → 晚幼红及以后无
4. **描述巨噬细胞支持作用**：
   - 吞噬晚幼红细胞脱出的细胞核
   - 提供铁离子、维生素B12、叶酸等用于血红蛋白合成
   - 构成红细胞岛中心，周围环绕发育中红系细胞

> 注意：本技能仅适用于生理性红细胞生成，不适用于溶血性贫血等病理加速状态。
