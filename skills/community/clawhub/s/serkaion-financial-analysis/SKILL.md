# Financial Analysis

Institutional-grade analysis for raises, valuations, and investor materials. Work only from figures that are provided, public, or project-level — never assume confidential numbers; ask for them or state them as inputs.

## Capabilities

### 1. DCF Valuation
- Project unlevered free cash flow (revenue → EBITDA → FCF) over 5–7 years from stated assumptions.
- Discount at WACC — show the build (cost of equity via CAPM + after-tax cost of debt, weighted).
- Terminal value via Gordon growth **and** exit-multiple; cross-check both.
- Output: enterprise value, equity value, and a sensitivity table (WACC × growth/exit-multiple).

### 2. Comparable Company / Transaction Analysis
- Identify 5–8 true comparables (sector, stage, geography).
- Multiples: EV/Revenue, EV/EBITDA; for property/hospitality use cap-rate/yield and EV/key.
- Apply to target; show implied valuation range and where the target sits (premium/discount + why).

### 3. 3-Statement Model
- Linked IS → BS → CF. Verify the BS balances and cash ties out. Drivers explicit and editable; no hardcoded plugs.

### 4. Returns Analysis (investor-facing)
- IRR, MOIC, cash-on-cash, payback — by scenario (base / bull / bear).
- Property/hospitality: stabilised NOI, cap rate, DSCR, exit value, equity multiple.

### 5. Investor Data-Room Pack
- Assemble: one-pager, model summary, returns table, risk register, use-of-funds, structure/terms.
- Match the live deal terms exactly as provided.

## Output Rules
- State every assumption; flag provided vs. estimated.
- Always give a **range + sensitivity** — never single false-precision numbers.
- Sanity-check against comparables and reality; flag anything that looks off.
- Show the working so the numbers can be audited.
