## 六大类型详细说明

---

## 类型1：行情查询

### 识别关键词
- "当前价格"、"现在多少钱"
- "行情"、"实时行情"
- "开盘价"、"最高价"、"最低价"
- "成交量"、"持仓量"
- "买一"、"卖一"

### 相关变量
| 变量 | 说明 | 示例 |
|------|------|------|
| CLOSE | 当前收盘价 | `CLOSE > 5000` |
| OPEN | 当前开盘价 | `REF(OPEN,1)` |
| HIGH | 当前最高价 | `HIGH > REF(HIGH,1)` |
| LOW | 当前最低价 | `LOW < REF(LOW,1)` |
| VOL | 成交量 | `VOL > 10000` |
| OI | 持仓量 | `OI > 50000` |

### 典型问题
```
Q: 螺纹钢现在多少钱？
Q: 沪铜今天的成交量是多少？
Q: 帮我查一下IF300的行情
```

### 回答要点
- 直接提供价格或数据获取方式
- 说明数据来源和更新频率

---

## 类型2：历史数据

### 识别关键词
- "历史K线"
- "收盘价序列"
- "过去N天的数据"
- "补充数据"
- "数据从哪里开始"

### 相关变量
| 变量 | 说明 | 示例 |
|------|------|------|
| CLOSE | 收盘价序列 | `MA(CLOSE,20)` |
| HIGH | 最高价序列 | `HHV(HIGH,10)` |
| LOW | 最低价序列 | `LLV(LOW,10)` |
| OPEN | 开盘价序列 | `REF(OPEN,5)` |
| VOL | 成交量序列 | `MA(VOL,5)` |

### 典型问题
```
Q: 如何获取过去一年的日线数据？
Q: 怎么写一个基于历史收盘价的策略？
Q: 数据不够怎么补充？
```

### 回答要点
- 说明数据范围设置方法
- 介绍K线周期选择
- 提示信号计算起始时间

---

## 类型3：账户查询

### 识别关键词
- "账户资金"
- "当前权益"
- "持仓盈亏"
- "浮动盈亏"
- "平仓盈亏"
- "保证金占用"

### 相关变量
| 变量 | 说明 | 示例 |
|------|------|------|
| MONEYTOT | 当前权益 | `MONEYTOT <= INITMONEY*0.9` |
| INITMONEY | 初始资金 | `INITMONEY + 10000` |
| BKVOL | 多头持仓手数 | `BKVOL > 0` |
| SKVOL | 空头持仓手数 | `SKVOL > 0` |
| BKPRICE | 多头开仓价 | `CLOSE < BKPRICE - 10` |
| SKPRICE | 空头开仓价 | `CLOSE > SKPRICE + 10` |

### 典型问题
```
Q: 怎么查询当前的账户权益？
Q: 如何根据权益变化做风控？
Q: 持仓盈亏怎么计算？
```

### 回答要点
- 区分理论持仓和实际持仓
- 说明权益计算方式
- 介绍资金管理函数

---

## 类型4：下单交易

### 识别关键词
- "买开"、"卖开"
- "买平"、"卖平"
- "BK"、"SK"、"BP"、"SP"
- "做多"、"做空"
- "开仓"、"平仓"

### 信号指令
| 信号 | 含义 | 语法 |
|------|------|------|
| BK | 买开 | `条件, BK;` |
| SK | 卖开 | `条件, SK;` |
| BP | 买平 | `条件, BP;` |
| SP | 卖平 | `条件, SP;` |
| BPK | 反手买 | `条件, BPK;` |
| SPK | 反手卖 | `条件, SPK;` |

### 加减仓指令
| 指令 | 含义 | 语法 |
|------|------|------|
| ADD_LONG | 多头加仓 | `条件, ADD_LONG(N);` |
| LOWER_LONG | 多头减仓 | `条件, LOWER_LONG(N);` |
| ADD_SHORT | 空头加仓 | `条件, ADD_SHORT(N);` |
| LOWER_SHORT | 空头减仓 | `条件, LOWER_SHORT(N);` |

### 典型问题
```
Q: 怎么写一个均线金叉买开的模型？
Q: 止损怎么写？亏损N跳止损
Q: 如何实现盈利加仓？
```

### 回答要点
- 说明信号必须带条件
- 说明加减仓必须带手数
- 强调AUTOFILTER的必要性

---

## 类型5：模拟运行

### 识别关键词
- "模拟交易"
- "页面盒子"
- "信号监测"
- "实盘运行"
- "自动下单"

### 使用场景
- 多周期多合约信号监测
- 策略信号验证
- 自动下单测试（无需真金白银）

### 页面盒子限制
| 支持 | 不支持 |
|------|--------|
| 信号语句 | 运行优化函数（SIGCHECK） |
| 基础模型 | 资金管理函数 |
| | 指定交易合约（TRADE_OTHER） |
| | 头寸管理函数 |

### 典型问题
```
Q: 怎么用页面盒子监测信号？
Q: 页面盒子支持哪些函数？
Q: 如何设置自动下单？
```

### 回答要点
- 说明页面盒子使用方法
- 列出不支持的函数
- 介绍信号预警配置

---

## 类型6：回测分析

### 识别关键词
- "回测"
- "测试收益"
- "夏普比率"
- "胜率"、"盈亏比"
- "最大回撤"
- "敏感性测试"

### 相关设置
| 项目 | 说明 |
|------|------|
| 初始资金 | 回测起始资金 |
| 保证金率 | 合约保证金比例 |
| 手续费率 | 交易手续费 |
| 信号计算起始时间 | 从何时开始计算信号 |

### 回测报告指标
| 指标 | 说明 |
|------|------|
| 收益率 | 总收益/本金 |
| 夏普比率 | (年化收益-无风险利率)/收益标准差 |
| 权益最大回撤 | 权益从高点到低点的最大跌幅 |
| 胜率 | 盈利次数/总交易次数 |
| 盈亏比 | 平均盈利/平均亏损 |

### 典型问题
```
Q: 怎么设置回测参数？
Q: 怎么解读回测报告？
Q: 如何做敏感性测试？
```

### 回答要点
- 说明回测参数配置
- 解读关键指标含义
- 介绍参数优化方法

---

## 分类准确度检查

### 易混淆类型

#### 行情 vs 历史数据
- 行情 = 当前这一刻的数据
- 历史数据 = 过去一段时间的数据序列

#### 账户查询 vs 下单交易
- 账户查询 = 读取数据（MONEYTOT、BKVOL）
- 下单交易 = 发送指令（BK、SP）

#### 回测 vs 模拟
- 回测 = 基于历史数据测试
- 模拟 = 实时信号监测和下单
