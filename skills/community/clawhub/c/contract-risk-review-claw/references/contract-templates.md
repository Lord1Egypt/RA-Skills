# 合同模板库

各类标准合同的关键条款要点，用于对比识别被修改或缺失的重要条款。

## 采购合同关键条款

### 必备条款清单
- [ ] 货物名称、规格、数量、单价、总价
- [ ] 交货时间、地点、方式
- [ ] 质量标准（引用具体标准编号）
- [ ] 验收方式、验收期限（建议 5-10 个工作日）
- [ ] 付款方式、付款时间（到货验收后 X 天内）
- [ ] 质保期（建议 12 个月以上）
- [ ] 违约责任（双方对等）
- [ ] 争议解决（仲裁或诉讼，明确管辖地）

### 风险关注点
- 价格调整条款：是否允许单方调价？
- 交货延迟：罚款比例是否合理？
- 质量问题：退换货流程是否明确？
- 知识产权：货物是否侵犯第三方 IP？

---

## 销售合同关键条款

### 必备条款清单
- [ ] 产品描述、规格、数量
- [ ] 价格及税费承担
- [ ] 交付条件（FOB/CIF 等，或具体交付地点）
- [ ] 所有权转移时间
- [ ] 付款条件（预付/货到付款/账期）
- [ ] 退货政策
- [ ] 保修条款
- [ ] 限制责任条款（合理范围内）

### 风险关注点
- 买方拒绝付款的条件是否过宽？
- 退货条件是否对卖方不利？
- 产品责任范围是否明确？

---

## 服务合同关键条款

### 必备条款清单
- [ ] 服务范围（详细描述，避免"等"字）
- [ ] 服务期限
- [ ] 服务标准/KPI（可量化）
- [ ] 交付物清单
- [ ] 费用及支付里程碑
- [ ] 变更管理流程（范围变更须书面确认）
- [ ] 知识产权归属（背景 IP vs 前景 IP）
- [ ] 保密条款
- [ ] 人员要求（关键人员替换须提前通知）
- [ ] 违约责任

### 风险关注点
- 服务范围是否有"兜底条款"（如"及其他相关工作"）？
- 验收标准是否客观可量化？
- 知识产权归属是否明确？
- 是否有竞业限制？补偿是否合理？

---

## 劳动合同关键条款

### 必备条款清单（依据《劳动合同法》第 17 条）
- [ ] 用人单位基本信息
- [ ] 劳动者基本信息
- [ ] 合同期限
- [ ] 工作内容和工作地点
- [ ] 工作时间和休息休假
- [ ] 劳动报酬
- [ ] 社会保险
- [ ] 劳动保护、劳动条件和职业危害防护

### 风险关注点
- 试用期是否超过法定上限？
- 竞业限制补偿是否符合法定标准（不低于月工资 30%）？
- 服务期条款是否合理（专项培训费用须明确）？

---

## 保密协议（NDA）关键条款

### 必备条款清单
- [ ] 保密信息定义（明确范围，避免过宽）
- [ ] 保密期限
- [ ] 保密义务（接收方义务）
- [ ] 例外情形（已公开信息、独立开发等）
- [ ] 信息返还/销毁义务
- [ ] 违约责任

### 风险关注点
- 保密信息定义是否过宽（如"所有接触到的信息"）？
- 保密期限是否合理（一般 2-5 年，永久保密需谨慎）？
- 是否有单向/双向之分？
- 违约金是否合理？

---

## 技术开发合同关键条款

### 必备条款清单
- [ ] 研发目标和技术指标
- [ ] 研发期限和里程碑
- [ ] 技术成果归属（委托开发 vs 合作开发）
- [ ] 专利申请权归属
- [ ] 技术秘密保护
- [ ] 验收标准
- [ ] 风险承担（研发失败如何处理）

### 风险关注点
- 委托开发：成果归委托方，但须明确背景技术使用权
- 合作开发：成果共有，须明确各方使用权和转让权
- 研发失败：是否约定了风险分担机制？
