# Herb Processing Five Methods

根据炮制目的（如减毒、便于切制、增强疗效等），选择并执行以下五类炮制方法之一或组合：

## 1. 修制法
适用于初步处理已采收的中药材，为后续炮制做准备。
- **纯净**：去除杂质及非药用部位。
- **粉碎**：将药材破碎至适宜粒度。
- **切制**：切成片、段、块等形状。

## 2. 水制法
用于清洁、降毒或软化药材以便切制。
- **漂洗**：反复换水浸泡清洗。
- **闷润**：喷水后密闭使药材软化。
- **浸泡**：长时间浸于水中。
- **喷洒**：表面喷水湿润。
- **水飞**：矿物药加水研磨成极细粉末。

## 3. 火制法
应用最广，通过加热改变药性或降低毒性。
- 包括炒、炙、烫、煅、炮、燎、烘等方法，根据温度、时间及辅料调整工艺。

## 4. 水火共制法
同时使用水与火进行处理。
- 基本方法包括蒸、煮、淖（焯）、淬、炖。

## 5. 其他制法
针对特殊药材或目的的处理方式。
- **制霜**（如巴豆霜）
- **发酵**（如神曲）
- **发芽**（如麦芽）
- **药拌**（与其他药物混合处理）

## Precautions
- **液体辅料**：酒、醋、蜂蜜、姜汁、甘草汁等，用于引药归经或减毒。
- **固体辅料**：白矾、食盐、麦麸、蛤粉、朱砂等，用于协同增效或改变药性。

> **注意**：本技能不适用于已制成中成药的成品；部分方法需特定辅料，应根据炮制目的选择。
