# 药物基因组学指导下的个体化用药

## 触发条件
当满足以下任一条件时启用本技能：
- 药物经CYP2D6、CYP2C19、NAT2等具有已知功能多态性的酶代谢
- 药物作用于具有多态性的转运体（如ABCB1）、受体（如β2肾上腺素受体、5-HT2A）或靶点（如ACE）
- 患者具备可检测的基因型数据
- 药物为相关基因产物的底物或配体

## 执行流程
1. **确认适用对象**：验证药物是否通过具有已知多态性的代谢酶、转运体、受体或靶点发挥作用。
2. **获取基因型信息**：明确患者在相关基因位点的基因型（如CYP2C19*1/*2）。
3. **推断代谢/功能表型**：根据基因型判断表型（如慢代谢者、超快代谢者）。
4. **结合种族背景**：参考不同人群的等位基因频率差异（如中国人群CYP2C19慢代谢率高于白种人）。
5. **制定用药建议**：依据表型调整药物选择或剂量，优先选用非依赖该多态性通路的替代药物（如CYP2C19慢代谢者避免使用奥美拉唑，改用雷贝拉唑）。

## 禁忌情形
- 药物反应主要由环境因素引起，无明确遗传基础
- 相关药物通路无已知功能性多态性
