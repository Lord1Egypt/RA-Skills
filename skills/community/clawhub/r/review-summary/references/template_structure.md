# 评审汇总表模板结构说明

## 四张表概览

| Sheet | 名称 | 用途 |
|-------|------|------|
| 1 | 评审底稿 | 封面 — 项目信息、送审/审定金额汇总、签名栏 |
| 2 | 汇总表 | 按单位工程×7大类别展开的审增减汇总矩阵 |
| 3 | 导表填写 | 逐条清单的审增减明细（数据源→分类映射） |
| 4 | 审增减明细生成表 | 按类别归集的审增减清单明细（正式输出） |

## Sheet 2: 汇总表 结构

```
行1: 标题 + 总送审金额
行2: 表头 (序号|名称|送审金额|审定金额|审增减金额|审增减率|建筑面积|单方指标|备注|分配责任人|互审人|索引列)
行3: 合计行 (一|电力工程合计|...)
行4+: 单位工程行 (每个单位工程占8行)
  - 主行: 序号=N.0, 送审/审定/审增减金额
  - 子行1: N.1 因重复计取审减X万元
  - 子行2: N.2 因漏项审增X万元
  - 子行3: N.3 因工程量审减X万元
  - 子行4: N.4 因定额子目审减X万元
  - 子行5: N.5 因材料价格审减X万元
  - 子行6: N.6 其他项审减X万元
  - 子行7: N.7 因基数调整审减X万元
```

## 7大审增减类别

| ID | 名称 | 判断依据 |
|----|------|---------|
| 1 | 重复计取审减 | 同一工作内容在多个清单项中重复计算 |
| 2 | 漏项审增 | 图纸/规范要求有但送审清单未列的工作 |
| 3 | 工程量审增减 | 工程量计算错误或多算/少算 |
| 4 | 定额子目审增减 | 定额子目套用错误，应套A实际套B |
| 5 | 材料价格审增减 | 材料/设备单价与市场价或信息价偏差 |
| 6 | 其他 | 无法归入上述5类的调整 |
| 7 | 基数调整审增减 | 措施费、税金等随计费基数变化而调整 |

## 数据来源文件

### 1. 分部分项汇总审核导表.xlsx
- Sheet "总表"
- 行1: 标题（含单位工程名称）
- 行2-3: 分部汇总行
- 行4+: 清单明细行
- 列结构: 序号|单位工程|项目编码|项目名称|特征描述|单位|送审工程量|送审综合单价|送审综合合价|审定工程量|审定综合单价|审定综合合价|增减金额|增减说明|增/减符|增减分析|增减原因

### 2. 总审核导表.xlsx
- Sheet "【项1】工程审核汇总对比表"
- 列: 序号|项目名称|报审金额|审定金额|调整金额

### 3. 材料汇总审核导表.xlsx
- Sheet "总表"
- 列: 单位工程|序号|材料编码|项目名称|规格型号|单位|送审(...)|审定(...)|增减|增减说明
- 用于材料价格审减的详细依据

## 分类规则优先级

脚本按以下顺序匹配关键字进行分类：

1. **基数调整**：包含"基数|税金|措施费|总价措施|安文|环境保护费|文明施工费|临时设施|扬尘|夜间施工|二次搬运|冬雨季"
2. **重复计取**：包含"重复"
3. **定额子目**：包含"定额|子目|套用|调整为|人工乘"
4. **材料价格**：包含"价格|材料|信息价|市场询价"
5. **漏项**：增减分析=[增项] 或包含"漏项|补项|未报送"
6. **工程量**：增减分析=[调量] 或 [调量,调价]
7. **[调价]（纯单价调整）**：默认归入材料价格
8. **兜底**：其他
