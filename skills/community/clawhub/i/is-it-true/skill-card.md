## Description: <br>
Conduct critical analysis of viewpoints, factual statements, or web content using the critical thinking framework from "Asking the Right Questions". <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[chenyichu86](https://clawhub.ai/user/chenyichu86) <br>

### License/Terms of Use: <br>

## Use Case: <br>
Employees, external users, and developers can use this skill to fact-check claims, separate facts from opinions, and examine argument quality in statements, articles, or web content. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Fact-checking may require web searches or source tracing that could expose confidential user-provided material. <br>
Mitigation: Avoid submitting confidential material unless the user is comfortable with it being used for verification searches. <br>
Risk: The included packaging helper can create distributable archives from skill directories. <br>
Mitigation: Run the packaging script only on skill directories intended for packaging and review the package contents before distribution. <br>
Risk: The skill can produce misleading confidence if source evidence is incomplete, stale, or unavailable. <br>
Mitigation: Use the skill's downgrade rules, uncertainty labels, and multi-source validation checklist before relying on a verdict. <br>
