# 房间隔缺损分流机制与肺动脉高压发展

1. 出生初期：左、右心室壁厚度相似，顺应性相近 → 左向右分流量不多。
2. 随年龄增长：肺血管阻力下降，右心室壁变薄，右心室充盈阻力低于左心室 → 心室舒张期左心房血液经缺损向右心房分流（左向右分流）。
3. 分流后果：右心血量增加 → 右心房、右心室增大；肺循环血量增加 → 早期引起动力学性肺动脉压力增高。
4. 晚期演变：持续肺血增多 → 肺小动脉肌层及内膜增厚、管腔狭窄 → 梗阻性肺动脉高压。
5. 肺动脉高压进展至右心室收缩压 ≥ 左心室收缩压时 → 左向右分流减少，甚至逆转为右向左分流（出现青紫）。
6. 关键影响因素包括：缺损大小、心房压力差、心室顺应性（尤其右心室充盈更容易是分流驱动力）。

**变量说明**：
- `shunt_volume`：左向右分流量，受缺损大小和心室顺应性影响
- `pulmonary_vascular_resistance`：肺血管阻力，随年龄增长而下降
- `ventricular_compliance`：心室顺应性，右心室通常低于左心室（充盈更易）
