## Description: <br>
Multi-source deep research agent using MCP search tools (minimax web_search or zai-mcp-server web_search_prime). <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wanghan0501](https://clawhub.ai/user/wanghan0501) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill to plan, execute, and synthesize multi-source web research into cited reports. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill uses configured MCP search providers and web requests for research topics. <br>
Mitigation: Install and run it only with trusted, approved MCP search providers, and avoid confidential topics unless those providers are approved. <br>
Risk: The skill can save generated research reports in shared workspaces. <br>
Mitigation: Review saved reports before sharing, publishing, or using them in shared environments. <br>
