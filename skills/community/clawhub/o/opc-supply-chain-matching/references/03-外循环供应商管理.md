# 外循环供应商管理

## 1. 供应商分类体系

### 1.1 九大环节分类

| 环节 | 涵盖范围 | 典型供应商 | 适用阶段 |
|------|---------|-----------|---------|
| **ID设计** | 外观造型、CMF设计、人机工程 | 工业设计公司、独立设计师 | L1-L2 |
| **结构设计** | 结构工程、堆叠设计、模具评估 | 结构工程师、ID工作室 | L2-L3 |
| **手板制作** | 3D打印、CNC手板、硅胶复模、钣金 | 手板厂、模型工作室 | L2-L3 |
| **模具制造** | 注塑模具、冲压模具、压铸模具、硅胶模 | 模具厂、模具服务商 | L3-L4 |
| **PCB制造** | PCB制板、FPC软板、多层板 | PCB厂商 | L2-L4 |
| **整机组装** | SMT贴片、DIP焊接、装配测试 | 组装厂、EMS厂商 | L3-L5 |
| **认证检测** | 安规认证、EMC测试、性能测试、可靠性测试 | 检测机构、认证咨询公司 | L4-L5 |
| **包装供应** | 包装设计、包材生产、说明书印刷 | 包装厂、包材供应商 | L4-L5 |
| **量产代工** | 大批量生产、OEM/ODM、产能外包 | 代工厂、品牌方合作 | L5 |

### 1.2 供应商画像要素

| 要素类别 | 具体内容 |
|---------|---------|
| **基本信息** | 公司名称、地址、联系方式、成立时间 |
| **能力信息** | 设备清单、技术团队、产能规模 |
| **资质信息** | 营业执照、行业资质、质量体系认证 |
| **合作信息** | 合作历史、付款条件、最小起订量 |
| **评估信息** | 交付质量、交期保障、配合度评分 |

## 2. 供应商准入标准

### 2.1 准入审核清单

| 审核维度 | 审核内容 | 通过标准 | 审核方法 |
|---------|---------|---------|---------|
| **资质合规** | 营业执照、行业许可证 | 证照齐全、合法经营 | 证件核查 |
| **能力验证** | 生产设备、技术团队 | 设备与描述匹配，团队专业 | 实地考察或视频验证 |
| **质量体系** | 质量管控文件、检验流程 | 有基本质量文件 | 文件审核 |
| **配合意愿** | 响应速度、沟通态度 | 3个工作日内响应 | 沟通测试 |
| **价格水平** | 报价单、市场比价 | 不高于市场均值20% | 比价分析 |
| **信用背调** | 工商信息、司法风险、舆情 | 无重大风险记录 | 第三方查询 |

### 2.2 准入流程

```
Step 1: 供应商申请
├─ 提交资质证明
├─ 填写能力问卷
└─ 提供样品或案例

Step 2: 资质审核
├─ 核对证照真实性
├─ 核查经营状态
└─ 查询信用记录

Step 3: 能力验证
├─ 审核设备清单
├─ 评估技术能力
└─ 必要时实地考察

Step 4: 试单合作
├─ 小批量试单
├─ 验证交付质量
└─ 评估配合表现

Step 5: 正式入库
├─ 综合评审通过
├─ 签署框架协议
└─ 进入供应商库
```

## 3. 供应商评估体系

### 3.1 四维评估模型

| 评估维度 | 核心指标 | 计算方法 | 权重 |
|---------|---------|---------|------|
| **交付质量** | 产品合格率、返工率、客户投诉率 | 批次抽检统计 | 35% |
| **成本控制** | 报价竞争力、性价比、降价配合度 | 多供应商比价 | 25% |
| **交期保障** | 准时交货率、平均交期波动、延期频率 | 交期记录统计 | 25% |
| **配合度** | 响应速度、问题解决效率、沟通满意度 | 沟通记录评价 | 15% |

### 3.2 评分计算

```
综合评分 = 交付质量得分×35% + 成本控制得分×25% + 交期保障得分×25% + 配合度得分×15%
```

| 评分等级 | 分数区间 | 供应商标签 |
|---------|---------|-----------|
| 优秀 | 90-100分 | 🟢 可用（推荐） |
| 良好 | 75-89分 | 🟢 可用 |
| 一般 | 60-74分 | 🟡 慎用 |
| 较差 | 45-59分 | 🔴 淘汰（观察） |
| 差 | <45分 | 🔴 淘汰 |

### 3.3 评估周期

| 评估类型 | 评估周期 | 触发条件 |
|---------|---------|---------|
| **常规评估** | 每季度 | 自动触发 |
| **事件评估** | 事件发生后 | 重大质量问题、延期等 |
| **年度评估** | 每年12月 | 综合评审 |
| **实时更新** | 每次合作后 | 合作结束即评价 |

## 4. 供应商库管理

### 4.1 供应商生命周期

```
【入库】新供应商申请 → 资质审核 → 能力验证 → 试单合作 → 正式入库
    ↓
【在库】正常合作 → 定期评估 → 动态调整
    ↓
【出库】严重问题 → 淘汰处理 → 档案留存
```

### 4.2 供应商状态管理

| 状态 | 定义 | 标识 | 可用性 |
|------|------|------|--------|
| **新增** | 刚提交申请 | ⏳审核中 | 不可用 |
| **试单** | 试单合作中 | 🔵试单中 | 限制使用 |
| **正常** | 正式合作，正常评估 | 🟢正常 | 可用 |
| **观察** | 存在问题，需关注 | 🟡观察中 | 慎用 |
| **暂停** | 暂停合作，整改中 | 🟠暂停 | 不可用 |
| **淘汰** | 终止合作 | 🔴淘汰 | 不可用 |

### 4.3 供应商退出机制

**主动退出**：
- 供应商申请退出
- 合作期满未续约
- 业务调整，能力不再需要

**被动退出**：
- 重大质量问题
- 连续3次评估不合格
- 严重违反合作协议
- 信用风险暴露

## 5. 供应商关系维护

### 5.1 分级维护策略

| 供应商级别 | 维护频率 | 维护方式 | 责任人 |
|-----------|---------|---------|--------|
| **核心供应商** | 月度 | 高层拜访、联合规划 | 三长老 |
| **重要供应商** | 季度 | 业务对接、问题解决 | 渠道经理 |
| **一般供应商** | 年度 | 常规沟通 | 专员 |

### 5.2 供应商赋能

- **能力提升**：分享行业趋势、技术要求
- **需求预测**：提前告知备货需求
- **联合创新**：共同开发新工艺、新材料
- **信用背书**：优秀供应商向社区推荐
