# 儿童1型糖尿病的诊断与实验室检查

## 何时使用
- 患儿有多饮、多尿、多食、体重下降或酮症酸中毒表现
- 尿糖阳性伴或不伴尿酮体
- 需排除应激性高血糖、暂时性糖尿或其他还原糖尿

## 执行步骤
1. **初步筛查**：
   - 尿糖定性（葡萄糖氧化酶法）
   - 尿酮体检测（阳性提示酮症）

2. **确诊标准（满足任一即可）**：
   - 典型症状 + 任意时间血糖 ≥11.1 mmol/L
   - 空腹血糖（FPG）≥7.0 mmol/L
   - OGTT 2小时血糖 ≥11.1 mmol/L

3. **辅助评估**：
   - **HbA1c**：<7.5%为控制良好，>9%提示控制不佳
   - **血气分析**：pH <7.30 且 HCO₃⁻ <15 mmol/L 提示DKA
   - **血脂**：胆固醇、甘油三酯常升高

4. **执行OGTT（必要时）**：
   - 方法：口服葡萄糖1.75g/kg（最大75g）
   - 采血点：0、60、120、180分钟
   - 诊断阈值：120分钟血糖 >11.1 mmol/L

5. **鉴别诊断**：
   - 排除果糖尿、戊糖尿（使用葡萄糖氧化酶法）
   - 排除范科尼综合征（依赖空腹血糖/OGTT）
   - 排除婴儿暂时性糖尿（6周内自愈）
   - 排除应激性高血糖（感染、手术后一过性）

> 注意：OGTT前需停用影响糖代谢药物；班氏试剂易受其他还原糖干扰，应避免使用。
