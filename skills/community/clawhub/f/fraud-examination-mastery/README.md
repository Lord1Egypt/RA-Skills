# Fraud Examination Mastery — 全球舞弊审查专业实务体系

> ACFE 2021教材 + 2026全球职业舞弊报告 + 中国舞弊审计指南 + 创始人回忆录

## 概述

融合5部权威著作：
1. **ACFE Fraud Examiners Manual（2021版）** — CFE考试官方教材，2010页
2. **2026 Report to the Nations** — 全球最大职业舞弊研究，2402个真实案例
3. **2026全球职业舞弊报告-精华版** — 数据可视化PPT
4. **舞弊审计实务指南** — 中国视角863页实务操作指南
5. **Fraud Fighter: My Fables and Foibles** — ACFE创始人Joseph T. Wells回忆录

## 核心内容

- CFE四大专业领域（财务交易与舞弊方案/法律/调查/预防与威慑）
- 舞弊树三大分类完整解读
- 2026全球职业舞弊报告核心数据
- 舞弊审查调查全流程五阶段
- Benford定律等数据分析工具
- 反舞弊控制体系（18项控制措施效果数据）
- 中国舞弊审计法律框架与特色方法
- 加密货币与新兴舞弊风险
- CFE与CGMA能力映射
