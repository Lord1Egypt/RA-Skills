## Description: <br>
React composition patterns for scalable component architecture, including refactoring boolean prop proliferation, building flexible component libraries, designing reusable component APIs, and working with compound components and context providers. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to refactor React components away from boolean prop proliferation and toward compound components, explicit variants, lifted provider state, and children-based composition. <br>
