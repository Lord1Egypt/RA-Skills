## Use Case: <br>
Developers and engineers use this skill to write and maintain tests for PydanticAI agents, including deterministic model overrides, custom model behavior, recorded API interactions, inline snapshots, tool-call tests, mocked dependencies, and pytest configuration. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Refreshing VCR cassettes or inline snapshots can rewrite files and accidentally capture sensitive or unintended data. <br>
Mitigation: Run only scoped record or fix commands against intended tests, use test credentials where possible, rerun without update flags, and inspect diffs before committing. <br>
