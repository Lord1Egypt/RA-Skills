# 失血性休克的输血指征与方案

## 触发条件
当患者因大量失血出现低血容量性休克时，使用本技能进行输血决策。

## 执行步骤

1. **评估失血量占总血容量的比例**（成人总血容量按70 ml/kg估算，通常为4000–5000 ml）：
   - **<10%（<400–500 ml）**：机体可代偿，无需输血。
   - **10%～20%（400–800 ml）**：若出现活动时心率增快、直立性低血压但HCT未变，补充胶体液或血浆代用品。
   - **≥20%（≥800 ml）且HCT下降**：除扩容外，应输注浓缩红细胞（CRBC）以提升携氧能力。

2. **遵循输血原则**：
   - 失血量 **<30%** 总血容量：不输全血。
   - 失血量 **≥30%**：可输全血与CRBC各半，并联合晶体液、胶体液及血浆扩容。

3. **特殊情况处理**：
   - 失血量 **>50%** 且大量输入库存血时，监测白蛋白、血小板、凝血因子，并及时补充。

4. **依据血红蛋白（Hb）水平决策**：
   - **Hb > 100 g/L**：不必输血。
   - **Hb < 70 g/L**：可输浓缩红细胞。
   - **Hb 70–100 g/L**：结合出血是否停止、患者一般状况、代偿能力及重要器官功能综合判断。

## 注意事项
- 仅适用于失血性/低血容量性休克患者。
- 输血前必须已完成失血量、失血速度、临床表现、Hb和HCT的评估。
- 避免对非失血性休克（如心源性、感染性）应用本方案。
