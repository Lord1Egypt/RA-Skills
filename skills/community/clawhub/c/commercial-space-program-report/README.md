# commercial-space-program-report

Convert a client brief and functional requirements into a structured commercial space program report — the prerequisite design document before schematic floor plans begin.

## When to use this skill

Use this skill when an interior designer, architect, project manager, or owner's representative needs to document space requirements for a new commercial interior project, tenant improvement, or building renovation before any design drawings are produced.

## What it produces

- Organizational inventory table with headcount (current and projected) and activity notes
- Room-type catalog with net area assignments and stated benchmark basis (GSA, BOMA, ANSI/BIFMA, or client-stated)
- Program totals by category (enclosed private, open collaborative, support, amenity) with gross-to-net load factor
- Functional adjacency matrix scored Must-Adjacent through Must-Separate
- Special systems and finish-quality tier requirements per space type
- Open-items list and design-guide narrative for the design team
- DRAFT report with unsigned designer review block

## Inputs needed

| Input | Description |
|---|---|
| Project type | Office, healthcare outpatient, retail, hospitality, education, mixed-use |
| Gross building area | If known; otherwise TBD |
| Department or group inventory | List of organizational groups or functional areas |
| Headcount per group | Current and 3–5 year projected occupancy |
| Special requirements | Labs, secure access, clinical infrastructure, AV-intensive, server rooms |
| Area standard preference | GSA, BOMA, ANSI/BIFMA, or client-stated standard |

## Safety notes

- Output is a DRAFT space program — not a permit document, code-compliance determination, or final design specification.
- Net area assignments are benchmarked estimates; final compliance with local code, ADA/ABA requirements, and lease constraints requires review by a licensed architect.
- Program totals are net usable area only; gross area must account for circulation, structure, and mechanical based on actual building efficiency.
