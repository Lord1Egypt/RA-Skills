# 临床实例与pKa参考表

## 典型药物及pKa
| 药物类型 | 示例药物 | pKa |
|----------|----------|-----|
| 弱酸性 | 苯巴比妥 | 7.3 |
| 弱酸性 | 水杨酸 | 3.0 |
| 弱碱性 | 苯丙胺 | 9.8 |
| 弱碱性 | 奎宁 | 8.5 |

## 详细机制说明
肾小管上皮细胞对非解离型药物具有高通透性，可通过被动扩散被重吸收回血液；而解离型因极性高、脂溶性低，难以跨膜，故随尿液排出。根据Henderson-Hasselbalch方程：

- 弱酸性药物：pH = pKa + log([A⁻]/[HA]) → 尿液碱化使[A⁻]↑，重吸收↓
- 弱碱性药物：pH = pKa + log([B]/[BH⁺]) → 尿液酸化使[BH⁺]↑，重吸收↓

## 注意事项
- 碱化治疗禁用于心衰、肾功能不全患者
- 酸化治疗禁用于肝病、代谢性酸中毒患者
- 尿液pH调节仅对肾小管分泌后仍可被重吸收的药物有效
- 不适用于已高度蛋白结合或主动分泌为主的药物
