# thiazolidinediones-and-alpha-glucosidase-inhibitor-application

## 触发条件
- T2DM患者存在胰岛素抵抗 → 考虑TZDs；
- T2DM或T1DM患者以餐后高血糖为主 → 考虑AGIs。

## 执行步骤
### A. 噻唑烷二酮类（TZDs）应用
1. **确认适应症**：T2DM伴胰岛素抵抗。
2. **禁忌与慎用**：
   - 禁用于骨质疏松或非外伤性骨折史者；
   - 65岁以上老年慎用；
   - 罗格列酮需评估心血管风险。
3. **药理机制**：激活PPAR-γ，增强胰岛素敏感性，改善β细胞功能，增加GLUT-4表达。
4. **联合用药**：可与二甲双胍或磺酰脲类联用，效果更佳。
5. **监测重点**：水肿、体重增加、心血管事件、膀胱癌风险（吡格列酮）。

### B. α-葡萄糖苷酶抑制药（AGIs）应用
1. **确认适应症**：餐后高血糖为主，可单用或联用。
2. **禁忌**：胃肠功能紊乱、妊娠、哺乳期、儿童。
3. **作用机制**：抑制小肠α-葡萄糖苷酶，延缓碳水化合物吸收，降低餐后血糖峰值。
4. **低血糖处理**：若与胰岛素或磺酰脲类合用发生低血糖，**必须直接给予葡萄糖**（口服或静脉），因蔗糖或淀粉无法被有效分解。
5. **不良反应**：腹胀、腹痛、腹泻；通常随用药时间减轻。

## 输出结果
- TZDs：改善胰岛素抵抗，需长期监测安全性；
- AGIs：有效控制餐后血糖，注意低血糖纠正方式。
