# 中文小说写作原则

这是一套面向中文长篇连载的写作护栏，用于未来起草、修订和自检；不会自动要求改写已接受旧章节。

好文字至少有五层：**声音、层次、细节、呼吸、活性**。章节规划只选本章最关键的 2-3 层，质量门再检查是否命中。

## 1. 声音

**目标**：叙事者有立场，不做摄像头。

**检查点**：

- 同样是“看见一封信”，叙事是否带出人物的警觉、厌恶、欲望或误判。
- 场景描写是否服务视角人物的处境，而不是中性罗列。
- 每段是否有判断方向：危险、诱惑、讽刺、疼痛、戒备、压迫。

**常见失败**：

- 摄像头式记录：人物进门、看见、坐下、说话，只有动作流水账。
- 设定说明压过人物视角。
- 叙事声音忽冷忽热，像不同作者接力。

**修改动作**：

- 把中性句改成带判断的观察。
- 用人物关心的物件或风险重排描写顺序。
- 删除“她觉得很复杂”这类空泛心理，换成视线、停顿、手势或选择。

## 2. 层次

**目标**：情绪和信息有起伏，不是单一色调。

**检查点**：

- 一章内至少有一次压力升级或关系位置变化。
- 情绪不是一路甜、一路虐、一路打，而有喘息、误判、反压或反转。
- 明线事件和暗线关系能互相加压。

**常见失败**：

- 原地暧昧、原地吵架、原地解释。
- 每场戏功能相同，读者读完只记得“很紧张”或“很伤心”。
- 权谋线、感情线、动作线彼此脱节。

**修改动作**：

- 给每个场景标注“进入状态”和“离开状态”，删掉没有变化的场景。
- 在情绪高点后安排一个更冷的选择或更具体的外部后果。
- 让情感互动同时承担情报、伤病、身份、任务或权力压力。

## 3. 细节

**目标**：情绪外化，有画面、有身体、有物证。

**检查点**：

- 关键情绪是否落在动作、物件、触感、光影、气味、温度或停顿上。
- 细节是否推动人物选择或揭示关系，而非只为了漂亮。
- 物件是否能回环：前文出现，后文改变含义或成为证据。

**常见失败**：

- 抽象心理过多：“害怕”“难过”“动容”“复杂”。
- 连续堆相近意象，画面互相抵消。
- 感官描写和剧情无关，像贴上去的装饰。

**修改动作**：

- 把抽象心理换成身体反应、手上动作或物件变化。
- 同一情绪节点只保留最准确的一处意象。
- 让关键物件至少承担一种功能：证据、筹码、误会、承诺、危险或亲密边界。

## 4. 呼吸

**目标**：长短句有张弛，读者能跟着压力呼吸。

**检查点**：

- 对峙、危险、揭露处是否有短句收紧。
- 过渡、观察、情绪回潮处是否允许一句承重的长句。
- 段落长度是否服务节奏，而不是平均铺开。

**常见失败**：

- 全是短句，像切碎的提纲。
- 全是长句，动作被淹没。
- 句式重复，连续用同一开头或同一否定排比。

**修改动作**：

- 关键动作单独成句。
- 信息密集处拆句；情绪回旋处合并一部分句子。
- 删除机械排比，尤其是无意义的“没有……没有……也没有……”。

## 5. 活性

**目标**：动词有力，选择有后果，文字在动。

**检查点**：

- 人物是否用动作证明性格，而不是被贴标签。
- 谓语动词是否具体：压、扣、折、掩、递、收、避、逼近。
- 每场戏是否留下可追踪后果：信息改变、关系改变、物件易手、承诺成立、危险逼近。

**常见失败**：

- 用形容词替代行动：“善良”“冷酷”“深情”“复杂”。
- 人物只想不做，或只说不选。
- 场景结束后世界状态没有变化。

**修改动作**：

- 把性格判断改成一次具体选择。
- 把“他很强势”改为他如何阻断退路、改变站位或控制信息。
- 给场景末尾补一个后果：谁失去筹码、谁多知道一点、谁被迫留下痕迹。

## 扩展原则

### 双线推进

**目标**：一章不只推进单一情绪或单一事件，而让至少两条压力线互相牵动，例如关系线与权谋线、伤病线与身份线、动作线与秘密线。

**检查点**：

- 本章是否同时改变了外部局势和人物关系，或同时推进了秘密、伤病、承诺、物件、任务中的两项。
- 情感戏是否带出信息、风险或选择，而不是只负责暧昧。
- 权谋/案件/战斗场景是否改变人物之间的距离和信任，而不是只负责“办事”。

**常见失败**：

- 只有拉扯没有剧情，读完关系像更近了，但世界状态没变。
- 只有 plot，没有人物关系的压力变化。
- 每条线轮流独立出现，互不咬合。

**修改动作**：

- 给每个场景标注“外部推进”和“关系推进”，缺一项就补一个选择、信息差或物件变化。
- 让亲密动作承担任务或危险，让权谋动作反过来改变关系边界。
- 删掉只重复气氛、不改变任一压力线的段落。

### 对白潜台词

**目标**：台词表面谈一件事，暗里交换另一件事；读者能听见未说出口的威胁、试探、退路或在意。

**检查点**：

- 每句关键台词是否至少有一个隐藏目的：试探、遮掩、逼供、讨价、划线、放饵。
- 对白是否符合人物身份和权力位置。
- 台词能否被动作、物件、沉默压住一部分，而不是把潜台词全说白。

**常见失败**：

- 人物直接说出内心说明书。
- 对白只有情绪宣泄，没有信息交换。
- 古风场景里台词太现代、太口号化，或像作者替人物解释设定。

**修改动作**：

- 把“我很担心你”改成问药、问伤、问规矩、问去处。
- 让角色答非所问，用回避、反问、命令或沉默暴露真实目的。
- 每段对话后检查：谁获得了信息，谁失去了退路，谁暴露了破绽。

### 关系边界与“差一点”

**目标**：关系推进不靠直白表白或露骨亲密，而靠边界被试探、逼近、退让、重新划定。

**检查点**：

- 亲密动作是否有目的：救治、试探、控制、拒绝、交换、确认秘密。
- 关系是否停在“差一点”：差一点说真话、差一点越界、差一点失控、差一点逃不掉。
- 靠近之后是否仍保留代价或危险，而不是立刻变甜。

**常见失败**：

- 肢体接触只为撒糖，不改变人物判断或局势。
- 角色突然软化、献祭或投降，失去原本张力。
- 关系推进只有“更亲密”，没有边界变化。

**修改动作**：

- 给每次接触加一个具体目的和一个后果。
- 把直接亲吻、告白、拥抱前移成停顿、扣住、避开、压住衣袖、递回物件。
- 让一方靠近，另一方同时保留后路或提出条件。

### 具体狠话

**目标**：威胁、冷话、权谋台词必须具体，有对象、有筹码、有代价。

**检查点**：

- 狠话是否说清“威胁谁、凭什么、代价是什么、下一步会怎样”。
- 话语是否贴合角色资源：身份、兵权、证据、药、债、人情、秘密、位置。
- 台词是否顺口、准确、符合语境，而非为了显得深。

**常见失败**：

- “你会后悔”“这是你的命”“我会让你付出代价”这类空泛狠话。
- 量词、搭配、称谓错误，导致气势塌掉。
- 用玄乎句子替代具体利益关系。

**修改动作**：

- 把抽象威胁改为具体动作：扣人、断路、交证、封口、换名单、移物证。
- 给狠话加筹码名词：账册、问帖、钥匙、旧债、伤口、亲族、门路。
- 读一遍台词，删掉不影响行动的漂亮词。

### 信息差与自利逻辑

**目标**：权谋、宅斗、江湖和群像人物都要有自利逻辑；冲突来自各方所知不同、所求不同、可付代价不同。

**检查点**：

- 每个重要人物是否知道不同的信息，而不是共享作者视角。
- 反派、权臣、长辈、盟友是否都有自保、求利、避祸或夺权的动机。
- 场景推进是否来自“误判/试探/交换/利用”，而不只是作者安排。

**常见失败**：

- 所有人都像为主角服务。
- 阵营只有正邪标签，没有利益结构。
- 角色突然说出不该知道的信息。

**修改动作**：

- 给每个角色写一句“他以为自己在做什么”和“他真正害怕失去什么”。
- 在场景前标清谁知道真相、谁只知道表象、谁故意放假线。
- 让错误判断产生后果，而不是马上被作者解释清楚。

### 场景推进

**目标**：每场戏都要改变状态，不做可删的过场。

**检查点**：

- 每场戏至少回答：谁想要什么、被什么阻挡、离开时什么改变了。
- 场景是否通过行动、冲突和后果显影设定。
- 3-5 个场景/动作节点是否足够支撑一章思路，避免章节规划写成长篇论文。

**常见失败**：

- 场景只解释背景或重复气氛。
- 人物进入和离开时关系、信息、物件、伤病、承诺都没变。
- 一章内多个场景功能重复。

**修改动作**：

- 写场景前先填“进入状态 -> 离开状态”。
- 删掉不能改变状态的场景，或把它并入更有冲突的场景。
- 用物件易手、秘密泄露、承诺成立、伤势变化、命令落下作为场景落点。

### 人物选择

**目标**：人物变化来自选择，不来自作者总结。

**检查点**：

- 主角是否在关键节点主动选择，而不是只被剧情推着走。
- 选择是否有代价，且代价会延续到后文。
- 人物的“心软、冷酷、偏执、善良”是否通过行动被证明。

**常见失败**：

- 人物只想不做，只说不选。
- 作者直接宣布“他变了”“她懂了”。
- 选择没有后果，下一场像什么都没发生。

**修改动作**：

- 把心理结论改成可见选择。
- 让人物失去一样东西、暴露一点信息、欠下一笔债或逼近一个雷点。
- 在下一场至少回收一次选择后果。

### 物象回环

**目标**：物象不是摆设，要成为情绪、证据或权力关系的一部分。

**检查点**：

- 关键物件是否能承担至少一种功能：证据、筹码、误会、承诺、危险、亲密边界。
- 物件重复出现时含义是否推进，而不是原样复述。
- 同一情绪节点是否只保留一个最准确的物象。

**常见失败**：

- 物象只漂亮，不改变剧情。
- 花、雪、灰、雾、月光等相近意象连堆，画面互相抵消。
- 前文重要物件后文不回扣，变成空置伏笔。

**修改动作**：

- 给物件安排“出现 -> 易手/变形/染痕 -> 变成证据或边界”的路径。
- 删掉同功能意象，只留最能改变情绪或行动的那个。
- 章末优先回扣一个前文物件，而不是写抽象感悟。

### 章末钩子

**目标**：章末落在动作、物件、命令、选择、外部压力或未解信息上，让下一章能直接接动作。

**检查点**：

- 结尾是否留下具体后续入口：谁要去哪里、谁扣住什么、谁下了命令、谁看见了证据。
- 情绪是否由外部动作或物件承载，而不是作者总结。
- 钩子是否同时牵动至少一条人物关系或剧情线。

**常见失败**：

- “她终于懂了”“这一刻他们都明白了”式感悟收束。
- 只留情绪，不留行动入口。
- 用大词收尾，实际没有新信息。

**修改动作**：

- 把顿悟句改为门响、信到、物件入袖、伤口裂开、命令落下。
- 让章末最后一个名词具体可见。
- 检查下一章第一场是否能从这个动作直接开始。

### 类型适配

**目标**：类型感来自规则、关系和代价，不来自标签。

**检查点**：

- 古装言情：关系推进、身份差、礼法边界、暧昧张力和选择代价是否清楚。
- 权谋/宫斗/宅斗：信息差、利益交换、台面话、暗手和后果是否比口号更重要。
- 武侠/仙侠/玄幻：动作逻辑、规则代价、修行/术法边界和江湖/宗门秩序是否自洽。
- 爽感强节奏：爽点是否来自铺垫后的反击、揭露或选择，而不是空喊。

**常见失败**：

- 古风只靠生僻词和称谓，缺少礼法与权力关系。
- 权谋只靠“棋局”“布局”口号，缺少具体筹码和误判。
- 爽点没有铺垫，像作者直接发奖励。

**修改动作**：

- 给类型场景补上规则：谁能说话、谁不能越界、谁握着什么资源。
- 把口号式“执棋”改成具体行动链。
- 让爽点之前至少有一个压迫、误判或代价。

## 坏习惯 -> 修法

| 坏习惯 | 修法 |
|---|---|
| 重复否定排比：连续写“没问、没看、没说” | 压成一个动作或反应，例如只写对方越过伤口去问差事。 |
| 相近意象堆叠：同一段连续写花、雪、灰、雾 | 保留一个最准确物象，让它承担情绪或证据功能。 |
| 空泛狠话：只说“你会后悔”“我会让你付代价” | 写具体筹码、具体代价和下一步动作。 |
| 情绪结论：直接写“他心疼了”“她终于明白” | 改成物件、触感、停顿、避让、收回手、扣住信。 |
| 只有暧昧没有事件 | 给亲密动作绑定任务、秘密、伤病、证据或权力压力。 |
| 只有 plot 没有人物关系 | 让事件改变称呼、站位、信任、欠债或退路。 |
| 章末感悟 | 改成动作钩子：信到、门响、物件易手、命令落下、伤口变化。 |

## AI味反例清单

优先在质量门中拦截：

- “命运的齿轮开始转动”
- “她终于明白了”
- “说不清的情绪”
- “复杂的情绪”
- “空气仿佛凝固了”连续滥用
- “这一刻，所有人都不知道……”式上帝视角
- 连续无意义排比或重复否定
- 用宏大词替代具体筹码，如“宿命”“羁绊”“救赎”而没有行动支撑

## 使用方式

- 章节规划阶段：只选 2-3 个最相关原则，写成一行 `本章写作原则`。
- 起草阶段：让原则服务本书风格，不把原则写成显眼说明。
- 自检阶段：原则失误通常标为 P1/P2；若造成 AI 腔硬伤、用户雷点、连续性错误或过期记忆，再标为 P0。
