# 吗啡呼吸抑制风险评估与识别

## 适用场景
- 患者使用吗啡后呼吸频率 < 12 次/分钟
- 出现意识障碍、针尖样瞳孔伴通气不足
- 联合使用镇静药、麻醉药或酒精后呼吸功能恶化
- 急性吗啡中毒的快速识别与抢救决策

## 执行步骤

### 1. 识别临床特征
- **主要表现**：呼吸频率显著减慢（最突出）
- **伴随表现**：潮气量↓、每分通气量↓
- **中毒危象**：呼吸频率可降至3–4次/分钟

### 2. 判断机制与影响因素
- 降低脑干呼吸中枢对CO₂张力的敏感性
- 抑制脑桥呼吸调节中枢
- **起效时间**：
  - 静脉注射：5–10分钟达峰
  - 肌内注射：30–90分钟达峰

### 3. 评估危险因素
- **协同抑制**：与麻醉药、苯二氮䓬类、酒精合用显著加重抑制
- **特殊风险**：CO₂蓄积 → 脑血管扩张 → 颅内压升高
- **关键区别**：吗啡抑制呼吸但**不抑制延髓心血管中枢**（不同于全麻药）

### 4. 紧急响应触发条件
若同时出现以下三项，立即视为吗啡中毒：
- 呼吸频率 < 8 次/分钟
- 针尖样瞳孔
- 意识障碍（嗜睡至昏迷）

> 立即启动抢救：保持气道通畅、辅助通气，并准备纳洛酮拮抗。
