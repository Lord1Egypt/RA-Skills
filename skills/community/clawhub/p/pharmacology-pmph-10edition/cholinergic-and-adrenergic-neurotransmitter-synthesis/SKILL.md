# 胆碱能与肾上腺素能神经递质合成机制

## 何时使用
- 用户询问乙酰胆碱或去甲肾上腺素在传出神经末梢的合成过程
- 需要确定限速步骤或限速酶（如TH或胆碱摄取）
- 分析药物（如密胆碱、Vesamicol）如何通过干预合成路径影响递质水平
- 排除非经典递质（如NO、ATP）或非神经元细胞的合成场景

## 执行步骤

### 1. 确认适用对象
仅适用于：
- 去甲肾上腺素能神经末梢
- 胆碱能神经末梢

不适用于中枢神经元以外无合成能力的细胞或非经典递质路径。

### 2. 选择递质类型并执行对应路径

#### A. 乙酰胆碱（ACh）合成
1. **原料获取**：
   - 胆碱通过钠依赖性高亲和力转运体A摄取（限速步骤，可被密胆碱阻滞）
   - 乙酰辅酶A（AcCoA）在线粒体内生成，需经枸橼酸穿梭进入胞质
2. **催化合成**：
   - 在胆碱乙酰转移酶（ChAT）催化下，胆碱 + AcCoA → ACh（胞质中）
3. **囊泡储存**：
   - ACh由囊泡乙酰胆碱转运体（转运体B）转运入囊泡，可被Vesamicol阻滞

#### B. 去甲肾上腺素（NA）合成
1. **原料摄取**：
   - 酪氨酸经钠依赖性转运体A进入末梢
2. **限速步骤**：
   - 酪氨酸羟化酶（TH）催化酪氨酸 → 多巴（dopa）
   - 此步受终产物（多巴胺/NA）反馈抑制
3. **后续转化**：
   - 多巴 → 多巴胺（DA） via 多巴脱羧酶（DDC）
   - DA经转运体B入囊泡
   - 囊泡内：DA → NA via 多巴胺β-羟化酶（DβH）
   - （可选）NA → 肾上腺素 via PNMT（主要在肾上腺髓质）

### 3. 识别关键调控点
- ACh合成限速因子：胆碱摄取（转运体A）
- NA合成限速酶：酪氨酸羟化酶（TH）

> **输出原则**：递质合成速率由限速步骤决定；干预限速环节可显著改变递质水平。
