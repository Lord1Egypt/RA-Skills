## Description: <br>
Game concept ideation using MDA framework, verb-first design, and mashup method to guide developers through a structured six-phase brainstorming process for a viable game concept. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
Game developers, designers, and teams use this skill to move from a blank page to a scoped game concept with inspiration inputs, emotion targets, core loops, design pillars, player validation, and MVP boundaries. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can save brainstorm output to a project file, which may overwrite an existing brainstorm-output.md or game-dev-studio/brainstorm-output.md. <br>
Mitigation: Check for an existing output file before saving and ask the user to confirm replacement or choose a new path. <br>
Risk: Broad trigger phrases may activate the skill for general game-idea requests. <br>
Mitigation: Invoke it intentionally when a structured game concept workflow is desired. <br>
