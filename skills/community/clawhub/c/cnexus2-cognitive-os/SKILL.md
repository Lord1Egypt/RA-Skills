# CNexus 2.0 — 极客的数字第二大脑：类脑认知架构与活体神经元代谢

## 项目概述

CNexus 2.0 不是一个普通的 RAG（检索增强生成）工具，它是在本地运行的一尊"活体数字生命"。

本项目的核心目标是为极客构建一个基于类脑认知科学（Cognitive Science）的本地化第二大脑。通过打破传统静态索引的限制，CNexus 2.0 引入了**动态神经元激活扩散**与**REM 睡眠代谢**机制，让 AI 不仅能"记住"你的每一个技术细节，更能随着对话深度自我进化，通过"虫洞协议"在语义空间中实现跨维度的联想与共振。

---

## 核心特性 (Key Features)

### 🧠 活体类脑架构 (Living Cognitive Architecture)

**神经元激活扩散 (Spreading Activation)：** 摒弃传统全量搜索。系统实时模拟大脑神经元放电，高频对话话题自动获得"潜意识"预热，实现毫秒级的精准上下文击穿。

**REM 深度睡眠代谢 (REM Sleep Metabolism)：** 引入周期性自清洁与压缩协议。系统在静息期自动完成突触剪枝（删除垃圾记忆）与知识压缩（将零碎对话提炼为长期常识骨干），确保大脑永不膨胀。

### 🌉 跨维度暗物质通道 (Wormhole Protocol)

**本地余弦虫洞：** 基于本地向量空间，让物理结构不相连的记忆节点通过语义相似度瞬间打通"虫洞"。即便你从未显式关联过两个话题，系统也能跨越空间直接联想。

### 🧬 多模态物理骨架感知 (Code & Visual Projection)

**AST 极速投影：** 零依赖解析 Python 代码库，将函数定义、类继承关系瞬间织成动态霓虹流光网络。

**多模态架构识别：** 通过本地视觉模型自动提炼架构图结构，让你的系统设计图与你的对话记忆在图谱上完美重合。

### ⚙️ 极客技术栈 (Tech Stack)

**零依赖设计 (Zero-Dependency)：** 深度利用 Python 标准库（如 `ast`），尽可能减少第三方庞大向量库的依赖，保持极致的轻量与主权。

**混合云边协议：** 支持本地 Ollama 离线计算，并具备针对云端 Embedding API 的智能弹性降级逻辑，确保在不同算力环境下的平滑运行。

**可观测性 (Observability)：** 提供完整的 `/api/status` 与 `/api/status/graph`，让用户能直观观察到神经网络的"心跳"与"代谢"。
