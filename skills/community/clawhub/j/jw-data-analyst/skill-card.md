## Description: <br>
JW Data Analyst helps agents generate Python data-processing scripts, visual charts, and statistical reports for Excel, CSV, JSON, database, and API data sources. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and data analysts use this skill to have an agent load, clean, analyze, visualize, and summarize provided data from files, databases, or APIs. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may analyze sensitive files, database connections, or API data supplied by the user. <br>
Mitigation: Provide only data the agent is allowed to inspect, avoid unnecessary raw sensitive data, and desensitize datasets when appropriate. <br>
Risk: Generated Python scripts and output files may write to paths or perform processing that should be reviewed before execution. <br>
Mitigation: Ask the agent to use a specific output folder and review generated scripts before running them. <br>
