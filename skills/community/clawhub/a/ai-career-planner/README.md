# AI时代职业规划助手 (AI Career Planner)

> 🧭 基于六维风险评估 + 五维能力模型 + 三梯队职业推荐的AI时代职业规划工具

## 功能

- **AI替代风险评估** — 六维模型(任务重复性/创造力/人际交互/非结构化决策/数字化程度/AI熟练度)，0-100量化评分
- **任务级分析** — 逐项分析核心工作任务的被替代风险（红/黄/绿三级）
- **AI能力雷达图** — 五维评估(AI协同时代/跨学科整合/审美判断力/原始创新力/情感领导力)
- **三梯队职业推荐** — 顺势升级 → 相近转型 → 全新赛道，含匹配度/难度/薪资前景
- **12个月行动计划** — 按季度拆分的可执行转型路线图
- **可视化HTML报告** — Chart.js交互式仪表盘+雷达图+时间线
