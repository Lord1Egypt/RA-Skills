## Use Case: <br>
External users use this skill to create habits, log completions, review streaks and progress, and set reminders through an AI assistant. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Habit messages and personal routine details may be sent to SkillBoss for processing, while the artifact also claims data is stored locally. <br>
Mitigation: Review the external setup guide and data flow before use, and avoid entering sensitive health or private routine information unless that processing is acceptable. <br>
Risk: The skill requires a sensitive SkillBoss API key. <br>
Mitigation: Use a dedicated, revocable API key stored in the environment, and rotate or revoke it if exposure is suspected. <br>
