# 淋菌性与非淋菌性尿道炎的鉴别与治疗

## 何时使用
当男性患者有不洁性交史，并出现尿道分泌物、排尿不适、尿道口红肿等症状时，使用本技能进行病原学鉴别和治疗决策。

## 执行步骤

1. **采集关键临床信息**：
   - 记录潜伏期（淋病：2～5日；非淋：1～5周）；
   - 观察分泌物性质（淋病：大量脓性；非淋：少量稀薄白色，晨起“糊口”）；
   - 确认是否有性接触史及配偶感染风险。

2. **进行实验室鉴别**：
   - 淋菌性尿道炎：尿道分泌物涂片见革兰阴性双球菌（位于多核白细胞内），尿三杯试验第一杯脓尿明显；
   - 非淋菌性尿道炎：涂片见10～15个WBC/HPF但无革兰阴性双球菌，需行衣原体/支原体核酸检测或培养；
   - 注意双重感染可能，应同步排查两种病原体。

3. **选择治疗方案**：
   - 淋菌性尿道炎：首选头孢曲松1.0g单次肌注或静注；重症或合并生殖系感染时延长疗程至7～14日，并加用口服喹诺酮类、头孢菌素或复方磺胺甲恶唑；
   - 非淋菌性尿道炎：首选米诺环素或红霉素；
   - 无论何种类型，配偶必须同步治疗以防止重复感染。

4. **处理并发症与特殊情况**：
   - 若发展为尿道狭窄，需定期尿道扩张或手术干预；
   - 新生儿感染多源于分娩过程，需特别关注；
   - 淋病治愈后无持久免疫力，可再感染，应加强健康教育。
