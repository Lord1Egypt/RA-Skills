# 第三代EGFR-TKI在非小细胞肺癌中的应用决策

## 触发条件
当患者满足以下任一条件时启用本技能：
- 局部晚期或转移性NSCLC且携带EGFR敏感突变（19外显子缺失或L858R）
- 既往接受第一/二代EGFR-TKI治疗后疾病进展，且经检测确认T790M突变阳性
- IB～ⅢA期EGFR突变阳性NSCLC术后需辅助治疗
- 存在脑或脑膜转移的EGFR突变阳性NSCLC

## 执行步骤

### 1. 确认适用对象
- **必须满足**：EGFR突变阳性（19del、L858R 或 T790M）
- **排除人群**：EGFR野生型、非指定突变类型、间质性肺炎高风险患者

### 2. 判断治疗线数与适应症
- **一线治疗**：初治的局部晚期或转移性NSCLC伴19del或L858R突变 → 推荐奥希替尼、阿美替尼或伏美替尼
- **二线治疗**：既往EGFR-TKI治疗失败且T790M阳性 → 推荐上述任一第三代TKI
- **辅助治疗**：仅奥希替尼获批用于IB～ⅢA期术后辅助

### 3. 脑转移优先策略
- 若存在脑或脑膜转移，**优先选择第三代EGFR-TKI**（因其优异的血脑屏障穿透能力）

### 4. 药物选择与监测计划
根据药物特性制定个体化方案：
- **奥希替尼**：监测皮肤反应、腹泻、Q-Tc间期延长
- **阿美替尼**：关注皮疹、瘙痒、血肌酸激酶升高；腹泻风险较低
- **伏美替尼**：密切监测ALT/AST（>20%患者出现升高）

### 5. 关键禁忌与随访
- 用药前**必须确认EGFR突变状态**
- 治疗期间定期进行：
  - 胸部/脑部影像学检查
  - 肝功能检测
  - 心电图（尤其使用奥希替尼时）
- 出现**间质性肺炎症状（如呼吸困难、咳嗽加重）立即停药并处理**

## 输出结果
明确回答：
- 是否适用第三代EGFR-TKI
- 推荐的具体药物
- 对应的监测与随访计划
