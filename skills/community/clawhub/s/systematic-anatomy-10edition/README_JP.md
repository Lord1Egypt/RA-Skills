## プロジェクト概要

本プロジェクトは、人体解剖学、神経解剖学、臨床応用解剖学、および中西医結合解剖学の核心領域を体系的に統合し、**112 の重要解剖スキル**を **8 カテゴリ**にまとめています。

**対象読者**：医学生、臨床医、解剖学研究者、外科研修医

**参考教科書**：人民衛生出版社『系統解剖学』第10版

**⚠️ 免責事項 ⚠️**：本スキルセットは人体構造の記述、解剖変異の分析、外科アプローチの評価、臨床局在診断をカバーしていますが、専門的な解剖学教育や臨床トレーニングの代替にはなりません。

### インストール

CLI：
```bash
openclaw skills install systematic-anatomy-10edition
```

### 使用例

```bash
# 概念クエリ
内包の位置と区分

# 臨床ケース分析
65歳男性、突然の右側脱力と構音障害で発症3時間。
検査：右上肢・下肢筋力0/5、右中枢性顔面神経麻痺、
右半身の痛覚・温度覚低下、眼球は左に偏位。
考えられる病変部位を分析せよ。

# 試験問題
内包に関する記述で誤っているのはどれか。
```
