## 三、匹配算法

### 3.1 匹配度计算

```
匹配度 = (赛道匹配×0.3 + 阶段匹配×0.25 + 关注点匹配×0.25 + 估值匹配×0.2) × 100

计算说明：
- 赛道匹配：强=1.0, 中=0.6, 弱=0.2
- 阶段匹配：完全符合=1.0, 相邻阶段=0.7, 相差较大=0.3
- 关注点匹配：项目优势与机构关注点重合度
- 估值匹配：项目估值在机构可接受区间=1.0, 偏高=0.5, 明显偏高=0.2
```

### 3.2 匹配优先级

```
优先级 = 匹配度 × 机构实力系数 × 合作历史系数

机构实力系数：
- 一线机构：1.2
- 知名机构：1.0
- 新锐机构：0.9

合作历史系数：
- 长期合作：1.3
- 成功案例：1.2
- 有过接触：1.0
- 首次合作：0.9
```
