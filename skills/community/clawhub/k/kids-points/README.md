# 📚 kids-points - 积分助手

> 简单、灵活的儿童积分管理工具

---

## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| 📝 **积分记录** | 语意识别，自动记账 |
| 💰 **积分消费** | 语意识别，自动扣减 |
| 📊 **余额查询** | 读取 balance.md，跨 Session 一致 |
| 📋 **输入日志** | 审计追踪，防重复 |

---

## 🚀 快速开始

### 1. 积分记录

直接说完成了什么任务，不需要固定格式：

```
今天完成了汉字抄写 2 课，口算题卡 2 篇全对
```

系统会自动：
- ✅ 理解语义，提取任务
- ✅ 写入 balance.md
- ✅ 写入 input.log（防重复）
- ✅ 返回确认消息

### 2. 积分消费

```
积分消费 买零食花了 20 分
```

### 3. 余额查询

```
现在多少分？
今日积分
```

---

## 📊 数据文件

### balance.md（唯一数据源）
```
kids-points/balance.md
```

所有积分记录都保存在这里，可以手动修正。

### input.log（审计日志）
```
kids-points/logs/input.log
```

记录每次输入，用于防重复检查和审计追踪。

## 💡 使用技巧

### 语意识别

系统能理解各种说法：

| 你说 | 系统理解 |
|------|---------|
| "写了 2 课生字" | 汉字抄写 +2 分 |
| "口算做了 2 篇，全对" | 口算题卡 +4 分 |
| "ABC Reading 读了 2 篇" | ABC Reading +3 分 |
| "忘带书包了" | 积分消费 -5 分 |

### 防重复

同一天说相同的内容会提示：
```
⚠️ 这条记录已经存在了
```

### 手动修正

直接编辑 `balance.md` 文件：
```markdown
| 2026-03-25 | 收入 | +2 | 98.8 | 汉字抄写 |
```

下次查询时会自动读取最新数据。

## ❓ 常见问题

**Q: 忘记说"学习积分"前缀可以吗？**  
A: 可以！系统能理解"完成了"、"做了"、"写了"等说法。

**Q: 记错了怎么办？**  
A: 直接编辑 `balance.md` 修正，或者通过调账记录修正。

**Q: 跨 Session 数据一致吗？**  
A: 一致！每次查询都读取 `balance.md`，不依赖记忆。
