# 子任务5：格式检查

## 任务目标
现场读取Excel实际格式，记录实际格式参数，输出格式确认清单，确保后续写入格式正确。

## 输入变量

| 变量名 | 说明 | 示例 |
|:---|:---|:---|
| `{{platform}}` | 目标平台名称 | 小红书、抖音、视频号 |
| `{{excel_path}}` | 文案库Excel路径 | [用户配置的路径] |
| `{{sheet_name}}` | 工作表名称 | 文案库 |

## 执行流程

1. 使用Python openpyxl读取Excel文件
2. 扫描标题行格式（字体、颜色、边框等）
3. 扫描数据行格式（A列、B-H列、I-N列）
4. 记录合并单元格规则
5. 输出格式确认清单

## 常见格式差异

| 项 | 可能差异 | 处理方式 |
|:---|:---|:---|
| 字体 | 微软雅黑/宋体 | 记录实际值 |
| 字号 | 10/11/12 | 记录实际值 |
| 颜色 | RGB值差异 | 记录实际RGB |
| 合并 | 跨1行/多行 | 现场确认 |

## 错误处理

```
如果 Excel文件不存在:
    输出: "⚠️ Excel文件不存在: {{excel_path}}"
    终止任务

如果 格式读取失败:
    输出: "⚠️ 格式读取失败: [错误详情]"
    尝试使用默认格式（提示风险）
```
