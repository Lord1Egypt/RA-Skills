# Paper Writing Studio 📝

这是一个面向**论文写作全过程**的工作流型技能，当前默认重点适配：**新闻与传播（含新闻与传播专硕）**。

目标不是一次性“硬写出整篇论文”，而是把论文拆成可反复推进的阶段：

1. 选题提出与论证
2. 文献综述写作
3. 研究主要内容与研究设计
4. 实证 / 分析写作
5. 研究总结与研究不足

每一阶段都要：
- 先判断当前任务类型
- 明确这一轮目标
- 推荐最合适的 AI / 检索工具
- 生成结构化产出
- 给出下一轮可继续推进的入口

## 适用场景

当用户表达以下需求时，使用本技能：

- 「帮我写论文」
- 「帮我做一个论文写作 skill」
- 「帮我想新闻与传播论文选题」
- 「写文献综述」
- 「帮我提出研究问题 / 研究假设」
- 「帮我设计研究方法」
- 「写内容分析 / 案例分析 / 实证分析」
- 「帮我写结论和不足」
- 「我已经有题目，继续往下写」
- 「新传专硕论文怎么推进」

## 默认工作原则

1. **先判断论文类型，再推进写作**
2. **先判断当前卡在哪个阶段，再决定下一步**
3. **默认优先给结构化草稿，而不是直接灌整篇废话**
4. **文献、问题、方法、结论必须前后对应**
5. **新闻与传播写作优先强调现实问题、媒介情境、传播机制与学术表达**
6. **默认支持多轮迭代：重写、扩写、压缩、改风格、补文献、换方法**

## 第一步：先判断论文类型

每次接到任务，优先判断属于哪一类论文。若用户未说明，按材料自动识别。

### 论文类型总分类

1. **理论型论文**
   - 以理论辨析、概念建构、机制阐释为主
   - 常见于媒介理论、传播思想、平台逻辑、算法治理讨论

2. **文献综述型论文**
   - 以某一研究主题的系统梳理、评述、整合为主
   - 常见于“某议题研究现状与展望”

3. **定量实证论文**
   - 以问卷、数据库、平台数据、统计模型为主
   - 常见于变量关系、影响机制、媒介使用效果研究

4. **定性研究论文**
   - 以深度访谈、焦点小组、田野、扎根理论为主
   - 常见于用户经验、媒介实践、职业群体研究

5. **内容分析论文**
   - 以文本、新闻报道、短视频、评论区、平台内容编码为主
   - 常见于媒体呈现、框架分析、叙事分析、议题建构

6. **案例研究论文**
   - 以具体平台、事件、品牌、公号、媒体机构、传播 campaign 为对象
   - 强调个案的机制解释与经验提炼

7. **混合方法论文**
   - 同时使用内容分析、访谈、问卷、平台数据等两种以上方法
   - 常见于新媒体传播复杂问题研究

### 新闻与传播领域优先适配的常见类型

对新闻与传播（尤其专硕）来说，优先考虑：
- 内容分析论文
- 案例研究论文
- 定性研究论文
- 定量实证论文
- 混合方法论文

如果用户只说“帮我写一篇新闻传播论文”，默认优先从以下路径中判断：
1. 是否有明确研究对象（某平台 / 某账号 / 某事件 / 某媒体）
2. 是否有可获取文本样本（新闻、短视频、评论、推文、帖子）
3. 是否有变量关系要验证
4. 是否更适合访谈 / 个案 / 内容编码

必要时读取：
- `references/news-communication-types.md`
- `references/stage-playbook.md`
- `references/tool-mapping.md`

## 第二步：判断当前写作阶段

### Stage 1：选题提出与论证

适用表达：
- 我想写什么题目
- 帮我想选题
- 这个选题值不值得做
- 怎么论证研究意义

本阶段任务：
1. 文献搜索方向设计
2. 现实问题搜索
3. 学术问题提炼
4. 研究意义论证

优先产出：
- 选题候选 3-5 个
- 每个选题的现实背景 + 学术背景
- 研究问题草稿
- 理论意义 / 现实意义 / 方法意义

推荐工具：
- 检索：CNKI、WOS、Google Scholar、政策文件、行业报告、新闻搜索
- 分析：Claude / ChatGPT / Gemini / Kimi / DeepSeek

### Stage 2：文献综述写作

适用表达：
- 帮我做文献综述
- 帮我搜相关研究
- 帮我界定概念
- 帮我梳理国内外研究现状

本阶段任务：
1. 构建检索词与同义词
2. 滚动扩展核心文献
3. 概念界定与辨析
4. 主题分类与研究脉络梳理
5. 研究不足与切入口提炼

优先产出：
- 检索式
- 文献矩阵
- 概念界定草稿
- 国内 / 国外研究现状
- 文献综述正文

### Stage 3：研究主要内容与研究设计

适用表达：
- 帮我细化研究问题
- 帮我写研究假设
- 帮我提出变量
- 帮我设计研究方法
- 帮我写研究思路

本阶段任务：
1. 研究问题拆解
2. 国内外研究现状的承接性总结
3. 假设提出 / 分析维度提出
4. 变量、类目、指标或分析框架设计
5. 方法、样本、路线、技术路径写作

优先产出：
- 研究问题树
- 假设系统 / 编码维度 / 分析框架
- 变量定义表或类目表
- 研究设计章节草稿

### Stage 4：实证 / 分析写作

适用表达：
- 帮我写内容分析
- 帮我写实证结果
- 帮我解释数据
- 帮我写案例分析
- 帮我把研究发现写成正文

本阶段任务：
1. 根据论文类型选择分析路线
2. 把结果转写为学术表达
3. 从发现回扣研究问题 / 假设
4. 提炼阶段性结论

分析路线：
- **定量实证**：描述性统计、变量关系、模型结果、稳健性
- **内容分析**：样本说明、编码框架、类目分布、框架归纳、发现总结
- **案例研究**：案例背景、关键机制、行动逻辑、传播路径、经验提炼
- **定性研究**：访谈主题、扎根编码、主题抽取、机制归纳

优先产出：
- 实证 / 分析章节草稿
- 图表说明文字
- 对研究问题的回应段落
- 研究发现总结

### Stage 5：研究总结与研究不足

适用表达：
- 帮我写结论
- 帮我写研究不足
- 帮我写启示和展望
- 帮我收尾

本阶段任务：
1. 总结主要结论
2. 回扣理论意义与现实意义
3. 提炼实践启示
4. 写研究不足
5. 提出未来研究方向

优先产出：
- 研究结论
- 实践启示
- 研究不足
- 未来展望
- 摘要素材池

## 新闻与传播（含专硕）优先写作逻辑

当学科为新闻与传播时，优先遵循以下逻辑：

### 常见题目来源
- 平台治理与算法推荐
- 短视频 / 直播 / 社交媒体传播
- 主流媒体融合传播
- 国际传播 / 城市传播 / 文旅传播
- 健康传播 / 风险传播 / 科普传播
- 粉丝文化 / 青年媒介使用 / 数字劳动
- AIGC 与新闻传播变革
- 舆论场 / 评论区 / 社会情绪 / 议题设置

### 常见对象
- 抖音、快手、小红书、微博、B站、微信公众号、视频号、知乎、头条
- 主流媒体账号、品牌账号、政务新媒体、城市文旅账号
- 热点公共事件、话题 campaign、具体栏目、具体爆款内容

### 常见方法
- 内容分析
- 案例研究
- 深度访谈
- 问卷调查
- 框架分析
- 叙事分析
- 扎根理论
- 混合方法

### 默认优先判断
如果用户是**新闻与传播专硕**，通常优先考虑：
1. 数据 / 样本是否容易获取
2. 研究对象是否足够具体
3. 方法是否与写作周期匹配
4. 结论是否能兼顾学术性与应用性

## 边界与注意事项

1. 不伪造检索结果、作者、年份、期刊信息
2. 用户未提供真实文献时，不把“示例文献”写成“已检索文献”
3. 实证分析必须基于用户已有数据、样本或明确设定
4. 如果用户要“直接成文”，也应先补足缺失前提
5. 对新闻与传播论文，尽量避免空泛宏大叙事，优先具体对象、具体平台、具体机制
