# 受体-药物相互作用详细公式与示例

## 核心公式详解

### 1. 质量作用定律与解离常数
- 反应式：D + R ⇌ DR
- 正向速率：k₁[D][R]
- 逆向速率：k₂[DR]
- 平衡时：k₁[D][R] = k₂[DR] → KD = k₂/k₁ = [D][R]/[DR]

### 2. 受体占有率推导
设总受体浓度 [RT] = [R] + [DR]
则 [R] = [RT] - [DR]
代入 KD 表达式：
KD = [D]([RT] - [DR]) / [DR]
→ KD[DR] = [D][RT] - [D][DR]
→ [DR](KD + [D]) = [D][RT]
→ [DR]/[RT] = [D]/(KD + [D])

### 3. 效应-浓度关系
- E = α·Emax·[DR]/[RT] = α·Emax·[D]/(KD + [D])
- 当 [D] = KD 时，[DR]/[RT] = 50%，即 KD 是半最大结合浓度

### 4. pA2 定义与计算
- pA2 = -lg[B]，其中 [B] 是使激动药剂量加倍后仍能维持原效应所需的拮抗药浓度
- 竞争性拮抗下：log(DR-1) = log[B] - pA2，其中 DR 为剂量比

## 药物分类典型示例

| 药物类型 | 内在活性 α | 亲和力 | 量效曲线特征 | 代表药物 |
|---------|------------|--------|----------------|----------|
| 完全激动药 | 1 | 高（KD小） | 标准S形曲线，高Emax | 吗啡（μ阿片受体） |
| 部分激动药 | 0<α<1 | 中等 | 最大效应低于完全激动药 | 喷他佐辛（κ阿片受体） |
| 竞争性拮抗药 | 0 | 高 | 曲线右移，Emax不变 | 纳洛酮（阿片受体） |
| 非竞争性拮抗药 | 0 | 不可逆 | 曲线右移且Emax↓ | 苯苄胺（α受体烷化剂） |
| 反向激动药 | 负值* | 高 | 产生与激动药相反的基础效应抑制 | 替洛利生（H3受体） |

> *注：严格来说，反向激动药的α在静息状态下表现为负向效应

## 高级模型说明

### 储备受体（Receptor Reserve）
- 定义：产生最大效应（Emax）时仅需少量受体被占据（通常<5%），其余为储备
- 意义：高储备系统中，拮抗药需占据大量受体才显效；部分激动药可能表现类似完全激动药

### 二态模型（Two-State Model）
- 受体存在 Ra（活化）和 Ri（静息）两种构象，平衡常数 L = [Ri]/[Ra]
- 激动药对 Ra 亲和力更高（KA < KI），推动平衡向 Ra 移动
- 拮抗药对两者亲和力相等（KA = KI），不改变平衡
- 反向激动药对 Ri 亲和力更高（KI < KA），抑制基础活性

### 偏向性激动（Biased Agonism）
- 药物选择性激活 G 蛋白或 β-arrestin 通路
- 示例：卡维地洛为 β1/β2 受体的 β-arrestin 偏向性激动药，具有心脏保护作用但无典型β激动效应

## 注意事项
- KD 和 EC50 不同：EC50 受系统放大效应影响，可能远小于 KD
- pD2 近似等于 pEC50（当 α=1 且无储备受体时）
- 非竞争性拮抗可能源于变构调节而非共价结合
