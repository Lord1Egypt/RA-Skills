# Presentation Video Maker 技能

## 快速使用

### 方式一：一键制作（推荐）

```
请帮我制作一个演示视频：
- 脚本文件：[提供内容或文件路径]
- 输出：[视频文件名]
```

示例：
```
请帮我制作一个演示视频，脚本如下：

# 项目成果
我们在过去一周完成了以下工作：
- 创建了记忆系统
- 突破了WeChat反爬
- 建立了知识复用机制

# 未来展望
未来我们可以一起：
- 数据可视化
- 智能邮件管理
- 自动化工作流

# 感谢观看
期待继续合作！
```

### 方式二：分步制作

1. **准备脚本**：创建文本文件，用 `# 标题` 分隔每页
2. **调用技能**：告诉我要制作视频，提供脚本内容
3. **获取结果**：视频生成后自动发送给你

## 脚本格式

```
# 幻灯片标题
内容要点1
内容要点2

# 下一页标题
更多内容...
```

## 输出文件

- `output/` 目录下的 `.mp4` 视频文件
- 包含幻灯片画面 + 中文讲解配音

## 依赖

技能会自动处理依赖安装：
- `python-pptx` - PPT处理
- `pillow` - 图片生成
- `edge-tts` - 语音合成
- `ffmpeg` - 视频合成
