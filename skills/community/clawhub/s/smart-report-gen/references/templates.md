# 📋 更多报告模板

## 📈 月报模板

适合月底总结，自动从每日/每周报告中汇总。

```markdown
# 📈 月报 | {year}年{month}月

## 月度概要
{3-5句话总结本月核心成果}

## 关键数据
| 指标 | 上月 | 本月 | 变化 | 目标 |
|------|------|------|------|------|
| {指标1} | {val} | {val} | ↑/↓/→ | {target} |
| {指标2} | {val} | {val} | ↑/↓/→ | {target} |

## 主要成果
### {领域/项目1}
- {成果1}
- {成果2}
### {领域/项目2}
- {成果1}

## 未完成事项
| 事项 | 原因 | 预计完成 |
|------|------|----------|
| {item} | {reason} | {date} |

## 本月教训
- {lesson 1}
- {lesson 2}

## 下月重点
1. {priority 1}
2. {priority 2}
3. {priority 3}
```

## 🏃 Sprint 迭代报告

适合敏捷开发团队，每轮迭代结束时使用。

```markdown
# 🏃 Sprint 报告 | Sprint #{number}

## 目标
{本轮迭代目标}

## 完成情况
| 任务 | 状态 | 备注 |
|------|:----:|------|
| {任务名} | ✅ | |
| {任务名} | 🔄 | 进行中 |
| {任务名} | ❌ | 阻塞：{原因} |

## 燃尽图数据
| 日期 | 剩余 | 完成 |
|------|------|------|
| 周一 | 12 | 0 |
| 周二 | 9 | 3 |
| ... | ... | ... |

## 回顾
### ✅ 做得好的
- {亮点}
### ⚠️ 要改进的
- {问题}
### 📋 行动项
- [ ] {改进措施} — {负责人} — {截止日期}
```

## 🚨 事件报告

适合系统故障、线上事故的复盘。

```markdown
# 🚨 事件报告 | {date} {time}

## 概要
- **事件**: {一句话描述}
- **影响范围**: {受影响的系统/用户数}
- **持续时间**: {开始} → {结束} ({总计X小时})
- **严重程度**: 🔴 P1 / 🟡 P2 / 🟢 P3

## 时间线
| 时间 | 事件 |
|------|------|
| 14:30 | 监控告警触发 |
| 14:35 | 确认数据库连接池耗尽 |
| 14:45 | 扩容连接池，服务恢复 |

## 根因
{根本原因分析}

## 修复
- **即时**: {紧急处理措施}
- **长期**: {根本解决方案}

## 经验教训
- {lesson 1}
- {lesson 2}
```

## 📁 客户项目周报

适合ToB项目，给客户汇报进展。

```markdown
# 📁 项目周报 | {项目名称}

## 项目健康度
🟢 正常 / 🟡 需关注 / 🔴 有风险

## 本周交付
- ✅ {交付物1}
- ✅ {交付物2}

## 进行中
- 🔄 {任务} — 预计完成：{日期}

## 里程碑追踪
| 里程碑 | 计划日期 | 实际日期 | 状态 |
|--------|---------|---------|:----:|
| {名称} | {date} | {date} | ✅/🔄/❌ |

## 风险
| 风险 | 概率 | 影响 | 应对措施 |
|------|:----:|:----:|---------|
| {描述} | 高/中/低 | 高/中/低 | {方案} |

## 需要客户确认
- [ ] {待确认事项1}
- [ ] {待确认事项2}

## 下周计划
1. {计划1}
2. {计划2}
```

## 🌐 English Daily Report

```markdown
# 📋 Daily Report | {date}

## ✅ Completed
- {task}

## 🔄 In Progress
- {task} ({progress}%)

## ⚠️ Blockers
- {blocker} → {workaround}

## 📅 Tomorrow
- {plan}

## 📊 Metrics
- {metric}: {value}
```

## 🌐 双语报告 Bilingual

```markdown
# 📋 日报 Daily Report | {date}

## ✅ 今日完成 / Completed
- {中文描述} / {English description}

## 🔄 进行中 / In Progress
- {中文} / {English}

## 📅 明日计划 / Tomorrow
- {中文} / {English}
```

## 💡 使用建议

| 场景 | 推荐模板 | 频率 |
|------|---------|------|
| 个人工作记录 | 日报 | 每天 |
| 团队站会汇报 | 日报（精简版） | 每天 |
| 向上汇报 | 周报 | 每周 |
| 客户沟通 | 客户项目周报 | 每周 |
| 敏捷团队 | Sprint报告 | 每迭代 |
| 故障复盘 | 事件报告 | 事发时 |
| 跨国团队 | 双语报告 | 按需 |
