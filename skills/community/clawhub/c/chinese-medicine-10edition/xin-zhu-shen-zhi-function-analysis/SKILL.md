# Xin Zhu Shen Zhi Function Analysis

## When to Use
当出现以下情况时使用本技能：
- 患者表现出精神、意识、思维或情志异常（如失眠、多梦、健忘、反应迟钝、谵妄、昏迷等）；
- 五脏六腑功能整体失调，且无明确单一脏腑病变指向；
- 需从中医整体观出发评估心神与血脉关系对症状的影响。

## Execution Steps
1. **区分广义之神与狭义之神**：
   - 广义之神：指人体整体生命活动及其外在表现，心为“五脏六腑之大主”，统摄全身脏腑协调；
   - 狭义之神：指意识、思维、情志等精神活动，心为“所以任物者”，主导对外界信息的接收与反应。

2. **评估正常功能表现**：
   - 精神振奋、神志清晰、思维敏捷、反应灵敏。

3. **识别功能异常的临床表现**：
   - 轻度：失眠、多梦、健忘、反应迟钝、精神萎顿；
   - 重度：谵妄、昏迷、不省人事、失语、舌强语塞。

4. **结合血脉状态综合判断**：
   - 血为神志活动的物质基础（“血者，神气也”）；
   - 心神亦反向调控血脉运行，二者互为因果。

5. **引用经典理论支持判断**：
   - 《素问·灵兰秘典论》：“心者，君主之官，神明出焉”；
   - 《灵枢·邪客》：“心者，五脏六腑之大主也，精神之所舍也”；
   - 《灵枢·口问》：“心动则五脏六腑皆摇”。

> **注意**：本技能不适用于仅从现代神经科学角度解释意识障碍的情形，必须结合中医整体观进行分析。
