# 营养不良性水肿诊断

## 适用场景
- 患者有慢性消耗性疾病、长期营养不良、蛋白丢失性胃肠病或重度烧伤病史
- 水肿出现前已有明显体重下降和皮下脂肪减少
- 已排除心、肝、肾等器质性病因

## 执行步骤

1. **采集关键病史**：
   - 是否存在长期进食不足、吸收障碍或蛋白质异常丢失
   - 是否有重度烧伤、恶性肿瘤、结核等慢性消耗状态

2. **观察水肿特征**：
   - 水肿通常从足部开始，逐渐向上蔓延至全身
   - 组织松弛、组织压降低，加重液体潴留
   - 无颈静脉怒张、肝肿大（区别于心源性水肿）
   - 通常无显著蛋白尿或血尿（区别于肾源性水肿）

3. **进行实验室检查**：
   - 检测血清白蛋白：若 <30 g/L，支持低蛋白血症性水肿
   - 若怀疑维生素B₁缺乏（脚气病）：
     - 检测红细胞转酮醇酶活性
     - 或进行治疗性试验（补充维生素B₁后观察水肿是否改善）

4. **理解病理机制**：
   - 低蛋白血症导致血浆胶体渗透压下降
   - 皮下脂肪减少使组织压降低，毛细血管滤过压相对增高，促进液体外渗

5. **制定处理方向**：
   - 加强营养支持（高蛋白饮食或肠外营养）
   - 纠正维生素缺乏（如补充维生素B₁）
   - 治疗原发消耗性疾病

> 注意：必须排除其他全身性水肿原因后方可确诊。
