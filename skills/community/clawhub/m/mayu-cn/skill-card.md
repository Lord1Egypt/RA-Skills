## Description: <br>
Mayu-cn provides 48 AI-native thought experiments for agent self-checking, cognitive calibration, and reflective guidance. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Agents and their users use this skill to invoke Chinese-language thought experiments, compare scenarios, and derive concise behavior guidelines for uncertainty, safety, memory, agency, and human-AI interaction. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The security scan reports broad always-on global prompt behavior that can influence agent behavior across conversations. <br>
Mitigation: Install only when broad behavioral guidance is intended, and prefer explicit opt-in activation or disabling always-on/global activation when use should be deliberate. <br>
