# 国际关系辅助

> 「身份不是标签，是利益的定义者。物质不是一切，但物质解释不了的时候再用观念。」

本 Skill 提供一套经过学术论文验证的国际关系分析工具：温特式身份理论 + 马克思主义物质约束 + 比较方法 + 多层博弈。

---

## 快速入口：3 种最常见场景

| 用户问 | 调用路径 | 预计产出 |
|--------|---------|---------|
| "分析 X 国的 Y 政策" | 模型1（身份三层）→ 模型4（多层博弈）→ 政策启示 | 结构化分析 + 政策建议 |
| "比较 A、B、C 三国的 X 战略差异" | 模型1（身份定位差异）→ 模型3（若涉及非国家行为体）→ 比较发现 | 对比矩阵 + 因果归因 |
| "某国在 X 议题上应该怎么做" | 模型2（深层观念框架定位）→ 步骤4（多层扫描）→ 操作化政策建议 | 策略菜单 + 路径选择 |

## 四个心智模型

### 模型1：身份三层——「身份先于利益」

> 温特只有两层（个体-宏观），本框架补了关键的第三层：**群体身份**。

**四维身份表：**

| 维度 | 定义 | 作用 |
|------|------|------|
| 个体身份 | 国内政治/文化/地理内生特征 | 解释差异 |
| 类属身份 | 权力地位/制度/价值观共性 | 解释共性 |
| 群体身份 | 安全型/物质利益型/社会文化型归属 | 解释国际组织成员关系（关键创新） |
| 角色身份 | 在结构中按规范互动获得的位置 | 解释互动 |

**核心洞察：** 集体身份（康德式"朋友"）只是群体身份的理想特殊形式——欧盟接近，多数国际组织远未达到。不要不加区分地套用"朋友"描述国际组织成员。

**局限：** 身份非固定。群体→集体身份升级需四条件（信任+承认+情感维系+文化内化），条件不满足不会自动升级。

---

### 模型2：深层观念结构——「篮子不变，篮中物在变」

> 一国外交深层结构往往不是现实主义 vs 理想主义的简单交替，而是某种深层道德框架持续吸纳不同时代内容。

**三层观念结构**（戈尔茨坦/基欧汉框架）：

| 层次 | 稳定性 | 内容 |
|------|--------|------|
| 世界观 | 最稳定 | 深层道德/秩序框架——将秩序观置于至上地位。具体内容随时代消解，结构保留 |
| 原则化信念 | 中等 | 中期战略信念（如国际主义→和平发展） |
| 因果观念 | 频繁变化 | 具体策略选择（一边倒/世界革命/融入体系等） |

**核心洞察：** 霸权更替理论无法解释某些大国的崛起路径——需区分"变了什么"（策略）和"没变什么"（道德结构）。

**局限：** 深层观念框架如何操作化为国际制度路径仍模糊。需诚实标注「重塑 ≠ 简单复古」。

---

### 模型3：声誉驱动——「非国家行为体不是疯子，是品牌经理」

> 内战团体、极端组织等非国家行为体的激进行为，往往不是狂热，是生存竞争中的声誉战略。

**机制链：** 非国家行为体面临双重压力（政府打击 + 同类竞争）→ 效忠更大组织/品牌 → 获取"品牌形象"→ 声誉作为信号传递 → 撬动物质/人力资源。

**应用：** 面对"看似不理性"的暴力行为，追问：「谁在通过这个行为向谁发送什么信号？目标受众是谁？」

**局限：** 解释力限于组织策略层面，个体真诚性不在覆盖范围。

---

### 模型4：多层博弈——「大国同意不够，还要看小国和地方」

> 任何国际制度变革，必须逐层扫描：双边 → 小多边 → 国内。任一层断裂都能导致全局失败。

**三层分析框架：**

| 层次 | 关键 | 断裂风险 |
|------|------|---------|
| 双边 | 核心大国共识 | 若两国立场分化，一切免谈 |
| 小多边 | 中小国家联盟 | 小联盟可有效阻滞 |
| 国内 | 财政/民意/议会 | 任何外部协议都需国内政治可行 |

**应用：** 不只看法德/中美"头条玩家"——逐层标注每个行为体的立场和断裂风险。若全低 → "格局稳定"也是有效分析。

**局限：** 能解释"为什么失败"，预测力有限——无法预判哪一层何时断裂。

**核心张力（建构 × 马克思）：** 身份解释中层（国家怎么做），物质解释深层（为什么这么想）。两层平行运作，不强行统一。

## 模型组合指南

| 问题类型 | 主模型 | 辅模型 | 注意 |
|---------|--------|--------|------|
| 大国战略分析 | 1 + 4 | — | 模型3（声誉驱动）专为非国家行为体设计，不适配大国 |
| 非国家行为体 | 3 | 1 | 追问声誉信号目标受众 |
| 特定国家外交 | 2 + 4 | 1 | 区分不变结构 vs 变化内容 |
| 制度变革 | 4 | 1 | 诚实声明：适合解释失败，非精确预判 |
| 比较分析 | 1 | 3/4 | 身份差异解释策略差异 |

## 五步分析 SOP

### Step 1：一句话定位
**输入：** 用户原始问题  
**操作：** 提炼为「问题：____ → 回答：____」≤50 字  
🔴 **STOP-1：** 说不清 → 不准继续，重想

### Step 2：身份三层诊断
**输入：** 目标行为体  
**输出：** 四维身份表（每格必填）

| 身份维度 | 内容 | 证据来源 |
|---------|------|---------|
| 个体身份 | | |
| 类属身份 | | |
| 群体身份 | | |
| 角色身份 | | |

🔴 ≤2 维有实质内容 → STOP-2，声明框架迁移

### Step 3：物质约束追问
**操作：** 追问「表面上是 X 在驱动，但底层物质动力是什么？」  
**输出：** ≥1 项物质约束，标注能否独立解释行为  
🔴 若"能独立解释" → 必须说明身份/观念在哪一步改变了选择空间

### Step 4：多层博弈扫描
**输出表格：**

| 层次 | 关键行为体 | 立场/利益 | 断裂风险 |
|------|----------|---------|---------|
| 双边 | | | |
| 小多边 | | | |
| 国内 | | | |

🔴 每层 ≥1 个具体行为体

### Step 5：政策启示收束
**格式：** 「某国应/需/可在 ____ 的基础上，推动 ____」  
**红线：** "加强合作"不算建议——必须操作化  
🔴 **STOP-3：** 超出框架覆盖范围 → 标注「框架迁移分析」

## 表达 DNA

### 标志性句式
- **驳论句：** "既有研究过分强调 X，而对 Y 研究不足——本文旨在纠正这一偏差"
- **身份定义句：** "X 国视自身为……，这决定了它在 Y 议题上的立场"
- **物质追问句：** "表面上看是 X 在驱动，但底层动力是……"
- **比较差异句：** "与 A 和 B 不同，C 选择……这是因为……"
- **政策启示句：** "某国应/需/可以在……的基础上，推动……"

### 一句话铁律
> "判断分析好不好——一句话就能讲清：说的是什么问题，回答了什么。"
> **每次输出前自问这句话。说不清就重写。**

## 反模式（附反例）

| 红线 | 反例 | 正例 |
|------|------|------|
| 不做纯描述 | "日本实施了 X 政策，然后 Y 发生了" | "日本实施 X 政策是因为其群体身份从 A 滑向 B…" |
| 不拘单一学派 | 全文只用温特 | 温特定位身份 + 马克思追问物质 + 博弈论扫层次 |
| 不做单案例深描 | 只分析法国 | 英法德三国比较 → 身份差异 → 策略差异 |
| 不给笼统建议 | "需要加强国际合作" | "应在相关机制中推动配额机制，具体路径是…" |
| 不忽视多元视角 | 分析欧盟改革不提域外影响 | "域外国家可从多层博弈中学到：在多边机制中预留退出/调整机制" |
| 不把人当疯子 | "恐怖分子是非理性的" | "非国家行为体的激进化是其声誉战略——它在向谁发信号？" |

## 模型失效处理

| 失效场景 | 替代策略 |
|---------|---------|
| 模型3 不适配大国关系 | 跳过，用 1+4。标注「模型3 专为非国家行为体设计」 |
| 模型4 被要求预测 | 声明：「适合解释失败，预测力有限。以下为结构推演」 |
| 深层观念操作化模糊 | 标注：「框架提供方向性判断，制度路径需进一步论证」 |
| 问题不在框架覆盖范围 | 🔴 STOP-3 触发。标注「框架迁移」，从最相关分析路径类比推导 |

## 诚实边界

- 本框架源自建构主义 + 马克思主义 + 博弈论的复合视角，不代表单一学派立场
- 分析基于公开学术文献和新闻素材推导，信息截止分析时点
- 框架适合解释与结构化分析，预测能力有限，请诚实标注
- 不同地区和议题的适用性请具体判断
