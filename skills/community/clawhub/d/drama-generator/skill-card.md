## Description: <br>
Automated drama video generator from script to multi-character drama videos with OpenAI TTS, Whisper, and Remotion. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ZhenStaff](https://clawhub.ai/user/ZhenStaff) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and content creators use this skill to set up and run OpenClaw Drama Generator workflows that convert formatted scripts into multi-character narrated vertical drama videos. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The workflow can install or run external npm/GitHub code. <br>
Mitigation: Install only trusted, preferably pinned versions and run in an isolated project when evaluating the skill. <br>
Risk: Scripts and audio data may be sent to selected AI or TTS providers and can incur usage costs. <br>
Mitigation: Use dedicated provider API keys, monitor usage, and avoid confidential, regulated, or copyrighted scripts unless the provider terms are acceptable. <br>
