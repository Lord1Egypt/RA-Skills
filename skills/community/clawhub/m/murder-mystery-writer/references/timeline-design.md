# 时间线设计指南

## 为什么时间线是骨架

时间线是推理本的命脉。玩家会拿着秒表盘你的时间线，任何一个矛盾都会被揪出来。

## 设计步骤

### 第一步：确定案发时间轴

先定死三个时间点：
- **死亡时间**：死者确切死亡的时间（精确到分钟）
- **最后目击**：最后一个见到死者活着的时间
- **发现时间**：尸体被发现的时间

### 第二步：铺设凶手动线

凶手的时间线是最核心的，必须先设计好：

```
准备阶段 → 接近死者 → 行凶 → 处理现场 → 制造不在场证明 → 回到公共区域
```

每个环节都要有：
- 具体时间
- 具体地点
- 可能被谁目击
- 留下的痕迹

### 第三步：铺设其他角色动线

每个角色的时间线要做到：
- **有交叉**：至少与2个其他角色在不同时间点相遇
- **有空白**：每个角色都有"没人能证明"的时间段
- **有嫌疑**：空白时间段与案发时间重叠

### 第四步：制造时间迷雾

让时间线变得复杂的手法：

| 手法 | 示例 |
|------|------|
| 时间误报 | 角色A说19:30见到死者，实际是19:15 |
| 时钟异常 | 某个钟被调快/调慢了15分钟 |
| 记忆偏差 | 角色B记错了顺序，把19:00的事记成19:30 |
| 身份混淆 | 角色C把X认成了Y |
| 伪造时间 | 凶手故意制造的时间证据 |

## 时间线表格模板

### 基础版（适合4-6人本）

```markdown
| 时间 | 死者 | 角色A | 角色B | 角色C | 角色D | 角色E |
|------|------|-------|-------|-------|-------|-------|
| 18:00 | 位置 | 位置 | 位置 | 位置 | 位置 | 位置 |
| 18:15 | 位置 | 位置 | 位置 | 位置 | 位置 | 位置 |
| 18:30 | 位置 | 位置 | 位置 | 位置 | 位置 | 位置 |
| 18:45 | 位置 | 位置 | 位置 | 位置 | 位置 | 位置 |
| 19:00 | 死亡 | 位置 | 位置 | 位置 | 位置 | 位置 |
| 19:15 | - | 位置 | 位置 | 位置 | 位置 | 位置 |
| 19:30 | - | 位置 | 位置 | 位置 | 位置 | 位置 |
```

### 详细版（适合硬核推理本）

```markdown
| 时间 | 地点 | 人物 | 动作 | 目击者 | 留下痕迹 | 备注 |
|------|------|------|------|--------|----------|------|
| 18:47 | 书房 | 凶手 | 进入书房 | 无 | 脚印 | 从后门进入 |
| 18:52 | 书房 | 凶手 | 行凶 | 无 | 凶器指纹 | - |
| 18:55 | 书房 | 凶手 | 处理现场 | 无 | 擦拭痕迹 | - |
| 19:00 | 走廊 | 凶手 | 离开 | 角色B远距离看到 | 无 | B以为是死者 |
```

## 时间线校验清单

完成后逐项检查：

- [ ] 每个角色每分钟都有去向（不能有"我不知道"的空白）
- [ ] 凶手动线物理可行（从A到B需要多少时间，来得及吗）
- [ ] 目击证词之间不矛盾
- [ ] 死亡时间有多个证据支持（不能只有一个钟停了）
- [ ] 凶手有至少一种不在场证明策略
- [ ] 至少有2个角色的时间线有交叉（互相证明）
- [ ] 至少有2个角色的时间线有空白（制造嫌疑）
- [ ] 时间线中有至少1个"时间迷雾"因素（增加推理难度）
