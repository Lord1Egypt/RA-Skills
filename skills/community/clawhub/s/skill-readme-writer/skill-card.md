## Description: <br>
Generates professional README.md documentation for local or GitHub projects, using project inspection, interactive confirmation, badge generation, and Chinese, English, or bilingual output. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[JustZeroX](https://clawhub.ai/user/JustZeroX) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and project maintainers use this skill to draft or improve README.md files for local projects, GitHub repositories, OpenClaw skills, CLIs, libraries, APIs, and frontend applications. It supports an interactive workflow for confirming project details, selecting documentation style, choosing optional sections, and iterating on the generated README. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated README content may be inaccurate, incomplete, or misleading if project details are detected incorrectly. <br>
Mitigation: Review the detected project details and generated README before publishing. <br>
Risk: README generation may overwrite or replace useful existing documentation. <br>
Mitigation: Prefer README.new.md, a backup, or a diff review before overwriting an existing README. <br>
Risk: Bilingual output may be unnecessary or inconsistent with the intended publication style. <br>
Mitigation: Choose a single-language mode when bilingual documentation is not desired. <br>
