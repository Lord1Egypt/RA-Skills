## 一、正典概述

### 1.1 基督教正典的形成

**旧约正典的形成**
- 犹太传统分为三部分：律法书（Torah）→ 先知书（Nevi'im）→ 圣卷（Ketuvim），即 Tanakh
- 律法书（摩西五经）：公元前5世纪前已定型
- 先知书：约公元前2世纪前确定
- 圣卷：公元90年雅麦尼亚会议（Council of Jamnia）后基本确定
- 正典化过程：以色列民在历史中逐步辨识哪些书卷是神默示的

**新约正典的形成**
- 公元2-4世纪逐步形成共识
- 确认标准：使徒传承、正统教义、普世接受、崇拜使用
- 亚他那修（Athanasius）公元367年的节日书信列出27卷正典
- 希波会议（公元393年）和迦太基会议（公元397年）正式确认

## 二、旧约各卷详细概要

### 2.1 律法书（摩西五经）—— Torah

#### 创世记（Genesis）—— Bereshit（起初）

| 项目 | 内容 |
|------|------|
| 作者 | 摩西（传统观点） |
| 成书年代 | 约BC1446-1406（旷野时期） |
| 时间跨度 | 从创造到约瑟之死（约BC4000-BC1806） |
| 关键主题 | 创造、堕落、拣选、约、应许、信心 |
| 结构 | 上古史（1-11章）→ 族祖史（12-50章） |

**上古史（1-11章）**：
- 创造叙事（1-2章）：六日创造、安息日、人受造
- 堕落（3章）：始祖犯罪、原罪、"始祖福音"（3:15）
- 该隐与亚伯（4章）：第一起谋杀
- 挪亚与洪水（6-9章）：挪亚之约、彩虹为记
- 巴别塔（11章）：语言变乱、分散列国

**族祖史（12-50章）**：
- 亚伯拉罕之约（12、15、17章）：土地、后裔、万国得福
- 以撒（21-28章）：应许之子
- 雅各（25-36章）：以色列12支派之父
- 约瑟（37-50章）：从奴仆到宰相，救赎叙事

**关键经文**：1:1, 1:26-27, 3:15, 12:1-3, 15:6, 22:1-14, 50:20

#### 出埃及记（Exodus）—— Shemot（名字）

**结构**：出埃及（1-18章）→ 西奈之约（19-24章）→ 会幕指示（25-40章）

- 摩西的蒙召与10灾（3-12章）
- 逾越节（12章）—— 旧约核心救赎事件
- 过红海（14-15章）—— 米利暗之歌
- 十诫（20章）
- 约书（21-23章）
- 金牛犊事件（32章）
- 会幕建造（35-40章）

**关键经文**：3:14（"我是自有永有的"），12:1-28（逾越节），14:21-31（过红海），20:1-17（十诫）

#### 利未记（Leviticus）—— Vayikra（祂呼叫）

**结构**：献祭制度（1-7章）→ 祭司制度（8-10章）→ 洁净律法（11-15章）→ 赎罪日（16章）→ 圣洁法典（17-27章）

- 五种祭：燔祭、素祭、平安祭、赎罪祭、赎愆祭
- 赎罪日（Yom Kippur）：两只羊（一为赎罪、一为阿撒泻勒）
- 圣洁的核心命令："你们要圣洁，因为我耶和华你们的神是圣洁的"（19:2）
- 安息年与禧年（25章）

**关键经文**：19:18（"爱人如己"），19:2（"你们要圣洁"）

#### 民数记（Numbers）—— Bemidbar（在旷野）

**结构**：在西奈（1-10章）→ 往迦南（11-20章）→ 在摩押（21-36章）

- 12探子（13-14章）：信心失败导致40年旷野漂流
- 可拉叛变（16章）
- 摩西击石出水（20章）
- 铜蛇（21章）—— 约3:14引用
- 巴兰的预言（22-24章）

**关键经文**：6:24-26（大祭司祝福），21:8-9（铜蛇）

#### 申命记（Deuteronomy）—— Devarim（话语）

**结构**：摩西的三篇告别讲论（1-30章）→ 摩西临终（31-34章）

- 重申十诫（5章）
- 示玛（Shema）—— 6:4-9（以色列啊，你要听！）
- 祝福与咒诅（28章）
- 摩西之死（34章）

**关键经文**：6:4-9（示玛），30:19（"生与福，死与祸"）

### 2.3 诗歌智慧书

#### 约伯记（Job）

**主题**：义人为何受苦？
- 天上帷幕的对话（1-2章）
- 约伯与三友的辩论（3-31章）
- 以利户的发言（32-37章）
- 耶和华的回答（38-41章）
- 约伯的恢复（42章）

**关键经文**：1:21, 19:25-27（"我知道我的救赎主活着"），42:5-6

#### 诗篇（Psalms）

**共150篇，分为5卷**（对应摩西五经）

| 卷 | 篇数 | 特征 |
|----|------|------|
| 卷一 | 1-41 | 多以大卫为作者 |
| 卷二 | 42-72 | 可拉后裔的诗 |
| 卷三 | 73-89 | 亚萨的诗 |
| 卷四 | 90-106 | 摩西和无名诗 |
| 卷五 | 107-150 | 哈利路亚诗集 |

**类型**：赞美诗、哀歌（个人/群体）、感恩诗、智慧诗、君王诗、上行诗（120-134）、弥赛亚诗（2、22、110等）

**关键篇章**：1、2、19、22、23、24、32、51、90、100、103、110、119、121、139、150

#### 箴言（Proverbs）

**主题**：敬畏耶和华是智慧的开端（1:7）
**传统作者**：所罗门（1:1、10:1、25:1）、亚古珥（30章）、利慕伊勒（31章）
**内容**：智慧格言集，涵盖言语、财富、家庭、勤劳、管教、妇女等主题
**关键章节**：31:10-31（才德妇人）

#### 传道书（Ecclesiastes）—— Qoheleth（传道者）

**主题**：虚空的虚空——在日光之下的一切劳碌有何意义？
**关键经文**：1:2（"虚空的虚空，凡事都是虚空"），3:1-8（"凡事都有定期"），12:1（"记念造你的主"），12:13（"敬畏神，谨守祂的诫命"）
**解读**：在日光之下的有限人生中寻求终极意义，最终指向日光之上的神

#### 雅歌（Song of Songs / Song of Solomon）

**主题**：爱情之歌——夫妻之爱的颂歌
**解读层次**：
- 字面：男女之间的爱情诗歌
- 寓意（犹太传统）：神与以色列的关系
- 预表（基督教传统）：基督与教会的关系
- 非以所罗门为作者（学者争议）

### 2.4 大先知书

**结构**：
- 审判篇（1-39章）：对犹大和列国的审判

#### 耶利米书（Jeremiah）

#### 以西结书（Ezekiel）

## 三、新约各卷详细概要

### 3.1 福音书

#### 马太福音（Matthew）

| 项目 | 内容 |
|------|------|
| 传统作者 | 马太（税吏、十二使徒之一） |
| 成书年代 | 约AD50-70 |
| 特色 | 最多引用旧约——耶稣是新摩西、新律法的颁布者 |
| 结构 | 家谱（1:1-17）→ 叙事（1:18-4:25）→ 五大讲论（5-25章）→ 受难复活（26-28章） |

**五大讲论**：
1. 登山宝训（5-7章）—— 八福、主祷文
2. 宣教使命（10章）
3. 天国比喻（13章）
4. 教会生活（18章）
5. 橄榄山讲论（24-25章）

**关键经文**：5:3-12（八福），6:9-13（主祷文），11:28（"凡劳苦担重担的人"），16:16-18（彼得认信），28:18-20（大使命）

#### 马可福音（Mark）

| 项目 | 内容 |
|------|------|
| 传统作者 | 马可（彼得的门徒、巴拿巴的表弟） |
| 成书年代 | 约AD55-65（最早的福音书） |
| 特色 | 节奏最快，强调耶稣的行动，"立刻"出现41次 |
| 读者 | 罗马的外邦基督徒 |

**结构**：
- 公开事工（1-8章）
- 转折点：彼得认信与登山变像（8-9章）
- 走向耶路撒冷（10章）
- 受难周（11-16章）

**关键经文**：1:11（"你是我的爱子"），8:29（"你是基督"），10:45（"不是要受服侍，乃是要服侍人"），15:34（"以利以利拉马撒巴各大尼"）

**文本问题**：16:9-20的"长结尾"是否属于原稿（最古老抄本不含）

#### 路加福音（Luke）

| 项目 | 内容 |
|------|------|
| 传统作者 | 路加（医生、保罗的旅伴） |
| 成书年代 | 约AD58-65 |
| 特色 | 最详细的福音书，强调耶稣的人性与怜悯、妇女地位、穷人关注 |
| 读者 | 提阿非罗（外邦人） |

**独特内容**：
- 耶稣降生的详细叙事（1-2章）—— 牧羊人、西面、亚拿
- 好撒玛利亚人比喻（10:25-37）
- 浪子回头比喻（15:11-32）

## 四、主要神学术语词典

| 术语 | 定义 |
|------|------|
| 称义（Justification） | 神宣告罪人为义——基于基督的义归算给信徒 |
| 成圣（Sanctification） | 信徒被分别出来、越来越像基督的过程 |
| 得荣耀（Glorification） | 信徒在基督再来时身体复活得荣耀 |
| 赎罪（Atonement） | 基督的死平息了神的忿怒并赎回了罪人 |
| 替代性刑罚（Penal Substitution） | 基督替我们承受罪的刑罚（改革宗观点） |
| 和好（Reconciliation） | 神与人在基督里被恢复了关系 |
| 挽回祭（Propitiation） | 基督的死平息了神的忿怒（约壹2:2） |
| 预表（Type） | 旧约的某人物/制度/事件预指新约的基督 |
| 末世论（Eschatology） | 关于末后之事的教义体系 |
| 基督论（Christology） | 关于基督的位格与工作的教义 |
| 圣灵论（Pneumatology） | 关于圣灵的位格与工作的教义 |
| 教会论（Ecclesiology） | 关于教会的本质与使命的教义 |
| 启示（Revelation） | 神向人类自我揭示的行动 |
| 默示（Inspiration） | 圣灵保守圣经作者写下神的话语 |
| 无误（Inerrancy） | 圣经在原文上完全没有错误（持此立场的福音派观点） |
| 无谬（Infallibility） | 圣经在信心和道德教导上不会误导人 |
