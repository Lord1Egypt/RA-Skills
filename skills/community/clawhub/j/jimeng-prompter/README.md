# 即梦 Prompt 大师 — 全栈 AI 提示词引擎

> 面向即梦全栈（Seedance 2.0 视频 + Seedream 4.x/5.0 图片）的提示词专家。不会写即梦提示词？你来创意，我来翻译成即梦能懂的专业语言。
> v2.0 新增：智能模型路由、参考图协作、时间线分段、运镜词库、情感转译、批量策略。

## ✨ 核心理念

即梦已发展为完整的双线平台——**图片生成 Seedream**（海报/电商/人像/数据图）和 **视频生成 Seedance 2.0**（短片/广告/叙事），每条线多个版本对提示词的偏好完全不同。

v2.0 的核心升级：用户不需要懂这些版本差异。Director 自动诊断意图 → 路由至正确模型 → Prompt Crafter 输出该模型的专属优化提示词。

| 角色 | 职责 |
|---|---|
| 🎬 **Director** | 模型路由、渐进式需求挖掘、参考图采集、全局风格锁定义、方案预演 |
| 🛠️ **Prompt Crafter** | 按模型版本输出专属提示词、时间线分段、批量变体、负向约束 |

**导演不写词，匠人不改设计。**

## 🆕 v2.0.0 升级 (2026-05-15)

基于 30+ 网页深度调研的全面升级：

### 🧭 Phase 0: 智能模型路由（全新）
- 自动识别用户意图 → 路由至 Seedance 2.0（视频）或 Seedream 4.0/4.5/5.0 Lite（图片）
- 图片流程追问场景类型（海报/纯画面/数据图），精确匹配最佳模型版本
- 图片 vs 视频提示词公式自动切换

### 📸 参考图协作流（全新）
- Phase 2 引导用户提供素材（角色照片/场景图/运镜参考视频）
- 自动适配即梦 @ 引用语法
- 告诉用户参考图对画面稳定度提升 50%+

### 🎞️ 时间线分段模板（全新）
- 中长分镜（≥10 秒）默认输出时间线分段
- 秒级精度：每段 [起始秒-结束秒] [画面] [镜头] [音效]
- 大幅提升长视频连贯性和稳定性

### 🎥 运镜词库标准化（全新）
- 内化 14 种专业运镜术语（推拉摇移环绕 POV 等）
- 用户说"镜头靠近" → 输出 "slow dolly in"
- 组合规则约束：一次最多 2-3 种运镜

### 😢 情感→可画面化转译（全新）
- 12 种情绪词各附 3-5 个微动作转译
- "孤独" → "独自坐在长椅上，手指摩擦杯沿，不看任何方向"
- 消除模糊词，让 AI 有具体画面可渲染

### 🎲 批量探索策略（全新）
- 图片流程：主动建议 n=3 变体快速探索
- 视频流程：同样提示词多生成 3 次挑最佳
- 探索 vs 生产两种模式分离

### 🛡️ 增强负向黑名单
- 从纯英文 → 中英文双轨
- 追加：禁止角色变脸、禁止突然偏色、禁止新增无关人物、禁止光线突变等

### 📋 图片流程完整体系（全新）
- Seedream 4.0/4.5/5.0 Lite 三版本专属提示词格式
- 文字渲染技巧（引号包裹、位置指定、字体风格）
- 批量参数建议表（n / size / web_search）
- 5 种常见错误+修复

### 流水线架构

```
用户输入
    ↓
Phase 0: Director 模型路由（视频 or 图片？哪个版本？）
    ↓
Phase 1: Director 渐进式需求挖掘（一次一问，最多5轮）
    ├── 同时捕获情感词 → 转译表
    └── Phase 2 前置：参考图采集 Q
    ↓
Phase 2: Director 方案预演 + 强制等待确认
    ↓
Phase 3: Prompt Crafter 生产级输出
    ├── [视频] 全局反向提示词 + 风格锁 + 时间线分段分镜表 + 运镜术语 + 概念图 + 批量建议
    └── [图片] 参数建议 (比例/分辨率/n) + 正向提示词 + 文字排版 + 批量策略
```

### 全覆盖能力矩阵

| 场景 | 模型路由 | 输出规格 |
|------|---------|---------|
| AI 短片/抖音 | Seedance 2.0 | 分镜表 + 时间线 + 运镜 + 概念图 |
| 海报/封面/广告 | Seedream 4.5 | 正向词 + 文字渲染 + 4K 建议 |
| 人像/风景/插画 | Seedream 4.0 | 正向词 + 风格锁定 + 批量变体 |
| 数据可视化 | Seedream 5.0 Lite | 正向词 + 逻辑关系 + web_search |

### 8 大避坑规范（视频）+ 5 大避坑规范（图片）

| 陷阱 | 规避 |
|---|---|
| 人物崩坏 | 画面 ≤2 人 + `anatomically correct hands` |
| 画风突变 | 全局风格锁每镜强制粘贴 |
| 多人错乱 | ≥3 人用远景 |
| 动作鬼畜 | 静态瞬间代替动态动词 |
| 文字乱码 | 不生成画面文字，后期剪映加 |
| 面部崩坏 | 近景加 `symmetrical face` |
| 夜景噪点 | 加 `well-lit night scene` |
| 角色不一致 | 风格锁含主角固定特征 |
