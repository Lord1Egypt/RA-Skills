## 💡 核心框架

### 1. 成本结构分析

在亚马逊卖货，你的成本结构如下：

```
总收入 = 售价 × 销量

成本项：
├── 商品成本（Cogs）
│   ├── 产品生产成本
│   ├── 包装成本
│   └── 国内运费到仓
├── 平台费用
│   ├── 推荐费（Referral Fee）：类目不同 6-15%
│   ├── FBA费用（如果使用）
│   │   ├── 仓储费
│   │   └── 配送费
│   └── 月度订阅费（如果专业卖家）
├── 营销费用
│   ├── 广告费
│   ├── Coupon/Prime Discount
│   └── 测评/站外推广
├── 运营成本
│   ├── 人工
│   ├── 工具/SaaS
│   └── 退款/退货
└── 其他
    ├── 汇率损耗
    └── 资金成本
```

### 2. 盈亏平衡点计算

**盈亏平衡售价** = 总成本 ÷ (1 - 推荐费率) + FBA相关费用

**计算示例**：
- 商品成本：$8
- 推荐费率：15%（3C电子类）
- FBA配送费：$3.5
- FBA仓储费：$0.5
- 期望利润率：20%

```
盈亏平衡价 = ($8 + $3.5 + $0.5) ÷ (1 - 15%) = $12 ÷ 0.85 = $14.12

包含利润的定价 = $14.12 ÷ (1 - 20%) = $17.65
```

### 3. 同行价格分析框架

**重要**：我们不追踪同行价格，我们**参考**同行价格来制定策略。

```
Step 1: 找到价格区间
   低价区（<25%分位）：通常是新入场者或清仓
   中价区（25%-75%分位）：市场主流价格
   高价区（>75%分位）：品牌溢价或差异化产品

Step 2: 分析价格分布
   如果价格集中在某一区间 → 这个区间是市场共识
   如果价格分散 → 市场尚未形成定价标准

Step 3: 结合评分分析
   高评分 + 高价格 = 品牌/品质溢价
   低评分 + 低价格 = 走量策略
   高评分 + 低价格 = 杀手级产品（小心！）

Step 4: 制定你的定价策略
   - 跟随策略：跟随中价区，稳定为主
   - 差异化策略：高于中价区，强调价值
   - 竞争策略：低于中价区，抢占份额（慎用！）
```

### 4. 定价策略矩阵

| 策略 | 适用场景 | 风险 | 利润率预期 |
|------|----------|------|------------|
| 渗透定价 | 新品类，需要教育市场 | 高（可能回不到高价） | 低 |
| 竞争定价 | 红海市场，必须争夺份额 | 中（价格战两败俱伤） | 偏低 |
| 价值定价 | 有差异化，能讲清故事 | 低 | 高 |
| 动态定价 | 标品，价格波动大 | 中（需要工具支持） | 波动 |
| 套餐定价 | 有捆绑空间，配件丰富 | 低 | 高 |

### 5. 促销定价计算

**Prime Day/黑五定价公式**：

```
促销价 = 平时价 × (1 - 折扣率)

折扣率参考：
- Prime Day：15-30% 较常见
- 黑五网一：20-40% 较常见
- 会员专享：10-20% 较常见
- Lightning Deal：25-50% 较常见

⚠️ 注意：计算时必须确保促销价 > 盈亏平衡价！
```

**优惠券叠加计算**：
```
实际到手价 = 标价 - 折扣 - Coupon - Prime Discount

例：标价 $30，折扣 20%，Coupon $2，Prime 5%
= $30 - $6 - $2 - $1.5 = $20.5
```

### 6. 利润优化方向

**提升利润的5个杠杆**：

| 杠杆 | 操作方向 | 提升空间 |
|------|----------|----------|
| 降低商品成本 | 谈判、换供应商、批量采购 | 5-20% |
| 优化包装 | 轻量化、简化包装 | 3-10% |
| 调整类目 | 移到推荐费率更低的类目 | 1-5% |
| 优化广告 | 降低ACOS，提升ROAS | 5-15% |
| 提升售价 | 配合产品升级或品牌化 | 10-50% |

### 多市场定价策略

1. **汇率加成法**：基础价 × 汇率 × (1 + 溢价率)
   - 适用于品类单一、规格统一
2. **市场对标法**：参考本地竞品价格
   - 适用于本地化需求强
3. **成本加成法**：成本 + 固定利润
   - 适用于成本占比高的品类

## ⚠️ 重要原则

1. **永远先算成本，再定价格** - 不要看着同行定，价格低于成本就是亏
2. **利润比销量重要** - 亏本冲排名是饮鸩止渴
3. **价格不是唯一武器** - 差异化、评价、服务都是竞争力
4. **旺季定价要有节奏** - 提前规划，不要临时抱佛脚
5. **定期复盘定价** - 市场在变，你的定价也要变
