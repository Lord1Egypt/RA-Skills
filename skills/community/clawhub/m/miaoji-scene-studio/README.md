# 秒技·原生场景图工作室

> 1张白底图 → 秒级生成TikTok原生感场景图提示词

## 核心价值

将亚马逊/TikTok Shop的白底产品图，转化为具有平台原生感的场景图AI绘图提示词，帮助用户提升内容点击率和转化率。

## 功能一览

| 功能 | 说明 |
|------|------|
| 场景图提示词生成 | 输入产品描述，秒级输出中英文AI绘图提示词 |
| 平台适配模板 | TikTok/Instagram/小红书 场景风格差异 |
| 内容偏好适配 | 适配平台内容偏好的视觉策略 |
| 多风格测试方案 | 同一产品3种场景风格对比 |

## 快速开始

```
用户：帮我生成[产品名]的场景图提示词
助手：自动输出场景方案和AI绘图提示词
```

## 使用示例

**输入**：
> 防晒帽的场景图提示词

**输出**：
- 中文场景描述
- 英文AI绘图提示词
- 构图方案（光线/角度/占比）
- 平台适配度评分
