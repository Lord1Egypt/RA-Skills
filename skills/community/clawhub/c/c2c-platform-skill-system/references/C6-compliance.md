## 步骤6.1 内容审核体系 (C6-01 to C6-02)

### C6-01 条目内容审核

- **输入**: 服务商提交的条目数据（标题/描述/价格/服务时长/服务区域/技能标签）、条目发布规范(C5-15)、违禁词库、品类白名单、价格异常阈值（同品类均价偏离±50%）、历史违规记录
- **输出**: 审核结果（通过/驳回+原因/待人工复审），驳回原因分类标签（违禁词/价格异常/虚假宣传/信息不完整/品类不符/图片违规），自动审核覆盖率≥80%
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C7-01(入驻审核条目质量维度)、C7-17(履约质量条目一致性检查)

### C6-02 图片内容审核

- **输入**: 条目配图/服务商相册/用户评价图片/履约过程照片、图片审核规则（涉黄/涉暴/涉政/侵权/广告/二维码/联系方式）、OCR提取文本、人脸检测（核实人证一致性）
- **输出**: 图片审核结果（通过/驳回+原因/待人工复审），涉黄/涉暴置信度分数，OCR违规关键词高亮，违规图片自动打码或拦截
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C6-01(条目审核图片维度)、C6-09(审核制度图片规则细化)

- **输入**: 交易业务模型（C2C撮合→订单→支付易点→履约→完成码确认→结算）、平台角色定位（信息中介/非交易当事方）、责任边界条款、免责条款、争议解决条款、保证金/备付金条款、ICP/EDI许可范围声明
- **输出**: 交易服务协议（协议主体→服务范围→双方权利义务→交易规则→支付规则→违约责任→争议解决→其他条款），明确平台不参与交易本身、仅提供信息撮合与技术服务的法律定位
- **依赖**: 无
- **AI自治度**: 🟨
- **组合接口**: → C6-06(二清合规协议依据)、C6-25(协议更新联动)

### C6-27 传销风险防控(3级上限)

- **输入**: 3级分销模型（L1:70%/L2:30%/L3:0%）、分销层级数据（关系链深度/广度）、传销风险特征（层级>3/团队计酬/入门费/拉人头）、分润规则说明(C5-18)、禁止行为清单、监管判定标准
- **输出**: 传销风险防控方案（3级硬限制技术实现→关系链深度实时监控→异常分润模式检测→违规处置规则→合规声明），含技术保障（数据库关系链查询≤3层、前端展示≤3级、分润计算≤3级）
- **依赖**: 无
- **AI自治度**: 🟨
- **组合接口**: → C6-13(反欺诈传销场景规则)、C7-07(信用分传销违规扣分)

### C6-28 保证金/备付金管理

- **输入**: 服务商保证金缴纳标准（按品类/等级差异化）、备付金计算逻辑（未结算订单金额/未提现易点余额/分润待发金额）、平台佣金1%计提、资金安全要求（专户管理/不挪用/不混同）、监管要求
- **输出**: 保证金/备付金管理制度（缴纳标准→专户管理→使用规则→退还条件→监管报告），含资金安全承诺书模板
- **依赖**: 无
- **AI自治度**: 🟨
- **组合接口**: → C6-06(二清合规资金安全依据)、C7-01(入驻审核保证金维度)

## 步骤6.9 合规运营支撑 (C6-29 to C6-33)

### C6-29 纠纷举证材料整理

- **输入**: 争议仲裁案件(C6-11)、订单全链路数据（下单→支付→履约→完成码确认→评价）、服务商资质与历史记录、服务过程证据（照片/GPS轨迹/录音）、易点交易记录、3级分销关系是否涉及利益关联
- **输出**: 纠纷举证材料包（证据清单→材料组织→时间线还原→法律意见→提交格式），支持仲裁/诉讼/监管调取三种场景
- **依赖**: C6-11
- **AI自治度**: ⬛
- **组合接口**: → C6-30(迎检举证材料)、C6-33(事件复盘证据)

### C6-30 监管检查迎检准备

- **输出**: 迎检准备材料包（检查项清单→对应材料索引→自查报告→整改台账→人员分工），支持网信办/工信部/市场监管局/人民银行四类检查
- **依赖**: C6-19
- **AI自治度**: 🟨
- **组合接口**: → C6-33(合规事件复盘迎检改进)

### C6-31 合规培训材料制作

- **输入**: 内容审核制度(C6-09)、用户隐私保护合规要求(C6-07)、AML规则(C6-22)、传销防控方案(C6-27)、数据安全等级(C6-20)、反欺诈案例(C6-14/C6-15)
- **输出**: 合规培训材料（课件+案例集+测试题），分角色版本：运营合规培训/客服合规培训/技术人员合规培训/管理层合规培训

- **输入**: 风控规则引擎配置(C6-17)、风控决策数据（拦截量/拦截率/误杀率/资损率/申诉通过率）、反欺诈识别数据(C6-14/C6-15)、交易合规数据(C6-05)、AML监控数据(C6-23)

- **依赖**: C6-17
- **AI自治度**: ⬛
