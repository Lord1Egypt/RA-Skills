## 地支关系

### 六合（地支六合）
子丑合土、寅亥合木、卯戌合火、辰酉合金、巳申合水、午未合日月（土）。

含义：合代表融洽、合作、结合，在占卜中多为吉兆，但需看具体的五行喜忌。

### 三合
- 申子辰 = 水局
- 亥卯未 = 木局
- 寅午戌 = 火局
- 巳酉丑 = 金局

三合成局力量更大，代表同类力量的汇聚。

### 六冲
子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲。

冲代表冲突、变动、分离，在占卜中多为变动之兆。

### 六害（穿）
子未害、丑午害、寅巳害、卯辰害、申亥害、酉戌害。

害代表暗中的损害、小人作祟。

### 三刑
- 无礼之刑：子刑卯、卯刑子
- 恃势之刑：寅刑巳、巳刑申、申刑寅
- 无恩之刑：丑刑戌、戌刑未、未刑丑
- 自刑：辰、午、酉、亥自刑

刑代表刑罚、冲突、纠纷。

## 旬空速算

六十甲子每十日为一旬，每旬有两地支轮空：

| 旬首 | 空亡地支 |
|------|----------|
| 甲子 | 戌亥 |
| 甲戌 | 申酉 |
| 甲申 | 午未 |
| 甲午 | 辰巳 |
| 甲辰 | 寅卯 |
| 甲寅 | 子丑 |

速算口诀：空亡地支 = 旬首地支向前推两位。
