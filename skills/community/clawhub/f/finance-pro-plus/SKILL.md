# 财务顶级强化包 (Finance Pro+)

## 概述
本技能包为 OpenClaw 提供企业级财务分析能力，让 AI 从"回答财务问题"升级为"执行完整财务工作流"。

## 核心能力矩阵

| 模块 | 能力 | 适用场景 |
|------|------|----------|
| 财报解析 | 三表（资产负债表、利润表、现金流量表）深度解读 | 投资分析、尽调 |
| 杜邦分析 | ROE 五因素拆解，盈利能力/运营效率/财务杠杆分析 | 业绩评估 |
| 风险识别 | 财务造假识别、债务风险、现金流断裂预警 | 风险管控 |
| 报告生成 | 自动生成专业财务分析报告 | 工作汇报 |
| 估值建模 | DCF、相对估值法、可比公司分析 | 投资决策 |
| 预算管理 | 预算编制、执行监控、差异分析 | 经营管理 |
| 税务管理 | 税负分析、税务筹划、合规检查 | 财税工作 |

## 执行流程 (Steps)

### 【标准财报分析流程】

```
第一步：数据收集与结构化
├── 识别财报类型（季报/半年报/年报）
├── 提取三表关键数据
├── 计算同比/环比变化
└── 建立分析框架

第二步：盈利能力分析
├── 营收增长率、毛利率、净利率
├── 营业利润 vs 归母净利润
├── 非经常性损益剥离
└── 趋势判断（3-5年）

第三步：资产质量分析
├── 应收账款周转天数
├── 存货周转天数
├── 固定资产周转率
├── 资产负债率、产权比率
└── 资产结构合理性

第四步：现金流健康度
├── 经营现金流 / 净利润 比值
├── 自由现金流计算
├── 筹资现金流分析
└── 现金流断裂风险评估

第五步：杜邦拆解（核心）
├── ROE = 净利率 × 资产周转率 × 权益乘数
├── 拆解五因素：净利率/毛利率/周转率/杠杆/税负
├── 与行业平均对比
└── 找出关键驱动因素

第六步：风险识别
├── 财务造假红旗信号
├── 关联交易异常
├── 商誉减值风险
├── 债务到期压力
└── 经营风险暴露

第七步：综合评级与建议
├── 给出财务评分（1-10分）
├── 核心优势/主要风险
└── 投资或合作建议
```

### 【DCF估值流程】

```
第一步：历史数据分析
├── 营收/利润 5年历史
├── 盈利能力趋势
└── 现金流稳定性

第二步：关键假设
├── 未来5年营收增长率（保守/中性/乐观）
├── 营业利润率假设
├── 永续增长率（通常 2-4%）
└── 折现率 WACC（8%-15%）

第三步：现金流预测
├── FCF = 净利润 + 折旧摊销 - 资本支出 - 营运资本增加
├── 分三阶段预测
└── 终值计算

第四步：估值输出
├── 股权价值 = 预测现金流现值 + 终值现值 - 净负债
├── 每股价值
└── 与当前市值/同行对比
```

### 【财务尽调流程】

```
第一步：基础筛查
├── 工商信息核实
├── 股权结构分析
├── 实际控制人识别
└── 关联方梳理

第二步：报表可信度验证
├── 三表一致性检查
├── 科目异常波动分析
├── 勾稽关系验证
└── 审计意见解读

第三步：核心风险点识别
├── 虚增收入迹象
├── 利润操纵识别
├── 资产质量问题
├── 债务隐藏手法
└── 现金流虚胖

第四步：债务债权核查
├── 有息负债明细
├── 对外担保情况
├── 民间借贷/隐性债务
├── 应收账款质量
└── 诉讼及或有负债

第五步：尽调报告
├── 企业画像
├── 财务健康度评级
├── 风险点清单
└── 合作建议
```

## 分析框架库

### 杜邦分析模型

```
┌─────────────────────────────────────────────────────────────┐
│                        净资产收益率 ROE                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐              │
│  │ 净利率   │ ×  │ 资产周转 │ ×  │ 权益乘数 │              │
│  │ (盈利)   │    │  (运营)  │    │  (杠杆)  │              │
│  └────┬─────┘    └────┬─────┘    └────┬─────┘              │
│       │               │               │                     │
│       ▼               ▼               ▼                     │
│  ┌─────────────────────────────────────────┐               │
│  │           ROE = 净利润/营业收入           │               │
│  │               × 营收/总资产               │               │
│  │               × 总资产/净资产             │               │
│  └─────────────────────────────────────────┘               │
│                                                             │
│  五因素拆解：                                               │
│  ① 毛利率 = 毛利 / 营收                                     │
│  ② 净利率 = 净利润 / 营收                                   │
│  ③ 资产周转率 = 营收 / 总资产                               │
│  ④ 权益乘数 = 总资产 / 净资产                               │
│  ⑤ 税前利润率 = 税前利润 / 营收                             │
│                                                             │
│  ROE = 毛利率 × 净利率/毛利率 × 资产周转率                  │
│      × 权益乘数 × 税前利润率/净利率                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 财务造假红旗信号清单

```
【收入类红旗】
☐ 营收大幅增长但经营现金流持续为负
☐ 应收账款增速远超营收增速
☐ 第四季度营收占比异常高
☐ 毛利率异常高于行业平均
☐ 客户集中度极高或突然变化
☐ 频繁变更收入确认政策

【利润类红旗】
☐ 非经常性损益占比过高
☐ 经常性利润与经营现金流严重背离
☐ 存货和固定资产突然大幅增加
☐ 费用资本化比例异常
☐ 计提减值准备不足或突然转回

【资产类红旗】
☐ 商誉占净资产比例超过50%
☐ 在建工程长期不转固
☐ 应收账款账龄结构恶化
☐ 存货周转天数突然延长
☐ 固定资产更新率过低

【现金流类红旗】
☐ 经营现金流长期低于净利润
☐ 筹资活动现金流依赖严重
☐ 应收账款回收异常缓慢
☐ 预付款项大幅增加

【关联交易红旗】
☐ 关联交易占比过高
☐ 关联销售定价异常
☐ 资金往来频繁且隐蔽
☐ 担保/抵押异常
```

## 输出规范

### 分析深度分级
- **L1 快速扫描**：5分钟内输出核心指标和风险提示
- **L2 标准分析**：包含杜邦拆解、风险评级、趋势判断
- **L3 深度尽调**：完整报告，包含估值、风险点清单、合作建议

### 数据呈现规范
- 金额统一用"亿元"或"万元"，保留2位小数
- 百分比保留1位小数
- 增长率用"+/-XX%"表示
- 时间标注用"YYYY-MM-DD"格式

### 判断标准
| 指标 | 优秀 | 良好 | 及格 | 较差 |
|------|------|------|------|------|
| 毛利率 | >40% | 20-40% | 10-20% | <10% |
| 净利率 | >20% | 10-20% | 5-10% | <5% |
| ROE | >20% | 10-20% | 5-10% | <5% |
| 经营现金流/净利润 | >1.2 | 0.8-1.2 | 0.5-0.8 | <0.5 |
| 资产负债率 | <40% | 40-60% | 60-80% | >80% |

## 注意事项 (Notes)

1. **数据来源**：优先使用上市公司公告、审计报告等官方披露数据
2. **非上市公司**：财务数据获取受限，分析时需标注数据局限性
3. **造假识别**：发现红旗信号时，应交叉验证多数据源
4. **估值校准**：DCF结果应与可比公司法交叉验证
5. **免责声明**：财务分析仅供决策参考，不构成投资建议
