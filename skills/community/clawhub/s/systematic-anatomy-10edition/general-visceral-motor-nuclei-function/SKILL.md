# 一般内脏运动核的功能与支配关系

## 适用场景
- 解释瞳孔收缩、晶状体调节、唾液腺/泪腺分泌或胸腹腔脏器活动的神经调控基础
- 分析涉及动眼神经、面神经、舌咽神经或迷走神经的副交感功能障碍
- 教学或临床中需明确副交感节前与节后纤维通路时

## 执行步骤
1. **确认目标效应器类型**：判断涉及的是瞳孔括约肌/睫状肌、唾液腺（腮腺/下颌下腺/舌下腺/泪腺）还是胸腹腔内脏。
2. **匹配对应的一般内脏运动核**：
   - 瞳孔与晶状体 → 动眼神经副核
   - 泪腺、下颌下腺、舌下腺 → 上涎核
   - 腮腺 → 下涎核
   - 颈、胸、腹腔大部分脏器 → 迷走神经背核
3. **追溯纤维路径**：
   - 节前纤维从中枢核团发出，加入相应脑神经
   - 在周围神经节（如睫状神经节、翼腭神经节、下颌下神经节、耳神经节或器官旁/内节）换元
   - 节后纤维支配效应器
4. **验证前提条件**：确保副交感通路完整且不涉及躯体运动、感觉或交感系统。

> 注意：本技能仅适用于副交感神经系统，不处理交感或躯体神经功能。
