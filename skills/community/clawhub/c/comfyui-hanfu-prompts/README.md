## 技术栈
- **模型**: Z-Image Turbo (DiT)
- **工作流**: 双采样 (768×1024→1024×1536, 6步结构+4步细节)
- **工具**: ComfyUI + comfy-batch
- **硬件要求**: RTX 5070 12GB+

## 使用方法
```bash
pip install comfy-batch
python -m comfy_batch "你的提示词" --double
```
