# Context Segment Template

A context segment documents one concept in the project's shared language. Multiple segments compose into CONTEXT.md.

```markdown
## [Concept Name]

**Definition:** One sentence explaining what this is.

**Why it matters:** [Why this concept deserves its own name.]

**Usage in code:**
- Variables, functions, types should use this name
- Related concepts: [list related terms]

**Examples:**
- Good: [correct usage]
- Bad: [misuse to avoid]
```
