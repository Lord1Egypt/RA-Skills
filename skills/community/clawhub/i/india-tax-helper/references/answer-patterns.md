# Answer patterns

Use a two-layer answer.

## Layer 1: short answer
Keep this to 3-7 bullets.

Recommended shape:
- what this thing is
- whether it likely applies to the user
- what they should do next
- what document/source is needed
- whether anything is uncertain

## Layer 2: detailed memo
Expand only when useful.

Recommended sections:
1. What this is
2. Why it matters
3. What usually happens in the salaried lifecycle
4. What the user should check next
5. What can go wrong if ignored
6. Missing info / source caveat

## Pattern: IT declaration / Form 12BB
Layer 1 should cover:
- declaration vs proof submission distinction
- whether it affects payroll TDS
- what broad categories go in it
- what happens if missed

## Pattern: ITR filing
Layer 1 should cover:
- whether the user likely still needs to file
- what documents usually matter
- whether refund/additional tax is plausible
- what happens if they miss filing

## Pattern: FD/RD / 15G / 15H
Layer 1 should cover:
- whether interest is generally taxable
- what 15G/15H is conceptually
- whether age/estimated total income matter
- whether exact eligibility is blocked pending facts/FY verification

## Pattern: Capital gains / dividends
Layer 1 should cover:
- asset type
- whether holding period matters
- whether exact tax treatment needs gain/holding/FY details
- what statement/document to fetch next

## Pattern: loan benefit questions
Layer 1 should cover:
- whether the question is about declaration, deduction, or filing
- what certificate/statement is needed
- whether the treatment is regime-sensitive
