# 子宫颈鳞状细胞癌组织学分期

## 适用条件
- 已经病理确诊为子宫颈鳞状细胞癌
- 癌细胞明确突破基底膜
- 需在显微镜下测量浸润深度
- 不适用于腺癌或其他非鳞癌类型

## 执行步骤
1. **确认浸润存在**：识别不规则癌细胞巢或条索侵入固有层间质。
2. **测量浸润深度**：从基底膜至癌巢最深浸润点的垂直距离（单位：mm）。
   - 若 **浸润深度 ≤ 5mm** → 诊断为“早期浸润癌”（微小浸润性鳞状细胞癌）。
   - 若 **浸润深度 > 5mm** → 诊断为“浸润癌”。
3. **评估分化类型**（仅对浸润癌）：
   - **角化型鳞癌**：可见角化珠或细胞内角蛋白。
   - **非角化型鳞癌**：无明显角化特征。
4. **输出结果**：包括分期（早期浸润癌 / 浸润癌）及分化类型（如适用），用于指导治疗方案选择。

> 注意：早期浸润癌预后良好，常可行保守手术；精确测量对避免过度治疗至关重要。
