# dacs-archival-finding-aid-drafter

Draft a DACS (Describing Archives: A Content Standard, 2nd edition, technical revision 2022) single-level-minimum-compliant finding aid for a processed archival collection, in the structure that an ArchivesSpace resource record expects: collection-level description (title, dates, extent, creator, biographical / historical note, scope and contents, arrangement, conditions governing access and use, languages, immediate source of acquisition, processing information, preferred citation, related materials) and an optional series-level container list.

## When to use

A practising archivist — processing archivist, accessions / acquisitions archivist, head of collections, special-collections librarian, university or institutional records archivist, manuscripts curator, audiovisual archivist, or community-archives volunteer — has:

- a processed (or partly processed) archival collection or accession,
- the collection-level facts (creator, dates, extent, restrictions, provenance, processing decisions), and
- needs a DACS-compliant finding aid ready to enter into ArchivesSpace, ArchivesEAS, AtoM, ArchivEra, or an equivalent collections-management system, or to render as PDF / EAD / HTML.

The skill does **not** perform original arrangement, do hands-on processing, make appraisal decisions, perform copyright clearance, or upload to the access system. It produces a DRAFT finding aid for the processing archivist and head of collections to review before publication.

## What it produces

- Collection title following the repository's documented naming convention (creator + form-of-materials term)
- Date statement (inclusive dates, bulk dates if applicable; predominant or single date when appropriate)
- Extent statement (linear feet / metres / cubic feet, plus item-count breakdown by material type when material types differ)
- Creator / Name of Creator(s) with authorised-form note (LCNAF / VIAF / locally authorised)
- Biographical note (for personal creators) or Historical / Administrative note (for corporate creators)
- Scope and contents note (overview of subjects, formats, time span, significant correspondents / events)
- Arrangement note (order imposed by the archivist, with series and subseries listing)
- Conditions Governing Access note (any restrictions: time-limited, donor-imposed, statutory, format-based, redacted material)
- Conditions Governing Use note (copyright holder of record; rights statement template — RightsStatements.org or institutional)
- Languages of Materials
- Immediate Source of Acquisition note (donor / source, year, accession number — confidential donor terms flagged)
- Processing Information note (processor, dates, level, intervention, refoulement / deaccession decisions, anti-oppressive review notes)
- Preferred Citation
- Related Materials (in this repository; in other repositories)
- Separated Materials (deaccessioned, transferred, returned to donor)
- Optional series-level container list with box / folder inventory in DACS Chapter 1 multilevel form
- ArchivesSpace-mappable field labels and EAD3 element correspondences

## Inputs the agent will ask for

One question at a time:

1. Repository name and naming-convention guidance (RAC-style "records / papers / collection," LC pattern, institution-local)
2. Collection name and creator(s) — with authorised-form choice (LCNAF / VIAF / local NAF)
3. Inclusive dates (and bulk dates if a clear bulk exists)
4. Total extent (linear feet / metres / cubic feet) and any item-count breakdown by material type
5. Processing level (DACS single-level minimum, intermediate, full / item-level, MPLP collection-level only)
6. Processor name, processing date range, and supervising archivist
7. Source of acquisition (donor / transfer / purchase / found-in-collection), year, accession number, and any donor-imposed restrictions
8. Repository's chosen rights-statement framework (RightsStatements.org / Creative Commons / local) and copyright status the archivist has determined
9. Access restrictions (open, restricted until [date], donor-permission-required, FERPA / HIPAA / equivalent, redaction policy)
10. Languages present
11. Arrangement decision (provenance / original order / imposed by archivist; series and subseries titles and date spans)
12. Notable subjects, correspondents, events, and any culturally sensitive content requiring a content advisory or reparative-description treatment
13. Related materials in this repository or elsewhere
14. Whether to draft the optional series-level container list (and the box / folder inventory if so)
