# 基于MCV和RDW的贫血形态学分类法

## 适用场景
- 患者确诊为贫血
- 已获取包含MCV和RDW的完整血常规报告
- 需对贫血类型进行初步病因推断

## 执行步骤
1. **确认前提条件**：
   - 血常规数据完整，包含MCV（参考区间：82～100 fL）和RDW-CV（参考区间：11.5%～15.1%）
   - 排除仪器干扰或样本溶血等技术因素

2. **根据MCV与RDW组合判断贫血类型**：
   - **MCV增高 + RDW正常** → 大细胞均一性贫血（如部分再生障碍性贫血）
   - **MCV增高 + RDW增高** → 大细胞非均一性贫血（如巨幼细胞贫血、MDS）
   - **MCV正常 + RDW正常** → 正常细胞均一性贫血（如急性失血性贫血）
   - **MCV正常 + RDW增高** → 正常细胞非均一性贫血（如再生障碍性贫血、PNH、G6PD缺乏症）
   - **MCV减低 + RDW正常** → 小细胞均一性贫血（如珠蛋白生成障碍性贫血、遗传性球形红细胞增多症）
   - **MCV减低 + RDW增高** → 小细胞非均一性贫血（典型见于缺铁性贫血）

3. **关键鉴别应用**：
   - 区分**缺铁性贫血**与**轻型β珠蛋白生成障碍性贫血**：两者均为小细胞低色素性贫血，但前者RDW增高（>15.1%），后者约88%患者RDW正常
   - **疗效监测**：缺铁性贫血治疗后即使血红蛋白恢复正常，若RDW仍升高，提示体内贮存铁尚未补足

4. **重要提醒**：
   - 本分类法不适用于混合性贫血难以归类的情况
   - 必须结合传统三参数（MCV、MCH、MCHC）及血涂片形态观察进行综合判断，避免误诊

## 输出建议
根据MCV/RDW组合结果，推荐下一步检查方向，例如：
- RDW↑ + MCV↓ → 建议检测铁代谢指标（血清铁、铁蛋白、总铁结合力）
- RDW正常 + MCV↓ → 建议行血红蛋白电泳以排除地中海贫血
