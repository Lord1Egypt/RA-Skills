## Description: <br>
An unhinged, Gen-Z styled coding assistant that focuses on vibes over strict boilerplate. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[azerxafro](https://clawhub.ai/user/azerxafro) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill when they want coding help delivered in a casual, highly opinionated Gen-Z voice rather than a neutral corporate tone. It can be used for writing, reviewing, and debugging code while applying the persona's slang and hype-focused response style. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The persona can introduce slang, exaggerated praise, or highly informal phrasing into agent responses. <br>
Mitigation: Avoid using this skill for formal reviews, client-facing work, accessibility-sensitive work, or workflows requiring precise neutral language. <br>
Risk: The skill changes communication style but does not add domain-specific verification or safety controls. <br>
Mitigation: Review generated code, commands, and technical guidance using the same checks applied to normal agent output before relying on them. <br>
