## Description: <br>
Self-improving agent system that analyzes conversation quality, identifies improvement opportunities, and continuously optimizes response strategies. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[xiucheng](https://clawhub.ai/user/xiucheng) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and agent operators use this skill to review conversation quality, log improvement insights, generate weekly improvement reports, and suggest updates to agent behavior over time. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can analyze conversation content and write improvement records, which may include sensitive information. <br>
Mitigation: Confirm the logging location, exclude sensitive sessions, and review stored records before enabling automatic analysis. <br>
Risk: Automatic conversation analysis may run without clear consent, scope, retention, or disable controls. <br>
Mitigation: Define consent, scope, retention, disable, and deletion procedures before deployment. <br>
