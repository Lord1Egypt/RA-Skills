# 脊髓丘脑束浅感觉传导通路分析

## 适用对象
- 躯干皮肤
- 四肢皮肤
- 脊髓丘脑束
- 背侧丘脑腹后外侧核（VPL）

## 前提条件
确保以下结构功能正常：
- 脊神经节
- 脊髓白质前连合
- 丘脑皮质投射通路

## 禁忌与限制
- **不适用于头面部**（由三叉丘脑束传导）
- **不传导精细触觉或本体感觉**

## 执行步骤

### 1. 第1级神经元
- 胞体位于脊神经节（假单极，中小型，薄髓或无髓）
- 周围突分布于皮肤感受器
- 中枢突经后根进入脊髓：
  - **痛温觉纤维**（细）→ 后根外侧部 → 背外侧束（Lissauer束）→ 终止于脊髓Ⅰ、Ⅳ～Ⅶ层
  - **粗略触觉/压觉纤维**（较粗）→ 后根内侧部 → 直接终止于同节段后角

### 2. 第2级神经元
- 胞体主要位于脊髓Ⅰ、Ⅳ～Ⅶ层
- 发出纤维上升1～2节段 → 经**白质前连合**交叉至对侧
- 组成：
  - **脊髓丘脑侧束**：传导痛温觉
  - **脊髓丘脑前束**：传导粗略触觉和压觉
- 合称**脊髓丘脑束**，上行路径：延髓下橄榄核背外侧 → 脑桥、中脑（位于内侧丘系外侧）→ 终止于背侧丘脑腹后外侧核（VPL）

### 3. 第3级神经元
- 胞体位于VPL
- 纤维经内囊后肢投射至中央后回中上部及中央旁小叶后部

### 4. 纤维排列规律与临床判断
- 脊髓内脊髓丘脑束纤维由外向内、由浅入深依次为：**骶 → 腰 → 胸 → 颈**
- **脊髓内肿瘤压迫**：先压迫颈胸部纤维 → 对侧上半身痛温觉障碍先出现，再向下扩展
- **脊髓外肿瘤压迫**（如神经鞘瘤）：先压迫骶腰部纤维 → 对侧下半身感觉障碍先出现，再向上扩展

## 输出结果
根据感觉障碍的起始部位（上半身或下半身）和进展方向，判断脊髓病变是髓内还是髓外。
