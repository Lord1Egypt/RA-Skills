## 🔧 配置说明（必读）

> ⚠️ **安全声明**：本技能**不内置任何 API 密钥**，所有密钥需用户自行配置。代码中提到的 `GS_API_KEY` 和 `TUSHARE_TOKEN` 均为**示例占位符**，请勿将真实密钥提交到任何公开仓库。密钥泄露请立即在对应平台轮换。

### 环境变量

| 变量 | 说明 | 获取方式 |
|:-----|:-----|:---------|
| `TUSHARE_TOKEN` | Tushare API Token | https://tushare.pro |
| `GS_API_KEY` | 国信证券 API Key | https://www.guosen.com.cn |

### 配置方法

```bash
# 配置 Tushare（替换为你的真实 Token）
export TUSHARE_TOKEN="你的真实_token_在这里"

# 配置国信证券（替换为你的真实 API Key）
export GS_API_KEY="你的真实_api_key_在这里"
```

### 安全注意事项

1. 切勿将 API 密钥硬编码到代码中
2. 切勿将 API 密钥提交到公共仓库
3. 优先使用环境变量或 .env 文件管理密钥
4. 如发现密钥泄露，请立即在对应平台轮换密钥
