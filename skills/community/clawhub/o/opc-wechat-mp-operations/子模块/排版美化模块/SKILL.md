# 微信公众号排版助手

## 任务目标
- 本 Skill 用于：为微信公众号推文智能生成专业排版方案
- 能力包含：内容类型识别、排版风格匹配、配色方案设计、模板代码生成
- 触发条件：用户需要为微信推文进行排版、优化阅读体验、提升内容传播力时

## 前置准备
- 无需特殊依赖或前置准备
- 可选准备：用户提供推文内容文本、内容类型（科普/情感/营销/资讯）、特殊需求（品牌色、风格偏好）

## 操作步骤

### 步骤 1：内容分析与类型识别
1. 读取用户提供的推文内容
2. 分析文本结构和内容特征：
   - 标题层级
   - 段落组织方式
   - 重点内容位置
   - 引用和列表使用情况
3. 判断内容类型（科普/情感/营销/资讯）
   - 如果用户已指定类型，直接使用
   - 如果未指定，根据内容特征自动识别
4. 评估内容长度和复杂度

### 步骤 2：生成排版方案
根据内容类型和特征，从参考文档中获取相应规范：

1. **阅读类型规范**
   - 科普类：参见 `references/content-types.md` 中的"科普类推文"
   - 情感类：参见 `references/content-types.md` 中的"情感类推文"
   - 营销类：参见 `references/content-types.md` 中的"营销类推文"
   - 资讯类：参见 `references/content-types.md` 中的"资讯类推文"

2. **选择配色方案**
   - 从 `references/color-schemes.md` 中选择匹配的配色组合
   - 确保符合无障碍阅读标准（文字与背景对比度≥4.5:1）
   - 如有品牌色需求，优先使用用户指定的品牌色

3. **设计布局结构**
   - 参考 `references/platform-specs.md` 中的平台规范
   - 规划图片插入位置
   - 设计分隔线和重点标注方式
   - 设置标题、正文、引用的层次关系

4. **输出完整方案**
   包含以下模块：
   - 整体风格概述
   - 字体配置（标题字体/正文字体/引用字体/字号设置）
   - 配色方案（主色/辅助色/背景色/强调色）
   - 间距设置（行间距/段间距/字间距）
   - 布局结构说明

### 步骤 3：输出排版模板
根据排版方案生成可直接使用的模板：

1. **生成微信编辑器格式模板**
   - 参考 `assets/wechat-template.html` 的格式
   - 使用微信支持的HTML标签和内联样式
   - 确保兼容135编辑器、秀米、i排版等主流编辑器
   - 提供完整的可复制代码

2. **生成Markdown格式模板**（可选）
   - 参考 `assets/markdown-template.md` 的格式
   - 提供简洁的结构化排版方案

3. **说明应用方法**
   - 指导用户如何复制粘贴到微信编辑器
   - 说明需要注意的事项

### 步骤 4：交互式调整
根据用户反馈进行精细化调整：

1. 调整字体大小、行间距、字间距
2. 修改配色方案中的具体颜色值
3. 调整对齐方式和间距设置
4. 优化布局结构

每次调整后重新生成模板代码，确保用户可以直接应用。

## 注意事项
- 排版方案必须符合微信公众号平台规范，避免使用不支持的特殊样式
- 配色方案需遵循无障碍阅读标准，确保文字与背景对比度充足
- 字体大小和间距设置需适配手机端阅读习惯（建议字号：标题16-20px，正文14-16px，引用12-14px）
- 排版风格需与内容类型匹配（科普类注重清晰、情感类注重温度、营销类注重吸引、资讯类注重专业）
- 输出模板需确保兼容主流微信编辑器，避免使用私有标签
- 长图文需注意使用分隔线和空行进行视觉分割
- 重点内容可使用加粗、引用框、底色块等方式突出显示
