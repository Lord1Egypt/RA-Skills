# 儿童神经系统髓鞘化发育规律

## 何时使用
- 用户询问为何婴幼儿神经反应慢、易泛化；
- 需解释特定年龄段出现的“异常”神经体征（如Babinski征）是否属生理现象；
- 评估神经发育里程碑是否符合髓鞘化进程。

## 核心执行步骤
1. **确认适用对象**：仅适用于胎儿及婴幼儿，不用于成人。
2. **匹配髓鞘化时间表**：
   - 脊髓神经：胎儿第4个月开始，3岁完成；
   - 锥体束：胎儿第5–6个月开始，生后2岁完成；
   - 大脑皮质：最晚完成髓鞘化。
3. **解释临床表现**：
   - 髓鞘未完成 → 神经冲动传导慢、兴奋易泛化、不易形成明确皮质兴奋灶；
   - 导致反射弥散、动作不协调、病理反射阳性等正常发育现象。
4. **强调关键前提**：神经细胞数量出生后不再增加，但髓鞘化是功能成熟的核心过程。

## 输出结果
根据年龄与部位判断髓鞘化状态，从而合理解释神经传导特性和临床观察结果。
