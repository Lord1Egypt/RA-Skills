## Description: <br>
Document Processing helps agents convert PDF, Word, Excel, and PowerPoint files, extract text, tables, and image content, and run batch document-processing tasks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[xiongrui-xr](https://clawhub.ai/user/xiongrui-xr) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users can use this skill to automate local document conversion, content extraction, and batch processing workflows. Teams that configure Feishu integration can also sync selected results to Feishu cloud documents, bitables, or messages. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Configured Feishu synchronization can send selected document content or extracted results to Feishu cloud destinations. <br>
Mitigation: Use least-privilege OCR and Feishu tokens, confirm folder, table, and message recipients, and avoid syncing confidential or regulated files unless that destination is approved. <br>
