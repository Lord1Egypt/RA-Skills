# 血-胸腺屏障结构与功能

## 何时使用
- 需阐明胸腺如何隔离外源大分子以保障T细胞正常发育
- 分析自身免疫风险是否与胸腺屏障损伤相关
- 区分胸腺皮质与髓质微环境差异
- 解释示踪实验中大分子无法进入胸腺皮质的现象

## 执行步骤
1. **确认适用条件**：仅适用于胸腺皮质区域，且胸腺处于发育或功能活跃状态；排除其他淋巴器官。
2. **识别屏障五层结构（由内向外）**：
   - 连续毛细血管（内皮细胞间具完整紧密连接）
   - 内皮周围连续基膜
   - 含巨噬细胞的血管周隙
   - 上皮基膜
   - 连续胸腺上皮细胞层（具突起，形成物理与生化屏障）
3. **明确功能机制**：阻止血液中大分子（如抗体、细胞色素C、铁蛋白、辣根过氧化物酶等）渗入胸腺皮质。
4. **关联生理意义**：维持胸腺内环境稳定，确保T淋巴细胞在无抗原干扰下完成阳性/阴性选择，建立中枢耐受。
5. **预警输出**：若屏障受损，应提示可能导致T细胞发育异常或自身免疫疾病风险升高。

> 注意：该屏障不存在于胸腺髓质或其他淋巴器官，使用时需严格限定解剖位置。
