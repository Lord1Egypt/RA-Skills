# Meta-Analysis Friendly SCI Journal Database

> **Data sources**:
> - **IF & JCR Quartile**: 2025 JCR (released June 2026), cross-referenced with verified data from ShowJCR/hitfyd GitHub repository
> - **CAS Quartile**: CAS journal ranking 2025 (中科院分区表 2025), same verified source
> - **Acceptance rates**: Estimates from published editorial reports; verify on journal websites.
> - See `references/jcr_verified_data.csv` for exact IF, JCR quartile, and CAS quartile for all 316 journals in this database.
> **★ = High MA acceptance** | **☆ = Moderate** | **APC** = OA Article Processing Charge; "Hybrid" = OA + subscription routes.
> **IF values below are approximate ranges for quick scanning; exact values are in jcr_verified_data.csv.**

## Scoring Guide for Journal Selection

### Factor Weights for Ranking

When comparing journals, apply these weights to produce a composite score (use exact IF and CAS quartile from `jcr_verified_data.csv`):

| Factor | Weight | Scoring Criteria |
|--------|--------|-----------------|
| Scope Match | 25% | 5=perfect match, 3=partial overlap, 1=peripheral |
| MA History | 20% | 5=published 10+ MAs/yr, 3=published 5-10/yr, 1=rarely |
| IF + CAS Quartile | 25% | 5=IF in reach AND CAS Q1-Q2; 3=IF competitive OR CAS Q2-Q3; 1=IF long shot AND CAS Q3-Q4 |
| Review Speed | 15% | 5=<4 wks, 4=4-6 wks, 3=6-8 wks, 2=8-12 wks, 1=>12 wks |
| Cost (APC) | 15% | 5=no APC, 4=<2,000, 3=2,000-3,000, 2=3,000-4,000, 1=>4,000 |

### Recommended Tier Thresholds

After computing composite scores, apply IF-based constraints. The final tier is the minimum of the score-based tier and the IF-constraint tier.

**Score-based tiers (from weighted algorithm):**
- Total Score 4.0-5.0 → reach candidate
- Total Score 3.0-3.9 → target candidate
- Total Score 2.0-2.9 → safety candidate

**IF-based constraints (override score-based tier if more restrictive):**

| Paper Grade | Safety IF Floor | Target IF Floor | Reach IF Floor |
|-------------|-----------------|-----------------|----------------|
| A (High) | IF ≥ 3 | IF ≥ 8 | IF ≥ 15 |
| B (Medium) | IF ≥ 1 | IF ≥ 3 | IF ≥ 7 |
| C (Modest) | IF ≥ 0.5 | IF ≥ 1 | IF ≥ 3 |

Example: A B-grade paper's candidate journal with composite score 4.2 (Reach candidate) but IF of 3.5 (< Reach floor of 7) → final tier = min(Reach, Target) = Target.

## Predatory/Questionable Journal Red Flags

DO NOT recommend journals with any of these characteristics:
- Not indexed in JCR/SCIE
- Not listed in DOAJ (for OA journals)
- Aggressive email solicitation
- Promises of extremely fast review (<1 week)
- No clear editorial board listed
- Website with grammatical errors or fake metrics
- Listed on Beall's List or Cabells Predatory Reports
- Imitates name of established journal (hijacked journal)
- APC-only business model with minimal peer review

When in doubt, verify via:
1. Master Journal List (https://mjl.clarivate.com/)
2. DOAJ (https://doaj.org/) for OA journals
3. Think. Check. Submit. (https://thinkchecksubmit.org/)
