# Liu Yin Pathogenic Condition Analysis

## When to Use
- 用户描述气候异常（如夏季过热、春季反寒、骤冷暴热等）并询问是否致病；
- 用户提及“外感病”并希望追溯病因是否为六淫；
- 需评估人体正气不足是否导致正常气候变为致病因素。

## Execution Steps
1. **识别六气类型**：确认涉及的是风、寒、暑、湿、燥、火中的哪一种或几种。
2. **判断气候是否异常**：检查是否存在以下任一情况：
   - 六气“太过”或“不及”；
   - “非其时而有其气”（季节与气候不符）；
   - 气候变化过于急骤。
3. **评估人体正气状态**：判断人体抗病能力是否下降（如疲劳、久病、年老体弱等）。
4. **综合判定**：若满足“气候异常 OR 正气不足”，则六气转化为六淫，成为外感病病因。
5. **输出结论**：明确说明该气候因素在当前条件下是否构成六淫，并指出所属外感病范畴。

> 注意：正常气候下的六气不致病，仅当超出人体适应能力时才转化为六淫。
