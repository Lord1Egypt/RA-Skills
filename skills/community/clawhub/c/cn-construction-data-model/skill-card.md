## Description: <br>
Designs construction project data models, including entity relationships, schemas, SQL database structures, JSON schemas, Mermaid ER diagrams, and model validation guidance. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, data architects, and construction project teams use this skill to design construction project entities, relationships, fields, SQL schemas, JSON schemas, and ER diagrams aligned with Chinese construction data-management workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Package metadata is not fully consistent across artifact files. <br>
Mitigation: Verify the publisher and package identity before installation or deployment. <br>
Risk: Generated SQL may not match the target database or project governance requirements. <br>
Mitigation: Review generated SQL before running it against any database. <br>
Risk: Construction project files can contain sensitive or unnecessary data. <br>
Mitigation: Provide only the project files intended for processing and choose export paths deliberately. <br>
