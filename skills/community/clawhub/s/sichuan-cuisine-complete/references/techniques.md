# 川菜烹饪技法大全

## 一、刀工

### 基本刀法
- **切丝**：先片后切，"片薄如纸，切细如发"
- **切片**：逆纹切（牛肉）/ 顺纹切（猪肉）
- **切丁**：先切条再切丁，大小均匀
- **花刀（剞刀）**：腰花、鱿鱼等，十字花刀深2/3不断

### 川菜特殊刀工
- **蒜泥白肉的"片"**：肉煮熟冷却后，用大平刀法片成透亮薄片，越薄越好（≤2mm）
- **回锅肉的"切"**：0.3cm厚，保证每片都有皮、肥、瘦三层

## 二、火候

### 川菜核心火候概念

| 火候 | 油温 | 用途 | 判断方法 |
|------|------|------|---------|
| 小火（低温） | 80-120°C | 炒豆瓣出红油 | 油面平静，微有波动 |
| 中火（中温） | 120-160°C | 滑肉丝/肉片 | 油面起波纹，有微烟 |
| 大火（高温） | 160-200°C | 爆炒、最后泼油 | 油面滚动，烟明显 |
| 猛火（极高） | 200°C+ | 炝炒蔬菜、泼油激香 | 大量青烟 |

### 关键火候要领
1. **炒豆瓣永远小火**——超过120°C豆瓣发苦发黑
2. **滑肉丝三四成热**——油温太高肉老
3. **炝炒蔬菜七成热**——锅要烧到冒烟
4. **泼油要到冒烟**——200°C+才能激发刀口辣椒香味

## 三、调味

### 川菜四大调味原则
1. **咸是基础**：任何味型都不能离开咸味打底
2. **糖是提鲜**：大部分菜都放少量糖，不是为甜，是为提鲜
3. **复合味型**：单一口味不是川菜，要追求层次
4. **先咸后甜再酸辣**：调味顺序不能乱

### 调味时间节点
- **腌制**（提前）：盐、料酒、酱油、淀粉上浆
- **底味**（炒制中）：豆瓣、酱料、姜蒜
- **定味**（收汁前）：酱油、盐、糖、醋
- **提香**（出锅前）：葱花、香油、花椒油
- **点缀**（装盘后）：花椒面、芝麻、花生碎

### 碗芡（对汁芡）
鱼香、宫保、荔枝等快炒菜的关键技术：
- 将所有液体调料+水淀粉提前在一个碗里调好
- 一次淋入，翻炒几下即出锅
- 避免逐一加调料，炒过头菜就老了

## 四、上浆与挂糊

| 类型 | 比例 | 用途 | 代表菜 |
|------|------|------|--------|
| 水淀粉上浆 | 肉+盐+料酒+水淀粉 | 保持嫩滑 | 鱼香肉丝、水煮肉片 |
| 蛋清上浆 | 肉+盐+料酒+蛋清+淀粉 | 更嫩 | 酸菜鱼鱼片 |
| 全蛋糊 | 蛋液+淀粉 | 炸制 | 糖醋里脊 |
| 干淀粉 | 直接拍干粉 | 煎制 | 煎鱼 |

## 五、勾芡

### 川菜勾芡三法
1. **包芡**（浓芡）：芡汁全部包裹在食材上，盘底无汁——宫保鸡丁、鱼香肉丝
2. **琉璃芡**（薄芡）：芡汁部分包裹，盘底略有流汁——麻婆豆腐、水煮牛肉
3. **汤芡**（更薄）：酸辣汤、三鲜汤等汤类勾芡

### 麻婆豆腐三次勾芡法（独有）
- 第一次：下豆腐烧开后，芡汁入锅，推匀
- 第二次：烧1分钟后，再次勾芡，汤汁开始收浓
- 第三次：再烧1分钟，最后勾芡，汤汁完全吸附在豆腐上
- 目的：保持豆腐嫩滑，汤汁浓而不稠

## 六、川菜特殊技法

### 1. 煸炒（干煸）
不放水/极少水，中火煸干水分——干煸四季豆、干煸牛肉丝

### 2. 小炒
不换锅，先炒肉再下配料，一气呵成——回锅肉、盐煎肉、小炒肉

### 3. 水煮
不是白水煮！是先炒底料加汤煮，最后泼油——水煮牛肉、水煮鱼

### 4. 炝炒
大火热油，干辣椒花椒爆出糊香后快速翻炒——炝炒莲白、炝炒土豆丝

### 5. 软烧
鱼不煎炸，直接入汤汁中小火烧熟——大蒜烧鳝段

### 6. 粉蒸
食材裹米粉，垫红薯/南瓜，蒸熟——粉蒸肉、粉蒸排骨

### 7. 火爆
极高温极短时间完成——火爆腰花、火爆黄喉

### 8. 干烧
不勾芡，大火将汤汁完全收干入食材——干烧鱼、干烧虾

## 七、烹饪顺序模板

### 小炒类（回锅肉、盐煎肉等）
```
预处理食材 → 热锅少油煸主料 → 拨到一边炒豆瓣出红油
→ 加酱料姜蒜 → 混合翻炒 → 加配料 → 调底味 → 出锅
```

### 滑炒类（宫保鸡丁、鱼香肉丝等）
```
食材上浆 → 调碗芡 → 温油滑主料 → 盛出 → 炒香小料
→ 回锅主料 → 淋碗芡 → 快速翻匀 → 出锅
```

### 水煮类
```
炒香料垫底 → 炒豆瓣加汤 → 煮主料 → 装碗 → 撒刀口辣椒
→ 泼热油
```

### 烧菜类（麻婆豆腐、家常豆腐等）
```
炒香豆瓣调料 → 加汤 → 下主料 → 小火烧入味 → 勾芡 → 出锅
```
