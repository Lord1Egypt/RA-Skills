## Description: <br>
Analyze geospatial data using geopandas with proper coordinate projections. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and analysts use this skill to guide GeoPandas workflows for loading geospatial files, filtering geometries, and calculating distances between geographic features such as earthquakes and plate boundaries. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Distance calculations can be misleading when the coordinate reference system is inappropriate for the region or task. <br>
Mitigation: Verify the coordinate reference system before relying on distance results, and project data to a suitable metric CRS before measurement. <br>
