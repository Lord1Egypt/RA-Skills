## 第一步：基础信息采集与真太阳时校正

### 操作指令

```
INPUT: 用户提供的出生信息（年月日时分、地点）
PROCESS: 
  1. 计算地方时差 = (出生地经度 - 120) × 4分钟
  2. 计算真太阳时 = 北京时间 + 地方时差
  3. 若出生在夏令时期间（6月-9月），再加1小时
OUTPUT: 真太阳时出生时间
```

### 真太阳时计算示例

**案例1：北京出生（经度116.4°E）**
```
出生时间：2020年5月15日 10:30（北京时间）
地方时差 = (116.4 - 120) × 4 = -3.6 × 4 = -14.4分钟
真太阳时 = 10:30 - 14分钟 = 10:16
```

**案例2：新疆乌鲁木齐出生（经度87.6°E）**
```
出生时间：2020年5月15日 10:30（北京时间）
地方时差 = (87.6 - 120) × 4 = -32.4 × 4 = -129.6分钟 ≈ -2小时10分钟
真太阳时 = 10:30 - 2:10 = 8:20
```

**案例3：哈尔滨出生（经度126.6°E）**
```
出生时间：2020年1月15日 10:30（北京时间）
地方时差 = (126.6 - 120) × 4 = 6.6 × 4 = 26.4分钟
真太阳时 = 10:30 + 26分钟 = 10:56
```

## 第二步：排出四柱八字

### 操作指令

```
INPUT: 真太阳时出生时间、性别
PROCESS:
  1. 确定年柱（查万年历或立春为界）
  2. 确定月柱（查月令表，以节令为界）
  3. 确定日柱（查六十甲子表或万年历）
  4. 确定时柱（查五鼠遁表）
OUTPUT: 完整四柱（年柱、月柱、日柱、时柱）
```

### 完整排盘示例

**案例**：1990年8月15日 14:30 北京时间 出生于上海

**第一步：真太阳时校正**
```
出生地：上海（经度121.5°E）
时差 = (121.5 - 120) × 4 = +6分钟
真太阳时 = 14:30 + 6分钟 = 14:36
```

**第二步：查万年历**
```
年柱：1990年8月（立秋后）→ 庚午年
月柱：8月15日处暑后 → 申月 → 壬申月
日柱：查表得 → 丙戌日
时柱：日干丙，按五鼠遁 → 丙申时
```

**最终四柱**：
```
年柱：庚午
月柱：壬申
日柱：丙戌
时柱：丙申
```

## 第三步：排出藏干与命宫

### 藏干排法

**查表确定各支藏干**：

| 地支 | 本气 | 中气 | 余气 |
|------|------|------|------|
| 子 | 癸 | - | - |
| 丑 | 己 | 癸 | 辛 |
| 寅 | 甲 | 丙 | 戊 |
| 卯 | 乙 | - | - |
| 辰 | 戊 | 乙 | 癸 |
| 巳 | 丙 | 庚 | 戊 |
| 午 | 丁 | 己 | - |
| 未 | 己 | 丁 | 乙 |
| 申 | 庚 | 壬 | 戊 |
| 酉 | 辛 | - | - |
| 戌 | 戊 | 辛 | 丁 |
| 亥 | 壬 | 甲 | - |

### 命宫排法

**公式**：
```
命宫地支 = (14 - 出生月份) + 出生时辰
若结果 > 12，则 -12
若结果 = 0，则 = 12
```

**命宫天干**：根据命宫地支和日干，查五虎遁表

**五虎遁（年起月法）**：
```
甲己年起丙寅月
乙庚年起戊寅月
丙辛年起庚寅月
丁壬年起壬寅月
戊癸年起甲寅月
```

### 胎元排法

**公式**：
```
胎元月干 = 月干进一位
胎元月支 = 月支进一位
```

### 排盘完整示例

**沿用上例**：1990年8月15日 14:36 上海出生

```
【四柱】
年柱：庚午（火） 藏干：庚、丁、己
月柱：壬申（金） 藏干：庚、壬、戊
日柱：丙戌（火） 藏干：戊、辛、丁
时柱：丙申（火） 藏干：庚、壬、戊

【命宫】
计算：(14 - 8) + 8 = 14 → 14 - 12 = 2 → 卯
日干庚，按五虎遁：庚卯月
命宫：庚卯

【胎元】
月干壬进一位：癸
月支申进一位：亥
胎元：癸亥
```

## 第五步：分析旺衰

### 操作指令

```
INPUT: 四柱八字、日干
PROCESS:
  1. 计算月令旺度
  2. 计算地支通根情况
  3. 计算天干帮扶情况
  4. 综合评估旺衰等级
OUTPUT: 旺衰等级（极旺/旺/中和/弱/极弱）
```

### 月令旺度表

| 月令 | 同气 | 相生 | 被生 | 克 | 泄 |
|------|------|------|------|------|------|
| 旺度 | +40% | +30% | +20% | -30% | -20% |

### 通根力量表

| 通根类型 | 力量 | 说明 |
|---------|------|------|
| 禄（帝旺） | 100% | 地支与天干同气 |
| 刃（阳刃） | 95% | 帝旺之支，阳干独有 |
| 根（长生） | 50% | 地支含天干之根 |
| 库（墓库） | 40% | 地支为天干之库 |
| 余气 | 20% | 地支含天干余气 |
| 虚浮 | 0% | 天干无根 |

### 旺衰等级判定

| 等级 | 综合得分 | 判定标准 | 喜忌原则 |
|------|---------|---------|---------|
| 极旺 | +80~+100 | 得令+得地+得助全占 | 宜泄不宜扶 |
| 旺 | +50~+79 | 得令或得地得助占两项 | 抑为佳 |
| 中和偏旺 | +30~+49 | 稍偏旺但不严重 | 慎动 |
| 中和偏弱 | -30~-49 | 稍偏弱但不严重 | 慎动 |
| 弱 | -50~-79 | 失令或失地失助占两项 | 扶为佳 |
| 极弱 | -80~-100 | 失令+失地+失助全占 | 宜扶不宜泄 |

### 旺衰分析示例

**案例**：庚午 壬申 丙戌 丙申

```
日干：丙火
月令：申月（金），丙火失令 → 月令得分：-30%

【通根情况】
年支午：午藏丁火，丙火帝旺 → +40%
日支戌：戌藏丁火余气 → +10%
时支申：申不含火 → 无

【天干帮扶】
时干丙：同我者比肩 → +15%

【综合得分】
-30% + 40% + 10% + 15% = +35%

【判定】
中和偏旺（+35分）
```

## 第七步：选取用神

### 操作指令

```
INPUT: 格局、旺衰、调候
PROCESS:
  1. 确定用神类型（扶抑/调候/通关/病药）
  2. 按优先级选取用神
  3. 确定相神、喜神、忌神
OUTPUT: 完整用神体系
```

### 用神选取原则

| 格局类型 | 核心用神 | 辅助用神 | 忌神 |
|---------|---------|---------|------|
| 官杀格 | 印星 | 财星、官杀 | 伤官、食神 |
| 财格 | 官杀、印星 | 食神、财星 | 比劫 |
| 印格 | 官杀、财星 | 印星、食神 | 财星过重 |
| 食神格 | 财星 | 印制枭神 | 枭神夺食 |
| 伤官格 | 财星、印星 | 印制伤官 | 官杀混杂 |
| 比劫格 | 官杀、财星 | 食神、伤官 | 比劫 |

### 用神层级关系

```
格局 → 用神 → 相神 → 喜神 → 忌神
  ↑_________↓（反馈调整）
```

**定义**：
- 用神：格局所需的核心五行
- 相神：辅助用神、保护用神的五行
- 喜神：用神所喜的五行
- 忌神：破坏格局的五行

### 用神选取示例

**案例**：庚午 壬申 丙戌 丙申

```
格局：财格
旺衰：中和偏旺

- 财格身旺：宜克泄，官杀泄财为用
- 财格身弱：宜生扶，印星助身为用

本例日主偏旺，壬水官星为用神
用神：壬水（官星）
相神：庚金（财星生官）
喜神：辛金（酉）、申中庚金
忌神：丁火、午火（比劫分官之力）

调候检查：
丙火生申月，不寒不暖 → 调候非必须
```
