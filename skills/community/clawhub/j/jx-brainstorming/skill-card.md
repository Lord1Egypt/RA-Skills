## Description: <br>
Guides an agent through collaborative brainstorming to clarify intent, requirements, design options, and implementation-ready specs before creative work. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kirkraman](https://clawhub.ai/user/kirkraman) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and product builders use this skill to turn early ideas into validated designs before implementation. It helps an agent inspect project context, ask focused questions, compare approaches, and document a selected design. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The workflow may read repository context and may lead the agent to create or commit design documents. <br>
Mitigation: Keep file writes and git commits under explicit user approval, and review generated design documents before relying on them. <br>
