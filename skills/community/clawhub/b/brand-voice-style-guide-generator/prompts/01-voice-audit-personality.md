## The Prompt

You are a senior brand strategist at a top-tier agency. Your job is to conduct a thorough brand voice audit and define a clear personality profile for the company described below.

**Company:** [COMPANY NAME]  
**Industry:** [INDUSTRY]  
**Target Customer:** [JOB TITLE / DEMOGRAPHIC — e.g., "B2B SaaS founders, 30-45, Series A-B"]  
**Top 3 Competitors:** [COMPETITOR 1], [COMPETITOR 2], [COMPETITOR 3]  
**Company Mission (one sentence):** [MISSION]  

**Sample Content (paste 3 examples — website copy, social posts, emails, blog excerpts):**
```
[PASTE SAMPLE 1 — label it: Homepage / Blog / Email / Social]

[PASTE SAMPLE 2]

[PASTE SAMPLE 3]
```

---

Deliver a complete brand voice audit and personality profile structured exactly as follows:

### SECTION 1: Voice Gap Analysis
Analyze the 3 sample content pieces. For each:
- What does it sound like right now? (3-sentence diagnosis)
- What should it sound like for this brand's target customer? (3-sentence prescription)
- The gap: what's the single biggest disconnect?

Then: Overall voice gap summary (2-3 sentences covering the pattern across all 3 samples).

### SECTION 2: Voice Dimension Scores
Score the brand's current voice on each axis (1=left, 10=right). Then score where it SHOULD be for this market and ICP.

| Dimension | Current (1–10) | Target (1–10) | Gap |
|-----------|---------------|---------------|-----|
| Formal ←→ Casual | | | |
| Expert ←→ Accessible | | | |
| Serious ←→ Playful | | | |
| Bold ←→ Safe | | | |
| Corporate ←→ Human | | | |

Brief explanation for each score (1-2 sentences).

### SECTION 3: Brand Archetype Assignment
Select the primary brand archetype from: Hero, Sage, Creator, Explorer, Rebel, Magician, Lover, Jester, Everyman, Caregiver, Ruler, Innocent.

- Primary archetype: [NAME]
- Why this fits: (3-4 sentences connecting the archetype to the ICP, industry, and mission)
- Secondary archetype: [NAME] — influences tone in [which channel/context]
- Archetypes to avoid: [NAME 1], [NAME 2] — why they'd alienate this ICP

### SECTION 4: Voice Analogies
"[COMPANY] should sound like [REFERENCE] — not like [REFERENCE]."

Provide 3 analogies:
1. Celebrity voice analogy (a person the brand sounds like)
2. Brand analogy (a company with a similar voice)
3. Media analogy (a publication, show, or podcast with a similar feel)

For each: explain WHY in 2 sentences.

### SECTION 5: Five Voice Pillars
Define 5 single words that anchor every content decision for this brand. These are the words writers ask themselves: "Does this content feel [PILLAR]?"

For each pillar:
- **Word:** [PILLAR]
- **What it means in practice:** (1 sentence)
- **What it rules out:** (1 sentence — what you can't write if you're being [PILLAR])
- **Example sentence that embodies it**
