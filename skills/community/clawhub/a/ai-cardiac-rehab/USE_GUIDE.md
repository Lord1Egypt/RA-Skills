# 使用指南

## 安装

```bash
pip install flask
python app.py
```

浏览器打开 `http://localhost:5000`。

## 工作流程

1. **注册/登录**：自动注册，密码 SHA256 加密
2. **填写档案**：必须如实填写年龄、心脏病诊断、EF值、合并症、药物
3. **获取 AI 建议**：系统根据临床数据自动生成风险分层和运动处方
4. **每日记录**：记录症状、心率、血压、运动情况
5. **持续监测**：7天趋势图 + 症状预警

## 环境变量

| 变量 | 说明 |
|------|------|
| `SECRET_KEY` | Flask 密钥（生产环境设置） |

## 生产部署

```bash
pip install gunicorn
gunicorn -w 2 -b 127.0.0.1:5000 app:app
```
