# 线粒体损伤级联反应分析

## 适用场景
- 观察到真核细胞线粒体结构异常（如肿胀、嵴消失）
- 检测到细胞内ATP水平下降≥5%
- 存在缺血、缺氧、化学毒物暴露、ROS升高或钙超载等损伤诱因
- 需评估细胞是否启动凋亡或走向坏死

## 执行步骤
1. **确认初始损伤表现**：检查线粒体是否出现肿胀、空泡化、嵴变短稀疏或基质内含钙致密体。
2. **评估ATP代谢状态**：
   - 测定ATP生成与消耗变化
   - 若ATP减少5%～10%，判定已产生显著细胞损伤效应
3. **识别继发性功能障碍**：
   - 检查钠泵/钙泵功能是否受损（表现为Na⁺、Ca²⁺内流）
   - 评估跨膜转运蛋白合成、脂质合成及磷脂酰基化状态是否下降或停滞
4. **判断凋亡启动**：检测细胞色素C是否从线粒体释放至胞质，若阳性则caspase通路已被激活
5. **评估代谢崩溃风险**：
   - 确认氧化磷酸化是否中止
   - 检测乳酸堆积与细胞内pH，判断是否发生酸中毒
6. **确定不可逆性**：若同时满足“线粒体形态异常 + ATP下降≥5% + 细胞色素C释放”，则判定为不可逆损伤高风险

> **注意**：本技能仅适用于含线粒体的真核细胞，不适用于原核细胞。
