## Description: <br>
Improve a skill package so AI agents can find, select, and trust it across registries and ecosystems, including Codex, Claude-style agents, OpenClaw, and other skill-based tools. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[LangYou](https://clawhub.ai/user/LangYou) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Skill authors, developers, and registry maintainers use this skill to improve skill names, descriptions, metadata, tags, listing copy, default prompts, and example prompts so agent systems can route to the skill more reliably. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated skill metadata or listing copy could overstate scope, platform support, trust, or capabilities. <br>
Mitigation: Review generated copy before publishing and keep claims aligned with the actual skill package. <br>
