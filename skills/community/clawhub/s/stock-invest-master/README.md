# Stock Invest Master

> 道法 × 十大师 × AI — 股票投资分析技能，支持 A股、港股、美股

⚠️ **投资免责申明**：本工具仅供学习和参考，不构成任何投资建议。所有投资决策由用户自行承担风险。

## 六层分析框架

从投资心性到量化工具，逐层递进，任一层不通过则触发对应处置：

| 层次 | 核心问题 | 主导大师 | 不通过处置 |
|------|---------|---------|-----------|
| **志** | 投资信仰是否端正？心性是否成熟？ | 格雷厄姆·巴菲特·段永平 | **停止分析** |
| **道** | 底层商业逻辑是否成立？ | 巴菲特·芒格·格雷厄姆 | **停止分析** |
| **势** | 市场周期与时机处于什么阶段？ | 索罗斯·马克斯·达利欧 | 需谨慎 |
| **法** | 值不值得投？如何系统化判断？ | 林奇·费雪·格雷厄姆 | 等待 |
| **术** | 具体怎么操作？仓位如何管理？ | 索罗斯·十大师标准 | 调整计划 |
| **器** | 工具与量化验证是否支持结论？ | 西蒙斯 | 核实数据 |

## 分析流程

```
识别市场(A股/美股/港股) → 25问快速筛选 → 六层逐层分析(注入大师视角)
  → 十大师共识结论 → 违背志·道·法专项诊断 → 保存报告
```

**25问快速筛选规则**：Q18(管理层诚信)=否 → 自动否决；任一主题>3个"否"或总计>8个"否" → 放弃

| 主题 | 问题数 | 关注点 |
|------|--------|--------|
| 志·投资信仰 | 4 | 价值信仰、风险认知、长期主义 |
| 道·商业本质 | 5 | 能力圈、护城河、内在价值 |
| 势·市场周期 | 4 | 反身性、周期定位、债务周期 |
| 法·分析流程 | 5 | 盈利质量、安全边际、PEG |
| 术·操作执行 | 3 | 仓位管理、卖出标准 |
| 器·工具技术 | 4 | 量化验证、回测、统计显著性 |

## 报告输出

分析完成后自动保存 Markdown 报告至 `~/.stock-invest-master/yyyymmdd_公司名.md`

报告包含：六层评估 + 十大师共识 + 违背"志·道·法"专项诊断 + 关键假设/风险/监控指标 + 数据来源校验声明
