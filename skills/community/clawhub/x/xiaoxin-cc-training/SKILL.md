# 超短交易教学（基于小新擒龙日记）

## 核心理念

### 1. 龙头股的定义与特征
- **什么是龙头**：引领板块上涨，具有最高辨识度的个股
- **龙头特征**：
  - 板块启动时最先涨停
  - 成交量最大、换手率最高
  - 回调时幅度最小
  - 有持续的消息/题材催化

### 2. 情绪周期判断
- **冰点期**：市场情绪极度低迷，涨停家数少
  - 标志：连板股少、炸板率高、跟风股大跌
  - 策略：防守为主，等待转折
- **修复期**：冰点后情绪开始回暖
  - 标志：涨停家数增加、炸板率下降
  - 策略：试错新题材
- **高潮期**：情绪亢奋，板块集体上涨
  - 标志：板块内多股涨停、持续性强
  - 策略：持股待涨、龙头战法
- **退潮期**：情绪开始回落
  - 标志：高位股大跌、炸板增多
  - 策略：止盈止损、观望

### 3. 竞价技巧
**核心观察指标：**
- 竞价量：对比昨日量能
- 竞价换手率：真实换手情况
- 竞价位置：高开/低开/平开
- 板块效应：同板块竞价情况

**竞价爆量模式：**
- 定义：竞价成交额超过昨日成交额的50%
- 意义：市场一致看好抢筹
- 买点：竞价确认后立即买入

### 4. 弱转强模式
**核心逻辑**：前一天烂板/炸板，今天超预期高开/涨停

**弱转强类型：**
- **涨停弱转强**：昨日炸板/烂板，今日快速涨停
- **分歧弱转强**：昨日高位震荡，今日分歧转一致

**实战要点：**
- 观察板块整体强度
- 确认跟风个股跟随情况
- 理解"弱转强"背后的逻辑（市场预期改变）

### 5. 复盘方法

**每日复盘步骤：**
1. **数据统计**：涨停家数、炸板率、连板高度
2. **板块梳理**：今日涨停主要集中在哪些板块
3. **龙头复盘**：连板龙头股走势分析
4. **明日预判**：推演明日可能的走势
5. **计划制定**：制定明日交易计划

**竞价复盘观察：**
- 昨日涨停股今日竞价表现
- 昨日炸板股今日修复情况
- 今日首板股竞价量能

### 6. 风险控制

**风险预警信号：**
- 监管特停（龙头被关）
- 板块高潮后分歧
- 大盘破位下跌
- 成交额持续萎缩

**回撤控制：**
- 单票止损线：-7%必须止损
- 单日回撤：-5%停止开新仓
- 连续回撤：3天回撤超10%空仓休息

### 7. 题材判断

**题材生命周期：**
1. **萌芽期**：只有1-2只涨停，看不懂
2. **发酵期**：板块效应出现，开始被认知
3. **高潮期**：板块持续上涨，全面爆发
4. **退潮期**：高位股开始大跌

**如何判断题材级别：**
- 政策力度（国家级>地方级）
- 想象空间（故事越大越好）
- 业绩落地时间（越远越有炒作空间）
- 参与资金量（成交量越大越健康）

## 实战案例分析

### 案例1：龙头弱转强
**背景**：某龙头股昨日炸板，形态较弱
**今日**：超预期高开5%，竞价量能放大
**分析**：
- 高开说明市场一致性看好
- 竞价量能放大说明抢筹
- 板块内跟风股竞价也不错
**结论**：符合弱转强模式，可以打板买入

### 案例2：情绪冰点判断
**观察指标**：
- 昨日涨停家数<20
- 炸板率>40%
- 连板高度<3板
- 跌停股增加

**应对策略**：
- 不开新仓
- 持有股有盈利就出
- 等待冰点后的反弹机会

## 常见问题解答

**Q: 总是卖飞龙头怎么办？**
A: 龙头信仰是超短的核心。卖飞是因为恐高，但真正的大龙头会持续上涨。解决方法：设定固定的止盈位，如不破5日线不出。

**Q: 如何克服恐惧心理？**
A: (1) 降低仓位到舒适区间 (2) 只做模式内的交易 (3) 复盘时做好计划，执行时就不慌

**Q: 账户总是过山车怎么办？**
A: 问题在于不止盈。盈利后要设置保护位，比如回撤20%时必须出。不要把盈利变成亏损。

**Q: 怎么判断题材持续性？**
A: (1) 政策消息级别 (2) 涨停家数多少 (3) 是否有换手龙 (4) 趋势中军是否趋势向上

## 交易纪律

1. **只做模式内**：只做龙头、弱转强、竞价爆量
2. **止损要坚决**：-7%必须割
3. **控制仓位**：单票不超30%
4. **复盘要细致**：每天2小时复盘时间
5. **心态要平和**：不要想一夜暴富

## 推荐学习路径

1. 理解情绪周期（冰点-修复-高潮-退潮）
2. 学会复盘（每日数据统计、板块梳理）
3. 掌握竞价技巧（竞价量能、位置、板块效应）
4. 练习弱转强模式（从假阳线开始）
5. 逐步建立自己的交易系统
