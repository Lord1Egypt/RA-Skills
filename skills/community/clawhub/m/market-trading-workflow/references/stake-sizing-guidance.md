# Stake sizing guidance

Use this only for the World Cup fixture trader.

## Rule of thumb
- **Normal/near-market edge:** size normally.
- **Moderate underdog or draw-leaning spot:** size down to a half stake when the side is still plausible and the model is near the market.
- **Extreme longshot:** pass unless the model clearly supports the upset.

## Practical interpretation
- Suitable for cases like a roughly 60/40 market where the underdog is not a true longshot.
- Not suitable for 80/20 or 90/10 style flips where the opposite side is mostly priced as a deep longshot.

## Decision wording
- Prefer `small bet` or `size down` for moderate underdogs.
- Prefer `pass` for extreme longshots.
- Paper and live should share the same decision rule; only the execution mode differs.
