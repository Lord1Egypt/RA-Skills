# vWD分型诊断的实验室比值判读

## 适用场景
- 患者存在出血倾向，临床怀疑血管性血友病（vWD）
- 已完成vWF:Ag、vWF:A和FVIII:C三项实验室检测
- 需在vWD各亚型、血友病A及血栓状态之间进行鉴别

## 执行步骤
1. **确认三项基础指标**：确保已测定vWF抗原（vWF:Ag）、vWF活性（vWF:A）和凝血因子Ⅷ活性（FVIII:C）。
2. **初步排除**：若三项指标均在正常参考范围内，则基本可排除vWD和血友病A。
3. **计算关键比值**：若任一指标降低，计算以下两个比值：
   - 比值1 = vWF:A / vWF:Ag
   - 比值2 = FVIII:C / vWF:Ag
4. **依据比值判读分型**：
   - 若比值1 ≥ 0.7 且 比值2 ≥ 0.7 → 诊断为 **vWD 1型**（数量减少型）
   - 若比值1 < 0.7 且 比值2 ≥ 0.7 → 提示 **vWD 2A/2B/2M亚型**（功能缺陷型），需进一步行RIPA和vWF多聚体分析
   - 若比值2 < 0.7 且 比值1 ≥ 0.7 → 提示 **vWD 2N亚型或血友病A**，需加测FVIII抗原（FVIII:Ag）以鉴别
5. **识别血栓状态**：若vWF:Ag与vWF:A均升高，且比值1 ≥ 1.0，提示血栓性疾病可能。

> **注意**：健康人群的参考范围受ABO血型影响，解读时应结合患者血型背景。

## 禁忌与限制
- 不适用于单纯血小板功能障碍患者
- 不适用于未同步检测vWF指标的血友病A疑似病例
