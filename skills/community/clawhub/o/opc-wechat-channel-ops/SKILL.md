## 真人出镜标准

### 形象要求

- **着装**：商务休闲，浅色衬衫/Polo衫，体现专业感
- **发型**：干净利落，拍摄前整理
- **表情**：自然微笑，眼神自信
- **姿态**：坐姿端正/站姿挺拔，镜头感舒适

### 场景布置

| 场景类型 | 适用内容 | 布置要点 |
|----------|----------|----------|
| 书架背景 | 政策解读、行业洞察 | 摆放专业书籍、技术类册子 |
| 办公室 | 案例分享、经验总结 | 简洁桌面，展示专业氛围 |
| 纯色背景 | 统一风格、后期特效 | 浅灰/浅蓝/白色，避免杂乱 |

### 设备清单

- **拍摄设备**：手机（iPhone 12以上/安卓旗舰） 或 相机
- **稳定设备**：手机支架/三脚架（高度与视线平齐）
- **补光设备**：环形补光灯（推荐40cm直径）
- **收音设备**：领夹麦克风（BOYA BY-M1 或同等）
- **网络**：稳定的WiFi环境

### 标准1分钟脚本结构

```

【核心内容】(45-50秒)
- 要点1：[具体内容]
- 要点2：[具体内容]
- 要点3：[具体内容]

【老胡观点】(5-8秒)
我的看法是：...（一句话总结）

【行动号召 CTA】(3-5秒)
关注老胡，懂技术转化。评论区说说你的看法！
```

### 脚本风格规范

| 要求 | 说明 |
|------|------|
| 口语化 | 像和朋友聊天，不要照读稿 |
| 有温度 | 适当加入个人经历和感受 |
| 有钩子 | 前3秒必须抓住注意力 |
| 有价值 | 每条视频至少1个干货点 |
| 有互动 | 结尾引导评论和关注 |

## 拍摄流程

### 拍摄前准备

1. **确定选题**：参考运营日历，提前1天准备
2. **撰写脚本**：按模板写出完整话术
3. **准备道具**：数据/案例/截图等素材
4. **场景布置**：调整灯光、背景、角度
5. **设备检查**：电量、存储、麦克风测试

### 拍摄中要点

1. **景别选择**：中近景为主（胸部以上）
2. **眼神方向**：看镜头，而非看提词器
3. **语速控制**：每分钟150-180字
4. **重录原则**：卡壳超过3秒就重来
5. **多拍几条**：同一条录3-5遍，选最佳

### 后期处理

| 工具 | 用途 | 难度 |
|------|------|------|
| 剪映（手机） | 剪辑、加字幕、滤镜 | 简单 |
| 必剪（B站） | 剪辑、字幕、素材库 | 简单 |
| Premiere | 精细调色、专业剪辑 | 复杂 |

**标准后期流程**：
1. 导入素材 → 2. 剪辑拼接 → 3. 添加字幕 → 4. 调色统一 → 5. 添加片头片尾 → 6. 导出
