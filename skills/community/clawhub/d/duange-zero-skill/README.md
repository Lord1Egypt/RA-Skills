# Duange Zero Skill Creator

这是一个面向零基础用户的 Skill 创建助手。

它适合帮助完全不懂 AI、不懂编程、不知道什么是 Skill 的人，用普通话把一个想法变成可用的 Codex Skill。

核心特点：
- 用生活化问题引导用户
- 自动把大白话整理成 Skill 结构
- 默认生成 `SKILL.md`、`README.md`、`examples.md`
- 内置安全提醒
- 支持后续打包成 zip 分发

典型触发方式：

```text
我想做一个 Skill
帮我做一个小助手
我不会写 Skill，你一步步问我
把这个流程变成 Skill
我要给别人生成一个 Skill 包
```
