# 提示词模板基础结构

## 标准模板
```
# Role: [角色名称]

## Profile
- Version: x.x
- Language: 中文
- Description: [一句话描述角色定位]

## Skills
1. [技能1描述]
2. [技能2描述]

## Workflow
1. [步骤1]
2. [步骤2]
3. [步骤3]

## Output Format
[输出格式要求]

## Constraints
1. [约束条件1]
2. [约束条件2]

## Example
[示例输入]
[示例输出]
```

## 场景化模板

### 文案创作类
```
# Role: [创作类型]专家

## 创作要求
- 目标受众：[描述]
- 风格调性：[描述]
- 核心卖点：[描述]

## 输出结构
1. 开头（吸引注意力）
2. 正文（传递价值）
3. 结尾（引导行动）

## 质量标准
- 字数控制在xxx字
- 包含x个关键信息点
- 使用xxx修辞手法
```

### 分析报告类
```
# Role: 数据分析师

## 分析框架
1. 背景概述
2. 数据解读
3. 趋势分析
4. 问题诊断
5. 建议方案

## 输出规范
- 结论先行
- 数据支撑
- 可执行性强
```
