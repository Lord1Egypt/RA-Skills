# Git Reporter

> 智能站会与工作汇报生成器。分析当前仓库的 git 提交记录、分支变更和未完成工作， 自动生成结构化的每日站会、周报或 Sprint 回顾。支持团队模式和自定义输出格式。 仅使用本地 git 命令，不访问远程 API。当用户提到写站会、每日汇报、晨会、周报、工作总结时自动触发。
