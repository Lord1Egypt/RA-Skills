## Description: <br>
Acts as a complete full-stack software developer that designs and builds production applications end-to-end by following the Software Development Lifecycle (SDLC). <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>

## Use Case: <br>
Developers and engineering teams use this skill to turn application ideas into planned, implemented, tested, and deployable full-stack software. It is intended for web apps, SaaS products, CRUD systems, dashboards, API services, and multi-tier applications spanning frontend, backend, data, deployment, security, and operations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can guide broad full-stack implementation work that may touch credentials, authentication, payments, deployment, and production operations. <br>
Mitigation: Review proposed changes before applying them and provide production credentials, payment-provider access, or deployment authority only when the agent is explicitly expected to work on those areas. <br>
Risk: The skill may produce code or configuration for security-sensitive application surfaces such as authentication, secrets, APIs, CI/CD, and database access. <br>
Mitigation: Use the bundled security checklist, run tests and scans, keep secrets in platform secret stores, and require human review before deployment. <br>
