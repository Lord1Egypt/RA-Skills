## Description: <br>
Compresses JPG, PNG, WebP, and GIF images to reduce file size while preserving visual quality. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[loverun321](https://clawhub.ai/user/loverun321) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and content operators can use this skill to request image compression for common web image formats, including bulk optimization workflows. It is suitable when reducing image size is more important than keeping all image processing local. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may send user images or image URLs to an external paid service. <br>
Mitigation: Use it only for images that are acceptable to share with the service provider, and prefer a local compressor for private or sensitive images. <br>
Risk: The release advertises paid usage and includes insufficient disclosure about quota or consent before processing. <br>
Mitigation: Review expected charges and obtain user approval before invoking compression that consumes paid quota. <br>
