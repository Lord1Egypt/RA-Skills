# 小说AI创作助手 - 技能介绍

## 📦 技能名称
**小说AI创作助手** (Novel AI Writer)

**技能标识**：`novel-ai-writer`

**版本**：v1.1

---

## 🎯 一句话介绍
从大纲到成稿，AI帮你写完整本小说！

---

## ✨ 核心功能

### 1️⃣ 故事框架构思
- 从零生成完整大纲
- 支持多种故事结构（三幕式、英雄之旅、七点法）
- 自动设计人物成长弧和伏笔
- 提供详细的章节细纲

**适用场景**：
- "我想写个都市修仙小说"
- "帮我构思一个悬疑故事"

---

### 2️⃣ 续写/扩写
- 智能接续，风格完全一致
- 支持多种续写策略（衔接、插入、分支）
- 自动标注新增伏笔
- 提供续写思路说明

**适用场景**：
- "接着写第4章"
- "这里可以怎么发展？"

---

### 3️⃣ 文字润色
- 轻度/中度/重度三种模式
- 支持风格转换（古龙风、金庸风、网络小说风等）
- 提供润色报告和修改说明
- 保留原文精华，提升文字质量

**适用场景**：
- "帮我润色这段文字"
- "改成古龙风格"

---

### 4️⃣ 逻辑一致性检查
- 自动检查人物/时间线/空间/因果一致性
- 生成详细检查报告
- 提供修改建议和优先级
- 附带辅助脚本（可检查多章节）

**适用场景**：
- "检查前后逻辑"
- "人物设定是否一致？"

---

## 💡 使用场景示例

### 场景1：从零开始写小说
```
用户："我想写个都市修仙小说，主角从废柴逆袭成强者"

AI：（调用技能）
    1. 询问细节（篇幅、风格、特殊要求）
    2. 生成完整大纲（含人物成长弧、伏笔计划）
    3. 与你确认后，开始写第1章
```

---

### 场景2：续写已有内容
```
用户：[粘贴第1-3章内容]
      "接下来怎么发展？"

AI：（调用技能）
    1. 分析写作风格和情节进展
    2. 生成续写方案（含思路说明）
    3. 输出第4章内容
```

---

### 场景3：润色文字
```
用户：[粘贴一段文字]
      "帮我润色，让对话更自然"

AI：（调用技能）
    1. 分析文本问题（对话书面化、描写过度等）
    2. 提供润色后版本
    3. 说明主要修改
```

---

### 场景4：逻辑检查
```
用户：[粘贴第1-10章内容]
      "检查前后逻辑是否一致"

AI：（调用技能）
    1. 检查人物/时间线/空间/因果一致性
    2. 输出检查报告（含问题清单）
    3. 提供修改建议
```

## 📦 包含内容

### 📚 参考文档（5个）
1. **story-structure.md** - 故事结构指南（三幕式、英雄之旅等）
2. **character-development.md** - 人物塑造方法（成长弧、人物卡等）
3. **plot-design.md** - 情节设计技巧（起承转合、节奏控制等）
4. **writing-style.md** - 文字润色和风格指南（语言层面、风格层面）
5. **logic-checklist.md** - 逻辑一致性检查清单（人物、时间线、空间、因果）

### 📋 模板文件（2个）
1. **outline-template.md** - 小说大纲模板
2. **chapter-template.md** - 章节撰写模板

### 🛠️ 辅助脚本（2个）
1. **check_consistency.py** - 逻辑一致性检查工具（支持多文件）
2. **validate_outline.py** - 大纲完整性验证工具

### 📖 示例输出（3个）
1. **outline-example.md** - 完整大纲示例
2. **continuation-example.md** - 续写示例
3. **polishing-example.md** - 润色示例
