## Description: <br>
OW Strategy 3 Step (大恩专题策略三步法) helps agents draft real-estate marketing strategy through core challenge definition, solution and positioning, and communication planning. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[qutongyuan](https://clawhub.ai/user/qutongyuan) <br>

### License/Terms of Use: <br>

## Use Case: <br>
External real-estate marketing strategists, project teams, and agents use this skill to quickly structure a focused strategy proposal for topic planning, quick pitches, small projects, and time-constrained campaigns. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The optional Python helper can write JSON files outside its intended project folder when given a crafted project name. <br>
Mitigation: Use simple project names without slashes, absolute paths, or '..' until the maintainer validates names and enforces writes inside the project state directory. <br>
