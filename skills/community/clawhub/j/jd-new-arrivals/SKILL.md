# 京东自营新品首发

提供京东新品首发查询工具，展示最新上架商品，支持国补优惠和以旧换新，基于京东官方数据。仅筛选京东自营商品，品质有保障。默认精选好评≥97%、近30天销量≥1000的品质商品，结果不足时自动逐级降级（先降销量到≥500，再降到≥100，最后不限销量），保证每次查询都有推荐。

## 能做什么

- **新品查询**：展示京东最新上架的商品，已精选好评≥97%、销量≥1000的品质商品
- **国补优惠**：支持查看享受国补优惠的新品
- **以旧换新**：适合有旧设备需要置换的用户
- **品类筛选**：按关键词筛选特定品类的新品（支持模糊匹配，如"家电"可匹配"家用电器"）
- **多种排序**：综合评分、到手价升序、优惠力度降序
- **品质自定义**：可自定义最低好评率（97/98/99/100），指定后好评率不降级，只放宽销量门槛
- **动态降级**：结果不足时自动放宽销量门槛（1000→500→100→不限），保证推荐数量
- **翻页浏览**：每页50条，说"下一页"查看更多

## 不能做什么

- 不能下单购买，只提供商品信息和购买链接
- 不能设置新品到货提醒通知
- 不能查询新品的首发预约信息

## 使用示例

1. "京东最近有什么新品手机" → 按品类筛选
2. "帮我看看2000元以下的新品" → 按价格筛选
3. "京东新品首发有什么好货" → 返回全部第1页（50条）
4. "优惠力度最大的新品" → 按优惠力度排序
5. "下一页" → 查看更多商品

## 注意事项

- 新品首发价格可能为首发优惠价，后续可能调整
- 部分新品需要预约或抢购，注意活动时间
- 仅筛选京东自营商品，专卖店和个人店已自动剔除
- 好评率97%为品质底线，系统不会降级到97%以下
- **数据流向说明**：查询通过云端代理转发到京东官方API，代理不存储用户数据

## 使用提示

- keyword为空时返回全部新品推荐，每页50条
- 带品类关键词时结果较少属正常现象，说明符合条件的好货稀缺
- sort=discount按优惠力度排序，适合找优惠最大的新品
- 新品销量天然偏低，降级机制确保即使小众品类也有推荐
- 关注国补优惠的新品，实际到手价更低
- 以旧换新品类（手机、电脑、家电）的新品更新最频繁
- 每条商品包含商品图片URL（image_url），请用 ![商品名](image_url) 格式渲染图片，图片下方放购买链接
- 购买链接点击后自动识别设备，手机端有京东APP则唤起APP
