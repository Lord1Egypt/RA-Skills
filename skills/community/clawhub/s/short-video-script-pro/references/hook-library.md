## Hook + Transition Combos

### Curiosity → Payoff
```
HOOK: "The reason your [X] isn't working has nothing to do with [obvious thing]"
TRANSITION: "It's actually about [surprising factor]. Let me show you..."
```

### Pain → Solution
```
HOOK: "If [painful situation], you're making this one mistake"
TRANSITION: "Here's exactly what to do instead — step by step"
```

### Bold Claim → Proof
```
HOOK: "I [achieved result] using only [simple method]"
TRANSITION: "Here's the exact process I followed:"
```

## Platform-Specific Hook Adjustments

### TikTok / Douyin
- Start with text overlay immediately (many watch muted)
- First frame should be high-energy or visually surprising
- Hook must work without sound

### Instagram Reels
- Aesthetic matters — make first frame beautiful or intriguing
- "POV:" and "Tell me..." hooks perform very well
- Slightly longer attention span than TikTok

### YouTube Shorts
- Title = hook for discovery (write for search too)
- Can be slightly more informational, less viral-bait
- "In this video" doesn't work — just start the value

### WeChat Channels (视频号)
- Emotional resonance > shock value
- Familiar scenarios work better than edgy
- Mid-career professional audience responds to career/family topics
