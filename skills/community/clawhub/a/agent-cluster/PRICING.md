# 外贸硅基军团 - 定价策略

## 定价方案

### Free（免费版）
- **适合**：个人外贸SOHO、创业初期团队
- **功能**：
  - Shopify基础连接（单店铺）
  - 库存查询
  - 采购订单基础审批
  - 4个Agent协作（库存/采购/财务/物流）
  - 每日100次API调用
  - 基础链路追踪

### Pro - ¥9.9/月
- **适合**：成长型外贸企业
- **功能**：
  - Shopify + WooCommerce + Magento全平台
  - 多店铺管理（最多5个）
  - 完整A2A多Agent协作
  - 财务审核流
  - 每日1000次API调用
  - 高级链路追踪 + Mermaid可视化
  - 邮件/飞书审批通知

### Enterprise（企业版）
- **适合**：大型跨境电商企业
- **功能**：
  - 无限店铺连接
  - 自定义Agent工作流编排
  - 企业级审批流配置
  - 无限API调用
  - 专属合规审核Agent
  - 技术支持 + SLA保障
  - 本地化部署选项
