# 联合单细胞与空间转录组技术用于病理诊断

## 何时使用
当满足以下任一条件时启用本技能：
- 需要同时解析肿瘤内细胞类型组成及其在组织中的空间分布；
- 研究目标涉及细胞间互作网络与微环境结构对病理状态的影响；
- 单独使用单细胞或空间转录组技术无法满足分析深度或空间保真度要求。

## 执行步骤

1. **评估样本与平台条件**
   - 确认具备高质量单细胞悬液（用于scRNA-seq）或完整组织切片（用于空间转录组）；
   - 确保拥有高通量测序平台及多模态数据分析能力。

2. **执行单细胞转录组测序（scRNA-seq）**
   - 对分离的单细胞进行cDNA文库构建与高通量测序；
   - 构建细胞图谱，鉴定细胞亚群、功能状态及分化轨迹；
   - 记录特征基因集（marker genes）用于后续空间映射。

3. **执行空间转录组分析**
   - 在保留组织空间坐标的前提下进行原位转录组捕获；
   - 获取各空间点位的基因表达谱，注意当前技术通常为“近单细胞”分辨率。

4. **整合分析策略**
   - 将scRNA-seq鉴定的细胞亚群特征基因映射至空间转录组数据；
   - 重建细胞类型在原始组织中的空间分布；
   - 联合推断细胞间通讯、微环境分区及病理相关异质性模式。

5. **结果输出与建议**
   - 输出整合后的细胞-空间图谱及互作网络；
   - 推荐在后续研究中引入单细胞多模态组学（如转录组+蛋白组+表观组）以提升诊断精度。

> 注意：单细胞技术会丢失空间位置信息，而空间转录组目前难以达到真正单细胞分辨率。联合使用可互补短板，但需谨慎处理数据对齐与批次效应。
