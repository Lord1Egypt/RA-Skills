# 商品评分规则说明

## 评分维度及权重

### 1. 销售贡献度（权重30%）

**指标构成：**
- 支付金额占比（15%）：商品支付金额 / 店铺总支付金额
- 支付买家数占比（10%）：商品支付买家数 / 店铺总支付买家数
- 新客贡献度（5%）：新买家支付金额 / 商品总支付金额

**评分逻辑：**
- 支付金额占比 >= 10%：满分100
- 支付金额占比 5%-10%：80分
- 支付金额占比 2%-5%：60分
- 支付金额占比 < 2%：按比例计分（占比 * 3000）

### 2. 流量效率（权重25%）

**指标构成：**
- 流量转化率（15%）：支付买家数 / 访客数
- 曝光转化率（5%）：IPV / 曝光量
- 加购率（5%）：加购UV / 访客数

**评分逻辑：**
- 流量转化率 >= 5%：满分100
- 流量转化率 3%-5%：80分
- 流量转化率 1%-3%：60分
- 流量转化率 < 1%：按比例计分（转化率 * 2000）

### 3. 成长潜力（权重20%）

**指标构成：**
- 平台标签加分（10%）：
  - 优品（is_hqp=1）：+30分
  - 潜力品（is_pwp=1）：+20分
  - 商机品（is_sjp=1）：+15分
  - 镇店之宝（is_zdzb=1）：+10分
- 成长分层（10%）：
  - gyp_growth_level高层级：+20分
  - gyp_growth_level中层级：+10分

**评分逻辑：**
- 累计加分，最高100分

### 4. 营销ROI（权重15%）

**指标构成：**
- 广告ROI：支付金额 / 广告消耗
- 自然流量占比：（访客数 - 广告点击数）/ 访客数

**评分逻辑：**
- 广告ROI >= 5：满分100
- 广告ROI 3-5：80分
- 广告ROI 1-3：60分
- 广告ROI < 1：30分
- 无广告消耗但有销售：按自然流量占比计分

### 5. 商品健康度（权重10%）

**指标构成：**
- 服务能力加分（5%）：
  - 七天无理由退货：+10分
  - 48小时发货：+10分
  - 15天包换：+5分
- 库存充足性（3%）：
  - 库存 > 100：+10分
  - 库存 50-100：+5分
  - 库存 < 50：0分
- 退款率（2%）：
  - 退款率 < 5%：+10分
  - 退款率 5%-10%：+5分
  - 退款率 > 10%：0分

**评分逻辑：**
- 累计加分，最高100分

## 综合评分计算

**总分 = 销售贡献度得分 × 30% + 流量效率得分 × 25% + 成长潜力得分 × 20% + 营销ROI得分 × 15% + 商品健康度得分 × 10%**

## 商品分层标准

基于综合评分将商品分为四个运营层级：

### S级：重点推广品（总分 >= 80）
- **特征**：销售贡献大、流量效率高、成长潜力强
- **运营策略**：
  - 加大广告投放预算，抢占更多流量
  - 优化商品详情页和主图，提升转化率
  - 参与平台营销活动，扩大曝光
  - 保证充足库存，避免缺货
  - 定期优化标题和关键词，维持搜索排名

### A级：潜力培育品（总分 60-80）
- **特征**：某些维度表现优秀，但综合表现有提升空间
- **运营策略**：
  - 针对性优化短板维度（如流量不足则增加推广，转化低则优化详情页）
  - 适度增加广告预算，测试流量潜力
  - 关注数据变化，及时调整策略
  - 考虑参加小型营销活动积累销量

### B级：维持运营品（总分 40-60）
- **特征**：表现平平，贡献有限但稳定
- **运营策略**：
  - 维持现有运营节奏，不增加额外投入
  - 定期检查库存和价格竞争力
  - 可作为关联销售的辅助商品
  - 观察长期趋势，判断是否需要升级或优化

### C级：优化调整品（总分 < 40）
- **特征**：多维度表现不佳，资源投入产出比低
- **运营策略**：
  - 停止广告投放，避免浪费预算
  - 深度诊断问题根因（价格、质量、市场需求等）
  - 考虑优化或下架处理
  - 若库存积压可通过促销清仓
  - 释放资源聚焦高价值商品

## 特殊情况处理

1. **新品保护**：上架不足30天的商品，成长潜力得分 +20
2. **季节性商品**：根据淡旺季标签调整评分权重
3. **严选商品**：is_yx=1 的商品，综合得分 +5
4. **零销售商品**：最近1天无销售的商品单独标记，建议诊断原因
5. **数据异常**：发现指标异常（如转化率>50%）时需人工复核
