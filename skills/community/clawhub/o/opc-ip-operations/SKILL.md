# 知识产权运营Skill (IP Operations Skill)

## 概述

本Skill为科研成果转化提供完整的知识产权运营变现指导，涵盖**许可、转让、作价入股、质押融资、证券化、拍卖、组合运营**七大路径。所有分析均基于**定量模型**，权重来源于权威标准（国知局+学术+行业），数据实时从CNIPA/WIPO/USPTO/智慧芽等权威渠道验证。

## 核心原则

| 原则 | 说明 |
|------|------|
| **定量优先** | 估值、尽调、风险评估全部出数字，禁止纯定性描述 |
| **数据实时** | 所有数据必须标注来源和查询时间，超过30天需重新验证 |
| **权重可查** | 每个权重参数必须标注来源（国知局/学术论文/行业报告） |
| **方法闭环** | 每个公式包含：定义+参数含义+取值范围+来源标注 |

## 估值权重体系（五维度）

> **来源**：国知局+中国技术交易所《专利价值分析指标体系操作手册》+Park Y学术研究

| 维度 | 权重 | 子指标 | 来源依据 |
|------|------|--------|----------|
| **创新性** | 25% | 新颖性(10%)+创造性(10%)+技术先进性(5%) | 国知局技术维度+Park Y |
| **实用性** | 20% | 可实施性(8%)+解决实际问题(7%)+TRL(5%) | 国知局经济维度+TRL标准 |
| **独占性** | 20% | 保护范围(8%)+权利要求数(5%)+同族(4%)+排他性(3%) | 国知局法律维度+Smolka |

## 估值方法体系

### 成本法
```
V = C研发 × (1 + 合理利润率) × 技术贬值系数
技术贬值系数 = 1 - 累计折旧率
折旧率 = 已使用年限 / 技术生命周期
```

### 收益法
```
V = Σ [ Ci × 技术贡献率 / (1 + r)^i ]
技术贡献率：制造业15-30% / IT 25-40% / 生物医药30-50%
r = WACC + IP风险溢价(3-8%)
```

### 市场法
```
V = 可比交易价格 × 调整系数(5维度)
调整系数 = f(创新性, 实用性, 独占性, 战略赛道, 未来市场)
```

## 数据实时验证要求

1. **法律状态**：每次操作前必须在CNIPA pss-system实时查询
2. **同族引文**：通过WIPO PATENTSCOPE或USPTO API验证
3. **市场数据**：优先使用智慧芽开放平台(免费额度)
4. **标注格式**：`[数据来源] | [查询时间: YYYY-MM-DD HH:mm]`

## 评分卡速查

| 风险等级 | 综合评分 | 建议 |
|----------|----------|------|
| A级 | ≥80分 | 优先推进，可直接进入成果转化 |
| B级 | 60-79分 | 积极推进，需补充1-2项尽调 |
| C级 | 40-59分 | 补强后推进，需解决关键风险点 |
| D级 | <40分 | 暂缓，建议完善IP布局后再评估 |
