# Herb Five-Flavor Theory Application

## When to Use
- 分析单味中药的功效机制
- 指导临床辨证选药
- 设计或解析复方配伍逻辑
- 结合四气、归经、升降浮沉综合判断药性

## Execution Steps
1. **确认药味**：明确目标中药的味道（辛、甘、酸、苦、咸、淡、涩）。
2. **结合四气**：不可孤立看待五味，必须与寒、热、温、凉四气综合分析。
3. **按味推效**：依据五味对应的功能机制推导主要功效方向：
   - **辛味**：能行、能散 → 行气、发散、活血（如麻黄、陈皮、川芎）
   - **甘味**：能补、能和、能缓 → 补益、调和、缓急止痛（如人参、甘草、饴糖）
   - **酸味**：能收、能涩 → 收敛固涩（如五味子、乌梅）
   - **苦味**：能泄、能燥 → 通泄、降泄、清泄、燥湿（如大黄、杏仁、黄连、苍术）
   - **咸味**：能下、能软 → 泻下通便、软坚散结（如芒硝、海藻）
   - **淡味**：能渗、能利 → 利水渗湿（用于水肿、小便不利）
   - **涩味**：作用类酸，主收敛固涩
4. **匹配证候**：将推导出的功效方向对应到具体中医证候（如气滞血瘀、虚损、久泻、实热、瘰疬等）。
5. **综合判断**：结合归经与升降浮沉理论，完善对药物作用部位与趋向的理解。

> **注意**：本技能不适用于无明确五味归属的现代合成药物。
