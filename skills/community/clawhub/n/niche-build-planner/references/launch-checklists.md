# Launch Checklists

Load this file ONLY when the user reaches Milestone 4 (Polish & Launch).

## Website Launch Checklist

- [ ] Domain purchased and DNS configured
- [ ] SSL certificate active (auto with Vercel/Netlify)
- [ ] Meta tags set: title, description, OG image for each page
- [ ] sitemap.xml generated and submitted to Google Search Console
- [ ] robots.txt configured
- [ ] Google Search Console verified
- [ ] Analytics tracking active (Vercel Analytics or Umami)
- [ ] Error monitoring active (Sentry free tier)
- [ ] Lighthouse score > 90 on mobile and desktop
- [ ] Responsive design tested on: iPhone, Android, iPad, desktop
- [ ] Payment flow tested end-to-end (test mode → live mode)
- [ ] Privacy policy page added
- [ ] Terms of service page added
- [ ] Contact/support channel visible (email, Twitter, or chat widget)

## Mini Program Launch Checklist

- [ ] WeChat developer account registered and verified
- [ ] AppID configured in project
- [ ] All pages render correctly on iOS and Android WeChat
- [ ] Navigation flow works without dead ends
- [ ] WeChat login flow tested on real device
- [ ] Search keywords configured (up to 10, use opportunity-finder results)
- [ ] Category selected (pick least competitive relevant category)
- [ ] Privacy policy page added (required for review)
- [ ] User data handling compliant with 微信小程序平台运营规范
- [ ] Payment flow tested (if applicable — requires merchant account)
- [ ] Share functionality works (forward to friend, share to moments)
- [ ] Submitted for review
- [ ] Review feedback addressed (if rejected)
- [ ] Published after approval

## Post-Launch (Day 1-7)

- [ ] Monitor error reports daily
- [ ] Set up user feedback channel
- [ ] Track the 3 success metrics from product-scoper
- [ ] Share on relevant communities (V2EX, Reddit, ProductHunt)
- [ ] Ask first 10 users for detailed feedback
