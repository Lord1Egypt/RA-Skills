# 特殊肿瘤命名情形详解

## 1. 母细胞瘤（-blastoma）
- **良性示例**：脂肪母细胞瘤、肾母细胞瘤（部分亚型）
- **恶性示例**：神经母细胞瘤、视网膜母细胞瘤
- 判断依据：需结合组织学分级与临床行为

## 2. 名称含“病”或“瘤”但实为恶性
- 白血病（leukemia）：造血系统恶性肿瘤
- 精原细胞瘤（seminoma）：睾丸生殖细胞恶性肿瘤
- 黑色素瘤（melanoma）：皮肤恶性肿瘤

## 3. 人名命名肿瘤
- 尤因肉瘤（Ewing sarcoma）
- 霍奇金淋巴瘤（Hodgkin lymphoma）
- 伯基特淋巴瘤（Burkitt lymphoma）

## 4. 形态命名肿瘤
- 透明细胞肉瘤（clear cell sarcoma）
- 小细胞癌（small cell carcinoma）
- 大细胞癌（large cell carcinoma）

## 5. 瘤病（-omatosis）
- 神经纤维瘤病（neurofibromatosis）：常染色体显性遗传，多发神经纤维瘤
- 血管瘤病（angiomatosis）：多发血管瘤

## 6. 畸胎瘤分类
- **成熟性畸胎瘤**：含三个胚层，组织分化良好 → 良性（ICD-O /0）
- **未成熟性畸胎瘤**：含未分化神经外胚层成分 → 恶性（ICD-O /3）
- **恶性转化畸胎瘤**：含明确癌或肉瘤成分 → 按具体成分编码
