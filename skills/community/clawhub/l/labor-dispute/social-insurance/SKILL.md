# 社保维权路径

## 一、社保的强制性

**法律依据：劳动法第72条**

> 国家发展社会保险事业，建立社会保险制度，设立社会保险基金，使劳动者在年老、患病、工伤、失业、生育等情况下依法从国家和社会获得物质帮助。

- 社保是强制性义务，用人单位和劳动者必须依法参加
- 缴纳社保是双方的法定义务，不能通过约定放弃
- 费用由用人单位（单位部分）和劳动者（个人部分）共同缴纳
- 缴费基数、缴费比例按当地社保经办机构规定执行

**未缴社保的后果：**
- 社保行政部门责令限期缴纳
- 逾期不缴的，加收滞纳金（劳动法第100条）
- 给劳动者造成损害的，用人单位承担赔偿责任

## 二、单位不缴社保，劳动者可以立即解除合同

**法律依据：劳动合同法第38条第三项**

> 用人单位有下列情形之一的，劳动者可以解除劳动合同：（三）未依法为劳动者缴纳社会保险费的；

这是劳动者的即时解除权，**不需要提前30天通知**，也不需要向用人单位说明理由。

### 解除合同后的经济补偿

依据劳动合同法第46条第一项：**因用人单位过错导致劳动者解除合同的，用人单位应当支付经济补偿**。

经济补偿按第47条计算：
- 工作每满1年 = 1个月工资
- 6个月以上不满1年 = 按1年计算
- 不满6个月 = 半个月工资
- 月工资 = 解除合同前12个月的平均工资

### 程序要点

劳动合同法第37条规定，普通情况下劳动者辞职应提前30日书面通知。但**第38条规定的即时解除权**，不需要提前通知。

操作步骤：
1. 收集证据：劳动合同、工资条、个税记录、工作证等，证明劳动关系和薪资水平
2. 向用人单位送达书面解除通知（注明理由：未缴社保）
3. 办理工作交接
4. 要求出具解除劳动合同证明（第50条，用人单位应在15日内办理）
5. 主张经济补偿

## 三、补缴社保找社保局，仲裁不受理

**法律依据：最高人民法院关于审理劳动争议案件适用法律问题的解释（一）第2条**

> 劳动者与用人单位因社会保险发生争议，当事人不服劳动争议仲裁机构作出的裁决，依法向人民法院提起诉讼的，人民法院应予受理。

但注意：**社会保险争议在仲裁阶段可能不被受理，或者被告知应向社会保险经办机构或劳动行政部门投诉**。

司法实践中：
- **能补缴的** → 向社保经办机构投诉举报，要求责令单位补缴
- **不能补缴的** → 就损失赔偿向劳动争议仲裁委员会申请仲裁

### 向社保局投诉的流程

1. 准备材料：
   - 劳动合同
   - 工资支付记录（银行流水/工资条）
   - 个税申报记录
   - 考勤记录
   - 身份证明

2. 向当地社保经办机构（或劳动监察大队）提交投诉

3. 社保部门核查劳动关系和欠缴金额

4. 责令用人单位限期补缴

5. 用人单位逾期不缴 → 加收滞纳金 → 申请法院强制执行

## 四、无法补缴时的赔偿

**法律依据：劳动合同法第38条 + 第46条 + 第47条**

当社会保险确实无法补缴（如用人单位已经注销、跑路），劳动者可以主张：

### 赔偿无法享受社保的损失

计算逻辑：
- 根据当地社保政策，推算劳动者本应享受的社保待遇（如养老金、医疗保险待遇）
- 折算为现金赔偿

司法实践中常见做法：
- 参考当地社保缴费标准，按应缴年限计算单位应承担的社保费用
- 部分法院支持劳动者主张用人单位赔偿其无法享受社保的实际损失

### 主张步骤

1. 确认无法补缴（社保部门出具书面证明）
2. 计算损失（可咨询社保部门或律师）
3. 向劳动争议仲裁委员会申请仲裁
4. 对裁决不服 → 15日内向人民法院起诉

---

## 五、执行步骤总结

```
步骤1：确认未缴事实
  ↓
  收集证据：劳动合同 + 工资记录 + 个税记录 + 考勤
  ↓
步骤2：选择维权路径

  路径A：继续工作 + 投诉补缴
    → 向社保经办机构投诉
    → 责令补缴 + 滞纳金

  路径B：解除合同 + 主张补偿
    → 书面通知解除（理由：未缴社保）
    → 办结交接
    → 主张经济补偿（每满1年赔1个月工资）
    → 可同时要求补缴

  路径C：无法补缴 → 主张赔偿
    → 社保部门出具无法补缴证明
    → 计算损失
    → 申请仲裁 → 起诉
```

## 六、关键法律条文

| 条款 | 内容要点 |
|---|---|
| 劳动法第72条 | 强制参加社保，费用由单位和劳动者共同缴纳 |
| 劳动法第73条 | 劳动者依法享受社会保险待遇 |
| 劳动法第100条 | 逾期不缴社保可加收滞纳金 |
| 劳动合同法第38条第三项 | 未缴社保 → 劳动者可即时解除合同 |
| 劳动合同法第46条第一项 | 因用人单位过错解除 → 支付经济补偿 |
| 劳动合同法第47条 | 经济补偿计算标准（每满1年赔1个月工资） |
| 劳动合同法第50条 | 解除合同出具证明 + 15日内办理档案社保转移 |
| 司法解释(一)第2条 | 社会保险争议仲裁前置，不服可起诉 |

## 七、注意事项

**证据意识：**
- 劳动合同是最重要的证据
- 工资条、银行流水、个税记录可证明实际收入
- 工作证、考勤记录、聊天记录可辅助证明劳动关系

**时效问题：**
- 劳动争议申请仲裁的时效为一年（从当事人知道或应当知道权利被侵害之日起）
- 补缴社保投诉无明确时效限制，但越早投诉越好

**关键区分：**
| 情形 | 补偿类型 | 金额 |
|---|---|---|
| 未缴社保 + 你主动解除合同（第38条） | 经济补偿 | N |
| 未缴社保 + 单位违法解除 | 赔偿金 | 2N |

> **记住**：未缴社保员工解除 = N（不是2N）。2N只在你被违法解除时才能主张。
