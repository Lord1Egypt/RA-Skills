## 一、买东西前的决策框架

### 需求分层筛选器
**适用场景**：任何消费决策，尤其是冲动消费

**步骤**：
1. 拿起东西或加入购物车前，先停3秒
2. 问自己四个问题：
   - 我需要它吗？（没有会影响健康/生存吗？）
   - 我爱它吗？（不考虑钱，这是我真正想要的吗？）
   - 我喜欢它吗？（只是当下觉得不错？）
   - 我想要它吗？（只是临时冲动？）
3. 判断标准：
   - 只有前两个（Need/Love）值得花钱
   - 后两个（Like/Want）是陷阱，放下

**经济学概念**：需求分层、边际效用递减

---

### 价格真实性检验
**适用场景**：看到"打折""促销""限时优惠"

**步骤**：
1. 遮住"原价"，只看"现价"
2. 问自己：如果我一直知道它卖这个价，我还会觉得便宜吗？
3. 判断标准：
   - 是 → 可能是真实需求
   - 否 → 被锚定效应骗了，不买

**经济学概念**：锚定效应、消费者剩余

## 二、时间与工作的决策框架

### 机会成本计算器
**适用场景**：选A还是选B、时间怎么分配、要不要做某事

**步骤**：
1. 列出你要选的选项（比如：加班 vs 学习）
2. 问自己：如果我选A，我放弃了B的什么好处？
3. 量化（如果可能）：
   - 加班3小时 = 200元加班费
   - 学习3小时 = 未来可能涨薪2000元的机会
4. 判断标准：
   - 放弃的 > 得到的 → 不选
   - 得到的 > 放弃的 → 选

**经济学概念**：机会成本、时间折现

---

### 比较优势分工表
**适用场景**：该自己做还是外包、团队任务分配、副业选择

**步骤**：
1. 列出你所有的技能/任务
2. 对每个任务，问自己：
   - 我做这件事的效率是别人的几倍？
   - 这件事是我的核心价值吗？
3. 判断标准：
   - 核心价值任务 → 自己做
   - 非核心但效率高 → 可以考虑自己做
   - 非核心且效率低 → 外包或交给别人

**经济学概念**：比较优势、二八法则

---

### 副业可行性评估
**适用场景**：想增加收入，不知道从何开始

**步骤**：
1. 列出你现有的技能（哪怕很小）
2. 问自己：
   - 这个技能能服务谁？
   - 他们愿意为此付多少钱？
   - 我能不能用"最小行动"验证？
3. 判断标准：
   - 能立即进账的短期工作 → 零工（先做起来）
   - 能持续产出且可复制 → 副业（规模化）
   - 收入超过主业且稳定 → 考虑转主业

**经济学概念**：比较优势、技能变现、规模化收入

## 三、财务健康诊断框架

### 月度财务体检
**适用场景**：每月复盘财务状况

**步骤**：
1. 收入到账立刻分流：
   - 家庭账单（房租/水电/保险）
   - 个人账单（债务还款）
   - 日常消费（吃饭/交通/购物）
   - 应急储蓄（至少3个月生活费）
   - 梦想基金（投资/旅行/学习）
2. 问自己：
   - 日常消费账户花完了吗？
   - 有没有动其他账户的钱？
   - 应急基金够3个月吗？
3. 判断标准：
   - 日常消费不超支 + 应急基金充足 → 健康
   - 经常超支或挪用 → 需要调整预算

**经济学概念**：强制储蓄、心理账户、系统思维

---

### 债务清偿优先级
**适用场景**：有多笔债务，不知道先还哪个

**步骤**：
1. 列出所有债务：金额、利率、最低还款额
2. 按利率从高到低排序
3. 每月还款策略：
   - 所有债务还最低还款额
   - 剩余的钱全部砸向利率最高的那笔
   - 还清一笔后，把这笔钱加到下一笔上
4. 判断标准：
   - 利率 > 10% → 优先还清
   - 利率 5-10% → 平衡还债和储蓄
   - 利率 < 5% → 可以慢慢还，同时投资

**经济学概念**：债务雪球、时间折现

---

### FIRE进度测算
**适用场景**：想知道自己离财务自由还有多远

**步骤**：
1. 计算年开销：每月开销 × 12
2. 计算FIRE目标：年开销 × 25
3. 计算当前进度：现有积蓄 ÷ FIRE目标
4. 问自己：
   - 按当前储蓄率，需要多少年？
   - 如果降低开销，能提前多少年？
5. 判断标准：
   - 进度 < 25% → 专注节流和增收
   - 进度 25-50% → 开始投资
   - 进度 50-75% → 优化投资组合
   - 进度 > 75% → 规划退休后的生活

**经济学概念**：FIRE公式、复利、时间折现

## 四、关系中的金钱决策

### 伴侣财务透明对话
**适用场景**：和伴侣谈钱、制定共同财务目标

**步骤**：
1. 各自列出：收入、债务、固定支出
2. 共同讨论：
   - 我们的共同目标是什么？（买房？旅行？FIRE？）
   - 各自承担什么比例？
   - 有多少"自由支配"空间？
3. 建立系统：
   - 共同账户（家庭开支）
   - 个人账户（自由支配）
4. 判断标准：
   - 双方对目标一致 + 对分工认可 → 健康
   - 一方回避或隐瞒 → 需要深入沟通

**经济学概念**：金钱观传承、财务透明、系统思维

## 五、快速决策清单

### 5秒决策法
**适用场景**：小事快速决策，不纠结

| 问题类型 | 问自己 | 决策标准 |
|---------|--------|---------|
| 买不买 | "没有它，我的生活会受影响吗？" | 会 → 买；不会 → 不买 |
| 做不做 | "1年后我会后悔没做吗？" | 会 → 做；不会 → 不做 |
| 选A还是B | "哪个选项的机会成本更低？" | 放弃的更可惜 → 选另一个 |
| 要不要继续 | "再投入的边际效用还大于成本吗？" | 是 → 继续；否 → 停止 |
| 现在做还是以后做 | "现在做的额外收益是什么？" | 收益 > 成本 → 现在做 |

## 六、每日行动检查表

### 早上（5分钟）
- [ ] 查看日常消费账户余额
- [ ] 确认今天有没有"必须花"的钱
- [ ] 提醒自己：需求 > 热爱 > 想要 > 喜欢

### 花钱前（3秒）
- [ ] 我需要它吗？
- [ ] 我爱它吗？
- [ ] 这是Like/Want还是Need/Love？

### 晚上（5分钟）
- [ ] 记录今天花了多少
- [ ] 有没有冲动消费？
- [ ] 明天有什么可以优化？

### 每月（30分钟）
- [ ] 复盘各账户支出
- [ ] 检查应急基金
- [ ] 评估副业/投资进度
- [ ] 调整下月预算
