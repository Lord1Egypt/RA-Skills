# Changelog

## 1.0.0 (2026-06-03)

### ✨ Features
- 首次公开发布
- 完整的 Slidev 语法指南（frontmatter / layout / transition / v-click）
- 常用布局模板（cover / default / center / two-cols / section）
- 语法特性参考（代码高亮 / 图标 / Mermaid / LaTeX）
- 自动初始化脚本（init-slidev.sh）
- 导出命令支持（PDF / PPTX / PNG）
- PPTX 反解支持（python-pptx 提取 + AI 重建）
- 内容溢出检测与自动拆页策略

## 1.0.1 (2026-06-03)

- 补充多种安装方式和推荐操作流程
