# 一致性检查规则

## 角色外观一致性

每次生成视频前，必须对照 `CHAR_XXX.md` 的**外观锁定项**逐项核对：

### 必检项

| 检查项 | 标准 |
|-------|------|
| 脸型 | 与锁定项一致 |
| 发型/发色 | 与锁定项完全一致，禁止换发型 |
| 肤色 | 与锁定项一致 |
| 服装 | 与锁定款一致，禁止换服装 |
| 身高体型 | 与锁定项一致 |
| 标志性特征 | 眼镜/纹身/疤痕等与锁定项一致 |
| 道具搭配 | 耳机、手机等与绑定场景一致 |

### 通过标准

- 全部必检项与资产卡一致 → ✅ 通过
- 任何一项不一致 → ❌ 返工

---

## 场景一致性

对照 `SCENE_XXX.md` 核对：

| 检查项 | 标准 |
|-------|------|
| 场景类型 | 与锁定场景一致 |
| 构图比例 | 与设定比例一致（16:9 / 9:16 / 1:1）|
| 光线/色调 | 与参考图一致 |
| 背景道具 | 与场景锁定项一致 |

---

## 道具一致性

| 检查项 | 标准 |
|-------|------|
| 道具外观 | 与 `PROP_XXX.md` 一致 |
| 绑定角色 | 与镜头卡中声明的角色一致 |
| 使用状态 | 符合该道具在场景中的用途 |

---

## 表情/姿态一致性

| 检查项 | 标准 |
|-------|------|
| 表情 | 必须在 `CHAR_XXX` 表情库范围内 |
| 手势 | 必须在 `CHAR_XXX` 手势库范围内 |
| 禁止 | 表情/手势超出库的范围 |

---

## 提示词 Positive/Negative

### Positive Prompt 规则

必须包含：
- 角色外观关键词（短发、淡金色头发等）
- 服装关键词（白色休闲服）
- 场景关键词（游戏房、室内）
- 质量关键词（写实风格、高质量）

### Negative Prompt 规则

必须包含：
- 禁止换发型的词
- 禁止换服装的词
- 禁止不同演员的词

---

## 返工阈值

以下情况必须进入返工流程：

1. 角色外观任何一项与锁定项不符
2. 场景类型或主要道具与设定不符
3. 表情/手势超出库的范围
4. 连续 2 次生成一致性检查失败
5. 视频中出现未绑定的道具

---

## 检查执行人

每次生成视频后：
- **AI** 自动对照资产卡初步检查
- **用户** 最终确认是否通过
- 有争议时以资产卡锁定版本为准
