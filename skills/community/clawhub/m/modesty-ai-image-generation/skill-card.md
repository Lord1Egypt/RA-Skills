## Use Case: <br>
Developers, designers, marketers, and other external users use this skill to generate, edit, upscale, and route image-generation requests through the SkillBoss API Hub. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill sends image prompts, referenced image URLs, and generated outputs to the SkillBoss API. <br>
Mitigation: Use the skill only when SkillBoss is trusted for that data, and avoid sending secrets, sensitive personal data, confidential business data, or private image URLs. <br>
Risk: The skill requires a user-provided SkillBoss API key. <br>
Mitigation: Use a limited API key where possible and store it in the SKILLBOSS_API_KEY environment variable rather than hard-coding it in prompts or files. <br>
