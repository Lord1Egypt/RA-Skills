# 香豆素类抗凝药（华法林）出血监测与逆转

## 适用场景
- 患者服用华法林进行房颤、深静脉血栓或瓣膜置换术后抗凝
- 需评估抗凝强度是否在治疗窗内
- 出现出血症状（如鼻衄、血尿、颅内出血）或INR显著升高
- 妊娠女性误用华法林（绝对禁忌）

## 执行步骤
1. **监测抗凝强度**：
   - 测定PT（凝血酶原时间），目标18–24秒（正常约12秒）
   - **优先使用INR**（国际标准化比值）：
     - 房颤/深静脉血栓：INR 2.0–3.0
     - 机械瓣膜（高危）：INR 2.5–3.5

2. **评估出血风险**：
   - INR >4.0：出血风险显著增加
   - INR >10.0 或活动性出血：需紧急干预

3. **出血处理**：
   - **立即停用华法林**
   - **轻度升高（INR 4–10）无出血**：给予维生素K 1–2.5 mg 口服
   - **严重出血或INR >10**：
     - 静脉注射维生素K 5–10 mg（缓慢输注≥30分钟）
     - 同时输注新鲜冰冻血浆（FFP）或凝血酶原复合物（PCC）补充功能性凝血因子

4. **特殊注意事项**：
   - **孕妇禁用**：华法林可通过胎盘，致胎儿华法林综合征（鼻发育不全、骨骺点状钙化）
   - **皮肤坏死风险**：用药后3–7天出现，可能与蛋白C/S抑制有关；起始剂量勿过大（建议≤5 mg/日）
   - **起效延迟**：抗凝作用需12–24小时显现，1–3天达峰

> **输出结果**：通过PT或INR判断抗凝强度是否在治疗窗内，并据此调整剂量或干预出血。
