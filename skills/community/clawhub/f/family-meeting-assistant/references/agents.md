> 版本: v2.0.0 | 创建日期: 2026-06-07
> 
> 本文档定义家语的工作流程和规范。

## 工作流程

家语的工作流程分为4个阶段：

### 阶段1：日常记录（随时）

1. **接收记录**
   - 接收家庭成员的文字/语音/图片
   - 提取事件（人物、事件、类别、时间）
   - 归档到家庭时间轴

2. **主动追问**
   - 发现值得记录的时刻，主动询问细节
   - 但不强迫回答

3. **准备材料**
   - 读取历史记录
   - 准备周期性整理材料

### 阶段2：周期整理（每周/每月）

1. **家庭周记**（每周生成）
   - 本周发生
   - 每个人变化（成长、情绪、生活）
   - 值得纪念

2. **家庭月报**（每月生成）
   - 本月新增记录统计
   - 孩子成长
   - 家庭活动
   - 重要决定
   - 家庭关键词

3. **AI洞察**（可选）
   - 积极发现（如"最近三个月家庭活动明显增加"）
   - 风险提醒（如"已连续45天没有家庭共同活动记录"）

### 阶段3：成长追踪（持续）

1. **成员成长档案**
   - 每个人拥有成长页
   - 记录成长变化（第一次骑车、学会游泳等）
   - 能力成长曲线（阅读、运动、社交）

2. **家庭成长图谱**
   - 展示家庭关系网络（谁与谁互动更多）
   - 共同活动趋势（如"2025年家庭活动12次，2026年28次"）

### 阶段4：传承沉淀（每年/持续）

1. **家庭编年史**（每年底生成）
   - 年度大事
   - 重要照片
   - 成员成长
   - 家庭感悟
   - 生成电子书

2. **长辈故事采访**（持续）
   - AI主动采访长辈
   - 如"您小时候住在哪里？"、"第一次工作的经历是什么？"
   - 逐步形成家族口述史

3. **家庭文化沉淀**（持续）
   - 形成家庭价值观（如诚信、陪伴、学习）
   - 来自长期记录分析

## 数据存储规范

```

│   └── 2026/

│       ├── 2026-04.md   # 2026年4月事件
│       └── ...
├── memory/               # 家语时刻记录
│   └── 2026/
│       └── 2026-06.md
├── growth/               # 成员成长档案
│   ├── 爸爸.md
│   ├── 妈妈.md

│   └── ...
├── legacy/               # 家庭传承资料
```

### 文件格式

**members.json**（家庭成员信息）：
```json
[
  {
    "name": "爸爸",
    "role": "家庭主力",
    "wecom_id": "zhangsan",
    "birthday": "1980-05-15",
    "permissions": ["创建记录", "审批决策"]
  }
]
```

**timeline/2026/2026-06.md**（家庭时间轴 - 2026年6月）：
```markdown
# 2026年6月家庭时间轴

## 2026-06-07
- **孩子第一次骑自行车**（成长记录）
  - 记录人：妈妈
  - 详情：今天下午，小明第一次独立骑自行车，没有摔倒。
- **全家厦门旅行**（家庭活动）
  - 参与人：爸爸、妈妈、小明
  - 详情：周五出发，周日回来，住了两晚。

## 2026-06-14
- **爸爸换工作**（重要事件）
  - 记录人：爸爸
  - 详情：决定接受新公司的offer，下个月入职。
```

**growth/小明.md**（成员成长档案）：
```markdown
# 小明成长档案

## 2026年
- **2026-06-07** - 第一次独立骑自行车
```

## 定时任务配置

### Cron 任务列表

| 任务名称 | 执行时间 | 说明 |
|---------|---------|------|
| 家语时刻 | 每天20:00 | 每天一句，降低表达门槛 |
| 家庭周记 | 每周日20:00 | 生成本周家庭周记 |
| 家庭月报 | 每月1号20:00 | 生成上月家庭月报 |
| 家庭编年史 | 每年12月31日 | 生成年度家庭编年史 |
| 长辈故事采访 | 每月15号20:00 | 主动采访长辈故事 |

### Cron 配置示例

```json
{
  "name": "家语时刻",
  "schedule": {"kind": "cron", "expr": "0 20 * * *", "tz": "Asia/Shanghai"},
  "payload": {"kind": "agentTurn", "message": "发送今日家语时刻问题到家庭群"},
  "sessionTarget": "isolated"
}
```

## 技能依赖

### 必需技能
- `jiayu` - 核心能力（专用skill包，从 `family-meeting-assistant` 升级而来）
- `wecom-weisheng-scrm` - 企微消息推送
- `tencent-docs` - 家庭记录存储到腾讯文档（可选）

### 可选技能
- `xlsx` - 家庭成长档案导出
- `pdf` - 家庭编年史导出为PDF
- `online-search` - 搜索旅游地点、餐厅推荐等
