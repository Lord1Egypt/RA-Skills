# 连接叙事模板

## 适用场景
客户考察、常规签约（小型）、领导视察、行业交流、项目考查

## 事件级别
L2 标准（绝大多数情况）

## 结构骨架

```
相遇缘起 → 互动现场 → 共识发现 → 下一步行动
```

## 填写表

| 段落 | 写什么 | 字数 | 铁律 |
|------|--------|------|------|
| 相遇缘起 | 对方为什么来/为什么走到一起 | 40-60字 | 写真实的动因，不说"慕名而来" |
| 互动现场 | 一个有画面感的细节 | 60-80字 | **白描**，不夸张 |
| 共识发现 | 双方找到了什么共同点 | 50-70字 | 具体，有内容 |
| 下一步 | 具体的后续动作 | 30-50字 | 可跟进，可验证 |

## 变量槽

| 变量 | 说明 | 示例 |
|------|------|------|
| {{VISITOR}} | 考察方名称+背景 | XX集团供应链总监李总 |
| {{REASON}} | 真实来意 | 华南仓周转效率遇到瓶颈 |
| {{SCENE_DETAIL}} | 现场画面细节 | 在产线前站了20分钟，问了三个问题 |
| {{CONSENSUS}} | 具体共识 | 先把基础打牢，再谈智能 |
| {{NEXT_ACTION}} | 后续动作 | 下个月去总部带方案 |

## 自检

- [ ] 是"李总带团队来了"还是"XX公司一行莅临"？（要前者）
- [ ] 互动现场有画面感吗？能"看见"吗？
- [ ] 共识是具体内容还是"达成广泛共识"？（要前者）
- [ ] 下一步能跟进吗？

## 常见问题

| 问题 | 修正 |
|------|------|
| "莅临参观指导" | 写对方真实的职务和来意 |
| "深入友好的交流" | 写一个具体的画面或对话 |
| "达成广泛共识" | 写清楚共识的具体内容 |

## 示例

> **相遇缘起**：李总本来只是路过广州，顺道来看看。他在我们的系统前站了半小时，问了三个问题。
>
> **互动现场**：他没看PPT，直接打开自己的手机说："你看，这是我们现在的流程，从下单到出库要过7个人。"
>
> **共识发现**：双方都认同一件事——数字化不是把人替代掉，是让7个人变成3个人，剩下4个人去做更有价值的事。
>
> **下一步**：约了下个月去他们总部。带上针对7人流程的改进方案。
