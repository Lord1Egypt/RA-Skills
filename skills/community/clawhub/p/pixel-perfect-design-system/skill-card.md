## Description: <br>
像素级完美设计系统 - 为 Vibe coding 应用提供专业级站点设计。支持中英文双语站点，58 个顶级产品设计模板，9 部分完整设计规范，AI 可读 Markdown 格式，开箱即用无需 Figma。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, designers, and AI coding agents use this skill to generate AI-readable DESIGN.md guidance for product-style UI systems, including colors, typography, layout rules, component styling, responsive behavior, and prompt guidance. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: A user-provided output path can overwrite an existing file. <br>
Mitigation: Choose output paths deliberately, avoid sensitive locations, and review the target path before running the generator. <br>
Risk: Documentation badges may contact external badge sites when a Markdown viewer loads remote images. <br>
Mitigation: Use a Markdown viewer that blocks remote images or review generated documentation offline when external image loading is not acceptable. <br>
