## 链路概述

把四轴连接成运行路径：

| 链路 | 起点 | 终点 | 断裂点 |
|------|------|------|--------|
| 控制链 | 目标 | 反馈 | 执行不自主 |
| 解释链 | 行为 | 认同 | 理由不稳定 |
| 能量链 | 兴趣 | 强化 | 试错被惩罚 |
| 责任链 | 选择 | 自我认知 | 责任被外包 |

---

## 控制链

### 正常路径

```
目标（方向定义）
    ↓
路径（选择空间）
    ↓
行为（自主执行）
    ↓
反馈（现实校正）
```

### 断裂表现

**断在执行层**：
- 孩子不是"做"，而是"被做"
- 行为是外部驱动的表演
- 缺乏内在执行意愿

**断在路径层**：
- 没有选择空间
- 只有唯一路径
- 探索被消灭

**断在目标层**：
- 目标是父母的
- 孩子不知道"为什么"
- 方向感缺失

### 诊断问题

1. 孩子的行为有多少是自主驱动的？
2. 孩子是否只有在"被要求"时才行动？
3. 孩子的探索空间有多大？
4. 孩子是否知道"为什么"要做？

---

## 解释链

### 正常路径

```
行为（孩子的行动）
    ↓
理由（理解原因）
    ↓
内化（接受逻辑）
    ↓
认同（形成自律）
```

### 断裂表现

**断在理由层**：
- 只有命令，没有解释
- "因为我说"
- "别问为什么"

**断在内化层**：
- 解释不稳定，随情绪变化
- 今天是A理由，明天是B理由
- 孩子无法建立稳定认知

**断在认同层**：
- 表面服从，内心抵抗
- 规则是外部强加的
- 没有真正认同

### 诊断问题

1. 孩子的行为有多少有清晰的内部理由？
2. 父母的解释是否前后一致？
3. 孩子是否能用自己的话解释规则？
4. 孩子的服从是认同还是被迫？

---

## 能量链

### 正常路径

```
兴趣（自发好奇）
    ↓
探索（尝试行动）
    ↓
成功/失败（结果反馈）
    ↓
强化（动力积累）
```

### 断裂表现

**断在兴趣层**：
- 兴趣被筛选（"这没用"）
- 只能有"有用"的兴趣
- 探索欲望被压制

**断在探索层**：
- 行动被中断（"你这样不对"）
- 过程被监控
- 探索无法深入

**断在失败层**：
- 失败被提前惩罚
- "错了不行"
- 试错成本太高

**断在强化层**：
- 结果被否定（"这不算好"）
- 进步不被认可
- 动力无法积累

### 诊断问题

1. 孩子还有自发产生的兴趣吗？
2. 孩子的探索行为是否被允许？
3. 孩子失败后的体验是什么？
4. 孩子的进步是否被认可？

---

## 责任链

### 正常路径

```
选择（自主决定）
    ↓
行为（执行决定）
    ↓
结果（承担后果）
    ↓
自我认知（"我是什么样的人"）
```

### 断裂表现

**断在选择层**：
- 选择被替代
- "你决定"但选项都是父母定的
- 伪选择

**断在结果层**：
- 结果与选择不关联
- 好结果归父母，坏结果归孩子
- 因果链断裂

**断在认知层**：
- 无法从结果中学习
- "我是什么样的人"无法形成
- 主体感缺失

### 诊断问题

1. 孩子的选择有多少是真正自主的？
2. 选择与结果是否直接关联？
3. 好结果和坏结果的归因是否公平？
4. 孩子能否从结果中建立自我认知？

## 链路联动与断裂诊断

### 四链联动

```
控制链 → 解释链 → 能量链 → 责任链
   ↓
  循环
```

健康状态下，四链形成正向循环：
- 控制链清晰 → 解释链稳定
- 解释链内化 → 能量流顺畅
- 能量流闭环 → 责任归属明确
- 责任闭环 → 控制权被接受

### 断裂联动

失衡状态下，四链形成负向循环：
- 控制权集中 → 解释权外置
- 解释权混乱 → 能量流冻结
- 能量流中断 → 责任归属模糊
- 责任模糊 → 控制权更加集中

### 诊断顺序

1. **先查控制链**：控制权是否清晰分布？
2. **再看解释链**：解释权是否内化？
3. **然后能量链**：能量闭环是否完整？
4. **最后责任链**：责任归属是否明确？

### 修复顺序建议

按照链路顺序逆向修复：
1. 先重建控制链的分布式结构
2. 再帮助解释链内化
3. 然后修复能量链闭环
4. 最后重建责任链

## 链路断裂的典型模式

### 模式一：高压型断裂

```
控制链：完全集中
解释链：外部强加
能量链：兴趣被消灭
责任链：结果归父母
```

表现：孩子极度顺从但缺乏内在动力，表面配合但内心抵抗。

### 模式二：混乱型断裂

```
控制链：没有控制
解释链：没有解释
能量链：没有方向
责任链：无法归因
```

表现：孩子行为缺乏边界，无法建立结构，表面"自由"但内心焦虑。

### 模式三：混合型断裂

```
控制链：时而控制时而放手
解释链：解释随情绪变化
能量链：时而鼓励时而打压
责任链：归因混乱
```

表现：孩子无法建立稳定预期，规则不可预测，安全感缺失。

## 应用指南

### 诊断时

1. 逐条链路检查：每个节点是否有断裂？
2. 识别主要断裂链：哪条链是核心问题？
3. 分析断裂联动：断裂如何相互影响？

### 干预时

1. 从主要断裂链开始
2. 按顺序修复（逆向链路）
3. 观察联动效应
4. 动态调整策略

## 与依恋理论的整合

### 依恋视角下的四轴分析

**安全型家庭**：
- 四轴通常分布健康
- 控制权清晰，解释权可内化
- 能量流顺畅，责任归属明确

**不安全型家庭**：
- 控制链：过度控制（焦虑型）或情感疏离（回避型）
- 解释链：外部强加或缺失
- 能量链：依恋焦虑/回避影响能量流动
- 责任链：边界混乱或责任不清

### 依恋类型诊断

| 依恋类型 | 控制链 | 解释链 | 能量链 | 责任链 |
|----------|--------|--------|--------|--------|
| 焦虑型 | 过度控制、边界模糊 | 不稳定、随焦虑变化 | 过度依赖外部 | 责任不清、相互依赖 |
| 回避型 | 控制或忽视 | 缺失或冷漠 | 压抑、冻结 | 责任混乱或逃避 |
| 混乱型 | 不可预测、矛盾 | 混乱、恐惧关联 | 极度不稳定 | 责任完全混乱 |

## 与代际传递的整合

### 四轴的代际来源

**控制轴的代际来源**：
- 父母被过度控制 → 习得控制模式
- 父母被忽视 → 不知如何给予空间

**解释轴的代际来源**：
- 父母被简单指令 → 不懂得解释
- 父母有创伤 → 用控制处理焦虑

**能量轴的代际来源**：
- 父母能量被压制 → 不知如何保护孩子能量
- 父母缺乏支持 → 无法提供情感滋养

**责任轴的代际来源**：
- 父母责任被外包 → 不懂得让孩子承担
- 父母过度负责 → 替孩子承担一切

### 代际觉察问题

诊断四轴时，同时追问：
1. 这个模式在我小时候是什么样的？
2. 我父母是如何处理这个维度的？
3. 我是在复制还是在改变？
4. 我愿意为孩子选择什么不同的方式？

详细指南见 [references/intergenerational-transmission.md](intergenerational-transmission.md)。
