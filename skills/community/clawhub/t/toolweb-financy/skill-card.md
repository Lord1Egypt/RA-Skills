## Description: <br>
Professional personal finance management API with analytics, multi-currency support, and transaction tracking. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, financial advisors, and organizations building consumer finance tools use this skill to manage transactions, retrieve dashboard analytics, list categories, and export finance records through the Financy API. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can access, export, modify, and delete personal financial records without clear authentication or safety controls in the artifact evidence. <br>
Mitigation: Review before installing or using with real financial data; only use it if the ToolWeb/Financy service is trusted and authentication, per-user authorization, protected short-lived export links, and confirmation before transaction changes or exports are verified. <br>
