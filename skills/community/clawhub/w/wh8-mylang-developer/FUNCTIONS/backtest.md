## 一、AUTOFILTER

### 基本语法
```mql
AUTOFILTER;
```
**必须写！** 放在模型最后一行。

### 作用
- 过滤重复信号
- 保证一开一平机制

### 示例
```mql
MA5:=MA(C,5);
MA10:=MA(C,10);

CROSS(MA5,MA10), BK;
CROSSDOWN(MA5,MA10), SP;

AUTOFILTER;  // 必须写！
```

## 二、回测参数设置

### 设置入口
- 菜单：【量化】→【设置保证金/手续费的量化交易参数】
- 回测K线图工具条按钮：临时调整初始资金

### 主要参数

| 参数 | 说明 | 设置位置 |
|------|------|----------|
| 初始资金 | 回测起始资金 | 模组设置/回测工具条 |
| 保证金率 | 合约保证金比例 | 量化交易参数 |
| 手续费率 | 交易手续费 | 量化交易参数 |
| 信号计算起始时间 | 从何时开始计算信号 | 模组设置/右键菜单 |

### 设置方法

#### 菜单设置（全局默认）
```
右上角菜单【量化】→ 设置保证金/手续费的量化交易参数
```

#### 临时调整（回测时）
- 回测K线图工具条中的按钮
- 设置当前合约的临时参数

### 信号计算起始时间

> ⚠️ 重要：数据计算起始 ≠ 信号计算起始
> 详细说明见官方文档：https://www.wenhua.com.cn/popwin/xhjsqzsj.html

```
数据起始 = 合约数据开始的时间（挂牌日期）
信号起始 = 你开始实际交易的时间
```

**示例**：
```
沪铜2201合约
数据起始：2021年1月18日（挂牌）
信号起始：2021年11月1日（开始交易）

信号计算起始时间 = 2021年11月1日
```

**为什么区分？**
- 合约挂牌初期交易不活跃，缺少流动性
- 你不会从挂牌就开始交易
- 信号计算应该从有实际交易价值的时间开始

## 三、回测报告指标

### 收益指标

| 指标 | 说明 |
|------|------|
| 收益率 | 总收益/本金 |
| 年化单利收益率 | 收益率/(测试天数/365) |
| 年化复利收益率 | (最终权益/本金)^(365/测试天数)-1 |
| 夏普比率 | (年化收益-无风险利率)/(收益标准差×√250) |
| 索提诺比率 | (年化收益-无风险利率)/(下行收益标准差×√250) |

### 风险指标

| 指标 | 说明 |
|------|------|
| 权益最大回撤 | 权益从高点到低点的最大跌幅 |
| 权益最大回撤比率 | 回撤/回撤时的权益 |
| 本金风险率 | (本金-最小权益)/本金 |
| 年化收益风险比率 | 年化收益/本金风险率 |

### 交易统计

| 指标 | 说明 |
|------|------|
| 胜率 | 盈利次数/总交易次数 |
| 盈亏比率 | 平均盈利/平均亏损 |
| 交易次数 | 完整买卖次数 |
| 最大持续亏损次数 | 最大连续亏损次数 |
| 单次最大亏损 | 单笔交易最大亏损 |

## 四，主连链回测

### 适用场景
- 中低频趋势策略
- 需要多年历史数据验证
- 对多个年度主力月份合约连续交易效果回测

### 优势
- 用月份合约K线数据计算信号
- 避免主连数据换月跳空
- 新主力重新计算信号

### 模型要求
```mql
// 不要写 TRADE_OTHER('ZHULIAN')
// 直接把模型加载到品种主连上
```

### 机制说明
1. 信号用各月份合约挂牌以来的完整K线数据计算
2. 主力切换时旧主力清仓
3. 新主力重新计算信号（延续旧主力清仓后的资金）

## 回测操作流程

### 历史信号调试
1. 加载模型到回测K线图
2. 查看信号明细（右键 → 信号明细）
3. 核对信号位置是否正确

### 参数优化
1. 主图回测后右键 → 枚举/遗传
2. 设置参数范围和步长
3. 根据参考标准筛选参数

### 主连链回测
1. 模型加载到品种主连
2. 右键 → 模型回测报告
3. 可查看月份合约独立报告

### 敏感性测试
1. 回测后右键 → 敏感性测试
2. 分析滑点/手续费变化对收益的影响
3. 评估策略实战能力

## 常见问题

### Q: 保证金、手续费在哪里设置？
A: 右上角菜单【量化】→ 设置保证金/手续费的量化交易参数

### Q: 初始资金在哪里分配？
A: 
- 回测K线图工具条按钮（临时调整）
- 右上角菜单【量化】→ 设置默认回测参数（默认设置）

### Q: 如何补充更久的数据？
A: 回测K线图右键 → 补充历史数据

### Q: 主连链回测用什么数据？
A: 用各月份合约挂牌以来的完整K线数据，不直接用主连K线图数据

### Q: 信号计算起始时间怎么设？
A: 设置为你实际开始交易的时间，不是数据开始的时间。详细说明见：https://www.wenhua.com.cn/popwin/xhjsqzsj.html
