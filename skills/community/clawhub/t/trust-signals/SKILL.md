# Trust Signals

Audit product pages for missing or weak trust signals including reviews, guarantees, certifications, and social proof that affect purchase confidence. This skill systematically identifies every element that makes a shopper hesitate and prescribes specific copy, design, and social proof fixes — ordered by conversion impact.

## Quick Reference

| Decision | Strong | Acceptable | Weak |
|----------|--------|------------|------|
| Review count visible above fold | 100+ reviews shown with star rating | 25–99 reviews shown | No review count or <25 |
| Guarantee language | Specific ("60-day money-back, no questions") | General ("satisfaction guaranteed") | No guarantee mentioned |
| Certifications displayed | Certified badges with verifiable links | Certification text only | Certifications absent |
| Social proof type | Video testimonials + UGC photos | Text reviews + star rating | Star rating only |
| Trust badge placement | Near add-to-cart button | In product description | Footer only |
| Return policy clarity | Free return + exact timeline + process | Return accepted + timeline | Vague or absent |
| Payment security indicators | SSL badge + recognized payment logos near checkout | SSL badge only | No security indicators |

## Solves

1. **High add-to-cart, low checkout rate** — Shoppers are interested but something at checkout triggers doubt; missing security badges and return policy clarity are the primary culprits
2. **Low conversion from cold traffic** — Visitors who don't know the brand need 3× more trust signals than warm audiences; missing certification and press mentions lose these buyers
3. **Review count anxiety** — Products with <25 reviews lose new visitors before they read a single bullet; review count must be visible above the fold
4. **Guarantee hesitation** — Vague guarantees ("satisfaction guaranteed") are worse than no guarantee; specificity ("full refund in 60 days, keep the product") converts better
5. **Mobile trust gaps** — Desktop trust signals (sidebars, secondary images) collapse on mobile; trust badges must be in the main content flow
6. **Post-purchase doubt (buyer's remorse)** — Weak confirmation emails and no order-tracking trust signals increase refund requests
7. **Cross-border purchase friction** — International buyers need local payment methods, local language return policies, and local social proof to convert

## Workflow

### Step 1 — Audit Inputs
Collect:
- Product page URL(s) to audit
- Platform (Shopify, Amazon, TikTok Shop, Shopee, brand website)
- Target market (country/region — trust signals are culture-specific)
- Current conversion rate if known (benchmark context)
- Top 3 competitor URLs for comparison

### Step 2 — Above-the-Fold Trust Audit
Check everything visible without scrolling on mobile (375px) and desktop (1280px):
- Star rating and review count visible?
- Brand name / logo clearly present?
- At least one trust badge (secure payment, free returns, certified)?
- Price presented with confidence framing (compare-at price, "limited stock")?
- Primary image professional and lifestyle-appropriate?

### Step 3 — Social Proof Depth Audit
| Signal Type | Present? | Strength (1–5) | Fix |
|-------------|---------|----------------|-----|
| Star rating average | | | |
| Total review count | | | |
| Photo reviews | | | |
| Video testimonials | | | |
| UGC / influencer content | | | |
| Press mentions | | | |
| "Customers also bought" | | | |
| Q&A with answered questions | | | |
| Verified purchase badges | | | |

### Step 4 — Guarantee & Return Policy Audit
Evaluate the guarantee statement against the Trust Signal Hierarchy:
1. **Tier 1 (Strongest):** "Full refund + keep the product, no questions asked, 90 days"
2. **Tier 2:** "Full refund within 60 days, free return shipping included"
3. **Tier 3:** "Full refund within 30 days, buyer pays return shipping"
4. **Tier 4 (Weak):** "Satisfaction guaranteed" with no specifics
5. **Tier 5 (Harmful):** No guarantee mentioned; "all sales final"

### Step 5 — Certification & Compliance Signal Audit
Check which certifications apply to the product category and whether they're displayed:
- Food/supplements: FDA, NSF, GMP, USDA Organic
- Electronics: FCC, CE, UL, RoHS, Energy Star
- Beauty: Cruelty-Free (Leaping Bunny), Vegan, EWG Verified, Dermatologist Tested
- Textiles: OEKO-TEX, Fair Trade, GOTS Organic
- Baby/Kids: ASTM, CPSC, BPA-free (with documentation)

### Step 6 — Payment & Security Signal Audit
Verify near the purchase CTA:
- SSL padlock and "secure checkout" text
- Accepted payment method logos (Visa, Mastercard, PayPal, Apple Pay, Klarna)
- Buy Now Pay Later option (Klarna, Afterpay) for items >$50
- Local payment methods for target markets (Alipay for CN, GrabPay for SEA)

### Step 7 — Prescribe Fixes (Prioritized)
Rank fixes by conversion impact × implementation ease:
- **Quick wins (1–3 days):** Trust badge placement, guarantee copy rewrite, review count display
- **Medium effort (1–2 weeks):** Review request campaign, UGC collection, certification upload
- **High effort (1+ month):** Video testimonials, press outreach, third-party certification

## Examples

### Example 1 — Shopify Supplement Store Audit

**Page:** Collagen peptides product page, 87 reviews, 4.6 stars

**Audit findings:**
- ❌ Review count not visible above fold (requires scroll past product image)
- ❌ Guarantee says "30-day guarantee" with no further detail
- ❌ No GMP or third-party tested badge displayed despite having certifications
- ✅ SSL badge present near checkout
- ❌ No photo reviews visible in first 3 reviews displayed
- ❌ Return shipping policy buried in FAQ (2 clicks away)

**Prescribed fixes (ordered by impact):**
1. Move star rating + review count to directly below product title — **+8–12% CTR to reviews**
2. Rewrite guarantee: "60-day full refund, no questions asked — just contact us and we'll process it within 24 hours" — **estimated +5–7% add-to-cart**
3. Add GMP and NSF badges to product image gallery (slide 3–4) and near buy button — **estimated +3–5% checkout rate**
4. Prioritize photo reviews in display order — **estimated +4–6% purchase confidence**
5. Add "Free returns, we cover shipping" line directly under price — **estimated +2–4% checkout completion**

### Example 2 — Amazon Electronics Listing (Earbuds, 312 reviews)

**Audit findings:**
- ✅ 312 reviews, 4.4 stars shown above fold
- ❌ FCC certification only in compliance section (not highlighted in bullets)
- ❌ Warranty buried in bullet 5 ("1-year warranty" — no claim process explained)
- ❌ No comparison to previous model or competitor — shoppers unsure of upgrade value
- ✅ "Frequently bought together" active and relevant

**Prescribed fixes:**
1. Bullet 3: Add FCC certification call-out — "FCC and CE certified for safe use in US and EU"
2. Bullet 5 rewrite: "BACKED BY OUR 18-MONTH WARRANTY — Free replacement or full refund; register at [brand].com/warranty within 60 days for extended protection"
3. A+ comparison table: Add vs. previous model column — reduces buyer research friction

## Common Mistakes

1. **Generic badges with no credibility** — "Trusted by 10,000+ customers" with no verifiable link or third-party source is dismissed as marketing copy
2. **Hiding the return policy** — If a buyer must click away to find return terms, assume they won't; put the key terms (timeline, cost, process) on the product page
3. **Star rating without review count** — "4.8 ★" with no number of reviews raises suspicion of manufactured ratings; always show the count
4. **Certifications not linked to verification** — Claiming "NSF Certified" without a link to the NSF certification registry is a conversion leak and a compliance risk
5. **Desktop-only trust badges** — Sidebars collapse on mobile; badges placed only in the sidebar are invisible to 60%+ of traffic
6. **Overwhelming the buyer with badges** — More than 6 trust badges near the CTA creates visual noise and decision paralysis; pick the 3 most relevant
7. **Ignoring negative reviews** — Leaving negative reviews unanswered signals poor customer service; a professional response converts fence-sitters better than removing the review
8. **Weak guarantee framing** — "30-day guarantee" vs. "30-day guarantee — if you're not completely happy, we'll refund every cent" converts significantly differently
