# Xue Fu Zhu Yu Tang Application

## When to Use
- 胸中血瘀证患者，表现为明确的瘀血内阻于胸部、气机郁滞。

## When to Use
确认患者具备以下全部主症：
- 胸痛、头痛如针刺且部位固定
- 呃逆持续日久
- 心悸失眠
- 唇色紫暗
- 舌有瘀斑
- 脉象涩或弦紧

## Precautions
- 孕妇禁用
- 月经过多者禁用
- 无瘀血证候者不宜使用

## Execution Steps
1. **确认方剂组成**：桃仁四钱、红花三钱、当归三钱、生地黄三钱、川芎一钱半、赤芍二钱、牛膝三钱、桔梗一钱半、柴胡一钱、枳壳二钱、甘草二钱。
2. **煎服方法**：水煎服。
3. **解析配伍结构**：
   - 君药：桃仁、红花（破血行滞、活血化瘀）
   - 臣药：川芎、赤芍（助活血止痛），牛膝（祛瘀通脉，引血下行）
   - 佐药：柴胡与枳壳（疏肝理气），桔梗（开宣肺气以助血行），生地与当归（养血活血，防祛瘀伤阴）
   - 使药：甘草（调和诸药）
4. **理解核心原则**：体现“活血不伤血，行气以助血行”的配伍思想，源自桃红四物汤合四逆散加桔梗、牛膝。
5. **明确现代适应症**：可用于冠心病心绞痛、风湿性心脏病、胸部挫伤、肋软骨炎、脑震荡后综合征、精神抑郁属瘀阻气滞者。

> 注意：本技能仅适用于中医辨证为胸中血瘀证的场景，不可用于无瘀血表现的患者。
