# Lung Yin Deficiency Differentiation

## When to Use
- 患者主诉长期干咳，痰少而黏，或反复咯血（血色鲜红）
- 伴随口干咽燥、声音嘶哑、形体消瘦
- 出现虚热征象：午后潮热、颧红、五心烦热、盗汗
- 舌象为舌红少津或少苔，脉象为细数
- 有久咳伤阴或痨虫袭肺病史

**排除情形**：若咳嗽伴明显恶寒发热、鼻塞流涕、痰多清稀或舌苔厚腻，则不属于本证。

## Execution Steps
1. **确认主症**：干咳无痰，或痰少而黏，或痰中带血。
2. **观察兼症**：口干咽燥、声音嘶哑、形体消瘦、颧红、潮热、盗汗。
3. **查验舌脉**：舌红少津（或少苔/无苔），脉细数。
4. **分析病机**：肺阴不足 → 虚火内灼 → 肺气上逆为咳；肺津被灼 → 痰少质黏；虚火灼络 → 痰中带血；津不上承 → 口干咽燥；虚热扰营 → 盗汗潮热。
5. **确定治法**：滋阴润肺，清虚热，宁络止血。
6. **推荐基础方**：百合固金汤（生地黄、熟地黄、麦冬、贝母、百合、当归、白芍、甘草、玄参、桔梗）。
7. **指导调养**：忌食辛辣燥热之品，避免耗伤肺阴。

> 注意：本证需与肝火犯肺、燥邪伤肺等证鉴别。若咯血量多、潮热显著或咽干严重，应参考加减用药指南。
