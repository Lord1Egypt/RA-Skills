# Turbos CLMM SDK Skill for OpenClaw

OpenClaw skill for integrating Turbos Finance CLMM SDK on Sui.

## Install

```bash
npx clawhub install turbos
```

## Modules

Contract, Pool, Trade, Position, Account, Math
