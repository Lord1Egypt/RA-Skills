# 面试题库与评分细则

## 模块一：AI流利度面试

### 基础题（L1-L2区分）

1. **请描述你日常使用AI工具的方式和频率。**
   - L1信号：主要是ChatGPT问答、代码补全
   - L2信号：根据不同任务选择不同工具，有固定的prompt模板

2. **用过的AI工具中，哪个让你印象最深？为什么？**
   - L1信号：回答停留在功能层面
   - L2信号：能对比不同工具的优劣，说出选择理由

### 进阶题（L2-L3区分）

3. **如果让你为公司选一个AI工具栈，你会怎么选？考虑哪些因素？**
   - L2信号：列出工具但缺乏系统性思考
   - L3信号：考虑成本/隐私/集成/学习曲线等多维度，有决策框架

4. **你如何评估一个新AI工具是否值得引入？**
   - L2信号：试用后凭感觉判断
   - L3信号：有评估框架（ROI/团队适配/维护成本等）

### 高阶题（L3-L4区分）

5. **如果让你设计一个团队级别的AI使用规范，你会包含哪些内容？**
   - L3信号：技术层面的规范（用什么工具、怎么写prompt）
   - L4信号：包含安全/合规/人机分工/效果评估等全方位规范

### 评分细则

| 分数 | 表现 |
|------|------|
| 1 | 只会用基础工具，表达停留在功能层面 |
| 2 | 能主动选择工具，有自己的使用心得 |
| 3 | 有工具评估框架，能系统性思考工具链 |
| 4 | 能设计规范，影响他人和团队 |

## 模块二：人机判断力面试

### 基础题（L1-L2区分）

1. **你遇到过AI输出错误的情况吗？怎么处理的？**
   - L1信号：没太注意/偶尔发现一两次
   - L2信号：能举出具体案例，有检查的习惯

2. **你怎么判断AI给的建议是否靠谱？**
   - L1信号：看起来合理就信了
   - L2信号：有检查清单（检查逻辑/数据来源/运行测试等）

### 进阶题（L2-L3区分）

3. **请设计一个验证AI输出质量的方法。**
   - L2信号：人工抽查/简单对比
   - L3信号：自动化验证流程/设定质量指标/建立反馈机制

4. **什么场景下你会选择不用AI？为什么？**
   - L2信号：回答模糊
   - L3信号：能说出具体场景（高精度/合规/创造性决策等）及原因

### 高阶题（L3-L4区分）

5. **如果让你制定一个部门的AI使用边界，你会怎么定义"AI可以做"和"AI不能做"？**
   - L3信号：技术层面的划分
   - L4信号：从业务风险/法律责任/组织能力等角度系统性定义

### 评分细则

| 分数 | 表现 |
|------|------|
| 1 | 基本信任AI，验证意识弱 |
| 2 | 有验证习惯，能识别常见问题 |
| 3 | 有系统验证方法，知道AI边界 |
| 4 | 能定义组织级规范，设计人机协作流程 |

## 模块三：综合能力面试（微案例）

### 案例1：效率提升

**背景**：假设你是某电商公司的技术负责人，公司每天有大量客服工单需要人工分类和回复。

**问题**：
1. 你会如何用AI解决这个问题？
2. 预期的效果和风险是什么？
3. 如何评估这个方案是否成功？

**评分关注点**：
- L1：直接说"用ChatGPT自动回复"
- L2：考虑分类+回复两阶段，但缺乏风险评估
- L3：设计完整方案（分类准确率/人工兜底/效果监控/成本核算）
- L4：不只解决当前问题，还考虑如何重构客服流程（AIfirst）

### 案例2：技术选型

**背景**：团队要在产品中集成AI能力，需要在"调用API"和"自建模型"之间选择。

**问题**：
1. 你会怎么决策？考虑哪些因素？
2. 两个方案的利弊是什么？
3. 如果选择调用API，如何确保数据安全？

**评分关注点**：
- L1：简单二选一
- L2：能列出一些因素但不够系统
- L3：系统性评估（成本/延迟/隐私/可控性/维护成本），知道Tradeoff
- L4：从长期战略角度思考，考虑组织能力建设

### 案例3：流程重构

**背景**：公司运营团队每月花大量时间做数据报表，流程是"导出数据→Excel处理→做PPT"。

**问题**：
1. 你如何重新设计这个流程？
2. 引入AI后，人和AI的分工是怎样的？
3. 如何让运营团队接受这个新流程？

**评分关注点**：
- L1：建议用AI生成报表
- L2：拆解流程找到可AI化的环节
- L3：重新设计端到端流程，明确人机分工
- L4：不只解决报表问题，推动运营团队整体AI转型
