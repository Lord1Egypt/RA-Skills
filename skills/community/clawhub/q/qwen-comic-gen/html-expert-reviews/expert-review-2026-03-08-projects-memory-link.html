🎯 规范动作指导（一步步完成）

步骤 1：识别关键词规范（5 分钟）

目标：定义什么内容应该被识别为任务

规范：

  • ✅ [待办] [进行中] [规划] 标记的内容
  • ✅ 会议纪要中的"行动项""TODO""Action"
  • ✅ 包含"要""需要""计划""准备"等动词的句子
  • ❌ 纯想法记录（无明确行动）
  • ❌ 已完成事项（除非需要跟进）

步骤 2：提取内容规范（5 分钟）

目标：统一任务描述的格式

规范：

  • 格式：动词 + 宾语 + （可选）预期结果
  • 示例：✅ "配置飞书 OAuth（获取 user_access_token）"
  • 示例：❌ "飞书 OAuth"（缺少动词）
  • 长度：≤50 字，超出部分放入备注

步骤 3：专家分析规范（10 分钟）

目标：建立任务分类的决策树

决策流程：

  1. 有明确截止时间？→ 是 → 紧急
  2. 影响重大业务价值？→ 是 → 重要
  3. 依赖他人？→ 是 → 需要跟进
  4. 5 分钟内能完成？→ 是 → 立刻做

步骤 4：优先级规范（10 分钟）

目标：四象限分类的明确标准

象限 标准 策略 示例
P0 紧急重要 有截止 + 影响大 立刻做 飞书 OAuth（影响任务管理）
P1 重要不紧急 无截止 + 影响大 排期做 地理知识库（长期价值）
P2 紧急不重要 有截止 + 影响小 委托/批量 临时会议
P3 不紧急不重要 无截止 + 影响小 延后/不做 浏览社交媒体
