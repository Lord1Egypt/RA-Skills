## Known Risks and Mitigations: <br>
Risk: The -o/--output option writes to the user-provided path and may replace an existing file depending on how it is used. <br>
Mitigation: Use explicit output paths, avoid sensitive locations, and review the target path before writing formatted JSON. <br>
