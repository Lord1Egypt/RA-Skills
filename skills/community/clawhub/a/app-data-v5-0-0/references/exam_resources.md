### 重点分录示例模板

**F-05 长期股权投资（权益法）**
```
【业务场景】对被投资单位具有重大影响或共同控制
【口诀】权益法，看权益，净利润要调公允
【标准分录】
1. 初始投资成本调整（成本<可辨认净资产公允份额的差额）
   借：长期股权投资——投资成本
     贷：营业外收入
2. 被投资单位实现净利润
   借：长期股权投资——损益调整
     贷：投资收益
   （注意：调整投资时点公允价值与账面价值差额对净利润的影响）
3. 被投资单位宣告分配现金股利
   借：应收股利
     贷：长期股权投资——损益调整
4. 被投资单位其他综合收益变动
   借：长期股权投资——其他综合收益
     贷：其他综合收益
5. 被投资单位除净损益、其他综合收益以外的其他权益变动
   借：长期股权投资——其他权益变动
     贷：资本公积——其他资本公积
【易错点】
❌ 忘记调整投资时点的公允价值与账面价值的差额
❌ 内部交易未实现损益未抵销
❌ 超额亏损未按顺序冲减
```

**F-10 收入（五步法）**
```
【业务场景】客户取得相关商品控制权时确认收入
【口诀】识别合同识别义务，确定价格分摊价格，最后确认收入
【核心分录】
1. 满足时点履约义务
   借：银行存款/应收账款
     贷：主营业务收入
       应交税费——应交增值税（销项税额）
   借：主营业务成本
     贷：库存商品
2. 满足时段履约义务（按履约进度）
   借：合同结算——收入结转
     贷：主营业务收入
   借：主营业务成本
     贷：合同履约成本
3. 附有销售退回条款的销售
   借：应收账款
     贷：主营业务收入
       预计负债——应付退货款
       应交税费——应交增值税（销项税额）
   借：主营业务成本
     应收退货成本
     贷：库存商品
【易错点】
❌ 混淆时点履约与时段履约的判断条件
❌ 可变对价估计错误
❌ 重大融资成分未考虑
```

### 重点公式示例模板

**G-35 杜邦分析法**
```
【公式】净资产收益率 = 营业净利率 × 总资产周转率 × 权益乘数
- 营业净利率 = 净利润 / 营业收入  （盈利能力）
- 总资产周转率 = 营业收入 / 平均总资产  （营运能力）
- 权益乘数 = 平均总资产 / 平均所有者权益 = 1 / (1 - 资产负债率)  （偿债能力/杠杆）

【口诀】营净率、周转率、权益乘，三者相乘ROE
【适用条件】杜邦分析体系，用于综合评价企业盈利能力
【典型例题】
某公司营业净利率5%，总资产周转率2次，资产负债率50%，求净资产收益率。
解：权益乘数 = 1/(1-50%) = 2
    净资产收益率 = 5% × 2 × 2 = 20%
【易错点】
❌ 总资产用期末数还是平均数？——用平均数更准确
❌ 权益乘数与资产负债率的换算错误
❌ 混淆杜邦三因素各自反映的能力维度
```

**G-20 应收账款机会成本**
```
【公式】应收账款机会成本 = 日销售额 × 信用期/平均收现期 × 变动成本率 × 资本成本率
= 应收账款平均余额 × 变动成本率 × 资本成本率

【口诀】日销乘信用期，变动成本资本率，四数相乘机会成
【适用条件】应收账款信用政策决策
【典型例题】
某企业年销售额3600万元，信用期30天，变动成本率60%，资本成本率10%，求应收账款机会成本。
解：日销售额 = 3600/360 = 10万元
    应收账款平均余额 = 10 × 30 = 300万元
    机会成本 = 300 × 60% × 10% = 18万元
【易错点】
❌ 忘记乘以变动成本率（固定成本不占用资金）
❌ 资本成本率用税前还是税后？——题目没说就用给定的
❌ 有现金折扣时，平均收现期的计算错误
```

**G-21 经济订货基本模型**
```
【公式】EOQ = √(2KD/Kc)
- EOQ：经济订货批量
- K：每次订货的变动成本
- D：存货年需求量
- Kc：单位变动储存成本
【相关公式】
- 最佳订货次数 = D/EOQ
- 最佳订货周期 = 360/最佳订货次数
- 相关总成本 = √(2KDKc)
- 经济订货量平均占用资金 = EOQ/2 × 单价

【口诀】二KD除以Kc，开根号得EOQ
【适用条件】
- 存货年需求量稳定且能预测
- 存货单价不变，无数量折扣
- 货物集中到货，不是陆续入库
- 仓储条件和所需资金不受限制
【典型例题】
某材料年需求量7200千克，每次订货变动成本200元，单位变动储存成本2元/年，求经济订货批量。
解：EOQ = √(2×7200×200/2) = √1,440,000 = 1200千克
    最佳订货次数 = 7200/1200 = 6次
    相关总成本 = √(2×7200×200×2) = √5,760,000 = 2400元
【易错点】
❌ 把固定订货成本/固定储存成本算进相关成本
❌ 陆续供货模型与基本模型混淆
❌ 再订货点与经济订货批量混淆
```

### 重点法条示例模板

**T-15 担保物权（抵押vs质押vs留置）**
```
【法条原文】
- 抵押权：为担保债务的履行，债务人或者第三人不转移财产的占有，将该财产抵押给债权人，债务人不履行到期债务或者发生当事人约定的实现抵押权的情形，债权人有权就该财产优先受偿。
- 质权：为担保债务的履行，债务人或者第三人将其动产出质给债权人占有，债务人不履行到期债务或者发生当事人约定的实现质权的情形，债权人有权就该动产优先受偿。
- 留置权：债务人不履行到期债务，债权人可以留置已经合法占有的债务人的动产，并有权就该动产优先受偿。

【记忆口诀】
「抵押不转移，质押要交付，留置先合法占有」
「不动产只能抵，动产可抵可质，留置只适用于动产」

【对比表格】
| 类型 | 标的 | 是否转移占有 | 设立方式 | 适用情形 |
|------|------|-------------|---------|---------|
| 抵押权 | 不动产、动产 | 不转移 | 不动产登记设立；动产合同生效设立 | 借款、买卖等 |
| 质权 | 动产、权利 | 转移交付 | 动产交付设立；权利登记/交付设立 | 借款等 |
| 留置权 | 动产 | 已合法占有 | 法定，符合条件即产生 | 保管、运输、承揽等 |

【典型案例】
甲向乙借款10万元：
① 以自有房屋抵押 → 办理抵押登记，乙取得抵押权（不转移占有）
② 以自有汽车质押 → 交付汽车给乙，乙取得质权（转移占有）
③ 甲的汽车坏了，送到丙修理厂修理，欠修理费2万 → 丙可以留置该车（已合法占有+同一法律关系）

【易错点】
❌ 不动产可以质押——错，不动产只能抵押
❌ 留置权不需要合同约定——对，是法定担保物权
❌ 抵押权和质权并存时，按设立时间先后顺序受偿
❌ 同一财产既设立抵押权又设立质权的，按登记、交付的时间先后确定清偿顺序
```

**T-23 增值税应纳税额计算**
```
【法条原文】
- 一般计税方法：应纳税额 = 当期销项税额 - 当期进项税额
- 销项税额 = 销售额 × 税率
- 销售额为纳税人发生应税销售行为收取的全部价款和价外费用，但不包括收取的销项税额
- 简易计税方法：应纳税额 = 销售额 × 征收率

【记忆口诀】
「一般计税销减进，简易计税直接乘；价外费用都算入，含税收入要价税分离」

【核心要点】
1. 销售额确定
   - 一般销售：全部价款+价外费用
   - 折扣销售：同一张发票金额栏注明的，按折扣后
   - 以旧换新：按新货物同期售价（金银首饰除外）
   - 还本销售：按货物售价，不得扣减还本支出
   - 以物易物：双方都作购销处理

2. 进项税额抵扣
   - 准予抵扣：增值税专用发票、海关专用缴款书、农产品收购/销售发票
   - 不得抵扣：用于简易计税、免税、集体福利、个人消费；非正常损失；购进的贷款服务、餐饮服务、居民日常服务、娱乐服务

3. 价税分离
   - 含税销售额 ÷ (1 + 税率/征收率) = 不含税销售额

【典型案例】
某商场为增值税一般纳税人，本月零售商品取得含税销售额113万元，本月购进货物取得增值税专用发票注明税额10万元。
解：不含税销售额 = 113 ÷ (1+13%) = 100万元
    销项税额 = 100 × 13% = 13万元
    应纳税额 = 13 - 10 = 3万元

【易错点】
❌ 价外费用不含税——错，价外费用默认含税，要价税分离
❌ 免税项目的进项税可以抵扣——错，要转出
❌ 非正常损失的进项税不需要转出——错，必须转出
```

## 🆚 资源库5：易混对比表

> 收录三科易混考点对比，帮助考生快速区分相似概念

### 会计实务易混考点

| 对比项目 | 概念A | 概念B | 核心区别 |
|---------|-------|-------|---------|
| 账面余额vs账面价值 | 账面余额=账面原价 | 账面价值=账面余额-折旧/摊销-减值准备 | 是否扣减备抵科目 |
| 公允价值变动vs资产减值损失 | 公允价值变动：交易性金融资产等公允价值波动 | 资产减值：可收回金额低于账面价值的减值 | 暂时性vs永久性？减值可转回的有限制 |
| 资本公积vs其他综合收益 | 资本公积：资本溢价/股本溢价+其他资本公积 | 其他综合收益：未计入当期损益的利得损失 | 是否计入当期损益；以后能否转损益不同 |
| 同一控制vs非同一控制合并 | 同一控制：受同一方或相同多方最终控制，交易不公允，按账面价值 | 非同一控制：非关联方之间，交易公允，按公允价值 | 合并对价按账面还是公允；商誉确认不同 |
| 追溯调整法vs未来适用法 | 追溯调整：视同交易事项初次发生时就采用新政策，调整期初留存收益 | 未来适用：新政策仅适用于变更当期及以后期间 | 是否调整以前期间报表 |

### 财务管理易混考点

| 对比项目 | 概念A | 概念B | 核心区别 |
|---------|-------|-------|---------|
| 经营杠杆vs财务杠杆 | 经营杠杆：固定经营成本导致的，DOL=EBIT变动率/销量变动率 | 财务杠杆：固定资本成本导致的，DFL=EPS变动率/EBIT变动率 | 杠杆产生的原因不同；作用不同 |
| 净现值vs内含收益率 | NPV：绝对数，反映投资效益，受折现率影响 | IRR：相对数，反映投资效率，不受折现率影响 | 评价标准不同；互斥项目可能结论冲突 |
| 静态回收期vs动态回收期 | 静态：不考虑时间价值，计算简单 | 动态：考虑时间价值，更准确 | 是否折现；结果不同 |
| 成本中心vs利润中心vs投资中心 | 成本中心：只对成本负责，无收入决策权 | 利润中心：对成本+收入+利润负责 | 投资中心：对成本+收入+利润+投资效果负责 | 控制权范围不同；考核指标不同 |
| 剩余收益vs投资报酬率 | ROI：相对数，便于部门间比较，易导致本位主义 | 剩余收益：绝对数，与企业整体目标一致，不便比较 | 相对vs绝对；本位主义vs整体利益 |

### 经济法易混考点

| 对比项目 | 概念A | 概念B | 核心区别 |
|---------|-------|-------|---------|
| 抵押权vs质权vs留置权 | 抵押：不转移占有，不动产/动产 | 质押：转移占有，动产/权利 | 留置：已合法占有，法定担保 | 占有方式不同；设立方式不同；标的不同 |
| 要约vs要约邀请 | 要约：内容具体确定，希望订立合同，受约束 | 要约邀请：希望他人向自己发出要约，如寄送价目表、拍卖公告 | 是否有明确的订约意思表示；内容是否具体 |
| 无效合同vs可撤销合同vs效力待定合同 | 无效：自始无效，如违法、虚假意思表示 | 可撤销：撤销前有效，撤销后自始无效，如欺诈、胁迫 | 效力待定：效力悬而未决，如限制民事行为能力人 | 效力状态不同；原因不同；后果不同 |
| 有限合伙人vs普通合伙人 | 普通合伙人：无限连带责任，可执行合伙事务，竞业禁止 | 有限合伙人：有限责任，不执行合伙事务，可竞业 | 责任形式不同；事务执行权不同；竞业限制不同 |
| 增值税视同销售vs进项税额转出 | 视同销售：货物转移了所有权或用于外部，要算销项税 | 进项转出：货物用于内部非应税项目或发生非正常损失，进项不能抵 | 一个是销项，一个是进项不能抵；方向不同 |
