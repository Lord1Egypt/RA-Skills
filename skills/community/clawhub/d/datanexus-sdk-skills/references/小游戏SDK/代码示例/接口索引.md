# 微信小游戏 SDK 接口索引

## 初始化

| 接口 | 说明 |
|------|------|
| `new SDK(config)` | 构造并初始化 SDK 实例 |

## 用户标识

| 接口 | 说明 |
|------|------|
| `sdk.setOpenId(openid)` | 设置微信 OpenId（优先使用） |
| `sdk.setUnionId(unionid)` | 设置微信 UnionId |

## 通用上报

| 接口 | 说明 |
|------|------|
| `sdk.track(action_type, action_param?)` | 通用行为上报 |

## 专用上报

onRegister / onCreateRole / onTutorialFinish / onPurchase / onAppStart / onAppQuit / onEnterForeground / onEnterBackground / onAddToWishlist

## 全局配置

| 接口 | 说明 |
|------|------|
| `SDK.setDebug(true/false)` | 调试模式 |
| `SDK.setRequestConcurrency(n)` | 并发数 |

## 自动采集事件

TICKET / START_APP / ENTER_FOREGROUND / ENTER_BACKGROUND / LOGIN / SHARE / START_PAY / FINISH_PAY / ADD_TO_WISHLIST / CREATE_GAME_ROOM
