# 跨境电商产品合规速查

## Purpose

当用户询问跨境电商产品合规问题时，查询内置 8 品类 × 8 市场合规数据库，
输出结构化全套合规要求，并生成交互式 HTML 可视化合规报告。

## 适用品类与市场

**8 大品类:**
1. 电子产品 (Electronics & Electrical)
2. 玩具 (Toys)
3. 化妆品 (Cosmetics & Personal Care)
4. 食品 (Food & Beverages)
5. 纺织品/服装 (Textiles & Apparel)
6. 医疗器械 (Medical Devices)
7. 家居用品 (Home Goods & Furniture)
8. 儿童用品 (Children's Products, 不含玩具)

**8 大市场:**
1. 美国 (USA)
2. 欧盟 (EU)
3. 英国 (UK)
4. 日本 (Japan)
5. 澳大利亚 (Australia)
6. 加拿大 (Canada)
7. 韩国 (South Korea)
8. 中东 GCC (沙特/阿联酋为核心)

## Workflow

### Step 1: 解析用户输入

从用户消息中提取:
- `category` — 产品品类，映射到上述 8 大品类之一
- `market` — 目的地市场，映射到上述 8 大市场之一 (或 GCC 中的具体国家)
- `platform` (可选) — 平台名称 (如亚马逊/速卖通/Temu/Coupang)，用于额外要求标注

若品类或市场不明确，主动询问确认。若用户给出具体产品名称 (如 "蓝牙耳机")，自动映射到最接近的品类 (如 电子产品)。

### Step 2: 查询合规数据库

读取 `references/compliance-database.md` 中对应市场的对应品类章节。

查询逻辑:
1. 先定位市场 (如 `## 美国 (USA)`)
2. 再定位品类 (如 `### 1. 电子产品`)
3. 提取该品类所有维度的合规数据

**对于加拿大和英国**: 基础合规与美国 (加拿大) 和欧盟 (英国) 高度相似，查询时:
- 加拿大: 按美国数据输出，再读取 "加拿大 (Canada)" 概要章节，标注差异点
- 英国: 按欧盟数据输出，再读取 "英国 (UK)" 概要章节，标注 UKCA/UK REACH/UK 责任人等差异
- 北爱尔兰提及: 注意 Windsor Framework 特殊规则

### Step 3: 组织输出维度

无论品类和市场，按以下标准维度组织输出:

| 维度 | 内容 |
|------|------|
| **强制认证** | 必须取得的认证标志 (FCC/CE/UKCA/KC/PSE/RCM 等)，认证路径 (自我声明 vs 第三方检测) |
| **标签要求** | 标签上必须包含的信息 (成分/警告/制造商/年龄分级/原产国)、标签语言要求、标志 (CE/UKCA 等) 最小尺寸 |
| **包装要求** | 包装材料合规 (如食品接触材料)、安全警告印刷、防篡改/防窒息、运输测试标准 (ISTA) |
| **测试标准** | 必须通过的产品测试标准编号 (ASTM/EN/IEC/JIS/AS/KS 等)，关键测试项目 |
| **限制物质** | 禁止或限制使用的化学物质 (铅/邻苯/RoHS/REACH SVHC/甲醛等)，限量值 |
| **进口文件** | 清关必备文件 (商业发票/装箱单/提单/CoC/测试报告/卫生证明等) |
| **平台额外要求** | 亚马逊/Coupang 等电商平台的特殊上传要求 (CPC/DoC/测试报告/合规图片) |
| **特殊要求** | 电压/插头匹配、无线设备额外认证、清真认证 (GCC)、双语标签 (加拿大)、婴幼儿产品特殊标准等 |

### Step 4: 生成 HTML 可视化报告

生成一份专业的交互式 HTML 合规报告，结构如下:

1. **报告头部** — 产品品类 + 目的地市场 + 生成时间 + 风险等级标识
2. **合规仪表盘** — 8 维度卡片展示，标注强制/推荐的标签，完成状态指示灯
3. **关键认证概览** — 核心认证标志列表 (大图标+认证名称+认证路径)
4. **标签要求详情** — 标签语言、必须包含的信息清单、标签模板示意
5. **测试标准表格** — 标准编号+标准名称+适用范围+是否强制
6. **限制物质清单** — 物质名称 + 限量值 + 法规依据 + 风险等级
7. **进口文件清单** — 文件名称 + 必要性 (强制/推荐/可选) + 备注
8. **亚马逊合规要求** (如适用) — 平台特定上传要求
9. **行动检查清单** — 按优先级 (P0/P1/P2) 排列的实操 checklist
10. **风险提示** — 常见不合规点 + 规避建议

HTML 报告要求:
- 移动端响应式设计
- 支持打印 (CSS @media print)
- 色彩方案: 专业商务风 (深蓝 #1a365d 为主色，绿色 #38a169 代表合规，橙色 #ed8936 代表注意，红色 #e53e3e 代表强制)
- 中英双语关键术语 (如 "FCC (联邦通信委员会)")
- 使用 Tab 切换或折叠面板减少滚动，适合手机端查看

### Step 5: 输出摘要

生成 HTML 报告后，向用户输出一份简洁的文本摘要，包含:
- 核心 3-5 项强制认证
- 标签语言要求
- 最容易忽视的 2-3 个坑点
- 直接可操作的下一步建议

## 特殊场景处理

### 多品类查询
若用户同时询问多个品类 (如 "蓝牙耳机和儿童玩具出口美国"):
- 依次查询各品类，在一个 HTML 报告中分 Tab 展示

### 多市场查询
若用户询问同一产品出口多国 (如 "电子产品出口美国和欧盟"):
- 依次查询各市场，在一个 HTML 报告中分市场对比展示

### 平台特定查询
若用户提及具体平台:
- 亚马逊: 重点标注 "亚马逊要求" 维度
- 速卖通 (AliExpress): 标注 CE/FCC 等目的国认证 (速卖通无独立认证体系)
- Temu: 标注目的国合规+平台禁售品类
- Coupang (韩国): 标注 KC 认证+韩文标签+韩国本地责任人要求

### 模糊产品映射
若用户提供的产品名称无法精确匹配:
- 智能手表 → 电子产品 (如强调健康功能 → 可能涉及医疗器械)
- 电动牙刷 → 电子产品 (含电池 → 注意电池运输 UN38.3)
- 太阳镜 → 纺织品/服装大类中的配件 (实际为 PPE — 个人防护装备，需 CE/ANSI)
- 蜡烛 → 家居用品 (注意易燃品安全标准)
- 宠物玩具 → 玩具大类 (注意 - 宠物玩具在美国和欧盟有不完全同儿童玩具的合规要求)
- 圣诞灯饰 → 电子产品 + 节日装饰品双重管理

## 数据来源说明

合规数据库基于以下来源整理 (截至 2026 年 6 月):
- 各国政府法规官网 (FDA, CPSC, EU Commission, UK Gov, METI, ACCC, Health Canada, KATS, SASO, ESMA 等)
- 国际标准组织 (ISO, IEC, ASTM, EN, JIS, AS/NZS)
- 电商平台合规政策 (Amazon Seller Central, Coupang Wing, Temu 商家中心等)
- 第三方检测认证机构指南 (SGS, Intertek, Bureau Veritas, TÜV)

> **免责声明**: 法规持续更新，本数据库提供参考方向，正式出口前建议通过官方渠道或第三方认证机构确认最新要求。
