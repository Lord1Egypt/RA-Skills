# 波士顿矩阵分析模板

## 概述

波士顿矩阵（BCG Matrix）是一种产品组合分析工具，用于评估企业产品线的竞争地位和发展策略。

## 核心概念

### 两个维度
- **市场增长率**：衡量产品所在市场的成长性
- **相对市场份额**：企业产品相对于最大竞争对手的份额

### 判断标准
| 维度 | 高 | 低 |
|------|----|----|
| 市场增长率 | >10% | <10% |
| 相对市场份额 | >1（大于竞争对手） | <1（小于竞争对手） |

## 四象限定义

### 明星业务（Star）
```
特征：高增长 + 高份额
定义：处于快速增长市场中并占有主导份额的产品
```

**战略建议**：
- 加大资源投入
- 保持竞争优势
- 追求市场份额增长
- 为现金牛阶段储备能力

**资源配置**：优先投入

### 现金牛业务（Cash Cow）
```
特征：低增长 + 高份额
定义：成熟市场中占有主导份额的产品
```

**战略建议**：
- 维持现状
- 控制成本
- 贡献稳定现金流
- 支持明星和问题业务

**资源配置**：维持投入

### 问题业务（Question Mark）
```
特征：高增长 + 低份额
定义：快速增长市场中份额较低的产品
```

**战略建议**：
- 选择性投资
- 分析增长潜力
- 决定是否转化为明星
- 或果断退出

**资源配置**：选择性投入

### 瘦狗业务（Dog）
```
特征：低增长 + 低份额
定义：成熟但竞争力弱的产品
```

**战略建议**：
- 收缩或退出
- 降低运营成本
- 释放资源
- 或寻找细分市场机会

**资源配置**：最小投入或退出

## 分析流程

### 第一步：识别产品/业务线
从商业计划书中提取所有产品或业务线：
1. 核心产品
2. 衍生产品
3. 服务业务
4. 解决方案

### 第二步：评估市场增长率
数据来源：
- 商业计划书中的市场数据
- 行业研究报告
- 公开市场数据

### 第三步：评估相对市场份额
计算方法：
```
相对市场份额 = 企业该产品销售额 / 最大竞争对手该产品销售额
```

### 第四步：绘制矩阵定位

```
                             ↑
                             │
    ┌────────────────────────┼────────────────────────┐
    │                        │                        │
    │     问题业务           │        明星业务        │
    │   （高增长             │       （高增长         │
    │    低份额）            │        高份额）         │
    │   资源配置：           │       资源配置：        │
    │   选择性投入           │       优先投入          │
    │                        │                        │
    ├────────────────────────┼────────────────────────┤
    │                        │                        │
    │     瘦狗业务           │        现金牛业务       │
    │   （低增长             │       （低增长          │
    │    低份额）            │        高份额）         │
    │   资源配置：           │       资源配置：        │
    │   最小投入或退出       │       维持投入          │
    │                        │                        │
    └────────────────────────┼────────────────────────┘
                             │
                             ↓
                          低增长率
          ← 低市场份额              高市场份额 →
```

## 输出格式

### 产品线汇总表
| 产品/业务 | 市场增长率 | 相对市场份额 | 象限定位 | 战略建议 | 资源配置 |
|-----------|------------|--------------|----------|----------|----------|
| | | | | | |

### 详细分析模板
```
## [产品/业务名称]

### 定位信息
- 市场增长率：[数值]%（[判断：高/低]）
- 相对市场份额：[数值]（[判断：高/低]）
- 象限：[明星/现金牛/问题/瘦狗]

### 当前状态
- 市场份额：[具体数据或描述]
- 增长趋势：[描述]
- 竞争地位：[描述]

### 战略建议
- 短期策略：[1年内建议]
- 中期策略：[1-3年建议]
- 资源配置：[优先级]

### 风险提示
- [如有问题或瘦狗业务，说明风险]
```

## 商业计划书提取指南

### 提取内容
1. **产品线列表**
   - 主要产品/服务
   - 产品定位
   - 收入占比

2. **市场数据**
   - 目标市场规模
   - 年增长率
   - 市场份额

3. **竞争信息**
   - 主要竞争对手
   - 竞争格局
   - 差异化优势

### 推断规则
如果商业计划书中没有明确的产品线划分，根据业务描述进行推断：
- 按业务类型分类
- 按收入来源分类
- 按目标客户分类

**标注方式**：`【推断】`

## 注意事项

1. **数据尽量量化**：如无法获取具体数值，使用描述性判断
2. **注明数据来源**：引用商业计划书中的原文
3. **标注推断项**：无法确定的用【推断】标注
4. **动态视角**：考虑产品线的发展阶段
5. **不编造**：无法提取的信息如实标注
