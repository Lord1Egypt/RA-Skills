# 写作任务清单与依赖拓扑

网络小说创作按价值链组织的任务类型清单，附任务间依赖关系和UTOS元操作映射提示。

**域间逻辑流（5层归属）**：
- 📥 输入层：素材与世界观(N1)
- 🏗️ 设计层：架构与大纲(N2)
- ✍️ 执行层：角色(N3) → 情节(N4) → 场景(N5)
- 🔍 质控层：叙事节奏(N6) → 修订与润色(N7)
- 💰 商业层：商业运营(N8)

## 依赖拓扑摘要

以下为任务间的主要依赖链路，UTOS执行轴可据此自动编排管线：

**题材-世界观链**: N1-01(题材定位) → N1-02(世界观设定) → N1-03(力量体系)
**世界观-架构链**: N1-02 → N2-01(核心宣言) → N2-02(三幕) → N2-03(大纲) → N2-05(章纲) / N1-03 → N2-03
**架构-分卷链**: N2-03 → N2-09(分卷架构) → N8-07(日更计划)
**架构-角色-情节链**: N2-02 → N3-01 → N3-03 → N4-01 → N4-03 → N4-05
**角色-关系-冲突链**: N3-01 → N3-02 → N3-05 → N3-07 → N4-03
**情节-场景-正文链**: N4-01 → N2-05 → N5-01 → N5-02/N5-04/N5-07/N5-08/N5-09
**伏笔-回收闭环**: N4-01 → N4-04(埋设) → N4-04(回收追踪) → N7-01(验证)
**节奏-悬念链**: N6-01 → N6-03 → N6-05 / N4-06 → N6-04 → N6-05
**节奏-日更闭环**: N6-01 → N2-09 → N8-07(日更计划) → N8-06(数据复盘) → N8-07
**正文-修订闭环**: N5-01~N5-09 → N7-08(正文修订) → N7-01~N7-06 / N1~N4 → N7-07(设定修订) → N7-01
**数据-运营闭环**: N8-06 → N8-03 → N8-05 / N8-02 → N8-01 → N8-04
**素材-世界观-全文链**: N1-05 → N1-08 → N1-02 → [N2→N3→N4→N5] → N7 → N8
**黄金三章加速链**: N2-01 → N2-02 → N4-01 → N5-01 → N5-02 → N6-05（前三章快速产出管线）

更多组合由UTOS根据具体任务动态推导。
