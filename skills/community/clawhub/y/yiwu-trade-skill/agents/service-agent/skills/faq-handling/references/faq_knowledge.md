# FAQ知识库 (FAQ Knowledge Base)

## 一、产品类FAQ

### Q1: 产品起订量(MOQ)是多少?
```
A: 我们的MOQ根据产品不同而有所区别:
- 标准产品: 50-100件
- 定制产品: 200-500件
- 大型产品: 20-50件

部分产品支持混批，详情请咨询客服。
```

### Q2: 可以提供样品吗?
```
A: 是的，我们提供样品服务:
- 样品费用: 通常是批量价格的1.5倍
- 样品运费: 到付
- 样品时间: 3-5个工作日
- 样品抵扣: 大货订单可抵扣货款

如果您在确认样品后下大货，样品费可以抵扣。
```

### Q3: 产品是什么材质/规格?
```
A: 我们的产品材质和规格多样，常见包括:
- 塑料类: ABS, PP, PVC, 硅胶
- 金属类: 铁, 不锈钢, 铝合金, 锌合金
- 布料类: 棉, 涤纶, 尼龙, 帆布
- 木材类: 实木, 密度板, 竹木

具体产品请查看详情页或咨询客服获取规格表。
```

### Q4: 可以定制Logo/包装吗?
```
A: 可以，我们提供以下定制服务:
- Logo定制: 丝印/烫印/贴标
- 包装定制: 公版盒/彩盒/礼盒
- 说明书定制: 多语言

定制MOQ通常为500-1000件，具体根据产品而定。
```

### Q5: 产品有哪些认证?
```
A: 我们产品获得的常见认证包括:
- CE: 欧盟安全认证 (电子/玩具类)
- FCC: 美国联邦通信认证 (电子产品)
- ROHS: 欧盟环保认证
- REACH: 欧盟化学品注册
- ISO9001: 质量管理体系认证
- BSCI: 社会责任审核

特定产品请咨询是否有对应认证。
```

## 二、订单类FAQ

### Q6: 如何下单?
```
A: 下单流程如下:
1. 选择商品，添加到购物车
2. 确认商品信息和数量
3. 填写收货地址
4. 选择运输方式
5. 选择付款方式
6. 提交订单并支付

也可以通过WhatsApp/Email联系我们帮您下单。
```

### Q7: 可以修改订单吗?
```
A: 订单修改规则:
- 未付款: 可随意修改
- 已付款未生产: 可修改数量/地址
- 生产中: 仅可修改地址
- 已发货: 无法修改，请联系物流

请尽快联系客服进行修改。
```

### Q8: 可以取消订单吗?
```
A: 取消政策:
- 未付款: 可直接取消
- 已付款未发货: 扣除5%手续费后退款
- 生产中: 扣除已生产部分费用
- 已发货: 无法取消

请确认后谨慎下单。
```

### Q9: 多个订单可以合并发货吗?
```
A: 可以合并发货，但需满足:
- 订单尚未全部发货
- 收货地址相同
- 可联系客服申请合并

合并后根据总重量重新计算运费。
```

## 三、物流类FAQ

### Q10: 发货时间需要多久?
```
A: 生产时间:
- 现货产品: 1-3天
- 标准产品: 7-10天
- 定制产品: 15-20天

运输时间(参考):
- 空运: 5-7天
- 海运(集装箱): 15-30天
- 快递(DHL/FedEx): 3-5天

具体时间以下单时确认为准。
```

### Q11: 运费如何计算?
```
A: 运费计算方式:
1. 海运: 按体积(CBM)或重量计算
2. 空运: 按实际重量计算
3. 快递: 按件数或重量

我们会根据您的订单提供多个方案供选择。

预估运费 = 基础运费 + 附加费 + 保险费(可选)
```

### Q12: 可以追踪物流吗?
```
A: 是的，发货后我们会提供:
- 运单号
- 物流公司名称
- 追踪链接

您可以通过:
- 我们提供的链接直接查询
- 物流公司官网查询
- 联系客服帮您查询

对于DDP订单，我们提供门到门追踪服务。
```

### Q13: 清关需要多久?
```
A: 清关时间因国家而异:
- 东南亚: 2-3天
- 中东: 3-5天
- 欧洲: 3-7天
- 非洲: 5-10天
- 南美: 5-15天

我们会提供清关所需文件，买方配合清关即可。
```

## 四、支付类FAQ

### Q14: 支持哪些付款方式?
```
A: 我们支持以下付款方式:
1. T/T (电汇): 30%定金，余款发货前付清
2. PayPal: 手续费3.5%由买方承担
3. Western Union: 小额付款
4. 信用证(L/C): 大额订单，需审核

我们强烈推荐T/T，安全快捷。
```

### Q15: 可以月结吗?
```
A: 月结服务仅对符合条件的客户开放:
- 月均订单额 ≥ $5,000
- 合作满3个月以上
- 信用审核通过

如需申请，请联系客服提交公司资料审核。
```

### Q16: 如何开具发票?
```
A: 发票类型:
- 增值税专用发票(17%税率)
- 增值税普通发票
- 形式发票(Proforma Invoice)
- 商业发票(Commercial Invoice)

请在付款后告知发票类型和抬头，我们将随货寄出。
```
