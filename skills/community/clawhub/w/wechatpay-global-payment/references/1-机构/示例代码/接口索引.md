# 机构模式 接口索引

> 本文档是境外微信支付（跨境）**接口路径 + 官方文档**的完整索引，同时收录示例代码的获取方式。
>
> ‼️ **接入前必读**：
> 1. **请求域名**：`https://apihk.mch.weixin.qq.com`
> 2. **路径前缀**：所有 API 以 `/v3/global/` 开头（**非**境内的 `/v3/pay/partner/`）
> 3. **请求体必带字段**：`sp_mchid` + `sp_appid` + `sub_mchid`（`sub_appid` 在需要子商户 OpenID 时必填）
> 4. **签名主体**：机构自身私钥（`sp_mchid` 对应），详见 [📄 签名与验签规则](../接入指南/签名与验签规则.md)

---

## 如何获取示例代码

> 本 Skill **不内置代码**，请按以下流程引导用户获取官方代码：

```
第一步：判断产品类型
  → 确认用户使用的支付产品（付款码 / Native / JSAPI / APP / 小程序 / H5 / 委托代扣）
    或扩展能力（子商户进件 / 跨境分账 / 海关报关）

第二步：到「官方开发指引快速入口」检索开发指引
  → 在下方「官方开发指引快速入口」找到对应产品或能力
  → 给用户该产品的开发指引链接
  → 开发指引页面包含完整的接入说明与页面底部的 Java / PHP / Go 代码示例

第三步（按需）：查询具体 API 字段说明
  → 如用户需要某个接口的详细字段说明（如请求参数、响应字段、错误码）
  → 到下方「业务接口索引」找到对应接口，给出具体接口文档链接
```
