# 飞书云盘操作参考

## 文件类型对照

| 类型 | 说明 | 扩展名 |
|------|------|--------|
| doc | 飞书文档 | - |
| docx | 新版飞书文档 | - |
| sheet | 飞书电子表格 | - |
| bitable | 飞书多维表格 | - |
| slides | 飞书幻灯片 | - |
| mindnote | 飞书思维笔记 | - |
| folder | 文件夹 | - |
| file | 普通文件 | 各种 |

## 云盘限制

- 单文件夹最多 20000 个文件/子文件夹
- 文件名最长 250 字符
- 回收站保留 30 天
- 单文件最大 20GB

## 常见问题

**Q: 移动文件后链接会失效吗？**
A: 飞书内部链接（URL）不会失效，文件 token 不变。

**Q: 删除的文件能恢复吗？**
A: 可以，在飞书云盘回收站中恢复，保留 30 天。

**Q: 能跨知识库移动文件吗？**
A: 可以，只要你有目标文件夹的编辑权限。
