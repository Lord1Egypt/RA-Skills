# 小小 Platform Content Optimization

## 核心原則

小小在進行平台內容優化時必須遵守以下原則：

- **核心訊息一致**：不同平台呈現的形式可以不同，但核心訊息與品牌調性應保持一致。
- **平台特性優先**：尊重每個平台的用戶習慣與內容形式（例如 IG 重視視覺與簡短、電郵重視深度與結構）。
- **受眾適配**：根據目標受眾在不同平台上的行為與偏好調整內容。
- **細節與吸引力**：注重細節優化，讓內容在各平台都能保持吸引力和專業感。
- **效率與一致性平衡**：在保持內容一致性的同時，提高多平台產出的效率。
- **測試與調整**：不同平台的效果會有差異，要願意根據實際表現進行調整。

## 主要能力

### 1. 平台特性分析與適配
- 了解不同平台的用戶特徵、內容偏好與演算法特性
- 根據平台特性調整內容長度、語氣、格式與呈現方式
- 設計適合該平台的內容形式（例如 IG 輪播圖、限時動態、文章、短影片腳本等）

**適用場景**：需要把同一訊息發佈到多個平台時。

### 2. 語氣與風格調整
- 根據平台與受眾調整語氣（例如 IG 較活潑、LinkedIn 較專業、電郵較正式）
- 保持品牌調性一致的同時，做出平台適配的調整
- 確保語氣能有效觸達目標受眾

**適用場景**：同一內容需要在不同平台使用不同語氣時。

### 3. 內容格式與結構優化
- 根據平台特性重新組織內容結構（例如 IG 重點放前面、電郵用清晰段落）
- 優化標題、開頭與結尾在不同平台的效果
- 加入適合平台的元素（例如 hashtag、emoji、視覺建議、連結位置等）

**適用場景**：需要為不同平台重新設計內容呈現形式時。

### 4. 多平台內容策略設計
- 規劃同一核心訊息在不同平台的呈現策略
- 設計內容的變形版本（例如一篇長文 → IG 系列貼文 + 電郵版本）
- 確保各平台內容之間的連貫性與互補性

**適用場景**：需要制定多平台內容發布策略時。

### 5. 效果評估與持續優化
- 建議追蹤不同平台內容的表現指標
- 根據數據反饋調整內容策略與呈現方式
- 建立各平台的內容優化經驗與最佳實踐

**適用場景**：內容發布後需要檢討與優化時。

## 標準平台內容優化流程（小小推薦）

1. **明確核心訊息** → 確定不管在哪個平台都要傳達的核心內容
2. **分析目標平台** → 了解各平台的用戶習慣與內容特性
3. **設計平台版本** → 為每個平台設計適合的內容形式與語氣
4. **優化細節** → 調整標題、開頭、格式、視覺建議等
5. **發布與追蹤** → 上線後觀察表現並記錄數據
6. **檢討與調整** → 根據表現持續優化各平台策略

## 注意事項與最佳實踐

- 不要為了「一樣」而強求所有平台內容完全相同，適度差異化是正常的。
- 重視開頭與結尾在不同平台的表現，這兩部分對點擊率與轉換影響很大。
- 善用 Target Audience Analysis 的結果，確保各平台內容都貼近目標受眾。
- 可與 Copywriting 和 Storytelling 搭配，讓各平台版本都有吸引力和說服力。
- 建議建立各平台的內容模板與 checklist，提高產出效率。
- 重要內容建議先在主要平台測試，再擴散到其他平台。

## 常見應用場景（通用版）

- 同一活動或產品在多平台的內容發布策略
- 將長篇文章轉化為多個社媒貼文版本
- 設計跨平台內容系列（例如教育系列、產品推廣系列）
- 優化現有內容在不同平台的表現
- 建立多平台內容的標準流程與模板
