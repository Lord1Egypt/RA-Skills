# ✍️ ai-content-polish

> 中文 AI 内容去痕与润色工具 — 让 AI 写的东西看起来不像 AI 写的。

## 它能做什么？

1. **AI 痕迹检测** — 扫描中文文本中的 8 大类 AI 写作特征
2. **人性化改写** — 提供具体的改写建议和改写版本
3. **人类感评分** — 0-100 分，评估文本的自然程度

## 为什么需要它？

AI 生成的内容越来越泛滥，读者一眼就能认出 "AI味"。无论你是：
- 内容创作者，不想让读者觉得你在用 AI
- 市场文案，需要文字有人味儿
- 学生/研究者，需要论文读起来自然
- 产品经理，文档不能像机器写的

这个 Skill 帮你找出 AI 痕迹，并给出具体的改写建议。

直接对话即可：

```

```

```
帮我把这篇文章改得更自然一些：[你的文本]
```

```
给这段内容打个分：[你的文本]
```

## 检测的 8 大类 AI 痕迹

| # | 类别 | 说明 |
|---|------|------|
| 1 | 空洞过渡语 | "值得注意的是"、"总而言之" 等 |
| 2 | 排比与三段式 | "不仅A，而且B，更是C" |
| 3 | 形容词堆砌 | "强大的、全面的、高效的" |
| 4 | 机械列举 | 每项结构完全相同的列表 |
| 5 | 过度总结 | 每段末尾都有总结句 |
| 6 | 口癖重复 | 同一篇文章反复出现相同的词 |
| 7 | 缺乏个人声音 | 没有"我"、没有口语、没有情感 |
| 8 | 过度工整 | 段落长度均匀、句式统一、标点规范 |

## 评分等级

- 🟢 90-100：非常自然
- 🟢 70-89：较自然
- 🟡 50-69：有明显 AI 痕迹
- 🔴 30-49：AI 味很重
- 🔴 0-29：典型 AI 输出

## 三种模式

| 模式 | 说明 | 使用场景 |
|------|------|---------|
| 检测报告 (report) | 输出详细检测报告 + 改写建议 | 想知道具体哪里有问题 |
| 直接改写 (rewrite) | 输出润色后的文本 | 直接要改好的版本 |
| 快速评分 (score) | 只输出评分和一句话总结 | 快速判断 |
