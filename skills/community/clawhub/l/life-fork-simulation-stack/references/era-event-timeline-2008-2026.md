# 2008-2026 事件知识库

用途：为人生岔路模拟提供时代、行业、城市、资产、技术、家庭和全球背景。事件只用于过去复盘和情景分析，避免宏观预测、焦虑制造和投资建议。

## 使用规则

- 先按用户的年份、城市、行业、资产状态、海外经历、家庭责任和技术路径检索事件。
- 每份报告只选 3-8 个高相关事件。
- 每个事件必须写清：时间范围、影响人群、影响变量、适用岔路、使用边界。
- 已发生事件可以用于过去复盘；未来变化只能作为情景变量。
- 必须区分个人选择、时代环境和随机运气。
- 涉及资产、健康、法律、心理议题时读取 `safety-boundaries.md`。

## 事件卡片格式

每次调用事件库时，按下面结构引用：

## 阶段索引

### 2008-2010：金融危机、刺激政策、地产上行起点、移动互联网前夜

#### 全球金融危机与中国刺激政策

- 时间范围：2008-2009
- 影响人群：毕业生、制造业、外贸、金融、地产和基建相关从业者。
- 影响变量：就业稳定性、行业景气、城市迁移、家庭安全感、现金流缓冲。
- 适用岔路：毕业后留大城市 / 回家；进入外贸金融 / 进入内需基建；买房 / 观望。
- 影响机制：外贸和金融路径承压，基建、地产和内需相关机会增加，家庭对稳定的需求上升。
- 使用边界：只分析当年就业和城市承受力，避免推导具体资产收益。
- 待确认问题：当年你的行业更靠外贸、金融、地产、基建，还是消费？

#### 城市化与房地产上行

- 时间范围：2008-2010
- 影响人群：一线城市白领、返乡购房者、婚育人群、家庭首付支持者。
- 影响变量：房价预期、城市绑定、家庭关系、婚育节奏、现金流压力。
- 适用岔路：买房 / 租房；留一线 / 回家；先结婚 / 延后婚育。
- 影响机制：房子逐渐从居住选择变成家庭安全感、身份和城市绑定变量。
- 使用边界：只分析现金流和路径依赖，避免推荐买卖。
- 待确认问题：当年房子对你的婚恋、父母期待或留城选择影响多大？

#### 3G 与智能手机前夜

- 时间范围：2009-2010
- 影响人群：通信、互联网、内容、产品、营销和媒体岗位。
- 影响变量：技能迁移、行业入口、城市机会、作品形态。
- 适用岔路：传统媒体 / 移动互联网；线下营销 / 线上增长；硬件行业 / 软件平台。
- 影响机制：移动入口开始形成，内容、产品和营销岗位出现新窗口。
- 使用边界：只分析机会窗口和技能迁移，避免把早期接触写成必然优势。
- 待确认问题：你当年是否接触 App、移动内容、手机互联网或线上营销？

### 2011-2013：移动互联网爆发、社交平台迁移、一线城市机会密度上升

#### 微信和移动社交兴起

- 时间范围：2011-2013
- 影响人群：内容、营销、社交、电商、媒体、公关从业者。
- 影响变量：内容分发、私域关系、职业能见度、组织资源。
- 适用岔路：门户 / 移动社交；传统媒体 / 新媒体；员工路径 / 个人内容资产。
- 影响机制：社交关系和内容分发从 PC 迁移到移动端，个人表达和机构传播边界变化。
- 使用边界：避免把平台红利写成个人能力；必须检查用户是否真的参与相关项目。
- 待确认问题：你当年是否从线下、门户、传统媒体迁到移动社交？

#### 移动 App 创业潮

- 时间范围：2011-2013
- 影响人群：产品、运营、设计、程序员、市场、公关、创业者。
- 影响变量：城市机会、平台履历、融资环境、薪资弹性、试错成本。
- 适用岔路：北京 / 深圳 / 上海 / 杭州；大厂 / 创业公司；稳定岗位 / 高波动项目。
- 影响机制：一线城市创业机会密度提高，平台和融资环境开始影响城市选择。
- 使用边界：同时分析项目失败率和生活成本，避免只写机会。
- 待确认问题：目标城市给你的主要变量是平台、融资、圈层，还是岗位质量？

#### 公众号和内容行业变化

- 时间范围：2012-2013
- 影响人群：媒体人、品牌营销、公关、内容创业者、知识型工作者。
- 影响变量：表达资产、个人品牌、组织能见度、平台依赖。
- 适用岔路：继续做机构内容 / 做个人内容资产；传统传播 / 社交传播。
- 影响机制：内容表达和职业影响力开始平台化，个人作品对职业机会影响增强。
- 使用边界：只分析可展示作品和平台窗口，避免神化早期入场。
- 待确认问题：你当年是否抓住公众号、微博或移动内容红利？

### 2014-2016：资本热、平台补贴、房价上行、留学路径变化

#### O2O 和资本热

- 时间范围：2014-2015
- 影响人群：互联网从业者、创业者、运营、市场、地推、产品经理。
- 影响变量：薪资弹性、项目存活率、职业履历、现金流波动。
- 适用岔路：创业公司 / 大公司；高薪机会 / 稳定路径；转行 / 留原行业。
- 影响机制：岗位扩张快，机会看起来更弹性，同时项目死亡率高。
- 使用边界：必须同时写机会和失败概率，避免把资本热当成长期价值。
- 待确认问题：当年机会是否来自补贴、融资、快速扩张或平台流量？

#### 共享经济和平台补贴

- 时间范围：2014-2016
- 影响人群：出行、本地生活、外卖、城市服务、运营和增长岗位。
- 影响变量：城市生活方式、岗位结构、平台依赖、收入波动。
- 适用岔路：平台岗位 / 传统行业；一线城市 / 新一线城市。
- 影响机制：平台型岗位增多，城市生活和就业结构变化。
- 使用边界：避免把补贴期数据当成长期收入能力。
- 待确认问题：你的路径是否依赖补贴、流量增长或城市服务扩张？

#### 一二线房价快速上行

- 时间范围：2015-2016
- 影响人群：城市白领、婚育家庭、留城者、回乡者、家庭首付支持者。
- 影响变量：现金流、负债、城市绑定、婚育节奏、家庭关系。
- 适用岔路：买房 / 没买房；一线 / 新一线；结婚 / 延后；留城 / 回家。
- 影响机制：买房与否显著影响长期现金流、家庭安全感和城市自由度。
- 使用边界：只做路径依赖和现金流分析，禁止投资建议。
- 待确认问题：当年是否有买房窗口？月供、首付和家庭期待分别多重？

#### 留学大众化和海归标签变化

- 时间范围：2011-2016
- 影响人群：海外硕士、本科留学生、计划回国进入外企、互联网、品牌或金融岗位的人。
- 影响变量：海外经历、语言环境、家庭预算、实习经历、回国时点、职业身份。
- 适用岔路：留海外 / 回国；继续读书 / 求职；国际化身份 / 中文职场安全感。
- 影响机制：海外经历从稀缺标签逐渐变成需要被作品、实习、语言和行业入口承托的履历材料。回国发展的人，容易在多年后复盘自己有没有把海外经历转成跨文化协作、英文表达或出海判断。
- 使用边界：避免把海归身份写成优势或劣势；必须结合学校、专业、签证、实习、语言和家庭现金流。
- 待确认问题：你当年留学经历给你带来的核心资产是语言、身份、视野、职业入口，还是一段仍未被使用的经验？

#### Brexit 与英国留学 / 就业环境变化

- 时间范围：2016-2020
- 影响人群：英国留学生、海外求职者、计划留英或回国人群。
- 影响变量：身份、签证、就业市场、文化归属、父母距离、跨境成本。
- 适用岔路：留英 / 回国；继续读书 / 求职；海外身份 / 国内机会。
- 影响机制：签证、身份和就业环境变化，留学后去留需要重新评估。
- 使用边界：避免替用户判断移民、签证和法律路径；需要用户补充当时身份条件。
- 待确认问题：当时留下来的现实门槛是签证、钱、孤独、工作，还是家庭距离？

#### 短视频和直播前夜

- 影响机制：流量入口变化，个人表达和品牌增长方式重构。
- 使用边界：必须检查用户是否具备持续作品能力，避免只谈平台红利。
- 待确认问题：你当年是否从图文、公众号、微博转向短视频或直播？

#### 互联网平台成熟

- 时间范围：2017-2019
- 影响人群：大厂、平台型公司、创业者、增长岗位。
- 影响变量：组织能见度、岗位分工、平台履历、健康消耗。

- 影响机制：流量红利收窄，岗位更专业，平台履历价值和组织压力同步上升。
- 使用边界：同时分析履历价值和组织消耗。
- 待确认问题：你当年在扩张型平台，还是成熟型组织？

#### 地产调控和城市分化

- 时间范围：2017-2019
- 影响人群：买房者、租房者、换城者、新一线迁移者。
- 影响变量：城市资产、职业机会、租售压力、迁移自由度。
- 适用岔路：留一线 / 去新一线；买房 / 继续租；职业机会 / 居住稳定。
- 影响机制：不同城市房价和机会分化，城市选择同时涉及资产和职业。
- 使用边界：不做城市资产排序，只评估路径承受力。
- 待确认问题：目标城市的房价和职业机会是否方向一致？

#### 移动支付和新零售成熟

- 时间范围：2017-2019
- 影响人群：零售、电商、品牌、市场、用户运营、门店管理、本地生活岗位。
- 影响变量：用户数据、门店效率、履约体验、增长打法、线上线下协作。
- 适用岔路：传统零售 / 平台零售；品牌市场 / 用户运营；线下经验 / 数字化岗位。
- 影响机制：支付、会员、门店和线上流量连接后，市场岗位开始更依赖数据、转化和履约体验。只会讲品牌故事的人，可能感到自己的优势被增长指标重新定义。
- 使用边界：避免把数字化写成唯一方向；需检查用户是否接触过会员、数据、门店和履约项目。
- 待确认问题：你当年有没有机会把线下用户、门店经验或内容能力接到线上数据和复购上？

#### 教培扩张和教育焦虑外包

- 时间范围：2017-2021
- 影响人群：K12 教培从业者、家长、内容和销售岗位、转行进入教育行业的人。
- 影响变量：收入弹性、岗位安全、家庭教育支出、销售压力、行业合规。
- 适用岔路：进教培 / 留原行业；销售增长 / 内容教研；市场化高收入 / 稳定岗位。
- 影响机制：教培扩张期让大量内容、销售、运营和管理岗位快速成长，也把家庭焦虑、业绩压力和政策风险压到从业者身上。复盘时要区分个人能力增长和行业扩张带来的短期收入。
- 使用边界：不评价家长选择和教育方式；只分析行业周期、岗位迁移和现金流承受。
- 待确认问题：你当时拿到的是可迁移能力，还是平台扩张期的岗位红利？

#### 新消费和消费品牌兴起

#### 内容合规和平台治理常态化

- 时间范围：2019-2022
- 影响人群：内容创作者、MCN、品牌市场、公关、直播团队、平台运营。
- 影响变量：表达边界、平台规则、账号安全、商业化稳定性、品牌风险。
- 适用岔路：公开表达 / 幕后岗位；单平台增长 / 多平台组合；内容创业 / 公司岗位。
- 影响机制：内容增长从“谁更敢说、谁更会追热点”进入规则、证据和长期人设并重的阶段。复盘内容路径时，要看用户是否具备稳定栏目、事实校验和商业承接能力。
- 使用边界：避免鼓励擦边、搬运或制造冲突；只分析内容能力、风险边界和可持续交付。
- 待确认问题：你的表达能力有没有沉淀成稳定栏目和可信身份，还是只停留在临时热点判断？

### 2020-2022：疫情、远程办公、平台监管、教育变化、地产回落

#### 疫情冲击

- 时间范围：2020-2022
- 影响人群：城市迁移者、留学生、家庭照护者、线下行业、跨境流动人群。
- 影响变量：城市流动、健康风险、家庭距离、远程办公、职业稳定性。
- 适用岔路：回国 / 留海外；留一线 / 回家；线下行业 / 线上化岗位。
- 影响机制：流动受限、健康风险和家庭责任被放大，远程协作能力变重要。
- 使用边界：避免制造健康焦虑，只分析实际约束和支持网络。
- 待确认问题：疫情是否改变你的城市、家庭照护、海外计划或工作节奏？

#### 远程办公和线上化

- 时间范围：2020-2022
- 影响人群：互联网、教育、内容、咨询、跨境协作岗位。
- 影响变量：城市位置、自我管理、线上表达、工具使用、边界感。
- 适用岔路：换城市 / 留原地；线下岗位 / 线上岗位；公司平台 / 自由职业。

- 适用岔路：国内市场 / 海外市场；内容市场 / 出海运营；硬件供应链 / 纯平台业务。
- 影响机制：出海从“把货卖出去”变成渠道、认证、售后、文化和雇佣的组合题。懂行业、懂表达、懂海外用户的人，可能把过去的行业经验重新打包成国际化资产。
- 使用边界：避免宏观预测；只分析已发生贸易规则、本地化和能力迁移。
- 待确认问题：你是否接触过海外用户、供应链、英文材料、本地渠道或跨境客户？
