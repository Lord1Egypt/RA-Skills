# Source Trust Levels / 来源可信度等级

## trusted / 可信来源

Official, primary, stable, or user-approved sources. Examples: government portals, standards bodies, verified company primary documents, approved internal knowledge bases.

官方、一手、稳定或用户批准来源。例如政府网站、标准机构、已验证公司一手文件、已批准内部知识库。

## allowed / 可用来源

Useful sources that can be staged but usually require review before permanent archive.

有用但通常需要审核后永久归档的来源。

## review / 待审核来源

Ambiguous, unverified, OCR-derived, user-uploaded sensitive, or context-dependent sources.

不明确、未验证、OCR 提取、用户上传敏感文件或依赖上下文判断的来源。

## blocked / 屏蔽来源

Sources that should not be archived because they are unreliable, prohibited, irrelevant, malicious, or outside user policy.

因不可靠、禁止、无关、恶意或违反用户策略而不应入库的来源。
