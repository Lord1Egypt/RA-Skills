## Known Risks and Mitigations: <br>
Risk: Statistical guidance can be misapplied if assumptions, sample size, or multiple testing are not reviewed. <br>
Mitigation: Validate test assumptions, report effect sizes with p-values, and apply multiple-testing correction when running several tests. <br>
Risk: The skill references a Python package whose runtime code is not included in this documentation-only bundle. <br>
Mitigation: Install any referenced package only from a trusted source and review that package separately before execution. <br>
