# 多非利特基于肾功能的用药决策

## 触发条件
当以下任一情况成立时触发本技能：
- 患者拟使用多非利特治疗心房颤动
- 患者存在肾功能异常或尚未评估肾功能

## 执行步骤

1. **确认前提条件**：
   - 患者确诊为心房颤动，需恢复或维持窦性心律
   - 已准确测定肌酐清除率（CrCl）

2. **根据CrCl分层决策**：
   - 若 CrCl ≥60 mL/min：可使用标准剂量（如500 μg，每日两次）
   - 若 CrCl 40–59 mL/min：剂量减半（如250 μg，每日两次）
   - 若 CrCl 20–39 mL/min：使用更低剂量（如125 μg，每日两次），且必须住院并进行心电监护
   - 若 CrCl <20 mL/min（肾衰竭）：**绝对禁用**，立即发出红色警报

3. **实施安全监测**：
   - 所有患者首次给药必须在医院进行，连续心电监护至少3天
   - 监测QTc间期，若 >500 ms 应停药
   - 纠正电解质紊乱，确保血钾 >4.0 mmol/L、血镁正常
   - 避免联用其他延长QT间期的药物

4. **输出结果**：
   - 明确是否允许使用多非利特
   - 推荐具体剂量（如适用）
   - 列出必须的监测与预防措施
   - 若 CrCl <20 mL/min，输出“红色警报：多非利特禁用”

> **注意**：未评估肾功能前不得启动多非利特治疗。
