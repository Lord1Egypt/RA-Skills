# 儿科药物剂量计算

## 何时使用
- 需为儿童（含新生儿、婴儿、青少年）开具处方药物；
- 已获取患儿实测体重（kg）和药物在儿童中的推荐剂量数据；
- 药物说明书或指南未直接提供该年龄段剂量，需自行计算。

## 核心计算方法选择
1. **优先按体重计算**：适用于大多数药物，尤其是急诊用药（如退热药、镇静剂）。
2. **按体表面积计算**：用于治疗窗窄、毒性大的药物（如化疗药、某些抗生素），精度更高。
3. **按年龄估算**：仅限剂量范围宽、安全性高的药物（如维生素、营养补充剂）。
4. **从成人剂量折算**：仅在缺乏儿童剂量数据时作为最后手段。

## 执行步骤
1. 确认患儿体重（kg）和药物类型（是否需精确给药）；
2. 根据药物特性选择合适计算方法；
3. 应用对应公式计算初始剂量；
4. 检查是否超过成人剂量上限；
5. 结合患儿年龄、肝肾功能等临床因素进行调整；
6. 输出最终剂量（精确至毫克或毫升）、给药频次与途径。

## 关键约束
- 新生儿或肝肾功能不全者通常需减量；
- 年长儿按体重计算若超成人剂量，以成人剂量为上限；
- 抢救用药（如阿托品）可突破常规剂量数倍至数十倍；
- 需穿透血脑屏障的药物（如化脑治疗用青霉素）应适当加量。
