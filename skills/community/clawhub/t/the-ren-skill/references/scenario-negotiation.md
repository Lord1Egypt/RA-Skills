# 场景四：识对手（博弈·竞对分析·冲突应对）

> 商业博弈中看穿对手：判断对手类型、预测下一步行动、在冲突中获得信息优势。
>
> **⚡ 协作：** 先用 `personality-model.md` 六型画像判断对手，再用本文件做博弈落地。
> 本文件的"虎狼狐龟"四型与六型人格有以下映射关系。

---

## 一、对手类型快速识别

### 六型人格在博弈场景的映射

**博弈场景是"零和/负和"思维，日常场景是"正和"思维。同一类型的人，在做对手时的表现和做同事时不同。**

| 日常六型 | 博弈形态 | 博弈别名 | 核心弱点 |
|---|---|---|---|
| 将帅型 | 虎型（攻击性） | 正面硬刚型对手 | 傲慢——看不起"弱"，会被比他更稳的人打乱节奏 |
| 谋士型 | 狼型（算计型） | 暗中布局型对手 | 过度分析——给他太多不相关信息可以拖慢他的决策 |
| 先锋型 | 狐型（虚张声势型） | 借势压人型对手 | 泡沫怕针——追问细节、沉默、用数据打脸都能让他崩 |
| 基石型 | 龟型（保守型） | 稳守不出型对手 | 怕不确定性——不给他安全感他就会一直缩着不出招 |
| 伪将帅/嘴炮型 | 混合虎+狐 | 最危险的对手 | 有能力+没底线——唯一的应对是不跟他同台竞技 |
| 成长型 | 通常不作为独立对手出现 | — | — |

### 博弈四型速查（传统分类保留，与六型对应）

| 类型 | 对应的六型 | 面相/行为特征 | 应对策略 |
|---|---|---|---|
| **虎型（攻击性）** | 将帅型 | 眼神犀利/下巴微抬/说话时手势大/坐姿侵占空间/音量偏大/被质疑时前倾对抗 | 不硬碰，用逻辑和数据消耗他的攻击力；让他先出招，你后发制人 |
| **狼型（算计型）** | 谋士型 | 眼神游离但聚焦时锐利/说话慢而有停顿/被质疑时不急回应先观察/很少真笑/法令纹深但笑不到眼睛 | 信息不对等是对他最有效的武器；少给他信息，多观察 |
| **狐型（虚张声势型）** | 先锋型/嘴炮型 | 音量刻意大/手势多而夸张/眼神要么死盯要么飘/被质疑时防御激烈/名表名车LOGO满身 | 用沉默和追问细节刺破他的泡沫；泡沫破了他自己会乱 |
| **龟型（保守型）** | 基石型 | 说话慢/姿态收缩/眼神回避/被质疑时沉默/不主动出招/法令纹浅或没有 | 给他安全感他反而会配合；施压太重他会缩回壳里 |

### 快速判断对手类型（30秒）

1. 他的坐姿占了多少椅子？（全占=虎/狐，半占=狼，1/3=龟）
2. 他的眼神和你对视的频率？（高=虎/狐，中=狼，低=龟）
3. 你打断他时他的反应？（立刻反击=虎，观察你=狼，激烈防御=狐，沉默=龟）

### 用六型判断对手的下一步

| 对手六型 | 压力下的默认动作 | 你要怎么预判 |
|---|---|---|
| 将帅/虎型 | 加大施压，正面猛攻 | 他出重拳时往往侧翼空虚——注意他团队中他不太关注的细节 |
| 谋士/狼型 | 等待，收集更多信息再做决定 | 他"不动"的时候其实在算——不要被他的沉默骗出底牌 |
| 先锋/狐型 | 制造紧迫感，"再不决定就没机会了" | 他的所有"最后期限"几乎都是假的——顶住节奏你就赢了 |
| 基石/龟型 | 拖延，"再考虑考虑" | 他不是在拖延是在等你让步——给他一个安全的台阶他会下的 |
| 伪将帅/嘴炮 | 切换策略极快，什么好用用什么 | 没有固定模式就是他的模式——守住你的底线，不被他带节奏 |

## 二、博弈中的读心术

### 对手在说谎时的信号

| 信号 | 机制 | 你该怎么做 |
|---|---|---|
| 回答前有微停顿（比平时长0.5秒以上） | 认知负荷：在脑子里构建谎言 | 在这个话题上追问细节 |
| 回答时肢体突然减少 | 大脑资源被编造谎言占满 | 让他倒序讲一遍 |
| 说完后摸鼻子/摸后颈 | 安慰行为：对自己说的话感到不安 | 沉默5秒，让他自己补充（越描越黑） |
| 眼睛死盯着你 | 过度补偿：知道自己在说谎，用眼神"证明"清白 | 不要被盯怕，保持自然对视 |
| 重复你的问题 | "你问我有没有X？我当然有X" | 在争取时间编造，继续追问 |

### 对手在虚张声势时的信号

| 信号 | 含义 | 应对 |
|---|---|---|
| 音量突然变大+身体前倾+手势增加 | 用气势掩盖底气不足 | 不退不让，平静反问细节 |
| "我还有别的选择"但说不出具体是什么 | 在威胁，没筹码 | "那很好，你可以去比较一下"（反向施压） |
| "我不在乎"但频繁看那个东西 | 极度在乎 | 不要被他带节奏 |
| 说"最后一次机会"但还在继续谈 | 不是最后一次 | 继续按你的节奏谈 |

### 对手真的准备离开时的信号

| 信号 | 含义 |
|---|---|
| 身体转向门口/脚指向出口方向 | 身体已经准备好离开 |
| 开始收拾物品（合上本子/收起笔/拿包） | 心理已经在结束 |
| 语气变平淡，情绪消失 | 已经不在意结果了 |
| 从"你可以考虑一下"变成"那就先这样吧" | 给台阶下→不再给台阶 |

## 三、竞对分析

### 通过公开信息判断竞争对手创始人的性格

| 信息来源 | 分析方法 |
|---|---|
| 他的公开演讲/采访视频 | 看他的眼神（定vs飘）、手势（自然vs夸张）、被追问时的表情（开放vs防御） |
| 他写/发的社交媒体内容 | 归因模式：夸别人多还是夸自己多？分析失败还是分析成功？ |
| 他的团队的流动率/离职评价 | 高流动率+差口碑=对手自身可能有问题 |
| 他的产品的用户体验细节 | 细节做得好→对手是注重细节的人；细节粗糙→对手可能粗放 |

### 判断对手是否会出阴招

| 危险信号 | 含义 |
|---|---|
| 他的社交媒体全是黑别人的，从不夸人 | 攻击性是内在性格，不是策略 |
| 他之前的合作伙伴全都"背叛"了他 | 概率上不可能所有人都背叛他→问题在他 |
| 他的眼神中有一种"什么都做得出来"的冷 | 没有道德约束感 |
| 他聊竞争对手时的微表情是兴奋（不是冷静） | 他享受攻击，不是理性竞争 |

## 四、冲突中的识人

### 谈判陷入僵局时的对手分析

| 对手的反应 | 面相/行为信号 | 含义 |
|---|---|---|
| 沉默+身体不动 | 在思考/在等你先让步 | 谁先开口谁弱势。如果你不急，可以等他开口 |
| 皱眉+嘴角紧绷 | 在压抑愤怒 | 不要在这时施压，给他台阶 |
| 眼神飘向同伴 | 在寻求支持 | 他的立场可能在软化 |
| 开始玩手机/看窗外 | 兴趣在降低 | 需要在3分钟内重新建立兴趣点 |
| 身体前倾+语速变慢 | 准备认真谈条件了 | 这是真正的转机信号 |

### 对方情绪失控时

**永远不要在对方情绪失控时做决策——无论是让步还是强硬。** 情绪中的决策事后都会后悔。

| 失控类型 | 表现 | 应对 |
|---|---|---|
| 愤怒型 | 声音变大/面色变红/手势激烈 | 不跟着大声，反而降低音量，用平静让他冷静 |
| 崩溃型 | 眼圈红/嘴唇颤抖/声音哽咽 | 暂停谈判，给对方空间（"我们休息10分钟"） |
| 攻击型 | 人身攻击/贬低你的产品或人品 | 不接招（接招你就输），"我们回到正题" |
| 冷暴力型 | 沉默/不看你/摆弄手机 | "我们今天先到这，改天再约" |

## 五、博弈后的复盘

每次重要博弈/谈判后，强制自己复盘：

1. **对手在哪个时刻露出了真实的底线？**（回想微表情变化的时间点）
2. **我有没有被对手的光环效应影响？**（因为他穿得好/开好车/头衔高就高估了他？）
3. **对手是什么六型/博弈类型？**（虎/狼/狐/龟，对应将帅/谋士/先锋或嘴炮/基石）→我的应对策略对不对？
4. **如果重新来一次，我会在他哪一个微表情出现后改变策略？**
5. **这次博弈中，对手的压力默认动作是什么？**（将帅加大施压/谋士等待/先锋制造紧迫感/基石拖延）→我有没有被他的节奏带着走？

## 附：商业饭局中的识人

**饭局是最好的识人场景——酒精放松了伪装，餐桌暴露了本性。**

| 观察点 | 信号分析 |
|---|---|
| 对服务员的态度 | 最真实的共情测试（见前文"服务员测试"） |
| 喝酒后的眼神变化 | 眼神变柔和=真放松；眼神变凶=压抑的攻击性释放；眼神变飘=可能有什么在隐瞒 |
| 挑菜的方式 | 翻来翻去挑最好的=自我中心；先让给别人=共情高 |
| 买单时的反应 | 主动买单+不张扬=有担当；假装买单（掏钱包慢动作）=虚伪；从不买单+理所应当=不感恩 |
| 说"我干了，你随意"时的眼神 | 真诚期待vs施压逼迫——看眼轮匝肌是否参与（真诚的笑眼睛会眯） |
