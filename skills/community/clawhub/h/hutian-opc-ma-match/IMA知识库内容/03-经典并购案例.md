# 经典并购案例汇编

## 一、大陆A股经典案例

---

### 案例1：中芯国际收购中芯北方（科创板最大重组案）

#### 基本信息
| 项目 | 内容 |
|------|------|
| 买方 | 中芯国际（688981.SH） |
| 卖方 | 国家集成电路基金等5名股东 |
| 标的 | 中芯北方49%股权 |
| 交易金额 | 406.01亿元 |
| 交易时间 | 2026年5月过会 |

#### 交易结构
- **发行价格**：74.2元/股
- **支付方式**：发行股份购买资产
- **标的业务**：12英寸集成电路晶圆代工
- **交易后**：中芯北方将成为全资子公司

#### 交易亮点
1. **科创板开板以来规模最大的重组交易**
2. **审核快速**：2月25日受理→5月11日过会，约2.5个月
3. **产业整合**：中芯北方为控股子公司，本次交易实现100%控股
4. **协同效应**：提升晶圆制造产业集约化、规模化发展

#### 监管审批
- 上交所受理
- 重组委会议通过
- 符合"科创板八条"产业整合导向

#### 启示
- 半导体产业链纵向整合成为主流
- 大型央国企并购审核效率显著提升
- 符合国家战略的产业并购获政策大力支持

### 案例3：长电科技收购星科金朋（A股首单跨境收购全球一流封测商）

#### 基本信息
| 项目 | 内容 |
|------|------|
| 买方 | 长电科技（600584.SH） |
| 标的 | STATS ChipPAC Ltd.（星科金朋） |
| 交易金额 | 约7.8亿美元（企业价值18亿美元） |
| 交易时间 | 2015年完成 |

#### 交易结构
- **联合收购方**：长电科技+国家集成电路产业投资基金+芯电半导体
- **收购主体**：JCET-SC（新加坡特殊目的公司）
- **方式**：自愿有条件全面要约私有化

#### 交易亮点
1. **A股上市公司首次跨境收购全球一流封测公司**
2. **"蛇吞象"式并购**：长电科技当时市值不足百亿，收购规模超百亿
3. **产业基金撬动**：联合国家集成电路产业投资基金完成
4. **获取核心技术**：扇出型封装、芯片凸点倒装等先进封装技术

#### 收购协同
| 协同类型 | 具体表现 |
|----------|----------|
| 技术协同 | 先进封装技术全球领先 |
| 客户协同 | 进入高通、博通等国际客户供应链 |
| 产能协同 | 全球四地布局制造测试中心 |

#### 启示
- 跨境并购需借助产业基金撬动资金
- "走出去"获取先进技术是重要战略
- 私有化退市后整合是关键挑战

### 案例6：顺丰控股借壳鼎泰新材（2017年完成）

#### 基本信息
| 项目 | 内容 |
|------|------|
| 壳公司 | 鼎泰新材（002352.SZ） |
| 借壳方 | 顺丰控股 |
| 交易金额 | 433亿元 |
| 完成时间 | 2017年2月 |

#### 交易结构
```
第一步：资产置换
鼎泰新材置出原有资产（作价8亿元）
顺丰控股100%股权置入（作价433亿元）

第二步：发行股份购买资产
差额部分由鼎泰新材向顺丰控股全体股东发行股份购买

第三步：配套融资
80亿元（引入战略投资者）
```

#### 审核要点
- 顺丰控股为非上市公司，需按重组上市审核
- 需满足《上市公司重大资产重组管理办法》第13条要求
- 配套融资需遵循再融资规定

#### 启示
- 借壳上市是快递等轻资产行业快速登陆A股的路径
- 资产置换+发行股份是经典借壳结构
- 2019年修订后借壳等同IPO审核，借壳案例大幅减少

### 案例7：恒生银行私有化（2025年）

#### 基本信息
| 项目 | 内容 |
|------|------|
| 要约人 | 汇丰亚太 |
| 方式 | 协议安排 |
| 价格 | 155港元/股 |
| 溢价 | 30%（较停牌前收盘价119港元） |
| 总估值 | 约2903亿港元 |
| 现金对价 | 约1062亿港元 |

#### 私有化目的
1. **简化决策流程**：汇丰与恒生两个品牌可更有效发挥协同效应
2. **提升风险管理**：增强资本投放质量和效益
3. **发掘协同效应**：提升营运效益

#### 启示
- 银行间合并可采用协议安排私有化
- 高溢价是获得股东支持的关键
- 保留品牌独立运营可减少整合阻力

### 案例8：康基医疗私有化（医疗板块估值困境案例）

#### 基本信息
| 项目 | 内容 |
|------|------|
| 上市时间 | 2020年6月 |
| 退市时间 | 2025年12月 |
| 上市开盘价 | 23.679港元/股 |
| 私有化价格 | 9.25港元/股 |
| 溢价 | 9.9%（较停牌前收盘价） |
| 估值 | 14亿美元 |

#### 私有化原因
1. **股价长期低迷**：从33.429港元高点跌至8.42港元
2. **流动性不足**：股票交易不活跃
3. **融资受限**：难以通过股权市场融资
4. **行业压力**：带量采购政策影响

#### 财团构成
- 创始人钟鸣夫妇（保留40%持股）
- TPG（全球另类资管巨头）
- 卡塔尔投资局（主权财富基金）

#### 启示
- 港股医疗板块估值困境是普遍现象
- 高溢价是私有化成功的前提
- 国际资本的参与有助于交易完成

### 案例10：万达商业私有化（经典H股私有化）

#### 基本信息
| 项目 | 内容 |
|------|------|
| 上市时间 | 2014年12月 |
| 私有化时间 | 2016年9月 |
| 要约价格 | 52.8港元/股 |
| 溢价 | 10%（较发行价48港元） |
| 总出资 | 344.55亿港元 |

#### 私有化原因
1. **A股估值更高**：当时A股房地产公司估值显著高于H股
2. **回归A股**：计划在A股重新上市
3. **战略调整**：轻资产转型

#### 交易结构
- 成立多个境内外SPV
- 境内存款+境外贷款完成资金汇集
- 收购全部H股股东持有的6.52547亿股

#### 启示
- A+H股估值差异是H股私有化的重要动因
- 需与重要股东事先沟通取得支持
- 溢价合理是成功关键

## 四、失败案例分析

---

### 案例11：海光信息与中科曙光吸并重组终止

#### 基本信息
| 项目 | 内容 |
|------|------|
| 类型 | 吸收合并 |
| 终止原因 | 市场环境变化，条件尚不成熟 |

#### 终止分析
1. **交易规模大、涉及方多**：重大资产重组方案论证历时较长
2. **市场环境变化**：方案筹划至今市场发生较大变化
3. **未达失败**：因商业原因暂缓，非实质性障碍

#### 启示
- 超大型并购交易周期长，市场窗口把握难度大
- 宏观环境变化是并购终止的重要原因

---

### 案例12：帝奥微重组终止

#### 终止原因
- 交易双方就**交易方案、交易价格、业绩承诺**等核心条款未能达成一致

#### 启示
- 民营上市公司并购以市场化谈判为主
- 商业条款分歧是终止的主要原因
- 约30%的重组在草案公告前终止

## 五、案例总结与启示

### 5.1 成功并购要素

| 要素 | 说明 |
|------|------|
| 战略协同 | 产业整合、产业链延伸 |
| 定价合理 | 双方认可的估值水平 |
| 团队稳定 | 核心管理层留任 |
| 审批顺畅 | 符合监管导向 |
| 时机恰当 | 市场环境配合 |

### 5.2 并购失败主要原因

| 原因类别 | 占比 | 典型情形 |
|----------|------|----------|
| 商业条款分歧 | 高 | 估值、对赌、支付方式 |
| 市场环境变化 | 中 | 股价波动、行业周期 |
| 监管障碍 | 低 | 政策限制、审批未获 |
| 尽调问题 | 中 | 发现实质性障碍 |

### 5.3 政策导向影响

1. **"并购六条"后**：
   - 审核效率显著提升
   - 跨界并购获更多包容
   - 未盈利资产可被接受

2. **科创板八条后**：
   - 硬科技并购活跃
   - 产业整合成主流
   - 简易审核程序落地

3. **港股新规后**：
   - 借壳上市门槛提高
   - 反收购规则趋严
   - 私有化溢价需提高
