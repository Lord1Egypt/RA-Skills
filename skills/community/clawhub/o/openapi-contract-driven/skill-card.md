## Description: <br>
OpenAPI Contract Driven helps multi-agent teams standardize REST API collaboration with OpenAPI contracts, rule checks, generated TypeScript clients, CI checks, migration guidance, and role-specific agent instructions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineering teams use this skill to keep frontend, backend, architecture, operations, and coordinator agents aligned around one REST API contract. It is intended for new projects, existing REST API migrations, and debugging recurring frontend/backend endpoint mismatches. <br>
