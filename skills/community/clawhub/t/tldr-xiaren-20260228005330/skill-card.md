## Use Case: <br>
Developers, engineers, and coding agents use this skill to get concise command examples from the local tldr CLI before consulting longer manual pages. <br>

### Deployment Geography for Use: <br>

## Known Risks and Mitigations: <br>
Risk: Concise command examples can omit important context for destructive, privileged, or unfamiliar commands. <br>
Mitigation: Verify those commands with man, --help, or official documentation before running them. <br>
