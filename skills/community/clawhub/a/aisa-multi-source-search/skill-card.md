## Description: <br>
Intelligent search for agents with multi-source retrieval and confidence scoring across web, academic, smart, and Tavily sources. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[AIsaPay](https://clawhub.ai/user/AIsaPay) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agents use this skill to run web, academic, smart, full-text, and Tavily searches through AIsa APIs, then inspect confidence-scored results for research, market analysis, news aggregation, and source synthesis. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Search terms, target URLs, retrieved content, and result bundles are sent to AIsa's external API. <br>
Mitigation: Use a scoped API key and avoid confidential prompts, private URLs, or internal content. <br>
Risk: Crawl and map features may access third-party sites. <br>
Mitigation: Use those features only with authorization and within the target site's acceptable-use expectations. <br>
