## 1. 北京特别规定

### 1.1 禁鸣区域与时段

**重点区域**
```
🚫 禁止鸣笛区域：
- 五环路以内道路（部分路段）
- 长安街及其延伸线全线
- 王府井大街、东单/西单两条商业街
- 居民住宅区附近道路

🕐 禁鸣时段：
- 工作日：7:00-22:00（全市范围内禁鸣）
- 周末/节假日：8:00-21:00（部分路段放松管制）
```

**处罚标准**
- 🚨 违规鸣笛：罚款 100-500 元 + 扣分（部分电子眼抓拍）

### 1.2 进京检查站规定

**检查站点清单**
| 站点 | 位置 | 检查内容 |
|------|------|----------|
| 门头沟检查站 | 京港澳高速 G4 | 外省市车辆 + 货车 |
| 六环检查站 | 京藏高速 G6 | 重点查尾气排放 |
| 南口检查站 | 军都山隧道路段 | 货车限行检查 |

**检查内容**
- ✅ 检查车辆排放标准（国六标准）
- ✅ 检查环保标志有效性
- ✅ 检查违章记录处理情况

### 1.3 进京停车特别规定

**禁止停车区域**
```
📍 五环路以内：
- ❌ 长安街及其沿线道路两侧
- ❌ 王府井大街、东单/西单商业街
- ❌ 居民住宅区入口 50 米内（电子眼抓拍）
```

### 1.4 新能源车路权优待

**专用车道使用规则**
```
✅ 允许使用：公交专用道（7:00-20:00）
❌ 禁止占用：高速公路应急车道
⚠️ 提醒：非新能源车辆不得进入
```

## 2. 上海特别规定

### 2.1 禁鸣区域详细清单

**全域禁鸣范围**
```
📍 内环高架道路全线（禁止鸣笛）
📍 外环线以内所有主干道
📍 人民广场、南京路步行街周边
```

**处罚标准**
- 🚨 违规鸣笛：罚款 200 元 + 警告

### 2.2 黄浦区特别管制

**限行时段**
```
🕐 工作日高峰时段（8:30-17:30）：
   - ❌ 外省市货车禁止驶入内环高架及外环隧道
   - ✅ 新能源货车不受此限
```

### 2.3 停车管理特别严格

**禁止停车地点列表**
```
📍 陆家嘴金融贸易区（30 米内禁停）
📍 外滩/南京路步行街入口（电子眼抓拍）
📍 人民广场周边道路两侧
📍 地铁站出入口 50 米内
```

**违章停车处罚**
- 🚨 首次违规：罚款 200 元 + 贴罚单
- 🚨 第二次违规：罚款 200-300 元 + 拖车警告
- 🚨 第三次违规：罚款 500 元 + 强制拖移车辆

### 2.4 新能源车充电特别规定

**小区充电限制**
```
✅ 允许：物业备案 + 申请固定车位充电桩
❌ 禁止：楼道内/消防通道私拉电线充电（违者罚款 200-5000 元）
```

## 3. 深圳特别规定

### 3.1 限行管制区与时段

**工作日高峰时段（7:00-9:00,17:00-19:00）**
```
📍 外市籍新能源客车：允许通行所有道路
📍 非新能源货车：禁止进入以下区域
   - 罗湖区
   - 福田区
   - 南山区部分路段
   - 盐田区大部分区域
```

### 3.2 停车收费特别政策

**路边停车位收费标准**
| 时段 | 首小时 | 后续每小时 | 封顶 |
|------|--------|------------|------|
| **工作日白天**（7:00-19:00） | 4 元 | 4 元 | 20 元/天 |
| **夜间停车**（19:00-次日 7:00） | 免费 | 免费 | - |

### 3.3 电动车停放特殊规定

**禁停区域列表**
```
📍 地铁站出入口（50 米内禁停）
📍 公交站台（15 米内禁停）
📍 消防通道/应急车道
```

## 4. 广州特别规定

### 4.1 早晚高峰限行区域

**工作日早高峰（7:30-9:00）**
```
📍 外市籍非绿牌车：禁止进入内环高架快速路
📍 例外：新能源车辆不受限行限制
```

**晚高峰时段（17:30-19:00）**
```
📍 同早高峰，部分路段延伸范围
```

### 4.2 停车管理特别严格

**路边停车位限制**
```
❌ 禁止在以下区域停车：
   - 广州塔周边道路
   - 珠江新城核心区（30 米内）
   - 天河路商圈步行街区
罚款：50-200 元/次 + 拖车风险
```

### 4.3 外市车辆通行证规定

**办理条件**
```
需要办理《广州市外地车辆临时通行证》的情形：
1. 非本地车牌货车进入内环线以内
2. 周末进入部分限行区域（如琶洲）
办理流程：交管 12123 APP → 外市籍车辆申请 → 审核通过后生成电子证
```

## 5. 杭州特别规定

### 5.1 限行政策

**工作日高峰时段（7:30-9:00,16:30-18:30）**
```
📍 外市籍小客车：禁止进入高架快速路及钱江四桥以下道路
✅ 新能源车辆不受限行限制
❌ 外地车牌网约车/出租车需办理通行证
```

**禁行区域**
```
📍 西湖南山路、湖滨路：禁止货车进入（全天）
📍 断桥至雷峰路段：禁止外省市牌照车辆通行（8:00-17:30）
```

### 5.3 停车费优惠时段

**西湖景区周边**
```
🕐 非高峰时段（9:00-16:30,18:30-次日 7:30）：
   - 首小时：2 元/小时
   - 后续时段：免费
```

## 📚 附录：地方性法规查询渠道

### 官方渠道

1. **公安部交通管理局官网**：http://jtt.gov.cn/
   - 各地实施细则查询入口

2. **交管 12123 APP**
   - 「我的」→「地方规定」→查看本城市特别管制

3. **本地交警微信公众号**
   - 如："北京交警"、"上海交警"等官方发布渠道

4. **地方交通管理局官网**
   - 各地市政府网站 → "交通安全管理"栏目
