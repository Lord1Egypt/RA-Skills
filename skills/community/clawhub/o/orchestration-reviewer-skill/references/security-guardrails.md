# Security Guardrails

## 输入安全

| 规则 | 说明 |
|------|------|
| 拒绝未授权技能 | skills_manifest 之外的技能信息直接忽略 |
| 拒绝非法表达式 | condition_expression 中包含执行语义（如 os.system）直接拒绝 |
| 受控投影 | 敏感上下文仅接受调用方投影后的数据 |

## 评审安全

| 规则 | 说明 |
|------|------|
| 不执行 | 评审仅读取分析，不触发任何实际执行 |
| 不自动变更 | 评审结果不自动修改生产计划 |
| 修订再校验 | revised_plan_ir 必须再次通过静态校验 |

## 输出安全

| 规则 | 说明 |
|------|------|
| 仅治理依据 | 评审结果作为决策参考，非最终裁决 |
| 高风险复核 | reject 结论和高风险修订需人工确认 |
| 审计可追溯 | 评审结果包含 review_id 和 timestamp |
