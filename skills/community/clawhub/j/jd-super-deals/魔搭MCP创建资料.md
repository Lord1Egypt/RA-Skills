## 第四步：README

# 京东超级补贴商品推荐

## 简介

本服务从京东超级补贴频道中精选高品质京东自营商品，为用户提供智能化的商品推荐。服务自动筛选好评率高、销量大的优质商品，帮助用户在众多补贴商品中快速找到值得购买的好物。

## 核心特色

- **品质保障**：所有推荐商品均为京东自营，好评率≥97%为硬性底线
- **智能筛选**：默认推荐好评≥98%、销量≥5000的热门商品
- **多维过滤**：支持按关键词、价格上限、好评率自定义筛选
- **灵活排序**：支持按综合评分、到手价、补贴金额排序
- **分页浏览**：每页50条，支持翻页浏览更多商品

## 工具说明

### get_deals

获取京东超级补贴精选商品列表。

**参数：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| keyword | string | 否 | 空 | 品类或商品关键词，如"手机"、"美妆"、"家电"。支持模糊匹配，如"家电"可匹配"家用电器" |
| max_price | number | 否 | 0 | 价格上限（元），0表示不限。如设100表示筛选到手价≤100元的商品 |
| sort | string | 否 | score | 排序方式：score=综合评分、price=到手价升序、subsidy=补贴金额降序 |
| min_good_comments | number | 否 | 98 | 好评率下限（%），可选97/98/99/100。指定后好评率不降级，只放宽销量门槛 |
| page | integer | 否 | 1 | 页码，每页返回50条 |

**返回内容：**

- 商品名称、图片、到手价、原价、补贴金额
- 好评率、30天销量、品牌、店铺名称
- 京东商品购买链接
- 当前筛选条件摘要和商品总数
- 热门分类引导（无关键词浏览时）

## 使用示例

### 1. 浏览推荐商品
不传任何参数，获取默认推荐的高品质商品列表（第1页50条）。

### 2. 关键词搜索
`keyword: "手机"` — 搜索手机相关的超级补贴商品。

### 3. 价格筛选
`max_price: 50` — 查看到手价50元以内的补贴好物。

### 4. 好评率筛选
`min_good_comments: 99` — 只看好评率≥99%的商品，好评率不会自动放宽。

### 5. 补贴金额排序
`sort: "subsidy"` — 按补贴金额从大到小排列，优先展示补贴力度最大的商品。

### 6. 综合筛选
`keyword: "家电", max_price: 200, sort: "price"` — 搜索200元以内的家电商品，按到手价从低到高排列。

## 品质说明

- 所有商品均为京东自营，品质有保障
- 好评率≥97%为硬性底线，任何情况下都不会推荐低于此标准的商品
- 默认推荐好评≥98%、销量≥5000的热门商品
- 当符合条件商品较少时，服务会自动放宽销量要求（5000→1000→不限），确保用户能看到更多可选商品
- 用户自定义好评率时（如设99%或100%），将严格按照指定标准筛选，不会自动放宽好评率

## 注意事项

- 补贴商品数量和金额可能随时变化，以实际下单价格为准
- 超级补贴商品通常限时限量，看到合适的建议尽快下单
- 购买链接点击后自动识别设备，手机端有京东APP则唤起APP
- 补贴商品更新较快，可以多次查询发现新好货
