**域间逻辑流**：保险产品 → 保险销售 → 保险理赔 → 保险管理

## 依赖拓扑摘要

### 一、域内链路

**保险产品链路**: T1-1 → T1-2 → T1-3 → T1-4
**保险销售链路**: T2-1 → T2-2 → T2-3 → T2-4
**保险理赔链路**: T3-1 → T3-2 → T3-3 → T3-4
**保险管理链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整保险服务链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T3-1 → T3-2 → T3-3 → T3-4 → T4-1 → T4-2 → T4-3 → T4-4

**反馈改进链路**: T4-1 → T1-1（风险管理反馈到产品设计）

**理赔驱动链路**: T3-3 → T1-2（理赔审核驱动产品定价）

更多组合根据具体任务动态推导。
