## Description: <br>
Provides quick-reference guidance for authorized CTF binary exploitation challenges, including overflow, ROP, heap, format string, sandbox, and kernel techniques. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[gandli](https://clawhub.ai/user/gandli) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and security practitioners use this skill for authorized CTF, training, and isolated lab binary exploitation workflows, including triage, technique selection, exploit planning, and command reference. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill provides broad, actionable exploitation and privilege-escalation guidance that could be misused outside authorized labs. <br>
Mitigation: Install only for authorized CTF, training, or isolated lab exploitation, and require human approval before running exploit commands, making network callbacks, changing system files, weakening kernel protections, or attempting privilege escalation. <br>
