# 中国标书格式规范

## 一、页面设置

| 项目 | 标准值 | 备注 |
|------|--------|------|
| 纸张 | A4 (210mm × 297mm) | 除非招标文件另有要求 |
| 上边距 | 2.54cm (1英寸) | — |
| 下边距 | 2.54cm (1英寸) | — |
| 左边距 | 3.17cm (1.25英寸) | 装订侧 |
| 右边距 | 3.17cm (1.25英寸) | — |
| 页眉 | 1.5cm | 可含项目名称/公司名 |
| 页脚 | 1.75cm | 页码位置 |

## 二、字体规范

| 用途 | 字体 | 字号 | 行距 | 备注 |
|------|------|------|------|------|
| 封面标题 | 黑体/方正小标宋 | 二号 (22pt) | — | 加粗，居中 |
| 一级标题 (章) | 黑体 | 三号 (16pt) | 1.5倍 | 加粗，居中 |
| 二级标题 (节) | 黑体 | 小三号 (15pt) | 1.5倍 | 加粗，左对齐 |
| 三级标题 (条) | 黑体 | 四号 (14pt) | 1.5倍 | 加粗，左对齐 |
| 正文 | 宋体 | 小四号 (12pt) | 1.5倍 | 首行缩进2字符 |
| 表格内容 | 宋体 | 五号 (10.5pt) | 单倍 | — |
| 页眉 | 宋体 | 五号 (10.5pt) | — | — |
| 页码 | Times New Roman | 小五号 (9pt) | — | 底部居中 |

## 三、标题编号规则

```
第一章  [标题]          ← 一级标题（章）
  1.1  [标题]           ← 二级标题（节）
    1.1.1  [标题]       ← 三级标题（条）
      1. [标题]         ← 四级标题（款）
        (1) [标题]      ← 五级标题（项）
```

## 四、表格规范

| 项目 | 要求 |
|------|------|
| 表格类型 | 三线表（顶线、栏目线、底线） |
| 表题位置 | 表上方居中 |
| 表题格式 | 黑体五号，如"表3.1 技术参数响应表" |
| 表内文字 | 宋体五号 |
| 表格注 | 表下方，宋体小五号 |

### 三线表格式示例
```
表3.1 技术参数响应表
┌──────┬──────────┬──────────┬──────────┬──────────┐
│ 序号 │ 招标要求 │ 投标响应 │ 偏离情况 │ 说明     │  ← 顶线+栏目线
├──────┼──────────┼──────────┼──────────┼──────────┤
│ 1    │ ...      │ ...      │ ...      │ ...      │
│ 2    │ ...      │ ...      │ ...      │ ...      │
└──────┴──────────┴──────────┴──────────┴──────────┘  ← 底线
注：偏离情况填写"无偏离"、"正偏离"或"负偏离"
```

## 五、页码规范

| 位置 | 规则 |
|------|------|
| 封面、目录 | 不编页码 / 单独编罗马数字 |
| 正文 | 从第一章开始，阿拉伯数字连续编号 |
| 页码位置 | 页面底部居中 |
| 页码格式 | - 1 - / 第 1 页 |

## 六、图表编号规范

| 类型 | 编号格式 | 示例 |
|------|---------|------|
| 图 | 图X.X | 图3.1 系统架构图 |
| 表 | 表X.X | 表4.2 设备清单 |
| 公式 | 式(X-X) | 式(3-1) |

## 七、装订规范

| 项目 | 要求 |
|------|------|
| 装订方式 | 左侧胶装/线装（活页装订通常不允许） |
| 正本/副本 | 封面标注"正本"、"副本"字样 |
| 密封 | 正副本分别密封，加盖公章 |
| 外层封面 | 注明项目名称、投标人名称、"开标前不得启封"字样 |

## 八、电子投标补充规范

| 项目 | 要求 |
|------|------|
| 文件格式 | PDF扫描件（含签字盖章） |
| 扫描分辨率 | 不低于300dpi |
| 文件命名 | 按招标文件要求 |
| 加密要求 | 按电子招投标平台要求 |

## 九、常见错误规避

1. ❌ 正副本内容不一致 → ✅ 正副本必须完全一致
2. ❌ 缺少页码或页码不连续 → ✅ 逐页检查页码
3. ❌ 错别字/格式不统一 → ✅ 逐字校对，统一格式
4. ❌ 目录与正文标题不一致 → ✅ 生成目录后逐项核对
5. ❌ 签字盖章处遗漏 → ✅ 列出所有需要签字盖章的位置
6. ❌ 资质证书过期 → ✅ 检查所有证书有效期
7. ❌ 投标函法定内容缺失 → ✅ 对照招标文件逐项检查
