# Launch Readiness Checklist (Go / No-Go)

A 40-point quality gate. A launch is GO only when every Critical item is checked.

## 1. Listing (Critical)
- [ ] Title keyword in first 30 characters
- [ ] All variants created with correct SKUs
- [ ] 5+ images including hero, in-use, scale
- [ ] Product video on the card
- [ ] Specs / dimensions / care complete

## 2. Media & Quality
- [ ] All media passed approval
- [ ] Listing quality score ≥ 90%
- [ ] No pending review flags

## 3. Pricing & Promo (Critical)
- [ ] Anchor price set above launch price
- [ ] Launch coupon created with a cap/end
- [ ] Margin positive at fully-promoted price
- [ ] Bundle / quantity break configured (if used)

## 4. Inventory (Critical)
- [ ] Stock confirmed vs forecast peak
- [ ] ≥ 14 days of cover (≥ 30 preferred)
- [ ] 3PL receipt confirmed (if applicable)
- [ ] Restock PO drafted with trigger level

## 5. Affiliates
- [ ] Target creator list built
- [ ] Outreach sent with commission + sample
- [ ] Samples shipped 4–5 days out
- [ ] Brief shared (hooks, claims, link)
- [ ] 3+ creators confirmed

## 6. Compliance (Critical)
- [ ] Claims reviewed and cleared
- [ ] No trademarked terms in title/tags
- [ ] Category and attributes correct
- [ ] Required certificates uploaded
- [ ] Promo type allowed for category

## 7. Launch-Day Readiness
- [ ] Buy button live in target region
- [ ] Promo active and visible
- [ ] First creator content scheduled
- [ ] Test order completed successfully
- [ ] CS replies / FAQ ready

## 8. Post-Launch Setup
- [ ] Review-request flow live
- [ ] Monitoring dashboard / thresholds set
- [ ] Restock trigger defined
- [ ] Creator cadence planned (2–3/day)
- [ ] Escalation contact confirmed

## 9. Sign-off (Critical)
- [ ] Owner assigned to every open task
- [ ] All Quick-Reference rows ≥ Acceptable
- [ ] No Critical item unchecked
- [ ] GO/NO-GO recorded with date
