# Lottery Predictor - 彩票预测分析工具

中国体育彩票/福利彩票预测分析工具，支持多种玩法的数据分析和号码推荐。

## ⚠️ 免责声明

**本工具仅供娱乐和学习数据分析使用，不构成任何投注建议。**

彩票开奖是独立随机事件，历史数据不具备预测未来开奖结果的能力。请理性购彩，量力而行，切勿沉迷。

## 支持的彩票类型

| 彩票 | 类型 | 特点 |
|------|------|------|
| 排列三 | 数字型 | 3位数字，每日开奖 |
| 排列五 | 数字型 | 5位数字，每日开奖 |
| 大乐透 | 乐透型 | 5+2，前区1-35，后区1-12 |
| 双色球 | 乐透型 | 6+1，红球1-33，蓝球1-16 |
| 七星彩 | 数字型 | 7位数字 |
| 足彩14场 | 竞猜型 | 14场比赛胜平负预测 |

## 使用方法

```bash
# 排列三分析
python3 lottery_predictor.py pl3 --analyze --recommend

# 大乐透分析
python3 lottery_predictor.py dlt --analyze --recommend

# 双色球分析
python3 lottery_predictor.py ssq --analyze --recommend
```

## 分析方法

- **冷热号分析**: 统计号码出现频率
- **遗漏值分析**: 追踪号码未开出的期数
- **走势分析**: 和值、奇偶比、大小比
- **足彩赔率分析**: 基于赔率计算胜平负概率
