# Herb Harvesting Season Determination

## When to Use
- 中药材（包括植物药、动物药、矿物药）

## When to Use
- 已明确药材的药用部位（如根、茎、叶、花、果实、全草等）
- 已了解该药材主要有效成分的动态变化规律

## Execution Steps
1. **确认药用部位**：明确目标药材的入药部分。
2. **查询有效成分高峰期**：依据文献或研究数据，确定该部位中主要有效成分含量最高的时期。
3. **结合传统采收原则**：参考《本草蒙筌》等典籍中的采收规律：
   - 茎叶：采未老枝茎，因“汁正充溢”；
   - 花蕊：摘将开之花，因“气尚包藏”；
   - 果实：收已熟者，因“味纯”；
   - 叶片：采新生叶，因“力倍”。
4. **综合判断最佳采收时间**：结合现代成分分析与传统经验，确定具体采收季节或月份。
5. **规避禁忌**：不得在有效成分低谷期采收，不得违背自然生长节律，避免“恣孟浪”（随意采收）。

## Output
返回该中药材的最佳采收时间点（如季节、月份或物候阶段），确保有效成分含量最大化，保障药材质量与临床疗效。
