# Install-time activation quickstart

## What this page is for

Use this page when you want the skill to start doing gentle proactive care after installation.

## Minimal setup

1. Install the skill.
2. Confirm the user wants proactive care.
3. Choose a small number of jobs:
   - bedtime check-in
   - sparse idle-time greeting
4. Send through the active channel, not only the local dialog.
5. Keep frequency low and make opt-out easy.

## Recommended default

- Bedtime check-in: 22:00 in the user's timezone.
- Idle-time check-in: only after a meaningful quiet period.
- Frequency cap: at most one proactive message per day unless the user asks for more.
- Tone: short, warm, and easy to ignore.

## Good reminder text

- `晚上好，人～ 该去睡觉啦 ( ๑-๑ )`
- `人，猫来给你说晚安了…今晚也要好好休息喔`
