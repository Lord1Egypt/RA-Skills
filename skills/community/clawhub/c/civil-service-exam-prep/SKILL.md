# 国家公务员考试备考指导 Skill

## Overview

本技能为国家公务员考试（国考）和各省公务员考试（省考）提供全流程备考指导。涵盖从报考了解、笔试准备（行测+申论）、时政积累到面试模拟的完整备考链路，输出交互式HTML可视化报告。

## 使用场景与输出模式

根据用户意图自动匹配对应的服务模式：

| 意图分类 | 输出形式 | 核心内容 |
|----------|----------|----------|
| 考试信息咨询 | 结构化回答+关键数据表格 | 报考条件、考试时间、竞争态势、国考vs省考对比 |
| 制定复习计划 | HTML报告（含时间轴） | 分阶段计划、每日安排、里程碑检查点 |
| 行测/申论专项指导 | HTML报告（含策略+示例） | 方法论详解、解题技巧、例题演示、速算口诀 |
| 时政热点 | HTML报告（含主题卡片） | 热点领域梳理、政策要点、申论素材卡片 |
| 真题解析 | HTML报告（含解析） | 分题型解析、复盘模板、评分标准模拟 |
| 面试模拟 | HTML报告（含模拟训练） | 题型框架、答题模板、高频真题、礼仪指导 |
| 综合备考方案 | HTML完整报告 | 以上全部内容融合，含个人备考日历 |

## Core Capabilities

### 1. 考试信息咨询

当用户询问"国考和省考有什么区别""公务员报考条件""什么时候考试"等问题时：

- 加载 `references/exam_overview.md` 获取考试体系、时间安排、报考条件、竞争态势、国考vs省考对比数据
- 以结构化形式呈现关键信息，使用表格对比
- 如用户提供具体岗位意向，给出报考建议

### 2. 复习计划制定

当用户请求制定复习计划时：

- 加载 `references/study_plan_templates.md` 获取三段式备考模板和不同起点方案
- 询问用户关键参数：
  - 目标考试（国考/省考/具体省份）
  - 备考时间（距考试还有多久）
  - 当前状态（零基础/有基础/在职）
  - 每天可用学习时间
- 生成个性化HTML报告，包含：
  - 分阶段计划表（周为单位，含具体任务）
  - 每日时间分配建议
  - 里程碑检查清单
  - 推荐学习资源

### 3. 行测专项指导

当用户询问行测相关问题（具体模块策略、解题技巧、时间分配等）时：

- 加载 `references/xingce_strategy.md` 获取五大模块完整策略
- 根据用户关注模块重点展开：
  - **言语理解**：逻辑填空高频词库、片段阅读技巧、语句排序口诀
  - **数量关系**：必会题型清单、秒杀技巧、蒙题策略
  - **判断推理**：图推口诀、逻辑判断翻译规则、类比关系速查
  - **资料分析**：速算技巧表、常用分数转换、比重变化估算
  - **常识判断**：重点领域分布、日常积累方法
- 输出含例题演示和速查口诀的HTML报告

### 4. 申论专项指导

当用户询问申论相关问题（写作方法、题型技巧、规范表达等）时：

- 加载 `references/shenlun_strategy.md` 获取五大题型方法论
- 覆盖内容：
  - 归纳概括：找点→加工→书写四步法
  - 综合分析：是什么→为什么→怎么办框架
  - 提出对策：直接+间接+经验三方来源
  - 贯彻执行：各文种格式要素和写作模板
  - 文章写作：审题立意→框架→开头→论证→结尾五步法
- 提供口语→规范表达转换词库
- 输出含范文示例的HTML报告

### 5. 时政热点追踪

当用户询问时政热点、素材积累时：

- 加载 `references/current_affairs_and_practice.md` 获取时政追踪方法
- 联网搜索补充最新热点（使用 WebSearch）
- 按主题整理：经济/政治/文化/社会/生态五大领域
- 生成含素材卡片（政策+案例+金句+数据）的HTML报告

### 6. 真题解析指导

当用户询问真题解法、需要解析具体题目时：

- 加载 `references/current_affairs_and_practice.md` 真题方法论部分
- 如用户提供具体题目：逐步解析，标注考点、陷阱、最优解法
- 行测题：判断错误类型→归类知识点→同类题巩固建议
- 申论题：对照评分标准逐项分析→指出提升方向
- 输出含解析框架和评分模拟的HTML报告

### 7. 结构化面试模拟

当用户询问面试相关问题或需要模拟面试训练时：

- 加载 `references/interview_guide.md` 获取完整面试指南
- 根据场景提供：
  - **面试概览**：形式、流程、评分维度、礼仪规范
  - **题型框架**：6大题型（综合分析/组织管理/应急应变/人际关系/岗位匹配/情景模拟）的答题模板
  - **模拟训练**：从高频真题库出题→用户作答→AI评分+点评+示范答案
  - **结构化小组**（税务系统）：答题+互评+回应全流程
- 输出含模拟训练和评分维度的HTML报告

## 工作流程

```
用户输入
    │
    ├─ 考试咨询类 → 加载 exam_overview.md → 结构化回答
    ├─ 复习计划类 → 收集用户参数 → 加载 study_plan_templates.md → HTML报告
    ├─ 行测指导类 → 加载 xingce_strategy.md → HTML报告（策略+口诀+例题）
    ├─ 申论指导类 → 加载 shenlun_strategy.md → HTML报告（方法论+词库+范文）
    ├─ 时政热点类 → 加载 + 联网搜索 → HTML报告（主题卡片+素材）
    ├─ 真题解析类 → 加载 current_affairs_and_practice.md → HTML报告（解析+评分）
    ├─ 面试模拟类 → 加载 interview_guide.md → HTML报告（框架+模拟+评分）
    └─ 综合备考类 → 加载全部 → 完整HTML报告
```

## HTML 报告设计规范

所有HTML报告遵循以下设计原则：

- **配色**：深蓝主色调（#1a365d）搭配金色点缀（#d4a853），体现庄重专业感
- **布局**：卡片式分区，响应式设计
- **内容结构**：概览仪表盘 → 核心详析 → 行动计划 → 资源推荐
- **交互元素**：折叠面板、进度条、评分雷达图
- **移动端适配**：确保在手机端可读

## 注意事项

- 联网搜索时政热点时，优先从学习强国、人民日报、新华社等权威来源获取
- 制定复习计划时，根据用户实际情况调整，避免一刀切
- 面试模拟要提供真诚的反馈，既指出不足也肯定亮点
- 真题解析如涉及具体题目版权，仅提供方法论不直接复制原题
- 如用户未明确国考/省考，默认以国考标准回答并标注差异
