## Description: <br>
Convert Windows paths to WSL paths for file operations in WSL environment <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to convert Windows-style file paths into WSL-accessible paths before reading files, accessing project resources, or working with images and other assets in WSL. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Converted paths may be incorrect if Windows-style paths appear inside code snippets, logs, quoted text, or other surrounding content. <br>
Mitigation: Double-check converted paths before using them for file operations. <br>
