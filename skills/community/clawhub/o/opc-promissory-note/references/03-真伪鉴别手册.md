# 真伪鉴别手册

> 物理鉴别四招、仪器检测、数字化验证的完整指南

## 一、物理鉴别四招

### 1.1 纸张鉴别

| 鉴别项 | 真本票 | 假本票 |
|--------|--------|--------|
| **纸张材质** | 专用防伪纸张 | 普通印刷纸张 |
| **纸张手感** | 挺括、厚度均匀 | 较薄、质地松软 |
| **水印** | 多层次水印图案 | 无水印或印刷水印 |
| **安全线** | 金属安全线或荧光线 | 无安全线或假安全线 |
| **纤维丝** | 随机分布的彩色纤维 | 无纤维或印刷纤维 |

**鉴别方法**：
1. 手感厚度：真票手感挺括，假票偏软
2. 透光检查：查看水印图案
3. 触感检查：触摸安全线位置
4. 纤维检查：在紫外灯下查看荧光纤维

### 1.2 油墨鉴别

| 鉴别项 | 真本票 | 假本票 |
|--------|--------|--------|
| **油墨配方** | 银行专用配方，保密 | 商业油墨替代 |
| **颜色标准** | 符合银行标准色 | 颜色偏差 |
| **印刷精度** | 精细、清晰 | 模糊、重影 |
| **光变油墨** | 有特定角度变色的油墨 | 无或效果差 |
| **荧光油墨** | 有特定紫外荧光 | 无或异常 |

**鉴别方法**：
1. 颜色对比：与已知真票对比
2. 触摸检查：真票油墨凸起感
3. 角度观察：光变油墨随角度变色
4. 放大检查：使用放大镜查看印刷细节

### 1.3 号码鉴别

| 鉴别项 | 真本票 | 假本票 |
|--------|--------|--------|
| **号码字体** | 统一规范 | 不一致 |
| **排列顺序** | 整齐规范 | 歪斜错位 |
| **间隔距离** | 均匀一致 | 不均匀 |
| **凹凸感** | 触摸有凹凸感 | 平滑无感 |
| **编号规则** | 符合银行编号规则 | 格式错误 |

**鉴别方法**：
1. 目视检查：号码排列是否整齐
2. 触摸检查：是否有凹凸感
3. 规则核对：编号是否符合规范格式
4. 流水号查询：可向银行核实

### 1.4 签字鉴别

| 鉴别项 | 真本票 | 假本票 |
|--------|--------|--------|
| **笔迹流畅** | 自然流畅 | 生硬呆板 |
| **签字位置** | 正确 | 位置偏差 |
| **与印鉴一致** | 印鉴一致 | 不一致 |
| **授权核实** | 与印鉴册规定一致 | 越权签字 |
| **整体协调** | 协调自然 | 不协调 |

**鉴别方法**：
1. 笔迹分析：运笔是否自然流畅
2. 印鉴核对：签字与印鉴是否一致
3. 授权核实：签字人是否有权签字
4. 样本对比：与银行预留签字样本对比

## 二、仪器检测方法

### 2.1 紫外线荧光检测

**设备**：紫外灯（365nm或254nm）

**检测内容**：
| 检测项 | 真票反应 | 假票反应 |
|--------|----------|----------|
| 纸张荧光 | 无荧光或特定荧光 | 漂白纸张发蓝白荧光 |
| 水印荧光 | 特定颜色荧光 | 无或异常 |
| 安全线荧光 | 特定颜色发光 | 无或印刷替代 |
| 纤维丝 | 彩色荧光纤维 | 无荧光纤维 |

### 2.2 红外线扫描检测

**设备**：红外扫描仪

**检测原理**：
- 不同油墨对红外线的吸收特性不同
- 可识别被涂改的票面信息

**应用场景**：
- 检测票面文字是否被篡改
- 识别不同批次的油墨

### 2.3 磁性检测

**设备**：磁性检测仪

**检测内容**：
| 检测项 | 真票 | 假票 |
|--------|------|------|
| 磁性油墨 | 有特定磁性信号 | 无磁性或异常 |
| 磁条位置 | 符合规范 | 位置错误 |
| 磁性强度 | 标准范围内 | 异常 |

### 2.4 RFID芯片检测

**设备**：RFID读卡器

**检测内容**：
- 芯片信息与票面信息交叉验证
- 芯片唯一序列号核实
- 芯片加密等级检查

**适用情况**：仅限带有RFID芯片的新型票据

## 三、数字化验证方法

### 3.1 二维码验证

**操作步骤**：
1. 使用银行官方APP扫描票面二维码
2. 获取票据电子信息
3. 与票面信息交叉验证

**验证内容**：
| 验证项 | 一致性检查 |
|--------|------------|
| 票据号码 | 必须一致 |
| 金额 | 必须一致 |
| 出票日期 | 必须一致 |
| 收款人 | 必须一致 |
| 有效期 | 在有效期内 |

### 3.2 银行查询系统验证

**操作步骤**：
1. 登录开证行官方网站
2. 进入票据验证页面
3. 输入票据号码和验证码
4. 获取验证结果

**验证内容**：
| 验证项 | 说明 |
|--------|------|
| 票据状态 | 有效/已兑付/已挂失 |
| 出票信息 | 是否真实存在 |
| 金额信息 | 是否与票面一致 |

### 3.3 SWIFT核实

**报文类型**：MT799（自由格式查询）

**核实流程**：
```
1. 通过开户行发起SWIFT查询
2. 填写票据信息（号码、金额、开证行）
3. 等待开证行加押回复
4. 根据回复确认真实性
```

**回复类型**：
| 回复类型 | 含义 |
|----------|------|
| 加押确认 | 票据真实，银行承认 |
| 加押否认 | 票据不存在或伪造 |
| 无法确认 | 需进一步核实 |

### 3.4 电话核实

**核实步骤**：
1. 通过银行官方电话核实
2. 提供票据号码
3. 核实出票信息
4. 确认票据状态

**注意事项**：
- 使用银行官网公布的电话
- 不要使用对方提供的电话
- 记录核实过程

## 四、要素审核要点

### 4.1 金额审核

| 审核点 | 标准 | 异常情况 |
|--------|------|----------|
| 大写金额 | 中文大写规范（壹、贰、叁等） | 简化字、错别字 |
| 数码金额 | 阿拉伯数字清晰 | 模糊、被篡改 |
| 一致性 | 大写与数码完全一致 | 不一致 |
| 小数点 | 规范使用 | 多位小数、位置错误 |

### 4.2 日期审核

| 审核点 | 标准 | 异常情况 |
|--------|------|----------|
| 完整性 | 年、月、日完整 | 缺少年或日 |
| 合理性 | 票据日期合理 | 未来日期 |
| 连续性 | 日期与票据流程一致 | 日期矛盾 |

### 4.3 签章审核

| 审核点 | 标准 | 异常情况 |
|--------|------|----------|
| 印鉴清晰 | 印鉴完整清晰 | 模糊、缺角 |
| 一致性 | 与预留印鉴一致 | 不一致 |
| 签字权限 | 与印鉴册规定一致 | 越权签字 |
| 位置正确 | 签章位置规范 | 位置错误 |

### 4.4 密押核对

| 审核点 | 标准 | 异常情况 |
|--------|------|----------|
| 密押格式 | 符合SWIFT规范 | 格式错误 |
| 密押一致性 | 与预留密押一致 | 不一致 |
| 有效期 | 在密押有效期内 | 密押过期 |
