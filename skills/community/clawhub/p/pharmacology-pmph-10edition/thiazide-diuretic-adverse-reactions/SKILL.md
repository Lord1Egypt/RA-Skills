# 氢氯噻嗪不良反应识别与监测

## 适用场景
- 患者长期（>2周）或大剂量（≥50 mg/日）使用氢氯噻嗪
- 患者合并糖尿病、高尿酸血症、肝硬化或肾功能不全
- 出现乏力、心律失常、痛风发作、意识模糊或黄疸等症状

## 执行步骤
1. **评估电解质紊乱风险**：
   - 监测血钾、钠、氯水平；警惕低钾血症（<3.5 mmol/L）、低钠血症（<135 mmol/L）及低氯性碱中毒
   - 若出现心律失常或肌无力，优先排查低钾

2. **监测代谢指标**：
   - 定期检测空腹血糖、血脂谱（总胆固醇、甘油三酯、HDL、LDL）及血尿酸
   - 血糖升高或新发糖耐量异常提示胰岛素抵抗
   - 血尿酸 >420 μmol/L（男）或 >360 μmol/L（女）需警惕痛风

3. **评估器官功能影响**：
   - 肾功能：监测血肌酐、尿素氮；脱水可致肾小球滤过率下降
   - 肝功能：肝病患者注意血氨水平，防范肝性脑病

4. **识别罕见但严重反应**：
   - 过敏反应（皮疹、光敏感）、血液系统异常（白细胞减少、血小板减少性紫癜）
   - 胆囊炎、胰腺炎、性功能减退或色觉障碍

5. **干预建议**：
   - 调整剂量或联用保钾利尿剂（如螺内酯）纠正低钾
   - 痛风急性发作期禁用，高尿酸血症者慎用
   - 严重肾功能不全（eGFR <30 mL/min）避免使用

> **输出结果**：提示需定期监测电解质、肾功能、血糖、血脂及尿酸，并警惕心律失常、痛风发作等并发症。
