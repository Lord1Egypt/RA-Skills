## Known Risks and Mitigations: <br>
Risk: Provider API keys may be exposed if copied into source files or shared scripts. <br>
Mitigation: Keep real provider API keys in environment variables or a secret store, and avoid committing credentials. <br>
Risk: Prompts or outputs sent to external model providers may include sensitive information. <br>
Mitigation: Review provider data-handling policies before sending sensitive prompts or production data. <br>
