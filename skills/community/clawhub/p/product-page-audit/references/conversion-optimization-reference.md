## Scoring Rubric

### Composite Score Calculation
- Image Score (weight 25%): Average of image sub-criteria
- Copy Score (weight 25%): Average of title, bullets, description sub-criteria
- SEO Score (weight 15%): Average of technical SEO element checks
- Mobile Score (weight 20%): Average of mobile rendering and performance checks
- Trust Score (weight 15%): Average of trust signal presence and placement

### Grade Boundaries
| Grade | Score Range | Interpretation |
|---|---|---|
| A+ | 95-100 | Exceptional — industry-leading page |
| A | 90-94 | Excellent — minor refinements only |
| B+ | 85-89 | Very good — few targeted improvements needed |
| B | 75-84 | Good — clear optimization opportunities exist |
| C+ | 65-74 | Average — multiple dimensions need attention |
| C | 60-64 | Below average — significant gaps present |
| D | 40-59 | Poor — major overhaul recommended |
| F | 0-39 | Critical — fundamental elements missing |

## Page Speed Benchmarks

### Mobile Load Time Targets
| Metric | Good | Needs Work | Poor |
|---|---|---|---|
| Largest Contentful Paint (LCP) | ≤2.5s | 2.5-4.0s | >4.0s |
| First Input Delay (FID) | ≤100ms | 100-300ms | >300ms |
| Cumulative Layout Shift (CLS) | ≤0.1 | 0.1-0.25 | >0.25 |
| Time to Interactive (TTI) | ≤3.8s | 3.8-7.3s | >7.3s |
| Total page weight | ≤2MB | 2-5MB | >5MB |

### Impact of Load Time on Conversion
| Load Time | Relative CVR Impact |
|---|---|
| 1-2 seconds | Baseline (optimal) |
| 2-3 seconds | -7% to -12% |
| 3-5 seconds | -15% to -25% |
| 5-7 seconds | -30% to -45% |
| 7+ seconds | -50%+ (most users bounce) |
