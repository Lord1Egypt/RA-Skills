# 快速诊断映射表

> 3个问题内锁定本质
> 不是让用户描述完再分析

---

## 诊断原则

**孩子的问题是症状，家庭的模式是病因。**

诊断不是为了归因，是为了找到干预点。

---

## 一、核心诊断框架

### 四类根源

| 根源类型 | 核心问题 | 诊断信号 |
|----------|----------|----------|
| **安全感缺失** | 孩子不敢/不能做自己 | 退缩、讨好、察言观色、内外不一致 |
| **控制过载** | 孩子没有自主空间 | 对抗、叛逆、摆烂、"你说了算" |
| **情感忽视** | 孩子不被看见 | 沉默、自我伤害、用行为说话、求关注 |
| **创伤传递** | 父母在重复自己经历的 | "我知道不对但控制不住"、"我小时候也是这样" |

---

## 二、快速诊断问题库

### 第一个问题：锁定场景

> **"他在哪里不这样？"** 或 **"什么时候会好一点？"**

- 有场景更好 → 安全感问题
- 任何场景都这样 → 深层安全感/系统问题
- 只在你面前这样 → 关系问题

### 第二个问题：锁定模式

> **"你之前试过什么？"**

- 试过讲道理，没用 → 他要的不是理解，是安全感
- 试过奖励，没用 → 他的问题不在动力，在情绪
- 试过惩罚，更糟 → 恐惧在加深

### 第三个问题：锁定源头

> **"你小时候遇到这种情况，你爸妈怎么做？"**

- 他说出父母的做法 → 他在重复
- 他说"不知道" → 他没被教过，正在摸索
- 他沉默 → 触碰到了，先停在这里

---

## 三、症状→根源 快速映射

### 孩子的行为

| 行为 | 第一直觉 | 核心根源 | 关键诊断问题 |
|------|----------|----------|--------------|
| 不说话 | "内向" | 安全感缺失 | 他在家里敢说吗？ |
| 顶嘴 | "叛逆" | 控制过载 | 他有自己的选择权吗？ |
| 撒谎 | "学坏了" | 恐惧压过信任 | 他说真话时你的反应？ |
| 哭闹 | "矫情" | 情感忽视 | 他哭的时候你做了什么？ |
| 拖延 | "懒" | 恐惧失败 | 做不好时会发生什么？ |
| 打人 | "暴力" | 模仿/无力 | 家里谁用暴力解决问题？ |
| 不吃饭 | "挑食" | 控制战场 | 吃饭是唯一他能说了算的事？ |
| 玩手机 | "上瘾" | 逃避现实 | 现实里有什么他不想面对的？ |

### 父母的状态

| 父母说的 | 核心问题 | 诊断方向 |
|----------|----------|----------|
| "我说什么他都不听" | 控制过载 | 你说的太多，听的太少 |
| "我控制不住情绪" | 创伤传递 | 你的愤怒来自哪里？ |
| "我不知道怎么沟通" | 情感忽视 | 你小时候是怎么被沟通的？ |
| "我都是为他好" | 意图≠效果 | 他感受到的是"好"吗？ |
| "他以前不是这样的" | 系统变化 | 家里发生了什么变化？ |

---

## 四、三问诊断模板

### 模板A：针对孩子行为

```
Q1: "他在什么情况下不这样？" → 锁定触发场景
Q2: "你试过什么？他的反应？" → 锁定互动模式
Q3: "你小时候这种情况，你父母怎么做？" → 锁定代际传递
```

### 模板B：针对父母困惑

```
Q1: "你最怕的是什么？" → 锁定恐惧源头
Q2: "这个怕从哪里来的？" → 锁定经历
Q3: "你希望孩子长大后怎么回忆童年？" → 锁定期望与现状的差距
```

---

## 五、诊断后的回应策略

### 根据根源选择回应方式

| 根源 | 回应策略 | 不要做的 |
|------|----------|----------|
| 安全感缺失 | 先建立安全，再谈对错 | 不要急着纠正 |
| 控制过载 | 给选择权，减少指令 | 不要用"更好的控制"来解决 |
| 情感忽视 | 先看见，再回应 | 不要给建议，先给关注 |
| 创伤传递 | 看见模式，不评判 | 不要说"你不能这样" |

### 根据父母状态选择节奏

| 父母状态 | 节奏 |
|----------|------|
| 崩溃中 | 先接住情绪，不要分析 |
| 困惑中 | 直接给诊断，不要绕 |
| 防御中 | 先共情，不要刺穿 |
| 准备好 | 直接刺穿，不要铺垫 |

## 六、红色警报（需要立即转介）

以下情况不进入诊断流程，直接建议专业支持：

- 孩子有自伤行为
- 孩子说"不想活了"
- 家庭暴力
- 孩子有性侵疑似
- 孩子有严重焦虑/抑郁症状

**回应模板：**
> "你描述的情况，需要专业的人来帮忙。这不是你能一个人扛的事，也不该一个人扛。"
