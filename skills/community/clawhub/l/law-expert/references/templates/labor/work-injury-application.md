# 工伤认定申请指引

## 适用场景
劳动者在工作时间、工作场所因工作原因受伤，或患职业病后，用人单位/劳动者向人社部门申请工伤认定，确认工伤性质后获得工伤保险待遇。

## 提交机构
用人单位所在地的人力资源和社会保障局（工伤认定部门）

## 模板正文

```
                    工伤认定申请指引

一、申请主体及时限
□ 用人单位申请：事故伤害发生之日/诊断鉴定为职业病之日起
                 30日内申请；
□ 劳动者/近亲属申请：用人单位不申请的，劳动者或近亲属
                       可在1年内自行申请。

二、可认定为工伤的情形（《工伤保险条例》第14条）
□ 在工作时间和工作场所内，因工作原因受到事故伤害的；
□ 工作时间前后在工作场所内，从事与工作有关的预备性
  或收尾性工作受到事故伤害的；
□ 在工作时间和工作场所内，因履行工作职责
  受到暴力等意外伤害的；
□ 患职业病的；
□ 因工外出期间，由于工作原因受到伤害或发生事故下落不明的；
□ 在上下班途中，受到非本人主要责任的交通事故
  或城市轨道交通、客运轮渡、火车事故伤害的；
□ 法律规定的其他应当认定为工伤的情形。

三、视同工伤的情形（《工伤保险条例》第15条）
□ 在工作时间和工作岗位，突发疾病死亡或在48小时内
  经抢救无效死亡的；
□ 在抢险救灾等维护国家利益、公共利益活动中受到伤害的；
□ 职工原在军队服役，因战、因公负伤致残，已取得革命伤残
  军人证，到用人单位后旧伤复发的。

四、申请材料清单
1. 工伤认定申请表（人社部门标准表格）；
2. 与用人单位存在劳动关系的证明材料（劳动合同/社保记录/
   工资流水/工作证/考勤记录/同事证言等）；
3. 医疗机构出具的受伤后诊断证明书（病历/出院小结）；
4. 事故情况说明（时间、地点、经过、原因、伤情）；
5. 目击证人证言（含证人身份信息、联系方式和证言内容）；
6. 如为上下班途中交通事故，需提供交通事故认定书、
   路线图、上下班时间证明；
7. 申请人身份证明复印件。

五、工伤认定申请表关键填写提示
事故时间：______年______月______日______时______分
事故地点：________________________
受伤部位及程度：________________________
事故经过简述：______________________________（200字以内）
受伤害职工签名（本人无法签名的按手印）

六、流程与时限
1. 提交申请材料→人社部门受理/补正通知
2. 受理后60日内作出认定决定（事实清楚的15日内）
3. 领取《工伤认定决定书》
4. 认定为工伤的→进行劳动能力鉴定→享受工伤保险待遇

七、注意事项
- 用人单位不为劳动者申请的不影响劳动者自行申请；
- 劳动关系不明确的，需先通过劳动仲裁确认劳动关系；
- 对认定结论不服的，可申请行政复议或提起行政诉讼；
- 用人单位未参加工伤保险的，由用人单位按标准支付待遇；
- 工伤认定期间注意保留所有医疗票据和相关文件。

法律依据：《工伤保险条例》《工伤认定办法》
```

## 注意事项
- **关键填写点**：事故时间和地点要精确（关系到是否在"工作时间+工作场所"）；诊断证明是核心材料，医院等级无硬性要求；劳动关系证明材料最关键（无合同的可用工资流水、社保等替代）
- **常见错误**：错过申请时限（单位30日/个人1年内）；未及时保留事故现场证据（拍照、录像、证人联系方式）；误认为"上下班途中所有事故都是工伤"（仅限非本人主要责任的交通事故等）
- **份数要求**：正本1份 + 全套材料1份；建议自留完整复印件
- **认定后的程序**：工伤认定→劳动能力鉴定（1-10级）→工伤保险待遇领取/工伤赔偿
- **未参保情形**：单位未缴工伤保险的，全部工伤待遇由单位承担（比有保险时更重的后果）
