## Description: <br>
干系人沟通，针对开发/产品/管理层不同角色定制测试沟通内容和汇报方式。当需要与不同角色沟通测试进展或推动问题时激活。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
QA practitioners use this skill to tailor stakeholder communication for developers, product managers, and management. It helps transform bug reports, release assessments, and quality metrics into audience-specific communication content. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may activate on fairly general communication requests. <br>
Mitigation: Confirm the request is about QA stakeholder communication before applying its templates. <br>
Risk: The skill allows read-only file access for context gathering. <br>
Mitigation: Limit file reads to relevant bug reports, release assessments, and quality metrics intended for the communication task. <br>
