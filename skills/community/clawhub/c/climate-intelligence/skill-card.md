## Description: <br>
Provides multi-domain climate data analysis across emissions, renewable energy, carbon markets, policy, physical risk, finance, innovation, attribution, and net-zero progress. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>

## Use Case: <br>
Developers, analysts, and climate-focused teams use this skill to structure climate intelligence requests, route them to relevant public sources, compare evidence across domains, and present uncertainty-aware findings. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Server security evidence marked the release suspicious despite finding no explicit malicious artifacts. <br>
Mitigation: Install only if the publisher is trusted and the skill is needed for the intended climate intelligence workflow. <br>
Risk: The skill can produce policy, market, risk, and projection guidance that may become stale or carry uncertainty. <br>
Mitigation: Require data vintage, source URLs, scenario assumptions, and confidence ranges before using outputs for decisions. <br>
Risk: Security guidance warns to review commands before execution when using related maintainer workflows. <br>
Mitigation: Review any generated command or workflow step before running it, especially steps that could change public content or account state. <br>
