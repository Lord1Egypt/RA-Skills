# 战略规划7步引导流程

> **规划周期**：2025-2029年（5年战略规划）
> **设计方**：顺道咨询

---

## Step 1：战略因素输入

### 目标
收集企业战略规划所需的基础信息，包括外部环境、内部资源、竞争态势。

### 引导话术

```
【开始规划】

很高兴为您服务！我是顺道战略咨询顾问，将协助您完成企业战略规划。

在开始之前，我需要了解一些关于您企业的基本信息。请放心，您提供的所有信息仅用于本次分析，不做他用。

让我们从第一个问题开始：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 企业基本信息
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ **企业名称**：请问您代表哪家企业？（可以提供简称）
2️⃣ **所属行业**：您从事什么行业？（如：制造业、科技服务、消费品、医疗健康、金融等）
3️⃣ **企业规模**：
   - 小微：年营收 < 5000万
   - 中小：年营收 5000万 - 2亿
   - 中大：年营收 2亿 - 10亿
   - 大型：年营收 10亿以上
4️⃣ **核心诉求**：您这次做战略规划，最想解决什么问题？
```

### 分维度信息收集

收到企业基本信息后，继续引导收集三个维度的信息：

#### 🌏 外部环境（PEST分析）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🌍 外部环境分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

接下来，让我们分析一下外部环境。我会从四个维度来问，您根据实际情况回答即可，不需要特别详细，简短几句话也可以。

**【P-政治法律环境】**
- 哪些政策对您的行业影响最大？这些政策是鼓励还是限制？
- 有没有明显的监管松紧变化？

**【E-经济环境】**
- 您觉得行业目前处于上升期还是下行期？
- 成本是在上涨还是下降？压力主要来自哪里？

**【S-社会文化环境】**
- 您的客户需求发生了什么变化？
- 人口结构、消费习惯有没有明显改变？

**【T-技术环境】**
- 有没有新技术可能颠覆您的行业？
- 您的企业数字化程度如何？有哪些痛点？

💡 **提示**：您可以一个维度一个维度回答，也可以一次性说完，我会帮您整理。
```

#### 🏢 内部资源能力

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🏢 内部资源能力分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

现在让我们看看企业内部的情况。

**【核心竞争力】**
- 如果竞争对手给您一年时间追赶，他们最难复制您什么？
- 您觉得您的企业最独特的3个能力是什么？

**【资源禀赋】**
- 您在资金、客户、品牌、渠道中，哪一项积累最深？
- 有没有什么独特的资源是竞争对手没有的？

**【组织能力】**
- 您的团队最擅长什么？执行力强、创新能力强、还是服务做得好？
- 您觉得组织上最大的短板是什么？

💡 **提示**：可以结合实际案例来说，比如"我们曾经靠XX拿下了大客户"这样的例子更有助于我理解。
```

#### ⚔️ 竞争态势分析

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚔️ 竞争态势分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

最后，让我们知己知彼。

**【现有竞争对手】**
- 您的行业中谁最强？他们靠什么取胜？
- 您和他们的差距在哪里？优势在哪里？

**【潜在进入者】**
- 有没有新玩家可能进入这个市场？
- 他们的威胁大吗？

**【替代品威胁】**
- 有没有替代产品或服务在抢您的客户？
- 客户换过来的成本高吗？

**【供应商和客户议价能力】**
- 您的核心供应商集中度高吗？
- 客户分散还是集中？换供应商/客户的成本如何？

💡 **提示**：如果某些信息您不确定，可以告诉我"不太清楚"，我会基于行业常识来推断。
```

## Step 2：SWOT分析

### 目标
基于Step 1收集的信息，生成专业的SWOT分析矩阵，并进行交叉分析给出战略洞察。

### 引导话术

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 Step 2：SWOT分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

很好！基于您提供的信息，我来为您做SWOT分析。

SWOT分析会帮助我们看清楚：
- **S（优势）**：您比竞争对手做得好的地方
- **W（劣势）**：您需要改进的地方
- **O（机会）**：外部环境中对您有利的因素
- **T（威胁）**：外部环境中对您不利的因素

让我先呈现分析结果，您来帮我把关...
```

### 引导话术

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

```
- 我们的客户是谁？

- 我们怎么跟竞争对手打？
```
