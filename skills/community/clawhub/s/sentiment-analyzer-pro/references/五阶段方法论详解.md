# 五阶段方法论详解

> Duncan 五阶段舆情分析框架，基于《民航舆情管理》专著理论体系提炼

## 方法论起源

Duncan 在长期民航舆情管理实践中，发现舆情演进存在可预测的阶段性规律。将舆论生命周期划分为五个阶段，每个阶段具有不同的动力机制、传播特征和应对策略。

## 五个阶段详解

### 阶段1：发生期（Occurrence）

**定义**：事件触发，信号微弱，仅局部知晓

**核心特征**：
- 事件初发，尚未被广泛关注
- 仅有少量媒体报道或社交讨论
- 百度/微信指数 < 1000
- 未上热搜，无算法推荐

**传播机制**：
- 信源单一（往往来自当事人或单一媒体）
- 传播路径线性，无交叉扩散
- 平台算法尚未识别为"热门内容"

**应对策略**：
- 快速核实事实，确认事件真实性
- 评估事件性质和潜在影响范围
- 准备基线回应口径，不主动扩散
- 建立监测机制，跟踪传播动态

**关键指标**：
| 指标 | 阈值 |
|------|------|
| 百度/微信指数 | < 1000 |
| 微博讨论量 | < 100 条 |
| 媒体报道数 | < 5 篇 |
| 热搜排名 | 未上榜 |

---

### 阶段2：爆发期（Outbreak）

**定义**：舆论发酵，传播加速，开始破圈

**核心特征**：
- 媒体跟进报道，大V/博主开始转发
- 指数快速攀升，突破 5000
- 登上热搜（通常排名 10-30）
- 社交平台出现话题标签

**传播机制**：
- 关键节点转发（媒体账号、行业KOL）
- 算法识别热门信号，开始推荐扩散
- 平台话题页建立，讨论聚合

**应对策略**：
- 发布第一份官方声明（承认关注+核实中）
- 明确回应主体和联络渠道
- 统一内部口径，避免员工/部门各自发声
- 同步启动内部调查和整改措施

**关键指标**：
| 指标 | 阈值 |
|------|------|
| 百度/微信指数 | 5000-10000 |
| 微博讨论量 | 100-1000 条 |
| 媒体报道数 | 5-20 篇 |
| 热搜排名 | 10-30 名 |

---

### 阶段3：高潮期（Climax）

**定义**：舆论峰值，全网讨论，各方密集回应

**核心特征**：
- 指数突破 10000，热搜排名 < 5
- 主流媒体介入，形成主流叙事
- 公众情绪极化（愤怒/质疑/支持分化）
- 各方密集回应（涉事方、监管、第三方）

**传播机制**：
- 主流媒体定调，形成舆论框架
- 情绪传染加速，理性讨论减少
- 跨平台联动（微博→微信→抖音→小红书）

**应对策略**：
- 发布正式回应/通报（态度端正+事实清晰+措施具体）
- 高层出面发声（增加回应权重）
- 主动披露调查进展和整改进度
- 借助第三方权威背书（专家/协会/媒体）

**关键指标**：
| 指标 | 阈值 |
|------|------|
| 百度/微信指数 | > 10000 |
| 微博讨论量 | > 1000 条 |
| 媒体报道数 | > 20 篇 |
| 热搜排名 | 前 5 名 |

---

### 阶段4：缓和期（Mitigation）

**定义**：热度回落，新热点出现，舆论转向

**核心特征**：
- 讨论量下降 > 50%，指数回落
- 热搜退出前十，新话题占据注意力
- 涉事方发布跟进措施/整改通报
- 深度报道/复盘文章出现

**传播机制**：
- 新热点出现，公众注意力自然转移
- 算法推荐权重下降，话题流量枯竭
- 媒体转向新议题，报道频率降低

**应对策略**：
- 发布整改进展通报（具体动作+时间节点+可验证结果）
- 主动邀请媒体/公众监督整改落实
- 启动内部复盘，形成改进闭环
- 准备恢复性传播（正面内容填充搜索结果）

**关键指标**：
| 指标 | 阈值 |
|------|------|
| 百度/微信指数 | 下降 > 50% |
| 微博讨论量 | 下降 > 50% |
| 热搜排名 | 退出前十 |
| 新热点竞争 | 出现 1+ 个替代话题 |

---

### 阶段5：降温期（Cooling）

**定义**：舆论消散，改进措施落地，形成闭环

**核心特征**：
- 指数 < 1000，热搜完全消失
- 七日内无新增讨论
- 整改/问责措施落地并公示
- 公众注意力完全转移

**传播机制**：
- 无新增触发点，话题自然沉底
- 搜索结果被正面/中性内容填充
- 长效机制建立，类似事件防范能力提升

**应对策略**：
- 发布最终整改报告/问责结果
- 主动披露制度改进和流程优化
- 开展正面传播，修复品牌形象
- 形成案例总结，纳入舆情案例库
- 更新应急预案，提升组织韧性

**关键指标**：
| 指标 | 阈值 |
|------|------|
| 百度/微信指数 | < 1000 |
| 微博讨论量 | < 50 条/日 |
| 热搜排名 | 无 |
| 七日内新增讨论 | 0 条 |

---

## 阶段间转换信号

| 转换 | 关键信号 | 决策点 |
|------|---------|--------|
| 发生→爆发 | 指数突破5000，大V转发 | 是否发布第一份声明？ |
| 爆发→高潮 | 指数>10000，主流媒体介入 | 高层是否出面回应？ |
| 高潮→缓和 | 指数下降>50%，新热点出现 | 是否发布整改通报？ |
| 缓和→降温 | 指数<1000，七日无新增 | 是否发布最终报告？ |

---

## 方法论应用要点

1. **阶段定位优先**：先判断当前处于哪个阶段，再制定应对策略
2. **信号交叉验证**：不依赖单一指标，至少3个信号共同确认阶段转换
3. **提前半步布局**：阶段转换前就开始准备下一阶段的应对动作
4. **对标历史案例**：每次分析都参考同类事件的历史数据
5. **闭环才是终点**：阶段5的整改落地才是完整舆情管理的终点
