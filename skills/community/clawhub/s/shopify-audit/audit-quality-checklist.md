# Shopify Audit Coverage Checklist

Confirms the audit itself is complete and the report meets the bar. Sections 1 and 8 are blocking.

## 1. Inputs & baseline
- [ ] Store URL and primary PDP/collection target confirmed
- [ ] Device split obtained (or assumed mobile-majority, stated)
- [ ] Top traffic source identified
- [ ] Baseline conversion / ATC / checkout numbers recorded (or "not provided" noted)
- [ ] Theme identified where possible

## 2. Homepage
- [ ] 5-second test result recorded
- [ ] Headline assessed (benefit-led, ≤8 words)
- [ ] Hero media weight checked
- [ ] Primary CTA singular and clear
- [ ] Announcement bar content assessed
- [ ] Nav depth ≤6 verified

## 3. Collection
- [ ] Card info sufficiency (price, stars, variants)
- [ ] Filters present and decisive-attribute coverage
- [ ] Sort defaults checked
- [ ] Image consistency across cards
- [ ] Empty-state handling tested
- [ ] Compare-at display on cards checked

## 4. PDP
- [ ] Image set ≥5 with lifestyle + scale + zoom
- [ ] Description benefit-led and scannable
- [ ] Stars + count visible without scroll (mobile)
- [ ] Shipping/returns near ATC
- [ ] Variant picker and size guide usability
- [ ] Sticky mobile ATC checked
- [ ] Urgency elements verified honest
- [ ] Cross-sell placement assessed

## 5. Cart & checkout
- [ ] Cost-surprise prevention (estimate/threshold in cart)
- [ ] Guest checkout enabled
- [ ] Express payments (Shop Pay/Apple/Google) enabled
- [ ] Discount-code exit risk assessed (automatic discounts during sales)
- [ ] Shipping options include a fast and a cheap lane
- [ ] Trust links in checkout footer

## 6. Performance
- [ ] PageSpeed run on homepage, top collection, top PDP (mobile)
- [ ] LCP recorded vs. <2.5s target
- [ ] Heaviest 3 assets listed with sizes
- [ ] Installed apps listed; removal/defer candidates flagged
- [ ] Image format coverage checked

## 7. Mobile pass
- [ ] Dedicated mobile pass completed (not desktop-resized)
- [ ] Thumb reach of ATC verified
- [ ] Tap targets and type legibility checked
- [ ] Sticky elements don't cover content/CTA
- [ ] Chat widget placement vs. ATC checked

## 8. Report quality
- [ ] Findings ≤15, sorted impact × effort
- [ ] Every finding: location, evidence, executable fix
- [ ] Copy findings include exact before → after rewrites
- [ ] Theme-editor paths or settings named for each fix
- [ ] No aesthetic-only findings
- [ ] Top-3 wins called out in snapshot
- [ ] Re-audit trigger and comparison metrics stated
