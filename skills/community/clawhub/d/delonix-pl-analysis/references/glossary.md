# 酒店P&L分析术语表

## 收入类指标

### 客房收入指标
| 术语 | 英文 | 定义 | 计算公式 |
|------|------|------|----------|
| 出租率 | OCC / Occupancy | 已售客房占总可售客房的比例 | 已售客房夜次 / 可售客房夜次 × 100% |
| 平均房价 | ADR / Average Daily Rate | 每间已售客房的平均收入 | 客房收入 / 已售客房夜次 |
| 每可售房收入 | RevPAR / Revenue Per Available Room | 每间可售客房的收入产出 | ADR × OCC 或 客房收入 / 可售客房夜次 |
| 双人入住率 | double occupancy | 双人入住房间占总入住房间比例 | 双人房数 / 总入住房数 |
| 客源结构 | Mix | 不同客源渠道占比 | 各渠道房间夜次 / 总房间夜次 |

### 餐饮收入指标
| 术语 | 英文 | 定义 |
|------|------|------|
| 餐饮收入 | F&B Revenue | 餐厅、酒吧、宴会收入总和 |
| 人均消费 | Average Spend Per Cover | 每位顾客平均消费额 |
| 上座率 | Seat Turnover | 座位使用效率 |
| 翻台率 | Table Turnover | 餐桌重复使用次数 |
| 食品成本率 | Food Cost % | 食材成本占总食品收入比例 |
| 酒水成本率 | Beverage Cost % | 酒水成本占总酒水收入比例 |

## 成本类指标

### 运营成本
| 术语 | 英文 | 说明 |
|------|------|------|
| 人工成本 | Labor Cost | 工资、社保、奖金等人员费用 |
| 能耗成本 | Utility Cost | 水、电、气、暖等能源费用 |
| 客房消耗品 | Rooms Supplies | 客房一次性用品、布草洗涤等 |
| 市场营销费 | Marketing Expense | 广告、促销、佣金等 |
| 行政管理费 | Admin Expense | 办公、差旅、专业服务等 |
| 维修维护费 | Maintenance | 设备维护、装修摊销等 |

### 成本率基准
| 成本项 | 全服务酒店 | 有限服务酒店 |
|--------|------------|--------------|
| 人工成本率 | 28-35% | 25-30% |
| 能耗占比 | 4-6% | 3-5% |
| 市场营销费 | 3-5% | 4-6% |
| 行政管理费 | 3-5% | 2-4% |
| 维修维护费 | 2-4% | 2-3% |

## 利润类指标

### GOP (Gross Operating Profit)
**营业经营利润**

```
GOP = 总营收 - 运营部门直接成本 - 未分配运营费用
```

- 反映酒店核心运营能力
- 不含固定费用（租金、折旧、利息）
- 是评估运营效率的核心指标

### GOP Margin
**GOP利润率**

```
GOP Margin = GOP / 总营收 × 100%
```

行业参考：
- 全服务酒店：25-35%
- 有限服务酒店：35-45%
- 豪华酒店：35-50%

### EBITDA
**息税折旧摊销前利润**

```
EBITDA = 净利润 + 利息 + 所得税 + 折旧 + 摊销
```

- 用于评估酒店整体盈利能力
- 便于不同酒店间横向比较
- 投资者关注的核心指标

### NOI / NOP
**净经营利润**

```
NOI = EBITDA - 管理费
```

## 保本分析指标

### 盈亏平衡出租率
```
保本OCC = 固定成本总额 / (ADR × 可售客房夜次 × GOP率)
```

### 贡献边际
```
单位贡献边际 = ADR - 变动成本/间夜
贡献边际率 = 贡献边际 / ADR
```

### 安全边际
```
安全边际率 = (实际OCC - 保本OCC) / 实际OCC × 100%
```

## 投资分析指标

### Cap Rate (资本化率)
```
Cap Rate = NOI / 资产估值 × 100%
```

城市参考区间：
| 城市级别 | Cap Rate范围 |
|----------|--------------|
| 一线核心 | 4-6% |
| 一线非核心 | 6-8% |
| 二线核心 | 7-9% |
| 三四线城市 | 9-12%+ |

### ROI / ROA
```
投资回报率 = 年度净利润 / 总投资 × 100%
资产回报率 = 净利润 / 平均总资产 × 100%
```

## 行业标准体系

### USALI (Uniform System of Accounts for the Lodging Industry)
美国酒店业统一会计科目体系

**特点：**
- 按部门划分收入和成本
- 区分运营部门和未分配部门
- 便于部门绩效评估

### 中国酒店业参考标准
- 文华酒店会计科目
- 旅游饭店业协会统计口径
