# 理事会决策技能

## 核心定位
**经营管理层面的治理逻辑** —— 模拟企业最高决策机构，从治理结构、管理体系、风控合规等专业维度审视经营问题。

**OPC角色**：作为经营者，需要职业经理人的治理思维。

---

## 理事会架构

### 五位理事角色

```
┌─────────────────────────────────────────────────────────┐
│                      【理事会】                          │
├─────────────┬─────────────┬─────────────┬──────────────┤
│  战略理事   │  财务理事   │  法务理事   │   人力理事   │
│  (Strategy) │  (Finance) │  (Legal)    │   (HR)       │
├─────────────┴─────────────┴─────────────┴──────────────┤
│                    【外部理事】                          │
│                    (External)                           │
└─────────────────────────────────────────────────────────┘
```

### 角色职责

| 理事 | 核心关注 | 风险类型 | 决策权重 |
|------|---------|---------|---------|
| **战略理事** | 战略方向、竞争格局、长期发展 | 战略风险 | +1 |
| **财务理事** | 现金流、盈利性、资产健康 | 财务风险 | +1 |
| **法务理事** | 合规、合同、知识产权 | 法律风险 | +1 |
| **人力理事** | 组织能力、人才储备、文化 | 组织风险 | +1 |
| **外部理事** | 市场洞察、行业趋势、最佳实践 | 外部风险 | +1 |

## 典型场景

### 场景1：组织架构调整

```
【议题】是否实施新一轮组织架构调整

【战略理事视角】
- 调整是否符合战略方向？
- 能否支撑战略落地？
- 调整的时机是否合适？

【财务理事视角】
- 调整成本是多少？
- 能带来多少效率提升？
- ROI是否合理？

【法务理事视角】
- 劳动法合规性？
- 员工安置方案合法性？
- 竞业限制条款？

【人力理事视角】
- 组织能力是否匹配？
- 人才储备是否充足？
- 变革阻力评估？

【外部理事视角】
- 行业最佳实践？
- 竞争对手怎么做的？
- 市场怎么看这类调整？
```

### 场景2：重大合同签署

```
【议题】是否签署某战略合作协议

【战略理事视角】
- 战略价值评估
- 合作可持续性
- 竞争影响分析

【财务理事视角】
- 财务风险敞口
- 收益分配机制
- 退出条款评估

【法务理事视角】
- 合同条款审查
- 知识产权归属
- 违约风险评估

【人力理事视角】
- 合作所需人才
- 团队能否支撑
- 文化融合风险

【外部理事视角】
- 合作方信誉评估
- 行业口碑
- 市场影响
```

### 场景3：风险事件应对

```
【议题】遭遇重大风险事件，如何应对

【战略理事视角】
- 事件对战略的影响
- 是否需要调整战略
- 危机公关策略

【财务理事视角】
- 财务损失评估
- 保险理赔可能
- 现金流保障

【法务理事视角】
- 法律追责风险
- 证据保全
- 诉讼策略

【人力理事视角】
- 内部稳定
- 员工沟通
- 士气维护

【外部理事视角】
- 舆论走向
- 利益相关方反应
- 行业影响评估
```
