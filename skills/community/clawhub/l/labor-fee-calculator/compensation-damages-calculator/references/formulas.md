# 计算公式说明 - 补偿金及赔偿金计算

本文件包含三大模块的详细计算规则：**未签合同赔偿金**、**离职补偿及赔偿金**、**竞业限制补偿金**。

---

## 模块一：未签书面劳动合同赔偿金

### 适用场景
- 用人单位自用工之日起超过**1个月不满1年**未与劳动者订立书面劳动合同

### 核心公式

```
二倍工资赔偿金 = 劳动者应得月工资 × 赔偿月数
赔偿月数 = 从"用工满1个月的次日"至"补签合同的前1日"（或离职日）的自然月数
```

> ⚠️ 注意：二倍工资赔偿金中的**"多付的1倍"**才是赔偿部分；正常1倍工资为应支付的劳动报酬，不是赔偿金。
>
> 法律意义上的赔偿金 = 额外多出的1倍 × 月数

### 不满整月的折算规则（司法解释二第6条）

赔偿月数不满一个月的，**按该月实际工作日折算**：
```
不满整月赔偿额 = 月工资 ÷ 当月应工作日 × 实际工作日
```
> ⚠️ SKILL 计算时若涉及不满整月情形，须在备注中注明此折算规则，提示用户核实当月实际工作日数据。

### 月工资口径
- 月工资 = 劳动者实际应得工资（包含计时/计件工资 + 奖金 + 津贴 + 补贴等货币性收入）
- **不包含**：非货币性福利、年终奖（须按约定折算）
- **下限保障**：若劳动者月工资低于当地最低工资标准，以**当地最低工资标准**计算
- 工作不满12个月：按实际工作月数计算平均工资

### 计算期间规则（重要）

| 情形 | 起算时间 | 截止时间 |
|------|---------|---------|
| 补签了合同 | 用工满1个月次日 | 补订合同的前1日 |
| 未补签，继续工作 | 用工满1个月次日 | 满1年的前1日（之后视为订立无固定期限合同） |
| 用人单位满1年仍未签 | 用工满1个月次日 | 满1年前1日（共11个月） |
| 劳动者拒绝签合同 | 无赔偿义务 | — |

### 连带规则：满1年后的无固定期限合同处理

若用工满1年仍未签书面合同：
1. 视为已订立**无固定期限劳动合同**（《劳动合同法》第14条）
2. 满1年前的11个月：每月支付2倍工资（即额外1倍赔偿）
3. 满1年当日起：若仍不签，每月继续支付2倍工资（《劳动合同法》第82条第2款）

### 仲裁时效说明
- 劳动争议申请仲裁时效为**1年**，从当事人知道或应当知道权利被侵害之日起计算
- 未签合同系持续性侵权，每月产生一次赔偿请求权，实务中存在时效起算争议

### 免责抗辩与排除情形（司法解释二第7、9条）

**用人单位可举证免付二倍工资的情形（司法解释二第7条）：**
- 因不可抗力导致未订立
- 因劳动者本人故意或重大过失未订立
- 法律、行政法规规定的其他情形

**视为无固定期限合同后不再支持二倍工资（司法解释二第9条）：**
- 满1年后视为已订立无固定期限劳动合同，劳动者再主张此后期间的二倍工资，**不予支持**
- 仅满1年前1日的11个月期间可主张

## 模块二：离职补偿及赔偿金

### 2A. 月平均工资（S）—— 所有计算的基础

```
S = 劳动合同解除/终止前12个月的应得月工资总额 ÷ 12
```

**特殊规则：**
1. **下限保障**：若 S < 当地最低工资标准 → S = 当地最低工资标准（《劳动合同法实施条例》第27条）
2. **工作不满12个月**：按实际工作的月数计算平均工资（实际月数 ≤ 12）
3. **月工资口径**：包含计时/计件工资、奖金、津贴、补贴等货币性收入；不含非货币性福利

**三倍封顶规则（高收入劳动者）：**
> 若 S > 当地上年度职工月平均工资 × 3倍
> - 经济补偿计算基数改用：社平工资 × 3（即以三倍封顶）
> - 同时：补偿年限（N）最高不超过**12年**
> - 注意：此双封顶仅适用于经济补偿金，**不适用于2倍赔偿金**

### 2B. 工龄系数（N）

```
满1年 → 计1个月
满6个月不满1年 → 计1个月
不满6个月 → 计0.5个月
```

**工龄起算规则：**
- 以本单位实际用工之日起计算
- 劳动者非因本人原因从原单位被安排到新单位工作，原单位工龄合并计算
- 原单位已付经济补偿的工龄，新单位不再重复计算
- 2008年1月1日起计算全部工龄；2008年前工龄按当时规定（双方协商）处理

### 2C. 标准经济补偿金（N）

**适用情形（《劳动合同法》第46条）：**
- 用人单位提出的协商解除（第36条，单位主动提出）
- 非过失性辞退（第40条：医疗期满、不胜任工作、情势变更）
- 经济性裁员（第41条）
- 劳动者被迫离职（第38条：未付工资、未缴社保等）
- 固定期限合同到期终止（单位未维持/提高条件，或单位不续签）
- 用人单位破产、吊销营业执照等（第44条第四、五项）

**公式：**
```
经济补偿金 = S × N
```

**双封顶计算（高收入者）：**
```
若 S > 社平工资 × 3：
  经济补偿金 = (社平工资 × 3) × min(N, 12)
```

### 2D. N+1（代通知金）

**适用情形：**
- 非过失性辞退（第40条），且用人单位选择**不提前30天书面通知**，改为支付额外1个月工资

**公式：**
```
N+1 = 上一个月实际工资 + S × N
```

> ⚠️ 代通知金的1个月按**上一个月实际工资**（《实施条例》第20条），不是S；经济补偿金按S×N

### 2E. 违法解除赔偿金（2N）

**适用情形（《劳动合同法》第87条）：**
- 用人单位**违法解除**或**违法终止**劳动合同（没有第36、39、40、41、44条规定的合法理由）
- 常见情形：无故辞退、违法裁员、怀孕期间辞退、工伤期间辞退

**公式：**
```
赔偿金（2N） = S × N × 2
```

**关键规则：**
1. 已支付赔偿金（2N）的，**不再另付经济补偿金（N）**（《实施条例》第25条）
2. 赔偿金计算年限从**用工之日起**计算，不受"2008年1月1日"限制
3. **高收入者三倍封顶不适用于2N赔偿金**（仅适用于N）
4. 劳动者可选择：要求继续履行合同，**或**要求支付2N赔偿金

### 2E补充：2N vs 继续履行的程序路径（司法解释二第16、18条）

劳动者主张违法解除时，可在以下两条路径中选择：

**路径A——要求继续履行：**
- 适用条件：劳动合同有继续履行可能（不属于第16条列明的不能履行情形）
- 待遇：用人单位支付违法解除决定作出后至劳动合同**继续履行前一日**的工资（按正常劳动工资标准）
- 注意：继续履行期间工资与2N赔偿金**不可兼得**

**路径B——放弃继续履行：**
- 直接主张违法解除赔偿金2N = S × N × 2

**"劳动合同已不能继续履行"的认定（第16条）：**
- 合同在仲裁/诉讼期间期满（无法定续订情形）
- 劳动者已开始享受基本养老保险待遇
- 用人单位被宣告破产
- 用人单位解散（合并/分立除外）
- 劳动者已与其他单位建立劳动关系且严重影响本职工作，经提出拒不改正
- 其他客观不能履行的情形

### 2F. 无需支付补偿的情形（0）

以下情形用人单位无需支付任何经济补偿或赔偿（《劳动合同法》第39条）：
- 试用期内不符合录用条件
- 严重违纪违规
- 严重失职、营私舞弊造成重大损害
- 同时与其他单位建立劳动关系且造成严重影响
- 以欺诈方式订立合同
- 被依法追究刑事责任

> ⚠️ 劳动者**主动辞职**（第37条）也不享有经济补偿，除非属于第38条的被迫离职

### 2G. 离职场景快速判断

| 离职原因 | 补偿类型 | 公式 |
|---------|---------|-----|
| 双方协商解除（单位主动提出） | 经济补偿 | N × S |
| 双方协商解除（劳动者主动提出） | 无 | 0 |
| 劳动者被迫离职（单位违法，第38条） | 经济补偿 | N × S |
| 劳动者主动辞职（第37条） | 无 | 0 |
| 单位非过失辞退，提前30天通知（第40条） | 经济补偿 | N × S |
| 单位非过失辞退，未提前通知（第40条） | 经济补偿+代通知金 | N × S + 上月工资 |
| 经济性裁员（第41条） | 经济补偿 | N × S |
| 合同到期，单位不续签或降条件续签 | 经济补偿 | N × S |
| 合同到期，劳动者不同意续签（单位维持条件） | 无 | 0 |
| 单位过失解除（第39条） | 无 | 0 |
| 单位违法解除/终止（无合法理由） | 违法解除赔偿金 | 2 × N × S |
| 单位破产/吊销（第44条第4、5项） | 经济补偿 | N × S |

## 模块三：竞业限制补偿金

### 3A. 定义与适用范围

**适用人员：**
- 高级管理人员
- 高级技术人员
- 其他负有保密义务的人员

**竞业限制期限：**
- 最长不超过**2年**（《劳动合同法》第24条）
- 期间用人单位须按月支付经济补偿

### 3A补充：在职竞业限制无需支付补偿（司法解释二第14条）

**新增规则（2025.09.01生效）：**
- 用人单位与高级管理人员/高级技术人员/其他负有保密义务人员约定**在职期间**竞业限制条款，无需支付经济补偿
- 劳动者以"在职竞业限制+未支付补偿"主张条款无效的，**不予支持**（即此类约定有效）
- **离职后的竞业限制**：补偿规则不变（须按月支付，不得低于30%下限）

> ⚠️ 若仅约定离职后竞业限制但未明确在职期间是否有竞业限制约定，实务中存在争议，SKILL计算时须在备注中提示用户核实在职期间是否有竞业义务。

### 3A-2. 竞业限制条款有效性前提（司法解释二第13条）

竞业限制条款有效的前提是劳动者**实际知悉、接触用人单位的商业秘密或知识产权相关保密事项**：
- 劳动者从未知悉/接触保密事项 → 劳动者可请求确认条款**不生效**
- 竞业限制的范围、地域、期限与劳动者实际知悉的保密事项**不相适应** → 超过合理比例的部分可被认定为**无效**

> ⚠️ SKILL 在竞业限制计算前，应在备注中提示用户确认：劳动者是否实际接触保密事项、竞业限制范围是否与职务匹配。此为条款有效性的前提条件，不影响计算公式本身。

**司法解释（一般规则）：**
```
月经济补偿 ≥ 离职前12个月平均工资 × 30%
且 ≥ 劳动合同履行地最低工资标准
```

**实务建议区间：**
- 竞业限制期限 ≤ 1年：月补偿 ≥ 月均工资 × 30%
- 竞业限制期限 > 1年（实务中不宜低于）：月补偿 ≥ 月均工资 × 50%
- 具体数额以劳动合同/保密协议约定为准，高于30%的约定有效

**月平均工资（W）的口径：**
```
W = 劳动合同解除/终止前12个月的应得月工资总额 ÷ 12
```
（含奖金、津贴、补贴等货币性收入）

**总补偿金（约定期限内全额）：**
```
竞业限制总补偿金 = 月经济补偿 × 竞业限制月数
```

### 3C. 违约金规则

**劳动者违约时：**
- 须退还已收到的竞业限制经济补偿
- 须支付约定违约金
- 违约金数额一般不超过竞业限制经济补偿总额的**5倍**（参考司法实践）

**用人单位3个月未付补偿时：**
- 劳动者可要求解除竞业限制约定（《劳动争议司法解释四》第7条）
- 无需退还已收到的补偿

### 3D. 用人单位单方解除竞业限制

- 用人单位可以单方解除竞业限制协议
- 解除时需额外支付劳动者**3个月竞业限制经济补偿**（《劳动争议司法解释四》第9条）

## 重要数据口径说明

### 月工资 vs 月均工资（S）

| 项目 | 说明 |
|------|------|
| 离职经济补偿的月工资（S） | 离职前12个月的平均应得工资，下限为当地最低工资 |
| 未签合同二倍工资的月工资 | 当月实际应得工资（逐月计算，非平均） |
| 代通知金的1个月工资 | 上一个月实际工资（非S） |
| 竞业限制月均工资（W） | 离职前12个月平均应得工资（与S口径相同） |

### 社平工资口径

**三倍封顶判断用"当地上年度职工月平均工资"，须区分以下口径：**

| 口径 | 含义 | 工资水平 | 实务适用情况 |
|------|------|---------|------------|
| **城镇非私营单位就业人员平均工资** | 机关事业单位、国有企业、外资等非私营单位职工 | ★★★ 最高 | **默认口径**——大多数省市仲裁委、法院据此判断三倍封顶 |
| **全口径城镇单位就业人员平均工资** | 城镇非私营 + 私营单位全部职工 | ★★ 居中 | 2019年起用于核定社保缴费基数上下限；广东、上海等部分省市工伤/补偿待遇可适用此口径 |
| **城镇私营单位就业人员平均工资** | 仅私营单位职工 | ★ 最低 | 极少单独使用，仅作补充参考 |

**三倍封顶规则（高收入劳动者）：**

> 若 S（月均工资）> 当地社平工资 × 3倍：
> - 计算基数改用：社平工资 × 3（即以三倍为上限）
> - 同时：补偿年限 N 最高不超过 **12年**
> - 此双封顶**仅适用于经济补偿金（N 和 N+1）**，**不适用于违法解除赔偿金（2N）**

**数据查找规则：**
1. 优先请求政策数据 API/cache 中的 `social_avg_wage`
2. 若 API 返回 `needs_web_search: true`，执行联网检索（关键词：`"[省市]YYYY年城镇非私营单位就业人员平均工资"`）
3. 如联网也无法确认，询问用户是否知晓当地数据；**不得代入其他省份数据**
