# 身体部位纹身匹配指南

本文档是纹身设计中"部位 × 风格 × 图案"匹配的完整参考。
当用户询问"纹在哪里合适"或需要身体部位推荐时加载本文档。

---

## 部位评估框架

每个身体部位从以下维度评估：

| 维度 | 说明 |
|------|------|
| **疼痛指数** | 1-10 分，1=几乎无感，10=难以忍受 |
| **变形容错率** | 该部位随体重/年龄变化的变形程度，★越多越稳定 |
| **可见度** | 日常生活可见程度和遮盖难易度 |
| **推荐风格** | 该部位最适合的纹身风格 TOP 3 |
| **推荐图案** | 适合该部位形态的图案类型 |
| **特别警告** | 该部位特有的风险和注意事项 |

---

## 各部位详解

### 1. 头部/面部

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 8-10（颅骨无肉，针直接打在骨头上的感觉） |
| 变形容错率 | ★★★★★（面部骨骼固定） |
| 可见度 | 最高，几乎无法完全遮盖 |
| 推荐风格 | Fine line、小字母、小几何（面部不宜大面积） |
| 推荐图案 | 太阳穴小符号、耳后小图案、发际线隐藏纹身 |
| 特别警告 | ⚠️ 职业风险极高（多数行业不接受面部纹身）；头皮纹身受脱发影响 |

### 2. 颈部

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 前颈 6-7、后颈 3-4 |
| 变形容错率 | ★★★★ |
| 可见度 | 高（后颈可披发遮盖，前颈难遮） |
| 推荐风格 | 后颈：Fine line、小花朵、小几何；前颈：Blackwork、字符 |
| 推荐图案 | 后颈中轴线竖构图、两侧对称小图案 |
| 特别警告 | ⚠️ 前颈纹身职场接受度低；喉结区域皮肤薄且活动量大，易晕色 |

### 3. 锁骨/胸前（上胸部）

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 锁骨 7-8（骨骼突出），胸骨 6-7 |
| 变形容错率 | 女性★★★★（有文胸影响），男性★★★★★ |
| 可见度 | 中（普通衣服可遮，低领/泳装露出） |
| 推荐风格 | 水彩花卉、写实、Fine line、几何对称 |
| 推荐图案 | 锁骨横构图（文字、花枝）、胸骨中轴线竖构图（曼陀罗、几何） |
| 特别警告 | 女性需考虑文胸肩带对图案的遮挡/摩擦 |

### 4. 上臂（大臂外侧）

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 2-3（最不痛的部位之一） |
| 变形容错率 | ★★★★★（肌肉支撑，非常稳定） |
| 可见度 | 中（短袖露出，长袖遮盖） |
| 推荐风格 | 所有风格都适合！这是"万能画布" |
| 推荐图案 | 任何图案，常见的花臂/半甲都在此区域 |
| 特别警告 | 无明显风险，注意与内侧/下臂的风格衔接 |

### 5. 前臂（小臂）

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 外侧 3-4，内侧 4-5（内侧皮肤薄） |
| 变形容错率 | ★★★★★ |
| 可见度 | 中-高（短袖半露，比大臂更常暴露） |
| 推荐风格 | Old School、水墨、Blackwork、Fineline、写实 |
| 推荐图案 | 竖构图为主，适合竹子/剑/蛇/花枝等条状图案 |
| 特别警告 | 前臂纹身已是"社会可见"纹身，谨慎选择图案 |

### 6. 手腕

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 5-6（骨骼靠近表面，皮肤薄） |
| 变形容错率 | ★★★★★ |
| 可见度 | 高（手表/手链可临时遮盖） |
| 推荐风格 | Fine line、小字符、小符号、小曼陀罗 |
| 推荐图案 | 环腕式、内侧小图案、脉搏线 |
| 特别警告 | ⚠️ 手腕频繁活动+日晒，褪色速度较快；某些职业（医护/食品）有遮盖要求 |

### 7. 手指/手背

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 手指 7-8，手背 5-6 |
| 变形容错率 | ★★★（手部随年龄老化变化明显） |
| 可见度 | 极高（几乎无法遮盖） |
| 推荐风格 | Fine line、小符号、Blackwork 指环 |
| 推荐图案 | 戒指纹身、指节字母、手背中心小图案 |
| 特别警告 | ⚠️ 手指纹身褪色极快（频繁洗手+摩擦+日晒），通常 2-3 年需补色；手指 blowout（晕色）率极高；强烈不建议首纹选择手指 |

### 8. 背部（全背 / 上背）

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 脊柱 6-7（骨骼），肩胛骨 5-6，下背 4-5 |
| 变形容错率 | ★★★★★（人体最大的稳定画布） |
| 可见度 | 低（日常完全遮盖，泳装/露背装露出） |
| 推荐风格 | 日式传统全背、写实大图、New School 满背、几何对称 |
| 推荐图案 | 满背神话场景、大型花卉、凤凰展翅、几何对称 |
| 特别警告 | 全背纹身面积巨大（通常 20-40+ 小时），需分多次完成，预算高 |

### 9. 肋骨/侧腰

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 8-9（最痛的部位之一，肋骨皮肤极薄） |
| 变形容错率 | ★★★★（体重稳定时较稳定） |
| 可见度 | 低 |
| 推荐风格 | 水墨竖构图、水彩花卉、Fine line、文字 |
| 推荐图案 | 竖排文字/诗歌、花枝、竹子、藤蔓 |
| 特别警告 | ⚠️ 体重波动大的人慎选此部位（变形明显）；呼吸时图案会随之移动，需考虑动态效果 |

### 10. 腹部

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 5-7（腹部皮肤敏感） |
| 变形容错率 | ★★（最容易受体重变化影响的部位之一） |
| 可见度 | 低 |
| 推荐风格 | 写实、New School、Blackwork |
| 推荐图案 | 对称图案、动物头像、花卉 |
| 特别警告 | ⚠️ 怀孕/体重变化会导致严重变形；此部位纹身师通常不建议做大面积 |

### 11. 下背（腰 / "Tramp Stamp"）

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 4-5 |
| 变形容错率 | ★★★ |
| 可见度 | 低（低腰裤/泳装露出） |
| 推荐风格 | 几何对称、花卉、Blackwork 对称 |
| 推荐图案 | 中心对称图案、蝴蝶、花卉横构图 |
| 特别警告 | ⚠️ 下背纹身在西方有文化污名（"tramp stamp"），在欧美建议谨慎选择；体重变化影响大 |

### 12. 大腿

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 外侧 3-4，内侧 5-6 |
| 变形容错率 | ★★★★（肌肉大，吃重不易变形） |
| 可见度 | 低（日常遮盖） |
| 推荐风格 | New School、写实、水彩、日式大图、Old School |
| 推荐图案 | 大幅图案、全身像、神话场景 |
| 特别警告 | 大腿内侧神经丰富，痛感高于外侧；大腿根部和臀部交界处痛感突增 |

### 13. 小腿

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 外侧 4-5，胫骨 7-8，后侧 6-7 |
| 变形容错率 | ★★★★★ |
| 可见度 | 中（裤子大部分遮盖，短裤/裙露出） |
| 推荐风格 | 水墨竖构图、Old School、日式、几何 |
| 推荐图案 | 竹子、日式锦鲤、几何图腾、老学校玫瑰 |
| 特别警告 | 胫骨（小腿前侧）是骨骼凸出部位，痛感极高；愈合期穿长裤会摩擦纹身 |

### 14. 脚踝/脚部

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 脚踝 7-8（骨骼），脚背 7-8 |
| 变形容错率 | ★★★★★ |
| 可见度 | 中（鞋子+袜子遮盖，凉鞋露出） |
| 推荐风格 | Fine line、小几何、小花卉 |
| 推荐图案 | 环绕脚踝的链式图案、脚背小图案 |
| 特别警告 | ⚠️ 脚部纹身褪色快（穿鞋摩擦+出汗）；脚背纹身愈合期不能穿鞋，影响生活 |

### 15. 膝盖/肘部

| 属性 | 评估 |
|------|------|
| 疼痛指数 | 膝盖 8-9（骨+关节），肘部 7-8 |
| 变形容错率 | ★★★★（关节本身不变，但弯曲时图案变形） |
| 可见度 | 膝盖：中；肘部：中-高（短袖） |
| 推荐风格 | 几何曼陀罗（利用圆形优势）、Blackwork |
| 推荐图案 | 圆形/放射状图案、蜘蛛网 |
| 特别警告 | ⚠️ 关节频繁活动使愈合极其困难；弯曲时图案严重变形需在设计阶段考虑；关节纹身褪色快且补色效果差 |

## 纹身师选择指南

### 按风格找纹身师

| 风格 | 关键考察点 | 避雷信号 |
|------|-----------|----------|
| Old School | line work 稳定性、色彩饱和度 | 线条抖动、颜色暗沉 |
| New School | 色彩过渡、构图想象力、smooth shading | 颜色脏/泥浆感 |
| 日式传统 | 浮世绘构图理解、背景处理功底 | 满填无 skin break |
| 水墨 | wash shading 技法、毛笔笔触感 | 渐变生硬、有硬边 |
| 写实 | 素描功底、人像比例把握 | 五官比例失调 |
| 水彩 | 色彩透明感、soft edge | 颜色脏、无透明感 |
| 几何 | 对称性、线 precision | 不对称、线条歪曲 |
| Fineline | 单针稳定度、深度控制 | 线条粗细不均、blowout |

### 选择纹身师的红绿灯

**绿灯信号** ✅:
- 作品集风格专一（不万金油）
- 同一风格作品数量 > 50 张
- 有 healed photo（愈后照片）对比
- 工作室卫生许可证齐全
- 明确告知价格和预估时间
- 会给出专业建议甚至拒绝不合理要求

**红灯信号** 🚫:
- 作品集风格杂乱（什么都做=什么都不精）
- 没有愈后照片
- 价格异常低（低于市场价 50%+）
- 工作室环境脏乱
- 什么都答应，不提出专业建议
- 使用非一次性针具/染料来源不明
- 无证经营
