## Description: <br>
Provides cryptography attack techniques for CTF challenges involving encryption, hashing, signatures, ZKP, PRNG, RSA, AES, ECC, lattices, LWE, CVP, number theory, Coppersmith, Pollard, Wiener, padding oracles, GCM, key derivation, and stream or block cipher weaknesses. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, security researchers, and CTF participants use this skill as a quick-reference guide for selecting and applying cryptographic attack techniques in authorized CTF challenges, labs, and defensive research. It helps agents triage crypto challenge patterns and produce analysis, code snippets, shell commands, and tool setup guidance. <br>
