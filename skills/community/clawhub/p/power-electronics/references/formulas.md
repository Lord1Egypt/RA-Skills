# 电力电子公式速查（四大变换）



符号：D 占空比，fsw 开关频率，α 触发角(°)，M 调制比，U2 相电压有效值，Vm=√2·U2。



---



## 一、AC-DC 整流



### 1.1 不可控整流



| 拓扑 | 平均输出 Vd | 系数（相对 U2） | 纹波频率 |

|------|-------------|-----------------|----------|

| 单相半波 | Vm/π | 0.45 | f |

| 单相全波/桥式 | 2Vm/π | 0.9 | 2f |

| 三相半波 | √3·Vm/π | 1.17 | 3f |

| 三相桥式 | 3√2·Vm/π | 2.34 | 6f |



### 1.2 全控整流（理想，连续电流）



| 拓扑 | Vd |

|------|-----|

| 单相全控桥 | (2√2/π)·U2·cosα ≈ **0.9·U2·cosα** |

| 三相半波可控 | (3√6/(2π))·U2·cosα ≈ **1.17·U2·cosα** |

| 三相全控桥 | (3√6/π)·U2·cosα ≈ **2.34·U2·cosα** |



- 整流区：α < 90°，Vd > 0

- 逆变区：α > 90° 且存在足够反电动势/源

- 三相全控桥线电压：UL = √3·U2



### 1.3 滤波



**电容滤波（轻载近似）**：Vo ≈ √2·U2（单相全波），重载 Vd 下降。



**电容选型**：C ≥ Io / (fripple · ΔV)



| 整流类型 | fripple |

|----------|---------|

| 单相半波 | f |

| 单相全波 | 2f |

| 三相桥 | 6f |



**电感滤波（大电流）**：Vd ≈ 0.9·U2·cosα（全控），纹波以电流为主。



**LC 滤波**：截止频率 fc = 1/(2π√LC) ≪ fripple。



### 1.4 功率因数（相控整流）



- 位移因子 cosφ ≈ cosα（近似）

- α 越大，PF 越低 → 大功率需 PFC（Boost CRM/DCM）

## 二、DC-AC 逆变



### 2.1 单相 H 桥 SPWM



| 量 | 公式 |

|----|------|

| 基波峰值 | Vo1_peak = M · Vdc / 2 |

| 基波有效值 | Vo1_rms = M · Vdc / (2√2) |

| 线性调制 | M ≤ 1 |



### 2.2 三相 SPWM



| 量 | 公式 |

|----|------|

| 线电压基波峰值 | Vl_peak = M · Vdc / 2 |

| 相电压基波有效值 | Vph_rms = M · Vdc / (2√2) |



### 2.3 三相 SVPWM



| 量 | 公式 |

|----|------|

| 线电压基波峰值（线性） | Vl_peak = (√3/3) · Vdc · M |

| 线电压基波有效值 | Vl_rms = Vdc · M / √6 ≈ 0.408·Vdc·M |

| 相电压基波有效值 | Vph_rms = Vl_rms / √3 |

| 六阶梯波峰值 | Vl = √3/3 · Vdc |



**扇区判定**：由 Vα、Vβ 分 6 扇区（各 60°）。



**七段式作用时间**（扇区内角 θ）：

- T1 = m·T·sin(60°−θ)/sin(60°)

- T2 = m·T·sin(θ)/sin(60°)

- T0 = T − T1 − T2



### 2.4 死区与过调制



- 死区 td：桥臂互补开关间插入，防止直通

- M > 1：过调制，基波增大但 THD 上升

- 过调制极限（六阶梯）：M ≈ 1.15（SPWM）或更高（SVPWM）

## 三、DC-DC 斩波



### 3.1 Buck（CCM）



| 量 | 公式 |

|----|------|

| Vo | D · Vin |

| L | (Vin−Vo)·D / (k·Io·fsw) |

| C | Io·D / (fsw·ΔV) |

| Lcrit | (Vin−Vo)·Vo / (2·Io·Vin·fsw) |



### 3.2 Boost（CCM）



| 量 | 公式 |

|----|------|

| Vo | Vin / (1−D) |

| Iin | Io / (1−D) |

| L | Vin·D / (k·Iin·fsw) |

| Lcrit | Vin·D / (2·Iin·fsw) |



### 3.3 Buck-Boost（CCM，反相）



| 量 | 公式 |

|----|------|

| \|Vo\| | D·Vin / (1−D) |

| D | \|Vo\| / (Vin + \|Vo\|) |

| 开关/二极管电压 | Vin + \|Vo\| |



### 3.4 Flyback（DCM）



| 量 | 公式 |

|----|------|

| 输出功率 | Po ≈ Vin²·D²·T / (2·Lm) |

| 峰值电流 | Ipk = Vin·D·T / Lm |

| 开关 Vds | Vin + Vo·Np/Ns |

## 四、AC-AC 交流变换



### 4.1 相控调压（电阻负载）



单相/三相每相：



**Vo_rms = U2 · √[(π − α + sin2α/2) / π]**



- α = 0° → Vo = U2

- α = 90° → Vo ≈ 0.707·U2

- α = 180° → Vo = 0



### 4.2 交流斩波（电阻负载）



**Vo_rms = Vin_rms · √D**



双向开关 + 反并联二极管；D 为导通占空比。



### 4.3 交交变频（ Cycloconverter ）



- 输出频率 fout = fin / N（N 为脉波拍数，近似）

- 约束：fout ≪ fin，通常 fout ≤ fin/3 ~ fin/10

- 三相 6 脉波：最低输出约 fin/6



### 4.4 矩阵变换器



- 直接 AC→AC，无 DC 链路

- 输出频率上限 fout < fin/3（理论）

- 四象限：可调 V、可调 f、可回馈

## 五、损耗估算



| 类型 | 近似 |

|------|------|

| 导通 | P = I²rms·Rds(on) 或 Vf·If_avg |

| 开关 | P = (Eon+Eoff)·fsw |

| 电容 ESR | ΔV = ΔI·ESR |

## 六、标准值取整



- 电感 E12：10, 12, 15, 18, 22, 27, 33, 39, 47, 56, 68, 82, 100 …

- 电容耐压：×1.5~2 标准档 (16, 25, 50, 100, 450V …)

## 七、阻感负载整流（单相全控桥）

- **临界角**：α_crit = arctan(ωL/R) = arctan(2πfL/R)
- **α > α_crit 且连续**：Vd = 0.9·U2·cosα
- **α < α_crit**：电流断续，Vd 需分段积分（小于 0.9U2cosα）
- **平波电感经验**：L(H) ≥ 2.5·U2 / (1.414·Id)

---

## 八、Boost PFC（CRM）

- **D_max** = 1 − √2·Vin_rms / Vo
- **电感**：L = (√2·Vin_rms)² / (2·Po·fsw_min)
- **峰值电流近似**：Ipk ≈ 2√2·Po / (η·Vin_rms)
- Vo 必须 > 输入峰值

---

## 九、Flyback DCM

- **匝比 n = Ns/Np**
- **D** = n·Vo / (Vin + n·Vo)
- **Lm** = Vin²·D²·T / (2·Pin)
- **Ipk** = Vin·D·T / Lm
- **Vds** ≈ Vin + Vo/n

---

## 十、LLC 谐振

- **fr** = 1 / (2π√(Lr·Cr))
- **fn** = fsw / fr，**m** = Lm / Lr
- **Rac** = (8/π²)·n²·Vo² / Po
- **Q** = √(Lr/Cr) / Rac
- FHA 增益（近似）：M = m / √[(1+m−m/fn²)² + (Q(1/fn²−1))²]

---

## 十一、DCM 斩波补充

**Buck DCM**：Vo = Vin·D²·R / (2·L·fsw)

**Buck 临界负载**：Io_crit = (Vin−Vo)·Vo / (2·L·Vin·fsw)

**Boost DCM**：Vo 与负载强相关，需数值求解或查图

---

## 十二、QR Flyback

- **Lm** = Vin_min² / (2·Pin·fsw_max)
- **fsw** ∝ Vin²（恒功率）
- **Ipk** = √(2·Pin/(Lm·fsw))

---

## 十三、NPC 三电平逆变

- **Vl_peak** ≈ (√3/4)·Vdc·M
- **Vph_peak** ≈ M·Vdc/2
- **Vds_stress** = Vdc/2
