# 肥达反应的诊断逻辑

## 适用对象
- 发热伴疑似伤寒/副伤寒患者
- 疫区接触史者
- 不明原因长期发热患者

## 执行步骤
1. **确认检测方法与参考区间**：使用直接凝集法检测伤寒H、O及副伤寒甲、乙、丙抗体，参考区间为：
   - 伤寒H < 1:160
   - 伤寒O < 1:80
   - 副伤寒各型 < 1:80

2. **评估单份血清结果**：若O ≥ 1:80 且 H ≥ 1:160，则具有初步诊断意义。

3. **优先采用动态观察**：比较急性期与恢复期血清抗体效价，若升高≥4倍，诊断价值更高。

4. **按O/H组合解读结果**：
   - **O和H均升高**：提示伤寒可能性大（多见于病程第2周）。
   - **O不高、H升高**：考虑预防接种反应或非特异性回忆反应。
   - **O升高、H不高**：可能为感染早期，或由其他沙门菌引起的交叉反应。

5. **综合判断并发出警示**：结合临床表现、流行病学史及其他检查，避免孤立依赖单一时间点结果。

## 注意事项
- 已接种伤寒疫苗者可能出现H抗体升高，慎用本逻辑。
- 肥达反应易受既往感染、交叉反应等因素干扰，必须结合临床综合分析。
