# 北美扩展详细指南

## 一、加拿大市场深度攻略

### 1.1 市场规模与特点

**核心数据**：
- 年GMV：约$50亿
- 年增长率：15-20%
- 第三方卖家占比：~50%
- 主要站点：Amazon.ca

**消费者特点**：
- 与美国消费者高度相似
- 对价格敏感（加元汇率影响）
- 英语为主（约75%英语区）
- 法语区（魁北克）需要法语

**热卖品类**：
1. 电子产品
2. 家居用品
3. 服装
4. 运动户外
5. 美妆护肤

### 1.2 入门要求

**账号要求**：
- 美国卖家账号可直接添加加拿大市场
- 无需单独注册

**税务要求**：
- GST（Goods and Services Tax）：5%
- 建议注册，Amazon可代扣代缴
- 如果年销>$30,000 CAD，需注册

**产品合规**：
- 类似美国的CPSC/UL要求
- 部分品类需要IC认证（电子产品）
- 食品/保健品审查更严

### 1.3 物流方案

**方案1：FBA泛美计划**
```
优点：
- Amazon处理仓储和配送
- Prime标志
- 退货处理

缺点：
- 偏远地区额外费用
- 库存管理复杂
- 费用较高
```

**方案2：FBM自配送**
```
优点：
- 控制成本
- 灵活调整

缺点：
- 无Prime标志
- 转化率低
- 客户服务压力大
```

**方案3：第三方仓库**
```
优点：
- 比FBA便宜
- 控制灵活

缺点：
- 需要自己处理配送
- Prime问题
```

### 1.4 定价策略

**定价公式**：
```
加拿大售价（CAD）= 美国售价（USD）× 汇率 × 1.05~1.15

考虑因素：
- 汇率波动
- 5% GST
- 运费差异
- 竞争环境
```

**定价参考**：
| 美国售价 | 建议加拿大售价 | 说明 |
|----------|----------------|------|
| $10 USD | $14-15 CAD | 低价品，加价少 |
| $25 USD | $32-35 CAD | 中价品 |
| $50 USD | $65-70 CAD | 高价品 |

**注意事项**：
- ⚠️ 不要定价比美国还低
- ⚠️ 考虑FBA偏远附加费
- ⚠️ 留出汇率波动空间

### 1.5 运营要点

**Listing适配**：
- 翻译为英语（美式英语即可）
- 图片可以共用美国版本
- 考虑法语音频/图片（魁北克用户）

**客户服务**：
- 英语回复即可
- 响应时间要求同美国
- 假期考虑加拿大节日

**常见问题**：
1. **是否需要发FBA？**
   - 建议发，Prime标志很重要
   - 可以先小批量测试

2. **魁北克要不要单独处理？**
   - 先不用，等销量上来再说
   - 有销量后再考虑法语

3. **关税怎么处理？**
   - Amazon会代收关税
   - 收件人支付

## 二、墨西哥市场深度攻略

### 2.1 市场规模与特点

**核心数据**：
- 年GMV：约$20亿
- 年增长率：25-30%
- 第三方卖家占比：~40%
- 主要站点：Amazon.com.mx

**消费者特点**：
- 年轻人口多
- 手机购物占比高（>70%）
- 价格敏感
- 喜欢分期付款
- 对国际品牌有好感

**热卖品类**：
1. 电子产品（手机配件、电脑配件）
2. 服装鞋帽
3. 家居用品
4. 母婴用品
5. 美妆护肤

### 2.2 入门要求

**账号要求**：
- 需要单独注册墨西哥卖家账号
- 可与美国账号统一管理

**RFC税号**：
- 必须注册RFC（Registro Federal de Contribuyentes）
- 个人或企业RFC
- 影响发票开具

**银行账户**：
- 理想情况：墨西哥本地银行账户
- 备选：支持墨西哥收款的第三方服务商

### 2.3 物流方案

**FBA方案**：
```
FBA Mexico流程：
美国工厂/仓库 → 墨西哥FBA仓库 → Amazon配送

优点：
- Amazon处理最后一公里
- Prime标志
- 退货处理

缺点：
- 清关复杂（2-3周）
- 关税高（部分品类15-30%）
- FBA费用比美国贵
```

**FBM方案**：
```
美国直接发货：
美国仓库 → 墨西哥消费者

优点：
- 无需墨西哥FBA库存
- 测试市场

缺点：
- 配送慢（2-4周）
- 关税由买家支付
- 客户体验差
```

**第三方仓库**：
- 建议找清关能力强的服务商
- 可以降低清关风险

### 2.4 合规与清关

**清关注意事项**：
| 问题 | 解决方案 |
|------|----------|
| 清关时间长 | 提前备货，预留时间 |
| 关税高 | 选择低关税品类，或加价覆盖 |
| 文件要求 | 发票/装箱单必须规范 |
| 假货风险 | 找有经验清关行 |

**常见清关文件**：
- 商业发票（西语/英语）
- 装箱单
- 原产地证（可能需要）
- 产品认证文件

### 2.5 定价策略

**定价公式**：
```
墨西哥售价（MXN）= (美国成本 + 关税 + 运费 + 利润) × 汇率

关税参考：
- 电子产品：15-20%
- 服装：30%
- 家居：20%
- 玩具：20%

⚠️ 注意：墨西哥有IVA（增值税）16%
```

**定价示例**：
```
产品成本：$15 USD
关税（20%）：$3 USD
运费到FBA：$3 USD
美国费用（15%）：按售价

建议定价：
- 美国价 $30 USD → 墨西哥 $650-750 MXN
- 墨西哥消费者对价格敏感，不建议太高
```

### 2.6 本地化要点

**语言**：
- 产品标题：西班牙语
- 产品描述：西班牙语
- 说明书：西班牙语
- 客服：西班牙语

**支付**：
- 信用卡
- OXXO现金支付（墨西哥特色）
- 分期付款（Credomatic等）

**图片**：
- 可以用英语市场图片
- 建议添加西语主要信息
- 避免文化冲突

### 2.7 常见问题

**Q：要不要发FBA墨西哥？**
A：建议发，但要先测试。清关是最大风险。

**Q：墨西哥退货怎么处理？**
A：FBA会自动处理，但可能退回美国或销毁。建议提前规划。

**Q：被跟卖了怎么办？**
A：墨西哥假货问题严重，要有心理准备。注册当地商标有帮助。

## 三、北美多市场协同

### 3.1 库存协同策略

**FBA库存分配**：
```
策略1：集中美国
- 美国FBA库存充足
- 墨西哥少量或不发FBA
- 墨西哥用FBM测试

策略2：美国+墨西哥FBA
- 美国热销SKU发墨西哥FBA
- 测试后再决定是否扩大
- 注意墨西哥仓储费

策略3：加拿大+美国泛欧
- 利用FBA泛美计划
- 库存自动分配
- 偏远附加费需考虑
```

**库存计算**：
```
北美总需求 = 美国需求 + 加拿大需求 + 墨西哥需求

建议分配：
- 美国：70%
- 加拿大：20%
- 墨西哥：10%

⚠️ 墨西哥需求难预测，建议保守
```

### 3.2 定价协同

**统一定价策略 vs 差异化策略**：

| 策略 | 适用情况 | 做法 |
|------|----------|------|
| 统一美元定价 | 小规模试水 | 美国价=北美价 |
| 区域差异化 | 正式运营 | 各市场独立定价 |
| 市场领导策略 | 成熟运营 | 加拿大跟随美国，墨西哥试探 |

**汇率管理**：
- 建议设置自动调价规则
- 汇率波动>5%时考虑调整
- 不要频繁调价（影响排名）

### 3.3 运营协同

**Listing管理**：
```
优先级：
1. 美国listing优化（最重要）
2. 加拿大listing同步（次要）
3. 墨西哥listing本地化（最后）

建议：
- 美国listing完善后再扩展
- 用工具批量管理多市场
```

**客户服务协同**：
- 英语客服覆盖加拿大
- 墨西哥需要西班牙语
- 考虑外包或工具
