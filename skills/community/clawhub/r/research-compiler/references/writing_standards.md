# 科研成果写作规范：无废话原则与创新性表达

---

## 一、核心写作原则

### 原则 1：首句即核心（段落结构铁律）

**错误示范**：
> 随着我国城镇化进程的不断加速，城市建设面临着越来越多的新问题与新挑战，在此背景下，本研究认为……

**正确示范**：
> 城市存量更新阶段，社区微更新机制的缺失是制约老旧社区品质提升的核心障碍。

**规则**：段落首句即论点，不铺垫，不渲染背景。

---

### 原则 2：创新点显式标注

在正文中，凡涉及本研究创新之处，须用以下标签显式标注：

| 标签 | 适用情形 |
|------|---------|
| `【理论创新】` | 提出新概念、新命题、修正已有理论 |
| `【方法创新】` | 使用新研究方法、开发新测量工具 |
| `【实践创新】` | 提出新对策模型、新机制设计、应用新技术 |
| `【数据创新】` | 建立新数据集、提供一手调查数据 |

示例：
> 【理论创新】本研究在传统"政府主导"模式之外，提出"社区协商共治"的第三种微更新路径，完善了存量更新治理的理论谱系。

---

### 原则 3：数字优先于形容词

| 废话表达 | 精准表达 |
|---------|---------|
| 大幅提升了效率 | 效率提升 37%（与对照组相比） |
| 得到了广泛应用 | 已在 6 省 12 市推广应用 |
| 取得了显著成效 | 试点社区满意度从 62% 提升至 89% |
| 覆盖面较广 | 样本覆盖 23 个省份，856 个社区 |
| 具有重要意义 | [直接说明意义内容，如"填补了……领域的研究空白"] |

---

### 原则 4：禁用词清单

以下表达在科研报告中一律不得使用：

**万能开头废话**：
- ❌ 随着……时代的到来
- ❌ 在当今社会……
- ❌ 众所周知……
- ❌ 不言而喻……
- ❌ 本研究认为……（直接说研究发现）

**冗余过渡废话**：
- ❌ 综上所述，不难看出……
- ❌ 由此可见……（直接写结论）
- ❌ 值得注意的是……（去掉这句，直接写内容）
- ❌ 在此基础上，我们进一步……

**假谦虚废话**：
- ❌ 本研究仅做了初步探索
- ❌ 由于作者水平有限……
- ❌ 研究结论仅供参考

**无意义强调词**：
- ❌ 非常重要、极为关键、至关重要（换成具体说明"重要"在哪）
- ❌ 进一步深入研究（说清楚研究什么）

### 原则 5：引用规范

#### 中文文献（GB/T 7714-2015）

**期刊论文**：
```
作者. 文章题名[J]. 刊名, 年份, 卷(期): 起止页码.
例：李明, 王华. 城市社区微更新机制研究[J]. 城市规划学刊, 2023, 65(2): 45-53.
```

**专著**：
```
作者. 书名[M]. 出版地: 出版社, 年份: 引用页码.
例：张伟. 城市更新理论与实践[M]. 北京: 中国建筑工业出版社, 2022: 123.
```

**报告/政策文件**：
```
发布机构. 文件名[R/Z]. 发布年份.
例：住房和城乡建设部. 城镇老旧小区改造技术指南[Z]. 2020.
```

#### 外文文献（APA 第7版）

**期刊论文**：
```
Author, A., & Author, B. (Year). Title of article. Journal Name, Volume(Issue), Pages. https://doi.org/xxxxx
```

**专著**：
```
Author, A. (Year). Title of book. Publisher.
```

## 二、各章节写作要点

### 绪论写作要点

**研究背景**（避免流水账，用"问题导向"结构）：
```
结构：宏观背景（2句）→ 领域问题（2-3句）→ 具体研究缺口（1-2句）→ 本研究定位（1句）
```

**研究综述**（避免"文献罗列"，用"评述"逻辑）：
- 不要：作者A研究了XX，作者B研究了YY，作者C研究了ZZ……
- 要：按**研究主题**或**研究方法**或**演进脉络**组织综述
- 每个综述段落末尾须有**评价性结论**（如"然而上述研究均未关注……"）

### 核心章节写作要点

每节论证结构（PREP原则）：
```
P（Point）：   论点 —— 本节核心主张（首句）
R（Reason）：  理由 —— 为什么成立（第2-3句）
E（Evidence）：证据 —— 数据/文献/案例（主体内容）
P（Point）：   重申论点 —— 结尾呼应（末句）
```

### 结论写作要点

**错误示范**：
> 本研究通过对……的研究，得出了一系列重要结论，对今后的研究具有重要参考意义。

**正确格式**：
```
结论一：[一句话陈述]。本研究通过……（数据/案例），发现……，表明……。
结论二：[一句话陈述]。……
结论三：[一句话陈述]。……

研究创新：
（1）【理论创新】……
（2）【方法创新】……
（3）【实践创新】……
```

## 三、摘要写作规范（结构化摘要）

标准结构化摘要（适用于大多数研究报告）：

```
【研究目的】（1-2句）：明确说明研究解决什么问题
【研究方法】（1-2句）：研究方法、样本/数据来源
【主要发现】（3-5句）：核心研究结论，含数据
【创新贡献】（1-2句）：理论/方法/实践创新
【应用价值】（1句）：成果可用于什么
```

示例：
```
【研究目的】本研究旨在揭示城市老旧社区微更新中多元主体协同机制的失灵成因，并构建可操作的协同治理框架。
【研究方法】采用混合方法，对12个典型社区进行田野调查，结合856份居民问卷数据和38位利益相关者深度访谈。
【主要发现】研究发现：（1）资金分配机制碎片化是制约微更新可持续性的首要因素，78.3%的受访社区存在资金断层问题；（2）居民参与呈现"伪参与"特征，实质决策参与率仅为12.4%；（3）物业管理缺位社区微更新满意度显著低于有物业管理社区（p<0.01）。
【创新贡献】【理论创新】提出"分级参与-资金闭环-技术赋能"三维协同模型，修正了传统单一主体治理理论。
【应用价值】研究结论可直接应用于地方政府制定老旧小区改造政策及社区规划实践。
```

## 四、图表规范

### 图的规范

- 图题：图序（阿拉伯数字）+ 图题（简洁，含变量/主题信息），置于图下方
- 图注：数据来源、说明，紧随图题之后
- 格式：图题 12pt，黑体；图注 10.5pt，宋体

示例：`图 3  2015-2023年各省老旧小区改造面积（万平方米）`

### 表的规范

- 表题：表序（阿拉伯数字）+ 表题，置于表上方
- 三线表格式（上边线、标题下线、下边线，无竖线）
- 表注：数据来源、缩写说明，紧随表格之后

示例：`表 2  受访社区基本特征统计（N=12）`

## 五、内容深化：网络资料检索策略

在 Phase 3 内容深化过程中，针对各类信息需求使用以下检索策略：

| 需求类型 | 推荐来源 | 检索关键词建议 |
|---------|---------|-------------|
| 政策依据 | 中国政府网、国务院文件库 | "课题关键词" + "政策/意见/规划" |
| 统计数据 | 国家统计局、各部委统计年鉴 | 指标名称 + 年份 |
| 学术文献 | 中国知网、万方、Web of Science | 核心概念 + 近3-5年 |
| 行业标准 | 国家标准全文公开系统（openstd.samr.gov.cn）| GB/GBT + 关键词 |
| 地方实践案例 | 住建部官网、各省住建厅 | 试点名称 + 经验/模式 |
| 国际经验 | World Bank、UN-Habitat报告 | 英文核心词 + policy/report |

### 文献整合原则

1. **时效性**：政策文件优先采用近3年；统计数据优先采用最新年度
2. **权威性**：国家级 > 省部级 > 地方/行业；SCI/SSCI > 核心期刊 > 普通期刊
3. **相关性**：文献须与所在板块的论点直接相关，不做泛泛引用
4. **平衡性**：支持性文献与质疑性文献均需列出，体现批判性视角
