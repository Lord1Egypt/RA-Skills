## 移民服务任务清单

移民服务按子域组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：移民评估 → 移民申请 → 移民安置 → 移民管理

## 依赖拓扑摘要

### 一、域内链路

**移民评估链路**: T1-1 → T1-2 → T1-3 → T1-4
**移民申请链路**: T2-1 → T2-2 → T2-3 → T2-4
**移民安置链路**: T3-1 → T3-2 → T3-3 → T3-4
**移民管理链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整移民服务链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T3-1 → T3-2 → T3-3 → T3-4 → T4-1 → T4-2 → T4-3 → T4-4

**反馈改进链路**: T4-4 → T1-1（后续服务反馈到资格评估）

**政策驱动链路**: T1-2 → T2-1（条件分析驱动材料准备）

更多组合根据具体任务动态推导。
