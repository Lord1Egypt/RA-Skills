# Zhongyi Basic Theoretical Characteristics

## When to Use
- 分析中医诊疗思路或解释中医为何强调“四诊合参”
- 制定个体化中医治疗或养生方案
- 对比中西医思维差异
- 解读“同病异治”“异病同治”等中医特有原则
- 在疾病预防或慢病管理中融入中医理念

> **注意**：本技能不适用于急症抢救场景（需结合现代医学），也不适用于基于机械还原论的生物医学模型分析。

## Execution Steps

### 1. 应用整体观念
- 将人体视为五脏六腑、形体官窍通过经络气血相互联系的有机整体。
- 考察患者所处的自然环境（如四时、地域、气候）与社会环境（如情志、人际关系）对其健康的影响。
- 在诊断中综合望、闻、问、切四诊信息；在治疗中关注全身状态而非仅针对局部症状。

### 2. 运用恒动观念
- 认识生命处于持续运动变化中，包括生理上的气血运行与脏腑功能活动。
- 理解疾病具有发生、发展、传变与转归的动态过程。
- 根据病情变化动态调整治法方药，例如依据《伤寒论》原则随证加减。

### 3. 实施辨证论治
- **辨证**：整合四诊资料，分析病因、病位、病性及邪正关系，归纳为特定“证候”。
- **论治**：依据证候确定治则治法（如“寒者热之”“虚则补之”）。
- 区分“同病异治”（相同疾病因证不同而治法不同）与“异病同治”（不同疾病因证相同而治法相同）。

### 4. 整合三大特点
- 以整体观念为认识基础，恒动观念为动态视角，辨证论治为实践核心，三者协同指导中医临床与养生实践。

> **前提要求**：使用者需已掌握阴阳五行、藏象、气血津液等中医基础概念。
