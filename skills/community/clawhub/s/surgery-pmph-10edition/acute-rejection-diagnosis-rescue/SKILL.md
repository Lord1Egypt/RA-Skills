# 急性排斥反应诊断与挽救治疗

## 触发条件
当移植受体出现以下任一表现时，立即启动本流程：
- 不明原因发热
- 移植区域胀痛
- 移植器官功能减退（如血清肌酐升高、胆红素上升、射血分数下降等）

## 诊断步骤
1. **确认适用对象**：仅适用于已接受实体器官移植（肾、肝、心脏、胰腺等）且正在接受免疫抑制治疗的患者。
2. **排除非适用情况**：不适用于超急性排斥反应（通常在移植后数分钟至数小时内发生）或慢性排斥反应（表现为缓慢进展的器官功能丧失）。
3. **执行移植物活检**：
   - 所有疑似急性排斥反应病例均应行移植物活检；
   - 活检是诊断金标准，不可依赖生化或免疫学指标替代。
4. **病理分型**：
   - T细胞介导的排斥反应（TCMR）：以间质和血管周围炎症细胞浸润为特征；
   - 抗体介导的排斥反应（AMR）：以毛细血管C4d沉积、微血栓形成为特征；
   - 两者可单独或共存。

## 挽救治疗流程
1. **一线治疗：大剂量糖皮质激素冲击**
   - 使用甲泼尼龙 500–1000 mg/日，静脉给药，连续3天；
   - 启动时机：确诊后立即开始。
2. **二线治疗：激素无效时升级方案**
   - 使用抗淋巴细胞球蛋白（ALG）或抗胸腺细胞球蛋白（ATG）；
   - 对AMR患者同步进行血浆置换，清除循环供体特异性抗体（DSA）。
3. **调整基础免疫抑制方案**
   - 增加钙调磷酸酶抑制剂（CNI）或霉酚酸酯（MMF）剂量；
   - 根据药物浓度监测结果个体化调整。

## 预后提示
- 早期诊断并及时干预者预后良好；
- 延误治疗可能导致不可逆损伤，进展为慢性排斥反应。

> **注意**：本流程不涵盖超急性或慢性排斥反应的处理，亦不适用于非实体器官移植（如骨髓移植）。
