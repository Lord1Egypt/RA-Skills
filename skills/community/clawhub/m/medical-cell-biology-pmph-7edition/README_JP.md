## プロジェクト概要

本プロジェクトは医学細胞生物学のコア知識体系を体系的に統合し、**146のキースキル**を**20の主要カテゴリ**にわたって網羅しています。細胞構造と機能、分子メカニズム、シグナル伝達、細胞周期制御、幹細胞生物学、細胞死と老化、オルガネラ動態、膜輸送、遺伝子発現制御、疾患メカニズム、最先端研究技術など、多岐にわたる分野をカバーしています。

**対象読者**：医学生、研究者、臨床医、教育者

**参考教科書**：人民衛生出版社『医学細胞生物学』第7版

**⚠️免責事項**：本スキルは細胞生物学の概念、分子メカニズム、病理分析、実験技術を網羅していますが、独立した医療アドバイスや実験指導として誤用される可能性があります。

対策：出力は教育または研究の参考としてのみ使用し、最新のガイドラインと分野のコンセンサスに照らして推奨事項を検証してください。

## プロジェクト構造

```

├── <skill-name>/         # 個別スキル定義
│   ├── SKILL.md          #   スキル詳細（トリガー、手順、参考資料）
│   └── references/       #   オプションの補足資料
├── scripts/              # 実行可能ユーティリティスクリプト
├── config/               # 設定ファイル
├── assets/               # 静的リソース（アイコン、画像）
└── tests/                # 検証とテスト
```
