# 行业研究参考手册

## 数据源优先级

### 价格数据
1. 英文主流媒体：AP News、BBC、NPR → 搜 `commodity price 2025 2026`
2. 中文财经：界面新闻、彭博中文、东方财富网、生意社
3. 宏观数据：国家统计局、海关总署

### 新闻
- AP News → BBC → NPR（英文）
- 界面新闻 → 财新 → 澎湃（中文）

### 上市公司财务
- 东方财富网（财务数据页面）
- 新浪财经（个股页面）
- 巨潮资讯网（年报原文）

## 常用搜索关键词模板

### 价格走势
```
"{商品}" price trend 2025 2026
"{商品}" 价格上涨 原因
"{商品}" 市场价格 最新
```

### 产业链
```
"{商品}" 上游 原料
"{商品}" 下游 用途 消费占比
"{商品}" 产业链 上下游
```

### 上市公司
```
A股 "{商品}" 上市公司 龙头
"{商品}" 概念股 有哪些
```

### 投资分析
```
"{商品}" 投资机会 2026
"{商品}" 价格波动 影响 上市公司
```
