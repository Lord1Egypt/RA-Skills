# 抗癫痫药作用机制分类

## 何时使用
- 解释为何苯妥英钠对强直阵挛发作有效但对失神发作无效
- 选择针对特定发作类型的药物（如乙琥胺用于失神发作）
- 分析广谱抗癫痫药（如丙戊酸钠）的多靶点机制

## 机制分类

### 第一大类：调节神经递质平衡
1. **增强GABA作用**：
   - 激活GABA_A受体 → Cl⁻内流 ↑ → 超极化 → 兴奋性 ↓
   - 代表药：苯巴比妥、苯二氮䓬类
2. **拮抗谷氨酸作用**：
   - 抑制谷氨酸释放或受体激活
   - 代表药：丙戊酸钠（抑制谷氨酸释放）

### 第二大类：干扰离子通道（膜稳定作用）
1. **阻滞电压依赖性Na⁺通道**：
   - 延长失活时间，提高动作电位阈值
   - 代表药：苯妥英钠、卡马西平、拉莫三嗪
2. **阻滞Ca²⁺通道**：
   - T型通道阻滞 → 抑制丘脑3Hz放电（乙琥胺，用于失神发作）
   - N/L型通道阻滞 → 减少递质释放（苯妥英钠）
3. **影响K⁺通道**：
   - GABA_B激活 → K⁺外流 ↑ → 超极化

## 药物-机制对应示例
- **苯妥英钠**：主要阻断异常放电扩散（Na⁺通道），不抑制病灶放电
- **乙琥胺**：特异性抑制T型Ca²⁺电流，仅用于失神发作
- **丙戊酸钠**：广谱，通过↑GABA + ↓Na⁺/T型Ca²⁺通道

> 注意：本分类不适用于代谢性脑病引起的癫痫样活动。
