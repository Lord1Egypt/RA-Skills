## 范本1：中年人年度健康管理全流程

### 题目
45岁企业高管的年度健康管理全流程

### 背景
张先生，45岁，企业高管，工作压力大，经常出差，饮食不规律，缺乏运动。父亲有高血压病史，母亲有糖尿病病史。希望进行全面的年度健康管理。

### Step 1: 健康评估阶段

#### 1.1 基础体检规划 (H1-01)
**输入**: 年龄45岁，男性，高管职业，家族史(高血压/糖尿病)
**执行**:
1. 确定体检项目：基础体检+心血管专项+肿瘤筛查
2. 选择体检机构：三甲医院体检中心
3. 安排体检时间：避开出差高峰期
4. 体检前准备：空腹8小时，避免剧烈运动

**输出**: 个性化体检方案

#### 1.2 实验室检查解读 (H1-02)
**输入**: 体检报告(血常规/生化/肿瘤标志物/心电图/超声)
**执行**:
1. 识别异常指标：血脂偏高(总胆固醇6.2mmol/L)，血糖临界(空腹血糖6.0mmol/L)
2. 分析异常原因：饮食不规律，缺乏运动，压力大
3. 评估临床意义：心血管风险增加，糖尿病前期
4. 给出建议措施：生活方式干预，3个月后复查

**输出**: 检查解读报告

#### 1.3 生活方式评估 (H1-04)
**输入**: 饮食(外卖为主，油腻)，运动(几乎无)，睡眠(6小时，质量差)，压力(高)
**执行**:
1. 收集生活方式信息：详细记录一周生活
2. 评估各维度得分：饮食2分，运动1分，睡眠3分，压力2分
3. 识别主要风险因素：饮食和运动是主要问题
4. 确定优先改善领域：先改善运动，再调整饮食

**输出**: 生活方式评估报告

#### 1.4 健康风险分层 (H1-05)
**输入**: 体检结果，实验室检查，生活方式评估，家族史
**执行**:
1. 汇总所有风险因素：年龄，血脂高，血糖临界，家族史，生活方式差
2. 使用风险评估工具：Framingham心血管风险评分
3. 计算综合风险评分：10年心血管风险15%
4. 确定风险等级：中高风险
5. 制定风险管理策略：强化生活方式干预，考虑药物预防

**输出**: 风险分层报告

**校准点1**: 风险等级确认
- 确认风险等级为中高风险
- 确认主要风险因素
- 确认干预优先级

### Step 2: 疾病预防阶段

#### 2.1 心血管风险预防 (H2-03)
**输入**: 血压130/85mmHg，血脂高，血糖临界，吸烟史(已戒5年)
**执行**:
1. 评估心血管风险因素：血脂高是主要风险
2. 计算风险评分：10年风险15%
3. 确定风险等级：中高风险
4. 制定干预策略：
   - 生活方式：低脂饮食，规律运动，控制体重
   - 药物考虑：他汀类药物(如血脂3个月后仍高)
5. 监测计划：每月血压，3个月血脂，6个月血糖

**输出**: 心血管预防方案

#### 2.2 代谢性疾病预防 (H2-04)
**输入**: 血糖6.0mmol/L(临界)，BMI 26(超重)，腰围92cm
**执行**:
1. 评估代谢指标：血糖临界，超重，腹型肥胖
2. 识别代谢综合征：符合3项标准(血糖临界，超重，腹型肥胖)
3. 确定风险等级：糖尿病前期
4. 制定生活方式干预：
   - 饮食：控制总热量，增加膳食纤维
   - 运动：每周150分钟中等强度运动
   - 体重：减重5-7%
5. 监测计划：3个月空腹血糖+糖化血红蛋白

**输出**: 代谢性疾病预防方案

#### 2.3 癌症筛查计划 (H2-02)
**输入**: 年龄45岁，男性，吸烟史(已戒)，家族史(无特殊)
**执行**:
1. 确定适用筛查项目：肺癌(低剂量CT)，结直肠癌(肠镜)，胃癌(胃镜)
2. 评估风险等级：中风险(年龄+吸烟史)
3. 确定筛查频率：肺癌每年，结直肠癌每5年，胃癌每3年
4. 制定筛查计划：今年做肺癌CT，明年做肠镜
5. 安排随访：筛查后定期随访

**输出**: 癌症筛查计划表

**校准点2**: 预防方案确认
- 确认心血管预防策略
- 确认代谢性疾病干预措施
- 确认癌症筛查计划

### Step 3: 营养运动阶段

#### 3.1 营养需求评估 (H6-01)
**输入**: 年龄45岁，男性，中等活动水平，超重，血脂高
**执行**:
1. 收集基本信息：身高175cm，体重80kg，BMI 26
2. 评估营养需求：控制总热量，增加膳食纤维，控制脂肪
3. 识别营养风险：膳食纤维不足，钙摄入不足
4. 确定重点营养素：膳食纤维，钙，维生素D
5. 给出建议：增加全谷物，蔬菜，低脂乳制品

**输出**: 营养需求评估报告

#### 3.2 膳食计划设计 (H6-02)
**输入**: 营养需求，饮食偏好(喜欢面食)，工作忙碌
**执行**:
1. 确定营养目标：每日1800kcal，脂肪<30%，膳食纤维>25g
2. 设计食谱：
   - 早餐：全麦面包+鸡蛋+牛奶+水果
   - 午餐：杂粮饭+瘦肉+蔬菜(提前准备)
   - 晚餐：清淡为主，蔬菜+豆制品
3. 安排餐次：三餐+两次加餐(坚果/水果)
4. 制定采购清单：全谷物，瘦肉，蔬菜，水果，坚果
5. 烹饪指导：少油少盐，蒸煮为主

**输出**: 个性化膳食计划

#### 3.3 运动处方制定 (H6-03)
**输入**: 健康状况(心血管风险)，体能水平(差)，运动目标(改善代谢)
**执行**:
1. 评估健康状况：心血管风险中高，需要渐进运动
2. 确定运动目标：改善心肺功能，降低血脂血糖
3. 选择运动类型：有氧运动(快走/游泳)+抗阻运动
4. 制定运动计划：
   - 频率：每周5次
   - 强度：中等强度(心率120-140次/分)
   - 时间：每次30-45分钟
   - 渐进：从每次15分钟开始，每周增加5分钟
5. 安全注意事项：运动前热身，运动后放松，不适停止

**输出**: 个性化运动处方

#### 3.4 体重管理 (H6-04)
**输入**: 当前体重80kg，目标体重75kg，BMI 26
**执行**:
1. 评估当前体重状态：超重，腹型肥胖
2. 设定合理目标：6个月减重5kg，每月减0.8kg
3. 制定饮食计划：每日减少500kcal摄入
4. 制定运动计划：每周消耗2000kcal
5. 进度监测：每周称重，每月测量腰围

**输出**: 体重管理计划

**校准点3**: 营养运动方案确认
- 确认膳食计划可行性
- 确认运动处方安全性
- 确认体重管理目标合理性

### Step 4: 健康数据管理阶段

#### 4.1 可穿戴设备选型 (H7-01)
**输入**: 监测需求(心率/运动/睡眠)，功能要求，预算2000元
**执行**:
1. 确定监测需求：心率监测，运动追踪，睡眠分析
2. 了解可用设备：Apple Watch，华为手环，小米手环
3. 比较功能价格：Apple Watch功能全但贵，华为性价比高
4. 选择合适设备：华为手环(性价比高，功能满足)
5. 使用指导：正确佩戴，数据同步，APP使用

**输出**: 设备选型建议

#### 4.2 健康指标追踪 (H7-02)
**输入**: 追踪指标(血压/血糖/血脂/体重/运动)，监测频率
**执行**:
1. 确定追踪指标：血压(每周)，血糖(每月)，血脂(每3月)，体重(每周)，运动(每天)
2. 设定监测频率：根据指标重要性
3. 选择记录方式：手机APP+Excel表格
4. 设置提醒：测量提醒，复查提醒
5. 数据分析：每月分析趋势

**输出**: 指标追踪体系

#### 4.3 趋势分析报告 (H7-04)
**输入**: 3个月健康数据
**执行**:
1. 收集历史数据：血压，血糖，血脂，体重，运动数据
2. 选择分析指标：趋势变化，达标情况
3. 数据可视化：制作趋势图表
4. 趋势识别：血压稳定，血脂改善，体重下降
5. 建议措施：继续当前方案，加强运动

**输出**: 趋势分析报告

**校准点4**: 健康数据管理确认
- 确认设备选型合适
- 确认追踪体系可行
- 确认数据分析有效

### Step 5: 综合评估与调整

#### 5.1 健康报告生成 (H1-07)
**输入**: 健康评估结果，风险分层结果，数据整合结果
**执行**:
1. 汇总评估结果：整体健康状况，主要风险因素
2. 生成摘要：关键发现，主要建议
3. 详细分项报告：各系统评估结果
4. 个性化建议：针对性改善建议
5. 行动计划：下一步行动计划

**输出**: 综合健康报告

#### 5.2 下一年度规划
**输入**: 本年度健康管理效果，新的健康目标
**执行**:
1. 评估本年度效果：目标达成情况，健康改善程度
2. 确定新目标：进一步降低风险，提高生活质量
3. 调整管理策略：根据效果调整
4. 制定新计划：下一年度健康管理计划

**输出**: 下一年度健康管理计划

## 范本2：慢性病（糖尿病）长期管理

### 题目
55岁2型糖尿病患者的长期管理

### 背景
李女士，55岁，教师，2型糖尿病病史5年，目前服用二甲双胍。血糖控制不佳(糖化血红蛋白8.5%)，伴有轻度高血压和血脂异常。希望改善血糖控制，预防并发症。

### Step 1: 诊断评估阶段

#### 1.1 症状分析与鉴别 (H3-01)
**输入**: 主要症状(多饮多尿，乏力，视力模糊)，持续时间(5年)
**执行**:
1. 详细描述症状：多饮多尿，乏力，视力模糊，伤口愈合慢
2. 分析症状特征：典型糖尿病症状，提示血糖控制不佳
3. 列出可能诊断：2型糖尿病，糖尿病并发症(视网膜病变，神经病变)
4. 鉴别诊断要点：排除1型糖尿病，其他原因引起的血糖升高
5. 建议进一步检查：糖化血红蛋白，眼底检查，神经传导检查

**输出**: 症状分析报告

#### 1.2 检查项目建议 (H3-03)
**输入**: 症状信息，可能诊断，检查目的
**执行**:
1. 确定诊断需求：评估血糖控制，筛查并发症
2. 选择检查项目：
   - 血糖相关：空腹血糖，餐后血糖，糖化血红蛋白
   - 并发症筛查：眼底检查，尿微量白蛋白，足部检查，神经传导
   - 心血管评估：血脂，血压，心电图
3. 解释检查目的：了解控制情况，早期发现并发症
4. 注意事项：空腹检查，携带病历
5. 结果解读准备：了解正常范围

**输出**: 检查项目清单

#### 1.3 健康风险分层 (H1-05)
**输入**: 检查结果，病史，家族史
**执行**:
1. 汇总所有风险因素：血糖控制不佳，高血压，血脂异常，病程5年
2. 使用风险评估工具：糖尿病并发症风险评估
3. 计算综合风险评分：高风险(血糖控制不佳+多重危险因素)
4. 确定风险等级：高风险
5. 制定风险管理策略：强化血糖控制，综合管理多重危险因素

**输出**: 风险分层报告

**校准点1**: 诊断确认
- 确认2型糖尿病诊断
- 确认并发症风险等级
- 确认综合管理必要性

### Step 2: 治疗管理阶段

#### 2.1 治疗方案比较 (H3-04)
**输入**: 确定诊断，可选治疗方案，患者偏好
**执行**:
1. 列出所有治疗方案：
   - 口服药：二甲双胍，磺脲类，DPP-4抑制剂，SGLT-2抑制剂
   - 注射药：GLP-1受体激动剂，胰岛素
2. 比较疗效：SGLT-2抑制剂和GLP-1受体激动剂有心血管获益
3. 比较风险：低血糖风险，胃肠道反应，体重影响
4. 比较成本：药物费用，医保覆盖
5. 综合推荐：加用SGLT-2抑制剂(达格列净)

**输出**: 治疗方案比较表

#### 2.2 用药管理 (H4-01)
**输入**: 药物名称(二甲双胍+达格列净)，剂量，用药时间
**执行**:
1. 确认药物信息：
   - 二甲双胍：500mg，每日2次，餐中服用
   - 达格列净：10mg，每日1次，早餐前服用
2. 制定用药计划：设置服药提醒
3. 设置提醒：手机闹钟，药盒分装
4. 监测副作用：
   - 二甲双胍：胃肠道反应，维生素B12缺乏
   - 达格列净：泌尿生殖系统感染，低血压
5. 疗程管理：长期用药，定期评估

**输出**: 用药管理计划

#### 2.3 慢性病管理 (H4-03)
**输入**: 疾病诊断，治疗目标，监测指标
**执行**:
1. 确定管理目标：
   - 血糖：糖化血红蛋白<7%，空腹血糖<7mmol/L，餐后血糖<10mmol/L
   - 血压：<130/80mmHg
   - 血脂：LDL-C<2.6mmol/L
2. 制定治疗计划：
   - 药物治疗：二甲双胍+达格列净
   - 生活方式：饮食控制，规律运动
3. 监测方案：
   - 血糖：每日自我监测，每3月糖化血红蛋白
   - 血压：每周测量
   - 血脂：每6月检查
4. 生活方式调整：糖尿病饮食，规律运动
5. 定期评估：每3月评估控制效果

**输出**: 慢性病管理计划

**校准点2**: 治疗方案确认
- 确认药物治疗方案
- 确认管理目标
- 确认监测计划

### Step 3: 生活方式干预阶段

#### 3.1 特殊饮食指导 (H6-05)
**输入**: 饮食类型(糖尿病饮食)，营养要求
**执行**:
1. 了解特殊饮食要求：控制碳水化合物，均衡营养

## 范本3：心理健康危机干预

### 题目
30岁职场人士的焦虑抑郁危机干预

### 背景
王女士，30岁，互联网公司产品经理，近期工作压力大，出现严重焦虑和抑郁症状，失眠，食欲下降，工作效率降低，有自杀念头。需要紧急心理干预。

### Step 1: 识别与评估阶段

#### 1.1 心理健康筛查 (H1-03)
**输入**: 标准化量表(PHQ-9/GAD-7)，症状持续时间，功能影响程度
**执行**:
1. 选择合适的筛查工具：
   - PHQ-9(抑郁筛查)：评分22分(重度抑郁)
   - GAD-7(焦虑筛查)：评分18分(重度焦虑)
2. 完成量表评估：详细回答每个问题
3. 计算评分：PHQ-9 22分，GAD-7 18分
4. 判断严重程度：重度抑郁，重度焦虑
5. 给出建议：立即专业干预，有自杀风险

**输出**: 心理健康筛查报告

#### 1.2 症状分析与鉴别 (H3-01)
**输入**: 主要症状(焦虑，抑郁，失眠，自杀念头)，持续时间(2个月)
**执行**:
1. 详细描述症状：
   - 情绪症状：持续悲伤，焦虑，易怒
   - 躯体症状：失眠，食欲下降，乏力
   - 认知症状：注意力不集中，自我否定
   - 行为症状：工作效率降低，社交回避
   - 自杀念头：有，但无具体计划
2. 分析症状特征：符合重度抑郁发作伴焦虑
3. 列出可能诊断：
   - 重度抑郁发作
   - 广泛性焦虑障碍
   - 适应障碍
4. 鉴别诊断要点：排除双相障碍，躯体疾病所致
5. 建议进一步检查：甲状腺功能，心理评估

**输出**: 症状分析报告

#### 1.3 急诊判断 (H3-07)
**输入**: 症状描述，症状严重程度，发病时间
**执行**:
1. 评估症状严重程度：有自杀念头，重度抑郁
2. 判断是否需急诊：需要紧急评估
3. 急诊前准备：联系心理危机热线，准备就医
4. 就医指导：前往精神卫生中心急诊
5. 紧急处理措施：24小时监护，移除危险物品

**输出**: 急诊判断指南

**校准点1**: 危机确认
- 确认心理危机严重程度
- 确认自杀风险等级
- 确认紧急干预必要性
