# 染色体畸变平衡性判定

## 何时使用
- 在染色体核型解读中发现结构异常（如相互易位、倒位、缺失、重复等）
- 需评估携带者是否存在临床表型或生育风险
- 表型异常但核型显示“平衡”时，判断是否需进一步分子检测

## 执行步骤

1. **初步分类畸变类型**
   - 根据核型结果识别畸变类型（如 del、dup、inv、t、r、i 等）
   - 记录检测方法（常规显带、高分辨、FISH、array CGH）

2. **判定是否为平衡畸变**
   - **平衡畸变标准**：
     - 无遗传物质净增减
     - 仅基因位置或顺序改变
     - 典型类型：相互易位、倒位、插入（无片段丢失/增加）
     - 携带者通常无临床表型
   - **非平衡畸变标准**：
     - 存在遗传物质净增或净减
     - 导致基因剂量失衡
     - 典型类型：缺失、重复、环状染色体、等臂染色体、双着丝粒染色体
     - 常表现为发育迟缓、畸形等染色体病综合征

3. **处理特殊情况**
   - 若表型异常但核型显示“平衡”，建议进行分子检测（如 array CGH 或 FISH）以排除微缺失/微重复
   - 注意嵌合体现象：可能同时存在平衡与非平衡细胞系
   - 警惕“假平衡”：染色体水平看似平衡，但分子水平存在致病性 CNV

4. **输出临床建议**
   - 若判定为平衡畸变：提供遗传咨询与生育风险指导
   - 若为非平衡畸变或表型与核型不符：推荐分子遗传学检测以明确诊断
