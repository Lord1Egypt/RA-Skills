## Description: <br>
Guides users through a project development template when they want to define a new project, describe requirements, or create an application. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[2907555270](https://clawhub.ai/user/2907555270) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, product teams, and external users use this skill to turn an early project idea into structured project overview, requirements, architecture, API, and change-record templates. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can trigger broadly and persistently update project documents without clear user confirmation. <br>
Mitigation: Require explicit confirmation before entering the workflow and preview any main.md or requirements.md changes before writing files. <br>
