# 地支互动关系表
在查询地支互动和十二长生时，必须参照参照此表

| 关系类型 | 具体组合 | 核心意象与化气结果 |
| :--- | :--- | :--- |
| **六合** | 子丑合土，寅亥合木，卯戌合火，辰酉合金，巳申合水，午未合土/火 | 亲近、羁绊、合作。天干有引化时可化为对应五行。 |
| **暗合** | 寅丑暗合，午亥暗合，卯申暗合 | 暗中苟合、私下交易、无形的牵制与压力。 |
| **三合** | 申子辰合水，亥卯未合木，寅午戌合火，巳酉丑合金 | 长远同盟。由长生、帝旺、墓库组成，中神（子午卯酉）为核心。 |
| **三会** | 亥子丑会水，寅卯辰会木，巳午未会火，申酉戌会金 | 抱团排外、气势最强。凑齐一方之气，力量大于三合与六合。 |
| **六冲** | 子午冲，丑未冲，寅申冲，卯酉冲，辰戌冲，巳亥冲 | 剧烈冲突、走动、破裂。方位相对，五行相克（辰戌丑未为本气冲）。 |
| **六害(穿)** | 子未害，丑午害，寅巳害，卯辰害，申亥害，酉戌害 | 暗中破坏、互相倾轧。破坏了原有的六合局，防不胜防。 |
| **相刑** | **无恩**：寅巳申；**恃势**：丑戌未；**无礼**：子卯；**自刑**：辰午酉亥 | 纠纷、折磨、病痛、法律问题或自我内耗。 |
| **相破** | 子酉破，午卯破，巳申破，寅亥破，辰丑破，戌未破 | 内部结构的破裂、中途打断、局部的不完美。 |
