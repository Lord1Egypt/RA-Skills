# 儿童计划免疫规划

## 适用场景
- 制定儿童免疫接种时间表
- 评估公共卫生系统中的免疫覆盖率
- 解释国家免疫规划的科学依据

## 执行步骤
1. **确认适用对象**：仅适用于儿童、人群及公共卫生系统；不适用于成人常规免疫或非传染性疾病预防。
2. **理解核心定义**：计划免疫是根据小儿免疫特点和传染病流行情况制定的系统性免疫程序。
3. **明确实施方式**：通过有计划地使用经验证的生物制品（疫苗）进行预防接种。
4. **识别关键要素**：
   - 基于婴幼儿免疫系统发育阶段
   - 结合特定传染病的流行病学特征（如发病率、传播模式）
   - 采用标准化疫苗接种时间安排
5. **达成最终目标**：提高人群免疫水平，控制乃至消灭目标传染病。

> **注意**：使用前需掌握小儿免疫特点及当地传染病流行数据。
