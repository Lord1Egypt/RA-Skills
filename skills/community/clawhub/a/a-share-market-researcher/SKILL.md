# A股市场研究助手（Market Researcher CN）

基于 Anthropic `sector-overview` + `idea-generation` 原版架构本土化改造，
融合四层信号框架（政策/资金/景气/情绪），适配A股行业动态追踪与投资标的筛选场景。

## Overview

创建全面的**A股行业研究报告**，涵盖行业动态、竞争格局、关键参与者和主题趋势，
并基于四层信号框架标记风险与机会。

**报告规格：**
- 深度报告：20-30页（行业深度）
- 快速概览：5-10页（热点解读）
- 核心产出：行业景气分析 + 重点标的清单 + 风险预警标记

---

## When to Use

**适用场景：**
- "帮我研究新能源汽车行业现状和投资机会"
- "半导体板块最近政策变化怎么看"
- "消费医疗行业竞争格局分析"
- "帮我筛选[行业]中有50%上涨空间的个股"
- "这个板块最近资金流向和情绪怎么样"

**不适用场景：**
- 单只股票深度分析 → 用财报审阅助手
- 宏观经济分析 → 用宏观研究模板
- 已持仓股票跟踪 → 用组合监控工具

## Process

### 阶段1：四层信号扫描（30-60分钟）

> 在进入传统行业分析前，先运行四层信号框架快速定位行业所处位置。

#### 1.1 政策信号扫描（Policy Signal）

**信号来源：**
- 国务院/发改委/工信部官网最新文件
- 行业主管部委官方通知（如能源局/卫健委/证监会）
- 全国人大/政协相关提案与报告
- 地方政府配套政策（重点：北京/上海/广东/浙江）

**评分维度：**

| 维度 | 内容 | 信号强度 |
|-----|-----|---------|
| 政策方向 | 鼓励/中性/限制 | +2/0/-2 |
| 政策力度 | 国家级/部委级/地方级 | +2/+1/0 |
| 政策时效 | 近3个月/近半年/更早 | +2/+1/0 |
| 执行进度 | 落地实施/发文/规划 | +2/+1/0 |

**政策信号总分**：___/8分

**标记规则：**
- 🟢 6-8分：政策强力支持，短期催化明确
- 🟡 3-5分：政策中性或方向不明
- 🔴 0-2分：政策收紧或面临监管压力

---

#### 1.2 资金信号扫描（Capital Signal）

**信号来源：**
- 北向资金（沪深港通）净流入数据（Wind/同花顺）
- 融资余额变化趋势（东方财富/同花顺）
- 行业ETF资金流向（各大ETF基金公告）
- 机构持仓变化（基金季报/年报）
- 大宗交易记录（Wind大宗交易数据）

**评分维度：**

| 维度 | 内容 | 信号强度 |
|-----|-----|---------|
| 北向资金 | 连续净流入/流出 | +2/-2 |
| 融资余额 | 持续增加/持续下降 | +2/-2 |
| ETF规模 | 明显扩张/收缩 | +1/-1 |
| 机构仓位 | 超配/标配/低配 | +2/0/-2 |

**资金信号总分**：___/8分

---

#### 1.3 景气信号扫描（Prosperity Signal）

**信号来源：**
- 行业PMI数据（国家统计局/财新）
- 行业月度产销数据（工信部/行业协会）
- 中观高频数据（如：发电量/货运量/商品房销售面积/挖掘机开工小时）
- 上下游价格走势（Wind价格库）
- 上市公司盈利预期修订方向

**评分维度：**

| 维度 | 内容 | 信号强度 |
|-----|-----|---------|
| PMI走势 | 连续回升/下行 | +2/-2 |
| 量价趋势 | 量价齐升/量价齐跌 | +2/-2 |
| 高频数据 | 持续改善/恶化 | +1/-1 |
| 盈利预期 | 持续上调/下调 | +2/-2 |

**景气信号总分**：___/8分

---

#### 1.4 情绪信号扫描（Sentiment Signal）

**信号来源：**
- 板块换手率（相对历史分位数）
- 板块PE/PB历史分位（Wind估值库）
- 新媒体热度（微博/微信/雪球热搜）
- 分析师评级分布（一致预期看多/看空比例）
- 散户情绪指标（融资盘比例/龙虎榜数据）

**评分维度：**

| 维度 | 内容 | 信号强度 |
|-----|-----|---------|
| 估值分位 | <30%分位（低估）/>70%分位（高估） | +2/-2 |
| 换手率 | 温和放量/极度热炒 | +1/-1 |
| 分析师情绪 | 普遍悲观（逆向买点）/狂热看多 | +1/-1 |
| 新媒体热度 | 刚升温/已过热 | +1/-1 |

**情绪信号总分**：___/8分

---

#### 四层信号汇总

| 信号层 | 得分 | 信号方向 | 描述 |
|-------|-----|---------|-----|
| 政策信号 | /8 | 🟢/🟡/🔴 | |
| 资金信号 | /8 | 🟢/🟡/🔴 | |
| 景气信号 | /8 | 🟢/🟡/🔴 | |
| 情绪信号 | /8 | 🟢/🟡/🔴 | |
| **综合评分** | **/32** | | |

**综合判断：**
- 24-32分：强烈看多，优先重点研究
- 16-23分：中性，结构性机会，精选个股
- 8-15分：谨慎，需等待信号改善
- 0-7分：规避，风险大于机会

### 阶段2：市场规模与行业结构（1-2小时）

#### 2.1 市场规模测算

| 指标 | 数据 | 来源 | 日期 |
|-----|-----|-----|-----|
| 行业总市场规模（亿元） | | | |
| 5年CAGR（历史）| | | |
| 预测CAGR（未来3年）| | | |

**注意事项：**
- 所有规模数据必须标注来源（行业协会/发改委/第三方研究机构）
- 区分TAM（总可寻址市场）与实际市场份额
- 注明数据日期，标记可能过时的预测

#### 2.2 产业链图谱

绘制产业链上中下游结构：
- 上游：原材料/核心零部件供应商
- 中游：制造/集成商/平台
- 下游：终端客户/应用场景
- 标注：价值量主要集中在哪个环节？

### 阶段3：竞争格局分析（1-2小时）

#### 3.1 市场集中度

- CR3/CR5（前3/前5家合计市占率）
- 行业竞争结构：垄断/寡头/充分竞争
- 市场份额变化趋势（谁在扩张，谁在收缩）

#### 3.2 重点公司梳理（前5-8家）

| 公司名称 | 股票代码 | 营收（亿元） | 市占率 | 核心护城河 | 近期动态 |
|---------|---------|------------|-------|----------|---------|
| | | | | | |

**每家公司简述（3-5句）：**
- 核心业务定位
- 竞争优势（技术/品牌/渠道/规模）
- 当前估值（PE/PB，历史分位）
- 近期重要动态（并购/新品/政策受益）

#### 3.3 国产替代/进口替代进度（适用时）

| 细分领域 | 国产化率 | 主要国产厂商 | 进展评估 |

| | | | |

### 阶段4：风险标记（30-60分钟）

> 这一步是A股市场研究的关键差异点，必须明确标记各类风险。

#### 4.1 政策监管风险
- [ ] 行业是否面临反垄断调查风险
- [ ] 是否有价格管控/利润率压缩政策
- [ ] 是否存在资质/牌照收紧风险
- [ ] 出口管制/地缘政治风险（涉及进出口的行业）

#### 4.2 景气周期风险
- [ ] 当前处于景气周期的哪个阶段（上行/顶部/下行/底部）
- [ ] 主要下行催化剂是什么
- [ ] 库存周期位置（去库存/补库存）

#### 4.3 竞争恶化风险
- [ ] 是否存在价格战迹象
- [ ] 新进入者对现有格局的冲击
- [ ] 跨界竞争者的影响

#### 4.4 基本面风险
- [ ] 行业整体ROE是否持续下滑
- [ ] 资产负债率是否过高
- [ ] 应收账款/库存异常堆积

**风险综合评级：**
- 🔴 高风险：存在2项以上重大风险
- 🟡 中风险：存在1-2项一般性风险
- 🟢 低风险：风险可控

---

### 阶段5：投资标的筛选（30-60分钟）

基于前述分析，从行业中筛选具有**50%以上上涨空间潜力**的标的：

#### 5.1 筛选标准

**基本面标准（必须全部满足）：**
- [ ] 营收增速 > 行业平均增速
- [ ] 净利润率稳定或改善
- [ ] 经营性现金流为正且质量良好
- [ ] ROE > 10%（或处于改善趋势）

**估值标准（至少满足1项）：**
- [ ] 当前PE < 历史中枢50%分位
- [ ] PEG < 1（PE/净利润增速 < 1）
- [ ] PB < 行业均值 且 ROE > 行业均值（价值洼地）

**催化剂标准（至少存在1个）：**
- [ ] 有明确的盈利拐点预期（如新产品放量/订单爆发）
- [ ] 四层信号评分 ≥ 20分
- [ ] 近期重大政策利好直接受益方

#### 5.2 标的清单输出

| 股票代码 | 公司名称 | 当前价格 | 目标价 | 上涨空间 | 核心逻辑 | 主要风险 |
|---------|---------|---------|-------|---------|---------|---------|
| | | | | % | | |

### 阶段6：报告生成（1小时）

**报告结构（深度版20-30页 / 快速版5-10页）：**

```
封面：行业名称 + 四层信号综合评级 + 报告日期

第1页：核心观点摘要
  - 四层信号评分卡
  - 3-5条核心结论
  - 重点关注标的（3-5只）

第2-4页：四层信号详细分析
  - 政策/资金/景气/情绪逐层解析

第5-8页：行业深度分析
  - 市场规模与结构
  - 产业链图谱
  - 竞争格局

第9-12页：重点公司逐一分析

第13-14页：风险提示（分层次）

第15页：标的筛选清单与目标价测算

附录：数据来源清单
```

**数据标注要求（⭐ 强制）：**
- 所有行业数据标注来源机构 + 日期
- 估值数据标注查询日期（Wind/Choice）
- 一致预期标注来源平台 + 样本分析师数量

## Common Rationalizations（常见借口警示）

| 借口 | 正确做法 |
|-----|---------|
| "政策信号不好量化" | 按四层信号框架逐项评分，给出理由 |
| "市场规模预测差别很大" | 引用多个来源，给出区间估计，说明差异原因 |
| "找不到好的标的" | 如果筛选后确实没有50%空间标的，如实说明，不要凑数 |
| "资金数据实时性不够" | 标注数据截止日期，提醒读者自行验证最新数据 |

---

## Red Flags（执行异常警告）

- 🚨 四层信号中有2层以上为🔴 → 重新评估是否推荐该行业
- 🚨 找不到行业市场规模数据来源 → 不得捏造，说明数据缺失
- 🚨 筛选标的PE超历史70%分位 → 不满足估值安全边际，不纳入推荐
- 🚨 政策信号与资金信号严重背离 → 重点分析矛盾原因

---

## Verification（可验证的证据要求）

完成后必须能回答：
1. 四层信号综合得分是多少？各层信号依据是？
2. 行业市场规模数据来源是哪个机构？发布时间？
3. 筛选出的标的，目标价测算方法和依据是什么？
4. 主要风险项中，哪个风险在当前最需要关注？
5. 所有数据的最新更新日期？
