# 尿液颜色异常的鉴别诊断

## 触发条件
当尿液呈现以下非典型颜色之一时启动本流程：红色、深黄色、白色、深棕色/黑色、蓝色、绿色。

## 执行步骤

1. **确认前提条件**：
   - 已排除常见食物或药物导致的颜色干扰（如维生素B2致亮黄、利福平致橙红）。
   - 使用新鲜尿液样本，避免因久置氧化（如胆红素转为胆绿素）造成误判。

2. **按颜色分类鉴别**：
   - **红色尿液**：区分血尿（云雾状、带凝块）、血红蛋白尿（暗红/酱油色）、肌红蛋白尿（粉红/暗红）和卟啉尿（红葡萄酒色）。
   - **深黄色尿液**：考虑胆红素尿（豆油样、泡沫黄、胆红素定性阳性），注意久置后可变棕绿。
   - **白色尿液**：鉴别乳糜尿/脂肪尿（乳白有光泽、乳糜试验阳性）、脓尿/菌尿（浑浊）、结晶尿（灰白/淡粉红，加酸或加热后澄清）。
   - **深棕/黑色尿液**：排查重症血尿、黑尿酸症、酚中毒或黑色素瘤。
   - **蓝色尿液**：考虑蓝尿布综合征、靛青代谢异常或特定药物影响。
   - **淡绿色尿液**：警惕铜绿假单胞菌感染或服用吲哚美辛、亚甲蓝等药物。

3. **整合辅助检查**：
   - 结合尿液透明度、化学检测（胆红素、尿胆原）、显微镜检查（红细胞、细菌、结晶）结果综合判断。

4. **输出建议**：
   - 明确最可能的病理类型。
   - 推荐进一步检查项目（如显微镜检查、胆红素定性试验、乳糜试验等）。

> 注意：陈旧尿液颜色可能失真，务必使用新鲜样本；所有初步判断均需实验室确认。
