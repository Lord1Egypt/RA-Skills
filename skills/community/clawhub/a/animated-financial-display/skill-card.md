## Description: <br>
Patterns for animating financial numbers with spring physics, formatting, and visual feedback for financial dashboards and trading UIs. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and designers use this skill to add animated financial values, price tickers, percentage changes, metric cards, and formatted numbers to dashboards and trading interfaces. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The README includes an npx installation command that fetches from an external GitHub path. <br>
Mitigation: Confirm the repository is trusted before running the command, or install from the reviewed registry artifact. <br>
Risk: Animated financial values can distract users or make financial changes feel more significant than the underlying data supports. <br>
Mitigation: Use animation only on actual value changes, keep durations brief, preserve tabular numeric formatting, and avoid rapid decimal animation. <br>
