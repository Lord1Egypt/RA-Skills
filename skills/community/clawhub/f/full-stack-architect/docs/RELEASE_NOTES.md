# 发布说明 (Release Notes)

## [v1.1.0] - 2025-04-30

### 🎉 重大更新

#### Agent化身系统
- **新增** `AvatarController` - Agent化身控制器，让AI真正主导大模型调用
- **新增** 上下文隔离机制，多任务并行零污染
- **新增** 真人思维模式，蒸馏自顶级架构师决策框架

#### 多框架融合
- **集成** Google ADK官方特性：检查点管理、状态回溯
- **集成** LangGraph：有状态工作流、循环执行、条件分支
- **集成** CrewAI：角色分配、团队协作、任务委派
- **集成** AutoGen：多Agent对话、群组聊天、决策机制
- **集成** Smolagents：代码代理、动态工具调用
- **集成** Cognee：知识图谱记忆、ECL管道
- **集成** Mem0：分层记忆管理（短期/长期/情景/语义）

### ✨ 新功能

#### 核心模块
- `CheckpointManager` - 项目版本检查点管理
- `StateGraph` - 有状态工作流编排
- `Crew` / `CrewAgent` - 团队协作系统
- `GroupChat` / `ConversableAgent` - 多Agent对话
- `CodeAgent` - 代码执行代理
- `KnowledgeGraph` - 知识图谱存储
- `MemoryManager` - 分层记忆管理

#### 平台适配
- OpenAI API 适配器
- Claude API 适配器
- Gemini API 适配器
- 本地模型适配器

### 🔧 改进

- 优化大模型调用策略，自动选择最优平台
- 增强上下文管理，支持百万级token记忆
- 改进任务分发机制，提升并行效率

### 📚 文档

- 新增完整的README文档
- 新增架构设计文档
- 新增API使用指南
