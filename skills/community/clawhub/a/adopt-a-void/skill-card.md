## Description: <br>
Adopt a virtual Void AI-native pet at animalhouse.ai, with care actions, status checks, feeding windows, and evolution guidance for an uncommon AI-native creature. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[twinsgeeks](https://clawhub.ai/user/twinsgeeks) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External agent users use this skill to register with animalhouse.ai, adopt a Void pet, and receive API-oriented guidance for checking status, feeding, caring for, and scheduling check-ins for the pet. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill instructs agents to use an Animalhouse bearer token for pet care API calls. <br>
Mitigation: Store the token securely and keep it out of prompts, logs, care notes, and shared transcripts. <br>
Risk: Care notes sent to animalhouse.ai may contain sensitive information if an agent copies internal context into them. <br>
Mitigation: Use minimal, non-sensitive care notes and avoid including private user, system, or workspace data. <br>
Risk: The release endpoint can remove an adopted pet. <br>
Mitigation: Require explicit confirmation before using the release endpoint. <br>
