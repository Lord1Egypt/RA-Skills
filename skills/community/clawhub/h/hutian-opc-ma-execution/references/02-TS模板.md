# 投资意向书（TS/LOI）模板框架

> ⚠️ **法律声明**：本模板仅供参考，投资意向书通常不具有法律约束力（除特定条款外），实际使用需根据具体交易情况调整，并经专业律师审核。

---

## 标准投资意向书框架

### 协议主体

```
【投资意向书】

甲方（投资方）：[公司全称]
统一社会信用代码：[代码]
注册地址：[地址]
法定代表人：[姓名]

乙方（目标公司）：[公司全称]
统一社会信用代码：[代码]
注册地址：[地址]
法定代表人：[姓名]

丙方（现有股东）：[股东名称]（如适用）
身份证号/统一社会信用代码：[代码]

签订日期：[年]年[月]月[日]
```

### 核心条款框架

```
第一条 交易概述

1.1 交易类型
本意向书旨在记录各方就以下交易（以下简称"本交易"）的初步意向：
□ 股权转让
□ 增资扩股
□ 股权转让+增资
□ 其他：[具体描述]

1.2 交易标的
本交易的标的为乙方[股权比例]%的股权，对应注册资本[金额]万元。

1.3 交易背景
[简要描述交易背景，如行业情况、战略考量等]
```

```
第二条 交易条件

2.1 初步估值
各方初步商定，本交易项下乙方的企业估值如下：

| 估值方法 | 估值结果 | 备注 |
|---------|---------|------|
| 投前估值 | [X]万元 | - |
| 投后估值 | [X]万元 | - |
| 对应估值倍数 | [X]倍 | P/E |

2.2 融资金额
本交易的融资总额为人民币[大写]万元（[小写]万元）。

2.3 股权比例
本次交易完成后，甲方将持有乙方[比例]%的股权。

2.4 估值调整机制
各方理解并同意，最终估值将根据尽职调查结果进行调整，具体调整机制如下：
（一）若尽调发现重大风险或问题，估值下调幅度不超过[X]%
（二）若目标公司存在未披露负债，每出现[X]万元，估值下调[X]万元
（三）其他调整事项：[具体约定]
```

```
第三条 投资方案

3.1 投资方式
甲方以[现金/资产/混合]方式支付投资款。

3.2 支付安排
分期付款计划：
├── 第一期款：[X]%
├── 第二期款：[X]%（根据[里程碑事件]支付）
└── 第三期款：[X]%（根据[里程碑事件]支付）

3.3 资金用途
投资款主要用于：
├── 研发投入：[X]%
├── 市场拓展：[X]%
├── 团队建设：[X]%
└── 运营资金：[X]%
```

```
第四条 排他条款

4.1 排他期
乙方同意，在本意向书签暑之日起[X]日内（以下简称"排他期"）：

（一）除甲方外，乙方及其关联方不得与其他任何第三方就本交易相关事宜进行任何接触、谈判或签订任何协议；

（二）乙方应将所有与本交易相关的信息仅向甲方披露；

（三）乙方应全力配合甲方的尽职调查工作。

4.2 排他期延长
如因甲方原因导致尽职调查延期的，经乙方书面同意，排他期可延长[X]日。

4.3 违约责任
若乙方违反本条约定，应向甲方支付违约金人民币[大写]万元（[小写]万元）。
```

```
第五条 尽职调查

5.1 尽调安排
甲方将在本意向书签署后[X]日内启动尽职调查，乙方应全力配合。

5.2 尽调范围
尽职调查范围包括但不限于：
├── 财务尽职调查
├── 法律尽职调查
├── 业务尽职调查
├── 技术尽职调查（如适用）
└── 其他甲方认为必要的调查事项

5.3 尽调费用
尽职调查费用由甲方自行承担。
```

```
第六条 关键假设

本交易基于以下关键假设：

6.1 乙方不存在重大未披露负债、诉讼或行政处罚；

6.2 乙方的核心团队成员在交割时仍在公司任职；

6.3 乙方的主要业务合同在到期后能够正常续期；

6.4 乙方已取得的证照、资质在交易后持续有效；

6.5 目标公司所处行业未发生重大不利变化；

6.6 乙方知识产权权属清晰，不存在第三方主张。
```

```
第七条 核心条款

7.1 对赌安排（初步）
各方初步同意，本交易将设置以下业绩对赌条款：

| 指标 | 年度 | 承诺值 | 备注 |
|-----|------|--------|------|
| 净利润 | 第1年 | [X]万元 | - |
| 净利润 | 第2年 | [X]万元 | - |
| 净利润 | 第3年 | [X]万元 | - |

补偿机制：若未能达到业绩承诺，按照[约定方式]进行补偿。

7.2 竞业禁止
乙方实际控制人及核心团队成员应在交割后[X]年内履行竞业禁止义务。

7.3 董事会安排
交割后，甲方有权提名[X]名董事进入乙方董事会。

7.4 优先认购权
若乙方后续进行融资，甲方有权按照持股比例优先认购。

7.5 反稀释保护
若乙方后续融资估值低于本轮估值，甲方有权获得额外股权补偿。
```

```
第八条 锁定期与转让限制

8.1 锁定期
乙方现有股东同意，自交割之日起[X]年内不出售其持有的公司股权。

8.2 共同出售权
若乙方股东拟出售其持有的股权，在同等条件下，甲方有权按比例共同出售。

8.3 拖带权
若甲方出售其持有的公司股权，经[X]%以上其他股东同意，可要求其他股东共同出售。
```

```
第九条 时间表

| 里程碑 | 预计完成时间 | 责任方 |
|-------|------------|-------|
| 签署本意向书 | [日期] | 双方 |
| 完成尽职调查 | [日期] | 甲方/乙方配合 |
| 签署正式协议 | [日期] | 双方 |

```

```
第十条 保密义务

10.1 各方应对本意向书的内容及谈判过程严格保密，未经其他方书面同意，不得向第三方披露。

10.2 前述保密义务不适用于：
（一）法律或监管机构要求披露的信息；
（二）已公开的信息；
（三）向专业顾问披露的信息（须承担保密义务）。
```

```
第十一条 法律效力

11.1 本意向书自各方签署之日起生效。

11.2 本意向书以下条款具有法律约束力：
（一）保密条款（第十条）
（二）排他条款（第四条）
（三）法律适用与争议解决（第十三条）
（四）违约责任（第十一条）

11.3 除上述条款外，本意向书其他条款仅为各方目前的意向表示，不构成具有法律约束力的承诺，最终交易以各方正式签署的协议为准。
```

```
第十二条 违约责任

12.1 若甲方违反本意向书约定，导致本交易无法完成，甲方应赔偿乙方因此遭受的合理损失。

12.2 若乙方违反本意向书约定，导致本交易无法完成，乙方应向甲方支付违约金人民币[大写]万元（[小写]万元）。
```

```
第十三条 争议解决

13.1 本意向书适用中华人民共和国法律。

13.2 因本意向书引起的或与本意向书有关的任何争议，各方应友好协商解决。

13.3 协商不成的，任一方可向[被告住所地/合同签订地]有管辖权的人民法院提起诉讼。
```

```

14.2 本意向书自各方签署之日起生效，有效期至[日期]或正式协议签署之日（以较早者为准）。
```

## OPC特别注意事项

### 技术方谈判要点

```
【估值调整】
□ 争取设置估值下限保护
□ 明确尽调调整的计算公式
□ 争取"重大不利变化"的排除情形

【排他期】
□ 争取较短的排他期（建议≤45天）
□ 设置排他期延长的条件限制
□ 争取"分手费"条款（对方毁约赔偿）

【日落条款】
□ 设置本意向书的有效期（建议60-90天）
□ 超过有效期自动失效

【关键假设】
□ 争取将"核心团队稳定"作为条件而非假设
□ 明确假设变化时的重新谈判机制
```

### 常见陷阱与规避

| 陷阱类型 | 风险描述 | 规避建议 |
|---------|---------|---------|
| 排他期过长 | 被锁定而交易无法推进 | 控制排他期≤45天 |
| 过于详细估值 | 后期难以调整 | 设置估值调整公式 |
| 约束性条款过多 | 失去灵活性 | 仅保留必要约束条款 |
| 无分手费 | 对方毁约成本低 | 设置违约金条款 |
| 期限无限制 | 长期绑定无结果 | 设置日落条款 |
