# Xiabi 的人生军师系统

## 🎯 系统概述

这是一个**单一智能 Agent**系统，由 Claw 🐾 担任你的**人生军师**，基于你的全局情况（时间、精力、财务、家庭）做最优决策。

**核心理念**：你 40 岁，时间是最稀缺资源。需要的不是 4 个顾问，而是一个懂你全局的军师。

## 🚀 使用方法

### 1. 查看工作看板

打开 `dashboard.html` 文件，即可看到所有 Agent 的工作状态：

```bash
# 在浏览器中打开
start agents/dashboard.html
```

### 2. 更新工作日志

每天更新每个 Agent 的工作日志：

```bash
# 健康顾问
code agents/health_advisor/2026-03-04.md

# 育儿顾问
code agents/parenting_advisor/2026-03-04.md

# 创业顾问
code agents/startup_advisor/2026-03-04.md

# 投资顾问
code agents/investment_advisor/2026-03-04.md
```

### 3. 配置 Agent

编辑 `config.json` 文件来调整 Agent 设置：

```json
{
  "agents": [
    {
      "id": "health_advisor",
      "name": "健康顾问",
      "enabled": true,
      "schedule": {
        "daily_check": "08:00",
        "weekly_review": "Sunday 20:00"
      }
    }
  ]
}
```

## 🎯 使用建议

### 每日流程
1. **早上**：查看工作看板，了解今日任务
2. **白天**：执行任务，更新工作日志
3. **晚上**：复盘今日完成情况，规划明天

### 每周流程
1. **周日晚上**：查看每周复盘报告
2. **调整计划**：根据本周完成情况调整下周计划
3. **庆祝成就**：回顾本周的成就和进步

## 🔗 集成 OpenClaw

### 通过子代理系统
- 每个 Agent 可以作为一个子代理 session
- 通过 `sessions_list` 查看所有 Agent 状态
- 通过 `sessions_history` 查看每个 Agent 的工作记录

### 通过定时任务
- 配置 cron 任务，定时提醒每个 Agent 的工作
- 通过 `cron list` 查看所有定时任务
