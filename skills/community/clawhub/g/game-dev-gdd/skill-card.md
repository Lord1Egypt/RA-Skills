## Description: <br>
Guided Game Design Document writing using structured templates for section-by-section system, mechanics, and narrative design. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[toilanguyen2910](https://clawhub.ai/user/toilanguyen2910) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Game designers, developers, and teams use this skill to turn prompts into a comprehensive GDD covering gameplay, systems, narrative, UI/UX, audio, content, monetization, and technical specifications. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad document-writing triggers could activate the workflow for requests that are not specifically about game design. <br>
Mitigation: Confirm the request is game-related before applying the GDD workflow. <br>
Risk: Generated Markdown files could conflict with an existing gdd/ folder or existing design documents. <br>
Mitigation: Ask where to place the generated files and check for existing GDD or design documents before writing. <br>
