# cn-shuffle-lines

文本行随机排序工具。将文本行随机打乱顺序。

## 使用方法

```bash
python3 scripts/shuffle.py --text "第一行\n第二行\n第三行"
python3 scripts/shuffle.py --file input.txt
```

纯Python标准库。
