## Description: <br>
Memory Forge analyzes local Claude, ChatGPT, and Cursor conversation history for KPI statistics, cost tracking, topic distribution, and efficiency insights. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[melody1015](https://clawhub.ai/user/melody1015) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and AI tool users use this skill to inspect local conversation history, summarize cost and usage trends, and identify practical efficiency improvements. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill reads local AI conversation history, which may contain sensitive prompts, project details, or credentials. <br>
Mitigation: Install only when this local access is acceptable, prefer date or project filters for narrower reports, and review generated summaries before sharing. <br>
Risk: Optional dashboard commands install and run the external Memory Forge package. <br>
Mitigation: Verify the package and environment before running pip install memory-forge[all] or dashboard commands. <br>
