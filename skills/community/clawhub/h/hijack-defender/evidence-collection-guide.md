# Evidence Collection Guide — Listing Hijack Cases

## Why Evidence Quality Determines Enforcement Speed

Amazon resolves IP complaints on a tiered basis:
- **Tier 1 (resolved in 1–3 days):** Trademark + test buy + photos showing clear counterfeiting
- **Tier 2 (resolved in 3–7 days):** Trademark + screenshots showing unauthorized brand use
- **Tier 3 (resolved in 7–30 days or rejected):** Screenshots only, no trademark, vague description

Build Tier 1 evidence packages for every counterfeit case.

---

## Test Buy Protocol

### Before the Purchase
1. Create a dedicated "test buy" Amazon account separate from your brand account and any personal accounts. Use a neutral email address.
2. Use a shipping address that is not your business address (team member's home, P.O. Box, or package forwarding service).
3. Screenshot the listing before purchasing: show seller name, price, "Sold by" field, all listing images, and the date in the URL bar.
4. Note the specific offer you're buying (if multiple sellers exist, select the hijacker's offer explicitly).

### During the Order
- Screenshot the checkout screen showing: seller name, price, and estimated delivery.
- Note the order ID (format: 114-XXXXXXX-XXXXXXX).
- Screenshot the order confirmation email.

### Upon Delivery
Document the following the moment the package arrives — do not open fully until you're ready to photograph:

**Package exterior:**
- Photo of the package as received (all sides)
- Photo showing return address label (seller's ship-from location)
- Photo showing your order confirmation number on the packing slip

**Product comparison photos (authentic vs. hijacker):**
- Side-by-side packaging (front, back, sides)
- Logo comparison (color accuracy, font, placement)
- Barcode / UPC comparison (they often match but product inside differs)
- Any authentication features: holographic stickers, serial numbers, QR codes
- Inner packaging: foam, inserts, accessories, documentation
- Product itself: material quality, construction, color accuracy
- Any authenticity cards or certificates (present in authentic, absent or different in counterfeit)

**Product testing (if safe and applicable):**
- Weight comparison (counterfeits are often lighter)
- Performance test documentation (e.g., battery life, output quality)
- Ingredient or material discrepancy (for consumables or skincare)

### What to Preserve
- The entire package, product, and all materials in the original received state
- Digital photos organized by category (see above)
- Order confirmation email as PDF
- Tracking information from delivery

*Do NOT dispose of any test buy materials until the enforcement case is fully resolved and any legal threats have passed.*

## Screenshot Documentation Checklist

For every hijack incident, capture and label the following screenshots:

### Listing Screenshots
- [ ] Full listing page showing: title, brand, images, price, "Sold by [Hijacker Name]" field
- [ ] "All Offers" display showing the hijacker's offer selected
- [ ] Hijacker's seller profile page (storefront)
- [ ] Hijacker's feedback page (account age, feedback count, rating, recent reviews)
- [ ] Date visible in browser: use URL bar or open a second tab with google.com → search today's date

### Your Brand Documentation
- [ ] Your Amazon Brand Registry enrollment page (or confirmation email)
- [ ] Your product's ASIN on Amazon with you as the listed brand
- [ ] Your trademark registration certificate (from USPTO TESS or equivalent)
- [ ] Your product's manufacturer invoice or purchase order (shows you as the authorized seller)

### Post-Submission
- [ ] Amazon Brand Registry report submission confirmation (case number)
- [ ] Any Amazon responses to the case
- [ ] Seller's response to your cease-and-desist message (screenshot from Amazon messaging)

      order-confirmation.pdf

      [additional comparison photos]
    /brand-documentation/
      trademark-registration.pdf

      manufacturer-invoice.pdf
    /enforcement/
      amazon-report-confirmation.pdf

### Naming Convention
`[ASIN]_[SellerID]_[Date]_[EvidenceType]`

Example: `B08XYZ1234_A3SELLER987_2026-06-06_listing-screenshot.png`

## Amazon Brand Registry IP Report — Field Guide

**Where:** brand.amazon.com → Protect → Report a Violation

| Field | What to Enter |
|-------|---------------|
| Violation type | "Counterfeit" if product is fake; "Trademark" if brand name is used without authorization |
| ASIN | The ASIN of your listing being hijacked |
| Infringing product URL | Direct URL to the hijacker's offer page (copy from "All Offers" display) |
| Seller name | Exact seller name as shown on Amazon |
| Your trademark | Enter your U.S. trademark registration number (or international equivalent) |
| Evidence description | Brief factual description: "Test purchase conducted [date], order [ID]. Product received differs from authentic product in [specific ways]. Photos attached." |
| Attachments | Upload: listing screenshot, comparison photos, order confirmation |

**Tips for faster resolution:**
- Keep the description factual and specific — avoid vague language like "looks fake"
- Reference the test buy order ID explicitly
- Upload 4–6 clearly labeled photos, not 20 unlabeled images
- Submit separate reports for each seller (don't combine multiple sellers in one report)

## Escalation Paths

| Situation | Escalation Action |
|-----------|------------------|
| Amazon report rejected (no TM found) | Confirm TM registration status on USPTO.gov; resubmit with registration certificate attached |
| Amazon report rejected (insufficient evidence) | Conduct test buy, resubmit with test buy evidence |
| Case resolved but seller returns within 30 days | Resubmit with note "Repeat infringer — previously removed on [case #]" |
| Multiple sellers on same listing (organized attack) | Flag as potential coordinated infringement ring; request Counterfeit Crimes Unit referral |
| Seller is large enough to have legal representation | Escalate to your trademark attorney; consider federal court filing |
| Product appears on Alibaba/AliExpress as well | File separate takedown on those platforms; may indicate factory-level counterfeiting |
