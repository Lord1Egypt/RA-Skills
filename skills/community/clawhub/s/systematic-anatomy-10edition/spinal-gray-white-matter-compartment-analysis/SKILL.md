# 脊髓灰质与白质分区分析

## 何时使用
- 分析脊髓横切面图像或示意图
- 解释患者出现的感觉缺失、运动障碍等症状与脊髓节段的关系
- 定位脊髓病变（如外伤、肿瘤、炎症）的解剖层面
- 理解感觉与运动传导通路在脊髓内的走行与交叉位置

## 执行步骤

1. **确认脊髓节段**（如C5、T10、L2），注意侧角仅存在于T1–L2（交感）和S2–S4（副交感）。

2. **识别灰质结构**（中央，呈H形）：
   - **前角**：含躯体运动神经元，支配骨骼肌；损伤导致同侧下运动神经元性瘫痪。
   - **后角**：接收后根传入的感觉信息，含胶状质和后角固有核；参与痛觉、温度觉及部分触觉处理。
   - **侧角**（若存在）：含内脏运动节前神经元，参与自主神经调控。

3. **识别白质结构**（周围，分为三索）：
   - **后索**：含薄束与楔束，传导同侧本体感觉和精细触觉。
   - **外侧索**：含皮质脊髓侧束（同侧随意运动）和脊髓丘脑侧束（对侧痛温觉）。
   - **前索**：含皮质脊髓前束等，参与双侧躯干运动控制。

4. **检查连合结构**：
   - **白质前连合**：痛温觉纤维在此交叉至对侧。
   - **灰质前/后连合**：连接左右灰质角。

5. **关联临床表现**：
   - 若出现同侧运动与本体感觉障碍 + 对侧痛温觉丧失，提示**Brown-Séquard综合征**（脊髓半切），印证上述分区功能。

> 注意：本技能不适用于脑干及以上中枢结构，也不直接用于功能性MRI信号解读。
