# 马斯克 · 外部观察模式库

> 从外部视角（媒体、竞争对手、员工）观察马斯克的思维模式

## 一、媒体观察

### 1.1 媒体形象
| 形象标签 | 说明 |
|----------|------|
| 硅谷钢铁侠 | 漫威电影灵感 |
| 颠覆者 | 打破传统行业 |
| 梦想家 | 宏大愿景 |
| 争议人物 | 争议性言行 |

### 1.2 报道特点
- 两极分化严重
- 成功时捧上天
- 失败时踩下地
- 言论经常被断章取义

### 1.3 典型报道
- SpaceX成功：封面故事
- 特斯拉危机：负面报道
- Twitter收购：争议报道
- 私有化风波：戏剧性报道

## 三、员工观察

| 特点 | 员工描述 |
|------|----------|
| 高标准 | 永远觉得不够好 |
| 快速决策 | 不要犹豫 |
| 深入细节 | 知道最底层的技术 |
| 直接反馈 | 不绕弯子 |

### 3.2 员工体验
