# 民事答辩状

## 适用场景
收到法院送达的起诉状副本后，作为被告需要针对原告的诉讼请求和事实理由进行反驳时使用。

## 提交机构
受诉人民法院（提交至承办法官）

## 模板正文

```
                    民事答辩状

答辩人：姓名______，性别______，民族______，出生日期______
        身份证号：________________________
        住址：____________________________
        联系电话：________________________

因原告______诉答辩人______纠纷一案（案号：______），现提出答辩如下：

一、关于原告诉讼请求的答辩意见
原告主张______________________________的诉讼请求，答辩人认为该请求缺少事实和法律依据。理由如下：
__________________________________________________。

二、关于原告主张事实的反驳
原告在起诉状中声称______________________________，
但实际情况是____________________________________。
（逐点反驳原告主张的事实，提供相反证据或证明事实不存在）

三、答辩人的主张
综上所述，请求人民法院依法驳回原告的全部/部分诉讼请求。
（如有反诉，可在此一并提出）

此致
____________人民法院

附：
1. 本答辩状副本______份；
2. 证据材料______份。

                                    答辩人（签名）：
                                    ______年______月______日
```

## 注意事项
- **关键填写点**：写明案号便于法院归档；针对原告每一项诉讼请求和事实逐一回应；证据目录与答辩意见一一对应
- **常见错误**：仅情绪化反驳而不提供证据支撑；超过答辩期限（收到起诉状副本后15日内）提交；遗漏反诉请求
- **份数要求**：法院1份正本 + 原告人数对应份数的副本
- **时限提示**：收到起诉状副本后15日内提交答辩状，超过期限虽可提交但不免除答辩义务
