# 短视频爆款选题方法论

## 选题来源矩阵

### 1. 平台数据挖掘
- **抖音热搜榜**：实时 + 7日榜，筛选与账号垂类相关的话题
- **巨量算数**（bytedance.com/app/toutiao/data）：关键词热度趋势
- **快手磁力引擎**：快手平台数据工具
- **微信指数**：判断话题在微信生态的热度

### 2. 竞品账号分析
- 找同类TOP账号，按点赞量排序，逆向拆解其爆款
- 关注点：标题规律、封面风格、时长分布、开头方式
- 工具：飞瓜数据、新榜、卡思数据

### 3. 长尾内容挖掘
- 搜索目标关键词，看"相关搜索"和"猜你想搜"
- 评论区高赞评论 = 用户真实需求未被满足的地方
- 知乎/小红书热门问题 → 转化为视频选题

## 爆款标题公式

### 悬念式
- "我用 X 天做到了 Y，秘诀只有一个"
- "没人告诉你的 X 个真相"
- "为什么 XX 人都在这么做？"

### 数字式
- "X 个让你 [效果] 的 [方法]"
- "[时间] 内 [达成目标]，只需这 X 步"
- "学会这 X 招，[改变/收益]"

### 对比式
- "同样是 XX，为什么他/她能 YY？"
- "做 XX 的人和不做 XX 的人，10年后差距有多大"
- "普通人 vs 厉害的人，处理 XX 的方式差在哪"

### 痛点式
- "你以为你在 XX，其实你在 YY"
- "一直 XX，难怪你 YY"
- "不是你不努力，是你不知道 XX"

## 账号定位与内容矩阵

```
账号定位 = 垂直领域 + 目标人群 + 差异化风格

内容矩阵示例（职场成长号）：
├── 引流款（50%）：通用职场干货，选题大众，易传播
├── 留存款（30%）：深度内容，建立专业形象，促关注
└── 变现款（20%）：课程/服务软植入，转化粉丝
```

## 数据指标参考

| 指标 | 健康值 | 说明 |
|------|--------|------|
| 完播率 | >40% | 核心算法指标 |
| 点赞率 | >3% | 内容认可度 |
| 评论率 | >0.5% | 互动深度 |
| 转发率 | >1% | 传播价值 |
| 关注转化 | >0.5% | 账号吸引力 |

## 内容生产SOP

**Day 1（选题）**
- [ ] 浏览热榜，记录10个备选选题
- [ ] 检查竞品同类内容近期表现
- [ ] 确定3个本周要做的选题

**Day 2（脚本）**
- [ ] 写出核心论点（一句话）
- [ ] 设计3个Hook版本，选最强的
- [ ] 完成完整脚本初稿
- [ ] 大声朗读审核节奏

**Day 3（拍摄/剪辑）**
- [ ] 准备道具、场景、服装
- [ ] 按脚本顺序拍摄素材
- [ ] 粗剪 → 精剪 → 调色 → 字幕 → 音乐

**Day 4（发布）**
- [ ] 选择最佳发布时间（晚7-9点/午12点）
- [ ] 撰写发布文案+话题标签
- [ ] 首小时内回复前20条评论
