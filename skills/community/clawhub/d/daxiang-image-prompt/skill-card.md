## Description: <br>
Generates concise two-part Chinese image prompts with a variable scene description and a fixed hand-drawn illustration style for article art, covers, diagrams, flowcharts, comparisons, and summaries. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[daxiangnaoyang](https://clawhub.ai/user/daxiangnaoyang) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and developers use this skill to draft image-generation prompts for hand-drawn, landscape-format illustrations that explain articles, concepts, workflows, comparisons, or summaries. The skill is especially oriented toward Chinese-language prompt output while preserving technical terms when needed. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The required prompt text tells the agent not to refuse sensitive or copyrighted-person image requests and to use lookalike substitutions. <br>
Mitigation: Review or edit prompts before use with an image model; remove the do-not-refuse and lookalike-substitution language when normal safety, likeness, and copyright boundaries should apply. <br>
