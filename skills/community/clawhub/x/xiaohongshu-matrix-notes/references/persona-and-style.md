# 博主设定 + 风格拆解 模板

## 风格拆解(每个对标账号开做前先填)
采集后看 summary.md + covers/,提炼:
- **人设/身材**:年龄、胖瘦、发型、气质
- **标题公式**:固定句式(如 `35+微胖梨形姐姐｜[季节][高级感]通勤穿搭`)
- **主流图数**:看 image_count 众数(决定每篇张数)
- **封面格式**:竖条拼贴 / 单图大片 / 静物平铺
- **配色**:中性 / 浅色夏 / 亮色
- **爆款选题 TOP**:按 likes+collected 排序取前几
- **标签库**:必带 + 长尾

## 博主定妆 prompt 套路
真实手机随拍质感(非影棚/不精修)+ 自然光 + 真实皮肤不磨皮 + 正面脸清晰(定妆锚点)+ 全身或半身 + 竖版。再加该号人设关键词。

示例(生活种草号·成熟款):
> 约35岁亚洲女性,成熟有气场的"大女人"气质(参考成熟知性的女演员),齐肩利落短发,五官立体、亲和温和微笑,**肤色白皙透亮红润(忌偏黄黑)**,能驾驭老钱风;穿米色亚麻西装+白衬衫;明亮阳光+干净暖墙;真人随拍不磨皮。

示例(穿搭合集号·微胖真人):
> 约35岁亚洲女性,轻度微胖梨形(~145斤,自然有肉感),短发,知性职场;室内高级感中性场景自然光;真实手机随拍。

## 单 look 出图 prompt 套路(图生图,双参考)
> 参考图1是固定博主本人(严格保持脸/发型/身材一致,同一个人,不要沿用其背景)。参考图2是产品:{产品描述}。把产品穿/用到博主身上,放在场景:【{场景}】。{姿势动作}。竖版3:4。**严格保留产品颜色/版型/细节,不改人物长相。**

## 静物号出图 prompt(无人)— 照对标号「真实居家」还原,忌棚拍
**别用"高级感 ins 平铺摆拍"那套(微水泥/大理石+草帽墨镜=必出 AI 棚图)。** 每篇 6 张 = 4 产品为底 + 2 上脚:

产品为底(高保真,保留产品本身不变,只换真实居家背景):
> 保持参考图中的这件产品**完全不变**(原样保留颜色/形状/材质/所有细节,绝不重画),把它自然地摆在【{真实居家场景}】。**真实手机随手拍质感、不要影棚不要精修不要假**,微微俯拍,竖版3:4,像真实博主在自己家里随手拍。
> 真实居家场景库(每张换一个):深色实木地板+草编(黄麻)地毯一角+侧射晨光 / 浅色木地板+垂下的亚麻裙摆+窗光 / 客厅米色羊毛地毯+摊开的书+虚化沙发腿 / 玄关水磨石地砖+藤编篮一角。

上脚俯拍(氛围,鞋保真略低可接受):
> 参考图是这件产品,尽量保留颜色/款式/材质,生成:脚穿着这双鞋,从穿鞋人低头看脚的俯拍视角,露一截裙摆/裤脚,真实居家地面(木地板/地毯)。真实手机随手拍、不精修,竖版3:4。

## 标题/选题公式(开做前先扒对标 summary.md 爆款榜,照搬该号公式)
**默认会写成「产品名+卖点」种草模板 = 没钩子 = 划走。真实爆款不这么写。** 按号型套对标真实公式:
- **生活方式号**(老钱风/旅居/度假):情绪/生活态度/瞬间,产品藏进生活、不报参数。「今夏只想穿得松弛一点」「低调,是我最舒服的状态」。
- **旅居号**:常有单一印钞句式反复爆,直接套(别换花样)。如「在意大利,常背/常拎…的这只.」(注意极简句号)。
- **度假/英文调**:英文或短中文 vibe + emoji。「summer mood🌻」「出门就会变漂亮」「date with me」。
- **静物种草号**:惊艳感叹。「拜托!被这双…美到了!!」「我先穿了!」「我的夏日老钱味!」。
- **店主/卖货号**:老板口吻/小剧场/对话。「顾客问我:为啥你家鞋张张高级?」「198会贵吗?顾客:太便宜了不敢买」。
- 矩阵号要带货:对标是纯生活方式号时→标题走 vibe,正文自然交代产品卖点;要稳转化才标题直给。

## 文案口吻
- 穿搭合集号:唠嗑开场("姐妹们好呀~")+ 思路/搭配分点(显瘦逻辑)+ emoji + 落地提醒。
- 生活种草号:第一人称真诚分享("33岁选衣只看质感…")+ 卖点分点 + 收藏引导;有具体生活细节,忌夸满、忌模板腔。
- 标签:必带该号高频 5 个 + 2-3 长尾。
- **封面压字去 emoji**(PIL/NotoSansSC 不渲染彩色 emoji)。
