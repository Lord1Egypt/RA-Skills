# 膀胱叩诊鉴别诊断法

## 适用场景
- 膀胱触诊结果不明确
- 怀疑患者存在尿潴留但尚未确诊
- 需在导尿或排尿前快速评估膀胱充盈状态

## 执行步骤

1. **准备条件**：确保患者未排尿或尚未导尿，且耻骨联合上方区域可叩诊。
2. **叩诊操作**：从耻骨联合上方由上向下轻叩，注意听辨音调变化。
3. **判断结果**：
   - **正常（膀胱空虚）**：因肠管位于耻骨上方，叩诊呈鼓音，无法叩出膀胱轮廓。
   - **异常（膀胱充盈）**：耻骨上方出现圆形浊音区，其弧形上缘**凸向脐部**（关键特征）。
4. **鉴别诊断**：
   - 若浊音区上缘**凹向脐部**，提示腹腔积液而非膀胱胀大。
   - 妊娠子宫、子宫肌瘤或卵巢囊肿也可致浊音，需结合病史及妇科检查排除。
5. **确诊验证**：指导患者排尿或行导尿后复查叩诊；若浊音区转为鼓音，则确认为尿潴留所致膀胱胀大。

## 注意事项
- 假阳性可能由妊娠、盆腔肿块或腹水引起，务必结合临床背景综合判断。
- 浊音区形态（凸 vs 凹）是区分膀胱胀大与腹腔积液的核心依据。
