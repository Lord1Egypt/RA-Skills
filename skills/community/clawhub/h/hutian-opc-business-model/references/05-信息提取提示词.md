# 信息提取提示词

## 概述

本文件提供从商业计划书中提取各维度信息的提示词模板，用于指导AI准确提取所需信息。

## 提取原则

1. **忠于原文**：严格提取原文内容，不做推测
2. **标注来源**：每项信息标注原文位置
3. **如实记录**：找不到的信息标记"未提取到"
4. **分类整理**：按九宫格维度分类整理

## PESTL维度提取提示词

### P-政治环境提取

```
请提取以下信息（严格基于原文）：
1. 政策支持
   - 国家/地方有哪些扶持政策？
   - 是否有补贴、税收优惠？
   - 政府采购机会？

2. 政府态度
   - 政府对行业的态度？
   - 是否有政府合作案例？
   - 政企关系如何？

3. 监管趋势
   - 行业监管政策走向？
   - 监管是趋严还是放松？
   - 合规要求有哪些？

4. 产业政策
   - 专项产业政策？
   - 发展规划？
   - 禁止或限制事项？

提取示例格式：
- 政策1：[原文引用]（来源：第X页/第X段）
- 政策2：[原文引用]（来源：第X页/第X段）
- 未提取到：[维度描述]
```

## 注意事项

1. **逐条提取**：每项信息单独提取，便于追溯
2. **标注来源**：必须标注原文位置
3. **原文引用**：保留原文表述，不做二次加工
4. **如实记录**：未找到的信息明确标注
5. **不编造**：严禁推测或补充原文未提及的信息
