# 子宫内膜癌分子分型指导治疗

## 适用条件
- 患者已确诊为子宫内膜癌
- 已完成病理诊断
- 可获取肿瘤组织用于基因检测
- 不适用于无法获得足够组织样本的患者

## 执行步骤

1. **进行全面分子检测**：对所有符合条件的新确诊患者，检测以下四项指标：
   - POLE基因致病性突变
   - MMR蛋白表达（或微卫星不稳定性MSI）
   - p53免疫组化状态
   - HER2表达（辅助判断）

2. **根据结果划分为四型并制定策略**：
   - **POLE超突变型**：存在POLE致病突变 → 预后最佳，即使高危病理特征也可考虑降级治疗（减少放化疗强度）。
   - **dMMR型**（错配修复缺陷）：MMR蛋白缺失或MSI-H → 预后中等；晚期/复发患者推荐PD-1/PD-L1抑制剂免疫治疗。
   - **NSMP型**（无特异分子改变）：POLE野生型、MMR完整、p53正常 → 预后中等；按传统临床病理因素（如分期、分级、肌层浸润）决策治疗。
   - **p53异常型**：p53弥漫强阳性或完全阴性 → 预后最差，侵袭性强；应强化辅助治疗（如含铂化疗），患者从中显著获益。

3. **林奇综合征筛查**：所有患者均需行MMR蛋白免疫组化。若结果异常（dMMR），应进一步行胚系基因检测以排除林奇综合征。

4. **优先使用分子分型**：分子分型优于传统组织学分型，因其不依赖形态学，能更精准预测预后并指导个体化治疗决策。

> 注意：最终治疗方案需结合FIGO临床分期、患者体能状态及其他合并症综合判断。
