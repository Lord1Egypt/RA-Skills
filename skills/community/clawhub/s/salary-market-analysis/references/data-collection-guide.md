## 方案 1：全网搜索 + 手动收集 ⭐ 推荐

**适用场景**：最稳定，推荐首选
**效率**：★★★
**技术要求**：无

### 操作步骤

1. **使用 web-search-plus 搜索**
   ```bash
   python3 skills/web-search-plus/scripts/search.py -q "[岗位名称] 薪酬 薪资 [城市]" --count 10
   ```

2. **逐条查看搜索结果**
   - 打开链接查看薪酬信息
   - 提取关键数据（岗位、薪酬区间、公司、地点等）

3. **填写 CSV 模板**
   ```bash
   # 复制模板
   cp assets/salary_data_template.csv data.csv
   # 手动填写数据
   ```

4. **数据验证**
   - 检查薪酬口径是否一致（月薪/年薪）
   - 确认数据时效性（近 6 个月内）
   - 标记数据来源

### 优点
- 最稳定，不易被封
- 数据质量可控
- 无需技术配置

### 缺点
- 效率较低
- 需要人工判断

## 方案 2：Tampermonkey 用户脚本

**适用场景**：批量采集招聘网站
**效率**：★★★★
**技术要求**：低

### 配置步骤

1. **安装 Tampermonkey**
   - Chrome 扩展商店搜索 "Tampermonkey"
   - 安装并启用

2. **创建用户脚本**
   ```javascript
   // ==UserScript==
   // @name         薪酬数据收集
   // @match        https://www.zhipin.com/*
   // @grant        none
   // ==/UserScript==

   // 提取薪酬数据逻辑
   ```

3. **运行脚本**
   - 打开招聘网站搜索页面
   - 脚本自动提取数据
   - 导出为 CSV

### 支持网站
- BOSS 直聘
- 猎聘
- 前程无忧
- 智联招聘

## 方案 5：Playwright 自动化 ⭐ 高效

**适用场景**：大批量自动化采集
**效率**：★★★★★
**技术要求**：高

### 安装配置

```bash
# 安装 Playwright
pip install playwright
playwright install chromium

# 运行自动化脚本
python3 scripts/auto_collector.py --config config.json
```

### 示例配置

```json
{
  "targets": [
    {
      "url": "https://www.zhipin.com/web/geek/job?query=HRBP&city=101280600",
      "selector": ".job-card-wrapper",
      "fields": {
        "position": ".job-title",
        "salary": ".salary",
        "company": ".company-name",
        "location": ".job-area"
      }
    }
  ],
  "output": "data.csv",
  "delay": 3
}
```

### 注意事项
1. **遵守 robots.txt**：不要违反网站使用条款
2. **控制频率**：设置合理延迟（3-5 秒）
3. **使用代理**：避免 IP 被封
4. **数据验证**：自动化收集后需人工验证

## 常见问题

### Q1：找不到足够的薪酬数据怎么办？
A：扩大搜索范围：
- 使用相近岗位名称
- 扩大地域范围
- 参考相近行业数据

### Q2：薪酬格式不统一怎么办？
A：统一转换为月薪区间：
- 年薪 ÷ 12 = 月薪
- 年薪 ÷ 月薪数 = 月薪
- 单一薪资 → 区间（±10%）

### Q3：如何判断数据可信度？
A：来源权重：
- 行业报告 > 财报 > 招聘网站 > 论坛
- 交叉验证多个来源
- 关注数据时效性
