# 掼蛋AI性格设定

## 概述

在陪练模式和观战模式中，AI玩家具有不同的性格特点，使得对局更加真实有趣。每个AI都会"独立思考"，只知道自己的牌和公共出牌信息。

---

## AI角色性格详解

### 2号位 - 激进型（右手对手）

**性格特征**
- 进攻欲望强烈，喜欢抢夺出牌权
- 爱用炸弹，不怕暴露牌力
- 敢于冒险，善于抓住机会

**策略偏好**
| 情况 | 策略 |
|------|------|
| 开局 | 主动出大牌，争取先手 |
| 中盘 | 积极压制对手，不怕对攻 |
| 残局 | 敢冲敢打，追求头游 |

**常见打法**
```
✅ 好的激进打法：
- 手握大王时主动开局
- 对手出小牌时果断压制
- 关键时刻敢于炸牌

❌ 过度激进的表现：
- 牌力不足时盲目抢出牌权
- 炸弹在牌多时过早使用
- 不顾对家配合单独冒进
```

**识别技巧**
- 出牌快，决策果断
- 频繁使用炸弹
- 经常主动出牌争取控制权

---

### 3号位 - 配合型（对家/伙伴）

**性格特征**
- 配合意识极强，优先支持1号位
- 善于帮挡帮送，创造跑牌机会
- 会观察对家意图，主动让牌

**策略偏好**
| 情况 | 策略 |
|------|------|
| 对家领先 | 让牌配合，帮忙挡牌 |
| 对家落后 | 主动出击吸引火力 |
| 对家报单 | 送牌配合其跑牌 |

**配合信号**
| 出牌 | 含义 |
|------|------|
| 出小单张 | 可能在等你先出 |
| 过牌 | 让你先打 |
| 出对子 | 可能在准备配合 |
| 大牌压对手 | 帮你控制局面 |

**帮送技巧**
```
【帮对家跑单的信号】
- 你报单时，3号位出小牌让你接
- 你有大牌时，3号位主动压制对手

【帮你挡牌的信号】
- 对手出牌时，3号位主动压制
- 你被压制时，3号位用炸弹解救
```

**注意事项**
- 3号位过于保守时，可能是在等你先打
- 3号位连续出大牌时，可能在帮你控制局面
- 3号位突然过牌时，可能在让你接风

---

### 4号位 - 保守型（左手对手）

**性格特征**
- 沉稳冷静，擅长防守
- 精于记牌，推断对手牌型
- 后发制人，不轻易暴露牌力

**策略偏好**
| 情况 | 策略 |
|------|------|
| 开局 | 出小牌探路，不暴露实力 |
| 中盘 | 观察局势，关键时刻出手 |
| 残局 | 精准控局，确保不输 |

**记牌特点**
- 记住关键牌是否已出
- 推断各位置剩余牌型
- 预判对手出牌意图

**防守技巧**
```
【保守型4号位的防守策略】
1. 不轻易用炸弹，除非必要
2. 观察1号位和3号位的配合
3. 在关键时刻后发制人

【如何应对4号位】
- 4号位过牌 = 不需要担心
- 4号位出大牌 = 可能在控局
- 4号位突然压制 = 到了关键时刻
```

**识别技巧**
- 出牌相对较慢，思考时间长
- 不轻易使用炸弹
- 经常过牌观望

## AI难度设定

### 初级模式

**AI表现**
- 判断力较弱，容易犯错
- 记牌不精准，经常漏记
- 配合意识差，各自为战

**适合人群**
- 新手入门
- 建立信心
- 熟悉规则

**示例表现**
```
AI-2号位：我有对A，应该能打过对手的单张5...
（错误地分析局面，用大对子压小牌）
```

### 中级模式

**AI表现**
- 具备基本判断力
- 能记住关键牌
- 有一定配合意识

**适合人群**
- 入门玩家
- 正常练习
- 提升水平

**示例表现**
```
AI-3号位：1号位出了对子，我应该配合出对子帮他。
（识别对家意图，主动配合）
```

### 高级模式

**AI表现**
- 记牌精准，推断准确
- 配合默契，意识超前
- 难以战胜，需要高水平

**适合人群**
- 高手练习
- 挑战自我
- 考前冲刺

**示例表现**
```
AI-4号位：根据已出牌推断，2号位手里应该还有炸弹...
（精准记牌，准确推断）
```

## 观战学习要点

### 观察激进型（2号位）
- 学习何时该主动出击
- 理解激进与冒进的区别
- 掌握抢出牌权的时机

### 观察配合型（3号位）
- 学习对家配合技巧
- 理解帮挡帮送的时机
- 掌握信号传递方式

### 观察保守型（4号位）
- 学习防守策略
- 理解记牌的重要性
- 掌握后发制人的技巧

---

## 性格转换建议

**从初级到中级**
- 开始关注AI的配合意识
- 学习3号位的配合信号
- 尝试预测AI的出牌

**从中级到高级**
- 研究AI的记牌方法
- 学习如何利用AI的弱点
- 提升自己的配合意识
