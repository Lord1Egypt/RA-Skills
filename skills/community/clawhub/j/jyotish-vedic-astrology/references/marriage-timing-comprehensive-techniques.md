## 一、KN Rao Double Transit 精确规则

### 核心法则
- **木星（自然祝福星）** 必须关联事件宫/宫主/宫主D9星座
- **土星（自然业力星）** 必须关联事件宫/宫主/宫主D9星座
- 吉祥事件中木星**必须**主要影响宫主星
- Dasha 占 65-70%，Transit 占 30-35%
- 功能9主=功能木星，功能10主=功能土星，与自然木星/土星配合使用

### 精度数据
- KN Rao 体系在 110-115 个星盘测试，准确率 97%（使用 D9 Navamsha）
- 建议可用 D-60 Shashtiamsha 替代 D9 获更高精度

### 相位定义
- 木星相位：5, 7, 9 宫位
- 土星相位：3, 7, 10 宫位

## 二、VP Goel 功能征象星扩展法

### 规则
- 出生盘 9主 = 功能木星（祝福星）
- 出生盘 10主 = 功能土星（业力星）
- 若自然木星/土星恰好是 9主/10主，力量最强
- 功能征象星过境关联事件宫/宫主时，与自然木星/土星同等效力

### 应用场景
当自然木星/土星不直接照射目标宫位时，检查功能征象星是否照射 → 可作为"替代性 Double Transit"

## 三、Jaimini DK 激活法（木星过境法）

### 核心规则（来源：DKScore, Shankar Hegde）
- 当木星**过境**或**相位** DK 所在星座时，婚姻被触发
- 这是 Jaimini 婚姻预测中**最强的规则之一**

### Jaimini 星座相位规则
- 开创星座（Aries/Cancer/Libra/Capricorn）→ 相位固定星座
- 固定星座（Taurus/Leo/Scorpio/Aquarius）→ 相位变动星座
- 变动星座（Gemini/Virgo/Sagittarius/Pisces）→ 相位开创星座

### DK 与 UL 区别
- DK 被激活 + UL 未完全激活 = 约会/订婚，非正式婚姻
- DK + UL 同时被激活 = 正式婚姻

## 四、Upapada Lagna (UL) 完整体系

### 计算方法
1. 取第12宫及其宫主星
2. 从12宫数到宫主星所在星座 = 距离N
3. 从宫主星所在星座再数N步 = UL

### 例外规则
- 宫主星在12宫本宫 → 跳到第10宫
- 宫主星在12宫的第7宫 → 跳到第4宫

### UL 内行星 — 配偶本质
| 行星 | 配偶特征 |
|------|---------|
| 木星 | 高贵、传统、智慧 |
| 金星 | 美丽、艺术、浪漫 |
| 水星 | 年轻、聪明、像朋友 |
| 太阳 | 权威、主导 |
| 土星 | 严肃、年长、稳定 |
| 火星 | 精力旺盛、攻击性 |

### UL 第2宫 — 婚姻延续性（核心秘法）
- **吉星**（木/金/水/月）在 UL 第2宫 = 婚姻有"黏合剂"，不会轻易离婚
- **凶星**（日/火/土/罗喉）在 UL 第2宫 = 危险区域，预示破裂
- 凶星庙旺则"风暴后可能存活"

### UL 与 DK 配合
- DK = 配偶的灵魂（个人层面）
- UL = 婚姻的制度（关系质量、配偶家庭背景）
- 两者需同时考察

## 七、踩坑记录

### Double Transit 判断错误
- 2026年上半年：木星从 Gemini 照 Aquarius（7宫）✓，但土星从 Pisces 不照 Aquarius ✗
  - 土星3/7/10相位从Pisces照：Taurus/Virgo/Sagittarius，不照Aquarius
- **教训**：必须逐一验证每个行星的相位是否真正照射目标宫位，不能想当然

### UL 计算距离
- Cancer(12宫) → 12宫主=Moon，Moon在Aquarius
- UL计算：从12宫(Cancer, 第4星座)数到12宫主所在星座(Aquarius, 第11星座) = 距离 = 11 - 4 = 7步
- 再从Aquarius向前数7步：Aquarius(11)→Pisces(12)→Aries(1)→Taurus(2)→Gemini(3)→Cancer(4)→Leo(5) → **UL = Leo** ✓
- 例外1：若12宫主在12宫本宫 → UL = 第10宫
- 例外2：若12宫主在12宫的第7宫 → UL = 第4宫
