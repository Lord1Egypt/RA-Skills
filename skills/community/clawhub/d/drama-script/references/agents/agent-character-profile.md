# 人物小传 · 代理工作手册

> 通用规则见 `../common-agent-rules.md`。

## 定位

人物小传是整部剧的人物基石。下游所有环节（故事概要、逐集概要、剧本）都依赖人物小传来保证人设一致性和性格辨识度。

产出必须让下游拿到后清楚知道：每个人物是谁、什么性格、什么背景、在剧中的核心功能、会经历什么变化。

---

## 任务

- **输入**：创意原点
- **输出**：人物小传.md（含人物架构 + 全部人物小传）
- **目标**：从创意原点出发，完成人物规划与人物小传撰写，产出一份含人物架构和所有人物简练小传的完整文件

---

## 流程

### 1. 盘点创意原点人物

提取创意原点中已出现的所有人物，逐一判断其在剧情中的功能定位。记录：人物名 + 创意原点中的原始定位 + 重新评估的定位。

### 2. 确定总人数

根据集数确定最低人数，不设上限：

| 集数 | 最低人数 |
|------|----------|
| ≤20集 | 5人 |
| 21-50集 | 8人 |
| 51-80集 | 12人 |
| 81-120集 | 16人 |
| >120集 | 20人 |

低于最低人数剧情撑不起来，但多加人由创作需要决定。

### 3. 差异化填充

确保：

- 每个人物有且仅有一个核心功能，同功能不重复
- 每个人物有独立的性格内核，不重复他人的核心特质
- 每个人物有至少一个闪光点——让观众记住的独特设计（口头禅、行为习惯、反差萌、独特技能等）

### 4. 性别比检查

- 总体男女比 1:1，允许偏差±1人
- **不达标时增加人物来凑比例，不允许修改现有人物性别**
- 新增人物同样遵循差异化原则，不能是凑数工具人

### 5. 次要人物补充

创意原点中的主角、反派与对立面、主角团、中间派与摇摆人物是骨架，但剧情运转还需要其他人物填充——跟班小弟、司机保镖、店员路人等。

**补充原则**：

- **只加有剧情功能的**：推动特定场景冲突、提供信息差、衬托主要人物性格、制造意外变量——每个次要人物至少承担一项功能
- **不加纯背景板**：如果某个场景只需要一个路人甲走过，不需要给名字和小传，直接在剧本里写"路人"即可
- **按需分配**：预估各阶段剧情场景，按实际需要补充，不设硬性数量。通常每个阶段补充 2-4 个次要人物
- **区分等级**：有名字有台词的次要人物 → 写小传；只有一句两句台词的工具人 → 不写小传，直接在逐集概要/剧本中处理

### 6. 撰写人物小传

1. **按重要度排序**展开小传（主角→核心配角→次要配角→功能性人物，同级别内按阵营或关系排序）
2. **结合创意原点**丰满人物细节
3. 设计所有人物弧线
4. 保存文件

---

## 输出

```markdown
# 人物小传

## 人物架构

- 总人数：X人（主要人物X人，次要人物X人）
- 性别比：男X人 / 女X人

## 人物表

| 人物名 | 阵营 | 描述 | 性别 | 等级 |

---

### {人物名}

- 性别：
- 年龄：
- 身份：
- 外貌特征：{一句话}
- 性格：{2-3个标签，一句话行为例证}
- 背景故事：{1-2个关键事件}
- 人物弧线：{一句话}
```

### 字段说明

- **人物表·等级**：`主要` / `次要`，主要人物为创意原点中的主角、反派与对立面、主角团、中间派与摇摆人物，次要人物为本环节补充的人物
- **外貌特征**：一句话，写辨识度最高的特征，不罗列五官数据
- **性格**：2-3个标签 + 一句话行为例证，不用纯形容词堆砌
- **背景故事**：只保留塑造人物的关键事件，1-2句，禁止流水账生平
- **人物弧线**：一句话概括转变轨迹（本以为…不料…最终…）

### 格式铁律

- 只输出上述格式中的章节和内容，禁止添加产出格式以外的任何章节、表格、说明或补充
- 如有任何需要说明的内容，融入对应字段中，不得额外增加章节
- 所有人物按统一模板输出，不因等级差异跳过字段

### 写法要求

**人物小传以简练为第一优先级。** 每个人物的总字数不超过150字（含所有字段）。背景故事只保留1-2个关键事件，禁止流水账。人物弧线一句话概括，禁止展开。

次要人物的小传可更简短，重点写清身份、与主要人物的关系、在剧中的功能，背景故事和弧线可不写。

---

## 自检

生成完成后，逐条检查以下致命问题。**命中任何一条必须立即修改，全部通过后才能交付。**

### 1. 人物是标签堆砌而非活人

- 人物小传读完，脑海中无法浮现一个具体的人。性格全是形容词罗列，缺乏行为例证；背景故事是流水账而非塑造人物的关键事件。

### 2. 功能重复

- 两个人物在剧中的核心功能相同或高度重叠。去掉其中一人，另一人完全可以替代其全部剧情功能。

### 3. 主要人物无弧线或弧线虚假

- 主要人物没有弧线，或弧线是假的（"从极度自卑到极度自信"中间没有任何事件推动转变）。次要人物可以没有弧线，但主要人物必须有且可信。

### 4. 人物去掉后剧情不塌

- 拿掉某个人物后，其他人物能覆盖其功能，所有关键事件可被合理分配给其他人。短剧人物名额有限，每个人物都必须不可替代。

### 5. 性格辨识度不足

- 随便抽一句台词（假设已写），不看角色名无法判断是谁说的。多个角色的说话方式、反应模式高度相似。

### 6. 背景故事与世界观/剧情脱节

- 人物的背景故事只涉及私人恩怨，与世界观的核心冲突、力量体系、势力格局没有交叉点。人物行动路径和世界观运行规则互不干涉。
