# 细胞凋亡异常与疾病关联分析

## 何时使用
- 观察到肿瘤、自身免疫病、神经退行性疾病、AIDS 或心血管疾病的临床或实验表型
- 怀疑细胞凋亡调控异常（如凋亡减少或过度）
- 已排除急性创伤性坏死等非凋亡性细胞死亡
- 可检测组织或细胞中的凋亡水平变化

## 执行步骤

### 1. 判断凋亡水平异常方向
- **凋亡过低（减少）**：提示与肿瘤、自身免疫病相关
- **凋亡过高（过度）**：提示与神经退行性疾病、AIDS、心血管疾病相关

### 2. 根据凋亡异常方向匹配疾病机制

#### 凋亡过低相关疾病
- **肿瘤**：
  - 检查 p53 基因是否突变或缺失 → 导致 NOXA/PUMA 无法诱导 → 线粒体凋亡通路受阻
  - 检测肿瘤细胞 Fas/FasL 表达：高 FasL + 低 Fas → 诱导淋巴细胞凋亡并实现免疫逃逸
- **系统性红斑狼疮（SLE）**：
  - 检查 Fas 表达缺陷 → 自身反应性 T 细胞阴性选择失败 → 外周积累 → 自身免疫攻击

#### 凋亡过度相关疾病
- **神经退行性疾病（如 AD、PD）**：
  - 检测 β 淀粉样蛋白堆积是否诱导神经元凋亡
  - 检查 caspase-3 活性及神经元凋亡抑制蛋白（IAP）基因是否突变
- **AIDS**：
  - HIV gp120 与 CD4⁺ T 细胞结合 → 诱导凋亡
  - 感染细胞对 TRAIL/FasL 超敏感 → 加速 T 细胞耗竭
- **心血管疾病**：
  - 检测血管平滑肌细胞、巨噬细胞、心肌细胞或传导系统细胞是否过度凋亡

### 3. 提出诊断与治疗启示
- **肿瘤治疗**：考虑恢复 p53 活性、使用 IAP 抑制剂或 Smac 类似物
- **神经保护**：考虑使用抗氧化剂或 caspase 抑制剂
- **免疫调节**：考虑靶向 Fas/FasL 通路

> 注意：本技能不适用于急性创伤性组织坏死，且需排除坏死性凋亡（necroptosis）、焦亡（pyroptosis）等非经典凋亡形式的干扰。
