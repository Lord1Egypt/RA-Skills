## 一、核心概念

- **Ashta** = 八，**Varga** = 分配/来源
- 通过Bindus（吉点）评估每个星座的力量强度
- **八大来源**：太阳、月亮、火星、水星、木星、金星、土星、Lagna
- ⚠️ Rahu和Ketu不参与计算

---

## 二、双层计算框架

### Bhinna Ashtakavarga (BAV) - 个人评分表
- 每个行星对12星座的独立贡献
- 每个星座获得0-8个Bindus

### Sarva Ashtakavarga (SAV) - 综合评分表
- 汇总所有8个来源的贡献
- **总分恒定为337点**（宇宙常数）

## 三、分数解读标准

### BAV评分

| 分数范围 | 强度等级 | 含义 |
|---------|---------|------|
| 0-2 bindus | 弱 | 困难、延迟或损失 |
| 3-4 bindus | 中性 | 功能正常但不突出 |
| 5-6 bindus | 强 | 过境产生有利结果 |
| 7-8 bindus | 极强 | 显著正面效果 |

### SAV评分

| 分数范围 | 强度等级 | 人生体验 |
|----------|----------|----------|
| **30+** | 极吉 | 自然顺遂、成功 |
| **28-30** | 吉利 | 正向结果 |
| **25-28** | 中等 | 混合结果 |
| **<25** | 弱/挑战 | 需要刻意努力 |

## 六、过境预测应用

### 核心公式
```
结果质量 = (行星在该星座的BAV) × (该宫的SAV强度)
```

### 应用原则
- **高SAV星座（30+）**：即使凶星也产生吉效
- **低SAV星座（<25）**：即使吉星也难以发挥正面作用

### 示例
- 土星过境10宫（SAV=35）→ 事业稳步上升
- 土星过境5宫（SAV=19）→ 子女/创作受阻

## 八、SAV聚合方法（v2.0.0新增）

### 8.1 SAV计算步骤

1. **确定每颗行星所在的星座**
2. **为每颗行星计算其BAV在12星座上的分布**
   - 从该行星所在星座开始，按BAV基础表分配
   - 叠加其他行星位置带来的额外Bindhu
3. **汇总所有8个来源（7行星+Lagna）**
4. **每个星座的SAV = 所有8个来源在该星座的Bindus之和**

### 8.2 SAV总分校验

```
12个星座的SAV之和 = 337（宇宙常数）
```

如果计算结果≠337，说明有计算错误。

### 8.3 SAV关键阈值实战

| SAV分数 | 10宫（事业）含义 | 7宫（关系）含义 |
|---------|---------------|---------------|
| ≥35 | 事业极度顺利 | 婚姻极度和谐 |
| 30-34 | 事业顺利 | 关系良好 |
| 28-29 | 事业平稳 | 关系正常 |
| 25-27 | 需要努力 | 需要经营 |
| <25 | 事业困难 | 关系挑战 |

## 九、常见错误警示

| 错误类型 | 正确做法 |
|----------|----------|
| 从上升点数宫位 | 从**该行星所在位置**数起 |
| 包含Rahu/Ketu | **排除**这两个影子行星 |
| 忽略Lagna贡献 | Lagna是**第8个来源**，必须计入 |
| 只看SAV不看BAV | **必须双看**：SAV=整体环境，BAV=具体行星在该宫的表现 |
| BAV分配使用错误星座 | **必须从行星实际所在星座开始数**，不是从白羊座 |
