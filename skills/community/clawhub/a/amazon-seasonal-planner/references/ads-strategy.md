# 旺季广告策略详解

## 一、旺季广告核心逻辑

### 1.1 旺季广告的特点

| 特点 | 影响 | 应对策略 |
|------|------|----------|
| 流量激增 | 广告成本上涨 | 提前占位 |
| 竞争激烈 | ACOS上升 | 提高预算 |
| 转化率高 | 是冲量的好时机 | 不要太保守 |
| 排名窗口期 | 旺季表现影响长期排名 | 全力以赴 |

### 1.2 旺季广告目标

**主要目标**：
1. 冲排名 - 旺季排名决定全年
2. 抢流量 - 旺季流量最贵
3. 去库存 - 旺季清仓最好时机

**次要目标**：
1. 测试新品
2. 推广新链接
3. 建立品牌认知

## 二、旺季广告分阶段打法

### 2.1 预热期（旺季前2-4周）

**目标**：积累权重，提升自然排名

**策略**：
```
预算：平时日预算的 120-150%
竞价：平时竞价的 80-100%
ACOS目标：可接受偏高

主要工作：
1. 提高广告预算
2. 降低竞价积累点击
3. 优化listing关键词
4. 开启手动广泛匹配
```

**具体操作**：
```
广告活动设置：
- 自动广告：开启，收集关键词
- 手动广泛：核心关键词，出价适中
- 手动词组/精准：长尾词，出价略高

预算分配：
- 自动广告：30%
- 手动广泛：40%
- 手动词组/精准：30%
```

### 2.2 冲刺期（旺季前1周）

**目标**：冲排名，抢流量

**策略**：
```
预算：平时日预算的 200-300%
竞价：提高到首页竞价的 110-120%
ACOS目标：可接受 30-50%

主要工作：
1. 竞价提高抢首屏
2. 开启更多关键词
3. 提高预算上限
4. 监控排名位置
```

**具体操作**：
```
竞价调整：
- 核心词：提价 30-50%
- 长尾词：提价 20-30%
- 自动广告：保持或略提

预算分配：
- 预热期预算 × 2-3
- 重点词加投
- 灵活调整
```

### 2.3 旺日（旺季当天）

**目标**：最大化转化

**策略**：
```
预算：冲刺期预算 × 1.5-2
竞价：保持高位
ACOS目标：不要过度保守，让它跑

主要工作：
1. 实时监控调整
2. 关注库存
3. 调整无效词
4. 记录数据
```

**旺日注意事项**：
```
⚠️ 不要因为ACOS高就暂停广告
⚠️ 不要在旺日大幅降价（除非库存真的卖不完）
⚠️ 实时关注库存，准备断货预案
⚠️ 记录实时数据，方便复盘
```

### 2.4 延续期（旺日后1-2周）

**目标**：维护排名，清理库存

**策略**：
```
预算：旺日预算的 50-70%，逐步降低
竞价：逐步降低
ACOS目标：回归正常

主要工作：
1. 逐步降低预算
2. 清理旺季没卖完的库存
3. 分析数据做复盘
4. 为下次旺季做准备
```

## 三、各旺季广告策略差异

### 3.1 Prime Day

**特点**：
- 时间短（48小时）
- 流量集中
- 竞争激烈

**策略**：
```
Prime Day广告策略：
- 提前3-4周预热
- 当天全力冲刺
- 之后快速收尾

预算分配：
- 前1周：30%
- Prime Day当天：40%
- 后1周：30%
```

### 3.2 黑五网一

**特点**：
- 时间长（4-5天）
- 销量峰值最高
- 竞争最激烈

**策略**：
```
黑五网一广告策略：
- 提前2-3周预热
- 黑五当天全力
- 网一延续冲量
- 之后清库存

预算分配：
- 前1周：25%
- 黑五：30%
- 网一：25%
- 后1周：20%
```

### 3.3 圣诞旺季

**特点**：
- 时间长（近1个月）
- 礼品属性强
- 节后断崖

**策略**：
```
圣诞广告策略：
- 11月开始预热
- 12月初全力
- 12月中控制
- 12月底清仓

预算分配：
- 11月：40%
- 12月初（1-15）：35%
- 12月中（16-25）：15%
- 12月底（26-31）：10%
```

## 五、旺季广告预算

### 5.1 预算计算方法

**方法1：基于目标销售**
```
旺季预算 = 旺季目标销量 × ACOS × 客单价

示例：
- 旺季目标销量：1000单
- 目标ACOS：30%
- 客单价：$30
- 旺季预算 = 1000 × 0.3 × $30 = $9000
```

**方法2：基于平时倍数**
```
旺季预算 = 平时日预算 × 旺季天数 × 旺季倍数

示例：
- 平时日预算：$200
- 旺季天数：7天
- 旺季倍数：3倍
- 旺季预算 = $200 × 7 × 3 = $4200
```

## 六、旺季广告优化

### 6.1 旺日实时调整

**每2-4小时检查**：
```
1. 广告表现
   - 点击量是否正常
   - 转化率如何
   - ACOS是否可控

2. 库存状态
   - 库存消耗速度
   - 预估断货时间

3. 排名位置
   - 核心词排名
   - 是否有下滑

4. 预算消耗
   - 是否烧太快/太慢
   - 是否需要调整
```

### 6.2 旺日常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 预算烧太快 | 竞价太高 | 降低竞价 |
| 预算花不出去 | 竞价太低 | 提高竞价 |
| ACOS爆表 | 转化差 | 检查listing/调整人群 |
| 断货 | 销量超预期 | 提高售价减缓/暂停广告 |
| 排名下滑 | 竞争对手加强 | 提高竞价 |

### 6.3 旺日后优化

**优化清单**：
```
□ 暂停无效关键词
□ 调整ACOS高的词
□ 优化点击率低的词
□ 调整预算分配
□ 分析数据做复盘
□ 为下次旺季积累经验
```

## 七、旺季ACOS参考

### 7.1 各品类ACOS目标

| 品类 | 平日ACOS | 旺季ACOS | 可接受上限 |
|------|----------|----------|------------|
| 电子产品 | 15-25% | 20-35% | 40% |
| 家居 | 15-25% | 20-35% | 40% |
| 服装 | 20-35% | 25-45% | 50% |
| 美妆 | 15-30% | 20-40% | 45% |
| 玩具 | 20-35% | 25-45% | 50% |
| 宠物 | 15-25% | 20-35% | 40% |

### 7.2 ACOS与利润平衡

```
旺季ACOS接受原则：
- 利润充足（>30%毛利率）：ACOS 40-50% 可接受
- 利润一般（20-30%毛利率）：ACOS 30-40% 可接受
- 利润较薄（<20%毛利率）：ACOS 25-35% 可接受

核心逻辑：
旺季的排名权重更重要！
一次旺季好排名 = 接下来几个月的自然流量
```
