# 使用最低肺泡有效浓度（MAC）评估吸入麻醉药强度

## 何时使用
- 需要比较多种吸入麻醉药的效价强度
- 为手术患者选择合适的吸入麻醉药
- 调控麻醉深度以维持术中无体动反应
- 解释为何某种吸入麻醉药所需浓度更低

## 如何执行
1. 确认适用对象为吸入麻醉药（如七氟烷、异氟烷、氧化亚氮等），排除静脉麻醉药。
2. 获取该药物的标准MAC值（单位：vol%），参考权威药理学资料。
3. 比较不同药物的MAC值：**MAC值越低，麻醉作用越强**。
4. 根据目标麻醉深度，调节吸入气体中药物的肺泡浓度至目标MAC倍数（如1.0 MAC用于切皮无反应，0.5 MAC用于维持浅麻醉等）。
5. 注意：MAC反映的是50%患者对伤害性刺激（如切皮）无体动反应的肺泡浓度，不代表意识消失或完全镇痛。
6. 麻醉结束时，药物主要经肺排出，患者通常可快速清醒。

> 提示：氧化亚氮MAC极高（约105%），无法单独达到1.0 MAC，常与其他吸入麻醉药联合使用以减少后者用量。
