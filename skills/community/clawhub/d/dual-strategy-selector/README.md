# dual-strategy-selector

双策略A股选股器 - 基于仙人指路和老鸭头形态的技术分析策略

## 策略说明

### 策略1: 仙人指路B
- **形态**: 小实体长上影线
- **条件**: 上影线>实体2倍，实体<3%，换手3-7%，量比>1
- **持股**: 20天
- **预期胜率**: 54.2%
- **预期收益**: +3.53%

### 策略2: 老鸭头B
- **形态**: 5日金叉10日，回调至20日线附近
- **条件**: 5日均线上穿10日均线，20日均线向上，股价回调至20日线±2%
- **加成**: 10日上穿60日 + MA10向上
- **持股**: 5天
- **预期胜率**: 72%
- **预期收益**: +0.77%

## 使用方法

### 1. 准备数据

CSV文件必须包含以下列:

| 列名 | 说明 | 示例 |
|------|------|------|
| symbol | 股票代码 | 000001 |
| name | 股票名称 | 平安银行 |
| date | 日期 | 2026-04-30 |
| open | 开盘价 | 11.50 |
| high | 最高价 | 11.60 |
| low | 最低价 | 11.46 |
| close | 收盘价 | 11.49 |
| volume | 成交量 | 113924162 |
| turnover | 换手率(%) | 3.5 |
| ma5 | 5日均线 | 11.37 |
| ma10 | 10日均线 | 11.20 |
| ma20 | 20日均线 | 11.17 |
| ma60 | 60日均线 | 10.99 |
| volume_ratio | 量比 | 1.5 |
| pct_change | 涨跌幅(%) | -0.26 |
| is_up_limit | 是否涨停(0/1) | 0 |

### 2. 运行选股

```bash
python dual_strategy_selector.py --data your_data.csv
```

指定日期:
```bash
python dual_strategy_selector.py --data your_data.csv --date 2026-04-30
```

### 3. 获取数据接口

本技能不包含数据源，请接入您自己的数据接口:

**推荐数据源**:
- 同花顺问财API
- Tushare Pro
- 东方财富Choice
- 通达信本地数据

**数据获取示例** (Python):
```python
import tushare as ts

# Tushare示例
pro = ts.pro_api('your_token')
df = pro.daily(trade_date='20260430')
```

## 参数配置

可在代码中修改 `STRATEGY_CONFIG` 调整策略参数:

```python
STRATEGY_CONFIG = {
    '仙人指路B': {
        'params': {
            'upper_shadow_pct': 2.0,  # 上影线>%
            'body_pct_max': 2.0,      # 实体<%
            'turnover_min': 3.0,      # 换手>%
            'turnover_max': 7.0,      # 换手<%
            'volume_ratio_min': 1.0,  # 量比>
        }
    },
    '老鸭头B': {
        'params': {
            'near_pct': 0.02,         # 回调至20日线±%
            'require_10x60': True,     # 是否要求10日上穿60日
            'require_ma10_up': True,   # 是否要求MA10向上
        }
    }
}
```

## 注意事项

1. **数据质量**: 确保换手率、量比等字段计算正确
2. **时效性**: 策略基于历史数据，市场变化可能影响效果
3. **风险控制**: 建议设置3-5%止损，单只仓位不超过10%
4. **热点叠加**: 可结合当前热点板块进一步过滤
