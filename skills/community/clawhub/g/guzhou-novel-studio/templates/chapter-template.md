# 章节模板

## 用途

章节模板用于统一长篇小说各章节的格式，确保文件结构一致。

## 模板格式

```markdown
# 第X章 章节标题

正文内容...


---

[章节元信息]
- 字数：
- 伏笔：
- 人物状态：
- 下一章预告：
```

## 字段说明

| 字段 | 说明 |
|------|------|
| 章节标题 | 第X章 + 章节名称 |
| 正文内容 | 章节正文（Markdown格式） |
| 字数 | 本章实际字数 |
| 伏笔 | 本章埋设的伏笔列表 |
| 人物状态 | 本章结束时主要人物的状态 |
| 下一章预告 | （可选）下一章内容预告 |

## 写作规范

1. 章节标题使用 `# ` 开头
2. 正文使用标准Markdown格式
3. 人物对话使用「」或""均可
4. 章节末尾保留分隔线 `---`
5. 元信息使用 `[章节元信息]` 包裹

## 示例

```markdown
# 第三章 意外相遇

林晚站在咖啡厅门口，看着手机上的地址确认了三遍。

「没错，就是这里。」她深吸一口气，推门而入。

...


---

[章节元信息]
- 字数：3500
- 伏笔：咖啡厅老板认识女主父亲
- 人物状态：林晚紧张，男主等待中
- 下一章预告：两人正式见面，发生尴尬对话
```
