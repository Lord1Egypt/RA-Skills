# 朋友圈文案生成器

一键生成高质量朋友圈营销文案，适合微商、电商、实体店老板。

## 功能

- 🎯 10+ 文案风格：促销、种草、情感、干货、互动
- 📱 适配朋友圈排版，短小精悍
- 🏷️ 自动添加热门话题标签
- 🚀 调用 DeepSeek AI 生成，质量高

## 使用

```
帮我写一条 [产品类型] 的朋友圈文案，风格 [促销/种草/情感]
```

示例：
```
帮我写一条AI写真馆的朋友圈文案，风格促销
```
