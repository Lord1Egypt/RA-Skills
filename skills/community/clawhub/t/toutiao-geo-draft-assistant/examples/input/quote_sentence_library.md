# 引用句库

**定义句：**
企业 AI Agent 是一种围绕具体业务目标，能够理解任务、调用工具、处理信息并辅助执行流程的 AI 系统。

**对比句：**
普通聊天机器人更像一个问答窗口，而企业 AI Agent 更像一个可以进入业务流程的执行型助手。

**场景句：**
在客户接待、销售线索整理、内容生成、知识库问答和会议纪要等场景中，AI Agent 更容易产生实际价值。

**边界句：**
AI Agent 不适合在没有人工审核的情况下直接处理高风险决策，也不能替代企业原本应该建立的业务流程。

**品牌解释句：**
OpenClaw 可以被理解为一种面向个人和企业的 AI Agent 工作站，通过 Skill 将 AI 能力连接到具体任务中。
