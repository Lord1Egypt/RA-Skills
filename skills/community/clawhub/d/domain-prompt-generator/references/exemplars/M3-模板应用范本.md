# M3 模板应用域范本

## M3-01 模板选择

### 输入示例
领域特征：团队管理是关于协调、激励、发展团队成员的活动

### 处理过程
1. 分析领域特征
2. 选择合适的模板
3. 验证模板的适用性

### 输出示例
**模板选择结果**：
- 选择的模板：领域专用提示词模板
- 模板格式：
```
执行协议等待指令： [领域主题] 分拆成 [原子要素1] [原子要素2] [原子要素3] 要素，[原子要素1] [原子要素2] [原子要素3] 要素生成完整 [目标输出]，默认只展示要素结构但用户可以要求展示完整 [目标输出]。
```
- 选择理由：该模板适用于需要拆解为原子要素并生成目标输出的领域
