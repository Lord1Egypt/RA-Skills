# 福尔摩斯思维方法库：逻辑规则版

> **格式：** 每条规则 = 名称 + 符号定义 + 口诀 + 案例应用
> **用途：** AI遇到新问题，直接套用这些规则进行推导
> **触发：** 当观察到特定条件时，知道下一步怎么推

---

## 一、排除法（Elimination Rule）

**定义：** 如果 A 是唯一能解释所有观察的可能性，且其他所有假设都被证据否定，则 A 必为真相。

**符号：**
```
H1 ∧ H2 ∧ H3 ∧ ... Hn = 所有假设
H1 → 矛盾
H2 → 矛盾
...
Hn-1 → 矛盾
∴ Hn = 真
```

**口诀：** 否定一切不可能，剩余即使难以置信也必定是真相

**案例：**
```
假设集合：H1(自杀) / H2(德国人所为) / H3(抢劫杀人)
H1(自杀) → 口袋仍有钱 → 矛盾 → 排除
H2(德国人所为) → 字体非德国习惯 → 矛盾 → 排除
H3(抢劫杀人) → 无矛盾 → 保留
∴ H3 = 真（凶手不是自杀，不是德国人，是谋杀）
```

**适用：** 当存在多个假设，且部分假设可以被明确证据否定时。

---

## 二、假言推理（Conditional Reasoning）

**定义：** 如果 A → B，且 A 成立，则 B 必成立（肯前必肯后）。

**符号：**
```
A → B
A为真
∴ B为真
```

**口诀：** 肯前必肯后——肯定前件一定能推出后件

**案例1：**
```
如果 符号是密码（O4支持）
且 斯莱尼使用这个密码系统
那么 斯莱尼知道符号含义
斯莱尼知道符号含义 → 斯莱尼能读懂埃尔西的回复
斯莱尼能读懂埃尔西的回复 → 斯莱尼在追踪埃尔西
∴ 斯莱尼在追踪埃尔西
```

**案例2：**
```
如果 房间密室（所有入口封堵）
那么 威胁必须从"未被封堵的通道"进入（O4物理封堵完好）
如果 威胁从通风口进入（O6新开通风口）
且 铃绳新装不通铃（O6铃绳不工作）
那么 通风口和铃绳之间存在关联
∴ 威胁从通风口进入且与铃绳有关
```

**适用：** 当前提A成立时，直接推出结论B，无需额外验证。

---

## 三、逆否推理（Contrapositive Rule）

**定义：** 如果 A → B，则 非B → 非A（否定后件一定能否定前件）。

**符号：**
```
A → B
¬B
∴ ¬A
```

**口诀：** 否后必否前——否定后件一定能推出否定前件，但否定前件推不出确定结论

**案例：**
```
如果 这是恶作剧（假设H1）
那么 埃尔西不会吓死（应为无害玩笑）
但 埃尔西脸色煞白立刻烧掉信（O1）→ 不是无害
∴ 这不是恶作剧（¬B → ¬A）

应用：
如果 凶手是德国人（假设H2）
那么 字体应该符合德国书写习惯
但 RACHE 写法不符合德语正字法
∴ 凶手不是德国人
```

**适用：** 当无法直接证明A时，通过否定B来否定A。

---

## 四、传递规则（Transitivity Rule）

**定义：** 若 A → B，且 B → C，则 A → C（链条式推理）。

**符号：**
```
A → B
B → C
∴ A → C
```

**口诀：** 链条传递——推理链可以跳过中间步骤直接得出结论

**案例：**
```
O4(埃尔西能主动回复符号) → 能读懂符号含义
能读懂符号含义 → 符号是编码而非乱涂
符号是编码 → 使用者是有组织的通信行为
∴ O4 → 符号是编码
∴ 符号是编码 → 使用者有组织
∴ O4 → 使用者有组织
```

**适用：** 多步骤推理，每一步都以前一步为前提，形成完整链条。

---

## 五、必要条件规则（Necessary Condition Rule）

**定义：** 只有 A 成立，B 才成立，记作 B → A（否后必否前）。

**符号：**
```
B → A
¬A
∴ ¬B
```

**口诀：** 否后必否前——否定必要条件一定会否定结果

**案例1：**
```
只有 马活着（必要条件），赌马集团才能操控比赛结果
马被杀了（¬A）
∴ 赌马集团无法操控比赛结果（排除H2：马被卖掉/杀掉）
∴ 马必须活着 → 在某处找到马
```

**案例2：**
```
只有 斯特普尔顿知道继承人行踪，他才能在合适时机放出猎犬
斯特普尔顿不知道亨利爵士的具体行程（¬A）
∴ 斯特普尔顿无法精确作案
但 猎犬在雾中精准出现（实际可作案）
∴ 斯特普尔顿通过其他方式（偷靴子）追踪
```

**适用：** 明确必要条件，当必要条件不满足时，直接排除该假设。

---

## 六、矛盾触发规则（Contradiction Rule）

**定义：** 如果 O1 与 O2 不能同时为真，则至少有一个是假的。

**符号：**
```
O1 → H
O2 → ¬H
∴ O1 与 O2 不能同时为真
```

**口诀：** 两个观察不能同时为真 → 至少有一个需要重新审视

**案例：**
```
O1: 斯莱尼说"我从没威胁过埃尔西" → H1: 斯莱尼无辜
O2: 密码内容显示"AM HERE ABE SLANEY" → H2: 斯莱尼正在追踪埃尔西
H1 与 H2 矛盾
∴ 斯莱尼的陈述与证据矛盾 → 斯莱尼在说谎 → H1排除
```

**适用：** 当两个观察得出完全相反的假设时，触发矛盾分析。

---

## 七、因果链构建规则（Causality Chain Rule）

**定义：** 每个结果都有前因，每个前因都应有证据支撑或合理推测。

**符号：**
```
E1 → E2 → E3 → ... → En
每一步：证据/推测 → 结论
```

**口诀：** 链条不断——每一步都必须有"因为...所以..."，不允许跳跃

**案例：**
```
金币悬赏550（O1）→ 主人拒绝赎金（O5）
→ 主人知道真正动机不是钱（推理）
→ 真正动机与比赛结果有关（推理）
→ 盗贼想让马无法赢比赛（假设）
→ 马必须活着（必要条件）
→ 马在某处（结论）
→ 找到废弃矿井中的马（验证）
```

**适用：** 当需要从结果倒推原因，或从原因推导结果时使用。

---

## 八、充分条件否定规则（Modus Tollens）

**定义：** 如果 A → B，且 B 为假，则 A 必为假。

**符号：**
```
A → B
¬B
∴ ¬A
```

**口诀：** 否定后件必否定前件

**案例：**
```
如果 斯莱尼是凶手（假设A）
那么 他应该在现场留下线索（推理B）
但 现场没有斯莱尼的任何物证（¬B）
∴ 斯莱尼不是凶手（¬A）→ 重新审视假设
```

**适用：** 当前提A必然导致B，但B未发生时，排除A。

---

## 九、或然性推理规则（Probabilistic Reasoning）

**定义：** 当确定性和排他性都不足时，用概率最高假设作为工作假设，但标注置信度。

**符号：**
```
P(H1|E) > P(H2|E) > P(H3|E)
∴ 暂时采用 H1，但持续验证
```

**口诀：** 最大概率假设优先，但保持验证警觉

**适用：** 信息不足时，给出最可能结论但标注需要进一步验证。

---

## 十、异常锚定规则（Anomaly Anchoring Rule）

**定义：** 最异常的观察点通常是突破口，应优先分析。

**符号：**
```
O1: 普通观察
O2: 反常观察
∴ 优先分析 O2
```

**口诀：** 平淡无奇的案子最难——异常细节不是干扰，而是突破口

**案例：**
```
银色马案中：
O1(马被偷) → 普通 → H1(赎金动机) → 证据支持
O2(主人拒绝赎金) → 反常 → H2(比赛操控) → 无矛盾
∴ 优先分析 O2（拒绝赎金）→ 发现赌博动机
```

**适用：** 当多个观察中有一个明显反常时，优先处理反常点。

---

## 十一、主动验证规则（Active Verification Rule）

**定义：** 当关键信息缺失时，主动设计验证方案，而不是猜测。

**符号：**
```
缺失信息 I
设计验证 V
V结果X → 支持 H1
V结果Y → 排除 H1
```

**口诀：** 主动验证，不猜测

**案例：**
```
登失物招领启事，用戒指作诱饵
结果：来的是乔装老太婆的年轻人
→ 验证失败本身揭示：新线索（凶手有帮手且演技精湛）
```

**适用：** 信息不足时，设计行动引出新信息。

---

## 十二、反向工程规则（Reverse Engineering Rule）

**定义：** 从已知结果状态，反推需要满足的前提条件，然后验证前提。

**符号：**
```
结果状态 S
反推：S为真 → 需要满足 P1, P2, P3
验证：P1? P2? P3?
若任一不满足 → S为假
```

**口诀：** 从结果倒推前提，验证每一个前提是否成立

**案例：**
```
结果：血字"RACHE"
反推：若凶手故意留下德语线索 → 凶手有德裔背景
验证：死者是英美混合背景（O3）
→ 德裔背景合理
→ 假设成立
```

**适用：** 当需要解释一个现象时，先问"这个结果要成立需要什么前提"。

---

## 十三、归谬法（Reductio ad Absurdum）

**定义：** 假设命题成立，若能推导出矛盾结论，则原假设不成立。

**符号：**
```
假设 A
若 A 成立 → B
B → 矛盾（与已知事实矛盾）
∴ A 不成立
```

**口诀：** 假设推出矛盾 → 假设必假

**案例1：**
```
假设：斯莱尼是无辜的（与埃尔西威胁无关）
若斯莱尼无辜 → 他不应该知道密码含义
但 斯莱尼能读懂埃尔西的回复（O4主动回复）
→ 斯莱尼知道密码含义
→ 斯莱尼与威胁有关
∴ 斯莱尼无辜假设不成立
```

**案例2：**
```
假设：斑点带子中的"带子"指吉普赛人（band = 一帮人）
若"带子"=吉普赛人 → 凶手应该是一群人行动
但 物理密室（所有入口封堵O4）→ 多人行动不可能悄无声息
→ 矛盾
∴ "带子"不是吉普赛人
```

**案例3：**
```
假设：银色马案中马被杀了
若马被杀 → 马主不会拒绝赎金（赎回尸体）
但 马主拒绝赎金（O5）
→ 矛盾
∴ 马没有被杀 → 马还活着（重要结论）
```

**适用：** 当直接证明A为假很困难时，用归谬法假设A为真再导出矛盾。
