# ClawSentry Readme

ClawSentry 是火山引擎 AI 助手安全为 OpenClaw 深度定制的安全防护组件。在保障 OpenClaw 正常作业的同时，提供实时风险检测与防御能力，涵盖提示词注入、敏感数据泄露及高危命令执行等核心安全痛点场景。

## 防护功能

- **高危操作拦截**
  - 实时识别并拦截删除、外发、越权调用等高危行为，防止风险操作真正落地。

- **提示词攻击防护**
  - 检测并防御提示词注入、越狱诱导和恶意指令劫持，保障 AI 助手执行边界可信。

- **敏感信息泄露防护**
  - 自动识别银行卡号、手机号等 PII 隐私数据，阻断敏感信息输出与外传。

- **风险 Skill 扫描**
  - 对 AI 助手已集成的 Skill 进行安全扫描，识别恶意组件，提前发现潜在安全隐患。

## 安装方式（仅需极简三步操作）

#### 通过对话直接安装

支持通过对话交互的方式完成 ClawSentry skill 包安装，例如你可以通过与 OpenClaw 对话完成安装。

#### 前提条件

在开始之前，请确保你已经正确安装了 OpenClaw 。

#### 步骤一 运行 ClawSentry Skill

在对话框中输入：帮我安装 claw-sentry 插件

预期结果：执行完成后，在对话框中会返回安装成功的提示并提供登录授权链接。

#### 步骤二 授权与自动配置

1. 在打开的页面上，请仔细阅读相关服务协议和隐私政策。勾选同意后，单击 **“同意并开通”**。
2. 浏览器页面显示“已成功创建”信息即表示完成。
3. 返回对话框，按照提示回复已登录并授权，继续执行自动配置步骤完成 claw-sentry 插件的配置。

#### 步骤三 验证安装结果

完成安装后，你可以向你的 OpenClaw 通过对话框发送以下类似指令，来验证 claw-sentry 插件是否已成功安装并启用：

1. 查看 claw-sentry 的启用状态；
2. 查看防护状态。

**预期结果**：对话框返回的信息中应包含 claw-sentry 插件，状态为“已启用”。
