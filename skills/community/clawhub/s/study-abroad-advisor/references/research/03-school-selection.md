## 第一章：选校的核心逻辑

### 1.1 选校不是在选排名，是在选「四年后的自己」

顾问在做选校建议时，核心要回答的问题：

```
1. 这个学生的背景最适合去哪？（Background Fit）
2. 这个学生的目标最需要什么资源？（Resource Fit）
3. 哪些学校真的有录取他的概率？（Chance Assessment）
4. 哪些学校的 Return on Investment 最高？（ROI）
```

### 1.2 「中国申请者陷阱」——排名崇拜症

```
常见误判：
❌ 只看 QS/US News 总排名
❌ 以为 UCLA > UCSD（计算机方向相反）
❌ 只申请前50，忽略专业强校
❌ 忽视奖学金差异（同排名学校给 50% 或 0% 的差距）

正确逻辑：
✓ 看专业排名（US News 分专业排名）
✓ 看 Placement Rate（毕业生就业/升学去向）
✓ 看 Faculty Research（与学生研究方向匹配度）
✓ 看 Class Size（大课 vs 小研讨班）
✓ 看 Geographic Location（就业市场辐射范围）
```

## 第二章：美国院校分析框架

### 2.1 美国本科院校分层

```
Tier 1：Ivy+ / MIT/Caltech/Stanford
  特征：品牌效应最强，课外活动/网络资源顶级
  适合：有强钩子 + 文书极强 + 综合 Profile 无明显短板

Tier 2：T10–T20（如 JHU/Northwestern/Duke/Rice）
  特征：专业顶尖，校园文化各有特色
  适合：Profile 强但无超强钩子的学生

Tier 3：T20–T50（如 UIUC/UCSD/UT Austin/UMich）
  特征：工科/商科专业力量极强，录取相对可及
  适合：GPA/标化扎实，专业目标明确

Tier 4：Liberal Arts Colleges（如 Williams/Amherst/Swarthmore）
  特征：师生比低，本科科研机会极多
  适合：想要深度本科教育、考虑 PhD 的学生
```

### 2.2 热门专业最强院校（专业方向选校）

| 专业方向 | 顶尖院校（不按总排名，按专业实力） |
|---------|----------------------------------|
| CS / AI | MIT, Stanford, CMU, UIUC, UC Berkeley |
| 金融/商科 | Wharton, Booth, Stern, Columbia, HBS（MBA）|
| 工程 | MIT, Caltech, Georgia Tech, Purdue |
| 生物/医学 | Johns Hopkins, UCSF, Duke, Vanderbilt |
| 法学（LLM）| Yale, Harvard, Columbia, NYU |
| 公共政策 | Harvard Kennedy, Princeton SPIA, Georgetown |
| 艺术/设计 | RISD, Pratt, Parsons, CalArts |

### 2.3 Location vs. Ranking 权衡

```
🏙️ 纽约优势：
  - 金融/媒体/时尚行业就业网络
  - 哥大/NYU/Fordham 的校招资源
  
🌁 硅谷/湾区优势：
  - 科技行业实习/校招最密集
  - Stanford/Berkeley/Santa Clara 的产业连接

🌆 波士顿优势：
  - 科研/生物医药产业密集
  - MIT/Harvard 的学术氛围

⚠️ 小城市学校（如 Dartmouth/Cornell）：
  - 校园体验极佳，但就业需要主动出击
```

## 第三章：英国院校分析框架

### 3.1 英国院校分层

```
G5（顶尖五校）：Oxford / Cambridge / Imperial / UCL / LSE
  特征：世界顶级学术声誉，入读竞争极激烈
  注意：各校优势专业不同（LSE=社科经济，Imperial=工程理工）

Russell Group（24所）：QS Top 100 主体
  特征：研究型综合大学，毕业生就业强
  代表：Edinburgh, King's College, Manchester, Bristol

专业强校：
  艺术：St Martins (UAL), RCA
  商科：LBS（MBA/研究生）
  法律：Oxford, Cambridge, LSE, King's
```

### 3.2 英国一年制研究生的独特价值

```
优势：
  ✓ 一年完成，总成本（学费+生活）低
  ✓ QS Top 50 院校学历，国际认可度高
  ✓ Graduate Route 签证（2年工作权利）
  ✓ 进入外企/咨询/金融的跳板

适合：
  → 本科毕业想要一个国际硕士镀金后回国就业
  → 想转行到金融/咨询（LSE/Oxford/Cambridge 牌子加持）
  → 想要 PhD 的学术准备期

劣势：
  ✗ 时间短，科研积累相对薄弱
  ✗ 英国本地就业市场竞争激烈
```

### 4.3 港新选择的战略价值

```
战略一：「跳板策略」
  → 港新 → 积累国际工作经验 → 申请欧美博士
  
战略二：「亚太就业策略」
  → 港新硕士 → 跨国企业亚太区岗位（外资银行、咨询、科技）
  
战略三：「性价比策略」
  → 对比英美，港新学费更低 + 生活成本可控
  → QS前50院校学历，认可度高
  
风险：
  → 香港政治环境变化对某些学生可能有顾虑
  → 新加坡本地就业对国际生有限制
```

## 第五章：选校策略的顾问操作规范

### 5.1 背景评估标准化流程

```
Step 1：收集硬件数据
  □ GPA（百分制/4.0/5.0换算）
  □ 标化成绩（SAT/ACT/TOEFL/IELTS）
  □ AP/IB课程数量和成绩
  □ 竞赛获奖（等级/范围）

Step 2：软件评估
  □ 课外活动质量（是否有钩子）
  □ 研究/实习经历（本科/研究生）
  □ 推荐信来源强度
  □ 文书潜力（是否有好故事）

Step 3：目标院校初始列表
  □ 结合硬件给出「理性」目标范围
  □ 结合软件给出「钩子加成」可能范围
  □ 与学生确认意愿排序

Step 4：最终校单确认
  □ 冲刺：2–3 所
  □ 匹配：4–5 所
  □ 保底：2–3 所
  □ 确保每所都是真心愿意就读
```

### 5.2 中国学生的特殊注意事项

```
1. 竞争激烈的中国学生池（Chinese Applicant Pool）
   → 顶校对中国学生实际上存在隐性配额
   → 背景雷同度极高（数理竞赛 + AMC + 模联）
   → 需要差异化叙事，避免成为「另一个中国数学奖学金学生」

2. 大陆高中 vs 国际高中
   → 大陆高中申请者：高考成绩 + AP/SAT双线备考
   → 国际课程（IB/A-Level）申请者：与海外申请者同一标准

3. 内地高考生的海外申请窗口
   → 部分院校（港三/新两）接受高考直申
   → 英国部分院校接受高考等效认定
   → 美国通常需要同时准备 SAT/ACT
```
