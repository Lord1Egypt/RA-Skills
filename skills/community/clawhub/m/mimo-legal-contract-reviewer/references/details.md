# legal-contract-reviewer 详细参考

## 一、合同分类与审查视角判定

### 合同类型速查表
| 类型 | 核心关注点 | 常见弱势方 | 高频风险 |
|------|------------|-----------|---------|
| 劳动合同 | 试用期/竞业/社保 | 员工 | 单方调岗/违法解除 |
| 租赁合同 | 押金/解约/维修 | 租客 | 霸王涨租/扣押金 |
| 采购合同 | 验收/质量/交付 | 采购方 | 货不对板/验收困难 |
| 销售合同 | 退货/质保/售后 | 消费者 | 限定责任/取消退换 |
| 合作协议 | 分成/退出/贡献 | 小股东 | 被稀释/踢出局 |
| 保密协议 | 范围/期限/泄密认定 | 打工人 | 范围过宽/永久有效 |
| 授权合同 | 范围/地域/再授权 | 被授权方 | 单方撤销/独家陷阱 |
| 服务合同 | 交付/验收/修改次数 | 甲方 | 无限修改/拖欠款项 |

### 审查视角判定
- **甲方（优势方）**：关注对方履约能力、违约成本
- **乙方（弱势方）**：关注权利保障、违约金合理性、退出机制
- **对等合同**：关注双方权利义务是否对等

## 二、风险等级判定标准

### 🔴 高危风险（必须修改）
1. **违反法律强制性规定**（如试用期满不转正）
2. **明显显失公平**（权利义务严重不对等）
3. **模糊表述导致无法维权**（如"视为满意"的定义权在对方）
4. **单方权利过大**（如任意解除权、任意调整权）
5. **违约金/赔偿无上限**

### 🟡 中危风险（建议修改）
1. 条款模糊需进一步明确
2. 缺失重要条款（建议补充）
3. 风险分配略偏但可接受
4. 争议解决方式不便于己方

### 🟢 低危风险（可接受）
1. 格式性条款表述优化
2. 语言表达更精确
3. 次要权利义务微调

## 三、条款逐项审查指南

### 3.1 合同主体审查
```
检查项：
□ 合同双方名称是否与资质文件一致
□ 法定代表人/授权代表是否有效授权
□ 对方是否具备履约资质（特殊行业许可证）
□ 主体是否存在（吊销/注销/破产风险）
□ 联系地址/账号是否准确
```

### 3.2 标的与质量审查
```
检查项：
□ 标的物/服务描述是否具体可量化
□ 质量标准是否有明确依据（国标/行标/企标/样品）
□ 验收标准是否可操作
□ 样品封存是否有记录
```

### 3.3 权利义务审查
```
检查项：
□ 甲乙方权利义务是否对等
□ 是否有"霸王条款"（只规定一方义务）
□ 关键节点是否有时间限制
□ 变更/补充条款的触发条件是否合理
```

### 3.4 违约责任审查
```
检查项：
□ 违约金是否合理（过高可申请调减）
□ 损失赔偿是否可预见/可量化
□ 免责条款是否过于宽泛
□ 不可抗力定义是否合理
□ 解除权行使条件是否清晰
```

### 3.5 争议解决审查
```
检查项：
□ 管辖法院/仲裁委是否便于己方
□ 适用法律是否明确（涉外合同）
□ 送达条款是否有效
□ 诉讼/仲裁时效是否清晰
```

## 四、高频风险条款详解

### 劳动合同高危条款
| 条款 | 风险 | 修改建议 |
|------|------|---------|
| "甲方可单方调整乙方岗位" | 岗位调整无限制 | 限定调整条件，需协商或正当理由 |
| "试用期不符合录用条件"（无具体标准） | 任意辞退 | 明确录用条件/考核标准 |
| "竞业限制期内每月补偿X元"（低于工资30%） | 补偿过低 | 争取提高至50%以上或约定随工资调整 |
| "离职后Y年内不得从事相关行业"（范围过宽） | 限制过严 | 限定地域/具体业务/竞争关系 |

### 租赁合同高危条款
| 条款 | 风险 | 修改建议 |
|------|------|---------|
| "涨租幅度不超过市场行情" | 无具体标准 | 约定涨幅上限（如CPI+3%） |
| "提前解约需赔偿剩余租金的X%" | 违约金过高 | 争取降低或约定免赔期 |
| "押金退还时扣除正常损耗" | 损耗标准不清 | 明确正常损耗范围/拍照留档 |
| "房东可随时进入房屋" | 侵犯隐私 | 改为提前通知+紧急情况例外 |

### 商业合同高危条款
| 条款 | 风险 | 修改建议 |
|------|------|---------|
| "甲方有权随时解除合同" | 无保障 | 约定乙方损失赔偿或限定解除条件 |
| "付款以甲方验收合格为准"（无验收期限） | 无限拖延 | 约定验收期限+逾期视为通过 |
| "知识产权归甲方所有" | 丧失成果 | 区分背景IP/ foreground IP |
| "乙方向甲方支付律师费" | 不合理 | 改为败诉方承担 |

## 五、修改建议模板

### 格式：问题→依据→建议

```markdown
**风险点**：{原条款内容}
**问题描述**：{具体问题}
**法律依据**：《民法典》第XXX条/《劳动合同法》第XXX条
**修改建议**：
  - 修改前：`{原文}`
  - 修改后：`{建议稿}`
**协商优先级**：必须修改/建议修改/可接受
```

### 示例
```
**风险点**：甲方有权随时调整乙方工资
**问题描述**：单方调薪权剥夺乙方知情权和协商权
**法律依据**：《劳动合同法》第35条（变更劳动合同需协商一致）
**修改建议**：
  - 修改前：甲方有权根据经营情况调整乙方工资
  - 修改后：甲方调整工资需提前30日书面通知乙方，并说明理由
**协商优先级**：必须修改
```

## 七、合规边界清单

### ❌ 不涉及的情形
- 从零开始起草合同 → 建议使用合同起草类工具
- 代理诉讼/仲裁 → 需要律师执照
- 提供正式法律意见书 → 需要律师出具

### ✅ 本技能适用场景
- 识别合同中的风险条款
- 提供修改建议参考
- 辅助理解合同含义
- 判断是否需要咨询律师

## 八、常见陷阱识别

### 1. 格式条款陷阱
- 提供方未以合理方式提示注意（可主张不成为合同内容）
- 加重对方责任/排除对方权利（无效）

### 2. 数字陷阱
- "不少于"/"不超过"是否有包含关系
- 百分比计算基数是否明确
- 货币单位是否清晰（人民币/美元）

### 3. 生效条件陷阱
- "本合同自双方签字盖章之日起生效"（骑缝章是否完整）
- 附条件生效（条件是否可操纵）
- 审批生效（审批期限是否明确）

### 4. 附件效力陷阱
- 附件与正文冲突时以哪个为准
- 附件是否需要签字盖章
- 附件编号引用是否准确

## 九、谈判话术参考

### 争取修改霸王条款
```
"这个条款和我们之前谈的不太一样，能否调整为..."
"根据行业惯例，这个违约金比例通常是..."
"如果这个条款不改的话，我们这边可能很难通过审批..."
```

### 要求补充缺失条款
```
"这个情况如果发生的话，责任怎么算？"
"最好能把这个约定写进去，避免以后争议..."
"我们法务要求这类合同必须包含XX条款..."
```
