# 三命通会

## 概述

《三命通会》，明代万民英撰，是中国古代命理学的一部巅峰之作，也是子平命理学的集大成者。全书共十二卷，系统总结了前人命理学说，广征博引，资料丰富，是学习八字命理的必读经典。

---

## 作者简介

**万民英**，字汝豪，号育吾，明代著名命理学家。生于明嘉靖年间，卒于万历年间。博通经史，精于命理，著有《三命通会》《星学大成》等命理名著。

---

## 成书背景

明代命理学发展到顶峰，《三命通会》汇集了前人命理精华，包括：
- 《渊海子平》的核心理论
- 《子平真诠》的用神体系

## 核心理论

### 一、五行生克

**生克次序**：
```
木生火、火生土、土生金、金生水、水生木
木克土、土克水、水克火、火克金、金克木
```

**旺相休囚死**：
| 四季 | 旺 | 相 | 休 | 囚 | 死 |
|------|------|------|------|------|------|
| 春 | 木 | 火 | 水 | 金 | 土 |
| 夏 | 火 | 土 | 木 | 水 | 金 |
| 秋 | 金 | 水 | 土 | 火 | 木 |
| 冬 | 水 | 木 | 金 | 土 | 火 |

### 二、十神体系

| 十神 | 定义 | 象义 |
|------|------|------|
| 正官 | 克我之物，阴阳异类 | 官职、名誉、约束 |
| 七杀 | 克我之物，阴阳同类 | 权威、武力、凶险 |
| 正财 | 我克之物，阴阳异类 | 妻子、财运、产业 |
| 偏财 | 我克之物，阴阳同类 | 父亲、横财、情妇 |
| 正印 | 生我之物，阴阳异类 | 母亲、学业、贵人 |
| 偏印 | 生我之物，阴阳同类 | 继母、偏业、冷门 |
| 食神 | 我生之物，阴阳同类 | 子女、福气、口福 |
| 伤官 | 我生之物，阴阳异类 | 才华、傲气、克官 |
| 比肩 | 同类同阴阳 | 兄弟、朋友、竞争 |
| 劫财 | 同类异阴阳 | 姐妹、争夺、破财 |

### 三、用神选取

**原则**：
1. 日主强：宜克泄耗
2. 日主弱：宜生扶帮
3. 寒暖燥湿：调候为急
4. 格局清浊：取清去浊

**方法**：
1. 看日主强弱
2. 看月令提纲
3. 看格局成败
4. 看寒暖燥湿
5. 看组合流通

### 四、格局成败

**成格条件**：
1. 用神有力
2. 喜神辅佐
3. 忌神受制
4. 组合流通

**败格条件**：
1. 用神受损
2. 忌神猖獗

## 命例分析

### 命例一：富贵命

**四柱**：甲子 乙亥 丙寅 戊戌

**分析**：
1. 日主丙火，生于亥月，不得令
2. 但寅木长生，戌土库根，日主不弱
3. 用神：壬水七杀（亥中壬水得令）
4. 喜神：木（印星生身）
5. 忌神：金（财星破印）
6. 格局：七杀格，杀印相生
7. 大运：丙子、丁丑、戊寅、己卯、庚辰、辛巳
8. 流年：逢金水流年吉，逢木火流年平，逢土流年凶

### 命例二：贫贱命

**四柱**：庚申 戊寅 壬子 丙午

**分析**：
1. 日主壬水，生于寅月，不得令
2. 但申子半合水局，日主不弱
3. 用神：丙火正财（午中丙火得令）
4. 喜神：木（食伤生财）
5. 忌神：金（印星生身太过）
6. 格局：正财格，财印相战
7. 大运：己卯、庚辰、辛巳、壬午、癸未、甲申
8. 流年：逢木火流年吉，逢金水流年凶，逢土流年平
