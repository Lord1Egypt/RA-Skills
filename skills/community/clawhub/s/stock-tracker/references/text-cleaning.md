# 公告正文清洗规则

共 14 条规则，按应用顺序依次执行。每规则为正则模式，使用 `re.MULTILINE` 编译。

## 规则 1: 股票代码表头行

**匹配示例：**
- `证券代码:600961 证券简称:株冶集团 公告编号:2026-021`
- `证券代码:601669 股票简称:中国电建 公告编号:临2026-028`

**正则：** `证券代码[：:]\s*\S+\s+(?:证券|股票)简称[：:]\s*\S+\s+(公告编号[：:]\s*\S+)?`

**行为：** 全部移除，不保留任何信息。

---

## 规则 2: 纯公司名行

**匹配示例：**
- `株洲冶炼集团股份有限公司`
- `中国电力建设股份有限公司`

**正则：** `^\s*\S{4,20}(股份有限)?公司\s*$`

**行为：** 删除独立成行的公司全称（仅限于单独占一行的情况，不删除段落内的公司名）。

---

## 规则 3: 董事会/监事会免责声明

**10 个变体：**

| # | 变体模式 |
|---|---------|
| 3a | 本公司及董事会全体成员保证……虚假记载……误导性陈述……重大遗漏……法律责任 |
| 3b | 本公司董事会及全体董事保证……虚假记载……误导性陈述……重大遗漏……法律责任 |
| 3c | 本公司及全体董事……虚假记载……误导性陈述……重大遗漏……法律责任 |
| 3d | 本公司及……监事……保证……虚假记载……误导性陈述……重大遗漏……法律责任 |
| 3e | 本公司及……高级管理人员……保证……虚假记载……误导性陈述……重大遗漏……法律责任 |
| 3f | 本公司……董事……高级管理人员……保证……虚假记载……误导性陈述……重大遗漏……法律责任 |
| 3g | 不存在……虚假……误导……重大遗漏……承担……责任 |
| 3h | 对其内容的真实性、准确性和完整性承担法律责任 |
| 3i | 全体董事……保证……内容……真实……准确……完整……责任 |
| 3j | 保证……不存在……虚假……误导……遗漏……法律 |

**行为：** 完整移除整句声明。注意：3a-3f 的保证范围涵盖 "虚假记载、误导性陈述或者重大遗漏"（含"或者"变体），同时支持 "承担法律责任" 和 "承担个别及连带法律责任" 等变体。

---

## 规则 4: "特此公告" 结语 + 日期落款

**匹配：** `\n\s*特此公告[。\s]*\n?(?:\s*\S*?公司\s*(?:董事会|监事会)?\s*\n?)?(?:\s*<日期>\s*)?`

**行为：** 将 "特此公告" 及其后的公司名和日期整体移除，替换为换行符。

**规则 4b（独立日期行）：** `\n\s*<日期>\s*$`

**行为：** 移除独立的日期落款行，支持中文数字日期（如 `二〇二六年五月二十九日`）和阿拉伯数字日期（如 `2026年5月29日`）。

---

## 规则 5: 股票信息行

**匹配示例：**
- `股票简称:申万宏源 股票代码:000166`

**正则：** `股票简称[：:]\s*\S+\s+股票代码[：:]\s*\S+`

**行为：** 移除出现在募集说明书等文档开头的股票简称/代码行。

---

## 规则 6: "重要内容提示" 标记

**匹配：** `重要内容提示[：:]\s*\n?`

**行为：** 只移除 "重要内容提示：" 这个标题标记，保留下方的提示内容。

---

## 规则 7: PDF 页脚/分页噪声

**三个子规则：**

| # | 匹配内容 | 正则 |
|---|---------|------|
| 7a | `第1页 共3页` 类页脚 | `\n\s*第\s*\d+\s*页\s*共\s*\d+\s*页\s*\n?` |
| 7b | 纯数字页码行 | `^\s*\d+\s*$` |
| 7c | PDF 水印文字 | `[\w\-]+\.pdf\s*\d{4}-\d{2}-\d{2}\s*\d{2}:\d{2}:\d{2}` |

**行为：** 7a 替换为换行符，7b 和 7c 直接移除。

---

## 规则 8: 声明板块

**匹配：**
- `声 明\n[^。]*?中国证监会[^。]*?证券交易所[^。]*?责任`
- `声 明\n[^。]*?本募集说明书[^。]*?真实[^。]*?准确[^。]*?完整[^。]*?责任`

**行为：** 移除年报、募集说明书开头的法律声明板块。

---

## 规则 9: 投资者关系活动记录表头

**三个子规则：**

| # | 匹配内容 |
|---|---------|
| 9a | `证券代码：xxx 证券简称：xxx` 行 |
| 9b | `xxx股份有限公司投资者关系活动记录表` |
| 9c | `投资者关系活动记录表` |

**行为：** 移除机构调研公告的标准表头。

---

## 规则 10: 地址/联系信息模板

**匹配：**
- `住所[：:].*?(?:\d{3,6}室|号|楼|大厦|路\d+号)`
- `邮政编码[：:]\s*\d{6}`
- `联系电话[：:]\s*\S+`
- `传真[：:]\s*\S+`

**行为：** 移除文档末尾的注册地址、邮编、电话、传真等联系信息。

---

## 规则 11: 募集说明书评级/担保字段

**匹配：**
- `信用评级结果[：:][^。]*?[。，]`
- `信用评级机构[：:][^。]*?[。，]`
- `增信措施情况[：:][^。]*?[。，]`
- `担保或其他增信情况[：:][^。]*?[。，]`
- `发行金额[：:][^。]*?[。，]`
- `牵头主承销商[：:].*`
- `联席主承销商[：:].*`
- `簿记管理人[：:].*`
- `受托管理人[：:].*`

**行为：** 移除募集说明书中固定格式的评级、担保、承销商信息。

---

## 规则 12: 目录页

**匹配：** `目\s*录\s*\n[\s\S]*?(?=议案|一[、）]|\d[、）])`

**行为：** 从 "目 录" 标记开始到第一个议案/章节标题之间的内容全部移除。适用于年报、股东会会议资料等包含目录的文档。

---

## 规则 13: H股表格模板文字

**匹配：**
- `翌日披露報表\s*\([^)]*\)\s*`
- `表格類別[：:]\s*\S+\s+狀態[：:]\s*\S+\s*`
- `如上市發行人的已發行股份[^。]*?作出披露，必須填妥第一章節。`

**行为：** 移除港股翌日披露报表的标准表格模板文字。

---

## 规则 14: 多余空白行

**匹配：**
- `\n{3,}` → 替换为 `\n\n`（保留最多一个空行）
- `^\s*\n` → 移除文档开头的空行

**行为：** 压缩多余连续空行，清理文档首尾空白，最后执行 `strip()`。
