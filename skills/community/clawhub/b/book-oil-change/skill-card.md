## Use Case: <br>
External users can use this skill to search for oil-change providers, check appointment availability, and create bookings through Lokuli. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill uses Lokuli as an external booking service and may submit customer contact details when creating a booking. <br>
Mitigation: Install only if the user is comfortable using Lokuli, and confirm the provider, service, date, time, and contact details before creating a booking. <br>
Risk: The skill could be invoked for general oil-change information when no booking service is needed. <br>
Mitigation: Use it only for search, availability, or booking workflows that require contacting the booking service. <br>
