# 安魂符修持指南

## 概述

**安魂符**，道教经典符箓，用于**镇魂安魄、护身安康**。适用于魂魄不安、失眠多梦、神志不清等症状。

**起源**：相传为太上老君传授，后由道士整理成符箓。

**核心作用**：
1. 镇魂安魄
2. 护身安康
3. 安神定志
4. 驱邪避鬼

---

## 适用症状

### 一、主要症状

| 症状 | 表现 | 适用符法 |
|------|------|----------|
| 魂魄不安 | 失眠多梦、易醒 | 安魂符 |
| 神志不清 | 言语错乱、行为异常 | 安魂符+辟邪符 |
| 夜间多动 | 夜游、夜啼 | 安魂符+北斗神咒 |
| 术后昏迷 | 手术后昏迷不醒 | 安魂符+还受生债 |
| 附体嫌疑 | 言语不似本人 | 安魂符+金光神咒 |

### 二、王兵案例适配

**王兵症状**：
1. 胃切除后昏迷，刚清醒
2. 白天不语（魂魄不全）
3. 晚上9-11点话多（亥时附体）
4. 对家属不理不睬（情感淡漠）

**适用符法**：**安魂符**（首选） + **辟邪符**（辅助）

## 符咒内容

### 一、安魂符咒

**完整咒语**：
```
奉请太上老君，安魂定魄，保神安康。
魂归本位，魄守其身，邪不可干，病不可侵。
急急如律令！
```

**简咒**（日常念诵）：
```
太上老君，安魂定魄。
魂归本位，魄守其身。
急急如律令！
```

### 二、符形结构

**安魂符构成**：
1. **上部**："魂"字（镇魂）
2. **中部**："安"字（安魄）
3. **下部**："魄"字（定魄）
4. **四周**：雷令纹（驱邪）

**符形示意图**：
```
    [魂]
   /    \
[安]    [魄]
   \    /
    [令]
```

## 书符方法

### 一、择日

**宜择日**：
1. **天德日**：天德星值日（黄道吉日）
2. **月德日**：月德星值日（黄道吉日）
3. **甲子日**：天德合日（大吉）
4. **乙丑日**：月德合日（大吉）

**忌择日**：
1. **破日**：日月相冲（凶）
2. **害日**：日月相害（凶）
3. **煞日**：煞星值日（凶）

**查询方法**：
- 查《黄历》或《通胜》
- 选"宜祈福、宜祭祀"之日

### 二、择时

**宜择时**：
1. **卯时（5-7点）**：日出时分，阳气初生（最宜）
2. **辰时（7-9点）**：龙行时分，阳气旺盛（次宜）
3. **午时（11-13点）**：正午时分，阳气最盛（驱邪宜）

**忌择时**：
1. **子时（23-1点）**：夜半时分，阴气最盛（凶）
2. **亥时（21-23点）**：人定时分，阴气旺盛（凶）

### 三、斋戒

**斋戒要求**：
1. **前三天**：戒荤腥（不吃肉、不吃葱蒜韭菜）
2. **前三天**：戒房事（不与妻妾同房）
3. **前三天**：戒饮酒（不喝酒）
4. **前一日**：沐浴更衣（洗澡换干净衣服）

**斋戒目的**：
1. 清净身心
2. 增强灵力
3. 感通神灵

### 四、净身

**净身步骤**：
1. **沐浴**：洗澡（洗净身体）
2. **更衣**：换干净衣服（最好白色或青色）
3. **净口**：刷牙漱口（不说脏话）
4. **净手**：洗手（不摸脏物）
5. **净心**：静坐片刻（平心静气）

### 五、设坛

**设坛要求**：
1. **设坛位置**：安静、干净、向阳的房间
2. **神位设置**：供太上老君神位（或画像）
3. **供品准备**：清水一杯、香三支、蜡烛一对
4. **符纸准备**：黄纸（或红纸）三张
5. **笔墨准备**：朱砂墨（或黑墨）、新笔

**设坛示意图**：
```
    [神位]
      |
   [供品]
      |
    [书案]
```

### 六、书符

**书符步骤**：
1. **净笔**：用香熏笔（或念净笔咒）
2. **净纸**：用香熏纸（或念净纸咒）
3. **念咒**：念安魂符咒三遍
4. **书符**：一气呵成，书安魂符
5. **加盖**：加盖天师印（或指印）

**书符禁忌**：
1. 不净身不书符
2. 不斋戒不书符
3. 心不诚不书符
4. 时辰不对不书符

### 七、焚化/佩戴

**焚化法**：
1. **焚化位置**：户外向阳处（或香炉内）
2. **焚化时间**：书符当日（或次日清晨）
3. **焚化方法**：用香火焚化（边焚化边念咒）
4. **灰烬处理**：撒于流水处（或埋于洁净处）

**佩戴法**：
1. **佩戴位置**：胸前（或手臂）
2. **佩戴时间**：书符当日（或次日清晨）
3. **佩戴方法**：用红绳系好（或放入香囊）
4. **佩戴期限**：七七四十九日（或至病愈）

## 修持方法

### 一、日常修持仪轨

**每日修持**：
1. **清晨**：面向东方，念安魂符咒21遍
2. **中午**：面向南方，念安魂符咒49遍
3. **晚上**：面向西方，念安魂符咒21遍

**修持要点**：
1. **诚心**：心诚则灵
2. **持之以恒**：每日修持，不可中断
3. **记录**：记录修持感受（有助于精进）

### 二、结合其他符法

**安魂符 + 辟邪符**：
1. **用途**：魂魄不安 + 驱邪避鬼
2. **方法**：先书安魂符，再书辟邪符
3. **修持**：先念安魂符咒，再念辟邪符咒

**安魂符 + 金光神咒**：
1. **用途**：魂魄不安 + 护身驱邪
2. **方法**：先书安魂符，再诵金光神咒
3. **修持**：先念安魂符咒，再诵金光神咒

**安魂符 + 北斗神咒**：
1. **用途**：魂魄不安 + 护魂消灾
2. **方法**：先书安魂符，再诵北斗神咒
3. **修持**：先念安魂符咒，再诵北斗神咒
