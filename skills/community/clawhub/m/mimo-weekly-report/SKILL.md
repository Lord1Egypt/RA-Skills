# 周报生成器 📋

## 核心流程（5 Steps）

### Step 1: 信息收集与分类
- 收集本周工作内容（用户输入/对话记录/任务列表）
- 按项目/模块/优先级分类整理
- 标注关键成果、数据和里程碑
- 识别未完成任务和卡点

### Step 2: 成果量化
- 将模糊描述转化为量化表达
- 计算完成率、增长率等关键指标
- 对比上周/目标值体现进展
- 标注超预期完成的亮点

### Step 3: 结构生成
- 本周完成：按优先级排列，量化成果
- 进行中：进度百分比+预计完成时间
- 问题与卡点：风险描述+需要的支持
- 下周计划：目标+关键任务+预期产出
- 个人反思：本周收获/改进方向（可选）

### Step 4: 模板匹配
- 简洁版：要点式，适合快速汇报/站会/Slack
- 详细版：含数据分析和背景说明，适合正式邮件
- OKR版：对齐目标关键结果，适合OKR团队
- 项目制版：按项目维度组织，适合多项目并行

### Step 5: 输出优化
- 语言精炼，删除冗余
- 重要成果前置（倒金字塔结构）
- 数据可视化（进度条/趋势对比）
- 控制篇幅（简洁版1页/详细版2页）

## 输出模板

### 简洁版
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 周报 | {姓名} | {日期范围}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## ✅ 本周完成
1. {事项} — {成果/数据}
2. {事项} — {成果/数据}

## 🔄 进行中
- {事项}：{进度}%，预计{日期}完成

## ❌ 卡点
- {问题}：需要{支持类型}

## 📅 下周计划
1. {任务}（预期产出）
2. {任务}（预期产出）
```

### OKR版
```
## O1: {目标}
- KR1: {关键结果} — 完成{X}%（{具体数据}）
- KR2: {关键结果} — 完成{X}%（{具体数据}）
## O2: {目标}
- KR1: ...
## ⚠️ 风险
- {OKR达成风险及应对}
## 📅 下周OKR推进计划
```

## 边界约束
1. **真实性**：不虚构未完成的工作
2. **量化**：尽量用数据说明成果
3. **简洁**：避免流水账式罗列
4. **重点突出**：最多5条核心成果，其余归入"其他"
5. **前瞻性**：下周计划要具体可执行

中文输出
