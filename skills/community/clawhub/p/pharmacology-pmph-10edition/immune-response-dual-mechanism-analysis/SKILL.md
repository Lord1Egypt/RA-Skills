# immune-response-dual-mechanism-analysis

## 何时使用
- 用户询问免疫系统如何应对病原体入侵
- 需要区分固有免疫与适应性免疫的特征、组成或功能
- 分析免疫反应的速度、特异性或记忆性
- 解释T细胞或B细胞在免疫中的作用

## 执行步骤

1. **识别触发条件**：确认机体是否接触外源生物性物质且免疫系统结构完整。

2. **判断主导机制**：
   - 若反应在数分钟至数小时内发生、无抗原特异性、无记忆性 → 判定为**固有免疫**
   - 若反应需数天、具有高度抗原特异性、具备免疫记忆 → 判定为**适应性免疫**

3. **解析固有免疫组成与功能**：
   - 涉及吞噬细胞（巨噬细胞、中性粒细胞）、补体系统、干扰素等
   - 功能包括快速清除异物、启动炎症反应、向适应性免疫传递信号

4. **解析适应性免疫分支**：
   - **细胞免疫**：由T细胞介导，针对细胞内病原体（如病毒、肿瘤细胞）
   - **体液免疫**：由B细胞介导，产生抗体对抗细胞外病原体
   - 受IL-2、IFN-γ等细胞因子调控

5. **说明协同机制**：
   - 固有免疫通过抗原提呈激活适应性免疫
   - 适应性免疫产物（如抗体、细胞因子）可增强固有免疫效应

> 注意：本技能不适用于无免疫系统的生物体，也不涵盖对非生物性损伤（如物理创伤、化学刺激）的反应。
