## 🔍 阶段0：快筛（1分钟·一票否决）

**目的**：90%的标的在这一关就淘汰了，不进深研。

| **#** | **检查项**  | **方法**      | **通过标准** | **否决条件**          |
| ----- | -------- | ----------- | -------- | ----------------- |
| 1     | **能力圈**  | 自评1-5分      | ≥3分      | 1-2分→不入（❌华测导航）    |
| 2     | **估值锚点** | PE/PB 3年分位  | \<70%分位  | >80%→不入（❌中钨高新94%） |
| 3     | **叙事拥挤** | 朋友圈/雪球/研报密度 | 正常或冷清    | 🔴🔥极度拥挤→打折       |

> 🚫 任一否决 → 不入。三全绿 → 进入深研。

## 🔬 阶段1-5：五阶深研

### ① 本末 —— 先验校准

**核心**：建立P(好)的先验分布——不校准直接看证据=本末颠倒。

```
五行：🔥火 → 上火照本质
```

#### 能力圈深度标注

| **评分** | **含义**      | **先验P调整** |
| ------ | ----------- | --------- |
| 5分     | 深耕十年·赚过钱    | 不变        |
| 4分     | 从业五年·边界清晰   | 向0.5收缩20% |
| 3分     | 跨行研究·深度不够   | 向0.5收缩50% |
| 2分     | 外围了解·依赖公开信息 | 向0.5收缩80% |
| 1分     | 完全不懂        | 强设0.5     |

**不禁止投资**，但强制调整先验置信度。圈浅+证据强=可以通过后验翻回来。

#### 赛道拓扑判型

| **类型**   | **特征**    | **代表**     | **判据**        |
| -------- | --------- | ---------- | ------------- |
| **幂律**   | 赢家通吃·γ\<3 | 矿山·芯片·社交平台 | 问"砍掉所有链接还剩什么" |
| **对数正态** | 多玩家共存     | 水泥·设备·消费品  | 看市占率分散度       |

#### 先验来源差异

| **​**  | **股票**             | **股权**            |
| ------ | ------------------ | ----------------- |
| P(好)来源 | 行业β + 市场情绪 + 分析师共识 | 行业 + 团队 + 退出 + 竞争 |
| 信息基础   | 公开信息·相对对称          | 尽调信息·高度不对称        |
| 格栅权重   | 心理高·数学高            | 生物高·社会高·网络高       |

#### 🗹 一阶强制四问

| **#** | **问题**                  |
| ----- | ----------------------- |
| ①     | 能力圈打高了吗？先验已按评分调整？       |
| ②     | PE/PB在3年什么分位？（快筛复核）     |
| ③     | "我知道自己不知道"的三件事列出来       |
| ④     | 如果明天先验被证明错的离谱——最可能因为什么？ |

### ② 终始 —— 终局与生态

**核心**：投前四准则独立打分→确定P置信区间。

```
五行：🌳木 → 左木定方向
```

#### 四准则

| **准则** | **核心问题**               | **信息来源标注**             |
| ------ | ---------------------- | ---------------------- |
| **业务** | 护城河是什么？可持续多久？          | 标注判断日期                 |
| **管理** | 资本配置能力？损害过股东吗？         | "郭文叁时代"信息判"杨军时代"=自动降模糊 |
| **财务** | ROE·负债·FCF自己算的？改善还是恶化？ | 标注计算来源                 |
| **市场** | 估值 vs 内在价值？两种方法交叉验证    | 标注估值方法                 |

#### 四准则 → 仓位天花板

| **完成度** | **P区间** | **仓位上限** | **守城期上限** |
| ------- | ------- | -------- | --------- |
| 四全通     | ±10%    | ≤25%     | ≤10%      |
| 三通一模糊   | ±20%    | ≤15%     | ≤5%       |
| 两项模糊+   | ±30%    | ≤8%      | ≤3%       |
| 关键硬伤    | P不可靠    | 0%       | 0%        |

**护城河五维检查**（业务准则补充）：

| **维度** | **问题**      |
| ------ | ----------- |
| 用户粘性   | 用户离开的成本有多高？ |
| 供应商关系  | 上游集中度？成本可控？ |
| 竞争格局   | 价格战还是稳定？    |
| 政策壁垒   | 牌照/认证/政府关系？ |
| 技术迭代   | 被替代的风险？     |

### ③ 变量 —— 约束与盲区

**核心**：识别什么能杀死这个投资，什么能让它爆发。

```
五行：⚔️金 → 右金收边界
```

#### A. 死法卡片（每科一条）

| **学科** | **死法**      | **活路**      |
| ------ | ----------- | ----------- |
| 物理     | 矿难·厂毁·技术故障  | 保险·备份·冗余    |
| 数学     | 戴维斯双杀·现金流断裂 | 低负债·高现金·高分红 |
| 生物     | 竞争者入场·替代品出现 | 护城河加宽·专利·品牌 |
| 社会     | 政策突变·治理丑闻   | 合规·分散国别     |
| 心理     | 市场恐慌·叙事崩坏   | 低杠杆·长持有期    |

#### B. 估值锚点

| **PE/PB分位** | **判断** |
| ----------- | ------ |
| \<30%       | 🟢 便宜  |
| 30-70%      | 🟡 合理  |
| 70-90%      | 🟡 偏贵  |
| >90%        | 🔴 极端  |

> 周期股用PB band，港股上沿放宽5-10%。

#### C. 事件驱动检查清单

**触发条件**（任一→启动）：财报·股价±10%·重大公告·行业催化剂·信息流异常

| **#** | **问题**                  |
| ----- | ----------------------- |
| ①     | 事件改变基本面了吗？还是只变了价格？      |
| ②     | 价格是否过度反应？（1天+15%→大概率过度） |
| ③     | 其他人反应是否同质化？（都在说同一句话→危险） |
| ④     | 如果现在不操作，1个月后回头看会后悔吗？    |

> 四答全绿→可行动。两答以上红→不动。

#### D. 叙事拥挤度

| **观察**    | **判断**  |
| --------- | ------- |
| 几乎没人说     | 🟢 不拥挤  |
| 偶尔看到      | 🟡 温和   |
| 到处在说·说辞一致 | 🔴 拥挤   |
| 有人晒收益截图   | 🔴🔴 泡沫 |

#### E. 信息论三注入

1. 🟢官方 🟡第三方 🔴传闻 → 每条信息源可信度
2. 末尾标注 **知道X / 相信Y / 不知道Z**
3. 月度检查活跃标的数+分析深度（防信道过载）

## 🧠 阶段6：行为金融纠偏

### 卡尼曼三问（强制·不可跳过）

| **#** | **对抗式措辞**                | **仓位冷却**   |
| ----- | ------------------------ | ---------- |
| ①     | "如果不在我手里——我今天会用这个价格买它吗？" | ≤5%直接决策    |
| ②     | "哪些是客观数字？哪些是主观判断？客观占比？"  | 5-15%冷却24h |
| ③     | "三年后回头看，这次判断错了，最可能因为什么？" | >15%冷却72h  |

### 三重纠偏

| **检查**   | **问题**                            |
| -------- | --------------------------------- |
| **锚定检查** | 被某个数字锚定了吗？（买入价·分析师目标价·朋友成本价·历史最高） |
| **参照点**  | 换一个参照点，结论变吗？                      |
| **内外视角** | 用的是内部视角（自己的故事）还是外部视角（行业基准成功率）？    |
