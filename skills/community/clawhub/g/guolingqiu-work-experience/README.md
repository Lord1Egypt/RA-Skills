# 果粒球的工作经历skill 🎯

> 果粒球3年工作全记录——平台运营 → 电商产品经理 → 项目管理 → AI产品经理

## 这个Skill能帮你做什么？

如果你正在求职、准备面试、或者想梳理自己的工作经历，这个Skill就是为你准备的～

### 🔑 五大核心能力

| 能力 | 怎么用 |
|------|--------|
| **工作经历查询** | 聊聊你的某段工作经历，帮你结构化梳理 |
| **行业知识补充** | 不懂互联网/AI行业的术语？随时问我，用大白话讲清楚 |
| **工作细节完善** | 告诉你的经历框架，帮你补充合理的工作细节 |
| **简历编写** | 基于STAR法则生成简历初稿 |
| **面试准备** | 高频面试问题清单 + 模拟面试 |

### 🎯 适用人群

- 刚毕业在找第一份工作的应届生
- 工作1~2年想转岗/转行的职场新人
- 对互联网/AI行业好奇但不太了解的女生
- 想从运营/销售等岗位转型产品经理的求职者
