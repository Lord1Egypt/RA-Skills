# 妇科肿瘤免疫治疗方案选择

## 触发条件
- 患者为晚期、转移性或复发性妇科肿瘤；
- 已完成 MSI、dMMR、TMB 或 PD-L1 等生物标志物检测；
- 或已确认表达可靶向肿瘤抗原（如 HPV E6/E7、NY-ESO-1、WT1）。

## 执行流程

### 第一步：判断是否适用免疫检查点抑制剂（ICI）
1. **子宫颈癌**：
   - 一线治疗：在含铂化疗 ± 贝伐珠单抗基础上加用帕博利珠单抗（无论 PD-L1 状态）；
   - 后线治疗：铂类失败且 PD-L1 阳性者可单用 PD-1 抗体；既往含铂失败者可考虑 PD-1/CTLA-4 双特异性抗体。
2. **子宫内膜癌**：
   - MSI-H/dMMR 或 TMB-H：可在一线或后线使用 PD-1 抗体；
   - pMMR：联合仑伐替尼 + PD-1 抗体；
   - 最新证据支持无论生物标志物状态，一线联合化疗均可显著获益。
3. **卵巢癌**：仅限铂耐药复发且满足 MSI-H/dMMR 或 TMB-H 条件时使用 PD-1 抗体。
4. **妊娠滋养细胞肿瘤**：高危耐药且 PD-L1 广泛表达者可选用 PD-1 抗体。
5. **外阴/阴道癌**：PD-L1 阳性、TMB-H 或 MSI-H/dMMR 者可考虑 PD-1 抗体；黑色素瘤可考虑 CTLA-4 和/或 PD-1 抗体。

### 第二步：评估是否适用治疗性肿瘤疫苗
1. **适用场景**：
   - HPV 相关子宫颈病变（尤其术后预防复发）；
   - 表达 NY-ESO-1（约43%卵巢癌）或 WT1 的妇科肿瘤；
   - 免疫功能基本正常，无广泛晚期病灶。
2. **疫苗类型选择**：
   - 蛋白质/肽疫苗（如 WT1 肽疫苗）；
   - DNA/RNA 疫苗（编码 HPV E6/E7 等抗原）；
   - 树突状细胞（DC）疫苗（荷载 HPV 抗原，用于高级别鳞状上皮内病变）。
3. **注意事项**：
   - 治疗性疫苗对已建立的晚期肿瘤疗效有限；
   - 预防性 HPV VLP 疫苗对已感染者无效，但治疗后接种可降低复发率。

### 第三步：整合决策
- 若同时满足 ICI 和疫苗条件，优先考虑 ICI（尤其一线）；
- 疫苗更适用于微小残留病灶或维持治疗阶段；
- 可探索 ICI 与溶瘤病毒（如 GL-ONC1、KD01）联合策略（需参考临床试验）。

> 注意：所有免疫治疗决策需结合患者整体状况、既往治疗史及可用药物批准状态。
