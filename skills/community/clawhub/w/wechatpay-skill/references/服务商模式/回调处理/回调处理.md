# 服务商模式 - 回调处理

> 服务商模式下回调通知的处理

## 服务商回调特点

服务商模式下，支付回调会包含子商户信息：

```json
{
    "event_type": "TRANSACTION.SUCCESS",
    "resource": {
        "algorithm": "AEAD_AES_256_GCM",
        "ciphertext": "...",
        "nonce": "...",
        "associated_data": "transaction"
    }
}
```

## 解密后结构

```json
{
    "sp_mchid": "服务商商户号",
    "sub_mchid": "子商户号",
    "transaction_id": "微信订单号",
    "out_trade_no": "商户订单号",
    "trade_state": "SUCCESS",
    "amount": {
        "total": 100,
        "payer_total": 100,
        "currency": "CNY",
        "sub_order": {...}
    }
}
```

## 服务商回调处理示例

```python
class PartnerNotifyHandler:
    """服务商回调处理器"""
    
    def __init__(self, sp_mchid, api_v3_key):
        self.sp_mchid = sp_mchid
        self.api_v3_key = api_v3_key
    
    def handle_payment_notify(self, body, headers):
        """处理支付成功回调"""
        
        # 1. 验签（使用服务商公钥）
        if not self.verify_signature(body, headers):
            raise ValueError("验签失败")
        
        # 2. 解密
        payment_data = self.decrypt(body)
        
        # 3. 获取子商户信息
        sub_mchid = payment_data.get("sub_mchid")
        transaction_id = payment_data.get("transaction_id")
        
        # 4. 分发到对应子商户处理
        self.dispatch_to_sub_merchant(sub_mchid, payment_data)
        
        return {"code": "SUCCESS", "message": "成功"}
    
    def dispatch_to_sub_merchant(self, sub_mchid, payment_data):
        """根据子商户分发处理"""
        
        # 根据sub_mchid找到对应的子商户处理器
        handler = self.get_sub_handler(sub_mchid)
        
        # 处理子商户业务
        handler.process_payment(payment_data)
```

## 合单支付回调处理

```python
def handle_combine_notify(body, headers):
    """处理合单支付回调"""
    
    payment_data = decrypt(body)
    
    # 合单回调结构
    combine_out_trade_no = payment_data.get("combine_out_trade_no")
    sub_orders = payment_data.get("sub_orders", [])
    
    # 处理每个子单
    for sub_order in sub_orders:
        sub_mchid = sub_order.get("mchid")
        sub_out_trade_no = sub_order.get("out_trade_no")
        trade_state = sub_order.get("trade_state")
        
        # 处理子单
        process_sub_order(sub_mchid, sub_out_trade_no, trade_state)
    
    return {"code": "SUCCESS", "message": "成功"}
```

## 子商户回调分发

```python
# 子商户回调处理器注册
sub_handlers = {
    "sub_mchid_1": SubMerchantHandler1(),
    "sub_mchid_2": SubMerchantHandler2(),
}

def get_sub_handler(sub_mchid):
    """获取子商户处理器"""
    return sub_handlers.get(sub_mchid)

def process_sub_order(sub_mchid, out_trade_no, trade_state):
    """处理子商户订单"""
    handler = get_sub_handler(sub_mchid)
    
    if handler:
        handler.process(out_trade_no, trade_state)
    else:
        # 未知子商户，记录日志
        logger.warning(f"Unknown sub_mchid: {sub_mchid}")
```

## 注意事项

1. **必须验签**：使用服务商公钥验签
2. **子商户分发**：根据sub_mchid分发处理
3. **幂等处理**：子商户维度做幂等
4. **日志记录**：记录完整处理流程
