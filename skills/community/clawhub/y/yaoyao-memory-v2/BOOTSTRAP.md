## 📖 模块详细介绍

### ✅ core - 核心记忆（必装）

**包含**：记忆存取、FTS5 搜索、查询缓存、智能召回、记忆分类、质量管理

**功能**：
- 🧠 自动存储重要对话内容
- 🔍 快速 FTS5 全文搜索
- 📁 四层记忆架构（短期→中期→长期→档案）
- 🏷️ 自动分类（decision/preference/task/info）
- ⭐ 重要性分级（Critical/High/Normal/Low）

**适合**：所有用户

---

### ✅ health_check - 健康检测（必装）

**包含**：健康检查、自动修复

**功能**：
- 🔍 系统健康度检测
- 🔧 自动修复常见问题
- 📊 数据库状态监控
- ⚡ 性能基准测试
- 🔔 异常告警通知

**适合**：所有用户（自动保持系统健康）

---

### ✅ security - 安全治理（必装）

**包含**：安全检查、上下文隔离、工具过滤、权限管理、审计日志

**功能**：
- 🛡️ Prompt 注入检测
- 🔒 子 Agent 工具隔离
- 📋 操作审计日志
- 🚨 熔断保护机制
- 🔐 RBAC 权限控制

**适合**：注重安全的用户

---

### ☁️ cloud_backup - 云端备份（可选）

**包含**：多云适配器、备份管理、同步工具

**功能**：
- ☁️ 支持多种云服务（IMA/坚果云/阿里云 OSS/S3/Samba/SFTP）
- 📥 一键备份到云端
- 📤 一键从云端恢复
- 🔄 定时自动同步
- 🔍 增量备份（只备份变化的部分）
- 📊 备份状态监控
- 🔒 加密传输（HTTPS/SSH）

**适合**：需要多设备同步、注重数据安全或想备份到私有云的用户

**☁️ 关于 IMA**：IMA 是**腾讯**推出的 AI 智能工作台（ima.copilot），不是飞书的！集成了 DeepSeek-R1，适合做知识库。

**云服务支持**：
| 服务 | 类型 | 特点 |
|------|------|------|
| 🥜 坚果云 | WebDAV | 国内可用，稳定 |
| ☁️ **IMA (ima.copilot)** | AI 知识库 | **腾讯出品**，DeepSeek 加持 |
| 🪣 阿里云 OSS | 对象存储 | 企业级、低成本 |
| 🖥️ Samba/NAS | 局域网 | 高速、不耗流量 |
| 📡 SFTP | SSH | 安全、可自建 |

## 🔧 安装命令

```bash
# 交互式安装
python3 install_modules.py --interactive

# 完整安装（核心 + 基础模块）
python3 install_modules.py --full-install

# 查看所有模块
python3 install_modules.py --list

# 安装指定模块
python3 install_modules.py --install cloud_backup

# 查看状态
python3 install_modules.py --status
```

## 🔄 从旧版迁移

如果你已经安装了旧版 yaoyao-memory（未拆分版），运行迁移：

```bash
python3 migrate.py          # 检查迁移状态
python3 migrate.py --dry   # 预览迁移（不执行）
python3 migrate.py --force # 执行迁移
```

迁移会自动：
1. 备份现有数据到 `~/.openclaw/backup/pre-migration/`
2. 转换为模块化架构
3. 保留所有记忆文件
4. 更新配置指向新路径
