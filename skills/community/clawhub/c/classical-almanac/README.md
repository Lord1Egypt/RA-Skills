# 古典黄历（Classical Almanac）

> 基于清乾隆钦定《协纪辨方书》的古典择日系统

## 简介

这是一个基于中国传统历法《协纪辨方书》的择日系统，融合了多个维度的吉凶判断，为用户提供专业的择吉服务。

与市面上简化的黄历应用不同，本系统严格遵循《协纪辨方书》的正统体系，综合考虑建除十二神、黄道黑道、天德月德天赦、凶煞、彭祖百忌、奇门遁甲、五行生克等多个因素，提供科学、准确的择日建议。

## 核心特性

### 🏛️ 正统体系

基于清乾隆钦定《协纪辨方书》，这是中国古代择日学的集大成之作，共36卷，系统化地整理了历代择日方法。

### 📊 多维度评分

| 维度 | 权重 | 说明 |
|------|------|------|
| 建除十二神 | ±30分 | 建/除/满/平/定/执/破/危/成/收/开/闭 |
| 黄道黑道 | ±25分 | 青龙/明堂/天德/玉堂/司命（吉）vs 天刑/朱雀/白虎/天牢/玄武/勾陈（凶） |
| 天德月德天赦 | +25分 | 三大吉神值日加分 |
| 凶煞 | -25分 | 劫煞、月破等负面神煞 |
| 宜忌配合 | ±15分 | 日课宜忌与活动类型匹配度 |
| 日冲 | -20分 | 与用户八字日柱相冲 |
| 值符值使 | ±16分 | 奇门遁甲吉凶 |
| 五行生克 | ±15分 | 日课五行与用户日主关系 |

### 🎯 10类活动支持

结婚、开业、搬家、签约、订婚、装修、出行、祭祀、求财、上任

### 🔮 个性化定制

支持输入用户八字，根据日主五行定制评分，提供个性化的择日建议。

## 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/your-username/classical-zeri.git

# 进入目录
cd classical-zeri

# 无需安装依赖，直接运行
node scripts/zeri.js
```

### 使用

```bash
# 查看2026年6月结婚吉日
node scripts/zeri.js 2026-06 结婚

# 查找2026年6月最佳开业日
node scripts/zeri.js best 2026-06 开业

# 带八字的个性化择日
node scripts/zeri.js best 2026-06 结婚 "庚午 辛巳 庚辰 辛巳"
```

## 技术实现

### 算法核心

1. **建除十二神计算**: 以月支起寅，按日支顺数
2. **黄道黑道计算**: 12神按固定顺序排列
3. **天德月德查找**: 按月查日干
4. **天赦计算**: 按季查日干支
5. **凶煞判断**: 劫煞、月破等
6. **奇门遁甲**: 值符值使计算
7. **五行生克**: 日课五行与用户日主关系

### 代码特点

- **纯 JavaScript**: 无外部依赖，Node.js >= 18 即可运行
- **模块化设计**: 各计算模块独立，便于维护和扩展
- **详细输出**: 完整显示所有评分因素

## 历史背景

《协纪辨方书》是清乾隆四年（1739年）钦定编纂的择日学巨著，共36卷。该书系统整理了历代择日方法，删繁就简，建立了以五行生克衰旺为核心的择日体系，是清代官方认定的择日标准。

本系统以此为理论基础，用现代编程技术实现了传统择日算法，既保留了古法的精髓，又提供了便捷的使用体验。
