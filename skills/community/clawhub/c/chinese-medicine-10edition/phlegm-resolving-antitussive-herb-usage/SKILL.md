# Phlegm-Resolving and Antitussive Herb Usage

## When to Use
- 辨明痰的性质：观察痰的颜色、质地、量及是否易咯出，判断属寒、热、燥或湿。
- 区分主症：明确以痰盛、咳嗽或气喘为主导症状。

## Execution Steps

### 1. 辨痰定类，选择药物类别
- **痰黄黏稠、舌红苔黄** → 选用「清化热痰药」（如前胡、贝母）
  - 禁忌：脾胃虚寒、寒痰或湿痰者慎用。
- **痰白清稀、量多易咯、舌苔白腻** → 选用「温化寒痰药」（如半夏）
  - 禁忌：痰热咳嗽、阴虚燥咳、咯血者禁用。
- **以咳喘为主，无论寒热** → 选用「止咳平喘药」（如苦杏仁、桔梗）
  - 注意：根据兼夹证调整配伍。

### 2. 根据兼夹证候进行配伍
- 外感风热咳嗽：前胡配桑叶、薄荷
- 肺热燥咳：川贝母配沙参、麦冬
- 寒饮伏肺：半夏配干姜、细辛
- 肠燥便秘兼咳喘：优先选用具润肠作用的苦杏仁

### 3. 规避关键禁忌
- 半夏反乌头类药材，严禁同用；生品有毒，必须炮制后使用。
- 苦杏仁有小毒，用量不宜过大，婴儿慎用。
- 桔梗性升散，呕吐、气逆者不宜使用；大剂量可致恶心。

## Output
返回具体药物类别、推荐药材、必要配伍及明确禁忌提示，确保安全有效。
