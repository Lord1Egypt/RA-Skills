# 团队协作SOP模板

## 任务分派模板

```markdown
## 任务：XXX

### 目标
_一句话说清楚要做什么_

### 步骤
1. [步骤1] → @谁
2. [步骤2] → @谁
3. [步骤3] → @谁

### 验收标准
- [ ] 交付物是什么
- [ ] 完成标志是什么
- [ ] 截止时间
```

## 常见协作流程SOP

### 内容发布流程
```
CEO定选题 → 运营写文案 → 设计出配图 → CEO审核 → 发布
```
各角色责任：
- CEO：定方向、审核
- 运营：文案产出、关键词优化
- 设计：配图、排版

### BUG修复流程
```
CEO接反馈 → 开发定位 → 开发修复 → 开发自测 → CEO复测 → 发布
```
- 开发：定位+修复+自测
- CEO：判断优先级+复测

### 客户成交流程
```
销售接咨询 → 销售了解情况 → CEO/技术出方案 → 销售报价 → 客户付款 → 交付
```
- 销售：全程跟进
- CEO/技术：技术方案支持
