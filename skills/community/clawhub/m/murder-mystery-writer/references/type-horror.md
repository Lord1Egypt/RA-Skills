# 恐怖本创作指南

## 恐怖本的定义

恐怖本的核心体验是"惊吓"和"紧张"——让玩家在安全的环境中体验恐惧。

## 恐怖类型分类

### 心理恐怖

**特点**：不依赖视觉惊吓，而是制造心理压力。

**手法**：
- 细思极恐的细节
- 身份不确定性（"我到底是谁？"）
- 信息不对称（玩家知道危险但不知道是什么）
- 氛围营造（光线、声音、暗示）

**适合**：不喜欢jump scare的玩家

### 惊悚恐怖

**特点**：直接的感官刺激。

**手法**：
- Jump scare（突然出现的惊吓）
- 血腥场面描写
- NPC演绎（突然出现、追逐）
- 特殊道具（假血、面具）

**适合**：喜欢刺激的玩家

### 诡异恐怖

**特点**：说不清道不明的不安感。

**手法**：
- 违反常理的事件
- 真实与虚幻的界限模糊
- 不可名状的存在
- 循环/重复的异常

**适合**：喜欢克苏鲁风格的玩家

## 恐怖本设计要点

### 氛围营造

**环境设计**：
- 光线：昏暗、闪烁、突然全黑
- 声音：低频噪音、突然的响声、诡异的音乐
- 温度：刻意调低温度
- 气味：使用香薰（霉味、血腥味模拟）

**文字描写**：
- 用五感描写制造代入感
- 留白比详细描写更恐怖
- 用日常物品的异常制造不安

### 恐怖节奏

**节奏安排**：
```
铺垫期（平静）→ 小惊吓 → 缓冲期 → 大惊吓 → 高潮 → 缓冲 → 结局
```

**关键原则**：
- 不能一直吓人，玩家会疲劳
- 缓冲期也很重要，让玩家积累紧张感
- 最大的惊吓放在中后段

### 恐怖与推理的平衡

恐怖本不能只有恐怖，也要有推理。

**平衡方法**：
- 恐怖事件本身就是线索
- 破解恐怖事件的真相可以减轻恐惧
- 推理过程中的恐怖元素增加紧张感

## 恐怖本结构模板

### 标准结构（3-4小时）

```
第一幕：平静的开始（30-40分钟）
  - 建立日常感
  - 隐藏的异常细节
  - 第一个"不对劲"的信号

第二幕：异常升级（60-80分钟）
  - 恐怖事件开始发生
  - 玩家开始调查
  - 第一次惊吓（小高潮）
  - 发现更多秘密

第三幕：真相浮现（60-80分钟）
  - 恐怖事件的真相
  - 角色之间的关联
  - 第二次惊吓（大高潮）
  - 最终抉择

第四幕：结局（20-30分钟）
  - 根据选择走向不同结局
  - 恐怖解除（或延续）
```

## 恐怖本示例：6人恐怖本

### 《夜半钟声》（示例）

**类型**：恐怖本
**人数**：6人（3男3女）
**时长**：3-4小时
**背景**：现代·废弃精神病院

**故事梗概**：
六个陌生人收到神秘邀请函，来到一座废弃精神病院参加"试胆大会"。随着夜幕降临，他们发现这座病院有着不为人知的秘密，而他们每个人的到来都不是偶然。

**恐怖元素**：
- 环境恐怖：阴暗走廊、闪烁灯光、突然关闭的门
- 心理恐怖：每个人都有被遗忘的记忆
- 诡异事件：镜中出现不存在的人、时钟倒转

**推理元素**：
- 调查病院的历史
- 发现每个人之间的关联
- 破解"试胆大会"的真正目的

**结局类型**：
- 逃脱结局：所有人成功离开
- 牺牲结局：有人留下封印邪恶
- 循环结局：发现一切会重新开始
