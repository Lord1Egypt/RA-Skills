## Description: <br>
Scrapes a LinkedIn profile and generates a savage audio roast about being replaced by AI. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[xuanyuzhou98](https://clawhub.ai/user/xuanyuzhou98) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and agents use this hosted Constants skill to submit a LinkedIn profile URL and receive a humorous roast script, profile name, and generated MP3 audio. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Submitted LinkedIn profile URLs and generated roast content are processed by Constants' hosted service. <br>
Mitigation: Use the skill only when you are comfortable with Constants' service handling the profile data, and review Constants' privacy and retention practices before submitting profiles. <br>
Risk: The skill requires a Constants API key. <br>
Mitigation: Store the API key securely, avoid committing it to source control, and rotate it if exposure is suspected. <br>
Risk: Roast output can involve personal profile data and harsh or insulting language. <br>
Mitigation: Submit profiles only with consent or another appropriate basis, and review generated text before sharing it. <br>
