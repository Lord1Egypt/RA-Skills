# Altered Self-Antigen Autoimmunity Analysis

## 适用场景
- 患者有明确外伤、感染、药物或化学物质暴露史
- 出现无法用典型感染或肿瘤解释的多系统炎症或组织损伤
- 需要判断自身免疫反应是否由抗原暴露或结构相似性触发

## 执行步骤

### 1. 判断是否涉及隐蔽抗原暴露
- 检查受损组织是否属于免疫豁免部位（如眼、睾丸、中枢神经系统）
- 确认是否存在物理屏障破坏（如外伤、手术、感染）
- 若存在，则考虑隐蔽抗原释放引发自身免疫

### 2. 评估是否存在分子模拟
- 回顾近期感染病原体（如病毒、细菌）
- 查询该病原体抗原是否与人体组织抗原有已知同源性
- 若存在交叉反应性抗体或T细胞应答，则支持分子模拟机制

### 3. 分析是否发生表位扩展
- 观察自身抗体谱是否随时间扩大（如从抗组蛋白发展为抗DNA）
- 确认炎症是否持续未控制，导致组织持续破坏和新抗原释放
- 若免疫应答从优势表位蔓延至隐蔽表位，则符合表位扩展

## 输出结果
- 明确主导机制类型（单一或复合）
- 提供机制与临床表现的关联解释
- 建议针对性干预方向（如清除持续抗原、阻断交叉反应等）
