# COC 第七版 游戏系统规则

> 本章内容基于《克苏鲁的呼唤》第七版守秘人规则书第五章"游戏系统"及第八章"理智"、第六章"战斗"相关部分整理。

---

## 1. 技能检定系统

### 1.1 何时掷骰

- 只在**紧张时刻**才需要掷骰，日常琐事无需检定
- 当角色行动存在冲突、挑战或戏剧性时，才需要进行技能检定
- 玩家应在掷骰前**声明目标**，明确要实现什么
- 守秘人有最终决定权，决定使用何种技能或属性检定

### 1.2 检定方式

- 使用百分骰（1D100），骰出结果需**小于等于**目标值才能成功
- 检定结果等于或低于目标值即为成功，不存在"更高标准的成功"
- 原则上应由玩家掷骰，守秘人应尽量让玩家比自己更常掷骰

---

## 2. 成功等级

### 2.1 三种难度等级

| 难度等级 | 说明 | 掷骰要求 |
|---------|------|---------|
| **常规** | 对胜任者构成挑战的一般情形 | 骰出 ≤ 技能/属性值 |
| **困难** | 对专业人士构成挑战 | 骰出 ≤ 技能/属性值的**一半** |
| **极难** | 对专家构成挑战，触及人类可能性边界 | 骰出 ≤ 技能/属性值的**五分之一** |

> 绝大部分情况使用常规难度。困难和极难应谨慎使用。

### 2.2 六种检定结果

| 结果 | 说明 |
|------|------|
| **大成功** | 骰出 01，不仅成功还有额外好处 |
| **极难成功** | 骰出 ≤ 技能值的 1/5 |
| **困难成功** | 骰出 ≤ 技能值的 1/2 |
| **常规成功** | 骰出 ≤ 技能值 |
| **失败** | 骰出 > 技能值（非大失败） |
| **大失败** | 骰出 96-100（若目标值 < 50）；或骰出 100（若目标值 ≥ 50） |

### 2.3 面对活物时的难度等级

- 对手技能/属性 **< 50**：常规难度
- 对手技能/属性 **≥ 50**：困难难度
- 对手技能/属性 **≥ 90**：极难难度

---

## 3. 对抗检定

### 3.1 基本规则

- 双方都声明**互斥目标**（一方胜则另一方败）
- 双方各选一项技能或属性，经守秘人许可后掷骰
- 比较双方的**成功等级**，高等级胜出

### 3.2 比较优先级

```
大成功 > 极难成功 > 困难成功 > 常规成功 > 失败 > 大失败
```

### 3.3 平手处理

- 若打成平手，**技能/属性更高的一方获胜**
- 若仍然平手，达成**僵局**或双方重骰

### 3.4 对抗检定限制

- 对抗检定**不能孤注一掷**
- 属性通常比技能高，若一方使用属性值，另一方应获得选择属性或技能的机会

---

## 4. 奖励骰与惩罚骰

### 4.1 基本规则

- 只有理由**充分显著**时才使用，不能微不足道
- 一个奖励骰与一个惩罚骰**互相抵消**
- 通常最多增减**一个**奖惩骰，极端情况可用两个

### 4.2 奖励骰操作

1. 在标准百分骰（个位骰 + 十位骰）之外，**额外骰一个十位骰**
2. 共三个骰子：一个个位骰 + 两个十位骰
3. 选择**较小的**（较有利的）十位骰与个位骰组合

### 4.3 惩罚骰操作

1. 同样额外骰一个十位骰
2. 选择**较大的**（较不利的）十位骰与个位骰组合

### 4.4 使用时机

| 类型 | 适用场景 |
|------|---------|
| **技能检定** | 通过**难度等级**体现优劣势 |
| **对抗检定** | 通过**奖惩骰**体现优劣势 |

---

## 5. 属性检定

### 5.1 智力（INT）检定

- 用于破解智力谜题、解答问题
- 骰出 ≤ 智力值即成功
- 守秘人可据此提供解决方案线索

### 5.2 灵感检定

- 当调查陷入僵局时使用，同样以智力值为目标
- 成功与否决定遗漏线索是被轻松找到，还是调查员要付出某种代价

### 5.3 知识检定

- 以**教育（EDU）值**为目标
- 代表角色基于所受教育而知晓的信息
- 若有更适当的技能，应使用该技能而非知识检定

---

## 6. 幸运检定

### 6.1 使用时机

- 当**外部环境因素**主导，或需要决定命运之手时使用
- 若有适用的技能或属性，应优先使用该检定

### 6.2 团体幸运检定

- 由队伍中**幸运值最低**的玩家来骰
- 守秘人可简单判定最低幸运值的调查员遭遇不幸

### 6.3 花费幸运值（可选规则）

- 技能检定后可以**1:1花费幸运值**改变掷骰结果
- **不能**花费在：幸运检定、伤害检定、理智检定
- 大成功、大失败与枪械故障**不能**通过幸运值改变
- 使用幸运值改变结果后**不会获得技能成长标记**

### 6.4 回复幸运值

- 每次游戏聚会后，可进行幸运增强检定
- 骰 1D100，若结果 > 当前幸运值，则增加 1D10
- 幸运值**不会超过 99**

---

## 7. 理智检定基础

### 7.1 理智值（SAN）

- 每位调查员都有理智值，衡量其对精神创伤的适应性和恢复力
- **最大理智值 = 99 - 克苏鲁神话技能值**
- 理智值降至 0 时，调查员陷入**永久性疯狂**，不再由玩家控制

### 7.2 理智检定方式

- 骰百分骰，**小于等于当前理智值**即为成功
- **奖励骰与惩罚骰不适用于理智检定**（唯一例外："自救"规则）
- 大失败（100）意味着失去该遭遇中**最大理智损失量**

### 7.3 理智损失表示法

格式：`SAN X/Y`（如 `SAN 1/1D4+1`）

- 斜线**左侧**：理智检定**成功**时的损失量
- 斜线**右侧**：理智检定**失败**时的损失量

### 7.4 失败的后果

- 一个失败的理智检定会让调查员**不由自主地行动**，包括：
  - 猛的一跳（掉落物品）
  - 因恐惧而大叫
  - 不由自主的动作（如方向盘转向危险侧）
  - 不由自主的战斗行为
  - 呆立不动

### 7.5 理智损失范例

| 损失量 | 事件 |
|-------|------|
| 0/1D2 | 发现被撕碎的动物尸体 |
| 0/1D3 | 发现尸体的一部分 |
| 0/1D4 | 看到一地流淌着的鲜血 |
| 1/1D4+1 | 发现一具血肉模糊的恐怖尸体 |
| 0/1D6 | 目击一个朋友因暴力行为而死亡 |
| 0/1D6 | 看到一只食尸鬼 |
| 1/1D10 | 看到死尸复活 |
| 1D10/1D100 | 直视伟大的克苏鲁 |

### 7.6 疯狂状态

| 触发条件 | 疯狂类型 | 持续时间 |
|---------|---------|---------|
| 一次性损失 ≥ 5 点理智值 | **临时性疯狂** | 1D10 小时 |
| 一天内损失 ≥ 1/5 理智值 | **不定性疯狂** | 直到治愈或恢复 |
| 理智值降至 0 | **永久性疯狂** | 不可恢复 |

## 8. 伤害与生命值

### 8.1 生命值（耐久值/HP）

- 生命值 = (体质 + 体型) / 2（向下取整）
- 生命值不会记录负数，最低为 0

### 8.2 伤害加值

- 由力量（STR）和体型（SIZ）的总和决定
- 用于近战和投掷武器的伤害计算

### 8.3 伤害程度分类

| 伤害程度 | 判定标准 | 后果 |
|---------|---------|------|
| **轻伤** | 单次伤害 < 最大生命值的一半 | 无特殊效果，生命值归零时昏迷 |
| **重伤** | 单次伤害 ≥ 最大生命值的一半 | 立刻倒地，须体质检定避免昏迷 |
| **死亡** | 单次伤害 > 最大生命值 | 直接死亡 |

### 8.4 重伤后果

1. 在角色卡**重伤栏**做标记
2. 角色**立刻倒地**
3. 必须进行**体质检定**以避免昏迷
4. 重伤标记存在时生命值归零 → 进入**濒死状态**

### 8.5 生命值归零

- 只受轻伤：不会死亡，随时间恢复
- 受过重伤：未及时急救则**死亡**
- 单次伤害 > 最大生命值：**必死无疑**

### 8.6 濒死状态

- 重伤角色生命值归零时进入濒死
- **立刻昏迷**，每轮结束时须体质检定
- 体质检定失败 → **立即死亡**
- 只有**急救**能稳定濒死角色（医学不行）
- 急救稳定后获得 1 点临时生命值，之后每小时须体质检定

---

## 9. 治疗规则

### 9.1 急救

- 必须在受伤后**一小时内**进行
- 成功后恢复 **1 点生命值**
- 只能尝试**一次**，之后构成孤注一掷
- 两人可同时急救，任一成功即可
- 濒死角色：急救只能**暂时稳定伤势**

### 9.2 医学

- 需花费**至少一小时**
- 可在受伤后任意时间进行
- 非当天治疗需**困难成功**
- 成功后恢复 **1D3 点生命值**（追加在急救之外）
- 濒死角色：须先急救稳定，再接受医学检定

### 9.3 轻伤恢复

- 未受重伤时，以**每天 1 点**的速度恢复生命值

### 9.4 重伤恢复

每周结束时进行**体质检定**：

| 结果 | 恢复量 |
|------|-------|
| 失败 | 本周无恢复 |
| 常规成功 | 恢复 1D3 点 |
| 极难成功 | 恢复 2D3 点 |

**奖励骰条件**（可叠加）：
- 完全静养休养
- 获得有效医疗服务（照料者每周进行医学检定）

**惩罚骰条件**：
- 休养环境差且无法充分休息
- 照料者医学检定大失败

**重伤治愈条件**：
- 体质检定获得极难成功，或
- 生命值恢复到最大生命值的一半或以上

> 大失败可能导致永久性伤害、并发症（如终生跛行、单眼失明、断肢等），记入角色卡"伤口与疤痕"栏。

---

## 附：孤注一掷

- 失败后的**第二次也是最后一次**机会
- 玩家必须**合理描述**额外努力或时间花费
- 时间总是在检定间流逝（可能是几秒，也可能是几小时）
- **不能**用于：幸运检定、理智检定、战斗检定、伤害与理智损失掷骰
- 孤注一掷失败：守秘人可自由支配后果（伤害、理智损失、装备损失、被俘等）
- 守秘人可在孤注一掷前**揭示预兆**（告知失败后果）
