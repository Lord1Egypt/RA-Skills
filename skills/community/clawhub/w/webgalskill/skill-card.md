## Description: <br>
Assists with writing, checking, and optimizing WebGAL visual novel scripts using WebGAL command syntax, variables, scene flow, and presentation controls. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[zhenruyan](https://clawhub.ai/user/zhenruyan) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, writers, and visual novel creators use this skill to produce WebGAL-compatible scripts, troubleshoot syntax, and apply best practices for dialogue, scenes, assets, variables, audio, video, animation, and effects. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated WebGAL scripts may change project behavior or include incorrect syntax. <br>
Mitigation: Review generated scripts before applying them to project files and test changes in the WebGAL project. <br>
Risk: Optional build steps or configuration edits could affect game assets or project settings if applied unintentionally. <br>
Mitigation: Run build steps or edit configuration and game files only when those actions are intentional and based on trusted WebGAL sources. <br>
