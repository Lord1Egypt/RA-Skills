## Description: <br>
Connects an AI agent to BestYou health intelligence via MCP so it can query readiness, manage daily action plans, generate workouts, analyze meals, and summarize fitness, nutrition, sleep, and recovery insights. <br>

This skill is ready for commercial/non-commercial use. <br>
