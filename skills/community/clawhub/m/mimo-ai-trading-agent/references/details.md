# ai-trading-agent 详细参考

## 7大策略详解
1. 均值回归：价格偏离均值后回归，适合震荡市
2. 动量策略：强者恒强，适合趋势市
3. 突破策略：关键位突破跟进，适合盘整后
4. 网格策略：区间内高抛低吸，适合震荡市
5. 配对交易：两品种价差回归，市场中性
6. 统计套利：基于统计规律的价格偏差
7. 趋势跟踪：MA/通道跟踪，适合单边市

## TradingView指标开发
- Pine Script v5语法
- 内置指标模板：MACD背离/布林带突破/多时间框架MA
- 自定义告警条件

## Kelly公式
f* = (bp - q) / b
- b = 赔率（盈利/亏损比）
- p = 胜率
- q = 1 - p
- 实际使用建议：Kelly/2（半Kelly）更稳健

## 风控框架
- 单笔止损：≤总资金2%
- 日最大亏损：≤总资金5%
- 周最大亏损：≤总资金10%
- 相关性控制：同方向持仓≤总仓位60%
