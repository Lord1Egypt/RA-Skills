## Description: <br>
Build UI components and pages using the AIOZ UI V3 design system, including tokens, Tailwind classes, icon imports, chart components, and Figma MCP to React code mappings. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to translate Figma MCP output into production React UI built with AIOZ UI V3 components, tokens, typography utilities, icons, and charts. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Setup guidance may modify TypeScript, package, or CSS configuration and assumes local AIOZ UI workspace package paths. <br>
Mitigation: Review configuration snippets and adapt package paths and dependencies to the target project before applying them. <br>
Risk: Generated UI code depends on the AIOZ UI packages and their documented import paths being available in the project. <br>
Mitigation: Verify @aioz-ui/core, @aioz-ui/core-v3, and @aioz-ui/icon-react are installed or mapped correctly before relying on generated code. <br>
