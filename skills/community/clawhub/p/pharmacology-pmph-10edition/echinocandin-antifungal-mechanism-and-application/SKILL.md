# 棘白菌素类抗真菌药的作用机制与适用范围

## 何时使用
- 患者确诊或高度怀疑由**念珠菌属**或**曲霉菌属**引起的深部真菌感染；
- 需评估抗真菌药物选择的合理性，特别是考虑细胞壁靶向机制；
- 排除对**毛霉菌属、隐球菌属、镰刀菌属**等无效菌种的误用；
- 考虑联合用药方案（如与两性霉素B或唑类联用）以增强疗效。

## 如何执行
1. **确认病原体类型**：仅适用于念珠菌属和曲霉菌属；若为毛霉菌属、隐球菌属或镰刀菌属，则禁用。
2. **验证前提条件**：
   - 真菌处于活跃细胞壁合成状态；
   - 药物可达到有效组织浓度。
3. **理解作用机制**：
   - 棘白菌素类通过非竞争性抑制**葡聚糖合成酶**；
   - 抑制**β-1,3-D-葡聚糖**合成，破坏真菌细胞壁完整性；
   - 导致菌体裂解死亡。
4. **评估安全性**：因人体细胞无细胞壁，该类药物毒性较低，安全性较好。
5. **确定临床适应症**：
   - 念珠菌败血症；
   - 食管念珠菌病；
   - 侵袭性曲霉病。
6. **考虑联合治疗**：在重症或难治性感染中，可与其他抗真菌药（如两性霉素B、唑类）联用。

> 注意：本技能不适用于无细胞壁的病原体，且对非敏感菌种无效，务必先明确病原学诊断。
