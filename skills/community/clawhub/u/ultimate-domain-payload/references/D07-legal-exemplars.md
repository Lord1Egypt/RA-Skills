## 范本1：创业公司法律架构搭建

### 题目
互联网科技创业公司的法律架构搭建全流程

### 背景
李先生和王先生计划创办一家互联网科技公司，专注于人工智能应用开发。初始团队5人，计划天使轮融资500万元。需要搭建完整的法律架构，包括公司设立、股权架构、知识产权保护、劳动用工等方面。

### Step 1: 公司设立阶段

#### 1.1 公司设立 (L5-01)
**输入**: 创业团队信息、业务计划、初始资金
**执行**:
1. 确定公司类型：有限责任公司
2. 准备设立文件：公司章程、股东会决议、设立申请书
3. 办理工商登记：提交设立申请，获取营业执照
4. 办理后续手续：刻章、银行开户、税务登记

**输出**: 公司设立完成，取得营业执照

#### 1.2 股权架构设计 (L5-02)
**输入**: 创始人信息、业务计划、融资需求
**执行**:
1. 分析创始人贡献：技术、资金、资源
2. 设计股权结构：创始人60%，期权池20%，预留投资人20%
3. 考虑控制权安排：投票权、董事会席位
4. 设计激励机制：期权计划、成熟条件
5. 制定股权架构方案

**输出**: 股权架构方案

**校准点1**: 股权架构确认
- 确认股权比例合理
- 确认控制权安排明确
- 确认激励机制可行

### Step 2: 知识产权保护阶段

#### 2.1 专利策略 (L4-01)
**输入**: 技术方案、商业目标、竞争环境
**执行**:
1. 分析技术方案：AI算法、应用场景
2. 评估商业价值：市场潜力、竞争优势
3. 检索现有专利：专利数据库检索
4. 制定申请策略：核心专利、外围专利
5. 规划专利布局：申请时间、地域布局

**输出**: 专利策略报告

#### 2.2 商业秘密保护 (L4-04)
**输入**: 秘密信息范围、保密措施需求
**执行**:
1. 确定秘密范围：算法、数据、客户信息
2. 制定保密措施：访问控制、加密、监控
3. 签订保密协议：员工保密协议、合作伙伴保密协议
4. 建立保密制度：保密管理制度、保密培训
5. 监控保密执行：定期检查、违规处理

**输出**: 商业秘密保护方案

### Step 3: 劳动用工阶段

#### 3.1 劳动法风险评估 (L1-06)
**输入**: 用工模式、劳动合同需求、规章制度
**执行**:
1. 分析用工模式：全职、兼职、实习生
2. 审查劳动合同：必备条款、竞业限制、保密条款
3. 评估规章制度：考勤、绩效、奖惩
4. 识别法律风险：劳动纠纷风险、合规风险
5. 提出改进建议：合同优化、制度完善

**输出**: 劳动法风险评估报告

#### 3.2 合规制度设计 (L3-02)
**输入**: 劳动法要求、公司实际情况
**执行**:
1. 分析合规要求：劳动法、社保法、公积金条例
2. 设计制度框架：员工手册、管理制度
3. 制定具体制度：考勤制度、薪酬制度、绩效制度
4. 审核制度内容：合法性审查
5. 发布实施：培训、公示、执行

**输出**: 合规制度文件

**校准点2**: 劳动用工合规确认
- 确认劳动合同合法有效
- 确认规章制度合规
- 确认社保公积金合规

### Step 4: 融资准备阶段

#### 4.1 融资法律准备 (L5-05)
**输入**: 融资计划、公司资料、投资人信息
**执行**:
1. 确定融资方式：股权融资
2. 准备融资文件：商业计划书、财务预测、法律文件
3. 进行法律尽职调查：公司合规性审查
4. 起草融资协议：投资协议、股东协议
5. 完成融资交割：签署协议、资金到账

**输出**: 融资法律文件清单

#### 4.2 合同审查 (L2-03)
**输入**: 融资协议、投资条款清单
**执行**:
1. 审查投资协议：估值、股权比例、董事会席位
2. 审查股东协议：投票权、反稀释、优先权
3. 评估法律风险：条款风险、合规风险
4. 提出修改建议：条款优化、风险防控
5. 出具审查意见：法律意见书

**输出**: 合同审查意见

**校准点3**: 融资法律架构确认
- 确认融资协议合法有效
- 确认股东协议公平合理
- 确认法律风险可控

## 范本2：劳动合同纠纷处理

### 题目
科技公司劳动合同纠纷处理全流程

### 背景
某科技公司员工张某因绩效考核不合格被公司解除劳动合同，张某认为公司解除程序违法，申请劳动仲裁。公司需要处理这起劳动合同纠纷，维护合法权益。

### Step 1: 纠纷评估阶段

#### 1.1 纠纷评估 (L6-01)
**输入**: 纠纷事实、劳动合同、绩效考核记录
**执行**:
1. 了解纠纷事实：解除原因、程序、员工反应
2. 分析法律关系：劳动关系、解除依据
3. 评估证据材料：劳动合同、绩效记录、解除通知
4. 分析法律风险：程序违法风险、赔偿风险
5. 制定解决策略：协商优先、仲裁准备

**输出**: 纠纷评估报告

#### 1.2 证据收集 (L6-05)
**输入**: 纠纷相关证据需求
**执行**:
1. 确定证据需求：解除依据、程序合法性、员工过错
2. 识别证据来源：公司记录、员工档案、沟通记录
3. 收集证据材料：劳动合同、绩效考核表、培训记录
4. 保全证据：公证、电子存证
5. 整理证据清单：证据分类、证明目的

**输出**: 证据清单

**校准点1**: 纠纷评估确认
- 确认纠纷事实清楚
- 确认证据材料充分
- 确认法律风险可控

### Step 2: 协商调解阶段

#### 2.1 协商调解 (L6-02)
**输入**: 纠纷评估报告、调解意愿
**执行**:
1. 确定调解意愿：公司愿意协商解决
2. 选择调解方式：公司内部调解、劳动仲裁调解
3. 准备调解方案：经济补偿、离职安排
4. 进行调解：与员工沟通、寻求共识
5. 达成调解协议：签署调解协议、履行协议

**输出**: 调解方案

#### 2.2 合同纠纷处理 (L2-07)
**输入**: 纠纷事实、合同约定、证据材料
**执行**:
1. 分析纠纷原因：绩效考核标准、解除程序
2. 评估法律风险：程序违法风险、赔偿金额
3. 制定解决策略：协商方案、仲裁准备
4. 进行协商：与员工沟通、寻求和解
5. 执行解决方案：签署协议、履行义务

**输出**: 合同纠纷处理方案

**校准点2**: 协商调解确认
- 确认调解意愿真实
- 确认调解方案可行
- 确认调解协议合法

### Step 3: 仲裁准备阶段

#### 3.1 仲裁准备 (L6-03)
**输入**: 调解失败情况、仲裁需求
**执行**:
1. 确认仲裁协议：劳动合同仲裁条款
2. 准备仲裁申请：申请书、证据材料
3. 收集证据材料：补充证据、证人证言
4. 选择仲裁员：劳动法专业仲裁员
5. 准备庭审：庭审提纲、质证意见

**输出**: 仲裁准备文件清单

#### 3.2 庭审应对 (L6-06)
**输入**: 仲裁申请、证据材料、法律依据
**执行**:
1. 准备庭审提纲：争议焦点、举证计划
2. 整理证据材料：证据清单、证明目的
3. 准备质证意见：对员工证据的质证
4. 准备辩论意见：法律适用、程序合法性
5. 进行庭审：陈述、举证、辩论

**输出**: 庭审准备文件

**校准点3**: 仲裁准备确认
- 确认仲裁申请合法
- 确认证据材料充分
- 确认庭审准备充分

### Step 4: 执行阶段

#### 4.1 执行与上诉 (L6-07)
**输入**: 仲裁裁决书
**执行**:
1. 分析仲裁结果：胜诉、败诉、部分胜诉
2. 决定是否上诉：评估上诉必要性
3. 准备执行申请：如胜诉，申请强制执行
4. 申请强制执行：提交执行申请
5. 处理执行异议：如员工提出异议

**输出**: 执行/上诉方案

#### 4.2 合规改进 (L3-07)
**输入**: 纠纷处理结果、暴露的问题
**执行**:
1. 评估整改效果：绩效考核制度改进
2. 识别改进机会：劳动合同管理、解除程序
3. 制定改进计划：制度优化、培训加强
4. 实施改进措施：修订制度、培训员工
5. 评估改进效果：合规检查、风险评估

**输出**: 合规改进报告

**校准点4**: 执行确认
- 确认执行程序合法
- 确认整改措施有效
- 确认风险得到控制

## 范本3：跨境知识产权保护

### 题目
中国科技企业海外知识产权保护全流程

### 背景
某中国科技企业计划进军美国市场，需要在美国进行知识产权保护。企业拥有多项核心技术和商标，需要在美国进行专利申请、商标注册，并建立知识产权保护体系。

### Step 1: 知识产权评估阶段

#### 1.1 知识产权评估 (L4-07)
**输入**: 知识产权清单、评估目的
**执行**:
1. 确定评估范围：专利、商标、商业秘密
2. 选择评估方法：成本法、市场法、收益法
3. 收集评估数据：技术资料、市场数据、财务数据
4. 进行价值评估：评估各项知识产权价值
5. 出具评估报告：评估结论、价值分析

**输出**: 知识产权评估报告

#### 1.2 法律环境扫描 (L1-03)
**输入**: 目标市场、法律领域
**执行**:
1. 确定扫描范围：美国知识产权法律
2. 收集法律法规：专利法、商标法、商业秘密法
3. 分析政策趋势：审查标准、保护力度
4. 识别法律风险：申请风险、侵权风险
5. 提出应对建议：申请策略、保护措施

**输出**: 法律环境报告

**校准点1**: 知识产权评估确认
- 确认评估范围全面
- 确认评估方法适当
- 确认评估结论可靠

### Step 2: 专利保护阶段

#### 2.1 专利策略 (L4-01)
**输入**: 技术方案、商业目标、竞争环境
**执行**:
1. 分析技术方案：核心技术、应用场景
2. 评估商业价值：市场潜力、竞争优势
3. 检索现有专利：美国专利数据库检索
4. 制定申请策略：临时申请、正式申请
5. 规划专利布局：申请时间、权利要求设计

**输出**: 专利策略报告

#### 2.2 专利申请准备
**输入**: 专利策略、技术资料
**执行**:
1. 准备申请文件：说明书、权利要求书、摘要
2. 翻译文件：英文翻译、术语校对
3. 选择代理人：美国专利代理人
4. 提交申请：临时申请或正式申请
5. 跟踪审查：答复审查意见、修改申请

**输出**: 专利申请文件

**校准点2**: 专利保护确认
- 确认专利策略合理
- 确认申请文件完备
- 确认申请程序合规

### Step 3: 商标保护阶段

#### 3.1 商标注册 (L4-02)
**输入**: 商标图样、商品/服务类别
**执行**:
1. 设计商标图样：英文商标、图形商标
2. 确定注册类别：核心类别、关联类别
3. 进行商标检索：美国商标数据库检索
4. 准备申请材料：商标申请书、使用证据
5. 提交注册申请：美国专利商标局申请

**输出**: 商标注册申请

#### 3.2 商标保护策略
**输入**: 商标注册情况、市场计划
**执行**:
1. 制定使用规范：商标使用标准、使用证据收集
2. 建立监控机制：商标侵权监控、市场巡查
3. 准备维权材料：侵权证据、维权策略
4. 建立应急机制：侵权应对、法律行动
5. 制定管理规范：商标管理制度、档案管理

**输出**: 商标保护策略

**校准点3**: 商标保护确认
- 确认商标注册顺利
- 确认保护策略可行
- 确认管理制度完善

### Step 4: 综合保护阶段

#### 4.1 知识产权许可 (L4-05)
**输入**: 知识产权清单、许可需求
**执行**:
1. 确定许可标的：专利、商标、技术秘密
2. 选择许可方式：独占许可、排他许可、普通许可
3. 确定许可条件：许可费、使用范围、期限
4. 起草许可合同：许可协议、保密条款
5. 签订许可协议：谈判、签署、备案

**输出**: 知识产权许可合同

#### 4.2 知识产权维权 (L4-06)
**输入**: 侵权信息、权利证明
**执行**:
1. 收集侵权证据：侵权产品、销售证据
2. 分析侵权行为：侵权类型、侵权范围
3. 制定维权策略：警告函、行政投诉、诉讼
4. 采取维权措施：发送警告函、提起诉讼
5. 评估维权效果：维权结果、损失赔偿

**输出**: 知识产权维权方案

**校准点4**: 综合保护确认
- 确认许可安排合理
- 确认维权策略有效
- 确认保护体系完善

### Step 5: 持续管理阶段

#### 5.1 合规管理 (L3-04)
**输入**: 知识产权管理制度、合规要求
**执行**:
1. 制定检查计划：定期检查、专项检查
2. 设计检查清单：专利维护、商标使用、保密措施
3. 实施检查：现场检查、文件审查
4. 记录检查结果：检查发现、问题记录
5. 分析检查发现：合规状况、改进建议

**输出**: 合规检查报告

#### 5.2 持续改进 (L3-07)
**输入**: 检查结果、改进需求
**执行**:
1. 评估整改效果：问题解决情况
2. 识别改进机会：管理优化、技术升级
3. 制定改进计划：改进目标、改进措施
4. 实施改进措施：制度修订、系统升级
5. 评估改进效果：合规状况、管理效率

**输出**: 持续改进报告

**校准点5**: 持续管理确认
- 确认合规检查有效
- 确认改进措施落实
