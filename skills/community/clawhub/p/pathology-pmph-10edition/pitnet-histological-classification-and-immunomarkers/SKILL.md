# PitNET组织学分型与免疫标志物判读

## 适用前提
- 已获取垂体肿瘤组织标本
- 已完成免疫组织化学染色（包括PIT1、TPIT、SF1及GH、PRL、TSH、ACTH、FSH、LH等）
- 病理确诊为垂体神经内分泌瘤（PitNET）

## 执行步骤

### 1. 确定细胞谱系
通过三大核心转录因子的免疫表达确定基本谱系：
- **PIT1阳性** → PIT1谱系（包括GH、PRL、TSH相关肿瘤）
- **TPIT阳性** → TPIT谱系（促肾上腺皮质激素细胞瘤）
- **SF1阳性** → SF1谱系（促性腺激素细胞瘤）

### 2. 细分PIT1谱系亚型
若为PIT1谱系，结合激素表达进一步分类：
- **生长激素细胞瘤**：GH↑，IGF1↑，PIT1+，GH+
- **催乳素细胞瘤**：PRL↑，PIT1+，PRL+
- **催乳素生长激素细胞瘤**：同一细胞GH+且PRL+，PIT1强+
- **促甲状腺激素细胞瘤**：TSH↑（少数），TSH+，多为嫌色细胞
- **成熟性多激素PIT1谱系瘤**：GH广泛+，伴PRL/TSH+，老年人多见
- **未成熟PIT1谱系瘤**：TSH/GH/PRL局灶表达，核异型明显
- **嗜酸性干细胞瘤**：PIT1强+，PRL±，GH灶性或阴性，浸润性生长
- **混合型GH-PRL瘤**：含两种独立细胞群，分别表达GH或PRL

### 3. 判读非PIT1谱系
- **TPIT谱系**：ACTH+，可致Cushing综合征或Nelson综合征
- **SF1谱系**：FSH/LH+，临床表现为性功能减退

### 4. 处理无明确谱系病例
- **零细胞瘤**：所有转录因子及激素阴性，仅Syn/CgA+
- **多激素瘤**：单一或多形态细胞分泌多种激素，常PIT1+SF1共表达

### 5. 特殊情况
如存在多发同步不同谱系肿瘤，应分别记录各成分的亚型及免疫表型。

## 输出要求
输出具体的PitNET组织学亚型名称及其对应的免疫表型组合，用于临床预后评估和治疗决策。
