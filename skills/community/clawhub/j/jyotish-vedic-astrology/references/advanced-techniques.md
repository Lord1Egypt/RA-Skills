# 印度占星高级技法参考

## 一、Gochara 行星过境（Transit）

### 核心原则
过境（Gochara）以**月亮星座（Janma Rashi）**为基准宫位，评估各行星过境的吉凶。

### 土星过境（最重要）

**Sade Sati（七年半土星压力期）**
土星过境月亮前一宫、月亮所在宫、月亮后一宫，共约7.5年。

| 阶段 | 土星位置 | 主题 |
|------|----------|------|
| 第一阶段 | 月亮前一宫 | 家庭压力、精神准备 |
| 第二阶段（峰值） | 月亮所在宫 | 业力清算、健康挑战、重大变化 |
| 第三阶段 | 月亮后一宫 | 收尾、适应新秩序 |

**Ashtama Shani（8宫土星）**
土星过境月亮第8宫，约2.5年，为次要挑战期。

**土星过境各宫吉凶（以月亮宫为1宫）**
- 1、2、4、5、8、9、12宫：挑战期
- 3、6、11宫：相对有利

### 木星过境（每约12个月换一宫）

**Jupiter Transit 吉宫（以月亮宫为基准）**
- 2、5、7、9、11宫：扩张、机遇、祝福
- 1、6、8、12宫：挑战或收缩

**木星与上升的关系**
- 过境上升：身体健康改善，整体运势提升
- 过境5宫：子女、创造力、智慧方面有喜讯
- 过境9宫：命运转折，贵人出现

### Rahu/Ketu 过境（约1.5年换一宫）

Rahu 过境带来外来能量、欲望激增、不稳定；
Ketu 过境带来分离、内化、灵性倾向。

> **来源标签**: 【现代演绎·Skill整合】 — 高级技法整合（Transit+Dasha+AV交叉分析框架）

## 二、Ashtakavarga（八宫积分系统）

### 原理
每个行星对命盘中8个关键点（7大行星+上升）各贡献积分，形成各宫位的总分（Sarvashtakavarga）。

**宫位积分解读（Sarvashtakavarga）**
| 分数 | 解读 |
|------|------|
| 25分以上 | 该宫位领域非常强大，行星过境此宫带来收益 |
| 20-25分 | 中等偏强 |
| 15-20分 | 中等 |
| 15分以下 | 该宫位较弱，行星过境此宫需注意 |

**土星 Ashtakavarga 特别用途（Bhinna Ashtakavarga）**
土星自身的积分表，用于精准判断某一年土星过境哪个宫位最吉或最凶。

## 三、Shadbala 行星力量六要素

评估每个行星在命盘中的综合力量，由六个维度组成：

| 要素 | 梵文 | 含义 |
|------|------|------|
| 1. 位置力 | Sthana Bala | 行星在旺/庙/友宫的力量加成 |
| 2. 方向力 | Dig Bala | 行星在最强方位宫的力量（木星/水星在1宫最强；金星/月亮在4宫；火星/土星在10宫；太阳在9宫） |
| 3. 时间力 | Kala Bala | 白天/夜晚出生对不同行星有利 |
| 4. 运动力 | Chesta Bala | 行星顺行/逆行的力量 |
| 5. 自然力 | Naisargika Bala | 固定顺序：日>月>金>木>水>火>土 |
| 6. 相位力 | Drig Bala | 受其他行星相位影响的加减 |

**实用简化判断：**
- 旺（Exalted）：力量最强 ✅
- 庙（Moolatrikona/Own Sign）：力量强 ✅  
- 友宫（Friendly Sign）：力量中等 ✅
- 敌宫（Enemy Sign）：力量弱 ⚠️
- 弱（Debilitated）：力量最弱 ❌
- 逆行（Retrograde）：力量加强但内化，效果迟到或颠覆性
- 燃烧（Combust，距太阳过近）：力量被压制 ❌

## 四、特殊行星状态

### 逆行（Vakri）
- 行星逆行时力量实际增强，但其效果会延迟、内化、或以非常规方式显现
- 逆行土星：业力主题尤为强烈
- 逆行木星：已有的好事可能被收回或推迟

### 燃烧（Combust / Asta）
行星距太阳过近，光芒被太阳压制：
| 行星 | 燃烧度数（与太阳的角距） |
|------|--------------------------|
| 月亮 | 12°以内 |
| 水星 | 14°以内（逆行时8°） |
| 金星 | 10°以内（逆行时8°） |
| 火星 | 17°以内 |
| 木星 | 11°以内 |
| 土星 | 15°以内 |

### 战争（Graha Yuddha / Planetary War）
两颗行星在同一星座且经度差在1°以内，发生"行星战争"。
纬度较低（或黄经较小）的行星获胜，另一方力量受损。

### 深陷（Ati Neecha）与深旺（Ati Uchcha）
超过旺/弱度数后会递减效果，恰好在旺度数上力量最强。

## 五、Navamsa（D-9 分盘）解读要点

Navamsa 是印度占星最重要的分盘，又称"第二命盘"。

**用途：**
1. 婚姻与伴侣质量的深层评估
2. 命运在后半生的深化展现（36岁后尤为重要）
3. 验证本命盘行星的真实力量

**关键规则：**
- 行星在本命盘旺，但在 Navamsa 弱 → 实际效果打折
- 行星在本命盘弱，但在 Navamsa 旺 → 后期逆转（Neecha Bhanga 效应增强）
- Vargottama（本命盘与Navamsa同星座）：该行星力量极强

**婚姻评估：**
- 7宫主在 Navamsa 中的位置与状态
- Navamsa 上升主星力量
- 金星（女命）/ 木星（男命）在 Navamsa 中的位置

## 六、Neecha Bhanga（弱势逆转）规则

行星处于弱位（Debilitation），但以下条件可逆转其弱势：

1. 弱位行星的旺主星在角宫（1/4/7/10宫）
2. 弱位行星的主星在角宫
3. 弱位行星与旺主星形成交换或相望
4. 弱位行星位于角宫

**效果：** 拥有 Neecha Bhanga 的人，弱星所代表的领域往往历经磨难后大放异彩，成就反而可能超越普通人。

## 七、特殊宫位组合

### Bhavat Bhavam（宫之宫）
某宫与它自身等距离所指向的宫位，形成共鸣：
- 5宫的5宫 = 9宫（5宫主题的扩展）
- 7宫的7宫 = 1宫（婚姻反映自我）
- 4宫的4宫 = 7宫（家庭与婚姻关联）

### 行星相位（Graha Drishti）
- **所有行星**：7宫对冲相位（互望）
- **木星**：额外相位 5宫、9宫
- **土星**：额外相位 3宫、10宫
- **火星**：额外相位 4宫、8宫
- **Rahu/Ketu**：5宫、9宫（部分传统认可）
