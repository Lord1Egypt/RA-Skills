# 神经元发生与成熟过程

## 触发条件
当胚胎发育进入第4周后且神经上皮开始分化时，启动神经元发生程序。

## 执行步骤

### 1. 中枢神经系统神经元发生
- 起源于神经管的成神经细胞。
- 初始为无极成神经细胞（圆形，无突起）。
- 发出两个突起成为双极成神经细胞。
- 朝向神经管腔的突起退化，外周突起增长形成原始轴突，转变为单极成神经细胞。
- 内侧端再形成多个短突起（原始树突），成为多极成神经细胞。
- 最终发育为成熟神经元。

### 2. 周围神经系统神经元发生
- **感觉神经元**（脑、脊神经节）：
  - 神经嵴细胞 → 成神经细胞 → 双极神经元；
  - 因细胞体不均等生长，两突起基部靠拢融合，形成假单极神经元。
- **交感神经节细胞**：
  - 神经嵴细胞 → 交感成神经细胞 → 多极交感神经节细胞。
- **副交感神经节**：起源尚存争议（可能来自神经管或脑神经节成神经细胞）。

### 3. 神经元数量调控
- 初始生成数量远超最终存活数。
- 未能与靶细胞建立连接或位置异常的神经元发生凋亡。
- 存活依赖靶细胞释放的神经营养因子（如NGF、FGF、EGF）。
- 亦依赖与其他神经元形成足够传入性突触。

### 4. 结构成熟标志（以第20周大脑皮质为例）
- 胞体与树突内出现成层排列的粗面内质网。
- 含大量微丝、微管和溶酶体。

## 注意事项
- 本流程不适用于胶质细胞。
- 不适用于成年后神经再生（除海马齿状回、嗅球等特定区域）。
- 前提条件：神经管或神经嵴已形成，且存在成神经细胞。
