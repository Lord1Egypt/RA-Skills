# 参考范围与T₄/TBG比值计算说明

## 实验室参考区间
- **TT₄**：65～155 nmol/L
- **FT₄**：10.3～25.7 pmol/L
- **TBG**：15～34 mg/L

## T₄/TBG比值计算方法
1. 将TT₄单位转换为 μg/L：
   - 1 nmol/L TT₄ ≈ 0.0777 μg/L
   - 公式：`TT4_ug_per_L = TT4_nmoll * 0.0777`
2. 计算比值：
   - `T4_TBG_ratio = TT4_ug_per_L / TBG_mg_per_L`

## 临床意义解释
- **比值在正常范围（3.1～4.5）**：即使TT₄异常，若比值正常，提示甲状腺功能实际正常，异常由TBG变化引起
- **比值降低（0.2～2.0）**：支持甲减诊断，尤其当TT₄降低但TBG也降低时
- **比值升高（7.6～14.8）**：支持甲亢诊断，尤其当TT₄升高但TBG也升高时

## 常见干扰因素
- **TBG升高**：妊娠、雌激素、急性间歇性卟啉病、HIV治疗、遗传性TBG增多症
- **TBG降低**：雄激素、糖皮质激素、严重肝病、肾病综合征、遗传性TBG缺乏症

> 注意：T₄/TBG比值仅在TT₄与临床不符且无法获取可靠FT₄时作为辅助手段；现代实验室普遍提供FT₄，应优先使用。
