# Pop-Up Shop Budget Planning Guide

## Budget Templates by Scale

### Micro Pop-Up ($3,000-8,000)
Best for: Market/fair booths, 1-3 day events, first-time testing
| Category | % Range | Typical Amount |
|---|---|---|
| Venue/booth fee | 20-30% | $600-2,400 |
| Display fixtures + signage | 25-35% | $750-2,800 |
| Marketing | 15-20% | $450-1,600 |
| Staffing | 10-15% | $300-1,200 |
| Inventory (incremental) | 5-10% | $150-800 |
| Contingency | 10% | $300-800 |

### Standard Pop-Up ($10,000-25,000)
Best for: 1-2 week retail storefront, brand awareness campaigns
| Category | % Range | Typical Amount |
|---|---|---|
| Venue (lease + utilities + insurance) | 18-25% | $1,800-6,250 |
| Build-out (fixtures + signage + lighting) | 20-28% | $2,000-7,000 |
| Marketing (ads + PR + influencer) | 20-25% | $2,000-6,250 |
| Staffing (wages + training) | 15-20% | $1,500-5,000 |
| Inventory (product + packaging) | 5-10% | $500-2,500 |
| Technology (POS + WiFi + data tools) | 3-5% | $300-1,250 |
| Contingency | 7-10% | $700-2,500 |

### Premium Pop-Up ($25,000-75,000)
Best for: 2-6 week immersive experience, market testing for permanent retail
| Category | % Range | Typical Amount |
|---|---|---|
| Venue (lease + utilities + insurance) | 20-25% | $5,000-18,750 |
| Build-out (custom fixtures + full design) | 22-30% | $5,500-22,500 |

| Technology + data capture | 3-5% | $750-3,750 |
| Events/activations (DJ, workshops, etc.) | 3-5% | $750-3,750 |
| Contingency | 7-10% | $1,750-7,500 |

## Cost Line Item Details

### Venue Costs
| Item | Range | Notes |
|---|---|---|
| Lease/rent | $50-200/sq ft/month | Varies wildly by city and location |
| Security deposit | 1-2 months rent | Often negotiable for short-term |
| Utilities (electric, water) | $200-800/month | Ask if included in lease |
| WiFi/internet | $100-300/month | Essential; verify speed for POS |
| General liability insurance | $300-800/event | Required by most landlords |
| Property damage deposit | $500-2,000 | Refundable if no damage |
| Permits/licenses | $100-1,000 | Varies by city; check 4+ weeks ahead |
| Sales tax registration | $0-100 | Required in most states |

### Build-Out Costs
| Item | Range | Notes |
|---|---|---|
| Modular display fixtures | $500-5,000 | Reusable across pop-ups (invest here) |
| Signage (exterior) | $200-2,000 | Vinyl banners cheapest; neon/LED premium |
| Signage (interior) | $150-1,500 | Price tags, category markers, brand story |
| Lighting (supplemental) | $200-1,500 | Track lighting, spotlights, accent |
| Flooring (if needed) | $300-2,000 | Rugs, vinyl overlay, branded mat |
| Paint/wall treatment | $200-1,000 | Check lease for restore requirements |
| Fitting room (if needed) | $300-1,500 | Curtain + mirror minimum |
| Content backdrop | $200-800 | Instagram-worthy photo moment |

### Marketing Costs
| Item | Range | Notes |
|---|---|---|
| Social media ads (geo-targeted) | $500-5,000 | 10-mile radius, 4-week campaign |
| Influencer partnerships | $500-10,000 | Micro-influencers: $200-500 each |
| PR outreach | $0-3,000 | DIY or hire local PR firm |
| Printed materials (flyers, postcards) | $100-500 | For local distribution |
| Event listing fees | $0-200 | Many free local event calendars |

### Staffing Costs
| Role | Hourly Range | Notes |
|---|---|---|
| Pop-up lead/manager | $25-45/hr | Most experienced team member |
| Brand ambassador | $18-30/hr | Personality and brand fit critical |
| POS operator | $16-25/hr | Retail experience preferred |
| Content creator | $20-40/hr | Or contract per day/event |
| Security (if needed) | $20-35/hr | Required for high-value inventory |
| Training time | Same rates | Budget 3-4 hours per staff member |

## ROI Calculation Framework

### Direct ROI
```
Direct ROI = (Pop-Up Revenue - Total Cost) / Total Cost × 100
```
Note: Many pop-ups have negative direct ROI. This is expected if the primary objective is brand awareness or customer acquisition.

### Customer Lifetime Value ROI
```
LTV ROI = (New Customers × Average LTV × Conversion Rate) / Total Cost × 100
```
Where:
- New Customers = emails/SMS captured
- Average LTV = average customer value over 12-24 months
- Conversion Rate = % of captured leads who become paying customers (typically 10-25%)

### Total Attributed ROI
```
Total ROI = (Direct Revenue + Post-Event Online Lift + LTV Value of New Customers + PR/Media Value) / Total Cost × 100
```

### Break-Even Calculator
```
Break-Even Transactions = Total Fixed Costs / Average Margin per Transaction
Break-Even Daily Traffic = Break-Even Transactions / Duration Days / Expected Conversion Rate
```

Example:
- Total cost: $18,000
- Average margin per transaction: $28
- Duration: 5 selling days
- Expected conversion rate: 20%
- Break-even transactions: 643
- Break-even daily traffic: 643 / 5 / 0.20 = 643 visitors/day

If expected foot traffic is 300/day, direct break-even is unlikely — the pop-up must be justified by customer acquisition value and brand awareness.
