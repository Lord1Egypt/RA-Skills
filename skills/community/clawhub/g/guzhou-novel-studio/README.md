# 🚢 孤舟小说工作室 | Guzhou Novel Studio

> 一人一舟，渡创作之海

一套完整的AI辅助小说创作技能系统，整合调研、创作、风格、质检、长篇管理五大模块，支持从短篇到长篇的全流程创作。

---

## ✨ 核心亮点

### 🧠 风格学习与进化

提取任意作品的风格特征（句式节奏、用词偏好、叙事视角等），保持全文风格统一。支持风格自我迭代进化，越写越像你想要的调性。

### 🔍 QA自检系统

自动检测逻辑漏洞、人物设定矛盾、时间线错乱等常见问题。确保伏笔回收、前后文一致，告别「写着写着圆不回来」的困境。

### 📏 多篇幅智能适配

- **短篇**：800-3000字，紧凑节奏
- **中篇**：8000-30000字，人物丰满
- **长篇**：每章5000字+，长期连载

根据篇幅自动调整叙事密度、支线数量、节奏把控。

### 🎯 五大模块协同

```
调研(Research) → 创作(Creation) → 风格(Style) → 质检(QA) → 长篇管理(Studio)
```

| 模块 | 核心能力 |
|------|----------|
| **调研** | 开题/续写/改写调研，先问再写 |
| **创作** | 短篇/中篇/长篇，多种结构 |
| **风格** | 风格提取、保持、进化 |
| **质检** | 10维度检测，问题追踪 |
| **长篇** | 伏笔追踪，时间线管理 |

### 🔄 智能路由，自动调度

根据用户需求自动识别场景，调度对应模块：

| 用户说 | 自动路由到 |
|--------|-----------|
| 「写个故事」「帮我创作」| 创作模块 |
| 「开题」「调研」「卡文了」| 调研模块 |
| 「续写」「写下一章」| 长篇创作 + 质检 |
| 「检查一下」「有问题吗」| 质检模块 |
| 「学学XX风格」| 风格模块 |

### 📝 支持多种篇幅

| 类型 | 字数 | 特点 |
|------|------|------|
| **短篇故事** | 800-3000字 | 完整故事，紧凑节奏，适合快速阅读 |
| **中篇小说** | 8000-30000字 | 人物丰满，情节层次，适合深度叙事 |
| **长篇连载** | 每章5000字起，可连载数十万至上百万字 | 完整世界观，多线叙事，长期陪伴 |

## 🎨 内置风格库

四种经典风格，一键切换：

### 王小波风格：荒诞中的自由

> 「生活就是个缓慢受锤的过程，人一天天老下去，奢望也一天天消失。」

**核心特点**：
- 黑色幽默：用玩笑消解严肃
- 狂欢化叙事：打破常规，自由跳跃
- 智性反思：在荒诞中思考人生

**适用**：黑色幽默、都市讽刺、对权力结构的解构

### 古龙风格：留白中的侠义

> 「他没有动，因为他知道，只要他一动，就有人会死。」

**核心特点**：
- 电报体短句：三字一句，五字一段
- 留白艺术：不写尽，让读者自己补
- 存在主义：孤独、选择、责任

**适用**：硬汉侦探、武侠江湖、杀手故事

### 王家卫电影风格：碎片中的时光

> 「不知道从什么时候开始，在什么东西上面都有个日期。」

**核心特点**：
- 碎片叙事：跳跃、重组、拼贴
- 时间独白：精确到分秒的时间标记
- 都市疏离：霓虹灯下的孤独

**适用**：都市情感、边缘人物、文艺叙事

### 吴宇森电影风格：暴力中的诗意

> 「暴力不是目的，暴力是到达目的的手段。」

**核心特点**：
- 暴力美学：血与鸽子的视觉碰撞
- 双雄对决：英雄相惜，正邪模糊
- 慢镜头诗化：暴力变成舞蹈

**适用**：犯罪题材、黑帮故事、双雄叙事

## 🔧 模块详解

### 1. 调研模块 (Researcher)

**核心原则**：先读 → 先问 → 再给方案

**三大场景**：
- **开题调研**：从零开始，帮你理清人物、情节、世界观
- **续写调研**：卡文时帮你理清后面几章怎么走
- **改写调研**：不满意时先问清楚再改

**触发方式**：
- 「帮我理一下后面几章怎么走」
- 「卡文了，先调研再写」
- 「这段不满意，先调研再改」

### 2. 创作模块 (Creator)

**支持篇幅**：

| 类型 | 字数 | 结构建议 |
|------|------|----------|
| 短篇 | 800-3000字 | 单线叙事，一个核心冲突，干净利落的结局 |
| 中篇 | 8000-30000字 | 双线或三线，人物成长弧光，层次丰富 |
| 长篇 | 每章5000字+ | 多线并行，完整世界观，伏笔系统 |

**写作技法支持**：
- 双线镜像叙事：两条线相互映照，最终汇聚
- 灰色人设：正邪模糊，每个角色都有自己的立场
- 草蛇灰线：伏笔隐蔽，回头再看方知深意
- 硬核智斗：靠智力而非外挂解决问题

### 3. 风格模块 (Stylist)

**核心能力**：
- **Style-DNA 提取**：从范文中提取风格特征
- **风格进化分析**：以上一章为标杆持续进化
- **去AI化**：杜绝翻译腔、模板句、机械重复

**使用方式**：
```
用户：用古龙风格写一章
系统：📖 已加载【古龙风格】
      - 短句留白，三字一句五字一段
      - 不写尽，让读者自己补
      开始创作...
```

### 4. 质检模块 (QA)

**10大检测维度**：

| 维度 | 检查内容 |
|------|----------|
| 人设一致性 | 性格、习惯、能力前后矛盾 |
| 时间线一致性 | 日期顺序、事件先后、年龄对应 |
| 情节逻辑 | 因果关系、动机合理性 |
| 对话合理性 | 说话方式、语境违和 |
| 伏笔闭环 | 悬而未决的线索 |
| 称呼一致性 | 角色称呼前后统一 |
| 场景一致性 | 地点、空间布局矛盾 |
| 物品一致性 | 关键物品合理性 |
| 情感逻辑 | 情绪反应是否合理 |
| 文风自然度 | AI味、翻译腔检测 |

**问题分级**：
- 🔴 **致命**：完全破坏故事逻辑，必须修复
- 🟠 **严重**：认真读者必然发现的不一致
- 🟡 **一般**：可感知但不影响阅读
- 🟢 **轻微**：瑕疵级别

### 5. 长篇管理模块 (Studio)

**核心能力**：
- 项目初始化：统一的目录结构
- 强制写作流程：读设定 → 风格进化 → 写作 → 检查
- 记忆系统：伏笔追踪、时间线管理、人物状态
- 本地备份：章节自动保存
