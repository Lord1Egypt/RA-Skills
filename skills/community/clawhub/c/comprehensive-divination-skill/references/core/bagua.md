## 八卦卦象速查表

### 乾（☰）——天
- 五行：金
- 人物：君、父、老人、官贵
- 身体：头、骨、肺
- 动物：马、龙
- 场所：京都、高堂、楼台
- 物象：金银珠宝、圆形物、刚硬物
- 时间：戌亥年月日时

### 兑（☱）——泽
- 五行：金
- 人物：少女、妾、歌手、巫师
- 身体：口、舌、肺
- 动物：羊
- 场所：沼泽、废井、山崩地裂
- 物象：金属乐器、缺口之物、带刃之物
- 时间：酉年月日时

### 离（☲）——火
- 五行：火
- 人物：中女、文人、目疾者
- 身体：目、心、上焦
- 动物：雉（野鸡）、龟、蟹
- 场所：南方、窑灶、炉冶
- 物象：文书、字画、铠甲、干燥物
- 时间：午年月日时

### 震（☳）——雷
- 五行：木
- 人物：长男
- 身体：足、肝、发
- 动物：龙、蛇
- 场所：东方、山林、大道
- 物象：花草树木、乐器（竹木）、声响
- 时间：卯年月日时

### 巽（☴）——风
- 五行：木
- 人物：长女、寡妇、僧道
- 身体：股、气、胆
- 动物：鸡
- 场所：东南方、园林、寺庙
- 物象：绳索、木工具、香气
- 时间：辰巳年月日时

### 坎（☵）——水
- 五行：水
- 人物：中男、盗贼、渔人
- 身体：耳、肾、血
- 动物：猪、鱼
- 场所：北方、江河湖海、酒肆
- 物象：液体、水具、弓轮
- 时间：子年月日时

### 艮（☶）——山
- 五行：土
- 人物：少男、山人
- 身体：手、指、背、鼻
- 动物：狗、鼠
- 场所：东北方、山路、门阙
- 物象：土石、果实、阻隔物
- 时间：丑寅年月日时

### 坤（☷）——地
- 五行：土
- 人物：母、老妇、众人
- 身体：腹、脾胃
- 动物：牛
- 场所：西南方、田野、仓库
- 物象：布帛、容器、柔软物
- 时间：未申年月日时

## 六十四卦速查

梅花易数和六爻均使用六十四卦。八卦两两相叠（上卦/外卦 + 下卦/内卦），共 8×8=64 卦。
