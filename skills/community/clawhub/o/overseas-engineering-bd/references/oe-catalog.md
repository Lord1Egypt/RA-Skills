# 海外工程BD任务清单与依赖拓扑

海外工程业务拓展按价值链组织的任务清单，附依赖关系和UTOS元操作映射提示。

## 依赖拓扑摘要

以下为跨域的主要依赖链路，UTOS执行轴可据此自动编排管线：

**项目情报主链路**: OE1-01 → OE1-02 → OE2-02 → OE2-04 → OE3-03
（爬虫采集 → PDF处理 → 信息提取 → 背景调查 → 看板展示）

**伙伴名录主链路**: OE1-03 → OE2-03 → OE2-06 → OE3-01
（文库搜索 → 数据清洗 → 画像生成 → 导入CRM）

**触达转化主链路**: OE3-01 → OE4-02 → OE4-03 → OE5-01
（CRM数据 → 邮件设计 → 话术生成 → SOP执行）

**联系补全链路**: OE2-02 → OE2-05 → OE3-01
（信息提取 → 联系补全 → 导入CRM）

**WhatsApp触达链路**: OE2-02 → OE4-01 → OE4-03 → OE5-01
（信息提取 → 号码预检 → 话术生成 → 触达执行）

更多组合由UTOS根据具体任务动态推导。
