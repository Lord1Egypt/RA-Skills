# Judging Strategy

## Positioning

Neway Commerce OS is not just a site generator. It is a reusable commerce operating system for launching digital product storefronts and multi-product creator businesses with AI-guided conversion.

## Why this can score well

### 1. Website completeness
It generates:
- storefront
- cart
- checkout
- account
- admin
- assistant entrypoint
- edge deployment structure

### 2. Skill quality
It ships with:
- clear SKILL.md trigger language
- reusable references
- scaffold script
- validation script
- deployment notes
- handoff expectations

### 3. Technical depth
It explicitly combines:
- React + Vite frontend
- Hono function structure
- edge + node split
- stripe-style payment flow
- AI assistant integration point
- EdgeOne deployment setup

### 4. Story and传播力
The story is easy to explain:
- one prompt
- one operating system
- many commerce launches

## Demo recommendation

Best demo prompt:
“Build a premium multi-product storefront for NewayStudio that sells UI kits, AI workflow packs, and creator tools, with an AI shopping concierge and Stripe checkout.”

## What to emphasize in a submission

- repeatability
- reusable business system
- short-prompt generation
- clean architecture
- clear extension points
- conversion-focused UX
