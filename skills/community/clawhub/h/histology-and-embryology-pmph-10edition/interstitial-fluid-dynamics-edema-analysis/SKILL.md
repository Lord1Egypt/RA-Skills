# 组织液动态平衡与水肿分析

## 何时使用
- 用户提及组织水肿、局部肿胀、脱水或微循环异常
- 涉及毛细血管通透性、血浆胶体渗透压、静脉压或淋巴引流功能的问题
- 需评估体液在组织间隙的分布是否失衡

## 执行步骤
1. **确认适用对象**：仅适用于开放组织间隙（如皮下、肌肉等），不适用于封闭腔隙（如关节腔）或急性出血场景。
2. **评估组织液生成**：检查毛细血管动脉端是否有小分子水溶液（含电解质、单糖、气体）渗出至基质。
3. **评估组织液回流**：
   - 约90%应在毛细血管静脉端重吸收回血
   - 约10%应经毛细淋巴管进入淋巴系统并最终回流入血
4. **判断平衡状态**：
   - 若生成 > 回流（因血浆蛋白↓、毛细血管通透性↑、静脉压↑或淋巴管阻塞）→ 提示**水肿风险**
   - 若生成 < 回流（因严重脱水或低血容量）→ 提示**脱水风险**
   - 否则 → 判定为**正常**
5. **输出结果**：返回“提示组织液平衡状态：正常 / 水肿风险 / 脱水风险”。

## 关键前提条件
- 正常毛细血管通透性
- 血浆蛋白浓度稳定
- 淋巴回流通畅

> 注意：若前提条件不满足，需优先考虑病理机制而非生理性平衡。
