# 腋窝淋巴结压痛评估

## 触发条件
- 正在进行淋巴结体格检查
- 触及腋窝区域淋巴结

## 准备工作
- 患者取坐位或仰卧位
- 上肢外展
- 检查者右手查左侧，左手查右侧

## 执行步骤

1. **系统触诊五组淋巴结**
   - 中央组
   - 前（胸肌）组
   - 后（肩胛下）组
   - 外侧（肱）组
   - 尖组

2. **压痛评估方法**
   - 施加轻柔但稳定的压力
   - 询问患者是否感到“疼痛”或“不适”
   - 观察面部表情或退缩反应

3. **判定标准**
   - **存在压痛**：提示急性炎症、感染或近期免疫激活（如乳腺炎、上肢感染）
   - **无压痛**：常见于慢性病变、肿瘤转移或正常淋巴结

4. **综合判断要点**
   - “压痛”是炎症性/反应性肿大的重要体征
   - 无痛性肿大需高度警惕恶性肿瘤
   - 必须结合大小、硬度、活动度、边界等综合评估

> 输出应为临床警报：若压痛阳性，提示感染可能；若无痛性肿大，建议进一步排查肿瘤。
