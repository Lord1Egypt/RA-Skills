# Brief Master Examples

Start with [`messy-to-clean-brief.md`](messy-to-clean-brief.md).

It shows the core move of Brief Master: take a fuzzy request, preserve the intent, and turn it into an agent brief with scope, constraints, acceptance criteria, non-goals, and verification commands.

Use it when you want to see what changes between "please improve this" and a brief an implementation agent can actually execute.
