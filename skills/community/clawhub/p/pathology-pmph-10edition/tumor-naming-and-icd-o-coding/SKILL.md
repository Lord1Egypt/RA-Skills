# 肿瘤命名与ICD-O编码

## 何时使用
当需要对已明确组织来源（如上皮、间叶、造血等）和生物学行为（良性、交界性、原位癌或恶性）的肿瘤进行标准化命名或分配ICD-O编码时，应用本技能。适用于病历书写、肿瘤登记、医保申报及流行病学统计等场景。

## 执行步骤

### 1. 确定命名规则
- **良性肿瘤**：组织/细胞类型 + “瘤”（-oma），例如腺瘤、平滑肌瘤。
- **恶性肿瘤**：
  - 上皮来源 → 称“癌”（carcinoma），如鳞状细胞癌、腺癌；
  - 间叶来源 → 称“肉瘤”（sarcoma），如脂肪肉瘤、骨肉瘤；
  - 同时含癌与肉瘤成分 → 称“癌肉瘤”（carcinosarcoma）。

### 2. 处理特殊命名情形
识别并正确应用以下特例：
- “母细胞瘤”（-blastoma）可能为良性或恶性；
- 名称含“病”或“瘤”但实为恶性（如白血病、精原细胞瘤）；
- 以人名命名（如尤因肉瘤）、以形态命名（如透明细胞肉瘤）；
- “瘤病”（-omatosis）表示多发状态（如神经纤维瘤病）；
- 畸胎瘤需根据成熟度判断良恶性。

### 3. 分配ICD-O编码
- 使用四位主码标识肿瘤类型（如肝细胞肿瘤为8170）；
- 添加斜线后缀表示生物学行为：
  - `/0` = 良性
  - `/1` = 交界性或生物学行为未定
  - `/2` = 原位癌（CIS）或非侵袭性
  - `/3` = 恶性
- 组合格式：`主码/行为码`（如肝细胞腺瘤 = 8170/0；肝细胞癌 = 8170/3）。

### 4. 输出结果
输出标准化中文名称及完整ICD-O编码，用于病历记录、统计与医保申报。

> **注意**：不适用于未明确组织来源的未分化肿瘤的初步命名；历史特例需单独记忆。
