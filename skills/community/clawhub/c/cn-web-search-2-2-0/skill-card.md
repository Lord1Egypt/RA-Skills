## Known Risks and Mitigations: <br>
Risk: Search terms are sent to public search engines and content sites. <br>
Mitigation: Avoid entering secrets, confidential business information, private personal details, or sensitive research topics as queries. <br>
Risk: Search results from public web scraping can be stale, incomplete, or misleading. <br>
Mitigation: Cross-check important answers across multiple engines and review source pages before acting on the information. <br>
Risk: The reviewed artifacts include stale README/package metadata relative to the skill frontmatter and server release metadata. <br>
Mitigation: Use the server-resolved release metadata for the public release version and review artifact metadata before republishing or updating the skill. <br>
