# 个人财务基准参考

> 本文档提供常见的财务健康指标，用于在月度报告中给出有意义的参考对比。
> 所有数据为通用参考值，个人情况千差万别，仅作方向性指引，不作硬性标准。

## 1. 储蓄率目标

| 阶段 | 推荐储蓄率 | 说明 |
|------|-----------|------|
| 刚工作 / 还债期 | ≥ 10% | 先养成习惯，金额不重要 |
| 稳定期 | 20% – 30% | 大多数理财建议的"甜蜜点" |
| 财务自由追求期 | 30% – 50% | 加速积累，需要较强的消费纪律 |

**怎么算：**
```
储蓄率 = (收入 − 支出) / 收入 × 100%
```

**月度报告中的表述示例：**
- 储蓄率 > 30%："本月储蓄率 35%，超过推荐的 30% 目标，继续保持！"
- 储蓄率 10%–30%："本月储蓄率 22%，在健康范围内。如果目标是加速储蓄，可以看看餐饮和购物有没有优化空间。"
- 储蓄率 < 10%："本月储蓄率偏低（8%）。建议先排查是否有大额意外支出，后续可以尝试把目标设到 15% 起步。"

## 2. 分类支出占比参考

以下为税后月收入的百分比参考，适用于中国城市普通工薪家庭：

| 分类 | 健康范围 | 警戒线 | 说明 |
|------|---------|--------|------|
| 🏠 居住（房租/房贷+水电物业） | 20% – 30% | > 40% | 含房租、房贷、水电燃气、物业 |
| 🍜 餐饮 | 15% – 25% | > 35% | 含一日三餐、外卖、零食饮料 |
| 🚌 交通 | 5% – 10% | > 15% | 含通勤、打车、公共交通 |
| 🛍️ 购物 | 5% – 15% | > 25% | 含服饰、日用品、电子产品 |
| 🎮 娱乐 | 3% – 8% | > 15% | 含视频会员、游戏、电影、社交 |
| 📚 教育 | 2% – 5% | > 10% | 含课程、书籍、技能培训 |
| 🏥 医疗 | 2% – 5% | > 10% | 含体检、药品、保险（不含社保） |
| 📱 通讯 | 1% – 3% | > 5% | 含话费、网费 |
| 💝 社交人情 | 3% – 8% | > 15% | 含红包、礼物、请客 |
| 💰 储蓄/投资 | 20% – 30% | < 10% | 这是"剩下来"的部分 |

## 3. 健康财务指标速查表

### 核心三问

| 指标 | 健康值 | 怎么算 |
|------|--------|--------|
| 应急基金 | ≥ 3 个月支出 | 把最近 3 个月的月均支出 × 3 |
| 储蓄率 | ≥ 20% | (收入 − 支出) / 收入 |
| 负债收入比 | ≤ 35% | 月还贷总额 / 月收入 |

### 进阶指标

| 指标 | 说明 | 健康范围 |
|------|------|---------|
| 流动性比率 | 能随时拿出的钱 / 月支出 | 3 – 6 |
| 负债资产比 | 总负债 / 总资产 | < 50% |
| 投资占资产比 | 投资类资产 / 总资产 | 随年龄递增，25岁>10%，35岁>30% |
| 保险覆盖率 | 年保费 / 年收入 | 5% – 10% |

### 简易评分卡

用以下 5 项快速评估财务健康度，每项 20 分：

1. **每月有结余**（储蓄率 > 10%）— 20 分
2. **有应急基金**（≥ 1 个月支出）— 20 分
3. **无高息负债**（无信用卡分期、网贷）— 20 分
4. **知道钱花在哪了**（有记账习惯）— 20 分
5. **有储蓄/投资目标**（哪怕是模糊的）— 20 分

- 80–100：财务状况良好 👍
- 60–79：基本健康，有优化空间
- 40–59：需要关注，建议从记账开始
- < 40：需要行动，优先处理高息负债

## 4. 中国普通家庭消费结构参考

数据来源：国家统计局居民消费支出分类（2023 年大致水平），城镇居民人均年消费支出约 3 万元。

| 类别 | 占比 |
|------|------|
| 食品烟酒 | 29% |
| 居住 | 24% |
| 交通通信 | 14% |
| 教育文化娱乐 | 10% |
| 医疗保健 | 9% |
| 衣着 | 6% |
| 生活用品及服务 | 5% |
| 其他用品及服务 | 3% |

**注意：** 这是全国平均水平，一二线城市居住占比会显著更高（可达 35%–40%）。

## 5. 不同收入水平的通用理财建议

### 月收入 < 5000 元
- **优先目标**：建立记账习惯 + 应急基金（哪怕每月存 200 元）
- **消费建议**：居住控制在 1500 以内，餐饮自己做为主
- **投资建议**：先别想投资，把钱存银行活期或货币基金就行
- **关键动作**：开始记账，知道钱去哪了

### 月收入 5000 – 10000 元
- **优先目标**：应急基金攒到 1–2 万 + 偿还高息负债
- **消费建议**：居住 30% 以内，餐饮 25% 以内
- **投资建议**：可以从货币基金、定期存款开始，学习理财基础知识
- **关键动作**：让储蓄率稳定在 15% 以上

### 月收入 10000 – 20000 元
- **优先目标**：应急基金 3–6 个月 + 开始配置保险 + 建立投资组合
- **消费建议**：居住 30% 以内，娱乐消费设预算上限
- **投资建议**：指数基金定投（沪深300 / 中证500），每月固定比例
- **关键动作**：建立系统性理财规划，不只是存钱

### 月收入 20000 元以上
- **优先目标**：资产配置多元化（房产/基金/债券/保险）
- **消费建议**：避免"生活方式膨胀"（赚得多花得更多）
- **投资建议**：根据风险承受能力分配资产，考虑长期规划（养老、子女教育）
- **关键动作**：关注资产增值效率，而不仅仅是存钱速度

## 在月度报告中的使用方式

Agent 在生成月度报告时，应：

1. **计算用户的实际储蓄率**，与 20% 基准对比
2. **计算各分类占收入百分比**，标注超出警戒线的项目
3. **如果数据充足（≥ 3 个月）**，给出趋势判断（改善 / 恶化 / 稳定）
4. **每次只提 1–2 个最值得关注的点**，不要把所有指标都列一遍
5. **用鼓励为主的语言**，避免让用户产生焦虑

**报告建议的标准模板：**

```
💡 本月财务小结

储蓄率 XX%（目标 ≥20%）：[达标/偏低/优秀]
最值得关注的支出：[分类名] 占收入 XX%，[正常/偏高/需留意]
[一句话具体建议]
```
