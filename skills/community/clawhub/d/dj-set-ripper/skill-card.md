## Description: <br>
Download individual songs from a DJ set or mix by extracting a tracklist from supported set pages or pasted text, then sourcing each track and producing a status log. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Robinnnnn](https://clawhub.ai/user/Robinnnnn) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and music workflow users use this skill to turn DJ set or mix links, or pasted tracklists, into individual downloaded track files with a status log for each track. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Bulk download and rename workflows may make broad file changes based on untrusted track names. <br>
Mitigation: Use a fresh dedicated output folder, review the extracted tracklist before downloads, and sanitize artist and title values before filename normalization. <br>
