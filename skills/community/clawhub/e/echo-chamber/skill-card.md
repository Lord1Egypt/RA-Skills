## Description: <br>
Echoes back whatever the user says when asked to echo or repeat something. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[bristy](https://clawhub.ai/user/bristy) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Users invoke this skill when they want an agent to repeat supplied text exactly, or to return a simple default echo phrase when no input is supplied. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill repeats supplied text, which can make secrets or private content visible in the conversation. <br>
Mitigation: Avoid invoking it on secrets, credentials, or private content that should not be echoed. <br>
