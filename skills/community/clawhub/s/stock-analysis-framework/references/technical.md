# 技术面分析详细指南

## 一、均线系统（MA）

### 常用均线
| 均线 | 周期 | 意义 |
|:----:|:----:|------|
| MA5 | 5日 | 短线攻击线 |
| MA10 | 10日 | 短线生命线 |
| MA20 | 20日 | 月线，中线趋势 |
| MA60 | 60日 | 季线，中期牛熊分界 |
| MA120 | 120日 | 半年线 |
| MA250 | 250日 | 年线，长期牛熊分界 |

### 均线形态
```
多头排列：MA5 > MA10 > MA20 > MA60 → 上涨趋势 🟢
空头排列：MA5 < MA10 < MA20 < MA60 → 下跌趋势 🔴
粘合：均线纠缠一起 → 即将选择方向
乖离过大：价格远离均线 → 可能回归
```

### 金叉/死叉
- **金叉**：短线上穿长线 → 买入信号
- **死叉**：短线跌破长线 → 卖出信号
- 金叉/死叉在底部/顶部更有效
- 结合成交量：放量金叉更可靠

## 二、常用技术指标

### RSI（相对强弱指标）
- 参数：默认14日
- 超买：>70，可能回调
- 超卖：<30，可能反弹
- 背离：
  - 顶背离：股价新高，RSI不新高 → 见顶
  - 底背离：股价新低，RSI不新低 → 见底

### MACD（指数平滑异同移动平均线）
- 参数：12日EMA / 26日EMA / 9日信号线
- 0轴以上 = 多头市场
- 0轴以下 = 空头市场
- 金叉（DIF上穿DEA）= 买入信号
- 死叉（DIF跌破DEA）= 卖出信号
- 柱体（红绿柱）：
  - 红柱变长 = 多头力量增强
  - 红柱变短 = 多头力量减弱
  - 绿柱变长 = 空头力量增强
  - 绿柱变短 = 空头力量减弱

### KDJ（随机指标）
- K值：快线
- D值：慢线
- J值：方向线
- 超买：K>80，D>70
- 超卖：K<20，D<30
- 金叉：K上穿D
- 死叉：K跌破D

### BOLL（布林带）
- 中轨：MA20
- 上轨：中轨+2倍标准差
- 下轨：中轨-2倍标准差
- 碰触上轨 = 超买
- 碰触下轨 = 超卖
- 开口=波动加大
- 收口=波动减小，突破在即

## 五、成交量分析

| 现象 | 含义 |
|------|------|
| 底部放量 | 可能见底，有资金进场 |
| 底部缩量 | 下跌动能减弱 |
| 顶部放量滞涨 | 出货信号 🔴 |
| 顶部缩量上涨 | 动能不足 ⚠️ |
| 突破放量 | 趋势确认 ✅ |
| 突破缩量 | 假突破 ⚠️ |

## 六、技术分析纪律

1. **不要用太多指标** — 1个趋势型+1个震荡型足够
2. **按周期分析** — 先看日线定方向，再看60分/30分钟找买卖点
3. **量价要配合** — 没有成交量的技术信号不可靠
4. **支撑压力位跟踪** — 前高前低是天然的支撑压力
5. **有效期有限** — 技术分析在趋势行情最有效，震荡市容易吃假信号
