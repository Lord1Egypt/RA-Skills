## 命题类型识别

### 审题定性规范（硬性要求）

审题的第一步必须用**标准术语**对题目进行定性，这是高考作文审题的基本规范。

**四大标准术语**：

| 标准术语 | 定义 | 判定标志 |
|----------|------|----------|
| **材料作文** | 提供文字/漫画/名言等材料，要求从中提炼观点展开论述 | 题目有材料但无特定身份/文体/情境指令 |
| **任务驱动型作文** | 材料+任务指令，要求在特定情境中完成写作任务 | 题目出现"写一篇演讲稿/倡议书/书信"等文体要求，或"以XX身份""给XX"等身份/对象要求，或"在XX活动中"等情境要求 |
| **命题作文** | 直接给定标题 | 题目给出完整标题，如"以《XX》为题" |
| **话题作文** | 给定话题范围，自定立意 | 题目出现"以XX为话题"，不限定具体观点 |

**严禁使用的非标准表述**：
- ❌ "应用文体写作题" → ✅ 应为"任务驱动型作文"
- ❌ "给材料作文" → ✅ 应为"材料作文"
- ❌ "新材料作文" → ✅ 应为"材料作文"或"任务驱动型作文"（根据是否有任务指令判定）
- ❌ "半命题作文" → ✅ 应为"命题作文"（半命题是命题的子类）

**输出格式要求**：审题分析开头第一行必须为标准术语定性，格式如：
> **题目类型：任务驱动型作文**

---

### 类型特征速查

| 类型 | 特征 | 占比 | 审题重点 |
|------|------|------|----------|
| 材料作文 | 文字/漫画/名言为载体，提炼观点展开论述 | >80% | 紧扣材料，不可脱离 |
| 任务驱动型作文 | 材料+任务指令，限定身份/文体/读者 | 全国卷主流 | 审材料+审任务，重点审任务 |
| 命题/话题作文 | 给定标题或话题范围 | 较少 | 挖掘标题/话题内涵 |

**快速判断法**：题目中出现"请写一篇演讲稿/倡议书/书信"等明确文体要求，或"以XX身份""给XX"等身份/对象要求，即为**任务驱动型作文**。

## 审题三法

### 方法一：关键词法
**适用**：材料中有明确核心概念

**步骤**：
1. 圈画高频词、强调词、对比词
2. 分析每个关键词的内涵
3. 梳理关键词之间的关系（因果/递进/对比/辩证）

**示例**：2022年全国乙卷"本手、妙手、俗手"
- 本手=基础，妙手=创造，俗手=失误
- 关系：本手是妙手的前提，妙手是本手的升华，俗手是偏离本手的结果
- 立意：夯实基础方能创新，急功近利易致失败

### 方法二：逻辑分析法
**适用**：材料有因果/递进/对比逻辑

**步骤**：
1. 识别材料中的逻辑关系
2. 梳理"现象→原因→结果"或"观点→论据→结论"
3. 找到逻辑链条的薄弱环节或深层本质

**示例**：2023年全国甲卷"人·技术·时间"
- 现象：技术改变时间感知
- 本质：人应保持对时间的主体性
- 立意：让技术成为时间的赋能者而非吞噬者

### 方法三：任务指令法
**适用**：任务驱动型作文

**步骤**：
1. 识别四大任务要素：身份、对象、场景、文体
2. 明确内容指令：围绕什么内容写
3. 确认写作边界：哪些必须写、哪些不能写

**任务要素识别表**：

| 要素 | 常见表述 | 示例 |
|------|----------|------|
| 身份 | "以XX身份""请你作为XX" | 青年学生/人大代表/志愿者 |
| 对象 | "写给XX""向XX介绍" | 同龄人/校领导/外国友人 |
| 场景 | "在XX活动中""针对XX问题" | 班级演讲/论坛发言/网络评论 |
| 文体 | "写一篇XX""以XX形式" | 演讲稿/倡议书/辩论词/书信 |

## 立意三策

### 策略一：本质挖掘法
**路径**：现象→本质→规律

**操作**：
1. 描述材料呈现的表层现象
2. 追问"为什么"，挖掘背后的价值观/方法论/社会规律
3. 再追问"所以呢"，提炼具有启发性的观点

### 策略二：辩证思维法
**路径**：对立→统一→转化

**操作**：
1. 找到材料中的对立概念（传统与现代/个人与集体/快与慢）
2. 分析二者的统一关系（相辅相成/互为前提）
3. 探讨转化条件（在什么情况下A转化为B）

### 策略三：角度创新法
**路径**：常规→升维→独特

**操作**：
1. 先找到最常规的立意角度（个人层面）
2. 尝试升维到社会层面/文化层面/哲学层面
3. 结合个人体验使抽象观点落地

**立意深度对照**（必须使用"浅层/中层/深层"标准等级标注，不可替换为其他描述）：

| 层次 | 特征 | 示例 |
|------|------|------|
| 浅层 | 就事论事，停留现象 | "要重视基础" |
| 中层 | 揭示关系，辩证思考 | "基础与创新相辅相成" |
| 深层 | 透视本质，启发思考 | "真正的创新，是对基础的理解达到自由后的自然绽放" |

**立意层面框架**（立意方向必须按此框架组织）：

| 层面 | 立意范围 | 示例（以"坚守"话题为例） |
|------|----------|----------|
| 个人层面 | 个体品质、个人成长、自我修养 | 坚守初心、坚守诚信、坚守自我 |
| 社会层面 | 社会风气、文化传承、公共价值 | 坚守公德、坚守匠心、坚守文化根脉 |
| 国家层面 | 家国情怀、民族精神、时代担当 | 坚守家国、坚守民族精神、坚守文化自信 |

**文体适配指导**：当题目"文体不限"时，必须为不同文体推荐适合的立意方向：

| 文体 | 适合的立意特点 | 示例（以"不平凡的时代"为例） |
|------|----------------|------|
| 议论文 | 逻辑性强、辩证关系明确 | 时代的不平凡由平凡人的坚守构成 |
| 记叙文 | 有故事性、有人物命运 | 一个普通人在疫情中的选择 |
| 散文 | 有情感深度、有意象渲染 | 在喧嚣中守住内心的锚

## 结构三式

### 式一：递进式（推荐首选）
**适用**：观点型作文，需要层层深入

```
开头（15%）：引述材料+亮明观点
  ↓
第一层：是什么——界定核心概念
  ↓
第二层：为什么——分析原因/意义
  ↓
第三层：怎么办——提出路径/方法
  ↓
结尾（15%）：升华主题+回扣材料
```

**段落黄金三角**：每段 = 论点句 + 论据（事例/名言） + 分析句（揭示本质）

### 式二：并列式
**适用**：多角度分析的作文

```
开头：引述材料+亮明观点
  ↓
分论点一（角度A）——论点+论据+分析
分论点二（角度B）——论点+论据+分析
分论点三（角度C）——论点+论据+分析
  ↓
结尾：归纳升华
```

**注意**：分论点之间必须有逻辑关联，不能简单堆砌。

### 式三：对比式
**适用**：有正反案例的作文

```
开头：引述材料+亮明观点
  ↓
正面论证：正面事例+分析
反面论证：反面事例+分析
辩证综合：正反对比+提炼规律
  ↓
结尾：总结升华
```

## 任务驱动型写作指南

### 三步满分攻略

**第一步：拆任务**——精准识别题目中的"任务密码"
- 找到任务关键词（藏在"写作任务""请完成下列写作任务"等字样后）
- 识别四大任务要素：身份、对象、场景、文体

**第二步：定框架**——根据任务类型选择"黄金结构"
- 判断文体类型：议论类 or 应用类
- 套用标准结构：每种文体有约定俗成的格式
- 调整内容比例：不同文体侧重点不同

**第三步：填内容**——"观点+论据+任务语言"三合一
- 亮明观点：开篇或段首明确核心论点
- 精选论据：选择能支撑观点的典型素材
- 嵌入任务语言：在关键位置回扣任务要求

### 常用应用文体格式

**演讲稿/发言稿**：
```
标题
尊敬的XX，亲爱的XX：（顶格称呼）
大家好！（问候语空两格）
……（正文，口语化表达，有互动感）
我的发言完毕，谢谢大家！（结束语）
```

**倡议书**：
```
标题
尊敬的XX：（称呼）
……（倡议背景+理由）
为此，我们倡议：（过渡）
一、……
二、……
三、……（具体措施）
让我们……！（号召）
倡议人：XX
XXXX年XX月XX日
```

**辩论稿**：
```
标题
尊敬的主席，各位辩友，观众：（称呼，顶格写）
大家好！（问候语空两格写）
我今天是正（反）方代表，我方的观点是……（明确立场）
首先，……其次，……再次，……（分层论证）
综上所述，我方坚持认为……（重申观点）
我的发言到此结束，谢谢大家！（结束语）
```

辩论稿格式要求（必须标注）：
- 字数限制：一辩稿建议800-1200字
- 时间要求：立论陈词3-5分钟，攻辩环节每轮2-3分钟
- 辩手分工：一辩负责开篇立论，二辩负责攻辩，三辩负责总结陈词
- 称呼规范："尊敬的主席，各位辩友，观众"顶格写
- 立场声明：开篇必须明确"我方的观点是……"

**书信**：
```
标题（正标）——致XX的一封信（副标）
尊敬的XX：
您好！
……（正文）
此致
敬礼！
XX
XXXX年XX月XX日
```

## 语言提升技巧

### 技巧一：具象化表达
用场景描写替代空洞议论。

| 空洞表达 | 具象表达 |
|----------|----------|
| "奋斗是青春的底色" | "凌晨五点的教室，笔尖划过试卷的沙沙声，与朝阳爬上窗棂的光影，共同织就青春的奋斗图谱" |
| "他很努力" | "他的草稿纸叠起来比课本还高" |
| "传统文化很重要" | "敦煌的壁画是凝固的史诗，故宫的红墙是沉默的见证，而我们，是文明长河中掌灯的渡人" |

### 技巧一附：书面语与口语对照

诊断语言问题时，必须识别口语化表述并转化为书面语：

| 口语化表述（需修改） | 书面语表达（应改为） |
|------------------------|------------------------|
| "压力很大，工作很累，生活很难" | "重压之下，疲惫与困顿如影随形" |
| "我们要勇敢面对困难" | "困境面前，唯有迎难而上" |
| "遇到挫折的时候，要想办法解决" | "每逢挫折，当以智慧与韧性破局" |
| "只有这样，我们才能成功" | "唯有如此，方能在困局中开辟坦途" |

**口语化问题的识别标准**：
1. 使用"很+形容词"结构（很大、很累、很难）→ 应改为更精准的书面表达
2. 使用口语化连词（的时候、这样、那样）→ 应改为书面连词（之际、如此、彼时）
3. 句式过于短促直白，缺乏修饰 → 应增加修辞和节奏变化
4. 词汇过于日常（想办法、不要放弃）→ 应改为更凝练的书面表达（谋策、坚守）

### 技巧二：修辞化表达
- **比喻**：化抽象为具象（"技术是一把钥匙，能打开时间的门，但钥匙的主人是我们自己"）
- **排比**：增强气势（"可为的时代需要有为的青年，有为的青年成就可为的时代"）
- **呼告**：增强互动感（演讲稿中"同学们！"）

### 技巧三：名言化表达
引用贴切的名言、诗句、典故增强文化底蕴，注意：
- 贴切性：与观点一致
- 多样性：古今中外结合
- 适度性：不宜过多，2-3处为宜

### 技巧四：金句打造
发展等级加分项，每200字出现1个以上哲学化表达：

**金句公式**：
- **矛盾统一式**："不是A，而是B" → "不是看到希望才坚持，而是坚持了才看到希望"
- **条件转化式**："A的尽头是B" → "基础的尽头是自由的创造"
- **隐喻升华式**："A是B的C" → "苦难是民族精神的淬火炉"

## 素材运用策略

### 策略一：一材多用
同一素材从不同角度切入。

**示例——苏轼**：
- 乐观精神："乌台诗案后，他把贬谪的苦酒酿成'竹杖芒鞋轻胜马'的豁达"
- 突围智慧："从'兼济天下'到'独善其身'，他在人生转弯处寻得精神突围"
- 文化传承："他的词章是宋韵的活化石，千年后仍以'但愿人长久'的温情叩击现代心灵"

### 策略二：热点转化
将新闻事件转化为论述素材。

**示例——嫦娥探月**：
- 科技角度："从'嫦娥奔月'的神话到'嫦娥五号'的探月，让神话照进现实"
- 文化角度："科技与浪漫的共生，古老的飞天梦以现代方式续写"

### 策略三：素材新鲜度
**避免陈旧素材**：司马迁、爱迪生、牛顿等已被过度使用，将被降档。

**推荐新鲜素材方向**：
- 近三年时代人物：樊锦诗、黄文秀、张桂梅
- 科技前沿：AI、量子计算、深空探测
- 社会热点：乡村振兴、文化传承、生态文明

## 常见误区与规避

| 误区 | 表现 | 规避方法 |
|------|------|----------|
| 偏离材料 | 只写关键词不写关系 | 紧扣所有关键词，分析内在关系 |
| 立意肤浅 | 停留现象描述不深入 | 用本质挖掘法，从现象到本质 |
| 结构混乱 | 段落间无逻辑衔接 | 用过渡句连接，明确逻辑主线 |
| 语言空洞 | 全是口号无具体论据 | 用具体事例+数据支撑观点 |
| 文体失范 | 演讲稿写成议论文 | 严格遵守应用文体格式要求 |
| 素材陈旧 | 反复用司马迁、爱迪生 | 关注近三年新鲜素材 |
| 情感虚假 | 无病呻吟、感情泛滥 | 真情实感，适度表达 |
| 风格杂糅 | 不文不白、不土不洋 | 保持风格一贯性 |
