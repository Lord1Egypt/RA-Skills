# 小儿问诊要点

执行以下步骤以完成儿科中医问诊：

1. **询问出生及围产期情况**：包括产程、分娩方式、Apgar评分等。
2. **核实预防接种记录**：确认是否完整且按时接种。
3. **排查法定传染病史**：明确是否曾患麻疹、水痘等，以及是否有相关接触史。
4. **重点评估三大常见致病因素**：
   - 是否受寒（如衣着不当、气候变化）；
   - 喂养情况（如饮食不节、过食生冷、乳食不化）；
   - 是否受惊吓（如夜啼、惊惕、抽搐）。
5. **采集家族健康史**：必须询问父母及兄弟姐妹健康状况，特别关注遗传性疾病（如癫痫、先天性心脏病、代谢病等）。
6. **记录家族相似症状或慢性病**：若存在，需注明具体疾病名称及发病年龄。

完成上述步骤后，整合信息形成完整的儿科四诊资料，用于辨证分型。
