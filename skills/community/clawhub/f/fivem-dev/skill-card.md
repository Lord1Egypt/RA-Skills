## Known Risks and Mitigations: <br>
Risk: Generated server, database, or configuration changes may affect live FiveM environments if applied without review. <br>
Mitigation: Review proposed changes, test them in a staging environment, and back up existing resources and configuration before deployment. <br>
Risk: SSH private keys or SFTP credentials created while following the skill's guidance are sensitive credentials. <br>
Mitigation: Store keys securely, avoid overwriting existing keys, use least-privilege SFTP accounts, and rotate credentials if exposure is suspected. <br>
