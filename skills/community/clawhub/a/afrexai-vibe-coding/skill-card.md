## Known Risks and Mitigations: <br>
Risk: Copied rules-file templates may accidentally include secrets, private data, or conventions that do not fit the target project. <br>
Mitigation: Customize templates per project, omit secrets and private data, and review configuration before committing or sharing it. <br>
Risk: Example git recovery commands can discard local changes if copied without understanding their effect. <br>
Mitigation: Review each command before execution and preserve local work before using checkout or other recovery commands. <br>
Risk: AI-generated code produced through this methodology can still contain security, correctness, or maintainability defects. <br>
Mitigation: Use the included review, testing, and production-readiness checklists before deploying generated code. <br>
