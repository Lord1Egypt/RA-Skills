# 页面生成与后期添加分层标准 v2

## 用户最新确认

PPT 生产不是“生图只做纯背景”。正确流程是：

1. 生图 / 导演台阶段可以直接生成一张接近成品的 PPT 页面主体图；
2. main / PPT 组装阶段在交付给用户前，统一添加固定品牌元素和附加元素；
3. 最终交付给用户的是后期加料后的完整 PPT 页面 / PPT 文件。

## 生图阶段负责什么

生图阶段可以负责：

- 页面主体视觉；
- 页面整体版式；
- 页面内容的视觉表达；
- 标题区、内容区、卡片区、流程区的整体设计；
- 与内容有关的图形、模块、结构、流程、符号化表达；
- 黑金风格、高级感、页面空间关系。

也就是说：可以把页面内容、页面标题、支撑点、流程结构、版式要求告诉导演台，让它按 PPT 页面方向直接生成页面主体。

## 生图阶段仍然禁止什么

即使生图直接做 PPT 页面，也禁止生成：

- 逐梦学院 Logo；
- 页脚；
- 页码；
- 下标 / 页标；
- 二维码；
- 需要真实准确的品牌 Logo；
- 需要真实准确的产品截图；
- 需要真实准确的联系方式；
- 任何不可控的错误品牌元素。

这些必须由后期统一添加。

## 关于文字

生图阶段可以按页面内容做版式，但如果文字准确性不可控，则必须：

- 要么生成无文字版主体图，后期加真实文字；
- 要么生成带版式占位的页面图，再由后期覆盖/替换关键文字；
- 正式交付前必须核对所有文字，不允许错字、乱码、伪中文。

## 后期添加负责什么

PPT 组装层 / main 最后必须添加或校正：

- 逐梦学院 Logo；
- 页脚；
- 页码；
- 下标 / 页标；
- Part 标签；
- 二维码；
- 固定品牌元素；
- 需要绝对准确的标题、正文、流程文字；
- 真实素材、真实截图、真实二维码。

## 最终交付前门禁

给用户前必须检查：

- 生图页面主体是否符合内容；
- 文字是否准确；
- 是否存在伪中文/错字/乱码；
- Logo、页脚、页码、下标是否已后期添加；
- 页面是否像完整 PPT，而不是单纯背景图；
- 是否存在品牌元素重复或错位。
