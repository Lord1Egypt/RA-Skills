## About

This project systematically integrates core domains of human anatomy, neuroanatomy, clinical applied anatomy, and integrative Chinese-Western anatomical analysis, covering **112 key anatomy skills** organized into **8 categories**.

**Target Audience**: Medical students, clinicians, anatomy researchers, surgical residents

**Reference Textbook**: *Systematic Anatomy* (People's Medical Publishing House, 10th Edition)

**⚠️ Disclaimer ⚠️**: This skill set covers anatomical structure descriptions, anatomical variation analysis, surgical approach evaluation, and clinical localization diagnosis. It is not a substitute for professional anatomy education or clinical training.

### Usage

Each skill contains four sections:
1. **When to Use** — applicable clinical or teaching scenarios
2. **Steps** — standardized anatomical analysis procedure
3. **Cautions** — anatomical variations and clinical warnings
4. **References** — supplementary documentation

### Query Examples

#### 1. Concept Query
```bash
Internal capsule: location and divisions
```

#### 2. Clinical Case Analysis
```bash
A 65-year-old male presents with sudden right-sided weakness and slurred speech for 3 hours. Exam: right upper and lower limb strength 0/5, right central facial palsy, decreased pain/temperature sensation on the right, eyes deviated to the left. Analyze the likely lesion location.
```

#### 3. Exam Question
```bash
Regarding the internal capsule, which statement is INCORRECT?
A. Located between the caudate nucleus, lentiform nucleus, and thalamus
B. The genu contains the corticonuclear tract
C. The posterior limb contains the corticospinal tract
D. Internal capsule lesions do NOT cause sensory deficits
```
