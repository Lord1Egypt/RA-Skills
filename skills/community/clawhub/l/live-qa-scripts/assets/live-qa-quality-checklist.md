# Live Q&A Script Quality Checklist

A 40-point gate. Scripts are ready only when every Critical item is checked.

## 1. Coverage (Critical)
- [ ] Top 10 objections addressed
- [ ] Price objection has a strong answer
- [ ] Authenticity objection covered
- [ ] Sizing/fit covered (if relevant)
- [ ] Shipping + returns covered

## 2. Answer Quality (Critical)
- [ ] Each answer is 1–2 spoken sentences
- [ ] Each answer acknowledges the concern
- [ ] Each answer includes specific proof
- [ ] No vague "trust me" reassurance
- [ ] Sayable out loud naturally

## 3. CTA (Critical)
- [ ] Every answer ends with a CTA
- [ ] CTA points to cart/code/timer
- [ ] Price + code restated periodically
- [ ] Scarcity used honestly

## 4. Variants
- [ ] Short variant for each question
- [ ] Long variant for each question
- [ ] Rapid-fire cue card included
- [ ] Demo cues noted where useful

## 5. Organization
- [ ] Grouped by theme (Product/Price/Trust/Logistics)
- [ ] Findable in seconds mid-stream
- [ ] Skimmable cue-card layout
- [ ] Header with product, promo, policies

## 6. Tone & Brand
- [ ] On-brand voice
- [ ] Warm and confident, not defensive
- [ ] Not robotic or over-scripted
- [ ] Consistent across hosts

## 7. Compliance (Critical)
- [ ] No banned/medical claims
- [ ] Unsubstantiated claims softened/removed
- [ ] Approved guarantee wording used
- [ ] Safe line for sensitive questions

## 8. Multi-Host & Ops
- [ ] Handoff cues included
- [ ] Comment-pinning checklist
- [ ] Reset line for chat floods
- [ ] Pre-stream checklist attached

## 9. Maintenance (Critical)
- [ ] Reflects current price and promo
- [ ] Reflects current shipping/returns
- [ ] Reviewed before each major stream
- [ ] Owner assigned to keep it updated
