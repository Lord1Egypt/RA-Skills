# 展会跟进策略

## 一、展会前准备

### 目标设定
| 目标类型 | 具体指标 |
|---------|---------|
| 名片收集 | 目标: 200+ 张/天 |
| 高质量线索 | 目标: 50+ 个 |
| 意向客户 | 目标: 20+ 个 |
| 即时成交 | 目标: 3-5 个 |

### 物料准备清单
- [ ] 公司名片（中英文）
- [ ] 产品目录（纸质+电子）
- [ ] 价格表（可现场提供）
- [ ] 样品（轻便易带）
- [ ] 名片收集盒
- [ ] iPad/平板（展示视频）
- [ ] 笔记本和笔

## 二、展会中技巧

### 快速识别高价值客户
- [ ] 采购决策人（Director/VP级别）
- [ ] 有明确采购需求
- [ ] 采购量级符合目标
- [ ] 有合理采购周期


## 三、展会后跟进

### 黄金跟进时间线
| 时间 | 动作 |
|-----|------|
| 展会期间 | 即时WhatsApp/微信联系 |
| 展会结束当天 | 感谢邮件 |
| Day 3 | 个性化跟进邮件 |
| Day 7 | 详细报价+资料 |
| Day 14 | 询问进展 |
| Day 30 | 再次跟进 |

### 邮件策略
1. **个性化第一封**：提及展会见面的具体细节
2. **价值导向**：不要重复目录内容
3. **行动号召**：明确下一步

## 四、客户分类跟进

### A类客户（高优先级）
- 立即发送详细报价
- 48小时内安排样品
- 主动电话跟进

### B类客户（次优先级）
- 发送标准报价
- 常规邮件跟进
- 邀请参观工厂

### C类客户（观察名单）
- 发送电子资料
- 保持品牌印象
- 资源允许时跟进
