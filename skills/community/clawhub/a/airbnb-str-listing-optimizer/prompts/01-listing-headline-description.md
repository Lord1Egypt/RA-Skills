# Prompt 1: Listing Headline + Description (FREE)

**What it produces:** A high-converting Airbnb/VRBO title (under 50 characters) + a 400–500 word listing description optimized for platform search ranking and guest conversion.

---

## The Prompt

```
You are an expert short-term rental copywriter who has optimized 500+ Airbnb and VRBO listings. Your copy consistently increases booking rates by 30–60% because you understand both platform SEO and guest psychology.

Write a complete listing headline and description for this property:

**Property type:** [condo / house / cabin / villa / townhome / suite / other]
**Bedrooms / Bathrooms / Max guests:** [e.g., 2BR / 2BA / 4 guests]
**Location:** [City, neighborhood, and distance to 2–3 key landmarks or attractions]
**Top 5 amenities:** [e.g., heated pool, resort-style gym, fast WiFi 500 Mbps, dedicated workspace, EV charger]
**Unique selling point:** [e.g., Strip views from the balcony / mountain retreat 10 min from skiing / steps to the French Quarter]
**Target guest:** [e.g., couples on a romantic getaway / business travelers / family reunion groups / bachelorette parties]
**Vibe / design style:** [e.g., modern luxury / cozy mountain cabin / beachy boho / sleek minimalist]

---

OUTPUT FORMAT:

**TITLE (under 50 characters):**
[Headline — include 1 location reference + 1 standout feature]

**DESCRIPTION (400–500 words):**

[Opening hook — 2–3 sentences that paint the guest experience, NOT a list of features. Make them feel they're already there.]

[Paragraph 2 — The space. Cover layout, sleeping arrangements, standout design elements. Use sensory language.]

[Paragraph 3 — Top amenities that matter to this guest type. Lead with the 2–3 that differentiate this property from comparables in the area.]

[Paragraph 4 — The location story. What's walkable? What's a 5-minute drive? What local experience can only guests staying HERE have?]

[Closing — 2–3 sentences with a clear call to action and one line about the host's commitment to a great stay.]

---

RULES:
- Naturally include these Airbnb search keywords (do NOT stuff): [pool / Strip views / walkable / workspace / entire home / king bed / fast wifi / self check-in / free parking — use whichever apply]
- Write in second person ("you'll love", "your morning coffee")
- Zero corporate-speak. No "immaculate" or "charming" — show, don't tell.
- Title: avoid ALL CAPS. Use sentence case or title case.
- Description: no bullet lists. Flowing paragraphs only.
```

---

## Usage Notes

- Run this first. The title and description are the #1 ranking and conversion factors.
- For Airbnb: the title must stay under 50 characters (including spaces). Claude will comply if you include the property details accurately.
- For VRBO: you get up to 80 characters — ask Claude to produce a VRBO variant after running the Airbnb version.
- Refresh this every 90 days or when you add a major amenity.
