帮助用户构建具有强穿透力的极致品牌内核；当用户需要创建新品牌、升级现有品牌或设计品牌战略时使用。 <br>

## Use Case: <br>
External brand builders, founders, consultants, and marketers use this skill to define or upgrade a brand core, assess penetration strength, compare competitor positioning, and generate structured brand strategy guidance. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill uses strong persuasion concepts such as dependency, emotional connection, and decision-path control, which could lead to manipulative or unsupported brand claims. <br>
Mitigation: Review generated strategy before public use, validate claims against evidence, and remove manipulative framing from customer-facing materials. <br>
Risk: Server security guidance notes that at least one included script may need a bug fix for reliable CLI use. <br>
Mitigation: Run scripts on noncritical inputs first, inspect JSON output for errors, and fix CLI issues before relying on the reports. <br>
