# Cron表达式示例

## 常见cron表达式示例

### 基本示例

| Cron表达式 | 描述 |
|-----------|------|
| `* * * * *` | 每分钟执行 |
| `0 * * * *` | 每小时整点执行 |
| `0 0 * * *` | 每天午夜执行 |
| `0 0 * * 0` | 每周日午夜执行 |
| `0 0 1 * *` | 每月1号午夜执行 |
| `0 0 1 1 *` | 每年1月1号午夜执行 |

### 工作日示例

| Cron表达式 | 描述 |
|-----------|------|
| `0 9 * * 1-5` | 工作日（周一到周五）上午9点执行 |
| `0 18 * * 1-5` | 工作日（周一到周五）下午6点执行 |
| `*/30 9-17 * * 1-5` | 工作日每30分钟执行，从9点到17点 |
| `0 9-17 * * 1-5` | 工作日每小时整点执行，从9点到17点 |

### 间隔执行示例

| Cron表达式 | 描述 |
|-----------|------|
| `*/5 * * * *` | 每5分钟执行 |
| `0 */2 * * *` | 每2小时整点执行 |
| `0 */6 * * *` | 每6小时整点执行 |
| `0 0 */3 * *` | 每3天午夜执行 |
| `0 0 * */2 *` | 每2个月午夜执行 |

### 特定时间示例

| Cron表达式 | 描述 |
|-----------|------|
| `0 8 * * *` | 每天上午8点执行 |
| `0 12 * * *` | 每天中午12点执行 |
| `0 18 * * *` | 每天下午6点执行 |
| `0 22 * * *` | 每天晚上10点执行 |
| `30 14 * * *` | 每天下午2点30分执行 |
| `0 9,12,18 * * *` | 每天上午9点、中午12点、下午6点执行 |

### 复杂示例

| Cron表达式 | 描述 |
|-----------|------|
| `0 0 1,15 * *` | 每月1号和15号午夜执行 |
| `0 0 * * 0,6` | 每周六和周日午夜执行 |
| `0 8-18 * * 1-5` | 工作日每小时整点执行，从8点到18点 |
| `*/15 9-17 * * 1-5` | 工作日每15分钟执行，从9点到17点 |
| `0 0 1 1,4,7,10 *` | 每季度第一天（1月、4月、7月、10月1号）午夜执行 |

## 特殊字符说明

### 星号 (*)
- 表示"任何值"
- 示例：`*` 在分钟字段表示"每分钟"

### 逗号 (,)
- 分隔多个值
- 示例：`1,3,5` 表示"1、3和5"

### 连字符 (-)
- 表示范围
- 示例：`1-5` 表示"从1到5"

### 斜杠 (/)
- 表示步长
- 示例：`*/5` 表示"每5个单位"
- 示例：`0-30/5` 表示"从0到30，每5个单位"

### 问号 (?)
- 仅在日和星期字段使用
- 表示"无特定值"
- 用于避免日和星期字段的冲突

## 字段取值范围

| 字段 | 允许值 | 特殊值 |
|------|--------|--------|
| 分钟 | 0-59 | - |
| 小时 | 0-23 | - |
| 日 | 1-31 | ? (无特定值) |
| 月 | 1-12 或 JAN-DEC | - |
| 星期 | 0-6 或 SUN-SAT | 0和7都表示周日，? (无特定值) |

## 月份和星期名称

### 月份名称
- JAN, FEB, MAR, APR, MAY, JUN
- JUL, AUG, SEP, OCT, NOV, DEC

### 星期名称
- SUN, MON, TUE, WED, THU, FRI, SAT

注意：名称不区分大小写。

## 实用技巧

### 1. 避免冲突
当同时指定日和星期时，cron通常只使用其中一个。使用`?`可以避免冲突：
- `0 0 ? * 1` - 每周一午夜执行（日字段无特定值）
- `0 0 1 * ?` - 每月1号午夜执行（星期字段无特定值）

### 2. 最后一天
表示每月最后一天：
- `0 0 L * *` - 每月最后一天午夜执行（某些cron实现支持）

### 3. 工作日
表示工作日（周一到周五）：
- `0 9 * * 1-5` - 工作日上午9点执行

### 4. 周末
表示周末（周六和周日）：
- `0 9 * * 6,0` - 周末上午9点执行
