## Description: <br>
Use CamScanner to detect whether an image was generated by an AI model, returning an AI verdict, confidence score, and human-readable result text. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[camscanner-ai](https://clawhub.ai/user/camscanner-ai) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and agents use this skill to check whether a provided image appears to be real, suspected AI-generated, or AI-generated. The skill is suited for image authenticity questions where the user wants a concise verdict with confidence. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill uploads user-provided images to CamScanner servers for analysis. <br>
Mitigation: Use it only for images the user chooses to submit, and avoid sensitive or private images unless the user accepts CamScanner's privacy and retention claims. <br>
Risk: AI-generated-image detection can return a suspected result rather than a definitive result. <br>
Mitigation: Report the confidence score and preserve the distinction between suspected AI-generated and AI-generated. <br>
