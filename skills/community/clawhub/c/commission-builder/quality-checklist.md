# Commission Builder Quality Checklist

---

## Program Design

- [ ] Unit economics calculated (AOV, gross margin, max commission budget)
- [ ] Commission rate verified to preserve ≥15% net margin after all costs
- [ ] Three-tier structure defined with clear performance thresholds
- [ ] Attribution window defined (days after click)
- [ ] Attribution model defined (last-click, multi-touch, etc.)
- [ ] Commissionable customers defined (all vs. new-only)
- [ ] Coupon code attribution policy written
- [ ] Returns/refunds treatment defined (commissions clawed back or kept)

## Partner Quality

- [ ] Partner profile defined (platform, follower range, engagement rate, category)
- [ ] Manual review process in place (not fully automated acceptance)
- [ ] Coupon/deal site policy defined (included, excluded, or lower-tier)
- [ ] Activity minimum defined (e.g., 1 qualifying action per 90 days)

## Legal and Compliance

- [ ] Affiliate agreement includes: attribution rules, exclusivity terms, prohibited claims, payment terms
- [ ] FTC disclosure requirement communicated to all partners (#ad, #sponsored)
- [ ] Prohibited claims list shared during onboarding
- [ ] GDPR/CCPA data handling for affiliate personal data addressed

## Tracking and Attribution

- [ ] Affiliate tracking platform selected and set up
- [ ] All tracking links tested and confirmed working
- [ ] Coupon codes set up and confirmed working
- [ ] Attribution is consistent with platform settings

## Onboarding Experience

- [ ] Welcome email template ready
- [ ] Product samples or purchase credits available for new partners
- [ ] Brand asset folder created and accessible
- [ ] Approved claims / messaging guide shared
- [ ] 30-day check-in scheduled for all new partners

## Ongoing Management

- [ ] Monthly performance review scheduled
- [ ] Tier graduation criteria are automated or reviewed monthly
- [ ] Inactive affiliate notification at 90 days is set up
- [ ] Payment processing confirmed for Net-30 cadence
- [ ] Partner communication cadence (monthly update) scheduled
