## Description: <br>
Helps agents onboard a repository into Claude Code by creating or improving CLAUDE.md and identifying useful follow-on setup. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wimi321](https://clawhub.ai/user/wimi321) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill to analyze a repository and create practical CLAUDE.md onboarding guidance with architecture, commands, conventions, and useful follow-on setup. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated CLAUDE.md content could include secrets, private URLs, or proprietary repository details. <br>
Mitigation: Review the generated file before committing or sharing it, and remove sensitive information. <br>
Risk: Repository guidance may be inaccurate if the project structure, commands, or conventions are incomplete or ambiguous. <br>
Mitigation: Check generated commands and conventions against the repository before relying on them. <br>
