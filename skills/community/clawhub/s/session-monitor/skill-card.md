## Description: <br>
Automatically displays concise session status information, including model, token usage, context usage, and configurable feature state. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[yangdaowan](https://clawhub.ai/user/yangdaowan) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agent users use this skill to append compact model, token, and context status to responses and to control that display through simple commands or configuration. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Session status appended to responses can expose model, token, or context details when responses are shared externally. <br>
Mitigation: Turn the display off or use hidden/no-display mode before sharing responses or when strict output formatting is required. <br>
Risk: The skill saves local display preferences in the configured workspace. <br>
Mitigation: Use it in a workspace where local preference files are acceptable and review workspace permissions before deployment. <br>
