## 🆕 核心功能

### 1️⃣ 格式互转（LibreOffice引擎）
| 源格式 | 目标格式 |
|--------|---------|
| Word (.docx) | PDF, TXT, MD, HTML |
| PDF | TXT, MD, DOCX |
| Markdown (.md) | DOCX, PDF, HTML |
| 纯文本 (.txt) | DOCX, PDF, MD |
| HTML | DOCX, PDF, TXT |

### 2️⃣ PPT智能生成
| 模板风格 | 适用场景 |
|---------|---------|
| 🏢 商务风 | 工作汇报、方案演示 |
| 🎓 学院风 | 学术报告、作业展示 |
| 🏛 会议风 | 内部讨论、项目评审 |
| ✨ 简约风 | 通用简洁、快速展示 |

**输入方式：**
- 上传文档 → 提取内容 → 生成PPT
- 直接粘贴文本 → 生成PPT
- 文档+指定风格 → 生成PPT

### 3️⃣ 文档摘要（保留原版功能）
- URL/粘贴文本 → 全文摘要
- 多文档对比摘要
- 合同关键条款提取

## 👤 为什么做这个

之前v2.0做文档摘要，有用户反馈说：能不能直接帮我改格式？能不能直接生成PPT？这些是文档处理最刚需的功能。v3.0一次性解决。

## 💡 使用小贴士
- PDF转Word：复杂排版可能有偏差，建议人工核对
- PPT生成：内容越详细，PPT质量越高
- 格式互转：支持批量文件处理

## ⚠️ 免责与已知限制
- PDF扫描件无法转换（需要OCR）
- PPT模板效果基于标准样式，复杂排版需微调
- 文件大小限制：单个文件≤20MB
