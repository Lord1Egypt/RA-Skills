# /sector — 行业报告

对指定行业（申万分类）执行全景分析，输出行业规模、格局、趋势和竞争分析报告。

## 执行步骤

1. 通过 MCP 获取该行业全部成分股列表
2. 计算行业全景指标：营收总规模、增速中位数、利润率中位数、集中度（CR5/CR10）
3. 判断行业生命周期（导入/成长/成熟/衰退）
4. 产业链分析：上游/中游/下游利润分配、议价能力分布
5. 竞争格局：波特五力 + 龙头对比 + 市场份额分布
6. 生成行业对标基准值（75分位/中位/25分位）

## 输入格式

- 行业名称：如 "白酒"、"新能源汽车"、"半导体"
- 申万行业代码：如 "SW801010"

## 输出要求

- 行业基本画像（规模/增速/生命周期）
- 竞争格局分析
- 产业链利润分布
- 行业对标基准表（供公司分析参考）
- 行业趋势判断与风险提示
