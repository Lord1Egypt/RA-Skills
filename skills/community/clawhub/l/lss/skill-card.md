## Description: <br>
Applies the Learn Social Studies brand system to React interfaces using CSS variables, reusable component conventions, light theme by default, and dark theme as optional override. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[lssmanager](https://clawhub.ai/user/lssmanager) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to restyle or standardize React apps, dashboards, admin panels, forms, landing pages, and educational product interfaces with the Learn Social Studies visual system while preserving product logic. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The artifact license file says complete terms are still to be defined even though server evidence lists MIT-0. <br>
Mitigation: Confirm the final license terms with the project owner before installing or redistributing the skill. <br>
Risk: The recommended Google Fonts snippet loads resources from Google-hosted domains. <br>
Mitigation: Review whether third-party font loading is allowed in the target environment; self-host or remove the font links if required. <br>
Risk: CSS custom properties and component class names may collide with an existing application design system. <br>
Mitigation: Review token and class names before integration, then namespace or map them to existing app conventions where needed. <br>
