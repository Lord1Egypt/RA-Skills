## 发育预警（需要就医信号）

| 能区 | 警示信号 |
|------|---------|
| 大运动 | 4个月抬头不稳 / 8个月不会翻身 / 18个月不会走 |
| 语言 | 12个月无咿呀 / 18个月无词汇 / 24个月词汇<50 |
| 社交 | 6个月不认亲人 / 12个月不指物 / 18个月不模仿 |

---

## 早产儿校正规则

**公式：** 实际月龄 - (40周 - 出生周龄) / 4

**举例：** 32周早产，現已6个月
校正 = 6个月 - (40-32)/4 = 6-2 = 4个月（按4个月发育标准评估）
