# GB/T 9704-2012 Formatting Rules (格式规范)

## Overview

GB/T 9704-2012 "党政机关公文格式" specifies the format requirements for official documents of party and government organs.

## Page Setup (页面设置)

### Paper Size

```
规格：A4
尺寸：210mm × 297mm
方向：纵向（一般情况）
```

### Margins

```
上白边：37mm ± 1mm
下白边：35mm ± 1mm
左白边：28mm ± 1mm
右白边：26mm ± 1mm

版心尺寸：156mm × 225mm
```

### Line Spacing

```
行距：固定值 28磅（约0.99cm）
每页：22行
每行：28字
```

## Document Elements (公文要素)

### 版头 (Header)

#### 1. 份号

```
位置：版心左上角第一行
字体：黑体，3号
格式：如 "000001" 或 "秘密000001"
说明：涉密公文应标注份号
```

#### 2. 密级和保密期限

```
位置：版心左上角第二行
字体：黑体，3号
格式：如 "机密★5年" 或 "绝密"
说明：涉密公文必须标注
```

#### 3. 紧急程度

```
位置：版心左上角
字体：黑体，3号
格式：特急 或 加急
说明：紧急公文应标注
```

#### 4. 发文机关标志

```
位置：居中
字体：小标宋体，红色
格式：
  - 全称：如 "XX省人民政府文件"
  - 简称：如 "XX省府文件"
说明：联合行文时，各机关标志并列排列
```

#### 5. 发文字号

```
位置：发文机关标志下空二行，居中
字体：仿宋，3号
格式：机关代字 + 六角括号〔年份〕+ 序号 + 号
示例：国发〔2024〕1号

正确格式：
- XX发〔2024〕1号 ✓
- XX发[2024]1号 ✗
- XX发（2024）1号 ✗
```

#### 6. 签发人

```
位置：发文字号右侧
字体：仿宋，3号
格式：签发人：姓名
说明：上行文必须标注签发人姓名
```

### 主体 (Body)

#### 1. 标题

```
位置：分隔线下空二行，居中
字体：小标宋体，2号
格式：发文机关 + 事由 + 文种
示例：国务院关于XX的通知

要求：
- 一般不加标点符号
- 可分多行排列
- 排列应整齐美观
```

#### 2. 主送机关

```
位置：标题下空一行，顶格
字体：仿宋，3号
格式：机关全称、规范化简称或同类型机关统称
示例：各省、自治区、直辖市人民政府

说明：
- 多个主送机关用顿号或逗号分隔
- 最后一个主送机关名称后标全角冒号
- 某些公文可不标注主送机关
```

#### 3. 正文

```
位置：主送机关名称下一行
字体：
  - 正文：仿宋，3号
  - 第一层：黑体，3号
  - 第二层：楷体，3号
  - 第三、四层：仿宋，3号
行距：28磅

层次序号：
第一层：一、二、三、……
第二层：（一）（二）（三）……
第三层：1. 2. 3. ……
第四层：（1）（2）（3）……
```

#### 4. 附件说明

```
位置：正文下空一行，左空二字
字体：仿宋，3号
格式：
  附件：1. XXXXX
       2. XXXXX

说明：
- 附件较多时应标注序号
- 附件名称不加书名号
- 附件名称后不加标点
```

#### 5. 发文机关署名

```
位置：正文右下方
字体：仿宋，3号
格式：机关全称或规范化简称
说明：署名应与发文机关标志一致
```

#### 6. 成文日期

```
位置：署名下方
字体：仿宋，3号
格式：XXXX年X月X日（阿拉伯数字）
说明：
- 用阿拉伯数字
- 不加"年""月""日"之外的标点
- 右空四个汉字
```

#### 7. 印章

```
位置：骑年盖月（印章下沿压在成文日期上）
要求：
- 上沿不压正文
- 下沿压成文日期
- 印章端正
```

### 版记 (Footer)

#### 1. 抄送机关

```
位置：版记第一行
字体：仿宋，4号
格式：抄送：XX、XX
```

#### 2. 印发机关和印发日期

```
位置：版记最后
字体：仿宋，4号
格式：XX印发    XXXX年X月X日印发
```

#### 3. 页码

```
位置：版心下边缘下方
字体：宋体，4号，半角
格式：— 1 —
对齐：单页右空一字，双页左空一字
```

## Special Formats (特殊格式)

### 信函格式

```
特点：
- 发文机关名称居中
- 发文字号右上角
- 无版头
- 无抄送机关
- 页码在正文下方

适用：
- 函
- 部分批复
```

### 命令格式

```
特点：
- 发文机关标志用全称或规范化简称
- 无主送机关
- 签发人职务和姓名
- 标题可省略

适用：
- 命令
- 令
```

### 纪要格式

```
特点：
- 无主送机关
- 标注出席人员、列席人员
- 正文记录会议情况和议定事项
- 无发文机关署名和印章

适用：
- 会议纪要
```

## Common Errors

### 格式错误

| 错误 | 正确 |
|-----|------|
| 国发[2024]1号 | 国发〔2024〕1号 |
| 二〇二四年三月十五日 | 2024年3月15日 |
| I、II、III | 一、二、三 |
| 1、2、3 | 1. 2. 3. |
| （一） | （一） |

### 排版错误

- 行距不一致
- 字体字号不规范
- 标题不居中
- 日期格式错误
- 页码格式错误

## Checklist

```
□ 发文机关标志正确
□ 发文字号格式正确
□ 标题完整、居中
□ 主送机关完整
□ 正文层次序号正确
□ 字体字号规范
□ 发文机关署名正确
□ 成文日期格式正确
□ 页码格式正确
□ 整体排版美观
```
