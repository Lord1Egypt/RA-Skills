# 选品分析框架参考

## 趋势分析框架

### Google Trends分析

| 指标 | 含义 | 选品参考 |
|-----|------|---------|
| 搜索量趋势 | 同比变化 | 增长型/稳定型/衰退型 |
| 地区热度 | 哪些地区搜索多 | 目标市场选择 |
| 相关查询 | 热门长尾词 | 产品差异化方向 |
| 季节性 | 周期性变化 | 备货时机 |

## 竞争分析方法

### 市场集中度

| CR值 | 市场状态 | 策略建议 |
|-----|---------|---------|
| CR3 < 30% | 竞争分散 | 可进入 |
| CR3 30-50% | 中度集中 | 差异化 |
| CR3 > 50% | 高度垄断 | 谨慎进入 |

### 竞争壁垒评估

| 壁垒类型 | 表现 | 威胁程度 |
|---------|------|---------|
| 品牌壁垒 | 知名品牌主导 | 高 |
| 评论壁垒 | 头部评论1000+ | 高 |
| 价格壁垒 | 头部价格极低 | 中 |
| 技术壁垒 | 专利产品 | 中 |
| 认证壁垒 | 特殊认证需求 | 中 |

### 盈亏平衡分析

```
盈亏平衡销量 = 固定成本 / (单价 - 变动成本)

ROI = 利润 / 投入成本 × 100%

回本周期 = 初始投入 / 月利润
```

## 选品检查清单

### 基础核查
- [ ] 市场需求量如何（月销多少）
- [ ] 竞争激烈程度（头部销量多少）
- [ ] 产品差异化可能性
- [ ] 利润空间是否满足要求

### 合规核查
- [ ] 目标市场认证要求
- [ ] 是否有专利风险
- [ ] 是否有品牌授权
- [ ] 进口限制和禁运

### 供应核查
- [ ] 能否找到稳定供应商
- [ ] MOQ是否可接受
- [ ] 交期是否可控
- [ ] 样品质量如何

### 运营核查
- [ ] 是否适合FBA配送
- [ ] 产品是否易损
- [ ] 包装要求
- [ ] 退货率预估
