# Live Q&A Library — Output Template

Organize answers by theme so a host can find one mid-stream in seconds. Each entry has a short and long variant and ends with a CTA.

## Header

- **Product:** {name} — {price}
- **Promo / code:** {launch code, discount, timer}
- **Guarantee / returns:** {policy}
- **Shipping:** {terms}
- **Brand voice:** {3 adjectives}
- **Banned claims:** {list}

## Theme: Product & Quality

**Q: {question}**
- *Short:* {1 sentence + proof + CTA}
- *Long:* {2 sentences + demo cue + CTA}

(repeat for each product question)

## Theme: Price & Promo

**Q: "It's too expensive"**
- *Short:* {reframe value + code + CTA}
- *Long:* {per-use math + guarantee + CTA}

**Q: "Will it go on sale later?"**
- *Short:* {scarcity + timer + CTA}

## Theme: Trust & Authenticity

**Q: "Is this legit / authentic?"**
- *Short:* {proof of authenticity + CTA}

**Q: "Does it really work?"**
- *Short:* {demo cue + result + CTA}

## Theme: Logistics (Shipping & Returns)

**Q: "When will it arrive?"**
- *Short:* {shipping window + CTA}

**Q: "What if it doesn't fit / I don't like it?"**
- *Short:* {returns policy + CTA}
