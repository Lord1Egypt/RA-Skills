# AI Interview Trainer

Job interview preparation covering behavioral questions, technical assessments, case interviews, and salary negotiation for all career stages and industries.

## Core Capabilities

### 1. Behavioral Interviews
- STAR method: Situation, Task, Action, Result framework
- Common questions: leadership, conflict, failure, teamwork, innovation
- Competency mapping: aligning stories to job requirements
- Story bank building: 10-15 versatile anecdotes
- Delivery practice: pacing, conciseness, confidence markers

### 2. Technical Interviews
- Coding interviews: data structures, algorithms, system design
- Whiteboard problem-solving framework: clarify, plan, pseudocode, implement
- Take-home assignments: scope management, documentation
- Portfolio review preparation: project selection, narrative
- Domain-specific: product, design, data science, engineering

### 3. Case Interviews
- Framework application: profitability, market entry, M&A, pricing
- Structured communication: hypothesis-driven approach
- Quantitative analysis under pressure
- Fit questions integration with case performance

### 4. First Impressions
- Elevator pitch crafting: 30-60 second versions
- Researcher's mindset: company, role, interviewer background
- Questions to ask: team, culture, challenges, success metrics
- Virtual interview: lighting, background, eye contact, audio

### 5. Negotiation
- Compensation research: salary bands, equity, bonus structure
- Multiple offer leverage, timing strategies
- Total compensation evaluation: base, equity, benefits, perks
- Counter-offer etiquette, acceptance/decline communication
