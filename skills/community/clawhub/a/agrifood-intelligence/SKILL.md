# AgriFood Intelligence

## 核心能力

| 能力模块 | 功能描述 |
|---------|---------|
| 病虫害识别 | 基于作物照片识别常见病虫害（覆盖50+作物、300+病害），输出防治方案与推荐药剂 |
| 土壤分析 | 解读土壤检测报告（pH/NPK/有机质/微量元素），输出改良方案与施肥建议 |
| 市场价格 | 追踪全国主要农产品批发市场价格走势，支持同比/环比/季节性分析 |
| 食品安全 | 农药残留MRL合规检查（覆盖中国GB 2763/EU/Codex标准），输出风险等级 |
| 种植方案 | 基于气候/土壤/市场需求的多作物轮作方案优化，含投入产出比估算 |

## 触发场景

- 作物出现异常症状，需要远程诊断
- 土壤检测报告解读与改良方案
- 农产品定价与市场时机判断
- 出口农产品农药残留合规审查
- 种植结构调整与轮作规划

## 执行流程

1. **问题采集**：获取作物照片/症状描述/土壤数据/市场关注区域
2. **智能诊断**：病虫害图像识别 + 土壤参数解读 + 市场数据抓取
3. **方案生成**：综合输出防治方案/施肥计划/市场分析报告
4. **合规审查**：对目标市场进行农药残留合规比对
5. **经济分析**：计算各方案的成本收益，辅助决策

## 输出格式

- 病虫害诊断报告（病害名+置信度+防治方案+推荐药剂）
- 土壤改良方案（现状分析+改良措施+施肥量计算）
- 市场价格分析（趋势图数据+价格预测+交易建议）
- 食品安全合规报告（风险项+超标检测项+贸易影响）
- 种植规划方案（作物配置+轮作时间线+经济收益预估）

## 注意事项

- 病虫害识别基于视觉特征，部分病害需要实验室确诊
- 市场价格预测基于历史趋势，极端天气/政策变化可能造成偏差
- 农药使用建议需遵循当地法律法规和标签说明
- 食品安全MRL标准定期更新，建议每次使用前确认版本
- 本技能适用于中国及东南亚主要作物区
