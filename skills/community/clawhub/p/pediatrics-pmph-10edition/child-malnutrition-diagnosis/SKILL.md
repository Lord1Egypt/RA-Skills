# 儿童营养不良诊断指标应用

## 适用场景
- 临床筛查7岁以下儿童营养状况
- 公共卫生营养调查
- 生长发育评估中识别营养不良

## 执行步骤
1. **准备参照数据**：获取同年龄、同性别儿童的身高体重标准（均值 X̄ 和标准差 S）。
2. **测量并计算三项指标**：
   - **体重低下**：实际体重 < X̄−2S（反映急/慢性营养不良）
   - **生长迟缓**：身高 < X̄−2S（反映长期慢性营养不良）
   - **消瘦**：体重 < 同身高参照人群 X̄−2S（反映近期急性营养不良）
3. **判断分度**：
   - 中度：(X̄−3S) ~ (X̄−2S)
   - 重度：< X̄−3S
4. **综合诊断原则**：
   - 符合任一指标即可诊断营养不良
   - 多项阳性提示混合型或长期营养问题
   - 若存在水肿（如Kwashiorkor），优先依赖消瘦和生长迟缓指标

> **注意**：水肿可能掩盖体重下降，需结合临床表现综合判断。
