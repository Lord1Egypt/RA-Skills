## 话术库

### 开场话术

```
暖场话术：
- "欢迎XX进入直播间"
- "感谢XX的关注"
- "新朋友点点关注，老朋友扣1"

自我介绍：
- "我是XX，专注XX行业X年"
- "我们有自己的工厂/源头供应链"
- "今天给老铁们带来最优惠的价格"

福利预告：
- "今天的福利非常大，先点关注等通知"
- "准备了X波秒杀，错过等一周"
```

### 产品话术

```
引出产品：
- "接下来给大家介绍一款XX"
- "很多人问我们有没有XX，今天来给大家讲讲"

产品介绍：
- "这款产品有XX特点"
- "材质是XX，规格是XX"
- "适合XX人群/场景"

价值塑造：
- "市面上同质量的要卖XX"
- "我们用的是XX品牌的原料"
- "这个工艺是XX认证的"

价格说明：
- "原价XX，今天直播间只要XX"
- "这是我们给直播间的专属价"
- "全网最低价，买贵退差价"
```

### 互动话术

```
互动引导：
- "觉得有用的扣666"
- "想要这个的扣1"
- "有问题打在公屏上"

催单话术：
- "只剩XX单了"
- "5分钟后下架"
- "已经XXX人下单了"

感谢话术：
- "感谢XX的关注"
- "感谢XX的礼物"
- "老铁们太热情了"
```

### 收尾话术

```
回顾总结：
- "今天给大家介绍了X款产品"
- "错过直播的可以看回放"

感谢告别：
- "感谢大家陪伴"
- "爱你们哟"
- "明天见"

预告引导：
- "明天的直播更精彩"
- "想来的点个预约"
- "关注不迷路"
```

## 数据复盘指标

### 核心指标定义

| 指标 | 定义 | 计算方式 |
|-----|------|---------|
| 场观 | 总观看人数 | 唯一UV之和 |
| 峰值在线 | 同时在线最高人数 | 数据峰值 |
| 新增关注 | 直播新增粉丝 | 直播前后差值 |
| 互动率 | 互动人数/场观 | 评论+点赞/场观 |
| 商品曝光 | 商品展示次数 | 后台数据 |
| 点击率 | 点击/曝光 | 点击/曝光 |
| 转化率 | 下单/点击 | 下单/点击 |
| GMV | 成交金额 | 付款金额 |
| 客单价 | GMV/订单数 | 平均订单金额 |

### 优秀参考值

| 指标 | 优秀 | 良好 | 需改进 |
|-----|------|------|--------|
| 场观 | 5000+ | 2000-5000 | <2000 |
| 互动率 | 10%+ | 5-10% | <5% |
| 点击率 | 15%+ | 8-15% | <8% |
| 转化率 | 8%+ | 3-8% | <3% |
| 客单价 | 150+ | 80-150 | <80 |
| GPM | 15%+ | 8-15% | <8% |
