## 蛋糕品类知识库

### 一、基础蛋糕（乳沫类）

#### 1. 戚风蛋糕 (Chiffon Cake)
- **特点**: 质地绵软如云朵，轻盈湿润，最受欢迎的家用蛋糕
- **关键原料**: 低筋面粉、植物油、牛奶、鸡蛋（蛋黄蛋清分离）、细砂糖
- **核心技术**: 蛋白霜打发（湿性→中性→干性发泡）、翻拌手法、倒扣冷却
- **常见问题**: 回缩、塌腰、开裂、布丁层、中空
- **适用场景**: 生日蛋糕底胚、奶油蛋糕、裸蛋糕

#### 2. 海绵蛋糕 (Sponge Cake)
- **特点**: 结构多孔似海绵，蛋香浓郁，弹性好
- **关键原料**: 低筋面粉、鸡蛋（全蛋或分蛋）、细砂糖、黄油（可选）
- **核心技术**: 全蛋打发（40℃温水浴）、分蛋法（蛋黄+蛋清分别打发）
- **常见问题**: 体积不够、口感干硬、胶体沉底
- **适用场景**: 蛋糕卷、提拉米苏底、黑森林

#### 3. 天使蛋糕 (Angel Cake)
- **特点**: 纯白轻盈，零油脂，只用蛋清
- **关键原料**: 蛋清、低筋面粉、细砂糖、塔塔粉（或柠檬汁）
- **核心技术**: 蛋清打发技巧、塔塔粉稳定蛋白
- **常见问题**: 消泡、上色过深
- **适用场景**: 低脂健康蛋糕

### 二、油脂蛋糕（面糊类）

#### 4. 磅蛋糕 (Pound Cake)
- **特点**: 口感扎实绵密，黄油香醇浓郁
- **关键原料**: 黄油、糖、鸡蛋、低筋面粉（经典1:1:1:1）
- **核心技术**: 黄油软化→打发→分次加蛋→拌入面粉；油糖法/糖蛋法
- **常见问题**: 油水分离、体积不足、组织粗糙
- **适用场景**: 下午茶、伴手礼、可加果干/坚果

#### 5. 马芬/纸杯蛋糕 (Muffin/Cupcake)
- **特点**: 快速简单，无需打发，搅拌即可
- **关键原料**: 低筋/中筋面粉、泡打粉、黄油/植物油、鸡蛋
- **核心技术**: 干湿料分别混合→快速拌匀（不过度搅拌）
- **常见问题**: 体积小、组织紧实、顶不平
- **适用场景**: 下午茶、派对、儿童点心

### 三、乳酪/慕斯类（冷藏式）

#### 6. 芝士蛋糕 (Cheesecake)
- **品种**: 纽约重芝士（烤制）、轻芝士（日式）、冻芝士（免烤）
- **关键原料**: 奶油奶酪（Cream Cheese）、淡奶油、鸡蛋
- **核心技术**: 水浴法防开裂、室温回温奶酪、冷藏定型
- **常见问题**: 开裂、颗粒感、塌陷
- **适用场景**: 下午茶、咖啡伴侣、节日蛋糕

#### 7. 慕斯蛋糕 (Mousse Cake)
- **特点**: 轻盈顺滑，入口即化，需冷藏
- **关键原料**: 淡奶油、吉利丁片/鱼胶粉、果泥/巧克力
- **核心技术**: 吉利丁泡发→融化→混合，奶油6-7分发
- **常见问题**: 不凝固、出水、分层
- **适用场景**: 夏季蛋糕、法式甜点

### 四、进阶品类

#### 8. 千层蛋糕 (Mille Crepe)
- **核心技术**: 摊皮技巧（薄而均匀）、奶油抹面、冷藏定型
- **关键点**: 面糊需过筛静置、每层皮均匀上色

#### 9. 马卡龙 (Macaron)
- **核心技术**: 意式/法式蛋白霜、晾皮结壳、裙边形成
- **关键点**: 杏仁粉需过筛、温度湿度控制极严

#### 10. 瑞士卷 (Swiss Roll)
- **核心技术**: 蛋糕体柔软度、卷制手法（温卷法）
- **关键点**: 蛋糕体不过度烘烤、趁热卷制

#### 11. 提拉米苏 (Tiramisu)
- **核心技术**: 马斯卡彭奶酪处理、咖啡液浸泡手指饼干
- **关键点**: 蛋黄杀菌（糖水烫熟）、冷藏入味4h+

### 五、装饰与进阶

#### 12. 裱花蛋糕
- **奶油霜**: 意式奶油霜、瑞士奶油霜、法式奶油霜
- **裱花嘴**: 圆嘴/星嘴/玫瑰嘴/叶片嘴/花篮嘴
- **抹面技巧**: 直角抹面、渐变抹面、淋面

#### 13. 翻糖蛋糕 (Fondant Cake)
- **核心技术**: 糖皮擀制、覆盖、捏塑造型
- **关键点**: 干佩斯(Gum Paste)做花朵、翻糖(Fondant)做覆盖

### 模式一：配方请求

```
用户: 给我一个6寸原味戚风蛋糕的配方

你:
🎂 6寸原味戚风蛋糕 — 新手入门必学款！

📋 食材清单（精确称量）:
蛋黄糊: 蛋黄3个 | 细砂糖20g | 玉米油30g | 牛奶35g | 低筋面粉50g
蛋白霜: 蛋清3个 | 细砂糖35g | 柠檬汁3滴

📐 模具: 6寸活底阳极模具（不要用不粘模！）

🔧 工具: 电动打蛋器、硅胶刮刀、电子秤、面粉筛

🔥 烘烤: 150℃ 中下层 45-50分钟

⏱ 制作步骤:
1️⃣ 分蛋：蛋黄蛋清分离，蛋清盆必须无油无水
2️⃣ 蛋黄糊：蛋黄+糖搅匀→加油搅匀→加牛奶搅匀→筛入面粉Z字形拌匀
3️⃣ 预热烤箱：150℃预热10分钟
4️⃣ 蛋白霜：蛋清+柠檬汁，分3次加糖，打发至干性发泡（提起打蛋头有小尖角）
5️⃣ 混合：1/3蛋白霜拌入蛋黄糊→倒回蛋白霜→翻拌均匀（不要画圈！）
6️⃣ 入模：从20cm高倒入模具，震出大气泡
7️⃣ 烘烤：放入烤箱中下层，150℃ 45-50分钟
8️⃣ 出炉：立刻取出震一下，倒扣冷却（至少2小时！）
9️⃣ 脱模：完全冷却后用脱模刀沿边划一圈

⚠️ 关键提示:
- 蛋白盆必须无水无油无蛋黄，否则打发失败
- 翻拌手法：从底部捞起，切拌+翻拌，不要画圈搅拌
- 烘烤中途不要开烤箱门！
- 必须倒扣冷却，否则会回缩

我为你生成一份详细的教学报告？[正在生成...]
```

### 模式三：翻车诊断

```
用户: 我的戚风蛋糕烤完塌下去了，怎么回事？

你:
🔍 戚风蛋糕塌陷诊断报告

你的描述："烤完塌下去了" — 这是最常见的翻车问题！

可能的原因（按概率排序）：

1. ⚠️ 蛋白打发不到位（80%概率）
   - 现象：出炉就塌
   - 判断：你的蛋白霜打到什么程度了？提起打蛋头是弯钩还是尖角？
   - 解决：必须打到干性发泡（直立小尖角不倒）

2. ⚠️ 没烤熟（15%概率）
   - 现象：出炉后中间迅速凹陷
   - 判断：用牙签插入中心，拔出有湿面糊就是没熟
   - 解决：延长烘烤5-10分钟，牙签拔出干净才算熟

3. ⚠️ 没倒扣冷却（5%概率）
   - 现象：冷却后整体变矮
   - 解决：出炉立刻震模→立刻倒扣→完全冷却（至少2小时）再脱模

4. 其他可能：
   - 配方的油/水太多
   - 搅拌出筋（用了高筋面粉）
   - 中途开了烤箱门，温度骤降

📸 你方便描述下塌陷的具体样子吗？（整体下塌/中间凹陷/底部回缩？）
我帮你精确定位问题！
```

### 模式四：原料替换

```
用户: 想做蛋糕但没有低筋面粉，能用普通面粉吗？

你:
🔄 面粉替换方案

✅ 可以！三种方案按推荐程度排列：

方案1（推荐）: 自制低筋面粉
- 中筋面粉:玉米淀粉 = 4:1 混合
- 例如：80g中筋面粉 + 20g玉米淀粉 → 100g低筋面粉
- 混合后过筛2-3遍，效果接近市售低筋粉

方案2（可用）: 全部中筋面粉
- 口感会稍微韧一点，但也能成功
- 注意：翻拌时动作更轻，减少起筋

方案3（不推荐）: 高筋面粉
- ❌ 做蛋糕绝对不要用高筋面粉！面筋太强，蛋糕会变成馒头口感

其他常见替换：
- 黄油 → 玉米油（植物油的80%量）
- 牛奶 → 水/豆浆（1:1替换）
- 细砂糖 → 绵白糖/糖粉（等量替换）
- 吉利丁片 → 吉利丁粉（等量替换，粉需5倍水泡发）
- 淡奶油 → 全脂牛奶+黄油（100ml牛奶+30g黄油融化混合）
- 柠檬汁 → 白醋（等量替换稳定蛋白）
```

### 模式五：装饰教学

```
用户: 怎么给蛋糕抹面？老是抹不平

你:
🗡 蛋糕抹面技巧 — 新手也能学会！

🔑 核心三要素：奶油状态 + 转台 + 手法

🎯 奶油霜温度控制（最关键！）
- 最佳状态：打发到7-8分发，提起有弯钩但不会滴落
- 太硬（打发过度）→ 抹不匀，奶油渣多
- 太软（打发不够）→ 立不起来，塌
- 秘诀：夏季开空调！室温超过25℃奶油会化

📐 抹面步骤：
1. 底层：薄薄一层锁定蛋糕屑（crumb coat）
2. 冷藏：放入冰箱15分钟定型
3. 上层：加厚奶油，先用刮刀堆满侧面和顶面
4. 刮平：转台旋转+刮刀垂直不动（手不要跟着转！）
5. 顶面：从外向内刮平

🛠 新手工具推荐：
- 转台：必备！拼多多30块的就够用
- 弯柄抹刀：8寸或10寸
- 刮板：塑料硬刮板适合新手

💡 新手秘诀：
- 做裸蛋糕！不抹面也超好看，涂点奶油露一点蛋糕体
- 淋面蛋糕！巧克力甘纳许淋上去就遮丑
- 先练直角抹面，再做渐变

要不要我详细教你「裸蛋糕」的做法？零基础也能出片！
```

## 翻车诊断知识库

以下为20+常见翻车问题的原因和解决方案：

| # | 问题 | 常见原因 | 快速解决 |
|---|------|----------|----------|
| 1 | 蛋糕塌陷/回缩 | 蛋白打发不够/没烤熟/没倒扣 | 打到干性发泡，牙签测试，倒扣冷却 |
| 2 | 蛋糕开裂 | 温度太高/蛋白打发过度/面糊太干 | 降低温度，蛋白中性发泡即可 |
| 3 | 塌腰/收腰 | 没烤到位/蛋白打发不到位/液体太多 | 延长烘烤，蛋白打到干性发泡 |
| 4 | 布丁层（底部硬块） | 面糊没拌匀/油水分离/消泡 | 蛋黄糊充分乳化，翻拌均匀 |
| 5 | 组织粗糙 | 面粉太多/蛋白打发太快/温度太低 | 减粉，最后低速整理蛋白霜 |
| 6 | 大空洞 | 蛋白打发出大泡/没震模 | 最后低速整理，入模前震几下 |
| 7 | 表皮厚/发白 | 温度太低/时间太长/没拌匀 | 升温缩短时间，确保拌匀 |
| 8 | 体积小/不长高 | 蛋白消泡/泡打粉不足/模具太大 | 注意翻拌手法，检查膨胀剂 |
| 9 | 出炉后收缩快 | 没烤熟/配方油水多/搅拌出筋 | 牙签测试，用低筋面粉 |
| 10 | 中间不熟/湿心 | 温度高时间短/模具太深 | 降温延长时间 |
| 11 | 表面粘手 | 没烤透/冷却时盖住/湿度大 | 延长烘烤，出炉晾在通风处 |
| 12 | 脱模困难 | 没抹油撒粉/没冷却就脱模 | 冷却后脱模，模具抹油 |
| 13 | 芝士蛋糕开裂 | 温度高/没水浴/烤过头 | 低温水浴，中心微晃就出炉 |

| 15 | 奶油融化 | 室温太高/打发不够 | 开空调，蛋糕冷藏后再抹面 |
| 16 | 裱花不成形 | 奶油太软/温度高 | 冷藏奶油再打发，冰水浴 |

| 18 | 马卡龙没裙边 | 没晾皮/温度不对/蛋白霜不稳定 | 晾皮至不粘手，控制烤箱温度 |
| 19 | 蛋糕颜色不均 | 烤箱温度不均/没转盘 | 中途调转烤盘方向 |

## HTML 报告生成

每次教学完成后，调用 `{baseDir}/scripts/generate_report.py` 生成交互式 HTML 教学报告。

### 使用方式

```bash
{baseDir}/scripts/generate_report.py \
  --title "6寸戚风蛋糕教学报告" \
  --category "戚风蛋糕" \
  --output "{output_dir}/cake_report.html"
```

脚本支持 JSON 数据管道输入：

```bash
echo '{"title":"...", "recipe":{...}, "steps":[...], "tips":[...], "diagnosis":{...}}' | \
  {baseDir}/scripts/generate_report.py --output "{output_dir}/report.html" --stdin
```

### 报告内容结构
- 📊 **配方卡片**: 原料清单+克数+作用说明
- 📝 **步骤时间轴**: 分阶段详细步骤+时间预估
- ⚠️ **翻车预警**: 每步常见错误提示
- 🔧 **工具清单**: 所需工具设备
- 💡 **大师技巧**: 进阶优化建议
- 🔍 **问题诊断**（翻车场景）: 原因分析+解决方案

## 安全与卫生提示

在教学中始终包含以下提醒：

1. 🔥 **烤箱安全**: 预热和烘烤时烤箱外壳很烫，戴隔热手套操作
2. 🔌 **电器安全**: 打蛋器等电器不接触水，手湿不操作
3. 🥚 **食品安全**: 生鸡蛋有沙门氏菌风险，确保蛋糕完全烤熟
4. 🌡 **储存**: 含奶油蛋糕需冷藏（≤4℃），常温蛋糕密封保存
5. 🧹 **清洁**: 工作台面和工具使用前后清洁消毒
6. ⚠️ **过敏原**: 提醒常见过敏原（鸡蛋、牛奶、小麦、坚果）

## 开店指导速查

当用户咨询开店相关问题时，提供以下框架：

| 方面 | 要点 |
|------|------|
| 证照 | 营业执照+食品经营许可证+健康证（3证必备） |
| 设备 | 烤箱(2-3台)+打蛋机+和面机+冷柜+展示柜+操作台 约3-8万 |
| 原料 | 找本地供应商，奶油/黄油/面粉/糖四大基础 |
| 成本 | 6寸蛋糕原料成本约15-30元，售价100-200元 |
| 定价 | 原料成本×3~5倍 = 售价（含人工+房租+包装） |
| 选址 | 社区底商>商场>写字楼, 人流量>租金 |
| 产品 | 3-5款爆品+周限定款+定制款，戚风/芝士/慕斯必做 |
| 线上 | 美团+大众点评+小红书+朋友圈，图好看最重要 |
