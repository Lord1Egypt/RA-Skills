# 儿童慢性胃炎与消化性溃疡的鉴别诊断

## 何时使用
当接诊以下患儿时立即启用本技能：
- 反复腹痛的儿童患者
- 主诉上腹部不适的患儿
- 伴有消化道症状（如恶心、呕吐、食欲缺乏）的学龄儿童

**前提条件**：已采集完整病史并完成基础体格检查。

**排除情况**：不适用于急性外科急腹症（如阑尾炎穿孔）的初步判断。

## 执行步骤

### 1. 评估腹痛特征
- **时间规律**：
  - 消化性溃疡：饥饿时或夜间多发，进食后缓解；
  - 慢性胃炎：进食中或餐后出现，无明显规律；
  - 功能性腹痛：发作性，持续数十分钟至数小时，熟睡后消失。
- **疼痛性质**：
  - 慢性胃炎：间歇性隐痛、钝痛，严重者为剧烈绞痛；
  - 消化性溃疡（年长儿）：烧灼感、胀痛；
  - 嗜酸性粒细胞性胃肠炎：类似胃炎但常规治疗无效。

### 2. 分析伴随症状
- 呕血、黑便 → 提示胃黏膜糜烂出血或消化性溃疡并发症；
- 营养不良、生长发育迟缓 → 更倾向慢性胃炎或克罗恩病；
- 吐虫、排虫史、异食癖 → 考虑肠蛔虫病（现已少见）；
- 心理因素、家庭焦虑、情绪波动 → 支持功能性腹痛；
- 外周血嗜酸性粒细胞升高 → 提示嗜酸性粒细胞性胃肠炎。

### 3. 指导辅助检查
- **胃镜检查**：直接观察黏膜病变——胃炎表现为充血、水肿、糜烂；溃疡呈圆形/椭圆形病灶伴灰白苔；
- **粪便虫卵检查**：排除肠蛔虫病；
- **内镜下活检**：嗜酸性粒细胞性胃肠炎可见大量嗜酸性粒细胞浸润；
- **功能性腹痛**：临床及辅助检查通常无阳性发现。

### 4. 注意特殊事项
- 年龄越小，症状越不典型；
- “熟睡后症状消失”是功能性腹痛的重要线索；
- 嗜酸性粒细胞性胃肠炎对常规胃炎治疗无效，需免疫调节治疗。

## 输出结果
综合上述信息，生成鉴别诊断建议，明确最可能的病因方向，并推荐下一步检查项目（如胃镜、粪检、血嗜酸细胞计数等）。
