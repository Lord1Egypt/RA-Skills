# 内容资产评估框架

## 一、内容生产能力评估

### 1.1 产出量评估

**核心指标：**

| 指标 | 定义 | 计算方式 | 评估标准 |
|-----|------|---------|---------|
| 日均产出量 | 每日发布内容数 | 总发布数/统计周期天数 | 稳定>断更 |
| 周均产出量 | 每周发布内容数 | 总发布数/周数 | ≥3条为活跃 |
| 月均产出量 | 每月发布内容数 | 总发布数/月数 | ≥12条为稳定 |
| 内容更新频率 | 相邻内容发布时间间隔 | 平均间隔时间 | 规律>随意 |

**评估方法：**
1. 调取后台发布记录，统计近3-6个月发布频率
2. 识别断更周期，分析断更原因（主动/被动/节假日）
3. 评估内容储备量（已制作未发布内容）
4. 对比同类账号平均发布频率

**判断标准：**
```
🟢优质：日更或隔日更，无超过7天断更
🟡合格：周更3条以上，断更不超过2周
🔴风险：月更不足4条，或有超过1个月断更
```

### 1.2 内容质量评估

**质量维度：**

| 维度 | 指标 | 计算公式 | 行业参考值 |
|-----|------|---------|----------|
| 完播率 | 完整观看用户占比 | 完播人数/播放人数×100% | >40%为优质 |
| 互动率 | 互动用户占比 | (赞+评+转+藏)/播放量×100% | >5%为优质 |
| 传播系数 | 分享带来的二次传播 | 分享数/播放数×100% | >2%为优质 |
| 吸粉率 | 观看后关注占比 | 新增粉丝/观看人数×100% | >3%为优质 |

**深度质量评估：**

```
内容质量得分 = 完播率×30% + 互动率×25% + 传播系数×25% + 吸粉率×20%
```

**内容类型对比：**

| 内容类型 | 平均完播率 | 平均互动率 | 变现能力 |
|---------|----------|----------|---------|
| 短视频(15-60s) | 30-50% | 3-8% | 中等 |
| 中视频(1-5min) | 40-60% | 5-10% | 较强 |
| 长视频(>5min) | 50-70% | 8-15% | 强 |
| 图文/图文笔记 | N/A | 2-6% | 中等 |
| 直播 | 实时互动 | 10-30% | 强 |

### 1.3 内容迭代能力评估

**评估维度：**

| 能力类型 | 评估内容 | 判断标准 |
|---------|---------|---------|
| 热点响应速度 | 从热点发生到内容产出时间 | <72小时为合格 |
| 热点复刻能力 | 爆款内容模式复制成功率 | >50%为优秀 |
| 内容形式迭代 | 新形式尝试频率与效果 | 有迭代记录 |
| 差异化能力 | 与同类账号的内容区分度 | 明显差异化 |

**评估方法：**
1. 梳理近半年热点事件，核查内容响应情况
2. 分析爆款内容规律，验证可复制性
3. 对比同类账号，评估差异化程度
4. 访谈核心创作者，了解迭代流程

### 1.4 创意团队评估

**团队结构分析：**

| 角色 | 职责 | 关键评估点 |
|-----|------|---------|
| 核心创作者 | 内容方向把控、爆款产出 | 从业年限、代表作品、粉丝量 |
| 执行团队 | 内容制作、执行落地 | 专业度、稳定性 |
| 运营团队 | 数据分析、发布运营 | 平台运营经验 |
| 商务团队 | 客户对接、商业变现 | 客户资源、服务能力 |

**团队稳定性指标：**

```
年人员流动率 = 年度离职人数 / 年度平均人数 × 100%
核心人员留存率 = 期末核心人员 / 期初核心人员 × 100%
```

**风险信号：**
- 🔴 核心创作者为兼职或外部合作
- 🔴 核心创作者合同即将到期且未续签
- 🔴 团队流动率>30%/年
- 🔴 核心人员年龄偏大或精力不足

### 2.2 粉丝质量评估

**核心指标：**

| 指标 | 定义 | 优质标准 | 风险信号 |
|-----|------|---------|---------|
| 真粉率 | 真实用户关注占比 | >70% | <50%为刷量风险 |
| 活跃粉丝占比 | 近30天有互动用户占比 | >30% | <10%为僵尸粉 |
| 互动粉丝占比 | 有评论/转发用户占比 | >5% | <2%为异常 |
| 铁粉占比 | 持续互动核心粉丝占比 | >3% | <1%为缺乏忠诚度 |

**检测方法：**

```python
# 粉丝质量检测逻辑
def check_fan_quality(comments, likes, followers):
    # 1. 评论用户与粉丝比
    comment_ratio = unique_commenters / followers
    
    # 2. 互动集中度
    top_comments_ratio = top_10_comments / total_comments
    
    # 3. 评论内容分析
    real_comments = detect_real_comments(comments)
    
    # 4. 综合评分
    quality_score = calculate_score(...)
    return quality_score
```

**异常信号识别：**
- 🟠 粉丝暴涨后无对应内容产出
- 🟠 评论内容高度相似或为无意义内容
- 🟠 点赞数与评论数比例异常
- 🔴 粉丝画像与内容定位不匹配

### 2.4 流量成本分析

**成本指标：**

| 指标 | 定义 | 计算方式 | 行业参考 |
|-----|------|---------|---------|
| CPM | 千次曝光成本 | 投放金额/曝光量×1000 | 10-50元 |
| CPC | 单次点击成本 | 投放金额/点击量 | 0.5-3元 |
| CPA | 单次行动成本 | 投放金额/行动数 | 5-50元 |
| ROI | 投放回报率 | GMV/投放金额 | >2为合格 |

**成本趋势分析：**
- 统计近6个月CPM/CPC变化
- 分析成本上升/下降原因
- 评估成本变化对盈利的影响

### 2.5 私域资产评估

**私域资产类型：**

| 私域类型 | 评估指标 | 质量判断 |
|---------|---------|---------|
| 微信社群 | 群数量、人数、活跃度 | 500人×群数，有效转化 |
| 微信公众号 | 关注数、阅读量、打开率 | 打开率>3%为合格 |
| 微信小程序 | 用户数、DAU、留存率 | 次日留存>30% |
| 会员体系 | 会员数量、付费率、续费率 | 续费率>60% |

**私域价值计算：**

```
私域月GMV = 私域用户数 × 购买转化率 × 客单价 × 月均购买频次
私域LTV = 月GMV / 用户数 × 平均生命周期(月)
```

## 三、IP资产评估

### 3.1 自有IP统计

**IP资产清单：**

| IP名称 | 内容类型 | 粉丝量 | 生命周期阶段 | 版权状态 |
|-------|---------|-------|-------------|---------|
| - | - | - | 成长期/成熟期/衰退期 | 已注册/未注册 |

**生命周期阶段判断：**
```
成长期：粉丝增长>20%/季度，内容热度上升
成熟期：粉丝稳定，内容产出稳定，变现成熟
衰退期：粉丝负增长，内容热度下降，更新减少
```

### 3.2 IP价值评估方法

**方法一：成本法**

```
IP估值 = 历史投入成本 × 系数
系数参考：
- 成熟期IP：3-5倍
- 成长期IP：5-10倍
- 衰退期IP：0.5-1倍
```

**方法二：市场法**

```
IP估值 = 同类IP授权均价 × 调整系数
调整因素：
- 粉丝量差异
- 内容质量差异
- 商业化成熟度
- 法律保护完善度
```

**方法三：收益法**

```
IP估值 = Σ(预期年收入 × 折现系数) / 折现率
保守估计周期：3-5年
折现率：10-15%
```
