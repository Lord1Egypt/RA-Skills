## Description: <br>
UI Designer helps agents produce product-grade UI and front-end design guidance with React, Vue, design-token, responsive-system, and performance considerations. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[WesLon1987](https://clawhub.ai/user/WesLon1987) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and product designers use this skill to prompt an agent for polished web UI descriptions, design token tables, and runnable Vue/TailwindCSS examples. It is suited to front-end ideation and implementation planning where React, Vue, responsive layout, SSR, and product-level design quality matter. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated Vue/TailwindCSS or other front-end code may be incomplete, incorrect, or unsuitable for a production codebase. <br>
Mitigation: Review, test, and scan generated code before using it in a real project. <br>
Risk: Generated UI guidance may not fully satisfy project accessibility, brand, localization, or performance requirements. <br>
Mitigation: Validate generated designs against the target product requirements before implementation. <br>
