# 继发性肺结核病的定义与发病机制

## 适用场景
- 成人患者出现肺部新发浸润或空洞病变
- 患者有明确的既往结核分枝杆菌感染史
- 免疫力下降（如HIV感染、糖尿病、长期使用免疫抑制剂）背景下出现结核样症状

## 执行步骤
1. **确认前提条件**：
   - 患者为成人；
   - 存在既往结核感染史及静止的原发病灶。

2. **排除不适用情况**：
   - 若为初次感染者，应诊断为原发性肺结核；
   - 儿童患者罕见继发性肺结核，需谨慎鉴别。

3. **识别典型特征**：
   - 病变起始部位多位于肺尖部（区别于原发性肺结核的上叶下部/下叶上部近胸膜处）；
   - 主要播散途径为支气管播散（而非淋巴道或血道）。

4. **分析发病机制**：
   - 可由外源性再感染引起，或由静止病灶在免疫力下降时内源性再活化所致；
   - 因机体已致敏，病理反应以干酪样坏死和空洞形成为主。

5. **评估病程特点**：
   - 病程具有长期性与波动性；
   - 通常不能自愈，需规范抗结核治疗控制病情。

6. **综合诊断建议**：结合影像学表现（如肺尖空洞）、临床症状（咳嗽、咯血、低热）及结核病史，做出继发性肺结核病诊断。
