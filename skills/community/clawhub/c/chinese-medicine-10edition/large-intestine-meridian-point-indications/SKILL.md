# Large Intestine Meridian Point Indications

## When to Use
当患者出现以下任一情况时，考虑使用本技能：
- 头面部病症（如齿痛、目赤肿痛、鼻衄、咽喉肿痛、口眼喎斜、头痛）
- 热病或神志病（如发热无汗、神昏、癫狂）
- 上肢局部病症（如手指麻木肿痛、肘臂不遂、肩痛）

**禁忌**：孕妇禁用合谷等特定穴位；内脏实质性器质病变且未经中医辨证者不宜使用。

## Execution Steps
1. **确认适应症**：判断患者症状是否属于手阳明大肠经循行所过部位（上肢外侧前缘、下齿、肩颈、头面）及相关功能范畴。
2. **选择穴位**：
   - 面口五官疾患 → 选用合谷（LI4）
   - 热病、皮肤病（如瘾疹、湿疹）、瘰疬 → 选用曲池（LI11）
   - 咽喉肿痛、高热神昏 → 选用商阳（LI1），可点刺出血
3. **执行操作**：按针灸规范进行针刺或点刺，注意避开禁忌人群。
4. **配伍提示**：虚寒证需配合温补类穴位或疗法，不可单独使用本经穴位。

> 注意：本经穴位以疏风解表、清热通络为主，主要用于实证、热证。
