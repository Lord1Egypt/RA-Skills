# 牛妖股评分体系

A股超短线选股工具，基于题材强度、股性、开盘涨幅、换手率、市场情绪和连板高度六维评分，从近3日涨停池中筛选最具接力价值的龙头候选。

---

## 功能概述

**核心定位**：游牧型超短线 —— 只做主线、只做最强、1-2天T+1套利。

从近3日涨停池中六维评分，选出次日最具接力价值的Top20龙头候选。

---

## 核心特性

### 六维评分体系（总分100）

| 维度 | 权重 | 前瞻性 | 说明 |
|------|------|--------|------|
| 题材强度 | 25分 | ⭐⭐⭐⭐⭐ 盘前可知 | 排名基础分+占比加成 |
| 股性 | 20分 | ⭐⭐⭐⭐⭐ 盘前可知 | 近10日涨停次数 |
| 开盘涨幅 | 15分 | ⭐⭐⭐⭐ 竞价可知 | 竞价开板幅度 |
| 换手率 | 15分 | ⭐⭐⭐ 当天可知 | 实际换手率 |
| 市场情绪 | 15分 | ⭐⭐⭐ 当天可知 | 全市场涨停数 |
| 连板高度 | 10分 | ⭐⭐ 辅助验证 | 当前连板数 |

### 硬门槛过滤

1. 非ST/退市风险
2. 非一字独食板（换手率>0.5%）
3. 成交额>5000万
4. 流通市值<200亿（小票优先）

## 报告结构

### 市场概览
- 3日候选池统计
- 最强主线Top3
- 题材分布Top10

### Top20榜单
- 排名、股票名、代码、等级、总分
- 六维分项得分
- 今日是否涨停标记（🔴/⚪）

### 技术指标详情
- MA5/MA10/MA20/MA60 均线系统
- MACD（DIF/DEA/MACD柱）
- RSI6/12/24 相对强弱
- KDJ（K/D/J值）
- 布林带（上轨/中轨/下轨）
- SAR、CCI、WR6

### 关键信号解读
- MACD动能方向
- RSI超买超卖
- KDJ极端值预警

### 次日策略建议
- A级候选重点观察
- 等级划分与操作建议

## 等级划分

| 等级 | 分数 | 操作建议 |
|------|------|----------|
| S级 | ≥85 | 次日重点关注，竞价超预期直接上 |
| A级 | 70-84 | 强候选，结合盘面决策 |
| B级 | 55-69 | 观察，需其他信号确认 |
| C级 | 40-54 | 谨慎，仅作参考 |
| D级 | <40 | 排除 |
