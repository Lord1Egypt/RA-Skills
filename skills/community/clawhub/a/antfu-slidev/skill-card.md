## Description: <br>
Create and present web-based slidedecks for developers using Slidev with Markdown, Vue components, code highlighting, animations, and interactive features. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and technical presenters use this skill to create, edit, run, export, and publish Slidev decks for talks, workshops, code walkthroughs, and teaching material. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated workflows may include package-manager commands that create, run, build, or export Slidev projects. <br>
Mitigation: Review commands before execution and run them only in intended project directories. <br>
Risk: Optional Slidev remote control, Monaco write/run features, and remote asset loading can expose unintended access or content changes when enabled casually. <br>
Mitigation: Enable those features only where that access is intended and review generated slides before publishing. <br>
