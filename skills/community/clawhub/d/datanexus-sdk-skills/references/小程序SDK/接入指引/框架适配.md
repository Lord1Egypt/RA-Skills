# 微信小程序 SDK 框架适配指南

不同开发框架在引入和初始化 SDK 时存在差异，请根据项目使用的框架参考对应的适配方式。

---

## 原生小程序

```javascript
const SDK = require('./datanexus-sdk-mp');
App({
  onLaunch() {
    this.sdk = new SDK({ user_action_set_id: 'ID', secret_key: 'KEY', appid: 'APPID' });
  }
});
```

---

## Taro

```javascript
import SDK from '@dn-sdk/miniprogram';
const sdk = new SDK({ user_action_set_id: 'ID', secret_key: 'KEY', appid: 'APPID' });
export default sdk;
```

---

## mpvue

```javascript
import SDK from '@dn-sdk/miniprogram';
import Vue from 'vue';
Vue.prototype.$sdk = new SDK({ user_action_set_id: 'ID', secret_key: 'KEY', appid: 'APPID' });
```

---

## WePY 1.x / 2.x

在 `app.wpy` 的 `onLaunch` 中初始化。

---

## uni-app

```javascript
// #ifdef MP-WEIXIN
import SDK from '@dn-sdk/miniprogram';
import Vue from 'vue';
Vue.prototype.$sdk = new SDK({ user_action_set_id: 'ID', secret_key: 'KEY', appid: 'APPID' });
// #endif
```

---

## Kbone

在小程序侧入口初始化，Web 端不支持。

---

## Remax

```javascript
import SDK from '@dn-sdk/miniprogram';
const sdk = new SDK({ user_action_set_id: 'ID', secret_key: 'KEY', appid: 'APPID' });
export default sdk;
```

---

## 通用注意事项

1. 所有框架均需配置安全域名 `https://api.datanexus.qq.com`
2. npm 引入需「构建 npm」
3. SDK 实例建议全局单例
4. 如遇报错，检查构建工具是否正确处理 npm 包
