| 业务类型 | 正常佣金区间 | 说明 |
|----------|--------------|------|
| 即期本票贴现 | **1-2%** | 银行正常收费 |

| 特征 | 正常情况 | 说明 |

| 支付对象 | 银行或正规机构 | 有明确账户 |
| 支付时间 | 交易完成后 | 有保障 |
| 佣金名目 | 合理解释 | 名目清晰 |

## 二、异常佣金结构识别

### 2.1 异常佣金分级

| 佣金区间 | 风险等级 | 建议 |
|----------|----------|------|
| 3-5% | 🟡警示 | 需核实业务真实性 |
| 5-10% | 🟠警告 | 大概率存在问题 |
| ≥10% | 🔴危险 | 明确诈骗信号 |

### 2.2 异常佣金特征

| 特征 | 异常情况 | 风险提示 |
|------|----------|----------|
| 佣金比例 | >5% | 明显偏高 |
| 分配层级 | 多层分赃 | 诈骗特征 |
| 支付对象 | 个人或不明账户 | 高风险 |
| 支付时间 | 先付佣金 | 典型诈骗 |
| 佣金名目 | 名目不清 | 掩饰目的 |

### 2.3 佣金异常的典型话术

| 话术类型 | 举例 | 风险提示 |
|----------|------|----------|
| 高回报诱惑 | "只需3%佣金即可获得5亿本票" | 🟠警告 |
| 层层分赃 | "这笔钱要分给ABCD四个人" | 🟠警告 |
| 提前支付 | "先付诚意金，票到了再结清" | 🔴危险 |
| 名目模糊 | "这是通关费/手续费/茶水费" | 🟠警告 |
| 催促付款 | "机会难得，今天不付就没了" | 🟠警告 |

## 三、金字塔分赃结构识别

### 3.1 什么是金字塔分赃

**定义**：通过多层中间人分配佣金，每一层都抽取一定比例，形成金字塔形状的利益分配结构。

**特征**：
- 多层中间人（≥3层）
- 每层都有佣金抽成
- 总佣金比例极高（≥20%）
- 底层买家承担所有成本

### 3.2 正常结构 vs 金字塔结构

**正常结构（单层）**：
```
A（本票持有人）→ 收取 1-2% 佣金 → 交易完成
```

**正常结构（双层）**：
```
A（本票持有人）→ B（合法中介）→ 收取 1.5%
B → C（专业服务）→ 收取 0.5%
总计：2%
```

**金字塔分赃结构（诈骗特征）**：
```
A（本票持有人/诈骗者）
        ↓ 声称持有多张本票
B（第一中间人）→ 提取 3%
        ↓
C（第二中间人）→ 提取 5%
        ↓
D（第三中间人）→ 提取 8%
        ↓
E（第四中间人）→ 提取 12%
        ↓
F（第五中间人）→ 提取 15%+
        ↓
总计佣金：43%+（诈骗）
```

### 3.3 真实案例：5亿欧元本票佣金结构

**文档中的佣金要求**：
| 本票类型 | 佣金结构 | 总计 |
|----------|----------|------|
| 即期本票 | 3% | 3% |
| 远期本票 | 50% + 1% + 1% + 1% | 53% |

**分析**：
- 即期本票3%偏高（正常1-2%）
- 远期本票53%明显是金字塔结构
- 50%主佣金 + 3个1%可能是分赃给不同中间人

### 3.4 金字塔结构识别要点

| 识别要点 | 检查方法 |
|----------|----------|
| 询问佣金分配 | "佣金是怎么分配的？" |
| 观察中间人数量 | ≥3个中间人→警惕 |
| 计算总佣金比例 | ≥10%→危险 |
| 追问名目 | "这笔钱给谁？为什么？" |
