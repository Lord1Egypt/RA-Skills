# 打分标准参考表

## 一、团队维度评分细则（满分20分）

### 1.1 创始人背景（6分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 5-6分 | 985/211或海外名校背景；10年以上行业经验；知名企业高管经历；成功创业或退出经验 |
| 良好 | 4分 | 知名院校背景；5-10年行业经验；有知名企业工作经历 |
| 一般 | 2-3分 | 普通院校背景；3-5年行业经验；有相关工作经验 |
| 较差 | 0-1分 | 院校背景一般；行业经验不足3年；无相关背景 |

### 1.2 团队完整性（5分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 完整 | 5分 | 技术+产品+市场+运营 四类岗位齐全 |
| 基本完整 | 3-4分 | 技术+产品+市场/运营 三类岗位齐全 |
| 缺失较多 | 1-2分 | 仅技术团队，或团队严重不完整 |
| 极不完整 | 0分 | 团队信息缺失或不明确 |

### 1.3 团队互补性（5分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 5分 | 技能互补、性格互补、资源互补、价值观一致 |
| 良好 | 3-4分 | 技能有一定互补，资源配置基本合理 |
| 一般 | 1-2分 | 团队成员背景相似，互补性不明显 |
| 较差 | 0分 | 团队信息不足或明显不合理 |

### 1.4 股权结构（4分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 4分 | 创始人持股40-60%；预留10-15%期权池；有动态调整机制 |
| 良好 | 3分 | 创始人持股60-70%；有期权池设计 |
| 一般 | 1-2分 | 股权过度集中或分散；无期权池 |
| 较差 | 0分 | 股权结构存在明显隐患 |

## 二、市场维度评分细则（满分20分）

### 2.1 市场规模（5分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 5分 | 明确TAM/SAM/SOM三层数据；有权威来源；数据合理 |
| 良好 | 3-4分 | 有市场规模数据；来源可查；逻辑合理 |
| 一般 | 1-2分 | 有简单市场规模描述；缺乏细分数据 |
| 较差 | 0分 | 无市场数据或数据明显不合理 |

### 2.2 市场增速（4分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 4分 | 有行业增长率数据；有权威来源；趋势判断准确 |
| 良好 | 3分 | 有增速描述；趋势判断基本合理 |
| 一般 | 1-2分 | 提及市场增长但无具体数据 |
| 较差 | 0分 | 未提及或判断明显错误 |

### 2.3 痛点精准度（6分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 5-6分 | 痛点具体、真实、可量化；用户迫切性高；已有验证 |
| 良好 | 3-4分 | 痛点描述清晰；有一定针对性 |
| 一般 | 1-2分 | 痛点泛泛而谈；缺乏具体场景 |
| 较差 | 0分 | 痛点不明确或逻辑不通 |

### 2.4 用户清晰度（5分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 5分 | 有清晰用户画像；细分明确；可触达 |
| 良好 | 3-4分 | 有目标用户描述；画像基本清晰 |
| 一般 | 1-2分 | 用户描述模糊；画像不明确 |
| 较差 | 0分 | 目标用户不清晰或错误 |

## 三、创新维度评分细则（满分20分）

### 3.1 技术创新度（6分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 5-6分 | 有核心专利；技术壁垒高；研发实力强；国际领先 |
| 良好 | 3-4分 | 有一定技术积累；有自己的技术特点 |
| 一般 | 1-2分 | 技术描述模糊；无明显壁垒 |
| 较差 | 0分 | 无技术创新点或描述不可信 |

### 3.2 模式创新度（5分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 5分 | 商业模式独特；盈利路径清晰；可规模化 |
| 良好 | 3-4分 | 模式有一定创新；盈利逻辑可行 |
| 一般 | 1-2分 | 模式同质化严重；创新不明显 |
| 较差 | 0分 | 模式不清晰或明显不可行 |

### 3.3 差异化竞争力（5分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 5分 | 差异化明显；壁垒清晰；难以复制 |
| 良好 | 3-4分 | 有一定差异化；壁垒一般 |
| 一般 | 1-2分 | 差异化不显著 |
| 较差 | 0分 | 无差异化或照搬竞品 |

### 3.4 竞品对比（4分）

| 等级 | 分值 | 评判标准 |
|------|------|----------|
| 优秀 | 4分 | 竞品分析完整；优劣势客观；知己知彼 |
| 良好 | 3分 | 有竞品分析；对比基本准确 |
| 一般 | 1-2分 | 竞品分析简单或缺失 |
| 较差 | 0分 | 完全无竞品分析 |

## 六、综合评级标准

| 总分区间 | 等级 | 说明 | 建议 |
|----------|------|------|------|
| 90-100分 | 🏆 A+ | 顶尖项目 | 强烈推荐，重点关注 |
| 80-89分 | 🥇 A | 优秀项目 | 推荐，可进入下一轮 |
| 70-79分 | 🥈 B+ | 良好项目 | 可以考虑 |
| 60-69分 | 🥉 B | 一般项目 | 需要改进 |
| 50-59分 | 📋 C | 较差项目 | 不推荐 |
| 50分以下 | ❌ D | 差项目 | 淘汰 |
