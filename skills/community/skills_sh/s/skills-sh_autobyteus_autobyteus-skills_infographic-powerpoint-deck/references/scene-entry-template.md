# Scene entry template

Use this template when adding new rows to `scene-catalog.md`.

```text
| `<scene-id>` | `<short scene summary>` | `<tag1,tag2,tag3,...>` |
```

Guidelines:
- `scene-id`: lowercase + hyphen only (example: `urban-night-bridge`).
- Summary: one short sentence focused on composition and mood.
- Tags: 3–6 searchable tags (style, mood, domain, lighting).
- Keep scenes non-graphic and presentation-safe.
