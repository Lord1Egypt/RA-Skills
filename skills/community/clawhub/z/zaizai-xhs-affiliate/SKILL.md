```
选品推荐 → 文案创作 → 审核润色 → 图片生成 → 人工发布 → 挂载商品 → 赚取佣金
```

### 步骤2：文案创作 ✍️
```
输入：商品信息、目标人群
输出：标题×3、正文、标签×10
⚠️ 需主人审核润色
```

#### 文案生成提示词
```
【种草文案生成提示词】

## 商品信息
- 商品名称：[名称]
- 核心卖点：[卖点1、2、3]
- 使用场景：[场景描述]
- 价格区间：[价格]
- 佣金比例：[X]%

## 目标人群
[人群描述，如：25-35岁职场女性、注重生活品质的宝妈等]

## 文案要求
1. 标题（3个备选）
   - 风格：真实感、亲近感，避免过度营销
   - 结构：痛点型/测评型/好物推荐型
   - 字数：15-25字

2. 正文结构
   - 开头：痛点引入（引起共鸣）
   - 中间：产品介绍+使用体验
   - 结尾：行动引导+购买链接

3. 标签（10个）
   - 包含品类词、人群词、功效词、热词

## 风格要求
- 语言自然流畅，像真人分享
- 避免"姐妹们冲"、"绝绝子"等过度营销词
- 融入真实使用感受
- 可加入个人小故事或经历

输出格式：
【标题】
1. xxx
2. xxx
3. xxx

【正文】
[完整正文内容]

【标签】
#xxx #xxx #xxx...

⚠️ 注意：以上内容为AI生成初稿，请主人审核润色后发布
```

### 步骤3：图片生成 🖼️
```
输入：商品类型、风格偏好
输出：封面图×1、正文卡片图×3
⚠️ 一次生成一张，避免崩溃
```

#### 图片生成提示词

##### 封面图
```
封面图风格：治愈系/高级感/真实感
画面内容：[商品+使用场景]
尺寸：3:4（小红书标准）
要求：不要任何水印、不要任何AI生成字样
```

##### 正文卡片图
```
卡片风格：简洁清晰、突出卖点
内容：产品展示+核心卖点文字
配色：与封面风格统一
尺寸：3:4竖版
```

---

### 步骤4：内容审核 ✅
```
输入：待审核文案+图片
输出：审核报告+修改建议
```

#### 审核清单
| 检查项 | 合格标准 | 不合格处理 |
|--------|----------|------------|
| AI味检查 | 读起来自然流畅 | 重新润色 |
| 违禁词检查 | 无平台禁用词 | 替换敏感词 |
| 图片质量 | 清晰无水印 | 重新生成 |
| 合规性检查 | 正规商品+真实宣传 | 下架不合规商品 |
| 版权检查 | 无侵权内容 | 删除侵权部分 |

#### 审核提示词
```
【内容审核提示词】

请审核以下小红书种草内容：

【文案内容】
[待审核文案]

【图片描述】
[图片内容描述]

请检查：
1. AI味检查：读起来是否自然？是否有明显AI痕迹？
2. 违禁词检查：是否包含"最"、"第一"、"绝对"等极限词？
3. 合规性检查：是否存在虚假宣传？
4. 敏感话题检查：是否涉及医疗、金融等敏感领域？

输出格式：
【审核结果】
✅ 通过 / ❌ 不通过

【问题列表】
- [问题1]
- [问题2]

【修改建议】
- [建议1]
- [建议2]

【最终版本】
[修改后的文案]
```

## 五、风险控制

### 5.1 平台红线 ❌
- 禁止纯AI自动化发布
- 禁止虚假宣传/夸大功效

- 禁止医疗/金融敏感推广
- 禁止出现"AI生成"字样

### 5.2 合规要求 ✅
- 真人审核润色所有内容
- 使用真实购买体验
- 注明利益关系（如"品牌合作"）
- 保留推广证据
