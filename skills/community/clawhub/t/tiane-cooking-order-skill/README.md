# 天鹅到家 · 做饭钟点工下单助手

跟助手说一句话，就能预约阿姨上门做饭。从说出需求到支付完成，全程在对话里搞定。

## 能帮你做什么

**一句话下单。** 告诉助手你想在哪里、什么时候需要阿姨上门做饭，它会自动匹配地址、查询可约时段、生成订单，你确认后即可支付。

**听得懂你的话。** 你不需要按固定格式说话。说"明天中午"它知道是 11:30，说"3个人吃饭"它知道安排 3 小时服务，说"少油少盐"它会自动备注给阿姨。

**不啰嗦。** 能从你的话里推断出来的信息不会再问，规格默认 3 小时不需要你选，只有真正缺少的信息才会问你一次。

## 目前支持的服务

当前助手支持**上门做饭钟点工**的预约下单。日常保洁、家电清洗等更多品类正在开发中，敬请期待。

## 关于阿姨匹配

目前助手会根据你的服务地址和时间，由系统自动为你匹配合适的做饭阿姨。如果你对阿姨的服务满意，后续可以通过购买服务套餐把她固定下来，每次都由她为你服务。

## 怎么用

**说出你的需求就行。** 比如：

- "我想找个做饭阿姨"
- "明天中午帮我找个做饭的到望京花园3号楼"
- "后天晚上，4个人吃饭，喜欢川菜，少辣"

助手会根据你提供的信息量自动调整流程：信息全就直接出订单预览，信息少就一步步引导你补充。

**首次使用需要授权。** 助手会发一个授权链接，点击完成授权后回复"已授权"即可继续。授权是为了获取你的服务地址和下单能力。

**确认后才会下单。** 助手会先展示一张订单预览卡片（包含地址、时间、规格、费用），你说"确认"之后才会生成订单。不确认就不会产生任何费用。确认前你可以随时修改地址、时间、规格；确认支付后如需修改，需要取消订单重新下单。

## 常见问题

**包含买菜吗？** 做饭服务本身不含买菜。如果你需要阿姨代买菜也可以，买菜时间会计入服务时间，做菜时间会相应减少。

**能照顾老人孩子吗？** 做饭钟点工专注于上门烹饪。如果你需要照顾老人或孩子的服务，可以在天鹅到家 App 里选择对应的服务品类。

**下单后能改时间或地址吗？** 需要取消订单后重新下单。阿姨接单后取消会扣违约金（12 小时外取消全退，2-12 小时退 80%，2 小时内退 60%）。

**下单后多久要支付？** 订单生成后需要在 10 分钟内完成支付，超时订单会自动取消。

**阿姨什么时候上门？** 阿姨会在你预约的服务时间准时上门。下单时需要提前至少 2 小时预约。
