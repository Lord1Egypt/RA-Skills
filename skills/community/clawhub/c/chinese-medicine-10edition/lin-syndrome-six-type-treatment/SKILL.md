# Lin Syndrome Six-Type Treatment

## When to Use
- 已明确诊断为淋证
- 完成虚实辨别
- 具备典型证候表现（如小便灼热刺痛、尿中夹砂石、尿色红赤、少腹胀痛、小便乳白如米泔、形瘦乏力等）

## Execution Steps
1. **辨明淋证六型**：依据主症特征判断属于热淋、石淋、血淋、气淋、膏淋或劳淋。
2. **判断虚实属性**：每型需进一步区分实证、虚证或虚实夹杂。
3. **确定治法与方药**：遵循“实则清利，虚则补益；虚实夹杂者通补兼施，分清标本主次”的基本治则，选用对应方剂及加减原则。

### 各型辨治要点
- **热淋**（纯实证）：小便灼热刺痛、尿黄赤、舌红苔黄腻、脉滑数 → 清热利湿，通淋止痛 → 八正散加减；若热毒弥漫三焦，合用黄连解毒汤与五味消毒饮。
- **石淋**（实证为主）：尿中夹砂石、腰腹绞痛、尿道刺痛 → 清热利湿，通淋排石 → 石韦散加减。
- **血淋**：
  - 实证：尿赤涩痛剧、夹血块、心烦不寐 → 清热通淋、凉血止血 → 小蓟饮子加减。
  - 虚证：尿淡红、涩痛轻、腰膝酸软 → 滋阴清热、补虚止血 → 肾阴虚用知柏地黄丸，脾虚气陷用补中益气汤加金钱草等。
- **气淋**：
  - 实证：郁怒后小便涩滞、少腹胀痛、脉沉弦 → 理气疏导 → 沉香散加减。
  - 虚证：少腹坠胀、尿有余沥、面白脉虚 → 补中益气 → 补中益气汤加减。
- **膏淋**：
  - 实证：小便乳白如米泔、上有浮油、尿道热痛 → 清利湿热、分清泌浊 → 程氏萆薢分清饮加减。
  - 虚证：淋出如脂、形瘦乏力、腰膝酸软 → 补脾益肾、固摄 → 膏淋汤加减。
- **劳淋**（多属虚证）：
  - 脾肾阳虚：面白少气、手足不温、脉微弱 → 补益脾肾 → 肾气丸合补中益气汤加减。
  - 肾阴不足：潮热盗汗、五心烦热、脉细数 → 滋阴清热 → 知柏地黄丸加减。

## Precautions
- 不适用于非淋证的尿路感染或结石。
- 必须排除肿瘤、结核等器质性疾病后再应用本辨治规则。
- 输出应包含具体证型名称、治法、推荐方剂及加减原则，用于临床处方参考。
