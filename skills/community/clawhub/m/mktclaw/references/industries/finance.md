## 5. 合规与监管

### 5.1 金融广告专属限制（**重点**）

| 限制类型 | 禁用 | 法规依据 |
|---------|------|---------|
| 收益承诺 | "保本""稳赚""零风险""高额回报" | 《广告法》+ 金融监管 |
| 风险提示 | 所有金融产品广告必须含风险提示 | 《金融营销宣传管理办法》 |
| 数据引用 | 收益率必须标注"历史业绩不代表未来" | 《基金法》 |
| 比较广告 | 不得贬低其他金融机构 | 《广告法》 |
| 误导宣传 | "唯一""最佳""最高收益" | 《广告法》 |

### 5.2 行业特殊资质

| 业务类型 | 必备资质 |
|---------|---------|
| 银行/信用卡 | 银保监会金融许可证 |
| 保险 | 银保监会保险业务许可证 |
| 基金 | 证监会基金销售资格 |
| 消费金融 | 银保监会消费金融牌照 |
| 互联网保险 | 互联网保险业务许可 |

### 5.3 平台审核红线

| 平台 | 红线 |
|------|------|
| 抖音 | 收益承诺、无风险提示、误导宣传 |
| 小红书 | 虚假"理财达人"、收益承诺 |
| 微信 | 多级分销、非法集资暗示 |
| 全平台 | 无资质机构发布金融广告 |

### 5.4 特殊合规要求

| 要求 | 说明 |
|------|------|
| 风险提示 | "投资有风险，理财需谨慎"（基金/保险） |
| 收益标注 | 必须标注"历史业绩不代表未来" |
| 利率标注 | 必须为年化利率（APR），不得用日息/月息 |
| 资质展示 | 广告中必须展示金融许可证编号 |

| 雷区 | 严重度 | 规避方法 |

| 无资质机构发布广告 | 🔴 致命 | 验证资质 |
