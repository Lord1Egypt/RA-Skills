# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Initial Release

### Added
- `prompts/`: Core prompt files (`intake.md`, `content_reader.md`, `toutiao_rewrite.md`, `title_generator.md`, `cover_prompt_generator.md`, `micro_post_generator.md`, `keywords_generator.md`, `quality_check.md`) for generating Toutiao-friendly content.
- `automation/`: Playwright automation documentation, draft flow, and strict safety rules ensuring human-in-the-loop.
- `templates/`: Markdown templates for articles, summaries, micro-posts, and checklists.
- `examples/`: Sample input files (`brand_profile.md`, `website_faq.md`, etc.) and output artifacts for a sample brand (PowerMatrix).
- `scripts/draft_to_toutiao.example.py`: Local Playwright script for assisting with Toutiao draft entry without automatic publishing.
- `SKILL.md`: Core skill definitions, boundaries, and methodology.
- `README.md`: User-facing documentation and usage instructions.
