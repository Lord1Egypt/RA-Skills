# 2.5 样式匹配标准（按元素重要性分级）

| 标准 | 适用场景 | 必须匹配的属性 |
|------|---------|--------------|
| **标准1: 完整匹配** | 列表项、卡片、主容器 | 盒模型 + 尺寸 + 布局 + 文字 + 边框 + 视觉效果 + 交互状态 |
| **标准2: 简化匹配** | 筛选项、按钮、标签 | 尺寸 + 布局 + 文字 + 边框 + 视觉效果 + 交互状态 |
| **标准3: 文字匹配** | 标题、描述文本 | font-family + font-size + color + font-weight + line-height |
| **标准4: 尺寸匹配** | 图片、图标 | width + height + border + border-radius + box-shadow |
