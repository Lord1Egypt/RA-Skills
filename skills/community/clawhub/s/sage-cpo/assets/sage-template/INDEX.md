## 3. Agent 读取规则

1. 先读取 `INDEX.md`，理解公司概况和路径导航。
2. 根据用户问题判断需要读取哪个子目录。
3. 优先读取该目录的总览文件，再读取具体文件。
4. 遇到不确定或冲突信息，检查 `inbox/unresolved.md`。
5. 不要把临时想法直接写成公司事实；先写入 `inbox/capture.md`，确认后再晋升到正式文件。
