# 盗墓小说创作主编 · 天下霸唱

> 全流程自动化盗墓小说创作工具——从大纲碰撞到排版交付，一步到位。

天下霸唱（张牧野）统筹四大名家（南派三叔、蔡骏、贰十三、那多）碰撞大纲，写手执笔正文，专属检查官把关质量，最终交付 `.txt` 成品。

## 使用方式

只需告诉主编你的想法，剩下的全自动：

```
# 无脑启动
帮我写一本盗墓小说

# 指定方向
写一个以唐代古墓为核心的探险故事

# 续写已有作品
我之前写了个开头，你接着往下写
```

**主编全程不问任何问题**，自己决定创作方向、拍板大纲、调度写手，最终交付成品。

## 创作流程

```
用户发起 → 断点检测 → 大纲产出 → 正文撰写 → 名家审读 → 检查修改 → 排版交付
```

| 阶段 | 步骤 | 产出 |
|------|------|------|
| 断点检测 | 扫描已有文件，确定续写起点 | 直接跳过已完成的卷 |
| 第一阶段 | 四大名家 3 轮碰撞 → 主编整合大纲 → 主编自审 | `05_大纲.md` |
| 第二阶段 | 写手逐卷正文 → 名家审读 → 检查修改循环 | 逐卷正文 + 修改版 |
| 第三阶段 | 统一排版 → 生成目录 → 输出 txt | 最终 `.txt` 成品 |

## 核心机制

- **去AI化写作** — 禁用 AI 句式、强化对话辨识度、五感平衡（尤其声音和气味的恐怖渲染）、句子长短落差、写三分留七分
- **断点续写** — 自动检测已有大纲/卷章，无缝衔接已有内容
- **名家审读** — 每卷随机选 2 位名家审读，共识问题优先处理
- **修改循环** — 检查 → 修改 → 复查，至少 2 轮通过才放行
- **全程自主** — 不向用户提问，主编自行完成所有创作决策

## 盗墓小说创作原则

- **玄而不虚，吓而不假** — 超自然元素要有民间传说的根，恐怖场面要有现实质感
- **真实感是魂** — 风水有出处、历史有依据、民俗有来路。读者信了，才怕
- **写三分留七分** — 恐怖留白比写满更吓人，不替读者解释和总结
- **呼吸感** — 恐怖不能全程高压，紧张中要穿插日常感
