# 任务类型默认映射表

> **加载时机**：步骤 2.5 首次执行时加载；后续可缓存

---

## 使用规则

- 用户任务文本未能提供某 5W2H 维度时，使用本表填入默认值
- 用户明确提供的维度 → 覆盖默认值，约束强度按约束词判断
- 默认值标注 `[推断]`，最终输出时保留此标注
- 本表仅为启发式参考，模型可根据上下文调整默认值

---

## 任务类型 1：制作 PPT

| 维度 | 默认值 | 说明 |
|------|--------|------|
| Why | 有效传达信息 | 核心动机 |
| What | 结构清晰、视觉简洁 | 交付物标准 |
| Who | 目标听众（从上下文推断或设为"决策者"） | 受众 |
| When | 截止时间（若未给，设为"尽快"） | 时间约束 |
| Where | 会议/线上 | 场景 |
| How | 使用常用演示工具 | 方法 |
| How much | 页数10-20页、说服力评分 | 量化指标 |
| 阻力 | 内容组织困难、时间不足 | 常见障碍 |
