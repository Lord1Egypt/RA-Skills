# 编排前自检表

## 1. 是否应启用本 Skill

- [ ] 这是正式产出，而非随手草稿
- [ ] 需要交付给用户 / 学员 / 外部渠道
- [ ] 涉及 PPT、课程、漫画、文章、行业方案、代码、视觉等高质量产出
- [ ] 需要审核记录和交付包

任意一项为是，即应启用本 Skill。

## 2. 是否需要 SubAgent

- [ ] 正式 PPT / Deck：media-director / PPT 专职
- [ ] 漫画 / 视觉 / 生图：media-director
- [ ] 公众号正文 / 长文：writer-pro / 写作专职
- [ ] 代码 / 系统修改：code-engineer
- [ ] 审核：独立 reviewer / critic 或双视角审核

main 不得因流程麻烦而绕过专业 SubAgent。

## 3. 是否具备必要依赖

- [ ] 目标用户明确
- [ ] 使用场景明确
- [ ] 输出清单明确
- [ ] 必需素材路径明确
- [ ] 缺失项已登记
- [ ] 样章 / 样页门禁已写入计划

## 4. 是否设置停止条件

- [ ] 缺真实素材不伪造
- [ ] 双审不通过不放行
- [ ] 最多返工轮数明确
- [ ] BLOCKED 条件明确
- [ ] 外部发送 / 发布前需确认或走唯一渠道
