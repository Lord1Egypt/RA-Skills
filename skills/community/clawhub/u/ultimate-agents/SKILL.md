# Ultimate AI Agent (DeFi + dApp)

## Deskripsi
Agent otomatis berbasis AI yang dapat melakukan transaksi blockchain dan interaksi dengan dApp.

## Fitur
- Multi wallet
- Multi RPC
- Auto transaction
- Smart swap (DEX)
- Adaptive learning system

## Cara Pakai
1. Isi file .env
2. Jalankan:
   node ultimate-agent.js

## Catatan
Gunakan testnet terlebih dahulu.
