## Known Risks and Mitigations: <br>
Risk: The skill has broad activation language and may steer many frontend requests toward the Next.js, Tailwind CSS, and shadcn/ui stack. <br>
Mitigation: Confirm the desired framework and design-system constraints before applying generated implementation guidance. <br>
Risk: Generated dependency or setup commands could affect the project workspace. <br>
Mitigation: Review commands before execution and run them only in the intended project environment. <br>
Risk: The server evidence includes unrelated crypto and purchase capability tags for a markdown-only frontend design skill. <br>
Mitigation: Treat those capability tags as publisher metadata to correct and rely on the security verdict for behavior assessment. <br>
