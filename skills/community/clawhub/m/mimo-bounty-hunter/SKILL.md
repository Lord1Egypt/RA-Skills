# 赏金猎人 💰

## 核心流程（4 Steps）

### Step 1: 能力评估
- 评估用户技能栈：编程开发/设计/翻译/写作/测试/产品/运营/视频
- 确定可接任务类型（按技能匹配）
- 评估时薪预期（新手/进阶/专家三级）
- 风险偏好：稳定小单/高风险大单

### Step 2: 任务扫描
**外包平台**：
- 国际：Upwork / Fiverr / Freelancer / Toptal / Guru / PeoplePerHour / SimplyHired
- 国内：猪八戒 / 码市 / 一品威客 / 人人都是产品经理兼职 / 电鸭社区

**翻译平台**：
- 国际：Gengo / OneHourTranslation / ProZ / TranslatorsCafe / TextMaster
- 国内：有道人工翻译 / 译客 / 我译网 / 做到网

**漏洞赏金**：
- 国际：HackerOne / Bugcrowd / Synack / Intigriti / OpenBugBounty / Cobalt.io
- 国内：补天 / 漏洞盒子 / CVE平台 / 360众测 / Wuyun

### Step 3: 任务筛选
- 按匹配度/报酬/截止日期排序
- 过滤条件：
  - ❌ 低信誉雇主（评分<4/无支付历史）
  - ❌ 不合理报价（低于预期50%）
  - ❌ 版权风险（侵权素材要求）
  - ❌ 模糊需求（无法评估工作量）
- 匹配度计算：技能匹配度×平台可靠性×报酬合理性

### Step 4: 执行指导
- 投标/接单模板（定制化）
- 交付物检查清单
- 争议处理建议
- 收款安全建议

## 边界约束
1. **合规优先**：不推荐任何灰色/违法项目（刷单/套现/侵权）
2. **汇率波动**：国际任务金额为估算，需核实当日汇率
3. **平台规则**：遵守各平台服务条款，注意平台抽成
4. **收款安全**：优先选择平台托管，避免私下转账
5. **个人信息**：投标时注意保护个人隐私信息
