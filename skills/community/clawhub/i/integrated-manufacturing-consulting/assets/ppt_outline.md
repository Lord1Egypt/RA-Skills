## PPT制作要点

### 设计原则
- **一页一个观点**，不要在一页塞太多信息
- **用图表说话**，减少大段文字
- **关键数据突出**，用红/绿色标注好坏
- **照片胜过千言**，现场照片是最有力的证据

### 配色建议
- 主色：深蓝（专业感）
- 强调色：红色（问题/痛点）、绿色（改善/效益）
- 背景：白色/浅灰色

### 每页结构
```
标题（一句话概括本页核心观点）
  ┌──────────────────────────────────────┐
  │  正文内容                            │
  │  · 关键发现                          │
  │  · 数据支撑                          │
  │  · 图表/照片                         │
  │                                      │
  └──────────────────────────────────────┘
  底部：页脚（公司名/项目名/页码）
```

### 汇报节奏建议
| 部分 | 时间分配 | 重点 |
|------|:--------:|------|
| 开场+背景 | 5分钟 | 建立信任 |
| 现状+诊断 | 25分钟 | 核心内容，用数据说服 |
| 改善规划 | 15分钟 | 展示专业能力和价值 |
| 合作建议 | 10分钟 | 推动下一步行动 |
| QA | 5分钟 | 答疑解惑 |
| **总计** | **60分钟** | |
