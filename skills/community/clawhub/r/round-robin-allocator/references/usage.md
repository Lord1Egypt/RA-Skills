# 使用指南

> 本文件为 round-robin-allocator 的详细使用说明，配合 SKILL.md 渐进式加载。

## 命令行使用

```bash
# 一行描述，自动解析
python scripts/main.py --input "33个项目，4个月，5套方案，比例7:8:10:3:5"

# 指定后处理模式
python scripts/main.py --input "..." --mode fair
python scripts/main.py --input "..." --mode random --seed 42
python scripts/main.py --input "..." --mode custom --repeat-ratio "4:2:7:1"

# 配置管理
python scripts/main.py --input "..." --no-confirm   # 跳过本次确认
python scripts/main.py --input "..." --always        # 永久跳过确认
python scripts/main.py --input "..." --set-default-mode fair  # 配置默认后处理
```

## 三要素规则（NTK）

| 参数 | 含义 | 是否必须 | 默认值 |
|------|------|---------|--------|
| T | 周期/轮次数 | **必须** | — |
| N | 对象数（项目/学生/用户…） | **必须** | — |
| K | 选项数（方案/策略/类型…） | **必须** | — |
| ratios | 各选项分配比例 | 可选 | 不指定则默认等分 |
| --mode | 后处理模式 | 可选 | 默认 algorithm |

**重要**：N/K/T 不会自动推断或编造。缺任何一个都无法执行，必须通过菜单选项 9 补充。

## 确认表

```
  ┌──────────┬────────────────────────────────────────┐
  │ 参数     │ 值                                    │
  ├──────────┼────────────────────────────────────────┤
  │ N (对象) │ 33                                    │
  │ T (轮次) │ 4                                     │
  │ K (方案) │ 5                                     │
  │ 比例     │ 7:8:10:3:5                            │
  │ 后处理   │ 4-ID排序                               │
  ├──────────┴────────────────────────────────────────┤
  │ 1=确认执行  2=取消  3=永久跳过确认         │
  │ 后处理: 4=ID排序  5=随机打乱  6=均匀分布    │
  │         7=自定义                            │
  │ 8=设默认后处理  9=补充更新参数              │
  └─────────────────────────────────────────────┘
```

| 按键 | 功能 |
|------|------|
| **1** | 确认当前参数并执行 |
| **2** | 取消本次操作 |
| **3** | 永久跳过确认 |
| **4/5/6/7** | 切换后处理并立即执行 |
| **8** | 设默认后处理（二级菜单） |
| **9** | 补充/更新参数（自然语言输入） |

### 缺参数时

```
  │ K (方案) │ ? (缺) 按9补充                            │
  │ 比例     │ ? (缺)（未指定，默认等分）按9补充           │
```

### 按9补充参数

```
  请输入选项 [1]: 9
  ════════════════════════════════════════
  当前参数：N=33, T=4
  缺失/可补充：K(方案数), 比例(默认等分)
  ════════════════════════════════════════
  请输入补充/更新内容：5套方案，比例7:8:10:3:5
  ℹ️  已更新参数
```

支持：`5套方案`（K=5，比例默认等分）、`5套方案，比例7:8:10:3:5`、`比例1:2:1`、`N=50` 等自然语言输入。

## 后处理模式

| 模式 | 菜单 | 命令行 | 说明 |
|------|------|--------|------|
| algorithm | 4=ID排序 | 默认 | ID 顺序优先，可复现 |
| random | 5=随机打乱 | `--mode random --seed N` | 随机打乱，消除 ID 偏见 |
| fair | 6=均匀分布 | `--mode fair` | 重复均匀分摊到各月 |
| custom | 7=自定义 | `--mode custom --repeat-ratio "4:2:7:1"` | 指定月间重复比例 |

## 配置系统

配置文件在 `.standardization/round-robin-allocator/data/_allocator_config.json`

```json
{"skip_confirm": false, "default_mode": "algorithm"}
```

| 字段 | 说明 | 更新方式 |
|------|------|---------|
| skip_confirm | 是否跳过确认表 | 菜单按 3 或 `--always` |
| default_mode | 默认后处理模式 | 菜单按 8 或 `--set-default-mode` |

## 输出文件

| 文件 | 内容 |
|------|------|
| allocation_result.md | 分配明细表 + 轮次分布统计 |
| allocation_result.csv | 仅在询问时按 y 才导出 |
| allocation_result.html | 热力表格 + 分布图表 + 3D 散点图 |
