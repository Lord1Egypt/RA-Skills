# Listing Quality Checklist

Run before finalizing any listing. Every ✅ must be checked before delivery.

## Title (8 checks)
- [ ] Primary keyword appears in the first 5 words
- [ ] Character count within platform limit (Amazon 200, Shopify 70 SEO, TikTok 34 visible)
- [ ] Title case applied (Amazon); sentence case (Shopify/TikTok)
- [ ] No ALL CAPS words (Amazon — abbreviations only)
- [ ] No prohibited phrases: "best", "#1", "cheapest", "free shipping"
- [ ] Brand name included (first position for Amazon)
- [ ] Key variant (size, color, pack count) included if applicable
- [ ] Renders fully on mobile 375px screen without critical truncation

## Bullet Points / Feature Highlights (10 checks)
- [ ] 5 bullets present (Amazon); 4–6 highlights (Shopify/TikTok)
- [ ] Each bullet opens with a benefit, not a feature
- [ ] Each bullet follows BENEFIT → FEATURE → PROOF structure
- [ ] ALL CAPS hook phrase on each bullet (Amazon style)
- [ ] No period at end of bullets (Amazon)
- [ ] No HTML tags (Amazon bullets)
- [ ] Under 200 visible characters for above-the-fold display
- [ ] Each bullet covers a distinct benefit (no overlap)
- [ ] At least 2 mid-tail keywords placed naturally across bullets
- [ ] No competitor names mentioned

## Description / A+ Content (8 checks)
- [ ] Primary keyword in opening sentence
- [ ] At least 2 secondary keywords placed in first 100 words
- [ ] Comparison table present (A+ or Shopify — 5+ criteria)
- [ ] FAQ module present with 4+ real buyer questions
- [ ] Social proof element included (rating, testimonial, or award)
- [ ] Under 2,000 characters (plain description) or within A+ module limits
- [ ] Readability grade 7 or lower (use Hemingway App)
- [ ] No duplicate content from competitor pages

## Backend Keywords — Amazon Only (6 checks)
- [ ] No words repeated from title
- [ ] No competitor brand names
- [ ] Byte limit maximized (target: 240+ of 250 bytes)
- [ ] Words separated by spaces, no commas or punctuation
- [ ] Includes common misspellings / regional spelling variants
- [ ] Seasonal modifiers added where appropriate

## Compliance (7 checks)
- [ ] No medical claims (treat, cure, diagnose) without substantiation
- [ ] Health claims use compliant language ("supports", "helps maintain")
- [ ] All certifications mentioned (FCC, CE, CARB, NSF) are documented and accurate
- [ ] No "guaranteed results" language for supplements or beauty
- [ ] No pesticide efficacy claims without EPA registration number
- [ ] Images described in alt-text match listing claims
- [ ] If hazardous material (batteries, chemicals) — SDS uploaded to seller central

## Keyword Coverage (3 checks)
- [ ] Primary keyword: present in title + at least 2 bullets
- [ ] Secondary keywords: 3+ distributed across bullets and description
- [ ] Long-tail terms: covered in backend keywords or FAQ section

## Final Review (4 checks)
- [ ] Listing previewed on mobile device (or mobile emulator)
- [ ] Spell-checked in target market language (US English vs UK English)
- [ ] Brand voice consistent with brand guidelines
- [ ] Client / team sign-off completed before submission
