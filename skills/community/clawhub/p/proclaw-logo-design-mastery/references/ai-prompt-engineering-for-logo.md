## 概览

AI 提示词工程（Prompt Engineering）是 AI 时代 Logo 设计的核心技能。精准的提示词能够引导 AI 生成高质量的 Logo 设计方案，模糊的提示词则会导致低质量、不可用的输出。

本章提供系统化的 AI Logo 提示词工程方法论，从结构化框架到关键词库，从模板库到实战案例，帮助设计师掌握顶级 AI 提示词工程能力。

**核心能力目标**：
- ✅ 能够编写结构化、精准的 Logo 设计提示词
- ✅ 能够根据品牌特征生成定制化提示词
- ✅ 能够评估和优化 AI 输出质量
- ✅ 能够通过迭代提升 AI 生成质量

**适用 AI 工具**：
- Midjourney v6
- DALL-E 3
- Stable Diffusion
- Adobe Firefly
- 其他图像生成 AI

## Logo 设计提示词的结构化框架

### 八个核心要素

高质量的 Logo 设计提示词包含 8 个核心要素，每个要素都有明确的参数和选项。

**要素 1：主体（Subject）**

**定义**：Logo 的主要视觉元素

**参数选项**：
- 字母组合：首字母、品牌名称、缩写
- 几何形状：圆形、三角形、正方形、六边形
- 自然元素：动物、植物、天体、自然现象
- 抽象符号：线条、形状、组合
- 具象物体：产品、工具、场景

**示例**：
```
主体：字母组合 "A" 和 "M"
主体：抽象几何形状
主体：猫的简化图形
```

**要素 2：风格（Style）**

**定义**：Logo 的设计风格

**参数选项**：
- 极简主义：简洁、纯粹、留白
- 几何抽象：几何化、数学化、精确
- 负空间：双重意义、隐藏元素
- 线性设计：线条、轮廓、简洁
- 字母组合：字母、排版、字体
- 象征符号：符号化、隐喻、象征
- 3D/立体：立体、深度、材质
- 渐变/光效：渐变、光效、现代

**示例**：
```
风格：极简主义
风格：几何抽象
风格：负空间设计
```

**要素 3：色彩（Color）**

**定义**：Logo 的色彩方案

**参数选项**：
- 单色：单色、黑白、高对比
- 双色：双色、对比色、互补色
- 三色：三色、主色+辅色+点缀色
- 渐变：渐变、多色渐变、渐变过渡
- 无色彩：黑白、灰色、无色

**示例**：
```
色彩：单色 - 蓝色
色彩：双色 - 蓝色和橙色
色彩：渐变 - 蓝色到紫色
```

**要素 4：构图（Composition）**

**定义**：Logo 的构图方式

**参数选项**：
- 中心构图：中心、聚焦、稳定
- 对称构图：对称、平衡、秩序
- 不对称构图：不对称、动感、创新
- 放射构图：放射、扩散、能量
- 层叠构图：层叠、深度、层次

**示例**：
```
构图：中心构图
构图：对称构图
构图：不对称构图
```

**要素 5：抽象度（Abstraction Level）**

**定义**：Logo 的抽象程度

**参数选项**：
- 等级 1（具象）： recognizable, detailed, realistic
- 等级 2（半具象）：semi-abstract, simplified, recognizable
- 等级 3（半抽象）：abstract, symbolic, stylized
- 等级 4（抽象）：highly abstract, minimal, symbolic
- 等级 5（符号化）：pure symbol, minimal, iconic

**示例**：
```
抽象度：等级 3（半抽象）
抽象度：等级 4（抽象）
抽象度：等级 5（符号化）
```

**要素 6：情感（Emotion）**

**定义**：Logo 传达的情感

**参数选项**：
- 温暖：温暖、友好、亲切
- 力量：力量、稳定、权威
- 创新：创新、未来、科技
- 信任：信任、可靠、专业
- 活力：活力、能量、热情
- 高端：高端、奢华、优雅

**示例**：
```
情感：信任、可靠、专业
情感：创新、未来、科技
情感：高端、奢华、优雅
```

**要素 7：技术（Technical）**

**定义**：Logo 的技术要求

**参数选项**：
- 分辨率：矢量、高分辨率、可缩放
- 格式：SVG、PNG、JPG
- 尺寸：各种尺寸、小尺寸适用
- 背景：透明背景、白色背景、无背景

**示例**：
```
技术：矢量格式，高分辨率
技术：透明背景，小尺寸适用
技术：SVG 格式，可缩放
```

**要素 8：约束（Constraints）**

**定义**：Logo 的设计约束

**参数选项**：
- 负面提示词：避免的元素
- 禁止元素：禁止的形状、符号
- 风格约束：避免的风格
- 色彩约束：避免的色彩

**示例**：
```
约束：避免复杂细节，避免渐变
约束：禁止使用常见符号，避免过度装饰
约束：避免 3D 效果，避免阴影
```

### 提示词的语法和逻辑结构

**基础语法**：

```
[主体] + [风格] + [色彩] + [构图] + [抽象度] + [情感] + [技术] + [约束]
```

**示例**：
```
字母组合 "A" 和 "M"，极简主义风格，单色蓝色，中心构图，
等级 3 半抽象，传达信任和可靠性，矢量格式透明背景，
避免复杂细节，避免渐变
```

**逻辑结构**：

**结构 1：优先级排序**
- 主元素 → 次元素 → 细节元素
- 核心特征 → 辅助特征 → 装饰特征

**结构 2：因果关系**
- 风格 → 抽象度 → 简洁程度
- 情感 → 色彩 → 色彩选择

**结构 3：对比关系**
- 正向提示词 → 负向提示词
- 核心元素 → 禁止元素

**高级语法**：

**语法 1：权重控制（Midjourney）**
```
元素1::2 元素2::1 元素3::0.5
```
- ::2：高权重
- ::1：正常权重
- ::0.5：低权重

**语法 2：参数控制（Midjourney）**
```
--ar 1:1（宽高比）
--q 2（质量）
--style raw（原始风格）
```

**语法 3：否定控制（DALL-E）**
```
主体，风格，色彩，避免元素1，避免元素2
```

### 提示词优化的五个步骤

**步骤 1：明确设计目标**

**目标清单**：
- ✅ 品牌类型：科技、金融、零售、服务等
- ✅ 品牌价值：信任、创新、品质、服务等
- ✅ 目标受众：年龄、性别、职业、地区
- ✅ 应用场景：数字、印刷、产品、环境

**输出成果**：
- 设计目标文档（1-2 页）
- 关键词清单（10-20 个关键词）
- 约束清单（5-10 个约束）

---

**步骤 2：选择核心要素**

**选择原则**：
- 主体：符合品牌类型和象征意义
- 风格：符合品牌调性和目标受众
- 色彩：符合品牌色彩和心理效应
- 构图：符合品牌定位和应用场景

**选择清单**：
- ✅ 主体：_______（字母/几何/自然/抽象/具象）
- ✅ 风格：_______（极简/几何/负空间/线性/组合）
- ✅ 色彩：_______（单色/双色/三色/渐变）
- ✅ 构图：_______（中心/对称/不对称/放射）
- ✅ 抽象度：_______（等级 1-5）
- ✅ 情感：_______（温暖/力量/创新/信任/活力/高端）

**输出成果**：
- 核心要素文档（1 页）
- 要素决策清单（每个要素的选择原因）

---

**步骤 3：编写初始提示词**

**编写模板**：
```
[主体]，[风格]风格，[色彩]，[构图]构图，
[抽象度]抽象度，传达[情感]情感，
[技术]要求，避免[约束]
```

**示例**：
```
字母组合 "T" 和 "E"，极简主义风格，单色深蓝色，
中心构图，等级 4 抽象，传达创新和科技感，
矢量格式透明背景，避免复杂细节，避免渐变，避免装饰
```

**输出成果**：
- 初始提示词（1-3 个版本）
- 提示词说明文档（解释每个要素的选择）

---

**步骤 4：测试和评估**

**测试方法**：
- 生成测试：AI 生成 5-10 个方案
- 识别性测试：3 秒内能否识别？
- 质量评估：是否符合设计目标？
- 创意评估：是否有创意和独特性？

**评估标准**：
- 识别性：> 85%（3 秒识别）
- 质量：> 80%（符合设计目标）
- 创意：> 75%（有创意和独特性）
- 一致性：> 85%（与品牌一致）

**输出成果**：
- 测试报告（2-3 页）
- 评估清单（每个方案的评分）

---

**步骤 5：迭代优化**

**优化方法**：
- 调整权重：调整要素权重
- 修改参数：修改风格、色彩、构图
- 添加约束：添加负面提示词
- 精细化：细化描述，增加细节

**优化示例**：
```
初始：字母组合 "T" 和 "E"，极简主义风格
优化：字母组合 "T" 和 "E"::2，极简主义风格::1.5，
深度深蓝色，几何抽象，等级 4 抽象，传达创新和科技感，
矢量格式透明背景，--ar 1:1 --q 2 --style raw，
避免复杂细节，避免渐变，避免装饰，避免常见符号
```

**输出成果**：
- 优化提示词（1-2 个版本）
- 对比报告（初始 vs 优化）

**负面关键词**：

**示例提示词**：

**核心关键词**：
- negative space, white space, hidden meaning
- dual meaning, visual pun, clever design
- space between, space within, hidden shapes

**技巧关键词**：
- gap recognition, internal recognition, boundary recognition
- layering, cutout, composition
- double meaning, visual metaphor, hidden symbol

**构图关键词**：
- space balance, positive-negative balance
- hidden elements, embedded shapes
- clever use of space

**色彩关键词**：
- high contrast, black and white, single color
- two-color, color blocking
- solid colors, no gradient

**细节关键词**：
- clean edges, sharp boundaries
- clear positive space, clear negative space
- hidden shapes, hidden letters, hidden symbols

**负面关键词**：
- cluttered, busy, no negative space
- no hidden meaning, no clever design
- no visual pun, no dual meaning

**示例提示词**：
```

矢量格式透明背景，利用字母间隙创造负空间箭头，
平衡正负空间比例，避免 cluttered，避免复杂细节，--ar 1:1 --q 2
```

**细节关键词**：
- clean lines, smooth curves, no fill
- no shadow, no depth, no texture
- no decoration, no pattern, no ornament

**负面关键词**：

**示例提示词**：

## 常见提示词错误与修正

### 错误 1：提示词过于笼统

**错误示例**：
```
设计一个 Logo
```

**问题**：
- 没有明确主体
- 没有明确风格
- 没有明确约束
- AI 输出随机性高

**修正示例**：
```
字母组合 "A" 和 "Z"，极简主义风格，单色深蓝色，
中心构图，等级 4 抽象，传达创新和科技感，
矢量格式透明背景，避免复杂细节，避免渐变，
--ar 1:1 --q 2
```

**改进点**：
- ✅ 明确主体：字母组合 "A" 和 "Z"
- ✅ 明确风格：极简主义风格
- ✅ 明确色彩：单色深蓝色
- ✅ 明确约束：避免复杂细节，避免渐变

---

### 错误 2：忽略负面提示词

**错误示例**：
```
字母组合 "A" 和 "Z"，极简主义风格，单色深蓝色
```

**问题**：
- 没有负面约束
- AI 可能生成复杂细节
- AI 可能添加渐变
- AI 可能添加装饰

**修正示例**：
```
字母组合 "A" 和 "Z"，极简主义风格，单色深蓝色，
中心构图，等级 4 抽象，传达创新和科技感，
矢量格式透明背景，避免复杂细节，避免渐变，
避免装饰，避免阴影，--ar 1:1 --q 2
```

**改进点**：
- ✅ 添加负面约束：避免复杂细节
- ✅ 添加负面约束：避免渐变
- ✅ 添加负面约束：避免装饰
- ✅ 添加负面约束：避免阴影

---

### 错误 3：缺乏具体参数

**错误示例**：
```
字母组合 "A" 和 "Z"，极简主义风格，蓝色
```

**问题**：
- 色彩不具体（蓝色 vs 深蓝色）
- 构图不明确
- 抽象度不明确
- 技术要求不明确

**修正示例**：
```
字母组合 "A" 和 "Z"，极简主义风格，单色深蓝色，
中心构图，等级 4 抽象，传达创新和科技感，
矢量格式透明背景，--ar 1:1 --q 2 --style raw
```

**改进点**：
- ✅ 色彩具体：单色深蓝色
- ✅ 构图明确：中心构图
- ✅ 抽象度明确：等级 4 抽象
- ✅ 技术参数：--ar 1:1 --q 2 --style raw

---

### 错误 4：风格不一致

**错误示例**：
```
字母组合 "A" 和 "Z"，极简主义风格，单色深蓝色，
不对称动态构图，等级 5 符号化，传达创新和科技感
```

**问题**：
- 极简主义风格 vs 不对称动态构图（矛盾）
- 极简主义风格 vs 等级 5 符号化（过度抽象）
- 风格不一致导致 AI 输出混乱

**修正示例**：
```
字母组合 "A" 和 "Z"，科技现代风格，单色深蓝色，
不对称动态构图，等级 4 抽象，传达创新和科技感，
矢量格式透明背景，--ar 1:1 --q 2
```

**改进点**：
- ✅ 风格一致：科技现代风格
- ✅ 构图匹配：不对称动态构图
- ✅ 抽象度匹配：等级 4 抽象

---

### 错误 5：忽略测试验证

**错误示例**：
```
生成一次，接受结果
```

**问题**：
- 没有测试验证
- 没有评估质量
- 没有迭代优化
- 可能接受低质量输出

**修正流程**：
```
1. 生成 5-10 个方案
2. 使用 10 维度评估清单评估
3. 识别问题和不足
4. 优化提示词
5. 重新生成
6. 对比初始和优化版本
7. 选择最佳方案或继续迭代
```

**改进点**：
- ✅ 多方案生成
- ✅ 系统化评估
- ✅ 迭代优化
- ✅ 对比选择
