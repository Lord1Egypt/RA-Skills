# Seasonal Plan Quality Checklist

Run before delivering any seasonal keyword plan.

## Inputs & scope
- [ ] 5–30 keywords captured and in scope.
- [ ] Product category identified to anchor the seasonal pattern.
- [ ] Target market(s) confirmed; hemisphere accounted for.
- [ ] Known launch/promo dates incorporated.

## Classification
- [ ] Every keyword tagged evergreen / seasonal / event-driven.
- [ ] Hybrid terms (evergreen + seasonal lift) identified.
- [ ] Evergreen terms are not assigned a false "season."

## Demand curve
- [ ] Peak month(s) stated for each seasonal/event term.
- [ ] Rising shoulder, falling shoulder, and trough described.
- [ ] Curves expressed in months, not vague quarters.
- [ ] Multi-peak terms (e.g. spring + fall) captured.

## Timing & windows
- [ ] Early-mover window identified for each seasonal term.
- [ ] Content lead time (4–6 weeks) respected.
- [ ] Listing re-index lead time (2–3 weeks) respected.
- [ ] Ad budget shifts precede peaks, not chase them.

## Evidence & rigor
- [ ] Each curve labeled as evidenced vs assumed.
- [ ] Assumption-based peaks flagged for validation with real data.
- [ ] Seller's own historical data used where available.

## Coordination
- [ ] Overlapping keywords coordinated to avoid cannibalization.
- [ ] Plan covers listings, ads, AND content per month.
- [ ] Trough months redirect effort to off-season terms.

## Output
- [ ] Classification table included.
- [ ] Full 12-month action plan, keyed to specific keywords.
- [ ] Early-mover summary a manager can act on immediately.
- [ ] Measurement metrics defined (rank, CPC/impression share, conversion).
- [ ] Recommendation to revisit the plan quarterly.
