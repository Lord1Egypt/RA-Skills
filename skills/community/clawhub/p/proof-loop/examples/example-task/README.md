# Example Task: UI Language Fix

This example shows what a completed Proof Loop task looks like inside a repository.

The task artifacts live under:

```text
.agent/tasks/ui-language-fix/
```

Read them in this order:

1. `spec.md` - frozen acceptance criteria and verification plan
2. `evidence.md` - what was changed and checked
3. `verdict.json` - structured verifier result
4. `problems.md` - empty because the final verifier pass found no open issues

The example is intentionally small. Its job is to prove the artifact shape, not to ship a real app.
