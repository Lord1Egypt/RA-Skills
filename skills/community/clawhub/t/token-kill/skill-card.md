## Use Case: <br>
OpenClaw users and developers use this skill to reduce token consumption by clearing or compressing context, moving repetitive polling and data-fetching work into scripts, and selecting lower-cost model tiers for simple tasks. <br>

### Deployment Geography for Use: <br>

## Known Risks and Mitigations: <br>
Risk: Memory-clearing or compression commands can remove useful context if used without review. <br>
Mitigation: Use context-clearing and compression commands deliberately, after confirming that important task details have been retained or saved elsewhere. <br>
Risk: Automation scripts for email, orders, APIs, or scheduled checks can affect external data or accounts if granted broad permissions. <br>
Mitigation: Keep script permissions narrow and review actions that change data before enabling unattended execution. <br>
