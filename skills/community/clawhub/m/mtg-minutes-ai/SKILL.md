# Meeting Minutes - 会议纪要整理

10 秒整理 1 小时会议内容。

## 功能

- 自动提取关键决策点
- 识别待办事项和负责人
- 生成结构化会议纪要
- 追踪行动项完成状态

## 使用方法

### 整理录音/文字
```
整理会议纪要：
<paste 录音转文字或会议记录>
```

### 生成特定格式
```
生成会议纪要，格式：<标准/简洁/详细>
会议类型：<周会/评审会/1on1/项目启动>
```

## 输出格式

### 标准模板
```
## 会议基本信息
- 时间：YYYY-MM-DD HH:MM
- 参会人：@姓名1 @姓名2
- 会议主题：XXX

## 关键决策
1. 决策1 → 决策人：@姓名
2. 决策2 → 决策人：@姓名

## 待办事项
| 事项 | 负责人 | 截止日期 | 优先级 |
|------|--------|----------|--------|
| XXX | @姓名 | YYYY-MM-DD | P0/P1/P2 |

## 讨论要点
- 要点1
- 要点2

## 下次会议
- 时间：
- 议题：
```

## 支持会议类型

- 周会/站会
- 需求评审会
- 技术评审会
- 项目启动会
- 1on1 面谈
- 复盘会议
