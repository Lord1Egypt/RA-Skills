# Return Cost Calculator

## Why Most Brands Undercount Return Costs

The common mistake: brands track return rate as "refunds ÷ revenue" and treat the refund amount as the return cost. This undercounts the true cost by 40–100%.

A $40 product returned doesn't cost you $40. It costs you:

1. The original outbound shipping ($6–14)
2. The return shipping label ($6–14)
3. Processing labor ($3–8)
4. Restocking cost if the product can be resold ($1–4)
5. Write-off cost if it cannot be resold (COGS of $12–25 for a $40 product)
6. Customer service time ($2–5)
7. Lost gross profit on the original sale (the margin you would have kept)

Total for a $40 product: $35–70 all-in, depending on restocking rate.

## ROI of Return Reduction Initiatives

Use this formula to evaluate whether a return-reduction investment is worth it:

```
Annual Saving = (Current return rate − Target return rate) × Annual orders × Cost per return

ROI = Annual Saving ÷ Investment Cost

Example:
- Current return rate: 22%
- Target return rate: 15% (−7 percentage points)
- Annual orders: 18,000
- Cost per return: $26
- Annual saving = 7% × 18,000 × $26 = $32,760
- Investment: $5,000 (listing photography + email sequence setup)
- ROI = $32,760 ÷ $5,000 = 6.6× return (655% ROI)
```

## Exchange vs. Refund Value

When a customer returns due to sizing, an exchange keeps the customer and the revenue. Calculate the value of exchanges vs. refunds:

| Scenario | Revenue Impact | Customer Retention |
|---------|----------------|-------------------|
| Full refund | −AOV − Return cost | Customer likely churned |
| Exchange (size swap) | −Return cost only | Customer retained |
| Exchange with $10 incentive | −Return cost − $10 | Customer retained + goodwill |

**Exchange conversion rate benchmarks:**
- Without incentive: 15–25% of sizing returners choose exchange
- With $5 incentive: 30–40% choose exchange
- With $10 credit (exchange only): 40–55% choose exchange
- With free return label for exchange only (vs. customer-paid for refunds): 35–50% choose exchange

At 1,000 apparel returns/month from sizing issues:
- Baseline (no exchange incentive): 200 exchanges, 800 refunds
- With $10 exchange incentive: 450 exchanges, 550 refunds → 250 additional exchanges
- Revenue retained: 250 × $52 AOV = $13,000/month additional revenue retained
- Cost: 250 × $10 credit = $2,500/month
- Net benefit: $10,500/month
