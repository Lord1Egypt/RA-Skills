## 一、核心概念

**Badhaka（बाधक）= 障碍、阻碍**

在吠陀占星中，Badhaka描述的是：**某些特定的宫位和其主宰星，会在人生中制造隐藏的障碍、延迟或困难**。

- **Badhaka Sthana（障碍宫）** — 制造障碍的宫位
- **Badhakesh（障碍星主）** — 障碍宫的主宰星

## 四、Badhaka 实战分析步骤

### Step 1: 确认上升星座类型
- 启动（Aries/Cancer/Libra/Capricorn）→ Badhaka = 11宫
- 固定（Taurus/Leo/Scorpio/Aquarius）→ Badhaka = 9宫
- 双元（Gemini/Virgo/Sagittarius/Pisces）→ Badhaka = 7宫

### Step 2: 确认 Badhakesh
- 找到 Badhaka 宫位的主宰星

### Step 3: 评估 Badhakesh 状态
- 庙旺/落陷/燃烧/逆行
- 与吉星/凶星的合相和相位
- 在D9中的状态

### Step 4: 评估影响范围
- Badhakesh 对哪些目标宫有相位/合相
- Badhaka 宫内有哪些行星

### Step 5: 整合到综合分析
- Badhaka 因素不独立断事，作为辅助判断
- 与其他Yoga/Dasha/Transit交叉验证

## 五、示例分析：Leo 上升

### 一楠星盘（Leo 上升 12.54°）

| 项目 | 数据 |
|------|------|
| 上升 | Leo（固定星座） |
| Badhaka宫 | **9宫 = Aries** |
| Badhakesh | **Mars** |
| Mars位置 | Cancer 1.32°（12宫，落陷+Neecha Bhanga） |
| Badhaka宫(Aries)内 | 无行星 |

### 分析
- **Mars = Badhakesh = DK = 女性日盘丈夫征象星 = Yogakaraka** → 四重身份叠加
- Mars落陷12宫 → 障碍因素隐藏在"远方/灵性/隐秘"领域
- Neecha Bhanga → 障碍可被化解（需努力）
- Badhaka宫Aries空宫 → 障碍更多来自Badhakesh本身而非宫内行星
- **婚姻领域的Badhaka影响**: Mars同时是Badhakesh和DK → 配偶关系天然带有"需要克服的障碍"属性
