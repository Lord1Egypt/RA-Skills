# Study Plan Templates by Goal Type

## Template 1: Professional Certification (8-12 weeks)

### Example: AWS Solutions Architect Associate

```
## 📚 Study Plan: AWS Solutions Architect Associate
**Goal**: Pass SAA-C03 exam | **Deadline**: 10 weeks | **Daily Time**: 2 hours | **Level**: Beginner

### Curriculum Overview (120 total hours)
Domain 1: Design Resilient Architectures — 30 hours (25%)
Domain 2: Design High-Performing Architectures — 24 hours (20%)
Domain 3: Design Secure Applications — 30 hours (25%)
Domain 4: Design Cost-Optimized Architectures — 18 hours (15%)
Domain 5: Practice Tests & Review — 18 hours (15%)

### Phase Breakdown
**Phase 1 (Weeks 1-4): Foundation**
- AWS fundamentals, core services (EC2, S3, VPC, IAM)
- Priority: breadth, service overview

**Phase 2 (Weeks 5-7): Deep Dive**
- Architecture patterns, best practices
- Priority: hands-on labs, real scenarios

**Phase 3 (Weeks 8-10): Exam Prep**
- Practice tests, weak area drilling
- Priority: exam strategy, time management

### Week 1 Schedule
Monday: AWS Overview + EC2 basics — 2h
Tuesday: S3 + CloudFront — 2h
Wednesday: Hands-on lab: Launch EC2 + S3 — 2h
Thursday: VPC fundamentals — 2h
Friday: IAM + Security basics — 2h
Saturday: Practice quiz (Domain 1) — 1h
Sunday: Review notes, flashcards — 1h

### Milestone Checklist
□ Week 2: Complete AWS Cloud Practitioner practice test (80%+)
□ Week 4: Build a 3-tier web app architecture (hands-on)
□ Week 6: Score 70%+ on first SAA practice test
□ Week 8: Score 80%+ on practice tests consistently
□ Week 10: Pass actual exam (720+ score)

### Recommended Resources
- Primary: Stephane Maarek's AWS SAA course (Udemy)
- Practice: Tutorials Dojo practice tests
- Hands-on: AWS Free Tier account
- Reference: AWS Well-Architected Framework docs
```

## Template 2: Programming Language (12-16 weeks)

### Example: Python for Data Science

```
## 📚 Study Plan: Python for Data Science
**Goal**: Build portfolio projects | **Deadline**: 16 weeks | **Daily Time**: 1.5 hours | **Level**: Programming beginner

### Curriculum Overview (168 total hours)
Foundation: Python basics — 40 hours (24%)
Data Manipulation: Pandas + NumPy — 35 hours (21%)
Visualization: Matplotlib + Seaborn — 25 hours (15%)
Analysis: Statistics + ML basics — 35 hours (21%)
Projects: 3 portfolio projects — 33 hours (19%)

### Phase Breakdown
**Phase 1 (Weeks 1-5): Python Foundation**
- Syntax, data types, control flow, functions
- Priority: coding fluency, problem-solving

**Phase 2 (Weeks 6-10): Data Tools**
- Pandas for data manipulation, basic visualization
- Priority: real datasets, cleaning workflows

**Phase 3 (Weeks 11-16): Applied Projects**
- End-to-end analysis projects for portfolio
- Priority: storytelling with data, presentation

### Daily Schedule Pattern
**Weekdays (1.5h)**:
- 45 min: New concept learning (video/reading)
- 30 min: Coding practice (exercises)
- 15 min: Review previous day's code

**Weekends (2h)**:
- Saturday: Project work or challenging problems
- Sunday: Review week, plan next week

### Milestone Checklist
□ Week 3: Complete 50 basic Python exercises
□ Week 6: Clean and analyze your first real dataset
□ Week 9: Create 5 different chart types in matplotlib
□ Week 12: Complete Project 1 (exploratory data analysis)
□ Week 14: Complete Project 2 (predictive model)
□ Week 16: Complete Project 3 + portfolio website

### Recommended Resources
- Primary: "Python Crash Course" book + Kaggle Learn
- Practice: HackerRank Python domain, LeetCode easy problems
- Projects: Kaggle datasets, FiveThirtyEight data
- Reference: Pandas documentation, Python.org tutorial
```

## Template 3: Language Learning (6 months)

### Example: Spanish to Conversational Level

```
## 📚 Study Plan: Spanish (A1 → B1 Conversational)
**Goal**: Hold 30-min conversation | **Deadline**: 24 weeks | **Daily Time**: 45 minutes | **Level**: Complete beginner

### Curriculum Overview (180 total hours)
Vocabulary: Core 2000 words — 60 hours (33%)
Grammar: Essential structures — 45 hours (25%)
Listening: Comprehension practice — 30 hours (17%)
Speaking: Conversation practice — 30 hours (17%)
Reading: Graded texts — 15 hours (8%)

### Phase Breakdown
**Phase 1 (Weeks 1-8): Foundation (A1)**
- Present tense, basic vocabulary (500 words)
- Priority: pronunciation, basic sentence structure

**Phase 2 (Weeks 9-16): Building (A2)**
- Past/future tenses, expanded vocabulary (1500 words)
- Priority: listening comprehension, basic conversations

**Phase 3 (Weeks 17-24): Fluency (B1)**
- Complex grammar, cultural context, real media
- Priority: speaking confidence, natural flow

### Daily 45-Minute Schedule
**Monday/Wednesday/Friday**:
- 15 min: Duolingo/Babbel lesson
- 15 min: Anki flashcards (spaced repetition)
- 15 min: Grammar exercises

**Tuesday/Thursday**:
- 20 min: Listening practice (podcasts, videos)
- 15 min: Speaking practice (shadowing, recording)
- 10 min: Vocabulary review

**Saturday**:
- 30 min: Conversation practice (iTalki tutor)
- 15 min: Cultural content (Spanish YouTube, news)

**Sunday**: Light review, plan next week

### Milestone Checklist
□ Week 4: Introduce yourself for 2 minutes (recording)
□ Week 8: Complete A1 level test (80%+)
□ Week 12: Watch 10-min Spanish video with 70% comprehension
□ Week 16: Complete A2 level test (80%+)
□ Week 20: Have 15-min conversation with native speaker
□ Week 24: Pass B1 conversation assessment

### Recommended Resources
- Primary: Babbel or Busuu structured course
- Vocabulary: Anki with frequency-based Spanish deck
- Listening: SpanishPod101, News in Slow Spanish
- Speaking: iTalki tutors, HelloTalk language exchange
- Grammar: SpanishDict grammar guide
```

## Template 4: Exam Cramming (2-4 weeks)

### Example: Final Exam Preparation

```
## 📚 Study Plan: Biology Final Exam (Cramming Mode)
**Goal**: Score 85%+ on final | **Deadline**: 3 weeks | **Daily Time**: 4 hours | **Level**: Covered in class

### High-Yield Topic Prioritization (84 total hours)
**Tier 1 (60% of exam, 50 hours)**:
- Cell biology, genetics, evolution
- Focus: memorization + application problems

**Tier 2 (30% of exam, 25 hours)**:
- Ecology, plant biology
- Focus: concept understanding

**Tier 3 (10% of exam, 9 hours)**:
- Detailed mechanisms, edge cases
- Focus: only if time permits

### Week-by-Week Breakdown
**Week 1: Content Review**
- Days 1-3: Tier 1 topics (intensive review)
- Days 4-5: Tier 2 topics
- Weekend: First practice test, identify gaps

**Week 2: Active Practice**
- Days 1-3: Problem sets, past exams
- Days 4-5: Weak area drilling
- Weekend: Second practice test

**Week 3: Final Prep**
- Days 1-3: Spaced review, flashcards
- Days 4-5: Final practice tests
- Day 6: Light review, rest
- Day 7: EXAM DAY

### Daily 4-Hour Schedule
**Hour 1**: Active reading (textbook + notes)
**Hour 2**: Practice problems or flashcards
**Hour 3**: Past exam questions
**Hour 4**: Review mistakes, create summary notes

### Emergency Shortcuts
- Focus on professor's emphasized topics (lecture time = importance)
- Memorize diagrams and processes (high-yield for biology)
- Use acronyms and memory palaces for lists
- Study with classmates for last-minute clarifications
```
