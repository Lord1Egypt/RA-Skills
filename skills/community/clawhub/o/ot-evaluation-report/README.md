# OT Evaluation Report Drafter

Drafts an initial Occupational Therapy evaluation report from clinician-provided data, aligned to the AOTA Occupational Therapy Practice Framework 4th edition and CMS / Medicare documentation standards.

## What It Does

Takes referral information, occupational profile data, ADL/IADL performance observations, standardized assessment scores, and clinician impressions, then produces a structured DRAFT evaluation report covering:

- Occupational profile (roles, routines, priorities, environments)
- Analysis of occupational performance (ADLs, IADLs, work, play/leisure, social participation)
- Standardized assessment scores table
- Clinical impressions and functional limitations summary
- SMART short-term and long-term goals table
- Intervention plan with frequency, duration, and skilled-service rationale
- Unresolved-information list for OT to address before sign-off

## Who It's For

- Licensed Occupational Therapists (OTR/L) in outpatient, inpatient, school-based, or home-health settings
- Occupational Therapy Assistants (OTA/COTA) drafting under supervising OT
- Rehab documentation specialists supporting OT teams

## When To Use

Use when you need to convert raw evaluation notes, intake data, or clinical observations into a formatted OT evaluation report. Suitable for initial evaluations, re-evaluations, and school-based IEP-related OT evaluations.

## Important Limitations

All output is a **DRAFT** for licensed OT review and signature. The skill does not render a clinical decision, diagnose, or substitute for the OT's professional judgment. Output must be reviewed, edited, and signed by the treating OT before entry into any medical record, school record, or payer submission.

## Compliance References

- AOTA Occupational Therapy Practice Framework, 4th edition (OTPF-4)
- CMS Medicare Part B documentation requirements (42 CFR § 410.59, MLN 905364)
- IDEA Part B (school-based OT evaluation requirements)
- AOTA Occupational Profile Template (2017)
