# 胰岛细胞类型、分布及其激素分泌功能

## 何时使用
- 分析血糖调节异常（如持续高血糖或低血糖）
- 解释胰腺内分泌部功能障碍
- 探讨1型或2型糖尿病中胰岛细胞变化的潜在机制
- 区分胰腺内分泌与外分泌疾病

## 如何执行
1. **确认适用对象**：仅适用于胰腺内分泌部、胰岛或血糖调节系统相关问题；排除外分泌胰腺疾病。
2. **识别胰岛主要细胞类型及其比例**：
   - β细胞（约75%）：位于胰岛中央，分泌胰岛素
   - α细胞（约20%）：分布于胰岛周边，分泌胰高血糖素
   - δ细胞（约5%）：散在分布，分泌生长抑素
   - PP细胞（极少）：分泌胰多肽
3. **关联激素功能与血糖调节**：
   - 胰岛素降低血糖，促进葡萄糖摄取与储存
   - 胰高血糖素升高血糖，促进肝糖原分解与糖异生
   - 生长抑素通过旁分泌抑制α和β细胞，维持胰岛内稳态
   - 胰多肽调节胰腺外分泌及胃肠运动
4. **结合临床意义解释机制**：
   - β细胞破坏 → 胰岛素绝对缺乏（如1型糖尿病）
   - α细胞功能亢进 → 高胰高血糖素血症 → 加重高血糖

> 注意：本技能不直接解释1型或2型糖尿病的全部发病机制，仅聚焦胰岛细胞层面的功能分工。
