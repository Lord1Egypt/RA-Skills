# C3 活动运营与执行 — 活动引擎

管理从学术会议到全国巡讲的全生命周期。辅助策划、自动检测排期冲突、生成物料和预算清单、管理讲者行程与费用报销、回收现场数据，自动生成复盘报告。

- **依赖**: 无（入口单元）
