## Description: <br>
Skill Activation Booster - Fix the 40% activation problem. Optimize triggers, descriptions, and context so agents actually use your skills. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and skill authors use this guidance skill to audit and improve trigger wording, descriptions, activation profiles, and activation tests so agents use relevant skills more reliably. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Future versions could add scripts, broad local scanning, or automatic edits that change agent behavior without explicit user approval. <br>
Mitigation: Review and scan new versions before installation, and require explicit user approval before accepting edits or executing new automation. <br>
Risk: Trigger optimization advice can increase false activations if overlap checks, anti-triggers, and activation tests are skipped. <br>
Mitigation: Use the skill's activation profile, anti-trigger examples, and test tables before deploying changed skill descriptions. <br>
