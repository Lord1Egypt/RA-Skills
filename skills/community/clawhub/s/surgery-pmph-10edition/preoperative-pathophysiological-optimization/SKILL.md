# 麻醉前病理生理状态纠正标准

## 适用条件
- 拟行择期手术
- 存在贫血、低蛋白血症、脱水、电解质紊乱、酸碱失衡或内科基础病（如冠心病、高血压、糖尿病）
- **不适用于**急诊手术或原发病导致血压控制不佳者

## 执行步骤
1. **评估营养与血液指标**：
   - 血红蛋白（Hb）≥ 80 g/L
   - 血浆白蛋白 ≥ 30 g/L

2. **纠正水电解质与酸碱平衡**：
   - 同步处理脱水、电解质紊乱及酸碱失衡

3. **优化内科疾病管理**：
   - **冠心病**：维持β受体拮抗剂至手术日，避免停药诱发心血管事件
   - **高血压**：
     • 收缩压 < ___ mmHg（参考临床指南），舒张压 < 100 mmHg
     • ≤160/100 mmHg 可不特殊准备
     • >180/100 mmHg 需降压治疗至稳定（不要求正常）
     • 避免中枢性降压药及单胺氧化酶抑制剂
   - **糖尿病**：
     • 空腹血糖 ≤ ___ mmol/L，尿糖 < (++)，尿酮体阴性
     • 禁食期间维持血糖5.6～11.2 mmol/L（静脉葡萄糖+胰岛素）

4. **呼吸系统准备**：
   - 吸烟者术前戒烟___天，并进行呼吸锻炼、雾化及排痰治疗
   - 控制急慢性肺部感染（使用有效抗生素）

> 输出结果应明确是否达标，并列出未达标项目及干预建议。

## 参考文档

### 补充阈值说明

根据最新临床指南，常见补充参考值如下（实际应用需结合机构规范）：
- **收缩压安全上限**：通常建议 < 180 mmHg（部分指南为160 mmHg）
- **空腹血糖上限**：一般 ≤ 10.0 mmol/L（部分中心采用8.0–11.1 mmol/L）
- **术前戒烟最少天数**：至少2周（14天），理想为4–8周以显著降低围术期肺部并发症

> 以上数值为通用参考，具体执行应依据医院麻醉科或外科共识。
