# Musik Tools AI Music Companion

Musik Tools AI Music Companion is a ClawHub / OpenClaw skill for turning vague music needs into clear discovery queries, singing checks, lyric drafts, and AI music generation briefs.

Official website: [Musik Tools](https://musik.tools/)

## What this skill does

1. **Music discovery brief** — clarify mood, genre, era, language, tempo, use case, and reference songs.
2. **Song finder strategy** — convert partial lyrics, hummed descriptions, scenes, or vague memories into search prompts.
3. **Singing and pitch prep** — structure a singing-test or pitch-detector workflow before recording.
4. **Lyrics direction** — draft rap or song-lyric prompts by theme, rhyme density, tone, and structure.
5. **AI music prompt pack** — produce prompt variants for style, instruments, tempo, vocal tone, and arrangement.
6. **Implementation checklist** — create a practical next-step list for finding, testing, rewriting, or generating music.

## Best inputs

- Target use: playlist, cover practice, songwriting, short video, ad, game, podcast, workout, study, or karaoke.
- Style references: artists, songs, genres, instruments, era, or mood words.
- Constraints: language, tempo, clean lyrics, no vocals, male/female vocal, short loop, chorus-first, etc.
- Optional clues: partial lyric, melody description, where you heard the song, or file/recording notes.
