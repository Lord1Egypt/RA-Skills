# AI工程造价+法律知识库 Skill

基于腾讯 IMA「AI工程造价+法律知识库」构建的造价+法律交叉领域AI技能，帮助工程与法律人士精准管控风险，提升项目效益。

✅ **真实知识库支撑**：基于1394人订阅、3149个内容的权威知识库

## 安装方法

### 方法一：直接安装（推荐）

1. 下载本仓库
2. 将 `AI工程造价+法律知识库_skill` 文件夹复制到你的 Marvis skills 目录
3. 重启 Marvis 或重新加载技能

### 方法二：通过 GitHub 安装

```bash
# 克隆仓库
git clone https://github.com/你的用户名/AI工程造价+法律知识库-skill.git

# 复制到 skills 目录
cp -r AI工程造价+法律知识库-skill/AI工程造价+法律知识库_skill ~/.marvis/skills/
```

### 方法三：手动安装

1. 在 Marvis 的 skills 目录下创建 `ai-engineering-legal-knowledge` 文件夹
2. 将 `SKILL.md` 文件复制到该文件夹
3. 创建 `prompts`、`references`、`examples` 子目录（可选）
4. 重启 Marvis

## 使用示例

### 触发词
- 造价+法律、工程司法鉴证、合同风控
- 变更索赔、结算纠纷、鉴定意见
- 证据链、工期索赔、费用索赔

### 基本用法

```bash
# 合同风险审查
用户：帮我审查这份施工合同的风险条款
技能：自动调用合同审查Prompt，分析计价风险、变更管理、结算条款

# 变更索赔分析
用户：分析这个设计变更的索赔合理性
技能：自动调用变更索赔分析框架，评估变更合法性、费用计算

# 结算纠纷诊断
用户：结算被审减了30%，如何应对？
技能：自动检索知识库中类似案例，提供反驳策略

# 司法鉴定准备
用户：案件要进入司法鉴定，如何准备材料？
技能：提供证据链构建指南、鉴定意见撰写模板
```

## 知识库内容概览

**AI工程造价+法律知识库**包含7大类内容：

1. **造价规范**：工程量清单计价标准、定额标准、计价规范
2. **法律法规**：建设工程相关法律法规、司法解释、政策文件
3. **合同范本**：施工合同示范文本、专用条款编写要点
4. **案例库**：最高院建设工程纠纷案例、工程价款优先受偿权案例
5. **实务指南**：合同风控、变更索赔、结算纠纷解决方案
6. **专业书籍**：工程价款管理、建设项目工程总承包计价规范
7. **审计方法**：工程造价电子数据审计、招标采购审计应用

## 适用人群

- **工程造价人员**：需要法律风险防控的造价工程师
- **工程法律顾问**：需要造价专业知识的法律人士
- **施工企业管理人员**：需要合同风控和索赔管理的项目经理
- **建设单位成本人员**：需要结算审核和纠纷处理的成本经理
- **司法鉴定人员**：需要造价+法律交叉知识的鉴定专家
- **工程审计人员**：需要电子数据审计方法的审计师
