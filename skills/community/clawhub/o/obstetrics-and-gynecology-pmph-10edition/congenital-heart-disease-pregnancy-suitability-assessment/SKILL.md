# 先天性心脏病孕妇妊娠适宜性判断

## 何时使用
- 患者为育龄期女性，已确诊先天性心脏病
- 正在进行孕前咨询或妊娠≤12周的早期评估
- 已完成心功能NYHA分级并明确心脏解剖参数
- 不适用于风湿性心脏病、围产期心肌病等非先天性心脏病

## 执行步骤

### 1. 按先天性心脏病类型分类评估

**A. 左向右分流型**
- **房间隔缺损（ASD）**：
  - 缺损面积 <1 cm² → 可耐受妊娠
  - 缺损面积 >2 cm² → 建议手术矫治后再妊娠
  - 若存在肺动脉高压或右向左分流 → 判定为不宜妊娠
- **室间隔缺损（VSD）**：
  - 缺损面积 <1.25 cm²、无心衰史、无并发症 → 可继续妊娠
  - 缺损面积大、未手术修补、伴肺动脉高压 → 禁止妊娠，若已妊娠应于早孕期终止
- **动脉导管未闭（PDA）**：
  - 管径小、肺动脉压正常 → 可继续妊娠
  - 管径大、存在肺动脉高压或右向左分流 → 建议终止妊娠

**B. 右向左分流型**
- **法洛四联症（未手术）** → 不宜妊娠，若已妊娠应尽早终止
- **艾森门格综合征** → 绝对禁忌妊娠

**C. 无分流型**
- **重度肺动脉瓣狭窄** → 宜妊娠前行手术矫治
- **主动脉缩窄** → 谨慎评估，妊娠并发症发生率高
- **马方综合征（MFS）**：
  - 主动脉根部直径 >45 mm 或有夹层病史 → 不宜妊娠
  - 已妊娠者需严格控压，全程超声心动图监测（至产后6个月）

### 2. 结合心功能分级综合判断
- NYHA I～II级且无严重解剖异常 → 可考虑妊娠
- NYHA III～IV级 → 不宜妊娠

### 3. 输出结论与后续建议
- 输出三类结论之一：
  - "可以妊娠"
  - "不宜妊娠需终止"
  - "需手术矫治后评估"
- 对所有“不宜妊娠”者：
  - 孕前应加强避孕宣教
  - 若意外妊娠，建议在≤12周内终止，并注意围手术期镇痛及抗感染

## 关键输入变量
- 心脏缺损面积（cm²）
- 是否存在肺动脉高压（是/否）
- 分流方向（左向右 / 右向左 / 无分流）
- NYHA心功能分级（1–4级）
- 主动脉根部直径（mm，仅马方综合征适用）
