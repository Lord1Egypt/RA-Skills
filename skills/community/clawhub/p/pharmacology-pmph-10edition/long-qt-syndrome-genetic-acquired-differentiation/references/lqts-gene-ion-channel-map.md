# LQTS致病基因与对应离子通道

| 基因 | 影响的离子电流 | 通道功能 |
|------|----------------|----------|
| KCNQ1, KCNE1, AKAP9 | I_Ks（慢延迟整流钾电流） | 复极储备，心率加快时增强 |
| KCNH2, KCNE2 | I_Kr（快延迟整流钾电流 / hERG） | 主要复极电流，多数药物靶点 |
| SCN5A, CAV3, SCN4B, SNTA1 | I_Na（钠电流） | 动作电位0相除极，部分突变导致晚钠电流增加 |
| ANK2 | 多种（Na⁺, K⁺, Ca²⁺） | 锚定蛋白，影响多种通道定位 |
| KCNJ2 | I_K1（内向整流钾电流） | 维持静息电位，参与终末复极 |
| CACNA1C | I_Ca(L)（L型钙电流） | 平台期维持，突变可延长复极 |
| KCNJ5 | I_KACh（乙酰胆碱敏感钾电流） | 迷走神经激活时开放，影响心房复极 |

## 典型获得性LQTS药物示例
- **抗心律失常药**：奎尼丁、索他洛尔、胺碘酮
- **抗生素**：大环内酯类（红霉素）、氟喹诺酮类（莫西沙星）
- **抗精神病药**：氟哌啶醇、齐拉西酮
- **抗肿瘤药**：三氧化二砷（As₂O₃）— 同时抑制hERG、I_Ca(L)、I_K1

## 风险增强因素
- 低钾血症、低镁血症
- 心动过缓
- 女性性别
- 高龄
- 药物相互作用（如CYP3A4抑制剂增加hERG阻断药浓度）
