## Description: <br>
Helps agents apply Remix v2 routing patterns for flat-route filenames, nested layouts, resource routes, root modules, and v1-to-v2 migrations. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[anderskev](https://clawhub.ai/user/anderskev) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill when implementing or reviewing Remix v2 route modules, route file naming, nested layouts, resource routes, and root.tsx scaffolding. It also helps compare v1 and v2 routing conventions during migrations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated route or configuration changes can alter Remix application behavior, especially resource routes and migration adapter setup. <br>
Mitigation: Review proposed changes before applying them and verify resource routes, parent-loader assumptions, and migration adapter configuration against the application. <br>
