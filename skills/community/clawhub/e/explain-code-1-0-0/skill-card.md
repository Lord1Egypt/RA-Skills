## Description: <br>
Explains code with visual diagrams and everyday analogies when users ask how code works or need help learning a codebase. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[zhangyingzhuangk](https://clawhub.ai/user/zhangyingzhuangk) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and learners use this skill to get code explanations that combine analogies, ASCII diagrams, step-by-step walkthroughs, and notes about common mistakes. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The trigger and language guidance may make the skill activate for broad code-explanation requests or favor Chinese-oriented responses. <br>
Mitigation: Review or adjust the trigger and wording before deployment if the environment needs narrower activation or a different response language. <br>
Risk: Analogies and ASCII diagrams can simplify code behavior in ways that miss important implementation details. <br>
Mitigation: Compare explanations against the source code before relying on them for debugging, review, or onboarding decisions. <br>
