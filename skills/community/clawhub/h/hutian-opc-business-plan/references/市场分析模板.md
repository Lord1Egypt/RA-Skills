# 市场分析模板

## 一、市场规模测算框架

### TAM/SAM/SOM模型

```
                    ┌─────────────────────────┐
                    │         TAM              │
                    │   (潜在市场总量)          │
                    │   100%潜在需求            │
                    │                         │
                    │   ┌─────────────────┐   │
                    │   │      SAM        │   │
                    │   │  (可服务市场)    │   │
                    │   │  地理/渠道覆盖   │   │
                    │   │                 │   │
                    │   │   ┌─────────┐   │   │
                    │   │   │   SOM   │   │   │
                    │   │   │(目标市场)│   │   │
                    │   │   │ 3-5年   │   │   │
                    │   │   │ 可获得  │   │   │
                    │   │   └─────────┘   │   │
                    │   └─────────────────┘   │
                    └─────────────────────────┘
```

### 测算方法

**方法1：自上而下法（Top-Down）**
```
TAM = 目标用户群体 × 产品使用频率 × 客单价

示例（ SaaS 产品）：
- 目标企业数量：1000万家
- 目标付费渗透率：5%
- 年均客单价：2万元
- TAM = 1000万 × 5% × 2万 = 1000亿元
```

**方法2：自下而上法（Bottom-Up）**
```
基于具体细分市场逐一测算

| 细分市场 | 用户规模 | 付费意愿 | 年均ARPU | 市场规模 |
|---------|---------|---------|---------|---------|
| 大型企业 | 10万     | 80%     | 10万    | 80亿    |
| 中型企业 | 100万    | 40%     | 3万     | 120亿   |
| 小型企业 | 2000万   | 10%     | 5000    | 100亿   |
| 合计     |          |         |         | 300亿   |
```

**方法3：替代分析法**
```
基于替代现有方案的市场空间推算

原方案市场规模 × 替代率 × 价值捕获率 = 新市场空间

示例：
- 现有解决方案市场规模：500亿
- 预期替代率：20%
- 新方案价值捕获率：60%
- 新市场 = 500亿 × 20% × 60% = 60亿
```

```
    ┌───────────────────┼───────────────────┐
```

### 竞争策略分析

**波特五力分析**
```
                         供应商议价能力
                              ▲
                              │
                    ┌─────────┴─────────┐
                    │                     │
        新进入者 ───┤     本行业         ├── 替代品威胁
        威胁       │   竞争程度          │   威胁
                    │                     │
                    └─────────┬─────────┘
                              │
                    买方议价能力
                              ▼
```

**五力评估表**
| 五力 | 强度评估 | 主要影响因素 | 应对策略 |
|-----|---------|-------------|---------|
| 新进入者威胁 | 高/中/低 | 壁垒高度、资本需求 | 加速跑马圈地、构建护城河 |
| 供应商议价能力 | 高/中/低 | 供应商集中度、可替代性 | 多元化采购、战略合作 |
| 买方议价能力 | 高/中/低 | 买方数量、集中度 | 差异化、锁定策略 |
| 替代品威胁 | 高/中/低 | 替代品性价比、切换成本 | 持续创新、提升转换成本 |
| 行业内竞争 | 高/中/低 | 行业集中度、差异化程度 | 差异化、成本领先 |

## 四、市场进入策略模板

### 进入策略选择矩阵

| 策略类型 | 适用条件 | 优势 | 风险 | 资源需求 |
|---------|---------|-----|-----|---------|
| 闪电战 | 窗口期短、资本充足 | 快速占领、规模效应 | 投入大、风险高 | 高 |
| 渗透战 | 产品成熟、市场教育完成 | 风险可控、口碑积累 | 时间长、可能被复制 | 中 |
| 利基战 | 资源有限、差异化明显 | 细分领先、利润率高 | 市场天花板低 | 低 |
| 生态战 | 平台型产品、开放战略 | 网络效应、护城河深 | 构建周期长 | 高 |

### 渠道策略规划

**渠道组合矩阵**
```
              渠道覆盖面
              大▲
              │
    直销     │  ★ 直销（KA客户）
              │
    经销     │      ★ 经销（渠道商）
              │

              │

              │

                            低      高
```

| 直销 | 低 | 低 | 高 | 人力成本 | 早期 |
| 经销 | 中 | 中 | 中 | 渠道分成 | 成长期 |
| 代理 | 高 | 高 | 中低 | 代理费 | 扩张期 |
| 线上 | 极高 | 高 | 中 | 营销成本 | 全阶段 |

```
               曝光
```

## 五、市场分析报告模板

### 标准市场分析报告结构

```
一、执行摘要
   1. 核心发现
   2. 关键数据
   3. 战略建议

二、市场概况
   1. 市场规模（TAM/SAM/SOM）
   2. 增长趋势
   3. 驱动因素

三、市场特征
   1. 用户画像
   2. 需求分析
   3. 消费行为

四、竞争格局
   1. 竞争者列表
   2. 市场份额分布
   3. 竞争态势分析

五、进入时机
   1. 市场成熟度评估
   2. 窗口期判断
   3. 风险因素

六、战略建议
   1. 目标市场选择
   2. 进入策略建议
   3. 资源配置建议
```
