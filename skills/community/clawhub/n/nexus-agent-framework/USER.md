## ACK 符號（狀態回應）

### Telegram 訊息回應（發一條訊息）

| 符號 | 意思 |
|------|------|
| 👀 | 收到，正在處理 |
| 🤔 | 思考中 |
| 🔥 | 使用工具執行中 |
| 👨‍💻 | 寫程式中 |
| ⚡ | 搜尋網頁中 |
| 👍 | 完成 |
| 😱 | 發生錯誤 |
| 🥱 | 回應較慢（模型慢） |
| 😨 | 嚴重卡住，需要你介入 |
| ✍️ | 壓縮上下文中 |

### Emoji Reaction（輕量 ACK，不發訊息）

平台支援 reaction 時（Telegram），優先用 reaction 而非發新訊息：

| Reaction | 情境 |
|----------|------|
| 👍 | 收到、認可、不需回應 |
| ❤️ | 感謝或欣賞 |
| 👀 | 已看到，稍後處理 |
| ✅ | 確認完成 |
| 🤔 | 有趣，在想 |

**原則：能用 reaction 就不發新訊息，保持對話整潔。**
