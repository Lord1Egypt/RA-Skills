## Description: <br>
Generate submission issues for GitHub awesome-lists, including formatted issue and pull request drafts for popular awesome repositories. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[bytesagain1](https://clawhub.ai/user/bytesagain1) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, founders, and marketers use this skill to draft GitHub awesome-list submissions, choose relevant repositories, follow contribution formats, and track submission status. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated issue or pull request drafts may not match a repository's contribution rules, scope, or expected tone. <br>
Mitigation: Review each draft for relevance, formatting, repository rules, and tone before posting. <br>
Risk: Printed shell snippets may need product-specific or repository-specific edits before use. <br>
Mitigation: Inspect and adapt any command before running it with real product, repository, or tracking details. <br>
