# 大学智慧 / The Great Learning Wisdom

## 1. 核心概述 / Core Overview

《大学》是儒家“内圣外王”的标准范式。全文提出**"三纲领八条目"**的实践体系，影响中国两千余年。
*The Great Learning* provides a complete progressive path from inner sagehood to outer kingship, emphasizing self-cultivation as the root.

**核心价值 / Core Value**：
为个体与组织提供从内圣（修身）到外王（治国平天下）的完整进阶路径。
*Provides a path from Inner Sagehood to Outer Kingship.*

---

## 2. 三纲领：人生最高理想 / The Three Guidelines

1.  **明明德 (Illustrious Virtue)**：彰显自身本有的光明德性，去除物欲遮蔽。
    *Manifest the luminous virtue inherent within oneself.*
2.  **亲民 (Renewing the People)**：推己及人，教化民众，从独善其身到兼善天下。
    *Extend oneself to others, transform the people.*
3.  **止于至善 (Highest Excellence)**：达到完美的道德境界并持守，不妥协。
    *Attain the perfect moral state and maintain it.*

---

## 3. 八条目：实践进阶路径 / The Eight Steps

> “古之欲明明德于天下者，先治其国……致知在格物。”
> *Ancient kings wishing to manifest virtue globally started with the state, then family, then self... ending with investigating things.*

### 3.1 内圣功夫（向内）/ Inner Sagehood
1.  **格物 (Investigate things)**：研究事物原理，博学审问。
2.  **致知 (Extend knowledge)**：获得真知灼见，明辨是非。
3.  **诚意 (Sincere intentions)**：意念真诚，毋自欺，慎独。
4.  **正心 (Rectify the mind)**：心态端正，去忿懥、恐惧、好乐、忧患。

### 3.2 外王事业（向外）/ Outer Kingship
5.  **修身 (Cultivate the person)**：完善人格品德，**“一是皆以修身为本”**。
6.  **齐家 (Regulate the family)**：治理家庭，孝悌慈，以身作则。
7.  **治国 (Govern the state)**：推己及人，絜矩之道，以德服人。
8.  **平天下 (Bring peace to the world)**：天下太平，以义为利。

## 4. 核心心法 / Core Mental Models

### 4.1 慎独功夫 / Watchfulness in Solitude
*   **定义**：独处时如“十目所视，十手所指”，不欺暗室。
*   **应用**：道德自律的最高境界，表里如一。
*   *See `references/original-text.md` for details.*

### 4.2 絜矩之道 / The Principle of Reciprocity (Empathy)
*   **定义**：己所不欲，勿施于人。换位思考的六维模型。
*   **公式**：
    *   所恶于上，毋以使下 (Don't treat subs how you hate being treated by superiors)
    *   所恶于下，毋以事上 (Don't treat supers how you hate being treated by subs)
    *   前/后、左/右同理。
*   **应用**：解决组织内部矛盾、人际冲突的万能钥匙。

### 4.3 知止六步法 / Six Steps to Attainment
*   **流程**：知止 → 有定 → 能静 → 能安 → 能虑 → 能得
*   **应用**：在面临混乱决策时，先定目标（知止），让心静下来，再深思熟虑，最终有所得。

### 4.4 德本财末 / Virtue as Root, Wealth as Branch
*   **因果链**：德 → 人 → 土 → 财 → 用
*   **原则**：先有德行和文化（本），再有人心和资源，最后才有财富（末）。
*   **应用**：企业战略排序，不可本末倒置（为了利润牺牲价值观）。

### 4.5 日新精神 / Spirit of Daily Renewal
*   **名言**：“苟日新，日日新，又日新。”
*   **应用**：持续改进，每天都要像洗澡一样洗涤心灵，自我更新。

## 5. 应用场景 / Application Scenarios

| 场景 | 核心工具 | 执行逻辑 |
| :--- | :--- | :--- |
| **个人迷茫** | 八条目 | 从“格物致知”开始，搞清楚现状，再谈目标。 |
| **团队管理** | 絜矩之道 | 建立“己所不欲勿施于人”的团队文化。 |
| **创业/商业** | 德本财末 | 优先做产品（德/本），而不是先想怎么圈钱（财/末）。 |
| **心态焦虑** | 知止六步 | 停止内耗，先确定一个“止”点（目标），静下来思考。 |
| **自我约束** | 慎独 | 在没人监督时，依然保持高标准。 |
