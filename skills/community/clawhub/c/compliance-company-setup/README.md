# 跨境电商注册地推荐（Company Setup Check）

> 输入产品与目标市场，一分钟生成跨境公司注册地推荐方案。

覆盖 8 个热门管辖区的成本、税率、流程和平台兼容性对比，适用于跨境电商卖家、独立站创业者、工贸一体小工厂在海外注册公司时不知道选哪个国家的场景。

📖 详细使用说明见 [SKILL.md](./SKILL.md)
📊 完整数据见 [references/company-setup-data.md](./references/company-setup-data.md)

---

## 快速上手

直接说下面任意一句：

> 「我在亚马逊卖 3C 配件，月销 4 万美元，想正规化经营」
> 「对比香港和新加坡，哪个更适合跨境电商注册？」
> 「已注册美国 Wyoming LLC，想进 EU 市场销售」
> 「注册美国公司多少钱？流程是什么？」

## 核心输出

| 输出类型 | 说明 |
|---------|------|
| 多管辖区对比表 | 成本、税率、流程、平台兼容并排对比 |
| 注册地推荐 | 基于用户经营模式给出最优推荐 + 理由 |
| 步骤化清单 | 选定注册地后的完整操作清单 + 时间线 |
| 多国联动方案 | 多市场销售时的"总部+入口"组合建议 |
