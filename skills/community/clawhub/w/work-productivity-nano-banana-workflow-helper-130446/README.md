# Nano Banana Image Workflow Helper

A workflow skill for planning and debugging AI image generation or editing work. It turns a visual goal into prompt packs, reference-image instructions, retry budgets, and visual QA checks.

## Best For

- Product, marketing, avatar, thumbnail, and concept-image prompt packs.
- Editing workflows with reference images, masks, style constraints, or brand rules.
- Debugging failed generations caused by unclear prompts, bad references, queue cost, wait time, or tool setup.
- Packaging image-generation workflows into reliable ClawHub skills.

## Outputs

- Base prompt, negative prompt, style anchors, composition notes, and variation prompts.
- QA checklist for artifacts, text, anatomy, object counts, brand consistency, and safety.
- Retry plan with attempt limits, fallback choices, and stop criteria.
- Optional deployment checklist for making first use hard to misconfigure.

## Evidence

Demand was supported by Nano Banana Pro-style ClawHub popularity, Nano PDF adjacency, a V2EX discussion about AI image/video generation experience problems, and a GitHub request for guided first deployment.
