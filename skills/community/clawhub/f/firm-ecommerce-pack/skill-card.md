## Description: <br>
Firm Ecommerce Pack is a curated e-commerce operations prompt bundle for platforms, D2C brands, and marketplace operators that configures department-oriented agents for catalog, campaign, customer support, and revenue workflows. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
E-commerce, D2C, and marketplace teams use this skill to coordinate marketing, commercial, engineering, support, finance, and operations work through reusable firm-style routing profiles and prompt examples. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Recommended companion install commands may add skills that connect to commerce, CRM, analytics, payment, social media, or catalog systems. <br>
Mitigation: Review each companion skill and its permissions before installation, and install only the connectors needed for the workspace. <br>
Risk: The pack uses session-spawning OpenClaw tools for orchestration. <br>
Mitigation: Run it in a workspace approved for agent read or write access, and confirm session-spawning governance in the OpenClaw setup before use. <br>
