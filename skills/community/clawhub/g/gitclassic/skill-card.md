## Description: <br>
GitClassic is a fast, no-JavaScript GitHub browser optimized for AI agents to browse public repositories, read files, view READMEs, search GitHub content, and optionally access private repositories through PRO GitHub OAuth. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
Developers and AI agents use GitClassic to browse GitHub repositories quickly, read code and documentation, view READMEs, and search users, organizations, or repositories through clean server-rendered HTML. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Optional private-repository access uses GitClassic's GitHub OAuth flow and may involve repositories with confidentiality requirements. <br>
Mitigation: Review the requested GitHub OAuth scopes and GitClassic privacy, caching, and data-retention practices before use; use direct GitHub access for repositories with stricter confidentiality requirements. <br>
