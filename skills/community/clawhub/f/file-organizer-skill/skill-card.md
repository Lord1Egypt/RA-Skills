## Description: <br>
Organize files in directories by grouping them into folders based on their extensions or date. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[1999AZZAR](https://clawhub.ai/user/1999AZZAR) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers, operators, and general users can use this skill to organize local directories by file extension or modification date, preview planned moves, and undo moves using the generated history file. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The organizer can move files in the selected directory, and recursive mode expands the scope of those moves. <br>
Mitigation: Run with --dry-run first, target a narrow folder, and enable --recursive only when the full directory tree is intended. <br>
Risk: Undo relies on paths in organize_history.json and can restore files incorrectly if the history file is untrusted or unrelated. <br>
Mitigation: Use --undo only with organize_history.json generated by this script from a folder you trust. <br>
