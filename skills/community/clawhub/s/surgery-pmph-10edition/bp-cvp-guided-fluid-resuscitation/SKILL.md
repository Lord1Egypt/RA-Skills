# 基于血压与中心静脉压联合指导的补液决策

## 何时使用
当患者属于以下任一类别且存在低血压或血流动力学不稳定时：
- 休克患者
- 低血容量患者
- 术后危重病人

同时满足前提条件：
- 可测量动脉血压
- 已置入中心静脉导管并可准确测量CVP

**注意**：本技能不适用于严重心律失常或三尖瓣病变患者（因CVP测量可能失真），且必须结合尿量、乳酸等其他指标综合判断。

## 执行步骤
根据实测的血压（BP）和中心静脉压（CVP）值，按以下决策树执行：

1. **BP正常 + CVP低（<5 cmH₂O）**
   - 判断：血容量不足
   - 行动：适当补液

2. **BP正常 + CVP高（>15 cmH₂O）**
   - 判断：容量血管过度收缩
   - 行动：考虑使用血管扩张剂（如山莨菪碱），避免继续补液

3. **BP低 + CVP低（<5 cmH₂O）**
   - 判断：血容量严重不足
   - 行动：充分补液，快速输注晶体液和胶体液

4. **BP低 + CVP正常（5~10 cmH₂O）**
   - 判断：可能为心功能不全或隐匿性血容量不足
   - 行动：进行补液试验（快速输注200~300ml液体，观察CVP和BP变化）

5. **BP低 + CVP高（>15 cmH₂O）**
   - 判断：心功能不全
   - 行动：启动强心治疗（如多巴酚丁胺、去乙酰毛花苷），严禁盲目补液

## 关键定义
- **BP正常**：通常指收缩压 ≥90 mmHg
- **CVP正常范围**：5~10 cmH₂O
- **补液试验**：在严密监测下快速输注少量液体（200~300ml），评估循环反应

动态评估治疗反应，并根据后续生命体征和灌注指标调整治疗方案。
