## Use Case: <br>
Agent operators and researchers use this skill to plan web research, prioritize current search evidence, maintain citation logs, and synthesize sourced reports. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can overstate trust in user input and real-time tool output. <br>
Mitigation: Treat search results as research leads and verify important claims against authoritative sources before use. <br>
Risk: Web search and scraping may expose the agent to adversarial or irrelevant page content. <br>
Mitigation: Review source content and keep the agent focused on the original research task, especially for high-stakes topics. <br>
