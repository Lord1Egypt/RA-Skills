## Use Case: <br>
HR teams and authorized business users use this skill to request Indonesian-language summaries and recommendations for employee monitoring, performance reviews, payroll metrics, attendance, and workforce risk signals. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill is intended for HR, payroll, attendance, finance, and employee-sensitive information. <br>
Mitigation: Install only for authorized HR use, define approved data sources, enforce role-based access, set audit expectations, and document rules for handling sensitive employee and payroll data before connecting real systems. <br>
Risk: Several reference files contain TODO data sources and placeholder operational details, so generated insights may be incomplete or based on missing context. <br>
Mitigation: Verify the source data, reporting period, department, employee scope, payroll cutoff, and approval chain before using the skill's output for decisions. <br>
