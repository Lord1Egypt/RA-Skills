### 1.2 渐变效应（关键！）

**旺衰不是全有或全无的开关，而是渐变的：**

- 行星**越接近**精确入庙度数，正面力量越强；在精确度数达到峰值后开始下降
- 行星**越接近**精确落陷度数，负面影响越重；远离精确点则影响减弱
- **实操判断**：需同时报告行星度数与精确旺衰度数的距离

**示例**：太阳在白羊座2.62° → 距精确入庙度10°差7.38° → 入庙力量中等偏弱（尚未到峰值）

## 二、行星生命阶段（Avastha by Degree）

行星的度数和所在星座的奇偶性决定了它的"生命阶段"，直接关系行星能否有效交付结果。

### 2.1 奇数星座（白羊、双子、狮子、天秤、射手、水瓶）

| 度数区间 | 阶段 | 力量 |
|---------|------|------|
| 0°–6° | 婴儿期（Infant） | 弱 |
| 6°–12° | 青年期（Young） | 中等 |
| **12°–18°** | **成年期（Adult）** | **最强** |
| 18°–24° | 老年期（Old） | 中等 |
| 24°–30° | 衰退期（Declining） | 弱 |

### 2.2 偶数星座（金牛、巨蟹、处女、天蝎、摩羯、双鱼）

**顺序完全相反！**

| 度数区间 | 阶段 | 力量 |
|---------|------|------|
| 24°–30° | 婴儿期（Infant） | 弱 |
| 18°–24° | 青年期（Young） | 中等 |
| **12°–18°** | **成年期（Adult）** | **最强** |
| 6°–12° | 老年期（Old） | 中等 |
| 0°–6° | 衰退期（Declining） | 弱 |

### 2.3 交界点（Sandhi Degrees）

- **0° 和 29°** 的行星处于极不稳定状态
- 即使是入庙行星，落在0°或29°时力量也会变得迟缓、不稳定
- 类比：正在搬家的人，既没脱离旧环境也没融入新环境

## 三、燃烧（Combustion/Astanga）

当行星距太阳太近时，被太阳强光"燃烧"，特质被削弱或掩盖。

### 3.1 精确燃烧度数阈值

| 行星 | 燃烧阈值（与太阳的距离） | 备注 |
|------|------------------------|------|
| 🌙 月亮 | **12°** | — |
| ♂ 火星 | **17°** | — |
| ☿ 水星 | **14°**（顺行） / **8°**（逆行） | 注：部分传统认为逆行为12°，Parashari标准为8° |
| ♃ 木星 | **11°** | — |
| ♀ 金星 | **10°**（顺行） / **8°**（逆行） | — |
| ♄ 土星 | **15°** | — |

**注意**：罗睺和计都作为阴影行星不受燃烧影响。

### 3.2 燃烧的影响

1. **星性减弱**：该行星主宰的宫位和天然象征无法充分发挥
2. **自我冲突**：太阳代表自我与权威，燃烧=该行星特质被自我问题遮蔽
3. **结果延迟**：成果会延迟或伴随困难才实现
4. **健康隐患**：该行星象征功能相关的健康脆弱性

### 3.3 燃烧减轻因素

| 减轻因素 | 说明 |
|---------|------|
| 逆行燃烧 | 逆行增加一定力量，减轻负面影响 |
| 入庙/旺相燃烧 | 固有力量可部分抵消燃烧 |
| 与太阳为友 | 与太阳友好的行星（木星、火星）损害较小 |
| 度数较远 | 处于燃烧范围边缘时影响较温和 |

### 3.4 判定步骤（实操）

1. 计算该行星与太阳的黄经差（绝对值）
2. 对照上表阈值
3. 若在阈值内 → 燃烧
4. 检查是否有减轻因素
5. 越接近太阳（度数差越小）→ 燃烧越严重

## 四、落陷取消（Neecha Bhanga）完整规则

### 4.1 五大取消条件（满足任一即触发）

| 编号 | 条件 | 说明 |
|------|------|------|
| 1 | **定位星在角宫** | 落陷行星所在星座的守护星（定位星），位于命宫的角宫（1/4/7/10） |
| 2 | **定位星相对月亮在角宫** | 落陷行星的定位星，位于月亮的角宫 |
| 3 | **互补入庙星在角宫** | 在该星座入庙的行星，位于命宫或月亮的角宫 |
| 4 | **与定位星产生关联** | 落陷行星与定位星合相或被定位星注视（产生相位） |
| 5 | **星座互换/互容** | 落陷行星与定位星之间发生星座互换 |

### 4.2 Parasara三法（P.V.R. Narasimha Rao 教学版）

| 方法 | 条件 | 效果强度 |
|------|------|---------|
| **逆行** | 落陷 + 逆行 | ≈ **实质等同于入庙**（最强取消！） |
| **Navamsha入庙** | D1落陷 + D9入庙 | 强取消 |
| **Kendra定位** | 在该星座入庙的行星或星座守护星从命宫/月亮起算在角宫 | 中等取消 |

### 4.3 特别注意

- **同座相会不算Neecha Bhanga！** 入庙行星必须从命宫或月亮起算在角宫，而非仅与落陷行星同座
- **逆行+入庙 = 糟糕！** 入庙行星如果同时逆行，反而会产生 Uchchabhanga（入庙被取消）
- **Dusthana宫主（6/8/12）落陷 = 好事！** 障碍本身被削弱了，不要去取消它的落陷
- **Dusthana宫主的Neecha Bhanga = 不可取！** 取消落陷等于加强了制造障碍的行星

### 4.4 从Neecha Bhanga升级为Raja Yoga的进阶条件

取消落陷 ≠ Raja Yoga。要真正形成Neecha Bhanga Raja Yoga，还需：
1. 该落陷行星掌管角宫或三方宫
2. 参与吉祥组合
3. 在强劲的大运/小运期被激活
4. 在分盘（D9/D10）中恢复力量
5. 对特定上升是功能吉星

## 六、丈夫征象星：木星 vs 火星

### 6.1 经典立场

| 角度 | 征象星 | 依据 |
|------|--------|------|
| **Parashara主流体系** | **木星** = 丈夫征象星 | 丈夫的传统角色（智者、引导者、达摩支柱）= 木星特质 |
| **少数派/现代** | **火星** = 爱情婚姻中的丈夫征象 | 现代婚姻中丈夫=平等伴侣（火星与金星平等） |

### 6.2 核心论证

**支持木星的理由：**
- 婚姻在吠陀文化中是**达摩关系**而非激情关系
- 丈夫 = 家庭之首、智慧决策者、灵性引导者 → 木星
- 火星 = 兄弟征象星，非丈夫征象星
- 木星关联第9宫（宗教/誓约），婚姻以誓约维系

**支持火星的场景：**
- 爱情婚姻（非包办）中，激情和吸引力是主要驱动力 → 火星
- 火星与金星是平等伴侣关系，更符合现代夫妻动态
- 部分南印学派使用火星作为女性盘的丈夫征象

### 6.3 实操建议

**多层分析法（推荐）**：不二选一，而是同时考虑两者，根据Sect和宫位权重分配：

| 条件 | 权重分配 |
|------|---------|
| 女性日间盘 | **木星**（入Sect，权重最高）+ 火星（出Sect，次之） |
| 女性夜间盘 | **木星**（出Sect，权重降低）+ **火星**（入Sect，权重升高） |
| 传统婚姻背景 | 木星权重↑ |
| 现代爱情婚姻 | 火星权重↑ |

## 九、DaraKaraka（DK）完整分析协议

### 9.1 计算方法
