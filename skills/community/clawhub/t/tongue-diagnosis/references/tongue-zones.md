# 舌部分区与脏腑对应

## 一、舌面分区

```
     ┌────── 舌尖 ──────┐   → 心肺
     │                   │
     │   舌边（左→肝）       │   → 肝胆（两侧）
     │   舌  中         │   → 脾胃
     │   舌边（右→胆）       │   → 肝胆（两侧）
     │                   │
     └────── 舌根 ──────┘   → 肾
```

### 舌面分区对应脏腑
| 分区 | 对应脏腑 | 临床参考 |
|------|----------|---------|
| 舌尖 | 心肺 | 心火亢盛 → 舌尖红赤/芒刺；心肺气虚 → 舌尖淡白 |
| 舌中 | 脾胃 | 脾胃湿热 → 舌中黄厚腻苔；脾虚 → 舌中淡白有齿痕 |
| 舌根 | 肾 | 肾阴虚 → 舌根无苔；肾阳虚 → 舌根淡白 |
| 舌边 | 肝胆（左肝右胆） | 肝胆火旺 → 舌边红赤；气滞血瘀 → 舌边青紫 |

## 二、舌诊原理

### 2.1 舌与经络的联系
- 手少阴心经之别 **系舌本**
- 足太阴脾经 **连舌本、散舌下**
- 足少阴肾经 **夹舌本**
- 足厥阴肝经 **络舌本**
- 脏腑发生病理变化，可以通过经络反映在舌象上

### 2.2 舌与气血津液
- 气血津液充足、运行正常 → 舌色淡红、舌苔湿润
- 气血不足 → 舌色淡白
- 津液不足 → 舌苔干燥
- 气滞血瘀 → 舌色紫暗

### 2.3 舌与心脏的特殊关系
- 「手少阴之别……循经入于心，系舌本」→ 心经将气血输送于舌
- 「心气通于舌，心和则舌能知五味矣」→ 舌的味觉与心相关
- 「夫舌为心窍」→ 强调心与舌的密切关系
- 心气充沛、血液充盈 → 舌色淡红润泽
- 心气不足、心血亏虚 → 舌色淡白
- 心血瘀阻 → 舌色青紫
- 心火炽盛 → 舌色红赤、口舌生疮

## 三、正常舌象
- **概括**：淡红舌，薄白苔
- **具体特征**：
  - 舌质淡红，鲜明润泽
  - 舌体运动灵活、大小适中
  - 舌苔薄白均匀
- **生理性变化**：可随年龄、体质、外界环境等因素有正常变异

## 四、舌诊方法

### 环境要求
- 诊室有充足的自然光线
- 室温适宜，不能过高或过低
- 自然光线不足时，可借助色温 6500K 左右的标准日光灯

### 患者要求
- 伸舌自然，舌尖略向下，舌面平展，尽量暴露

### 检查顺序
1. **先观察舌质** → 按舌尖、舌中、舌边、舌根的顺序
2. **再观察舌苔**
3. **最后观察舌下络脉**
4. 一次观察不清楚时，让患者休息片刻后继续
5. 必要时拍照保存，为动态分析提供依据
