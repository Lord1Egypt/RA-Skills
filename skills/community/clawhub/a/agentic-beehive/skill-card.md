This skill is ready for commercial/non-commercial use. <br>

Developers and agent builders use this skill to select an agent architecture posture, coordinate multi-agent work, and produce operational guidance, code-delivery handoffs, MCP server setup steps, or content-generation workflows. <br>

## Known Risks and Mitigations: <br>
Risk: The release includes exposed credential or session-shaped examples, personal paths, and hardcoded messaging details. <br>
Mitigation: Review and edit before installation; remove exposed keys or JWT-shaped examples, replace personal paths, and configure messaging recipients explicitly. <br>
Risk: The bundled AI market report tooling includes broad scraping and local persistence behavior. <br>
Mitigation: Restrict or disable scraping unless that report workflow is needed, review configured sources, and run only in an environment approved for the data being collected. <br>
Risk: The AutoGen Studio command can expose a service on 0.0.0.0. <br>
Mitigation: Bind local services to localhost or place them behind protected network controls and authentication before use. <br>
