# 示例：漫画 P1 样章门禁

## 适用场景

漫画 / 小白手册 / 连续图文页。

## 流程

1. 先完成完整文字版；
2. main 自审文字是否合理；
3. 收集主角图、角色参考、风格参考、Logo、二维码真实资产；
4. 只下发 P1 样章；
5. P1 审核：画风、人物、气泡、文字密度、品牌位；
6. P1 未确认，不得扩展全套；
7. P1 确认后，把 P1 标准写成后续页面金标准；
8. 扩展 P2-Pn；
9. 全图审核后进入 HTML；
10. HTML / ZIP 终审。

## P1 审核结论

- PASS：可扩展；
- NEEDS_REVISION：只返工 P1；
- BLOCKED：缺主角图 / 风格 / Logo / 文字未定稿。
