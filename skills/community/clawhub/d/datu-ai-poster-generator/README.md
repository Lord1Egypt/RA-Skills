# Digilifeform AI大图生成器

将文本、文档或已有图片快速转换为 4K 高质量信息海报，支持 AI 修图、继续修图、`9:16` / `9:21` / `1:3` / `3:1` 扩展比例、大图主流程 `magic_wand` 开关，以及可选的 `deep_research` 深度研究报告。

## 核心能力

| 能力 | 说明 | 成本 |
|---|---|---|
| 文本生成大图 | 直接从文本生成海报 | 5 积分(4K) |
| 文件上传生成 | 支持 PDF、Word、Markdown、图片等上传 | 5 积分(4K) |
| 深度研究 | 全网深度抓取并分析，生成图片同时提供研究报告文件 | 额外 5 积分，通常额外 5-15 分钟 |
| AI 修图 | 支持原图、标注图、参考图组合修图 | 3 积分(4K) |
| 比例支持 | `16:9`、`21:9`、`4:3`、`1:1`、`3:4`、`9:16`、`9:21`、`1:3`、`3:1` | 包含在生成流程中 |
| 魔法棒开关 | 设计化 Prompt 或原始 Prompt 直出 | 包含在大图流程中 |

## AI 修图命中规则

当用户提供已有图片并要求“改这张图、修改这张海报、基于原图修改、继续改图、换字、替换 logo、调整颜色、保留版式”等，应优先调用 AI 修图接口，而不是重新生成一张新图。

- 被修改的主图作为 `image`
- 标注图作为 `annotated_image`
- logo、二维码、风格图、产品图等参考资料作为 `ref_images`
- 只有用户明确要求“解析这张图片内容并生成新海报”时，图片才作为文件解析生成的内容素材

## 关键行为约束

- 当 `magic_wand=true` 时，先做上传前安全检查；检查通过后，不要由外层代理先对用户资料做摘要、提炼、压缩或改写
- 对普通业务资料，敏感信息检查通过后，把用户给的原始信息直接交给大图技能
- Datu 内部会自己完成信息提炼和设计化 Prompt 生成
- 如果用户选择 `deep_research=true`，任务完成后不仅返回图片，也应返回或下载 Word 研究报告文件
- `deep_research=true` 只能在用户明确选择或确认后启用
- 如果比例、清晰度、`magic_wand` 已明确，应直接创建任务，不要先输出“整理后的方案摘要”再执行

## 隐私与敏感数据

- 本技能会把文本、文件或图片发送到 `https://datu.digilifeform.com` 外部 API
- Required env vars: `DATU_API_KEY`
- Primary credential: `DATU_API_KEY`
- API 请求头：`X-API-Key`
- 宿主/IM 工具收到用户提供的 Datu API Key 后，应保存为 `DATU_API_KEY` 持久凭据，并在后续调用中自动注入
- 不要反复要求用户重新提供 API Key；只有持久凭据缺失、被删除、失效或认证失败时，才提示用户重新配置
- 如果飞书、微信、OpenClaw、ClawHub 等宿主反复索要 key，应优先检查宿主的凭据/密钥库/技能授权是否正确保存和注入了 `DATU_API_KEY`
- 对普通业务资料，上传前安全检查通过后，仍按技能规则提交用户给出的原始信息
- 如果内容明显包含密钥、口令、身份证号、银行卡号、医疗记录或受限数据，应先提醒并获得用户明确授权
- 风险控制不等于先做摘要；重点是“先提示、再授权”，用户不确认时应停止上传或请用户提供脱敏版本

## 大图生成工作模式

### `magic_wand=true`

- 由 Datu 内部完成信息提炼与设计化 Prompt 生成
- 适合长文、方案、政策、品牌传播内容
- 外层代理不要先做摘要或缩写；上传前安全检查通过后，再把用户资料保真提交给 Datu

### `magic_wand=false`

- 直接使用用户输入或文件解析文本
- 不再做大图方案改写
- 适合已经准备好 Prompt 的场景

说明：

- `magic_wand` 只作用于大图生成
- 修图仍使用独立的 `magic_think`

### `deep_research=true`

- 由 Datu 先进行全网深度抓取并分析，再进入大图生成流程
- 适合行业分析、政策解读、产品发布、市场洞察、趋势报告等需要补充背景资料的任务
- 通常需要额外 5-15 分钟，并额外消耗 5 积分
- 完成后应检查 `research_report_download_url`，并将 Word `.docx` 研究报告文件与图片结果一起交付给用户

## 质量说明

系统会统一给最终图像 Prompt 补充 4K 高清质量要求，以提升生成质量与稳定性。
## 新版本提醒

- 技能可低频检查 `GET /api/skill/version`
- 建议同一会话或同一用户 24 小时最多检查一次
- 如果发现有新版本，只在正常结果后顺手提醒一句，不打断主流程

## 发布者备用包

每次更新技能包后，发布者可运行 `python -m scripts.publish_skill_package --upload` 打包并上传 zip 到 `cdn.aimindschool.com`。如果用户私下反馈 ClawHub/OpenClaw 下载受限，可由发布者单独发送脚本输出的 CDN URL。
