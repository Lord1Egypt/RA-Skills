# Si Wu Tang Formula Application

## When to Use
当患者出现以下血虚证候时使用：
- 面色萎黄
- 头晕目眩
- 心悸失眠
- 唇爪无华
- 月经量少或闭经
- 舌淡、脉细

适用于血虚证、月经不调、产后调理及营血不足所致诸症。

## Execution Steps
1. **确认基础组成**：熟地黄、当归、白芍、川芎。
2. **理解配伍意义**：
   - 熟地黄为君药，滋阴补血；
   - 当归为臣药，补血活血；
   - 白芍养血敛阴，助补血之效；
   - 川芎活血行气，使补而不滞。
3. **掌握煎服方法**：常规水煎服，熟地黄宜先煎，川芎用量不宜过大以防辛燥伤阴。
4. **识别禁忌与加减**：
   - 阴虚内热或实热证慎用；
   - 脾胃虚弱易腹泻者需加减使用；
   - 可根据病情衍生为桃红四物汤（加桃仁、红花）或圣愈汤（加人参、黄芪）等变方。

> 注意：本方为补血调血基础方，必须根据具体证候进行个体化加减化裁。
