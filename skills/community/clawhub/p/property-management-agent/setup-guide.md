**Professional Property Setup (40 Minutes)**

1. Install ThumbGate: `npx thumbgate init --agent openclaw`
2. Load the skill: `openclaw skill load property-management-agent`
3. **Voice Desk:** Set up a professional receptionist voice via [ElevenLabs](https://elevenlabs.io/affiliates).
4. **CRM Sync:** Use [Make.com](https://make.com) to route maintenance tickets into Buildium, AppFolio, or Slack.
5. **Privacy Shield:** Ensure the ThumbGate privacy rules are active to protect door codes and sensitive owner data.

Test by submitting a "leaky faucet" request and verify the agent doesn't mark it as an emergency.
