# 行政专家 — 工具参考

## 行政效率指标

| 指标 | 计算公式 | 行业基准 |
|------|---------|---------|
| 流程周期时间 | 从发起至完成的总时长 | 报销<3天，采购<7天 |
| 人均行政费用 | 行政总费用 / 员工数 | 1-3万/年 |
| 空间利用率 | 实际使用面积 / 总面积 | 70-80% |
| SLA 达成率 | 按时完成的工单 / 总工单 | ≥95% |
| 员工满意度 | 调研评分（1-5分） | ≥4.0 |
