## Description: <br>
Generates a daily Markdown digest from stored memory notes, summarizing decisions, lessons, actions, and questions into a dated journal entry. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[pmaeter](https://clawhub.ai/user/pmaeter) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and ClawHub users use this skill to turn local daily memory notes into a structured Markdown digest for journaling, follow-up tracking, and review. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Private details from local memory notes may be carried into persistent digest files. <br>
Mitigation: Review generated journals before sharing, syncing, or publishing them. <br>
Risk: A recurring scheduler can automatically summarize sensitive memory notes without a fresh manual review. <br>
Mitigation: Enable cron or other scheduled execution only after confirming the local memory contents and destination are appropriate. <br>
