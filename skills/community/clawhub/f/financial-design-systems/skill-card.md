## Description: <br>
Patterns for building dark-themed financial charts and data visualizations, including chart theming, gain/loss color scales, and real-time data display for trading dashboards and financial analytics. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wpank](https://clawhub.ai/user/wpank) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and designers use this skill to create dark-themed financial chart interfaces for trading dashboards, portfolio views, price histories, and financial analytics products. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The README includes an npx installation command from a GitHub URL. <br>
Mitigation: Prefer the reviewed ClawHub package, or inspect and pin the source before using the command, especially for global installation. <br>
Risk: Financial chart designs can over-rely on red and green encodings. <br>
Mitigation: Use the skill's guidance to add shapes, patterns, labels, or other non-color cues for accessibility. <br>
