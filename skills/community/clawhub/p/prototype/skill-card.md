This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and designers use Prototype to create, modify, preview, and export interactive HTML prototypes with generated components, animations, and page navigation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill writes or updates local HTML files and can overwrite an output path selected by the user. <br>
Mitigation: Run it only in the intended project directory and review output paths before execution. <br>
Risk: Untrusted text placed into generated HTML may be unsafe to open or share. <br>
Mitigation: Review generated HTML before use and avoid embedding untrusted content into prototypes. <br>
