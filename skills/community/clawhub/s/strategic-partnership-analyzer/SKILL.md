
### 名称
`executive-speech-analyzer`

### 定位（用户可见）
搜集并剖析企业核心高管近期的公开言论，精准洞察其当前的战略重心与业务痛点，为您生成靶向拜访话术。

### 描述（LLM识别）
精通企业战略分析与高管心理洞察的商务情报专家。当用户需要分析某企业董事长/CEO近期关注焦点、准备拜访前获取战略意图与话术建议、或了解领导层对行业风口的态度时激活此技能。

执行逻辑：
1. 提取用户指令中的“企业名称”与“高管姓名”实体。
2. 触发联网搜索：定向检索该高管在过去12-24个月内的媒体专访、峰会演讲、财报会议实录及公开文章。
3. 文本挖掘：利用阅读理解能力，提取高频关键词，识别“业务焦虑（痛点）”与“战略野心（需求）”。
4. 策略生成：结合用户身份（如招商人员、投资人、销售），将高管诉求转化为具体沟通切入点与谈判话术。

### 适用场景
- 招商人员拜访目标企业前，准备精准破冰话术
- 投资人评估企业战略方向与创始人认知
- 销售团队针对高管痛点定制解决方案提案
- 竞争对手分析高管最新表态中的信号

### 输入参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `company_name` | string | 是 | 目标企业名称 |
| `executive_name` | string | 是 | 高管姓名（支持“CEO”“董事长”等代称，需自动映射） |
| `time_range` | int | 否 | 搜索时间范围（月），默认12个月 |
| `user_role` | string | 否 | 用户身份，如“招商人员”“投资人”“销售”，默认“招商人员” |
| `target_region` | string | 否 | 用户所在区域（用于话术中加入区域优势） |

### 输出格式
输出一份结构化《高管情报与谈判白皮书》，包含以下模块：

1. **高管基础画像**：姓名、职位、任职时间、简要背景
2. **近期公开发言核心观点提炼**（按时间倒序排列，每项附来源及日期）
3. **高管当前最关注的3大核心诉求**（分类：资金/市场/技术/政策/人才/供应链，标注优先级）
4. **痛点与野心分析**：业务焦虑点 + 战略扩张方向
5. **沟通切入点建议**：针对用户角色的2-3个破冰话题
6. **核心谈判话术**：模拟对话示例，可直接使用
7. **禁忌与风险提示**：应避免提及的内容

### 执行流程（内部步骤）

1. **实体提取**  
   - 从用户输入中识别 `company_name` 和 `executive_name`。  
   - 若用户仅提供“某公司CEO”，需联网搜索该公司现任CEO姓名（如“华为 CEO” → 获取当前实际高管名）。  
   - 支持中英文名模糊匹配（如“Pony马” → 马化腾）。

2. **联网搜索策略**（使用 `web_search` 工具）  
   - 搜索关键词组合示例：
     - `"{高管姓名} {公司名称} 专访 2025"`
     - `"{高管姓名} 演讲 2026"`
     - `"{公司名称} 财报会议 实录 2026"`
     - `"{高管姓名} 论坛 发言 2025"`
     - `"{公司名称} 战略发布会 2026"`
   - 限定时间范围（如 `after:2025-05-01`）  
   - 优先级：权威财经媒体（36氪、虎嗅、第一财经、新浪财经）、公司官网新闻中心、行业协会峰会报道。

3. **文本挖掘与关键词提取**  
   - 对搜集到的文本进行高频词统计（排除停用词）。  
   - 使用LLM阅读理解，识别两类信号：  
     - **痛点信号**：抱怨、瓶颈、成本、挑战、竞争、合规、人才短缺等  
     - **野心信号**：扩张、布局、第一、引领、百亿、千亿、出海、科创板、生态等  
   - 综合提炼出3个核心诉求，并标注对应的证据语句（直接引用）。

4. **策略生成**  
   - 根据 `user_role` 定制话术方向：  
     - 招商人员：侧重区域政策、土地用工成本、本地市场订单、产业配套  
     - 投资人：侧重增长空间、盈利模型、融资需求、估值逻辑  
     - 销售：侧重痛点解决方案、ROI、标杆案例  
   - 若提供了 `target_region`，话术中融入该区域的独特优势（如“我们区有低空经济专项补贴”）。

5. **输出报告**  
   - 按Markdown格式组织，附上所有引用来源的链接或标题。  
   - 话术部分用引用块或代码块突出，便于复制。

### 搜索与引用规范
- 每个观点必须附带至少一个来源（媒体名称+日期）。  
- 若某信息多渠道验证，标注“据XX、XX等多家媒体报道”。  
- 避免引用未署名的自媒体或谣言站点。  
- 若搜索不到足够内容，返回“公开信息有限，仅能基于以下X条结果分析”并列出实际结果。

### 错误处理
| 错误类型 | 处理方式 |
|---------|----------|
| 未找到该高管任何近期公开言论 | 返回提示，建议缩短时间范围或更换高管姓名 |
| 搜索返回内容过少（<3篇） | 标注“信息有限，以下分析可能存在偏差”，仍输出已有内容 |
| 高管姓名与公司名称不匹配 | 提示用户确认，并给出推测的正确姓名（如“您是否指XX?”） |

### 使用示例

**用户输入**：
```
我要去拜访边界智控的CEO潘亮，帮我分析一下他最近关注什么，我是江门蓬江区招商局的。
```

**Agent输出**（节选）：

```markdown
# 高管情报与谈判白皮书

**目标**：潘亮（边界智控创始人/CEO）  
**用户角色**：江门蓬江区招商人员  
**分析时段**：2025年5月 - 2026年5月

## 一、近期公开发言核心观点

1. **2026年5月14日，B轮融资发布会**  
   - “本轮融资将主要用于珠江西岸产能扩张，我们正在考察深圳周边的低成本制造基地。”  
   - “飞控系统的成本大头在于芯片和测试，希望能在新基地附近找到传感器配套企业。”  
   （来源：36氪）

2. **2026年3月，低空经济产业峰会**  
   - “适航认证周期太长，希望地方政府能帮助对接民航局资源，缩短取证时间。”  
   - “我们缺的不是订单，而是有经验的飞控测试工程师。”  

## 二、当前最关注的3大核心诉求

| 优先级 | 诉求类别 | 具体描述 | 证据 |
|--------|----------|----------|------|
| 1 | 成本 | 寻找珠西低成本的制造基地，降低综合落地成本 | 融资发布会原话 |
| 2 | 人才 | 急需飞控测试、嵌入式开发工程师 | 峰会发言提及 |
| 3 | 政策 | 希望获得适航取证专项支持 | 同上 |

## 三、痛点与野心

- **痛点**：深圳综合成本过高，制约30万套产能落地；适航取证流程慢，影响eVTOL客户交付节奏。  
- **野心**：两年内成为国内低空飞控市占率第一；将珠西基地建成辐射东南亚的出口中心（据2025年底内部信推测）。

## 四、破冰沟通切入点（针对招商人员）

1. **从成本痛点切入**：  
   “潘总，我们注意到您在B轮融资时提到珠西产能扩张。蓬江工业地价仅为深圳1/5，我们准备了1500亩连片熟地，可以为您定制代建厂房。”

2. **从人才痛点切入**：  
   “关于飞控测试工程师短缺，蓬江有五邑大学和职业技术学院，我们可以开设定向培养班，毕业生优先供您选拔。”

3. **从适航政策切入**：  
   “对于适航取证，广东省有50亿元低空经济专项，我们可以协助对接民航中南地区管理局，申请绿色通道。”

## 五、核心谈判话术

> **开场白（10秒抓住注意力）**：  
> “潘总，我是江门蓬江区招商局的。我们看了您B轮融资的报道，知道您正在找珠西的制造基地。蓬江可以做到：拿地即开工，第一年综合成本比深圳低40%，还有本地8000万的飞控订单等您接手。您下周方便来实地看看吗？”

> **应对“我要对比周边城市”**：  
> “您当然应该对比。但只有蓬江，能让您落地第一天就有本地客户——擎动千机和保卫鹰已经同意将您作为首选供应商，这是他们签好的采购意向函。另外我们的供应链配套，50公里内能找到航电机壳和线束供应商，这是中山没有的。”

## 六、禁忌与风险提示

- 避免主动提及“你们在深圳有什么不好”，容易引发防御心理。  
- 不要在初次见面时追问融资估值或财务细节。  
- 如果潘亮表现出对环保或ESG的关注，请准备好蓬江绿电园区资料（据观察他曾点赞过碳中和文章）。

> 数据截止至2026年5月22日，信息来源：36氪、新浪财经、低空经济产业联盟官网。
```
