# TCM Oncology Differentiation Four Key Points

## When to Use
- 已通过现代医学手段明确肿瘤的解剖部位、病理类型、临床分期及预后；
- 已完成中医四诊（望、闻、问、切）合参，获得完整证候信息；
- 不适用于急诊抢救或仅针对单一症状的临时对症处理。

## Execution Steps

### 1. 辨病和辨证相结合
- **辨病**：依据影像学、病理学等现代医学检查，明确肿瘤的生物学特性；
- **辨证**：运用中医理论分析证候类型（如气滞血瘀、痰湿凝聚、阴虚火旺等），实现基于病种与证型的分类分期论治。

### 2. 辨阴阳虚实
- **初期**：以邪实为主，正气未明显受损，治疗应以祛邪为主；
- **中晚期**：正虚显著（如气虚、阴伤、气血不足、阴阳两虚），同时邪实仍盛，需采用攻补兼施策略。

### 3. 辨标本缓急
- **急则治标**：针对咳嗽、发热、疼痛、出血等紧急症状，采用镇咳化痰、通腑退热、止痛安神等法；
- **缓则治本**：以扶正培本（如健脾益肾、益气养阴）、活血化瘀、软坚散结为核心；
- **中晚期**：多采用“标本兼顾”原则，避免偏颇。

### 4. 辨局部和整体
- 认识局部病灶对全身的影响（如压迫、消耗、转移）；
- 评估全身状态对局部治疗效果的决定性作用；
- 治疗上坚持“局部攻邪 + 全身扶正”相结合，体现“以人为本”的中医核心思想。

### 特别提示
- 在患者接受放化疗期间，因西医“攻”法副作用较大，应侧重扶正调理，以增强疗效、减轻毒副反应。

## Output
制定个体化的中医治疗方案，明确扶正与祛邪的主次关系、实施先后顺序及药物配伍策略。
