# 内置类比库

处理选题时，优先从以下类比库中匹配。

## 政治与权力类比

| 历史/知识场景 | 现代类比 |
|-------------|---------|
| 王朝管理 | 公司运营（CEO、股东、KPI、融资、破产） |
| 权力斗争 | 办公室政治（站队、背锅、抢功劳） |
| 变法失败 | 改革触动老部门利益（裁员、重组、架构调整） |
| 宦官专权 | CEO被秘书/助理架空 |
| 藩镇割据 | 加盟店失控、分公司不听总部 |
| 朝堂争论 | 董事会吵架 |

## 自创类比原则

如果类比库中没有合适项，自创类比必须符合：

1. **3秒能懂** - 听的人不需要额外背景知识
2. **画面感强** - 能直接在脑海中形成画面
3. **贯穿全篇** - 这个类比可以从头用到尾
4. **反差萌** - 古代场景用现代梗，有喜剧效果
