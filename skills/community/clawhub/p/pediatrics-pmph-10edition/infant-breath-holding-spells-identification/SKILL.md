# 婴幼儿屏气发作识别与家庭干预

## 何时使用
当婴幼儿（6～18个月）或学龄前儿童（5岁前）在情绪急剧变化（如发怒、恐惧、剧痛、剧烈叫喊）后出现以下表现时：
- 呼吸暂停
- 口唇发绀（青紫）
- 意识丧失或昏厥
- 躯干四肢挺直或轻微抽动
- 发作持续约30秒至1分钟，随后自行恢复

**注意**：必须先排除癫痫等器质性神经系统疾病，不可用于已确诊癫痫或其他神经系统异常的患儿。

## 如何执行

### 1. 识别典型表现
- **诱因**：情绪剧烈波动（哭闹、愤怒、惊吓、疼痛）
- **机制**：哭喊导致换气过度 → 呼吸中枢暂时抑制 → 呼吸暂停
- **症状序列**：哭喊 → 呼吸停止 → 口唇发绀 → 意识丧失 → 肢体强直/抽动 → 自行恢复（呼吸恢复、面色转红、清醒）
- **持续时间**：通常0.5～1分钟
- **频率**：一日可发作数次

### 2. 评估性格特征
观察患儿是否具有以下行为倾向：
- 性格暴躁
- 任性
- 易发脾气

### 3. 实施家庭干预
- 遇到冲突时，采用耐心解释代替打骂或惩罚
- 主动减少诱发哭闹和情绪爆发的情境
- 建立稳定、一致的家庭教养环境，促进情绪调节能力发展

### 4. 鉴别诊断提醒
- 屏气发作为良性自限性行为现象，**非癫痫**
- 若发作无明确情绪诱因、持续时间过长（>2分钟）、发作后意识恢复缓慢，或伴有其他神经系统异常，应建议就医排除器质性疾病

> **输出目标**：指导家长实施非惩罚性教养策略，预防发作复发，并支持儿童情绪健康发展。
