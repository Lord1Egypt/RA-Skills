# 劳动合同审查技能

## 概述

本技能为劳动者提供劳动合同逐条审查与风险标注服务，覆盖以下核心功能：
1. **必备条款审查** — 检查《劳动合同法》第17条规定的8项必备条款是否齐全、合规
2. **风险条款识别** — 识别合同中的违法条款、霸王条款和潜在风险条款
3. **试用期审查** — 试用期时长、工资、社保合规性检查
4. **薪资条款分析** — 薪资结构、加班费基数、年终奖条款风险评估
5. **竞业限制审查** — 适用人员、期限、范围、补偿金合法性检查
6. **审查报告生成** — 输出结构化审查报告，含风险等级和修改建议

---

## 工作流程

### 总流程

```
用户提供合同 → 逐条解析 → 必备条款检查 → 风险条款识别 → 生成审查报告
```

### 阶段一：合同信息提取

收集以下信息（通过用户上传合同文本/照片或口述）：

**必采信息：**
1. 用人单位全称
2. 劳动者姓名
3. 合同期限（起止日期）
4. 试用期（时长/起止日期）
5. 工作岗位和工作地点
6. 薪资结构（基本工资/绩效/补贴等）
7. 工作时间制度
8. 社保条款
9. 竞业限制条款（如有）
10. 其他特殊条款（培训/保密/服务等）

### 阶段二：必备条款审查

加载参考文档：`references/contract_review_guide.md`

逐项检查《劳动合同法》第17条规定的8项必备条款：
1. 用人单位信息（名称、法定代表人、地址）
2. 劳动者信息（姓名、身份证号、住址）
3. 合同期限
4. 工作内容和工作地点
5. 工作时间和休息休假
6. 劳动报酬
7. 社会保险
8. 劳动保护/劳动条件/职业危害防护

### 阶段三：风险条款识别

加载参考文档：`references/risk_clauses.md`

逐条审查合同条款，标注风险等级：

- 🔴 **高风险** — 违法条款或严重损害劳动者权益，可能导致重大损失
- 🟡 **中风险** — 条款有瑕疵但不当然违法，需关注后续影响
- 🟢 **低风险** — 条款基本合规，仅有轻微注意事项

### 阶段四：专项审查

根据合同内容进行专项审查：

**试用期专项：**
- 时长是否符合法定上限
- 工资是否≥转正工资80%
- 是否约定两次试用期
- 试用期是否不缴社保

**薪资专项：**
- 基本工资是否过低（影响经济补偿金基数）
- 是否有"公司有权调整薪资"条款
- 加班费计算基数是否合理
- 年终奖发放条件是否明确

**竞业限制专项：**
- 适用人员是否符合法定范围
- 期限是否≤2年
- 补偿金是否≥月工资30%
- 范围是否合理

**解除/终止专项：**
- 是否有"随时解除"条款
- 离职通知期是否超过30日
- 是否有"离职扣工资"条款
- 是否有排除劳动者法定权利的条款

### 阶段五：生成审查报告

输出结构化审查报告，包含：
1. 合同基本信息概览
2. 必备条款完整性检查表
3. 风险条款清单（按风险等级排列）
4. 专项审查结果
5. 修改建议
6. 签约注意事项

## 关键规则与提醒

### 法律依据快速索引

| 审查事项 | 法律依据 |
|---------|---------|
| 必备条款 | 《劳动合同法》第17条 |
| 试用期 | 《劳动合同法》第19条、第20条、第21条 |
| 薪资 | 《劳动合同法》第18条、第20条 |
| 竞业限制 | 《劳动合同法》第23条、第24条 |
| 服务期 | 《劳动合同法》第22条 |
| 合同解除 | 《劳动合同法》第36-41条 |
| 违法条款效力 | 《劳动合同法》第26条、第27条 |

### 使用注意事项
1. 本技能提供的信息仅供参考，不构成法律意见
2. 各地裁判口径可能不同，建议咨询当地律师
3. 审查发现违法条款不代表合同整体无效，仅违法条款无效
4. 保留合同原件是维权的基础，签约前务必拍照/复印留存
5. 合同空白处应划掉或填"无"，防止事后添加
