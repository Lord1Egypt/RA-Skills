## 第二章：关键比率对比

### 2.1 比率计算

```markdown
| 序号 | 收藏率 | 点赞率 | 投币率 | 分享率 | 综合评分 |
|------|--------|--------|--------|--------|----------|
| 1    |        |        |        |        |          |
| 2    |        |        |        |        |          |
| 3    |        |        |        |        |          |
| ...  |        |        |        |        |          |

【计算公式】
收藏率 = 收藏数 ÷ 播放量 × 100%
点赞率 = 点赞数 ÷ 播放量 × 100%
投币率 = 投币数 ÷ 播放量 × 100%
分享率 = 分享数 ÷ 播放量 × 100%

【综合评分计算】
综合评分 = 收藏率×0.4 + 点赞率×0.3 + 投币率×0.2 + 分享率×0.1
（权重可根据业务调整）
```

## 📝 示例：珠宝工艺类竞品分析

```
【参评视频】
- 视频A: "清代宫廷金镯复原全过程" - 见筑师 - 114万播放 - 收藏率5.6%
- 视频B: "花丝镶嵌工艺揭秘" - 工艺大师 - 89万播放 - 收藏率5.2%
- 视频C: "古法金选购指南" - 珠宝鉴定师 - 156万播放 - 收藏率4.8%
- 视频D: "錾刻绝技展示" - 非遗传承人 - 67万播放 - 收藏率4.5%

【核心发现】
1. 历史文化类收藏率最高（5.2-5.6%）
2. 选购实操类播放量最高但收藏率偏低（4.8%）
3. 工匠/工艺类内容稀缺但效果好

【差异化机会】
历史文化 + 现代选购 = 蓝海组合

【入场建议】
✅ 强烈建议入场
赛道成熟度: 18/25
理由: 市场需求强、竞争适中、内容稀缺
```
