# 文风规范与保持指南

## 核心原则

文风是小说的灵魂。涟漪修改和章节生成必须保持原文风格的一致性，不能因为修改或续写导致"换了一个作者"的感觉。

## 文风维度

### 1. 叙事节奏
- **快节奏**（热血/爽文）：短句为主，动作密集，少心理描写
- **中等**：叙事与描写平衡，张弛有度
- **慢热**：长句铺陈，大量环境与心理描写，氛围感重

### 2. 语言风格
- **古风**：半文半白，用词典雅，善用典故
- **现代**：口语化，接地气，网络用语可适当使用
- **文艺**：比喻丰富，修辞密集，注重音韵
- **硬核**：术语密集，逻辑严密，少修饰

### 3. 情感表达
- **含蓄**：不直说情感，通过行为和环境暗示
- **直白**：角色情感直接表达，内心独白丰富
- **克制**：情感点到即止，留白多

### 4. 视角处理
- **第一人称**：完全代入主角，信息受限
- **第三人称限制**：跟随主角视角，偶尔切换
- **第三人称全知**：上帝视角，可切换多角色
- **多视角**：章/节级别切换视角角色

## 文风保持策略

### 生成时

1. **提取文风样本**：从 project.yaml 的 style.sample 字段获取
2. **分析文风特征**：
   - 平均句长
   - 修辞密度（比喻/排比/对仗的出现频率）
   - 心理描写占比
   - 对话风格（简洁/华丽/口语/文雅）
   - 常用意象和比喻域
3. **生成指令注入**：在LLM上下文中明确标注文风要求

### 重写时

1. **最小修改原则**：只改矛盾部分，不改风格
2. **风格对齐**：重写段落必须与前后段落的风格一致
3. **句式模仿**：重写时模仿原文的句式结构和节奏
4. **词汇一致性**：使用原文中出现过的同类词汇，不引入风格外的新词

### 续写时

1. **衔接检测**：续写开头必须与前文结尾自然衔接
2. **节奏延续**：保持前文的叙事节奏，不突然变快或变慢
3. **角色语言一致**：角色的说话方式、口头禅必须与已建立的一致

## 文风样本提取方法

当 project.yaml 的 style.sample 为空时，从前3章中自动提取：

```
分析内容：
1. 句式结构：主句平均长度、从句使用频率、断句习惯
2. 修辞手法：比喻类型（自然/战争/日常）、排比频率、环境描写风格
3. 对话风格：平均对话长度、语气词使用、方言/口音标记
4. 叙事距离：近距（内心独白多）vs 远距（客观描述多）
5. 情感浓度：克制 vs 饱满
```

提取后写入 style.sample，供后续所有生成和重写使用。

### 《凡人修仙传》— 忘语文笔特征

**定位：朴实细腻、简练传神的"凡人流"开创者**

#### 核心文风指纹

| 维度 | 特征 | 典型表现 |
|------|------|---------|
| 叙事节奏 | 慢热稳步 | 前期铺垫长，升级循序渐进，张弛节奏="观察→潜伏→爆发"三段式 |
| 语言风格 | 半文半白+白描 | 不堆辞藻，细节精准；四字句/对仗句频繁，但日常对话保持现代口语 |
| 心理描写 | 理性算计型极高频 | 每章15-20次"韩立心想/暗道"，永远是利弊分析+风险评估+机会捕捉 |
| 环境描写 | 克制服务型 | 渐进式铺陈（远景→近景→细节），不喧宾夺主，善在环境描写中埋伏笔 |
| 对话风格 | 高度口语化+谨慎型 | "晚辈""前辈""道友"等修仙界通用称呼，韩立说话永远留有余地 |
| 战斗描写 | 策略性>动作性 | 重法宝轻招式，重效果轻过程，重心理轻动作——打得过打，打不过跑 |
| 信息释放 | 层层递进型 | 伏笔可跨百万字回收，"千里伏笔"；主角视角限制，最后知真相 |
| 独特标记 | "韩跑跑"模式 | "韩立眉头一皱，退至众人身后"——遇险先跑，摸尸必富，团灭发动机 |

#### 写作时可复用的文笔模式

**1. 心理独白模板（凡人流核心）**
```
<角色>微微一怔，随即眉头皱起，心中迅速盘算：
- <对自身利弊的判断>
- <风险等级评估>
- <可行策略选择>
```

**2. 战斗节奏模板**
```
观察对手（3-5句判断修为/法宝/弱点）
→ 选择策略（内心独白：硬拼/偷袭/撤退）
→ 执行动作（密集描写法宝激发/符箓激活）
→ 结果收尾（胜/败/平，搜刮储物袋）
```

**3. 环境铺陈模板**
```
远景勾勒（空间范围，一句话）
→ 近景聚焦（具体场景，2-3句）
→ 细节注入（灵气/建筑/天气+暗示性伏笔，1-2句）
```

#### 凡人流写作禁忌
- ❌ 主角冲动行事、热血上头
- ❌ 无因果链的突然升级
- ❌ 过度煽情的情感描写
- ❌ 全知视角透露主角不知的信息
- ✅ 永远给主角留后手
- ✅ 升级有清晰因果链
- ✅ 情感克制，行为驱动
- ✅ 信息差制造悬念

### 《诡秘之主》— 爱潜水的乌贼文笔特征

**定位：维多利亚翻译体+克苏鲁信息恐怖+吐槽式内心独白的暗黑史诗**

#### 核心文风指纹

| 维度 | 特征 | 典型表现 |
|------|------|---------|
| 叙事节奏 | 多线并进+爆点收束 | 章节内多线程推进（值夜者/塔罗会/调查），结尾汇总释放 |
| 语言风格 | 维多利亚翻译体 | 欧化长句+半文言词汇+被动语态，刻意营造异域感 |
| 心理描写 | 吐槽式内心直播 | 克莱恩几乎不间断的思维直播，"表里两层"叙事张力 |
| 环境描写 | 电影镜头式+多感官 | 静态铺陈（空间布局/光影/物品），蒸汽朋克+哥特+克苏鲁意象体系 |
| 对话风格 | 阶层化语言差异 | 贵族优雅/底层朴素/教会庄重/各途径者各有气质 |
| 恐怖描写 | "不可言说"策略 | 不正面描写恐怖，通过角色反应/环境异常/他人警告侧面传达 |
| 信息释放 | 渐进式+假信息干扰 | 像剥洋葱一样揭示真相，不可靠叙述者故意误导读者 |
| 独特标记 | SCP式物品档案 | 封印物格式化介绍（名称-序列-外形-能力-负面效果） |

#### 写作时可复用的文笔模式

**1. 维多利亚翻译体句式模板**
```
<主语> <被动/倒装结构>，<长定语从句>，<多重修饰>。
→ 示例："那本被层层封印的、散发着微弱灵性波动的皮面笔记，被小心翼翼地推到了桌面中央。"
```

**2. 克苏鲁式恐怖描写模板**
```
日常场景中渗透异常（一次普通占卜/一页笔记/一个梦境）
→ 角色细微反应（手指颤动/瞳孔收缩/呼吸急促）
→ 环境异变暗示（阴影蠕动/空气变冷/声音扭曲）
→ 留白（不解释全貌，让读者自己想象最可怕的可能）
```

**3. 吐槽式内心独白模板**
```
<表面行为/庄重场景>
→ <内心吐槽/世俗化评论>（制造反差喜感）
→ 示例：面对神灵般存在时心里想的却是"这要花多少钱"
```

**4. 封印物/SCP式物品档案模板**
```
【名称】<物品名> — 序列<等级>
- 外形：<客观描写>
- 能力：<规则化描述>
- 负面效果：<代价/限制>
```

**5. 信息差叙事模板**
```
A角色知道某信息，B角色不知道
→ 读者通过A的视角知道真相
→ B的行动因信息缺失而产生误判
→ 读者产生"不要啊！"的紧张感
```

#### 诡秘体写作禁忌
- ❌ 正面描写不可名状之物（必须侧面暗示）
- ❌ 破坏叙事基调的过度幽默（吐槽只在内心，外在永远谨慎）
- ❌ 一次性倾泻大量设定（必须渐进式释放）
- ❌ 违反已建立规则的情节
- ✅ 日常中渗透恐怖
- ✅ 仪式感描写（烛光/香料/咒文/肃穆笔调）
- ✅ 多视角拼图式叙事
- ✅ 用现代概念类比超凡现象（"核弹论""大气污染"）

## 两部作品的文笔对比速查

| 维度 | 凡人修仙传 | 诡秘之主 |
|------|-----------|---------|
| 句式 | 短中句为主，白描简练 | 长句为主，欧化翻译体 |
| 心理描写 | 理性算计，利弊分析 | 吐槽式直播，表里两层 |
| 恐怖/危险 | 不渲染恐惧，冷静应对 | 克苏鲁式侧面暗示，氛围浓重 |
| 幽默 | 几乎无（严肃叙事） | 大量内心吐槽+罗塞尔日记梗 |
| 信息控制 | 主角视角限制（最后知真相） | 多视角拼图+不可靠叙述者 |
| 战斗 | 策略博弈，一击定胜负 | 信息博弈，陷阱布局，棋局式 |
| 世界观呈现 | 递进式探索 | 渐进式揭秘+格式化说明 |
| 核心氛围 | 冷静理性，凡人挣扎求生 | 神秘压抑，理智与疯狂一线之隔 |
| 适用题材 | 仙侠/玄幻/修真 | 悬疑/暗黑奇幻/克苏鲁/蒸汽朋克 |
