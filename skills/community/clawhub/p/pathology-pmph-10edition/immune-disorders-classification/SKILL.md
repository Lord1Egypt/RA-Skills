# 免疫相关疾病分类与机制

## 适用场景
- 自幼反复感染、易患肿瘤或自身免疫病
- 实体器官移植后出现功能异常
- 怀疑自身免疫病但需明确机制

## 执行步骤

### 一、原发性免疫缺陷病分类
确认为先天性（非HIV/营养不良等继发）后，按受损成分分类：
1. **体液免疫缺陷为主**：B细胞/抗体缺陷（如丙种球蛋白缺乏症）
2. **细胞免疫缺陷为主**：T细胞功能障碍（如DiGeorge综合征）
3. **联合免疫缺陷**：T+B均受损（如重症联合免疫缺陷病）
4. **非特异性免疫缺陷**：吞噬细胞、补体或自身炎症性疾病

### 二、实体器官移植排斥分型
根据时间窗与病理特征判断：
- **超急性**（数分钟–数小时）：预存抗体→Ⅲ型超敏反应→广泛血栓+坏死
- **急性**（数天–数月）：
  - 细胞型：T细胞浸润+肾小管炎
  - 血管型：抗体介导→内膜增厚
- **慢性**（>1年）：血管纤维化+间质纤维化，药物难治

### 三、免疫耐受丧失机制
若出现以下任一情况，警惕自身免疫病：
1. T细胞获得异常协同刺激（如B7表达升高）
2. 活化诱导凋亡（AICD）失效（Bim/Fas通路异常）
3. 调节性T细胞（Tr）与辅助性T细胞（Th）失衡

> **关键提示**：肝移植中超急性排斥罕见；慢性排斥是长期存活主要障碍。
