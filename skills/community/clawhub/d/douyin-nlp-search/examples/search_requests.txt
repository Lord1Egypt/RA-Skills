# Douyin Scraper - 自然语言搜索请求示例
# 以下是所有支持的自然语言查询格式示例

# === 基础搜索 ===
搜索一下海鲜视频
找一下猫咪视频
搜一下美食教程
查找健身相关视频
帮我找旅游攻略视频

# === 带数量的搜索 ===
找5个海鲜视频
搜索10个猫咪搞笑视频
查找3个最新的美食视频

# === 带排序的搜索 ===
找最热的海鲜视频
搜索最新的科技资讯
找点赞最多的健身教程

# === 组合条件搜索 ===
找5个最热美食视频
搜索10个最新科技资讯视频
找3个点赞最多的猫咪搞笑视频

# === 更自然的表达 ===
抖音上有什么好吃的海鲜视频吗
帮我搜一下猫咪的搞笑短视频
有没有好玩的游戏视频推荐
想看看做饭的教程视频

# === 预期行为 ===
# 所有以上查询都会被自动解析为:
# - keyword: 搜索关键词 (海鲜、猫咪、美食等)
# - count: 返回数量，默认10条
# - sort: 排序方式 general(综合)/latest(最新)/most_liked(最热)
