# 著作权登记申请指引

## 适用场景
对作品进行著作权登记，获取官方登记证书，便于维权时证明权利归属。

## 登记机构
中国版权保护中心（www.ccopyright.com.cn）

## 申请流程

### 第一步：准备材料
| 材料 | 要求 |
|------|------|
| 作品登记申请表 | 从版权保护中心网站下载 |
| 作品样本 | 完整作品复制件（电子或纸质） |
| 身份证明 | 个人：身份证复印件 / 企业：营业执照复印件 |
| 权利归属证明 | 如非本人创作：委托合同、职务创作证明等 |
| 作品说明书 | 简要说明创作意图、过程、独创性 |

### 第二步：在线/线下提交
- 在线：登录中国版权保护中心 → 著作权登记 → 作品登记
- 线下：邮寄或亲临办理

### 第三步：审查看审查 与 缴费
- 审查周期：一般30个工作日
- 费用：依作品类型不同，约100-2000元

## 适用作品类型
- 文字作品（文章/小说/诗歌/剧本）
- 美术作品（绘画/书法/雕塑）
- 摄影作品
- 视听作品（视频/电影/动画）
- 音乐作品（词/曲）
- 计算机软件（另走软件著作权登记）
- 设计图/地图/示意图等

## 注意事项
- 著作权自动产生（创作完成即享有），登记是自愿的，但登记证书是最有力的权利证明
- 登记证书是维权诉讼中的重要证据
- 软件著作权走单独通道（中国版权保护中心软件登记部）
- 作品说明书要体现独创性——你的作品到底「独」在哪里
