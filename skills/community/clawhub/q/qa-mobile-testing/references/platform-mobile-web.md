# 移动Web/H5测试专项

## 测试维度

### 1. 响应式测试

```
├─ 布局适配
│   ├─ 流式布局：宽度自适应
│   ├─ 弹性布局：Flexbox/Grid
│   ├─ 媒体查询：断点适配
│   └─ 单位适配：rem/vw/vh
│
├─ 屏幕适配
│   ├─ 手机竖屏：320px-480px
│   ├─ 手机横屏：480px-768px
│   ├─ 平板竖屏：768px-1024px
│   ├─ 平板横屏：1024px-1200px
│   └─ 桌面：>1200px
│
└─ 元素适配
    ├─ 文字：字体大小/行高
    ├─ 图片：尺寸/清晰度
    ├─ 按钮：点击区域
    └─ 表单：输入体验
```

### 2. 触摸测试

```
├─ 手势支持
│   ├─ 点击：tap事件
│   ├─ 长按：longpress事件
│   ├─ 滑动：swipe事件
│   ├─ 缩放：pinch事件
│   └─ 旋转：rotate事件
│
├─ 手势冲突
│   ├─ 滚动冲突：页面滚动vs组件滚动
│   ├─ 缩放冲突：浏览器缩放vs组件缩放
│   └─ 手势冲突：左滑返回vs组件手势
│
└─ 触摸体验
    ├─ 响应速度：<100ms
    ├─ 防抖处理：防止误触
    ├─ 禁用缩放：user-scalable=no
    └─ 点击延迟：300ms问题
```

### 3. 浏览器兼容

```
├─ iOS Safari
│   ├─ 特性支持：-webkit-前缀
│   ├─ 安全限制：HTTPS要求
│   ├─ 存储限制：localStorage
│   └─ 弹性滚动：-webkit-overflow-scrolling
│
├─ Android WebView
│   ├─ 版本差异：不同Android版本
│   ├─ 厂商差异：三星/华为/小米
│   ├─ 内核差异：Chrome/系统WebView
│   └─ 性能差异：渲染性能
│
└─ 其他浏览器
    ├─ 微信内置浏览器
    ├─ QQ内置浏览器
    ├─ 微博内置浏览器
    └─ 其他App内置浏览器
```

### 4. 性能测试

```
├─ 加载性能
│   ├─ 首屏时间：<2秒
│   ├─ 完全加载时间：<5秒
│   ├─ 资源加载：图片/JS/CSS
│   └─ 缓存策略：强缓存/协商缓存
│
├─ 渲染性能
│   ├─ 首次渲染：<1秒
│   ├─ 重排重绘：减少reflow
│   ├─ 动画帧率：>55fps
│   └─ 滚动流畅：无卡顿
│
├─ 交互性能
│   ├─ 点击响应：<100ms
│   ├─ 表单输入：无延迟
│   ├─ 页面切换：<300ms
│   └─ 数据加载：<2秒
│
└─ 网络性能
    ├─ 请求次数：减少请求
    ├─ 请求大小：压缩资源
    ├─ 弱网表现：可接受体验
    └─ 离线缓存：Service Worker
```

### 5. SEO测试

```
├─ 基础SEO
│   ├─ 标题标签：title唯一
│   ├─ 描述标签：description准确
│   ├─ 关键词：keywords合理
│   └─ 结构化数据：Schema.org
│
├─ 内容SEO
│   ├─ 标题层级：H1-H6正确
│   ├─ 图片Alt：Alt属性完整
│   ├─ 链接：内链/外链合理
│   └─ 内容质量：原创/有价值
│
└─ 技术SEO
    ├─ URL规范：规范化URL
    ├─ sitemap：站点地图
    ├─ robots.txt：爬虫协议
    └─ 移动适配：移动端声明
```
