## 四、拍摄参数设置

### 手机拍摄参数

```
iPhone设置：
- 分辨率：4K 30fps（最佳）或 1080P 60fps
- 网格：开启（辅助构图）
- 稳定性：开启（防抖）
- HDR：关闭（便于后期）
- 曝光：自动（拍摄时轻点面部测光）

安卓手机（通用）：
- 分辨率：4K 30fps
- 对焦：自动+触屏对焦
- 白平衡：自动
- ISO：自动（不超过1600）
```

### 相机拍摄参数

```
索尼ZV-E10推荐设置：
- 模式：视频模式
- 分辨率：4K 25fps
- 文件格式：XAVC S 4K
- ISO：自动（100-3200）
- 光圈：f/2.0-f/4.0（背景虚化）
- 对焦：实时眼控AF
- 美颜：关闭（专业感）
```

## 六、拍摄注意事项

### 常见问题及解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 面部暗淡 | 灯光角度不对 | 调整灯位，45度侧光 |
| 背景过亮/过暗 | 曝光不准确 | 拍摄时点触面部测光 |
| 声音嘈杂 | 环境音干扰 | 使用领夹麦，远离噪音源 |
| 画面晃动 | 手持不稳 | 使用三脚架，启用防抖 |
| 表情僵硬 | 紧张 | 多拍几条，放松心态 |

### 拍摄前检查清单


## 七、拍摄技巧

### 克服镜头紧张

1. **提前预演**：正式拍摄前练习2-3遍
2. **看镜头下方**：减少直视压力
3. **想象对话**：把镜头当成熟人朋友
4. **分段录制**：一句一句录，减少心理压力
5. **深呼吸**：开始前做3次深呼吸放松

### 自然表达技巧

1. **语速控制**：每分钟150-180字
2. **适当停顿**：重要观点前后停顿
3. **眼神交流**：看镜头，不要飘
4. **手势配合**：自然的手部动作
5. **表情管理**：跟随内容有适当表情

### 重录标准

建议重录的情况：
- 口误超过2处
- 卡壳超过3秒
- 表情明显不自然
- 语句逻辑混乱
- 语气过于僵硬

## 八、后期基础

### 剪映（手机）剪辑流程

```
1. 导入素材
   └→ 选择视频片段

2. 粗剪
   └→ 裁剪多余部分
   └→ 调整片段顺序

3. 添加字幕
   └→ 自动识别
   └→ 校对修正

4. 调色
   └→ 亮度/对比度
   └→ 饱和度微调
   └→ 一键美颜（可选）

5. 添加BGM
   └→ 背景音乐
   └→ 音量调节

6. 导出
   └→ 1080P/4K
   └→ 发送
```

### 字幕规范

| 类型 | 格式 | 示例 |
|------|------|------|
| 对话字幕 | 白字黑边 | 大家好，我是老胡 |
| 强调字幕 | 渐变色+放大 | 重点！必须记住 |
| 数据字幕 | 数字突出 | 转化率 ↑ 30% |

---

> **拍摄口诀**：光线要均匀，背景要简洁，表情要自然，内容要口语，眼神要有神，结尾要引导。
