===========================================
SKILL-TTS-001 - TTS 自动化（文本转语音 + 自动播放）
===========================================

【这个技能是干啥的】
把文字转换成语音（阿里云 TTS），生成 MP3 文件后自动播放。

【什么时候用】
- 用户说"生成语音"
- 用户说"转语音"
- 用户说"朗读这段"
- 需要语音提醒/点评/汇报

【核心步骤】
1. 用户提供文本（≤1500 字符）
2. 调用阿里云 TTS API
3. 生成 MP3 文件（workspace/tts/voice-时间戳.mp3）
4. 检测锁屏状态
5. 屏幕活跃时自动播放
6. 锁屏时跳过（记录日志）

【举个例子】
用户：把这段话转成语音："当前最优先任务是飞书 OAuth 配置"
    ↓
调用：SKILL-TTS-001
    ↓
处理：
  - 文本："当前最优先任务是飞书 OAuth 配置"
  - 调用阿里云 TTS API
  - 生成：workspace/tts/voice-1772869000000.mp3
  - 检测：屏幕未锁定
  - 播放：自动播放
    ↓
输出：MP3 文件 + 自动播放（0 步操作）

【对应阿福的 Skill】
skills/tts-automation/SKILL.md

【关联原子动作】
- ATOM-VISUAL-008：生成 TTS 语音（底层）
- ATOM-DELIVERY-013：本地播放语音（自动调用）

【常见错误】
❌ 错误：文本>1500 字符（会被截断）
✅ 正确：分段生成，每段≤1500 字符

❌ 错误：API Key 无效
✅ 正确：检查阿里云 API Key 配置

❌ 错误：锁屏时播放
✅ 正确：检测 LockApp 进程，锁屏时跳过

【核心原则】
> 文本转语音 → 自动播放，0 步操作！

【TTS 配置】
- 提供商：阿里云（Aliyun）
- 声音：zh-CN-yunxi
- 语言：zh-CN（中文）
- 格式：MP3
- 采样率：24000 Hz

【文件管理】
- 存储位置：workspace/tts/
- 命名规则：voice-时间戳.mp3
- 统一归档：所有 TTS 文件集中管理

【播放策略】
- 自动播放：是（生成后立即播放）
- 锁屏检测：是（锁屏时跳过）
- 日志记录：是（Temp/voice-playback-log.txt）

【触发词】
- "生成语音"
- "转语音"
- "朗读这段"
