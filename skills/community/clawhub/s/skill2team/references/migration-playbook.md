# Migration Playbook

Stages:

1. Inventory.
2. Workflow extraction.
3. Candidate design.
4. Runtime artifact generation.
5. Shadow run.
6. Limited rollout.
7. Guarded rollout.
8. Full replacement or hybrid route.

Hybrid options include baseline for simple tasks, team for high-risk tasks, original skill as a sub-skill, team as reviewer, or platform-neutral design before runtime deployment.
