# Amap Travel Planner

> 系统化的旅行规划助手，融合高德路径规划 API 提供真实路线数据。从目的地研究、行程设计、预算管理到清单检查，覆盖旅行全生命周期。基于高德地图真实路径规划数据（驾车/步行/公交），告别拍脑袋估时间，让每一段行程都有精准的距离、时间和路线支撑。最终生成可在高德地图 App 扫码打开的旅行地图。
