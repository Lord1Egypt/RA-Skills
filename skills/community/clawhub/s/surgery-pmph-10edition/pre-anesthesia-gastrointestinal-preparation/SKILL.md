# 麻醉前胃肠道准备规范

## 适用对象
- 择期手术患者
- 小儿患者
- 胃肠功能异常患者
- 胃食管反流高风险患者

**前提条件**：患者计划接受全身麻醉或椎管内麻醉，且无急诊指征。

**不适用情况**：急诊手术；胃肠动力障碍者需个体化调整，不可直接套用标准规则。

## 执行步骤

### 1. 根据食物类型确定禁食时间

**成人患者**：
- 易消化固体食物或非母乳类奶制品：至少禁食6小时
- 油炸、高脂食物或肉类：至少禁食8小时
- 清液体（水、无果肉果汁、苏打饮料、清茶、黑咖啡，不含酒精）：术前2小时内可少量饮用

**小儿患者**：
- 固体食物：禁食4～6小时
- 清水：禁水2～3小时
- 新生儿/婴幼儿：
  - 母乳：至少禁食4小时
  - 非母乳或婴儿配方奶、易消化固体食物：至少禁食6小时

### 2. 特殊情况处理

对合并胃肠排空延迟或胃食管反流高风险者：
- 适当延长禁食时间
- 可考虑加用促胃动力药（如甲氧氯普胺）和抑酸药（如H2受体拮抗剂或质子泵抑制剂）

### 3. 胃肠道手术的额外准备

- 术前1～2日开始进流质饮食
- 幽门梗阻患者：需进行胃减压，必要时行术前洗胃（详见参考资料）
- 结直肠手术：
  - 术前2～3天进流食
  - 口服肠道制菌药物（如新霉素、甲硝唑）
  - 术前一日及手术当日清晨行清洁灌肠或口服泻剂（详见参考资料）

### 4. 输出要求
生成包含以下内容的个体化方案：
- 各类食物/液体的最后允许摄入时间
- 是否建议使用促胃动力药或抑酸药
- 胃肠道手术患者的附加准备措施

> **目的**：防止麻醉或术中因呕吐导致胃内容物反流误吸，引发窒息或吸入性肺炎。

## 参考文档

### 胃肠道手术特殊准备补充说明

**幽门梗阻患者**
- 术前需进行持续胃肠减压，以排空胃内容物
- 必要时可行术前洗胃，确保胃腔清洁，降低误吸风险

**结直肠手术肠道准备**
- **清洁灌肠**：通常使用磷酸钠灌肠剂或生理盐水，术前一日晚及手术当日清晨各一次
- **口服泻剂替代方案**：如聚乙二醇电解质散（PEG），按说明书分次服用，直至排出清水样便
- **抗生素使用**：新霉素 + 甲硝唑联合口服，通常从术前1～2天开始

**注意事项**
- 老年或肾功能不全患者慎用含磷酸盐的灌肠剂
- 口服泻剂可能导致水电解质紊乱，需监测并补液
- 抗生素预防应结合医院耐药谱和患者过敏史调整
