# 去渍指南

## 基本原则

1. **越早处理越好**：新鲜污渍比陈旧污渍容易去除
2. **先测试**：在不显眼处先测试，确保不损坏衣物
3. **由外向内**：从污渍边缘向中心擦拭，防止扩散
4. **冷水优先**：热水会让蛋白质污渍（血、蛋）凝固

---

## 常见污渍处理

### 🩸 血渍
- **新鲜**：冷水冲洗，加洗衣液搓洗
- **干涸**：浸泡冷水30分钟，用肥皂或洗洁精搓洗
- ❌ 不要用热水！会凝固

### 🍳 油渍（食用油、沙拉酱）
1. 在污渍处涂洗洁精
2. 静置10分钟
3. 用热水（可承受的最高温）搓洗
4. 正常洗涤

### ☕ 咖啡/茶渍
1. 立即用冷水冲洗
2. 涂洗衣液，静置5分钟
3. 如果还有痕迹，用稀释白醋（1:1）点涂
4. 正常洗涤

### 🍷 红酒渍
1. 立即用盐覆盖，吸收液体
2. 用沸水从高处浇下（如果布料耐热）
3. 或用苏打水冲洗
4. 正常洗涤

### 🖊 墨水渍
1. 在污渍下垫纸巾
2. 用酒精（或含酒精的爽肤水）从背面点涂
3. 墨水会转移到纸巾上
4. 换纸巾重复，直到墨水吸完
5. 正常洗涤

### 💄 口红/化妆品
1. 用卸妆水点涂
2. 或用酒精轻擦
3. 涂洗衣液搓洗
4. 正常洗涤

### 🟡 汗渍/黄渍
1. 衣领处涂洗发水（针对皮脂）
2. 或用小苏打+水调成糊状涂抹
3. 静置30分钟
4. 正常洗涤

### 🌱 草渍
1. 用酒精或白醋浸泡
2. 或用洗洁精+双氧水
3. 搓洗后正常洗涤

---

## 工具推荐

| 污渍类型 | 最佳工具 |
|---------|---------|
| 油渍 | 洗洁精 |
| 蛋白质污渍（血、蛋） | 冷水+肥皂 |
| 色素污渍（咖啡、酒） | 白醋 |
| 墨水 | 酒精 |
| 通用 | 氧漂剂（彩色衣物用） |

---

## 注意事项

- **丝绸、羊毛**：送去干洗，不要自己处理
- **皮革**：用专用皮革清洁剂
- **不确定材质**：先测试小面积

如果多次尝试仍无法去除，考虑送去干洗店，他们有专业设备和溶剂。
