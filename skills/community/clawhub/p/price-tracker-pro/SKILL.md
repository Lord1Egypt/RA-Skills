# Price Tracker - 商品价格追踪器

自动监控商品价格，降价时第一时间通知你。

## 功能

- 添加商品监控（支持淘宝、京东、拼多多链接）
- 设置目标价格，到达时自动提醒
- 查看价格历史走势
- 批量管理监控列表

## 使用方法

### 添加监控
```
监控这个商品：<商品链接>
目标价格：¥XXX
```

### 查看列表
```
查看我的价格监控
```

### 删除监控
```
停止监控 <商品名/编号>
```

## 工作原理

1. 解析商品链接获取当前价格
2. 定期抓取价格（每 6 小时）
3. 价格低于目标时发送提醒
4. 记录历史价格生成走势图表

## 支持平台

- ✅ 淘宝/天猫
- ✅ 京东
- ✅ 拼多多
