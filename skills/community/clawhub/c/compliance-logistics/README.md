# 跨境出口认证与清关查询（Logistics Compliance Check）

> 货被扣了、被退回了、平台下架了——90% 都是合规问题。输入产品 + 目的国，一分钟看清出口认证/标签/包装/测试/限制全套要求。

覆盖 8 大目标市场的合规要求，适用于跨境电商卖家在发货前或扣货后快速查询合规信息。

📖 详细使用说明见 [SKILL.md](./SKILL.md)
📊 完整数据见 [references/logistics-data.md](./references/logistics-data.md)

---

## 快速上手

直接说下面任意一句：

> 「电子产品出口德国要什么认证？」
> 「货在美国海关被扣了，怎么办？」
> 「玩具出口日本和美国，要求有什么不同？」
> 「我有 CE 认证，出口英国还要什么？」

## 支持的市场

🇺🇸 美国 · 🇬🇧 英国 · 🇩🇪 德国 · 🇪🇪 爱沙尼亚 · 🇸🇬 新加坡 · 🇯🇵 日本 · 🇦🇺 澳大利亚 · 🇭🇰 香港

## 核心输出

| 输出类型 | 说明 |
|---------|------|
| 五大类合规清单 | 认证 / 标签 / 包装 / 测试 / 限制 |
| 海关扣货应急指南 | 扣货原因分析 + 补办路径 + 时间线 |
| 多国横向对比 | 同一产品出口不同国家的合规差异 |
| 已有资质匹配 | 基于用户已有认证计算合规缺口 |

---

⚠️ AI 输出为合规参考，不构成清关建议。实际清关以当地海关判定为准。
