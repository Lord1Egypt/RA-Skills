# Education Search Skill

学历教育与职业培训搜题工具

## 功能简介

从习题库中查找习题、真题、试卷等资源，涵盖：
- 教资考试
- 考研真题
- 行测
- 各类职业资格考试

## 使用方法

直接在对话中输入需求即可，例如：
- "帮我找考研数学真题"
- "教资考试幼儿保教知识真题"
- "行测图形推理题"

## 依赖要求

- `baidu-search` skill（用于兜底搜索）
- 需要 `BAIDU_API_KEY` 环境变量

## 更新方法

```bash
openclaw skills update education-search
```
