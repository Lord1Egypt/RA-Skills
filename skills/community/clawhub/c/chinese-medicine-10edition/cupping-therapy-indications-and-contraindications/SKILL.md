# Cupping Therapy Indications and Contraindications

## When to Use
- 患者表现为风寒湿痹、颈肩腰腿疼痛、感冒咳嗽或面瘫等适应证
- 已排除所有禁忌证
- 需选择合适的拔罐类型（留罐、走罐、闪罐或刺络拔罐）

## Execution Steps
1. **确认适应证对应方案**：
   - 肩背痛、腰腿痛 → 留罐或多罐
   - 胃痛 → 单罐拔中脘穴
   - 冈上肌腱炎 → 单罐拔肩髃穴
   - 感冒、咳嗽 → 走罐于腰背部
   - 面瘫 → 闪罐至皮肤潮红
   - 痤疮等皮肤病 → 刺络拔罐（先点刺出血，再拔罐吸出3～5ml血液）
2. **严格排查禁忌证**（任一存在即禁用）：
   - 急性严重疾病、接触性传染病、严重心脏病、心力衰竭
   - 皮肤过敏、传染性皮肤病、皮肤肿瘤/溃烂
   - 血小板减少性紫癜、白血病、血友病
   - 心尖区大动脉搏动处、静脉曲张处
   - 精神分裂症、抽搐、高度神经质及不合作者
   - 急性外伤性骨折、中重度水肿
   - 瘰疬、疝气、活动性肺结核
   - 眼耳口鼻等五官孔窍部
3. **选择拔罐类型并控制时间**：
   - 留罐：5～10分钟（风寒湿痹）
   - 走罐：涂介质后推拉至潮红（肌肉丰厚部位）
   - 闪罐：反复吸拔至潮红（面瘫）
   - 刺络拔罐：仅限特定皮肤病
   - 总时长一般5～20分钟；年老体弱或儿童应缩短
4. **正确起罐**：一手握罐底稍倾斜，另一手按压罐口边缘皮肤，使空气进入后取下。

> 警告：若存在任一禁忌证，必须禁止使用拔罐疗法，否则可能导致严重不良反应。
