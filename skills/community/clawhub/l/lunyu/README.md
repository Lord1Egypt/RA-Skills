# 论语 — 私人语录 Skill

一个可以写属于你自己的《论语》的 skill。

在这里，你就是孔子，你的 Agent 就是你的学生。很多时候你会突然冒出一句很妙的感悟、一段漂亮的吐槽、或者像 Tim 那样的"色话"，但如果当时不记下来，这些话就再也不会出现在你的人生当中了。

这种灵感时刻非常零碎——它们发生在通勤路上、洗澡时、深夜失眠的时候。你不可能每次都掏出笔记下来。所以你需要一个工具，能在你随口说出来的那一刻就完成捕捉，然后在后台默默整理成一本浓缩的"论语 of your own"。

> 你说，它记。零摩擦，零延迟。

## 快速开始

在 Hermes Agent 中启用此 skill：

```
hermes curator install lunyu
```

然后直接在对话中使用：

```
记论语：所有的焦虑都来自对不确定性的抗拒，而真正的勇气是在不确定中依然向前走。
```

Agent 会自动将这句话存入你的私人语录集。

## 核心功能

- **主动记录**：`记论语：xxx`、`论语：xxx`、`子曰：xxx`
- **被动捕捉**：当你说出精彩句子时，Agent 会主动问你要不要记
- **分类管理**：按章节（论生活、论学习、论情感、色话集等）、标签、类型组织
- **查询检索**：按章节浏览、按标签/类型筛选、全文搜索
- **每日随机**：每日一语功能，每天随机推送一条你的金句
- **导出编译**：一键生成编译版 `论语.md`

## 文件结构

```
~/.hermes/lunyu/
├── analects.json   # 原始数据（JSON）
└── 论语.md          # 编译版预览（可手动导出到 Obsidian）
```

## 命令一览

| 命令 | 作用 |
|------|------|
| `记论语：xxx` | 记录金句 |
| `论语览` | 浏览所有语录 |
| `论语览 论学习` | 浏览某章节 |
| `论语搜 xxx` | 搜索关键词 |
| `论语统计` | 统计数据 |
| `论语今日` | 随机一条每日一语 |
| `论语导出` | 生成编译版 |
| `论语删 xxx` | 删除语录 |
| `论语改 xxx 论XX` | 修改章节 |
