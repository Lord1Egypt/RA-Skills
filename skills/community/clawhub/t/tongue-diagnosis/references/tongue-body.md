# 舌质诊断（舌体）

## 一、舌神

### 1. 荣舌（有神）
- **舌象特征**：舌色红活鲜明，舌体滋润、运动灵活
- **临床意义**：气血津液充足，脏腑功能正常。见于健康人或轻病患者

### 2. 枯舌（无神）
- **舌象特征**：舌色晦暗无光，舌体干枯、运动不灵
- **临床意义**：气血津液枯竭，脏腑功能衰竭。见于重病患者

---

## 二、舌色

### 1. 淡红舌
- **舌象特征**：舌体颜色淡红润泽、白中透红
- **临床意义**：气血调和的征象，常见于正常人或轻病患者

### 2. 淡白舌
- **舌象特征**：舌色比正常浅淡，白色偏多红色偏少
- **临床意义**：主气血两虚、阳虚
- **细分**：
  - 淡白瘦薄 → 气血两虚
  - 淡白胖大 → 阳虚水湿内停
  - 淡白湿润 → 寒湿

### 3. 红舌
- **舌象特征**：舌色较正常舌色红，呈鲜红色
- **临床意义**：主热证
- **细分**：
  - 舌尖红赤 → 心火亢盛
  - 舌边红赤 → 肝胆火旺
  - 全舌鲜红 → 气分热盛
  - 嫩红舌 → 虚热证

### 4. 绛舌
- **舌象特征**：较红舌更深的或略带暗红色
- **临床意义**：主热入营血，或阴虚火旺

### 5. 青紫舌
- **舌象特征**：全舌呈均匀青色或紫色，或在舌色中泛现青紫色
- **细分**：
  - 淡青紫舌 → 淡舌泛青紫
  - 紫红/绛紫舌 → 红绛舌泛青紫
  - 瘀斑/瘀点舌 → 局部青紫色斑点，大小不一，不高于舌面
- **临床意义**：主气血运行不畅（血瘀）

## 三、舌形

### 1. 老、嫩舌
- **老舌**：舌体坚敛苍老，纹理粗糙或皱缩，舌色较暗 → 多见于实证
- **嫩舌**：舌体浮胖娇嫩，纹理细腻，舌色浅淡 → 多见于虚证

### 2. 胖、瘦舌
- **胖大舌**：舌体比正常的大而厚，伸舌满口 → 体内水湿停滞的表现
  - 胖大色淡白 → 气虚、阳虚
  - 胖大色红 → 里热
- **肿胀舌**：舌体肿大，舌色鲜红或青紫，甚则不能缩回口中 → 心脾热盛、外感湿热
- **瘦薄舌**：舌体比正常瘦小而薄 → 舌失濡养
  - 瘦薄色淡白 → 久病气血两虚
  - 瘦薄色红绛，舌干少苔或无苔 → 阴虚火旺
- **齿痕舌**：舌体两边有齿痕（常伴胖大）→ 主脾虚、湿证
  - 舌体不胖而有齿痕，舌质嫩 → 气血两虚

### 3. 点、刺舌（芒刺）
- **舌象特征**：蕈状乳头肿胀或高突
  - 点 → 蕈状乳头体积增大、数目增多，充血水肿
  - 刺 → 蕈状乳头增大、高突，形如芒刺
- **临床意义**：主脏腑阳热亢盛，或血分热盛
- **定位参考**：
  - 舌尖芒刺 → 心火亢盛
  - 舌中芒刺 → 胃肠热盛

### 4. 裂纹舌
- **舌象特征**：舌面上出现各种形状的裂纹、裂沟，深浅不一
- **临床意义**：主精血亏虚，或阴津耗损
- **细分**：
  - 裂纹色红，无苔 → 阴液亏虚
  - 裂纹色淡白 → 血虚

## 四、舌态

### 1. 痿软舌
- **舌象特征**：舌体软弱无力，不能随意伸缩回旋
- **临床意义**：主阴虚，或气血俱虚

### 2. 强硬舌
- **舌象特征**：舌体失其柔和，卷伸不利，或板硬强直，不能转动
- **临床意义**：主热入心包；或高热伤津；或风痰阻络

### 3. 歪斜舌
- **舌象特征**：伸舌时舌体偏向一侧
- **临床意义**：主肝风夹痰，或痰瘀阻络（中风征兆！）

### 4. 颤动舌
- **舌象特征**：舌体不自主地颤动，动摇不宁
- **临床意义**：主肝风内动
- **细分**：
  - 颤动舌色淡白 → 血虚生风
  - 颤动舌色红绛 → 热极生风

### 5. 吐弄舌
- **舌象特征**：舌伸出口外不能回缩（吐舌）；舌尖反复舔口唇（弄舌）
- **临床意义**：主心脾有热

### 6. 短缩舌
- **舌象特征**：舌体短缩，不能伸长
- **临床意义**：主气血虚衰，或风痰阻络
- **注意**：先天性舌系带过短（绊舌）为生理性，无辨证意义
