# COC 第七版 战斗与追逐规则

## 第六章 战斗

### 6.1 战斗顺序（敏捷决定）

- 战斗以**回合制**进行，每轮中每个角色拥有一个回合的行动机会
- **行动顺序**：由高到低按敏捷（DEX）排列；敏捷相同时，拥有较高战斗技能的角色先行动
- 每回合角色可执行以下**一项**动作：
  - 使用格斗或射击技能发动一次攻击
  - 执行一次战技
  - 脱离战斗
  - 施展一个法术
  - 进行其他需要时间的动作（如撬锁）
- 角色可选择**延迟行动**到另一角色行动后；若所有角色都延迟，本轮可能无人行动

> **可选规则：先攻检定** — 每人进行一次敏捷检定决定先攻顺序：极难成功 > 困难成功 > 常规成功 > 失败。大成功获得战术优势/奖励骰，大失败可能失去第一次攻击机会。

---

### 6.2 近战攻击和闪避

#### 徒手格斗
- 使用**斗殴**（格斗技能的专攻项）
- 被攻击时可选择：**闪避**或**反击**
- 近战结果由**对抗检定**决定

#### 目标选择反击
- 攻击者格斗技能 vs 被攻击者格斗技能
- 成功等级较高的一方命中对手并造成伤害
- **平手**时视为攻击者胜利
- 双方均失败则无人受伤

#### 目标选择闪避
- 攻击者格斗技能 vs 被攻击者闪避技能
- 攻击者成功等级更高则命中
- **平手**时视为闪避者躲开了攻击
- 双方均失败则无人受伤

#### 持械格斗
- **基础武器**（匕首、棍棒等简单武器）：使用格斗（斗殴）
- **复杂武器**（剑等）：需要对应的专攻项技能（如格斗（剑））
- 武器技能归属和伤害见**表XVII：武器列表**

#### 临时武器
- 守秘人将临时武器与武器列表中类似武器比较决定伤害
- 鼓励玩家利用环境（台球杆、酒瓶、椅子等）

#### 寡不敌众
- 一轮内已进行过闪避或反击的角色，接下来对他的所有近战攻击检定可获得**1个奖励骰**
- 不适用于远程攻击

#### 突袭
- 被突袭的目标若未察觉攻击，可能无法闪避或反击
- 攻击者可使用潜行技能设定目标的聆听/侦查/心理学检定难度
- 未察觉的目标：攻击自动成功（除非大失败），远程攻击仍需检定
- 从背后敲头（未被发现）：自动成功（除非大失败）

---

### 6.3 战技

用于达成非纯伤害的目标（缴械、撞倒、推下悬崖等）。

#### 步骤1：比较体格
| 体格差距 | 惩罚骰 |
|---------|--------|
| 比对方小1级 | 1个惩罚骰 |
| 比对方小2级 | 2个惩罚骰 |
| 比对方小3级或以上 | 战技不可能成功 |

#### 步骤2：进行攻击检定
- 通常使用格斗（斗殴）检定
- 对手可选择闪避或反击（反击成功时可造成伤害）
- 平手规则与普通近战相同

#### 战技成功可达成的目标
- 将目标缴械或夺下物品
- 使目标陷入劣势（后续行动承受1个惩罚骰或盟友获得1个奖励骰）
- 从物理压制中脱离
- 将对手推倒/撞落

---

### 6.4 远程攻击和射程

#### 射击基础规则
- 投掷百分骰，比较与射击技能
- **不是对抗检定**，难度取决于目标距离

#### 远程攻击难度等级
| 距离 | 难度等级 |
|-----|---------|
| 基础射程内 | 普通 |
| 基础射程 x2 | 困难 |
| 基础射程 x4 | 极难 |

#### 奖励骰和惩罚骰汇总
| 奖励骰 | 惩罚骰 |
|--------|--------|
| 瞄准一轮 | 高速移动目标（MOV 8+） |
| 抵近射击（敏捷1/5英尺内） | 小目标（体格-2以下） |
| 大目标（体格4+） | 手枪连射 |
| | 装填一发弹药并击发 |
| | 射击近战中的目标 |
| | 部分（一半以上）遮蔽 |

#### 抵近射击
- 目标在射击者敏捷值的1/5英尺内：获得**1个奖励骰**
- 目标无法像面对格斗攻击时那样反击或闪避

#### 瞄准
- 声明仔细瞄准，下一轮射击时获得**1个奖励骰**
- 瞄准前若受到伤害或移动则失去优势

#### 手枪连射
- 一轮内射击2或3发子弹：每次检定承受**1个惩罚骰**

#### 自动武器射击
- **短点射**：2或3发子弹，视为一组弹幕
- **全自动射击**：弹幕数 = 技能值/10（向下取整），最少3发
- 第一次攻击检定：正常难度
- 之后每次：增加1个惩罚骰（超过2个惩罚骰时，难度等级提升1级）
- **命中结果**：成功 = 一半子弹命中；极难成功 = 全部命中，前半贯穿

#### 武器故障
- 攻击检定结果 >= 武器故障值：枪械未开火
- 转轮式/中折式：仅哑弹
- 杠杆式：卡壳，修理需1D6战斗轮 + 成功的机械维修/射击检定

#### 弹药装填
- 手枪/步枪/霰弹枪：装填2发需1轮，换弹夹需1轮
- 机枪弹链：2轮
- 装填1发并射击：承受**1个惩罚骰**

#### 投掷武器和投射武器
- 投掷武器可被闪避
- 面对投射/投掷武器时，只有距离 < 敏捷1/5英尺才能反击
- 依赖力量的投掷/投射武器：伤害加值的一半加到伤害上

### 6.5 伤害和护甲

#### 计算伤害
- 徒手格斗（拳打脚踢）：**1D3** 伤害
- 武器伤害见**表XVII：武器列表**

#### 极难成功的额外伤害
- **非贯穿武器**（钝器）：伤害骰和伤害加值取**最大值**
- **贯穿武器**（刀刃、子弹）：伤害骰和伤害加值取最大值 + **额外一份武器伤害骰**

> 示例：+1D4伤害加值的角色用弹簧刀极难成功 → 1D4 + 4(武器最大) + 4(加值最大) = 1D4+8 (9~12伤害)

#### 护甲
- 护甲值直接从伤害中扣除
- 护甲不减少魔法、毒素、溺水等伤害

#### 护甲数值范例
| 护甲 | 数值 |
|------|-----|
| 厚重皮夹克 | 1点 |
| 一战标准钢盔 | 2点 |
| 1英寸硬木 | 3点 |
| 现代美军头盔 | 5点 |
| 重型凯芙拉防弹背心 | 8点 |
| 军用防暴服 | 12点 |
| 1.5英寸防弹玻璃 | 15点 |
| 1英寸钢板 | 19点 |
| 大号沙包 | 20点 |

#### 脱离近身战斗
- 花费该回合的行动逃离近战范围（前提：有逃脱路线且未被物理压制）

### 6.6 生命值和重伤

#### 轻伤 vs 重伤
| 条件 | 类型 |
|------|------|
| 单次伤害 < 最大生命值的一半 | **轻伤** |
| 单次伤害 >= 最大生命值的一半 | **重伤** |
| 单次伤害 > 最大生命值 | **死亡** |

#### 轻伤
- 无特殊影响，生命值归零时角色昏迷
- 角色不会因轻伤死亡

#### 重伤
- 重伤栏标记
- 角色**立刻倒地**
- 必须进行**体质检定**以避免昏迷
- 重伤栏已标记时生命值归零 → 濒死

#### 生命值归零
- 角色昏迷（不记录负数生命值）
- 只受轻伤：不会死亡，随时间恢复
- 已受重伤：濒死（需急救）
- 伤害超过最大生命值：必死无疑

#### 急救
- 必须在**一小时内**进行
- 成功恢复**1点生命值**
- 只能尝试一次，之后构成孤注一掷
- 濒死角色：急救只能暂时稳定伤势

#### 医学
- 需花费至少**一小时**
- 成功恢复**1D3生命值**（追加在急救之外）
- 非当天进行需困难成功

#### 濒死
- 重伤角色生命值归零 → 濒死
- 立刻昏迷
- 每轮结束进行**体质检定**，失败则死亡
- **急救**稳定伤势 → 获得1点临时生命值 → 需接受医学检定
- 医学成功后：擦除濒死标记，恢复1D3生命值

#### 轻伤恢复
- 每天恢复**1点生命值**

#### 重伤恢复
- 每周进行**体质检定**
  - 失败：无恢复
  - 成功：恢复1D3生命值
  - 极难成功：恢复2D3生命值
- 奖励骰：完全静养 / 有效医疗服务
- 惩罚骰：休养环境差 / 照料者医学大失败
- 重伤治愈条件：体质极难成功 或 生命值恢复到最大生命值的一半以上
- 大失败：永久性伤害/并发症

### 6.8 其他类型伤害

| 伤情 | 伤害 | 范例 |
|------|------|------|
| 轻度 | 1D3 | 拳打脚踢、头槌、弱酸、坠落到淤泥地（每10英尺） |
| 中度 | 1D6 | 坠落到草地（每10英尺）、棍子、强酸、小口径子弹、火焰 |
| 重度 | 1D10 | .38子弹、坠落到水泥地（每10英尺）、斧头、火焰喷射器 |
| 致命 | 2D10 | 被30mph汽车撞击、手榴弹3~6码内、强毒剂 |
| 终结 | 4D10 | 被超速汽车撞击、手榴弹3码内、致死毒剂 |
| 血肉横飞 | 8D10 | 高速直接撞击、被火车撞击 |

- **窒息/溺水**：每轮体质检定，失败则每轮受伤直至死亡
- **毒药**：体质极难成功可减伤，大成功可能摆脱效果

## 第七章 追逐

### 7.1 追逐系统

追逐场景按**回合**进行（同战斗），由一系列**地点**构成。

#### 五部分结构
1. **建立追逐**：速度检定决定追逐是否开始
2. **切入追逐**：布置场景、放置参与者、决定行动顺序
3. **移动**：使用行动点在地点间移动，处理险境和障碍
4. **冲突**：角色/载具之间的攻击和战技
5. **补充规则**：可选规则

#### 速度检定
- 步行：**体质检定**
- 载具：**汽车驾驶技能检定**
- 结果：
  - 成功：MOV不变
  - 极难成功：MOV +1
  - 失败：MOV -1

#### 比较速度
- 追逐者调整后MOV >= 逃离者 → 追逐开始
- 逃离者调整后MOV > 追逐者 → 成功逃离

---

### 7.2 追逐中的行动

#### 地点
- 地点表示追逐中的**方位**，距离不等同
- 地点间放置**险境**和**障碍**
- 例：旅馆 = 旋转门 → 大堂 → 楼梯 → 走廊 → 餐厅 → 厨房...

#### 追逐轮
- 按回合进行，按**敏捷顺序**行动
- 战斗轮与追逐轮互通

#### 行动点
- 所有角色默认获得**1点行动点**
- MOV比最慢者每高1点 → 获得**1点额外行动点**
- 行动点用于从一个地点移至下一个地点

#### 追逐中的行动
在回合中，角色可：
- 前进（花费行动点）
- 发起一次攻击（格斗/射击/汽车驾驶）*消耗1行动点
- 施放法术
- 其他动作（如撬锁）

> 注：追逐中不可孤注一掷，但可在下轮再次尝试

---

### 7.3 距离和移动

#### 移向下一个地点
- 无障碍区域：花费**1点行动点**移动1个地点

#### 险境
险境迫使角色放慢速度或承担风险：
- 角色可花费行动点换取**奖励骰**（1点=1个奖励骰，2点=2个）
- 技能检定失败：可能受伤 + 损失1D3行动点
- 险境放置在两个地点之间

#### 常用技能对应险境
| 技能 | 适用险境 |
|------|---------|
| 攀爬 | 翻越围栏/墙壁、从高处窗户爬下 |
| 游泳 | 穿过河流/湖泊 |
| 闪避 | 穿越人群、躲避落物 |
| 力量 | 穿越泥沼 |
| 敏捷 | 墙顶奔跑防摔落 |
| 汽车驾驶 | 汽车追逐险境 |

#### 步行追逐险境范例
- 矮围栏/矮墙、闩住的窗户、河流/泥沼、拥挤街道、窄巷、屋顶、螺旋阶梯等

#### 汽车追逐险境范例
- 交通堵塞、U形弯、道路施工、路面事故、十字路口、急转弯、路上动物等

#### 障碍
- 阻拦前进直到被越过或摧毁
- 失败：通常仅拖延行动，无法前进
- 破坏障碍：载具每点体格造成1D10伤害

#### 障碍物耐久值范例
| 障碍 | 耐久值 |
|------|--------|
| 室内门/薄围栏 | 5 |
| 普通后屋门 | 10 |
| 坚固室外门 | 15 |
| 9寸厚砖墙 | 25 |
| 大树 | 50 |
| 混凝土桥墩 | 100 |

---

### 7.4 追逐中的冲突

- 无火器时只能攻击**同一地点**的目标
- 攻击需消耗**1行动点**
- 受到攻击时总有反击/闪避机会（无论是否有行动点剩余）

#### 载具作为武器
- 每点体格造成**1D10伤害**
- 载具也受到造成伤害一半的自身伤害
- 每10点伤害损失1点体格
- 载具损失的体格值不超过目标的体格值

#### 追逐中的战技
- 成功的战技：目标损失1D3行动点 + 可能受伤
- 体格限制同普通战技

#### 追逐中的远程攻击
- **停下脚步**：正常射击（花费1行动点）
- **行进中射击**：所有射击检定承受**额外1个惩罚骰**（不花费额外行动点）
- **射击轮胎**：1个惩罚骰，3点护甲，2点伤害则爆胎

#### 驾驶员受伤
- 重伤但清醒：必须通过**困难难度的险境检定**维持控制
- 重伤失去意识：自动失去控制

### 7.5 可选规则

#### 猛踩油门
| 移动方式 | 行动点消耗 | 险境惩罚 |
|---------|-----------|---------|
| 普通移动 | 1行动点移动1个地点 | 无 |
| 加速（2-3地点） | 1行动点 | 1个惩罚骰 |
| 高速（4-5地点） | 1行动点 | 2个惩罚骰 |

- 必须在移动前声明
- 加速移动中险境检定失败则立即停止移动

#### 躲藏
- 逃离者可在追逐中尝试躲藏
- 追逐者需通过**侦查检定**才能发现
- 载具需**潜行+驾驶**组合检定

#### 突发险境
- 玩家或守秘人可要求**团体幸运检定**
- 成功：玩家放置险境
- 失败：守秘人放置险境
- 双方轮流提出要求

#### 乘客
- 无行动点，按DEX顺序可执行一次行动
- 可射击/投掷物品
- 导航/地图辅助：成功后载具加速少承受1个惩罚骰

#### 载具速度表
| 载具 | MOV | 体格 | 乘客护甲 | 乘客 |
|------|-----|------|---------|------|
| 经济型轿车 | 13 | 4 | 1 | 3或4 |
| 标准轿车 | 14 | 5 | 2 | 4 |
| 豪华轿车 | 15 | 6 | 2 | 4 |
| 跑车 | 16 | 5 | 2 | 1 |
| 中型卡车 | 13 | 7 | 2 | 2+ |
| 半挂式卡车 | 13 | 9 | 2 | 3+ |
| 轻型摩托车 | 13 | 1 | 0 | 1 |
| 重型摩托车 | 16 | 3 | 0 | 1 |
| 马（载骑手） | 11 | 4 | 0 | 1 |

#### MOV等级说明
- 每1点MOV约等于增加50%速度
- 健康人类：MOV 9（约12MPH，5分钟跑完1英里）
- 世界级短跑选手：MOV 11

#### 交通事故表
| 事故 | 体格伤害 | 范例 |
|------|---------|------|
| 轻微事故 | 1D3-1 | 侧面刮擦、擦过路灯柱 |
| 中度事故 | 1D6 | 撞到牛/大型鹿、与重型摩托车相撞 |
| 严重事故 | 1D10 | 与普通轿车/路灯柱/树木相撞 |
| 场面混乱 | 2D10 | 与卡车/巴士/大树相撞 |
| 道路惨案 | 5D10 | 与重型卡车/列车相撞 |

#### 多人追逐
1. 所有参与者进行速度检定
2. 按速度排序，分为追逐者和逃离者
3. 速度超过最快追逐者的逃离者可逃离
4. 速度低于最慢逃离者的追逐者被淘汰
5. 放置追逐者（最慢者在起始点，其余按速度差前移）
6. 放置逃离者（最慢者在追逐者前2个地点，其余按速度差前移）
7. 设置险境和障碍
8. 计算行动点
9. 确定敏捷顺序
