# 常见技术骗局特征清单

## 一、绝对红旗（发现即回避）

### 1.1 违反物理定律
以下任何一项出现，基本可判定为骗局或严重夸大：

| 技术领域 | 红旗信号 | 物理边界 |
|----------|----------|----------|
| 电池 | 能量密度 > 500 Wh/kg | 锂空气理论极限约500 Wh/kg |
| 电池 | 循环寿命 > 10万次 | 现有最优约5000次 |
| 电池 | 充电时间 < 5分钟(大容量) | 热管理极限 |
| 超导 | 室温超导无迈斯纳效应 | 超导定义：零电阻+完全抗磁性 |
| 超导 | 常压室温超导 | 目前所有室温超导需高压 |
| 热电 | 效率 > 卡诺效率 | 热力学第二定律 |
| 永动机 | 任何"永动机" | 能量守恒定律 |
| 量子计算 | 无量子纠错谈"实用" | 量子计算基础要求 |
| 核聚变 | Q > 1但无能量回收 | 当前Q值约0.5 |

### 1.2 履历造假
- 核心成员无任何可查学术/职业记录
- 声称"前NASA/Google/特斯拉"但无证据
- 学历/职称与实际不符
- 伪造诺贝尔奖/院士头衔

### 1.3 专利造假
- 专利号无法在专利局网站查询
- 专利内容与声称技术不符
- 专利已过期或被驳回

## 二、高度红旗（需重点核实）

### 2.1 技术参数可疑
- [ ] 性能指标比现有最优技术好10倍以上
- [ ] 无具体测试数据，只有"显著提升"等模糊表述
- [ ] 数据无法提供原始测试报告
- [ ] 测试条件不明确（温度、压力、环境）
- [ ] 性能指标来自单一来源，无第三方验证

### 2.2 商业化路径不清
- [ ] 声称"颠覆性技术"但无明确目标市场
- [ ] 声称市场规模万亿但无具体客户
- [ ] 无竞争分析或声称"没有竞争对手"
- [ ] 商业模式画布多个要素缺失
- [ ] 估值远超同阶段行业水平

### 2.3 团队问题
- [ ] 核心技术只有1人，声称"独家秘方"
- [ ] 团队无相关领域背景
- [ ] 核心成员不愿露面或匿名
- [ ] 团队频繁变动
- [ ] 顾问团队明显造假或挂名

### 2.4 宣传手法可疑
- [ ] 大量"突破性"、"革命性"、"颠覆性"用语
- [ ] 频繁出现在非学术媒体的夸张报道
- [ ] 声称"已通过验证"但不提供证据
- [ ] 使用名人站台但无技术背书
- [ ] 饥饿营销（名额有限、即将截止）

---

## 三、中度红旗（需警惕）

### 3.1 学术基础薄弱
- [ ] 无已发表的相关学术论文
- [ ] 论文发表在掠夺性期刊
- [ ] 论文被撤回或质疑
- [ ] 无专利布局或专利质量差
- [ ] 引用量极低

### 3.2 资金使用可疑
- [ ] 融资金额与团队规模不匹配
- [ ] 资金用途不透明
- [ ] 早期项目频繁更换融资主体
- [ ] 关联方交易过多

### 3.3 历史问题
- [ ] 创始人有创业失败/项目烂尾历史
- [ ] 公司曾有法律纠纷
- [ ] 关联方有负面新闻

---

## 四、低度红旗（需关注）

### 4.1 沟通层面
- [ ] 过度保密，不愿透露技术细节
- [ ] 回应问题避重就轻
- [ ] 承诺无法兑现
- [ ] 对竞品了解不足

### 4.2 文件层面
- [ ] 商业计划书粗糙，错误百出
- [ ] PPT有明显抄袭痕迹
- [ ] 法律文件不完善

---

## 五、按技术领域分类

### 5.1 新能源/电池
| 红旗信号 | 说明 |
|----------|------|
| 能量密度超理论极限 | 当前锂离子极限约300Wh/kg |
| 充电速度违反热管理 | 大容量电池快充会产生热失控 |
| 成本低于原材料 | 连原材料成本都不够 |
| 无着火/爆炸风险 | 目前无法完全避免 |
| 循环寿命无限 | 不符合物理衰减规律 |

**真实技术判断**：
- 看有没有国际期刊论文
- 看有没有车企/电池厂商合作
- 看有没有通过IEC等安全认证

### 5.2 量子科技
| 红旗信号 | 说明 |
|----------|------|
| 量子计算机"已实用" | 目前量子计算机仍在NISQ时代 |
| 量子通信无量子密钥分发 | 违背量子通信基本原理 |
| 量子传感器夸大精度 | 需对比传统传感器的实际优势 |
| 量子AI无量子优势证明 | 需 Shor 算法等证明 |

**真实技术判断**：
- 看是否在Nature/Science等顶刊发表
- 看是否有国家量子计算项目支持
- 看是否有IBM/Google等头部企业合作

### 5.3 AI/机器学习
| 红旗信号 | 说明 |
|----------|------|
| 声称AGI即将实现 | 需警惕过度的技术炒作 |
| AI超越人类但无benchmark | 需第三方评测 |
| AI能解决所有问题 | 特定领域AI有局限性 |
| 无需训练数据 | 违反机器学习基本原理 |
| 黑盒不可解释AI宣称透明 | 技术矛盾 |

### 5.4 生物医药
| 红旗信号 | 说明 |
|----------|------|
| 100%治愈率 | 违反医学统计规律 |
| 无副作用 | 不存在无副作用的药物 |
| 治愈周期极短 | 与疾病自然病程不符 |
| 无需临床试验 | 违反监管要求 |
| 独家秘方无专利 | 商业逻辑可疑 |

## 六、快速核查清单

### 第一分钟扫描
- [ ] 搜索"项目名 + 骗局/骗子/质疑"
- [ ] 搜索核心成员名字
- [ ] 验证专利号
- [ ] 检查物理边界

### 第三分钟深度
- [ ] 检索学术论文
- [ ] 核实商业模式
- [ ] 分析估值合理性
- [ ] 评估TRL等级

### 综合判断
- [ ] 红旗数量：1个绝对红旗 → 建议回避
- [ ] 红旗数量：3个以上高度红旗 → 建议回避
- [ ] 红旗数量：1-2个高度红旗 → 极度谨慎

---

## 七、真实vs虚假技术对比

| 维度 | 真实技术 | 虚假/夸大技术 |
|------|----------|---------------|
| 性能描述 | 有具体参数、测试条件 | 模糊描述、绝对化用语 |
| 数据来源 | 可提供原始数据、第三方验证 | 内部数据、无可查证来源 |
| 学术基础 | 有同行评审论文 | 无学术记录或掠夺性期刊 |
| 团队背景 | 可查证的真实履历 | 匿名、模糊、造假 |
| 商业计划 | 合理的市场分析 | 万亿市场、无竞争者 |
| 估值逻辑 | 与阶段匹配 | 远超行业水平 |
| 沟通态度 | 开放但保护IP | 过度神秘或过度吹嘘 |
| 时间线 | 合理可行 | 过于激进 |
