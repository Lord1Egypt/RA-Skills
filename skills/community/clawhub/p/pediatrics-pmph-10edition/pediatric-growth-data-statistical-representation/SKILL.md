# 儿童体格生长数据的统计学表示方法

## 何时使用本技能
- 需对儿童身高、体重等体格测量数据进行统计汇总或可视化
- 需将个体测量值与标准化生长参照数据比较
- 需绘制或解读生长曲线
- 需评估儿童生长是否在正常范围内

## 执行步骤

1. **判断数据分布形态**
   - 若数据近似正态分布，可优先考虑均值离差法或Z评分
   - 若数据呈偏态分布或分布未知，推荐使用百分位数法或中位数法

2. **选择合适的统计表示方法**
   - **均值离差法**：适用于正态分布数据，用均值 ± n个标准差（S）表示范围（±2S 覆盖约95%人群）
   - **百分位数法**：适用于各类分布，常用 P₃～P₉₇ 定义正常范围（覆盖94%人群）
   - **Z评分（标准差离差法）**：计算 Z = (X - X̄) / S，用于跨年龄、性别、指标的精确比较，正常范围为 [-2, +2]
   - **中位数法**：当存在极端值或非完全正态时，用中位数（P₅₀）代表中心趋势，因其稳健性优于算术均值

3. **解释结果并避免常见误区**
   - 输出统计位置（如 Z = -1.2 或 P₂₅）
   - 判断是否在正常范围：-2 ≤ Z ≤ 2 或 P₃ ≤ P ≤ P₉₇
   - **切勿将均值、P₅₀ 或中位数视为个体必须达到的“标准”**，应尊重生长个体差异

4. **确保前提条件满足**
   - 使用标准化生长参照数据（按年龄与性别分组）
   - 明确样本分布形态以指导方法选择
