# 車位拆算方法

## 為什麼要拆算？

台灣常見將車位含在總價內，但車位價值與房屋價值不同，需分開計算才能正確估值。

## 三種車位類型

| 類型 | 權狀登記 | 坪數計算 | 拆算方法 |
|------|----------|----------|----------|
| 法定車位 | 含在公設 | 約8~15坪 | 依公設比例估算 |
| 增設車位 | 獨立權狀 | 登記坪數 | 直接扣除登記坪數 |
| 機械車位 | 通常無權狀 | 使用權 | 約定2~5坪空間 |

## 拆算公式

```
不含車位總價 = 總價 − 車位價值
不含車位單價 = 不含車位總價 ÷ （總坪數 − 車位坪數）
```

| 地區 | 平面車位 | 機械車位(下層) | 機械車位(上層) |
|------|:--------:|:--------------:|:--------------:|
| 台北市蛋黃區 | 300~400萬 | 180~250萬 | 150~200萬 |

| 其他縣市 | 80~150萬 | 40~80萬 | 30~60萬 |

## 實際案例

### 案例一：大樓含車位
```
總價：2,500萬
總坪數：50坪
車位類型：平面車位（約10坪）
車位價值（新北市第一圈）：約220萬

不含車位總價 = 2,500 - 220 = 2,280萬
不含車位坪數 = 50 - 10 = 40坪
不含車位單價 = 2,280 / 40 = 57萬/坪
```

### 案例二：公寓無車位
```
總價：1,200萬
總坪數：30坪
無車位
單價 = 1,200 / 30 = 40萬/坪
（不需拆算）
```

### 案例三：機械車位
```
總價：1,800萬
總坪數：35坪（含機械車位使用權5坪）
車位類型：機械上層
車位價值（台北一般區）：約170萬

不含車位總價 = 1,800 - 170 = 1,630萬
不含車位坪數（概估）= 35 - 5 = 30坪
不含車位單價 = 1,630 / 30 = 54.3萬/坪
```

## 注意事項

1. **法定車位無法獨立買賣** → 拆算時不能分拆權狀
2. **車位管理費另計** → 一般300~600元/月（平面），500~2,000元/月（機械含保養）
3. **車位單獨出售行情** → 同社區車位過去成交價是最準參考
4. **坡道平面 vs 升降平面** → 坡道平面價值較高（+10~20%）
5. **機械車位尺寸** → 確認車輛是否可停放，無法使用等於無價值
