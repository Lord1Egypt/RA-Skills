## 快捷录入示例

### 示例1：数学选择題

```
数學 选择题
题目：已知 f(x) = x² - 4x + 3，求 f(x) 的最小值
选项：A.3 B.1 C.-1 D.0
错选：A
正解：C
原因：配方错误，f(x) = (x-2)² - 1
知识点：二次函数
```

### 示例2：物理计算题

```
物理
题目：质量为2kg的物体，在水平面上受到10N的拉力，摩擦力为4N，求加速度
错答：a = F/m = 10/2 = 5 m/s²
正答：a = (F-f)/m = (10-4)/2 = 3 m/s²
原因：忘记考虑摩擦力
知识点：牛顿第二定律
难度：简单
```

### 示例3：英语语法

```
英语
题目：He suggested that we ____ a meeting.
错答：will have
正答：(should) have
原因：suggest后的从句用虚拟语气，(should)+动词原形
知识点：虚拟语气
```

## 图片录入

提供图片路径时，系统会自动OCR识别：

```

科目：数学
知识点：二次函数
```
