# 最小必填字段

任何正式产出启动前，至少必须明确以下字段。

## 1. 任务定义

- 任务名称：必填
- 任务类型：必填
- 目标用户：必填
- 使用场景：必填
- 核心目标：必填
- 最终输出清单：必填

## 2. 生产边界

- 是否正式产出：必填
- 是否需要 SubAgent：必填
- 执行 owner：必填
- 审核 owner：必填
- 不允许做的事：必填

## 3. 素材依赖

- Logo：按需必填
- 二维码：按需必填
- 主角图/角色参考：漫画/人物视觉必填
- 风格参考：视觉类必填
- 数据来源：数据页必填
- 真实案例：案例页必填
- 产品截图/界面：产品/方案类按需必填

## 4. 样章门禁

- 样章范围：必填
- 样章确认人：必填
- 样章确认后才能进入哪一步：必填

## 5. 审核与停止条件

- 双审是否必需：必填
- PASS 标准：必填
- NEEDS_REVISION 标准：必填
- BLOCKED 标准：必填
- 最大返工轮数：必填
