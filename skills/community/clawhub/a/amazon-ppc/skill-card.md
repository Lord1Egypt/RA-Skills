## Description: <br>
Build Amazon Sponsored Products and Sponsored Brands campaign structures, keyword match strategies, and bid logic for profitable paid visibility. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External Amazon sellers, marketers, and account operators use this skill to design or restructure Amazon Ads campaigns, set margin-based bids, plan match-type strategy, manage negatives, allocate budgets, and run optimization reviews. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Campaign recommendations could lead to wasted ad spend or unprofitable bids if product margins, conversion rates, budgets, or account data are wrong. <br>
Mitigation: Verify unit economics, ACoS targets, budgets, and proposed campaign changes before applying them in Amazon Ads. <br>
Risk: Future versions could change the technical security posture if they add executable code, credentials, API access, or automatic account changes. <br>
Mitigation: Re-review and rescan future releases before deployment, especially if automation or account access is added. <br>
