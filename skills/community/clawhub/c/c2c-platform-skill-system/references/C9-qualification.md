# C9 资质证照与准入 — 准入专员

20 units organized into sub-steps:

## 步骤9.1 基础资质申请 (C9-01 to C9-03)

### C9-01 ICP备案申请

- **输入**: 域名证书(如 yidian-service.com)、服务器接入信息(阿里云ECS实例ID/IP)、营业执照扫描件、法人身份证正反面、网站负责人身份证及核验照、平台服务内容说明(C2C本地生活服务撮合平台)
- **输出**: ICP备案号(如京ICP备2026XXXXX号)、备案审核通过通知书、备案信息变更流程指引、备案年报提醒日历
- **依赖**: 无
- **AI自治度**: 🟨
- **组合接口**: → C9-02公安备案, → C9-03 EDI许可证, → C9-09应用商店上架, → C9-17证照到期提醒

### C9-02 公安备案申请

- **输入**: ICP备案号(C9-01)、网站域名及IP地址、平台交互式服务说明(用户注册/发布/评论/即时通讯)、信息安全管理制度、用户个人信息保护制度
- **输出**: 公安备案号、全国互联网安全管理服务平台备案凭证、交互式服务备案确认、安全评估报告存档
- **依赖**: C9-01
- **AI自治度**: 🟨
- **组合接口**: → C9-13等保备案, → C9-17证照到期提醒

### C9-03 EDI许可证申请

- **输入**: ICP备案号(C9-01)、营业执照(经营范围含"增值电信业务")、公司章程、验资报告(注册资本≥100万)、3名以上技术人员社保证明、平台业务说明(C2C撮合交易/易点虚拟币/3级裂变分润模式)、信息安全保障措施
- **输出**: EDI许可证(增值电信业务经营许可证-在线数据处理与交易处理业务)、许可证编号、经营范围、有效期、年检提醒
- **依赖**: C9-01
- **AI自治度**: 🟨
- **组合接口**: → C9-12增值电信年检, → C9-09应用商店上架, → C9-17证照到期提醒

## 步骤9.2 支付与通道资质 (C9-04 to C9-08)

### C9-04 支付通道资质审核

- **输入**: 营业执照、法人身份证、ICP备案号(C9-01)、EDI许可证(C9-03)、平台业务模式说明(服务商撮合+易点虚拟币+3级裂变分润)、银行开户许可证、组织机构代码证、公司对公账户信息
- **输出**: 支付通道资质审核报告，含资质完整性检查清单、各通道准入要求对比(微信/支付宝)、风险点标注(虚拟币模式合规性)、审核状态跟踪
- **依赖**: 无
- **AI自治度**: 🟨
- **组合接口**: → C9-05微信支付商户号, → C9-06支付宝商户号, → C8-10支付通道供应商

### C9-05 微信支付商户号申请

- **输入**: 支付通道资质(C9-04)、微信商户平台注册信息、对公银行账户信息、法人微信认证、经营场景说明(本地生活服务-家政/维修/跑腿/咨询)、费率申请(默认0.6%，大交易量可申请优惠)
- **输出**: 微信支付商户号(mch_id)、API密钥配置确认、商户平台登录权限、费率确认函、结算周期确认(T+1)、微信支付接入技术文档清单(JSAPI/H5/小程序/Native)
- **依赖**: C9-04
- **AI自治度**: 🟨
- **组合接口**: → C10-04支付系统架构, → C8-10支付通道供应商, → C9-17证照到期提醒

### C9-06 支付宝商户号申请

- **输入**: 支付通道资质(C9-04)、支付宝开放平台注册信息、对公银行账户信息、法人支付宝认证、经营场景说明、费率申请(默认0.6%)
- **输出**: 支付宝商户号(app_id)、RSA密钥对配置确认、支付宝开放平台权限、费率确认函、结算周期确认(T+1)、支付宝接入技术文档清单(App支付/H5支付/当面付)
- **依赖**: C9-04
- **AI自治度**: 🟨
- **组合接口**: → C10-04支付系统架构, → C8-10支付通道供应商, → C9-17证照到期提醒

### C9-07 短信通道资质审核

- **输入**: 营业执照、ICP备案号(C9-01)、短信使用场景说明(注册验证码/订单通知/营销短信)、短信模板样例、用户隐私保护制度
- **输出**: 短信通道资质审核报告，含阿里云/腾讯云短信服务准入要求对比、资质完整性检查、短信模板合规性评估(含退订字样/无诱导内容)、审核状态跟踪
- **依赖**: 无
- **AI自治度**: 🟨
- **组合接口**: → C9-08云服务短信接入, → C8-11短信通道供应商

### C9-08 阿里云/腾讯云短信接入

- **输入**: 短信通道资质(C9-07)、阿里云AccessKey/腾讯云SecretKey、短信签名申请(如"易点服务")、短信模板申请(验证码/通知/营销)
- **输出**: 短信服务接入配置，含签名ID与模板ID映射表、发送频率限制配置(验证码1条/分钟/营销50条/天)、发送回执配置、费用告警阈值
- **依赖**: C9-07
- **AI自治度**: 🟨
- **组合接口**: → C10-07消息队列架构, → C8-11短信通道供应商

## 步骤9.3 应用上架 (C9-09 to C9-11)

### C9-09 应用商店上架审核

- **输入**: ICP备案号(C9-01)、App名称与简介(C2C本地生活服务平台)、应用图标与截图、隐私政策链接、用户协议链接、应用权限说明(定位/相机/通讯录/存储)
- **输出**: 应用商店上架审核清单，含各商店审核要求对比(苹果/华为/小米/OPPO/vivo/应用宝)、审核常见拒审原因汇总、审核通过预计时间、审核状态跟踪
- **依赖**: C9-01
- **AI自治度**: 🟨
- **组合接口**: → C9-10 App Store上架, → C9-11安卓市场上架

### C9-10 苹果App Store上架

- **输入**: 应用商店审核准备(C9-09)、Apple开发者账号(年费$99)、App Store Connect配置信息、Flutter 3.11构建的IPA包、TestFlight测试版本、App审核信息(分类/年龄分级/定价)
- **输出**: App Store上架结果，含App ID、审核通过时间、审核反馈(如有被拒记录及解决方案)、版本更新提审流程、苹果审核指南关键条款标注(含虚拟货币/打赏相关2.1条款)
- **依赖**: C9-09
- **AI自治度**: 🟨
- **组合接口**: → C8-04上线发布, → C9-17证照到期提醒(开发者账号续费)

### C9-11 安卓应用市场上架

- **输入**: 应用商店审核准备(C9-09)、各市场开发者账号(华为/小米/OPPO/vivo/应用宝)、Flutter 3.11构建的APK/AAB包、各市场SDK适配(推送/更新/统计)、软著证书(C9-15)
- **输出**: 安卓市场上架结果，含各市场应用链接、各市场审核通过时间、各市场更新提审流程、各市场特殊要求备注(华为需隐私测评/小米需安全检测)
- **依赖**: C9-09
- **AI自治度**: 🟨
- **组合接口**: → C8-04上线发布, → C9-15软著登记

## 步骤9.4 年检与等保 (C9-12 to C9-16)

### C9-12 增值电信业务年检

- **输入**: EDI许可证(C9-03)、上年度经营报告(业务收入/利润/纳税)、公司年报信息、电信业务经营情况表、用户权益保护措施报告
- **输出**: 年检申报材料，含经营情况统计表、年检报告书、年检状态跟踪(待审核/通过/不通过)、年检整改意见(如有)、下次年检提醒
- **依赖**: C9-03
- **AI自治度**: 🟨
- **组合接口**: → C9-17证照到期提醒, → C8-37法务合同审核

### C9-13 数据安全等级保护备案

- **输入**: 系统定级报告(建议等保三级-C2C平台涉及用户资金/个人信息)、系统拓扑图(C10-01架构规划)、安全管理制度、数据分类分级方案(用户身份/交易/易点余额为敏感数据)、公安备案号(C9-02)
- **输出**: 等保备案证明，含备案编号、系统等级(三级)、备案信息表、公安机关审核意见、备案有效期
- **依赖**: 无
- **AI自治度**: 🟨
- **组合接口**: → C9-14等保测评准备, → C10-09安全体系设计, → C9-17证照到期提醒

### C9-14 等保测评准备

- **输入**: 等保备案证明(C9-13)、等保三级测评要求(技术要求/管理要求共73项)、现有安全措施清单(JWT认证/数据加密/审计日志/访问控制)、安全差距分析
- **输出**: 等保测评准备清单，含技术整改项(MySQL数据加密/RabbitMQ传输加密/Redis访问控制/日志审计完善)、管理整改项(安全组织架构/制度建设/人员管理/应急演练)、整改时间计划、测评机构推荐清单
- **依赖**: C9-13
- **AI自治度**: 🟨
- **组合接口**: → C10-09安全体系设计, → C10-12数据加密方案, → C10-14数据备份策略

### C9-15 软件著作权登记

- **输入**: 软件源代码(前后端核心模块60页)、软件设计文档、软件名称及版本号(如"易点本地生活服务平台V1.0")、营业执照、申请表
- **输出**: 软件著作权登记证书，含登记号、软件名称、著作权人、首次发表日期、权利范围、有效期
- **依赖**: 无
- **AI自治度**: 🟨
- **组合接口**: → C8-38知识产权保护, → C9-11安卓市场上架, → C9-17证照到期提醒

### C9-16 商标注册申请

- **输入**: 商标图样(易点品牌Logo/文字商标)、商标分类选择(第35类广告销售/第38类电信/第42类科技服务)、商标查询报告(近似风险)、营业执照、申请表
- **输出**: 商标注册申请受理通知书、商标注册证(通过后)、商标编号、保护范围(注册类别)、有效期(10年)、续展提醒
- **依赖**: 无
- **AI自治度**: 🟨
- **组合接口**: → C8-38知识产权保护, → C8-39品牌商标管理, → C9-17证照到期提醒

## 步骤9.5 资质运维 (C9-17 to C9-20)

### C9-17 证照到期提醒

- **输入**: 所有证照信息(ICP备案C9-01/公安备案C9-02/EDI许可证C9-03/微信商户号C9-05/支付宝商户号C9-06/苹果开发者C9-10/等保备案C9-13/软著C9-15/商标C9-16)、证照有效期、续期办理周期
- **输出**: 证照到期提醒日历，含30天/60天/90天三级预警、续期办理倒计时、续期所需材料清单、责任人指派、逾期风险说明(证照过期导致业务下架/罚款/监管处罚)
- **依赖**: C9-01~C9-16
- **AI自治度**: ⬛
- **组合接口**: → C9-18证照续期更新, → C8-22风险问题升级, → C8-31运营指标仪表盘

### C9-18 证照续期/更新流程

- **输入**: 到期提醒(C9-17)、续期申请材料、证照变更事项(公司信息/业务范围/技术方案变更)、审批流程
- **输出**: 续期/更新申请工单，含申请状态(准备中/已提交/审核中/通过)、新证照信息、变更记录、审批流程节点、办理进度跟踪
- **依赖**: C9-17
- **AI自治度**: 🟨
- **组合接口**: → C9-17证照到期提醒(更新), → C6合规审核

### C9-19 监管检查应对准备

- **输入**: 监管检查通知(网信办/工信部/公安/市场监管)、检查范围(数据安全/个人信息保护/交易合规)、现有合规文档(C6合规审核输出)、等保测评报告(C9-14)
- **输出**: 监管检查应对方案，含检查材料准备清单(营业执照/备案证/安全制度/审计日志/数据脱敏方案)、迎检人员安排、自查问题整改清单、模拟检查Q&A、历史检查问题回顾
- **依赖**: 无
- **AI自治度**: 🟨
- **组合接口**: → C6合规审核, → C9-20准入合规自查, → C8-41舆情危机响应

### C9-20 准入合规自查清单

- **输入**: 所有证照状态(C9-01~C9-16)、到期提醒(C9-17)、等保测评结果(C9-14)、监管检查记录(C9-19)、业务模式变更(新类目/新功能/易点规则变更)
- **输出**: 准入合规自查报告，含证照完整性检查(是否齐全/是否过期)、业务模式合规性评估(易点虚拟币是否符合非货币预付卡规定/3级裂变分润是否涉嫌传销/服务商准入是否完备)、风险项清单(高/中/低)、整改建议与时间表
- **依赖**: C9-01~C9-16
- **AI自治度**: ⬛
- **组合接口**: → C6合规审核, → C9-19监管检查应对, → C8-22风险问题升级
