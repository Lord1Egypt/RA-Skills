# 劳动法 Skill 库 — labor-dispute

> 劳动者维权技能体系。覆盖从入职到仲裁诉讼全流程。

## 包含的8个技能卡

| # | 技能卡 | 触发场景 | 核心法条 |
|---|---|---|---|
| 1 | `contract-signing/` 劳动合同签订防坑 | 入职签合同 / 试用期谈判 / 签保密协议 | 劳动合同法第7-25条 |
| 2 | `compensation-n/` 经济补偿N计算 | 离职补偿金计算 / 仲裁赔偿谈判 | 劳动合同法第46-47条 |
| 3 | `illegal-dismissal-2n/` 违法解除赔偿2N | 被解除合同 / 违法裁员 / 离职协议谈判 | 劳动合同法第48条、第87条 |
| 4 | `labor-arbitration-process/` 劳动仲裁全流程 | 申请仲裁 / 仲裁进行中 / 对裁决不服 | 调解仲裁法 + 司法解释(一) |
| 5 | `evidence-burden/` 举证责任分配 | 准备仲裁 / 收集证据 / 证明加班 | 司法解释(一)第42-44条 |
| 6 | `non-compete/` 竞业限制权利义务 | 签竞业协议 / 离职后被追责 | 劳动合同法第23-24条 + 司法解释第36-40条 |
| 7 | `social-insurance/` 社保维权路径 | 发现单位没缴社保 / 要求补缴 | 劳动法第72-76条 + 劳动合同法第38条 |
| 8 | `work-injury-compensation/` 工伤/职业病赔偿 | 发生工伤 / 职业病 / 单位侵权 | 劳动法 + 民法典第1179-1192条 |

## 法律渊源

| 法律文件 | 适用内容 |
|---|---|
| 劳动合同法（2012修正） | 合同签订/解除/补偿 |
| 劳动法（2009修正） | 基准制度（工时/工资/安全卫生） |
| 劳动争议调解仲裁法（2007） | 仲裁程序/时效/终局裁决 |
| 劳动争议司法解释(一)（2021） | 裁判规则/举证/执行 |
| 民法典第1179-1192条 | 工伤/职业病 人身损害赔偿 |

## 快速决策树

```
遇到劳动问题 → 判断场景
  ↓
入职/签合同 → contract-signing（10项检查）
  ↓
计算补偿金 → compensation-n（N=年限×月工资）
  ↓
被解除合同 → illegal-dismissal-2n（6种禁止解除情形）
  ↓
申请仲裁 → labor-arbitration-process（6步流程）
  ↓
收集证据 → evidence-burden（劳动关系三层证据表）
  ↓
竞业限制 → non-compete（2年/30%/3个月三数字）
  ↓
社保问题 → social-insurance（补缴找社保局）
  ↓
工伤/职业病 → work-injury-compensation（两套系统对比）
```

## 核心口诀

**入职：查三样——期限、工资、社保**
**离职：算补偿——N还是2N先判断**
**仲裁：备证据——劳动关系证明优先**
**工伤：双轨走——工伤保险+民法典都算一遍**
