## Why This Exists

Every generic AI tool writes "duct cleaning improves your air quality" — which directly contradicts the EPA's own published guidance stating that "duct cleaning has never been shown to actually prevent health problems." This creates FTC deceptive practices exposure for every contractor using that copy.

Add the NADCA certification misrepresentation problem ("EPA certified"), the biocide FIFRA trap (applying unregistered chemicals = federal pesticide law violation), and the mold claim liability minefield — and most duct cleaning marketing is a legal liability waiting to happen.

This kit enforces 7 compliance moats so the output is accurate, defensible, and differentiated.

Serving property managers, HOAs, schools, medical offices. Key need: ASHRAE 180 compliance framing for commercial; OSHA General Duty Clause IAQ awareness; post-COVID IAQ documentation for facilities managers.

Serving multiple duct cleaning clients. Key need: scalable template system that doesn't create FTC liability across client portfolio; NADCA compliance verification workflow.
