## 证券投资任务清单

证券投资按子域组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：投资研究 → 投资决策 → 投资执行 → 投资管理

---

| ID | 任务类型 | 说明 | 依赖 | 元操作映射 |
|----|----------|------|------|----------|
| T1-1 | 市场研究 | 研究宏观经济和市场趋势，包括GDP、通胀、利率、政策 | 无（入口） | S→C |
| T1-2 | 行业分析 | 分析行业发展趋势，包括行业周期、竞争格局、政策影响 | T1-1 | S→C |

## 依赖拓扑摘要

### 一、域内链路

**投资研究链路**: T1-1 → T1-2 → T1-3 → T1-4
**投资决策链路**: T2-1 → T2-2 → T2-3 → T2-4
**投资执行链路**: T3-1 → T3-2 → T3-3 → T3-4
**投资管理链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整证券投资链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T3-1 → T3-2 → T3-3 → T3-4 → T4-1 → T4-2 → T4-3 → T4-4

**反馈改进链路**: T4-4 → T1-4（投资调整反馈到投资策略）

**风险驱动链路**: T4-2 → T2-2（风险控制驱动风险评估）

更多组合根据具体任务动态推导。
