# Fortune Rules — 离线确定性规则

让同样输入尽量得到稳定结果。所有计算都是确定性算术,无随机。

## 1) 输入归一化

- **生日 B**:`YYYY-MM-DD`,取年月日数字相加。如 `1990-05-15` → `B = 1990 + 5 + 15 = 2010`。
- **查询日期 D**:同样规则。如 `2026-05-18` → `D = 2026 + 5 + 18 = 2049`。
- **星期 W**:查询日期的 weekday,周日=0,周一=1,…,周六=6。
- **问题类型**:`overall | wealth | love | career | study | health | social | interview | exam | dating | launch | business-trip | couple-today`

若未提供生日:用 `B = 0`,运势细度降级,只输出基础四元素 + 评级,跳过"详细拆解"中的命格基调一项。

## 2) 基础数值

```
R       = (B * 131 + D * 17) mod 1000
W       = weekday(D)
S_base  = 40 + (R mod 45)
bonus   = 8 if W in {1, 3, 5} else 3
penalty = 10 if (R mod 7) == 0 else 0
S       = clamp(S_base + bonus - penalty, 0, 100)
```

评级:

| 区间 | 等级 |
|---|---|
| 80-100 | 大吉 |
| 65-79  | 中吉 |
| 45-64  | 平 |
| 0-44   | 小凶 |

## 3) 幸运四元素

### 3.1 幸运方位 — 8 选 1

```
dir_idx = (R + W) mod 8
```

| idx | 方位 |
|---|---|
| 0 | 正北 |
| 1 | 东北 |
| 2 | 正东 |
| 3 | 东南 |
| 4 | 正南 |
| 5 | 西南 |
| 6 | 正西 |
| 7 | 西北 |

### 3.2 幸运颜色 — 主色 + 辅色

```
c1 = (R div 7) mod 10
c2 = (R div 13 + W) mod 10
若 c2 == c1: c2 = (c2 + 1) mod 10
```

颜色表(idx 0-9):红、橙、黄、绿、青、蓝、紫、白、金、黑。

### 3.3 幸运数字 — 2-3 个

```
n1 = (R mod 9) + 1
n2 = ((R div 9) mod 9) + 1
n3 = ((R div 81) mod 9) + 1
```

去重后取 2-3 个(若三个都相同则 fallback 取 `n1, (n1 mod 9) + 1`)。

### 3.4 幸运物品 — 12 选 1

```
item_idx = (R + n1 + n2) mod 12
```

物品表(idx 0-11):记事本、银色钥匙扣、绿色植物、蓝色水杯、红绳手链、金属书签、薄荷糖、白色手帕、木质小摆件、香囊、护手霜、雨伞。

## 4) 分项分数

```
wealth = clamp(S + ((R mod 11) - 5),  0, 100)
love   = clamp(S + ((R mod 13) - 6),  0, 100)
career = clamp(S + ((R mod 9)  - 4),  0, 100)
social = clamp(S + ((R mod 7)  - 3),  0, 100)
```

## 5) 宜 / 忌池

每天从池中取 3-5 条,索引公式:`idx_i = (R + i) mod len(pool)`。

### 5.1 宜池(9 条)

1. 先做一件最重要的小任务
2. 上午处理沟通事项
3. 整理桌面与文件
4. 主动表达感谢
5. 进行 20 分钟运动
6. 用幸运色配件出门
7. 朝幸运方位出行或就座
8. 记一笔今日收支
9. 晚上早睡 30 分钟

### 5.2 忌池(9 条)

1. 冲动消费
2. 情绪化回复消息
3. 同时开启过多任务
4. 熬夜刷短视频
5. 与人硬碰硬争辩
6. 忽略备份与检查
7. 临时做高风险决定
8. 拖延关键事项
9. 空腹喝咖啡

### 5.3 强禁忌预警

定义风险分:`risk = 100 - S`

| risk | 输出几条强禁忌 |
|---|---|
| ≥ 50 | 2 条 |
| 30-49 | 1-2 条 |
| < 30 | 1 条(轻量) |

候选事项池(按条件触发):

| 编号 | 事项 | 触发条件(任一即启用) |
|---|---|---|
| 1 | 不宜出门远行 | `risk ≥ 50` 或 `(R + W) mod 6 ∈ {0, 3}` |
| 2 | 不宜使用铁器/刀剪 | `risk ≥ 45` 或 `(R mod 8) ∈ {1, 4, 7}` |
| 3 | 不宜夜间独行 | `social ≤ 45` 或 `(R mod 5) == 0`,时间窗 21:00 当日 - 01:00 次日 |
| 4 | 不宜签高风险承诺 | `career ≤ 48` 或 `wealth ≤ 48`,时间窗:即刻起至 end |

### 5.4 时间窗计算

```
start  = 查询日期 00:00
hours  = 12 + (R mod 25)     # 12-36 小时
end    = start + hours
```

若高风险(≥50),追加第二条更长时间窗:

```
hours2 = 36 + (R mod 37)     # 36-72 小时
end2   = start + hours2
```

每条强禁忌必须输出四段:**禁忌事项 / 禁忌时间 / 原因 / 替代方案**。禁止只写"不要做 X"而不写时间和原因。

## 6) 本周模式(weekly)

输入查询周(默认本周,周一为起点)。对 7 个日期分别计算 `R_i` 和 `S_i`,然后:

- **峰值日**:`argmax(S_i)` — 最适合冲刺/重要决策
- **低谷日**:`argmin(S_i)` — 最适合休整/避免高风险动作
- **平均评级**:`avg(S_i)` 转评级
- **周序列**:列出每日评级一行

输出:

```
【本周运势 | YYYY-MM-DD ~ YYYY-MM-DD】

✨ 整周评级:中吉 (avg 72)
📈 峰值日:YYYY-MM-DD (大吉 88) — <一句话用途建议>
📉 低谷日:YYYY-MM-DD (平 52) — <一句话避雷建议>

🗓️ 每日速览
- 周一 YYYY-MM-DD:中吉 70 · 方位 <…> · 主色 <…>
- 周二 YYYY-MM-DD:平 60 · ...
- ...

✅ 周宜
1. <动作 1>
2. <动作 2>
3. <动作 3>

❌ 周忌
1. <规避 1>
2. <规避 2>
3. <规避 3>

⚠️ 本周强禁忌(选低谷日 + 周累计 risk ≥ 80 时触发)
- 禁忌事项 / 时间 / 原因 / 替代方案
```

## 7) 本月模式(monthly)

对当月每天计算 `S_i`(可批量执行,只列出焦点日)。然后:

- **焦点日**:取 `S_i` 前 25% 的日期(约 7-8 天),按日期排序输出
- **避坑日**:取 `S_i` 后 10% 的日期(约 3 天)
- **月主题**:由月份 `M` 与生日 `B` 决定的简单建议(`tag = (B + M) mod 6` → 6 个月主题轮换)

月主题表(0-5):

| tag | 月主题 |
|---|---|
| 0 | 积累与基础 |
| 1 | 沟通与连接 |
| 2 | 突破与决断 |
| 3 | 收获与整理 |
| 4 | 反思与休整 |
| 5 | 出走与变化 |

输出:

```
【本月运势 | YYYY-MM】

✨ 月度主题:<月主题>
📈 焦点日(优先安排重要事):YYYY-MM-DD, YYYY-MM-DD, ...
📉 避坑日(避免重大决定):YYYY-MM-DD, ...

🌟 月度专项
• 财运:<结论>
• 桃花:<结论>
• 事业:<结论>
• 人际:<结论>

💡 月度建议
1. <策略 1>
2. <策略 2>
3. <策略 3>
```

## 10) 解释模板

输出"🔮 详细拆解"时按顺序展开:

1. **命格基调**:由 `B` 的奇偶 + 数位和(`sum_digits(B)`)给"偏稳 / 偏动"判断。偏稳:数位和为偶且 `B` 是偶数;偏动:其他。
2. **日期气场**:由 `D` 和 `W` 给出"偏沟通 / 偏执行 / 偏收敛"。
   - `W in {1, 3}` → 偏沟通
   - `W in {2, 4, 5}` → 偏执行
   - `W in {0, 6}` → 偏收敛
3. **叠加结论**:为什么得出当前评级;为什么是这些方位/颜色/数字/物品。必须引用上面的具体数值(`R`、`S`、`dir_idx` 等)。

## 11) 一致性

同一输入下,核心元素(评级、方位、颜色、数字)必须稳定。允许:

- 解释措辞变化(同一结论用不同句式)
- 宜忌池条目从同索引附近 ±1 取

不允许:

- 同一日不同次查询给出不同评级
- 幸运色/方位频繁切换
