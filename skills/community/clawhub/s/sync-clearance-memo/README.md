# sync-clearance-memo

Evaluate the clearance path for a composition and master recording and produce a DRAFT sync clearance memo for film, TV, advertising, video games, and branded content placements.

## When to use this skill

Use this skill when a music supervisor, sync licensing director, music publisher, or artist manager needs to assess the rights landscape for a song placement and document findings in a structured clearance memo before entering deal negotiations.

## What it produces

- Composition rights map (writers, shares, publishers, PRO affiliations, clearance contacts)
- Master rights map (label or artist ownership, documentation source)
- One-stop clearance assessment with complexity rating (Low / Medium / High)
- Sample chain verification with blocking-issue flags for uncleared samples
- Territory and term restriction notes
- Indicative fee range table by use type and media
- DRAFT clearance memo with open-items list and unsigned licensing-team review block

## Inputs needed

| Input | Description |
|---|---|
| Song title and ISRC | Identifies the recording |
| Songwriter and publisher names | Starting point for composition rights research |
| Master rights holder | Label, independent artist, or distributor |
| Intended use | Media type, scene or ad description, territory, term, exclusivity |
| Sample status | Does the recording contain any sampled or interpolated material? |

## Safety notes

- Output is a DRAFT memo only — not a license, rights authorization, or legal opinion.
- All fee estimates are indicative market ranges only — not binding quotes or offers.
- Uncleared samples are flagged as blocking issues; placement must not proceed until resolved.
- Confidential deal terms shared during intake are not logged or disclosed in the output.
- Rights determinations require confirmation from actual rights holders before any deal is executed.
