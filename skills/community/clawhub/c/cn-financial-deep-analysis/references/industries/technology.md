# 科技业（TMT）

## 核心财务特征

- 轻资产为主，人力资本密集
- 研发费用率高（5-20%），无形资产占比高
- 毛利率跨度大（软件60-90%，硬件15-40%）
- 成长性高但波动大，产品生命周期短
- 股权激励费用影响利润

## 关键分析指标

| 指标 | 健康值 | 警戒值 | 说明 |
|------|--------|--------|------|
| 研发费用率 | >8% | <3% | 技术驱动型公司核心投入 |
| 研发资本化率 | <30% | >50% | 过高则存在利润操纵嫌疑 |
| 毛利率 | 软件>60% 硬件>25% | 低于行业均值15% | 技术壁垒的直接体现 |
| 人均创收 | >50万 | <30万 | 人效指标 |
| 应收账款周转天数 | <90天 | >150天 | 软硬件差异大 |
| 无形资产/总资产 | 20-40% | >60% | 商誉减值风险 |

## 风险评估重点

- 技术迭代风险：产品生命周期、研发管线储备
- 人才流失风险：核心技术人员持股、薪酬竞争力
- 客户集中风险：大客户依赖（尤其政企客户）
- 商誉减值：并购 > 2年前标的业绩不达预期
- 应收账款：部分行业账期长（安防、政企软件）

## 舞弊高发区

- 研发费用过度资本化（资本化率 > 50%）
- 软件确认收入提前（签订合同即确认而非验收确认）
- 关联交易虚增收入（合资公司循环交易）
- 商誉不减值（业绩承诺未达标仍不减值）
