# 生时校正交互提示词

## 开场白
```
🔮 生时校正系统已启动

检测到您的命盘存在高影响校验点，系统已自动生成 {count} 个候选校正方案。

生时校正原理：
通过 ±1、±2 时辰偏移，重新排盘并评估置信度变化，选择最优方案。

请从以下候选方案中选择一个：
{table}

提示：
- 置信度越高越好（建议 ≥0.7）
- 高影响校验点越少越好（建议 =0）
- 可参考"关键变化"辅助判断
```

## 候选方案对比表模板
```
方案  偏移        置信度  高影响点  关键变化
----  ----------  -------  --------  -------------------------
0     {shift0}    {conf0}  {high0}    {changes0}
1     {shift1}    {conf1}  {high1}    {changes1}
2     {shift2}    {conf2}  {high2}    {changes2}
```

## 用户选择确认
```
✅ 您选择了方案 {index}：{shift_description}
校正时间：{corrected_dt}
置信度：{confidence:.3f}
关键变化：
{changes_list}

是否确认？(y/n)
```

## 跳过校正
```
⏭️ 您选择跳过生时校正。
维持原命盘不变。
如需后续校正，可重新发起请求。
```

## 无候选方案
```
⚠️ 无法生成有效校正候选。

可能原因：
1. 出生时间格式错误
2. 依赖的 ziwei 技能未就绪
3. 所有候选均超时或失败

建议：
- 检查出生时间参数
- 确认 ziwei 技能可用
- 稍后重试
```

## 低置信度警告
```
⚠️ 自动校正置信度不足

最佳方案：方案 {index}（{shift_description}）
置信度：{confidence:.3f}（阈值要求 ≥{threshold:.2f}）
剩余高影响点：{high_impact}

建议：
1. 人工复核该命盘
2. 尝试交互模式手动选择
3. 检查 birth_dt 是否准确

是否仍要应用此方案？(y/n)
```
