# 企服助手 · 能力概览（4 大板块）

> 2026-06-11 更新：从"7 大功能模块"重构为"4 大板块"，按企服人员日常工作流组织。

---

```
客户管理 | 费用催缴 | 续租预警 | 工单分派 | 库存监控 | 增值服务 | 走访计划
```

## 4 大板块详解

### ☀️ 今日工作台（每日操作面板）

企服人员每天打开的第一站，所有日常工作在此完成。

| 模块 | 功能 |
|------|------|
| 🔴 今日重点关注 | 风险企业（欠费/续租/高频报修/低满意度），自动标记 |
| 📋 今日走访安排 | A级/续租/欠费自动排程，含路线建议 |
| ⚡ 今日待办清单 | 费用催缴 | 三级逾期→话术→催缴→闭环 |
| | 工单处理 | 自动分派→SLA→超时升级→闭环 |
| | 续租跟进 | 三级预警→价值评估→方案→流转招商 |
| | 服务商机 | 需求识别→服务推荐→成交推进→闭环 |
| 📊 工作概览 | 待办数量、已完成、目标达成 |

### 🎯 企业服务（服务支撑能力）

走访和沟通中随时查阅的支撑工具。

| 模块 | 功能 | 来源 |
|------|------|------|
| 客户档案查询 | 按单元号/租户名/合同号快速查询 | 原模块一 |
| 企业画像分析 | 聚合费用、服务、报修、能耗记录 | 原模块一 |
| 风险标记计算 | 7级风险自动标记 | 原模块一 |
| 赋能服务推荐 | 8大子类（政务/人才/金融/知产/政策/法律/数字化/社群） | 原模块六 |
| 入驻前/中/后清单 | 按时间线提供对应服务 | 原模块六 |
| 一企一策策略 | 9种企业场景自动匹配个性化方案 | 原模块六 |

### 📦 后勤保障

| 模块 | 功能 | 来源 |
|------|------|------|
| 库存监控 | 水位计算、四级分级（缺货/预警/关注/正常） | 原模块五 |
| 补货提醒 | 按级别生成采购建议 | 原模块五 |

### 📝 闭环记录

干完活必须记录的入口。闭环→更新状态→生成下一步待办→反哺明日工作台。

| 模块 | 功能 | 来源 |
|------|------|------|
| 走访记录录入 | 租户名/时间/情绪/详情 | 原模块七 |
| 服务记录归档 | 服务类别/成交金额/反馈 | 原模块六 |
| 工单闭环确认 | 维修结果/客户确认/满意度 | 原模块四 |
| 客户状态更新 | 缴费确认/备注 | 原模块二 |

## 一企一策场景映射

| 企业场景 | 策略定位 | 对应待办 |
|---------|---------|---------|
| 初创科技企业 | 科创加速包 | 政策申报+融资对接+人才招聘 |
| 制造型企业 | 制造业稳商包 | 安环能评+安全培训+职称申报 |
| 龙头/标杆企业 | VIP专属包 | 资源对接+私董会+品牌宣传 |
| 欠费风险企业 | 柔性方案 | 经营诊断+分期方案+服务置换 |
| 续租到期企业 | 稳商包 | 空间优化+租金定制+产业链价值说明 |

## Skill 归属映射

| Skill | 自然归属 | 说明 |
|-------|---------|------|
| 投诉处理 | ⚡ 待办清单 → 工单处理 | 投诉本质是紧急工单，共用SLA监控 |
| 履约异常 | 🔴 今日重点关注 | 风险企业自动包含欠费/续租/履约异常 |
| 客户入驻 | 🎯 企业服务 | 入驻是"服务"的一部分，走访/沟通时查阅 |
| 满意度调查 | 每年两次专项任务 | 到时间 Agent 提醒，不融入日常面板 |
| 运营监测 | on-demand 触发 | 键入"季度报告"时触发生成 |
