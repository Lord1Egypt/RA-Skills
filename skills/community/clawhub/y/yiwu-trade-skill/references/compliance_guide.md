# 1039市场采购贸易合规指南

## 一、政策背景

### 什么是1039市场采购贸易？

1039市场采购贸易方式是海关总署专为小商品多品种、小批量出口设计的监管方式，2011年在义乌率先试点，2014年上升为国家战略。

### 核心优势
- **免征不退**：无需增值税发票即可出口，免征增值税
- **简化申报**：按章归类，单票货值15万美元以下简化报关
- **灵活收汇**：个体户/小微企业可合法自主收汇结汇
- **通关便利**：通关时效缩短50%以上

### 适用条件
- 注册地在义乌市
- 通过义乌市场采购商品
- 单票货值≤15万美元
- 在指定口岸报关

## 二、资质办理全流程

```

- 经营场所证明（租赁合同/自有房产证）
```

```
审核时间：1-3个工作日
```

## 三、合规操作要点

### 3.1 货物采购合规

#### 必须通过义乌市场采购
```
合规采购地：
✅ 义乌国际商贸城1-5区
✅ 义乌篁园市场
✅ 义乌生产资料市场
✅ 义乌周边专业市场（经认定）

非合规采购：
❌ 直接从外地工厂采购（需走一般贸易）
❌ 从其他批发市场采购
```

#### 单证要求
```
必备单证：
- 采购合同/订单
- 装箱单
- 商户档口信息
- 货物照片

建议保留：
- 付款凭证
- 拿货视频/照片
- 供应商名片
```

### 3.2 报关申报合规

#### 货值申报
```
申报原则：
- 真实申报实际成交价
- 申报价≥实际成交价70%
- 单票≤15万美元

常见错误：
❌ 低报货值逃避关税（目的国风险）
❌ 高报货值骗取退税（虽1039不退税，但影响统计）
```

#### 商品归类
```
简化申报规则：
- 按"章"归类（前4位税号）
- 无需精确到10位
- 特殊商品除外（食品、化妆品等）

注意事项：
- 敏感商品需精确归类
- 侵权商品禁止出口
- 危险品需特殊申报
```

### 3.3 收汇结汇合规

#### 合规收汇路径
```
路径一：义支付（Yiwu Pay）
- 直接连Chinagoods平台
- 自动关联订单
- 实时结汇

路径二：银行柜台结汇
- 携带合同、报关单
- 填写结汇申请书
- 银行审核后结汇

路径三：网银自助结汇
- 银行开通网银外汇功能
- 在线操作结汇
- 实时到账
```

#### 结汇红线
```
严禁行为：
❌ 地下钱庄换汇
❌ 虚假贸易骗汇
❌ 拆分结汇规避监管
❌ 私账收公款

合规要求：
- 贸易背景真实
- 收汇金额与报关一致
- 单证留存3年以上
```

## 五、常见问题解答

### Q1：1039和一般贸易有什么区别？

| 项目 | 1039市场采购 | 一般贸易 |
|------|-------------|----------|
| 增值税发票 | 不需要 | 必须提供 |
| 出口退税 | 无（免征不退） | 可申请退税 |
| 报关方式 | 简化申报 | 完整申报 |
| 适用范围 | 小商品多品种 | 大批量单一品种 |
| 收汇方式 | 个体户可收汇 | 需企业账户 |

### Q2：单票超过15万美元怎么办？

```
方案一：拆单
- 合理拆分为多票
- 每票≤15万美元
- 注意拆单合理性

方案二：转一般贸易
- 改走一般贸易报关
- 需要增值税发票
- 可申请退税
```

### Q3：外地货物可以走1039吗？

```
原则上不行，但有两种方式：
1. 货物进入义乌市场集货仓
   - 需办理入库手续
   - 取得市场采购证明

2. 通过义乌供应商采购
   - 供应商在义乌有档口
   - 采购合同在义乌签订
```

### Q4：被海关查验怎么办？

```
查验流程：
1. 海关通知查验
2. 货主/代理到场
3. 开箱检查
4. 确认无误放行

应对要点：
- 单证齐全，现场出示
- 货物与申报一致
- 配合海关工作
- 查验费用由货主承担

查验风险：
- 申报不实 → 改单/罚款
- 侵权产品 → 扣货/罚款/追究责任
- 走私嫌疑 → 立案调查
```

## 六、合规风险案例警示

### 案例1：低报货值被目的国罚款

```
情况：出口到某欧洲国家，申报价$5000，实际成交$8000
后果：目的国海关查验，认定低报，罚款$2400（3倍差价）
教训：申报价要真实，低报省不了钱反而风险大
```

### 案例2：侵权产品被扣

```
情况：出口带有某品牌logo的产品，未获授权
后果：海关扣货，罚款，列入黑名单
教训：选品必须排查侵权风险
```

### 案例3：地下钱庄换汇被查

```
情况：为图方便，通过地下钱庄换汇人民币
后果：账户被冻结，接受调查，罚款
教训：收汇必须走正规银行渠道
```
