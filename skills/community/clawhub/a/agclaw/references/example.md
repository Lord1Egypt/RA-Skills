## 示例2：创意卖点设计 + 广告脚本生成

### 输入

```
超自然行动组的创意卖点设计，并根据其核心卖点生成一个游戏广告素材创作脚本
```

### 输出

---

#### 一、《超自然行动组》创意卖点提炼

**核心定位：** 中式恐怖探宝 + 多人联机搞笑 — 1人玩是恐怖游戏，4人玩是喜剧游戏。代言人：赵露思（强化"闺蜜/哥们一起玩"的社交氛围）。

##### 卖点一：摸金暴富，开棺出大金

| 层次 | 内容 |
|------|------|
| **功能卖点** | 开棺/开柜/开金冠掉落随机珍宝（鬼玺、天山雪莲、双鱼玉佩、圣杯、大宝剑等） |
| **绑定场景** | 非酋连续破产 → 突然开出大金 → 逆袭撤离 |
| **解决的情绪** | 日常生活中的"运气不好"焦虑 → 逆袭的快感满足 |

##### 卖点二：一人恐怖，四人爆笑（社交反差）

| 层次 | 内容 |
|------|------|
| **功能卖点** | 1-4人联机组队，中式恐怖场景+灵异怪物 |
| **绑定场景** | 朋友/闺蜜/同学约着一起玩，本来是恐怖氛围结果全员抽象搞笑 |
| **解决的情绪** | 一个人不敢玩恐怖游戏的"怂" → 和损友一起"坑队友"的欢乐 |

##### 卖点三：角色技能差异化

| 层次 | 内容 |
|------|------|
| **功能卖点** | 多个角色各有专属技能：薇薇安（重铸开棺→双倍机会）、艾伦（扫描定位棺椁）、爱德华（飞行）、小黄（快速位移）、蓝（优雅输出） |
| **绑定场景** | 连续不出货 → 换角色/使用技能 → 瞬间翻盘 |
| **解决的情绪** | 随机性带来的"无力感" → 技能操控带来的"掌控感" |

##### 卖点四：非酋逆袭，绝境翻盘（情绪过山车）

| 层次 | 内容 |
|------|------|
| **功能卖点** | 撤离倒计时压力 + 高价值宝藏随机掉落 + 怪物追击/传送器机制 |
| **绑定场景** | 一路倒霉啥也没摸到 → 最后关头搏一把 → 绝境出大金 → 极限撤离 |
| **解决的情绪** | "今天运气真差"的沮丧 → 绝境翻盘的极致肾上腺素满足 |

##### 卖点五：赵露思代言 + 大厂品质（信任背书）

| 层次 | 内容 |
|------|------|
| **功能卖点** | 巨人网络出品，赵露思全球代言人，周年庆/联动活动丰富 |
| **绑定场景** | "露丝同款快乐" → 玩家想和偶像玩一样的游戏 |
| **解决的情绪** | "这游戏靠谱吗？"的信任疑虑 → "明星都在玩，肯定好玩"的从众心理 |

**核心公式：恐怖氛围 × 多人联机 × 随机爆率 = 高情绪波动 + 高社交传播**

---

#### 二、广告脚本创作

**脚本类型：** 真人实拍 + 游戏混剪 | **时长：** 30-45秒 | **风格：** 搞笑反转 + 真实感

**目标玩家：** 18-28岁泛社交型玩家，喜欢和朋友/闺蜜一起组队玩游戏，对中式恐怖又怂又爱玩

##### 【第1段：痛点钩子 — 怂人设建立】0-5秒

| 画面 | 台词 |
|------|------|
| 真人出镜：女生/男生对着镜头，表情有点怂 | "我闺蜜非要拉我玩这个超自然行动组，说是什么中式恐怖探宝……" |
| 快速闪现游戏昏暗地宫场景 + 怪物突然出现 | "我一看这画面，我说算了吧，我胆子小。" |

##### 【第2段：情绪转折 — 从恐怖到搞笑】5-15秒

| 画面 | 台词 |
|------|------|
| 切到组队界面：四个人一起进入游戏 | "结果架不住她天天念叨，我就上了。" |
| 队友被蜘蛛怪追着跑撞墙、被鬼新娘吓到尖叫 | "进去发现……确实吓人！但你猜怎么着？我三个队友比我叫得还响！" |
| 快速剪辑：队友各种抽象操作 | "本来恐怖的气氛，被这群活宝整得我笑到肚子疼。" |

##### 【第3段：游戏机制植入 — 摸金爽点展示】15-25秒

| 画面 | 台词 |
|------|------|
| 角色找到金冠，点击开棺，金光一闪 | "好家伙，笑归笑，金还是要摸的。" |
| 大金掉落：鬼玺/天山雪莲/双鱼玉佩弹窗 | "我随手开了个棺材——好家伙，直接出了个大鬼玺！" |
| 薇薇安使用重铸技能，紫变金 | "队友说用薇薇安技能还能重铸，我试了一下——双鱼玉佩直接到手！" |
| 全员传送撤离画面 | "四个人背着满满一包金，极限撤离，那种感觉——爽！" |

##### 【第4段：情绪高潮 + 行动号召】25-35秒

| 画面 | 台词 |
|------|------|
| 真人出镜，表情兴奋，竖起大拇指 | "现在我每天上线第一句就是：哥们儿，摸金不？" |
| 游戏组队界面 + "点击下载"按钮特效 | "一个人玩是恐怖游戏，四个人玩是欢乐喜剧。" |
| 游戏logo + 赵露思代言画面 | "超自然行动组，拉上你的冤种队友，一起摸金去！" |
| | 字幕：**点击左下角，组队摸金！** |

##### 备选版：女性向社交场景（抖音/快手）

- 开场：*"好闺闺，整天说想玩又不敢？我当你靠山！"*
- 中间：*"你只需要挂在我身上，怪物交给我，你只管开箱摸金！"*
- 结尾：*"好闺闺带带我，这个古城摸金没你真不行！"*
