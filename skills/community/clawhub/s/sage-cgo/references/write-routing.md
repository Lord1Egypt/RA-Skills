# CGO 写入路由

所有路径都以 `$HOME/.sage/` 为根目录。`sage-mirror/` 只用于工作区浏览。

## 路由表

| 信息类型 | 写入位置 |
| :--- | :--- |
| 平台账号、渠道状态、平台选择 | `growth/channels.md` |
| 内容支柱、栏目、选题模板 | `growth/content-pillars.md` |
| 目标受众、粉丝分层、用户旅程 | `growth/audience.md` |
| 增长实验、假设、结论 | `growth/experiments.md` |
| 数据指标、爆款归因、复盘 | `growth/metrics.md` |
| 变现路径、私域、转化漏斗 | `growth/monetization.md` |
| 平台知识、通用策略 | 不写入 `.sage`，优先使用 `references/platforms.md` |
| 重大增长决策 | `memory_and_insights/recent_decisions.md` |
| 不确定增长信息 | `inbox/unresolved.md` |

## 写入动作

- 数据复盘：追加日期、平台、指标、判断。
- 渠道策略变化：更新当前状态，保留关键历史。
- 实验：必须有假设、动作、指标、周期、结论。
- 爆款归因：区分可复制变量和偶然因素。
