# My Research Lab 我的研究院

> Your personal research institute — 一个真正自动运转、自进化的个人智库。

## 这是什么

一句话：**告诉院长你的研究方向，它帮你每天采集→深度思考→动手产出，全程自动。**

院长不只是信息搬运工，而是你的 **PM（项目经理）**：管理信源、安排大师轮值、跟踪项目进度、从反馈中自我进化。

## 三层运转体系

```
第一层 · 每日发现   信息采集 + 三层质检 + 大师点评 + 与你的关联分析
第二层 · 头脑实验   AI主动发起深度探讨 + 大师圆桌 + 你不回也做
第三层 · 动手实践   院长做PM跟踪、拆AI任务和用户任务、推进到完成
```

## 核心能力

| 能力 | 说明 |
|------|------|
| 冷启动自动配置 | 检测推送渠道 + 沉淀空间 + 大师扫描 + 创建定时任务，一次搞定 |
| 推送+沉淀=生命线 | 没有推送和沉淀不算启动，自动检测已安装的存储平台 |
| 三层质检 | 内容质检 + 摘要质检 + 关联质检（结合用户记忆找关联） |
| 院长精选 | 每期选1条最值得关注的高亮推荐 |
| 排重机制 | 语义排重，30天窗口，同一事件进展更新不算重复 |
| 大师协同 | 自动扫描已安装的女娲蒸馏大师 skill，绑定为顾问团，≥3位时轮值排班 |
| 头脑实验 | AI主动选题，召集大师圆桌讨论，用户不回复也会做 |
| 动手实践 | 院长做PM，拆任务（🤖AI做/👤用户做），跟踪进度到完成 |
| 自进化真正运行 | 信源自动调权、反馈改变行为、大师排班轮值——写进定时任务强制执行 |
| 多研究院并行 | 同一平台可运行多个研究院，独立配置、独立定时、独立沉淀 |
| 多渠道适配 | 邮件=完整版HTML，IM=摘要版Markdown，沉淀=完整版存档 |
| Skill 安全策略 | 推荐安装 Skill 时三色评级，绝不静默安装 |

## 安装

```bash
npx clawhub@latest install my-research-lab
```

## 快速启动

```
"研究 AI 公益"
"帮我建一个美食研究院"
"启动研究院"
```

院长只问一个问题：**你的研究方向是什么？** 然后自动配置一切，立刻执行第一次采集。

## 与女娲蒸馏 Skill 联动

如果你安装了 [nuwa-skill](https://github.com/nuwa-skill) 蒸馏的大师 perspective skill，研究院会自动识别并绑定为顾问团，每条资讯附加大师多视角点评。没有安装也不影响基础运转。
