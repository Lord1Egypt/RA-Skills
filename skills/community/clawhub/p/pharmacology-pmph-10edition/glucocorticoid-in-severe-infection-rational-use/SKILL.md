# glucocorticoid-in-severe-infection-rational-use

## 何时使用本技能
当患者满足以下条件时启用：
- 患有严重急性感染（如中毒性菌痢、暴发性流脑、败血症、重症COVID-19肺炎、结核性脑膜炎等）；
- 同时存在休克或显著全身炎症反应（如ARDS、进行性肺损伤）；
- 已开始足量有效的抗菌或抗结核治疗；
- 无绝对禁忌证（如水痘、带状疱疹、角膜溃疡等）。

## 执行步骤

### 1. 判断适用情形
确认是否属于以下任一情况：
- 中毒性细菌感染伴休克（如中毒性菌痢、暴发型流脑、败血症）；
- 重症病毒感染导致进行性肺损伤或ARDS（如SARS-CoV-2）；
- 以渗出为主的结核病（如结核性脑膜炎、心包炎、胸膜炎、腹膜炎）。

### 2. 验证用药前提
- 必须已启动足量、有效的抗菌（或抗结核）治疗；
- 糖皮质激素仅作辅助，不可替代病原治疗。

### 3. 确定剂量与疗程
- **中毒性休克**：大剂量突击使用（如氢化可的松200–400 mg静脉滴注），微循环改善、脱离休克后立即停药；
- **结核性渗出性疾病**：小剂量（常规剂量的1/2–2/3），短程使用；
- **重症COVID-19**：恰当使用可减轻肺渗出和纤维化，但需警惕股骨头坏死等远期并发症。

### 4. 排查禁忌与慎用情况
- **绝对禁忌**：水痘、带状疱疹（因免疫抑制可致播散）；
- **眼科制剂禁用**：角膜溃疡患者（延缓愈合）；
- **病毒性感染无特效药者原则上不用**，除非病情危重且炎症反应为主导病理过程。

### 5. 把握用药与停药时机
- 糖皮质激素应在抗菌药物之后使用；
- 停药应在撤抗菌药之前，避免感染反弹。

> 提示：本技能不适用于普通感染、非炎症主导的病毒感染或未使用有效抗菌药物的情况。
