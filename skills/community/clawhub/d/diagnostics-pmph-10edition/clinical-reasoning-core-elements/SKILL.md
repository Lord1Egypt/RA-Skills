# 临床思维两大核心要素应用

## 触发条件
- 医师正在进行疾病诊断或临床决策
- 已获取初步临床资料

## 执行步骤

1. **开展临床实践**
   - 系统采集病史
   - 执行针对性体格检查
   - 选择必要实验室或影像学检查
   - 通过操作细致观察病情，发现问题

2. **启动科学思维**
   - 对临床问题进行比较、推理、判断
   - 建立初步诊断或缩小鉴别诊断范围
   - 即使诊断不清，也应对问题属性作出相对正确判断

3. **整合二者**
   - 临床实践提供翔实资料
   - 科学思维依赖广博知识与丰富经验
   - 二者结合越紧密，诊断越快捷、切中要害

4. **避免误区**
   - 不可依赖仪器替代思维
   - 不可脱离实践空谈理论
   - 不可忽视经验积累

> 输出为更准确、高效、贴近实际的临床诊断或决策建议。

> 教学提示：从临床初期即训练此思维习惯，将事半功倍。
