# 淋巴结滤过淋巴与免疫应答机制

## 触发条件
当输入淋巴管携带有抗原物质（如细菌、病毒）进入淋巴结时，立即启用本技能。

## 执行步骤

### 1. 滤过淋巴功能
- 引导淋巴从输入淋巴管进入被膜下窦和小梁周窦。
- 确保淋巴在窦内缓慢流动，以利于巨噬细胞高效清除抗原。
- 验证输出淋巴中细菌等抗原是否被显著清除（正常清除率可达99.5%）。

### 2. 启动免疫应答
- 确认巨噬细胞和交错突细胞已捕获并处理抗原，并将其呈递给初始或记忆T细胞。
- 监测副皮质区（胸腺依赖区）是否出现T细胞活化与增殖，标志细胞免疫启动。
- 检查浅层皮质中B细胞是否在Th细胞辅助下增殖分化：
  - 观察淋巴小结是否增多增大（体液免疫激活标志）。
  - 确认髓索中浆细胞数量是否增加。
  - 测量输出淋巴管中抗体含量是否显著上升。
- 注意：细胞免疫与体液免疫通常同步发生。

### 3. 支持淋巴细胞再循环
- 跟踪约10%的血液淋巴细胞通过高内皮微静脉进入副皮质区。
- 允许B细胞进一步迁移至淋巴小结参与反应。

### 4. 输出免疫效应
- 确保输出淋巴富含活化的免疫细胞（如效应T细胞）和抗体，形成系统性免疫防御。

## 关键变量监控
- **细菌清除率**（bacterial_clearance_rate）：应接近99.5%。
- **淋巴细胞再循环率**（lymphocyte_recirculation_rate）：约10%的血液淋巴细胞参与。

## 使用限制
仅在存在抗原刺激时使用；静息状态淋巴结功能低下，不应触发本流程。
