# 半导体赛道汇总

## 一、赛道概述

半导体是国产替代最确定的赛道之一，涵盖芯片设计、制造、封装测试、设备、材料、EDA/IP等多个细分领域。2025年半导体行业进入AI算力驱动+周期复苏+国产替代三重共振期。

### 1.1 赛道规模
- **全球市场**：2025年全球半导体销售额突破7900亿美元，同比增长25.6%
- **中国市场**：国产替代空间巨大，整体国产化率约15%，高端领域普遍低于20%

### 1.2 核心趋势
1. **AI算力爆发**：驱动先进封装、高端芯片需求
2. **国产替代加速**：设备/材料/EDA等环节突破
3. **先进制程突破**：成熟制程扩产，先进制程追赶
4. **碳化硅/氮化镓**：第三代半导体爆发

## 七、OPC视角并购匹配建议

### 7.1 买方推荐

**战略投资型（具备并购能力）**：
1. **北方华创**（002371）- 设备平台龙头，820亿+订单支撑并购
2. **长电科技**（600584）- 先进封装龙头，持续外延扩张
3. **华大九天**（301269）- EDA龙头，已启动行业整合
4. **海光信息**（688041）- CPU+DCU龙头，扭亏后扩张意愿强
5. **韦尔股份**（603501）- CIS龙头，国际整合经验丰富

**未动潜力型（整合空间大）**：
1. **龙芯中科**（688047）- 管理层明确表示要并购EDA、封测
2. **晶方科技**（603005）- 高毛利隐形冠军，历史上并购少
3. **安路科技**（688107）- FPGA龙头，定增后整合意愿强
4. **芯原股份**（688521）- AI芯片定制，盈亏平衡后扩张

### 7.2 标的推荐

**高匹配度标的（并购价值高）**：
1. **中小型设备公司** - 估值合理，技术互补
2. **材料公司** - 国产化率低，整合空间大
3. **设计公司** - 产品补全型并购
4. **封测公司** - 产能整合型并购

### 7.3 风险提示

1. **估值泡沫**：部分设备/材料公司估值偏高
2. **技术壁垒**：高端环节国产化需要时间
3. **竞争加剧**：更多资本进入，可能价格战
4. **治理风险**：如三安光电实控人调查
