## Use Case: <br>
Developers and engineers use this skill to create, configure, test, troubleshoot, and deploy Botpress ADK bots with TypeScript project conventions, CLI commands, integrations, workflows, tools, tables, triggers, and knowledge bases. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill documents remote ADK installer commands that execute with the user's local privileges. <br>
Mitigation: Review the Botpress ADK installer source or use official manual release downloads, and run installer commands only from trusted Botpress or GitHub URLs. <br>
Risk: Botpress credentials, environment variables, and generated agent identifiers may be exposed if project files are committed carelessly. <br>
Mitigation: Keep `.env`, `agent.json`, Botpress credentials, `node_modules/`, and `.botpress/` out of source control as the skill recommends. <br>
