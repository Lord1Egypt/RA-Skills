# Product Page Audit Quality Checklist

## Image Quality (12 items)
- [ ] Hero image meets platform-specific background requirements
- [ ] Hero image product fill is 85%+ of frame
- [ ] All images are minimum required resolution for platform
- [ ] At least 7 images present (or platform maximum)
- [ ] Lifestyle image shows product in use by target demographic
- [ ] Infographic text is readable at 375px mobile width
- [ ] Scale/size reference image included
- [ ] Image sequence follows logical narrative flow
- [ ] Color consistency across all product images
- [ ] No watermarks, promotional text, or banned elements
- [ ] Images compressed for performance (<500KB each for web)
- [ ] Alt text present and descriptive on every image

## Copy Quality (10 items)
- [ ] Title contains primary keyword in first 5 words
- [ ] Title includes benefit or differentiator language
- [ ] Title within platform character limit
- [ ] Bullets lead with benefits, not features
- [ ] Bullets include specific numbers, metrics, or timeframes
- [ ] Bullets address top 3-5 buyer objections
- [ ] Description has scannable structure (not a wall of text)
- [ ] Readability score at Flesch-Kincaid grade 6-8 for consumer products
- [ ] Brand story or origin element present in description
- [ ] No spelling, grammar, or formatting errors

## SEO Quality (8 items)
- [ ] Title tag present and ≤60 characters with primary keyword
- [ ] Meta description present and ≤155 characters with CTA
- [ ] All images have descriptive, keyword-rich alt text
- [ ] Product schema markup implemented (Product, Offer, AggregateRating)
- [ ] URL slug is clean and contains primary keyword
- [ ] Canonical tag present and correct
- [ ] No duplicate title tags across site
- [ ] Internal linking to related products present

## Mobile Experience (8 items)
- [ ] All tap targets are ≥44x44px with ≥8px spacing
- [ ] Body text is ≥16px, secondary text ≥14px
- [ ] No horizontal scroll required at 375px viewport
- [ ] Product title, price, image, and CTA visible above fold
- [ ] Page loads in ≤3 seconds on 3G connection
- [ ] Images resize properly (no cropping of critical content)
- [ ] Bullet points display key benefits before truncation point
- [ ] Sticky add-to-cart or persistent CTA on scroll

## Trust Signals (9 items)
- [ ] Star rating and review count visible near price
- [ ] Review snippet quotes displayed on page
- [ ] Trust/security badges visible (payment, shipping, authenticity)
- [ ] Money-back guarantee or satisfaction guarantee stated
- [ ] Return policy visible within one scroll of CTA
- [ ] Social proof indicators present (purchase count, trending)
- [ ] Negative reviews addressed with brand responses
- [ ] Certifications and compliance badges displayed where applicable
- [ ] Brand credibility section (story, origin, team) present

## Price Presentation (5 items)
- [ ] Price clearly visible and prominent
- [ ] Anchor pricing shown (strikethrough original price) where applicable
- [ ] Per-unit or per-use cost breakdown displayed
- [ ] Savings amount called out in dollars and/or percentage
- [ ] Payment plan options (Afterpay/Klarna) mentioned if available

## Audit Process Quality (5 items)
- [ ] Platform correctly identified and platform-specific standards applied
- [ ] Mobile-first evaluation completed before desktop review
- [ ] Benchmark data provided for context (category averages, competitor comparison)
- [ ] Priority matrix used with maximum 3-5 P1 recommendations
- [ ] Projected score improvement calculated for P1+P2 items
