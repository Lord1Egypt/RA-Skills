## Description: <br>
Code Oracle turns local code metrics and git history into playful tarot-style readings, crystal-ball forecasts, horoscopes, and fortune-cookie reports. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[jcools1977](https://clawhub.ai/user/jcools1977) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and teams use this skill to review repository health signals, git-history patterns, code quality trends, test coverage, dependency risk, and merge risk through entertaining reports that encourage action. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may inspect repository files and git history that contain secrets or sensitive business context. <br>
Mitigation: Use it only on repositories the agent is authorized to read, and avoid invoking it on codebases containing unreviewed secrets. <br>
Risk: Playful forecasts and metaphors could be mistaken for definitive engineering conclusions. <br>
Mitigation: Treat outputs as prompts for review and confirm any proposed action against the underlying metrics and project context. <br>
