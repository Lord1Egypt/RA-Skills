# 线索道具卡生成模板

> 线索系统是剧本杀的"骨架"。玩家通过线索推导真相，线索的设计直接决定游戏是否好玩。

---

## 线索设计原则

### 三轮发放结构

| 轮次 | 时机 | 性质 | 数量建议 |
|------|------|------|---------|
| 第一轮 | 开场搜证 | 公开，所有玩家可获取，建立场景基础认知 | 4-6张 |
| 第二轮 | 中场讨论后 | 半公开，玩家主动搜证获取，推进推理深度 | 4-6张 |
| 第三轮 | 投票前 | 终极线索，锁定真相，可有剧情反转 | 2-4张 |

### 线索类型分类

| 类型 | 说明 | 举例 |
|------|------|------|
| 直接证据 | 直接指向凶手身份或作案事实 | 带有凶手指纹的道具、目击证人证词 |
| 间接证据 | 需要与其他线索结合才有意义 | 时间表、某角色的日记片段 |
| 人物线索 | 揭露某角色的秘密，制造或澄清嫌疑 | 一封情书、一张欠条 |
| 场景线索 | 描述现场状况，帮助重建事件过程 | 脚印方向、打翻的椅子 |
| 时间线索 | 确定某事发生的时间 | 相机时间戳、服务员的记忆 |
| 误导线索 | 看起来指向某人，实际是烟雾弹 | 嫌疑人丢失的手套（实际与案件无关） |
| 情感线索 | 揭露角色关系和情感背景，情感本重用 | 日记、照片、录音 |

## 单张线索卡模板

```
━━━━━━━━━━━━━━━━━━━━━━
线索卡 #[编号]
━━━━━━━━━━━━━━━━━━━━━━

【线索名称】[简洁且有意象的名字，如"染血的手帕"/"破碎的钟表"]

【发放轮次】第X轮 / 触发条件：[...]

【获取方式】
  搜证地点：[玩家去哪个地点才能找到]
  搜证条件：[是否需要满足特定条件，如"只有[角色X]的玩家可以找到"或"任何玩家都可以找到"]

【线索内容（玩家看到的原文）】
  [用游戏世界观语言描写这件道具/证据，
   写清楚玩家能直接观察到的特征，
   不要透露解读，让玩家自己分析]
  
  示例格式：
  "一块怀表，表盖上刻有字母'R.H.'。表盘停在了11时47分，
   表链断裂，链条末端沾有少量泥土。"

【设计备注（仅主持人和作者可见）】
  真实含义：[这条线索在真相中代表什么]
  指向：[与哪个角色/哪个时间线事件相关]
  关联线索：[需要结合哪条线索才能发挥最大作用]
  是否为误导：[是/否，如果是，说明误导逻辑]
  关键度：[核心/重要/辅助/烟雾弹]
━━━━━━━━━━━━━━━━━━━━━━
```

## 三轮发放设计指引

### 第一轮线索设计要求

目标：让玩家对案件有基本认知，让每个角色都有初步嫌疑。

- 至少1条线索与死亡方式/时间有关
- 至少1条线索让每个角色都有被怀疑的理由（分散嫌疑）
- 不能有直接指向凶手的证据
- 氛围营造型线索（如场景描写、死者遗留的物品）

第一轮线索卡示例格式：

```
【公开发放线索清单（第一轮）】

主持人在开场搜证阶段，按照以下方式发放：
- [地点A]可以找到：线索卡 #1（任何人可找到）、线索卡 #2（任何人可找到）
- [地点B]可以找到：线索卡 #3（任何人可找到）
- [地点C]可以找到：线索卡 #4（仅[角色X]可以理解其含义，但内容向所有人公开）
- 主持人宣读：线索卡 #5（主动告知所有人的公开信息）
```

---

### 第二轮线索设计要求

目标：深化矛盾，让一些角色的秘密被部分揭露，推进关键推理。

- 应有1-2条让玩家开始锁定嫌疑范围的证据
- 至少1条让非凶手角色的某个秘密被揭露（制造混乱感）
- 可以开始出现指向性较强的间接证据

第二轮触发方式建议：
- 玩家去特定地点搜证
- 玩家完成某个行动（如"向主持人交出一张线索卡换取新线索"）
- 时间节点自动触发（如"第二小时开始，发放第二轮线索"）

---

### 第三轮线索设计要求（终极线索）

目标：在投票前提供决定性证据，同时触发情感高潮。

- 必须有1条"锁定凶手"的关键证据（如凶器上的指纹、作案当时的目击者证词）
- 可以有1条揭示"真正动机"的情感线索（让玩家理解凶手为什么这样做）
- 可以有1条"反转线索"（推翻某个之前认为是真的事实）
- 终极线索通常由主持人在规定时间点强制发放

## 常见线索道具类型参考

**物理道具类：**
- 凶器（刀/绳/毒药/重物）
- 遗留物（手套/纽扣/头发/烟蒂）
- 个人物品（钱包/手机/包包）

**文字类：**
- 日记/日程本
- 书信/情书/遗书（真假不定）
- 欠条/合同/遗嘱
- 账本/收据

**影像/记录类（现代场景）：**
- 监控截图（时间戳）
- 聊天记录截图
- 相片（日期和地点线索）

**场景线索类：**
- 脚印/血迹/痕迹
- 打开/锁住的门
- 移动过的家具
- 打碎的杯子

**古风/奇幻道具类：**
- 玉佩/令牌/印章
- 符咒/卷轴
- 血书/密函
- 毒药瓶/暗器
