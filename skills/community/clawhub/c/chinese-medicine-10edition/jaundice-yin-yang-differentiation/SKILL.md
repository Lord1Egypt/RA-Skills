# Jaundice Yin-Yang Differentiation

## When to Use
当患者出现目黄、身黄、小便黄三联征时，立即启动本辨证流程。

> **注意**：萎黄（目不黄）和黄胖（虫积所致）不适用此分类。

## Execution Steps

### 1. 阳黄辨识
- **特征**：黄色鲜明如橘子色，起病急，病程短，多属实证、热证。
- **分型与治法**：
  - **湿热兼表**：伴恶寒发热、头身重痛 → 使用麻黄连翘赤小豆汤合甘露消毒丹。
  - **热重于湿**：伴烦渴、便秘、苔黄糙 → 使用茵陈蒿汤。
  - **湿重于热**：伴头身困重、便溏、苔厚腻 → 使用茵陈五苓散。
  - **胆腑郁热**：伴右胁剧痛放射肩背、寒热往来 → 使用大柴胡汤。

### 2. 阴黄辨识
- **特征**：黄色晦暗如烟熏，起病缓，病程长，多属虚证、寒证。
- **分型与治法**：
  - **寒湿内困**：伴畏寒、便溏、苔白腻 → 使用茵陈术附汤。
  - **脾虚湿郁**：伴乏力、心悸、脉濡细 → 使用六君子汤加茵陈、柴胡。
  - **瘀血内阻**：伴胁下癥积、舌紫有瘀斑 → 使用膈下逐瘀汤。

### 3. 急黄识别（危重证）
- **特征**：起病急骤，黄疸迅速加深，色如金；伴高热、神昏谵语、衄血便血、肌肤瘀斑；舌红绛，苔黄燥，脉洪大或弦滑数。
- **处理**：视为危重症，立即采用清热解毒、凉血开窍法，主方为犀角地黄汤（以水牛角代犀角）加减。

### 4. 转化关系
- 阳黄久病伤阳可转为阴黄。
- 阴黄复感湿热可转为阳黄。

## Output
若符合急黄证候，必须发出危重警示，并建议立即采取清热解毒凉血开窍治疗。
