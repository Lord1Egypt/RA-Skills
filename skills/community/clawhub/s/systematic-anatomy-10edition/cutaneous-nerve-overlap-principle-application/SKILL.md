# 皮神经分布重叠性原理应用

## 何时使用
- 患者报告局部皮肤麻木，但查体发现感觉未完全消失
- 单一皮神经切断术后仍保留部分触觉
- 怀疑神经损伤但感觉障碍范围不符合典型皮节分布

## 执行步骤
1. **确认损伤范围**：
   - 若仅**一条皮神经**受损 → 预期表现为**感觉迟钝**（hypesthesia）
   - 若出现**感觉完全丧失**（anesthesia） → 提示**两条或以上相邻皮神经同时受损**

2. **应用重叠原理**：
   - 躯干和四肢皮肤均由相邻皮神经**网状重叠支配**
   - 单一神经损伤时，邻近神经可代偿部分功能

3. **临床推理**：
   - 若单神经损伤却出现完全感觉丧失，应怀疑：
     - 多节段神经根或丛损伤
     - 中枢感觉通路病变（如脊髓或丘脑）

4. **输出警示**：
   - “若仅单神经损伤但出现完全感觉丧失，应怀疑多神经受累或中枢病变”

> 注意：此原理不适用于肌支、内脏神经或极少数绝对支配区；四肢因神经丛重组，重叠模式更复杂但仍适用。
