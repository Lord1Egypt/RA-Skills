## Description: <br>
Batch-generate images via OpenAI Images API. Random prompt sampler + `index.html` gallery. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[saschaSpoonbill](https://clawhub.ai/user/saschaSpoonbill) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agents use this skill to batch-generate images from structured or supplied prompts through the OpenAI Images API and review the resulting PNGs in a local HTML gallery. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Untrusted OPENAI_BASE_URL or OPENAI_API_BASE values could send API requests and credentials to an unintended endpoint. <br>
Mitigation: Run with those variables unset or set only to a trusted endpoint before invoking scripts/gen.py. <br>
Risk: The generated HTML gallery can include prompt text supplied by the user or caller. <br>
Mitigation: Avoid opening index.html generated from untrusted prompt text until prompt escaping is fixed; inspect prompts.json and image files directly when needed. <br>
Risk: The skill requires an OpenAI API key for image generation. <br>
Mitigation: Use a protected API key and avoid passing credentials through shared shell history or untrusted environments. <br>
