> 来源：Albert Humphrey, 斯坦福大学  
> 用途：系统分析创业项目的优势、劣势、机会与威胁

## 二、每个维度详解

### 2.1 Strengths（优势）

**核心问题**：你比其他竞争者做得好的地方是什么？

| 优势类型 | 问题引导 | 示例 |
|---------|---------|------|
| **资源优势** | 你拥有什么独特资源？ | 专利技术、独家数据、资金充足 |
| **能力优势** | 你擅长什么？ | 营销、技术、运营、供应链 |
| **关系优势** | 你认识谁？ | 行业专家、渠道伙伴、投资者 |
| **品牌优势** | 用户为什么信任你？ | 口碑、背书、历史积累 |
| **规模优势** | 你能做到多大规模？ | 用户基数、成本优势 |

**优势评估标准**：
- ✅ 是真正的优势（竞品难以复制）
- ✅ 可以持续（不是暂时的）
- ✅ 对用户有价值（能转化为用户利益）

---

### 2.2 Weaknesses（劣势）

**核心问题**：你需要改进的地方是什么？

| 劣势类型 | 问题引导 | 示例 |
|---------|---------|------|
| **资源缺失** | 缺少什么资源？ | 资金不足、人才匮乏 |
| **能力短板** | 什么做不好？ | 技术落后、管理混乱 |
| **规模限制** | 什么做不到？ | 产能不足、覆盖不够 |
| **成本劣势** | 什么让你成本高？ | 没有规模效应、效率低下 |
| **品牌劣势** | 用户为什么不信你？ | 新品牌、无背书 |

**劣势评估标准**：
- ⚠️ 是可控的（可以通过努力改变）
- ⚠️ 影响到用户价值
- ⚠️ 被竞争对手利用

---

### 2.3 Opportunities（机会）

**核心问题**：外部环境中对你有利的趋势是什么？

| 机会类型 | 问题引导 | 示例 |
|---------|---------|------|
| **市场机会** | 市场需求在增长？ | 新兴市场、用户痛点未被满足 |
| **技术机会** | 技术进步带来可能？ | AI、5G、云服务成本下降 |
| **政策机会** | 政策变化带来利好？ | 扶持政策、监管放松 |
| **社会机会** | 社会趋势对你有利？ | 消费升级、环保意识 |
| **竞争机会** | 竞品犯错让你有机可乘？ | 竞品弱点、用户迁移 |

**机会评估标准**：
- 🌟 是真实的（不是假设的）
- 🌟 你可以抓住（有能力承接）
- 🌟 有时间窗口（不会很快消失）

---

### 2.4 Threats（威胁）

**核心问题**：外部环境中对你不利的趋势是什么？

| 威胁类型 | 问题引导 | 示例 |
|---------|---------|------|
| **市场威胁** | 市场在萎缩？ | 需求下降、替代品出现 |
| **技术威胁** | 技术变革威胁你？ | 新技术颠覆、成本下降 |
| **政策威胁** | 政策变化不利？ | 监管收紧、准入限制 |
| **竞争威胁** | 竞品在抢夺市场？ | 大公司入场、价格战 |
| **经济威胁** | 经济环境恶化？ | 消费降级、融资困难 |

**威胁评估标准**：
- 🔴 是真实的（概率较高）
- 🔴 影响很大（会重创业务）
- 🔴 你难以控制（只能应对）

## 四、SWOT组合策略

### 4.1 四种策略组合

```
┌────────────────────────────────────────────────────────────────┐
│                      SWOT 策略矩阵                               │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│                         内部因素                                │
│                    优势(S)         │        劣势(W)            │
│              ┌────────────────────┼────────────────────┐       │
│         机会 │                    │                      │       │
│      (O)     │   【SO策略】       │     【WO策略】       │       │
│              │   增长策略         │     改善策略         │       │
│              │                    │                      │       │
│              │ 利用优势抓住机会   │ 弥补劣势抓住机会     │       │
│              ├────────────────────┼────────────────────┤       │
│        威胁 │                    │                      │       │
│      (T)     │   【ST策略】       │     【WT策略】       │       │
│              │   防御策略         │     防御策略         │       │
│              │                    │                      │       │
│              │ 利用优势规避威胁   │ 弥补劣势规避威胁     │       │
│              └────────────────────┴────────────────────┘       │
│                         外部因素                                │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### 4.2 策略详解

| 策略 | 行动 | 适用条件 |
|-----|------|---------|
| **SO（增长策略）** | 利用优势抓住机会 | 优势明显 + 机会窗口 |
| **WO（改善策略）** | 弥补劣势抓住机会 | 劣势可控 + 机会存在 |
| **ST（防御策略）** | 利用优势规避威胁 | 优势明显 + 威胁可控 |
| **WT（收缩策略）** | 弥补劣势规避威胁 | 劣势明显 + 威胁严重 |

### 4.3 策略示例

```
案例：在线教育平台

【SO策略 - 增长策略】
├─ 利用团队的技术优势，抓住AI教育的机会
├─ 利用已有的用户基础，拓展新品类
└─ 利用融资优势，加大市场投入抢占份额

【WO策略 - 改善策略】
├─ 引入行业专家弥补内容短板
├─ 建立用户增长团队解决获客问题
└─ 融资解决资金压力

【ST策略 - 防御策略】
├─ 利用技术壁垒抵御新进入者
├─ 利用规模优势打价格战
└─ 利用品牌影响力应对舆论风险

【WT策略 - 收缩策略】
├─ 裁员降低运营成本
├─ 聚焦核心产品线
└─ 寻找被大公司收购的机会
```

## 五、SWOT到行动的转化

### 5.1 行动优先级矩阵

```
行动优先级 = 价值 × 可行性

         高价值
            ▲
            │
     ┌──────┴──────┐
     │   1.立即做   │  ←  高价值 + 高可行性
     │  (SO策略)    │
     ├─────────────┤
     │   2.计划做   │  ←  高价值 + 低可行性
高可 │  (WO策略)    │
行性 ├─────────────┤
     │   3.监控做   │  ←  低价值 + 高可行性
     │  (ST策略)    │
     ├─────────────┤
     │   4.暂不做   │  ←  低价值 + 低可行性
     │  (WT策略)    │
     └─────────────┘
            │
            ▼
         低价值
```

## 六、SWOT分析检查清单

```
✅ 分析完整性
   □ 每个维度至少列出3-5个点
   □ 区分了"想要的"和"真实的"
   □ 区分了"内部因素"和"外部因素"

✅ 分析质量
   □ 优势是真正的优势（竞品难以复制）
   □ 劣势是会影响业务的点
   □ 机会是有时间窗口的
   □ 威胁是概率较高的

✅ 策略转化
   □ 为每个SO组合制定了行动
   □ 为每个WO组合制定了改善计划
   □ 为ST和WT组合制定了应对方案
   □ 行动有明确的优先级
```
