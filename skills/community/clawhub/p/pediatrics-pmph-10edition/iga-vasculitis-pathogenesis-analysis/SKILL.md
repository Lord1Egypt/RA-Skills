# IgA血管炎（过敏性紫癜）发病机制分析

## 何时使用
当患儿（尤其2～8岁）出现以下表现时启动本分析：
- 血小板计数正常但有紫癜
- 同时存在关节肿痛、腹痛、消化道出血或血尿/蛋白尿
- 需与血小板减少性紫癜或其他系统性血管炎鉴别

## 执行步骤

### 1. 评估遗传易感背景
- 确认是否携带HLA-DRB1*07、HLA-DW35等易感基因
- 检查补体C2水平，判断是否存在C2缺乏
- 询问家族史，注意同胞先后发病情况

### 2. 识别潜在诱发因素
- 回顾近期感染史（尤其A组溶血性链球菌呼吸道感染）
- 评估食物（蛋、乳、豆类）、药物（阿司匹林、抗生素）、疫苗或寄生虫暴露
- 若可行，检测肾小球NAP1r抗原沉积（提示链球菌相关诱因）

### 3. 分析免疫异常激活
- 检测血清IgA（尤其IgA1）水平是否升高
- 评估T细胞和单核细胞CD40L表达状态
- 判断是否存在IgA类免疫复合物形成

### 4. 关联组织损伤与临床表现
- 确认IgA免疫复合物、补体C3及纤维蛋白在皮肤、肠道或肾脏毛细血管沉积
- 结合病理特征（白细胞碎裂性小血管炎、中性粒细胞浸润、红细胞外渗等）解释多系统症状

### 5. 鉴别诊断要点
- 与IgA肾病区分：后者仅有肾小球IgA沉积，无皮疹及系统性血管炎表现
- 排除血小板减少性紫癜及其他类型血管炎

## 输出建议
识别高风险患儿（如C2缺乏或HLA易感基因阳性者），并在链球菌感染后密切监测系统性表现，尤其是肾脏受累迹象。
