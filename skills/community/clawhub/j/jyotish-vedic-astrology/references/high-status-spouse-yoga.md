## 一、核心判定规则（3条）

判断"嫁给高社会地位配偶且婚后命运全面上升"需要**同时满足**以下三条规则：

### 规则1：7宫+7宫主必须强于Lagna+Lagna主

**适用范围**：D1（本命盘）和D9（Navamsa）中都必须满足

**原理**：
- Lagna + Lagna主 = 命主本人的力量
- 第7宫 + 第7宫主 = 配偶的力量
- 当配偶力量 > 命主力量 → 配偶来自更高社会阶层
- 婚后命主会因为配偶而获得巨大提升

**判断方法**：
1. 比较7宫内行星 vs Lagna内行星的力量
2. 比较7宫主 vs Lagna主的力量（尊贵度、相位、合相）
3. 两项都满足"配偶方更强" → 规则1成立

### 规则2：D9（Navamsa）中7宫主存在Rajyoga

**原理**：
- D9是婚姻结果盘，7宫主在D9中形成Rajyoga = 配偶有地位且富有
- Rajyoga的判断标准：
  - 三方宫主（Trikona lord）+ 四角宫主（Kendra lord）的合相或互换
  - 7宫主与Yogakaraka的关系
  - 7宫主入庙/入旺且受吉星相位

### 规则3：从7宫起算的Upachaya宫被行星占据

**原理**：
- Upachaya = 成长宫（第3、6、10、11宫）
- 从第7宫起算的成长宫被行星占据 → 婚后会有成长
- 被占据的数量越多、行星越强 → 成长越显著

**Upachaya中凶星更有效的原理解释**：
- 增长是一种"超越他人"的活动 → Rajas和Tamas行星更适合
- Rajas行星（水星、金星）→ 不伤害他人的方式超越
- Tamas行星（Rahu、Ketu、土星、火星）→ 通过制服/消灭对手实现
- 因此凶星在Upachaya中反而比吉星更有效

**判断方法**：
1. 以D1和D9的第7宫为起点
2. 找出从7宫起算的第3、6、10、11宫
3. 检查这些宫位是否被行星占据
4. 检查占据行星的力量（入庙/入旺/友宫 > 落陷/敌宫）

## 二、实操步骤

### Step 1：D1盘分析

1. **Lagna力量评估**：
   - Lagna内行星（吉星=强，凶星=弱）
   - Lagna主的位置（尊贵度、相位）

2. **7宫力量评估**：
   - 7宫内行星
   - 7宫主的位置和力量
   - 7宫主参与的Yoga（Rajyoga、Dhanayoga等）

3. **力量对比**：7宫系统 vs Lagna系统

### Step 2：D9（Navamsa）验证

> **关键原则**：D9才是结果盘。D1看到倾向，D9验证结果。

1. **D9的7宫**：星座本身暗示配偶的出生环境
   - 狮子座 = 皇家/显赫家族
   - 金牛座 = 富裕/稳定家庭
   - 天秤座 = 优雅/外交家庭
   - 等

2. **D9的7宫主**：
   - 宫主星=具体的人（后天潜质和个人能力）
   - 检查7宫主是否形成Rajyoga
   - 7宫主落入位置（与Lagna的关系）

3. **D9的Upachaya验证**：
   - 从D9的7宫起算成长宫
   - 检查是否被占据

### Step 3：综合判定

| 规则 | D1满足 | D9满足 | 判定 |
|:----:|:------:|:------:|:----:|
| 规则1 | ✅ | ✅ | 强确认 |
| 规则1 | ✅ | ❌ | 有倾向但不确定 |
| 规则2 | — | ✅ | 配偶有地位 |
| 规则3 | ✅ | ✅ | 婚后显著成长 |

## 三、关键概念区分

### 宫位 vs 宫主星

> **宫位 = 环境/背景**，**宫主星 = 具体的人/能力**

- 7宫强大 → 对方**出生的环境好**、家庭背景强大
- 7宫强大 ≠ 对方很有能力（败家的也多的是）
- 7宫主强大 → 对方**后天的潜质和个人能力**
- 两者都需要检查

**常见误区**：
- "嫁入好家庭"≠ 夫妻关系顺利
- "对方有地位"≠ 对方能给你幸福
- "家庭条件好"≠ 对方学识/能力也强
- 本方法**仅用于查看婚姻中物质繁荣的情况**

### D1 vs D9 的分工

| 维度 | D1盘 | D9盘 |
|:----:|:----:|:----:|
| 7宫含义 | 所有合作关系的泛化 | 专指婚姻结果 |
| 看到什么 | 倾向（可能） | 结果（确认） |
| 宫位 | 环境+背景 | 婚后实际生活 |
| 分析顺序 | 先看 | 后验证 |

### 婚前 vs 婚后的人生分水岭

- Navamsa盘的Rajyoga → 婚后才开始显现
- 从7宫起算的Upachaya → 婚后的成长轨迹
- 人生可分为两个阶段：婚前（Lagna主导）和婚后（7宫系统开始发力）
- 有些人的命运**完全因婚姻而改变**

## 四、案例验证：Sonya Gandhi

### 基本数据
- 配偶：Rajiv Gandhi（印度前总理，尼赫鲁家族继承人）
- 婚前：普通意大利女性
- 婚后：成为国大党领袖，掌权多年

### D1盘验证

| 要素 | 数据 | 判定 |
|:----:|:----:|:----:|
| Lagna | 巨蟹座 | 受土星+火星双重刑克 → 弱 |
| Lagna主 | 月亮 | 12宫双子座+受土星相位 → 有Rajyoga但需赴异地 |
| 7宫 | 摩羯座 | 土星通过相位加强 → 强 |
| 7宫主 | 土星 | 受Yogakaraka火星相位 → Rajyoga → 非常强 |
| 力量对比 | 7宫系统 >> Lagna系统 | ✅ 规则1成立 |

**细节发现**：
- 7宫主土星落入Lagna → 伴侣对命主有重大影响力
- 但火星（Yogakaraka在6宫）相位土星 → 配偶会受伤害（Rajiv被刺杀）
- Lagna主月亮在12宫+受Yogakaraka土星相位 = Rajyoga但需去异地/异国 → 完美吻合（从意大利到印度）

### D9（Navamsa）验证

| 要素 | 数据 | 判定 |
|:----:|:----:|:----:|
| D9 Lagna | 水瓶座 | 太阳在水瓶座 → 出身环境普通 |
| D9 7宫 | 狮子座 | 皇家星座 → 丈夫出身显赫家族 ✅ |
| D9 7宫内 | 水星 | 健谈+受木星火星相位 → 非常强 |
| D9 7宫主 | 太阳 | 落入Lagna+与5主水星互相位 → Rajyoga ✅ |
| D9 Lagna主 | 土星 | 天蝎座+受金星影响 → 不错 |
| 力量对比 | 7宫系统 > Lagna系统 | ✅ 规则1 D9验证 |

**D9 Upachaya验证**（从7宫狮子座起算）：
- 第3宫（从7宫）= 天秤座 → 空
- 第6宫（从7宫）= 摩羯座 → 空
- 第10宫（从7宫）= 金牛座 → **被主星金星占据** ✅
- 第11宫（从7宫）= 双子座 → **被耀升Rahu占据** ✅

4个Upachaya中2个被强行星占据 → **婚后显著成长** ✅
