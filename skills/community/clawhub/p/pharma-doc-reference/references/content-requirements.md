# 内容要求清单

每种文档类型的组件槽位定义，供UTOS内容轴清单法使用。标注 `[待补充]` 的字段需企业根据自身产品和合规要求定义。

---

## D1 市场情报

### D1-01 标准化市场数据表
- **必选组件**: 产品名称、治疗领域、数据时间范围、市场份额、增长率、竞品排名、区域分布
- **可选组件**: 竞品列表、目标市场范围、数据来源标注
- **组装顺序**: 基础信息→市场份额→增长趋势→区域分布→竞品对比
- **合规约束**: [待补充] 数据来源合规要求
- **格式**: Excel/CSV

### D1-02 政策追踪日报表
- **必选组件**: 政策名称、发文单位、核心变化摘要、影响等级、截止日期
- **可选组件**: 关注省份/城市、政策类型筛选、历史对比
- **组装顺序**: 基本信息→核心变化→影响评估→行动建议
- **合规约束**: [待补充] 内部政策解读审批流程
- **格式**: Markdown表格

### D1-03 医院列名进度表
- **必选组件**: 医院名、等级、列名状态、关键节点、障碍备注
- **可选组件**: 目标医院名单、关注区域、预计完成时间
- **组装顺序**: 基本信息→当前状态→关键节点→障碍分析
- **合规约束**: 无特殊
- **格式**: Excel

### D1-04 竞品监测报告
- **必选组件**: 监测周期、重点竞品、份额变化、新品动态、策略调整、异常信号
- **可选组件**: 历史对比、竞品推广活动
- **组装顺序**: 摘要→份额变化→动态汇总→异常信号→建议
- **合规约束**: [待补充] 竞品信息采集合规边界
- **格式**: Word/Markdown

### D1-05 PESTEL分析报告
- **必选组件**: 政治环境、经济环境、社会环境、技术环境、环境因素、法律环境、机会标注、威胁标注、趋势方向
- **可选组件**: 行业报告引用、治疗领域聚焦
- **组装顺序**: 概述→六维度逐一分析→机会威胁汇总→趋势判断
- **合规约束**: 无特殊
- **格式**: Word/Markdown

### D1-06 竞争态势分析报告
- **必选组件**: 市场份额变动、竞品策略演变、区域热点、风险预警
- **可选组件**: 历史对比月份、竞品推广活动分析
- **组装顺序**: 摘要→份额分析→策略演变→区域热点→风险预警→建议
- **合规约束**: [待补充] 内部分析结论传播限制
- **格式**: Word/Markdown

### D1-07 竞争应对策略矩阵
- **必选组件**: 竞品策略→我方应对→推荐话术→可用证据
- **可选组件**: 目标客户群、重点竞品筛选
- **组装顺序**: 策略矩阵→话术库→证据支撑→实施建议
- **合规约束**: [待补充] 话术合规红线（超适应症、比较用语规范）
- **格式**: Excel矩阵 + Word话术库

### D1-08 政策趋势预判报告
- **必选组件**: 1-3年政策走向、影响评估、应对建议
- **可选组件**: 历史政策演变轨迹、国际对标
- **组装顺序**: 摘要→趋势推演→影响评估→应对建议
- **合规约束**: [待补充] 预判结论发布审批
- **格式**: Word/Markdown

## D2 战略规划

### D2-01 基础数据处理与图表
- **必选组件**: 数据来源、数据清洗说明、可视化图表、关键指标摘要

- **合规约束**: 数据来源必须标注
- **格式**: Excel + 图表

### D2-02 SWOT与机会评估

### D2-03 产品组合分析矩阵

- **可选组件**: BCG矩阵、GE矩阵
- **组装顺序**: 产品概览→矩阵定位→资源建议
- **合规约束**: 无特殊

### D2-04 品牌资产评估

- **可选组件**: 历史趋势、竞品对标
- **组装顺序**: 指标体系→数据→评估→趋势→建议

### D2-05 品牌医学差异化定位

- **组装顺序**: 差异化主张→证据链→竞品对比→定位声明
- **合规约束**: [待补充] 差异化声明合规边界
- **格式**: Word/Markdown

### D2-06 品牌视觉识别规范
- **必选组件**: 品牌主色/辅色、标准字体、版式规范、图标/Logo使用规则
- **可选组件**: 模板套用示例、合作代理商视觉交付标准
- **组装顺序**: 品牌概述→色彩→字体→版式→Logo/图标→应用示例
- **合规约束**: [待补充] 品牌视觉合规审查要求
- **格式**: PDF/品牌手册

### D2-07 ABP九宫格
- **必选组件**: 市场分析、目标设定、策略、资源分配、预算、时间线、风险、KPI、里程碑
- **可选组件**: 历史对比、竞品对标
- **组装顺序**: 市场分析→目标→策略→资源→预算→时间线→风险→KPI
- **合规约束**: [待补充] ABP审批流程
- **格式**: PPT/Word九宫格模板

### D2-08 推广策略规划(STP)
- **必选组件**: 市场细分、目标市场选择、产品定位
- **可选组件**: 区域差异化策略、资源优化配置
- **组装顺序**: 细分→目标→定位→策略→资源
- **合规约束**: [待补充] 推广策略合规边界
- **格式**: Word/Markdown

### D2-09 年度目标季度分解
- **必选组件**: 年度目标、季度KPI、里程碑、责任人
- **可选组件**: 风险预案、资源需求
- **组装顺序**: 年度总目标→Q1→Q2→Q3→Q4→风险预案
- **合规约束**: 无特殊
- **格式**: Excel

### D2-10 三年战略规划(STRAP)
- **必选组件**: 战略愿景、市场分析、三年目标、分年策略、资源规划、风险预案
- **可选组件**: 情景分析、敏感度分析
- **组装顺序**: 愿景→市场分析→目标→策略→资源→风险
- **合规约束**: [待补充] 战略审批流程
- **格式**: Word

### D2-11 预算执行追踪表
- **必选组件**: 预算项、预算额、实际支出、偏差率、预警线
- **可选组件**: 周度/月度追踪粒度
- **组装顺序**: 预算概览→分项追踪→偏差分析→预警
- **合规约束**: [待补充] 预算审批与超支处理流程
- **格式**: Excel

- **组装顺序**: 证据缺口→阶梯设计→研究规划→时间线→资源

### D4-07 RWE生成与统计分析
- **必选组件**: 数据治理结果、统计分析、证据产出
- **可选组件**: 证据价值评估
- **组装顺序**: 数据→分析→证据→评估
- **合规约束**: [待补充] RWE用于监管申报的标准

### D4-08 HTA证据包

- **可选组件**: 支付方证据需求映射
- **组装顺序**: 临床证据→经济证据→PRO→对比→适配需求
- **合规约束**: [待补充] HTA机构特定要求
- **格式**: Word + Excel

### D4-09 上市后研究承诺追踪
- **必选组件**: PMR/PMC承诺清单、交付状态、截止日期、责任部门、延迟风险标记

- **组装顺序**: 承诺清单→状态追踪→延迟预警→沟通记录→下一步计划

- **可选组件**: 目标受众分层、发布渠道策略
- **组装顺序**: 核心信息→受众适配→Q&A→时间线→审批
- **合规约束**: [待补充] 对外沟通合规审批
- **格式**: Word/Markdown
