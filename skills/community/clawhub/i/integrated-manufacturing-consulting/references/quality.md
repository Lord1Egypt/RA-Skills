# 质量管理调研指南

## 调研目标
- 评估质量管理体系完整性与执行有效性
- 识别质量控制断点和可追溯性短板
- 评估质量数据利用水平

## 调研重点

### 1. 质量体系
- ISO/TS体系认证情况
- 质量方针与目标分解
- 质量组织架构与责任划分

### 2. 来料质量(IQC)
- 来料检验流程与抽样标准
- 免检物料管理
- 来料异常处理（MRB流程）
- 供应商质量绩效管理

### 3. 过程质量(IPQC)
- 首件检验制度
- 过程检验频次与方式
- SPC统计过程控制应用
- 自检/互检机制
- 不合格品管理（标识/隔离/处理）

### 4. 出货质量(OQC)
- 成品检验标准
- 出货检验流程
- 客户验货配合

### 5. 质量追溯
- 生产批次追溯能力
- 物料批次追溯（正向/逆向）
- 质量问题可追溯到的粒度

### 6. 质量改进
- 客诉处理流程（8D/CAR）
- 质量数据收集与分析
- 质量改善项目机制
- 质量成本分析

## 常见问题诊断库

| 问题现象 | 可能根因 | 典型影响 |
|---------|---------|---------|
| 质量数据"孤岛" | 各环节数据未打通/无系统集成 | 无法做趋势分析/预防 |
| 追溯困难 | 无批次管理/扫码缺失 | 召回成本高/客诉处理慢 |
| 不良品重复发生 | 根因分析不深入/纠正措施不落地 | 质量成本居高不下 |
| 检验依赖"人治" | SPC缺失/自动化检验不足 | 错检漏检/效率低下 |
| 客诉处理周期长 | 无8D流程/责任界定慢 | 客户关系受损 |
| 来料不良波动大 | 供应商管控弱/免检标准不合理 | 产线停线/返工 |
| 过程质量失控 | IPQC频次不够/控制计划未执行 | 不良品流到下游 |
