# 我要退休 - 退休规划计算器

## 概述

此 Skill 帮助用户规划退休，提供以下核心功能：

1. **计算退休年龄**：根据中国最新延迟退休政策（2025年1月1日起实施），计算用户的确切退休年月
2. **计算社保缴费总额**：根据用户工资、所在城市和剩余工作年限，计算需要缴纳的社保总金额

## 使用方法

### 第一步：收集用户信息

#### 通用信息（所有人都要问）

1. **出生日期**（格式：YYYY-MM-DD）
2. **性别**（男性/女性）
3. **女性需确认身份**：工人或干部（影响退休年龄计算）
4. **所在城市**（用于确定社保缴费基数上下限）
5. **就业类型**：
   - **职工社保**（单位代缴）- 有工作的人
   - **灵活就业**（自己缴纳）- 自由职业者、个体户等

> ⚠️ **关于就业类型**：这个选择对结果影响巨大！灵活就业者需要自己承担全部社保费用（养老20% vs 职工8%），每月缴费通常是职工的 **2-3倍**。

#### 职工社保专属问题

6. **月工资收入**（用于计算缴费基数）
7. **是否缴纳住房公积金**（部分单位不交，需要确认）

#### 灵活就业专属问题

6. **选择缴费档次**（向用户展示当地基数范围，供其选择）

灵活就业者可以**自由选择**缴费基数，在当地最低基数和最高基数之间任选。应展示以下信息供用户参考：
- 当地最低基数（如上海：¥7,384）
- 当地最高基数（如上海：¥36,921）
- 不同档次对应的月缴费金额和退休前总费用
- 建议根据收入稳定性选择合适的档次

### 第二步：执行计算

使用脚本 `scripts/calculate_retirement.py` 进行计算：

```bash
python scripts/calculate_retirement.py <出生日期> <性别> <身份类型> <月工资> <城市> <就业类型> <是否含公积金>
```

参数说明：
- `出生日期`：格式 YYYY-MM-DD，如 1985-06-15
- `性别`：male 或 female
- `身份类型`：worker（工人）或 cadre（干部），女性需要填写
- `月工资`：数字，如 15000
- `城市`：城市名称，如 北京、上海
- `就业类型`：**employee**（职工社保）或 **flexible**（灵活就业）
- `是否含公积金`：**true 或 false** - 职工社保时询问，灵活就业一般为 false

### 示例

**职工社保（有单位）：**
```bash
python scripts/calculate_retirement.py 1985-06-15 male worker 15000 北京 employee true
```

**灵活就业（自由职业）：**
```bash
python scripts/calculate_retirement.py 1985-06-15 male worker 15000 北京 flexible
```

### 第三步：解析并展示结果

脚本返回 JSON 格式的结果，包含以下信息：

#### 退休信息
- 原退休年龄和目标退休年龄
- 延迟月数
- 实际退休年月（哪年哪月）
- 距离退休还有多少个月/年

#### 社保缴费信息
- 缴费基数（根据工资和城市上下限确定）
- 每月各项社保缴费金额
- 从现在到退休的总缴费金额
- 分项明细（养老、医疗、失业、公积金）

## 延迟退休政策说明

### 政策背景
根据2024年发布的《关于实施渐进式延迟法定退休年龄的决定》，从2025年1月1日起实施渐进式延迟退休：

| 人群 | 原退休年龄 | 目标退休年龄 | 延迟规则 |
|------|-----------|-------------|---------|
| 男性 | 60岁 | 63岁 | 每4个月延迟1个月 |
| 女性干部 | 55岁 | 58岁 | 每4个月延迟1个月 |
| 女性工人 | 50岁 | 55岁 | 每2个月延迟1个月 |

详细政策参考 `references/retirement_policy.md`

## 社保缴费说明

### 两种就业类型对比

| 项目 | 职工社保（有单位） | 灵活就业（自己交） |
|------|------------------|-------------------|
| **养老保险** | 个人缴 8% | **个人缴 20%** |
| **医疗保险** | 个人缴 2% | **个人缴 6%-11%**（各地不同） |
| **失业保险** | 个人缴 0.5% | **无** |
| **住房公积金** | 个人缴 5%-12% | **一般无**（极少自缴） |
| **每月总费用** | 较低 | **约为职工的 2-3 倍** |

### 职工社保缴费比例
- 养老保险：8%
- 医疗保险：2%
- 失业保险：0.5%
- 住房公积金：5%-12%（各地不同，计算时默认7%）

### 灵活就业缴费比例
- **养老保险：20%**（其中8%进个人账户，12%进统筹账户）
- **医疗保险：6%-11%**（各地差异较大，如上海11%，武汉6%）
- **失业保险：无**
- **住房公积金：一般无**

### 缴费基数计算
缴费基数 = max(当地最低基数, min(本人月工资, 当地最高基数))

即：如果工资低于最低基数，按最低基数缴纳；如果高于最高基数，按最高基数缴纳；如果在中间，按实际工资缴纳。

### 灵活就业各城市医疗保险比例

| 城市 | 医疗缴费比例 | 城市 | 医疗缴费比例 |
|------|-------------|------|-------------|
| 北京 | 6.5% | 南京 | 10% |
| 上海 | 11% | 重庆 | 10% |
| 广州 | 8% | 天津 | 7.5% |
| 深圳 | 8% | 苏州 | 10% |
| 杭州 | 9.5% | 其他城市 | 8%（默认） |
| 成都 | 8.8% | | |
| 武汉 | 6% | | |
| 西安 | 6% | | |

## 注意事项

1. **就业类型选择**：这是最重要的变量！灵活就业缴费约为职工的 **2-3倍**，务必确认用户实际情况：
   - 有单位代缴 → 选"职工社保"
   - 自由职业/个体户/无业自己交 → 选"灵活就业"

2. **公积金变量**：职工社保务必询问单位是否缴纳公积金，这对最终缴费总额影响巨大（约占40%）。如果不交，用户表面少交钱，实际损失了福利。

3. **灵活就业的利弊**：
   - **弊端**：每月缴费高（养老20%全额自付），没有失业险和公积金
   - **好处**：养老金待遇与职工相同（都是8%进个人账户），医保报销比例相同
   - **建议**：如果收入不稳定，可以选择只交养老保险，或按最低基数缴纳

4. **数据时效性**：社保缴费基数每年7月左右调整，脚本中的基数为2024年数据，实际计算时请提醒用户以当地最新公布为准

5. **地区差异**：各地社保政策可能存在细微差异，灵活就业医疗保险比例各地不同，计算结果仅供参考

6. **政策变化**：延迟退休政策为2025年新实施政策，如有调整需更新脚本

7. **提前退休**：特殊工种、病退等提前退休情况未在本Skill中考虑

## 关于住房公积金的重要说明

### 法律规定
根据《住房公积金管理条例》，单位**必须**为员工缴纳公积金，具有强制性。

### 现实情况
部分单位（尤其是中小企业、创业公司、某些民营企业）存在不交或少交公积金的情况。

### 对计算的影响
- 有公积金：缴费总额包含公积金部分（可提取，相当于强制储蓄）
- 无公积金：缴费总额减少约40%，但损失长期福利

### 建议
如果发现单位不交公积金：
1. 向当地住房公积金管理中心投诉（12329热线）
2. 要求单位补缴（可以追溯）
3. 在计算时明确告知用户两种情况的差异
