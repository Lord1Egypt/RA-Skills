# 示例 07：已有封面诊断

## 输入

用户上传已有封面，反馈：点击率低。

## 诊断输出模板

### 技术问题

- 比例是否匹配平台。
- 图片是否模糊。
- 文字是否太小。
- 背景是否过乱。

### 策略问题

- 最大关键词是否清楚。
- 人物情绪是否匹配标题。
- 是否有一个明确点击钩子。
- 是否把完整标题硬塞进封面。

### 优先修改顺序

1. 缩短封面标题，提炼最大关键词。
2. 放大人物或主体。
3. 降低背景复杂度。
4. 增加标题描边/色块。
5. 重新匹配人物表情和手势。

### 重做方案

输出 A/B 两版：

- A：强点击版。
- B：信任干货版。
