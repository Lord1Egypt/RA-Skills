# 小红书爆款文案生成器

一键生成高互动率的小红书笔记。

## 功能

- 生成 10 个爆款标题选项
- 撰写完整笔记正文（含表情、话题标签）
- 提供封面图建议
- 优化文案钩子提升互动率

## 使用方法

### 生成笔记
```
写一篇关于<主题>的小红书
赛道：<美妆/穿搭/美食/旅行/职场/学习>
风格：<干货/种草/吐槽/励志>
```

### 优化现有文案
```
优化这篇小红书文案：<粘贴内容>
要求：提高互动率，增加钩子
```

## 输出格式

每篇笔记包含：
1. **封面建议** - 标题 + 视觉风格
2. **标题选项** - 10 个爆款标题
3. **正文内容** - 含表情和排版
4. **话题标签** - 15-20 个相关标签
5. **发布建议** - 最佳发布时间

## 支持赛道

- 美妆护肤
- 穿搭时尚
- 美食探店
- 旅行攻略
- 职场干货
- 学习成长
- 家居生活
- 健身运动
