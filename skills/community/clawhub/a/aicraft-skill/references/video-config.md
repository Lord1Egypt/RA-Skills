# 视频生成配置详解

## 生成模式

### 新手模式 (mode: 1)
- 上传图片后，系统自动识别商品信息并生成脚本
- 适合快速出视频
- 额外消耗：脚本生成 2 积分

### 专业模式 (mode: 2)
- 上传图片后，系统生成多镜头分镜脚本（含镜头、台词、动作）
- 适合精细控制视频内容
- 额外消耗：脚本生成 2 积分

## 视频通道 (videoChannel)

**当前可用通道：**

| 通道 | 值 | 时长选项 | 积分消耗 | 特点 |
|------|-----|----------|----------|------|
| 高稳通道 | `stable` | 10秒 / 15秒 | 150 / 200 | 稳定性极高，画面真实自然，适合电商爆款视频 |

> 标准通道(`standard`)、增强通道(`enhanced`)、极速通道(`fast`) 当前已禁用。

**新手模式总积分示例：**
- stable 10秒 + 新手模式 = 150 + 2 = **152 积分**
- stable 15秒 + 新手模式 = 200 + 2 = **202 积分**

## 视频比例 (orientation)

| 比例 | 值 | 适用平台 |
|------|-----|----------|
| 9:16 竖屏 | `9:16` | 抖音、快手等短视频平台（默认） |
| 16:9 横屏 | `16:9` | B站、YouTube等横屏平台 |

## 视频清晰度 (size)

| 清晰度 | 值 | 说明 |
|--------|-----|------|
| 普清 | `small` | 标清画质，文件较小（默认） |
| 高清 | `large` | 高清画质，推荐使用 |

## 视频时长 (duration)

- **stable 通道**: 10秒 或 15秒

## 视频类型 (videoType)

| 类型 | 值 | 说明 |
|------|-----|------|
| 商品推广 | `0` | 突出商品卖点和优势 |
| 带货视频 | `1` | 直播带货风格 |
| 商品讲解 | `2` | 详细介绍商品特点、功能和使用方法（默认） |
| 商品促销 | `3` | 促销活动宣传 |
| 商品测评 | `4` | 客观测评，优缺点分析 |
| 商品口播 | `5` | 口播介绍类 |

## 视频语言 (language)

| 语言 | 值 |
|------|-----|
| 简体中文 | `0`（默认） |
| 繁体中文 | `1` |
| 英语 | `2` |
| 泰语 | `3` |
| 俄语 | `4` |
| 越南语 | `5` |
| 马来语 | `6` |
| 葡萄牙语 | `7` |
| 西班牙语 | `8` |

## 默认配置推荐

**电商商品讲解视频（新手模式）：**
- mode: 1
- videoChannel: `stable`
- duration: 10
- orientation: `9:16`
- size: `large`
- videoType: `2` (商品讲解)
- language: `0` (简体中文)
- 总积分：**152 积分**

**电商商品讲解视频（专业模式）：**
- mode: 2
- videoChannel: `stable`
- duration: 10
- orientation: `9:16`
- size: `large`
- videoType: `2` (商品讲解)
- language: `0` (简体中文)
- 总积分：**150 积分**

## API 请求示例

```json
{
  "conversationId": "",
  "projectType": "VIDEO",
  "mode": 1,
  "type": 1,
  "payload": {
    "prompt": "介绍这款商品的优点和使用场景",
    "duration": 10,
    "orientation": "9:16",
    "size": "large",
    "language": "0",
    "videoType": "2",
    "videoChannel": "stable",
    "images": ["https://oss.fzputi.com/..."]
  }
}
```
