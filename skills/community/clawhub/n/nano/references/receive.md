# xno-skills receive

```
Usage: cli receive [options]

Receive pending blocks

Options:
  --wallet <name>  OWS wallet name
  --hash <hash>    Receive only this send block hash
  --count <n>      Max receivable blocks to consider (default: 10)
  -j, --json       Output in JSON format
  -h, --help       display help for command
```
