## Description: <br>
README authoring workflow for audience definition, value proposition, quickstart, configuration, troubleshooting, contributing, and badges/links hygiene. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and documentation maintainers use this skill to plan or revise README files so repository users can quickly understand purpose, setup, configuration, contribution, and maintenance expectations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: README guidance can still lead to inaccurate setup instructions, exposed secrets, or stale operational details if copied without verification. <br>
Mitigation: Review generated README content against the actual repository, avoid publishing real secrets, and verify setup commands before release. <br>
