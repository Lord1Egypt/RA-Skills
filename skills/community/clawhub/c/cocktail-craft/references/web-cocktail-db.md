## 实时查询优先级

查配方时：
1. 本地库（IBA + 品牌 + 酒吧签名）→ 命中直接出
2. 未命中 → AnySearch 搜索 `"Difford's Guide <酒名>"`（首选）
3. Difford's 404 → `"Liquor.com <酒名>"` 或 `"Imbibe <酒名>"`
4. 查趋势/新酒 → `"PUNCH <酒名> cocktail"`
