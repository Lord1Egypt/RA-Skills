# 酒精性与代谢性脂肪性肝病鉴别诊断

## 适用条件
- 患者已排除病毒性肝炎、自身免疫性肝病等其他明确肝损因素
- 肝活检或影像学证实存在肝脂肪变性
- 患者属于以下任一人群：
  - 慢性饮酒者
  - 肥胖或糖尿病患者
  - 无酗酒史但有代谢异常者

## 执行步骤

### 第一步：确认是否符合酒精性肝病（ALD）诊断路径
- **必须满足**：存在慢性酒精中毒史
- **评估病理表现**（基于肝活检）：
  1. **脂肪肝**：肝脏肿大、质软、呈黄色油腻外观；镜下见肝细胞含脂滴空泡，以中央区为主；戒酒后可逆
  2. **酒精性肝炎**：三联征包括：
     - 肝细胞脂肪变性
     - Mallory小体（HE染色为胞质嗜酸性凝集物）
     - 点灶状坏死伴中性粒细胞浸润
  3. **酒精性肝硬化**：假小叶形成，为终末阶段
- **机制提示**：酒精及乙醛直接毒性、营养不良（蛋白质/维生素缺乏）、可能的自身免疫反应

### 第二步：若无酗酒史，评估是否符合代谢紊乱相关脂肪性肝病（MAFLD）
- **必须满足**：无酗酒史（男性<30g/天，女性<20g/天）
- **定义**：除外酒精及其他明确肝损因素，以弥漫性肝细胞脂肪变性为主的代谢综合征相关肝病（2023年欧洲肝病会议命名更新为MAFLD）
- **评估病理表现**：
  - 单纯性脂肪变、脂肪性肝炎、脂肪性纤维化
  - 可见Mallory小体，但数量通常少于ALD
  - 可进展至肝硬化
- **应用“二次打击”学说解释进展机制**：
  - **第一次打击**：遗传、环境、胰岛素抵抗、肥胖 → 肝细胞游离脂肪酸堆积
  - **第二次打击**：氧化应激、炎症因子释放、线粒体功能障碍 → 肝星状细胞激活 → 纤维化

### 第三步：输出诊断结论与干预建议
- 明确肝病类型：酒精性 vs 代谢性（MAFLD）
- 指导核心干预：
  - ALD：严格戒酒 + 营养支持
  - MAFLD：改善代谢状态（减重、控制血糖/血脂）
- 评估纤维化风险，决定是否需进一步监测或抗纤维化治疗
