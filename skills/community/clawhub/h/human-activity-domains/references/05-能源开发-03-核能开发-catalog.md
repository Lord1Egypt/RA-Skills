## 核能开发任务清单

核能开发按子域组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：核能研究 → 核电站设计 → 核电站建设 → 核电站运营

## 依赖拓扑摘要

### 一、域内链路

**核能研究链路**: T1-1 → T1-2 → T1-3 → T1-4
**核电站设计链路**: T2-1 → T2-2 → T2-3 → T2-4
**核电站建设链路**: T3-1 → T3-2 → T3-3 → T3-4
**核电站运营链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整核能开发链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T3-1 → T3-2 → T3-3 → T3-4 → T4-1 → T4-2 → T4-3 → T4-4

**反馈改进链路**: T4-3 → T2-4（设备维护反馈到设备设计）

**安全驱动链路**: T4-2 → T1-3（安全监管驱动核安全研究）

更多组合根据具体任务动态推导。
