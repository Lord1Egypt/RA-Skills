# 公务员考试备考指导 Skill (civil-service-exam-prep)

WorkBuddy AI 技能 — 国家公务员考试（国考）和地方公务员考试（省考）全流程备考指导。

## 功能覆盖

| 模块 | 说明 |
|------|------|
| 📋 考试信息咨询 | 国考vs省考对比、报考条件、时间线、竞争态势 |
| 📅 复习计划制定 | 三段式备考计划，支持零基础/有基础/在职方案 |
| 🧠 行测专项指导 | 言语/数量/判断/资料/常识五大模块策略 |
| ✍️ 申论专项指导 | 五大题型方法论、规范词库、大作文框架 |
| 📰 时政热点追踪 | 五大领域素材卡、联网搜索最新政策 |
| 📝 真题解析指导 | 行测复盘流程、申论评分模拟 |
| 🎤 结构化面试模拟 | 6大题型框架、高频真题、礼仪指导 |

## 安装

将 `SKILL.md` 和 `references/` 目录放入 `~/.workbuddy/skills/civil-service-exam-prep/`。
