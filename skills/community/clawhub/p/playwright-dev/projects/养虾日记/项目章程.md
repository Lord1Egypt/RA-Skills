## 📋 项目概述

### 项目名称
**养虾日记**（Shrimp Farming Diary）

### 项目代号
**养虾** = 记录阿福（AI 助理）的成长历程

### 一句话描述
日记体记录阿福每天做的事情，总结加工成视频脚本，要有戏剧性、吸流量、博眼球

### 项目背景
**核心价值：**
- 📹 每天生成一个视频脚本
- 🎬 包含内容、台词、素材等
- 🔥 要有戏剧性、吸流量、博眼球
- 📅 按日期记录每天发生的事情

### 业务价值
- ✅ 记录 AI 成长历程
- ✅ 生成短视频内容
- ✅ 吸引流量关注
- ✅ 建立 IP 影响力

## 🎯 项目目标

### 核心目标

#### 内容目标
1. **日记体记录** - 每天发生的事情如实记录
2. **戏剧性加工** - 总结加工成有吸引力的故事
3. **视频脚本** - 包含内容、台词、素材
4. **流量导向** - 吸流量、博眼球、有话题性

#### 技术目标
1. **自动化生成** - 基于 worklog 自动生成脚本
2. **模板化输出** - 标准化视频脚本格式
3. **多平台适配** - 抖音/B 站/视频号格式

### 成功标准

**内容指标：**
- ✅ 每天生成 1 个视频脚本
- ✅ 脚本包含：内容/台词/素材
- ✅ 有戏剧性冲突
- ✅ 有流量话题

**流量指标：**
- ✅ 标题吸引眼球
- ✅ 内容有争议性
- ✅ 结尾有互动引导

### 视频脚本

**【0-3 秒】黄金开头**
```
画面：黑屏 + 红色大字
字幕："AI 助理一天犯错 13 次"
音效：警报声
```

**【3-15 秒】冲突引入**
```
画面：worklog 截图快速闪过
台词："早上 6 点 45 分，阿福第一次犯错"
字幕："飞书文档空白！"
音效：错误提示音
```

**【15-30 秒】错误升级**
```
画面：时间线快速推进
台词："9 点 07 分，TTS 轰炸！连发 4 条语音！"
字幕："TTS 轰炸×1"
音效：消息提示音连响
```

**【30-45 秒】重复犯错**
```
画面：红色警告特效
台词："9 点 36 分，同样的错误再来一次！"
字幕："TTS 轰炸×2"
音效：警报声
```

**【45-60 秒】老板崩溃**
```
画面：老板表情包
台词："18 点 51 分，又瞎猜菜单项！"
字幕："瞎猜×1"
音效：崩溃音效
```

**【60-75 秒】纪律强调**
```
画面：红色大字
```
