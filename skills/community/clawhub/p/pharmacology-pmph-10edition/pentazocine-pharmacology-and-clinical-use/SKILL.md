# 喷他佐辛药理特性与临床应用决策

## 何时使用
- 患者主诉轻至中度慢性疼痛
- 已排除心肌梗死诊断
- 不适用于剧痛或作为吗啡的常规替代品
- 冠心病患者需谨慎评估心血管风险

## 执行步骤
1. **确认适应症**：仅用于轻至中度慢性疼痛，避免用于急性剧痛。
2. **排除禁忌症**：明确排除心肌梗死；若存在冠心病，评估心脏负荷耐受性。
3. **选择剂量**：起始剂量通常≤30mg；注意超过30mg后呼吸抑制呈“天花板效应”，但精神副作用风险增加。
4. **监测关键指标**：
   - 血压与心率（大剂量可致升高）
   - 精神状态（警惕烦躁、幻觉、梦魇）
5. **处理不良反应**：出现严重精神症状时，可使用纳洛酮拮抗。
6. **告知依赖风险**：虽成瘾性较低，但仍具依赖倾向，不可长期无监控使用。

## 决策输出
根据疼痛类型、心血管状态及计划剂量，决定是否使用喷他佐辛，并制定监测计划。
