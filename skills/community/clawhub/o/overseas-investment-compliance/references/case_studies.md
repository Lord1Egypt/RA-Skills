# 典型场景分析（虚构场景，供方法论学习）

> **声明**：以下案例为虚构场景，用于展示对外投资合规的方法论和常见问题，不指向任何真实交易或企业。

---

## 核心前置判断：资产流向判定

> ⚠️ **每个案例必须首先明确**：资产在哪里？要去哪里？

| 流向类型 | 合规框架 | 典型场景 |
|---------|---------|---------|
| 境内→境外 | ODI备案+发改委+商务部+外管 | 境内企业出海 |
| 境外→境内 | 外商投资法+负面清单+安全审查 | 外资入华 |
| 境外→境外 | 目标地法规+交易所规则 | 境外资产港股上市 |
| 境内→港澳 | 参照执行+CEPA特殊安排 | 内地企业赴港 |

---

## 一、成功合规场景

### 场景A：境内→境外（制造企业境外建厂）

| 维度 | 虚构设定 |
|------|----------|
| **资产流向** | 境内资金 → 境外某东南亚国家 |
| **投资方** | 境内民营制造企业 |
| **投资标的** | 东南亚某国工业产业园 |
| **投资金额** | 8000万美元 |
| **投资方式** | 绿地投资（新建）+与当地企业合资 |

#### 合规路径执行

| 步骤 | 操作 | 合规要点 |
|------|------|----------|
| 1 | 完成发改委备案 | 省级发改委，7工作日 |
| 2 | 完成商务部备案 | 关联发改委编码，3工作日 |
| 3 | 完成外汇登记 | 银行代办，1-3工作日 |
| 4 | 开立ODI专户 | 指定银行 |
| 5 | 资金合规汇出 | 6个月内首笔≥30% |
| 6 | 获得当地BOI优惠 | 享受政策支持 |

#### 合规亮点

| 要点 | 说明 |
|------|------|
| **流程合规** | 严格按照"发改委→商务部→外管局"顺序 |
| **本地化承诺** | 带动当地就业，获得政府支持 |
| **政策利用** | 充分利用目的地国产业优惠政策 |

#### 经验总结
> 绿地投资是合规风险较低的"境内→境外"投资方式，关键是在投资前完成完整的ODI备案流程。

---

### 场景B：境外→境内（外资收购境内限制类企业）

| 维度 | 虚构设定 |
|------|----------|
| **资产流向** | 境外资金 → 中国境内 |
| **投资方** | 某境外企业 |
| **投资标的** | 中国境内某省新材料生产企业 |
| **投资金额** | 2亿美元 |
| **投资方式** | 收购境内企业60%股权 |

#### 合规路径执行

| 步骤 | 操作 | 合规要点 |
|------|------|----------|
| 1 | 外商投资准入申请 | 向省级商务部门提交 |
| 2 | 安全审查（如触发） | 涉及关键材料，触发审查 |
| 3 | 行业主管部门意见 | 新材料行业需工信部意见 |
| 4 | 发改委项目核准 | 大额投资项目核准 |
| 5 | 生态环境部门环评 | 制造业项目环评 |
| 6 | 市场监管登记 | 股权变更登记 |

#### 合规亮点

| 要点 | 说明 |
|------|------|
| **前置沟通** | 提前与商务部门沟通准入可能性 |
| **合资模式** | 选择中方控股，降低审批难度 |
| **材料准备** | 完整准备资金证明、国别风险评估 |

#### 经验总结
> "境外→境内"投资需重点关注负面清单和是否触发安全审查，限制类行业建议选择中方控股合资模式。

---

### 场景C：境外→港澳（境外科技公司港股上市）

| 维度 | 虚构设定 |
|------|----------|
| **资产流向** | 境外资产 → 香港 |
| **投资方** | 境外注册科技公司（创始人原为境内居民） |
| **投资标的** | 拟在港交所上市 |
| **架构设计** | 创始人→BVI→开曼（上市主体）→香港→境内运营 |

#### 合规路径执行

| 步骤 | 操作 | 合规要点 |
|------|------|----------|
| 1 | 37号文登记（创始人） | 设立SPV后30日内完成 |
| 2 | 目标地法律尽调 | 开曼/香港法律合规 |
| 3 | 港交所上市申请 | 按港交所规则准备材料 |
| 4 | 中国证监会备案 | 境外上市需备案 |
| 5 | CRS合规 | 实际控制人信息申报 |

#### 合规亮点

| 要点 | 说明 |
|------|------|
| **37号文前置** | 创始人SPV设立后及时登记 |
| **架构清晰** | 股权穿透图清晰，便于审核 |
| **两地协同** | 境内外律师团队协同 |

#### 经验总结
> "境外→港澳"（港股上市）需同时满足：①境内37号文登记、②港交所规则、③CRS合规。三个维度缺一不可。

## 二、失败教训场景（合规红线警示）

### 场景D：境内→境外，先投资后备案

| 维度 | 虚构设定 |
|------|----------|
| **资产流向** | 境内资金 → 境外 |
| **投资方** | 境内某制造企业 |
| **投资内容** | 境外收购同行业公司 |
| **违规行为** | 资金已出境，ODI备案尚未完成 |

#### 违规分析

| 违规点 | 说明 |
|--------|------|
| **流程倒置** | 资金出境先于ODI备案完成 |
| **主观故意** | 企业心存侥幸 |
| **监测发现** | 外汇管理局监测发现异常 |

#### 处罚结果

| 处罚类型 | 说明 |
|----------|------|
| 罚款 | 涉案金额5‰-5%（顶格） |
| 交易 | 可能被认定为无效 |
| 责任人 | 可能追究法律责任 |

#### 教训总结
> **"境内→境外"场景：ODI备案必须先于资金出境完成**，这是最基本的合规红线。

---

### 场景E：境内→境外，分拆购汇购置境外房产

| 维度 | 虚构设定 |
|------|----------|
| **资产流向** | 境内个人资金 → 境外 |
| **投资方** | 境内居民个人 |
| **投资内容** | 境外某城市购置房产 |
| **违规行为** | 借用家人5人额度，分拆购汇约25万美元 |

#### 违规分析

| 违规点 | 说明 |
|--------|------|
| **分拆购汇** | 多人购汇汇往同一境外账户 |
| **用途虚假** | 申报用途与实际不符 |
| **额度滥用** | 借用他人便利化额度 |

#### 处罚结果

| 处罚类型 | 说明 |
|----------|------|
| 罚款 | 涉案金额10%-30% |
| 名单 | 列入外汇违规关注名单 |
| 处置 | 可能被要求出售房产 |

#### 教训总结
> **"境内→境外"个人场景：境外房产购汇没有合规渠道**，任何"绕过"方式都是违规的。

---

### 场景F：境外→境内，通过身份转换规避安全审查

| 维度 | 虚构设定 |
|------|----------|
| **资产流向** | 境外资金 → 中国境内 |
| **投资方** | 境内某科技企业+创始人变更国籍 |
| **投资内容** | 拟被境外某国企业收购部分股权 |
| **违规行为** | 创始人变更国籍，试图规避安全审查 |

#### 违规分析

| 违规点 | 说明 |
|--------|------|
| **身份转换** | 创始人变更为第三国国籍 |
| **穿透审查** | 监管机构穿透至实质控制人 |
| **规避意图** | 变更时间与交易时间高度吻合 |

#### 处罚结果

| 结果 | 说明 |
|------|------|
| 交易被否 | 安全审查拒绝批准 |
| 公开警示 | 案例被作为典型通报 |
| 后续影响 | 企业和负责人进入关注名单 |

#### 教训总结
> **"境外→境内"场景：安全审查将穿透至实质控制人，身份转换无法规避监管。**

## 三、案例启示总结

### 成功要素（按流向区分）

| 流向 | 成功要素 |
|------|----------|
| **境内→境外** | 流程合规（先备案后投资）、本地化运营、政策利用 |
| **境外→境内** | 前置沟通、选择合资模式、完整材料准备 |
| **境外→港澳** | 37号文前置登记、两地法律协同、CRS合规 |

### 失败教训（按流向区分）

| 流向 | 失败教训 |
|------|----------|
| **境内→境外** | 先投资后备案、分拆购汇、地下钱庄 |
| **境外→境内** | 身份转换规避审查、忽视安全审查、材料不完整 |
| **境外→港澳** | 37号文遗漏登记、CRS穿透风险 |
