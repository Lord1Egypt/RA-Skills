## Known Risks and Mitigations: <br>
Risk: The skill requires a global install of pipiads-mcp-server@1.0.3. <br>
Mitigation: Review the package and install it only in an environment where global npm packages are acceptable. <br>
Risk: The skill requires PIPIADS_API_KEY access and searches may consume paid credits. <br>
Mitigation: Provide a scoped account key where possible and review PipiAds/PipiSpy billing, recharge, and credit usage before running searches. <br>
