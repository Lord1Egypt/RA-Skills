# Pipeline Template

Use one simple file to keep momentum visible.

```markdown
# Social Pipeline

## Habitats to Test
- [ ] Tuesday run club
- [ ] Thursday language exchange

## Active People
- Name:
  Source:
  Last interaction:
  Next move:
  Reciprocity: green | yellow | red

## Second-Meeting Candidates
- Person:
  Suggested plan:
  Timing:

## Pause / Archive
- Person or context:
  Reason:
```

If the file gets crowded, move stale items to `archive/` instead of deleting context.
