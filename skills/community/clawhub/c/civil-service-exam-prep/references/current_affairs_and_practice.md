## 真题解析方法论

### 行测真题使用策略

1. **分模块刷**（基础阶段）：按模块做真题，掌握各模块出题规律
2. **分套刷**（强化阶段）：整套计时做，训练时间感和考试节奏
3. **重点刷**（冲刺阶段）：近3-5年真题反复做，特别是错题

### 行测真题复盘流程
```
做错一道题 →
1. 判断：是知识点盲区？还是粗心？还是时间不够？
2. 如果是知识点问题：回到方法论，做3道同类题巩固
3. 如果是粗心：标注陷阱类型（单位换算？选非题？偷换概念？）
4. 如果是时间问题：找更快的解题方法（秒杀技巧/估算/排除）
5. 记录到错题本
```

### 申论真题使用策略

1. **精做**（基础阶段，2-3天/套）：
   - 先自己作答
   - 对照参考答案逐题分析差距
   - 抄写一遍参考答案，体会规范表达
2. **模仿**（强化阶段）：限定时间内作答，逼迫自己快速提取材料
3. **反复做**（冲刺阶段）：同一套真题可以做2-3遍，每次侧重点不同

### 申论小题评分标准模拟
```
归纳概括题（20分）：
- 要点全面（12分）：每个要点2-3分，缺一个扣相应分
- 条理清晰（4分）：分类合理、逻辑清楚
- 语言规范（4分）：用词准确、表述简洁

综合分析题（20分）：
- 观点明确（5分）：态度清晰，不模棱两可
- 分析深入（10分）：多角度、有层次
- 逻辑严谨（5分）：论证有条理
```

### 大作文评分参考
| 档次 | 分数 | 标准 |
|------|------|------|
| 一类文 | 32-40 | 立意深刻、结构严谨、论证充分、语言优美 |
| 二类文 | 24-31 | 立意准确、结构完整、论证较充分 |
| 三类文 | 16-23 | 立意基本准确、结构基本完整 |
| 四类文 | 0-15 | 偏题跑题、结构混乱、字数不足 |

### 面试真题训练方法

1. **看题思考**：拿到题目，先自己思考30秒-1分钟
2. **录音作答**：用手机录音自己的回答（控制3-4分钟/题）
3. **回听复盘**：听录音，从内容、逻辑、流畅度三个维度自评
4. **对照参考**：看优质答案，补充自己遗漏的角度
5. **重新作答**：消化后再次作答，对比进步
