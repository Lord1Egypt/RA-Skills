## Description: <br>
Creates Vision Node AI button controls from a customer description, generating aiui.json and index.html files and packaging them as a ZIP archive. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wujianyu09](https://clawhub.ai/user/wujianyu09) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and Vision Node builders use this skill to create packaged AI button controls with metadata, browser-based interaction logic, and parent-page messaging behavior. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill creates generated control files and ZIP packages on the user's machine. <br>
Mitigation: Review generated files and expected output paths before deployment. <br>
Risk: Generated buttons communicate with a parent page. <br>
Mitigation: Embed generated controls only in trusted Vision Node environments and review index.html before use. <br>
