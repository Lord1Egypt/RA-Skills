# D03 教育领域任务目录

## 领域定义

教育领域(D03)关注个体知识获取、能力发展和潜能实现。核心驱动需求：N4成长。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人学习规划 | 学校运营管理 |
| 知识获取方法 | 课程设计服务 |
| 技能培养指导 | 教师培训 |
| 学习效果评估 | 教育政策制定 |
| 学习资源推荐 | 教育产品开发 |
| 终身学习规划 | 学历认证 |
