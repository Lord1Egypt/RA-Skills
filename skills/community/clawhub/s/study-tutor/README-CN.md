# Study Tutor Skill

**Study Tutor** 是一个精简版、基于认知科学的学习辅导 Skill，用来帮助学生和自学者系统学习。它覆盖学前诊断、引导式讲解、练习、复盘、间隔复习、错题分析和学习档案追踪。

## 本版本做了什么修改

这个版本把原来很长的 Skill 压缩成更适合模型执行的 `SKILL.md`。删除了大量重复示例、过长模板和中英文混排内容，以减少 token 消耗。Skill 名称统一为 **Study Tutor**。

## 核心功能

- 学前诊断：目标、基础、资料、截止时间、可用时间。
- 教师备课：优先阅读用户提供的教材、讲义、课件和题目。
- 四种学习模式：引导式、批量式、问题驱动式、混合式。
- 教学闭环：讲解 → 检验 → 反馈 → 练习 → 记录 → 复习。
- 主动回忆与间隔重复。
- 错题分析与薄弱点修复。
- 独立学习档案，支持跨会话延续学习。

## 触发示例

用户说出类似内容时可以使用该 Skill：

- “我要学习数据结构。”
- “这个概念我不懂。”
- “帮我看一下这道作业题。”
- “我快考试了，帮我复习。”
- “帮我整理错题。”
- “帮我制定学习计划。”

## 推荐工作流

1. 诊断学习目标、当前基础、学习资料和可用时间。
2. 优先根据用户提供的教材、笔记、课件或题目备课。
3. 选择学习模式。
4. 按“直觉 → 细节 → 例题 → 易错点 → 检验题”的结构讲一个知识单元。
5. 根据用户回答给出反馈。
6. 记录进度、错题、薄弱点和下次复习内容。
7. 用主动回忆和间隔重复巩固知识。

## 学习模式

| 模式 | 适合情况 | 方式 |
|---|---|---|
| 引导式 | 零基础、基础薄弱、追求高分 | 一次一个概念，讲完立即检验。 |
| 批量式 | 有一定基础、时间紧张、复习场景 | 一次讲几个相关知识点，再综合检测。 |
| 问题驱动式 | 具体概念不懂、作业卡住 | 解决当前问题，同时补底层知识点。 |
| 混合式 | 大多数学习场景 | 根据难度动态切换模式。 |

## 学习档案

长期学习时，建议单独创建：

```text
memory/{subject}-study.md
```

学习档案应记录：

- 学习目标与基础；
- 各章节/知识点进度；
- 薄弱点；
- 错题记录；
- 下次复习和练习计划。

## 科学依据

该 Skill 借鉴了以下学习方法：

- 主动回忆（Active Recall）
- 间隔重复（Spaced Repetition）
- 测试效应（Testing Effect）
- 精细化加工（Elaboration）
- 费曼技巧（Feynman Technique）
- 康奈尔笔记法（Cornell Note-taking）

## 设计原则

`SKILL.md` 应保持短小精炼，只放模型真正需要执行的行为规则、核心模板和必要流程。长示例、解释性文字和宣传性内容应放到 README 或外部文档里。
