# Cloud Game

通过自然语言在天翼云游戏平台搜索并启动游戏。

## 安装

```bash
clawhub install cloud-game
```

## 使用方式

### 玩指定游戏
直接告诉 AI 你想玩的游戏名称：
- "我要玩原神"
- "打开飞速骑行"

AI 会自动匹配游戏，展示简介，并在浏览器中打开游戏页面。

### 搜索推荐游戏
告诉 AI 你想玩什么类型的游戏：
- "有什么赛车游戏"
- "推荐冒险类游戏"

AI 会展示匹配的游戏列表，你选择后在浏览器中打开。
