# 舌苔诊断

## 一、苔质

### 1. 薄、厚苔
- **薄苔**：透过舌苔能隐隐见到舌体
  - 临床意义：属正常舌苔，有病见之亦属病轻浅，邪在表
- **厚苔**：不能透过舌苔见到舌体
  - 临床意义：主病邪入里，或食积痰湿

### 2. 润、燥苔
- **润苔**：舌苔干湿适中，不滑不燥
  - 临床意义：正常舌苔，津液上承之征
- **滑苔**：苔面有过多的水分，滑利而湿
  - 临床意义：主水湿内停
- **燥苔**：苔面干燥，望之枯涸
  - 临床意义：主热盛伤津或阴液亏耗

### 3. 腻、腐苔
- **腻苔**：苔质致密，颗粒细腻，舌边苔薄舌中苔厚，刮之难去，状如油腻覆盖舌面
  - 临床意义：主湿浊、痰饮、湿热
  - 白腻苔 → 寒湿内停
  - 黄腻苔 → 湿热内蕴
- **腐苔**：苔质疏松，颗粒较大，舌边舌中皆厚，刮之易去，状如豆腐渣堆积舌面
  - 临床意义：主食积胃肠，或痰浊内蕴

### 4. 剥苔
- **舌象特征**：舌本有苔，忽然脱去，脱落处光滑无苔
  - 部分剥脱 → 胃气、胃阴不足
  - 地图舌 → 剥苔呈地图状分布
  - 镜面舌 → 全舌之苔脱落，舌面光洁如镜
- **临床意义**：
  - 剥苔 → 胃气、胃阴不足
  - 镜面舌 → 胃气、胃阴枯竭（危重证候）

### 5. 有根、无根苔
- **有根苔**：舌苔坚敛着实，刮之不去
  - 临床意义：有胃气的表现，见于实证、热证
- **无根苔**：舌苔如浮涂舌上，刮之即去
  - 临床意义：胃气大伤的表现，见于虚证、寒证

## 二、苔色

### 1. 白苔
- **舌象特征**：舌苔为白色
- **细分**：
  - 薄白苔 → 正常或表证初起
  - 白厚苔 → 寒湿内盛或食积
- **临床意义**：主表证、寒证

### 2. 黄苔
- **舌象特征**：舌苔为黄色
- **细分**：
  - 淡黄苔（微黄）→ 在薄白苔上出现均匀的浅黄色，多由薄白苔转化而来
  - 深黄苔（正黄）→ 苔色黄而略深厚
  - 焦黄苔（老黄）→ 正黄色中夹有灰褐色苔
- **临床意义**：主热证、里证
- **黄色越深**：热邪越重
  - 淡黄 → 热轻
  - 深黄 → 热重
  - 焦黄 → 热极

### 3. 灰黑苔
- **舌象特征**：苔色为灰色或黑色
- **细分**：
  - 黑润苔 → 苔色黑而湿润 → 寒极
  - 焦黑苔 → 苔色黑而干燥 → 热极津枯
- **临床意义**：主里热或里寒的重证
- **判断要点**：
  - 苔质润燥是关键：润→寒证，燥→热证
