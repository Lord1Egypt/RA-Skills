## 范本1：新城市安顿全流程

### 题目
30岁专业人士搬迁到新城市的安顿全流程

### 背景
赵女士，30岁，市场经理，因工作调动需要从北京搬到成都。需要在新城市建立完整的生活系统，包括住房、通勤、社交、生活设施等。希望在3个月内完全安顿下来，恢复正常生活节奏。

### Step 1: 住房选择阶段

#### 1.1 住房选择 (LO2-01)
**输入**: 需求分析框架，位置评估标准，预算约束
**执行**:
1. 分析住房需求：
   - 工作地点：高新区某写字楼
   - 通勤时间：希望30分钟内
   - 住房类型：一室一厅或两室一厅
   - 预算：月租3000-4000元
   - 附加需求：电梯、安保、周边配套
2. 确定目标区域：
   - 首选：高新区（近公司）
   - 次选：武侯区（生活便利）
3. 收集房源信息：
   - 链家、贝壳找房
   - 小红书、豆瓣租房小组
   - 公司内部租房信息
4. 实地考察评估：
   - 考察3-5个房源
   - 评估交通、配套、环境
   - 检查房屋状况
5. 做出租房决策

**输出**: 租房决策报告

#### 1.2 搬家规划 (LO2-02)
**输入**: 搬家时间表，物品清单，搬家公司选择
**执行**:
1. 制定搬家时间表：
   - 提前1个月：整理物品，联系搬家公司
   - 提前2周：打包非必需品
   - 提前1周：打包日常用品
   - 搬家前1天：最后打包和清洁
2. 整理物品清单：
   - 必需品：衣物、工作用品、重要文件
   - 可选品：书籍、装饰品、厨房用品
   - 处理品：不需要的物品捐赠或出售
3. 选择搬家公司：
   - 比较价格和服务
   - 查看评价和口碑
   - 确认保险和赔偿
4. 准备打包材料：
   - 纸箱、气泡膜、胶带
   - 标签和记号笔
5. 执行搬家计划

**输出**: 搬家执行方案

**校准点1**: 住房安顿完成
- 确认租房合同签订
- 确认搬家完成
- 确认基本生活设施就绪

### Step 2: 生活系统建立

#### 2.1 日历系统搭建 (LO1-01)
**输入**: 日历工具选择，日历分类框架，同步设备清单
**执行**:
1. 选择日历工具：Google Calendar（跨平台同步）
2. 建立日历分类：
   - 工作：会议、项目、截止日期
   - 个人：社交、爱好、自我提升
   - 家庭：家人联系、家庭事务
   - 健康：运动、体检、休息
3. 设置同步设备：手机、电脑、平板
4. 迁移重要日程：工作安排、重要约会
5. 建立使用习惯：每天查看，每周规划

**输出**: 日历系统配置

#### 2.2 通勤规划 (LO3-01)
**输入**: 通勤路线选项，交通方式比较，时间成本分析
**执行**:
1. 分析通勤需求：
   - 起点：新租房地址
   - 终点：公司地址
   - 时间：希望30分钟内
2. 收集路线选项：
   - 地铁：1号线转7号线
   - 公交：直达公交
   - 骑行：共享单车
   - 打车：网约车
3. 比较交通方式：
   - 时间：地铁35分钟，公交45分钟，骑行20分钟
   - 成本：地铁5元，公交2元，骑行1.5元，打车20元
   - 舒适度：地铁中等，公交较低，骑行较高
4. 计算时间成本：选择地铁+骑行组合
5. 选择最佳方案：地铁为主，天气好时骑行

**输出**: 通勤规划方案

#### 2.3 居住环境优化 (LO2-07)
**输入**: 环境评估标准，优化目标设定，优化方案设计
**执行**:
1. 评估当前环境：
   - 优点：位置好，交通便利
   - 缺点：家具简单，缺少收纳
2. 设定优化目标：
   - 短期：满足基本生活需求
   - 中期：提升生活舒适度
   - 长期：打造温馨居住环境
3. 设计优化方案：
   - 收纳：购买收纳盒、衣架
   - 舒适：添加地毯、靠垫
   - 美化：绿植、装饰画
4. 准备优化材料：
   - 预算：2000元
   - 采购：淘宝、宜家
5. 实施优化计划

**输出**: 居住环境优化方案

**校准点2**: 生活系统建立
- 确认日历系统正常运行
- 确认通勤方案可行
- 确认居住环境改善

### Step 3: 生活设施对接

#### 3.1 居家维护 (LO2-05)
**输入**: 维护项目清单，维护频率规划，维护工具准备
**执行**:
1. 列出维护项目：
   - 日常：清洁、通风、垃圾处理
   - 定期：空调滤网、热水器、燃气灶
   - 季节：空调清洗、地暖检查
2. 确定维护频率：
   - 日常：每天/每周
   - 定期：每月/每季度
   - 季节：每年
3. 准备维护工具：
   - 基本工具：螺丝刀、扳手
   - 清洁工具：吸尘器、拖把
   - 安全工具：灭火器、急救包
4. 建立维护计划
5. 记录维护情况

**输出**: 居家维护计划

#### 3.2 邻里关系 (LO2-06)
**输入**: 邻里交往原则，社区规则了解，沟通技巧
**执行**:
1. 了解社区规则：
   - 垃圾分类规定
   - 噪音控制要求
   - 公共设施使用
2. 学习邻里交往原则：
   - 礼貌问候
   - 尊重隐私
   - 适度帮助
3. 主动建立联系：
   - 介绍自己
   - 了解邻居
   - 交换联系方式
4. 参与社区活动：
   - 社区清洁日
   - 邻里聚会
5. 维护良好关系

**输出**: 邻里关系建立计划

#### 3.3 紧急联系人 (LO6-04)
**输入**: 联系人分类，联系信息收集，联系方式管理
**执行**:
1. 分类紧急联系人：
   - 家人：父母、兄弟姐妹
   - 朋友：成都本地朋友
   - 同事：公司同事、领导
   - 服务：物业、维修、急救
2. 收集联系信息：
   - 家人：已有
   - 朋友：新认识的朋友
   - 同事：公司通讯录
   - 服务：物业、维修电话
3. 管理联系方式：
   - 手机通讯录分组
   - 紧急联系人设置
   - 联系信息备份
4. 创建联系人列表
5. 定期更新信息

**输出**: 紧急联系人列表

**校准点3**: 生活设施对接
- 确认居家维护计划可行
- 确认邻里关系良好
- 确认紧急联系人完备

   - 早晨：每天

## 范本2：双职工家庭生活系统

### 题目
35岁双职工家庭的生活系统建立全流程

### 背景
刘先生和刘太太，35岁，都是企业员工，有一个5岁的孩子。双方工作繁忙，经常加班，家务劳动分配不均，生活效率低下。希望建立高效的家庭生活系统，平衡工作与家庭，提升生活质量。

### Step 1: 现状评估

#### 1.1 生活质量评估 (LO7-01)
**输入**: 评估维度框架，评估标准设定，评估方法选择
**执行**:
1. 确定评估维度：
   - 时间管理：工作时间、家庭时间、个人时间
   - 家务效率：清洁、烹饪、整理
   - 家庭关系：夫妻关系、亲子关系
   - 个人健康：运动、睡眠、压力
2. 设定评估标准：1-10分
3. 选择评估方法：家庭会议讨论
4. 执行生活质量评估：
   - 时间管理：4分（工作时间过长）
   - 家务效率：3分（缺乏系统）
   - 家庭关系：6分（基本良好）
   - 个人健康：5分（缺乏运动）
5. 分析评估结果：主要问题是时间管理和家务效率

**输出**: 家庭生活质量评估报告

#### 1.2 时间审计 (LO1-07)
**输入**: 时间记录方法，时间分类框架，效率评估标准
**执行**:
1. 记录一周时间使用：
   - 工作：每天10小时
   - 通勤：每天2小时
   - 家务：每天2小时
   - 陪伴孩子：每天1小时
   - 个人时间：每天0.5小时
   - 睡眠：每天6小时
2. 分类时间使用类型：
   - 必要时间：工作、通勤、睡眠
   - 家庭时间：家务、陪伴
   - 个人时间：自我提升、休息
3. 评估时间使用效率：
   - 工作时间过长
   - 家务时间低效
   - 个人时间不足
4. 识别优化空间：
   - 减少无效工作时间
   - 优化家务流程
   - 增加个人时间
5. 制定优化策略

**输出**: 时间审计报告

**校准点1**: 现状评估完成
- 确认生活质量评估准确
- 确认时间审计详细
- 确认问题识别清晰

### Step 2: 系统设计

#### 2.1 家务分工 (LO4-06)
**输入**: 家务项目清单，家庭成员能力评估，分工原则
**执行**:
1. 列出所有家务项目：
   - 日常：做饭、洗碗、洗衣、扫地
   - 定期：清洁卫生间、整理衣柜、买菜
   - 特殊：照顾孩子、辅导作业
2. 评估家庭成员能力：
   - 刘先生：擅长做饭、修理
   - 刘太太：擅长整理、照顾孩子
   - 孩子：5岁，可以做简单家务
3. 制定分工原则：
   - 能力匹配
   - 时间平衡
   - 轮换制度
4. 分配家务任务：
   - 刘先生：做饭、修理、周末清洁
   - 刘太太：日常整理、照顾孩子
   - 孩子：收拾玩具、摆碗筷
5. 建立协调机制：家庭会议、家务清单

**输出**: 家务分工方案

#### 2.2 时间块规划 (LO1-02)
**输入**: 时间块类型定义，典型日程模板，优先级框架
**执行**:
1. 分析个人精力曲线：
   - 刘先生：早晨精力好，晚上疲惫
   - 刘太太：下午精力好，早晨疲惫
2. 定义时间块类型：
   - 专注块：工作、学习
   - 休息块：午休、晚上放松
   - 缓冲块：通勤、过渡
3. 创建典型日程模板：
   - 早晨：6:30-7:30（个人时间）
   - 上午：8:00-12:00（工作）
   - 中午：12:00-13:00（午休）
   - 下午：13:00-18:00（工作）
   - 晚上：18:00-22:00（家庭时间）
4. 应用到实际日程
5. 根据效果调整

**输出**: 时间块规划表

#### 2.3 提醒系统 (LO1-05)
**输入**: 提醒工具选择，提醒层级设计，提醒触发规则
**执行**:
1. 选择提醒工具：
   - 手机提醒
   - 智能音箱
   - 日历提醒
2. 设计提醒层级：
   - 紧急：立即处理
   - 重要：当天处理
   - 一般：本周处理
3. 设置触发规则：
   - 时间触发：固定时间提醒
   - 位置触发：到家提醒
   - 事件触发：完成某事提醒
4. 测试提醒效果
5. 优化提醒策略

**输出**: 提醒系统配置

**校准点2**: 系统设计完成
- 确认家务分工合理
- 确认时间块规划可行
- 确认提醒系统有效

### Step 3: 工具配置

#### 3.1 日历系统搭建 (LO1-01)

#### 3.2 待办事项管理 (LO1-03)

## 范本对比分析

### 共同点
1. 都从现状评估开始
2. 都包含系统设计
3. 都涉及习惯养成
4. 都进行持续优化
5. 都注重生活质量提升

### 差异点
1. 范本1：新城市安顿，重点在生活系统建立
2. 范本2：双职工家庭，重点在时间管理和家务分工
3. 范本3：极简生活转型，重点在物品精简和消费控制

### 关键成功因素
1. 现状评估准确
2. 系统设计合理
3. 习惯养成坚持
4. 持续优化改进
5. 家庭支持配合

### 常见挑战
1. 旧习惯抵抗
2. 缺乏时间精力
3. 家庭成员不配合
4. 系统过于复杂
5. 缺乏持续动力

### 应对策略
1. 小步骤改变
2. 简化系统设计
3. 家庭会议沟通
4. 定期复盘调整
5. 建立奖励机制
