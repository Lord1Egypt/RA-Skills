### 🔴 红线 6：空间站位禁止脑补

**铁律**：空间位置必须逐字对照原文，禁止添加原文未提及的方位词

> **与 Step 4（抽象→具象转译）的区别**：
> - 红线 6 管的是**空间位置**（谁在哪里、朝向哪里）——必须有原文依据，不可脑补
> - Step 4 管的是**情绪/心理描写**（内心崩溃、鼓起勇气）——原文没有具象描述时，允许转译为可拍画面

**禁止事项**：
- ❌ 禁止添加"左/右/前/后方"等原文未提及的方位词
- ❌ 禁止添加原文未描述的视线关系
- ❌ 禁止"合理脑补"——即使逻辑通顺，原文没写就不能加

**正确做法**：
```
原文："收银台后，林屿抬头"
✅ 正确：林屿(收银台后，抬头)
❌ 错误：林屿(收银台右后方，看向两人)
```

---

### 🔴 红线 7：人物移动路径必须写清

**铁律**：每个角色的位置变化必须包含"从 A 走到 B"的过程描述

**正确做法**：
```
❌ 错误（只写结果）：小满(手边放热水)
✅ 正确（写清路径）：小满(从收银台走到小桌旁，放热水到唐柚手边)

❌ 错误（模糊描述）：小满(远处看过来)
✅ 正确（精确锚定）：小满(收银台后，远远看向小桌方向)
```

---

### 🔴 红线 8：防越轴检查表禁止只打✅

**铁律**：防越轴检查表每格必须填写具体描述

**正确做法**：
```
❌ 错误：移动过程 | ✅ | ✅ | ✅
✅ 正确：移动过程 | 推门过渡 | 进门→货架 | 货架→收银台
```

### 🔴 红线 9：分镜字段禁止两套标准混用

**铁律**：每个镜头必须且只能按核心规范第 2 节的 9+1 字段顺序填写，禁止简写或遗漏字段。

**禁止事项**：
- ❌ 禁止 8 要素式简写（如只写"特写，静止，50mm"）
- ❌ 禁止遗漏焦段/光圈/机位/构图四项
- ❌ 禁止在机位字段只写"仰拍近景"（必须按机位强制模板）

> 字段顺序与示例见 → [核心规范 §2 镜头 9+1 字段公式](#2-镜头-91-字段公式)

### 🔴 红线 10：负面词禁止照搬通用列表

**铁律**：每个剧本必须根据类型动态生成负面词，禁止直接复制通用负面词列表。

**必须包含三层**：
```
Layer 1 基础负面词（所有类型通用）
Layer 2 类型专属负面词（根据剧本类型选配）

```

**正确做法**：
```
✅ 都市剧负面：基础 + 避免服装突变 + 避免夜景噪点
```

### 📝 Step 2: 台词逐句提取（P0）

**流程**：标记对话 → 标记心理描写 → 标记旁白 → 提取表格 → 二次核对 → 统计字数 → 计算时长 → 超长拆分

**7 种语速标准**：

| 类型 | 语速 | 适用场景 |
|:---|:---:|:---|
| 舒缓抒情温柔对话 | 3.0 字/秒 | 浪漫场景 |
| 沉思安静内心独白 | 3.2 字/秒 | 内心独白、回忆 |
| 默认正常情节对白 | 4.4 字/秒 | 日常对话 |
| 剧情推进对话 | 4.6 字/秒 | 情节发展 |
| 客观叙事旁白 | 4.8 字/秒 | 旁白叙述（无停顿、无情绪起伏，语速最匀） |
| 紧张对峙激烈冲突 | 5.0 字/秒 | 对峙、争吵 |
| 慌乱惊恐急促情绪 | 5.5 字/秒 | 逃跑、惊恐 |

**时长计算**：单句时长 = 字数 ÷ 语速；超长台词（>40 字或>15 秒）必须拆分

**台词时长是镜头时长的硬下限**：
- 计算出台词时长后，对应镜头时长 **必须 ≥ 台词时长**
- ❌ 禁止反向操作：用镜头时长倒推压缩台词语速
- 示例：台词 22 字，语速 4.4 字/秒 → 时长 5 秒 → 该镜头时长不得低于 5 秒

### 📍 Step 3: 空间状态追踪（P0）

**流程**：
1. 通读原文，识别所有场景
2. 逐句标记原文中的位置描述
3. 记录每个角色的初始位置（附原文依据）
4. 标记剧情推进中的走位（附原文依据）
5. 按段落拆分，记录段首/段尾位置
6. 建立段落间位置传递链
7. 填写防越轴检查表（逐格写具体描述，见红线 5 模板）

**空间状态追踪表模板**：

| 段落 | 场景 | 角色 A 位置（原文依据） | 角色 B 位置（原文依据） | ... |
|:---:|:---|:---|:---|:---|
| 1 | 场景名 | 位置 + "原文" | 位置 + "原文" | ... |
| 2 | 场景名 | 从 X→Y + "原文" | 位置 + "原文" | ... |

### 🎭 Step 4: 抽象→具象转译（P0）

**核心拷问**："这个描述，摄影机拍得到吗？麦克风录得到吗？拍不到/录不到的，必须转译。"

**流程**：
1. 通读原文，扫描所有心理描写、情绪描述、抽象状态
2. 对每个抽象描述，追问"演员会用身体哪一部分来演？"
3. 将抽象描述转译为摄影机能拍到的具象画面
4. 转译结果直接用于分镜生成，不单独输出转译表

**转译速查表**：

| 抽象描述（不可拍） | 具象转译（可拍） |
|---|---|
| 内心崩溃 | 茶碗指节发白；杯沿出现细裂纹；周围声音渐弱 |
| 时间停滞 | 灰尘悬浮；水珠悬空；环境音消失 |
| 心一横/决断 | 握紧拳头；咬肌跳动；瞳孔收缩 |
| 鼓起勇气 | 吸管搅拌动作慢一拍；抬头时机比对方晚 0.5 秒 |
| 心动/暧昧 | 眼神短暂下移再上移；指尖无意识摩挲杯沿 |
| 关系疏远 | 物理距离拉大；中间放置障碍物；同框但不同景深 |
| 不祥预感 | 风吹动纸张；蜡烛火苗倾斜；远处铃铛单声 |
| 故作冷静 | 手指无意识摩挲杯沿，节奏比正常快 1.5 倍 |
| 强忍泪水 | 眼眶微红，鼻尖发红，嘴唇压紧不动 |
| 渐生情愫 | 抬头时机错位 0.3-0.5 秒；身体微微前倾；杯子向对方推 1cm |

## 🔧 核心规范

### 0. 组结构定义

**核心概念**：组 = 最小上传单元。每组独立生成、独立上传到 Seedance，AI 不感知前后组。

**组的基本规则**：

| 属性 | 规则 |
|:---|:---|
| 时长 | 每组 ≤15 秒 |
| 时间码 | 组内归零——第 N 组的镜头 1 永远从 00:00 开始 |
| 自包含 | 每组是独立完整包，不依赖前后组的任何内容 |
| 结构 | 1 组 = 1 个单镜头到底（最长 15s），或 1 组 = 2-4 个镜头快切 |

**组内两种模式**：

```
模式 A：单镜到底（1 组 = 1 个长镜头）
  【镜头 1】【00:00 - 00:15】（16:9）单镜 tracking shot
    ┃ 00:00-00:05 ┃ 阶段描述
    ┃ 00:05-00:10 ┃ 阶段描述
    ┃ 00:10-00:15 ┃ 阶段描述

模式 B：多镜快切（1 组 = 2-4 个镜头，时间码累计 ≤15s）
  【镜头 1】【00:00 - 00:06】（16:9）... 9+1字段 ...
  【镜头 2】【00:06 - 00:12】（16:9）... 9+1字段 ...
  【镜头 3】【00:12 - 00:15】（16:9）... 9+1字段 ...
```

**每组自包含结构**（仅输出用户提交到 Seedance 的内容，AI 思考中间产物不输出）：

```
┌─────────────────────────────────────┐
│ 组标题 + 空间状态（一行）            │
├─────────────────────────────────────┤
│【全局风格与环境】                     │
│  ├ 核心限制（时间/天气/氛围）         │
│  └ 画质与光影（含色彩主题/视觉符号）   │
├─────────────────────────────────────┤
│【镜头 1/2/3...】9+1字段，组内归零    │← 可单镜可多镜
├─────────────────────────────────────┤
│【关键约束】五层级（仅本组内）         │
├─────────────────────────────────────┤
│【音频规范】全局音轨+逐镜头            │
├─────────────────────────────────────┤
│【负面词】三层完整                     │
└─────────────────────────────────────┘
```

**不输出**（AI 内部使用，无需展示给用户）：情绪曲线、语速类型、参考影片

### 1. 全局强制风格层

AI 内部规划时包含三层锚定（参考影片+色彩主题+关键视觉符号），输出时仅保留色彩主题+关键视觉符号，参考影片不出现在最终输出中。输出格式：

```markdown
【全局风格与环境】

🌍 核心限制
时间：{晨/午/黄昏/夜}
天气：{晴/阴/雨/雪/雾}
氛围：{压抑/温暖/紧张/梦幻/冷峻}

🎨 画质与光影
电影质感：{cinematic / realistic / 3DCG} | 胶片：{35mm中颗粒 / ARRI / UE5} | 画幅：{16:9 / 9:16 / 2.35:1}
布光方式：{主光位+光质+光比}
色调：{主调} | 色处理：{通道偏移/色相隔离/色温逻辑}
色彩主题：{主色调} + {辅色调} + {情绪色}
关键视觉符号：{0-2个反复出现的视觉元素}
人物面部质感：{肤色+高光+纹理+质地}
拒绝项：{不要X, NOT Y}
```

### 2. 镜头 9+1 字段公式

每个镜头必须且只能按以下 9+1 字段顺序填写：

```
【镜头 N】【时间码】（画幅）
1️⃣ 视角/景别 + 2️⃣ 机位 + 3️⃣ 焦段/光圈 + 4️⃣ 运镜 +
5️⃣ 主体动作（含台词）+ 6️⃣ 环境细节 + 7️⃣ 构图/取景 +
8️⃣ 焦点/景深 + 9️⃣ 画质标识 + 10️⃣(可选) 光线
```

**字段 7 扩展**：构图后加取景边界（如"取景腰部以上，头顶距画上沿 1/8"），控制画面裁切位置。

**字段 10 规则**：仅当本镜头光线偏离全局默认设置时填写，遵循全局默认时省略。

### 3. 精确时间码（组内归零，0.5 秒精度）

**铁律**：每组时间码从 00:00 开始独立计时，禁止跨组累计。

组内多镜头时（时间码累计 ≤15s）：
```
【镜头 1】【00:00 - 00:06】（16:9）
【镜头 2】【00:06 - 00:12】（16:9）
【镜头 3】【00:12 - 00:15】（16:9）
```

组内单镜分段时（单镜到底，内部分段）：
```
【镜头 1】【00:00 - 00:15】（16:9）单镜
  ┃ 00:00-00:05 ┃ 阶段描述
  ┃ 00:05-00:10 ┃ 阶段描述
  ┃ 00:10-00:15 ┃ 阶段描述
```

**组间规则**：第 N 组的镜头 1 永远从 00:00 开始。跨组时间码不连续，因为每组独立上传。—— ❌ 禁止写【镜头 2】【00:15 - 00:27】（这是跨组累计，每组归零）

### 4. 人物文生图提示词规范

**前置格式**：比例 21:9，真人写实风格，右侧角色头部高清特写，左侧角色全身三视图（正面/侧面/背面），中性全局照明（Neutral Global Illumination），纯白无缝背景，8K 超高清分辨率。

**人物一致性铁律**（每项打✓方可进入下一步）：人物形象统一 · 五官统一 · 发型统一 · 服装统一 · 饰品统一 · 年龄统一 · 气质统一

**核心结构（10 大系统，缺一不可）**：

| 系统 | 必填要素 |
|:---|:---|
| **1. 人物基础身份** | 年龄 + 身份气质 + 情绪气质 + 城市/地区 + 职业/身份风格 + 核心人物气场 |
| **2. 脸型骨相系统** | 颅顶特征 + 脸型轮廓 + 面部折叠度 + 骨相类型 + T 区（眉骨+鼻梁）+ 下颌线 + 下颌角 + 面中 + 颧骨 |
| **3. 核心视觉记忆点** | 必须包含 1-2 个辨识度特征（鼻梁眼镜压痕/黑眼圈/疤痕/黑痣/晒斑/红血丝/嘴角歪斜/缺失手指/胡茬/烫伤/厚茧等），特写与三视图全程 100% 统一 |
| **4. 眼部系统** | 瞳色 + 眼型 + 眼尾 + 下眼睑 + 睫毛 + 眼神气质 |
| **5. 眉毛系统** | 眉形 + 眉峰 + 眉尾 + 毛流感 + 修饰程度，强调职业感与生活感 |
| **6. 鼻部系统** | 山根 + 鼻梁 + 海鸥线 + 鼻头 + 鼻翼 + 鼻骨感 |
| **7. 唇部系统** | 唇厚 + 唇峰 + 唇线 + 嘴角 + 唇色 + 纹理 + 情绪感 |
| **8. 发型系统** | 发色 + 长度 + 造型 + 头发状态（如略显油腻/通勤毛躁/海风吹乱/刚下夜班等） |
| **9. 皮肤与生活痕迹** | 肤色 + 皮肤纹理 + 皱纹 + 黑眼圈 + 晒痕 + 红血丝 + 青春痘 + 劳损；双手职业痕迹（薄茧/烫伤/划痕/指节变形等） |
| **10. 服装系统** | 上身 + 内搭 + 下身 + 鞋子；服装状态（发旧/起皱/洗白/油渍/毛边等），真实生活感，避免 cosplay 感 |

**饰品/职业道具规则**：手表 · 项链 · 文件夹 · 工具 · 包 · 工作证 · 武器 · 杯子 · 车钥匙等 —— 所有道具必须服务职业身份，细节必须真实生活化

**三视图同步规则**：三视图可呈现另一套造型，但人物形象/骨相/五官/年龄/发型/气质必须 100% 统一

**最终气质收束**：整体呈现 {电影类型} 质感 —— 超写实/非 AI 感/非网红脸/真实演员感/强职业辨识度/电影级角色设定图


**禁止**：描述光影/色调/效果（如"暖光照射""侧光"等），纯白背景以外的背景描述

### 4.1 场景文生图提示词规范

**前置格式**：比例 21:9，35mm 人文焦距，中性全局照明，**实景**，**画面中禁止出现任何人物**

**描述结构**：
1. **空间布局**：空间大小、纵深/平面关系、地面材质
2. **关键物件**：固定家具/设备的位置和材质（如收银机、货架、桌椅）
3. **光线氛围**：光源类型（荧光灯/台灯/自然光）、色温（暖黄/冷蓝）
4. **背景细节**：窗外/墙面/天花板可见元素
5. **焦距光圈**：35mm 人文焦距，f/2.8-f/4.0 适中景深

**禁止事项**：
- ❌ 禁止出现任何人物或人体部位
- ❌ 禁止纯白/纯色背景（必须是实景）
- ❌ 禁止描述光影效果（只描述光源，不描述光效）
- ❌ 禁止描述色调/情绪氛围词（如"温暖""诡异"）

**示例**：
```
【便利店外景】 比例21:9，35mm人文焦距，中性全局照明，实景，画面中禁止出现任何人物。
深夜街角，暖黄色"25小时便利店"招牌挂在玻璃门上方，门口立着小型广告牌，
玻璃门透出暖橘色灯光，冷蓝色夜空背景，f/4.0。
```

### 4.2 道具文生图提示词规范

**前置格式**：比例 16:9，50mm 微距焦距，f/1.4-f/1.8 浅景深，**纯白背景**，**无明显光源方向**，全局均匀照明

**描述结构**：
1. **材质**：木质/金属/纸质/塑料等
2. **颜色**：主色 + 辅色（如"浅黄色笔杆，深灰色铅芯"）
3. **形态**：形状、尺寸感、表面纹理（如"木质纹理可见""边缘有齿孔痕迹"）
4. **状态**：静止/使用中的姿态（如"削尖的铅芯""卷曲状态自然"）
5. **多角度**：单一物体多角度展示（正面/侧面/特写），用逗号分隔

**禁止事项**：
- ❌ **禁止描述任何文字内容**（AI 文生图无法精准生成指定文字，会乱码）
- ❌ 禁止描述光影效果（如"暖光照射""高光反射"）
- ❌ 禁止纯白背景以外的背景描述
- ❌ 禁止描述用途（如"用来写字的"）

**示例**：
```
【真话铅笔】 比例16:9，50mm微距焦距，f/1.8浅景深，纯白背景，无明显光源方向，全局均匀照明。
木质铅笔，浅黄色笔杆，木质纹理可见，橡皮擦端，削尖的铅芯，无文字印刷。

【小票】 比例16:9，50mm微距焦距，f/1.4极浅景深，纯白背景，无明显光源方向，全局均匀照明。
白色热敏纸卷，边缘有打印齿孔痕迹，纸面空白或仅有极淡的灰色横纹，卷曲状态自然。
```

### 5. 分组拆分规范

**公式**：按总字数 ÷ 默认语速(4.4字/秒) ÷ 15秒/组 计算。例：1000字 ÷ 4.4 ÷ 15 ≈ 14组（浮动 ≤5%）。每组≤15秒为硬上限，组数向上取整。

**组 = 最小上传单元**，每组独立提交到 Seedance，AI 不感知前后组。

**分组前必须核算台词时长**：

在拆分分组之前，先计算每个剧情段落的台词总时长：
```
段落台词总时长 = Σ（每句字数 ÷ 对应语速）
```
- 台词总时长 ≤ 15s → 可以放入一组，镜头时长按台词时长分配
- 台词总时长 > 15s → **必须拆分为多组**，不得压缩语速来凑进 15s
- ❌ 禁止：台词总时长 18s，强行塞入一组并压缩为 15s

**分组优先级**：
1. 剧情闭环节点 > 站位变化节点 > 语义自然断点 > 15 秒硬限制

**组内结构选择**：

| 场景类型 | 推荐模式 | 说明 |
|:---|:---|:---|
| 连续动作/情绪高潮 | 单镜到底（模式 A） | 一镜贯穿，内部分段，最长 15s |
| 对话/对峙 | 多镜快切（模式 B） | 2-3 个正反打，累计 ≤15s |
| 建立空间/全景 | 单镜到底（模式 A） | 固定长镜，12-15s |
| 快速节奏/追逐 | 多镜快切（模式 B） | 3-4 个短镜快速堆叠 |

**镜头视觉节奏规则**（组内台词 >20 字时强制启用）：

| 组内台词总字数 | 最低分镜要求 | 理由 |
|:---:|:---|:---|
| 0 字（纯画面） | 1 镜即可 | 视觉本身有信息量 |
| 1-20 字 | 1-2 镜 | 短台词，单镜可承受 |
| 20-40 字 | **≥2 镜** | 中等长度，需切一次换视觉节奏 |
| 40-55 字 | **≥2 镜或单镜+视觉动线** | 长台词，必须切镜或视觉变化 |
| 55 字+ | **≥3 镜或单镜+强视觉动线** | 超长台词，必须有多次视觉变化 |

满足字数要求时，选以下任一方案：

```
① 多镜切分法（推荐）
   按台词情绪节点切镜，每个节点至少换一次景别或角度。
   示例：55字/15s = 第7s切一次镜，或分成2-3镜

② 单镜视觉动线法（仅必须一镜到底时）
   镜头内必须有≥2处视觉变化：走位入画/出画、摄运镜、
   焦点转移、前景物体遮挡。
   ❌ 禁止固定镜头拍摄长对话无任何视觉变化

③ 反应镜头穿插法
   长独白中至少切1次听众反应镜头（忍让/握拳/冷笑/围观）
```

**每组输出内容**（仅用户提交到 Seedance 所需，缺一不可）：
- ✅ 组标题 + 空间状态
- ✅ 全局风格与环境（核心限制 + 画质光影，含色彩主题/视觉符号）
- ✅ 镜头描述（9+1字段，时间码组内归零）
- ✅ 关键约束（五层级，仅本组内）
- ✅ 音频规范（全局音轨+逐镜头）
- ✅ 负面词（三层完整）

**禁止**：纯字数硬切、切断走位/对话/造成逻辑断层、跨组共享任何元素

### 6. 音频规范

**铁律**：AI 默认倾向添加背景音乐——必须显式控制。简单场景可禁止 BGM，有叙事需求时须逐镜头设计声场演变。

**输出格式**——分两段：

```markdown
【音频规范】
全局音轨（如有）：{跨镜头持续的单一音轨描述}
逐镜头设计：
  镜头 1：{声音描述}
  镜头 2：{声音描述}
  ...（单镜时按时间分段描述）
```

**规则**：
- 有全局统一音轨（如持续 drone/环境音）时写在"全局音轨"行
- 逐镜头写声音差异；无声/纯对话场景简写为"禁止 BGM"
- 当声音参与叙事推进时（如提示词 1 的机械马金属声逐渐压过环境声），必须在对应镜头/分段中写明声场变化
- ❌ 禁止无音乐声明直接让 AI 自由发挥（AI 默认加 BGM）
- ❌ 禁止有对话时音乐覆盖人声
- ✅ 配音尾音自然渐弱，不硬切

**示例**（无声场景）：
```
【音频规范】禁止生成任何背景音乐，配音声音不要戛然而止，要留自然的尾音
```

**示例**（逐镜设计）：
```
【音频规范】
全局音轨：Tibetan dungchen drone 全程延续（slow majestic drone）
逐镜头设计：
  镜头 1：大漠风声低频 drone + 脚步沙沙 + 金属铠甲吱嘎 + 远方号角 solo
  镜头 2 前段：疲惫呼吸 + 沙地摩擦 + 齿轮卡顿 + 号角延续
  镜头 2 末尾：金属声渐大占主导，其他环境声被遮挡压制，号角延续低频
```

### 7. 单镜持续与时间分段规范

**铁律**：当一段戏为一个连续长镜头（single continuous tracking shot），不允许硬切时，必须在镜头内部按时间轴分段描述。

**格式**：
```
【镜头 1】【00:00 - 00:07】（16:9）单镜 continuous tracking shot
  ┃ 00:00-00:01 ┃ 阶段描述：画面内容/运镜/光线变化
  ┃ 00:01-00:03 ┃ 阶段描述：画面内容/运镜/光线变化
  ┃ 00:03-00:05.5 ┃ 阶段描述：画面内容/运镜/光线变化
  ┃ 00:05.5-00:07 ┃ 阶段描述：画面内容/运镜/光线变化
```

**规则**：
- 第一行标记"单镜"标识，运镜字段写全程运镜方式
- 每个分段用 `┃` 分隔，格式固定：`┃ 时间范围 ┃ 描述（画面+运镜+光线变化）`
- 分段数量 2-5 段，每段不少于 1 秒
- 分段描述中必须写明该阶段"前景遮挡/揭示了什么"（如果有进出画物体）
- 末段描述必须写明镜头结束时的画面状态（全黑/全白/固定画面等）
- 单镜模式下禁止写"镜头 2"，整段只有一个镜头编号

**示例**：
```
【镜头 1】【00:00 - 00:07】（9:16）单镜 slow horizontal tracking
  ┃ 00:00-00:01 ┃ 前景立柱完全遮挡镜头，全黑，仅木质纹理可见
  ┃ 00:01-00:02.5 ┃ 立柱出画右，露出寝宫，空黄袍散落地面，烛光闪烁
  ┃ 00:02.5-00:05.5 ┃ 揭示巨蛇躯干紧贴地面，鳞片缓慢起伏，烛光昏暗
  ┃ 00:05.5-00:07 ┃ 另一立柱入画左，逐渐遮挡画面至全黑
```

### 7.1 运镜衔接 5 模式

**铁律**：一段戏内相邻镜头的运镜方式必须服从衔接规则，禁止随机切换运镜。

| 模式 | 定义 | 适用场景 |
|:---|:---|:---|
| **同向衔接** | 两镜运镜方向一致，速度匹配 | 追逐/行进/连续动作 |
| **反向衔接** | 第二镜以相反方向回应第一镜的方向感 | 对话/对峙/因果反应 |
| **静切动** | 固定镜头切入手持/运动镜头，速度突变制造节奏感 | 情绪转折/突发动作 |
| **动切静** | 运动镜头突然切入固定镜头，制造戛然而止的窒息感 | 震惊/发现/定格时刻 |
| **同质叠加** | 两镜使用相同运镜方式，景别递进或保持 | 仪式/审视/情绪累积 |

**禁止事项**：
- ❌ 禁止一组内 3 种以上运镜模式混用（风格失控）
- ❌ 禁止无理由的静切动/动切静（必须对应情绪节拍）
- ❌ 禁止同向衔接时速度突变（除非有明确剧情动机）

### 8. 层级约束区块

**铁律**：每组输出的末尾（音频规范和负面词之前），必须按执行层级输出【关键约束】区块，确保 AI 逐级扫描确认不遗漏。

**格式**：
```markdown
【关键约束】

🔴 空间级
- {空间继承约束，逐条描述}

🔴 光线级
- {光线约束，逐条描述}

🔴 运镜级
- {运镜约束，逐条描述}

🔴 画面级
- {取景/构图/可见性约束，逐条描述}

🔴 风格级
- {风格/通用约束，逐条描述}
```

**规则**：
- 五层级缺一不可，无约束的层级写"无"
- 每条约束用 NOT / 禁止 / 必须 开头，明确是"禁止"还是"要求"
- 从两组提示词提炼的常见约束条目示例：

| 层级 | 约束示例 |
|:---|:---|
| 空间级 | 镜头2必须承接镜头1空间——士兵是纵队一员；背景必须有其他残兵和机械马 |

| 风格级 | 真人实拍质感，NOT cartoon/3D render/CGI/anime；无字幕无水印 |

## ⚠️ 注意事项

以下条目为核心规范中未单独展开的补充要点，其余规则以正文红线及核心规范为准：
