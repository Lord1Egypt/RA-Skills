## 🏗️ 系统架构

```
用户（自然语言）
      ↓
┌─────────────────┐
│   幕僚长 (ChiefOfStaff)  │  ← LangGraph 状态机，需求理解 + 方案路由
└────────┬────────┘
         ↓ 分发
   ┌─────┼─────┬─────┬─────┐
   ↓     ↓     ↓     ↓     ↓
  采购  生产  销售  财务  通用
  Agent Agent Agent Agent Agent  ← 20个专业Agent，并行生成方案
   ×4    ×4    ×4    ×4    ×4
```
