## 可再生能源生成要求

可再生能源每种任务类型的"零件清单"——必选/可选组件、组装顺序、领域约束。按清单逐项产出。

---

### 1 资源评估

#### T1-1 资源调查

- **必选组件**: 调查范围、调查方法、资源数据、调查报告
- **可选组件**: 调查工具、调查团队、调查预算
- **组装顺序**: 确定调查范围 → 选择调查方法 → 收集资源数据 → 分析资源 → 形成报告
- **约束**: 范围必须明确；方法必须科学；数据必须可靠
- **格式**: 资源调查报告（Markdown）

#### T1-2 资源分析

- **必选组件**: 分析维度、分析方法、分析结果、分析结论
- **可选组件**: 分析工具、分析模型、分析报告
- **组装顺序**: 确定分析维度 → 选择分析方法 → 执行分析 → 形成结论
- **约束**: 维度必须全面；方法必须科学；结论必须客观
- **格式**: 资源分析报告（Markdown）

#### T1-3 资源规划

- **必选组件**: 规划目标、规划内容、规划方案、规划预算
- **可选组件**: 规划评审、规划调整、规划监控
- **组装顺序**: 确定规划目标 → 设计规划内容 → 制定规划方案 → 编制规划预算
- **约束**: 目标必须明确；内容必须完整；方案必须可行
- **格式**: 资源规划方案（Markdown）

#### T1-4 资源优化

- **必选组件**: 优化目标、优化方案、优化措施、优化效果
- **可选组件**: 优化工具、优化模型、优化验证
- **组装顺序**: 确定优化目标 → 设计优化方案 → 实施优化措施 → 评估优化效果
- **约束**: 目标必须明确；方案必须可行；效果必须可验证
- **格式**: 资源优化方案（Markdown）

### 2 技术开发

#### T2-1 技术研发

- **必选组件**: 研发目标、研发方案、研发资源、研发成果
- **可选组件**: 研发团队、研发设备、研发预算
- **组装顺序**: 确定研发目标 → 设计研发方案 → 配置研发资源 → 执行研发 → 产出成果
- **约束**: 目标必须明确；方案必须可行；资源必须充足
- **格式**: 技术研发方案（Markdown）

#### T2-2 技术选型

- **必选组件**: 候选技术、选型标准、对比分析、选型结论

- **组装顺序**: 列出候选技术 → 确定选型标准 → 对比分析 → 验证评估 → 形成结论
- **约束**: 候选必须全面；标准必须合理；对比必须客观
- **格式**: 技术选型报告（Markdown）

#### T2-3 技术集成

- **必选组件**: 集成方案、集成接口、集成测试、集成效果
- **可选组件**: 集成工具、集成团队、集成预算
- **组装顺序**: 设计集成方案 → 定义集成接口 → 执行集成测试 → 评估集成效果
- **约束**: 方案必须可行；接口必须清晰；测试必须充分
- **格式**: 技术集成方案（Markdown）

#### T2-4 技术优化

- **必选组件**: 优化目标、优化方案、优化措施、优化效果
- **可选组件**: 优化工具、优化模型、优化验证
- **组装顺序**: 确定优化目标 → 设计优化方案 → 实施优化措施 → 评估优化效果
- **约束**: 目标必须明确；方案必须可行；效果必须可验证
- **格式**: 技术优化方案（Markdown）
