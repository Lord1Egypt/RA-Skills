# 成都各区小学初中划片对应关系

## 使用说明

本文档记录成都主要区域的小学→初中划片对应关系，供小升初规划参考。

**数据更新时间**：2025年12月
**重要提醒**：划片范围每年可能调整，务必以当年各区教育局官方发布为准！
官方查询渠道：各区教育局官网（锦江区教育局、青羊区教育局、高新区教文卫健局等）

## 四、大摇号（市直属学校）

以下学校面向成都市主城区（或全市）招生，通过自愿报名+摇号方式录取：

| 学校 | 招生范围 | 特点 |
|---|---|---|
| 七中育才（部分校区） | 主城区 | 成都最强初中之一 |
| 树德实验 | 主城区 | 树德系嫡系 |
| 石室联合（部分） | 主城区 | 石室系嫡系 |
| 泡桐树中学 | 主城区 | 泡桐树系嫡系 |
| 四川天府新区师大一中 | 全市/主城区 | 民办优质 |

**大摇号报名条件**：
- 具有成都市主城区户籍（或在主城区就读小学）
- 参加当年小升初信息采集
- 自愿报名，不强制

## 五、典型小学→初中路径案例

### 案例1：锦江区娇子小学
```
娇子小学（新希望D10附近）
→ 根据划片，可能进入：盐道街中学（初中部）或田家炳中学
→ 初中毕业后参加中考，冲七中/树德/石室
```

### 案例2：青羊区泡桐树小学
```
泡桐树小学
→ 根据划片，进入泡桐树中学或石室联合
→ 初中毕业后参加中考，冲树德宁夏/石室文庙
```

### 案例3：高新区玉林小学
```
玉林小学
→ 根据划片，进入玉林紫荆或玉林中学
→ 可享受七中高新调剂名额，初中毕业后冲七中高新
```

## 七、换区策略分析

### 从锦江区换到其他区

| 目标区 | 成本 | 收益 | 适合人群 |
|---|---|---|---|
| **青羊区** | 购房/迁户籍 | 树德实验系质量高，竞争略低于锦江 | 预算充足，想换环境 |
| **高新区** | 购房/迁户籍 | 区内高中容量大，调剂机会多 | 不想太卷，追求稳妥 |
| **天府新区** | 购房 | 新区学校设施好，但距离远 | 接受较远距离 |

**注意**：换区需提前考虑小升初资格（户籍/学籍），建议至少提前1-2年布局。
