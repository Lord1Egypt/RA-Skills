# 尿液pH调控弱酸/碱性药物肾排泄

## 何时使用
- 需加速弱酸性或弱碱性药物排泄（如急性中毒抢救）
- 药物主要经肾脏排泄，且属于可解离的弱电解质
- 药物pKa符合有效范围：弱酸性药pKa 3.0～8.0，弱碱性药pKa 6.0～11.0
- 存在可用的尿液pH调节手段（如碳酸氢钠、氯化铵）

## 如何执行
1. **判断药物性质**：确认药物为弱酸性（如苯巴比妥）或弱碱性（如苯丙胺）。
2. **确定目标尿液pH**：
   - 弱酸性药物 → 碱化尿液（pH > 7.5）
   - 弱碱性药物 → 酸化尿液（pH < 6.0）
3. **选择调节剂**：
   - 碱化：静脉或口服碳酸氢钠
   - 酸化：口服氯化铵（注意禁忌症）
4. **监测与调整**：持续监测尿液pH，维持目标范围以最大化排泄效果。
5. **评估适用性**：排除强酸/强碱、非电解质或主要经胆汁/肺排泄的药物。

> 注意：超出pKa有效范围的药物，尿液pH改变对其排泄影响甚微，不应盲目应用。
