# 中毒急救指南

> **⚠️ 免责声明**：本文仅供参考，不能替代专业急救培训。中毒是紧急情况，请同时拨打120和中毒咨询热线。

---

## 一、中毒通用处理原则

### 识别中毒信号

| 信号 | 可能表现 |
|------|----------|
| 神经系统 | 意识模糊、昏迷、抽搐、头晕、头痛 |
| 呼吸系统 | 呼吸困难、呼吸缓慢、肺水肿 |
| 消化系统 | 恶心、呕吐、腹痛、腹泻 |
| 心血管系统 | 心率异常、低血压/高血压、心律失常 |
| 眼部 | 瞳孔缩小（有机磷/阿片类）/ 散大（阿托品类） |
| 皮肤 | 出汗、干燥、发红、发绀 |
| 特殊气味 | 蒜味（有机磷）、苦杏仁味（氰化物）、酒精味 |

### 通用急救步骤

1. **确保自身安全**：戴手套/口罩，避免二次中毒（如毒气环境）
2. **拨打120**，同时联系 **中毒咨询中心**
3. **收集信息**：毒物名称、剂量、摄入时间、患者年龄体重
4. **保留毒物样本**：药瓶、呕吐物、剩余的食物/液体 → 带给医生
5. **不盲目催吐**！多数情况催吐是危险的

> **中国中毒急救咨询热线**：各地均有中毒中心，120可转接。

## 二、各类中毒专项处理

### 经口中毒（误食/过量服药）

#### 禁止催吐的情况（绝大多数！）：
- 意识不清/已昏迷
- 吞服腐蚀性物质（强酸/强碱）→ 催吐造成二次灼伤
- 吞服石油类产品（汽油/煤油）→ 吸入性肺炎
- 吞服尖锐物体
- 抽搐中

#### 一般处理：
1. 如患者清醒且毒物非腐蚀性：可给予 **活性炭**（需在中毒后1小时内效果最佳）
2. 保持气道通畅，侧卧位防止呕吐物窒息
3. 绝对不要给患者强制灌水

### 一氧化碳中毒

**识别**：
- 轻度：头痛、头晕、恶心、乏力（"流感样"症状）
- 中度：面色潮红（樱桃红）、口唇呈樱桃红色、意识模糊
- 重度：昏迷、抽搐、呼吸停止

**处理**：
1. **立即开窗通风**，关闭燃气
2. 将患者移到通风处（施救者低姿进入，因CO比空气轻浮于上层）
3. 解开领口、腰带
4. 拨打120
5. 如呼吸停止 → 立即CPR
6. **高压氧治疗**是最有效方法，尽快送医

### 食物中毒

| 类型 | 潜伏期 | 典型症状 | 处理 |
|------|--------|----------|------|
| 细菌性 | 1-48小时 | 恶心、呕吐、腹泻、腹痛、发热 | 补充水分电解质（口服补液盐）、禁食6-12小时 |
| 毒蘑菇 | 0.5-24小时 | 幻觉、肝损伤、溶血 | ⚠️ 致命风险！立即送医，保留蘑菇样本 |
| 河豚毒素 | 10分钟-3小时 | 口唇麻木→全身麻痹→呼吸停止 | ⚠️ 致命！立即送医，做好CPR准备 |
| 亚硝酸盐 | 1-3小时 | 口唇指甲发紫、头晕 | 送医，亚甲蓝特效解毒 |

**肉毒杆菌中毒**（⚠️ 极罕见但致死率高）：
- 症状：视物模糊、吞咽困难、下行性肌无力（从头部向四肢发展）
- 需抗毒素血清治疗，立即送医

### 农药中毒

| 类型 | 典型特征 | 处理 |
|------|----------|------|
| 有机磷（敌敌畏/乐果） | 瞳孔缩小、大汗、流涎、肌颤、蒜味 | 脱去污染衣物、清水冲洗皮肤；阿托品+解磷定特效解毒 |
| 百草枯 | 口腔溃烂、进行性肺纤维化 | ⚠️ 极高致死率，尽早洗胃+血液灌流 |

### 药物过量

| 药物类型 | 处理要点 |
|----------|----------|
| 安眠药/镇静剂 | 侧卧位防呕吐窒息，保持呼吸道通畅 |
| 阿片类（吗啡/海洛因/芬太尼） | ⚠️ 呼吸抑制是主要死因；纳洛酮特效解毒 |
| 对乙酰氨基酚（扑热息痛） | 超量72小时内用N-乙酰半胱氨酸解毒，否则肝坏死 |

## 三、中毒急救绝对禁忌

| ❌ | ✅ |
|----|-----|
| 盲目催吐（大多数情况催吐有害） | 遵医嘱或120指导 |
| 用牛奶/生鸡蛋"解毒" | 牛奶可稀释腐蚀性物质但非解毒剂 |
| 让昏迷患者喝水 | 窒息风险，侧卧位 |
| 自行用民间偏方 | 可能加重中毒 |
| 销毁毒物证据 | 保留给医生判断 |

---

## 四、中毒预防要点

- 药品、清洁剂放在儿童接触不到的地方
- 不要将化学品装入饮料瓶
- 使用燃气热水器时保持通风
- 不认识的野生菌类/植物不采不食
- 过期药品妥善处理
