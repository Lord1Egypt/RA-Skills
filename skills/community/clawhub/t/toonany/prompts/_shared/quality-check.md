## 质量检查清单

生成内容后，请自检以下要点：

### 叙事一致性
- [ ] 角色名称在全文中保持一致
- [ ] 场景描述相互匹配
- [ ] 剧情逻辑连贯
- [ ] 情绪曲线合理
- [ ] 经典台词符合角色性格

### 格式规范
- [ ] 提示词具体且可执行
- [ ] 每个分镜包含景别+角度+构图
- [ ] 角色描述不包含样貌（年龄/五官/身材），仅限服化道
- [ ] 时间使用具体词汇（清晨/正午/黄昏），不用笼统的"日/夜"
- [ ] 无抽象描写（气氛尴尬、心中五味杂陈等），全部转化为具体画面

### 资产引用
- [ ] 角色/场景/道具名称与 data.json 完全一致
- [ ] 无缩写、无近义词替换
