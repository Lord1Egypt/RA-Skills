# Plant Watering Vacation Buddy Note

## Purpose
Create a quick room-by-room plant handoff note for a vacation sitter. The note captures plant locations, water amount, timing, and simple owner-provided warnings so the sitter knows what to do and when to skip.

## Use When
- The user is leaving home and someone else must water plants.
- The user wants a printable plant care instruction sheet for a sitter.
- The user needs room-by-room plant notes with water amount and timing.
- The user wants photo slots or labels for easier handoff.

## Do Not Use For
- Plant disease diagnosis, pest diagnosis, or long-term horticulture planning.
- Claims about plant toxicity or safety for pets, children, or people.
- Guessing unknown plant needs when the owner is unsure.
- Replacing owner judgment, nursery guidance, or local expert advice.

## Inputs To Ask For
Ask for:
1. Travel dates and sitter visit dates.
2. Rooms or areas where plants are located.
3. Plant names or simple labels, if known.
4. Water amount per plant, using the user's preferred units.
5. Watering timing: every visit, once mid-trip, only if dry, or skip.
6. Any owner-provided notes such as "do not move," "bottom water," "avoid leaves," or "skip if unsure."
7. Contact method for quick questions, if the user wants it included.

If the user does not know a plant name or water amount, leave a blank line or mark "ask owner" rather than inventing care instructions.

## Workflow
1. Group plants by room or area.
2. Assign each plant a short label the sitter can match to a pot or photo.
3. Record exact water amount and timing only when the owner provides it.
4. Add a "skip if unsure" line for unclear plants, unclear amounts, or questionable soil conditions.
5. Add a simple visit checklist for the sitter.
6. Add optional photo slots so the user can paste or attach photos before printing.
7. Produce a clean handoff note suitable for taping to a door, fridge, or watering can.

## Output Format
Return the following sections.

### Plant Vacation Buddy Note
- Owner:
- Travel dates:
- Sitter visit dates:
- Water source or watering can location:
- Contact for quick questions, optional:
- Main rule: If unsure about a plant, amount, or condition, skip it and ask the owner.

### Room-By-Room Plant Table
Use these columns:
- Room or area
- Plant label
- Photo slot
- Water amount
- Timing
- Special owner note
- If unsure

For the "If unsure" column, use clear phrases such as:
- Skip and text owner.
- Leave dry until confirmed.
- Do not move until confirmed.

### Sitter Visit Checklist
- [ ] Check the date on this note.
- [ ] Match each plant by room, label, and photo.
- [ ] Water only plants with a listed amount and timing for this visit.
- [ ] Skip anything unclear and ask the owner.
- [ ] Wipe spills if needed.
- [ ] Send a quick done message if requested.

### Photo Slot List
Create numbered placeholders:
1. Room:
   Plant label:
   Photo attached:
2. Room:
   Plant label:
   Photo attached:
3. Room:
   Plant label:
   Photo attached:

### Final Hand-Off Note
Write a short friendly note the user can give the sitter. Keep it practical, grateful, and clear.

## Safety And Boundaries
- This skill is for sitter handoff only.
- Do not make toxic plant claims or safety claims about pets, children, or people.
- Do not invent plant species, care needs, water amounts, light needs, or pest treatments.
- If the owner is unsure, use "skip if unsure" and ask the owner.
- If there is an urgent leak, electrical issue, mold, breakage, or unsafe access problem, tell the sitter to stop and contact the owner or appropriate local help.

## Example Prompts

- "I'm going away for two weeks. Help me write a plant watering note for my neighbor."
- "Make a room-by-room watering guide for my vacation plant sitter."
- "Turn my list of plants into a simple handout with when to water and when to skip."
