# Asset Inventory Prompt

Ask the user to list assets and bundled local resources as:

| Name | Type | Current job | Inputs | Outputs | Tools/data | Evidence status | Owner today | Problems | Preserve? |
|---|---|---|---|---|---|---|---|---|---|

Types: skill, agent, prompt, script, tool/API, data source, document, workflow step, human review step, reference file, asset library, template, example, index/manifest, config, release/package metadata.

Evidence status choices: authoritative evidence, derived evidence, advisory reference, visual/style-only, tool-only, template-only, example-only, packaging-only, unknown.
