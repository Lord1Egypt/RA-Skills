# Structure Contract - Academic Literature Presentation

The presentation must follow this exact section hierarchy and content flow.

## Required Slide Sections (in order)

### 1. Cover Slide
- English paper title
- Chinese translated title
- Author names (original)
- Journal name and year
- Impact factor (if available)
- Presenter name (if provided by user)

### 2. Table of Contents (TOC)
- List all main sections with numbered ordering
- Section names in Chinese
- Adapt section names to match the actual paper structure

### 3. 综述 (Literature Review)

**Purpose**: Summarize the theoretical background and prior research that motivated the study.

**Content rules**:
- Translate the introduction/literature review sections of the original paper
- Cover key theories cited in the paper
- Summarize relevant prior studies with citations preserved
- Build the logical case for the current study
- Use 2-5 slides depending on literature complexity
- Split across multiple slides if content exceeds the per-slide limits in the style contract

**Must preserve**:
- All citations in (Author, Year) format
- Key theoretical concepts
- The logical flow from general topic to specific research gap

### 4. 研究假设 (Research Hypotheses)

**Purpose**: Present all formal hypotheses tested in the study.

**Content rules**:
- List each hypothesis with its formal designation (H1, H2, etc.)
- State the expected relationship between variables
- Include sub-hypotheses if present (H1a, H1b, etc.)
- Preserve exact hypothesis wording translated from the paper
- Place hypotheses as plain text directly on white background — no cards, no containers

**Format**:
```
假设 1：xxx（H1）
假设 2：xxx（H2a）。这一效应通过xxx中介（H2b）。
...
```

### 5. 研究理论模型图 (Theoretical Model Diagram)

**Purpose**: Visualize the hypothesized relationships between variables.

**Content rules**:
- Recreate the theoretical model from the paper as a diagram
- Show all independent variables, mediators, and dependent variables
- Draw arrows indicating hypothesized directions of effects
- Label mediators and direct/indirect paths clearly
- Use simple boxes with thin borders on white background
- The model diagram itself is the focal point of this slide

### 6. 各研究 (Individual Studies)

For each study/experiment reported in the paper, include these subsections in order:

#### 6a. 研究参与者 (Participants)
- Sample size (N = xxx)
- Demographics (age, gender distribution)
- Recruitment method
- Selection criteria / inclusion-exclusion criteria
- Compensation/ethics approval info

#### 6b. 实验材料与刺激物 (Materials and Stimuli)
- Description of stimuli/materials used
- Number of items/conditions
- Source or creation method of materials
- Any relevant images of materials (preserve from original)

#### 6c. 实验程序 (Procedure)
- Step-by-step experimental procedure
- Order of tasks/measurements
- Timing information (duration, intervals)
- Randomization/counterbalancing details

#### 6d. 研究方法/测量工具 (Methods/Measures)
- Name of scales/questionnaires used with citations
- Number of items, response scale format
- Reliability coefficients if reported (Cronbach's alpha)
- Any physiological measures with equipment names

#### 6e. 结果 (Results) — MUST INCLUDE ALL FIGURES AND TABLES

**CRITICAL**: This section must include ALL figures and tables from the original paper.

**For each result**:
- Report descriptive statistics (M, SD)
- Report inferential statistics (F, t, p, effect sizes, confidence intervals)
- Copy exact statistical values — never round or approximate
- Preserve significance markers (* p < .05, ** p < .01, *** p < .001)
- Include all figures (bar charts, box plots, path diagrams, etc.)
- Include all tables (correlation matrices, ANOVA tables, regression tables)
- Present results in same order as original paper
- When a table is too large for one slide, split it across slides with continuation indicators

**Statistical reporting format**:
```
F(df1, df2) = xx.xx, p = .xxx, partial eta-squared = .xx
t(df) = x.xx, p = .xxx
coefficient = .xxx, 95% CI = [xxx, xxx]
```

**Hypothesis support indication**: Use simple inline text, NOT visual badges:
- "假设1得到支持" or "假设1未获支持"
- NOT colored SUPPORTED/REJECTED labels

#### 6f. 结论 (Study Conclusion)
- Brief summary of key findings for this specific study
- Whether each hypothesis was supported or not (plain text)
- 1-2 sentences connecting to broader implications

### 7. 总讨论 (General Discussion)

**Purpose**: Synthesize findings across studies, connect to theory, discuss implications.

**Content rules**:
- Translate the general discussion section of the paper
- Summarize main findings and their theoretical significance
- Connect findings back to the theoretical framework
- Discuss practical/applied implications
- Use 2-4 slides, splitting if content is long

### 8. 结论 (Conclusions)

**Content rules**:
- Key take-home message from the authors
- Theoretical contributions
- Practical recommendations
- Keep concise — 1-2 slides maximum

### 9. 局限性与未来展望 (Limitations and Future Directions) — Optional

- Include if the paper has a limitations section
- Sample limitations, methodological constraints
- Suggestions for future research

### 10. 谢谢 (Thank You)

- Simple closing slide with "谢谢！"

## Citation Handling Rules

1. **Preserve original format**: All citations remain in (Author, Year) format
2. **Do not translate author names**: "Kaplan, 2001" stays as "Kaplan, 2001"
3. **Multiple authors**: Preserve ampersand format: "(Hartig & Staats, 2003)"
4. **Multiple citations**: Preserve semicolon separation: "(Kaplan, 2001; Hartig, 2012)"
5. **Et al. handling**: If original uses "et al.", preserve it: "(Staats et al., 2016)"
6. **Never fabricate citations**: Only include citations that appear in the original paper

## Content Fidelity Rules

1. **Translation only**: Convert English text to Chinese. Do not add, remove, or re-interpret content.
2. **No invention**: Never invent hypotheses, results, statistical values, or conclusions not present in the original.
3. **Exact statistics**: Report all numbers (p-values, F-values, t-values, R-squared, means, SDs) exactly as in the original.
4. **Complete figures/tables**: Include ALL figures and tables from the original. None may be omitted.
5. **Original emphasis**: If the paper highlights certain findings as key results, preserve that emphasis.
6. **Annotation markers**: If the user adds personal annotations (e.g., "核心结论：xxx"), include them as presented.
