# Scarring vs Non-Scarring Moxibustion Selection

## When to Use
- **使用瘢痕灸**：当病证属于顽固性慢性虚弱性疾病，且患者体质可耐受强刺激时。
- **使用无瘢痕灸**：当需要温和刺激、避免留疤，或用于一般性调理及广泛适应症时。

## When to Use
确保满足以下前提条件：
- 施灸部位皮肤无感染、溃疡、瘢痕或肿瘤；
- 部位不在颜面、心前区或大血管处；
- 排除禁忌人群：孕妇腰骶部和少腹部禁用瘢痕灸；中暑、高血压危象、肺结核咯血者禁灸。

## Execution Steps

### 1. 瘢痕灸（化脓灸）操作
1. 在穴位涂少量大蒜汁以增强刺激；
2. 将小艾炷（约半麦粒大小）直接置于穴位上点燃；
3. 艾炷燃尽后立即续接新艾炷，共灸5～10壮；
4. 若患者感灼痛，术者可在穴位周围轻拍或抓挠以分散注意力；
5. 灸后约一周出现无菌性化脓（灸疮），4周后结痂脱落并留瘢痕。

### 2. 无瘢痕灸（非化脓灸）操作
1. 在穴位涂少量凡士林；
2. 放置艾炷点燃；
3. 当艾炷燃过半、皮肤潮红并有灼痛感时，立即用镊子移除；
4. 更换新艾炷继续，共灸3～5壮；
5. 此法不引起化脓，不留瘢痕。

### 3. 共同注意事项
- 艾炷大小：大者如半枣粒，小者如半麦粒；
- 每一炷称为“一壮”；
- 灸后若起大疱，需消毒针抽液并涂甲紫防感染。
