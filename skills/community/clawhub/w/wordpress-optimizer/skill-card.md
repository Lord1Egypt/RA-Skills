## Use Case: <br>
WordPress hosting providers, managed service providers, development agencies, and DevOps teams use this skill to analyze WordPress sites, apply optimization routines, tune performance settings, and receive configuration recommendations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill asks for WordPress administrative credentials or tokens. <br>
Mitigation: Use staging first when possible, provide a temporary scoped account or revocable token, and revoke credentials after use. <br>
Risk: Optimization and tuning actions can make broad changes to a live WordPress site without detailed rollback information. <br>
Mitigation: Take a backup, request a clear list of proposed changes before applying them, and review results before using the changes in production. <br>
