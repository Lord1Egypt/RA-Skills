# 措施项目费计算规则（山东安装工程）

> **适用标准**：
> - GB/T 50500-2024《建设工程工程量清单计价标准》
> - 《山东省建设工程费用项目组成及计算规则》2025年版
> - 《山东省安装工程消耗量定额（2025版）》
>
> **对标对象**：广联达GCCP5.0"措施项目"模块

---

## 一、措施项目分类体系

广联达将措施项目分为两大类，cnsdoce1技能应遵循此分类：

```
措施项目费
├── 单价措施项目（可计量，套定额计算）
│   ├── 脚手架搭拆费
│   ├── 垂直运输费
│   ├── 超高施工增加费
│   ├── 大型机械设备进出场及安拆费
│   └── 施工排水、降水费
│
└── 总价措施项目（按费率/给定金额计算）
    ├── 安全文明施工费（强制性，不可竞争）
    ├── 夜间施工增加费
    ├── 二次搬运费
    ├── 冬雨季施工增加费
    ├── 已完工程及设备保护费
    └── 临时保护设施费
```

---

## 二、安装工程特有措施费

安装工程除通用措施外，还有以下**专业特有措施费**（广联达单独列项）：

### 2.1 安装与生产同时进行增加费

| 项目 | 内容 |
|------|------|
| 计取条件 | 在设备安装的同时，生产（使用）系统仍在运行 |
| 计算基数 | 人工费 |
| 费率 | **10%** |
| 包含内容 | 管理费 + 利润（已含在10%内） |
| 是否再计取管理费利润 | **否** |

> ⚠️ 注意：此费率已包含管理费和利润，不得重复计算。

### 2.2 在有害身体健康环境中施工增加费

| 项目 | 内容 |
|------|------|
| 计取条件 | 在有害气体、粉尘、高温、高湿、噪声等有害环境中施工 |
| 计算基数 | 人工费 |
| 费率 | **26%** |
| 包含内容 | 管理费 + 利润（已含在26%内） |
| 是否再计取管理费利润 | **否** |

> ⚠️ 注意：此费率已包含管理费和利润，不得重复计算。

### 2.3 高层建筑增加费

| 建筑物高度/层数 | 增加费率（人工费%） |
|:-------------:|:-----------------:|
| 6层或20m以上 | 2% |
| 8层或30m以上 | 3% |
| 12层或40m以上 | 5% |
| 20层或70m以上 | 8% |
| 30层或100m以上 | 12% |

| 项目 | 内容 |
|------|------|
| 计算基数 | 人工费 |
| 包含内容 | 管理费 + 利润（已含在费率内） |
| 是否再计取管理费利润 | **否** |
| 计算范围 | 主体工程的人工费，不包括脚手架、垂直运输等措施项目 |

## 三、总价措施项目费（通用）

### 3.1 安全文明施工费（强制性）

| 属性 | 说明 |
|------|------|
| 性质 | **不可竞争费用**，投标时必须按标准费率计取，不得下浮 |
| 计算基数 | 分部分项工程费 + 单价措施项目费（不含设备费） |
| 费率 | 按山东省2025版费用规则执行，安装工程一般为 **3%~5%**（以最新文件为准） |
| 包含内容 | 环境保护费、文明施工费、安全施工费、临时设施费 |

> ⚠️ **重要**：安全文明施工费费率由省建设行政主管部门发布，不得自行调整。最新费率需查阅山东省住建厅发布的年度费率文件。

### 3.2 夜间施工增加费

| 项目 | 内容 |
|------|------|
| 计取条件 | 因施工条件或工期要求，在夜间（22:00~次日6:00）施工 |
| 计算基数 | 人工费 |
| 费率 | 一般为 **2%~4%**（按合同约定或山东省规定） |

### 3.3 二次搬运费

| 项目 | 内容 |
|------|------|
| 计取条件 | 因施工场地条件限制，材料、构配件需进行二次搬运 |
| 计算基数 | 人工费 |
| 费率 | 一般为 **1%~2%** |

### 3.4 冬雨季施工增加费

| 项目 | 内容 |
|------|------|
| 计取条件 | 在冬季或雨季施工期间 |
| 计算基数 | 人工费 |
| 费率 | 一般为 **1%~3%** |

### 3.5 已完工程及设备保护费

| 项目 | 内容 |
|------|------|
| 计取条件 | 竣工验收前，对已完工程和设备采取保护措施 |
| 计算基数 | 按实际发生计取，或按人工费的 **0.5%~1%** |

## 四、单价措施项目费（可计量措施）

### 4.1 脚手架搭拆费

安装工程的脚手架搭拆费，广联达处理方式：

| 工程类型 | 处理方式 |
|---------|---------|
| 电气安装 | 套用第4册相关脚手架定额子目 |
| 给排水/采暖/燃气 | 套用第8册/第10册相关脚手架定额子目 |
| 通风空调 | 套用第7册相关脚手架定额子目 |
| 工业管道 | 套用第8册相关脚手架定额子目 |
| 刷油/绝热/防腐 | 套用第12册脚手架定额子目 |

> 安装工程脚手架搭拆费一般按**人工费的5%~8%**计取，或按定额规定执行。

### 4.2 超高施工增加费

| 项目 | 内容 |
|------|------|
| 计取条件 | 操作物高度超过定额规定（一般为3.6m或5m） |
| 计算方式 | 按定额规定的超高系数调整人工费和机械费 |
| 注意 | 安装工程各册定额有各自的超高规定，需查阅各册说明 |

### 4.3 大型机械设备进出场及安拆费

| 项目 | 内容 |
|------|------|
| 计取条件 | 使用大型施工机械（如塔吊、施工电梯、大型起重机等） |
| 计算方式 | 按定额或实际发生计取 |
| 注意 | 安装工程中大型机械使用相对较少，一般按实际签证计取 |

## 五、措施项目费汇总计算示例

```
【项目概况】某高层办公楼安装工程，20层，人工费合计 500,000 元

一、分部分项工程费
   人工费 = 500,000 元
   材料费 = 800,000 元
   机械费 = 100,000 元
   基价 = 1,400,000 元
   管理费 = 500,000 × 48.08% = 240,400 元
   利润 = (500,000 + 240,400) × 29.15% = 216,117 元
   分部分项工程费 = 1,400,000 + 240,400 + 216,117 = 1,856,517 元

二、措施项目费
   1. 安全文明施工费（假设费率4%）
      = 1,856,517 × 4% = 74,261 元
      
   2. 夜间施工增加费（假设2%）
      = 500,000 × 2% = 10,000 元
      
   3. 高层建筑增加费（20层/70m以上，8%）
      = 500,000 × 8% = 40,000 元
      （已含管理费、利润，不再另计）
      
   4. 脚手架搭拆费（假设6%）
      = 500,000 × 6% = 30,000 元
      
   措施项目费合计 = 74,261 + 10,000 + 40,000 + 30,000 = 154,261 元

三、其他项目费
   暂列金额 = 50,000 元（按招标文件）
   
四、税前造价
   = 1,856,517 + 154,261 + 50,000 = 2,060,778 元
   
五、税金（9%）
   = 2,060,778 × 9% = 185,470 元
   
六、工程造价
   = 2,060,778 + 185,470 = 2,246,248 元
```

## 六、cnsdoce1技能中措施项目费的实现建议

### 6.1 数据库存储方案（quota_reference.db）

在 `quota_reference.db` 的 `fee_rates` 表中增加措施项目费率记录：

| fee_type | rate | base_item | note |
|----------|------|-----------|------|
| `management` | 0.4808 | labor | 管理费 |
| `profit` | 0.2915 | labor+management | 利润 |
| `tax` | 0.09 | pre_tax | 税金 |
| `high_rise_6f` | 0.02 | labor | 高层增加费6层 |
| `high_rise_8f` | 0.03 | labor | 高层增加费8层 |
| `high_rise_12f` | 0.05 | labor | 高层增加费12层 |
| `high_rise_20f` | 0.08 | labor | 高层增加费20层 |
| `high_rise_30f` | 0.12 | labor | 高层增加费30层 |
| `concurrent_work` | 0.10 | labor | 安装与生产同时进行 |
| `hazard_env` | 0.26 | labor | 有害环境施工 |
| `safety_civilized` | 0.04 | subtotal | 安全文明施工费（示例） |

### 6.2 脚本调用逻辑

```python
# 伪代码：措施项目费计算

def calc_measure_items(labor_cost, building_height, has_concurrent_work, has_hazard_env):
    measures = {}
    
    # 高层建筑增加费
    if building_height >= 30:  # 30层或100m
        measures['high_rise'] = labor_cost * 0.12
    elif building_height >= 20:  # 20层或70m
        measures['high_rise'] = labor_cost * 0.08
    elif building_height >= 12:
        measures['high_rise'] = labor_cost * 0.05
    elif building_height >= 8:
        measures['high_rise'] = labor_cost * 0.03
    elif building_height >= 6:
        measures['high_rise'] = labor_cost * 0.02
    
    # 安装与生产同时进行
    if has_concurrent_work:
        measures['concurrent'] = labor_cost * 0.10
    
    # 有害环境施工
    if has_hazard_env:
        measures['hazard'] = labor_cost * 0.26
    
    return measures
```

## 七、与fee_standard.md的衔接

本文件专用于**措施项目费**，与 `fee_standard.md`（取费标准）的关系：

| 文件 | 内容 | 用途 |
|------|------|------|
| fee_standard.md | 管理费、利润、税金、高层增加费等**通用费率** | 任何组价都必须引用 |
| cost_composition.md | 费用构成框架、计价程序流程 | 理解整体计价逻辑 |
| measure_items.md | 措施项目费的**分类、计取条件、费率** | 计算措施项目费时引用 |

三者的调用关系：
```
组价计算流程
    │
    ├──> fee_standard.md → 管理费、利润、税金
    │
    ├──> cost_composition.md → 费用构成、计价程序
    │
    └──> measure_items.md → 措施项目费分类与计算
```
