## Description: <br>
Guides an agent in using the agent-browser CLI to automate browser navigation, clicks, form entry, screenshots, and page information capture. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent operators use this skill when a task requires browser automation, including sign-ins or check-ins, form filling, button clicks, screenshots, and extracting web page content for research material. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can guide browser clicks, form fills, submissions, screenshots, scraping, and scheduled check-ins, which may affect accounts or capture private page content. <br>
Mitigation: Verify the target pages and npm package before use, review pages before submit or check-in actions, avoid storing real passwords in scripts, and avoid screenshots or scraping on private pages unless the captured content is approved. <br>
