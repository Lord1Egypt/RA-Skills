# 学术阅读提示词集合

## 1. 章节分析提示词

```
请用以下结构分析这本书的第X章：

## 核心概念
- 列出本章3-5个最重要的概念
- 用一句话解释每个概念

## 理论要点
- 本章的理论框架是什么？
- 主要结论和论据是什么？

## 与个人进化的融合
- 这个理论如何应用于我的工作/学习？
- 有什么可以立即实践的行动？

## 批判性思考
- 这个理论的局限性是什么？
- 有什么问题作者没有回答？
```

## 2. 深度反思提示词

```
请基于这本书的内容，进行深度批判性思考：

1. 这个理论/方法论的核心假设是什么？
2. 这些假设在什么情况下可能不成立？
3. 如果把这个理论应用到我的领域，可能有什么问题？
4. 有什么是作者可能遗漏的重要维度？
5. 这个理论和我已有的知识有什么冲突或补充？

请详细分析并给出你的独立判断。
```

## 3. 执行计划生成提示词

```
基于这本书的理论，请帮我生成一个执行计划：

1. 理论中提到的核心方法是什么？
2. 我如何在接下来的一周内开始实践？
3. 我如何衡量自己是否在进步？
4. 可能的障碍是什么？如何克服？

请给出具体的、可执行的行动步骤。
```

## 4. 快速摘要提示词

```
请用300字概括这本书的核心论点：
- 解决的问题
- 提出的方法
- 关键洞见
- 对我的意义
```
