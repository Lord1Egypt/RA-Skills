# 审增减分类规则详解

## 7大类别

| ID | 类别名 | 判断依据 | 典型场景 |
|----|--------|---------|---------|
| 1 | 重复计取审减 | 同一工作内容在多个清单项中重复计算 | 墙砖下的水泥砂浆既作为独立项又含在面层定额中 |
| 2 | 漏项审增 | 图纸要求有但送审清单遗漏的工作 | 电缆头、接线端子、桥架支架等小项未列 |
| 3 | 工程量审增减 | 工程量算多或算少 | 电缆长度量差、管线长度核对、设备数量差异 |
| 4 | 定额子目审增减 | 套用的定额子目与工作内容不匹配 | 应套一般钢结构定额却套了H型钢定额 |
| 5 | 材料价格审增减 | 材料/设备单价偏离市场或信息价 | 高压柜、电缆、钢板等主材单价调整 |
| 6 | 其他 | 无法归入前5类的调整 | 零星项、计算规则差异等 |
| 7 | 基数调整审增减 | 随计费基数变化而调整的措施费/税金 | 因审减后分部分项工程费减少，同步下调安文费、税金 |

## 自动分类优先级

脚本按以下顺序匹配 `增减说明` + `增减原因` 组合文本：

1. **基数调整** → 匹配：`基数|税金|措施费|总价措施|安文|环境保护费|文明施工费|临时设施|扬尘|夜间施工|二次搬运|冬雨季`
2. **重复计取** → 匹配：`重复`
3. **定额子目** → 匹配：`定额|子目|套用|调整为|人工乘`
4. **材料价格** → 匹配：`价格|材料|信息价|市场询价|材料价格`
5. **漏项** → `[增项]` 或 `漏项|补项|未报送`
6. **工程量** → `[调量]` / `[调量,调价]` / `[减项]`
7. **[调价] / [调项]** → 默认归入**材料价格**
8. **兜底** → 其他

## 常见误分类场景

### [调价] → 材料价格 vs 定额子目

纯 `[调价]` 项默认归入材料价格，但以下情况应为定额子目：
- 增减原因提到"定额调整为"、"子目调整"、"套用XX定额"
- 项目特征变化导致定额子目变更（非单纯材料价差）

**复核方法**：查看 `增减原因` 列是否含"定额"或"子目"关键词。

### [调量,调价] → 工程量

工程量同时变化时归入"工程量"类别。但若主要原因是材料替换或定额调整，应人工改为材料价格或定额子目。

### 措施费/税金未命中"基数"关键词

如果 `增减原因` 描述为"税金调整"、"措施费调整"但未包含"基数"字眼，会被兜底归入"其他"。

**复核方法**：搜索输出文件中金额较大的"其他"类条目，检查是否为措施/税金调整。

## 复核清单生成

复核时可运行脚本查看分类摘要，关注：
1. "其他"类中金额 > 1万元的条目
2. "材料价格"类中增减原因含"定额"字眼的条目
3. 各类别合计是否与总审减金额勾稽一致
