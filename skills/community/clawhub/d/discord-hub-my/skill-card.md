## Use Case: <br>
Developers and engineers use this skill to plan Discord bot workflows, design commands and interactions, and prepare direct HTTPS requests for Discord API operations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The runnable scripts can post messages to an external Discord webhook. <br>
Mitigation: Use only a webhook you control and confirm the destination and message content before execution. <br>
Risk: The run script loads a local .env file broadly before sending a message. <br>
Mitigation: Keep unrelated secrets out of the skill's .env file and review or replace the loader before running it. <br>
