## 核心理念

**放弃单图思维 + 放弃 MJ 两阶段。** 使用 Gemini 文生图模型，图片上的中文文字在 prompt 中直接指定，一步出图。本指南所有设计基于 5-12 张 Carousel 轮播图，覆盖完整的「点击→阅读→收藏→评论→关注」转化链路。

---

## 第一部分：Carousel 5图结构

### 标准轮播结构

```
图1：封面图      → 大字标题 + 年龄徽章 + 钩子           → 点击率
图2：核心内容图   → 主要干货，bento-grid/flow 布局        → 完读率
图3：补充展开图   → 细节清单、里程碑对照                 → 收藏动机
图4：误区对比图   → 正确 ✓ vs 错误 ✗ 左右对比            → 评论动机
 （可选扩展：分步详解 / 场景案例 / 清单模板 — 接在图4后）
末张：金句收尾图   → 一句戳心励志金句，情绪释放            → 共鸣转发（始终在最末尾）
```

### 各图定位详解

| 图位 | 目的 | 内容类型 | 视觉策略 | 文字量 |
|------|------|----------|----------|--------|
| 封面图 | 让人停下来点进去 | 标题+年龄段+钩子 | 大标题占50-60%版面，留白多，颜色对比强 | ≤20字 |
| 核心内容图 | 提供核心价值 | 5类能力/bento-grid | 圆角卡片排列，每卡左边emoji+右边文字 | ≤50字（分块） |
| 补充展开图 | 加深信任 | 发育里程碑/数据 | 与图2风格统一但布局有明显差异（列表/时间轴） | ≤60字（分块） |
| 误区对比图 | 制造讨论 | 正确✓ vs 错误✗ 对比 | 绿色区（对）vs 红色区（错），视觉冲击 | ≤40字 |
| 金句收尾图（末张） | 共鸣转发 | 一句戳心金句 | 大量留白，温暖留白，居中金句 | ≤25字 |

## 第二部分：布局类型（在 prompt 中如何描述）

### 中文 prompt 中的布局描述方式

不再使用 MJ 的英文关键词（`bento-grid`、`vertical-flow`），改用**中文场景化描述**：

| 布局 | Prompt 中的中文描述方式 |
|------|------------------------|
| **bento-grid** | 「5个圆角卡片纵向排列，每个卡片左侧一个大号emoji，右侧排列小标题和两行说明文字，卡片之间间距均匀有柔和阴影」 |
| **timeline** | 「一条纵向时间轴从顶部延伸到底部，每个时间节点左侧是时钟图标+时间，右侧是圆角卡片内容，节点之间用装饰虚线连接」 |
| **vertical-flow** | 「自上而下的流程图，3-4个步骤用向下的弧形箭头连接，每个步骤在圆角框内，左上角有步骤编号徽章」 |
| **numbered-list** | 「大号数字1到5纵向排列，每个数字在彩色圆形徽章中，右侧配小标题和说明文字，行距宽松」 |
| **comparison** | 「左右两栏对比布局，左栏浅绿色底（正确✓），右栏浅红色底（错误✗），中间有柔和分隔线，每栏3条内容」 |
| **checklist** | 「纵向清单布局，每行左边是方形复选框+emoji图标，右边是简短操作短语，行与行之间有浅灰色分割线」 |

## 第五部分：Carousel 视觉节奏

```
图1 (封面) ：视觉冲击最强 —— 大标题 + 大量留白 + 年龄徽章
    ↓ 滑动
图2 (核心) ：信息密度适中 —— 5个卡片纵向排列，结构工整
    ↓ 滑动  
图3 (补充) ：信息密度最高 —— 列表/时间轴，布局与前图不同（防疲劳）
    ↓ 滑动
图4 (误区) ：视觉冲击回升 —— 绿色✓ vs 红色✗ 对比，色彩反差大
    ↓ 滑动
 （可选扩展：分步详解/场景案例/清单模板，信息密度持续提供）
    ↓ 滑动
末张 (金句) ：视觉回到极简 —— 大量留白 + 居中金句 + 温暖心形/小花（始终在末尾收尾）
```

### 避免 Carousel 常见错误

❌ 5张图看起来一模一样（用户以为只有1张，跳过）
❌ 图2信息量比图1还少（点进来失望，滑走）
❌ 图之间配色跳变太大（视觉不连贯，取关）
❌ 最后一张突然变风格（信任感断裂）
✅ 每张图有独立的视觉钩子但整体色调/风格统一
✅ 封面主打视觉冲击，图2-3信息密度，图4对比冲击，末张始终金句收尾留白
✅ 最后一张用心形/缩略图等温暖元素提醒「这是最后一张了」

## 第六部分：5图通用元素规范

每张内容图（图2-图4）必须包含：

| 元素 | 位置 | 样式规范 |
|------|------|----------|
| **年龄段徽章** | 顶部，圆角气泡/标签 | 「适用年龄：3-4岁」，浅绿色底+深绿字，醒目但不大过标题 |
| **主标题/章节标题** | 年龄段下方 | 中号圆体粗字，深灰色，与封面标题形成层级关系 |
| **内容区块** | 画面主体 | 圆角卡片，有柔和阴影，色彩按区块功能区分 |
| **底部装饰线/金句** | 画面底部10%区域 | 不抢眼，点缀性装饰或一句金句 |

## 第七部分：知识挖掘与年龄段适配

| 年龄段 | 知识深度 | 家长关注点 | 图中呈现方式 |
|--------|----------|-----------|-------------|
| 0-3个月 | 基础护理 | 喂养/睡眠/黄疸 | 简洁流程图，文字极少 |
| 4-11个月 | 中等 | 辅食/大运动/分离焦虑 | 时间轴+清单 |
| 1-2岁 | 进阶 | 语言/社交/如厕训练 | 对比图+步骤详解 |
| 2-3岁 | 进阶 | 规则建立/情绪管理 | 场景化+案例 |
| 3-6岁 | 深度 | 学习启蒙/社交/幼小衔接 | 系统化+数据 |
| 6-8岁 | 深度 | 学习习惯/专注力/阅读启蒙 | 清单+方法卡+计划表 |
| 8-10岁 | 深度 | 学习方法/时间管理/兴趣培养 | 图表+对比+进阶路径 |
| 10-12岁 | 深度 | 青春期预备/自主学习/小升初衔接 | 策略图+流程+资源包 |

✅ 好知识（有数字、有步骤、可操作）：
> 「38.5°C以上且精神状态差才用退烧药，对乙酰氨基酚间隔≥4h，布洛芬间隔≥6h，24h内不超4次」

❌ 差知识（空洞无物）：
> 「发烧要多喝水，注意观察体温」

## 第八部分：情绪锚点

为每个选题挖掘1-2个情绪锚点，贯穿封面文案 → 正文钩子 → 金句收尾图：

| 情绪锚点 | 对应设计 | 示例 |
|----------|----------|------|
| 崩溃/焦虑 | 封面用「你懂我」感 | 「凌晨3点第5次量体温」 |
| 被理解 | 正文钩子用场景共鸣 | 「你是不是也经历过...」 |
| 有底气了 | 干货用「照着做就行」 | 「打印贴冰箱，不用再纠结」 |
| 原来这么简单 | 对比展示「别人vs你」 | 「就3步，比想象的简单100倍」 |
| 必须收藏 | 信息密度足够大 | 「花了3天整理，全网最全」 |

## 第九部分：布局风格自检清单

- [ ] 是否选择了 Carousel 5图方案（封面+核心+补充+误区+引导）？
- [ ] 封面图是否为大标题+钩子+留白型（非信息堆砌）？
- [ ] 每张图是否有独立的视觉价值和信息增量？
- [ ] 图之间配色/风格是否统一（同一调色盘）？
- [ ] 图片比例是否统一为 3:4 竖版？
- [ ] 风格是否与选题类型和年龄段匹配？
- [ ] 知识是否适配目标年龄段（深度、表述方式）？
- [ ] 情绪锚点是否在封面/正文/引导图中贯穿？
- [ ] 图4是否有正确✓ vs 错误✗ 对比设计？
- [ ] 金句收尾图是否在末张？
- [ ] 金句收尾图是否是一句戳心金句 + 温暖留白（非关注引导/广告）？
- [ ] 所有中文文字在 prompt 中是否用「」标记？
- [ ] 每个区块的文字位置、颜色、字号是否精确描述？
