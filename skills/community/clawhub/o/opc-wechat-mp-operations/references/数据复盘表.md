# 数据复盘表

> 适用定位：技术成果转化、科技创新政策解读  
> 适用场景：单篇文章复盘、周度/月度数据分析

---

## 一、数据指标体系

### 1.1 核心指标定义

| 指标名称 | 定义 | 计算公式 | 行业参考值 |
|----------|------|----------|------------|
| **送达人数** | 文章推送后送达的关注者数量 | - | 取决于粉丝基数 |
| **阅读人数** | 点击文章的人数（去重） | - | - |
| **阅读次数** | 文章被阅读的总次数（含重复） | - | 建议>阅读人数的1.1倍 |
| **打开率** | 阅读人数/送达人数×100% | 5% | 3-8%为正常 |
| **完读率** | 阅读完整篇文章的比例 | 目测估算 | 40-60%为合格 |
| **分享人数** | 分享到朋友圈/好友的人数 | - | - |
| **分享率** | 分享人数/阅读人数×100% | 3% | 1-5%为正常 |
| **在看数** | 点击在看/收藏的人数 | - | 阅读人数×1-3% |
| **新增关注** | 文章带来的新关注者数量 | - | - |
| **取关数** | 文章期间取消关注的人数 | - | 需关注变化趋势 |
| **留言数** | 主动留言的人数 | - | - |
| **留言率** | 留言人数/阅读人数×100% | 1% | 0.5-2%为正常 |

### 1.2 数据获取路径

```
公众号后台 → 内容与互动 → 内容分析 → 选择文章
```

可获取数据：
- 阅读量趋势
- 流量来源（搜一搜/看一看/朋友圈/朋友在看/其他）
- 关注趋势（新增/取关）
- 留言/在看数据

## 四、问题诊断与分析

### 4.1 数据异常诊断

| 问题现象 | 可能原因 | 验证方法 | 解决方案 |
|----------|----------|----------|----------|
| 打开率过低（<3%） | 标题吸引力不足 | 对比同阅读量文章的标题 | 优化标题公式 |
| 打开率过低（<3%） | 发布时间不当 | 检查推送时间 | 调整发布时间 |
| 打开率过低（<3%） | 粉丝活跃度下降 | 查看取关趋势 | 提升内容质量 |
| 分享率过低（<1%） | 内容缺乏共鸣点 | 检查是否有传播钩子 | 增加互动引导 |
| 分享率过低（<1%） | 话题专业性过强 | 评估选题适配性 | 调整选题方向 |
| 留言率过低（<0.5%） | 缺少互动引导 | 检查结尾是否有互动 | 增加互动话题 |
| 取关率异常升高 | 内容与粉丝期待不符 | 分析取关时间节点 | 优化内容定位 |
| 搜一搜流量为0 | 关键词未布局 | 检查标题/正文关键词 | SEO优化 |
