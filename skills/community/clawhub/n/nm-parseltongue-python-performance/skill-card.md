## Known Risks and Mitigations: <br>
Risk: Profiling live Python processes can expose runtime behavior or sensitive workload details. <br>
Mitigation: Use profiling commands only on approved processes and review generated profile files before sharing them. <br>
Risk: Suggested profiling tools may require package installation or local process access. <br>
Mitigation: Review package installs and run profiling commands in controlled environments with the least access needed. <br>
