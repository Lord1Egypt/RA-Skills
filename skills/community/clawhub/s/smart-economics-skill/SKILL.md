# Skill: 聪明生活经济学

> 经济学不是学问，是武器。每天一个概念，变成下意识的思考方式。

---

## 触发场景

### 1. 定时触发（每日推送）
- 安装此 Skill 时，自动创建一个每天 08:00 的日程任务
- 每天从 `daily-queue.md` 按顺序取一条概念推送
- 推完后循环回第一条重新开始

### 2. 对话关联（主动解读）
当用户对话涉及以下场景时，主动用经济学概念重新解读：

| 用户场景 | 关联概念 |
|---------|---------|
| 买不买 / 已经花了钱舍不得放弃 | 沉没成本 |
| 选哪个 / 做不做 / 时间怎么分配 | 机会成本、比较优势 |
| 要不要跳槽 / 换工作 | 机会成本、信号传递 |
| 定价 / 收费 / 买东西觉得贵或便宜 | 价格歧视、锚定效应、消费者剩余 |
| 时间分配 / 该自己做还是外包 | 比较优势、边际效用 |
| 值不值 / 该不该 / 选A还是B | 边际效用、机会成本、沉没成本 |
| 存钱还是花钱 / 投资还是消费 | 时间折现、复利、心理账户 |
| 副业 / 创业 / 增加收入 | 比较优势、规模化收入 |
| 还债 / 负债 / 信用卡 | 债务雪球、现金流管理 |
| 想买很多东西 / 冲动消费 | 需求分层、边际效用递减 |

关联方式：
> "你说的这个，用经济学的 **XX概念** 来看就是——[重新解读]"

注意：不是硬蹭，是真能帮用户换个角度看问题。

### 3. 主动查询（深度解释）
用户直接问某个经济学概念时，展开完整解释：
- 概念定义 → 背后原理 → 生活场景(2-3个) → 课程案例 → 金句

## 功能1：每日定时推送

### 推送格式
```
💡 今日经济学 · 【概念名】

一句话：XXX

生活场景：XXX

金句：XXX
```

### 示例
```
💡 今日经济学 · 【需求分层】

一句话：把钱花在需求和热爱上，你被生活掌控；花在喜欢和想要上，你在掌控生活。

生活场景：月底发现信用卡又刷爆了，但想不起买了什么——那些"手滑下单"的东西，大多只是"喜欢"，不是"热爱"。

金句："在需求和爱上花钱，是你在掌控生活；当你关注喜欢和想要，你在被生活掌控。" ——第2课
```

## 功能2：对话主动关联

### 判断逻辑
当用户输入包含以下关键词或语义时触发：

- **沉没成本**："已经花了"、"都买了"、"舍不得"、"来都来了"、"不看可惜"、"分手可惜"
- **机会成本**："选哪个"、"去不去"、"做不做"、"时间不够"、"放弃了"、"如果当初"
- **边际效用**："值不值"、"还要不要"、"腻了"、"没感觉了"、"再买一件"
- **比较优势**："自己做还是"、"外包"、"分工"、"谁更适合"、"专长"
- **价格歧视/锚定**："原价"、"打折"、"贵不贵"、"值这个价吗"、"心理价位"
- **需求分层**："想买"、"冲动"、"需要吗"、"该不该买"、"控制不了手"
- **时间折现**："现在就要"、"等不及"、"以后再说"、"即时满足"
- **复利/投资**："存钱"、"理财"、"投资"、"退休"、"财务自由"
- **副业/开源**："副业"、"兼职"、"创业"、"多赚点"、"技能变现"
- **债务管理**："还债"、"信用卡"、"贷款"、"负债"、"利息"

### 回应模板
```
你说的这个，用经济学的【概念名】来看就是——

[一句话解释概念]

放到你的情况里：
[重新解读用户的具体场景]

[给出一个可操作的判断标准或行动建议]
```

```
**原理**：背后的经济学理论（1-2段，口语化）

**生活场景**：
1. [具体场景1]
2. [具体场景2]
3. [具体场景3]

**课程案例**：
[来自纪录片的具体人物故事]

**金句**：
"[课程原文]" ——第X课

**模块**：[所属模块]
```

## 核心原则

1. **口语化中文，不要学术腔**——像朋友聊天一样解释
2. **概念必须配生活场景**——不能只给定义
3. **金句保留课程原文**——标注第几课
4. **不是硬蹭，是真能帮用户换个角度看问题**
5. **经济学不是学问，是武器**——每天一个概念，变成下意识的思考方式
