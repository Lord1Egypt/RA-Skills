# 评估青春期发育阶段与进程

## 何时使用
- 女孩年龄在8～13岁之间，且出现任何第二性征（如乳房萌发、阴毛生长）
- 需要判断当前发育阶段是否符合正常时序
- 需预测月经初潮或其他后续发育事件的时间窗
- 怀疑性早熟（<8岁出现性征）或青春期延迟（>13岁无乳房发育）

## 执行步骤
1. **确认起始事件**：识别最早出现的第二性征，通常为乳房萌发（thelarche）。
2. **按四阶段顺序评估当前所处阶段**：
   - **阶段1：乳房萌发**（约10岁开始，历时约3.5年）
   - **阶段2：肾上腺功能初现**（阴毛先于腋毛，伴随雄激素升高）
   - **阶段3：生长加速**（11～12岁，年均增长约9cm）
   - **阶段4：月经初潮**（平均晚于乳房发育2.5年）
3. **验证关键时序逻辑**：确保发育顺序为“乳房发育 → 阴毛出现 → 生长突增 → 月经初潮”。
4. **判断异常**：
   - 若任一性征在<8岁出现，提示性早熟
   - 若>13岁仍无乳房发育，提示青春期延迟
5. **预测下一阶段**：基于当前阶段和已知间隔（如乳房发育至初潮平均2.5年），估算后续事件时间窗。
6. **综合心理变化**：同步关注性意识萌发、情绪波动等心理发育表现。

> 注意：初潮后5～7年内月经多不规律，属正常现象，不代表排卵障碍。
