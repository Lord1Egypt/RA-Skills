# Outreach Quality Checklist

Run every batch against this list before sending. A draft failing any item in Categories 1–3 must be revised.

## 1. Opener specificity
- [ ] First sentence references something only this creator does (post, series, format)
- [ ] Reference is recent (within ~60 days) — not their pinned video from 2 years ago
- [ ] Opener would FAIL if pasted into a message to a different creator
- [ ] Creator's name spelled exactly as displayed on their profile
- [ ] No generic flattery ("love your content", "huge fan")

## 2. Earning math
- [ ] Per-sale dollar amount stated explicitly
- [ ] Weekly/per-video scenario uses conversion within the realistic band for their tier
- [ ] Low-end figure used, not best case
- [ ] No "guaranteed", "passive income", or "easy money" phrasing
- [ ] Numbers trace to stated AOV × commission × conversion

## 3. Call to action
- [ ] Exactly one CTA in the message
- [ ] CTA is the lowest-friction available step (reply trigger or free-sample yes)
- [ ] No first-message links in TikTok/Instagram DMs
- [ ] Email contains at most one link
- [ ] CTA sentence works standalone

## 4. Channel fit
- [ ] TikTok DM 120–160 words
- [ ] Email 200–260 words with ≤9-word subject
- [ ] Marketplace pitch 80–120 words
- [ ] Register matches channel (casual DM / structured email / fit-first marketplace)
- [ ] Inbox-preview first line contains the personal reference

## 5. Audience fit & honesty
- [ ] One sentence explicitly connects product to creator's audience segment
- [ ] Target market/shipping countries stated so mismatched creators self-filter
- [ ] Unique selling hook included if provided
- [ ] No claims about the product that the brand can't substantiate
- [ ] Disclosure expectation (FTC/platform) covered in FAQ

## 6. FAQ block
- [ ] All 5 items present: sampling, exclusivity, payment, approval, link setup
- [ ] Each answer 1–2 sentences
- [ ] Payment schedule and method specific (not "fast payouts")

## 7. Follow-up plan
- [ ] Exactly one follow-up scheduled at 3–4 days
- [ ] Follow-up leads with a DIFFERENT angle than the original
- [ ] No second follow-up planned
- [ ] Lapsed-affiliate messages acknowledge past performance with real numbers
- [ ] Re-engagement offers something genuinely new

## 8. Batch hygiene
- [ ] No two creators in the same niche receive identical openers
- [ ] Send volume rate-limited (≤20 DMs/session per platform account)
- [ ] Tone consistent with requested brand voice across all variants
- [ ] All bracketed placeholders removed
- [ ] Names/handles cross-checked against the target list
