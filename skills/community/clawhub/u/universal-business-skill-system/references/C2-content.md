# C2 内容生产与传播 — 详细单元规格

## C2-01 核心信息架构(Message House)

| 字段 | 内容 |
|------|------|
| **输入** | 必选：产品/服务核心卖点、目标受众；可选：品牌调性、竞品差异化 |
| **输出** | Message House文档（品牌承诺→3大支柱→证据→话术），Markdown |
| **依赖** | 无 |
| **AI自治度** | 🟨 半自动（需人工确认核心信息和优先级） |
| **组合接口** | →C2-03, →C2-04, →C2-05, →C2-10, →C2-11, →C2-13, →C2-15, →C6-02 |

**执行要点**：
1. 明确品牌核心承诺（一句话价值主张）
2. 拆解3大支撑支柱
3. 为每个支柱匹配证据/数据
4. 衍生不同场景话术
