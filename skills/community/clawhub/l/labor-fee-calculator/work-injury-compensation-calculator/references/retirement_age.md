# 法定退休年龄计算（渐进式延迟退休，2025年起施行）

> 法律依据：《全国人民代表大会常务委员会关于实施渐进式延迟法定退休年龄的决定》（2024年）
> 施行时间：2025年1月1日起

---

## 重要说明

2025年起中国实施渐进式延迟退休制度。计算工伤赔偿中的"距退休年龄"时，**必须使用以下公式**，不得使用旧的60/55/50岁固定标准。

---

## 一、男性职工

- **原定退休年龄：** 60周岁
- **延迟上限：** 63周岁（即最多延迟3年/36个月）

### 适用情形分类

| 出生时间范围 | 退休年龄 | 计算方法 |
|-----------|---------|---------|
| 1976年9月1日以后 | **63周岁** | 直接按63岁算，无需公式 |
| 1965年1月1日 至 1976年8月31日 | 60岁 + 延迟月数 | 用下方公式计算 |
| 1965年1月1日以前 | **60周岁**（不受影响） | 直接按60岁算 |

### 延迟月数公式（适用1965.1.1-1976.8.31）

```
延迟月数 = CEILING([(出生年份 + 60 - 2025) × 12 + 出生月份] / 4)
（结果保留整数，小数部分不足0.5也进1，即向上取整）
```

**计算示例：**
- 男，1972年3月出生：延迟月数 = CEILING([(1972+60-2025)×12+3]/4) = CEILING([7×12+3]/4) = CEILING(87/4) = CEILING(21.75) = **22个月** → 退休年月：1972年3月 + 60年 + 22个月 = **2034年1月**

---

## 二、女性职工（非管理岗/工人）

- **原定退休年龄：** 50周岁
- **延迟上限：** 55周岁（最多延迟5年/60个月）

| 出生时间范围 | 退休年龄 | 计算方法 |
|-----------|---------|---------|
| 1984年11月1日以后 | **55周岁** | 直接按55岁算 |
| 1975年1月1日 至 1984年10月31日 | 50岁 + 延迟月数 | 用下方公式计算 |
| 1975年1月1日以前 | **50周岁**（不受影响） | 直接按50岁算 |

### 延迟月数公式（适用1975.1.1-1984.10.31）

```
延迟月数 = CEILING([(出生年份 + 50 - 2025) × 12 + 出生月份] / 2)
（同样向上取整）
```

**计算示例：**
- 女（工人），1980年6月出生：延迟月数 = CEILING([(1980+50-2025)×12+6]/2) = CEILING([5×12+6]/2) = CEILING(66/2) = **33个月** → 退休年月：1980年6月 + 50年 + 33个月 = **2033年3月**

---

## 三、女性职工（管理岗/干部）

- **原定退休年龄：** 55周岁
- **延迟上限：** 58周岁（最多延迟3年/36个月）

| 出生时间范围 | 退休年龄 | 计算方法 |
|-----------|---------|---------|
| 1981年9月1日以后 | **58周岁** | 直接按58岁算 |
| 1970年1月1日 至 1981年8月31日 | 55岁 + 延迟月数 | 用下方公式计算 |
| 1970年1月1日以前 | **55周岁**（不受影响） | 直接按55岁算 |

### 延迟月数公式（适用1970.1.1-1981.8.31）

```
延迟月数 = CEILING([(出生年份 + 55 - 2025) × 12 + 出生月份] / 4)
（同样向上取整）
```

## 四、计算"距退休剩余年数"的方法

用于判断离职补助金折减比例：

```
退休年月（精确到月） = 出生年月 + 法定退休年龄（含延迟月数）
距退休剩余月数 = 退休年月 - 计算当月
距退休剩余年数 = 剩余月数 / 12（向上取整，不足1年算1年）
```

**折减规则（大多数省份适用，具体以 `policy_data.leaving_subsidy_standards` 返回为准）：**
- 剩余年数 ≥ 5年 → 全额支付
- 不足5年 → 每少1年，就业补助金递减20%
  - 剩余4-5年：按80%支付
  - 剩余3-4年：按60%支付
  - 剩余2-3年：按40%支付
  - 剩余1-2年：按20%支付
  - 不足1年：按10%支付
- 已达退休年龄 或 符合领取养老金条件 → 支付金额 = 0（《劳动合同法》第38条情形除外）

> 注：重庆市等地区可能存在特殊折减规则，以政策数据 API 返回的地区规则或特殊说明为准。

---

## 五、注意事项

1. 当用户未指明岗位性质（管理岗/工人）时，默认按女工人（50岁原定）计算，并提示用户确认
2. 2025年前已退休或已到原退休年龄者，不受延迟退休政策影响
3. 如对具体人员退休年龄有争议，建议以人社局12333咨询结果为准
