## 查找组件文档

直接读取 `src/组件名/index.zh-CN.md` 获取完整 API 文档。

示例: 查找 DForm 文档
```
Read: src/DForm/index.zh-CN.md
```

示例: 搜索某个 API
```
Grep: "DFormProps" include: "index.zh-CN.md" path: "src"
```
