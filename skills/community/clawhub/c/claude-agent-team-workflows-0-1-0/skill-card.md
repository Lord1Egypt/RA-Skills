## Description: <br>
Universal multi-agent workflow orchestration using Claude Code Agent Teams. Use when user asks to run a team workflow, create an agent team, or coordinate parallel work across multiple teammates - for any domain (software, content, data, strategy, research, etc.). <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[Sieyer](https://clawhub.ai/user/Sieyer) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and other Claude Code users use this skill to coordinate a five-agent team, select a workflow pattern, assign role cards, manage handoffs, and synthesize the final deliverable for software, content, data, strategy, or research work. <br>
