# 正常组织在T₁WI与T₂WI中的信号表现参考表

| 组织       | T₁WI信号 | T₂WI信号 | 说明 |
|------------|----------|----------|------|
| 脑白质     | 较高（白灰） | 中等（灰） | T₁WI比灰质亮，T₂WI比灰质暗 |
| 脑灰质     | 中等（灰） | 较高（白灰） | T₁WI比白质暗，T₂WI比白质亮 |
| 脂肪       | 高（白） | 中等偏高（白灰） | T₁WI最亮组织之一；T₂WI因含少量水分呈中等偏高信号 |
| 脑脊液     | 低（黑） | 高（白） | 典型长T₁、长T₂表现 |
| 肌肉       | 中等偏低（深灰） | 中等偏低（深灰） | T₁WI与T₂WI均呈均匀中低信号 |
| 韧带       | 低（黑） | 低（黑） | 因胶原纤维致密，T₂弛豫极短 |
| 骨皮质     | 极低（黑） | 极低（黑） | 几乎无自由水质子，信号缺失 |
| 红骨髓     | 中等至较高 | 中等 | 含脂肪成分，T₁WI可呈较高信号 |
| 黄骨髓     | 高（白） | 中等 | 富含脂肪，类似皮下脂肪信号 |

> 注：以上为常规1.5T或3T MRI在标准序列下的典型表现。场强、序列参数（如TR/TE）、年龄（如儿童骨髓含水多）等因素可影响实际信号。
