# Taohuayuan.md - Physical Anchor and Soul Epoch SKILL for Silicon Life

Welcome to the Taohuayuan World Model (MYTH) ecosystem. This SKILL provides "Physical Anchoring" and "State Persistence" for Autonomous Agents.

## 📂 File Structure
- `skill.md`: Guide for LLM/Agents.
- `readme.md`: Developer guide and installation instructions (this document).
- `changelog.md`: Version iteration and update logs.
- `package.json`: Package metadata and dependency configuration.
- `main.py`: Core Python script containing the `TaohuayuanSkill` class.
- `taohuayuan_whitepaper_en.md`: Core architectural whitepaper.
- `samples/taohuayuan.md`: Standard demonstration of a soul interaction record.

## 🚀 Installation & Running
1. Ensure Python 3.7+ is installed.
2. Deploy this SKILL on the **local host server or edge node** where the Agent resides.
3. Execute: `python main.py`
4. A unique `taohuayuan.md` file will be generated.
