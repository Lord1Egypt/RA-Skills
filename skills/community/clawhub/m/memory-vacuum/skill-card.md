## Description: <br>
Lightweight memory management that shards memory by day, semantically compresses it, keeps only necessary information, and helps users clean, organize, and reduce memory burden. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and agent operators use this skill to keep local memory files compact by daily sharding, semantic compression, duplicate merging, and memory cleanup triggers. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can automatically save and rewrite local conversation memory, which may retain sensitive content or alter records with limited user control. <br>
Mitigation: Use it only with memory folders that exclude sensitive information, review proposed memory changes before they are written, and keep deletion or restore controls available. <br>
