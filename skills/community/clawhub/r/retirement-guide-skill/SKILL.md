# 中国养老规划助手

基于最新政策（截至2026年6月）的全面养老规划指导技能。

## 核心能力

### 1. 养老现状分析
- 中国老龄化数据：2026年60岁以上约3.2亿，2030年达3.9亿
- 养老金三大支柱体系解读（基本养老保险、企业/职业年金、个人养老金）
- "十五五"（2026-2030）为老龄化最快发展期

### 2. 财务缺口计算
使用 **4%法则**（退休金 = 年支出 × 25）计算退休目标金额：
- 基础生活（月5,000元）→ 需150万元
- 舒适生活（月8,000元）→ 需240万元
- 品质生活（月15,000元）→ 需450万元

需要考虑：
- 社保养老金抵扣（当前月均约3,000-4,000元）
- 通货膨胀修正（2%年通胀，30年后购买力减半）
- 投资收益率对积累速度的影响
- 复利效应：每晚10年开始规划，月投入需翻倍

### 3. 分年龄段行动方案

**30岁阶段：强制储蓄，时间换空间**
- 开立个人养老金账户，每年缴满12,000元（享税优）
- 长缴费期养老年金或养老FOF定投
- 先配健康险（百万医疗+重疾），再谈理财
- 月收入10%-15%强制划入养老账户

**40岁阶段：黄金配置窗口**
- 评估家庭总资产，计算养老资金缺口
- 补足寿险/重疾险/医疗险保障
- 锁定当前利率配置养老年金险
- 夫妻共同规划，避免单方承担
- 考虑养老社区入住资格

**50岁阶段：快速补充，精确测算**
- 精确测算退休后月度支出（含医疗）
- 选择3-5年短期缴费快速返还型产品
- 降低高风险投资，转向稳健现金流
- 预留30-50万医疗备用金
- 评估住房适老化改造需求

### 4. 2026年重大政策：长期护理保险
- 全民参保，费率约工资0.3%
- 保障重度失能人员，逐步扩至中度失能
- 不设起付线，报销护理服务费用
- 36项全国统一服务项目（20项生活照护+16项医疗护理）
- 2028年底全国基本覆盖

### 5. 医疗保障三层防护
- **基础层**：职工/居民医保
- **补充层**：百万医疗险（年保费300-800元）
- **兜底层**：重疾险 + 长护险

### 6. 居住与生活方式
三种养老模式对比：
- **居家养老**（90%主流）：成本3,000-5,000元/月，需适老化改造
- **社区养老**（7%）：5,000-10,000元/月，有社交和基本医护
- **机构养老**（3%）：8,000-30,000+元/月，24小时专业照护

适老化改造重点：卫生间扶手/防滑、感应夜灯、紧急呼叫、消除门槛

### 7. 法律与传承
- 遗嘱（40岁后建议）
- 意定监护协议（60岁前后）
- 预立医疗指示
- 数字资产盘点

### 8. 资产配置建议
按年龄的配置比例（安全/稳健/进取）：
- 30-35岁：30%/40%/30%
- 36-45岁：40%/40%/20%
- 46-55岁：50%/40%/10%
- 56+岁：60%/35%/5%

2. 使用4%法则帮助计算退休资金目标
3. 根据年龄段给出针对性建议
4. 提供可执行的具体行动清单
5. 提醒2026年长护险新政策
