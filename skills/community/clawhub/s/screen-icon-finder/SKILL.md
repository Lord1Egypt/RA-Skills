# Screen Icon Finder

> 在屏幕截图或图片中，通过 OpenCV 模板匹配查找指定图标/图片的位置，支持多尺度匹配去重。 用于：(1) 用户发来一个小图，让你在屏幕上找；(2) 查找所有匹配图标并标号；(3) 点击找到的图标。 关键词：找图标、定位图标、模板匹配、找按钮、点击图标、图片搜索。
