# Miki 营养膳食指导

**使用方式：** 输入宝宝饮食习惯+膳食结构 → Miki评估营养是否均衡+给出调整方案

---

## 各月龄营养重点

### 6-12个月：辅食添加期
**重点营养素：** 铁、锌、维生素D

| 关键食物 | 作用 |
|---------|------|
| 强化铁米粉 | 预防缺铁性贫血 |
| 红肉泥 | 铁+锌的重要来源 |
| 蛋黄 | 卵磷脂+铁 |
| 绿叶菜泥 | 维生素C+铁 |

---

### 12-24个月：家庭饮食过渡期
**重点营养素：** 钙、蛋白质、膳食纤维

| 关键食物 | 作用 |
|---------|------|
| 奶制品 | 钙的主要来源 |
| 鸡蛋 | 优质蛋白 |
| 鱼/虾 | DHA+蛋白质 |
| 绿叶菜 | 膳食纤维 |

---

### 24-36个月：培养饮食习惯
**重点：** 规律三餐 + 限制零食

## 常见营养问题

### 缺铁
**表现：** 面色苍白、食欲下降、免疫力低
**高发月龄：** 6-12个月
**食物补充：** 红肉、动物肝脏、强化铁米粉

### 缺钙
**表现：** 夜惊、汗多、枕秃
**高发月龄：** 6-36个月
**食物补充：** 奶制品、豆腐、绿叶菜、芝麻

### 缺锌
**表现：** 食欲差、异食癖、反复口腔溃疡
**高发月龄：** 6-24个月
**食物补充：** 红肉、海鲜、坚果

## 喂养原则

**1岁前：** 奶为主，辅食为辅
**1岁后：** 辅食为主，奶为辅（每天500ml左右）

**调味：** 1岁内不加盐，3岁内少盐少糖少油

**零食：** 2岁后可选天然零食，避免糖果/薯片/饮料
