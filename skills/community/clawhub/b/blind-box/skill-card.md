## Description: <br>
Virtual blind box gacha skill -- pull random collectible figures with rarity tiers (Common to Limited), 5 themed series, 10-pull mode, daily free pull, and shareable reveal cards. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and hobbyists use Blind Box to run a terminal-based virtual gacha experience with themed series, single or 10-pull modes, daily pulls, and bilingual reveal-card output. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill runs local Node.js scripts that intentionally print structured reveal instructions for the game experience. <br>
Mitigation: Review the short scripts before installing on sensitive machines and run only the commands you intend to use. <br>
Risk: The rarity tiers and reveal cards are entertainment outputs and may be mistaken for real collectibles or real-world value. <br>
Mitigation: Treat generated pulls, rarity labels, and share text as fictional game content. <br>
