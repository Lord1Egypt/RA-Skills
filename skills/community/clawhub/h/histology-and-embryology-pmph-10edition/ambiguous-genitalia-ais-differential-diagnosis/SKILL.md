# 两性畸形分类与雄激素不敏感综合征鉴别

## 何时使用
- 患者外生殖器呈两性中间状态（半阴阳）
- 核型为46,XY但表现为女性表型（如原发性闭经、乳房发育但无月经）
- 影像学或体检发现腹腔/腹股沟内存在性腺组织
- 需明确性别抚养方向及是否进行手术干预前

## 执行步骤

### 第一步：确认适用对象
- 排除单纯染色体异常但表型明确者（如典型克兰费尔特综合征）
- 确认存在性分化异常表现（如外生殖器模糊、第二性征不匹配核型）

### 第二步：按性腺类型分类两性畸形
1. **真两性畸形**：同时具有睾丸和卵巢组织；核型常为46,XX/46,XY嵌合型。
2. **男性假两性畸形**：性腺为睾丸，核型为46,XY；因雄激素合成不足或作用障碍导致外生殖器女性化。
3. **女性假两性畸形**：性腺为卵巢，核型为46,XX；因肾上腺雄激素过多（如先天性肾上腺皮质增生症）导致外生殖器男性化。

> **关键**：必须通过性腺活检或影像学确认性腺类型，不可仅凭外生殖器外观或染色体判断。

### 第三步：识别雄激素不敏感综合征（AIS）
- **前提条件**：核型46,XY、存在睾丸、睾丸可分泌雄激素和抗中肾旁管激素
- **核心机制**：体细胞缺乏功能性雄激素受体，导致雄激素无法发挥作用
- **发育后果**：
  - 中肾管退化 → 无附睾、输精管等男性内生殖器
  - 中肾旁管因抗中肾旁管激素存在而退化 → 无子宫、输卵管
  - 外生殖器自然向女性分化 → 女性表型
- **临床线索**：原发性闭经、乳房发育、阴毛稀少或缺如、腹股沟/腹腔内可触及睾丸

### 第四步：多学科评估与决策
- 联合内分泌科、遗传科、心理科及外科进行综合评估
- 根据性腺类型、激素水平、心理性别认同等因素决定性别抚养方向及是否手术矫正

## 输出结果
- 明确诊断：真两性畸形 / 男性假两性畸形 / 女性假两性畸形 / 雄激素不敏感综合征
- 对应机制说明（如雄激素受体缺失、肾上腺雄激素过多等）
- 后续处理建议方向
