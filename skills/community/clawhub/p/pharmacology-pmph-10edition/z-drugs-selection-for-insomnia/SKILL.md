# Z-药在失眠治疗中的选择与应用

## 触发条件
当满足以下条件时启用本技能：
- 患者已确诊为失眠症
- 需要短期催眠治疗（通常≤4周）
- 希望降低药物依赖风险和次晨残余效应

## 执行步骤

### 1. 确认适用对象
仅适用于以下四种Z-药：
- 唑吡坦（zolpidem）
- 扎来普隆（zaleplon）
- 佐匹克隆（zopiclone）
- 右佐匹克隆（dexzopiclone）

### 2. 排除禁忌证
若患者存在以下任一情况，禁用部分或全部Z-药：
- 严重肝肾功能不全（尤其扎来普隆）
- 睡眠呼吸暂停综合征
- 重症肌无力
- 需长期维持治疗（Z-药不适用）

### 3. 匹配失眠类型与药物特性
- **入睡困难为主**：选择起效快、半衰期短的药物
  - **唑吡坦**：显著缩短入睡潜伏期，次日警觉性好，无反跳性失眠
  - **扎来普隆**：超短效（t₁/₂ 1～1.5小时），仅改善入睡，无蓄积、无宿醉
- **全程失眠（入睡+维持困难）**：选择作用时间较长的药物
  - **佐匹克隆**：起效快（15～30分钟），改善各类失眠，t₁/₂ 3.5～6小时
  - **右佐匹克隆**：延长慢波睡眠和总睡眠时间，t₁/₂约6小时，不良反应轻微

### 4. 考虑药理特性
- 所有Z-药对GABA_A受体α₁亚基亲和力高，但右佐匹克隆对α₂/α₃也有高亲和力
- 优先选择次晨残余作用低、依赖风险小的药物
- 中毒时唑吡坦可用氟马西尼解救

## 输出结果
根据患者失眠类型和禁忌情况，推荐具体Z-药：
- 入睡困难 → 唑吡坦 或 扎来普隆
- 全程失眠 → 佐匹克隆 或 右佐匹克隆
