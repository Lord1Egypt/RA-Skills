## Description: <br>
Navigator is an instruction-only troubleshooting coach that reads a user's recent technical action, explains whether it worked, identifies one learning gap, helps close it, and prompts a backup checkpoint before continuing. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[contrario](https://clawhub.ai/user/contrario) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers, learners, solo founders, career changers, and self-taught technical users use Navigator to check whether a command, API call, code change, or setup step worked and to recover with one clear next action. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Users may paste secrets or private account details while asking the skill to review command output, errors, API calls, or setup steps. <br>
Mitigation: Avoid sharing API keys, passwords, tokens, private account details, or other sensitive data when using the skill. <br>
Risk: Suggested troubleshooting commands or code changes could affect local files, deployments, billing, or accounts if run without review. <br>
Mitigation: Review any command or change before running it, and keep the skill's backup checkpoint before continuing. <br>
