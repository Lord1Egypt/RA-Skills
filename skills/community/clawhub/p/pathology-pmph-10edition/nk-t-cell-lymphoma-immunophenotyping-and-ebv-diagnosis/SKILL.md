# NK/T细胞淋巴瘤的免疫表型与EB病毒诊断

## 何时使用
- 患者为中年男性，出现中线面部破坏性病变（如鼻腔或鼻咽部溃疡、骨质破坏、血管中心性浸润伴坏死）
- 疑似侵袭性T/NK细胞淋巴瘤，需与慢性中线肉芽肿、Wegener肉芽肿或其他外周T细胞淋巴瘤鉴别
- 已获取组织活检标本，并具备免疫组化和EBER原位杂交检测条件

## 执行步骤

1. **评估免疫表型组合**：
   - 检测CD56：应呈强阳性（NK细胞标志）
   - 检测胞质CD3ε：应阳性（注意膜CD3通常阴性）
   - 检测细胞毒性分子：TIA-1、穿孔素、颗粒酶B应表达

2. **进行EB病毒检测**：
   - 对肿瘤细胞执行EBER原位杂交（EBER ISH）
   - **所有确诊病例必须EBER阳性**；若阴性，则排除典型NK/T细胞淋巴瘤

3. **辅助判断TCR基因状态（如可行）**：
   - 多数病例呈胚系构型（支持NK细胞起源）
   - 少数可有克隆性TCR重排（提示细胞毒性T细胞来源，但仍符合诊断）

4. **结合临床与地域背景**：
   - 在亚洲地区高发（占非霍奇金淋巴瘤5%~20%）
   - 典型表现为快速进展的中线破坏性病灶伴坏死

5. **做出诊断结论**：
   - 若满足：CD56+、胞质CD3+、EBER+，且临床符合中线破坏性病变 → 确诊鼻型结外NK/T细胞淋巴瘤
   - 推荐治疗路径：放疗联合化疗（如SMILE或GELOX方案）

> **注意**：本技能不适用于其他类型T细胞淋巴瘤（如PTCL-NOS）。EBV阴性病例需谨慎，应重新评估诊断。
