# 各平台GEO特点与内容规范

本文档详细说明各主流自媒体平台的特点、发布规范和GEO优化策略。

---

## 1. 百家号（百度系SEO核心）

### 平台特点
- **搜索权重**：⭐⭐⭐⭐⭐ 最高，百度搜索直接收录
- **内容形式**：图文、视频、动态
- **推荐机制**：搜索引擎+信息流双通道

### 发布规范
| 项目 | 要求 |
|------|------|
| 标题字数 | 15-30字，必须含核心关键词 |
| 正文字数 | 800-2000字 |
| 图片数量 | 3-9张，配图需添加ALT描述 |
| 标签数量 | 3-5个行业标签 |

### GEO优化技巧
```
标题公式：[地域]+[行业关键词]+[差异化词]
示例：深圳装修公司哪家好？10年口碑老店教你选对装修公司

关键词布局：
- 首段必须出现核心关键词
- 小标题包含长尾关键词
- 结尾总结含关键词

图片优化：
- 封面图含文字说明
- 每张图片添加ALT描述（含关键词）
```

### 浏览器自动化发布
**后台地址**：https://baijiahao.baidu.com/

**发布流程**：
1. `agent-browser open https://baijiahao.baidu.com/`
2. 如需登录，等待用户扫码/输入验证码
3. 点击「发布文章」
4. 填入标题 → type "#title-input" "[标题]"
5. 填入正文 → type "#content-editor" "[正文]"
6. 上传封面图 → agent-browser click "[上传按钮]"（需手动选择文件）
7. 添加标签 → type "#tag-input" "[标签]"
8. 点击发布 → agent-browser click "[发布按钮]"
9. 截图确认

## 2. 头条号（字节系内容核心）

### 平台特点
- **搜索权重**：⭐⭐⭐⭐⭐ 字节搜索+今日头条搜索
- **内容形式**：图文、视频、微头条、问答
- **推荐机制：强算法推荐，爆款潜力大

### 发布规范
| 项目 | 要求 |
|------|------|
| 标题字数 | 15-28字，含热点词更佳 |
| 正文字数 | 600-1500字 |
| 图片数量 | 1-9张 |
| 标签数量 | 5个标签，可蹭热点 |

### GEO优化技巧
```
标题公式：[热点词]+[需求词]+[地域]（可选）
示例：2024装修报价单曝光！北京业主真实花费清单分享

蹭热点策略：
- 关注平台热榜话题
- 结合节假日/事件营销
- 使用平台热门BGM（视频）

关键词布局：
- 正文前100字必须有关键词
- 每300字出现一次关键词
- 使用自然语义，避免堆砌
```

### 浏览器自动化发布
**后台地址**：https://mp.toutiao.com/

**发布流程**：
1. `agent-browser open https://mp.toutiao.com/`
2. 如需登录，等待用户操作
3. 点击「发头条」（发布图文）
4. 填入标题 → type "[标题输入框]" "[标题]"
5. 填入正文 → type "[正文编辑框]" "[正文]"
6. 添加标签 → 在标签输入框输入关键词
7. 选择分类 → agent-browser click "[分类选项]"
8. 点击发布/定时发布

## 3. 知乎（权威内容核心）

### 平台特点
- **搜索权重**：⭐⭐⭐⭐⭐ 百度/搜狗搜索高权重
- **内容形式**：回答、文章、想法、视频
- **用户画像**：高学历、高收入、高决策权

### 发布规范
| 项目 | 要求 |
|------|------|
| 回答字数 | 300-2000字 |
| 文章字数 | 1000-5000字 |
| 图片数量 | 不限，建议每500字配图 |
| 话题标签 | 选择2-5个相关话题 |

### GEO优化技巧
```
标题公式：问题型+含关键词
示例：[地域]+[行业]哪家好？真实用户10年经验分享

问答策略：
- 搜索"[行业]哪家好"、"怎么办"、"如何选择"等问题
- 优先回答已有高浏览量问题
- 文章可同步投稿相关问题

内容结构：
- 开头：直接回答问题核心
- 中段：详细论证+案例
- 结尾：总结+行动建议+联系方式
```

### 浏览器自动化发布
**后台地址**：https://www.zhihu.com/creator/create-article

**发布流程**：
1. `agent-browser open https://www.zhihu.com/creator/create-article`
2. 登录知乎账号
3. 填入标题 → type "[标题输入框]" "[标题]"
4. 填入正文 → type "[正文编辑框]" "[正文]"
5. 添加话题 → agent-browser click "[话题按钮]" → type "[话题输入框]" "[话题]"
6. 点击「发布回答」或「发布文章」

## 4. 小红书（种草转化核心）

### 平台特点
- **搜索权重**：⭐⭐⭐⭐ 微信搜一搜+小红书搜索
- **内容形式**：图文笔记、视频笔记
- **转化路径**：种草→搜索→咨询→成交

### 发布规范
| 项目 | 要求 |
|------|------|
| 标题字数 | 20字以内，含emoji更佳 |
| 正文字数 | 300-800字 |
| 图片数量 | 3-9张，封面图最重要 |
| 话题标签 | 3-5个，如#深圳探店 #装修灵感 |

### GEO优化技巧
```
标题公式：[情绪词]+[需求词]+[地域]+[emoji]
示例：必看！深圳业主装修前必知的10个避坑指南建议收藏

封面图设计：
- 文字清晰，大小适中
- 使用对比图（前后/好坏）
- 加入品牌水印

正文结构：
- 开头：痛点共鸣
- 中段：干货内容+产品植入
- 结尾：行动召唤+位置标签

标签策略：
- 地域标签：#深圳 #福田
- 行业标签：#装修 #软装
- 场景标签：#家居灵感 #旧房改造
```

### 浏览器自动化发布
**后台地址**：https://creator.xiaohongshu.com/

**发布流程**：
1. `agent-browser open https://creator.xiaohongshu.com/`
2. 登录小红书账号
3. 点击「发布笔记」
4. 上传封面图和正文章图片
5. 填入标题 → type "[标题输入框]" "[标题]"
6. 填入正文 → type "[正文编辑框]" "[正文]"
7. 添加话题标签 → type "[话题输入框]" "[#话题]"
8. 添加地点标签（如需要）
9. 点击「发布」

## 5. 抖音（流量爆发核心）

### 平台特点
- **搜索权重**：⭐⭐⭐⭐ 抖音搜索+百度搜索
- **内容形式**：短视频（15秒-10分钟）、图文
- **变现路径**：曝光→引流→转化

### 发布规范
| 项目 | 要求 |
|------|------|
| 视频时长 | 15-60秒（完播率高） |
| 图文要求 | 最多9张图 |
| 话题标签 | 3-5个热门话题 |
| 封面设置 | 需单独设置，决定点击率 |

### GEO优化技巧
```
视频脚本结构（前3秒法则）：
- 0-3秒：制造悬念/冲突，吸引停留
- 3-15秒：快速抛出价值点
- 15-60秒：详细讲解+引导互动

标题/文案公式：[悬念/情绪]+[关键词]+[行动]
示例：深圳业主必看！装修合同这5条不签亏大了

话题策略：
- 行业话题：#装修 #软装设计
- 地域话题：#深圳装修 #广东装修
- 热点话题：#2024装修风格 #家居美学

SEO技巧：
- 视频描述含核心关键词
- 评论区置顶含关键词
- 合集名称含行业词
```

### 浏览器自动化发布
**后台地址**：https://creator.douyin.com/

**发布流程**：
1. `agent-browser open https://creator.douyin.com/`
2. 登录抖音账号
3. 点击「上传视频」或「发布图文」
4. 选择视频/图片文件
5. 填入标题/描述 → type "[描述输入框]" "[文案内容]"
6. 添加话题标签 → type "[话题输入框]" "[#话题]"
7. 设置封面（如需）
8. 点击「发布」

**注意**：视频文件需手动选择上传，自动化可处理文案和标签部分

## 平台选择建议

| 商家类型 | 优先平台 | 内容策略 |
|----------|----------|----------|
| 本地服务（装修/餐饮/家政） | 抖音+小红书+百家号 | 短视频+图文+SEO |
| 零售电商 | 小红书+抖音 | 种草+直播 |
| B2B企业 | 知乎+百家号+头条号 | 专业内容+SEO |
| 知识付费 | 知乎+头条号 | 问答+专栏 |
| 实体店铺 | 小红书+抖音+快手 | 打卡+探店+种草 |
