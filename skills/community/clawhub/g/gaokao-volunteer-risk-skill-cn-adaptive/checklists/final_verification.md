# Final Verification Checklist

Before giving final recommendations, verify:

## Student data
- [ ] Province and year
- [ ] Category/track
- [ ] Score
- [ ] Rank
- [ ] Selected subjects
- [ ] Preferences and avoid fields
- [ ] Tuition constraints

## Province policy
- [ ] Admission mode
- [ ] Application unit
- [ ]志愿数量
- [ ] Whether major adjustment exists
- [ ] Current year score lines
- [ ] Previous 1–3 years score lines

## Each recommendation
- [ ] Subject requirement satisfied
- [ ] Historical cutoff/rank checked
- [ ] Line margins calculated
- [ ] Major-level fit checked
- [ ] Plan count checked
- [ ] Tuition checked
- [ ] High-fee/sino-foreign flag checked
- [ ] Adjustment risk checked

## Report honesty
- [ ] Official sources cited
- [ ] Unverified data marked as需复核
- [ ] No unsupported precision claims
