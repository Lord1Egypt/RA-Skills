## Platform attributes

- Users: female-skewed, long decision cycles, save-heavy / compare-heavy
- Content forms: image posts (cover weight is highest) + short video (vertical-dominant)
- Algorithm cares about: completion rate / save rate / comment rate > like rate
- Traffic formula: good cover + good hook + saveable point

## Platform viral formulas (XHS high-frequency structures)

### Image posts
- **18-image outfit collection** — model + flat-lay grouped by scene
- **Store deep-dive** — first image is a wide shot, then individual items, last image has the address
- **Recommendation card** — first image is a big-text cover, then multi-angle product shots

### Video posts
- **Immersive merchandising** — no voiceover, pure visuals + BGM + on-screen text
- **Founder interview** — founder on-camera + contrast opener + values closer
- **Product craft close-up** — hands / material / process + cultural meaning

## Tone adaptation (DNA → XHS)

> Implementing [[brand/brand-voice]] on the XHS platform:
- Title: 1 emoji as accent OK (preferably aesthetic ones like 💚✨📍)
- Body: address forms like "girls" / "babes" used **sparingly** — once per post max
- Short paragraphs, lots of line breaks, visual breathing room
- CTA uses 📍 to point to address; never "buy now" / "must-have" pressure

## Hook templates (XHS-specific)

> See [[engine/hooks]] for the general library. These work especially well on XHS:
- **Location + contrast**: "Found the {category} mecca in {location}"
- **Number + comparison**: "{Year-of-birth} owner runs a {size} store"
- **Identity + experience**: "What does it feel like to wear {X}?"
- **Comparison + question**: "Why do some people in {category} look effortlessly {trait}?"

## Taboos (XHS platform)

> See [[engine/taboo]] for general bans. XHS-specific:
- Don't put product prices directly in the body (high traffic-throttle risk)
- Don't include WeChat / external contact in body (use comment-based redirects instead)
- Don't use absolutist superlatives ("best / first / only")
- Don't stack sales emojis in title (💰🔥🎁 looks cheap)
