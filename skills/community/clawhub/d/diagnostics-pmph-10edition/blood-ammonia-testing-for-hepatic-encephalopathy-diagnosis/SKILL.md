# 血氨检测在肝性脑病诊断中的应用

## 触发条件
当患者有严重肝病且出现意识障碍时，立即启动血氨检测以评估肝性脑病风险。

## 执行步骤

1. **确认适用对象**：仅适用于以下人群：
   - 肝硬化患者
   - 暴发性肝衰竭患者
   - 上消化道出血患者

2. **验证前提条件**：
   - 已采集静脉血，并在15分钟内完成血浆分离
   - 排除高蛋白饮食或剧烈运动等生理性干扰因素

3. **排除禁忌与干扰**：
   - 不适用于无肝病的单纯肾功能不全患者
   - 确保标本未因处理延迟导致假性升高

4. **执行检测与解读**：
   - 参考区间：18～72 μmol/L
   - 若血氨 >72 μmol/L 且患者存在严重肝病背景，提示肝性脑病高风险
   - 结合临床症状（如意识障碍、扑翼样震颤）综合判断

5. **输出警报**：
   - 生成标准化结果提示：“血氨 >72 μmol/L 且存在严重肝病，提示肝性脑病高风险；需结合临床症状判断”

## 注意事项
- 血氨升高也可能由上消化道出血（肠道产氨增加）、尿毒症或肝外门体分流引起，需鉴别诊断
- 血氨正常不能完全排除肝性脑病，应结合其他临床指标综合评估
