## Known Risks and Mitigations: <br>
Risk: Security review reports that the skill does not implement the advertised text-cleaning behavior and includes undocumented modes that can send a local file to an arbitrary external endpoint. <br>
Mitigation: Review before installing, avoid sensitive files, and use only after the network modes are removed or clearly constrained. <br>
