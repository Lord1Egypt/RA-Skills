# 工伤赔偿计算公式大全

> 数据来源：《工伤保险条例》（2010修订）及配套文件
> 最后更新：2026年

---

## 一、月工资基数的认定（所有计算的起点）

**月工资 = 劳动合同约定工资 + 奖金 + 津贴 + 补贴 + 加班工资**
- 不包括非常规性福利；年终奖需按约定折算
- 计算公式：伤前12个月平均月缴费工资（总工资 ÷ 12，或月度工资直接提供）

**月工资上下限（适用于伤残补助金、伤残津贴计算基数）：**
```
实际计算基数 = MAX(本人月工资, 统筹地区上年度职工月平均工资 × 60%)
实际计算基数 = MIN(实际计算基数, 统筹地区上年度职工月平均工资 × 300%)
```
> 注：此上下限规则适用于**一次性伤残补助金**和**伤残津贴**，不适用于离职两项补助金（各省自有规定）

**日工资 = 月工资基数 ÷ 21.75**
**小时工资 = 日工资 ÷ 8**

## 二、工伤伤残赔偿项目（1-10级）

### 2.1 一次性伤残补助金（工伤保险基金支付，全国统一）

| 伤残等级 | 月数 |
|---------|------|
| 一级    | 27个月 |
| 二级    | 25个月 |
| 三级    | 23个月 |
| 四级    | 21个月 |
| 五级    | 18个月 |
| 六级    | 16个月 |
| 七级    | 13个月 |
| 八级    | 11个月 |
| 九级    | 9个月  |
| 十级    | 7个月  |

**计算公式：** `实际计算基数（含60%-300%限制）× 对应月数`

### 2.2 伤残津贴（按月支付，1-6级适用）

| 伤残等级 | 比例 | 支付主体 | 备注 |
|---------|------|---------|------|
| 一级    | 90% | 工伤保险基金 | 长期按月 |
| 二级    | 85% | 工伤保险基金 | 长期按月 |
| 三级    | 80% | 工伤保险基金 | 长期按月 |
| 四级    | 75% | 工伤保险基金 | 长期按月 |
| 五级    | 70% | 用人单位   | **须难以安排工作时才支付** |
| 六级    | 60% | 用人单位   | **须难以安排工作时才支付** |

**计算公式：** `实际计算基数（含60%-300%限制）× 对应比例`

> 1-4级：保留劳动关系，不再上班，工伤保险基金按月支付。
> 5-6级：保留劳动关系，优先安排工作；难以安排时，由单位按月支付伤残津贴。

### 2.3 停工留薪期待遇

**停工留薪期工资：** `本人月工资 × 停工留薪期月数`

- 支付主体：**用人单位**
- 期限：以劳动能力鉴定委员会确认结果为准，一般不超12个月；特殊情形经确认可延长至24个月
- 如未经鉴定，参考本技能停工留薪期参考目录（附件），并提示以官方鉴定为准

**停工留薪期护理费：**
- 支付主体：**用人单位**
- 条件：职工确需护理，且单位未派专人护理
- 具体认定标准参考当地规定

### 2.4 评残后护理费（工伤保险基金支付，1-4级按需）

**基数：** 统筹地区上年度职工月平均工资

| 护理依赖等级 | 比例 |
|-----------|------|
| 完全不能自理 | 50% |
| 大部分不能自理 | 40% |
| 部分不能自理  | 30% |

**计算公式：** `统筹地区上年度月平均工资 × 对应比例`
> 需经劳动能力鉴定委员会确认护理依赖等级

### 2.5 离职补偿（5-10级，解除/终止劳动合同后）

包含两笔：
1. **一次性工伤医疗补助金**（工伤保险基金支付）
2. **一次性伤残就业补助金**（用人单位支付）

**各省具体月数和计发基数不同，由 `policy_data.leaving_subsidy_standards` 提供；本文件仅说明通用计算结构。**

**职业病附加规则：** 患职业病的，**一次性工伤医疗补助金在各省标准基础上额外增发20%**（部分省份规定不同，详见各省特殊情形说明）

**退休年龄折减规则（严格执行）：**
```
若距法定退休年龄 ≥ 5年：全额支付
若距法定退休年龄不足5年：
  - 每少1年，就业补助金递减20%
  - 不足1年按10%支付
若已达法定退休年龄 或 符合领取养老金条件：支付金额 = 0
  （例外：《劳动合同法》第38条情形除外）
```

> 具体计算：先用 retirement_age.md 中的公式算出该劳动者的确切退休年月，再计算距今的剩余年数（不足1年按1年算）

### 2.6 住院伙食补助费

- 按住院地标准，不超当地机关一般工作人员出差补助
- 多数地区标准：100元/天
- 外地治疗可报销1名陪护人员伙食费

### 2.7 交通费、住宿费

- 凭正式票据报销
- 票据需与就医地点、时间、人数、次数匹配

### 2.8 医疗费

- 工伤保险基金**实报实销**
- 需符合工伤保险诊疗目录、药品目录、住院服务标准

### 2.9 工伤康复费

- 工伤保险基金按规定报销

### 2.10 伤残辅助器具费

- 工伤保险基金按国家规定标准支付
- 需提供具体配置费用，无固定公式

## 三、工亡赔偿项目

### 3.1 一次性工亡补助金（全国统一，工伤保险基金支付）

**计算公式：** `死亡时上一年度全国城镇居民人均可支配收入 × 20`

> 注意：按**死亡时间**对应的**上一年度**数据，不是当年
> 具体年度金额由 `policy_data.work_death_subsidy` 提供。

### 3.2 丧葬补助金（工伤保险基金支付）

**计算公式：** `统筹地区上年度职工月平均工资 × 6`

### 3.3 供养亲属抚恤金（工伤保险基金按月支付）

| 亲属类型 | 计算比例 |
|---------|---------|
| 配偶    | 死者本人月工资 × 40%（每月） |
| 其他亲属 | 死者本人月工资 × 30%（每人每月） |
| 孤寡老人或孤儿 | 在上述基础上 +10% |

**封顶规则：** 所有供养亲属抚恤金之和 **不得超过职工本人月工资的100%**

> "本人工资"指死者伤前12个月平均月缴费工资，同样适用60%-300%上下限

## 四、注意事项

1. **两项补助金申领后，工伤保险关系终止**，但不影响失业保险待遇和经济补偿金领取
2. 多次工伤：部分省份规定按最高伤残级别计发（如山西），或每多一次额外增发（如吉林）
3. 未参保用人单位：两项补助金由**用人单位全额支付**（山东等省份有明确规定）
4. 如果在本地知识库和联网搜索中都找不到该省的某项补助金具体标准，请明确告知用户"该地区该项标准缺失，建议致电当地社保局12333咨询"，切勿自行套用其他省份标准
