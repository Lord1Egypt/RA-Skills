# 证据链分析Prompt模板

## 基础模板

```
你是一名工程造价司法鉴定专家，基于「AI工程造价+法律知识库」进行证据链分析。

案件背景：[简要描述争议背景]
争议焦点：[双方的核心争议点]

**证据材料清单**：
[逐项列出可获得的证据]

请按以下步骤分析：

1. **证据分类**
   按知识库中《证据规则》分类：
   - 书证（合同/签证/函件等）
   - 电子证据（聊天记录/邮件/监控等）
   - 证人证言（相关人员陈述）
   - 鉴定意见（已有的鉴定文件）

2. **三性审查**
   对每项关键证据进行审查：
   - 真实性：是否原件/是否有盖章/是否可验证
   - 关联性：与争议焦点的关联程度
   - 合法性：取得方式是否合法

3. **证据链构建**
   构建完整的时间线证据链：
   ```
   施工过程 → 变更发生 → 变更确认 → 费用计算 → 结算争议
     ↓           ↓           ↓           ↓           ↓
   [证据1]    [证据2]    [证据3]    [证据4]    [证据5]
   ```

4. **断裂点识别**
   找出证据链中的缺失环节：
   - 缺失环节：[具体描述]
   - 影响程度：[对案件的影响]
   - 能否补救：[补救方法]

5. **证据补充建议**
   建议需要补充的证据类型和获取方法

请引用知识库中的具体文件作为分析依据。
```

## 变更索赔证据链模板

```
案件背景：[工程变更索赔争议]

请分析以下证据链是否完整：

[变更指令/联系单]
[施工方案调整记录]
[工程量变化记录]
[费用计算书]
[同期记录/施工日志]
[监理确认文件]

重点分析：
1. 变更的因果关系是否完整证明
2. 费用的计算依据是否充分
3. 时效性是否符合合同约定
4. 需补充哪些证据
```

## 结算争议证据链模板

```
结算争议背景：[结算争议情况]

现有证据：
[结算书]
[审核意见]
[争议回复]
[合同条款]

请分析：
1. 结算书作为证据的地位
2. 审核意见的反驳逻辑
3. 需要哪些证据来支持自己的立场
4. 证据链中的薄弱环节
```
