## 二、第一步：收集命主信息

### 2.1 必填信息

| 信息项 | 说明 | 用途 |
|--------|------|------|
| **姓名** | 全名（含姓） | 三才五格计算 |
| **性别** | 男/女 | 格数吉凶判断 |
| **出生公历时间** | 年月日时（精确到时辰） | 排八字 |
| **出生农历时间** | 可选，用于校验 | 避免时辰误差 |
| **出生地点** | 省市即可 | 五运六气参考 |
| **父亲姓氏** | 即姓氏 | 天格计算 |
| **生肖** | 可不填，从八字推算 | 属相配合 |

### 2.2 信息收集模板

```
【起名信息登记表】

姓名：_______
性别：男 / 女
出生公历：____年____月____日____时（____点）
出生农历：____年____月____日____时
出生地点：____省____市

备注：有无特殊要求？（如：避开某字、喜欢某风格等）
```

### 2.3 八字排盘方法回顾

#### 排年柱
- 以**立春**为分界（不是正月初一）
- 例：2026年立春后为丙午年

#### 排月柱
- 地支固定：正月寅、二月卯...十二月丑
- 天干由年干决定（五虎遁）

```
甲己之年丙作首
乙庚之岁戊为头
丙辛必定寻庚上
丁壬壬寅顺水流
若问戊癸何处起
甲寅之上好追求
```

#### 排日柱
- 需查万年历，60天一循环
- 子时（23:00）为日分界

#### 排时柱
- 地支固定对应时辰
- 天干由日干决定（五鼠遁）

```
甲己还加甲
乙庚丙作初
丙辛从戊起
丁壬庚子居
戊癸推壬子
```

### 3.2 日主强弱判断

#### 判断三要素

| 要点 | 说明 | 权重 |
|------|------|------|
| **得令** | 月支是否生扶日主 | 50% |
| **得地** | 地支是否有根气 | 30% |
| **得势** | 天干是否有比劫印星 | 20% |

#### 旺相休囚死对照表

| 月令 | 日主状态 |
|------|----------|
| 当令（月令与日主同五行） | 旺 |
| 月令所生 | 相 |
| 生月令者 | 休 |
| 克月令者 | 囚 |
| 被月令克 | 死 |

#### 快速判断法（数数法）

```
同类（比劫）+ 生我（印星）≥ 3个 → 身强
同类（比劫）+ 生我（印星）≤ 2个 → 身弱
```

> ⚠️ 仅为初步参考，需综合月令、地支藏干、合化等因素

### 3.3 格局判断

| 格局类型 | 特征 | 喜忌原则 |

## 四、第三步：确定喜用神

### 4.1 喜用神确定原则

| 原则 | 适用情况 | 用神方向 |
|------|----------|----------|
| **扶抑** | 日主过旺或过弱 | 旺则抑之，弱则扶之 |
| **调候** | 冬生寒命、夏生燥命 | 寒命喜火暖，燥命喜水润 |
| **通关** | 两行相战不相让 | 用第三方化解 |
| **从势** | 极旺或极弱 | 顺势而行 |

### 4.2 身强格的喜忌

| 身强类型 | 用神（喜） | 忌神（忌） |
|----------|------------|------------|
| 官杀旺 | 官杀（克身） | 印星、比劫 |
| 食伤旺 | 食伤（泄秀） | 印星、比劫 |
| 财星旺 | 财星（耗身） | 印星、比劫 |

### 4.3 身弱格的喜忌

| 身弱类型 | 用神（喜） | 忌神（忌） |
|----------|------------|------------|
| 印星为用 | 印星（生身） | 官杀、食伤、财星 |
| 比劫为用 | 比劫（助身） | 官杀、食伤、财星 |

### 4.4 缺≠补原则

> ⚠️ **最大误区**：不是八字缺什么五行就补什么！

**五种"缺而不补"的情况**：

| 情况 | 说明 | 举例 |
|------|------|------|
| 所缺为忌神 | 缺的正是命局所忌，补之反而为祸 | 火旺命缺金，金为忌 |
| 缺而不显 | 表面缺，实际藏于地支 | 寅中藏甲丙戊 |
| 从格专旺 | 特殊格局忌破格 | 炎上格忌水 |
| 调候不及 | 补之无力 | 冬命缺火但火太弱 |
| 流通有情 | 强行补缺破坏流通 | 五行流转顺畅 |

**正确原则**：所缺五行必须是喜用神才补！

## 五、第四步：三才五格配置

### 5.1 五格定义

| 五格 | 别称 | 含义 | 代表人生阶段 |
|------|------|------|--------------|
| **天格** | 根运 | 祖先、先天 | 幼年（1-12岁） |
| **人格** | 苗运 | 自身、主运 | 中青年（25-36岁）**核心** |
| **地格** | 花运 | 名字、前运 | 早年（13-24岁） |
| **外格** | 叶运 | 社交 | 中年（37-48岁） |

### 5.2 五格计算方法

> ⚠️ **必须使用繁体字/康熙字典笔画数！**

| 姓名类型 | 天格 | 人格 | 地格 | 外格 | 总格 |

### 5.6 三才配置原则

#### 相生相克关系

```
相生：金生水 → 水生木 → 木生火 → 火生土 → 土生金
相克：金克木 → 木克土 → 土克水 → 水克火 → 火克金
```

#### 最佳配置

| 配置类型 | 吉凶 | 说明 |
|----------|------|------|
| 连续相生 | ★★★ 大吉 | 天→人→地或人→地连续相生 |
| 循环相生 | ★★★ 大吉 | 如：金→土→金（循环） |
| 单向相生 | ★★ 吉 | 只有一重相生关系 |
| 比和 | ★ 平 | 同五行，比和 |
| 相克 | ★★ 慎 | 需结合命局评估 |

#### 李明哲案例

```
李（7画）明（8画）哲（10画）

天格 = 7 + 1 = 8 → 8→金
人格 = 7 + 8 = 15 → 5→土
地格 = 8 + 10 = 18 → 8→金

三才：金→土→金（循环相生）→ 大吉配置
```

## 八、起名检查清单

### 8.1 八字检查

- [ ] 八字排盘是否正确？
- [ ] 日主五行是否判断准确？
- [ ] 日主强弱是否评估正确？
- [ ] 格局类型是否判断无误？
- [ ] 喜用神是否确定正确？
- [ ] 是否避免补忌神之误？

### 8.2 三才五格检查

- [ ] 笔画计算是否使用康熙字典繁体？
- [ ] 人格数理是否为吉数？
- [ ] 地格、总格是否避开大凶数？
- [ ] 女性是否避开孤寡运数？
- [ ] 三才配置是否相生或比和？
- [ ] 三才五行是否配合喜用神？

### 8.3 五行用字检查

- [ ] 字形五行是否判断正确？
- [ ] 字义五行是否与形符一致？
- [ ] 是否优先选用喜用神对应的字？
- [ ] 是否规避了忌神对应的字？
- [ ] 男女用字是否适配？

### 8.4 音韵检查

- [ ] 声调是否有起伏？（避免全平或全仄）
- [ ] 声母是否"打架"？（避免双声）
- [ ] 韵母是否单调？（避免叠韵）
- [ ] 尾音是否响亮？（优先开口音）
- [ ] 姓氏与名字声调搭配是否合理？

### 8.5 谐音检查

- [ ] 普通话快速连读3遍有无问题？
- [ ] 方言读一遍有无负面谐音？
- [ ] 有无容易被拆解的绰号？
- [ ] 是否与网络热梗冲突？

### 8.6 寓意检查

- [ ] 字义是否积极正面？
- [ ] 男女风格是否适配？
- [ ] 与姓氏搭配是否协调？
- [ ] 是否太过浮夸或太满？
- [ ] 是否经得起时间考验？

## 十、学习回顾与总结

### 10.1 起名方法论核心要点

```
┌────────────────────────────────────────────┐
│           起名实战三原则                    │
├────────────────────────────────────────────┤
│                                            │
│  ① 八字为根本                              │
│     - 喜用神是核心依据                      │
│     - 缺≠补，忌神不补                       │
│                                            │
│  ② 三才为框架                              │
│     - 人格必须为吉                          │
│     - 三才相生优先                          │
│     - 配合喜用神五行                        │
│                                            │
│  ③ 音义为血肉                              │
│     - 平仄相间有韵律                        │
│     - 寓意温润不浮夸                        │
│     - 谐音歧义全排查                        │
│                                            │
└────────────────────────────────────────────┘
```

### 10.2 快速起名流程图（简化版）

```
1. 收集信息（出生时间、性别）
      ↓
2. 排八字 → 定日主强弱
      ↓
3. 找喜用神（扶抑/调候/通关）
      ↓
4. 筛选五行用字（喜神优先）
      ↓
5. 计算三才五格（避开大凶）
      ↓
6. 音韵搭配（平仄平/仄平仄）
      ↓
7. 谐音排查 → 确定名字
```

### 10.3 常见误区提醒

| 误区 | 正确做法 |
|------|----------|
| 缺什么补什么 | 喜用神才补，忌神不补 |
| 三才五格决定一切 | 八字为本，五格为辅 |
| 数理越大吉越好 | 配合命局，适合最重要 |
| 名字越复杂越好 | 简洁大气为佳 |
| 只看五行不看音义 | 形神音义四位一体 |
