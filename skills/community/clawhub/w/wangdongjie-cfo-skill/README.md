## 📦 安装方式

### 方式一：ClawHub一键安装（推荐）
访问 [ClawHub页面](https://clawhub.ai/yjkj999999/wangdongjie-cfo-system)，WorkBuddy 用户可直接一键安装。

### 方式二：手动安装
```bash
# 克隆仓库
git clone https://github.com/yjkj999999/wangdongjie-cfo-system.git

# 复制 SKILL.md 到 WorkBuddy skills 目录
cp wangdongjie-cfo-system/SKILL.md ~/.workbuddy/skills/wangdongjie-cfo-system/
```

### 方式三：直接下载
1. 点击本仓库右上角的 **Code → Download ZIP**
2. 解压后将 `SKILL.md` 放入 `~/.workbuddy/skills/wangdongjie-cfo-system/`
3. 重启 WorkBuddy
