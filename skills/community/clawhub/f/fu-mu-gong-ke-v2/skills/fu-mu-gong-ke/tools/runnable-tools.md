## 可运行工具

本技能包含以下可执行脚本（位于 `scripts/` 目录）：

| 脚本 | 功能 |
|------|------|
| `assessment.py` | 错位程度评估问卷 |
| `reflection.py` | 听见防御日记（每日反思） |
| `breathing_timer.py` | 30秒呼吸计时器 |
| `scenario_practice.py` | 场景对话练习 |
| `visualize.py` | 家庭互动模式可视化 |

运行方式：`python scripts/YOUR_SCRIPT.py`
