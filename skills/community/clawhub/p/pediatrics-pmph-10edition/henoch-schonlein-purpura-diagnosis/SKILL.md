# Henoch-Schönlein紫癜（过敏性紫癜）临床诊断

## 何时使用
- 患儿出现典型皮肤紫癜（对称分布、压之不褪色的紫红色斑丘疹）
- 同时存在以下任一附加表现：弥漫性腹痛、大关节肿痛、肾脏损害（血尿/蛋白尿）、或活检证实IgA沉积
- 需与免疫性血小板减少性紫癜（ITP）、风湿性关节炎、败血症或外科急腹症等鉴别

## 如何执行诊断
1. **确认典型皮肤紫癜**：
   - 观察皮疹是否为紫红色斑丘疹，高出皮面，压之不褪色
   - 分布是否对称，主要位于四肢及臀部伸侧，面部及躯干少见
   - 是否呈分批出现，2～4周内可自行消退，可能复发

2. **评估附加标准（满足其一即可）**：
   - **腹痛**：弥漫性，常位于脐周或下腹部，可伴呕吐、黑便
   - **关节症状**：大关节（膝、踝、肘、腕）肿痛，活动受限，无关节腔出血
   - **肾损害**：出现血尿、蛋白尿、管型尿，可伴高血压或水肿
   - **组织学证据**：任何部位活检显示IgA免疫复合物沉积

3. **确诊条件**：
   - 存在典型皮肤紫癜 **且** 满足至少一项附加标准 → 确诊为过敏性紫癜（IgA血管炎）

4. **鉴别诊断（若表现不典型）**：
   - **ITP**：血小板减少，出血时间延长，无系统性症状
   - **风湿性关节炎**：类风湿因子阳性，关节破坏更显著
   - **败血症**：高热、中毒貌、炎症指标显著升高
   - **外科急腹症**：需影像学（如超声）排除肠套叠、肠穿孔等

> 注意：本诊断标准不适用于尚未出现典型皮疹的早期病例。
