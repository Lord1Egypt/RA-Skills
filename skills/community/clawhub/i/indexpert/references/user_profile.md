# 用户画像

## 写入说明

首次对话收集到回答后，按以下格式写入"当前画像"，空字段填"未知"。
每次更新时在"版本历史"追加一行，不删除旧记录。

**判断"是否有数据"的标准**：`风险偏好` 字段不为"未知"即视为已有画像。
