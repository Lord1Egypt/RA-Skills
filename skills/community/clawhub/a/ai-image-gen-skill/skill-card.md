## Description: <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and creators use this skill to prepare prompts and shell commands for generating or editing images, including product visuals, article illustrations, and social media cover images. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill relies on a separately installed image-generation helper that is not reviewed in this release evidence. <br>
Mitigation: Install and use it only after reviewing and trusting the nano-banana-pro helper it invokes. <br>
Risk: Prompts, reference images, and a Gemini API key may expose sensitive data to an external provider or helper. <br>
Mitigation: Use a scoped API key and avoid sensitive prompts or private images unless the helper and provider data handling have been verified. <br>
