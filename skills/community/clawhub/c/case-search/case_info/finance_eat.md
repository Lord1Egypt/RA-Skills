# 财产倾吞类案件

案号：湖南省怀化市中级人民法院发布4起拒不执行判决、裁定罪典型案例之三：黄某某拒不执行判决、裁定案—未报告理财产品等财产情况逃避执行
概括：被执行人黄某某因未向法院报告其持有的理财产品等财产情况，企图逃避执行，被以拒不执行判决、裁定罪追究刑事责任。

案号：(2023)桂10刑终409号
概括：覃某清、黄某莉、谭某梅作为村集体管理人员，利用职务便利侵吞公摊水电费盈余及虚构租赁环节获利，被认定构成职务侵占罪。

案号：最高人民法院发布4个涉民营企业产权和民营企业家权益保护再审典型案例之三：窦某某职务侵占、挪用资金、隐匿会计凭证、会计账簿再审改判无罪案
概括：窦某某因个人财产与公司财产高度混同，原审认定的职务侵占、挪用资金等犯罪事实不清、证据不足，经再审被宣告无罪。

案号：(2020)最高法民再69号
概括：最高人民法院在该案中明确了破产管理人对债权人行使抵销权有异议时应通过诉讼解决，其自行否定并转移债务人财物的行为构成侵权。

案号：(2023)沪02刑终550号
概括：冯某某利用负责销售的职务便利，伙同他人通过虚设中介公司侵占单位银粉销售差价，其行为构成职务侵占罪。

案号：(2023)沪0106刑初161号
概括：桂某在担任国家出资企业不同职务期间（非国家工作人员与国家工作人员），利用职务便利侵吞单位钱款的行为分别被认定为职务侵占罪和贪污罪。

案号：(2022)沪0101刑初116号
概括：被告人项甲、项乙在引渡程序中自愿回国受审并如实供述，虽系被抓获归案，但其主动性与自愿性仍被法院认定为自首。

案号：(2021)辽02刑终40号
概括：张某某利用负责军粮拨付的职务便利，将低价补贴军粮按市场价出售赚取差价，其行为被认定为贪污罪。

案号：(2021)辽02民终10531号
概括：破产公司法定代表人咸家福擅自出售公司地上建筑及附属物，所得款项未交还公司，被法院认定为侵占公司财产，应予返还。
