# 分镜推导流程详情 (Layer 1 — 按需加载)

> 本文件包含 Phase 0-7 的完整推导指引。仅在进入对应阶段时加载。

---

## Phase 0: 模式选择与意图澄清 (Perception)
> **Harness 层级**: Read — 通过上下文管理器将用户输入翻译为结构化意图
- 确认创作目标（电影概念 / 卖货 / 种草 / 品牌 / 叙事 / 艺术实验）
- 选择对应 Mode（含 Mode 0 创意探索），明确核心约束
- 确定输入素材类型（纯文本 / 产品图 / 参考图 / 参考视频）
- **产出**: Mode 确认 + 输入策略 + 创作意图结构化描述

## Phase 1: 核心创意锁定 (Core Concept) (Planning)
> **Harness 层级**: Eval — 调用拦截器捕获创意意图并路由到正确的模式引擎
- 明确主体（产品 / 人物 / 场景）
- 定义单镜头内唯一的动作或信息点
- 确定情感基调和观众预期反应
- **拒绝策略**: 如果用户在一个镜头里塞了 > 1 个动作或 > 1 个信息点，主动建议拆分
- **产出**: Subject + Action 描述

## Phase 2: 时间轴与节奏 (Timeline & Rhythm) (Action)
> **Harness 层级**: 执行器 — 将时间规划注入确定性时间线模型
- 规划时长内的时间分配（默认 15s，电影模式支持 30-180s）
- 定义关键帧节点（至少 3 个：入场 / 核心 / 收尾）
- 多镜头模式：规划 Cut-to 节点和镜头间节奏关系
- **产出**: Timeline 描述 + 关键帧定义

## Phase 3: 视觉构图 (Visual Composition) (Action)
> **Harness 层级**: 执行器 — 构图参数化输出
- 确定画面比例、构图方式
- 主体在画面中的占比和位置
- 角色 / 产品与环境的空间关系
- **电商提醒**: 产品必须占画面显著位置，不要放在边角
- **UGC 提醒**: 保持自然取景，不要过度设计构图
- **电影提醒**: 遵循三分法/黄金分割/对称构图等经典电影语言
- **产出**: Composition 描述 + 画幅建议

## Phase 4: 镜头运动 (Camera Movement) (Action)
> **Harness 层级**: 调用拦截器 — 捕获镜头运动意图并路由到运动强度参数
- 选择 1 个主导镜头运动（不可在一个镜头内混用多种运动）
- 在 Timeline 上标注运动变化
- 给出 Motion Strength 建议值
- **原则**: 镜头运动服务于叙事动机，不炫技
- **产出**: Camera 描述（含运动强度）+ Motion Strength 参数

## Phase 5: 光影与质感 (Lighting & Texture) (Action)
> **Harness 层级**: 执行器 — 光影参数化输出
- 光源方向、色温、对比度
- 环境细节（烟雾、颗粒、反射、材质）
- 整体色调和视觉风格
- **电商提醒**: 光源不要遮挡产品标签
- **UGC 提醒**: 自然光优先，避免棚拍感
- **电影提醒**: 每个光源有情绪动机，参考 `assets/cheat_sheet.json` 中的 `cinematic_lighting_extended`
- **产出**: Lighting 描述 + 风格预设

## Phase 6: 声音设计 (Sound Design) (Action)
> **Harness 层级**: 执行器 — 声音参数化输出
- 确定音效类型（环境音 / 音效 / 对白 / 音乐）
- 音乐风格和节奏（参考 `assets/cheat_sheet.json` 中的 `sound_design_presets`）
- 音画同步点（关键动作匹配音效）
- **电商提醒**: 音乐不分散注意力，让产品说话
- **UGC 提醒**: BGM 保持在语音之下
- **电影提醒**: 参考导演声音签名（如 Nolan 的低频冲击）
- **产出**: Sound Design 建议

## Phase 7: 最终合成与参数输出 (Export) (Feedback)
- 整合所有 Phase 产出为完整提示词
- 生成 Negative Prompt（基于 Mode 自动注入）
- 输出多模态建议（参考图 / 参考视频 / 音频建议）
- 输出平台适配参数（根据目标平台约束自动适配）
- 质量自检：对照质量门禁逐项检查
- **降级策略**:
  - 结构不完整 → 回退到上一 Phase 补充
  - 参数超范围 → 自动 clamp 到安全区间
  - 模式不匹配 → 提示用户确认或自动切换兼容模式
- **产出**: 完整分镜方案
