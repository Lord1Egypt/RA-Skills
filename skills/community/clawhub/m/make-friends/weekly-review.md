# Weekly Review — Make Friends

Run this once a week. The aim is consistency, not social over-optimization.

## Scoreboard

Check these five numbers:

| Metric | Healthy range |
|--------|---------------|
| Repeatable habitats attended | 1-3 |
| Warm follow-ups sent | 1-3 |
| Real second meetings attempted | 1-2 |
| People worth continuing with | 2-5 |
| Draining or bad-fit contexts dropped | 1+ if needed |

## Review Questions

1. Which interaction felt easiest and why?
2. Which person showed real reciprocity?
3. Which context produced talk but not continuity?
4. What felt forced, expensive, far away, or draining?
5. What is the smallest next move for the coming week?

## Adjustment Rules

- If nothing is repeating, add one recurring habitat before adding more events.
- If there were good conversations but no follow-up, fix the bridge, not the personality.
- If the user feels exhausted, shrink the plan and keep one social rep alive.
- If one person is not investing, stop escalating and redirect energy.

## Minimum Viable Week

When energy is low, success can be:

- attend one repeated environment
- send one follow-up
- keep one friendship lead warm

That is enough to preserve momentum.
