## Description: <br>
双色球选号辅助系统 v2 基于历史数据生成概率优化的双色球候选号码，支持动态冷热周期、形态筛选、蓝球分析和多策略输出。 <br>

This skill is ready for commercial/non-commercial use. <br>

MIT-0 <br>


## Use Case: <br>
开发者或彩票数据分析用户可用该技能在本地更新双色球历史数据、生成 2 注主推和 3 注备选号码，并回测选号规则的历史表现。输出应视为娱乐或统计分析辅助，不是中奖预测或保证。 <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Users may treat statistically generated lottery picks as predictions or guarantees. <br>
Mitigation: Use the output only for entertainment or analysis; the artifacts state that lottery results are random and winning is not guaranteed. <br>
Risk: The skill installs PyPI packages, fetches lottery history from a third-party website, and writes CSV/JSON files locally. <br>
Mitigation: Review the dependency install command and run the scripts only in an environment where this network access and local file output are acceptable. <br>
