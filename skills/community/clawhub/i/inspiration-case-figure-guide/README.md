# Inspiration Case Figure Guide

这个 skill 用来把论文核心的“为什么要做”转化为 inspiration 图。它帮助作者从案例、失败、观察或对比中提炼动机，形成适合论文开篇或答辩使用的视觉叙事，让读者快速理解研究价值与必要性。

## 设计动机

很多论文的方法图能说明“怎么做”，但读者最先需要理解的是“为什么这件事值得做”。如果动机只写在文字里，问题场景、失败现象和关键观察往往不够直观。这个 skill 的目标是把论文的动机压缩成一张可信、清楚、有记忆点的图，让读者在进入方法细节前先看见研究问题本身。

它特别适合用来表达：

- 一个能暴露研究空缺的具体案例
- 现有方法的失败或局限
- 前后状态、旧方法与新需求的对比
- 某个观察如何引出研究假设
- 审稿人可能关心的边界案例

## 主要步骤

1. 收集论文材料  
   先确认论文主题、目标位置和想表达的动机来源。

2. 判断图的类型  
   根据材料判断更适合做案例走查、失败说明、前后对比、观察启发、场景故事板或设计空间缺口。

3. 提炼读者要看到的重点  
   明确这张图要回答的核心问题：为什么这篇论文是必要的。

4. 给出多种候选方向  
   先比较不同叙事方式，而不是过早锁定单一画法。

5. 生成并比较候选图  
   用多张候选图判断哪种视觉方向最清楚、最可信、最适合论文位置。

6. 整理定稿说明  
   在选定方向后，补齐标题、图注、图例和正文中的衔接文字。

## 使用建议

使用时最好提供论文摘要、引言片段、关键案例或失败现象。如果已经有参考图，也可以说明想借鉴它的风格、布局、密度或标注方式。最终图中的案例、数据和结论应始终来自论文材料本身，不应为了视觉效果而添加未验证内容。
