# behavior-intervention-plan-drafter

Draft a BACB Ethics Code–compliant Behavior Intervention Plan (BIP) for an ABA therapy client, ready for supervising BCBA review and approval before implementation.

## Overview

Board Certified Behavior Analysts (BCBAs), BCaBAs, and behavior technicians under BCBA supervision must produce individualized, function-based Behavior Intervention Plans for every client receiving ABA services. A compliant BIP must include operational definitions of target behaviors, a synthesized functional behavior assessment (FBA), function-matched antecedent modifications, replacement behavior teaching procedures, consequence strategies, a crisis/de-escalation protocol, and a generalization and maintenance plan.

Without a structured process, BIPs are often function-mismatched, omit required components, or include consequence strategies that are not reinforcement-based — creating compliance risk with the BACB Ethics Code and payer documentation requirements.

This skill guides the complete intake and produces a DRAFT BIP labeled for supervising BCBA review and sign-off before implementation with any client.

## Use When

- A BCBA, BCaBA, or supervised behavior technician needs to draft a new BIP from FBA data
- An existing BIP requires a major revision (new function identified, new target behaviors, or treatment failure)
- A treatment team onboarding process requires a structured BIP intake document before the BCBA's clinical review

## Out of Scope

- Aversive consequence procedures, response cost, or any restrictive procedure plan — these require a separate document with enhanced oversight
- Experimental functional analysis (FA) design or execution — this skill synthesizes existing FBA data
- Psychotropic medication recommendations
- Diagnosis or re-evaluation reports
- Insurance authorization letters or billing documentation

## Workflow Summary

1. Client identification and consent confirmation (PII-safe intake)
2. Target behavior operational definitions and baseline data
3. FBA synthesis: function, antecedents, maintaining consequences, function confidence rating
4. Antecedent modification strategies (function-matched)
5. Replacement behavior selection and teaching procedures (function-matched reinforcement)
6. Consequence strategies (reinforcement and response strategies; safety note on restrictive procedures)
7. Crisis and de-escalation protocol
8. Generalization and maintenance plan
9. Data collection and progress monitoring plan
10. DRAFT BIP assembly with BCBA authorization block

## Output

A DRAFT Behavior Intervention Plan with:
- Target behavior operational definition table with baseline data
- FBA summary with function confidence rating
- Antecedent strategy table
- Replacement behavior and teaching procedure table
- Consequence strategies table
- Crisis protocol narrative
- Generalization and maintenance plan
- Data collection and monitoring plan
- BCBA authorization sign-off block

All output is labeled **DRAFT — NOT APPROVED FOR IMPLEMENTATION** until the supervising BCBA reviews and signs off.
