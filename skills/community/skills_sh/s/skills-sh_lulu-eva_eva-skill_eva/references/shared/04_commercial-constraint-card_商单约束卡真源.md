# Eva-skill 商单约束卡真源

本文件是商单约束卡、商单内容任务卡和二者转换规则的唯一真源。

使用方式：

- `/eva-brief` 输出、校验或补齐商单约束卡时读取本文件。
- `/eva-title` 做商单标题搜索、候选判断或把商单约束卡追加为商单内容任务卡时读取本文件。
- `/eva-script` 在第一句话链路中把商单约束卡转换为商单内容任务卡时读取本文件。
- `02a2/02a3/02b` 承接商单约束、检查真实体验缺口或判断成稿置信度时读取本文件。

商单约束卡不是标题交接卡、第一句话交接卡或正文入口。它只能提供品牌硬要求、卖点边界、真实素材缺口和合规风险。

## 商单约束卡

正式商单约束卡字段：

```markdown
## 商单约束卡

- 产品：
- 品类：
- 品牌目标：
- 目标用户：
- 必须保留：
- 可以转述：
- 禁止触碰：
- 推荐主讲卖点：
- 可以轻轻带过：
- 不建议主讲：
- 学员可用素材：
- 需要补充的真实体验：
- 合规风险：
- 可搜索验证词：
- 标题限制：
- 不可承诺内容：
- 可迁移样本结构：
- 推荐内容入口：
- 推荐接力模块：/eva-title / /eva-script 开头分支
```

正式输出商单约束卡前，至少必须有：

- 产品。
- 目标用户。
- 推荐主讲卖点。
- 必须保留。
- 禁止触碰。
- 不可承诺内容。
- 学员可用素材字段。
- 可搜索验证词。
- 推荐内容入口或推荐接力模块。

缺产品、目标用户、推荐主讲卖点、必须保留、禁止触碰、不可承诺内容或可搜索验证词时，停在 `/eva-brief` 补 Brief，不进入标题或正文。

缺少“学员可用素材”这个字段时，停在 `/eva-brief` 补字段。

字段存在但值为“未提供 / 暂无 / 待补充 / 已授权回捞-未命中 / 主动回捞-未命中 / 未命中 / 未扫描-无目录 / 未扫描-无权限 / 未扫描-未执行”时，可以进入低置信度标题搜索或低置信度路线图，但不得进入高置信度正文成稿。

## 进入标题/第一句话链路

商单约束卡输出后，先判断内容入口，不直接进正文：

- 小红书、有标题、有封面标题或平台不清 -> 接 `/eva-title` 做爆款标题搜索或候选判断。
- 抖音、视频号、无标题口播、前 5 秒、第一句话或不要封面标题 -> 接 `/eva-script` 的开头分支，先形成第一句话交接卡。

只有已经存在合格的标题交接卡或第一句话交接卡时，商单约束卡才可以转换为商单内容任务卡，并随同内容交接卡进入 `/eva-script`。

## 商单内容任务卡

商单内容任务卡是正文链路里的约束字段，不是正文入口。

```markdown
## 商单内容任务卡

- 产品：
- 目标用户：
- 推荐主讲卖点：
- 必须保留：
- 可以转述：
- 禁止触碰：
- 不可承诺内容：
- 学员可用素材：
- 需要补充的真实体验：
- 合规风险：
- 可迁移样本结构：
- 推荐内容入口：
```

商单约束卡转换为商单内容任务卡时：

- 不得删除必提、禁区、不可承诺内容和真实体验缺口。
- 不得把“推荐主讲卖点”改成新的选题承诺。
- 不得把“学员可用素材”里的未命中状态改写成已有体验。
- 不得用商单内容任务卡替代标题交接卡或第一句话交接卡。

缺产品、目标用户、推荐主讲卖点、必须保留、禁止触碰、不可承诺内容或学员可用素材字段时，不转换，退回 `/eva-brief`。

## 真实素材和置信度

学员可用素材状态按 `02_asset-state_资产状态归一表.md` 归一。

命中或已提供真实素材时，只能使用卡片或本轮输入里的真实字段。

未提供、未命中或未扫描时：

- 商单标题搜索可以继续，但必须标注低置信度。
- 正文路线图可以继续，但成稿置信度不得为高。
- 使用感受、经历、效果、反馈位置必须标注 `[待补充真实使用感受]`。
- 不得编造“我用了多久”“朋友亲测”“数据提升多少”“用户反馈很好”。

## 已有商单稿检查

商单原稿改稿交接卡只描述原稿与 Brief 的冲突和下一步接力，不替代商单约束卡。

已有标题交接卡或第一句话交接卡，且商单约束卡字段齐全时，才可以接 `/eva-script`；仍必须先经过正文逻辑链推理和 `02a3` 正文路线图。

只有标题文本、第一句话文本或普通原稿，没有内容交接卡时，按平台回 `/eva-title` 或开头分支补入口。

## 禁止

- 禁止把商单约束卡当成正文入口。
- 禁止没有标题交接卡或第一句话交接卡就转换商单内容任务卡。
- 禁止绕过 `/eva-brief` 自行补品牌必提、禁区或不可承诺内容。
- 禁止用低置信度商单卡输出可发布终稿。
- 禁止因为用户催稿就跳过真实体验缺口。
