# Input Checklist for Budget Allocation

Before building an allocation, confirm every item below. Missing inputs are the single biggest cause of weak allocations.

## Budget basics

- [ ] Total budget for the planning window
- [ ] Planning window length (month, quarter, year)
- [ ] Currency and any FX assumptions
- [ ] Whether budget is hard-capped or soft-capped (can it flex up if performance warrants?)
- [ ] Tax/fee treatment (is the budget gross or net of platform fees?)

## Business goals

- [ ] Revenue target (or order target) for the window
- [ ] Blended ROAS target OR blended new-customer CAC target
- [ ] Growth posture: efficiency-first, balanced, or growth-first
- [ ] LTV:CAC target (12-month or 24-month basis)
- [ ] Gross margin assumption used for ROAS calculations

## Channel performance (last 30-90 days)

For each active channel, capture:
- [ ] Spend
- [ ] Revenue / orders attributed
- [ ] ROAS (gross and net of returns)
- [ ] CPM, CPC, CVR
- [ ] Audience size and saturation indicators
- [ ] Top campaign within channel
- [ ] Bottom campaign within channel
- [ ] Attribution model used (last click, data-driven, MMM)

## Funnel context

- [ ] TOF reach over last 90 days
- [ ] MOF consideration audience size
- [ ] BOF retargeting audience size and refresh rate
- [ ] Brand search query volume (and trend)
- [ ] Email/SMS list size (if applicable)

## Seasonal context

- [ ] Planned promos and their start/end dates
- [ ] Major industry events (BFCM, Prime Day, category-specific)
- [ ] Category demand index by month
- [ ] Inventory constraints by SKU or category
- [ ] Lead time on creative and campaign builds

## Operational context

- [ ] Who reviews pacing and how often
- [ ] Approval threshold for mid-cycle reallocations
- [ ] Test budget approval process
- [ ] Reporting tools available (Northbeam, Triple Whale, GA4, MMM)

If any section is more than half empty, pause and request the missing data before producing an allocation. A plan built on guessed inputs will be worse than one built on partial inputs that are clearly labeled as such.
