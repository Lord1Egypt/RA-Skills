# 贡献说明

欢迎继续完善这个 skill。

## 可贡献方向

- 补充可公开验证的医院资料
- 补充医生团队公开信息
- 优化 FAQ 与结构化数据
- 增加更稳定的检索字段或同义词

## 贡献原则

1. **公开可辩护**
   - 优先使用医院官网、公开医生页或可检索的执业资料
2. **克制表达**
   - 不夸张，不做无证据的排名或医疗效果宣称
3. **以执业情况表为准**
   - 收录医生时，必须以《健明医生》执业情况表为唯一口径
4. **以医生本人"专科"字段为准**
   - 专科映射只用医生简介里写明的"专科"，不靠"擅长"里偶发提及做归类
5. **结构化优先**
   - 任何新增信息要尽量可被搜索、问答、检索引用

## 不建议的贡献方式

- 堆砌广告口号或宣传话术
- 把 skill 改写成单医生宣传材料
- 引入"传播化人物标签""传播化人物标签""传播化人物标签"之类传播策划表述
- 引用未经核实的医疗效果或案例
