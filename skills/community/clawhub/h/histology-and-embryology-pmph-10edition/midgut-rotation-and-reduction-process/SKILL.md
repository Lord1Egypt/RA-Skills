# 中肠袢旋转与回纳流程

## 何时使用
- 分析胚胎消化系统发育（第6–10周）
- 评估肠管最终解剖位置
- 排查肠转位异常（如肠旋转不良、肠扭转）的胚胎学基础

## 执行步骤
1. **第6周启动机制**：因肠生长迅速且腹腔容积小，中肠袢突入脐腔（生理性脐疝）。
2. **首次旋转（90°）**：在脐腔内，以肠系膜上动脉为轴，从腹面观**逆时针旋转90°**，使头支移至右侧，尾支移至左侧。
3. **第10周回纳**：腹腔容积增大，肠袢按**头支先、尾支后**顺序回纳入腹腔。
4. **继续旋转（180°）**：回纳过程中再逆时针旋转180°，**总旋转270°**。
5. **确定最终位置**：
   - 头支 → 空肠与回肠大部分（腹腔中部）
   - 尾支 → 结肠（腹腔周边）
   - 盲肠突从肝下移至右髂窝，形成盲肠与阑尾
   - 降结肠尾段移向中线形成乙状结肠

> 注意：本技能仅适用于正常胚胎发育；若旋转异常，需结合其他诊断技能评估畸形类型。
