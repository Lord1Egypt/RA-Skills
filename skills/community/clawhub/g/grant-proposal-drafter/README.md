# Grant Proposal Drafter

**Platforms:** Claude · Openclaw · Codex
**Domain:** Nonprofit

## Purpose

Turns a funder RFP and a project brief into a structured, compliance-checked grant proposal draft. Extracts every funder requirement, gathers organizational and project context, drafts each required section, and produces an RFP-to-section traceability matrix so reviewers can verify compliance at a glance.

## When to Use

- Drafting a foundation, federal, state, or corporate grant proposal in response to an RFP, NOFO, or LOI invitation
- Converting a concept note or program brief into a full proposal narrative
- Rapidly identifying and flagging missing compliance information before submission
- Producing a reviewer-ready draft for an executive director, program lead, or grants committee
- Building a traceability matrix that maps every funder requirement to a proposal section

## What It Does

**Phase 1: Decompose the RFP**
1. Collects the funder RFP/NOFO/guidelines and extracts every requirement
2. Builds a requirements register: required sections, page/word limits, attachments, eligibility criteria, evaluation rubric, formatting rules, and submission logistics
3. Flags eligibility risks before proceeding

**Phase 2: Gather Project and Organizational Context**
4. Collects project context one question at a time: problem, target population, geography, goals, activities, timeline, evaluation plan, budget total, key staff, organizational capacity, prior grants
5. Maps each piece of context to one or more requirements in the register
6. Identifies and flags every funder requirement that lacks supporting input

**Phase 3: Draft and Verify**
7. Drafts each required section in the funder's preferred order, respecting page/word limits and rubric weights
8. Builds an RFP-to-section traceability matrix with a status per requirement (covered / partial / missing)
9. Produces unresolved-info flags and a pre-submission checklist for organizational review

## Output

A complete grant proposal draft containing: cover sheet placeholder, executive summary, statement of need, project description, goals and measurable objectives, methods/activities, evaluation plan, sustainability plan, organizational capacity, budget narrative, attachments checklist, RFP compliance matrix, and unresolved-information flags. Ready for nonprofit staff review and refinement before submission.

## Notes

The skill never invents grant outcomes, statistics, prior award amounts, or organizational facts not provided by the user. Citations and external statistics are flagged for human verification. Funder confidentiality clauses, embargoed program information, or unpublished evaluation data shared during the session are not used in examples, tool calls, or external searches.
