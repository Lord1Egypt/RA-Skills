## Description: <br>
Generate videos using OpenAI's Sora API. Use when the user asks to generate, create, or make videos from text prompts or reference images. Supports image-to-video generation with automatic resizing. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and creators use this skill to generate MP4 videos from text prompts or optional reference images through OpenAI's Sora API. It supports prompt guidance, model selection, duration and resolution choices, polling, and local download of generated media. <br>
