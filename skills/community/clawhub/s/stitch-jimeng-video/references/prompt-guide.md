# 即梦AI 视频生成 — 提示词参考手册

> 基于 Seedance/即梦 同源引擎的提示词最佳实践

---

## 核心原则：细节消灭模糊

**每个镜头必须包含三个具体物理细节：**

1. **环境压力** — 冷藏柜的蓝绿光、湿漉漉的柏油路、闪烁的日光灯、水壶的蒸汽
2. **身体微动作** — 下巴收紧、指关节发白、嘴唇抿成一条线、艰难地咽口水
3. **声音锚点或视觉母题** — 手机屏幕的反光、同一扇窗户上的雨滴、每次切镜前的日光灯闪烁

**禁用词汇（不会渲染，暴露偷懒）：**
- ❌ "电影感"、"专业"、"高质量"、"杰作"、"震撼"、"史诗"
- ❌ "美丽的光影"、"动态镜头"、"紧张时刻"、"强大场景"
- ❌ "他很难过"、"她很生气" — 没有身体动作的情绪描写

**替代方式：**
- ❌ "他很难过" → ✅ "下巴收紧，手指抠住门框，艰难地咽了一下口水"
- ❌ "美丽的厨房" → ✅ "冷白色日光灯从头顶打下，不锈钢台面上凝着水珠"

## 六步提示词公式（快速场景）

```
主体：[谁或什么]
动作：[现在时态动词，一个清晰动作]
镜头：[运动 + 构图 + 焦段]
环境：[地点、道具、氛围]
光线：[方向 + 质量 + 颜色]
风格：[写实程度、类型参考]
```

**用完整句子写，不要用标签堆砌。** 即梦偏好清晰的电影化散文。

## 11格生产级骨架（重要场景必用）

```
[1] 角色锁定
    使用 @img1 作为主角参考，保持完全一致的人物：
    <脸型、瞳色、肤色、发型、发色、胡须、体型、标志性特征>
    穿着：<具体服装>。无裸露。无眼镜（除非参考图有）。
    无额外角色。

[2] 时长 + 类型 + 剪辑意图
    生成一个 <时长> 秒的多镜头 <类型> 序列，<快/慢/阶梯式> 动态剪辑。

[3] 故事（一段话）
    <这个片段中具体物理事件的现在时态描述。从头到尾发生了什么变化。
    命名转折点。>

[4] 视觉风格
    <色盘、对比度、颗粒感、色温、要避免什么>
    <纹理提示——写实皮肤、食物细节、织物、表面>

[5] 镜头风格
    <机身/质感（如 Sony FX3 手持感）>
    <焦段及用途：35mm/50mm 中景，85mm 情感特写，100mm 微距，24mm 全景>
    <手持微晃或固定。镜头间硬切。不要连续长镜头。>

[6] 剪辑风格
    <硬切 vs 匹配剪辑。节奏升级。停顿在哪里。冲击点在哪里。>
    <节奏阶梯：长 → 更短 → 更短 → 停顿 → 冲击>

[7] 声音
    <环境音、微动作音效、冲击音、静默时刻、最终提示音>
    <例如：冰箱嗡鸣、远处雨声、2.3秒时肚子咕噜一声、突然安静>
    无对白 / 无字幕 / 无屏幕文字

[8] 逐镜头时间线
    镜头1，0.0-X.X秒：<景别、焦段、镜头运动、动作、环境细节、情绪的身体表达>
    镜头2，X.X-Y.Y秒：<...>
    ...

[9] 光线（回顾与细节）
    <主光源、补光、轮廓光。方向。色温。它在心理上承载什么>

[10] 构图
    <主体在画面中的位置。负空间。反光。剪影。前景遮挡。
    最终画面必须命名。>

[11] 输出规格
    <精确时长。宽高比。写实程度。>
```

## 5秒镜头节奏模板

```
0.0-0.8秒 | 建立     | 极近特写或插入镜头，锚定情绪/情境
0.8-1.6秒 | 动作     | 中景，主角移动或反应
1.6-2.5秒 | 转折     | 新构图揭示变化（主观、过肩、焦点转换）
2.5-3.6秒 | 反应     | 紧凑特写，缓慢推进，情绪落在身体上
3.6-5.0秒 | 高潮     | 英雄镜头，低角度，慢动作（如果配得上），最终画面
```

## 多镜头语法（即梦独有能力）

即梦可以在一次生成中创建多个不同镜头。使用切镜标记：

```
镜头1. [描述]
切到. [描述]
镜头切换到. [描述]
```

**5秒内建议2-3个镜头。** 4-5个镜头需要每个镜头都很短且物理上独特。超过6个镜头会压缩成不可读的运动。

## 首尾帧特别规则

使用首尾帧模式时：
- **不要重复图片中已有的元素** — 模型看得到
- **只描述运动和镜头运动**
- 首帧和尾帧之间的过渡需要有明确的物理动作驱动

❌ 错误："一个穿红衬衫的男人站在厨房里。他走向冰箱。"
✅ 正确："他犹豫地走向冰箱，打开门，看到空荡荡的货架时僵住了。从背后跟拍，35mm。"

## 情绪→身体翻译表

| 情绪 | 身体表现 |
|------|---------|
| 恐惧 | 下巴收紧，停止呼吸一个节拍，手指抠住门框 |
| 悲伤 | 肩膀塌陷，视线低垂半寸，嘴唇微微颤抖 |
| 愤怒 | 指关节发白握紧，嘴唇抿成直线，呼吸变重 |
| 内疚 | 目光闪避，手指无意识摩挲手机屏幕，吞咽 |
| 希望 | 眼睛微微睁大，嘴角不自觉上扬一点点，呼吸变轻 |
| 孤独 | 身体前倾缩成一团，手机蓝光照亮面部，周围大片空白 |

| 光线 | 心理效果 |
|------|---------|
| 冷白日光灯 | 审判、官僚、恐惧 |
| 冰箱蓝绿光 | 饥饿、孤独、深夜 |
| 暖黄台灯 | 安全、怀旧、脆弱 |
| 逆光剪影 | 神秘、威胁、距离 |
| 闪烁荧光灯 | 衰败、不安、精神紧张 |
| 窗光+雨影 | 压抑的悲伤、内疚 |
| 顶部冷光 | 审判、被注视 |

## 节奏阶梯

```
慢热剧情：4秒、4秒、3秒、2秒、1秒、停顿、2秒
产品弧线：3秒、2秒、1.5秒、1秒、0.5秒（产品微距）、2秒（英雄镜头）
焦虑升级：2秒、1秒、1秒、0.5秒、0.5秒、0.3秒、停顿、1秒
冲击场景：停顿、0.2秒（闪）、2秒（静止余韵）
```

**规则：在最大冲击点之前，必须有至少一个停顿。没有停顿的速度就是视觉绞肉机。**

## 五个锚点（每个短片必须命名）

1. **一个核心情绪**
2. **一个视觉母题**（反复出现的视觉元素）
3. **一个锚定物体**
4. **一个断裂点**
5. **一个最终画面**

示例：
- 情绪：内疚
- 母题：主角不断被玻璃表面反射（窗户、手机屏幕、电梯门）
- 锚定物体：一条未读消息的手机
- 断裂点：他删除了消息
- 最终画面：他的脸倒映在熄灭的手机屏幕上
