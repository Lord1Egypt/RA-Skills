## Description: <br>
Transform AI agents from task-followers into proactive partners that anticipate needs and continuously improve with WAL Protocol, Working Buffer, Autonomous Crons, and tested operating patterns. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent users use this skill to configure a proactive personal-agent workflow with persistent memory, onboarding, heartbeat checks, security review patterns, and self-improvement routines. It is intended for agents that maintain workspace files and propose or perform helpful follow-up work with user oversight. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill enables broad persistent memory and automatic capture of user context. <br>
Mitigation: Review and edit memory/profile files before use, avoid storing secrets, and require explicit approval before retaining sensitive personal or project information. <br>
Risk: The skill describes autonomous background work, crons, isolated agents, and private email or calendar checks. <br>
Mitigation: Disable these behaviors by default or require explicit opt-in for each automation, account access, and spawned-agent workflow. <br>
Risk: The skill includes cleanup authority such as closing apps or moving files to trash. <br>
Mitigation: Require confirmation before cleanup, deletion, file movement, or other irreversible local actions. <br>
