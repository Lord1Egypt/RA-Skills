# 城市美食数据库格式说明

## 文件位置
`data/city_food_db.json`

## 用途
当百度地图API不可用时，作为兜底方案提供该城市的代表性美食推荐。

## 数据结构

```json
{
  "cities": {
    "城市名": {
      "specialties": ["特色菜系1", "特色菜系2", "代表菜品1", "代表菜品2"],
      "famous_restaurants": [
        {
          "name": "餐厅名",
          "type": "菜系类型",
          "area": "所在商圈/区域",
          "signature": ["招牌菜1", "招牌菜2"],
          "budget": "人均消费范围",
          "description": "餐厅简介"
        }
      ]
    }
  }
}
```

## 示例

```json
{
  "cities": {
    "杭州": {
      "specialties": ["杭帮菜", "西湖醋鱼", "东坡肉"],
      "famous_restaurants": [
        {
          "name": "楼外楼",
          "type": "杭帮菜",
          "area": "西湖区",
          "signature": ["西湖醋鱼", "东坡肉"],
          "budget": "人均120-200",
          "description": "百年老字号"
        }
      ]
    }
  }
}
```

## 扩展建议

如需支持更多城市，按相同格式添加即可。建议优先覆盖：
1. 一线城市（北上广深）
2. 新一线城市（杭州、成都、南京等）
3. 旅游热门城市（西安、厦门、丽江等）
