# 梅花诗

## 概述

《梅花诗》，传为宋代邵雍（邵康节）撰，是一部**预言宋朝至未来国运**的谶纬之作。全诗共十首，每首预言一朝代，以梅花为题，含蓄隐晦。

---

## 作者简介

**邵雍**（1011-1077），字尧夫，号安乐先生，谥号康节。北宋著名理学家、数学家、命理学家、预言家。著有《皇极经世》《铁板神数》《梅花易数》《梅花诗》等。

---

## 成书背景

邵雍精于易理，感悟天机，撰《梅花诗》十首，预言宋朝至未来国运。全诗以梅花为题，含蓄隐晦，需细品方知其意。

## 核心内容

### 一、梅花诗全文

**其一（宋）**：
```
荡荡天门万古开，几人归去几人来。
山河虽好非完璧，不信黄金是祸胎。
```

**其二（元）**：
```
湖山一梦事全非，再见云龙向北飞。
三百年来终一日，长天碧水叹弥弥。
```

**其三（明）**：
```
天地相乘数一原，忽逢甲子又兴元。
年华二纪全堪笑，搔首荒唐欲问天。
```

**其四（清）**：
```
三百年来事不顺，虎头带犬乱纷纷。
平安吉庆其家福，天人庆会时方春。
```

**其五（民国）**：
```
如棋世事局初残，共济和衷却大难。
豹死犹留皮一袭，最佳秋色在长安。
```

**其六（未来）**：
```
火龙飞去海波平，万里浮云一旦收。
多少胡人留此土，如何草木起戈矛。
```

（其余四首，预言更远未来）

### 二、预言内容

**宋朝**：
1. 天门开，天人往来
2. 山河非完璧，外患不绝
3. 黄金祸胎，经济繁荣下的危机

**元朝**：
1. 湖山一梦，南宋灭亡
2. 云龙向北飞，宋室南迁
3. 三百年终一日，元朝国祚

**明朝**：
1. 天地相乘，朱明兴起
2. 甲子又兴元，周期性兴替
3. 二纪全堪笑，兴衰无常

**清朝**：
1. 三百年事不顺，清朝国祚
2. 虎头带犬，满清入关
3. 天人庆会，康乾盛世

**民国**：
1. 世事如棋，军阀混战
2. 共济和衷，国共合作
3. 豹死留皮，英雄逝去

**未来**：
1. 火龙飞去，科技昌明
2. 海波平，世界和平
3. 胡人留此土，民族融合

## 预言示例

### 示例一：宋朝

**预言**：
```
荡荡天门万古开，几人归去几人来。
山河虽好非完璧，不信黄金是祸胎。
```

**解读**：
1. 天门开：指宋朝开国
2. 山河非完璧：指外患不绝（辽、金、元）
3. 黄金祸胎：指经济繁荣下的危机

### 示例二：未来

**预言**：
```
火龙飞去海波平，万里浮云一旦收。
多少胡人留此土，如何草木起戈矛。
```

**解读**：
1. 火龙飞去：指科技昌明（火龙喻科技）
2. 海波平：指世界和平
3. 胡人留此土：指民族融合
4. 草木起戈矛：指未来仍有冲突
