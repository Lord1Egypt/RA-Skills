# Integrative Unstable Angina Treatment

## When to Use
- 患者已确诊为不稳定型心绞痛
- 正在接受或计划启动西医“双抗+降脂”基础治疗
- 存在典型心前区压榨样疼痛等症状
- 难以长期服用西药或出现明显药物不良反应

## Execution Steps

1. **确认西医诊断与基础治疗**
   - 明确患者因冠状动脉病变导致心肌急性缺血缺氧
   - 启动或维持标准西医治疗：双抗（如阿司匹林+氯吡格雷）联合他汀类降脂药

2. **评估中医外治适用性**
   - 排除禁忌：急性心肌梗死发作期、严重出血倾向、对针灸/贴敷过敏者禁用或慎用
   - 辨证分型：根据阳虚寒凝、痰瘀互结等证型选择对应外治方法

3. **联合中医外治法**
   - **针灸**：取内关、膻中、心俞等穴，每周3–5次，每次20–30分钟，以通经活络
   - **艾灸**：适用于阳虚寒凝证，温灸关元、气海，每日1次，7–10天为一疗程
   - **穴位贴敷**：将丹参、川芎等活血化瘀中药膏贴于心俞、厥阴俞，每日更换
   - **耳穴压豆**：取心、神门、交感等耳穴，用王不留行籽按压，每日3–5次，每穴1分钟

4. **监测与评估**
   - 观察心绞痛发作频率、持续时间及硝酸甘油用量变化
   - 定期复查心电图及心功能指标
   - 评估是否实现“活血化瘀、化痰降浊”治疗目标，改善心肌供血

> **注意**：中医外治仅为辅助手段，不可替代急性期西医急救措施。
