# Wind-Cold Lung vs Exterior Differentiation

## When to Use
- 患者初诊为感冒或外感咳嗽；
- 同时存在咳嗽和典型风寒表证表现（恶寒、发热、无汗、头身疼痛）；
- 需明确病位以决定治法（宣肺 vs 解表）。

> **注意**：不适用于纯内伤咳嗽或无表证者；若发热重于恶寒，应考虑风热证而非本技能适用范围。

## Execution Steps

1. **判断主症重心**：
   - 若**咳嗽、气喘为主症**，即使伴有轻微恶寒发热 → 判定为**风寒束肺证**；
   - 若**恶寒发热为主症**，咳嗽轻微或无 → 判定为**风寒表证**（太阳伤寒）。

2. **对比辅助症状**：
   - **风寒束肺证必见**：咳嗽气喘、痰稀色白、鼻塞流清涕、咽痒；
   - **风寒表证核心**：恶寒重、发热轻、无汗、头身疼痛显著，咳嗽可有可无且轻微。

3. **评估舌脉**：
   - 两者均可见舌苔薄白、脉浮紧，**不可单独依赖舌脉区分**。

4. **确定病位**：
   - 风寒束肺证：病位在**肺**，肺卫失宣；
   - 风寒表证：病位在**肌表**，卫阳被郁。

5. **指导治疗**：
   - 风寒束肺证 → 宣肺止咳散寒（如杏苏散）；
   - 风寒表证 → 解表发汗（如麻黄汤）。

## Output
若咳嗽为主，即使有轻微表证，也应诊断为风寒束肺证，治疗重点在宣肺而非单纯解表。
