# AI记忆术指南

从记忆宫殿到世界记忆大师。名字/数字/考试全覆盖。

## 核心能力

### 1. 记忆原理
- **艾宾浩斯遗忘曲线**：20分钟后遗忘42%/1小时后56%/1天后74%→必须在遗忘点前复习
- **间隔重复Spaced Repetition**：第1天→第3天→第7天→第14天→第30天(黄金复习节律)
- **工作记忆=7±2**：记住任意序列的能力约5-9项/用组块打包
- **图像记忆>>抽象记忆**：大脑进化用来记地点、面孔、食物→不是用来记数字的
- **神经可塑性**：伦敦出租车司机考"知识"Knowledge/海马体后部明显增大
- **情绪+记忆**：情绪越强烈/记忆越深刻/所以编故事
- **记忆≠智商**：世界记忆冠军的平均智商约110(略高于平均但不是天才)

### 2. 记忆宫殿Method of Loci(核心大技)
- **原理**：利用大脑强大的空间记忆/把你熟悉的地点与要记住的信息绑定
- **构建宫殿5步**：
  1. 选熟悉的地点(你家/你的学校/你的上班路线)
  2. 规划一条固定路线(按顺时针/不重复)
  3. 沿途选10-20个"记忆桩"(门→鞋柜→沙发→电视→餐桌→厨房水槽……)
  4. 把要记住的每个项目→变成疯狂/夸张/动感的图像→放在记忆桩上
  5. "走"你的宫殿→提取每个桩上的图像=提取信息
- **图像规则**：越大越好/越荒谬越好/越详细越好/动起来比静止好/与自己有关比他人有关好
- **例子**：记购物清单"牛奶/面包/苹果"→ 门(一大瓶牛奶倒在门上)/鞋柜(巨大的法式长棍当鞋穿)/沙发(一颗巨大的苹果坐在沙发上看电视)
- **多少个宫殿**：可以建很多个/每个30-100个桩/轮流用
- **刷新桩**：用过一次的桩→过1-2天旧图像自然消失/可以重用

### 3. 数字记忆
- **Major System(主流数字编码)**：数字→辅音→加元音→变成单词或图像
  - 0=S/Z(soft C)/1=T,D/2=N/3=M/4=R/5=L/6=J,SH,CH/7=K,G(hard)/8=F,V/9=P,B
  - 例：12=TN→TiN(锡罐)/34=MR→MaRe(母马)/79=KB→CaB(出租车)
- **PAO系统(Person-Action-Object)**：每两位数字=一个人物+动作+物品(记忆竞赛顶级选手使用)
- **Dominic System**：00-99/每位数字=人名首字母(A=1/B=2……Z=26)→每两位=一个人
- **卡片编码**：52张牌=52个图像/黑桃A=某名人/红心7=某动作
- **扑克牌记忆**：世界纪录(约12秒记一副52张)/用卡片编码/每3张牌组成一个图像=17个图像→放进17个桩上
- **数字编码示例简易版(00-99)**：00=眼镜/01=树/02=鸭子/03=弹簧/……99=胶囊
- **记日期和历史**：1865→图像组合→放在历史时间轴上

### 4. 名字与面孔记忆
- **见面90秒内复述名字3次**
- **将名字转化为图像**："王磊"→一个"王"和一堆"雷"(磊=石+雷)
- **将图像贴在对方脸上**：额头上/鼻子上/下巴上
- **特征连接**：对方戴眼镜→名字图像与眼镜结合
- **最难记的名字**：外国名字/无意义音节→强拆成中文图像
- **聚会/面试前预习**：拿到名单→提前做图像→见面时已存储
- **名字记忆的社交价值**：记住对方名字=最高级别的尊重

### 5. 外语词汇记忆
- **关键词法Keyword Method**：外语发音→找发音相近的中文词→编入图像
  - 例：西班牙语"perro(狗)"→发音像"贝罗"→贝多芬养了一条罗威纳
- **词根词缀**：拉丁/希腊词根=记忆的积木
- **Anki/SRS软件**：间隔重复的自动化/自动给你该复习的词
- **语境记忆**：单词放在句子/故事里/而非孤零零的词典
- **多感官**：读+写+听+说+画/5个通道同时输入
- **不背单词≠不记忆**：大量阅读+听=自然记忆/有上下文的单词更容易留在脑中

### 6. 学习与考试应用
- **费曼技巧**：学了以后/假装教给别人(8岁孩子)/哪里解释不下去=哪里没真懂
- **主动回忆Active Recall**：不看书的回忆/比反复读有效4倍
- **思维导图Mind Map**：层级化/颜色/图片/一页看完
- **SQ3R阅读法**：Survey(浏览)→Question(提问)→Read(读)→Recite(复述自己)→Review(复习)
- **"编故事"记公式**：每一个公式的符号=一个角色/角色之间发生故事
- **讲课录音**：自己讲→录→睡前听
- **考前策略**：先建记忆宫殿/把要点放进去/进考场后在纸上画桩→逐个提取
