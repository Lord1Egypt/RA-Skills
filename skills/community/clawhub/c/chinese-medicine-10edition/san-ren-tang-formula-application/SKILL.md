# San Ren Tang Formula Application

## When to Use
当满足以下全部条件时启用本技能：
- 证属湿温初起
- 邪在气分
- 湿重于热
- 主症包括头痛恶寒、身重疼痛、胸闷不饥、午后身热、舌白不渴、脉弦细而濡
- 排除纯热无湿或阴虚内热证

## Execution Steps
1. **确认方剂组成**：严格按照《温病条辨》记载配伍——杏仁、白豆蔻、薏苡仁、厚朴、半夏、通草、滑石、竹叶。
2. **理解核心配伍逻辑**：
   - 杏仁宣利上焦肺气
   - 白豆蔻芳香化湿、畅中焦脾气
   - 薏苡仁渗湿健脾、疏导下焦
   - 厚朴、半夏行气燥湿和胃
   - 滑石、通草、竹叶清利湿热
3. **掌握煎服要点**：白豆蔻宜后下以保留挥发油；常规水煎服。
4. **明确治疗原则**：体现“宣上、畅中、渗下”三焦分消法，重在宣畅气机、清利湿热。

> 注意：本方仅适用于湿重于热之证，不可用于纯热或阴虚内热者。
