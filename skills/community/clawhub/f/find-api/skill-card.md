## Description: <br>
Find API helps agents identify reliable data APIs for external data needs across financial, weather, news, maps, e-commerce, and related domains instead of relying first on web scraping. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[yesilsin-netizen](https://clawhub.ai/user/yesilsin-netizen) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, engineers, and agents use this skill to choose practical API sources, authentication approaches, and basic request patterns before fetching external data. It is most useful when structured data access is preferable to scraping. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad trigger words may cause the skill to activate for generic API, data source, or scraping requests. <br>
Mitigation: Confirm that external data-source selection is actually needed before applying the skill's recommendations. <br>
Risk: Recommended third-party APIs can involve costs, keys, quotas, or source reliability tradeoffs. <br>
Mitigation: Confirm the selected provider, pricing, authentication requirements, and data suitability before making calls or installing packages. <br>
