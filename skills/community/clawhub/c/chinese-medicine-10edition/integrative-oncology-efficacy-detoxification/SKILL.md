# Integrative Oncology Efficacy and Detoxification

## When to Use
- 患者正在接受放疗、化疗、靶向治疗或免疫治疗，出现明显毒副反应（如骨髓抑制、消化道反应、肝肾功能损伤等）
- 西医治疗进入平台期或疗效受限，需联合中医手段提升整体抗肿瘤效果
- 患者希望在规范西医治疗基础上，通过中医药调理改善生活质量、延长生存期

## Execution Steps
1. **评估西医治疗方案及当前阶段**：明确患者所处的治疗周期（诱导、巩固、维持）、用药种类及已出现的毒副反应。
2. **辨证分型**：依据中医理论对患者进行四诊合参，确定证型（如气阴两虚、脾肾阳虚、瘀毒内结等）。
3. **制定增效策略**：选择具有扶正固本、清热解毒、活血化瘀等功效的中药复方或单味药，以增强机体免疫功能或直接抑制肿瘤生长。
4. **实施减毒干预**：针对具体毒副反应选用对应中药（如黄芪、当归改善骨髓抑制；姜半夏、竹茹缓解恶心呕吐；茵陈、栀子保护肝功能）。
5. **动态调整方案**：根据治疗反应和证候变化，定期调整中药配伍与剂量，确保安全有效。

## Precautions
- 必须在规范西医肿瘤治疗基础上使用，不可替代手术、放化疗等标准疗法
- 警惕中药与西药潜在相互作用（如影响CYP450酶代谢），必要时咨询临床药师
- 强调个体化治疗，避免套用固定方剂

> 提示：若需具体方剂组成、剂量参考或毒副反应分级标准，请查阅相关参考资料。
