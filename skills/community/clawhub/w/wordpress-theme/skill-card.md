## Description: <br>
Guides agents through converting static HTML, CSS, and JavaScript websites into WordPress themes while preserving design fidelity and adding WordPress theme integration patterns. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[junice](https://clawhub.ai/user/junice) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agents use this skill when converting static sites or CMS migrations into WordPress themes with expected templates, menus, enqueueing, Customizer settings, security checks, and testing steps. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: WordPress security examples may be incomplete for custom fields, attributes, URLs, IDs, or arbitrary user-controlled output. <br>
Mitigation: Review generated theme code and separately verify escaping and sanitization for the specific WordPress data sources being used. <br>
