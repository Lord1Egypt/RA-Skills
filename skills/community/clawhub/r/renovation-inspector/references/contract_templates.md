# 装修补充协议条款模板库

> 本模板仅供参考，不构成法律建议。重大纠纷建议咨询律师。
> 使用方法：根据业主需求组合条款，填写【】内的占位符，作为合同附件签署。

---

## 一、补充协议封面模板

```
装修施工补充协议

甲方（业主）：【全名】          身份证号：【号码】
乙方（装修公司）：【公司全称】   统一信用代码：【代码】
项目地址：【完整地址】
主合同签订日期：【日期】

本协议为双方主合同的补充，与主合同具有同等法律效力。如本协议与主合同条款冲突，以本协议为准。
```

---

## 二、材料品牌锁定条款

```
第X条 材料品牌与规格约定

1. 主材清单（详见附件一）中列明的所有材料，乙方必须按指定品牌、型号、规格采购，不得以"同等品"或"替代品"替换，除非取得甲方书面同意。

2. 材料进场前，乙方须提前24小时通知甲方，甲方有权在材料进场时现场核验品牌标签、合格证及检测报告。

3. 进场材料不符合约定的，甲方有权拒收，乙方须在【3】个工作日内重新提供合格材料。因此导致的工期延误，由乙方承担。

4. 乙方须在施工完工后，向甲方提供主材的购买凭证（发票或收据）。
```

---

## 三、水电工程包干/封顶条款

```
第X条 水电工程费用约束

1. 水电工程（含所有点位、管线、线盒安装）以包干价计，总额不超过人民币【X】元。包干价已包含图纸设计内的全部点位及合理余量。

2. 若甲方在施工过程中主动增加点位，经甲方书面确认后方可计入增项，每增加一处插座/开关按【X】元计，每增加一处水路接口按【X】元计。

3. 乙方不得以"实际用料超预期""图纸未标注部分"等理由要求追加费用，除非甲方已书面同意该增项。
```

---

## 四、防水工程质量保证条款

```
第X条 防水工程验收与质保

1. 防水施工完成后，乙方须在蓄水前通知甲方，由甲方或其委托人在场监督闭水试验。

2. 闭水试验标准：卫生间蓄水深度不低于20mm，静置不少于48小时，试验期间楼下无渗漏痕迹为合格。试验结果须拍照存档，由甲乙双方在场确认。

3. 防水工程质量保修期不少于【5】年，自竣工验收合格之日起计算。保修期内出现渗漏，乙方须在接到通知后【5】个工作日内到场处理，并承担全部维修费用（含对楼下造成的损失）。

4. 若乙方未在甲方在场情况下进行闭水试验，视为乙方放弃该试验结果的证明效力，日后如发生渗漏，举证责任由乙方承担。
```

## 五、增项管控条款

```
第X条 增项确认程序

1. 施工过程中如乙方认为需要增加工程量或费用，须提前以书面形式（书面通知或微信文字，发至甲方【联系方式】）说明：增项原因、具体内容、工程量、单价及总价。

2. 甲方须在收到增项申请后【3】个工作日内书面回复是否同意。甲方未回复的，视为不同意。

3. 未经甲方书面同意擅自施工的增项，甲方有权拒绝支付该部分费用。

4. 工程增项总金额不得超过主合同金额的【15%】；如超出，须重新签署增项合同。
```

## 六、工期与违约金条款

```
第X条 工期约定与违约责任

1. 工程总工期为【X】个日历天，开工日期：【日期】，约定竣工日期：【日期】。

2. 因乙方原因（非甲方要求变更、非不可抗力）导致工期延误的，每逾期一天，乙方须向甲方支付违约金人民币【X】元（建议为合同总额的0.1%-0.3%/天）。

3. 因甲方原因（如主材供货延误）导致的工期延误，双方协商顺延，不计违约金。

4. 工期延误责任的认定，以乙方向甲方的书面工期报告为依据。
```

---

## 七、竣工验收与尾款条款

```
第X条 竣工验收与付款程序

1. 乙方在认为施工完成后，须提前【3】个工作日书面通知甲方进行竣工验收。

2. 验收由甲乙双方共同进行，验收标准以国家规范GB 50327及合同约定为准。

3. 验收通过后，甲方须在【7】个工作日内支付尾款。

4. 如验收发现不合格项，乙方须在甲方约定的期限内（不少于【5】个工作日）完成整改，整改完成后再行验收，验收通过后再付尾款。

5. 尾款不得少于合同总额的【10%】，作为质量保证金，验收合格后全额支付。
```

---

## 八、材料环保指标条款

```
第X条 材料环保要求

1. 乙方提供的所有木质板材（细木工板、多层板、饰面板、密度板等）须达到E1级标准（甲醛释放量≤1.5mg/L），乙方须在材料进场时提供相应检测报告或合格证。

2. 乳胶漆、胶粘剂等涂料须符合GB 18582、GB 18583等强制性环保标准，乙方须提供相应合格证。

3. 竣工后，甲方有权委托第三方机构进行室内空气质量检测，若不达标（依据GB/T 18883《室内空气质量标准》），乙方须承担整改及复检费用。
```

## 使用说明

1. **组合使用**：根据实际谈判需求选取对应条款，不需要全部使用
2. **填写要点**：【】内的金额/天数/比例需双方协商后填入，不宜留空签署
3. **保存证据**：签署后拍照或扫描，各自保存电子版
4. **附件一**：主材清单应单独列表，作为本协议附件，双方逐页签字/盖章
