# Emblem Vault Prompt Examples

## Vault Discovery

```text
List my Emblem Vaults and group them by status.
Show vault 12345 with balances, collection details, and deposit addresses.
List my unminted Emblem Vaults.
Refresh the live balance for this vault and summarize what it contains.
```

## QuickVault And Creation Flows

```text
QuickVault 2 of my PEPECASH, but stop for confirmation first.
Vault 3 of my TESTNETPEPE and wait for my approval before executing.
Create a new Darkfarms vault and tell me the deposit address.
Create a vault for this curated collection and show the next step after creation.
List available vault collections before creating anything.
```

## Minting, Revealing, And Cleanup

```text
Mint my unminted vault as an NFT on Ethereum.
Explain the risks before revealing keys for vault 12345.
Reveal keys for vault 12345 only after showing the warning and tradeoff.
Delete this empty vault after confirming it has no assets.
```

## Intent Disambiguation And Safety

```text
Show vault 4616027839.
Vault my PEPECASH.
Review whether this is a QuickVault request or a vault-details request before taking action.
Explain why my Hustle account private key cannot be revealed, and point me to the safer backup option.
```
