## Description: <br>
Reconstruct 3D scenes from single images using depth estimation, producing depth maps, point clouds, and meshes. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[moroiser](https://clawhub.ai/user/moroiser) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to guide local workflows that reconstruct 3D scene structure from satellite, aerial, or standard photos using Depth Anything 3. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The workflow depends on external Python packages, a pretrained model, and local project scripts that may change or process sensitive images. <br>
Mitigation: Run it in a virtual environment, verify package and model sources, and review referenced project scripts before using important or sensitive images. <br>
