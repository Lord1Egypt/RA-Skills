## Description: <br>
将剧本转为AI分镜提示词。当用户提供剧本并要求生成分镜、AI提示词、漫剧/动画提示词时使用。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External creators, storyboard artists, and video producers use this skill to convert user-provided scripts into structured AI storyboard prompt segments for AI video, animation, comic-drama, or trailer workflows. It helps preserve original dialogue while planning scene anchors, character positions, action beats, visual treatment, and sound cues. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may activate for broad script or AI-prompt requests where the user did not intend a screenplay-to-storyboard conversion. <br>
Mitigation: Confirm the user wants script-to-storyboard prompt generation before applying the workflow. <br>
Risk: User-provided scripts may include material the user does not have rights or permission to process. <br>
Mitigation: Use the skill only with scripts the user is authorized to provide and transform. <br>
Risk: Generated storyboard prompts may change pacing, structure, or emphasis even when preserving original dialogue. <br>
Mitigation: Review the generated segments against the source script before using them in production. <br>
