# 取费标准（山东安装工程）

> **适用标准**：
> - 国标：《建设工程工程量清单计价标准》GB/T 50500-**2024**（2025年9月1日起山东实施）
> - 省级：《山东省建设工程费用项目组成及计算规则》（**2025年版**，2025年11月1日起施行）
> - 省级定额：《山东省安装工程消耗量定额》（2025版，2025年11月1日起执行）
> - 市级：《济南市价目表(2026版)》
>
> 本文件为权威取费依据，组价计算时必须引用本章参数。

---

## 0. 重大变化：规费科目取消（GB/T 50500-2024）

> ⚠️ **2025年11月1日起执行**
>
> 依据 GB/T 50500-2024《建设工程工程量清单计价标准》：
> - **取消"规费"科目**
> - 社会保险费、住房公积金并入人工费和企业管理费
> - 费用结构简化：人工费 + 管理费（含社保统筹）+ 利润 + 税金
> - 不再单独计算规费费率

---

## 1. 费率总览

| 费用名称 | 费率 | 说明 |
|---------|------|------|
| 管理费 | **48.08%** | 以**人工费**为基数（山东省安装工程标准） |
| 利润 | **29.15%** | 以（**人工费**+管理费）为基数 |
| 税金 | **9%** | 增值税一般纳税人 |
| 综合费率（管理费+利润） | **91.25%** | (1+48.08%)×(1+29.15%)-1，**仅适用于人工费口径** |
| 含税综合系数 | **2.0846** | 1.9125 × 1.09，**仅适用于人工费口径** |

> **计算基础**：管理费以**人工费**为基数；利润以（人工费+管理费）为基数。
>
> **重要区分**：
> - **基价** = 人工费 + 材料费 + 机械费（定额单价全口径）
> - **计费基础** = 人工费（仅人工费部分参与管理费和利润的计算）
> - 综合费率 91.25% 和含税系数 2.0846 只能乘以**人工费**，不能乘以**基价**
> - 税前综合单价 = 基价 + 管理费 + 利润 = 基价 + 人工费 × 综合费率
>
> 依据：《山东省建设工程费用项目组成及计算规则》2025年版 第三章费率表说明

## 2. 综合单价计算公式（2025版）

### 2.1 取费步骤（规费已取消）

```
已知：基价 = 人工费 + 材料费 + 机械费

Step 1: 管理费  = 人工费 × 48.08%
Step 2: 利润    = (人工费 + 管理费) × 29.15%
Step 3: 税前综合单价 = 基价 + 管理费 + 利润
                       = 基价 + 人工费 × 91.25%
                       ≠ 基价 × 1.9125  （错误！）
Step 4: 税金    = 税前综合单价 × 9%
Step 5: 含税综合单价 = 税前综合单价 + 税金
```

### 2.2 Excel 公式参考

| 列名 | Excel 公式 | 说明 |
|------|-----------|------|
| 管理费 | `=[@人工费]*48.08%` | 基数是人工费，不是基价 |
| 利润 | `=([@人工费]+[@管理费])*29.15%` | 基数是人工费+管理费 |
| 税前综合单价 | `=[@基价]+[@管理费]+[@利润]` | 基价+管理费+利润 |
| 含税综合单价 | `=[@税前综合单价]*(1+9%)` | 加增值税 |
| 税金 | `=[@税前综合单价]*9%` | |
| 合价 | `=[@工程量]*[@含税综合单价]` | |

## 3. 快速计算工具

### 3.1 综合单价速查表

> 假设人工费占基价的比例为 80%（安装工程常见），则：
> 管理费+利润 = 人工费 × 91.25% = 基价 × 80% × 91.25% = 基价 × 73.00%
> 税前综合单价 ≈ 基价 × 1.73

| 人工费(元) | 管理费 | 利润 | 管理费+利润 | 基价参考(元) | 税前单价参考(元) | 含税单价参考(元) |
|:---------:|:-----:|:----:|:---------:|:----------:|:-------------:|:-------------:|
| 100 | 48.08 | 42.99 | 91.07 | 130 | 221.07 | 240.97 |
| 200 | 96.16 | 85.98 | 182.14 | 260 | 442.14 | 481.93 |
| 500 | 240.40 | 214.95 | 455.35 | 650 | 1105.35 | 1204.83 |

> 注：以上"基价参考"按人工费/基价=77%估算（安装工程典型比例）。
> 实际使用时必须按定额中的人工费、材料费、机械费分别计算。

### 3.2 Excel 速算公式

```
管理费 = 人工费 × 48.08%
利润   = (人工费 + 管理费) × 29.15%
税前综合单价 = 基价 + 人工费 × 91.25%
含税综合单价 = 税前综合单价 × 1.09

注意：不能用"基价 × 2.0846"计算含税综合单价！
正确公式：含税综合单价 = (基价 + 人工费 × 91.25%) × 1.09
```

## 4. 人工费价差调整

| 项目 | 说明 |
|------|------|
| 调整依据 | 济南市建设工程人工费指导价（2026） |
| 调整方式 | 按实调整，计入税前造价 |
| 适用标准 | GB/T 50500-2024 第×章价格调整条款 |

> 注：人工费价差调整计入税前工程造价，不参与管理费和利润的计算。

---

## 5. 材料费价差调整

| 项目 | 说明 |
|------|------|
| 调整依据 | 济南市材料信息价（《济南工程造价信息》月刊） |
| 调整方式 | 按施工期平均信息价与定额基价之差调整 |
| 税金处理 | 价差部分按规定税率（9%）计算税金 |

## 7. 安装与生产同时施工增加费

按人工费的 **10%** 计取（含管理费、利润，不含税金）。

---

## 8. 在有害身体健康环境中施工增加费

按人工费的 **26%** 计取（含管理费、利润，不含税金）。
