# 胆道闭锁的病因与病理机制

## 适用场景
- 新生儿存在进行性黄疸、白陶土样大便
- 影像学或核素扫描提示胆道梗阻
- 需排除非梗阻性胆汁淤积、代谢性或感染性肝炎

## 执行步骤
1. **确认疾病特征**：胆道闭锁是一种发生于胎儿后期、生后早期及新生儿期的进行性病变，表现为肝内和/或肝外胆管阻塞，导致胆汁排泄通道完全或部分梗阻。
2. **分析病因构成**：
   - 约80%由围产期病毒感染引发的胆道炎症所致；
   - 约10%源于先天性胆管发育不良；
   - 其余10%可能与其他因素（如毒性、免疫等）相关。
3. **理解病理过程**：感染 → 胆管炎症 → 纤维化闭锁 → 胆汁淤积 → 肝损伤 → 肝硬化。
4. **关键鉴别点**：
   - 炎症性闭锁通常有TORCH等感染史；
   - 发育不良型多伴其他先天畸形。
5. **临床意义**：“围产期病毒感染”是主要可干预窗口，强调早期识别感染标志物的重要性。

## 输出结果
提示胆道闭锁高度可能，建议尽快行肝胆核素扫描（如99mTc-EHIDA）及病因筛查。
