# 购车流程指南

## 一、选车阶段

### 1. 确定需求
- [ ] 预算范围（落地价）
- [ ] 用车场景（通勤/家用/自驾游）
- [ ] 车型偏好（轿车/SUV/MPV）
- [ ] 动力类型（油车/混动/纯电）
- [ ] 必备配置（L2智驾/座椅加热/全景天窗等）

### 2. 初选车型
- 搜索该价位热门车型
- 查看销量排行（月销5000+为安全线）
- 筛选3-5款目标车型

### 3. 深度对比
- 参数对比（懂车帝/汽车之家）
- 口碑调研（车主论坛、投诉平台）
- 实际续航（新能源重点关注）

## 二、试驾阶段

### 试驾准备
- 提前预约（周末人多）
- 带上驾照
- 试驾路线包含：市区道路、快速路、颠簸路面

### 试驾关注点

**驾驶感受：**
- [ ] 起步/加速是否平顺
- [ ] 刹车脚感
- [ ] 转向手感（轻/重/虚位）
- [ ] 底盘滤震（过减速带感受）
- [ ] 噪音控制（60km/h、80km/h）

**乘坐感受：**
- [ ] 后排空间（膝盖、头部）
- [ ] 座椅舒适度
- [ ] 视野盲区

**新能源专属：**
- [ ] 能耗显示是否真实
- [ ] 充电接口位置
- [ ] 智驾功能体验

## 三、谈价阶段

### 价格构成
```
落地价 = 裸车价 + 购置税 + 保险 + 上牌费 - 优惠
```

**新能源车**（30万以内）：免购置税

### 谈价技巧
1. **货比三家** — 同城至少问3家店
2. **问落地价** — 不要只看裸车价
3. **分期优惠** — 问清楚总利息+手续费
4. **赠送项目** — 脚垫、贴膜、保养（价值有限）
5. **库存车** — 库存超3个月可砍价

### 砍价幅度参考
| 车型 | 通常优惠空间 |
|------|-------------|
| 热门燃油车 | 5-10% |
| 热门新能源 | 3-5%（政策优惠外） |
| 冷门车型 | 10-15%+ |
| 库存车 | 额外3-5% |

## 四、订车阶段

### 订车合同要点
- [ ] 车型、颜色、配置明确
- [ ] 落地价明细
- [ ] 预计提车时间
- [ ] 延迟交付赔偿条款
- [ ] 赠送项目写进合同
- [ ] 贷款方案（利率、期数、月供）

### 定金 vs 订金
- **定金**：法律规定不退（慎重！）
- **订金**：可退（建议用这个）

## 五、提车阶段

### 提车检查清单

**外观：**
- [ ] 漆面无划痕、色差
- [ ] 轮胎无磨损、生产日期（6个月内）
- [ ] 玻璃无裂纹

**内饰：**
- [ ] 座椅无破损
- [ ] 各按键功能正常
- [ ] 里程数（50km内正常）

**随车物品：**
- [ ] 备胎/补胎工具
- [ ] 三脚架、反光背心
- [ ] 说明书、保养手册
- [ ] 两把钥匙

**新能源专属：**
- [ ] 充电线缆
- [ ] 充电桩安装服务确认

### 证件确认
- [ ] 购车发票
- [ ] 车辆合格证
- [ ] 车辆一致性证书
- [ ] 保险单
- [ ] 贷款合同（如适用）

## 六、上牌流程

### 无锡上牌
1. 购置税申报（新能源免）
2. 交强险生效
3. 车管所验车
4. 选号（线上12123或现场）
5. 领取行驶证、登记证

**新能源上牌**：直接上绿牌，无摇号限制
