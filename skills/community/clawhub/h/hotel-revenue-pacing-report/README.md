# Hotel Revenue Pacing Report

Draft a weekly or monthly revenue pacing analysis report with occupancy, ADR, RevPAR, segment analysis, and competitive set benchmarking.

## Overview

Hotel revenue managers must produce regular pacing reports that compare current booking position to budget, prior year, and competitive set — and translate those findings into actionable pricing and channel strategy recommendations. This skill guides structured data intake and produces a DRAFT pacing report for revenue manager review before distribution to ownership, asset management, or the general manager.

## Use Cases

- Weekly revenue management pacing reports
- Monthly performance and forecast reviews
- Budget variance analysis (actual vs. budget vs. prior year)
- Group displacement analysis
- Forward-demand review for upcoming high-demand dates
- STR or OTA Insight competitive index reporting

## Audience

- Hotel Revenue Managers and Directors of Revenue Management
- General Managers with revenue management responsibilities
- Asset managers and ownership representatives
- Corporate revenue management analysts overseeing multi-property portfolios

## What the Skill Produces

1. **Property and period summary block** — hotel name, type, star category, comp set, reporting period
2. **Historical performance table** — occupancy, ADR, RevPAR (actual vs. budget vs. prior year) with variance %
3. **Pacing analysis** — on-the-books position vs. STLY and budget with RED/YELLOW/GREEN pacing tier flags
4. **Segment performance breakdown** — transient, group, corporate, OTA production vs. budget
5. **Competitive set benchmarking** — OCC Index, ADR Index, RevPAR Index (RGI); MPI/ARI flags where available
6. **Forward demand review** — upcoming compression dates, events, demand generators
7. **Pricing and strategy recommendations** — rate strategy by date range, channel recommendations, restriction flags
8. **Revenue manager review block** — data source citations, reviewer sign-off, and distribution list

## Key Metrics

| Metric | Definition |
|--------|------------|
| ADR | Average Daily Rate |
| RevPAR | Revenue Per Available Room |
| RGI | Revenue Generation Index (property RevPAR ÷ comp set RevPAR) |
| MPI | Market Penetration Index (property OCC ÷ comp set OCC) |
| ARI | Average Rate Index (property ADR ÷ comp set ADR) |
| STLY | Same Time Last Year |

## Safety Boundaries

- All revenue projections are labeled PRELIMINARY ESTIMATES — not financial commitments.
- Competitive set index data must cite source (STR, OTA Insight, Duetto, etc.); fabricated benchmarks are never acceptable.
- Pricing strategy recommendations require revenue manager approval before deployment in any property management system (PMS) or channel manager.
- Seasonal comparisons spanning holidays or major demand shifts must include an adjustment flag.
- This skill does not replace the revenue manager's market judgment — it structures analysis inputs and report output.
