# AccountingOnFeishu

> 飞书多维表格智能记账助手 —— 支持文字记账、图片小票OCR识别，首次使用自动学习你的记账习惯。

## ✨ 功能特性

- **📝 文字记账**：一句话记账，支持数学表达式（如 `100*0.8`）
- **📷 图片 OCR**：拍照小票自动识别，批量记账
- **🧠 智能学习**：首次使用读取你的现有记账本，学习字段结构和分类习惯
- **🏗️ 一键建表**：没有记账本时自动创建飞书多维表格
- **🎯 自动分类**：根据关键词智能匹配一级/二级分类
- **📅 日期自动**：日期和月份自动填充，无需手动填写

## 🚀 安装

```bash
openclaw skill install accounting-on-feishu
```

或通过 ClawHub 安装：
```
https://clawhub.ai/YOUR_USERNAME/accounting-on-feishu
```

## 📖 使用方法

### 首次使用（初始化）

直接说：
```
我要记账
```

助手会询问：
1. **有现成记账本** → 提供飞书多维表格链接，助手学习你的字段结构
2. **没有记账本** → 引导创建新的记账本（简单/标准/完整版可选）

### 文字记账

```
记账 吃饭 50
记账 打车 23.5
收入 工资 5000
记账 超市购物 128 支出
```

**支持表达式：**
```
记账 分摊 150/3
记账 折扣 200*0.85
```

### 图片小票记账

发送图片并说：
```
记账
```

助手自动 OCR 识别小票上的每个项目，批量写入。

## 🗂️ 表格结构

默认创建的记账本包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| 备注 | 文本 | 消费描述（主键） |
| 日期 | 日期 | 自动填充 |
| 金额 | 数字 | 消费金额 |
| 收支 | 单选 | 收入 / 支出 |
| 分类 | 单选 | 餐饮、交通、购物、娱乐等 |
| 二级分类 | 单选 | 餐饮-日常/蔬果、餐饮-外卖等 |
| 月份 | 公式 | 自动计算 |
| 复盘 | 单选 | 必要 / 想要 / 稳定 / 风险 |

**一级分类选项：**
`收入`、`居住`、`餐饮`、`交通`、`购物`、`娱乐`、`健康`、`学习`、`社交`、`投资/储蓄`、`宠物`、`长辈`、`无匹配类别`

**二级分类示例：**
- `餐饮-日常/蔬果`、`餐饮-零食`、`餐饮-大餐/外卖`、`餐饮-饮品`
- `交通-停车`、`交通-油费电费`、`交通-保养维修`、`交通-路费`
- `购物-生活日用`、`购物-衣物鞋包`、`购物-电子产品`、`购物-兴趣爱好`
- `居住-房贷/房租/物业`、`居住-电费/水费`、`居住-通讯/其他`
- `宠物-洗护/医疗`、`宠物-狗粮/零食`、`宠物-用品`
- `健康-医疗/药品`、`健康-保健`
- `学习-书籍`、`学习-课程资料`
- `社交-礼物`、`社交-红包`、`社交-捐赠`
- `收入-工资`、`收入-补贴`

## ⚙️ 配置

### 使用现有记账本

提供你的飞书多维表格链接：
```
https://my.feishu.cn/base/APP_TOKEN?table=TABLE_ID
```

助手会自动学习你的字段名和分类体系。

### 配置文件

学习后的配置保存在：
```
~/.openclaw/workspace/skills/AccountingOnFeishu/config.json
```

### 自定义分类

如果你希望使用自己的分类体系，可以：
1. 在飞书中先创建好多维表格，设置好分类字段
2. 让助手学习你的表格结构

## ⚠️ 注意事项

1. **金额只写数字**，不要带 ¥ 或 元
2. **日期自动填充**，不需要手动填写
3. **复盘字段留空**，供用户后续手动复盘时填写
4. **图片小票**需清晰可辨认，系统会自动解析每个项目
5. **配置文件中不要包含敏感 token**（发布版本请删除 config.json，保留 config.json.example）
