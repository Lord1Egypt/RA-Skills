# ABO血型鉴定与交叉配血操作规范

## 何时使用
- 患者即将接受输血或器官移植
- 怀疑新生儿溶血病（尤其母O型、胎A/B型）
- 供血者与受血者血液相容性评估
- 受检者有输血史、妊娠史、输血反应史或曾生育新生儿溶血病患儿

## 如何执行

### 1. ABO血型鉴定（必须同时进行正定型与反定型）
- **正定型（Beth-Vincent法）**：将标准抗A、抗B、抗AB（即O型）血清分别与受检者红细胞混合，观察凝集反应。
- **反定型（Simonin法）**：将标准A、B、O型红细胞分别与受检者血清混合，观察凝集反应。
- **结果判定**：
  - A型：正定型抗A(+)、抗B(-)；反定型A细胞(-)、B细胞(+)
  - B型：正定型抗A(-)、抗B(+)；反定型A细胞(+)、B细胞(-)
  - AB型：正定型抗A(+)、抗B(+)；反定型A/B细胞均(-)
  - O型：正定型抗A(-)、抗B(-)；反定型A/B细胞均(+)
- **特殊要求**：
  - 必须加入O型血清以避免Ax亚型误判为O型
  - 加入O型红细胞可检测非ABO系统异常抗体（如冷凝集素）

### 2. 交叉配血试验
- **主侧**：受血者血清 + 供血者红细胞 → 检测受血者是否存在抗供者红细胞抗体
- **次侧**：供血者血清 + 受血者红细胞 → 检测供血者是否存在抗受血者红细胞抗体
- **结果判断**：
  - 主侧无凝集且次侧无凝集 → 配血相合，可输血
  - 主侧有凝集 → **绝对禁止输血**（无论次侧结果）
- **方法选择**：
  - 无输血/妊娠史者：使用盐水介质凝集试验
  - 有相关病史者：必须进行间接抗球蛋白试验（检测不完全抗体）
  - 大量输血（48小时内≥5L）：除常规交叉配血外，还需供者之间相互交叉配血

### 3. 临床注意事项
- “O型万能供者”存在风险：约30.2%的O型人含免疫性抗A/抗B（IgG），全血输注可致溶血
- “AB型万能受者”不绝对：A₁B型约3%含抗“O”抗体；A₂B型约25%含抗A₁抗体，输入相应亚型可致溶血
- 新生儿溶血病多见于母O型、胎A/B型，因母体IgG抗A/抗B经胎盘致病，首胎即可发生

> **警报规则**：若主侧凝集，立即发出“禁止输血”警报；若正反定型结果不符，提示需进一步鉴定亚型或排查异常抗体。
