# Part 7:诊断与干预

## 使用方式

读取本份的典型场景:
- 用户问"健康家庭电路画像是什么样"、"我们家是哪一型"
- 要求列出"电路层面的临床干预清单"
- 在 SKILL.md 主表已经定位完后,需要给出**完整的处方建议**(干预 = 调哪个元件)
- 用户讨论 negative affect reciprocity、self-fulfilling prophecy 等"开关被另一变量控制"的现象时,看末尾的**晶体管扩展**指针

不要一开始就把这一份倒出来。先用 `dc-and-zvector.md` 的 Z 平面五型表 + `conjugate-and-damping.md` 的拓扑判断,定到具体类型后再来这一份找处方。

## 健康家庭的电路画像(六维体检表)

把所有 RLC + 二极管 + 拓扑 + DC 综合起来,健康家庭对应六个同时成立的条件:

| 维度 | 健康指标 | 反向问诊 |
|---|---|---|
| **拓扑** | 并联共轭 —— 两人是互补的不同处理路径 | "你俩是同一种反应模式,还是互补的两种?" |
| **元件值** | R / L / C 三者均衡;\|Z\| 中等;\|φ\| 不大 | "压力来时家里是过载、扁平、还是有适度反应?" |
| **DC 偏置** | 稳定的正向 DC > AC 振幅(对应 Gottman 5:1) | "积极互动 ÷ 消极互动 ≥ 5 吗?" |
| **阻尼(Q)** | 适度 R 让特征根成为有适度实部的复共轭根 | "每次大冲突最长多久能真的翻篇?" |
| **谐振点 ω₀** | 不在常见外部压力频段 | "什么节奏的事最能引爆你们?这种事经常发生吗?" |
| **二极管** | 健康边界 + 适度积极错觉 + 完整 capitalization 配对 | "好消息他第一反应是兴奋还是泼冷水?背叛能不能扛?" |

任意一维失分都不致命;**两维以上同时失分**才进临床干预候选区。

## 不健康家庭的六类电路画像

每一类都有**主导失分维度**(粗体)+ 次发症状。同一个家庭可能落在多类(例如"燃烧 + 击穿"是常见的双重病理)。

| 类型 | 主要电路病理 | 临床表征 | Miller 章节落点 |
|---|---|---|---|
| **燃烧型** | **串联 LC + 小 R + 高 Q** | 大爱大恨周期循环到耗尽;"我们什么都同频" | ch03 similarity, ch11 volatile, ch08 浪漫之爱 |
| **室友型** | **大 R + 小 LC** | 没冲突也没温度;各自外部生活,家内空 | ch11 avoider, ch07 dyadic withdrawal 反例 |
| **隐忍型** | **大 C + 小 R** | 表面稳;蓄势直到电容击穿(突然离婚 / 躯体化) | ch10 hurt feelings, ch11 voice/exit |
| **惯性型** | **大 L** | 老套路应对一切新挑战;反激强 | ch01 attachment(僵化),ch14 维持机制失败 |
| **内耗型** | **并联 LC + R = 0** | 永动振荡;你侬我侬讲不清,话题闭环 | ch11 attributional conflict, ch04 重构记忆 |
| **击穿型** | **Zener 阈值过低 / 缺失** | 任何反向冲击直接破坏;Reactance 易激;背叛后无修复通道 | ch10 betrayal, ch12 intimate terrorism |

> 双重病理示例:**燃烧 + 击穿** = volatile couple + 缺乏 Forgiveness Zener;每次和好后 DC 又被下一次反向尖峰打回去 → 解体倒计时。**惯性 + 隐忍** = 大 L + 大 C 的家庭;能装但更新极慢,典型华人家庭代际剧本。

## 六手干预:把家庭治疗技术翻译成电路操作

家庭治疗大部分技术从电路角度看就是这六类操作之一(或组合)。临床上**不存在"这一手最有效"**,要看具体失分维度。

### ① 加大 R —— 开拓外部耗散通道

**目标病理**:燃烧型、内耗型、串联 LC、Q 极高

**电路含义**:把能量散到家外面去,不让所有情绪都在家内循环。

**临床技术**:
- **结构上扩大社交圈**(各自的朋友、家人、同事)
- **个体化兴趣 / 运动 / 爱好**
- **个人咨询 + 夫妻咨询并行**
- **有意识的独处时间**(不是冷战,是恢复电荷)
- **数字断联时段**(不让情绪 24/7 直流通)

**Miller / 文献**:Bowen differentiation of self;ch14 维持机制中的"独立 + 联结"。

**注意**:R 加得太大会变室友型。**目标是适度阻尼,不是无情绪流动**。

### ② 改 L —— 改写僵化模式

**目标病理**:惯性型、依恋僵化(尤其 dismissing / preoccupied)、Demand-Withdraw 死循环

**电路含义**:家庭的"反应惯性"是长年沉积的 L 值,不会因一次谈话就变 —— 必须**走低频通道**(电感对低频通)。

**临床技术**:
- **认知行为干预**(CBCT)针对自动化反应模式
- **原生家庭工作 / Bowen genogram**
- **角色脚本重写**(Schema therapy)
- **持续半年以上的小步骤新行为**(L 的累积过程,不可瞬时)

**Miller / 文献**:ch01 attachment 修改路径;ch14 Michelangelo 现象(双方相互调谐 L)。

**关键:不要尝试瞬间切换 L** —— 强行加速或刹车都会出反激火花(摔门 / 冷战 / 躯体化)。

### ③ 改 C —— 扩容承压或建立泄洪

**目标病理**:隐忍型、过低 C 易爆发、过高 C 待击穿

**电路含义**:压力蓄水池 —— 既要"装得下日常",也要"装满前定期泄洪"。

**临床技术**:
- **扩容**:情绪命名训练、emotion regulation skills(DBT)、正念
- **泄洪**:**定期"大事化小"沟通仪式**(Gottman 每周对话、State of the Union)、写日记、疗愈性诉说
- **预击穿预警**:让伴侣识别"我快撑不住"的早期信号

**Miller / 文献**:ch05 self-disclosure;ch11 voice 路径。

**注意**:**只扩 C 不开泄洪 = 制造更大的延后击穿**。两者必须同时做。

### ④ 稳 DC 偏置 —— 维护承诺与共同身份

**目标病理**:DC 衰减(disillusionment)、AC > DC(危险区)、击穿后工作点掉到负

**电路含义**:DC = 承诺 + 长期共同利益 + 深层依恋,这是所有 AC 扰动叠加的基线。**抬 DC 通常比压 AC 更直接**。

**临床技术**:
- **Gottman 软启动 / 修复尝试**(repair attempts)
- **共同记忆唤起**(老照片、共同纪念日)
- **共同未来想象 / 共同意义建构**(shared meaning)
- **积极错觉的有意维护**(Murray:看见伴侣的好)
- **承诺仪式**(纪念日、共同投资:房、孩、宠物,这些都是 L 增大也是 DC 充电)

**Miller / 文献**:ch06 Rusbult commitment;ch08 Sternberg 三角的 commitment 维度;ch13 barrier model;ch14 cognitive interdependence。

**这是危机干预的优先项** —— 当 \|AC\| 已经接近 \|DC\| 时,先抬 DC 再说其他。

### ⑤ 改拓扑 —— 从串联到并联

**目标病理**:燃烧型(灵魂伴侣即焚)、过度融合(fusion)、缺乏 differentiation

**电路含义**:把"两人在同一回路里共振"重构为"两条互补支路在节点处自然相消"。

**临床技术**:
- **差异化(differentiation)**:各自有独立朋友圈、爱好、职业身份
- **接受互补不是缺陷**:"我们的反应方式不一样"是好事,不是危机
- **角色非对称化**:同一压力,A 走电感支路(慢但持久),B 走电容支路(快但能储)
- **降 Q**:不每次都高强度复盘,允许有些事情就过

**Miller / 文献**:ch03 complementarity > similarity 在长期关系的优势;ch14 cognitive interdependence("一模一样"重构为"互补不同");Bowen differentiation。

**注意**:这是**最难的一手**(年量级),也是把燃烧型救成 validator 的唯一通道。

### ⑥ 调二极管 —— 修边界、修整流、修配对

**目标病理**:Stonewalling、Reactance、低 Zener、Capitalization 失配、被污染整流器

**电路含义**:RLC 处理不了的非对称、阈值、击穿、单向叙事 —— 这些都得调二极管。

**临床技术清单**(按二极管类型):

| 二极管病 | 干预 | 推荐演示 |
|---|---|---|
| Stonewalling 反向截止极致 | EFT 软化追加者 / Gottman 物理生理冷静 | app08 |
| Demand-Withdraw 半波整流 | 让 B 的二极管偶尔正向导通(回应训练) | app08 |
| Capitalization 配对失败 | 训练 Active-Constructive 响应风格 | app09 |
| 低 Zener / Reactance 易激 | 提高反向耐受;识别触发器 | app10 |
| 高 Zener(扛但不放) | **主动激活 Zener**:让背叛能受控泄放,而非死扛 | app10 |
| 被污染整流器(自我服务/重构) | 共同回顾 / 第三方记录 / attributional reframing | — |
| 积极错觉缺失(抑郁性现实) | 整流 + 稳压器:有意识看见伴侣的好 | — |
| Sociometer photodiode 黑暗 | 重建外部社会光照(朋友 / 工作 / 兴趣) | — |

**Miller / 文献**:ch11 demand-withdraw;ch07 capitalization;ch10 forgiveness/betrayal;ch12 IPV;ch04 social cognition 偏差;ch01 sociometer。

## 干预排序的临床决策树

来访者一次咨询无法六手齐下。优先级原则:

1. **危机优先**:自伤 / 家暴 / 强迫性背叛 → 先安全资源,**不在电路框架内解决**。
2. **DC < AC 优先抬 DC**(④):基线没了,其他都白做。
3. **拓扑明显错的优先改拓扑**(⑤):串联无救于内部,先并联化。
4. **二极管病优先调二极管**(⑥):一只坏二极管比 RLC 失衡更具诊断意义。
5. **R 是最易调的**(①):任何阶段都可以加。
6. **L 是最慢调的**(②):放在长程治疗里,不指望几次谈话改 L。
7. **C 扩容必须配泄洪**(③):单做一边制造更大的击穿。

## 模型的边界

| 适用 | 不适用 |
|---|---|
| 近线性区段 / 稳态运行 | 极端创伤、PTSD 的非系统性闯入 |
| 边界、阈值、不对称(二极管擅长的部分) | 测量学(具体量表的信度效度) |
| 振荡 / 谐振 / 击穿 / 整流的结构性预测 | 神经生化具体机制(虽 sociometer 有脑区证据) |
| 跨章节连接、教学、咨询师内部建构 | 替代实证心理学治疗、临床诊断、危机干预 |

**核心提醒**:类比是**结构化思考工具**,不是物理事实。当用户问"L 的具体单位是多少 H"时,提醒这是同构而非同一,聚焦于**结构性预测**(振荡 / 谐振 / 击穿模式)而不是数字。

## 下一步扩展:晶体管(放在视野里)

源文件 `circintimate.md` 末尾留下了一个明确的扩展指针:**晶体管**。

晶体管的关键性质是**一方状态调控另一方的导通**(基极小电流控制集电极大电流)。这正好对应一类 RLC + 二极管都难以表达的现象 —— **"开关被另一变量控制"**:

| 现象 | 晶体管语义 |
|---|---|
| **Negative affect reciprocity**(Gottman) | A 的负面情绪状态作为"基极信号",**调控** B 的二极管导通阈值 → 越吵越易吵 |
| **Self-fulfilling prophecy**(社会认知) | 信念作为基极偏置,**放大**新进证据的导通方向 |
| **依恋激活模型**(secure base 假说) | 安全感水平作为基极,**门控**探索行为电路的开放程度 |
| **Triangulation 三角化**(Bowen) | 第三方作为基极,**调控**夫妻间的电流流动 |
| **Therapeutic alliance** | 治疗关系作为外部基极,**临时**调高 Forgiveness Zener 阈值 |

**目前 skill 不展开晶体管**;但当用户提出"开关被另一变量控制"型现象时,可以**指向这个扩展点**:"这是 RLC + 二极管碰到的边界,需要引入晶体管才完整。"
