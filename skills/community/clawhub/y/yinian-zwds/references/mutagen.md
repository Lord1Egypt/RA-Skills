# 四化体系 — 一念紫微斗数

## 什么是四化

四化 = 化禄 + 化权 + 化科 + 化忌

**有四个来源：**
1. 生年四化（本命四化）— 天干决定，一生不变
2. 飞星四化（宫干飞化）— 各宫天干飞入他宫
3. 大限四化 — 大限天干引发的四化
4. 流年四化 — 流年天干引发的四化

## 四化性质

### 化禄
- 性质：增加、融合、顺利
- 宫位：该宫主题和谐顺畅
- 吉凶：大吉，舒缓凶象

### 化权
- 性质：增加、主导、掌控
- 宫位：该宫主题主控性强
- 吉凶：吉带刚，掌权势

### 化科
- 性质：文雅、名声、调和
- 宫位：该宫主题有文化气息
- 吉凶：吉善，化解风波

### 化忌
- 性质：缺失、纠葛、收缩
- 宫位：该宫主题有缺憾
- 吉凶：凶，需注意的领域

## 飞星派四化补充

- **宫干飞四化**：本宫天干飞出四化到各宫
  - 如：命宫天干戊，贪狼化禄入戌（财帛宫）
- **自化**：本宫星曜自带四化（天干飞化入本宫）
  - 表示该宫自我变动特质
- **追禄**：他宫化禄入某宫
- **追忌**：他宫化忌入某宫
