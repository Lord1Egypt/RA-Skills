## Description: <br>
Skill Lint is a skill quality acceptance and format review tool for checking Claude Code Skill structure, frontmatter, reference consistency, release versioning, business-flow depth, evaluability, and security risk. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[cat-xierluo](https://clawhub.ai/user/cat-xierluo) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and skill maintainers use Skill Lint after drafting or changing a Claude Code Skill to review publishing readiness, structural compliance, business-flow depth, evaluability, and security posture. It is intended for release checks, third-party skill reviews, migration assessments, and regression reviews rather than creating new skills or performing general code review. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated review reports or archived assessments can summarize private project structure, quality findings, or local review context. <br>
Mitigation: Review generated reports before sharing or committing them, and keep local review profiles or private context out of public artifacts unless explicitly intended. <br>
