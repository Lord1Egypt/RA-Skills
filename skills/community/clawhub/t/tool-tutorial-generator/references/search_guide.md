# 中文互联网平台搜索策略

## 平台特性与搜索技巧

### 百度搜索
- 优势：覆盖面广，适合找安装配置类文章
- 技巧：加上"教程"、"入门"、"安装"等限定词
- 注意：筛选近1-2年的内容，技术工具迭代快

### 知乎
- 优势：深度内容多，有系统性的学习经验和路线图
- 搜索：「XX 教程 推荐」「XX 学习路线」「XX 怎么学」
- 特点：高赞答案通常质量有保障

### B站
- 优势：视频教程丰富，适合跟随操作
- 搜索：「XX 教程 2025」「XX 手把手」「XX 实战」「XX 新手」
- 注意：关注播放量和发布时间判断质量

### CSDN
- 优势：技术类教程最密集，安装配置文章多
- 搜索：「XX 安装教程」「XX 环境配置」「XX 入门教程」
- 注意：注意文章时效性，优先选择近期的

### 掘金
- 优势：前端/全栈领域内容质量高，有深度技术文章
- 搜索：「XX 入门」「XX 实战」「XX 进阶」
- 注意：适合前端/Node/React/小程序等技术方向

### 微信公众号
- 优势：中文互联网最深度的长文来源之一
- 搜索：「XX 教程」「XX 使用技巧」「XX 效率提升」
- 注意：微信搜一搜可以搜公众号文章

### 小红书
- 优势：生活方式类工具（设计/效率/笔记）教程多
- 搜索：「XX 教程」「XX 使用」「XX 推荐」
- 注意：偏向非技术类工具

### GitHub
- 优势：开源项目、awesome列表、脚手架模板
- 搜索：「awesome XX」「XX starter」「XX template」

## 搜索关键词变体生成规则

输入：{工具名}

自动生成以下关键词组合：
1. `{工具名} 教程` / `{工具名} 入门` / `{工具名} 使用指南`
2. `如何学习{工具名}` / `{工具名} 从入门到精通`
3. `{工具名} 安装配置` / `{工具名} 下载`
4. `{工具名} 实战项目` / `{工具名} 案例`
5. `{工具名} 常见问题` / `{工具名} 踩坑` / `{工具名} 技巧`
6. `{工具名} 进阶` / `{工具名} 高级用法` / `{工具名} 最佳实践`
7. `{工具名} 教程 2026` / `{工具名} 最新教程`（加当年年份）

## 内容可信度评估

| 信号 | 权重 | 说明 |
|------|------|------|
| 发布时间 | 高 | 技术工具优先1年内内容 |
| 作者背景 | 中 | 官方/大V/活跃社区成员更可信 |
| 互动量 | 中 | 点赞/收藏/评论数反映社区认可 |
| 实操性 | 高 | 包含可执行命令/截图/步骤的更可信 |
| 原创度 | 中 | 原创>翻译>转载 |

## 搜索时的topic参数选择

- 编程/开发类工具 → `topic: "programming"`
- 设计/效率/办公类工具 → `topic: "technology"`
- 通用/生活类工具 → `topic: "general"`
- 学术/研究类工具 → `topic: "academic"`
