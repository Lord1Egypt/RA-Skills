# 喉返神经损伤的分类与恢复预测

## 何时使用
- 患者接受过涉及喉返神经区域的甲状腺或头颈外科手术
- 术后出现声音嘶哑、失音、呼吸困难或窒息风险
- 需要区分损伤机制并制定后续管理策略

> **注意**：本技能不适用于非手术性声带麻痹（如肿瘤压迫、病毒感染等）。

## 执行步骤

### 1. 判断损伤机制与类型
- **暂时性损伤**：由挫夹、牵拉、血肿压迫引起，神经结构完整未断裂
- **永久性损伤**：由神经被切断、缝扎等直接破坏连续性所致

### 2. 分析症状出现时间
- **术中或术后立即出现**：提示直接损伤（如切断、缝扎）
- **术后数日出现**：可能因瘢痕牵拉或迟发性压迫

### 3. 评估临床表现
- **单侧损伤**：主要表现为声音嘶哑
- **双侧损伤**：可导致失音、严重呼吸困难，甚至窒息；需立即评估气道，必要时行气管切开

### 4. 预测恢复与决策
- **暂时性损伤**：给予嗓音休息、理疗等支持治疗，通常在3～6个月内逐渐恢复
- **永久性损伤**：不可逆，需长期嗓音康复或考虑声带内移术等外科干预

### 5. 预防建议（用于未来手术规划）
- 推荐术中使用喉返神经监测（IONM）技术以降低损伤风险

## 输出指引
根据损伤类型和神经受累侧别，生成以下决策建议：
- 若为暂时性单侧损伤 → 观察+康复
- 若为永久性单侧损伤 → 转诊至嗓音专科
- 若为双侧损伤 → 立即评估气道，警惕窒息，准备紧急干预
