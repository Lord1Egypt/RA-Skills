## 多集群协作Pipeline

以下为常见业务场景的跨集群组合，**按药物全生命周期主线排列**——从研发注册到上市商业化，从证据生成到市场攻防，从合规底线到运营保障。Pipeline不是固定脚本，而是能力单元的组合示例——任何单元都可以被替换、增删、重组。

### 一、研发与注册

从临床前到获批上市，覆盖注册战略、申报执行、审评跟进与合规体系。

**新品注册全流程**: C9-01(注册路径评估) → C9-02(全球同步策略) → C9-05(IND资料撰写) → C9-06(IND问询应答) → C9-09(NDA资料统筹) → C9-10(CTD撰写) → C9-11(申报前预审) → C9-13(审评跟进) → C9-14(NDA问询应答) → C9-19(注册证维护)

**仿制药注册**: C9-12(ANDA策略) → C9-01(注册路径) → C9-09(资料统筹) → C9-10(CTD撰写) → C9-13(审评跟进)

**全球注册协同**: C9-20(法规差异解读) → C9-02(全球同步策略) → C9-07(CTA跨区域管理) → C9-04(时间线规划) → C8-18(注册部门协作)

**审评答辩与专家评审**: C9-14(NDA问询应答) → C9-15(咨询委员会/专家评审准备) → C7-08(学术会议参与策略) → C2-09(对外沟通策略) → C9-13(审评跟进)

**上市后变更管理**: C9-16(上市后变更申报) → C10-08(生产变更管理) → C9-17(补充申请/备案) → C9-13(审评跟进) → C10-09(工艺验证)

**再注册全流程**: C9-18(再注册/延续注册管理) → C9-19(注册证维护台账) → C10-13(APR/PQR) → C6-23(PSUR撰写) → C9-13(审评跟进)
