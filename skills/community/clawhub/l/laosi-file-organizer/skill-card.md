## Description: <br>
Organizes Desktop and Downloads files into type-based folders and generates a cleanup report. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[534422530](https://clawhub.ai/user/534422530) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Individuals and developers use this skill to preview and organize cluttered Desktop and Downloads folders by file type, then review a report of scanned, moved, skipped, and errored files. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Execute mode can move files in Desktop and Downloads into category subfolders, changing where users find those files. <br>
Mitigation: Run dry-run mode first, review the generated report, and execute only after confirming the proposed moves. <br>
