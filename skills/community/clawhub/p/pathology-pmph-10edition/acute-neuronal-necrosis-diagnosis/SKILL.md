# 神经元急性损伤与可逆性评估

## 触发条件
当神经元出现急性损伤表现（如意识障碍、卒中、中毒症状）或显微镜下见胞体异常时，启动本评估。

## 执行流程

### 第一步：识别是否为急性坏死（红色神经元）
1. 检查是否存在以下全部形态学特征：
   - 核固缩（核缩小、深染）
   - 胞体缩小变形
   - 尼氏小体完全消失
   - HE染色下胞质呈深红色
2. 确认存在急性诱因：缺血、缺氧、感染或中毒。
3. 若符合 → 诊断为“神经元急性坏死”，不可逆。

### 第二步：若未达坏死标准，评估是否为中央性尼氏小体溶解
1. 检查是否具备以下可逆性损伤特征：
   - 神经元肿胀变圆
   - 细胞核偏位（核边置）
   - 核仁增大明显
   - 胞质中央尼氏小体消失，仅周边残留
   - HE染色呈苍白均质状，或尼氏染色见中央透亮区
2. 判断病因是否可去除（如纠正缺氧、补充维生素B、控制病毒感染、解除轴突压迫）。
3. 若病因可及时去除 → 判定为“可逆性损伤”，需积极干预。
4. 若病因持续存在 → 预警将进展为不可逆坏死。

## 输出
- 若满足急性坏死标准 → 确诊“红色神经元”，不可逆。
- 若为中央性尼氏小体溶解且病因可逆 → 提示可逆损伤，建议干预。
- 否则 → 预警神经元死亡风险。
