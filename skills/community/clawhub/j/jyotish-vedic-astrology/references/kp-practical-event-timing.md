## 一、Sub-Lord事件时机判断核心法则

### 1.1 子领主三层判定法

每个事件判断都经过三层分析：

```
第一层：星座领主（Sign Lord）
  → 确定大方向（这个星座的主星主宰什么领域）

第二层：星宿领主（Star Lord / Nakshatra Lord）
  → 确定中观方向（星宿的主星更精细地定位）

第三层：子领主（Sub-Lord）
  → 决定性因素（决定有利/不利/中性）
```

**核心原则**：
- 星座领主和星宿领主都指示同一方向，但子领主不同 → **结果完全不同**
- 子领主是"最终裁判"

## 二、事件时机判断实战流程

### 2.1 标准判断步骤

```
步骤1：确定问题类型
  → 婚姻？事业？财富？健康？

步骤2：确定相关宫位
  → 事业：10宫（职业）+ 2/6/11宫（财富/竞争/收入）
  → 婚姻：7宫（配偶）+ 2/11宫（家庭/愿望实现）

步骤3：提取Cuspal Sub-Lord
  → 查看相关宫位的宫头子领主

步骤4：子领主指示分析
  → 子领主指示哪些宫位？
  → 有利/不利/中性？

步骤5：Significator层级分析
  → 1级Significator（最强）
  → 2级Significator
  → 3级Significator
  → 4级Significator

步骤6：Ruling Planets匹配
  → 当前统治行星
  → 与Significator是否匹配

步骤7：Dasha+Bhukti+Antra匹配
  → 三级Dasha领主是否与Significator一致

步骤8：最终判断
  → 综合所有层级给出结论
```

### 2.2 是/否判断标准

| 条件 | 结果 |
|------|------|
| Cuspal Sub-Lord有利 + Significator一致 + Dasha匹配 | ✅ **是**（90%+） |
| Cuspal Sub-Lord有利 + Significator部分一致 + Dasha部分匹配 | ⚠️ **可能**（60-89%） |
| Cuspal Sub-Lord中性 + Significator不一致 | ⚠️ **不确定**（40-59%） |
| Cuspal Sub-Lord不利 | ❌ **否**（60%+） |
| Cuspal Sub-Lord不利 + Significator一致有利 | ⚠️ **困难但有转机**（30-50%） |

## 三、实战案例详解

### 案例1：求职预测——"我能拿到这份工作吗？"

**背景**：
- 男性，32岁，软件工程师
- 面试了一家科技公司，等待结果
- 当前时间：2026年5月

**分析过程**：

**步骤1-2：确定问题与宫位**
- 问题类型：事业
- 相关宫位：10宫（职业）+ 2宫（收入）+ 6宫（日常工作）+ 11宫（愿望实现）

**步骤3：Cuspal Sub-Lord分析**
- 10宫宫头子领主：木星
- 木星指示宫位：2、6、10、11
- 判断：✅ 有利

**步骤4：Significator分析**
- 1级Significator（最强）：火星（在10宫内行星的星宿中）
- 2级Significator：太阳（在10宫内）
- 3级Significator：水星（在10宫主星的星宿中）
- 4级Significator：木星（10宫主星）

**步骤5：Ruling Planets**
- 日主：木星（周四面试）
- 月亮星座领主：火星
- 月亮星宿领主：太阳
- 与Significator匹配度：高（火星、太阳、木星全部匹配）

**步骤6：Dasha匹配**
- 当前Mahadasha：木星（匹配10宫主星）
- 当前Bhukti：太阳（匹配2级Significator）
- 当前Antra：火星（匹配1级Significator）

**综合判断**：
- Cuspal Sub-Lord：✅ 有利
- Significator：✅ 一致
- Ruling Planets：✅ 匹配
- Dasha：✅ 完全匹配

**结论**：**是，能拿到这份工作**（置信度90%+）
**时间窗口**：木星-太阳-火星Antra期间（约2-4周内）

### 案例3：投资决策——"现在适合投资加密货币吗？"

**背景**：
- 男性，35岁，自由职业者
- 想投资10万到加密货币
- 当前时间：2026年5月

**分析过程**：

**步骤1-2：确定问题与宫位**
- 问题类型：财富/投资
- 相关宫位：2宫（资产）+ 5宫（投机）+ 11宫（收益）

**步骤3：Cuspal Sub-Lord分析**
- 5宫宫头子领主：Rahu
- Rahu指示宫位：6、8、12
- 判断：❌ 不利（投机领域有风险）

- 2宫宫头子领主：金星
- 金星指示宫位：2、7、11
- 判断：✅ 有利（基本资产管理能力好）

- 11宫宫头子领主：土星
- 土星指示宫位：2、6、10
- 判断：⚠️ 中性（收入稳定但投机收益不确定）

**步骤4：Significator分析**
- 5宫 1级Significator：Rahu
- 5宫 2级Significator：火星
- 5宫 3级Significator：土星
- Rahu指示6/8/12 → 不利投机

**综合判断**：
- 5宫Sub-Lord不利 → 投机领域有风险
- 2宫Sub-Lord有利 → 基本理财能力好
- 建议：**不建议加密货币投机**，但可以稳健投资

**结论**：**不建议此时投资加密货币**（置信度75%）
**替代建议**：选择2宫Sub-Lord有利的投资方式（稳健型理财）

## 五、问答占星（Horary）实战指南

### 5.1 数字问卜法

**方法**：求问者在1-249之间凭直觉选一个数字。

**步骤**：
1. 求问者说出数字（如：137）
2. 找到数字137对应的黄道位置
3. 该位置成为问卜盘的上升点
4. 分析问卜盘的Cuspal Sub-Lord

### 5.2 问答占星的判断优先级

| 优先级 | 分析内容 | 权重 |
|--------|---------|------|
| 最高 | 宫头子领主（Cuspal Sub-Lord） | 40% |
| 高 | 1级Significator | 25% |
| 中 | Ruling Planets匹配 | 20% |
| 低 | 行星状态（庙旺陷落） | 15% |

### 5.3 问答占星适用场景

| 适用 | 不适用 |
|------|--------|
| "我会得到这份工作吗？" | "我的人生方向是什么？" |
| "他会和我结婚吗？" | "我应该和谁结婚？" |
| "投资会赚钱吗？" | "我应该投资什么？" |
| "病会好吗？" | "我为什么会生病？" |
| "能出国吗？" | "我应该去哪个国家？" |

**原则**：问答占星适合**是/否**类问题，不适合**开放性**问题。

## 六、KP准确率提升技巧

### 6.1 五大常见错误

| 错误 | 描述 | 修正方法 |
|------|------|---------|
| 只看宫主星 | 忽视星宿领主和子领主 | 使用四层级Significator |
| 忽视Ruling Planets | 不考虑当前天象匹配 | 始终检查RP与Significator匹配 |
| 过度依赖单一宫位 | 只看10宫判断事业 | 结合2/6/10/11多宫分析 |
| 忽视Dasha层级 | 只看Mahadasha | 必须看Mahadasha+Bhukti+Antra |
| 忽视否定宫位 | 只看有利宫位 | 同时检查不利宫位（1/5/9对事业不利） |

### 6.2 准确率提升清单

- [ ] 检查Cuspal Sub-Lord → 是有利还是不利？
- [ ] 检查四层级Significator → 哪些行星指示目标宫位？

- [ ] 检查Dasha三级 → 是否与Significator一致？
- [ ] 检查否定宫位 → 是否有不利指示？
