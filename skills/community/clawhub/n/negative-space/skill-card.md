## Description: <br>
Metacognitive pre-execution analysis that teaches an agent to identify missing assumptions, hidden constraints, absent context, and likely failure modes before acting. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[jcools1977](https://clawhub.ai/user/jcools1977) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agent users use this skill to make an agent pause before meaningful actions, verify key assumptions, and avoid wasted turns, failed approaches, and scope drift. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Overuse can add unnecessary verification or clarifying questions to simple tasks. <br>
Mitigation: Apply the skill before meaningful actions and skip it for trivial operations, as described in the artifact. <br>
Risk: The skill changes agent behavior by encouraging extra caution before action. <br>
Mitigation: Use it when that caution is desired and review agent responses for excessive hesitation in low-risk workflows. <br>
