## Description: <br>
Generates daily weather-themed character image guidance based on mood, real-time weather, and a selected city. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[waffle105](https://clawhub.ai/user/waffle105) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users use this skill to create a daily weather-themed character concept or image workflow from their mood, local weather, and one of 27 Chinese cities. It also guides setup for interactive city and mood selection, custom character replacement, and optional daily scheduling. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The submitted artifact references Python scripts, README.md, and character image files that are not included in the package. <br>
Mitigation: Treat the package as documentation-only unless those files are obtained separately, and review and scan any separate files before running them. <br>
Risk: The documented scheduler can run recurring daily automation. <br>
Mitigation: Enable scheduling only when recurring execution at 7:30 AM is intended and the runtime environment has been reviewed. <br>
