# 代理记账行业校准模板

本文件定义四种代账机构类型的校准参数。使用技能前须先确定机构类型，按对应模板校准各集群参数。

## 校准维度（11个）

| 维度 | 说明 |
|------|------|
| 核心术语 | 行业特有术语（10-15个） |
| C1 情报核心维度 | 重点关注的市场情报方向 |
| C2 内容核心规格 | 内容产出的类型和风格 |
| C3 增长核心方式 | 主要获客和增长方式 |
| C4 分析核心KPI | 重点关注的数据指标 |
| C5 知识核心领域 | 知识库建设的重点领域 |
| C6 合规核心要求 | 合规审核的重点方向 |
| C7 客户核心对象 | 主要客户类型和服务方式 |
| C8 协同核心部门 | 主要协作部门和角色 |
| C9 认证核心路径 | 资质认证的重点路径 |
| C10 运营核心规范 | 工具和运营的重点规范 |

---

## 类型一：传统代账公司

**核心特征**：线下为主，中小规模，人工+工具混合运营，以稳定续费和口碑为核心竞争力

### 校准参数

| 维度 | 校准值 |
|------|--------|
| **核心术语** | 代账、报税、记账、凭证、汇算清缴、工商年报、税负率、小规模、一般纳税人、查账征收、核定征收 |
| **C1 情报核心维度** | 区域代账公司动态、财政局政策、税局通知、园区招商政策、本地价格竞争 |
| **C2 内容核心规格** | 实用型财税科普、朋友圈/微信群内容、线下活动宣传物料、客户通知模板 |
| **C3 增长核心方式** | 老客户转介绍(核心)、园区地推、工商局驻点引流、同行合作、朋友圈口碑 |
| **C4 分析核心KPI** | 续费率(>85%目标)、人效比(30-50户/人)、户均收入、差错率(<2%)、应收账款 |
| **C5 知识核心领域** | 小企业会计准则、增值税/个税操作、常见税务风险、本地税局偏好、凭证规范 |
| **C6 合规核心要求** | 代理记账许可证年检、逾期申报零容忍、发票合规、客户数据保密、凭证规范 |
| **C7 客户核心对象** | 小微企业(小规模为主)、个体工商户、初创公司，按行业和规模分层 |
| **C8 协同核心部门** | 客户经理↔主办会计↔助理会计↔外勤，以线性分工为主 |
| **C9 认证核心路径** | 代理记账许可证→年检→财政局备案→地方财税协会 |
| **C10 运营核心规范** | 金蝶/用友单机版+电子税务局+微信群+纸质档案，稳定优先 |

### 业务模式特点
- 人均服务30-50户，以手工+半自动化为主
- 收费标准：小规模200-300元/月，一般纳税人500-800元/月
- 汇算清缴季需要临时增聘人手
- 核心风险：逾期申报、凭证差错、客户流失
- 增值服务：工商变更、社保代办、税务咨询

## 类型二：互联网代账平台

**核心特征**：线上获客为主，高度标准化，AI+自动化驱动，以规模增长和效率为核心竞争力

### 校准参数

| 维度 | 校准值 |
|------|--------|
| **核心术语** | SaaS代账、智能记账、OCR识别、自动申报、RPA、客户门户、在线签约、批量处理 |
| **C1 情报核心维度** | 竞品平台功能/定价/融资、全电发票改革、电子税务局升级、AI代账技术趋势、互联网获客成本 |
| **C2 内容核心规格** | SEO长尾词矩阵、抖音/小红书短视频、在线直播课、自动化营销邮件、产品功能宣传 |
| **C3 增长核心方式** | SEM/信息流投放、SEO获客、内容营销、线上免费工具引流、裂变拉新、品牌投放 |
| **C4 分析核心KPI** | 获客成本(CAC)、LTV/CAC比(>3目标)、转化漏斗、自动申报率(>90%)、人效比(100+户/人)、NPS |
| **C5 知识核心领域** | API对接文档、RPA流程脚本、自动申报规则库、批量处理SOP、异常处理知识库 |
| **C6 合规核心要求** | 数据安全等保三级、客户数据隔离、自动申报准确性、电子档案合规、平台服务协议 |
| **C7 客户核心对象** | 全国范围小微客户(线上签约)、自助+半自助服务模式、按套餐分层 |
| **C8 协同核心部门** | 产品↔技术↔运营↔客服，以敏捷迭代为主，AI替代重复劳动 |
| **C9 认证核心路径** | ICP+EDI+等保三级+软件著作权+商标+代理记账许可证 |
| **C10 运营核心规范** | 云端代账SaaS+OCR+RPA+电子税务局API+客户门户+自动化监控 |

### 业务模式特点
- 人均服务100+户，高度自动化
- 收费标准：小规模99-199元/月，一般纳税人399-599元/月
- 核心竞争力：价格+效率+便捷
- 核心风险：自动申报差错、数据安全、客户粘性低
- 增值服务：AI税务顾问、经营数据分析、融资对接

## 类型三：财税咨询公司

**核心特征**：高客单价，税务筹划+财务咨询为主，专家驱动，以价值创造和深度服务为核心竞争力

### 校准参数

| 维度 | 校准值 |
|------|--------|
| **核心术语** | 税筹、税务优化、财务顾问、合规架构、股权转让、跨境税务、税收洼地、核定征收、合伙企业、递延纳税 |
| **C1 情报核心维度** | 税收政策变动、税收洼地政策、行业并购动态、高净值客户需求、跨境税务政策 |
| **C2 内容核心规格** | 深度行业报告、税务筹划案例白皮书、高管财税内训、闭门沙龙内容、专业论文 |
| **C3 增长核心方式** | 行业口碑(核心)、闭门沙龙/私董会、会计师事务所合作、律所合作、高净值圈层营销 |
| **C4 分析核心KPI** | 客单价(5000+元/月)、项目利润率(>40%)、客户留存率(>90%)、转介绍率(>30%)、项目交付周期 |
| **C5 知识核心领域** | 企业会计准则、国际税收协定、税收筹划案例库、并购重组税务、跨境税务、股权转让税务 |
| **C6 合规核心要求** | 税务筹划合规边界、反避税规则、关联交易合规、跨境税务合规、执业责任保险 |
| **C7 客户核心对象** | 中大型企业、拟上市公司、跨境企业、高净值个人，按复杂度和价值分层 |
| **C8 协同核心部门** | 税筹专家↔审计团队↔法律顾问↔客户经理，以项目制协作 |
| **C9 认证核心路径** | 代理记账许可证+注册会计师资质+税务师资质+证券从业资质+律师事务所合作 |
| **C10 运营核心规范** | 专业审计软件+税务数据库(Wind/同花顺)+高端CRM+项目管理工具+视频会议 |

### 业务模式特点
- 人均服务5-15户，但单户价值高
- 收费标准：按项目收费(1万-100万+)或年度顾问费(5万-50万/年)
- 核心竞争力：专业深度+定制化方案+风险把控
- 核心风险：筹划方案被否定、执业责任风险、政策变动风险
- 增值服务：并购税务尽调、上市税务合规、跨境架构设计、家族信托税务

## 类型四：自动化代账平台

**核心特征**：持有代理记账许可证，以SaaS+全自动报税为核心产品，客户自助操作为主、少量审账人员兜底，以极低价格+零门槛+规模效应颠覆传统代账

> 典型代表：自记账（zijizhang.com，30万+企业，1100万+次自动报税，TSC5级信用）、伽玛账（gamaz.cn）、自会计（zikuaiji.com）

### 校准参数

| 维度 | 校准值 |
|------|--------|
| **核心术语** | 自动报税、自助记账、全税种申报、拍照记账、零门槛、极简记账、客户自助、审账兜底、SaaS订阅、批量自动化 |
| **C1 情报核心维度** | 同类自动化平台动态（自记账/伽玛账/自会计的功能/定价/用户增长）、传统代账客户流失到自助平台趋势、全电发票+电子税务局API能力边界、AI记账技术突破、监管对自动化代账的态度 |
| **C2 内容核心规格** | 极简教程（30秒教会记账）、价格对比冲击内容（"代账公司1/8价格"）、用户证言视频、App功能演示短视频、零基础创业财税指南 |
| **C3 增长核心方式** | ASO/App Store获客(核心)、SEM/信息流投放、SEO长尾词（"自己记账""不请代账"）、口碑裂变（推荐返现）、应用市场评分运营、免费试用转化 |
| **C4 分析核心KPI** | 月活/日活、自动申报成功率(>99%)、审账人工干预率(<10%)、CAC(<100元)、LTV/CAC(>5)、月费续费率、客户自助完成率、App评分(>4.5) |
| **C5 知识核心领域** | 全税种自动申报规则库、各省级电子税务局接口差异、OCR票据识别训练集、常见记账差错自动检测规则、客户自助操作FAQ |
| **C6 合规核心要求** | 代理记账许可证+TSC信用评级、自动申报准确性（罚款全额赔款承诺）、客户数据安全与隔离、电子会计凭证合规归档、退款保障机制 |
| **C7 客户核心对象** | 个体工商户(核心)、小微企业(小规模)、创业小白——价格极度敏感、业务简单、不懂财税、不需要人工服务 |
| **C8 协同核心部门** | 产品↔技术(核心驱动)↔审账团队(极少量)↔客服(一对一答疑)，以产品自动化替代人力为第一优先级 |
| **C9 认证核心路径** | 代理记账许可证→TSC涉税信用评级→ICP+等保二级→App上架资质→软件著作权→商标 |
| **C10 运营核心规范** | 自研App+SaaS后端+OCR+电子税务局API直连+银行回单自动导入+自动凭证生成+自动全税种申报，全链路自动化 |

### 业务模式特点
- 人均服务500+户，以产品自动化替代人工，审账人员仅兜底
- 收费标准：小规模29.99-99元/月，一般纳税人99-149元/月（传统代账1/8~1/5）
- 核心竞争力：价格碾压+零门槛+全自动+数据在自己手中
- 核心风险：自动申报差错导致客户罚款（对冲：全额赔款承诺）、复杂账务无法覆盖（对冲：标注"暂不支持"并引导退回传统代账）、客户粘性低月费模式易流失
- 增值服务：审计评估报告、工商年报工具、发票API/报税API（面向开发者）、注册地址服务
- 关键壁垒：全税种自动申报的技术实现、各省级税务局接口适配、OCR+规则引擎的记账准确性
