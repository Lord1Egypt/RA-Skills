# 京东自营秒杀

提供京东秒杀频道商品查询工具，展示限时秒杀好货，基于京东官方数据。仅筛选京东自营商品，品质有保障。默认精选好评≥98%、近30天销量≥5000的品质商品，结果不足时自动逐级降级（先降销量到≥2000，再降到≥1000，再到≥500，最后不限销量），好评率≥98%固定不降级，保证每次查询都有推荐。

## 能做什么

- **秒杀查询**：展示京东秒杀频道的限时好货，已精选好评≥98%、销量≥5000的品质商品
- **品类筛选**：按关键词筛选特定品类的秒杀商品（支持模糊匹配，如"家电"可匹配"家用电器"）
- **多种排序**：综合评分、到手价升序、折扣力度降序
- **品质自定义**：可自定义最低好评率（98/99/100），好评率固定不降级，只放宽销量门槛
- **动态降级**：结果不足时自动放宽销量门槛（5000→2000→1000→500→不限），好评≥98%不降级，保证推荐数量
- **翻页浏览**：每页50条，说"下一页"查看更多

## 不能做什么

- 不能下单购买，只提供商品信息和购买链接
- 不能设置秒杀开始提醒通知
- 不能查询秒杀的精确倒计时

## 使用示例

1. "京东秒杀有什么手机" → 按品类筛选
2. "帮我看看100元以下的秒杀" → 按价格筛选
3. "京东秒杀有什么好货" → 返回全部第1页（50条）
4. "折扣最大的秒杀" → 按折扣力度排序
5. "下一页" → 查看更多商品

## 注意事项

- 秒杀价格可能随时变动，以京东页面实际价格为准
- 部分秒杀商品数量有限，售完即止
- 仅筛选京东自营商品，专卖店和个人店已自动剔除
- 好评率≥98%为品质标准，固定不降级
- **数据流向说明**：查询通过云端代理转发到京东官方API，代理不存储用户数据

## 使用提示

- keyword为空时返回全部秒杀推荐，每页50条
- 带品类关键词时结果较少属正常现象，说明符合条件的好货稀缺
- sort=discount按折扣力度排序，适合找折扣最大的秒杀好货
- 秒杀商品更新频繁，建议多翻几页发现更多好货
- 每条商品包含商品图片URL（image_url），请用 ![商品名](image_url) 格式渲染图片，图片下方放购买链接
- 购买链接点击后自动识别设备，手机端有京东APP则唤起APP
