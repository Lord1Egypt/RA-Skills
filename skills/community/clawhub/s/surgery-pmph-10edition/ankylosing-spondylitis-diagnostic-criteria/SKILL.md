# 强直性脊柱炎诊断标准

## 适用对象
- 疑似强直性脊柱炎患者（存在腰背痛、脊柱活动受限、胸廓扩展受限或骶髂关节炎影像学证据）

## 使用前提
确保患者满足以下任一前提条件：
- 存在持续≥3个月的炎性腰背痛（活动后缓解）
- 脊柱多方向活动受限
- 胸廓扩展范围低于同龄同性别正常值
- 影像学提示骶髂关节炎

## 执行步骤
1. **确认影像学标准**：检查X线或CT结果是否满足以下任一条件：
   - 双侧骶髂关节炎Ⅱ~Ⅳ级，或
   - 单侧骶髂关节炎Ⅲ~Ⅳ级。
2. **验证临床标准**：若影像学标准满足，则检查是否同时具备以下任一临床表现：
   - 腰背痛持续≥3个月，且活动后缓解；
   - 脊柱在前屈、后伸、侧弯、旋转四个方向均受限；
   - 胸廓扩展范围（深吸气与深呼气胸围差）低于同年龄、同性别正常参考值。
3. **综合判断**：仅当同时满足影像学标准和至少一项临床标准时，方可诊断为强直性脊柱炎。
4. **排除干扰**：确认患者类风湿因子阴性，且无仅手部小关节受累表现，以排除类风湿关节炎等其他疾病。

## 输出结果
返回“符合强直性脊柱炎诊断标准”或“不符合”，用于辅助临床确诊。

> 注意：本标准需结合HLA-B27检测、完整病史及鉴别诊断综合应用，不可孤立使用。
