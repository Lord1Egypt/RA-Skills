# 脊柱生理性弯曲与病理性变形识别

## 准备工作
- 患者脱去上衣，取站立位或坐位
- 双手自然下垂
- 医生熟悉关键解剖标志

## 执行步骤

1. **背面观检查**
   - 正常脊柱呈一条直线
   - 若偏离中线，提示可能存在侧弯

2. **触诊确认侧弯**
   - 用示指、中指沿棘突自上而下滑压，形成红色充血痕作为参考线
   - 观察是否偏离中线

3. **侧面观检查四个生理弯曲**
   - 颈段：稍向前凸
   - 胸段：稍向后凸
   - 腰椎：明显向前凸
   - 骶椎：明显向后凸

4. **识别病理性变形类型**
   - **后凸（驼背）**：胸段过度后弯
     - 儿童：佝偻病（坐位明显，仰卧可消失）
     - 青少年：脊柱结核（成角畸形）
     - 成人：强直性脊柱炎（弓形后凸、固定）
     - 老人：退行性变
   - **前凸**：腰椎过度前凸，见于妊娠、腹腔积液、腰椎滑脱
   - **侧凸**：分姿势性（可纠正）与器质性（不可纠正）

5. **利用解剖标志定位节段**
   - 第7颈椎棘突最长
   - 肩胛冈连线 → 第3胸椎
   - 肩胛下角连线 → 第7胸椎
   - 髂嵴最高点连线 → 第4腰椎下部或L4-L5间隙
   - 髂后上棘连线 → L5-S1棘突之间

> 输出应包含弯曲类型、变形名称、可能病因及是否需影像学进一步检查。
