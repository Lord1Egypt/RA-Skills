## 模块概述

**定位：** 解决外贸交易"付款"环节的完整闭环

**核心价值：**
- 提供标准化的支付方案选择
- 建立完善的支付风控体系
- 指导汇率管理与避险操作
- 规范账期管理与催收流程

## 二、收汇结汇流程（1039政策）

### 2.1 收汇流程SOP

```
【1039市场采购贸易收汇SOP】

Step 1: 银行账户准备
├── 开立义乌稠州商业银行/中国银行外汇账户
├── 确认账户支持1039贸易项下收汇
├── 开通网银外汇功能
└── 保存银行联系人电话

Step 2: 收款确认
├── 客户打款后，银行发送收汇通知
├── 登录网银查看收汇明细
├── 核对：金额、汇款人、附言
└── 记录收汇编号

Step 3: 收汇入账
├── 银行审核通过后入账
├── 查看外汇账户余额
├── 记录收汇日期和金额
└── 更新订单状态

Step 4: 结汇操作
├── 登录网银外汇业务
├── 选择结汇币种和金额
├── 确认结汇汇率
└── 提交结汇申请

Step 5: 人民币入账
├── 结汇成功后人民币到账
├── 核对入账金额
├── 转账至人民币账户
└── 归档收汇凭证
```

### 2.3 1039结汇注意事项

```
【1039收汇注意事项】

✅ 合规操作：
├── 通过正规银行收汇结汇
├── 保留完整单证（合同/发票/物流单）
├── 如实申报资金性质
└── 单证留存至少3年

❌ 禁止行为：
├── 借用他人账户收汇
├── 拆分结汇规避监管
├── 地下钱庄换汇
└── 虚假贸易背景收汇

⚠️ 常见问题：
├── 收汇被拒：检查外汇管理规定
├── 结汇限额：了解个人/企业额度
└── 账户冻结：配合银行调查
```

### 3.2 支付风控流程

```
【支付风控标准流程】

收到订单 → 客户背调 → 风控评估 → 方案设计 → 执行监控 → 异常处理

Step 1: 客户背调（必须）
├── 核查公司工商信息（天眼查）
├── 查询制裁名单（OFAC/欧盟）
├── 历史合作记录（如有）
└── 邓白氏信用报告（如有大额）

Step 2: 风控评估
├── 评估结论：A/B/C级
├── A级（信用良好）：常规操作
├── B级（有一定风险）：加强风控
└── C级（高风险）：拒绝或全额预付

Step 3: 方案设计
├── 根据评估结果设计付款方案
├── 明确预付比例
├── 确定托管方式
└── 与客户确认条款

Step 4: 执行监控
├── 监控付款到账
├── 核实水单真伪
├── 确认正本提单
└── 全程保持沟通记录

Step 5: 异常处理
├── 未按时付款：催款+警告
├── 付款异常：暂停发货
└── 欺诈嫌疑：立即报警
```

### 3.3 账户变更验证SOP

```
【账户变更防诈骗SOP】

触发：收到账户变更通知

Step 1: 立即暂停付款
└── 通知财务暂停该订单所有付款

Step 2: 多渠道核实
├── 电话核实（使用原始号码）
├── 视频核实（如可行）
└── 邮件核实（回复原始邮件）

Step 3: 内部审批
├── 填写《账户变更审批表》
├── 财务+老板双重审批
└── 通过后方可付款

Step 4: 记录归档
└── 保存全部沟通记录
```

## 四、汇率管理

### 4.1 汇率管理工具

```
【汇率风险管理工具箱】

工具1：远期结汇
├── 原理：锁定未来某日汇率
├── 适用：订单周期>30天
├── 优点：确定性高
└── 成本：银行手续费约0.1%

工具2：外汇期权
├── 原理：购买结汇选择权
├── 适用：汇率波动剧烈
├── 优点：风险有限
└── 成本：期权费约0.5%-1%

工具3：自然对冲
├── 原理：美元收+美元支
├── 适用：有美元采购渠道
├── 优点：零成本
└── 条件：需配套采购渠道

工具4：汇率保险
├── 原理：保险产品兜底
├── 适用：大额订单
├── 优点：风险转移
└── 成本：保费约0.3%-0.5%
```

### 4.3 汇率波动应对策略

```
【汇率波动应急处理】

情况1：汇率大幅下跌（美元贬值）
├── 立即评估影响：利润损失计算
├── 短期措施：与供应商谈判降价
├── 中期措施：提高新订单报价
└── 长期措施：增加非美货币结算

情况2：汇率大幅上涨（美元升值）
├── 短期措施：加速结汇
├── 中期措施：观望等待更高点
└── 长期措施：考虑美元采购对冲

情况3：汇率剧烈波动
├── 暂停大额结汇
├── 分批结汇分散风险
└── 咨询银行锁汇方案
```

### 5.2 信用证（L/C）入门

```

├── 买卖双方首次合作
└── 高风险交易

├── 银行作为中间担保方
├── 买家银行开立信用证
```

## 七、模块总结

### 7.1 核心流程速查

| 场景 | 操作SOP | 关键点 |
|------|---------|--------|
| 新客户首单 | 背调→预付50%→发货→付尾款 | 背调必须做 |
| 账户变更 | 暂停付款→多渠道核实→审批 | 电话确认 |
| 大额订单 | 背调→部分锁汇→分阶段付款 | 锁汇建议 |
| 高风险国家 | Escrow或全额预付 | 安全第一 |
