## Use Case: <br>
Developers, operators, and marketing teams use this skill to integrate Nudgen APIs, operate the Nudgen CLI, and plan email retention and deliverability workflows. It helps agents produce implementation guidance, commands, troubleshooting steps, and campaign-quality recommendations. <br>

## Known Risks and Mitigations: <br>
Risk: The skill uses Nudgen account tokens and other sensitive credentials for API and CLI workflows. <br>
Mitigation: Use least-privilege tokens, keep secrets server-side or in secure storage, and avoid printing credential values. <br>
Risk: Team-scoped API or CLI operations can mutate or delete data in the wrong Nudgen team context. <br>
Mitigation: Verify the active team before mutating or deleting data and use a read or identity check before write operations. <br>
Risk: The artifact includes a remote shell installer example for the Nudgen CLI. <br>
Mitigation: Inspect or pin the installer before execution, or use an auditable install path such as a local build or versioned package command. <br>
Risk: Broad trigger wording can cause the skill to activate for general marketing automation requests that are not Nudgen-specific. <br>
Mitigation: Install and use it only for Nudgen-specific work, and confirm the task is about Nudgen before applying product-specific API, CLI, or campaign guidance. <br>
