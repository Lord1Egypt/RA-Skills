# 设备与模具管理调研指南

## 调研目标
- 评估设备管理与TPM体系成熟度
- 识别设备故障与OEE损失的主要根因
- 评估模具管理与换模效率

## 调研重点

### 1. 设备基础管理
- 设备台账（种类/数量/关键程度分类）
- 设备点检制度与执行频率
- 设备故障报修流程及响应时效
- 点检维修记录管理方式

### 2. 设备保全体系(TPM)
- 操作员自主保全(AM)执行情况（日常清洁/点检/润滑/紧固）
- 专业保全(PM)计划与实施率
- OEE数据采集（自动/手工/未统计）
- MTBF/MTTR追踪
- 设备故障信息汇总与根因分析
- 防止再发对策体系

### 3. 备品备件管理
- 关键备件清单
- 备件采购流程与周期
- 备件安全库存管理

### 4. 设备数字化
- 设备是否具备数据输出功能
- 设备联网与数据采集(IoT)现状
- 设备管理看板（实时状态）

### 5. 模具管理
- 模具台账与寿命管理
- 模具库管理（存放/领用/归还流程）
- 模具完好性保障
- 模具保养计划与执行
- 模具备件采购

### 6. 换模管理
- 换模人员配置与技能
- 换模频次与计划性
- 换模准备（内部作业/外部作业区分）
- 换模/调试平均时间（实测数据）

## 常见问题诊断库

| 问题现象 | 可能根因 | 典型影响 |
|---------|---------|---------|
| OEE低 | 故障多/换型长/质量损失 | 产能不足/成本高 |
| 设备频繁故障 | 自主保全缺失/PM计划执行差 | 停线损失/交期延误 |
| 模具状态不可控 | 模具库管理缺失/寿命无追踪 | 生产异常/质量波动 |
| 换模时间长 | 内外部作业未区分/改善不足 | 柔性差/被迫大批量 |
| 备件响应慢 | 备件清单缺失/采购流程长 | MTTR长/停线延长 |
| 设备数据不透明 | 无数据采集/无OEE看板 | 改善无依据/管理凭感觉 |
| 点检流于形式 | 检查表不实用/监督缺失 | 突发故障/安全风险 |

## 数据采集项
- OEE（可用率/性能率/质量率）
- 设备故障率
- MTBF/MTTR
- PM计划执行率
- 换模频次与时间
- 备件库存金额与周转
- 模具寿命与状态
