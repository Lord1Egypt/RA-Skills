Global <br>

## Known Risks and Mitigations: <br>
Risk: The self-update command replaces the local skill instructions from the remote main branch. <br>
Mitigation: Use /skillnav update only when you trust the SkillNav repository, and review the updated skill instructions before relying on them. <br>
Risk: Search results may include install commands for third-party MCP servers or tools. <br>
Mitigation: Review install commands returned by SkillNav before running them in a local environment. <br>
