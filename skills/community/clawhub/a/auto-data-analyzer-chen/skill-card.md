## Description: <br>
Auto Data Analyzer helps agents analyze CSV or Excel datasets and generate reports with data cleaning, descriptive statistics, visualization, correlation analysis, and regression analysis. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[jason-aka-chen](https://clawhub.ai/user/jason-aka-chen) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, analysts, and business users can use this skill to quickly analyze tabular datasets and prepare data cleaning summaries, statistical descriptions, charts, and analysis reports. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated reports and charts may contain sensitive values from the source dataset. <br>
Mitigation: Keep outputs in a controlled folder and share them only when the underlying data is safe to disclose. <br>
Risk: Package installation and example execution may run local code against user-provided datasets. <br>
Mitigation: Install dependencies in a virtual environment and run examples only on datasets intentionally selected for analysis. <br>
