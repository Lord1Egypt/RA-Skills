# Frontend Design Skill

Build distinctive UIs — avoid the generic "AI slop" aesthetic.

## Rules
- NEVER use Inter, Roboto, Arial, system-ui as primary fonts
- Use distinctive fonts: Space Mono, Syne, Clash Display, Cabinet Grotesk, Satoshi, Fraunces
- Avoid purple gradients on white — pick a real color palette
- Use asymmetric layouts, not 3-card grids
- Add micro-interactions and personality

## Good Patterns
- Bold typography with tight tracking
- Unexpected accent colors
- Generous whitespace with intentional density
- Scroll animations on key content

## Bad Patterns (avoid)
- Hero → Features → Testimonials → CTA → Footer
- Purple/blue gradient backgrounds
- "AI-generated" card grids
- Generic stock illustration style
