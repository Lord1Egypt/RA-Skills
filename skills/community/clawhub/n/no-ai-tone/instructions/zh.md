# 中文：避免 AI 生成腔调

> 调研来源：知乎、小红书、CSDN、即刻、腾讯新闻、澎湃新闻、学术论文（北京语言大学 ACL 2023），综合社媒用户真实吐槽，8轮搜索综合。

核心主张：**AI 文字的问题不只是用了哪些词，而是整体腔调——面面俱到、面无表情、节奏如节拍器。** 好的文字有取舍、有立场、有时候不完美。

---

## 一、词汇黑名单

### 1.1 连接词/过渡词（最高频 AI 信号）

这些词本身无错，问题在于 AI 每段必用、高密度堆叠。写作时主动删除，用上下文自然衔接替代。

**立即删除型（出现即红牌）**
- 综上所述、总而言之、总的来说
- 值得注意的是、值得一提的是
- 不可否认、毋庸置疑、显而易见
- 由此可见、不言而喻
- 可见（作为结论引导词，结论往往并不自然得出）
- 不仅……而且……（高频并列结构，密集出现即露馅）

**控制频率型（每篇最多出现 1 次）**
- 此外、另外、与此同时
- 然而、不过（用来制造虚假转折）
- 因此、由此、故而、基于此
- 首先……其次……最后（三件套禁止整段使用）
- 同时（作为段首衔接词，极易累积）

### 1.2 企业/营销空话词

直接替换为具体动词或描述：

| 禁用 | 替换思路 |
|---|---|
| 赋能、全面赋能、深度赋能 | 说清楚具体做什么，让谁获得了什么能力 |
| 深耕、聚焦 | 说多久、做了什么具体的事 |
| 驱动、赋动 | 用具体机制替代 |
| 重塑、颠覆 | 说改变了什么，前后对比 |
| 蓬勃发展、焕发新机 | 给数据或具体变化 |
| 深度融合、全面升级 | 说具体整合了什么，升了什么 |
| 举足轻重、不容忽视 | 直接说影响有多大 |
| 赋予……新的活力 | 说具体变化 |
| 致力于、不断追求 | 说做了什么，而不是"致力于做什么" |
| 打造（打造生态/打造品牌） | 说建了什么、结果是什么 |
| 赛道（切入赛道/赛道布局） | 直接说这个行业/市场，赛道是比喻不是描述 |
| 降本增效 | 说省了多少钱或时间，具体数字 |
| 核心竞争力 | 说到底强在哪，用一个具体例子 |
| 生态、生态位（泛指任何产品组合） | 说具体涉及哪些产品或合作方 |
| 版图（业务版图/市场版图） | 说进入了哪些地区或领域 |
| 一站式（一站式解决方案） | 说用户原来要跑几个地方，现在只用一个 |

### 1.3 空洞陈述开头（段首黑名单）

**通用型（学术/资讯 AI 最爱）**
- "近年来，……已引起广泛关注"
- "随着……的快速发展"
- "在当今社会……"
- "……是一个复杂的问题"
- "……具有重要的现实意义"

**自媒体/内容营销 AI 专用**
- "深度解析：……"（标题格式，几乎全是 AI）
- "一文读懂……"（夸大收益，内容往往不够）
- "保姆级……教程"（口语化包装 AI 内容的常见手法）
- "避坑指南：……"（用问题感包装空洞建议）
- "越来越多的人开始……"（无数据的伪趋势陈述）
- "想象一下，……"（AI 的场景代入套路）
- "你知道吗？……"（fake discovery 结构）

### 1.4 频率异常词（统计研究发现）

AI 写作中这些词出现频率显著偏高：
- **以及**（AI 频率约 0.53%，真人约 0.16%）
- 无论、真的、非常（单独无问题，密集出现露馅）
- 成语密度过高：古色古香、跌宕起伏、应有尽有、如虎添翼、锦上添花

## 二、句式禁区

### 2.1 立即避开的结构

```
❌ 首先，……。其次，……。最后，……。
❌ 第一，……；第二，……；第三，……。
❌ [每段结尾] 综上所述，……
❌ 不是 A，而是 B（直接说 B 就好）
❌ 在……方面，……；在……方面，……（过度并列）
```

### 2.2 节拍器节奏

**AI 文字读起来像节拍器**——所有句子长度均匀，没有喘息感。

修复方法：
- 主动加入 2-4 字的短句作为呼吸点
- 一段长句后跟一个短句，或反过来
- 北京语言大学研究：AI 对偶句平均 4 次/篇，真人仅 0.67 次——刻意减少排比

### 2.3 长句堆砌

AI 偏好"句子特别长，内容连贯但中间没有断开的标点"。  
→ 遇到超过 30 字的句子，考虑是否能断成两句。

## 三、语气规范

### 3.1 核心问题：无立场、和稀泥

AI 为避免争议，语气永远中立。这是最难掩盖的 AI 信号。

**检查清单：**
- 这段话有没有表达出"我们认为……"或"我们不认同……"？
- 有没有取舍？说了 A 的好，有没有同时暗示 B 不是我们的选择？
- 结论是真的结论，还是"因此，X 是重要的"这种无效总结？

### 3.2 过度正确反而露馅

> 知乎用户："AI 最牛的地方就是太正确——没有错别字、没有语法错误。这恰恰也是 AI 致命的缺点。"

允许：
- 不完整的句子（口语节奏）
- 轻微的语气词（嗯、其实、说实话）
- 带有倾向性的用词

### 3.3 情绪要真实，不要饱满

AI 写的感受"情绪过于饱满，反而不真实"。人对某件事的感受是混杂的、有保留的。  
→ 不要把每件事都写成"令人振奋"。说"这件事我们想了很久"比"这是一个激动人心的时刻"更真实。

## 四、排版/格式规范

被网友戏称「**豆包体**」的视觉特征，一看就是 AI：

```
❌ 每段开头加 emoji 标注
❌ 每个 H2/H3 下面整整齐齐三条加粗要点
❌ **加粗** 用于所有关键词（Markdown 渗入自然文字）
❌ 整齐的三层层级结构
❌ 标点使用"过于精准"——该用省略号的地方全是句号
```

**正常文字的格式感**：段落有长有短，重点通过句子本身强调，不依赖 Markdown 加粗。

## 五、内容规范

### 5.1 给具体细节，不给泛泛描述

```
❌ 我们提供高效、便捷、全面的解决方案
✓  我们把三个月的对接工作压缩到了一周

❌ 团队充满热情，致力于为用户创造价值
✓  我们在发布前重写了三次首页，因为第一版太假了
```

### 5.2 不要同义反复

AI 习惯用不同句式把同一个意思说三遍（撑字数）。  
→ 每个句子必须贡献新信息，否则删掉。

### 5.3 不要"全覆盖"

AI 的内容面面俱到，因为它没有真正的立场。  
→ 好文案要有取舍——说了 A，就意味着没说 B。这种取舍本身传递立场。

### 5.4 感官和动作细节（叙述性内容）

```
❌ 用户感到满意，获得了良好的体验
✓  打开 App 的第一秒，地图就已经加载好了
```

## 六、改写流程

收到中文文案修改任务时，按以下顺序检查：

1. **扫词**：对照黑名单，标出所有禁用词和过渡词
2. **测节奏**：大声读出来，找到节拍器感的段落，加入短句打断
3. **问立场**：这段话在说什么"我们认为"？如果没有，补上
4. **砍重复**：找出同义反复的句子，保留最准确的那一句
5. **加细节**：找一个可以换成具体数字/动作/感官描述的地方

## 七、改写示例

### 示例 1：产品介绍

```
❌ AI腔原文：
我们的平台聚焦于为企业赋能，深度融合先进技术，
全面提升用户体验。我们拥有强大的团队，致力于
创新，推动行业蓬勃发展。综上所述，选择我们，
就是选择未来。

✓ 改写后：
我们做一件事：让公司内部的信息不再藏在 Excel 里。
过去三年，帮助 40 家公司把周报从两小时压到了二十分钟。
```

### 示例 2：关于我们

```
❌ AI腔原文：
我们是一支充满热情的团队，深耕行业多年，
以用户为中心，不断追求卓越，为客户提供
全面、高效、便捷的解决方案。

✓ 改写后：
我们三个人，2021 年从同一家公司辞职，
因为受不了每次上线都要填八张审批表。
所以我们自己做了一个不需要填表的版本。
```

### 示例 3：小红书/公众号推文开头

```
❌ AI腔原文：
随着数字化转型的深入推进，越来越多的人开始
关注个人知识管理。深度解析：如何打造属于你的
第二大脑？想象一下，如果你的笔记能够自动整合
并赋能你的工作，会是什么体验？

✓ 改写后：
我的笔记 App 里有 3000 条记录，我上周能找到的不超过十条。
不是搜不到，是不记得自己存过。
后来我改了一个习惯，解决了这个问题。
```

## 八、核心判断标准

写完一段文案后，用三个问题自测：

1. **能预测吗？** 读完前半句，下半句是不是已经知道了——如果是，改掉
2. **有人在说话吗？** 把这段话想象成一个真实的人说出来——听起来奇怪吗？
3. **有没有取舍？** 说了什么，就意味着没说什么——如果什么都说了，什么都没表达

> 网友的终极比喻：**AI 写的文章像预制菜——成分齐全、摆盘漂亮、没有锅气。**
