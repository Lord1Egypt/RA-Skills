## Workflow

### 1. Capture Page Context

Collect the URL or pasted page copy, audience, offer, traffic source, and desired conversion action.

### 2. Score Conversion Drivers

Score each area as pass / warning / fail:

- Headline clarity
- CTA specificity
- Trust and proof
- Page flow
- Mobile readability
- Risk reversal / objection handling

### 3. Produce Prioritized Fixes

Return fixes in this order:

1. Critical conversion blockers
2. Trust/proof improvements
3. Copy and CTA rewrites
4. Mobile/layout quick wins
5. A/B tests worth running
