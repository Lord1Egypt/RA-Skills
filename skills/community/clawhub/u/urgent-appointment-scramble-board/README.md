# Urgent Appointment Scramble Board

Track urgent appointment calls across multiple providers with a live availability board, call log, openings, callbacks, and confirmation details.

## What it does
- Builds a ranked target list and a concise call script before dialing
- Logs every call outcome — slots, callbacks, waitlists, prerequisites, and fees
- Ranks available openings by logistics criteria (earliest time, travel, documents, cost)
- Provides a confirmation checklist to verify before treating a slot as secured

## Example scenarios

**1. Scramble for a dermatologist within 3 days**
> 👤 "I need a dermatologist appointment within 3 days for a suspicious mole. Here are 5 clinics I can go to."
> 🤖 Ranked priority list by cancellation-history likelihood, distance, and insurance acceptance. Generates call scripts for each clinic's front desk.

**2. Monitor patient portals for last-minute openings**
> 👤 "Monitor the patient portals for these 3 clinics and alert me if a slot opens before Friday."
> 🤖 Sets monitoring check cadence. When a slot opens, sends an urgency alert with the time, provider, and a one-tap booking link.

**3. 三甲医院号满了怎么捡漏**
> 👤 "牙疼得不行，但上海三甲医院的号全满了，最早的也要两周后。有没有办法捡漏或加号？"
> 🤖 三甲挂号攻略：每天早7点/下午5点刷新退号集中时间→挂同科室普通门诊再请医生转诊→有明显疼痛/发烧直接走急诊（24小时开放）→查附近二甲/区中心医院互认→黄牛渠道（300-1000元，有风险但管用）。关键：正规渠道优先，急诊兜底。
