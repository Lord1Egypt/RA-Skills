## Description: <br>
Guides structured help for Godot Project Checklist using clear templates, checks, and safe defaults. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[abdullah944](https://clawhub.ai/user/abdullah944) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and game creators use this skill to turn Godot project planning goals into concise plans, reusable checklists, next actions, and editable draft copy without sharing secrets. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Users could paste secrets or sensitive private project data into planning prompts. <br>
Mitigation: Keep inputs limited to non-secret goals, constraints, and notes; avoid passwords, keys, wallet data, OTPs, session secrets, and sensitive private project details. <br>
Risk: The artifact includes an external PayPal donation link. <br>
Mitigation: Treat the donation link as optional and external; it is not required for using the checklist guidance. <br>
