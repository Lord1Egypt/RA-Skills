# Qi-Tonifying Herb Differentiation

## When to Use
当患者出现以下气虚所致症状之一时使用：
- 神疲乏力、少气懒言、自汗、脉弱
- 食少便溏、易感冒（脾肺气虚）
- 脱肛、子宫脱垂、久泻不止（中气下陷）
- 心悸、失眠、健忘（心气不足）
- 吐血、衄血、崩漏、便血而色淡质稀（气不摄血）

## Execution Steps

### 1. 辨证确认气虚类型
- **脾肺气虚**：食少便溏、气短懒言、易感外邪。
- **中气下陷**：内脏下垂、久泻、小便浑浊。
- **心气不足**：心悸怔忡、失眠多梦、健忘。
- **气不摄血**：慢性出血，血色淡、质稀。

### 2. 选药依据
- **人参**：大补元气，用于元气欲脱、气津两伤或心脾肺肾俱虚。
- **黄芪**：补气升阳、固表止汗、利水消肿，尤适用于中气下陷及卫表不固。
- **白术、甘草**：健脾益气，常作基础配伍。

### 3. 配伍规则
- 脾虚：配白术、茯苓、甘草（四君子汤基础）。
- 中气下陷：配升麻、柴胡（补中益气汤结构）。
- 气不摄血：配当归、阿胶、龙眼肉。
- 气阴两伤：配麦冬、五味子（生脉散结构）。

### 4. 用法与禁忌
- 人参救脱：15–30g 文火另煎兑服。
- 黄芪：补气升阳宜蜜炙，托毒排脓宜生用。
- **禁忌**：实证、热盛、湿阻中焦者不宜单用；阴虚火旺者慎用性温补气药；人参反藜芦、畏五灵脂。

> 注意：必须先确认为气虚证，排除实证、热证或湿阻，方可使用补气药。
