# 卵子发生的三个阶段与减数分裂停滞机制

## 何时使用
- 用户询问卵子如何从原始生殖细胞发育为可受精的成熟卵子
- 需要解释为何排卵时排出的是次级卵母细胞而非成熟卵子
- 探讨减数分裂为何在胎儿期启动却停滞数十年
- 分析LH峰如何触发减数分裂恢复
- 理解透明带、皮质颗粒等结构的功能及其形成时机

## 执行步骤

### 1. 增殖期
- 确认原始生殖细胞（PGC）已迁移至卵巢生殖嵴
- PGC通过有丝分裂大量扩增形成卵原细胞（可达500万～700万个）
- 大部分卵原细胞凋亡，仅约5万个留存至出生

### 2. 生长期
- 卵原细胞体积增大，分化为初级卵母细胞（2n）
- 在胎儿期启动第一次减数分裂：DNA复制、同源染色体配对与交换
- 减数分裂停滞于前期Ⅰ双线期（dictyate stage），可持续数十年
- 初级卵母细胞被包裹于卵泡中，依次经历：
  - **原始卵泡**：单层扁平卵泡细胞包围，通过缝隙连接物质交换
  - **初级卵泡**：卵泡细胞变为立方/柱状，分泌透明带（zona pellucida），卵母细胞积累皮质颗粒
  - **次级卵泡**：形成卵泡腔，颗粒细胞构成卵丘，放射冠紧贴透明带

### 3. 成熟期
- 成熟卵泡直径达15～20mm，突出卵巢表面
- LH峰触发初级卵母细胞完成第一次减数分裂，不对称分裂产生：
  - 次级卵母细胞（保留大部分细胞质，n=23，含两条姐妹染色单体）
  - 第一极体（小，最终退化）
- 次级卵母细胞立即进入第二次减数分裂，但停滞于中期Ⅱ
- **排卵时排出的是处于中期Ⅱ的次级卵母细胞**
- 受精后完成第二次减数分裂，产生成熟卵子和第二极体；三个极体最终退化

### 4. 减数分裂停滞调控机制
- **前期Ⅰ停滞维持**：
  - 卵泡颗粒细胞分泌Nppc → 激活卵丘细胞NPt2受体 → 生成cGMP
  - cGMP经Cx37缝隙连接进入卵母细胞 → 抑制PDE3A → cAMP升高 → 激活PKA
  - PKA激活Wee1B（使Cdk1失活）并抑制Cdc25B → MPF（Cdk1-cyclin B）活性低
  - APC/C^Cdh1持续降解cyclin B1，Emi1蛋白进一步抑制MPF
- **LH峰解除停滞**：
  - LH surge关闭缝隙连接 → cGMP无法进入卵母细胞
  - PDE3A水解cAMP → cAMP下降 → PKA失活
  - Wee1B失活、Cdc25B激活 → Cdk1去磷酸化激活 → MPF活化
  - 核膜破裂，减数分裂恢复

> 注意：出生后不再新增卵原细胞；一生中仅约400–500个卵泡最终排卵。
