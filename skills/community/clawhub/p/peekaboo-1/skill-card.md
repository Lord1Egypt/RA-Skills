## Description: <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and automation agents use this skill to capture, inspect, and automate macOS UI workflows with the Peekaboo CLI, including screenshots, element targeting, input automation, and app or window management. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill enables an agent to automate visible macOS UI and may act on sensitive apps or data shown on screen. <br>
Mitigation: Use it only when you intend UI automation, avoid sensitive content while it runs, and review actions before clicks or typing in important apps. <br>
Risk: Screen Recording and Accessibility permissions grant broad local interaction capability. <br>
Mitigation: Grant permissions only for the needed workflow and revoke them when the tool is no longer required. <br>
