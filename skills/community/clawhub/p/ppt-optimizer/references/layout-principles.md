# PPT 排版原则

## 布局黄金法则

### 60-30-10 原则
- **60% 主色**：背景和主要区域
- **30% 辅色**：次要元素和装饰
- **10% 强调色**：重点信息和引导

### 留白原则
- 页面边距：至少 10% 页面宽度
- 元素间距：统一为 8px 或 16px 倍数
- 留白 ≠ 浪费，留白 = 呼吸感

## 常见页面布局

### 1. 标题页布局
```
┌─────────────────────────┐
│                         │
│                         │
│       [大标题]          │
│       [副标题]          │
│                         │
│    [日期/姓名/Logo]     │
│                         │
└─────────────────────────┘
```
- 标题居中或左对齐
- 字号：标题 44-60pt，副标题 24-32pt

### 2. 要点列表布局
```
┌─────────────────────────┐
│        [标题]           │
├─────────────────────────┤
│  • 要点 1               │
│    说明文字             │
│                         │
│  • 要点 2               │
│    说明文字             │
│                         │
│  • 要点 3               │
│    说明文字             │
└─────────────────────────┘
```
- 要点数量：3-5 个最佳
- 每要点 1-2 行文字

### 3. 图文左右布局
```
┌───────────┬─────────────┐
│           │             │
│           │  [标题]     │
│  [图片]   │             │
│           │  • 要点 1   │
│           │  • 要点 2   │
│           │  • 要点 3   │
│           │             │
└───────────┴─────────────┘
```
- 图片占 40-50% 宽度
- 文字左对齐或两端对齐

### 4. 图文上下布局
```
┌─────────────────────────┐
│        [标题]           │
├─────────────────────────┤
│                         │
│        [图片]           │
│                         │
├─────────────────────────┤
│  说明文字或要点列表     │
└─────────────────────────┘
```
- 适用于图片需要强调的场景
- 图片高度占 50-60%

### 5. 对比布局
```
┌───────────┬─────────────┐
│  [标题 A] │  [标题 B]   │
├───────────┼─────────────┤
│  • 要点   │  • 要点     │
│  • 要点   │  • 要点     │
│  • 要点   │  • 要点     │
└───────────┴─────────────┘
```
- 适用于对比分析
- 左右内容对称

### 6. 三栏布局
```
┌──────────┬──────────┬──────────┐
│ [图标 1] │ [图标 2] │ [图标 3] │
│  标题 1  │  标题 2  │  标题 3  │
│  描述 1  │  描述 2  │  描述 3  │
└──────────┴──────────┴──────────┘
```
- 适用于并列信息展示
- 每栏宽度相等

### 7. 数据图表布局
```
┌─────────────────────────┐
│        [标题]           │
├───────────┬─────────────┤
│           │  关键数据   │
│  [图表]   │  • 数据 1   │
│           │  • 数据 2   │
│           │  • 数据 3   │
└───────────┴─────────────┘
```
- 图表为主，数据为辅
- 关键数据突出显示

### 8. 引用/金句布局
```
┌─────────────────────────┐
│                         │
│   "引用文字或金句"      │
│                         │
│           —— 作者       │
│                         │
└─────────────────────────┘
```
- 居中对齐
- 字号放大，字间距加宽
- 简洁有力

## 对齐原则

### 统一对齐方式
- **左对齐**：最常用，符合阅读习惯
- **居中对齐**：标题、引用、强调内容
- **右对齐**：日期、数字、特殊效果

### 对齐技巧
- 使用参考线辅助对齐
- 相关元素靠近，不相关元素远离
- 视觉重心平衡

## 字体规范

### 字号建议
| 元素 | 字号范围 |
|------|----------|
| 大标题 | 44-60pt |
| 小标题 | 28-36pt |
| 正文 | 18-24pt |
| 注释 | 12-16pt |

### 字体搭配
- **标题**：粗体、无衬线（思源黑体、微软雅黑）
- **正文**：常规、无衬线
- **强调**：加粗或颜色
- **全篇字体 ≤ 2 种**

### 行间距
- 标题：1.2 倍行高
- 正文：1.5 倍行高
- 多行文字必须设置行间距

## 排版检查清单

- [ ] 页面边距是否一致？
- [ ] 元素对齐是否准确？
- [ ] 留白是否充足？
- [ ] 字号层级是否清晰？
- [ ] 配色是否统一？
- [ ] 重点是否突出？
- [ ] 视觉重心是否平衡？
- [ ] 全篇风格是否一致？
