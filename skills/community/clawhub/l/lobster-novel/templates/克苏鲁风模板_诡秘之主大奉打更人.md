# 克苏鲁风模板——诡秘之主与大奉打更人写作风格提取

> 数据来源：中国作家网专访、知乎专栏研究、Scribd学术论文、观察者网解读、百度百科、腾讯新闻等

## 一、两书定位

这两部作品代表了克苏鲁风在中国网文中的两种落地形态：

| 维度 | 诡秘之主 | 大奉打更人 |
|------|---------|-----------|
| 作者 | 爱潜水的乌贼 | 卖报小郎君 |
| 年代 | 2018-2020 | 2020-2021 |
| 字数 | 约445万 | 约380万 |
| 核心标签 | 克苏鲁+蒸汽朋克+SCP+维多利亚 | 悬疑探案+仙侠+搞笑+穿越 |
| 克苏鲁浓度 | 直系——核心设定就是克苏鲁 | 旁系——用探案悬疑营造诡秘感 |
| 恐怖来源 | 不可名状的外神、序列失控、疯狂 | 案件背后的黑暗真相、权谋阴谋 |
| 整体基调 | 压抑、诡秘、理性对抗疯狂 | 搞笑轻松为主、悬疑为辅 |

## 二、诡秘之主——克苏鲁风的正统标杆

**定位**：起点中文网巅峰之作，克苏鲁神话本土化的天花板。2018-2020多次打破起点纪录，被评「超脱了网络文学的小说」。

### 2.1 克苏鲁核心设定体系

**1. 序列晋升体系（魔药途径）**
- **22条序列途径**，每条从序列9到序列0（真神），最后晋升「旧日」
- 晋升方式：**喝魔药→扮演消化→晋升下一序列**
- **扮演法**（全书核心设定）：你要「扮演」对应的序列角色，但不能入戏太深
- 核心格言：**「你只是在扮演」**
- 如果不扮演消化，魔药中的残余精神会侵蚀你→失控→变成怪物

**2. 非凡特性三大定律**
- **非凡特性不灭定律**：特性不会消失，只会转移
- **非凡特性聚集定律**：同途径的特性会相互吸引
- **守恒定律**：总量不变，零和博弈

**3. 疯狂值系统**
- 了解越多→越接近真相→越容易疯狂
- 窥视高位存在→听到呓语→san值狂掉
- 对应克苏鲁核心：「知道得太多本身就是危险」

**4. 锚（Anchor）系统**
- 与普通人/信徒建立社会关系→形成「锚」→稳固人性
- 没有足够的锚→神性压倒人性→变成没有感情的怪物
- 本质：人的本质是一切社会关系的总和

### 2.2 克苏鲁氛围营造技法

**1. 不可名状的恐怖**
- 高位存在不能直接描述，只能通过「呓语」「疯狂」「扭曲」侧面暗示
- 读者永远不知道「福生玄黄天尊」「上帝」「原初造物主」到底长什么样
- 知道越多越危险——与克苏鲁神话「san值」机制完美对应

**2. 理性与疯狂的对峙**
- 主角克莱恩·莫雷蒂（周明瑞）始终保持理性克制
- 用序列笔记、塔罗会会议、情报收集等系统化手段对抗混乱
- **「我们是守护者，也是一群时刻对抗着危险和疯狂的可怜虫」**
- 高度理性的人设本身就是对克苏鲁恐怖的最大反衬

**3. 多层世界的悬疑感**
- 现实世界（伦堡、鲁恩）→非凡世界→神灵世界→旧日宇宙
- 每揭开一层，发现上一层的「真相」只是更大恐怖的冰山一角
- 世界观的真相是「末日将至，外神窥视」
- 悬念层层递进，绝不一次性透露全部设定

**4. 维多利亚时代+蒸汽朋克的落地感**
- 伦堡=维多利亚时代的伦敦：雾气弥漫的街道、马车、报纸、煤气灯
- 蒸汽与机械：开局主角是「值夜者」小职员，坐马车、喝红茶、去公司打卡
- 真实的历史细节让克苏鲁的荒诞感更有冲击力
- 乌贼自称「查阅大量资料还原维多利亚时代社会」

**5. SCP式的文档风格**
- 「序列9占卜家魔药配方：……辅助材料：……」
- 塔罗会的会议记录、值夜者的档案风格
- 用一种「极端理性」「档案化」的语言描述极端荒诞的事物
- 反差产生恐怖：用官僚报告的语气写吃人怪物

### 2.3 叙事结构

**三卷式递进**
- 第一卷「小丑」：克莱恩成为非凡者，学习规则，建立塔罗会（世界观引入）
- 第二卷「无面人」：克莱恩前往海上，扮演「愚者」，势力扩张（世界观展开）
- 第三卷「旅行家」至第五卷「红祭司」：真相逐步显现，「愚者」身份坐实（世界观收束）
- **塔罗会作为长期暗线**：每次塔罗会都是信息交换+伏笔回收的节点

**叙诡手法**
- 克莱恩的多重身份（周明瑞/克莱恩/夏洛克/格尔曼·斯帕罗/道恩·唐泰斯/「愚者」）
- 每一重身份都是一个面具，读者和书中其他角色一样不知道哪层是真实的
- 大量的隐藏身份、伪装、叙事误导

### 2.4 语言风格

- **白描为主，冷静克制**：即使描写恐怖场景也不煽情，用客观叙述制造细思极恐
- **文档化语言**：序列配方、塔罗会记录、值夜者档案——理性文体书写非理性内容
- **大量伏笔和暗示**：一个不起眼的细节可能在数十万字后回收
- **克苏鲁式禁忌**：从不直接描写神的面貌，只用「不可名状」「扭曲」「疯狂的呓语」
- 整体风格：冷峻、精密、缜密

### 2.5 写作要点提取

| 技法 | 说明 | 应用方式 |
|------|------|---------|
| **序列体系** | 22条途径、每途径10级、魔药+扮演 | 为克苏鲁世界观提供逻辑骨架 |
| **扮演法** | 扮演对应身份消化魔药 | 解决「为什么要做任务」的逻辑 |
| **锚系统** | 社会关系稳固人性 | 将「神性vs人性」具象化 |
| **不可名状** | 高位存在永远不直接描写 | 保留神秘感和恐怖感 |
| **理性文风** | 用冷静白描写荒诞恐怖 | 反差增强恐怖 |
| **SCP文档** | 档案式语言写超自然 | 增加真实感和代入感 |
| **多层世界** | 一层层揭开更大的恐怖 | 持续制造悬疑 |
| **塔罗会** | 固定节点做信息交换+伏笔回收 | 长篇叙事的结构锚点 |
| **维多利亚落地** | 真实历史细节打底 | 让荒诞有根基 |
| **san值机制** | 知道越多越危险 | 控制信息释放节奏 |

## 三、大奉打更人——悬疑探案仙侠的克苏鲁旁系

**定位**：2020年现象级仙侠，探案+修仙+搞笑三合一，长期起点月票榜前二。克苏鲁浓度不高，但悬疑探案线的「谜团逐层揭露」结构与克苏鲁的「不可知」精神相通。

### 3.1 核心设定体系

**1. 七大修行体系（10品阶制）**
- 武夫（打更人主力）：9品→超品武神
- 术士（司天监）：9品医师→超品天命
- 儒家（云麓书院）：9品开窍→超品儒圣
- 佛门（西域佛国）：9品沙弥→超品佛陀
- 道门（天地人三宗）：9品筑基→超品道尊
- 巫师（巫神教）：9品血灵→超品巫神
- 蛊师（十万大山）：9→超品蛊神

**2. 打更人组织（垂直职级体系）**
- 白役（编外）→铜锣（8品武夫）→银锣（7-6品）→金锣（4品）→魏渊（2品）
- **公务员式的升级体系**——网文中罕见的「有编制的修仙」
- 职责：夜间巡逻、监察百官、情报收集、案件侦破

**3. 探案驱动的世界观揭示**
- 每个案子揭开世界真相的一角（如剥洋葱）
- 前200万字是「一个大案子由众多环环相扣的小案件组成」
- 读者的视角随案件推进不断扩展：京城→南方→北方→西域→海外

### 3.2 悬疑与克苏鲁精神的结合

**1. 「知得越多，漩涡越大」的递进结构**
- 开局：许七安只是个小捕快，想查清自己的冤案
- 中期：查一个案子引出更大的阴谋→涉及皇室→涉及神魔
- 后期：发现背后是500年的布局、神魔复活的深渊
- 这种「查一个案子→发现更大的黑暗」的结构，在感觉上与克苏鲁精神相通

**2. 超凡手段的悬疑用法**
- 主角用现代推理+超凡能力破案
- 术士的望气术、占卜师的水镜术、武夫的炼神感应力→都变成了探案工具
- 但超自然力量不能直接告诉你答案→必须推理才能组合线索

**3. 恐怖氛围的间歇性插入**
- 并非全篇恐怖，而是在搞笑日常中突然插入恐怖场景
- 诸如桑泊案的白骨、血屠三千里、蛊术的诡异感
- 克制使用——正因为平时搞笑，恐怖场景的冲击力被放大

### 3.3 搞笑+悬疑的双轨并行

**双线叙事**
- 主线：悬疑案件（桑泊案→云州案→楚州案→国战）
- 副线：搞笑日常（许七安吐槽、许铃音吃、杨千幻装逼、勾栏听曲）

**核心技巧：悬疑和搞笑不互相削弱**
- 搞笑段子集中在人物互动和吐槽——不影响案件推理解谜
- 案件推理保持严肃——不影响搞笑段子的节奏
- 两者交替出现，形成「紧绷→松弛→紧绷」的节奏

### 3.4 人物塑造

**许七安（主角）**
- 穿越者，警校毕业，有现代推理能力
- 性格：有原则有底线、大是大非分明，但喜欢捉弄人、轻浮、贪财
- 身世极复杂：父亲许平峰（司天监大弟子+潜龙城国师）、母亲姬白晴（皇室后裔）
- 最像普通人的「大侠」——读者共鸣感强

**群像塑造的突出之处**
- 许铃音（吃货小妹）——热度不比主角低
- 杨千幻（装逼大王）——读者最爱配角之一
- 魏渊（一身青衣的宦官首领）——孤寂又强大的布局者
- 卖报强调「我立起了一个又一个有趣的人物」

### 3.5 写作要点提取

| 技法 | 说明 | 应用方式 |
|------|------|---------|
| **案中案式大结构** | 前200万字是一个大案子由小案串联 | 保持长篇悬疑的持续吸引力 |
| **多修行体系** | 7大体系+10品阶 | 提供丰富的世界观博弈空间 |
| **公务员修仙** | 打更人是有编制的组织 | 为成长线提供天然阶梯 |
| **搞笑+悬疑双轨** | 轻松日常与紧张探案交替 | 调节阅读节奏、防止疲劳 |
| **群像制胜** | 每个配角都有记忆点 | 增加世界真实感和读者黏性 |
| **现代知识破案** | 穿越者的认知不对称 | 提供「爽感」来源 |
| **克制恐怖** | 恐怖场景只在关键时刻出现 | 放大冲击力（好钢用在刀刃上） |
| **层层揭示** | 每个案件揭开世界的一角 | 保持读者好奇心 |
| **7大势力博弈** | 朝廷+江湖+宗教+神魔 | 提供长线政治/权谋张力 |
| **正能量底调** | 主角维护秩序、弘扬正义 | 爽文需要的基本价值观 |

## 四、克苏鲁风通用模板

### 4.1 克苏鲁浓度的分级运用

| 浓度等级 | 代表作品 | 核心手法 | 适用场景 |
|---------|---------|---------|---------|
| **正统克苏鲁** | 诡秘之主 | 序列体系+不可名状+疯狂值+锚 | 严肃诡秘向、硬核克系 |
| **悬疑克苏鲁** | 大奉打更人 | 案中案+层层揭真相+黑暗权谋 | 轻克系、探案为主 |
| **氛围克苏鲁** | 一般网文借鉴 | 只取「不可知」「细思极恐」元素 | 当作调味料而非主菜 |

### 4.2 克苏鲁风核心工具箱

**一、设定层**
1. **序列/等级体系**：为超自然力量建立逻辑骨架，避免力量体系混乱
2. **代价机制**：获得力量必有代价（疯狂/失控/寿命/人性）
3. **扮演/锚系统**：让力量获取与角色成长挂钩，而非纯粹的数值增长
4. **禁忌知识**：了解越多越危险——控制信息释放节奏

**二、氛围层**
5. **不可名状**：高位存在永远不直接描写（「扭曲的」「疯狂的」「呓语」「不可名状」）
6. **理性文风**：用冷峻白描写荒诞恐怖——反差制造恐惧
7. **客观化语言**：SCP文档/档案/报告风格写超自然
8. **间歇性恐怖**：安静日常中突然插入恐怖场景（不要全程压抑）

**三、结构层**
9. **多层世界**：真相是一层层剥开的，每一层揭示更大的恐怖
10. **塔罗会式固定锚点**：定期举行的信息交换/伏笔回收节点
11. **案中案结构**：小案→大案→更大的案（大奉式）
12. **多重身份/面具**：主角有多个身份，读者也分不清哪个是真的

**四、世界层**
13. **真实历史落地**：用维多利亚/明清/架空王朝的真实细节打底
14. **势力博弈**：朝廷/江湖/宗教/神魔多条线并行
15. **末日倒计时**：给读者「时间紧迫感」

### 4.3 诡秘之主 vs 大奉打更人的路线选择

| 维度 | 选诡秘路线 | 选大奉路线 |
|------|-----------|-----------|
| 基调 | 压抑、诡秘、严肃 | 轻松、搞笑、有底线的爽 |
| 恐怖 | 全程笼罩的可控san值丢失 | 间歇性的恐怖场景 |
| 主角 | 理性克制、步步为营 | 轻浮幽默、有弱点接地气 |
| 力量体系 | 22条途径+扮演法（复杂） | 7大体系+10品（相对简单） |
| 笑点 | 极少（乌贼式冷幽默偶尔出现） | 极多（卖报的搞笑功底） |
| 适合人群 | 硬核设定党、克苏鲁原教旨 | 大众读者、喜欢轻松悬疑 |

### 4.4 克苏鲁风写作禁忌

❌ 把神/外神写得明明白白——「不可名状」一旦具象化就不可怕了
❌ 全程高能压抑——读者会疲劳（需要间歇性松弛，像大奉的搞笑日常）
❌ 力量体系不严谨——克苏鲁风的恐怖依赖于「规则明确、代价清晰」
❌ 信息一次性释放——悬念的核心是「让读者知道得比主角少」
❌ 全是恐怖的廉价惊吓——克苏鲁的恐怖是细思极恐、而不是突然跳脸吓人

## 五、风格质量门控（refqc-cthulhu）

### 5.1 门控概览

| 层级 | 检测项 | 检查周期 | 通过线 |
|------|-------|---------|-------|
| L1 | 克苏鲁风格一致性 | 每3000字 | 全部通过 |
| L2 | 设定完整性 | 每20000字 | 通过率≥85% |
| L3 | 恐怖节奏疲劳 | 每20000字 | 无黄/红预警 |

### 5.2 L1 风格一致性检测（每3000字）

克苏鲁风是最容易跑偏的风格——写着写着就把神写明白了，或者恐怖感丢了。

| 编号 | 检查项 | 说明 | 不通过表现 |
|------|-------|------|----------|
| K-L1-01 | 不可名状原则 | 高位存在不直接描写形貌/语言/思维 | 神直接出场、张嘴说话、外表详细描写 |
| K-L1-02 | 恐惧来自未知 | 恐怖感源于「理解不了」而非「打不过」 | 变成普通打怪升级/战斗力差距 |
| K-L1-03 | 冷静客观语调 | 叙述者保持冷静理性，不煽情 | 过度的感叹号、夸张的形容词堆砌 |
| K-L1-04 | 有代价的超自然力量 | 使用超自然力量必有代价（san/寿命/人性） | 无代价施法或使用能力 |
| K-L1-05 | 信息控制 | 重要设定不一次性交代，有控制地释放 | 一段话揭露全部世界观 |
| K-L1-06 | 无跳脸惊吓 | 恐怖来自氛围积累而非突然跳出吓人 | 开门杀/回头杀等廉价吓人手段 |

### 5.3 L2 设定完整性校验（每20000字）

| 编号 | 检查项 | 说明 |
|------|-------|------|
| K-L2-01 | 力量体系未膨胀 | 没有凭空出现碾压当前阶段的超规格力量 |
| K-L2-02 | 疯狂值/代价体系一致 | 获得力量的代价没有因为主角光环而减免 |
| K-L2-03 | 规则类设定无自相矛盾 | 序列等级/扮演法则/锚系统的规则前后一致 |
| K-L2-04 | 「知道越多越危险」维持 | 主角没有因为变强就不再受到禁忌知识的威胁 |
| K-L2-05 | 悬疑嵌套结构未断层 | 每揭开一层真相，都要引出更大的未知 |
| K-L2-06 | 历史/蒸汽朋克落地感未丢失 | 细节描写没有缩水变成纯粹的架空 |

### 5.4 L3 恐怖节奏与疲劳检测（每20000字）

克苏鲁风最大的风险是读者**恐怖疲劳**——全程压抑反而没感觉了。

| 预警级别 | 判断标准 | 处置方式 |
|---------|---------|---------|
| 🟢 正常 | 恐怖场景与平静场景交替，间隔不超过3章全是恐怖 | 继续 |
| 🟡 黄色预警 | 连续3章以上全都是压抑/恐怖/紧张 | 插入至少1章日常/对话/温馨场景缓冲 |
| 🔴 红色预警 | 连续5章以上高恐怖密度，或恐怖描写手法重复超过3次 | 大幅降速，插入2章以上非恐怖章节重置节奏 |

**恐怖手段多样性检测**：恐怖描写的方式（环境暗示/心理描写/真相揭露/氛围渲染等）在连续10章中至少切换3种不同手段，避免单一手法用到底。
