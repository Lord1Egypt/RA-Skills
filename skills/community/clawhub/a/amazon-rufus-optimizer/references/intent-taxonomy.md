# Rufus 意图标签分类参考表

## 四维意图体系

Rufus通过四个维度的意图标签来匹配用户需求和产品：

---

## 1. 使用意图 (Usage Intent)

产品的主要使用目的和功能场景。

| 标签 | 适用产品示例 | 用户搜索词示例 |
|------|-------------|---------------|
| 日常使用 | 家居用品、厨房工具 | "每天都能用的..." |
| 专业用途 | 专业设备、工具 | "专业摄影师用的..." |
| 旅行便携 | 旅行配件、收纳包 | "旅行携带方便" |
| 运动健身 | 运动装备、健身器材 | "适合健身用" |
| 办公商务 | 办公用品、电子设备 | "办公室用" |
| 户外探险 | 户外装备、露营用品 | "户外活动必备" |
| 家居装饰 | 装饰品、家具 | "提升家居品味" |
| 送礼收藏 | 礼品、收藏品 | "送人的礼物" |

## 意图标签优先级建议

### 高价值标签（必填）

根据Rufus推荐逻辑，以下标签对推荐影响最大：

1. **核心使用意图** - 产品的主要功能
2. **主要人群意图** - 目标用户群体
3. **高频场景意图** - 最常见的使用场景
4. **核心情感诉求** - 用户最关心的感受

### 扩展标签（建议补充）

提升跨类目推荐概率：

1. **户外场景** → 匹配户外装备类目
2. **办公场景** → 匹配办公用品类目
3. **送礼标签** → 匹配礼品类目
4. **健康安全** → 匹配健康护理类目

## 意图覆盖度计算

```
意图覆盖度 = 已覆盖标签数 / 应覆盖标签总数 × 100%

建议目标：
- 核心意图：100% 覆盖
- 扩展意图：>50% 覆盖
- 总覆盖率：>60% 视为健康
```

## 常见缺失意图问题

| 问题 | 解决方案 |
|------|---------|
| 缺少场景词 | 在标题/BP中补充"office/户外/旅行"等场景词 |
| 缺少人群词 | 补充"送给爸妈/新手适用/家庭必备"等人群词 |
| 缺少情感词 | 强调"安全无毒/舒适耐用/高颜值"等情感词 |
| 参数堆砌 | 改写为对话式："出汗也不滑——防滑纹理" |
