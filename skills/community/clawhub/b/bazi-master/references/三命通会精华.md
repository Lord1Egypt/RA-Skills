# 三命通会 精华摘要

> 《三命通会》由明代万民英编撰，是子平命理的集大成之作。此书博采众长，以长生十二宫和神煞体系为特色，兼顾格局理法，是命理师案头必备的参考书。

## 核心理念

### 一、长生十二宫

长生十二宫是分析日主在命局中状态的重要参照：

| 状态 | 含义 | 象征 |
|------|------|------|
| 长生 | 初生、新生 | 希望、生长 |
| 沐浴 | 洗礼、裸裎 | 桃花、败露 |
| 冠带 | 加冠、成人 | 礼仪、体面 |
| 临官 | 出仕、临民 | 独立、承担 |
| 帝旺 | 极盛、顶巅 | 强势、顶点 |
| 衰 | 转弱 | 退守、沉淀 |
| 病 | 生病 | 缺失、问题 |
| 死 | 终结 | 僵化、结束 |
| 墓 | 收藏、埋葬 | 库房、归宿 |
| 绝 | 绝地 | 破局、重生 |
| 胎 | 胚胎、孕育 | 机会、萌芽 |
| 养 | 养育、成形 | 酝酿、准备 |

#### 应用原则

1. 日主在十二宫的状态，反映日主在命局中的"生命力"
2. **长生、冠带、临官、帝旺、胎、养**为"进气"
3. **衰、病、死、墓、绝、沐浴**为"退气"
4. 结合月令、岁运，判断吉凶

#### 命局层次

- **日主坐长生**：有生气，有韧性
- **日主坐帝旺**：刚强、自负
- **日主坐死墓绝**：弱势、被动
- **日主坐沐浴**：桃花、情感丰富

4. 关键看神煞与其他十神的配合

### 四、六亲论断

#### 六亲对应

| 六亲 | 对应十神 | 宫位 |
|------|---------|------|
| 父亲 | 偏财 | 年柱 |
| 母亲 | 正印 | 年柱 |
| 兄弟 | 比肩、劫财 | 月柱 |
| 姐妹 | 比肩、劫财 | 月柱 |
| 配偶 | 正财（男）、正官（女） | 日支 |
| 子女 | 食神、伤官 | 时柱 |

#### 六亲吉凶判断

1. **六亲星得令、无伤**：该六亲吉利
2. **六亲星失令、有伤**：该六亲有缺
3. **六亲宫位被冲克**：该六亲变动或分离
4. **六亲星被合走**：该六亲缘分薄

### 五、大运流年

#### 大运

- 始于月柱，按阳男阴女顺排，阴男阳女逆排
- 每运管十年
- 大运的五行与命局的生克制化，决定这十年的基本走势

#### 流年

- 以年柱为主
- 流年地支为"太岁"，为一年之主
- 流年天干为"值事"，主当年之事

#### 限运

- 年柱管1-18岁
- 月柱管19-35岁
- 日柱管36-53岁
- 时柱管54岁以后

### 六、断运要诀

1. **大运与命局合看**：生助为吉，克制为凶
2. **流年引动**：流年引发命局中的吉凶神煞
3. **冲合并见**：冲合会改变命局的平衡
4. **进退气**：进气之运吉，退气之运凶
5. **天克地冲**：大运与命局天克地冲，多有重大变动

### 七、格局分类

#### 正格外格

正格之外，还有一些特殊格局：

| 格局 | 条件 | 特点 |
|------|------|------|
| 从财格 | 日主无根，满盘财星 | 富屋贫人 |
| 从杀格 | 日主无根，满盘官杀 | 贵气凛然 |
| 从儿格 | 日主无根，满盘食伤 | 才华横溢 |
| 从旺格 | 日主得令，满盘比劫 | 刚强自立 |
| 化气格 | 合化成象 | 专用所化之五行 |
| 专旺格 | 五行纯一 | 气势凝一 |

#### 格局成败

- **成格**：用神有力，配合有情
- **破格**：用神受损，配合无情
- **从格**：日主无根，不得不从
