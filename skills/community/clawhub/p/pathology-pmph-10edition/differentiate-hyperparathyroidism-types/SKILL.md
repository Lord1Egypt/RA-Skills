# 甲状旁腺功能亢进分型与鉴别

## 何时使用
当患者满足以下任一条件时启动本技能：
- 血清PTH升高，同时伴有血钙和/或血磷异常
- 存在典型症状（如肾结石、便秘、多饮多尿）
- 有慢性肾病或其他可能导致钙磷代谢紊乱的病史

## 执行步骤

1. **确认基本生化异常**：验证是否存在高PTH，并记录血清钙、磷浓度及肾功能指标（肌酐、尿素氮、肌酐清除率）。

2. **评估病史**：明确是否存在慢性低钙血症病史或慢性肾脏病等基础疾病。

3. **根据以下标准进行分型**：
   - **原发性甲旁亢**：
     - 高PTH + **高钙** + **低磷**
     - 肾功能正常
     - 无明确继发诱因
   - **继发性甲旁亢**：
     - 高PTH + **低钙** + **高磷**
     - 肾功能异常（肌酐↑、尿素氮↑、清除率↓）
     - 存在导致低钙的基础疾病（如慢性肾病）
   - **三发性甲旁亢**：
     - 高PTH + **高钙** + **高磷**
     - 肾功能异常
     - **既往有明确低钙病史**（即曾诊断为继发性甲旁亢）

4. **关键注意事项**：
   - 不可仅凭甲状旁腺病理形态（如腺瘤或增生）区分类型
   - 必须结合生化数据与临床背景综合判断

5. **输出结果**：明确甲旁亢类型（原发性/继发性/三发性）并列出支持该判断的关键依据。
