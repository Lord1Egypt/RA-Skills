# broadband-grant-application-drafter

**Domain:** Broadband Planning  
**Skill:** `broadband-grant-application-drafter`  
**Version:** 0.1.0

## What It Does

Converts project facts — applicant information, service area, technology selection, deployment timeline, and cost estimates — into a structured grant application narrative for federal and state broadband funding programs.

Supports:
- **BEAD** (Broadband Equity, Access, and Deployment) — NTIA, $42.45B program
- **USDA ReConnect** — rural broadband loan and grant program
- **E-Rate** — FCC Schools and Libraries program
- **State broadband grant programs** — state broadband offices and digital equity offices

Produces a **DRAFT** narrative for applicant review, engineering verification, and legal counsel sign-off before portal submission.

## When to Use

Use this skill when an ISP, rural electric cooperative, municipality, tribal organization, or broadband office needs to:

- Draft the project narrative section of a BEAD subgrantee application
- Prepare a ReConnect application project description
- Build a structured service area description, technology justification, and deployment plan for a state broadband grant
- Develop an affordability and adoption plan required by most federal broadband programs
- Compile an application readiness checklist before a grant deadline

## Scope and Boundaries

**This skill covers:**
- Program identification and eligibility pre-check
- Applicant profile and qualifications narrative
- Service area definition and unserved/underserved confirmation
- Technology selection and speed commitment narrative
- Phased deployment plan with milestone schedule
- Project budget with cost-per-location analysis
- Affordability plan, low-income program commitments, and digital equity measures
- Reviewer sign-off block and open-items list

**This skill does not:**
- Submit applications to any government portal
- Generate FCC Form filings (FCC Form 471, 472, 473, etc.)
- Certify cost estimates — engineering verification is required
- Provide legal advice on eligibility, compliance, or procurement
- Guarantee award or compliance with program rules — legal counsel review is required

## Output

A structured Markdown application narrative labeled **DRAFT**, with all information gaps flagged and a reviewer sign-off table.

## Example Use Cases

- A rural electric cooperative in Kentucky needs a BEAD subgrantee application narrative to connect 2,400 unserved locations with FTTP
- A tribal government in the Southwest is applying for USDA ReConnect Round 4 to build a fiber ring serving a reservation
- A small WISP in rural Iowa needs a state broadband grant application narrative to upgrade tower backhaul and expand FWA coverage to 600 new locations
- A municipal utility is applying for state digital equity grant funding for a community digital literacy program
