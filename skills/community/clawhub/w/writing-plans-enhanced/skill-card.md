## Description: <br>
Use when you have a spec or requirements for a multi-step task, before touching code. Enhanced version with milestone timelines, data flow diagrams, mockups, and risk tables. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[paudyyin](https://clawhub.ai/user/paudyyin) <br>

## Use Case: <br>
Developers and engineers use this skill to create detailed implementation plans before code changes, including timelines, data flow, mockups, risks, and task-by-task execution handoff. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated plans can influence future code changes through task steps, commit guidance, and execution handoff instructions. <br>
Mitigation: Review generated plans before using them to drive implementation, especially suggested tasks, commits, and subagent workflow. <br>
