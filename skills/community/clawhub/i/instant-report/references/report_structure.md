# 产业研究报告结构模板

## A. 公司研究（Company Research）

```
第一章 公司概况
  1.1 基本信息（成立时间、总部、主要业务）
  1.2 发展历程与里程碑
  1.3 股权结构（主要股东、机构持仓）

第二章 业务与产品
  2.1 业务布局（各业务线收入占比）
  2.2 核心产品/服务
  2.3 上下游产业链位置

第三章 财务表现
  3.1 营收与利润趋势（3-5年）
  3.2 盈利能力分析（毛利率、净利率）
  3.3 现金流状况
  3.4 财务预测

第四章 竞争格局
  4.1 行业定位
  4.2 SWOT 分析
  4.3 核心竞争优势
  4.4 主要竞争对手对比

第五章 战略展望
  5.1 未来战略方向
  5.2 潜在增长点
  5.3 风险因素
```

## B. 行业研究（Industry Research）

```
第一章 行业概述
  1.1 行业定义与分类
  1.2 行业发展历程
  1.3 产业链结构

第二章 市场规模与增长
  2.1 全球/区域市场规模（TAM/SAM/SOM）
  2.2 历史增长率
  2.3 细分市场分析
  2.4 市场增长驱动因素

第三章 竞争格局
  3.1 主要玩家与市占率
  3.2 市场集中度
  3.3 竞争壁垒
  3.4 波特五力分析

第四章 关键趋势与驱动因素
  4.1 技术路线演进
  4.2 政策环境
  4.3 需求端变化
  4.4 供给端变革

第五章 未来展望
  5.1 2024-2030 市场规模预测
  5.2 新兴机会
  5.3 主要风险与挑战
```

## C. 对比分析（Comparative Analysis）

```
第一章 比较框架
  1.1 比较对象
  1.2 比较维度与方法论

第二章 横向对比
  2.1 市场规模对比（表格）
  2.2 财务指标对比
  2.3 产品/服务对比
  2.4 技术实力对比

第三章 竞争定位
  3.1 差异化定位图
  3.2 各玩家优劣势分析

第四章 综合评价
  4.1 评分矩阵
  4.2 分析结论
  4.3 战略建议
```
