# 推运应期分析模板（v2.0）

**定位**：从本命征象到精确应期的标准化分析框架，确保推运结论可验证、可追溯。
**版本**：2.0.0
**更新**：2026-04-24 — 增加Pratyantar微运+Double Transit+事件专属应期公式+五级时间精度

---

## 使用说明

1. 确保PDF数据已通过质量门（见 `pdf-chart-reading-guide.md` 第四、五节）
2. 按模板逐层填写分析
3. **五级验证必须全部完成才能给出精确应期**
4. 缺少任何一层，必须在结论中标注"精度受限"
5. 最终应期必须给出：主窗口（月级）+ 次窗口（周级）+ 确认信号

## 附录：事件专属应期公式

### A. 婚姻应期公式

**触发条件**：
1. Dasha层：当前Dasha涉及7宫主、Venus（男）/Mars+Jupiter（女）、DK
2. Transit层：木星或金星过境/相位7宫（从Lagna和Chandra Lagna双看）
3. Double Transit：土星+木星同时与7宫产生联系
4. D9确认：D9的7宫被激活

**精确应期窗口**：
- 主窗口 = 木星过境/相位D1的7宫期间 + Dasha涉及7宫主/Venus/DK
- 次窗口 = 金星过境/相位7宫期间（更短但更精确）
- 确认信号 = 木星和金星同时在7宫相关位置

### B. 事业突破应期公式

**触发条件**：
1. Dasha层：当前Dasha涉及10宫主、AmK、太阳、土星
2. Transit层：土星过境/相位10宫（压力=成长）；木星过境/相位10宫（机遇）
3. Double Transit：土星+木星同时与10宫产生联系
4. D10确认：D10的10宫被激活
5. ⚠️ 从Chandra Lagna看6宫 = 新工作信号（不可遗漏）

**精确应期窗口**：
- 主窗口 = 土星+木星Double Transit激活10宫期间
- 确认信号 = 木星过境10宫（从Lagna或Chandra Lagna看）

### C. 财富增长应期公式

**触发条件**：
1. Dasha层：当前Dasha涉及2宫主、11宫主、木星、金星
2. Transit层：木星过境/相位2宫或11宫
3. Double Transit：土星+木星同时与2宫或11宫产生联系
4. D2确认：D2相关宫位被激活

**精确应期窗口**：
- 主窗口 = 木星过境2宫或11宫期间 + Dasha涉及2/11宫主
- 次窗口 = 金星过境2宫或11宫期间

### D. 子女应期公式

**触发条件**：
1. Dasha层：当前Dasha涉及5宫主、PK、木星
2. Transit层：木星过境/相位5宫
3. Double Transit：木星+土星与5宫产生联系
4. D7确认：D7的5宫被激活

### E. 健康事件应期公式

**触发条件**：
1. Dasha层：当前Dasha涉及6宫主、8宫主、上升主星受克
2. Transit层：土星过境6宫/8宫/12宫；火星过境相关宫位
3. ⚠️ 6/8/12宫Transit叠加 = 高风险期
4. D6确认：D6相关宫位被激活
