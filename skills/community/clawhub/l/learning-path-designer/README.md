# learning-path-designer

Design structured, competency-aligned corporate learning paths from skill gaps to measurable outcomes.

## Overview

This skill guides L&D professionals and instructional designers through building learning paths for corporate training programs. It covers competency gap analysis (current vs. target proficiency), Bloom's taxonomy–aligned objective writing, module design with modality selection, prerequisite-respecting sequencing, and assessment planning using the Kirkpatrick model. Produces a DRAFT learning path curriculum map for L&D-lead review before content development begins.

## Use When

- Designing new hire onboarding, upskilling, role transition, or leadership development programs
- Mapping competency gaps to structured learning objectives and modules
- Building a curriculum map and build/buy/curate task list before committing resources to content development
- Aligning training design to Bloom's taxonomy and Kirkpatrick evaluation levels

## Not For

- Writing eLearning scripts, facilitator guides, or assessment items (content development phase)
- K-12 lesson plans — use `education/lesson-plan-architect` instead
- Replacing SME review of technical content accuracy

## Domain

`learning-development`

## Workflow Summary

1. **Learner and context intake** — audience, business driver, constraints, existing content inventory
2. **Competency gap matrix** — current vs. target proficiency per competency, gap priority
3. **Learning objectives** — Bloom's taxonomy–aligned objectives per gap
4. **Module design** — modality, duration, prerequisites, content status (Existing / Build / Buy / Curate)
5. **Sequencing** — scaffolded, prerequisite-respecting learning path table by week/sprint
6. **Assessment design** — formative + summative per competency cluster, Kirkpatrick level
7. **DRAFT learning path** — full curriculum map with build/buy/curate task list and L&D lead review block

## Output

DRAFT learning path document (competency map, module catalog, sequenced path table, assessment plan, task list) for L&D lead review before content development begins.
