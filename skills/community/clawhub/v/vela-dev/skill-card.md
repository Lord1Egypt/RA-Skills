This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers use this skill to scaffold, modify, debug, build, and package Xiaomi Vela quick apps for wearable devices, including `.ux` pages, manifest/config files, and `.rpk` outputs. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated or edited Vela project files and `.rpk` outputs may not match the user's intended behavior or device constraints. <br>
Mitigation: Review generated file changes and package outputs before sharing, installing, or deploying them. <br>
Risk: Build results may vary when npm dependencies resolve differently over time. <br>
Mitigation: Pin npm dependencies when reproducible builds are required. <br>
