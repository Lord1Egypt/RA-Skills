# 十五五规划纲要完整结构（2026-2030）

> 来源：中国政府网 gov.cn，新华社2026年3月13日受权播发
> 十四届全国人大四次会议2026年3月12日表决通过

---

## 一、规划总定位

"十五五"时期（2026-2030年）是我国全面建设社会主义现代化国家的关键时期，地位**承前启后**——夯实基础、全面发力，为到2035年人均GDP比2020年翻一番、达到中等发达国家水平打好基础。

## 二、七大目标领域及关键指标

### 1. 高质量发展取得显著成效
- GDP增长保持在合理区间
- 全要素生产率稳步提升
- 居民消费率明显提高
- 全国统一大市场建设纵深推进
- 新型工业化、信息化、城镇化、农业现代化取得重大进展

### 2. 科技自立自强水平大幅提高
- 全社会研发经费投入 **年均增长7%以上**
- 教育科技人才一体发展格局基本形成
- 基础研究能力显著增强
- 关键核心技术快速突破
- 并跑领跑领域明显增多

### 3. 进一步全面深化改革取得新突破
- 社会主义市场经济体制更加完善
- 高水平对外开放体制机制更加健全
- 公平更有活力的市场环境加快形成
- 全过程人民民主制度化

### 4. 社会文明程度明显提升
- 文化自信更加坚定
- 文化创新创造活力激发
- 中华文化影响力显著增强

### 5. 人民生活品质不断提高
- 城镇调查失业率 **<5.5%**
- 居民收入增长与经济增长同步
- 劳动报酬提高与劳动生产率提高同步
- 中等收入群体持续扩大
- 劳动年龄人口平均受教育年限 → **11.7年**
- 人均预期寿命 → **80岁**

### 6. 美丽中国建设取得新重大进展
- 碳达峰目标如期实现
- 单位GDP二氧化碳排放 **降低17%**
- 地级及以上城市PM2.5浓度 → **<27μg/m³**
- 优良水体比例 → **85%**
- 森林覆盖率 → **25.8%**

### 7. 国家安全屏障更加巩固
- 粮食综合生产能力 → **1.45万亿斤左右**
- 能源综合生产能力 → **58亿吨标准煤**
- 建军一百年奋斗目标如期实现

## 三、十八篇完整框架

### 第一篇：奋力开创中国式现代化建设新局面
- 第一章 发展环境
- 第二章 指导方针
- 第三章 主要目标

### 第二篇：建设现代化产业体系，巩固壮大实体经济根基
- 第四章 优化提升传统产业（钢铁/石化/船舶升级、智能制造、绿色制造、整治"内卷式"竞争）
- 第五章 培育壮大新兴产业和未来产业（新一代信息技术、新能源、生物医药、量子科技、氢能核聚变、具身智能、6G）
- 第六章 促进服务业优质高效发展（生产性服务业高端化、养老托育补短板）
- 第七章 构建现代化基础设施体系（八纵八横高铁网、非化石能源十年倍增、新型电力系统、多层次算力设施）

### 第三篇：加快高水平科技自立自强，引领发展新质生产力
- 第八章 加强原始创新和关键核心技术攻关（集成电路/工业母机/高端仪器/基础软件全链条突破、"揭榜挂帅""赛马"制度、长周期资助基础研究）
- 第九章 提高体系化创新能力（国家实验室/研究型大学/科技领军企业布局）
- 第十章 强化企业科技创新主体地位（创新资源向企业集聚、研发费用加计扣除提比例、债券市场"科技板"、大力发展创业投资）
- 第十一章 一体推进教育科技人才发展（适度扩大"双一流"、超常规布局AI/集成电路学科、高技术人才移民制度）

### 第四篇：深入推进数字中国建设，提升数智化发展水平
- 第十二章 强化算力算法数据高效供给（超大规模智算集群、国家数据资源"一本账"、"数据要素×"行动）
- 第十三章 全方位推进数智技术赋能（制造业"智改数转网联"、智慧农业、数智生活、政府治理数智化）
- 第十四章 营造健康有序的发展生态（数据产权/流通交易制度、AI治理算法备案安全评估、AI全球治理合作）

### 第五篇：建设强大国内市场，加快构建新发展格局
- 第十五章 大力提振消费（释放服务消费潜力、文旅/冰雪/低空消费、住房汽车以旧换新、落实带薪休假、探索中小学生春秋假）
- 第十六章 扩大有效投资（优化政府投资结构、激发民间投资参与铁路核电等）
- 第十七章 纵深推进全国统一大市场建设（破除地方保护、公平竞争审查刚性约束、降低物流成本、制定统一大市场建设条例）

### 第六篇：加快构建高水平社会主义市场经济体制
- 第十八章 充分激发各类经营主体活力（国资国企改革聚焦主业战略性重组、落实民营经济促进法）
- 第十九章 加快完善要素市场化配置体制机制（城乡统一建设用地市场、新型能源价格形成机制、REITs常态化发行）
- 第二十章 健全宏观经济治理体系（更加积极宏观政策、零基预算改革、消费税征收环节后移下划地方、建设金融强国）

### 第七篇：扩大高水平对外开放，开创合作共赢新局面
- 第二十一章 积极扩大自主开放（扩大电信/互联网/医疗开放、人民币国际化、自贸试验区提升、海南自贸港）
- 第二十二章 提升贸易投资合作质量（数字贸易/绿色贸易、外资服务保障）
- 第二十三章 高质量共建"一带一路"（硬联通+软联通、丝路海运、空中丝绸之路）
- 第二十四章 推动构建人类命运共同体

### 第八篇：加快农业农村现代化，扎实推进乡村全面振兴
- 第二十五章 提升农业综合生产能力（新一轮千亿斤粮食产能提升、种业振兴、耕种收综合机械化率→80%+）
- 第二十六章 推进宜居宜业和美乡村建设（县域基础设施一体化、整治农村高额彩礼）
- 第二十七章 提高强农惠农富农政策效能（第二轮土地承包到期后再延长30年试点、防止返贫致贫常态化）

### 第九篇：优化区域经济布局，促进区域协调发展
- 第二十八章 增强区域发展协调性（西部大开发、东北全面振兴、中部加快崛起、东部加快推进现代化、京津冀/长三角/大湾区动力源）
- 第二十九章 促进区域联动发展（城市群协作、经济大省挑大梁、产业转移税收碳排放指标分享）
- 第三十章 优化国土空间发展格局（主体功能区、耕地/生态红线/城镇开发边界管控）
- 第三十一章 深入推进以人为本的新型城镇化（全面设立公共户口、现代化都市圈、超大特大城市治理现代化）

### 第十篇：激发全民族文化创新创造活力
- 第三十二章～第三十六章 文化繁荣相关内容

### 第十一篇：完善人口发展战略，促进人口高质量发展
- 第三十七章 建设生育友好型社会
- 第三十八章 办好人民满意的教育（受教育年限→11.7年）
- 第三十九章 加快建设健康中国（预期寿命→80岁）
- 第四十章 积极应对人口老龄化

### 第十二篇：加大保障和改善民生力度，扎实推进共同富裕
- 第四十一章 促进高质量充分就业（失业率<5.5%）
- 第四十二章 完善收入分配制度
- 第四十三章 健全社会保障体系
- 第四十四章 推动房地产高质量发展（释放刚性和改善性住房需求）
- 第四十五章 稳步推进基本公共服务均等化
- 第四十六章 保障各类群体发展权益（妇女/儿童/残疾人/青年等）

### 第十三篇：加快经济社会发展全面绿色转型，建设美丽中国
- 第四十七章 积极稳妥推进和实现碳达峰（PM2.5→<27μg/m³、森林覆盖率→25.8%）
- 第四十八章 持续改善环境质量
- 第四十九章 提升生态系统多样性稳定性持续性
- 第五十章 加快形成绿色生产生活方式（非化石能源十年倍增行动）

### 第十四篇：推进国家安全体系和能力现代化
- 第五十一章 加强国家安全体系和能力建设
- 第五十二章 保障国家经济安全
- 第五十三章 提高公共安全治理水平
- 第五十四章 完善社会治理体系

### 第十五篇：如期实现建军一百年奋斗目标
- 第五十五章～第五十六章 国防和军队现代化

### 第十六篇：发展全过程人民民主，完善法治体系
- 第五十七章～第五十八章 民主与法治

### 第十七篇：坚持和完善"一国两制"，推进祖国统一
- 第五十九章 促进香港、澳门长期繁荣稳定
- 第六十章 推动两岸关系和平发展、推进祖国统一大业

### 第十八篇：加强规划实施保障
- 第六十一章 坚持和加强党中央集中统一领导
- 第六十二章 健全规划实施全周期推进机制

## 五、七大战略主线

1. **新质生产力主线**：科技创新引领产业创新，AI赋能千行百业
2. **扩大内需主线**：消费投资良性循环，全国统一大市场
3. **制度型开放主线**：对标高标准经贸规则
4. **城乡区域协调主线**：新型城镇化+乡村振兴双轮驱动
5. **民生保障主线**：从扎实稳固迈向品质提高，应对老龄化
6. **绿色转型主线**：碳达峰+美丽中国，能源体系根本性变革
7. **安全韧性主线**：粮食/能源/产业链/金融/数据全领域安全

## 六、与十四五关键变化对比

| 方面 | 十四五(2021-2025) | 十五五(2026-2030) |
|------|-------------------|-------------------|
| 阶段定位 | 开启全面建设社会主义现代化国家新征程 | 夯实基础、全面发力，承前启后 |
| GDP目标 | 未设具体增速，保持合理区间 | 保持合理区间 |
| 科技 | 研发强度目标>2.4% | 研发投入年均增长7%+ |
| 数字经济 | 数字经济核心产业增加值占GDP 10% | 全面实施"人工智能+"行动 |
| 人口 | 应对人口老龄化上升为国家战略 | 建设生育友好型社会 |
| 就业 | 城镇调查失业率<5.5% | 城镇调查失业率<5.5%(延续) |
| 碳达峰 | 制定2030年前碳达峰行动方案 | 碳达峰目标如期实现 |
| 城镇化 | 常住人口城镇化率65% | 全面设立公共户口 |
| 安全 | 统筹发展和安全 | 安全体系化构建(粮食/能源/产业链/数据) |
| 统一大市场 | 提出概念 | 纵深推进，制定条例 |
