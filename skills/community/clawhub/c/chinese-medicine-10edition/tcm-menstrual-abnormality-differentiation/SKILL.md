# TCM Menstrual Abnormality Differentiation

## When to Use
- 患者为育龄期或围绝经期女性
- 已排除妊娠状态
- 症状持续观察至少3个月经周期
- 不适用于青春期初潮后1年内或绝经后出血

## When to Use
当用户主诉“月经不调”“月经提前/推迟”“经量过多/过少”“闭经”或“痛经”等，且明确非妊娠状态时，启动本辨证流程。

## Execution Steps

### 1. 判断月经周期异常类型
- **月经先期**：经期提前7天以上，连续≥3个周期 → 考虑血热妄行或气虚不能摄血
- **月经后期**：经期延后7天以上，连续≥3个周期 → 考虑血虚、寒凝气滞或经血不利
- **月经先后不定期**：经期或提前或延后均超7天，连续≥3个周期 → 考虑肝郁气滞

### 2. 判断经量异常
- **经量过多**：
  - 色红质稠 → 实证、热证
  - 色淡质稀 → 气虚证
- **经量过少**：
  - 色淡量少 → 精血亏虚证
  - 色紫暗有块 → 瘀血证

### 3. 判断闭经
- 非妊娠状态下停经≥3个月 → 考虑化源不足、血海空虚，或寒凝气滞血瘀

### 4. 判断痛经
- 行经期间或前后小腹阵痛，甚则痛引腰骶：
  - **实证**：寒凝、气滞血瘀
  - **虚证**：气血两虚、阳虚

### 5. 综合经色与经质辨证
- 色浅淡、质清稀 → 气血亏虚
- 色鲜红、质浓稠 → 血热
- 色紫黑、有瘀块 → 血瘀

## Output
输出具体中医证型诊断（如“肝郁气滞证”“气血两虚证”“血热妄行证”等），用于指导后续治法选择。

> 注意：本技能仅用于辨证参考，不替代专业医师诊疗。
