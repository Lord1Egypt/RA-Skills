# 大数投资选股技能

## 核心框架

大数投资基于**概率统计**原理，以**国民经济整体**为投资对象，通过**分层抽样**构建拟合全市场的投资组合。

**三大原则**：组合投资 + 低估值选股 + 长期持有

## 选股标准（优先级顺序）

1. **市净率 PB ≤ 1.5**（优先破净股 PB < 1）
2. **市盈率 PE ≤ 20**（剔除亏损股，特殊情况除外）
3. **行业龙头**（市值排名行业前3优先）
4. **ROE ≥ 5%**（盈利能力合格线）
5. **无财务异常**（无ST、无重大违规）

## 实时选股流程

> 每次必须实时搜索最新数据，不得使用过时信息！

### Step 1：搜索市场整体估值

```
搜索：A股 低估值 破净股 行业龙头 {当前月份}
搜索：上证指数 深证成指 创业板 {当前日期}
```

### Step 2：按行业筛选低估值标的

```
搜索：{行业名} 低估值 破净股 {当前月份}
搜索：{行业名} 市盈率 市净率 龙头 {当前月份}
```

### Step 3：核实关键指标

通过**东方财富**或**新浪财经**核实个股：
- PE（TTM）、PB、ROE
- 近3年分红情况
- 有无ST或退市风险

### Step 4：输出选股结果

每只股票输出：行业 | 股票名称 | 代码 | PE | PB | 选股理由
