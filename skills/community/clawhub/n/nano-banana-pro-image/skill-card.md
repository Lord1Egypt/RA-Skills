## Description: <br>
Generate and edit images using Google's Nano Banana Pro through Atlas Cloud, including professional visual assets with precise text, reference-image editing, and high-resolution outputs. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users, developers, and creative teams use this skill to prepare prompts, API requests, and workflow guidance for generating or editing images through Atlas Cloud. It is suited to marketing assets, infographics, menus, diagrams, concept art, product visuals, and other image tasks where text rendering and composition quality matter. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Prompts, image URLs, and uploaded images are sent to Atlas Cloud for generation or editing. <br>
Mitigation: Avoid confidential images or sensitive prompt content unless Atlas Cloud's data handling terms are acceptable for the intended use. <br>
Risk: Image generation can incur API usage or billing costs through the configured Atlas Cloud account. <br>
Mitigation: Use a dedicated API key where possible and monitor usage or billing for the account tied to ATLASCLOUD_API_KEY. <br>
