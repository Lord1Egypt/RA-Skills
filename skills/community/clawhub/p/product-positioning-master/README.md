# 产品定位大师 / Product Positioning Master

## 概述

**产品定位大师** 是一款专业的 AI 技能，能够对任何产品/服务/品牌进行系统化的产品定位分析。它结合了三大经典商业分析框架，输出一份结构完整的《产品定位分析报告》。

## 核心框架

1. **ST-SWOT 交叉复合论证** — 市场细分(ST)与内外部环境(SWOT)的交叉比对分析
2. **定位理论** — 基于用户心智、产品优势、竞争差异的定位推导
3. **4P 营销组合** — 产品、价格、渠道、推广完整策略（含品牌宣言、推广语、销售话术）

## 适用场景

- 新产品上市前的定位策略制定
- 现有产品的重新定位与品牌升级
- 竞品分析与差异化策略研究
- 营销方案制定前的战略诊断
- 创业项目的商业模式梳理

## 使用方式

在 WorkBuddy 中通过以下关键词即可触发：
- `产品定位` / `定位分析` / `市场定位`
- `SWOT` / `STP` / `4P`
- `竞品分析` / `品牌定位` / `品牌宣言`
- `市场分析` / `细分市场` / `目标客户`
