# 儿童体格生长数据统计方法详解

## 1. 均值离差法
- **适用条件**：数据服从或近似正态分布
- **公式**：范围 = X̄ ± n·S
- **覆盖比例**：
  - ±1S → 68.3%
  - ±2S → 95.4%
  - ±3S → 99.7%
- **临床常用**：正常范围定义为 X̄ ± 2S

## 2. 百分位数法
- **定义**：Pₖ 表示有 k% 的参照人群测量值低于该数值
- **正常范围**：P₃ 至 P₉₇（覆盖94%人群）
- **优势**：
  - 不依赖分布假设
  - 能准确反映偏态分布
  - 在正态分布下与均值离差法结果接近

## 3. Z评分（标准差离差法）
- **公式**：Z = (X - X̄) / S
  - X：实测值
  - X̄：同年龄同性别参照均值
  - S：参照标准差
- **特点**：
  - 可用于不同指标间比较（如身高Z与体重Z）
  - 支持纵向追踪（同一儿童不同时间点的Z变化）
  - 精确量化偏离程度
- **临界值**：
  - Z < -2：生长迟缓/低体重
  - Z > +2：超重/过高

## 4. 中位数法
- **适用场景**：数据含异常值、轻度偏态或样本量小
- **原理**：中位数（P₅₀）不受极端值影响，比均值更稳健
- **注意**：中位数本身不提供变异程度信息，需配合四分位间距（IQR）使用

## 5. 重要原则与禁忌
- **禁止行为**：
  - 将均值、P₅₀ 或中位数作为个体“达标线”
  - 忽略年龄与性别分组直接比较
  - 在偏态分布下强行使用均值离差法
- **推荐实践**：
  - 结合多种方法交叉验证
  - 使用权威参照标准（如WHO 2006生长标准）
  - 向家长解释时强调“正常范围”而非“理想值”

## 6. 变量说明
| 符号 | 类型 | 描述 |
|------|------|------|
| X | measurement | 儿童体格指标实测值 |
| X̄ | reference_mean | 参照人群同年龄同性别均值 |
| S | standard_deviation | 参照人群标准差 |
| P₃/P₉₇ | percentile | 第3和第97百分位，常用作正常范围边界 |
