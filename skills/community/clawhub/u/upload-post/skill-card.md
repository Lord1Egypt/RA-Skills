## Description: <br>
Upload videos, photos, text, and documents to TikTok, Instagram, YouTube, LinkedIn, Facebook, X (Twitter), Threads, Pinterest, Reddit, and Bluesky through the Upload-Post API with scheduling, analytics, FFmpeg processing, and upload history support. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External developers, social media operators, and automation agents use this skill to prepare Upload-Post API requests for publishing, scheduling, editing, canceling, and reviewing posts across connected social accounts. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can publish, schedule, edit, or cancel posts across connected social accounts without built-in confirmation safeguards. <br>
Mitigation: Confirm the exact content, destination platforms, profile, visibility, and scheduled time before every publish or edit, and regularly review scheduled jobs. <br>
Risk: Connected account access and uploaded content depend on trust in the Upload-Post service and API key handling. <br>
Mitigation: Use a dedicated limited profile or API key where possible, avoid confidential documents unless they are intended for posting, and rotate the API key if it is exposed. <br>
