# 复习计划制定指南

## 科学备考的5个原则

1. **先摸底后规划**：先做一套真题了解水平和薄弱环节
2. **模块化推进**：逐个模块攻克，不要眉毛胡子一把抓
3. **刷题+复盘双循环**：做题和总结时间比例约3:1
4. **循序渐进**：基础→强化→冲刺，每阶段目标明确
5. **定期模拟**：每周至少1次全真模拟

## 三段式备考计划模板

### 阶段一：基础夯实（4-8周）

**目标**：系统学习各模块方法论，建立知识框架

| 周次 | 行测 | 申论 | 其他 |
|------|------|------|------|
| 第1周 | 摸底测试+分析 | 摸底测试+了解题型 | 制定个人计划 |
| 第2-3周 | 言语理解+判断推理方法论 | 归纳概括+综合分析 | 每日时政10分钟 |
| 第4-5周 | 资料分析+数量关系方法论 | 提出对策+贯彻执行 | 人民日报评论阅读 |
| 第6-7周 | 常识判断+综合练习 | 大作文写作法 | 开始每周1次模考 |
| 第8周 | 薄弱模块回炉 | 大作文强化 | 阶段总结 |

**每日学习建议**：
- 行测：2-3小时（1个模块精练）
- 申论：1-1.5小时（1道小题或半篇作文）
- 时政：碎片时间（学习强国/人民日报APP）

### 阶段二：强化提升（4-6周）

**目标**：大量刷题，提升速度和正确率

| 周次 | 重点 | 目标 |
|------|------|------|
| 第9-10周 | 行测模块化刷题 | 每个模块正确率提高5-10% |
| 第11-12周 | 申论各题型专项 | 小题达到80%得分率 |
| 第13-14周 | 套题训练+限时提速 | 行测控制在115分钟内 |

**每日学习建议**：
- 上午：行测套题1套+逐题复盘（3小时）
- 下午：申论1道小题+大作文框架练习（2小时）
- 晚上：错题整理+薄弱点补习（1小时）

### 阶段三：冲刺模拟（2-4周）

**目标**：全真模拟，查漏补缺，保持状态

| 时间 | 内容 |
|------|------|
| 每日 | 上午行测模拟+复盘，下午申论模拟+复盘 |
| 每周 | 2-3次完整双科模拟（严格按考试时间） |
| 考前1周 | 回归基础，不做新题，看错题本和笔记 |

## 不同起点考生方案

### 零基础（备考时间6个月）
```
前2个月：看视频课程，跟学方法论
中2个月：分模块刷题，开始申论写作
后2个月：套题冲刺，全真模拟
```

### 有基础（备考时间3个月）
```
第1个月：弱项补齐，方法论回顾
第2个月：大量刷题，限时训练
第3个月：全真模拟，保持手感
```

### 在职备考（备考时间4-6个月）
```
工作日：每天保证2-3小时（晚上+通勤碎片时间刷题）
周末：全天学习（模拟+复盘）
关键：碎片时间利用（时政、常识、逻辑填空）
```

## 10周全职备考精确到天模板

| 阶段 | 周一-周五 | 周六 | 周日 |
|------|-----------|------|------|
| 基础(1-3周) | 上午行测方法论/下午申论/晚上时政 | 行测模块练习 | 全真模考+复盘 |
| 强化(4-7周) | 上午行测套题/下午申论专项/晚上错题 | 弱项专项强化 | 全真模考+复盘 |
| 冲刺(8-10周) | 上午全真行测/下午申论作文/晚上回顾 | 全真双科模拟 | 深度复盘+错题 |
