# 小餐饮菜单设计参考 🍽️

## 一、菜单结构设计

### 黄金菜单公式

```
完美菜单 = 引流产品（10-15%）+ 利润产品（60-70%）+ 形象产品（15-25%）

引流产品：
- 特点：大众口味、成本低、视觉好
- 数量：占总SKU的10-15%
- 定价：低于竞品15-25%
- 作用：吸引流量，提升排名

利润产品：
- 特点：独特口味、成本可控、复购率高
- 数量：占总SKU的60-70%
- 定价：成本×3.5-4.5
- 作用：贡献核心利润

形象产品：
- 特点：看起来很贵/很特别/分量足
- 数量：占总SKU的15-25%
- 定价：成本×5-8
- 作用：衬托其他产品性价比高
```

### 各品类菜单SKU数量参考

| 品类 | 推荐SKU数 | 说明 |
|------|---------|------|
| 奶茶饮品 | 15-25个 | 核心款+季节限定，不要超过30 |
| 炸鸡汉堡 | 20-30个 | 主食+小食+饮品，套餐要有 |
| 麻辣烫 | 30-50个食材 | 素菜/荤菜/丸子类分好区 |
| 黄焖鸡米饭 | 10-15个 | 口味×规格+配菜，简单为主 |
| 卤味熟食 | 20-35个 | 荤素搭配，凉菜热菜分开 |
| 早餐包子 | 15-25个 | 包子+豆浆+粥+小菜 |
| 煎饼果子 | 10-18个 | 基础款+升级款+套餐 |

## 二、菜单定价公式

### 外卖定价法

```
外卖价 = (房租分摊 + 人工分摊 + 平台扣点 + 配送成本) ÷ 预期单量 + 食材成本 + 预期利润

举例（炸鸡店）：
- 月房租：6000元 → 每天200元
- 月人工：12000元 → 每天400元（夫妻店算0）
- 平台扣点：23%（每单收）
- 预期每天：50单
- 食材成本：8元/份

日均固定成本分摊 = (200+0) ÷ 50 = 4元/单
平台扣点 = 外卖价 × 23%
设外卖价为X：
X = 4 + X×23% + 8 + 2（利润）
77%X = 14
X = 18元

所以这个店的外卖定价应该在18-25元区间（根据定位调整）
```

### 堂食定价法

```
堂食价 = 食材成本 × 3.5-4.5

举例：
- 炸鸡腿饭：食材成本6元 → 定价21-27元
- 麻辣烫（素菜1元/串）：食材成本12元 → 定价42-54元

注意：堂食要考虑翻台率！
翻台率低的位置，定价要更高来弥补
```

### 套餐定价法

```
套餐价值 > 各单品之和，但实际定价 < 各单品之和

举例（炸鸡店）：
单品：炸鸡腿（18元）+ 可乐（6元）+ 薯条（10元）= 34元
套餐定价：26.8元

设计逻辑：
- 用套餐引导顾客多点
- 套餐毛利率仍要 ≥ 60%
- 套餐要设置"占便宜"的感觉
```

## 四、爆款打造方法

### 爆款选择标准

```
不是所有产品都适合做爆款！选爆款要看：

1. 受众广 → 80%的人都会喜欢
2. 出品稳 → 每次做味道一样
3. 成本可控 → 毛利 ≥ 60%
4. 视觉好 → 拍照好看，颜色有食欲
5. 出餐快 → 5分钟内能出
6. 适合外送 → 凉了也好吃

爆款候选：
- 奶茶：珍珠奶茶、杨枝甘露（全国通用）
- 炸鸡：香辣鸡腿、脆皮炸鸡
- 麻辣烫：番茄锅底+毛肚虾滑
- 黄焖鸡：招牌微辣款
```

### 爆款打造步骤

```
Step 1：选品（1-2周）
- 测试3-5个候选款
- 收集顾客反馈
- 看复购率哪个最高

Step 2：强化（2-4周）
- 确定1个爆款
- 优化出品流程，确保稳定
- 研发配套产品（套餐/小食）

Step 3：推广（持续）
- 爆款放在菜单第一位/头图
- 设置爆款专属优惠
- 朋友圈/社群持续种草
- 引导好评围绕爆款

Step 4：迭代
- 根据季节调整口味
- 每年1-2次小幅升级
- 保持新鲜感但不变核心
```

### 爆款定价策略

```
爆款不是越便宜越好！

错误做法：爆款赔钱卖 → 吸引来的人都是来占便宜的
正确做法：爆款高性价比 → 让顾客觉得值，但你不亏

爆款定价参考：
- 比竞品低5-10%：有竞争力
- 比自己其他产品贵：体现价值
- 设置套餐搭配：爆款+利润产品组合

举例（奶茶店）：
- 普通款：16元，毛利60%
- 爆款波波奶茶：14元，毛利55%
  → 定价稍低但不赔钱
  → 引导加3元升级大杯
  → 引导加2元加布丁（利润产品）
  → 实际客单价可能更高
```

## 五、菜单视觉设计

### 线上菜单（外卖平台）

```
头图设计原则：
- 菜品实拍，禁止用logo当头图
- 背景干净，突出菜品
- 文字少而精
- 食欲色：红/橙/黄 > 蓝/绿

菜单分类：
- 分类名称清晰
- 分类顺序：招牌→经典→新品→小食→饮品
- 爆款打"🔥爆"标签
- 新品打"🆕"标签

产品图规范：
- 同一光源（不要有的亮有的暗）
- 同一角度（俯拍/45度角）
- 背景统一（白底/木桌/大理石）
- 产品完整（不要切一半）
```

### 线下菜单（门贴/桌贴）

```
菜单材质：
- 小店：打印覆膜（便宜耐用）
- 连锁：铜版纸覆膜骑马钉
- 高端：特种纸+覆膜+烫金

菜单尺寸参考：
- 奶茶店：A4纸大小，2-4页
- 快餐店：A3纸大小，折页2-4面
- 炸鸡店：A4纸大小，正反面

菜单内容排序：
正面：招牌/爆款 → 主力产品 → 套餐
背面：小食/饮品 → 联系方式 → 二维码

价格设计：
- 价格写在菜名后面，不要另起一行
- 价格字体不要太大（比菜名小一号）
- 套餐价格要突出："套餐立省XX元"
```
