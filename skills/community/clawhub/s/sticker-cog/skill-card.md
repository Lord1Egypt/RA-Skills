## Description: <br>
Sticker Cog helps agents generate CellCog-powered sticker packs and emoji sets with character-consistent expressions, transparent backgrounds, batch generation, and platform-ready packaging. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, creators, and workspace administrators use this skill to prompt CellCog for custom sticker packs, mascot or pet stickers, platform-specific emoji sets, and brand reaction assets. The skill provides prompt patterns, platform constraints, setup guidance, and example API usage for CellCog-backed generation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may send prompts, reference photos, mascot images, pet images, or brand assets to CellCog. <br>
Mitigation: Avoid uploading confidential, regulated, or highly personal assets unless CellCog's privacy and retention terms are acceptable. <br>
Risk: The skill requires the CELLCOG_API_KEY credential. <br>
Mitigation: Store the API key securely, avoid exposing it in prompts or logs, and rotate it if disclosure is suspected. <br>
