# 论文 MATLAB 复现 Skill 流程方案

## 一、优化后的总体流程

这个 skill 将论文复现拆成 8 个阶段，核心原则是“先证据、后代码、再对比修正”。

1. 读取论文文件地址，建立论文信息索引。
2. 从算法图、算法表、流程图中逐一抽取算法伪代码。
3. 对伪代码涉及的参数做分层查找：仿真结果部分优先，其次图表描述，最后才基于论文内容推断。
4. 对算法图表中引用其他文献的操作，用 Google Scholar 优先检索对应论文，读取所需操作并补全伪代码。
5. 根据论文系统模型生成 MATLAB 主仿真框架。
6. 根据每个伪代码生成独立 `.m` 算法函数，并由 `main*.m` 调用。
7. 运行所有主函数，和论文结果图表做相似度对比。
8. 如果结果差异明显，最多自动诊断并优化 MATLAB 代码 3 次，最后输出代码、结果图和相似度报告。

## 二、详细执行步骤

### 1. 用户输入检查

必须要求用户提供论文文件地址，可接受：

- 本地 PDF 路径
- 在线 PDF URL
- arXiv 链接
- DOI
- 已上传文件

如果缺少论文文件地址，先询问用户。可选询问输出目录、是否只复现指定图表、是否允许近似复现。

### 2. 论文读取与信息抽取

读取论文后抽取：

- 标题、作者、年份、期刊/会议
- 摘要和关键词
- 系统模型
- 算法图、算法表、流程图
- 仿真设置
- 仿真结果图表
- 参考文献列表

建立映射：

```text
算法图表 -> 论文位置 -> 输入输出变量 -> 参数 -> 对应结果图 -> MATLAB 文件
```

### 3. 算法伪代码生成

对论文中每一个算法图表生成一个伪代码块。判断范围包括：

- 标准 Algorithm 环境
- 流程图
- 描述算法步骤的表格
- 由图注和正文共同描述的过程

如果有几个算法图表，就生成几个伪代码。每个伪代码必须包含：

- 来源位置
- 目的
- 输入
- 输出
- 参数
- 循环结构
- 约束条件
- 终止条件
- 对应 MATLAB 函数文件名

### 4. 引用文献操作补全

如果算法步骤中出现：

- `using [n]`
- `following [n]`
- `as described in [n]`
- `based on Method X`
- 操作被引用但当前论文没有定义

则优先用 Google Scholar 搜索对应文献。读取后只提取复现该操作所需内容，不需要完整复现被引用论文。

如果 Google Scholar 不可访问，使用 arXiv、出版社页面、Semantic Scholar、Crossref 或作者公开 PDF 作为备选，并在报告中说明。

### 5. 参数设置提取

参数查找顺序固定为：

1. 仿真结果或实验设置部分
2. 图注、表注、算法说明、附录
3. 引用文献中的操作定义
4. 根据论文内容、量纲、领域常见设置推断

每个参数都要记录来源：

- `paper-explicit`: 论文仿真部分明确给出
- `caption-or-table`: 图表描述给出
- `cited-source`: 引用文献给出
- `inferred`: 自行推断

凡是 `inferred` 参数，必须明确告知用户。

### 6. MATLAB 文件生成

默认文件结构：

```text
main.m
main_<环境差异后缀>.m
alg1_<算法名>.m
alg2_<算法名>.m
helpers/
results/
paper_artifacts/
reproduction_log.md
```

生成规则：

- 系统模型、仿真循环、对比指标、绘图和结果保存放在 `main.m`。
- 如果存在多种仿真环境，生成多个 `main` 文件。
- 后缀按环境差异命名，例如 `main_snr_sweep.m`、`main_rayleigh_channel.m`、`main_user_density.m`。
- 一个算法伪代码对应一个 `.m` 函数文件。
- 参数统一放入 `cfg` 结构体，避免散落的 magic number。

### 7. 运行与相似度对比

运行所有 `main*.m`。对比对象包括：

- 论文中的结果曲线
- 论文表格数值
- 文本中描述的性能提升比例
- 算法收敛趋势

推荐相似度指标：

- NRMSE
- 相对误差
- Pearson 相关系数
- 关键点误差
- 趋势一致性

最终给出每个图表和整体复现相似度。

### 8. 自动诊断与最多 3 次优化

如果仿真结果和论文不一致，按顺序检查：

1. 参数数值和单位
2. 系统模型公式
3. 引用文献操作
4. 伪代码到 MATLAB 的翻译
5. 随机种子和 Monte Carlo 次数
6. 指标定义和绘图坐标轴

最多优化 3 次。每次优化都记录：

- 差异现象
- 诊断原因
- 修改内容
- 修改后的相似度

### 9. 最终输出

最终交付：

- MATLAB 仿真代码
- 每个算法对应的 `.m` 函数
- 一个或多个 `main*.m`
- 仿真结果图
- 参数表
- 伪代码文件
- 引用文献操作说明
- 相似度报告
- 仍无法复现或依赖推断的部分说明
