# 📈 Fund Report - 基金持仓播报工具

实时抓取场外基金净值和估算涨跌，支持多基金同时查询。

## 功能

- 实时基金净值估算
- 支持多个基金代码同时查询
- 包含涨跌幅度和估算收益
- 支持A股/港股/纳斯达克基金

## 使用方法

```bash
python3 scripts/fund_report.py
```

## 基金代码示例

- 161725 - 招商中证白酒
- 008971 - 大成纳斯达克100
- 011452 - 华泰柏瑞质量成长
