# UI 设计系统与美学规范

> 基于 12 大设计法则、631 配色原则、8 点网格系统的完整 UI 设计规范。

## 12 大核心 UI 设计法则

| # | 法则 | 原理 | 实战应用 |
|---|------|------|----------|
| 01 | **连续性** | 视觉追随连续排列轨迹 | 导航横向排列、卡片单侧截断 |
| 02 | **相似性** | 特征相似的元素归为一类 | 统一功能图标尺寸造型 |
| 03 | **封闭性** | 人脑会补全残缺图形 | 用部分元素暗示完整结构 |
| 04 | **接近法则** | 邻近物体视为同一组 | 用间距/线条分组内容 |
| 05 | **图底关系** | 前景背景明确区分 | 卡片阴影、毛玻璃效果 |
| 06 | **共同命运** | 同方向运动元素视为一组 | 滚动、展开收起动效 |
| 07 | **对称法则** | 人脑偏好对称 | 列表/表单布局对称 |
| 08 | **中心焦点** | 视线优先落中心 | 核心功能放视觉中心 |
| 09 | **黄金比例** | 1:1.618 自然美感 | 图片裁剪、卡片比例 |
| 10 | **斐波那契** | 自然递增尺寸体系 | 字号 8/13/21/34/55 |
| 11 | **三分法则** | 九等分画面 | 关键按钮放置交点 |
| 12 | **Z型/F型** | 自然阅读路径 | 左上放最重要内容 |

## 色彩系统

### 631 配色原则

- **60% 主色** — 背景色、大面积色块
- **30% 辅助色** — 导航栏、卡片、分割线
- **10% 强调色** — 按钮、链接、关键操作

### 色彩心理学选择

| 颜色 | 心理效应 | 适用场景 |
|------|----------|----------|
| 🔵 蓝 | 信任、安全、专业 | 金融、社交 |
| 🔴 红 | 紧迫、热情、警示 | 促销、错误提示 |
| 🟢 绿 | 成功、成长、环保 | 确认消息、健康 |
| 🟠 橙 | 活力、友好、创意 | 娱乐、教育 |
| 🟣 紫 | 奢华、创意、神秘 | 美妆、高端品牌 |
| ⚫ 黑 | 高端、力量、优雅 | 奢侈品、夜间模式 |

### Asset Catalog 颜色实践

```swift
// 错误 ❌ 硬编码颜色
Color(red: 0.2, green: 0.4, blue: 0.8)

// 正确 ✅ Asset Catalog 命名颜色 (支持 Light/Dark 双变体)
Color("Primary")
Color("Background")
Color("TextPrimary")
Color("Surface")
```

## 排版系统

### 字号规范

| 层级 | 尺寸 | 字体 |
|------|------|------|
| 大标题 | 34px | SF Pro Display |
| 标题1 | 28px | SF Pro Display |
| 标题2 | 22px | SF Pro Display |
| 副标题 | 17px | SF Pro Text |
| 正文 | 15px | SF Pro Text |
| 辅助文字 | 11px | SF Pro Text |

字号比例：**3:2:1**（标题:正文:辅助）

### 字体选择

| 场景 | 字体 |
|------|------|
| 标题/显示 | SF Pro Display |
| 正文 | SF Pro Text |
| 等宽 | SF Mono |
| 中文简/繁 | PingFang SC / PingFang TC |
| 日文 | Hiragino Sans |
| 韩文 | Apple SD Gothic Neo |

**禁止超过 2 种字体家族。**

## 间距系统 (8点网格)

```swift
// 间距规范
let spacing: [CGFloat] = [4, 8, 12, 16, 24, 32, 48]
// 微间距 4/8 → 内间距 12/16 → 组件间距 16/24 → 大间距 32/48
```

### 圆角规范

| 类型 | 圆角 | 应用 |
|------|------|------|
| 小型 | 6px | 按钮、标签、输入框 |
| 中型 | 12px | 卡片、弹窗、sheet |
| 大型 | 20px | 模态视图、全屏组件 |

## 按钮优先级体系

- **主按钮 (Primary)** — 1个页面最多1个，强调色填充，带阴影
- **次按钮 (Secondary)** — 线框按钮，透明背景+边框色
- **三级按钮 (Tertiary)** — 纯文字按钮，无边框

## 反AI流行模式检查清单

| ❌ 反模式 | ✅ 替代方案 |
|-----------|-------------|
| 紫蓝渐变背景 | 有目的性的色彩方案 |
| Emoji 作为功能图标 | SF Symbols 或 SVG 图标 |
| 左侧带彩色边框的圆角卡片 | 自定义卡片设计 |
| 居中对齐文本+渐变背景的"英雄区" | 有层次的内容布局 |
| 使用 Inter/Roboto 作为显示字体 | SF Pro 或系统字体 |
| AI 生成的 SVG 剪贴画 | 干净的灰色占位符 |

## 交互反馈规范

| 交互类型 | 反馈方式 | 响应时间 |
|----------|----------|----------|
| 按钮点击 | 缩放动画 + 颜色变化 | < 100ms |
| 加载中 | 骨架屏 + 进度指示器 | < 400ms |

| 操作失败 | 红色错误提示 + 震动反馈 | 即时 |
| 拖拽 | 实时跟随 + 弹性效果 | 实时 |
