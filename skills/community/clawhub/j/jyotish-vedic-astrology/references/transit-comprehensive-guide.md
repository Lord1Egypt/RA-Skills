## 二、Double Transit 理论（双过境理论）

### 2.1 核心原理

**重大人生事件需要土星+木星的双重过境激活同一宫位或其三方宫**。

- **土星** = 事件的确切性（业力承诺，"这件事会发生"）
- **木星** = 事件的时机性（恩典与扩张，"这件事在此时发生"）

### 2.2 双过境组合表

| 事件类型 | 土星过境宫位 | 木星过境宫位 | 触发机制 |
|---------|-------------|-------------|---------|
| **婚姻** | 7宫 或 7宫主星 | 7宫 或 7宫三方（5/9） | 双方过7宫最强 |
| **生育** | 5宫 或 5宫主星 | 5宫 或 5宫三方（1/9） | 5宫直接激活 |
| **事业突破** | 10宫 或 10宫主星 | 10宫 或 10宫三方（2/6） | 10宫直接激活 |
| **购房/搬迁** | 4宫 或 4宫主星 | 4宫 或 4宫三方（8/12） | 4宫直接激活 |
| **出国/远行** | 9宫/12宫 | 9宫/12宫 | 双方过12宫最强 |
| **重病/手术** | 6宫/8宫 | 6宫/8宫（不利相位） | 土星单方过6/8也够 |
| **死亡** | 8宫/2宫（Mrityu Bhaga） | 无需木星 | 土星+Rahu即可 |

### 2.3 实战判断步骤

```
Step 1: 确认事件对应的宫位（如婚姻=7宫）
Step 2: 检查土星当前过境位置（确切性）
Step 3: 检查木星当前过境位置（时机性）
Step 4: 两者是否都激活了目标宫位或其三方？
Step 5: 结合Dasha确认——是否有相关Dasha运行？
Step 6: 三层吻合 → 高概率事件
```

### 2.4 实战案例

**案例：婚姻时机预测**
- 命盘：7宫主金星在8宫，7宫内有土星
- 当前Dasha：Rahu-Venus（激活7宫主+8宫）
- 土星过境：水瓶座（11宫）→ 三方7宫 ✅
- 木星过境：白羊座（9宫）→ 三方5宫 → 间接激活7宫 ✅
- **结论**：双过境+Dasha三重激活7宫，婚姻窗口开启

## 三、Ashtakavarga 引导的过境评分

### 3.1 BAV 引导的过境判断

**原理**：某行星过境某宫时，如果该宫在该行星的 BAV 中得分高，则过境效果好；反之则差。

#### 土星 BAV 过境评分表（示例）

| BAV分数 | 过境该宫效果 | 判断 |
|---------|------------|------|
| 4-5分 | 土星带来正面结果（结构化、纪律、稳定） | ✅ 有利 |
| 3分 | 中性 | ⚠️ 平稳 |
| 0-2分 | 土星带来困难（延迟、阻碍、压力） | ❌ 不利 |

#### 木星 BAV 过境评分表（示例）

| BAV分数 | 过境该宫效果 | 判断 |
|---------|------------|------|
| 5-6分 | 木星带来显著好运（扩张、机遇、智慧） | ✅ 大吉 |
| 3-4分 | 中等偏正 | ✅ 有利 |
| 0-2分 | 木星的祝福被减弱 | ⚠️ 一般 |

### 3.2 SAV 引导的宫位强弱判断

**关键阈值**：

| SAV分数 | 过境该宫的效果 |
|---------|--------------|
| >30分 | 该宫领域强大，任何行星过境都偏向正面 |
| 25-30分 | 中等偏强，吉星过境好，凶星过境需注意 |
| 20-25分 | 中等，过境效果取决于具体行星 |
| <20分 | 该宫领域薄弱，凶星过境尤为不利 |

### 3.3 综合过境评分公式

```
过境评分 = 行星吉凶性 + BAV得分 + SAV宫位得分 + Dasha配合度
```

| 因素 | 吉星加分 | 凶星加分 |
|------|---------|---------|
| BAV ≥ 4 | +3 | +1 |
| BAV 3 | +1 | 0 |
| BAV ≤ 2 | -2 | -3 |
| SAV ≥ 28 | +2 | +1 |
| SAV 20-27 | +1 | 0 |
| SAV < 20 | -1 | -2 |
| Dasha配合 | +3 | +2 |
| Dasha不配合 | -2 | -3 |

**评分解读**：
- +5以上：强烈正面
- +2~+4：温和正面
- -1~+1：中性
- -4~-2：温和负面
- -5以下：强烈负面

## 六、过境事件触发时机精确法

### 6.1 Ashtakavarga 积分法

当外行星过境某宫时，用BAV积分确定最佳/最差时间段：

**步骤**：
1. 确定行星过境的星座
2. 将该星座划分为BAV积分高/低区域
3. 行星在高分区度数时 → 事件偏向正面
4. 行星在低分区度数时 → 事件偏向负面

### 6.2 月亮过境触发法

外行星的位置确定了**可能性**，月亮过境确定了**确切触发日**：

| 月亮过境位置 | 触发效果 |
|-------------|---------|
| 过境外行星所在星座 | 事件最可能发生 |
| 过境外行星三方星座 | 次要触发 |
| 与外行星形成精确相位 | 精确触发日 |

### 6.3 Retrograde 过境影响

| 行星状态 | 影响 | 事件节奏 |
|---------|------|---------|
| 顺行进入 | 事件正常展开 | A→B→C |
| 逆行进入 | 事件延迟、重复、回溯 | A→B→A→B→C |
| 逆行后恢复顺行 | 延迟的事件终于落地 | A→...→B→C（集中爆发） |

**关键规则**：
- 木星逆行 = 好事延迟但不会取消
- 土星逆行 = 压力加倍，尤其是回退到原位置时
- 金星逆行 = 旧关系可能重现
- 水星逆行 = 沟通/合同/交易问题
