# Resolution Decision Matrix

The strategy decision is economics + evidence + relationship + precedent. Run all four; the loudest factor wins only if the others don't veto.

## 1. The four factors

**Economics.** Refund cost = product cost + outbound shipping (sunk) + refund amount. Fight cost = your time (value it honestly: 30-60 min/case) + loss probability × refund anyway + platform-relations wear. Below ~$15-25 at stake, fighting is usually negative-EV even when you're right — unless precedent (factor 4) applies.

**Evidence.** Grade your case: Strong (dispositive dated artifacts: GPS delivery photo, return unboxing video, listing screenshot matching delivery), Medium (records that support but don't prove), Weak (your word vs. theirs). Fight only Strong, occasionally Medium with high stakes; never Weak — convert Weak cases into goodwill resolutions.

**Relationship.** Buyer LTV: repeat customer with clean history → bias toward generosity, the refund is a marketing spend with measurable ROI. New account, first order, instant dispute → no LTV bias. Compute simply: (orders/year × margin × expected years). A $40 goodwill refund to a $300/yr-margin customer is obvious.

**Precedent/pattern.** Serial refunders and organized abuse learn which sellers fold. If this buyer or this claim pattern has hit you before, the case value includes future losses prevented. Fighting a $20 case can be rational the third time. Likewise: a fix-the-root-cause refund (your listing really was ambiguous) prevents the next ten disputes better than ten fights.

## 2. Strategy selection table

| Situation | Strategy |
|---|---|
| Genuine defect/damage, confirmed | Replacement (margin allows + buyer wants the product) or full refund; apologize once, fix QC |
| Genuine but minor (cosmetic blemish, late delivery) | Partial refund (10-30%) with the reasoning stated |
| Ambiguous fault (listing unclear, expectations mismatch) | Partial or full refund + fix the listing today; do not fight your own ambiguity |
| Claim contradicted by Strong evidence, meaningful value | Fight with evidence; respectful buyer message |
| Claim contradicted, trivial value, first offense | Refund + log + report if reportable; not worth the hour |
| Repeat abuser / organized pattern | Fight + report + block-list, regardless of size |
| Review/refund extortion | Report the extortion FIRST; resolve the underlying claim on its merits separately |
| Chargeback with delivered goods | Always represent (respond) — non-response is auto-loss and raises your chargeback ratio anyway |

## 3. Partial refund calibration

- 10-15%: cosmetic issue, buyer keeps a fully functional product.
- 20-30%: real but tolerable defect, or significant delivery failure with eventual arrival.
- 40-50%: shared-fault returns where return shipping makes return irrational; or missing accessory the buyer can live without.
- Anchor: offer what makes the buyer whole for the actual gap, not what makes them stop typing. State the reason for the number — unexplained partials read as haggling and invite counter-haggling.

## 4. Goodwill spend policy (set once, apply consistently)

Define per-shop: auto-approve threshold (e.g., ≤$10 claims from clean-history buyers refunded without investigation — investigation costs more), goodwill budget per month, one-time-courtesy flag per buyer (the second "lost" package gets signature-required handling instead of a second credit), and the LTV multiplier above which you always choose generosity (e.g., at-stake < 10% of annual buyer margin).

## 5. When losing is the plan

Some cases you respond to knowing you'll lose (platform policy favors buyers on, say, sub-$20 INR). Respond anyway, minimally: it builds your dispute-rate record, feeds your log, and flags the buyer in platform systems. Budget 5 minutes, use the template, move on. The goal is portfolio-level: dispute COSTS down, not every case won.
