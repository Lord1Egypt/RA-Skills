# 抖音脚本公式模板库

## 公式一：痛点钩子公式

**适用场景**：功能性产品、解决日常困扰类

```
结构：痛点描述 → 引出产品 → 解决过程 → 效果呈现 → CTA
```

示例框架：
- 开头（3秒钩子）：「你有没有遇过...」「最近发现一个困扰我好久的问题终于解决了」
- 引入：「直到我用了这个...」
- 过程：展示使用步骤，突出1-2个核心卖点
- 效果：「现在...感觉真的不一样」
- 结尾CTA：「感兴趣的姐妹直接去链接看看」

---

## 公式二：对比反转公式

**适用场景**：有竞品差异化优势、升级换代产品

```
结构：旧方案痛点 → 意外转折 → 新产品登场 → 对比效果 → 种草
```

示例框架：
- 开头：「以前我一直觉得这类产品都差不多」
- 转折：「直到有人推荐我试了这个」
- 对比：「和我之前用的相比，明显感觉...」
- 结尾：「已经回购第三瓶了」

---

## 公式三：好奇心悬念公式

**适用场景**：新品、联名款、有故事感的产品

```
结构：神秘感开场 → 铺垫好奇 → 揭晓产品 → 卖点展示 → 互动引导
```

示例框架：
- 开头：「今天分享一个我偷偷用了三个月没敢说的东西」
- 铺垫：「因为我怕说出来就断货」
- 揭晓 + 卖点：产品名 + 1-3个核心特点
- 互动：「用过的扣1告诉我」

---

## 公式四：口播测评公式

**适用场景**：美食、美妆、日化等直接体验类产品

```
结构：开门见山推荐 → 外观/颜值 → 使用体验 → 关键卖点 → 适合人群 → 总结
```

示例框架：
- 开头：「今天来测评一下最近在用的...」
- 颜值描述：包装、外观、第一印象
- 体验：「打开之后...味道/质地/手感...」
- 卖点：2-3点具体特点
- 适合谁：明确受众描述
- 总结：一句话种草语

---

## 公式五：场景代入公式

**适用场景**：生活方式类、情感共鸣类产品

```
结构：生活场景开场 → 情绪共鸣 → 产品作为解决方案 → 生活改变 → 号召行动
```

示例框架：
- 场景：「每次...的时候总觉得少了点什么」
- 共鸣：「相信很多人都有这种感觉」
- 产品：「后来我发现了...」
- 改变：「现在每次...都会想到它」
- 行动：「强烈安利给同款生活方式的你」

---

## 公式六：数字冲击公式

**适用场景**：性价比突出、成分含量/使用次数等有数据支撑的产品

```
结构：数字开场（制造冲击）→ 解释数字 → 产品呈现 → 价值对比 → CTA
```

示例框架：
- 数字开场：「花了X元，用了X个月，我要说实话了」「同款产品，价格差了X倍」
- 解释：说明数字背后的故事
- 产品：引出推荐
- 价值：为什么值得/划算
- CTA：引导购买或收藏

|------|---------|
| 痛点型 | 「你是不是也有这个问题...」 |

| 场景型 | 「每次[具体场景]，我都会想到它」 |
