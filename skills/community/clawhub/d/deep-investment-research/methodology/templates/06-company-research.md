> 用法：填 {公司名} {代码/拟上市} {核心问题}，作为一次研究任务 prompt。
> 依赖：research-core.md（命门变量+反共识引擎+反证+质量门控）/ company-research.md（七维框架+5独门分析法）/ source-registry.md / content-reading-protocol.md

【公司研究：{公司名}（{代码}） — {核心问题}】

# 这类研究最容易犯的错（动笔前先警惕）
0. 闭门造车/凭记忆作答（不搜资料就下判断=井底之蛙）→ 必须先充分联网搜索+读一手财报/招股书全文
1. 形容词堆砌当研究（"龙头、稳健、护城河深、管理层优秀"=没研究）→ 每个判断必须有数字和趋势
2. 只读财报摘要/新闻 → 必须读财报/招股书/电话会正文相关章节（红线3）
3. 把"护城河叙事"当事实 → 必须用数据验证护城河真伪与方向
4. 描述数据而非审讯数据 → 必须用下面的独门分析法，从数字里逼出洞察

# 研究设计（动笔取数前先做）
先写三句，决定本次研究的重心：
1. 命门变量：决定这家公司价值/生死的 2-3 个变量是什么？（如某SaaS=NRR；某药企=核心管线；某制造=单位成本曲线+需求结构）70%火力打这几个。
2. 驱动类型：这门生意主要被什么决定（成本曲线/需求结构/技术迭代/牌照/网络效应）？
3. 据此裁剪：哪些维度重笔墨、哪些一句话带过。
然后过【七棱镜】判断相关性（相关取数、不相关说明为何忽略）：
①宏观 ②政策/监管 ③地缘 ④产业链上下游 ⑤资金面 ⑥新闻/事件 ⑦公司自身（主战场）

# 🔑 独门分析方法（公司研究的灵魂——不是描述数据，是审讯数据）
对取到的每组数字，主动用下面的方法挖洞察，而不是把数字罗列出来：

【方法A·商业模式解构三问】把"靠什么赚钱"拆成 价值创造→传递→捕获 三层并量化：
  - 创造：为客户解决什么、创造多少价值（客户因此省多少钱/多赚多少/避免多大损失）
  - 传递：渠道成本占比、谁掌握渠道议价权
  - 捕获：公司captures了创造价值的多少（定价 vs 客户感知价值的gap=提价空间；毛利率=捕获效率）
  → 创造强但捕获弱=有未释放的提价空间（潜在预期差）；捕获强但创造在被侵蚀=毛利将下行

【方法B·单位经济推演】把整体财务数字除到"单客户/单件"，看一单生意的本质：
  - 制造业：单位ASP / 单位成本 / 单位毛利的时间序列——成本曲线在降还是升？ASP在涨还是被价格战压？
  - 订阅/平台：LTV/CAC>3？NRR>100%？
  - 关键追问：规模扩大后单位经济是改善（真规模效应）还是恶化（伪规模）？用不同规模年份的单位数据对比验证

【方法C·护城河压力测试】对每种护城河不问"有没有"，问"什么情况会让它塌"：
  网络效应/规模成本/转换成本/无形资产/特许——逐项做"如果X发生，护城河是否瓦解"
  → 方向判定（最关键）：用 市占率趋势+毛利率趋势+定价权(提价是否掉客)+流失率 四个数据印证护城河在变宽还是变窄，不靠叙事

【方法D·财报异常信号探测】不孤立看单指标，看"本该同步却背离"的地方：
  应收增速vs收入增速 / 存货增速vs收入 / 经营现金流vs净利润 / 资本化研发占比 / 关联交易占比 / 毛利率vs行业趋势
  → 每一处背离都是要追问到底的线索；正常一句话带过，背离深挖

【方法E·管理层言行一致性审计】用历史数据，不用印象：
  拉出管理层过去2-3年的公开指引/承诺，逐条对照实际兑现——说到做到吗？资本配置track record如何？
  → 爱画饼但不兑现→其未来指引打折；历史兑现率高→其指引可信度高（直接影响你对成长预测的取信程度）

1. 看连续性：读 ${RESEARCH_OUTPUT_DIR:-~/research-output}/{行业}/ 最近2-3篇相关研究作背景（有则读，注意时效；无则跳过）

   ⚠️ 充分搜索原则：先广撒网搜够（多源、正反都搜），再聚焦读全文；绝不凭记忆/单一来源
   - 财报/招股书全文：edgar_filings(美)/hkex_announcements(港)/巨潮(A)，读最近1-2期+招股书的"业务/财务/风险"章节，不读摘要

# 推理脚手架（每个命门变量先写推理链，再下判断）

反共识三步：① 市场对这门生意的共识是什么 → ② 共识哪个假设最脆弱（用数据挑战）→ ③ 我跟随还是偏离

# 📐 材料分析示范（看原始数据如何变成洞察——这是本模板的核心）
下面展示"拿到一组财报数字后，用独门分析法逼出洞察"的完整推理，照此对你研究的公司做：

  原始数据（招股书）：四足单位成本 2.23万(2022)→1.21万(2025)；人形ASP 59.3万(2023)→16.6万(2025)；综合毛利 44%→60%；外购件占成本14-18%。
  ↓ 用方法B（单位经济推演）审讯：
  成本腰斩=全栈自研的规模效应是真的；但ASP暴跌72%=价格战在吞噬单价。两股力量反向拉扯。
  ↓ 必须拆开看（不能只看综合毛利上升就乐观）：
  综合毛利上升其实是"产品结构变化"撑的（高毛利人形占比从0→52%），不是单品在改善。
  ↓ 用方法C（护城河方向）印证：
  成本护城河在变宽（自研+规模），但定价权在变窄（ASP暴跌、被迫降价抢份额）——护城河方向是分化的。
  ↓ 洞察（别人看摘要看不到的）：
  "毛利率上升"这个表面利好，掩盖了"单品定价权正在流失"。一旦产品结构红利吃完（人形占比见顶），
  综合毛利会暴露出单品被价格战压制的真相。这是生意质量的隐藏裂缝。

  ❌ 平庸写法："公司成本控制优秀，毛利率持续提升，盈利能力强。"（描述数据，无洞察）
  ✅ 深刻写法：上面整条推理链（审讯数据，挖出裂缝）

# 输出结构（每段按标准写，禁止"龙头/稳健/优秀"等无证据形容词）
🏢 {公司}（{代码}）公司研究 | {date}
结论：[这是门什么生意 + 生意质量评级(高/中/差及理由) + 护城河方向(变宽/变窄/分化) + 成长可持续性(高/中/低) + 确信度]

═══ 命门变量及其当前状态 ═══
2-3个决定价值/生死的变量 + 每个的当前数据状态

═══ 商业模式：靠什么赚钱（方法A） ═══
价值 创造→传递→捕获 三层 + 收入结构占比 + 提价空间判断
  量化锚：收入结构带占比和增速；捕获效率=毛利率；不写"模式清晰"

═══ 单位经济：一单生意的本质（方法B） ═══
单位ASP/成本/毛利时间序列 或 LTV/CAC/NRR + 规模效应真伪判断

═══ 护城河：方向比有无更重要（方法C） ═══
逐种护城河压力测试 + 用4个数据(市占/毛利/定价权/流失率)印证方向
  ✅ 示范："转换成本护城河，流失率仅5%且毛利3年75%→82%提价未掉客=变宽"

═══ 财务质量与异常信号（方法D） ═══
盈利质量/现金流vs利润/ROIC + 探测到的背离信号(应收/存货/资本化/关联交易)
  正常项一句话带过，背离项深挖追问

═══ 竞争格局与成长性 ═══
份额趋势 + 同业横比(具体数字) + TAM倒推 + 成长驱动拆解(量/价/新品/份额) + 成长是内生还是烧钱买的

═══ 管理层与治理（方法E） ═══
历史指引兑现率 + 资本配置track record + 治理红旗 + 激励一致性

═══ 风险与脆弱性 ═══
最脆弱一环 + 单一依赖(客户/供应商/产品) + kill condition(看到什么说明生意逻辑变坏)

═══ 结论与衔接 ═══
生意质量评级理由 + 信息盲区 + 一句话"进入投资判断(stock-deep-dive)时该重点验证的预期差线索"
