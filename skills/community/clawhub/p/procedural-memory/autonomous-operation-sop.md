# 自主行动 SOP

当用户超过 30 分钟不响应时，小咪可以自主执行以下任务：

## 允许的自主任务

### 记忆维护
- 检查记忆文件完整性
- 清理过期的临时文件
- 更新 STARTUP.md
- 备份 GitHub 和 E盘

### 技能整理
- 检查新安装的 skill 是否有更新
- 检查 skill 配置完整性
- 归档无用的临时文件

### 自我改进
- 复盘最近的错误，记录到 corrections.md
- 检查 heartbeat-state.md 的状态
- 扫描 self-improving/ 目录，更新 index

### 信息收集
- 搜索新的 AI 技术动态
- 收集可能有用的 skill 信息

## 不允许自主行动的
- 修改核心人格设定
- 删除重要记忆
- 发布内容到外部
- 安装未经审查的 skill

## 执行后的报告

自主任务完成后，写入 `~/self-improving/autonomous-reports/`：
```markdown
# YYYY-MM-DD HH:MM 自主任务报告

## 任务类型
[记忆维护/技能整理/自我改进/信息收集]

## 执行内容
[具体做了什么]

## 结果
[成功/失败/部分成功]

## 学到的
[如果有新发现]
```

## 权限边界
- 无需批准：只读检查、cwd 内写操作、脚本实验
- 需要报告待审：修改全局记忆、安装软件、外部 API 调用
