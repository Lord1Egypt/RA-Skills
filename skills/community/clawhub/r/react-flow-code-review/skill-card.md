## Description: <br>
Reviews React Flow code for anti-patterns, performance issues, and best practices. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[anderskev](https://clawhub.ai/user/anderskev) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to review React Flow implementations for provider-boundary mistakes, unstable node or edge type definitions, rendering performance issues, missing citations, and checklist gaps. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The agent may inspect sensitive files if the review scope is broader than the React Flow code that needs review. <br>
Mitigation: Confirm the intended review scope before use and exclude sensitive or unrelated paths. <br>
