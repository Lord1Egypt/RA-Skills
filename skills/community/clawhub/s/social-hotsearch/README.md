# 社媒热搜助手 (social-hotsearch)

> 微博/小红书/抖音/知乎/百度热搜与社媒舆情分析。一句话查热搜、做话题分析、采样原帖。免费 200 次,无需注册。

## 你会得到什么

- **跨 5 大平台的实时数据能力**:微博 / 小红书 / 抖音 / 知乎 / 百度(其中小红书走话题分析,不走热搜榜)
- **对单个话题的多维度分析**:声量、情感指数、互动数、平台分布、关联话题
- **真实原帖样本**:看到舆论现场的具体讨论,不只是数字
- **支持视频/图文深度搜索**:可选搜索视频 ASR(语音)/ OCR(画面)/ 图片 OCR(适合短视频和小红书图文笔记)
- **零配置开箱即用**:首次使用自动注册一个匿名 ID,无需手动管理 token

## 适合谁用

- **社会化营销 / 内容运营**:每天追热点、找选题(小红书种草、抖音爆款话题)
- **公关 / 品牌**:监测品牌相关话题、舆论复盘、传播效果评估
- **媒介投放**:评估热点话题/综艺/影视的投放价值
- **行业研究 / 咨询**:跨平台情报快速汇总,对比内容差异

## 快速体验

skill 装好后,直接对你的 AI 助手说自然语言即可:

> "今天微博热搜前 10 是什么?"
> "看下小红书最近'夏日防晒'有哪些爆款话题"
> "瑞幸最近一周在微博和小红书的口碑怎么样"
> "某热播综艺这一周的全网讨论量级,值不值得投"
> "AI 大模型最近在知乎 vs 微博的话题分布"

AI 会自动调用对应脚本并把结果整理给你看。

## 命令行直接体验

也可以从 skill 根目录直接跑脚本:

```bash
# 1. 查询微博热搜前 10
python3 ./scripts/query_hot_list.py --source 微博 --size 10

# 2. 分析话题(默认覆盖 微博+小红书+短视频)
python3 ./scripts/analyze_topic.py --topic '瑞幸' --days 7

# 3. 采样原帖,小红书图文笔记加图片 OCR
python3 ./scripts/sample_posts.py --topic '夏日防晒' --datasource 小红书 \
    --search-fields title content screenshot --size 20

# 4. 短视频场景加 ASR/OCR 多捞内容
python3 ./scripts/analyze_topic.py --topic '某综艺' --datasource 抖音 \
    --search-fields title content video_asr video_ocr

# 5. 查询当前剩余额度
python3 ./scripts/ensure_user.py
```

无需 `pip install`,仅需 Python 3.10+ 和能访问公网的环境。

## 适用边界(请提前了解)

✅ **擅长**:头部消费品牌、品类词、热门事件、综艺/影视等**有充分社媒讨论**的话题

❌ **不擅长**:
- 长尾品牌 / B2B / 小众话题:讨论量太少,统计意义弱
- 实时危机预警:数据 T+1,不做秒级监测
- 小红书热搜榜:平台无此数据(但小红书话题分析完全可用)

## 免费额度

- **首次使用自动注册**,无需手动操作
