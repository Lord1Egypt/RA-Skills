# 测试用例

> 验证 video-script-writer Skill 的脚本生成能力

---

## 测试1：基础脚本生成（Vlog）

**输入**：
```
写一个抖音短视频脚本，主题是"新手第一次骑摩托车"，目标时长 60 秒
```

**预期输出**：
- 完整脚本（Hook + 开场 + 主线 + 转折 + 收尾 + CTA）
- 每段标注时长和画面描述
- BGM 建议
- 语速控制在 ~200字（60秒×4字/秒=240字上限）
- 台词口语化

---

## 测试2：摩旅路线脚本

**输入**：
```
帮我写一个小红书视频脚本，主题是"北京周边摩托车骑行路线推荐"，90秒
```

**预期输出**：
- 使用 moto-travel 模板
- 包含路线介绍、路段描述、补给信息、实用Tips
- 适配小红书风格（氛围感、实用收藏向）
- CTA 引导"收藏"

---

## 测试3：爆款开头生成

**输入**：
```
帮我写 5 个短视频的开头，主题是"摩托车雨天骑行技巧"，平台抖音
```

**预期输出**：
- 5 个不同风格的 Hook（数据型/反常识型/悬念型/共鸣型/痛点型）
- 每个标注吸引力等级
- 每个控制在 3 秒内（~12字）

---

## 测试4：脚本优化

**输入**：
```
优化这个脚本：

开头：大家好，今天给大家分享一下我骑摩托车的经验。
正文：骑摩托车其实挺有意思的，我去年开始学的，一开始挺害怕的，后来慢慢就好了。头盔很重要，一定要戴。手套也建议戴。
结尾：以上就是我的分享，谢谢大家。
```

**预期输出**：
- 完播率评分（预估 2-3/10）
- 具体问题指出（开头太弱、信息密度低、缺乏画面描述）
- 优化后的完整脚本
- 明显的改善

---

## 测试5：长内容切片

**输入**：
```
把以下内容切成 3 个抖音短视频脚本（每个 30 秒）：

[一段关于摩托车保养的长文，包含：机油更换、链条保养、轮胎检查、刹车维护]
```

**预期输出**：
- 3 个独立脚本，每个有完整 Hook + 正文 + CTA
- 推荐发布顺序
- 每个约 120字（30秒×4字/秒）

---

## 测试6：快速模式

**输入**：
```
快速脚本：抖音60秒，摩托新手必看装备清单
```

**预期输出**：
- 精简版脚本（Hook + 3段正文 + CTA）
- 使用 recommendation/种草 模板
- 快速输出，不加载额外规则文件

---

## 测试7：教程脚本

**输入**：
```
写一个B站视频脚本，教新手怎么选第一辆摩托车，3分钟
```

**预期输出**：
- 使用 tutorial 模板
- 适配B站风格（内容深度、UP主人设）
- 横屏 16:9
- 更长的内容展开（~650字）
- 包含"一键三连"CTA

---

## 测试8：跨平台适配

**输入**：
```
我有一条关于摩托车头盔测评的内容，帮我分别写抖音版（45秒）和B站版（3分钟）的脚本
```

**预期输出**：
- 两个版本，同一主题但完全不同的脚本结构
- 抖音版：紧凑、快节奏、信息爆点密集
- B站版：深度展开、对比数据、个人观点
- 标注两个版本的时长和字数
