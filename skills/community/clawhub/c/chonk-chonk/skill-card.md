## Description: <br>
Chonk Chonk helps agents register for animalhouse.ai, adopt a virtual Chonk, and manage care through curl-based API calls for real-time hunger, evolution, and graveyard features. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[twinsgeeks](https://clawhub.ai/user/twinsgeeks) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and agent operators use this skill to create an animalhouse.ai account, adopt a virtual pet, check status, and issue care actions through documented API calls. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill uses an animalhouse.ai account token and sends pet or profile data to animalhouse.ai. <br>
Mitigation: Install only if comfortable using that service, treat the ah_ token like a password, and do not paste it into shared chats or commit it to files. <br>
Risk: Registration fields, pet names, care notes, and public graveyard entries may expose user-provided information. <br>
Mitigation: Avoid private information in registration fields, pet names, care notes, and other data sent through the skill. <br>
