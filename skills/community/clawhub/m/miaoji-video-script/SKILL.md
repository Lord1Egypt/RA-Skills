# 秒技·爆款视频脚本

> 1张图 + 1个对标 → 秒级生成带货视频脚本

## 🎯 核心定位

**不是视频生成器，而是脚本生成器。**

本技能专注于**拆解热门视频结构 + 生成适配产品的视频脚本方案**，帮助亚马逊卖家、跨境电商从业者快速产出高转化率的带货视频脚本。

---

## 📋 技能能力

### 1. 爆款结构拆解器
输入任意爆款视频描述，智能拆解：
- 节奏段落（开场/发展/高潮/结尾）
- 运镜方式（特写/全景/推进/平移）
- 脚本结构（钩子+痛点+方案+证明+号召）
- 情绪曲线（0-3秒刺激→3-15秒共鸣→15-30秒转化）

### 2. 产品适配脚本生成
基于爆款结构 + 你的产品信息，生成完整视频脚本：

| 要素 | 说明 |
|------|------|
| 分镜编号 | 镜头序号 |
| 时长 | 建议时长（秒） |
| 画面描述 | 运镜、场景、道具 |
| 旁白文案 | 主播话术/口播内容 |
| BGM建议 | 背景音乐风格 |

### 3. 平台差异模板

| 平台 | 时长 | 节奏特点 |
|------|------|----------|
| TikTok | 15-60s | 快节奏、强钩子、病毒式传播 |
| YouTube Shorts | 60s | 略深度、可添加字幕 |
| Instagram Reels | 30s | 精致感、生活方式导向 |

### 4. 前3秒钩子库（20种）

| 类型 | 模板示例 |
|------|----------|
| 痛点型 | "还在为XX问题头疼？90%的人都做错了..." |
| 惊喜型 | "你绝对想不到XX居然还能这样用！" |
| 对比型 | "普通款 VS 我们的 = 省下XX元/时间" |
| 故事型 | "上周有个客户找到我，说..." |
| 数字型 | "3个技巧，让你XX效率提升200%" |
| 悬念型 | "这个功能，80%的买家都不知道..." |

## 🚀 使用方式

### 输入格式

```
产品描述：[产品名称+核心卖点+目标人群]
爆款参考：[爆款视频描述/风格偏好]
目标平台：[TikTok / YouTube Shorts / Instagram Reels]
视频时长：[X秒]
```

### 示例输入

```
产品描述：便携式咖啡机，小巧可加热，适合办公室人群
爆款参考：类似"早晨5分钟搞定咖啡"的晨间routine风格
目标平台：TikTok
视频时长：30秒
```

### 输出内容

1. **爆款结构分析**：拆解参考视频的脚本骨架
2. **完整分镜脚本**：含5-8个镜头，每个镜头有画面+文案
3. **BGM建议**：推荐2-3个风格方向
4. **注意事项**：平台合规、字幕建议等

---

## ⚠️ 重要说明

1. **本技能生成的是视频脚本和分镜方案**，不包含视频生成能力
2. 生成的脚本需配合视频拍摄或剪辑工具使用
3. 建议根据实际产品调整脚本内容
4. 平台政策可能随时更新，请以官方最新规范为准

---

## 💡 最佳实践

1. **参考爆款越多，脚本越精准** - 可提供2-3个对标视频
2. **产品卖点要具体** - "好用"太抽象，"3秒速热"更具体
3. **首帧决定生死** - 重点打磨前3秒钩子
4. **CTA要明确** - 结尾必须有清晰行动号召

## 交付标准

优秀的输出应该让你一眼看到：

- 前3秒有明确的钩子，能抓住观众注意力
- 时长设置匹配目标平台的最佳时长
- 包含BGM风格建议或参考音乐类型
- 每个镜头有清晰的视觉描述和运镜建议
- 结尾有明确的CTA（行动号召）

如果输出缺少以上任何一项，建议补充后再交付。

## 不适用场景

本技能不适用于以下情况：
- 需要实际生成视频（本技能只生成脚本，不生成视频）
- 纯图片设计（需使用图片设计工具）
- 长篇内容创作（超过60秒的视频脚本建议拆分为系列内容）

## 安全边界声明

本技能为纯脚本创作工具，所有脚本内容由用户自主决定如何使用。本技能不生成任何视频文件，不连接任何视频平台接口，不存储任何媒体内容，不自动发布或同步视频。
