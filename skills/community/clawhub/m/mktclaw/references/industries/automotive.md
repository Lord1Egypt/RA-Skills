### 竞争格局特征

- **高客单 + 长决策**：决策周期 1-3 个月
- **O2O 链路**：线上种草 → 线下试驾 → 4S 店成交
- **新能源 vs 燃油**：新能源主打智能化，燃油车主打可靠性
- **品牌力极强**：BBA / 造车新势力 / 自主品牌分层清晰
- **政策驱动**：补贴、牌照、限行政策影响巨大

## 5. 合规与监管

### 5.1 广告法专属限制

| 限制类型 | 禁用 |
|---------|------|
| 极限词 | "最快""最强""第一"（无依据时） |
| 性能虚标 | 续航/加速/油耗虚标 |
| 比较广告 | 不得贬低竞品 |
| 安全承诺 | "最安全""零事故" |

### 5.2 行业特殊资质

| 业务类型 | 必备资质 |
|---------|---------|
| 汽车广告 | 营业执照 + 品牌授权 |
| 二手车 | 二手车流通许可证 |
| 后市场维修 | 道路运输经营许可证 |
| 新能源补贴 | 符合国家补贴目录 |

### 5.3 平台审核红线

| 平台 | 红线 |
|------|------|
| 抖音 | 虚假测评、参数造假、贬低竞品 |
| 懂车帝 | 数据无来源、虚假"专业" |
| B站 | 利益相关未声明 |

### 5.4 特殊合规要求

| 要求 | 说明 |
|------|------|
| 续航标注 | 必须标注 CLTC/NEDC/WLTP 工况 |
| 加速成绩 | 必须标注测试条件 |
| 价格 | 必须为官方指导价或经销商报价 |
