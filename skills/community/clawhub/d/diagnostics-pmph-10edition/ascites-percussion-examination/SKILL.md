# 腹腔积液叩诊检测

## 适用场景
- 患者疑似存在腹腔积液
- 正在进行腹部体格检查中的叩诊环节
- 患者能够配合完成体位变换

## 执行步骤

### 1. 移动性浊音检查（适用于积液量 ≥1000ml）
1. 让患者取仰卧位：
   - 腹中部因肠管浮起呈鼓音
   - 两侧腹部因积液积聚呈浊音
2. 从脐水平向左侧叩诊至出现浊音处，固定板指位置
3. 嘱患者转为右侧卧位，再次叩诊该固定位置：
   - 若声音由浊音转为鼓音，提示浊音随体位移动
4. 同法向右侧叩诊得浊音后，嘱患者左侧卧位验证
5. 判定标准：体位改变后浊音区发生变动即为阳性

### 2. 水坑征检查（适用于少量积液，≥120ml）
1. 让患者取肘膝位（膝胸卧位），使脐部成为腹腔最低点
2. 从侧腹部向脐部方向叩诊
3. 若叩诊音由鼓音转为浊音，提示存在腹腔积液（水坑征阳性）
4. 替代方法：患者站立位时，下腹部浊音上界呈水平线，其上方为鼓音（浮动肠曲）

## 注意事项
- 肠梗阻或巨大卵巢囊肿可能导致假阳性结果
- 移动性浊音阴性不能排除少量积液，应结合水坑征进一步评估
- 所有操作需采用间接叩诊法

## 结果解读
阳性结果提示腹腔积液存在，需结合临床表现及其他辅助检查（如超声）确认；阴性结果不排除少量积液可能。
