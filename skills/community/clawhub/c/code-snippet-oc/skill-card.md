## Use Case: <br>
Developers use this skill to save, tag, search, highlight, and copy reusable code snippets during coding workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Saved snippets may include secrets, proprietary code, or other sensitive material. <br>
Mitigation: Avoid storing API keys, passwords, private keys, proprietary code, or other sensitive content unless local retention is intentional and deletion procedures are understood. <br>
Risk: Locally retained snippet and tag files may persist beyond the immediate session. <br>
Mitigation: Review and delete snippets.json and tags.json when retained snippets should no longer be available. <br>
