# 接案评估量化口径

用本文件统一“胜诉概率、诉讼周期、资金成本、时间成本、接案建议”的测算方式。先评分，再给结论；先写依据，再写区间。

## 一、胜诉概率

### 1. 评分结构

按下列五部分评分：

| 维度 | 分值 | 判断重点 |
| :--- | :--- | :--- |
| 实体请求权基础 | 0-35 | 法律关系是否清晰、请求权是否稳定、构成要件是否可覆盖 |
| 证据质量 | 0-30 | 核心证据是否齐全、证明链是否闭环、证据三性是否稳固 |
| 程序可行性 | 0-15 | 管辖、时效、仲裁条款、主体适格、举证责任分配是否可控 |
| 法条与类案支持度 | 0-20 | 检索到的规范和类案是否对我方路径形成稳定支撑 |
| 重大风险扣分 | 0 到 -25 | 时效临界、证据违法取得、合同无效、履行抗辩、执行落空等重大问题 |

计算方式：

```text
基础得分 = 实体请求权基础 + 证据质量 + 程序可行性 + 法条与类案支持度 + 重大风险扣分
胜诉概率中心值 = 将基础得分限制在 5 到 95 之间
胜诉概率区间 = 中心值 ± 5%
```

出现下列情况时，将区间扩大为 `中心值 ± 10%`：

- 关键信息缺失较多
- 仅有法规检索，没有有效类案支撑
- 双方事实版本严重冲突
- 关键证据仍待核验真伪

### 2. 评分锚点

#### 实体请求权基础 0-35

- `28-35`：法律关系和请求权路径明确，构成要件基本齐备，抗辩空间有限。
- `18-27`：请求权路径总体可行，但关键要件需要解释、推定或补证。
- `0-17`：请求权基础存在重大分歧，案由或适法路径不稳定。

#### 证据质量 0-30

- `24-30`：有核心书证或交易闭环，证据链完整，三性较稳。
- `14-23`：有一定证据基础，但关键节点仍依赖截图、录音、证人等间接材料。
- `0-13`：关键事实缺少直接证据，或证据来源、完整性明显存疑。

#### 程序可行性 0-15

- `12-15`：管辖清晰、时效安全、主体适格、程序障碍较少。
- `6-11`：程序上可推进，但存在管辖异议、时效争议、仲裁条款解释等变量。
- `0-5`：程序障碍重大，足以显著削弱受理或支持前景。

#### 法条与类案支持度 0-20

- `15-20`：核心规则明确，且能检索到事实结构接近、裁判倾向一致的类案。
- `8-14`：有一定规范和类案支撑，但相似度或稳定性一般。
- `0-7`：规范适用存在较大争议，或公开类案对我方不利。

#### 重大风险扣分 0 到 -25

常见扣分项：

- 诉讼时效或除斥期间风险：`-5` 到 `-15`
- 关键证据真实性 / 合法性缺陷：`-5` 到 `-10`
- 合同效力或主体资格重大瑕疵：`-5` 到 `-10`
- 对方强抗辩显著成立可能：`-5` 到 `-10`
- 执行回收高度不确定且直接影响接案目标：`-3` 到 `-8`

### 3. 概率结论用语

| 概率区间 | 结论用语 |
| :--- | :--- |
| 80%-95% | 胜诉前景较强 |
| 65%-79% | 胜诉前景较为积极 |
| 50%-64% | 胜诉前景中性偏谨慎 |
| 35%-49% | 胜诉前景偏弱 |
| 5%-34% | 胜诉前景较低 |

写结论时同时说明“主要拉分项”和“主要扣分项”。

## 二、诉讼周期

### 1. 基础区间

先按复杂度给出一审基础区间：

| 案件复杂度 | 一审基础周期 |
| :--- | :--- |
| 事实清楚、证据集中、无鉴定 | 4-6 个月 |
| 存在实质争议、需多轮补证 | 6-10 个月 |
| 多方主体、需鉴定 / 审计 / 专业认定 | 10-18 个月 |

### 2. 加减项

在基础区间上叠加以下因素：

| 因素 | 影响 |
| :--- | :--- |
| 管辖异议 | `+1` 到 `+3` 个月 |
| 鉴定、审计、评估 | `+3` 到 `+8` 个月 |
| 公告送达或送达困难 | `+2` 到 `+6` 个月 |
| 追加当事人 / 反诉 | `+1` 到 `+4` 个月 |
| 和解或法院强力调解 | `-1` 到 `+2` 个月 |
| 二审概率高 | 再增加 `3` 到 `6` 个月 |
| 执行程序可能性高且回收难 | 再增加 `3` 到 `12` 个月 |

### 3. 输出方式

优先输出“阶段拆分”而不是只给总数：

```text
一审：6-10个月
二审（如进入）：3-5个月
执行（如进入）：4-8个月
综合判断：从立案到回款，预计约 10-20 个月
```

## 三、资金成本

### 1. 拆分口径

至少拆为以下项目：

- 诉讼费 / 申请费
- 财产保全与担保成本
- 鉴定、审计、评估、公证、翻译等第三方费用
- 差旅、送达、调档等程序性支出
- 律师费或外部专业支持费用

### 2. 输出规则

- 已知标的额且收费口径可靠时：输出人民币区间，并说明测算假设。
- 未知标的额或收费标准不稳时：输出成本等级与主要构成，不虚构精确金额。

### 3. 资金成本等级

| 等级 | 典型特征 |
| :--- | :--- |
| 低 | 无保全、无鉴定、庭次少、跨地域因素弱 |
| 中 | 可能保全或多轮开庭，但外部第三方费用有限 |
| 高 | 涉鉴定 / 审计 / 评估 / 保全担保，或标的额较大、执行投入显著 |

如果必须量化但金额资料不足，可补充“资金成本指数 1-5”：

- `1-2`：低
- `3`：中
- `4-5`：高

## 四、时间成本

从律师团队与客户两侧同时估算：

| 等级 | 律师投入工时 | 客户配合工时 | 典型场景 |
| :--- | :--- | :--- | :--- |
| 低 | 20-40 小时 | 8-16 小时 | 事实清晰、证据集中、争议点少 |
| 中 | 40-80 小时 | 16-40 小时 | 需补证、需多轮沟通、存在抗辩交锋 |
| 高 | 80-160 小时 | 40-80 小时 | 多主体、专业问题强、需鉴定或多地协调 |
| 很高 | 160 小时以上 | 80 小时以上 | 高标的、跨区域、程序链条长 |

## 五、接案建议

按“胜诉概率 + 回收预期 + 成本承受 + 证据可补强性 + 商业目标匹配度”综合判断：

| 建议 | 适用情形 |
| :--- | :--- |
| 建议接案 | 胜诉概率较高，关键证据齐备，回收或目标实现路径清晰 |
| 有条件接案 | 方向可做，但需补证、调整诉请、优化被告选择或先行保全 |
| 谨慎接案 | 存在较大实体或程序风险，成本和回收预期不成比例 |
| 不建议接案 | 胜诉概率偏低，或诉讼无法实现客户核心目标，或合规风险过高 |

写“有条件接案”时，必须列出至少 2 项前置条件。

## 六、限制说明

出现以下情况时，主动降低结论强度：

- 仅凭单方陈述、未见任何核心原始证据
- 关键法规或类案检索受限
- 涉及地方性审判尺度差异，但缺少对应法院公开样本
- 争议更适合仲裁、行政复议、执行异议或刑民交叉路径
