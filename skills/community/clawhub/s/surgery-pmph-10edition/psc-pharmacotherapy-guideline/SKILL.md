# 原发性硬化性胆管炎药物治疗规范

## 适用条件
- 患者已确诊为原发性硬化性胆管炎（PSC）
- 无药物治疗禁忌证
- 肝功能可评估

## 禁忌情形
- 继发性硬化性胆管炎
- 已确诊合并胆管癌

## 核心操作流程
1. **推荐使用熊去氧胆酸（UDCA）**，剂量严格控制在 **17～23 mg/(kg·d)**。
2. 此剂量范围可改善临床症状及肝功能指标（如碱性磷酸酶 ALP、胆红素）。
3. **禁止使用大剂量 UDCA（>28 mg/(kg·d)）**，因其不仅无临床获益，反而显著增加以下不良事件风险：
   - 静脉曲张发生率升高
   - 肝移植需求比例增加
   - 整体临床预后恶化
4. **停用无效药物**：硫唑嘌呤、甲氨蝶呤、泼尼松龙、环孢素等免疫抑制剂因无法改善肝脏生化指标或临床结局，现已不推荐用于PSC治疗。
5. **用药期间监测**：定期评估肝功能、门静脉高压相关并发症及症状变化。

## 输出行动
根据患者体重计算并选择合适 UDCA 剂量，启动或调整治疗方案，并停用已被证实无效的免疫抑制剂。
