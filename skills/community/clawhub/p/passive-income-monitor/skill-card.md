## Description: <br>
Monitors and summarizes earnings from bandwidth nodes, decentralized storage, DeFi yield positions, and staking rewards with configurable alerts for drops or offline status. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and agent operators use this skill to monitor passive-income streams, inspect earnings summaries, configure alerts, and export local earning history for accounting or review. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may involve privacy-sensitive wallet, node, earnings, and alert configuration data stored locally. <br>
Mitigation: Secure local configuration and earnings files and avoid sharing exported histories unless they have been reviewed. <br>
Risk: Optional API keys and webhook alert endpoints could expose account or notification data if over-privileged or public. <br>
Mitigation: Use read-only or least-privilege API keys and send webhook alerts only to private HTTPS endpoints you control. <br>
Risk: Crypto and DeFi monitoring can surface financial data that may be incomplete, delayed, or unsuitable for transaction decisions. <br>
Mitigation: Treat results as monitoring summaries, verify important balances and yields against authoritative sources, and never provide seed phrases or private keys. <br>
