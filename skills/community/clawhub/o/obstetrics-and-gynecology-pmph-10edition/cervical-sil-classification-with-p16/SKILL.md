# 子宫颈鳞状上皮内病变（SIL）的二级分类与P16辅助诊断

## 何时使用
- 组织学切片显示鳞状上皮异型增生，初步诊断为CIN2；
- 形态学特征模糊，无法可靠归类为LSIL（CIN1）或HSIL（CIN2/3）；
- 病例为子宫颈来源，且排除腺上皮内病变。

## 执行步骤
1. **进行P16免疫组化染色**：对目标组织切片实施标准P16免疫组化检测。
2. **判读染色结果**：
   - 若P16呈**阴性**（无表达或仅局灶弱表达），则归类为**低级别鳞状上皮内病变（LSIL，相当于CIN1）**；
   - 若P16呈**强而弥漫的片状阳性**，则归类为**高级别鳞状上皮内病变（HSIL）**。
3. **明确HSIL亚型**：根据WHO（2020）指南，在诊断HSIL时需进一步注明为CIN2或CIN3，以支持临床精准管理。
4. **记录诊断依据**：在病理报告中说明P16结果及其对分类的决定性作用。

> 注意：本规则仅适用于鳞状上皮内病变，不可用于外阴、阴道或腺上皮病变的诊断。
