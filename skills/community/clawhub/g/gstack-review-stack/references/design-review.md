# Design Review

Use this for UI/UX plans, landing pages, store listings, extension popups, side panels, and screenshots.

## Score 0-10

Rate:
- First-screen clarity
- User job clarity
- Trust and permission comfort
- Visual originality
- Interaction completeness
- Mobile/responsive behavior
- Copy simplicity
- Empty/error/loading states

For each score below 8, write the exact change that would raise it.

## AI-Slop Flags

Flag:
- Generic centered hero with vague claims
- Overdecorated cards with no workflow density
- Text that a nontechnical user cannot understand
- Screenshots that do not show the actual product
- Permission-heavy UX without trust explanation
- Missing empty/error states
- Unclear call to action

## Required Output

Give concrete edits:
- Replace this phrase with that phrase
- Move this control here
- Add/remove this state
- Change screenshot composition
- Simplify the listing promise
