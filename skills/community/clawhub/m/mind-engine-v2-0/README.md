# Mind Engine — Universal Thinking Framework

A 7-stage thinking engine that transforms any AI assistant into a structured reasoning partner. When a user asks a question, the engine runs through Problem Diagnosis → Model Matching → Dialogue Exploration → Hypothesis Generation → Exhaustive Verification → Recommendation Output → Cognitive Consolidation.

## What It Does

- **Structured Thinking**: Replaces gut-feeling advice with method-driven analysis
- **Methodology-Cited**: Every recommendation tagged with its source framework
- **Multi-Hypothesis**: Always generates 3+ distinct options, never a single answer
- **Conversational**: Asks methodology-driven questions instead of jumping to conclusions
- **Self-Evolving**: Evaluates its own performance and updates after each use

## The 7 Stages

| Stage | What Happens |
|-------|-------------|
| 1. Problem Diagnosis | Classify problem type, uncertainty, stakeholders, biases |
| 2. Model Matching | Map the problem to proven frameworks (game theory, probability, systems, etc.) |
| 3. Dialogue Exploration | Ask methodology-driven questions before giving answers |
| 4. Hypothesis Generation | Produce 3+ distinct paths with conditions, outcomes, risks |
| 5. Exhaustive Verification | Test each hypothesis against ergodicity, recursion, antifragility |
| 6. Recommendation Output | Deliver structured options with full reasoning chains |
| 7. Cognitive Consolidation | Evaluate and improve the framework itself |

### Manual

## Customization

The engine is designed to work with your own knowledge bases. Replace the generic model matching table with your specific methodology inventory, or append your knowledge base file paths to the customization section in SKILL.md.

## Who Is This For

- Decision-makers who want structured analysis instead of gut feelings
- Knowledge workers managing complex, multi-stakeholder problems
- Anyone who has accumulated reading notes/knowledge bases and wants an AI to actually use them
- People who are tired of AI assistants that give confident-but-shallow answers
