## 🚨 2025-2026年重大监管更新

### 一、证监会《上市公司董事会秘书监管规则》

| 项目 | 内容 |
|------|------|
| **发布日期** | 2026年4月24日 |
| **施行日期** | 2026年5月24日 |
| **文号** | 证监会公告〔2026〕8号 |
| **意义** | 🎯 **中国首部专门针对董秘的部门规章** |

**核心变化：**

1. **任职禁止（第二十六条）**
   - ⛔ 董事会秘书**不得兼任**：经理、分管经营业务的副经理、财务负责人
   - 影响：A股约2400家上市公司需调整董秘兼任情况
   
2. **任职资格大幅提升**
   - 具备**5年以上**财务、会计、审计、法律合规、金融从业工作经验；**或**
   - 取得**法律职业资格证书** 且 具有5年以上工作经验；**或**
   - 取得**注册会计师证书** 且 具有5年以上工作经验

3. **职责明确**
   - 作为上市公司信息披露活动**组织者**
   - 对定期报告的异常情形予以关注、核实
   - 对临时报告的真实、准确、完整承担**主要责任**

4. **过渡期安排**
   - 任职、兼职等事项过渡期至 **2027年12月31日**
   - 过渡期内与《董秘规则》不一致的，应逐步调整至符合规定

### 三、香港交易所《董事會及董事企業管治指引》（2025年更新）

**核心要求：**
- 公司秘书须为香港Institute of Chartered Secretaries (HKICS) 等认可专业机构会员
- 必须为**香港居民**，且随时可向董事会提供建议
- 兼任限制：同一人不得担任超过**3家**无关联上市公司的公司秘书（除非证明有足够能力）
- 2025年7月1日起，IPO申请人上市时不得有超额任职独立非执行董事

**新增职责（2025版）：**
- ESG策略支持与披露合规
- 网络安全与数据治理风险建议
- 年度管治审查与董事绩效评估支持
