## Description: <br>
Automate TikTok engagement on Android using ADB with topic search, AI-generated or template comments, and an interactive setup wizard. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and operators use this skill to configure and run Android ADB workflows that search TikTok topics or For You feed videos and post public comments using static templates or AI-generated text. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can automatically post public TikTok comments from the logged-in Android account. <br>
Mitigation: Use a dedicated account and device, start with very small limits, and monitor public activity before increasing usage. <br>
Risk: AI comment mode may send screenshots to external AI providers. <br>
Mitigation: Use static comments when screenshots should not leave the device, or review provider settings and data handling before enabling AI mode. <br>
Risk: Scheduled automation can continue posting if behavior is no longer desired. <br>
Mitigation: Avoid unattended cron runs until monitoring and a simple disable path are in place. <br>
