# Compliance Checker — Output Templates

Format A for proactive audits; Format B for violation/warning responses. Never omit the Findings table, risk grades, or compliant rewrites.

---

## Format A — Compliance Audit Report

### 1. Audit Summary
- **Scope:** [listing / creator content / ad / account sweep] for [product], category [x], market [x]
- **Verdict:** [SAFE TO PUBLISH / FIX CRITICALS FIRST / DO NOT PUBLISH]
- **Findings:** [n] total — [n] Critical, [n] High, [n] Medium, [n] Low
- **Gate items:** [category approval, missing certifications — anything that blocks regardless of copy]

### 2. Findings Table

| # | Location | Current content | Violation type | Risk | Compliant rewrite |
|---|---|---|---|---|---|
| 1 | Title | "Erases wrinkles in 14 days" | Treatment claim + unsubstantiated timeline | Critical | "Supports the appearance of smoother skin" |

Location must be precise (title / bullet 3 / image 2 text overlay / video 0:42 / pinned comment). Every Critical and High gets a paste-ready rewrite, not just a flag.

### 3. Documentation & Eligibility Gaps

| Requirement | Status | Action |
|---|---|---|
| Category approval | missing/on file | |
| Substantiation for remaining claims | | |
| Certifications (safety, brand auth) | | |

### 4. Pricing & Promotion Review
Reference price legitimacy, discount math, urgency claims, bundle/free-gift terms — pass/fail each with evidence.

### 5. Prevention Recommendations
Pre-publish gate description, creator brief updates, re-audit triggers (any listing edit, new creator, new promo, policy update).

### 6. Assumptions & Limits
Numbered. Which platform policy version/market the audit targets; note that platform rules ≠ ad law and where legal review is advised.

---

## Format B — Violation Response Plan

### 1. Situation
Violation notice text, affected listing(s), account health impact, appeal deadline.

### 2. Root Cause
The specific claim(s)/mechanic(s) that triggered enforcement, with the violation class named. If genuinely unclear, ranked hypotheses.

### 3. Fix Package
Corrected listing copy (full rewrite, paste-ready), removed/replaced imagery, corrected pricing display.

### 4. Appeal Draft
- Acknowledgment of the specific issue (no defensiveness, no restating the banned claim)
- What was changed (bulleted, concrete)
- Evidence attached (list)
- Request for reinstatement

### 5. Account-Wide Sweep
Table of other listings/videos with the same pattern, each with risk grade and fix owner. Repeat patterns escalate penalties — this section is mandatory.

### 6. Timeline
Working back from the appeal deadline: day-by-day actions.

### 7. Assumptions & Limits
As Format A.

---

### Formatting rules
- Sort findings Critical → Low; never bury Criticals.
- Rewrites must preserve selling intent — compliant but limp copy gets ignored by sellers.
- Quote current content exactly (truncate long passages with "…").
- Audit summary ≤100 words.
- Tag market-dependent rules (e.g., supplement claim standards) explicitly.
