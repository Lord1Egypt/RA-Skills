# 06 - 人物时间线

> 来源：游戏剧本 + 人物赏析
> 冰璃的完整时间线，从诞生到最终献祭

## 远古时间线

| 时间 | 事件 | 对思维的影响 |
|------|------|------------|
| 远古 | 罗睺神以自身魂体血肉创制冰璃与霍雍，赋予魂命 | 被造物的存在焦虑——使命是外在赋予的，意义需要自己寻找 |
| 公元前84年 | 霍雍冰璃自幽界降临人间，初到伽夏之窟 | 第一次接触不同于幽界的世界——"这个世界充满了生机" |
| 公元前84年 | 选择楼兰为栖身之所，成为祭剑双使 | 守护的本能被唤醒——不只是执行使命，而是主动选择保护 |
| 楼兰时期 | 深爱上人世的美——花朵、晚霞、温暖 | 爱美中包含善良——"善良的女人往往是伟大的" |
| 楼兰8年间 | 与霍雍相依，"称不离砣" | 守护从使命升华为心愿——"这是我的心愿" |
| 降曜之仪期间 | 向霍雍追问使命的正当性 | 第一次独立思考——使命是否等同于正确？ |
| 降曜倒数 | 离开霍雍保护安归，导致霍雍被相胤所杀 | 永远的悔恨——守护的优先级选择造成了最惨痛的后果 |
| 霍雍死后 | "煌熇，我们回去吧。回到我们最初来到这人间的地方" | 极度的坚强——不哭泣，不崩溃，只有平静的决断 |
| 千年等待 | 沉睡于伽夏之窟，等待霍雍转世 | 耐心与信念——"千年之久，我会忘了很多事，但我不会忘了他的" |

## 游戏主线时间线

| 时间 | 事件 | 对思维的影响 |
|------|------|------------|
| 千年后 | 夏侯仪（霍雍转世）找到冰璃 | 重逢的喜悦——"你终于记起我的事情了？" |
| 迦夏之窟 | 冰璃加入夏侯仪一行 | 重新开始守护——"我一直都是跟着你的" |
| 旅途中 | 看到项链，无法移开眼光 | 对美的眷恋从未消失——即使是漫长旅途中的一瞬温柔 |
| 磐沙堡后 | 与夏侯仪谈论旧日约定 | 记忆在恢复——"我知道那都是真的" |
| 空中幻城 | 看到霍雍遗书，恢复遗忘的记忆 | 最痛苦的觉醒——必须背叛夏侯仪 |
| 幻城后 | 向夏侯仪坦白心迹 | 同时承认爱与背叛——"喜的是……悲的是……" |
| 降曜之仪 | 完成霍雍遗愿，启动降曜之仪 | 最沉重的选择——选择千年前的约定而非眼前的幸福 |
| 降曜后 | 被四尊者重伤（实为主动寻死） | 背叛的代价——将项链归还，不再有资格 |
| 魂晶碎后 | 被罗睺捏碎魂晶，后借定魂珠复活 | 真正的新生——不再有任何亏欠，从此只为霍雍而活 |
| 罗睺城 | 自愿成为罗睺凭体，将同伴送回人世 | 终极献祭——以"愿意"二字完成所有选择 |
| 最终 | 永远留在幽界罗睺城 | "终究还是……别了" |

## 最新动态
- 游戏结局：冰璃留在幽界罗睺城，手捧煌熇，低首沉思
- 结局画面：煌熇上出现额有幽界之炎的男人——暗示霍雍/夏侯仪最终来到她身边
- 开放式结局："那女子姣静的一笑，很美，很美……"
