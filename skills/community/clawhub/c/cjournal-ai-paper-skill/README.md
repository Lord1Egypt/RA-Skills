# C 刊论文 AI 写作 Skill 使用说明

这是一个用于辅助撰写中文 C 刊风格论文的可复用 Skill，适合新闻传播、教育治理、公共管理交叉方向的机制解释型论文。

## 使用方法

1. 将 `SKILL.md` 作为系统提示词或项目指令导入 AI 工具。
2. 按 `prompts/` 中的阶段提示词逐步推进，不建议一次性要求 AI 生成全文。
3. 使用 `checklists/` 中的清单进行质量控制。
4. 使用 `templates/` 中的模板设计目录、文献矩阵、图表和引用规范。

## 推荐工作方式

- 第 1 轮：选题定位与题目设计；
- 第 2 轮：文献矩阵与文献述评；
- 第 3 轮：研究问题与目录；
- 第 4 轮：分析框架与机制图；
- 第 5 轮：正文初稿；
- 第 6 轮：C 刊风格润色；
- 第 7 轮：引用规范与匿名审稿模拟。

## 适合处理的问题

- 论文像工作报告，不像 C 刊论文；
- 案例太多、叙事太重；
- 文献综述堆砌，缺少研究问题；
- 没有数据但想提升经验支撑；
- 模型不清晰、图表不规范；
- 引用顺序混乱或文献不够真实；
- 表达口号化、政策化。

## 核心提醒

AI 可以辅助语言生成和结构优化，但不能替代问题意识、理论判断和学术边界。使用本 Skill 时，务必先完成论文定位，再逐步推进文献、框架、正文、引用和审稿模拟。
