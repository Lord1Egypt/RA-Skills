# D07 法律领域任务目录

## 领域定义

法律领域(D07)关注个体在规则框架内保护权益。核心驱动需求：N2安全。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人法律知识 | 法学研究 |
| 合同法律问题 | 法律服务提供 |
| 劳动权益保护 | 劳动仲裁服务 |
| 婚姻家庭法律 | 婚姻咨询服务 |
| 财产法律保护 | 资产管理服务 |
| 纠纷解决 | 律师代理服务 |
| 法律风险管理 | 法律合规咨询 |
