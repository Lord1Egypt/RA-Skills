## Description: <br>
Converts text descriptions into Barbie-style AI portrait requests and returns generated fashion-doll image URLs through the Neta AI image generation API. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[omactiengartelle](https://clawhub.ai/user/omactiengartelle) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External creators, social media teams, and developers use this command-line skill to turn short descriptions or optional reference image UUIDs into Barbie-style portrait image URLs for creative content. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Prompts, optional reference image UUIDs, and the supplied Neta API token are sent to api.talesofai.com. <br>
Mitigation: Use a dedicated low-scope or short-lived token when available, avoid sensitive personal data in prompts, and pass secrets through an environment variable rather than directly in shell history. <br>
Risk: The skill returns externally hosted image URLs generated by a third-party API. <br>
Mitigation: Review generated images and URL handling before sharing or embedding them in production workflows. <br>
