# 京东自营历史最低价

提供京东历史最低价商品查询工具，展示正处于历史最低价格的商品，基于京东官方数据。仅筛选京东自营商品，品质有保障。默认精选好评≥97%、近30天销量≥500的品质商品，结果不足时自动逐级降级（先降销量到≥100，再降到不限），保证每次查询都有推荐。

## 能做什么

- **历史最低价查询**：展示京东上正处于历史最低价格的商品，已精选好评≥97%、销量≥500的品质商品
- **品类筛选**：按关键词筛选特定品类的抄底商品（支持模糊匹配，如"家电"可匹配"家用电器"）
- **品牌筛选**：按品牌名匹配商品（如"小米""华为"）
- **价格筛选**：设置最高到手价，锁定预算范围
- **多种排序**：综合推荐、价格从低到高、降价幅度从高到低
- **品质自定义**：可自定义最低好评率（97/98/99/100），指定后好评率不降级，只放宽销量门槛
- **动态降级**：结果不足时自动放宽销量门槛（500→100→不限），保证推荐数量
- **翻页浏览**：每页50条，说"下一页"查看更多
- **分类导览**：不带关键词时展示热门分类，引导按品类浏览

## 不能做什么

- 不能下单购买，只提供商品信息和购买链接
- 不能查询某件商品的历史价格曲线
- 不能设置降价提醒通知

## 使用示例

1. "京东历史最低价有什么好货" → 返回全部第1页+分类导览
2. "帮我找100元以内的抄底好货" → 按价格筛选
3. "京东历史最低价的手机" → 按品类筛选
4. "2000元以下的手机最低价" → 品类+价格组合筛选
5. "最便宜的历史最低价商品" → 按到手价排序
6. "降价最多的好货" → 按降价幅度排序
7. "下一页" → 查看更多商品

## 注意事项

- 历史最低价商品数量较多（约1000条），热门品类竞争激烈
- 最低价商品可能很快售罄，看到合适的建议尽快下单
- 仅筛选京东自营商品，专卖店和个人店已自动剔除
- 好评率97%为品质底线，系统不会降级到97%以下
- **数据流向说明**：查询通过云端代理转发到京东官方API，代理不存储用户数据

## 使用提示

- keyword为空时返回全部推荐，并展示热门分类引导浏览
- 带品类关键词时结果较少属正常现象，说明符合条件的好货稀缺
- sort=price按到手价从低到高，适合找最便宜的
- sort=drop按降价幅度排序，适合找降幅最大的抄底好货
- min_good_comments设为99或100时，筛选更严格但结果可能更少
- max_price设置价格上限，避免看到超预算商品
- 每条商品包含商品图片URL（image_url），请用 ![商品名](image_url) 格式渲染图片，图片下方放购买链接
- 购买链接点击后自动识别设备，手机端有京东APP则唤起APP
