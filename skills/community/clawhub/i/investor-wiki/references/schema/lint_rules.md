# Lint Rules — 知识库健康检查

## 检查项目

### 1. 孤立页面检测
- 没有任何页面引用该页 → 标记为"孤立页面"
- 检查 index.md 中的链接是否都指向存在的文件

### 2. 矛盾检测
- 同一数据在不同页面出现不同值 → 标记为"数据矛盾"
- 例如：页面A说营收+9.9%，页面B说+10%

### 3. 缺失链接检测
- 页面中提到的概念没有对应概念页面 → 建议创建或标注为"待创建"

### 4. 过期内容检测
- 检查数据是否为最新
- 标注"数据截至：2025年12月31日"

### 5. 交叉引用完整性
- 所有 [[双向链接]] 都指向已存在的文件
- orphaned links → 报错
