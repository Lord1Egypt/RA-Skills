## W2-内容生成域范本

### W2-01 PPT课件生成范本

**任务目标**：生成演示文稿、幻灯片、汇报课件

**执行步骤**：

1. **确定课件主题**
   - 汇报主题：工作汇报、项目汇报
   - 演讲主题：会议演讲、活动演讲
   - 培训主题：技能培训、知识培训

2. **编写课件内容**
   - 封面：主题+作者+日期
   - 目录：内容概览
   - 正文：详细内容
   - 总结：回顾重点

3. **设计课件页面**
   - 页面风格：与主题一致
   - 页面布局：清晰、美观
   - 页面元素：文字、图片、图表

4. **添加课件动画**
   - 页面切换：流畅、自然
   - 元素动画：有节奏、有重点
   - 互动效果：增强参与感

**输出示例**：

```markdown
## PPT课件设计方案

### 课件主题
- 主题：年度工作汇报
- 目标：展示工作成果

### 课件内容
- 封面：年度工作汇报+姓名+日期
- 目录：工作概述、工作成果、问题分析、未来计划
- 正文：详细内容
- 总结：重点回顾、感谢聆听

### 课件页面
- 页面风格：现代、专业
- 页面布局：图文并茂、清晰美观
- 页面元素：文字、图片、图表、图标

### 课件动画
- 页面切换：淡入淡出
- 元素动画：逐步显示
- 互动效果：点击展开
```

### W2-02 教学课件生成范本

**任务目标**：生成教学课件、教案、教学设计

**执行步骤**：

1. **确定教学目标**
   - 知识目标：掌握某个知识点
   - 技能目标：学会某个技能
   - 态度目标：培养某种态度

2. **编写教学内容**
   - 导入：引起兴趣
   - 讲解：系统讲解
   - 练习：巩固知识
   - 总结：回顾重点

3. **设计教学页面**
   - 页面风格：清晰、易读
   - 页面布局：层次分明
   - 页面元素：文字、图片、动画

4. **设计教学互动**
   - 提问环节：引发思考
   - 练习环节：巩固知识
   - 讨论环节：加深理解

**输出示例**：

```markdown
## 教学课件设计方案

### 教学目标
- 知识目标：掌握Python基础语法
- 技能目标：学会编写简单程序
- 态度目标：培养编程兴趣

### 教学内容
- 导入：展示Python能做什么
- 讲解：变量、数据类型、运算符
- 练习：编写简单程序
- 总结：回顾重点知识

### 教学页面
- 页面风格：简洁、清晰
- 页面布局：代码+解释+示例
- 页面元素：代码块、文字说明、运行结果

### 教学互动
- 提问环节：这个代码是什么意思？
- 练习环节：编写一个简单程序
- 讨论环节：你学到了什么？
```

### W2-03 培训课件生成范本

**任务目标**：生成企业培训、员工培训、技能培训课件

**执行步骤**：

1. **确定培训目标**
   - 知识目标：掌握某个知识
   - 技能目标：学会某个技能
   - 行为目标：改变某种行为

2. **编写培训内容**
   - 理论知识：系统讲解
   - 实践操作：动手练习
   - 案例分析：加深理解
   - 考核评估：检验效果

3. **设计培训页面**
   - 页面风格：专业、有品牌感
   - 页面布局：清晰、有条理
   - 页面元素：文字、图片、视频

4. **设计培训评估**
   - 课前测试：了解基础
   - 课中练习：巩固知识
   - 课后考核：检验效果

**输出示例**：

```markdown
## 培训课件设计方案

### 培训目标
- 知识目标：掌握销售技巧
- 技能目标：学会客户沟通
- 行为目标：提升销售业绩

### 培训内容
- 理论知识：销售心理学、沟通技巧
- 实践操作：角色扮演、模拟销售
- 案例分析：成功案例、失败案例
- 考核评估：笔试+实操

### 培训页面
- 页面风格：专业、有品牌感
- 页面布局：理论+实践+案例
- 页面元素：文字、图片、视频、图表

### 培训评估
- 课前测试：销售基础知识测试
- 课中练习：角色扮演练习
- 课后考核：销售技能考核
```

### W2-04 在线课件生成范本

**任务目标**：生成网络课程、在线学习、MOOC课件

**执行步骤**：

1. **确定课程主题**
   - 知识课程：传授知识
   - 技能课程：教授技能
   - 兴趣课程：培养兴趣

2. **编写课程内容**
   - 课程介绍：课程目标、适合人群
   - 课程大纲：章节安排、学习内容
   - 课程视频：讲解视频、演示视频
   - 课程资料：课件、练习、参考资料

3. **设计课程页面**
   - 页面风格：适合在线学习
   - 页面布局：清晰、易导航
   - 页面元素：视频、文字、互动

4. **设计课程互动**
   - 讨论区：交流学习心得
   - 作业区：提交作业
   - 测试区：检验学习效果

**输出示例**：

```markdown
## 在线课件设计方案

### 课程主题
- 主题：Python编程入门
- 目标：学会Python基础编程

### 课程内容
- 课程介绍：适合零基础学习者
- 课程大纲：10个章节，每章30分钟
- 课程视频：讲解视频+演示视频
- 课程资料：课件+练习+参考资料

### 课程页面
- 页面风格：简洁、现代
- 页面布局：视频+课件+资料
- 页面元素：视频播放器、文字说明、下载链接

### 课程互动
- 讨论区：交流学习心得
- 作业区：提交编程作业
- 测试区：章节测试+期末测试
```
