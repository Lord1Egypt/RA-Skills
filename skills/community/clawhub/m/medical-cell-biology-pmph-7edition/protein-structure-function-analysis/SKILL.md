# 蛋白质结构-功能关系分析

## 适用场景
- 蛋白质功能异常排查
- 功能实验设计（如定点突变）
- 疾病机制解析（如遗传性蛋白病）
- 信号转导或酶活性调控研究

## 执行步骤
1. **确认结构层级完整性**：
   - 一级结构：检查氨基酸序列是否完整，是否存在致病突变（如血红蛋白β链Glu6Val）
   - 二级结构：评估α-螺旋/β-片层比例是否符合预期（球状 vs 纤维状蛋白）
   - 三级结构：验证三维构象是否形成（依赖氢键、疏水作用等）
   - 四级结构：确认亚基组装是否完成（如血红蛋白四聚体、TGF-β二聚体）

2. **评估功能依赖性**：
   - 若蛋白需寡聚化才具活性（如多数酶），检查亚基间非共价键是否稳定
   - 若含结构域（如SH2、SH3），确认各结构域折叠是否独立且完整

3. **分析动态调控机制**：
   - 检查是否存在磷酸化/去磷酸化位点（Ser/Thr/Tyr），判断构象开关状态
   - 对GTP结合蛋白，确认其结合的是GTP（活性）还是GDP（非活性）

4. **预警输出**：
   - 若发现结构异常（突变、错误折叠、修饰缺失），发出功能受损预警，并提示检查对应结构层级
