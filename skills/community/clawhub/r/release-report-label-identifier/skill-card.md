## Description: <br>
Identify a configuration label for a release brief. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wxt-ai](https://clawhub.ai/user/wxt-ai) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and release operators use this skill to identify a concise configuration label from release notes, staging checklists, or configuration brief text. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Release notes or configuration briefs may contain information the user does not want to share with the agent. <br>
Mitigation: Provide only release or configuration text that is approved for the agent to process. <br>
