# 心悸的病因与发生机制分类

## 何时使用
- 患者明确主诉“心慌”、“心跳快”、“心跳不规律”或“自觉心跳异常”；
- 需判断心悸是否由器质性疾病引起；
- 需排除药物、生理或心理因素干扰后进行病因归类。

## 执行步骤

### 第一步：按六大类病因分类
1. **心脏搏动增强**
   - 生理性：运动、紧张、咖啡因摄入、妊娠；
   - 病理性：心室肥大、甲状腺功能亢进、贫血、发热、低血糖、嗜铬细胞瘤。
2. **心律失常**
   - 心动过速：窦性心动过速、室上性心动过速；
   - 心动过缓：高度房室传导阻滞、病态窦房结综合征；
   - 心律不齐：早搏、心房颤动。
3. **心力衰竭**：任何原因导致的心衰均可因代偿性心率增快引起心悸。
4. **心脏神经症**：无器质性病变，多见于青年女性，伴焦虑、失眠等神经衰弱症状。
5. **β受体亢进综合征**：紧张时发作，心电图可见ST-T改变，普萘洛尔试验可逆转。
6. **其他原因**：大量胸腔积液、高原病、胆心综合征、绝经综合征等。

### 第二步：分析发生机制
- **血流动力学改变**：如心室肥大→收缩力增强→搏动感明显；甲亢/贫血→心率加快→心悸。
- **心律失常机制**：
  - 心动过速：舒张期缩短→心室充盈减少→收缩期压力上升速率增快→心悸；
  - 心动过缓：舒张期延长→充盈增加→代偿性收缩增强→搏动感强；
  - 期前收缩：长间歇后强收缩+提前搏动似连续跳动→心悸或停跳感。
- **神经体液调节**：心衰时交感兴奋、肾素-血管紧张素系统激活→心率↑、收缩力↑→心悸。
- **神经精神因素**：焦虑、紧张时自主神经紊乱→即使心率正常也可感心悸。

### 第三步：关键诊断要点
- **询问诱因**：运动、情绪波动、咖啡/茶、药物、月经周期等；
- **评估伴随症状**：
  - 胸痛 → 警惕冠心病；
  - 发热 → 考虑心肌炎；
  - 消瘦、多汗 → 提示甲亢；
  - 晕厥 → 可能为严重心律失常。
- **必须进行心电图检查**；若怀疑β受体亢进综合征，可行普萘洛尔试验鉴别功能性ST-T改变。

## 输出建议
根据分类结果，建议进行以下检查并决定转诊方向：
- 心电图（必做）；
- 甲状腺功能、血常规（排查甲亢、贫血）；
- 若无器质性证据且伴明显焦虑，考虑转诊心理科；
- 若发现严重心律失常或结构性心脏病，立即转诊心内科。
