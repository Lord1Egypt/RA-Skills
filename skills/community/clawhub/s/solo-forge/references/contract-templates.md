# 合同场景模板

> ⚠️ 仅适用于中国大陆，不构成法律意见。重大金额建议咨询律师。

## 服务合同

```markdown
# 服务合同

**甲方**（委托方）：{公司/个人}
**乙方**（服务方）：{你的名字/公司}
**签订日期**：{日期} | **合同编号**：{编号}

## 服务内容
1. {服务内容}
交付物：| 序号 | 交付物 | 规格 | 交付时间 |

## 费用
总费用：**人民币 {金额} 元（大写：{大写}）**
付款：首付 **{比例}%** / 里程碑 **{比例}%** / 尾款 **{比例}%** | 收款：银行转账/支付宝/微信

## 周期与验收
周期：{开始日期}—{结束日期}，甲方原因延期→顺延
验收：交付后 **{X}** 工作日，逾期未异议→视为通过 | 变更：超出范围→书面确认+补充报价

## 权利保护
- 知识产权：付清全款后转甲方，尾款前乙方保留
- 质保：验收通过后 **{X}天** 内免费修复 bug（不含新需求），超出质保期按 **{金额}元/次** 或 **{比例}%** 合同额收取维护费
- 付款保护：逾期15天→暂停 / 逾期30天→终止，已收不退
- 违约：逾期付款/交付→每日 **0.5%**，累计不超过合同总金额的 **30%** / 不可抗力免责
- 解约：提前7天书面通知 / 甲方→已付不退+付已完成 / 乙方→退未完成+移交
- 争议：协商→乙方所在地法院

## 签署
| | 甲方 | 乙方 |
| 签名 | | |
| 日期 | | |
```

💡 生成合同/NDA/SOW 时，将模板中的压缩要点展开为带编号的完整条款（如"第六条 知识产权：..."），输出正式法律文书格式。

⚖️ 模板已包含30%上限。如用户口头约定更高的日违约金但未设上限，提醒：根据《民法典》第585条，过高违约金可被法院调减。

## NDA（双向保密）

```markdown
# 保密协议（NDA）

**甲方**：{公司/个人} | **乙方**：{你的名字/公司} | **日期**：{日期}

保密信息：商业计划/客户信息/定价策略/技术方案/源代码/设计文档/财务数据/营销策略/合作条款
双向义务：严格保密不向第三方 / 仅用于本合作 / 采取合理措施
例外：已公开(非因对方过错) / 合作前已掌握 / 从无义务第三方获取
归还与销毁：合作终止或对方书面要求后 **30** 日内归还或销毁全部保密资料及副本
不招揽：合作期间及终止后 **{X}年** 内，不主动招揽对方客户和员工
期限：有效 **{X}年**，终止后持续 **{X}年**
违约：赔偿一切直接和间接损失 / 紧急情况下可申请行为保全（禁令救济）

## 签署
| | 甲方 | 乙方 |
| 签名 | | |
| 日期 | | |
```

## SOW（项目确认书）

```markdown
# 项目确认书（SOW）| {项目名称}

**甲方**：{客户} | **乙方**：{你} | **关联合同**：{编号}

范围：✅ 包含：{列表} / ❌ 不包含：{列表}
交付物：| 序号 | 交付物 | 标准 | 日期 |
计划：| 阶段 | 内容 | 开始 | 结束 | 甲方配合 |
变更：范围内→免费 / 超出→书面确认+另行报价
验收：交付后 **{X}** 工作日，逾期→视为通过

## 签署
| | 甲方 | 乙方 |
| 签名 | | |
| 日期 | | |
```

💡 SOW 核心价值：锁定范围边界。没有SOW，"服务内容"太笼统→甲方无限加需求。
