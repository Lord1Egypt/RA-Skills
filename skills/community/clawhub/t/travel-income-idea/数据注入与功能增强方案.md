## ✈️ 新增功能：行程预订集成

### 功能需求
当用户生成执行计划后，在"行程准备"环节提供：
1. **机票航班智能推荐** - 根据目的地、时间推荐航班
2. **火车票智能推荐** - 根据目的地、时间推荐车次
3. **酒店智能推荐** - 根据目的地、预算推荐酒店
4. **一键预订入口** - 跳转到携程/飞猪/12306

### 实现方案

#### 方案 A：调用飞猪/携程 API（推荐）

**步骤 1：在 1D 后台添加新的 API 服务**

```
请为旅行创收助手添加行程预订功能。需要集成以下 API：

1. 航班查询 API（飞猪/携程）
   - 输入：出发城市、到达城市、出发日期
   - 输出：航班列表（航班号、起降时间、价格）

2. 火车查询 API（12306）
   - 输入：出发站、到达站、出发日期
   - 输出：车次列表（车次号、起降时间、票价）

3. 酒店查询 API（飞猪/携程）
   - 输入：城市、入住日期、离店日期、预算
   - 输出：酒店列表（名称、价格、评分、位置）

请创建新的服务文件 travelBookingService.ts，实现这些 API 调用。
```

**步骤 2：创建行程预订组件**

```
请创建一个新的行程预订组件 BookingPanel.tsx，包含：

1. 航班预订模块
   - 城市选择器（出发/到达）
   - 日期选择器
   - 航班列表展示
   - 跳转飞猪/携程预订链接

2. 火车预订模块
   - 站点选择器（出发/到达）
   - 日期选择器
   - 车次列表展示
   - 跳转 12306 预订链接

3. 酒店预订模块
   - 城市选择器
   - 入住/离店日期
   - 预算滑块
   - 酒店列表展示
   - 跳转飞猪/携程预订链接

请将此组件集成到 PlansPage.tsx 的执行计划页面中。
```
