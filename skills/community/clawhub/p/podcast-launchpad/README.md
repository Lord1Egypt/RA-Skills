# Podcast Launchpad

Podcast Launchpad is a document-only skill for turning a podcast idea into a launch-ready show package.

## Best For

- New creators planning a first show
- Teams formalizing an interview, solo, narrative, or hybrid format
- Creators who need a repeatable trailer-to-launch checklist

## Core Outputs

- Show concept, listener persona, positioning, and mission statement
- Format decision table and repeatable episode structure
- Equipment and recording setup matched to budget and production load
- Trailer outline, episode 1 outline, and two follow-up episode briefs
- Editing, loudness, show notes, and metadata checklist
- Directory submission tracker for Apple Podcasts, Spotify, YouTube Music, Amazon Music, Overcast, Pocket Casts, and audience-relevant platforms
- Launch QA checklist for RSS, artwork, metadata, playback, links, and consent

## Boundaries

This skill is educational. It does not guarantee audience growth, platform acceptance, rankings, sponsorships, or monetization. Platform rules change, so current submission requirements must be verified before claiming a show is listed or approved.
