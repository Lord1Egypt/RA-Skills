# iOS SDK 接口索引

## 初始化接口

| 接口 | 说明 |
|------|------|
| `+[GDTAction init:secretKey:]` | SDK 初始化，传入 `actionSetId` 和 `secretKey` |
| `+[GDTAction start]` | 启动 SDK（v2.1.0 及以上版本**必须调用**） |

## 启动上报接口

| 接口 | 说明 |
|------|------|
| `+[GDTAction logAction]` | 上报 START_APP 事件（常规启动），在 `applicationDidBecomeActive:` 中调用 |
| `+[GDTAction logAction:actionParam:]` | 上报 START_APP 事件（URL Scheme / Universal Links 唤起），需传入 `@"open_url"` 参数 |

## 预定义行为接口（GDTAction+convenience）

需引入 `GDTAction+convenience.h`。

| 接口 | 说明 |
|------|------|
| `+[GDTAction reportRegisterActionWithMethod:isSuccess:]` | 注册行为上报 |
| `+[GDTAction reportLoginActionWithMethod:isSuccess:]` | 登录行为上报 |
| `+[GDTAction reportBindSocialAccountActionWithType:isSuccess:]` | 绑定社交账号行为上报 |
| `+[GDTAction reportFinishQuestActionWithQuestID:questType:questName:questNumer:description:isSuccess:]` | 完成任务行为上报 |
| `+[GDTAction reportCreateRoleActionWithRole:]` | 创建角色行为上报 |
| `+[GDTAction reportUpgradeLevelActionWithLevel:]` | 等级提升行为上报 |
| `+[GDTAction reportViewContentActionWithContentType:contentName:contentID:]` | 浏览内容行为上报 |
| `+[GDTAction reportAddingToCartActionWithContentType:contentName:contentID:contentNumber:isSuccess:]` | 加入购物车行为上报 |
| `+[GDTAction reportCheckoutActionWithContentType:contentName:contentID:contentNumber:isVirtualCurrency:virtualCurrencyType:realCurrencyType:isSuccess:]` | 发起结算行为上报 |
| `+[GDTAction reportPurchaseActionWithContentType:contentName:contentID:contentNumber:paymentChannel:realCurrency:currencyAmount:isSuccess:]` | 付费行为上报 |
| `+[GDTAction reportAddPaymentChannelActionWithChannel:isSuccess:]` | 添加支付渠道行为上报 |
| `+[GDTAction reportRateActionWithRate:]` | 评分行为上报 |
| `+[GDTAction reportShareActionWithChannel:isSuccess:]` | 分享行为上报 |

## 通用上报接口

| 接口 | 说明 |
|------|------|
| `+[GDTAction logAction:actionParam:]` | 通用行为上报，支持自定义事件名和参数字典 |

## CAID 接口

| 接口 | 说明 |
|------|------|
| `+[GDTAction getCaid]` | 获取 CAID 信息，返回 NSDictionary（v2.0.11 及以上） |

## 自动采集事件

以下事件由 SDK 自动采集，无需手动调用：

| 事件 | 说明 |
|------|------|
| `TICKET` | 凭证事件 |
| `ENTER_FOREGROUND` | 应用进入前台 |
| `ENTER_BACKGROUND` | 应用进入后台 |

> **注意：** iOS 端 SDK v2.1.4+ 已默认自动采集 `START_APP`，无需手动上报。低版本（< v2.1.4）需开发者手动在 `applicationDidBecomeActive:` 中调用 `logAction:actionParam:` 上报。

## 特殊 actionParam 参数

| Key | 值 | 说明 |
|-----|-----|------|
| `audience_type` | `1` | 标记为回流用户 |
| `outer_action_id` | 自定义字符串 | 外部去重 ID |
| `length_of_stay` | `1` | 次留标记（用于次日留存统计） |
