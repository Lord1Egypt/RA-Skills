# OnboardAI — AI 新员工入职引导系统

> 一键式新员工入职管理，覆盖从 Offer 确认到 90 天转正评估的全链路。面向 HR、新人、Buddy 三角色。

## 适用场景

- 公司每月入职 10-20 人，HR 手忙脚乱：offer 确认没人跟、设备账号开通遗漏、新人第一天找不到工位
- 需要标准化入职流程，减少人工遗漏和重复沟通
- 新人自助查询常见问题（工位/设备/账号/食堂/培训/请假等）

## 6 大核心模块

| 模块 | 功能 | 核心用户 |
|------|------|----------|
| 📋 入职前准备 | 18 项自动化 checklist + 进度条 + 4 条自动化规则 | HR / IT |
| 🎯 首日引导 | 上午/下午分段时间线，步进式打卡 | 新人 + Buddy |
| 📌 首周看板 | 三列 Kanban（待开始/进行中/已完成），拖拽切换 | 新人 + Buddy |
| 🗺️ 成长路线 | 30-60-90 天三阶段里程碑，点击标记完成 | 新人 + Leader |
| 🤖 AI 问答 | 覆盖 10+ 高频场景智能问答 | 新人自助 |
| 📊 满意度追踪 | NPS 仪表盘 + 6 维热力图 + 逐条反馈列表 | HR / 管理层 |

## 使用方式

1. 加载技能后，Agent 会打开 `assets/onboarding-system.html` 交互式仪表盘
2. 所有数据存储在浏览器 localStorage 中，刷新不丢失
3. 点击「导出报告」生成 JSON 格式数据，可接入 BI 或汇报
4. 点击「新增员工」录入新入职人员，系统自动生成 checklist 和首日日程

## 技术特性

- **纯前端零依赖**：单个 HTML 文件，双击即用，无需服务器
- **localStorage 持久化**：数据存储在浏览器本地
- **响应式设计**：支持桌面和移动端
- **JSON 导出**：一键导出结构化数据报告

## 演示数据

系统预置了 5 名员工、18 项 checklist、10 项首日任务、7 项首周看板任务、完整的 30-60-90 天路线图和满意度样本数据，开箱即可体验完整流程。
