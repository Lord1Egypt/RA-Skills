# 一季度法则

> **核心**：不是回顾已披露的季度业绩，而是**预判未来一个季度（下一个季报窗口）的业绩是否会超预期**。提前布局"预期差"，在财报披露前基于产业数据、订单节奏、价格走势等先行指标，推算 upcoming quarter 的景气方向。

---

## 一、法则本质

### 1.1 什么是"一季度法则"

| 误区 | 正解 |
|------|------|
| ❌ 看已经披露的一季报/季报好不好 | ✅ **预判未来一个季度业绩会不会超预期** |
| ❌ 财报出来后的被动反应 | ✅ **财报披露前的主动推演** |
| ❌ 后视镜总结 | ✅ **望远镜预测** |

**核心问题**：
> "现在（当前时点），下一个季度（Q2/Q3/Q4/Q1）的业绩，市场一致预期是多少？真实业绩可能比这高还是低？"

### 1.2 为什么叫"一季度"法则

- "一季度"指的是**一个季度的时间跨度**，不是特指每年1-3月
- 在任意时点，往前看一个季度，判断业绩方向
- 法则精髓：**永远提前一个季度做预判，提前布局**

## 二、预判框架：先行指标 → 业绩推演

### 2.1 先行指标矩阵

| 指标类型 | 具体数据 | 数据来源 | 领先业绩多久 |
|----------|---------|---------|------------|
| **产品价格** | 月度价格走势、价差变化 | 生意社/卓创/百川 | 1-2个月 |
| **订单/排产** | 大客户框架协议、产能利用率 | 公司公告/调研纪要 | 1-3个月 |
| **上游数据** | 原材料采购量、供应商出货 | 产业链调研 | 1-2个月 |
| **下游验证** | 终端销量、渠道库存、客户提货 | 行业协会/海关数据 | 1-2个月 |
| **产能/良率** | 新产线投产、良率爬坡进度 | 公司公告/调研 | 2-3个月 |
| **政策/事件** | 补贴退坡、技术路线切换、大客户认证 | 新闻/公告 | 即时影响 |

### 2.2 业绩推演逻辑链

```
先行指标（当前可见）
    │
    ├── 产品涨价 + 订单饱满 → 营收↑ 毛利率↑ → 下季度业绩大概率超预期
    │
    ├── 产能释放 + 良率爬坡 → 出货量↑ 成本↓ → 下季度业绩大概率超预期
    │
    ├── 大客户导入 + 认证通过 → 新增收入确认 → 下季度业绩大概率超预期
    │
    ├── 价格下滑 + 库存累积 → 营收↓ 毛利率↓ → 下季度业绩可能不及预期
    │
    └── 竞争加剧 + 新进入者涌入 → 价格战 → 下季度业绩可能不及预期
```

## 三、一致预期 vs 真实业绩：寻找预期差

### 3.1 一致预期的来源

| 来源 | 特点 | 可靠性 |
|------|------|--------|
| 券商研报盈利预测 | 分析师覆盖度高的标的有共识 | ⚠️ 往往滞后 |
| Wind/Choice一致预期 | 汇总多家券商预测 | ⚠️ 更新不及时 |
| 市场隐含预期 | 股价走势反映的市场预期 | ⚠️ 需要解读 |
| 季度业绩指引 | 公司给出的正式指引（美股常见） | ✅ 最可靠 |

### 3.2 预期差的四种情形

| 情形 | 一致预期 | 真实业绩（预判） | 策略 |
|------|---------|----------------|------|
| **大超预期** | 增速30% | 实际可能60%+ | **提前布局，财报前买入** |
| **小超预期** | 增速30% | 实际可能40% | **持有，但不新增** |
| **不及预期** | 增速30% | 实际可能15% | **提前减仓，财报前卖出** |
| **暴雷** | 增速30% | 实际可能负增长 | **立即离场** |

### 3.3 预期差的量化标准

| 偏离度 | 定义 | 操作 |
|--------|------|------|
| > 50%超预期 | 真实业绩比一致预期高50%以上 | **强买入信号** |
| 20-50%超预期 | 真实业绩比一致预期高20-50% | **买入/持有** |
| ±20%以内 | 与一致预期基本一致 | **持有观察** |
| 不及预期20%+ | 真实业绩低于一致预期20%以上 | **减仓/卖出** |

## 四、实战推演：以光模块为例

### 4.1 2024年Q2预判（2024年4月时点）

**一致预期**：Q2营收环比+10-15%，同比增速+50%

**先行指标推演**：
- 800G光模块单价：3月较1月涨15% ✅
- 英伟达/谷歌订单：Q1末追加订单，排产至Q3 ✅
- 中际旭创产能利用率：95%+，扩产中 ✅
- 上游光芯片：供应紧张，涨价 ✅

**预判**：Q2营收环比可能+25-30%，同比+80-100%，**大超预期**

**操作**：4-5月布局，6月财报前持有

### 4.2 2025年Q4预判（2025年10月时点）

**一致预期**：Q4营收环比+20%，同比增速+60%

**先行指标推演**：
- 1.6T光模块：10月尚未大规模出货，仅小批量 ✅/⚠️
- 硅光方案：客户认证延迟 ⚠️
- 价格战：800G单价Q3环比下滑10% ⚠️
- 库存：部分客户库存水位上升 ⚠️

**预判**：Q4营收可能环比+10-15%，同比+40%，**不及预期**

**操作**：10-11月减仓，等财报验证

## 五、不同行业的先行指标

| 先行指标 | 数据来源 | 领先时间 |

| 下游开工率 | 产业链调研 | 1个月 |

| 上游芯片供应 | 供应商出货数据 | 1个月 |

## 六、推演模板

### 6.1 季度业绩预判表

| 项目 | 一致预期 | 先行指标显示 | 我的预判 | 偏离度 | 策略 |
|------|---------|-------------|---------|--------|------|
| 营收增速 | +30% | 订单饱满，涨价传导 | +50% | **+67%超预期** | **买入** |
| 毛利率 | 35% | 原材料降价，产品涨价 | 38% | **+3pct超预期** | **买入** |
| 净利润增速 | +40% | 营收↑+毛利率↑ | +70% | **+75%超预期** | **买入** |
| 经营现金流 | 正 | 回款良好 | 大幅正 | 超预期 | 验证真实 |

### 6.2 预判检查清单

每个季度末（财报披露前1个月），完成以下推演：

- [ ] 当前一致预期是多少？（券商预测汇总）
- [ ] 过去一个月产品价格涨跌？
- [ ] 订单/排产是否饱满？有无新增大客户？
- [ ] 产能利用率？新产线是否投产？
- [ ] 上游原材料供应/价格如何？
- [ ] 下游客户库存/提货节奏？
- [ ] 有无政策/技术路线变化？
- [ ] 竞争对手动态？价格战是否加剧？
- [ ] 我的预判 vs 一致预期的偏离度？
- [ ] 偏离度是否足以支撑买入/卖出决策？

## 七、与四浪理论的结合

### 7.1 预期浪的核心就是"一季度法则"

```
预期浪的本质：
    ├── 基于先行指标 → 预判下季度业绩超预期
    ├── 在财报披露前 → 提前布局
    ├── 财报披露时 → 业绩兑现，股价主升
    └── 等业绩浪 → 市场共识形成，估值抬升

所以"一季度法则"就是预期浪的操作手册：
    - 不是等业绩出来再买
    - 是提前一个季度推演业绩方向
    - 在"预期差"最大的时候介入
```

### 7.2 业绩浪也需要"一季度法则"

即使进入业绩浪，每季度也要预判：
- 下个季度增速是加速还是放缓？
- 如果加速 → 业绩浪延续，持有
- 如果放缓 → 业绩浪结束，考虑退出

## 八、经典案例

### 案例1：磷化工2021年Q3预判（2021年7月时点）

**一致预期**：Q3磷酸铁锂需求旺，但磷矿价格平稳

**先行指标**：
- 磷矿石：7月启动涨价，月环比+15% ✅
- 磷酸铁锂订单：宁德时代/比亚迪锁定产能 ✅
- 磷三杰产能：满产满销，新增产能Q4才释放 ✅

**预判**：Q3营收环比+30%，同比+150%，毛利率提升，**大超预期**

**操作**：7-8月买入磷三杰，Q3财报后主升浪

### 案例2：光模块2025年Q2预判（2025年4月时点）

**一致预期**：800G放量，Q2环比+20%

**先行指标**：
- 1.6T认证：4月英伟达/谷歌开始小批量 ⚠️
- 硅光方案：良率爬坡中，尚未大规模 ✅/⚠️
- 价格战：800G单价环比下滑5% ⚠️

**预判**：Q2环比+15%，同比+50%，**符合预期但不再超预期**

**操作**：持有但不新增仓位，等Q3 1.6T放量信号

## 九、禁忌清单

1. **不要等财报出来再反应**：等出来就不是"一季度法则"了
2. **不要只看历史业绩增速**：环比趋势比同比更重要
3. **不要忽略季节性**：有些行业Q2天然比Q1好，要调整预期
4. **不要过度乐观**：先行指标好不代表一定能超预期，要看一致预期已经 price-in 了多少
5. **不要只盯单一指标**：价格、订单、产能、客户要交叉验证

| 问题 | 答案位置 |
|------|---------|
| 如何预判下季度业绩？ | 先行指标矩阵（2.1节） |
| 一致预期从哪来？ | 券商研报/Wind/Choice（3.1节） |
| 偏离度多大才操作？ | >50%超预期=强买入（3.3节） |
| 各行业看什么先行指标？ | 行业先行指标对照表（5.1-5.3节） |
| 和四浪理论怎么配合？ | 预期浪=提前一个季度布局（7.1节） |

**生命法庭原话**：
> "预期浪才是最好的阶段，等业绩出来就晚了。"
> "炒股要炒高景气阶段的股，每个产业都有生命周期。"
