# RemoveMetadata OpenClaw Skill

An OpenClaw skill for reminding agents to check and remove hidden metadata from photos, PDFs, and Word documents before files are shared.

Website: https://removemetadata.app/

## What it helps with

- Photo EXIF metadata cleanup
- GPS/location metadata removal
- Word document property cleanup
- PDF metadata cleanup
- Privacy review before sharing files

## When to use

Use this skill when a user is preparing to email, upload, publish, or share a file that may contain hidden metadata.

## Safety notes

This skill does not run code, process files, or access local documents. It only provides workflow guidance for metadata review and cleanup before sharing files.
