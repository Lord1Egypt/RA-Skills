## Description: <br>
Guides agents in creating, optimizing, or auditing product features page content, including structure, benefit-led copy, feature lists, headline options, per-feature page guidance, and SEO metadata. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kostja94](https://clawhub.ai/user/kostja94) <br>

### License/Terms of Use: <br>

## Use Case: <br>
Developers, marketers, and content teams use this skill to guide agents in planning and drafting product features pages that communicate capabilities, benefits, proof, calls to action, and search metadata. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated or revised features page content could misstate product capabilities, differentiation, proof points, pricing relevance, or SEO intent. <br>
Mitigation: Review output against product source material, approved claims, and the local project context before publishing or using it in customer-facing pages. <br>
Risk: The skill can read local project context files when present, which may expose private positioning, product, or proof-point details to the agent session. <br>
Mitigation: Use it only in a trusted ClawHub development or maintainer environment and avoid adding sensitive material to project context files unless the session is authorized for that content. <br>
