# 性格模型 · 六型人格识人框架

> 四维评分给的是"数据"，六型人格给的是"画像"。
> 先画像 → 再验证 → 再决策。这是老板的识人认知入口。
>
> **⚡ 协作文件：** `four-dimension-model.md`（评分引擎）、`physiognomy-handbook.md`（信号采集）、`veto-checklist.md`（红线校验）、`scorecard-templates.md`（评分模板）

---

## 为什么需要这个模型

四维模型是精确的——四个维度每个打1-5分，加权计算，得到分数。

但老板在实战中不会说"这个人情绪稳定性4分、掌控感3分"。老板会说：

> "这个人什么路数？"

六型人格回答的就是这个问题。它不是替代四维模型，而是四维模型的**用户界面层**。

**工作流：** 见一个人 → 六型画像（秒级判断） → 四维打分（分钟级验证） → 一票否决扫描（最终校验） → 场景决策

---

## 六个类型的总览

| 类型 | 一句话 | 四维特征 | 最适合 | 最大风险 |
|---|---|---|---|---|
| **将帅型** | 松弛的掌控感 | 稳↑↑ 控↑ 诚↑ 共→ | CEO/合伙人/总经理 | 过度专断/金玉其外 |
| **先锋型** | 点燃全场的那个人 | 稳→ 控↑ 诚→ 共↑↑ | 销售/BD/0-1创业 | 过度承诺/可持续性差 |
| **谋士型** | 脑内的世界足够精彩 | 稳↑↑ 控↓ 诚↑↑ 共↓ | 技术/战略/风控/CFO | 不善管理/孤僻 |
| **基石型** | 你说什么他都能接住 | 稳↑↑ 控↓ 诚↑↑ 共↑ | 运营/HR/PM/执行层 | 缺乏突破力/被忽视 |
| **成长型** | 现在不一定最好，但在变好 | 四维在正向变化中 | 潜力股/接班人/转型期 | 成长不及预期/伪成长 |
| **跨界型** | 两种类型的复合体 | 双型特征兼有 | 复杂岗位/转型角色 | 内在冲突/角色混乱 |

# 第一篇：五种主型

## 类型一：将帅型 · The Commander

**"他不需要证明什么，所以看起来反而很放松。"**

### 四维特征

| 维度 | 水平 | 说明 |
|---|---|---|
| 情绪稳定性 | 极高（4-5） | 泰山崩于前面不改色——不是装的，是真的不怕 |
| 掌控感 | 高（4-5） | 有"松弛的掌控感"——想要的东西会拿，但不需要用姿势证明 |
| 诚实度 | 高-极高（4-5） | 对自己的失败坦诚，能从错误中学习 |
| 共情能力 | 中-高（3-4） | 能理解人，但不会被情绪绑架决策 |

### 面相特征

| 信号 | 特征 | 心理学机制 |
|---|---|---|
| 眼神 | 定而有穿透力，不过度施压也不回避 | 长期处于"知道自己在做什么"的状态 |
| 印堂 | 开阔，即使有悬针纹也是决策压力纹（直且浅） | 承担决策的生理成本，不是因为抱怨 |
| 坐姿 | 舒展但有框架——不是瘫，占的空间"刚好够" | 内化的自信，不需要用扩张来证明 |
| 嘴角 | 自然平直或微上扬 | 长期满足感，不讨好不抱怨 |
| 法令纹 | 向外上方延展（真笑纹） | 真心笑的时候多，不是社交假笑 |
| 常态表情 | 面无表情时也看起来平和 | 情绪基线稳定，不是"绷着" |
| 被质疑时 | 身体前倾、倾听、再回应——有好奇心 | 安全感够，不怕被挑战。核心区分点 |

### 面相对比：真将帅 vs 伪将帅（金玉其外型）

| 信号 | 真将帅 ✅ | 伪将帅 ❌ |
|---|---|---|
| 被质疑的第一反应 | 好奇、前倾、追问 | 防御、后靠、抱胸 |
| 谈失败 | 讲自己的责任部分+系统改进 | 全是别人的错+自己是受害者/英雄 |
| 倒序讲述 | 流畅，细节逻辑一致 | 卡壳、矛盾、烦躁 |
| 眼神 | 自然对视，不证明什么 | 刻意死盯（过度补偿）或飘忽 |
| 坐姿 | 有框架的舒展 | 过度的扩张（占领别人空间）或刻意的"松弛" |
| 对下位者 | 平视，不施压不轻视 | 颐指气使或表演性亲和 |

### 核心面试测试

1. **质疑测试：** "你刚才说的那个决策，如果换一种做法会不会更好？" → 真将帅会认真思考再回答，伪将帅会立刻防御
2. **倒序测试：** "从结果倒过来讲一遍那个项目" → 真将帅流畅，伪将帅卡壳
3. **沉默测试：** 他说完后沉默5秒 → 真将帅不会慌，伪将帅会忍不住补充

### 适合场景

- CEO/总裁/事业部总经理
- 需要独立决策+承担后果的岗位
- 转型期企业（需要有人稳住舵）

### 不适合场景

- 需要高频共情的岗位（一线客服/心理咨询）
- 需要完全服从的岗位（将帅型很难做纯执行者）
- 强矩阵组织中做"传话筒"角色（会憋死）

### 老板管理将帅型下属

- 给他目标，不要给方法。将帅型需要决策自主权
- 挑战他的想法可以，但要基于事实和逻辑，不要基于权力
- 最怕的上级：控制狂。将帅+控制狂=要么走人，要么暗中对抗
- 最好的搭档：一个能让他"不用管运营细节"的基石型COO

## 类型二：先锋型 · The Trailblazer

**"他到哪里，哪里的能量就起来。但他不一定是守城的那个人。"**

### 四维特征

| 维度 | 水平 | 说明 |
|---|---|---|
| 情绪稳定性 | 中-高（3-4） | 能扛压力但不是"不变色"——情绪写在脸上但不崩 |
| 掌控感 | 高（4-5） | 想要被看见、想要赢、享受征服感 |
| 诚实度 | 中（3-4） | 基本诚实，但会选择性呈现（"把最好的一面展示出来"） |
| 共情能力 | 极高（4-5） | 镜像神经元高度活跃，能快速建立人际连接 |

### 面相特征

| 信号 | 特征 | 心理学机制 |
|---|---|---|
| 眼神 | 有攻击性的温和——坚定但带笑意 | 高睾酮+高催产素的混合信号 |
| 变色龙效应 | 明显——会不自觉模仿你的姿势/语速 | 镜像神经元高度活跃，天生社交脑 |
| 手势 | 说话时手势多且自然、幅度大 | 思维活跃+表达欲强，不是表演型 |
| 鱼尾纹 | 丰富，真心笑的时候多 | 长期高频真笑留下的生理痕迹 |
| 法令纹 | 向外上扬（笑纹走向） | 社交中的"标准笑容"发自内心 |
| 眉毛 | 说话时眉飞色舞，眉毛动态丰富 | 情绪表达不压抑，透明度高 |
| 常态表情 | 活力充沛，有"精气神" | 情绪基线偏高，自带能量场 |

### 面相对比：真先锋 vs 嘴炮型

| 信号 | 真先锋 ✅ | 嘴炮型 ❌ |
|---|---|---|
| 谈失败 | "那次确实是我太乐观了，后来我学会了先做小规模验证" | "市场环境不好/团队没跟上/对手耍手段" |
| 追问细节 | 能说出具体数字和经过 | "大概是那个数"/"具体数字我回去查" |
| 被拒绝时 | 短暂挫败，迅速调整，找新的切入点 | 脸色瞬间垮掉/眼神变凶/放弃 |
| 眼神 | 有攻击性的温和 | 说话时死盯你（表演），不说时涣散（维持不住） |
| 手势 | 自然为表达服务 | 用动作"演"气势，手势和内容不同步 |
| 背调 | 前同事评价两极分化但总体正面 | 前同事评价高度一致（要么都喜欢=会演，要么都不喜欢=已穿帮） |

### 核心面试测试

1. **归因测试：** "你最大的失误是什么？" → 真先锋会承认自己"太乐观"或"太快"，嘴炮会归因于外部
2. **被拒模拟：** 故意否定他的一个观点 → 真先锋会找新角度说服你，嘴炮会变防御或放弃
3. **极限承诺测试：** 给一个明显不可能的目标 → 真先锋会说"这个有挑战，我需要XX资源才有可能"，嘴炮会说"没问题"

### 适合场景

- 销售总监/BD/市场/品牌
- 0-1创业阶段（需要有人引爆市场）
- 对外谈判/商务拓展
- 一切需要"打开局面"的岗位

### 不适合场景

- 需要长期深耕同一件事（先锋型会无聊）
- 需要高度精确的工作（财务/审计）
- 守成期（先锋会在稳定运营中失去能量）

### 老板管理先锋型下属

- 给他舞台和认可。先锋型的核心燃料是"被看见"
- 设定结果底线但不要控制过程。说"你需要做到X"，不要说"你应该这样做"
- 给他配一个基石型的搭档——先锋签了单，基石接住执行
- 警惕他的过度承诺。每月复盘一次"答应的事做到了多少"
- 最怕的上级：不给认可只给批评。先锋没了能量就废了

## 类型三：谋士型 · The Strategist

**"脑内的世界足够精彩，不需要用外在证明自己。"**

### 四维特征

| 维度 | 水平 | 说明 |
|---|---|---|
| 情绪稳定性 | 极高（4-5） | 情绪不写在脸上——不是因为压抑，是因为脑内正在深度加工 |
| 掌控感 | 中低（2-3） | 对人际空间没有争夺欲望，争夺发生在智力领域 |
| 诚实度 | 极高（4-5） | 说实话是最省认知资源的策略——不说实话太累 |
| 共情能力 | 中低（2-3） | 能理解但不想处理——"我知道你不开心，但事实就是这样" |

### 面相特征

| 信号 | 特征 | 心理学机制 |
|---|---|---|
| 眼神 | 有"内视"习惯——思考时往侧面/上方看 | 深度提取记忆或构建模型，不是回避社交 |
| 社交对视 | 偏少（40-50%），但思考时眼神聚焦 | 内向型大脑，社交对视消耗认知资源 |
| 说话节奏 | 偏慢，有明显停顿，但逻辑极清晰 | 说之前已在大脑里完整跑了一遍 |
| 坐姿 | "刚好够用"——不刻意舒展也不缩 | 对空间无争夺欲，自我在脑内完成 |
| 表情幅度 | 小，常态偏"严肃"或"无表情" | 情绪不写在脸上，不是因为不开心 |
| 手势 | 少但精确——偶尔有刀刃般精准的手势 | 肢体为思维服务，不为社交服务 |
| 幽默 | 冷幽默/反讽——笑点高但命中率也高 | 在脑内完成了幽默加工，不需要用笑声证明 |

### 面相对比：真谋士 vs 阴水型（算计型）

| 信号 | 真谋士 ✅ | 阴水型 ❌ |
|---|---|---|
| 被质疑时 | 眼神"让我想想"（思考启动），沉默或慢速回应 | 眼神变冷/变凶（攻击性压抑），或假意附和但嘴角紧绷 |
| 谈同事 | 客观评价优缺点，有具体数据支撑 | "他们不行"但说不出具体原因，或过分夸赞（掩饰真实想法） |
| 归因模式 | "当时我漏掉了XX因素，是我的分析不够全面" | "信息不够/别人没告诉我/情况特殊" |
| 眼神 | 内视型+思考时聚焦 | 眼睛频繁扫视（在衡量利害关系），或刻意回避关键话题 |
| 第6感 | 你感觉他在思考，不是在算计 | 你感觉他在"观察"你——那种审视让人不舒服 |

### 核心面试测试

1. **深度题+沉默观察：** 给一个需要深度思考的问题，然后闭嘴等。观察他思考的过程（看哪里、多久、表情变化）比看他的答案更有信息量
2. **写得怎么样：** 让他在面试后提交一份书面分析。谋士型写的往往比说的好10倍
3. **社交需求测试：** "你更喜欢一个人攻克难题，还是和团队一起？" → 谋士会诚实地说前者（不会伪装社交偏好）

### 适合场景

- CTO/首席科学家/技术VP
- CFO/风控/合规/审计
- 战略顾问/行业分析师
- 一切需要"深度思考+精准判断"的岗位

### 不适合场景

- 一线带大团队（谋士会累死自己，下属也会觉得被冷落）
- 销售/BD/对外公关（社交消耗太大，续航力不足）
- 需要"鼓舞士气"的领导者角色

### 老板管理谋士型下属

- 给他难题，不要给琐事。谋士的核心燃料是"被问题吸引"
- 沟通方式：写>说。重要的事发文字，不要只口头交代
- 不要逼他社交。团建让他坐在角落里比强迫他活跃更让他舒服
- 给他独立空间。开放式办公室是谋士的噩梦
- 最怕的上级：每天追问"你在想什么"——谋士需要时间完成脑内闭环
- **关键：** 谋士的判断通常是对的，但不一定"好听"。学会听难听话，是老板用好谋士的前提

## 类型四：基石型 · The Cornerstone

**"你说什么他都能接住，然后默默地就把它做了。"**

### 四维特征

| 维度 | 水平 | 说明 |
|---|---|---|
| 情绪稳定性 | 极高（4-5） | 不爆发也不压抑——情绪像流水，过去就过去了 |
| 掌控感 | 中低（2-3） | 不需要掌控全局，但掌控"自己那一亩三分地" |
| 诚实度 | 极高（4-5） | "说到做到"带来的长期一致性——不需要撒谎 |
| 共情能力 | 高（4-5） | 能接住别人的情绪，但不会被带着走——有"边界感的共情" |

### 面相特征

| 信号 | 特征 | 心理学机制 |
|---|---|---|
| 眼神 | 柔和稳定，不犀利但也不涣散 | 长期安于自己所处的位置，不焦虑不退缩 |
| 常态表情 | 平和，没有大起大落 | 情绪基线稳定，不需要通过大表情调节情绪 |
| 说话 | 不快，但每句话都落地 | "说对了"比"说得多"重要 |
| 嘴角 | 自然平直或微上扬 | 长期满足感——"不争"带来的松弛 |
| 坐姿 | "刚好"——不抢也不缩 | 领地行为和自我认同精准匹配 |
| 镜像反应 | 有但不过度——会共情但不被带跑 | 有边界感的共情，情绪不卷入 |
| 对细节 | 对流程/时间/数字敏感 | 基石的大脑在高频扫描"会出什么问题" |

### 面相对比：真基石 vs 泥型（停滞型）

| 信号 | 真基石 ✅ | 泥型 ❌ |
|---|---|---|
| 承诺后 | 承诺前会认真评估，承诺后100%做到 | "好的"挂嘴上但不了了之 |
| 眼神 | 柔和稳定 | 呆滞/无光/缺乏活力 |
| 给挑战时 | 先评估再承诺——"我需要XX天" | 直接答应但不做，或直接拒绝挑战 |
| 变化 | 面相在微变好（稳中有升） | 面相无变化或微变差 |
| 主动性 | 在自己的领域内会主动发现问题 | 别人推一下才动一下 |

### 核心面试测试

1. **等待测试：** "你等我一会，有个临时电话" → 真基石会安静等，泥型会玩手机/发呆
2. **细节追问：** "你上次负责的那个项目，预算是多少？实际花了多少？差在哪？" → 真基石能准确回答，泥型模糊
3. **服务员测试：** 用餐/咖啡时间观察他对服务员的态度 → 真基石自然礼貌，泥型可能无视或挑剔

### 适合场景

- COO/运营总监/行政总监
- HR负责人/HRBP
- 项目经理/PMO
- 财务中层/采购/供应链管理
- 一切需要"长期靠谱"而不是"短期亮眼"的岗位

### 不适合场景

- 从0到1的创新（基石需要已有的框架才能发挥）
- 高度不确定性的岗位（基石需要"确定自己能搞定"才能承诺）
- 需要"画大饼"的融资/PR角色

### 老板管理基石型下属

- **面试时最容易漏掉他。** 基石不炫，不会在面试中发光。强制规则：终面名单至少包含一个"挑不出毛病但说不出哪里特别亮眼"的人
- 给他清晰的目标和边界。基石在明确的框架里效率最高
- 认可他的方式是**在众人面前说"交给他我从不担心"**——基石不追求聚光灯，但需要被信任
- 不要频繁改变优先级。基石最痛苦的是"昨天让我做A，今天让我做B"
- 最怕的上级：朝令夕改的老板。基石会因此失去"确定感"这个核心燃料

## 类型五：成长型 · The Grower

**"他现在不一定是最好的，但他在变好——而且变好的速度很快。"**

### 核心定义

**成长型和前四型不在一个分类轴上。** 前四型回答"你现在是什么"，成长型回答"你在往什么方向变"。

成长型是"善能改相"的活标本——四维分数可能不高，但变化斜率是正的。

### 面相变化信号（判断成长型的关键）

| 变化信号 | 之前的状态 | 现在的状态 | 说明 |
|---|---|---|---|
| 眉头 | 习惯性紧锁 | 逐渐舒展 | 不钻牛角尖了，焦虑水平在降 |
| 眼神 | 飘忽/回避/涣散 | 逐渐稳定/聚焦 | 自信心在建立，找到了方向 |
| 嘴角 | 习惯性下垂 | 逐渐平直或上扬 | 停止了抱怨，心态在转向 |
| 姿态 | 蜷缩/收缩 | 逐渐舒展 | 安全感和自我认同在提升 |
| 面色 | 暗沉/灰暗 | 逐渐清亮 | 皮质醇在降，生活节律在改善 |
| 被质疑 | 防御/冻结 | 开始能倾听 | 不再在"被否定=我不行"的循环里 |

**关键判断标准：** 上述信号中至少3个同时出现正向变化，且持续3个月以上。

### 面相对比：真成长 vs 伪成长（新环境红利型）

| 信号 | 真成长 ✅ | 伪成长 ❌ |
|---|---|---|
| 第1-3个月 | 面相改善，但伴随"努力"的痕迹（比如同时有疲惫感） | 面相改善，因为新环境的新鲜刺激 |
| 第3-6个月 | 持续改善，且改善在加速 | 开始回落，回到基线 |
| 遇到挫折后 | 短暂回落但迅速反弹 | 回落且不反弹——退回原来的模式 |
| 独处时的状态 | 依然在变好（变化是内化的，不是给别人看的） | 有人看时好，没人看时回到老样子 |
| 改善的驱动 | 内在驱动（"我想变好"） | 外部驱动（"新老板要求高"） |

### 最值得投资的成长型信号

1. **"苦相在消退"：** 之前眉头紧锁+嘴角下垂+眼神飘，现在三个信号都在消退 → **一级投资标的**
2. **"眼神在定"：** 之前眼神飘/涣散，现在越来越有聚焦 → 找到了方向，或建立起了自信
3. **"姿态在开"：** 之前蜷缩，现在越来越舒展 → 安全感在建立，自我认同在提升
4. **"笑在变真"：** 之前只有社交笑容（不动眼睛），现在眼轮匝肌参与增加 → 真的在变开心

### 适合场景

- 潜力股员工（目前绩效一般但面相在改善→值得投资）
- 接班人培养（面相在变好=方向对）
- 转型期高管（从技术转管理、从IC转带人）
- 年轻人才的早期识别（25岁以下，面相变化斜率比绝对分数更重要）

### 不适合场景

- 需要"立即可用"的关键岗位（成长型需要时间）
- 危机时刻的救火队长（还没成长到能扛）

### 老板投资成长型的三个原则

1. **土壤决定上限。** 成长型在最对的环境里能长成任何类型。放到不好的环境（坏老板/烂团队），"善能改相"会被逆转。你不仅是他的老板，你是他的土壤。
2. **第3-6个月是关键窗口。** 新环境的兴奋感在第3个月消退，第3-6个月还在变好的=真成长。第3-6个月停滞或回落的=伪成长。
3. **投资忠诚度ROI极高。** 一个在"善能改相"过程中被你信任和投资的人，对这段关系的忠诚度远超市场平均水平。不是因为感恩，是因为他的成长和你绑定在一起。

# 第二篇：跨界型

---

## 跨界型 · The Hybrid

**"他不是单一类型，而是两种类型的复合体。这种复合本身就是一种稀缺资产。"**

### 跨界型不是"判断不确定"

判断一个人是跨界型之前，先确认你不是因为信息不足才无法归类。

**跨界型的判断标准：**
- 在两个类型上有**同等显著**的信号集合（各至少3个独立信号）
- 这两个类型的特征在不同**场景下稳定切换**（不是同一场景下随机摇摆）
- 两个类型的核心资产他都具备，核心风险他也都可能触发

### 六种核心跨界组合

#### 将帅×先锋（The Warlord）

**特征：** 既有将帅的决策力+担当，又有先锋的感染力+拓展力。

| 信号 | 表现 |
|---|---|
| 将帅面 | 眼神定、印堂开阔、被质疑时前倾 |
| 先锋面 | 变色龙效应强、鱼尾纹丰富、说话眉飞色舞 |

**核心资产：** 能定战略又能带团队冲锋——最稀缺的复合型，天生的CEO。

**核心风险：** 过于自信+过度承诺的双重叠加。将帅的"我能搞定"×先锋的"没问题"=可能在极端乐观中翻车。

**管理要点：** 必须给他配一个谋士型或基石型的副手——一个敢说"老板你这不对"的人。

---

#### 将帅×谋士（The Philosopher King）

**特征：** 既有将帅的担当+掌控感，又有谋士的深度思考+精准判断。

| 信号 | 表现 |
|---|---|
| 将帅面 | 姿态舒展、被质疑时开放、自然对视 |
| 谋士面 | 说话有停顿、思考时内视、表情幅度小 |

**核心资产：** 既能深度思考又能拍板执行——最适合技术驱动型公司的CEO，或对冲基金的PM。

**核心风险：** 过度思考+行动犹豫。谋士的"再想想"和将帅的"现在就做"在脑内打架，决策速度可能反而变慢。另一个风险：社交续航不足——外表是将帅，内里是谋士，带大团队开会可能很消耗。

**管理要点：** 给他独处思考的时间，不要连续塞满他的日程。他的决策质量依赖于独处时间。

---

#### 先锋×基石（The Anchor）

**特征：** 既有先锋的感染力，又有基石的靠谱和落地能力。

| 信号 | 表现 |
|---|---|
| 先锋面 | 变色龙效应、眼神带笑意、手势丰富 |
| 基石面 | 对细节敏感、说话落地、眼神柔和稳定 |

**核心资产：** 能开拓又能守住——最适合销售运营/客户成功/区域总经理。先锋签单，基石接住并深耕。

**核心风险：** 内在矛盾——先锋想要新市场新客户，基石想要把现有的做到极致。可能会在"拓展"和"深耕"之间反复拉扯。另一个风险：过度给予——先锋的慷慨×基石的负责=可能被客户/团队过度消耗。

**管理要点：** 帮他明确"当前阶段是拓展还是深耕"，避免两个方向互相拉扯。他自己不一定能意识到这个矛盾。

---

#### 谋士×基石（The Architect）

**特征：** 既有谋士的深度思考，又有基石的执行落地能力。

| 信号 | 表现 |
|---|---|
| 谋士面 | 内视型眼神、逻辑清晰、表情严肃 |
| 基石面 | 对流程敏感、眼神柔和稳定、承诺后100%执行 |

**核心资产：** 能设计系统又能把系统跑起来——最适合COO/技术VP/首席架构师/风控总监。想得清楚+做得踏实。

**核心风险：** 过于保守。谋士的"风险尚未穷尽"×基石的"不确定的不承诺"=可能错失需要冒险的机会。另一个风险：缺乏外显领导魅力——团队会尊重他但不会被他"点燃"。

**管理要点：** 他是团队里最不需要你操心的人。你需要操心的是：他有没有因为"太稳"而错过了该抓住的机会？定期和他探讨"如果我们不做会怎样"。

---

#### 先锋×成长（The Rising Star）

**特征：** 先锋的底色+正在快速成长的过程中。

| 信号 | 表现 |
|---|---|
| 先锋面 | 感染力强、共情高、镜像反应活跃 |
| 成长面 | 面相在持续改善（眉头在舒展/眼神在定/姿态在开） |

**核心资产：** 有天赋+在加速成长——最值得投资的年轻人才。给他时间，他可能长成将帅×先锋。

**核心风险：** 成长的波动性——先锋型本身就有情绪起伏，叠加成长过程中的不确定性，状态可能大起大落。另一个风险：过度自信——先锋的"我能行"+成长型的"我在变好"=可能高估自己的成长速度。

**管理要点：** 给他挑战但要设置安全边界。让他摔跤但不要让他摔到骨折。每月一次的1-on-1不能省——你需要持续追踪他的"成长斜率"有没有变平。

---

#### 基石×成长（The Late Bloomer）

**特征：** 基石的底色+在缓慢但稳定地向更好的方向变化。

| 信号 | 表现 |
|---|---|
| 基石面 | 靠谱、稳定、说到做到 |
| 成长面 | 面相在微改善（可能很慢，但方向对） |

**核心资产：** 稳中有升——可能不会给你惊喜，但绝对不会给你惊吓。长期持有的价值投资标的。

**核心风险：** 成长速度太慢，可能被组织成长速度甩在后面。或者因为"太稳"而被忽视，没有被给予应有的成长机会。

**管理要点：** 不要拿他和先锋×成长型比速度——他的成长是马拉松不是百米。关注的是"方向对不对"而不是"快不快"。在组织快速扩张时，警惕他被边缘化。

---

### 跨界型判断流程

```
观察到多个信号 → 先尝试匹配单一类型 → 能匹配吗？
                                    ↓
                        ┌─ 能 ──→ 就是该单一类型
                        │
                        └─ 不能 ──→ 观察信号是否来自两个不同类型
                                        ↓
                                ┌─ 是，且在不同场景下稳定 → 跨界型
                                │
                                └─ 信号混乱/同一场景下摇摆 → 信息不足，继续观察
```

# 第三篇：团队生克关系

---

## 相生关系（互补增强 · 团队搭配指南）

**相生 = 类型A能滋养、补充、增强类型B。**

### 将帅 → 谋士（金生水）

**机制：** 将帅提供方向和"保护伞"，让谋士可以安心做深度思考，不用消耗认知资源处理政治和组织噪音。

**为什么有效：** 谋士最需要的是一个"不怕听难听话"的决策者。将帅不怕被挑战，所以谋士可以放心说出"你想错了"。谋士最怕的是"要面子"的上级——说了实话会被穿小鞋。

**最佳搭配场景：** CEO（将帅）+ CTO/CFO（谋士）

**警惕：** 如果将帅实际上是个伪将帅（金玉其外型），他被谋士挑战后会防御/记恨/架空谋士。这是很多技术合伙人被踢出局的底层逻辑。

---

### 谋士 → 成长型（水生木）

**机制：** 谋士的深度思考和方法论，能滋养成长型人才的认知升级。谋士不抢功不抢风头，恰好给了成长型舒展的空间。

**为什么有效：** 成长型正处于"在变好"的过程中，最怕的是被过度施压或过度比较。谋士天然不是施压型——他关注的是"问题本身对不对"，不是"你行不行"。

**最佳搭配场景：** 资深技术（谋士）带新人（成长型）；战略顾问（谋士）陪跑年轻CEO（成长型）

**警惕：** 如果谋士的耐心不够，或者成长型需要的情感支持超出了谋士能给的（谋士共情偏低），成长型可能感觉"被冷落"。

---

### 成长型 → 先锋型（木生火）

**机制：** 从内部培养起来的成长型人才，其成长动力和归属感能转化为先锋型的"燃烧燃料"。内部提拔的先锋比外部空降的先锋更可持续——因为他知道自己被投资过。

**为什么有效：** 成长型在"善能改相"的过程中积累了真实的成长经历——这些经历会转化成先锋型面对客户/市场时的底气和感染力。他不是在"演自信"，是真的经历过从不行到行的过程。

**最佳搭配场景：** 内部提拔的销售总监（成长→先锋）；创始人从技术转型商务（成长→先锋）

**警惕：** 成长→先锋的转变如果太快（拔苗助长），成长型可能在还没准备好的时候就冲出去了。需要一个将帅型老板在他飞太高的时候把他拉回来。

---

### 先锋型 → 基石型（火生土）

**机制：** 先锋打开局面后，基石稳稳接住，把"一锤子买卖"变成"长期关系"。先锋带来客户，基石深耕客户。

**为什么有效：** 先锋的弱项是"可持续性"（签了单但懒得跟进细节）。基石的天赋恰恰是"持续交付"。先锋×基石的组合，能让客户体验从"签单前很爽"延续到"签单后也很放心"。

**最佳搭配场景：** 销售总监（先锋）+ 客户成功/运营总监（基石）

**警惕：** 先锋对接客户时的"过度承诺"会变成基石的"交付灾难"。需要建立机制：先锋做承诺前要经过基石评估可行性。

---

### 基石型 → 将帅型（土生金）

**机制：** 稳固的运营体系支撑将帅敢于做出更大胆的决策。将帅知道"后面有人兜底"，所以敢于冒险。

**为什么有效：** 将帅最大的限制不是决策能力，是决策后的执行焦虑——"我决定了但如果执行崩了怎么办"。基石型的存在消解了这种焦虑，从而释放了将帅的决策勇气。

**最佳搭配场景：** COO（基石）+ CEO（将帅）

**警惕：** 基石也可能过度保守，让将帅觉得"你这个不支持那个不支持"。需要将帅理解：基石的谨慎不是针对他，是基石的本能。沟通方式需要调整。

---

## 相克关系（冲突预警 · 团队避坑指南）

**相克 = 类型A和类型B天然存在张力，需要管理，否则可能互相消耗。**

### 将帅 克 成长型（金克木）

**冲突机制：** 将帅的"高标准+直接反馈"可能压垮还在成长中的人。将帅觉得"我给你说了问题在哪，你自己去解决"，成长型听到的是"他说我不行"。

**典型场景：** 一个强CEO（将帅）带一个潜力股（成长型）。CEO直接指出问题，成长型感受到的是否定而非指导。

**管理方法：**
- 将帅对成长型，需要多一步："我说的是这件事的问题，不是你人的问题"
- 给成长配一个中间层（谋士或基石），缓冲将帅的直接冲击
- 不要因为将帅觉得"我当年也是这么过来的"就默认成长型也能这样

---

### 先锋型 克 将帅型（火克金）

**冲突机制：** 先锋的过度承诺和乐观预期，不断倒逼将帅去收拾残局。将帅的耐心和信任会被消耗——"你又答应了一个你做不到的事？"

**典型场景：** 强销售VP（先锋）对CEO（将帅）说"这个客户没问题"，然后出了问题CEO去道歉。反复几次后，CEO会开始不信任销售VP的判断。

**管理方法：**
- 建立"承诺-兑现率"追踪——不是看先锋说了什么，是看兑现了多少
- 先锋做外部承诺前，必须经过内部校验（"给我12小时确认"）
- 将帅对先锋的反馈要基于数据，不要基于情绪

---

### 谋士型 克 先锋型（水克火）

**冲突机制：** 谋士的"这个不行，数据不支持"会浇灭先锋的战斗热情。先锋需要的是"我们冲！"，谋士给的是"建议先做A/B测试，控制变量，样本量不够的话再扩大……"。

**典型场景：** CTO（谋士）对销售VP（先锋）说"这个功能技术上至少3个月"。先锋听到的是"不行"，实际上是"需要3个月"。

**管理方法：**
- 谋士对先锋说话需要多加一句："不是不行，是需要XX条件"
- 不要让他们两直接对接——中间需要一个翻译者（基石型最合适）
- 先锋需要理解：谋士说的"风险"不是拒绝，是他思考问题的默认模式

---

### 基石型 克 谋士型（土克水）

**冲突机制：** 基石型对"确定性"的需求，和谋士型对"可能性"的探索，天然矛盾。基石问"什么时候能交付"，谋士说"取决于我发现了什么"。

**典型场景：** 一个强运营（基石）追着一个强技术（谋士）要排期。基石想锁定时间，谋士觉得"需求还在变，锁定没意义"。双方都觉得对方不职业。

**管理方法：**
- 给谋士一个"安全缓冲区"——比如排期=他的估计×1.5
- 给基石一个"不要锁定太死"的授权——让基石接受"分阶段承诺"
- 或者：不给谋士排期压力，让另一个谋士去评估他的进度

---

### 成长型 克 基石型（木克土）

**冲突机制：** 成长型的变化和不确定性，让基石型感到不安。基石习惯了"可预测"，而成长型的本质是"在变化中"。

**典型场景：** 一个快速成长的年轻员工（成长型）和一个资深运营经理（基石型）合作。年轻员工这周说A方案，下周说B方案更好。基石内心："你能不能先想清楚？"

**管理方法：**
- 让成长型理解：对基石来说，"说到做到"比"做对"更重要
- 给成长型配一个缓冲角色——不是他直接对接基石，而是通过一个中间人承诺稳定版本
- 基石需要被提醒：成长型的"变"不是不靠谱，是他的核心竞争力

---

## 团队组合原则

### 三原则

1. **不要全是一种类型。** 全将帅的公司每天都在打仗但没人守城；全基石的公司稳如泰山但永远不会爆发。
2. **CEO类型决定公司气质。** 将帅CEO=决策驱动；先锋CEO=市场驱动；谋士CEO=产品驱动；基石CEO=运营驱动。你要什么气质的公司，就选什么类型的CEO。
3. **必有一生一克。** 核心团队里至少有一组相生（给你能量）和一组相克（给你约束）。只有相生没有相克=回音壁效应，一群人在互相确认偏见。只有相克没有相生=每天都在内耗。

### 四角色最小可行团队

任何创业/项目的最小核心团队，建议覆盖这四个角色：

| 角色 | 最佳类型 | 变通类型 | 缺失的后果 |
|---|---|---|---|
| **拍板的** | 将帅型 | 将帅×谋士 | 没人做决策，或决策了没人担 |
| **搞钱的** | 先锋型 | 先锋×基石 | 产品很好但没人知道 |
| **做出来的** | 谋士型 | 谋士×基石 | 想法停留在想法 |
| **守住的** | 基石型 | 基石×成长 | 交付质量不可控 |

**一个人可以覆盖两个角色（跨界型的价值就在这），但不能覆盖全部四个。** 四角色全覆盖=一个人说了算=没有真正的团队。

# 第四篇：完整识人流程

## 从"见到一个人"到"做出决策"的五步法

```
Step 1: 六型画像（秒级）
    ↓ 先判断这个人"什么路数"
Step 2: 四维打分（分钟级）
    ↓ 用四个维度精确验证画像
Step 3: 面相交叉验证（分钟级）
    ↓ 至少3个独立面相信号指向同一结论
Step 4: 一票否决扫描
    ↓ 有没有触发致命/高危红线
Step 5: 场景决策
    ↓ 在这个具体场景中，用还是不用？
```

### Step 1: 六型画像（30秒-3分钟）

**目标：** 形成第一假设。

**观察重点：**
- 眼神（定/飘/内视/带笑意）
- 姿态（舒展/刚好/缩/瘫）
- 被质疑后的第一反应（好奇/防御/沉默思考/紧张）

**输出：** "这个人看起来像XX型"

### Step 2: 四维打分（3-15分钟深度交流）

**目标：** 用结构化评分验证画像假设。

**如果四维分数和六型画像矛盾：**
- 四维说"情绪稳定性2分"，但画像认为是"将帅型"→画像错了（将帅需要稳↑↑），重新判断
- 四维和画像一致→进入下一步

### Step 3: 面相交叉验证

**目标：** 用面相信号做独立验证。

**规则：** 该类型的核心面相信号中，至少3个确认存在。如果不足3个→信号不够，暂不下结论。

### Step 4: 一票否决扫描

**目标：** 检查是否有致命/高危红线。

**规则：** 致命级出现1个→不管画像多好，pass。高危级出现2个以上→暂停，收集更多信息。

### Step 5: 场景决策

**目标：** 基于画像+评分+红线，做出具体决策。

**决策维度：**
- 这个类型适合这个岗位吗？
- 团队现有类型组合中，加上他会产生怎样的生克关系？
- 如果录用/合作，管理要点是什么？

# 附录：类型快速判断问题集

## 3个问题快速定位主型

**Q1: 被当众质疑时，你的第一反应是什么？**
- "我想知道他为什么这么想" → 将帅/谋士
- "我会立刻回应，说服他" → 先锋
- "我会先消化一下，想想他说的对不对" → 基石/成长

**Q2: 一个人攻克难题 vs 带领团队达成目标，你更享受哪一个？**
- "攻克难题" → 谋士
- "带领团队" → 将帅/先锋
- "把分配给我的事情做到极致" → 基石
- "我正在学习如何……" → 成长

**Q3: 你最希望别人怎么评价你？**
- "他是一个能扛事的人" → 将帅
- "有他在就有办法" → 先锋
- "他说的话不会错" → 谋士
- "交给他，我放心" → 基石
- "他最近进步很大" → 成长
