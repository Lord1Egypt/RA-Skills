## Description: <br>
Applies an iterative Ralph loop workflow to help an agent define completion criteria, test work, improve outputs, and report final status for complex tasks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[cloudyli](https://clawhub.ai/user/cloudyli) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, skill authors, and agent users use this skill to structure complex work into measurable deliverables, validation steps, bounded iteration, and a final completion report. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Validation commands produced during use may target unintended files or modify important data. <br>
Mitigation: Review each command before approval and confirm that it targets the intended files and that modification is part of the task. <br>
Risk: The iterative workflow can add process overhead when a task is simple or lacks measurable completion criteria. <br>
Mitigation: Use the skill for complex work with clear deliverables, validation methods, and a bounded iteration limit. <br>
