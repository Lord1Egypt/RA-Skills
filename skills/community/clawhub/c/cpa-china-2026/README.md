# CPA China 2026 — 中国注册会计师考试专家

基于2026年中国注册会计师全国统一考试辅导教材（中国注册会计师协会 组织编写），涵盖专业阶段全部6科知识体系。

## 功能

- 📊 **会计** — 30章，从总论到政府及民间非营利组织会计
- 🔍 **审计** — 审计全流程，从风险评估到审计报告
- 💰 **财务成本管理** — 从货币时间价值到业绩评价
- 🏢 **公司战略与风险管理** — 从战略分析到内部控制
- ⚖️ **经济法** — 从民事法到反垄断法
- 🧾 **税法** — 从增值税到18个税种全面覆盖

## 安装

```bash
skillhub install cpa-china-2026
```

## 使用

直接提问CPA相关科目问题即可，无需特殊配置。
