# Qing Re Herb Classification and Contraindications

## When to Use
- 患者存在里热证候，如高热、口渴、痈肿、痢疾、骨蒸潮热等。

## Execution Steps
1. **辨识热证类型**：
   - 气分实热（高热、汗出、脉洪大）→ 选用清热泻火药
   - 热毒痈疮、瘟毒发斑 → 选用清热解毒药
   - 营血分热（身热夜甚、吐衄）→ 选用清热凉血药
   - 湿热内蕴（泻痢、黄疸、带下）→ 选用清热燥湿药
   - 暑热证（发热烦渴、吐泻）→ 选用清热解暑药
   - 目赤肿痛因肝火或风热 → 选用清热明目药
   - 阴虚内热（骨蒸潮热、盗汗）→ 选用清虚热药

2. **选择配伍组合**：
   - 根据具体证型参考经典配伍，如石膏配知母治肺胃实热，金银花配连翘治温病初起。

3. **评估禁忌体质**：
   - 若患者为脾胃虚弱、虚寒体质或便溏，则所有清热药均需慎用，因其大多苦寒，易伤脾胃阳气。
   - 特别注意：清热燥湿药“苦寒伐胃、性燥伤阴”，对津伤阴亏者亦应慎用。

4. **输出建议**：
   - 选择对应类别的清热药，并根据体质调整剂量或配伍扶正药物；若属禁忌人群，则避免使用或在严密监护下短期使用。
