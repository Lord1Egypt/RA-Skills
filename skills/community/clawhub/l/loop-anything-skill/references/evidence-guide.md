# Evidence Guide

Final review should include evidence appropriate to the artifact.

Use only evidence that exists or can be reasonably produced.

## Evidence By Artifact Type

Prompt or delegation instruction:

- target outcome checklist
- objective resource checklist
- constraint and acceptance-criteria checklist
- ambiguity scan

Code:

- tests, build, lint, typecheck
- changed files
- runtime logs
- reproduction steps

Design:

- screenshot or render
- interaction notes
- accessibility or readability checks
- design-system consistency checks

Plan or strategy:

- assumptions table
- risk table
- decision-readiness checklist
- stakeholder fit checklist

Research or analysis:

- source list
- claim audit
- uncertainty list
- decision-usefulness check

Operational workflow:

- dry-run output
- rollback plan
- monitoring or alerting evidence
- operator checklist
