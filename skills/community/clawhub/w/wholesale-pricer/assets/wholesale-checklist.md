# Wholesale Account Onboarding Checklist

Go through every item before the first PO ships. 38 checks across 7 categories.

## 1. Account setup (6 checks)

- [ ] Business license on file
- [ ] Resale / sales-tax certificate on file
- [ ] Point of contact: ordering contact + AP contact captured
- [ ] Shipping address + billing address recorded
- [ ] Freight account number recorded if retailer prefers to use their own
- [ ] Account assigned to a sales rep

## 2. Pricing agreement (6 checks)

- [ ] Tier decided (Tier 1 / Tier 2 / Tier 3)
- [ ] Exceptions noted and approved in writing (e.g., exclusive SKU, custom case pack)
- [ ] Case pack and MOQ confirmed and matches your catalog
- [ ] First-order promo applied (and expiry date set, 30 days default)
- [ ] Any MDF or co-op commitments documented
- [ ] Target annual volume noted (for rep reporting)

## 3. Payment terms (5 checks)

- [ ] Payment term decided (prepay / Net-30 / Net-60)
- [ ] Credit check run if term is Net-30 or longer and order is >$10k
- [ ] Three trade references checked (for Net-60+)
- [ ] Early-pay discount clause included (2/10 Net-30 default)
- [ ] Currency and tax handling confirmed for international accounts

## 4. MAP policy (4 checks)

- [ ] Retailer has received the MAP policy document
- [ ] Retailer has been pointed to the current MAP price per SKU
- [ ] Retailer acknowledged receipt (email record)
- [ ] Retailer added to MAP monitoring service

## 5. Freight and shipping (5 checks)

- [ ] Threshold for free freight communicated ($1,200 default)
- [ ] Retailer's preferred carrier noted (if any)
- [ ] Delivery window expectations agreed (standard lead time per PO)
- [ ] Return freight policy explained (defective paid by brand, slow movers not returnable)
- [ ] Split-case policy confirmed (no broken cases)

## 6. Communication and ordering process (5 checks)

- [ ] Ordering method: EDI / portal / email confirmed
- [ ] Order confirmation and shipping notification templates set up
- [ ] Invoice delivery method confirmed (email to AP vs. portal)
- [ ] Dispute resolution contact (ours + theirs) exchanged
- [ ] Cadence check-in scheduled (monthly for top 10%, quarterly for others)

## 7. Ongoing account hygiene (7 checks)

- [ ] First three invoices tagged for close monitoring (payment on time?)
- [ ] MAP compliance check at 30 days post-first-ship
- [ ] Reorder prompted via rep or email automation 45 days post-ship
- [ ] Tier review at 6 months (promote if earned)
- [ ] Credit limit reviewed annually or when order size grows materially
- [ ] Chargebacks and defective-unit returns tracked per account
- [ ] Account profitability reported quarterly (gross margin − freight − chargebacks − working capital cost)
