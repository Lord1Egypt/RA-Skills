# 短剧情绪模式库

## 六种基础情绪模式

### 1. 初次相遇（First Meeting）
**情绪弧：好奇→微紧张→心动→期待**
```
开场：对方出现，打破日常节奏
第一眼：外表/气场留下印象
互动：对话中有来有往，有攻防
结尾：有意无意制造下一次见面契机
```
**镜头设计：**
- 远景建立对方出场（低角仰拍）
- 近景聚焦眼神交流（视线交错）
- 中景对话（越肩镜头）

---

### 2. 误解冲突（Misunderstanding）
**情绪弧：期待→落差→委屈→防御→对峙**
```
触发：某句话/动作被错误解读
发酵：双方各自脑补
爆发：情绪对抗，言语交锋
结尾：冷战或僵持（钩子）
```
**镜头设计：**
- 越肩对峙（压迫感）
- 特写手部动作（握拳/松开）
- 硬切退出（摔门/转身）

---

### 3. 分离与牵挂（Separation）
**情绪弧：不舍→孤独→思念→决心**
```
离别：表面平静，内心翻涌
独处：空镜头+内心独白
思念：对方物品/习惯触发回忆
决心：做出某个决定（要去找对方/要放下）
```
**镜头设计：**
- 背影远景（送别）
- 空镜头（对方留下的物品）
- 特写（目光远眺）

---

### 4. 重逢与突破（Reunion）
**情绪弧：紧绷→试探→释放→确认**
```
重逢：意外/有意，双方都在试探
破冰：某件事打破僵局
坦白：内心话说出来
确认：关系升级或明确
```
**镜头设计：**
- 从远景到近景（走近）
- 特写眼神变化（防备→软化）
- 动作细节（手指接触/拥抱）

---

### 5. 情绪高潮（Climax）
**情绪弧：蓄力→爆发→失控→崩溃/胜利**
```
蓄力：事件逐步积累，压力增加
触发点：一个动作/话语达到临界
爆发：情绪决堤（哭/愤怒/崩溃）
结果：关系/局势不可逆转改变
```
**镜头设计：**
- 面部特写（眼泪/颤抖）
- 快速剪辑（情绪强度）
- 静音/极简背景音（放大情绪）

---

### 6. 结局落版（Resolution）
**情绪弧：震荡→平静→新平衡**
```
结果：事件有了结果（好/坏/开放）
反思：角色对事件的感受
成长：内心发生变化
落版：新的状态/新的开始
```
**镜头设计：**
- 远景收尾（场景开阔或封闭）
- 静帧落版（情绪定格）
- 留白/悬念（开放结局）

## 虐点与爽点设计

### 虐点（Pain Points）
让人心疼的点，让观众替角色难过

| 类型 | 写法 | 示例 |
|------|------|------|
| 隐忍 | 不说，憋着，情绪从细节漏出 | 眼眶红了但声音平稳 |
| 牺牲 | 为了别人放弃自己的利益 | 卖掉唯一的手表付学费 |
| 无声 | 有机会但不解释，任由误解 | 看着对方转身离开，不追 |
| 习惯 | 习惯了痛苦，当成正常 | "没事，习惯了" |

### 爽点（Reward Points）
让人痛快的点，让观众觉得"解气"

| 类型 | 写法 | 示例 |
|------|------|------|
| 反击 | 以彼之道还施彼身 | "你说我没用？给你看看什么叫有用" |
| 翻身 | 从被动到主动 | 原本被踩的人反过来掌控局面 |
| 揭示 | 真相大白，反派打脸 | "你不知道吧，我才是幕后" |
| 守护 | 关键时刻站出来保护 | 在所有人面前站在他身边 |

## 情绪曲线模板

### 标准60秒短剧曲线
```
0-5秒：抓人（钩子/困境）
5-15秒：建立（角色处境/关系）
15-40秒：发展（冲突/误解/拉扯）
40-50秒：高潮（情绪爆发/转折）
50-60秒：落版（留下悬念/情绪定格）
```

### 快节奏曲线（情绪强烈）
```
0-3秒：冲突爆发
3-20秒：对抗升级
20-35秒：关键转折
35-45秒：情绪最高点
45-60秒：钩子/悬念
```

### 慢热曲线（情感积累）
```
0-15秒：日常建立
15-30秒：微妙变化
30-45秒：关系转折
45-55秒：内心确认
55-60秒：行动/决定
```

## 平台适配

| 平台 | 情绪节奏 | 特点 |
|------|---------|------|
| 抖音 | 快节奏，前3秒必抓人 | 15-60秒，强刺激 |
| 快手 | 接地气，情绪强烈 | 同上，偏下沉 |
| 微信 | 情感向，细腻 | 60秒内，情感积累 |
| 海外短剧 | 强情节，高潮密集 | 45-60秒，虐爽交替 |
