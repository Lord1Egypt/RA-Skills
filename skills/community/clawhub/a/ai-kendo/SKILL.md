# AI剑道指南

全球600万剑士。从摺足到残心，气剑体一致。

## 核心能力

### 1. 剑道入门
- **剑道vs剑术**：剑道(Kendo)=现代体育化/剑术(Kenjutsu)=古流实战/剑道由剑术演变而来
- **剑道理念**："剑道は剣の理法の修錬による人間形成の道である"→剑道是通过剑之理法的修炼而到达人格完成之道
- **不是暴力**：剑道是"对人尊重/谦让/克己"的武道
- **竹刀Shinai**：四片竹片或碳纤维组成/长120cm(男子一般)/重约510g
- **防具Bogu**：面Men(头盔)/胴Do(胸甲)/小手Kote(手套)/垂Tare(腰垂)
- **道服**：剑道衣Kendogi(上衣/蓝或白)+袴Hakama(裙裤)
- **足**：赤足/在木地板上练

### 2. 基本构与步法
- **中段Chudan-no-kamae**（核心构）：竹刀剑尖对准对手咽喉/最平衡/攻防兼备/所有人的起点
- **上段Jodan-no-kamae**：剑高举过头(火之构)/纯粹进攻/左手在前(左撇子反过来)
- **下段Gedan-no-kamae**：剑尖放低对准膝盖(地之构)/防守/等待对手出手
- **八相Hasso-no-kamae**：剑在右肩侧(林之构)/剑道形中用
- **胁构Waki-gamae**：剑隐藏在身体右侧后(古流多用/剑道形)
- **摺足Suriashi**：脚底不离开地板/滑动走路/前足推动/后足跟上(不要跨步不要跳)
- **方向转换**：以左脚为轴/身体像圆规转
- **间合Maai(距离)**：一足一刀的间距/进一步能打到/退一步能逃开

### 3. 素振Suburi与打击
- **素振=空挥**：每天先做100-200次/练正姿势
- **正面Men**：双手挥→正面头顶/落地响亮
- **左右面Sayu-Men**：先右面→再左面(15-30度角)
- **小手Kote**：打右手前臂(腕部)
- **胴Do**：打右腹部(斜砍)
- **刺击Tsuki**：直刺喉部/高级技术/初段以上才用(危险)
- **气剑体一致Ki-Ken-Tai-Icchi**：喊(气)+剑打到(剑)+身体跟上(体)=同步/不同步=无效攻击
- **残心Zanshin**：打击后保持警醒/不放松/不回头看/这是剑道最核心的精神

### 4. 切返与稽古
- **切返Kirikaeshi**(最重要的基本功)：
  1. 打正面→身体冲撞Taiatari
  2. 连续打左右面(前进4步×5次/后退5次)
  3. 最后一击打正面→穿过去/保持残心→转身
  4. 重复左右面→最后一击正面
- **互格稽古Kakari-Geiko**：短时间/全力连续攻击/锻炼体能和意志
- **地稽古Ji-Geiko**：自由对练/像实战一样
- **稽古结构**：热身→素振→切返→互格稽古→地稽古→整理

### 5. 日本剑道形Nippon Kendo Kata
- **木刀Bokuto**：剑道形用木刀/不用竹刀
- **太刀Tachi之形(1-7本)**：长刀vs长刀/7个套路
- **小太刀Kodachi之形(8-10本)**：短刀(小太刀)vs长刀/3个套路
- **打太刀Uchidachi(老师)**vs**仕太刀Shidachi(学生)**：老师先攻/学生应对→最后学生胜
- **形vs稽古**：形是原理教科书/稽古是应用
- **升段必考**：考初段到五段必须考剑道形

### 6. 试合(比赛)规则
- **有效打突Yuko-Datotsu**：气剑体一致+残心/用竹刀的正确部位/打到正确的目标
- **得分部位Datosu**：正面Men/左小手或右小手Kote/右胴Do/突刺Tsuki(喉)
- **一本Ippon**：裁判3面旗2面同时举起=一本得分/2本先取者胜
- **犯规Hansoku**：出界/消极/危险动作/不当言行
- **2个犯规=对手得一本**
- **比赛时间**：3-5分钟/延长赛Encho(先取得一分者胜)
- **团体赛**：5人(先锋/次锋/中坚/副将/大将)
- **段位**：级(6-1级)→初段~八段/八段是最高实际段位(九段十段极少)

### 7. 装备与护理
- **竹刀**：39号(高中生以上男子)/38号(女子或中学生)/竹片定期检查/裂缝=换
- **防具**：面金(前网)要检查/久用有凹陷→防刺穿/胴乳革(胸绳)检查
- **道服**：蓝(靛蓝染)/白(夏季或少儿)/每次训练后晾干
- **手拭Tenugui**：头巾/垫在面下/吸汗
- **胴紐Do-himo**：细腰带/扎在袴下面
- **护理仪式**：训练后/竹刀擦干/防具通风/道服折叠
- **竹刀修理**：换竹片(四竹全换或单换)/弦Tsuru(刀背)松了重新拉

### 8. 礼仪与文化
- **道场礼Reiho**：进门鞠躬→向神棚(或正面)鞠躬→老师/先辈鞠躬→开始稽古→结束时同样顺序
- **立礼Ritsurei**：立鞠躬(15度)/日常
- **座礼Zarei**：跪座鞠躬/正式
- **蹲踞Sonkyo**：竹刀对竹刀/蹲下/准备开始
- **剑道=生涯武道**：7岁到70岁都可以/有年龄分组
- **"先先之先"Sen-Sen-no-Sen**：在对手动念之前就出手/剑道的最高境界
- **剑道vs其他武术**：更重礼仪/不追求"制服"对手/而是共同进步
- **全世界剑道**：韩国Kumdo/台湾/巴西(日裔多)/法国/美国/中国发展快
