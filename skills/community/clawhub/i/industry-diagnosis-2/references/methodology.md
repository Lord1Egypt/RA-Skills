## 二、评分标准详表

### 容量引力

| 赛道规模 | 评分 | 等级 |
|---------|------|------|
| >1,000亿 | 90 | 大赛道 |
| 100-1,000亿 | 60-80 | 中赛道 |
| <100亿 | 30-50 | 小赛道 |

### 趋势动力

| 增速区间 | 评分 | 判断 |
|---------|------|------|
| >15% | 90 | 高增长 |
| 5-15% | 60-75 | 稳定增长 |
| 0-5% | 30-45 | 存量竞争 |
| <0% | 10-20 | 萎缩 |

### 竞争友好度

| CR3 | 格局 | 进入难度 | 评分 |
|-----|------|---------|------|
| >60% | 高度集中/寡头垄断 | 极难 | 20-30 |
| 40-60% | 寡头竞争 | 很难 | 30-50 |
| 20-40% | 竞争分散 | 中等 | 50-70 |
| <20% | 高度分散 | 较易 | 70-90 |

### 差异化引力

| 差异化空间 | 评分 | 判断 |
|-----------|------|------|
| 三个层次都有明确突破口 | 80-90 | 充足 |
| 1-2个层次有突破口 | 60-70 | 一般 |
| 仅定位差异化有空间 | 40-50 | 有限 |
| 三层都被占满 | 10-30 | 极度饱和 |

## 三、趋势四指标

| 指标 | 判断标准 |
|------|---------|
| 品类规模增速 | >15%高增长 / 5-15%稳定 / <5%存量 |
| 门店数量增速 | 开店>关店=扩张 / 反之=出清 |
| 头部品牌开店速度 | 净增=信心 / 净减=退缩 |
| 资本投向 | 资本进入=看好 / 资本退出=看空 |

## 四、趋势交叉信号

| 信号组合 | 含义 | 建议 |
|---------|------|------|
| 增速高+开店多+资本进 | 黄金赛道 | 快速进入 |
| 增速降+开店多+资本进 | 红海前夜 | 谨慎进入，打差异化 |
| 增速降+关店多+资本退 | 行业出清 | 不进入，或等出清结束后进入 |
| 增速稳+开店稳+资本观望 | 成熟期 | 只做差异化细分 |

## 五、差异化三层次

| 层次 | 护城河 | 典型路径 |
|------|--------|---------|
| 产品差异化 | 最深 | 独家原料/工艺/配方 |
| 服务差异化 | 中等 | 服务体验/数字化 |
| 定位差异化 | 最灵活 | 人群/场景/价格带 |

## 六、差异化三道检验

| 检验 | 标准 |
|------|------|
| 明显吗？ | 顾客进门第一眼能感知 |
| 稳定吗？ | 每次来都能体验到 |
| 难复制吗？ | 跟独占资源/深层能力绑定 |

## 七、定位三策

| 策略 | 适用场景 | 执行要点 |
|------|---------|---------|
| 找空白 | 赛道有未被满足的细分需求 | 画赛道地图→找无人区→验证需求 |
| 切弱点 | 赛道已有强者但有明显短板 | 分析竞品差评→单点突破→做到极致 |
| 做擅长 | 自身有独占资源或核心能力 | 盘点自身优势→匹配赛道需求→打透 |

## 八、扩张节奏

| 阶段 | 店数 | 核心目标 | 节奏 |
|------|------|---------|------|
| 验证期 | 1→3家 | 验证模型 | 慢 |
| 深耕期 | 3→10家 | 区域扎根 | 稳 |
| 扩张期 | 10→100家 | 规模复制 | 快 |

## 九、扩张开关闭合红线

| 指标 | 合格线 | 警戒线 |
|------|--------|--------|
| 单店月度平均净利润率 | >=20% | <12% |
| 核心客群月度复购率 | >=40% | <22% |
| 顾客净推荐值(NPS) | >=75 | <55 |
| SI标准化飞检执行得分 | >=95分 | <80分 |

闭合规则：四项指标任一跌破警戒线 → 暂停扩张

## 十、风险评级

| 综合评分 | 风险等级 | 建议 |
|---------|---------|------|
| >=80 | 低风险 | 推荐进入，可考虑"找空白"策略 |
| 60-79 | 中等风险 | 谨慎进入，建议"切弱点"策略 |
| 40-59 | 较高风险 | 不推荐进入，除非"做擅长" |
| <40 | 高风险 | 不推荐进入 |

## 十一、数据采集SOP

### 搜索关键词模板

```python
# 第1步：市场容量
search_1 = f"{industry}市场规模 亿元"
search_2 = f"{industry}增长率 增速"

# 第2步：头部品牌
search_3 = f"{industry}品牌排名 TOP10 门店"
search_4 = f"{industry}头部品牌 市占率"

# 第3步：竞争格局
search_5 = f"{industry}竞争格局 CR3 集中度"
search_6 = f"{industry}倒闭 关店 闭店率"

# 第4步：差异化
search_7 = f"{industry}差异化竞争 定位"
search_8 = f"{industry}产品创新 趋势"
```

### 数据交叉验证

| 原则 | 方法 |
|------|------|
| 多源验证 | 同一数据至少2个来源佐证 |
| 时效性 | 优先近6个月的数据 |
| 来源标注 | 每个数据标注来源网站+发布时间 |
| 存疑标注 | 无法验证的数据标注"待验证" |
| 口径分离 | 区分"核心战力"与"生态网络"数据 |
