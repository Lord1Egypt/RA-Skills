## Description: <br>
Generate publication-quality figures for Nature-series journals using Python (matplotlib) or R (ggplot2), including multi-panel layouts, Nature-style color palettes, correct typography, and SVG/PDF/PNG exports. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External researchers, developers, and scientific authors use this skill to create or polish Nature-style scientific figures from user-provided data. It helps produce runnable Python or R plotting code, export instructions, figure legends, and submission checklists. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The fast auto-figure shortcut references a separate local helper script that is not included in the reviewed package. <br>
Mitigation: Verify that the local helper script is present and trusted before running it, or ask the agent to generate transparent Python or R code and run it only on data files you choose. <br>
