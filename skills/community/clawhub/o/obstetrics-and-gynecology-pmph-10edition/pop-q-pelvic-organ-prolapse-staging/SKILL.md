# POP-Q盆腔器官脱垂分度评估

## 适用条件
- 患者为女性，主诉下坠感、阴道肿物脱出、排尿或排便困难
- 患者能配合完成最大用力屏气（Valsalva动作）
- 已完成妇科检查并可测量各解剖点位置
- 排除急性炎症期或无法配合检查者

## 执行步骤

1. **测量六个核心解剖指示点**（单位：cm；以处女膜缘为0点，上方为负值，下方为正值）：
   - Aa：阴道前壁中线距处女膜3cm处（正常范围：-3～+3）
   - Ba：阴道前壁上段最远脱出点（无脱垂时为-3，完全外翻时可达+TVL）
   - C：宫颈或阴道顶端最远端（范围：-TVL～+TVL）
   - D：后穹隆位置（仅在有宫颈时存在；子宫切除后此项空缺）
   - Ap：阴道后壁中线距处女膜3cm处（正常范围：-3～+3）
   - Bp：阴道后壁上段最远脱出点（对应Ba）

2. **记录辅助参数**：
   - gh（生殖裂隙长度）：尿道外口至后处女膜缘中点距离
   - pb（会阴体长度）：阴裂后缘至肛门中点距离
   - TVL（总阴道长度）：用于校正C、D、Ba、Bp的上限

3. **判定POP-Q分度**（依据最远脱出点的量化值）：
   - **0度**：Aa、Ap、Ba、Bp均为-3 cm，且C/D ≤ -(TVL - 2)
   - **I度**：最远脱出点 ≥1 cm 在处女膜上方（量化值 ≤ -1 cm）
   - **II度**：最远脱出点位于处女膜±1 cm 内（量化值 ≥ -1 且 ≤ +1）
   - **III度**：最远脱出点超出处女膜 ≥1 cm 但 ≤ TVL - 2（量化值 ≥ +1 且 ≤ TVL - 2）
   - **IV度**：完全外翻，最远脱出点 ≥ TVL - 2（量化值 ≥ TVL - 2）

4. **注意事项**：
   - 所有测量必须在Valsalva动作下进行
   - 先填写3×3 POP-Q表格记录具体数值，再进行分度判定
   - TVL允许±2 cm误差，用于0度和IV度的边界判定
   - 不得将传统分度法与POP-Q混用于同一患者的疗效评价
   - 建议结合PFIQ-7和PISQ问卷综合评估症状对生活质量的影响

## 输出结果
输出完整的POP-Q分度（0–IV度）及各解剖点的具体测量数值，用于临床决策、手术规划及随访疗效对比。
