## Known Risks and Mitigations: <br>
Risk: User-initiated install, setup, update, publish, or compiler commands may affect the local development environment or release artifacts. <br>
Mitigation: Review commands before execution and run them only in trusted PerryTS projects and environments. <br>
Risk: Installing PerryTS through npm, Homebrew, or winget depends on trusting the referenced package or tap. <br>
Mitigation: Verify the PerryTS package source before installation. <br>
Risk: The Geisterhand UI testing server exposes a localhost control surface when enabled. <br>
Mitigation: Enable the localhost testing server only for projects you trust. <br>
