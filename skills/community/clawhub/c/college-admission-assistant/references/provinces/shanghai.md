# 上海市高考数据参考

> 数据来源：上海市教育考试院公开信息
> 最后更新：2026年4月

---

## 一、高考模式

**3+3模式**（2017年首届新高考）
- 总分：660分
- 语文150 + 数学150 + 外语150 + 6选3（等级赋分各70分，合计210分）
- ⚠️ 上海总分660分（非750分），注意换算

## 四、选科要求特色

1. **上海高考总分660**：是全国唯一非750分的省份
2. **外语一年两考**：取最高成绩计入总分
3. **综合评价录取**：复旦、交大、同济、华师大等有综合评价批次，是主要录取渠道
4. **上海711所重点高校**：上海高水平大学密集

---

## 五、志愿填报规则

1. **批次**：综合评价批 → 提前批 → 普通批
2. **志愿单位**：院校专业组
3. **普通批**：24个平行志愿
4. **综合评价批次**：最早批次，是进复旦、交大的主渠道
