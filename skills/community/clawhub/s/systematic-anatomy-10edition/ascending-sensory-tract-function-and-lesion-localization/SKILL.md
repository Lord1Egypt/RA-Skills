# 上行感觉传导束功能与损伤定位

## 何时使用
- 患者出现躯干或四肢的感觉障碍（如闭目不能判断肢体位置、痛温觉减退、两点辨别觉丧失）
- 需区分感觉障碍为同侧还是对侧，以判断损伤节段是否在感觉通路交叉点以上或以下
- 神经系统检查中需将症状映射至特定传导束

## 执行步骤
1. **识别感觉障碍类型**：
   - 本体感觉（位置觉、运动觉、振动觉）和皮肤精细触觉 → 提示薄束/楔束或内侧丘系受损
   - 痛觉、温度觉减退 → 提示脊髓丘脑侧束受损
   - 粗略触压觉障碍 → 提示脊髓丘脑前束受损

2. **判断损伤侧别与交叉关系**：
   - 脊髓后索（薄束、楔束）损伤 → **同侧**本体觉与精细触觉障碍
   - 内侧丘系交叉（延髓）以上损伤 → **对侧**本体觉与精细触觉障碍
   - 脊髓丘脑束在脊髓内1–2节段交叉 → 损伤平面以下1–2节段**对侧**痛温觉障碍

3. **结合解剖路径定位**：
   - T5及以下本体觉障碍 → 薄束受损
   - T4及以上本体觉障碍 → 楔束受损
   - 全身性协调障碍伴非意识性本体觉异常 → 考虑脊髓小脑束

4. **输出临床提示**：
   - 根据感觉缺失模式，明确最可能受损的传导束及损伤节段
   - 若同时存在多种感觉障碍，考虑多束受累或广泛性脊髓病变

> 注意：此技能不适用于周围神经病变；内脏感觉通常伴随脊髓丘脑束上行，但临床评估较少单独依赖。
