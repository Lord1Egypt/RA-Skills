# 29 ䷜ 坎卦 — 连续风险模型

## 卦象结构

![卦象图](../hexagrams/29-kan.svg)
- 下坎 ☵ 上坎 ☵（习坎，双水相叠）
- 内部状态：深陷不确定性和风险之中（坎为水为险）
- 外部环境：同样是风险和未知，双重叠加

## 情境识别

什么时候你会用到这个模型？
- 连环危机——一个问题还没解决，新的问题又来了
- 现金流危机 + 人才流失 + 客户投诉同时爆发
- 行业下行 + 竞争对手围剿 + 内部管理混乱
- 创业中的"至暗时刻"——连续受挫
- 多重压力叠加：投资人的施压、团队的焦虑、市场的质疑
- 任何"一波未平一波又起"的时期

## 核心原则

1. **习坎，有孚，维心亨**：在连续的风险中（习坎=反复经历坎）保持信心（有孚），心态通达（维心亨）。心态不崩是第一位的。
2. **坎不盈，祗既平**：坑不要挖太深（坎不盈=坑不要填满水），差不多平了就收手。不要陷入"越挣扎陷得越深"的困局。
3. **纳约自牖**：从窗户递进东西（不是走正门）。在连续危机时，不按常理出牌——走窗户、走后门、用非常规渠道解决问题。
4. **樽酒簋贰，用缶**：简单到极致（樽酒簋贰=一樽酒两盘菜，用缶=用土陶器），但诚心诚意。危机中只需要做几件简单但正确的事。
5. **系用徽纆，寘于丛棘**：如果你在连续风险中乱来，就会被绳索捆住、丢进荆棘丛里。危机中的错误决策会有级联效应。

## 阶段地图（六爻 → 连续风险阶段）

- **初六·习坎，入于坎窞，凶** → 掉进坑里：第一次坎还可以应对，连续坎就"入于坎窞"（掉进了深坑）。这个阶段是一个危险信号——你进入了连续风险周期。
- **九二·坎有险，求小得** → 小步求存：坑里有风险，不追求大翻盘，只求"小得"——小小的进展。能止血就止血，能保住一个客户就保住一个。
- **六三·来之坎坎，险且枕，入于坎窞，勿用** → 进退皆坎：往前走是坎（来之坎），往后退也是坎（坎坎）。风险和枕头一样伴随你（险且枕）。这个时候"勿用"——不要轻举妄动，任何动作都可能让情况更糟。
- **六四·樽酒簋贰，用缶，纳约自牖，终无咎** → 极简应对：一壶酒两碟菜，用最简单的器具。从窗户递进联系（不正式、不排场、但有效）。在连续危机中，精简到最核心的几件事。
- **九五·坎不盈，祗既平，无咎** → 不再加深：坑里的水不再涨了（坎不盈=坑不满），差不多平了就停了。"祗既平"= 刚好够平复。这个阶段的核心是止住恶化的趋势。
- **上六·系用徽纆，寘于丛棘，三岁不得，凶** → 被锁死：被绳索捆住（系用徽纆），丢进荆棘丛。三年脱不了身。这是连续危机中最糟糕的结果——如果在前几个阶段没有"纳约自牖"和"坎不盈"，就会走到这一步。

## 关键警示

- ❌ 最常见的错误：在"来之坎坎"时（进退都是坎）继续乱动——乱裁员、乱砍业务、乱借钱。结果越动坑越深
- ❌ 心态崩了——坎卦的核心是"维心亨"（保持内心通达），连续风险最大的伤害不是现实损失，是心理崩塌
- ❌ 坎卦六爻只有九二和九五给了"无咎"或"求小得"，说明在连续风险中，不要期待大胜。活下来就是胜利

## 关联模型（格栅连接）

- ䷝ **离卦（错卦+后卦）**：坎是"风险和黑暗"，离是"光明和依附"。坎→离的卦序暗示：连续风险之后会有转机（光明）。但离卦的智慧是"附着于正确的东西"——找到对的平台、对的合作伙伴。
- ䷚ **颐卦（互卦）**：坎的内部机制是颐 → 在连续风险中，最需要的是"颐"的智慧——细水长流的生存策略。不是一口吃个胖子，是每一口都吃得稳。
- ䷜ **坎自综**：坎是自综卦。连续风险不管从哪个角度看，都是风险。应对之道也不是一个方向的——既要从内部稳住（下坎），又要从外部防御（上坎）。
