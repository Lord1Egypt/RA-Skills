## Description: <br>
Skill Finder CN Pro helps Chinese and bilingual users discover relevant ClawHub skills, inspect results, and consider install commands for their needs. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[lvjunjie-byte](https://clawhub.ai/user/lvjunjie-byte) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and developers use this skill to turn a natural-language need into ClawHub skill searches, review candidate skill details, and decide whether to install a matching skill. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Installing a recommended skill can change what an agent is able to do. <br>
Mitigation: Review the target skill's description, publisher, files, and permissions before running any suggested install command. <br>
