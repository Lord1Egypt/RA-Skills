> 体系：北斗主星 + 南斗六星 + 中天诸星 + 辅佐煞忌星
> 用于：命盘格局分析、星曜组合判断、身宫迁移论断

## 〇、安命宫法（排盘核心步骤）

> 紫微斗数排盘的第一步是确定命宫位置，然后依次安十二宫、安星曜

### 第一步：定命宫

以出生年月日时（农历），从寅宫起正月，顺数至出生月，再从该宫起子时，逆数至出生时辰，所落之宫即为命宫。

```
寅宫起正月 → 顺数到出生月 → 得临时宫位
从临时宫位起子时 → 逆数到出生时辰 → 即为命宫
```

**示例**：农历三月午时生
- 寅起正月→卯二月→辰三月（临时宫=辰）
- 辰起子时→卯丑→寅寅→卯卯→辰辰→巳巳→午午时（命宫=巳）

### 第二步：安十二宫

从命宫起，逆时针依次安：

```
命宫 → 兄弟 → 夫妻 → 子女 → 财帛 → 疾厄 →
迁移 → 奴仆 → 官禄 → 田宅 → 福德 → 父母
```

### 第三步：安身宫

从寅宫起正月，顺数至出生月，再从该宫起子时，**顺数**至出生时辰，所落之宫即为身宫。

> 命宫是逆数时辰，身宫是顺数时辰，方向不同！

### 第四步：安十四主星

**确定命宫天干**：
根据出生年干，用五虎遁法则给命宫及各宫配天干：

| 年干 | 寅宫天干 |
|------|----------|
| 甲己 | 丙寅起 |
| 乙庚 | 戊寅起 |
| 丙辛 | 庚寅起 |
| 丁壬 | 壬寅起 |
| 戊癸 | 甲寅起 |

**确定局数**：
命宫天干+地支的纳音五行决定局数：

| 纳音五行 | 局数 |
|----------|------|
| 水二局 | 2 |
| 木三局 | 3 |
| 金四局 | 4 |
| 土五局 | 5 |
| 火六局 | 6 |

> 命宫天干地支纳音查法：用六十甲子纳音表（见bazi_theory.md）查命宫干支对应的纳音五行

**紫微星安法**：
1. 以出生日数除以局数（水二局/木三局/金四局/土五局/火六局），得紫微星所在宫位
2. 局数由命宫天干纳音决定
3. 紫微星确定后，其余十三主星按固定规则排列

**紫微星系排列**（从紫微星起，按固定顺序逆排）：
```
紫微 → 天机 →（ skip空位）→ 太阳 → 武曲 → 天同 →（空位）→ 廉贞
```

**天府星系排列**（从天府星起，天府与紫微始终在对宫位置，按固定顺序顺排）：
```
天府 → 太阴 → 贪狼 → 巨门 → 天相 → 天梁 → 七杀 →（空位三合位）→ 破军
```

### 第五步：安辅佐煞忌星

- **左辅右弼**：左辅从辰起正月顺数，右弼从戌起正月逆数
- **文昌文曲**：文昌从戌起子时逆数，文曲从辰起子时顺数
- **天魁天钺**：以年干定（甲戊庚见丑未，乙己见子申，丙丁见亥酉，壬癸见卯巳）
- **擎羊陀罗**：以年干定禄存位，擎羊在禄存后一宫，陀罗在禄存前一宫
  - 禄存安法：甲禄在寅，乙禄在卯，丙戊禄在巳，丁己禄在午，庚禄在申，辛禄在酉，壬禄在亥，癸禄在子
- **火星铃星**：以年支+时辰安星
- **地空地劫**：以时辰安星

| 戊 | 贪狼 | 太阴 | 右弼 | 天机 |
| 己 | 武曲 | 贪狼 | 天梁 | 文曲 |

### 第七步：安命主星与身主星

- **命主星**：以命宫地支定
  子贪狼、丑巨门、寅禄存、卯文曲、辰廉贞、巳武曲、午破军、未武曲、申廉贞、酉禄存、戌文曲、亥巨门

## 八、身宫

身宫与命宫同宫或相邻，论**后天际遇与主观能动性**。

- 身宫在命宫：主观强，自我意识浓厚
- 身宫在财帛：理财意识强，财运后天努力
- 身宫在夫妻：感情经历深刻，感情主导人生
- 身宫在迁移：外出发展机遇多，人生多变动
