# 🚀 ResumeExpert Pro: AI 语义级简历审计与美化引擎

**ResumeExpert Pro** 是一个专为程序员设计的简历优化工具。它不仅提供高审美的排版，更内置了基于正则与语义分析的“HR审计大脑”，专门识别并修复技术简历中的逻辑缺陷。

## 🛠️ 核心技术特性

* **🔍 语义级逻辑审计**：自动检测入职、毕业、离职时间线，精准识别时间轴倒挂等逻辑冲突。
* **📊 量化产出诊断**：智能扫描项目描述，对缺乏 QPS、性能提升百分比等量化指标的区块给予优化建议。
* **🛡️ 一键隐私脱敏**：内置掩码算法，可快速生成隐藏手机号、邮箱的“投递安全版”简历。
* **🎨 自适应极客排版**：
    * **技术栈自动加粗**：识别 Java、微服务、分布式等核心词汇并进行视觉高亮。
    * **布局智能适配**：根据字数自动调整行间距与字号，确保页面视觉饱满。

## 📂 模块说明
* `resume_beautifier.py`: 底层渲染引擎，负责高亮渲染与 Docx/PDF 生成。
* `resume_beautifier_pro.py`: 业务逻辑大脑，负责 AI 审计、打分与脱敏策略。
