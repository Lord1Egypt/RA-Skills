# 中英翻译桥映射表 — Translation Bridge Map

用于将中文 query 关键词翻译为英文，使 all-MiniLM-L6-v2 embedding 能匹配英文 skill 库。

格式说明：
- 左列：中文触发词（按长度降序匹配，长词优先于短词）
- 右列：英文翻译短语（多个同义词空格分隔，增强语义信号）
