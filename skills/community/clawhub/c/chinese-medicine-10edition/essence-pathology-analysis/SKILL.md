# Essence Pathology Analysis

## When to Use
当出现以下任一表现时启动本技能：
- 小儿身材矮小、囟门迟闭等生长发育障碍
- 成人不孕不育、精少、性功能减退
- 早衰征象（齿落发白、耳鸣健忘、精神萎靡）
- 排精异常（滑精、梦遗、早泄、排精不畅、精道疼痛、睾丸坠胀等）

## Execution Steps

### 第一步：判断是否属于“精的失常”
- 排除单纯气血津液失调
- 女性患者需结合冲任理论另行分析，本技能主要适用于男性患者及生长发育障碍者
- 确认存在肾精或生殖之精的功能异常

### 第二步：辨识具体病理类型

#### 1. 精虚（肾精不足）
- **成因**：先天禀赋不足、后天失养、过劳伤肾、久病及肾、脾失健运
- **关键表现**：
  - 生长发育障碍（小儿）
  - 生殖功能减退（男女不孕不育，男子精少）
  - 早衰征象
  - 若由脾源性导致，兼见面色萎黄、肌肉瘦削、头昏目眩、疲倦乏力、舌淡脉弱

#### 2. 失精（排泄过度）
- **成因**：房事过度、肾气虚衰封藏失职；或相火偏亢、肝疏泄太过
- **关键表现**：滑精、梦遗、早泄，伴精力不支、失眠健忘、腰膝酸软、尿频肢冷
- **危重提示**：若精泄不止，警惕“精脱”→“气脱”的危候

#### 3. 精瘀（排泄障碍）
- **成因**：忍精不泄、手淫、久旷、惊恐伤肾；或瘀血/湿热/痰浊阻滞精道；或肾气虚推动无力、肝郁疏泄失司
- **关键表现**：排精不畅、精道疼痛、睾丸坠胀、小腹挛痛、精索硬结如串珠、腰痛头晕

### 第三步：输出病机结论与治疗方向
- 明确属于精虚、失精或精瘀
- 指导对应治法：
  - 精虚 → 补肾填精，兼顾健脾益气
  - 失精 → 固精止遗，调补肾气，清泻相火（如有）
  - 精瘀 → 通精化瘀，疏肝理气，清利湿热或化痰散结（依病因）

> 注意：本技能聚焦于男性及生长发育相关精的失常，女性生殖问题需结合冲任理论单独处理。
