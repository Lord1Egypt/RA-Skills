# Canon References / 原著背景档案

This folder holds **fan-authored canon digests** for YumFu's source-based worlds — one `<world-id>.md` per IP.

## What these are
Each file is an **original, in-our-own-words setting summary** of a source work: world overview, era, faction/house/clan structure, a staged plot scaffold, a character roster, magic/system anchors, themes, and an "anchor vs. free-to-diverge" guide.

They exist so YumFu can keep turns **faithful to the source world** (real geography, factions, characters, plot beats) instead of drifting into generic atmosphere — see SKILL.md "World grounding rule".

## What these are NOT
- ❌ **Not** the original novels/scripts. No copyrighted source text is reproduced here.
- These are transformative summaries/digests for non-commercial fan play. Plot may diverge from canon.

## Why no full original text?
Most of these IPs (Game of Thrones, LOTR, Harry Potter, Warriors, and Jin Yong's wuxia) are **still under copyright**. Bundling and publicly distributing the full original text would be infringement. Public-domain classics (e.g. *Journey to the West*, *Records of the Three Kingdoms*, the *Great Tang Records on the Western Regions*) may carry fuller source text where appropriate.

## How YumFu uses them
When running a world that has a canon file here, prefer loading `worlds/canon/<world-id>.md` alongside the mechanics in `worlds/<world-id>.json` to anchor: next moves, NPC pressure, travel routes, factions, artifacts, and the main story line. The JSON drives systems; the canon.md keeps the *story* on-source.
