# 视觉传导通路损伤与视野缺损的对应关系

## 何时使用
- 患者出现明确模式的视野缺损（如中央暗点、同向性偏盲、颞侧半偏盲等）
- 需要根据视野表现初步判断视觉通路损伤解剖位置
- 用于指导后续影像学检查（如MRI）的靶向定位
- 适用于眼科、神经内科或神经外科的临床评估场景

## 如何执行
1. **确认前提条件**：
   - 已掌握视野定义
   - 理解视网膜成像倒置规律（鼻侧视野投射至颞侧视网膜，反之亦然）

2. **根据视野缺损类型反推损伤部位**：
   - **视网膜损伤**：
     - 视神经盘损伤 → 视野中出现较大暗点
     - 黄斑部受损 → 中央视野暗点
     - 其他区域损伤 → 对应视野区域暗点
   - **一侧视神经损伤** → 该侧眼全盲
   - **视交叉中央交叉纤维损伤** → 双眼视野颞侧半偏盲
   - **一侧视交叉外侧部不交叉纤维损伤** → 患侧眼鼻侧半视野偏盲
   - **一侧视束、视辐射或视皮质损伤** → 双眼对侧同向性偏盲（例如右侧通路受损 → 右眼鼻侧半 + 左眼颞侧半偏盲）

3. **输出建议**：
   - 提示特定神经通路损伤可能性
   - 建议进行针对性影像学检查（如头颅MRI）以确认病灶

> 注意：本技能仅适用于单侧或局灶性损伤，不适用于弥漫性脑病或多系统病变。
