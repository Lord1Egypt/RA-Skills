# 剪映(CapCut)后期合成操作指南

## 剪映合成七步法

### Step 1: 导入素材

1. 打开剪映专业版 → 点击「开始创作」→ 新建项目
2. 命名项目：`[主题]_科普视频`
3. 将 `视频素材/` 中所有文件（视频+图片）全选 → 拖入媒体库/点击「导入」
4. 将 `配音音频/` 中所有文件全选 → 拖入媒体库

**素材管理技巧**：
- 视频素材和图片素材分不同文件夹导入，便于区分
- 导入后在媒体库中按文件名排序，与分镜表镜号对应
- 建议先预览每个素材确认内容正确

---

### Step 2: 排列素材到时间轴

按分镜表顺序将素材拖入主视频轨道（V1）：

```
操作流：
媒体库选中素材 → 拖入时间轴V1轨道 → 调整时长 → 下一个素材
```

**排列规则**：
- 严格按分镜表镜号顺序从左到右排列
- 视频素材：保持原始时长，如需截取用「分割」工具
- 图片素材：默认3-5秒，按脚本预估时长调整
  - 设置方法：选中图片 → 拖动右边缘 → 或右键「时长设置」
- 每个素材间预留转场位置（重叠0.3-0.5秒）

**时间轴快捷键**：
| 操作 | 快捷键 |
|------|--------|
| 分割 | Ctrl+B |
| 删除片段 | Delete |
| 撤销 | Ctrl+Z |
| 放大/缩小时间轴 | Ctrl+滚轮 |
| 吸附开关 | N |

---

### Step 3: 添加配音音频

1. 将音频片段按顺序拖入音频轨道 A1
2. **对齐规则**：每段音频的起始点与对应画面的起始点对齐
3. 音频间添加 0.5-1 秒空白间隔（制造段落停顿感）
4. **音量统一**：
   - 选中所有音频 → 右键「调节音量」
   - 统一调至 -3dB 至 -6dB
   - 或选中单段 → 音量条下拉调整

**音频处理技巧**：
- 音频开头/结尾添加淡入淡出（0.2-0.5秒）避免爆音
- 如某段音频偏长/偏短，用「变速」微调（保持音调）
- 检查音频与画面口型/字幕是否同步

---

### Step 4: 添加字幕

#### 方式A：智能字幕（有配音时推荐）

1. 选中所有音频片段（Ctrl+多选）
2. 点击顶部「文本」→「智能字幕」→「识别字幕」
3. 等待AI识别完成
4. **校对修正**：
   - 逐条检查文字准确性
   - 特别注意：人名、专业术语、英文、数字
   - 如发现错误，双击字幕条直接修改文字

#### 方式B：手动添加字幕（无配音时使用）

1. 点击「文本」→「新建文本」
2. 按脚本中的字幕标注逐条输入
3. 调整每条字幕的起止时间
4. 拖动字幕条到画面合适位置

#### 字幕样式设置

**全选字幕后统一设置**（Ctrl+A选中所有字幕条）：

| 参数 | 标准设置 | Hook/金句设置 |
|------|---------|--------------|
| 字体 | 思源黑体 Bold / 阿里巴巴普惠体 Bold | 同左 |
| 字号 | 根据画面大小调整（正文偏小）| 偏大（占画面宽度80%）|
| 颜色 | 纯白 #FFFFFF | 金色渐变 #FFD700 → #FFA500 |
| 描边 | 黑色描边 3-4px | 黑色描边 4-5px |
| 背景 | 无 | 可选半透明黑色底 |
| 位置 | 画面下方1/4处 | 画面居中偏下 |
| 对齐 | 居中对齐 | 居中对齐 |

**字幕动画（可选增强）**：
- Hook字幕：「文本动画」→「入场」→「打字机」或「弹跳进入」
- 金句字幕：「文本动画」→「入场」→「缩放弹出」
- 数据字幕：「文本动画」→「入场」→「淡入」

---

### Step 5: 添加BGM

1. 点击「音频」→「音乐库」
2. 按情绪关键词筛选音乐：

| 视频段落 | 搜索关键词 | 风格描述 |
|---------|-----------|---------|
| Hook | 悬疑、紧张、神秘 | 低音渐入、心跳感 |
| 知识铺垫 | 轻快、科技、清新 | 电子节拍、律动感 |
| 核心发现 | 励志、震撼、史诗 | 弦乐渐进、高潮感 |
| CTA结尾 | 温暖、积极、轻快 | 吉他或钢琴收尾 |

3. 选中合适音乐 → 拖入音频轨道 A2（在配音下方）
4. **音量设置**：
   - BGM整体音量：-20dB 至 -25dB
   - 确保不盖过人声配音
   - 方法：选中BGM → 音量条下拉 → 或右键「调节音量」
5. **淡入淡出**：
   - BGM开头：淡入 1-2秒
   - BMG结尾：淡出 2-3秒
   - 段落切换处：轻微淡入淡出过渡

**BGM分段技巧**（同一视频多段不同BGM）：
- 在段落切换处用「分割」切断BGM
- 后一段BGM添加淡入，前一段添加淡出
- 确保两段BGM风格相近，过渡自然

---

### Step 6: 特效与动画增强

#### 图片素材动画

选中图片 → 右侧「动画」面板：

| 动画类型 | 推荐效果 | 适用场景 |
|---------|---------|---------|
| 入场 | 轻微放大/淡入 | 所有图片默认 |
| 出场 | 轻微缩小/淡出 | 所有图片默认 |
| 组合 | 缓慢推近 | 数据图表（强调感）|
| 组合 | 左右平移 | 宽图展示 |

**设置方法**：选中图片 → 动画 → 选择效果 → 调整时长（1-2秒）

#### 转场效果

选中两个素材间的转场图标 → 选择效果：

| 转场类型 | 时长 | 适用场景 |
|---------|------|---------|
| 叠化（Dissolve）| 0.3-0.5s | 默认通用，最常用 |
| 闪黑 | 0.2-0.3s | 段落切换 |
| 闪白 | 0.2s | 高潮顿悟时刻 |
| 模糊溶解 | 0.5s | 柔和过渡 |

#### 画面调色（可选）

选中所有视频素材 → 右侧「调节」：

| 参数 | 科普视频推荐值 | 效果 |
|------|--------------|------|
| 亮度 | +5 至 +10 | 整体提亮 |
| 对比度 | +5 至 +10 | 增强层次感 |
| 饱和度 | +5 至 +10 | 色彩更鲜明 |
| 色温 | 偏冷 -5 | 科技感 |
| 锐化 | +10 至 +15 | 画面更清晰 |

> 注意：图片素材如已设计好配色，谨慎调色避免破坏。

#### 特效添加

- **Hook画面**：「特效」→「画面特效」→「震动」或「闪烁」（短暂0.5秒）
- **数据展示**：「特效」→「画面特效」→「科技边框」或「扫描线」
- **金句时刻**：「特效」→「画面特效」→「聚光灯」或「光晕」

---

### Step 7: 封面设置与导出

#### 封面制作

1. 时间轴拖动到最醒目的画面帧（通常是Hook或高潮画面）
2. 点击播放器右上角「封面」→「设置封面」→「选择帧」
3. 点击「去编辑」添加封面文字：
   - 主标题：视频核心主题（大字、粗体、高对比色）
   - 副标题（可选）：补充说明
4. 封面文字位置：画面中央偏下，避开抖音UI遮挡区
5. 点击「完成」

**封面设计原则**：
- 文字占画面面积 30%-40%
- 颜色对比强烈（深色底+亮色字 或 亮色底+深色字）
- 最多2行文字，每行不超过10个字
- 确保在小尺寸缩略图中也能清晰可读

#### 导出设置

点击右上角「导出」：

| 参数 | 设置值 | 说明 |
|------|-------|------|
| 分辨率 | 1080×1920 | 抖音竖版9:16 |
| 帧率 | 30fps | 标准流畅 |
| 码率 | 推荐/更高 | 推荐即可 |
| 编码 | H.264 | 兼容性最佳 |
| 格式 | MP4 | 通用格式 |
| 文件名 | `[主题]_科普.mp4` | 清晰命名 |

**导出后检查**：
- [ ] 播放检查音画是否同步
- [ ] 检查字幕是否有错别字
- [ ] 检查BGM音量是否合适
- [ ] 检查转场是否流畅
- [ ] 检查视频时长是否符合预期

## BGM推荐库

按情绪分类的剪映音乐库搜索关键词：

### 悬疑/紧张（Hook段）
- 搜索：悬疑、紧张、神秘、侦探
- 推荐风格：低音渐入、心跳节拍、电子氛围

### 轻快/科技（知识段）
- 搜索：科技、轻快、清新、电子
- 推荐风格：明亮合成器、轻快节奏、未来感

### 震撼/史诗（高潮段）
- 搜索：史诗、震撼、励志、宏大
- 推荐风格：弦乐渐进、打击乐高潮、管弦混合

### 温暖/积极（结尾段）
- 搜索：温暖、积极、治愈、轻快
- 推荐风格：吉他、钢琴、柔和节拍

## 常见问题解决

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 音画不同步 | 素材帧率不一致 | 统一素材帧率或手动对齐 |
| 字幕被遮挡 | 位置设置不当 | 调整字幕Y轴位置，避开画面主体 |
| BGM盖过人声 | BGM音量过大 | BMG降至-22dB以下 |
| 图片播放太快 | 默认时长太短 | 手动延长每张图片时长 |
| 导出文件太大 | 码率设置过高 | 选择"推荐"码率 |
| 转场生硬 | 转场时长太短 | 延长至0.5秒以上 |
