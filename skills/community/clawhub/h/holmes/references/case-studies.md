## 一、排除法（Elimination Formula）

**公式：**
```
H1 → 矛盾 （如果假设H1成立，则与已知矛盾）
H2 → 矛盾 （如果假设H2成立，则与已知矛盾）
...
Hn → 无矛盾 （如果假设Hn成立，无矛盾）
∴ Hn （∴ Hn为真）
```

**口诀：** 否定一切不可能，剩余即使难以置信也必定是真相

**说明：** 枚举所有假设，用已知证据逐一排除与证据矛盾的假设，唯一无法排除的即为真相。

**原文依据：**
- 《血字的研究》：凶手不是德国人（排除）→ 不是自杀 → 是谋杀
- 《跳舞的小人》：斯莱尼口供与证据矛盾 → 排除无辜假设

---

## 二、选言推理（Disjunctive Syllogism）

**公式：**
```
A ∨ B （A或B至少有一个成立）
¬A （A不成立）
∴ B （∴ B成立）
```

**口诀：** 否一必肯一

**说明：** 当已知"A或B至少有一个为真"，且确认"A不成立"时，必然得出"B为真"。

---

## 三、假言推理（Conditional Reasoning）

**公式：**
```
A → B （如果A成立，则B成立）
A （A成立）
∴ B （∴ B成立）
```

**口诀：** 肯前必肯后

**说明：** 当前提A成立时，可以直接推出结论B成立。

**原文依据：**
- 《血字的研究》：阿富汗归来→医务特征；海军队服→海军陆战队

---

## 四、逆否推理（Contrapositive Formula）

**公式：**
```
A → B （如果A成立，则B成立）
¬B （B不成立）
∴ ¬A （∴ A不成立）
```

**口诀：** 否后必否前

**说明：** 如果一个假设能成立，它必然导致某个可观察现象；若该现象未出现，则该假设必不成立。

**原文依据：**
- 《血字的研究》：凶手是德国人→德文字体，但RACHE不符合德语正字法→凶手不是德国人
- 《跳舞的小人》：这是恶作剧→埃尔西不会吓死，但埃尔西脸色煞白→这不是恶作剧

---

## 五、传递公式（Transitivity Formula）

**公式：**
```
A → B （如果A成立，则B成立）
B → C （如果B成立，则C成立）
∴ A → C （∴ 如果A成立，则C成立）
```

**口诀：** 链条传递，推理可跳过中间步骤

**说明：** 多个蕴含关系可以串联，形成跨步骤的直接推论。

**原文依据：**
- 《血字的研究》：泥土来源→伦敦区域→凶手活动范围
- 《跳舞的小人》：符号→密码→斯莱尼追踪→威胁埃尔西

---

## 六、必要条件公式（Necessary Condition Formula）

**公式：**
```
B → A （只有A成立，B才成立）
¬A （A不成立）
∴ ¬B （∴ B不成立）
```

**口诀：** 否前必否后

**说明：** A是B的必要条件。如果A不成立，则B必定不成立。

**原文依据：**
- 《银色马》：《银色马案中只有马活着，赌马集团才能操控比赛结果；马被杀了→赌马集团无法操控》
- 《巴斯克维尔的猎犬》：只有斯特普尔顿知道继承人行踪，他才能在合适时机放出猎犬

---

## 七、矛盾触发公式（Contradiction Formula）

**公式：**
```
O1 → H （如果观察到O1，则支持假设H）
O2 → ¬H （如果观察到O2，则否定假设H）
∴ O1 与 O2 不能同时为真（至少有一个需要重新审视）
```

**口诀：** 两个观察指向相反结论，至少有一个需要重审

**说明：** 当不同观察推导出互相矛盾的假设时，提示某个假设的前提有问题。

**原文依据：**
- 《跳舞的小人》：斯莱尼说"从没威胁过埃尔西" vs 密码显示"AM HERE ABE SLANEY"→口供与证据矛盾

---

## 八、因果链构建公式（Causality Chain Formula）

**公式：**
```
E1 → E2 （如果E1成立，则E2成立）
E2 → E3 （如果E2成立，则E3成立）
E3 → ... → En
∴ E1 → En （∴ 链条传递到底）
```

**口诀：** 链条不断，每一步都必须有"因为...所以..."，不允许跳跃

**说明：** 每个结果都有前因，每个前因都应有证据支撑或合理推测。推理链环环相扣，任何一环断开则整链失效。

**原文依据：**
- 《血字的研究》：泥渍→伦敦区域→马车→到达时间→凶手特征
- 《斑点带子》：铃绳新装→不通铃铛→通风口新开→极小孔径→蛇的专用通道

---

## 九、肯定后件（Invalid——注意避免）

**公式：**
```
A → B （如果A成立，则B成立）
B （B成立）
∴ A （∴ A成立）  ← 无效
```

**口诀：** B成立不能证明A成立

**说明：** 这是常见推理谬误。福尔摩斯的方法论强调——只能通过否定后件来否定前件，不能通过肯定后件来肯定前件。信息不足时，可作为工作假设使用，但需标注置信度并持续验证。

---

## 十、或然性推理公式（Probabilistic Reasoning）

**公式：**
```
P(H1|E) > P(H2|E) （假设H1的概率高于假设H2）
P(H2|E) > P(H3|E)
∴ 暂时采用 H1，但持续验证
```

**口诀：** 最大概率假设优先，但保持验证警觉

**说明：** 当确定性和排他性都不足时，用概率最高假设作为工作假设，但标注置信度。

---

## 十一、异常锚定公式（Anomaly Anchoring Formula）

**公式：**
```
O普通 （普通观察，支持常规假设）
O异常 （异常观察，与常规假设矛盾）
∴ 优先分析 O异常
```

**口诀：** 异常不是干扰，而是突破口

**说明：** 最平淡无奇的案子往往最难——正是异常细节降低了问题难度，提供了关键突破口。

**原文依据：**
- 《血字的研究》："其实，最平淡无奇的案子往往最难解……正是由于异乎寻常的细节，案子的难度系数不仅没有增加，反而降低了"
- 《银色马》：《主人拒绝赎金》vs 马被偷→优先分析拒绝赎金这一反常点

---

## 十二、主动验证公式（Active Verification Formula）

**公式：**
```
缺失信息 I
设计验证 V
V结果X → 支持 H1
V结果Y → 排除 H1
```

**口诀：** 主动验证，不被动猜测

**说明：** 当关键信息缺失时，主动设计验证方案。即使验证失败，失败本身也是新信息。

**原文依据：**
- 《血字的研究》：登失物招领启事，用戒指作诱饵引出凶手
- 《血字的研究》：两粒药丸实验——第一粒无毒→修正假设→第二粒验证成功

---

## 十三、反向工程公式（Reverse Engineering Formula）

**公式：**
```
结果 S
若 S 为真 → 需要满足 P1, P2, P3
验证：P1? P2? P3?
若任一不满足 → S 为假
```

**口诀：** 从结果倒推前提，逐一验证每个前提

**说明：** 先接受一个结果为真，反推它必须满足的前提条件，再验证这些前提是否实际成立。

**原文依据：**
- 《血字的研究》：从地板脚印分布→重建凶手行为顺序（站立位置→激动程度→写字顺序）
- 《斑点带子》：从固定地板+铃绳+通风口→重建当晚蛇进入路径

---

## 十四、归谬法（Reductio ad Absurdum）

**公式：**
```
假设 A
若 A 成立 → B
B → 矛盾（与已知事实矛盾）
∴ A 不成立
```

**口诀：** 假设推出矛盾，假设必假

**说明：** 先假设某个命题为真，如果能推导出明显的矛盾结论，则该命题为假。

**原文依据：**
- 《银色马》：《假设马被杀了→马主不会拒绝赎金，但马主拒绝了→矛盾→马还活着》
- 《跳舞的小人》：假设斯莱尼是无辜的→他不应该知道密码含义，但O4显示他主动回复→矛盾→斯莱尼涉案
- 《斑点带子》：假设"带子"指吉普赛人→多人行动无法实现密室，但密室封堵完好→矛盾→"带子"不是吉普赛人
