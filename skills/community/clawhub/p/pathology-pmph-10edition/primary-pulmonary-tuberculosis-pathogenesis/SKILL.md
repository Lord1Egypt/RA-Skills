# 原发性肺结核病理发展过程分析

## 适用场景
- 患者为儿童或未曾感染结核分枝杆菌的成人
- 有明确的初次吸入结核分枝杆菌暴露史
- 胸部影像显示哑铃状阴影或其他提示原发复合征的表现
- 需评估病灶可能的转归（自愈、进展或播散）

## 执行步骤

1. **确认初始感染定位**：检查是否在肺通气良好区域（上叶下部或下叶上部近胸膜处）形成直径1.0～1.5cm的原发病灶（Ghon灶），表现为灰白色炎性实变伴中央干酪样坏死。

2. **评估淋巴播散情况**：判断是否存在沿淋巴管向肺门淋巴结的细菌播散，导致结核性淋巴管炎和肺门淋巴结肿大伴干酪样坏死。

3. **识别原发复合征**：综合肺原发病灶、淋巴管炎及肺门淋巴结结核三要素，确认是否构成原发复合征；影像学上典型表现为哑铃状阴影。

4. **预测临床转归**：
   - 若患者营养状况良好且无合并感染，预计约95%病例在细胞免疫建立后（感染后30～50天）病灶纤维化或钙化而自愈；
   - 若存在营养不良、免疫抑制或合并其他感染，警惕病灶扩大、空洞形成、粟粒性肺结核或全身播散风险；
   - 持续进展的肺门淋巴结病变可能发展为支气管淋巴结结核。

5. **特殊人群注意**：对免疫功能严重受损的成人，即使非儿童，也可能表现为原发性肺结核，因其缺乏既往致敏状态。

> 注意：本技能不适用于既往感染过结核或卡介苗接种后再次感染者。
