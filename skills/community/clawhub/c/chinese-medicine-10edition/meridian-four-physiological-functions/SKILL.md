# Meridian Four Physiological Functions

## When to Use
- 分析疾病的发生、发展与传变过程（如外感病由表入里）
- 解释人体脏腑与体表组织之间的功能联系
- 阐明针灸治疗中“得气”“行气”的作用机制
- 理解中医辨证体系（如卫气营血辨证）的经络基础

## Execution Steps

### 1. 沟通内外，网络全身
- 引用《灵枢·海论》：“十二经脉者，内属于府藏，外络于支节”
- 十二经脉及其经别连接体表与脏腑、脏腑之间
- 十五络脉强化体表与体表、体表与脏腑的联系
- 奇经八脉加强各经脉间的横向联络
- 标本、气街、四海结构强化人体前后腹背及头身分段的整体性
- 最终形成以头身四海为总纲、十二经为主体、三百六十五络遍布全身的有机网络

### 2. 运行气血，协调阴阳
- 引用《灵枢·本藏》：“行血气而营阴阳，濡筋骨，利关节”
- 气血依赖经络输布至全身，为生命活动提供物质基础
- 宗气、原气、营气、卫气均通过经络营运周身
- 实现“内溉藏府，外濡腠理”（《灵枢·脉度》）
- 维持“阴平阳秘，精神乃治”（《素问·生气通天论》）的动态平衡

### 3. 抗御病邪，反映证候
- 孙络“以溢奇邪，以通营卫”（《素问·气穴论》），为抗邪第一线
- 病邪传变路径：皮毛 → 孙脉 → 络脉 → 经脉 → 内连五脏，散于肠胃（《素问·缪刺论》）
- 温病学派据此建立“卫、气、营、血”辨证体系
- 经络可反映局部、一经、数经或整体证候（如太阳病见“热结膀胱”，阴虚见五心烦热、盗汗）

### 4. 传导感应，调整虚实
- 针灸“得气”“行气”是经络传导感应的直接体现
- 经络具有双向调节能力，引导机体向康复方向转化
- 通过激发经络功能、疏通经气，实现“泻其有余，补其不足，阴阳平复”（《灵枢·刺节真邪》）

> **注意**：本技能仅适用于中医理论框架，不直接对应现代医学病理机制。使用前需确保用户问题基于中医语境。
