# Platform · 小红书 Playbook

> 拆解视角 + 平台规则 + 观察日志。拆解时对照这里判断"是否符合 XHS 平台逻辑"。

## 平台属性

- 用户：女性占比高，决策周期长，重收藏 / 重对比
- 内容形态：图文（封面权重最高）+ 短视频（横屏少、竖屏多）
- 算法关注：完播率 / 收藏率 / 评论率 > 点赞率
- 流量公式：好封面 + 好钩子 + 收藏点

## 拆解时重点看

| 维度 | 看什么 | 影响哪些字段 |
|---|---|---|
| **封面** | 信息密度、字体、对比、是否第一秒看清主题 | "参考封面文案"、"情绪钩子" |
| **标题** | 是否有数字 / 反差 / 身份感 / 地点 | "参考发布主题" |
| **正文结构** | 钩子句 → 痛点共鸣 → 干货展开 → 行动引导 | "旁白逻辑分析"、"可学习要点" |
| **标签** | 大词 + 中长尾 + 地点 + 痛点 | "参考热门标签" |
| **评论区** | 用户问什么 / 求什么 / 夸什么 | "评论关键词" |

## 平台爆款公式（XHS 高频结构）

### 图文常见
- **18 图穿搭合集** — 模特图 / 平铺图按场景分组
- **门店探店** — 第一张大场景 + 后续单品 + 最后一张地址
- **种草卡片** — 第一张大字封面 + 后续多角度产品图

### 视频常见
- **沉浸式陈列** — 无人声、纯画面 + BGM + 字幕信息
- **人设访谈** — 主理人出镜 + 反差开场 + 价值观结尾
- **产品工艺特写** — 手部 / 材质 / 制作过程 + 文化寓意

## 语气适配（DNA → XHS）

> 引用 [[brand/brand-voice]]，做平台落地：
- 标题允许 1 个 emoji 做点缀（💚✨📍 偏审美感的）
- 正文用「家人们 / 姐妹们」这类称呼时**克制**——一篇用一次就够
- 段落短、空行多、视觉留白
- CTA 用 📍 引地址，不用「快冲」「必入」这类词

## 钩子模板（XHS 专版）

> 通用钩子见 [[engine/hooks]]。XHS 特别管用的：
- **地点 + 反差**：「在 X 挖到 Y 的天花板」
- **数字 + 对比**：「N 后开了 X 平大店」
- **身份 + 体验**：「把 X 穿在身上是什么体验」
- **对比 + 设问**：「同样是 X，为什么有人 Y」

## 禁忌（XHS 平台）

> 通用禁忌词见 [[engine/taboo]]。XHS 特别要避开的：
- 不要在文案里直接写商品价格（容易限流）
- 不要带店铺微信号（评论引导即可）
- 不要带"最 / 第一 / 唯一"等极限词
- 不要在标题用太多营销 emoji（💰🔥🎁 容易显廉价）

### 2026-04-26 · 本地服务型短视频公式
- 来源：AIC-260426-002（皮小客武汉改衣店，14s 视频）
- 公式：地点+品类定位（0-2s 顶部大字）→ 服务能力罗列（3-6s 中部字幕）→ 痛点钩子（7-10s "贵 + 不要扔"）→ 工艺特写（11-12s 极特写剪刀缝纫）→ 成品+年限背书（13s+ 水印）
- 互动表现：4 赞 / 1 藏 / 2 评 / 2 转（极低），**但 1 条评论是高质量精准询问**（"羽绒服袖子抬手有点紧可以改大吗 什么价格"）
- 字幕分层：顶部固定品牌定位 + 底部滚动具体内容，避免信息拥挤
- 推论：结构正确但缺"破圈钩子"——前 3 秒平庸只能拿精准流量不能拿泛流量。适合本地服务账号做日常输出，月度建议加 1 条「具体案例 + 价格反差」开场的种草感视频拉曝光
- 适用品类：本地服务（改衣 / 美甲 / 皮具护理 / 修表 / 干洗 / 二手翻新）
- 配合：必须打地点标签（缺这个等于放弃同城推流）；评论必须 24h 内人工回复

### 2026-04-26 · 沉浸式陈列短视频公式
- 来源：AIC-260426-001（见花开新中式陈列）
- 观察：34 秒纯陈列、无人声、无字幕条（仅底部固定 watermark），全靠 BGM + 镜头节奏
- 互动表现：330 赞 / 231 藏 / 133 评，**藏赞比 70%**（XHS 上藏赞比 >50% 视为高保存价值，说明用户认为"以后会用得上/想再看"）
- 镜头节奏：全景 → 单品（绿马甲特写）→ 单品（白衬衫油菜花特写）→ 单品（绿裙特写）→ 全景收尾
- 推论：高客单 / 高审美品类适用此公式；不适用低客单冲动消费类目
- 配合：发布文案要长且画面化（见花开这条 desc 351 字，描绘"祖母绿真丝"等具体材质细节），补足无声画面的信息缺口
