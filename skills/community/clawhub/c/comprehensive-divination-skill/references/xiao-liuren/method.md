# 小六壬推演方法

## 核心原理

小六壬又称"马前课"、"掐指一算"，是最简捷的占卜法。通过在左手掌上定位六宫，以月、日、时三个时间点为坐标进行推算，最终落在哪个宫位即得吉凶。

## 六宫布局（顺时针）

```
      大安 (1)
空亡 (6)    留连 (2)
小吉 (5)    速喜 (3)
      赤口 (4)
```

定位口诀：大安起正月，月上起日，日上起时。

## 推算步骤

### 第一步：月上起日
1. 从大安起正月（农历正月）
2. 顺时针数到当前月份，落位记为 A
3. 从 A 起初一
4. 顺时针数到当前日期（农历日），落位记为 B

### 第二步：日上起时
1. 从 B 起子时（23:00-1:00）
2. 顺时针数到当前时辰
3. 最终落位即为此事宫位 C

## 算法规则

- 六宫循环：1-2-3-4-5-6 → 1-2-3...（无上限循环）
- 月数 = 当前农历月份数字
- 日数 = 当前农历日期数字
- 时数 = 时辰序数（子=1, 丑=2, ..., 亥=12）
- 每一步计算：起始宫位序号 + 点数 - 1，结果 mod 6（余0=6）

## 用户报数模式

如果用户直接报三个数字（如 3、8、15），跳过月份日期自动推算，直接使用：
- 从大安起，数第一个数 → 落宫 A
- 从 A 起，数第二个数 → 落宫 B
- 从 B 起，数第三个数 → 最终落宫 C

## 输出格式

推演完成后按以下格式输出：

1. **推演过程**：展示月→日→时三步推算的落宫变化
2. **最终落宫**：宫名 + 吉凶等级 + 五行属性 + 数字信息
3. **总断**：该宫的总断语
4. **分事断**：对照用户问事类型，给出对应领域的断语（参考 references/xiao-liuren/liugong.md）
5. **五行补充**：结合季节的五行生克，给出补充建议
