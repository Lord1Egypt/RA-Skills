This reference provides complete structural templates for all 10 report types. For live writing, load the relevant template below.

## Part A: Daily Reports (5 Types)

### 0. Investigation & Analysis Report (调研分析报告/专题调研报告)

**Purpose**: In-depth investigation and multi-dimensional analysis of emerging phenomena. Combines factual discovery, legal framework analysis, social/political/economic/cultural impact assessment, and risk evaluation. Best for: crypto/blockchain investigations, emerging technology risks, cross-border financial phenomena, cult/religious-linked activities, new social trends, policy-adjacent phenomena.
**Length**: 2,000-5,000 chars
**Style**: 调研报告 tone — factual, investigative, legal, multi-dimensional impact analysis, risk-forward

#### Template

```
Title: Investigation & Analysis of [Core Phenomenon/Threat] — Full descriptive title

[Opening paragraph — 工作发现/调研发现]
"Work has found that, since [time period], driven by [key actors], [phenomenon] has emerged..."
Summarize: core finding + timeline + key actors + scale + risk implication in 3-5 sentences.

一、[Core Phenomenon] Basic Facts (基本情况)
  一是 [Fact category 1 — detailed description of first dimension].
    - Background, origin, key milestones, scale, key data points
    - Cite specific events, actors, timelines, numerical data
    
  二是 [Fact category 2 — second dimension].
    - Additional factual dimensions
    - Parallel descriptions where applicable

二、Operation / Dissemination / Business Model Analysis (运营推广模式)
  一是 [Model characteristic 1 — how it operates].
    - Describe the mechanism: cultural packaging, authority endorsement, community-driven, etc.
    - Use specific examples
  
  二是 [Model characteristic 2].
  
  三是 [Model characteristic 3].

三、Domestic / Local Impact Assessment (境内传播/本地影响)
  一是 [Impact category 1 — specific manifestation within jurisdiction].
    - Platform, actors, scale, channels
  
  二是 [Impact category 2].
  
  三是 [Impact category 3].

四、Social-Political-Economic-Cultural Impact Analysis (社会政治经济文化影响)
  一是 [Social impact — effects on population groups, social stability, public sentiment].
    - Specific affected populations
    - Social stability implications
    - Potential for group incidents or mass events
  
  二是 [Political impact — effects on governance, policy, institutional credibility].
    - Government credibility implications
    - Policy enforcement challenges
    - Political stability dimensions
  
  三是 [Economic impact — effects on industry, markets, financial security].
    - Industry-specific economic consequences
    - Market disruption potential
    - Financial security risks
  
  四是 [Cultural impact — effects on values, norms, social fabric].
    - Cultural value implications
    - Social norm shifts
    - Ideological dimensions

五、Risk Assessment (风险评估)
  一是 [Risk 1 — detailed risk analysis with legal basis].
    - Legal/regulatory framework: cite specific laws, regulations, notices
    - Risk mechanism: how the threat operates
    - Consequences: impact on stakeholders
    - Cross-reference with impact analysis above
  
  二是 [Risk 2].
  
  三是 [Risk 3].
  
  四是 [Risk 4 if needed].

六、Countermeasures & Recommendations (对策建议)
  针对以上风险建议：
  一是 [Recommendation 1 — responsible body + specific action + expected outcome].
  二是 [Recommendation 2].
  三是 [Recommendation 3].
  四是 [Recommendation 4 if needed].
```

#### Key Features of This Report Type

| Feature | Description | Example from Sample Document |
|---------|-------------|------------------------------|
| **Opening formula** | "工作发现" / "调研发现" + time + actors + core finding | "工作发现，自2025年3月以来，在阿联酋主权基金MGX、币安创始人赵长鹏等的推动下..." |
| **一是/二是/三是 enumeration** | Each major section uses this nested structure | "一是' Mubarak'虚拟币基本情况..." / "二是' Eid Mubarak'虚拟币基本情况..." |
| **Legal citation** | Cites specific laws, regulations with full title and date | "2026年2月，中国人民银行等八部门联合发布《关于进一步防范和处置虚拟货币等相关风险的通知》..." |
| **Fact-first structure** | Facts before analysis — establishes foundation before evaluating | Sections 一→二→三 are factual; Sections 四→五→六 are impact+risk+recommendations |
| **Bilingual naming** | Important terms in both Chinese and English on first mention | "Mubarak（中文名称'穆巴拉克币'）" |
| **Data precision** | Specific numbers, dates, platform names | "6小时内突破4000万美元" / "一天内交易量迅速超过7亿美元" |
| **Multi-dimensional impact** | Social/political/economic/cultural analysis in separate section | Social (affected groups, stability), Political (governance, policy), Economic (markets, financial security), Cultural (values, norms) |

#### When to Use vs. Other Report Types

| Scenario | Use This | Use Other |
|----------|----------|-----------|
| Emerging tech/finance phenomenon | ✅ Investigation & Analysis | ❌ Rapid Assessment (too shallow) |
| New regulatory risk area | ✅ Investigation & Analysis | ❌ Special Report (sentiment-focused) |
| Single breaking news event | ❌ Rapid Assessment | ✅ |
| Need deep sentiment analysis | ❌ Special Report | ✅ |
| Monthly summary | ❌ Monthly Report | ✅ |
| Cross-border financial threat | ✅ Investigation & Analysis | — |

#### Writing Style Notes
- **一是/二是/三是** is the preferred enumeration format (vs. First/Second/Third or 1/2/3)
- Opening must establish investigative posture: "工作发现" / "走访发现" / "网络巡查发现" / "调研发现"
- Legal citations: full regulatory document title + issuing body + date + article number
- Foreign terms: first mention in both original and Chinese translation
- Risk section transitions logically from factual findings
- Recommendations are concrete, assigned to specific responsible bodies, actionable, and measurable

### 1. Rapid Assessment Report (快报)

**Purpose**: Quick assessment of breaking events
**Length**: 800-1,500 chars

#### Template

```
Title: [Brief Event Description] — Risk Assessment

I. Incident Summary
  - Core elements (time, location, actors, cause, sequence)
  - Current communication data (message volume, primary platforms, peak metrics, views/plays)
  - Cite initial reporting media and key dissemination nodes

II. Risk Assessment
  - 3-5 risk points using "First/ Second/ Third" format
  - Each point: trigger condition → risk behavior → consequence

III. Recommendations
  - 3-5 action items using "First/ Second/ Third"
  - Cover: online monitoring, offline response, coordinated control, root-cause prevention, media engagement
```

#### Subtypes

**Type A (Event-driven)**: Specific exposed incident exists
- Includes communication data (message count, platform distribution, views)

**Type B (Early-warning)**: Only sparse info but high potential risk
- Add "Communication Status" subsection (when data is limited)
- Focus on escalation pathways and potential coordinated amplification

### 2. Special Report (专报)

**Purpose**: Systematic deep-dive on a specific incident or phenomenon
**Length**: 2,000-4,000 chars

#### Template

```
Title: [Event] Online Sentiment Analysis Report

I. Overview / Communication Analysis
  (A) Escalation Phase (time period)
  (B) Explosion Phase (time period)
  (C) Decline Phase (time period)
  → Each phase: key dissemination node + specific data + sentiment characteristics

II. Related Analysis (selective, based on event type)
  - Account analysis (self-media-driven events) — follower count, positioning, content, influence
  - Platform analysis
  - Sentiment tendency analysis
  - Netizen opinion distribution

III. Risk Assessment
  - 5-8 risk points

IV. Response Actions Taken (if any)
  - Step-by-step measures

V. Lessons Learned / Recommendations
```

#### Subtypes

**Type A (Event-driven)**: Single incident full tracking
**Type B (Phenomenon-oriented)**: Multiple related events aggregated
**Type C (Risk-warning)**: Other-location incident → local risk implications

### 3. Deep Research Report (研报)

**Purpose**: Policy-oriented deep research with industry + sentiment analysis
**Length**: 3,000-6,000 chars

#### Template

```
Title: Risk/Impact/Countermeasure Analysis of [Core Issue]

I. Overview / Recent Developments / Risk Landscape
  - Macro background (2-3 sentences identifying core tension)
  - Specific events/dynamics enumerated
  - Cite specific media reports, data, timestamps

II. Local Impact & Risk / Risk Analysis
  - 5-8 risks, 200-300 chars each
  - Each: nature of risk → specific manifestation → deeper consequences
  - Include trend assessment and cascade effects
  - Cover both near-term and structural risks

III. Recommendations
  - Macro-strategic level
  - Mid-level institutional mechanisms
  - Micro-operational level
```

#### Subtypes

**Type A (Assessment-oriented)**: Risk assessment as spine + recommendations
**Type B (Warning-reference)**: Other-location event as lead-in → local risk warning

### 4. Retrospective Report (复盘)

**Purpose**: Post-mortem analysis of a major incident
**Length**: 2,000-4,000 chars

#### Template

```
Title: [Event] Online Sentiment Retrospective

I. Incident Overview
  - Full picture (time, location, actors, nature, outcome)

II. Sentiment Development & Response Process
  (A) Initial Phase (time period)
    - Initial platform, account, time
    - Initial communication data
    - Key netizen opinions/emotions
  (B) Escalation Phase (time period)
    - Key forwarding nodes (media intervention, influencer sharing)
    - Escalation data (shares, views, topic heat)
    - Sentiment escalation characteristics
  (C) Decline Phase (time period)
    - Cooling factors (control / natural decay / official response)
    - Decline data (growth rate decrease, etc.)

III. Retrospective Findings
  - 4-6 findings
  - Focus on: communication patterns, platform characteristics, public psychology, response outcomes, institutional issues
  - Each finding backed by data or case evidence

IV. Lessons & Recommendations
  - 5-8 actionable improvement suggestions
```

### 5. Monthly Report (月报)

**Purpose**: Monthly sentiment overview + next-month risk forecast
**Length**: 2,500-4,000 chars

#### Template

```
Title: [Month] Online Sentiment Overview for [Entity]

Introduction:
  - Overall monthly assessment (stable / complex / severe)
  - Key positive areas
  - Main negative categories
  - Risk nodes for next month

I. Positive Sentiment Highlights
  - 5-8 topics
  - Each: topic summary + 2-3 specific report examples

II. Negative Sentiment Information
  - Categorized (e.g., business operations, corruption, academic misconduct, livelihood, rights defense, safety accidents, etc.)
  - Each category: 1-3 case examples
  - Each case: media source + event summary + brief sentiment assessment

III. Next-Month Risk Forecast
  - 8-10 forecast items
  - Link to calendar events (holidays, sensitive anniversaries, meetings), seasonal patterns, social mood
  - Each: trigger → possible manifestations → prevention suggestions
```

### 6. International/Foreign Pressure Report

**Purpose**: Analysis of foreign pressure, trade friction, geopolitical sentiment
**Length**: 3,000-6,000 chars

#### Template

```
Title: New Developments in [Core Issue] — Impact on China and Countermeasures

Introduction (strategic assessment):
  - Overall situation in one sentence
  - Core characteristics of pressure/stigmatization
  - General response approach

I. Recent Developments / Foreign Narratives
  - Enumerate specific events and dynamics
  - Each: time + actor + specific action + underlying intent
  - Cite specific countries, institutions, media, legislation names and data

II. Local Impact & Risk Analysis
  - 5-8 risks, 200-300 chars deep dive each
  - Logic: risk nature → specific impact → deeper consequences
  - Distinguish near-term shocks from structural risks

III. Recommendations
  - 4-6 items
  - Macro-strategic: supply chain autonomy, standard-setting power
  - Mid-level: early warning mechanisms, diversification
  - Micro-operational: narrative struggle, diplomatic engagement
```

### 7. Ethnic/Religious/Ideological Report

**Purpose**: Analysis of ethnic policy, anti-China narratives, ideological confrontation
**Length**: 2,500-5,000 chars

#### Template

```
Title: Foreign Forces' "XX" Narrative Smearing [China Policy/Region]

I. Foreign Narrative Landscape
  - Categorized by actor (5-6 categories):
    1. Foreign media attack angles
    2. Human rights orgs / anti-China groups
    3. Separatist forces
    4. Foreign governments / politicians
    5. NGOs / think tanks
  - Each: 2-4 specific cases
  - Direct quotes from foreign media (with brief rebuttal)

II. Risk Analysis
  - 3-5 deep-dive risk points
  - Cover: ideological security, social stability, diplomatic environment, economic sanctions
  - Note domestic-foreign linkages

III. Recommendations
  - 3-5 items
  - Cover: domestic publicity/legal education, external narrative struggle, targeted countermeasures, border stability
```

### 8. Industrial-Economic Security Report

**Purpose**: Supply chain security, trade barriers, economic sanctions sentiment analysis
**Length**: 3,000-6,000 chars

#### Template

```
Title: [Issue] — Requires Urgent Attention

I. Overview
  (A) Reasons for export/industry obstruction
    - Political factors (legislation, sanctions)
    - Narrative factors (stigmatization)
    - Market factors (trade protectionism)
  (B) Foreign narrative landscape
    - Media narratives
    - Anti-China faction initiatives
    - Supply chain relocation moves

II. Local Risk & Vulnerability Analysis
  - 4-6 deep-dive risks
  - Each: risk → specific data → deeper consequences
  - Cover: market share, employment, social stability, supply chain security, demonstration effect

III. Recommendations
  - 4-6 systematic recommendations
  - Cover: relief measures, market diversification, traceability certification, countermeasures
```

### 9. Governance Innovation / Social Phenomena Report

**Purpose**: Dialectical analysis of social governance innovation and emerging trends
**Length**: 2,500-5,000 chars

#### Template

```
Title: Sentiment Impact Analysis of [Phenomenon]

I. Characteristics of [Phenomenon] in Practice
  - 5-7 dimensions, each 100-150 chars
  - Each: summary + specific case + effect/significance

II. Sentiment Impact
  - Positive impacts (2-3): dimension + case + public feedback
```

## Part C: General Writing Standards

### Language Style
- Formal, rigorous, objective official document style
- No emotional expressions; use "requires vigilance", "should monitor", "attention needed"
- Data analysis precise to specific numbers

### Structure Elements
- Level 1 headings: I, II, III, IV, V
- Level 2 headings: (A)(B)(C) or 1. 2. 3.
- Level 3 enumeration: First/Second/Third or 1./2./3.
- Each paragraph has a clear topic sentence at head

### Risk Analysis Three-Layer Model (Core)

| Layer | Function | Example Phrase |
|-------|----------|----------------|
| **1. Trigger/Context** | Sets the stage | "Currently...", "As X approaches..." |
| **2. Risk Behavior + Actors** | Who does what | "Some netizens may...", "Self-media accounts..." |
| **3. Consequences** | Chain of impact | "Leading to...", "Exacerbating...", "Causing..." |

### Recommendation Writing
- Each: **Responsible body + Specific action + Expected outcome**
- Macro before micro
- Online and offline in parallel
- Both symptoms and root causes
- Quantity guidelines: Rapid 3-5, Special/Retro 5-8, Research 4-6, Monthly 8-10

### Common Official Stock Phrases

| Context | Chinese Stock Phrase | Meaning |
|---------|---------------------|---------|
| Opening | 密切关注 / 高度重视 / 需警惕 | Close monitoring / high priority / vigilance needed |
| Development | 持续发酵 / 迅速升温 / 引发热议 | Continuous escalation / rapid heating / sparking discussion |
| Response | 及时回应 / 正面引导 / 妥善处置 | Timely response / positive guidance / proper handling |
| Goal | 防止扩散 / 遏制升级 / 维护稳定 | Prevent spread / contain escalation / maintain stability |
| Characterization | 恶意炒作 / 煽动性信息 / 不实信息 | Malicious hype / inciting content / false information |
| Measures | 强化监测 / 精准引导 / 协调管控 | Strengthen monitoring / precise guidance / coordinated control |

### Citation & Data Format
- Media reports: "On [date], [media outlet] reported that..."
- Online posts: "[Platform] user '[handle]' posted that..."
- Communication data: "As of [datetime], total relevant posts: X, including video X, news client X..."
- Topic heat: "[Weibo] topic [#hashtag] has been read X million times"

### Sensitive Information Handling
- Replace specific city names with "xx" (maintain consistency throughout)
- Foreign force descriptions: stay objective, cite original text, add rebuttal context
- Individual privacy: use surnames or "xx" as needed
