# 和解协议书

## 适用场景
发生民事纠纷的当事人之间，在未经法院判决/仲裁裁决的情况下，通过协商自愿达成一致意见，就争议事项的解决方式（如赔偿金额、履行方式、责任划分等）签署书面协议。常见于：人身损害赔偿、合同违约、交通事故赔偿、劳动争议、相邻权纠纷等各类民事纠纷。可由当事人自行签署，也可在律师协助下签署。

## 提交机构
无需提交特定机构（当事人各执一份保存）；如需强制执行，可向法院申请司法确认或凭和解协议提起诉讼。

## 模板正文

### 通用版和解协议书

```
                        和 解 协 议 书

甲方（赔偿义务人/债务人-自然人）：
  姓名：________  性别：____  身份证号：____________________
  住址：________________________________________
  联系电话：________________

甲方（赔偿义务人/债务人-单位）：
  单位名称：________________________
  统一社会信用代码：__________________
  法定代表人：________  职务：________
  住所地：____________________________
  联系电话：________________

乙方（赔偿权利人/债权人-自然人）：
  姓名：________  性别：____  身份证号：____________________
  住址：________________________________________
  联系电话：________________

（注：如有多个甲方或乙方，请逐一列明。文中甲方/乙方的表述仅为方便叙述，不代表权利义务的强弱判断。）

鉴于：
1. 甲乙双方于____年____月____日因 [简述纠纷起因，如：驾驶车辆在XX路段发生碰撞 / 签订《XX合同》后发生履行争议 / 劳动关系解除产生争议 / 其他情形] ，产生纠纷；
2. 双方本着自愿、平等、互谅互让的原则，经过友好协商，就该纠纷的解决达成如下协议，以资共同遵守。

第一条  基本事实确认

双方共同确认以下事实：
1. ________________________________________；
2. ________________________________________；
3. ________________________________________。

（注：签署前双方确认的事实写清楚，防止事后一方否认）

第二条  赔偿/补偿/给付金额及方式

1. 甲方同意向乙方支付 [款项性质，如赔偿款/补偿款/货款/借款本息] 共计人民币（大写）__________________元整（¥________元）。

2. 上述款项包含但不限于：[列举费用构成，如医疗费、误工费、护理费、营养费、后续治疗费、精神损害抚慰金、合同价款、违约金等] ，为一次性了结全部争议的最终金额。

3. 支付方式：
   □ 一次性支付：甲方于____年____月____日前向乙方一次性付清上述全部款项。
   □ 分期支付：
      - 第一期：____年____月____日前支付 ¥________元；
      - 第二期：____年____月____日前支付 ¥________元；
      - 第三期：____年____月____日前支付 ¥________元。

4. 支付账户：
   乙方指定收款账户：
   开户行：________________
   户名：__________________
   账号：__________________

第三条  违约责任

1. 若甲方未按本协议约定的时间和金额支付款项，每逾期一日，应按逾期金额的____%（建议0.05%-0.1%/日，即年化约18%-36%）向乙方支付违约金。

2. 若甲方逾期支付超过____日，乙方有权要求甲方一次性支付全部剩余款项，并有权就本协议全部未履行部分向人民法院申请强制执行/提起诉讼。

3. 甲方履行完毕本协议项下的全部义务后，乙方不得就本协议所涉纠纷再以任何形式向甲方主张任何权利。

第四条  权利义务了结

1. 甲方按本协议约定足额支付全部款项后，乙方确认双方的上述纠纷一次性了结，双方之间的民事权利义务关系就此终结。

2. 乙方放弃就上述纠纷向行政机关投诉举报、向人民法院提起诉讼、向仲裁委员会申请仲裁等一切权利（法定不得放弃的权利除外）。

3. 乙方应在收到全部款项后____日内，向 [相关机构，如法院] 申请撤回对甲方的 [起诉/仲裁申请/执行申请] （如有）。

第五条  保密条款（可选）

双方确认，本协议的存在及内容属于双方的保密信息，除按照法律要求所需外，任何一方不得向任何第三方披露。违反保密义务的一方应赔偿由此给对方造成的一切损失。

第六条  争议解决

因本协议产生的或与本协议有关的任何争议，双方应首先友好协商解决；协商不成的，任何一方有权向 [甲方/乙方] 住所地有管辖权的人民法院提起诉讼。

第七条  其他条款

1. 本协议自双方签字（甲方为单位的须加盖单位公章）之日起生效。
2. 本协议一式____份，甲方持____份，乙方持____份，各份具有同等法律效力。
3. 本协议的修改、补充须经双方协商一致并签署书面文件。
4. [其他约定事项，如：甲方另有身份证复印件作为本协议附件等]

甲方（签名/盖章）：__________________
                                      ____年____月____日

乙方（签名/盖章）：__________________
                                      ____年____月____日

见证人（如有）：
姓名：________  身份证号：____________________
签名：__________________  日期：____________

见证人2（建议律师或双方信任的第三方）：
姓名：________  身份证号：____________________
签名：__________________  日期：____________
```

### 特定场景补充条款

#### 人身损害赔偿（交通事故/伤害/医疗纠纷）

```
补充条款（插入"第一条  基本事实确认"之后）：

事故经过：
____年____月____日____时许，甲方________（简述事故经过）导致乙方________受伤。
乙方于____医院治疗，诊断为：________________。
截至____年____月____日，乙方已发生的医疗费为 ¥________元。

除第二条所列赔偿款项外，甲方另承担/垫付：
（注：如甲方已先行垫付部分费用，在此写明确认）

乙方确认：签署本协议时，对本协议所涉法律后果有充分认识，系在自愿、知情的情况下签署，今后不以"后续治疗费增加"/"伤情加重"等任何理由再向甲方主张额外赔偿。
```

#### 劳动争议和解

```
补充条款：

双方确认劳动关系于____年____月____日解除。

本协议约定的款项支付完毕后，双方之间不存在任何未了结的劳动报酬、经济补偿金/赔偿金、社会保险、公积金、年休假、加班费及其他一切劳动争议。

乙方确认放弃就与甲方劳动关系存续期间及解除后的任何劳动争议向劳动行政部门投诉、申请劳动仲裁或提起诉讼的权利。
```

#### 合同纠纷和解

```
补充条款：

双方确认自本协议生效之日起，《XX合同》自动终止，双方恢复至合同履行前的状态或维持XXXX状态。

甲方按本协议约定支付款项后，乙方应向甲方返还/移交：[对方已交付的原始合同/发票/证照/设备等]。

双方互不追究《XX合同》项下的违约责任。
```

## 注意事项

### 关键填写点
1. **事实确认条款**：签署前先把双方认可的共同事实写下来，防止事后赖账
2. **款项包含范围要穷尽**：写明赔偿/补偿款具体包含哪些项目，防止事后又主张遗漏项目
3. **支付时间要精确**：写具体日期不要只写"尽快"或"N日内"
4. **违约责任要保留**：对方按期付款是和解的价值所在，必须约定违约后果
5. **"一揽子了结"条款**：第二条/第四条的组合，确保事后不能再就同一事件主张权利
6. **见证人签名**：非亲属的第三方见证人有助于事后证明协议是自愿的、非胁迫的

### 常见错误
- 忘记了结条款——赔偿付了但没说"事后不得再追究"，对方可能事后再起诉/索赔
- 款项构成不明确——笼统写"赔偿XX元"之后对方可能就某一费用再主张
- 没有违约责任条款——对方不付款你没有直接的执行办法
- 重大误解/显失公平——赔偿金额与损害严重失衡的，事后可能被法院撤销（《民法典》第147条/第151条）
- 伤情未鉴定就和解——涉及人身损害的，建议先鉴定/等伤情稳定后再签协议，否则"后续并发症/后遗症"可能不被免责条款覆盖
- 违反法律强制性规定——不得预先免除工伤赔偿/人身伤害的基本保障（合同中预先免除人身伤害责任的条款无效，《民法典》第506条）

### 和解协议的效力
1. **合同效力**：和解协议本质上是一种合同，对双方具有法律约束力
2. **不能直接强制执行**：未经司法确认的和解协议不能直接申请强制执行；如一方不履行，另一方需以和解协议为证据另行提起诉讼
3. **如何获得强制执行力**：
   - 签订和解协议后，向人民法院申请司法确认 → 获得《民事裁定书》→ 可申请强制执行
   - 或先签和解协议，再将协议内容转化为《民事调解书》→ 由法院出具
4. **与法院调解书的区别**：法院出具的《民事调解书》可直接强制执行；当事人自行签署的和解协议不能

### 份数/格式要求
- 当事人各执1份原件
- 建议每人持原件（而非复印件），所有原件均需亲笔签名
- A4纸打印，宋体小四号，1.5倍行距
- 建议每份原件都有双方签名（即每人各签各持的每份）
- 有见证人的，见证人应在每份原价上签名
