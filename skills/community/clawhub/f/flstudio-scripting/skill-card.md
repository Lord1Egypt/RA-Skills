## Use Case: <br>
Developers and music-technology engineers use this skill to generate guidance, code snippets, and reference-backed answers for FL Studio MIDI controller scripts, piano roll and Edison automation, and PyFLP project-file workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated FL Studio or PyFLP scripts can change project state, controller behavior, or .flp files. <br>
Mitigation: Review generated code before running it, use it only with intended projects and devices, and keep backups or work on copies before batch-modifying FLP files. <br>
