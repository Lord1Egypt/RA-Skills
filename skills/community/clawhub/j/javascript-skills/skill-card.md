## Known Risks and Mitigations: <br>
Risk: Broad activation may shape any request that mentions or implies JavaScript, including casual or mixed-language requests. <br>
Mitigation: Enable the skill only when JavaScript style-guide behavior is desired, and disable it for conversations where neutral or project-specific JavaScript guidance is required. <br>
Risk: Generated JavaScript guidance or code may not match a project's local conventions or runtime constraints. <br>
Mitigation: Review generated output against the target project's standards, tests, and supported JavaScript environment before use. <br>
