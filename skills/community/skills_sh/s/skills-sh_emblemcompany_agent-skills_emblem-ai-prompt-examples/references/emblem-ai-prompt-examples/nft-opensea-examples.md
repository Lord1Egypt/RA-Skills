# OpenSea NFT Prompt Examples

## Portfolio And Discovery

```text
Show my NFTs on Ethereum.
Find trending NFT collections on OpenSea right now using integrated marketplace data, then summarize key caveats.
Show me the cheapest listings for this collection.
Summarize recent sales, owners, and floor-price trend for this collection.
```

## Collection Research

```text
Summarize available marketplace stats for this collection slug and note source coverage, timestamps, and verification needs.
Show recent listing, sale, and transfer activity for this NFT collection from available marketplace data and include timestamps.
Compare two NFT collections by floor price, volume, and owner count.
```

## Listings, Offers, And Buy Drafts

```text
Review listing parameters for this NFT and show the exact order summary without submitting anything.
Compare offer parameters for this NFT on Base and stop at the review stage.
Show current listings for this token, rank candidates, and wait for my review before any action.
Review a marketplace order draft for this NFT by showing order hash, price, and protocol address without submitting anything.
Review cancellation parameters for my active NFT listing and show the transaction preview without submitting anything.
```

## Transfers

```text
Prepare an NFT movement summary to another wallet for review only.
Show me the exact NFT movement summary before any wallet action.
Draft an ERC1155 movement summary and confirm token, chain, and destination before asking for explicit execution approval.
```
