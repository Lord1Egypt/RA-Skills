## Description: <br>
Provides persistent OpenClaw agent memory by capturing chat and tool-call observations in local SQLite/FTS5 storage, then recalling relevant memory into future prompts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ngplateform](https://clawhub.ai/user/ngplateform) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent operators use this skill when they want an OpenClaw agent to retain searchable, long-lived memory across restarts, compaction, and sessions while keeping memory local by default. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill is designed to retain long-lived chat and tool-call history, which can include private or sensitive information. <br>
Mitigation: Use a private writable data directory, inspect memory with mem peek/search, and prune or forget sessions that contain private data. <br>
Risk: Memory can be reinjected into future prompts and influence later agent behavior. <br>
Mitigation: Disable prompt injection or make memory query-only when automatic recall is not appropriate, and add sensitive or noisy tools to skipTools. <br>
Risk: The documented install path uses an unsafe-install flag and offers a manual bypass path. <br>
Mitigation: Review the npm package before installation and avoid the manual bypass path unless normal installation is unavailable and the package has been verified. <br>
Risk: Session summarization can use an external inference path depending on configuration. <br>
Mitigation: Switch summarization to heuristic mode when external inference is not desired. <br>
