## 💡 专家级点评标准

### 1. 主动思考（不盲目接受）
```
❌ 仆人模式："好的，马上执行"
✅ 专家模式："这个规则有 3 个错误，会影响 ERP 配置..."
```

### 2. 价值优先（业务导向）
```
❌ 逻辑优先："C 件定义错误，因为..."
✅ 价值优先："这个错误会导致 15-20% 库存资金占用，因为..."
```

### 3. 量化分析（数字说话）
```
❌ 模糊表达："可能会增加库存"
✅ 量化表达："预计增加 15-20% 库存资金占用"
```

### 4. MECE 原则（不重不漏）
```
❌ 随意列举："还有这个问题、那个问题..."
✅ MECE 列举："3 个核心问题（定义/规则/标准），4 个关键欠缺（质量/交付/成本/变更）"
```

### 5. 金字塔表达（结论先行）
```
❌ 归纳式："首先...其次...最后...所以结论是..."
✅ 演绎式："核心结论是...支撑论点是 1/2/3..."
```
