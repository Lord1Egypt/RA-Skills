## 六、评分要素与得分策略指南

### 价格分计算方式（常见三种）

**方式A：最低价法**
```
价格得分 = 最低有效投标价 ÷ 本次投标价 × 价格权重分
```
策略：在成本可控前提下尽量降低报价。

**方式B：平均价法（偏离度）**
```
评标基准价 = 所有有效报价的算术平均值
价格得分 = (1 - |投标价 - 基准价| / 基准价) × 价格权重分
```
策略：报价接近平均价；如能了解竞争者数量，可预测平均价区间。

**方式C：去头去尾平均法**
```
去掉最高和最低报价后求平均
```
策略：避免报价过低（可能被去掉），也避免过高。

### 技术分提升要点

| 评分项 | 提升策略 |
|--------|---------|
| 方案完整性 | 对照招标要求逐条响应，不漏项 |
| 业绩 | 选择最相关、金额最大的案例，格式规范 |
| 人员资质 | 突出项目经理资历，证书齐全 |
| 管理措施 | 具体化、量化，避免空话 |
| 创新亮点 | 提供额外价值（超出招标要求的内容）|
| 绿色低碳 | 提供碳排放数据/节能认证/环保产品认证，2026年起已为法定评审项 |
| 信用评价 | 维护信用中国/政府采购信用记录，一处失信处处受限 |

## 七、废标/无效投标常见原因清单

### 一票否决（绝对废标）
- [ ] 投标文件未在规定时间前递交
- [ ] 投标保证金金额不足或形式不符（银行保函/保险保函/电汇）
- [ ] 法人或授权代表未签字/未盖章
- [ ] 不满足招标文件规定的资格条件（资质、业绩、人员）
- [ ] 报价超出采购预算（如文件有明确规定）
- [ ] 投标文件数量不足（正本/副本数量要求）

### 可能被质疑/取消资格
- [ ] 资质证书超出有效期
- [ ] 业绩材料不完整（缺合同关键页、缺验收报告）
- [ ] 投标函金额大小写不一致
- [ ] 技术参数存在负偏离（低于招标要求的最低值）
- [ ] 前后矛盾（不同位置报价不一致）

## 八、各类型项目特殊要点

### 政府采购服务类
- 必须具有营业执照、组织机构代码（或三证合一）
- 注意是否需要政府采购资格证书
- 注意是否需要提供增值税一般纳税人资格

### 政府采购货物类
- 注意品牌/型号响应要求
- 国产品/进口品问题
- 节能、环保产品认证要求（绿色采购）

### 工程类（含服务）
- 施工许可、安全生产许可
- 项目经理需有注册建造师证书
- 注意分包限制条款

### 团体保险类
- 保险公司资质要求（经营团险的保监许可）
- 偿付能力充足率要求
- 服务网络覆盖要求
- 历史赔付率、满期赔付数据
- 增值服务（员工关怀、健康管理）加分项

### 🆕 新能源/绿色项目类
- 碳排放测算与控制目标（需附第三方核查报告/碳核查声明）
- 绿色建材使用比例承诺（2026要求≥30%）
- 建筑垃圾资源化利用率（≥50%）
- 全生命周期碳排放评估（设计+施工+运营阶段）
- ESG绩效提升方案

### 🆕 数字化服务/政务云类
- 信创适配声明（国产CPU/OS/数据库兼容性）
- 数据安全方案（等保2.0+数据分类分级）
- 大模型/智能体备案合规声明
- 算法安全与可解释性承诺
- 数据本地化存储与安全可控承诺
