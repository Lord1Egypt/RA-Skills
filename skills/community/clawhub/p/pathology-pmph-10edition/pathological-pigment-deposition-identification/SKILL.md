# 病理性色素沉着识别

## 适用场景
当组织病理检查发现细胞内或间质中存在异常增多的色素沉积，且排除正常生理范围内的少量沉积时，应用本技能进行鉴别诊断。

## 执行步骤

1. **确认前提条件**：
   - 存在内源性或外源性色素
   - 患者处于出血、老化、炎症或代谢异常等病理状态

2. **识别四种主要病理性色素**：
   - **含铁血黄素**：金黄色或褐色颗粒，普鲁士蓝染色阳性；见于陈旧性出血、溶血性疾病。
   - **脂褐素**：黄褐色微细颗粒，位于心肌或肝细胞核周；见于老年人或营养耗竭患者，又称“消耗性色素”。
   - **黑色素**：由酪氨酸氧化聚合生成，可聚集于角质细胞或真皮巨噬细胞；局部增多见于色素痣、黑色素瘤，全身性增多见于艾迪生病（Addison病）。
   - **胆红素**：胞质中呈粗糙金色颗粒；血中增高导致皮肤黏膜黄染（黄疸）。

3. **综合判断依据**：
   - 镜下形态（颜色、颗粒性）
   - 特殊染色反应（如普鲁士蓝染色）
   - 临床背景（如溶血史、年龄、内分泌疾病）
   - 沉积部位（细胞内或细胞间质）

4. **输出结果**：
   返回色素类型、沉积特征（部位、形态、染色特性）及对应病理意义，用于支持临床诊断。

> 注意：本技能不适用于正常生理范围内的少量色素沉积。
