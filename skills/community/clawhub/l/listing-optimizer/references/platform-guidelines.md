# Platform Listing Guidelines Reference

## Amazon

### Title Rules
- Max 200 characters (most categories); some categories enforce 80 chars — check category-specific style guide
- Capitalize first letter of each word (title case)
- Numbers should be written as numerals (5, not five)
- No ALL CAPS (except abbreviations like USB, LED)
- No promotional phrases: "best seller", "top rated", "#1", "hot deal"
- No price, promotion, or shipping information
- No HTML or special characters (%, &, $, @, ~, !, {, }, ^, ¦, ¢)
- Include: Brand + Keywords + Size/Color/Pack/Quantity where applicable

### Bullet Points
- Start each bullet with a capital letter
- Do NOT use periods at the end
- ALL CAPS opening phrase is allowed and recommended for scannability
- Max 500 characters per bullet (display truncates at ~200 chars above the fold)
- No HTML; no special characters listed above
- No seller information (contact, URL, return policy — save for description)

### Product Description
- Max 2,000 characters
- Basic HTML allowed: `<b>`, `<br>`, `<li>` (if not A+ enrolled)
- A+ Content replaces basic description for Brand Registry sellers
- A+ supports up to 5 modules; comparison tables are highest-converting

### Backend Keywords
- 250 bytes per field (1 byte per ASCII character)
- Separate words with spaces, NOT commas
- No repeating words from title or bullets (A9 sees them; don't waste bytes)
- No competitor brand names (policy violation; can result in listing suppression)
- No misleading terms (ASMR, organic if not certified)
- Include: synonyms, misspellings (colour/color), regional variants, use cases

### Prohibited Content
- Medical claims without substantiation
- Competitor brand names in copy
- Pesticide claims without EPA registration
- False or misleading statements
- Adult content in standard categories

## Shopify

### Product Title (SEO)
- Recommended: under 70 characters for full display in Google SERPs
- No hard platform limit, but respect SEO character budget
- Format: Primary Keyword + Product Type + Differentiator
- Avoid: "buy", "cheap", "free shipping" in title (Google demotes promotional titles)

### Product Description
- No hard character limit
- Use H2/H3 headings to create scannable structure
- Include primary keyword in first 100 words
- Use bullet lists for features; paragraphs for story/benefit copy
- Avoid duplicate content if cross-posting to Amazon

### Meta Description (SEO)
- 150–160 characters
- Include primary keyword naturally
- Conversational CTA ("Discover", "Shop now", "Free delivery over $50")

### Image Alt Text
- Describe the image AND include the primary keyword
- Format: "[Product Name] [key feature] — [Brand]"
- Max 125 characters

## TikTok Shop

### Product Title
- Displayed title: 34 characters visible in feed
- Full title limit: 255 characters (used in search index)
- Lead with hook phrase or conversational opener
- Include product name + size/flavor/variant
- Avoid: excessive punctuation, emoji in title (renders inconsistently)

### Product Description
- Short form (shown in listing): 800 characters
- Focus on: social proof hook, top 3 benefits, CTA
- TikTok buyers are impulse-driven — lead with emotional outcome, not specs

### Compliance — TikTok
- No health claims without documentation uploaded to seller center
- No "guaranteed results" for supplements, skincare, or devices
- Prohibited: weapons, tobacco, alcohol (standard categories)
- Required disclosures: affiliate relationships, material connections
- Weight loss claims: require FDA disclaimer or registered dietitian review

## Shopee (Southeast Asia)

### Title
- Max 120 characters
- Include: product name + brand + key attribute + variant (if space allows)
- Local language preferred for SEA markets; bilingual (EN + local) accepted

### Description
- Max 3,000 characters
- Use bold and line breaks for readability
- Include: specs table, usage instructions, what's in the box, warranty info

### Keywords (Shopee Search)
- Add via "Product Keywords" field: up to 5 keywords, comma-separated
- Do not repeat keywords across fields
