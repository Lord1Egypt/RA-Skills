# 基于ALP的黄疸鉴别诊断

## 适用场景
- 患者出现黄疸症状
- 已完成血清ALP、总胆红素和ALT（或其他转氨酶）检测
- 需快速区分黄疸的可能病因以指导进一步检查或治疗

## 执行步骤
1. **确认实验室数据**：获取患者的血清ALP水平、总胆红素浓度（μmol/L）和ALT活性（U/L）。
2. **分析三者变化模式**：
   - 若 **ALP明显升高（↑↑↑）**、**胆红素显著升高**、**ALT仅轻度增高（↑）** → 考虑**梗阻性黄疸**；
   - 若 **胆红素中度升高（通常<171 μmol/L）**、**ALT显著升高（↑↑↑）**、**ALP正常或轻度升高（↑）** → 考虑**肝细胞性黄疸**；
   - 若 **ALP明显升高（↑↑或↑↑↑）**、**ALT无明显升高**、**胆红素大多正常** → 考虑**肝内局限性胆道阻塞**（如肝癌、肝脓肿）。
3. **结合临床背景验证**：
   - 梗阻性黄疸常见于胆总管结石、胰头癌；
   - 肝细胞性黄疸多见于病毒性肝炎、酒精性肝病；
   - 肝内占位性病变需影像学（如超声、CT）支持诊断。
4. **排除例外情况**：部分肝硬化或慢性肝炎患者可有ALP轻度升高，但不符合典型梗阻模式，需谨慎解读。

> ⚠️ **注意**：本技能不适用于单纯骨骼疾病或妊娠等生理性ALP升高但无黄疸的患者。
