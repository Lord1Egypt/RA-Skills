## Content Formulas

### 1. Affiliate Product Review (好物推荐/带货文章)

```
📌 Title formats:
- "用了3个月的XX，今天说点大实话"
- "2026年最值得入手的5款XX，第3个我囤了8次"
- "拼多多9.9的XX，和199的有什么区别？实测告诉你"

📝 Structure:
1. 开头钩子(100字): 共鸣痛点 或 反常识发现
2. 产品介绍(200字): 外观、参数、核心功能
3. 使用体验(300字): 真实感受、细节描述
4. 横向对比(200字): 和竞品的差异
5. 购买建议(100字): 适合谁/不适合谁
6. 结尾CTA: "链接放评论区了，自取👇"

💡 Affiliate tips:
- 插入京东联盟/淘宝联盟链接
- 用"同款"、"自用"、"回购"等词增加信任
- 每周更新爆款产品推荐
```

### 2. Listicle / Tool Roundup (好物合集)

```
📌 Title formats:
- "2026年必备的10个效率工具，第7个免费还好用"
- "打工人续命神器TOP8，个个不到50块"
- "学生党必看的5个学习APP，学长亲测"

📝 Structure:
1. 开篇(100字): "最近整理了..."
2. 每个推荐(150字×N):
   - 名称+价格
   - 核心功能(2-3点)
   - 适用场景
   - 小缺点（增加真实感）
3. 总结表格(50字)
4. 获取方式+CTA
```

### 3. How-to Guide (干货教程)

```
📌 Title formats:
- "3步搞定XX，小白也能学会（附模板）"
- "用了这个XX方法，效率提升300%"
- "90%的人不知道的XX黑科技，我教你"

📝 Structure:
1. 痛点引入(100字)
2. 错误方法盘点(100字) - 建立权威
3. 正确步骤详解(500字) - 编号步骤，配图
4. 避坑指南(100字)
5. 资源下载/福利引导(50字)
```

### 4. Emotional/Story-driven (情感故事)

```
📌 Title formats:
- "那个月薪3000的女孩，后来怎样了？"
- "离职那天，我在工位上哭了"

📝 Structure:
1. 场景钩子(50字)
2. 故事展开(400字)
3. 观点升华(200字)
4. 互动提问(50字)
```
