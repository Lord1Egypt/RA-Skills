# 个股分析要点

## 一、分析模块总览

| 模块 | 子模块 | 核心问题 |
|------|--------|----------|
| **模块一：基本信息** | 公司概况、行业分类、市值、上市信息 | 这家公司是做什么的？规模多大？ |
| **模块二：行情走势** | 价格走势、多周期涨跌幅、成交量 | 近期股价表现如何？趋势向上还是向下？ |
| **模块三：经营分析** | 主营分析、财务分析（杜邦）、公司重大事项 | 业务结构是否健康？ROE 驱动力来自哪？ |
| **模块四：行情分析** | 估值分析、资金流向分析、技术分析 | 当前估值贵不贵？资金在进还是在出？ |
| **模块五：综合结论** | 优势、风险、关注要点、投资观点 | 综合来看，这只股票值得关注吗？ |

---

## 二、各模块分析要点

### 模块一：基本信息

**分析目标**：了解公司"是谁"——基本属性和市场地位

| 维度 | 关键点 |
|------|--------|
| 公司全称与股票代码 | 唯一标识，注意 A 股 vs 港股 vs 美股代码差异 |
| 所属行业 | 申万行业分类 |
| 上市板块 | 上交所主板/科创板、深交所主板/创业板、北交所 |
| 总市值 / 流通市值 | 规模量级，决定股票流动性和关注度 |
| 上市日期 | 上市不满 3 年的公司需注意财务数据期数较少 |
| 主营业务描述 | 公司核心收入来源，用于判断业务属性 |

---

### 模块二：行情走势

**分析目标**：了解股价"怎么走"——短期趋势和波动特征

| 维度 | 关键点 |
|------|--------|
| 最新收盘价与涨跌幅 | 当日或最近交易日收盘价，相比大盘涨跌情况 |
| 成交量与成交额变化 | 量价配合——放量上涨/缩量整理/放量下跌各有含义 |
| 趋势判断 | 上升趋势 / 震荡整理 / 下降趋势 |

---

### 模块三：经营分析

#### 3.1 主营分析要点

**分析目标**：公司收入结构——哪些业务贡献最大？各业务是否在增长？

**分析要点**：
- 各业务板块占营业收入的比重（主力业务是哪个？占比多少？）
- 各业务板块毛利率差异（哪个业务最赚钱？）
- 主营收入增长率（整体增长靠什么驱动？）
- 业务集中度风险（是否过度依赖单一业务？）

#### 3.2 财务分析—杜邦分析要点

**分析目标**：拆解 ROE（净资产收益率）的驱动力，找出盈利质量的核心来源

**杜邦三因子模型**：
```
ROE = 净利率 × 资产周转率 × 权益乘数
     = (净利润/营业收入) × (营业收入/总资产) × (总资产/净资产)
```

| 因子 | 含义 | 分析关注点 |
|------|------|-----------|
| **净利率**（盈利能力） | 每元收入能赚多少净利润 | 反映定价权、成本控制能力；关注毛利率与净利率的差距（费用率） |
| **资产周转率**（运营效率） | 每元资产能产生多少收入 | 反映资产利用效率；重资产 vs 轻资产模式差异大 |
| **权益乘数**（财务杠杆） | 负债程度放大收益的能力 | 反映负债水平；金融/地产偏高；关注有息负债率 |

**综合判断框架**：

- ROE 高且稳定 → "好公司"候选，重点看净利率是否可持续
- ROE 靠高杠杆驱动 → 警惕负债风险，尤其是利率上行周期
- ROE 靠高周转驱动 → 关注行业景气度和资产质量
- ROE 靠高净利率驱动 → 关注护城河（品牌/技术/许可），护城河是否可持续

#### 3.3 公司重大事项要点

**分析目标**：识别可能影响公司估值和股价的"大事"

| 事项类型 | 分析关注点 |
|----------|-----------|
| 分红送配 | 分红率是否可持续？高分红是否意味增长乏力？ |
| 再融资 | 定增/可转债稀释现有股东权益，目的合理性 |
| 股权激励 | 行权条件是否过高？是否可能损害股东利益？ |
| 股份回购 | 回购用途（注销/激励）？金额是否显著？ |
| 诉讼/监管处罚 | 是否涉及主营业务合规？是否可能造成重大损失？ |

### 模块四：行情分析

#### 4.1 估值分析要点

**分析目标**：判断当前股价"贵不贵"，相对历史和行业处于什么位置

**核心指标与判断标准**：

| 指标 | 计算方式 | 解读要点 |
|------|----------|----------|
| **PE（TTM）** | 市价/每股收益（近12个月） | 与行业均值对比；负 PE 无意义；周期股 PE 失真 |
| **PB** | 市价/每股净资产 | 破净（PB<1）可能是价值陷阱，也可能是资产质量问题 |
| **PS** | 市价/每股营业收入 | 适合营收稳定但利润波动的公司（如电商、SaaS） |
| **历史分位** | 当前估值在近3年中所处百分位 | <20%低估；20%~50%合理偏低；50%~80%合理偏高；>80%高估 |
| **行业均值** | 同行业公司平均估值 | 判断相对行业是溢价还是折价 |

**分析框架**：
1. 先看行业平均 PE/PB，确定行业估值中枢
2. 再看公司历史分位，判断当前在历史中的位置
3. 结合成长性（净利润增速）判断估值是否匹配
4. 警惕低 PE 陷阱（银行/地产的表外负债、周期股的景气顶点）

#### 4.2 资金流向分析要点

**分析目标**：跟踪市场资金的实际选择——谁在买？谁在卖？

**核心指标**：

| 指标 | 含义 | 分析要点 |
|------|------|----------|
| **主力净流入** | 超大单净流入 + 大单净流入 | 反映机构/大资金态度；个人投资者资金归入散户 |
| **连续流入/流出天数** | 连续 N 日主力净流入为正/负 | 连续3日+净流入 = 主力持续买入信号；连续3日+净流出 = 主力离场信号 |
| **净流入占比** | 主力净流入 / 成交额 | >5% = 当日主力明显介入；<1% = 当日主力参与度低 |
| **北向资金（沪深港通）** | 外资通过沪股通/深股通持股变化 | 持续净买入 = 外资长期看好；大幅净卖出 = 外资短期避险 |

**资金流向综合判断**：
- 主力持续净流入 + 股价上涨 → 资金推动型上涨，可持续性较强
- 主力净流出 + 股价却上涨 → 警惕"散户接盘"，上涨难以持续
- 主力净流出 + 股价下跌 → 确认下跌趋势，谨慎抄底
- 北向资金大幅净流入 → 关注外资重点加仓板块

#### 4.3 技术分析要点

**分析目标**：从历史价格和成交量规律中寻找趋势信号，作为买卖参考

**趋势类指标**：

| 指标 | 周期 | 核心信号 | 分析要点 |
|------|------|----------|----------|
| **MA 均线** | MA5 / MA20 / MA60 | 价格在均线上方 = 偏多；均线多头排列（5>20>60）= 强势 | MA60 为中期生命线，跌破需警惕 |
| **MACD** | DIF / DEA / 柱状线 | **金叉**：DIF 从下穿上 DEA → 买入信号 | MACD 顶背离（股价新高但 DIF 低点）= 上涨动能衰竭 |
| MACD | | **死叉**：DIF 从上穿下 DEA → 卖出信号 | MACD 底背离（股价新低但 DIF 高点）= 下跌动能衰竭 |
| **KDJ** | K / D / J 值 | J > 80 = 超买，警惕回调 | J 触顶后拐头向下 = 短线卖出参考 |
| KDJ | | J < 20 = 超卖，关注反弹机会 | J 触底后拐头向上 = 短线买入参考 |

**量价配合分析**：

- 放量上涨 + 缩量回调 = 健康的多头趋势
- 缩量上涨 + 放量下跌 = 主力出货嫌疑
- 地量（成交量极度萎缩）= 可能变盘前兆
