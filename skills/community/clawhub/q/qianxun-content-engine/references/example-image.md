## 四、可学习要点

1. **图文笔记的"封面图 = 标题"**：本条没用文字封面，第一张图本身就是钩子（手 + 耳饰 + 深色背景），靠视觉自带信息。如果你的产品视觉够强，可以省略大字封面。
2. **文化锚点比卖点更打动**：通篇没有提价格、克重、买点，全程在讲"紫藤""金陵""民国""旧时光"。先建立"我也想成为这样的人"的向往，购买决策自然跟上。
3. **诗化开头 + 散文体正文**：「紫藤垂紫绶，风拂碎香柔」一句先定调，把读者从滑动状态切换到阅读状态。后面是节奏舒缓的散文，跟产品调性一致。
4. **图文 4 帧公式**：全局 → 特写（看清工艺）→ 人像（建立向往）→ 对称（形式美学）。是"种草型珠宝图文"的稳定结构，可复用。
5. **评论区高意向不来自"快冲"，来自"文化共鸣 + 实体保障"**：评论里的"询价 / 蹲克重 / 实体店"说明用户已经被文化打动，缺的是"信任补全"。生成对应内容时，CTA 不是"快冲"，是"地址 + 营业时间 + 工艺背书"。
