# 报告结构模板

## 模板 A：决策型正式报告

适用于需要给出验收结论的场景。

章节：

- 验收总览
- 范围、环境、角色、假设、排除项
- 验收基线
- RTM 需求追踪矩阵
- 业务流程验收结果
- 模块/菜单验收结果
- 角色权限/数据范围验收结果
- 接口/后台/异步/集成补充验证结果
- UAT 用例执行明细
- 正式缺陷清单
- 阻塞项、待确认项、范围外事项
- 文档/系统差异
- 风险与遗留问题
- 证据索引
- 最终验收建议
- 签署确认页

## 模板 B：模块型报告

适用于测试模块较多的场景。

每个模块包含：

- 模块结论
- 覆盖菜单/页面
- 已执行用例
- 正式缺陷
- 阻塞项/待确认项
- 证据引用
- 遗留风险

## 模板 C：轻量探索型 UAT 记录

仅适用于小范围或早期探索。

包含：

- 目标与范围
- 已测试内容
- 通过内容
- 发现问题
- 阻塞与未知项
- 建议下一步
