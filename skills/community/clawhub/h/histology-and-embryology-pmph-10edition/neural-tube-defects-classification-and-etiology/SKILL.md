# 神经管缺陷类型与成因判断

## 何时使用
- 胚胎发育至第4周末，神经孔未能正常闭合
- 影像或病理发现颅脑或脊柱结构异常
- 需区分无脑畸形、脊柱裂及其亚型

## 执行步骤
1. **判断未闭合部位**：
   - **前神经孔未闭** → 无脑畸形（anencephaly），常伴颅顶骨缺如（露脑）
   - **后神经孔未闭** → 脊髓裂（myeloschisis），常伴脊柱裂（spina bifida）
2. **脊柱裂亚型分类**：
   - **脊膜膨出**：囊袋仅含脊膜与脑脊液
   - **脊髓脊膜膨出**：囊袋含脊膜、脑脊液、脊髓及神经根
3. **颅部膨出类型**：
   - 脑膜膨出 / 脑膜脑膨出（多见于枕部）
   - 积水性脑膜脑膨出（合并脑室膨出）
4. **识别致病机制**：
   - 脊索诱导作用丧失
   - 环境致畸因子（如叶酸缺乏）
5. **输出诊断结果**：返回具体畸形名称及伴随异常

> 预防提示：备孕期补充叶酸可显著降低神经管缺陷发生率。
