## Description: <br>
Scan WordPress sites for security vulnerabilities, misconfigurations, and potential threats. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[krishnakumarmahadevan-cmd](https://clawhub.ai/user/krishnakumarmahadevan-cmd) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
WordPress site owners, security professionals, penetration testers, and hosting providers use this skill to assess public WordPress installations for vulnerabilities, weak configurations, missing security headers, and other security risks. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The scanner sends submitted WordPress site URLs to an external scanning provider. <br>
Mitigation: Use the skill only when you are comfortable sharing the target URL with that provider. <br>
Risk: Security scanning can affect or be inappropriate for sites the user does not control. <br>
Mitigation: Scan only WordPress sites you own, administer, or have explicit permission to test. <br>
