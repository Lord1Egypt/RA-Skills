## Description: <br>
Converts text to speech with Microsoft Edge TTS, including real-time streaming playback, generated audio files, configurable voices, and voice listing. <br>

This skill is ready for commercial/non-commercial use. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Subprocess execution and install paths can create host security exposure. <br>
Mitigation: Review or patch subprocess usage before installation; prefer argument-array process execution, run in a sandbox, and avoid non-sensitive hosts only if review is incomplete. <br>
Risk: Text submitted for speech generation may be sent to an external TTS service. <br>
Mitigation: Disclose external TTS use to users and avoid submitting confidential, regulated, or sensitive text unless the deployment has approved that data flow. <br>
Risk: Audio playback and file path handling may interact with local host files. <br>
Mitigation: Limit playback to files generated by the skill, validate file paths, and keep generated audio in a controlled temporary directory. <br>
Risk: Dependencies and playback tools affect the host runtime environment. <br>
Mitigation: Pin and review dependencies, install ffmpeg or ffplay from trusted sources, and verify the runtime configuration before enabling streaming playback. <br>
