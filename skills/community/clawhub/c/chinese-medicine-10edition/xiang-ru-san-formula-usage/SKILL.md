# Xiang Ru San Formula Usage

## When to Use
当满足以下全部条件时应用本技能：
- 发病季节为夏季（暑月）；
- 患者因贪凉饮冷导致外感风寒、内伤湿邪；
- 症状包括恶寒发热、无汗、头重头痛、胸闷泛恶、腹痛吐泻；
- 舌象为舌苔白腻，脉象为脉浮。

**禁忌情形**：
- 不适用于阳暑证（如高热烦渴、汗多尿赤等实热证）；
- 非暑湿夹寒证者禁用。

## Execution Steps
1. **确认方剂组成与剂量**：
   - 香薷（去土）一斤（君药）
   - 厚朴（去粗皮，姜汁炙熟）半斤（臣药）
   - 白扁豆（微炒）半斤（佐药）

2. **明确各药功效**：
   - 香薷：辛温芳香，解表散寒，祛暑化湿；
   - 厚朴：苦辛而温，行气除满，燥湿行滞；
   - 白扁豆：甘平，消暑和中，兼能化湿。

3. **指导用法**：
   - 水煎服，可加少量酒同煎以助药力发散。

4. **判断适用病证**：
   - 临床适用于上呼吸道感染、急性胃肠炎等属“暑湿外感风寒”者。
