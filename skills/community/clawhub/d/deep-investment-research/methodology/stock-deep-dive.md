# stock-deep-dive.md — 个股深度研究框架

> 引用 research-core.md + source-registry.md + content-reading-protocol.md。
> 用途：判断一只个股值不值得投——核心是预期差，不是"好不好"。

## 研究目标定义
一句话：要回答这只票的什么问题（如"中芯国际当前价位是否有未被定价的上行逻辑"）。

## 🔑 个股深度研究独门分析方法（核心·算出预期差与非对称赔率，不抄卖方）

> 研究结构解决"看哪几块"，下面四个分析法解决"怎么算出预期差、自己估值、排催化剂、用资金面验证"。**主动用方法算，不是复述共识。**

### 方法A · 预期差提取（从一致预期反推→质疑→算差额）
不直接接受卖方一致预期，反向拆解：
- 取市场一致预期EPS/增速 → 反推它隐含了哪些假设（多少营收增速/毛利率/份额）
- 质疑最脆弱的那个假设：有无一手证据显示它会比共识更好或更差？
- 若该假设错，EPS会差多少？这个差额就是预期差的量级
> 洞察来源：**预期差不是"我觉得好/差"，是"市场隐含假设X，但证据指向Y，差额=Z"。算得出差额才是真预期差。**

### 方法B · 估值锚定自算（禁止抄分析师目标价，红线4）
- 用 新EPS × 合理PE区间 自算（PE区间=同行分位 + 成长溢价/折价，不是单点）
- 或 DCF/SOTP/PB-ROE，至少1种，给区间不给单点
- **分析师目标价是滞后指标**（业绩beat后才上修），只作"市场预期"参照，绝不当上行天花板（教训12）
> 洞察来源：**自算估值才能发现"市场目标价滞后于基本面"的机会。抄目标价=放弃了最大的预期差来源。**

### 方法C · 催化剂确定性排序（时间窗口×确定性×弹性）
对每个潜在催化剂三维打分排序：
- 时间窗口（多久内发生）× 确定性（发生概率）× 弹性（发生后股价反应幅度）
- 排出"最可能驱动股价的前2-3个催化剂"，而非罗列一堆
> 洞察来源：**催化剂决定"预期差何时被市场认知/兑现"。没有近期高确定性催化的预期差，可能要等很久（接时间成本）。**

### 方法D · 资金面验证（基本面判断与资金动向是否一致）
用资金面交叉验证基本面判断：
- 南向/北向、机构持仓变动、沽空比例、超级投资者动向——方向与你的基本面判断一致吗？
- 一致=增强信心；背离=要么市场还没认知(机会),要么你漏了什么(预警),必须想清楚是哪种
> 洞察来源：**资金面是基本面判断的"测谎仪"。聪明钱与你反向时，先假设是你错了再去找证据，能避免大坑。**

## 深度研究结构

### 1. 公司基本面（读财报全文，非摘要）
- 主营业务拆分：收入/利润来自哪、各占多少、增速
- 盈利质量：毛利率/净利率趋势、现金流 vs 利润、ROE/ROIC
- 资产负债：杠杆、有息负债、商誉、应收
- 最近一期财报关键变化（环比/同比，beat/miss 共识多少）

### 2. 预期差（核心）
- 一句话假设："市场定价 [X]，但 [财报/政策/事件/数据] 显示 [Y]，如果 Y 正确则 [Z]"
- 写不出预期差 → 直接结论"无明显预期差"，不硬做
- 市场共识是什么（一致预期EPS/评级/目标价分布）？我哪里看法不同？

### 3. 估值（自己算，红线4）
- 用 ≥1 种方法自算合理价值：新EPS×合理PE / DCF / SOTP / PB-ROE
- **不直接用分析师目标价当天花板**（滞后指标，业绩beat后会上修）
- 给估值区间，不是单点

### 4. 催化剂（前瞻）
- 未来 3-12 个月可能的催化剂：业绩、新品、政策、订单、事件
- 时间窗口与确定性

### 5. 风险与证伪
- 下行风险 + 各自概率
- kill condition：看到什么就证明判断错了

### 6. 量化情景
- Bull / Base / Bear：各情景假设、目标价、概率
- 期望值判断（哪个情景概率最大、风险收益比如何）

## 产出
- 结论：明确立场（看多/看空/中性，或值得/不值得深入跟踪）+ 确信度
- 核心逻辑链（A→B→C→D）
- 估值区间 + 关键假设
- 催化剂与跟踪信号
- 风险 + 信息盲区
- 不替用户下单，给判断与依据

## 多源验证（每条≥具体数值，红线2）
财报 / 一致性预期 / 资金面(南向/机构) / 公告 / 政策 / 同业对比——每个维度有数字，不写"确认/看好"空话。

## 质量对照（肤浅 vs 深刻）

**❌ 肤浅**：“公司基本面稳健，行业龙头，估值不贵，长期看好。”
—— 无预期差、无自算估值、无催化剂、无 kill condition，等于没研究。

**✅ 深刻**：“市场用成熟制程周期股给中芯定价(fwdPE X)，但 80% 国产强制令把先进产能变成刚需；若 2027 先进产线满载，EPS 可到 Y，对应合理价 Z（自算：EPS×合理PE），距现价 +W%。kill：HBM 国产化滞后/良率不达预期。”
—— 有定价检查、有自算估值、有情景与概率、有证伪条件。
