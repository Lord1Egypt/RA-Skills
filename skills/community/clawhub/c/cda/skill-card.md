## Description: <br>
Provides a Chinese-language reference pack for Causal Dynamics Architecture (CDA), including causal mechanism networks, Hamiltonian constraints, causal encapsulation recursion, physics-constrained neural networks, and causal simulation concepts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, researchers, and technical reviewers use this skill to ask an agent for structured explanations and references about CDA architecture concepts, proposed system layers, data structures, limitations, and implementation roadmap. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The CDA material is a research proposal and reference source rather than audited implementation code. <br>
Mitigation: Treat architecture claims and implementation guidance as reference material and validate them before relying on them for production engineering decisions. <br>
Risk: The source material is primarily Chinese-language and may be misunderstood if an agent answers in a different language without preserving technical nuance. <br>
Mitigation: Specify the desired response language and ask the agent to cite the relevant reference section when precision matters. <br>
