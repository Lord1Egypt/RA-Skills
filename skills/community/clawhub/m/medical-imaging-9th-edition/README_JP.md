## プロジェクト概要

本プロジェクトは、医学画像診断学、インターベンショナルラジオロジー、AI支援分析、マルチモーダル画像技術選択などの核心領域を体系的に統合し、**136 の重要臨床スキル**を13のカテゴリーに分類して収録しています。

**対象読者**：放射線科医、臨床各科医師、医学生、インターベンショナルラジオロジーチーム、医学教育者

**参考教科書**：人民衛生出版社『医学画像学』第9版（国家衛生健康委員会「第十四次五カ年計画」教材）

**⚠️ リスク ⚠️**：本スキルセットには画像診断、造影剤使用、インターベンション治療適応評価、レポート解釈が含まれており、独立した診断や治療決定に誤用される可能性があります。

対策：出力は教育目的または臨床医レビュー用の参考資料としてのみ使用し、現在の公式ガイドライン、地域プロトコル、および資格を持つ放射線専門医の助言に従って確認してください。

### 使用例

**質問：**
```bash
Bosniak腎嚢胞分類2019年版の基準
```

**回答：**
```bash
## Bosniak腎嚢胞分類（2019年版）

| 分類 | 画像特徴 | 悪性リスク | 管理 |
|------|---------|-----------|------|
| I型 | 単純性嚢胞、薄壁、隔壁なし | <1% | 良性、経過観察不要 |
| II型 | 少数（≤3）の薄い隔壁、微細な石灰化 | <5% | 良性、経過観察不要 |
| IIF型 | 複数（≥4）の薄い隔壁、軽度の壁肥厚 | ~10% | 経過観察 |
| III型 | 造影増強する充実成分、不規則な壁肥厚 | ~55% | 手術切除 |
| IV型 | 明らかな造影増強を示す充実成分 | >90% | 手術切除 |
```
