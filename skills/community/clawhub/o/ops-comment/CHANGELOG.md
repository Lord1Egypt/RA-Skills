# Changelog

## v1.0.0 (2026-05-16)

### 新增
- 互动引擎：点赞、收藏、关注、评论四类动作
- 浏览模拟：变速滚动、随机停顿、回滚行为
- 反检测层：贝塞尔曲线鼠标移动、打字节奏模拟、坐标随机偏移
- 速率控制：每日限额、会话节奏、高斯/均匀/泊松抖动、防突发
- 内容过滤：笔记数值过滤、关键词过滤、用户过滤、历史去重
- AI 评论生成：多模型路由（DeepSeek/GPT/Claude）、安全过滤、多样性检查
- 四种任务路径：发现页、搜索页、用户主页、评论区
- 多账号管理：Profile 体系、独立计数器、跨账号去重
- 定时调度：OpenClaw cron 集成、前置检查、启动抖动、故障恢复
- 数据看板：每日/周报告、Chart.js 可视化、效果追踪
- 异常处理：五级风险响应、验证码检测、登录态监控
- 会话生命周期：预热→正式互动→中场休息→收尾
