## Description: <br>
Retrieves saved memory by date, keyword, type, or natural-language query and returns matching historical context with source and time information. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and agent operators use this skill to search prior conversations, diary files, project decisions, people, and dated notes from a permanent memory system. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Memory recall can surface private or sensitive history when a user asks about past conversations, dates, projects, people, or decisions. <br>
Mitigation: Review recalled content before sharing it outside the chat and limit prompts to the history needed for the current task. <br>
