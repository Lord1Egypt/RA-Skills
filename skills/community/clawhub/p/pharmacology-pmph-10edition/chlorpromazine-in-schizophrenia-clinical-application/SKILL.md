# chlorpromazine-in-schizophrenia-clinical-application

## 何时使用
- 患者确诊为以阳性症状（如进攻、亢进、妄想、幻觉）为主的精神分裂症，尤其处于急性期；
- 或患者为伴有精神病性症状的其他精神障碍或器质性精神疾病（如脑动脉硬化性、感染中毒性精神障碍），且无禁忌证。

## 如何执行
1. **确认适应症**：仅适用于以阳性症状为主的精神分裂症；对阴性症状（如意志减退、快感缺乏）效果不显著，不应作为主要治疗选择。
2. **评估禁忌证**：禁用于有癫痫或惊厥史、青光眼、乳腺增生症或乳腺癌患者；老年及动脉硬化、高血压患者慎用。
3. **制定给药方案**：
   - 对典型精神分裂症患者，可按标准剂量起始并长期维持，甚至终身治疗；
   - 对器质性精神疾病患者，必须使用较小剂量。
4. **实施停药策略**：器质性精神疾病患者在症状控制后应立即停药，避免长期使用。
5. **监测不良反应**：特别关注直立性低血压、锥体外系反应及高催乳素血症等。

> 注意：本技能不适用于阴性症状主导的精神分裂症，也不替代个体化临床判断。
