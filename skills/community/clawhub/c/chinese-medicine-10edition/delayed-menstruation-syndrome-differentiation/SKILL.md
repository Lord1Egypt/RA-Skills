# Delayed Menstruation Syndrome Differentiation

## When to Use
患者月经周期连续2次以上延后7天以上，排除青春期初潮1年内及围绝经期生理性延迟，且存在明确证候表现。

## Execution Steps

### 第一步：确认诊断标准
- 周期延后≥7天，连续≥2周期。
- 排除偶然性及生理阶段波动。

### 第二步：分型辨治
- **肾虚**：量少、色淡质稀 + 头晕耳鸣、腰膝酸软 → 当归地黄饮加减
- **血虚**：量少、色淡红质稀 + 头晕眼花、心悸少寐 → 大补元煎加减
- **血寒（虚寒）**：量少、色淡质稀 + 小腹隐痛喜温按、小便清长 → 温经汤《金匮要略》加减
- **血寒（实寒）**：量少、色暗有血块 + 小腹冷痛拒按、得热痛减 → 温经汤《妇人大全良方》加减
- **气滞**：量少、色暗红或有血块 + 小腹胀痛、情志抑郁 → 乌药汤加减
- **痰湿**：量少、夹黏液 + 形体肥胖、脘闷呕恶 → 苍附导痰丸加减

### 第三步：治疗总则与风险评估
- 总则：“虚者补之，实者泻之，寒者温之，滞者行之，痰者化之”，重在调理冲任、疏通胞脉。
- 若伴经量过少，评估闭经风险，制定早期干预方案。

## Precautions
- 仅延后3–5天或偶然一次不作病论。
- 无其他证候的周期波动不适用本技能。
