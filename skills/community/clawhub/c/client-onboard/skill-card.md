## Description: <br>
Generate a complete client onboarding package from a project description, including a project brief, tech stack, milestones, folder structure, CLAUDE.md, and a ready-to-send kickoff message. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kevdogg102396-afk](https://clawhub.ai/user/kevdogg102396-afk) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Freelancers and developers use this skill after closing a new client to turn a project description into an onboarding package with scope, stack, milestones, project context, and a client kickoff message. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated business assumptions, payment terms, and kickoff copy may not match the actual client agreement. <br>
Mitigation: Review the generated project brief, milestone plan, payment schedule, and kickoff message before sending them to a client. <br>
Risk: The generated CLAUDE.md may encode project assumptions that are unsuitable for reuse in the final project repository. <br>
Mitigation: Review CLAUDE.md before copying it into a project and update stack, commands, environment variables, and conventions to match the actual implementation. <br>
