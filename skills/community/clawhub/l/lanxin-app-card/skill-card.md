## Description: <br>
Helps an agent produce Lanxin appCard JSON messages for application cards with structured titles, fields, links, and card targets. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[iamdacai](https://clawhub.ai/user/iamdacai) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and developers use this skill when they want an agent to respond to Lanxin application-card requests with a clean JSON appCard payload instead of prose or Markdown. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad Lanxin and card-related trigger words may activate the skill during ordinary card-related conversations. <br>
Mitigation: Install it only where Lanxin appCard JSON output is intended, and narrow trigger conditions if the host environment supports trigger customization. <br>
