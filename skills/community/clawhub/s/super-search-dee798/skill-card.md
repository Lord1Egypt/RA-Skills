## Description: <br>

## Use Case: <br>
Developers and coding agents use this skill for lightweight guidance on choosing search-style workflows when locating code or text patterns. It is best suited for brief learning prompts and reminders rather than deep search automation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The broad trigger terms may activate the skill in unrelated conversations. <br>
Mitigation: Use narrower routing terms or confirm the user wants search-workflow guidance before applying the skill. <br>
Risk: The skill provides lightweight learning notes rather than independently verified search methodology. <br>
Mitigation: Treat its advice as general guidance and review recommendations before changing code or workflows. <br>
