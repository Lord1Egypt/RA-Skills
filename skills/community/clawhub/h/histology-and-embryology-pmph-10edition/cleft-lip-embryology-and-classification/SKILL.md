# 唇裂的胚胎学成因与分类

## 何时使用
- 用户询问唇裂的胚胎发育机制
- 需要根据裂隙位置判断融合失败的面部突起
- 评估唇裂是否合并腭裂或其他颜面畸形
- 提供唇腭裂预防建议（如孕期营养、避免致畸因素）

## 执行步骤

1. **确认正常颜面融合过程**：
   - 左右下颌突 → 形成下唇与下颌
   - 上颌突 + 同侧内侧鼻突 → 形成上唇外侧部与上颌
   - 左右内侧鼻突融合 → 形成鼻梁、鼻尖、人中（上唇正中部）

2. **识别融合失败部位并分类唇裂**：
   - **最常见类型**：上颌突与同侧内侧鼻突未融合 → 单侧或双侧唇裂，裂沟位于人中外侧
   - **罕见类型**：
     - 左右内侧鼻突未融合 → 上唇正中裂
     - 左右下颌突未融合 → 下唇正中裂
     - 内侧鼻突发育不良 → 宽大正中唇裂伴人中缺损

3. **检查是否合并其他畸形**：
   - 牙槽突裂
   - 腭裂（前腭裂、正中腭裂或全腭裂）

4. **提供预防干预要点**：
   - 孕期保证营养均衡（尤其叶酸等维生素）
   - 避免病毒感染、X线照射、内分泌紊乱等致畸暴露
   - 推荐定期产前检查

> 注意：本技能仅适用于人类胚胎颜面发育，不适用于其他物种或非面部裂隙。
