## Description: <br>
Code2Flow analyzes codebase structures and source files to generate clear Mermaid.js diagrams that visualize system architecture and workflows. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[radenwahid](https://clawhub.ai/user/radenwahid) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to analyze pasted source files or directory structures and produce architectural summaries with Mermaid diagrams for documentation and review. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The security evidence notes that powerful maintainer and review workflows should be run deliberately because they can affect user or skill state. <br>
Mitigation: Verify the target and reason before any state-changing workflow and use safer review options when evaluating untrusted code. <br>
Risk: Generated architecture diagrams may omit or misrepresent behavior when the provided code context is incomplete. <br>
Mitigation: Review the Markdown summary and Mermaid diagram against the source files before using the output as authoritative documentation. <br>
