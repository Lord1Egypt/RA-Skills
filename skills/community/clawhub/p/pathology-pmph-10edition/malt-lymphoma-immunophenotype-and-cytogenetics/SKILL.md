# MALT淋巴瘤免疫表型与细胞遗传学特征分析

## 适用条件
- 已确诊或高度怀疑为胃或肺MALT淋巴瘤
- 具备免疫组化和细胞遗传学检测能力

## 执行步骤

### 1. 免疫表型检测
- **阳性标记**：检测CD20、CD79a、表面免疫球蛋白IgM和IgA，应呈阳性表达
- **阴性标记**：确认CD5、CD10、CD23、周期蛋白D1、IgD为阴性

### 2. 细胞遗传学检测
- 检测是否存在t(11;18)(q21;q21)染色体易位
  - 若存在，表明形成API2::MALT1融合基因
  - 此结果预示对幽门螺杆菌根除等抗生素治疗反应差

### 3. 分子机制补充（如可行）
- 检测TNFAIP3基因突变状态
  - TNFAIP3是NF-κB通路的负调控因子，其失活突变支持MALT淋巴瘤诊断

### 4. 结果解读
- 若免疫表型符合且存在t(11;18)易位，可确诊MALT淋巴瘤并预测抗生素治疗无效
- 注意：t(11;18)易位主要见于胃和肺MALT淋巴瘤，其他部位可能不具此特征

## 禁忌情形
- 不适用于DLBCL、滤泡性淋巴瘤等其他B细胞淋巴瘤
- 非MALT来源的淋巴瘤禁用本流程
