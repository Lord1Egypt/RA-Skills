|------|--------|------|

| 数据一致性 | 缓存与 DB 不一致场景 | 通用 |
| 并发 | 乐观锁/悲观锁竞争场景 | 通用 |
| 并发 | 重复提交/幂等性 | 通用 |
| 边界 | 数据量为零（空列表、空表） | 通用 |
| 边界 | 跨天/跨月/跨年的业务逻辑 | 通用 |
| 权限 | 越权访问（修改不属于自己的数据） | 安全 |

| 集成 | 第三方服务超时/不可用 | 通用 |

| 数据 | SQL 注入、XSS | 安全 |
