## Description: <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Researchers and analysts use this skill to clean SPSS, CSV, Excel, or TSV datasets for survey, experiment, public-opinion, and similar research workflows. It guides diagnosis, user confirmation, cleaning, validation, export, and report generation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Cleaning operations may alter or remove data in ways that affect research conclusions. <br>
Mitigation: Use a copy of the original dataset and review the proposed cleaning plan before approving changes. <br>
Risk: User-provided datasets may contain sensitive personal or research data. <br>
Mitigation: Handle input files according to the user's data-governance requirements and avoid exposing sensitive data unnecessarily. <br>
Risk: Suggested Python dependencies and generated scripts run in the user's environment. <br>
Mitigation: Install packages in an isolated environment and review generated scripts before execution. <br>
Risk: The artifact is written for Chinese-language workflows. <br>
Mitigation: Use the skill only when Chinese-language prompts and reports are acceptable for the workflow. <br>
