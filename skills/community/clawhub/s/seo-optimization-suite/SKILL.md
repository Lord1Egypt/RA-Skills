# SEO Optimization Suite

## Capabilities

| Capability | Description |
|------------|-------------|
| **Dual-engine optimization** | Baidu SEO + Google SEO parallel strategies |
| **Technical SEO audit** | HTML/CSS code-level analysis, structured data (Schema.org), Core Web Vitals (LCP/FID/CLS), mobile-friendliness, crawl efficiency |
| **Content strategy** | Keyword matrix (core/long-tail/question/geo), content gap analysis, search intent classification, internal linking |
| **Competitor analysis** | Top 3 competitor keyword布局拆解, backlink source analysis, content strategy comparison |
| **Link building** | High-quality backlink sources, anchor text distribution, brand mention monitoring |
| **Priority ranking** | P0 (fix immediately), P1 (this week), P2 (this month), P3 (ongoing) |
| **Ranking timeline** | Expected improvement: 1 week/1 month/3 months/6 months |

## Trigger Keywords
- "SEO", "搜索引擎优化", "网站优化", "排名优化", "关键词", "Google SEO", "Baidu SEO", "technical SEO", "内容策略"

## Workflow

### Phase 1: Website Diagnosis
Technical audit, performance analysis, current ranking assessment.

### Phase 2: Competitor Analysis
Identify top 3 competitors, analyze their SEO strategies and gaps.

### Phase 3: Content Strategy Development
Keyword research, content planning, internal linking structure.

### Phase 4: Action Plan & Timeline
Output prioritized action list with expected results timeline.

## Output Format
- **Technical Audit Report** (HTML issues, structured data recommendations, performance scores)
- **Competitor Analysis Matrix** (Comparison table)
- **Keyword Matrix** (Excel/CSV with volume, difficulty, intent)
- **Content Calendar** (3-month plan)
- **Action Priority List** (P0-P3 with specific tasks)
- **Ranking Projection Timeline** (Chart showing expected improvements)

## Examples

**Example 1: E-commerce Site SEO**
```
Website: example-shop.com
Current ranking: Page 3 for main keywords
Output:
- Technical: Fix duplicate meta tags, add product Schema, improve LCP from 3.2s to <2.5s
- Keywords: 5 core keywords, 20 long-tail, 50 question-based
- Content: 10 product comparison articles, 5 buying guides
- Timeline: Page 2 in 1 month, Page 1 in 3 months
```

**Example 2: B2B Service Site**
```
Website: b2b-service.com  
Target: Baidu + Google
Output:
- Baidu: Focus on Baijiahao, Zhihu, forum backlinks
- Google: Technical SEO + E-E-A-T content
- Competitors: Analyze 3 industry leaders' backlink profiles
- Action: P0 - Fix broken links, P1 - Create case studies, P2 - Guest posting
```

## Notes
- Requires website URL or sitemap for analysis
- Optimization suggestions are quantifiable with specific code changes
- Ranking timelines are estimates based on industry data
- Never recommend black-hat SEO techniques
- Baidu and Google have different algorithm priorities (Baidu: authority signals, Google: E-E-A-T)
- Regular monitoring and adjustment needed
