# 跨境多国税务速查（Tax Compliance Check）

> 报了六国税、熬了三个通宵、还算不准——输入销售国家，一分钟看清多国税率、申报周期、注册门槛和特殊规则。

覆盖 8 大目标市场税制，支持 EU OSS 一站式申报方案，适用于跨境电商卖家税务合规自查。

📖 详细使用说明见 [SKILL.md](./SKILL.md)
📊 完整数据见 [references/tax-data.md](./references/tax-data.md)

---

## 快速上手

直接说下面任意一句：

> 「我在亚马逊卖到德法意西澳美六个国家，税怎么报？」
> 「我月销 3 万美元，卖到德国和美国，需要注册 VAT 吗？」
> 「Amazon 代扣了 VAT，我还需要申报吗？」
> 「EU 怎么报税最省事？」

## 核心输出

| 输出类型 | 说明 |
|---------|------|
| 多国税制总览 | 税率、门槛、周期并排对比 |
| 注册义务检查 | 基于用户销售额判断是否触发注册 |
| 申报日历 | 各国截止日一目了然 |
| EU OSS 方案 | 一站式申报指南 + 推荐注册地 |
| 税务优化建议 | 合规红线 + 省税策略 |

---

⚠️ AI 输出为税务参考，不构成税务建议。最终申报以当地税务局要求为准。
