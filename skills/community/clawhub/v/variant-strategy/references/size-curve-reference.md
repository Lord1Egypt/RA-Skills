## Size Curve Optimization Principles

### Minimum Viable Size Range
- **Apparel launch**: S, M, L, XL minimum (captures 77-85% of demand)
- **Expanded range**: Add XS and XXL when initial sizes validate (captures 95%+)
- **Footwear launch**: Cover peak ±2 sizes (e.g., Women's 6-10 captures ~84%)

### Inventory Allocation by Size
The recommended approach is sales-velocity-weighted allocation:
1. Use the category benchmark curve as starting allocation
2. After 30 days, adjust based on actual sell-through rates
3. Maintain 2-week safety stock on peak sizes (M, L typically)
4. Allow 1-week safety stock on edge sizes (XS, XXL)

### Size-Related Return Rate Benchmarks
| Category | Average Return Rate | Size-Related Returns |
|---|---|---|
| Women's apparel | 25-35% | 40-50% of returns cite sizing |
| Men's apparel | 15-20% | 30-40% of returns cite sizing |
| Footwear | 20-30% | 50-60% of returns cite sizing |
| Accessories | 8-12% | 10-15% of returns cite sizing |

### Reducing Size-Related Returns
- Include detailed size chart with body measurements (not just S/M/L labels)
- Add "true to size" / "runs small" / "runs large" indicator based on review data
- Show model measurements and size worn in lifestyle photos
- Consider fit quiz integration for DTC stores (reduces returns 15-25%)
- Analyze return data by size to identify systematic fit issues
