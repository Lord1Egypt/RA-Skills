## Description: <br>
Search Shopify stores and products with PPSPY. Use this skill as a general Shopify library and Shopify spy tool for store, product, theme, and category research. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, operators, and commerce researchers use this skill to query PPSPY for Shopify store, product, theme, traffic, category, and bestselling-product intelligence. It requires a PPSPY API key and consumes PPSPY account credits for store and product search records. <br>
