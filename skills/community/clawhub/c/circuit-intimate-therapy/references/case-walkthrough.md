# 咨询案例走完整流程(Case Walkthrough)

## 使用方式

读取本份的典型场景:
- 用户给一段案例描述(冲突循环、冷战、出轨、控制-顺从循环、宽恕困难等),要求"电路层诊断"
- 用户在咨询师培训 / 督导 / 论文写作场景下,需要一个**可复用的工作流模板**
- 已经掌握元件/拓扑/二极管的知识,但不知道**先后顺序**

不要倒出整个文件。**按用户当前所在阶段(采集→读数→拓扑→二极管→干预→演示)挑出对应小节**;用户给案例时套模板,按六步走。

---

## 六步流程总览

```
① 信息采集(safety check + 现象描述)
   ↓
② 电路读数(R / L / C / DC + |Z| / φ + ω₀)
   ↓
③ 拓扑判定(串联 vs 并联;阻尼程度)
   ↓
④ 二极管检查(单向 / 阈值 / Zener / 整流哪只坏)
   ↓
⑤ 干预排序(六手按优先级)
   ↓
⑥ 演示选择(配 1–2 个 app 让来访者自己玩一遍)
```

每一步给到下一步前,**先核对来访者是否同意你的读数**。这是苏格拉底式教学,不是单方面诊断。

---

## ① 信息采集 —— 安全 + 现象

### 安全清单(先过这一关)

只要案例触及以下,**立即跳出电路框架**:
- 自伤 / 自杀念头
- 家庭暴力(身体 / 性 / 经济)
- 强迫性出轨 / 控制
- 儿童虐待 / 忽视
- 严重精神疾病急性发作

→ 给当地紧急资源(中国大陆心理援助热线 400-161-9995;家暴庇护与法律援助;儿童保护热线 12345),**确认安全后**再问是否回到电路框架。

### 现象采集九问(按需问 4–6 个,不必全问)

按"压力源 / 反应模式 / 时序 / 边界 / 修复 / 外部"六个维度:

| 维度 | 问题 | 在测什么 |
|---|---|---|
| 压力源 | 最近三次大冲突的导火索是什么? | 触发频段(测 ω₀) |
| 压力源 | 你们各自工作 / 健康 / 经济压力有多大? | V 振幅 |
| 反应模式 | 一方表达不满,另一方第一反应是什么? | 二极管极性 |
| 反应模式 | 你们俩的反应方式是更像还是更不一样? | 拓扑(串/并) |
| 时序 | 上一次大冲突过了多久才真的"翻篇"? | Q / 阻尼 |
| 时序 | 情绪是马上来,慢一拍,还是当时没事后来才发作? | φ(测电感/电容) |
| 边界 | 各自在家外面有多少自己的生活? | R |
| 修复 | 重大伤害(背叛 / 严重失言)能不能扛过去?怎么扛的? | Zener |
| 外部 | 你们俩对同一件事的回忆,经常一致还是经常完全不同? | 整流器是否被污染 |

## ② 电路读数 —— 给出 (R, L, C, DC) + (|Z|, φ) + ω₀

把九问的答案翻译成五个**结构化读数**:

### R(外部耗散)
- 大:朋友圈、运动、独立工作、独处时间充足
- 小:24/7 黏一起 / 没朋友 / 工作也是同一个

### L(关系惯性)
- 大:老套路应对一切;反应慢但持久
- 小:行为脚本未沉淀(新婚 / 跨文化 / 创伤后重建)

### C(压力蓄水池)
- 大:能装能忍,情绪悄悄涨
- 小:一点小事就炸

### DC(承诺基线)
- 强正:积极互动 ÷ 消极互动 ≥ 5;愿意为对方共同未来牺牲
- 弱正:还在一起但已开始合理化"也许我配不上更好的"
- 反向:已在想象离婚;基线情绪是负

### |Z|(反应钝感度)
- 极小 → 过载敏感;极大 → 钝感疏离
- 适中 = 健康

### φ(反应时序)
- → +90°:滞后型 / 反刍型(强电感)
- → 0°:即时耗散(纯阻)
- → -90°:警觉 / 隐忍(强电容)

### ω₀ = 1/√(LC)
- 高:高频小事就引爆(每天通勤一句话不合就吵)
- 低:长周期慢压力才引爆(代际冲突 / 慢性病 / 十年量级转型)

读数完成后用一句话总结:**"听起来你们家是 (R/L/C 偏向…) + (DC …) + (敏感频段在 …)。我说的对吗?"**

---

## ③ 拓扑判定 —— 串/并 + 阻尼

### 串联 vs 并联

判别问题:"你们俩的反应模式是更像,还是更不一样?"

| 答 | 拓扑 | 落点 |
|---|---|---|
| "我们想得一模一样、什么都同频" | 串联 LC | 燃烧型(危险) |
| "他用他的方式接,我用我的方式接,不一样但能接住" | 并联共轭 | 健康(若 R 适中)/ 内耗型(若 R = 0) |
| "他根本不接,我自己处理" | 退化为单元件 | 室友型 / 隐忍型 |

### 阻尼判别

判别问题:"上一次大冲突过了多久才真的翻篇?翻篇后再提还会不会重燃?"

| 答 | 阻尼 | 含义 |
|---|---|---|
| "永远翻不完" | R = 0 | 慢性内耗(并联永动) |
| "几个月才过 / 重燃严重" | 欠阻尼 | 振荡缓慢衰减 |
| "几天到一周内能过 / 提了不重燃" | 临界附近 | **健康** |
| "半小时之内一切归零、不再提" | 过阻尼或抑制 | 死板 / 假性翻篇(可能是隐忍) |

---

## ④ 二极管检查 —— 哪只坏了

按四性质走一遍,**只挑明显的 1–2 只**展开,不必全检:

### 单向性
- 有没有 stonewalling?(冷战不回)
- 有没有 dismissing 入向阻断?(他关心你,你接不进来)
- 有没有 lesser interest 的不对称?(谁更愿意低头?长期是同一个人吗?)

### 正向阈值
- 是什么强度的事件之后他/你才**真的**有反应?
- 有没有微表情漏电流?(他想 cool 但闪过一下)
- 自我表露的阈值在哪?(轻度披露能不能开启互惠?)

### Zener(反向击穿 / 修复通道)
- 严重伤害(背叛 / 严重失言 / 一次大错)能不能扛?怎么扛的?
- 是低 Zener(任何小事就当背叛)?
- 还是高 Zener(扛但不放,后期突然崩)?

### 整流(认知偏差)
- 同一件事的回忆是不是经常完全不同?(被污染整流器)
- 有没有积极错觉的有意维护?(看见对方的好)
- 自我服务偏差是否极端?(成功是我功劳,失败是他责任)

完成后用一句话:**"听起来你们家最明显的二极管病是 (X)。"**

---

## ⑤ 干预排序 —— 六手按优先级

调用 `diagnosis-intervention.md` 的决策树,**每个案例最多挑 2–3 手**(不要列出全部六手淹没来访者):

1. **危机优先** → 安全资源(本步在 ① 已过滤,这里再核对一次)
2. **DC < AC?** → 抬 DC(④ 稳偏置)优先于一切
3. **拓扑明显错?** → 改拓扑(⑤)
4. **二极管病明显?** → 调二极管(⑥)
5. **R 总能加** → 任何阶段都可以(①)
6. **L 是长程项目** → 不指望几次谈话改 L(②)
7. **C 扩容必须配泄洪**(③)

每一手要给:**做什么 + 怎么做 + 多久 + 谁主动**。

## ⑥ 演示选择 —— 配 1–2 个 app

参考 `apps-guide.md`。规则:

- 一次咨询**不超过 2 个 app**(注意力会分散)
- **先讲清楚静态映射,再演示动力学**(app 是实验台,不是 PPT)
- 让来访者**自己拖滑块**,你只问问题
- 留作业:"回家用 app11 把你们的电路调到目标态,下次带来讨论"

# 示例案例 1:燃烧型 + 击穿型(双重病理)

## 来访者描述

A 女(33)+ B 男(35),结婚 6 年,一儿子(3)。来访主诉:**"我们三天一大吵,但不吵的时候特别甜。今年他出过一次轨,我'原谅'了,但每次他出差我就开始想这件事,然后又吵。"**

A:"他能完全懂我,我们俩的反应一模一样。"
B:"我们什么都同频。"
A:"我们朋友都说看不懂我们俩 —— 这么相爱怎么吵这么凶。"

## ① 信息采集

- 安全:无 IPV、无自伤、出轨为单次但伤害深。✓ 通过。
- 压力源:工作、孩子、双方父母期望。
- 反应模式:**A 哭闹 + 强迫沟通,B 也哭闹 + 强迫沟通**(同步反应,不是 demand-withdraw)。
- 时序:大冲突当天爆发,3–5 天和好,但**和好后再被勾起就立刻重燃**。
- 边界:**朋友圈高度重叠,几乎无独立社交;独自爱好极少**。
- 修复:"原谅"后**反复重燃**,B 说"她从没真的过去"。
- 外部:朋友说"你们这么有戏"。

## ② 电路读数

| 量 | 读数 | 依据 |
|---|---|---|
| R | **极小** | 朋友圈重叠、无独立爱好 |
| L | **小** | 6 年但反应仍像新婚那样剧烈,没沉淀 |
| C | **小** | 一点小事就炸 |
| DC | **弱正** | 还相爱,但出轨已使基线下移 |
| \|Z\| | **小** | 任何刺激都引爆巨大流动 |
| φ | ≈ 0(同步) | 都即时反应 |
| ω₀ | **高** | 高频小事(每周 1–2 次) |

## ③ 拓扑判定

- "我们一模一样" → **串联 LC**
- "和好后还重燃" → **欠阻尼,Q 极高**

## ④ 二极管检查

- **Zener 阈值过低**:出轨后名义上"原谅",但每次小触发又激活反向 → Zener 没真的吸收能量
- 整流器 OK(双方对事件回忆较一致)
- 没有明显 stonewalling,没有 demand-withdraw

## ⑤ 干预排序(给本对夫妻的处方)

按优先级**前三手**:

1. **改拓扑(⑤)从串联到并联**
   - **做什么**:鼓励双方各自发展独立朋友圈、爱好
   - **怎么做**:每周各自有一个晚上是"自己的朋友的"
   - **多久**:6 个月起
   - **理由**:燃烧型唯一长期出路是改拓扑

2. **调 Zener 阈值(⑥)处理出轨遗留**
   - **做什么**:**主动激活**而非压抑反向能量(Zener 受控泄放)
   - **怎么做**:固定时间 + 固定话题去**完整地谈这件事**(而不是每次小触发偶发地谈)
   - **多久**:每两周一次,持续到反向冲击不再激活(可能 3–6 个月)
   - **理由**:扛但不放 = 高 Zener 后期崩

3. **加 R(①)**
   - **做什么**:扩大独立社交(可与第 1 手合并)
   - **怎么做**:见上
   - **理由**:R 是这一型最易调,效果直接

(暂不动 L —— 长程,先看前三手 6 个月后)

## ⑥ 演示选择

- **本次**:`~/circapps/app05-conjugate.html` —— 让他俩看串/并联对比,理解为什么"一模一样"是危险信号
- **回家作业**:`~/circapps/app11-circuit-designer.html` —— 选燃烧型模板,试着调到 validator 区,讨论改了哪些参数 → 这些在生活里对应什么

# 示例案例 2:隐忍型 + Demand-Withdraw(单手处方)

## 来访者描述

C 男(45)+ D 女(43),结婚 18 年,两女儿(15、12)。来访主诉:**"我们 18 年没吵过架,但她最近说要离婚。我完全没准备,以为我们没问题。"**

C:"我从不主动提矛盾,她也不提。"
D:"我提过几次,但他总是'嗯'一声就转身去工作。我已经放弃了。"
D:"我已经爱不起来了,但我也不知道现在是什么感觉,就是空的。"

## ① 信息采集

- 安全:无 IPV、无自伤。✓ 通过。
- 压力源:孩子青春期、双方父母老去、经济稳定但工作高强度。
- 反应模式:**D 提出问题(导通方向),C 不回应(反向截止)**;**长期累积 D 已不再提**(Demand-Withdraw 已演化为 Withdraw-Withdraw)。
- 时序:**没有"冲突",所以没有"翻篇"**(假性过阻尼)。
- 边界:各自工作、各自朋友圈,**家内交流极少,但仍同住、共同养孩子**。
- 修复:从未发生过需要修复的明显事件,但**关系基线已经悄悄掉到负**。
- 外部:朋友、亲戚都说"你们家好稳"。

## ② 电路读数

| 量 | 读数 | 依据 |
|---|---|---|
| R | 中 | 各自外部生活 |
| L | **大** | 18 年僵化模式 |
| C | **极大,接近击穿** | 18 年隐忍累积 |
| DC | **从强正衰减到弱正接近反向** | "我已经爱不起来了" |
| \|Z\| | 大(对外) | 朋友看不出 |
| φ | → -90° | 强电容,情绪先于事件浮动 |
| ω₀ | **极低** | 18 年量级慢压力才显化 |

## ③ 拓扑判定

- 不是串联(各走各的) —— 但也不是健康并联
- **更像 D 走电容支路独自承担,C 走电感支路(老套路)** —— 表面像并联,**但 R 内部支路 = 0(没有沟通)** 所以无法收敛
- 阻尼:**伪过阻尼**(看起来死板,其实是 C 还在偷偷充电)

## ④ 二极管检查

- **C 的反向截止极致 = Stonewalling**(早期是 Withdraw,18 年后已固化)
- D 的二极管:**导通已停**(她不再提了)
- Zener:从未被激活过,因为反向冲击都被 C 的 stonewalling 吃掉了 —— 但**电容 C 已快击穿**

## ⑤ 干预排序

这是**最难的一类:DC 已弱正、电容快击穿、又是 18 年模式**。优先级**前两手**:

1. **稳 / 抬 DC(④)** —— **危机干预级别**
   - **做什么**:在 D 决定离婚之前,**先看 DC 还能不能抬起来**
   - **怎么做**:Gottman State of the Union 每周仪式;共同记忆唤起(早期照片、第一次见面);共同未来想象(如果再给 5 年,你想要什么)
   - **多久**:4–6 周,**这是诊断窗口**;如果 DC 抬不起来,后续治疗的基础就没了
   - **理由**:DC 反向化是最危险的,先抢这一项

2. **改 C —— 扩容必须配泄洪(③)**
   - **做什么**:**泄洪**(D 18 年累积的事一项一项谈)+ **扩容**(情绪命名 / 表达练习)
   - **怎么做**:专业咨询师陪同(单干会反激);C 的任务是**学习接住**而不是解决
   - **多久**:6–12 个月
   - **理由**:不泄洪等于在等更大的击穿;但单纯泄洪没有扩容会让 D 觉得"还是说不通"

(改 L 几乎不可能,18 年模式;改拓扑暂缓;调二极管在第 2 手里隐含)

## ⑥ 演示选择

- **本次**:`~/circapps/app01-rlc-response.html` —— 让 C 看大 C + 小 R 的波形,**直观看到电容慢慢充电直到突然击穿**
- **下次咨询**:`~/circapps/app08-demand-withdraw.html` —— 让 C 看自己的"反向截止"在 18 年里做了什么,以及"偶尔回应一下"的电路效果

## 流程的常见误用

避免下列做法:

| 误用 | 为什么有问题 | 怎么改 |
|---|---|---|
| 跳过①直接给读数 | 来访者没有共识就被诊断,会抗拒 | 每一步都核对"我说的对吗" |
| 一次列六手干预 | 来访者无法消化,执行率为 0 | 一次最多 2–3 手 |
| 立刻推 4 个 app | app 是工具不是装饰;注意力分散 | 一次 1–2 个 |
| 把电路类比当成事实 | "你 L 是 0.5 H" —— 来访者认为这是测出来的 | 类比是同构不是同一,聚焦结构性预测 |
| 跳过安全清单 | 在 IPV / 自伤场景里建议加 R 是危险的 | ① 永远先过 |
| 苦劝来访者"接受我的诊断" | 这不是医学诊断,是协作建模 | 苏格拉底式;让来访者自己定位 |
