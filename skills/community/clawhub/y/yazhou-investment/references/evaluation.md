# 崖州区 招商评估框架与背景调查标准
> 来源：崖州区政府产业规划、崖州湾科技城管理局公开信息 | 更新日期：2026年4月

---

## 一、招商评估维度与评分标准

### 1.1 标准评估框架

| 评估维度 | 权重 | 评分要点 |
|---------|------|---------|
| **产业契合度** | 30% | 是否属于园区六大重点产业；与崖州区核心产业的协同性 |
| **企业实力** | 25% | 营收规模、资产规模、融资情况、资质认定（高企/独角兽等） |
| **科技创新能力** | 20% | 专利数量与质量、研发投入比例、核心技术团队 |
| **带动效应** | 15% | 能否引入上下游配套企业、带动产业链集聚 |
| **可落地性** | 10% | 在崖州湾科技城注册运营的可行性、核心团队到位程度 |

### 1.2 契合度评分说明（1-10分）

| 分数 | 含义 | 建议动作 |
|-----|------|---------|
| 9-10分 | 高度契合，产业方向完全匹配，且企业实力强 | **优先重点跟进**，建议管理局领导出面洽谈 |
| 7-8分 | 较契合，产业方向匹配或企业综合实力突出 | **积极跟进**，安排城小二专项对接 |
| 5-6分 | 一般契合，部分方向有交集 | **保持联系**，观察后续发展 |
| 3-4分 | 弱契合，关联度不高 | **一般关注**，有明确需求再跟进 |
| 1-2分 | 基本不契合，产业方向冲突或企业资质不足 | **暂不跟进**，不建议浪费资源 |

## 二、优先招引企业标准

### 2.1 高优先级（重点跟进）

- 深海科技或南繁种业领域独角兽企业
- 行业细分领域前5名企业
- 拥有国家级实验室/研发中心的企业
- 拟将崖州区作为全球/区域研发总部的企业

### 2.2 次优先级（积极跟进）

- 国家级高新技术企业（有意从三亚市外迁入）
- 行业细分领域前20名企业
- 有明确融资需求且符合园区产业方向的成长期企业

### 2.3 一般关注（建议先调查）

- 传统制造类企业（需评估是否有数字化转型方向）
- 与崖州区主导产业关联度不强的服务业企业

---

## 三、谨慎引入 / 不引入类型

| 类型 | 说明 |
|------|------|
| 高污染制造业 | 与生态旅游定位冲突 |
| 劳动密集型低端加工业 | 不符合园区科创定位 |
| 纯地产开发 | 管理局有专项管控 |
| 与国家政策限制相违背的行业 | 依据海南省相关规定 |

## 四、企业背景调查清单

### 基础信息核查
- 企业全称、注册地、成立时间
- 实控人/法定代表人背景
- 企业类型（国企/民营/外资/上市公司）
- 主营业务与主要产品/服务
- 员工规模与核心团队构成

### 经营与财务情况
- 近3年营收与利润趋势
- 资产规模（总资产/净资产）
- 融资情况（轮次/金额/投资机构）
- 是否上市（A股/港股/美股）及市值
- 是否存在重大诉讼或经营风险

### 科技创新能力
- 是否认定为高新技术企业
- 专利数量（发明专利/实用新型/外观）
- 是否拥有国家级/省级研发平台
- 核心技术在行业中的地位

### 与崖州区的契合度
- 业务方向是否属于深海科技/南繁种业/六大产业
- 是否已有在海南/自贸港的布局意向
- 竞争对手中是否已有企业入驻崖州湾科技城
- 企业是否有"出海"需求（可结合出海服务联合体）

### 招商可行性评估
- 企业是否有注册或搬迁意愿
- 核心团队是否愿意到崖州湾科技城
- 是否有明确的政策需求（税收/人才/场地）
- 预计落地时间与规模

## 五、行业分析要素

当用户要求分析某个行业时，应覆盖以下维度：

1. **行业规模与增速**：全球/中国/海南的市场规模及增长率
2. **竞争格局**：头部企业（全球/中国 TOP 10）、市场集中度
3. **发展趋势**：技术路线变化、政策影响、主要驱动因素
4. **龙头企业画像**：什么类型的企业是行业领导者
5. **与崖州区的关联**：该行业企业在崖州区发展的优势（政策/地理/资源/配套）
6. **潜在挑战**：与国内其他园区相比，崖州区的竞争劣势
7. **重点目标企业**：建议优先招引的3-5家企业名单及理由
