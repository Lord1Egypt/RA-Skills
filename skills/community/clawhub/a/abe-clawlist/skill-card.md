## Description: <br>
Clawlist helps agents plan, execute, track, and verify multi-step, long-running, or ongoing monitoring tasks with checkpoint validation. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[abeltennyson](https://clawhub.ai/user/abeltennyson) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and operators use clawlist to structure agent work into brainstorming, planning, execution, and verification phases, with checkpoint tracking for one-off projects and ongoing monitoring workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad mandatory activation language may steer agents to apply this workflow more often than users intend. <br>
Mitigation: Require explicit user opt-in before using the skill for long-running, infinite, or monitoring workflows. <br>
Risk: Ongoing automated task execution can continue without clear stop controls or action boundaries. <br>
Mitigation: Define allowed automated actions, set a review or stop date, and require approval before any public, account-affecting, financial, or destructive action. <br>
