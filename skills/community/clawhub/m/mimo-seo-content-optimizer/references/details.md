## 关键词类型详解

| 类型 | 定义 | 示例 | 优化策略 |
|------|------|------|---------|
| 主关键词 | 核心主题词 | "AI写作工具" | 标题首位、H1、开头100字 |
| 长尾关键词 | 细分需求词 | "AI写作工具免费" | H2/H3、段落中 |
| LSI关键词 | 语义相关词 | "人工智能写作" | 自然融入全文 |
| 问题关键词 | 用户问题型 | "AI写作工具哪个好" | FAQ模块、问答式标题 |
| 商业关键词 | 购买意图 | "AI写作工具价格" | CTA区域、对比表格 |
| 地域关键词 | 本地化 | "北京AI写作培训" | Title前部、正文首段 |

## Title优化公式

```
公式1：主关键词 + 修饰词 + 品牌
"AI写作工具推荐2026（10款实测）| 工具名"

公式2：疑问句 + 主关键词
"AI写作工具哪个好？2026年最全测评"

公式3：数字 + 主关键词 + 效果
"2026年最好用的AI写作工具推荐（附免费测评）"

公式4：主关键词 + 差异化词
"最全面的AI写作工具测评：免费/付费/国产全覆盖"

公式5：How-to + 主关键词
"如何用AI写作工具提升10倍效率？完整指南"

公式6：对比型 + 主关键词
"ChatGPT vs 文心一言：2026年AI写作工具终极对决"

Power Words（提升CTR的修饰词）：
中文：最全/必看/实测/免费/独家/深度/权威/终极/完整/最新
英文：Ultimate/Complete/Free/Proven/Expert/Definitive/Top/Best
```

## E-E-A-T优化指南

### Experience（经验）
- 展示作者实际使用经验（第一人称叙述）
- 包含个人案例/截图/数据
- 注明内容创作背景

### Expertise（专业性）
- 作者资质展示（简介/认证/学历）
- 引用权威来源和研究
- 使用专业术语但提供解释

### Authoritativeness（权威性）
- 获得其他权威网站引用/链接
- 作者在其他平台的背书
- 行业奖项/认证/媒体报道

### Trustworthiness（可信度）
- 更新日期标注
- 引用可验证的数据来源
- 透明的利益披露
- 联系方式和About页面

## AI搜索适配策略

### Google AI Overview / SGE
```
关键策略：
1. 直接回答格式：每个H2下第一段直接给出核心答案
2. 结构化列表：步骤/要点用编号/项目符号
3. 事实陈述：数据+来源引用
4. 对比表格：产品/方案对比用表格
5. 独特观点：AI倾向整合多角度，提供差异化见解
6. 引用标记：引用权威来源增加可信度
```

### 百度AI摘要
```
关键策略：
1. 百度知道/百科格式：问答式内容优先
2. 时效性标记：标题和内容加年份
3. 百度百科引用：引用百科内容增加权威
4. 结构化段落：清晰的因果关系链条
5. 图文并茂：配图增加内容丰富度
```

### 语音搜索优化
```
关键策略：
1. 自然语言：使用口语化问答格式
2. 简短回答：FAQ答案控制在40-60字
3. 本地化：包含地理位置信息
4. 问题关键词："附近""怎么""什么""多少"
5. Featured Snippet：争取精选摘要位置
```

## Core Web Vitals优化建议

| 指标 | 目标值 | 优化方向 |
|------|--------|---------|
| LCP | <2.5s | 图片压缩/CDN/预加载关键资源 |
| FID/INP | <100ms | 减少JS执行/代码分割/延迟非关键 |
| CLS | <0.1 | 固定图片尺寸/字体预加载/预留广告位 |

## 关键词布局矩阵

| 位置 | 关键词要求 | 密度建议 |
|------|-----------|---------|
| Title标签 | 主关键词前置 | 30字内 |
| Meta描述 | 主关键词+长尾 | 80-120字 |
| H1标题 | 主关键词 | 出现1次 |
| 首段100字 | 主关键词 | 出现1次 |
| H2/H3 | 长尾关键词 | 各出现1次 |
| 正文 | 关键词+LSI | 自然分布 |
| URL | 主关键词 | 简短英文 |
| ALT标签 | 描述性文本 | 含关键词 |
| 首张图 | 主关键词相关 | 文件名含关键词 |
| 最后一段 | 主关键词+CTA | 总结性出现 |

## 竞品SERP分析方法

```
步骤1: 搜索目标关键词，记录前10结果
步骤2: 分析每个结果的：
  - 标题结构和字数
  - 内容长度（字数）
  - 内容结构（H2/H3数量和用词）
  - 是否有多媒体（图片/视频/表格）
  - 页面加载速度
  - 更新频率/发布时间
步骤3: 找出共性特征和差异化机会
步骤4: 制定超越策略（更长/更新/更深/更好结构）
```

## 异常处理

| 场景 | 处理方式 |
|------|---------|
| 内容过短(<200字) | 建议扩充到800字以上，提供内容大纲 |
| 关键词堆砌(>5%) | 提供去堆砌化建议，改写自然融入 |
| 竞品URL不存在 | 忽略，仅分析自有内容 |
| 内容严重不足 | 建议扩充或提供内容大纲 |
| 内容与标题不符 | 建议重写或调整标题 |
| 关键词无搜索量 | 建议更换为有搜索量的替代词 |
| 高度竞争词 | 建议长尾策略+内容差异化 |
