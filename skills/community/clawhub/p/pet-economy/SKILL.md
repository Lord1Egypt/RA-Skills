# 宠物经济技能

## 核心能力

| 能力 | 说明 |
|------|------|
| 品种选购 | 根据居住条件/预算/时间投入推荐犬猫品种，含纯种/领养渠道 |
| 科学喂养 | 定制BARF生骨肉/商业粮方案，计算每日热量与营养素 |
| 健康管理 | 疫苗接种日历、驱虫提醒、常见疾病症状识别（皮肤病/肠胃/泌尿） |
| 保险对比 | 对比平安/众安/太平洋等宠物医疗保险方案 |
| 行为训练 | 社会化训练、分离焦虑、乱咬乱叫行为矫正教程 |

## 触发场景

- "帮我推荐适合公寓养的狗"
- "猫咪软便怎么办"
- "宠物保险哪个好"
- "制定柯基幼犬喂养方案"
- "狗狗分离焦虑怎么训练"

## 执行流程

### 阶段一：需求识别
1. 收集宠物信息（品种/年龄/体重/绝育状态）
2. 了解主人的生活环境和时间投入
3. 识别当前问题或需求

### 阶段二：方案生成
1. 计算每日热量需求和营养素配比
2. 生成免疫与驱虫时间表
3. 匹配合适的保险产品和训练方案

### 阶段三：持续追踪
1. 生成健康管理日历
2. 训练进度跟踪表
3. 异常症状就医指引

- 喂养方案需根据宠物个体差异调整
- 保险产品信息以保险公司最新条款为准
