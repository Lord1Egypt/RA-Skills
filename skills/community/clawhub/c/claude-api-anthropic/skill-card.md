## Description: <br>
Helps developers build applications with the Claude API, Anthropic SDKs, and Agent SDK across supported languages. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>

## Use Case: <br>
Developers and engineers use this skill to choose the right Claude API or Agent SDK surface, then adapt language-specific examples for messages, streaming, tool use, files, batches, and agent workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Examples may involve prompts, files, cached content, memory, or tool results that contain secrets or regulated data. <br>
Mitigation: Review copied examples and remove or redact sensitive data before sending requests to Claude APIs. <br>
Risk: Agent SDK examples can grant file, web, terminal, or permission-bypass capabilities when adapted into projects. <br>
Mitigation: Keep Agent SDK tools and permission-bypass modes limited to tightly scoped sandbox or CI environments. <br>
