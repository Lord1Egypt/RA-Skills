# Sweating Pattern Differentiation

## When to Use
IF 患者出现白昼自汗或寐中盗汗，且无明显外界高温、运动等生理诱因，同时已排除脱汗（亡阳/亡阴危象）、战汗（正邪交争）、黄汗（湿热染衣）等特殊汗证 THEN 启动本辨证流程。

## Execution Steps

### 1. 初步分类
- **自汗**：白昼时时汗出，动则益甚 → 多属气虚或阳虚。
- **盗汗**：寐中汗出，醒后即止 → 多属阴虚或血虚。
> 注意：不可绝对化，明代张介宾指出“自汗盗汗亦各有阴阳之证”。

### 2. 五型辨证与对应治法
根据伴随症状、汗液性质及舌脉表现，判断以下五型之一：

| 证型 | 主要表现 | 治法 | 基础方剂 |
|------|--------|------|--------|
| 肺卫不固 | 汗出恶风、易感冒、体倦乏力、面色少华、脉细弱 | 益气固表 | 玉屏风散加减 |
| 营卫不和 | 汗出恶风、周身酸楚、时寒时热、或半身汗出 | 调和营卫 | 桂枝汤加减 |
| 心血不足 | 自汗或盗汗、心悸少寐、面色不华、舌淡 | 补养心血 | 归脾汤加减 |
| 阴虚火旺 | 夜寐盗汗、五心烦热、午后潮热、颧红口渴、舌红少苔、脉细数 | 滋阴降火 | 当归六黄汤加减 |
| 邪热郁蒸 | 蒸蒸汗出、汗黏腻、面赤烘热、口苦尿黄、苔黄脉弦数 | 清肝泄热、化湿和营 | 龙胆泻肝汤加减 |

### 3. 动态调整治法
- 自汗日久可伤阴，转为气阴两虚；
- 盗汗日久可伤阳，转为阴阳两虚；
→ 需根据病程变化调整辨证与方药。

### 4. 通用处理原则
各证型均可酌情加入固涩敛汗药，如浮小麦、糯稻根、五味子、煅牡蛎等。

### 5. 鉴别要点（仅用于排除）
- **脱汗**：大汗淋漓如油，伴肢厥脉微 → 危证，立即转诊；
- **战汗**：先战栗后汗出 → 正胜邪退之象，无需干预；
- **黄汗**：汗出色黄染衣 → 多因湿热，另作处理。

## Output
根据辨证结果选择对应基础方剂，并结合患者具体情况加减药物，同时考虑加入固涩敛汗药。
