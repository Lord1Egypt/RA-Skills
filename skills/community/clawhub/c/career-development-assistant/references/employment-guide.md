## 一、简历撰写指南

### 简历结构

```
个人信息
  ├── 姓名、电话、邮箱（必填）
  └── 求职意向（必填，要具体不泛泛）

教育背景
  ├── 学校 + 专业 + 学历 + 时间
  ├── 成绩排名（好才写）
  └── 相关课程（与求职岗位相关的课程）

实习/项目经历（核心部分）
  ├── 公司/项目名称 + 时间 + 职位
  ├── 用STAR法则写：情境-任务-行动-结果
  └── 使用量化成果（"提升效率20%"优于"提高效率"）

技能特长
  ├── 专业技能列在前
  ├── 证书列出与求职相关的主要证书
  └── 语言能力和办公技能

自我评价（可选）
  └── 简短的2-3句话，体现岗位匹配度
```

### STAR法则写法模板

```
不良写法：
"负责公司微信公众号运营"

STAR写法：
"负责公司公众号运营（行动），累计发布80+篇原创内容（任务），
粉丝从2000增长到10000+（结果），单篇最高阅读量5万+（量化）"
```

### 简历注意事项
1. **一页纸原则**：应届生简历不超过一页
2. **针对性**：针对不同岗位微调简历内容
3. **真实性**：不可造假，但可以突出优势
4. **格式简洁**：PDF格式发送，字体统一
5. **联系方式醒目**：手机号和邮箱放在显眼位置

## 二、面试准备

### 面试流程

```
面试前
├── 研究公司和岗位（官网、产品、团队）
├── 准备自我介绍（1分钟/3分钟两个版本）
├── 准备常见问题的回答
├── 准备2-3个问面试官的问题
└── 着装得体，提前15分钟到场

面试中
├── 自信、真诚、专注
├── 回答问题用STAR法则
├── 不懂就说"不太了解，但我可以学习"
└── 结束时简短感谢

面试后
├── 24小时内发感谢邮件
└── 约定时间内没有回复，可礼貌跟进
```

### 常见面试问题

| 问题 | 回答策略 |
|:---:|---------|
| "请自我介绍" | 1-3分钟，突出与岗位匹配的经历 |
| "你的优缺点" | 优缺点都围绕岗位要求 |
| "为什么选我们公司" | 展现你对公司的了解 |
| "职业规划" | 2-3年的短期规划，贴合岗位发展 |
| "期望薪资" | 提前了解行业水平，给出合理区间 |
| "你还有什么问题" | 问团队/成长空间/岗位职责 |

## 三、实习求职渠道

### 主流求职平台

| 平台 | 特点 | 推荐使用场景 |
|:---:|------|------------|
| 智联招聘 | 综合类岗位最多 | 全行业求职 |
| Boss直聘 | 互联网/科技类 | 互联网求职首选 |
| 前程无忧 | 综合类 | 传统行业 |
| 拉勾网 | 互联网垂直 | 互联网岗位 |
| 脉脉 | 职场社交+内推 | 人脉拓展和求职 |
| 实习僧 | 实习岗位为主 | 学生找实习 |
| 学校就业网 | 校内招聘信息 | 校招首选 |
| 企业官网 | 校招信息 | 目标企业求职 |

### 职校学生求职建议
1. **学校的校企合作企业优先投递**
2. **校园招聘会积极参加**
3. **技能大赛获奖经历是硬通货**
4. **1+X证书有助于证明技能水平**
5. **参加职业技能展示活动，建立行业人脉**

## 四、实习须知

### 实习注意事项
1. **签署实习协议**：明确实习期限、工作内容、补贴标准
2. **薪资待遇**：实习补贴一般低于正式员工工资
3. **保险保障**：正规企业应为实习生购买意外险
4. **劳动权益**：每日工作时间一般不超过8小时
5. **实习转正**：提前了解实习期后是否有转正机会

### 维权渠道
- 劳动监察投诉电话：12333
- 学校就业指导中心
- 如有权益受损，保留劳动合同/实习协议和相关证据

## 五、就业市场趋势（2025-2026）

### 热门行业
1. **人工智能/大数据**：持续高增长，人才缺口大
2. **新能源/新能源汽车**：政策驱动，产业扩张期
3. **生物医药**：研发和生产岗位持续增长
4. **智能制造**：工业4.0推动，技师缺口大
5. **数字经济**：电商、直播、数字内容创作

### 技能型人才需求
- 拥有实操技能的中高级技工供不应求
- 年薪30万+的高级技工并不少见
- 智能制造、新能源汽车等领域的技师最紧缺

### 就业建议
1. **技术为王**：有真技术不怕找不到工作
2. **持续学习**：技能更新速度快，要持续学习
3. **证书+经验**：学历不够证书凑，证书不够经验凑
4. **灵活就业**：平台经济下的灵活就业机会日益增多
5. **创业意识**：职校生动手能力强，适合技能型创业
