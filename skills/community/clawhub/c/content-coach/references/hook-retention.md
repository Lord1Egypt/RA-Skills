# Hook 与留存 (Hook & Retention)

> "If you haven't hooked them in the first 5 seconds, the rest of the video doesn't exist." — MrBeast

## MrBeast 的黄金法则

MrBeast 团队公开的原则：

### 前5秒（Hook）的四个类型

1. **惊喜型**：「你绝对猜不到我用30天做了什么」
2. **悬念型**：「我花了10万块验证一个想法，结果让我震惊」
3. **共鸣型**：「如果你也是个总在焦虑的人，这条视频是给你的」
4. **结果型**：「我用这个方法3个月涨了10万粉，今天全部分享」

### 每30秒的检查点

MrBeast 的终极原则：
> "Every 30 seconds, ask: has the viewer had a reason to keep watching?"

如果连续30秒没有新的刺激点，观众就会划走。

### 节奏公式

```
Hook → Payoff → New Hook → Payoff → New Hook → Payoff...
```

每个 Hook 给一个理由留下，每个 Payoff 给一个满足感。

## 数据驱动的标题优化

### 标题公式

| 公式 | 例子 |
|------|------|
| [数字] + [形容词] + [名词] + [结果] | 5个让你涨粉翻倍的内容策略 |
| [如果/如何] + [问题] + [在X时间内] | 如何在30天内从0到1000粉丝 |
| [什么] + [为什么] + [连词] + [解决方案] | 为什么你的内容没人看（以及怎么改） |

### 封面设计原则

- 文字不超过5个字
- 有情绪（惊讶/疑惑/期待）
- 色彩对比强烈
- 人物+文字是最好的组合

## Justin Welsh 的 Hook 法则

- **LinkedIn**：用提问开头，让人想回答
- **Newsletter**：第一段就要给一个具体的洞察
- **Twitter**：前两行决定一切

**通用原则**：
1. 具体 > 抽象（"月入3万" > "提高收入"）
2. 反差 > 平铺（"我大厂裸辞了" > "今天聊聊职业规划"）
3. 对谁说清楚（"给还在犹豫的职场人" > "给大家一些建议"）
