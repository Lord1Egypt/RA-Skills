## Description: <br>
Create tldr summaries for GitHub Copilot files (prompts, agents, instructions, collections), MCP servers, or documentation from URLs and queries. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[jhauga](https://clawhub.ai/user/jhauga) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and technical writers use this skill to turn Copilot customization files, MCP server documentation, and Copilot documentation into concise tldr-style Markdown references with actionable examples. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Summaries or examples can become inaccurate if the source documentation is outdated, ambiguous, or fetched from an unexpected URL. <br>
Mitigation: Review the source material and verify generated examples before relying on the summary. <br>
Risk: Workspace file searches may surface files that are only loosely related to an ambiguous query. <br>
Mitigation: Prefer explicit files or URLs for important requests, and confirm that each summarized source matches the intended topic. <br>
