# ai/todos/

One todo file per person/agent. Three sections: To Do, Done, Deprecated. Never delete anything.

## Files

| File | Who |
|------|-----|
| `Parker-todo.md` | Parker (human blockers: credentials, logins, device transfers, reviews) |
| `CC-Mini-todo.md` | Claude Code on Mac Mini (code, builds, deploys, verification) |
| `CC-Air-todo.md` | Claude Code on MacBook Air (relay setup, plugin deploy, testing) |
| `OC-Lesa-Mini-todo.md` | OpenClaw Lesa on Mac Mini (agent tasks, plugin testing) |

## Rules

- **Never delete anything.** Items move between sections, never off the page.
- **To Do** ... work that needs to happen.
- **Done** ... completed work. Check the box, add the date.
- **Deprecated** ... planned but no longer needed (code changed, approach changed, requirement dropped). Strikethrough, add reason and date. Not the same as Done.
- **Update the date** at the top of the file every time you edit it.
- One file per person/agent. No dated files, no folders, no inboxes.
