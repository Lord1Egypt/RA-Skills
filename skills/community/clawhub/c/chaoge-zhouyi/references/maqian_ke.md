# 马前课

## 概述

《马前课》，传为三国诸葛亮撰，是一部**预言后世**的谶纬之作。全书共十四课，每课预言一朝代或时期，从三国一直预言至未来。

---

## 作者简介

**诸葛亮**（181-234），字孔明，号卧龙，三国蜀汉丞相。著名政治家、军事家、发明家、预言家。相传撰《马前课》《诸葛神数》等预言书。

---

## 成书背景

诸葛亮精通易理、天文、兵法，感悟天机，撰《马前课》十四课，预言后世。每课一卦，含蓄隐晦，需细品方知其意。

## 核心内容

### 一、马前课全文

**第一课（三国）**：
```
无力回天，鞠躬尽瘁。
阴居阳拂，八千女鬼。
```

**第二课（两晋）**：
```
火上有火，光烛中土。
称名不正，江东有虎。
```

**第三课（南北朝）**：
```
扰扰中华，山河无主。
二三其位，从此推下。
```

**第四课（隋唐）**：
```
十八男儿，起于太原。
动则得解，日月丽天。
```

**第五课（五代十国）**：
```
五十年中，其数自起。
八牛垂手，弓长起。
```

**第六课（宋朝）**：
```
云无心，见真龙。
泛泛不系，万里望风。
```

**第七课（元朝）**：
```
有客西来，至东而止。
木火金水，洗此大耻。
```

**第八课（明朝）**：
```
日月丽天，群阴慑服。
百灵来朝，双羽四足。
```

**第九课（清朝）**：
```
水月有主，古月为君。
十传绝统，相敬如宾。
```

**第十课（民国）**：
```
豕后牛前，千人一口。
五二倒置，朋来无咎。
```

**第十一课（未来）**：
```
四门乍辟，突如其来。
晨鸡一声，其道大衰。
```

**第十二课（更未来）**：
```
新圣人出，大有为焉。
天地反覆，斯文乃兴。
```

（其余二课，预言更远未来）

### 二、预言内容

**三国**：
1. 无力回天：诸葛亮无力回天
2. 鞠躬尽瘁：诸葛亮鞠躬尽瘁
