# 天禧视觉设计规范

## 配色系统

```css
:root {
  --bg-primary:    #F5F6FA;   /* 主背景·浅灰白 */
  --bg-surface:    #FFFFFF;   /* 顶栏底栏·纯白 */
  --brand:         #6C63FF;   /* 天禧品牌紫 */
  --color-high:    #FF4757;   /* 高优先级·红 */
  --color-mid:     #F0920A;   /* 中优先级·橙（加深适配浅色背景） */
  --color-low:     #18A85A;   /* 低优先级·绿（加深适配浅色背景） */
  --text-primary:  #1A1A2E;   /* 主文字·深色 */
  --text-secondary:#6B7280;   /* 次要文字 */
  --border:        rgba(108, 99, 255, 0.18); /* 边框 */
}
```

## 字体规范

```
中文优先栈：'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif
禁止使用：Inter / Roboto / Arial
字重：正文 400，标签/徽章 600，数字高亮 700
```

## 反套路原则（Frontend Design Anti-Pattern）

```
❌ 紫色渐变装饰（区别于品牌紫的功能性使用）
❌ Inter 字体
❌ 居中卡片网格
❌ 纯装饰动画
❌ 随机配色（颜色必须有语义）
❌ 深色背景（可读性差，用户体验优先）

✅ 浅色背景（#F5F6FA 主背景 + #FFFFFF 顶栏底栏）
✅ 颜色语义化（红=高优/危险，橙=中优，绿=低优/安全）
✅ 中文字体优先
✅ 信息层级清晰（顶栏状态 + 主图 + 底栏统计）
✅ 动效仅用于交互反馈（节点展开/收起）
✅ 品牌色作为高亮和边框，不做大面积填充
```

## Markmap 配置

```yaml
markmap:
  colorFreezeLevel: 2       # 冻结颜色到二级，保持一级节点颜色一致
  initialExpandLevel: 3     # 初始展开3层
  maxWidth: 280             # 节点最大宽度
  paddingX: 16              # 节点水平内边距
  duration: 400             # 展开动画时长(ms)
```

## 顶部信息栏规范

```
布局：左侧徽章组 + 右侧图例
徽章内容：考试类型 / 距考天数 / 目标分数
徽章样式：半透明背景 + 对应语义色文字 + 圆角20px
图例：高优先(红点) / 中优先(橙点) / 低优先(绿点) / ⚠️薄弱点 / 📖待补充
```

## 底部统计栏规范

```
内容：考点总数 / 薄弱点数 / 建议总时长 / 品牌署名
布局：左侧统计 + 右侧品牌
数字高亮色：--brand (#6C63FF)
品牌署名："天禧 AI Claw · 考前冲刺"，半透明品牌紫
```
