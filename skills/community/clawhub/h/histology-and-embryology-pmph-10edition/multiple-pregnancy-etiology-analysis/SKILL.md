# 多胎妊娠成因分析

## 适用场景
- 一次分娩产出 ≥3 个新生儿
- 患者接受过辅助生殖技术（如 IVF、ICSI、PGD）或促排卵治疗
- 需判断多胎类型及其临床成因

## 执行步骤
1. **确认多胎定义**：一次分娩产出两个以上新生儿（本技能聚焦三胎及以上）。
2. **分类多胎类型**：
   - **单卵性**：一个受精卵分裂为三个或更多胚胎（罕见）
   - **多卵性**：多个卵子同时受精（如三卵三胎）
   - **混合性**：部分来自同一受精卵分裂，部分来自不同受精卵（最常见）
3. **评估辅助生殖技术影响**：
   - 检查是否使用促性腺激素导致超排卵
   - 确认是否进行体外受精-胚胎移植（IVF-ET），并记录移植胚胎数量
4. **输出结果**：明确多胎类型、可能成因及是否与辅助生殖技术相关。

## 注意事项
- 自然状态下多胎妊娠发生率极低，显著上升主要归因于辅助生殖技术应用。
- 多胎妊娠显著增加母婴并发症风险，建议提示临床严密监测。
