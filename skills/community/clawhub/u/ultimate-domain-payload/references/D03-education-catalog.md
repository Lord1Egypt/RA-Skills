# D03 教育领域任务目录

## 领域定义

教育领域(D03)关注个体知识获取、技能培养和能力提升的全过程。核心驱动需求：N4自我实现。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人学习规划 | 学校运营管理 |
| 知识获取方法 | 课程内容制作 |
| 技能训练指导 | 教学方法研究 |
| 学习效果评估 | 教育政策制定 |
| 学习资源选择 | 教材出版发行 |
| 学习习惯养成 | 学术研究管理 |
| 终身学习规划 | 教育技术开发 |
