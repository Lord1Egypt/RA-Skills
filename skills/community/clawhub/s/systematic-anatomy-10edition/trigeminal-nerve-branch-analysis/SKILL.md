# 三叉神经分支、分布与功能分析

## 触发条件
当出现以下情况之一时应用本技能：
- 头面部特定区域感觉减退或丧失
- 咀嚼肌无力或瘫痪
- 角膜反射消失
- 下唇、上唇、鼻翼、额部等区域皮肤或黏膜感觉异常

## 执行步骤

1. **确认三叉神经基本组成**：
   - 三叉神经（V）为混合性脑神经，包含一般躯体感觉纤维（胞体位于三叉神经节）和特殊内脏运动纤维（起自三叉神经运动核）。

2. **按三大分支逐一评估**：
   - **眼神经（V1）**：仅含感觉纤维，经眶上裂入眶。若额顶、上睑、鼻背或角膜感觉障碍，提示V1受损。
   - **上颌神经（V2）**：仅含感觉纤维，经圆孔出颅。若下睑、鼻翼、上唇、上颌牙或腭部感觉异常，提示V2受损。
   - **下颌神经（V3）**：混合性，经卵圆孔出颅。若下唇以下皮肤、舌前2/3、颊部感觉障碍，或咀嚼肌无力，提示V3受损。

3. **判断功能类型**：
   - 感觉障碍 → 涉及感觉根或相应分支
   - 咀嚼肌瘫痪 → 涉及运动根（仅存在于V3）

4. **综合定位**：
   - 结合症状分布区域、颅底孔道（眶上裂、圆孔、卵圆孔）及神经走行，确定最可能受损的分支及损伤部位。

5. **注意特殊传导通路**：
   - 舌神经接受面神经鼓索（味觉+副交感），耳颞神经携带舌咽神经副交感纤维至腮腺，泪腺神经接受面神经副交感控制——这些不属三叉神经本身功能，但影响临床表现。

> 提示：运动根还包含传导咀嚼肌本体感觉的纤维，投射至三叉神经中脑核，此功能常被忽略。
