# Insurance Agent Training Question Bank Templates / 保险代理人训练问题库模板

## 模板说明 / Template Description

本文档提供标准化的问题库模板，支持从产品文档自动生成题库。
每类问题均按5级难度梯度设计，适用于L1-L3各级别代理人。

---

## 一、产品知识题 / Product Knowledge Questions

### 1.1 选择题模板 (Multiple Choice)

```markdown
## 【单选题】⭐ 基础题

**题目**: [产品名称]的[核心条款/数值/规则]是？

A. [选项A - 错误]
B. [选项B - 正确]
C. [选项C - 干扰项]
D. [选项D - 混淆项]

**正确答案**: B

**知识点**: [章节名，例如：等待期条款]

**适用级别**: L1新人

**参考话术**:
> "这款产品的等待期是90天，这也是行业大多数重疾险的标准等待期。在等待期内发生重疾，我们退还已交保费，但不会赔付保险金。不过，意外伤害导致的重大疾病是没有等待期限制的，这一点对客户很重要。"

---

## 【多选题】⭐⭐⭐ 进阶题

**题目**: 满足以下哪些条件时，[产品名称]可以申请[保险金类型]？

A. [条件A]
B. [条件B]
C. [条件C]
D. [条件D]

**正确答案**: [A,C]

**知识点**: [条款名]

**适用级别**: L2进阶

**解析**:
- A正确，因为...
- B错误，因为...
- C正确，因为...
- D错误，因为...

**评估标准**:
- 优秀：能准确识别全部正确选项，并解释原因
- 合格：选择正确，但解释不完整
- 不合格：漏选或错选关键选项
```

### 1.2 计算题模板

```markdown
## 【计算题】⭐⭐⭐⭐ 高阶题

**场景**: [年龄]岁，[职业类型]，[年收入]，现有[保障情况]，希望规划[财务目标]

**计算任务**:
1. 计算该客户的合理保额区间
2. 计算10年缴/20年缴的年缴保费
3. 对比不同缴费期的总保费差异

**参考计算过程**:

| 指标 | 计算公式 | 结果 |
|------|---------|------|
| 生命价值法保额 | 年收入×(65-当前年龄)×10% | XX万 |
| 家庭需求法保额 | 家庭负债+子女教育+养老预留-已有资产 | XX万 |
| 合理保费(10年缴) | 家庭年收入×15% | XX元/年 |
| 合理保费(20年缴) | 家庭年收入×15% | XX元/年 |

**适用级别**: L2-L3

**评估标准**:
- 优秀：两种方法均正确计算，并给出综合建议
- 合格：至少一种方法正确，有基本逻辑
- 不合格：计算错误或逻辑混乱
```

## 二、异议处理题 / Objection Handling Questions

### 异议处理话术框架 (A-C-E Framework)

```
A - Acknowledge（认同情绪）: "是的，我理解您的顾虑..."
C - Clarify（澄清需求）: "您主要担心的是...对吗？"
E - Educate（教育引导）: "其实，根据您的实际情况..."
```

### 2.1 高频异议处理模板

```markdown
## 【异议处理】异议类型：需求认知类

**客户异议**: "我有社保了，不需要商业保险。"

**适用产品**: 医疗险、重疾险

**L1 参考话术（入门级）**:
> "李先生，社保确实是我们国家的基本保障，很重要。不过社保有一个局限，就是有报销比例和范围的限制。比如，进口药、自费药社保是不能报销的，重大疾病治疗中的很多靶向药一个月就要几万块，都是社保外费用。商业保险正好可以报销这部分。我帮您算一下，30岁男性，一年500元的医疗险，社保报销后超过1万的部分可以100%报销，包括社保外的自费药。这个您看对您有没有用？"

**L2 参考话术（进阶级）**:
> "李先生，我先问您一个问题——如果您或家人突然生了一场大病，需要50万的治疗费用，社保能报销多少？
> 实际上，社保平均报销比例大约在50%-60%左右，还有很多特效药、进口器材社保完全不报。50万的治疗费，社保大概只能报20-30万，还有20-30万的缺口需要自己承担。
> 您年收入30万，拿出500元/年买个医疗险，撬动200万的保障，把这个风险缺口完全覆盖住。这是'小钱撬大保障'的逻辑，您觉得值不值？"

**L3 参考话术（专家级）**:
> "李先生，我们做个测算——
> 假设您45岁，标准治疗一场癌症需要30-50万，社保报销约15-25万，自费部分约15-25万。
> 更重要的是：大病治疗期间您可能3-5年无法正常工作，家庭收入中断，但房贷、孩子教育、父母赡养的费用不会停。
> 一个人倒下，最危险的不是医疗费，而是'收入损失'和'家庭责任'。所以我建议您考虑一个'医疗险+重疾险'的组合——医疗险报销治疗费用，重疾险一次性给付，补偿收入损失。
> 您目前的家庭责任有多重？"

**评估标准**:
- 🌟 优秀：主动算账 + 量化风险 + 逻辑递进 + 引入重疾险组合
- ✓ 合格：能反驳异议，有基本逻辑，但说服力不足
- ✗ 不合格：直接否定客户，或未能给出合理解释
```

```markdown
## 【异议处理】异议类型：产品比较类

**客户异议**: "XX公司的产品比你们的好，保费还便宜。"

**适用产品**: 全品类

**L2 参考话术**:
> "张先生，您提到的XX公司确实是大公司，产品也不错。买东西货比三家是对的。
> 不过保险产品比较有个特点——不能只看保费，要看'性价比'和'适合度'。
> 我帮您做个对比表：[产品名称] vs [竞品]"

| 对比维度 | [产品名称] | [竞品] |
|---------|-----------|--------|
| 核心保障 | [我方优势] | [对方描述] |
| 保障范围 | [详细] | [描述] |
| 等待期 | 90天 | 180天 |
| 特色功能 | [独有] | [缺失] |
| 综合性价比 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

> 选保险更重要的是选'条款'，不是选'公司'。因为保险公司受银保监会严格监管，每家都有实力赔付。
> 您方便把他们的条款发给我吗？我帮您逐条对比，找到最适合您需求的方案。
```

```markdown
## 【异议处理】异议类型：信任类

**客户异议**: "保险都是骗人的，到时候这也不赔那也不赔。"

**L1 参考话术**:
> "李先生，我能理解您的担心。这种印象主要来自于两个方面：一是早期保险销售不专业，导致客户买错了产品；二是很多人没有认真看过保险条款，不知道什么赔什么不赔。
> 现在监管要求'双录'，保险合同也越来越透明。我会帮您把条款逐条解释清楚，您签字之前完全理解保障内容。我们公司有'秒赔'服务，只要是条款内的责任，上传资料后24小时内到账。
> 您之前遇到的是什么情况？我帮您分析一下，是产品选择问题还是条款理解问题。"

**评估标准**:
- 🌟 优秀：不反驳客户情绪，先共情，再引导，最后主动提出解决方案
- ✓ 合格：能共情但解决方案模糊
- ✗ 不合格：直接否定客户，或过度承诺
```

### 3.2 高净值客户案例

```markdown
## 【案例分析】⭐⭐⭐⭐⭐ 专家级案例

### 客户档案
| 项目 | 内容 |
|------|------|
| 姓名 | [化名] |
| 年龄/性别 | 50岁男性 |
| 职业 | 私企老板（制造业） |
| 年收入 | 200-500万元 |
| 资产状况 | 净资产2000万+（含企业资产） |
| 家庭状况 | 离异，子女2人（15岁、20岁） |
| 风险担忧 | 企业债务风险、家庭传承、税务风险 |
| 核心诉求 | 资产隔离 + 税务优化 + 子女传承 |

### 需求挖掘问题链
1. "您企业目前负债情况如何？有没有个人连带担保？"
2. "您希望孩子将来继承的是企业还是现金资产？"
3. "您目前的税务顾问有没有做过个人所得税的专项规划？"
4. "您对子女教育金的规划是希望确定性给付还是弹性安排？"

### 专家级方案设计要点

**核心产品**: 终身寿险（高保额）+ 保险金信托

| 方案层次 | 产品/工具 | 功能 | 核心价值 |
|---------|---------|------|---------|
| 第一层 | 高额终身寿险 | 身故保险金杠杆 | 10倍杠杆，指定传承 |
| 第二层 | 保险金信托 | 信托架构 | 防止子女挥霍，按条件分批给付 |
| 第三层 | 年金险 | 养老现金流 | 55岁起每年领取，保证品质养老 |
| 第四层 | 高端医疗 | 医疗资源 | 全国三甲医院绿色通道，海外二次诊疗 |

**关键销售逻辑**:
> "李总，您现在的企业资产和家庭资产没有做好隔离。一旦企业发生债务风险，家庭资产可能被追索。
> 终身寿险的作用是：用5万/年的保费，撬动500万的身故保障，这500万是保险金，不属于遗产，不用于清偿债务，可以完整留给您的两个孩子。
> 如果再通过保险金信托架构，可以设定孩子年满30岁才可领取，且每月领取不超过5万，防止他们一次性拿到大笔资金后挥霍。
> 这是目前高净值客户最常用的'资产隔离+定向传承'的法律工具之一。"

**评估标准**:
- 🌟 优秀：能识别企业债务隔离+传承核心需求，设计保险金信托架构，引入税务顾问联动
- ✓ 合格：识别了传承需求，有基本方案但深度不足
- ✗ 不合格：仍以推销产品为主，未触及核心风险
```

## 四、促成话术题 / Closing Technique Questions

### 4.1 促成信号识别

```markdown
## 【促成话术】促成信号识别与应对

### 常见促成信号（出现即响应）
1. 🔔 "这个产品听起来不错，多少钱一年？"
2. 🔔 "如果我现在买，什么时候生效？"
3. 🔔 "能不能把保费降低一点？"
4. 🔔 "我回去和家人商量一下。"
5. 🔔 "我有个朋友也想买，能不能一起介绍？"

### L1 标准促成话术（假设成交法）
> "李先生，如果这款产品您觉得合适的话，我们今天就把计划书做出来，您看看每年存3万，存10年，保障终身，您看可以吗？"

### L2 进阶促成话术（利益汇总法）
> "李先生，我们回顾一下：您选择了[产品名称]，每年存[X]元，存[X]年，可以获得[核心保障]。今天投保的话，90天等待期后您的保障就生效了。现在是[当前月份]，正好是我们公司的[活动名称]，额外赠送[附加权益]。您看今天把合同签了？"

### L3 专家级促成话术（回马枪+压力促成）
> "李先生，我看您今天对这款产品确实有需求，也很认可我们的方案。
> 我有个担心——您今天回去商量是应该的，但保险产品越早买越好，有两个原因：
> 一是年龄越大保费越贵，每大一岁保费大约贵3%-5%；
> 二是健康状况是不可逆的，如果体检查出什么异常，可能就买不了或需要加费。
> 您今天要不要先把保费最低的方案确定下来？后续有任何调整我们再优化。"

### 评估标准
- 🌟 优秀：能识别2个以上促成信号，并在最佳时机主动促成，话术自然不生硬
- ✓ 合格：能识别一个促成信号，有基本促成尝试
- ✗ 不合格：错过促成时机，或促成方式过于生硬让客户反感
```
