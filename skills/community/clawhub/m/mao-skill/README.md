# 毛泽东著作 Skill

> 毛泽东著作全集合辑——从《中国社会各阶级的分析》到《论十大关系》，涵盖政治、军事、哲学、经济、文化等各个领域的核心著作

## 版本说明

**V1.0.0（初版）**

- 毛泽东主要著作 30 余篇逐篇精要提炼
- 四大理论体系：新民主主义革命理论、人民战争理论、党的建设理论、社会主义建设理论
- 哲学思想：实践论、矛盾论、群众路线、实事求是
- 历史背景与时代脉络：大革命→土地革命→抗日战争→解放战争→社会主义建设
- 重要概念词典：新民主主义、工农武装割据、人民民主专政、群众路线、三大法宝
- 输出模板：著作精读报告、理论体系梳理、时代背景分析、概念辨析、比较研究

## 概述

本 Skill 将毛泽东著作知识体系封装为 AI 可调用的知识技能，支持大模型完成：

- 毛泽东各篇著作的内容提炼、结构分析、核心观点归纳
- 理论体系的跨著作串联（如"农村包围城市"思想的演进）
- 马克思主义中国化的理论脉络梳理
- 历史背景分析与著作时代定位
- 概念辨析与术语解释
- 毛泽东思想与马克思主义经典理论的比较
- 历史文献解读与原文精读
- 毛泽东思想对当代的启示与应用

## 发布指南

### 发布到 OpenAI GPTs

1. 新建 GPT → 在 **Instructions** 中粘贴 `SKILL.md` 全部内容
2. 在 **Knowledge** 中上传 `mao-knowledge-base.md` 和 `output-templates.md`
3. 配置 Conversation starters 为常见毛泽东著作问题

### 发布到 Dify

1. 创建 Agent → 系统提示词中粘贴 `SKILL.md`
2. 创建知识库 → 导入 `mao-knowledge-base.md` 和 `output-templates.md`

### 发布到 Coze（扣子）

1. 创建 Bot → Persona & Prompt 中粘贴 `SKILL.md`
2. 知识库 → 上传 `mao-knowledge-base.md`（分段入库）

### 通用 LLM 调用

```python
system_prompt = open("mao-skill/SKILL.md").read()
response = llm.chat(
    messages=[
        {"role": "system", "content": system_prompt},

    ]
)
```

## 使用指南（面向最终用户）

### 推荐使用方式

#### 方式一：单篇著作精读

> "帮我精读《中国社会各阶级的分析》，包括写作背景、结构大纲、核心观点"

AI 会自动：
1. 介绍写作背景（时间、地点、历史形势）
2. 梳理文章结构
3. 提炼核心观点
4. 分析历史影响

#### 方式二：理论体系追踪

> "毛泽东的'农村包围城市'思想是如何形成的？从哪些著作中可以看到这个思想的演进？"

AI 会：
1. 从早期著作追踪到成熟著作
2. 列出关键文献
3. 分析思想发展脉络
4. 总结理论贡献

#### 方式三：概念辨析

> "什么是'新民主主义'？它与旧民主主义、社会主义的区别是什么？"

AI 会：
1. 定义核心概念
2. 列出经典表述
3. 对比分析
4. 标注出处篇目

#### 方式四：比较研究

> "对比《湖南农民运动考察报告》与《中国社会各阶级的分析》的分析方法"

AI 输出结构化对比分析

### 用户可以提供的信息

| 信息类型 | 说明 |
|---------|------|
| 著作名称 | 可直接使用全称或简称 |
| 核心概念 | 如"群众路线""人民民主专政"等 |
| 历史时期 | 大革命时期、土地革命、抗日战争、解放战争、社会主义建设时期 |
| 理论主题 | 军事理论、哲学思想、党的建设、经济思想、文化理论 |

### 客观评价原则

在涉及历史评价的问题上，保持学术中立，客观呈现：
- 历史成就与理论贡献
- 历史局限性与时代表征
- 不同学术视角（中国国内学界、西方学界、国际共运学界）
- 著作的历史地位与现实意义

### 最低产出物

无论何种查询，AI 必须在答复中提供：
1. **文献出处** — 所有引用必须标注具体篇名、成文时间
2. **原文引用** — 关键论述引用原文，确保忠实于原著
3. **历史背景** — 结合具体历史情境理解著作
