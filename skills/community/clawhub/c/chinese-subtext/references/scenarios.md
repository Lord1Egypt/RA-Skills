# 高频场景潜台词模式库

每类：典型话术 → 真实含义 → 识别信号。命中场景时优先调用，更准更快。

## 通用解码原则（跨场景，最先适用）

1. **主动否定 = 暗示肯定**："不是钱的问题""我不在乎"——主动提的往往正是在意的。
2. **转折词是分界线**："不过 / 只是 / 但是 / 就是"后面才是真意，前面是缓冲垫。
3. **越模糊越是拒绝**：不给时间点、不给具体承诺的"再看看 / 改天 / 研究下"通常 = 软拒绝。
4. **反常即信号**：回复长度、速度、时间点的突变，比字面更诚实。
5. **谦辞要反读**："随便 / 别麻烦 / 小意思"在中式语境常需反向理解。

## 一、职场 · 上下级 / 同事

| 典型话术 | 真实含义 | 识别信号 |
|---|---|---|
| "这个方案不错，再完善一下" | 不满意，要返工 | 没有具体肯定点，只给"完善/打磨/优化"等空泛词 |
| "我再看看 / 我考虑一下" | 多半是婉拒或搁置 | 无下文承诺、不给时间点 |
| "你这个想法很好，不过……" | 转折后才是重点，前半句是缓冲 | "不过/只是/但是"后面才是真意 |
| "辛苦了，先这样吧" | 到此为止，别再推进/别再争 | "先/暂时"= 不想当场否定 |
| "有空我们约一下" | 客套，通常不会真约 | 无具体时间、纯社交话 |
| "这事你看着办" | 充分授权 或 甩锅试探——看语气 | 配"出问题你负责"=甩锅；配"我信你"=授权 |
| 领导深夜发"在吗" | 有事且多半不轻松 | 时间反常、无正文 |

## 二、商务 · 谈判 / 甲乙方

| 典型话术 | 真实含义 | 识别信号 |
|---|---|---|
| "价格不是问题，主要看效果" | 价格恰恰是问题，在压价或找理由 | 主动提"不是问题"= 正是问题 |
| "我们研究研究 / 内部对一下" | 拖延或软拒绝，决策没到位 | 不给推进节点 |
| "你们方案我们很认可，就是预算……" | 认可是铺垫，"就是"后才是拒绝/压价点 | 转折词后接困难 |
| "再给我们一点诚意" | 要降价 / 要额外让利 | 商务语境"诚意"= 钱 |
| "和领导汇报一下再定" | 真要汇报 或 拖延挡箭牌 | 反复使用且无进展 = 拖延 |
| "长期合作"挂嘴边 | 当下想压这一单的价 | 用未来换当下让利 |

## 三、亲密关系 · 恋人 / 暧昧

| 典型话术 | 真实含义 | 识别信号 |
|---|---|---|
| "我没事 / 随便" | 通常有事，在等你主动追问/哄 | 语气冷、回复变短 |
| "你忙你的吧" | 希望你别真去忙，要你陪 | 带情绪、前面刚有冷落 |
| "随便你" | 不满，已经不想商量 | 从积极参与转为撒手 |
| "我们是不是不太合适" | 认真在提分手或重大不满，不是问句 | 反复出现 = 信号升级 |
| "别人家对象都……" | 在表达对你的不满和期待 | 拿对比施压 |
| 秒回变慢、"嗯""哦"变多 | 热度下降或在赌气 | 回复长度/速度断崖 |

## 四、家庭 · 父母 / 亲戚

| 典型话术 | 真实含义 | 识别信号 |
|---|---|---|
| "我们不用你管，自己能行" | 反话，其实希望你多关心/常回家 | 语气委屈、带"你忙吧" |
| "别人家孩子……" | 对你现状不满 + 期待，核心是关心 | 横向对比 |
| "随便做点就行，别麻烦" | 客气，实际希望你重视、用心准备 | 中式谦辞，别当真 |
| "你什么时候带对象回来" | 催婚，且已积累焦虑 | 反复问、亲戚联合问 |
| "钱够不够用" | 既是关心，也可能暗示自己需要被惦记 | 看家庭经济与语气 |

## 五、人情往来 · 朋友 / 社交

| 典型话术 | 真实含义 | 识别信号 |
|---|---|---|
| "改天请你吃饭" | 客套，多数不落地 | 无具体时间 |
| "小事一桩，包在我身上" | 看交情：可能真帮，也可能场面话 | 后续是否主动跟进 |
| "你先垫一下" | 可能要不回来，需警惕 | 含糊还款时间 |
| "我就直说了啊" | 后面是早想好的请求/批评，非临时 | 铺垫式开场 |
| "有困难随时找我" | 客气话居多，别当真兑现承诺 | 泛泛而谈、无具体 |
| "都是朋友，谈钱伤感情" | 往往是想让你让利/帮忙不付钱 | 用关系换利益 |
