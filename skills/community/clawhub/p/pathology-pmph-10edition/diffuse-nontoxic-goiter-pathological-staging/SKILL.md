# 弥漫性非毒性甲状腺肿病理分期

## 触发条件
仅在满足以下全部前提时应用本技能：
- 临床诊断为弥漫性非毒性甲状腺肿（单纯性甲状腺肿）
- 甲状腺功能正常（T3/T4正常，无甲亢症状）
- 无自身免疫性甲状腺抗体（如TRAb、TPOAb阴性）
- 已排除Graves病、甲状腺炎、甲状腺癌及高功能结节

## 执行步骤

1. **收集病理数据**
   - 获取甲状腺重量（克）
   - 记录肉眼观特征（大小、对称性、切面质地、有无出血/囊变/钙化）
   - 在光镜下评估：
     - 滤泡上皮高度（扁平 / 立方 / 柱状 / 乳头样）
     - 胶质量（少 / 多 / 大量贮积）
     - 滤泡大小分布及间质改变

2. **对照三期标准进行分期**

   **(1) 增生期（弥漫性增生性甲状腺肿）**
   - 甲状腺重量 ≤150g，弥漫性对称性中度增大，表面光滑
   - 镜下：滤泡上皮呈立方或低柱状，小滤泡形成，胶质较少，间质充血

   **(2) 胶质贮积期（弥漫性胶性甲状腺肿）**
   - 甲状腺重量 200–300g，显著增大，切面呈淡褐或棕褐色、半透明胶冻状
   - 镜下：滤泡大小不等，上皮复旧变扁平，滤泡腔高度扩大，腔内大量胶质贮积；可伴局灶小滤泡上皮增生或乳头形成

   **(3) 结节期（结节性甲状腺肿）**
   - 甲状腺不对称结节状增大，结节境界不清、无完整包膜
   - 切面常见出血、坏死、囊性变、钙化和瘢痕
   - 镜下：滤泡上皮局灶性增生（柱状/乳头样）、复旧与萎缩并存，胶质分布不均，间质纤维组织增生分割形成结节

3. **输出结果**
   - 明确当前所处分期：增生期 / 胶质贮积期 / 结节期
   - 强调全程无甲状腺功能亢进表现，T3/T4正常，无淋巴细胞浸润及吸收空泡，以区别于毒性甲状腺肿

> ⚠️ 若发现高代谢症状、T3/T4升高、淋巴细胞浸润或吸收空泡，立即终止本流程并考虑毒性甲状腺肿或其他甲状腺疾病。
