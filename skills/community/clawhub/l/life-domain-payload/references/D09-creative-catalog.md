# D09 创意领域任务目录

## 领域定义

创意领域(D09)关注个体如何创造新的表达和作品。核心驱动需求：N4成长 + N5传承。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人创意表达 | 艺术教育培训 |
| 创意技能提升 | 创意产业运营 |
| 创意工具使用 | 创意产品开发 |
| 创意作品创作 | 艺术品交易 |
| 创意分享传播 | 媒体内容生产 |
| 创意社群参与 | 文化活动策划 |
