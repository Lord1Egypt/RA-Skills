## 联系话术模板

### 首次接触（电话/邮件）

```
您好，我是XX投资/XX资本的项目负责人，关注到贵公司所在的XXX行业近期发展动态，想与公司管理层就投资者关系和业务合作方面进行交流，请问可以安排一个电话沟通的时间吗？
```

### 深度接触（试探卖壳意愿）

```
在与公司沟通的过程中，可以适当提及：
"我们关注到贵公司在XXX领域有深厚积累，公司对未来的战略规划是怎样的？是否有考虑通过并购重组等方式实现业务转型或股东价值最大化？"
```

### 注意事项
1. 初次接触以投资者关系名义为宜
2. 避免直接提及"买壳"，先用"战略合作"、"产业整合"等中性词汇
3. 通过证券事务代表建立信任后，再寻求与董事会直接沟通的机会
