# 高中英语题型教研分析参考

## 一、完形填空（Cloze Test）

### 考查核心
- 词汇辨析（近义词、固定搭配、语境用词）
- 逻辑衔接（上下文语义连贯）
- 语篇理解（段落主旨、作者态度）

### 解题要点
1. 通读全文，把握主旨
2. 结合上下文语境推断词义
3. 注意固定搭配与习惯用法
4. 利用首尾句确认文章基调

### 常见易错类型
- 只看空格前后，忽视段落语义
- 近义词辨析不清（如 discover/invent/find）
- 忽略固定搭配（如 regardless of / regardless）

---

## 二、阅读理解（Reading Comprehension）

### 题型分类
| 题型 | 考查重点 |
|------|---------|
| 细节题 | 信息定位、同义替换 |
| 主旨题 | 段落/全文中心思想 |
|推断题 | 隐含信息、作者态度 |
| 词义题 | 语境推断生词含义 |
| 结构题 | 文章逻辑结构（七选五） |

### 七选五专项
- 考查段落间逻辑衔接
- 分析上下文主题词、代词指代、逻辑连接词
- 常用标志词：however, therefore, in addition, for example

---

## 三、语法填空（Grammar Cloze）

### 给词类
| 给词形式 | 常考变形 |
|---------|---------|
| 动词原形 | 时态、语态、非谓语（to do/doing/done） |
| 形容词/副词 | 比较级、最高级 |
| 名词 | 单复数、所有格 |

### 不给词类
| 填写内容 | 判断依据 |
|---------|---------|
| 冠词 | 特指/泛指/零冠词规律 |
| 介词 | 固定搭配、语义逻辑 |
| 连词 | 句子逻辑关系（并列/转折/因果/条件） |
| 代词 | 指代对象、格的变化 |
| 关系词 | 定语从句先行词类型 |

### 时态判断三步法
1. 找时间状语（yesterday, now, since, when 等）
2. 判断句子结构（主动/被动）
3. 确认主从句动作时间关系

## 四、书面表达（Writing）

### 题型分类
- 应用文：书信、通知、邀请函、申请书
- 读后续写：故事续写（逻辑+情感+语言）
- 概要写作：提炼主旨+关键信息

### 评分维度
| 维度 | 权重 | 关注点 |
|------|------|-------|
| 内容 | 40% | 要点完整、逻辑清晰 |
| 语言 | 40% | 词汇丰富、句式多样、语法准确 |
| 格式 | 20% | 文体规范、段落清晰 |

### 高频失分点
- 要点缺失或遗漏
- 句式单一（过多简单句）
- 中式英语思维（直译）
- 书信格式错误（称呼/署名位置）

## 五、语法专项考点速查

### 非谓语动词
| 形式 | 逻辑主语关系 | 典型语境 |
|------|------------|---------|
| to do | 与主语是主动关系，表目的/将来 | 目的状语、be+adj+to do |
| doing | 与主语是主动关系，表同时/正在 | 时间状语、伴随状语 |
| done | 与主语是被动关系 | 过去分词作状语/定语 |
| having done | 动作先于谓语完成 | 完成时非谓语 |

### 定语从句 vs 同位语从句
| 区别点 | 定语从句 | 同位语从句 |
|-------|---------|---------|
| 先行词类型 | 名词（具体事物） | 抽象名词（fact/idea/news 等） |
| that 是否充当成分 | 是（宾语/主语） | 否（纯连接词） |
| 能否用 which 替换 | 能（非限制性） | 不能 |

### 虚拟语气
| 类型 | 从句时态 | 主句结构 |
|------|---------|---------|
| 与现在相反 | did/were | would/could/might + do |
| 与过去相反 | had done | would/could/might + have done |
| 与将来相反 | did/were to do | would + do |
