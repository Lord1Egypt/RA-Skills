# 三段尿试验血尿定位诊断

## 适用条件
- 患者出现肉眼或镜下血尿
- 无严重排尿困难或尿潴留
- 患者能配合完成一次性连续分段排尿
- 已清洁外阴，避免污染

## 禁忌情形
- 无法自主排尿者
- 女性处于月经期（可能污染尿液）

## 执行步骤
1. **准备容器**：取三个清洁、干燥、标记清晰的玻璃杯，分别标注为“第一杯（起始段）”、“第二杯（中段）”、“第三杯（终末段）”。
2. **采集尿液**：指导患者一次性连续排尿，并按以下方式分段收集：
   - 起始段（前1/3尿量）→ 第一杯
   - 中段（中间1/3尿量）→ 第二杯
   - 终末段（后1/3尿量）→ 第三杯
3. **观察颜色**：立即肉眼观察各杯尿液颜色，判断血尿分布模式：
   - **仅第一杯呈红色** → 提示出血部位在**尿道**
   - **仅第三杯呈红色** → 提示出血来自**膀胱颈部、三角区、后尿道、前列腺或精囊腺**
   - **三杯均呈均匀红色（全程血尿）** → 提示出血来源于**肾脏或输尿管**
4. **镜下血尿处理**：若为镜下血尿，需对每段尿液离心后进行显微镜检查，比较红细胞数量分布。
5. **结果解读与后续建议**：
   - 肾脏出血常表现为暗红色、均匀混合的全程血尿
   - 膀胱或前列腺出血多为鲜红色，可伴血凝块
   - 本结果仅为初步定位，应结合影像学检查（如超声、CT）或尿红细胞形态学进一步明确诊断

## 注意事项
- 尿液必须新鲜，采集后立即观察，避免红细胞溶解导致假阴性
- 严格清洁外阴，防止污染干扰判断
- 若患者排尿中断或分段不连续，结果不可靠，应重新采集
