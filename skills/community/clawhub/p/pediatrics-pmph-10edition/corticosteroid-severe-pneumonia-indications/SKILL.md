# 糖皮质激素在儿童重症肺炎中的使用指征

## 何时使用
仅在以下全部条件满足时启用：
- 患儿确诊为以下任一疾病：
  - 重症难治性支原体肺炎
  - 重症腺病毒肺炎
  - A组链球菌肺炎
  - 难治性脓毒症休克
  - 病毒性脑病
  - 急性呼吸窘迫综合征（ARDS）
  - 上述疾病合并哮喘或喘息
- 常规治疗效果不佳
- 存在明确的炎症或免疫介导病理过程

**禁止用于**：普通轻中度肺炎、无上述特定指征的患儿、单纯病毒感染无并发症者。

## 如何执行
1. **确认适用对象**：核对诊断是否属于上述六类重症情况之一，或合并哮喘/喘息。
2. **选择糖皮质激素种类及剂量**（静脉滴注）：
   - 甲泼尼龙：1～2 mg/(kg·d)
   - 琥珀酸氢化可的松：5～10 mg/(kg·d)
   - 地塞米松：0.1～0.3 mg/(kg·d)
3. **确定疗程**：短期使用，通常不超过5–7天，根据临床反应个体化调整。
4. **监测与注意事项**：
   - 使用前评估感染控制状态，避免掩盖感染进展
   - 密切监测血糖、血压、电解质及精神状态
   - 不推荐在普通肺炎中常规使用

> 提示：具体药物选择应结合医院可用制剂、患儿肝肾功能及既往用药史综合判断。
