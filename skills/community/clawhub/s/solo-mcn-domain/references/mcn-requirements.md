# 一人MCN自媒体运营内容要求清单

---

### M1-01 热点追踪
- **必选组件**: 热门话题列表（话题名+热度+来源平台）、追踪时间
- **可选组件**: 行业事件、竞品动态、平台热搜
- **组装顺序**: 平台热搜搜索→行业事件搜集→竞品动态监控→热点汇总→时效标注
- **约束**: 热点数据标注来源和时间；实时数据标注"可能有延迟"
- **格式**: 热点列表（Markdown表格）

### M1-02 选题生成
- **必选组件**: 5-8个候选选题（标题+热度评级+流量类型+推荐平台）
- **可选组件**: 选题来源标注（热点驱动/长尾搜索/用户痛点/竞品空白）
- **组装顺序**: 热点分析→领域匹配→选题发散→热度评估→筛选排序
- **约束**: 每个选题必须有差异化角度；长尾选题≥2个（不全追热点）
- **格式**: 选题卡片列表（Markdown）

### M1-03 差异化角度
- **必选组件**: 每个选题2-3个角度（角度名+切入点+目标读者+吸引力评级）
- **可选组件**: 竞品已有角度分析、差异化空间说明
- **组装顺序**: 选题分析→角度发散→竞品比对→差异化确认→吸引力评估
- **约束**: 角度之间必须互斥（不重叠）；每个角度必须有具体切入点
- **格式**: 角度分析表（Markdown表格）

### M1-04 竞品分析
- **必选组件**: 已有同类内容列表（标题+作者+数据+发布时间）、差异化空间
- **可选组件**: 竞品内容风格分析、竞品不足之处
- **组装顺序**: 关键词搜索→同类内容筛选→数据分析→差异化空间识别
- **约束**: 数据来源标注；竞品数量≥3篇才有分析意义
- **格式**: 竞品分析表（Markdown表格）

### M1-05 选题评估
- **必选组件**: 评估维度（热度/竞争度/可执行性/变现潜力）、评分、首选推荐
- **可选组件**: 系列组合建议、时效提醒
- **组装顺序**: 各维度评分→加权计算→排序→首选推荐→理由说明
- **约束**: 评估必须基于数据而非直觉；时效性选题标注截止时间
- **格式**: 评估矩阵（Markdown表格）

### M2-01 标题优化
- **必选组件**: 10种公式标题（数字/悬念/痛点/对比/身份/热点/反常识/情绪/工具/合集）、评分表
- **可选组件**: AB测试建议、平台适配调整
- **组装顺序**: 核心卖点提取→10种公式逐一生成→评分→首选推荐→AB测试建议
- **约束**: 标题字数按平台约束（小红书≤18字/抖音≤15字/公众号≤22字）
- **格式**: 标题列表+评分表（Markdown）

### M2-02 正文写作
- **必选组件**: 完整稿件（按平台风格）、标签/话题、互动引导
- **可选组件**: 配图建议、发布时间、SEO关键词
- **组装顺序**: 大纲生成→正文撰写→风格适配→标签添加→互动引导→字数校验
- **约束**: 字数按平台约束（小红书300-800字/公众号≤2000字/知乎1000-2000字）
- **格式**: 完整稿件（Markdown）

### M2-03 SEO优化
- **必选组件**: 核心关键词、长尾关键词、标题改写建议、标签建议
- **可选组件**: 首段优化、布局检查、语义词
- **组装顺序**: 内容分析→关键词提取→搜索量评估→标题改写→标签建议→布局检查
- **约束**: 关键词密度≤5次（避免堆砌）；小红书SEO靠标题+标签
- **格式**: SEO优化报告（Markdown）

### M2-04 脚本撰写
- **必选组件**: 完整口播脚本（含时间标注）、分镜表（镜号/时间/景别/画面/文案/运镜）
- **可选组件**: 设备建议、剪辑要点、封面AI prompt
- **组装顺序**: 结构设计→Hook设计→核心内容→CTA→分镜表→设备建议
- **约束**: 前3秒必须有强hook；每5秒切一次画面；字幕必须有
- **格式**: 脚本+分镜表（Markdown）

### M2-05 互动引导
- **必选组件**: 评论区引导话术、CTA设计（关注/点赞/收藏/评论）
- **可选组件**: 互动话题设计、抽奖活动、UGC征集
- **组装顺序**: 内容主题分析→互动话题设计→CTA选择→话术撰写
- **约束**: CTA必须自然融入内容；不要"求赞"式引导
- **格式**: 互动方案（Markdown）

### M3-01 封面设计
- **必选组件**: 3套方案（风格描述+配色HEX+排版布局+字体建议+AI生图prompt）
- **可选组件**: 尺寸规范、实现工具推荐
- **组装顺序**: 平台确认→尺寸确认→风格设计→配色→排版→AI prompt
- **约束**: 文字3-7个字最佳；高饱和度颜色点击率高30%+；不要放二维码
- **格式**: 设计方案卡片（Markdown）

### M4-01 平台适配
- **必选组件**: 目标平台规则（标题字数/正文长度/标签数量/封面尺寸）、适配后内容
- **可选组件**: 发布时间建议、引流话术
- **组装顺序**: 平台规则确认→内容适配→标题改写→标签调整→封面尺寸→发布检查
- **约束**: 每条平台规则必带"可能已更新"提醒；极限词必须检查
- **格式**: 适配后内容（Markdown）

### M4-02 社群运营
- **必选组件**: 30天运营日历（每天时间+内容+话术）、话术模板（欢迎/日常/冷场/转化）
- **可选组件**: 内容比例（干货40%+互动30%+娱乐15%+品牌10%+通知5%）
- **组装顺序**: 社群定位→成员画像→栏目设计→日历编排→话术撰写
- **约束**: 红包每周≤2次；广告≤10%；群规正面表述
- **格式**: 30天日历+话术模板（Markdown）

### M5-01 数据采集
- **必选组件**: 最近10-20篇内容数据（阅读量/点赞/评论/收藏/发布时间/内容类型）
- **可选组件**: 行业基准数据、历史数据对比
- **组装顺序**: 数据请求→数据整理→基准对比→数据质量检查
- **约束**: <10篇数据不够分析；数据标注来源和时间
- **格式**: 数据表格（Markdown）

### M5-02 爆款分析
- **必选组件**: Top3爆款（标题+数据+成功原因+可复制经验）
- **可选组件**: 爆款共性总结、可复用模式
- **组装顺序**: 数据排序→Top3筛选→原因分析→经验提炼→可复用模式总结
- **约束**: 原因分析必须基于数据对比而非主观判断
- **格式**: 爆款分析报告（Markdown）

### M5-03 低效诊断
- **必选组件**: Bottom3低效内容（标题+数据+问题诊断+改进方向）
- **可选组件**: 共性问题总结、避免策略
- **组装顺序**: 数据排序→Bottom3筛选→问题诊断→改进方向→避免策略
- **约束**: 诊断必须区分标题/内容/发布时间/封面等因素
- **格式**: 诊断报告（Markdown）

### M6-01 报价计算
- **必选组件**: 各形式报价区间（植入/专场/种草/直播/长期）、计算公式
- **可选组件**: 沟通话术、砍价应对
- **组装顺序**: 基础数据确认→公式计算→领域系数调整→互动加成→报价区间→话术
- **约束**: 先收款再发布；<1000元单子建议不做；独家>30天慎重
- **格式**: 报价方案（Markdown）

### M6-04 排期管理
- **必选组件**: 一周排期表（日期/平台/时间/内容/平台适配）、发布前检查清单
- **可选组件**: 跨平台引流策略、最佳发布时间
- **组装顺序**: 内容清单确认→平台确认→时间安排→适配标注→检查清单
- **约束**: 同一内容跨平台≥2小时间隔；周一早/周日晚避开
- **格式**: 排期表（Markdown表格）
