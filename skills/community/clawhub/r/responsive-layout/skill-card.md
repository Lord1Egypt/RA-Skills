## Use Case: <br>
Developers and frontend engineers use this skill to design or troubleshoot responsive layouts, choose breakpoints and flex/grid strategies, and adapt CSS units, safe areas, and touch targets across devices. <br>

### Deployment Geography for Use: <br>

## Known Risks and Mitigations: <br>
Risk: Broad triggers such as layout, flex, or grid may activate the skill for general frontend conversations. <br>
Mitigation: Confirm the user wants responsive layout guidance before applying detailed CSS recommendations. <br>
Risk: Generated responsive CSS or layout advice may introduce visual regressions when applied directly to production interfaces. <br>
Mitigation: Review proposed CSS against the target design system and test it across relevant viewport sizes before release. <br>
