# ClawHub API 枚举值说明

## Family (包类型)

- `skill`: 技能
- `code-plugin`: 代码插件
- `bundle-plugin`: Bundle插件

## Channel (渠道)

- `official`: 官方
- `community`: 社区
- `private`: 私有

## VerificationTier (验证层级)

- `structural`: 结构化
- `source-linked`: 源码链接
- `provenance-verified`: 来源验证
- `rebuild-verified`: 重建验证

## ScanStatus (扫描状态)

- `clean`: 干净
- `suspicious`: 可疑
- `malicious`: 恶意
- `pending`: 待处理
- `not-run`: 未运行
- `error`: 错误

## ModerationState (审核状态)

- `approved`: 已批准
- `quarantined`: 已隔离
- `revoked`: 已撤销

## ArtifactKind (工件类型)

- `legacy-zip`: 传统ZIP
- `npm-pack`: NPM包
