## Description: <br>
Hizal Start initializes or resumes a Hizal context memory session when an agent begins work, picks up a task, or enters a coding session. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[parkerscobey](https://clawhub.ai/user/parkerscobey) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Agent operators and developers use this skill to start or resume Hizal context memory before normal work, then register task focus, search prior context, and record decisions during a task. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill auto-starts or resumes persistent Hizal memory sessions for ordinary work, which can store task details and decisions beyond the immediate chat. <br>
Mitigation: Install only where persistent memory is intended; review Hizal retention and access controls, and have the agent ask before storing sensitive task details. <br>
