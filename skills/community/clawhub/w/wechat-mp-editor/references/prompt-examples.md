## 提示词通用原则（持续沉淀）

_这些原则从实战中来，每条都是有教训的经验。_

- **即梦对中文理解很好**，不需要写蹩脚英文 prompt
- **具体场景 > 抽象概念**（"办公桌上堆满周报" > "工作效率"）
- **指定画风关键词**：电影感、3D渲染、水彩、水墨、CG模型、摄影等
- **光影词汇很有用**：柔光、逆光、暖色调、冷色调、强烈光影对比
- **"近镜头""特写"** 能有效控制人脸崩坏
- **人物场景加情绪词**：专注、微笑、疲惫、悠闲，比光说"一个人"丰富得多
- **模型版本建议**：图片优先用 5.0，画质和一致性最好
- **比例经验**：横版 16:9 适合场景图，1:1 适合头像/图标，竖版 9:16 适合手机壁纸/封面

## 即梦 5.0 已知能力边界

_这些边界来自于多次尝试，帮助判断什么时候该用 AI 图、什么时候该用 SVG 示意图。_

| 能力 | 表现 | 适用场景 |
|------|------|---------|
| 人物肖像 | 🟢 非常好 | 单人头像、半身照、表情 |
| 人物半身/全身 | 🟢 好 | 职场场景、生活场景 |
| 多人互动 (2人) | 🟡 中等 | 要有明确主次关系 |
| 多人互动 (3人+) | 🔴 不稳定 | 建议用 SVG 示意图替代 |
| 场景/风景 | 🟢 非常好 | 城市、自然、室内场景 |
| 光影/氛围 | 🟢 非常好 | 电影感、情绪场景 |
| 文字/标签 | 🔴 不可靠 | 绝对不要指望 AI 生成清晰文字 |
| 抽象概念 | 🔴 不行 | 用生活类比替代 |
| 动态模糊 | 🟡 中等 | 简单运动可行，复杂运动不行 |
| 文字生图 | 🟢 非常好 | 核心能力，5.0 最稳定 |
| 图生图（参考图） | 🟢 好 | 风格迁移、换背景、局部修改 |
| 图片超清（2K→4K） | 🟢 好 | VIP 功能，未测试 |

## ⚠️ 使用规范

1. **首次写推文** → 先读这个文件，参考成功案例的风格
2. **生成提示词后** → 告知用户配图数量和位置，用户用即梦官网生成
3. **用户反馈效果后** → 更新此文件：
   - 效果好 → 追加到成功案例，写上当时文章主题和模型参数
   - 效果差 → **移除**这条，并更新能力边界
   - 同主题有更佳方案 → 替换旧的，不保留多个版本
4. **定期整理**（每 5 篇配图后） → 检查有没有矛盾或过时的经验，清理掉

### 案例 4：AI工具箱概念图
- **文章**：你的AI正在变成一个工具箱
- **配图用途**：文章开头，视觉化"工具箱"概念
- **模型 / 比例**：5.0 / 16:9
- **分辨率**：2K
- **提示词**：
  ```
  俯拍视角，一张浅木色桌面上散落着各种工具——扳手、螺丝刀、画笔、剪刀，但所有工具都被柔和的光线笼罩，呈现出科技感，画面中央有一团蓝色的数字光晕在流动，暗示这些是 AI 工具，暖色调，电影感，细节丰富，横版 16:9
  ```

- **效果**：⭐⭐⭐⭐ — 构图丰富，科技感氛围好
- **经验**：实物+科技的混搭描述很有效，"数字光晕"这类词能引导AI做出预期的科幻效果
