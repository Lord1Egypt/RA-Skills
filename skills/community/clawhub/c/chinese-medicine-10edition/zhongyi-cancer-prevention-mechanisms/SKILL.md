# Zhongyi Cancer Prevention Mechanisms

## When to Use
当满足以下任一情形时启动本技能：
- 患者被诊断为慢性萎缩性胃炎伴肠化生或中重度不典型增生
- 存在食管上皮重度增生
- 肝硬化患者（尤其脾虚湿盛、嗜酒者）
- 结直肠息肉患者
- 肿瘤术后需预防复发转移

且已排除浸润性癌，并完成基础病因治疗（如抗HP治疗）。

## Execution Steps

1. **辨识癌前病变类型**：明确患者所属的适用对象类别。

2. **选择对应干预策略**：
   - **慢性萎缩性胃炎伴肠化生/不典型增生**：采用健脾益气、酸甘生津、疏肝理气、清热解毒、活血化瘀等复合治法，代表药物包括党参、麦冬、柴胡、陈皮、黄连、半枝莲等。
   - **肝硬化高危人群**：以健脾、理气、化湿为主，推荐香砂六君子汤以防止肝癌变。
   - **食管上皮重度增生**：使用六味地黄丸降低食管癌发生风险。
   - **肿瘤术后防复发转移**：核心治法为活血化瘀联合扶正培本，靶向抑制新生血管、促进凋亡、调节MMPs活性、增强免疫。

3. **确认长期用药计划**：强调需规律、持续服药，不可中断。

4. **监测与调整**：定期评估病变逆转情况及耐药风险，必要时调整方药组合。

## Precautions
- 已确诊浸润性癌者不属于本技能适用范围。
- 中药干预应建立在规范西医诊断和基础治疗完成的前提下。
- 多药耐药（MDR）逆转可通过下调耐药转运蛋白表达、调控代谢酶系统实现，可作为辅助策略纳入整体方案。
