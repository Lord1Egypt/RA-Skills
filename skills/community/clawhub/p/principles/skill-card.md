## Description: <br>
Principles is a personal knowledge system that captures thoughts, tracks source credibility, detects conflicts with existing beliefs, and graduates tested wisdom into principles over time. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[andyhcwang](https://clawhub.ai/user/andyhcwang) <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and developers use this skill to maintain a workspace-local personal knowledge system for capturing raw thoughts, organizing sourced wisdom, resolving conflicts, and reviewing personal principles. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill creates and updates a personal/ folder in the workspace, which may contain sensitive personal notes if the user stores them there. <br>
Mitigation: Install only if comfortable with workspace-local note files, avoid storing secrets, and review generated notes before relying on them. <br>
Risk: Processing the inbox can clean up inbox.md after routing entries, which may remove raw notes the user wanted to preserve. <br>
Mitigation: Review or back up inbox.md before running inbox processing when raw notes should be preserved. <br>
