# 公文文种详解

按《党政机关公文处理工作条例》(2012) 第八条规定，党政机关法定公文共 **15 种**。本文档分为两部分：

- **第一部分**：15 种法定公文（党政机关用）
- **第二部分**：非法定但常用公文（企业 / 事业 / 个人）

---

## 第一部分：15 种法定公文

### 1. 决议 (Resolution)

**性质**：会议讨论通过的重大决策事项。
**适用场景**：党的代表大会、人大、政协全会等通过的决议。
**行文方向**：下行 / 公开。
**关键句式**："会议决定 / 一致通过 / 全体表决通过……"
**注意**：
- 必须经会议讨论表决，单个领导签发不能称"决议"
- 区别于"决定"：决议是会议通过的，决定可由领导机关单独作出

### 2. 决定 (Decision)

**性质**：对重要事项作出决策和部署、奖惩有关单位和人员、变更或撤销下级机关不适当的决定事项。
**行文方向**：下行。
**典型用途**：奖励 / 处分 / 机构变更 / 重大事项部署
**示例标题**："国务院关于设立中国（海南）自由贸易试验区的决定"

### 3. 命令（令）(Order)

**性质**：公布行政法规和规章；宣布施行重大强制性措施；批准授予和晋升衔级；嘉奖有关单位和人员。
**行文方向**：下行 / 公开。
**使用机关**：国务院、国家主席、国务院总理、国务院各部部长等有签发权的领导。
**特点**：法律效力最强的文种之一，普通机关基本不用。

### 4. 公报 (Communique)

**性质**：公布重要决定或重大事项。
**典型用途**：中央委员会全会公报、外交联合公报、统计公报。
**行文方向**：公开。
**特点**：仅由特定高层机关使用，普通单位不写公报。

### 5. 公告 (Public Announcement)

**性质**：向国内外宣布重要事项或法定事项。
**典型用途**：人大常委会公告、国务院公告、中央银行公告。
**行文方向**：公开。
**与"通告"的区别**：公告面向国内外，事项重大；通告面向社会一定范围。

### 6. 通告 (Notice / Circular)

**性质**：在一定范围内公布应当遵守或周知的事项。
**典型用途**：交通管制通告、防汛通告、限号通告。
**发文主体**：各级政府、行政部门均可发。
**行文方向**：面向社会公众。

### 7. 意见 (Opinion / Guidance)

**性质**：对重要问题提出见解和处理办法。
**行文方向**：**可上可下可平**（按内容定）。
**典型用途**：
- 上行：向上级提出建议性意见（"关于…的意见"，请上级研究）
- 下行：指导下级工作（"关于深化…改革的指导意见"）
- 平行：协调与不相隶属单位的工作

**示例**：国务院《关于深化国有企业改革的指导意见》（下行指导）

### 8. 通知 (Notification / Notice) ⭐

**最常用文种之一。** 占党政机关日常公文的 30%+。

**性质**：
- 发布、传达要求下级机关执行和有关单位周知或者执行的事项
- 批转、转发公文
- 任免人员

**行文方向**：下行 或 平行（**不相隶属机关用函，不用通知**）。

**子类：**
- **指示性通知**：要求下级照办（"现就…通知如下，请认真贯彻执行"）
- **知照性通知**：仅周知 / 不要求行动（"现将…告知如下"）
- **批转 / 转发性通知**：转发上级或同级文件
- **任免性通知**：人事任免

**标题套路**：`[发文机关]关于[事由]的通知`

### 9. 通报 (Briefing / Circular Report)

**性质**：表彰先进、批评错误、传达重要精神和告知重要情况。
**行文方向**：下行。
**典型用途**：表彰通报、批评通报、情况通报、事故通报。
**与"通知"的区别**：通报重在"告知情况以引起注意"，通知重在"要求行动"。

### 10. 报告 (Report) ⭐

**最常用文种之一。** 仅次于通知。

**性质**：向上级机关汇报工作、反映情况、回复上级机关的询问。
**行文方向**：**上行**。

**子类：**
- **工作报告**：阶段性 / 年度工作汇报
- **情况报告**：反映重要情况或问题（不要求批复）
- **答复报告**：回复上级询问

**铁律：报告中绝对不得夹带请示事项。** 如果你写到"请上级批准 / 请审示"——立即停笔，拆成"请示"另发。

**结尾套语**：
- "专此报告" / "特此报告"
- 不用"请批复" / "妥否请批示"——那是请示的语言

### 11. 请示 (Request for Instructions) ⭐

**性质**：向上级机关请求指示、批准。
**行文方向**：**上行**。

**三铁律：**
1. **一文一事**——一份请示只请求一件事，多事拆成多份
2. **不得越级**——逐级上行；必须越级时抄送被越过机关并说明
3. **不夹带其他事项**——不能把请示和报告混写

**结构：**
- 请示缘由（为什么需要请示）
- 请示事项（具体请求什么）
- 结尾套语：**"妥否，请批示"** / "以上请示当否，请批复"

### 12. 批复 (Reply)

**性质**：答复下级机关的请示事项。
**行文方向**：下行（针对下级请示）。
**特点**：被动文种 — 没有下级请示，就没有批复。

**结构：**
- 引述：明确指明所批复的请示文件标题和发文字号
- 批复意见：同意 / 不同意 / 同意原则但提出修改意见
- 结尾："此复"

**示例开头**：`你（厅 / 局 / 委）《关于……的请示》（X 文 [2026] X 号）收悉。经研究，现批复如下：……`

### 13. 函 (Official Letter)

**性质**：不相隶属机关之间商洽工作、询问和答复问题、请求批准和答复审批事项。
**行文方向**：**平行**（不相隶属）。

**子类：**
- **去函（商洽函 / 询问函 / 请求批准函）**
- **复函**（答复对方来函）

**关键判断**：双方有无隶属关系？
- 有隶属（上下级）：用请示 / 批复 / 通知
- 无隶属（兄弟单位、跨系统）：**用函**

**示例**：教育部致财政部"关于…经费保障的函"（两个部委之间互不隶属）。

**结尾套语**："此致 / 特此函达 / 盼复"。

### 14. 议案 (Motion / Proposal)

**性质**：各级人民政府按照法律程序，向同级人民代表大会或者人民代表大会常务委员会提请审议事项。
**使用主体**：人民政府提请人大。
**普通机关基本不用此文种。**

### 15. 纪要 (Minutes / Memorandum) ⭐

**性质**：记载会议主要情况和议定事项。
**行文方向**：会议参与方共同遵守（横向 + 下行兼有）。
**特点**：纪要是**会议产物**，不是个人观点。

**结构：**
- 会议概况（时间、地点、主持人、参会人员）
- 会议议程
- 议定事项（一、二、三……分条列出）
- 责任分工与时限（如有）

**示例标题**：`[会议名称]纪要` 或 `关于[议题]的会议纪要`

## 第二部分：非法定但常用公文（企业 / 事业 / 个人）

### 述职报告

**用途**：领导干部 / 中层管理者 / 党员 民主生活会上汇报履职情况。
**频次**：年度 / 任期满 / 调动前。
**关键结构：**
1. **履职情况**（做了什么）— 占 50%
2. **主要成绩**（取得什么）— 占 20%
3. **存在问题**（不足是什么）— 占 15%（必写，不能省）
4. **下一步打算**（怎么改 / 怎么干）— 占 15%

**注意**：
- 必须谈不足，且具体（不能写"理论学习不够"这种空话）
- 第一人称用"本人"或"我"均可
- 篇幅一般 2000-5000 字

### 工作总结

**用途**：年度 / 季度 / 项目结束后回顾。
**与述职报告区别**：述职以"人"为主线（我做了什么），总结以"事"为主线（这件事做得如何）。

**结构：**
1. 基本情况 / 总体回顾
2. 主要工作及成效
3. 经验做法
4. 问题与不足
5. 下一步计划

### 调研报告 / 立项报告 / 可行性研究报告

属于专业报告，结构因领域而异，但都遵循：
- 背景 / 目的
- 现状分析
- 问题诊断
- 对策建议
- 实施路径

### 工作汇报材料 / PPT 讲稿

非正式公文，但企业 / 国企日常高频。语言介于公文与口语之间，建议：
- 保留"层级清晰、要点突出"的公文骨架
- 适度使用更生动的语言（避免完全口语化）
- 必须有数据支撑

## 选用文种的快速判断表

```
我要做什么？
├── 让下级 / 同级周知或执行某事        → 通知（隶属）/ 函（不相隶属）
├── 向上级请求批准或答复               → 请示
├── 向上级汇报工作或情况               → 报告（不夹带请求）
├── 答复下级的请示                     → 批复
├── 协调不相隶属单位                   → 函
├── 对重要问题提建议 / 给指导          → 意见
├── 记录会议议定事项                   → 纪要
├── 表彰先进 / 批评错误 / 通报情况     → 通报
├── 向社会公布事项                     → 通告（区域）/ 公告（重大）
├── 党组织作重大决策                   → 决议 / 决定
├── 公布法规、宣布措施                 → 命令（令）
├── 政府向人大提请审议                 → 议案
├── 个人履职汇报                       → 述职报告（非法定）
└── 项目 / 阶段性回顾                  → 工作总结（非法定）
```
