# Early Menstruation Syndrome Differentiation

## When to Use
患者月经周期连续2次以上提前7天以上，且具备完整的经量、经色、经质及全身证候信息，排除生理性波动。

## Execution Steps

### 第一步：观察月经特征
- **经量多、色淡、质稀** → 属气虚
  - 兼神疲肢倦、气短懒言 → **脾气虚证**
  - 兼腰膝酸软、头晕耳鸣 → **肾气虚证**
- **经量多或少、色红、质稠** → 属血热
  - 兼面红口干、尿黄便结 → **阳盛血热证**
  - 兼两颧潮红、手足心热 → **阴虚血热证**
  - 兼烦躁易怒、口苦咽干 → **肝郁血热证**

### 第二步：对应治法与方药
- **脾气虚**：健脾益气，摄血调经 → 补中益气汤加减
- **肾气虚**：补益肾气，固冲调经 → 固阴煎加减
- **阴虚血热**：养阴清热调经 → 两地汤加减
- **阳盛血热**：清热凉血调经 → 清经散加减
- **肝郁血热**：疏肝清热，凉血调经 → 丹栀逍遥散加减

### 第三步：治疗原则与风险监测
- 核心原则：“益气固冲，清热调经”。
- 监测是否合并月经过多或崩漏风险，必要时升级干预。

## Precautions
- 偶发性周期提前不适用本技能。
- 青春期初潮1年内或围绝经期生理性波动不纳入辨证范围。
