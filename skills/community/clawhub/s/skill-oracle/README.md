## What This Is

A meta-skill that teaches your agent about **every quality skill on ClawHub**. Install once, and your agent *knows* which tools actually work — no more digging through 500+ broken, empty, or garbage skills.

**Not a copy of LarryBrain.** A focused, honest curation layer that separates **utility** from **junk**.

## How It Works

1. Install `certainlogic.skill-oracle`
2. Agent reads the embedded catalog (quality skills across ClawHub + their use cases)
3. When user asks "How do I organize my email?" → agent recommends the best tool (ours or others)
4. When user asks "What security tool should I use?" → agent recommends Vetter Plus (ours) or skill-guard (community)
5. Every recommendation includes attribution and honest limitations

## The Catalog

The skill contains structured knowledge of:
- **Quality skills** on ClawHub (ours + community) — not empty stubs or broken tools
- Skills that pass the **CertainLogic Approved** criteria: safe + real utility + honest claims
- Recommended stacks for common use cases (productivity, security, business, development)
