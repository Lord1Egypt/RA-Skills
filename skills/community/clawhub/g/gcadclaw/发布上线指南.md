## 四、发布到 ClawHub（Web UI 方式）

### 步骤 1：注册 ClawHub 账号

1. 访问 https://clawhub.ai
2. 点击右上角 **「Sign in with GitHub」** 用 GitHub 账号授权登录
3. 无需额外注册，直接绑定 GitHub 账户即可
4. 登录后用户名为你的 GitHub 用户名
5. 如需使用组织身份，可在「Publishers」板块管理

### 步骤 2：创建技能列表

1. 登录后进入「我的技能」页面
2. 点击「发布新技能」
3. 填写以下信息：

| 字段 | 填写内容 |
|------|---------|
| 技能名称 | `gcadclaw` |
| 分类 | `开发工具` / `工程工具` |
| 描述 | 通过自然语言生成和修改 GstarCAD 二维工程图纸 |
| 标签 | `cad`, `gstarcad`, `dwg`, `mechanical`, `engineering`, `automation` |

### 步骤 3：上传技能包

1. 选择「上传文件」
2. 选择：`c:\Users\55341\WorkBuddy\20260526143702\gcadclaw-v1.0.0.zip`
3. 系统将自动验证文件结构

### 步骤 4：提交审核

1. 预览信息确认无误
2. 点击「提交审核」
3. 等待 **1-3 个工作日**审核
4. 审核状态追踪：待审核 → 审核中 → 已通过（上架）
5. 如被拒绝，查看原因修改后重新提交
