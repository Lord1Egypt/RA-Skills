# 脊髓损伤类型、临床表现及病理演变规律

## 触发条件
当患者有脊柱外伤史，并出现以下任一表现时，应用本技能：
- 损伤平面以下感觉障碍
- 运动功能减退或丧失
- 反射异常（如腱反射消失或亢进）

## 执行流程

### 1. 判断损伤类型
- **脊髓震荡**：无结构损伤，功能暂时抑制；数小时至数天内完全恢复。
- **不完全性脊髓损伤**：保留部分感觉或运动功能；识别四种综合征。
- **完全性脊髓损伤**：损伤平面以下所有功能丧失；经历脊髓休克期后转为痉挛性瘫痪。
- **特殊部位损伤**：包括脊髓圆锥和马尾神经损伤。

### 2. 识别不完全性损伤综合征
- **脊髓前综合征**：四肢瘫（下肢重于上肢），保留深感觉；预后最差。
- **脊髓后综合征**：深感觉丧失，运动与痛温觉保留。
- **中央管综合征**：四肢瘫（上肢重于下肢），无感觉分离；常见于颈椎过伸伤。
- **Brown-Séquard综合征**：同侧运动与深感觉丧失，对侧痛温觉丧失。

### 3. 评估完全性损伤分期与表现
- **脊髓休克期（2–4周）**：弛缓性瘫痪，骶段功能（肛周感觉、括约肌收缩）完全丧失。
- **休克期后**：转为痉挛性瘫痪，肌张力增高、腱反射亢进、病理征阳性。
- **按节段判断瘫痪形式**：
  - 胸段 → 截瘫
  - 颈段 → 四肢瘫
    - C1–C4：四肢均为痉挛性瘫
    - C5–C7：上肢弛缓性瘫，下肢痉挛性瘫

### 4. 处理特殊情形
- **SCIWORA（无放射学异常的脊髓损伤）**：X线/CT正常但MRI显示脊髓异常，多见于儿童或颈椎外伤。
- **电生理辅助**：SEP与MEP均不能引出提示完全性损伤。

### 5. 输出结果
输出包含以下要素：
- 脊髓损伤类型（震荡/不完全/完全/特殊部位）
- 具体综合征名称（如适用）
- 瘫痪性质（弛缓性或痉挛性）
- 预后判断依据（基于病理演变与临床特征）

## 参考文档

### 脊髓损伤病理演变时间线

**不完全性损伤**
- 3小时内：灰质少量出血
- 6–10小时：出血扩大，神经水肿
- 24–48小时后：逐渐消退

**完全性损伤**
- 3小时：灰质多灶出血
- 6小时：灰质出血增多，白质水肿
- 12小时：白质出血，轴索退变开始
- 24小时：灰质中心坏死
- 48小时：灰质软化，白质广泛退变
- 晚期：胶质瘢痕替代

### 各综合征详细临床特征

**脊髓前综合征**
- 病因：颈脊髓前方受压或前中央动脉闭塞
- 运动：四肢瘫，下肢重于上肢
- 感觉：位置觉、震动觉保留；痛温觉、轻触觉部分保留
- 预后：最差，恢复有限

**脊髓后综合征（罕见）**
- 常由脊髓后动脉供血障碍或压迫引起
- 运动：基本保留
- 感觉：深感觉（位置觉、震动觉）丧失；痛温觉、触觉存在

**中央管综合征**
- 机制：颈椎过伸致中央管周围传导束受损
- 运动：上肢瘫痪重于下肢（因手部纤维位于中央）
- 感觉：无典型感觉分离
- 常见人群：老年颈椎退变患者

**Brown-Séquard综合征**
- 同侧：运动瘫痪 + 深感觉丧失
- 对侧：痛温觉丧失（通常在1–2节段以下）
- 常见原因：穿透伤、椎管内肿瘤
