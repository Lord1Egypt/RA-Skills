# 大女主小说创作主编 · 苏幕遮

> 全流程自动化大女主小说创作工具——从大纲碰撞到排版交付，一步到位。

苏幕遮统筹四大名家（琼瑶、亦舒、流潋紫、天下归元）碰撞大纲，桐华执笔正文，专属检查官把关质量，最终交付 `.txt` 成品。

## 使用方式

只需告诉主编你的想法，剩下的全自动：

```
# 无脑启动
帮我写一本大女主小说

# 指定方向
写一个后宫为背景的权谋故事

# 续写已有作品
我之前写了个开头，你接着往下写
```

**主编全程不问任何问题**，自己决定创作方向、拍板大纲、调度写手，最终交付成品。

## 创作流程

```
用户发起 → 断点检测 → 大纲产出 → 正文撰写 → 名家审读 → 检查修改 → 排版交付
```

| 阶段 | 步骤 | 产出 |
|------|------|------|
| 断点检测 | 扫描已有文件，确定续写起点 | 直接跳过已完成的卷 |
| 第一阶段 | 四大名家 3 轮碰撞 → 主编整合大纲 → 主编自审 | `05_大纲.md` |
| 第二阶段 | 写手逐卷正文 → 名家审读 → 检查修改循环 | 逐卷正文 + 修改版 |
| 第三阶段 | 统一排版 → 生成目录 → 输出 txt | 最终 `.txt` 成品 |

## 核心机制

- **去AI化写作** — 禁用 AI 句式、强化角色对话辨识度、五感平衡、句子长短落差、写三分留七分
- **断点续写** — 自动检测已有大纲/卷章，无缝衔接已有内容
- **名家审读** — 每卷随机选 2 位名家审读，共识问题优先处理
- **修改循环** — 检查 → 修改 → 复查，至少 2 轮通过才放行
- **全程自主** — 不向用户提问，主编自行完成所有创作决策

## 四大核心标准

1. **女主不花瓶** — 美不能是她唯一的武器
2. **女人不为难女人** — 复杂真实的女性关系
3. **爱情不是全部** — 事业、复仇、自我实现同等重要
4. **不写圣母** — 女主可以善良，但不能愚蠢
