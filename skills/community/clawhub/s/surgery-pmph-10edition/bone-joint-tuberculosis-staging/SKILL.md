# 骨与关节结核病理分期评估

## 适用对象
- 骨组织
- 关节滑膜
- 关节软骨

## 前提条件
- 患者存在原发肺结核或胃肠道结核病灶
- 结核分枝杆菌经血行播散至骨或关节

## 禁忌场景
- 急性化脓性骨关节感染
- 非结核性肉芽肿疾病

## 执行流程
1. **确认初始病灶位置**：检查是否位于长骨干骺端。
2. **评估当前病理阶段**：
   - 若病灶局限于骨或滑膜，且关节软骨面完好 → 判定为**单纯性骨结核**或**单纯性滑膜结核**。
   - 若病灶已侵入关节腔并破坏关节软骨 → 判定为**全关节结核**。
3. **判断是否出现晚期并发症**：
   - 如有关节结构毁损、窦道或瘘管形成，提示进入终末阶段。
4. **制定干预策略**：
   - 单纯性阶段：推荐规范抗结核药物治疗，可完全保留关节功能。
   - 全关节结核：需考虑手术干预（如病灶清除、关节融合等）。
   - 窦道形成伴混合感染：需联合抗感染及外科处理，但必然遗留功能障碍。

## 关键判断点
- “全关节结核”是不可逆关节损伤的转折点，必须在单纯性阶段积极干预。
- 关节软骨状态（完好 vs 破坏）是分期的核心依据。

## 输出结果
根据病理阶段明确：
- 预后（是否可完全恢复）
- 是否需要手术干预
- 是否已进入不可逆损伤阶段
