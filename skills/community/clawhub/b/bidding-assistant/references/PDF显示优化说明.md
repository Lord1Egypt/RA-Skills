# PDF显示优化说明

## 优化内容

### 问题
用户反馈收到的PDF月报中：
- ❌ 项目标题显示不全（被截断）
- ❌ 链接显示不清（被截断）
- ❌ 单行显示不够，信息丢失

### 解决方案
将PDF表格从"单行显示"改为"两行显示"模式：

**优化前**：
```
| 网站 | 日期 | 标题（截断50字符） | 链接（截断30字符） |
```

**优化后**：
```
| 网站 | 日期 | 项目名称（完整显示） |
|      |      | 项目链接（完整显示） |
```

## 技术实现

### 修改位置
- 文件：`scripts/pdf_generator.py`
- 方法：`_create_monthly_platform_pages()`

### 关键改动

**1. 调整列宽**
```python
# 优化前：4列，每列宽度有限
colWidths=[4*cm, 2.5*cm, 6*cm, 2.5*cm]

# 优化后：3列，项目名称+链接合并为11cm
colWidths=[4*cm, 2.5*cm, 11*cm]
```

**2. 改为两行显示**
```python
# 每个项目占用两行
# 第一行：网站、日期、项目名称
row1 = [site, date, project_name]

# 第二行：链接（占满整行）
row2 = ['', '', detail_url]

table_data.append(row1)
table_data.append(row2)
```

**3. 样式优化**
```python
# 表头两行
table_data = [['网站', '日期', '项目名称']]
table_data.append(['', '', '项目链接'])

# 链接样式
('FONTSIZE', (2, 3), (2, -1), 8),  # 链接字体稍小
('TEXTCOLOR', (2, 3), (2, -1), colors.blue),  # 链接蓝色

# 行高调整
('ROWHEIGHT', (0, 2), (-1, -1), 25),  # 项目名称行高
('ROWHEIGHT', (0, 3), (-1, -1), 18),  # 链接行高
```

## 优化效果

### 显示效果

**优化前**：
```
| 盐城市政府采购网 | 2026-04-12 | 某某采购项目招标公告（截断...） | http://xxx.com/xxx/xxx... |
```

**优化后**：
```
| 盐城市政府采购网 | 2026-04-12 | 某某采购项目招标公告完整标题显示（自动换行） |
|                  |            | http://xxx.com/xxx/xxx/完整链接显示（自动换行） |
```

### 优势

✅ **完整显示**
- 项目名称不再截断，完整显示
- 链接不再截断，完整显示

✅ **自动换行**
- 使用 `WORDWRAP` 属性，长文本自动换行
- 不限制字符长度，根据内容自动调整

✅ **视觉优化**
- 链接使用蓝色显示，更易识别
- 链接字体稍小，层次分明
- 表格行高适中，便于阅读

✅ **空间优化**
- 将链接单独一行，充分利用空间
- 项目名称+链接合并列宽，从8.5cm增加到11cm

## 总结

通过将PDF表格从"单行显示"改为"两行显示"，并优化列宽和样式，彻底解决了标题和链接显示不全的问题。

**核心改进**：
- 项目名称和链接分开两行显示
- 扩大列宽，充分利用空间
- 自动换行，不限制字符长度
- 链接蓝色显示，易于识别

**用户体验提升**：
- 信息完整，不再遗漏
- 清晰易读，层次分明
- 空间利用更合理
