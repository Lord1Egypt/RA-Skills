## Description: <br>
Human-centric memory management organized by time, conversation, and topics. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[amlyx](https://clawhub.ai/user/amlyx) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Users and AI assistants use this skill to maintain local memory across daily work logs, full conversation backups, and topic-based folders. It is intended to make prior decisions, context, and project materials easier to find and reuse. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Persistent memory files can retain sensitive preferences, personal data, or secrets longer than intended. <br>
Mitigation: Redact sensitive content, keep memory files out of shared or synced locations unless appropriate, and periodically review or delete old memory files. <br>
Risk: Complete conversation backups may capture more private context than is needed for future recall. <br>
Mitigation: Prefer concise summaries when full transcripts are not necessary and save complete chats only when the user intentionally wants that level of local memory. <br>
