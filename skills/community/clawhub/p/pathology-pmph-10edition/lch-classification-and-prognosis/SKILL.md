# LCH分型与预后判断

## 适用前提
确保满足以下确诊条件后再执行本技能：
- **组织学**：增生细胞核呈折叠状、凹陷或分叶状，有核沟
- **免疫表型**：CD1a、Langerin（CD207）、S-100蛋白阳性（Langerin为特异性标记）
- **电镜**：胞质内可见Birbeck颗粒（杆状，200–400nm长，一端可呈网球拍状）
- **排除**：非朗格汉斯细胞来源的组织细胞增生症及反应性组织细胞增多

## 执行步骤

### 1. 确定受累器官系统数量
- 统计明确受累的器官系统（如骨骼、皮肤、肝、脾、肺、骨髓、垂体等）
- 记录是否涉及“危险器官”：肝、脾或骨髓

### 2. 进行LCH分型
- **单系统LCH**：仅累及一个器官系统（可为单灶或多灶）
- **多系统LCH**：累及≥2个器官系统

### 3. 判断预后风险等级
- **低危**：单系统LCH（无论是否多灶），无危险器官受累
- **高危**：多系统LCH 且 累及至少一个危险器官（肝、脾、骨髓）

### 4. 输出结果
根据上述判断，输出风险等级，并提示治疗强度建议：
- 低危：局部治疗（手术、放疗）或观察，预后良好
- 高危：需启动强力系统性化疗，密切监测

> **注意**：若可行，应检测BRAF p.V600E突变状态，因其在多系统LCH中常见，可能影响靶向治疗选择。
