# 🎯 中文内容精修助手 — Content Polisher ZH

> **AI 写的中文一眼就能看穿？这个技能帮你藏好 AI 尾巴。**

## 这技能解决什么问题？

你用过 ChatGPT、Claude 写中文内容吧？发到小红书、公众号、知乎上，评论总有人说"这是 AI 写的吧"。

不是内容不好，是**味儿太冲了**——那些"首先其次最后"、"总的来说"、"不可否认"，一看就是 AI 写的。

这个 Skill 就是干这个的：**去 AI 味 + 平台排版，一步到位。**

| 功能 | 说明 |

## 举个栗子 🌰

**AI 原文：**
```
在这个快节奏的时代，越来越多的人开始关注健康生活方式。
首先，合理饮食是保持健康的基础。其次，适量运动也是不可或缺的。
总而言之，只有三者兼顾才能真正拥有健康的身体。
```

**精修后（小红书版）：**
```
姐妹听我说💡健康这事真没那么复杂

吃饭别太狠，八分饱就好
运动不用天天泡健身房，每天散步30分钟也行
别熬夜，真的别熬

三样做好，身体会感谢你

#健康生活 #养生日常 #生活小习惯
```

看出差别了吗？这就是这个技能做的事。

在 OpenClaw 里输入：

