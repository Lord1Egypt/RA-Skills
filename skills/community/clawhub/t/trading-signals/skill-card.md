## Description: <br>
量化交易信号系统，基于 MACD、RSI、KDJ、布林带和均线等技术指标为股票和加密货币生成交易信号。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and market-analysis agents use this skill to fetch recent market data and summarize technical trading signals for stock and cryptocurrency symbols. The output is informational market analysis and should not be treated as personalized financial advice or automatic trading instructions. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated trading signals may be mistaken for personalized financial advice or automatic trading instructions. <br>
Mitigation: Treat the output as informational market analysis only and independently verify investment decisions. <br>
Risk: The documented dependency installation command uses --break-system-packages, which can affect the system Python environment. <br>
Mitigation: Install dependencies in a Python virtual environment instead of using --break-system-packages. <br>
