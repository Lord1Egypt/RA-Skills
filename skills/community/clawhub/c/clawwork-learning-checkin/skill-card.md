## Description: <br>
Workplace check-in skill for Agent (Claw) with AI-generated personalized greetings. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[daizongyu](https://clawhub.ai/user/daizongyu) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Employees and workplace users use this skill to perform daily check-ins, track streak status, and receive personalized welcome, success, and greeting messages generated by the agent. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The local helper saves the user's nickname, language preference, and recent generated greetings under the skill directory. <br>
Mitigation: Install only when local profile and greeting-history storage is acceptable, and review or clear the skill data folder when retention is no longer desired. <br>
Risk: The skill depends on the separate learning-checkin skill for check-in behavior. <br>
Mitigation: Review and trust the learning-checkin dependency before approving installation or use. <br>
Risk: Broad check-in phrasing could trigger unintended state updates. <br>
Mitigation: Use explicit check-in phrasing and confirm user intent before running commands that update check-in or greeting history. <br>
