# 文案公式库与爆款案例

## 标题公式大全

### A类：数字利益型
公式：`[数字]+[时间/数量]+[结果/成就]`
案例：
- 「28天减掉10斤，就靠这3件事」
- 「花了500块，把家里改造成了展厅」
- 「用了1年，整理出一套超好用的收纳系统」

### B类：对比反差型
公式：`同样是[共同条件]，为什么[她们/他们][好结果]而你[坏结果]`
案例：
- 「同样护肤，为什么她40岁看起来像25岁」
- 「同样做副业，为什么她月入2万你还在焦虑」

### C类：反常识型
公式：`[常见行为]千万别[动作]！除非你[特定条件]`
或：`[大多数人以为X]，其实[真相Y]`
案例：
- 「买护肤品，千万别看成分表！除非你懂这个」
- 「大家都在买的XX，我劝你慎重」

### D类：痛点共鸣型
公式：`买了[时间][商品]，终于找到[解决方案/平替]`
或：`[痛苦经历]之后，我终于明白...`
案例：
- 「熬了3年夜班，才发现这才是真正有效的护眼方法」
- 「买了10支口红踩雷，总结出这个选色公式」

### E类：情绪触发型
公式：`看到/用了[事物]的那一刻，我[强烈情绪反应]了`
案例：
- 「看到这个沙发的那一刻，我破防了」
- 「第一次用这款精华，我当场哭了」

### F类：干货分享型
公式：`[领域][动作]指南｜[承诺：踩过的坑/省钱/避坑]都帮你[结果]了`
案例：
- 「租房装修避坑指南｜花了8万踩过的坑都帮你踩了」
- 「旅行省钱攻略｜便宜好玩的地方我全给你整合了」

## 开头钩子模板

### 痛点式（引发共鸣）
```
你有没有这种感觉——[具体困境描述]？
我曾经也是这样的……直到遇见[转折]
```

### 悬念式（制造好奇）
```
说一个可能让你重新考虑[行为]的事实：[反常识信息]
很多人不知道[认知盲点]，包括我自己……
```

### 场景代入式（真实感）
```
上个月[具体场景]，我发现了一个[商品/方法]……
昨天闺蜜问我[问题]，我直接把这个发给她了
```

### 强利益式（直给价值）
```
[结果]只需要[简单方法]，今天把方法告诉你们
这个[方法/产品]让我[好处]，不分享出来说不过去
```

## 带货话术公式

### 价格锚点话术
```
原价[X元]，现在只要[Y元]，相当于[Z折]
换算下来每天只要[小数额]，[价值对比]
```

### 稀缺性话术
```
今天只有[N]件，卖完不补了
这个价格只有今天，明天就恢复原价
已经有[N]个人买了，手慢无
```

### 信任背书话术
```
我自己用了[时间]，真实感受是……
这是我回购[N]次的产品，不是托
[知名人物/媒体]也在用/推荐的
```

### 逼单收尾话术
```
真的，不用想了，就买这个，后悔包赔
点下面的购物车，今天的[优惠/赠品]还有
评论区扣「1」我给你发[优惠券/攻略]
```
