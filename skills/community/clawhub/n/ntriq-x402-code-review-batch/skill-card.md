## Description: <br>
Batch AI code review for up to 500 snippets. Flat $15.00 USDC via x402. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ntriq-gh](https://clawhub.ai/user/ntriq-gh) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineering teams use this skill to submit batches of code snippets to a third-party x402 API for AI review focused on security, performance, quality, or all categories. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Selected code snippets are sent to a third-party endpoint for review. <br>
Mitigation: Submit only code you are authorized to share, and avoid secrets, credentials, proprietary code, and regulated data unless the provider is trusted and sharing is permitted. <br>
Risk: Each call may authorize a $15.00 USDC payment through x402. <br>
Mitigation: Confirm the x402 payment request, network, amount, and policy approval before sending the request. <br>
