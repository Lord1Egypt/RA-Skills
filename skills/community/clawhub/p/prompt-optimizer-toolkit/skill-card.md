## Description: <br>
Analyzes, optimizes, templates, and scores LLM prompts with built-in prompt engineering strategies. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kaiyuelv](https://clawhub.ai/user/kaiyuelv) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and prompt engineers use this skill to evaluate prompt quality, generate improvement suggestions, apply common prompt templates, and manage reusable prompts locally. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Optional prompt library storage can write saved prompts to local plain JSON files. <br>
Mitigation: Avoid saving API keys, credentials, proprietary prompts, or personal data unless the storage path and cleanup process are controlled. <br>
