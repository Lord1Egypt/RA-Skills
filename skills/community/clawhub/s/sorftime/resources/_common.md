# Sorftime CLI 公共参考（_common.md）

> 所有 resources/*.md 共用的 Domain 表、错误码表、返回结构、CLI 调用模板。各接口文档**只保留接口本身的参数/字段/示例**，公共部分一律引用本文件。

---

## 1. CLI 调用模板

```bash
sorftime api <Endpoint> '<json-params>' --domain <N> --profile <name>
```

| 选项 | 说明 |
|------|------|
| `<Endpoint>` | 接口名，**严格区分大小写**，按本目录各文件中的 PascalCase 写法（如 `ProductRequest`、`KeywordBatchSubscription`）。CLI 的 README 里小写形式仅作示例，实际请求严格按 PascalCase。 |
| `<json-params>` | 单引号包裹的 JSON 字符串。多参数接口可省略不必需字段，按各接口文档要求传。 |
| `--domain` | 站点代码（见下表）。Amazon 1-14、Shopee 201-208、Walmart 21。**忘传或传错会直接 401/404**。 |
| `--profile` | profile 名称，对应 `sorftime add` 创建的鉴权配置（含 token 与默认 domain）。可省略，省略时用 `sorftime use` 当前选中的 profile。 |


**Profile 管理速查**：
```bash
sorftime add <profile-name> <api-key>          # 添加 profile-name
sorftime list                                  # 查看所有 profile-name
sorftime use <profile-name>                    # 切换默认 profile-name
sorftime whoami                                # 查看当前 profile-name
sorftime rm <profile-name>                     # 删除 profile-name
```

## 5. 通用返回结构

所有接口统一返回 JSON：

```json
{
  "requestleft": 3000,
  "requestconsumed ": 1,
  "code": 0,
  "message": "success",
  "data": { ... }
}
```

| 字段 | 说明 |
|------|------|
| `requestleft` | 剩余request数 |
| `requestconsumed` | 本次请求消耗请求数 |
| `code` | 业务状态码，**判断成功的唯一依据**（见错误码表）。`0` 表示成功；非 0 即为业务错误。 |
| `message` | 状态描述，错误时含具体原因。 |
| `data` | 业务数据，结构各接口不同。 |

## 6. 错误码表

### Amazon 业务错误码

| code | 含义 | 排查建议 |
|------|------|---------|
| `0` | 执行成功 | — |
| `4` | 积分余额不足 | 充值积分或等待下月重置 |
| `10` | 请求参数错误 | 检查接口参数格式、类型、必填项 |
| `11` | 暂无数据 | 该条件下无数据返回，尝试调整参数 |
| `12` | 数据已存在 | 重复提交相同数据，无需重复操作 |
| `13` | 首页评论数据不存在 | 需先通过 ProductRealtimeReviews 接口采集评论 |
| `14` | ASIN 数量超限 | 单次最多支持 100 个 ASIN |
| `15` | ASIN 查询数量超限 | 单次最多支持 10 个 ASIN |
| `16` | 接口不支持 | 该站点不支持此接口 |
| `17` | ASIN 采集数量超限 | 单次最多支持 30 个 ASIN 采集详情 |
| `18` | 任务详情查询超限 | 单次最多支持 20 个任务详情查找 |
| `19` | 仅支持细分类目 | 非细分类目不支持查询 |
| `20` | 实时数据计算中 | 请稍后重试 |
| `21` | 部分 ASIN 计算中 | 部分 ASIN 仍在计算，请稍后重试 |
| `22` | ASIN 监控数量超限 | 库存监控仅支持单 ASIN |
| `23` | 超两年数据不支持 | 暂不支持查询近两年以外的数据 |
| `24` | 不支持月份查询 | 暂不支持输入月份的历史数据查询 |
| `-1` | 执行失败 | 一般性错误，查看 message 获取详情 |
| `97` | ASIN 不存在 | 检查 ASIN 是否正确 |
| `98` | 采集失败 | 稍后重试，或联系 Sorftime 客服 |
| `99` | 采集中 | 正在实时抓取，预计耗时5分钟，请稍后重试 |
| `400` | 非认证 IP | 当前 IP 未在白名单中 |
| `401` | 请求接口未开放 | 检查接口名称是否正确，套餐是否包含此接口 |
| `402` | 无权查看此数据 | 检查账号权限 |
| `500` | 本月请求数量已达限额 | 等待下月重置或升级套餐 |
| `501` | 一分钟内请求已达限额 | 降低请求频率，1分钟后重试 |
| `502` | 一天内请求已达限额 | 等待次日重置或升级套餐 |
| `503` | 注册任务失败 | 该时刻任务量超限，换个时段重试 |
| `694` | 请求数量不足 | 请求参数不满足最低数量要求 |
