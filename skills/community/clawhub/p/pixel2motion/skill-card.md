## Description: <br>
Pixel2Motion turns raster logos into smooth, motion-ready SVGs and choreographed standalone HTML animations with QA artifacts for vector fit and motion continuity. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[frede2010](https://clawhub.ai/user/frede2010) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Designers, developers, and brand teams use Pixel2Motion to convert logo images into editable SVGs and interactive logo reveals with motion specs and QA evidence. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can run local scripts, install browser automation dependencies, launch Chrome or Chromium, and write generated design and QA files. <br>
Mitigation: Install and run it in a local project workspace, review commands before execution, and keep sensitive files outside generated output paths. <br>
Risk: The optional video export path and generated HTML/media outputs may create files that are not appropriate to publish unchanged. <br>
Mitigation: Review the optional export script and generated artifacts before sharing or committing them. <br>
