# Travel.Skills - 全能型家庭旅游规划智能体

## 简介

Travel.Skills 是一个专为家庭出游设计的AI智能体，特别适合五一、国庆等旅游旺季使用。它能：

- 🚫 **智慧避堵**: 识别热门拥挤景区，推荐小众宝藏地
- 👴 **适老化设计**: 平路优先、强制午休、门票优惠查询
- 👶 **亲子化体验**: 年龄匹配、任务卡生成、互动体验
- 🎫 **预订助手**: 门票酒店比价、官方渠道推荐
- 📋 **可视化路书**: 时间轴行程、出行清单、预算预估

## 使用场景

### 场景1: 五一全家游
```
用户: 五一我想带全家出去玩，一家5口，有70岁的老人和8岁的小孩。
     不想去人挤人的地方，老人腿脚不好，小孩喜欢动物。
```

### 场景2: 周末亲子游
```
用户: 这个周末带孩子去玩，孩子6岁，喜欢恐龙，预算500元。
```

### 场景3: 老人休闲游
```
用户: 想带父母出去玩，他们都是65岁，腿脚不太好，想去轻松的地方。
```

## 核心模块

### 1. 智慧避堵雷达
- 排雷热门景区（标记高风险）
- 挖掘小众宝藏地
- 路况预测和出发时间建议

### 2. 全家福行程引擎
| 模式 | 适用人群 | 核心特点 |
|------|----------|----------|
| 适老模式 | 60岁+ | 平路优先、午休强制、政策查询 |
| 亲子模式 | 0-14岁 | 年龄匹配、任务卡、互动体验 |

### 3. 门票酒店预订助手
- 官方预订渠道提供
- 价格对比建议
- 不直接代订

### 4. 可视化路书生成
- 时间轴式行程
- 出行清单检查
- 预算预估

## 技术实现

### 变量配置
```yaml
travel_days: 天数
elderly_count: 老人人数
elderly_age: 老人年龄
kids_count: 儿童人数
kids_age: 儿童年龄
budget: 预算范围
destination: 目的地（可选）
travel_date: 出行日期
```

### 插件需求
- 搜索插件（必应/头条）：获取实时客流、路况
- 地图插件（高德/百度）：距离计算、拥堵分析
