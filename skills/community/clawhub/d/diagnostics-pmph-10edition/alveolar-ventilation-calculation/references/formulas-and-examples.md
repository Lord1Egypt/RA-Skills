# 公式详解与临床示例

## 核心公式

肺泡通气量（Alveolar Ventilation, VA）表示每分钟进入肺泡并参与气体交换的气体体积，计算公式如下：

### 基本形式
```
VA = (VT - VD) × RR
```

### 比值形式
```
VA = VT × (1 - VD/VT) × RR
```

两种形式等价，后者更便于分析 VD/VT 比值的影响。

## 参数说明
- **VT（潮气容积）**：每次平静呼吸吸入或呼出的气体量，正常成人约 500 ml。
- **VD（生理无效腔）**：包括解剖无效腔（约 150 ml）和肺泡无效腔（通常可忽略），总和约 150 ml。
- **RR（呼吸频率）**：每分钟呼吸次数，正常成人约 15 次/分。
- **VD/VT**：无效腔与潮气容积比值，正常范围 0.3～0.4。

## 正常值计算示例
```
VA = (500 ml - 150 ml) × 15 /min = 350 × 15 = 5250 ml/min = 5.25 L/min
```

## 异常情况模拟
若患者采用浅快呼吸（VT=250 ml, RR=30 次/分），且 VD 不变（150 ml）：
```
VA = (250 - 150) × 30 = 100 × 30 = 3000 ml/min = 3.0 L/min
```
尽管每分钟通气量（VE = VT × RR = 7.5 L/min）高于正常，但 VA 反而下降，说明通气效率降低。

若 VD/VT 升高至 0.7（如肺栓塞、肺气肿）：
```
VA = 500 × (1 - 0.7) × 15 = 500 × 0.3 × 15 = 2250 ml/min = 2.25 L/min
```
显著低于正常，提示严重通气效率障碍。

## 临床推论
- **深缓呼吸**（如 VT=750 ml, RR=10）：VA = (750-150)×10 = 6.0 L/min → 效率高
- **浅速呼吸**（如 VT=250 ml, RR=30）：VA = 3.0 L/min → 效率低

因此，“浅速呼吸的通气效率逊于深缓呼吸”的本质在于 VD/VT 比值相对增大，导致有效肺泡通气减少。
