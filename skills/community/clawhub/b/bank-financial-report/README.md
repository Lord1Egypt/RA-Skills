# Bank Financial Report Analyst
# 银行财报分析助手

> *AI-powered financial statement analysis — balance sheet, income statement, cash flow, DuPont analysis, peer comparison, and financial health diagnostics.*

> AI驱动的财务报表分析工具——资产负债表、利润表、现金流量表、杜邦分析、同业对比、财务健康诊断，一站式完成。

---

## Overview

This skill provides comprehensive financial statement analysis for bank analysts and credit officers. It covers three-statement analysis, ratio analysis, DuPont decomposition, Altman Z-Score, cash flow quality assessment, peer comparison, and financial red flag identification.

本工具为银行分析师和信贷人员提供全面的财务报表分析，涵盖三表分析、财务比率、杜邦分解、Altman Z-Score、现金流质量评估、同业对比及财务风险预警。

## Installation


## Core Capabilities

| Capability | Description |
|------------|-------------|
| Three-Statement Analysis | Balance sheet, income statement, cash flow breakdown |
| Ratio Analysis | Profitability, leverage, liquidity, efficiency |
| DuPont Analysis | ROE decomposition and diagnosis |
| Altman Z-Score | Financial distress prediction |
| Cash Flow Quality | CFO/net income ratio, FCF sustainability |
| Peer Comparison | Multi-company cross-sectional analysis |
| Red Flag Detection | Financial fraud indicators |

## Use Cases

- **Credit financial analysis** (信贷财务分析)
- **Investment research** (投资研究报告)
- **Due diligence financial review** (尽调财务审查)
- **Peer benchmarking** (同业对标分析)
- **Financial health check** (财务健康诊断)

## Analysis Methods

- Altman Z-Score (制造业上市公司)
- DuPont Analysis (ROE归因)
- Cash Conversion Cycle
- Net Income to CFO Ratio
- EBITDA Coverage Analysis
