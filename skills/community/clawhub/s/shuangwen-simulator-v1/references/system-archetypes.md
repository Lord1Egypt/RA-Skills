# System Archetypes

Use these when the player asks for a different 爽文 system or when designing a new short scenario.

## Shared Loop

```text
concrete humiliation / betrayal / pressure
→ system binding or system refresh
→ inspect shop / item / evidence / mission
→ player chooses how to use the advantage
→ public reversal
→ reward and next pressure
```

Rules:

- One item must help the immediate scene.
- One item must create a bigger hook.
- One item must be flexible for player creativity.
- Redeeming an item does not resolve the scene by itself. Ask how the player uses it.
- The player should control reveal timing.

## 神豪系统

Fantasy: the player is despised as poor or low-status, then buys the social stage out from under the villain.

Resources:

- 消费额度
- 神豪点
- 返现倍数
- 资产清单
- 身份可信度

Shop examples:

- 至尊黑卡
- 酒店即时收购协议
- 停车场产权转让
- 私人管家团队
- 限量超跑钥匙
- 品牌晚宴邀请函
- 热搜反转券

Good openings:

- luxury shop clerk refuses service,
- hotel manager blocks entry,
- ex-lover mocks the player at a reunion,
- parking space is occupied and the other party insults the player,
- family dinner treats the player as failure.

System voice:

```text
叮！检测到宿主遭受公开羞辱。
神豪系统已绑定。
新手额度：1000000 元。
目标越看不起你，返现倍数越高。
```

## 无限系统

Fantasy: everyone thinks this is the first timeline, but the player remembers betrayal and death.

Resources:

- 轮回点
- 保留槽位
- 存档次数
- 死亡节点倒计时
- 证据可信度

Shop examples:

- 上一世监控回放
- 时间锚点
- 合同替换卡
- 死亡预警
- 未来新闻截屏
- 因果锁链

Good openings:

- forced to sign a document that killed the player last time,
- sibling repeats the exact lie from the previous life,
- partner starts the same betrayal,
- disaster countdown begins.

System voice:

```text
检测到宿主死亡节点重启。
本轮倒计时：4 天。
每轮仅可保留 1 件证据。
请谨慎选择。
```

## 装逼打脸系统

Fantasy: the villain says something arrogant in public; the player locks that sentence and later returns it.

Resources:

- 装逼值
- 震惊值
- 破防值
- 围观热度
- 承诺锁定

Shop examples:

- 气势外放
- 台词暴击
- 隐藏境界体验卡
- 全场静音
- 反派承诺锁定
- 吃瓜群众起哄
- 身份揭示倒计时

Good openings:

- public退婚,
- boss says the player will never enter the industry,
- sect elder says "if you can take one strike, I kneel",
- villain says nobody will believe the player.

System voice:

```text
叮！检测到公开羞辱。
对方原话已锁定。
本场目标：让他说完，再让他兑现。
```

## Moral Contrast

Use one concrete bad act and one concrete good act.

Villain details:

- greed,
- betrayal,
- coercion,
- hypocrisy,
- opportunism,
- social climbing,
- kicking someone while down.

Good person details:

- warns the player when they are low-status,
- refuses to lie,
- shares last resource,
- saves a piece of evidence,
- protects a child or elder without asking for reward.

Payoff:

- expose,
- remove leverage,
- rescue victim,
- make public apology,
- reverse contract,
- protect dignity,
- create title or status.
