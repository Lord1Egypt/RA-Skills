## Description: <br>
Etsy SEO analyzer and optimizer for improving search visibility with title optimization, tag analysis, description scoring, keyword research, SEO scoring, long-tail keyword suggestions, and prioritized action plans. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[phheng](https://clawhub.ai/user/phheng) <br>

## Use Case: <br>
External sellers and ecommerce operators use this skill to analyze Etsy listing text, score SEO quality, and generate title, tag, description, keyword, and action-plan improvements. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The documented installation path uses a global external package install. <br>
Mitigation: Review or pin the referenced package before using the global install command. <br>
Risk: Etsy business data pasted into the tool is processed as local input. <br>
Mitigation: Only provide listing data appropriate for local processing in the execution environment. <br>
