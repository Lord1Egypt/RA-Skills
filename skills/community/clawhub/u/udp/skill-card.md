## Description: <br>
Helps with UDP sockets, datagram send/receive, and network debugging. Use when the user needs to send or receive UDP packets, write UDP client/server code, test UDP ports, or debug packet loss and firewall issues. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to produce UDP command examples, language-specific socket code, and troubleshooting guidance for datagram communication, port testing, packet loss, firewalls, broadcast, and multicast. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated UDP commands or code can send real network traffic to a host and port. <br>
Mitigation: Verify that the target system is owned by the user or explicitly authorized before running generated UDP tests or examples. <br>
Risk: Public UDP services can be exposed to spoofed traffic, abuse, or amplification patterns. <br>
Mitigation: Use application-level authentication and validation for public UDP services, and review firewall or security group rules before deployment. <br>
