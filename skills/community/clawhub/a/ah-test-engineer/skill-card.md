## Description: <br>
Test Engineer helps agents design comprehensive testing strategies and automation across testing types, frameworks, the test pyramid, and BDD/TDD approaches. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to plan test strategy, choose frameworks, draft test cases, define automation and CI/CD integration, and identify coverage or quality metrics. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated tests, cleanup commands, or CI/CD changes may introduce incorrect guidance or alter project behavior if applied without review. <br>
Mitigation: Review generated test implementations, cleanup steps, and pipeline changes before running or committing them. <br>
