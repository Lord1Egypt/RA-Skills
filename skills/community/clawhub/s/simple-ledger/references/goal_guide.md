# 财务目标指南

## 什么是财务目标

财务目标是有截止日期和目标金额的存款计划。比如：
- 旅行基金：¥10,000，年底前
- 应急基金：¥30,000，1年内
- 买车：¥50,000，3年内

## 基础操作

### 创建目标

```bash
python scripts/goal.py create "旅行基金" 10000 2026-12-31
```

### 存入金额

```bash
python scripts/goal.py deposit "旅行基金" 2000
```

### 查看进度

```bash
python scripts/goal.py progress "旅行基金"
```

### 列出所有目标

```bash
python scripts/goal.py list
```

### 删除目标

```bash
python scripts/goal.py delete "旅行基金"
```

## 预测逻辑

系统会根据你的存款历史计算月均存款速度，判断能否按时达成：

- 月均存款速度基于存款记录自动计算
- 预测结果会在进度中显示
- 如果速度不够，会告诉你每月需要存多少才能达标

## 常见目标类型

### 应急基金（优先级最高）

目标金额 = 3-6 个月生活支出
截止日期：随时开始，越早越好
建议策略：先把紧急目标存满

### 旅行基金

目标金额：按目的地估算
截止日期：出发前 1 个月
建议：把旅行费用单独设账本，避免和生活费混在一起

### 购房首付

目标金额：按目标城市房价估算
截止日期：5-10 年
建议：用定期存款 + 基金定投组合

### 养老储备

目标金额：按退休后生活水平估算
截止日期：退休前
建议：越早开始越好，利用复利效应

## 实用建议

1. **一个时间只追一个目标** — 同时追 3 个目标等于没有目标
2. **截止日期设紧一点** — 宽松的截止日期让人拖延
3. **记录每一笔存款** — 用 `deposit` 指令，积累存款历史用于预测
4. **达成后立即设新目标** — 不要让攒钱这件事停下来
5. **应急基金优先于投资** — 应急金没存满之前不要投资高风险产品
