# 新生儿黄疸风险评估与生理性黄疸诊断

## 适用条件
- 新生儿胎龄 ≥ 35 周
- 已测定血清总胆红素（TSB）
- 已知出生后小时龄及胎龄
- 已完成高危因素评估
- **不适用于**胆红素脑病急性期
- **不可仅凭单次TSB值**做出判断

## 执行步骤

### 1. 风险分层评估
使用Bhutani小时龄胆红素风险评估曲线（见参考资料），根据以下要素确定风险分区：
- 出生后小时龄
- 胎龄（足月/晚期早产）
- 存在的高危因素数量

输出结果为：低危、中危、高危或极高危。

### 2. 高危因素识别
确认是否存在以下任一高危因素（越多风险越高）：
- 新生儿溶血
- 头颅血肿
- 窒息
- 缺氧
- 酸中毒
- 脓毒症
- 低蛋白血症
- 低血糖

### 3. 生理性黄疸诊断（排除性诊断）
仅当**同时满足以下全部条件**时，可诊断为生理性黄疸：
- 一般情况良好
- 黄疸出现与消退时间符合典型规律：
  - 足月儿：生后2～3天出现，4～5天达峰，5～7天消退（≤2周）
  - 早产儿（≥35周）：3～5天出现，5～7天达峰，7～9天消退（最长≤3～4周）
- 胆红素上升速率未超标：
  - 生后24小时内：每小时增加 < 0.3 mg/dl（≈5.1 μmol/L）
  - 24小时后：每小时增加 < 0.2 mg/dl（≈3.4 μmol/L），即每日升高 < 5 mg/dl（85 μmol/L）
- TSB水平低于对应小时龄Bhutani曲线的P95百分位，且未达到光疗干预阈值

### 4. 单位换算与临床提示
- 胆红素单位换算：1 mg/dl = 17.1 μmol/L
- 肉眼可见黄疸通常出现在TSB ≥ 5 mg/dl（85.5 μmol/L）

> **注意**：传统“生理性/病理性”二分法已被基于小时龄的风险分层模型取代，应优先依据Bhutani曲线进行决策。

## 输出结果
- 风险分区（低/中/高/极高危）
- 是否符合生理性黄疸诊断标准
- 是否建议光疗干预（需结合光疗阈值表）
