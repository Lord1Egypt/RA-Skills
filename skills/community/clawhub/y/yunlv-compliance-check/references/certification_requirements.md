# 产品认证要求清单（按目标市场）

|------|------|----------|------|--------|
| **UL** | Underwriters Laboratories | 美国安全认证（自愿性，但大量采购商要求） | 4-8周 | 持续（年度审厂） |
| **ETL** | Electrical Testing Labs | 美国安全认证（UL竞争认证） | 3-6周 | 持续（年度审厂） |

## 认证准备清单

### 认证申请前准备
- [ ] 产品技术规格书（英文）
- [ ] 产品电路图/结构图（如适用）
- [ ] 关键元器件清单（BOM）及认证证书
- [ ] 工厂质量管理体系证书（ISO9001优先）
- [ ] 产品照片（外观+铭牌）
- [ ] 英文使用说明书
- [ ] 目标市场的产品标签设计（含警告语）

### 认证时间预估
```
简单电子设备（手机充电线、耳机）：
- CE: 2-3周
- FCC: 2-4周

复杂电子设备（LED灯具、家电）：
- CE: 4-6周（含RoHS测试）
- UL/ETL: 6-10周（含工厂审查）

机械/工业设备：
- CE (MD): 8-16周（含风险评估）
- CCC: 3-6个月

消费品（玩具、纺织）：
- CPSC/REACH: 4-8周
- OEKO-TEX: 4-8周
```
