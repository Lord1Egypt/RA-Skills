# 🎬 短视频全链路编导 (video-script-to-post)

> 版本 `1.1.0` | 创作者/MCN/品牌方内容引擎 | 分镜规划 → AI提示词 → 平台排期 → 合规校验

## 🎯 功能简介

面向短视频创作者、MCN机构与品牌运营团队，提供从创意到发布的标准化工程包。自动匹配平台算法偏好（完播率/互动权重/推荐标签），生成秒级分镜脚本、AI视觉生成参数、封面标题策略与合规风险提示，大幅降低内容制作试错成本。

## 📋 支持的平台

| 平台 | 标识符 | 特性 |
|------|--------|------|
| 抖音 | `douyin` | 竖屏 9:16，强算法推荐，3 秒完播率是关键 |
| B站 | `bilibili` | 横屏 16:9，弹幕互动文化，中长视频友好 |
| 视频号 | `video_account` | 微信生态社交裂变，好友点赞推荐 |
| 小红书 | `xiaohongshu` | 搜索 SEO 权重高，图文+视频混合，生活方式调性 |

## 📥 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `topic_or_product` | string | ✅ | 核心主题/产品名/活动名称 |
| `target_platform` | string | ✅ | 发布平台标识符（见上表） |
| `video_duration` | string | ❌ | 目标时长：`15s` / `30s` / `60s` / `3min`（默认 `30s`） |
| `tone_style` | string | ❌ | 内容调性：`专业科普` / `剧情反转` / `情绪共鸣` / `干货拆解` / `生活记录` |

## 🚀 快速开始

### 1. 明确内容定位
- 确定核心主题/产品卖点/活动节点
- 选择目标平台与内容调性（不同平台节奏与审核尺度差异极大）

### 2. 配置调用

```json
{
  "skill": "video_script_to_post",
  "inputs": {
    "topic_or_product": "智能办公升降桌",
    "target_platform": "douyin",
    "video_duration": "30s",
    "tone_style": "干货拆解"
  }
}
```

### 3. 获取输出

返回标准 Markdown 工程包，包含以下 4 个章节：

| 章节 | 内容 |
|------|------|
| ① 分镜脚本表 | 秒级画面描述、台词字幕、音效BGM、AI视觉提示词 |
| ② AI生成参数包 | Midjourney / ComfyUI / Runway / Pika / Kling / Sora / Vidu / 即梦 参数配置 |
| ③ 平台运营策略 | 封面标题、推荐标签、发布时间、评论区互动话术 |
| ④ 合规与版权提醒 | 广告法敏感词、字体/音乐版权状态、平台公约红线 |

## 🤖 支持的 AI 工具

| 类别 | 工具 | 用途 |
|------|------|------|
| 文生图 / 图生视频 | Midjourney, ComfyUI | 画面生成、角色/场景设计 |
| 视频生成 | Runway, Pika, Kling（可灵） | 动态镜头、运动控制 |
| 文生视频 | Sora, Vidu, 即梦 | 长视频生成、风格化短片 |

> 💡 具体参数配置示例见 `SKILL.md` 输出模板章节。
