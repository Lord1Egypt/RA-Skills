# 量-效关系曲线分析与安全性评价

## 适用场景
- 需要确定药物的最小有效剂量、最大效应或半数有效/致死剂量
- 比较两种或多种药物的效价强度或安全性
- 设计临床或动物实验的给药方案
- 评估药物在剂量依赖性反应中的风险收益比

## 执行步骤

### 1. 判断反应类型
- **量反应**（graded response）：效应连续变化（如血压、心率），适用于单个生物单位；绘制真数剂量得直方双曲线，对数剂量得S形曲线。
- **质反应**（all-or-none response）：效应为全或无（如死亡/存活），需群体数据；累计阳性率对剂量作图得S形曲线。

### 2. 提取关键参数
- **量反应参数**：
  - 最小有效剂量/浓度（阈剂量）
  - 最大效应（Emax，反映效能）
  - 半最大效应浓度（EC50）
  - 效价强度（EC50越小，效价越高）
  - 曲线斜率（陡峭提示药效剧烈）
- **质反应参数**：
  - 半数有效量（ED50）
  - 半数致死量（LD50）

### 3. 安全性评价
- 计算治疗指数：TI = LD50 / ED50（TI越大越安全）
- 若ED与LD曲线重叠，改用更保守指标：LD1/ED99 或 LD5/ED95 的剂量间距

### 4. 注意事项
- 区分**效能**（Emax）与**效价强度**：二者不平行，例如呋塞米Emax高于氢氯噻嗪，但后者效价更高。
- 本方法不适用于非剂量依赖性反应（如变态反应）。
