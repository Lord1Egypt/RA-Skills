# 联系方式获取渠道

> 本文档详细说明合法合规获取联系方式的渠道清单和操作流程。

---

## 一、官方公开渠道

### 1.1 优先级：高

#### 1.1.1 上市公司公告

| 获取内容 | 说明 |
|----------|------|
| **董秘联系方式** | 年度报告、招股说明书中披露 |
| **高管信息** | 董事会成员、高管团队名单 |
| **薪酬信息** | 高管薪酬水平（间接反映重要性） |
| **股权变动** | 大股东增减持信息 |

**获取路径**：
```
巨潮资讯网（www.cninfo.com.cn）
→ 搜索目标公司
→ 选择"公告"栏目
→ 筛选"年度报告"/"招股说明书"
→ 下载PDF查看
```

#### 1.1.2 投资者关系页面

| 获取内容 | 说明 |
|----------|------|
| **董秘邮箱** | 投资者关系页面通常公示 |
| **投资者热线** | 电话和传真号码 |
| **IR团队** | 投资者关系团队联系方式 |

**获取路径**：
```
目标公司官网
→ 投资者关系（IR）
→ 联系方式/Contact Us
```

#### 1.1.3 互动易/上证e互动

| 获取内容 | 说明 |
|----------|------|
| **董秘回复记录** | 了解董秘风格和关注点 |
| **公司回应** | 了解公司对特定问题的态度 |
| **提问入口** | 可直接提问（但回复较慢） |

**获取路径**：
```
深交所互动易：irss.cninfo.com.cn
上交所e互动：sns.sseinfo.com
→ 注册账号
→ 搜索目标公司
→ 查看互动记录
```

#### 1.1.4 投资者调研记录

| 获取内容 | 说明 |
|----------|------|
| **机构调研名单** | 参与调研的机构和分析师 |
| **调研纪要** | 了解公司关注点 |
| **联系方式** | 机构联系人信息 |

**获取路径**：
```
巨潮资讯网
→ 搜索"投资者关系活动记录"
→ 查看调研详情
```

## 二、商业信息平台

### 2.1 优先级：中

#### 2.1.1 天眼查/企查查

| 获取内容 | 说明 |
|----------|------|
| **高管任职信息** | 当前任职和历史任职 |
| **股权穿透** | 实控人、股东结构 |
| **关联企业** | 高管关联的其他公司 |
| **联系方式** | 部分平台提供联系方式（需付费） |

**使用技巧**：
- 搜索公司名称 → 查看"高管"栏目
- 查看"股东信息"了解股权结构
- 使用"族谱"功能查看关联关系

#### 2.1.2 东方财富/同花顺

| 获取内容 | 说明 |
|----------|------|
| **董秘详细信息** | 简历、教育背景 |
| **机构持仓** | 主要持股机构 |
| **研报信息** | 分析师联系方式 |
| **高管变动** | 历史变动记录 |

**使用技巧**：
- 个股详情页 → "高管团队"
- 研报栏目 → 查看分析师信息
- 股东分析 → 查看主要机构股东

#### 2.1.3 Wind/Choice（机构用户）

| 获取内容 | 说明 |
|----------|------|
| **完整高管简历** | 详细职业履历 |
| **薪酬数据** | 高管薪酬水平对比 |
| **并购历史** | 公司历史并购记录 |
| **分析师覆盖** | 跟踪该公司的分析师 |

## 三、职场社交平台

### 3.1 优先级：中低

#### 3.1.1 LinkedIn

| 获取内容 | 说明 |
|----------|------|
| **高管履历** | 完整职业路径 |
| **教育背景** | 毕业院校、专业 |
| **共同好友** | 寻找中间人 |
| **公司主页** | 公司动态 |

**使用注意事项**：
- 需要真实账号，避免虚假profile
- 通过共同好友建立联系
- 关注公司主页获取动态

#### 3.1.2 脉脉

| 获取内容 | 说明 |
|----------|------|
| **职场八卦** | 公司内部动态 |
| **人脉关系** | 共同好友分析 |
| **离职员工** | 可能了解内部情况 |
| **招聘动态** | 公司扩张/收缩信息 |

## 四、线下活动渠道

### 4.1 优先级：高（效果最好）

#### 4.1.1 行业峰会/论坛

| 活动类型 | 参与价值 | 获取信息 |
|----------|----------|----------|
| **行业年会** | 高管公开演讲 | 演讲人信息、联系方式 |
| **产业论坛** | 政策制定者参与 | 行业趋势、高管观点 |
| **投资峰会** | 机构投资者云集 | 接触投资总监、CFO |
| **创业大赛** | 评委多为产业专家 | 潜在独董资源 |

**参与技巧**：
- 提前了解参会嘉宾名单
- 选择目标人物主持的分论坛
- 准备有深度的提问

#### 4.1.2 投资者交流会

| 活动类型 | 接触对象 | 参与方式 |
|----------|----------|----------|
| **业绩发布会** | CFO、IR团队 | 机构投资者可参加 |
| **券商策略会** | 上市公司高管 | 报名参加 |
| **反向路演** | 上市公司邀请机构 | 需有机构背景 |
| **股东大会** | 全体股东 | 以股东身份参加 |

**股东大会参与要点**：
- 提前预约参会资格
- 设计有深度的问题
- 会后主动交流

#### 4.1.3 商会/协会活动

| 活动类型 | 接触对象 | 参与方式 |
|----------|----------|----------|
| **地方商会** | 企业家群体 | 申请加入 |
| **行业协会** | 产业链上下游 | 会员优先 |
| **校友会** | 校友资源 | 校友活动 |
| **EMBA校友** | 企业高管 | 商学院资源 |

## 五、联系方式获取流程

### 5.1 标准流程

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        联系方式获取标准流程                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  Step 1：明确目标                                                       │
│  └─ 确定需要联系的关键人及其职位                                         │
│                                                                          │
│  Step 2：选择渠道                                                       │
│  └─ 按优先级选择合适的获取渠道                                           │
│                                                                          │
│  Step 3：信息收集                                                       │
│  └─ 系统性收集公开信息                                                  │
│                                                                          │
│  Step 4：交叉验证                                                       │
│  └─ 多渠道信息互相验证                                                  │
│                                                                          │
│  Step 5：记录归档                                                       │
│  └─ 记录信息来源和使用方式                                               │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

### 5.2 渠道优先级选择

| 情况 | 推荐渠道顺序 |
|------|--------------|
| **需要快速联系董秘** | 互动易 → 官网IR → 年报公告 |
| **需要接触高管层** | 行业活动 → 股东大会 → 商会 |
| **需要找到中间人** | LinkedIn → 共同好友 → 行业活动 |
| **需要了解背景** | 天眼查 → Wind → 研报 |

## 七、禁止事项（重要提醒）

### 7.1 绝对禁止的行为

| 禁止类型 | 具体行为 | 法律风险 |
|----------|----------|----------|
| **数据购买** | 从黑市购买个人信息 | 侵犯公民个人信息罪 |
| **非法爬取** | 自动化爬取未授权信息 | 违反《网络安全法》 |
| **虚假伪装** | 冒充其他身份获取信息 | 欺诈罪、诈骗罪 |
| **骚扰行为** | 反复电话/邮件轰炸 | 侵犯名誉权、骚扰 |

### 7.2 合规使用原则

| 原则 | 说明 |
|------|------|
| **最小必要** | 只获取实现目的所必需的信息 |
| **来源合法** | 只使用公开渠道获取的信息 |
| **使用正当** | 只用于正常的商业沟通 |
| **保护隐私** | 不泄露、不买卖获取的信息 |
