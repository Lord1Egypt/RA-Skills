## 5. 合规与监管

### 5.1 医疗广告专属限制（**重点**）

| 限制类型 | 禁用 | 法规依据 |
|---------|------|---------|
| 医疗广告审查 | 所有医疗广告需取得《医疗广告审查证明》 | 《医疗广告管理办法》 |
| 效果承诺 | "保证效果""无创""无痛""零风险" | 《广告法》 |
| 案例对比 | Before/After 过度使用 | 平台审核 |
| 医生背书 | 需医生书面同意 + 资质真实 | 《广告法》 |
| 价格欺诈 | "原价 X" 必须有真实销售记录 | 《价格法》 |

### 5.2 行业特殊资质

| 业务类型 | 必备资质 |
|---------|---------|
| 医疗机构 | 《医疗机构执业许可证》 |
| 医生 | 《医师资格证》+《医师执业证》 |
| 药品/器械 | NMPA 注册/备案 |
| 医疗广告 | 《医疗广告审查证明》 |

### 5.3 平台审核红线

| 平台 | 红线 |
|------|------|
| 抖音 | Before/After 过度、效果承诺、低俗擦边 |
| 小红书 | 软广未声明、虚假案例、夸大效果 |
| 全平台 | 未取得医疗广告审查证明的广告 |

### 5.4 特殊合规要求

| 要求 | 说明 |
|------|------|
| Before/After | 必须标注"效果因人而异"，不得过度 P 图 |
| 医生 IP | 必须真实医生，不得用演员冒充 |
| 案例展示 | 必须取得患者书面同意 |
| 项目命名 | 不得使用"永久""彻底"等绝对化用语 |

## 7. 常见雷区

| 雷区 | 严重度 | 规避方法 |
|------|:-----:|---------|
| 未取得医疗广告审查证明 | 🔴 致命 | 投放前必备 |
| 效果承诺 | 🔴 致命 | 用"改善""提升"代替 |
| Before/After 过度 | 🔴 高 | 标注"效果因人而异" |
| 医生资质虚假 | 🔴 致命 | 真实展示医师证书 |
| 价格欺诈 | 🔴 高 | 真实原价记录 |
| 暗示替代正规治疗 | 🔴 高 | 明确"医美非医疗" |
