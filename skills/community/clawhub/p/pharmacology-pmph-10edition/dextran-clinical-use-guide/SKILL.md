# dextran-clinical-use-guide

## 适用场景
- 患者因大量失血、烧伤或创伤导致血容量下降
- 存在微循环障碍（如心脑血管血栓高风险）
- 需预防休克后期弥散性血管内凝血（DIC）

**前提条件**：无出血倾向、凝血功能正常

**禁忌证**：血小板减少症、出血性疾病、纤维蛋白原低下；心衰、肺水肿、肾功能不全者慎用

## 执行步骤

1. **识别休克类型与治疗目标**
   - 若仅为扩容维持血压 → 优先考虑右旋糖酐70
   - 若需改善微循环并抗血栓 → 选用右旋糖酐40或10

2. **按分子量分类选择**
   - **右旋糖酐70（中分子，~70kDa）**：扩容作用强，维持约12小时，适用于单纯低血容量性休克
   - **右旋糖酐40（低分子，~40kDa）**：兼具扩容与抗血栓作用，维持约3小时
   - **右旋糖酐10（小分子，~10kDa）**：显著改善微循环、抗聚集、渗透性利尿明显，维持约3小时

3. **评估禁忌与风险**
   - 检查血小板计数、纤维蛋白原水平
   - 避免连续长期使用以防蓄积致凝血障碍

4. **实施治疗并监测**
   - 给药后观察是否出现过敏反应（发热、荨麻疹）或严重不良反应（血压下降、呼吸困难）
   - 根据临床反应调整剂量或停药

## 输出结果
根据休克类型和微循环需求，选择合适分子量右旋糖酐进行扩容或抗血栓治疗。
