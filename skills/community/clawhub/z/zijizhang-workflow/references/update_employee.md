## 更新员工

```shell
zijizhang-cli employee update_employee '<employee_id>' '<data>'
```

### 参数说明

| 参数          | 类型     | 必填 | 说明                  |
|:------------|:-------|:---|:--------------------|
| employee_id | string | ✅  | 员工id                |
| data | string | ✅  | 待更新字段 JSON 字符串；支持部分字段更新 |

data json 结构说明

| 参数   | 类型     | 必填 | 说明                                                                         |
|:-----|:-------|:---|:---------------------------------------------------------------------------|
| xm   | string | 否  | 员工姓名                                                                       |
| zjhm | string | 否  | 证件号码                                                                       |
| sjhm | string | 否  | 手机号                                                                        |
| rzrq | string | 否  | 入职日期，格式 yyyy-mm-dd 如 2020-02-03                                            |
| jbgz | float  | 否  | 基本工资(元)，必须大于或等于 0                                                          |
| zjlx | int    | 否  | 证件类型，1:居民身份证；2:中国护照；3:港澳居民往来内地通行证；4:刚好居民居住证；5:台湾居民来往大陆通行证；6:台湾居民居住证；7:外国护照 |
| rzlx | int    | 否  | 任职受雇从业类型 1:雇员 2:劳务兼职 3:实习学生（全日制学历教育）                                       |
| gjdq | int    | 否  | 国籍， 1:中国                                                                   |
| xb   | int    | 否  | 性别，1:男，2:女                                                                 |
| csrq | string | 否  | 出生日期，格式 yyyy-mm-dd 如 2020-02-03                                            |

### 使用例子

#### 更新全部资料

```shell
zijizhang-cli employee update_employee 'xxxxxxx' '{"xm": "夏天", "zjhm":"888888888888888888","sjhm":"18888888888", "rzrq": "2026-01-01", "jbgz": 0}'
```

#### 只更新员工证件号码

```shell
zijizhang-cli employee update_employee 'xxxxxxx' '{"zjhm":"888888888888888888"}'
```

### 返回参数

| 参数   | 类型     | 说明                      |
|:-----|:-------|:------------------------|
| code | int    | 状态码，`200` 代表成功，其他代表失败   |
| msg  | string | 提示信息，失败时为异常信息           |

### 返回例子

#### 更新成功

```json
{
    "code": 200,
    "msg": ""
}
```

### 更新失败，身份证不正确

```json
{
    "msg": "请输入正确的身份证号",
    "code": 500
}

```
