## 十大核心轴

### 轴1：沟通轴

**定义**：伴侣之间信息传递、理解接收、表达回应的模式与质量

**核心节点**：
```
沟通轴
├── 表达层
│   ├── 自我表达（需求、感受、想法）
│   └── 接收反馈
├── 理解层
│   ├── 积极倾听
│   └── 意图解读
├── 回应层
│   ├── 情绪调节
│   └── 行为回应
└── 模式层
    ├── 沟通频率
    ├── 沟通深度
    └── 沟通风格
```

**常见卡点**：
| 节点 | 卡点类型 | 表现 |
|-----|---------|-----|
| 表达层 | 表达障碍 | 无法说清需求、习惯压抑 |
| 理解层 | 倾听缺失 | 打断、防卫、选择性倾听 |
| 回应层 | 反应过度 | 情绪化、攻击、防御 |
| 模式层 | 沟通固化 | 固定模式、无效循环 |

**常见卡点**：

## 五大关键链路

### 链路一：能量链路（已有）

```
夫妻关系质量 → 情绪基底 → 决策质量
                    ↓
             自我价值感 → 外部竞争力
                    ↓
             能量恢复能力 → 持续输出能力
                    ↓
             家庭氛围 → 子女模型复制
                    ↓
             生理系统 → 健康状态
```

### 链路二：冲突升级链路（已有）

```
触发事件 → 表层反应 → 情绪升级 → 人身攻击 → 旧伤复发
                                                         ↓
                                            关系创伤累积
                                                         ↓
                                            模式固化/关系崩溃
```

### 链路三：性-亲-情链路 ✦ 新增

```
性需求表达 → 性行为互动 → 性满意度 → 情感连接
                                           ↓
                                     关系安全感
                                           ↓
                                     亲密表达意愿
                                           ↓
                                  更高性满意度 ←
```

### 链路四：代际传递链路 ✦ 新增

```
原生家庭经历 → 依恋风格 → 关系期望 → 伴侣选择 → 婚姻互动
                                                              ↓
                                              重复/修复/创造新模式
                                                              ↓
                                                     传递给下一代
```

### 链路五：系统震荡链路 ✦ 新增

```
外部压力事件 → 内部系统失衡 → 夫妻关系紧张
                                     ↓
                           原生家庭/子女/工作介入
                                     ↓
                              边界冲突爆发
                                     ↓
                           夫妻联盟瓦解/稳固
```
