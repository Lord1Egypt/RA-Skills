## Description: <br>
Macos Toolkit is a Bash command-line utility that records local activity notes for run, check, convert, analyze, generate, preview, batch, compare, export, config, status, and report workflows. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
Developers and technical users can use this skill to record local command-line workflow notes, inspect recent entries, search activity history, and summarize locally retained logs. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: User-entered notes are retained locally and may later appear in search results or exports. <br>
Mitigation: Do not enter passwords, API tokens, private file contents, or sensitive system details; review and delete retained logs in ~/.local/share/macos-toolkit when no longer needed. <br>
