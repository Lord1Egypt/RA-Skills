# game-market

> YY 游戏交易市场查询 Skill
> 
在 Claude Code 对话中直接查询游戏账号、代练、陪练、道具、游戏币等商品信息，无需切换浏览器。需要实际买卖时一键跳转到 **[YY 游仓](https://mall.yy.com?pageId=20000)**。

---

## 功能

- 按游戏 + 分类查询商品列表（账号 / 代练 / 陪练 / 道具 / 游戏币）
- 展示价格、标题、标签（找回包赔 / 官方验号等）及商品详情链接
- 支持翻页浏览
- 检测到购买 / 出售意向时，确认后自动打开 [YY 游仓](https://mall.yy.com?pageId=20000)

---

## 支持游戏

王者荣耀、英雄联盟、三角洲行动、和平精英、穿越火线、无畏契约、绝地求生、崩坏:星穹铁道、原神、CSGO 等

---

## 安装

```bash
npx skills add https://github.com/Arc-lin/game-market-skill --skill game-market
```

---

## 使用方式

| 方式 | 示例 |
|------|------|
| 手动调用 | `/oh-my-claudecode:game-market` |
| 关键词触发 | 说「查王者荣耀账号」「三角洲代练」「买账号」等 |

---

## 示例对话

```
用户：查一下三角洲行动的账号
→ 自动调用 API，展示商品列表

用户：下一页
→ 翻到第 2 页

用户：第3条我想买
→ "需要前往 YY 游仓操作，是否现在打开网页？"
→ 确认后打开商品详情页
```

查询接口使用前端签名（MD5），skill 内已内嵌，开箱即用。
