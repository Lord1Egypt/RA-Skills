# 远端肾小管性酸中毒的氯化铵负荷试验诊断

## 适用对象
- 疑似远端肾小管性酸中毒（Ⅰ型RTA）患者
- 无全身性酸中毒表现的不典型或不完全RTA患者

## 前提条件
- 患者已停用所有酸、碱类药物
- 已排除全身性酸中毒
- 已完成氯化铵负荷试验（短程法或长程法）并收集至少5份尿液样本

## 禁忌与限制
- 禁用于已有明显全身性酸中毒的患者（以免加重病情）
- 禁用于严重肝肾功能衰竭或对氯化铵过敏/禁忌者

## 执行步骤

1. **试验前准备**：受试者饮食不限，但必须停用所有酸、碱类药物；服药前排空膀胱并收集基础尿液样本。

2. **给药方式**：
   - **短程法（单剂法）**：成人按每千克体重0.1 g NH₄Cl一次性口服；
   - **长程法（Elkinson法）**：停用碱性药物2天后，按0.1 g/(kg·d)剂量分3次口服，连续3天。

3. **尿液采集时间点**：
   - **短程法**：服药后第3、4、5、6、7、8小时各留取一次尿液（含服药前1次，共至少5份有效样本）；
   - **长程法**：第3日末次服药后第3、4、5、6小时留取4次尿液，加上服药前样本，共5份。

4. **检测指标**：测定所有尿液样本的pH值。

5. **结果判读**：
   - **正常反应**：至少1次尿pH < 5.5，表明远端肾小管酸化功能正常；
   - **异常反应（诊断标准）**：5次尿样pH均 > 5.5，提示远端肾小管无法在酸负荷下分泌H⁺，符合远端RTA（Ⅰ型）诊断；
   - 典型患者尿pH常维持在6.0～7.0，呈现“血液与尿液pH分离现象”。

6. **注意事项**：仅适用于无全身酸中毒者；若已存在代谢性酸中毒，禁止进行本试验。
