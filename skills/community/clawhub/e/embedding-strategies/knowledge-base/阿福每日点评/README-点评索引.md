## 📈 使用方式

### 新增点评
```
1. 在 knowledge-base/阿福每日点评/ 创建文件
2. 命名：YYYY-MM-DD-主题.md
3. 使用模板：_模板/每日点评-template.md
4. 更新本索引文件
```

### 查找点评
```
1. 按日期查找 → knowledge-base/阿福每日点评/YYYY-MM/
2. 按主题搜索 → 搜索文件名或内容
3. 按领域筛选 → 查看文件标签
```

### 回顾复盘
```
1. 每周回顾 - 周日晚查看本周点评
2. 每月复盘 - 月末查看本月点评
3. 季度总结 - 提取关键洞察到 MEMORY.md
```
