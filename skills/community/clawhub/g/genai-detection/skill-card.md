## Description: <br>
Detect whether an image was generated by AI using the Scam.ai API, guiding the user through API key setup when needed and then analyzing the image. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[yuchennnnnnn](https://clawhub.ai/user/yuchennnnnnn) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and developers use this skill to submit an image file for AI-generation detection through Scam.ai and receive a concise verdict with confidence. It is suited for checking common image formats when third-party image processing is acceptable. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill stores a Scam.ai API key locally. <br>
Mitigation: Use a dedicated, revocable API key and keep the local key file protected as a secret. <br>
Risk: Images submitted through the skill are sent to Scam.ai for third-party processing. <br>
Mitigation: Do not upload sensitive, private, or regulated images unless third-party processing is acceptable. <br>
