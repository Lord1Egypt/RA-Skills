## 一、三大诈骗类型详解

### 类型一：伪造/假冒本票诈骗

#### 五大特征
| 序号 | 特征 | 说明 |
|------|------|------|
| 1 | 出票人签名与印鉴不符 | 有模仿迹象，笔迹生硬 |
| 2 | 票据要素不全 | 内容错漏百出 |
| 3 | 纸质较薄 | 油墨较淡，印刷粗糙 |
| 4 | 填制方法不合常规 | 格式不规范 |
| 5 | 伪冒银行多为中银集团成员 | 或密切相关银行 |

#### 识别要点
- **签字检查**：仔细比对签字与印鉴的一致性
- **要素核对**：检查六项要素是否完整规范
- **纸张检查**：手感厚度、印刷质量
- **银行核实**：向相关银行核实票据真伪

#### 防范措施
1. 仔细核对印鉴、密押
2. 与来电信函交叉验证
3. 委托银行对外查实

---

### 类型二：涂改/传真本票诈骗

#### 五大特征
| 序号 | 特征 | 说明 |
|------|------|------|
| 1 | 出票行无法查到 | 国际著名银行或虚构银行 |
| 2 | 签名权限不符 | 与印鉴册规定不符 |
| 3 | 涂改痕迹明显 | 或以传真件出现 |
| 4 | 内容含糊不清 | 票据内容欠缺 |
| 5 | 金额巨大 | 诈取暴利 |

#### 识别要点
- **真实性核查**：核实出票行是否真实存在
- **原件检查**：确认为原件而非传真件
- **涂改痕迹**：检查是否有刮擦、消字液痕迹
- **文义核对**：内容是否清晰准确

#### 防范措施
1. 查询银行年鉴确认银行存在
2. 不接受传真件作为凭证
3. 仔细检查票面是否有涂改

---

### 类型三：假证实/担保/保兑本票诈骗

#### 五大特征
| 序号 | 特征 | 说明 |
|------|------|------|
| 1 | 出票行伪造或假冒 | 虚构银行或冒用银行名义 |
| 2 | 附随担保书/保兑函 | 加发银行证实电 |
| 3 | 担保行多为国际大银行 | 增加欺骗性 |
| 4 | 金额大、关系复杂 | 迷惑性强 |
| 5 | 打着贸易或投资幌子 | 以合法外衣掩盖 |

#### 识别要点
- **担保函核实**：核实担保函的真实性
- **大银行警惕**：对"大银行担保"保持警惕
- **交易背景审查**：审查背后的交易是否真实
- **核实机制**：通过官方渠道核实

#### 防范措施
1. 对担保函进行独立核实
2. 不轻信大银行担保
3. 审查完整的交易链条

## 二、九大高危信号清单

| 序号 | 高危信号 | 风险等级 | 详细说明 |
|------|----------|----------|----------|
| 1 | **超大金额** | 🔴极高 | 单张票据金额超过1亿美元/欧元，远超正常贸易规模 |
| 2 | **超高折扣** | 🔴极高 | 要求以50%-93%折扣购买，不可能的市场优惠 |
| 3 | **先付后交割** | 🔴极高 | 要求先付款，几天后才交票，典型诈骗模式 |
| 4 | **术语错误** | 🔴极高 | SWIFT拼写错误、PGL等伪造术语 |
| 5 | **复杂佣金链** | 🟠高 | 多层分赃、佣金比例异常（>10%） |
| 6 | **非银行开证** | 🟠高 | 非银行金融机构开证，无票据资质 |
| 7 | **缺乏法律管辖** | 🟠高 | 无法明确适用法律或管辖法院 |
| 8 | **紧迫感/保密性** | 🟠高 | 催促决策、限时优惠、要求绝对保密 |
| 9 | **无正规渠道** | 🟠高 | 无法通过正规银行验证 |

### 高危信号快速检查

| 检查项 | 是 | 否 |
|--------|----|----|
| 票据金额≥1亿美元/欧元 | □ | □ |
| 折扣≥50%（折价≤50%） | □ | □ |
| 要求先付款后交割 | □ | □ |
| 有术语错误（SWIFT拼写错误等） | □ | □ |
| 佣金≥10%或多层分赃 | □ | □ |
| 非银行机构开证 | □ | □ |
| 无明确法律管辖 | □ | □ |
| 催促决策/要求保密 | □ | □ |
| 无法通过银行验证 | □ | □ |

> ⚠️ **任一勾选"是"即需高度警惕！**

## 三、术语错误识别表

| 错误术语 | 正确术语 | 识别说明 |
|----------|----------|----------|
| **SWFIT** | **SWIFT** | 专业金融机构绝不会犯此拼写错误 |
| **PGL** | 无此标准术语 | 金融领域无此术语，属伪造 |
| **银行安全电子邮件** | 无此标准通讯方式 | 银行间通讯使用SWIFT MT系列报文 |
| **银行官员亲自交票** | 无此操作 | 正规银行从不派人送票上门 |
| **MT700国际信用证格式** | 本票不使用此格式 | 混淆不同票据类型 |
| **银行保函买断** | SBLC/备用信用证 | 术语混淆 |
| **循环出证买断证** | 非标准术语 | 可能是虚构操作 |

### 术语正确性快速检查

| 术语检查项 | 是否正确 |
|------------|----------|
| SWIFT拼写正确 | □ 是 □ 否 |
| 无PGL等伪造术语 | □ 是 □ 否 |
| 无"银行安全邮件" | □ 是 □ 否 |
| 无"银行官员送票" | □ 是 □ 否 |
| 票据格式规范 | □ 是 □ 否 |

## 四、佣金结构异常识别

### 正常佣金参考

| 业务类型 | 正常区间 | 说明 |
|----------|----------|------|
| 即期本票贴现 | 1-2% | 银行正常收费 |
| 远期本票 | 1-2% | 略高于即期 |
| SBLC开证 | 1-3% | 银行正常收费 |
| 中介服务费 | 0.5-1% | 合理居间费用 |

### 异常佣金识别

| 佣金区间 | 风险等级 | 建议 |
|----------|----------|------|
| 3-5% | 🟡警示 | 需核实业务真实性 |
| 5-10% | 🟠警告 | 大概率存在问题 |
| ≥10% | 🔴危险 | 明确诈骗信号 |

### 金字塔分赃结构

**正常结构**：
```
A（本票持有人）→ B（单层中介）→ 收取 1-2%
```

**金字塔分赃（诈骗特征）**：
```
A（本票持有人）→ 声称持有多张本票
        ↓
B（中间人1）→ 提取 3-5%
        ↓
C（中间人2）→ 提取 5%
        ↓
D（中间人3）→ 提取 10%
        ↓
E（中间人4）→ 提取 15%
        ↓
F（中间人5）→ 提取 15%+
        ↓
总计佣金：50%+（明确诈骗）
```

## 五、真实案例：5亿欧元本票诈骗案完整分析

### 案例概述

某客户收到一份"国际本票流程"文件，声称持有5亿欧元本票，愿意以93%折扣（即7折）出售。

### 诈骗特征分析

| 特征项 | 文档描述 | 风险分析 |
|--------|----------|----------|
| 票据金额 | 5亿欧元/美元 | 🔴 超大金额，远超正常贸易规模 |
| 资金证明 | >5000亿欧元/美元 | 🔴 荒谬要求，诈骗话术 |
| 折扣比例 | 93%折扣（7折购买） | 🔴 超高折扣，不可能的市场优惠 |
| 付款流程 | 先付93%资金，3天后给票 | 🔴 先付后交割，典型诈骗模式 |
| 即期本票佣金 | 3% | 🟠 偏高，正常应为1-2% |
| 远期本票佣金 | 50%+1%+1%+1% | 🔴 金字塔分赃，明确诈骗信号 |
| 术语错误 | SWFIT（应为SWIFT） | 🔴 专业机构不会犯此错误 |
| 伪造术语 | PGL（无此术语） | 🔴 虚构术语，欺骗性内容 |

### 作案手法还原

```
┌─────────────────────────────────────────────────────────────┐
│                   5亿欧元本票诈骗流程                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  阶段一：建立信任                                            │
│  ┌────────────────────────────────────────────────────┐   │
│  │ 1. 通过中间人接触潜在受害者                          │   │
│  │ 2. 展示"巨额"本票照片（通常是伪造的）                 │   │
│  │ 3. 提供虚假的资金证明和银行文件                       │   │
│  └────────────────────────────────────────────────────┘   │
│                           ↓                                  │
│  阶段二：诱导入局                                            │
│  ┌────────────────────────────────────────────────────┐   │
│  │ 1. 以超高折扣（93折）吸引受害者                       │   │
│  │ 2. 强调"机会难得"、"限时优惠"                        │   │
│  │ 3. 要求保密，制造紧迫感                              │   │
│  └────────────────────────────────────────────────────┘   │
│                           ↓                                  │
│  阶段三：骗取资金                                            │
│  ┌────────────────────────────────────────────────────┐   │
│  │ 1. 要求先支付"诚意金"或"佣金"（通常是3-5%）          │   │
│  │ 2. 收到款项后消失或继续索要更多"费用"                 │   │
│  │ 3. 受害者永远拿不到本票或拿到假票                     │   │
│  └────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 关键教训

| 教训 | 说明 |
|------|------|
| **正规本票不打折** | 本票是支付工具，不是投资产品，银行不会打折出售 |
| **银行不通过中间人** | 正规业务通过银行网点或企业账户操作，不会有中间人兜售 |
| **术语暴露本质** | SWIFT拼写错误等专业疏漏暴露诈骗本质 |
| **超高折扣是陷阱** | 任何超过市场规律的"优惠"都是陷阱 |

## 六、快速识别口诀

> **一看二查三核四不**
>
> **一看**：看票据要素是否完整、格式是否规范
>
> **二查**：查术语是否正确、佣金是否正常
>
> **三核**：核银行是否真实、渠道是否正规
>
> **四不**：不先付款、不信折扣、不急决策、不贪高回报
