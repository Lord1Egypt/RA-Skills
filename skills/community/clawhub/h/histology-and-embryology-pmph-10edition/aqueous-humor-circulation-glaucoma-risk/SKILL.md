# 房水循环与青光眼风险评估

## 适用场景
- 解释正常眼压（10–21 mmHg）的维持机制
- 分析青光眼发病的解剖与生理基础
- 区分开角型与闭角型青光眼的病因差异
- 评估房水流出通道阻塞对视力的潜在威胁

## 执行步骤
1. **确认房水生成**：由睫状体非色素上皮细胞主动分泌及血液超滤形成。
2. **追踪房水流向**（按顺序）：
   - 起始于后房（虹膜与晶状体之间）
   - 经瞳孔流入前房（角膜与虹膜之间）
   - 到达前房角（角膜缘内侧）
   - 穿过小梁网（胶原纤维核心+内皮覆盖）
   - 进入巩膜静脉窦（环形静脉通道）
   - 汇入睫状前静脉，回流至全身循环
3. **验证动态平衡**：房水生成速率 ≈ 回流速率，维持正常眼压。
4. **识别阻塞风险点**：
   - 小梁网结构或功能障碍 → 开角型青光眼
   - 瞳孔阻滞致房角关闭 → 闭角型青光眼
5. **预警青光眼后果**：若任一环节阻塞 → 房水积聚 → 眼压升高 → 压迫视神经乳头 → 视野缺损甚至失明。
6. **排除无关系统**：本技能仅适用于房水循环，不涉及玻璃体或泪液系统。

> **行动提示**：一旦发现房水回流路径受阻迹象，立即建议眼压监测与青光眼筛查。
