# 六爻断卦方法与规则

## 断卦总纲

六爻断卦按"先看用神，次查五行，再审六兽，后详动变"的顺序进行。

## 一、用神旺衰判断

用神的旺衰是全卦断吉凶的根本标准：

### 得生得扶为旺（吉）
- 用神得月建生扶 → 旺
- 用神得日辰生扶 → 旺
- 用神得动爻生 → 旺
- 用神本身发动 → 旺（但看变爻如何）
- 临月建或日辰 → 最旺

### 被克被制为衰（凶）
- 用神被月建克制 → 衰
- 用神被日辰克制 → 衰
- 用神被动爻克制 → 衰
- 用神入旬空 → 衰弱
- 用神入墓库 → 衰弱

### 特殊状态
- **旬空**：用神落入当日旬空的地支，能量暂失，出空后可恢复
- **入墓**：用神在日辰或月建中入墓库，被限制能量
- **伏吟**：用神与变爻地支相同，事情停滞反复
- **反吟**：用神与变爻地支相冲，事情反复无常

## 二、各事项断法细则

### 感情/婚姻
**核心爻位**：妻财爻（男）/ 官鬼爻（女）+ 世爻 + 应爻

- 用神旺相 + 用神生世爻 → 感情顺利，对方有心
- 用神衰弱 + 用神克世爻 → 感情不顺，对方无意或有利害关系
- 兄弟爻持世 → 竞争激烈，易有第三者
- 应爻与世爻相合 → 两人有缘
- 应爻与世爻相冲 → 两人性格不合
- 妻财爻临青龙 → 女方条件好、貌美
- 官鬼爻临青龙 → 男方条件好、有官职

### 事业/求职
**核心爻位**：官鬼爻 + 父母爻 + 世爻

- 官鬼旺相 + 生世爻 → 事业顺利，有晋升机会
- 官鬼衰弱 + 克世爻 → 工作压力大，职位不稳
- 官鬼临青龙 → 有贵人提携
- 父母爻旺 → 单位/公司有实力
- 妻财爻旺 → 待遇好
- 子孙爻动 → 有下属/学生，但也克制官鬼（影响升迁）
- 官鬼入旬空 → 暂时无职或缺位

### 财运/财富
**核心爻位**：妻财爻 + 子孙爻

- 妻财旺 + 得生 → 财运好
- 妻财衰弱 + 受克 → 财运差，谨防破财
- 子孙爻旺 → 财有来源（子孙生财）
- 兄弟爻旺而动 → 破财、竞争消耗（兄弟克妻财）
- 妻财临青龙 → 正财可期
- 兄弟临玄武 → 谨防被骗

### 健康/疾病
**核心爻位**：官鬼爻（病灶）+ 子孙爻（医药）

- 官鬼弱 + 子孙旺 → 病可痊愈
- 官鬼旺 + 子孙弱 → 病难痊愈
- 官鬼克世爻 → 疾病侵袭自身
- 官鬼在何五行 → 对应何脏腑有问题（参考五行与身体对应）
- 官鬼临白虎 → 病情严重（尤其金爻+白虎）
- 子孙爻临青龙 → 良医良药

## 三、动爻的特殊作用

### 动爻对用神的直接作用
- 动爻生动 → 事情有推动力（吉）
- 动爻克动 → 事情有阻碍（凶）
- 动爻合用神 → 事情被绑定/有人相助
- 动爻冲用神 → 突发变故

### 动爻化出（变爻）
每个动爻都化出一个变爻，判断动爻对最终结果的影响：
- 动爻化回头生 → 力量增强（如寅木动化子水回头生）
- 动爻化回头克 → 力量被削弱甚至失效（如寅木动化申金回头克）
- 动爻化进神 → 力量增强（如寅化卯）
- 动爻化退神 → 力量衰退（如卯化寅）
- 动爻化空 → 动而无力

## 四、常用吉凶断法速记

### 定吉兆的情况
1. 用神旺相得日月生扶
2. 用神动而生世
3. 用神持世（用神恰好在世爻）
4. 忌神（克制用神的爻）空亡受制
5. 元神（生用神的爻）动而生用神

### 定凶兆的情况
1. 用神衰弱被克
2. 用神被日月冲破
3. 忌神动而克用神
4. 用神动化回头克
5. 用神入墓、旬空长期不出

## 五、六兽辅助判断

用于补充细节信息（但不可取代五行分析）：

| 六兽 | 在感情中 | 在事业中 | 在财运中 | 在健康中 |
|------|----------|----------|----------|----------|
| 青龙 | 正缘、喜酒 | 贵人提携 | 正财可靠 | 良医 |
| 朱雀 | 口舌是非 | 文书通知 | - | 炎症 |
| 勾陈 | 拖延 | 土建相关 | 土地房产 | 慢性病 |
| 螣蛇 | 纠缠 | 阴谋诡计 | 假账 | 虚惊 |
| 白虎 | 暴力倾向 | 武职/管理者 | 破大财 | 急症/开刀 |
| 玄武 | 暗昧关系 | 暗中操作 | 被骗 | 隐疾 |

## 六、时间应期判断

用户常问"什么时候能成/能好"：
- 用神空亡 → 出空之日（走出旬空范围时）
- 用神入墓 → 冲墓之日（被冲开时）
- 动爻克用 → 冲动爻/冲开制神之日
- 用神被合 → 冲开合神之日
- 用神不入空 → 用神逢生之日
