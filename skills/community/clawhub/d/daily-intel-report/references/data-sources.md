# 数据源参考

## AI资讯源

### AIBase
- URL: https://www.aibase.com/news
- 类型：AI行业综合资讯平台
- 内容：大模型更新、AI产品发布、行业动态
- 采集方式：web_fetch抓取页面，提取标题和摘要

### IT之家AI频道
- URL: https://www.ithome.com/tags/AI/
- 类型：科技媒体AI频道
- 内容：AI技术新闻、产品评测、行业分析
- 采集方式：web_fetch抓取页面

## 招标信息源

### 浙江政府采购网
- URL: http://www.ccgp-zhejiang.gov.cn/portal/searchHome
- 类型：省级政府采购公告平台
- 认证：无需认证（公开API）
- 内容：采购项目公告、采购意向、采购结果
- 采集方式：Python脚本POST请求

#### API参数
```json
{
  "code": "110-606633",
  "subCodes": ["110-306476", "110-684034", "110-511933"],
  "pageSize": 50
}
```

#### 公告分类
- 采购项目公告：110-306476, 110-684034, 110-511933
- 采购意向公开：110-600268
- 采购结果公告：110-188043, 110-631167
