# 剧本 · 代理工作手册

> 通用规则见 `../common-agent-rules.md`。

## 定位

剧本是最终的拍摄执行文档。将逐集概要展开为完整剧本（含分镜表格），导演、摄影师、演员拿到后可以直接执行。

逐集概要中每集约500字，剧本需要在此基础上充分展开细节、补充对话和动作。剧本完成后，逐场次编写分镜表格。

---

## 任务

- **输入**：逐集概要（所属批次）+ 世界观与运行规则 + 人物小传
- **输出**：剧本/EP_{X}.md（含完整剧本 + 分镜表格）
- **目标**：将逐集概要展开为可拍摄的完整剧本，并为每个场次编写分镜

---

## 流程

### 1. 撰写前提回顾

EP1 无前提回顾。EP2 起，在头部信息后撰写前提回顾：
- 用1-2句话简述上一集结尾状态和关键事件，让观众/读者快速衔接上下文
- 字数不超过100字
- 只回顾上一集，不做更早的追溯
- 回顾内容必须与上一集剧本的实际内容一致

### 2. 撰写剧本

将逐集概要展开为可拍摄的完整剧本。

1. 拆分场次（地点、时间、出场人物）
2. 撰写场景描述和对话
3. 标注音效/配乐/特效
4. 估算时长
5. 自查一致性

写作约束：

- **字数限制**：每集剧本正文不超过2500字
- **场景描述**：写画面，不写心理，白描为主，短句断行。竖屏画幅横向空间有限，场景描述聚焦人物动作和细节，环境信息用台词带出
- **台词**：台词承担双重功能——推进戏剧冲突，同时补偿画面无法交代的信息（背景、身份、立场、空间位置、人物关系）。画面拍不出来的用台词说，画面拍得出来的台词不重复。信息性台词必须裹在冲突或情绪里，禁止生硬解说式独白
- **人物名**：加粗，括号标注情绪/动作
- **画外音/旁白**：加粗人物名后标注（O.S.）或（VO）
- **音效/特效**：`*[音效：xxx]*` 格式
- **场景切换**：`---` 分隔
- **时长估算**：台词约3-4字/秒，动作按实际表演时间估算，每集总时长在目标时长±10%以内
- **驱动性**：每个场景至少承担一项功能——推进主线 / 深化人物 / 制造冲突，不存在纯过渡的废戏
- **钩子**：集末必须有钩子驱动下一集

### 3. 编写分镜

剧本写完后，逐场次拆分镜头序列。分镜必须完整覆盖剧本中的全部场景、台词、旁白和关键动作，不能遗漏。

1. 从逐集概要头部提取画幅信息（竖屏/横屏），据此选择景别和运镜
2. 选择景别、运镜，撰写画面描述
3. 标注台词/旁白引用、音效/配乐
4. 估算每镜头时长
5. 自查完整覆盖

#### 景别

| 景别 | 用途 |
|------|------|
| 全景 | 交代环境、人物关系、场面调度 |
| 中景 | 日常对话、动作、人际互动 |
| 近景 | 表情、情绪传递、重要对话 |
| 特写 | 关键道具、情绪爆发、悬念细节 |
| 大特写 | 极少用，只用于最重要视觉符号 |

景别原则：
- 有节奏变化，不连续多镜头同一景别；景别渐进衔接（全景→中景→近景→特写），跳跃不超过两档
- 单场戏搭配2-3种景别制造层次感
- 关键节点用特殊景别放大情绪：爽点/反转/情感高光→特写或大特写；压抑/孤独→远景或全景
- **竖屏**纵向画幅天然贴合人体比例，中景/近景/特写表现力强；全景横向信息被裁切，建议用于纵向纵深场景（走廊、楼梯、高楼）

#### 运镜

| 运镜 | 用途 | 适用场景 |
|------|------|----------|
| 固定 | 稳定、冷静、日常 | 台词戏、情绪平稳、铺垫段落 |
| 推 | 聚焦、压迫、情绪升级 | 爽点落地、情绪高光、揭晓关键信息 |
| 拉 | 揭示全貌、远离、孤独感 | 结局留白、人物孤影、展示全貌 |
| 摇（Pan） | 水平摇：跟随目光、展示空间 | 跟随动作、揭示新信息 |
| 摇（Tilt） | 垂直摇：展示高度、人物全貌 | 人物站起/倒下、展示建筑纵深 |
| 甩摇（Whip Pan） | 快速甩动产生模糊过渡 | 场景切换、时间跳转、情绪急转 |
| 移（Track/Dolly） | 跟随人物运动、展示环境 | 走路跟随、场景巡视 |
| 跟 | 沉浸感、紧迫感 | 追逐、逃亡、人物移动 |
| 环绕（Arc） | 以主体为圆心环绕 | 展示人物气场、环境与主体的空间关系 |
| 升降（Crane/Pedestal） | 气势、揭示、情绪转换 | 强者登场、全貌揭示、俯瞰全局 |
| 变焦（Zoom） | 不动机位放大/缩小画面 | 快速聚焦细节、制造戏剧感 |
| 急推变焦（Crash Zoom） | 瞬间推近产生强烈冲击 | 震惊、发现关键线索、角色觉醒 |
| 滑动变焦（Dolly Zoom） | 主体不变但背景拉伸/压缩 | 不安、眩晕、现实崩塌的瞬间 |
| 移焦（Rack Focus） | 焦点从A切换到B | 引导注意力转移、揭示隐藏信息 |
| 手持 | 真实感、紧张感、混乱感 | 冲突爆发、混乱场面、主观视角 |
| 升格（Slow Motion） | 放慢时间、屏息感 | 情感高光、关键动作、视觉奇观 |

运镜原则：
- 服务于情绪，不炫技；紧张=快切+手持，温馨=慢推+固定
- 内容优先，运镜为辅——每个运镜都有明确目的，不为了动而动
- 高光时刻运镜放慢，让人物和剧情成为焦点
- 运镜时长控制在1-3秒，跟镜头和环绕不超过3秒
- **竖屏**纵向空间充裕，上下摇（Tilt）、升降、纵向推拉表现力强；横向空间有限，水平摇（Pan）和横向平移建议缩短时长或减少幅度

### 4. 保存文件

按输出格式将剧本和分镜整合为完整文件（含附录），保存到指定路径。

## 自检

生成完成后，逐条检查以下致命问题。**命中任何一条必须立即修改，全部通过后才能交付。**

### 1. 场景描述不是画面

- 场景描述中出现大量心理活动、情感分析、文学修辞、背景知识交代。"她内心充满了矛盾""这个决定改变了一切""命运的齿轮开始转动"——全是文学语言，摄像机拍不出来。

### 2. 台词不符合人物性格

- 某句台词的语气、用词、表达方式和该人物在人物小传中的性格设定完全矛盾。不看角色名，完全猜不到是谁说的。

### 3. 存在废戏

- 某个场景同时不满足：推进主线、深化人物、制造冲突。删掉它，前后衔接完全正常，观众不会觉得少了什么。

### 4. 节奏感缺失

- 场景之间的切换没有节奏变化——连续多个场景都是同一情绪强度、同一节奏类型。连续5个安静对话，或连续5个激烈冲突，中间没有节奏转换。

### 5. 集末没有强钩子

- 一集的最后一个场次没有制造悬念或留下驱动下一集的钩子。故事在某个自然段落结束，观众看完没有"然后呢？"的冲动。

### 6. 分镜遗漏剧本内容

- 剧本中的某个场景、台词、旁白、关键动作在分镜表格中找不到对应条目。分镜的镜头序列没有覆盖剧本的全部内容。

### 7. 分镜画面不可执行

- 分镜表格的"画面"列写的是模糊描述而非具体可执行的画面。"气氛紧张""主角做出反应""场景发生变化"——没有具体的视觉信息。摄影师拿到不知道怎么拍。

### 8. 时长严重偏离

- 剧本预估总时长超出目标时长±10%的范围。目标120秒，实际80秒（内容不够）或170秒（内容塞不下）。
