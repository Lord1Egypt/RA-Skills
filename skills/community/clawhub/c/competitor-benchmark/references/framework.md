## 一、波特五力模型

**用途**：分析行业竞争强度和吸引力

**五力定义**：

| 五力 | 定义 | 评估要点 |
|-----|------|---------|
| 现有竞争者 | 市场中直接与你竞争的企业 | 数量/规模/差异化程度 |
| 潜在进入者 | 可能进入市场的新玩家 | 进入壁垒/品牌优势/资金需求 |
| 替代品 | 能满足同样需求的其他方案 | 替代程度/转换成本 |
| 供应商议价能力 | 关键供应商的议价能力 | 集中度/替代性/整合威胁 |
| 买家议价能力 | 客户的议价能力 | 数量/价格敏感度/信息获取 |

**评分标准**：

| 强度 | 分值 | 含义 |
|-----|------|------|
| 极强 | 5分 | 对你构成重大威胁 |
| 强 | 4分 | 需要认真应对 |
| 中等 | 3分 | 一般性竞争 |
| 弱 | 2分 | 影响有限 |
| 极弱 | 1分 | 几乎无影响 |

## 二、SWOT分析框架

**用途**：系统评估内外部环境

```
┌─────────────────────────────────────────────────────────────┐
│  SWOT分析矩阵                                             │
├─────────────────────────┬─────────────────────────────────┤
│                         │                                 │
│  【S】Strengths         │  【W】Weaknesses                │
│  内部·优势              │  内部·劣势                       │
│                         │                                 │
│  • 独特资源/能力        │  • 资源缺口/能力短板             │
│  • 竞争优势            │  • 竞争劣势                      │
│  • 可持续的优势        │  • 需要改进的地方               │
│                         │                                 │
├─────────────────────────┼─────────────────────────────────┤
│                         │                                 │
│  【O】Opportunities     │  【T】Threats                   │
│  外部·机会              │  外部·威胁                       │
│                         │                                 │
│  • 市场机会             │  • 市场威胁                      │
│  • 趋势性机会           │  • 竞争威胁                      │
│  • 政策/技术机会        │  • 风险因素                      │
│                         │                                 │
└─────────────────────────┴─────────────────────────────────┘
```

**策略组合**：

| 组合 | 策略 | 行动 |
|-----|------|------|
| S+O | 进攻策略 | 利用优势抓住机会 |
| S+T | 防御策略 | 利用优势应对威胁 |
| W+O | 转型策略 | 改善劣势抓住机会 |
| W+T | 收缩策略 | 改善劣势应对威胁 |

## 四、护城河评估标准

**六大护城河详解**：

### 4.1 网络效应

| 评分 | 类型 | 特征 |
|-----|------|------|
| 8-10 | 强网络效应 | 双向网络，用户越多价值指数增长 |
| 5-7 | 中网络效应 | 单向网络，有一定增长曲线 |
| 0-4 | 弱网络效应 | 几乎没有网络效应 |

### 4.2 切换成本

| 评分 | 类型 | 特征 |
|-----|------|------|
| 8-10 | 极高切换成本 | 数据/习惯/学习成本极高 |
| 5-7 | 中切换成本 | 需要时间和精力 |
| 0-4 | 低切换成本 | 轻松可切换 |

### 4.3 品牌护城河

| 评分 | 类型 | 特征 |
|-----|------|------|
| 8-10 | 强品牌 | 品牌=品类代名词 |
| 5-7 | 中品牌 | 有一定认知度和信任 |
| 0-4 | 弱品牌 | 品牌认知不足 |

### 4.4 数据护城河

| 评分 | 类型 | 特征 |
|-----|------|------|
| 8-10 | 强数据壁垒 | 数据难以获取且持续积累 |
| 5-7 | 中数据壁垒 | 有一定数据积累 |
| 0-4 | 弱数据壁垒 | 数据可复制 |

### 4.5 规模护城河

| 评分 | 类型 | 特征 |
|-----|------|------|
| 8-10 | 强规模优势 | 成本/采购/定价优势明显 |
| 5-7 | 中规模优势 | 有一定规模效应 |
| 0-4 | 弱规模优势 | 规模优势不明显 |

### 4.6 技术护城河

| 评分 | 类型 | 特征 |
|-----|------|------|
| 8-10 | 强技术壁垒 | 专利/算法/工程壁垒高 |
| 5-7 | 中技术壁垒 | 有一定技术积累 |
| 0-4 | 弱技术壁垒 | 技术可快速复制 |

## 五、差异化定位检查表

**差异化要素**：

| 维度 | 检查问题 | 是/否 |
|-----|---------|-------|
| 用户差异化 | 你服务的是竞品忽视的用户群体吗？ | |
| 场景差异化 | 你解决的是竞品不关注的场景吗？ | |
| 功能差异化 | 你有竞品没有的核心功能吗？ | |
| 体验差异化 | 你的用户体验有质的飞跃吗？ | |
| 价格差异化 | 你的定价策略与竞品显著不同吗？ | |
| 渠道差异化 | 你的获客渠道与竞品不同吗？ | |
| 品牌差异化 | 你的品牌定位与竞品有明显区分吗？ | |
