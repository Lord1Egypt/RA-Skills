# textbook-immunomodulatory-drug-page-index-rules

## 何时使用
- 用户提供药物名称及对应页码，且页码出现在460及以上（尤其是463–469区间）。
- 用户需要快速确认某药物是否属于“影响免疫功能的药物”章节内容。
- 用户为医学教育者、临床药师或住院医师，正在查阅该特定教材。

## 如何执行
1. 获取药物关联的所有页码。
2. 检查是否存在页码 ≥463 且 ≤469。
3. 若存在，则根据具体页码确定药物类别与作用机制：
   - **页码463**：环孢素（ciclosporin）、环孢菌素A（cyclosporin A）→ 钙调磷酸酶抑制剂，强效免疫抑制。
   - **页码465**：环磷酰胺、吗替麦考酚酯、来氟米特、芬戈莫德 → 多机制免疫抑制剂。
   - **页码467**：干扰素、卡介苗 → 免疫增强剂。
   - **页码468**：白介素-2、左旋咪唑 → 免疫刺激或调节。
   - **页码469**：异丙肌苷 → 免疫增强。
4. 注意环磷酰胺可能同时出现在抗肿瘤章节（如页码434）和免疫章节（页码465），表明其双重用途。
5. 输出提示信息，建议用户查阅对应页码以确认详细机制。

> ⚠️ 本规则仅适用于该教材所列药物，不适用于未标注页码460+的潜在免疫药物。
