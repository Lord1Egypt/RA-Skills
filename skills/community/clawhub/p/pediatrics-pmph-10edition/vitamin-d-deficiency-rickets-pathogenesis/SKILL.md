# 维生素D缺乏性佝偻病发病机制分析

## 触发条件
当满足以下全部条件时启用本技能：
- 患者为婴幼儿或处于快速生长期的儿童
- 出现颅骨软化（3–6月龄）、方颅、肋骨串珠、手足镯或下肢弯曲（O/X型腿）等骨骼畸形
- 血清25-(OH)D₃水平 <12 ng/ml（<30 nmol/L）

## 执行步骤
1. **确认维生素D缺乏**：评估是否存在长期摄入不足或日光暴露不足，导致肠道钙、磷吸收减少。
2. **识别低钙血症反应**：低血钙刺激甲状旁腺激素（PTH）分泌增加。
3. **分析PTH代偿效应**：
   - PTH促进破骨细胞活性，动员骨钙入血
   - PTH抑制肾小管对磷的重吸收，导致尿磷增多、低磷血症加重
4. **评估矿化障碍**：计算血钙×血磷乘积（单位：mg/dL），若 <30 mg²/dL²，提示骨基质无法正常矿化。
5. **解释骨骼病理改变**：
   - 骺板软骨细胞分化紊乱，临时钙化带消失
   - 成骨细胞代偿增生，堆积未矿化骨样组织，形成肋骨串珠、手足镯、方颅等体征
   - 骨皮质变薄、骨质疏松，负重后出现下肢弯曲
6. **关联生化标志物**：碱性磷酸酶（ALP）因成骨活跃而显著升高。

## 输出结论
综合骨骼畸形、低25-(OH)D₃、高ALP和低磷，判定为活动期维生素D缺乏性佝偻病，需立即启动维生素D及钙剂干预。

> 注意：本技能仅适用于营养性维生素D缺乏所致佝偻病，排除遗传性低磷血症性佝偻病等非营养性病因。
