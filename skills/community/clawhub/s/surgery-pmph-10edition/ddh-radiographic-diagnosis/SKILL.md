# 发育性髋关节脱位（DDH）的影像学诊断标准

## 适用条件
- 患儿年龄 ≥ 3个月（髋臼骨化中心已部分出现）
- 已获取标准骨盆正位X线片
- 不适用于3个月以下婴儿（因髋臼主要为软骨，X线显影不佳）

## 执行步骤

### 1. 测量髋臼指数（Acetabular Index）
- 连接双侧Y形软骨中心形成Y线；
- 从同侧Y形软骨中心向髋臼外上缘连线形成C线；
- C线与Y线之间的夹角即为髋臼指数；
- 对照年龄对应的正常值范围判断是否存在髋臼发育不良。

### 2. 应用Perkin象限划分法（需股骨头骨骺已骨化）
- 以Y线（双侧髋臼中心连线）为基础；
- 从髋臼外缘向Y线作垂线（P线），将髋关节区域划分为四个象限；
- 判断股骨头骨骺所在象限：
  - 内下象限 → 正常位置；
  - 外下象限 → 半脱位；
  - 外上象限 → 全脱位。

### 3. 评估Shenton线连续性
- Shenton线由股骨颈内缘与闭孔上缘构成的平滑弧线；
- 观察该线是否中断或失去连续抛物线形态；
- 中断提示可能存在髋关节脱位。

## 综合判断
结合上述三项指标结果，评估髋关节脱位程度及发育状态，为临床治疗决策提供依据。

## 参考文档

### 髋臼指数正常值参考范围

| 年龄       | 正常髋臼指数范围（度） |
|------------|------------------------|
| 新生儿     | 30°–40°                |
| 1岁        | 23°–28°                |
| 3岁        | 20°–25°                |
| 12岁       | 约15°                  |

> 超过对应年龄正常上限提示髋臼发育不良。

**Perkin象限法使用前提**
- 仅在股骨头骨骺已开始骨化后适用（通常≥4–6个月）
- 若骨骺未显影，应依赖髋臼指数与Shenton线评估

**Shenton线判读要点**
- 正常情况下应为连续、光滑的抛物线
- 轻微中断可能见于体位不佳，需结合其他指标综合判断
- 明显中断高度提示脱位
