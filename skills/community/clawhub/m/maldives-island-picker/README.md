# maldives-island-picker Skill 介绍

## 📖 概述

**maldives-island-picker** 是一个专业的马尔代夫选岛助手，帮助用户根据旅行类型、个人诉求、交通偏好和预算，从海量攻略中智能筛选出最适合的马尔代夫岛屿，并生成结构化的选岛报告。

### 核心功能

- **🎯 智能需求收集**：通过结构化引导，精准捕捉用户的旅行类型（亲子/蜜月/朋友等）、核心诉求（浮潜/水屋/美食等）、交通偏好和预算范围
- **🔍 多源攻略搜索**：并行搜索知乎、小红书、马蜂窝等多个平台，获取真实用户评价和专业攻略
- **💰 实时商品查询**：集成飞猪 API，获取岛屿的实时价格和可直接预订的商品链接
- **📊 智能匹配推荐**：基于权重算法（核心诉求35% + 旅行类型25% + 预算25% + 交通15%）排序推荐最优岛屿
- **📝 结构化报告生成**：输出包含信息表格、推荐理由、真实反馈、商品卡片、横向对比表的完整选岛报告

### 工作流程

1. **收集需求** → 2. **搜索攻略** → 3. **查询商品** → 4. **生成报告** → 5. **导出文件**

### 触发关键词

当用户提到以下关键词时自动触发：
- "马尔代夫选岛"、"马代选岛"、"马尔代夫去哪个岛"
- "马尔代夫推荐"、"马代推荐岛屿"
- "马尔代夫亲子/蜜月/度假选岛"、"马尔代夫预算"
- 英文：Maldives island selection, Maldives resort recommendation, which Maldives island to choose
