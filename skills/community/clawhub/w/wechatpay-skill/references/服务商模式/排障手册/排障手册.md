# 服务商模式 - 排障手册

> 服务商模式下特有问题的排查

## 服务商特有错误码

| 错误码 | 含义 | 解决方案 |
|--------|------|----------|
| 268443705 | 子商户mchid不能与自身mchid相同 | 检查sub_mchid参数 |
| 268443704 | 受理机构必须传入sub_mch_id | 下单时需传子商户号 |
| 268443701 | 受理关系不存在 | 子商户未进件或未绑定 |
| 268443702 | 子商户状态异常 | 检查子商户审核状态 |
| 268443715 | 子商户权限不足 | 服务商后台配置权限 |

## 子商户相关问题

### 子商户无法支付

排查步骤：
1. 检查子商户是否审核通过
2. 检查子商户AppID是否与mchid绑定
3. 检查服务商是否配置子商户权限
4. 检查子商户是否签约支付产品

### 分账失败

排查步骤：
1. 检查子商户是否已添加为分账接收方
2. 检查分账比例是否超限
3. 检查子商户是否微信支付实名
4. 检查资金是否已到位

### 合单支付问题

排查步骤：
1. 检查子单数量是否超限
2. 检查子商户是否绑定同一AppID
3. 检查子单金额是否超限
4. 检查关单只能关合单

## 服务商模式调试建议

1. **区分问题来源**：先确认是服务商问题还是子商户问题
2. **检查参数传递**：确认sp_mchid、sub_mchid正确传递
3. **查看服务商后台**：登录服务商后台查看配置
4. **查看子商户状态**：检查子商户进件和绑定状态
