# 药动学公式与参数详解

## 关键变量定义
- **C̄ₛₛ**：平均稳态血浆浓度（单位：mg/L 或 μg/mL）
- **τ**：给药间隔时间（单位：h）
- **Vₛₛ**：稳态表观分布容积（单位：L）
- **CL**：药物清除率（单位：L/h 或 mL/min）
- **F**：生物利用度（无量纲，0–1 或 %）
- **t₁/₂**：消除半衰期（单位：h）
- **Kₛ**：消除速率常数，Kₛ = 0.693 / t₁/₂

## 详细公式

### 峰浓度与谷浓度
- Cₛₛ,ₘₐₓ = (F·D) / [Vₛₛ·(1 - e⁻ᴷˢᵗᵃᵘ)]
- Cₛₛ,ₘᵢₙ = Cₛₛ,ₘₐₓ · e⁻ᴷˢᵗᵃᵘ

### 波动度（DF）
- DF(%) = [(Cₛₛ,ₘₐₓ - Cₛₛ,ₘᵢₙ) × 2] / (Cₛₛ,ₘₐₓ + Cₛₛ,ₘᵢₙ)

### 累积因子（R）
- R = 1 / (1 - e⁻ᴷˢᵗᵃᵘ)
- 当 τ = t₁/₂ 时，R ≈ 1.44
- τ < t₁/₂ → R > 1.44（易蓄积）
- τ > t₁/₂ → R < 1.44（不易蓄积）

## 稳态达成时间
- 经 4 个 t₁/₂ 达到约 94% 稳态
- 经 5 个 t₁/₂ 达到约 97% 稳态
- 提高剂量或频率不能缩短达稳态时间，仅影响浓度水平

## 个体化给药流程
1. 设定靶浓度范围（MEC ~ MTC）
2. 估算患者 CL 和 Vₛₛ（基于体重、肝肾功能等）
3. 计算初始负荷剂量与维持剂量
4. 给药并采集稳态谷浓度（或峰谷浓度）
5. 利用实测浓度反推个体 CL/Vₛₛ
6. 调整剂量或间隔，重复直至达标
