# CPR质量实时监测的关键指标与阈值

## 何时使用本技能
- 患者处于心搏骤停状态
- 正在进行胸外按压
- 具备以下任一监测设备：呼气末二氧化碳（PETCO₂）监测仪、动脉导管、心电监护仪、CPR反馈装置

## 执行步骤

### 1. 使用视听反馈装置
- 实时监控按压频率（目标100–120次/分）、深度（5–6 cm）、充分回弹
- 根据设备提示立即纠正技术偏差

### 2. 监测动脉血压（如有创通路）
- 确保动脉舒张压 ≥20 mmHg（反映冠状动脉灌注压）
- 维持足够平均动脉压以保障脑灌注

### 3. 心电监测
- 持续识别可电击心律（如室颤、无脉性室速）
- 为除颤时机和抗心律失常药物选择提供依据

### 4. 呼气末二氧化碳（PETCO₂）监测
- **CPR有效标准**：PETCO₂ >10 mmHg
- **CPR质量不足警报**：PETCO₂ ≤10 mmHg → 立即优化按压
- **自主循环恢复（ROSC）标志**：PETCO₂突然升至 ≥40 mmHg

## 警报触发条件
- 若 **PETCO₂ ≤10 mmHg** 或 **动脉舒张压 <20 mmHg**，立即发出“CPR质量不足”警报，并调整按压技术。

## 注意事项
- 非心搏骤停患者禁用此监测逻辑
- 动脉血压监测需有创操作，仅在具备条件时使用
