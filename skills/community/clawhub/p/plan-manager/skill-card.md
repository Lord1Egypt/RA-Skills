## Description: <br>
Unified plan management system for coordinating OpenClaw Agent and Human plans. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wx528](https://clawhub.ai/user/wx528) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and OpenClaw users use this skill to organize multi-step agent execution plans alongside human task plans, with shared status tracking, detailed plan templates, and archive conventions. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated plan documents may contain private project details. <br>
Mitigation: Review generated plans before committing, publishing, or sharing them. <br>
Risk: Persistent planning files can add unnecessary workspace state for quick one-off todos. <br>
Mitigation: Use the skill when saved planning artifacts are useful, and skip it for short-lived task lists. <br>
