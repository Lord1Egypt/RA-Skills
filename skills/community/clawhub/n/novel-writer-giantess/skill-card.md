## Description: <br>
Chinese-language novel-writing agent that helps analyze reference style, manage complex size-hierarchy worldbuilding, generate outlines, draft chapter segments, check outline drift, and review completed chapters. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[sherlockeinstein](https://clawhub.ai/user/sherlockeinstein) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Writers and creative-writing agents use this skill to run a structured long-form Chinese novel workflow: initialize project files, analyze reference prose style, maintain setting and character consistency, generate outlines, draft chapter segments, and produce review reports. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The workflow may create or update local story project files. <br>
Mitigation: Use a clear project directory and review generated or modified planning, chapter, character, setting, and review files before relying on them. <br>
Risk: Reference text supplied for style analysis may be summarized into locally stored style notes. <br>
Mitigation: Avoid providing private or sensitive reference material unless local derived notes are acceptable. <br>
Risk: The included templates cover explicit giantess and sexualized size-hierarchy fiction. <br>
Mitigation: Install and use the skill only for appropriate creative-writing contexts and adjust templates when that content is not desired. <br>
