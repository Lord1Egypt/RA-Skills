# 电除颤操作规范

## 适用对象
- 室颤（VF）患者
- 无脉性室速（PVT）患者

## 使用前提
- 已确认为可除颤心律（VF 或 PVT）
- 除颤仪处于可用状态
- 心肺复苏（CPR）已开始

## 禁忌与限制
- 不适用于无脉性电活动（PEA）或心室停搏
- 禁止在易燃环境或水中使用

## 操作流程

### 1. 除颤原则
- 室颤和无脉性室速视为“可除颤心律”，应尽早除颤
- 优先使用双相波除颤仪
- 初始能量设置：
  - 双相波：200 J（或按设备厂商建议）
  - 单相波：360 J

### 2. 操作步骤
- **体位准备**：去枕平卧，充分暴露胸部，左上肢外展
- **皮肤处理**：擦干胸部皮肤，移除金属物品；若患者有植入式起搏器，避开其位置
- **设备设置**：开启除颤仪，选择非同步模式
- **电极安放**：采用“前-侧位”
  - 右侧电极：胸骨右缘第二肋间（心底部）
  - 左侧电极：左腋中线第五肋间（心尖部）
- **导电与充电**：均匀涂抹导电糊或使用粘贴式电极片，充电至设定能量
- **放电警示**：高声喊出“准备除颤，大家离开！”确保无人接触患者
- **实施放电**：双手同时按下放电按钮，并对电极板施加约10 kg压力

### 3. 除颤后处理
- 立即恢复高质量CPR（5个循环，约2分钟）
- 2分钟后再次评估心律与脉搏
- 若仍为可除颤心律，重复“除颤 → CPR → 肾上腺素给药”循环

### 4. 注意事项
- 除颤准备不应导致CPR长时间中断（中断时间应<10秒）
- 首次除颤后必须立即恢复胸外按压，不得延迟
- 公共场所可使用AED，其可自动识别心律并语音指导非专业人员操作

> **关键提示**：本技能仅用于VF或PVT；若为PEA或心室停搏，应继续CPR并查找可逆病因，而非除颤。
