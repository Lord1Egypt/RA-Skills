# 基质的分子筛屏障机制

## 何时使用
- 分析结缔组织对细菌（如溶血性链球菌）或癌细胞的物理屏障能力
- 评估细胞外基质在炎症、感染或肿瘤微环境中的通透性变化
- 判断透明质酸酶活性是否可能破坏基质防御功能

## 执行步骤
1. **确认基质结构完整性**：检查是否存在完整的蛋白聚糖聚合体网络，包括：
   - 透明质酸主干（长度可达2.5μm）
   - 硫酸化糖胺聚糖（如硫酸软骨素、硫酸角质素等）与核心蛋白共价结合形成的辐射状亚单位
   - 结合蛋白介导的亚单位与透明质酸主干的连接

2. **评估分子筛功能状态**：
   - 若结构完整，则分子筛允许小分子（水、电解质、单糖、气体、激素、代谢产物）自由通过
   - 同时阻挡大分子物质（如细菌、大分子蛋白质）

3. **检测屏障失效风险**：
   - 若存在能分泌透明质酸酶的病原体（如溶血性链球菌）或癌细胞，则屏障可能被破坏
   - 此时需进一步评估透明质酸酶活性

4. **输出判断结果**：
   - 基质是否具备有效防御屏障功能：是 / 否
   - 若为“否”，建议评估透明质酸酶活性及基质降解程度
