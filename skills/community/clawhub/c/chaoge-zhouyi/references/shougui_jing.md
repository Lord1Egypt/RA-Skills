# 守鬼经

## 概述

《守鬼经》为古代驱鬼、镇鬼、守灵类典籍，记载鬼神之性与应对之法。

---

## 卷一：识鬼

### 鬼之本性

鬼者，阴气之聚也。

**鬼有三性**：
1. 畏阳气：日出而隐，日落而出
2. 喜阴气：聚于阴地，避于阳方
3. 惧正气：正气足者，鬼神避之

### 鬼之形态

| 鬼类 | 形态 | 出没 | 性质 |
|------|------|------|------|
| 游魂 | 飘忽不定 | 墓地荒野 | 无害 |
| 冤鬼 | 哭泣呼冤 | 凶案之地 | 有害 |
| 厉鬼 | 面目可憎 | 忌日 | 大害 |
| 善鬼 | 和善安详 | 祭祀之地 | 护主 |

### 鬼之强弱

**弱鬼**：新死未久，阳气未尽，力弱易驱
**中鬼**：死有年月，阴气凝聚，需法术驱
**强鬼**：积怨深重，厉气成煞，需高人解

## 卷二：鬼入

### 鬼入之兆

鬼欲入宅，必先示兆：

1. 冷风突至，温度骤降
2. 灯光忽明忽暗
3. 异声异响（脚步、叹息）
4. 动物异常（犬吠、猫叫）
5. 灵梦示警

### 鬼入之径

| 方位 | 时间 | 鬼类 |
|------|------|------|
| 正北 | 子时 | 水鬼 |
| 正南 | 午时 | 火鬼 |
| 正东 | 卯时 | 木鬼 |
| 正西 | 酉时 | 金鬼 |
| 四隅 | 四季 | 土鬼 |

### 鬼入之因

1. 阳气衰退，阴气上升
2. 招惹鬼神，犯忌触讳
3. 宅地不净，阴气聚集
4. 时运不济，命犯阴煞

## 卷三：守宅

### 守宅之法

#### 一、镇宅符

**作用**：镇宅辟邪，保家平安

**贴法**：
- 大门正中
- 窗户四角
- 卧室床头

#### 二、镇宅物

| 物品 | 作用 | 摆放 |
|------|------|------|
| 八卦镜 | 照妖辟邪 | 门楣悬挂 |
| 桃木剑 | 驱鬼斩煞 | 客厅悬挂 |
| 貔貅 | 招财辟邪 | 面向大门 |
| 麒麟 | 镇宅化煞 | 门口两侧 |
| 龙龟 | 化煞招财 | 客厅明堂 |

#### 三、守宅咒

```
天圆地方，律令九章。
吾今下笔，万鬼伏藏。
二十四气，七十二候。
五行相生，八卦相扶。
左青龙，右白虎。
前朱雀，后玄武。
中央戊己土，护卫吾宅。
急急如律令。
```

### 守宅禁忌

- 不在夜半开门
- 不在阴处久留
- 不带外物入宅
- 不妄语鬼神

## 卷四：守灵

### 守灵之法

人死之后，七七四十九日内为守灵期。

**守灵要点**：

1. **头七**：魂魄归家，家人守候
2. **三七**：亡魂渐醒，祭奠祈福
3. **五七**：审判功过，超度加持
4. **七七**：转世投胎，功德圆满

### 守灵禁忌

| 禁忌 | 原因 |
|------|------|
| 不哭泣过度 | 令亡魂不舍 |
| 不争吵喧哗 | 惊扰亡魂 |
| 不动刀剪 | 恐伤亡魂 |
| 不穿红衣 | 喜气冲灵 |
| 不食荤腥 | 净身敬灵 |

### 守灵护身

守灵之人，宜：
- 佩戴护身符
- 默念护身咒
- 保持正气

## 卷五：驱鬼

### 驱鬼步骤

#### 第一步：辨鬼

观其形态，闻其声音，知其性质

#### 第二步：请神

念请神咒，请护法神

```
恭请三清道祖，太上老君。
天兵天将，护法龙神。
弟子有难，恳请降临。
驱邪斩煞，护卫生灵。
```

#### 第三步：驱鬼

念驱鬼咒，焚驱鬼符

```
太上老君敕令，天兵天将听令。
四方神明速至，斩妖除魔留情。
若有妖魔鬼怪，速速退避隐形。
违者化作脓血，永世不得超生。
急急如律令。
```

#### 第四步：谢神

礼毕谢神，焚香叩首

### 驱鬼器具

| 器具 | 用途 |
|------|------|
| 桃木剑 | 斩鬼驱邪 |
| 法铃 | 惊鬼震煞 |
| 符水 | 净宅化煞 |
| 柳枝 | 打鬼驱邪 |
| 糯米 | 撒鬼退避 |

## 卷六：镇鬼

### 镇鬼之法

鬼已入宅，不可单驱，需镇之。

#### 一、镇鬼符

**画法**：
- 黄纸朱砂
- 面北背南
- 念咒书符

#### 二、镇鬼咒

```
天雷地雷，护吾元神。
三魂七魄，不散不离。
若有邪鬼，敢来侵犯。
五雷正法，化为尘泥。
急急如律令。
```

#### 三、镇鬼物

- 镇鬼瓶：装鬼封印
- 镇鬼镜：照鬼原形
- 镇鬼印：镇压邪鬼

## 卷七：防鬼

### 养正气

正气内存，邪不可干。

**养气之法**：
- 早起早睡
- 言行端正
- 积德行善
- 清心寡欲

### 避鬼之法

| 方法 | 说明 |
|------|------|
| 不夜行 | 子时后不出门 |
| 不独行 | 夜行结伴 |
| 不回头 | 闻声不回头 |
| 不应声 | 有人叫名不应 |
| 不捡物 | 路上物不捡 |

### 护身之物

- 护身符
- 桃木饰品
- 朱砂手链
- 五帝铜钱
- 本命佛牌

## 卷八：问答

**问：见鬼如何？**

答：勿惊勿惧，正气护身，念咒退避。

**问：鬼可杀人乎？**

答：正气足者，鬼不能伤。心虚胆怯，鬼乘其隙。

**问：驱鬼危险否？**

答：德行不足，反受其害。量力而行，不可强为。

## 结语

守鬼之道，在守己心。

心正无惧，万邪不侵。

修身养德，鬼神敬之。
