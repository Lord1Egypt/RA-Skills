# 病理学核心知识框架

## 适用场景
- 医学生学习病理学课程
- 临床医师分析疾病机制以指导诊断与治疗
- 医学研究人员设计实验或解读病理数据

## 执行步骤
1. **区分总论与各论内容**：
   - 总论（第一至八章）：聚焦细胞和组织的共同病理规律，包括适应与损伤、修复、血液循环障碍、炎症、肿瘤等。
   - 各论（第九至十八章）：基于总论，解析各器官系统疾病的特殊规律（如肝炎、肾炎、肺炎虽同属炎症，但表现各异）。

2. **整合五大核心维度**：对任一疾病，同步分析其：
   - 病因（etiology）
   - 发病机制（pathogenesis）
   - 病理变化（pathological change）
   - 结局与转归
   - 临床病理联系

3. **遵循学习原则**：总论与各论必须互相参照，不可偏废——认识共性有助于理解特性，反之亦然。

4. **结合现代技术**：第十九章所述的免疫组织化学、分子病理学及人工智能等技术，为诊断与科研提供支撑，应在适当时机整合应用。

> **注意**：本框架仅适用于医学领域内与疾病机制相关的任务，不适用于非医学或与病理无关的研究。
