## 一、SPAC概述

### 1.1 什么是SPAC

**SPAC**（Special Purpose Acquisition Company）即"特殊目的收购公司"，是一种先上市后并购的融资工具。

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         SPAC运作原理                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   【第一阶段：IPO期】                                                       │
│   ┌─────────────────────┐                                                  │
│   │    SPAC发起人       │                                                  │
│   │   (Sponsor)        │                                                  │
│   └──────────┬──────────┘                                                  │
│              │ 设立+IPO                                                             │
│              ↓                                                             │
│   ┌─────────────────────┐                                                  │
│   │    空白SPAC         │ → 上市，募集信托资金                              │
│   │  (仅有现金，无业务)  │                                                  │
│   └─────────────────────┘                                                  │
│                                                                             │
│   【第二阶段：并购期】                                                      │
│   ┌─────────────────────┐                                                  │
│   │    寻找并购标的     │                                                  │
│   │  (De-SPAC)         │                                                  │
│   └──────────┬──────────┘                                                  │
│              │ 找到目标公司                                                 │
│              ↓                                                             │
│   ┌─────────────────────┐                                                  │
│   │    并购完成         │ → 原SPAC股东换成目标公司股东                      │
│   │   (Business Combo) │                                                  │
│   └──────────┬──────────┘                                                  │
│              │                                                             │
│              ↓                                                             │
│   ┌─────────────────────┐                                                  │
│   │    更名+上市        │ → 变成真正的上市公司                              │
│   └─────────────────────┘                                                  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 SPAC vs 传统IPO

| 对比维度 | SPAC上市 | 传统IPO |
|----------|----------|----------|
| **时间周期** | 3-6个月 | 9-18个月 |
| **不确定性** | 低（先确定壳再找标的） | 高（市场波动影响） |
| **定价机制** | 谈判定价 | 市场询价 |
| **费用结构** | 较低（无高额承销费） | 较高（2-4%承销费） |
| **稀释程度** | 较高（发起人股权+认股权证） | 较低（仅发行新股） |
| **适合阶段** | 成长期企业 | 成熟期企业 |

## 二、香港SPAC规则要点

### 2.1 发起人要求

| 要求项 | 具体标准 |
|--------|----------|
| **人数要求** | 最少1名SPAC发起人 |
| **资质要求** | 须为香港证监会持牌人士或机构 |
| **持股要求** | 发起人在上市日须持有≥10%发起人股份 |

### 2.2 投资者要求

| 投资者类型 | 最低资格要求 |
|------------|--------------|
| **机构专业投资者** | 无限制 |
| **散户** | 仅可在IPO和赎回时参与 |
| **并购完成后** | 散户可自由买卖 |

### 2.3 集资规模

| 要求项 | 具体标准 |
|--------|----------|
| **最低集资额** | 10亿港元 |
| **最高集资额** | 无上限 |
| **发行价** | 每股≥10港元 |

### 2.4 并购规则

| 要求项 | 具体标准 |
|--------|----------|
| **并购期限** | 必须在上市后24个月内完成并购 |
| **延期条款** | 可申请延期最多6个月（需股东投票） |
| **继承公司市值** | 须达港交所上市规则的市值门槛 |
| **股东批准** | 须 majority shareholders 批准 |

## 三、适合的OPC项目类型

### 3.1 匹配度评估

| OPC项目类型 | SPAC匹配度 | 说明 |
|-------------|------------|------|
| **AI/大模型** | ⭐⭐⭐⭐ | 成长性高，故事好讲 |
| **SaaS/企业软件** | ⭐⭐⭐⭐ | 现金流可预测 |
| **新能源/储能** | ⭐⭐⭐⭐⭐ | 热门赛道，投资人关注 |
| **医疗健康** | ⭐⭐⭐ | 需要临床数据支撑 |
| **消费品牌** | ⭐⭐⭐ | 需要规模验证 |

### 3.2 适配条件

**必须满足**：
- 目标公司估值≥SPAC信托资产的80%
- 有清晰的商业模式和增长路径
- 创始团队愿意接受SPAC结构
- 可接受一定的股权稀释

## 四、SPAC并购流程

### 4.1 完整流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         SPAC并购完整流程                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ 【阶段一：寻找标的期 12-24个月】                                             │
│    │                                                                      │
│    ├── SPAC上市（IPO）                                                    │
│    ├── 寻找并购标的                                                        │
│    └── 尽职调查                                                            │
│    ↓                                                                      │
│ 【阶段二：谈判期 2-4个月】                                                  │
│    │                                                                      │
│    ├── 估值谈判                                                            │
│    ├── 结构设计                                                            │
│    ├── 股东权益保护条款                                                    │
│    └── 签署并购协议                                                        │
│    ↓                                                                      │
│ 【阶段三：审批期 2-3个月】                                                  │
│    │                                                                      │
│    ├── 港交所审批                                                          │
│    ├── 股东投票                                                            │
│    ├── 赎回选择（股东可选择赎回）                                          │
│    └── 完成并购                                                            │
│    ↓                                                                      │
│ 【阶段四：过渡期 1-2个月】                                                  │
│    │                                                                      │
│    ├── 更名                                                               │
│    ├── 人员调整                                                            │
│    └── 正式交易                                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 4.2 关键时间节点

| 里程碑 | 时间点 | 说明 |
|--------|--------|------|
| SPAC IPO | 月份0 | 壳公司上市 |
| 开始寻找标的 | 月份1 | 通常与IPO同步 |
| 签署并购协议 | 月份6-18 | 取决于谈判进度 |
| 港交所审批 | 协议后2-3个月 | 审批周期 |
| 完成并购 | IPO后24个月内 | 必须完成 |

## 五、估值与稀释

### 5.1 估值方式

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| **P/E倍数** | 按净利润倍数估值 | 盈利企业 |
| **P/S倍数** | 按营收倍数估值 | 成长期企业 |
| **用户数×单用户价值** | 互联网估值法 | 用户驱动型 |
| **DCF** | 现金流折现 | 稳定现金流企业 |
| **可比公司** | 参考同类上市公司 | 通用 |

### 5.2 稀释计算示例

| 项目 | 说明 |
|------|------|
| **SPAC信托资金** | 10亿港元 |
| **并购估值** | 20亿港元（信托资金的200%） |
| **原股东获得** | 66.7% 继承公司股权 |
| **发起人权益** | 通常为20%（含认股权证） |
| **公众股东** | 剩余股权 |

### 5.3 发起人收益结构

| 组成部分 | 比例 | 说明 |
|----------|------|------|
| **发起人股份** | 10-20% | 以极低价格获得 |
| **认股权证** | 一定比例 | 可按约定价格认购股票 |
| **激励股权** | 视条款而定 | 并购完成后奖励 |

## 六、优缺点分析

### 6.1 优点

| 优点 | 说明 |
|------|------|
| ✅ **时间短** | 3-6个月完成，比传统IPO快 |
| ✅ **确定性高** | 先上市再并购，减少市场波动影响 |
| ✅ **费用低** | 无高额承销费 |
| ✅ **谈判灵活** | 估值和条款可协商 |
| ✅ **品牌背书** | SPAC发起人可为项目背书 |

### 6.2 缺点

| 缺点 | 说明 |
|------|------|
| ❌ **稀释较大** | 发起人股份+认股权证导致稀释 |
| ❌ **限制多** | 24个月须完成并购，压力较大 |
| ❌ **信任问题** | 投资者对壳公司有顾虑 |
| ❌ **赎回风险** | 股东可能选择赎回不参与并购 |
| ❌ **条款复杂** | 法律架构复杂，成本高 |

## 七、如何选择SPAC

### 7.1 选择考量因素

| 因素 | 评估内容 |
|------|----------|
| **发起人背景** | 行业经验、资源整合能力 |
| **信托资金规模** | 决定可并购的标的规模 |
| **发起人股权比例** | 影响原股东稀释程度 |
| **并购行业偏好** | 是否与自身业务匹配 |
| **剩余期限** | 时间压力评估 |

## 八、行动计划

### 8.1 自评估清单

```
□ 项目是否处于成长期，需要快速融资？
□ 是否愿意接受一定的股权稀释？
□ 是否能接受24个月内完成并购的时间压力？
□ 创始人团队是否愿意参与SPAC结构？
□ 是否了解SPAC的法律和财务影响？
□ 是否有合适的SPAC发起人资源？
```

### 8.2 下一步行动

| 时间 | 行动项 | 负责方 |
|------|--------|--------|
| **+1个月** | 了解市场上活跃的SPAC | 创始人+FA |

| **+6个月** | 评估SPAC vs IPO的利弊 | 创始人+投行 |
| **+12个月** | 决定是否走SPAC路径 | 创始人+董事会 |
