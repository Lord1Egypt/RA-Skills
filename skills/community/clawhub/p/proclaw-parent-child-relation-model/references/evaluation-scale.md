## 使用说明

### 评估目的

1. **追踪进展**：客观记录改变
2. **识别问题**：发现需要调整的地方
3. **指导决策**：基于数据调整策略
4. **赋能家长**：让家长看到进步

### 评估频率

| 阶段 | 频率 |
|------|------|
| 密集期（每月2-4次） | 每2周评估 |
| 改善期（每月1-2次） | 每月评估 |
| 维持期（每月1次） | 每2-3月评估 |

### 评估方式

- **自评**：家长填写
- **观察**：指导者观察记录
- **访谈**：与孩子/家长对话
- **综合**：多方式结合

### 里程碑评估

**短期里程碑（1-2个月）**：
- 家长能识别触发点

- 孩子行为改善
- 亲子沟通质量提升

## 评估结果解读

### 分数解读

| 分数范围 | 解读 | 建议 |
|----------|------|------|
| 4.0-5.0 | 优秀/显著改善 | 继续保持 |
| 3.0-3.9 | 良好/有进步 | 继续当前策略 |
| 2.0-2.9 | 一般/进展缓慢 | 评估策略调整 |
| 1.0-1.9 | 不足/问题严重 | 深度评估/转介 |

### 预警指标

**需要特别关注的信号**：
- 任何维度持续低于2.0
- 分数持续下降
- 家长报告与观察不符
- 孩子功能明显受损

### 评估后的行动

| 评估结果 | 行动 |
|----------|------|
| 显著改善 | 强化、庆祝、进入维持 |
| 有进步 | 继续当前策略、微调 |
| 进展缓慢 | 重新评估、调整策略 |
| 问题严重 | 深度评估、考虑转介 |
