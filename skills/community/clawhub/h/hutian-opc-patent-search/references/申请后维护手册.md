# 申请后维护手册

> 本手册涵盖专利/商标/软著授权后的法律状态跟踪、维护规则、使用合规指南、权利转化与运营。

## 第1页/共6页

## 一、法律状态跟踪

### 1.1 专利审查进度跟踪

**关键节点监控表**：

| 阶段 | 正常时长 | 异常预警信号 | 应对措施 |
|------|---------|-------------|---------|
| 受理 | 1周内 | 超2周未受理 | 联系代理所核查 |
| 初审 | 3-12月 | 超12月无消息 | 主动查询进度 |
| 公布 | 18月 | 申请日满18月未公布 | 可请求提前公布 |
| 实审请求 | 申请日3年内 | 超期未提实审请求 | 视为撤回 |
| 实审 | 2-3年 | 超3年无结果 | 可提加快审查PPH |
| 授权 | 收到通知2月内 | 超期未缴费 | 视为撤回，2个月内可恢复 |

**专利审查高速路（PPH）加速条件**：
- 在其他国家/地区已有同申请可授权/允许的审查意见
- 对应申请与目标申请的权利要求充分对应
- 适用类型：首次申请+PCT申请

### 1.2 商标审查进度跟踪

**关键节点监控表**：

| 阶段 | 正常时长 | 异常预警信号 | 应对措施 |
|------|---------|-------------|---------|
| 受理 | 1个月内 | 超1个月 | 联系代理所 |
| 初审公告 | 4-7月 | 超9月无公告 | 查询审查进度 |
| 异议期 | 3个月 | 公告期起算 | 监控是否有异议 |
| 注册公告 | 异议期满 | 有异议则暂停 | 准备异议答辩 |
| 领证 | 1-2月 | 超2月未领证 | 联系代理所 |

### 1.3 审查意见通知书应对

**专利审查意见答复模板**：

```markdown
# 审查意见答复书

专利申请号：__________
发明名称：__________

针对审查员于____年____月____日发出的审查意见通知书，申请人答复如下：

## 一、对审查意见的总体意见
[同意/部分同意/不同意审查意见，说明理由]

## 二、对权利要求的修改
[修改后的权利要求书，说明修改依据]

## 三、具体答复意见
[逐条答复审查意见，说明如何克服指出的缺陷]

## 四、如有必要，补充实验数据
[证明技术效果的数据]

## 五、结论
申请人认为，经过上述修改，本申请已克服审查意见指出的全部缺陷，
符合专利法的规定，请求审批授予专利权。

申请人（签章）：__________
____年____月____日
```

**常见审查意见应对策略**：

| 通知书类型 | 常见问题 | 答复策略 |
|-----------|---------|---------|
| 初审补正通知 | 形式问题 | 按要求补正即可 |
| 实审意见（新颖性） | 与对比文件相似 | 修改权利要求，增加区别特征 |
| 实审意见（创造性） | 技术方案显而易见 | 论证技术效果，强调非显而易见 |
| 实审意见（不支持） | 说明书不支持 | 缩小保护范围或补充实施例 |
| 驳回决定 | 上述问题未克服 | 评估是否复审 |

## 二、专利授权后维护

### 2.1 年费缴纳时间表

**发明专利年费标准**（费减前/费减70%后）：

| 年度 | 第1-3年 | 第4-6年 | 第7-9年 | 第10-12年 | 第13-15年 | 第16-20年 |
|------|--------|--------|--------|----------|----------|----------|
| 标准(元) | 900 | 1200 | 2000 | 4000 | 6000 | 8000 |
| 费减后(元) | 270 | 360 | 600 | 1200 | 1800 | 2400 |

**实用新型/外观设计年费标准**（费减前/费减85%后）：

| 年度 | 第1-3年 | 第4-5年 | 第6-8年 | 第9-10年 |
|------|--------|--------|--------|----------|
| 实用新型(元) | 600 | 900 | 1200 | 2000 |
| 外观设计(元) | 600 | 900 | 1200 | 2000 |
| 费减后(元) | 90 | 135 | 180 | 300 |

**滞纳金规则**：
- 缴费期满6个月内补缴：每月加收当年年费的5%
- 超6个月未缴：专利权自应期限届满之日起终止

### 2.2 年费管理策略

**维持决策矩阵**：

| 商业价值 | 技术领先性 | 建议策略 |
|---------|-----------|---------|
| 高 | 领先 | 按时缴费，维持全面保护 |
| 高 | 落后 | 按时缴费，但评估技术升级 |
| 低 | 领先 | 按时缴费，可考虑许可/转让 |
| 低 | 落后 | 评估放弃，节省维护成本 |
| 不确定 | 领先 | 按时缴费，或考虑开放许可 |
| 不确定 | 落后 | 评估放弃 |

**开放许可制度**：
- 专利权人声明愿意许可任何人实施其专利
- 许可费标准公开
- 可在专利权存续期间随时撤回声明
- 优势：年费减缴50%

### 2.3 专利权终止与恢复

**终止情形**：
- 年费逾期未缴
- 书面声明放弃专利权
- 保护期限届满

**恢复条件**：
- 因不可抗力或正当理由导致延误期限
- 自收到专利权终止通知起2个月内
- 提交恢复请求+补缴年费+滞纳金

## 三、商标授权后维护

### 3.1 商标使用义务与"撤三"风险

**三年不使用撤销制度（撤三）**：

| 要点 | 说明 |
|------|------|
| 法律依据 | 《商标法》第49条 |
| 触发条件 | 注册满3年且连续3年未使用 |
| 启动方式 | 任何人均可向商标局提出申请 |
| 答辩期限 | 收到撤销通知2个月内 |
| 使用的界定 | 商业性真实使用 |
| 使用证据要求 | 发票/合同/广告/包装/产品等 |

**使用证据准备清单**：

| 证据类型 | 具体示例 |
|---------|---------|
| 发票 | 增值税发票、普通发票 |
| 合同 | 销售合同、服务合同 |
| 广告 | 广告合同、媒体发布证明 |
| 包装/标签 | 实物照片、产品包装 |
| 展会材料 | 参展合同、现场照片 |
| 网络证据 | 网页截图、销售平台后台 |

**常见撤三答辩理由**：
- 因不可抗力等正当理由未使用
- 已有实际使用行为，仅是使用证据暂时缺失
- 他人联合注册/防御性注册等特殊情况

### 3.2 商标续展管理

**续展时间线**：

```
到期前12个月          到期日           宽展期(6个月)        注销
    ↓                   ↓                   ↓                ↓
可申请续展 ─────────→ 到期 ──────────→ 滞纳金续展 ──────→ 权利终止
```

**续展费用**：

| 项目 | 标准费用 | 宽展期费用 |
|------|---------|-----------|
| 续展注册费 | 500元/类 | 500+250=750元/类 |
| 变更费 | 250元/类 | 250元/类 |

**注意事项**：
- 商标名称/地址变更需先办变更再续展
- 续展后有效期重新计算10年
- 建议提前1年关注，避免遗忘

### 3.3 商标变更备案

**必须变更的情形**：
| 变更类型 | 是否必须备案 |
|---------|-------------|
| 注册人名义变更 | ✅ 必须 |
| 注册人地址变更 | ✅ 必须 |
| 商品/服务范围变化 | ❌ 需另行申请 |

**变更材料**：
- 变更申请书
- 变更证明文件（工商变更登记证明）
- 委托书（如代理办理）

## 四、软著授权后维护

### 4.1 软著保护期限

| 主体类型 | 保护期限 | 起算时间 |
|---------|---------|---------|
| 自然人 | 终身 + 死后50年 | 首次发表日期 |
| 法人/组织 | 50年 | 首次发表日期 |
| 未发表作品 | 50年 | 创作完成日期 |

### 4.2 版本更新登记

**需要重新登记的情况**：
- 软件名称变更
- 著作权人变更
- 主要功能发生重大变化
- 代码改动超过30%

**变更登记材料**：
- 变更登记申请表
- 原登记证书
- 变更证明材料
- 变更后的部分代码（若涉及代码变更）

### 4.3 软著使用与声明规范

**版权声明格式**：
```markdown
© [年份] [著作权人名称]. All rights reserved.
软件著作权登记号：[20XXXXXXXX]
```

**不规范使用风险**：
- 未声明可能被他人抢先登记
- 声明不当可能产生法律风险
- 许可使用需签订书面协议

## 五、使用合规指南

### 5.1 专利标识标注规范

**正确标注方式**：

| 标注位置 | 标注内容 | 示例 |
|---------|---------|------|
| 产品本身 | 专利号+专利类型 | 中国发明专利 ZL202310001234.5 |
| 产品包装 | 专利号 | ZL202310001234.5 |
| 说明书 | 专利号 | 专利号：ZL202310001234.5 |

**常见错误标注**：
| 错误类型 | 正确做法 |
|---------|---------|
| 仅标注"专利产品" | 必须标注具体专利号 |
| 标注失效专利号 | 仅能标注有效专利 |
| 标注他人专利号 | 不得标注非己方专利 |
| 标注申请号（非专利号） | 只能标注已授权专利号 |

### 5.2 商标使用合规要点

**合规使用要求**：

| 要求 | 说明 | 违规后果 |
|------|------|---------|
| 使用注册标记® | 注册商标可标注®，非注册不得标注 | 未注册标注构成虚假宣传 |
| 不改变注册图样 | 不得自行改变字体/颜色/结构 | 可能丧失商标专用权 |
| 在核定商品范围使用 | 不得擅自扩大使用范围 | 可能构成商标侵权 |
| 注册人名义一致 | 不得变更名义使用 | 工商查处风险 |
| 保持显著性 | 不得将商标通用化 | 可能被撤销 |

**不规范使用情形**：
- 将®标用于未注册商标
- 自行改变注册商标字体颜色
- 将TM误用为®（TM无法律效力）
- 超范围使用（核定商品外使用）

### 5.3 软著使用合规要点

**署名规范**：
| 署名方式 | 说明 |
|---------|------|
| 个人开发 | 署真实姓名或笔名 |
| 合作开发 | 署全体开发者姓名 |
| 职务开发 | 署单位名称 |
| 委托开发 | 按合同约定 |

**许可使用合同要点**：
- 许可类型（独占/排他/普通）
- 许可范围（地域/期限/使用方式）
- 费用及支付方式
- 改进成果归属
- 侵权责任划分

## 六、权利转化与运营

### 6.1 专利许可合同要点

**许可类型对比**：

| 类型 | 许可人 | 被许可人 | 许可人能否用 | 转许可 |
|------|-------|---------|-------------|-------|
| 独占许可 | ❌ | ✅ | ❌ | 视合同约定 |
| 排他许可 | ✅ | ✅ | ❌ | 视合同约定 |
| 普通许可 | ✅ | ✅ | ✅ | 通常不允许 |

**专利许可合同核心条款**：

| 条款类型 | 关键内容 |
|---------|---------|
| 许可范围 | 地域范围、期限、使用方式（制造/销售/进口） |
| 许可费用 | 入门费、提成费、一次总算 |
| 最低使用费 | 约定最低销售量或最低提成 |
| 技术支持 | 是否提供技术支持及范围 |
| 改进回馈 | 改进成果的归属和使用 |
| 侵权处理 | 侵权监控和维权分工 |
| 保密条款 | 技术秘密保护义务 |

### 6.2 商标许可合同要点

**许可合同核心条款**：

| 条款类型 | 关键内容 |
|---------|---------|
| 许可类型 | 独占/排他/普通许可 |
| 许可商品 | 具体使用哪些核定商品 |
| 许可地域 | 具体在哪些地区使用 |
| 许可期限 | 许可使用期限 |
| 质量监督 | 许可人对被许可人商品质量的监督权 |
| 许可备案 | 需向商标局备案才可对抗第三人 |
| 费用 | 许可使用费及支付方式 |

### 6.3 软著许可/转让合同要点

**许可合同核心条款**：

| 条款类型 | 关键内容 |
|---------|---------|
| 许可范围 | 使用方式（复制/发行/信息网络传播） |
| 许可期限 | 使用期限 |
| 许可费用 | 费用及支付方式 |
| 署名权 | 是否保留署名权 |
| 改编权 | 是否允许改编及成果归属 |

**转让合同核心条款**：

| 条款类型 | 关键内容 |
|---------|---------|
| 转让范围 | 全部还是部分权利转让 |
| 转让费用 | 一次性转让费用 |
| 登记变更 | 需向版权局办理变更登记 |
| 生效时间 | 合同生效≠权利生效，需登记后生效 |

### 6.4 与知识产权运营Skill联动

```
创新（36计+DIKWP）
     ↓
查新（本Skill：风险预判）
     ↓
申请（模块一~五：全流程护航）
     ↓
运营（知识产权运营Skill：估值→交易→保护闭环）
```

**联动场景**：
- 专利许可/转让前：先用知识产权运营Skill做估值
- 商标品牌运营：估值+许可合同+质量监督
- 软著变现：转让/许可给有意向的企业
- 成果转化：对接技术需求方，完成交易闭环

## 附录一：常见法律状态术语

| 术语 | 含义 |
|------|------|
| 实质审查生效 | 专利进入实审阶段 |
| 专利权有效 | 专利已授权且有效 |
| 专利权终止 | 保护期届满或未缴年费 |
| 专利权无效 | 被无效宣告程序撤销 |
| 撤回 | 申请人主动撤回申请 |
| 驳回 | 审查员驳回申请 |
| 复审 | 对驳回决定不服，向专利复审委员会申诉 |
| 撤销 | 对已授权专利提出无效程序 |

## 附录二：维权途径

| 维权方式 | 适用场景 | 费用 | 时效 |
|---------|---------|------|------|
| 协商和解 | 侵权轻微、双方愿意和解 | 低 | 快 |
| 行政投诉 | 侵权明显、证据易固定 | 中 | 中 |
| 民事诉讼 | 损失较大、需要赔偿 | 高 | 慢 |
| 刑事报案 | 侵权严重、构成犯罪 | 视情况 | 慢 |
