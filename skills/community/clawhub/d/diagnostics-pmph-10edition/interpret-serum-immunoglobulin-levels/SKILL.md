# 血清免疫球蛋白水平解读

## 使用前提
- 已通过免疫比浊法、ELISA等方法获得血清IgG、IgA、IgM或IgE的定量结果。
- 明确患者年龄（<16岁需使用年龄校正参考值）。

## 执行步骤

1. **确认适用参考区间**：
   - 若患者为成人（≥16岁），使用标准成人参考区间；否则需调用儿童/新生儿校正值（详见参考资料）。

2. **比对实测浓度与参考区间**：
   - IgG: 8.0～15.0 g/L
   - IgA: 0.7～3.5 g/L
   - IgM: 0.5～2.6 g/L
   - IgE: 0.1～0.9 mg/L（ELISA）或 <100 IU/ml（免疫比浊法）

3. **判断异常方向**：
   - 浓度高于上限 → 病理性增高
   - 浓度低于下限 → 病理性降低

4. **结合临床背景解读病理意义**：
   - **IgG↑**：考虑慢性感染、自身免疫病（如SLE、类风湿关节炎）、IgG型多发性骨髓瘤。
   - **IgA↑**：考虑肝硬化、IgA型骨髓瘤、肾脏疾病。
   - **IgM↑**：提示原发感染（如病毒性肝炎初期）、宫内感染（脐血IgM>0.2 g/L）、巨球蛋白血症。
   - **IgE↑**：提示过敏性疾病（哮喘、过敏性鼻炎）、寄生虫感染、IgE型骨髓瘤。
   - **所有Ig↓**：提示先天性或获得性体液免疫缺陷、肾病综合征（蛋白丢失）、免疫抑制治疗影响。
   - **IgM↓**：可见于IgG型重链病、淋巴瘤。
   - **IgE↓**：见于丙种球蛋白缺乏症、某些恶性肿瘤。

5. **特殊注意事项**：
   - IgG是唯一可通过胎盘的免疫球蛋白，新生儿早期依赖母体来源。
   - IgM为五聚体结构，是初次免疫应答中最早出现的抗体。
   - 若某单一Ig显著升高，需警惕单克隆丙种球蛋白病（如M蛋白），建议进一步行血清蛋白电泳或免疫固定电泳。
   - 局部黏膜感染时，分泌型IgA（SIgA）变化不能仅凭血清IgA水平反映。

> **触发场景示例**：用户输入“患者血清IgE为150 IU/ml，是否异常？可能原因？”或“IgM升高代表什么？”
