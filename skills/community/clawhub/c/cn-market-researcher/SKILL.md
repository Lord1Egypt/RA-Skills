# 市场调研助手 🔍

快速完成行业分析、竞品调研和市场趋势洞察，输出结构化的分析报告。帮助在信息收集和分析环节提升效率。

## 什么时候使用

- 需要快速了解一个陌生行业的格局时
- 产品立项前需要做竞品分析时
- 需要评估市场机会和进入策略时
- 需要跟踪行业最新趋势和变化时
- 准备商业计划书或投资分析时

## 核心能力

1. **行业全景分析**：快速勾勒行业规模、增长趋势、价值链结构
2. **竞品对标分析**：多维度对比主要竞争对手的产品、定价、市场策略
3. **趋势识别**：识别技术趋势、政策变化、用户行为变迁
4. **SWOT分析**：输出优势、劣势、机会、威胁的结构化分析
5. **市场机会评估**：识别市场空白、差异化方向和进入策略建议
6. **报告生成**：输出可交付的Markdown分析报告

## 使用说明

### 快速开始

用户可以说"帮我分析一下AI编程助手市场"或"调研一下国内低代码平台"，助手会：
1. 确认调研范围和深度
2. 收集公开信息和分析
3. 输出结构化报告

### 详细工作流

按以下流程执行调研：

1. **调研范围确定**
   - 确认调研对象（行业/公司/产品/技术）
   - 确认调研深度（速览版 / 标准版 / 深度版）
   - 确认关注重点（市场格局、技术趋势、商业模式等）

2. **信息收集**

   **公开数据源（使用curl或fetch）：**
   - 行业报告摘要（艾瑞、亿欧、36氪研究院）
   - 竞品官网和产品文档
   - 用户评价和社区讨论（知乎、V2EX、Reddit）
   - GitHub Stars和活跃度（开源项目）
   - 招聘信息和融资新闻

3. **结构化分析**

   **市场格局分析：**
   - 市场规模和增长率的公开数据
   - 主要玩家和市场份额估算
   - 市场集中度（CR3/CR5）

   **竞品对标（表格形式）：**
   | 维度 | 竞品A | 竞品B | 竞品C |
   |------|-------|-------|-------|
   | 产品定位 | | | |
   | 目标用户 | | | |
   | 价格策略 | | | |
   | 核心功能 | | | |
   | 技术架构 | | | |
   | 用户规模 | | | |

   **SWOT分析：**
   - 优势：技术壁垒、用户基础、品牌认知
   - 劣势：功能缺失、体验问题、覆盖不足
   - 机会：新兴需求、政策红利、技术变革
   - 威胁：新进入者、替代品、监管风险

4. **报告输出**
   - 标题 + 一句话结论
   - 关键发现（Top 3）
   - 详细分析
   - 建议和行动项
   - 数据来源说明

## 输出示例

```
# 🔍 AI编程助手市场调研报告

**一句话结论：** AI编程助手进入快速增长期，国内市场预计2026年达50亿规模。

## 关键发现
1. 市场竞争从"有无AI"转向"AI质量"竞争
2. 开源方案正在快速缩小与商业产品的差距
3. 企业级需求增长超过个人开发者

## 市场格局
| 产品 | 定位 | 定价 | 特点 |
|------|------|------|------|
| GitHub Copilot | 全能型 | $10/月 | 生态整合强 |
| 通义灵码 | 国内深耕 | 免费/企业版 | 中文优化 |
| CodeGeeX | 开源 | 免费 | 可私有部署 |

## 建议
- 差异化方向：垂直领域（如嵌入式开发）的深度优化
- 短期策略：利用开源模型降低推理成本
- 长期策略：积累行业特定数据形成壁垒
```

## 注意事项

- 市场数据来源为公开信息，可能存在滞后性
- 竞品分析基于公开可获取的信息，内部数据无法获取
- 建议结合一手调研（用户访谈、问卷）验证分析结论
- 涉及商业决策时，建议交叉验证关键数据
