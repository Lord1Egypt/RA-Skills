## これは何ですか？

**Gingiris SEO & GEO デュアルエンジン成長プレイブック** は、2026 年の検索の新現実に対応した完全な SEO/GEO 戦略ガイドです。現在、30% 以上の検索トラフィックが AI チャットボット（ChatGPT、Perplexity、Claude、Gemini）から始まっており、Google だけの最適化では不十分です。

本プレイブックは、従来の検索エンジンでランキングを獲得しながら、同時に AI 検索で引用されるコンテンツの作り方を教えます — これが GEO（Generative Engine Optimization、生成式エンジン最適化）です。

時代遅れの 2023 年 SEO アドバイス（キーワード密度、被リンク構築）とは異なり、本プレイブックは JSON-LD Schema パターン、E-E-A-T ライティングボイスシステム、比較ページテンプレート、IndexNow プッシュなど、AI エンジンが実際に引用するコンテンツパターンをカバーしています。

## コアフレームワーク：SEO + GEO デュアルエンジン

```
  従来の検索 (Google/Bing)
  ┌──────────────────────────┐
  │  キーワード + 被リンク    │ ← SEO
  │  Schema + Core Web Vitals│
  └────────────┬─────────────┘
               │
        ┌──────▼──────┐
        │  構造化コン  │ ← 共通基盤
        │  テンツ      │
        └──────┬──────┘
               │
  ┌────────────▼─────────────┐
  │  IndexNow + AI クローラー │ ← GEO
  │  直接回答 + 比較テーブル  │
  └──────────────────────────┘
  AI 検索 (ChatGPT/Perplexity/Claude)
```

## 5つのコア原則

1. **BOFU からスタートし上へ** — 高意図キーワード優先（価格設定、競合比較）
2. **本物の声 = 最良の E-E-A-T** — 創業者ストーリー、実数値、一次体験
3. **構造化 = 引用可能** — Key Stats テーブル、FAQ Schema、比較マトリックスで AI エンジンが直接引用可能
4. **IndexNow は GEO のインフラ** — コンテンツ更新を秒単位でプッシュ、クローラーを待たない
5. **比較ページは SEO のゴールドマイン** — 競合ごとに 1 ページ、意思決定段階のユーザーを獲得

## 内容一覧

- **キーワードファネル戦略** — TOFU/MOFU/BOFU マトリックスと優先順位付け

## よくある質問

**Q：一般的な SEO スキルと何が違いますか？**
A：多くの SEO スキルはまだ 2023 年のアドバイス（キーワード密度、被リンク）を教えています。本プレイブックは 30%+ のトラフィックが AI チャットボットから来る新しいデュアルリアリティに対応した書き方を教えます。

**Q：Schema マークアップと JSON-LD テンプレートは含まれていますか？**
A：はい。SoftwareApplication、Article、FAQ、HowTo、Comparison のコピー＆ペーストテンプレートと、Bing/Yandex への IndexNow プッシュ手順が含まれています。

**Q：プログラマティック SEO にも対応していますか？**
A：はい。キーワードファネルセクションでスケーラブルなテンプレートの特定、薄いコンテンツペナルティなしでのランディングページ生成、AI オーバービューで引用されるための構造化について解説しています。

**Q：日本語コンテンツの SEO/GEO 戦略は英語と異なりますか？**
A：コア方法論は同じですが、Google Japan の特性やローカル競合の強さ、日本語の自然な E-E-A-T 表現は異なります。プレイブックではマルチ言語 SEO の考慮点もカバーしています。

**Q：効果が出るまでどのくらいかかりますか？**
A：SEO コンテンツは通常 3〜6 ヶ月で効果が現れ始めます。比較ページは高意図キーワードを狙うため、4〜8 週間で転換をもたらす可能性があります。GEO（AI 引用）はフィードバックが早く、通常 2〜4 週間で観察可能です。
