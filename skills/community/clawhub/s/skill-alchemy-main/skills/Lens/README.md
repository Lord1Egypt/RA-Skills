## 它做什么

你描述一个任务时，天然会遗漏东西：没考虑过的视角、领域内讳莫如深的坑、拿到成品后才发现没做对的标准。Lens 在你动手之前把问题空间拉完整。不问你任何问题，不评判你的任务，不说教。可以直接用，也可以被编排器调用。

## 快速开始

```
npx skills add agentsope/SkillAlchemy/skills/Lens
```

然后用自然语言触发：

```
帮我想想这个项目有什么我没考虑到的
分析一下这个需求
帮我生成一个 skill
```

## 怎么工作

在心里跑完四步，全程不跟你交互：

1. **定性** — 你在干什么？给谁看？产出什么？
2. **展开** — 向上抽象一层，拆 subject/method，追问「什么叫好」
3. **自检** — 换名测试、术语测试、区分测试，任何一项不过就启动定向搜索
4. **收敛** — 匹配野心级别，只留 5-10 个影响力最大的维度

两种输出模式：**模式 A** — 隐性维度清单 + 快速检查项；**模式 B** — 增强版 description，可直接喂给下游引擎。
