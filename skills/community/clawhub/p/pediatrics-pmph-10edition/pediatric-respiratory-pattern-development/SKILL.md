# 儿童呼吸模式与急促判定

## 适用场景
- 患儿年龄处于新生儿至7岁之间
- 观察到呼吸频率增快或呼吸困难
- 已完成动脉血气分析（用于呼吸衰竭判断）

## 执行步骤
1. **判断预期呼吸类型**：
   - 0-2岁：腹式呼吸为主；
   - 2-7岁：过渡为胸腹式呼吸；
   - >7岁：接近成人胸腹协同模式。
2. **判定呼吸急促**（安静清醒状态下）：
   - <2个月：≥60次/分；
   - 2个月~1岁：≥50次/分；
   - 1~5岁：≥40次/分；
   - >5岁：≥30次/分。
3. **判断呼吸衰竭**（动脉血气，吸入空气）：
   - PaO₂ < 60 mmHg（8.0 kPa）；
   - PaCO₂ > 50 mmHg（6.67 kPa）；
   - SaO₂ < 85%。
4. **综合评估**：若呼吸急促伴鼻翼扇动、吸气性凹陷、发绀，或血气达阈值，提示严重呼吸系统疾病。

## 输出结果
- 根据年龄判断呼吸类型是否异常；
- 判定是否为“呼吸急促”；
- 若血气指标达标，发出“呼吸衰竭”警报，需紧急干预。
