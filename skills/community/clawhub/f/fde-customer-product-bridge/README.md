# FDE Customer-Product Bridge

This OpenClaw skill helps a Field Development Engineer connect customer discovery, product thinking, R&D input, onsite demos, and case-study packaging.

## Use Cases

- Customer discovery interview guides and meeting notes
- Onsite demo flow and talk track
- Customer case packaging and reusable success stories
- Customer feedback translated into product and engineering requirements
- Acceptance criteria, edge cases, and follow-up action lists

## Privacy

This skill contains only generic workflow instructions and templates. It does not include credentials, local paths, customer data, API keys, private URLs, or executable scripts.
