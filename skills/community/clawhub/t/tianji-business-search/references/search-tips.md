# 百度搜索高级技巧

## 基础语法

### 引号精确匹配
```
"Exact phrase"
```
搜索结果必须包含完整短语。

### 排除词
```
word1 -word2
```
包含 word1 但不包含 word2。

### OR 逻辑
```
word1 OR word2
```
包含任一关键词。

### 数字范围
```
手机 3000..5000
```
价格在 3000-5000 之间的手机。

## 限定来源

### 按网站
```
site:tianyancha.com "公司名"
site:qichacha.com "公司名"
site:36kr.com "公司名"
site:ithome.com "公司名"
```

### 按域名后缀
```
"公司名" site:.gov.cn    # 政府网站
"公司名" site:.edu.cn    # 教育机构
```

## 时间筛选

### 最近时间
```
after:2024-01-01
before:2024-12-31
```

### 相对时间
```
pd  # 最近24小时
pw  # 最近7天
pm  # 最近30天
py  # 最近365天
```

## 组合查询示例

### 查找最近融资新闻
```
"公司名" 融资 (A轮 OR B轮 OR C轮) after:2024-01-01
```

### 查找司法风险
```
"公司名" (诉讼 OR 判决 OR 被执行) site:tianyancha.com
```

### 查找竞品对比
```
"竞品A" vs "竞品B" 对比
```

### 查找高管信息
```
"公司名" CEO 创始人 简历
```
