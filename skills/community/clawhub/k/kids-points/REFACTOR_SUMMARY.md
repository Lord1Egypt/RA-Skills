## 🏗️ 架构设计

### 三层架构

```
┌─────────────────────────────────────┐
│  Agent (LLM) - 语义理解层            │
│  - 理解用户意图                      │
│  - 提取任务数据                      │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  技能代码 - 操作层                   │
│  - 防重复检查                        │
│  - 写入 balance.md                   │
│  - 写入 input.log                    │
│  - 保证数据一致性                    │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  文件系统 - 存储层                   │
│  - balance.md (唯一数据源)           │
│  - input.log (审计日志)              │
└─────────────────────────────────────┘
```

### 职责分工

| 组件 | 职责 | 实现方式 |
|------|------|----------|
| **语义理解** | 理解用户意图，提取任务 | ✅ Agent (LLM) |
| **积分计算** | 根据任务计算分数 | ⚠️ 灵活计分 |
| **记账操作** | 写入文件，保证数据一致 | ✅ 代码（确定性） |
| **防重复检查** | 检查是否重复记录 | ✅ 代码（确定性） |
