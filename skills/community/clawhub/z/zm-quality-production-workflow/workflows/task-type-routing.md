# 任务类型路由表

## 课程章节

- 主控：main / course-director
- 执行：course-director / media-director / writer-pro 视内容而定
- 必用：课程章节检查表、双 Agent 审核
- 样章：1-2 页课件 + 讲师提示卡

## PPT / HTML Deck

- 主控：main
- 执行：media-director / PPT 专职链路
- 协作 Skill：zm-ppt-deck-production-workflow、gpt-image2-ppt、powerpoint-pptx、pptx-2
- 样页：1-3 页
- 硬门禁：大纲未锁定不进入视觉；样页未确认不全量

## 漫画 / 小白手册

- 主控：main
- 执行：media-director
- 协作 Skill：happy-img2-direct
- 样章：P1 单页
- 硬门禁：缺主角图/角色参考不得全量；P1 未确认不得扩 P2-Pn；漫画图本体不得生成二维码

## 公众号文章 / 长文

- 主控：main / editorial-director
- 执行：writer-pro
- 审核：reviewer / critic / compliance
- 样稿：标题 + 开头 + 大纲
- 硬门禁：main 不裸写正式正文；无审核不放行

## 行业 AI 方案

- 主控：main
- 执行：方案专职 / code-engineer / media-director 按产物拆分
- 样章：一页方案结构或售前页
- 硬门禁：真实案例和交付边界不清，不得包装成已验证方案

## 代码 / 工具

- 主控：main
- 执行：code-engineer
- 样章：架构简审 / 最小可行验证
- 硬门禁：main 不直接写正式业务代码；必须有测试/验证/回滚说明
