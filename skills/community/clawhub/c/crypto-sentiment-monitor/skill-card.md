## Known Risks and Mitigations: <br>
Risk: Example commands reference an external xreach tool and a local analyze_sentiment.py script that were not bundled with the reviewed skill. <br>
Mitigation: Review and trust those tools or scripts before running them, and adapt the examples to approved local data sources. <br>
Risk: Crypto workflows may lead users to share wallet credentials or private keys even though this skill does not require them. <br>
Mitigation: Do not provide wallet credentials, private keys, seed phrases, or signing access when using this guidance skill. <br>
Risk: Market sentiment summaries can be incomplete or misleading if treated as financial advice. <br>
Mitigation: Use generated sentiment outputs as analysis inputs only and verify conclusions against independent market data before acting. <br>
