# 星币牌组 Pentacles

- **元素**：土
- **领域**：物质、身体、工作、金钱、健康、现实世界
- **心理维度**：感觉功能（Sensation）—— 与物质世界和身体的连接

---

## 星币王牌 Ace of Pentacles

- **正位关键词**：新机遇 / 物质基础 / 种子 / 潜力 / 丰盛的开端
- **正位含义**：一枚金币从云中递出——现实世界中一个新的可能性正在出现。可能是新工作、新项目或改善物质生活的机会。关键在于：你是否愿意接住它并播种。
- **逆位关键词**：错失机会 / 物质不安全 / 计划延迟 / 贪婪 / 根基不稳
- **逆位含义**：机会在眼前却没有抓住，或物质层面的不安全感阻碍了行动。也可能是过于贪婪，想要的太多反而什么都得不到。

## 星币二 Two of Pentacles

- **正位关键词**：平衡 / 灵活 / 多任务处理 / 适应 / 变化中的稳定
- **正位含义**：双手交替抛接两枚金币，背后的海浪起伏——在变化中保持动态平衡。生活需要灵活性，你正在学习在多重责任之间找到节奏。
- **逆位关键词**：失衡 / 过度承诺 / 杂乱 / 顾此失彼 / 财务紧张
- **逆位含义**：球太多了，开始掉地上。试图同时应对太多事情导致了混乱，需要做出取舍。

## 星币三 Three of Pentacles

- **正位关键词**：合作 / 精湛技艺 / 学习 / 团队协作 / 精益求精
- **正位含义**：工匠在教堂中雕刻——技艺、规划和合作的完美结合。你的专业能力正在被认可，协作带来了超越个人的成果。
- **逆位关键词**：缺乏团队协作 / 质量低下 / 不被认可 / 独断 / 学习不足
- **逆位含义**：独自蛮干而不寻求帮助，或者工作品质没有达到应有的标准。也许需要向他人学习。

## 星币四 Four of Pentacles

- **正位关键词**：保守 / 执着于安全 / 控制 / 守财 / 物质依附
- **正位含义**：紧紧抱住四枚金币——头上一枚，脚下一枚，手中两枚。对物质安全的执着可能正在限制你的成长。抓得太紧的手无法接受新的馈赠。
- **逆位关键词**：释放控制 / 慷慨 / 冒险 / 过度花费 / 放手
- **逆位含义**：开始放松对物质和控制的执着，学会在给予中获得。或者走向另一个极端——毫无节制地消费。

## 星币五 Five of Pentacles

- **正位关键词**：匮乏 / 困难 / 被排斥 / 身体不适 / 精神贫瘠
- **正位含义**：两个衣衫褴褛的人走过亮着灯的教堂窗外——帮助就在那里，但他们没有走进去。这张牌问的是：你是否因为羞耻或骄傲而拒绝了可以获得的帮助？
- **逆位关键词**：恢复 / 接受帮助 / 走出困境 / 灵性转化 / 重新获得安全感
- **逆位含义**：困难时期正在过去，你开始愿意接受支持。物质层面的匮乏可能是一次精神层面的觉醒契机。

## 星币六 Six of Pentacles

- **正位关键词**：慷慨 / 给予与接受 / 公平 / 分享资源 / 互惠
- **正位含义**：一手持天平一手分发金币——给予和接受之间的平衡。问问自己：你在关系中更常扮演给予者还是接受者？两种角色都需要勇气。
- **逆位关键词**：不平等 / 带条件的给予 / 依赖 / 权力不对等 / 自私
- **逆位含义**：给予变成了控制的工具，或者接受帮助让你感到低人一等。给予和接受之间的权力动态需要被审视。

## 星币七 Seven of Pentacles

- **正位关键词**：耐心 / 评估 / 等待收获 / 长期投资 / 反思
- **正位含义**：农夫停下来审视他种下的庄稼——你已经付出了努力，现在需要耐心等待果实成熟。这也是一个评估的时刻：投入的方向是否正确？
- **逆位关键词**：急于求成 / 回报不足 / 错误投资 / 放弃太早 / 缺乏耐心
- **逆位含义**：对收获的速度感到焦虑，或者开始怀疑自己的努力方向是否正确。也许需要调整策略，但不要在果实即将成熟时放弃。

## 星币八 Eight of Pentacles

- **正位关键词**：精进 / 专注 / 技能提升 / 勤勉 / 匠人精神
- **正位含义**：工匠专注地打磨每一枚金币——通过反复练习达到精湛。这是一个需要沉下心来磨练技艺的时期。成功来自日积月累的精进，而非一朝的灵感。
- **逆位关键词**：敷衍了事 / 完美主义 / 重复劳动无意义 / 缺乏热情 / 技术停滞
- **逆位含义**：对工作失去了热情，变成了机械的重复。或者走向完美主义的极端——永远觉得不够好而无法交付。

## 星币九 Nine of Pentacles

- **正位关键词**：独立 / 丰盛 / 自我价值 / 享受成果 / 优雅
- **正位含义**：优雅的女子站在葡萄园中——通过自己的努力创造了物质和精神的丰盛。这是一种自给自足的满足感，不依赖他人来定义自己的价值。
- **逆位关键词**：过度独立 / 物质依赖 / 自我价值怀疑 / 安全感焦虑 / 孤独
- **逆位含义**：独立变成了隔离——不愿让任何人进入自己精心维护的花园。或者物质的丰盛无法填补内心的空虚。

## 星币十 Ten of Pentacles

- **正位关键词**：家族 / 传承 / 长期稳定 / 世代财富 / 根基深厚
- **正位含义**：三代人在家族大宅前——你正在或即将建立持久的基业。这不仅是物质的传承，也是价值观、智慧和爱的代际传递。
- **逆位关键词**：家族问题 / 遗产纠纷 / 不稳定 / 打破家族模式 / 短视
- **逆位含义**：家族中的未解议题浮出水面——也许是需要打破的不健康模式，也许是关于遗产和责任的冲突。

## 星币侍从 Page of Pentacles

- **正位关键词**：学习 / 新技能 / 务实的好奇心 / 机会 / 踏实
- **正位含义**：年轻人专注地审视手中的金币——以认真踏实的态度面对新的学习机会。不急不躁，一步一步地建立自己的能力基础。
- **逆位关键词**：缺乏方向 / 不切实际 / 浪费机会 / 懒散 / 短视
- **逆位含义**：有机会却不够专注或不愿付出努力来把握。也可能是太过务实而失去了梦想的能力。

## 星币骑士 Knight of Pentacles

- **正位关键词**：坚定 / 可靠 / 勤勉 / 稳步前进 / 尽责
- **正位含义**：唯一没有在奔跑的骑士——他骑在静止的马上，耐心而坚定。进度也许缓慢，但每一步都是扎实的。可靠性和耐心是他最大的力量。
- **逆位关键词**：停滞 / 过度保守 / 无聊 / 固执 / 工作狂
- **逆位含义**：稳定变成了僵化，可靠变成了无趣。陷入了日复一日的例行公事，失去了前进的动力和方向感。

## 星币王后 Queen of Pentacles

- **正位关键词**：滋养 / 务实的关怀 / 丰盛 / 与自然连接 / 身体智慧
- **正位含义**：花园中的王后温柔地抱着金币——她以最务实的方式关怀他人：确保你吃饱穿暖，有一个安全的空间。她提醒我们：灵性成长也需要物质基础。
- **逆位关键词**：自我忽视 / 物质主义 / 不安全依赖 / 过度付出 / 与身体断联
- **逆位含义**：照顾所有人唯独忘了自己，或者将物质等同于爱。也可能指向与身体的断联——忽视了身体发出的信号。

## 星币国王 King of Pentacles

- **正位关键词**：富足 / 安全 / 实业 / 可靠的领导 / 物质成就
- **正位含义**：土元素的最高表达——将远见和努力转化为持久的物质成就。他的富足不是偶然的运气，而是智慧经营的结果。他代表一种扎根于大地的稳定力量。
- **逆位关键词**：贪婪 / 物质执着 / 工作至上 / 控制性 / 地位焦虑
- **逆位含义**：对财富和地位的追求遮蔽了生活中其他重要的面向。用物质成就来填补内在的空虚，或以金钱作为控制他人的工具。
