# Example: write-from-anova

## Input

User provides ANOVA results and asks for Chinese Results paragraph:

> "请根据以下 ANOVA 结果写一段中文 Results：因素A 主效应 F(1, 58) = 12.34, p = .001, ηp² = .18；因素B 主效应 F(1, 58) = 3.21, p = .078, ηp² = .05；交互作用 F(1, 58) = 6.78, p = .012, ηp² = .10。"

## Expected Output

```
【结果组织建议】
先报告交互作用（因为交互显著），再分别报告简单效应分析结果。
主效应作为上下文提及但不作为重点展开。

【可直接使用的结果段】
方差分析显示，因素 A 与因素 B 的交互作用显著，F(1, 58) = 6.78, p = .012, ηp² = .10。
简单效应分析进一步表明，在因素 B 的水平 1 下，因素 A 的效应显著……（需补充简单效应结果）。
因素 A 的主效应显著，F(1, 58) = 12.34, p = .001, ηp² = .18；因素 B 的主效应未达到显著水平，F(1, 58) = 3.21, p = .078, ηp² = .05。

【统计报告检查】
- ✅ F、df、p、ηp² 均报告
- ✅ 交互作用优先报告
- ⚠️ 因素 B 主效应 p = .078 → 已使用"未达到显著水平"，未写"无差异"
- ⚠️ 缺少简单效应分析的具体统计值。如用户提供，可在"进一步表明"后补充

【结果与讨论边界提醒】
- 不要写"因素 A 影响……" → 根据实验设计，可用"因素 A 的操纵影响了……"
- 因素 B 不显著不能写"因素 B 无作用"
- 交互作用的方向和模式解释应放入 Discussion

【可选替代表达】
- "因素 B 的主效应接近但未达到传统显著性水平，p = .078"
- 如果后续有 post-hoc：可考虑"事后比较（Bonferroni 校正）显示……"
```
