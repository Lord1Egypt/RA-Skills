## Description: <br>
Creates Baiyin Open Platform digital-human video tasks, checks task status, and retrieves final video results for lip-sync and body-driven digital-human models. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[jiuping520](https://clawhub.ai/user/jiuping520) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External users and developers use this skill to create digital-human videos through the Baiyin Open Platform, upload required face or audio media when needed, select an available model, and check task status or result links. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill requires a Baiyin API key and may upload selected face or audio media to the provider. <br>
Mitigation: Use only approved credentials, confirm user intent before uploading media, and avoid sending sensitive media unless the user has authorized that transfer. <br>
Risk: The security summary reports silent remote version checks and possible local self-updates before the requested task runs. <br>
Mitigation: Prefer a release that requires explicit approval before changing installed instructions, or review the installed skill version before execution. <br>
