### NORMAL

• **NORMAL** = ``0``

普通态： 更新定位数据时不对地图做任何操作

___

### FOLLOWING

• **FOLLOWING** = ``1``

跟随态，保持定位图标在地图中心

___

### COMPASS

• **COMPASS** = ``2``

罗盘态，显示定位方向圈，保持定位图标在地图中心
