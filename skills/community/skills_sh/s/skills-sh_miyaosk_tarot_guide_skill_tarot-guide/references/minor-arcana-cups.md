# 圣杯牌组 Cups

- **元素**：水
- **领域**：情感、关系、爱、直觉、潜意识、内在世界
- **心理维度**：情感功能（Feeling）—— 心灵的流动与深度

---

## 圣杯王牌 Ace of Cups

- **正位关键词**：爱的涌现 / 情感新开始 / 直觉觉醒 / 慈悲 / 灵性连接
- **正位含义**：心灵的杯子被充满——一种新的情感体验正在涌入。可能是爱、创造灵感或深层的内在平静。允许自己接受这份馈赠。
- **逆位关键词**：情感封闭 / 空虚 / 被压抑的情感 / 自爱缺失 / 情感枯竭
- **逆位含义**：心灵的杯子被翻转，水流了出去。你可能正在阻挡情感的流动——害怕去爱，或害怕被爱。

## 圣杯二 Two of Cups

- **正位关键词**：连接 / 伙伴关系 / 相互吸引 / 和谐 / 情感交流
- **正位含义**：两颗心的真诚相遇——不仅是浪漫关系，也代表任何形式的深层连接。两个杯子的交汇象征着平等、尊重的情感交换。
- **逆位关键词**：关系失衡 / 断裂 / 误解 / 不平等的付出 / 与自我断联
- **逆位含义**：关系中的给予与接受失去了平衡，或者与自己内在的对立面（阿尼玛/阿尼姆斯）断开了连接。

## 圣杯三 Three of Cups

- **正位关键词**：庆祝 / 友谊 / 社群 / 共享喜悦 / 创造性合作
- **正位含义**：与他人共享快乐的时刻。三只杯子高高举起，象征情感的丰盛与人际连接的滋养。归属感和被接纳的喜悦。
- **逆位关键词**：社交过度 / 群体压力 / 排斥感 / 放纵 / 表面社交
- **逆位含义**：社交活动变成了逃避内在感受的方式，或在群体中感到格格不入。需要区分真正的连接和表面的热闹。

## 圣杯四 Four of Cups

- **正位关键词**：冷漠 / 内省 / 不满足 / 被忽视的机会 / 情感退缩
- **正位含义**：身边的三只杯子已无法满足你，但你也没有注意到云中伸出的第四只杯子。内在的不满可能是一种信号——你的灵魂渴望更深层的东西。
- **逆位关键词**：觉察新机会 / 走出冷漠 / 重新参与 / 感恩 / 接受
- **逆位含义**：从情感的麻木中醒来，开始看到之前忽略的可能性。重新投入生活的意愿正在回归。

## 圣杯五 Five of Cups

- **正位关键词**：失落 / 悲伤 / 遗憾 / 执着于失去 / 未完成的哀悼
- **正位含义**：三只杯子倒了，你的全部注意力都在失去的东西上。但身后还有两只完好的杯子——悲伤是必要的，但不要让它遮住了仍然存在的祝福。
- **逆位关键词**：接受失去 / 开始疗愈 / 转身看到希望 / 放下遗憾 / 从悲伤中走出
- **逆位含义**：悲伤的过程正在完成，你终于转身看到了那两只仍然完好的杯子。疗愈不是忘记，而是学会与失去共存。

## 圣杯六 Six of Cups

- **正位关键词**：怀旧 / 童年记忆 / 纯真 / 善意 / 重逢 / 内在小孩
- **正位含义**：与内在小孩的重新连接。过去的美好记忆正在疗愈当下。有时也指向一个问题的根源在童年——需要带着成人的智慧回顾那段时光。
- **逆位关键词**：困在过去 / 理想化回忆 / 拒绝长大 / 童年创伤 / 怀旧过度
- **逆位含义**：将过去理想化，用怀旧来逃避当下的挑战。或者童年的伤痕正在影响成年后的关系模式，需要被正视。

## 圣杯七 Seven of Cups

- **正位关键词**：幻想 / 选择过多 / 白日梦 / 诱惑 / 投射
- **正位含义**：七只杯子中盛满了各种诱人的幻象——但哪些是真实的，哪些是幻觉？这张牌提醒我们区分真正的渴望和心理投射，区分直觉和一厢情愿。
- **逆位关键词**：清晰抉择 / 脚踏实地 / 打破幻想 / 聚焦 / 面对现实
- **逆位含义**：迷雾散去，你开始分辨哪些愿望是真实的，哪些只是逃避现实的幻想。做出了基于现实而非幻象的选择。

## 圣杯八 Eight of Cups

- **正位关键词**：离开 / 寻求更深的意义 / 放弃已有的 / 内在召唤 / 勇敢告别
- **正位含义**：八只杯子整齐排列，但你转身走向月光下的山路。不是因为杯子空了，而是因为它们已经无法满足灵魂更深的渴望。这是一次勇敢的精神出走。
- **逆位关键词**：害怕离开 / 安于不满 / 逃避成长 / 漂泊无根 / 反复离开
- **逆位含义**：明知应该离开却不敢迈出那一步，或者陷入了反复"出走"的模式——总是在离开，却不知道自己在寻找什么。

## 圣杯九 Nine of Cups

- **正位关键词**：满足 / 愿望实现 / 幸福 / 感恩 / 情感充盈
- **正位含义**：九只杯子排列在身后如同荣耀的弧线。你的情感需求得到了满足，这是一个享受当下成果的时刻。真正的满足来自内心，而非外在。
- **逆位关键词**：不满足 / 贪婪 / 空虚的拥有 / 物质主义 / 真正需要被忽略
- **逆位含义**：拥有了很多却仍感空虚——也许追求的是表面的满足而非灵魂真正的渴望。需要重新审视"我真正需要什么"。

## 圣杯十 Ten of Cups

- **正位关键词**：家庭和谐 / 情感圆满 / 幸福 / 归属 / 爱的完成
- **正位含义**：情感世界的最高理想——彩虹下的圆满。这不仅是外在家庭的和谐，更是内在各个情感面向的整合与安宁。
- **逆位关键词**：家庭不和 / 理想破灭 / 情感断裂 / 对幸福的错误定义 / 不切实际的期望
- **逆位含义**：对"完美幸福"的执着反而破坏了真实的关系。也许需要重新定义什么是"足够好"的家庭和情感生活。

## 圣杯侍从 Page of Cups

- **正位关键词**：情感萌芽 / 直觉信息 / 创造力 / 温柔 / 内在小孩的声音
- **正位含义**：一个来自潜意识的温柔信息——也许是一个梦、一个灵感、一份突如其来的感动。像孩子一样对情感世界保持开放和好奇。
- **逆位关键词**：情绪化 / 幼稚 / 逃入幻想 / 情感不成熟 / 忽视直觉
- **逆位含义**：情感表达停留在孩童水平，或以白日梦逃避现实的情感需求。

## 圣杯骑士 Knight of Cups

- **正位关键词**：浪漫 / 追求理想 / 情感驱动 / 优雅 / 提议
- **正位含义**：带着杯子的骑士缓缓而来——他不是冲锋陷阵，而是以优雅和真诚追求内心的理想。跟随心的方向，但不失行动力。
- **逆位关键词**：不切实际 / 情感操控 / 虚假浪漫 / 善变 / 逃避现实
- **逆位含义**：以"跟随心灵"为名逃避责任，或用浪漫的外表掩盖情感上的不真诚。美丽的包装里可能是空的。

## 圣杯王后 Queen of Cups

- **正位关键词**：共情 / 直觉深邃 / 情感智慧 / 滋养 / 潜意识的主人
- **正位含义**：水元素的最高女性表达——她能感受他人的感受，读懂潜意识的语言，同时保持自身的情感稳定。她的杯子是封闭的——她能共情但不会被他人的情绪淹没。
- **逆位关键词**：情感边界丧失 / 过度共情 / 情绪淹没 / 殉道 / 与直觉断联
- **逆位含义**：杯子打开了，别人的情绪涌入没有边界。过度共情导致自我丧失，或者反过来——为了保护自己而完全封闭情感。

## 圣杯国王 King of Cups

- **正位关键词**：情感成熟 / 沉稳 / 智慧的慈悲 / 外交 / 情绪掌控
- **正位含义**：坐在波涛汹涌的大海中却保持平静——情感的成熟不是没有感受，而是能在感受的风暴中保持中心。他将情感智慧转化为对他人的引导和支持。
- **逆位关键词**：情感压抑 / 操控 / 情绪爆发 / 冷漠伪装 / 酗酒/逃避
- **逆位含义**：以"理性"和"冷静"压抑情感，直到某天堤坝崩溃。或以情感智慧操控他人而非真诚连接。
