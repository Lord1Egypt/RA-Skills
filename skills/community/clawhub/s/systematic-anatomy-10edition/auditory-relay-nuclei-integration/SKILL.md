# 听觉中继核的功能整合

## 适用对象
- 上橄榄核
- 外侧丘系核
- 下丘核
- 蜗神经核
- 外侧丘系

## 前提条件
- 双侧听觉通路完整
- 脑干听觉核团结构正常

## 使用限制
- 不适用于非听觉感觉（如视觉或平衡觉）
- 不涉及皮质高级听觉处理

## 执行步骤

### 1. 蜗神经核（初级中继）
- **位置**：菱形窝听结节深面，分为腹侧核（含前、后亚核）和背侧核
- **功能**：接受蜗神经初级听觉纤维
- **传出路径**：
  a) 大部分纤维在脑桥基底与被盖之间形成斜方体 → 越中线交叉 → 对侧被盖腹外侧 → 上橄榄核外侧转折上升
  b) 小部分纤维不交叉，同侧上行
  c) 交叉与未交叉纤维共同构成外侧丘系

### 2. 上橄榄核（空间定位关键）
- **位置**：脑桥中下部被盖腹侧，内侧丘系背外侧
- **输入**：接受双侧蜗腹侧前核传出纤维
- **功能**：与蜗腹侧前核协同，依据双耳接收声音的「时间差」和「强度差」实现声音空间定位
- **输出**：发出纤维加入双侧外侧丘系

### 3. 外侧丘系核
- **位置**：自脑桥中下部至中脑尾侧，伴随外侧丘系分布
- **结构变化**：
  - 上橄榄核上方：散在于外侧丘系背内侧
  - 脑桥上部：被外侧丘系环绕
- **输入**：接受蜗腹侧前核及外侧丘系纤维
- **输出**：发出纤维越边（cross midline），加入对侧外侧丘系

### 4. 下丘核（高级中继与反射中枢）
- **中央核**：接受外侧丘系纤维 → 传出经下丘臂至内侧膝状体
  - 分层结构具音频定位功能：腹侧部处理高频声波，背侧部处理低频声波
- **下丘周灰质**：接受中央核、内侧膝状体、听皮质和小脑输入 → 参与听觉负反馈调节和声源定位
- **听觉反射**：发出纤维至上丘深部 → 经顶盖脊髓束完成头眼转向声源的「听觉惊恐反应」

## 触发条件
当任务涉及以下任一情况时应用本技能：
- 分析声音的空间定位机制
- 解释听觉反射的神经通路
- 阐述听觉传导通路中的中继整合过程
