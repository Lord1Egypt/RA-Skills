## Description: <br>
Helps frontend teams diagnose and fix mobile responsiveness, navigation layout, and cross-device UI issues with tailored guidance, checklists, workflows, and code-support artifacts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kyro-ma](https://clawhub.ai/user/kyro-ma) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Frontend developers, designers, startup teams, and maintainers use this skill to turn mobile layout, responsive navigation, and cross-device usability requests into actionable fixes, checklists, workflows, code changes, and verification notes. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad frontend and layout wording may cause the skill to activate on requests better handled by another specialized skill. <br>
Mitigation: Explicitly name the intended skill or disable implicit invocation when precise routing matters. <br>
Risk: Responsive layout guidance or code changes may not match the user's actual device, breakpoint, or framework constraints. <br>
Mitigation: Review generated changes against stated success criteria and verify them on target mobile and desktop viewports before deployment. <br>
