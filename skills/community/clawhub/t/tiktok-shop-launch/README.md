# TikTok Shop 出海助手

> 全球本土化矩阵，助力卖家开拓流量新大陆

## 🎯 核心价值

**从"搬运视频"升维到"全球本土化矩阵"**

帮助跨境卖家制定区域化、本土化的TikTok Shop运营策略，覆盖美区、东南亚、欧洲、日本四大核心市场。

## ✨ 功能矩阵

| 功能 | 说明 |
|------|------|
| 🔍 **区域化策略** | 根据目标市场提供差异化运营路线 |
| 📝 **内容本地化** | 视频风格/文案调性/视觉偏好建议 |
| 🎬 **直播脚本** | 适配目标市场的直播带货脚本框架 |
| ✅ **合规准入** | 电商法规/税务要求/资质清单 |
| 💰 **投流策略** | 投放组合/预算分配/效果评估 |

## 🌍 市场策略

| 区域 | 核心路线 | 策略重点 |
|------|----------|----------|
| 🇺🇸 **美区** | 品牌故事路线 | 美式价值观、品牌叙事、个性/环保/家庭主题 |
| 🌏 **东南亚** | 直播驱动路线 | 直播带货、价格敏感型选品、本土语言 |
| 🇪🇺 **欧洲** | 合规准入路线 | GDPR合规、电商法规、品质导向内容 |
| 🇯🇵 **日本** | 细节信赖路线 | 信赖建立、品质细节、文化适配 |

## 📥 使用方法

### 输入格式

```
产品描述：[描述产品功能、特点、卖点]
目标市场：[美区/东南亚/欧洲/日本，可多选]
预算范围：（可选）[如：$1000/月]
```

### 示例

```
产品描述：便携式咖啡机，轻便易用，适合上班族
目标市场：美区
预算：$2000/月
```

## 📤 输出内容

1. **市场洞察** - 目标市场特点分析
2. **选品建议** - 适合该市场的产品卖点提炼
3. **内容策略** - 本土化内容方向建议
4. **直播规划** - 直播脚本框架（如适用）
5. **合规要点** - 需要关注的法规/资质
6. **投流建议** - 预算分配和投放节奏

## ⚠️ 免责声明

- 所有建议仅供参考
- 实际运营需遵守当地法规和平台规则
- 内容创作需符合平台内容规范
- 跨境电商需关注目标市场的税务和海关要求
