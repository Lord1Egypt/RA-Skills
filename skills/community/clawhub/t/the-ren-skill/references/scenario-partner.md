# 场景三：识伙伴（合伙人·投资人·供应商·战略联盟）

> 这是商业决策中最"重"的场景——选错伙伴的代价可能比选错员工大100倍。
>
> **⚡ 协作：** 先用 `personality-model.md` 六型画像定位，再用本文件做合伙人专项判断。
> 合伙场景的六型推荐和禁忌见下方。

---

## 一、合伙人识人（最重决策）

**合伙人的本质是"婚姻"——一旦绑定，分手成本极高。**

### 合伙人六型适配指南

**不同类型的人适合做不同类型的合伙人。不是所有类型都适合当合伙人。**

| 你的类型 | 最佳合伙人类型 | 为什么 | 警惕 |
|---|---|---|---|
| 将帅型CEO | 谋士型（技术/产品）+ 基石型（运营） | 将帅定方向，谋士做产品，基石守运营 | 另一个将帅→权力斗争 |
| 先锋型CEO | 将帅×谋士型（战略）+ 基石型（执行） | 先锋冲市场，需要有人拉住他+有人接住 | 另一个先锋→两个人都只顾往前冲 |
| 谋士型CEO | 先锋×基石型（商务+运营） | 谋士做产品，需要有人把产品卖出去并稳住公司 | 另一个谋士→都在思考没人执行 |
| 基石型CEO | 先锋型（市场）+ 谋士型（产品） | 基石做运营，但需要有人开拓和有人做产品 | 另一个基石→稳如泰山但永远长不大 |

**合伙人团队的三条铁律：**

1. **至少覆盖"拍板的+做出来的+搞钱的"三个角色。** 缺任何一个，公司大概率卡在某个阶段。
2. **不能有两个同类型的人占据同一角色。** 两个将帅=必有一战。两个谋士=产品永远在打磨。两个先锋=承诺满天飞。
3. **人格类型可以互补，价值观必须一致。** 类型互补让你跑得快，价值观分裂让你死得快。

### 合伙人的六型红线

| 类型 | 适合合伙吗 | 条件 |
|---|---|---|
| 将帅型 | ✅ 适合（CEO位） | 必须通过伪将帅测试（倒序讲述+质疑反应） |
| 先锋型 | ⚠️ 可合伙（商务/市场位） | 必须配一个基石型接住他的承诺，否则公司会被过度承诺拖垮 |
| 谋士型 | ✅ 适合（CTO/CPO位） | 必须接受"他的判断通常是对的但不好听"，且他需要独处时间 |
| 基石型 | ✅ 适合（COO位） | 最容易错过的合伙人类型——面试不发光但合伙后最稳 |
| 成长型 | ❌ 不建议作为联合创始人 | 成长型需要时间，联合创始人需要"立即可用"。但可以作为早期员工重点培养 |
| 跨界型 | ✅ 稀缺资产 | 将帅×谋士=技术型CEO，先锋×基石=能签单能交付的合伙人 |
| 伪将帅 | 🔴 绝对不行 | 能力极强+诚信低=有能力作大恶。这是合伙人的头号杀手 |

### 合伙前必查的四维+面相

#### 维度0：见面第一感的"直觉账"

**最高优先。** 如果你和一个潜在的合伙人第一次深入交流后有"说不上来但哪里不对"的感觉——**相信你的直觉**。你的潜意识可能捕捉到了你自己没意识到的微表情/微体态信号。

#### 维度1：利益分配的坦诚度

| 信号 ✅ | 信号 ❌ |
|---|---|
| 大方聊股权/分配，愿意说具体的数字和比例 | 回避利益话题，"我们先做事，利益好说" |
| 谈利益时眼神稳定，嘴角自然 | 谈利益时眼神飘忽，频繁摸鼻子 |
| 能说出"你需要拿X%，因为……"的合理理由 | "我们五五分吧"随口而出（没经过认真思考） |
| 愿意把约定写下来 | 抗拒书面化，"我们兄弟还签什么协议" |

#### 维度2：格局与胸怀

| 信号 ✅ | 信号 ❌ |
|---|---|
| 客观评价竞争对手，能从对手身上看到可学之处 | 贬低、嘲讽所有竞争对手 |
| 把成功归因于团队和运气 | 把成功全归因于自己"看准了/做对了" |
| 对前合伙人/前合作伙伴的客观描述 | 一说前合伙人就激动/全是对方的错 |
| 印堂开阔，眉心舒展 | 印堂窄/悬针纹深+钻牛角尖 |

#### 维度3：执行力与"说到做到"

**面相方法：看"长期行为痕迹"**
- 观察他过往2-3年的面相变化：面相在变好（面色清亮、眼神更定、姿态更舒展）=他在做正向积累→靠谱
- 面向在变差（面色暗沉、眼神浑浊、姿态萎缩）=他在走下坡路→合伙后你可能要替他填坑
- 对过往承诺的兑现度：让他举3个"说到做到"的具体例子，观察描述时的眼神和语速（真的做到→自然描述；编的→认知负荷超载）

#### 维度4：逆商（在低谷时的表现）

| 测试方法 | 信号 ✅ | 信号 ❌ |
|---|---|---|
| 问"你最困难的时候怎么过来的" | 具体描述过程+学到了什么 | 模糊/"都过去了不想说" |
| 问"如果这次创业失败了怎么办" | 理性分析风险+有Plan B | 要么回避/要么"不可能失败"（盲目乐观） |
| 描述困难时的眼神 | 坚定+有反思的痕迹（眼神略微内视） | 飘忽/愤怒（还在怨恨）/麻木 |

### 合伙人面相信号特别篇

合伙人的面相有几个特别重要的区域：

**1. 眼神——最重要，没有之一**
- 合伙人的眼神必须"定"。不是犀利，是定。
- 如果他的眼神让你感觉"看不透"——不是"深邃"的看不透，而是"不知道他在想什么"的看不透——要极度警惕。合伙人之间需要的是透明，不是神秘。
- 和他聊未来3-5年的愿景时，观察他眼神是否发亮。如果发亮=真的有热情。如果眼神不变=只是觉得这个生意能赚钱。

**2. 印堂——格局的生理印记**
- 印堂开阔+自然舒展=能容人、能放权
- 印堂狭窄+皱眉肌紧张=大概率是个control freak，合伙后事事要插手

**3. 法令纹——真实笑容的痕迹**
- 法令纹向外上扬=这个人是真的经常笑→团队会被他带动
- 向下弯的法令纹=长期不快乐→合伙后负能量会传染整个公司

## 二、投资人评估（拿谁的钱很重要）

### 好投资人 vs 坏投资人

| 观察点 | 好投资人 ✅ | 坏投资人 ❌ |
|---|---|---|
| 谈被投企业 | "XX做得不错，尤其是在YY方面" | "那家公司不行/那个创始人不懂" |
| 谈自己的角色 | "我的价值是……"具体能帮什么 | "我认识很多人/资源很多"（模糊承诺） |
| 谈失败案例 | 坦诚聊，分析原因 | 回避/说自己从没失败过 |
| 眼神 | 和你交流时自然对视 | 看手机比看你多 |
| 握手 | 力度适中，不表演 | 用力过猛或不用力 |
| 对你行业的理解 | 问的问题在点上 | 问的问题让你觉得"你不懂" |
| 谈判时的姿态 | 开放，愿意被你反问 | 居高临下，"你应该感谢我给你钱" |

### 投资人面相信号

- **面色清亮+眼神有光：** 这个投资人自己的状态很好，不会因为自己的焦虑而过度干预你
- **面色暗沉+眼神焦虑：** 他可能自己的基金状况不好，投你是为了翻盘——后续会给你巨大压力
- **眉头习惯性紧锁：** 大概率是个micromanager型投资人，投后会频繁干预

## 三、供应商/战略联盟评估

### 长期供应商可靠性判断

| 观察点 | 可靠信号 ✅ | 不可靠信号 ❌ |
|---|---|---|
| 对合同的认真度 | 逐条看，提出合理疑问 | 扫一眼就签（要么不care，要么不打算履约） |
| 谈过往客户 | 客观评价，不说坏话 | 吐槽前客户（将来也会吐槽你） |
| 交货时间承诺 | 给出保守估计+缓冲 | "没问题肯定能按时"（过度承诺） |
| 报价 | 透明，给出成本结构 | 模糊，回避成本细节 |
| 出现问题时的反应 | 主动承认+给出补救方案 | 先推卸责任再想办法 |
| 企业主的面向 | 面相舒展，气色好 | 面相紧绷，面色差（企业可能在走下坡路） |
| 员工状态 | 员工面带笑容，工作区有序 | 员工面色暗沉，怕老板 |

### 战略联盟/合资伙伴选择

除了和"合伙人识人"相同的四维检查外，额外注意：

**"对等感"检查：**
- 双方交流时，谁的姿态更占空间？如果一方始终在缩小身体（蜷缩/抱胸/小声音），合作从一开始就是不对等的。
- 对等的合作：双方都舒展、都敢直视、都敢说不。
- 不对等的合作：一方在俯视，一方在仰视。这种合作迟早出问题。

**"共情匹配"检查：**
- 双方在交流中的"变色龙效应"是否双向？如果只有你在模仿他而他从不模仿你→他对这段关系没有真正的共情→可能只是短期利用。

## 四、伙伴关系的"面相动态监控"

**核心逻辑：面相在变，关系在变。定期"扫描"你的关键伙伴。**

### 每季度/每半年观察一次：

| 变化 | 含义 | 行动 |
|---|---|---|
| 对方面相在变好 | 他的生意/状态在改善 | 加深合作 |
| 对方面相在变差 | 可能有你不知道的问题 | 增加沟通频率，了解发生了什么事 |
| 对方对你的镜像反应减弱 | 他对这段关系的投入感在降低 | 约一次深入交流，了解是否有不满 |
| 对方的眼神从"定"变"飘" | 可能在隐瞒什么 | 增加信息验证渠道，不要只听他说 |
| 对方的姿态从舒展变防御 | 可能在你们的关系中感到不安 | 主动询问是否有什么需要聊的 |

**最重要的预警：如果对方开始回避和你眼神接触——这是所有关系中最可靠的"出问题"信号。不要忽略。**
