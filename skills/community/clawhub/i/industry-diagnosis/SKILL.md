# 行业诊断 (Industry Diagnosis)

## 概述

覆盖全部 **31 个申万一级行业**，根据各行业独特的产业链结构，定制化分析框架。
对超出申万一级的细分行业，可动态推导产业链结构并生成分析指标。

## 工作流程

### Step 1: 确定行业归属

1. 确认该行业是否属于申万一级行业 → 使用对应行业模板
2. 若为细分行业（如"光伏"是电力设备的下级）→ 使用母行业框架 + 细分补充
3. 若为跨行业新产业（如"AI+医药"）→ 使用动态分析

### Step 2: 产业链结构拆解

按行业模板中的**产业链结构图**，将行业拆分为上游/中游/下游三个环节，
识别每个环节的核心参与者（公司或商品）。

### Step 3: 采集核心指标

根据模板中定义的各环节核心指标，通过以下工具采集数据：

- **价格/供需指标** → mx-finance-data（商品期货、板块行情）
- **公司财务指标** → mx-finance-data（营收/利润/库存/产能利用率）
- **行业销量/产量** → mx-stocks-screener（板块龙头数据）
- **政策/国际数据** → web_search / web_fetch（海关总署、能源局、协会数据）
- **价量技术指标** → mx-finance-data（板块指数、资金流向、估值水位、技术指标）

### Step 4: 执行独特分析逻辑

按行业模板中的"独特分析逻辑"执行 — 每个行业都有独立的分析框架：
- 有的侧重库存周期（如半导体、白酒）
- 有的侧重原料价格传导（如钢铁、化工）
- 有的侧重政策驱动（如新能源、军工）
- 有的侧重品牌/渠道（如消费、医药）

### Step 5: 综合诊断输出格式（严格模板 — 不可省略任何子项）

**每次输出必须严格按照下面的模板格式，所有子项必须逐项输出，不可合并、不可跳过、不可用"同上"/"参考上文"替代。**

**标题行：**
```
# 🏷️ 行业诊断：XXX（申万一级/细分行业）
**YYYY-MM-DD**
```
