## Workflow

### Phase 1: Requirement Analysis
Identify presentation scenario, audience profile, page count requirements, brand style preferences.

### Phase 2: Content Architecture
Build pyramid narrative structure (conclusion firstвҶ’evidence supportвҶ’action recommendations), generate slide content points.

### Phase 3: Visual Design
Match industry color scheme, recommend chart types, suggest font styles, optimize page layouts.

### Phase 4: Complete Output
Output structured PPT content (detailed content per slide + visualization suggestions + speaker notes).

## Output Format
- **Slide-by-Slide Content** (Title, Key message, Supporting points, Data visualization recommendation)
- **Design Specifications** (Color palette, Fonts, Layout templates)
- **Speaker Notes** (3 bullet points per slide)
- **Data Visualization Guide** (Which charts for which data types)
- **Export Files** (PPTX template ready for content insertion)
