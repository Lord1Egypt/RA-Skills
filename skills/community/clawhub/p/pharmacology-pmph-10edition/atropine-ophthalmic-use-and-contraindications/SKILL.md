# 阿托品眼科应用与禁忌规则

## 何时使用本技能
- 患者需要进行**验光**或**眼底检查**
- 患者被诊断为**虹膜睫状体炎**
- 需评估患者是否适合使用阿托品（尤其关注青光眼病史和年龄）

## 执行步骤

1. **确认禁忌证**：
   - 若患者有**青光眼病史**（尤其是急性闭角型青光眼），**禁止使用阿托品**。
   - 前列腺增生患者慎用（因全身吸收可能加重排尿困难）。

2. **判断适用人群**：
   - **儿童验光必须使用阿托品**，因其睫状肌调节力强，短效药物无法充分麻痹。
   - 成人验光或眼底检查可考虑阿托品，但因作用时间长（7–12天），现多优先选用后马托品或托吡卡胺。

3. **确定临床用途与浓度**：
   - **虹膜睫状体炎**：使用0.5%–1%阿托品滴眼，每日1–3次，以松弛虹膜与睫状肌。
   - **验光/眼底检查**：使用1%阿托品滴眼，通常每日2–3次，连续3天以达充分调节麻痹与扩瞳效果。

4. **告知患者注意事项**：
   - 视近模糊可持续7–12天，避免驾驶或精细作业。
   - 瞳孔扩大期间畏光明显，建议佩戴墨镜。

> ⚠️ **关键原则**：无青光眼 + 有明确适应证 → 可用；有青光眼 → 绝对禁用。
