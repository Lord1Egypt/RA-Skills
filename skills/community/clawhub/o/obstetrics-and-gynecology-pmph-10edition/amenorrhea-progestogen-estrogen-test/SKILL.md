# 闭经诊断中的孕激素与雌孕激素序贯试验

## 适用对象
- 闭经患者
- 疑似低雌激素状态的女性
- 孕激素试验阴性且已排除妊娠的闭经患者

## 使用前提
- 已排除妊娠
- 患者无活动性阴道出血
- 无禁忌证（如血栓高风险、急性肝病）
- 不适用于已知无子宫或流出道完全梗阻者

## 执行流程

### 第一步：孕激素试验
1. **选择任一孕激素方案**（连续用药）：
   - 黄体酮注射液：20 mg/次，肌注，每日1次，连用3日
   - 醋酸甲羟孕酮：10 mg/次，口服，每日1次，连用7–10日
   - 地屈孕酮：20 mg/次，口服，每日1次，连用7–10日
   - 微粒化黄体酮：100 mg/次，口服，每日2–3次，连用7–10日
   - 黄体酮凝胶：90 mg/次，阴道给药，每日1次，连用10日

2. **停药后观察14天内是否出现撤退性出血**：
   - **阳性**（有出血）：提示内源性雌激素水平足够，子宫内膜反应正常，生殖道通畅
   - **阴性**（无出血）：可能为低雌激素、子宫性病变（如Asherman综合征）或下生殖道梗阻

3. **若阴性且已排除妊娠**，进入第二步：雌孕激素序贯试验

### 第二步：雌孕激素序贯试验
1. **雌激素阶段**：连续21日每晚口服以下任一雌激素：
   - 戊酸雌二醇 2 mg
   - 17β-雌二醇 2 mg
   - 结合雌激素 1.25 mg
   - 或使用固定复方制剂（如雌二醇/地屈孕酮复合片）

2. **孕激素叠加阶段**：在最后10日同时加用孕激素：
   - 地屈孕酮 10–20 mg/日，口服
   - 醋酸甲羟孕酮 6–10 mg/日，口服

3. **停药后观察14天内是否出现撤退性出血**：
   - **阳性**：子宫内膜功能正常，闭因为低雌激素（如下丘脑-垂体-卵巢轴功能障碍）
   - **阴性**：重复一次完整试验；若仍无出血，考虑子宫性或下生殖道性闭经

4. **辅助评估**：结合妇科超声（子宫内膜厚度、宫腔形态）及基础性激素（FSH、LH、E2）提高诊断准确性

## 判读要点
- 孕激素试验阳性 → 内源性雌激素充足，无需序贯试验
- 孕激素试验阴性 + 序贯试验阳性 → 功能性闭经（低雌激素）
- 孕激素试验阴性 + 序贯试验阴性（重复确认） → 器质性闭经（子宫或流出道病变）
