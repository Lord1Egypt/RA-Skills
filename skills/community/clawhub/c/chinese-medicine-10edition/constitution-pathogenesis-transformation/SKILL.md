# Constitution and Pathogenesis Transformation

## When to Use
- 患者已感受外邪或出现内生病理产物；
- 已明确患者的中医体质类型（如阳虚、阴虚、气虚、湿盛等）；
- 已知致病邪气性质（如风寒、湿邪、燥邪等）；
- 处于辨证论治或慢性病调理阶段，需预判病情演变趋势。

## Execution Steps
1. **确认前提条件**：确保已知患者体质类型与邪气性质。
2. **判断从化方向**：依据以下规律预测病机转化：
   - **素体偏阳或阳盛者** → 多从热化；
   - **素体偏阴或阳虚者** → 多从寒化；
   - **素体津亏血耗者** → 多从燥化；
   - **素体气虚湿盛者** → 多从湿化。
3. **结合具体邪气分析**：
   - 同感风寒：阳质易化热，阴质易化寒；
   - 同感湿邪：阳质转湿热，阴质转寒湿。
4. **输出预警建议**：提示可能的证候转化方向，并警示避免误用相反药性（如阳虚者慎用寒凉药物）。
5. **指导治疗方向**：强调“因人制宜”，将体质作为调整治法的核心依据。

> 注意：本技能不适用于急诊处理或缺乏体质信息的场景。
