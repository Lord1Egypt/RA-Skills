# 集数识别正则表达式参考

本文件供 AI 在处理集数识别时参考，列出各类常见格式的正则表达式。

---

## 带季号的完整格式

```regex
# S01E01 / s01e01 / S1E1（最常见）
[Ss](\d{1,2})[Ee](\d{1,3})

# 1x01（西剧常见）
(\d{1,2})x(\d{2,3})

# S01E01-E03（多集合并）
[Ss](\d{1,2})[Ee](\d{1,3})[-~][Ee](\d{1,3})

# S01E01E02（连写多集）
[Ss](\d{1,2})[Ee](\d{1,3})[Ee](\d{1,3})
```

## 仅集号格式（无季号）

```regex
# EP01 / Ep.01 / E01
[Ee][Pp]?\.?(\d{1,3})

# 第01集 / 第1集
第\s*(\d{1,3})\s*集

# 01话 / 第01话（动漫常见）
第?\s*(\d{1,3})\s*[话話]

# Episode 01（英文完整写法）
[Ee]pisode\s+(\d{1,3})

# 纯数字（需排除年份等干扰，通常两到三位）
(?<!\d)(\d{2,3})(?!\d)
```

## 季号单独识别

```regex
# Season 01 / Season 1（文件夹名）
[Ss]eason\s+(\d{1,2})

# S01（简写）
[Ss](\d{1,2})(?![Ee])

# 第一季/第二季（中文，需映射）
第([一二三四五六七八九十百]+)季

# 第1季/第01季
第(\d{1,2})季
```

## 特别篇关键词正则

```regex
(?i)(特别篇|SP\d*|Special|OVA\d*|OAD|番外|花絮|幕后|Making\s*Of|预告|Trailer|PV\d*|\bOP\d*\b|\bED\d*\b|NCOP|NCED|采访|访谈)
```

## 识别优先级

1. 文件名中的 `SxxExx` 格式（最高优先级）
2. 文件名中的 `NxNN` 格式
3. 父文件夹名称中的季号信息
4. 文件名中的纯集号格式（`EP01`、`第01集`等）
5. 纯数字（最低优先级，可能误判）

## 集名称提取规则

当文件名中集号标识之后、噪音标签之前存在文字时，该文字应作为集名称保留。

### 提取步骤

1. **定位集号结束位置**：找到 `第N话`、`第N集`、`EP01`、`E01`、`S01E01` 等集号标识的结束位置
2. **截取集号后文字**：从集号结束后的空格/下划线/横线之后开始，到噪音标签之前结束
3. **剥离噪音**：移除画质、编码、发布组等标签（见 SKILL.md 噪音排除规则）
4. **验证**：剩余文字若非空且长度合理（1-50字符），则作为集名称

### 提取正则参考

```regex
# 第N话 后的集名称
第\s*\d{1,3}\s*[话話]\s+(.+?)(?:\s*[_\-\s]\s*(?:1080[pP]|720[pP]|4K|2160[pP]|x264|x265|HEVC|H\.26[45]|WEB-?DL|WEBRip|BluRay|BDRip|HDTV|AAC|DDP|DTS|HDR|DV|10bit|_高清|\[|$))

# S01E01 后的集名称（英文常见）
[Ss]\d{1,2}[Ee]\d{1,3}\s*[-–—]\s*(.+?)(?:\s*[_\-\s]\s*(?:1080[pP]|720[pP]|\[|$))

# EP01 / E01 后的集名称
[Ee][Pp]?\.?\d{1,3}\s+(.+?)(?:\s*[_\-\s]\s*(?:1080[pP]|720[pP]|\[|$))

# 纯数字集号后的集名称（如 "01 集名称"）
(?:^|[\s_])(\d{1,3})\s+(.+?)(?:\s*[_\-\s]\s*(?:1080[pP]|720[pP]|\[|$))
```

### 噪音排除清单

以下内容出现在集号之后时应被剥离，不属于集名称：
- 画质标签：`1080P`、`1080p`、`720P`、`4K`、`2160p`、`高清`
- 编码标签：`x264`、`x265`、`HEVC`、`H.264`、`H.265`
- 来源标签：`WEB-DL`、`WEBRip`、`BluRay`、`BDRip`、`HDTV`
- 音频标签：`AAC`、`DDP5.1`、`DD+5.1`、`DTS`、`Atmos`
- 色彩标签：`HDR`、`DV`、`Dolby Vision`、`10bit`、`8bit`
- 发布组标识：`[ANi]`、`[SubGroup]`、`【某某字幕组】` 等方括号/花括号内容
- 无意义后缀：`_高清`、`+`、连续的空格/横线/下划线
