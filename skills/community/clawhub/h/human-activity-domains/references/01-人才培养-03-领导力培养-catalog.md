## 领导力培养任务清单

领导力培养按子域组织的任务清单，附任务间关系和元操作映射提示。

**域间逻辑流**：领导力诊断 → 领导力训练 → 实践发展 → 导师辅导

---

### 1 领导力诊断

| ID | 任务类型 | 说明 | 依赖 | 元操作映射 |
|----|----------|------|------|----------|
| T1-1 | 能力诊断 | 通过测评工具诊断个人当前领导力水平和风格特征 | 无（入口） | S→C |
| T1-2 | 潜力评估 | 评估个人的领导潜力，包括学习能力、抗压能力、影响力 | T1-1 | C |
| T1-3 | 发展目标 | 基于诊断结果设定领导力发展目标和提升方向 | T1-2 | C |
| T1-4 | 培养方案 | 设计个性化的领导力培养方案，确定培养路径和资源 | T1-3 | C→A |

## 依赖拓扑摘要

### 一、域内链路

**领导力诊断链路**: T1-1 → T1-2 → T1-3 → T1-4
**领导力训练链路**: T2-1 → T2-2 → T2-3 → T2-4
**实践发展链路**: T3-1 → T3-2 → T3-3 → T3-4
**导师辅导链路**: T4-1 → T4-2 → T4-3 → T4-4

### 二、跨域协同链路

**完整领导力培养链路**: T1-1 → T1-2 → T1-3 → T1-4 → T2-1 → T2-2 → T2-3 → T2-4 → T3-1 → T3-2 → T3-3 → T3-4 → T4-1 → T4-2 → T4-3 → T4-4

**反馈改进链路**: T4-4 → T1-3（持续改进反馈到发展目标）

**实践驱动链路**: T3-4 → T2-1（危机处理反思驱动理论学习）

更多组合根据具体任务动态推导。
