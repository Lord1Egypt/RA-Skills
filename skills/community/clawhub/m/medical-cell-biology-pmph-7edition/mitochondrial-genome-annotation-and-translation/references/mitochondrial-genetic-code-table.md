# 哺乳动物线粒体遗传密码表（对比核基因组）

| 密码子 | 核基因组含义 | 哺乳动物线粒体含义 |
|--------|---------------|---------------------|
| UGA    | 终止          | 色氨酸 (Trp)        |
| AGA    | 精氨酸 (Arg)  | 终止                |
| AGG    | 精氨酸 (Arg)  | 终止                |
| AUA    | 异亮氨酸 (Ile)| 甲硫氨酸 (Met)      |

## 其他注意事项
- 起始氨基酸为甲酰甲硫氨酸（fMet），类似原核生物
- 成熟mRNA含约50个A的poly(A)尾，但无5'帽结构
- 某些终止密码子（如U或UA）需依赖polyadenylation完成终止信号

## 物种差异提示
不同脊椎动物类群（如鱼类、鸟类）的线粒体密码子规则可能存在差异，建议查阅NCBI Mitochondrial Code Tables（编号2–33）确认具体规则。
