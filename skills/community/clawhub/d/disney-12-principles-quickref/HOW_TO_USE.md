# How to Use This Skill

Hey AI—I just added the "animation-script-master" skill. Can you help me create an animation script?

## Example Invocations

**Example 1:**
Hey AI—I just added the "animation-script-master" skill. Can you help me写一个30秒的护肤品动画短片脚本？

**Example 2:**
Hey AI—Using the animation-script-master skill, 帮我写一个60秒的科普动画，主题是"为什么我们要多喝水"。

**Example 3:**
Hey AI—I just added the "animation-script-master" skill. 我想做一个45秒的搞笑小动画，主角是一只橘猫，帮我出分镜脚本。

## 工作流程

1. AI会先问你：想讲什么故事？核心信息是什么？
2. 然后确认视频总时长
3. 接着让你上传角色、场景、物品的参考图并编号
4. AI根据以上信息生成分镜脚本，每个镜头标注迪士尼十二法则
5. 最终输出为MD文档

## 注意事项

- 参考图越多，脚本对画面的描述越精准，AI视频生成时一致性越高
- 如果没有参考图，AI也能生成脚本，但画面描述会偏向文字想象
- 每个镜头都会严格匹配一条迪士尼十二法则，不会自编法则
