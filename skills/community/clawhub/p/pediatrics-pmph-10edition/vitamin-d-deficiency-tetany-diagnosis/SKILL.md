# 维生素D缺乏性手足搐搦症的诊断

## 适用场景
- 患儿年龄 ≤ 6个月
- 已存在维生素D缺乏
- 出现无热惊厥、喉痉挛或典型手足搐搦
- 同时具有活动期佝偻病体征（如方颅、肋串珠等）
- 排除发热性惊厥、中枢神经系统感染等其他病因

## 诊断流程

### 1. 确认典型临床表现
- 发作时无发热
- 惊厥、喉痉挛或手足搐搦突发
- 发作后神志迅速恢复清醒
- 无局灶性神经系统体征
- 存在活动期佝偻病表现

### 2. 检测血钙水平
- **总血钙** < 1.75～1.8 mmol/L（7～7.5 mg/dl）
- 或 **离子钙** < 1.0 mmol/L（4 mg/dl）

### 3. 执行隐匿型体征激发试验（如临床高度怀疑但血钙临界）
- **面神经征（Chvostek sign）**：轻叩颧弓与口角之间面颊部，出现同侧眼睑或口角抽动为阳性（注意：新生儿可假阳性）
- **腓反射（Peroneal sign）**：叩击腓骨小头处腓神经，足部向外侧收缩为阳性
- **陶瑟征（Trousseau sign）**：血压袖带充气至收缩压与舒张压之间并维持5分钟，出现腕关节屈曲痉挛、拇指内收呈“助产士手”为阳性

### 4. 排除关键鉴别诊断
- **低血糖**：检测血糖，若 < 2.2 mmol/L 需优先处理
- **低镁血症**：检测血镁，若 < 0.58 mmol/L 可能为病因
- **婴儿痉挛症**：表现为点头样痉挛、发育倒退，EEG示高度失律
- **原发性甲状旁腺功能减退**：PTH显著降低，常伴高磷血症，无佝偻病背景

### 5. 判断结果与后续行动
- 若满足血钙阈值且临床表现相符，**立即启动静脉钙剂急救治疗**
- 急救后给予维生素D补充治疗
- 记录PTH状态以评估甲状旁腺代偿能力

> **注意**：本技能不适用于有发热、颅内感染征象、癫痫家族史或其他明确神经系统疾病的患儿。
