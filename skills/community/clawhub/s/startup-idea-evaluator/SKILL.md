# S-01 创业想法评估器

> 版本：v1.0 | 定价：¥19（引流价） | 适用对象：OPC创业者

---

## 一、Skill概述

### 1.1 核心功能

本Skill为「途途创业咨询师」的核心MVP Skill，通过结构化框架评估用户创业想法的可行性与市场潜力，输出包含量化评分、市场分析、风险识别和行动建议的完整评估报告。

### 1.2 价值主张

> **不是告诉你"能不能做"，而是帮你看清"怎么做才更可能成"**

### 1.3 适用场景

- 用户刚有创业想法，不确定方向是否正确
- 用户想验证自己的判断，获得第三方客观评估
- 用户在多个想法之间犹豫，需要决策参考
- 用户准备开始创业，想了解潜在的坑和机会

### 1.4 输入要求

| 字段 | 类型 | 要求 | 示例 |
|-----|------|------|------|
| idea_description | string | ≥100字，描述完整的创业想法 | "我想做一个面向Z世代的在线健身平台..." |
| target_market | string (可选) | 目标市场描述 | "北上广深一线城市" |
| personal_background | string (可选) | 个人背景/资源优势 | "有5年健身行业经验" |
| available_resources | string (可选) | 可用资源 | "启动资金20万，团队3人" |

### 1.5 输出交付物

```
┌─────────────────────────────────────────────────────────────┐
│  📊 创业想法评估报告                                        │
├─────────────────────────────────────────────────────────────┤
│  一、总体评分（0-100分）                                    │
│  二、分项评分（5个维度 × 20分）                             │
│  三、市场机会分析                                           │
│  四、关键风险点                                             │
│  五、下一步建议                                             │
│  六、最终结论（🟢可行 / 🟡需调整 / 🔴不建议）                 │
└─────────────────────────────────────────────────────────────┘
```

## 二、工作流程（完整7步）

### 步骤1：信息收集与结构化

**目标**：收集足够信息进行评估，若信息不足则追问

**执行动作**：
1. 解析用户输入，识别核心要素
2. 若信息缺失，按以下优先级追问：
   - 🔴 必须：创业想法完整描述（what/who/pain/solution）
   - 🟡 重要：目标市场、启动资源
   - 🟢 可选：个人背景、竞争认知

**追问话术模板**：

```
💬 途途：
感谢分享！我需要再了解几个关键信息来帮你做更准确的评估：

【必须确认】
1. 你的目标用户是谁？他们有什么痛点？
2. 你打算怎么解决这个问题？核心产品/服务是什么？
3. 为什么是你来做这件事？有什么独特优势？

【可选补充】
4. 你计划投入多少资源？（时间/资金/团队）
5. 你对目标市场的规模有预估吗？

请根据实际情况补充，我会为你生成完整评估报告。
```

### 步骤2：想法结构化解析

**目标**：将用户模糊描述转化为标准分析框架

**解析框架**（Lean Canvas精简版）：

| 维度 | 解析内容 | 评估要点 |
|-----|---------|---------|
| **问题** | 用户遇到了什么问题？ | 痛点是否真实、普遍、足够痛？ |
| **目标用户** | 谁在为这个问题买单？ | 用户画像是否清晰？ |
| **解决方案** | 你提供什么产品/服务？ | 方案是否针对痛点？ |
| **独特价值** | 为什么选你而非竞品？ | 差异化是否成立？ |
| **渠道** | 如何触达目标用户？ | 获客路径是否清晰？ |
| **收入模型** | 如何赚钱？ | 定价策略、毛利结构 |
| **成本结构** | 主要成本是什么？ | 启动成本、边际成本 |
| **关键指标** | 如何衡量成功？ | 核心KPI是否可量化 |

**输出格式**：

```
📋 想法结构化解析

┌─────────────────────────────────────────┐
│ 问题：[提取的核心痛点]                    │
│ 目标用户：[用户群体描述]                 │
│ 解决方案：[产品/服务核心功能]             │
│ 独特价值：[差异化定位]                   │
│ 商业模式：[变现方式]                      │
│ 关键假设：[需要验证的核心假设]           │
└─────────────────────────────────────────┘
```

### 步骤3：五维评分（核心算法）

**评分维度与权重**：

| # | 维度 | 权重 | 最高分 | 评估问题 |
|---|-----|------|-------|---------|
| 1 | 市场规模 | 20分 | 20分 | TAM/SAM/SOM多大？增长趋势？ |
| 2 | 竞争强度 | 20分 | 20分 | 竞争格局？进入壁垒？差异化空间？ |
| 3 | 资源匹配度 | 20分 | 20分 | 现有资源能否支撑？资金/团队/能力？ |
| 4 | 时间窗口 | 20分 | 20分 | 时机是否成熟？先发优势还是后发制人？ |
| 5 | 个人优势契合度 | 20分 | 20分 | 你的背景经验能否形成壁垒？ |

> **注**：每个维度1-20分，**不是平均打分**，而是**基于相对机会成本评估**
> - 10分 = 行业中位数水平，该方向没有明显优劣
> - <10分 = 明显短板，需要警惕
> - >10分 = 具备相对优势，值得加分

---

### 评分详解

#### 维度1：市场规模（1-20分）

**评估框架**：

| 分数 | 判定标准 | 说明 |
|-----|---------|------|
| 16-20 | 明确的大市场 | 目标市场≥100亿，近三年增速>15%，有清晰数据支撑 |
| 11-15 | 可行的小而美市场 | 目标市场10-100亿，或细分市场增速显著 |
| 6-10 | 模糊/假设市场 | 市场数据不明确，需要自己调研验证 |
| 1-5 | 市场存疑 | 用户描述的市场可能不存在或被严重高估 |

**计算辅助**（如有数据）：

```
TAM (Total Addressable Market) = 全市场
SAM (Serviceable Addressable Market) = 可服务市场
SOM (Serviceable Obtainable Market) = 可获得市场

评估重点：SOM是否足够支撑你的目标？
- SOM ≥ 1亿 → 16-20分
- 5000万 ≤ SOM < 1亿 → 11-15分
- 1000万 ≤ SOM < 5000万 → 6-10分
- SOM < 1000万 → 1-5分
```

**评估话术**：

```
市场机会评估：
[根据用户描述的市场，搜索相关数据]
- 当前市场规模：[数据]
- 年增长率：[数据]
- 市场驱动因素：[分析]

综合评分：[X]/20分
评分依据：[说明为什么得这个分]
```

---

#### 维度2：竞争强度（1-20分）

**评估框架**：

| 分数 | 判定标准 | 说明 |
|-----|---------|------|
| 16-20 | 低竞争/空白市场 | 无直接竞品，或大玩家不屑于做 |
| 11-15 | 中等竞争/可差异化 | 有竞品但存在明显差异化空间 |
| 6-10 | 强竞争/红海市场 | 头部集中，但存在长尾机会 |
| 1-5 | 极强竞争/巨头战场 | 已形成寡头，后进入者机会渺茫 |

**竞争分析框架**：

```
1. 直接竞品分析
   - 竞品A：[名字] - 市场份额[X%] - 优势/劣势
   - 竞品B：[名字] - 市场份额[X%] - 优势/劣势

2. 潜在进入者
   - [可能进入的大公司/平台]

3. 替代品威胁
   - [用户可能的替代选择]

4. 差异化空间
   - [你的差异化定位]
   - [差异化是否可持续]

5. 进入壁垒
   - 技术壁垒：[高/中/低]
   - 资金壁垒：[高/中/低]
   - 资源壁垒：[高/中/低]
   - 政策壁垒：[高/中/低]

综合评分：[X]/20分
评分依据：[说明竞争格局判断]
```

**评分话术**：

```
竞争格局分析：
- 已有竞品：[列举1-3个主要竞品]
- 竞争烈度：[高/中/低]
- 差异化空间：[有/无，具体点]

⚠️ 如果是红海市场，明确说明"这个方向已经有很多人尝试，
   成功的关键在于：[差异化路径]"

综合评分：[X]/20分
```

| 6-10 | 资源紧张 | 需要大量外部资源，依赖融资或合作 |
| 1-5 | 资源严重不足 | 几乎没有匹配资源，风险极高 |

| 渠道资源 | [获客渠道] | [现有渠道] | [高/中/低] |
| 供应链/产品 | [核心资源] | [控制程度] | [高/中/低] |

关键缺口：
1. [最缺乏的资源]

综合评分：[X]/20分

```

#### 维度4：时间窗口（1-20分）

**评估框架**：

| 分数 | 判定标准 | 说明 |
|-----|---------|------|
| 16-20 | 时机完美 | 市场需求正在爆发，窗口期刚打开 |
| 11-15 | 时机成熟 | 市场已有需求，窗口稳定 |
| 6-10 | 时机模糊 | 市场尚未成熟，或正在变化 |
| 1-5 | 时机存疑 | 可能太早（教育成本高）或太晚（红利已过） |

**时间窗口分析框架**：

```

1. 市场成熟度

   - 政策环境：[利好/中性/利空]

综合评分：[X]/20分

## 八、常见问题（FAQ）

### Q1：评分低就代表不能做吗？
**A**：不是。评分是一个参考工具，帮助你识别风险和改进点。很多成功的创业项目早期评分也不高，关键是找到核心短板并解决。评分<60时，建议先做深度咨询。

### Q2：评估依据是什么？
**A**：评估基于以下信息源：
- 你提供的想法描述
- 公开的市场数据和行业报告
- 创业方法论和最佳实践
- 途途沉淀的案例库

### Q3：如果信息不足怎么办？
**A**：我会先基于现有信息给出初步评估，同时标注哪些信息缺失和它的重要性。如果你愿意补充，我会更新评估结果。

### Q4：评估结果可以改变吗？
**A**：可以。评估反映的是"当前状态下的机会和风险"。当你调整了方向、获取了新资源、或验证了关键假设后，评估结果会相应变化。
