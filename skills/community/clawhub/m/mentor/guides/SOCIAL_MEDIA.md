### 3.1 社交媒体采集（真人/明星/名人）

社交媒体发言是明星/名人最重要的数据来源——这是 TA **亲自写的文字**，直接体现说话风格、价值观、个性。

**⚠️ 不采集粉丝评论**——那是别人说的话，不是 TA 的人格。

#### 采集维度和优先级

| 维度 | 重要性 | 说明 | 上限 |
|------|-------|------|------|
| ⭐ 原创发帖 | ★★★★★ | **最核心** — TA 主动表达的想法和态度 | 最新 200 条 |
| ⭐ 评论/回复 | ★★★★★ | **最真实** — TA 怎么跟人互动，语气最自然 | 最新 100 条 |
| 个人简介/签名 | ★★★★ | 最浓缩的自我定义 | — |
| 置顶内容 | ★★★★ | TA 最想让别人看到的 | 全部 |
| 转发/引用 | ★★★ | TA 认同什么、传播什么 | 最新 100 条 |
| 长文/专栏 | ★★★★ | 深度表达，书面风格核心 | 全部 |

#### 微博采集（核心平台）

微博是中国明星/名人最主要的公开发言平台。

**采集流程**：

```python
# 使用 scripts/weibo_collect.py
python3 scripts/weibo_collect.py \
  --uid {微博UID} \
  --mcp-endpoint {endpoint} \
  --api-key {api_key} \
  --pages 20 \
  --output /tmp/{角色名}_weibo
```

**采集后需要做的数据清洗**：

| 步骤 | 说明 | 为什么重要 |
|------|------|---------|
| 1. 去重 | 按文本前50字去重 | API翻页可能有重复 |
| 2. 区分原创/转发 | `retweeted_status` 为空 = 原创 | 转发内容不是 TA 的话 |
| 3. **过滤广告/代言** | 含"旗舰店/天猫/代言/品牌大使"等关键词 | 广告文案由PR团队写的，不代表个人风格 |
| 4. **标记疑似团队运营** | 语气生硬+配图精美+固定格式 | 明星微博很多是工作室代发 |
| 5. 提取个人表达 | 非广告、非转发、有个人语气的微博 | 这才是说话风格的真实来源 |

**⚠️ 明星微博的重大挑战：团队代发 vs 本人发言**

大部分明星微博有 70-90% 是工作室/PR团队代发的（广告、宣传、活动图）。只有 10-30% 是本人的真实表达。

**识别本人发言的线索**：
- 口语化/随意（如"好累啊"、"新年快乐！！！！"）
- 短微博（< 30 字）
- emoji/表情风格与正式宣传不同
- 深夜发布（工作室通常白天发）
- 无商业 hashtag
- 用手机客户端发的（可从API字段`source`判断）
- 有错别字或语法不规范

**识别团队代发的线索**：
- 固定格式（如"#剧名# + 角色台词 + @搭档"）
- 含品牌名称/官方话术
- 精修图配文
- 工作日白天定时发布
- 语气正式、无个人情感

> 💡 **关键原则**：宁可少采，不能把团队代发的内容当成本人风格。
> 一条真正的个人微博 > 十条团队宣传微博。

**微博采集的特殊数据**：

除了帖子内容，还需关注：
- **超话发言**：明星在自己超话里的发言通常更随意、更真实
- **评论区回复**：明星回复粉丝的内容是最真实的互动风格
- **微博故事/vlog文案**：通常是本人写的

#### 抖音采集

抖音主要是视频平台，文字内容较少，但：

| 维度 | 说明 |
|------|------|
| 视频标题/文案 | 简短，但能看出风格 |
| 评论区回复 | 如果有创作者回复 |
| 直播切片 | 最真实的口语（通过 Whisper 转写，见 Section 3.3） |
| 短视频口播 | Whisper 转写（见 Section 3.3） |

#### 其他平台

| 平台 | 采集重点 | 说明 |
|------|---------|------|
| B站 | 动态/专栏/视频字幕 | UP主风格明显的平台 |
| 小红书 | 笔记文案 | 生活化表达多 |
| X/Twitter | 推文/回复 | 海外名人的核心平台 |
| Instagram | 图片文案/Stories | 视觉为主，文字为辅 |
| 知乎 | 回答/想法 | 深度表达 |
| 豆瓣 | 短评/日记 | 文艺/影评向 |

#### 社交媒体书面风格分析

采集完成后，对所有原创/个人微博进行书面风格统计：

```python
# 分析维度
分析项目 = {
    "平均帖子长度": "字数统计",
    "感叹号频率": "！的使用密度",
    "问号频率": "？的使用密度——反问型还是请教型",
    "省略号频率": "……的使用——欲言又止型",
    "emoji类型和频率": "常用哪些emoji，多久用一次",
    "hashtag习惯": "喜不喜欢加话题标签",
    "@习惯": "经常@谁",
    "分段习惯": "长段落还是短句分行",
    "中英混用": "是否混用英文，在什么场景",
    "自造词/梗": "有没有自己发明的表达方式",
}
```

> 💡 书面风格 + 口语风格（来自视频字幕）= 完整的说话风格画像。两者缺一不可，且差异巨大。
