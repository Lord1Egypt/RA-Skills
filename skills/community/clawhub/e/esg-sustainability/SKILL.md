# ESG与可持续发展技能

## 核心能力

| 能力 | 说明 |
|------|------|
| ESG报告生成 | 基于企业数据自动生成GRI/TCFD/ISSB标准ESG报告 |
| 碳足迹核算 | Scope 1/2/3碳排放自动计算，支持GHG Protocol |
| 评级对标 | 对标MSCI、Sustainalytics、CDP等主流ESG评级体系 |
| 气候风险分析 | TCFD气候情景分析（物理风险+转型风险） |
| 合规检查 | EU CSRD、SEC气候披露规则合规性审查 |

## 触发场景

- "帮我生成一份ESG报告"
- "计算公司的碳排放"
- "对标MSCI ESG评级"
- "检查是否符合CSRD要求"
- "分析气候风险"

## 执行流程

### 阶段一：数据收集
1. 收集企业能耗数据、废弃物数据、员工数据
2. 识别价值链上下游排放源
3. 收集治理结构和政策文件

### 阶段二：核算与分析
1. 按GHG Protocol分类Scope 1/2/3排放
2. 计算碳排放强度、能耗强度等KPI
3. 对标行业基准和评级标准

### 阶段三：报告生成
1. 按GRI/TCFD/ISSB模板生成报告
2. 输出碳减排目标和路线图建议
3. 生成可视化图表（排放趋势、构成饼图等）

## 输出模板

```markdown
# ESG报告

## 1. 环境绩效
- Scope 1排放：__tCO2e
- Scope 2排放：__tCO2e
- Scope 3排放：__tCO2e
- 可再生能源占比：__%

## 2. 社会绩效
- 员工满意度：__
- 多元化比例：__
- 社区投资：__万元

## 3. 治理绩效
- 董事会独立性：__%
- 反腐败政策覆盖率：__%

## 4. 改进建议
```

## 注意事项
- 碳排放核算需遵循GHG Protocol最新版
- ESG报告须标注数据来源和核算边界
- 评级对标仅供参考，不构成投资建议
