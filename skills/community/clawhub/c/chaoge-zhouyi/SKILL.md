## 先天卦数（梅花易数专用）

> ⚠️ 起卦一律用**先天数**，与后天方位数不同

```
乾1  兑2  离3  震4  巽5  坎6  艮7  坤8
```

## 除法取余规则

- 除以8取余数，余1~7对应先天数
- **余0 → 作8**（整除时视为余8）
- 除以6取动爻，**余0 → 作6**

---

## 标准解读流程

### 第一步：确认起卦信息

| 输入形式 | 对应起卦法 |
|---------|-----------|
| 农历年月日时 | 时间起卦法 |
| 两个数字 | 数字起卦法 |
| 汉字/文字 | 字画起卦法 |
| 已有卦象（如"地天泰"） | 直接解读 |
| 姓名+出生时间 | 命卦流程（读 `birth_chart.md`） |

> ⚠️ 旺相分析需要知道当前农历月份。

### 第二步：时间起卦法

```
年支数 = 地支序数（子1 丑2 寅3 卯4 辰5 巳6 午7 未8 申9 酉10 戌11 亥12）
上卦  = (年支数 + 月 + 日) ÷ 8 → 取余数（余0作8）
下卦  = (年支数 + 月 + 日 + 时辰数) ÷ 8 → 取余数（余0作8）
动爻  = (年支数 + 月 + 日 + 时辰数) ÷ 6 → 取余数（余0作6）
```

> ⚠️ 下卦与动爻都用**年+月+日+时辰**的**总和**，不是上卦数加时辰

**时辰数**：子1 丑2 寅3 卯4 辰5 巳6 午7 未8 申9 酉10 戌11 亥12

**输出格式**：
```
【起卦】年支X + 月X + 日X + 时辰X = 总和X
        上卦 ÷ 8 → 余X → X卦
        下卦 ÷ 8 → 余X → X卦
        动爻 ÷ 6 → 余X → 第X爻
【主卦】䷊ 地天泰
【变卦】䷋ 天地否
```

### 第三步：互卦取法

从主卦六爻中提取：

- 互卦下卦 = 主卦二三四爻
- 互卦上卦 = 主卦三四五爻

互卦代表事物发展过程中的**隐藏因素和中间状态**。在 `hexagrams.md` 中查找互卦对应卦象。

### 第四步：卦象解读（古文+白话）

在 `hexagrams.md` 中查找主卦，依次输出：卦辞 → 象传 → 彖传 → 动爻爻辞 → 卦象象数。

同理查找互卦和变卦的卦辞作为参考。

### 第五步：体用分析

### 问事卦与命卦通用体用规则

梅花易数的体用核心原则：**无动爻的卦为体，有动爻的卦为用**

- **体** = 无动爻的卦（代表"我"、本体、不变之方）
- **用** = 有动爻的卦（代表"事"、变化、所用之方）

> ⚠️ 当上下卦都有动爻时（多爻动），以本卦上卦为用、下卦为体

**五行生克**：

| 体用关系 | 符号 | 断语 |
|---------|------|------|
| 体克用 | 得 | 自身驾驭，主动争取 |
| 用克体 | 失 | 自身受制，需避凶 |
| 体生用 | 泄 | 自身消耗，难度较大 |
| 用生体 | 助 | 外部得助，事半功倍 |
| 比和 | 合 | 内外相助，诸事顺遂 |

**旺相休囚**（需知农历月份）：

| 状态 | 条件 |
|------|------|
| 旺 | 与月令五行相同 |
| 相 | 月令所生 |
| 休 | 我生月令 |
| 囚 | 我克月令 |
| 死 | 克我月令 |

### 第六步：应期判断

- 动爻属卦之数 → 对应周期（一六期/二七期/三八期/四九期）
- 近事当日起，远事数十日，大事数月年

### 第七步：趋避建议

- **宜**：宜静/宜行/宜速/宜缓/宜变
- **忌**：忌何事/避何方
- **方位**：利X方，避X方
- **一句话总结**

## 解读示例

**用户输入**：甲子年三月初五午时问出行

**【起卦】** 年支子=1，上卦=(1+3+5)÷8余**1**→乾，下卦=(1+3+5+7)÷8=16÷8=2余**0→作8**→坤，动爻=(1+3+5+7)÷6=16÷6余**4**→第4爻
主卦：天地否䷋（上乾下坤）→ 变卦：风地观䷓（第4爻属上卦，上卦乾→巽）

**【互卦】** 主卦否䷋（乾上坤下），六爻：上阳、五阳、四阳（变为阴→变卦）、三阴、二阴、初阴。互卦上卦=三四五爻=阴+阳+阳=巽☴，互卦下卦=二三四爻=阴+阴+阳=震☳，互卦=风雷益䷩

**【卦辞】** 否之匪人，不利君子贞。大往小来。
> 白话：闭塞不通，小人得势，不利于君子守正。大的去了小的来了，出多入少。

**【体用】** 否䷋第4爻属**上卦乾**（上卦为四五六爻），故上卦有动爻为**用**，下卦坤无动爻为**体**。体坤☷（土）用乾☰（金），土生金体生用为**泄**，自身消耗。春令木旺土死，体卦极弱。变卦风地观䷓：上卦变巽☴（木），下卦坤不变仍为体，体坤（土）用巽（木），木克土用克体为**失**。主卦泄、变卦失，皆不利。

**【趋避】** 不利出行，宜守不宜动，方位利南/中（火生土），避东（木克土）/北（水耗土）。

## 紫微斗数速解流程

用于分析命盘格局、星曜组合、身宫迁移：
