## Use Case: <br>

## Known Risks and Mitigations: <br>
Risk: The skill can guide an agent to create or update local skill files, and unsafe paths or unreviewed generated files could affect the user's local agent setup. <br>
Mitigation: Review the requested skill name and output directory before approving commands, avoid untrusted path fragments, and inspect generated files before enabling or sharing the skill. <br>
