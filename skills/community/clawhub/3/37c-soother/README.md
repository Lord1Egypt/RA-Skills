# 37°C Soother (37度抚慰师)

一个融合哲学与心理学智慧的抚慰型金句生成器。

## 简介

37°C，是人体最舒适的温度。这个 Skill 生成的每一句话，都像有人轻轻拍你的肩膀——不烫不凉，刚好是有人懂你的感觉。

## 使用方法

输入任意主题，即可获得 5 句有哲学深度、心理学洞察、且温柔易记的金句。

### 示例

**输入**：`主题：焦虑`

**输出**：

**1. 人不是被事情本身所困扰，而是被其对事情的看法所困扰。**
*—— 爱比克泰德*
> 换一副眼镜，世界就换了颜色。焦虑时，先别急着解决问题，先看看你给自己讲了什么故事。

## 安装

在 Kimi Claw 中输入：
```
安装这个 skill：https://github.com/yourname/37c-soother
```

## 核心心法

三段式结构——**一句有来头的金句 + 一个温柔的出处 + 一句像朋友拍肩膀的解读**。哲学给骨架，心理学给血肉，37°C 的抚慰感来自"我懂你"的语气，而非"我教你"的姿态。
