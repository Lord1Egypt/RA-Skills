# 家庭急救包与急救证指南

> **⚠️ 免责声明**：本文仅供参考，具体配置应根据家庭实际情况调整。

---

## 第一部分：家庭急救包清单

### 一、基础配置原则

- 存放在**固定位置**，全家人都知道
- 放在儿童接触不到但成人**方便取用**的地方
- 每 **3-6个月** 检查一次，更换过期物品
- 不要放在潮湿的卫生间
- 可准备2份：家中1份 + 车上1份

### 二、分级配置清单

#### Level 1 — 基础版（必备，每家庭至少一套）

**外伤处理**：
| 物品 | 数量 | 用途 |
|------|------|------|
| 无菌纱布（不同尺寸） | 各5片 | 覆盖伤口、止血 |
| 弹性绷带 | 3卷 | 固定敷料、加压包扎 |
| 三角巾 | 2条 | 悬吊、固定、包扎 |
| 医用胶带/创可贴 | 若干 | 小伤口 |
| 一次性手套（无粉） | 5双 | 防止感染 |
| 剪刀（圆头） | 1把 | 剪绷带/衣物 |
| 镊子 | 1把 | 取异物/蜱虫 |

**消毒清洁**：
| 物品 | 用途 |
|------|------|
| 碘伏棉签/碘伏溶液 | 伤口消毒（比酒精温和） |
| 75%酒精棉片 | 器械消毒、皮肤消毒 |
| 生理盐水（小支装） | 伤口冲洗 |
| 肥皂/洗手液 | 咬伤冲洗 |

**常用药品**：
| 药品 | 用途 |
|------|------|
| 布洛芬/对乙酰氨基酚 | 退热、止痛 |
| 氯雷他定/西替利嗪 | 过敏、虫咬 |
| 蒙脱石散 | 腹泻 |
| 口服补液盐 | 腹泻/中暑补水电解质 |
| 活性炭 | 中毒（遵医嘱使用） |

**工具类**：
| 物品 | 用途 |
|------|------|
| 电子体温计 | 测体温 |
| 一次性冰袋（即捏即冷型） | 扭伤、中暑降温 |
| 急救手册/卡片 | 急救流程速查 |
| 急救毯（保温毯） | 失温、户外应急 |

#### Level 2 — 进阶版（推荐）

在Level 1基础上增加：

| 物品 | 用途 |
|------|------|
| 止血带（CAT/ SOFT-T） | 四肢大出血（需培训） |
| 止血粉/止血纱布 | 严重出血 |
| 口对口呼吸面罩 | CPR人工呼吸（防止交叉感染） |
| 烧伤敷料/水凝胶敷料 | 烧伤专用 |
| AED（自动体外除颤器） | 心脏骤停急救（💰 较贵但可救命） |
| 血氧仪 | 监测血氧饱和度 |
| 血压计 | 监测血压 |
| 夹板（SAM夹板） | 骨折固定 |
| 颈托 | 颈椎保护 |
| 眼罩/洗眼液 | 眼部受伤 |

#### Level 3 — 特殊需求版

| 人群 | 额外物品 |
|------|----------|
| 有心脏病史 | 硝酸甘油、阿司匹林、血压计 |
| 有哮喘 | 备用吸入器 |
| 严重过敏史 | 肾上腺素自动注射笔（EpiPen） |
| 糖尿病 | 血糖仪、葡萄糖片、胰岛素 |
| 婴幼儿 | 儿童退热药（对乙酰氨基酚滴剂）、喂药器、鼻吸器 |
| 宠物 | 宠物专用急救品 |

## 第三部分：急救证（中国）

### 一、主要急救培训机构

| 机构 | 发证类型 | 特点 |
|------|----------|------|
| **中国红十字会** | 红十字救护员证 | 全国通用，最常见，培训费低 |
| **AHA（美国心脏协会）** | Heartsaver / BLS | 国际通用，含CPR+AED，证书2年有效 |
| **ERC（欧洲复苏委员会）** | BLS / ILS | 欧洲体系 |
| **地方急救中心/120** | 急救技能培训证 | 各地组织，偏实操 |

### 二、获取流程

#### 红十字会救护员证（推荐入门）

1. **报名**：联系当地红十字会或官网查询培训时间
2. **培训**：通常为16学时（2天）：
   - 理论：急救概论、CPR、创伤救护
   - 实操：CPR+AED、止血包扎固定、搬运
3. **考核**：理论考试 + 实操考核
4. **发证**：通过后颁发《红十字救护员证》
5. **费用**：各地不同，约200-500元（部分地区免费）

#### AHA Heartsaver 急救员课程

1. **报名**：通过AHA授权培训中心（中国各大城市均有）
2. **培训**：约8小时（1天）
   - 成人/儿童/婴儿CPR+AED
   - 内科急症、创伤急症、环境急症
3. **考核**：技能测试（无笔试）
4. **发证**：Heartsaver First Aid CPR AED 证书
5. **有效期**：2年（到期需复训换证）
6. **费用**：约800-1500元

#### AHA BLS（基础生命支持）

- 面向**医疗从业者**，内容更深入
- 含团队协作CPR、球囊面罩通气
- 费用约600-1000元

### 三、持证者权利保障

《中华人民共和国民法典》第184条（"好人法"）：
> 因自愿实施紧急救助行为造成受助人损害的，救助人不承担民事责任。

这意味着：持证急救者在紧急情况下施救受到法律保护。

## 第四部分：急救包维护检查清单

每3个月检查一次：

- [ ] 药品有效期（过期立即更换）
- [ ] 无菌包装完好无损
- [ ] 碘伏/酒精未干涸
- [ ] 一次性冰袋完好
- [ ] 手电筒电池有电
- [ ] 手套无破损
- [ ] AED电极片/电池在有效期内（如有）
- [ ] 肾上腺素笔在有效期内（如有）
- [ ] 补充已使用的物品
- [ ] 急救手册完好可读
