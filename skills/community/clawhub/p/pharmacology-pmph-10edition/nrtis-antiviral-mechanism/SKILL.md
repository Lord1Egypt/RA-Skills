# NRTIs抗病毒作用机制

## 何时使用
- 用户咨询NRTIs（如齐多夫定、拉米夫定、替诺福韦等）如何抑制HIV或HBV
- 需解释NRTIs为何必须与其他抗病毒药物联用
- 分析NRTIs相关不良反应（如骨髓抑制、乳酸酸中毒）的机制
- 判断某药物是否属于NRTIs及其作用前提

## 执行步骤
1. **确认适用病毒**：NRTIs主要针对处于反转录阶段的HIV-1、HIV-2；部分（如拉米夫定、替诺福韦）也对HBV有效。
2. **验证激活条件**：确保宿主细胞内存在胸苷酸激酶等激酶，可将NRTIs磷酸化为活性三磷酸形式（NRTI-TP）。
3. **执行链终止机制**：
   - NRTI-TP与天然dNTP竞争，被病毒反转录酶掺入新生DNA链；
   - 因NRTIs缺乏3'-OH基团，无法形成磷酸二酯键，导致DNA链合成提前终止；
   - 病毒DNA无法完成合成，复制被阻断。
4. **评估限制与风险**：
   - 对已整合入宿主基因组的前病毒无效；
   - 单药易致耐药，必须联合NNRTIs、INSTIs等其他抗病毒药；
   - 高浓度可能抑制宿主DNA聚合酶，引发线粒体毒性。
5. **监测与应对**：建议定期监测血常规、肝功能、乳酸水平，警惕骨髓抑制和乳酸酸中毒。

> 注意：本机制仅在病毒RNA正在反转录为DNA时生效。
