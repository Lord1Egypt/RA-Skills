# Dietary Impropriety Pathogenesis Analysis

## When to Use
当出现以下任一情况时启用本技能：
- 用户提及饮食无规律（如过饥、过饱、不定时进食）
- 用户报告食用不洁、变质或有毒食物
- 用户长期偏好寒凉、辛辣、咸味或单一食物类型
- 用户为健康人群或病后恢复者，出现消化不适、乏力、面色异常等症状

> **注意**：本技能不适用于完全禁食、肠外营养等非经口摄入情形。

## Execution Steps

### 1. 判断饮食失宜类型
根据用户描述，归类为以下三类之一：
- **饮食不节**：包括过饥、过饱、食无定时
- **饮食不洁**：摄入污染、腐败、有毒或寄生虫污染食物
- **饮食偏嗜**：长期偏好寒热性质、五味（酸苦甘辛咸）或食物种类（如只吃肉、不吃蔬果）

### 2. 评估脾胃功能状态
确认当前脾胃处于正常、虚弱或已受损状态，此将影响病理演变速度与严重程度。

### 3. 推演致病机制与预警表现
依据失宜类型，按以下逻辑推导可能的病理变化：

#### 若为饮食不节：
- **过饥** → 气血生化乏源 → 面色不华、气短心悸、神疲消瘦、易感外邪
- **过饱** → 饮食停积 → 嗳腐吞酸、脘腹胀满、吐泻；严重可致“食厥”；久则生痰化热；婴幼儿可成疳积；病后过饱可致“食复”
- **食无定时** → 脾胃气机紊乱 → 气滞血瘀或津停生湿 → 胃痛、反酸、聚积、便血等

#### 若为饮食不洁：
- 引发痢疾、霍乱样症状（腹痛、吐泻）
- 可致肠道寄生虫病（腹痛、嗜异物、面黄肌瘦、蛔厥）
- 严重中毒可致昏迷甚至死亡

#### 若为饮食偏嗜：
- **寒热偏嗜**：生冷伤阳致寒湿（脘腹冷痛、泄泻）；辛燥助热致胃肠积热（口臭、便秘、痔疮）
- **五味偏嗜**：如过咸伤肾乘心 → 胸闷、面色无华、血脉瘀滞
- **种类偏嗜**：结构失衡 → 瘿瘤（缺碘）、佝偻病（钙磷障碍）、夜盲（维A缺乏）；烟酒过度 → 脾胃湿热（口苦、苔腻、纳差）

### 4. 输出风险预警
综合上述分析，明确提示用户可能面临的证候类型、典型症状及进一步发展风险，建议调整饮食行为并必要时就医。
