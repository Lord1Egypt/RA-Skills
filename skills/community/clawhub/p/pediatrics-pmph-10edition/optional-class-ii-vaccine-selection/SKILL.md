# 非强制性第二类疫苗选择指南

## 何时使用
- 当地存在乙脑、流脑等疾病高发风险
- 即将进入流感等季节性传染病高发期
- 家长主动询问或表达接种意愿（如因旅行、托幼入学等）
- 儿童已完成国家免疫规划规定的基础疫苗接种

## 执行步骤
1. **确认前提条件**：
   - 儿童已完成国家免疫规划基础疫苗
   - 获得家长知情同意
   - 评估当地是否存在相关疾病流行风险

2. **识别适用的第二类疫苗**（常见选项）：
   - 乙型脑炎疫苗（JE）
   - 流行性脑脊髓膜炎疫苗（MenACWY/MenA）
   - 流感疫苗（每年接种）
   - 甲型肝炎病毒疫苗（HepA）
   - 水痘疫苗（Varicella）
   - b型流感嗜血杆菌疫苗（Hib）
   - 肺炎球菌疫苗（PCV）
   - 轮状病毒疫苗（Rotavirus）

3. **进行个体化评估**：
   - 检查儿童当前健康状况
   - 回顾既往过敏史及疫苗不良反应
   - 确认与其他疫苗接种的时间间隔是否合规

4. **提供决策支持**：
   - 向家长说明疫苗的预防价值、自费情况及潜在风险
   - 强调虽非强制，但在高风险情境下强烈推荐接种

> **提示**：这些疫苗虽非国家强制，但在高风险情境下具有重要预防价值。
