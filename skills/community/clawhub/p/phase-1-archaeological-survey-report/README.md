# phase-1-archaeological-survey-report

Draft a Phase I Archaeological Survey (Cultural Resources Assessment Survey) report for a development project, covering background research, field methods, site documentation, and NRHP eligibility recommendations aligned to Section 106 of the National Historic Preservation Act.

## When to use this skill

Use this skill when a CRM archaeologist, principal investigator, or cultural resource specialist needs to draft a Phase I Archaeological Survey report for a Section 106 NHPA review, state CEQA/SEPA compliance, or other federal or state cultural resource management undertaking before ground-disturbing construction or development proceeds.

## What it produces

- APE description with direct and indirect effect areas, datum, and coordinate system
- Background research documentation (SHPO site files, NRHP database, historic maps, soil surveys, GLO records, environmental databases, prior surveys within 1-mile buffer)
- Environmental and cultural context summary with regional culture-history narrative (Paleo-Indian through Historic)
- Field methodology documentation (shovel-test interval and dimensions, screen size, transect bearing, crew qualifications)
- Site-by-site results table with state trinomial, NRHP criteria evaluation, eligibility recommendation, and data potential assessment
- Summary recommendations (No Further Work / Phase II Evaluation / Data Recovery / SHPO/THPO Consultation)
- DRAFT report with bibliography stubs, appendix placeholders (shovel-test log, site forms, photo log), and unsigned principal investigator review block

## Inputs needed

| Input | Description |
|---|---|
| Project name and location | County, state, USGS 7.5-minute quadrangle name |
| APE acreage and description | Direct and indirect effects area, how boundaries were determined |
| Undertaking agency | Federal or state agency triggering Section 106 |
| Survey type | Areal, linear, or point-source |
| Scope of work | Any SHPO- or agency-approved SOW; if none, note |
| Field findings summary | Shovel test counts, depths, recovery data, site discoveries |

## Safety notes

- Output is a DRAFT report — not a final SHPO submission, NRHP eligibility determination, or Section 106 compliance finding.
- NRHP eligibility determinations require principal investigator review and formal SHPO or ACHP concurrence; this skill cannot substitute for that regulatory process.
- GPS coordinates and locational data for archaeological sites are subject to ARPA and state confidentiality requirements; do not include precise site locations in unprotected documents without PI authorization.
- Tribal consultation records and THPO communications are confidential and must not be incorporated into the DRAFT without PI and agency authorization.
- Not a substitute for licensed professional archaeological judgment or regulatory compliance review.
