# 消费业

## 核心财务特征

- 轻资产或品牌资产驱动
- 现金流质量高（预收款+现金交易占比大）
- 毛利率跨度大（必需品15-35%，可选品40-80%）
- 渠道掌控力决定盈利能力
- 消费升级/降级趋势影响估值中枢

## 关键分析指标

| 指标 | 健康值 | 警戒值 | 说明 |
|------|--------|--------|------|
| 毛利率 | 细分差异大 | 低于行业均值10% | 品牌溢价/成本优势信号 |
| 销售费用率 | <25% | >40% | 过高则品牌驱动失效 |
| 应收账款周转天数 | <30天 | >90天 | 消费业回款通常快 |
| 存货周转天数 | <60天 | >120天 | 快消vs耐用品差异 |
| 预收账款/营收 | >5% | 持续下降 | 下游需求先行指标 |
| 经营性现金流/净利润 | >1 | <0.7 | 消费业现金流质量应较好 |

## 风险评估重点

- 品牌老化：营收增速 < 行业、毛利率持续下降
- 渠道依赖：单一渠道占比 > 50%
- 消费降级挤压：提价能力丧失
- 存货跌价：时尚/电子消费品类
- 食品安全/质量事件（食品饮料）

## 舞弊高发区

- 渠道压货虚增收入（经销商库存积压）
- 销售返利跨期调节
- 关联经销商虚构销售
- 存货不跌价
