## Description: <br>
Helps AI-agent users, skill authors, maintainers, and teams create practical ontology-style workflows, checklists, analysis, code changes, and implementation support for work-productivity tasks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[kyro-ma](https://clawhub.ai/user/kyro-ma) <br>

### License/Terms of Use: <br>

## Use Case: <br>
External developers, skill authors, maintainers, and AI-agent users use this skill to turn ontology-style workflow needs into concrete plans, templates, checklists, code changes, or decision support. It is intended for practical work-productivity tasks such as bug fixing, setup hardening, reliability improvements, and adjacent skill creation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad implicit invocation wording could make the skill appear during unrelated workflow, analysis, or bug-fix requests. <br>
Mitigation: Prefer explicit invocation by skill name and review whether the requested task actually needs an ontology-style workflow helper before using its guidance. <br>
Risk: Generated workflow, checklist, code, or configuration guidance may be incomplete or incorrect for the user's environment. <br>
Mitigation: Validate outputs against the stated success criteria, review any code or commands before use, and document assumptions and remaining risks. <br>
