# Prompt 3 — Review & Referral System (Windshield Buddy Program)

## Instructions

Fill in the variables below, then paste this entire prompt into Claude (claude.ai) or any capable AI model.

---

## PROMPT

You are a customer retention specialist for local auto glass businesses. Generate a post-installation review and referral system using only the information provided.

**Shop details:**
- Shop name: [SHOP_NAME]
- City, State: [CITY], [STATE]
- Google Business Profile link: [URL or "I'll add later — use placeholder [GOOGLE_REVIEW_LINK]"]
- Referral reward: [e.g., "$25 Visa gift card", "free rock chip repair (chip <1 inch)", "10% off next service", "NONE"]
- Customer mix (approximate): [e.g., "70% insurance, 30% cash-pay" or "mostly cash-pay"]
- ADAS recalibration performed on this job: [YES / NO / SOMETIMES — note: this affects the confirmation message]
- Preferred follow-up channel: [SMS / EMAIL / BOTH]
- Shop phone or text number for responses: [PHONE]

**Compliance rules:**
1. Review requests never say "please leave us a positive review" or "5-star review" — always "share your experience" or "tell us how we did."
2. Referral rewards clearly state "per referred customer who completes a paid service" — no ambiguous "earn $X" without conditions.
3. If ADAS recalibration = YES, confirmation message includes "your ADAS camera has been recalibrated." If NO or SOMETIMES, omit.
4. Google review links go to the GBP review URL (provided or placeholder).
5. SMS messages include "Reply STOP to unsubscribe" on first message in sequence.

**Generate:**

**1. Post-Install SMS Sequence (3 messages)**

Message 1 (send same day, 2 hours after pickup/completion):
- Thank you + ADAS confirmation (if applicable)
- Soft review ask
- Max 160 characters each

Message 2 (send day 3):
- Follow-up + review ask with link
- Max 160 characters

Message 3 (send day 14 — referral ask):
- Windshield Buddy referral program intro
- Max 160 characters

**2. Post-Install Email Sequence (3 emails)**

Email 1 (same day):
- Subject + preview text
- Thank you, service summary, ADAS confirmation if applicable
- Review ask (soft)
- 150–200 words

Email 2 (day 3):
- Subject + preview text
- Follow-up, emphasize how much reviews help small businesses
- Direct review link
- 100–150 words

Email 3 (day 14):
- Subject + preview text
- Windshield Buddy referral program introduction
- How it works + reward
- 150–200 words

**3. Referral Card Copy (print-ready)**

Front:
- Program name: Windshield Buddy
- Headline: "Know someone who needs a new windshield?"
- Reward for referrer + reward conditions

Back:
- How it works (3 steps max)
- Shop name, phone, address
- "Your friend gets a free rock chip repair with their windshield replacement" (or equivalent based on reward)
- Expiration note if applicable

**4. Google Review Response Templates (5 variations)**

Template A: 5-star standard (warm, specific, no keyword stuffing)
Template B: 5-star with ADAS mention (customer noted the recalibration — validate and reinforce)
Template C: 4-star (acknowledge positive + "we'd love to hear how we can earn that 5th star")
Template D: 3-star concern (professional, offer to make it right, move conversation offline)
Template E: 1-star / negative (professional, no defensiveness, clear resolution path)

Each response: 50–100 words.

**5. Windshield Buddy Landing Page Copy (200 words)**

Headline, how-it-works section (3 steps), reward details, fine print (one-per-customer, referral must complete paid service, no cash value, expires [DATE or "12 months from issue"]).
