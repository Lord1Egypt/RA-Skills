## Description: <br>
UI component generator for producing standalone HTML/CSS snippets for forms, tables, cards, modals, navigation bars, and related interface elements. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and designers use this skill to quickly generate editable UI component HTML/CSS and design reference guidance for common web interface patterns. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: One helper script records command arguments in a local history file, which may expose sensitive names or values passed to the skill. <br>
Mitigation: Avoid passing secrets, credentials, or sensitive project details as command arguments, and review local history files before sharing an environment. <br>
Risk: The included shell scripts expose different command sets, which can cause users to run a different helper path than intended. <br>
Mitigation: Check which script the launcher invokes and review the available commands before relying on generated output. <br>
Risk: Generated UI snippets are starting points and may not meet a project's accessibility, browser support, or design-system requirements without review. <br>
Mitigation: Review and adapt generated HTML/CSS for labels, focus states, contrast, responsiveness, and project-specific component standards. <br>
