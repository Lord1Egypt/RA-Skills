## 12步制作流程

### Step 1：选题定位（15分钟）

```
选题检查清单：
□ 题材类型确定（6种选1）
□ 目标平台确定（抖音/快手/小红书/B站/视频号）
□ 目标受众画像（年龄/性别/兴趣）
□ 参考作品（找3部同类型爆款短剧）
□ 差异化卖点（和竞品比有什么不同）
```

**选题热度参考（2026年）**：
- 🔥🔥🔥 都市甜宠、悬疑推理（永恒热门）
- 🔥🔥 古风仙侠、职场逆袭（持续热门）
- 🔥 科幻未来、恐怖惊悚（小众但精准）

### Step 2：剧本大纲（30分钟）

1. 用 `script-templates.md` 选择剧本类型
2. 填写8集大纲
3. 确定每集的钩子和反转点
4. 写出黄金3秒开头

### Step 3：角色设计（20分钟）

1. 用 `prompt-templates.md` 的角色卡模板
2. 生成主角+配角的角色卡
3. 用AI工具生成角色定妆照（正面/侧面/全身各1张）
4. **关键：保存角色卡和参考图，后续每张图都要参考**

### Step 4：分镜设计（45分钟）

1. 用 `prompt-templates.md` 的分镜生成Prompt
2. 为每集生成8-15个镜头的分镜表
3. 标注每个镜头的：景别、画面、运动、对白、配乐
4. 输出：完整分镜脚本（Excel/Notion/飞书表格）

### Step 5：AI绘图（画面素材）（2-3小时）

```
批量生产流程：
1. 先生成场景图（不需要角色一致性，效率高）
2. 再生成角色+场景合成图（需要角色一致性，慢）
3. 最后生成特写/表情图

效率技巧：
- 一次生成同一场景的多个角度（正面/侧面/远景）
- 用种子号(seed)保持一致性
- 风格关键词固定不变
- 批量生成时保持Prompt结构一致
```

### Step 6：AI视频生成（2-4小时）

```
推荐流程（Runway Gen-3为例）：
1. 图片转视频模式（Image to Video）
2. 输入分镜图 + 运动描述
3. 生成4秒片段（标准长度）
4. 每个镜头生成3-5个变体，选最好的
5. 关键帧之间加过渡效果

注意事项：
- 每次生成控制在4-6秒
- 运动描述要具体（不要"有趣的运动"，要"缓慢推镜头，人物微微转头"）
- AI视频有随机性，多生成几个版本
- 角色面部容易变形，特写镜头建议用静态图+Ken Burns效果替代
```

### Step 7：配音制作（1小时）

```
流程：
1. 整理对白脚本（每行对白 + 角色名 + 情绪标注）
2. 用ElevenLabs/剪映AI配音生成
3. 每句对白单独生成，方便后期对齐
4. 旁白单独录制
5. 调整语速和情感参数

声音一致性技巧：
- 每个角色固定一个声音ID
- 保存声音设置参数
- 同一角色的情绪变化用同一个声音ID的不同参数
```

### Step 8：音效与配乐（30分钟）

```
配乐选择原则：
- 每集1首主BGM + 2-3个情绪转折点
- 开头30秒用紧张/悬念型配乐
- 反转时刻用静默→骤起
- 甜宠场景用轻快钢琴/吉他

音效清单（每集必备）：
- 环境音（雨声/街道/办公室）
- 动作音（开门/脚步/手机响）
- 情绪音（心跳/倒计时/雷声）
- 转场音（whoosh/鼓点）
```

### Step 9：视频剪辑（1-2小时）

```
剪映专业版推荐流程：
1. 新建项目，按集分类
2. 导入所有视频片段，按分镜顺序排列
3. 添加配音轨道，对齐画面
4. 添加BGM和音效
5. 添加字幕（剪映自动识别+手动校对）
6. 添加转场效果
7. 调色（统一色调）
8. 导出（H.264, 1080x1920, 30fps）

剪辑节奏控制：
- 都市场景：快节奏，1-3秒/镜头
- 古风场景：中速，2-4秒/镜头
- 悬疑场景：慢+快交替
- 情感高潮：慢镜头+特写
```

### Step 10：平台适配（30分钟）

```
各平台规格：

抖音：
- 竖屏 9:16（1080x1920）
- 每集1-3分钟
- 前3秒决定生死
- 添加合集功能（连续观看）

快手：
- 竖屏 9:16
- 每集1-3分钟
- 偏接地气，标题要有梗
- 小剧场功能

小红书：
- 竖屏 3:4 或 9:16
- 偏精致，封面图重要
- 文案描述要完整
- 标签要用好

B站：
- 横屏 16:9（可选竖屏）
- 可以更长（3-10分钟）
- 弹幕互动设计
- 分P功能

视频号：
- 竖屏 9:16
- 社交推荐为主
- 封面图+标题是关键
```

### Step 11：发布运营（持续）

```
发布策略：
- 每天1-2集，保持更新节奏
- 固定时间发布（晚8-10点最佳）
- 每集结尾放预告（下一集看点）
- 引导互动："你觉得XX会怎么做？"
- 评论区置顶引导看合集

涨粉技巧：
- 前3集免费，后续引流到合集/付费
- 和同类型短剧账号互动
- 用热门BGM和话题标签
- 定期做粉丝回馈（投票选下一部题材）
```

### Step 12：数据复盘（每周）

```
关键指标：
- 完播率（>40%算合格）
- 3秒留存率（>70%算合格）
- 互动率（点赞+评论+转发）
- 涨粉率（每1000播放涨粉数）

A/B测试：
- 封面图（测试2-3个版本）
- 标题（悬念型 vs 直白型）
- 开头3秒（不同钩子）
- 发布时间（早/中/晚）
```

| 角色设计 | 20min | 10min |
| 分镜设计 | 45min | 20min |
| AI绘图 | 2-3h | 1-1.5h |
| AI视频 | 2-4h | 1-2h |
| 配音 | 1h | 30min |
| 音效配乐 | 30min | 15min |
| 视频剪辑 | 1-2h | 45min |
| 平台适配 | 30min | 15min |
| **总计** | **8-13h** | **4-6h** |

**目标**：熟练后每部短剧（8集）控制在4-6小时内完成。
