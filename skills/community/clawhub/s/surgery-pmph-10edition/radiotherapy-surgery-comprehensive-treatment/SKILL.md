# 放疗-手术综合治疗规范

## 适用场景
在以下条件同时满足时启用本技能：
- 肿瘤类型属于乳腺癌、食管癌、支气管肺癌、直肠癌、脑肿瘤（含垂体瘤）、宫颈癌、外阴癌、阴茎癌、肢体及躯干皮肤癌或软组织肉瘤；
- 肿瘤为局部进展但尚未发生广泛远处转移；
- 患者经多学科团队评估，确认可耐受手术与放疗联合治疗；
- 存在高局部复发风险。

## 执行步骤
1. **确认适用肿瘤类型**：核对是否属于上述七类主要适应症。
2. **选择放疗时机**：
   - **术前放疗**：用于缩小肿瘤体积以提高手术切除率，适用于Ⅲ期乳腺癌、直肠癌、骨肉瘤等；
   - **术后放疗**：用于清除残留病灶、降低局部复发率，常见于乳腺癌、大肠癌等；
   - **术中放疗**：在肿瘤切除后对瘤床及淋巴结引流区给予单次大剂量照射，目前限于临床试用阶段。
3. **整合全身治疗**：明确放疗与手术仅为局部治疗手段，必须结合化疗、靶向治疗等全身治疗以控制潜在微转移。
4. **设定治疗目标**：以提高局部控制率、延长无病生存期为核心，同时最大限度保留器官功能、减少治疗相关损伤。

## 注意事项
- 本方案对以远处转移为主的肿瘤效果有限；
- 不可替代全身系统性治疗；
- 必须由多学科团队（MDT）共同决策并全程管理。
