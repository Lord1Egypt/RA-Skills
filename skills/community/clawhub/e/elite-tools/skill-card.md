## Description: <br>
Elite CLI Tools guides agents toward modern CLI utilities for efficient shell execution, directory traversal, file manipulation, structured parsing, and cleaner diffs. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and engineering agents use this skill during shell work to choose concise, structured tools for file discovery, viewing, replacement, code search, JSON/YAML handling, diffs, command help, and HTML conversion. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill includes powerful shell examples that may involve deletion, in-place edits, project-wide rewrites, configuration changes, global Git configuration changes, or outbound curl commands. <br>
Mitigation: Ask for previews, exact target paths, and diffs before allowing those operations, and review proposed commands before execution. <br>
