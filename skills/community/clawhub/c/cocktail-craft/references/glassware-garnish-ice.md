# 杯具指南

## 核心杯型与标准容量

| 杯型 | 标准容量 | 适合饮品 | 特点 |
|------|---------|---------|------|
| **Coupe** | 150-180ml | Martini, Daiquiri, Sidecar, 所有 shaken/stirred 短饮 | 宽口浅杯，香气扩散好 |
| **Nick & Nora** | 120-150ml | Martini, Manhattan | 比 Coupe 更小更深，防止洒出 |
| **Martini Glass (V-shape)** | 180-240ml | Martini 的经典选择 | 视觉优雅，但容易洒，专业酒吧逐渐弃用 |
| **Rocks / Old Fashioned** | 250-350ml | Old Fashioned, Negroni (on rocks), 所有加冰短饮 | 厚底矮杯，适合捣压和加冰 |
| **Highball / Collins** | 300-400ml | Gin & Tonic, Mojito, 所有长饮 | 高瘦杯，保持碳酸、减少冰融化 |
| **Julep Cup** | 250-350ml | Mint Julep, 部分 Smash | 金属杯（锡/铜/不锈钢），导热快，外结霜 |
| **Copper Mug** | 350-400ml | Moscow Mule | 铜杯是 Mule 标志，保持低温 |
| **Tiki Mug** | 350-500ml | Mai-Tai, Zombie, 所有 Tiki 饮品 | 陶瓷/木制，任何造型都行 |
| **Flute** | 150-180ml | Bellini, Mimosa, French 75, 所有气泡酒鸡尾酒 | 细长杯身保持气泡更持久 |
| **Irish Coffee Glass** | 240-300ml | Irish Coffee, Hot Toddy | 耐热玻璃，有把手 |
| **Sour Glass** | 120-150ml | Pisco Sour, Whiskey Sour | 略比 Coupe 窄，泡沫更集中 |
| **Shot Glass** | 30-60ml | 纯饮、分层 shot | — |

## 选择原则

- 无冰短饮（Stirred/Shaken + Strain）→ Coupe / Nick & Nora
- 加冰短饮 → Rocks 杯
- 含气泡长饮 → Highball / Collins
- 含蛋清/泡沫的 Sour → Sour 杯（泡沫更集中、更好看）
- 碎冰饮品 → Julep Cup / Tiki Mug
- 热饮 → 耐热玻璃

# 装饰指南

## 柑橘装饰

### Twist（扭皮）
- 用削皮刀削一条 5-7cm 的果皮，避开白色内层（pith 会苦）
- 在杯口上方拧转（释放精油喷雾到酒面），然后绕在杯沿或丢入酒中
- 适合：Martini, Manhattan, Old Fashioned

### Wheel / Slice（轮切）
- 横向切 3-5mm 厚的圆片
- 适合：长饮杯沿（Gin & Tonic, Bloody Mary）

### Wedge（楔形）
- 纵向切 1/6-1/8 瓣
- 适合：挤压入酒（Moscow Mule, Ranch Water）

### Zest（皮屑）
- Microplane 直接刨在酒面
- 适合：给酒表面加一层柑橘油雾（Negroni, Daiquiri）

## 其他常见装饰

| 装饰 | 适用饮品 | 处理方式 |
|------|---------|---------|
| **Mint Sprig** | Mojito, Julep, Southside | 拍打激活香气后插入 |
| **Olive** | Martini | 1-3 颗串签，dry Martini 标配 |
| **Cocktail Onion** | Gibson | Martini 变体，cocktail 小洋葱 |
| **Luxardo Cherry** | Manhattan, 大多数 Stir 饮品 | 不要用鲜红的 maraschino 樱桃 |
| **Rosemary Sprig** | 冬季/草本风格 | 可轻微炙烤增香 |
| **Edible Flower** | 花香饮品、特殊场合 | Pansy, Orchid, Nasturtium |
| **Cinnamon Stick** | Tiki, 热饮 | 既是装饰也是搅拌棒 |
| **Coffee Beans** | Espresso Martini | 3 颗浮在泡沫表面 |
| **Nutmeg Grating** | Flip, Eggnog | 现刨风味远超预磨粉 |
| **Salted Rim** | Margarita, Salty Dog | 半圈盐（只盐外面，留一半不盐供饮用） |
| **Sugared Rim** | Sidecar, Lemon Drop | 类似盐边，用砂糖 |

# 冰处理指南

## 冰的类型

| 类型 | 制作方式 | 用途 |
|------|---------|------|
| **Large Cube (5x5cm)** | 专用冰模或切冰 | Old Fashioned, Negroni on rocks——融化慢、稀释少 |
| **Large Sphere** | 球形冰模 | 视觉效应 + 同样融化慢 |
| **Standard Cube (3x3cm)** | 标准冰格 | Shake/Stir 用 |
| **Cracked Ice** | 用冰锤敲碎标准冰块 | Build 或 Shake 当需要更多稀释时 |
| **Crushed Ice** | 碎冰机 / Lewis Bag + 木槌 | Mint Julep, Tiki 饮品, Swizzle |
| **Block Ice (15cm+)** | 大容器冷冻 | Punch Bowl 用，超大块冰融化极慢 |
| **Collins Spear** | 专用冰格 | Highball 长饮用——从杯底到杯顶的一整条冰 |

## 冰的质量

- **透明冰**：用 directional freezing（定向冷冻）——只从一个方向结冰，气泡被推到另一端。用小型保温箱在冷冻室里慢冻 24 小时。
- **白冰**：普通冰格冷冻，里面有气泡和裂纹。功能上没问题，视觉上不够专业。
- **湿冰 vs 干冰**：刚从冷冻柜拿出的冰表面有霜（湿冰），需要先在水里涮一下再用，否则 Shake 时稀释不可控。

## 技法注意事项

- **Stir 前**：冰要先在 mixing glass 里预冷，倒掉融水，再加酒——避免初始稀释过量
- **Shake 前**：用结实的大块冰，不要用碎冰（碎冰融化太快 = 过度稀释）
- **Build**：先加冰再加酒 → 冰会浮起，可能导致溅出；先加酒再加冰更安全
- **Mint Julep**：必须用碎冰填满整个杯子，使劲压实——碎冰之间的缝隙形成冰水层，杯子外结霜才是对的
