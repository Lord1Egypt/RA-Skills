# cn-text-statistics

文本统计工具。统计文本的详细信息（字符、词频、行数等）。

## 使用方法

```bash
python3 scripts/text_stats.py --text "要统计的文本"
python3 scripts/text_stats.py --file document.txt --freq
```

纯Python标准库。
