# 髓母细胞瘤分子分型与预后评估

## 触发条件
- 病理诊断明确为髓母细胞瘤
- 患者为儿童或青少年
- 肿瘤位于小脑蚓部或第四脑室
- 已完成分子分型检测

## 执行步骤

1. **确认组织学亚型**（基于光镜）：
   - 经典型：瘤细胞圆形、核深染、胞质少，可见Homer-Wright菊形团
   - 促纤维增生型/结节型
   - 广泛结节型
   - 大细胞/间变型（预后最差）

2. **验证免疫表型**：
   - 确认表达突触素（Syn）、神经元特异性烯醇化酶（NSE）、NeuN
   - 注意GFAP阳性可能提示胶质分化

3. **整合分子分型结果**：
   - WNT激活型（约15%）：预后极好，对放疗高度敏感
   - SHH激活型：中等预后，可考虑靶向治疗
   - 第3组（非WNT/非SHH）：侵袭性强，预后差
   - 第4组（最常见，约44%）：中等预后

4. **评估临床行为与风险**：
   - 所有亚型均需评估脑脊液播散风险（高）
   - 推荐行脑脊液细胞学检查

5. **制定治疗建议**：
   - WNT型：可考虑降阶治疗（减少放化疗强度）
   - 其他亚型：标准三联疗法（手术 + 全脑全脊髓放疗 + 化疗）

> **注意**：本技能不适用于其他胚胎性肿瘤（如AT/RT）或成人胶质瘤。
