# 企微私域运营 SOP 与内容日历

> 本文件为 `insurance-private-domain-ops` Skill 配套参考

## 一、企微1v1运营 SOP

### 1.1 新客户激活 SOP（0-7天）

```
Day 0（保单生效当天）
├─ 发送：企微感谢消息 + 电子保单 + 服务承诺
└─ 话术：XX先生/女士，您的XX保单已生效！我是您的专属顾问XXX，
         有什么需要随时联系我。附上您的电子保单：[链接]

Day 3
├─ 发送：健康小贴士/服务介绍
└─ 话术：XX先生/女士，给您分享一个健康小知识：[内容]，有任何健康保障问题可以随时问我。

Day 7
├─ 发送：满意度回访
└─ 话术：XX先生/女士，您的保单已生效一周了，请问对服务还满意吗？
         有任何问题我都第一时间帮您解决。
```

### 1.2 沉默客户激活 SOP

```
触发条件：客户≥180天无任何互动
      ↓
第1次触达（企微消息）
├─ 内容：个性化价值内容（如：XX岁专属保障方案/近期健康热点）
└─ 观察48小时内是否回复
      ↓（无回复）
第2次触达（7天后）
├─ 内容：福利活动/产品升级通知
└─ 观察72小时
      ↓（无回复）
第3次触达（短信）
├─ 内容：简短提醒+企微二维码
└─ 如仍未回复，降低触达频率（季度触达）
```

## 三、公众号推送节奏

```
每月推送频次：8-12篇
├─ 产品类：2-3篇/月
├─ 健康类：3-4篇/月
├─ 理赔/服务类：1-2篇/月
└─ 热点/品牌类：2-3篇/月

推送时间：
├─ 早间 7:30-8:30（通勤时间，打开率高）
└─ 午间 12:00-13:00（休息时间，阅读率高）
```
