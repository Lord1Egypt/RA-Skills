## О проекте

Этот проект систематически объединяет основные компетенции клинической диагностики, охватывая **183 ключевых диагностических навыка** в 14 основных категориях, основанных на учебнике «Диагностика», 10-е издание (People's Medical Publishing House).

**Целевая аудитория**: Врачи, студенты-медики, ординаторы, лаборанты, специалисты по визуализации, медсестры и работники общественного здравоохранения

**Учебник**: «Диагностика», 10-е издание, PMPH (ISBN: 978-7-117-35276-8)

**⚠️ Предупреждение ⚠️**
- Этот проект предназначен **только для образовательных целей** и **клинического обучения** и не заменяет профессиональное медицинское заключение.
- Все диагностические решения должны приниматься квалифицированными клиницистами с учетом индивидуальных особенностей пациента.
- Исходный контент может не отражать последние клинические рекомендации; всегда сверяйтесь с текущими официальными руководствами, местными протоколами и мнением специалиста.
- Внедрите системные политики медицинской безопасности, требующие передачи решений по диагностике, назначению лечения и неотложной помощи квалифицированным врачам.
