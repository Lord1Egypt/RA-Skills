# Hijack Defender — Enforcement Readiness Checklist

Use this checklist before submitting any enforcement action. Incomplete evidence packages lead to rejected cases and allow the hijacker to continue operating.

---

## 1. Detection & Classification (5 items)
- [ ] Hijacker identified with full seller name and seller ID captured
- [ ] Violation type classified: Counterfeit / Unauthorized Reseller / Listing Copy / ASIN Piggyback
- [ ] Hijacker's price vs. your price documented
- [ ] Hijacker's seller account age and feedback rating noted
- [ ] Detection date logged (for measuring time-to-resolution later)

## 2. Evidence — Listing Documentation (6 items)
- [ ] Screenshot of hijacker's offer page (seller name, price, "Sold by" field visible)
- [ ] Screenshot of your listing showing you as brand owner
- [ ] Screenshot of "All Offers" display showing the specific hijacker offer
- [ ] Hijacker's seller profile and feedback page captured
- [ ] Date/timestamp visible in all screenshots
- [ ] All screenshots labeled with ASIN, seller name, and date

## 3. Evidence — Brand Documentation (4 items)
- [ ] Trademark registration number confirmed (USPTO TESS or equivalent)
- [ ] Brand Registry enrollment confirmation on hand
- [ ] Manufacturer invoice or purchase order confirming you as authorized seller
- [ ] Authorized reseller list reviewed (is this seller on it? If yes, investigate MAP/contract violation separately)

## 4. Test Buy (for Counterfeit Cases) (7 items)
- [ ] Test buy placed from a separate non-brand Amazon account
- [ ] Order shipped to a neutral address (not brand business address)
- [ ] Order ID documented
- [ ] Package exterior photographed upon arrival
- [ ] Side-by-side product comparison photos taken (authentic vs. received)
- [ ] Packaging, inserts, and all materials preserved
- [ ] Comparison photos clearly labeled: "authentic" vs. "hijacker"

## 5. Amazon Brand Registry Report (6 items)
- [ ] Report submitted at brand.amazon.com → Report a Violation
- [ ] Correct violation type selected (Counterfeit vs. Trademark)
- [ ] ASIN and infringing seller URL provided
- [ ] Trademark registration number included
- [ ] Evidence photos attached (4–8 photos, clearly labeled)
- [ ] Case reference number saved

## 6. Cease-and-Desist (4 items)
- [ ] C&D message sent via Amazon Buyer-Seller Messaging to hijacker
- [ ] Template customized with ASIN, trademark number, and specific violation details
- [ ] 48-hour deadline stated in the message
- [ ] Copy of sent message saved (screenshot or download)

## 7. Post-Resolution (6 items)
- [ ] Hijacker's offer confirmed removed from listing
- [ ] Reviews posted during hijack period reviewed for counterfeit complaints
- [ ] Review removal requests filed for reviews about products seller didn't ship
- [ ] Monitoring continued for 30 days post-removal (hijacker may return)
- [ ] Estimated revenue lost during hijack period calculated
- [ ] Incident logged in tracker with resolution date and method

## 8. Prevention (5 items)
- [ ] Amazon Transparency enrollment status reviewed / initiated
- [ ] Brand Gating request filed for affected ASIN (if not already gated)
- [ ] Supply chain investigation launched (how did the unauthorized inventory get there?)
- [ ] Supply chain leak closed (distributor agreement, liquidation channel, etc.)
- [ ] Authorized reseller policy published and accessible

---

**Total items: 43**
**Minimum before submitting a counterfeit report: Sections 1–4 complete**
**Minimum before submitting an unauthorized reseller report: Sections 1–3 complete (no test buy required)**
