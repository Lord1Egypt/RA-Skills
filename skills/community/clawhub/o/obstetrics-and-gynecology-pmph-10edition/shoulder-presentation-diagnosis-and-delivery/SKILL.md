# 肩先露诊断与分娩处理

## 适用场景
- 妊娠足月孕妇
- 胎膜已破且宫口开大（用于阴道检查确认）
- 腹部触诊提示横位（子宫横椭圆、宫底低、耻骨上空虚）

## 执行步骤

### 第一步：确诊肩先露
1. **腹部检查**：
   - 子宫呈横椭圆形，宫底高度低于孕周
   - 耻骨联合上方空虚
   - 宫体一侧触胎头，另一侧触胎臀
   - 脐周两侧胎心最清晰

2. **阴道检查**（需胎膜已破、宫口开大）：
   - 触及肩胛骨、肋骨、腋窝或肩峰
   - 根据腋窝尖端方向判断胎头位置（指向母体左/右侧）
   - 肩胛朝向母体后方为**肩后位**，反之为**肩前位**

3. **握手法鉴别脱出肢体**：
   - 遵循“前反后同”原则：
     - 肩左前位 → 脱出右手 → 仅能与检查者**右手**相握
     - 肩左后位 → 脱出左手 → 仅能与检查者**左手**相握

4. **超声辅助**：确认胎头、脊柱及胎心位置，明确胎方位

### 第二步：制定处理方案
- **妊娠期发现**：试行外倒转术（ECV）；失败则提前住院待产
- **分娩期处理**：
  - **活胎初产妇**：无论宫口大小，**立即剖宫产**
  - **经产妇**：
    - 首选剖宫产
    - 若宫口≥5cm、胎膜已破但羊水未尽、胎儿不大 → 可在麻醉下行**内转胎位术**转为臀先露后阴道分娩
  - **双胎第二儿肩先露**：立即行内转胎位术
  - **先兆子宫破裂**：无论胎儿存活与否，**紧急剖宫产**；若已破裂伴感染，考虑子宫切除
  - **胎死宫内 + 无子宫破裂 + 宫口开全**：全麻下行**毁胎术**，术后检查软产道并缝合，预防出血与感染

> **禁忌**：本技能不适用于头先露或臀先露；活胎嵌顿时不可行内转胎位术。
