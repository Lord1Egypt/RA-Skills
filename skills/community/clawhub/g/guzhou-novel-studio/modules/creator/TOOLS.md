# 创作模块整合说明

本目录整合了孤舟小说工作室的核心创作能力。

## 工具引用

### 1. novel-creation skill
> 短篇/长篇创作核心工具

**触发条件**：
- 用户请求创作微小说（800-2200字）
- 用户请求创作盐选短篇（知乎8000字起）
- 用户请求长篇大纲规划与逐章续写
- 用户请求多风格模仿（说书人、讲故事、猫腻、金庸、古龙等）
- 用户请求多题材创作（都市怪谈、侦探、恐怖、科幻、民俗志异等）
- 用户请求生成配图

**使用方式**：
```
skill_load(skill="novel-creation")
```

### 2. qingshan-novel-writing skill
> 专业技法支持工具

**触发条件**：
- 用户请求使用双线镜像叙事
- 用户请求人物小传法
- 用户请求草蛇灰线伏笔技法
- 用户请求诗化语言风格
- 用户请求灰色人设塑造
- 用户请求无系统硬核智斗设计
- 用户请求技法学习或指导

**使用方式**：
```
skill_load(skill="qingshan-novel-writing")
```

## 模块能力映射

| 创作类型 | 工具 | 描述 |
|---------|------|------|
| 微小说 | novel-creation | 800-2200字短篇故事 |
| 盐选短篇 | novel-creation | 知乎盐选风格8000字起 |
| 长篇创作 | novel-creation + studio | 大纲规划+逐章续写 |
| 技法应用 | qingshan-novel-writing | 专业叙事技法 |
| 风格模仿 | novel-creation + stylist | 多风格写作 |
| 插画配图 | novel-creation | 图片生成能力 |

## 协作流程

```
调研模块 → 风格模块 → 创作模块 → 质检模块
              ↓            ↓
         Style-DNA    章节正文
              ↓            ↓
         风格校准    章节终稿
```

## 文件规范

创作输出遵循统一格式：
- 章节文件：`chapters/chapter-{序号}.md`
- 章节元信息：末尾包含字数、伏笔、人物状态等
- 输出编码：UTF-8
- 格式：Markdown
