This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and operators use this skill to install and configure security governance controls for OpenClaw agents. It focuses on policy enforcement, auditing, identity signing, budget limits, process isolation, and emergency shutdown behavior. <br>

## Known Risks and Mitigations: <br>
Risk: The skill describes broad control over an agent and full activity logging without enough installation, privacy, or rollback detail. <br>
Mitigation: Before installation, obtain the full source repository, exact install command, pinned release, SAMMA_API_KEY scope, data-flow documentation, audit-log storage and retention rules, and disable or uninstall procedure from the publisher. <br>
Risk: Using the skill with sensitive or production agents may expose activity data or introduce hard-to-audit controls until implementation details are reviewed. <br>
Mitigation: Avoid sensitive or production deployment until the publisher documentation and security posture have been reviewed and tested in a controlled environment. <br>
