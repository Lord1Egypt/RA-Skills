# 视频谱 — 七大要素

视频谱是执行层面的镜头语言工具箱。每个分镜按需取用，N=0~7，不强求全覆盖。

---

## 1. 主体

角色/物体的详细描述，是画面的焦点。

- 外貌特征（体型、肤色、年龄）
- 服装装束
- 道具武器
- 眼神表情

**要点：** 多个主体时每个都需要独立的角色卡。

---

## 2. 景别

决定画面中主体的大小与空间关系。

| 景别 | 说明 | 用途 |
|------|------|------|
| 大远景 | 主体极小，环境为主 | 展现宏大规模、孤独感 |
| 远景 | 主体全身+环境 | 展现空间关系 |
| 全景 | 主体全身 | 展现动作全貌 |
| 中景 | 主体膝上/腰上 | 展现互动与情绪 |
| 近景 | 主体胸部以上 | 展现表情细节 |
| 特写 | 主体局部 | 展现关键细节 |
| 大特写 | 极度放大 | 极致细节冲击 |

---

## 3. 动作

主体的运动状态与细节。

- 运动轨迹（直线/弧线/螺旋）
- 速度（极速/中速/慢速/静止）
- 力量感（爆发/柔和/沉重）
- 微动作（咬牙/握拳/呼吸）

---

## 4. 场景

环境、光影、氛围的综合描述。

- 空间设定（室内/室外/地下/空中）
- 天气与自然条件
- 光源类型与方向
- 色彩基调
- 氛围情绪

---

## 5. 构图

画面元素的排列方式，引导观众视线。

| 构图类型 | 特点 | 适用场景 |
|---------|------|---------|
| 对称构图 | 左右/上下对称，庄重稳定 | 宫殿、仪式、对峙 |
| 三分构图 | 主体放在三分线交叉点 | 通用，自然舒适 |
| 引导线 | 线条引导视线指向主体 | 道路、走廊、河流 |
| 负空间 | 大面积留白，主体极小 | 孤独、渺小、末世 |
| 中心构图 | 主体居中，视觉冲击强 | 特写、关键瞬间 |
| 框架构图 | 前景元素框住主体 | 门窗、洞口、枝叶 |
| 对角线 | 对角方向排列，动感强 | 战斗、追逐、运动 |
| 黄金分割 | 1:1.618比例排列 | 经典美学，精致画面 |

---

## 6. 镜头视角

摄影机相对于主体的观察角度。

| 视角 | 特点 | 情绪效果 |
|------|------|---------|
| 仰视 | 从低处向上看 | 威压、崇敬、力量感 |
| 俯视 | 从高处向下看 | 渺小、无助、全局掌控 |
| 平视 | 与主体等高 | 客观、平等、日常 |
| 第一人称 | 主体主观视角 | 沉浸感、紧张感 |
| 过肩视角 | 从另一人肩后拍摄 | 对话、对峙 |
| 航拍镜头 | 高空俯瞰 | 宏大、辽阔、上帝视角 |
| 俯冲镜头 | 从高空快速下降 | 冲击力、进入感 |
| 低角度仰拍 | 接近地面仰望 | 威慑、压迫、力量 |

---

## 7. 镜头运动

摄影机本身的运动方式。

| 运动 | 特点 | 适用场景 |
|------|------|---------|
| 跟随 | 摄影机跟随主体移动 | 追逐、奔跑、飞行 |
| 环绕 | 摄影机围绕主体旋转 | 展示主体全貌、战斗 |
| 拉远 | 摄影机远离主体 | 揭示环境、结束感 |
| 推进 | 摄影机靠近主体 | 聚焦、紧张、发现 |
| 静止 | 摄影机不动 | 对峙、凝视、沉思 |
| 平摇 | 摄影机水平旋转 | 扫视环境、跟随移动 |
| 升降 | 摄影机垂直移动 | 揭示高度、场景转换 |
| 甩镜 | 快速转动镜头 | 惊吓、转折、紧迫 |
| 手持 | 人手持有晃动 | 纪实感、紧张、混乱 |
| 变焦 | 改变焦距不移动机位 | 聚焦/揭示、希区柯克效果 |

---

## 使用原则

1. **工具箱，不是清单** — 按需取用，不强行堆砌
2. **N=0~7** — 每个分镜用多少个要素由画面需要决定
3. **判断标准** — 这个要素用上去，画面更好吗？
4. **自然融入** — 构图/视角/运镜写入描述段落，不机械罗列标签
