# 中枢性尿崩症诊断（禁水-加压素试验）

## 适用对象
- 疑似中枢性尿崩症的稳定患儿

## 前提条件
- 已排除高渗性利尿、肾性尿崩症等其他多尿病因
- 患儿一般状况稳定，能配合试验

## 禁忌与终止条件
- **禁忌**：严重脱水、高钠血症或无法配合者不宜进行
- **试验中立即终止**：出现严重脱水症状、体重下降达5%、或满足其他终止标准

## 执行流程

### 1. 禁水试验准备
- 自前一日晚7–8时开始禁食水直至试验结束
- 试验当日晨8时排空膀胱，测量体重，采集血液检测血钠及血渗透压

### 2. 禁水期间监测
- 每小时排尿一次，记录尿量、测定尿渗透压（或尿比重）和体重
- **终止条件（满足任一即停止）**：
  - 相邻两次尿渗透压差值连续两次＜30 mmol/L
  - 体重下降达5%
  - 尿渗透压≥800 mmol/L
- 达终止条件后再次采血测血渗透压和血钠

### 3. 结果初步判读
- **正常儿童**：尿量减少，尿渗透压＞800 mmol/L，血钠和血渗透压正常
- **尿崩症患者**：持续排出低渗尿，血钠＞145 mmol/L，血渗透压＞295 mmol/L，体重下降3%～5%

### 4. 加压素试验（紧接禁水试验后）
- 皮下注射垂体后叶素5U 或 精氨酸加压素0.1U/kg
- 2小时内多次留尿测定尿渗透压
- **判定标准**：
  - 尿渗透压峰值上升＞给药前50% → 完全性中枢性尿崩症
  - 上升9%～50% → 部分性中枢性尿崩症
  - 上升＜9% → 肾性尿崩症

### 5. 辅助检查建议
- 结合血浆AVP测定：中枢性尿崩症AVP低于正常，肾性尿崩症AVP正常或升高但尿不浓缩
- 行头颅MRI排除下丘脑-垂体区域肿瘤

## 关键变量监控
- 尿渗透压（mmol/L）
- 血清钠浓度（mmol/L）
- 体重下降百分比（%）
