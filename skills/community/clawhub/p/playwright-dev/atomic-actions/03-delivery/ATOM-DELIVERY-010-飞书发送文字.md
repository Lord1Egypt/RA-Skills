# ATOM-DELIVERY-010 - 飞书发送文字

> 版本：V1.0  
> 状态：🟡 待规范  
> 最后更新：2026-03-07

---

## 📋 动作定义

**名称：** 飞书发送文字  
**分类：** 交付层（Delivery Layer）  
**编号：** ATOM-DELIVERY-010

**一句话描述：** 发送 Markdown 格式的文字消息到飞书

---

## 🎯 输入输出

### 输入
- **类型：** Markdown 文本
- **内容：** 消息内容

### 输出
- **类型：** 飞书消息
- **返回：** messageId

---

## ⚙️ 偏好设置

### 格式要求
- Markdown 格式
- emoji 点缀（每章节前加 icon）
- 结构化排版（标题/列表/表格）

### 消息长度
- **短消息：** <500 字符（直接发送）
- **长消息：** >500 字符（分段落，加标题）
- **超长消息：** >1500 字符（建议用文件附件）

### 目标用户
- **默认：** 当前会话用户（ou_e3a0d4a64a9e0932ee919b97f17ec210）
- **飞书 ID 格式：** ou_*（个人）或 chat_*（群聊）

---

## 📝 操作步骤

```powershell
# 1. 准备消息内容
$message = @"
## 📢 标题

核心内容...

### 详情
- 要点 1
- 要点 2
"@

# 2. 发送飞书消息
Invoke-MessageSend `
    -Action "send" `
    -Message $message `
    -Target "ou_e3a0d4a64a9e0932ee919b97f17ec210"

# 3. 确认
Write-Host "✅ 消息已发送"
```
