## 一、核心方法论体系

### 1.1 四大家广告方法论

**奥美（Ogilvy）—— 品牌叙事法**
核心：不是说服，是让人爱上你
工具：大卫·奥格威品牌形象论 / 品牌金字塔 / 360度品牌接触点
应用：创始人故事、品牌宣言、情感共鸣文案

**BBDO —— 品牌差异点进攻法**
核心：先找差异化，再放大差异
工具：品牌阶梯（Attributes → Benefits → Values → Brand Essence）
应用：竞品分析、主打卖点提炼、视觉差异化设计

**盛世长城（Saatchi & Saatchi）—— 绝对吸引力法则**
核心：没有注意力，就没有品牌
工具：PESO模型（Paid/Earned/Shared/Owned）
应用：钩子设计、传播爆发点、病毒传播预测

**扬罗必凯（Young & Rubicam）—— 品牌资产评估**
核心：品牌是资产，不是包装
工具：BrandZ品牌资产评估 / 品牌动力矩阵（Power/Diffusion）
应用：品牌健康度评估、品牌升级路径

---

### 1.2 七政·OASIS推演系统

核心架构：SIR+IC双模型叠加

**SIR模型（舆情传播预测）**
- S：Susceptible（易感人群）
- I：Infected（感染者/传播者）
- R：Recovered（免疫/衰退）
- K值：传播系数（K>1.5为爆款级）

**IC模型（舆情演化曲线）**
- I：Infection（舆情强度，0-100）
- C：Conversion（转化率）
- 轮次推演：R1→R2→R3...→Rn
- 极化指数>0.5为危险信号

**应用场景**
- 开业推广预测：K值/出圈时间/库存售罄时间
- 舆情危机推演：R0值/峰值热度/消退时间/竞品抢客次数
- 爆款活动预测：最优折扣/最优库存/最优时长

## 二、品牌营销完整流程（5步闭环）

### 第1步：品牌定位诊断（Brand Diagnosis）

**输入：** 企业基础资料（产品/创始人/市场定位/竞品）
**输出：** SWOT分析 + 品牌定位报告

**执行框架：**
```
1. 收集企业资料（官网/公众号/抖音/小红书/竞品官网）
2. SWOT四维分析（优势/劣势/机会/威胁）
3. 市场扫描（竞争格局/切入机会/定价区间）
4. 品牌差异化定位（Brand Ladder → Brand Essence）
5. 输出：品牌定位报告（SWOT+战略+核心主张）
```

---

### 第2步：品牌核心叙事构建（Brand Narrative）

**输入：** 品牌定位报告 + 创始人资料
**输出：** 品牌故事 + Slogan体系 + KV描述

**执行框架：**
```
1. 品牌故事（长版：公众号/品牌手册）
2. 品牌故事（短版：抖音字幕/小红书封面）
3. 创始人语录（IP内容素材）
4. Slogan体系（5层级：主口号/承诺/情感/号召/促销）
5. 主视觉KV描述（画面构成/色调规范/拍摄要求）
6. 产品线话术（每条产品线30字核心话术）
```

---

### 第3步：爆款推广推演（Promotion Simulation）

**输入：** 品牌资料 + 促销参数（折扣/库存/时长/预算）
**输出：** 舆情演化曲线 + 最优方案建议

**OASIS执行脚本：**
```bash
python3 /workspace/skills/qizheng-oasis/run_v2.py \
  --scenario "新店开业" \
  --n_agents 1000 \
  --stock 500 \
  --hours 72 \
  --rounds 4
```

**输出解读：**
- K值>1.5：爆款概率高，可加大投放
- 出圈时间<24小时：需提前备好第二波内容
- 库存售罄时间>72小时：库存充足，可持续促销

---

### 第4步：舆情危机推演（Crisis Simulation）

**输入：** 危机场景描述 + 品牌应急能力
**输出：** 舆情演化曲线 + 响应策略效果量化

**OASIS危机脚本：**
```bash
python3 /workspace/skills/qizheng-oasis/run_crisis.py \
  --crisis "食品安全" \
  --n_agents 1000 \
  --hours 48 \
  --rounds 6
```

**输出解读：**
- R0值>3：爆点级舆情，需CEO直接介入
- 极化指数>0.5：情绪已极化，常规公关无效
- 竞品抢客次数>100次：同步启动老客户锁客计划

---

### 第5步：完整执行方案输出（Execution Package）

**输入：** 品牌叙事 + 推演结果
**输出：** 4文件执行包

| 文件 | 内容 | 对应方法论 |
|------|------|-----------|
| 品牌核心文案包 | Slogan/故事/KV话术 | 奥美叙事 + BBDO差异化 |
| 地推执行手册 | D1-D14每天动作清单 | 盛世长城PESO + 执行SOP |
| KOC人设卡×20 | 用户画像+种草话术 | 扬罗必凯分型 + 真实用户洞察 |
| 危机应对预案 | 5大场景+黄金1小时流程 | 盛世长城危机PR + 量化响应 |

## 三、品牌诊断标准问卷

**在开始品牌项目前，必填：**

```
Q1: 品牌名称 + 主营业务？
Q2: 核心差异化是什么？（用一句话）
Q3: 主要竞争对手是谁？（列举3个）
Q4: 目前最大的痛点是什么？（获客/留存/品牌认知/转化）
Q5: 目标客群是谁？（年龄/城市/消费习惯）
Q6: 有没有创始人IP故事？
Q7: 这次推广的目标是什么？（新店开业/产品上新/品牌升级/危机应对）
Q8: 有没有时间节点要求？
Q9: 预算范围？（决定推广规模）
Q10: 已有哪些素材？（官网/产品图/视频/文案）
```
