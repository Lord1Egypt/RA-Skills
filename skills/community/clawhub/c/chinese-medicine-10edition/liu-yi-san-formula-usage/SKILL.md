# Liu Yi San Formula Usage

## When to Use
当患者符合以下条件时使用六一散：
- 主症：身热烦渴、小便不利或泄泻
- 病机：暑湿热下注
- 适用病证：膀胱炎、尿道炎、夏季皮炎、新生儿腹泻、病毒性肠炎、湿疹、带状疱疹等属暑湿热下注者

**禁忌与慎用**：
- 不适用于寒湿证或纯虚无实热者
- 脾胃虚寒、滑泄不禁者慎用

## Execution Steps
1. **确认组成比例**：滑石六两，甘草一两（6:1），取“天一生水，地六成之”之意。
2. **辨析药物作用**：
   - 滑石为君药，甘淡性寒，清解暑热、通利水道；
   - 生甘草为臣药，甘平偏凉，清热泻火、益气和中，兼护胃气、防伤津。
3. **选择用法**：
   - 传统用法：研细末，每服三钱（约9克），加蜜少许，温水调服，每日三次；
   - 现代用法：可作汤剂水煎服。
4. **注意鉴别**：
   - 原名益元散，又名天水散；
   - 若加朱砂则为另一方剂（如益元散加朱砂），不可混淆。

> 提示：使用前务必排除寒湿或虚寒证候，确保病机属实热夹湿。
