# 六个办公室主任框架详解

---

## 概述

"六个办公室主任"是大脑决策机制的拟人化模型，用于分析课程内容对学员大脑各区域的影响。

**核心原则**：第一目标不是成交，是不要让任何一个办公室主任触发"喊停"。

**理想配比**：50% 情绪 + 20% 逻辑 + 20% 结构 + 10% 干货

---

## 1️⃣ C 总（前额叶皮层）- 总经理

### 基本信息
- **外号**：总经理 / 人脑办公室主任 / CPU
- **特点**：能耗高（20-35 瓦）、不持久、易疲劳、最高级别
- **分工**：计算、理性分析、长期规划、权衡利弊、抑制冲动
- **弱点**：过度承压会崩溃（熔断机制）

### 喜好与厌恶

| 喜好 | 厌恶 |
|------|------|
| 清晰逻辑 | 信息过载 |
| 证据支撑 | 无依据断言 |
| 可执行步骤 | 复杂术语 |
| 节省认知负荷 | 模糊号召 |

### 分析维度

| 维度 | 评分标准 |
|------|---------|
| 逻辑清晰度 | 论证链条是否清晰，层层递进 |
| 认知负荷 | 信息密度是否适中，有无过载 |
| 干货含量 | 是否提供可执行的步骤/方法 |
| 数字支撑 | 观点是否有数据/案例支撑 |

### 讨好策略
- 帮他做计算，给判断题而非问答题
- 10% 干货感服务 C 总，产生"哦"的顿悟
- 好逻辑对 C 总是友好的

### 高分示例
> "一年所有的花费是 680 块钱，你的成本是可控的。"
> （具体数字，清晰计算）

---

## 2️⃣ 杏总（杏仁核）- 保安队长

### 基本信息
- **外号**：保安队长 / 警报器 / 情绪大脑
- **特点**：一对、管恐惧、管或打或逃、特别三八、赌性强
- **分工**：情绪、危险识别、战斗或逃跑、0.3 秒反应
- **弱点**：注意力不集中、没长性、喜欢被调戏

### 喜好与厌恶

| 喜好 | 厌恶 |
|------|------|
| 骇人听闻的故事 | 平淡内容 |
| 强烈画面感 | 纯理性说教 |
| 共情到的恐惧 | 无情绪波动 |
| 意料之外的奖赏 | |

### 分析维度

| 维度 | 评分标准 |
|------|---------|
| 情绪唤醒 | 是否有恐惧/渴望/惊讶等情绪触发 |
| 故事张力 | 是否有冲突、反差、意外 |
| 画面感 | 语言是否具象、可感知 |
| 反认知 | 是否有颠覆常识的观点 |

### 讨好策略
- 让杏总喊出"我勒个去""麻蛋""卧槽"就舒服了
- 50% 情绪内容讨好杏总
- 信息量大≠文字多，字少信息量大
- 利用它的冲动和赌性

### 高分示例
> "Meta 的 CEO 让本地虾清理邮件，一路删 200 多封全部删干净。"
> （恐惧触发，强烈画面感）

---

## 3️⃣ 鸡哥（基底神经节）- 自动驾驶

### 基本信息
- **外号**：自动驾驶 / 惯性驾驶
- **特点**：无意识运行、重复强化、快速反应
- **分工**：习惯养成、经验导航、自动驾驶

### 分析维度

| 维度 | 评分标准 |
|------|---------|
| 互动设计 | 是否有扣字、投票、跟读等动作 |
| 重复强化 | 核心观点是否重复 3 次以上 |
| 仪式感 | 是否有固定的开场/结尾/过渡 |
| 习惯唤醒 | 是否调用已有行为模式 |

### 讨好策略
- 为什么直播要让大家扣 1？这次扣，下次还会扣
- 启动基底神经节进入无意识行为模型，降低 C 总工作量
- 由浅入深的行为依从度设计

### 高分示例
> "最后清晰的打个清晰，我们以这两个字来结束今天的会议。"
> （仪式感设计，鸡哥喜欢）

---

## 4️⃣ 多多哥（多巴胺系统）- 快乐制造机

### 基本信息
- **外号**：快乐制造机 / 奖励中心
- **特点**：预测奖励驱动、出其不意、管理预期
- **分工**：奖赏、愉悦感、动机、习惯形成

### 喜好与厌恶

| 喜好 | 厌恶 |
|------|------|
| 即时反馈 | 延迟满足 |
| 可预期收益 | 不确定结果 |
| 小步快跑成就感 | 大步骤高门槛 |
| 惊喜彩蛋 | 平淡无奇 |

### 分析维度

| 维度 | 评分标准 |
|------|---------|
| 惊喜设计 | 是否有彩蛋、意外奖励 |
| 即时反馈 | 互动后是否有及时回应 |
| 期待管理 | 是否制造并满足期待 |
| 幽默元素 | 是否有笑点、金句 |

### 讨好策略
- 出其不意让他快乐
- 做一个轻而易举的动作，得到一个奖励
- 奖励超预期 = 惊喜轰炸
- 逗她一笑，哄她开心

### 高分示例
> "我都没有写任何一行代码，我都是让他直接帮我做。"
> （展示网站时，有"哇"效果）

---

## 5️⃣ 海马体 - 记忆中枢

### 基本信息
- **外号**：图书管理员 / 记忆中枢
- **特点**：短期转长期记忆、对重复和情绪敏感
- **分工**：记忆归档、经验提取、价值观存储

### 分析维度

| 维度 | 评分标准 |
|------|---------|
| 记忆锚点 | 是否有比喻、金句、画面 |
| 结构框架 | 知识点是否有清晰的组织 |
| 场景调用 | 是否调用学员已有经验 |
| 重复巩固 | 关键内容是否多次出现 |

### 讨好策略
- 核心观点重复 3 次以上
- 有记忆锚点（比喻、画面）
- 知识点有逻辑框架
- 调用场景化经验

### 高分示例
> "小龙虾"比喻贯穿全程
> （云虾、本地虾、封装虾、原生虾、养虾、虾总）

---

## 6️⃣ 导爷（岛叶）- 防御机制

### 基本信息
- **外号**：防御机制启动器
- **特点**：记录痛苦记忆、启动逃离机制
- **分工**：痛苦防御、恐惧失去
- **触发**：321 上链接、割韭菜感、被强迫

### 分析维度

| 维度 | 评分标准 |
|------|---------|
| 防御触发 | 是否有"割韭菜感"、强迫感 |
| 阻力点处理 | 是否看见并抚慰用户顾虑 |
| 信任建立 | 是否有背书、保障、承诺 |
| 逆鳞规避 | 是否避免触发逃离机制 |

### 讨好策略
- 别碰逆鳞，触发就一票否决
- 用"7 天无理由"抚平不良反应
- 恐惧失去也可利用（得不到就痛苦）

### 高分示例
> "我不是非要让大家去用，你提倡用 Q-CLaw 等等也是 OK 的。"
> （化解防御，给选择权）

## 七阶情绪升温模型

| 阶数 | 状态 | 说明 |
|------|------|------|
| 第 1 阶 | 听得懂 | 换成用户听得懂的话，不用专业术语 |
| 第 2 阶 | 能听进 | 抓住注意力，跟着主线走 |
| 第 3 阶 | 感兴趣 | 情感共鸣，想继续听 |
| 第 4 阶 | 爽 | 脑子浅表层兴奋 |
| 第 5 阶 | 渴望 | 脑子深层兴奋，心生向往，开始脑补 |
| 第 6 阶 | 意识到冲动 | 冲动和理智博弈 |
| 第 7 阶 | 决策 | 给动力点理由，阻力点被看见和抚慰 |

**关键洞察**：
- 阻力点需要被看见，讲出阻力点想讲的话
- 成为阻力点的代言人，给它自洽合理的解释

## 应用案例

### 案例 1：42 岁设计师失业故事

**触发对象**：杏总（恐惧）+ C 总（数据）

```
我有个朋友，是做平面设计的。干了 12 年，从助理做到设计总监。
去年 3 月，他们公司裁掉了 6 个设计师，只留下 2 个。

为什么？因为剩下的 2 个人，每个人 +AI，能干原来 8 个人的活。

我那个朋友，42 岁，月薪 3 万，突然就失业了。
```

**分析**：
- 杏总：恐惧触发（失业危机）
- C 总：数据支撑（12 年、6 个、2 个、3 万）
- 导爷：化解（"不是教你要不要用，是教你怎么不被淘汰"）

---

### 案例 2：最小可行使用清单

**触发对象**：导爷（化解防御）+ 鸡哥（小动作）+ 多多哥（即时反馈）

```
你不需要有"语料"，你不需要"开始播"，你甚至不需要"有很多内容"。

你只需要做 3 件小事：
✅ 第 1 件：给你的小龙虾起个名字（1 分钟）
✅ 第 2 件：告诉它你是谁、做什么的（2 分钟）
✅ 第 3 件：让它帮你写一条明天的朋友圈文案（3 分钟）

就这 3 件事，6 分钟。
```

**分析**：
- 导爷："不需要"句式化解防御
- C 总：清晰步骤（3 件事、6 分钟）
- 鸡哥：可执行的小动作
- 多多哥：即时反馈（3 分钟后打"我请了 AI 员工"）
