## Description: <br>
Generates machine-actionable Product Requirements Documents for AI coding agents by running a discovery interview before drafting phased, dependency-ordered specifications with testable acceptance criteria, file structures, API contracts, and build order. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[idcesares](https://clawhub.ai/user/idcesares) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, product managers, and technical leads use this skill to turn product ideas or rough requirements into PRDs that AI coding agents can execute with minimal clarification. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated PRDs or companion agent files may encode incorrect assumptions that influence later implementation work. <br>
Mitigation: Review generated PRDs and any AGENTS.md or CLAUDE.md companion files before another coding agent follows them. <br>
