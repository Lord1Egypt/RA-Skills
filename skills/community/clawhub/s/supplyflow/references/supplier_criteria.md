# 供应商评估标准

## 评估维度与权重

### 默认权重配置

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 价格 | 30% | 市场比价、年度降幅承诺 |
| 质量 | 35% | 来料合格率、质量认证 |
| 交期 | 20% | 准时交付率、交期稳定性 |
| 服务 | 15% | 响应速度、售后支持 |

### 价格评分 (0-10)

| 得分 | 条件 |
|------|------|
| 10 | 低于市场均价 >10% |
| 9 | 低于市场均价 5-10% |
| 8 | 低于市场均价 0-5% |
| 7 | 等于市场均价 |
| 6 | 高于市场均价 0-5% |
| 5 | 高于市场均价 5-10% |
| ≤4 | 高于市场均价 >10% |

### 质量评分 (0-10)

| 得分 | 条件 |
|------|------|
| 10 | 来料合格率 ≥99.9% |
| 9 | 来料合格率 99.5-99.9% |
| 8 | 来料合格率 99-99.5% |
| 7 | 来料合格率 98-99% |
| 6 | 来料合格率 95-98% |
| 5 | 来料合格率 90-95% |
| ≤4 | 来料合格率 <90% |

### 交期评分 (0-10)

| 得分 | 条件 |
|------|------|
| 10 | OTD ≥99% |
| 9 | OTD 97-99% |
| 8 | OTD 95-97% |
| 7 | OTD 90-95% |
| 6 | OTD 85-90% |
| 5 | OTD 80-85% |
| ≤4 | OTD <80% |

### 服务评分 (0-10)

| 得分 | 条件 |
|------|------|
| 10 | 响应 <1小时，技术支持完善 |
| 8-9 | 响应 <4小时，服务良好 |
| 6-7 | 响应 <8小时，服务一般 |
| 4-5 | 响应 <24小时 |
| ≤3 | 响应 >24小时 |

## 等级划分

| 等级 | 综合分 | 说明 |
|------|--------|------|
| A | 8-10 | 优秀供应商，战略合作 |
| B | 7-7.9 | 良好供应商，优先采购 |
| C | 6-6.9 | 合格供应商，正常使用 |
| D | <6 | 不合格，限期改善或淘汰 |

## 评估周期建议

| 供应商等级 | 评估频率 |
|------------|----------|
| A | 年度评审 |
| B | 半年度评审 |
| C | 季度评审 |
| D | 月度评审 |

## 使用 supplyflow 脚本

```bash
python3 scripts/supplier_eval.py \
  --suppliers '[{"name":"宏达五金","price_score":8,"quality_score":9,"delivery_score":7,"service_score":6}]' \
  --weights '[0.30,0.35,0.20,0.15]'
```
