Developers and designers use this skill to bootstrap or modify EPC management frontends with a React + Ant Design template and consistent navigation, layout, typography, color, table, metric, and component standards. <br>

### Deployment Geography for Use: <br>

## Known Risks and Mitigations: <br>
Risk: The template depends on npm packages and should be reviewed before installation in a project environment. <br>
Mitigation: Review dependencies, run npm install in a normal project directory, and keep dependency updates under the project's usual security review process. <br>
Risk: Demo frontend code and sample EPC management content may not match production data handling or security needs. <br>
Mitigation: Replace sample data and UI behavior during implementation, and avoid adding real credentials or production data until those changes have been separately reviewed. <br>
