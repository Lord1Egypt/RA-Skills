# 跨境税务义务详解

## 欧盟VAT（增值税）

### 注册触发条件
- 在欧盟境内有库存（FBA/海外仓）→ 必须注册VAT
- 远程销售超过阈值（€10,000/年，全欧盟合并计算）→ 需在消费国注册或使用OSS
- 使用Amazon Pan-European FBA → 每个有库存的国家都需注册

### OSS（一站式申报）
- 2021年7月启用
- 可在一个成员国申报全欧盟B2C远程销售VAT
- 简化合规，但不适用于有实体库存的情况

### 主要国家VAT税率
| 国家 | 标准税率 | 低税率 | 申报周期 |
|------|---------|--------|---------|
| 德国 | 19% | 7% | 月度/季度 |
| 法国 | 20% | 5.5%/10% | 月度 |
| 意大利 | 22% | 4%/5%/10% | 月度/季度 |
| 西班牙 | 21% | 4%/10% | 季度 |
| 荷兰 | 21% | 9% | 季度 |

### 关键红线
- 未注册VAT即销售 → 欠税 + 罚款 + Amazon冻结账户
- VAT号被注销后仍销售 → 刑事风险

## 英国VAT

### 注册阈值
- 远程销售超过£90,000/年
- 在英国有库存

### 税率
- 标准税率：20%
- 低税率：5%

### Brexit影响
- 北爱尔兰：仍适用欧盟VAT规则（针对商品）
- 英国本土：独立VAT体系
- EORI号：需分别申请英国和欧盟EORI

## 美国销售税

### 经济关联（Economic Nexus）
- 2018年Wayfair案后，各州可基于销售额/交易数征税
- 大多数州阈值：$100,000销售额 或 200笔交易/年
- 无联邦VAT，各州独立管理

### Marketplace Facilitator规则
- Amazon、eBay等平台代收代缴销售税
- 卖家通过平台销售 → 平台负责收税
- 独立站卖家 → 需自行判断是否需注册和收税

### 常见误区
- "我没有实体在美国所以不用交税" → 错，经济关联已取代物理关联
- "平台帮我收税了所以不用管" → 部分正确，但需确认平台覆盖范围

## 关税

### 基本规则
- 所有跨境货物进口均需申报关税
- 税率取决于HS编码和原产地

### 常见问题
| 问题 | 说明 |
|------|------|
| 低报货值 | 违法，可能被认定为走私 |
| 错误HS编码 | 可能导致多缴或少缴关税，均需纠正 |
| 原产地规则 | 影响关税税率，需准确申报 |
| 免税额度 | 美国$800/人/天，欧盟€150（但VAT仍需缴纳） |

### 欧盟IOSS
- 进口一站式申报（Import One-Stop Shop）
- 适用于≤€150的B2C远程销售
- 简化VAT申报，但需在销售时即收取VAT

## 东南亚税务

### 新加坡
- GST税率：9%（2024年起）
- 远程服务阈值：S$100,000/年
- 低价值商品：2023年起海外供应商需注册GST

### 泰国
- VAT税率：7%
- 非居民电商：年营收>180万泰铢需注册VAT

### 印尼
- VAT税率：11%（2025年起12%）
- PMSE（电商交易）：海外电商需指定本地代表并缴纳VAT

### 越南
- VAT税率：10%（部分品类5%）
- 海外供应商：年收入>20亿越南盾需注册

## 日本消费税

- 税率：10%（标准）/ 8%（减轻税率）
- 海外卖家：通过日本仓库发货需注册
- 阈值：1,000万日元/年

## 合规建议清单

1. **进入新市场前**：确认VAT/GST注册义务
2. **使用海外仓**：该国几乎一定需要注册VAT
3. **多国销售**：考虑使用OSS/IOSS简化申报
4. **独立站**：自行判断经济关联，必要时注册
5. **保留记录**：至少保留6年交易和税务记录
6. **定期申报**：逾期申报会产生罚款和利息
7. **专业协助**：复杂情况建议聘请当地税务顾问
