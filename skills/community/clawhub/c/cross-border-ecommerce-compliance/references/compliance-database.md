## 美国 (USA)

### 1. 电子产品 (Electronics & Electrical)

| 维度 | 要求 |
|------|------|
| **强制认证** | FCC (Federal Communications Commission) — 强制。FCC ID (无线设备) 或 FCC SDoC (非无线设备)。无线设备还需 FCC Part 15 测试报告 |
| **安全认证** | UL (Underwriters Laboratories) — 非强制但市场准入必备。ETL/CSA 可替代 UL。Amazon 常要求 UL 认证 |
| **能效认证** | ENERGY STAR (自愿) ，加州 CEC 能效强制 (BCS Title 20) |
| **标签要求** | FCC 声明标签 (FCC ID 或合规声明) ；UL/ETL 认证标志；电压/频率标注 (120V/60Hz)；原产国 (Made in China)；加州 Prop 65 警告 (如含铅/邻苯等) |
| **包装要求** | 英文说明书；安全警告 (小零件窒息警告如适用) ；防静电包装 (PCB/IC 类) ；ISTA 运输测试报告 |
| **测试标准** | FCC Part 15 (EMI/RFI) ；UL 60950-1/62368-1 (IT/AV 安全) ；ANSI C63.4 (FCC 测试方法) ；加州 Title 20 (能效) |
| **限制物质** | RoHS (加州 SB-20/SB-50，限制铅/汞/镉/六价铬/PBBs/PBDEs) ；加州 Prop 65 清单物质 (铅/邻苯二甲酸盐/双酚A 等 900+ 种) ；TSCA 管控物质 (如 PFAS 报告要求) |
| **进口文件** | 商业发票；装箱单；提单/空运单；FCC 测试报告/FCC ID；UL 认证证书 (如有) ；原产地证明 (如有优惠关税) |
| **亚马逊额外要求** | 儿童产品证书 (CPC) — 如产品针对 12 岁以下；UL 测试报告上传 Seller Central；合规性图片 (产品+包装多角度) |
| **特殊要求** | 无线设备 (WiFi/BT/蜂窝) 必须有 FCC ID 并标于产品和包装；激光产品需 FDA/CDRH 认证 (21 CFR 1040) ；IT 设备需 DOC (Declaration of Conformity) |

### 2. 玩具 (Toys)

| 维度 | 要求 |
|------|------|
| **强制认证** | ASTM F963 — 强制玩具安全标准。16 CFR Part 1250 (2008 年 CPSIA 修订) |
| **强制测试** | 第三方 CPSC 认可实验室检测 (强制) |
| **合规文件** | CPC (Children's Product Certificate) — 每批/每产品强制，需列出所有适用标准和测试实验室信息 |
| **标签要求** | 追踪标签 (Tracking Label) — 永久标记制造商/生产日期/批次号/产地；小零件窒息警告 (如适用 3 岁以下) ；年龄分级标签；ASTM F963 合规声明；Prop 65 警告 |
| **包装要求** | 英文包装说明；窒息警告 (小零件/气球/弹珠) ；透明包装需可见安全警告；无锐利边角 |
| **测试标准** | ASTM F963-23 (机械/物理/燃烧/化学) ；CPSIA Section 101 (总铅 ≤100ppm) ；CPSIA Section 108 (邻苯二甲酸盐 ≤0.1%: DEHP/DBP/BBP/DINP/DIDP/DNOP) ；16 CFR Part 1303 (表面涂层铅 ≤90ppm)；小零件测试 (16 CFR Part 1501) |
| **限制物质** | 铅 ≤100ppm (基材) / ≤90ppm (涂层)；6 种邻苯二甲酸盐 ≤0.1%；ASTM F963 限制的 8 种重金属 (锑/砷/钡/镉/铬/铅/汞/硒 — 可溶)；BPA (安抚奶嘴/牙胶) |
| **进口文件** | CPC 证书；CPSC 认可实验室测试报告；商业发票/装箱单/提单 |
| **亚马逊要求** | CPC 证书上传；ASTM F963 测试报告；CPSIA 铅/邻苯测试报告；产品+包装图片 (含追踪标签) |
| **特殊要求** | 电动玩具额外需 FCC + UL；艺术材料玩具需 LHAMA 评估 (Labeling of Hazardous Art Materials Act)；磁铁玩具强制标准 16 CFR Part 1262 |

### 3. 化妆品 (Cosmetics & Personal Care)

| 维度 | 要求 |
|------|------|
| **强制法规** | MoCRA (Modernization of Cosmetics Regulation Act, 2022) — FDA 监管 |
| **注册要求** | 设施注册 (Facility Registration) — 所有生产/加工化妆品的企业必须在 FDA 注册，每两年更新一次；产品列表 (Product Listing) — 每种化妆品必须向 FDA 列出，每年更新 |
| **安全要求** | 安全证据 (Safety Substantiation) — 企业须持有充分的安全性测试数据；不良事件报告 — 严重不良事件 15 日内报告 FDA |
| **标签要求** | 净含量 (法定单位+公制)；成分列表 (INCI 名称降序排列)；制造商/经销商名称地址；使用说明/警告语；原产国标识；Talc 含滑石粉产品需注明是否含石棉 (MoCRA) |
| **包装要求** | 防篡改包装 (液体口腔护理/眼部化妆品) ；防漏包装；英文标签 |
| **测试标准** | 微生物测试 (USP <61>/<62>) ；稳定性测试；防腐挑战测试 (USP <51>) ；重金属限量 (FDA 推荐: 铅 ≤10ppm, 汞 ≤1ppm, 砷 ≤3ppm, 镉 ≤3ppm) |
| **限制物质** | 11 种禁用/限制成分清单 (FDA)；色素添加剂须经 FDA 认证批次；防晒剂为 OTC 药品管理 (需 FDA 专论或 NDA) ；加州 Prop 65 物质；PFAS 禁用趋势 (多州立法中) |
| **进口文件** | FDA 设施注册号；产品列表编号；商业发票/装箱单 |
| **亚马逊要求** | FDA 注册证明 (如声称功效/治疗)；成分列表；安全性测试报告 |
| **特殊要求** | 声称"有机"/"天然" — 需 USDA NOP 认证 (有机)；防晒产品按 OTC 药品管理，需 FDA 批准；含 CBD 化妆品目前 FDA 不认可合法 |

### 4. 食品 (Food & Beverages)

| 维度 | 要求 |
|------|------|
| **强制认证** | FDA 注册 (食品设施注册 — 生产/加工/包装/仓储设施) ；FSVP (Foreign Supplier Verification Program — 进口商须验证外国供应商) |
| **食品安全** | FSMA (Food Safety Modernization Act) — 强制预防控制措施；HACCP (危害分析与关键控制点) — 海产品/果汁强制 |
| **标签要求** | Nutrition Facts 标签 (FDA 标准格式: 份量/卡路里/宏量营养素/微量营养素)；成分列表 (降序)；过敏原标示 (8 大过敏原: 牛奶/鸡蛋/鱼/甲壳类/树坚果/花生/小麦/大豆 — 2023 新增芝麻为第 9)；净含量；制造商/经销商信息；原产国 |
| **包装要求** | 食品级包装材料 (FDA 21 CFR 175-178)；防篡改包装；密封完好 |
| **测试标准** | 微生物测试；农药残留 (EPA 限量)；重金属 (FDA 行动水平)；营养成分分析 |
| **限制物质** | FSMA 预防控制；农药残留限量 (EPA tolerance)；食品添加剂须经 FDA 批准或 GRAS 认定；中国出口常见问题: 二氧化硫/亚硝酸盐/色素超标 |
| **进口文件** | FDA 预先通知 (Prior Notice) — 到港前提交；FDA 设施注册号；商业发票/装箱单/提单；原产地证明；有机认证 (如声称有机) |
| **亚马逊要求** | FDA 注册证明；营养标签；成分列表；保质期标注 |
| **特殊要求** | 膳食补充剂: 需符合 21 CFR Part 111 (cGMP)，标签需 Supplement Facts 格式；含新膳食成分 (NDI) 需提前 75 天通知 FDA；酒精饮料: TTB (烟酒税收贸易局) 审批+标签批准 |

### 5. 纺织品/服装 (Textiles & Apparel)

| 维度 | 要求 |
|------|------|
| **强制法规** | CPSIA (儿童产品)；FHSA (易燃织物法案 — Flammable Fabrics Act) 16 CFR Part 1610/1611/1615/1616 |
| **强制测试** | 易燃性测试 (16 CFR Part 1610 服装纺织品) ；儿童睡衣额外严格标准 (16 CFR Parts 1615/1616) |
| **标签要求** | FTC 护理标签 (Care Labeling Rule 16 CFR Part 423: 洗涤/漂白/干燥/熨烫)；纤维成分标签 (Textile Fiber Products Identification Act: 纤维名称+百分比降序，通用名称)；RN (Registered Number) 号码或公司名称；原产国标签 (Made in China)；儿童产品需追踪标签 (CPSIA) |
| **包装要求** | 塑料袋窒息警告 (如开口 ≥5 英寸, 厚度 <1 密耳)；别针/针等金属物安全包装 |
| **测试标准** | 16 CFR Part 1610 (服装易燃性: 1 级正常/2 级中等/3 级快速燃烧-禁止)；16 CFR Part 1615/1616 (儿童睡衣) ；AATCC 测试方法 (色牢度/缩水率等)；ASTM 测试 |
| **限制物质** | CPSIA 铅/邻苯二甲酸盐 (儿童产品)；Prop 65 物质；甲醛 (无强制联邦限制但部分州有限制)；PFAS (防水/防污面料 — 多州立法限制)；偶氮染料 (部分州立法) |
| **进口文件** | 商业发票/装箱单；易燃性测试报告 (如需要)；纤维成分证明；CPC (如为儿童产品) |
| **亚马逊要求** | 儿童产品需 CPC+测试报告；易燃性测试报告推荐上传；合规图片 |
| **特殊要求** | 羽绒/羽毛产品: 需标明成分 (如 "80% Duck Down")，部分州要求灭菌证明；皮草: 需标明动物种类/产地/是否染色 (Fur Products Labeling Act)；有机棉: 需 USDA NOP 或 GOTS 认证 |

### 6. 医疗器械 (Medical Devices)

| 维度 | 要求 |
|------|------|
| **强制认证** | FDA 医疗器械分类 (Class I/II/III): Class I — 510(k) 豁免但需企业注册+产品列名; Class II — 通常需 510(k) 上市前通知; Class III — 需 PMA (上市前批准) |
| **企业注册** | FDA 企业注册 (Establishment Registration) — 每年 10-12 月续期; 医疗器械列名 (Device Listing) |
| **质量体系** | QSR (Quality System Regulation 21 CFR Part 820) — 将过渡至 ISO 13485:2016 协调版 (2026 年生效) |
| **标签要求** | UDI (Unique Device Identification) — 强制唯一器械标识; 预期用途/适应症; 禁忌症/警告/注意事项; 灭菌状态; 一次性使用标识; Rx only 标识 (处方器械); 制造商信息 |
| **包装要求** | 无菌器械: ISO 11607 无菌屏障包装; 运输包装: ISTA 测试; 温度敏感器械: 冷链验证 |
| **测试标准** | IEC 60601 系列 (有源医疗器械电气安全+EMC); ISO 10993 系列 (生物相容性); ISO 11135/11137 (灭菌); IEC 62304 (软件生命周期); IEC 62366 (可用性工程) |
| **限制物质** | 无过敏性材料 (如天然胶乳需标注); 邻苯二甲酸盐 (DEHP 等 — FDA 推荐替代); BPA (婴儿护理器械) |
| **进口文件** | FDA 企业注册号; 器械列名号; 510(k)/PMA 批准函; 商业发票/装箱单; 灭菌证明 (如适用) |
| **亚马逊要求** | FDA 注册+列名证明; 510(k) 或 PMA 批准 (如适用); 只能通过 Amazon 医疗器械邀请制销售 |
| **特殊要求** | 含软件/App 的器械 (SaMD): 需符合 IEC 62304 + 网络安全指南; COVID-19 后 FDA 对远程医疗/可穿戴设备审批加速但审查趋严; Class I 虽免 510(k) 但仍需企业注册+列名; 非处方 (OTC) 器械路径简化 |

### 7. 家居用品 (Home Goods & Furniture)

| 维度 | 要求 |
|------|------|
| **强制法规** | CPSIA (如含儿童部件)；FHSA 16 CFR Part 1500 (危险物质)；16 CFR Part 1632/1633 (床垫易燃性)；加州 TB 117-2013 (软垫家具易燃性) |
| **认证要求** | 床垫: 16 CFR Part 1632 (香烟引燃) + Part 1633 (明火) 强制测试；软垫家具: 加州 TB 117-2013 (阻燃) ；儿童家具: CPSIA + CPC |
| **标签要求** | 软垫家具: TB 117-2013 合规标签；床垫: 易燃性合规标签；护理标签 (纺织品部件)；原产国；Prop 65 警告 (如含甲醛/阻燃剂等) |
| **包装要求** | 防刮擦包装；防潮包装；大件家具需运输安全测试 (ISTA 3A/6A) |
| **测试标准** | 家具稳定性 (ASTM F2057 — 防倾倒，2023 强制) ；16 CFR Part 1632/1633 (床垫) ；TB 117-2013 (软垫家具阻燃)；甲醛释放 (CARB 第 2 阶段 / EPA TSCA Title VI — 复合木制品) ；表面涂层铅 (16 CFR Part 1303) |
| **限制物质** | 甲醛 (复合木板: CARB Phase 2 ≤0.05/0.09ppm)；有机磷酸酯阻燃剂 (加州/多州限制)；PFAS (防污处理 — 多州立法)；Prop 65 物质 |
| **进口文件** | 商业发票/装箱单/提单；易燃性测试报告 (床垫/软垫家具) ；甲醛排放证明 (复合木制品)；CPC (儿童家具) |
| **亚马逊要求** | 床垫: 易燃性测试报告 (Part 1632+1633)；家具稳定性测试认证 (ASTM F2057)；儿童家具: CPC+测试报告 |
| **特殊要求** | 2023 年 ASTM F2057 防倾倒强制标准适用于 30 英寸以上储物家具；蜡烛/香薰产品需 ASTM F2417 和 FHSA 标签警告；厨具/餐具: FDA 食品接触材料 21 CFR 175-178 |

### 8. 儿童用品 (Children's Products — 不含玩具)

| 维度 | 要求 |
|------|------|
| **强制认证** | CPSIA (Consumer Product Safety Improvement Act) — 全面覆盖 12 岁以下儿童产品 |
| **合规文件** | CPC (Children's Product Certificate) — 每产品强制；第三方 CPSC 认可实验室测试 |
| **适用品类** | 婴儿推车/婴儿床/高椅/婴儿背带/安抚奶嘴/儿童餐具/儿童家具/婴儿车/学步车等 — 各自有独立 ASTM 标准 |
| **标签要求** | 追踪标签 (Tracking Label) — 制造商/日期/批次/产地；年龄分级；窒息/勒颈警告；注册卡 (婴儿耐用品 — 16 CFR Part 1130) |
| **包装要求** | 塑料袋窒息警告；小型配件密封包装；无锋利边角 |
| **测试标准** | 各品类特定 ASTM 标准 (如: 婴儿床 ASTM F1169, 推车 ASTM F833, 高椅 ASTM F404, 背带 ASTM F2907) ；CPSIA 铅 ≤100ppm / 涂层铅 ≤90ppm；CPSIA 6 种邻苯二甲酸盐 ≤0.1%；小零件测试 |
| **限制物质** | 铅 ≤100ppm；邻苯二甲酸盐 ≤0.1%；BPA/BPS (婴儿奶瓶/水杯已禁)；阻燃剂 (有机磷/卤系 — 多州/联邦限制)；PFAS |
| **进口文件** | CPC 证书；CPSC 认可实验室测试报告；商业发票/装箱单；产品注册卡样本 (婴儿耐用品) |
| **亚马逊要求** | CPC + 所有引用标准的测试报告；产品+包装图片；追踪标签图片；对婴儿耐用品注册卡要求严格 |
| **特殊要求** | 婴儿耐用品 (推车/床/高椅/游戏床等) 需额外提供产品注册卡 (Consumer Registration Card) ；摇篮/睡篮产品需符合最新 ASTM F2194 (倾斜角度/窒息风险) ；婴儿背带/包裹式产品受 2023 年新联邦安全标准管制 |

## 欧盟 (EU)

### 1. 电子产品 (Electronics & Electrical)

| 维度 | 要求 |
|------|------|
| **强制认证** | CE 标志 — 强制。需符合: RoHS 指令 2011/65/EU (有害物质限制)；EMC 指令 2014/30/EU (电磁兼容)；LVD 指令 2014/35/EU (低电压 50-1000V AC / 75-1500V DC)；无线设备指令 RED 2014/53/EU (含无线功能产品) |
| **能效认证** | ErP 指令 2009/125/EC + 能效标签法规 (EU) 2017/1369 |
| **WEEE** | WEEE 指令 2012/19/EU — 电子废弃物回收。需在目标国注册 WEEE，贴 WEEE 垃圾桶标志 |
| **标签要求** | CE 标志 (最小 5mm)；WEEE 垃圾桶带叉标志；RoHS 合规声明；电压/频率/功率标注 (220-240V/50Hz)；制造商/进口商名称地址 (EU 境内)；产品型号/批次号；使用语言: 目标国官方语言 |
| **包装要求** | 包装指令 94/62/EC — 最小化/可回收；多语言说明书 (至少目标国语言)；安全警告图标 |
| **测试标准** | EN 55032/55035 (EMC) ；EN 62368-1 (IT/AV 安全) ；EN IEC 63000 (RoHS 技术文档) ；EN 300 328/EN 301 893 (无线 RED) |
| **限制物质** | RoHS (10 种: Pb/Hg/Cd/Cr6+/PBBs/PBDEs/DEHP/BBP/DBP/DIBP)；REACH 法规 SVHC (240+ 种高关注物质 — 含量>0.1%需申报)；POPs 法规 (持久性有机污染物)；PFAS 限制提案 (2023) |
| **进口文件** | EU DoC (Declaration of Conformity)；技术文件 (Technical Documentation) ；CE 测试报告；WEEE 注册号；REACH SVHC 合规声明 |
| **亚马逊要求** | CE DoC 上传；RoHS 测试报告；WEEE 注册 (DE/FR 等国强制) ；合规标签图片 |

## 英国 (UK)

### 概要说明

英国脱欧后建立了独立合规体系 — UKCA (UK Conformity Assessed)。CE 标志在 2024 年底前仍接受 (过渡期多次延长)，但 2025 年起 UKCA 强制。建议新产品直接申请 UKCA。

**核心差异 vs EU:**
- UKCA 代替 CE (英国市场)
- UK REACH 代替 EU REACH (独立化学品管理)
- UK 指定标准 (Designated Standards) 代替 EU 协调标准
- UK Approved Body 代替 EU Notified Body
- UK 责任人 (UK Responsible Person) 代替 EU 授权代表

| 品类 | 关键 UK 差异 |
|------|-------------|
| **电子产品** | UKCA + UK RoHS；UK WEEE 注册 (在英国)；需 UK DoC + UK 责任人地址 |
| **玩具** | UKCA + Toys (Safety) Regulations 2011 (与 EU TSD 基本一致)；UK 指定标准 (实质同 EN 71) |
| **化妆品** | SCPN (Submit Cosmetic Product Notification) 代替 CPNP；UK 责任人；UK Cosmetics Regulation (Schedule 34) — 实质同 EU 1223/2009 |
| **食品** | UK Food Safety Authority (FSA/FSS)；UK 营养标签格式略有不同；UK REACH |
| **纺织品** | UK REACH；UK 纤维成分法规 (同 EU)；UKCA (PPE 仅) |
| **医疗器械** | UKCA + UK MDR 2002 (基于旧 EU MDD + MDR 部分条款)；UK Approved Body；MHRA 注册；UK 责任人 |
| **家居用品** | UKCA (如适用)；UK REACH；UK 家具易燃性法规 1988 (比 EU 更严 — 强制) |
| **儿童用品** | UKCA (如适用)；UK REACH；UK 标准同 EN 系列但为 UK 指定标准 |
| **北爱尔兰** | 特殊地位: 继续适用部分 EU 规则 (NI Protocol / Windsor Framework)，需 UKNI 标志或 CE |

> 详细 UK 每条目内容与 EU 高度相似，仅替换认证机构/法规名称。查询时按 EU 数据输出并标注 UK 差异点。

| **测试标准** | ST 2016 (机械/物理/燃烧) ；ST 2016 第 3 部分 (化学 — 8 种重金属溶出+邻苯+甲醛) ；食品卫生法第 4 条 (婴幼儿接触玩具) |

| **亚马逊要求** | ST 标志认证号；ST 测试报告 |
| **特殊要求** | 杂货玩具 (如食玩/扭蛋): 食品卫生法额外要求；电动玩具: 需额外 PSE 认证 (电安法) |

## 加拿大 (Canada)

### 概要说明

加拿大合规体系与**美国相似但独立**，主要差异:

- CSA/UL/ETL 认证 (电气安全 — 均接受但 CSA 更本土化)
- ISED (Innovation, Science and Economic Development) 无线认证 (代替 US FCC)
- CCPSA (Canada Consumer Product Safety Act) — 消费品安全法
- 双语标签 (英/法) — 加拿大独有要求
- 魁北克省额外 Charter of the French Language 要求

| 品类 | 加拿大 vs 美国关键差异 |
|------|----------------------|
| **电子产品** | 电气安全: CSA/UL/ETL/cUL (NRTL 认可 — 同美国)；无线: ISED 认证 (代替 FCC)；能效: NRCan 强制 (代替 ENERGY STAR) ；标签: 双语英文+法文 |
| **玩具** | CCPSA + Toys Regulations (SOR/2011-17 — 等效 ASTM F963)；加拿大卫生部监管 (代替 CPSC) ；双语警告 |
| **化妆品** | Food and Drugs Act + Cosmetic Regulations；健康加拿大 (Health Canada) 监管 (代替 FDA) ；化妆品通知表 (Cosmetic Notification Form — 上市后 10 天内) ；双语标签 |
| **食品** | Safe Food for Canadians Regulations (SFCR) ；CFIA (加拿大食品检验署) 监管；Nutrition Facts 双语格式 (英语/法语各一面) |
| **纺织品** | Textile Labelling Act + Regulations；双语纤维成分标签；护理标签 (CAN/CGSB-86.1) |
| **医疗器械** | Health Canada Medical Devices Regulations；MDL (Medical Device Licence) ；MDSAP (Medical Device Single Audit Program — 加拿大为倡议国) |
| **家居用品** | CCPSA 覆盖；特定产品有加拿大强制标准 (如婴儿床/蜡烛等) ；复合木制品: 加拿大甲醛排放标准 (CANFER) — 等同 CARB ATCM |
| **儿童用品** | CCPSA + 各品类法规 (婴儿床/推车/高椅/游戏床等) ；严格追踪/记录要求 (制造商/进口商保有记录 6 年) |

> 加拿大合规查询时默认按美国数据输出 + 标注上述加拿大差异点。
