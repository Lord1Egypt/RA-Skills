## 一、Vimshottari Dasha层级结构

```
Mahadasha（大运）        → 年度/十年级
  └─ Antardasha（小运）   → 月度/年级
      └─ Pratyantar（微运）→ 周/月级
          └─ Sookshma（细运）→ 日/周级
              └─ Prana（息运）→ 日级
```

### 各层级的时间精度

| 层级 | 时间跨度 | 精度 | 实战用途 |
|------|---------|------|---------|
| Mahadasha | 1-20年 | 年级 | 人生阶段 |
| Antardasha | 数月-3年 | 月级 | 年度趋势 |
| Pratyantar | 数日-数月 | 周/月级 | **具体事件时机** |
| Sookshma | 数日 | 日级 | 精确日期 |
| Prana | 数小时 | 时级 | 极精确定时 |

**⚠️ 核心原则**：Pratyantar是**实战预测的关键层级**——它将Antardasha的趋势缩小到具体的几周/几个月内。

## 二、Pratyantar计算方法

### 2.1 基本原理

**每个Antardasha内部的Pratyantar顺序**与Mahadasha顺序完全相同：
```
Ketu → 金星 → 太阳 → 月亮 → 火星 → Rahu → 木星 → 土星 → 水星
```

### 2.2 Pratyantar时长计算

**公式**：
```
Pratyantar时长 = (该行星在120年周期中的年数 / 120年) × Antardasha总时长（月）
```

### 2.3 各行星的120年周期占比

| 行星 | 120年周期年数 | 占比 | Nakshatra |
|------|-------------|------|-----------|
| Ketu | 7年 | 5.83% | Ashwini, Magha, Moola |
| 金星 | 20年 | 16.67% | Bharani, Purva Phalguni, Purva Ashadha |
| 太阳 | 6年 | 5.00% | Krittika, Uttara Phalguni, Uttara Ashadha |
| 月亮 | 10年 | 8.33% | Rohini, Hasta, Shravana |
| 火星 | 7年 | 5.83% | Mrigashira, Chitra, Dhanishta |
| Rahu | 18年 | 15.00% | Ardra, Swati, Shatabhisha |
| 木星 | 16年 | 13.33% | Punarvasu, Vishakha, Purva Bhadrapada |
| 土星 | 19年 | 15.83% | Pushya, Anuradha, Uttara Bhadrapada |
| 水星 | 17年 | 14.17% | Ashlesha, Jyeshtha, Revati |

### 2.4 标准化计算流程

**输入**：
1. 当前Antardasha的主星和时长
2. Antardasha开始日期

**步骤**：

1. **确定Antardasha总月数**
   ```
   Antardasha月数 = Antardasha年数 × 12
   ```

2. **按标准顺序排列Pratyantar**
   - 从Antardasha主星开始
   - 按Ketu→金星→太阳→月亮→火星→Rahu→木星→土星→水星循环
   - 注意：Antardasha主星的Pratyantar排第一个

3. **计算每个Pratyantar的月数**
   ```
   Pratyantar月数 = (行星年数 / 120) × Antardasha总月数
   ```

4. **从Antardasha开始日期累加**

## 四、Pratyantar解读核心原则

### 4.1 Pratyantar行星的征象激活

Pratyantar行星会激活以下维度：

| 维度 | 说明 |
|------|------|
| **行星本身征象** | 太阳=权威/父亲/自我，月亮=母亲/情绪/公众，等 |
| **行星作为宫主星** | 太阳=10宫主→事业，月亮=12宫主→灵性/海外，等 |
| **行星所在宫位** | 太阳在9宫→高等教育/父亲/导师 |
| **行星的Nakshatra** | 太阳的Nakshatra主宰星也会被激活 |

### 4.2 Pratyantar与过境的叠加规则

**核心公式**：
```
事件发生 = Mahadasha定领域 × Antardasha定趋势 × Pratyantar定时机 × Transit定触发
```

| 叠加层级 | 作用 | 权重 |
|---------|------|------|
| Mahadasha | 人生大主题 | 40% |
| Antardasha | 年度趋势 | 30% |
| Pratyantar | 月度/周度时机 | 20% |
| Transit | 精确触发日 | 10% |

### 4.3 Pratyantar快速判断法

**三步判断**：

1. **Pratyantar行星是什么？** → 确定激活的征象
2. **它在星盘中的位置/状态如何？** → 确定事件质量
3. **当前Transit是否配合？** → 确定是否触发

**口诀**：
> "Pratyantar定是什么，位置定好不好，Transit定什么时候"

## 五、进阶：Sookshma和Prana

### 5.1 Sookshma（细运）

**计算方法**：与Pratyantar完全相同，只是输入变为Pratyantar的总月数。

**示例**：月亮Pratyantar（1.17月）

| # | Sookshma行星 | 占比 | 天数 | 大约日期 |
|---|-------------|------|------|---------|
| 1 | 月亮 | 8.33% | 3天 | 04.28-05.01 |
| 2 | 火星 | 5.83% | 2天 | 05.01-05.03 |
| 3 | Rahu | 15.00% | 5天 | 05.03-05.08 |
| 4 | 木星 | 13.33% | 5天 | 05.08-05.13 |
| 5 | 土星 | 15.83% | 5天 | 05.13-05.18 |
| 6 | 水星 | 14.17% | 5天 | 05.18-05.23 |
| 7 | Ketu | 5.83% | 2天 | 05.23-05.25 |
| 8 | 金星 | 16.67% | 6天 | 05.25-05.31 |
| 9 | 太阳 | 5.00% | 2天 | 05.31-06.02 |

### 5.2 Prana（息运）

- 最精细的层级，可精确到**日/小时**
- 计算方法相同，输入变为Sookshma的总天数
- 实战中很少使用，因为出生时间精度往往不够

### 5.3 实用建议

| 预测需求 | 使用层级 | 说明 |
|---------|---------|------|
| 年度规划 | Mahadasha + Antardasha | 足够 |
| 月度预测 | + Pratyantar | 推荐 |
| 周度预测 | + Sookshma | 可选 |
| 日度预测 | + Prana | 仅精确出生时间 |

## 七、注意事项

1. **出生时间精度限制**：Pratyantar精度取决于出生时间精度。出生时间误差5分钟→Pratyantar误差约1-2天
2. **逆行行星的影响**：如果Pratyantar行星在逆行，其征象可能延迟或反复
3. **Rahu/Ketu的Pratyantar**：影子行星的Pratyantar特别关注突然事件和灵性体验
4. **与Transit的协同**：Pratyantar不单独预测，必须结合Transit才能确定事件是否真的触发
