## 摄影风格

### 1. 35mm Film Photography

**特征识别词**:
```
35mm film, film grain, color shift, high contrast, authentic film color grading, 
analog aesthetic, vintage film look, CineStill, Kodak Portra, Fuji Superia
```

**适用场景**: 街拍、人文纪实、复古风格人像、日常记录

**质量控制要点**:
```
- film grain visible
- natural color shift/warm tones
- slight overexposed highlights
- authentic analog imperfections
- no digital over-sharpening
```

---

### 2. Cinematic Photography

**特征识别词**:
```
cinematic, film still, anamorphic lens, shallow depth of field, cinematic color grading,
letterbox bars, movie frame aesthetic, dramatic lighting, narrative composition
```

**适用场景**: 电影感画面、故事性内容、情绪人像、游戏过场

**质量控制要点**:
```
- anamorphic lens flare
- 2.35:1 or 16:9 aspect ratio
- rich shadows with rim lighting
- film grain texture
- dramatic mood
```

---

### 3. Editorial Fashion

**特征识别词**:
```
fashion editorial, magazine cover, high fashion, clean studio lighting, 
sharp focus, professional retouching, model pose, luxury brand aesthetic
```

**适用场景**: 时尚摄影、品牌宣传、杂志封面、产品代言

**质量控制要点**:
```
- ultra-clean lighting
- sharp focus on subject
- magazine-quality composition
- natural skin texture
- no plastic skin
```

---

### 4. Documentary / Street Photography

**特征识别词**:
```
documentary, street photography, candid, unposed, authentic moment, 
environmental portrait, street scene, urban life, travel documentary
```

**适用场景**: 纪实摄影、旅行记录、城市故事、社会纪实

**质量控制要点**:
```
- natural unposed moment
- authentic environmental context
- available light preferred
- candid atmosphere
- story-driven composition
```

---

### 5. Golden Hour / Warm Tones

**特征识别词**:
```
golden hour, warm sunlight, golden light, sunset, sunrise, amber tones,
warm color grading, backlit, sun flare, dreamy atmosphere
```

**适用场景**: 逆光人像、风景、温馨氛围、浪漫时刻

**质量控制要点**:
```
- warm color temperature
- soft backlight with rim glow
- lens flare from sun
- golden specular highlights
- dreamy soft mood
```

---

### 6. Moody / Dark Aesthetic

**特征识别词**:
```
moody, dark aesthetic, low-key lighting, high contrast, dramatic shadows,
noir, deep shadows, cinematic atmosphere, mysterious, brooding
```

**适用场景**: 电影感人像、悬疑氛围、个性表达、艺术摄影

**质量控制要点**:
```
- deep blacks with preserved detail
- dramatic single light source
- cool or warm shadow tones
- volumetric atmosphere
- tension-building composition
```

## 插画/艺术风格

### 1. Watercolor Illustration

**特征识别词**:
```
watercolor, watercolour, soft washes, visible brushstrokes, paper texture,
wet-on-wet technique, soft edges, pastel colors, artistic hand-painted quality
```

**适用场景**: 绘本、贺卡、装饰画、文艺礼品、包装设计

**质量控制要点**:
```
- visible brushstroke texture
- watercolor bleeding edges
- paper grain texture
- soft color transitions
- artistic imperfect beauty
```

---

### 2. Oil Painting

**特征识别词**:
```
oil painting, classical painting, impasto texture, rich saturated colors,
Renaissance, Baroque, realistic brushwork, canvas texture, museum quality
```

**适用场景**: 艺术作品、古典风格、收藏级、人物肖像

**质量控制要点**:
```
- thick impasto brushwork
- rich color layers
- canvas texture visible
- classical composition
- museum-quality rendering
```

---

### 3. Hand-Drawn / Sketch

**特征识别词**:
```
hand-drawn, pencil sketch, loose strokes, artistic imperfections,
line art, ink drawing, doodle, scribble, rough sketch aesthetic
```

**适用场景**: 草图、线稿、设计概念、手绘感内容、涂鸦风格

**质量控制要点**:
```
- visible pencil/ink texture
- loose unfinished strokes
- paper background
- artistic imperfections
- authentic hand-drawn feel
```

---

### 4. Vector Illustration

**特征识别词**:
```
vector illustration, flat design, clean lines, scalable graphics,
minimalist illustration, geometric shapes, spot colors, corporate art style
```

**适用场景**: 图标设计、平面海报、UI配图、品牌视觉

**质量控制要点**:
```
- flat solid colors
- no gradients (or subtle gradients)
- clean geometric forms
- scalable crisp edges
- corporate/tech aesthetic
```

---

### 5. Gouache Painting

**特征识别词**:
```
gouache, matte finish, bold opaque colors, visible brush marks,
pigment concentration, poster paint, illustrative style
```

**适用场景**: 海报、插画、手绘质感、平面设计

**质量控制要点**:
```
- matte no-shine surface
- bold opaque pigments
- visible brush application
- slightly textured strokes
- flat graphic quality
```

---

### 6. Colored Pencil

**特征识别词**:
```
colored pencil, soft blending, layered colors, paper texture visible,
细腻渐变, detailed drawing, realistic texture, waxy finish
```

**适用场景**: 细腻插画、写实风格、儿童绘本、纹理感内容

**质量控制要点**:
```
- visible paper tooth texture
- soft color layering
- waxy crayon-like finish
- detailed rendering
- warm artistic mood
```

## 3D/特殊效果风格

### 1. 3D Render / CGI

**特征识别词**:
```
3D render, CGI, 3D illustration, digital art, ray-traced,
PBR materials, physically based rendering, 3D CGI art
```

**适用场景**: 产品展示、游戏资产、建筑可视化、概念设计

**质量控制要点**:
```
- realistic PBR materials
- soft ray-traced shadows
- clean 3D aesthetic
- proper depth of field
- studio lighting setup
```

---

### 2. Isometric 3D

**特征识别词**:
```
isometric, 2.5D, isometric illustration, isometric view,
consistent 30-degree angle, pixel-perfect, flat 3D
```

**适用场景**: 游戏UI、信息图、建筑、游戏场景、图标设计

**质量控制要点**:
```
- consistent isometric angle
- clean geometric shapes
- flat shading with subtle highlights
- limited color palette
- no perspective distortion
```

---

### 3. Low Poly

**特征识别词**:
```
low poly, geometric, faceted surfaces, triangulated mesh,
modern geometric art, digital sculpture
```

**适用场景**: 游戏美术、抽象艺术、图标、设计元素

**质量控制要点**:
```
- visible polygon facets
- angular geometric forms
- limited color palette
- modern digital aesthetic
- clean edges
```

---

### 4. Pixel Art

**特征识别词**:
```
pixel art, 8-bit, 16-bit, retro game, sprite, pixelated,
limited resolution, nostalgic gaming, NES/SNES style
```

**适用场景**: 游戏资产、像素画、复古内容、头像设计

**质量控制要点**:
```
- visible pixel grid
- limited color per sprite
- retro gaming aesthetic
- dithering patterns
- nostalgic 8-bit/16-bit style
```

---

### 5. Double Exposure

**特征识别词**:
```
double exposure, photo manipulation, silhouette, blending,
overlay effect, creative photography, artistic merge
```

**适用场景**: 创意摄影、艺术海报、头像、视觉创意

**质量控制要点**:
```
- seamless blending between layers
- clear silhouette definition
- complementary subject/background
- artistic high-end look
- sophisticated composition
```

## 风格组合指南

### 常见风格组合

| 目标效果 | 推荐组合 |
|---------|---------|
| 现代中国风 | 国潮 + 摄影 或 国潮 + 插画 |
| 赛博朋克人像 | 摄影 + 霓虹 + 故障艺术 |
| 复古杂志风 | 摄影(35mm) + 排版 + 暖色调 |
| 梦幻插画 | 水彩 + 柔光 + 莫兰迪色 |
| 硬核科幻 | 3D渲染 + 赛博朋克 + 暗色调 |
| 清新治愈 | 摄影 + 柔焦 + 低饱和色 |

### 风格切换关键词

```
切换到写实摄影: "photorealistic, authentic, documentary style"
切换到插画: "illustration, hand-drawn, artistic rendering"
切换到动漫: "anime style, cel-shaded, manga inspired"
切换到3D: "3D render, CGI, digital art, ray-traced"
切换到极简: "minimalist, clean, negative space, simple"
切换到复古: "vintage, retro, aged, nostalgic"
```
