# Childcare & Daycare Marketing Kit

**Compliance-aware marketing for licensed childcare centers, daycares, and preschools.**

Generate enrollment campaigns, parent communication, seasonal marketing, and digital ads with COPPA, state licensing, NAEYC, FTC, and CACFP compliance built in.

## Why This Skill Exists

Generic AI gets childcare marketing wrong in ways that create real legal exposure:

- **COPPA violations**: Running ads on child-directed platforms when you should be targeting parents
- **Licensing violations**: "Fully licensed" without displaying the required state license number
- **NAEYC false claims**: Referencing accreditation after it expired or was never obtained
- **FTC violations**: "My child started reading at age 2!" testimonials without typical-results context
- **CACFP mislabeling**: Advertising "nutritious meals included" without USDA program enrollment
- **Ratio overclaiming**: Advertising staff ratios better than licensed capacity

This skill generates every piece of copy with compliance flags and correct claim structures baked in.

## Compliance Coverage

- **COPPA**: Parent-only targeting; no child-directed ad placements
- **State Licensing**: License number in all ads; ratios within licensed capacity
- **NAEYC**: Accreditation language only when enrollment is current and confirmed
- **FTC 2023**: Testimonials with typical-results context; no incentivized reviews
- **FERPA-adjacent**: No individual child info in group communications
- **USDA CACFP**: Meal program claims only when enrolled
- **TCPA / CAN-SPAM**: Opt-out in all SMS/email templates
