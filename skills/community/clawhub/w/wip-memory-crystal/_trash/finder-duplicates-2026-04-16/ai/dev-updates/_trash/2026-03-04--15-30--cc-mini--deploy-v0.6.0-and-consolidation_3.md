## Key Decisions

1. **CE set to `enabled: false`, not removed from config.** OpenClaw auto-discovers plugins from extensions/. Removing the config entry wasn't enough ... had to explicitly disable.
2. **cc-mini backfill covers all agents.** The 801 files include both CC and OC transcripts. Running oc-lesa-mini second correctly skipped all duplicates.
3. **migrate-embeddings error reporting is buggy.** The CLI reports failures when the migration actually succeeds. The transaction commits fine; the error is in post-migration cleanup (trying to backup after DB close). Needs a code fix but not blocking.
