# 飞书集成配置指南 (feishu_integration.md)

## 概述

本文件定义 Deal Genie 与飞书的集成方式。用户提供飞书 webhook 后，选品分析、推品提醒、追单节点、复盘报告均可自动推送到飞书群/文档。不接飞书时所有功能不受影响。

---

## 快速配置（用户只需一步）

### 获取 webhook 地址

1. 打开目标飞书群
2. 点击右上角「设置」→「群机器人」
3. 点击「添加机器人」→「自定义机器人」
4. 设置机器人名称（如"带货精灵"），复制 webhook 地址
5. 粘贴给 AI："接入飞书，webhook 地址是 https://open.feishu.cn/open-apis/bot/v2/hook/xxxxx"

AI 收到后回复"飞书已接入，后续分析结果将自动推送到该群"。

**安全提示**：webhook 地址仅用于发送消息，无法读取群聊内容。更换 webhook = 更换推送目标。

## 消息模板设计原则

1. **信息分层**：标题（结论）+ 核心数据（3-5行）+ 行动建议（1行），不超过一屏
2. **视觉标识**：✅🟢🟡🔴 一目了然
3. **可操作**：每条推送含下一步动作，不是纯信息展示
4. **非侵入**：追单提醒每天最多 1 条，不在深夜时段推送

---

## 故障处理

| 问题 | 处理 |
|------|------|
| webhook 失效 | 提示用户重新获取（群解散/机器人被删除会失效） |
| 推送失败但不影响主流程 | 提示"飞书推送失败，分析结果不受影响，可手动查看 HTML 报告" |
| 用户更换飞书群 | 提供新 webhook 即可，旧地址自动失效 |

---

## 隐私说明

- webhook 地址仅用于发送消息
- 不存储、不转发、不读取群聊历史
- 用户随时可以说"断开飞书"来停止推送
- 未接飞书时，skill 所有功能不受影响
