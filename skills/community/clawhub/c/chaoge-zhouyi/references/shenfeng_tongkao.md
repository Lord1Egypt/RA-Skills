## 独到见解

### 一、用神可变论

**张楠曰**：
> 用神非不变，随岁运而变化。命局用神，岁运用神，不可一概而论。

**示例**：
```
命局：甲木日主，生于申月，用神丁火（伤官）
大运：壬水运，丁火受克，用神变为癸水（正印）
```

### 二、真假用神论

**张楠曰**：
> 用神有真假。真用神，得地得势得气；假用神，不得地不得势不得气。

**示例**：
```
命局：甲木日主，生于申月，用神丁火
丁火透干，但地支无根 → 假用神
丁火透干，且地支有根 → 真用神
```

### 三、组合流通论

**张楠曰**：
> 命局之吉凶，全在组合流通。组合流通，忌神也为福；组合阻塞，用神也为祸。

**示例**：
```
命局：甲木日主，生于申月，用神丁火
丁火透干，壬水透干，丁壬合木 → 组合流通，大贵
丁火透干，癸水透干，丁癸相战 → 组合阻塞，贫贱
```

### 四、寒暖调候论

**张楠曰**：
> 调候为急。寒暖失调，虽用神得地，亦不吉。

**示例**：
```
命局：甲木日主，生于亥月（寒）
虽用神丙火得地，但局中无丙火调候 → 不吉
局中有丙火调候 → 大贵
```
