# 慢性萎缩性胃炎分型与癌变风险评估

## 触发条件
当满足以下所有条件时启用本技能：
- 胃镜下见胃黏膜变薄、呈灰绿色、血管透见
- 病理检查确认胃腺体数量减少或消失
- 排除急性胃炎或非萎缩性胃炎

## 执行步骤

### 1. 确认基本病理特征
- 验证是否存在以下组织学改变：
  - 胃黏膜变薄，腺体减少或消失
  - 固有层大量淋巴细胞、浆细胞浸润（可伴淋巴滤泡）
  - 纤维组织增生
  - 腺上皮化生（主要为肠上皮化生，偶见假幽门腺化生）

### 2. 进行A/B型分型
根据病因、部位、实验室指标区分：

**A型（自身免疫性）：**
- 部位：胃体/底部
- 抗体：抗壁细胞抗体、抗内因子抗体阳性（>90%）
- 胃酸：明显降低
- 血清胃泌素：升高
- 维生素B12：降低（常伴恶性贫血）
- 不伴消化性溃疡

**B型（幽门螺杆菌相关，我国常见）：**
- 部位：胃窦部
- 抗体：阴性
- 胃酸：中度降低或正常
- 血清胃泌素：降低
- 维生素B12：正常
- 常伴消化性溃疡

### 3. 评估肠上皮化生类型及癌变风险
- **完全型（Ⅰ型/小肠型）：**
  - 含杯状细胞、吸收细胞、潘氏细胞
  - PAS染色刷状缘阳性
  - MUC2表达，胃黏蛋白（MUC1/5AC/6）减少
  - 癌变风险较低
- **不完全型（Ⅱ型）：**
  - Ⅱa型（胃型）：分泌中性黏液
  - Ⅱb型（结肠型）：分泌硫酸黏液；同时表达胃黏蛋白与MUC2
  - **Ⅱb型与肠型胃癌发生关系密切，需高度警惕**

### 4. 制定临床处理建议
- 对所有慢性萎缩性胃炎患者，尤其是伴有**Ⅱb型肠上皮化生**者：
  - 定期进行内镜随访（推荐间隔1–3年，依风险调整）
  - 监测异型增生及早期胃癌征象
  - 若发现异型增生，按癌前病变管理路径进一步处理

> **注意**：癌变路径通常为“萎缩 → 化生 → 异型增生 → 癌变”，异型增生是关键中间步骤，需重点监测。
