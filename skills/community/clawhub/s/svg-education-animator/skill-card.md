## Description: <br>
Creates K-12 friendly flat SVG animations as standalone HTML pages that can be previewed in a browser. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[shinelee2014](https://clawhub.ai/user/shinelee2014) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Educators, learning-content creators, and developers use this skill to turn K-12 topics into friendly animated SVG scenes embedded in single-file HTML for lesson previews or teaching aids. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated browser-previewable HTML could include scripts or external resources if later prompts request them. <br>
Mitigation: Review generated HTML, SVG, and CSS before opening or sharing it, especially when prompts ask for scripts or external resources. <br>
