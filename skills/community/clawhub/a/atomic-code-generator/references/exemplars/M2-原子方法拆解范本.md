## M2-02 方法定义

### 输入示例
方法识别结果

### 处理过程
1. 定义每个方法的功能
2. 明确方法的输入输出
3. 验证定义的准确性

### 输出示例
**方法定义**：
1. validateUsername(username: string): boolean
   - 功能：验证用户名是否符合格式要求
   - 输入：用户名字符串
   - 输出：是否有效

2. validatePassword(password: string): boolean
   - 功能：验证密码是否符合强度要求
   - 输入：密码字符串
   - 输出：是否有效

3. authenticateUser(username: string, password: string): User|null
   - 功能：验证用户身份
   - 输入：用户名、密码
   - 输出：用户对象或null
