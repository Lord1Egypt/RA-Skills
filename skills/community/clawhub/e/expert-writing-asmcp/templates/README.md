# 模板分类索引

本文档定义了 `expert-writing-asmcp` 技能支持的写作模板分类结构。

---

## 📁 目录结构

```
templates/
├── README.md                    # 本文件 - 分类索引
├── 文档类/                      # 项目立项类文档
│   ├── 01_可行性研究报告.md
│   ├── 02_项目建议书.md
│   └── 03_商业计划书.md
├── 报告类/                      # 总结汇报类文档
│   ├── 04_项目总结报告.md
│   ├── 05_年度工作报告.md
│   └── 06_市场调研报告.md
└── 方案类/                      # 技术/商务方案
    ├── 07_技术方案.md
    ├── 08_产品介绍文档.md
    └── 09_解决方案.md
```

## 🔑 模板匹配关键词

| 模板 | 关键词 | 触发场景 |
|-----|-------|---------|
| 可行性研究报告 | 可研,可行性研究,立项分析 | 项目立项、技术改造 |
| 项目建议书 | 项目建议书,立项申请,项目申请 | 立项申请、审批汇报 |
| 商业计划书 | 商业计划,BP,融资计划,商业提案 | 创业融资、商务合作 |
| 项目总结报告 | 总结,复盘,项目复盘,项目回顾 | 项目复盘、经验沉淀 |
| 年度工作报告 | 年度总结,年度报告,年终总结 | 部门总结、年度汇报 |
| 市场调研报告 | 调研,市场分析,行业研究,调查报告 | 市场分析、行业研究 |
| 技术方案 | 技术方案,实施方案,技术设计 | 实施方案、技术评审 |
| 产品介绍文档 | 产品介绍,产品手册,产品宣传 | 产品宣传、方案展示 |
| 解决方案 | 解决方案,投标,客户方案,标书 | 投标文件、客户方案 |

---

## 📝 模板使用方式

技能在生成大纲时，会根据用户指定的模板名称或描述的场景，自动读取对应模板文件获取章节结构。

用户可通过以下方式指定模板：
- 直接说模板名称：`"用商业计划书写"`
- 描述场景：`"写个融资用的计划书"` → 自动匹配商业计划书
- 不指定 → 默认使用【可行性研究报告】
