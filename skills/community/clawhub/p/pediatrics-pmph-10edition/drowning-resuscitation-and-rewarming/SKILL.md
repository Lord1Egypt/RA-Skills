# 溺水救治与复温策略

## 触发条件
若患者被救离水中 **且** 出现以下任一情况：
- 无自主呼吸
- 神志不清

则立即启动溺水ABC复苏流程。

## 执行步骤

### 1. 现场抢救
- **确保施救者安全**：避免二次溺水。
- **水中仅行通气**（限专业人员），**禁止胸外按压**。
- **上岸后立即处理**：
  - 清理口鼻泥沙、杂物、呕吐物，开放气道。
  - 若患者无反应、无呼吸 → 先给予 **2–5次有效人工呼吸**。
  - 若仍无脉搏 → 立即开始胸外按压，按儿童CPR比例（通常30:2或15:2）持续至自主循环恢复。
- **严禁控水**（如倒立、腹部挤压）：因吸入肺内水量少且迅速吸收，控水易致误吸和感染。
- **颈椎固定**：仅在有明确颈椎损伤机制（如跳水撞击）时使用；一般不常规固定（颈髓损伤发生率<0.5%）。
- **所有接受复苏者**，即使清醒，**必须送医评估**。

### 2. 院内救治
- **轻症患者**：急诊留观6–8小时；若神志清楚、无需吸氧、生命体征稳定，可离院。
- **重症治疗重点**：
  - **通气氧合**：无呼吸者立即气管插管，给予高浓度氧使SpO₂≥94%；采用肺保护通气策略（潮气量6–8 ml/kg，PEEP起始5 cmH₂O）；24小时内不轻易撤机。
  - **抗感染**：吸入污水者需监测体温、白细胞计数、胸片；必要时使用抗生素；可疑异物可行纤维支气管镜检查。
  - **循环支持**：建立静脉通路；低血压者先补液；持续低血压使用血管活性药物；心衰者可选用多巴酚丁胺或米力农。
  - **脑复苏**：维持通气、氧合与灌注；控制血糖；避免增加脑代谢；颅内高压时使用甘露醇或高张盐水；抽搐发作时用地西泮或苯巴比妥。

### 3. 复温策略
- **抢救期**（低温为不良预后标志）：
  - 脱去湿衣物，用干毯包裹。
  - 使用加温输液、红外线加热、温化呼吸气体。
  - 严重低温（<25°C）：考虑温热液体灌洗胃/膀胱/腹腔，或体外循环复温。
- **复苏成功后**（诱导性低体温具脑保护作用）：
  - 可维持体温32–34°C，持续24–48小时。

### 4. 预后判断
- 溺水时间 <10分钟：预后良好。
- 溺水时间 >25分钟：预后极差。
- 最初24–72小时的神经系统评估是预测长期预后的最佳指标。
