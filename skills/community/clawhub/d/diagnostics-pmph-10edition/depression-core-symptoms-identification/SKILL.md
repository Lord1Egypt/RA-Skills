# 抑郁核心症状识别框架

## 触发条件
当患者主诉情绪问题或行为改变，且已排除器质性疾病所致情绪障碍时，若出现以下任一情况，立即应用本框架：
- 持续两周以上的情绪低落
- 明显的兴趣或快感丧失
- 日常功能（工作、学习、社交）显著受损

> **注意**：本框架不适用于短暂情绪波动、正常悲伤反应（如丧亲初期）或非病理性情绪变化。

## 执行步骤

### 1. 确认核心症状（必须存在至少一项）
- **情绪低落**：深切悲伤、愁眉苦脸，自称“高兴不起来”“活着没意思”，有度日如年或生不如死感。
- **兴趣缺乏**：对既往喜爱活动明显失去兴趣（如不愿读书、逛街、社交）。
- **快感缺失**：即使参与日常活动也无法获得愉悦感，心不在焉。

### 2. 评估认知与思维症状
- **思维迟缓**：联想变慢、反应迟钝、自感脑子变笨、言语减少、语速慢、语声低。

### 3. 观察运动行为症状
- **运动性迟滞**：动作缓慢、活动减少；严重者呈木僵（不言不动不食）或亚木僵。
- **或激越表现**：烦躁不安、紧张、难以控制，偶有攻击行为。

### 4. 评估自杀风险与自我评价
- **自责自罪**：对轻微过失感到严重过错，认为自己罪孽深重。
- **自杀相关表现**：从死亡念头到具体计划、行动，甚至扩大性自杀（先杀害亲人再自杀）。

### 5. 识别躯体症状
- **睡眠障碍**：入睡困难、睡眠浅、早醒（比平时早2–3小时且无法再入睡）或嗜睡。
- **食欲体重变化**：多数食欲减退、体重下降；少数暴食、体重增加。
- **其他躯体不适**：性欲减退、便秘、无器质性依据的头痛/腹痛/胸痛等。
- **疲惫乏力**、自主神经功能紊乱。

### 6. 判断是否伴精神病性症状
- **幻觉**（如听到嘲弄声）或**妄想**（如罪恶妄想、被害妄想）。

### 7. 考虑特殊人群表现
- **儿童/青少年**：兴趣减退、活动减少、学习成绩下降。
- **老年人**：常伴焦虑、易激惹、多种躯体不适，病程易慢性化。
- **女性**：高发于月经前、产后、围绝经期。

## 输出判断
若满足：
- 至少1项核心症状 **且**
- 其他症状总数达到临床显著程度（通常≥4项，持续≥2周）

则应高度怀疑抑郁障碍，**立即评估自杀风险**，并考虑转诊精神科或启动干预措施。
