# Alias Tool - Command Aliases

Manage shell command aliases. Aliases create shortcuts for commonly used commands.

## Quick Start

```bash
alias ll='ls -la'
```

## Features

- Create new aliases
- List defined aliases
- Remove aliases
- Persistent alias config

## Examples

```bash
alias ll='ls -la'
alias -p
```
