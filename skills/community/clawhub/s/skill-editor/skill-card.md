## Description: <br>
Guides agents through editing, improving, and reviewing AgentSkills, including SKILL.md metadata, package.json consistency, environment variable declarations, and release checks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[romicboy](https://clawhub.ai/user/romicboy) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agents use this skill when creating or updating OpenClaw AgentSkills, especially to check frontmatter, metadata, package.json, environment variable declarations, triggers, and version consistency. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Edited skills can introduce misleading trigger descriptions, inconsistent versions, incorrect package metadata, or newly declared environment variables. <br>
Mitigation: Review diffs carefully before relying on edited skills, especially trigger descriptions, version fields, package.json, and environment variable declarations. <br>
