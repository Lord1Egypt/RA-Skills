# 错误状态码

## 状态码分类

| 范围 | 分类 | 说明 |
|------|------|------|
| 0 | 成功 | 命令顺利完成 |
| 1x | 基础/参数错误 | 工具初始化或参数解析问题 |
| 2x | 身份/凭证错误 | 账号登录或配额相关问题 |
| 3x | 重定向错误 | 站点强制重定向 |
| 4x | 用户输入受限 | 终端条件不足或内容不可用 |
| 5x | 服务端/网络异常 | 网络问题或服务器错误 |
| 50 | 未知错误 | 非 KmdrError 的预期外异常 |

## 恢复策略

| 状态码 | 恢复策略 |
|--------|----------|
| 1x | 检查命令和参数；尝试 `kmdr config --clear` 重置配置 |
| 2x | 检查凭证正确性；使用 `kmdr pool use <username>` 切换有配额账号；等待配额重置（每月） |
| 3x | 按错误消息提示更新 base_url：`kmdr config --set base_url=<new_url>` |
| 4x | 检查输入参数有效性；尝试配置代理：`kmdr config --set proxy=<proxy_url>`；尝试不同搜索关键字 |
| 5x | 检查网络连接；配置或更换代理；稍后重试；使用 `--retry` 增加重试次数 |
| 50 | 重试一次；若仍失败，报告用户错误消息（从 `msg` 字段获取）并建议提交 issue |

> 错误详情见输出的 `msg` 字段。以上策略覆盖所有子码，无需逐个记忆。
