# MST — 材料选择与跟踪（CMMI V3.0 Domain-specific PA）

## PA 基本信息
- **缩写**：MST
- **英文**：Material Selection and Tracking
- **中文**：材料选择与跟踪
- **所属**：Domain-specific PA（领域特定实践域）
- **适用领域**：硬件工程、制造、供应链管理

---

## CMMI V3.0 要求要点

### 核心目的
确保材料选择满足产品需求，并跟踪材料使用情况。

### 关键实践（Specific Practices）
| 实践编号 | 实践内容 | 说明 |
|---------|---------|------|
| MST 1.1 | 识别材料需求 | 基于产品需求识别材料特性要求 |
| MST 1.2 | 评估材料选项 | 评估候选材料的适用性 |
| MST 1.3 | 选择材料 | 基于评估结果选择材料 |
| MST 1.4 | 建立材料规范 | 定义材料的技术规范 |
| MST 2.1 | 跟踪材料使用 | 跟踪材料在项目中的使用情况 |
| MST 2.2 | 管理材料变更 | 控制材料变更的影响 |
| MST 3.1 | 监控材料性能 | 监控材料在实际使用中的性能 |

### 通用实践（Generic Practices）
- GP 1.1 执行特定实践
- GP 2.1 建立组织政策
- GP 2.2 策划过程
- GP 2.3 提供资源
- GP 2.4 分配责任
- GP 2.5 培训人员
- GP 2.6 管理配置
- GP 2.7 识别并记录干系人需求和接口
- GP 2.8 监督和控制过程
- GP 2.9 客观评估符合性
- GP 2.10 高层审查
