# A股客户会前准备助手（Meeting Preparer CN）

基于 Anthropic `meeting-preparer` 原版架构本土化改造，适配 A 股投资者关系与客户覆盖场景。

## Overview

本技能为基金经理、研究员、投资经理等专业投资者提供**一站式会前准备**服务，
自动整合目标公司的公开信息，生成结构化的《客户会前简报》供会议使用。

**输出规格：**
- 长度：5-8页
- 形式：Word（.docx）或 Markdown
- 风格：国内券商调研简报格式
- 数据来源：westock-data 工具（个股数据）+ 公开研报 + 公告

**适用场景：**

| 场景 | 说明 |
|------|------|
| 上市公司调研 | 基金经理/研究员实地拜访前准备 |
| 投资者交流会 | 参加公司举办的投资者开放日/说明会 |
| 路演/反向路演 | 参加公司路演前背景整理 |
| 电话会议 | 业绩说明会、专题电话会前准备 |
| 一对一小范围交流 | 与董秘/财务总监/CEO 的闭门交流 |

## 核心能力模块

### 1. 公司背景摘要（Company Profile）
- 公司概况：股票代码、行业分类、主营业务、收入结构（按产品/地区拆分）
- 管理层简介：董事长、总经理、董秘、财务总监背景
- 股权结构：实际控制人、前十大股东、机构持仓变动（近3个季度）
- 财务摘要：近2年营收/净利润/毛利率/ROE 趋势
- 一致预期：券商一致预期（营收/净利润/目标价）

### 2. 近期动态追踪（Recent Developments）
- 最近3个月的重要公告（定增、并购、股权激励、分红等）
- 最近2个季度业绩变化趋势
- 研报覆盖动态（最近调高/调低评级的机构及理由）
- 新闻与舆情跟踪（行业政策、媒体报道、ESG 争议等）
- 管理层变动/重要人事调整

### 3. 行业竞争格局（Competitive Landscape）
- 行业所处阶段（导入期/成长期/成熟期/衰退期）
- 主要竞争对手对比（市值、营收规模、毛利率、增速）
- 公司相对优劣势分析
- 行业近期大事记（政策、技术突破、新进入者）

### 4. 机构持仓与资金面（Institutional Holdings）
- 近3个季度基金重仓变化（增持/减持/新进/退出）
- 北向资金持仓变化趋势
- 融资融券余额变化
- 大宗交易与重要股东增减持

### 5. 会议议程与提问建议（Discussion Points）
- 建议重点关注话题（根据近期动态智能生成）
- 建议回避话题（如敏感期、合规红线）
- 可推荐的产品/服务议题
- 开放式问题清单（5-10个）

## 工作流程

```
用户输入 → 搜索公开信息 → 整理结构化数据 → 生成简报 → 可供打印/分享
   │           │                │              │
公司名称    westock-data    按模块清洗    输出Word/MD
股票代码    网上公开信息    交叉验证    附数据来源
会议类型    公开研报        剔除噪音    同步到腾讯文档
```

### 详细步骤

**Step 1: 信息采集**
1. 使用 `westock-data` 工具查询个股数据：
   - 公司概况、主营业务
   - 近8个季度财务数据（营收、净利润、毛利率、ROE）
   - 机构持仓数据（基金、北向）
   - 近期公告列表（最近3个月）
   - 融资融券数据
   - 券商一致预期
2. 使用网络搜索补充：
   - 近期研报（评级变化）
   - 行业新闻与政策
   - 管理层动态
   - 舆情监测

**Step 2: 数据整理与交叉验证**
1. 将原始数据按六大模块分类整理
2. 交叉验证关键数据（如：不同来源的营收数据一致性）
3. 剔除噪音信息（短期波动、非实质公告）
4. 标注所有数据来源

**Step 3: 生成会前简报**
1. 按标准输出结构编排内容
2. 制作关键对比表格（竞争对手、机构持仓、财务趋势）
3. 智能生成会议议程与提问建议
4. 标注高风险提示（如：业绩不及预期、大股东减持、监管问询等）

**Step 4: 输出交付**
1. 默认输出 Markdown 格式（方便复制到微信/钉钉/邮件）
2. 可选项：生成 Word (.docx) 格式供打印
3. 可选项：同步到腾讯文档供团队共享

## 质量控制清单

输出前必须逐项确认：

- [ ] 公司名称、股票代码准确无误
- [ ] 财务数据标注期间（如：2026Q1）
- [ ] 所有数据标注来源
- [ ] 竞争对手数据可比性合理（同行业、相似规模）
- [ ] 会议议题与近期动态有逻辑关联
- [ ] 回避话题不踩合规红线
- [ ] 不存在捏造/推测性数据
- [ ] 语言专业、简洁、客观
- [ ] 高风险事项使用 🔴 红色标记

---

## 风险标记机制

简报中需对以下事项进行特殊标注：

| 等级 | 标记 | 事项示例 |
|------|------|---------|
| 🔴 高危 | 红色加粗 | 业绩大幅不及预期、被立案调查、大股东减持超5%、ST风险 |
| 🟡 关注 | 黄色加粗 | 行业政策变化、管理层变动、商誉减值风险、诉讼未决 |
| 🟢 一般 | 绿色标注 | 行业景气度波动、正常季节性变化、技术路线争议 |

## 注意事项

1. **合规优先**：简报仅供参考，不构成投资建议；不涉及内幕信息
2. **时效性**：所有数据标注时间，优先使用最新财报数据
3. **数据溯源**：每项关键数据必须标注来源，严禁捏造
4. **客观中立**：以事实陈述为主，避免主观价值判断
5. **安全性**：不记录、不存储用户具体的客户信息和会议安排
