# 泄殖腔分隔形成直肠与尿生殖窦

## 何时使用
- 分析后肠末段与泄殖腔发育（第4–8周）
- 判断直肠、肛管、尿道等结构的胚胎来源
- 评估先天性畸形（如直肠尿道瘘、肛门闭锁）的成因

## 执行步骤
1. **确认初始结构**：泄殖腔为后肠末段膨大部，腹侧连尿囊，尾端由泄殖腔膜封闭。
2. **尿直肠隔形成（第4–7周）**：尿囊与后肠间充质增生，形成横向镰状隔，向尾端生长。
3. **融合与分隔**：尿直肠隔与泄殖腔膜融合，将泄殖腔分为：
   - **腹侧** → 尿生殖窦（参与泌尿生殖管道形成）
   - **背侧** → 原始直肠（分化为直肠与肛管上段）
4. **泄殖腔膜同步分裂**：
   - 腹侧 → 尿生殖膜
   - 背侧 → 肛膜
5. **肛管开通（第8周末）**：肛膜破裂，肛凹（外胚层来源）与肛管上段（内胚层来源）相通。
6. **识别齿状线**：内胚层与外胚层交界处，为临床重要分界（如痔疮分类依据）。

> 注意：若尿直肠隔发育不全，可导致直肠尿道瘘等畸形，需结合影像或临床检查进一步评估。
