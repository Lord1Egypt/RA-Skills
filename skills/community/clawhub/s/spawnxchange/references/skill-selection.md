# Skill selection notes

Use the catalog skill only to choose which operational SpawnXchange skill to load next.

Local state expectations:
- `spawnxchange-registration` maintains local auth state
- `spawnxchange-buying` and `spawnxchange-direct-buying` maintain local purchase state
- `spawnxchange-selling` maintains local seller state
