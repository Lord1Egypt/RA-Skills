# 高频主题追问库

用于在 2-3 轮问答中快速建立用户画像。每次只选最相关的 3 个问题。

## 城市迁移：北京 / 上海 / 深圳 / 回老家

优先问题：

1. 当年目标城市给你的核心机会是什么：平台、行业、人脉、收入、身份，还是信息流？
2. 留在原城市给你的核心收益是什么：家庭、现金流、熟悉关系、健康，还是生活节奏？
3. 你今天最想验证的是职业跃迁、城市身份、收入差距，还是生活状态？

事件提示：

- 移动互联网周期
- 短视频和内容平台
- 疫情和远程办公
- AI 工具浪潮
- 就业市场结构调整

## 留学 / 回国 / 再出国

优先问题：

1. 当时留下来的现实门槛是什么：签证、工作、钱、孤独、身份，还是父母距离？
2. 回国给你的核心收益是什么：行业机会、家人、关系网络、生活确定性，还是身份转换？
3. 今天怀念的是某个国家，还是当年重新开始的感觉？

事件提示：

- 全球金融危机后就业
- Brexit 和签证环境
- 疫情出入境影响
- 全球通胀和利率周期
- AI 和远程工作

## 体制 / 国企 / 市场化

优先问题：

1. 当年稳定路径给你的核心收益是什么：父母认可、低波动、城市稳定，还是长期确定性？
2. 市场化路径当年吸引你的是什么：收入、成长、自由、行业热度，还是离开熟人社会？
3. 现在复盘它，是因为行业波动、年龄压力、家庭期待，还是安全感下降？

事件提示：

- 平台经济扩张和监管
- 互联网裁员周期
- 就业市场结构调整
- 人口和家庭照护压力

## 买房 / 没买房

优先问题：

1. 当年是哪一年、哪个城市，月供或首付压力大概怎样？
2. 房子对婚恋、父母期待、留城和现金流的影响各有多大？
3. 当前复盘重点是资产变化、现金流压力、居住稳定，还是城市自由度？

事件提示：

- 2008 后地产和城市化
- 2015-2016 房价快速上行
- 2020 后地产融资收紧
- 2021-2024 局部房价快速回落

边界：

- 只做现金流和路径依赖复盘。
- 不给买卖建议。

## 离开上一家公司

优先问题：

1. 当时离开的直接原因是什么：成长、钱、领导、行业、健康，还是看不到未来？
2. 留下来的收益和代价分别是什么？
3. 现在怀念的是旧公司，还是旧环境提供的安全系统？

事件提示：

- 行业周期变化
- 平台成熟
- 裁员周期
- AI 技能重估

## AI / 内容 / 个人品牌转型

优先问题：

1. 你第一次意识到 AI / 内容 / 个人品牌机会，大概是哪一年？
2. 当时没有行动，主要卡在时间、技能、信心、工作压力，还是看不清回报？
3. 今天你已有的可迁移资产是什么：行业经验、表达、作品、项目合作、技术，还是渠道？

事件提示：

- 公众号和图文内容
- 短视频和直播
- 内容平台分化
- 生成式 AI
- Agent 工作流

## 亲密关系 / 婚育

优先问题：

1. 当年选择和城市、职业、家庭距离是否有关？
2. 你现在想复盘的是关系本身、人生阶段，还是当时的自我状态？
3. 这个问题是否涉及安全、控制、威胁或严重心理压力？

边界：

- 不做心理诊断。
- 不替用户做关系决定。
- 有安全风险时优先现实支持。
