# 卵巢浆液性癌的分级诊断标准

## 适用条件
- 肿瘤已确认为卵巢浆液性上皮性来源（镜下见纤毛状立方/柱状上皮，类似输卵管上皮）
- 已获取组织病理学切片和免疫组化检测结果
- **不适用于**黏液性或其他非浆液性卵巢肿瘤

## 执行步骤

1. **确认肿瘤类型**：确保病变为卵巢浆液性上皮性肿瘤。

2. **评估组织学与分子特征**，区分低级别与高级别浆液性癌：
   - **低级别浆液性癌特征**：
     - 起源：由良性或交界性浆液性肿瘤逐步进展而来
     - 组织结构：小巢状、腺状、乳头状或微乳头状
     - 细胞特征：瘤细胞大小一致，核异型性轻至中度，核大小差异 < 3倍
     - 核分裂象：≤2个/mm²，无异常核分裂象
     - 坏死：罕见
     - 分子标志：常见KRAS或BRAF突变
     - 预后：发展较慢，预后较好
   
   - **高级别浆液性癌特征**：
     - 起源：多起源于输卵管末端上皮
     - 组织结构：实性、乳头状、迷路状、腺状或筛网状
     - 细胞特征：瘤细胞显著异型，核大且异型明显，核大小差异 > 3倍
     - 核分裂象：>5个/mm²，伴异常核分裂象
     - 坏死：常见
     - 特征性细胞：可见奇异瘤巨细胞
     - 分子标志：95%以上存在TP53突变，常伴NF1、RB、BRCA1/2突变
     - 预后：发展迅速，预后较差

3. **结合辅助判断依据**：
   - 肉眼观：若肿瘤体积大、大面积坏死、卵巢外广泛累及，高度怀疑高级别
   - 双侧性：低级别和高级别通常均为双侧性，不能单独用于分级

4. **输出诊断结论**：明确判定为“低级别浆液性癌”或“高级别浆液性癌”，用于指导后续治疗方案选择和预后判断。
