## Description: <br>
Agentic AI lifecycle: create, improve, and persist workflows for converting functionality into Claude Code automations such as agents, skills, rules, hooks, commands, plugins, documentation updates, and memory entries. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[drumrobot](https://clawhub.ai/user/drumrobot) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and engineers use this skill to identify automation candidates, choose an appropriate Claude Code automation type, create or extend the automation, improve automation behavior from session feedback, and persist reusable knowledge. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated automation, hooks, global Claude Code changes, and persistent memory updates can alter future agent behavior or save inappropriate information. <br>
Mitigation: Review generated automations before accepting them, prefer project scope for project-specific details, and do not save secrets, credentials, private endpoints, or personal data into shared skills, rules, plugins, or memory. <br>
