# 成纤维细胞迁移增殖与细胞外基质积聚调控

## 适用条件
- 组织损伤已发生
- 肉芽组织形成且富含新生血管
- 炎症反应已启动并进入修复阶段
- 成纤维细胞已开始增殖

## 执行流程

### 第一阶段：成纤维细胞迁移与增殖
1. **血管通透性增高**：VEGF 增加血管通透性，促使血浆蛋白（如纤维蛋白原、纤维粘连蛋白）在细胞外基质中沉积，为成纤维细胞提供临时支架。
2. **生长因子释放**：血小板、巨噬细胞、肥大细胞等释放 TGF-β、PDGF、FGF、EGF 及促纤维化细胞因子（IL-1、TNF-α）。
3. **趋化与增殖**：上述因子引导成纤维细胞向损伤部位迁移并刺激其增殖。
4. **正反馈调节**：巨噬细胞上调 TGF-β、PDGF 和 FGF 表达，进一步放大成纤维细胞活性。
5. **TGF-β 核心作用**：促进单核细胞趋化、血管生成、成纤维细胞迁移增殖、胶原/纤维粘连蛋白合成，并抑制金属蛋白酶对 ECM 的降解。

### 第二阶段：细胞外基质积聚与瘢痕成熟
1. **胶原合成启动**：创伤后第3～5天，成纤维细胞开始合成胶原，持续数周。
2. **合成促进**：PDGF、FGF、TGF-β、IL-1、IL-4 等协同刺激胶原及其他 ECM 成分合成。
3. **降解抑制**：TGF-β 和类固醇抑制胶原酶活性，减少胶原降解。
4. **MMP/TIMP 动态平衡**：
   - 基质金属蛋白酶（MMPs）以酶原形式分泌，活化后降解 ECM；
   - 特异性抑制物 TIMP 快速结合活化 MMP，防止过度降解。
5. **瘢痕成熟**：肉芽组织转化为致密胶原、弹性纤维构成的苍白瘢痕，血管退化，胶原纤维沿皮肤张力方向平行排列。

## 注意事项
- 不适用于无血管或无炎症反应的组织环境
- 若感染未控制，ECM 积聚过程将受阻
- 急性炎症早期不启动此机制

## 输出结果
成功实现成纤维细胞迁移增殖，并形成结构稳定、具备抗拉强度的成熟瘢痕。
