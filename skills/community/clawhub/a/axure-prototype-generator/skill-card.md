## Known Risks and Mitigations: <br>
Risk: Generated JavaScript prototypes can execute in Axure preview contexts. <br>
Mitigation: Review generated code before running it, especially for unexpected external links, network requests, credential handling, or unrelated behavior. <br>
Risk: License codes may represent paid access information. <br>
Mitigation: Handle license codes as sensitive access data and avoid sharing them in prompts, public files, or generated prototypes. <br>
Risk: Promotional download, rating, or testimonial claims may not indicate security or quality. <br>
Mitigation: Use the server-resolved security verdict and direct code review rather than promotional claims when deciding whether to install or run the skill. <br>
