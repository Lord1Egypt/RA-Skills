# 小细胞低色素性贫血的实验室鉴别诊断

## 何时使用
- 患者血常规显示MCV < 80 fL且MCH < 27 pg
- 需要明确小细胞低色素性贫血的具体病因
- 实验室已提供铁代谢相关指标或可获取这些数据

## 执行步骤

1. **确认小细胞低色素性贫血诊断**：核实MCV和MCH是否符合标准。
2. **收集关键实验室指标**：包括血清铁、血清铁蛋白、总铁结合力（TIBC）、转铁蛋白饱和度（TSAT）、红细胞分布宽度（RDW）、血红蛋白电泳（如怀疑地中海贫血）。
3. **应用鉴别诊断矩阵**：根据指标组合判断最可能的病因类型。
4. **结合临床背景验证**：考虑患者年龄、性别、饮食史、慢性疾病史、家族史等因素。
5. **建议进一步检查**：如骨髓铁染色、基因检测等，若初步结果不明确。

> 注意：本技能聚焦于实验室指标驱动的鉴别流程，不替代临床综合判断。若存在混合病因（如缺铁合并地中海贫血），需谨慎解读结果。

参见 `references/diagnostic-matrix.md` 获取完整的指标阈值与判读规则。
