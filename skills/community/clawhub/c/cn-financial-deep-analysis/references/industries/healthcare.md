# 医疗业

## 核心财务特征

- 研发驱动，研发管线决定未来
- 销售费用率高（15-40%，学术推广）
- 毛利率极高（创新药 70-95%，仿制药 40-70%）
- 应收账款账期长（医院回款慢）
- 受集采等政策影响极大

## 关键分析指标

| 指标 | 健康值 | 警戒值 | 说明 |
|------|--------|--------|------|
| 研发费用率 | >10% | <5% | 创新驱动型核心投入 |
| 销售费用率 | <30% | >40% | 过高依赖带金销售 |
| 毛利率 | >60% | <40% | 仿制药集采后下降明显 |
| 应收账款周转天数 | <120天 | >200天 | 医院回款周期 |
| 研发资本化率 | <30% | >50% | 与科技业同理 |
| 商誉/净资产 | <20% | >40% | 并购标的减值风险 |

## 风险评估重点

- 集采政策冲击：核心品种被纳入集采，毛利率骤降
- 研发失败风险：关键管线临床数据不达预期
- 应收账款坏账：医院拖欠+医保控费
- 销售合规风险：商业贿赂调查
- 减值风险：并购标的业绩承诺未完成

## 舞弊高发区

- 销售费用虚增（咨询费、会议费套现）
- 研发费用资本化操纵
- 收入确认提前（发货即确认而非医院验收）
- 应收账款坏账不计提
- 存货（药品）效期减值不处理
