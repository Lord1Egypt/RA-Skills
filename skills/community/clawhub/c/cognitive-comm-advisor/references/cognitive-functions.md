# Cognitive Functions Reference

This is the core theoretical foundation of the cognitive-comm-advisor skill. The 8 Jungian
cognitive functions describe HOW people process information and make decisions — they are
the mechanism layer beneath observable personality. Four-letter type codes (INTP, ENFJ, etc.)
are used as convenient shorthand indexes into function stacks, not as the framework itself.

## The 8 Functions

| Code | Name | Core Drive |
|------|------|-----------|
| Te | Extraverted Thinking | Organize external world efficiently; measurable results |
| Ti | Introverted Thinking | Build internal logical frameworks; precision & consistency |
| Fe | Extraverted Feeling | Harmonize group dynamics; read social cues |
| Fi | Introverted Feeling | Authenticate values; personal meaning & integrity |
| Se | Extraverted Sensing | Engage concrete reality; act on present data |
| Si | Introverted Sensing | Catalog past experience; reliability & detail |
| Ne | Extraverted Intuition | Generate possibilities; divergent connections |
| Ni | Introverted Intuition | Converge on singular vision; long-range pattern |

## Communication Implications by Dominant Function

### Te-dom (ENTJ, ESTJ)
- Want: bottom-line up front, data, action items, brevity
- Dislike: rambling, vague feelings, lack of agenda
- Respect trigger: competence, efficiency, directness

### Ti-dom (INTP, ISTP)
- Want: logical coherence, autonomy, depth over breadth
- Dislike: forced small talk, emotional pressure, surface arguments
- Respect trigger: intellectual honesty, precision

### Fe-dom (ENFJ, ESFJ)
- Want: warmth, acknowledgment of people impact, consensus
- Dislike: coldness, dismissing feelings, lone-wolf moves
- Respect trigger: showing you care about the group

### Fi-dom (INFP, ISFP)
- Want: authenticity, space to process, values alignment
- Dislike: fakeness, pressure to conform, ignoring personal meaning
- Respect trigger: respecting individuality

### Ne-dom (ENTP, ENFP)
- Want: possibilities, brainstorm energy, big picture first
- Dislike: rigid rules without rationale, shutting down ideas early
- Respect trigger: engaging with their ideas before narrowing

### Ni-dom (INTJ, INFJ)
- Want: future vision, depth, strategic framing
- Dislike: excessive detail without context, constant pivots
- Respect trigger: showing you see the big picture

### Se-dom (ESTP, ESFP)
- Want: action, concrete examples, keep it real
- Dislike: over-theorizing, endless planning without doing
- Respect trigger: being present, responsive, practical

### Si-dom (ISTJ, ISFJ)
- Want: precedent, clear process, reliability
- Dislike: constant change without justification, ignoring what worked
- Respect trigger: honoring commitments, showing preparation

## Interaction Dynamics: Function Conflicts & Synergies

### Natural Synergies
- Te ↔ Ti: Both value logic; Te appreciates Ti's depth, Ti appreciates Te's execution
- Fe ↔ Fi: Both value people; can bridge if mutual respect exists
- Ne ↔ Ni: Both intuitive; Ne brings breadth, Ni brings focus
- Se ↔ Si: Both sensing; Se brings adaptability, Si brings stability

### Common Friction Points
- Te vs Fi: "Just do it" vs "But does it feel right?"
- Fe vs Ti: "How does everyone feel?" vs "But is it logically sound?"
- Ne vs Si: "What if we try something new?" vs "What worked before?"
- Se vs Ni: "Act now" vs "Wait, I need to see the whole picture"

### Inferior Function Stress Behaviors
When stressed, people regress to their inferior function in unhealthy ways:
- Te-inf (INFP/ISFP): sudden harsh criticism, obsessive list-making
- Ti-inf (ENFJ/ESFJ): paranoid logic, finding conspiracies
- Fe-inf (INTP/ISTP): emotional outbursts, over-sensitivity to rejection
- Fi-inf (ENTJ/ESTJ): feeling unappreciated, self-doubt spirals
- Ne-inf (ISTJ/ISFJ): catastrophizing, seeing only worst-case
- Ni-inf (ESTP/ESFP): paranoia about future, tunnel vision
- Se-inf (INTJ/INFJ): overindulgence, sensory overwhelm
- Si-inf (ENTP/ENFP): neglecting health/routine, forgetting commitments

## Dynamic Analysis Heuristic

When analyzing a pair, follow these steps:

1. **Dom-to-Dom relationship**: Compare both dominant functions. Same axis (both T or both F,
   both N or both S) = shared language but potential competition. Opposite axis = complementary
   but requires translation effort.

2. **Find the Aux bridge**: Check if either person's auxiliary matches the other's dominant or
   tertiary. This is the natural bridge point — the function they can both access comfortably.
   Lead communication advice through this bridge.

3. **Check inferior triggers**: Identify both parties' inferior functions. These are landmine
   zones — accidental activation causes disproportionate defensive reactions. Map specific
   workplace behaviors that might trigger each person's inferior.
