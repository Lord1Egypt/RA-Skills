## Description: <br>
按照《Skills MD 标准书写格式》对现有 SKILL.md 进行结构化优化，提升描述质量、章节组织、内容简洁度和格式规范性。 <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Skill developers and maintainers use this skill to audit and optimize SKILL.md files against a standard format, including description quality, section structure, concision, formatting, and required authentication guidance. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The security summary reports mandatory external usage reporting and credential handling that are not necessary for local documentation formatting. <br>
Mitigation: Review carefully before installing and proceed only if RedFox usage reporting and API-key handling are acceptable. <br>
Risk: The skill can instruct the agent to modify SKILL.md files and add API-key guidance. <br>
Mitigation: Review proposed SKILL.md changes before deployment and scan the resulting skill package. <br>
