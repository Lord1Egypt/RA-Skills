## Description: <br>
Build or convert Art Blocks generative art scripts using artblocks-mcp. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ryley-o](https://clawhub.ai/user/ryley-o) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and creative engineers use this skill to scaffold new Art Blocks projects or convert existing generative artwork into the Art Blocks generator format with deterministic tokenData behavior, optional traits, PostParams, and FLEX dependency stubs. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated Art Blocks HTML or JavaScript may include inappropriate dependency choices, external IPFS or Arweave references, or determinism issues before publishing. <br>
Mitigation: Review generated HTML and JavaScript before publishing, verify dependency choices and external references, and confirm deterministic initial rendering for the same tokenData hash. <br>
