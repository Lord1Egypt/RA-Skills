# 妊娠晚期胎动监测与缺氧预警

## 适用条件
- 妊娠≥28周
- 孕妇能自觉胎动
- 无急性产科并发症

## 执行流程

1. **指导孕妇在安静状态下计数胎动**，推荐固定活跃时段（如下午或夜间）
2. **记录2小时内胎动总次数**
3. **判定标准**：
   - ≥6次/2小时：视为正常，继续常规产检
   - ＜6次/2小时：提示可能存在胎儿宫内缺氧，需进一步评估
4. **注意事项**：
   - 胎儿睡眠周期可致20～40分钟无胎动，属生理现象
   - 若首次计数异常，应重复监测或结合电子胎心监护、生物物理评分综合判断
   - 此方法不能替代专业医疗评估

## 输出
若胎动＜6次/2小时，发出“胎儿缺氧风险”警报，建议立即就医进行电子胎心监护或生物物理评分。
