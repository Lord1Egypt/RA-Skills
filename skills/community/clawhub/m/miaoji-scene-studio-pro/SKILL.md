# 技能名称：miaoji-scene-studio-pro
# 显示名：秒技·场景工作室Pro版

---

## 定位升维

**敌人**：只拍"能用的图"、不追求"比同类产品更吸引眼球"的将就思维

**升维打击**：从"拍什么场景"升级到"如何让视觉成为竞争优势"

免费版告诉你"应该加一张生活场景图"，Pro版告诉你"TOP10同类产品的视觉弱点在哪，你如何通过A/B测试反超"。

---

## 核心定位

场景工作室Pro版是免费版的战略升级版，不仅提供场景建议，更输出完整的视觉战役方案。通过A/B测试方案设计、平台适配规格指南、季节性场景库和同类产品视觉分析，帮助卖家建立视觉护城河。

**与免费版的本质区别**：
- 免费版：单次场景建议 → 告诉你拍什么
- Pro版：视觉战役方案 → 告诉你怎么测、怎么比、什么时候拍什么

## Pro专属功能

### 1. A/B测试方案生成

同一产品生成3套不同风格构图方案：

```
【视觉A/B测试方案】
━━━━━━━━━━━━━━━━━━━━

📸 方案A：功能展示型（强调实用性）
  构图风格：产品+使用手势+功效对比
  主打人群：理性消费者，关心参数和功能
  适用平台：Amazon主图、A+功能区
  预期CTR提升：参考值 +8-12%
  
  场景要素：
  - [ ] 产品特写镜头（微距）
  - [ ] 手持使用状态
  - [ ] 功效数据可视化
  - [ ] 使用前后对比（如适用）

📸 方案B：生活方式型（强调情感共鸣）
  构图风格：产品+生活场景+情绪表达
  主打人群：感性消费者，关心体验和感觉
  适用平台：A+生活场景区、社交媒体
  预期CTR提升：参考值 +15-20%
  
  场景要素：
  - [ ] 目标用户在目标场景使用
  - [ ] 正面情绪表达（笑脸/满足感）
  - [ ] 故事性元素（咖啡+周末+放松）
  - [ ] 色彩氛围统一

📸 方案C：高端质感型（强调品质感）
  构图风格：极简+光影+细节
  主打人群：品质导向消费者
  适用平台：独立站、Instagram、高端评价区
  预期CTR提升：参考值 +10-15%
  
  场景要素：
  - [ ] 纯色/浅色背景
  - [ ] 戏剧性光线（侧光/逆光）
  - [ ] 极简道具搭配
  - [ ] 微距细节展示

📋 测试执行计划：
| 周次 | 测试组 | 样本量目标 | 评估指标 |
|------|--------|-----------|---------|
| W1-2 | A vs B | 各500曝光 | CTR |
| W3-4 | 胜出组 vs C | 各500曝光 | CTR+转化率 |
| W5 | 冠军组扩量 | - | ROI验证 |

⚠️ 测试注意：
- 同等流量分配，避免人为偏差
- 至少运行2周，数据才有统计意义
- 记录所有变量，控制单一变量
```

## 不适用场景（Do Not Use When）

- ❌ 需要直接生成图片（纯策略工具，不含AI生图）
- ❌ 已有完整拍摄团队和执行计划（可能与现有流程冲突）
- ❌ 产品还未定型（拍摄计划无法锁定）
- ❌ 只想看单次场景建议（建议使用免费版 miaoji-scene-studio）
