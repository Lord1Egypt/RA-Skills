# 示例提问

## 1. 正常使用：报志愿

```text
我是河南物理类，2026 年高考 610 分，位次 28000，想学计算机或电子信息，帮我分析高考志愿怎么填。
```

## 2. 做冲稳保

```text
广东物理类，位次 45000，不接受民办和中外合作，优先广州、深圳、珠海，专业想要自动化、电气、计算机，帮我做冲稳保框架。
```

## 3. 学校深度调查

```text
帮我查查山东科技大学怎么样，孩子想报它的计算机专业，位次 3 万左右，家境普通，想读研。
```

## 4. 复杂场景：带文件 + 学校体检

```text
我上传了一份往年录取位次表，孩子是江苏物化生，位次 32000，想留在长三角，不接受医学和师范。帮我筛一版稳妥方案，然后帮我查查南京工业大学的安全和就业情况。
```

## 5. 检查已有方案

```text
这是我列的 30 个院校专业组，帮我看看冲稳保是否合理，有没有滑档或专业调剂风险。
```

## 6. 不该触发

```text
帮我写一篇高考励志演讲稿。
```

这不是志愿填报分析任务，不需要使用本 Skill。
