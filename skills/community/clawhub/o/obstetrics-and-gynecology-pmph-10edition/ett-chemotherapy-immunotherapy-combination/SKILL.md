# ETT化疗与免疫联合治疗策略

## 适用条件
- 患者确诊为上皮样滋养细胞肿瘤（ETT）；
- 已接受手术或标准治疗但失败；
- 疾病处于复发或转移阶段；
- 不适用于初治局限性ETT。

## 执行步骤
1. **确认病理类型与治疗史**：核实患者为ETT，并已接受过标准治疗（如EMA-CO方案）但无效。
2. **评估免疫治疗生物标志物**：检测肿瘤组织中PD-L1表达状态（阳性/阴性或CPS评分），优先考虑PD-L1高表达患者。
3. **制定联合治疗方案**：采用针对胎盘部位滋养细胞肿瘤（PSTT）的化疗方案（如依托泊苷+顺铂）联合PD-1/PD-L1抑制剂（如帕博利珠单抗、阿替利珠单抗等）。
4. **多学科协作与监测**：在多学科团队指导下启动治疗，并密切监测疗效及免疫相关不良反应（如甲状腺功能异常、肺炎、结肠炎等）。
5. **记录与随访**：将患者纳入个体化随访计划，定期评估肿瘤标志物（如hCG）和影像学变化。

## 注意事项
- 此方案属于探索性治疗，缺乏大规模临床证据支持；
- 化疗对ETT本身疗效有限，联合免疫治疗旨在增强抗肿瘤免疫应答；
- 治疗通常用于二线及以上治疗线数。
