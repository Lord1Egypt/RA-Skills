# Example Prompts

## Safe Mode

Use AutoPostOnline. List my connected integrations first. Create platform-specific drafts. Do not publish until I approve.

## Autonomous Mode

Use AutoPostOnline as my autonomous publishing layer. List connected integrations first. Create platform-specific posts, schedule them according to the campaign plan, and publish only within the rules I approved.

## Founder Brand Agent

Use AutoPostOnline to run my founder-brand content workflow. Turn my ideas into LinkedIn and X posts, schedule them for the week, and report what was scheduled.

## Agency Operator Agent

Use AutoPostOnline to manage social publishing for this client campaign. Stay inside the approved brand voice, channels, and posting schedule. Report every draft, scheduled post, and published post.

## Product Launch Agent

Use AutoPostOnline to create and schedule launch posts for the approved product campaign. Adapt each post per platform. Do not publish outside the approved channels.
