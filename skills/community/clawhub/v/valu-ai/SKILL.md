# ValU AI 智能估值分析器

> 🤖 让专业估值走进大众投资者

## 功能概览

| 功能 | 说明 |
|------|------|
| **单股分析** | 支持所有A股、港股上市公司，输入股票代码生成完整 DCF 估值报告 |
| **批量对比** | 2-5 只股票横向对比（PE/PB/ROE 等） |
| **行业识别** | 自动从网络获取公司所属行业 |
| **数据验证** | 完整性和准确性校验 |

## 支持范围

```
✅ 所有A股上市公司（约5000+只）
   ├── 上交所主板 (600xxx, 601xxx, 603xxx)
   ├── 科创板 (688xxx)
   ├── 深交所主板 (000xxx, 001xxx)
   ├── 中小板 (002xxx)
   └── 创业板 (300xxx)

✅ 部分港股上市公司
```

## 使用方式

### 单股分析
```
输入：股票代码（如 000002、600519、300450、9888.HK）
输出：完整的达莫达兰估值分析报告
```

### 批量对比
```
输入：多个股票代码（逗号分隔）
输出：横向对比分析报告
```

## 报告内容

生成的估值报告包含：
1. **核心诊断摘要** - 生命周期、盈利能力、现金流评估
2. **三阶段 DCF 模型** - 熊市/基准/牛市三种情景
3. **相对估值验证** - PE/PB/PS 与行业对比
4. **敏感性分析** - WACC、永续增长率影响
5. **投资建议** - 风险提示与核心假设
6. **免责声明** - 合规声明

## 数据来源

| 数据类型 | 来源 | 说明 |
|---------|------|------|

| 财务指标 | AKShare | 86项详细财务指标 |

## 工作流程

```
1. 接收股票代码
     ↓
2. 获取公司基础信息（名称、行业）
     ↓
3. 获取历史价格数据
     ↓
4. 获取财务报表（86项指标）
     ↓
5. 数据完整性校验
     ↓
6. 构建达莫达兰分析提示词
     ↓
7. 调用 AI 生成报告
     ↓
8. 添加免责声明并保存
```
