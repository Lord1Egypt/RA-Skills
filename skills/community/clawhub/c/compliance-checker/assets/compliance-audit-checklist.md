# Compliance Audit Quality Checklist

Run before delivering any compliance audit or violation response. Unchecked = fix or disclose in Assumptions.

## Scope & eligibility
- [ ] Audit surface defined (listing / creator content / ad / sweep / appeal)
- [ ] Product category and target market identified
- [ ] Category gate checked FIRST (prohibited / restricted / open)
- [ ] Required documents inventoried (approvals, certs, brand authorization)

## Claims audit
- [ ] Title, bullets, description ALL extracted
- [ ] Image text overlays checked (OCR surface)
- [ ] Video speech and on-screen text transcribed and checked
- [ ] Captions, pinned comments, live scripts included where in scope
- [ ] Every claim classified by type (treatment / measurable / superlative / authority / guarantee / comparative / subjective)
- [ ] Substantiation status confirmed for every retained objective claim
- [ ] Every Critical/High finding has a paste-ready compliant rewrite
- [ ] Rewrites preserve selling intent (not compliance-by-blandness)
- [ ] No synonym-swap rewrites that keep the violating claim type

## Pricing & promotion
- [ ] Reference/strikethrough price verified as genuine recent selling price
- [ ] Discount math checked
- [ ] Urgency/countdown claims match real offer ends
- [ ] Free shipping/gift claims match checkout behavior

## Creator content (if in scope)
- [ ] Disclosure: platform label + verbal/caption per local law
- [ ] Prohibited formats checked (before/after, medical staging, fear hooks)
- [ ] Offer accuracy vs. live checkout verified
- [ ] Music/IP licensing checked
- [ ] Do/don't brief produced or updated for the creator

## Risk grading & output
- [ ] Findings graded Critical/High/Medium/Low with grading rationale consistent
- [ ] Findings sorted Critical-first; verdict line present (publish / fix first / do not publish)
- [ ] Locations precise (field, image #, timestamp)
- [ ] Documentation gaps listed with actions

## Violation responses (if applicable)
- [ ] Root cause named (or ranked hypotheses)
- [ ] Fix completed BEFORE appeal drafted
- [ ] Appeal acknowledges the issue without restating the banned claim
- [ ] Evidence list attached and available same-day
- [ ] Account-wide sweep for the same pattern included
- [ ] Timeline works back from the appeal deadline

## Limits
- [ ] Platform-policy version/market dependence stated
- [ ] Platform rules vs. ad law distinction stated; legal review advised for high-risk categories
- [ ] Assumptions numbered in their own section
