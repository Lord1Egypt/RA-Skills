# 梅花易数推演方法

## 核心原理

梅花易数先确定上卦和下卦，两卦叠成六十四卦之一，再取动爻确定变卦。以体卦（不含动爻的卦）和用卦（含动爻的卦）的生克关系为核心断事。

## 起卦方法

### 一、时间起卦法（默认）
以当前农历时间起卦：
1. 上卦 = 年支数 + 月数，除以 8，余数对应先天八卦数（乾1兑2离3震4巽5坎6艮7坤8，余0取8）
2. 下卦 = 年支数 + 月数 + 日数，除以 8
3. 动爻 = 年支数 + 月数 + 日数 + 时数，除以 6（余0=6爻）

### 二、报数起卦法
用户报两个或三个数字：
- **两个数字**：第一个为上卦数，第二个为下卦数（除以8取余）；动爻 = (上卦数+下卦数) 除以6取余
- **三个数字**：前两个取上下卦，第三个取动爻

### 三、物象起卦法
根据方位、人物、颜色、动物等物象对应到八卦。

## 确定体卦和用卦

- 本卦中，不含动爻的那个卦为**体卦**（代表自身/主体）
- 含动爻的那个卦为**用卦**（代表对方/外部环境）
- 若动爻在上卦，则上卦为用、下卦为体；反之亦然

## 变卦和互卦

- **变卦**：将动爻翻转（阳变阴、阴变阳），得到的新卦
- **互卦**：取本卦的二三四爻为互卦下卦，三四五爻为互卦上卦。互卦代表事情的中间发展过程

## 五行生克判断

### 判断规则（核心）
| 关系 | 含义 |
|------|------|
| 用生体（用卦五行生体卦五行） | 大吉。对方/环境有利于我 |
| 体用比和（五行相同） | 吉。主客相合，事情顺利 |
| 体克用（体卦克用卦） | 小吉。我方能掌控局面，但需付出努力 |
| 体生用（体卦生用卦） | 小凶。我在付出，耗泄自己 |
| 用克体（用卦克体卦） | 大凶。对方/环境压制我 |

### 结合季节旺衰
- 春天木旺、夏天火旺、秋天金旺、冬天水旺
- 四季末土旺（辰戌丑未月最后18天）
- 旺相之卦力量加倍；衰弱之卦力量减半

参考 references/core/yinyang-wuxing.md 获取完整五行生克和旺相休囚表。

## 输出格式

1. **起卦参数**：时间/报数 → 上卦数+下卦数+动爻
2. **卦象呈现**：本卦·变卦·互卦（含卦名和符号）
3. **体用判定**：体卦五行 + 用卦五行 + 生克关系
4. **卦辞引用**：引用周易中对应卦的基本卦辞
5. **综合断卦**：结合体用生克 + 五行旺衰 + 变卦互卦 + 具体场景给解读
6. **注意**：必须在最后附免责声明"占卜结果仅供参考，请结合实际情况理性判断"
