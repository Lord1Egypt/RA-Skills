# 阑尾解剖变异与症状定位

## 触发条件
当疑似急性阑尾炎患者出现以下情况时使用本技能：
- 腹痛起始或压痛点不在典型麦氏点
- 伴有排便里急后重、尿频或右腰痛等非典型症状

## 执行流程

1. **确认阑尾根部恒定**：
   - 根部始终位于盲肠末端三条结肠带汇合处。

2. **根据压痛部位反推尖端方位**（六种类型）：
   - **回肠前/后位（0–3点）**：典型右下腹痛，后者更深在。
   - **盆位（3–6点）**：耻骨上压痛，伴直肠刺激征。
   - **盲肠后位（9–12点）**：右侧腰部痛，体征轻，易误诊，手术显露困难。
   - **盲肠下位（6–9点）**：右下腹偏外侧压痛。
   - **盲肠外侧位（9–10点）**：盲肠外侧腹腔内。

3. **识别特殊变异**：
   - 高位阑尾（肝下）→ 右上腹痛。
   - 跨中线阑尾 → 左下腹痛。

4. **指导临床决策**：
   - 调整查体重点区域。
   - 选择合适手术切口（如McBurney切口对盲肠后位显露不佳，需考虑扩大切口或腹腔镜）。

> **限制**：不适用于阑尾先天缺如或严重解剖变异无法归类者。
