## 活六神

活六神顺序：青龙 -> 朱雀 -> 勾陈 -> 白虎 -> 玄武 -> 腾蛇。

六宫固定顺序：大安 -> 留连 -> 速喜 -> 赤口 -> 小吉 -> 空亡。

| 时辰 | 活六神起于 |
| --- | --- |
| 子时 / 午时 | 大安 |
| 丑时 / 未时 | 留连 |
| 寅时 / 申时 | 速喜 |
| 卯时 / 酉时 | 赤口 |
| 辰时 / 戌时 | 小吉 |
| 巳时 / 亥时 | 空亡 |

Formula:

```text
活六神索引 = (宫位索引 - 起始宫位索引 + 6) % 6
```

## 五行与六亲

五行相生：木 -> 火 -> 土 -> 金 -> 水 -> 木。

五行相克：木 -> 土 -> 水 -> 火 -> 金 -> 木。

以人宫五行为"我"：

| 关系 | 条件 | 含义 |
| --- | --- | --- |
| 兄弟 | 同五行 | 同辈、竞争、合作 |
| 父母 | 他生我 | 庇护、长辈、文书 |
| 子孙 | 我生他 | 产出、晚辈、福德 |
| 妻财 | 我克他 | 财富、收获、可控对象 |
| 官鬼 | 他克我 | 压力、约束、疾病、风险 |

## 六宫细义

### 大安

- 总象：平安、稳定、顺利。
- 疾病：较轻，易恢复。
- 婚姻：稳定和谐。
- 行人：平安，归来不远。
- 事业：稳步发展。
- 财运：稳定增长。
- 寻物：未远，较易找回。
- 官司：有惊无险。

### 留连

- 总象：拖延、迟滞、反复。
- 疾病：反复，恢复慢。
- 婚姻：有波折，需要耐心。
- 行人：行程延迟。
- 事业：进展缓慢。
- 财运：平平，来源不稳。
- 寻物：附近细找。
- 官司：拖延，中平。

### 速喜

- 总象：消息快、进展快、有喜象。
- 疾病：好转迅速。
- 婚姻：进展较快。
- 行人：将归或有音信。
- 事业：有助力，推进快。
- 财运：短期收益或意外机会。
- 寻物：较快找回。
- 官司：有较快解决机会。

### 赤口

- 总象：争执、口舌、短期冲突。
- 疾病：需重视，可能有明显不适。
- 婚姻：争吵、意见不合。
- 行人：途中有阻或有惊。
- 事业：竞争强、阻碍多。
- 财运：纠纷、收入不稳。
- 寻物：较难找回或损坏。
- 官司：多争执，不利。

### 小吉

- 总象：小成、小喜、稳中有进。
- 疾病：小病小痛。
- 婚姻：总体平稳，小有波折。
- 行人：平安。
- 事业：适合稳妥推进。
- 财运：小有收获。
- 寻物：可能找回，需要耐心。
- 官司：小胜小负，不宜扩大。

### 空亡

- 总象：虚耗、落空、阻隔。
- 疾病：复杂，需谨慎处理。
- 婚姻：缘分淡，难成。
- 行人：音讯少或受困。
- 事业：计划落空，成效弱。
- 财运：不宜冒进。
- 寻物：难找或已失。
- 官司：不利或无进展。

## Example

Input numbers: 3, 5, 7.

```text
processedNum1 = 3
processedNum2 = 5
processedNum3 = 7
天宫值 = 3 % 6 = 3 -> 速喜
地宫值 = (3 + 5 - 1) % 6 = 1 -> 大安
人宫值 = (1 + 7 - 1) % 6 = 1 -> 大安
```

## FAQ

Q: 输入 0 怎么处理？
A: 将 0 视为 10 参与运算。

Q: 体用怎么判断？
A: 人宫为体，起卦时辰为用。以人宫五行为"我"，看时辰五行与"我"的生克关系。

Q: 活六神怎么记？
A: 子午大安丑未留，寅申速喜卯酉赤，辰戌小吉巳亥空。
