# CHANGELOG - checktruth（别瞎说）🔍

## [Unreleased]

### Added
- 新增"关于多视角验证的说明"章节
- 新增 `reference/multi_model_verify.py`（真正的多模型API调用脚本）
- 新增 `reference/requirements.txt`
- 新增 `reference/README.md`（使用说明）

### Changed
- **版本号**：v0.3.1 → v0.4
- **核心功能**：加入"多视角交叉验证"流程（3轮验证：严谨核查员 → 质疑者 → 综合裁判）
- **描述更新**：明确"多视角交叉验证（零配置）"
- **示例更新**：展示3个视角的验证过程
- **移除**：homepage 中的 gitee.com 链接

### Fixed
- 修复示例1中魏建军持股比例的错误（56% → 正确应为约30%）
- 统一中英文触发词说明

---

## [0.3.1] - 2026-05-10

### Added
- 统一触发词：英文 `/checktruth`，中文 `/别瞎说`
- 自动模式判断（模式A/B）

### Changed
- 更新 `metadata.security`，明确区分核心功能（零配置）和可选功能（需API Key）
- 将 `src/` 移到 `reference/`（明确标注为可选参考代码）
- 版本号更新：v0.3 → v0.3.1

### Fixed
- 修复 ClawScan 3个Medium问题（误报需要外部API Key）

---

## [0.3] - 2026-05-09

### Added
- 支持两种模式：模式A（问答验证）、模式B（文章/论点验证）
- WebSearch 集成（获取参考信息）
- 原子事实分解
- 逐事实验证 + 置信度评分
- 内部一致性检测
- 综合评分系统（0-100分）

### Changed
- 核心功能零配置（无需外部API Key）
- `reference/` 文件夹包含可选参考代码（需外部API Key）

---

## [0.2] - 2026-05-08

### Added
- 初始版本
- 基于 FActScore 的事实核查逻辑
- 支持中英文输入

---

## [0.1] - 2026-05-07

### Added
- 项目初始化
- 基础事实核查功能

## 格式说明

本文档遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 规范。

**版本号规则**：
- **主版本号**：不兼容的API修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

**类型说明**：
- `Added`：新功能
- `Changed`：功能变更
- `Deprecated`：即将移除的功能
- `Removed`：已移除的功能
- `Fixed`：Bug修复
- `Security`：安全问题修复
