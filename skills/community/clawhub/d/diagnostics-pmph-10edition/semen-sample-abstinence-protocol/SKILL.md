# 精液标本采集前禁欲时间规范

## 何时使用
- 进行精液常规检查
- 评估男性生育功能
- 精子库供精者筛选
- 需要重复采集精液以验证结果一致性

## 如何执行
1. 要求受检者在精液标本采集前严格禁欲（包括无性交、无手淫、无遗精）持续 **2至7天**。
2. 若需多次采集精液，每次采集前的禁欲时间应尽可能保持一致，以减少个体波动对检测结果的影响。
3. 明确告知：禁欲时间过短（<2天）可能导致精液量和精子总数偏低；禁欲时间过长（>7天）可能导致精子活动率下降、异常形态精子比例升高。
4. 采集时必须准确记录实际禁欲天数，并将其作为精液分析报告的重要参考信息。
5. 若因人为因素（如标本丢失、禁欲时间不符）导致结果异常，应排除该次结果并安排重新采集。

## 注意事项
- 本规范不适用于紧急法医学鉴定等特殊场景。
- 实际禁欲时长必须由受检者如实提供并记录，不可估算。
- 禁欲期间若发生遗精，应视为禁欲中断，需重新计时。
