# Color Trend & Selection Guide

|---|---|---|---|

## Seasonal Color Performance Patterns

### Northern Hemisphere Seasonal Shifts
| Season | Trending Direction | Example Colors | Planning Window |
|---|---|---|---|
| Spring (Mar-May) | Pastels, fresh greens, soft pinks | Lavender, Sage, Blush, Sky Blue | Order inventory by January |
| Summer (Jun-Aug) | Bright, saturated, tropical | Coral, Turquoise, Lemon Yellow, Hot Pink | Order inventory by March |
| Fall (Sep-Nov) | Earth tones, warm depth | Burgundy, Rust, Olive, Mustard, Chocolate | Order inventory by June |
| Winter (Dec-Feb) | Deep jewel tones, metallics | Emerald, Ruby, Midnight, Gold, Silver | Order inventory by September |

### Color Lifecycle Stages
| Stage | Characteristics | Strategy |
|---|---|---|
| Emerging | Appearing on runways, fashion media, Pantone forecast | Monitor; small test order if aligns with brand |
| Rising | Growing search volume, influencer adoption | Test with limited inventory, measure demand |
| Peak | Maximum market penetration, mass retail adoption | Stock aggressively if data supports |
| Declining | Search volume dropping, retail markdowns common | Begin sell-through, reduce reorders |
| Dormant | Minimal demand, may cycle back in 5-10 years | Do not stock; archive data for future reference |

## Color Selection Framework

### Step 1: Secure Core (60-70% of allocation)
Always stock your category's must-have neutrals. These form the foundation and have year-round demand.

### Step 2: Add Seasonal Rotators (15-25% of allocation)
Select 2-3 colors aligned with the upcoming season's trends. Validate with:
- Google Trends search volume for "[color] + [product category]"
- Pinterest trend reports (strong predictor 3-6 months ahead)
- Competitor new color launches in past 60 days
- Pantone Color of the Year and seasonal palette announcements

### Step 3: Test Accent Colors (5-15% of allocation)
Add 1-2 bold or niche colors with limited inventory. These create urgency and social media shareability. Test criteria:
- Order minimum viable quantity only
- Set 30-day performance gate (must hit 3% of revenue)
- Use as limited edition positioning if successful

## Color-Specific Data Points to Track

| Metric | Why It Matters | Red Flag Threshold |
|---|---|---|
| Conversion rate by color | Some colors attract browsers but not buyers | CVR <50% of best-performing color |
| Return rate by color | Color accuracy issues drive returns | Return rate >2x product average |
| Review sentiment by color | "Color looks different in person" signals photo issues | Multiple mentions of color mismatch |
| Session-to-click rate | Do shoppers click on this color swatch? | <5% of product page sessions select this variant |
| Photography consistency | Color must look accurate and consistent across images | Visual mismatch between hero and lifestyle shots |

## Color Naming Best Practices

### Naming Impact on Conversion
Research shows descriptive color names outperform generic names:
- "Ocean Blue" outperforms "Blue" by 8-15% in CVR
- "Midnight Black" outperforms "Black" by 3-5% for premium products
- Overly creative names ("Whispered Dusk") can confuse and hurt CVR
- Names should be evocative but immediately understood

### Platform-Specific Naming
| Platform | Best Practice |
|---|---|
| Amazon | Searchable name first: "Navy Blue" not "Midnight Wave" (SEO matters) |
| Shopify DTC | Creative names okay if swatch image is clear: "Driftwood" for tan |
| TikTok Shop | Short, memorable names that work in short-form video mentions |

## Color Cannibalization Risk Matrix

| Pairing | Cannibalization Risk | Action |
|---|---|---|
| Black + Charcoal | High (75%+ overlap) | Only stock both if data proves incremental demand |
| Navy + Dark Blue | Very High (85%+ overlap) | Choose one based on naming convention |
| Beige + Tan + Cream | High (60-80% overlap) | Stock maximum 2 of 3 |
| Olive + Army Green + Sage | Medium (40-60% overlap) | Different enough if clearly differentiated in photos |
| Red + Burgundy | Low-Medium (30-40%) | Seasonal differentiation (Red=summer, Burgundy=fall) |
| Pink + Blush + Coral | Medium (50-65%) | Different audiences; validate with demographic data |
