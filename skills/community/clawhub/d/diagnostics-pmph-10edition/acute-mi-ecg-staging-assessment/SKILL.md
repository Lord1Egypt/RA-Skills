# 急性心肌梗死心电图分期评估

## 适用条件
- 患者为急性心肌梗死且未接受溶栓或介入等早期再灌注治疗
- 具备连续心电图监测数据
- 呈现典型自然病程（非小范围或非典型梗死）

## 执行步骤

1. **识别当前心电图特征**：重点观察ST段形态、T波方向与振幅、Q波是否存在及其稳定性。
2. **匹配四期演变阶段**：
   - **超急性期（发病数分钟至数小时）**：高大T波、ST段上斜型或弓背向上型抬高、QRS振幅增高但无异常Q波。
   - **急性期（数小时至数周）**：ST段显著抬高呈单向曲线后逐渐回落，R波降低或消失，出现异常Q波或QS波，T波由直立转为倒置。
   - **亚急性期（数周至数月）**：ST段恢复基线，T波倒置由深变浅，异常Q波持续存在。
   - **陈旧期（数月后）**：ST段和T波基本正常或T波恒定倒置/低平，残留坏死型Q波（长期可能缩小或消失）。
3. **考虑例外情况**：若患者曾接受有效再灌注治疗，典型演变可能被缩短或中断，此时应谨慎解释分期结果。
4. **输出分期结论**：明确当前所处阶段，并结合临床背景用于治疗决策或预后评估。

## 注意事项
- 小范围梗死可能不呈现典型演变，需结合心肌酶学及影像学综合判断。
- Q波理论上持续存在，但长期随访中可能因瘢痕重塑而变小甚至消失。
- 本技能仅适用于自然病程下的典型演变；若存在再灌注治疗史，应优先考虑治疗对心电图的影响。
