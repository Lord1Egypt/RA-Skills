# 大数投资完整标准与操作手册

## 理论依据

大数投资由中国人民大学齐东平教授创立，核心原理：
- **大数定律**：随机抽样足够多时，样本均值趋近于总体均值
- **中心极限定理**：组合的收益和风险可预测
- **国民经济映射**：上市公司整体代表国民经济

## 选股五步法

### 第一步：行业分层

将全市场上市公司划分为**30个以上行业样本群**，确保：
- 每个行业都有代表
- 行业间相关性尽可能低（分散风险）
- 行业划分参考证监会分类标准

### 第二步：估值筛选（核心！）

**主要指标**：
| 指标 | 含义 | 筛选标准 | 说明 |
|------|------|---------|------|
| PB | 市净率 | **≤ 1.5** | 越低越好，<1为破净 |
| PE | 市盈率 | **≤ 20** | 越低回本越快 |
| ROE | 净资产收益率 | **≥ 5%** | 反映盈利能力 |
| 股息率 | 分红比例 | **越高越好** | 熊市保护垫 |

**优先顺序**：PB破净（<1）> PB<1.5+高股息 > PE极低

### 第三步：排除雷区

排除以下股票：
- ✗ ST股 / *ST股
- ✗ 连续2年亏损
- ✗ 重大违规、财务造假
- ✗ 大股东高比例质押（>50%）
- ✗ 商誉占净资产>30%

### 第四步：行业龙头优先

同行业选股优先级：
1. **行业市值第一**—— 竞争优势最强
2. **央企/国企**—— 政策风险低
3. **分红稳定**—— 现金流验证

### 第五步：动态平衡

- **定期再平衡**：每季度检视，卖出高估、买入低估
- **不停牌原则**：不问不炒，长期持有
- **补仓纪律**：跌多了可补，上涨不追

## 仓位配置

### 基础配置

```
总资金 = 100份
每个行业 = 100 ÷ 30 ≈ 3.3份
每行业2只 = 每只约1.65份
现金储备 = 20份（补仓用）
```

### 估值调整仓位

| 估值情况 | 仓位建议 |
|---------|---------|
| PB < 0.7（深度破净）| 可超配，最多1.5倍 |
| PB 0.7~1.0（破净）| 标准仓位 |
| PB 1.0~1.5（合理偏低）| 标准仓位或偏低 |
| PB > 2（偏高）| 观望，等待回调 |

## 买入策略

### 一次性买入（强烈低估时）
- 目标：PB<0.8，PE<10
- 时机：市场整体恐慌时期
- 比例：可建仓60%仓位

### 分批买入（正常偏低时）
- 目标：PB 0.8~1.5
- 时机：估值合理偏低
- 比例：首批30%，后续逢跌加仓

### 不买（高估时）
- 条件：PB>2，或PE>30
- 原则：宁可错过，不追高

## 卖出策略

**触发卖出条件（满足任一）**：
1. PE > 50（极度高估）
2. PB > 3（相对历史高位）
3. 行业基本面永久恶化（如教培行业政策）
4. 达到个人收益目标（建议年化10%+）
5. 发现更好低估标的（换仓）

**不触发卖出**：
- 短期波动（情绪噪音）
- 行业周期性调整（不离场）
- 股价短期下跌（不加仓即可）

## 常见错误

1. ❌ 只看PE不看PB—— 重资产行业PB更重要
2. ❌ 集中持有单一行业—— 违背分散原则
3. ❌ 频繁交易—— 违背长期持有原则
4. ❌ 全仓操作—— 必须保留20%+现金
5. ❌ 跟随题材炒作—— 坚守低估值纪律
