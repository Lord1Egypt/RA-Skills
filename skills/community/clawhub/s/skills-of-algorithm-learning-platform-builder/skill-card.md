## Description: <br>
Builds interactive algorithm learning pages, comparison pages, and reusable teaching platforms with formulas, numerical substitution, charts, controls, upgrade guidance, and optional runnable HTML demos. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[doctorfff](https://clawhub.ai/user/doctorfff) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, educators, and learning-content builders use this skill to plan, structure, compare, and generate interactive algorithm teaching pages or reusable algorithm learning platforms. It is especially suited for turning abstract formulas into step-by-step, chart-rich, browser-runnable educational material. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated runnable HTML may include CDN-loaded demo libraries. <br>
Mitigation: Review generated HTML before publishing, and replace CDN scripts with pinned local copies when offline use, strict privacy, or supply-chain control is required. <br>
Risk: Broad request routing can produce educational pages that need subject-matter review for accuracy. <br>
Mitigation: Review formulas, numerical substitutions, chart behavior, and generated explanations before using outputs in coursework or public materials. <br>
