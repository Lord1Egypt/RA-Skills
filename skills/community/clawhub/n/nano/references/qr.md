# xno-skills qr

```
Usage: cli qr [options] <address>

Generate a QR code for a Nano address

Arguments:
  address                Nano address

Options:
  --amount-xno <amount>  Include amount in XNO
  --format <format>      QR code format (ascii or svg) (default: "ascii")
  -j, --json             Output in JSON format
  -h, --help             display help for command
```
