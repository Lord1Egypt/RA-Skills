# 抗菌药物按作用类型分类

## 何时使用
- 已知抗菌药物名称或类别
- 需判断其为抑菌药（bacteriostatic）或杀菌药（bactericidal）
- 患者存在免疫功能低下或普通感染

## 如何执行
1. **分类判断**：
   - **抑菌药**：仅抑制细菌生长，依赖宿主免疫清除（如四环素类、红霉素、磺胺类）
   - **杀菌药**：直接杀灭细菌，不依赖免疫系统（如青霉素类、头孢菌素类、氨基糖苷类）

2. **临床决策**：
   - 免疫功能低下患者：优先选择杀菌药
   - 普通感染：可根据病原体和药物特性灵活选择

3. **注意浓度依赖性**：
   - 某些药物（如氨基糖苷类）低浓度抑菌，高浓度杀菌

> 注意：本分类不适用于抗病毒或抗真菌药物，仅限细菌性感染。
