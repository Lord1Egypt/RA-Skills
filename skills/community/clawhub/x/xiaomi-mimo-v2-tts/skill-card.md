## Description: <br>
Xiaomi-MiMo-V2-TTS synthesizes Chinese or English speech with selectable voices, emotions, dialects, role styles, speed control, singing mode, and WAV, MP3, PCM16, or OPUS output. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[php737](https://clawhub.ai/user/php737) <br>

MIT-0 <br>


## Use Case: <br>
Developers and content creators use this skill to generate single or batch audio files from text through Xiaomi MiMo V2 TTS for chat, narration, character voice, dialect, and messaging workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Synthesis text and optional context are sent to Xiaomi MiMo's API. <br>
Mitigation: Avoid secrets and regulated or confidential content in synthesis text, and review the provider's terms before using the skill with sensitive data. <br>
Risk: The skill requires a Xiaomi MiMo API key. <br>
Mitigation: Use a dedicated revocable API key and verify the install slug and publisher before use. <br>
