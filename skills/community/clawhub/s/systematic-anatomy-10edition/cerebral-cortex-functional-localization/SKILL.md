# 大脑皮质功能定位

## 何时使用
- 解释由大脑皮质局灶性病变引起的神经功能缺损（如运动障碍、感觉异常、视觉/听觉障碍、语言障碍）
- 分析神经影像（如MRI、fMRI）中皮质激活或损伤区域的临床意义
- 推断特定皮质区域（按Brodmann分区）受损后可能出现的症状
- 区分皮质性与非皮质性（如脑干、脊髓）神经功能障碍

## 执行步骤

### 1. 应用核心定位原则
- 确认所涉区域属于大脑皮质（排除脑干、小脑、基底节等非皮质结构）
- 明确该功能在皮质上有相对集中但非唯一的“核心中枢”
- 考虑联络区（association areas）在高级整合功能中的作用
- 注意左右半球功能不对称性：左侧主导语言与逻辑分析，右侧主导空间、音乐与图形处理

### 2. 定位主要功能中枢并关联症状

**第Ⅰ躯体运动区（中央前回 + 中央旁小叶前部，Brodmann 4、6区）**
- 损伤导致对侧肢体上运动神经元瘫痪（上下颠倒、左右交叉）
- 注意双侧支配例外：眼球外肌、咽喉肌、咀嚼肌

**第Ⅰ躯体感觉区（中央后回 + 中央旁小叶后部，3、1、2区）**
- 损伤导致对侧半身痛、温、触、压及本体感觉障碍
- 敏感区域（如手指、唇）投射范围更大

**视区（距状沟上下方，17区）**
- 单侧损伤 → 对侧同向性偏盲
- 后1/3损伤 → 黄斑回避现象可能缺失

**听区（颞横回，41、42区）**
- 单侧损伤通常不引起完全耳聋（因双耳输入交叉投射）

**语言中枢（人类特有）**
- Broca区（额下回后部，44、45区）损伤 → 运动性失语（能理解但不能流利表达）
- 书写区（额中回后部，6、8区）损伤 → 失写症
- 听觉性语言区（颞上回后部，22区）损伤 → 感觉性失语（言语流利但无意义，理解障碍）
- 视觉性语言区（角回，39区）损伤 → 失读症
- Wernicke区（整合22、39、40、37区）损伤 → 严重感觉性失语伴错语

**其他功能区**
- 味觉区（43区）、嗅觉区（海马旁回钩）、内脏活动中枢（边缘叶）等较少见，需结合具体症状判断

### 3. 输出结果
- 明确指出受损皮质区域（含Brodmann编号）
- 预测或解释对应的神经功能缺损症状
- 注明是否符合典型投射规律（如交叉、倒置、敏感度相关范围）
- 提示是否需考虑半球优势（尤其语言功能）
