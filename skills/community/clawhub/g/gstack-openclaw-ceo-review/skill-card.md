## Description: <br>
Provides CEO/founder-mode plan reviews that challenge premises, rethink scope, examine implementation risks, and guide users through expansion, selective expansion, hold-scope, or scope-reduction modes. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[garrytan](https://clawhub.ai/user/garrytan) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, founders, product leaders, and agents use this skill to review implementation plans before coding. It helps identify wrong problem framing, scope mistakes, architecture risks, missing error paths, observability gaps, testing gaps, and UX edge cases while keeping scope changes under explicit user control. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can cause potentially sensitive plan review summaries to be saved to persistent memory without asking first. <br>
Mitigation: Review before installing for confidential product plans, security issues, customer data, or internal architecture; disable or edit the memory-saving instruction, or use it only when persistent storage of the final review summary is acceptable. <br>
