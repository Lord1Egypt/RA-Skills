# 氮平衡计算与临床意义判定

## 何时使用
- 需评估患者的蛋白质合成与分解代谢状态
- 监测营养支持治疗效果
- 判断术后、创伤或重症患者的恢复趋势

## 执行步骤
1. **确认适用对象**：仅适用于肾功能正常、能完整收集24小时尿液及其他排泄物的住院患者、重症病人、术后恢复期患者或营养支持治疗对象。
2. **收集必要数据**：
   - 准确记录24小时内所有蛋白质摄入量（g）
   - 完整收集24小时尿液，测定尿尿素氮（UUN，单位：g）
3. **计算氮平衡**：
   - 摄入氮量（g） = 蛋白质摄入量（g） ÷ 6.25
   - 排出氮量（g） = 尿尿素氮（UUN） + 4（估算非尿氮损失）
   - 氮平衡 = 摄入氮量 - 排出氮量
4. **判定氮平衡类型**：
   - **总平衡**：摄入氮 ≈ 排出氮 → 正常成人稳态
   - **正平衡**：摄入氮 > 排出氮 → 合成代谢占优（如康复期、孕妇）
   - **负平衡**：摄入氮 < 排出氮 → 分解代谢占优（如严重感染、烧伤、肿瘤）
5. **输出临床提示**：根据平衡类型给出营养干预建议。
6. **注意事项**：
   - 连续监测3–5天以提高可靠性
   - 急性应激状态下即使摄入充足也可能出现负平衡
   - 排除肾功能衰竭或未计入的非尿氮丢失（如伤口渗出、出血）

> 提示：若数据不完整或患者存在排泄异常，请勿使用本技能。

## 参考文档

### 氮平衡计算详细说明

**丢失氮的构成**
- 尿液中尿素氮（UUN）占尿液总氮的约85%
- 非尿氮损失（约4g/d）包括：
  - 粪便氮（约1-2g/d）
  - 皮肤丢失（约0.5g/d）
  - 其他不可测丢失

**不同应激状态下的氮丢失**
- 择期手术后：约10-15g/d
- 严重创伤：约20-30g/d
- 大面积烧伤：约30-40g/d

**临床调整示例**
- 连续3天负平衡 > -10g/d → 提示需增加蛋白质供给或控制分解代谢
- 从负平衡转为正平衡 → 提示合成代谢开始占优，预后改善信号
