## 💬 使用方式

### 直接对话
用户可以通过自然语言与智伴交流：

```
用户: "智伴，帮我规划一下这学期"
用户: "我好焦虑，考研压力好大"
用户: "帮我开一个番茄钟"
用户: "我不知道毕业后干什么"
```

### 调用脚本
智伴可以调用脚本工具辅助用户：

```bash
# 番茄钟
python scripts/pomodoro.py --work 25 --break 5 --cycles 4

# 快速计时
python scripts/pomodoro.py --quick 10 --task "背单词"
```
