# 重组人生长激素（rhGH）治疗与监测

## 适用对象
已确诊为生长激素缺乏症的患儿，且满足以下前提：
- 无恶性肿瘤或严重糖尿病
- 头颅MRI已排除颅内肿瘤

> **禁忌**：恶性肿瘤患者、严重糖尿病患者禁用。

## 执行流程

### 1. 初始剂量设定
- 起始剂量：30 μg/(kg·d) 或 0.1 U/(kg·d)
- 给药方式：每晚临睡前皮下注射一次，或每周总剂量分6～7次注射

### 2. 个体化剂量调整
根据疗效与安全性指标动态调整剂量。

### 3. 疗效与安全性监测
- **定期检测**血清IGF-1和IGFBP-3水平
- 若IGF-1持续高于同性别、同年龄、同青春期状态正常儿童上限，应减少rhGH剂量
- 若年生长速率低于2～2.5 cm，可考虑终止儿科剂量rhGH治疗

### 4. 辅助监测
- **甲状腺功能**：rhGH可能诱发甲状腺功能减退，必要时加用左旋甲状腺素
- **糖代谢**：治疗前及过程中定期检查空腹血糖、胰岛素；必要时行OGTT试验
- **肾上腺皮质功能**：对合并多种垂体激素缺乏者需重点监测

### 5. 副作用管理
- **常见**：注射部位红肿、少数产生抗体（通常不影响疗效）
- **少见但需警惕**：暂时性视乳头水肿/颅内高压、股骨头滑脱/坏死（极低发生率）
- **风险说明**：目前临床证据未显示rhGH增加肿瘤发生/复发风险或导致糖尿病，但仍严格禁用于禁忌人群

### 6. 治疗时机提示
- 年龄越小效果越好
- 第1年疗效最佳，年身高增长可达10～12 cm及以上，之后逐年下降

## 输出
根据监测结果调整rhGH剂量或决定是否终止治疗，并同步处理相关内分泌异常（如甲减、糖代谢异常等）。
