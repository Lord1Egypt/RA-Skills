# 🎯 散户决胜法 — Retail Edge Framework

> **VC 在从叙事驱动转向 DPI 驱动，散户也应该从投机驱动转向认知驱动。**

2026 Crypto VC 新周期下的散户认知驱动投资框架。从赛道定位、链上可验证性、VC筹码结构到散户优势位，五步判断机会是否适合散户参与。

## 框架

### 五步评估
1. **赛道定位** — 属于四大收窄方向吗？
2. **链上可验证性** — 有链上数据支撑吗？
3. **VC筹码结构** — 解锁压力对散户友好吗？
4. **散户优势位** — 散户有什么独特优势？
5. **可操作结论** — 参与吗？怎么参与？

### 评级
- 🎉 极优 — 链路清晰、可验证、散户优势明确
- 🟢 可以 — 值得跟踪研究，小仓位试探
- 🟡 观望 — 信息不足或风险不对称
- 🔴 回避 — 明显对散户不利
