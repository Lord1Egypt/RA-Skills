## Known Risks and Mitigations: <br>
Risk: The skill can guide an agent to make real DOOMSCROLLR account changes through a user-configured API key. <br>
Mitigation: Review live publishing, subscriber changes, imports, pricing, inventory, deletions, and payment-related actions before approving them. <br>
Risk: The skill requires sensitive credentials for MCP or REST API access. <br>
Mitigation: Use the official connector or REST client with a properly scoped key, and keep API keys out of chat, logs, screenshots, and final replies. <br>
Risk: Product import and affiliate workflows can publish or preserve external commercial links. <br>
Mitigation: Preview imports when possible and confirm the intended product, post, or affiliate-link behavior before creating public content. <br>
