# 全身性疾病相关眼底特征性改变识别

## 何时使用
当眼底检查发现以下任一异常表现时立即启用本技能：
- 视网膜出血（火焰状、点状、片状或白心出血）
- 渗出物（棉絮状或硬性）
- 动脉变细、反光增强、铜丝状/银丝状改变或动静脉交叉压迫
- 视盘水肿或边界不清

## 执行步骤
1. **记录关键眼底变量**：
   - 出血形态（火焰状、点状、片状、白心出血/Roth斑）
   - 动脉状态（痉挛、变细、铜丝状、银丝状）
   - 渗出类型（棉絮状 vs 硬性）

2. **匹配特征与疾病**：
   - **高血压动脉硬化**：动脉变细、反光增强、动静脉交叉压迫；晚期出现火焰状出血、棉絮状渗出、视盘水肿。
   - **慢性肾炎**：视盘及周围视网膜水肿、火焰状出血、棉絮状渗出。
   - **子痫前期-子痫**：视网膜动脉痉挛、水肿，严重时可致视网膜脱离。
   - **糖尿病**：后极部深层点状或片状出血，伴静脉扩张迂曲。
   - **白血病**：Roth斑（带白色中心的出血）、视网膜血管色淡曲张、视盘边界不清。

3. **关键鉴别要点**：
   - 火焰状出血 → 神经纤维层病变 → 高血压或肾炎
   - 深层点状出血 → 糖尿病
   - Roth斑 → 高度提示白血病
   - 铜丝/银丝样动脉 → 高血压硬化期特异性表现

4. **输出建议**：
   提示可能存在的全身性疾病，并明确建议结合实验室检查（如血压、尿蛋白、血糖、血常规）及影像学进一步确诊。注意：眼底改变仅为辅助诊断依据，部分早期全身性疾病可能无眼底表现。
