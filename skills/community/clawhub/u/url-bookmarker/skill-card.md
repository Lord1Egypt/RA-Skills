## Description: <br>
Save, categorize, search, and export web bookmarks using local JSON storage with tags and folders. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[fuzzyb33s](https://clawhub.ai/user/fuzzyb33s) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and external users use this skill to save, organize, find, remove, and export bookmarks with folders and tags. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Bookmark URLs and tags are stored in the skill's local JSON file. <br>
Mitigation: Use the skill only for bookmark data that is appropriate to keep in local storage, and review assets/bookmarks.json before sharing the skill directory. <br>
Risk: Remove operations edit the local bookmark file and may be permanent without a backup. <br>
Mitigation: Keep a backup of assets/bookmarks.json before destructive bookmark changes. <br>
