## Description: <br>
Generate sound effects from text prompts using the Tomoviee Text-to-Sound-Effect API through the Wondershare OpenAPI gateway. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wondershare-boop](https://clawhub.ai/user/wondershare-boop) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and content-production agents use this skill to create Tomoviee text-to-sound-effect tasks, poll for completion, and retrieve generated audio URLs for prompts describing sound events or ambience. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: API credentials and generated Basic tokens can be exposed through shell history, process listings, logs, or CI output. <br>
Mitigation: Run the scripts in a private local environment, avoid passing real secrets on shared command lines, keep tokens out of logs, and rotate credentials if exposure is suspected. <br>
