# 脚本结构模板

> 每种视频类型有不同的最佳结构。基于对热门短视频的分析总结。

---

## 1. Vlog 类型

**适用场景**：日常记录、旅行日记、摩旅 vlog

```
【Hook — 0-3秒】
  画面：最具视觉冲击力的一帧（美景/意外/情绪高潮）
  台词：一句话概括今天最精彩的部分或制造悬念
  示例："今天骑车差点翻下山……但看到了这辈子最美的日落"

【开场 — 3-8秒】
  画面：出发准备/路上状态
  台词：交代背景（去哪、为什么去、今天的目标）
  字幕：日期 + 路线

【主线1 — 8-25秒】
  画面：路上实拍、骑行 POV
  台词：叙述过程中的关键事件/感受
  重点：选1个核心事件展开

【转折 — 25-35秒】
  画面：意外状况/发现/情绪变化
  台词："但是..." / "没想到..." / "结果..."
  作用：制造波澜，避免平铺直叙

【主线2 — 35-50秒】
  画面：转折后的新发现/解决过程
  台词：分享转折后的体验

【收尾 — 50-55秒】
  画面：目的地到达/总结画面
  台词：总结感受，呼应开头

【CTA — 55-60秒】
  画面：下期预告画面 或 最美一帧
  台词："关注我，下期xxx" / "你觉得xxx？评论区见"
```

**BGM 建议**：
- 轻松日常：Lo-fi / 轻快吉他
- 震撼风光：Epic / 电影配乐
- 温馨治愈：钢琴 / 民谣

## 2. 教程类型

**适用场景**：教学、技巧分享、知识科普

```
【Hook — 0-3秒】
  画面：最终成果/效果展示
  台词：用一句话说清楚看完这个视频能学到什么
  示例："3分钟教你摩托车换机油，省下 200 块"

【痛点 — 3-10秒】
  画面：反面案例/常见错误/问题描述
  台词：为什么需要学这个？不学有什么后果？

【步骤1 — 10-20秒】
  画面：准备材料/工具展示
  台词：需要什么，注意什么
  字幕：步骤编号 + 关键词

【步骤2 — 20-35秒】
  画面：核心操作步骤
  台词：详细讲解（口语化，像教朋友）
  字幕：关键参数/数据

【步骤3 — 35-45秒】
  画面：完成效果对比/注意事项
  台词：做完后怎么验证/常见坑

【总结 — 45-55秒】
  画面：前后对比/要点回顾
  台词：3句话总结核心要点
  字幕：要点列表

【CTA — 55-60秒】
  台词："收藏这个视频下次用" / "有什么问题评论区问我"
```

**BGM 建议**：
- 轻快节奏感（不要太大声，不盖过讲解）

---

## 3. 探店/测评类型

**适用场景**：餐厅探店、产品体验、摩托车装备测评

```
【Hook — 0-3秒】
  画面：最诱人的菜品特写/产品最酷的角度
  台词：制造好奇或争议
  示例："北京最贵的摩托车头盔，值不值 3000 块？"

【背景 — 3-10秒】
  画面：到达/开箱/外观
  台词：为什么选这个？花了多少钱？什么来头？

【体验1 — 10-25秒】
  画面：使用/品尝/试戴 过程
  台词：第一印象、直观感受
  重点：说真话，有好有坏才真实

【体验2 — 25-40秒】
  画面：细节特写/使用场景
  台词：深入体验，对比同类产品
  重点：给出具体数据/参数

【总结评分 — 40-50秒】
  画面：总结画面
  台词：直接给评分（满分10分），列出优缺点
  字幕：评分卡片

【CTA — 50-60秒】
  台词："你们觉得值不值？评论区打分" / "想看测什么评论告诉我"
```

**BGM 建议**：
- 活泼轻快，带点节奏感

## 4. 摩旅骑行类型

**适用场景**：摩托车旅行、路线推荐、骑行体验

```
【Hook — 0-3秒】
  画面：骑行中最震撼的画面（山路/日出/弯道）
  台词：路线亮点/情绪触发
  示例："离北京 2 小时，这条山路骑过一次就上瘾"

【路线介绍 — 3-12秒】
  画面：地图标注/路线概览
  台词：起点终点、总里程、预计时长、难度等级
  字幕：路线关键信息卡片

【路段1 — 12-25秒】
  画面：第一段路的骑行实拍
  台词：路况描述、风景亮点
  重点：让没骑过的人也能感受到

【休息点/补给 — 25-35秒】
  画面：休息点/加油站/小店
  台词：补给建议、注意事项（加油站位置、手机信号）
  作用：实用信息，增加收藏价值

【路段2/高潮 — 35-48秒】
  画面：最美路段/最佳观景点
  台词：这段路为什么值得来
  重点：情绪到位，让观众想来

【实用信息 — 48-55秒】
  画面：装备/路况/天气
  台词：新手注意事项、最佳出行时间、装备建议
  字幕：Tips 列表

【CTA — 55-60秒】
  画面：返程或日落
  台词："骑过的朋友评论区聊聊" / "关注我，每周一条骑行路线"
```

**BGM 建议**：
- 自由/公路感：摇滚、公路民谣
- 壮阔风景：后摇、电影配乐

## 5. 种草推荐类型

**适用场景**：好物推荐、必买清单、装备分享

```
【Hook — 0-3秒】
  画面：全部产品展示/最有冲击力的单品
  台词：制造紧迫感或好奇心
  示例："骑了 3 年摩托，这 5 件装备后悔没早买"

【产品1 — 3-15秒】
  画面：产品特写+使用场景
  台词：是什么 + 为什么推荐 + 多少钱 + 哪里买
  字幕：产品名+价格

【产品2 — 15-25秒】
  画面：同上
  台词：同上格式
  重点：每个产品给一个"记忆点"

【产品3-N — 25-45秒】
  画面：同上（节奏加快）
  台词：同上格式（更精简）

【总结 — 45-55秒】
  画面：所有产品全家福
  台词：按预算推荐/按场景推荐
  字幕：价格汇总表

【CTA — 55-60秒】
  台词："最想买哪个？评论区告诉我" / "收藏慢慢看"
```

**BGM 建议**：
- 节奏明快，卡点感强

## 6. 挑战/实验类型

**适用场景**：各种挑战、测试、实验

```
【Hook — 0-3秒】
  画面：挑战最戏剧性的一刻（失败或成功的瞬间）
  台词：直接说出挑战内容+悬念
  示例："100 块钱能买到能骑的摩托车吗？"

【规则说明 — 3-10秒】
  画面：规则文字/道具展示
  台词：挑战规则是什么，成功/失败的标准

【过程1 — 10-25秒】
  画面：执行过程
  台词：实时解说+感受
  重点：展示真实反应，不要完美主义

【过程2/困难 — 25-40秒】
  画面：遇到的困难/意外
  台词：真实的挫败感/惊喜
  作用：真实感=观众粘性

【结果 — 40-50秒】
  画面：最终成果
  台词：挑战成功/失败 + 总花费/总耗时
  字幕：结果卡片

【CTA — 50-60秒】
  台词："你们觉得这个结果合理吗？" / "下期想看什么挑战？"
```

**BGM 建议**：
- 紧张/竞技感：快节奏电子乐

## 7. 知识科普类型

**适用场景**：冷知识、科普、行业揭秘

```
【Hook — 0-3秒】
  画面：与主题相关的视觉冲击画面
  台词：提出一个反直觉的问题或事实
  示例："90% 的摩托车手都戴错了头盔"

【悬念 — 3-10秒】
  画面：错误示范/常见误解
  台词：大多数人以为的是错的
  作用：制造知识落差

【知识核心 — 10-35秒】
  画面：图解/实拍/对比
  台词：讲清楚核心知识点
  字幕：关键数据/公式/结论
  重点：通俗化，用比喻

【延伸 — 35-48秒】
  画面：补充画面
  台词：额外的有趣信息或冷知识
  作用：增加信息密度

【总结 — 48-55秒】
  画面：要点回顾
  台词：一句话总结
  字幕：要点列表

【CTA — 55-60秒】
  台词："你知道这个吗？转发给骑摩托的朋友"
```

**BGM 建议**：
- 悬疑/探索感，中低音量

## 8. 通用模板（general）

**适用场景**：无法归入以上类型的通用短视频

```
【Hook — 0-3秒】
  画面：最具吸引力的一帧
  台词：一句话抓住注意力（提问/数据/反常/故事开头）

【正文分段 — 3-50秒】
  按逻辑分段，每段 8-15秒
  每段一个小节奏点（画面切换/音乐变化/信息密度变化）

【总结 — 50-55秒】
  画面：收束画面
  台词：一句话总结/金句

【CTA — 55-60秒】
  画面：下期预告/最美画面
  台词：明确的互动引导
```

## Hook 公式库

开头是短视频最关键的部分。以下是经过验证的高完播率 Hook 公式：

### 数据冲击型
```
"99% 的人不知道..."
"只需要 X 元/分钟/步..."
"这个方法让我省了 X 元"
```

### 反常识型
```
"千万别再 XXX 了"
"你以为的 XX 其实是错的"
"XX 的真相是..."
```

### 悬念型
```
"今天差点出事..."
"结果万万没想到..."
"骑到一半发现..."
```

### 共鸣型
```
"骑摩托的人一定懂..."
"每次 XX 都在想..."
"有没有人也这样？"
```

### 痛点型
```
"新手最容易犯的 X 个错误"
"XX 踩坑血泪史"
"花了 X 元买到的教训"
```

### 制造对比型
```
"X 元 vs X 元的 XX，差距有多大？"
"新手 vs 老手的区别"
"以为很丑，结果..."
```

## 时长控制指南

按语速约 **4字/秒** 计算：

| 目标时长 | 台词字数 | 画面段数 |
|---------|---------|---------|
| 15秒 | 30-45字 | 2-3段 |
| 30秒 | 70-100字 | 3-4段 |
| 60秒 | 150-200字 | 5-6段 |
| 90秒 | 230-300字 | 6-8段 |
| 3分钟 | 500-650字 | 8-12段 |

**关键**：宁可少说两句，不要超时。观众注意力按秒递减。
