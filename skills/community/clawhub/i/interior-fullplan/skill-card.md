## Description: <br>
Generates a Chinese-oriented interior design planning document with layout guidance, lighting, materials, budget estimates, design notes, and construction considerations from project inputs. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Interior designers, decorators, and agents supporting Chinese-language planning use this skill to turn basic project details into an editable early-stage design proposal. The generated plan is useful for client discussion, budgeting, and drafting a proposal package, but it still needs professional and local-code review before construction decisions. <br>

### Deployment Geography for Use: <br>
China <br>

## Known Risks and Mitigations: <br>
Risk: Project names are used in the generated Desktop filename, which can create filename-safety issues or overwrite an existing file. <br>
Mitigation: Use simple project names without slashes, '..', or absolute paths, and check for an existing Desktop file with the same name before running. <br>
Risk: Embedded design and construction norms may not satisfy local code, site conditions, or professional requirements. <br>
Mitigation: Treat the output as planning assistance and have a qualified professional review it before using it for compliance, procurement, or construction. <br>
