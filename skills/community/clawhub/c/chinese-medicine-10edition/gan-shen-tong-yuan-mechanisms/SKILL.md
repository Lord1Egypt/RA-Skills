# Gan Shen Tong Yuan Mechanisms

## When to Use
- 辨证肝肾精血两亏证
- 分析月经失调或生殖功能异常的病机
- 判断“水不涵木”型肝阳上亢
- 指导滋补肝肾、调和藏泄的治疗策略

## Execution Steps

1. **评估精血同源机制**：
   - 肾精可化生肝血，肝血充盛则肾有所藏
   - 精血互化障碍表现为：头晕目眩、耳聋耳鸣、视力减退、腰膝酸软
   - 别称“乙癸同源”（肝属乙木，肾属癸水）

2. **分析藏泄互用机制**：
   - 肝主疏泄（动），肾主封藏（静）
   - 共同调节月经来潮、排卵、泄精
   - 失调表现：
     - 女子：月经先期/后期、经量过多/闭经、不孕
     - 男子：遗精滑泄或阳强不泄

3. **判断阴阳互资互制机制**：
   - 肾阴滋养肝阴 → “水能涵木”
   - 肾阴不足 → 肝阴亏虚 → 肝阳上亢 → 眩晕、中风（“水不涵木”）
   - 肾阳资助肝阳 → 温煦肝脉
   - 肾阳虚 → 下焦虚寒 → 少腹冷痛、阳痿、宫寒不孕
   - 肝火过盛亦可下劫肾阴

4. **排除非适用情况**：
   - 急性外伤性出血
   - 非慢性虚损性疾病的初期阶段

5. **输出综合判断**：
   - 明确肝肾在精血、藏泄、阴阳三个维度的协同状态
   - 推荐滋补肝肾、调和阴阳、疏泄与封藏并调的治法
