## Use Case: <br>
Developers and agent authors use this skill to generate publishable OpenClaw skill directories with SKILL.md metadata, usage instructions, examples, README content, and optional references. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The generator can overwrite SKILL.md, README.md, or references/overview.md when pointed at an existing directory. <br>
Mitigation: Run it in a new or empty output directory and review generated files before publishing. <br>
