## 1. Search Intent Decoder

**Best for:** Writing content that actually ranks

### Match intent to content type

| Query Type | Intent | Content Format | Example |
|------------|--------|----------------|---------|
| "how to" | Informational | Step-by-step guide | "How to bake sourdough" |
| "what is" | Informational | Definition + context | "What is Padel tennis" |
| "brand + product" | Navigational | Product page + reviews | "Nike Air Max" |
| "buy", "price" | Transactional | Sales page with CTA | "Buy MacBook Pro" |
| "best", "vs", "review" | Commercial | Comparison + honest verdict | "Best CRM for startups" |

### How to check
1. Google your target keyword
2. Look at page 1 results — what format dominates?
3. Match that format or don't expect to rank

**Rule:** A transactional query with 10 comparison posts ranking? Write a comparison post, not a sales page.

## 2. Keyword Research Without Tools

**Best for:** Finding keywords when you have $0

### Free Sources

| Source | How to Use | What You Get |
|--------|------------|--------------|
| **Google autocomplete** | Type seed keyword, note completions | Real user queries |
| **People Also Ask** | Search keyword, expand PAA boxes | Question-format keywords |
| **Related searches** | Scroll to bottom of SERP | Long-tail variations |
| **Google Trends** | Compare keywords over time | Trend direction |
| **Search Console** | Check queries you already rank for | Your best opportunities |

### Process
```
Seed keyword → Autocomplete expansion → PAA questions → Related searches → Cluster into topic groups
```

### Priority Matrix

| Priority | Condition | Action |
|----------|-----------|--------|
| **P0** | You rank 5-15 | Optimize existing, easy wins |
| **P1** | High volume + low competition | Create new content |
| **P2** | Competitor gap | Write what's missing |
| **P3** | Brand terms | Own your name |

## 3. Content Structure for Rankings

**Best for:** Formatting content Google loves

### Requirements

- **First 100 words** answer the query directly (featured snippet optimization)
- **Table of contents** for long content (may appear as sitelinks)
- **FAQ section** at bottom (captures "People Also Ask")
- **Bulllet points** and **numbered lists** (preferred for snippets)
- **Short paragraphs** (2-3 sentences max, especially mobile)

### E-E-A-T Checklist

| Signal | Implementation |
|--------|----------------|
| **Experience** | First-hand case studies, data from your work |
| **Expertise** | Author bio with credentials |
| **Authoritativeness** | Cite authoritative sources with links |
| **Trustworthiness** | Contact info, about page, updated dates |

### What NOT to do
- Thin content (<300 words with no unique value)
- Duplicate content without canonical tags
- Changing publish date without updating content
- Writing for search engines instead of humans

## 4. Competitive Content Gap Analysis

**Best for:** Finding topics your competitors missed

### Template

| Factor | Competitor 1 | Competitor 2 | You |
|--------|--------------|--------------|-----|
| Word count | | | |
| Subtopics covered | | | |
| Media (images, video) | | | |
| Freshness (last updated) | | | |
| E-E-A-T signals | | | |

### Find gaps
1. What questions do they NOT answer?
2. What formats do they NOT use? (tools, calculators, videos)
3. What depth do they lack?

**Rule:** Don't beat them on THEIR terms. Find the gap and own it.
