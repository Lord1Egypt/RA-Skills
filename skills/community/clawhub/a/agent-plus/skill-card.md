## Description: <br>
Agent Plus provides templates, voice guidelines, adaptation rules, and personality frameworks for defining an agent's identity, voice, boundaries, and learning style. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent designers use this skill to shape agent identity, communication style, adaptation behavior, and permission boundaries. It is useful when creating or refining assistants that need consistent voice, personality, and escalation rules. <br>
