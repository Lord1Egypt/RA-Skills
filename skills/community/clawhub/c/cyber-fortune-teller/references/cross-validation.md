# 三系统交叉验证规则

> 本文档定义八字、面相、手相三大系统的交叉验证规则，确保AI分析结论的一致性和准确性。

---

## 一、交叉验证的理论基础

### 1.1 三学科的内在联系

```
八字命理 → 先天命数（时间维度）
面相学 → 先天禀赋（空间维度）
手相学 → 命运轨迹（动态维度）

三者关系：
┌─────────────────────────────────────┐
│  八字（先天命数）                      │
│  ├── 决定整体命局                      │
│  ├── 包含格局、用神、大运              │
│  └── 代表"命"                         │
│              ↓ 交叉验证                │
│  面相（先天禀赋）                      │
│  ├── 体现命局特征                      │
│  ├── 包含五官、骨相、神气              │
│  └── 代表"相"                         │
│              ↓ 交叉验证                │
│  手相（命运轨迹）                      │
│  ├── 反映命运变化                      │
│  ├── 包含主线、辅纹、气色              │
│  └── 代表"运"                         │
└─────────────────────────────────────┘
```

### 1.2 交叉验证的三原则

| 原则 | 说明 | 应用 |
|------|------|------|
| 一致性原则 | 三者结论应基本一致 | 命局旺则面相应饱满，手相应有力 |
| 互补性原则 | 三者相互补充验证 | 面相静态，手相动态，八字定量 |
| 优先级原则 | 八字为主，面相手相为辅 | 命局决定面相手相的吉凶程度 |

## 二、核心交叉验证规则

### 2.1 旺衰与形相的对应

**八字旺衰 → 面相形相**：

| 八字旺衰 | 面相特征 | 手相特征 |
|---------|---------|---------|
| 极旺 | 五官饱满、骨骼粗大、肌肉坚实 | 生命线粗深、掌丘丰隆 |
| 旺 | 五官端正、骨骼适中、肌肉有力 | 生命线清晰、掌丘有力 |
| 中和 | 五官均称、骨骼均衡、肌肉适度 | 生命线长度适中、掌丘平整 |
| 弱 | 五官偏瘦、骨骼细弱、肌肉单薄 | 生命线浅细、掌丘低平 |
| 极弱 | 五官瘦削、骨骼纤细、肌肉消瘦 | 生命线短浅、掌丘凹陷 |

**验证规则**：

```
【规则2.1.1】旺衰与形相应一致
IF 八字旺衰程度 = 旺
THEN 面相应具备：五官饱满、骨骼粗大
AND 手相应具备：生命线粗深、掌丘丰隆
ELSE 存在不一致，需要进一步分析

【规则2.1.2】旺衰决定形相的吉凶程度
IF 八字格局为贵格 AND 八字旺衰为旺
THEN 面相应达到：五官端正以上
AND 手相应达到：主线清晰有力

【规则2.1.3】旺衰与形相的例外情况
IF 八字旺衰为中和偏旺
THEN 面相可能出现：清秀型而非饱满型
AND 手相可能出现：智慧线发达而非生命线粗壮
```

### 2.2 格局与五官的对应

**八字格局 → 面相五官**：

| 八字格局 | 对应面相特征 |
|---------|-------------|
| 官杀格 | 眉清目正、眼神有力、印堂饱满 |
| 财格 | 鼻丰圆润、颧骨对称、地阁方圆 |
| 印格 | 额头丰满、耳白红润、眉清目秀 |
| 食神格 | 眼神温和、口型端正、唇红齿白 |
| 伤官格 | 眼神灵动、眉形多变、嘴巴灵活 |
| 比劫格 | 骨骼粗大、肌肉结实、意志坚强 |

**验证规则**：

```
【规则2.2.1】格局与五官应匹配
IF 八字格局 = 官杀格
THEN 面相应具备：眼神有力、眉毛清正
AND 命宫（印堂）应：平满光泽

IF 八字格局 = 财格
THEN 面相应具备：鼻丰圆润、颧骨对称
AND 财帛宫应：鼻部饱满无破

IF 八字格局 = 印格
THEN 面相应具备：额头丰满、耳白红润
AND 官禄宫应：额部丰隆无陷
```

### 2.3 用神与宫位的对应

**八字用神 → 面相宫位**：

| 八字用神 | 对应面相宫位 | 说明 |
|---------|-------------|------|
| 官杀 | 官禄宫、命宫 | 官运事业 |
| 财星 | 财帛宫、田宅宫 | 财运房产 |
| 印星 | 官禄宫、福德宫 | 学识声誉 |
| 食伤 | 迁移宫、相貌宫 | 才华迁移 |
| 比劫 | 兄弟宫、奴仆宫 | 人际关系 |

**验证规则**：

```
【规则2.3.1】用神得力对应宫位吉相
IF 八字用神为官杀星 AND 用神得力
THEN 官禄宫应：丰满明亮
AND 命宫应：平满光泽

IF 八字用神为财星 AND 用神得力
THEN 财帛宫应：鼻丰圆润
AND 田宅宫应：眼睑饱满

【规则2.3.2】用神受损对应宫位凶相
IF 八字用神为官杀星 AND 用神受损
THEN 官禄宫可能出现：凹陷纹冲
AND 命宫可能出现：狭窄暗淡

【规则2.3.3】用神与宫位的特殊对应
IF 八字调候用神为丙火
THEN 面相应具备：面色红润
AND 手相应具备：掌色红润
```

### 2.4 大运与流年的验证

**八字大运 → 面相气色**：

| 大运状态 | 面相气色 | 手相变化 |
|---------|---------|---------|
| 运势上升 | 气色明润、面色红润 | 掌色变好、纹路清晰 |
| 运势平稳 | 气色平和、无明显变化 | 掌色稳定、主线如常 |
| 运势下降 | 气色暗沉、面色晦暗 | 掌色暗淡、可能出现新纹 |
| 重大转折 | 气色突变、颜色异常 | 出现新纹或断裂 |

**验证规则**：

```
【规则2.4.1】大运吉凶应与气色对应
IF 当前大运为吉运
THEN 面相应具备：气色明润
AND 手相应具备：掌色红润

IF 当前大运为凶运
THEN 面相应出现：气色暗沉
AND 手相应出现：掌色晦暗

【规则2.4.2】流年吉凶应与部位对应
IF 流年天干为忌神
THEN 对应部位可能出现：气色变化
例：壬水为忌 → 可能出现：眼下青色

IF 流年地支冲命局用神
THEN 可能出现：相应部位气色突变
```

### 2.5 六亲与宫位的验证

**八字六亲星 → 面相手相对应**：

| 六亲 | 八字对应 | 面相对应 | 手相对应 |
|------|---------|---------|---------|
| 父亲 | 偏财星 | 父亲宫（额角） | 生命线内侧 |
| 母亲 | 正印星 | 母亲宫（耳） | 感情线起点 |
| 兄弟 | 比肩星 | 兄弟宫（眉毛） | 无名指下方 |
| 配偶 | 正财星（男）/正官星（女） | 妻妾宫（眼尾） | 感情线 |
| 子女 | 食伤星 | 子女宫（眼睑） | 子女线 |

**验证规则**：

```
【规则2.5.1】六亲星旺衰应与宫位对应
IF 八字偏财星旺而得用
THEN 父亲宫应：丰满无破
AND 与父亲缘分深

IF 八字偏财星弱而受克
THEN 父亲宫可能出现：凹陷纹冲
AND 与父亲缘分浅

【规则2.5.2】六亲星与手相的对应
IF 八字食神星旺
THEN 子女宫应：眼睑饱满
AND 子女线应：清晰多条
```

## 四、冲突处理规则

### 4.1 冲突类型定义

| 冲突类型 | 定义 | 示例 |
|---------|------|------|
| 完全冲突 | 三系统结论完全相反 | 八字吉但面相凶 |
| 部分冲突 | 两系统一致，一系统不同 | 八字吉面相吉手相凶 |
| 轻微差异 | 三系统结论方向一致，程度不同 | 八字大吉面相中吉手相小吉 |

### 4.2 冲突处理优先级

```
冲突处理优先级：

1. 八字为主原则
   当三系统冲突时，以八字结论为主

2. 稳定性原则
   先天特征（面相）优先于变化特征（手相）

3. 主流原则
   两系统一致时，以多数意见为准

4. 程度修正原则
   当结论一致但程度不同时，取中间值
```

### 4.3 冲突处理流程

```
冲突处理流程：

STEP 1: 检查三系统结论
   输出八字结论：A
   输出面相结论：B
   输出手相结论：C

STEP 2: 判断冲突类型
   IF A = B = C
   THEN 结论一致，无冲突 → 直接输出

   IF A = B ≠ C
   THEN 部分冲突（手相不同）
   → 以A、B为准，C作为参考

   IF A = C ≠ B
   THEN 部分冲突（面相不同）
   → 以A、C为准，B作为参考

   IF B = C ≠ A
   THEN 部分冲突（八字不同）
   → 以B、C为准，A作为主要参考

   IF A ≠ B ≠ C
   THEN 完全冲突
   → 进入STEP 3

STEP 3: 完全冲突处理
   提取八字旺衰、格局、用神
   提取面相神形、宫位、五官
   提取手相主线、掌丘、气色

   综合评估：
   IF 八字旺衰与面相手相的形相特征一致
   THEN 八字结论为主

   IF 面相神形与八字格局一致
   THEN 面相特征作为主要参考

   IF 手相主线与八字用神一致
   THEN 手相作为动态验证

STEP 4: 输出综合结论
   明确标注各系统的判断
   解释冲突原因
   给出最终结论
```

### 四步分析输出框架

| 步骤 | 内容 | 原则 |
|------|------|------|
| 第一步 | 列出真实卡点（1-3个） | 不超过3个，要具体 |
| 第二步 | 针对每个卡点给具体建议 | 可执行、可验证 |
| 第三步 | 解释为什么会有效 | 结合命理配置 |
| 第四步 | 给出验证标准 | 如何判断起效了 |

### 核心铁律

```
不说不痛不痒的话。
不回避冲突性结论。
不堆砌吉祥话。
不迎合用户。
只说真实有用的分析，哪怕难听。
光说问题不说解法是耍流氓。
```
