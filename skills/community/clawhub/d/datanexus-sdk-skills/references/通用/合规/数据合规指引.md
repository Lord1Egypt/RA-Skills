# 数据合规指引

## 概述

接入 DataNexus SDK 前，开发者必须确保应用符合数据合规要求，包括隐私政策声明、用户授权获取、个人信息保护等。**不同端（APP / 小游戏 / 小程序）的合规要求存在差异，请务必阅读对应端的官方合规文档（见第五部分）。**

## 一、SDK 信息收集说明

### 1.1 必选信息（SDK 功能运行所需）

SDK 不强制获取可选参数，不提供可选参数时基础功能仍可正常运行。

#### Android / iOS / 鸿蒙端

| 信息类型 | 具体字段 | 用途 |
|---------|---------|------|
| 设备信息 | 设备型号、设备启动时间、系统语言、连接的 Wi-Fi | 广告投放、广告效果优化 |
| 设备标识符 | OAID、Android ID（Android/鸿蒙）；IDFA、IDFV（iOS）；AAID（鸿蒙） | 广告归因 |
| 应用信息 | 开发者应用包名、使用状态、前台状态 | 广告投放 |
| 使用数据 | 广告曝光、应用启动退出行为（Android/鸿蒙） | 广告效果衡量 |

#### 小游戏端

| 信息类型 | 具体字段 | 用途 |
|---------|---------|------|
| 设备信息 | 设备型号、操作系统、分辨率、网络状态 | 广告投放、广告效果优化 |
| 应用环境 | 微信版本、场景值（scene） | 广告效果优化 |

#### 小程序端

| 信息类型 | 具体字段 | 用途 |
|---------|---------|------|
| 设备信息 | 设备信息、去标识化 ID | 广告效果衡量、归因分析、效果优化 |

### 1.2 可选信息（开发者选择是否提供）

| 适用端 | 信息类型 | 说明 |
|--------|---------|------|
| Android/iOS/鸿蒙 | 系统文件状态信息 | 开发者可选择是否授权 SDK 收集 |
| Android/iOS/鸿蒙 | 设备制造商、品牌信息 | 仅 Android 端可选 |
| Android/iOS/鸿蒙 | 用户行为数据 | 启动、付费、咨询等 |
| 小游戏 | 用户行为数据 | 注册、创建角色、完成新手教程、付费、升级、浏览商城、浏览活动、退出等 |
| 小游戏 | 用户 ID 与设备标识 | 开发者授权后收集 |
| 小程序 | 用户行为数据 | 由开发者根据实际授权情况提供 |

> 用途：广告投放、广告归因、广告效果优化。

## 二、开发者合规义务

### 2.1 用户授权流程

1. **展示隐私政策**：在用户首次使用应用时，以显著方式（如登录页/首次启动弹窗）展示隐私政策
2. **获取明确同意**：用户必须主动点击同意，不可默认勾选
3. **延迟初始化**：仅在用户同意隐私政策后调用 SDK 的 `init` 方法
4. **提供撤回途径**：为用户提供撤回授权、删除数据、注销账号的渠道

> ⚠️ **严禁**在用户未同意隐私政策前初始化 SDK 或收集任何个人信息。

### 2.2 隐私政策披露要求

开发者**必须**在应用隐私政策中披露以下信息：

- SDK 提供方名称：**深圳市腾讯计算机系统有限公司**
- SDK 产品名称：腾讯广告转化 SDK / 腾讯广告小游戏 SDK / 腾讯广告小程序 SDK
- 收集的个人信息类型、用途、处理方式
- SDK 隐私政策链接（见第五部分各端链接）

### 2.3 小程序/小游戏额外要求

- **登记第三方 SDK**：在微信公众平台官方渠道登记 SDK 提供方信息（深圳市腾讯计算机系统有限公司）、使用场景及处理的数据类型
- **补充协议**：添加关于 SDK 数据处理的补充说明

### 2.4 未成年人保护

- 涉及 14 周岁以下用户时，**必须事先取得其监护人的同意**
- 如发现未经授权收集了儿童个人信息，SDK 提供方将尽快删除该儿童的个人信息

### 2.5 禁止行为

- ❌ 未经用户同意收集个人信息
- ❌ 用户拒绝授权后反复弹窗请求无关权限
- ❌ 在未使用相关功能时申请敏感权限
- ❌ 超出功能必要范围过度收集信息
- ❌ SDK 无正当使用场景时自动启动

### 2.6 保持 SDK 版本更新

及时升级至满足监管新规的最新 SDK 版本，避免因版本过旧导致合规风险或功能异常。

## 三、权限申请（按端区分）

> 原则：**最小必要**，仅申请 SDK 功能实际需要的权限。小程序/小游戏端无系统权限概念，在此不适用。

### Android 端

| 权限 | 用途 | 必要性 |
|------|------|--------|
| `INTERNET` | SDK 数据上报 | **必要** |
| `ACCESS_NETWORK_STATE` | 判断网络环境，提升归因成功率 | 可选（建议开启） |
| `READ_PHONE_STATE` | 获取设备标识 | 可选 |
| `WRITE_EXTERNAL_STORAGE` | 归因优化 | 可选 |

### iOS 端

| 权限 | 用途 | 必要性 |
|------|------|--------|
| 网络访问 | SDK 数据上报 | **必要**（默认具备） |
| `NSUserTrackingUsageDescription`（ATT） | 获取 IDFA，用于广告归因 | 可选（建议配置） |

### 鸿蒙端

| 权限 | 用途 | 必要性 |
|------|------|--------|
| `ohos.permission.INTERNET` | SDK 数据上报 | **必要** |
| `ohos.permission.GET_NETWORK_INFO` | 判断网络环境 | **必要** |
| `ohos.permission.APP_TRACKING_CONSENT` | 获取 OAID，用于广告归因 | 可选（建议配置） |

## 四、数据存储与用户权利

### 4.1 数据存储

- 个人信息存储于**中国大陆境内**
- 存储期限遵循**最小必要原则**，仅在实现目的所需的最短时间内保留
- 如涉及跨境传输，需额外获取用户同意并完成合规评估

### 4.2 用户权利

用户有权请求：访问、复制、更正、删除个人信息，撤回授权同意，注销账号。

- 开发者应在自身平台提供用户行使权利的渠道
- SDK 提供方联系方式：**Dataprivacy@tencent.com**
- 响应时限：15 个工作日内

### 4.3 参考法规与标准

1. [《个人信息保护法》](https://www.gov.cn/xinwen/2021-08/20/content_5632486.htm)
2. [《网络安全标准实践指南—移动互联网应用程序（App）使用软件开发工具包（SDK）安全指引》](https://www.tc260.org.cn/front/postDetail.html?id=20201126161240)
3. [《网络安全标准实践指南—移动互联网应用程序（App）系统权限申请使用指南》](https://www.tc260.org.cn/front/postDetail.html?id=20200918163359)

> SDK 提供方保留对开发者合规实践进行审计的权利，包括审查隐私政策和应用功能，如发现不合规可能拒绝提供服务。
