## Description: <br>
Project Engineer skill that helps engineering agents analyze codebases, produce implementation plans and specs, review developer branches, handle escalations, and run heartbeat queue checks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[encryptshawn](https://clawhub.ai/user/encryptshawn) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineering agents use this skill to coordinate software project delivery, including code audits, technical assessments, implementation planning, task manifests, branch reviews, and escalation handling. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may update local planning and audit files in a shared project workspace. <br>
Mitigation: Confirm the team is comfortable with those workspace updates and review diffs before committing changes. <br>
