## 💡 使用技巧

### 1. 不知道用谁？→ 说场景
```
"我要做个重大决策" → 自动选芒格 + 巴菲特
"我产品做不出来" → 自动选 Jobs + 马斯克
```

### 2. 想全面分析？→ 说完整
```
"启动完整会诊" → 15 位顾问全上
```

### 3. 想不起来名字？→ 问列表
```
"顾问都有谁？" → 显示完整名单
"显示顾问速查表" → 显示这个页面
```

---

## 🚀 最简用法

**就记 3 个：**
1. "顾问会诊 XXX" → 自动分析
2. "查看顾问名单" → 显示列表  
3. "问问 [任何名字]" → 单顾问咨询

**其他都不用记！**
