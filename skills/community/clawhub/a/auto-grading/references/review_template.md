# 论文/毕业设计评审模板 v2.0

> 支持：期刊论文评审 / 本科毕业设计评审 / 多文档交叉审查 / PPT 答辩评审
> 串联方式：doc-browser 读取全部文档 → academic-workflow 多维度评分输出
> 
> ⚠️ **安全护栏 v2.1**：每条问题必须附证据原文 + 置信度标记，输出前执行三问自查（证据可溯源/排除提取噪声/排除文件混淆）

---

## 一、毕业设计评审维度（多文档综合）

### 单个文档评分维度

| 维度 | 满分 | 适用文档类型 | 说明 |
|------|:----:|------------|------|
| 内容完整性 | 40 | 创作报告、文献综述、开题报告 | 结构要素齐全（摘要/目录/绪论/正文/结论/致谢/参考文献）、各章节内容充实度 |
| 格式规范性 | 20 | 全部文档 | 字体字号行距统一、页眉页脚页码、目录与正文对应、图表编号连续、参考文献对标 GB/T 7714、模板残留/域代码清理 |
| 逻辑一致性 | 20 | 创作报告、开题报告、文献综述 | 章节之间逻辑递进、论证链条完整、概念定义清晰无矛盾 |
| 论证合理性 | 20 | 创作报告、文献综述 | 选题意义明确、文献有批判性分析而非简单罗列、方法可行、结论基于实际工作 |
| **单文档总分** | **100** | | |

### 多文档交叉审查维度

| 维度 | 满分 | 检查内容 |
|------|:----:|---------|
| 要素一致性 | 50 | 游戏名称/学生姓名/学号/指导教师在所有文档中完全一致、日期逻辑正确（开题<中期<答辩<提交）、审批栏完整 |
| 引用一致性 | 30 | 创作报告、文献综述、任务书三套参考文献重叠度合理、正文引用的文献均在参考文献列表中存在且反之亦然 |
| 分工一致性 | 20 | 小组分工表与创作报告中个人角色一致、申报表课题方向与分工表匹配 |
| **交叉审查总分** | **100** | |

### PPT 答辩评审维度

| 维度 | 满分 | 说明 |
|------|:----:|------|
| 基本信息完整 | 20 | 题目/姓名/学号/指导教师/学院/日期齐全 |
| 内容结构 | 30 | 选题背景→设计思路→过程展示→成果呈现→总结展望，逻辑清晰 |
| 视觉美感 | 25 | 版式统一、配色协调、图文比例合理、无文字堆砌 |
| 观点表达 | 25 | 核心创新点突出、设计过程有说服力、成果有实物/视频支撑 |
| **PPT 总分** | **100** | |

## 二、多文档评审工作流

### 步骤 1：doc-browser 批量读取
```bash
# 对每个学生的存档目录，读取所有文件：
# .doc/.docx → 提取纯文本
# .pdf（查重报告）→ 提取重复率数值
# .pptx → 提取幻灯片文本+统计图片数量
```

### 步骤 2：单文档独立评分
对每份文档（创作报告、文献综述、开题报告、外文翻译、申报表、任务书、进程表），按上表维度独立打分，输出评分表格。

### 步骤 3：跨文档交叉审查
- 抽取所有文档中的：游戏名称、学生姓名、学号、日期、参考文献列表
- 逐项比对一致性，标注矛盾项

### 步骤 4：PPT 评审
- 提取 PPT 文本内容
- 评分基本信息/内容结构/视觉美感/观点表达

### 步骤 5：输出汇总报告
```markdown
## 学生姓名 — 毕业设计终审报告

### 单文档评分
| 文档 | 内容完整性/40 | 格式规范性/20 | 逻辑一致性/20 | 论证合理性/20 | 总分/100 |

| 创作报告 | | | | | |

| 开题报告 | | | | | |
```

## 三、格式检查清单

- [ ] 参考文献全部对标 GB/T 7714-2015（作者名大小写、期号不补零、[EB/OL]含URL和日期、[C]//格式）
- [ ] 图表编号连续且章节对应（图1-1在图1-2之前，第2章图从图2-1开始）
- [ ] 正文"如图X-X所示"引用覆盖所有图注
- [ ] 模板残留清零（SHAPE \* MERGEFORMAT、TOC/HYPERLINK/PAGEREF/REF域代码、占位符、格式说明文字）
- [ ] 摘要/关键词与正文内容匹配
- [ ] 页眉页脚页码正确
- [ ] 英文标题与中文标题一致
- [ ] 诚信承诺书已签名
- [ ] 审批栏已签章

## 四、参考文献格式速查（GB/T 7714-2015）

| 类型 | 格式模板 |
|------|---------|
| 期刊 [J] | `作者. 标题[J]. 期刊名, 年, 卷(期): 页码.` 期号不补零：`(1)` 非 `(01)` |
| 专著 [M] | `作者. 书名[M]. 出版地: 出版社, 年.` |
| 会议 [C] | `作者. 标题[C]//编者. 论文集名. 出版地: 出版社, 年: 页码.` |
| 学位论文 [D] | `作者. 标题[D]. 出版地: 学校, 年.` 分隔符用 `:` |
| 网络 [EB/OL] | `作者. 标题[EB/OL]. (发布日期)[引用日期]. URL.` |
| 报告 [R] | `机构/作者. 标题[R]. 出版地: 出版社, 年.` |
| 其他 [Z] | `作者. 标题[Z]. 出版地: 出版者, 年.` |

> 英文作者：姓全大写，名缩写。如 `STOKES P` 非 `Stokes, P.`
> 英文标题：建议统一为 sentence case（仅首词和专有名词大写）
