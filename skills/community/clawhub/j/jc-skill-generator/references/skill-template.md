# Skill 生成模板

这是一个参考模板，用于生成新的 SKILL.md 文件。不要照抄，只取需要的部分。

```markdown
---
name: skill-name
description: "Short trigger phrase describing what this skill does."
---

# Skill Name

一句话说明适用场景。

## 前置条件

- 需要安装的依赖
- 需要配置的环境变量
- 需要的权限/Key

## 工作流程

1. 第一步做什么
2. 第二步做什么
3. ...

## 关键命令

```bash
# 核心命令1
xxx

# 核心命令2
yyy
```

## 输出

- 会产生什么文件
- 在哪里

## 注意事项 / 踩坑记录

- 容易出错的地方
- 平台差异（Windows vs macOS）
- 兼容性问题
