# Bland-Altman 图

## 概述

Bland-Altman图用于评估两种测量方法的一致性。X轴=两方法均值，Y轴=两方法差值。

## 何时使用

| 条件 | 说明 |
|------|------|
| 场景 | 比较两种测量方法的一致性 |
| 关键 | 不是检验相关性，是检验可互换性 |

## R 代码

```r
library(blandr)
blandr.draw(data$method1, data$method2)
```

## 解读

- 中间虚线=偏倚(Bias)→0=无系统偏差
- 上下虚线=95%一致限(LoA)
- 95%的点在LoA内且LoA在临床/实用上可接受 → 两个方法可互换
- 漏斗形→偏倚随测量值变化

## 关键参数

| 参数 | 作用 |
|------|------|
| `method1`,`method2` | 两种测量方法的数值向量 |
| `loa` | 一致限宽度(默认1.96) |
