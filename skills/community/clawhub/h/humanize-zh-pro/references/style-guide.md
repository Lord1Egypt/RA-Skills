# 中文去AI味 — 深度风格指南

## AI味的本质

AI写作的"味"来自三个根源：
1. **信息均匀分布** — 每个段落信息量差不多，没有起伏
2. **语言过度规范** — 语法完美、连接词标准、缺乏个性
3. **情感真空** — 客观中立，没有情绪波动和个人立场

## 去味金字塔

```
    ┌──────────────┐
    │  注入灵魂      │  ← 个人故事、独特观点、情感
    ├──────────────┤
    │  打破结构      │  ← 长短句、不规则段落、打断
    ├──────────────┤
    │  换词换调      │  ← 替换AI词库、加入口语
    ├──────────────┤
    │  保留信息      │  ← 核心内容不变
    └──────────────┘
```

## 7 个去味公式

### 公式1: 拆序列
输入: "第一...第二...第三..."
输出: "先说... 另外啊... 还有个事..."

### 公式2: 加冗余
输入: "这个方案效率很高"
输出: "这个方案吧，怎么说呢，效率是真的高，高到离谱那种"

### 公式3: 插故事
输入: "数据表明该产品销量增长了30%"
输出: "说起来你可能不信，这产品卖得比想象中好太多了。我上次去店里，店员说都断货了"

### 公式4: 露情绪
输入: "该结果令人满意"
输出: "看到这个结果，我自己都惊了，没想到能这么好"

### 公式5: 主动态
输入: "该决定被管理层做出"
输出: "管理层拍板定了这个"

### 公式6: 断句
输入: "这是一款功能强大且易于使用的产品"
输出: "这款产品。好用。真的。"

### 公式7: 口语连
输入: "此外，该产品还具有防水功能"
输出: "还有啊，它还能防水，下雨天也不怕"

## 平台差异速查

| 平台 | 字数 | emoji | 语气 | 段落 |
|------|------|-------|------|------|
| 知乎 | 500-3000 | 少量 | 理性+温度 | 中段 |
| 小红书 | 200-800 | 大量 | 活泼+亲密 | 短段 |
| 公众号 | 800-5000 | 适量 | 叙事+温暖 | 中长段 |
| 朋友圈 | 50-200 | 适量 | 随性 | 极短 |
| 博客 | 任意 | 少量 | 个人 | 自由 |

## 常见AI词替换表

| AI味 | 人味 |
|------|------|
| 此外 | 还有啊、对了、另外说一句 |
| 然而 | 但问题是、可实际上、不过吧 |
| 因此 | 所以啊、这就导致、于是乎 |
| 从而 | 就（直接省略） |
| 进而 | 接着、然后 |
| 综上所述 | 说白了、总结一下、一句话 |
| 众所周知 | 大家都知道、懂的都懂 |
| 在当今社会 | 现在这年头、如今 |
| 随着...的发展 | （直接省略，从具体现象说起） |
| 具有重要意义 | 挺重要的、很关键 |
| 引发了广泛关注 | 大家都在讨论、传疯了 |
| 希望本文对您有所帮助 | 希望能帮到你、有问题随时问 |
| 期待您的反馈 | 来评论区聊聊、等着听你说 |
