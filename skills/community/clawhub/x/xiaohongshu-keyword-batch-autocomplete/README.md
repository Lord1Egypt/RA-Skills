# 小红书关键词联想词批量采集工具

一键批量获取小红书搜索框**下拉推荐联想词**，支持多轮次深度联想、自动去重，用于选题策划、SEO 优化、种草关键词布局。

## 功能特点

- ✅ 批量关键词导入（支持文件 / 命令行）
- ✅ 多轮次深度联想（联想词再联想，最多 3 层）
- ✅ 自动去重 + 热度权重排序
- ✅ TXT 格式导出（内置支持）
- ✅ Excel 格式导出（可选，需 pandas）
- ✅ 多 Sheet  Excel 报告（详细数据 / 汇总 / 统计）
- ✅ 引用计数统计（某个词被多个来源推荐的次数）

## 安装依赖

```bash
# 基础依赖（仅 TXT 导出）
pip3 install requests

# 完整依赖（TXT + Excel 导出）
pip3 install requests pandas openpyxl
```

## 使用方法

### 单个关键词采集

```bash
python3 batch_keyword.py --keyword "夏季穿搭"
```

### 批量关键词采集

```bash
# 创建关键词文件，每行一个关键词（# 开头为注释）
echo -e "夏季穿搭\n护肤\n美妆\n# 这是注释" > keywords.txt

# 批量采集
python3 batch_keyword.py --file keywords.txt
```

### 深度联想采集（推荐！）

```bash
# 2 层深度（联想词的联想词）
python3 batch_keyword.py --keyword "护肤" --depth 2

# 3 层深度（深度挖掘长尾词）
python3 batch_keyword.py --keyword "美食" --depth 3
```

### 导出 Excel 格式

```bash
python3 batch_keyword.py --keyword "旅游" --format excel
```

### 完整参数示例

```bash
# 批量 + 深度 3 + Excel 导出 + 自定义延迟
python3 batch_keyword.py \
  --file keywords.txt \
  --depth 3 \
  --format excel \
  --delay 0.3 \
  --output ./result
```

### 关闭自动去重

```bash
python3 batch_keyword.py --keyword "健身" --no-dedup
```

## Excel 报告说明

导出的 Excel 文件包含 3 个 Sheet：

| Sheet 名称 | 说明 |
|-----------|------|
| **详细数据** | 所有采集到的原始数据，包含主关键词、联想词、热度、深度、来源等 |
| **汇总（去重）** | 去重后的唯一词条，按热度排序，适合直接使用 |
| **统计信息** | 采集统计：关键词数量、深度、去重率等 |

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--keyword` | 单个关键词 | - |
| `--file` | 批量关键词文件（每行一个，# 开头为注释） | - |
| `--depth` | 联想深度（1-3），数值越大采集的长尾词越多 | `1` |
| `--format` | 输出格式：`txt` / `excel` | `txt` |
| `--output` | 输出目录 | `./output` |
| `--delay` | 请求延迟（秒），建议 0.3-1.0 | `0.5` |
| `--no-dedup` | 关闭自动去重，保留所有重复词条 | 关闭 |

## 真实 API 配置（开发者）

当前版本使用模拟数据进行演示。要对接真实的小红书 API，请：

1. 使用浏览器抓包，访问小红书搜索页面
2. 查找 `/api/sns/v1/search/recommend` 或类似端点
3. 更新 `batch_keyword.py` 中的 `_generate_mock_suggestions` 方法
4. 可能需要配置 cookies 或 token 进行认证

## 注意事项

- 建议设置合理的请求延迟（`--delay` 参数），避免请求过于频繁
- 深度 3 可能会产生大量词条（数百个），建议先从深度 1-2 开始
- 请遵守小红书的使用条款和 robots 协议
- 本工具仅供学习和研究使用
