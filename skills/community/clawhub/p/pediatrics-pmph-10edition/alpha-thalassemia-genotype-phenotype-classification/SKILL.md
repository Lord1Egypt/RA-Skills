# α地中海贫血基因剂量效应与临床分型

## 适用对象
- α地中海贫血疑似患者
- 产前筛查胎儿
- 家族成员

## 前提条件
- 已知人类有4个α珠蛋白基因（位于16p13.3，每条染色体2个）
- 存在α珠蛋白基因缺失或点突变

## 禁忌范围
- 不适用于β地中海贫血或其他血红蛋白病

## 执行流程

1. **统计功能性α珠蛋白基因数量**（正常为4个）：
   - **缺失1个基因**（α⁺杂合子）→ **静止型α地贫**
     - α链合成略减，无临床症状
     - 出生时脐血Hb Bart 0.01～0.02，3个月后消失
     - 易漏诊
   - **缺失2个基因**（α⁺纯合子 或 α⁰杂合子）→ **轻型α地贫**
     - α链合成减少但仍有相当量
     - 无症状，红细胞轻度形态异常
     - 脐血Hb Bart 0.034～0.140，6个月后消失
   - **缺失3个基因**（α⁰/α⁺双重杂合子）→ **中间型（HbH病）**
     - α链极少，多余β链形成HbH（β₄）
     - HbH对氧亲和力高且不稳定，易形成包涵体
     - 临床表现：贫血、肝脾大、黄疸、特殊面容
     - 可被氧化药物诱发溶血危象
     - HbH含量0.024～0.44，包涵体试验阳性
   - **缺失4个基因**（α⁰纯合子）→ **重型（Hb Bart胎儿水肿综合征）**
     - 无α链，γ链形成Hb Bart（γ₄）
     - Hb Bart氧亲和力极高 → 严重组织缺氧
     - 胎儿期即出现水肿、腹腔/胸腔积液、巨大胎盘
     - 多于30～40周流产、死胎或出生后半小时内死亡
     - 血红蛋白几乎全为Hb Bart，无HbA、HbA₂、HbF

2. **输出分型结果**：包括临床类型、关键实验室指标（如Hb Bart、HbH水平）及典型临床特征。

## 触发条件
当患者或胎儿存在α珠蛋白基因缺失或缺陷时，立即应用本分型规则。
