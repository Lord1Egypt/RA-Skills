# 妊娠期肝内胆汁淤积症（ICP）的诊断与分度

## 使用时机
- 孕妇在妊娠中期或晚期出现无原发皮损的瘙痒（常始于手掌/脚掌，夜间加重，产后24–48小时缓解）
- 需结合实验室检查明确是否为ICP，并排除病毒性肝炎、胆道疾病、皮肤病等其他病因
- 已完成初步肝功能和病毒学筛查，需依据TBA结果进行诊断与分度

## 执行步骤

1. **确认临床表现**：核实瘙痒特征符合ICP典型表现，且无皮肤原发病变。

2. **获取实验室数据**：
   - 测定空腹或随机血清总胆汁酸（TBA）
   - 检查肝功能（重点关注ALT、AST，部分患者γ-GT和直接胆红素升高）

3. **应用诊断标准**：
   - 若 **空腹TBA ≥10 μmol/L** 或 **随机TBA ≥19 μmol/L**，且已排除其他肝病，则诊断为ICP。

4. **进行严重程度分度（基于TBA峰值）**：
   - **轻度**：空腹10 ≤ TBA <40 μmol/L 或 随机19 ≤ TBA <40 μmol/L
   - **重度**：40 ≤ TBA <100 μmol/L
   - **极重度**：TBA ≥100 μmol/L

5. **输出结论**：明确ICP诊断状态及分度，用于指导胎儿监护频率、药物治疗选择及分娩时机决策。

> **注意**：TBA ≥100 μmol/L 与胎死宫内风险显著相关，需高度警惕并加强监测。
