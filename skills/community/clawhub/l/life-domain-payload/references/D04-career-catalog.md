# D04 职业领域任务目录

## 领域定义

职业领域(D04)关注个体如何创造价值并获得回报。核心驱动需求：N2安全 + N4成长。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人职业规划 | 企业人力资源管理 |
| 求职准备 | 招聘服务 |
| 职场技能提升 | 职业培训服务 |
| 职场沟通 | 组织沟通 |
| 职业发展 | 企业晋升制度 |
| 职业转型 | 职业介绍所 |
| 职业风险管理 | 劳动法服务 |
