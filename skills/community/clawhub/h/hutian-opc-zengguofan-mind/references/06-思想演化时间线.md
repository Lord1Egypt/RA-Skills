## 一、人生五阶段划分

### 阶段一：早年求学（1811-1838）28岁中进士
- **特点**：普通农家子弟，资质平庸，靠死读书考上进士
- **核心思想**：科举、理学入门、立志做圣人
- **关键事件**：
  - 1833年（23岁）中秀才
  - 1834年（24岁）中举人
  - 1838年（28岁）中进士，入翰林院
- **思想状态**：年轻气盛，想做圣人，还没经历社会毒打

### 阶段二：京城做官（1838-1852）28-42岁
- **特点**：京城做官14年，从副处级到副部长级
- **核心思想**：理学修身、每日十二条日课、开始识人
- **关键事件**：
  - 师从唐鉴、倭仁，理学功夫大进
  - 每日写日记骂自己，克己功夫极深
  - 开始发现人才、结交人才
- **思想状态**：修身阶段，还没真正做事

### 阶段三：组建湘军（1852-1860）42-50岁
- **特点**：在家守孝，墨绖从戎，从零开始组建湘军
- **核心思想**：结硬寨打呆仗、屡败屡战、打脱牙和血吞
- **关键事件**：
  - 1854年（44岁）靖港惨败，跳水自杀
  - 1855年（45岁）湖口惨败，又要自杀
  - 1856年（46岁）坐困江西，被太平军包围
  - 1858年（48岁）三河惨败，李续宾战死
  - 1860年（50岁）祁门被困，差点死了
- **思想状态**：最痛苦的阶段，也是成长最快的阶段，从书生到统帅

### 阶段四：平定太平天国（1860-1864）50-54岁
- **特点**：真正的巅峰期，军事、政治、用人全面成熟
- **核心思想**：大功不居、功成身退、裁军自保
- **关键事件**：
  - 1861年（51岁）攻克安庆
  - 1864年（54岁）攻克南京，平定太平天国
  - 立刻裁撤湘军25000人，主动交权
- **思想状态**：极其成熟，知道进，更知道退

### 阶段五：晚年洋务（1864-1872）54-62岁
- **特点**：晚年，做直隶总督，办洋务
- **核心思想**：师夷长技以制夷、守旧但不顽固
- **关键事件**：
  - 创办安庆内军械所
  - 派幼童留美
  - 处理天津教案（最争议的一件事）
  - 1872年去世，谥号文正
- **思想状态**：看透一切，知道大清要完，但还是尽力而为

## 二、关键思想转折点

### 转折点1：1857年（47岁）守父丧在家
- **之前**：刚愎自用，和所有人对着干，官场上处处碰壁
- **之后**：大悔大悟，开始和光同尘，开始灵活，开始懂政治
- **核心变化**：从"我要做圣人"到"我要做成事"

### 转折点2：1864年（54岁）打下南京
- **之前**：还在想立功，还在想证明自己
- **之后**：知道功高震主，知道退，知道藏
- **核心变化**：从"成事"到"保身"

### 转折点3：1870年（60岁）天津教案
- **之前**：还是想做圣人，还是想留清名
- **之后**：知道在大势面前，个人名节不算什么
- **核心变化**：从"求名"到"求实"

## 五、OPC人生时间线模板

```
曾国藩式人生时间线：

第一阶段（0-30岁）：积累期
- 科举/学历 → 打好基础
- 理学/认知 → 建立框架
- 人脉/资源 → 积累种子

第二阶段（30-40岁）：试错期
- 创业/探索 → 小步快跑
- 失败/反思 → 打脱牙和血吞
- 调整/迭代 → 屡败屡战

第三阶段（40-50岁）：突破期
- 核心产品 → 打透一个点
- 核心团队 → 找到合伙人
- 核心客户 → 服务好种子用户

第四阶段（50-60岁）：成熟期
- 规模化 → 复制成功
- 品牌化 → 建立壁垒
- 生态化 → 建平台

第五阶段（60岁后）：传承期
- 培养接班人 → 建梯队
- 总结方法论 → 出书立传
- 退出/交班 → 功成身退
```
