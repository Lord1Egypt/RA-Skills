# 短视频脚本模板库

## 模板一：知识干货型（60秒）

```
【标题】X个[领域]技巧，[目标人群]必看

[0-3s - Hook]
画面：[引发好奇/震惊的视觉]
口播："[反常识观点 or 问题]？"
字幕：【大字加粗标题】

[3-6s - 承诺]
画面：主播正面出镜，表情自信
口播："今天教你X招，[具体效果]，学完直接用"
字幕：【今天教你X招】

[6-18s - 第1点]
画面：[演示/图表/截图]
字幕：①[要点标题]
口播："第一，[要点内容]。[一句具体说明/案例]"

[18-35s - 第2-3点]
（同上结构，节奏加快）

[35-50s - 总结强化]
画面：回顾3个要点，字幕列出
口播："记住这3点：[快速复述]"

[50-60s - CTA]
口播："点赞收藏，避免下次找不到。我是[昵称]，关注我，持续分享[领域]干货"
字幕：【点赞收藏不迷路】
```

## 模板二：情感共鸣型（90秒）

```
【标题】"[场景]，我终于明白了……"

[0-4s - Hook场景]
画面：[日常生活真实场景，引发代入感]
字幕：【你有没有也这样过？】
口播：（可不说话，纯画面+文字）

[4-15s - 建立共鸣]
口播："[描述普通人都经历过的困境/情绪]"
"[我也曾经……直到有一天……]"

[15-45s - 故事/转折]
口播："[讲述一个小故事或经历]"
"[遇到的困难/冲突]"
"[转折点]"

[45-70s - 升华/结论]
口播："后来我才明白，[核心洞见/金句]"
字幕：【[提炼出来的金句，加大加粗】

[70-90s - 引发互动+CTA]
口播："你有没有类似的经历？评论区聊聊"
"喜欢这类内容，记得关注我"
```

## 模板三：产品种草型（45秒）

```
【标题】[痛点] 终于找到解决办法了！

[0-4s - 痛点展示]
画面：[使用前的麻烦场景，真实]
口播："[痛点描述]，这个问题困扰了我好久"

[4-15s - 产品出现]
画面：[自然拿出产品，展示外观]
口播："直到我发现了这个——"
字幕：【[产品名/类别】

[15-30s - 使用过程]
画面：[真实使用演示，展示关键功能]
口播："[描述使用步骤/感受]，[使用亮点]"
字幕：【[功能点1] [功能点2]】

[30-40s - 效果展示]
画面：[before/after 对比 or 使用结果]
口播："效果真的没想到，[具体效果描述]"

[40-45s - 购买信息+CTA]
口播："链接放评论区了，需要的直接去看"
字幕：【🛒链接在评论区】
```

## 模板四：观点输出型（120秒）

```
【标题】说句实话，[反常识观点]

[0-5s - 争议性观点]
口播："我今天想说一个可能有点争议的事：[观点]"
字幕：【大字展示观点】

[5-20s - 问题引出]
口播："很多人觉得[主流观点]，但我发现……"
[举出1个具体反例]

[20-60s - 论据展开]
口播："原因有3个：
第一，[论据+例子]
第二，[论据+例子]
第三，[论据+例子]"

[60-90s - 应对方案]
口播："所以我建议，与其[传统做法]，不如试试：
[具体可操作建议]"

[90-120s - 总结+互动]
口播："你觉得我说的对吗？不同意见欢迎评论区怼我"
"同意的点个赞，让更多人看到这条视频"
```

## 常用金句结构

- "不是 A，而是 B"（纠正认知）
- "越 X，越 Y"（规律揭示）
- "只有 X，才能 Y"（条件强调）
- "当你 X 的时候，其实你已经 Y 了"（认知翻转）
- "大多数人 X，少数人 Y，这就是差距所在"（对比激励）
