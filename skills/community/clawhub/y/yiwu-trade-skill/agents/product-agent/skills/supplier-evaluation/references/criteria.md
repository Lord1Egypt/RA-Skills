# 供应商评估标准体系 (Supplier Evaluation Criteria)

## 一、四维评分体系

### 1. 价格竞争力 (权重: 25%)

#### 评分维度
| 评分 | 等级 | 标准 |
|------|------|------|
| 5分 | 极优 | 价格低于市场均价15%以上，报价及时准确 |
| 4分 | 优秀 | 价格低于市场均价5-15%，报价在24h内 |
| 3分 | 良好 | 价格与市场均价持平，报价48h内 |
| 2分 | 一般 | 价格高于市场均价5-15% |
| 1分 | 较差 | 价格高于市场均价15%以上 |

#### 评分因子
- **报价折扣率**: 批量订单折扣力度
- **价格稳定性**: 30天内价格波动幅度
- **价格响应速度**: 对询价的响应时间
- **含税含票价差**: 发票价格的合理性

#### 数据来源
- 历史报价记录
- 同期市场价格对比
- 同行竞品价格调研

---

### 2. 质量稳定性 (权重: 30%)

#### 评分维度
| 评分 | 等级 | 标准 |
|------|------|------|
| 5分 | 极优 | 来料合格率≥99%，连续12月无重大质量事故 |
| 4分 | 优秀 | 来料合格率≥96%，年度质量投诉<2次 |
| 3分 | 良好 | 来料合格率≥92%，年度质量投诉<5次 |
| 2分 | 一般 | 来料合格率≥85%，存在可改善问题 |
| 1分 | 较差 | 来料合格率<85%，存在严重质量问题 |

#### 评分因子
- **来料合格率**: IQCpass率
- **客验通过率**: 客户验货合格率
- **重大质量事故**: 退货、索赔事故次数
- **质量改进响应**: 问题发生后的改进速度

#### 质量缺陷分类
| 缺陷等级 | 定义 | 扣分权重 |
|----------|------|----------|
| 严重 | 影响使用安全 | -2分/次 |
| 主要 | 影响正常使用或外观严重瑕疵 | -1分/次 |
| 次要 | 不影响使用的轻微瑕疵 | -0.3分/次 |

---

### 3. 交期准时率 (权重: 30%)

#### 评分维度
| 评分 | 等级 | 标准 |
|------|------|------|
| 5分 | 极优 | 准时交付率≥98%，提前交付率>50% |
| 4分 | 优秀 | 准时交付率≥95%，延期<3天/次 |
| 3分 | 良好 | 准时交付率≥88%，延期<7天/次 |
| 2分 | 一般 | 准时交付率≥80%，偶尔延期但可沟通 |
| 1分 | 较差 | 准时交付率<80%，经常延期且沟通困难 |

#### 评分因子
- **订单按期交付率**: 统计周期内按期完成订单数/总订单数
- **平均延期天数**: 延期订单的平均延期时长
- **延期预警率**: 提前通知延期的比例
- **紧急订单处理**: 急单的完成情况

#### 评分计算公式
```
准时率得分 = 100 - (延期订单数/总订单数 × 100)
如延期5%，则得分95分
考虑延期天数: 延期>7天额外扣2分/次
```

---

### 4. 服务响应 (权重: 15%)

#### 评分维度
| 评分 | 等级 | 标准 |
|------|------|------|
| 5分 | 极优 | 平均响应<1h，问题解决率100% |
| 4分 | 优秀 | 平均响应<4h，问题解决率≥95% |
| 3分 | 良好 | 平均响应<12h，问题解决率≥85% |
| 2分 | 一般 | 平均响应<24h，问题解决率≥70% |
| 1分 | 较差 | 响应>24h，问题解决率<70% |

#### 评分因子
- **询盘响应时间**: 从发出到首次回复的平均时长
- **问题解决率**: 提出的问题得到有效解决的比例
- **服务态度评分**: 客户对服务态度的评价
- **专业度评分**: 客户对供应商专业度的评价

---

## 二、综合评分计算

### 评分公式
```
综合评分 = (价格竞争力×0.25) + (质量稳定性×0.30) + (交期准时率×0.30) + (服务响应×0.15)
```

### 等级划分
| 等级 | 综合评分 | 定义 | 建议 |
|------|----------|------|------|
| A+ | 4.7-5.0 | 卓越供应商 | 优先合作，战略伙伴 |
| A | 4.3-4.6 | 优质供应商 | 积极合作 |
| B+ | 4.0-4.2 | 良好供应商 | 正常合作 |
| B | 3.5-3.9 | 一般供应商 | 谨慎合作，需要扶持 |
| C | 3.0-3.4 | 问题供应商 | 限制合作，监督改进 |
| D | <3.0 | 不合格 | 淘汰/黑名单 |

## 三、风险评估标准

### 风险等级
| 等级 | 定义 | 指标表现 |
|------|------|----------|
| 低风险 | 无明显风险特征 | 无黑名单，无投诉，无资质问题 |
| 中风险 | 存在1-2项需关注项 | 偶发延期，小问题投诉 |
| 高风险 | 存在严重问题隐患 | 质量事故，频繁延期，拒绝整改 |
| 极高风险 | 已造成损失或违规 | 欺诈行为，假货，已被投诉 |

### 黑名单标准
符合以下任一条件，列入黑名单：
1. 欺诈行为（收款不发货、货不对板）
2. 严重质量问题导致重大损失
3. 侵犯知识产权被投诉成立
4. 拒绝配合整改或整改无效
5. 涉及法律诉讼且败诉

## 四、供应商画像标签

### 规模标签
| 标签 | 标准 | 说明 |
|------|------|------|
| 源头工厂 | 自有厂房，有生产线 | 可参观，有生产能力 |
| 贸易商 | 无自有工厂，撮合交易 | 资源整合能力强 |
| 作坊 | 小规模，家庭式生产 | 适合小批量 |
| 大厂 | 员工>200人，年产值>5000万 | 规模化生产能力强 |

### 专长标签
| 标签 | 标准 |
|------|------|
| 爆款专家 | 某品类销量TOP10 |
| 定制专家 | OEM/ODM经验丰富 |
| 快速出货 | 平均交期<7天 |
| 精品路线 | 注重品质，客单价高 |
| 性价比王 | 同品质价格最优 |

### 合规标签
| 标签 | 标准 |
|------|------|
| 实地认证 | 已完成第三方实地验厂 |
| 资质齐全 | 核心资质证书齐全有效 |
| 社保合规 | 员工社保缴纳合规 |
| 环保合规 | 环评/排放合规 |
