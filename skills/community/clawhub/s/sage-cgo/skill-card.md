## Known Risks and Mitigations: <br>
Risk: The security scan reports that the skill can automatically write persistent business-memory files and workspace agent-instruction files without a clear consent or dry-run step. <br>
Mitigation: Install and run it only in workspaces where changes to AGENTS.md, CLAUDE.md, and ~/.sage are acceptable; review scripts such as bootstrap_workspace_identity.sh before execution. <br>
