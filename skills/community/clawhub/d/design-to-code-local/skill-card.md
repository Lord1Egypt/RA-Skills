## Description: <br>
Implements UI from Figma, Sketch, or image-based design mockups as pixel-accurate frontend code with responsive behavior and design token alignment. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[longfer](https://clawhub.ai/user/longfer) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and frontend engineers use this skill to translate Figma, Sketch, or image mockups into componentized frontend code that follows the current project's stack, design tokens, semantic HTML, ARIA expectations, and responsive layout requirements. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Broad activation wording may cause the skill to be used for generic UI tasks without a concrete mockup or design link. <br>
Mitigation: Invoke it explicitly for design-driven work and confirm the input design source before applying generated frontend changes. <br>
Risk: Generated frontend code can diverge from project conventions or introduce accessibility and responsive-layout issues. <br>
Mitigation: Review generated code changes normally, with attention to existing design tokens, semantic HTML, ARIA behavior, breakpoints, and any noted design differences. <br>
