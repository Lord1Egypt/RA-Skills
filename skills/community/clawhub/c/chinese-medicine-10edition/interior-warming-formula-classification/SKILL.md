# Interior-Warming Formula Classification

## When to Use
- 患者明确诊断为里寒证，包括以下三类：
  - 脾胃虚寒证（如脘腹冷痛、呕吐清涎、食少便溏）
  - 阳气衰微寒厥证（如四肢厥逆、冷汗淋漓、脉微欲绝）
  - 寒凝经脉证（如肢体冷痛、麻木、肤色青紫）
- 已排除禁忌证：阴虚、血虚、血热体质，以及真热假寒证

## Execution Steps
1. **辨证确认**：严格区分“真寒假热”与“真热假寒”，避免误用。
2. **按证选方**：
   - 脾胃虚寒证 → 选用温中祛寒剂（如理中丸）
   - 阳气衰微寒厥证 → 选用回阳救逆剂（如四逆汤）
   - 寒凝经脉证 → 选用温经散寒剂（如阳和汤）
3. **注意用药安全**：
   - 辛温燥热药易耗伤阴血，阴虚、血虚、血热者禁用
   - 含附子等有毒药物须久煎减毒
   - 回阳救逆剂应“中病即止”，不可久服
4. **监测反应**：服药后若出现口干、咽痛、烦躁、便秘等热象，立即停药并重新辨证。

> ⚠️ 提醒：温里剂属“八法”中“温法”，核心在于扶阳抑阴。误用于热证或虚热证将助火伤阴，加重病情。
