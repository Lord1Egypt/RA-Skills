### 计算题

9. 一台他励直流电动机，PN = 10kW，UN = 220V，IN = 53.4A，nN = 1500r/min，Ra = 0.4Ω。求：
   (1) 额定电磁转矩 TN
   (2) 额定效率 ηN
   **解答**：
   (1) TN = 9.55 × PN/nN = 9.55 × 10000/1500 = 63.67 N·m
   (2) ηN = PN/(UN × IN) = 10000/(220 × 53.4) = 10000/11748 ≈ 85.1%

10. 一台并励直流发电机，PN = 6kW，UN = 230V，nN = 1450r/min，电枢回路电阻 Ra = 0.92Ω，励磁回路电阻 Rf = 280Ω。求额定负载时电枢电流 Ia 和电磁功率 Pem。
    **解答**：
    If = UN/Rf = 230/280 ≈ 0.82A
    IN = PN/UN = 6000/230 ≈ 26.09A
    Ia = IN + If = 26.09 + 0.82 ≈ 26.91A
    Pem = (UN + IaRa) × Ia = (230 + 26.91×0.92) × 26.91 ≈ 255.4 × 26.91 ≈ 6859W

### 计算题

49. 一台他励直流发电机，额定功率Pₙ = 10kW，额定电压Uₙ = 230V，额定转速nₙ = 1450r/min，电枢回路电阻Rₐ = 0.5Ω（含电刷接触电阻）。求额定负载时的电枢电流Iₐ、感应电动势E和电磁功率Pₑₘ。
   **解答**：
   步骤1：求额定电枢电流
   对于他励发电机，额定电枢电流等于额定电流：
   Iₐ = Iₙ = Pₙ/Uₙ = 10000/230 = 43.48A

   步骤2：求感应电动势
   E = Uₙ + IₐRₐ = 230 + 43.48 × 0.5 = 230 + 21.74 = 251.74V

   步骤3：求电磁功率
   Pₑₘ = EIₐ = 251.74 × 43.48 = 10945.7W ≈ 10.95kW

   最终答案：Iₐ = 43.48A，E = 251.74V，Pₑₘ ≈ 10.95kW

50. 一台他励直流电动机，额定数据：Pₙ = 7.5kW，Uₙ = 220V，Iₙ = 41A，nₙ = 1500r/min，电枢回路电阻Rₐ = 0.5Ω。求额定运行时的感应电动势Eₙ、电磁转矩Tₙ和效率ηₙ。
   **解答**：
   步骤1：求额定感应电动势
   他励电动机中，Iₐ ≈ Iₙ = 41A（忽略励磁电流）
   Eₙ = Uₙ - IₐRₐ = 220 - 41 × 0.5 = 220 - 20.5 = 199.5V

   步骤2：求额定电磁转矩
   电磁功率 Pₑₘ = EₙIₐ = 199.5 × 41 = 8179.5W
   角速度 Ωₙ = 2πnₙ/60 = 2π × 1500/60 = 157.08 rad/s
   Tₙ = Pₑₘ/Ωₙ = 8179.5/157.08 = 52.07 N·m

   步骤3：求效率
   输入功率 P₁ = UₙIₙ = 220 × 41 = 9020W
   ηₙ = Pₙ/P₁ = 7500/9020 = 83.15%

   最终答案：Eₙ = 199.5V，Tₙ = 52.07 N·m，ηₙ = 83.15%

51. 一台他励直流电动机，Uₙ = 220V，Iₐₙ = 50A，nₙ = 1000r/min，Rₐ = 0.4Ω。求：（1）固有机械特性的理想空载转速n₀；（2）电枢串入Rₛ = 1.6Ω电阻后，额定负载转矩时的转速n。
   **解答**：
   步骤1：求电动势常数CₑΦ
   Eₙ = Uₙ - IₐₙRₐ = 220 - 50 × 0.4 = 220 - 20 = 200V
   CₑΦ = Eₙ/nₙ = 200/1000 = 0.2 V·min/r

   步骤2：求固有机械特性理想空载转速
   n₀ = Uₙ/(CₑΦ) = 220/0.2 = 1100 r/min

   步骤3：求电枢串电阻后额定负载转矩时的转速
   额定负载转矩不变，电枢电流不变（仍为Iₐₙ = 50A）
   n = (Uₙ - Iₐₙ(Rₐ + Rₛ))/(CₑΦ) = (220 - 50 × (0.4 + 1.6))/0.2 = (220 - 100)/0.2 = 120/0.2 = 600 r/min

   最终答案：n₀ = 1100 r/min，n = 600 r/min

52. 一台他励直流电动机，Uₙ = 220V，Rₐ = 0.2Ω，CₑΦₙ = 0.14 V·min/r。若采用降压起动，要求起动电流Iₛₜ ≤ 2Iₐₙ（Iₐₙ = 100A），求起动时电枢两端应加的最低电压Uₛₜ。
   **解答**：
   步骤1：确定允许的最大起动电流
   Iₛₜ_max = 2Iₐₙ = 2 × 100 = 200A

   步骤2：求起动时应加的最低电压
   起动瞬间n = 0，E = 0
   Iₛₜ = Uₛₜ/Rₐ
   Uₛₜ = Iₛₜ_max × Rₐ = 200 × 0.2 = 40V

   最终答案：Uₛₜ = 40V

53. 一台他励直流电动机，Uₙ = 220V，Iₐₙ = 80A，nₙ = 800r/min，Rₐ = 0.3Ω。采用弱磁调速，将磁通从Φₙ减小到0.8Φₙ，求调速后的理想空载转速和额定电枢电流下的转速。
   **解答**：
   步骤1：求CₑΦₙ
   Eₙ = Uₙ - IₐₙRₐ = 220 - 80 × 0.3 = 220 - 24 = 196V
   CₑΦₙ = Eₙ/nₙ = 196/800 = 0.245 V·min/r

   步骤2：弱磁后CₑΦ = 0.8CₑΦₙ = 0.8 × 0.245 = 0.196 V·min/r

   步骤3：求弱磁后理想空载转速
   n₀' = Uₙ/(CₑΦ) = 220/0.196 = 1122.4 r/min

   步骤4：求额定电枢电流下的转速
   n' = (Uₙ - IₐₙRₐ)/(CₑΦ) = (220 - 80 × 0.3)/0.196 = 196/0.196 = 1000 r/min

   最终答案：弱磁后理想空载转速n₀' = 1122.4 r/min，额定电枢电流下转速n' = 1000 r/min

54. 一台他励直流发电机，Pₙ = 20kW，Uₙ = 230V，nₙ = 1500r/min，Rₐ = 0.3Ω。当转速降至1200r/min而励磁电流不变时，求空载电压和额定电流下的端电压。
   **解答**：
   步骤1：求额定电流
   Iₙ = Pₙ/Uₙ = 20000/230 = 86.96A

   步骤2：求额定运行时的感应电动势
   Eₙ = Uₙ + IₙRₐ = 230 + 86.96 × 0.3 = 230 + 26.09 = 256.09V

   步骤3：求转速降至1200r/min时的感应电动势
   励磁电流不变，Φ不变，E正比于n
   E' = Eₙ × (n'/nₙ) = 256.09 × (1200/1500) = 256.09 × 0.8 = 204.87V

   步骤4：求空载电压
   空载时I = 0，U₀' = E' = 204.87V

   步骤5：求额定电流下的端电压
   U' = E' - IₙRₐ = 204.87 - 86.96 × 0.3 = 204.87 - 26.09 = 178.78V

   最终答案：空载电压 = 204.87V，额定电流下端电压 = 178.78V

55. 一台他励直流电动机，Uₙ = 440V，Iₐₙ = 100A，nₙ = 960r/min，Rₐ = 0.16Ω。采用能耗制动，要求最大制动电流不超过2Iₐₙ，求电枢回路应串入的电阻Rₑₓ。
   **解答**：
   步骤1：求制动瞬间的感应电动势（近似等于额定运行时的电动势）
   E = Uₙ - IₐₙRₐ = 440 - 100 × 0.16 = 440 - 16 = 424V

   步骤2：求允许的最大制动电流
   I_zmax = 2Iₐₙ = 200A

   步骤3：能耗制动时，电枢脱离电源经电阻闭合
   制动瞬间：E = I_zmax × (Rₐ + Rₑₓ)
   Rₐ + Rₑₓ = E/I_zmax = 424/200 = 2.12Ω

   步骤4：求应串入的电阻
   Rₑₓ = 2.12 - Rₐ = 2.12 - 0.16 = 1.96Ω

   最终答案：Rₑₓ = 1.96Ω

56. 一台他励直流电动机，Uₙ = 220V，Iₐₙ = 50A，nₙ = 1000r/min，Rₐ = 0.4Ω。采用反接制动，要求最大制动电流不超过2Iₐₙ，求电枢回路应串入的限流电阻Rₐₑ。
   **解答**：
   步骤1：求制动瞬间的感应电动势
   E = Uₙ - IₐₙRₐ = 220 - 50 × 0.4 = 220 - 20 = 200V

   步骤2：求允许的最大制动电流
   I_zmax = 2Iₐₙ = 100A

   步骤3：反接制动时，电枢电压反向，与E同方向叠加
   Uₙ + E = I_zmax × (Rₐ + Rₐₑ)
   220 + 200 = 100 × (0.4 + Rₐₑ)
   420 = 100 × (0.4 + Rₐₑ)
   0.4 + Rₐₑ = 4.2

   步骤4：求应串入的限流电阻
   Rₐₑ = 4.2 - 0.4 = 3.8Ω

   最终答案：Rₐₑ = 3.8Ω

57. 一台4极直流电机，电枢绕组为单叠绕组，电枢总导体数N = 400，每极磁通Φ = 0.02Wb，转速n = 1500r/min。求感应电动势E。
   **解答**：
   步骤1：确定参数
   极对数p = 4/2 = 2
   单叠绕组并联支路对数a = p = 2

   步骤2：求电动势常数
   Cₑ = pN/(60a) = 2 × 400/(60 × 2) = 800/120 = 6.667

   步骤3：求感应电动势
   E = CₑΦn = 6.667 × 0.02 × 1500 = 200V

   最终答案：E = 200V

58. 一台4极直流电机，电枢绕组为单叠绕组，电枢总导体数N = 400，每极磁通Φ = 0.02Wb，电枢电流Iₐ = 50A。求电磁转矩T。
   **解答**：
   步骤1：确定参数
   极对数p = 2
   单叠绕组并联支路对数a = p = 2

   步骤2：求转矩常数
   Cₜ = pN/(2πa) = 2 × 400/(2π × 2) = 800/(4π) = 63.66

   步骤3：求电磁转矩
   T = CₜΦIₐ = 63.66 × 0.02 × 50 = 63.66 N·m

   最终答案：T = 63.66 N·m

19. 一台单相变压器，SN = 50kVA，U₁N/U₂N = 10000/230V，空载试验在低压侧测得：U₀ = 230V，I₀ = 6A，p₀ = 380W；短路试验在高压侧测得：Uk = 400V，Ik = 5A，pk = 600W。求：
   (1) 变比 k
   (2) 励磁参数 Rm、Xm（归算到高压侧）
   (3) 短路参数 Rk、Xk
   **解答**：
   (1) k = U₁N/U₂N = 10000/230 ≈ 43.48
   (2) 空载试验在低压侧，需归算到高压侧：
      Z₀(低压) = U₀/I₀ = 230/6 ≈ 38.33Ω
      R₀(低压) = p₀/I₀² = 380/36 ≈ 10.56Ω
      X₀(低压) = √(38.33² - 10.56²) ≈ 36.83Ω
      归算到高压侧：Z₀ = k² × Z₀(低压) = 43.48² × 38.33 ≈ 72380Ω
      R₀ = k² × R₀(低压) = 43.48² × 10.56 ≈ 19940Ω → Rm ≈ 19940Ω, Xm ≈ 72380Ω的虚部（实际需更细致计算，此处为简化）
   (3) 短路试验在高压侧（已归算）：
      Zk = Uk/Ik = 400/5 = 80Ω
      Rk = pk/Ik² = 600/25 = 24Ω
      Xk = √(80² - 24²) = √(6400 - 576) = √5824 ≈ 76.3Ω

20. 一台三相变压器，SN = 1000kVA，U₁N/U₂N = 10/0.4kV，pkN = 12kW，p₀ = 4kW。求当负载系数 β = 0.8 且cosφ₂ = 0.8 时，该变压器的效率。
    **解答**：
    η = βSNcosφ₂/(βSNcosφ₂ + p₀ + β²pkN)
    = 0.8×1000×0.8/(0.8×1000×0.8 + 4 + 0.64×12)
    = 640/(640 + 4 + 7.68)
    = 640/651.68 ≈ 98.2%

300. 一台单相变压器，空载试验在低压侧进行：U₀=220V，I₀=0.5A，P₀=20W；短路试验在高压侧进行：Uk=120V，Ik=4.55A，Pk=50W。已知变比k=10。求：（1）励磁参数Zm、Rm、Xm（折算到高压侧）；（2）短路参数Zk、Rk、Xk。  
**解答**：  
（1）空载试验在低压侧测得：  
Zm(低压)=U₀/I₀=220/0.5=440Ω  
Rm(低压)=P₀/I₀²=20/0.5²=20/0.25=80Ω  
Xm(低压)=√(Zm²−Rm²)=√(440²−80²)=√(193600−6400)=√187200≈432.7Ω  
折算到高压侧：Zm=k²×Zm(低压)=100×440=44000Ω  
Rm=k²×Rm(低压)=100×80=8000Ω  
Xm=k²×Xm(低压)=100×432.7=43270Ω  
（2）短路试验在高压侧测得：  
Zk=Uk/Ik=120/4.55≈26.37Ω  
Rk=Pk/Ik²=50/4.55²=50/20.70≈2.42Ω  
Xk=√(Zk²−Rk²)=√(26.37²−2.42²)=√(695.5−5.86)=√689.64≈26.26Ω  
最终答案：Zm=44000Ω，Rm=8000Ω，Xm≈43270Ω；Zk≈26.37Ω，Rk≈2.42Ω，Xk≈26.26Ω

301. 一台单相变压器，SN=50kVA，U₁N/U₂N=10000V/230V。空载损耗P₀=200W，短路损耗Pk=600W。求负载系数β=1（满载）且为纯电阻负载时的效率η。  
**解答**：  
满载纯电阻负载时，输出功率P₂=SN×cosφ₂=50×1=50kW=50000W（cosφ₂=1）  
铜损pCu=β²×Pk=1²×600=600W  
铁损pFe=P₀=200W  
总损耗p=pCu+pFe=600+200=800W  
效率η=P₂/(P₂+p)=50000/(50000+800)=50000/50800≈98.43%  
最终答案：η≈98.43%

302. 一台单相变压器，SN=100kVA，空载损耗P₀=600W，短路损耗Pk=2400W。求变压器效率最大时的负载系数β_m及最大效率η_max。  
**解答**：  
效率最大条件：可变损耗=不变损耗  
β_m²×Pk=P₀  
β_m=√(P₀/Pk)=√(600/2400)=√0.25=0.5  
最大效率时：cosφ₂=1（纯电阻负载），  
P₂=β_m×SN×cosφ₂=0.5×100×1=50kW=50000W  
pCu=β_m²×Pk=0.5²×2400=600W  
pFe=P₀=600W  
总损耗p=pCu+pFe=600+600=1200W  
η_max=P₂/(P₂+p)=50000/(50000+1200)=50000/51200≈97.66%  
最终答案：β_m=0.5，η_max≈97.66%

303. 两台变压器并联运行，额定容量分别为S₁N=100kVA和S₂N=200kVA，短路阻抗标幺值分别为Z_k1*=0.05和Z_k2*=0.04，总负载S=150kVA。求两台变压器各承担多少负载。  
**解答**：  
并联运行时，负载分配与额定容量成正比、与短路阻抗标幺值成反比：  
S₁/S₁N∝1/Z_k1*，S₂/S₂N∝1/Z_k2*  
即 S₁=S₁N×(1/Z_k1*)×S/(S₁N/Z_k1*+S₂N/Z_k2*)  
=100×(1/0.05)×150/(100/0.05+200/0.04)  
=100×20×150/(2000+5000)  
=3000000/7000≈428.6kVA  
但这超过了S₁N=100kVA，说明第一台变压器已过载。  
重新计算：当Z_k1*≠Z_k2*时，  
S₁=S×(S₁N/Z_k1*)/(S₁N/Z_k1*+S₂N/Z_k2*)  
=150×(100/0.05)/(100/0.05+200/0.04)  
=150×2000/(2000+5000)=150×2000/7000≈42.86kVA  
S₂=S×(S₂N/Z_k2*)/(S₁N/Z_k1*+S₂N/Z_k2*)  
=150×(200/0.04)/(2000+5000)  
=150×5000/7000≈107.14kVA  
验证：S₁+S₂=42.86+107.14=150kVA ✓  
最终答案：S₁≈42.86kVA，S₂≈107.14kVA

304. 一台三相变压器，SN=100kVA，U₁N/U₂N=6000V/400V，Y,y接法。空载试验在低压侧：U₀=400V，I₀=5A，P₀=400W。短路试验在高压侧：Uk=300V，Ik=9.63A，Pk=1200W。求：（1）励磁参数（折算到高压侧）；（2）短路参数。

U₀_ph=400/√3≈230.9V，I₀=5A  
Zm(低压)=U₀_ph/I₀=230.9/5≈46.18Ω  
Rm(低压)=P₀/(3×I₀²)=400/(3×25)=400/75≈5.33Ω  
Xm(低压)=√(46.18²−5.33²)=√(2132.6−28.4)=√2104.2≈45.87Ω

305. 一台单相自耦变压器，额定容量SN=20kVA，额定电压U₁N=220V，U₂N=180V。求：（1）变比k；（2）电磁容量和传导容量；（3）公共绕组电流。

306. 两台变压器并联运行，变比分别为k₁=10和k₂=9.5，短路阻抗分别为Zk₁=20Ω和Zk₂=22Ω（均折算到高压侧）。一次侧额定电压U₁N=10000V。求空载时环流的大小。

307. 一台单相变压器，SN=30kVA，U₁N/U₂N=3000V/230V，短路阻抗Zk=15Ω（高压侧），短路电阻Rk=2Ω（高压侧）。求满载且cosφ₂=0.8（滞后）时的电压变化率ΔU%。

308. 一台三相变压器，Y,d11连接组别。一次侧线电压为10000V，二次侧线电压为400V。画出连接组别的向量关系并说明钟面表示法的含义。  
**解答**：

钟面含义：11点位置意味着二次侧线电压相量指向钟面11点方向，与指向12点的一次侧线电压相量相差330°。  
最终答案：Y,d11表示一次侧Y接、二次侧d接，二次侧线电压滞后一次侧线电压30°（相位差330°）

294. 一台4极三相异步电动机，f=50Hz，定子槽数Z=36。求每极每相槽数q和槽距角α。
   **解答**：
   极对数 p = 4/2 = 2
   q = Z/(2mp) = 36/(2×3×2) = 36/12 = 3

   最终答案：q=3，α=20°

295. 一台4极三相电机，Z=36，采用双层短距绕组，节距y=7。求基波短距因数kp1。（已知α=20°）
   **解答**：
   极距 τ = Z/(2p) = 36/(2×2) = 9
   短距比 y/τ = 7/9
   α = p×360°/Z = 2×360°/36 = 20°
   kp1 = sin(yα/2) = sin(7×20°/2) = sin(70°) = 0.9397
   最终答案：kp1 = 0.9397

296. 一台4极三相电机，Z=36，q=3，α=20°。求基波分布因数kd1和绕组因数kN1（设短距因数kp1=0.9397）。
   **解答**：
   kd1 = sin(qα/2)/(q·sin(α/2))
        = sin(3×20°/2)/(3×sin(20°/2))
        = sin30°/(3×sin10°)
        = 0.5/(3×0.17365)
        = 0.5/0.52095
        = 0.9598
   kN1 = kp1 × kd1 = 0.9397 × 0.9598 = 0.9019
   最终答案：kd1 = 0.9598，kN1 = 0.9019

297. 一台三相4极电机，f=50Hz，每相串联匝数N=80，每极磁通Φ₁=0.025Wb，绕组因数kN1=0.9019。求每相基波感应电动势Eφ1。
   **解答**：
   Eφ1 = 4.44fNΦ₁kN1
        = 4.44 × 50 × 80 × 0.025 × 0.9019
        = 4.44 × 50 × 80 × 0.025 × 0.9019
        = 222 × 80 × 0.025 × 0.9019
        = 17760 × 0.025 × 0.9019
        = 444 × 0.9019
        = 400.04 V
   最终答案：Eφ1 ≈ 400 V

   **解答**：
   同步转速 n₁ = 60f₁/p = 60×50/2 = 1500 r/min
   额定转差率 sN = (n₁-nN)/n₁ = (1500-1450)/1500 = 50/1500 = 0.0333

   转子铜耗 pCu2 = sN × Pem = 0.04 × 12000 = 480 W
   机械功率 Pmec = (1-sN) × Pem = 0.96 × 12000 = 11520 W
   同步角速度 Ω₁ = 2πn₁/60 = 2π×1500/60 = 157.08 rad/s
   电磁转矩 T = Pem/Ω₁ = 12000/157.08 = 76.39 N·m

### 计算题

349. 一台三相隐极同步发电机，额定容量SN=250MVA，额定电压UN=10.5kV，额定频率fN=50Hz，极对数p=1，同步电抗Xd=1.8Ω（不饱和值），电枢电阻忽略不计。空载额定电压时的励磁电势E₀=10.5/√3 kV（相值）。求：（1）同步转速n；（2）额定运行时cosφ=0.8（滞后）的功角δ；（3）最大电磁功率Pem_max。
   **解答**：
   （1）同步转速：
   $$n = \frac{60f}{p} = \frac{60 \times 50}{1} = 3000 \text{ r/min}$$

   （2）求功角δ：
   额定相电压：
   $$U = \frac{U_N}{\sqrt{3}} = \frac{10500}{\sqrt{3}} = 6062.2 \text{ V}$$

   额定相电流：
   $$I_N = \frac{S_N}{\sqrt{3}U_N} = \frac{250 \times 10^6}{\sqrt{3} \times 10500} = 13746.4 \text{ A}$$

   励磁电势相值：
   $$E_0 = \frac{10500}{\sqrt{3}} = 6062.2 \text{ V}$$

   由隐极发电机电动势方程（忽略Ra）：
   $$\dot{E}_0 = \dot{U} + j\dot{I}X_d$$

   以Ú为参考相量，Ú=6062.2∠0° V
   $$\dot{I} = I_N \angle -\varphi = 13746.4 \angle -36.87° \text{ A}$$
   （cosφ=0.8，φ=36.87°）

   $$j\dot{I}X_d = j \times 13746.4 \angle -36.87° \times 1.8 = 24743.5 \angle 53.13° \text{ V}$$
   $$= 24743.5(\cos 53.13° + j\sin 53.13°) = 14846.1 + j19794.8 \text{ V}$$

   $$\dot{E}_0 = (6062.2 + 0) + (14846.1 + j19794.8) = 20908.3 + j19794.8 \text{ V}$$

   $$E_0 = \sqrt{20908.3^2 + 19794.8^2} = 28788.5 \text{ V}$$

   $$\delta = \arctan\frac{19794.8}{20908.3} = 43.5°$$

   （3）最大电磁功率：
   $$P_{em\max} = m\frac{E_0 U}{X_d} = 3 \times \frac{28788.5 \times 6062.2}{1.8} = 290.9 \text{ MW}$$

350. 一台三相凸极同步发电机，UN=11kV，fN=50Hz，极对数p=4，Xd=5.0Ω，Xq=3.0Ω，Ra忽略不计。额定运行时E₀=12kV（线值），U=11kV（线值），功角δ=30°。求：（1）同步转速n；（2）电磁功率Pem。
   **解答**：
   （1）同步转速：
   $$n = \frac{60f}{p} = \frac{60 \times 50}{4} = 750 \text{ r/min}$$

   （2）电磁功率：

## 六、综合与实验题

## 六、综合与实验题

44. 简述变压器空载试验和短路试验的目的及分别在哪个侧进行。
    **解答**：
    空载试验：目的为测定铁损 pFe、励磁阻抗 Zm = Rm + jXm。通常在低压侧进行（低压侧加额定电压，高压侧开路）。
    短路试验：目的为测定铜损 pCu（短路损耗）、短路阻抗 Zk = Rk + jXk。通常在高压侧进行（高压侧加低电压使电流达额定值，低压侧短路）。

45. 为什么电流互感器二次侧严禁开路？
    **解答**：
    正常运行时，二次侧电流产生的磁动势近似抵消一次侧磁动势，铁心中磁通很小。若二次侧开路，二次电流为零，一次磁动势全部用于励磁，铁心磁通急剧增大，导致：
    (1) 铁损剧增，铁心过热可能烧毁；
    (2) 二次侧感应出极高电压（可达数千伏），危及人身安全和设备绝缘。

46. 直流电动机为什么不能直接起动？常用的起动方法有哪些？
    **解答**：
    直接起动时，n=0 → E=0 → Ia = U/Ra，因 Ra 很小（约0.1~0.5Ω），起动电流可达额定电流的5~7倍甚至更高，造成：
    (1) 换向困难，产生严重火花；
    (2) 机械冲击，损坏转轴和传动机构；
    (3) 对电网造成过大冲击。
    常用起动方法：电枢串电阻起动、降压起动。

47. 试述三相异步电动机变频调速的原理及 V/f 恒定控制的原因。
    **解答**：
    变频调速原理：改变供电频率 f，同步转速 n₁ = 60f/p 随之改变，从而调节电动机转速。
    V/f 恒定控制原因：E₁ ≈ 4.44fNΦkw，若 f 降低而 U 不变，则 Φ 增大 → 磁路饱和 → 励磁电流剧增；若 f 升高而 U 不变，则 Φ 减小 → 转矩下降。保持 U/f 恒定可使 Φ 基本不变，从而保持转矩能力并避免磁路饱和。

48. 同步发电机并联运行时，如何调节有功功率和无功功率？
    **解答**：
    有功功率调节：调节原动机的输入转矩（如调节汽轮机蒸汽量或水轮机水量），增大原动机转矩 → 功角 δ 增大 → 输出有功功率增大。
    无功功率调节：调节励磁电流 If。增大 If → E₀ 增大 → 发出更多感性无功（过励）；减小 If → E₀ 减小 → 吸收感性无功（欠励）。V 形曲线描述了不同励磁下有功功率与无功功率的关系。

535. 他励直流电动机额定数据为：P_N = 10kW，U_N = 220V，I_N = 53A，n_N = 1000r/min，R_a = 0.3Ω。现采用电枢串电阻方法起动，要求起动电流I_st ≤ 2I_N，试求：
（1）应串入电枢回路的起动电阻R_st；
（2）起动转矩与额定转矩之比T_st/T_N。

   **解答**：
   （1）直接起动时的起动电流：
   I_st0 = U_N / R_a = 220 / 0.3 = 733.3A
   
   要求I_st ≤ 2I_N = 2 × 53 = 106A
   
   串电阻后起动电流：I_st = U_N / (R_a + R_st)
   
   R_a + R_st ≥ U_N / (2I_N) = 220 / 106 = 2.075Ω
   
   R_st ≥ 2.075 - 0.3 = 1.775Ω
   
   取R_st = 1.775Ω
   
   （2）额定运行时反电势：
   E_N = U_N - I_N × R_a = 220 - 53 × 0.3 = 204.1V
   
   CeΦ = E_N / n_N = 204.1 / 1000 = 0.2041
   
   起动转矩：T_st = CtΦ × I_st = 9.55 × CeΦ × I_st = 9.55 × 0.2041 × 106 = 206.9N·m
   
   额定转矩：T_N = 9.55 × P_N / n_N = 9.55 × 10000 / 1000 = 95.5N·m
   
   T_st / T_N = 206.9 / 95.5 = 2.17
   
   起动转矩约为额定转矩的2.17倍。

536. 一台他励直流电动机，U_N = 220V，I_N = 100A，n_N = 1500r/min，R_a = 0.2Ω。分别采用三种调速方法将转速调至1000r/min（恒转矩负载），试比较三种方法的效率并说明哪种方法最优。

   **解答**：
   额定反电势：E_N = U_N - I_N × R_a = 220 - 100 × 0.2 = 200V
   CeΦ = E_N / n_N = 200 / 1500 = 0.1333
   
   恒转矩负载，电枢电流不变：I_a = I_N = 100A
   
   **方法一：电枢串电阻调速**
   n = 1000r/min时：
   E = CeΦ × n = 0.1333 × 1000 = 133.3V
   U = E + I_a × R_a = 133.3 + 100 × 0.2 = 153.3V（电压不变，需串电阻R_s）
   实际：U_N = E + I_a(R_a + R_s)
   220 = 133.3 + 100(0.2 + R_s)
   R_s = (220 - 133.3)/100 - 0.2 = 0.667Ω
   
   输入功率：P₁ = U_N × I_a = 220 × 100 = 22000W
   输出功率：P₂ = E × I_a = 133.3 × 100 = 13330W
   效率：η₁ = 13330/22000 = 60.6%
   
   **方法二：降压调速**
   U = E + I_a × R_a = 133.3 + 100 × 0.2 = 153.3V
   
   输入功率：P₁ = U × I_a = 153.3 × 100 = 15330W
   输出功率：P₂ = 13330W
   效率：η₂ = 13330/15330 = 87.0%
   
   **方法三：弱磁调速**
   弱磁调速只能从基速向上调速，无法将转速从1500调至1000r/min（向下调速）。
   此方法不适用。
   
   **结论**：降压调速效率最高（87.0%），电枢串电阻调速效率最低（60.6%），弱磁调速不适用于向下调速。降压调速最优。

537. 一台单相变压器，S_N = 10kVA，U₁N/U₂N = 1000/400V。空载实验在低压侧测得：U₀ = 400V，I₀ = 2.5A，P₀ = 120W。短路实验在低压侧加压、高压侧短路测得：U_k = 50V，I_k = 25A，P_k = 200W。试求折算到高压侧的等效电路参数（r_m、x_m、r_k、x_k）。

   **解答**：
   变比：k = U₁N/U₂N = 1000/400 = 2.5

538. 试比较直流电动机、笼型异步电动机和同步电动机的起动方法、起动性能和适用场合，并分析各自的优缺点。

   **解答**：
   
   **一、直流电动机**
   - 起动方法：全压起动（小容量）、电枢串电阻分级起动（中大容量）、降压起动
   - 起动性能：起动电流大（可达10~20倍额定电流），需加以限制；起动转矩与电枢电流成正比，串电阻后可同时限制电流并获得较大起动转矩
   - 优点：起动转矩可以精确控制，起动性能好
   - 缺点：全压起动电流过大，需附加起动设备（电阻或电源）
   - 适用场合：需要大起动转矩和宽调速范围的场合（轧钢机、起重机等）

   - 起动方法：直接起动（电网容量允许时）、Y-Δ降压起动、自耦变压器降压起动、深槽/双笼型电动机

   - 缺点：起动电流大而起动转矩不大，降压起动时转矩更小
   - 适用场合：轻载或空载起动的通用场合（风机、水泵等）

   - 起动方法：异步起动法（借阻尼绕组/起动绕组产生异步转矩）+ 投励牵入同步；辅助电动机起动；变频起动
   - 起动性能：本身无起动转矩（靠阻尼绕组作异步起动），起动电流大，牵入同步过程复杂
   - 优点：运行时功率因数可调、转速恒定
   - 缺点：起动最复杂，不能带重载起动，需阻尼绕组或辅助设备
   - 适用场合：大功率、需恒速或改善功率因数的场合（大型压缩机、鼓风机等）

539. 一台三相笼型异步电动机，P_N = 7.5kW，U_N = 380V，I_N = 15A，n_N = 1450r/min，f = 50Hz，p = 2。采用Y-Δ降压起动，试求：

（2）Y接法起动转矩与Δ接法直接起动转矩之比；
（3）若直接起动电流为7I_N，求Y-Δ起动时的起动电流。

540. 试从能量转换角度综合分析直流电动机、异步电动机和同步电动机的功率平衡关系，比较三者的损耗构成及效率特点。

   **解答**：
   
   **一、直流电动机功率平衡**
   P₁（输入电功率）= P_em（电磁功率）+ p_Cuf（励磁铜耗）+ p_Cua（电枢铜耗）
   P_em = P₂（输出机械功率）+ p_mec（机械损耗）+ p_Fe（铁耗）+ p_s（附加损耗）
   效率：η = P₂/P₁
   
   损耗构成：电枢铜耗（随负载变化）、励磁铜耗（他励时不计入电枢回路）、铁耗（基本不变）、机械损耗（基本不变）、附加损耗
   
   **二、异步电动机功率平衡**
   P₁（输入电功率）= p_Cu1（定子铜耗）+ p_Fe（铁耗）+ P_em（电磁功率）
   P_em = p_Cu2（转子铜耗）+ P_mec（总机械功率）
   P_mec = P₂ + p_mec + p_s
   转差率关系：p_Cu2 = s × P_em
   
   损耗构成：定子铜耗、转子铜耗（=sP_em，随负载和转差率变化）、铁耗、机械损耗、附加损耗
   
   **三、同步电动机功率平衡**
   P₁（输入电功率）= p_Cu1（定子铜耗）+ p_Fe（铁耗）+ P_em（电磁功率）
   P_em = P₂ + p_mec + p_s
   
   损耗构成：定子铜耗、铁耗、机械损耗、附加损耗（无转子铜耗，因励磁由直流电源提供）
   
   **四、效率比较分析**
   
   | 特征 | 直流电动机 | 异步电动机 | 同步电动机 |
   |------|-----------|-----------|-----------|
   | 可变损耗 | 电枢铜耗 | 定转子铜耗 | 定子铜耗 |
   | 不变损耗 | 铁耗+机械耗 | 铁耗+机械耗 | 铁耗+机械耗 |
   | 转子侧损耗 | 无（机械侧） | 转子铜耗=sP_em | 励磁铜耗（另供） |
   | 最大效率条件 | 可变损耗=不变损耗 | 可变损耗=不变损耗 | 可变损耗=不变损耗 |
   
   - 大型同步发电机/电动机效率最高（可达98%以上），因结构大、散热好、相对损耗小
   - 中小型异步电动机效率约85%~95%
   - 直流电动机因换向器存在附加损耗，效率通常略低于同容量异步电动机
   - 异步电动机在轻载时效率下降明显（因不变损耗占比增大）
   - 变压器效率最高（无机械损耗和旋转损耗），可达99%以上

544. 试分析并励直流发电机自励建压的物理过程，说明三个自励条件的物理意义，并讨论当励磁回路电阻大于临界电阻时，为什么不能自励建压。

   **解答**：
   
   **一、自励建压的物理过程**
   
   1. 并励发电机的励磁绕组与电枢绕组并联，励磁电流由电枢自身供电。
   2. 起动时，电机主磁路存在剩磁Φ_rem，电枢旋转切割剩磁产生剩磁电势E_rem。
   3. E_rem在励磁回路中产生励磁电流I_f1，I_f1产生磁通Φ1与剩磁同方向叠加，使气隙磁通增大。
   4. 磁通增大使电枢电势升高为E1，E1又使I_f增大为I_f2，如此反复——这是一个正反馈自激过程。
   5. 随着I_f增大，磁路逐渐饱和，空载特性曲线弯曲，而励磁回路电阻线为直线，两者交点即为稳定工作点。
   
   **二、三个自励条件的物理意义**
   
   **条件1：主磁路有剩磁**
   - 物理意义：剩磁是自励的"种子"。没有剩磁，电枢电势为零，无法产生励磁电流，自励无从开始。
   - 若剩磁消失，需先用外部电源充磁恢复。
   
   **条件2：励磁绕组接线极性正确**
   - 物理意义：励磁电流产生的磁通必须与剩磁方向一致（正反馈），才能使磁通逐步增大。若接线反向，励磁磁通与剩磁抵消（负反馈），电势反而减小，无法建压。
   - 若接线错误，对调励磁绕组两端即可纠正。
   
   **条件3：励磁回路电阻小于临界电阻**
   - 物理意义：临界电阻R_cr等于空载特性曲线在原点处（剩磁附近）的切线斜率。当励磁电阻线斜率（R_f）小于空载特性切线斜率（R_cr）时，电阻线与空载特性曲线有交点（稳定工作点），可以自励建压。
   
   **三、R_f > R_cr时不能自励的分析**
   
   当励磁回路电阻R_f大于临界电阻R_cr时：
   - 励磁电阻线（斜率为R_f）比空载特性曲线在原点处的切线更陡。
   - 电阻线与空载特性曲线仅在原点附近相交，交点处电压极低（接近剩磁电压）。
   - 从剩磁电压开始，电枢电势E_a产生的励磁电流I_f对应的电阻线上的电压U = I_f × R_f始终大于空载电势E₀（电阻线在空载特性上方），即U > E₀，说明励磁回路"供不应求"，无法维持自激增长。
   - 物理上：R_f过大导致励磁电流太小，产生的磁通不足以使电势增长，正反馈无法建立，电压只能停留在剩磁电压附近。
   
   因此，R_f < R_cr是保证自励正反馈能够持续增长的必要条件。

   **（3）比较分析**
   最大效率97.09%出现在β = 0.5时，满载效率96.39%略低。
   变压器在铜耗等于铁耗时效率最高，此时β = √(p₀/p_kN) = 0.5，即半载运行最经济。
   设计中通常使p₀/p_kN在1/3~1/4之间，使最大效率出现在较高负载率，更符合实际运行工况。

546. 试综合分析异步电动机和同步电动机的机械特性，比较二者的形状特征、稳定运行区域和过载能力，并说明在负载变化时两种电动机转速响应的本质区别。

   **解答**：
   
   **一、异步电动机机械特性**
   
   异步电动机的机械特性n-T（或T-s）曲线特点：
   1. 曲线形状：从s=0（同步转速）开始，随s增大T先增大至最大转矩T_max（临界转差率s_m处），然后T随s增大而减小。
   2. 稳定运行区域：0 < s < s_m段为稳定运行区，机械特性斜率为负（dT/dn < 0），负载增大时转速略降但仍稳定。
   3. 不稳定区域：s_m < s < 1段为不稳定区，斜率为正，负载增大时转速继续下降直至停转。
   4. 过载能力：λ = T_max/T_N，一般约1.8~2.5。
   5. 额定运行点在稳定区，s_N很小（约0.02~0.06），转速接近同步转速。
   
   **二、同步电动机机械特性**
   
   同步电动机的机械特性：
   1. 曲线形状：正常励磁下为一条水平线（n = n₁ = 常数），转速不随负载变化。
   2. 功角特性：T = (m × U × E₀)/(Ω₁ × X_d) × sinδ + (m × U²)/(2Ω₁) × (1/X_q - 1/X_d) × sin2δ
   3. 稳定运行区域：0 < δ < 90°（隐极机）或0 < δ < δ_max（凸极机），功角在此范围内可稳定运行。
   4. 过载能力：λ = T_max/T_N = 1/sinδ_N，一般约2~3。
   5. 当负载转矩超过最大同步转矩时，电动机"失步"，不能维持同步运行。
   
   **三、转速响应的本质区别**
   
   | 特征 | 异步电动机 | 同步电动机 |
   |------|-----------|-----------|
   | 转速随负载变化 | 变化（s增大，n略降） | 不变（恒为n₁） |
   | 本质原因 | 转矩平衡依赖转差率产生转子电流 | 转矩平衡依赖功角变化 |
   | 负载增大时 | s↑→I₂↑→T↑至新平衡，n略降 | δ↑→T↑至新平衡，n不变 |
   | 超载时 | 逐渐减速至停转（"闷车"） | 突然失步，失去同步运行能力 |
   | 稳态转速 | n = n₁(1-s)，略低于n₁ | n = n₁，严格恒定 |
   
   **本质区别**：异步电动机通过转速差（转差率）自动调节电磁转矩与负载平衡，是"自适应"的；同步电动机通过功角变化调节转矩，转速始终不变，一旦超出最大功角则失步，是"刚性"的。

547. 在同步发电机空载/短路/零功率因数特性实验中，已知空载特性的气隙线和短路特性如下数据。空载气隙线：E₀ = 2.0 × I_f（V/A）；短路特性：I_k = 5.0 × I_f（A/A）。额定电压U_N = 200V。试求：
（1）直轴同步电抗不饱和值X_d（未饱和）；
（2）对应额定电压的励磁电流I_f0（按气隙线）；
（3）短路比K_c。

   **解答**：
   
   **（1）求X_d不饱和值**
   
   在空载气隙线上，对应任意I_f的空载电势：E₀ = 2.0 × I_f
   
   在短路特性上，同一I_f下的短路电流：I_k = 5.0 × I_f
   
   X_d（不饱和）= E₀ / I_k = (2.0 × I_f) / (5.0 × I_f) = 2.0/5.0 = 0.4Ω
   
   **（2）求对应额定电压的励磁电流I_f0**
   
   按气隙线：U_N = E₀ = 2.0 × I_f0
   I_f0 = U_N / 2.0 = 200 / 2.0 = 100A
   
   **（3）求短路比K_c**
   
   短路比定义：K_c = I_f0 / I_fk
   
   其中I_f0为产生额定空载电压所需的励磁电流（按气隙线），I_fk为产生额定短路电流所需的励磁电流。
   
   额定短路电流I_kN对应的励磁电流：
   I_kN = 5.0 × I_fk → I_fk = I_kN / 5.0
   
   但短路比的标准定义中：
   K_c = I_f0(空载额定电压) / I_fk(短路额定电流)
   
   其中I_fk为使短路电流等于额定电流I_N所需的励磁电流。
   
   假设额定电流I_N = 500A（需补充数据）：
   I_fk = I_N / 5.0 = 500/5.0 = 100A
   
   K_c = I_f0 / I_fk = 100/100 = 1.0
   
   若无额定电流数据，短路比可表示为：
   K_c = I_f0 / I_fk = (U_N/2.0) / (I_N/5.0) = 5.0 × U_N / (2.0 × I_N) = 2.5 × U_N / I_N

548. 试从实验角度综合论述：如何通过空载实验和短路实验的数据处理，分别求取变压器和异步电动机的等效电路参数？比较两种电机实验方法的异同，并分析实验误差的主要来源。

   **解答**：
   
   **一、变压器空载与短路实验求参数**
   
   **空载实验**（低压侧加额定电压，高压侧开路）：
   - 测取：U₀、I₀、P₀
   - 计算：Z_m = U₀/I₀，r_m = P₀/I₀²，x_m = √(Z_m²-r_m²)
   - 物理依据：空载时I₂=0，I₁≈I_m（励磁电流），P₀≈p_Fe（铁耗）
   - 可忽略：一次侧漏阻抗压降（因I₀很小）
   
   **短路实验**（低压侧加低电压使电流达额定值，高压侧短路）：
   - 测取：U_k、I_k、P_k
   - 计算：Z_k = U_k/I_k，r_k = P_k/I_k²，x_k = √(Z_k²-r_k²)
   - 物理依据：短路时铁耗可忽略（电压很低），P_k≈p_Cu（铜耗）
   - 近似：r₁ = r₂' = r_k/2，x₁ = x₂' = x_k/2
   
   **二、异步电动机空载与短路实验求参数**
   
   **空载实验**（加额定电压，电机空载运转）：
   - 测取：不同电压下的U₀、I₀、P₀
   - 数据处理：作P₀-U₀²曲线，延长至U₀=0处截距得p_mec（机械损耗），P₀-p_mec得p_Fe
   - 计算：Z_m = U₀/I₀，分离出r_m和x_m
   - 与变压器区别：异步电机空载时有机械损耗p_mec，需分离处理
   
   **短路（堵转）实验**（转子堵住，加低电压使电流达额定值）：
   - 测取：U_k、I_k、P_k
   - 计算：Z_k = U_k/I_k，r_k = P_k/I_k²，x_k = √(Z_k²-r_k²)
   - 分离：r₂' = r_k - r₁，x₁ = x₂' = x_k/2（需已知r₁由直流电阻测量得到）
   
   **三、异同比较**
   
   | 项目 | 变压器 | 异步电动机 |
   |------|--------|-----------|
   | 空载实验加压侧 | 低压侧 | 定子侧 |
   | 空载时损耗 | 仅铁耗 | 铁耗+机械损耗 |
   | 短路实验方法 | 电气短路（绕组短接） | 机械堵转（转子固定） |
   | 短路时损耗 | 铜耗 | 铜耗（铁耗可忽略） |
   | 参数分离 | 励磁支路+短路支路 | 励磁支路+短路支路 |
   | 需要分离机械损耗 | 否 | 是 |
   | 需要测直流电阻 | 可选 | 必须（分离r₁和r₂'） |
   
   **四、实验误差主要来源**
   
   1. **仪表误差**：电压表、电流表、功率表本身的精度限制；功率表在低功率因数下误差增大（空载实验cosφ₀很低，约0.1~0.3）。
   
   2. **方法误差**：
      - 变压器空载实验忽略了漏阻抗压降；
      - 异步电动机空载实验中机械损耗的分离依赖线性外推，但P₀-U₀²在低压段可能非线性；
      - 短路实验中假设铁耗为零是近似处理。
   
   3. **温度误差**：绕组电阻随温度变化，短路实验测得的r_k对应实验时温度，需换算到基准工作温度（通常75°C）。
   
   4. **电源误差**：电源电压波动、三相不对称、谐波含量等影响测量结果。
   
   5. **磁滞误差**：空载特性受磁滞影响，上升和下降分支不重合，取值有偏差。
   
   6. **转速误差**（异步电机）：堵转实验中若转子未完全固定，微小转动会导致测量误差。
