## Prompts

### Prompt 1: Customer Story Narrative
**File:** `prompts/01_customer_narrative.md`  
Writes the full case study narrative: customer background, the challenge, why they chose you, how the solution was implemented, and the story arc. 700–1,000 words. Structured for maximum credibility with a human voice.

### Prompt 2: Results & Metrics Section
**File:** `prompts/02_results_metrics.md`  
Produces the "Results" section with hard numbers, percentage improvements, and before/after comparisons. Includes data visualization suggestions (chart types, callout stats) and an ROI calculation table. B2B buyers make decisions on this section — it has to be specific.

### Prompt 3: Pull Quotes & Testimonial Extraction
**File:** `prompts/03_pull_quotes.md`  
Generates 5 pull quotes from the customer's perspective — each targeting a different buyer objection or stage (awareness, consideration, decision). Also produces a 3-sentence email testimonial and a one-sentence social proof snippet for ad copy. These are the lines your sales team will paste everywhere.

### Prompt 4: Multi-Format Package
**File:** `prompts/04_multi_format.md`  
Packages the full case study into four ready-to-publish formats:
- **Blog post** (SEO-optimized, 600–800 words)
- **Sales one-pager** (PDF-ready, 400 words, structured for print)
- **LinkedIn post** (300–400 words, story hook format)
- **Email snippet** (150 words, for sales outreach sequences)

## How to Use

Run prompts 1–4 sequentially. Each builds on the inputs from the last. You can stop after any prompt if you only need that output format.

### Required Inputs

Provide these details when running Prompt 1:

| Field | Example |
|-------|---------|
| Company name | Acme Corp |
| Industry | B2B SaaS / Healthcare / Manufacturing |
| Customer size | 45-person software agency |
| The problem they had | Losing 8 hours/week to manual client reporting |
| Your product/service | ReportLab (automated reporting platform) |
| How you solved it | Automated weekly client report generation |
| Key results (metrics) | 8 hrs/week saved, 3 new clients onboarded with time freed |
| Timeline | 3 months to full ROI |
| Customer contact | Jane Smith, VP of Operations |
| Quote if available | "We used to dread Fridays. Now reports just appear." |
