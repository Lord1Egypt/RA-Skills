## ⚠️ 重要声明

> 本工具仅供健康科普和参考，不能替代专业医生的诊断和治疗。如有任何健康问题，请及时就医。

## 🎯 功能介绍

这是一个专门帮助普通用户理解医学检验报告的工具，将专业术语转化为通俗易懂的语言。

### 支持的检验项目

| 类别 | 包含项目 |
|------|---------|
| 血常规 | 白细胞、红细胞、血小板、血红蛋白等 |
| 肝功能 | ALT、AST、胆红素、白蛋白等 |
| 肾功能 | 肌酐、尿素氮、尿酸等 |
| 血脂 | 总胆固醇、甘油三酯、HDL、LDL等 |
| 血糖 | 空腹血糖、餐后血糖、糖化血红蛋白 |
| 甲状腺 | T3、T4、TSH等 |

## 📖 使用示例

### 示例1：血常规解读
```
用户：帮我看看这份血常规报告
助手：
• 白细胞计数：6.5 ×10⁹/L（正常范围4-10）
• 红细胞计数：4.8 ×10¹²/L（正常范围4.3-5.8）
• 血红蛋白：145 g/L（正常范围130-175）
• 结论：各项指标在正常范围内
```

### 示例2：指标咨询
```
用户：我的甘油三酯是2.3，高吗？
助手：甘油三酯正常范围是<1.7 mmol/L，您的数值偏高...
```

## 💡 使用技巧

1. **描述要清晰** - 说明检验项目的完整名称
2. **提供数值** - 带上具体的检测数值
3. **注明单位** - 单位信息有助于准确判断
4. **说明背景** - 提供是否空腹、有无症状等信息
