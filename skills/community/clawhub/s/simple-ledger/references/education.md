# 财务知识科普

> 不讲高深理论，只讲跟日常记账相关的基础概念。
> 看完能理解"为什么这么记"，而不是死记硬背。

## 1. 单式记账：为什么选单式？

### 单式记账（本 Skill 采用的方式）

**一句话解释：花了一笔就记一笔，像流水账。**

```
2026-05-21  微信钱包  -50.00  餐饮:饮食  午餐
```

**特点：**
- ✅ 简单直觉：花了 50 就写 -50
- ✅ 学习成本几乎为零
- ✅ 适合日常"随手记"
- ❌ 无法自动校验是否漏记
- ❌ 不适合企业财务或复杂资产跟踪

### 传统双行记账

**一句话解释：每笔交易至少影响两个账户，有借必有贷，借贷必相等。**

比如"午餐花了 50"：
```
2026-05-21 * 午餐
  Expenses:餐饮:饮食        50.00 CNY
  Assets:微信钱包          -50.00 CNY
```

**特点：**
- ✅ 严谨：每笔交易都平衡，不会出现"钱从哪来"的漏洞
- ✅ 强大的报表能力：自动计算资产负债表、利润表
- ✅ 适合复杂场景（多币种、投资组合、税务）
- ❌ 学习门槛较高
- ❌ 日常记账"杀鸡用牛刀"

### 为什么 personal-ledger 选单式？

| 对比 | 单式记账 | 传统双行记账 |
|------|---------|---------|
| 记一笔午餐 | 1 行 | 3 行 |
| 需要知道的概念 | 金额、分类 | 借方、贷方、账户类型、货币 |
| 记错了怎么办 | 删掉重来 | 可能导致全局不平衡 |
| 适合谁 | 想随手记钱的普通人 | 需要专业财务报表的人 |

**结论：** 对于"知道钱花哪了"这个目标，单式记账完全够用。

## 2. 资产 vs 负债 vs 净资产

### 资产（你拥有的、值钱的东西）

- 银行卡里的存款 ✓
- 微信/支付宝里的钱 ✓
- 现金 ✓
- 你名下的房产 ✓
- 投资账户里的基金/股票 ✓

### 负债（你欠别人的）

- 信用卡欠款 ✓
- 花呗/白条 ✓
- 房贷 ✓
- 车贷 ✓
- 借别人的钱 ✓

### 净资产（你"真正"有多少）

```
净资产 = 资产 − 负债
```

**举例：**
- 银行卡有 5 万，微信有 3 千，现金 5 百 → 资产 = 53,500
- 信用卡欠 2 千，花呗欠 5 百 → 负债 = 2,500
- 净资产 = 53,500 − 2,500 = **51,000**

**一个容易混淆的点：**
信用卡里的钱不算资产！信用卡额度是"你能借多少"，不是"你有多少"。
只有你还款之后，银行卡里减少的那部分才是支出。

### 在记账中怎么体现？

本 Skill 的处理方式：
- **资产类账户**：微信钱包、支付宝、银行卡、现金 → 记余额变化
- **负债类账户**：信用卡、花呗 → 支出时记到负债账户，还款时从资产转到负债
- **净资产**：可通过"所有资产账户余额之和 − 所有负债账户余额之和"计算

## 3. 现金流管理基础

### 什么是现金流？

**现金流 = 钱进来的速度 − 钱出去的速度。**

通俗地说：你每个月进账多少，出账多少，中间的差额就是现金流。

### 现金流的三种状态

```
收入 > 支出  →  正向现金流 → 钱越来越多 → 健康 ✅
收入 = 支出  →  零现金流  → 没有积累   → 需要注意 ⚠️
收入 < 支出  →  负向现金流 → 钱越来越少 → 危险 ❌
```

### 为什么现金流比"赚多少钱"重要？

一个人月薪 2 万但每月花 2 万 1 千 → 每个月在"变穷"
一个人月薪 8 千但每月只花 5 千 → 每个月在"变富"

**赚多少决定了上限，花多少决定了实际积累。**

### 如何改善现金流？

**开源（增加收入）：**
- 提升技能涨薪（最可持续）
- 兼职 / 副业
- 出租闲置资源
- 理财收益（长期）

**节流（减少支出）：**
- 记账 → 发现不该花的钱
- 区分"需要"和"想要"
- 找更便宜的替代方案
- 减少冲动消费（等 24 小时再买）

**本 Skill 帮你做的是"节流"部分：** 通过记账让你看清支出，从而做出更好的消费决策。

## 4. 预算为什么重要？

### 没有预算的生活

```
月初：感觉有很多钱
月中：也没怎么花啊？
月底：钱呢？？？
```

### 有预算的生活

```
月初：这个月餐饮预算 2000，购物预算 1000
月中：餐饮已经花了 1500，还剩 500，控制一下外卖
月底：餐饮 1900，达标！购物 800，省了 200
```

### 预算的本质

**预算不是限制你花钱，而是让你主动选择钱花在哪。**

没有预算时，钱是被"被动消费"花掉的（广告、冲动、习惯）。
有预算时，你把钱分配给"真正重要的东西"。

### 怎么开始做预算？

**第一步：先记账 1–2 个月，搞清楚现状**
（别急着做预算，你都不知道现在花多少怎么做预算？）

**第二步：按分类设定月度预算上限**

参考"财务基准参考"中的占比，结合自己的实际情况设定。

**第三步：月底对比实际 vs 预算**

本 Skill 的月度报告会帮你做这个对比（未来版本会支持预算功能）。

### 预算的常见误区

| 误区 | 正确做法 |
|------|---------|
| "预算越低越好" | 预算要现实可行，否则坚持不下来 |
| "所有分类都要设预算" | 先管好 TOP 3 大头（通常是居住、餐饮、购物） |
| "超预算就是失败" | 超预算是正常的，关键是知道超了多少、为什么超 |
| "预算一定不能变" | 每月根据实际情况调整，预算是活的 |

## 5. 应急基金的重要性

### 什么是应急基金？

**一笔专门用来应对突发事件的存款。** 不投资、不消费、不动它，除非真有急事。

### 什么算"急事"？

- ✅ 突然失业，需要几个月的生活费
- ✅ 生病住院，自费部分
- ✅ 家里突然需要大额维修
- ✅ 必须的紧急出行
- ❌ 手机出了新款想换
- ❌ 朋友借钱（除非是真正的紧急情况）
- ❌ 双十一打折

### 需要多少？

| 安全等级 | 金额 | 适合谁 |
|---------|------|--------|
| 最低保障 | 1 个月支出 | 刚开始攒钱，有个底 |
| 基本安全 | 3 个月支出 | 大多数人的目标 |
| 比较安心 | 6 个月支出 | 有房贷/家庭责任的人 |
| 非常安心 | 6–12 个月支出 | 自由职业/收入不稳定 |

**怎么算 1 个月的支出？**
看你最近一个月的月度报告里的"总支出"数字，或者把最近 3 个月的支出平均一下。

### 放在哪里？

- **活期存款 / 货币基金**：随时能取，收益虽然低但安全性最高
- ❌ 不要放股票、基金（急用钱时可能正好在亏）
- ❌ 不要放定期（提前取会损失利息）

### 怎么攒？

1. 每个月发工资后，**先把应急基金的钱转出来**，剩下的才是可以花的
2. 从小目标开始：先攒 1 个月支出，再攒 3 个月
3. 记账帮你控制支出，省下的钱就往应急基金里存

### 为什么记账跟应急基金有关？

**因为记账让你知道：**
- 你每个月实际支出多少（决定了应急基金的目标金额）
- 你每个月能存多少（决定了多久能攒够）
- 你的支出里哪些可以砍（加速攒钱）
- 你的消费趋势是否在恶化（提前预警）

---

> **总结：** 记账是所有财务管理的第一步。你不需要懂投资、不需要懂税务，先把"钱去哪了"这个问题回答清楚，其他的都是后面的事。
