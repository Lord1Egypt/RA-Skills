# Jin Ye Metabolism Analysis

## When to Use
当患者出现以下任一表现时启用本技能：
- 口干、皮肤干燥、尿少（提示伤津）
- 目陷、骨瘦、神昏（提示脱液）
- 水肿、痰饮、多汗、大便溏泄（提示水湿内停）
- 中医养生调摄中需评估水液代谢状态

> 注意：本技能不适用于单纯电解质紊乱等西医急症处理，必须结合整体辨证。

## Execution Steps

### 1. 判断津液生成是否正常
- **关键脏腑**：胃、小肠、大肠、脾
- **检查要点**：
  - 胃能否受纳腐熟水谷？
  - 小肠是否泌别清浊？
  - 脾是否有效运化水谷精微以生成津液？

### 2. 分析津液输布通路
- **脾**：是否将津液上输于肺或布散全身？
- **肺**：宣发（至体表上部）与肃降（至肾与膀胱）功能是否协调？
- **肾**：是否通过气化蒸腾推动全身水液运行？能否将清者上输、浊者下注？
- **肝**：疏泄功能是否正常？“气行则津行”是否受阻？
- **三焦**：水道是否通利？

### 3. 评估津液排泄途径
- **汗液**：肺宣发异常可致无汗或多汗
- **呼气水分**：肺气失宣可影响
- **尿液**：肾气化不足则尿少或癃闭
- **粪便水分**：大肠主津功能失调可致便干或泄泻

### 4. 辨别津伤与液脱
- **伤津**：以失水为主，症见口渴、尿少、皮肤干
- **脱液**：兼损精微，症见目眶凹陷、形体消瘦、神志异常
- 记住：“伤津未必脱液，脱液必兼津伤；脱液重于伤津”

### 5. 输出调治方向
根据失衡环节提示具体病机，例如：
- 脾不运湿 → 健脾化湿
- 肺失宣降 → 宣肺化痰
- 肾不气化 → 温阳利水

> 最终输出应明确指出津液代谢障碍的具体环节及对应脏腑，为后续辨证论治提供依据。
