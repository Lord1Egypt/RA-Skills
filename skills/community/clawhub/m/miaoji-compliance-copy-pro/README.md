# 秒技·合规文案Pro版

## 快速开始

### 基本使用

1. 打开技能
2. 输入产品和目标市场
3. 选择优化目标
4. 获取合规文案方案

### 输入示例

```
目标市场：多市场（美国+欧盟）
产品品类：化妆品
产品名称：XXX精华液
当前Listing文案：[粘贴文案]
优化目标：Rufus适配+多市场监管
```

## 功能概览

### 多市场监管方案
获取：
- 美国FDA要求清单
- 欧盟CE/GDPR要求清单
- 日本PSE要求清单
- 差异化合规建议

### 法律风险评估
获取：
- 文案风险绿/黄/红分级
- 替换建议方案
- 风险应对策略

### Rufus深度优化
获取：
- 标题重构建议
- Bullet Points优化
- 30+条问答内容

### 季节性文案模板
获取：
- Q4促销合规文案
- Prime Day专属模板
- 节日节点提醒
