## Description: <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, technical writers, and repository maintainers use this skill to turn pasted tree or find output, or manually described folder structures, into single-file interactive HTML directory views for documentation, audits, and storage analysis. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated directory views may expose sensitive file or folder names from user-provided directory listings. <br>
Mitigation: Review and redact usernames, absolute paths, internal repository names, customer or project identifiers, .env or secret-related filenames, and other sensitive metadata before sharing the generated HTML. <br>
