# Project Lifecycle Navigator

## Short Description

A bilingual English/Chinese project lifecycle Skill for non-technical users and AI Coding Agent workflows. It automatically routes users into new project intake, mid-project realignment, or code review and upgrade planning.

## Best For

- Starting a new software, AI tool, internal system, or automation project without knowing how to write full requirements.
- Recalibrating a project that feels off-track, bloated, overbuilt, or overloaded with new ideas.
- Reviewing an existing codebase for security, dead code, performance, architecture, testing, dependencies, and upgrade planning.
- Turning user answers and project reviews into plans an AI Coding Agent or engineer can execute.

## Core Capabilities

1. **New Project Intake**  
   Uses staged questions to clarify users, core problems, MVP scope, data sources, AI needs, technical approach, and development plan.

2. **Mid-Project Realignment**  
   Identifies goal drift, scope creep, over-engineering, MVP failure, unclear user scenarios, sunk costs, and whether to continue, narrow, pivot, archive, or restart.

3. **Code Review & Upgrade Plan**  
   Reviews existing code for security, clutter, logic, performance, architecture, testability, dependencies, and config issues, then produces an executable, verifiable, rollback-aware fix plan.

## Trigger Examples

```text
I want to build a new project, but I am not technical. Ask me questions first.
```

```text
The project is halfway done and feels off-track. Help me recalibrate the direction.
```

```text
I have a codebase. Review it for vulnerabilities, dead code, architecture issues, and generate an upgrade plan.
```

## Safety Note

This is an instruction-only Skill:

- No code execution
- No API keys
- No environment variables
- No third-party package installation
- No network permissions
- No scripts
