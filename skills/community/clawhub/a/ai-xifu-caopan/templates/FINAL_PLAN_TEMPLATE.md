### 26.3 策略操作细则

**趋势跟踪策略(适用于强趋势市):**
```
入场:价格突破20日高点→做多;跌破20日低点→做空
止损:反向突破10日高低点
仓位:总资金的60%,分批开仓(首次40%,确认后加至60%)
适用品种:沪铜CU(外盘联动强,趋势性强)
```

**均值回归策略(适用于震荡市):**
```
入场:RSI<30→超卖买入;RSI>70→超买卖出
布林带:触及下轨+RSI<30→做多;触及上轨+RSI>70→做空
止损:突破前高/前低
仓位:总资金的40%,单次入场20%
```

**统计套利策略(适用于高波动率/跨品种):**
```
沪铜 vs LME铜:关注比价关系,偏离均值2倍标准差时开仓
近月 vs 远月:月间价差>1000元/吨时考虑跨期套利
仓位:总资金的20~30%,风险较低
```

**事件驱动策略(适用于数据公布日/政策窗口):**
```
美国非农/CPI公布前:降低仓位,避免持仓过夜
数据公布后:等待15分钟,方向明确后再入场
政策利好/利空:第一时间跟随,或等回踩确认
仓位:不超过20%,快进快出
```
