# DMAIC 流程改进向导

交互式六西格玛 DMAIC 流程改进助手，通过一问一答的方式引导用户完成完整的 DMAIC 项目。

## 📖 简介

DMAIC（定义-测量-分析-改进-控制）是六西格玛方法论的核心流程改进框架。本技能通过交互式对话，帮助用户系统化地完成 DMAIC 项目的每个阶段。

## ✨ 核心特性

- **🎯 五阶段流程**：定义 → 测量 → 分析 → 改进 → 控制
- **🛠️ 31个工具**：每个阶段支持多选工具，包括项目章程、SIPOC图、鱼骨图、FMEA等
- **💬 一问一答**：逐步引导，用户卡壳时提供智能引导
- **👨‍🏫 专家审核**：每个工具完成后由3位专家（质量、流程、客户）审核
- **📄 文档生成**：自动生成详细文档和完整项目报告

## 🚀 使用方法

### 安装

1. 下载 `dmaic-process-improvement.skill` 文件
2. 将文件解压到 OpenClaw 的技能目录：
   - Windows: `%USERPROFILE%\.qclaw\skills\`
   - macOS/Linux: `~/.qclaw/skills/`

### 触发方式

说以下任一触发词即可启动：
- "启动 DMAIC"
- "我想做流程改进"
- "质量分析"
- "六西格玛"

## 📊 工具清单

### 定义阶段（9个工具）
1. 项目章程
2. 利益相关者分析
3. SIPOC 图
4. 宏观流程图
5. 树图 / CTQ 树
6. VOC（客户之声）
7. KANO 模型
8. 亲和图
9. SWOT 分析

### 测量阶段（5个工具）
1. 价值流图（VSM）
2. 数据收集计划（DCP）
3. 测量系统分析（MSA）
4. 流程能力计算（Cp/Cpk, DPMO, Sigma水平）
5. 其他（用户自定义）

### 分析阶段（6个工具）
1. 5Why 分析法
2. 鱼骨图（因果图）
3. Pareto 分析（80/20规则）
4. 散点图 / 相关性分析
5. 假设检验（t检验、卡方等）
6. 其他（用户自定义）

### 改进阶段（5个工具）
1. 头脑风暴（列出对策）
2. 方案评估矩阵（利弊打分）
3. 试点设计（PDSA循环）
4. FMEA（失效模式与效果分析）
5. 其他（用户自定义）

### 控制阶段（6个工具）
1. 控制计划（Control Plan）
2. 统计过程控制（SPC控制图）
3. 标准化作业（SOP更新）
4. 培训与移交计划
5. 反应计划（OCAP）
6. 其他（用户自定义）
