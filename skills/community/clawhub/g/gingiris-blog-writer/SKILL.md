# SEO Blog Writer — E-E-A-T Content System

> Write blog posts that rank on Google AND get cited by ChatGPT, Perplexity, and Claude.

---

## Core Writing System: QAE Pattern

Every section follows the **Question → Answer → Evidence** pattern:

```
## [H2: Natural language question people actually search]

[Direct answer in 1-2 sentences — this is what AI engines extract]

[Evidence: data, case study, personal experience, expert quote]

[Key Stats table or comparison matrix if applicable]
```

### Why QAE Works

1. **For Google**: H2 matches search queries → featured snippet potential
2. **For AI Search**: Direct answer paragraph → high citation probability
3. **For Readers**: Clear structure → lower bounce rate → ranking signal

## Founder Voice Injection Rules

To pass E-E-A-T signals and avoid "AI slop" detection:

1. **Use first person**: "I tested", "We found", "In my experience"
2. **Include specific numbers**: "After 3 months, our traffic grew 47%"
3. **Admit limitations**: "This didn't work for us because..."
4. **Reference timeline**: "Back in Q3 2025, when we first tried..."
5. **Name real tools you used**: "We tracked this in Ahrefs/Mixpanel/..."
6. **Show behind-the-scenes**: Screenshots, dashboards, actual emails

## SEO + GEO Optimization Checklist

Before publishing, verify:

- [ ] Title contains primary keyword naturally
- [ ] First paragraph directly answers the search intent
- [ ] At least one Key Stats table per article
- [ ] FAQ section with 3-5 questions (for FAQ Schema)
- [ ] Comparison table if the topic involves alternatives
- [ ] Author byline with real credentials and links
- [ ] Internal links to 2-3 related articles
- [ ] External links to 2-3 authoritative sources
- [ ] Publication date visible
- [ ] Meta description includes CTA and keyword

## Cross-Posting Strategy

After publishing on your main blog:
1. **Dev.to** — Republish with `canonical_url` pointing to original
2. **Hashnode** — Same, with canonical
3. **Medium** — Import with canonical (Settings → Import)
4. **LinkedIn Article** — Shorter version with link back
5. **Twitter/X Thread** — Key takeaways as a thread

> ⚠️ Always set canonical URL to your original post to avoid duplicate content penalties.
