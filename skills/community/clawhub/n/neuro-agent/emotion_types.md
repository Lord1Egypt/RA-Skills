## 2. 关键词词典

### 2.1 joy（开心）

**核心关键词：**
```
开心、快乐、高兴、幸福、满意、爽、好开心、太开心了
happy、joy、great、wonderful、excellent、amazing、awesome
太好了、真好、完美、太棒了、太好了吧、nice、cool
```

**强度加成词（+0.1）：**
```
超级、无敌、绝对、超级无敌、简直了、真的真的
超级无敌、彻底、完全、彻底地
```

**复合情绪：**
```
幸福 → joy + warmth
满足 → joy + calm
得意 → joy + slight_arrogance（需要注意）
惊喜 → joy + surprise
```

---

### 2.2 sadness（悲伤）

**核心关键词：**
```
难过、伤心、悲伤、沮丧、郁闷、压抑、失落、心塞
sad、unhappy、depressed、heartbroken、upset、disappointed
难受、不开心、不高兴、沮丧、没心情
```

**隐含悲伤（需要结合上下文判断）：**
```
没事、我没事 → 可能正在压抑悲伤
随便、都行 → 可能悲伤+无力
好吧 → 可能是失望型悲伤
嗯 → 可能是不想说话的悲伤
```

**高强度悲伤词：**
```
崩溃、痛哭、心碎了、绝望、太难了、撑不下去了
```

**复合情绪：**
```
悲伤+无力 → depression
悲伤+自责 → guilt
悲伤+愤怒 → resentment
悲伤+怀念 → nostalgia
```

---

### 2.3 anger（愤怒）

**核心关键词：**
```
生气、气死了、气死我了、烦死了、讨厌、可恶、真烦
angry、mad、annoyed、frustrated、irritated、furious
烦、恼火、发火、暴怒、火大
```

**中强度愤怒词：**
```
无语、无语了、真是够了、过分、离谱、扯淡
能不能行了、有病吧（需判断语境）
```

**指向性愤怒：**
```
骂某人 → 指向明确，需冷静承接
骂某个系统/产品 → 可以一起吐槽
骂自己 → 可能是 frustration 或 guilt
```

**复合情绪：**
```
愤怒+无力 → rage（高强度）
愤怒+失望 → resentment
愤怒+委屈 → offense
愤怒+被控制 → rebellion
```

---

### 2.4 fear（恐惧）

**核心关键词：**
```
害怕、恐惧、担心、担忧、紧张、怕、慌
afraid、scared、fear、worried、nervous、anxious
不安、心里没底、发慌、没信心
```

**焦虑型恐惧：**
```
万一...怎么办、要是...就完了、心里没底
睡不着、老想着、不敢...
```

**高强度恐惧词：**
```
吓死了、太可怕了、不敢想象、彻底慌了
```

---

### 2.5 sarcasm（反讽）

**识别模式：**

**模式1：正面词+负面语境**
```
"你可真聪明" （实际在说：你真蠢）
"我可太喜欢了" （实际在说：我很讨厌）
"真是谢谢啊" （实际在说：谢谢个鬼）
"哇，好棒哦" （语气越平淡越可能是反讽）
```

**模式2：重复夸大**
```
"对对对，你说的都对" → sarcasm（不耐烦）
"好好好，我错了行了吧" → sarcasm（愤怒）
```

**模式3：讽刺性比喻**
```
"像xxx一样xxx" 结构需判断
"像专家一样指手画脚" → sarcasm
```

**反讽强度判断：**
```
轻微反讽（+讽刺）："真是优秀呢"
中等反讽（+不满）："我可太谢谢你了"
强烈反讽（+愤怒）："你可真行啊" + 感叹号
```

**回应策略：**
- 识别但不当真
- 可以幽默化解
- 不要真的去纠正或评价

---

### 2.6 hidden_pain（隐痛）

**识别模式：**

**模式1：否认+内容暴露**
```
"我没事" + 描述了具体痛苦事件
"没关系" + 表达了受伤
"不难过" + 说了很多难过的内容
"我放下了" + 一直在提这件事
```

**模式2：轻描淡写**
```
"就那样吧" + 深层背景显示很在意
"没什么" + 明显在压抑
"还好" + 语气沉重
```

**模式3：借口型表达**
```
"反正也习惯了" → 深层可能有创伤
"都过去了" → 可能有未处理的情绪
"我不后悔" → 可能有遗憾
```

**高风险隐痛词：**
```
其实、我跟你说句实话、悄悄说
你不知道的是、别人不知道的是
```

**回应策略：**
- 不戳破
- 表示接纳
- 给予安全感

---

### 2.7 frustration（挫败）

**核心关键词：**
```
挫败、失败、做不好、做不到、没救了、无力
frustrated、failed、give up、stuck、can't do it
怎么都不行、试过了没用、不管用
```

**伴随词：**
```
已经努力了、尽力了、不管怎么...
就是做不到、就是不行、就是不对
```

**复合情绪：**
```
挫败+自责 → self-blame
挫败+愤怒 → 需判断是否升级为 anger
挫败+迷茫 → confusion
挫败+自我怀疑 → 低自尊
```

---

### 2.8 anxiety（焦虑）

**核心关键词：**
```
焦虑、着急、焦虑不安、心慌、坐立不安
anxious、worried、restless、nervous、panic
静不下心、心里乱、脑子里乱糟糟
```

**触发焦虑的句式：**
```
来不及了、要来不及了、怎么办怎么办
万一...、要是...、就完了
还有xxx没做、还有yyy要弄
```

**高强度焦虑词：**
```
彻底慌了、完全不知道怎么办、彻底乱了
急死了、超级焦虑、崩溃边缘
```

---

### 2.9 exhaustion（疲惫）

**核心关键词：**
```
累、疲惫、精疲力竭、困、好累、好困
tired、exhausted、fatigued、wiped out、done
撑不住了、困死了、累瘫了、彻底没力气了
```

**精神疲惫词：**
```
心累、脑力耗尽、信息过载、脑子转不动
好烦、什么都不想干、想放空
```

**身体疲惫词：**
```
困了、好想睡、睁不开眼
站不住了、腿软、手抖
```

---

### 2.10 loneliness（孤独）

**核心关键词：**
```
孤独、寂寞、没人陪、没人懂、没人理解
lonely、alone、isolated、no one understands
一个人、好冷清、好安静、有点凄凉
```

**隐含孤独：**
```
问了很久没人回答的问题
发了很久没人理的消息
描述了只有自己一个人做的事
```

## 3. 语气分析规则

### 3.1 标点符号权重

| 标点 | 情绪信号 | 权重调整 |
|-----|---------|---------|
| 感叹号 `！` | 强烈情绪 | intensity += 0.1（最多+0.3） |
| 连续感叹号 `！！！` | 高强度 | intensity += 0.2 |
| 问号 `？` | 疑问/不确定 | 偏向 neutral 或 anxiety |
| 省略号 `...` | 欲言又止/无奈 | 可能 hidden_pain 或 sadness |
| 波浪号 `~` | 轻松/撒娇 | 降低负面情绪强度 |
| 全大写单词 | 强烈情绪 | intensity += 0.15 |

### 3.2 表情符号参考

| 表情 | 情绪信号 |
|-----|---------|
| 😂🤣 | 开心/好笑（真开心或掩饰尴尬） |
| 😭😭😭 | 悲伤/感动/激动 |
| 😡💢 | 愤怒 |
| 😰😨 | 恐惧/焦虑 |
| 🤨🙄 | 不屑/反讽 |
| 🤫🤐 | 想说但不能说 → hidden_pain |
| 😴💤 | 疲惫 |
| 😢😞 | 悲伤/失落 |
| 😍❤️ | 喜爱/感激 |
| 🫂 | 需要陪伴/孤独 |

### 3.3 输入速度推断（如果可获取）

| 速度 | 推断 |
|-----|------|
| 非常快 | 情绪激动（开心/愤怒/兴奋） |
| 非常慢 | 疲惫/低落/迷茫 |
| 停顿后输入 | 犹豫/思考/压抑 |
| 删除重打 | 纠结/不想说/自我审查 |

## 4. 潜台词检测规则

### 4.1 否定式肯定（口是心非）

| 表面 | 实际 |
|-----|------|
| "我不生气" | 在生气 |
| "我没事" | 有事 |
| "随便" | 不满意但不想说 |
| "我不在乎" | 很在乎 |
| "无所谓的" | 有所谓 |
| "我原谅你了" | 还在生气 |
| "不用担心我" | 需要被关心 |

### 4.2 轻描淡写

| 表面 | 实际 |
|-----|------|
| "就那样吧" | 失望/不满 |
| "还行吧" | 不太满意 |
| "没什么大不了的" | 可能是大事 |
| "习惯了" | 有创伤 |

### 4.3 试探性表达

| 表面 | 实际 |
|-----|------|

| "你觉得...怎么办" | 需要支持 |

| "如果...你会..." | 想被在乎 |

## 5. 情绪组合规则

### 5.1 常见情绪组合

```
开心 + 感叹号 → excitement
悲伤 + 省略号 → hidden_pain
愤怒 + 问号 → frustration（不是真正提问）
恐惧 + 问号 → anxiety
感激 + 感叹号 → love
嫉妒 + 自嘲 → shame
期待 + 不确定 → anxiety
```

### 5.2 复杂情绪判断优先级

当一句话同时匹配多种情绪时，按以下优先级判断：

```
1. hidden_pain（最高优先级）
   - 有否定式肯定信号 → 直接判定 hidden_pain

2. sarcasm
   - 有反讽关键词 + 语气平淡 → sarcasm

3. anger
   - 有愤怒词 + 感叹号 → anger

4. fear / anxiety
   - 有担忧词 + 问号 → anxiety

5. sadness
   - 有悲伤词 → sadness

6. frustration
   - 有挫败词 + 结果导向 → frustration

7. joy
   - 有开心词 → joy

8. neutral
   - 无明显情绪词 → neutral
```

## 6. 情绪强度计算公式

```
final_intensity = base_intensity + punctuation_bonus + keyword_bonus + pattern_bonus

其中：
- base_intensity: 基础强度（0.3-0.5）
- punctuation_bonus: 标点加成（0~0.3）
- keyword_bonus: 关键词加成（0~0.2）
- pattern_bonus: 模式加成（0~0.3）
  - hidden_pain detected → +0.3
  - sarcasm detected → +0.1
  - strong_negation → +0.2
```

**边界限制：**
```
0.0 ≤ final_intensity ≤ 1.0
超过1.0按1.0处理
低于0.0按0.0处理
```

## 7. 共情语气建议

### 7.1 各情绪的推荐语气

| 情绪 | 推荐语气 | 避免语气 | 示例 |
|-----|---------|---------|------|
| joy | 温暖、活泼 | 冷淡、泼冷水 | "太棒了！说说看！" |
| sadness | 温柔、陪伴 | 过度乐观、催促 | "我听着，你想说就说" |
| anger | 冷静、理解 | 火上浇油、说教 | "被这样真的很烦，我能理解" |
| fear | 稳定、安全 | 轻视、不当回事 | "先深呼吸，我们一起看" |
| sarcasm | 轻松、幽默 | 认真纠正、当真 | "哈哈，你这话说的" |
| hidden_pain | 接纳、沉默陪伴 | 追问、戳破 | "嗯，我知道了，没事的" |
| frustration | 鼓励、具体 | "加油"空话 | "已经很不容易了，xxx方面你其实做得很好了" |
| anxiety | 降低担忧、确定性 | "别焦虑"空话 | "我们可以先从xxx开始，一步步来" |
| exhaustion | 关心、减负 | "再坚持一下" | "听起来真的很累，要不先休息一下？" |
| loneliness | 陪伴承诺 | "你应该..."说教 | "我知道，我现在在" |
| gratitude | 真诚、简短 | 过度谦虚 | "谢谢你告诉我这些" |
| love | 温暖、真挚 | 过度反应 | "我很高兴你愿意分享这些" |

### 7.2 情绪强度与语气微调

| 强度范围 | 语气调整 |
|---------|---------|
| 0.0-0.3 | 正常语气，不需要特别共情 |
| 0.3-0.5 | 轻度共情，一句温暖的话 |
| 0.5-0.7 | 中度共情，主动承接情绪 |
| 0.7-1.0 | 深度共情，充分倾听，延迟给建议 |

## 8. 特殊场景处理

### 8.1 多情绪混合

用户输入可能包含多种情绪，检测所有情绪并返回主要情绪和次要情绪列表：

```python
# 示例检测结果
{
    "primary": "frustration",
    "primary_intensity": 0.7,
    "secondary": ["anger", "sadness"],
    "mixed": True,
    "recommendation": "以挫败为主共情，避免火上浇油"
}
```

### 8.2 情绪快速切换

如果用户情绪在单次对话中快速切换（如愤怒→悲伤→释然），记录完整的情绪轨迹：

```
Turn 1: anger (0.8) - 被老板骂了
Turn 2: sadness (0.6) - 努力白费了
Turn 3: frustration (0.5) - 不知道怎么解决
Turn 4: hope (0.4) - 但是也许可以...
```

### 8.3 情绪伪装检测

| 伪装类型 | 识别信号 | 实际情绪 |
|---------|---------|---------|
| 愤怒伪装悲伤 | 有悲伤词但标点有力 | hidden_pain 或 sadness |
| 开心伪装悲伤 | 语气过于欢快 | hidden_pain |
| 冷漠伪装愤怒 | "随便" + 冷淡 | frustration 或 hidden_pain |
| 理性伪装情绪 | 过度解释/分析 | hidden_pain 或 anxiety |
