## Description: <br>
Generates structured nonprofit grant proposal outlines that connect project goals, budgets, impact metrics, supporting evidence, and open risks. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
External nonprofit teams, grant writers, and proposal reviewers use this skill to turn project goals, beneficiary information, budgets, and evidence into a reviewable Markdown grant outline with missing information called out. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Server security evidence says the skill is advertised as a grant-proposal helper but includes broad local auditing and scanning behavior that users may not expect. <br>
Mitigation: Review the skill before installing, run it only in a disposable or non-sensitive workspace, and avoid broad directory inputs unless the publisher narrows path and file handling and clearly discloses audit behavior. <br>
Risk: Grant outlines may become misleading if the user asks the skill to fill gaps with unsupported beneficiary counts, compliance claims, or impact evidence. <br>
Mitigation: Treat output as a draft, preserve the listed confirmation items, and require source evidence and formal compliance review before submission. <br>
