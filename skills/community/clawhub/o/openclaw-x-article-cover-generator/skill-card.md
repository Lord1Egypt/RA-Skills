## Description: <br>
Generate OpenClaw-themed X (Twitter) article cover images with fixed composition: lobster logo on right quarter, text zone on left three quarters. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[yunhe-dev](https://clawhub.ai/user/yunhe-dev) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and content creators use this skill to generate OpenClaw-themed cover images for X articles, using a required title, an optional subtitle, and a primary reference image. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The title, subtitle, and reference image are sent to an external image API. <br>
Mitigation: Avoid sensitive prompts or reference images, and use a dedicated API key where possible. <br>
Risk: The script writes the returned image to a user-selected output path. <br>
Mitigation: Choose the output path carefully before running the generator. <br>
