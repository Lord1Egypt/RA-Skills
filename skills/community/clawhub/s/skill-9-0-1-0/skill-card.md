## Description: <br>
Installs and configures upstream tools that let an agent access Twitter/X, Reddit, YouTube, GitHub, Bilibili, XiaoHongShu, Douyin, LinkedIn, Boss Zhipin, WeChat, RSS feeds, and web pages. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent operators use this skill to set up and check platform access channels, configure credentials or proxies when required, and then call the installed upstream tools directly for content retrieval, search, posting, or platform-specific workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill enables broad internet access and account-backed actions such as posting, uploading, liking, commenting, and publishing through upstream tools. <br>
Mitigation: Install only when broad platform automation is intended, trust the publisher and upstream tools, and require explicit user confirmation before any account-changing action. <br>
Risk: The skill asks users to provide browser cookies or extract cookies from a local browser, which can expose account credentials and create account-safety risks. <br>
Mitigation: Use dedicated accounts, avoid pasting main-account cookies into chat, limit stored credentials, and follow the skill's dedicated-account guidance for cookie-based login. <br>
