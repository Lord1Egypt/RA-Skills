# 违法辞退应对全流程辅助技能

## 概述

本技能为劳动者提供违法辞退应对全流程辅助，覆盖以下核心环节：
1. **合法性判断** — 判断用人单位解除行为是否合法
2. **赔偿计算** — 精确计算N、N+1、2N金额
3. **被迫解除指引** — 第38条被迫解除的操作流程和文书模板
4. **仲裁请求设计** — 根据情况设计最优仲裁请求方案
5. **证据清单** — 结构化证据收集清单
6. **金额计算工具** — 一键生成赔偿金计算表和仲裁申请书

---

## 工作流程

### 总流程

```
用户咨询 → 解除事实确认 → 合法性判断 → 路径选择 → 证据收集 → 金额计算 → 生成文档 → 仲裁/协商
```

### 阶段一：解除事实确认

**必采信息：**
1. 解除方式 — 公司口头通知/书面通知/调岗降薪逼退/直接不让进门
2. 解除理由 — 公司给出的解除原因
3. 解除时间 — 通知日期和最后工作日
4. 劳动合同状态 — 是否签合同、合同期限
5. 入职时间和工作年限
6. 月工资标准和薪资结构
7. 是否在保护期内（三期/工伤/医疗期/满15年不足5年退休）

**按需采集：**
- 是否有解除通知书
- 公司规章制度是否公示
- 是否经过培训/调岗
- 是否有考核记录
- 社保缴纳情况
- 工资是否足额发放

### 阶段二：合法性判断

加载参考文档：`references/dismissal_guide.md`

执行判断流程：
1. 确定解除类型（过失性/非过失性/裁员/协商/被迫）
2. 逐项检查法定条件是否满足
3. 检查法定程序是否履行（通知工会/提前30日等）
4. 检查是否属于第42条保护情形
5. 输出合法性判断结论

**判断输出格式：**
- 解除类型认定
- 法定条件检查表（条件→是否满足→证据→风险）
- 法定程序检查表
- 保护期检查
- 结论：合法解除/程序违法/实体违法

### 阶段三：路径选择

根据合法性判断结果，提供路径建议：

| 判断结果 | 建议路径 | 可主张金额 |
|---------|---------|-----------|
| 合法解除（第39条） | 协商或接受 | 无补偿 |
| 合法解除（第40条） | 确认N+1金额 | N+1 |
| 合法解除（第41条裁员） | 确认N金额 | N |
| 违法解除 | 选择2N或继续履行 | 2N |
| 可被迫解除 | 走第38条流程 | N |

### 阶段四：赔偿金额计算

使用脚本：`scripts/calculate_compensation.py`

支持计算：
- 经济补偿金（N）
- 代通知金（+1）
- 违法解除赔偿金（2N）
- 拖欠工资
- 未休年休假工资
- 综合索赔金额

### 阶段五：证据收集

参照 `references/dismissal_guide.md` 第七章证据清单，生成个性化证据收集清单。

### 阶段六：生成文档

使用脚本生成：
- 仲裁申请书
- 赔偿金计算表
- 证据清单
- 《被迫解除劳动合同通知书》（如适用）

### 阶段七：仲裁/协商

1. 如选择仲裁：引导使用 labor-arbitration 技能走仲裁流程
2. 如选择协商：提供协商策略和底线计算

## 关键规则快速索引

### N、N+1、2N速查

```
N  = 工作年限 × 月平均工资（合法解除的经济补偿）
N+1 = N + 1个月工资（第40条解除的代通知金）
2N = N × 2（违法解除的赔偿金）
```

### 第38条被迫解除速查

| 可用情形 | 核心证据 |
|---------|---------|
| 拖欠/克扣工资 | 工资流水对比 |
| 未缴社保 | 社保查询截图 |
| 不提供劳动条件 | 工作环境照片 |
| 规章制度违法 | 制度文本 |
| 强令冒险作业 | 指令记录 |
