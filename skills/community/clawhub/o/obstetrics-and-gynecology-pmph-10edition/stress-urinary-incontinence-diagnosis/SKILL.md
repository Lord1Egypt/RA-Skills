# 压力性尿失禁诊断流程

执行以下步骤以明确是否为压力性尿失禁（SUI），并区分解剖型与尿道内括约肌障碍型：

1. **初步评估**：以患者主诉为核心，排除急迫性尿失禁、充盈性尿失禁及泌尿系统感染。

2. **压力试验（Stress Test）**：
   - 确保患者膀胱充盈，取截石位；
   - 嘱其咳嗽，观察尿道外口；
   - 若每次咳嗽均立即伴随尿液溢出，提示SUI；
   - 若为延迟溢尿或大量溢尿，提示非抑制性膀胱收缩；
   - 若截石位无溢尿，需让患者站立位重复试验。

3. **指压试验（Bonney Test）**：
   - 将中、示指放入阴道前壁尿道两侧，指尖位于膀胱颈处；
   - 向前上方抬高膀胱颈后再次诱发咳嗽；
   - 若漏尿消失，则为阳性，支持解剖型SUI诊断。

4. **棉签试验（Q-tip Test）**：
   - 患者仰卧，将涂有利多卡因凝胶的棉签置入尿道至膀胱颈；
   - 测量静息及Valsalva动作时棉签与地面夹角；
   - 角度差 <15°：解剖支持良好；
   - >30°：解剖支持薄弱；
   - 15°~30°：结果不确定。

5. **尿动力学检查**：
   - 进行膀胱内压测定以区分非抑制性逼尿肌收缩与SUI；
   - 同时行尿流率测定评估排尿功能。

6. **辅助检查**：根据需要安排尿道膀胱镜或盆底超声检查以进一步明确诊断。

综合上述结果，判断是否为SUI，并确定其亚型。
