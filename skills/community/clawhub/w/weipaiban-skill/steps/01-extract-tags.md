# Step 1：提取分类标签和关键词

从用户输入中提取以下信息：

- **类别标签**：用于筛选模板分类，必须从以下固定列表中匹配（可多选）：
  `基础` `推荐` `节日` `点击` `展开` `切换` `弹窗` `答题` `轮播` `滑动` `美化` `新年` `圣诞` `媒体` `路径` `音频` `视频` `链接` `特效`
  例如：用户提到"清明节"→ 匹配标签 `节日`；用户提到"新年贺卡"→ 匹配标签 `新年`
- **主题关键词**：用于按标题搜索相关模板（如"清明"、"端午"、"春节"）
- **具体要求**：用户对内容的特殊说明

## 初始化任务目录

提取完成后，创建任务目录并写入初始元信息：

1. 生成临时目录：`/tmp/weipaiban-task-tmp-{当前时间戳}/`（workId 在 Step 3 才获得）
2. 在目录中创建 `images/` 子目录
3. 写入 `meta.json`：

```json
{
  "taskDir": "/tmp/weipaiban-task-tmp-{时间戳}",
  "workId": null,
  "templateId": null,
  "theme": "用户主题",
  "tags": ["匹配到的标签"],
  "keywords": "主题关键词",
  "requirements": "具体要求（如无则为空字符串）",
  "currentStep": "01",
  "createdAt": "ISO时间戳"
}
```

记录 `taskDir` 路径，后续所有步骤都通过此路径访问任务目录。然后进入下一步。
