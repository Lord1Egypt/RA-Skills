## 5. 合规与监管

### 5.1 广告法专属限制

| 限制类型 | 禁用 |
|---------|------|
| 极限词 | 全行业通用 |
| 材质虚标 | "100% 羊毛"必须有检测报告 |
| 原产地虚假 | 不得虚标"意大利进口"等 |
| 价格欺诈 | "原价 X" 必须有真实销售记录 |

### 5.2 行业特殊资质

| 业务类型 | 必备资质 |
|---------|---------|
| 服装生产 | 营业执照 + 质检报告 |
| 进口服装 | 海关报关 + 中文标签 |
| 皮革制品 | 皮革材质检测报告 |
| 功能性（防紫外线/防水） | 功能性检测报告 |

### 5.3 平台审核红线

| 平台 | 红线 |
|------|------|
| 抖音 | 材质虚假、低俗擦边、假大牌 |
| 小红书 | 软广未声明、虚假"自用" |
| 全平台 | 仿冒大牌（A 货） |
