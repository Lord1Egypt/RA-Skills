# Bank Credit Memo Writer
# 银行信用备忘录撰写助手

> *AI-powered credit memo generator for commercial banks — from financial analysis to structured credit reports and facility recommendations.*

> AI驱动的商业银行信用备忘录生成工具——从财务分析到结构化授信报告，一键生成。

---

## Overview

This skill generates professional credit memos for bank credit officers. It covers borrower financial analysis, internal credit rating, credit facility recommendations, and complete credit memo documentation following standard bank formats.

本工具为银行信贷人员生成专业信用备忘录，涵盖借款人财务分析、内部信用评级、授信额度建议及符合行内规范的完整信用备忘录文档。

## Installation

```bash
clawhub install bank-credit-memo
```

## Core Capabilities

| Capability | Description |
|------------|-------------|
| Financial Analysis | 3-year ratio analysis, peer comparison, trend analysis |
| Credit Rating | Internal 5-tier rating (AAA→D) with external mapping |
| Memo Generation | Complete credit memo in bank standard format |
| Facility Recommendation | Amount, tenor, pricing, covenants, monitoring indicators |
| Green Credit Support | ESG/green credit memo with carbon/environmental metrics |

## Use Cases

- **New credit application** (新客户授信申请)
- **Credit renewal review** (存量客户续授信)
- **Credit upgrade/downgrade analysis** (评级调整分析)
- **Group credit assessment** (集团客户统一授信)
- **Green credit facility** (绿色信贷项目评估)

## Report Format

Follows standard commercial bank credit memo format with sections:
1. Executive Summary & Application
2. Credit Analysis (Financials + Qualitative)
3. Industry & Risk Assessment
4. Security & Collateral
5. Rating & Recommendation
6. Approval Sign-off
