## 一、十四主星详解

### 1.1 主星分类体系

紫微斗数将十四主星分为两大星系：

| 星系 | 主星 | 特点 |
|------|------|------|
| **紫微星系** | 紫微、天机、太阳、武曲、天同、廉贞 | 主外显、格局、气象 |
| **天府星系** | 天府、太阴、贪狼、巨门、天相、天梁、七杀、破军 | 主内蕴、根基、实务 |

两系在命盘上**两两对望**：
- 紫微 ↔ 七杀
- 天机 ↔ 巨门
- 太阳 ↔ 太阴
- 武曲 ↔ 贪狼
- 天同 ↔ 天梁
- 廉贞 ↔ 破军

### 2.4 辅星与主星的互动关系

| 组合 | 效果 |
|------|------|
| 紫微 + 左辅右弼 | 帝王得良臣，管理能力大幅提升 |
| 天府 + 禄存 | 财库稳固，善于积累 |
| 贪狼 + 文昌文曲 | 才艺财运双收 |
| 武曲 + 煞星 | 易破财、财务纠纷 |
| 太阳 + 天魁天钺 | 名气事业双收 |
| 天梁 + 魁钺 | 司法/医疗领域得贵人 |

### 2.5 辅星使用要点

1. **辅曜 vs 佐曜**：
   - 辅曜（天魁、天钺、左辅、右弼）→ **他力**，非个人努力可获
   - 佐曜（文昌、文曲、禄存、天马）→ **自力**，需个人争取

2. **成对出现效果更佳**：左右同宫、魁钺夹命时效力翻倍

3. **吉星过多反主依赖**：六吉齐会可能缺乏主见

4. **逢空劫/截空**：贵人助力落空

## 三、四化星详解

### 3.1 四化基础概念

四化是紫微斗数的**动态核心**，由十天干触发主星产生质变：

| 四化 | 五行 | 象征 | 含义 |
|------|------|------|------|
| **化禄** | 阴土 | 财富、机遇、顺遂 | 得到、满足、开始 |
| **化权** | 阳火 | 权力、竞争、掌控 | 强化、主控、变动 |
| **化科** | 阳木 | 名声、贵人、才华 | 显现、认可、健康 |
| **化忌** | 阴水 | 阻碍、消耗、执着 | 失去、压力、缺失 |

| 天干 | 化禄 | 化权 | 化科 | 化忌 |
|------|------|------|------|------|

| **乙** | 天机 | 天梁 | 紫微 | 太阴 |

### 3.3 四化象意详解

#### 化禄（缘起）
- 象征：福禄、喜欢、聪明、增加、开始
- 在夫妻宫：恋爱、感情甜蜜
- 在官禄宫：事业顺利
- 在财帛宫：财源广进
- **过盛**可能导致过度消费

#### 化权（缘变）
- 象征：掌控、能力、专业、增多、成就
- 在官禄宫：渴望权力、积极进取
- 在财帛宫：理财掌控欲强、善于投资
- **过盛**可能导致争斗、摩擦

#### 化科（继缘）
- 象征：健康、解厄、贵气、科考、才艺
- 在命宫：温文尔雅、学识渊博
- 在官禄宫：职业声誉好
- **过盛**可能导致恋旧、多愁善感

#### 化忌（缘减/缘灭）
- 象征：先得后失、变动、收藏、情义
- 在夫妻宫：婚姻波折、矛盾
- 在财帛宫：破财、投资失利
- **化忌种类**：
  - **飞出忌**：忌入对宫，流出、泄露
  - **逆水忌**：忌入对宫但被挡回，跑路且完败
  - **拆马忌**：化忌在四马地，分离、跳槽
  - **桃花忌**：化忌在四败地，不务正业
  - **入库忌**：化忌在四库地，保守顽固

### 3.4 四化与宫位交互

| 宫位/四化 | 化禄 | 化权 | 化科 | 化忌 |
|-----------|------|------|------|------|
| **命宫** | 乐观积极、贵气 | 自我意识强、有主见 | 注重形象、谦逊有礼 | 自责倾向、挫折较多 |
| **财帛宫** | 财源广进 | 理财掌控欲强 | 财运平稳、诚信得名 | 破财风险、收入不稳 |
| **官禄宫** | 事业顺利、晋升 | 渴望权力、积极进取 | 职业声誉好 | 职场阻碍、工作繁重 |
| **夫妻宫** | 感情甜蜜 | 控制欲强 | 名声与婚姻 | 婚姻波折、口舌是非 |

### 3.5 四化组合格局

| 组合 | 效果 |
|------|------|
| **禄权同宫** | 强势组合，财富权力双收 |
| **禄科同宫** | 名利双收，优雅得财 |
| **权科同宫** | 权力与声誉兼备，专业领域成功 |
| **禄忌冲对** | 能量最强，顺境突遭变故 |
| **忌入忌冲** | 课题深化，需直面困难 |

### 3.6 自化概念

- **向心自化**：某宫四化飞入本宫，代表内在能量聚集
- **离心自化**：某宫四化飞出冲本宫，代表外在变化

**自化判断**：
- 禄随忌走：化忌所在宫位揭示化禄机遇的代价
- 权科解忌：化权/化科可化解化忌的部分负面影响

## 四、命盘与起名

### 4.1 紫微斗数与命名的关联

1. **补足命盘缺失**
   - 命盘某宫位星曜较弱，可通过姓名用字五行补救
   - 例如：命宫星曜属木过弱，可选用木属性的字

2. **姓名与四化**
   - 姓名用字可对应不同星曜的五行属性
   - 配合命盘四化选择有利的字形字义

3. **姓名与宫位**
   - 根据十二宫位的象意选择姓名用字
   - 子女宫：选与子女缘分相关的字
   - 田宅宫：选与家宅、财富相关的字

### 4.2 起名参考原则

| 命盘特点 | 起名方向 |
|----------|----------|
| 紫微星强 | 稳重、领导、权威类 |
| 天机星强 | 智慧、策划、艺术类 |
| 太阳星强 | 热情、公益、光明类 |
| 武曲星强 | 刚毅、果决、财运类 |
| 天同星强 | 福气、和谐、享乐类 |
| 贪狼星强 | 交际、艺术、桃花类 |
| 化禄多 | 财运、机遇、贵人缘 |
| 化忌多 | 化解、调和、稳健 |

### 4.3 起名注意事项

1. **避免冲克**：姓名五行与命盘主星五行相冲
2. **补足不足**：命盘某宫位星曜落陷，用姓名补救
3. **结合四化**：根据生年四化选择有利于化解忌星的字
4. **音形义兼顾**：读音朗朗上口，字形端正美观

## 五、三合派与四化派对比

| 比较项目 | 三合派（南派） | 四化派（北派） |
|----------|---------------|---------------|
| **重视要素** | 星曜配置、格局 | 宫位飞化、四化星 |
| **使用星数** | 100+颗 | 约30颗（14主星+四化） |
| **核心理论** | 《紫微斗数全书》 | 《斗数秘仪》 |
| **擅长领域** | 性格分析、长期运势 | 短期运势、事件应期 |
| **学习特点** | 数据量大，需好记性 | 数据量少，需悟性 |
| **推断方式** | 三方四正看星组 | 宫干飞化论吉凶 |

### 门派分支

**北派（四化飞星）**：
- 河洛派（务实，曾国雄创立）
- 华山钦天密仪派（飞星理论）
- 仙宗派（结合子平法）

**南派（三合星性）**：
- 中州派（借宫安星）
- 紫云派（太岁入卦）

## 六、学习要点总结

### 本课核心概念
1. **十四主星**：分紫微星系（6颗）和天府星系（8颗），各有独特性格
2. **辅煞星**：六吉星辅助主星，六煞星增加变数
3. **四化飞星**：化禄、化权、化科、化忌，揭示命运动态变化
4. **命名关联**：根据命盘星曜和四化选择有利于运势的名字
