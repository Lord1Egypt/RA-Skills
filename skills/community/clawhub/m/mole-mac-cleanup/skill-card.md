## Use Case: <br>
Developers and Mac users use this skill to have an agent propose Mole CLI commands for previewing and running macOS cleanup, optimization, installer cleanup, and project artifact purge workflows. It is scoped to macOS systems with the Homebrew Mole package installed, and users should review dry-run output before running commands that delete local files or change system settings. <br>
