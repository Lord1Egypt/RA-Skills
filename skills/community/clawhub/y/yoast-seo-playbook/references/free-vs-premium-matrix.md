# Yoast SEO — Free vs Premium Matrix

Use this to avoid proposing unavailable features.

## Baseline (both Free and Premium)

- SEO title and meta description templates/overrides
- Readability analysis and SEO analysis feedback in editor
- XML sitemaps
- Schema output framework (core graph generation)
- Social metadata basics (OpenGraph/X controls in plugin scope)
- Robots/canonical/meta robots controls in normal Yoast flow
- REST API metadata exposure (`yoast_head`, `yoast_head_json`) for consumption

## Premium-only (treat as optional unless confirmed)

- Redirect manager workflows in Yoast Premium UI
- Internal linking suggestions panel
- Multiple focus keyphrases support
- Orphaned content tooling and related Premium optimization assists
- Premium support/update stream and premium-specific enhancements

## Agent Policy

1. If Premium unknown:
   - Deliver a Free-compatible plan first.
   - Add a separate “If Premium is active” section.
2. If Premium confirmed:
   - You may include Premium workflow accelerators.
3. Never block core SEO work waiting for Premium.

## Practical Split for Delivery

- **Free path:** keyword intent, title/meta quality, heading structure, schema sanity, crawlability basics.
- **Premium path:** speed up execution with linking/redirect/orphan tools, not fundamental strategy changes.
