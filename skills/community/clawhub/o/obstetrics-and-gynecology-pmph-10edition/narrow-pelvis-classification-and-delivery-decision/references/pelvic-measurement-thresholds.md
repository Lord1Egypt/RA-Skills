# 骨盆各平面狭窄分级详细标准（单位：cm）

## 入口平面（以对角径 DC 为主）
- 正常值：≥12.0
- Ⅰ级（临界性狭窄）：DC = 11.5
- Ⅱ级（相对性狭窄）：DC = 10.0～11.0
- Ⅲ级（绝对性狭窄）：DC ≤ 9.5

## 中骨盆平面
- 坐骨棘间径（IS）正常值：≥10.0
- 中骨盆后矢状径（MPSPD）正常值：≥13.5
- Ⅰ级：IS = 10.0 / MPSPD = 13.5
- Ⅱ级：IS = 8.5～9.5 / MPSPD = 12.0～13.0
- Ⅲ级：IS ≤ 8.0 / MPSPD ≤ 11.5

## 出口平面
- 坐骨结节间径（IT）正常值：≥8.5
- 出口后矢状径（OPSPD）正常值：≥15.0
- Ⅰ级：IT = 7.5 / OPSPD = 15.0
- Ⅱ级：IT = 6.0～7.0 / OPSPD = 12.0～14.0
- Ⅲ级：IT ≤ 5.5 / OPSPD ≤ 11.0

> 注：出口平面总和（IT + OPSPD）>15 cm 通常可经阴道分娩。
