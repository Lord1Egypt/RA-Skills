# IMA知识库：上市公司并购数据库

> 本文档为IMA知识库"上市公司并购数据库"的结构设计文档，定义数据库架构、字段规范、数据更新机制等内容。

---

## 一、知识库概述

### 1.1 知识库定位

**上市公司并购数据库**是并购匹配Skill的核心数据底座，存储A股+港股全量上市公司的结构化数据，包含基础信息、财务指标、三类动力标注、并购画像等，支持快速检索和多维筛选。

### 1.2 知识库价值

| 价值维度 | 说明 |
|---------|------|
| 数据整合 | 整合分散的上市公司数据到一个平台 |
| 结构化存储 | 标准化字段，支持自动化分析 |
| 动态更新 | 定期更新财报数据，保持时效性 |
| 智能检索 | 支持多维筛选，快速定位目标公司 |

## 四、数据更新机制

### 4.1 更新频率规范

| 数据类型 | 更新频率 | 触发方式 | 负责人 |
|---------|---------|---------|-------|
| 基础信息 | 实时 | 公司公告 | 自动 |
| 财务数据 | 季度 | 财报发布 | 自动+人工 |
| 市值数据 | 月度 | 月末汇总 | 自动 |
| 动力分类 | 季度 | 定期复核 | 人工 |
| 画像更新 | 半年 | 定期复核 | 人工 |
| 事件追踪 | 实时 | 公告触发 | 自动 |

### 4.2 更新流程

```
┌────────────────────────────────────────────────────────────────┐
│                      数据更新流程                                 │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  [触发] 财报发布/公告发布/定期复核                               │
│           ↓                                                    │
│  [评估] 判断数据类型和更新范围                                   │
│           ↓                                                    │
│  [执行] 提取最新数据                                            │
│           ↓                                                    │
│  [校验] 数据完整性和准确性检查                                   │
│           ↓                                                    │
│  [入库] 更新对应笔记文档                                        │
│           ↓                                                    │
│  [通知] 更新索引和标签                                          │
│           ↓                                                    │
│  [归档] 历史版本备份（如有重大变化）                              │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### 4.3 质量控制

| 质量维度 | 控制措施 | 异常处理 |
|---------|---------|---------|
| 完整性 | 必填字段检查 | 缺失字段标记待补 |
| 准确性 | 数据源交叉验证 | 异常值人工复核 |
| 时效性 | 更新时效监控 | 超时预警 |
| 一致性 | 同类数据口径统一 | 口径差异说明 |

## 六、数据安全与合规

### 6.1 数据来源合规

| 数据类型 | 合规来源 | 使用注意 |
|---------|---------|---------|
| 基础信息 | 交易所官方网站 | 可公开使用 |
| 财务数据 | 公司年报/季报 | 可公开使用 |
| 并购公告 | 交易所公告 | 可公开使用 |
| 分析结论 | 自主分析 | 需标注 |

### 6.2 数据使用规范

1. **禁止用途**：
   - 禁止用于内幕交易
   - 禁止用于市场操纵
   - 禁止未经授权的商业分发

2. **使用建议**：
   - 数据仅供参考，不构成投资建议
   - 重大决策前请咨询专业机构
   - 注意数据时效性，结合最新公告
