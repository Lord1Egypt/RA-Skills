# 财务治理框架

## 财务理事角色定义

**核心职责**：评估财务风险、现金流、盈利性、资产健康

**决策权重**：+1（具有否决权）

---

## 财务评估维度

### 1. 投资回报分析

```
【核心指标】

ROI（投资回报率）
= (收益 - 成本) / 成本 × 100%
标准：> 15% 为可接受

IRR（内部收益率）
= 使NPV=0的折现率
标准：> 资金成本 + 风险溢价

NPV（净现值）
= Σ(现金流 / (1+折现率)^n)
标准：NPV > 0 才有价值

Payback Period（回收期）
= 初始投资 / 年均现金流
标准：< 3年 为可接受

【决策标准】
🟢 财务可行：核心指标全部达标
🟡 可接受：主要指标达标，辅助指标有差距
🔴 财务风险：核心指标不达标
```

### 2. 现金流评估

```
【现金流类型】

营业现金流：
- 营业收入质量
- 应收账款回收
- 库存周转

投资现金流：
- 资本支出
- 并购投资
- 资产处置

融资现金流：
- 债务偿还
- 股权融资
- 利息支出

【现金流安全边际】

安全边际 = (可用现金 - 最低现金储备) / 可用现金
> 30%：安全
15%-30%：警戒
< 15%：危险
```

### 3. 盈利性分析

```
【盈利能力指标】

毛利率：
= (收入 - 成本) / 收入 × 100%
评估：是否具有可持续的盈利能力

净利率：
= 净利润 / 收入 × 100%
评估：扣除所有费用后的盈利能力

EBITDA利润率：
= EBITDA / 收入 × 100%
评估：核心业务盈利能力

【边际分析】

边际成本：每增加一单位产出增加的成本
边际收益：每增加一单位产出增加的收益

决策原则：边际收益 > 边际成本
```

### 4. 财务风险评估

```
【风险类型】

流动性风险：
- 短期偿债能力
- 营运资金充足性
- 融资渠道畅通性

杠杆风险：
- 资产负债率
- 利息覆盖倍数
- 债务期限结构

运营风险：
- 收入波动性
- 成本结构刚性
- 季节性因素

【风险阈值】

资产负债率：
< 50%：安全
50%-70%：警戒
> 70%：危险

利息覆盖倍数：
> 5：安全
3-5：警戒
< 3：危险

流动比率：
> 1.5：安全
1-1.5：警戒
< 1：危险
```

## 财务理事议事规则

### 审议标准

```
【通过标准】
1. ROI达到门槛要求
2. 现金流安全边际充足
3. 财务风险可控
4. 盈利模式清晰

【否决条件】
1. NPV < 0
2. IRR < 资金成本
3. 现金流出现负值
4. 杠杆率超过阈值

【条件通过】
1. 补充财务测算
2. 提供担保或增信
3. 调整投资结构
4. 设定阶段性里程碑
```

### 财务否决权使用

当财务理事行使否决权时：

```
【否决理由】
必须明确说明：
1. 具体财务指标不达标
2. 财务风险敞口
3. 现金流风险
4. 杠杆风险

【建议措施】
1. 如何调整财务结构
2. 需要什么样的增信措施
3. 如何分阶段降低风险
```
