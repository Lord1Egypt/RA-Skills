# D05 家庭领域任务目录

## 领域定义

家庭领域(D05)关注个体亲密关系的建立和维护。核心驱动需求：N3归属 + N5传承。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 亲密关系建立 | 社交关系管理 |
| 婚姻经营 | 婚介服务 |
| 亲子教育 | 学校教育 |
| 家庭沟通 | 组织沟通 |
| 家庭治理 | 家政服务 |
| 家庭建设 | 房屋装修 |
| 家庭关系 | 心理咨询 |
