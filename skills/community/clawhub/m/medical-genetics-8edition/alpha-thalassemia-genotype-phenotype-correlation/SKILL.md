# Alpha地中海贫血基因型-表型对应关系分析

## 触发条件
当患者满足以下实验室指标时启动本技能：
- MCV < 80 fL
- MCH < 27 pg
- HbA2 < 2.5%

## 执行流程

1. **确认适用对象**：仅适用于α-珠蛋白基因（HBA1/HBA2）、红细胞或胎儿相关场景；排除β-或其他类型地中海贫血。

2. **确定功能性α基因拷贝数**（0–4个）：
   - 4个：正常（αα/αα）
   - 3个：静止型（-α/αα 或 α^Tα/αα）
   - 2个：轻型（--/αα、-α/-α 或 α^Tα/α^Tα）
   - 1个：HbH病（--/-α 或 --/α^Tα）
   - 0个：重型（Hb Barts胎儿水肿综合征，--/--）

3. **关联临床表型并制定管理策略**：
   - **静止型/轻型**：无症状或轻度小细胞低色素贫血，建议遗传咨询与家庭成员筛查。
   - **HbH病**：评估溶血严重程度，监测HbH包涵体，根据贫血程度决定是否输血支持，注意感染诱发危象。
   - **重型（胎儿）**：若夫妻均为α⁰地贫携带者（如--^SEA），胎儿有1/4风险患水肿综合征，应在孕11–12周（绒毛）、16–20周（羊水）或19–23周（脐血）进行产前基因诊断。

4. **输出行动建议**：基于α基因缺失数量，明确后续步骤——携带者随访、HbH病支持治疗或胎儿产前干预。

> 注意：必须结合MCV/MCH/HbA2初筛结果使用，不可单独依赖基因检测结果解释表型。
