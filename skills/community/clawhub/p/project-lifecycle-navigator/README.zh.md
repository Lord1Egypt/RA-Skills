# 项目生命周期导航助手

这是一个中英双语、纯文本指令型 Skill，适合非技术用户、创业者、运营团队、产品负责人和 AI Coding Agent 用户使用。

它会根据用户处于的项目阶段，自动进入三种流程之一：

1. **新项目需求访谈**：从 0 到 1，把模糊想法变成 MVP 范围、项目方案、技术建议和开发计划。
2. **项目中期方向校准**：当项目做到一半感觉跑偏、功能太多、方向不清时，重新问诊、砍功能、止损或转向。
3. **代码审查与升级优化**：当项目已有代码后，审查安全、冗余、性能、架构、测试和依赖问题，并生成可执行修复计划。

## 适合谁

- 不懂编程但想做软件/AI 工具的人
- 用 AI Coding Agent 做项目的人
- 创业者、运营人员、产品负责人
- 项目做到一半需要重新把关的人
- 需要让 AI 审查代码并给升级计划的人

## 触发方式

你可以这样说：

- “我想做一个新项目，你先问我问题。”
- “项目做到一半有点跑偏，帮我重新校准。”
- “我已经有代码了，帮我审查漏洞、冗余代码和重构方案。”

## 发布说明

本 Skill 不需要安装第三方包，不需要 API Key，不需要联网权限，也不运行代码。它只提供项目访谈、方向校准和代码审查流程指令。
