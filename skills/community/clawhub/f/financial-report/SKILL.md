# 企业财务报表分析与预测工具

## 概述

这是一个专业的企业财务分析工具，支持从知识库或手动输入财报数据，自动计算关键财务指标，生成可视化图表，并预测未来业绩趋势。工具包含三大报表录入、自动指标计算、多维度分析和专业报告输出。

## 你的工作方式

### Phase 1：数据获取

从知识库检索或手动输入目标企业财报数据：

1. **知识库搜索**：用 `search(kb)` 搜索企业名称 + "年报/财报/财务数据"
2. **读取内容**：用 `fetch` 读取搜索结果中的关键文章
3. **手动输入**：用户也可使用 HTML 工具界面手动输入数据

提取以下核心数据（至少需要 3 年历史数据）：
- 营业收入、营业成本、毛利润、净利润
- 总资产、总负债、股东权益
- 经营活动现金流、投资活动现金流、筹资活动现金流

### Phase 2：财务指标计算

自动计算以下关键指标：

**盈利能力**
- 毛利率 = 毛利润 / 营业收入 × 100%
- 净利率 = 净利润 / 营业收入 × 100%
- ROE = 净利润 / 股东权益 × 100%
- ROA = 净利润 / 总资产 × 100%

**成长性**
- 营收同比增长率
- 净利润同比增长率

**健康度**
- 资产负债率 = 总负债 / 总资产 × 100%
- 流动比率 = 流动资产 / 流动负债
- 现金流/净利润比 = 经营现金流 / 净利润

### Phase 3：业绩预测

基于历史数据做趋势外推：
1. 计算近 3 年 CAGR
2. 假设增速逐年递减 20%（回归均值）
3. 生成乐观/中性/悲观三情景预测

## 可视化分析工具

本技能包含一个交互式 HTML 财务分析工具（`scripts/financial-analyzer.html`），提供：

### 📝 数据录入
- 支持 4 年自定义年份输入
- 三大报表完整录入：利润表（6 项）、资产负债表（5 项）、现金流量表（3 项）

### 📈 自动计算指标
- 毛利率、净利率、ROE、ROA、资产负债率、流动比率等

### 🎨 四大可视化图表
1. 收入利润趋势图（柱状图）
2. 盈利能力趋势图（折线图）
3. 资产负债结构图（柱状图）
4. 现金流分析图（面积图）

### 📥 内置示例数据
- 贵州茅台 2021-2024 年真实财务数据

### 使用方式
1. 下载 `scripts/financial-analyzer.html`
2. 双击在浏览器中打开
3. 点击"加载示例"或手动输入数据
4. 点击"开始分析"查看结果

## 行业基准参考

分析时参考以下健康标准：

| 指标 | 优秀 | 良好 | 警戒 |
|------|------|------|------|
| ROE | > 20% | 12-20% | < 8% |
| 毛利率 | > 30% | 15-30% | < 10% |
| 资产负债率 | < 50% | 50-65% | > 75% |
| 现金流/净利润 | > 1.2 | 0.8-1.2 | < 0.5 |
| 营收增速 | > 20% | 5-20% | < 0% |
