# 渠道适配手册

同一事件，不同渠道——不是说短说长的区别，而是视角切换。

---

## 核心理念

```
        同一事件核心
        ┌─────┼─────┐
    公众号版    官网版    内部版
   "讲给外人听" "记给历史看" "说给自家人"
    有料有趣     准确完整     有血有肉
```

---

## 三版对照总表

| 维度 | 公众号版 | 官网版 | 内部版 |
|------|----------|--------|--------|
| **对谁说** | 客户、行业、潜在合作方 | 投资人、媒体、公众 | 员工、管理层 |
| **目标** | 吸引关注、建立认知 | 正式记录、合规存档 | 凝聚共识、传递方向 |
| **标题** | 观点型/悬念型/数字型 | 陈述型/公告型（5W） | 告知型/号召型 |
| **开篇** | 场景镜头或问题切入 | 时间+主体+事件 | 一句话点明意义 |
| **篇幅** | 600-1200字 | 300-500字 | 200-400字 |
| **图片建议** | 3-5张，有视觉冲击 | 1-2张，正式场景 | 1-3张，真实现场 |
| **语气** | 亲和专业，可适当轻松 | 严谨客观，第三视角 | 真诚直接，"我们" |
| **禁用** | 过于内部的细节、敏感数字 | 口语表述、主观评价、第一人称 | 对外营销语言、竞争比较 |
| **视觉输出** | 公众号封面 PNG（900×500） | — | 内部通知图 PNG（750×400） |

## 公众号版

### 定位
不是"软文"，是一个有趣有用的行业人跟你聊天。

### 标题规则
- **观点型**：《XX提升了2.3%，背后是我们做对了这件事》
- **悬念型**：《这家世界500强走进产线，只为一件事》
- **数字型**：《14个月、7次交流、1.2亿——一个签约的故事》
- 忌：公告式标题、"重磅""官宣"等陈词

### 开篇规则
- 场景切入（"李总站在产线前看了20分钟，问了三个问题..."）
- 问题切入（"财务月底关账，最怕的就是..."）
- 忌：5W罗列（"近日，XX公司一行莅临..."）

### 正文规则
- 段落短（手机屏3-4段）
- 有金句（至少1句可被截图转发的话）
- 有画面（至少1个读者能"看见"的细节）
- 有数据（至少1个具体数字）
- 结尾：不说"欢迎垂询"，用自然的收束

---

## 官网版

### 定位
公司历史的正式记录，写给需要准确信息的人。

### 标题规则
- **陈述型**：《雄韬股份与XX集团签署战略合作协议》
- **公告型**：《雄韬股份发布2026年半年度报告》
- 忌：悬念、比喻、网络用语

### 开篇规则
- 标准5W：时间 + 主体 + 事件 + 地点 + 背景
- "2026年6月18日，广州雄韬信息技术有限公司与XX集团在广州签署战略合作协议。"
- 忌：场景描写、情感铺垫、第一人称

### 正文规则
- 信息完整：谁、什么事、为什么重要、下一步
- 语言客观：用"双方""公司"，不用"我们""他们"
- 可供引用：媒体可以直接摘取的段落
- 数字精确：不模糊化处理
- 称谓正式：全称+职务

---

## 内部版

### 定位
给并肩作战的伙伴看的，真诚的、内部视角的表达。

### 标题规则
- **告知型**：《XX产线升级完成！》
- **号召型**：《端午安康｜这个假期，账先放一放》
- 忌：对外宣传腔、官话

### 开篇规则
- 直接说为什么这件事对团队重要
- "今天XX客户签了。想跟大家说说背后的故事。"
- 忌：外部人听不懂的缩写、外人看不懂的内部梗（适度即可）

### 正文规则
- 有"我们"：第一人称，真诚直接
- 有背后故事：外界看不到的努力
- 有具体的人：谁做了什么，点名感谢
- 有下一步：对团队意味着什么，接下来要做什么
- 结尾：有温度，不说教

## 渠道转换自动化公式

```
核心稿（叙事骨架填完的母版）
    │
    ├──→ 公众号版：母版 × 1.5倍篇幅
    │     + 标题悬念化、场景描写、金句点缀
    │     - 内部细节、敏感数字
    │
    ├──→ 官网版：母版 × 0.6倍篇幅
    │     + 5W要素、时间戳、正式称谓
    │     - 口语表述、主观评价、第一人称
    │
    └──→ 内部版：母版 × 0.5倍篇幅
          + 团队署名、背后故事、下一步行动
          - 对外营销语言、竞争比较
```

## 特殊场景：节日/节气

节日通常不出官网版。只需：

| 公众号版 | 内部版 |
|----------|--------|
| 品牌温度展示 | 团队凝聚力 |
| 可对外转发 | 内部群发 |
| 500-800字 | 200-300字 |

## 追加渠道（扩展）

| 渠道 | 特征 | 适配规则 |
|------|------|----------|
| 抖音/视频号 | 口播稿 | 公众号版浓缩到200字 + 口语化 |
| 小红书 | 图文笔记 | 公众号版精炼到400字 + emoji + 标签 |
| LinkedIn | 英文/双语 | 官网版翻译 + 国际化表达 |
