## Description: <br>
Webアプリケーション開発のためのOpenClawワークスペース設定テンプレート。ユーザー体験（UX）優先、レスポンシブデザイン、パフォーマンスの原則。 <br>

This skill is ready for commercial/non-commercial use. <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and external users use this skill to set up an OpenClaw workspace for web application development. It provides Markdown templates for agent identity, workspace behavior, user preferences, and heartbeat-style project monitoring. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The templates grant broad ongoing memory and proactive maintenance authority. <br>
Mitigation: Before use, edit the templates to require explicit approval before deleting bootstrap files, updating dependencies, deploying, accessing logs or monitoring systems, or storing sensitive personal or project context. <br>
