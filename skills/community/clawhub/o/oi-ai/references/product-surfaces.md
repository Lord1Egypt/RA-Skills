# Oi Product Surfaces

Use current Oi product surfaces when naming or scoping work:

- Public website discovery and autocomplete.
- Public API search via `/search`.
- App search/discover via `/search/app`.
- Context generation suggestions.
- Workflow discovery and installation.
- Organization analytics and usage analytics.
- Search analytics and admin metrics.
- Organization onboarding, invites, switching, billing, and API keys.
- MCP/API access for Context and workflow usage.
- Super-user admin tools for users, organizations, and product metrics.

Avoid assuming backlog automation, repository sanitization, knowledge graphs, or cross-system intelligence features unless the user explicitly introduces them.
