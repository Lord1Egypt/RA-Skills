# 急性炎症病理类型分类

## 触发条件
当存在致炎因子并已发生急性炎症反应，且病变以明显渗出为主时，依据渗出物中占主导的成分进行分类。

## 执行步骤

1. **识别主导渗出成分**：观察组织或体液标本中以下哪一类成分占优势：
   - 浆液（清亮液体，含少量蛋白）
   - 纤维蛋白（红染网状/条状结构）
   - 脓液（含大量变性中性粒细胞）
   - 红细胞（大量出血）

2. **匹配炎症类型**：
   - **浆液性炎**：渗出物以含3%～5%白蛋白的浆液为主，常见于黏膜、浆膜、滑膜、皮肤或疏松结缔组织。
   - **纤维蛋白性炎**：以大量纤维蛋白渗出为特征，多见于细菌毒素或毒物所致，可表现为假膜性炎（黏膜）、绒毛心（心包）或大叶性肺炎（肺）。
   - **化脓性炎**：以中性粒细胞大量浸润、组织坏死及脓液形成为主，分为表面化脓/积脓、蜂窝织炎或脓肿。
   - **出血性炎**：渗出物含大量红细胞，提示血管严重损伤，常见于特定感染（如流行性出血热、鼠疫）。

3. **评估合并与转化可能**：注意各型可单独或合并存在（如浆液性纤维蛋白性炎），亦可在病程中相互转化。

4. **指导临床判断**：结合部位、病原体及转归风险（如喉头水肿窒息、脓肿迁徙、机化粘连等）提出预后与处理建议。

> **注意**：本技能仅适用于急性炎症阶段，不适用于慢性炎症；若无明显渗出性病变，不应强行分类。
