## 📋 Commands

```
pricing analyze          # 全面定价分析
pricing cost <asin>      # 成本结构分析
pricing peer <asin>      # 同行价格参考
pricing promotion        # 促销定价策略
pricing optimize         # 利润优化建议
pricing adjust <asin>    # 调价影响评估
pricing report           # 整体定价健康报告
```

## 📖 示例对话

**用户**：
> 我卖蓝牙耳机，成本$12，FBA配送费$4，想上架亚马逊美国

**助手**：
> 帮你分析一下：
> 
> **成本结构**：
> - 商品成本：$12
> - FBA配送费：$4
> - 推荐费（8%）：按售价计算
> - 盈亏平衡价：约 $17.39
> 
> **定价建议**：
> - 保守策略：$22（利润率约21%）
> - 平衡策略：$25（利润率约28%）
> - 进取策略：$29（利润率约35%）
> 
> 建议从 $24-26 开始，观察转化后再调整。
