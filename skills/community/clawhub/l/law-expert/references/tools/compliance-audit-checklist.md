# 多行业合规自查清单

## 用途
按行业分类提供三色合规自查框架（红线/风险/建议），帮助企业在各自行业中快速定位合规短板。

## 使用场景
- 企业年度合规体检
- 切入新行业/新业务前的合规尽调
- 应对监管检查前的自我排查
- 投资/并购项目中的标的合规评估

## 核心内容

### 🔴 = 红线（触及即致命，可能导致吊销/刑事责任）
### 🟡 = 风险（需整改，可能面临行政处罚/民事索赔）
### 🟢 = 建议（最佳实践，提升合规竞争力）

| 行业 | 🔴 红线 | 🟡 风险 | 🟢 建议 |
|------|---------|---------|---------|
| **电商** | 无证经营（ICP许可证缺失）；二选一垄断行为；价格欺诈（虚假原价）；大数据杀熟 | 七天无理由退货执行不到位；个人信息违规收集；刷单炒信；跨境电商无海关注册 | 建立合规中台统一审核商品发布；定期对标《网络交易监督管理办法》 |
| **餐饮** | 无《食品经营许可证》；超范围经营（凉菜/生食需单独许可）；使用过期/来源不明食材 | 从业人员无健康证；后厨卫生不达标；食品添加剂超标；外卖封签不合规 | 导入HACCP体系；明厨亮灶+用户评价公示；建立食材溯源台账 |
| **教培** | 义务教育阶段学科培训（"双减"违规）；无证办学；预收费超3个月且不纳入监管账户 | 教师无资质；虚假宣传"保过""包就业"；培训合同退费条款不公 | 资金监管账户全覆盖；课程内容合规预审；家长知情权条款透明化 |
| **医疗** | 无《医疗机构执业许可证》；超范围执业（科室）；使用无证医疗器械/药品 | 医师多点执业未备案；病历书写不规范；医疗广告未审批；患者隐私泄露 | 建立医疗质量安全核心制度；电子病历合规归档；医疗纠纷预防与处置预案 |
| **金融科技** | 无牌经营（支付/贷款/征信/保险牌照）；非法集资/资金池；为非法活动提供支付通道 | 利率超司法保护上限；暴力催收；用户适当性管理缺失；反洗钱义务履行不到位 | 接入央行征信系统；建立智能风控模型+人工复核；定期第三方合规审计 |
| **AI/互联网** | 利用算法实施价格歧视/信息茧房操纵；深度合成技术用于诈骗/造谣；训练数据侵权 | 算法备案未完成；AI生成内容未标识；用户数据用于模型训练未获同意；自动化决策无人工干预渠道 | 建立算法伦理委员会；AI生成内容水印技术嵌入；大模型安全评估报告存档 |
| **建筑** | 无资质/超越资质等级承揽工程；违法转包/挂靠；未取得施工许可证即开工 | 安全生产责任险未购买；农民工工资专户未设立；环保措施不达标（扬尘/噪音） | 工程建设全过程BIM档案；供应链合规穿透审查；绿色建筑星级认证对标 |
| **制造** | 排污许可证缺失/超证排污；危险化学品违规储存；职业病防护设施未"三同时" | 安全生产事故应急预案未备案；特种设备超期未检；产品质量标准低于国标 | ISO 14001环境管理体系认证；产品碳足迹追踪；供应链ESG合规审查 |
| **文娱** | 内容违规（涉政/涉黄/涉暴）；艺人偷逃税；网络表演未取得《网络文化经营许可证》 | 直播未实名制；未成年人打赏未设限额；版权使用未获得完整授权链 | 内容审核AI+人工双机制；版权区块链存证；MCN机构合规管理制度 |

### 快速自查方法

```
Step 1: 确认所在行业（可多选）
Step 2: 逐条核对🔴红线 → 有任何一项存在 → 立即停止相关业务并咨询律师
Step 3: 逐条核对🟡风险 → 制定整改时间表（30天内完成）
Step 4: 参照🟢建议 → 纳入年度合规计划
```

## 使用说明

1. **红线零容忍**：红线项触及即可能导致刑事追诉或吊销执照，必须优先处理
2. **行业交叉**：若企业跨行业经营（如电商+金融科技），需同时满足多个行业的清单要求
3. **动态更新**：监管政策频繁调整（如互联网行业），建议每季度对照最新法规更新清单
4. **证据留存**：合规整改过程需保留书面记录，以应对监管检查时的"尽职免责"举证
5. **专业兜底**：本清单为自查框架，涉及具体合规判断应咨询执业律师——特别是红线项
