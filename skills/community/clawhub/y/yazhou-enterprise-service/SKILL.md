# 崖州区企业服务技能

本技能为崖州区企业服务工作人员提供企业信息查询、优惠政策推荐、安全生产监管、企业发展建议等一站式服务。支持同时为多家企业生成服务报告。

## 适用场景

当用户（崖州区企业服务人员）需要以下服务时，使用本技能：
- 查询某企业的基本情况（工商信息、经营状态、资质等）
- 根据企业类型/行业/规模，推荐适用的优惠政策
- 解答企业关于政策的咨询问题
- 监管企业安全生产情况
- 结合崖州区产业发展情况，为企业提供发展建议
- 同时为多家企业生成服务报告
- 生成企业服务报告或监管记录

触发词示例：
- "查询XX公司的基本情况"
- "这家企业符合哪些优惠政策？"
- "推荐适合科技型中小企业的政策"
- "如何监管企业安全生产？"
- "给XX企业一些发展建议"
- "崖州区产业发展重点是什么？"
- "XX行业在崖州有什么发展机会？"
- "生成企业服务报告"
- "为XX、YY、ZZ这几家企业生成服务报告"

## 核心工作流程

### 0. 输出规范（必读）

**输出格式**：所有服务报告均以 Word 文档（.docx）输出。

**输出工具**：使用 `docx` 技能生成 Word 文档。调用方式：`Skill` 工具，参数 `skill: "docx"`。

**多企业报告**：当用户同时提供多家企业名称时，在同一个 Word 文档中，按企业顺序逐个罗列每家企业的完整服务报告。文档结构如下：

```
封面（报告标题、日期）
企业1 完整服务报告（含信息查询、政策推荐、发展建议、安全生产监管）
───────── 分页符 ─────────
企业2 完整服务报告
───────── 分页符 ─────────
企业3 完整服务报告
...
```

每家企业的报告内部结构参见"企业服务报告模板"。

**单企业报告**：当只查询一家企业时，文档结构同上但只有一个企业的报告。

### 1. 企业信息查询

使用 WebSearch 或 WebFetch 工具查询企业信息：
- 通过企业名称、统一社会信用代码查询
- 查询目标：工商信息、经营范围、注册资本、成立时间、股东信息、行政处罚、经营异常等
- 推荐查询渠道：国家企业信用信息公示系统、天眼查、企查查等
- 将查询结果结构化整理，便于后续政策匹配

查询步骤：
1. 确认企业全称或统一社会信用代码
2. 使用 WebSearch 搜索："[企业全称] 企业信用信息" 或 "[企业全称] 天眼查"
3. 使用 WebFetch 获取具体页面内容（如国家企业信用信息公示系统页面）
4. 提取关键信息：企业类型、行业、规模、资质、经营状态等

**判断是否为上市公司：**
- 使用 WebSearch 搜索："[企业全称] 上市公司" 或 "[企业全称] A股/创业板/科创板"
- 如使用 neodata-financial-search 技能，可查询股票代码（如：海南矿业 601969）
- 确认后标记为"上市公司"，进入工作流程 2

### 2. 上市公司财报与金融数据查询

**适用场景**：当企业被确认为上市公司时，执行本步骤。

**查询工具**：使用 `neodata-financial-search` 技能（自然语言金融数据查询）

**查询内容：**

1. **基本行情数据**：
   - 股票代码、当前股价、涨跌幅、成交量、成交额
   - 市盈率（PE）、市净率（PB）、总市值

2. **财务报表数据**（近3年）：
   - 利润表：营业收入、营业利润、净利润、归母净利润
   - 资产负债表：总资产、总负债、净资产、资产负债率
   - 现金流量表：经营活动现金流净额

3. **财务指标**：
   - 盈利能力：ROE（净资产收益率）、ROA（总资产报酬率）、毛利率、净利率
   - 成长能力：营收增长率、净利润增长率
   - 偿债能力：流动比率、速动比率、资产负债率
   - 运营能力：总资产周转率、存货周转率

4. **财报摘要**：
   - 最新季报/年报核心数据
   - 业绩变动原因说明
   - 审计意见类型

**操作方式**：
```
调用 Skill 工具，参数：skill: "neodata-financial-search"
查询示例：
- "[股票代码/公司名称] 2024年财报 营业收入 净利润"
- "[股票代码/公司名称] 最新股价 PE PB 市值"
- "[股票代码/公司名称] 近三年ROE 营收增长率"
```

**输出内容**：将查询结果整理后，加入企业服务报告的"二、资质与认定"或单独增设"上市公司财务概况"章节。

### 3. 优惠政策推荐

根据查询到的企业信息，匹配适用的优惠政策：

**政策分类（参考 references/policies.md）：**
- 税收优惠政策
- 财政补贴政策
- 融资支持政策
- 人才引进政策
- 科技创新政策
- 产业升级政策

**推荐流程：**
1. 提取企业特征：行业类型、企业规模、成立时间、资质认定（高企、科技型中小企业等）
2. 匹配政策：根据企业特征，从政策库中筛选适用政策
3. 优先级排序：根据政策力度、申请难度、时效性排序
4. 生成推荐报告：政策名称、适用条件、补贴金额、申请流程

### 4. 政策咨询解答

当用户（企业服务人员）或企业提出政策咨询时：
1. 理解问题核心：明确企业关注的政策类型或具体问题
2. 查询政策文件：使用 WebSearch 搜索最新政策文件
3. 提取关键信息：申请条件、材料清单、办理流程、时间节点
4. 生成解答：用清晰、简洁的语言回答，附上政策依据

### 5. 安全生产监管

**监管流程：**
1. 查询企业安全生产信息：
   - 使用 WebSearch 搜索："[企业名称] 安全生产"
   - 查询是否有行政处罚、事故记录
   - 检查是否按规定进行安全评价、应急预案备案

2. 生成监管清单：
   - 安全生产责任制是否建立
   - 安全管理人员是否配备
   - 安全培训是否定期开展
   - 隐患排查治理是否到位
   - 应急装备和演练是否落实

3. 生成监管报告：
   - 企业基本信息
   - 安全生产现状
   - 存在问题和风险点
   - 整改建议
   - 监管记录

### 5. 企业发展建议

结合崖州区产业发展情况，为企业提供发展建议：

**数据收集：**
1. 查询崖州区产业发展规划（参考 references/industrial-development.md）
2. 查询崖州区重点发展产业、产业政策、产业布局
3. 分析企业所属行业在崖州区的发展前景
4. 了解崖州区产业扶持政策和资源配置

**建议内容：**
1. **产业定位建议**：根据崖州区产业发展规划，建议企业如何定位自己
2. **市场机会建议**：分析崖州区市场需求，提出市场拓展建议
3. **技术创新建议**：根据崖州区科技创新政策，建议企业技术创新方向
4. **人才引进建议**：根据崖州区人才政策，建议企业如何引进和培养人才
5. **合作机会建议**：分析崖州区产业链，建议企业合作机会
6. **政策利用建议**：建议企业如何充分利用崖州区的产业扶持政策

**建议流程：**
1. 收集企业信息：行业、产品、技术、市场、团队等
2. 分析崖州区产业环境：重点产业、产业政策、产业布局、市场需求等
3. 匹配企业优势与产业环境：找出契合点和机会点
4. 生成发展建议：具体、可操作的发展建议
5. 提供政策支持：告知企业可以申请的产业扶持政策和资金支持

## 注意事项

1. **信息准确性**：企业信息、政策信息和产业发展信息应通过多个渠道验证，确保准确性
2. **政策时效性**：注意政策的生效时间和失效时间，优先推荐有效期内的政策
3. **隐私保护**：企业查询信息仅用于公共服务，不得泄露或滥用
4. **持续更新**：政策文件和产业发展规划会持续更新，定期刷新 references/ 下的文件
5. **建议可行性**：发展建议应结合企业实际情况和崖州区产业环境，确保可行性和实用性
6. **动态调整**：崖州区产业发展重点可能随时间和政策调整，建议定期更新产业发展信息
7. **多企业报告**：多个企业的报告按用户提供的顺序罗列，每个企业之间用分页符分隔
