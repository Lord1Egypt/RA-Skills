# 🦅 SIAS Research Skill v1
A specialized deep-research skill for the **SIAS Multi-Agent System** (metamaus).
Optimized for: FOIA requests, NGO network analysis Cologne/NRW, public funding research.
## 📦 Installation
cp -r sias-research-v1/ ~/.openclaw/skills/
## ⚙️ Modes
| Mode | Description |
|---|---|
| `ifg` | FOIA requests, deadlines, official responses |
| `ngo` | NGO networks, connections, funding structures |
| `mail` | Process incoming government correspondence |
| `rapid` | Quick research, key insight in 1 paragraph |
## 🤝 Compatible Agents
- **Rheingold** 🦅 — Primary agent (research & analysis)
- **Hermes** ⚡️ — Support agent (scraping & raw data)
