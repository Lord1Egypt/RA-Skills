# RedNote Dimensions Reference

## Style

| Value | Best For | Recommended Mapping |
| --- | --- | --- |
| `cute` | sweet sharing, soft lifestyle content | `anime` |
| `fresh` | clean, natural, healthy, light product content | `flat-illustration` |
| `warm` | friendly, healing, story-like content | `watercolor` |
| `bold` | warnings, reminders, and high-contrast opinions | `poster` |
| `minimal` | concise, professional, light business summaries | `editorial` |
| `retro` | nostalgic rankings and retro recommendations | `poster` |
| `notion` | knowledge cards, tools, and concept organization | `flat-illustration` |
| `chalkboard` | tutorials, explanation, classroom feel | `chalk` |
| `editorial` | commentary, opinion, culture, poster-like mood | `editorial` |

## Layout

| Value | Best For |
| --- | --- |
| `sparse` | 1 to 2 key points with strong visual impact |
| `balanced` | 3 to 4 key points, the most general option |
| `dense` | 5 to 8 points in a knowledge-card style |
| `list` | lists, rankings, and summary cards |
| `comparison` | comparisons, pros and cons, before/after |
| `flow` | steps, timeline, and process cards |
| `mindmap` | branching knowledge structures |
| `quadrant` | four-quadrant or category splits |

## Aspect

Recommended defaults:

- `3:4` for portrait card covers
- `1:1` for general social cards
- `9:16` for taller portrait-style cards

## Language

Recommendations:

- Follow the user's language by default.
- Keep one primary language across the whole series unless the user explicitly wants a mixed-language result.
- If image text is required, state the language explicitly in the prompt.
