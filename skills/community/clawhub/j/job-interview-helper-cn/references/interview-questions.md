## 行为面试题（STAR法则）

**Q: 请举例说明你如何与团队合作完成目标？**

高分话术模板：
```
S: 在XX项目中，我们需要完成XX目标
T: 我负责XX部分，需要与XX角色协作
A: 我主动建立了XX沟通机制，每周同步进度；遇到XX分歧时，我提出XX解决方案
R: 项目按期交付，团队协作顺畅，获得了XX评价
```

**Q: 请举例说明你如何在压力下工作？**

## 岗位专业题

### 产品经理

**Q: 你如何发现用户需求？**
```
高分思路：
1. 用户调研：访谈、问卷、观察
2. 数据分析：行为数据、反馈数据
3. 场景分析：用户旅程、痛点识别
4. 竞品分析：对标产品、差异化机会

示例：
在XX项目中，我通过XX方式接触了XX个目标用户，发现XX痛点，最终设计了解决方案XX...
```

**Q: 如何衡量一个功能是否成功？**
```
高分思路：
1. 北极星指标：核心价值指标
2. 过程指标：使用率、留存、转化
3. 业务指标：GMV、收入、成本

示例：
以XX功能为例，我会关注：使用率（XX%）、7日留存（XX%）、对核心指标的贡献（XX%）...
```

### 运营

**Q: 你如何做用户增长？**
```
高分思路：
AARRR模型：获取→激活→留存→变现→推荐
具体策略：渠道投放、裂变活动、内容营销、用户分层

示例：
在XX项目中，我通过XX渠道获取用户，设计XX活动促进激活，建立XX机制提升留存...
```

**Q: 如何提升用户留存？**
```
高分思路：
1. 找到流失节点：漏斗分析
2. 优化核心体验：Aha moment
3. 建立回访机制：Push、活动、权益
4. 用户分层运营：RFM模型

示例：
分析发现用户在XX环节流失率最高，我通过XX优化提升了XX%留存...
```

### 数据分析

**Q: 你如何理解数据分析的价值？**
```
高分思路：
1. 描述性分析：发生了什么
2. 诊断性分析：为什么发生
3. 预测性分析：未来会发生什么
4. 指导性分析：应该怎么做

示例：
在XX项目中，我通过数据分析发现XX问题，提出XX建议，最终实现了XX成果...
```

**Q: 常用的数据分析方法有哪些？**
```
高分思路：
1. 对比分析：同比、环比、对标
2. 漏斗分析：转化、流失
3. 留存分析：cohort分析
4. 相关分析：相关系数、回归
5. 分类分析：聚类、决策树

结合案例说明效果更好
```

### 市场营销

**Q: 你如何做营销策划？**
```
高分思路：
STP理论：细分市场→目标市场→定位
4P理论：产品、价格、渠道、促销
整合营销：线上线下结合

示例：
针对XX产品，我分析目标人群是XX，定位XX，通过XX渠道触达，设计XX活动...
```

## 群面题型

### 类型1：优先级排序

**题目示例：**
> 公司预算有限，请从以下8个项目中选出最重要的3个进行投资

**高分策略：**
```
1. 快速建立评价标准（如：ROI、战略价值、紧急性）
2. 引导团队达成标准共识
3. 按标准逐一评估
4. 总结归纳，形成结论

话术：
"我建议我们先确定评价标准，比如XX和XX，这样可以更客观地比较..."
"根据我们确定的标准，XX项目得分最高，因为..."
```

### 类型2：方案设计

**题目示例：**
> 为某产品策划一个营销活动方案，30分钟讨论

**高分策略：**
```
1. 快速框架化：目标→策略→执行→衡量
2. 贡献核心观点
3. 整合他人意见
4. 总结陈述

话术：
"我们可以从目标出发，先明确活动要达成什么..."
"XX同学的观点很好，我可以补充XX..."
"我来总结一下我们讨论的要点..."
```

### 类型3：问题解决

**题目示例：**
> 公司某产品销量下滑，请分析原因并提出解决方案

**高分策略：**
```
1. 结构化分析：外部环境→内部问题
2. 提出假设并验证
3. 制定可执行方案
4. 预估效果和风险

话术：
"我们可以从内外两个维度分析，外部可能是XX，内部可能是XX..."
"针对XX问题，我建议采取XX方案，预计可以XX..."
```

### 群面角色分工

| 角色 | 职责 | 关键行为 |
|------|------|---------|
| Leader | 组织推进 | 分配任务、把控时间、整合观点 |
| Timer | 时间管理 | 提醒进度、确保按时完成 |
| 记录员 | 整理输出 | 记录要点、形成框架 |
| 贡献者 | 内容输出 | 提出观点、补充细节 |

**注意：** 不要刻意抢角色，自然发挥即可

## 反问环节

### 高质量问题

**关于岗位：**
- 这个岗位最需要的能力是什么？
- 团队目前面临的最大挑战是什么？
- 入职后前3个月的期望是什么？

**关于发展：**
- 这个岗位的职业发展路径是怎样的？
- 公司对新人有什么培养机制？
- 团队的氛围如何？

**关于业务：**
- 公司未来的战略重点是什么？
- 这个业务线的核心指标是什么？
- 与竞品相比，我们的优势是什么？

### 避免的问题
- 薪资待遇（HR面再问）
- 加班情况（负面印象）
- 网上能查到的信息
- "没有问题了"（错失机会）
