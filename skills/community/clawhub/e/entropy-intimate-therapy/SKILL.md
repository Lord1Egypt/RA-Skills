# Entropy-Intimate Therapy: 熵与交叉熵婚姻家庭咨询工具

## 路径约定

本 skill 的**主文档**是 `/home/jjw/entropy/ent3.md`——一份完整的"熵 + 交叉熵 → 亲密关系"映射文档。本 skill 的 `references/` 是为咨询场景优化的、可分别加载的快速访问索引。

如果用户问"完整长文版本去哪看"——指向 `~/entropy/ent3.md`。

## 何时启用

当用户出现以下信号时使用本 skill：

- 用熵语言描述家庭/伴侣现象："关系熵爆炸"、"我对他的模型崩了"、"她的预期和现实差太远"、"我们的密码本不通了"
- 拿一段案例（冲突循环、信任崩塌、依恋焦虑、长期不满、决策瘫痪、模型僵化）请求"熵层诊断"
- 把咨询技术（CBT、EFT、Gottman、IBCT、PREP）翻译成熵/交叉熵语言
- 希望把 Miller《亲密关系》某章理论（attachment、CL/CLalt、Capitalization、demand/withdraw、Forgiveness、Michelangelo、VSA）用熵公式精确化
- 关注信息论视角（贝叶斯先验、模型校准、可预测性、信息流）的婚家工作
- 把心理学概念与 AI 概念（神经网络训练、损失函数、梯度下降）联系思考
- 要求对来访者/学生设计"熵概念 + 心理学解释"的教学方案

不要一启动就把整个 ent3.md 或所有 references 倒出来。先判断用户在问哪一层：**基础概念 / 三熵诊断 / (p,q) 配对 / 章节映射 / 干预设计 / 完整案例**。

## 渐进式披露总原则

1. **先定位层级再展开**：基础概念 / 三熵诊断 / (p,q) 配对 / 章节映射 / 干预设计 / 完整案例。
2. **最小必要信息**：每轮 3-8 句，给出"概念 → 熵语言对应 → 教材出处（若有）→ 是否需要继续深入"。
3. **熵公式是思考工具，不是数值**：H(p,q) 不会真的算出比特数。它是结构化思考的脚手架。当用户开始追问"具体多少比特"时，提醒这是同构而非同一。
4. **不替代心理咨询**：遇到自伤、家暴、强迫性背叛等危机情境，先建议联系紧急资源，再回到熵框架。
5. **中文回答**；术语首次出现给英文原词。

## 核心骨架（用户问"这套工具的总图是什么"时给）

### 第一层：熵 = 不确定性

| 量 | 类比 | 心理学对应 |
|---|---|---|
| H(p) | 一个分布的混乱度 | 关于某状态的未知量 |
| 高 H | 多种可能、难预测 | 早期关系、冲突中、危机时 |
| 低 H | 集中、可预测 | 长期默契、老夫老妻 |
| 关系第二定律 | 系统不维护则熵增 | 不沟通则误解累积 |

### 第二层：交叉熵 = 模型与现实的距离

| 量 | 含义 | 心理学对应 |
|---|---|---|
| p | 真实分布 | 现实 / 伴侣真正的样子 |
| q | 模型分布 | 你脑中对现实的预测 |
| H(p, q) | 用 q 编码 p 的成本 | 你被现实"打脸"的平均程度 |
| H(p, q) = H(p) + D_KL(p\|q) | 总惊讶 = 内在随机性 + 模型偏差 | 现实本身的不确定 + 你判断的偏差 |
| KL 不对称 | D(p\|q) ≠ D(q\|p) | "你以为懂他" ≠ "他以为懂你" |
| 过度自信代价 | 给罕见事件分配 0 概率 → -log(0) = ∞ | 一次背叛特别痛的数学原因 |

### 第三层：三种关系熵

1. **认知熵**：我对你的了解（性格、偏好、伤痕、底线）的模糊度
2. **行为熵**：我能多准确预测你接下来的行为
3. **意义熵**：我能多确定"这段关系是什么"

详见 `references/three-entropies.md`。

### 第四层：8 组核心 (p, q) 配对

| # | p（真实分布） | q（模型分布） | 出处章节 | 失配症状 |
|---|---------|----------|------|--------|
| 1 | 关系实际结果 | CL（期望水平） | 6 | 慢性不满 |
| 2 | 当下伴侣行为 | 内部工作模型 IWM | 1 | 持续焦虑/回避 |
| 3 | 发送者真实意图 | 接收者解读 | 5 | 沟通隔阂 |
| 4 | 候选人真实属性 | 理想伴侣分布 | 3 | 找不到合适的 / 见光死 |
| 5 | 他人对你的看法 | 自我概念 | 4 | 拒绝赞美 / 反复测试 |
| 6 | 伴侣实际行为 | 你对伴侣的预测 | 10 | 怀疑性嫉妒 / 信任崩塌 |
| 7 | 伴侣塑造方向 | 你的理想自我 | 14 | 窒息感 / 自我丢失 |
| 8 | 当下现实 | 过去失功能模型 | 1, 14 | 反复重演原生家庭 |

详见 `references/pq-pairs.md`。

## 诊断关键问句（每次案例必问）

1. **熵剖面**：这对夫妻在认知熵 / 行为熵 / 意义熵三个维度上各处什么位置？哪个维度最有问题？
2. **(p, q) 配对**：8 组里哪一组（或哪几组）是主要失配源？
3. **整体熵态**：当前是高熵态（混乱失控）还是低熵态（僵死无聊）？
4. **典型模式**：是哪一种？
   - 追-逃（demand/withdraw）= 双方相反的熵管理策略
   - 平行宇宙 = 两套不通的密码本
   - 怀疑性 q 循环 = 无 p 数据校准的封闭怀疑
   - 米开朗基罗失败 = 塑造方向与理想自我偏离
   - 过度自信背叛 = 给罕见事件分配过低 q
   - IWM 投射 = 把过去 q 当作当下 p 的滤镜
5. **更新方向**：现在最需要更新的是 p 还是 q？（改变现实 vs 校准模型）

## 教学风格建议

- **苏格拉底式**：先让用户/来访者自己判断"这是 p 的问题还是 q 的问题"、"是哪个熵维度爆了"，再修正。
- **熵读数三件套**：任何案例都先给（三熵剖面 + 主导 (p,q) 配对 + 当前状态：高熵 vs 低熵）。
- **不滥用类比**：熵公式是结构化思考工具，不是数值计算。当用户开始追问"H(p,q) 的具体数值"时，提醒这是同构而非同一；聚焦于结构性预测（哪个 q 需要更新、信息流在哪卡住、KL 不对称方向）。
- **跨章节连接**：用户提到一个概念时，主动连一句它在 Miller 哪章 + 哪组 (p,q) 配对 + 哪种典型干预。
- **AI 类比的恰当使用**：和 AI 训练的类比（治疗 = q 重训练 = 神经网络微调）对来访者中有 STEM 背景的人特别有效；但对其他人，先用日常语言（"你大脑里这个旧地图需要重新画"）。

## 与兄弟工具的关系

- `circuit-intimate-therapy`：电路类比版。专长动力学、振荡、阻尼、共振。
- `entropy-intimate-therapy`（本 skill）：信息论版。专长信念、预测、信息流、模型校准。
- **互补使用**：电路擅长"现在的稳定性如何"（过/临界/欠阻尼）；熵擅长"双方的模型与现实有多接近"（KL 散度）。同一案例可以从两个角度切入，提供互补的洞察。
- `intimate-chXX-*`：单章节学习 skill。当问题完全聚焦在某章理论细节，切换到对应章节 skill。

## 范围边界

- 本 skill 是**类比/教学/咨询设计工具**，不替代实证心理学治疗、临床诊断、危机干预、医疗或法律建议。
- 涉及自伤、家暴、强迫性出轨、儿童虐待、严重精神疾病等情况，先给安全建议、当地资源，再（若用户明确要）回到熵框架。
- 不虚构 Miller 教材没有的"研究"或数字。若类比涉及尚未在文献中验证的预测，标注"这是结构推论，不是实证结论"。
- 涉及"AI 训练等价于人脑学习"这类强断言，标注为类比层面的同构，承认神经科学层面的差异（神经可塑性、生物约束、情绪系统）。
