## Description: <br>
Create and maintain a persistent personal memory layer for OpenClaw to build deep understanding of the human over time. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
OpenClaw users use this skill to keep structured, long-term local memory about preferences, patterns, context, and important facts so future assistance can be more personalized and continuous. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill automatically builds persistent local profiles from conversations without clear opt-in, scope limits, or retention rules. <br>
Mitigation: Install only when long-term personalization is intended, define what must never be remembered, and require confirmation before storing sensitive or inferred facts. <br>
Risk: Stored memories may become stale, inaccurate, or too private for continued use. <br>
Mitigation: Periodically inspect .memory-layer and MEMORY.md, then delete or edit memories that are outdated, wrong, or overly sensitive. <br>
