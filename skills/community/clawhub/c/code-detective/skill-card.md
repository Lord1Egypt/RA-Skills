## Description: <br>
Code Detective is an interactive debugging practice game where users find bugs in Python and JavaScript snippets across three difficulty levels. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[russianoracle](https://clawhub.ai/user/russianoracle) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Learners and developers use this chat-based skill to practice debugging by identifying syntax, logic, and common programming errors in short Python and JavaScript examples. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Future versions could add executable files, external integrations, or account access beyond the current instruction-only game. <br>
Mitigation: Review future releases before installation if new executable behavior, integrations, or permission requests are added. <br>
Risk: Users may treat exercise snippets as production coding guidance. <br>
Mitigation: Present snippets as debugging practice examples and keep explanations focused on the specific exercise bug. <br>
