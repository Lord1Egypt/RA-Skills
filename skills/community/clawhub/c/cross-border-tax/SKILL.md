# 跨境多国税务速查 (Cross-Border Tax Quick Lookup)

跨境电商卖家税务速查利器。输入国家名或自然语言问题，一分钟看清多国税率、申报周期、注册门槛和特殊规则，支持 EU OSS/IOSS 一站式申报方案。

## 快速开始

```bash
python ~/.workbuddy/skills/cross-border-tax/scripts/cross_border_tax.py "<查询内容>"
```

- `德国VAT税率` / `Germany VAT`

## 查询能力

每次查询自动返回以下信息（如适用）：
1. **标准税率** — 含减税率、零税率说明
2. **注册门槛** — 以当地货币和换算金额标注
3. **申报周期** — 月度/季度/年度及截止日期
4. **EU OSS方案** — 是否符合OSS/IOSS使用条件
5. **特殊规则** — 平台代收代缴、低价值商品规则、数字服务税
6. **近期待关注变化** — 税率调整、新规生效

## 核心EU OSS/IOSS知识

- **OSS阈值**: 欧盟跨境B2C销售合计超过€10,000/年必须使用OSS或逐国注册
- **OSS申报**: 每季度一次（Q1→4月30日, Q2→7月31日, Q3→10月31日, Q4→1月31日）
- **IOSS**: 适用于从非欧盟进口的单件价值≤€150的B2C包裹，月度申报
- **2026.7.1**: 150欧元关税豁免废止，每个包裹征收€3过渡统一关税
- **2028.3**: IOSS 150欧元上限取消；2028.7: ViDA单一VAT注册生效

## 输出格式

默认输出为交互式HTML可视化报告，包含：
- 税率卡片（颜色分级：绿<10% / 黄10-20% / 橙20-25% / 红>25%）
- 申报截止日时间轴
- EU OSS/IOSS资格判断
- 多国对比表
- 关键提醒/风险标注
