## 项目简介

本项目系统整合人体解剖学、神经解剖学、临床应用解剖及中西医结合解剖分析等核心领域，涵盖 **112 项关键解剖学技能**，分为 8 大分类。

**适用人群**：医学生、临床医师、解剖学研究者、外科规培医师

**参考教材**：人民卫生出版社《系统解剖学》第 10 版（主审：丁文龙；主编：崔慧先、刘学政）

**⚠️ 风险提示 ⚠️**：本技能集涵盖人体结构描述、解剖变异分析、外科手术入路评估及临床定位诊断等内容，不能替代专业解剖教学或临床操作训练。

### 提问示例

#### 1. 概念查询
```bash
内囊的位置和分部
```
```bash
## 内囊（Internal Capsule）
内囊是位于尾状核、豆状核和背侧丘脑之间的白质板，在水平切面上呈 **「><」形**。

### 分部
| 分部 | 位置 | 通过的纤维束 |
|------|------|-------------|
| **前肢** | 位于尾状核头与豆状核之间 | 额桥束、丘脑前辐射 |
| **膝** | 前肢与后肢的转折处 | 皮质核束（皮质延髓束） |
| **后肢** | 位于豆状核与背侧丘脑之间 | 皮质脊髓束、丘脑中央辐射、听辐射、视辐射 |
...

### 临床意义
- **内囊膝**损伤 → 对侧舌肌和面肌下部瘫痪（核上瘫）
- **内囊后肢**损伤 → 对侧偏瘫（皮质脊髓束）+ 对侧偏身感觉障碍（丘脑中央辐射）
```

#### 2. 临床案例分析
```bash
患者男性，65岁，突发右侧肢体无力伴言语不清3小时。查体：右侧上下肢肌力0级，右侧中枢性面瘫，右半身痛温觉减退，双眼向左侧凝视。请分析可能的病变部位。
```
```bash
## 病例分析

| 项目 | 表现 |
|------|------|
| 年龄 | 65岁 → 脑卒中高危人群 |
| 起病 | 突发（急性）→ 血管性 |
| 运动障碍 | 右侧偏瘫（0级）+ 中枢性面瘫 → 左侧皮质脊髓束 + 皮质核束 |
| 感觉障碍 | 右半身痛温觉减退 → 左侧丘脑中央辐射 |
| 凝视 | 双眼向左侧凝视 → 左侧额叶凝视中枢（额眼区）或破坏性病灶 |

### 定位诊断
**左侧内囊后肢病变**（最可能）
- 皮质脊髓束、丘脑中央辐射同时受累
- 典型的 **"三偏"综合征** 不完整型（偏瘫 + 偏身感觉障碍）
- 内囊病变可同时压迫或影响内囊后肢多个纤维束

### 鉴别 - 与大脑皮质病变区别
| 特征 | 内囊病变 | 大脑皮质病变 |
|------|----------|-------------|
| 偏瘫范围 | 上下肢均等 | 可能上肢重于下肢（或反之） |
| 感觉障碍 | 偏身均等 | 可能某部分更明显 |
| 凝视 | 不一定出现 | 额叶凝视中枢病变 → 凝视偏向病灶侧 |
| 失语 | 无 | 优势半球病变可出现 |

### 紧急处理
1. 即刻头颅CT排除出血
2. 评估rt-PA溶栓适应证（发病3小时内）
3. 保持气道通畅，监测血压、血糖
4. 抗血小板、他汀类药物
```

#### 3. 考试题目
```bash
下列关于内囊的描述，错误的是：
A. 位于尾状核、豆状核和背侧丘脑之间
B. 膝部有皮质核束通过
C. 后肢有皮质脊髓束通过
D. 内囊损伤不会引起感觉障碍
```
```bash
**答案：D. 内囊损伤不会引起感觉障碍**

**解析：**
- A ✅ 正确。内囊位于尾状核、豆状核和背侧丘脑之间的白质板
- B ✅ 正确。内囊膝有皮质核束（皮质延髓束）通过，支配脑神经运动核
- C ✅ 正确。内囊后肢有皮质脊髓束通过，支配躯干和四肢的随意运动
- D ❌ 错误。内囊后肢还有**丘脑中央辐射**通过，传导全身的浅感觉和深感觉，
  因此内囊损伤可导致对侧偏身感觉障碍

**拓展：**内囊完全损伤 → 典型"三偏综合征"：
1. 对侧偏瘫（皮质脊髓束）
2. 对侧偏身感觉障碍（丘脑中央辐射）
3. 对侧同向性偏盲（视辐射）
```
