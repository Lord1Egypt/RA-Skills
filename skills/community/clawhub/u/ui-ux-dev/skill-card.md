## Description: <br>
Generate and serve live HTML/CSS/JS UI designs from natural language prompts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wing8169](https://clawhub.ai/user/wing8169) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and design-focused builders use this skill to generate, serve, visually review, iterate on, and export React-based UI prototypes or frontend pages from natural language prompts. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The setup script can make persistent privileged Nginx and system-directory changes that are broader than a design workflow needs. <br>
Mitigation: Run setup only in a disposable or isolated environment, review setup.sh before sudo use, choose a non-sensitive serve directory and port, and verify served or zipped files before sharing exports. <br>
