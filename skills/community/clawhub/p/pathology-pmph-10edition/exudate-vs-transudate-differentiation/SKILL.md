# 渗出液与漏出液的鉴别标准

## 适用场景
- 患者存在浆膜腔（胸腔、腹腔、心包）或组织间隙积液
- 已采集积液样本并完成实验室检测
- 需初步判断积液是否由炎症引起

## 执行步骤

1. **获取关键检测指标**：
   - 积液蛋白浓度（g/L）
   - 积液比重
   - 积液细胞计数（×10⁶/L）
   - 外观与凝固性

2. **对照以下标准进行判断**：

   | 指标 | 渗出液 | 漏出液 |
   |---|---|---|
   | **原因** | 炎症（感染、肿瘤、自身免疫病等） | 非炎症因素（心衰、低蛋白血症、肝硬化、肾病综合征等） |
   | **蛋白浓度** | >30 g/L | <30 g/L |
   | **比重** | ≥1.018（多数 >1.020） | <1.018 |
   | **细胞计数** | 通常 >500×10⁶/L | 通常 <100×10⁶/L |
   | **外观与凝固性** | 浑浊、易自凝 | 清亮、不自凝 |

3. **综合判断**：
   - 若多数指标符合渗出液特征，提示活动性炎症，应警惕感染、恶性肿瘤或结缔组织病，需进一步病因排查。
   - 若多数指标符合漏出液特征，应评估心、肝、肾功能及血清白蛋白水平，排查全身性病理状态。

4. **注意事项**：
   - 排除混合性积液（如心衰合并肺炎）的干扰。
   - 单一指标异常不足以确诊，需结合临床背景综合分析。

> **触发条件**：IF 患者出现浆膜腔或组织间隙积液 AND 可获取液体样本 THEN 应用本鉴别标准。
