# Methods and Functions

> [!NOTE]
> The `rhinoscriptsyntax` (Python) and `RhinoScript` (VBScript) APIs nearly 100% mirrors.
> Function names match; only the call syntax differs. Many studios, firms, and individual users have
> years of accumulated RhinoScript code they haven't bothered to port.

- Function names match; only the call syntax differs
- `RhinoScript` (*VBScript*) has 3 method/function and property categories not included in
 `rhinoscriptsyntax` (*Python*)
  - See [vbscript-methods.md](vbscript-methods.md) for VBScript exclusive methods and properties

> [!IMPORTANT]
> `RhinoScript` is exclusive to Windows.
