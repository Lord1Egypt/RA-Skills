# 角色视角参考词典

> **这不是固定列表，是参考。** AI 读数据后应该动态派生视角，而不是照着这个表背。
> 实际遇到新场景时，根据字段名、数值范围和语境自己推——字典是帮你理解"视角"这个概念的，不是让你抄的。

不同角色看同一份数据，看到的东西完全不同。本文定义各角色的分析滤镜，也包括非 C 级岗位和普通用户视角。

---

## CEO — 首席执行官

**决策焦点：** 增长、风险、竞争位置、整体业务健康度

| 看什么 | 会问的问题 |
|--------|-----------|
| **增长趋势** | 营收/用户/订单是往上还是往下？增速在加速还是放缓？ |
| **风险雷达** | 最大的隐患在哪？客户集中度？单一供应商依赖？合规风险？ |
| **竞争位置** | 我们的数据在行业里算什么水平？有没有被颠覆的迹象？ |
| **资源分配** | 钱和人应该往哪投？哪个业务线最值得加注？ |
| **健康信号** | 有没有一票否决的问题（现金流断裂、重大违规）？ |

**分析语言风格：** 大局观、简洁、先讲结论再讲数据、风险前置。

---

## CFO — 首席财务官

**决策焦点：** 成本结构、利润质量、现金流、ROI、预算偏差、财务风险

| 看什么 | 会问的问题 |
|--------|-----------|
| **利润质量** | 毛利率稳定吗？净利润的趋势？利润来源是主业还是非经常性？ |
| **成本结构** | 固定成本 vs 变动成本占比？成本有没有失控的趋势？ |
| **现金流** | 回款周期多长？应收账款质量？资金链压力？ |
| **ROI** | 每块钱投出去回来多少？哪个业务线最赚钱？哪个在烧钱？ |
| **预算偏差** | 实际 vs 预算的差距？超支原因可解释吗？ |
| **财务风险** | 坏账率？存货跌价？汇率/利率风险敞口？ |
| **营运资金** | 库存周转天数、应收天数、应付天数、现金转换周期 |

**分析语言风格：** 精确、量化优先、每一个结论必须有数字支撑、关注"多少"和"多久"。

---

## COO — 首席运营官

**决策焦点：** 运营效率、产能瓶颈、资源利用率、交付质量、SLA 达标率

| 看什么 | 会问的问题 |
|--------|-----------|
| **效率指标** | 单位产出/人/天？流程有没有瓶颈环节？ |
| **产能利用** | 资源（设备/人力）闲置了多少？峰值够用吗？ |
| **交付质量** | 按时交付率？次品率？客户投诉率？ |
| **流程瓶颈** | 哪个环节最慢？排队时间 vs 处理时间？ |
| **库存管理** | 库存周转率？滞销品占比？缺货率？ |
| **SLA 达标** | 服务水平协议指标达成情况？ |

**分析语言风格：** 流程导向、关注吞吐量和时延、用瓶颈思维找改善点。

---

## CMO — 首席营销官

**决策焦点：** 客户分层、获客成本、渠道效率、品牌健康、客户生命周期价值

| 看什么 | 会问的问题 |
|--------|-----------|
| **客户分层** | 哪些客户贡献了大部分利润？二八法则的分布？高价值客户的画像？ |
| **获客成本** | 每个渠道的 CAC（获客成本）？哪个渠道 ROI 最高？ |
| **客户留存** | 复购率？流失率？沉默客户占比？ |
| **LTV** | 客户生命周期价值（一个客户整个合作期带来的总利润）？CAC 回本周期多久？ |
| **渠道效率** | 各渠道的转化率？预算分配合理吗？ |
| **品牌健康** | 净推荐值 NPS？口碑舆情？品牌认知度？ |

**分析语言风格：** 客户中心、关注分群和画像、用 LTV/CAC 框架、关注长期价值。

---

## CHO / CHRO — 首席人力资源官

**决策焦点：** 人效、人才结构、流失率、招聘效率、培训 ROI、组织健康

| 看什么 | 会问的问题 |
|--------|-----------|
| **人效** | 人均产出？人力成本占比？人效趋势？ |
| **人才结构** | 职级分布？关键岗位覆盖率？继任计划？ |
| **流失率** | 主动离职率？新员工留存率？离职原因分布？ |
| **招聘效率** | 招聘周期？offer 接受率？人均招聘成本？ |
| **培训 ROI** | 培训后的绩效改善？培训覆盖率？ |
| **组织健康** | 员工满意度？敬业度？内部晋升率？ |

**分析语言风格：** 以人为本、关注结构分布、关注趋势、关注"人"的层面。

---

## 首席战略官

**决策焦点：** 市场定位、竞争格局、长周期趋势、新市场机会、战略风险

| 看什么 | 会问的问题 |
|--------|-----------|
| **市场定位** | 我们在价值链的什么位置？护城河够深吗？ |
| **竞争格局** | 市场份额变化？竞争对手在做什么？ |
| **长周期趋势** | 结构性变化还是周期性波动？这个赛道 3 年后还在吗？ |
| **新机会** | 数据里有没有新业务线索？可拓展的相邻市场？ |
| **战略风险** | 最大的结构性风险是什么？被替代的可能性？ |

**分析语言风格：** 长期视角、关注结构性变化、区分信号和噪音、多问"所以呢"。

---

## 首席传播官

**决策焦点：** 声誉风险、公众感知、沟通机会、利益相关方关系

| 看什么 | 会问的问题 |
|--------|-----------|
| **声誉风险** | 数据中有没有什么如果公开会有负面影响的？ |
| **沟通机会** | 有没有好故事可以讲？数据能不能支撑 PR 叙事？ |
| **舆情监控** | 数据异常是否和舆情事件相关？ |
| **利益相关方** | 不同群体（客户/投资者/监管）对这份数据会有什么反应？ |

**分析语言风格：** 风险敏感、关注叙事、用"如果公开会怎样"来筛发现。

---

## CDO — 首席数据官

**决策焦点：** 数据质量、数据治理、合规、数据资产价值、数据架构

| 看什么 | 会问的问题 |
|--------|-----------|
| **数据质量** | 完整性？准确性？一致性？时效性？ |
| **数据治理** | 字段有定义文档吗？数据血缘清晰吗？谁负责这个数据？ |
| **合规风险** | 有没有违规收集的数据？敏感数据有没有加密？ |
| **数据资产** | 这份数据的业务价值？能不能复用？ |
| **架构健康** | 数据孤岛严重吗？ETL 流程稳定吗？ |

**分析语言风格：** 严谨、关注元数据、关注流程和标准、关注风险。

---

## 首席采购官

**决策焦点：** 供应商绩效、采购效率、供应链风险、采购成本、合同履约

| 看什么 | 会问的问题 |
|--------|-----------|
| **供应商绩效** | 按时交货率？质量合格率？响应速度？ |
| **采购成本** | 单价趋势？有没有价格异常？比价结果？ |
| **供应链风险** | 单一供应商依赖度？替代供应商？地缘政治风险？ |
| **采购效率** | 采购周期？审批效率？下单频次？ |
| **合同履约** | 合同 vs 实际的价差？履约率？ |

**分析语言风格：** 供应商视角、风险分散思维、关注合同和履约。

---

## CBO — 首席业务官

**决策焦点：** 业务拓展、合作伙伴、营收多元化、新市场进入

| 看什么 | 会问的问题 |
|--------|-----------|
| **营收多元化** | 有没有过度依赖单一业务/客户/渠道？ |
| **合作伙伴** | 哪些合作伙伴贡献了最多价值？潜在的合作缺口？ |
| **新市场信号** | 数据中有没有新的需求信号？新客户类型？新地域？ |
| **业务杠杆** | 什么可以快速复制放大？什么东西有网络效应？ |

**分析语言风格：** 机会导向、关注增长和扩展、关注关系网络。

## 策划 / 策略运营

**决策焦点：** 活动效果、用户转化、内容策略、预算分配、ROI 归因

| 看什么 | 会问的问题 |
|--------|-----------|
| **活动效果** | 活动带来的增量有多大？ROI 多少？哪个活动的转化最好？ |
| **用户转化** | 用户从看到→点击→转化的漏斗各环节流失多少？ |
| **内容策略** | 什么内容/素材的点击率最高？用户偏好什么类型？ |
| **预算分配** | 钱投在哪条线最有效？渠道效率排名？ |
| **ROI 归因** | 最后转化的用户最开始是从哪来的？多触点归因？ |

**分析语言风格：** 活动导向、关注漏斗和转化、用归因思维、关注增量而非总量。

## 产品经理（PM）

**决策焦点：** 用户需求、功能采纳、产品健康度、体验优化、产品市场匹配

| 看什么 | 会问的问题 |
|--------|-----------|
| **功能使用** | 用户用了什么功能？使用频率？哪些功能没人用？ |
| **用户行为** | 用户路径是什么？流失点在哪？新用户激活率？ |
| **产品健康** | DAU/MAU 趋势？留存曲线？核心指标波动？ |
| **体验优化** | 哪些环节用户卡住了？加载/响应时间？报错率？ |
| **PMF 信号** | 用户愿不愿意付费？主动推荐比例？竞品切换成本？ |

**分析语言风格：** 用户中心、关注行为而非数据、关注留存和黏性、区分信号和噪音。

---

## 运营经理

**决策焦点：** 日常指标、用户分层、活动运营、内容运营、用户生命周期

| 看什么 | 会问的问题 |
|--------|-----------|
| **日常指标** | 核心指标（活跃/收入/转化）有没有异常波动？ |
| **用户分层** | 新用户、活跃用户、沉默用户、流失用户的分布和趋势？ |
| **活动运营** | 活动带来的用户质量？（留存/ARPU vs 自然用户） |
| **内容运营** | 内容的展示量、点击率、互动率？什么类型的内容效果好？ |
| **生命周期** | 用户从进来到流失的典型周期？什么时候最容易走？ |

**分析语言风格：** 指标敏感、关注波动和异常、关注分层、关注动作和反馈。

---

## 销售经理

**决策焦点：** 销售漏斗、商机转化、客户开拓、目标达成、团队绩效

| 看什么 | 会问的问题 |
|--------|-----------|
| **销售漏斗** | 线索→商机→报价→成交，各环节转化率？哪里卡住了？ |
| **商机质量** | 赢单率？平均客单价？赢单周期？ |
| **客户开拓** | 新客户 vs 老客户占比？获客渠道质量？ |
| **目标达成** | 各团队/个人的目标完成率？趋势是向上还是向下？ |
| **团队绩效** | 谁超了目标？谁落后？人均产出多少？ |

**分析语言风格：** 目标导向、关注漏斗转化、关注数字和排名、关注 actionable 的商机。

## 设计师

**决策焦点：** 用户体验、可用性、视觉效果、设计系统健康度

| 看什么 | 会问的问题 |
|--------|-----------|
| **可用性** | 用户在界面上卡在哪？点击热区？误操作率？ |
| **视觉效果** | 哪个视觉方案的点击/转化最好？AB 测试结果？ |
| **设计系统** | 组件复用率？设计规范偏离度？ |
| **用户反馈** | 用户满意度评分？NPS？用户评论中的设计关键词？ |

**分析语言风格：** 体验导向、关注行为而非审美、用 AB 测试思维、量化设计决策。

## 工程师 / 技术负责人

**决策焦点：** 系统性能、可靠性、技术债、研发效率、代码质量

| 看什么 | 会问的问题 |
|--------|-----------|
| **系统性能** | 响应时间？吞吐量？错误率？资源利用率？ |
| **可靠性** | 可用性 SLA？故障频次？MTTR？ |
| **研发效率** | 需求交付周期？部署频率？代码 Review 效率？ |
| **技术债** | 遗留代码占比？依赖版本落后程度？测试覆盖率？ |
| **成本效率** | 服务器成本 vs 业务增长？单位请求成本？ |

**分析语言风格：** 精确、关注指标趋势、追求根因、用 SRE 思维。

---

## 数据分析师

**决策焦点：** 数据质量、分析深度、可复现性、洞察质量、数据驱动决策

| 看什么 | 会问的问题 |
|--------|-----------|
| **数据质量** | 数据完整吗？准确吗？口径定义一致吗？ |
| **分析深度** | 是描述性（发生了什么）还是诊断性（为什么发生）？有没有预测性？ |
| **可复现性** | 分析过程能不能复现？逻辑有没有漏洞？数据血缘清晰吗？ |
| **洞察质量** | 这个是已知信息还是真洞察？用户能拿着它做决策吗？ |
| **数据驱动** | 有多少决策是数据驱动的？分析和决策之间有没有闭环？ |

**分析语言风格：** 严谨、关注方法论、区分描述/诊断/预测/规范、关注可复现性和血缘。

## 合规/审计官

**决策焦点：** 利益冲突、飞单/舞弊风险、违规信号、数据合规、内控制度有效性

| 看什么 | 会问的问题 |
|--------|-----------|
| **利益冲突** | 员工和客户/供应商之间有没有异常关联？自导自演的可能性？ |
| **舞弊信号** | 数据里有没有不符合正常模式的记录？集中度高？时间异常？金额异常？ |
| **合规风险** | 数据收集和使用是否符合法规（个保法、行业监管）？敏感字段有没有脱敏？ |
| **内控有效性** | 审批流程被绕过了吗？权限分配合理吗？异常记录有没有被标记和处理？ |
| **证据链** | 每个风险信号的证据能追溯到原始记录吗？证据够不够定性？ |

**分析语言风格：** 克制、不下定性结论（只标风险信号不定性违规）、强调"需人工核实"、用"疑似""待确认"而非"确认违规"、关注证据链完整性。
