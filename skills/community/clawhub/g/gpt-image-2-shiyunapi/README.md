# GPT-image-2 图片生成与编辑诗云API Skill

## 目录结构

```text
gpt-image-2-shiyunapi/
├── SKILL.md
├── README.md
├── scripts/
│   ├── generate_image.py
│   ├── edit_image.py
│   └── save_api_key.py
└── references/
    ├── api_docs.md
    ├── examples.md
    └── troubles.md
```

## 使用方式

- 生成新图片：使用 `scripts/generate_image.py`。
- 编辑已有图片：使用 `scripts/edit_image.py`。
- 保存诗云 API Key：使用 `scripts/save_api_key.py --api-key-stdin`。

默认模型为 `gpt-image-2`。
