## Description: <br>
This skill provides a day-by-day SOP for an autonomous SEO/GEO agent to run audits, analyze GSC data, track AI-search traffic, validate schema, configure IndexNow, and produce SEO reports and optimization actions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External developers, founders, product owners, and growth teams use this skill to guide an agent through recurring SEO/GEO operations, including daily ranking reports, keyword-to-landing-page mapping, CTA blocks, schema checks, and indexing workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may guide an agent to use analytics or search API accounts. <br>
Mitigation: Connect only the intended GSC, GA4, DataForSEO, IndexNow, or search API credentials, and scope credentials to the sites and actions needed. <br>
Risk: The skill may guide changes to public website SEO assets such as content, schema, robots.txt, publishing, or indexing settings. <br>
Mitigation: Review proposed content, schema, robots.txt, and publishing changes before allowing unattended execution. <br>
