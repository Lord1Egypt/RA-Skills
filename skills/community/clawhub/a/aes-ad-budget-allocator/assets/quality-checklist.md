# Allocation Quality Checklist

Run through every item before delivering an allocation plan. A "no" anywhere means revise before sending.

## Inputs verified
- [ ] Total budget number is confirmed in writing
- [ ] Window dates are correct (no off-by-one month)
- [ ] Currency and FX assumptions stated
- [ ] All channel-level performance is 30-90 days fresh
- [ ] Attribution model is documented
- [ ] Gross margin assumption is documented

## Principles documented
- [ ] Blended ROAS floor stated
- [ ] Funnel-stage target ratios stated
- [ ] Reserve percentage stated
- [ ] Channel min/max stated
- [ ] New-channel test policy stated

## Marginal analysis done
- [ ] At least 8 weeks of data per scaled channel
- [ ] Curve shape identified per channel
- [ ] Marginal ROAS estimated per channel
- [ ] Saturation flagged where present
- [ ] Channels with thin data labeled as "low confidence"

## Funnel logic applied
- [ ] Prospecting/retargeting/brand split documented
- [ ] Retargeting budget capped at audience refresh rate
- [ ] Brand search capped at expected cannibalization
- [ ] TOF audience sizes match prospecting spend
- [ ] Each line item tagged with funnel role

## Seasonality applied
- [ ] Demand index per week documented
- [ ] CPM inflation buffer applied per peak window
- [ ] CVR lift assumed per peak window
- [ ] Trough weeks identified and de-weighted
- [ ] Creative lead-time built into the plan

## Outputs complete
- [ ] Channel × month allocation table reconciles to total
- [ ] Funnel summary matches channel table
- [ ] Expected ROAS per row
- [ ] Expected revenue per row
- [ ] Confidence (H/M/L) per row

## Triggers and kill criteria
- [ ] Reallocation rule for each top-3 channel
- [ ] Kill criterion for every new test
- [ ] Reserve deployment rule
- [ ] Pacing-check cadence stated

## Risk and review
- [ ] Top 3 assumptions called out
- [ ] Impact of each assumption tagged H/M/L
- [ ] Review cadence agreed
- [ ] Stakeholders named for sign-off
