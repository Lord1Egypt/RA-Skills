## 完整说明文档

### 一、这是什么？

**黄毛决胜法（Alpha Game Mastery）** 是一套基于博弈论、社交心理学与情绪价值供给理论的**魅力提升体系**。

它回答了情感博弈中最核心的问题：

> **"为什么老实人玩不过黄毛？"**

答案不是"女人都瞎了眼"。
答案是：
> **老实人输在姿态，赢在兑现。黄毛赢在姿态，输在兑现。**
> **黄毛 Pro = 黄毛的姿态 + 老实人的兑现。**

### 二、底层逻辑

这套体系建立在三大支柱上：

1. **博弈论** — 亲密关系本质是一场非零和博弈，谁定义框架谁占优
2. **情绪价值理论** — 人们追求的是"和谁在一起时的感觉"，而非"谁更对我好"
3. **稀缺原理** — 任何东西一旦无限供给，立刻就贬值

### 三、为什么叫"黄毛"？

"黄毛"是网络用语中一类角色的统称——他们自信、主导、不在乎别人看法。

本技能的命名方式是**借喻**：

> 不是教你成为"黄毛"，而是解剖黄毛的武器库，把**可用的武器**提取出来，**扔掉有毒的部分**，为你所用。

### 四、谁适合使用？

| ✅ 适合 | ❌ 不适合 |

### 七、常见问答

**Q: 这不就是教人当渣男吗？**
A: 恰恰相反。渣男是黄毛——只有姿态没有兑现。黄毛 Pro 是把姿态和兑现结合起来，既要自信也要负责。

**Q: 我本来就很内向，能学会吗？**
A: 内向不是问题，问题是**低姿态**。黄毛 Pro 不是要你变成外向的人，而是要你**在现有的性格框架内建立高位姿态**。

**Q: 用这些技巧会不会显得不真诚？**
A: 真诚和低位不是绑定关系。你可以既有框架又真诚——**框架内的真诚才是最有力量的真诚。**
