## Description: <br>
Usage Visualizer provides advanced usage statistics and high-fidelity visual reporting for OpenClaw with local processing and no credential storage. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[VintLin](https://clawhub.ai/user/VintLin) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and OpenClaw users use this skill to turn local session usage logs into text, JSON, and PNG usage reports for cost, token, model, and provider analysis. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill scans local OpenClaw/Clawdbot session logs for usage metadata. <br>
Mitigation: Run it only in workspaces where local session log analysis is expected and acceptable. <br>
Risk: The skill keeps a local SQLite usage database derived from usage logs. <br>
Mitigation: Store and retain the generated database according to the user's local data handling requirements. <br>
Risk: Dependency installation and Chromium-based rendering add local execution surface. <br>
Mitigation: Install dependencies in a trusted environment, keep them current, and avoid custom output paths in sensitive directories. <br>
