## Description: <br>

## Use Case: <br>
Developers, designers, and agent users use this skill to keep coding and design requests structured, scoped, and efficient so assistants spend fewer tokens on repeated context or broad rewrites. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The broad trigger may make an assistant favor terse, structured, token-saving behavior even when the user did not explicitly ask for it. <br>
Mitigation: Use the skill when efficient coding or design collaboration is desired, and disable or override it for tasks where fuller exploration, explanation, or creative elaboration is more important. <br>
