# transfer-pricing-benchmarking-memo

Draft an OECD Transfer Pricing Guidelines–aligned benchmarking memo for a single intercompany (controlled) transaction, in the structure that tax authorities expect inside a BEPS Action 13 Local File: functional / asset / risk (FAR) profile → tested-party selection → method selection → search strategy → acceptance and rejection log → comparability adjustments → interquartile-range conclusion.

## When to use

A transfer-pricing professional — in-house tax manager, Big-4 / boutique TP consultant, TP economist, tax-controversy specialist, or APA / MAP team member — has:

- a single, defined intercompany transaction to defend (e.g. routine distribution, contract manufacturing, intra-group services, intercompany loan, IP licence, captive R&D),
- a tested-party candidate in mind (or wants help picking one),
- access to a commercial comparables database (e.g. Orbis / TP Catalyst, Royaltystat, Loan Connector, Compustat) or has run a search and exported the longlist, and
- needs a memo that documents every benchmarking decision so the analysis survives a tax-authority audit.

The skill does **not** give legal or tax advice, run the database search itself, or set a transfer price. It produces a DRAFT memo for licensed in-house tax / TP-economist review and sign-off.

## What it produces

- Transaction summary with currency, fiscal year, jurisdictions, and intercompany agreement reference
- Functional / asset / risk (FAR) profile for each party
- Tested-party rationale (least-complex party test) with explicit alternatives considered
- Method selection (CUP / Resale Price / Cost Plus / TNMM / Profit Split) with rejection reasons for the other four
- Profit-level indicator (PLI) and rationale, where TNMM is selected
- Search strategy: database + version date, geographic scope, NACE / SIC / NAICS codes, time period, independence test, quantitative & qualitative screens — fully reproducible
- Acceptance / rejection log: every comparable accepted or rejected with a coded reason
- Comparability adjustments (working capital, accounting differences, idle capacity) with formulas
- Interquartile range (25th / median / 75th) and tested-party result placement
- Conclusion (within range / outside range → adjustment needed) with adjustment quantum if applicable
- Penalty, secondary-adjustment, and APA / MAP escalation flag
- Open questions and documentation gaps list
- DRAFT label and required reviewer sign-off line

## Inputs the agent will ask for

One question at a time:

1. Multinational group, tested entity, and fiscal year under review
2. Intercompany transaction type, counterparties, and direction (inbound / outbound)
3. Currency, magnitude, and intercompany agreement reference
4. Tax jurisdictions involved and any specific country-rule overlay (e.g. India safe harbour, U.S. §482, China bulletins, EU JTPF guidance, UAE FTA, OECD-only)
5. Functional / asset / risk profile of each party (or supporting interview notes)
6. Comparables-database used (or instruction to design search from scratch)
7. Whether an APA or MAP is in progress and the agreed parameters, if any
8. Audit / dispute posture (planning, return preparation, exam defence, MAP, APA negotiation)
