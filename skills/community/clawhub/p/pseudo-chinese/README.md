# 伪中国语趣味翻译（偽中国語）

把中文翻译成「伪中国语」——模拟日语去掉假名后只剩汉字的效果，让每句话看起来像日语又不是日语，中国人能猜懂、日本人也能猜懂。

## 这个技能能做什么

- **中文 → 伪中国语翻译**：输入任意中文句子，输出纯汉字日式表达
- **六大转换规则**：和式词汇替换 → SOV 语序重排 → 日式婉转/夸张风格 → 日本汉字字形 → 省略补充 → 标点规范
- **500+ 条中日词汇对照**：覆盖 18 个场景分类（日常、食物、交通、购物、职场、科技等）
- **40+ 条假朋友警示**：走≠走（跑）、老婆≠老婆（老太婆）、人参≠人参（胡萝卜）等同形异义词
- **七级夸张前缀体系**：中々 → 結構 → 大変 → 物凄 → 超 → 激 → 鬼

## 适用场景

- 中日跨文化社交、朋友圈/SNS 趣味内容创作
- 日语学习者的汉字词汇对照参考
- 中日双语 meme / 梗图文案生成
- 伪中国语挑战、语言游戏

## 翻译示例

```
🇨🇳 原文：对不起，我迟到了，路上堵车很严重。
🇯🇵 偽中国語：申訳無、遅刻致、道路渋滞大変酷。
💬 解読：申訳無(对不起)、遅刻(迟到) 致(了)、道路(路上) 渋滞(堵车) 大変(很) 酷(严重)。
```

```
🇨🇳 原文：这家店的拉面超好吃，下次一起去吧。
🇯🇵 偽中国語：此店拉麺超美味、次回一緒行！
💬 解読：此(这) 店 拉麺(拉面) 超(超级) 美味(好吃)、次回(下次) 一緒(一起) 行(去)！
```

## 输出格式

每句翻译统一输出三行：
- 🇨🇳 原文
- 🇯🇵 偽中国語（翻译结果）
- 💬 解読（逐词对照注释）

## 触发方式

对 Agent 说「伪中国语翻译」「偽中国語」「日式汉字翻译」或直接给出中文要求翻译即可。
