# 使用指南

## 快速开始

### 1. 直接使用
双击 `index.html` 文件，在浏览器中打开即可使用。

### 2. 本地服务器（推荐）
```bash
# Python 3
python -m http.server 8080

# Node.js
npx serve .

# 然后访问 http://localhost:8080
```

---

## 功能说明

### 模块一：AI爆款文案生成

1. **选择平台**
   - 抖音：生成中文爆款文案
   - TikTok日区：生成日文爆款文案
   - TikTok美区：生成英文爆款文案

2. **输入信息**
   - 产品类型：如"面膜"、"耳机"等
   - 产品卖点：用逗号分隔多个卖点

3. **生成文案**
   - 点击"生成爆款文案"按钮
   - 日区/美区会自动翻译中文

4. **配音功能**
   - 选择女声/男声
   - 点击播放按钮试听
   - 点击下载按钮保存音频

---

### 模块二：爆款文案复刻

1. **粘贴文案**
   - 将爆款文案粘贴到输入框
   - 系统自动识别语言

2. **生成新文案**
   - 点击"重新生成"
   - 系统生成同风格新文案

---

### 模块三：保存记录

1. **保存文案**
   - 点击"保存"按钮
   - 文案保存到本地

2. **生成标题标签**
   - 点击"一键生成标题和标签"
   - 自动生成适合的标题和标签

---

## TTS语音说明

### 免费服务
本应用使用 Edge TTS 免费服务，无需注册，无使用限制。

### 支持语音
- **中文**：晓晓、云希等 20+ 种语音
- **英文**：Jenny、Guy 等
- **日文**：七海、大地等
- **韩文**：선희、인준等

### 音频格式
- 格式：MP3
- 采样率：16kHz
- 比特率：128kbps

## 常见问题

### Q: 无法播放音频？
A: 请检查网络连接，确保能访问 tts.wangwangit.com

### Q: 翻译失败？
A: 翻译服务有频率限制，请稍后重试

### Q: 数据会丢失吗？
A: 数据保存在浏览器 localStorage，清除浏览器数据会丢失
