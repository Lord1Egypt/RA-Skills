## Description: <br>
Test Publish Check helps developers prepare software releases with pre-publish checklists for code quality, APIs, deployment, versioning, launch readiness, and regression testing. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[ckchzh](https://clawhub.ai/user/ckchzh) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, QA engineers, release managers, and DevOps teams use this skill to generate release readiness guidance before deploying software, websites, apps, APIs, or content to production. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The release checklist prompts are low risk, but an extra script stores user input in local files and may retain sensitive deployment details if provided. <br>
Mitigation: Avoid passing secrets, customer data, production identifiers, or sensitive deployment details to the skill, and review or remove its local data directory after running it. <br>
