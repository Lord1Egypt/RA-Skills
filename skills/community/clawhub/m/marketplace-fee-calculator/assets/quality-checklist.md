# Marketplace Fee Calculation Quality Checklist

## 1. Product Data Accuracy (6 items)
- [ ] Selling price confirmed (not aspirational)
- [ ] COGS from actual supplier invoice (not estimate)
- [ ] Product weight measured with packaging
- [ ] Product dimensions measured (longest side, median side, shortest side)
- [ ] Category confirmed on each marketplace
- [ ] Shipping weight includes packaging materials

## 2. Amazon Fee Completeness (8 items)
- [ ] Referral fee calculated at correct category rate
- [ ] FBA fee matched to correct size tier and weight band
- [ ] Monthly storage fee estimated based on cubic footage
- [ ] Long-term storage risk assessed (aged inventory surcharge)
- [ ] Professional subscription amortized over monthly volume
- [ ] Return processing fee estimated (category return rate × processing cost)
- [ ] Advertising cost per unit estimated (10-25% of revenue typical)
- [ ] FBA inbound shipping cost included

## 3. eBay Fee Completeness (5 items)
- [ ] Final value fee at correct category rate (+ $0.30 per order)
- [ ] Store subscription cost amortized per unit
- [ ] Promoted listings cost estimated (if using)
- [ ] Shipping cost included (if seller-paid)
- [ ] International fees checked (if selling globally)

## 4. Etsy Fee Completeness (5 items)
- [ ] Transaction fee (6.5%) calculated on price + buyer-paid shipping
- [ ] Payment processing (3% + $0.25) included
- [ ] Listing fee ($0.20) included per sale
- [ ] Offsite ads fee included (12-15% for eligible sellers)
- [ ] Etsy Plus subscription included (if applicable)

## 5. Walmart Fee Completeness (4 items)
- [ ] Referral fee at correct category rate
- [ ] WFS fee calculated (if using Walmart fulfillment)
- [ ] WFS storage fee included
- [ ] Comparison made between WFS and self-fulfillment

## 6. Shopify DTC Cost Completeness (6 items)
- [ ] Payment processing at correct plan rate
- [ ] Transaction fee included (if not using Shopify Payments)
- [ ] Subscription amortized per unit
- [ ] Shipping cost per order estimated
- [ ] Customer acquisition cost (CAC) included
- [ ] App subscription costs amortized per unit

## 7. Comparison Quality (6 items)
- [ ] All marketplaces use same selling price (or justified differences)
- [ ] All fee components included for each channel
- [ ] Net profit calculated per unit AND per month
- [ ] Net margin percentages comparable across channels
- [ ] Annual profit projections included
- [ ] Ranking by net margin AND by total profit (may differ)

## 8. Optimization Review (5 items)
- [ ] Category optimization checked (lower-fee category available?)
- [ ] FBA vs FBM comparison done for Amazon
- [ ] Store tier analyzed for eBay
- [ ] Volume thresholds identified for subscription upgrades
- [ ] Cross-channel pricing strategy documented

## 9. Decision Quality (5 items)
- [ ] Primary and secondary channels recommended
- [ ] Channels to avoid identified with clear rationale
- [ ] Break-even volume per channel calculated
- [ ] Review timeline set for fee re-evaluation
- [ ] Fee changes monitored (marketplace fee updates happen annually)
