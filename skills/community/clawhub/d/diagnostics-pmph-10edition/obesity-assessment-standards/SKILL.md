# 肥胖的测量标准与分级

## 何时使用
- 需评估成人个体是否存在超重或肥胖
- 进行流行病学调查或健康风险筛查
- 制定体重管理或代谢疾病干预方案
- **注意**：不适用于运动员、孕妇、严重水肿患者；儿童需使用年龄别BMI标准

## 执行步骤
1. **确认前提条件**：已获取准确的身高（cm）和体重（kg）数据，并排除肌肉发达或水肿干扰。
2. **计算BMI**：使用公式 BMI = 体重（kg） ÷ 身高²（m²）。
3. **选择适用标准进行分类**：
   - **中国标准**：
     - 正常：18.5 ~ 23.9 kg/m²
     - 超重：24.0 ~ 27.9 kg/m²
     - 肥胖：≥28.0 kg/m²
   - **WHO标准**：
     - 正常：18.5 ~ 24.9 kg/m²
     - 超重：25.0 ~ 29.9 kg/m²
     - 肥胖：≥30.0 kg/m²
4. **如采用WHO标准，进一步分级肥胖**：
   - 1级：30.0 ~ 34.9 kg/m²
   - 2级：35.0 ~ 39.9 kg/m²
   - 3级（重度）：≥40.0 kg/m²
5. **结合腰围判断腹型肥胖**：
   - 男性腰围 ≥90 cm 或女性腰围 ≥85 cm，提示腹型肥胖
6. **输出结果**：报告BMI值、对应分类（正常/超重/肥胖级别），并注明是否为腹型肥胖。

> 提示：若需皮褶厚度测量、简单体重估算或其他详细参考值，请查阅附带参考资料。
