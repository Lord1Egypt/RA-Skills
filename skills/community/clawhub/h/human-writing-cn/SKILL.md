# 中文防AI味写作助手

> **基于 taste-skill (52K⭐) 深度本地化改造**
> 针对中文表达习惯优化：去模板化、口语化、情绪化

## 触发条件

当用户请求涉及以下操作时自动激活：
- AI生成文字的润色/改写
- "帮我写一篇文章/文案"
- "这段话太AI味了"
- "让文字更自然"
- 内容质量评审

## AI味文字特征（中文版）

### 🔴 严重AI味

| 特征 | 示例 | 改造后 |
|------|------|--------|
| 过度礼貌 | "您好！很高兴为您介绍..." | "说说这个产品" |
| 三段式排比 | "无论是...还是...抑或是..." | "不管是谁都能用" |
| 空洞形容词 | "令人惊叹的、卓越的、无与伦比的" | "好用/牛逼/绝了" |
| 机械连接词 | "首先...其次...最后...综上所述" | "先说...再说...总之" |
| 官腔套话 | "在当今快速发展的时代背景下" | "现在这个时代" |
| 过度总结 | "总而言之，我们可以得出以下结论" | "说白了就是" |

### 🟡 中度AI味

| 特征 | 示例 | 改造后 |
|------|------|--------|
| 被动语态 | "该功能被广泛应用于..." | "大家都在用这个功能" |
| 抽象名词 | "提供了一种有效的解决方案" | "帮你解决问题" |
| 长句堆砌 | "通过XXX技术，实现了XXX功能，从而达到XXX效果" | 拆成短句 |
| 数据罗列 | "据统计，XX%的用户表示..." | "大部分人都说..." |
| 教科书体 | "综上所述，XXX具有以下特点" | "这东西有几个特点" |

### 🟢 人味文字特征

| 特征 | 示例 |
|------|------|
| 口语化 | "说实话"/"你懂的"/"反正" |
| 短句为主 | 每句不超过20字 |
| 有情绪 | "绝了"/"救命"/"太真实了" |
| 有观点 | 敢说"我觉得"/"不建议" |
| 有细节 | 具体数字/具体场景/具体人 |
| 不完美 | 偶尔用"可能"/"大概"/"我觉得" |
| 有态度 | 敢说"不好用"/"不推荐" |

## 改造规则

### 规则1：拆长句

```
❌ 通过使用先进的自然语言处理技术，我们能够为用户提供更加精准、高效、智能化的服务体验。
✅ 我们用了最新的技术，帮你更准、更快地解决问题。
```

### 规则2：去官腔

```
❌ 在当今数字化转型的大背景下，企业需要积极探索创新路径。
✅ 现在企业都在想办法数字化，怎么搞？
```

### 规则3：加人味

```
❌ 该产品具有多种功能，可以满足不同用户的需求。
✅ 这东西功能挺多的，不管干啥都够用。
```

### 规则4：具体化

```
❌ 大部分用户对产品表示满意。
✅ 10个人里有8个说好。
```

### 规则5：去排比

```
❌ 无论是学习、工作还是生活，它都能为你提供有力支持。
✅ 学习工作生活都能用。
```

### 规则6：加态度

```
❌ 该产品在市场上具有一定的竞争力。
✅ 这产品在同类型里算能打的。
```

## 评审打分标准

### AI味检测（0-100分）

| 分数 | 等级 | 说明 |
|------|------|------|
| 90-100 | 纯人味 | 看不出AI痕迹 |
| 75-89 | 基本人味 | 偶尔有AI感 |
| 60-74 | 中度AI味 | 需要改造 |
| 40-59 | 严重AI味 | 大改 |
| 0-39 | 纯AI味 | 重写 |

### 检测维度

1. **句式**（25分）：长短句比例，是否过于整齐
2. **用词**（25分）：是否过度使用AI高频词
3. **态度**（20分）：是否有个人观点和情绪
4. **细节**（15分）：是否具体、有场景感
5. **结构**（15分）：是否过度模板化（三段式等）
