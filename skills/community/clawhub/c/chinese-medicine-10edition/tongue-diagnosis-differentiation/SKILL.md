# Tongue Diagnosis Differentiation

## When to Use
当进行中医望诊，发现患者舌体形态（如老嫩、胖瘦、芒刺、裂纹、齿痕等）或舌苔（如厚薄、润燥、腐腻、颜色、分布等）出现异常时，应用本技能进行辨证。

## Execution Steps

### 第一步：排除干扰因素
- 排除食物、药物染苔等外部干扰。
- 确保光线充足，舌伸自然。

### 第二步：分别辨析舌体与舌苔

#### A. 舌体形态辨证
1. **老舌**（坚敛苍老、纹理粗糙）→ 实证或热证。
2. **嫩舌**（浮胖娇嫩、边有齿痕）→ 虚证或寒证。
3. **胖肿舌** → 脾虚湿蕴；结合舌色苔质细分：
   - 淡白胖嫩+白滑苔 → 脾肾阳虚；
   - 红绛胖大+黄厚腻苔 → 脾胃湿热；
   - 赤肿胀+黄苔 → 热毒壅盛；
   - 紫暗肿胀 → 中毒。
4. **瘦瘪舌** → 气血虚或阴虚；
   - 淡红而嫩 → 心脾两虚；
   - 绛干 → 阴虚火旺。
5. **芒刺舌** → 热盛；位置提示病位：
   - 舌边 → 肝胆火盛；
   - 舌中 → 胃肠热甚；
   - 舌尖 → 心火上炎。
6. **裂纹舌** → 阴血亏虚；
   - 红绛少苔燥裂 → 热盛伤阴；
   - 浅淡有裂 → 血虚。
7. **齿痕舌**（常伴胖舌）→ 脾虚水湿内停。
8. **舌疮** → 心经热毒或阴虚火炎。
9. **舌下络脉青紫粗大迂曲** → 血瘀。

#### B. 舌苔辨证
1. **苔质**：
   - 薄苔 → 邪在表，病轻；
   - 厚苔 → 邪入里，病重。
2. **润燥**：
   - 润苔 → 津液未伤；
   - 滑苔 → 湿盛或阳虚水泛；
   - 燥苔 → 津伤或热盛。
3. **腐腻**：
   - 腐苔 → 食积、痰浊；
   - 腻苔 → 湿浊、痰饮；
   - 黄腻 → 湿热/痰热；
   - 白滑腻 → 寒湿。
4. **苔色**：
   - 白苔 → 表证、寒证；
   - 黄苔 → 里证、热证；
   - 灰黑苔 → 里寒或里热重证（湿润为寒，干燥为热）。
5. **苔形**：
   - 全苔 → 痰湿阻滞；
   - 偏苔 → 肝胆病；
   - 花剥苔/镜面舌 → 胃气阴不足或枯竭。

### 第三步：综合判断
- 结合舌体与舌苔特征，输出病性（寒热虚实）、病位（表里）、津液状态、湿浊程度及预后。

## Precautions
- 必须与全身症状及其他三诊信息合参。
- 舌象动态变化（如苔由薄变厚）反映病势进退，应重视。
