# 安全指数定义与计算公式

## 机械指数（MI）
MI = 超声波最大负值峰压（MPa） / √(换能器中心频率（MHz）)

- 反映空化效应风险
- 无量纲

## 热指数（TI）
TI = 实际温升（℃） / 使组织温升1℃所需能量

- 反映热效应风险
- 无量纲

## 组织特异性阈值依据
- **胎儿**：发育中组织对热和机械应力高度敏感，国际指南（如AIUM、WFUMB）建议MI ≤ 0.3，TI ≤ 0.4
- **眼球**：晶状体和视网膜易受热损伤，推荐更严格限制：MI ≤ 0.1，TI ≤ 0.2

## 设备要求
- 必须支持实时MI/TI显示（符合IEC 60601-2-37标准）
- 输出功率可调，具备低MI/TI预设模式
