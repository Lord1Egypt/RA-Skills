# 🏢 企服助手 — 安装指引

> **版本**: 2.0（工作流重构版）  
> **适用平台**: WorkBuddy  
> **安装方式**: 专家包导入 / 腾讯文档模板  
> **数据安全**: 本指引不含任何本地业务数据，可安全分享

---

## 📦 安装包获取

### 方式A：从 WorkBuddy 市场安装（推荐）

在 WorkBuddy 中搜索 **「企服助手」** 或 **「Enterprise Service Assistant」**，一键安装即可。

### 方式B：手动安装

```bash
# 将企服助手专家包放置在 WorkBuddy 插件目录
cp -r enterprise-service-assistant/ ~/.workbuddy/plugins/marketplaces/my-experts/plugins/
# 重启 WorkBuddy 使专家生效
```

> 💡 如需获取专家包，请联系你的系统管理员或从内部仓库下载。

## 🚀 安装流程

### 第1步：安装专家包

在 WorkBuddy 中搜索并安装「企服助手」，安装后首次对话会自动弹出欢迎介绍。

### 第2步：准备数据源（三选一）

#### 🅰 本地文件（推荐个人使用）
```
1. 复制腾讯文档模板到你本地 Excel
2. 填入真实数据
3. 放到 workspace 的 knowledge-base/ 目录
4. 或直接发送 Excel 文件给 @企服助手，AI自动格式化
```

#### 🅱 腾讯文档（推荐团队协作）
```
1. 打开模板链接 → 复制到你的腾讯文档
2. 团队共同维护数据
3. 在知识库配置中填写文档ID
```

#### 🅲 IMA知识库（推荐长期积累）
```
1. 在 IMA 创建知识空间
2. 上传企业档案、走访记录等
3. 将知识库ID填入配置
```

### 第3步：对话验证

发送以下任意消息验证是否配置成功：

```
@企服助手 今天有什么安排
@企服助手 检查配置
@企服助手 查看库存状态
```

## 🔧 数据表字段说明

### 客户管理表（16字段）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 企业名称 | 文本 | 入驻企业全称 |
| 楼栋号 / 单元号 | 文本 | 物理位置 |
| 所属行业 | 单选 | 信息技术/制造业/金融服务/生物医药/文化创意/商贸服务/教育培训 |
| 企业等级 | 单选 | S级/A级/B级/C级/D级 |
| 当前风险标记 | 单选 | 正常/关注/预警/风险/高危/退租/流失 |
| 企业标签 | 多选 | 高增长/高新企业/税收大户/需扶持/需走访 |
| 其他字段 | - | 联系人、电话、面积、租金、合同到期日、入驻人数、走访日期等 |

### 费用收缴表（14字段）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 费用类型 | 单选 | 租金/物业费/水电费/空调费/停车费/违约金/其他 |
| 应收金额 / 实收金额 / 欠费金额 | 文本 | 金额数据 |
| 缴费状态 | 单选 | 已缴清/部分缴款/未缴款 |
| 其他字段 | - | 账单日期、缴费截止日、逾期天数、催缴次数等 |

### 增值服务记录（10字段）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 服务类别 | 单选 | 政务办理/人才服务/金融服务/知识产权/政策申报/法律咨询/数字化/社群孵化 |
| 服务状态 | 单选 | 待推荐/已推荐/有意向/已成交/已交付 |
| 商机阶段 | 单选 | 挖掘中/跟进中/谈判中/已转化/已流失 |
| 企业反馈 | 单选 | 非常满意/满意/一般/不满意 |
| 其他字段 | - | 服务项目、日期、金额、负责人员、跟进记录等 |

### 企业赋能服务产品清单（8大类）

| 类别 | 典型服务 |
|------|---------|
| 政务办理 | 工商注册、变更登记、税务申报、行政审批 |
| 人才服务 | 招聘发布、政策咨询、培训定制、职称辅导 |
| 金融服务 | 融资对接、银企对接、投融资咨询、担保贷款 |
| 知识产权 | 商标注册、专利申请、维权服务 |
| 政策申报 | 高新认定、专精特新、补贴申请、资质认定 |
| 法律咨询 | 法律顾问、合同审查、纠纷调解 |
| 数字化 | 企业上云、办公系统、网络安全 |
| 社群孵化 | 创业沙龙、路演对接、孵化加速、产业对接 |

## 🔒 数据安全

| 原则 | 说明 |
|------|------|
| **数据不外发** | 所有数据存储在本地或你授权的腾讯文档/IMA中，不外传 |
| **仅读取** | AI 仅读取你提供的数据源链接 |
| **模板不含数据** | 所有模板文件不包含真实业务数据，可安全分享 |
| **迁移安全** | 更换环境时只需复制数据源，无需迁移 Skill 包 |

> ⚠️ **重要**：分享专家包时，请确保不包含 `workspace/knowledge-base/` 目录中的真实数据。
