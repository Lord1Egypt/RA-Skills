# 骨盆骨折部位分类与临床特征

## 分类与特征

### 1. 骨盆边缘撕脱骨折
- **机制**：肌肉猛烈收缩（多见于青少年）
- **类型**：
  - 髂前上棘（缝匠肌）
  - 髂前下棘（股直肌）
  - 坐骨结节（腘绳肌）
- **特点**：骨盆环完整、稳定

### 2. 髂骨翼骨折
- **机制**：侧方挤压暴力
- **特点**：可为粉碎性，但不影响骨盆环稳定性

### 3. 骶尾骨骨折
- **骶骨骨折（Dennis分型）**：
  - Ⅰ区（骶骨翼）：较少神经损伤
  - Ⅱ区（骶孔处）：可能损伤腰骶神经根
  - Ⅲ区（骶管区）：易损伤马尾神经
- **尾骨骨折**：跌倒坐地所致，常伴骶骨末端骨折，移位轻微

### 4. 骨盆环骨折
- **机制**：高能量暴力（交通伤、坠落）
- **特点**：多为双处骨折（如双侧耻骨支骨折、耻骨联合分离+骶髂关节脱位），骨盆变形，常伴严重并发症

## 应用要点
- 结合致伤机制与影像学定位骨折类型；
- 评估是否影响骨盆环稳定性；
- 对骶骨Ⅱ/Ⅲ区骨折，警惕神经损伤。
