## 三、六经辨证速查

### 太阳病（表证、阳证）
- **特征**：脉浮、头项强痛、恶寒
- **分型**：
  - 太阳中风（表虚）：汗出恶风，脉浮缓 → 桂枝汤
  - 太阳伤寒（表实）：无汗恶寒，脉浮紧 → 麻黄汤
- **变证**：蓄水（五苓散）、蓄血（桃核承气汤）

### 阳明病（里热实证）
- **特征**：大热、大渴、大汗、脉洪大
- **经证**（无形热邪）：四大证 → 白虎汤
- **腑证**（有形燥屎）：腹胀满、拒按、便秘 → 承气汤类
- **兼证**：湿热发黄 → 茵陈蒿汤

### 少阳病（半表半里）
- **特征**：口苦、咽干、目眩、胸胁苦满、往来寒热
- **主方**：小柴胡汤
- **兼证**：
  - 少阳阳明合病 → 大柴胡汤
  - 少阳兼水饮 → 柴胡桂枝干姜汤

### 太阴病（里虚寒证）
- **特征**：腹满而吐、食不下、自利益甚、时腹自痛
- **主方**：理中汤/四逆辈
- **重点**：脾阳虚，运化无权

### 少阴病（心肾虚衰）
- **特征**：脉微细、但欲寐（精神萎靡）
- **寒化证**：畏寒、手足逆冷、下利清谷 → 四逆汤
- **热化证**：心烦不眠、口干咽燥 → 黄连阿胶汤
- **兼证**：阳虚水泛 → 真武汤

### 厥阴病（寒热错杂）
- **特征**：消渴、气上撞心、心中疼热、饥不欲食
- **寒热错杂**：乌梅丸
- **血虚寒厥**：手足厥寒、脉细 → 当归四逆汤
- **厥热胜复**：判断疾病进退

## 四、脏腑辨证要诀

### 肝（疏泄之本）
- **肝气郁结**：胁痛、善太息、情绪抑郁、脉弦 → 四逆散/柴胡疏肝散
- **肝火上炎**：头痛目赤、口苦、易怒、脉弦数 → 龙胆泻肝汤
- **肝血不足**：目涩、爪甲枯、经少、脉细 → 四物汤
- **肝阳上亢**：头痛眩晕、面红目赤 → 天麻钩藤饮
- **倪师重要观点**：肝主疏泄，经期靠肝气疏泄，桂枝升肝是核心药

### 心（君主之官）
- **心阳虚**：心悸、胸闷、畏寒、脉结代 → 炙甘草汤
- **心阴虚**：心烦失眠、盗汗、舌红少苔 → 黄连阿胶汤 / 天王补心丹
- **心火上炎**：口舌生疮、心烦、小便赤 → 导赤散
- **心肾不交**：心烦失眠、腰膝酸软 → 交泰丸/黄连阿胶汤
- **倪师核心观点**：心与小肠相表里，心火下行温小肠，小肠热是全身热源

### 脾（后天之本）
- **脾气虚**：食少便溏、乏力、面色萎黄 → 四君子汤
- **脾阳虚**：腹冷痛、喜温喜按、下利清谷 → 理中汤
- **脾虚湿困**：脘腹胀满、苔腻、身重 → 平胃散/苓桂术甘汤
- **中气下陷**：脘腹坠胀、脱肛、子宫下垂 → 补中益气汤
- **倪师观点**：脾主运化水湿，"脾为生痰之源，肺为贮痰之器"

### 肺（华盖之脏）
- **风寒束肺**：恶寒发热、咳嗽、痰白稀 → 麻黄汤/小青龙汤
- **风热犯肺**：发热口渴、咳嗽、痰黄 → 麻杏甘石汤
- **痰湿阻肺**：咳嗽痰多、胸闷 → 二陈汤
- **肺阴虚**：干咳少痰、咽干、潮热 → 百合固金汤
- **倪师观点**：肺主皮毛，皮肤病多从肺治；肺与大肠相表里

### 肾（先天之本）
- **肾阳虚**：畏寒肢冷、腰膝酸软、阳痿、五更泄 → 金匮肾气丸/右归丸
- **肾阴虚**：腰膝酸软、头晕耳鸣、潮热盗汗 → 六味地黄丸/左归丸
- **肾精不足**：发脱齿摇、健忘、不育 → 五子衍宗丸
- **肾不纳气**：喘息、呼多吸少 → 肾气丸加五味子
- **倪师观点**：肾藏精，精生髓，髓通脑。肾为作强之官。

## 五、关键诊疗概念

### 1. 倪海厦的"经血生成说"（核心理论）
- 乳汁（心精所化）→下行至精宫→受小肠之热→化为经血
- 因此心与小肠的温度对女子月经至关重要
- 乳癌、子宫肌瘤皆与乳汁不下行有关

### 2. 水火既济
- 心火（离）下温肾水
- 肾水（坎）上济心火
- 心肾不交 = 失眠、烦躁、腰膝酸软

### 3. 桂枝升肝理论
- 桂枝是升肝之药（不是温病派说的"血证禁用"）
- 肝主疏泄 → 桂枝助肝疏泄 → 月经通畅、瘀血化解
- 桂枝茯苓丸用桂枝就是此理

### 4. 扶阳思想
- "阳气者，若天与日，失其所则折寿而不彰"
- 治病首重阳气
- 忌寒凉药过度使用
- 生冷食物是大忌

### 5. 血与水的关系
- "血不利则为水"
- 许多水肿病根在血分
- 活血即利水

### 6. 倪海厦特殊诊断法
- **眼诊**：白睛斑块（瘀血）、瞳孔反应（肾气）
- **舌诊**：特别注意舌下静脉（瘀血指征）
- **腹诊**：按之痛否（虚实之分）
- **脉诊**：注重六部脉的对比
