# Map Domain Experts

快速梳理一个新领域的代表专家、核心共识、关键分歧、对照表和入门学习路线。

This Codex/OpenAI skill turns "I want to quickly learn X" into a compact expert map: who matters, what they agree on, where they disagree, and how to start learning.

## Use

```text
我想快速学习【领域名称】。请找出代表专家、核心共识、重要分歧、对照表和入门学习路线。
```

## Output

The skill produces:

- 范围说明
- 代表人物
- 核心共识
- 关键分歧
- 对照表
- 入门学习路线
- 资料来源

## Files

- `SKILL.md`: skill instructions and workflow
- `agents/openai.yaml`: OpenAI agent display metadata and default prompt
