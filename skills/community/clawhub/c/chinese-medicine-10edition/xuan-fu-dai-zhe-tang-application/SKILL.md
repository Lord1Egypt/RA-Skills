# Xuan Fu Dai Zhe Tang Application

## When to Use
确认患者符合以下全部特征：
- 主症：胃脘痞闷或胀满、频频嗳气、呃逆、恶心、呕吐
- 舌脉：舌苔白腻，脉缓或滑
- 病机：胃气虚弱 + 痰浊内阻 + 胃气上逆

## Execution Steps
1. **确认禁忌**：排除实热证、阴虚火旺证及非痰阻所致的呕吐呃逆。
2. **开具方剂**：按经典配伍组成处方：
   - 旋覆花三两（君药）
   - 代赭石一两（臣药）
   - 生姜五两、半夏（洗）半升、人参二两、炙甘草三两、大枣（擘）十二枚（佐使药）
3. **说明煎服法**：以水煎煮，温服。
4. **解释配伍逻辑**：强调本方“降逆化痰、益气和胃”的核心功效，突出旋覆花降气、代赭石镇逆、生姜三重作用（止呕、助祛痰、制寒）、参草枣补中等协同机制。
5. **指导现代应用**：可延伸用于慢性胃炎、胃神经症、幽门不完全性梗阻、神经性呃逆等属胃虚痰阻者。

> 注意：代赭石用量宜轻（仅一两），以防苦寒伤胃；全方重在调和虚实、标本兼顾。
