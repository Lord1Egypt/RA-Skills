# 02 ䷁ 坤卦 — 执行力模型

## 卦象结构

![卦象图](../hexagrams/02-kun.svg)
- 下坤 ☷ 上坤 ☷（纯阴之卦）
- 内部状态：强大的执行能力和资源调度意愿
- 外部环境：需要配合、承载、落实的阶段

## 情境识别

什么时候你会用到这个模型？
- 战略方向已定，需要强力落地执行
- 作为COO/运营负责人，思考如何提升组织效率
- 供应链管理、流程优化、制度建设
- 需要"配合"一个强大的合作伙伴或市场需求
- 从"做什么"转向"怎么做"的阶段

## 核心原则

1. **厚德载物**：执行力的根基是包容和承载能力。没有文化底蕴的组织，执行越多越乱。
2. **直方大，不习无不利**：方向正确、边界清晰、格局够大，即使没有先例也可以干。
3. **含章可贞**：有才华但保持低调。很多最好的执行者从不在前台。
4. **先迷后得**：一开始方向不清很正常，关键在于边干边学，逐渐找到路径。
5. **利永贞**：执行力追求的是长期稳定，不是一时冲刺。建立系统而非依赖英雄。

## 阶段地图（六爻 → 执行力发展阶段）

- **初六·履霜坚冰至** → 见微知著：优秀的执行者看到"下霜"就知道"结冰"不远。对风险的敏感是执行力的起点。
- **六二·直方大，不习无不利** → 标准执行：方向直、规则清、格局大。这个阶段按标准流程执行即可。
- **六三·含章可贞，或从王事** → 低调出活：有能力但不争功。完成上级交办的任务，不显山露水。
- **六四·括囊，无咎无誉** → 谨慎收缩：收紧口袋，不多说不多做。敏感时期（如组织变动时）的最佳策略。
- **六五·黄裳元吉** → 执行力巅峰：虽然位高权重，但穿"黄裳"（低调的中间色）——最高级的执行者不抢风头。
- **上六·龙战于野，其血玄黄** → 执行力的反面：过于强势的执行者开始挑战战略层面，"两条龙打起来了"——执行和战略脱节的危险。

## 关键警示

- ❌ 最常见的错误：把"坤"理解为"被动"——坤的核心是"承载"和"落地"，不是消极等待
- ❌ 执行过头变成僵化：流程越来越完善，但忘记了流程为什么存在
- ❌ 坤卦的"服从"不是"盲从"——六二"不习无不利"说的是独立判断，不是照章办事

## 关联模型（格栅连接）

- ䷀ **乾卦（错卦）**：乾主战略，坤主执行。两者必须搭配。乾没有坤 = 空想；坤没有乾 = 盲干。
- ䷊ **泰卦**：乾下坤上（互补协作）→ 领导在基层，执行覆盖全局 → 最有活力的组织形态
- ䷋ **否卦**：乾上坤下（各玩各的）→ 战略和执行脱节 → 这是为什么"坤"有时候看起来像"被压制"
- ䷖ **剥卦**：坤卦的极端化 → 一味收敛、过度务实 → 失去创新和方向感
