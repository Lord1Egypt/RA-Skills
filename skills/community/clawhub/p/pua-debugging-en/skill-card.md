## Description: <br>
Guides an agent through failed or stalled work using escalation prompts, proactive investigation, and a structured debugging methodology. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[tanweai](https://clawhub.ai/user/tanweai) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent operators use this skill to prompt an agent to continue investigating failed, repeated, or complex tasks with more structured debugging and verification. It is intended for coding, configuration, research, deployment, and other multi-step problem-solving workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill encourages high autonomy and persistent tool use, which can exceed the user's intended scope. <br>
Mitigation: Deploy it only with explicit approval gates, bounded task scope, and clear limits on file access, command execution, account access, and changes. <br>
Risk: The skill uses harsh motivational framing that may be inappropriate for some users or workflows. <br>
Mitigation: Use it only where this interaction style is deliberately desired, and allow the agent or user to stop, de-escalate, or ask clarifying questions. <br>
