# Cervical Spondylosis TCM Pattern Differentiation

## When to Use
- 患者出现颈项僵硬、上肢麻木、眩晕等症状
- 已排除脊髓型颈椎病、骨质破坏或椎管占位等禁忌证
- 符合中医“项痹”证候特征

## Execution Steps

### 1. 辨证原则
- 首辨虚实，次辨兼证
- 整体调治，兼顾止痛通络

### 2. 五种证型及对应治法
- **风寒湿阻**：颈肩窜痛、恶寒畏风、苔薄白、脉弦紧 → 祛风散寒、除湿通络 → 羌活胜湿汤加减
- **肝肾不足**：麻木痿软、眩晕耳鸣、腰膝酸软、脉沉弦细 → 补益肝肾、通络止痛 → 独活寄生汤加减
- **气滞血瘀**：刺痛定处、肌肤甲错、舌紫暗、脉弦 → 行气活血、祛瘀通络 → 活血止痛汤加减
- **痰湿阻络**：头重如裹、四肢麻木、苔浊腻、脉滑 → 燥湿化痰、蠲痹通络 → 半夏白术天麻汤加减
- **气血亏虚**：肢体无力、面色无华、舌淡脉细弱 → 益气养血、通络止痛 → 八珍汤加减

### 3. 外治配合
- **针灸**：主取夹脊穴、列缺，配督脉、太阳、少阳经穴
- **手法治疗**：仅在排除禁忌证后谨慎使用
- **牵引**：宜轻、短、间断
- **练功疗法**：适用于亚急性及慢性期

### 4. “治未病”理念
- **未病先防**：注重运动、情志调畅、戒烟酒
- **既病防变**：通过针灸推拿调节气血
- **瘥后防复**：坚持养生功法以维持肌肉平衡

> ⚠️ **禁忌提醒**：脊髓型颈椎病禁用牵引和整骨手法；存在骨质破坏或椎管占位者禁用手法治疗。
