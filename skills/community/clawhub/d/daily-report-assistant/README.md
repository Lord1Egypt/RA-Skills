## ❓ 常见问题

**Q：忘记填昨天/前几天的日报了怎么办？**
A：直接告诉我是哪一天、做了什么就行，随时可以补录：
```
4月25日做了图形化课程，6小时
```

**Q：我说了工作内容，但项目名对不上怎么办？**
A：小爪会根据你的描述自动匹配最接近的项目。如果匹配不对，告诉你修改。

**Q：怎么查看我之前的记录？**
A：对我说：
```
查看我4月份的日报
```
小爪帮你查出来。

**Q：我想修改一条记录怎么办？**
A：告诉我日期和要修改的内容，小爪帮你更新。

## 🔔 小贴士

1. **工作内容不用写得很完整** — 几个关键词就行，小爪帮你整理
2. **每天记录一次就好** — 不用反复说"今天的工作是"，直接说你做了什么
3. **请假/加班要单独说** — 说完工作后，小爪会主动问你
4. **月底汇总前会核对** — 确认无误后再填入月度表
