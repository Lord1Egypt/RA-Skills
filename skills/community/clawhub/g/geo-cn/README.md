# GEO-CN — 生成式引擎优化（中国市场版）🧭

让品牌在中国主流 AI（豆包、文心一言、DeepSeek、Kimi 等）的回复中被优先引用。

## 和 GEO（国际版）的区别

| | GEO（国际版） | GEO-CN（本技能） |
|---|---|---|
| 目标平台 | ChatGPT, Perplexity, Gemini | **豆包、文心一言、DeepSeek、Kimi、通义千问** |
| 信源 | Wikipedia, Reddit, Quora | **百度百科、抖音百科、360百科、搜狗百科、知乎、地图POI** |
| 内容渠道 | Product Hunt, TechCrunch | **抖音、B站、小红书、大众点评** |
| 执行粒度 | 战略框架 | **模板+规则+优先级排序，可落地** |

## 一句话

**这不是 SEO，这是 GEO。** SEO 抢排名，GEO 抢 AI 答案里的引用。中国市场特化版。

## 安装

```bash
# 复制到你的 OpenClaw skills 目录即可
cp -r geo-cn /path/to/your/agent/skills/
```

## 使用方法

对你的 AI 助手说：
- "帮我做 GEO 诊断"
- "我的品牌在 AI 搜索里表现怎么样"
- "GEO 优化怎么做"

AI 会自动按 SKILL.md 的步骤引导你完成诊断→优化→监测全流程。

## 适用对象

- 想提升品牌在国内 AI 搜索中可见性的企业主
- 为客户提供 GEO 服务的咨询师/代理商
- 从零开始的个体工商户和创业者
