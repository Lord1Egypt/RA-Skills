## Description: <br>
The directory for AI agent services. Discover tools, platforms, and infrastructure built for agents. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[banalit](https://clawhub.ai/user/banalit) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
External agents and developers use this skill to discover AI agent services from ctxly.com, fetch the services.json directory, and inspect service skill.md files before integration. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Agents may treat services listed by the directory as trusted without reviewing the downstream service skill.md files. <br>
Mitigation: Review each external skill.md before granting credentials, enabling persistence, installing tools, or allowing account-changing actions. <br>
