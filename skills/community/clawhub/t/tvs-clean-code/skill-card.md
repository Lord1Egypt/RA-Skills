## Description: <br>
Cleans and organizes code while adding comments that clarify intent and reduce the cognitive cost of reading and maintaining it. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[inksnowhailong](https://clawhub.ai/user/inksnowhailong) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers use this skill to guide code cleanup, simplify redundant structure, improve naming, organize code order, and add targeted comments that explain important business logic and edge cases. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Cleanup guidance can change code structure or behavior if applied too broadly. <br>
Mitigation: Apply changes to specified files or modules, then review and test the resulting code before relying on it. <br>
Risk: Added comments can be misleading if the agent misunderstands the original business intent. <br>
Mitigation: Check new comments against the code behavior and remove or revise any explanation that is not directly supported by the implementation. <br>
