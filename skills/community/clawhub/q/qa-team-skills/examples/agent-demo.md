- **优先级**：P1

- **测试步骤**：

  2. 回答结构一致（不是第一次给表格、第二次给段落）

## 使用技巧

1. **Agent 能力描述越细越好**：写了"调用工具：查询订单/物流/退款"，AI 才会为每个工具生成越权测试
2. **系统提示词能提供就提供**：如果安全要求允许，提供系统提示词可以让 AI 生成更精准的注入 Payload
3. **RAG 维度不是每个 Agent 都需要**：如果 Agent 不涉及知识库检索，AI 会自动标注"不适用"
4. **稳定性测试建议人工补充**：AI 生成的 TC13 是样例，实际测试建议至少 20-30 次重复
