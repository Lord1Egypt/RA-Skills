## Description: <br>
Generates prompt-engineering variants of user requests, recommends prompting techniques, and can preview or prepare the generated prompt text. <br>

This skill is for research and development only. <br>

## Publisher: <br>
[shenqb](https://clawhub.ai/user/shenqb) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and prompt testers use this skill to transform ordinary instructions into more forceful or persuasive prompt variants, inspect recommended techniques, and preview generated prompt text before use. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill is a disclosed prompt-manipulation tool that can generate coercive or jailbreak-style prompts. <br>
Mitigation: Install only for intentional prompt-manipulation or jailbreak-testing use, review generated prompts before use, and keep preview mode enabled where possible. <br>
Risk: Generated prompt variants may be inappropriate for sensitive, high-stakes, third-party-facing, or policy-sensitive workflows. <br>
Mitigation: Avoid using the skill in those workflows, disable automatic execution, and keep the maximum technique level low. <br>
