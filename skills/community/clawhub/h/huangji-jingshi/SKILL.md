# 皇极经世智慧

> "以物观物，无我而后大"
> —— 邵雍

《皇极经世》是北宋学者邵雍（邵康节）的伟大著作，融合《周易》象数之学与历史哲学，提出了著名的"元会运世"周期理论。

## 一、核心概念

### 元会运世时间体系

| 单位 | 量级 | 量化关系 |
|------|------|----------|
| 1元 | 129,600年 | 人类文明一个大周期 |
| 1会 | 10,800年 | 12会为一元 |
| 1运 | 360年 | 30运为一会 |
| 1世 | 30年 | 12世为一运 |

### 十二会六十四卦分配

| 会序 | 地支 | 主卦 | 历史阶段 | 时间范围（约） |
|------|------|--------|----------|---------------|
| 第1会 | 子会 | 复卦 | 开天 | 前64560-前53760年 |
| 第2会 | 丑会 | 临卦 | 辟地 | 前53760-前42960年 |
| 第3会 | 寅会 | 泰卦 | 人生 | 前42960-前32160年 |
| 第4会 | 卯会 | 大壮卦 | 生长 | 前32160-前21360年 |
| 第5会 | 辰会 | 夬卦 | 壮大 | 前21360-前10560年 |
| 第6会 | 巳会 | 乾卦 | 鼎盛前 | 前10560-前240年 |
| **第7会** | **午会** | **姤卦** | **鼎盛（当前）** | **前240年-10740年** |
| 第8会 | 未会 | 遁卦 | 衰退 | 10740-21540年 |
| 第9会 | 申会 | 否卦 | 困境 | 21540-32340年 |
| 第10会 | 酉会 | 观卦 | 观察 | 32340-43140年 |
| 第11会 | 戌会 | 剥卦 | 剥落 | 43140-53940年 |
| 第12会 | 亥会 | 坤卦 | 归藏 | 53940-129600年 |

## 二、当前时代定位

```
元：第1元
会：第7会（午会，姤卦主事）——文明鼎盛期
运：第192运（随卦主事）——2023年尚在此运
世：第10世（2013-2042，观卦主事）——观察待变期
```

## 三、应用方法

### 历史大势分析
1. 确定当前所处会、运、世
2. 分析主卦的表象与爻义
3. 推演相关卦的影响
4. 预测后续发展趋势

### 个人决策参考
- 大势顺势而动，不逆时而行
- 在姤运相交时期，注意阴阳转化
- 观卦时期重在观察收集信息

## 五、六十四卦推演历史详解

邵雍将六十四卦按先天方圆图顺序分配给十二会，每会对应特定的历史阶段。

### 当前所在：第七会（午会）详解

**主卦：姤卦（天风姤，䷫）**

| 维度 | 内容 |
|------|------|
| **卦象** | 天在上，风在下，天下有风 |
| **卦义** | 相遇、邂逅、阴气初生 |
| **象辞** | "风行天下，姤；后以施命诰四方" |
| **核心** | 文明鼎盛，但阳极阴生，盛极而衰的种子已埋下 |

### 午会历史分期（第186-192运）

| 运序 | 时间范围 | 主卦 | 历史事件 |
|------|---------|------|---------|
| 第186运 | 公元前417-前57年 | 泰卦 | 秦汉盛世 |
| 第187运 | 公元前57-303年 | 否卦 | 汉末三国 |
| 第188运 | 公元303-663年 | 同人卦 | 隋唐 |
| 第189运 | 公元663-1023年 | 大有卦 | 唐宋 |
| 第190运 | 公元1023-1383年 | 谦卦 | 宋元明 |
| 第191运 | 公元1383-1743年 | 豫卦 | 明清 |
| **第192运** | **公元1743-2103年** | **随卦** | **近现代（当前）** |

### 第192运（随卦）三十世分布

| 世序 | 时间范围 | 卦象 | 历史特征 |
|------|---------|------|---------|
| 第1世 | 1743-1772 | 复卦 | 阳气初生 |
| 第2世 | 1773-1802 | 临卦 | 阳气渐长 |
| 第3世 | 1803-1832 | 泰卦 | 天地交泰 |
| 第4世 | 1833-1862 | 大壮卦 | 阳气大盛 |
| 第5世 | 1863-1892 | 夬卦 | 阴气消退 |
| 第6世 | 1893-1922 | 乾卦 | 阳气极盛 |
| **第7世** | **1923-1952** | **姤卦** | **阴气初生** |
| 第8世 | 1953-1982 | 遁卦 | 阴气渐长 |
| **第9世** | **1983-2012** | **否卦** | **天地不交** |
| **第10世** | **2013-2042** | **观卦** | **观察待变（当前）** |
| 第11世 | 2043-2072 | 剥卦 | 阴气剥阳 |
| 第12世 | 2073-2103 | 坤卦 | 阴气极盛 |

## 六、核心智慧启示

> **"君子见几而作，不俟终日"**——洞察先兆，及时行动

| 启示 | 应用 |
|------|------|
| 历史有周期 | 文明兴衰有规律，顺势而为 |
| 当前定位 | 我们处于鼎盛期，但盛极而衰的种子已埋 |
| 观卦十年 | 2013-2042是观察期，需审慎应对变局 |
| 剥卦将至 | 2043-2072压力增大，需提前准备 |
| 随运而动 | 识时务者为俊杰，随运而动 |
