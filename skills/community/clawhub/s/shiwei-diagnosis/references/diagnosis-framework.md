# 施炜诊断框架详细参考

## 一、财务表征分析框架

财务分析不是目的，而是问题表征的起点。从财务数据中读取企业经营"体征"。

### 必看指标

1. **营收增长率**：3-5年趋势，判断是否遇到增长瓶颈
2. **净利润率**：趋势走低→运营效率问题或竞争加剧
3. **毛利率**：保持高位但净利率走低→内部运营成本过高
4. **存货周转率**：持续下降→动销势能减弱
5. **研发费用率**：与同行对比，过高但无产出→研发管理问题；过低→未来竞争力存疑
6. **销售费用率**：与同行对比，过高→营销效率问题
7. **市值/P/E**：资本市场是否看好成长性
8. **收入集中度**：单一品种占比过高→集中度风险
9. **人均指标**：人均营收、人均利润→人效水平

### 财务分析输出格式

```
通过对公司财务状况的分析，可以得出以下基本结论：
① 结论一（带数据支撑）
② 结论二（带数据支撑）
...
```

## 二、六大诊断维度详细框架

> 这六大维度覆盖诊断报告的全部诊断域。其中"组织能力"维度有一个深钻子框架——巴奴案例中使用的"组织能力六维模型"（凝聚力/协同力、战斗力、生命力、领导），详见维度2末尾。

### 维度1：战略与运营

#### 战略管理闭环诊断清单

```
□ 是否有清晰的战略规划文本？
□ 战略规划是否被解码为年度计划？
□ 年度计划是否与预算挂钩？
□ 各部门目标是否互锁（互相承接）？
□ 是否有定期战略复盘会议？
□ 战略管理是否有明确的组织载体（战略部/PMO）？
□ 战略部的职能是研究型还是管理型？
□ 投资项目的决策-执行-退出是否有闭环？
□ 核心资源（资金、人才）是否聚焦到战略方向？
□ 战略规划是否根据外部环境变化动态调整？
```

#### 运营效率诊断清单

```
□ 研发→注册→生产→营销的价值链是否通畅？
□ 是否存在部门墙导致的脱节（如包装变更未通知市场部）？
□ 是否有定期的经营分析会？
□ 经营分析会是否有PDCA闭环？
□ 目标责任体系是否有效（目标自上而下or自下而上）？
□ 是否存在"流程长、节点多、反应慢"的问题？
□ 是否有项目管理机制覆盖重大项目？
□ 横向协调是依赖流程还是依赖人（点对点沟通）？
□ 是否有统一的信息共享平台？
□ 经营数据是否透明可获取？
```

---

### 维度2：组织能力

#### 组织架构诊断清单

```
□ 组织架构是否清晰反映业务逻辑？
□ 是否有明确的组织设计规划（非临时拼凑）？
□ 职能划分是否过散过细？
□ 总部职能部门是否发育不足？
□ 是否有强有力的管理中枢（经管会/总裁办）？
□ 分子公司管控模式是否清晰？
□ 是否存在"总部职能+下属机构职能"叠合？
□ 部门职责边界是否清晰？
□ 是否存在"距离总部越近评分越低"的现象？
```

#### 组织能力诊断清单

```
□ 战略规划与组织能力是否匹配？
□ 管理能力和专业能力是否达到战略要求？
□ 总部职能部门是"管控思维"还是"赋能思维"？
□ 研发是否形成有效的管理体系？
□ 质量管理体系是否与国际标准存在差距？
□ 是否有统一的研发规划？
□ 研发力量是否分散（各技术平台各自为政）？
□ 是否有能力平台（如采购中心、品质中心、IT中心）？
```

#### 组织能力深钻：巴奴"组织能力六维模型"

当需要对"组织能力"维度做更深层诊断时，使用巴奴案例中的六维子框架：

| 子维度 | 核心要素 | 诊断要点 |
|--------|---------|---------|
| **凝聚力/协同力** | 约定（共识+信任）、连接（流程/权力/交易） | 跨部门/跨层级是否协同？共识和信任是否到位？ |
| **战斗力** | 技能（素质/开发）、动力（评价/激励） | 团队专业能力是否达标？激励机制是否有效？ |
| **生命力** | 创新（机制/氛围）、负熵（复杂性/僵化/腐败） | 有无创新机制和创新氛围？组织是否在熵增——流程越长、人越多、效率越低？ |
| **领导** | 治理结构、领导体制、决策机制、领导方式、领导心智 | 领导体系是否支撑组织能力？ |

> ⚠️ 注意：此处的"六维"特指"组织能力"维度的细化，与上面覆盖全部诊断域的"六大诊断维度"是不同层级的概念。巴奴报告中此模型用于深入剖析组织端问题，法兰泰克/沃森案例中未独立使用此命名但逻辑一致。如访谈暴露出组织凝聚力和战斗力问题，可选择性启用此深钻框架。

### 维度3：人才

#### 人才密度与梯队

```
□ 人才密度是否满足战略需求？
□ 是否有地域因素导致的人才吸引困难？
□ 专家人才是否存在流失？
□ 是否有后备干部管理体系？
□ 核心岗位是否有继任计划？
□ "二级企业家"（事业部负责人级别）是否充足？
□ 干部年龄结构是否老化？
□ 干部自培比例vs外部引入比例？
□ 是否存在"逆淘汰"（优秀人才流失，平庸者留下）？
```

#### 激励与考核

```
□ 绩效考核指标是否科学（定性vs定量）？
□ 考核与战略目标是否关联？
□ 考核结果是否真正应用（能上能下、能左能右）？
□ 是否存在"大锅饭"式的均分？
□ 股权激励是否达到预期效果？
□ 薪酬是否存在倒挂？
□ 技术/管理双通道是否真正走通？
□ "奖励不到位、处罚也不到位"是否存在？
```

### 维度4：文化

#### 文化诊断清单

```
□ 创业精神是否在流失？
□ 是否出现官僚化倾向（不敢说话、形式主义）？
□ 是否存在"自满、封闭"现象？
□ 信任文化是否充分（高管之间、上下级之间）？
□ 是否存在文化"散装"（不同单元文化差异大）？
□ 管理风格是否"宽松有余、压力不足"？
□ 是否有"骄兵悍将"状态？
□ 文化是否具备吸引外部优秀人才的能力？
□ 是否有"科研院所+民营企业"双重文化属性？
```

#### 文化归因方法

成功要素必须追溯文化根源。模式：
```
关键成功要素 → 背后的文化品格
  坚守航道    → 使命情怀
  前瞻布局    → 高瞻远瞩
  研发先行    → 敢于创新
  产业转化    → 重视能力
  资源整合    → 开放包容
```

### 维度5：领导与决策

#### 领导体制诊断清单

```
□ 是否有真正意义上的高管团队？
□ 高管团队成员是否能力互补？
□ 是否有规范的高管沟通/议事机制？
□ 重大决策是否有民主讨论程序？
□ 是否存在"几个人就决定了"的现象？
□ 决策是否有质疑/反对机制？
□ 事务是否过度集中在企业家个人身上？
□ 授权是否充分？
□ 是否存在"该管的没管、不该管的管太多"？
□ 决策失败后是否有复盘机制？
```

#### 企业家分析模型（巴奴案例深度框架）

当需要正面分析企业家/创始人时（"他做对了什么"，而不仅是"哪里有问题"），启用以下四个模型：

**A. "四力"素质模型**：
| 素质 | 定义 | 诊断信号 |
|------|------|---------|
| 认知力 | 对行业/市场/技术的理解深度和前瞻性 | 战略方向是否正确？是否预判了行业拐点？ |
| 驱动力 | 自我驱动的强度和方向（成就动机/权力动机/亲和动机） | 扩张冲动 vs 稳健经营？是否有"不甘心"驱动？ |
| 人际力 | 理解人、影响人、凝聚人的能力 | 是否能吸引人才？高管团队是否死心塌地跟随？ |
| 实践力 | 将想法变为行动、取得结果的能力 | 执行力是否强？是否"想到就能做到"？ |

**B. "四有"素质模型**：
- 有追求：有超越利润之上的事业理想
- 有方向：对企业未来有清晰的判断和规划
- 有力量：有推动变革的意志和资源
- 有胸怀：能容纳比自己强的人，能分享权力和利益

**C. 领导行为图谱**（六维连续谱）：
```
目标导向 ←——————→ 关系导向
勇于冒险 ←——————→ 谨慎稳健
信任授权 ←——————→ 控制集权
鼓励创新 ←——————→ 强调执行
分享权力 ←——————→ 集中权力
个性鲜明 ←——————→ 低调内敛
```
在六个维度上标注企业家的位置，并对比企业当前阶段需要的领导风格。

**D. 双维平衡模型**：
```
        关注人
          ↑
          │  教练型     │   民主型
          │             │
          ├─────────────┼──────────→ 关注事
          │  放羊型     │   指令型
          │
```
理性-感性 × 关注人-关注事，判断企业家的主导风格是否匹配企业发展阶段需求。

> 使用策略：通常在成功经验部分的文化归因中，用"四力"和"四有"解释"为什么他能做成"；在问题诊断部分，用领导行为图谱和双维模型解释"为什么现在这套不够用了"。

### 维度6：经营压力（行业特定）

根据行业特点分析：

**业绩压力**：
- 收入和利润是否下滑？
- 收入是否集中于个别品种？
- 海外市场拓展进度是否符合预期？

**竞争压力**：
- 主力品种市场份额是否在下降？
- 竞争对手产品迭代是否更快？
- 替代品威胁有多大？

**发展压力**：
- 产品管线和平台布局是否聚焦？
- 新基地/产能是否空转？
- 研发管线是否存在断层？
- 政策环境是否趋严？

## 三、问卷分析方法

### 发展阶段锚定

| 问卷总分 | 对应阶段 |
|---------|---------|
| <3.5 | 创业期或机会成长早期 |
| 3.5-4.2 | 机会成长向系统成长过渡 |
| 4.2-4.5 | 系统成长初期 |
| >4.5 | 系统成长成熟，可向分蘖成长过渡 |

### 交叉分析维度

1. **层级交叉**：高管 vs 中基层。高管打分通常更低（更了解全貌），差距越大→问题越严重
2. **部门交叉**：各事业部/部门打分差异→识别管理状况和士气的差异
3. **维度交叉**：战略/运营/组织/人才/文化五个维度的得分雷达图→识别短板

### 异常信号识别

- 高管打分低于3.0的任何题目→严重问题
- 部门间打分差距超过0.8→存在明显管理质量差异
- 距离总部越近打分越低→可能是官僚化信号
- 中基层打分远高于高管→信息不对称或盲目乐观

## 四、根因分析：三大矛盾诊断

### 矛盾一："业务——能力"

**判断依据**：
- 财务上：营收增长但净利率持续走低
- 战略上：方向正确但执行不到位
- 人才上：管理能力和专业能力跟不上业务复杂度
- 组织上：管理体系（如战略管理、研发管理、项目管理）没有建立

**典型症状**：起个大早赶个晚集、项目拖期、机会抓不住

### 矛盾二："成长——治理"

**判断依据**：
- 企业规模已到中大型，但决策仍靠个人
- 缺少高管团队和集体决策机制
- 缺少组织规则和制度流程
- 企业家亲力亲为、事务缠身

**典型症状**：事事上报、授权不充分、总裁办公会流于形式

### 矛盾三："环境——进化"

**判断依据**：
- 核心价值观和认知方式不适应新环境
- 需要从机会导向→能力导向、从交易型→深耕型等"转向"
- 文化层面需要进行深层次变革

**典型症状**：文化稀释、创业精神流失、自满封闭

## 五、成功要素分析方法

### "赢的逻辑"七维框架

这是从巴奴案例中提炼的系统化成功要素分析方法。从宏观到微观、逐层下钻：

```
赢在赛道（业务选择）
  └→ 赢在空间（市场定位）
       └→ 赢在价值（价值主张）
            └→ 赢在模式（商业模式）
                 └→ 赢在胜点（致胜因素）
                      └→ 赢在节奏（战略展开）
                           └→ 赢在能力（关键专长）
```

**使用方式**：在诊断的第一部分（成功经验回顾）中，用此框架逐层分析企业过去的成功逻辑，而不是简单罗列成功要素。每一层都要回答一个"为什么赢"的问题：
1. 赛道：为什么是这个行业/品类？进入时机如何？
2. 空间：为什么是这个市场/客群？定位依据是什么？
3. 价值：给客户解决了什么问题？为什么客户愿意买单？
4. 模式：怎么赚钱？收入结构、成本结构、盈利逻辑是什么？
5. 胜点：凭什么比竞争对手强？核心差异点在哪？
6. 节奏：什么先做、什么后做？战略展开的顺序逻辑是什么？
7. 能力：哪些组织专长支撑了以上六点？

根据行业和企业情况，7层不一定全部展开——但至少覆盖赛道、价值、胜点、能力四层。

### 战略飞轮模型

当企业成功经验呈现出各要素之间的正反馈闭环时，使用战略飞轮图示总结：

```
外部产业机遇
    ↓
战略选择（定位/细分市场）
    ↓
能力建设（技术/渠道/供应整合）
    ↓
市场聚焦（成为细分冠军）
    ↓
业务升级（从产品到解决方案）
    ↓
空间突破（跨区域/国际化）
    ↓ （回到起点，飞轮加速）
外部产业机遇（新一轮）
```

在MD报告和PPT中，战略飞轮以一页图示呈现，用于展示成功逻辑的自强化机制——既是对过去的总结，也为第三部分的变革建议埋下伏笔（飞轮如果卡壳，卡在哪一环）。
