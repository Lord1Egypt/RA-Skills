# Duty Estimate Quality Checklist

Run through this list before publishing or pricing on the estimate. Any "no" is a blocker.

## HS code
- [ ] 6-digit code verified against destination tariff
- [ ] Full local code (8 or 10 digit) verified
- [ ] Description matches product
- [ ] Material composition documented
- [ ] If high stakes, binding ruling considered or obtained

## Valuation
- [ ] Valuation method documented (CIF / FOB / Transaction Value)
- [ ] All addable costs identified (freight, insurance, assists, royalties)
- [ ] Currency of declaration documented
- [ ] FX rate and date stated

## Duty calculation
- [ ] MFN rate confirmed against current tariff
- [ ] FTA eligibility evaluated
- [ ] Certificate of Origin obtainable if FTA used
- [ ] Section 301 / 232 / anti-dumping check (US imports from China especially)
- [ ] Anti-circumvention concerns reviewed (third-country routing)

## VAT / GST / consumption tax
- [ ] Correct rate for product category
- [ ] Tax base includes duty + addable costs
- [ ] Threshold regime applied (IOSS, £135 rule, Section 321, etc.)
- [ ] VAT registration in destination considered if volume warrants

## Surcharges
- [ ] MPF / HMF (US) included
- [ ] Brokerage line included
- [ ] Carrier disbursement fee included
- [ ] Category-specific levies (alcohol, tobacco, batteries) included

## Incoterm
- [ ] DDP vs DDU choice documented
- [ ] Cash-flow impact modeled if DDP
- [ ] Buyer-experience impact considered if DDU
- [ ] Prepay option evaluated if available

## Margin model
- [ ] Landed cost compared against destination retail
- [ ] Platform fees included
- [ ] FX margin modeled
- [ ] Returns reserve modeled
- [ ] Net margin acceptable per category target

## Documentation and operations
- [ ] Commercial invoice template ready
- [ ] Packing list template ready
- [ ] HS code embedded in product master
- [ ] Retention plan ≥5 years
- [ ] Reconciliation cadence agreed (monthly common)
- [ ] Variance threshold for investigation agreed (often 10%)

## Sign-off
- [ ] Finance signs off on landed cost
- [ ] Operations signs off on documentation flow
- [ ] Customer service briefed on duty messaging
