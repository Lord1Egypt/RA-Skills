# Income Tracker Skill

自动化每日收入追踪与债务管理系统。

## 安装

```bash
clawhub install income-tracker
```

## 功能

- 📊 每日收入自动记录
- 💳 债务追踪与进度可视化
- 📈 周期性财务报告
- ⚠️ 逾期提醒

## 使用

当提到以下关键词时自动激活：
- 收入、记账、还债、收支、财务、存款
