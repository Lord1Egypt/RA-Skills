## Description: <br>
FastPlayWright Skill provides high-performance browser automation with Fast Playwright MCP for token-optimized testing, form automation, screenshots, navigation flows, and diagnostics. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and automation engineers use this skill to configure and operate Fast Playwright MCP for browser testing, form automation, screenshot capture, navigation workflows, diagnostics, and efficient multi-step browser tasks. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill enables broad browser control, including web navigation, session access, form submission, JavaScript execution, and local file upload. <br>
Mitigation: Use isolated browser profiles, avoid sensitive logged-in sessions, and review each file upload, form submission, and JavaScript action before execution. <br>
Risk: The MCP server is installed through an npm package at runtime. <br>
Mitigation: Pin or audit the npm package before use in managed environments. <br>
