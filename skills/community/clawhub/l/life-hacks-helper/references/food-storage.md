# 食材储存指南

## 基本原则

1. **冰箱不是万能的**：有些食材不适合冷藏
2. **生熟分开**：防止交叉污染
3. **先买先吃**：注意保质期，合理安排
4. **适当包装**：保鲜盒、保鲜袋防串味防干缩

---

## 冷藏区（0-5°C）

### 可冷藏食材

| 食材 | 储存时间 | 注意事项 |
|-----|---------|---------|
| 生肉 | 1-2天 | 密封保存，放最下层 |
| 生鱼 | 1天 | 尽快食用 |
| 熟食 | 3-4天 | 密封保存 |
| 鸡蛋 | 3-4周 | 尖头朝下放 |
| 牛奶 | 5-7天 | 开封后尽快喝完 |
| 酸奶 | 1-2周 | 开封后3天内吃完 |
| 豆腐 | 3-5天 | 泡水里每天换水 |
| 绿叶菜 | 3-5天 | 包厨房纸+塑料袋 |
| 根茎类 | 1-2周 | 胡萝卜、萝卜等 |

### 冷藏技巧

- **绿叶菜**：用厨房纸包裹后装袋，纸会吸收多余水分
- **切开的水果**：用保鲜膜贴紧切面
- **剩菜**：等凉透再放冰箱，但不要超过2小时

---

## 冷冻区（-18°C以下）

### 适合冷冻

| 食材 | 冷冻时间 | 解冻方式 |
|-----|---------|---------|
| 肉类 | 3-6个月 | 冷藏室缓慢解冻最佳 |
| 鱼类 | 2-3个月 | 同上 |
| 海鲜 | 2-3个月 | 同上 |
| 面包 | 1-3个月 | 室温或烤面包机 |
| 饺子/包子 | 1-2个月 | 直接煮/蒸，不需解冻 |
| 冰淇淋 | 1-2个月 | |

### 冷冻技巧

- **分装**：大块肉分切成小份，每次取一份
- **标记日期**：写上冷冻日期
- **排出空气**：保鲜袋尽量排空空气，防冻伤

---

## 常温储存

### 适合室温

| 食材 | 储存方式 | 注意事项 |
|-----|---------|---------|
| 大米 | 密封罐 | 防潮防虫 |
| 面粉 | 密封罐 | 防虫，可放月桂叶 |
| 土豆 | 阴凉通风 | 避光，不与洋葱同放 |
| 洋葱 | 阴凉通风 | 不与土豆同放 |
| 大蒜 | 阴凉通风 | |
| 番茄 | 室温 | 未熟透时放室温 |
| 香蕉 | 室温 | 挂起来可延长保存 |
| 苹果 | 室温 | 可与土豆同放，延缓发芽 |

### ❌ 不要放冰箱

- **土豆**：低温会转化淀粉，影响口感
- **洋葱、大蒜**：潮湿会发霉
- **香蕉**：会变黑
- **番茄**：失去风味
- **蜂蜜**：会结晶
- **面包**：变干变硬更快（除非冷冻）

---

## 保鲜小技巧

1. **苹果+土豆**：放一起可延缓土豆发芽
2. **香蕉根部包裹保鲜膜**：延长保鲜2-3天
3. **切开的水果涂柠檬汁**：防止氧化变色
4. **青菜竖着放**：保持生长状态，更新鲜
5. **剩米饭冷冻**：冷藏会变硬，冷冻可保存1个月

---

## 变质判断

### 不能吃了

- 异味：酸臭、腐臭
- 颜色：发灰、发绿（非正常颜色）
- 质地：黏滑、发软、出水
- 霉点：任何可见霉菌

### 可以切除

- 水果、蔬菜的小块烂斑（切除周围2cm）
- 面包的一小块霉斑（如果只是表面，切除周围较大范围）

**有疑问就扔掉**，不值得冒险。
