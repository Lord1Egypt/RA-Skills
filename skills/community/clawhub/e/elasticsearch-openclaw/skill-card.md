## Known Risks and Mitigations: <br>
Risk: Examples may go beyond strict read-only use by creating API keys, modifying indexes, configuring inference, or changing cluster behavior. <br>
Mitigation: Review the examples before use and skip setup, indexing, pipeline, inference-endpoint, API-key-creation, or cluster-setting steps unless those changes are intentional. <br>
Risk: The skill requires Elasticsearch credentials for applied use. <br>
Mitigation: Use a narrowly scoped Elasticsearch API key limited to read and view-index-metadata privileges on specific indices, and avoid hardcoding credentials. <br>
