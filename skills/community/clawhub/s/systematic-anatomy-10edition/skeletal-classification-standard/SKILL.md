# 骨骼分类标准

## 何时使用
- 在解剖学教学中需对骨骼标本进行分类
- 在骨科临床评估中需明确骨的类型以指导诊断或手术
- 学习或复习人体骨骼系统时需应用标准分类方法

## 前提条件
- 已知目标骨的形态特征
- 具备基础解剖知识

## 使用限制
- 不适用于软骨或未骨化组织
- 不适用于病理状态下的骨变形（如骨折畸形愈合、肿瘤破坏等）

## 执行步骤
1. 观察骨的整体形态和结构特征。
2. 根据以下标准判断其解剖学类别：
   - **长骨**：长度显著大于宽度，具有骨干（diaphysis）和两个骨骺（epiphyses），例如股骨、肱骨。
   - **短骨**：长度与宽度相近，呈立方体状，例如腕骨、跗骨。
   - **扁骨**：薄而扁平，通常形成腔室以保护内脏，例如颅盖骨、胸骨、肩胛骨。
   - **不规则骨**：形状复杂，无法归入上述三类，例如椎骨、颞骨。
   - **含气骨**：内部含有与鼻旁窦相通的气腔，例如额骨、蝶骨、筛骨、上颌骨。
3. 若某骨兼具多种特征（如颞骨既有不规则骨部分，也含气腔），以其主要功能和结构为准进行归类。
4. 注意：干骺端、骺软骨和骺线是长骨发育过程中的关键结构，可用于判断骨龄，但不影响基本分类。

## 输出
返回骨的解剖学分类名称（如“长骨”、“含气骨”等）。
