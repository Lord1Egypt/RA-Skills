## 黄金/原油数据格式（新浪 hf_）

```
URL: https://hq.sinajs.cn/list=hf_GC
响应: var hq_str_hf_GC="4650.856,,4652.700,4653.800,4699.300,4626.200,10:01:36,4679.700,4675.000,0,1,1,2026-04-06,纽约黄金,0"

字段解析（逗号分隔）：
[0]  = 4650.856  ← 最新价
[1]  = ''        ← 空字段（占位）
[2]  = 4652.700  ← 开盘价（或昨收，需核实）
[3]  = 4653.800  ← 最高价
[4]  = 4699.300  ← 最低价
[5]  = 10:01:36  ← 更新时间
[6]  = 4679.700  ← 买价（bid）
[7]  = 4675.000  ← 卖价（ask）
[12] = 2026-04-06 ← 日期
[13] = 纽约黄金   ← 品种名称

⚠️ 注意：字段1是空占位，解析时需跳过
```

## A50解决方案

**问题**：Yahoo Finance 403屏蔽，新浪A50数据为空

**替代方案（已实现）**：
1. 港股恒生指数（hkHSI）作为亚太市场情绪代理
2. QDII ETF（纳指ETF 513100）盘中反映美股期货情绪
3. 国内商品期货（铁矿石/螺纹钢）反映工业需求
4. 黄金/原油反映全球宏观+地缘风险

**综合研判逻辑**：
```
A50走强 → 亚太市场情绪积极 → A股高开概率+
黄金涨 → 避险情绪升温 → 股市承压
原油涨 → 通胀预期+地缘风险 → 政策宽松预期-
纳指ETF涨 → 美股夜盘积极 → A股情绪+
```
