## 二、GIA 钻石证书详解

### GIA 证书类型
| 类型 | 适用 | 价格(USD) | 特点 |
|------|------|----------|------|
| Diamond Grading Report | 0.15ct+ 天然钻 | $70-250+ | 完整4C+净度图 |
| Diamond Dossier | 0.15-1.99ct | $40-75 | 精简版(无净度图) |
| Colored Diamond Report | 彩钻 | $120-400+ | 颜色等级+4C |
| Laboratory Grown Report | 培育钻石 | $60-150 | 标注Lab Grown |

### GIA 证书关键字段解读

```
GIA Report Number: 1234567890     ← 官网 gia.edu 验证真伪
Shape and Cutting Style: Round Brilliant
Measurements: 6.48 - 6.52 x 4.01 mm  ← 直径和高度

CARAT WEIGHT: 1.01 carat           ← 克拉重量
COLOR GRADE: G                     ← D-Z 颜色等级
CLARITY GRADE: VS2                 ← 净度等级
CUT GRADE: Excellent               ← 切工等级

POLISH: Excellent                  ← 抛光
SYMMETRY: Excellent                ← 对称性
FLUORESCENCE: None                 ← 荧光

CLARITY CHARACTERISTICS:           ← 内含物特征图
  Crystal, Feather                 ← 晶体包体、羽裂纹

INSCRIPTION(S): GIA 1234567890     ← 腰围镭射编码
```

### GIA 证书真伪验证
1. 访问 https://www.gia.edu/report-check
2. 输入 Report Number
3. 核对重量：证书重量需精确匹配（允许误差 0.001ct）
4. 核对净度图：内含物位置和类型需匹配实物

## 四、NGTC 国检证书解读

### 翡翠证书关键字段
```
检验结论：翡翠(A货)               ← 最关键的结论
总质量：56.23g
形状：手镯(圆镯)
颜色：绿色
透明度：半透明
质地：细粒
光性特征：非均质集合体
折射率：1.66(点测)
放大检查：粒状纤维交织结构        ← 天然翡翠特征
备注：-
```

### A货 vs 处理货证书对比
| 字段 | A货证书 | B货处理证书 |
|------|---------|------------|
| 检验结论 | 翡翠(A货) 或 翡翠 | 翡翠(处理) |
| 放大检查 | 粒状纤维交织结构 | 结构破坏/表面酸蚀网纹 |
| 备注 | 无特殊备注 | 注胶处理/染色处理 |
| 红外光谱 | 天然翡翠吸收峰 | 有机物吸收峰(胶) |

## 六、常见证书骗局
1. **假证书**：仿冒 GIA/GRS 证书模板，官网验证是唯一可靠方式
2. **套证**：真证书配假宝石，核对重量/尺寸/内含物是否匹配
3. **"国际证书"噱头**：不知名小机构出具的"国际鉴定证书"无参考价值
4. **PS 证书**：证书照片被盗用 P 图
5. **证书夸大评级**：不知名机构随意给"鸽血红"等评级，以 GRS/Gübelin 为准
