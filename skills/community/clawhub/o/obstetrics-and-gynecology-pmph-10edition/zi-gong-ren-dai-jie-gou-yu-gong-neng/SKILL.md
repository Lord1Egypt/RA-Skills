# 子宫韧带结构与功能

## 适用场景
- 评估子宫脱垂风险
- 妇科手术中识别关键解剖结构（如广泛子宫切除术）
- 分析骨盆底支持结构对子宫的固定作用

## 执行步骤

### 1. 子宫阔韧带（broad ligament）
- **结构**：双层腹膜皱襞，上缘游离；内2/3包绕输卵管（伞部无腹膜），外1/3包绕卵巢动静脉形成骨盆漏斗韧带（即卵巢悬韧带）；卵巢与子宫角间增厚部分为卵巢固有韧带。
- **功能**：限制子宫向两侧倾斜。
- **注意**：子宫旁组织位于其内，含血管、神经、淋巴管及疏松结缔组织；子宫动静脉和输尿管从基底部穿过。

### 2. 子宫圆韧带（round ligament）
- **结构**：圆索状，长12～14 cm，由平滑肌和结缔组织构成；起自子宫角前面上方，经阔韧带前叶向前外，穿腹股沟管止于大阴唇前端。
- **功能**：维持子宫前倾位置。

### 3. 子宫主韧带（cardinal ligament）
- **结构**：位于阔韧带下部，横行于宫颈与骨盆侧壁之间；由坚韧的平滑肌与结缔组织束构成。
- **功能**：固定宫颈，是防止子宫脱垂的主要支撑结构。

### 4. 子宫骶韧带（uterosacral ligament）
- **结构**：起自子宫体与宫颈交界后上方，绕过直肠止于第2、3骶椎前筋膜；含平滑肌、结缔组织及支配膀胱的神经。
- **功能**：向后上牵引宫颈，与圆韧带协同维持子宫前屈前倾位。
- **注意**：广泛子宫切除术中切断此韧带可导致术后尿潴留。

## 使用提示
- 在手术中优先识别主韧带和骶韧带，因其对子宫固定至关重要且邻近重要神经血管。
- 评估脱垂时，重点检查主韧带和骶韧带的张力与完整性。
- 阔韧带内结构复杂，操作时需警惕输尿管走行。
