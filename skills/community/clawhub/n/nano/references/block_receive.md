# xno-skills block receive

```
Usage: cli block receive [options]

Build an unsigned receive block

Options:
  -a, --account <address>  Recipient Nano address
  --hash <blockhash>       Hash of the pending send block
  --amount-raw <raw>       Amount in raw
  --amount-xno <xno>       Amount in XNO
  --url <url>              RPC URL override
  -j, --json               Output JSON with block hex + metadata
  -h, --help               display help for command
```
