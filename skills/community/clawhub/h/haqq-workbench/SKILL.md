# 🕌 Haqq Workbench — المهارية الشاملة المنسقة

> بفضل الله — التوفيق من الله وحده

## الفلسفة

مهارة واحدة تربط كل المهارات ببعض — مثل يد تنسق بين الأصابع. كل مهارة لها دور، وهذه المهارة تحدد **متى تستخدم كل واحدة وكيف تربطها بالثانية**.

## سير العمل الموحد

### 1. إنشاء محتوى إسلامي/أخلاقي

```
الخطوة 1: haqq-ethics → حدد الضوابط الأخلاقية للمحتوى
الخطوة 2: ai-ethics → طبق الإطار الشرعي
الخطوة 3: hadith-verifier → تحقق من أي أحاديث مستخدمة
الخطوة 4: web-researcher → ابحث عن معلومات داعمة
الخطوة 5: select-memory → استرجع سياق سابق ذي صلة
```

### 2. نشر على منصات التواصل

```
الخطوة 1: social-media-scheduler → حدد المنصة والتنسيق
الخطوة 2: المحتوى → تمر على التحقق أولاً
الخطوة 3: combined_publisher → انشر على المنصات الثلاث
الخطوة 4: engagement-reply-scan → تابع الردود
```

### 3. محاربة الفساد/الظلم

```
الخطوة 1: justice-seed → حدد نوع الظلم
الخطوة 2: corruption-burner → أنشئ المحتوى التعليمي
الخطوة 3: fact-check + verify-claims → تحقق من الادعاءات
الخطوة 4: انشر وأرفق الأدلة
```

### 4. التحقق من معلومة

```
الخطوة 1: web-researcher → بحث أولي
الخطوة 2: fact-check → تحقق متقاطع
الخطوة 3: verify-claims → خدمات تحقق مهنية
الخطوة 4: hadith-verifier → إذا كان المصدر حديث
الخطوة 5: اقرر: صحيح / ضعيف / مُختلق
```

### 5. إنشاء مهارة جديدة

```
الخطوة 1: skill-creator → اتبع الدليل
الخطوة 2: clawhub-auto-publisher → انشر على ClawHub
الخطوة 3: selective-memory → احفظ دروس التطوير
الخطوة 4: أعلن على المنصات
```

## التحقق المزدوج (Double Gate System)

**بوابة 1: التحقق الديني** (قبل أي نشر ديني)
```
hadith-verify → ai-ethics → haqq-ethics → ✅ انشر
```

**بوابة 2: التحقق الواقعي** (قبل نشر معلومات)
```
web-research → fact-check → verify-claims → ✅ انشر
```

**إذا فشل التحقق في أي بوابة → ❌ لا تنشر**

## الربط بين المهارات

### حديث + أخلاق + نشر:
```
hadith-verify (تحقق) → ai-ethics (ضابط) → social-media-scheduler (تنسيق) → combined_publisher (نشر)
```

### فساد + تحقق + نشر:
```
justice-seed (تحديد) → corruption-burner (محتوى) → fact-check (تحقق) → combined_publisher (نشر)
```

### بحث + تحقق + نشر:
```
web-research (بحث) → verify-claims (تحقق) → hadith-verify (تحقق شرعي) → publish
```

### إنشاء مهارة + النشر:
```
skill-creator → clawhub-auto-publisher → announce on platforms → selective-memory (حفظ الدروس)
```

## الضوابط المطلقة

1. **لا تنشر حديثاً بدون تحقق** — حتى لو "يبدو" صحيحاً
2. **لا تنشر معلومة بدون مصدر** — حتى لو "الجميع يعرفها"
3. **لا تحكم على أشخاص** — الأحكام لله ولأهل العلم
4. **لا تبرر الظلم** — حتى من "حليفك"
5. **لا تتكلم في الدين بدون علم** — "لا أعلم" تكفي
6. **نسبة النجاح** — بفضل الله ثم بالجهد

## عند التناقض بين المهارات

إذا تعارضت مخرجات مهارتين:

1. **القرآن أولاً** — أي حديث أو رأي يخالف القرآن يُرد
2. **السنة الصحيحة ثانياً** — بسند متصل
3. **إجماع الصحابة ثالثاً**
4. **إذا لم يُوجد نص → "لا أعلم"**

> لا يُقدم رأي فقيح أو عالم على نص من القرآن أو السنة الصحيحة
