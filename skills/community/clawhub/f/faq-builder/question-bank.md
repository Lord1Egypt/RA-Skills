# Question Bank by Product Category

## Apparel & Footwear
- Does it run true to size, small, or large?
- What are the exact measurements for size [X]?
- What material is it made from? Is it breathable?
- How should I wash and care for this item?
- Will the color fade after washing?
- Is it stretchy? How much stretch does it have?
- Is it see-through/sheer?
- Can it be worn in rain / is it water-resistant?
- Are the pockets functional?

## Electronics & Tech Accessories
- Is it compatible with [iPhone/Android/MacBook/Windows]?
- Does it require batteries or is it rechargeable?
- What is the battery life?
- Is it Bluetooth or does it require a cable?
- What's in the box — does it come with [charging cable/accessories]?
- Does it work with [Alexa/Google Assistant/Siri]?
- What's the warranty? What does it cover?
- Can I use it in multiple countries (voltage compatibility)?

## Kitchen & Home
- What are the dimensions (length × width × height)?
- Is it dishwasher safe?
- Is it BPA-free / food-safe?
- What materials is it made from?
- Is it compatible with induction / gas / electric stovetops?
- Does it come with a lid / accessories?
- Is it oven-safe? Up to what temperature?
- Will it scratch non-stick surfaces?

## Beauty & Personal Care
- Is it suitable for sensitive skin?
- Is it cruelty-free / vegan?
- Does it contain fragrance / parabens / sulfates?
- How long does one unit last with daily use?
- Is it dermatologist tested?
- Can it be used on [hair type / skin tone / specific concern]?
- Is there a return policy if it causes a reaction?

## Supplements & Health
- Are there any allergens? (gluten, soy, dairy, nuts)
- Is it third-party tested?
- What is the recommended dosage?
- Can it be taken with [medication X]? (answer: consult your doctor)
- Is it safe for pregnant / breastfeeding women?
- What's the expiry date / shelf life?
- What certifications does it have? (GMP, NSF, Informed Sport)

## Pet Products
- What size / breed is this designed for?
- Is it safe if my pet chews on it?
- What materials are used? Are they non-toxic?
- Is it machine washable?
- Can it be used for both cats and dogs?
- Is the size adjustable?

## Baby & Kids
- What age range is it appropriate for?
- Is it BPA-free / phthalate-free?
- Does it meet safety certifications? (CPSC, ASTM, EN71)
- Is it machine washable?
- Can small parts be swallowed? (choking hazard)
- What are the exact dimensions?

## Sports & Fitness Equipment
- What weight capacity does it support?
- What is the assembled size vs. storage/folded size?
- Does it require assembly? How long does it take?
- What are the warranty terms?
- Can it be used on carpet / hardwood / outdoors?
- Is it compatible with [fitness tracker / app]?

## Universal High-Priority Questions (Any Category)
- What is the return policy?
- How long does shipping take?
- Do you ship internationally?
- Is this the official/authentic product? (important for Amazon)
- What does the warranty cover and how long is it?
- How do I contact support if I have an issue?
