# Peerberry SDK

This site contains documentation for the `peerberry_sdk` Python package.

## What This SDK Provides

- High-level API client for Peerberry investor operations
- Typed models for most response entities
- Retry and timeout control for transport reliability
- Authentication and token refresh lifecycle management
- Structured exceptions with request context metadata
- Optional observability events for retries/errors/refreshes
