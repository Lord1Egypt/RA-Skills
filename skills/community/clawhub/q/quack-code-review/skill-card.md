## Description: <br>
AI-powered code analysis via LogicArt for finding bugs, security issues, complexity signals, suggestions, and logic flow. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[JPaulGrayson](https://clawhub.ai/user/JPaulGrayson) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and engineers use this skill to send code snippets or selected files to LogicArt for review and receive bug, security, complexity, suggestion, and logic-flow analysis. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Reviewed code is sent to LogicArt and may expose private source code. <br>
Mitigation: Avoid using the skill on secrets, credentials, private customer data, or proprietary code unless the user trusts LogicArt and its data-handling practices. <br>
