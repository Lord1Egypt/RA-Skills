# 论文润色标准参考

## 学术写作风格指南

### 语言风格要求
- **客观性**: 避免主观表达，使用被动语态或第三人称
- **简洁性**: 删除冗余词汇，保持句子紧凑
- **准确性**: 使用精确的专业术语，避免模糊表达
- **连贯性**: 确保段落间逻辑流畅，使用恰当的连接词

### 常见问题修正清单

#### 语法与句式
- [ ] 检查主谓一致性
- [ ] 修正悬垂修饰语
- [ ] 统一时态（方法部分用过去时，结论用现在时）
- [ ] 避免过长句子（建议不超过30词）
- [ ] 检查平行结构

#### 学术表达
- [ ] 避免绝对化词汇（prove, always, never）改用（suggest, indicate, demonstrate）
- [ ] 删除口语化表达（kind of, sort of, basically）
- [ ] 使用学术连接词（however, moreover, therefore, furthermore）
- [ ] 检查缩写首次使用是否全称标注

#### 中英文学术写作差异

**中文特点**:
- 多用主动语态
- 句子较长，结构松散
- 重复表达较多

**英文特点**:
- 多用被动语态（方法部分）
- 句子简洁，结构紧凑
- 避免重复，多用代词指代

### 术语对照表

## 润色工作流程

### 第一遍：结构审查
1. 检查段落完整性（主题句+支撑句+结论句）
2. 确认论证逻辑顺序
3. 标注需要扩展或精简的部分

### 第二遍：语言优化
1. 语法错误修正
2. 词汇升级（口语→学术）
3. 句式多样化
4. 连接词优化

### 第三遍：格式规范
1. 检查引用格式（APA/MLA/Chicago/IEEE）
2. 统一数字表达
3. 检查缩写一致性
4. 图表标注规范
