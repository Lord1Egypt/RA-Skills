# RedNote Presets

These presets are shortcuts that combine `style + layout`.

| preset | style | layout | Best For |
| --- | --- | --- | --- |
| `knowledge-card` | `notion` | `dense` | practical summaries and concept explainers |
| `checklist` | `notion` | `list` | checklists, tool recommendations, step summaries |
| `tutorial` | `chalkboard` | `flow` | tutorials and step-by-step cards |
| `study-guide` | `chalkboard` | `dense` | study notes and key-point review |
| `cute-share` | `cute` | `balanced` | light sharing and atmosphere-driven posts |
| `cozy-story` | `warm` | `balanced` | story-driven or emotional content |
| `warning` | `bold` | `list` | warnings, pitfalls, and alerts |
| `versus` | `bold` | `comparison` | direct comparison and A/B choices |
| `clean-summary` | `minimal` | `balanced` | concise summaries and professional opinions |
| `retro-ranking` | `retro` | `list` | rankings and retro-themed lists |
| `editorial-cover` | `editorial` | `sparse` | commentary covers and single hero visuals |

Recommendation rules:

- If the user does not specify a style, knowledge content should usually start with `knowledge-card`.
- Tutorial content should usually start with `tutorial`.
- Strong opinion content should usually start with `warning` or `versus`.
- Lifestyle content should usually start with `cute-share` or `cozy-story`.
