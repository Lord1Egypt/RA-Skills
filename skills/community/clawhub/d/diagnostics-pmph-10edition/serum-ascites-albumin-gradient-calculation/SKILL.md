# 腹腔积液白蛋白梯度（SAAG）计算与解读

## 适用条件
- 患者存在腹腔积液
- 需鉴别腹水是否由门静脉高压引起
- 已同时获取血清和腹水样本

## 执行步骤

1. 同时采集患者的血清样本和腹水样本。

2. 使用生化分析仪分别测定血清白蛋白和腹水白蛋白浓度（单位统一为 g/L 或 g/dL）。

3. 计算 SAAG：
   ```
   SAAG = 血清白蛋白浓度 - 腹水白蛋白浓度
   ```

4. 解读结果：
   - **SAAG ≥ 11 g/L（或 1.1 g/dL）**：提示门静脉高压性腹水（如肝硬化、布-加综合征、心源性腹水）；
   - **SAAG < 11 g/L（或 1.1 g/dL）**：提示非门静脉高压性腹水（如恶性肿瘤、结核性腹膜炎、胰腺炎）。

## 注意事项
- 必须使用同一时间点的血清与腹水样本；
- 若患者近期输注白蛋白，应在输注后至少6小时再采样；
- 不适用于胸腔积液等非腹腔积液；
- 样本溶血或污染可能影响结果准确性。
