# 项目筛选决策树

## 概述

通过结构化决策流程，实现从路演项目到精准对接的智能筛选。

---

## 二、分层决策标准

### 2.1 第一层：进入"10个重点对象"

**必要条件（必须全部满足）：**
- [ ] 关注度评分 ≥ 60分
- [ ] 无高风险预警
- [ ] 核心问题回答满意度 ≥ 3分（5分制）
- [ ] 至少1位投资人有跟进意向

**加分项（满足任意一项）：**
- [ ] 关注度评分 ≥ 80分（+5分）
- [ ] 有3位以上投资人有跟进意向（+5分）
- [ ] 回答质量全优（+3分）
- [ ] 团队主动要求深度对接（+2分）

### 2.2 第二层：进入"2-3个精准对接"

**必要条件（必须全部满足）：**
- [ ] 已进入重点对象池
- [ ] 有1位以上投资人明确表达意向
- [ ] 核心亮点与特定机构偏好匹配
- [ ] 团队配合度高
- [ ] 估值有谈判空间

**加分项：**
- [ ] 有2位以上投资人明确意向（+5分）
- [ ] 机构匹配度 ≥ 90%（+3分）
- [ ] 可提供独家尽调机会（+2分）

### 2.3 滚动决策

**降级标准：**
- 路演后2周内无任何跟进
- 投资人反馈持续负面
- 项目方态度消极
- 新发现重大风险

**升级标准：**
- 路演后投资人主动多次询问
- 项目取得重大进展
- 有新投资机构主动联系

## 三、决策判断矩阵

### 3.1 关注度 × 风险等级

| | 低风险 | 中风险 | 高风险 |
|------------|--------|--------|--------|
| **高关注度** | ★★★ 精准对接 | ★★ 重点培育 | ☆ 风险澄清 |
| **中关注度** | ★★ 重点培育 | ★ 继续观察 | ☆ 暂缓 |
| **低关注度** | ★ 继续观察 | ☆ 暂缓 | ☆ 暂缓 |

**说明：**
- ★★★ 直接进入精准对接
- ★★ 优先进入重点池
- ★ 进入正常观察
- ☆ 需要进一步评估或暂缓

### 3.2 意向强度 × 机构匹配度

| | 高匹配 | 中匹配 | 低匹配 |
|------------|--------|--------|--------|
| **明确意向** | 直接对接 | 优先对接 | 评估后对接 |
| **模糊意向** | 重点跟进 | 持续跟进 | 保持联系 |
| **无明确意向** | 培育跟进 | 观察 | 暂不跟进 |

## 四、具体决策规则

### 规则1：高关注度 + 低风险

```
IF 关注度≥80 AND 风险等级=低 THEN
  → 进入"精准对接"候选
  → 立即启动机构匹配
  → 优先安排深度对接会
```

### 规则2：高关注度 + 中风险

```
IF 关注度≥80 AND 风险等级=中 THEN
  → 进入"重点培育"
  → 需要风险澄清或补充材料
  → 安排一对一沟通
  → 澄清后重新评估
```

### 规则3：高关注度 + 高风险

```
IF 关注度≥80 AND 风险等级=高 THEN
  → 进入"风险观察"
  → 重大风险需项目方书面回复
  → 风险解除后可重新评估
  → 若风险持续，不推进对接
```

### 规则4：中关注度项目

```
IF 关注度 60-79 THEN
  → 进入"重点培育池"
  → 持续跟踪路演反馈
  → 若后续表现提升，可升级
  → 季度内3场路演后重新评估
```

### 规则5：低关注度项目

```
IF 关注度<60 THEN
  → 进入"培育观察池"
  → 滚动到下一季度继续培育
  → 不占用本季度重点资源
  → 下季度重新参加路演
```

## 五、决策时间节点

### 5.1 实时决策

- 路演结束后24小时内：初步关注度评估
- 路演结束后48小时内：风险标记
- 路演结束后72小时内：初步分类

### 5.2 周度决策

- 每周一：上周路演项目分类确认
- 重点池更新公告

### 5.3 月度决策

- 月末复盘：重点池调整
- 精准对接启动决策

### 5.4 季度决策

- 季度末：全面复盘
- 下季度项目池更新

## 六、决策豁免与例外

### 6.1 豁免条件

以下情况可豁免标准流程：
- [ ] 顶级机构主动表示兴趣
- [ ] 项目有重大技术突破
- [ ] 政策利好加持
- [ ] 导师特别推荐

### 6.2 特殊情况处理

| 情况 | 处理方式 |
|------|----------|
| 投资机构抢投 | 加速对接流程 |
| 项目估值突涨 | 重新评估性价比 |
| 团队重大变动 | 暂停对接，重新尽调 |
| 竞争对手注资 | 评估竞争格局变化 |
