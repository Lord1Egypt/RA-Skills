# xno-skills rpc account-balance

```
Usage: cli rpc account-balance [options] <address>

Fetch account balance and pending amount

Arguments:
  address            Nano address

Options:
  --url <url>        RPC URL override
  --timeout-ms <ms>  Timeout in milliseconds (default: 15000)
  --xno              Also include XNO-formatted values
  -j, --json         Output in JSON format
  -h, --help         display help for command
```
