# 瓦匠施工工序参考

## 装修整体顺序中的瓦匠阶段

```
水电改造 → 防水 → 瓦匠（贴砖） → 木工 → 油漆 → 安装
```

**瓦匠进场前提**：
- 水电改造完毕并验收
- 墙面/地面管线槽已封堵
- 厨房/卫生间排气道已安装
- 窗户已安装或框已装好

## 瓦匠标准工序（按执行顺序）

### 第 1 步：基层处理
1. 清理基层：浮灰、油污、松散层清除
2. 润湿基层：提前 4-6h 洒水湿润，无明水
3. 打点冲筋（找平用）：
   - 用激光水平仪确定最高点
   - 灰饼间距 1.2-1.5m
4. 墙面找平（误差 > 5mm 时需要）
5. 地面找平（铺木地板前需要）

### 第 2 步：防水工程
1. 管根/墙角抹弧（R角处理）
2. 刷第一遍防水
3. 贴网格布（管根、墙角、下水口）
4. 刷第二遍防水（与第一遍十字交叉）
5. 干透后做闭水试验 24-48h
6. 蓄水深度 ≥ 20mm，观察楼下无渗漏
7. 做保护层（水泥砂浆覆盖）

### 第 3 步：瓷砖铺贴

**铺贴前准备**：
- 检查瓷砖色号、规格一致（同一批号）
- 预排版（确定起铺点、避免窄条）
- 瓷片泡水 2h 以上（全瓷砖不泡水）
- 检查墙面垂直度、地面平整度

**铺贴顺序**：
1. 先墙后地（墙砖压地砖）
2. 先大面后排角落
3. 从门口最显眼处起铺
4. 墙砖从下往上贴

**关键工艺节点**：

| 节点 | 做法 | 备注 |
|------|------|------|
| 壁龛 | 坡度向外 1%，内侧略高外侧略低 | 避免积水 |
| 地漏 | 回字形铺贴，地漏处四角切砖 | 排水更快 |
| 海棠角 | 45° 倒角留 1-2mm 边 | 阳角保护 |
| 门槛石 | 比卫生间地面高 5-10mm | 挡水 |
| 窗台石 | 两侧入墙 ≥ 20mm，出墙 10-15mm | 美观/防水 |
| 阴阳角 | 阴角留缝 2-3mm，阳角海棠角 | 防止热胀开裂 |

### 第 4 步：养护与勾缝

**养护**：
- 铺贴后 24h 内：禁止踩踏（地砖）
- 铺贴后 48h：可轻步走动
- 铺贴后 7 天：完全固化

**勾缝/美缝时机**：
- 水泥砂浆铺贴：贴完 **3-5 天**后可勾缝
- 瓷砖胶铺贴：贴完 **24h** 后可美缝
- 条件：砖缝必须完全干燥（48h 以上保持干燥）→ 可做美缝

## 各空间瓦匠工序差异

| 空间 | 核心工序 | 工期（天） |
|------|---------|-----------|
| 卫生间 | 找平→防水→闭水→保护→墙砖→地砖 | 5-7 |
| 厨房 | 墙砖→地砖→窗台石 | 3-4 |
| 阳台 | 找坡→防水→闭水→地砖 | 3-5 |
| 客厅 | 找平（可选）→地砖 | 2-3 |

## 养护周期

| 工序 | 养护时间 | 养护方式 |
|------|---------|---------|
| 水泥砂浆找平 | 7 天 | 每日洒水 2-3 次 |
| 防水层 | 2-3 天 | 自然阴干 |
| 瓷砖铺贴 | 7 天 | 自然养护，禁水 |
| 美缝固化 | 24-48h | 禁踩踏、禁水 |
