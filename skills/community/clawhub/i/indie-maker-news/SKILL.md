## 核心价值

**变现导向精选**：只推送与"变现"相关的资讯，每条都有行动价值

**智能分类**：
- 🏆 成功案例：独立开发者变现经验
- 💡 创业模式：SaaS、数字产品、咨询服务
- ⚠️ 失败反思：避坑指南
- 🔧 工具推荐：效率/营销工具

---

## 🚀 一键使用

直接运行聚合脚本：
```bash
./scripts/aggregate.sh
```

**无需任何配置**！脚本会自动：
- 解析公开RSS源（36氪、知乎、V2EX、Hacker News）
- 智能筛选变现相关内容
- 输出结构化JSON数据

## 📡 数据源（10个可靠源）

**创业投资类（4个）**：
- 36氪（创业、融资、SaaS、变现）
- 虎嗅（商业模式、商业洞察）
- 创业邦（创业者、融资动态）
- **华尔街见闻**（财经资讯、投资、融资）✨ 新增

**开发者社区类（3个）**：
- V2EX综合（独立开发、副业、赚钱）
- V2EX程序员（程序员话题、开源）
- Indie Hackers（独立开发、revenue、profit）

**技术前沿类（3个）**：
- Hacker News（startup、side project）
- GitHub Trending（开源工具、效率）
- 掘金前端（技术社区、副业）

**智能筛选**：
- 关键词匹配：变现、创业、副业、SaaS、独立开发、赚钱等
