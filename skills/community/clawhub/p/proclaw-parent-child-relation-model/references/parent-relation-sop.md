## 二、阶段一：问题识别

### 2.1 触发识别

#### 自动触发（常规）
- 日常观察发现异常
- 老师反馈
- 孩子主动表达

#### 主动筛查（定期）
- 每月家庭评估
- 季度深度沟通
- 年度系统回顾

### 2.2 信息收集

#### 孩子侧信息

| 信息类型 | 收集内容 | 收集方法 |
|----------|----------|----------|
| 行为信息 | 具体表现/频率/情境 | 观察/记录 |
| 情绪信息 | 情绪表达/调节方式 | 对话/观察 |
| 关系信息 | 与父母/同伴关系 | 沟通/询问 |
| 学习信息 | 态度/动力/困难 | 交流/观察 |

#### 父母侧信息

| 信息类型 | 收集内容 | 收集方法 |
|----------|----------|----------|
| 感受信息 | 对孩子行为的感受 | 自我反思 |
| 应对信息 | 通常如何回应 | 自我观察 |
| 期待信息 | 希望孩子怎样 | 自我澄清 |
| 模式信息 | 自己的成长经历 | 自我探索 |

#### 系统侧信息

| 信息类型 | 收集内容 | 收集方法 |
|----------|----------|----------|
| 家庭结构 | 成员/关系/边界 | 家庭图谱 |
| 家庭动力 | 互动模式/冲突 | 观察/沟通 |
| 外部压力 | 工作/经济/时间 | 自我评估 |

### 2.3 初步判断

#### 问题分类

| 类型 | 特征 | 初步判断 |
|------|------|----------|
| 行为问题 | 孩子行为异常 | ___ |
| 情绪问题 | 情绪反应异常 | ___ |
| 关系问题 | 亲子互动异常 | ___ |
| 发展问题 | 发展里程碑延迟 | ___ |

#### 紧急程度

| 等级 | 标准 | 处理 |
|------|------|------|
| 紧急 | 涉及安全/伤害 | 立即处理 |
| 重要 | 持续存在/影响明显 | 优先处理 |
| 一般 | 偶发/影响有限 | 常规处理 |
| 观察 | 需进一步确认 | 持续观察 |

## 五、阶段四：执行实施

### 5.1 实施原则

| 原则 | 说明 | 应用 |
|------|------|------|
| 先连接后纠正 | 先建立关系再推动改变 | 每次互动 |
| 先听后说 | 先理解再表达 | 沟通原则 |
| 稳定一致 | 持续不因情绪变化 | 执行关键 |
| 允许反复 | 改变需要时间 | 耐心基础 |
| 关注正向 | 发现进步并强化 | 反馈原则 |

### 5.2 实施流程

#### 日常执行

```
每日流程：
┌────────────────────────────────────────────────┐
│ 早晨：确认当天计划                               │
│  ↓                                              │
│ 执行：按计划行动                                 │
│  ↓                                              │
│ 观察：记录孩子反应                              │
│  ↓                                              │
│ 调整：根据需要微调                              │
│  ↓                                              │
│ 晚间：简短回顾                                  │
└────────────────────────────────────────────────┘
```

#### 每周回顾
- 本周执行情况
- 孩子变化观察
- 需要调整的地方
- 下周计划

#### 每月评估
- 阶段性效果评估
- 方案调整
- 问题识别

### 5.3 实施记录模板


### 5.4 常见问题应对

| 问题 | 表现 | 应对策略 |
|------|------|----------|
| 孩子抵抗 | 不配合/哭闹 | 坚持但温和 |
| 老人干涉 | 做法不一致 | 沟通协调 |
| 配偶不配合 | 态度不一致 | 优先沟通 |
| 自己情绪失控 | 生气/后悔 | 暂停+修复 |
| 效果不明显 | 没有进展 | 评估+调整 |

## 六、阶段五：效果评估

### 6.1 评估维度

| 维度 | 指标 | 评估方法 |
|------|------|----------|
| 行为改变 | 问题行为是否减少 | 记录对比 |
| 关系改善 | 亲子互动是否改善 | 主观感受 |
| 功能提升 | 相关能力是否提升 | 观察评估 |
| 持续性 | 改变是否稳定 | 定期追踪 |

### 6.2 评估标准

#### 显著改善
- 问题行为减少≥50%
- 亲子关系明显改善
- 孩子状态积极

#### 部分改善
- 问题行为有所减少
- 有进步但不够明显
- 需要继续努力

#### 无改善
- 问题行为未减少
- 亲子关系未改善
- 需要重新诊断

#### 恶化
- 问题行为增加
- 亲子关系恶化
- 考虑转介

### 6.3 评估报告模板

```
【效果评估报告】

评估周期：___至___
评估时间：___

一、目标达成情况
- 目标1：___ 达成度：___
- 目标2：___ 达成度：___

二、指标变化
- 行为指标：___
- 关系指标：___
- 能力指标：___

三、方案执行情况
- 按计划执行：___%
- 关键调整：___

四、评估结论
- □ 显著改善 → 继续强化
- □ 部分改善 → 调整方案
- □ 无改善 → 重新诊断
- □ 恶化 → 考虑转介

五、下一步
- □ 继续当前方案
- □ 调整方案
- □ 重新诊断
- □ 转介
```

## 七、阶段六：迭代优化

### 7.1 优化时机

| 时机 | 触发条件 | 优化内容 |
|------|----------|----------|
| 定期优化 | 每月评估后 | 方案微调 |
| 效果不彰 | 无改善或恶化 | 方案重设 |
| 显著改善 | 达成目标 | 维持/扩展 |
| 环境变化 | 重大生活变化 | 方案适配 |

```
└────────────────────────────────────┘
                 ↓
┌────────────────────────────────────┐
│ Step 2: 识别调整点                 │
│ - 需要改变什么？                   │
│ - 可以保持什么？                   │
└────────────────────────────────────┘
                 ↓
┌────────────────────────────────────┐
│ Step 3: 设计优化方案               │
│ - 调整内容                         │
│ - 调整原因                         │

└────────────────────────────────────┘
                 ↓

│ Step 4: 实施优化方案               │
│ - 执行新方案                       │
```

## 八、特殊场景SOP

### 8.1 孩子情绪崩溃SOP

```
┌────────────────────────────────────────────────────────┐
│ Step 1: 保持冷静（自己的情绪管理）                      │
│ - 深呼吸                                                │
│ - 提醒自己这是孩子的问题，不是针对你                      │
│ - 避免被孩子的情绪带走                                   │
└────────────────────────────────────────────────────────┘
                           ↓
┌────────────────────────────────────────────────────────┐
│ Step 2: 提供安全空间                                    │
│ - 如果安全，让孩子有空间                                │
│ - 不在公共场合持续批评                                  │
│ - 告诉孩子"我在这里，等你准备好"                         │
└────────────────────────────────────────────────────────┘
                           ↓
┌────────────────────────────────────────────────────────┐
│ Step 3: 陪伴等待                                       │
│ - 在旁边但不强迫                                        │
│ - 不说话等待                                            │
│ - 准备好纸巾、水                                        │
└────────────────────────────────────────────────────────┘
                           ↓
┌────────────────────────────────────────────────────────┐
│ Step 4: 情绪确认                                       │
│ - "你哭得这么厉害，一定很伤心/生气"                      │
│ - "我能感觉到你很难过"                                  │
│ - 等待孩子平静                                          │
└────────────────────────────────────────────────────────┘
                           ↓
┌────────────────────────────────────────────────────────┐
│ Step 5: 连接修复                                       │
│ - 抱抱/拍拍                                             │
│ - "我在这里"                                           │
│ - "等你准备好了我们再聊"                                 │
└────────────────────────────────────────────────────────┘
                           ↓
┌────────────────────────────────────────────────────────┐
│ Step 6: 后续跟进                                       │
│ - 等完全平静后，轻声询问发生了什么                        │
│ - 倾听孩子                                              │
│ - 不急于给建议或教训                                    │
└────────────────────────────────────────────────────────┘
```

### 8.2 亲子冲突后修复SOP

```
┌────────────────────────────────────────────────────────┐

│ - 意识到冲突后，先暂停                                  │

└────────────────────────────────────────────────────────┘
                           ↓
┌────────────────────────────────────────────────────────┐

│ - 我做得好的是什么？                                     │
│ - 我可以改变什么？                                      │
└────────────────────────────────────────────────────────┘
                           ↓
┌────────────────────────────────────────────────────────┐
│ Step 3: 选择时机                                       │
│ - 等双方都平静                                          │
│ - 不是立即，而是在当天内                                │
│ - 创造私密安静的时光                                    │
└────────────────────────────────────────────────────────┘
                           ↓
┌────────────────────────────────────────────────────────┐
│ Step 4: 道歉与修复                                     │
│ - "刚才我对你___，我感到很抱歉"                         │
│ - "我不应该___"                                        │
│ - "我理解你可能有___的感受"                             │
│ - "我想要/我们可以___"                                  │
└────────────────────────────────────────────────────────┘
                           ↓
┌────────────────────────────────────────────────────────┐
│ Step 5: 肢体连接                                       │
│ - 牵手/拥抱                                            │
│ - 告诉孩子你爱他                                        │
│ - 重申你们的关系                                        │
└────────────────────────────────────────────────────────┘
                           ↓
┌────────────────────────────────────────────────────────┐
│ Step 6: 共同展望                                       │
│ - "下次我们可以___"                                     │

│ - 达成小小的约定                                        │

```
