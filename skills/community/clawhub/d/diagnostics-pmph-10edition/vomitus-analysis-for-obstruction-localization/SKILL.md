# 呕吐物性质对消化道梗阻定位的诊断价值

## 何时使用
- 患者有呕吐症状，且呕吐物可采集并肉眼观察；
- 临床高度怀疑为机械性消化道梗阻（如腹痛、腹胀、停止排便排气等伴随表现）；
- 需快速初步判断梗阻平面以指导后续影像学检查或内镜干预；
- **不适用于**无明确梗阻征象的呕吐（如妊娠反应、颅内压增高、药物副作用等）。

## 如何执行
1. **采集并观察呕吐物样本**，重点评估以下五项特征：
   - **气味**（发酵/腐败/粪臭/无特殊）；
   - **是否含胆汁**（是/否）；
   - **是否有酸味**（是/否）；
   - **颜色**（鲜红/暗红/咖啡色/棕褐色等）；
   - **是否含隔夜宿食**。

2. **根据特征组合进行定位判断**：
   - **带发酵或腐败气味** → 提示胃潴留；
   - **带粪臭味** → 提示低位小肠梗阻；
   - **不含胆汁** → 梗阻平面多在十二指肠乳头以上；
   - **含多量胆汁** → 梗阻平面在此以下；
   - **含大量酸性液体** → 考虑胃泌素瘤或十二指肠溃疡；
   - **无酸味** → 可能为贲门狭窄或贲门失弛缓症；
   - **呈咖啡色样** → 提示上消化道出血（血红蛋白经胃酸作用形成酸化正铁血红蛋白）；
   - **含隔夜宿食** → 见于幽门梗阻。

3. **综合判断**：
   - 必须结合呕吐发生时间、与进食的关系、伴随症状（如腹痛、腹胀、排便情况）及其他体征；
   - 避免仅凭单一指标做出结论，防止误判。

4. **输出结果**：
   - 给出最可能的梗阻部位或病因（如“十二指肠乳头以上梗阻”、“低位小肠梗阻”、“幽门梗阻伴胃潴留”等），用于指导下一步检查（如腹部X线、CT、胃镜）。
