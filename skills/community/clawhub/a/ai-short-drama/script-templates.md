# 📝 AI短剧剧本模板库

> 6大热门剧本类型 + 完整结构模板 + 写作技巧

---

## 一、剧本基础结构

### 短剧标准结构（每集1-3分钟）

```
第1集：黄金30秒
├── 0-5秒：钩子（反常识/冲突/悬念）
├── 5-15秒：背景交代（人物关系/核心矛盾）
├── 15-45秒：情节推进（第一个反转）
├── 45-60秒：小高潮 + 留悬念
└── 60-90秒：结尾钩子（引导看下一集）

第2-5集：节奏递进
├── 开头：回顾上集关键画面（3-5秒）
├── 中段：新冲突/新角色/新反转
├── 结尾：更大悬念或情感冲击

第6-8集（终章）
├── 所有线索汇聚
├── 最终反转
├── 情感高潮
└── 结局（开放/圆满/意难平）
```

### 爆款短剧的5个要素
1. **强冲突** — 开篇3秒内建立矛盾
2. **快节奏** — 每15秒一个信息点
3. **高频反转** — 每集至少1个意外
4. **情感共鸣** — 触发观众代入感
5. **悬念钩子** — 每集结尾让人想看下一集

## 二、6大剧本类型模板

### 类型1：都市甜宠 🌹

**核心公式**: 误会 → 接近 → 心动 → 误会 → 甜蜜

**人物设定模板**:
```
女主角：
- 身份：普通但有个性的职业（设计师/烘焙师/摄影师）
- 特点：善良但不圣母，有自己的底线
- 秘密：隐藏的身世/才能/过去

男主角：
- 身份：表面高冷实则暖心的"霸总"型
- 特点：对外冷酷，对女主温柔
- 弱点：某件往事造成的心理创伤

配角：
- 闺蜜/兄弟（推动剧情+搞笑担当）
- 反派（情敌/职场对手/家族成员）
```

**8集剧本大纲模板**:
```
EP1【相遇】：女主在重要场合出糗，男主恰好目睹
EP2【再遇】：被迫合作/同居/假扮情侣
EP3【心动】：意外发现对方的另一面
EP4【误会】：第三者介入或信息不对称
EP5【分离】：误会加深，关系破裂
EP6【真相】：配角助攻，真相大白
EP7【追回】：高调示爱/挽回名场面
EP8【甜蜜】：在一起+番外彩蛋
```

**爆款开头模板**:
```
"你被开除了。"
"可是这个方案是我熬了三个通宵..."
"我不看过程，只看结果。收拾东西走吧。"
（镜头切换：她不知道的是，他口袋里装着那份方案——上面盖着'特批通过'的章）
```

### 类型2：悬疑推理 🔍

**核心公式**: 异常事件 → 调查 → 线索 → 误导 → 真相

**人物设定模板**:
```
主角：
- 身份：侦探/记者/受害者家属/意外卷入的普通人
- 特点：观察力敏锐，有执念
- 弱点：过去的某个失败案例

案件结构：
- 表象：一个看似简单的案件
- 线索：5-8个真真假假的线索
- 干扰：1-2个误导性嫌疑人
- 真相：意料之外情理之中
```

**8集剧本大纲模板**:
```
EP1【案发】：一个不寻常的事件打破平静
EP2【调查】：主角开始追查，发现第一个异常
EP3【深入】：线索指向一个出人意料的人
EP4【误导】：看似找到了凶手/真相
EP5【反转】：发现之前的推理有致命漏洞
EP6【隐秘】：挖掘出隐藏的关系/动机
EP7【对决】：与真凶正面对峙
EP8【真相】：完整还原事件经过+隐藏结局
```

**爆款开头模板**:
```
"监控显示，那个房间里只有她一个人。"
"但法医报告说，她身上有两种不同力道的伤痕。"
"一种在死前。一种在死后。"
"——谁会在一个死人身上再动手？"
```

### 类型3：古风仙侠 ⚔️

**核心公式**: 宿命 → 相遇 → 羁绊 → 考验 → 涅槃

**人物设定模板**:
```
女主：
- 身份：隐藏身份的神族/妖族/凡人天选之女
- 特点：外表柔弱内心坚韧
- 能力：觉醒后拥有特殊力量

男主：
- 身份：高阶仙人/魔族王子/隐世高手
- 特点：背负使命，冷面热心
- 禁忌：不能动情/有寿命限制

世界观：
- 三界设定（天界/人界/魔界）
- 核心规则（如：仙凡不能相恋）
- 大事件背景（即将到来的浩劫）
```

**8集剧本大纲模板**:
```
EP1【初遇】：凡间意外邂逅，命运齿轮开始转动
EP2【羁绊】：被迫同行，发现对方的秘密
EP3【心动】：生死关头的守护，情愫暗生
EP4【禁忌】：身份暴露，面临天条/世俗压力
EP5【分离】：为保护对方选择离开/被拆散
EP6【磨难】：各自身处险境，经历成长
EP7【涅槃】：在绝境中觉醒/突破
EP8【归一】：战胜宿命，在一起或意难平
```

**爆款开头模板**:
```
（画面：漫天飞雪中，一袭白衣立于悬崖）
"三百年前，他亲手将她推下诛仙台。"
"三百年后，她在人间再次遇见了他。"
"——只是这一次，他什么都不记得了。"
```

### 类型4：职场逆袭 💼

**核心公式**: 被压制 → 隐忍 → 机会 → 爆发 → 登顶

**人物设定模板**:
```
主角：
- 起始：被看不起的底层员工/被排挤的新人
- 隐藏能力：过人的专业能力/特殊背景/天才头脑
- 信念：不甘平庸，有明确目标

反派：
- 类型：抢功的上司/富二代同事/行业大佬
- 特点：表面光鲜实则能力不足
- 弱点：过度依赖权力/人脉，忽视真才实学

关键NPC：
- 赏识主角的贵人（老板/客户/前辈）
- 背叛者（先友好后背刺）
```

**8集剧本大纲模板**:
```
EP1【屈辱】：公开场合被羞辱/抢功/陷害
EP2【隐忍】：表面认命，暗中积蓄力量
EP3【转机】：一个无人敢接的难题/项目出现
EP4【暗战】：主角暗中发力，初露锋芒
EP5【危机】：反派发现威胁，开始打压
EP6【底牌】：关键时刻亮出隐藏能力/身份
EP7【反转】：用实力碾压，打脸名场面
EP8【登顶】：获得认可+更大舞台+给续集留扣
```

**爆款开头模板**:
```
"这个方案，是我做的。"
"小王啊，你一个实习生做的方案，能有什么水平？" （领导当众嘲笑）
"那就让客户来评吧。"
（镜头切：客户看完方案，直接拍板——"这个项目，我只认这个人。"）
```

### 类型5：科幻未来 🚀

**核心公式**: 异常 → 探索 → 发现 → 抉择 → 新世界

**人物设定模板**:
```
主角：
- 身份：科学家/普通市民/AI觉醒体
- 特点：好奇心驱动，不盲从权威
- 困境：发现了一个被隐藏的真相

世界观：
- 时间：近未来（2050-2100）或远未来
- 核心科技：AI/脑机接口/时间旅行/平行宇宙
- 社会矛盾：技术带来的阶级分化/伦理困境
```

**8集剧本大纲模板**:
```
EP1【异常】：日常生活中的一个不寻常现象
EP2【探索】：开始调查，发现更多疑点
EP3【发现】：找到关键证据/进入未知区域
EP4【真相】：世界/身份的真相被揭开
EP5【抉择】：面临重大选择（红药丸/蓝药丸）
EP6【对抗】：与维持旧秩序的力量抗争
EP7【牺牲】：付出代价换取突破
EP8【新世界】：旧秩序崩塌/新世界开启
```

**爆款开头模板**:
```
"你有没有觉得，今天的太阳...和昨天的不太一样？"
"你在说什么？太阳每天都一样。"
"不。昨天的太阳，从东边升起。"
"今天也是。"
"——但你记得昨天的事吗？所有人都不记得了。只有我。"
```

### 类型6：恐怖惊悚 👻

**核心公式**: 异象 → 恐惧 → 逃避 → 面对 → 真相

**人物设定模板**:
```
主角：
- 身份：搬新家的普通人/夜班工作者/探险者
- 特点：理性务实，不信邪
- 弱点：某段不愿提起的过去

恐怖元素（选1-2个）：
- 空间类：密室/废弃建筑/荒村/镜子
- 时间类：循环/时间错位/倒计时
- 心理类：幻觉/记忆篡改/分身
- 实体类：鬼魂/怪物/未知生物
```

**8集剧本大纲模板**:
```
EP1【异象】：细微但不安的异常现象
EP2【升级】：无法忽视的恐怖事件
EP3【调查】：试图找到合理解释
EP4【无路】：发现逃不掉/外面更危险
EP5【真相初现】：恐怖源头露出端倪
EP6【规则】：发现生存规则/规律
EP7【对抗】：利用规则与恐怖力量正面对抗
EP8【结局】：逃出/同归于尽/开放式恐怖结局
```

**爆款开头模板**:
```
（画面：深夜，手机相册自动翻到一张照片）
"这张照片不是我拍的。"
"但照片里的人——是我。"
"而且我穿着...明天才买的衣服。"
（手机屏幕闪了一下，照片消失了。但那张脸——她永远记得）
```

## 三、黄金3秒开头公式

### 公式1：反常识型
```
"死刑犯的最后愿望是——嫁给法官。"
```

### 公式2：悬念型
```
"她收到一封信，寄件日期是——明天。"
```

### 公式3：冲突型
```
"婆婆跪下说：求你跟我儿子离婚。我说：阿姨，是你儿子不配。"
```

### 公式4：反转型
```
"所有人都以为她是受害者。包括那个真正的凶手。"
```

### 公式5：情感冲击型
```
"他用了十年，学会了忘记她。然后她在婚礼上出现了——不是新娘。"
```

## 四、角色对白技巧

### 短剧对白原则
1. **一句一信息** — 每句话推进一个信息点
2. **少用形容词** — 用动作/表情替代
3. **潜台词** — 说出口的只是冰山一角
4. **金句** — 每集至少1句可传播的台词

### 金句模板
```
都市甜宠："我不是不喜欢你，我是怕太喜欢你。"
悬疑推理："真相不会消失，它只是在等一个敢找它的人。"
古风仙侠："三界之大，却没有一个地方能让我不去想你。"
职场逆袭："你们笑我没背景。但你们不知道，我背后的伤疤就是我的背景。"
科幻未来："如果记忆可以被删除，那爱也可以吗？"
恐怖惊悚："最可怕的不是鬼。是确定这间房里只有你一个人的时候——你听到了第二个人的呼吸声。"
```
