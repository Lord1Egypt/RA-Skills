## Description: <br>
Work inside an existing R project to extend analyses, modify scripts, run statistical models, update visualizations, and regenerate reports. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[JackKuo666](https://clawhub.ai/user/JackKuo666) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers, analysts, and data scientists use this skill to continue work in existing R projects by extending statistical analyses, updating visualizations, debugging R code, and regenerating R Markdown or Quarto reports. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may edit R project files, regenerate reports, or overwrite analysis outputs. <br>
Mitigation: Use version control or backups and review diffs before accepting changes. <br>
Risk: Running project R code or rendering reports from untrusted sources can execute unsafe code or install packages. <br>
Mitigation: Run only trusted projects in an isolated environment and review package installation requests before proceeding. <br>
