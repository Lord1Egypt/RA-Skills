# 编程启蒙项目模板

适合中学生的第一个项目清单，按难度和类型分类。

---

## 🎮 游戏类项目

### Level 1: 入门级

#### 1. 猜数字游戏 ⭐
**学到的概念：** 变量、条件判断、循环、随机数
**时间：** 30-60分钟
**文件：** `scripts/guess_number_game.py`

#### 2. 石头剪刀布 ⭐
**学到的概念：** 变量、条件判断、用户输入
**时间：** 20-40分钟
**扩展：** 添加计分系统

#### 3. 文字冒险游戏 ⭐⭐
**学到的概念：** 条件判断、字符串、多分支逻辑
**时间：** 40-60分钟
**扩展：** 添加更多分支和结局

### Level 2: 进阶级

#### 4. 猜词游戏 (Hangman) ⭐⭐
**学到的概念：** 列表、字符串处理、循环
**时间：** 60-90分钟

#### 5. 21点游戏 ⭐⭐
**学到的概念：** 随机数、条件判断、累加逻辑
**时间：** 45-60分钟

#### 6. 文字RPG游戏 ⭐⭐⭐
**学到的概念：** 字典、函数、类（可选）
**时间：** 2-3小时

## 项目选择建议

### 根据兴趣
- 喜欢游戏 → 从猜数字开始，进阶到文字RPG
- 喜欢聊天 → 从聊天机器人开始，进阶到智能问答
- 喜欢实用工具 → 从计算器开始，进阶到自动化脚本
- 喜欢数据 → 从成绩计算开始，进阶到数据分析

### 根据时间
- 10-20分钟 → 密码生成器、单位转换器
- 30-45分钟 → 计算器、石头剪刀布
- 1-2小时 → 聊天机器人、待办事项
- 2小时以上 → 文字RPG、数据可视化

### 根据难度
- ⭐ 零基础可做
- ⭐⭐ 需要一定基础
- ⭐⭐⭐ 需要较多练习

## 项目组合建议

### 组合A：游戏开发者之路
```
猜数字 → 石头剪刀布 → 猜词游戏 → 文字RPG
```
约 4-6 小时，覆盖大部分基础概念

### 组合B：工具制作者之路
```
计算器 → 密码生成器 → 待办事项 → 自动化脚本
```
约 3-5 小时，偏实用方向

### 组合C：聊天达人之路
```
问答机器人 → 聊天机器人 → 日记本 → 通讯录
```
约 3-4 小时，偏社交互动

## 完成一个项目后

1. ✅ 运行成功 → 拍照/截图记录
2. 🔧 尝试修改 → 添加新功能
3. 📝 写注释 → 解释每行代码的作用
4. 🎯 选下一个 → 继续挑战

记住：做出来的东西 > 完美的设计！
