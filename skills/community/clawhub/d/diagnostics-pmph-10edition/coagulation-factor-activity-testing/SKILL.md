# 凝血因子活性检测临床应用

## 适用场景
- 患者有出血倾向、瘀斑、关节出血或术后出血不止
- 家族中有遗传性出血病史
- PT 或 APTT 延长，需明确具体凝血因子缺陷

## 执行步骤

### 一、基于筛检试验选择检测项目
- **APTT延长** → 测定 FVIII:C、FIX:C、FXI:C、FXII:C
- **PT延长** → 测定 FII:C、FV:C、FVII:C、FX:C

### 二、结果解读
1. **活性减低**：
   - 先天性缺乏：如血友病A（FVIII:C↓）、血友病B（FIX:C↓）
   - 获得性缺乏：肝病、维生素K缺乏、DIC、抗凝治疗
   - 特别注意：FVIII:C降低也可见于血管性血友病（vWD）或存在FVIII抑制物
2. **活性增高**：见于血栓前状态、妊娠、恶性肿瘤等
3. **无出血风险的异常**：FXII:C降低虽致APTT延长，但通常不引起临床出血

### 三、后续建议
- 根据具体因子缺乏情况，安排基因检测、vWF抗原/活性测定或抑制物筛查
- 对获得性缺乏，需评估肝功能、营养状态及凝血消耗标志物（如D-二聚体、纤维蛋白原）

## 注意事项
- 不适用于单纯血小板减少或血管性出血的初步评估
- 所有F:C结果需结合临床表现和其他凝血功能指标综合判断
