## 代码映射

```javascript
const GANSU_CITIES = [
  '兰州', '嘉峪关', '金昌', '白银', '天水', 
  '武威', '张掖', '平凉', '酒泉', '庆阳', 
  '定西', '陇南', '临夏', '甘南'
];

function cityOf(areaName) {
  for (const city of GANSU_CITIES) {
    if (areaName && areaName.includes(city)) return city;
  }
  return areaName || '未知';
}
```

## 经济特点

- **兰州**: 政治经济中心，项目最多
- **天水**: 工业基础好，监控项目多
- **酒泉**: 新能源项目多
- **庆阳**: 石油相关项目
