# P波正常参数判定与心房肥大心电图诊断

## 适用条件
- 标准12导联心电图记录
- 成人静息状态
- 排除显著心律失常、起搏器干扰、急性心包炎、肺栓塞等非肥厚性病因

## 执行流程

### 第一步：判断P波是否正常
1. **形态**：P波在Ⅰ、Ⅱ、aVF、V₄～V₆导联应向上，呈钝圆形（可有轻度切迹）；aVR导联向下；其余导联可双向、倒置或低平。
2. **时限**：P波时间 < 0.12秒。
3. **振幅**：
   - 肢体导联 < 0.25 mV
   - 胸导联 < 0.20 mV

若任一参数超出范围，进入第二步。

### 第二步：评估心房肥大类型
1. **右心房肥大（“肺型P波”）**：
   - Ⅱ、Ⅲ、aVF导联P波振幅 ≥ 0.25 mV
   - V₁导联直立P波 ≥ 0.15 mV，或双向P波正负振幅和 ≥ 0.20 mV
   - P波电轴 > +75°

2. **左心房肥大（“二尖瓣型P波”）**：
   - P波时限 ≥ 0.12秒，常呈双峰（峰间距 ≥ 0.04秒）
   - V₁导联P波终末电势（Ptf_V1）绝对值 ≥ 0.04 mm·s

3. **双心房肥大**：
   - P波时限 ≥ 0.12秒 且 振幅 ≥ 0.25 mV
   - V₁导联P波高大双相，上下振幅均超正常

### 第三步：输出结果
- 若所有参数正常，返回“P波正常”
- 若存在异常，提示“可能存在右/左/双心房肥大”，并注明依据
- 强调需结合临床及其他检查综合判断，不可单独确诊

> 注意：P波异常亦可见于心房内传导阻滞、心房负荷增加等情况，无特异性病因诊断价值。
