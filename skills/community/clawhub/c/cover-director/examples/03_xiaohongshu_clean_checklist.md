# 示例 03：小红书干货收藏封面

## 输入

- 标题：小红书新手起号前，先把这 7 个设置改好
- 平台：小红书
- 比例：3:4
- 领域：自媒体运营
- 目标：收藏
- 风格：干净、高级、笔记感

## 输出摘要

### 标题策略

- 类型：干货型 + 避坑型
- 最大关键词：7个设置
- 辅助信息：新手起号前先改好
- 情绪：亲和、认真
- 手势：拿手机展示 / 指向清单

### 构图

浅色背景，左上大标题，右下人物拿手机或手机界面卡片。整体更干净，不使用大红大黄爆炸贴。

### 无字底图提示词

```text
生成一张小红书 3:4 无字封面底图。画面干净高级，浅米白或浅粉白背景，真实女性博主半身出镜，表情亲和认真，一手拿手机，另一只手自然指向左上方标题留白。右下有简洁手机界面卡片占位，但不出现文字。柔和自然光，生活方式摄影质感，高清清晰，适合后期添加小红书干货标题。不要文字、水印、logo、杂乱背景。
```

### 排版

- 最大词：“7个设置”
- 辅助：“新手起号前 先改好”
- 字体：圆润粗黑体，颜色：深咖/黑 + 小面积玫粉强调。

### 质检预估

- 评分：90/100
