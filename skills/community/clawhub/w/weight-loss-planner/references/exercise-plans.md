# 运动计划模板库

## 运动水平判定标准

| 水平 | 适用人群 | 判定条件 |
|------|----------|----------|
| 初级 | 几乎不运动，BMI>28，久坐工作者 | 之前无规律运动习惯 |
| 中级 | 偶尔运动，BMI 24-28，有一定基础 | 每周1-2次运动经验 |
| 高级 | 经常运动，BMI<24，运动习惯良好 | 每周3次以上运动经验 |

## 力量训练模板

### 30分钟版

#### 初级
```
热身：5分钟动态拉伸
主训练(每个动作3组×12次，组间休息60秒)：
  1. 深蹲(自重)
  2. 俯卧撑(膝盖俯卧撑变体)
  3. 弓步蹲(自重)
  4. 平板支撑(30秒×3组)
  5. 臀桥
放松：5分钟静态拉伸
```

#### 中级
```
热身：5分钟动态拉伸
主训练(每个动作4组×10次，组间休息45秒)：
  1. 深蹲(负重/哑铃)
  2. 俯卧撑(标准)
  3. 弓步蹲(负重)
  4. 哑铃划船
  5. 平板支撑(45秒×3组)
放松：5分钟静态拉伸
```

#### 高级
```
热身：5分钟动态拉伸
主训练(每个动作4组×8-10次，组间休息30秒)：
  1. 深蹲跳
  2. 俯卧撑(窄距/钻石)
  3. 保加利亚分腿蹲
  4. 哑铃划船+肩推组合
  5. 平板支撑(60秒×3组 + 侧平板各30秒)
放松：5分钟深度拉伸
```

## 特殊人群注意事项

### 大体重人群(BMI>30)
- ❌ 避免：跑步、跳绳、高冲击跳跃
- ✅ 推荐：游泳、椭圆机、快走、骑车、瑜伽
- 力量训练以自重为主，避免大负重深蹲
- 运动心率控制在120-140区间

### 中年人群(40+)
- 每次训练前增加热身时间至8-10分钟
- 重视关节活动度训练
- 力量训练侧重功能性动作
- 避免极端HIIT，以中等恒定有氧为主

### 久坐办公族
- 增加髋关节和肩部灵活性训练
- 重点加强核心和背部力量
- 每30分钟坐姿后起身活动1-2分钟
- 推荐瑜伽+力量的组合

## 运动通用原则

1. **热身必做**：任何训练前5-8分钟热身，预防受伤
2. **拉伸必做**：训练后5-7分钟拉伸，促进恢复
3. **循序渐进**：每1-2周增加5-10%的训练量
4. **心率监控**：减脂最佳心率区间 = (220-年龄) × 60%-70%
5. **组间休息**：力量训练组间30-60秒，有氧不间断
6. **水分补充**：运动前、中、后均需补水
7. **训练多样性**：避免长期重复同一方案，每4周调整一次
8. **身体信号**：出现疼痛(非肌肉酸痛)立即停止
