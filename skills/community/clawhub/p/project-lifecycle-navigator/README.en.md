# Project Lifecycle Navigator

A bilingual English/Chinese instruction-only Skill for non-technical users, founders, operators, PMs, and AI Coding Agent workflows.

It routes the conversation into one of three project stages:

1. **New Project Intake**: turn a vague idea into an MVP scope, project plan, technical recommendation, and development execution plan.
2. **Mid-Project Realignment**: when a project feels off-track, bloated, confusing, or overloaded with new ideas, recalibrate direction and decide whether to continue, narrow, pivot, archive, or restart.
3. **Code Review & Upgrade Plan**: when code already exists, review security, dead code, performance, architecture, testability, dependencies, and generate an executable upgrade plan.

## Who it is for

- Non-programmers building software or AI tools
- People using AI Coding Agents
- Founders, operators, PMs, and business teams
- Teams that need mid-project direction control
- Users who need structured code review and refactoring plans

## Example triggers

- “I want to build a new project. Ask me questions first.”
- “The project is halfway done and feels off-track. Help me recalibrate.”
- “I have a codebase. Review it for vulnerabilities, dead code, and refactoring opportunities.”

## Publishing note

This is an instruction-only Skill. It requires no third-party packages, API keys, network access, binaries, install scripts, or code execution.
