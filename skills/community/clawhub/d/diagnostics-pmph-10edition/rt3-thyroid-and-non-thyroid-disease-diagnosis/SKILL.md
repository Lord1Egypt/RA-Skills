# rT₃在甲状腺功能与非甲状腺疾病中的诊断价值

## 何时使用
- 患者已测定rT₃、T₃、T₄，且存在以下情况之一：
  - 疑似或确诊甲亢/甲减
  - 患有重症非甲状腺疾病（如AMI、肝硬化、糖尿病酮症、心力衰竭等）
  - 正在接受甲状腺激素替代或抗甲状腺药物治疗
  - 老年患者出现甲状腺功能指标异常

> ⚠️ 注意：不得孤立解读rT₃结果，必须结合T₃、T₄及完整临床背景。

## 执行步骤

1. **确认前提条件**
   - 已获取rT₃、T₃、T₄的实验室结果
   - 明确患者基础疾病、用药史（如普萘洛尔、地塞米松、丙硫嘧啶等）及年龄

2. **判断rT₃变化方向**
   - 若rT₃ > 0.8 nmol/L → 进入“rT₃增高”分析路径
   - 若rT₃ < 0.2 nmol/L → 进入“rT₃减低”分析路径

3. **结合T₃、T₄与临床背景综合解读**
   - **rT₃增高时**：
     - 若T₃↑、T₄↑ → 支持甲亢诊断
     - 若T₃↓或正常、T₄正常/↓，且存在重症 → 考虑非甲状腺疾病综合征（NTIS）
     - 若正在接受甲状腺激素治疗且T₃↑、rT₃↑、T₄正常/↑ → 提示用药过量
   - **rT₃减低时**：
     - 若T₃正常或轻度↓、T₄↓ → 强烈提示甲减，尤其适用于亚临床或轻型病例
     - 若正在使用抗甲状腺药物且rT₃↓、T₄↓ → 警惕药物过量

4. **输出结论**
   - 明确区分甲亢、甲减与非甲状腺疾病所致的rT₃异常
   - 对治疗调整或进一步检查提出建议

> 参考区间：rT₃ 0.2～0.8 nmol/L；T₃ 1.6～3.0 nmol/L；T₄ 65～155 nmol/L
