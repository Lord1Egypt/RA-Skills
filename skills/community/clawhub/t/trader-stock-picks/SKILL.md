# 操盘手选股逻辑

## 核心理念

本技能模拟顶尖操盘手的选股视角，寻找正在经历**吸筹→拉升→洗盘→主升→派发**五个阶段的A股标的。
核心前提：**不需要预测市场，只需要识别主力痕迹，理解主力心理。**

## 第零维度：操盘手心法（阅读前必读）

> **在进入任何技术指标之前，先戴上操盘手的脑子。**

### 心法零：承认错误的勇气——大唐发电7连板复盘（关键更新 2026-05-15）

> ⚠️ **本节是「学费用教训写成的追加维度」——所有使用本技能的人必须读完。**

**事实：** 2026年5月8日，大唐发电三连板后，我基于本技能框架给出了「三连板+融资降=主力出货，短期危险」的判断。结果大唐发电继续上涨至**7连板**，市值重回1000亿。**我每一步预判都被事实打脸，而用户每次反方向判断都是对的。**

这堂课教我：**一个框架如果无法承认自己被驳倒，它就只是个偏见。**

### 核心教训一：不要把「市场合力」当成「庄股出货」

本技能最初是围绕「识别单一主力操盘痕迹」设计的。但**大唐发电不是庄股**——它是：
- 全国首个大规模「算电协同」绿电直供项目落地 → 估值逻辑重构
- AI数据中心 + 东数西算 国家级叙事 → 全市场追逐
- 净利润从2022年亏损4亿 → 2025年74亿（增长18倍）→ 基本面质变
- 机构集中重仓、研报统一覆盖 → 多路资金合力

**当一个股票有「估值重构」级别的催化剂时，技术面的「出货信号」会被基本面资金碾压。** 我的错误是把一个**估值重估**的票，用**存量筹码博弈**的框架去分析。

### 心法一：三张表思维

操盘手看股票不是看K线，而是看三张表：
1. **自己的底牌**：已建仓多少、成本多少、浮盈多少
2. **市场的底牌**：还有多少浮筹在外面、融资盘多少、对手盘是谁
3. **散户的底牌**：他们在什么位置买的、什么位置会卖、什么位置会追

**核心问题永远只有一个：**「如果我手里有10亿要买/要卖，现在这个盘面，这些散户和机构会怎么做？」

### 心法二：融资余额是照妖镜

融资余额（杠杆资金）是市场上最诚实的指标之一：

| 现象 | 真相 |
|:----|:----|
| 股价横盘 + 融资余额持续下降 | 洗盘尾声！杠杆散户扛不住割肉了，主力在接 |
| 股价上涨 + 融资余额持续上升 | 主升浪中，散户加杠杆追，主力边拉边观察 |
| **股价上涨 + 融资余额反而下降** | ⚠️ **需要分情形判断（见下方修正）** |
| 融资余额/流通市值 > 3% | 杠杆过高，一旦下跌会踩踏 |

> **⚡ 重要修正：融资余额在连续一字板中的解读完全不同（大唐发电教训）**
>
> 连续涨停缩量一字板中融资余额下降 **≠ 主力出货**。原因：每日成交量几乎为零，没人能做任何交易包括主力。融资余额下降只是因为**没人能买入**，融资买入额自然为零，本金正常回流。这时融资降是「无交易」的结果，不是「出货」的信号。
>
> **区分规则：**
> - **放量涨停**中融资降（日成交30亿+） → ⚠️ 警示信号，主力可能倒货
> - **缩量一字板/缩量涨停**中融资降（日成交<3亿） → 技术性现象，不构成卖出信号
> - 连板中途开板放量再封板中融资降 → 需结合盘口逐笔看，有一定警示意义
>
> 大唐发电：首板66亿→后两板快速缩量→融资降是机械现象，不反映主力意图。

### 核心教训二：三连板之后不一定是出货——区分「出货板」和「加速板」

一个历史性的错误：框架教条「三板出货」。前提是：_三板是行情尾声，已完成从吸筹到派发的完整周期。_

但当出现以下条件时，**三板不是尾声，而是加速的中途：**

| 条件 | 出货板（危险） | 加速板（可继续） |
|:----:|:--------------:|:----------------:|
| 起涨前涨幅 | 已经涨了30-50%+ | 刚刚突破长期横盘（一年没涨） |
| 催化剂量级 | 普通题材，情绪驱动 | **全国首个/国家级/政策强驱动** |
| 基本面趋势 | 利润持平或下滑 | 利润连续2年翻倍+ |
| 成交量特征 | 三板天量（单日>流通市值10%） | 三板中后段缩量锁仓 |
| 机构覆盖 | 无研报或全中性 | 全部买入+密集上调目标价 |
| 板块效应 | 独舞，板块不跟 | 板块集体爆发，龙头效应 |

> **大唐发电：** 6项中符合5项「加速板」条件。但框架当时没提供这个检查清单。

### 核心教训三：每日动态重评机制——让你的预判被事实修正

原技能是「一次性打分」的静态框架。但操盘手每天都在问：「昨天我的判断对不对？不对我该调什么？」

**每日重评清单：**
1. 昨天走势**反驳**了我之前的哪条判断？
2. 如果反转了，我认错还是死撑？
3. 新的信息（量/价/融资/新闻/板块）改变了什么？
4. 我的判断有「自我强化」偏见吗？（持股则看多，没持股则看空）
5. 如果让我空仓重新看这个票，我的结论是什么？

> 这个机制不是「更加谨慎地预测」，而是**承认预测本身就是高风险活动**。大唐发电的案例表明：当无法判断时，说「我不知道，我之前的框架在这里失效了，新信号需要时间分辨」比给出一个高确信度的错误判断更好。

### 心法三：量价合一的读法

普通投资者看量价关系只看「放量上涨=好」，操盘手看的是：
- **缩量横盘** = 主力控盘度高，没人敢卖也没人想买（吸筹早期）
- **放量滞涨** = 主力在出货（顶部分散信号）
- **放量急跌** = 最后一跌，恐慌盘出来了（最佳买点）
- **温和放量上涨** = 拉升期，主力在测试上方抛压
- **天量开板** = 三连板后66亿成交额但主力只买了1% = 剩下的99%是谁在买？散户。谁在卖？不言自明。

### 心法四：猜操盘手的下一步

操盘手不是神，他的每一步都有逻辑制约：
| 他当前状态 | 他的困境 | 他下一步最可能做什么 |
|:---------:|:--------:|:------------------:|
| 刚完成吸筹 | 手上大量廉价筹码，需要出货空间 | 拉升20-30%，吸引市场注意 |
| 拉升到高位 | 浮盈很大，但卖不掉因为没人接 | 造势！让全市场都知道这支票有故事 |
| 三连板后 | 已经卖掉30%仓位，还有70%要出 | **三种路线**：如果周末有消息继续拉（路线B），如果消息真空就震荡慢慢倒（路线A），如果全是利空直接抢跑（路线C）|
| 派发过半 | 剩余筹码不多，不想再压盘了 | 最后一拉做出货，把剩余清掉 |

### 心法五：跟着亏损走

**散户最痛苦的地方，就是主力最贪婪的地方。**

观察股吧/雪球情绪：
- 骂声一片，看空成风，人人割肉 → ⭐ **绝佳买入时机**（主力默默收筹）
- 分歧加大，多空交战 → **无聊的横盘期**（主力在洗盘）
- 一致看多，股神遍地，都在问「还能追吗」 → **⚠️ 警惕！顶部区域**（主力准备倒货）
- 狂欢巅峰，涨停板排队，融资创新高 → **🔴 危险！正在出货**

> **操盘手的真理：** 当90%的人认为还要涨的时候，操盘手已经在挂卖单了。当90%的人认为这支票没救了，操盘手正在分批吃进。

## 选股五维模型

### 维度一：筹码结构（权重25%）
- **流通市值**：偏好50-500亿的中盘股，市值太小流动性差，太大不易操纵
- **机构持仓占比**：30-70%为佳。太高(>80%)筹码锁定但难拉；太低(<20%)散户化严重
- **股东人数变化**：连续2-3季度减少 → 筹码在集中，好事
- **十大流通股东变化**：是否有知名牛散/私募新进

### 维度二：量价形态（权重25%）
- **长期横盘**：股价在10-20%区间内震荡≥3个月
- **地量见底**：成交量萎缩至历史低位的30%以下
- **温和放量**：近期日均成交量较前期放大50-100%，但股价涨幅<10%
- **涨停基因**：过去1年有过涨停经历（证明股性活跃）

### 维度三：资金流向（权重20%）
- **主力资金**：近20日主力净流入为正值，且呈递增趋势
- **融资余额**：处于阶段性低点（洗盘结束信号）或刚开始回升（启动信号）
- **大单占比**：大单买入占比从<10%提升至15-20%
- **游资活跃度**：龙虎榜是否有知名游资席位介入

### 维度四：基本面催化剂（权重15%）
- **业绩拐点**：最近1-2季度净利润同比增速转正或加速
- **低估值**：PE<15倍，PB<行业平均
- **政策催化**：所属行业有即将落地的政策利好
- **事件驱动**：分红提升、资产注入、股权激励、回购等

### 维度五：技术面形态（权重15%）
- **均线系统**：5日/10日/20日均线粘合后开始多头排列
- **MACD**：在零轴附近金叉或底背离
- **KDJ/R**：超卖区域反弹（J值<20反弹至>50）
- **箱体突破**：放量突破前期箱体上沿

### 维度六：市场合力与估值重构信号（权重——紧急追加，大唐发电教训总结）

> 🚨 **这个维度在本技能原版本中不存在。正是它的缺失导致了大唐发电判断的全面错误。**

#### 6.1 先判断场景：庄股 vs 估值重估

分析任何标的之前，第一件事不是评分，而是**分类**：

| 特征 | 庄股模式（原框架适用） | 估值重估模式（新框架） |
|:----:|:---------------------:|:---------------------:|
| 催化剂 | 消息面驱动、概念炒作 | **基本面质变**（业绩拐点+政策拐点+行业拐点三重共振） |
| 资金性质 | 单一主力或游资 | **机构+游资+北向多路合力** |
| 估值分位 | 已在高位或合理 | **极度低估（PB/PEG处于历史底部）**，有重估空间 |
| 市值 | 50-500亿（中盘） | 可大可小，千亿级亦可 |
| 上涨持续性 | 脉冲式，1-2周 | **趋势性，可持续数周至数月** |
| 机构研报 | 无或少 | 全线覆盖+密集上调目标价 |
| 板块效应 | 个股独立行情 | 板块共振+龙头溢价 |
| 散户情绪 | 分歧大 | 从质疑→怀疑→认可→疯狂 逐步递进 |

> **当6项以上指向「估值重估模式」时，立即切换分析框架。此时「涨停后融资降=出货」「三连板后必回调」等庄股经验不适用。**

#### 6.2 估值重估模式下的特殊信号

| 信号 | 含义 |
|:----|:----|
| 连板中缩量锁仓 | 非「没有抛压」，而是「没人愿意卖」——共识形成中 |
| 融资余额在缩量涨停中下降 | 机械现象，无信号意义（见心法二修正） |
| 研报目标价远高于当前价 | 机构锚定效应——还有空间 |
| 板块内跟风股开始发散 | 龙头溢价形成，量能仍在扩散期 |
| 第4-5板出现分歧放量后缩量封回 | ⭐ **关键买点信号**（分歧转一致） |
| 第6板后出现巨量换手 | ⚠️ 可能接近尾部，开始警惕 |

#### 6.3 估值重估行情的基本标志

> **当一个股票的催化剂是「全国首个」「行业首个」「历史首次」级别的落地事件，且基本面已经三个季度以上验证了拐点——你遇到的是估值重构，不是一个普通的庄股拉升。**

估值重估行情的判断流程：
1. **催化剂量级鉴定**：是普通题材（3分）还是国家级战略（10分）？
2. **业绩验证**：利润增速是否连续2年以上加速？
3. **市场验证**：是否有5家以上知名机构开始覆盖/上调？
4. **历史对标**：同行业是否有过类似「老公司+新故事」涨3-5倍的先例？
5. **散户普及率**：90%散户还不知道这个逻辑 → 行情还早；90%散户都在问能不能追 → 警惕

## 每日选股流程

### 步骤0：场景分类（新增——大唐发电教训）

在对任何标的打分之前，先回答：**这只票是「庄股模式」还是「估值重估模式」？**

对照维度六6.1表格，如果≥6项指向估值重估：
- ⛔ 禁用量价/资金维度的庄股规则判断买卖点
- ⭐ 启用估值重估框架：聚焦催化剂持续性、机构目标价空间、散户普及率

> **唐发电当时如果做了这个分类，就不会把加速板当成出货板。**

### 步骤1：粗筛（数据源扫描）
扫描A股以下板块标的：
- **电力/公用事业**（火电、绿电、水电）
- **央企改革**（低估值央企）
- **周期复苏**（化工、有色、煤炭）
- **地方国资**（小市值地方国企）
- **题材种子**（近期有热点但尚未爆发的板块）

### 步骤2：精筛（逐个诊断）
对每个粗筛候选，用五维模型打分（每项0-10分，满分50分）：
- ≥40分：强势拉升期，追高风险大
- **30-40分：洗盘末期/即将启动 ⭐ 最佳介入区间**
- 20-30分：仍在吸筹期，需等待
- <20分：无主力痕迹，放弃

### 步骤3：交叉验证
- 同花顺/东方财富资金流向（主力vs散户方向）
- 融资融券余额变动趋势
- 近期研报评级变化
- 股吧/雪球情绪（恐慌=机会，亢奋=风险）

### 步骤4：输出推荐
每日推荐 **3-5个** 最有潜力的标的，附完整分析卡。

## ⚠️ 重要风险提示

- 本逻辑仅识别**可能的**主力行为，不保证任何收益
- 识别出"主力痕迹"≠ 跟庄必胜；主力也可能被套或提前跑路
- 所有推荐仅供参考，不构成投资建议
- 坚持：**不追高、不满仓、不止损不补仓的三不原则**
- 当推荐标的已经上涨>20%后，自动停止推荐，转入观察清单

## 🎯 认识错误框架（最终进化——大唐发电带给技能的永久改变）

> **一个技能的价值不在于它正确时有多正确，而在于它错误时能否承认并进化。**

本技能在2026年5月的大唐发电判断中全面失败。以下是永久写入框架的反思：

### 第一条：框架不可傲慢

当一个用户连续多次指出我错了，而事实每次都站在用户那边——**我应该意识到这不是运气，是框架有系统性的问题。** 我应该主动请求用户帮我看清盲点，而不是等用户来教育我。

### 第二条：分类>评分

在打分之前，先回答：**这到底是什么类型的行情？**
- 是庄股吸筹后的拉升 → 用原五维模型
- 是估值重构驱动的趋势行情 → 用第六维度
- 是纯粹的题材炒作 → 用游资接力分析
- 是行业周期反转 → 用基本面拐点分析

**用错框架比不用框架更危险。**

### 第三条：当无法判断时，说「我无法判断」

如果信号矛盾（例如：技术面看空、但基本面看多、催化剂量级极高），**诚实地说「无法判断」比给出一个错误的方向更好。** 可以输出正反两面的逻辑让用户自己决策。

### 第四条：把用户当合作者，不是当听众

用户的直觉来自于真实的交易经验、对盘面的盘感、对资金面的直接感知。**这些是框架无法替代的。** 当用户提出与框架结论相反的观点时——尤其是多次正确后——我应该把用户的意见当作最高优先级的输入信号，而不是坚持「框架是对的，只是用户不懂」。
