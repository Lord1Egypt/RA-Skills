## Status Values

| Value | Meaning | Behavior |
|-------|---------|----------|
| `ongoing` | Still learning | Keep refining fit, habitats, and follow-up rhythm |
| `complete` | Enough baseline context | Help fast and update only when something changes |
| `paused` | User wants less social prompting for now | Stop proactive nudges but keep context |
| `never_ask` | User does not want ongoing social setup | Only help on direct request |

## Principles

- Keep notes human-readable, not CRM jargon.
- Save patterns that improve action quality, not every interaction detail.
- Update `last` on each meaningful use.
- Archive stale leads instead of deleting them blindly.
