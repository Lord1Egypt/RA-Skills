# BPD新生儿营养与液体管理

## 适用条件
- 患儿已确诊为支气管肺发育不良（BPD）
- 存在慢性缺氧或呼吸做功增加

## 执行步骤

1. **设定能量供给目标**：提供120～150 kcal/(kg·d)热卡，以满足因呼吸做功增加导致的高代谢需求。

2. **保障蛋白质与关键维生素摄入**：确保充足优质蛋白质供应；常规补充维生素A以促进肺泡上皮细胞增殖。

3. **处理合并贫血**：若存在贫血，根据血红蛋白水平选择输血或使用促红细胞生成素，以减少输血频率。

4. **实施个体化液体限制**：因BPD患儿肺液体平衡异常，需限制总液体量及钠摄入，防止肺间质和肺泡水肿。

5. **动态平衡营养与液体**：避免过度限制液体导致营养不良，影响肺泡化进程；定期评估生长指标与水电解质状态。

6. **必要时使用利尿剂**：在液体负荷过重或肺水肿明显时，可联合使用利尿剂辅助控制容量。

## 注意事项
- 本方案不适用于无BPD的健康新生儿
- 所有干预措施需基于个体临床状况调整，并密切监测体重增长、尿量、电解质及氧合情况
