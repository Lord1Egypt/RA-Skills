# Live Session Quality Checklist

## Lineup Structure (7 checks)
- [ ] Hook product is <$30, visually engaging, fast to demo
- [ ] Price arc follows Low → Mid → High → Flash → Bundle structure
- [ ] Price anchor product is shown before hero product
- [ ] Hero product scheduled during expected peak viewer window
- [ ] Flash deal product has a real, verifiable discount
- [ ] Closing bundle is unique to this session (not available in store at same price)
- [ ] 6–9 products total for a 60-min session (or 8–12 for 90 min)

## Script Completeness (6 checks)
- [ ] Intro hook written for every product (10 sec)
- [ ] Demo talking points listed for every product (3 points min)
- [ ] Close mechanic written for every product (countdown or urgency line)
- [ ] Forward hook written for every transition
- [ ] Engagement prompts scheduled every 10–12 minutes
- [ ] Opening and closing scripts drafted

## Stock & Logistics (5 checks)
- [ ] All products physically staged in lineup order
- [ ] Stock counts confirmed and written into script
- [ ] Restock plan in place for products likely to sell out
- [ ] Cart links tested on a separate device before going live
- [ ] Flash deal discount verified in platform backend (discount is real)

## Technical (5 checks)
- [ ] Internet upload speed >10 Mbps
- [ ] Ring light or studio lighting set up and tested
- [ ] Camera angle shows product clearly (not just host face)
- [ ] Backup phone or device available if primary fails
- [ ] Platform notifications / reminders sent to followers 30 min before

## Energy & Pacing (5 checks)
- [ ] Host has reviewed script and knows each product's key talking points
- [ ] Timer or clock visible to host to track segment pacing
- [ ] Comment reading plan (don't ignore chat for >2 min)
- [ ] Backup "dead air" questions prepared for slow comment periods
- [ ] Host water and any physical supplies staged off-camera
