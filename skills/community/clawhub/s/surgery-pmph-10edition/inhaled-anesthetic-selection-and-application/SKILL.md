# 吸入麻醉药选择与临床应用

## 触发条件
当满足以下条件时应用本技能：
- 计划使用吸入麻醉药进行全身麻醉
- 具备麻醉机和气体监测设备

## 执行步骤

### 1. 评估禁忌与约束
- 若患者存在封闭腔隙疾病（如肠梗阻、中耳炎），禁用氧化亚氮（N₂O）
- 若患者肝功能严重受损，慎用卤代烃类吸入麻醉药（如异氟烷、地氟烷）

### 2. 确定关键药理参数
- **MAC（最低肺泡有效浓度）**：值越小，麻醉效能越强
- **血/气分配系数**：值越低，诱导和苏醒越快
- **油/气分配系数**：值越高，脂溶性越强，麻醉效能越高

### 3. 选择合适药物并设定参数
- **氧化亚氮（N₂O）**：
  - 仅用于复合麻醉，不可单独维持
  - 常用浓度50%~70%，必须维持FiO₂ > 0.3
  - 停用后吸纯氧5~10分钟以防弥散性缺氧
- **七氟烷**：
  - 适用于诱导与维持，呼吸道无刺激
  - 维持浓度1.5%~2.5%，成人苏醒约10分钟
- **地氟烷**：
  - 可控性最佳，适用于门诊或心脏病患者
  - 需专用蒸发器，术后恶心呕吐发生率较低

### 4. 输出结果
- 推荐吸入麻醉药
- 预期诱导与苏醒时间
- 所需浓度范围
- 相关禁忌与安全警示

> 注：详细药理数据、代谢路径及毒性信息见下文参考文档。

## 参考文档

### 吸入麻醉药药理参数详表

**MAC值（%）**
- 氧化亚氮（N₂O）：≈105%
- 七氟烷：≈2.0%
- 地氟烷：≈6.0%
- 异氟烷：≈1.2%

**血/气分配系数**
- N₂O：0.47
- 七氟烷：0.65
- 地氟烷：0.42
- 异氟烷：1.4

**油/气分配系数**
- N₂O：1.4
- 七氟烷：55
- 地氟烷：20
- 异氟烷：98

**代谢与毒性说明**
- 大多数吸入麻醉药以原型经肺排出，肝肾毒性低
- 甲氧氟烷可致暴发性免疫性肝损害（已少用）
- 恩氟烷、异氟烷、地氟烷有潜在肝损风险，但临床发生率极低

**安全操作规范**
- 使用N₂O时必须确保FiO₂ ≥ 0.3
- 停用N₂O后应给予100%氧气5~10分钟
- 肝功能异常者优先考虑地氟烷或七氟烷
