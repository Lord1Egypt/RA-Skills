# 核编码线粒体蛋白的基质导入机制

执行以下步骤完成核编码前体蛋白向线粒体基质的定向转运与成熟：

1. **识别基质导入序列（MTS）**
   - 确认目标蛋白N端含有20～50个氨基酸的MTS，特征为疏水、带正电荷，富含Arg、Lys、Ser、Thr，极少含Asp、Glu
   - 由外膜受体TOM20/TOM22特异性识别并结合MTS

2. **维持前体蛋白非折叠状态**
   - 在胞质中确保前体蛋白与Hsp70（或少数情况下与NAC）结合，防止错误折叠或聚集
   - 若为内膜定位蛋白（如ATP/ADP反向转运体），先经TOM37/TOM70结合，再转移至TOM20/TOM22

3. **形成跨膜转运通道**
   - 协调TOM复合物（含通道蛋白TOM40）与内膜TIM23/17系统在转位接触点耦联
   - 构建跨越线粒体外膜与内膜的连续蛋白转运通道

4. **驱动穿膜转运（布朗棘轮模型）**
   - 利用多肽链在通道内的布朗运动
   - 一旦前导肽进入基质，立即被mtHsp70捕获，阻止回退
   - mtHsp70通过ATP水解引发构象变化，产生单向拖力，推动肽链持续进入
   - 随肽链深入，招募更多mtHsp70分子协同作用

5. **完成基质内加工与折叠**
   - 由定位于内膜的基质加工蛋白酶（MPP）切除MTS（早期事件）
   - 在Hsp60/Hsp10等分子伴侣协助下完成正确折叠
   - 形成具有生物学活性的成熟蛋白

6. **确保能量供应**
   - 验证线粒体膜电位（ΔΨ）正常
   - 提供充足ATP以支持mtHsp70功能及转运全过程

> **注意**：若蛋白无MTS、已折叠/聚集、或线粒体膜电位丧失，则此机制无法启动。
