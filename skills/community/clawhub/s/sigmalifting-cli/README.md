# Sigma CLI Skill

This skill teaches agents how to use the SigmaLifting CLI.

Install the CLI:

```bash
npm install -g https://github.com/sigmalifting/cli.git
```

Required executable:

```bash
sigmalifting-cli doctor --compact
sigmalifting-cli help --compact
sigmalifting-cli schema list --compact
```

The CLI is the executable schema and validation source of truth. The skill is the modeling guide.
