## О Проекте

Этот проект систематически объединяет основные области анатомии человека, нейроанатомии, клинической прикладной анатомии и интегративного китайско-западного анатомического анализа, охватывая **112 ключевых анатомических навыков**, организованных в **8 категорий**.

**Целевая аудитория** : Студенты-медики, клиницисты, исследователи анатомии, ординаторы-хирурги

**Справочный учебник** : «Систематическая анатомия» (People's Medical Publishing House, 10-е издание)

## Структура Проекта

```
systematic-anatomy-10edition/
├── SKILL.md                        # Реестр навыков (точка входа ClawHub)
├── catalog.md                      # Индекс навыков с навигацией по категориям
├── README.md                       # Документация на китайском
├── README_EN.md                    # Документация на английском
├── README_JP.md                    # Документация на японском
├── README_FR.md                    # Документация на французском
├── README_RU.md                    # Этот файл — Документация на русском
├── <skill-name>/                   # Детальные определения навыков
│   └── SKILL.md                    #   Когда использовать, процедура, ссылки
├── scripts/                        # Исполняемые скрипты
├── config/                         # Файлы конфигурации
├── tests/                          # Валидация и тестирование
└── assets/                         # Статические ресурсы (иконки, изображения)
```
