# 知识库配置

> 本文件配置园区专属知识库的位置。
> 知识库支持两种部署方式：本地文件（推荐个人使用）、腾讯文档（推荐团队协作）。
> 新用户请参考下方「首次配置引导」。

---

## 部署方式选择

### 方式A — 本地文件（推荐个人/单人使用）

1. 打开以下4个模板文件，将 `{TODO: 待填写}` 替换为真实数据：
   - `references/园区基础_房源信息.md`
   - `references/园区基础_租金报价.md`
   - `references/园区基础_配套资源.md`
   - `references/政策文件_地方产业政策.md`
2. 也可以在 workspace 目录下创建 `knowledge-base/` 存放真实数据，
   Agent 会优先读取 workspace 再 fallback 到 references 模板。
3. 编辑本文件，将下方「腾讯文档链接」表格留空即可。

### 方式B — 腾讯文档（推荐团队/多人协作）

1. 在腾讯文档创建4个智能表格/文档：
   - 《园区房源信息》
   - 《租金报价标准》
   - 《配套资源》
   - 《地方产业政策》
2. 将文档链接填入下方「腾讯文档链接」表格
3. 团队共同维护，Agent 实时读取最新数据

## 🔒 数据安全说明

| 安全原则 | 说明 |
|----------|------|
| **数据不外发** | 所有知识库数据存储在本地，不会发送到任何服务器 |
| **腾讯文档仅读取** | Agent 仅读取你提供的文档链接，不获取任何其他信息 |
| **workspace 优先** | 真实数据建议放在 `workspace/knowledge-base/` 目录，与 Skill 包物理隔离 |
| **模板不含数据** | `references/` 下的模板文件不包含任何真实业务数据，可安全分享 |

> ⚠️ 重要提醒：安装本 Skill 时，请勿将含真实数据的 `workspace/knowledge-base/` 目录分享给他人。
> 如需迁移，重新在新环境填入数据即可。

## 首次配置引导（新用户必读）

欢迎使用产业园招商助手！使用前需要完成以下两步：

**第1步：选择部署方式**
- 单人使用 → 直接编辑 `references/` 下4个模板文件，将 `{TODO}` 替换为真实数据
- 团队使用 → 在腾讯文档创建4个文档，链接填入本文件「腾讯文档链接」表格

**第2步：替换园区名称**
- 编辑 `SKILL.md`，将「📝 请填写园区名称」替换为你的园区名

完成以上两步后，直接对话即可开始使用。
