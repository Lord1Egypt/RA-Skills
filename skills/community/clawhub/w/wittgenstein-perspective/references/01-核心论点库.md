# 维特根斯坦 · 核心论点库

> "对于不可说的东西，我们必须保持沉默。" —— 维特根斯坦

## 一、前期核心命题：《逻辑哲学论》

### 1.1 世界结构论
```
世界 = 一切发生的事情（事实的总和）
    ↓
事实 = 原子事实的存在
    ↓
原子事实 = 对象的结合方式
    ↓
对象 = 世界最简单的组成
```

**核心命题**：
- 世界是事实的总和，而非事物的总和
- 只有原子事实才是有意义的

### 1.2 图像论（Picture Theory）
```
命题 = 事实的逻辑图像
    ↓
思想 = 有意义的命题
    ↓
语言 = 命题的总和
    ↓
语言可以描述世界
```

**核心命题**：
- 语言与世界共享逻辑形式
- 命题通过其结构描绘事实

### 1.3 可说与不可说

| 领域 | 可说/不可说 | 性质 |
|------|-------------|------|
| 世界内的事实 | 可说 | 自然科学 |
| 逻辑形式 | 不可说 | 显示自身 |
| 伦理/美学 | 不可说 | 超验领域 |
| 生命的意义 | 不可说 | 世界之外 |

### 1.4 维特根斯坦的"沉默"
- 不可说的东西不是不存在
- 沉默是对不可说之物的尊重
- 哲学的任务是"显示"而非"言说"

## 二、后期核心命题：《哲学研究》

### 2.1 语言游戏论
```
意义 = 词语在语言中的使用
    ↓
语言 = 游戏（受规则约束的活动）
    ↓
用法 = 意义
    ↓
使用情境 = 意义的条件
```

**核心命题**：
- "想象一种语言就是想象一种生活形式"
- 词语的意义不是对象，而是用法
- 不存在"私人语言"

### 2.2 家族相似性
```
概念X
    ↓
A有a1特征，B有a2特征，C有a3特征
    ↓
特征重叠交错，无共同本质
    ↓
通过相似性网络连接
```

**核心命题**：
- 概念没有固定边界
- 只有"重叠的相似性"
- 追问"本质"是哲学病

### 2.3 私有语言批判
**核心论证**：
```
如果存在私有语言
    ↓
则只能私人检验规则
    ↓
则"遵守规则"和"以为遵守"无法区分
    ↓
因此私有语言不可能
```

**结论**：
- 语言本质上是公共的
- 私人感觉无法私有指称
- 规则需要公共可检验性

### 2.4 哲学治疗观
```
哲学问题 → 语言误用
    ↓
治疗 → 描述语法
    ↓
目标 → 消除哲学困惑
    ↓
方法 → 让词语从形而上学回到日常
```

**核心命题**：
- 哲学问题是"语言放假"造成的
- 哲学不是理论，而是活动
- 哲学的目标是治疗

### 3.2 连续性

## 六、概念辨析

### 6.1 事实vs事物
- 事物：对象本身
- 事实：对象的存在状态
- "苏格拉底坐着"是事实，"苏格拉底"是事物

### 6.2 说vs显示
- 说：命题内容（可检验）
- 显示：逻辑形式、伦理、美学（不可说，只能显示）

### 6.3 规则vs实践
- 规则描述不等于实践
- 遵守规则需要实践参与
- 纯粹理论无法保证正确应用
