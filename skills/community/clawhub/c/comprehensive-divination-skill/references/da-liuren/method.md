# 大六壬推演方法

## 核心原理

大六壬是最复杂的占卜术之一。以月将加时辰构建天地盘，确定四课后使用九大法则发传，形成完整的事态推演链（初传→中传→末传），最后分析各位置的十二天将和五行生克来断课。

## 推演步骤

### 第一步：月将加时（构建天地盘）

1. 确定当前**月将**（太阳所在宫位，按节气确定）：
   - 大寒→子（神后）、雨水→亥（登明）、春分→戌（河魁）、谷雨→酉（从魁）
   - 小满→申（传送）、夏至→未（小吉）、大暑→午（胜光）、处暑→巳（太乙）
   - 秋分→辰（天罡）、霜降→卯（太冲）、小雪→寅（功曹）、冬至→丑（大吉）

2. 确定当前**时辰**（当地真太阳时，子=23-1点，丑=1-3点，...）

3. 构建天地盘矩阵：
   - 地盘（固定）：子丑寅卯辰巳午未申酉戌亥，顺时针排列
   - 天盘（月将+时辰）：将月将放在时辰位置的地盘上，其余顺排

### 第二步：定四课

四课由日干和日支（即当日的天干地支）推导：
1. **第一课**：日干 + 日干上方天盘的地支
2. **第二课**：第一课天盘的地支在地盘的位置 + 该位置上方天盘的地支
3. **第三课**：日支 + 日支上方天盘的地支
4. **第四课**：第三课天盘的地支在地盘的位置 + 该位置上方天盘的地支

### 第三步：发三传

按九大法则的优先级依次检查四课，找到适用的法则后发传：
1. 贼克法（最常用）→ 2. 比用法 → 3. 涉害法 → 4-9 按需

详细规则参考 references/da-liuren/sanzhuan-faze.md

### 第四步：天将布局

确定贵人位置（按日干查贵人诀），然后按顺逆排法布局十二天将。

参考 references/da-liuren/shier-tianjiang.md

## 输出格式

大六壬为复杂推演，输出应包含：

1. **起课参数**：日期、时辰、月将、日干日支
2. **天地盘**：以表格呈现天地盘矩阵
3. **四课**：展示四课推导过程和各课内容
4. **三传**：初传·中传·末传 + 所用法则
5. **天将布局**：十二天将在各位置的分布
6. **综合断课**：
   - 日干的五行状态
   - 三传中的吉凶天将
   - 四课中的生克关系
   - 事态三阶段推断（初传起因→中传发展→末传结局）
7. **注意**：必须在最后附免责声明"占卜结果仅供参考，请结合实际情况理性判断"
