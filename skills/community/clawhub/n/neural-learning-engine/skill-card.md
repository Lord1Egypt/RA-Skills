## Known Risks and Mitigations: <br>
Risk: Users may mistake the simulated memory loop for a production memory system. <br>
Mitigation: Treat the skill as conceptual response formatting unless persistent memory, event tracking, dashboards, or external AI APIs are added and separately reviewed. <br>
Risk: Extensions that add persistent memory or external AI APIs could introduce privacy, retention, or consent risks. <br>
Mitigation: Require clear user consent, retention limits, and data-minimization controls before adding those capabilities. <br>
