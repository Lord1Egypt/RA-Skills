# Investment Committee Memo

Turns first-meeting notes, deck contents, and diligence call summaries into a structured **Investment Committee (IC) memo** with an evidence-vs-assertion matrix, a kill-criteria checklist, and a recommendation.

## When to use

- Preparing a new opportunity for partnership review.
- Updating an IC memo after the next diligence call or data-room access.
- Writing the deal lead's recommendation memo for a seed, Series A, or follow-on round.

## What it produces

- 12-section IC memo: thesis, market, product, traction, team, GTM, competition, financials, terms, risks, kill criteria, recommendation.
- Evidence-vs-assertion matrix mapping every claim to a cited source (deck slide, call note, public source).
- Kill-criteria checklist: explicit conditions under which the partnership should pass.
- Verdict: **Pass / Track / Diligence / Term-sheet**, tied to the fund thesis.
- An open-questions list for the next founder call.

## What it does **not** do

- It does not constitute investment advice.
- It does not invent metrics. If a metric (ARR, growth rate, burn, runway) is missing, it is logged as **Unknown** — never inferred.
- It does not score "founder vibes"; team assessment is restricted to evidence the user supplied.

## Confidentiality

Founder material is treated as confidential. The skill never writes to external services, and it instructs the user to remove or redact any unfiled trade-secret content before pasting.
