## Description: <br>
Generate PNG/SVG charts (line, bar, hbar, pie, stacked, scatter, area) from CSV data using matplotlib. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>

## Use Case: <br>
Developers, analysts, and agents use this skill to turn local CSV data into BI-ready chart images for trend analysis, category comparison, and multi-series visualization. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The setup step installs Python dependencies into a local virtual environment. <br>
Mitigation: Review the setup command before running it and use a trusted Python package source. <br>
Risk: The script reads CSV files and writes chart files on local paths. <br>
Mitigation: Only point the skill at CSV files and output paths you intend to use. <br>
