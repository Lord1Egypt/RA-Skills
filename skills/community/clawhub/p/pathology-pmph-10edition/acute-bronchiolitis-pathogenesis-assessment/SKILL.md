# Acute Bronchiolitis Pathogenesis Assessment

## 何时使用
当面对4岁以下婴幼儿（尤其1岁以内）在冬季出现以下表现时立即启用本技能：
- 喘息、呼吸困难
- 听诊有哮鸣音
- 可疑呼吸道病毒感染史（如呼吸道合胞病毒）

**注意**：本技能不适用于成人、大支气管炎症或由化学气体等非病毒因素引起的气道病变。

## 执行步骤

1. **确认易感因素**：
   - 患儿年龄 ≤1 岁（90%病例集中于此）
   - 细支气管管径 <2mm，气流缓慢
   - 黏膜IgA水平低，免疫防御弱
   - 管壁缺乏软骨支撑，易塌陷

2. **识别典型病理变化**：
   - 黏膜充血肿胀
   - 纤毛上皮坏死脱落，被无纤毛上皮替代
   - 杯状细胞增生，黏液分泌增多
   - 管壁淋巴细胞与巨噬细胞浸润
   - 管腔内含纤维蛋白、炎症细胞及脱落上皮组成的渗出物

3. **评估功能后果**：
   - 若管腔**部分阻塞** → 考虑急性阻塞性肺气肿
   - 若管腔**完全阻塞** → 警惕小灶性肺不张
   - 若炎症扩散至肺间质/肺泡 → 判断是否合并细支气管周围炎或局限性肺炎

4. **预测临床结局**：
   - **轻症**：渗出物可吸收或咳出，预后良好
   - **重症**：
     - 管壁瘢痕修复
     - 渗出物机化导致永久性管腔闭塞
     - 可能进展为纤维闭塞性细支气管炎

5. **发出预警**：
   - 若出现进行性呼吸困难、发绀或血气异常，立即提示“管腔严重阻塞，需警惕呼吸衰竭或纤维闭塞性病变”
