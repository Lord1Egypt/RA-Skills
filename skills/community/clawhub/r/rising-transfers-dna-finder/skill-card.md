## Description: <br>
Find cheaper, style-matched footballer alternatives using Rising Transfers vector DNA search. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External football scouts, analysts, and developers use this skill to query Rising Transfers for similar or lower-value footballers and review ranked replacement candidates. It requires an RT_API_KEY and sends player names to the Rising Transfers API. <br>
