# 舌纹诊病（补充知识）

> 来源：《舌纹诊病》（汪汉，2000）
> 舌纹是舌面出现的纹理/裂纹，不同于传统的裂纹舌概念。

## 一、舌纹的基本概念

- **舌纹**：舌面上出现的各种纹理（裂纹），有粗有细，有深有浅
- 一般舌纹出现在舌中部，少数患者在舌尖、心肺、脾、肾等部位出现
- 舌纹多而乱 → 主病情复杂难治
- 舌纹少而浅 → 主病情轻易治
- 舌纹深而粗 → 病情危重
- 舌纹细而小 → 病情轻
- 舌纹由细小、细少转变粗多、粗乱 → 病情恶化
- 舌纹由粗多、粗乱转变细小、细少 → 病情康复

## 二、舌纹分类

### 字形纹类
- 各种汉字形状的舌纹（如"一"字纹、"丰"字纹、"人"字纹等）
- 不同字形与不同脏腑疾病对应

### 象形纹类
- 舌纹呈特定图形或图案
- 根据纹路的走向、形状判断疾病性质

### 舌苔纹类
- 舌苔表面形成的纹理
- 与胃气、病邪性质相关

### 舌下纹类
- 舌下表面的纹理
- 反映气血运行状况

## 三、舌纹诊病的临床意义

### 1. 诊断正气盛衰
- 舌质红润 → 气血旺盛
- 舌质淡白 → 气血衰微
- 舌苔薄白而润 → 胃气旺盛
- 舌光无苔 → 胃气衰败或胃阴枯竭

### 2. 分辨病位深浅
- 舌苔薄 → 疾病初期，邪入尚浅，病位在表
- 舌苔厚 → 病邪入里，病位较深
- 舌质绛 → 热入营血，病位更深
- 舌纹出现 → 病已形成

### 3. 区别病邪性质
- 黄苔 → 主热邪
- 白滑苔 → 主寒邪
- 腐腻苔 → 主食积痰浊
- 黄厚苔 → 湿热
- 舌偏斜 → 风邪
- 舌有瘀斑瘀点 → 血瘀
- 舌纹出现在舌尖 → 心脏病变
- 舌纹出现在舌正中间 → 脾胃病变
- 舌纹出现在舌根 → 肾脏病变
- 满舌出现舌纹 → 五脏综合病症

### 4. 推断病情进退
- 苔由白转黄，又进一步变为灰黑 → 病邪由表入里，由轻转重，由寒化热
- 苔由润转燥 → 热渐盛而津渐伤
- 苔由厚变薄、由燥转润 → 病邪渐退，津液复生

## 四、舌纹诊断的特殊要点

### 舌纹与五脏疾病
| 脏腑 | 舌纹位置 | 舌纹特征 |
|------|---------|---------|
| 心系疾病 | 舌尖 | 舌纹出现在舌尖部位 |
| 肺系疾病 | 舌面前部 | 舌纹出现在舌面前部 |
| 脾胃病 | 舌正中间 | 舌纹出现在舌中部 |
| 肝病 | 舌边 | 舌纹出现在舌边 |
| 肾病 | 舌根 | 舌纹出现在舌根部 |

### 舌纹与病型判断
- 舌纹出现在某脏腑对应区域 → 该脏腑有病变可能
- 舌纹粗大深长 → 病程长，病情重
- 舌纹细小浅短 → 病程短，病情轻
- 满舌舌纹 → 五脏综合性疾病

## 五、舌色与病色（补充）

### 望色十法
1. **浮** → 色显于皮肤之间 → 病在表
2. **沉** → 色隐于皮肤之内 → 病在里
3. **清** → 清明 → 病在阳
4. **浊** → 浊暗 → 病在阴
5. **微** → 色浅淡 → 正气虚
6. **甚** → 色深浓 → 邪气盛
7. **散** → 疏离 → 病近将解
8. **搏** → 壅滞 → 病久渐聚
9. **泽** → 气色润泽 → 主生
10. **夭** → 气色枯槁 → 主死

### 病色善恶
- **善色**（气至）：五色光明润泽 → 脏腑精气未衰，预后良好
- **恶色**（气不至）：五色晦暗枯槁 → 脏腑败坏，预后不良

## 六、舌诊注意事项（补充）

### 影响因素
- **年龄与体质**：
  - 老年人气血偏衰，舌多见裂纹，舌乳头也常萎缩
  - 小儿易患舌疾，舌面出现白屑或剥苔
  - 肥胖之人舌多略大质淡
  - 消瘦之人舌体略瘦而质偏红
- **季节与时间**：
  - 夏季湿盛时苔多厚，或见淡黄色
  - 秋季燥气当令时苔多薄而干
  - 冬季严寒舌常湿润
  - 晨起时苔多厚，白天进食后变薄
- **饮食影响**：
  - 牛乳、乳儿因乳汗关系附有白苔
  - 富含脂肪的食品使舌面附着黄白色渣滓
  - 酸梅、咖啡、葡萄汁等使舌呈黑褐或茶褐色
  - 蛋黄、柿子等使舌变黄
  - 过冷过热的饮食及刺激性食物可使舌色变白

### 诊断原则
- 舌诊结合脉诊、闻诊、问诊才能全面分析病情
- 察舌质重在辨正气的虚实
- 察舌苔重在辨邪气的深浅与性质
- 察舌纹重在辨具体是什么纹、出现在什么部位、病程久暂
- 血病观质，气病观苔，吉凶观舌纹（《形色外诊简摩》）
