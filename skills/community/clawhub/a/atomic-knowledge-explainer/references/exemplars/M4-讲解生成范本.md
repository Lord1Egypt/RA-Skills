# 机器学习基本概念讲解

## 什么是机器学习？
机器学习是人工智能的一个分支，它让计算机能够从数据中学习，而不需要明确编程。

## 核心概念

### 1. 模型(Model)
模型是从数据中学习到的函数映射。就像一个"经验公式"，能够根据输入预测输出。

**示例**：线性回归模型 y = wx + b

### 2. 训练(Training)
训练是使用数据优化模型参数的过程。就像"做题练习"，通过大量练习提高准确率。

**示例**：使用梯度下降算法调整w和b的值

### 3. 预测(Prediction)
预测是使用模型对新数据进行推理。就像"考试答题"，用学到的知识解决新问题。

**示例**：给定x=5，预测y的值

### 4. 评估(Evaluation)
评估是验证模型性能的过程。就像"查看考试成绩"，了解学习效果。

**示例**：计算准确率、精确率、召回率

## 概念关系
```
特征+标签 → 训练 → 模型 → 预测 → 结果 → 评估
```

## 总结
机器学习的核心流程：准备数据→训练模型→预测→评估
