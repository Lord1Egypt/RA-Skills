# 上海一网通办市级事项索引

本索引用于快速定位部门和事项。详情以官方一网通办页面和 021-12345 为准。

## 使用方法

1. 用用户描述中的关键词在本文件和 `service_items.jsonl` 中检索。
2. 同时检索个人/法人；如果身份明确，优先对应身份。
3. 返回候选事项时附 `guide_url` 和线上办理标记。
