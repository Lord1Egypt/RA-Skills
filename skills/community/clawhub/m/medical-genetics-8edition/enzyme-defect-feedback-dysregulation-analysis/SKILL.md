# 酶缺陷导致反馈调节失常的机制分析

## 适用场景
- 患者表现为性发育异常（如女性阴蒂肥大、男性假性早熟）
- 怀疑存在下丘脑-垂体-靶腺轴负反馈失效
- 需解释为何终产物缺乏却伴随上游激素（如ACTH）持续升高

## 执行步骤
1. **确认系统具备负反馈调节环路**：确保该通路中终产物可抑制上游激素分泌（如皮质醇抑制ACTH）。
2. **识别关键合成酶缺陷**：定位阻断点（如21-羟化酶缺陷导致皮质激素合成受阻）。
3. **追踪代谢流向改变**：分析前体物质是否转向替代通路（如孕酮前体转向雄激素合成）。
4. **推导反馈失常过程**：
   - 终产物（如可的松、醛固酮）不足 → 无法抑制垂体 → ACTH过度分泌
   - ACTH持续刺激靶腺（肾上腺皮质）→ 靶腺增生
   - 增生组织仍无法合成终产物，但过量产旁路激素（如睾酮）
5. **关联临床表型**：结合性别与年龄，判断是否符合常染色体隐性遗传模式下的典型表现。

## 输出警示
若新生儿出现外生殖器模糊或女性呈现男性化特征，应立即筛查21-羟化酶缺陷，并启动内分泌紧急评估流程。
