# CHANGELOG

## v1.0.0 (2026-05-18)

### 🎉 First Release

- **4 Categories, ~35 Rules**: Security (Critical), Performance/Design/Error Handling (Warning), Convention/Dependencies (Info)
- **3 Scan Modes**: Quick Scan (src/main), Full Scan (+tests +build), Incremental Scan (git diff)
- **Health Score 0-100**: Weighted formula based on issue count and severity
- **Auto-Fix**: Low-risk Info issues auto-fixed on user confirmation
- **Extensible Rule Engine**: Easy to add new language/framework rules
- **Markdown Report**: Professional, shareable health reports
- **Java/Spring Boot Focus**: MyBatis, JPA, Maven, application.properties complete coverage
- **False Positive Notes**: Each rule includes conditions where it might misfire
