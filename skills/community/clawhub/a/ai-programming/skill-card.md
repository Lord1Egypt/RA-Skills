## Known Risks and Mitigations: <br>
Risk: Generated code or automation guidance may perform file, network, deployment, or API-key-related actions that affect local systems or external services. <br>
Mitigation: Review generated code and commands before running them, especially when they read or write files, send network requests, handle secrets, deploy services, or automate actions. <br>
Risk: Broad AI programming guidance may include incorrect, incomplete, or insecure implementation details. <br>
Mitigation: Test generated code in a controlled environment and apply normal code review, dependency review, and security scanning before production use. <br>
