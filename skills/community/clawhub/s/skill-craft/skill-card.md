## Description: <br>
Create, optimize, or update AgentSkills with evidence-based design patterns, validation scripts, and reference material for skill structure, optimization, taxonomy, and state management. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[leonardo-lb](https://clawhub.ai/user/leonardo-lb) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent builders use this skill to create new AgentSkills, optimize existing skills against evidence-based design dimensions, validate skill structure, and package skills for release. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated or optimized skill content may be incorrect, incomplete, or misleading before review. <br>
Mitigation: Review generated files and run the bundled validation script before publishing or deploying the skill. <br>
Risk: The packaging script zips files under the selected skill directory, which can include unrelated private files if the wrong folder is selected. <br>
Mitigation: Package only a prepared skill directory and check its contents before publishing the generated .skill archive. <br>
