# 珍珠鉴定与选购指南

## 一、珍珠分类体系

```
珍珠
├── 天然珍珠（极少，价格极高）
└── 养殖珍珠（市场99%+）
    ├── 海水珍珠
    │   ├── Akoya珍珠（日本/中国）
    │   ├── 南洋珍珠
    │   │   ├── 南洋白珠（澳洲白蝶贝）
    │   │   ├── 南洋金珠（金唇贝）
    │   │   └── 南洋黑珠（大溪地黑蝶贝）
    │   └── 其他海水珠（马氏贝/企鹅贝等）
    └── 淡水珍珠（中国为主，诸暨/常德）
        ├── 无核淡水珍珠
        ├── 有核淡水珍珠（Edison/爱迪生）
        └── 异形/巴洛克淡水珍珠
```

## 二、珍珠品质评价标准

### 七大维度评分

| 维度 | 说明 | 顶级标准 |
|------|------|----------|
| **光泽** | 珍珠灵魂，最重要 | 镜面般清晰反射人影 |
| **形状** | 正圆最贵 | 精圆 > 近圆 > 椭圆 > 异形 |
| **大小** | 直径(mm) | 越大越稀有 |
| **颜色** | 体色+伴色+晕彩 | 品种决定颜色范围 |
| **表面光洁度** | 瑕疵程度 | 肉眼无瑕 |
| **珠层厚度** | 海水珍珠特有 | ≥0.5mm(Akoya)、≥2mm(南洋珠) |
| **匹配度** | 项链/耳钉配对 | 颜色/大小/光泽一致 |

### 光泽等级
| 等级 | 描述 | 反射效果 |
|------|------|----------|
| 极强 | 镜面般清晰反射 | 可清晰看到人脸五官 |
| 强 | 明亮反射 | 可看到人脸轮廓 |
| 中 | 一般反射 | 模糊反射 |
| 弱 | 暗淡 | 几乎无反射 |

## 四、真假鉴别

### 快速鉴别法
| 方法 | 真珍珠 | 假珍珠(塑料/玻璃/贝壳粉) |
|------|--------|--------------------------|
| **牙咬法** | 有沙砾感(粗糙感) | 光滑 |
| **摩擦法** | 两粒摩擦有阻力感+粉末 | 平滑或打滑 |
| **放大观察** | 表面有生长纹理(等高线) | 光滑无纹理或均匀颗粒 |
| **重量感** | 有分量，手感凉爽 | 轻飘飘，手感温热 |
| **钻孔观察** | 孔口锐利，可见珠层 | 孔口圆钝，涂层剥落 |
| **火烧法** | 轻微烧焦味，不黑 | 塑料味，变黑(不推荐) |

### 真假"金珠"鉴别
- 真金珠颜色由内而外，染色金珠颜色浮于表面
- 用棉签蘸酒精擦拭：染色金珠会掉色
- 观察钻孔处：染色金珠孔口颜色深

### Akoya 特殊认证
- **花珠**：Akoya 中的优品（前20-30%）
- **天女**：花珠中的极品（前1-3%），需同时满足白/粉/绿三种干涉色
- **真多麻**：天然银蓝色 Akoya（非辐照处理）
- **彩凛珠**：6mm 以下的小尺寸高品质 Akoya

## 六、选购建议

### 按场景推荐
| 场景 | 推荐品种 | 预算 |
|------|---------|------|
| 日常通勤 | 淡水有核/Akoya 5-7mm | 500-3,000 |
| 职场进阶 | Akoya 花珠 7.5-8mm | 3,000-10,000 |
| 重要场合 | 南洋白珠/大溪地 10-12mm | 10,000-50,000 |
| 婚嫁 | 天女 Akoya / 南洋白珠 | 15,000-100,000 |
| 送长辈 | 南洋金珠 11-13mm | 8,000-30,000 |
| 入门收藏 | 大溪地孔雀绿 12mm+ | 20,000+ |

### 保养要点
1. **最后戴、最先摘**：避免接触化妆品/香水/发胶
2. **远离水和化学品**：珍珠怕酸碱，洗澡/游泳需摘下
3. **单独存放**：用软布袋，避免与其他首饰摩擦
4. **定期佩戴**：珍珠需要吸收皮肤油脂保持光泽
5. **每1-2年换线**：丝线老化易断裂

## 七、常见陷阱
1. **淡水充海水**：淡水有核珍珠染色冒充南洋金珠/大溪地
2. **辐照改色**：Akoya 辐照变灰冒充"真多麻"
3. **染色金珠**：淡水珍珠染金色冒充南洋金珠
4. **贝壳珠**：贝壳粉压制假珍珠，外观逼真
5. **珠层过薄**：养殖时间不足导致珠层薄，易剥落
6. **"天然"噱头**：声称"天然珍珠"但实际是养殖珠（真天然珠价格百倍于养殖珠）
