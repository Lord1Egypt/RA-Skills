# Habits-v2.md — 可追溯、可校验的优势强化基因库

---

## 格式规范

```markdown
## Habit-YYYY-MM-DD-XX
- **信号**：[触发场景]
- **行为**：[一句话描述做得好的行为]
- **启示**：[这个行为背后的原因/原则，≤2行]
- **验证状态**：[待复核/✅已确认/❌已作废]
- **价值级别**：[🟩重大/🟦常规/🟪突破]
- **类型**：[行为模式/单次亮点]
- **关联ID**：[同根因旧ID，可选]
- **晋升记录**：[已晋升为工作手册/SOUL条目，可选]
```

## 字段填写标准

| 字段 | 标准 |
|------|------|
| **类型** | 行为模式=同一模式多次出现；单次亮点=偶然突破不一定重复 |
| **价值级别** | 🟩重大=被用户明确认可/避免信任受损；🟦常规=正常好判断；🟪突破=主动超预期 |
| **验证状态** | 新条目默认「待复核」，用户确认后改为「✅已确认」 |

## 流转规则

- 第1次：生成 Habit 条目，标「待复核」
- 第2次：关联旧ID，同类型行为模式确认
- 第3次：累计3次 → 整理时生成「最佳实践草案」→ 汇报确认后写入工作手册/SOUL
