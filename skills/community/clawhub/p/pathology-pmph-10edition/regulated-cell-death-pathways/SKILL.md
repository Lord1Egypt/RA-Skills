# 程序性细胞死亡通路鉴别

## 适用场景
- 细胞死亡兼具凋亡与坏死特征
- 出现线粒体皱缩但无染色质浓缩
- 需选择特异性抑制剂进行干预

## 执行步骤

### 1. 判断是否为细胞焦亡（Pyroptosis）
- **触发条件**：病原体感染或危险信号激活炎症小体
- **关键执行蛋白**：GSDMD（被caspase-1/4/5/11切割）
- **形态特征**：
  - 细胞膜穿孔（10–20 nm微孔）
  - 细胞肿胀破裂
  - 同时具备TUNEL阳性（凋亡样）与膜破裂（坏死样）
- **功能后果**：释放IL-1β/IL-18，引发强烈炎症

### 2. 判断是否为铁死亡（Ferroptosis）
- **触发条件**：GSH耗竭 + GPX4失活 + Fe²⁺存在
- **核心机制**：脂质过氧化失控 → lipid ROS累积
- **形态特征**：
  - 线粒体皱缩、嵴减少、膜密度增加
  - **无**凋亡小体、核碎裂
- **特异性抑制**：铁螯合剂（去铁胺）、ferrostatin-1、维生素E

### 3. 判断是否涉及自噬依赖性死亡
- **定义**：依赖ATG基因和自噬体形成的程序性死亡
- **结构标志**：自噬体（双层膜囊泡）、自噬溶酶体
- **注意**：自噬通常为保护性过程，仅在特定条件下导致死亡

> **排除原则**：凋亡抑制剂（如Z-VAD）对焦亡和铁死亡无效；焦亡/铁死亡抑制剂对凋亡无效。
