# 传出神经递质释放的量子化机制与胞裂外排

## 何时使用
- 需解释神经冲动如何触发递质释放并引发突触后反应
- 分析终板电位的产生机制（如微小终板电位 vs. 动作电位阈值）
- 理解特定神经毒素或药物对递质释放的抑制或促进作用
- 阐明共传递（cotransmission）现象的细胞基础

## 执行步骤

### 1. 胞裂外排（Exocytosis）机制
- **触发条件**：神经冲动到达末梢 → 突触前膜去极化 → 电压门控钙通道开放 → Ca²⁺内流。
- **融合过程**：Ca²⁺结合传感器蛋白（如synaptotagmin），促进囊泡膜蛋白（VAMPs）与突触前膜蛋白（SNAPs）相互作用，形成融合孔。
- **内容物释放**：囊泡内递质（如乙酰胆碱ACh或去甲肾上腺素NA）一次性全部排入突触间隙。
- **药理干预**：
  - 肉毒杆菌毒素：切割VAMPs，阻断胆碱能神经囊泡融合。
  - 溴苄铵/胍乙啶：抑制去甲肾上腺素能神经的胞裂外排。

### 2. 量子化释放（Quantal Release）学说
- **基本单位**：“量子” = 单个囊泡所含递质量（约5000个ACh分子，范围1000–50000）。
- **静息状态**：少量囊泡自发释放 → 微小终板电位（0.3–3.0 mV），不足以触发动作电位。
- **兴奋状态**：神经冲动引发≥100个囊泡同步释放 → 终板电位总和超过阈值 → 肌纤维产生动作电位和收缩。

### 3. 其他释放机制与共传递
- **静息溢出**：交感神经末梢有微量NA从囊泡渗漏，但无生理效应。
- **药物诱导释放**：酪胺、苯丙胺等被摄取入囊泡，置换储存的NA，导致非生理性大量释放。
- **共传递现象**：单个神经末梢可同时释放2–3种递质（如NA + ATP + 神经肽Y）。

> 注意：本机制不适用于非囊泡释放的递质（如一氧化氮NO），且量子化概念主要基于骨骼肌终板电位的电生理记录。
