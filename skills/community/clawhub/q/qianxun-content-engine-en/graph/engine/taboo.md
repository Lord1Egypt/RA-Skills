# Taboo & Compliance

> Words / phrases / practices to never use in copy. The generate mode will reference this.

## Platform compliance

### Superlatives (banned across platforms)
- best / first / only / top-tier / extreme / national-grade / world-class
- 100% / absolute / permanent / lifetime
- "magic" / "ultimate" / "lifesaver"

### Medical/efficacy claims (cosmetics, skincare, food — be careful)
- treat / cure / therapeutic
- anti-aging / anti-wrinkle (unless registered)
- whitening (regulated by category)
- weight loss / fat-burning (unless drug-licensed)

### Financial
- guaranteed / risk-free
- get-rich / financial freedom

## Brand taboos

> Not compliance issues — just things "our brand shouldn't say".

### Words that lower brand position
> Reference [[brand/brand-voice]] "use sparingly" + stricter "banned" list here
- _e.g., "viral", trendy slang, "must-buy", "begging you"_
- _e.g., "value-for-money" (if you're high-end, use sparingly)_

### Promises we don't make
- _e.g., don't promise "lose X pounds"; just say "slimming silhouette"_
- _e.g., don't promise "skin transformation"; just say "feel on application"_
