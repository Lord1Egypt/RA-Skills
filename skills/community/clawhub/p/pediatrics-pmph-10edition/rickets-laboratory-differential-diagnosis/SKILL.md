# 佝偻病实验室鉴别诊断

## 适用对象
- 疑似佝偻病患儿
- 对常规维生素D治疗无效者

## 使用前提
- 已确认存在佝偻病体征
- 已完成基础实验室检查（包括血钙、血磷、碱性磷酸酶等）

## 禁忌场景
- 非佝偻病性骨骼畸形（如成骨不全、软骨发育不良）

## 执行流程
1. **收集关键实验室指标**：血钙、血磷、碱性磷酸酶、25-(OH)D3、1,25-(OH)2D3、PTH、尿磷、血钾、血氨、尿pH及氨基酸尿结果。
2. **比对各型佝偻病特征模式**（详见参考表）：
   - 维生素D缺乏性佝偻病
   - 家族性低磷血症（抗维生素D佝偻病）
   - 远端肾小管性酸中毒
   - 维生素D依赖性佝偻病（I型与II型）
   - 肾性佝偻病
   - 肝性佝偻病
3. **识别关键区分点**：
   - 血磷水平（显著降低 vs 正常 vs 升高）
   - 25-(OH)D3 与 1,25-(OH)2D3 的相对变化
   - 是否存在氨基酸尿、低钾、碱性尿等伴随表现
   - 基础疾病史（肝病、肾病等）
4. **输出最可能的亚型诊断**，并给出针对性治疗方向建议。

> 注意：除肾性佝偻病外，其他类型通常伴有碱性磷酸酶升高。
