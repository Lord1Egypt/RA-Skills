# 中国九市0～36月龄儿童体格发育参考值使用指南

## 何时使用
- 需评估0～36月龄中国健康足月儿童的体格发育状况
- 儿童无严重慢性疾病且非早产儿
- 已知儿童性别、精确月龄及居住地类型（城区/郊区）

## 执行步骤
1. **确认适用对象**：确保儿童为0～36月龄、足月出生、无严重慢性疾病。
2. **确定关键参数**：
   - 性别（男/女）
   - 精确月龄并归入标准年龄区间（如0～<1、1～<2……30～<36月）
   - 居住地类型（城区或郊区）
3. **查询参考值**：根据上述参数，在对应表格中查找以下三项指标的均值±标准差：
   - 体重（kg）
   - 身长（cm）
   - 头围（cm）
4. **注意数据限制**：郊区0～<1月龄所有指标标记为“—”，表示无参考值。
5. **临床解读**：
   - 本表仅提供±1个标准差（±1s）数据，临床通常以±2s作为正常范围边界
   - 应结合多次测量形成的生长曲线趋势进行综合判断，而非依赖单次测量值

> ⚠️ 此参考值不适用于早产儿、低出生体重儿或患有影响生长发育的慢性疾病的儿童。
