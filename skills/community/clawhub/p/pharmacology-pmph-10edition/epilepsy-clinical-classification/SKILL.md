# Epilepsy Clinical Classification

## When to Use
- 患者有明确的反复癫痫样发作症状（如抽搐、意识丧失、感觉异常等）
- 已完成脑电图（EEG）检查并可获取结果
- 需排除非癫痫性事件（如晕厥、心因性非癫痫发作）

## How to Execute

1. **判断发作是否局限于大脑局部区域**：
   - 若发作仅表现为局部肢体运动或感觉异常，且意识清楚 → 归类为「局限性发作」
     - **单纯性局限性发作**：局部症状，持续20～60秒，意识保留
     - **复合性局限性发作（精神运动性发作）**：伴有意识障碍，出现无意识自动症（如唇抽动、摸索动作），病灶多位于颞叶或额叶，持续30秒～2分钟

2. **若发作涉及全脑并导致意识丧失** → 归类为「全身性发作」
   - **失神发作（小发作）**：多见于儿童，意识突然丧失5～30秒，EEG呈3Hz高幅对称同步棘慢波
   - **肌阵挛性发作**：部分肌群出现约1秒的休克样抽动，伴意识丧失，EEG呈短暂暴发性多棘波
   - **强直-阵挛性发作（大发作）**：意识丧失、全身强直后阵挛抽搐、口吐白沫、牙关紧闭，持续数分钟，EEG呈高幅棘慢波
   - **癫痫持续状态**：大发作反复持续或长时间昏迷，属急症，需立即干预

3. **结合辅助检查指导病因判断**：
   - 局限性发作（约占60%）常由结构性病变引起（如脑寄生虫、血管畸形、肿瘤、外伤），建议行CT/MRI检查
   - 全身性发作（约占40%）多与遗传因素相关（如青少年肌阵挛性癫痫）

## Output
返回具体的癫痫发作类型（例如“单纯性局限性发作”或“强直-阵挛性发作”），用于指导抗癫痫药物的选择和治疗方案制定。
