# concept-anatomy 处理流程

> **版本**：v1.0.0 | 2026-06-26
> **目的**：详细描述 8 维度处理流程，确保解剖完整深入

---

## 完整处理流程（8 步）

### Step 1：历史维度

**目标**：追溯概念的诞生背景、起源时间、提出者、最初解决的问题。

#### 关键问题
- 这个概念什么时候、谁提出来的？
- 提出的背景是什么（学科/事件/未解决的问题）？
- 最初是用来解决什么问题的？
- 概念本身经历了哪些演变？

#### 输出格式
```
历史：
- 提出者：[人名/学派/无明确提出者]
- 时间：[年份或时期]
- 背景：[学科/事件/社会背景]
- 解决：[最初要解决的问题]
- 演变：[后来的关键发展]
```

#### 投资特化
对投资类概念，还要补充：
- 金融市场历史上的关键节点（如"期权"→ 1973 年 CBOE 成立）
- 推动概念发展的标志性事件/危机

---

### Step 2：辩证维度

**目标**：找到概念的对立面/反面/镜像，理解概念的边界。

#### 关键问题
- 这个概念的对立面是什么？（与谁对立就与谁互为定义）
- 正反合是什么？（黑格尔辩证法的"正题-反题-合题"）
- 概念的"反例"是什么？
- 概念的"过度使用"会带来什么？

#### 输出格式
```
辩证：vs [对立概念]
- 正：[本概念的正面/价值]
- 反：[对立概念的正面/价值]
- 合：[更高层次的统一，如"阴阳"统一为"道"]
- 反例：[不符合本概念的场景]
- 过度：[过度强调本概念会怎样]
```

#### 投资特化
- 概念 vs 反义概念（如"风险"vs"收益"、"集中"vs"分散"）
- 投资中的"度"：过少/过多都会失效

---

### Step 3：现象维度

**目标**：在日常生活/工作中，找到 3 个能直接观察/经历的具体表现。

#### 关键问题
- 这个概念在日常生活中有什么表现？
- 哪些现象/事件/案例是这个概念的实例？
- 普通人怎么"看到"这个概念？

#### 输出格式
```
现象：
① [具体现象 1] — [为什么这是这个概念]
② [具体现象 2] — [为什么这是这个概念]
③ [具体现象 3] — [为什么这是这个概念]
```

#### 选现象的原则
- ✅ 普通人能直接观察/经历
- ✅ 不同领域（生活/工作/自然）
- ✅ 具象而非抽象

#### 投资特化
- 3 个市场现象（如"反脆弱"→ 疫情期表现好的资产）

---

### Step 4：语言维度

**目标**：拆解概念的文字构成（字/词根），找到同义词/反义词。

#### 关键问题
- 这个词由哪些字/词根构成？
- 每个字/词根的原意是什么？
- 同义词有哪些？反义词有哪些？
- 在不同语境下意思有变化吗？

#### 输出格式
```
语言：
- 字/词根：[拆解] → [原意]
- 同义词：[词1]、[词2]
- 反义词：[词1]、[词2]
- 语境变化：[在 X 语境下偏向 Y 含义]
```

#### 拆解方法
- 中文：拆字 + 查词典
- 英文：拆词根（希腊/拉丁词源）
- 专业术语：看该领域的标准定义

---

### Step 5：形式维度

**目标**：把概念抽象为数学/逻辑/图形表示，看清其精确结构。

#### 关键问题
- 能用数学公式/逻辑表达式/算法表示吗？
- 能用图形/图表/流程图表示吗？
- 关键变量是什么？变量之间的关系是什么？
- 边界条件/极端值是什么？

#### 输出格式
```
形式：
- 数学/逻辑表示：[公式/表达式/算法]
- 图形表示：[ASCII 图/结构图]
- 关键变量：[变量 1]、[变量 2]
- 变量关系：[正相关/负相关/复杂]
- 边界条件：[极端情况 1]、[极端情况 2]
```

#### 投资特化
- 用金融公式表达（如夏普比率、VaR、Beta）
- 用图表展示（如"风险-收益"散点图）

---

### Step 6：存在维度

**目标**：明确概念存在/成立的前提条件，以及失效/消失的边界。

#### 关键问题
- 什么前提/条件下这个概念存在？
- 概念消失/失效的边界条件是什么？
- 概念存在的时间/空间/范围？
- 概念的"生命周期"是什么？

#### 输出格式
```
存在：
- 存在条件：[条件 1] + [条件 2] + [条件 3]
- 消失条件：[条件 1 缺失] → [概念失效]
- 时间范围：[长期/中期/短期]
- 空间范围：[个人/组织/市场/宇宙]
- 生命周期：[诞生-发展-成熟-消亡]
```

#### 投资特化
- 概念成立的市场环境（牛/熊/震荡）
- 概念失效的临界点

---

### Step 7：美感维度

**目标**：找到概念在美学/艺术/设计/自然中的对应物，理解其"美"。

#### 关键问题
- 这个概念在艺术作品中的对应物是什么？
- 在设计中如何体现？
- 在自然中有类似的"美"吗？
- 概念的"对称性"、"简洁性"、"和谐性"在哪？

#### 输出格式
```
美感：
- 艺术对应：[艺术作品/艺术家] 体现了这个概念
- 设计对应：[设计案例/设计原则]
- 自然对应：[自然现象] 中有这个概念
- 美之所在：[简洁性/对称性/和谐性/...]
```

#### 重要性
- 美感是"深层理解"的标志
- 能用美感表达的概念，说明已经内化
- 也有助于长期记忆

---

### Step 8：元反思维度

**目标**：识别自己/他人对这个概念的常见误解，反思理解的盲区。

#### 关键问题
- 我对这个概念的理解可能错在哪？
- 其他人常见的误解是什么？
- 有没有"听起来对但其实错"的描述？
- 概念的"反直觉"特性是什么？

#### 输出格式
```
元反思：
- 常见误解1：[误解] — [为什么错]
- 常见误解2：[误解] — [为什么错]
- 反直觉点：[与直觉相反的特性]
- 自我盲区：[我可能漏掉的维度]
```

#### 价值
- 这是"知道 vs 理解"的分水岭
- 能识别误解 = 真理解
- 防止"达克效应"（不知道自己不知道）

---

## Final：核心顿悟

**目标**：把 8 个维度综合为 1 句不超过 30 字的核心顿悟。

### 公式

```
核心顿悟 = [概念本质] = [最简化的洞察]
```

### 评判标准

- [ ] 不超过 30 字
- [ ] 包含了 8 个维度的核心
- [ ] 有"啊哈"感（让人立刻明白）
- [ ] 可作为"概念名片"使用
- [ ] 不需要再展开就能传达本质

### 反例

- ❌ "这是一个重要的概念"（空洞）
- ❌ "它有很多维度"（没说什么）
- ❌ "它是 X 的反义词"（太简单）
- ✅ "在压力下变强，而非仅仅是抗压"（反脆弱）
- ✅ "用小钱锁定大钱的不对称下注"（期权）

---

## 投资概念特化模式

**触发条件**：用户启用 `-mode=investment` 或明确表示"投资概念"

### 调整点
| 维度 | 普通模式 | 投资特化模式 |
|------|---------|------------|
| 历史 | 学科起源 | + 金融市场历史 |
| 辩证 | 一般对立 | + 风险-收益/集中-分散 |
| 现象 | 日常现象 | + 3 个市场现象 |
| 形式 | 一般抽象 | + 金融公式/图表 |
| 存在 | 一般条件 | + 牛/熊/震荡市场 |
| 美感 | 艺术对应 | + 投资美学（不对称/复利） |
| 元反思 | 一般误解 | + 投资误区（幸存者偏差等） |

### 核心顿悟偏向
- 普通模式：通用洞察
- 投资特化：投资决策导向（杠杆点/风险边界）

---

## 失败回退

如果某个维度无法填充：
- **历史缺失**：概念太新/太冷门，标注"该领域较新，文献有限"
- **形式缺失**：概念偏定性（如"美感"），用隐喻代替
- **美感缺失**：实用性概念不需要美感维度，标注"该概念为工具性，无需美感维度"
- **元反思缺失**：直接基于"非黑即白"判断，标注"该概念较成熟，无重大争议"

任何维度缺失**不影响整体输出**，但要诚实标注。
