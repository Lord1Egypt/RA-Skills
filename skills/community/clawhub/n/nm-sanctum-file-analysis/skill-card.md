## Description: <br>
Maps file structure and module organization of a codebase for architecture reviews, refactoring planning, and migration scope estimation. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[athola](https://clawhub.ai/user/athola) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineers use this skill to quickly map repository structure, identify project organization patterns, and capture file-level hotspots before review or refactoring work. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can activate on broad analysis or exploration requests and summarize local file structure in sensitive repositories. <br>
Mitigation: Invoke it explicitly or narrow activation triggers when working in sensitive repositories, and review generated structural summaries before sharing them. <br>
