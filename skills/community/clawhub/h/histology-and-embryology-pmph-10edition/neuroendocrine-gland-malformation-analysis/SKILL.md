# 神经系统相关内分泌腺畸形分析

## 何时使用
- 患者有面部/腭部先天畸形伴内分泌功能低下
- 影像学提示前脑结构异常或鞍区占位
- 家族史提示遗传性神经内分泌发育障碍
- 排除后天获得性内分泌疾病及孤立性甲状腺病变

## 执行步骤
1. **评估前脑泡闭合状态**：
   - 若未正常闭合，则考虑前脑缺损、垂体发育不良或缺如，并评估是否继发甲状腺、肾上腺或性腺发育不良。
2. **检查第三脑室底部发育情况**：
   - 若存在发育障碍，警惕漏斗瘤（infundibular tumor），尤其在儿童期发病者。
3. **确认颅咽管退化状态**：
   - 若未退化且异常增殖，诊断为颅咽管瘤（craniopharyngioma），并评估垂体功能低下或下丘脑综合征表现。
4. **排查副肾上腺**：
   - 在主肾上腺附近寻找仅有皮质无髓质的额外腺体组织，确认是否为超数发生性畸形。

> 注意：本技能仅适用于先天性、胚胎源性的神经内分泌结构畸形，不适用于后天获得性疾病。
