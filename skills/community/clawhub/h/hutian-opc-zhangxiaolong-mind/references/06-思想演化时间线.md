# 张小龙 · 思想演化时间线

## 一、人物基本信息

| 项目 | 内容 |
|------|------|

## 二、Foxmail时期（1997-2005）

### 2.1 创业起点
- 1997年：开发Foxmail
- 一个人完成所有开发
- 产品口碑好但商业化差

### 2.2 核心特质
```
一人开发者
    ↓
极致产品
    ↓
忽视商业化
    ↓
被资本选择
```

### 2.3 这一时期的教训
- 好产品不等于好商业
- 产品经理需要商业思维
- 但核心价值观已经形成

## 三、QQ邮箱时期（2005-2010）

### 3.1 转折点
- 2005年：Foxmail被腾讯收购
- 加入腾讯，负责QQ邮箱
- 从个人开发者到团队领导

### 3.2 QQ邮箱的蜕变
```
早期：平庸产品
    ↓
中后期：业界领先
    ↓
转折：漂流瓶功能
    ↓
成功：邮箱与社交结合
```

### 3.3 核心能力发展
- 团队管理
- 产品迭代
- 用户洞察
- 平衡商业与体验

## 四、微信诞生（2010-2012）

### 4.1 创立背景
- 2010年：张小龙向马化腾提议做移动社交
- "微信"的命名
- 广州研发中心秘密开发

### 4.2 微信1.0
```
2011年1月：微信发布
    ↓
核心功能：语音
    ↓
用户增长缓慢
    ↓
张小龙：方向是对的，继续迭代
```

### 4.3 关键迭代
| 版本 | 时间 | 关键功能 |
|------|------|----------|
| 1.0 | 2011.1 | 语音消息 |
| 2.0 | 2011.5 | 语音漂流瓶 |
| 2.5 | 2011.8 | 附近的人 |
| 3.0 | 2011.10 | 摇一摇 |
| 4.0 | 2012.4 | 朋友圈 |

### 4.4 微信崛起
- 2012年3月：用户破亿
- 2012年9月：用户破2亿
- 微信成为中国社交基础设施

## 五、微信扩张期（2012-2017）

### 5.1 功能矩阵形成
```
2012：公众号
2013：微信支付
2014：微信红包
2015：朋友圈广告
2016：小程序
```

### 5.2 关键决策
| 决策 | 内容 | 意义 |
|------|------|------|
| 不做开屏广告 | 放弃短期收入 | 建立用户信任 |
| 去中心化 | 不给中心流量 | 生态繁荣 |
| 小程序 | 超级App平台 | 生态扩张 |
| 克制 | 不过度运营 | 用户体验 |

### 5.3 商业化探索
- 朋友圈广告（谨慎）
- 小程序生态
- 微信支付（成功）
- 公众号（生态）

## 六、微信成熟期（2017-2021）

### 6.1 微信7.0
- 2018年12月：微信7.0发布
- 界面重大改版
- 用户争议巨大
- 张小龙坚持："方向要大致正确"

### 6.2 视频号
```
2020年1月：视频号内测
    ↓
张小龙反思：纯算法推荐不够
    ↓
调整：社交推荐为主
    ↓
成功：成为短视频重要平台
```

### 6.3 十年演讲
- 2021年1月：微信十年
- 4小时公开课
- 完整阐述产品价值观
- "微信在做底层的连接"
