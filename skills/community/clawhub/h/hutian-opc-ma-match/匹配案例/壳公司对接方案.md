### 1.2 沟通话术要点

#### 初次接触开场白（电话/邮件通用版）

```
（电话场景）
您好，请问是凌秘书吗？我是XX资本的并购负责人，我们专注于化工新材料领域的产业整合。

（重点来了）我们在华东地区寻找一些符合公司战略布局的并购标的，
看到先锋新材在化工新材料领域有一定积累，想了解贵司是否有
对外战略合作或并购整合的意愿？

我们不是来做二级市场的，就是看看有没有产业协同的机会。
```

**关键点**：不提"买壳"，不提"控制权转让"，只用"产业整合"和"战略合作"包装。

#### 试探卖壳意愿的进阶话术

在第一次接触后1-2周，如对方表现出开放态度，可推进：

```
（第二次沟通）
我们研究下来，先锋新材的主业这几年承压比较明显，
不知道管理层的战略规划是继续坚守还是有所调整？

我们也在看一些类似的公司，比如XX公司最近就做了控制权的变更，
想了解一下行业同仁有没有什么动向？

（观察反应：若对方主动追问"你说的XX公司是什么情况"，
说明有开放意愿；若对方回避话题，说明时机未到）
```

#### 评估5100万关联担保风险的标准问法

```
我们做尽调时看到公司有5100万的关联担保，这个担保
是历史遗留的还是还在存续期的？

对手方是？有没有具体的到期安排？

如果需要的话，我们可以协助引入担保资源来置换这个担保，
减轻上市公司负担——不知道管理层有没有这个需求？
```

**话术逻辑**：不直接问"这个担保有没有风险"，而是提出"帮忙解决"，试探对方对风险的认知程度。

#### 评估信披违规处罚影响的标准问法

```
我们注意到公司之前有一些信披方面的监管记录，
想了解一下目前的状态——处罚是否已经落地？

如果后续有再融资或重大资产重组的需求，
这个事项会不会构成障碍？
```

### 1.3 尽调重点清单

#### 优先级P0（必须核查，否则不推进）

| 尽调事项 | 核查方法 | 核查标准 |
|----------|----------|----------|
| 5100万关联担保详情 | 要求提供担保合同、债权人信息、到期日 | 确认担保是否已解除或是否有置换方案 |
| 信披违规处罚状态 | 查询证监会/交易所官网、公司公告 | 确认处罚是否已落地，是否处于立案调查期 |
| 原实控人卢先锋债务 | 中国执行信息公开网、企查查 | 确认卢先锋是否有未了结的大额债务 |
| 立案调查期 | 咨询律师确认 | 如处于调查期，重组受限，无法推进 |

#### 优先级P1（影响交易对价）

| 尽调事项 | 核查方法 | 核查标准 |
|----------|----------|----------|
| 剩余资产质量 | 审计报告 + 实地盘点 | 清壳后剩余资产能否支撑上市地位 |
| 员工安置 | HR访谈 + 劳动合同统计 | 多少人、补偿成本多大、是否有劳资纠纷 |
| 或有负债 | 诉讼查询 + 律师意见 | 是否有未披露的诉讼/仲裁/担保 |

#### 优先级P2（影响交易结构设计）

| 尽调事项 | 核查方法 | 核查标准 |
|----------|----------|----------|
| 股权结构 | 中登公司 + 股东名册 | 前十大股东持股比例、是否有代持 |
| 董事会构成 | 公司章程 + 董事会名单 | 改选董事会的难度 |
| 重组承诺限制 | 问询函回复 + 律师意见 | 多久内不能做重大资产重组 |

### 1.4 交易结构初步方案

#### 方案一：协议转让（最推荐）

| 要素 | 内容 |
|------|------|
| 转让标的 | 第一大股东或一致行动人持有的股份 |
| 转让比例 | 29.9%（踩线触发要约收购）或以上 |
| 转让价格 | 市价溢价20%-40%（参考同类案例） |
| 付款安排 | 预付30%+过户后付70%或分期 |
| 前提条件 | 解除关联担保、处罚落地、债务清理 |

#### 方案二：表决权委托 + 后续认购定增

| 要素 | 内容 |
|------|------|
| 第一步 | 原股东将29.9%股份表决权委托给收购方 |
| 第二步 | 收购方主导定增，认购后持股比例超过30% |
| 优势 | 资金压力小，可绕过老股转让的税负 |
| 风险 | 原股东保留股份，可能后续出售形成竞争 |

#### 方案三：资产置换 + 表决权转让（最复杂）

适用于收购方有优质资产需要注入的场景：

```
第一步：收购方将资产注入上市公司
第二步：上市公司原资产置出
第三步：原股东获得资产对价 + 部分现金
第四步：原股东股份表决权转让给收购方
```

**估值区间参考**：
- 当前市值：23亿（截至尽调基准日）
- 壳资源溢价：约3-5亿（参考市场行情）
- 控制权转让价：预估4.5-6亿（需谈判）

### 1.5 风险清单与应对措施

| 风险项 | 风险描述 | 应对措施 |
|--------|----------|----------|
| **风险1：信披违规处罚导致重组受限** | 如处于立案调查期，证监会限制重大资产重组 | 先核查处罚状态；如在调查期，放弃推进或等处罚落地后再谈 |
| **风险2：无实控人状态下决策效率低** | 无实控人意味着没有"拍板人"，谈判对象不明确 | 先明确郎海涛等主要股东的意图，锁定关键少数；谈判时要求对方出具股东会授权 |
| **风险3：5100万担保敞口风险** | 如担保方违约，上市公司需承担连带责任 | 要求原股东提供反担保或现金抵押；要求在交割前解除担保 |
| **风险4：原实控人卢先锋遗留问题** | 债务纠纷、诉讼、关联交易未清理干净 | 通过企查查、中国执行网全面排查；要求原股东出具兜底承诺 |
| **风险5：清壳不彻底，注入资产受限** | 清壳后剩余资产质量差，后续重组空间小 | 实地盘点剩余资产；与管理层确认清壳计划；必要时在协议中约定清壳完成标准 |

## 二、森霸传感(300701)对接方案

**推荐优先级**：⭐⭐⭐（中等，卖壳意愿不明确）  
**核心逻辑**：实控人单森林家族减持中，但尚未明确转让控制权意图

### 2.1 对接路径

#### 路径A：实控人家族直接渠道

| 身份 | 姓名 | 备注 |
|------|------|------|
| 实控人 | 单森林 | 家族持股约45% |
| 关键人 | 子女/配偶 | 可通过商务活动接触 |

**注意**：实控人家族减持正在进行中，说明有退出意愿，但可能尚未下定决心卖壳。接触时要有耐心，不可操之过急。

#### 路径B：机构股东渠道

通过前十大股东中的机构股东（如公募基金、私募）了解减持进展和实控人态度。

### 2.2 沟通策略

#### 核心策略：观察减持意图是否升级

```
（初次接触，以产业合作为名）
我们注意到森霸传感在传感器领域有一定积累，
想了解公司是否有引入战略投资者或对外并购的规划。

（观察点：对方是否主动提到"实控人有退出想法"，
或是否表现出积极回应）
```

#### 格林通业绩补偿问题处理

格林通是森霸传感收购的子公司，业绩承诺期已过但可能存在未完成的业绩补偿。

| 问题 | 处理方式 |
|------|----------|
| 业绩补偿是否完成 | 要求公司提供业绩承诺实现情况和补偿执行公告 |
| 补偿未完成的法律后果 | 咨询律师，确认是否影响控制权转让 |
| 谈判策略 | 可将"解决格林通遗留问题"作为谈判筹码，争取更好的交易条件 |

### 2.3 交易结构方案

| 方案 | 适用场景 | 关键要素 |
|------|----------|----------|
| 协议转让 | 实控人明确卖壳意愿 | 家族持股45%，需转让30%+获得控制权 |
| 大宗交易 | 短期无法完成协议 | 通过大宗交易分批接货 |
| 定增 | 收购方有资产需注入 | 认购定增后持股超过30% |

**估值区间**：森霸传感市值约30亿+，主业有一定盈利能力，卖壳估值可能高于先锋新材。

## 三、德固特(300950)对接方案

**推荐优先级**：⭐⭐（低，长期跟踪）  
**核心逻辑**：主业稳健，无卖壳意愿，但需保持跟踪

### 3.1 监控指标（什么信号出现说明开始有卖壳意愿）

| 监控指标 | 信号含义 |
|----------|----------|
| 实控人开始减持 | 减持至30%以下为关键信号 |
| 董监高频繁更换 | 核心管理层离职可能预示战略调整 |
| 主营业务连续亏损 | 连续2年以上亏损可能倒逼卖壳 |
| 引入外部投资者 | 战投进入后可能带来控制权变更机会 |
| 公告"战略合作"但无实质内容 | 可能是卖壳前的铺垫 |

### 3.2 主动接触策略

```
（以业务合作为切入点）
我们看到德固特在高温换热领域技术领先，
我们在XXX领域有一些资源想看看能否合作。

（不主动提买壳，观察公司反应：
如果公司对业务合作表现积极，说明管理层开放；
如果公司敷衍回应，说明目前没有需求）
```

**建议动作**：每季度更新一次公司公告和股东变动情况，保持信息跟踪。
