# 稅種詳細規定與計算公式

## 1. 房地合一稅（105/1/1 後取得）

### 適用對象
- 2016年1月1日（含）以後取得之房地產
- 2014年1月2日以後取得，且持有未滿2年

### 稅率結構

| 持有期間 | 一般稅率 | 自住房地稅率 |
|----------|:--------:|:-----------:|
| 2年以內 | 45% | — |
| 2~5年 | 35% | — |
| 5~10年 | 20% | — |
| 10年以上 | 15% | — |
| 自住滿6年 | — | 10%（免稅額400萬）|

### 計算公式
```
稅基 ＝ 成交價 － 取得成本 － 相關費用 － 土地漲價總數額
稅額 ＝ 稅基 × 稅率
```

### 扣除項目
- 取得成本：買入價＋契稅＋代書費＋規費＋仲介費
- 相關費用：仲介費（賣出）＋廣告費＋清潔費＋搬運費
  - 以上合計若>成交價5%，仍以5%為上限
  - 若無單據，可直接扣除成交價3%（文件費）
- 土地漲價總數額：土增稅核定之漲價數額

### 申報時限
- 移轉登記完畢次日起**30日內**
- 未申報罰則：滯納金 + 補稅

## 2. 土地增值稅

### 適用對象
- 土地所有權移轉時（買賣、贈與、交換）
- 設定典權時

### 稅率結構

| 級距 | 一般稅率 | 自用住宅稅率 |
|------|:--------:|:-----------:|
| 第1級（漲價倍數<100%）| 20% | 10% |
| 第2級（100%~200%）| 30% | 10% |
| 第3級（>200%）| 40% | 10% |

### 自用住宅優惠
- 適用要件：
  1. 所有權人出售前1年無出租/營業
  2. 地上建物為所有權人或其配偶/直系親屬自住
  3. 都市土地≤3公畝，非都市≤7公畝
- 一生一次（或一生一屋）
- 優惠稅率：**10%**

### 計算公式
```
漲價總數額 ＝ 申報移轉現值 － 前次移轉現值 × （台灣地區消費者物價指數／100）－ 改良費用
稅額 ＝ 漲價總數額 × 稅率 － 累進差額
```

## 3. 財產交易所得稅（105/1/1 前取得）

### 適用對象
- 2015年12月31日（含）以前取得之房地產

### 計算方式

| 方式 | 說明 |
|------|------|
| 核實認定 | 實際成交價－實際取得成本－相關費用 |
| 標準認定 | 房屋評定現值 × 財政部核定所得標準率（各縣市不同）|

### 各縣市所得標準率（參考值，每年公告）

| 區域 | 標準率 |
|------|:------:|
| 台北市 | 約 41%~45%（依路段）|
| 新北市 | 約 25%~35% |
| 桃園市 | 約 18%~25% |
| 台中市 | 約 15%~23% |
| 高雄市 | 約 13%~20% |
| 其他縣市 | 約 8%~15% |

## 4. 契稅

### 適用對象
- 買賣、承典、交換、贈與、分割、占有

### 稅率

| 原因 | 稅率 |
|------|:----:|
| 買賣 | 6% |
| 典權 | 4% |
| 交換 | 2% |
| 贈與 | 6% |
| 分割 | 2% |
| 占有 | 6% |

### 計算公式
```
契稅 ＝ 申報契價（房屋評定現值）× 稅率
```

## 5. 房屋稅

### 適用對象
- 每年繳納，持有房屋者

### 稅率結構

| 用途 | 法定稅率 | 實際徵收率 |
|------|:--------:|:---------:|
| 自住住家 | 1.2%~2% | 1.2% |
| 非自住住家 | 1.5%~3.6% | 各縣市不同（1.5~3.6%）|
| 營業用 | 3%~5% | 3% |
| 非住非營 | 1.5%~2.5% | 2% |

### 計算公式
```
房屋稅 ＝ 房屋評定現值 × 稅率 × 持有月數／12
```

## 6. 地價稅

### 適用對象
- 每年繳納，持有土地者

### 一般稅率
- 基本稅率：**10‰**
- 累進稅率：10‰~55‰（依地價總額累進）

### 自用住宅稅率
- **2‰**（優惠極大！）
- 適用要件：
  1. 戶籍登記
  2. 無出租或營業
  3. 都市土地≤3公畝，非都市≤7公畝
  4. 申請截止日：9月22日前

### 計算公式
```
地價稅 ＝ 申報地價總額 × 稅率 − 累進差額
```

## 7. 贈與稅

### 免稅額
- 每年：**244萬**（2025年起調整）
- 配偶贈與：免稅

### 稅率
- 淨額 2,500萬以下：10%
- 2,500萬~5,000萬：15%（累進差額125萬）
- 5,000萬以上：20%（累進差額375萬）

### 房地贈與稅計算
```
贈與淨額 ＝ 土地公告現值＋房屋評定現值 － 免稅額244萬
贈與稅 ＝ 贈與淨額 × 稅率 － 累進差額
```
*實務上贈與房產非常不划算（土增稅+契稅+贈與稅三重課稅）*

## 8. 遺產稅

### 免稅額
- **1,333萬**（2025年起調整）

### 扣除額
- 配偶扣除額：553萬
- 直系卑親屬扣除額：56萬/人
- 父母扣除額：138萬/人
- 喪葬費扣除額：138萬

### 稅率
- 淨額 5,000萬以下：10%
- 5,000萬~1億：15%（累進差額250萬）
- 1億以上：20%（累進差額750萬）
