## What It Does

### Core Features

- **Auto-Archive**: Every conversation is automatically recorded — no manual "remember this"
- **Semantic Search**: Say "what did we decide about the database?" — instantly recalled
- **Permanent**: Written to MEMORY.md — survives restarts, device changes, 1 month or 1 year later
- **Never Overwrite**: Append-only — same topic accumulates multiple versions, never lost
- **Daily Sharding**: Raw logs in `memory/YYYY-MM-DD.md`, never slows loading

### Problems It Solves

❌ "What was that plan we agreed on last time?"
❌ "This AI feels like a new person every session"
❌ "Switched accounts, all memory gone"
❌ "Memory file grows infinitely, takes forever to load"

✅ "You remembered my preference from last week — perfect"
✅ "Found a decision from a year ago in 3 seconds"

### How It Works

```
MEMORY.md (<50KB)  ← Refined essence, always fast
memory/YYYY-MM-DD.md  ← Date-sharded raw logs, never bloated
```
