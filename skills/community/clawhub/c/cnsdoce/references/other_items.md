# 其他项目费计算规则（山东安装工程）

> **适用标准**：
> - GB/T 50500-2024《建设工程工程量清单计价标准》
> - 《山东省建设工程费用项目组成及计算规则》2025年版
>
> **对标对象**：广联达GCCP5.0"其他项目"模块
> **配套文件**：cost_composition.md（费用构成总框架）、fee_standard.md（取费标准）

---

## 一、其他项目费构成

```
其他项目费
├── 暂列金额          ── 招标人给定，结算按实调整
├── 暂估价
│   ├── 材料暂估价    ── 招标人给定单价，计入综合单价
│   └── 专业工程暂估价 ── 招标人给定金额，另计
├── 计日工            ── 招标人给定暂定数量×综合单价
├── 总承包服务费      ── 按分包工程费的比例计取
└── 索赔与现场签证    ── 结算阶段按实发生计取
```

---

## 二、暂列金额

### 2.1 定义

暂列金额是招标人在工程量清单中暂定并包括在合同价款中的一笔款项。用于施工合同签订时尚未确定或者不可预见的所需材料、设备、服务的采购，施工中可能发生的工程变更、合同约定调整因素出现时的工程价款调整以及发生的索赔、现场签证确认等的费用。

### 2.2 计取规则

| 项目 | 内容 |
|------|------|
| 计取方式 | 按招标文件给定金额直接列入 |
| 计入位置 | 其他项目费 → 暂列金额 |
| 是否计取管理费/利润 | 否 |
| 是否计取税金 | 是（进入税前造价后统一计税） |
| 结算处理 | 按实际发生调整，余额归发包人，不足由发包人补足 |

### 2.3 参考费率

| 工程类别 | 暂列金额参考比例（分部分项工程费） |
|---------|-------------------------------|
| 工业管道安装 | 3%~5% |
| 给排水/暖通 | 3%~5% |
| 电气安装 | 3%~5% |
| 消防工程 | 2%~4% |

> 注：具体比例以招标文件为准。国有资金投资项目必须按规范列项。

## 三、暂估价

### 3.1 定义

暂估价是指招标人在工程量清单中提供的用于支付必然发生但暂时不能确定价格的材料、工程设备的单价以及专业工程的金额。

### 3.2 材料暂估价

| 项目 | 内容 |
|------|------|
| 计取方式 | 按招标文件给定的材料单价列入 |
| 计入位置 | 分部分项工程费的综合单价中 |
| 是否计取管理费/利润 | **否**（材料暂估价本身不参与管理费利润计算，但该材料所在定额的**人工费**仍参与） |
| 是否计取税金 | 是 |
| 结算处理 | 按实际采购价调整，差额计取税金 |

> ⚠️ **重要**：材料暂估价在广联达中的处理逻辑——暂估价材料替换定额原价，但**不影响管理费和利润的计算**（管理费、利润只与人工费挂钩）。

### 3.3 专业工程暂估价

| 项目 | 内容 |
|------|------|
| 计取方式 | 按招标文件给定的金额直接列入 |
| 计入位置 | 其他项目费 → 专业工程暂估价 |
| 是否计取管理费/利润 | 否 |
| 是否计取税金 | 是 |
| 结算处理 | 按实际分包合同价调整 |

常见专业工程暂估价项目：
- 消防报警系统
- 智能化系统
- 空调系统（甲方指定品牌）
- 特种设备安装（电梯、锅炉等）

## 四、计日工

### 4.1 定义

计日工是为了解决现场发生的零星工作的计价而设立的。完成发包人提出的施工图纸以外的零星项目或工作，按合同中约定的综合单价计价。

### 4.2 计取规则

| 项目 | 内容 |
|------|------|
| 计取方式 | 暂定数量 × 综合单价 |
| 暂定数量 | 由招标人在清单中给出 |
| 综合单价 | 由投标人自主报价 |
| 结算处理 | 按实际发生数量结算 |

### 4.3 计日工综合单价构成

```
计日工综合单价 = 人工费 + 材料费 + 机械费 + 管理费 + 利润

其中：
  管理费 = 人工费 × 48.08%
  利润   = (人工费 + 管理费) × 29.15%
```

### 4.4 计日工参考单价（山东安装工程）

| 工种/材料/机械 | 单位 | 参考单价（元） | 说明 |
|-------------|------|:-----------:|------|
| 管道工（综合） | 工日 | 180~220 | 含基本工资 |
| 电工（综合） | 工日 | 180~220 | |
| 焊工（综合） | 工日 | 200~260 | 特种作业 |
| 保温工 | 工日 | 160~200 | |
| 普工 | 工日 | 120~150 | |
| 电焊条 E4303 φ3.2 | kg | 12~18 | |
| 氧气 | m³ | 4~6 | |
| 乙炔气 | kg | 18~25 | |
| 交流电焊机 30kVA | 台班 | 150~200 | |
| 砂轮切割机 φ400 | 台班 | 30~50 | |

> 注：以上为参考价格区间，实际以《济南工程造价信息》和项目所在时段为准。

## 五、总承包服务费

### 5.1 定义

总承包服务费是总承包人为配合协调发包人进行的专业工程分包、采购和保管（发包人供应材料、设备时），以及对施工现场进行统一管理、协调、服务所需的费用。

### 5.2 计取条件与费率

| 计取条件 | 计算基数 | 参考费率 | 说明 |
|---------|---------|:-------:|------|
| 仅要求对分包的专业工程进行总承包管理和协调 | 分包专业工程估算造价 | **1%** | 仅提供管理协调 |
| 要求对分包的专业工程进行总承包管理和协调，并配合提供总承包服务 | 分包专业工程估算造价 | **2%~3%** | 提供管理协调+配合服务 |
| 发包人供应材料、设备 | 供应材料/设备总价值 | **0.5%~1%** | 采购、保管、检验 |

### 5.3 服务内容

| 服务类型 | 具体内容 |
|---------|---------|
| 管理协调 | 对分包工程的进度、质量、安全进行管理协调 |
| 配合服务 | 提供水电接驳、垂直运输、脚手架使用、施工场地提供、临时设施共用等 |
| 采购保管 | 对发包人供应的材料设备进行接收、检验、保管、发放 |

## 六、索赔与现场签证

### 6.1 索赔

| 项目 | 内容 |
|------|------|
| 定义 | 因非承包人原因导致的工期延误或费用增加，承包人向发包人提出的费用/工期补偿要求 |
| 计取方式 | 按实际发生计取，需提供证据和计算依据 |
| 计取管理费/利润 | 工程量增加部分需重新组价，按取费标准计算管理费和利润 |
| 计取税金 | 是 |

### 6.2 现场签证

| 项目 | 内容 |
|------|------|
| 定义 | 发包人现场代表与承包人现场代表就施工过程中涉及合同价款之外的责任事件所作的签认证明 |
| 计取方式 | 按签证内容实际发生计取 |
| 计取管理费/利润 | 签证涉及的工作内容需重新组价，按取费标准计算 |
| 计取税金 | 是 |

## 七、其他项目费汇总计算示例

```
【项目概况】某工厂管道安装工程，含消防报警系统分包

一、暂列金额
   = 分部分项工程费 2,000,000 × 3% = 60,000 元

二、暂估价
   1. 材料暂估价：DN200不锈钢管 500m × 180 元/m = 90,000 元
   2. 专业工程暂估价：消防报警系统 = 150,000 元
   
三、计日工
   招标给定：焊工 50工日 × 230 元/工日 = 11,500 元
   
四、总承包服务费
   1. 消防报警系统分包协调（1%）：150,000 × 1% = 1,500 元
   2. 甲方供应阀门保管（0.8%）：阀门总价值 80,000 × 0.8% = 640 元
   总承包服务费合计 = 1,500 + 640 = 2,140 元

五、其他项目费合计
   = 60,000 + 90,000 + 150,000 + 11,500 + 2,140 = 313,640 元

六、计入工程造价
   税前造价 = 分部分项工程费 + 措施项目费 + 其他项目费
   税金 = 税前造价 × 9%
```

## 八、在cnsdoce1技能中的应用

### 8.1 与 quota_project.db 的关系

其他项目费应在 `quota_project.db` 的 `projects` 表中预留字段：

| 字段名 | 类型 | 说明 |
|-------|------|------|
| `provisional_amount` | REAL | 暂列金额 |
| `professional_est` | REAL | 专业工程暂估价 |
| `material_est` | REAL | 材料暂估价合计 |
| `day_work_amount` | REAL | 计日工合计 |
| `general_service_fee` | REAL | 总承包服务费 |
| `claim_amount` | REAL | 索赔金额 |
| `visa_amount` | REAL | 现场签证金额 |
| `other_items_total` | REAL | 其他项目费合计 |

### 8.2 LLM调用逻辑

当用户查询"其他项目费如何计取"时，LLM应引用本文件，按以下逻辑回复：

1. 确认项目类型（安装工程）
2. 列出暂列金额参考比例（3%~5%）
3. 询问是否有材料暂估价、专业工程暂估价
4. 询问是否有分包需要计取总承包服务费
5. 汇总计算
