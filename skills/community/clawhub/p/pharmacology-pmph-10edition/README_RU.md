## Описание

Проект систематически интегрирует знания по фармакологии, охватывающие механизмы действия лекарств, клиническое рациональное использование, управление побочными реакциями, коррекцию дозировки для особых групп населения, стратегии антибактериальной и противоопухолевой терапии, применение лекарств для нервной системы, сердечно-сосудистую фармакологию, регуляцию эндокринных и метаболических препаратов, иммуномодулирующую терапию и принципы разработки новых лекарств — **180 ключевых клинических навыков**.

**Целевая аудитория**: Студенты-медики, ординаторы, клинические фармацевты, преподаватели фармакологии

**Учебник**: «Фармакология», 10-е издание, People's Medical Publishing House (PMPH)
