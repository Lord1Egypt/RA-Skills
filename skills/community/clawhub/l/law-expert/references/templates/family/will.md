# 自书遗嘱

## 适用场景
个人自行书写遗嘱，指定死后个人财产的分配方式。适合财产关系简单、继承人不多的情形。

## 提交机构
无需提交机构。建议自行保管或交可信赖的人保管；也可向公证处办理公证遗嘱（效力更强）

## 模板正文

```
                        遗    嘱

立遗嘱人：姓名______，性别______，身份证号：__________________
          住址：__________________，联系电话：__________________

为避免本人去世后继承人之间因财产继承问题发生纠纷，
本人经慎重考虑，在神志清醒、意志自由的情况下，
根据《中华人民共和国民法典》的规定，立下本遗嘱。

一、财产情况
本人现有财产如下：
1. 房产：位于_______________（产权证号：______，面积：______㎡）；
2. 存款：______元（开户行：______，账号：____________）；
3. 车辆：______（车牌号：______）；
4. 股票/基金：____________（账户号：______）；
5. 其他财产：________________________。

上述财产均为本人个人合法财产，本人有权处分。

二、财产处分
1. 位于__________的房产，由______（身份证号：____________）
   继承___% / 全部；
2. 存款______元，由______（身份证号：____________）继承；
3. 车辆______，由______继承；
4. _________________________，由______继承。

三、遗嘱执行人
本人指定______（身份证号：____________，联系电话：____________）
为本遗嘱的执行人。

四、特别声明
1. 本遗嘱是本人真实意思表示，未受任何胁迫或欺诈。
2. 本遗嘱的全部内容由本人亲笔书写。
3. 此前本人所立任何遗嘱，均以本遗嘱为准。

立遗嘱人（本人亲笔签名）：
日期：______年______月______日
（注：全文需亲笔书写，签名处需亲笔签名并注明日期）
```

## 注意事项
- **关键填写点**：自书遗嘱必须全文亲笔书写，不得打印；必须亲笔签名并注明日期（年月日完整）；财产信息尽可能详细准确；法定继承人之外的人受遗赠须在知道后60日内表示接受
- **常见错误**：打印后仅签名（无效）；他人代写（不满足自书条件）；未注明日期或日期不完整；遗忘保留法定继承人的"必留份"（无劳动能力又无生活来源的继承人）；处分了他人的财产（如夫妻共同财产中另一半）
- **形式要求**：全文亲笔书写 + 亲笔签名 + 手写日期，三者缺一不可
- **多份遗嘱**：有公证遗嘱的以最后一份公证遗嘱为准；无公证遗嘱的以最后一份有效遗嘱为准
- **建议**：重大财产建议办理公证遗嘱，虽然《民法典》不再赋予公证遗嘱最高效力，但公证遗嘱证明力强，不易引起签名真伪争议
