**任务域协调链路**: M0-01 → M0-02 → M0-03 → M0-04
**核心概念提取链路**: M1-01 → M1-02 → M1-03 → M1-04 → M1-05
**无关概念引入链路**: M2-01 → M2-02 → M2-03 → M2-04 → M2-05
**强制组合链路**: M3-01 → M3-02 → M3-03 → M3-04 → M3-05 → M3-06
**荒谬连接保留链路**: M4-01 → M4-02 → M4-03 → M4-04 → M4-05
**结果交付链路**: M5-01 → M5-02 → M5-03 → M5-04 → M5-05
