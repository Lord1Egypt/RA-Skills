## Description: <br>
Analyze GitHub user contribution quality. Evaluates owned repos, external contributions, output quality, and social influence. Triggers on "github cred", "analyze github". <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[zerone0x](https://clawhub.ai/user/zerone0x) <br>

### License/Terms of Use: <br>
MIT <br>


## Use Case: <br>
Developers and teams use this skill to assess a GitHub user's contribution quality with a four-tier scoring system covering owned repositories, external contributions, output quality, and social influence. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Manual installation from the README's GitHub clone command may fetch files outside this reviewed text-only bundle. <br>
Mitigation: Install from the ClawHub package when possible; inspect the external GitHub repository separately before using a manual clone. <br>
Risk: Profile scores may be incomplete or misleading if GitHub data is missing, stale, or interpreted without context. <br>
Mitigation: Treat the score as an aid for review and verify important conclusions against the user's public GitHub profile and contribution history. <br>
