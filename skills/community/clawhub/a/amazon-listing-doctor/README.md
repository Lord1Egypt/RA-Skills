# Amazon Listing Doctor 🩺

亚马逊Listing全方位诊断工具 — 专注于Rufus适配度优化

## 核心能力

本技能是市场上首个专门针对亚马逊Rufus（AI导购助手）优化的诊断工具。

**5维诊断体系**：
1. **Rufus适配度**（30%）— 分析Listing在Rufus对话中的可见度潜力
2. **场景词覆盖**（25%）— 检查核心使用场景词覆盖情况
3. **转化逻辑**（20%）— 评估五点描述的痛点-方案-利益点结构
4. **防御性文案**（15%）— 基于行业特征提供防御性内容建议
5. **视觉优化**（10%）— 主图合规和A+模块布局建议

## 极简输入

只需提供：
- 产品类目
- 产品标题
- 五点描述

纯文本粘贴即可，无需账号

## 输出

- 5维诊断评分
- 综合等级评定（A/B/C/D级）
- 针对性优化建议
- 优化后的Listing文本

## 使用示例

```
【产品类目】：家居 > 保温杯

【产品标题】：Stainless Steel Insulated Water Bottle 32oz

【五点描述】：
1. Premium stainless steel construction
2. Double-wall vacuum insulation
3. Keeps drinks hot 24h/cold 48h
4. BPA-free and leak-proof
5. Perfect for office, gym, outdoor
```

## 评分等级

| 等级 | 分数 | Rufus可见度 |
|------|------|-------------|
| A级 | 85+ | 强推荐 |
| B级 | 70-84 | 可匹配 |
| C级 | 55-69 | 低可见 |
| D级 | <55 | 不可见 |
