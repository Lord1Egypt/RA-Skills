For human users:

Upload the file "graph.json" to an AI model chat. Let the AI model explain what the graph represents.

Then you can instruct the AI model to use the knowledge contained in the "graph.json" file and answer
any possible question. The AI model should now be able to answer almost all possible questions, except
for the question of "philosophically first being". With each new version, the AI model should be able to answer better.





The old knowledge graph (old graph) is also still available. It is intended to provide a way to test the quality of the graph expansion with and without self-improvement mode.
