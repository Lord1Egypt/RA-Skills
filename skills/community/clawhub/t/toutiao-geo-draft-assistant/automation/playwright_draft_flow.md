# Playwright Draft Flow

本文件描述 `scripts/draft_to_toutiao.example.py` 的执行流程。

1. **读取本地文件**
   - 脚本启动时，仅读取 `output/toutiao/toutiao_article.md`。
   - 分离标题和正文内容。

2. **启动浏览器**
   - 使用 Playwright 启动一个非无头 (headless=False) 浏览器实例。
   - 不加载任何历史数据，不注入任何 Cookie。

3. **导航至头条号创作中心**
   - 打开 URL: `https://mp.toutiao.com/profile_v4/graphic/publish`
   - 脚本进入等待状态，监控页面元素。

4. **用户人工登录**
   - **用户必须在弹出的可见浏览器中，使用手机扫码或密码登录。**
   - 如果遇到拼图验证码、短信验证，由用户人工完成。
   - 脚本被动等待，直到检测到编辑器加载成功。

5. **填入草稿**
   - **定位标题**：填入解析出的文章标题。
   - **定位正文**：由于头条编辑器可能是富文本或 Prosemirror，脚本将正文内容粘贴至编辑器中。
   - 若能稳定定位摘要、封面图入口，则进行辅助点击或填入（受限于头条界面变化，可能需要人工操作）。

6. **停止脚本并交出控制权**
   - 脚本执行完毕，**不做任何保存或发布操作**。
   - 打印提示：“草稿已填入，请人工检查排版、上传封面，并手动点击发布。”
   - 浏览器保持开启状态，由用户接管。
