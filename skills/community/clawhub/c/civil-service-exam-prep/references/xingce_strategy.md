# 行测（行政职业能力测验）备考策略

## 总体框架

行测五大模块，按性价比排序：**资料分析 > 判断推理 > 言语理解 > 数量关系 ≈ 常识判断**

## 一、言语理解与表达（40题/副省，约40分）

### 核心题型
1. **逻辑填空（20题）**：实词辨析、成语辨析、混合考察
2. **片段阅读（15题）**：主旨概括、意图判断、细节理解、词句理解
3. **语句表达（5题）**：语句排序、语句衔接、下文推断

### 备考策略
- **逻辑填空**：背高频成语（600-800个）和近义实词辨析（200组）
  - 技巧：看搭配、找对应、辨色彩、析语境
  - 切忌凭语感，要结合上下文找线索词
- **片段阅读**：掌握"转折后为重点""首尾句定主旨""举例前是观点"
  - 意图判断题：优选对策项
  - 细节理解题：逐项带回原文比对
- **语句排序**：确定首句→捆绑验证→确定尾句

### 做题顺序建议
- 先做片段阅读（得分率高），再做逻辑填空，最后语句表达

## 二、数量关系（15-20题）

### 核心题型
1. **数学运算**：行程问题、工程问题、利润问题、排列组合、概率、容斥、几何、年龄/日期
2. **数字推理**（部分省份）：等差/等比/幂次/递推数列

### 备考策略
- **取舍策略**：国考数量放在最后做，挑3-5道简单题确保做对，其余统一蒙
- **必会题型**：工程问题、利润问题、简单行程、容斥问题
- **技巧**：代入排除法、赋值法、数字特性法（整除/奇偶/尾数）
- **蒙题策略**：全蒙B/C，正确率约25-30%

### 重点公式速记
```
路程=速度×时间   利润=售价-成本   利润率=利润/成本
工作量=效率×时间   浓度=溶质/溶液
排列：A(n,m)=n!/(n-m)!   组合：C(n,m)=n!/[m!(n-m)!]
概率=满足条件数/总情况数
```

## 三、判断推理（40题）

### 核心题型
1. **图形推理（10题）**：位置类、样式类、属性类、数量类、空间重构
2. **定义判断（10题）**：单定义、多定义
3. **类比推理（10题）**：集合关系、逻辑关系、语义关系、语法关系
4. **逻辑判断（10题）**：翻译推理、加强削弱、真假推理、归纳推理

### 备考策略
- **图形推理**：
  - 口诀："元素相同看位置，元素相似看样式，元素不同先属性后数量"
  - 空间重构（折纸盒）：用"相对面法"和"时针法"
- **定义判断**：抓关键词法，用定义中的条件逐条验证选项
- **类比推理**：造句子 → 看词性 → 辨关系
  - 常见关系：种属、组成、对应、近义、反义、功能、因果、顺序
- **逻辑判断**：
  - 翻译推理背口诀：如果就前推后，只有才后推前，且或要么不
  - 加强削弱：找论点论据，优先加强/削弱论证联系

## 四、资料分析（20题）

### 核心考点
- 增长率、增长量、比重、倍数、平均数、基期现期、同比环比

### 速算技巧
| 技巧 | 适用场景 | 精度 |
|------|----------|------|
| 截位直除 | 普通除法 | 高 |
| 特征数字法 | 乘除分数 | 中 |
| 有效数字法 | 复杂乘除 | 高 |
| 差分法 | 分数比较 | 高 |
| 比重变化估算 | a-b的绝对值 | 中 |

### 备考策略
- **最值得投入的模块**：分值高、技巧性强、提分快
- 牢记常用分数-小数转换（1/6≈16.7%、1/7≈14.3%、1/8=12.5%、1/9≈11.1%、1/11≈9.1%）
- 每篇资料分析4分钟完成：读题30秒→列式&计算3.5分钟/5题

## 五、常识判断（20题）

### 知识领域
- 政治（时政、马哲、中特）：约30%
- 法律（宪法、行政法、民法、刑法）：约20%
- 科技（前沿科技、生活常识）：约15%
- 文史（中国古代史、文学、文化）：约15%
- 地理（国情地理、自然地理）：约10%
- 经济（宏观经济、微观经济）：约10%

### 备考策略
- 日常积累为主，突击效果差
- 关注当年重要会议、政策文件、法律修订
- 做题策略：7-8分钟快速过，不会就蒙，不纠结
