# 羊膜腔穿刺术标准操作流程

## 适用条件
- 妊娠≥15周
- 有遗传学检查指征（如高龄、筛查高风险、超声异常等）
- 无流产征兆、感染、凝血功能异常
- 已签署知情同意书

## 执行流程

### 1. 术前准备
- 复核指征，充分告知风险，签署同意书
- 完善检查：生命体征、血常规、凝血功能、胎心监测

### 2. 操作步骤
- 孕妇排空膀胱，仰卧位
- 腹部消毒铺巾
- 实时超声引导下确定安全穿刺路径（避开胎儿、胎盘、脐带等）
- 局部麻醉（可选），经皮穿刺进入羊膜腔
- 弃去初始2ml羊水（防母体细胞污染）
- 抽取所需量羊水用于检测

### 3. 术后处理
- 观察胎心及孕妇症状（腹痛、流血等）
- Rh阴性者必须注射Rh免疫球蛋白

### 4. 关键注意事项
- 严格无菌操作
- 避免宫缩时穿刺
- 最多不超过3次穿刺尝试
- 监测羊水栓塞征兆（咳嗽、呼吸困难、发绀）

## 输出
成功获取无污染羊水样本用于遗传学、代谢病或感染检测，并完成术后监护。
