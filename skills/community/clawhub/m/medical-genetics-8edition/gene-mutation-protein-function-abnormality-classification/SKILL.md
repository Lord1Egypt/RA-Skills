# 基因突变致蛋白功能异常分类

## 何时使用
- 用户提供一个影响蛋白质编码或调控区域的基因突变（如点突变、缺失、重复、动态突变等）；
- 需要判断该突变如何导致蛋白质功能异常，以指导分子诊断、遗传咨询或治疗策略；
- 突变对象为编码蛋白质的基因、遗传病相关基因或其调控序列。

## 执行步骤
1. **确认突变基本信息**：
   - 确定突变位置（编码区、调控区等）；
   - 明确突变类型（无义、错义、移码、缺失、重复、动态突变等）；
   - 判断目标蛋白是否为多亚基复合体（如胶原三螺旋结构）。

2. **应用决策树分类**：
   - **若突变导致蛋白活性完全或部分丧失** → 归为“丢失功能”；
     - 包括：无义/移码突变致截短蛋白、基因缺失、调控区甲基化沉默表达；
     - 注意显性负效应（突变亚基干扰正常亚基功能）也属此类。
   - **若突变导致蛋白活性异常增强或剂量增加** → 归为“增强功能”；
     - 包括：基因拷贝数增加、染色体三体、突变使蛋白结合力/酶活性异常升高。
   - **若突变赋予蛋白全新有害特性** → 归为“获得新性状”；
     - 典型如错义突变导致蛋白在特定条件下异常聚集（如HbS镰变）。
   - **若突变导致蛋白在错误时间或位置表达** → 归为“异时或异位表达”；
     - 多由调控区突变引起，如胎儿血红蛋白出生后持续表达。

3. **输出结果**：
   - 明确指出所属功能异常类型；
   - 提示可能关联的遗传病机制及后续验证方向。

> 注意：本技能不适用于非编码RNA基因突变的直接效应分析。解释结果时必须结合具体蛋白的结构与生理功能背景。
