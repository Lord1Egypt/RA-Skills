# GB/T 7714-2015 Citation Format Reference

## Core Rules

### Author Formatting

**Chinese authors:**
- 3 or fewer: list all, comma separated. Example: `张三, 李四, 王五`
- More than 3: first author + `, 等`. Example: `张三, 等`

**English authors:**
- Surname ALL CAPS, given name as initials (no periods between initials unless ambiguous)
- 3 or fewer: list all. Example: `SMITH J D, WANG X M, BROWN R`
- More than 3: first author + `, et al.`
- Format: `SURNAME GivenNameInitials`. Example: `SMITH J` or `SMITH J D`
- Hyphenated surnames: keep hyphen, all caps. Example: `WILLIAMS-ELLIS A`
- Multi-part surnames (with prepositions): all caps. Example: `DE MORGAN A`
- Western author translated to Chinese: only record the surname. Example: original `S. Unwin (Stephen Unwin)` → `昂温`
- If same surname and need to distinguish: surname + given name initial. Example: `昂温G, 昂温PS`

**Chinese pinyin names:**
- Surname ALL CAPS, given name abbreviated to first letters of each character
- Example: `Li Jiangning` → `LI Jiangning` or `LI JN`
- Based on GB/T 28039-2011

### Organization as Author
When the responsible party is an organization, record the organization name directly as "author".
```
[1] 全国信息与文献标准化技术委员会. 信息与文献 都柏林核心元数据元素集: GB/T 25100—2010[S]. 北京: 中国标准出版社, 2010: 2-3.
[2] 中国造纸学会. 中国造纸年鉴: 2003[M/OL]. 北京: 中国轻工业出版社, 2003[2014-04-25]. http://...
```

### Sequence Format (顺序编码制)

Used by default. Citations numbered consecutively in text.

### Author-Date Format (著者-出版年制)

An alternative to sequence numeric format. In the reference list, citations are organized by author surname (alphabetically) and publication year.
```
尼葛洛庞帝, 1996. 数字化生存[M]. 胡泳, 范海燕, 译. 海口: 海南出版社.
汪冰, 1997. 电子图书馆理论与实践研究[M]. 北京: 北京图书馆出版社: 16.
杨宗英, 1996. 电子图书馆的现实模型[J]. 中国图书馆学报(2): 24-29.
BAKER S K, JACKSON M E, 1995. The future of resource sharing[M]. New York: The Haworth Press.
CHERNIK B E, 1982. Introduction to library services for library technicians[M]. Littleton, Colo.: Libraries Unlimited, Inc.
```

Note: When using author-date format:
- Citations in text use `(Author, Year)` format instead of `[n]`
- Reference list is organized by author surname alphabetically
- For Chinese authors, can organize by 汉语拼音 or by 笔画笔顺
- For multiple works by same author in same year, use `2010a`, `2010b` to distinguish

### Combined Titles (合订题名)
When 2 or more works are bound together without a general title, separate each title with `;`.
```
[1] 顾炎武. 昌平山水记; 京东考古录[M]. 北京: 北京古籍出版社, 1992.
```

## Format Templates

### Journal Article [J]

**Template:**
```
[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
[序号] 作者. 题名[J/OL]. 刊名, 年, 卷(期): 起止页码[引用日期]. DOI.
```

**Examples:**
```
[1] 张三, 李四. 基于深度学习的图像识别方法研究[J]. 计算机学报, 2023, 46(5): 1123-1135.
[2] SMITH J D, WANG X M, BROWN R. Deep learning for image recognition[J]. Nature, 2022, 612(7938): 45-52.
[3] 王五, 等. 新型材料在航空领域的应用[J/OL]. 航空学报, 2024, 45(3): 234-245[2024-05-01]. https://doi.org/10.xxxx/xxxxx.
```

### Book [M]

**Template:**
```
[序号] 作者. 书名[M]. 版本项. 出版地: 出版者, 出版年: 引文页码.
[序号] 作者. 书名[M/OL]. 版本项. 出版地: 出版者, 出版年: 引文页码[引用日期]. URL.
```

**Examples:**
```
[1] 张三. 人工智能导论[M]. 北京: 清华大学出版社, 2022: 45-48.
[2] RUSSELL S, NORVIG P. Artificial intelligence: a modern approach[M]. 4th ed. Boston: Pearson, 2020: 123-125.
[3] 李四. 数据结构与算法[M]. 2版. 上海: 上海交通大学出版社, 2023.
```

**Note:** Only include edition if it is 2nd or higher. For 1st edition, omit "1版" or "1st ed."
**Note:** For books with no known place of publication, use `[出版地不详]` for Chinese or `[S.l.]` for English. For no publisher, use `[出版者不详]` or `[s.n.]`. For electronic resources with no place/publisher, these may be omitted.
**Note:** For ancient/rare books, record the version type (刻本, 抄本, 影印本, etc.) in the edition field.
```
[1] 王夫之. 宋论[M]. 刻本. 金陵: 湘乡曾国荃, 1865(清同治四年).
[2] 汪昂. 增订本草备要: 四卷[M]. 刻本. 京都: 老二酉堂, 1881(清光绪七年).
[3] 康熙字典: 已集上: 水部[M]. 同文书局影印本. 北京: 中华书局, 1962: 50.
```
**Note:** Multiple places of publication → record the first one only. Multiple publishers → list all, separated by `:`.
**Note:** Non-Gregorian calendar years → convert to Gregorian and put in parentheses.
```
[1] 某某. 书名[M]. 出版地: 出版者, 1865(清同治四年).
```
**Note:** Unpublished works → record publication year as "印刷中".

### Conference Paper [C] / [C/OL]

**Template:**
```
[序号] 作者. 题名[C]//编者. 会议论文集名. 出版地: 出版者, 出版年: 起止页码.
[序号] 作者. 题名[C/OL]//编者. 会议论文集名. 出版地: 出版者, 出版年: 起止页码[引用日期]. URL.
```

**Examples:**
```
[1] 张三, 李四. 基于Transformer的文本分类方法[C]//中国计算机学会. 第30届全国信息检索学术会议论文集. 北京: 电子工业出版社, 2024: 56-62.
[2] SMITH J, WANG X. Attention mechanisms in NLP[C]//Proceedings of the 2023 Conference on Empirical Methods in Natural Language Processing. Stroudsburg: ACL, 2023: 1234-1240.
[3] 贾东琴, 柯平. 面向数字素养的高校图书馆数字服务体系研究[C]//中国图书馆学会. 中国图书馆学会年会论文集: 2011年卷. 北京: 国家图书馆出版社, 2011: 45-52.
```

```
[序号] 作者. 题名[D/OL]. 保存地: 保存单位, 年份[引用日期]. URL.
```

**Examples:**

### Standard [S] / [S/OL]

**Template:**
```
[序号] 主要责任者. 标准编号, 标准名称[S]. 出版地: 出版者, 出版年: 页码.
[序号] 主要责任者. 标准编号, 标准名称[S/OL]. 出版地: 出版者, 出版年: 页码[引用日期]. URL.
```

**Examples:**
```

[2] 全国广播电视标准化技术委员会. 广播电视音像资料编目规范: 第2部分 广播资料: GY/T 202.2—2007[S]. 北京: 国家广播电影电视总局广播电视规划院, 2007: 1.
[3] 国家环境保护局科技标准司. 土壤环境质量标准: GB 15616—1995[S/OL]. 北京: 中国标准出版社, 1996: 2-3[2013-10-14]. https://wenku.baidu.com/view/...
[4] Information and documentation-the Dublin core metadata element set: ISO 15836:2009[S/OL]. [2013-03-24]. https://www.iso.org/...
```

**Note:** Standards may have a responsible organization as "author". The standard number is part of the title. For online standards without place/publisher, these may be omitted.

```
[序号] 专利申请者或所有者. 专利题名: 专利号[P/OL]. 公告日期或公开日期[引用日期]. URL.

```

**Template:**
```
[序号] 作者. 题名[EB/OL]. (发布日期)[引用日期]. URL.
[序号] [佚名]. 题名[EB/OL]. [无日期][引用日期]. URL.
```

**Examples:**

## Punctuation Rules

| Symbol | Usage |
|---|---|
| `.` | Separates elements: title, other title info, other responsible parties, edition, publication, source, access path, DOI. Ends each reference. |
| `:` | Before other title info, publisher, page numbers, extracted page numbers, patent number |
| `,` | Between authors with same role, before "等"/"et al.", before year, before volume in journal year/volume/issue |
| `;` | Between combined titles by same author; between subsequent year/volume/issue/page in serial articles |
| `//` | Before the source of extracted literature from monographs |
| `()` | For issue number in journals, newspaper edition, update/modification date of electronic resources, non-Gregorian calendar years |
| `[]` | For reference sequence number, document type identifier, access date, self-created info |
| `/` | Between combined issue numbers; before document carrier identifier |
| `-` | Between starting-ending sequence numbers and page numbers |

## Special Cases

### No Author (顺序编码制)
For sequence numeric format, if no author, omit the author field and start with the title directly.
```
[1] 题名[文献类型标识]. ...
```
For English documents with no author, use `[Anon]` or omit and start with title.

### No Place of Publication
- Chinese: `[出版地不详]`
- English: `[S.l.]` (sine loco)

### No Publisher
- Chinese: `[出版者不详]`
- English: `[s.n.]` (sine nomine)

### No Date
Use `[无日期]` for Chinese, `[n.d.]` for English.

### Article Number vs Page Range
If journal uses article number instead of page range:
```
[1] SMITH J. Title[J]. Journal, 2023, 45: e123456.
```

### Combined Issues (合期)
For articles from combined issues:
```
[1] 作者. 题名[J]. 刊名, 2001(9/10): 36-39.
```

### Supplement Issues (增刊)
For articles from supplement issues:
```
[1] 作者. 题名[J]. 刊名, 2012, 22(增刊2): 81-86.
```

### Serial Articles (连载)
For serial articles in the same journal, append subsequent parts to the original citation:
```
[1] 作者. 题名[J]. 刊名, 2011, 33(2): 20-25; 2011, 33(3): 26-30.
```

### Online-First Articles
```
[1] 张三. 标题[J/OL]. 刊名, 2024[2024-05-01]. https://doi.org/10.xxxx/xxxxx.
```

### Electronic Resource with Update/Modification Date
When an electronic resource has been updated or modified, include the update date in `()` before the access date in `[]`.
```
[1] DEVERELL W, IGLE D. A companion to California history[M/OL]. New York: John Wiley & Sons, 2013: 21-22 (2013-11-15)[2014-06-24]. http://...
```

### DOI Format
DOI is placed at the end of the citation after the URL, prefixed with `DOI:`.
```
[1] 武丽丽, 华一新, 张亚军, 等. "北斗一号"监控管理网设计与实现[J/OL]. 测绘科学, 2008, 33(5): 8-9[2009-10-25]. http://... DOI:10.3771/j.issn.1009-2307.2008.05.002.
[2] FRESE K S, KATUS H A, MEDER B. Next-generation sequencing[J/OL]. Biology, 2013, 2(1): 378-398[2013-03-19]. http://... DOI:10.3390/biology2010378.
```

### Other Title Information (副标题/其他题名信息)
Use `:` before subtitles or other title information.
```
[1] 哈里森, 沃尔德伦. 经济数学与金融数学[M]. 谢远涛, 译. 北京: 中国人民大学出版社, 2012: 235-236.
[2] 徐光宪, 王祥云. 物质结构[M]. 北京: 科学出版社, 2010.
```

### Compilation [G] / [G/OL]

**Template:**
```
[序号] 编者. 汇编题名[G]. 出版地: 出版者, 出版年.
[序号] 编者. 汇编题名[G/OL]. 出版地: 出版者, 出版年[引用日期]. URL.
```

**Examples:**
```
[1] 中国职工教育研究会. 职工教育研究论文集[G]. 北京: 人民教育出版社, 1985.
[2] 国家环境保护总局生物安全管理办公室. 中国国家生物安全框架实施国际合作项目研讨会论文集[G/OL]. 北京: 中国环境科学出版社, 2002[2002-05-28]. https://example.com/compilation.pdf.
```

### Archives [A] / [A/OL]

**Template:**
```
[序号] 主要责任者. 档案名[A]. 出版地: 出版者, 出版年.
[序号] 主要责任者. 档案名[A/OL]. 出版地: 出版者, 出版年[引用日期]. URL.
```

**Examples:**
```
[1] 中国第一历史档案馆, 辽宁省档案馆. 中国明朝档案总汇[A]. 桂林: 广西师范大学出版社, 2001.
```

### Map/Atlas [CM]

**Template:**
```
[序号] 主要责任者. 图名[CM]. 出版地: 出版者, 出版年.
```

**Example:**
```
[1] 国家测绘地理信息局. 中国地图集[CM]. 北京: 中国地图出版社, 2020.
```

### Multivolume Works (多卷书)
When citing from a multivolume work, specify the volume number:
```
[1] 张伯伟. 全唐五代诗格会考[M]. 南京: 江苏古籍出版社, 2002: 288.
[2] 陈登原. 国史旧闻: 第1卷[M]. 北京: 中华书局, 2000: 29.
[3] 师伏堂日记: 第4册[M]. 北京: 北京图书馆出版社, 2009: 155.
[4] 胡承正, 周详, 缪灵. 理论物理概论: 上[M]. 武汉: 武汉大学出版社, 2010: 112.
```

### Series (丛书)
For books that are part of a series, the series name may be included:
```
[1] 宋史卷三: 本纪第三[M]//宋史: 第1册. 北京: 中华书局, 1977: 49.
```

### Bilingual Citations (双语著录)
The standard allows citations to be recorded in both original language and another language. First record in the original language, then in the other language.
```
[1] 李炳穆. 图书馆法规总览: 第1卷[M]. 首尔: 九美贸易出版部, 2005: 67-68.
    LEE B M. Library laws and regulations: Vol. 1[M]. Seoul: Kumi Trading Publishing, 2005: 67-68.
[2] 熊平, 吴颉. 从交易费用的角度谈如何构建药品流通的良性机制[J]. 中国物价, 2005(8): 42-45.
    XIONG P, WU X. Discussion on how to construct benign medicine circulation mechanism from transaction cost perspective[J]. China price, 2005(8): 42-45.
```

### Dataset [DS]

**Template:**
```
[序号] 主要责任者. 数据集名[DS]. 出版地: 出版者, 出版年.
```

**Example:**
```
[1] 中国科学院. 中国科学数据中心数据集[DS]. 北京: 科学出版社, 2023.
```

### Database [DB/OL]

**Template:**
```
[序号] 数据库名[DB/OL]. [引用日期]. URL.
```

**Example:**
```
[1] 中国知网[DB/OL]. [2024-05-01]. https://www.cnki.net.
```

### Translated Works
```
[1] 原作者. 书名[M]. 译者, 译. 出版地: 出版者, 年份.
[2] 哈里森, 沃尔德伦. 经济数学与金融数学[M]. 谢远涛, 译. 北京: 中国人民大学出版社, 2012: 235-236.

```

### Extracted Literature from Monographs (专著中析出文献)
Use `//` to indicate the source monograph.
```
[1] 程根伟. 1998年长江洪水的成因与减灾对策[M]//许厚泽, 赵其国. 长江流域洪涝灾害与科技对策. 北京: 科学出版社, 1999: 32-36.
[2] 陈晋镳, 张惠民, 朱士兴, 等. 蓟县震旦亚界研究[M]//中国地质科学院天津地质矿产研究所. 中国震旦亚界. 天津: 天津科学技术出版社, 1980: 56-114.
[3] WEINSTEIN L, SWERTZ M N. Pathogenic properties of invading microorganism[M]//SODEMAN W A, Jr, SODEMAN W A. Pathologic physiology: mechanisms of disease. Philadelphia: Saunders, 1974: 745-772.
```

### Extracted Literature from Journals (期刊中析出文献)
From journals, use `.` (not `//`) to connect to the journal name.

### Reading vs Cited References
- **Reading reference**: Page numbers record the article's starting-ending pages or starting page
- **Cited reference**: Page numbers record the specific page(s) where the cited information appears

Example (reading reference):
```
[1] 杨宗英. 电子图书馆的现实模型[J]. 中国图书馆学报, 1996(2): 24-29.
```

Example (cited reference):
```
[1] 杨宗英. 电子图书馆的现实模型[J]. 中国图书馆学报, 1996(2): 26.
```
