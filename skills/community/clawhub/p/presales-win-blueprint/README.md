# presales-win-blueprint

售前方案通关秘籍：按行业、阶段、竞品形态和风险信号生成赢单打法。

## 安装与运行

```bash
npm test
node src/index.js --quick --client "某连锁企业" --industry "零售" --stage "需求期" --competitors "平台型厂商,低价竞争者" --budget "已立项"
```

## 输入参数

- `--client`：客户或项目代称
- `--industry`：政府、零售、制造、金融等
- `--stage`：线索期、需求期、POC阶段、招投标、商务谈判
- `--competitors`：竞品形态，逗号分隔
- `--budget`：预算状态
- `--decision-role`：主要决策角色
- `--signals`：风险信号，逗号分隔

## 输出模块

- 阶段打法
- 行业打法
- 竞品攻防
- 风险预警
- 方案自检
- 下一步动作

## 内容原则

本 Skill 使用行业化经验方法论，不包含真实个人、真实客户、真实厂商、真实项目名称或具体项目数据。
