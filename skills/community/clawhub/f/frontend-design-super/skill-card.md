## Description: <br>
Frontend Design Super helps agents create production-grade static web interfaces across React, Vue 3, Nuxt, Tailwind CSS, and shadcn-style component stacks from plain text requirements. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, designers, and agents use this skill to plan and build distinctive static sites, landing pages, portfolios, dashboards, and framework-specific frontend projects from natural-language requirements. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Initialization scripts use npm and npx package workflows that can fetch and execute current third-party packages. <br>
Mitigation: Review the selected init script before running it, use a fresh project directory, and pin or audit dependencies for production work. <br>
Risk: Generated privacy, security, and marketing copy may be sample content rather than organization-approved language. <br>
Mitigation: Replace sample copy and review generated pages before publishing them externally. <br>
