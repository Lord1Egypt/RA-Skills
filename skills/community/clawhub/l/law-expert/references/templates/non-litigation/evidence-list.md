# 证据清单格式（庭审用）

## 适用场景
民事/行政/仲裁案件庭审中，当事人或代理律师向人民法院/仲裁庭提交证据时，附带编制的证据目录。用于清晰展示证据体系、方便庭审质证和法庭/仲裁庭查阅。

## 提交机构
受理案件的人民法院 或 仲裁委员会

## 注意事项

### 关键填写点
1. **每个证据都要有唯一的证据编号**（证1、证2...），方便庭审质证时快速定位
2. **证明目的要具体**：不是"证明被告违约"，而是"证明被告于XX日未按约定交付货物，构成违约"
3. **分组逻辑**：按诉讼要件的证明逻辑分组，例如：
   - 第一组：法律关系成立
   - 第二组：已履行义务
   - 第三组：对方违约/侵权
   - 第四组：损失计算
4. **原件还是复印件**要明确标注，原件开庭时带齐备查
5. **页数要准确**：每份证据的页数要提前数清楚，庭审提交前核对

### 常见错误
- 证据编号混乱——各组证据独立编号或统一连续编号均可，但必须一致
- 证明目的太宽泛或空泛——"证明被告存在过错"不是好的证明目的，应该写"证明被告驾车闯红灯导致碰撞"
- 遗漏证据——编好清单后再与起诉状中的事实逐一核对，确保每个关键事实都有对应证据
- 证据没有标注页码——庭审时法官要求翻到某页，没有页码就无法快速定位
- 把不同来源/不同性质的证据混在一组——同一组内的证据应围绕同一证明主题

### 特殊证据处理
| 证据类型 | 提交方式 | 特别要求 |
|----------|----------|----------|
| 微信/QQ聊天记录 | 截图打印+光盘/电子版 | 标注聊天人身份；关键内容高亮或划线；保留原始设备备查 |
| 电子邮件 | 打印页面+邮件原文 | 显示发件人/收件人/时间/主题全部信息 |
| 录音/录像 | 整理成文字版+光盘 | 文字版标明说话人、时间点，光盘贴标签 |
| 证人证言 | 书面证言或申请证人出庭 | 书面证言需证人签名；出庭需提前提交申请 |
| 公文书证 | 复印件 | 庭审时需带原件核对 |
| 域外证据 | 需公证+认证 | 域外形成的证据需所在国公证+中国使领馆认证（或海牙认证） |

### 份数/格式要求
- 提交份数：对方当事人人数+法院/仲裁庭留存1份
- 提交时间：一般在举证期限内提交（起诉/答辩时一并提交，或按法院/仲裁庭指定的举证期限）
- 电子版：建议同时提交证据的电子版（U盘或光盘），方便法院/仲裁庭查阅
- 格式：A4纸，打印清晰，表格格式规整
