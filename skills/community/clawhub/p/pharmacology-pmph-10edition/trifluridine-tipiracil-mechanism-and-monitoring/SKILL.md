# 曲氟尿苷替匹嘧啶作用机制与用药监测

## 何时使用本技能
- 患者正在接受或考虑使用曲氟尿苷替匹嘧啶治疗转移性结直肠癌
- 需要解释该药物为何必须以复方形式给药
- 出现骨髓抑制或胃肠道不良反应，需关联其作用机制进行评估
- 对比其他核苷类似物治疗方案时需阐明其独特保护机制

## 核心操作流程

1. **确认药物组成**：曲氟尿苷（trifluridine）与替匹嘧啶（tipiracil hydrochloride）按物质的量比1:0.5（重量比1:0.471）配制。

2. **理解保护机制**：替匹嘧啶抑制胸苷磷酸化酶，防止曲氟尿苷在外周被快速降解，显著提升其系统暴露量（AUC）和生物利用度。

3. **识别抗肿瘤作用路径**：
   - 曲氟尿苷进入肿瘤细胞后被磷酸化为三磷酸活性形式
   - 整合入DNA链，干扰复制与修复，导致DNA功能障碍
   - 最终抑制肿瘤细胞增殖，尤其适用于对其他化疗耐药的转移性结直肠癌

4. **执行关键监测**：
   - 定期检查血常规（重点关注贫血、中性粒细胞减少、血小板减少）
   - 评估胃肠道症状（恶心、呕吐、腹泻、腹痛、食欲下降）
   - 监测乏力、疲乏及发热等全身反应

5. **应用禁忌筛查**：
   - 禁用于对任一成分过敏者
   - 重度骨髓抑制患者禁用

> **重要提醒**：不可单独使用曲氟尿苷，必须依赖替匹嘧啶的酶抑制保护作用才能发挥疗效。
