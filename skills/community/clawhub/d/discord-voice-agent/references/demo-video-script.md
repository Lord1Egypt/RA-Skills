# Demo video script

## Goal

Show that Discord Voice Agent is easy to install, easy to understand, and actually useful.

## Length

60–120 seconds.

## Scene 1 — Hook

On screen:
- title card: "Discord Voice Agent for OpenClaw"
- subtitle: "A normal chat agent, but in voice"

Voiceover:
- "This is Discord Voice Agent, a beginner-friendly OpenClaw skill that turns Discord into a live voice assistant."

## Scene 2 — Setup simplicity

On screen:
- show the setup checklist
- highlight only the missing fields
- highlight the default model behavior

Voiceover:
- "If Discord is already installed, you only need the voice channel id. If not, you add the Discord plugin, token, and channel id. The model can stay on the default OpenClaw chat behavior at first."

## Scene 3 — One-command test

On screen:
- run `npm run smoke`
- show a passing result

Voiceover:
- "First, run the smoke test to confirm the voice pipeline is wired correctly."

## Scene 4 — Live voice demo

On screen:
- bot joins the voice channel
- `/status`
- `/say hello`
- speak a short question
- show short reply

Voiceover:
- "Then the bot joins voice, listens, and replies briefly through OpenClaw."

## Scene 5 — Why it feels good

On screen:
- short bullet callouts: fast, short replies, fallback, status, model can change later

Voiceover:
- "It keeps replies short, handles fallback paths, and feels like a normal chat agent instead of a special demo."

## Scene 6 — Close

On screen:
- repo name
- install / download call to action

Voiceover:
- "If you want a voice assistant that’s simple to start and easy to grow, this is the one to try."
