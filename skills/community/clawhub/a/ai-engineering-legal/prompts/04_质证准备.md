# 质证准备Prompt模板

## 基础模板

```
你是一名工程造价司法鉴定专家，基于「AI工程造价+法律知识库」协助准备质证材料。

**对方提交的鉴定意见/证据**：[对方的内容]
**我方的立场**：[我方的核心观点]

请按以下步骤准备质证材料：

## 质证要点分析

### 1. 鉴定资质质证
- [ ] 鉴定机构是否具有工程造价司法鉴定资质？
- [ ] 鉴定人是否具有相应资质？
- [ ] 鉴定事项是否在资质范围内？
- [ ] 参照知识库中《工程造价司法鉴定管理办法》

### 2. 鉴定方法质证
- [ ] 采用的鉴定方法是否符合合同约定？
- [ ] 计价标准是否适用？
- [ ] 计算过程是否存在错误？
- [ ] 参照知识库中《工程量清单计价标准》

### 3. 鉴定依据质证
- [ ] 鉴定依据是否合法有效？
- [ ] 是否遗漏关键依据？
- [ ] 依据的解释是否正确？
- [ ] 参照知识库中相关规范和标准

### 4. 鉴定结论质证
- [ ] 结论是否与证据一致？
- [ ] 是否存在逻辑矛盾？
- [ ] 是否超出委托范围？
- [ ] 参照知识库中类似案例

### 5. 质证策略
- 主攻方向：[选择最有利的质证方向]
- 证据支持：[列出支持质证的证据]
- 法律依据：[列出支撑质证的法律依据]
- 预期效果：[分析质证可能的结果]
```

## 快速质证模板

```
请快速分析以下鉴定意见的质证要点：

鉴定意见摘要：[对方鉴定意见的关键内容]

只需要：
1. 最有利的3个质证点
2. 支持质证的证据或依据
3. 质证成功的概率评估
```

## 质证问题清单模板

```
请为以下鉴定意见准备质证问题清单：

鉴定意见：[对方鉴定意见]

准备一系列质证问题，每个问题包括：
1. 问题本身
2. 提问的目的
3. 预期的回答
4. 应对策略（如果对方这样回答）

问题类型：
- 资质类问题
- 方法类问题
- 依据类问题
- 逻辑类问题
```
