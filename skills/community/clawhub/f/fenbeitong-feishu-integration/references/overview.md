# 版本与选型

## 版本说明

**市场版**：分贝通以独立应用身份上架飞书应用市场，小程序/H5 形式。客户管理员可在飞书应用市场直接搜索安装，即开即用。

**嵌入版**：分贝通将 Web 应用以 iframe 形式嵌入客户飞书域名的子路径，H5 形式。需要客户在飞书开放平台创建自建应用，权限灵活性大。

---

## 版本选型决策树

当客户问"我该选市场版还是嵌入版"时，按以下流程快速判断：

```
客户需要同步手机号？
    └─ ✅ 是 → 必须嵌入版
    └─ 否 ↓
客户需要集团版对接？
    └─ ✅ 是 → 必须嵌入版
    └─ 否 ↓
客户有定制化需求？
    └─ ✅ 是 → 嵌入版
    └─ 否 ↓
客户要求申请单同步用户状态栏标签？
    └─ ✅ 是 → 嵌入版
    └─ 否 ↓
客户仅需基础功能（消息、简单待办）且希望快速上线？
    └─ ✅ 是 → 市场版
```

**一句话结论**：有手机号同步、集团版、定制化、状态栏标签需求 → 嵌入版；仅需消息推送 + 基础待办 → 市场版。

## 交付周期与里程碑

### 上线周期

| 版本 | 典型周期 | 需要的配合 |
|------|---------|----------|
| 市场版 | **1 天** | 客户管理员 + 分贝通交付实施 |
| 嵌入版 | **1 天** | 客户管理员 + 分贝通交付实施 |

> ⚠️ 以上为标准场景周期，复杂定制化需求需额外评估。

### 典型交付里程碑

1. **组织架构同步完成** — 部门/成员数据准确同步到分贝通
2. **申请单同步完成** — 差旅申请单可正常创建、审批、流转
3. **飞书待办消息正常发出** — 审批通知、待办提醒可触达员工

---

## 交付注意事项

1. **市场版安装**：客户管理员在飞书应用市场搜索"分贝通"直接授权安装。
2. **嵌入版**：需在飞书开放平台创建自建应用，有一定配置工作量，建议销售提前与交付沟通。
3. **个人状态同步**：如客户需要此功能，**必须在售前阶段明确走飞书嵌入版路线。**
4. **考勤数据同步**：如分贝通每天晚上加班打车均需要先拉取飞书考勤数据，统计符合要求后方可发起打车。**必须在售前阶段明确走飞书嵌入版路线。**
