## Description: <br>
Automatically searches ClawHub for matching OpenClaw skills when no local skill fits a user's request, presents results, and installs the selected skill after confirmation. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[lpb123](https://clawhub.ai/user/lpb123) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
OpenClaw users and agents use this skill to find a suitable ClawHub skill when local skills do not cover a specialized task, then install the chosen skill with explicit user confirmation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: A selected third-party skill may add new permissions, instructions, or behavior after installation. <br>
Mitigation: Review the search result, publisher, target skill security notes, and ask the user for confirmation before installing. <br>
