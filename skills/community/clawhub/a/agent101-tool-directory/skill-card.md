## Description: <br>
Open directory of 500+ tools, APIs, and services for AI agents. Fetch any category page to get structured tool metadata with auth, free_tier, actions, input/output, example calls, and recommendations. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[vcup-date](https://clawhub.ai/user/vcup-date) <br>

### License/Terms of Use: <br>

## Use Case: <br>
Developers and agent operators use this skill when an agent needs to discover an external tool, API, or service and retrieve category-specific tool metadata for evaluation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Directory recommendations may be incomplete, outdated, or inappropriate for the user's security, cost, privacy, or compliance requirements. <br>
Mitigation: Treat recommendations as references only; verify services against official documentation, pricing, and internal allowlists before use. <br>
Risk: Some listed tools can involve payments, account access, secrets, public posting, code execution, browser automation, or private data. <br>
Mitigation: Require explicit approval before using high-impact tools or sharing sensitive information with any recommended service. <br>
