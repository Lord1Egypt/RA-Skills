# β-内酰胺酶抑制药联合用药原则

## 何时使用
- 病原菌确认或高度怀疑产β-内酰胺酶（如ESBL、AmpC、OXA-48等）；
- 拟选用β-内酰胺类抗生素进行治疗；
- 需对抗β-内酰胺酶介导的耐药机制。

**不适用情形**：
- 细菌不产β-内酰胺酶；
- 耐药机制非β-内酰胺酶介导（如孔蛋白缺失、外排泵过表达等）；
- 单独使用β-内酰胺酶抑制药（因其抗菌活性弱或无）。

## 执行步骤
1. **确认产酶类型**：明确病原菌产生的β-内酰胺酶类别（A/B/C/D类），仅产酶菌适用本策略。
2. **选择匹配组合**：根据酶类型选择对应的抑制药-抗生素复方：
   - **克拉维酸**：抑制A类酶，常与阿莫西林（口服）或替卡西林（静脉）联用；
   - **舒巴坦**：抑酶谱略广于克拉维酸，可与氨苄西林、头孢哌酮或头孢噻肟联用；
   - **他唑巴坦**：抑酶作用强于舒巴坦，常与哌拉西林或头孢洛扎联用；
   - **新型抑制剂**（阿维巴坦、雷利巴坦、法硼巴坦）：可抑制A类和C类酶，阿维巴坦还可抑制OXA-48（D类），常与头孢他啶、亚胺培南或美罗培南联用。
3. **核对药动学匹配性**：确保两药半衰期、分布容积、清除率相近，以维持同步血药浓度。
4. **严格限定适应证**：仅用于产酶菌感染或混合感染，避免滥用。
5. **采用标准复方比例给药**：如阿莫西林0.5g + 克拉维酸0.125g（口服），哌拉西林4g + 他唑巴坦0.5g（静脉），不得随意更改配比。
6. **排除干扰组合**：含西司他汀（肾脱氢肽酶抑制剂）或倍他米隆的复方旨在保护碳青霉烯类免受肾酶降解，不属于β-内酰胺酶抑制范畴，勿混淆。

> 注意：本技能输出为具体复方制剂推荐及给药方案，需结合患者感染部位、肝肾功能及当地耐药谱综合判断。
