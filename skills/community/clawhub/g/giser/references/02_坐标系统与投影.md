# 第一篇：坐标系基础

## 1.1 地球的形状与椭球体

地球不是完美的球体，而是两极稍扁、赤道略鼓的椭球体。GIS 中使用**参考椭球体（Spheroid/Ellipsoid）**来近似地球形状。

**关键参数：**
- **长半轴 (a)**：赤道半径
- **短半轴 (b)**：极半径
- **扁率 (f)**：f = (a - b) / a

**常用椭球体（含 WKID）：**

| **WGS_1984** | 7030 | 6,378,137 | 298.25722356 | GPS/全球 |

## 1.2 大地基准面（Datum）

**大地基准面**定义了椭球体与地球的相对位置关系。同一个椭球体可以有不同的定位方式。

**定位类型：**
- **地心基准**：椭球中心与地球质心重合（如 WGS84、CGCS2000）
- **参心基准**：椭球中心与地球质心有偏移，目的是使椭球面最佳拟合某一区域（如 北京54、西安80）

- **三参数**：ΔX, ΔY, ΔZ（平移）

- 不同基准面之间的坐标转换需要通过**转换参数**完成

## 1.3 地理坐标系（GCS）

**定义**：用经纬度（角度单位）表示地球上位置的三维球面坐标系统。

**组成要素：**
1. 参考椭球体（Spheroid）
2. 大地基准面（Datum）
3. 本初子午线（Prime Meridian）
4. 角度单位（Angular Unit）：通常为度（Degree, 0.0174532925199433 rad）

# 第二篇：地图投影

## 2.1 投影基本概念

**为什么需要投影？** 地球表面是三维的不可展曲面，必须通过数学变换投影到二维平面上才能制作地图。

**投影变形分类：**
- **等角投影（Conformal）**：保持角度不变，形状正确，但面积会变形
- **等积投影（Equal Area）**：保持面积不变，但角度和形状会变形
- **等距投影（Equidistant）**：沿特定方向保持距离不变
- **折衷投影（Compromise）**：各种变形都控制在一定范围内

**投影方法分类：**
1. **圆柱投影（Cylindrical）**：投影面为圆柱
2. **圆锥投影（Conic）**：投影面为圆锥
3. **方位投影（Azimuthal）**：投影面为平面

---

## 2.2 高斯-克吕格投影（Gauss-Kruger）

中国测绘最核心的投影方式！

**特征：**
- 等角横切椭圆柱投影
- 中央经线无变形，离中央经线越远变形越大
- 属于**等角投影**

**分带方式：**

| 分带 | 经差 | 带号范围 | 应用 |
|---|---|---|---|
| **6°带** | 6° | 13-23（中国） | 1:2.5万~1:50万比例尺 |
| **3°带** | 3° | 24-45（中国） | 1:1万及更大比例尺 |

**6°带计算公式：** 带号 = floor((L+3)/6)
**3°带计算公式：** 带号 = floor(L/3)

例：东经 111° → 6°带：floor((111+3)/6) = 19 带 → 中央经线 = 111°

**带号与中央经线关系：**
- 6°带中央经线 L0 = 6n - 3
- 3°带中央经线 L0 = 3n

**投影参数：**
- **False Easting（东伪偏移）**：500,000 m（避免坐标出现负值）
- **False Northing（北伪偏移）**：0（北半球）/ 10,000,000 m（南半球）
- **Central Meridian（中央经线）**：分带中央经线
- **Scale Factor（比例因子）**：1.0（高斯-克吕格）或 0.9996（UTM）
- **Latitude of Origin（起始纬度）**：0°

**ArcGIS 投影参数数组索引：**

| 索引 | 参数 | 说明 |
|---|---|---|
| 0 | False_Easting | 东伪偏移 |
| 1 | False_Northing | 北伪偏移 |
| 2 | Central_Meridian | 中央经线 |
| 3 | Standard_Parallel_1 | 第一标准纬线 |
| 4 | Standard_Parallel_2 | 第二标准纬线 |
| 5 | Scale_Factor | 比例因子 |
| 6 | Latitude_Of_Origin | 原点纬度 |
| 7 | Azimuth | 方位角 |

**比例因子选择原则：**
- **Gauss-Kruger**：固定 1.0，通过分带控制变形
- **UTM**：固定 0.9996，中央经线处略有压缩，扩大低变形带宽度
- **Lambert 1SP**：可自定义，通常接近 1.0

## 2.3 UTM 投影（Universal Transverse Mercator）

国际上使用最广泛的投影系统。

**与高斯-克吕格的区别：**
- 属于**等角横轴割圆柱投影**（高斯-克吕格是切圆柱）
- 比例因子 = 0.9996（中央经线缩小 0.04%）
- 分带方式：6°分带，经差 6°
- 带号从 180°W 向东编号（1-60）

| 特性 | 高斯-克吕格 | UTM |
|---|---|---|
| 投影面 | 切圆柱 | 割圆柱 |
| 比例因子 | 1.0 | 0.9996 |
| 中央经线变形 | 无 | 缩小 0.04% |
| 边缘变形 | 较大 | 较均匀 |
| 中国应用 | 测绘标准 | 卫星影像/国际数据 |

## 2.5 投影选择原则

| 区域特征 | 推荐投影 |
|---|---|
| 中纬度、东西延伸 | Lambert Conformal Conic |
| 中纬度、南北延伸 | Transverse Mercator（横轴墨卡托） |
| 赤道附近 | Mercator |
| 极地地区 | Azimuthal（方位投影） |
| 世界范围 | Robinson / Winkel Tripel（折衷投影） |
| 中国标准测绘 | **Gauss-Kruger 3°/6°带** |
| Web 地图 | Web Mercator (EPSG:3857) |
