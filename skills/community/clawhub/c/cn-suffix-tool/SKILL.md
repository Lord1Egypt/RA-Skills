# cn-suffix-tool

文本后缀添加工具。为每行添加指定后缀。

## 使用方法

```bash
python3 scripts/add_suffix.py --text "第一行\n第二行" --suffix ";"
python3 scripts/add_suffix.py --text "第一行\n第二行" --suffix " ✓"
```

纯Python标准库。
