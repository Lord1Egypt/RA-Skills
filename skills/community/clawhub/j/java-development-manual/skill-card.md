## Description: <br>
Provides Java coding standards guidance based on the Alibaba Java Development Manual Songshan edition, covering programming conventions, exception logging, unit testing, security, MySQL, project structure, and design. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and code reviewers use this skill to write, review, and troubleshoot Java projects against documented conventions for code style, exception handling, logging, tests, security, MySQL schema and query design, project structure, and design practices. <br>
