# 晨报模板

## 标准格式

```
早上好，霞姐 ☀️

**📋 项目进展**

[项目名] — [一句话状态]
  → [最关键风险或下一步行动]

[项目名] — [一句话状态]
  → [最关键风险或下一步行动]

**⏳ 待处理事项**

- [机器/系统] [具体行动] （来源：YYYY-MM-DD 日志）
- ...

**📅 今日日程**（如有）

- HH:MM–HH:MM [日程名]
- ...

——小C
```

## 示例

```
早上好，霞姐 ☀️

**📋 项目进展**

PG长稳压测（vm10-47-7-81） — listen_addresses 未恢复，VIP 5432 拒连
  → 执行 ALTER SYSTEM SET listen_addresses='*'; pg_ctl reload（需 postgres 用户）

PG长稳压测（vm10-47-7-197） — ⚠️ 磁盘 100% 满，sysbench prepare 失败
  → 联系基础设施团队修复底层块设备，或换实例；死循环压测（5并发）仍在运行

**⏳ 待处理事项**

- vm10-47-7-81：恢复 listen_addresses + 重启压测（4进程×32线程）
- vm10-47-7-197：磁盘虚报问题上报基础设施，sysbench 待换实例

**📅 今日日程**

- 14:30–16:00 会议（未绑定会议室）

——小C
```

## 无项目时

```
早上好，霞姐 ☀️

没有待跟进的项目，今天轻松一点 🎉

**📅 今日日程**

- 暂无日程

——小C
```
