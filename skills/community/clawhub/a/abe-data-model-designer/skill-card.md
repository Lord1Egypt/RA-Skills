## Description: <br>
Design data models for construction projects. Create entity-relationship diagrams, define schemas, and generate database structures. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[alvisdunlop](https://clawhub.ai/user/alvisdunlop) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Construction data teams, developers, and project analysts use this skill to define project entities, relationships, schemas, SQL DDL, JSON Schema, and Mermaid ER diagrams for construction data workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can send construction project content to a third-party AI API. <br>
Mitigation: Use it only with data that is approved for transmission to the heybossai/SkillBoss service, and prefer a local-only workflow when project confidentiality requires it. <br>
Risk: Generated schemas, relationships, and diagrams may be incomplete or incorrect for a project. <br>
Mitigation: Review generated data models against project requirements, construction standards, and database constraints before deployment. <br>
