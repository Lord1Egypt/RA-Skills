# 本质分析工具：benzhi

## 两种分析模式

### ⚡ 快速模式（Quick Mode）

当用户只是随口问，或者需要快速洞察时，输出3行：

```
【现象】...
【核心维度】...
【本质结论】一句话
```

### 🔬 深度模式（Deep Mode）

当现象复杂，启用完整10链流程。

---

## 触发条件

**✅ 触发：**
- "XX的本质是什么？"
- "分析一下XX的本质"
- "挖一下XX的本质"
- "为什么XX？"
- 任何明确要求分析本质的问题

**自动判断：**
- 简单现象（一句话能说清核心维度）→ 快速模式
- 复杂现象（多维度、多假设）→ 深度模式

**❌ 不触发：**
- 事实查询、操作指导、无关闲聊

---

## 核心工具链（深度模式）

### 第一链：现象扫描 + 十维假设（带权重）

**第一步：判断现象类型，分配维度优先级**

```
【个体行为类】（个人决策、情绪、习惯、成瘾）
  优先级：潜意识 > 生物 > 思维 > 生活 > 哲学 > 逻辑 > 商业 > 政治 > 文化 > 历史

【社会互动类】（人际、群体、关系、聚会）
  优先级：生活 > 政治 > 文化 > 逻辑 > 潜意识 > 思维 > 哲学 > 商业 > 生物 > 历史

【商业组织类】（公司、市场、竞争、利润）
  优先级：商业 > 政治 > 逻辑 > 思维 > 生活 > 经济 > 生物 > 文化 > 哲学 > 潜意识

【认知思考类】（学习、创新、决策、偏见）
  优先级：思维 > 逻辑 > 潜意识 > 生物 > 哲学 > 文化 > 生活 > 政治 > 商业 > 历史

【存在意义类】（生命、死亡、自由、孤独）
  优先级：哲学 > 潜意识 > 生物 > 历史 > 文化 > 思维 > 生活 > 逻辑 > 政治 > 商业

【权力结构类】（政治、制度、控制、垄断）
  优先级：政治 > 历史 > 经济 > 商业 > 逻辑 > 哲学 > 文化 > 生活 > 思维 > 生物

【文化现象类】（习俗、节日、语言、艺术）
  优先级：文化 > 历史 > 生活 > 哲学 > 逻辑 > 潜意识 > 思维 > 生物 > 政治 > 商业
```

**第二步：十维假设（每维度生成最可能的假设）**

```
┌─────────────────────────────────────────────────────────────┐
│                      十维框架                                │
├──────┬──────────────────────────────────────────────────────┤
│ 商业  │ 利益、成本收益、交易结构、竞争格局、资源配置           │
│ 生活  │ 习惯习俗、社会角色、人情世故、日常便利                 │
│ 逻辑  │ 因果链条、条件关系、逆否命题、同义反复                 │
│ 生物  │ 本能驱动、进化适应、神经化学、生理节律                 │
│ 思维  │ 认知模型、决策偏见、心智模式、信息处理                 │
│ 潜意识│ 欲望驱动、防御机制、创伤印痕、未满足需求               │
│ 哲学  │ 存在意义、自由意志、本体论、道德判断                   │
│ 政治  │ 权力博弈、控制结构、利益集团、合法性来源 ⭐新增        │
│ 文化  │ 价值观、符号系统、集体记忆、社会建构 ⭐新增            │
│ 历史  │ 演化路径、路径依赖、历史积累、阶段特征 ⭐新增          │
└─────────────────────────────────────────────────────────────┘

**⚠️ 文化维度使用限制（必须满足全部条件才可使用）：**
```
❌ 禁止："这是文化" → 泛泛而谈，没有具体机制
❌ 禁止："A文化这样，B文化那样" → 跨文化比较，无充分了解
❌ 禁止："人这样行为，因为文化" → 文化决定论，忽略个体差异

✅ 必须：指定"哪个文化"的什么机制（价值观/符号/记忆）
✅ 必须：说明文化→行为的传导机制（不是"文化导致"而是"A文化通过B机制影响C行为"）
✅ 必须：承认同一文化内的个体差异
✅ 必须：跨文化比较时，有充分的第一手知识
```

**⚠️ 历史维度使用限制（必须满足全部条件才可使用）：**
```
❌ 禁止："历史证明X是对的" → 历史是事后总结，不是原因
❌ 禁止："路径依赖，所以必须这样" → 路径依赖≠唯一结果
❌ 禁止："从古至今都是" → 声称连续性，但未验证

✅ 必须：历史维度是"时间视角"，不是"因果解释"
✅ 必须：说清楚"在这个历史节点上，什么机制导致了Y"
✅ 必须：承认历史 alternatives（其他可能性真实存在）
✅ 必须：区分"相关性"和"因果性"，不把时序当因果
```

生成规则：
- 优先级前3的维度：各生成2个假设（最可能的 + 备选）
- 优先级4-6的维度：各生成1个假设
- 优先级后4的维度：快速判断有关联才生成，无关联跳过

格式：
【维度名称】H1: [假设内容]
  快速理由：[为什么这个假设可能是对的]
```

**假设数量：目标6-10个（比v2.5的8个更精简）**

### 第二链：快速验证

**快速排除标准（任一满足即排除）：**

```
❌ 有1个基本事实反例
❌ 与已保留假设重复（被其他假设覆盖）
❌ 无法区分于其他假设（边界模糊）
```

**快速确认标准：**
```
✅ 有日常体验支撑
✅ 有跨学科研究支持
✅ 能区分于其他假设
```

---

### 第三链：深度追问（带停止标准）

**⚠️ 停止标准（5个，满足任一即停止）：**

```
1️⃣ 触及"不可还原的物理/化学/生物机制"
   → 例："因为多巴胺分泌" → 到根（神经科学层面，无法追问为什么有多巴胺）

2️⃣ 触及"价值选择"而非"事实原因"
   → 例："因为人不想痛苦" → 到根（价值层面，继续追问无意义）

3️⃣ 触及"进化本能"
   → 例："因为生存需要" → 到根（进化层面）

4️⃣ 回答变成"同义反复"
   → 例："为什么x？因为x就是这样" → 停止，换角度追问

5️⃣ 下一个"为什么"不产生新信息
   → 例："因为A→B，而B的本质就是A的变体" → 停止

⚠️ 触及"宇宙/存在的终极问题" → 停止
   → 例："为什么存在宇宙？" → 哲学深渊，不是本质分析的目标

✅ 收敛信号：多个追问链指向同一个变量
```

**⚠️ 多假设追问规则：**

```
只深度追问保留假设中的前3个维度（优先级最高的）
其他维度假设：在"收敛检查"时整合

每假设追问链不超过4轮
如果4轮后仍未收敛，标记"待定"，继续下一个假设
```

---

### 第四链：正推验证

```
如果核心变量成立（→），会发生什么？
- 推导链条：A → B → C → D
- 检验：D是否真实会发生？
- D不发生 → 核心变量可能有问题，需要修正
```

---

### 第五链：反推验证

```
如果现象不发生，缺失了什么？
- 缺失核心变量，现象是否消失？
- 现象依然存在 → 核心变量找错了
```

---

### 第六链：反例攻击（⚠️ 必须强制执行）

```
反例攻击是强制步骤，不可跳过。

模板（两个方向都要检查）：

方向A："有没有A成立，但现象不发生的情况？"
方向B："有没有现象发生，但A不成立的情况？"

处理方式：
- 有反例 → 记录反例 → 修正结论（缩小适用范围）
- 无反例 → 置信度提升

⚠️ 强制要求：必须找到至少1个反例
如果找不到 → 问自己：是因为结论太强（不可证伪）吗？
```

**反例挖掘方向提示：**

```
从以下角度找反例：
- 个体差异（有人不这样，为什么？）
- 文化差异（别的文化不这样，为什么？）
- 历史差异（过去不这样，为什么？）
- 极端情况（到了极致会反转吗？）
- 相邻现象（类似现象为什么不成立？）
```

---

### 第七链：收敛验证

```
正推 ✅ + 反推 ✅ + 反例攻击 ✅

收敛检查：
- 三个方向是否指向同一核心变量？
- 是 → 核心变量确认
- 否 → 返回第三链，重新追问
```

---

### 第八链：悖论挖掘

```
各维度的内在矛盾：
- 越强调A，越削弱B？
- 最有效的手段，会不会反噬目标？
- 各维度之间有没有结构性冲突？
```

---

### 第九链：边界检验

```
各维度的失效条件：
- 当...时，结论不成立
- 极端情况会怎样？到了极致会反转吗？
```

### 第十链：元层追问（Meta-Inquiry）⭐新增

**⚠️ 这是本版本最重要的创新。框架的盲区，只能靠元层追问来弥补。**

```
分析完成后，自动执行元层追问：

┌─────────────────────────────────────────────────────────────┐
│                    元层追问（强制执行）                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  🔍 我的框架遗漏了什么？                                      │
│     - 有什么反例，我的框架解释不了？                           │
│     - 有什么视角，完全不在十维框架内？                          │
│     - 这次分析中，什么被默认假设但没有被追问？                  │
│                                                             │
│  🔍 我的分析过程有什么问题？                                   │
│     - 有没有跳过反例攻击？                                    │
│     - 追问有没有过早停止？                                    │
│     - 有没有预设结论，然后找证据？                            │
│                                                             │
│  🔍 这次分析发现了什么值得更新到框架？                          │
│     - 新假设被验证 → 加入假设库                               │
│     - 框架维度有盲区 → 记录，下次分析注意                       │
│     - 停止标准有问题 → 修正标准                               │
│     - 文化/历史维度被正确使用了吗？（是否违反使用限制？）          │
│                                                             │
│  🔍 这个结论的元层面结论是什么？                                │
│     - 如果这个结论是对的，那么它的"父结论"是什么？             │
│     - 这个结论能被更大的框架解释吗？                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**示例（元层追问结果）：**

```
现象：为什么人会上瘾

元层追问结果：
- 【框架盲区】没有政治维度，但成瘾产业涉及权力（政府vs烟草公司）
- 【框架盲区】没有历史维度，但"上瘾"这个概念本身是现代建构
- 【过程问题】潜意识维度的两个假设被过早排除——可能是对的但被生物维度覆盖了
- 【新发现】"成瘾=回路锁定"这个框架可以解释其他成瘾现象——自我强化学说
- 【元结论】成瘾的本质是"自我主权的丧失"，回路锁定是机制，不是本质
```

**元层追问的输出格式：**

```markdown
### 元层发现
- 框架盲区：[...]
- 过程问题：[...]
- 框架更新：[...]
- 元结论：[...]
```

## 完整分析流程（深度模式）

```
① 现象扫描 + 判断类型 + 分配维度优先级
                    ↓
② 十维假设（6-10个假设，生成即筛选）
                    ↓
③ 快速验证（找1个基本反例即排除）
                    ↓
④ 深度追问前3优先级维度（带停止标准）
                    ↓
⑤ 正推验证 + 反推验证
                    ↓
⑥ 反例攻击 ← ⚠️ 必须强制执行，至少1个反例
                    ↓
⑦ 收敛验证（通过 → 确认核心变量）
                    ↓
⑧ 悖论挖掘 + 边界检验
                    ↓
⑨ 元层追问 ← ⭐ 框架盲区自查，更新框架本身
                    ↓
⑩ 本质结论 + 元层发现
```

## 自我学习机制

### 被动记录（每分析后）

```
CONTEXT: [现象]
CONCLUSION: [结论]
CORE_DIMENSION: [核心维度]
```

## 核心原则

1. **精确触发** — "本质"或"为什么"触发，快速/深度由现象决定
2. **权重优先** — 先判断类型，按优先级追
3. **生成即筛选** — 每维度1-2个假设，总数6-10个
4. **追问有标准** — 5个停止标准，避免过早/过晚停止
5. **反例强制** — 必须找到至少1个反例，找不到要反思
6. **收敛为止** — 正推+反推+反例全部通过
7. **元层追问** — ⭐ 每次分析后必须追问框架本身
8. **悖论+边界** — 按维度检查矛盾和失效条件
9. **记录+迭代** — 被动记录+主动更新框架
