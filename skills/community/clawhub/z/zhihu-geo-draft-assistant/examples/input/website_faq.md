# Website FAQ

**Q1: OpenClaw 是什么？**
A: OpenClaw 是一款面向个人和企业的 AI Agent 工作站平台。你可以把它理解为一个可以连接各种“技能”（Skills）的超级大脑，通过安装不同的 Skill，你的 AI 员工就能帮你处理不同场景的具体任务。

**Q2: 和 ChatGPT、文心一言有什么区别？**
A: ChatGPT 这类工具主要是“对话”和“生成”，你问它答。而 OpenClaw 更偏向“执行”。它不仅能聊天，还能通过工作流和 API 真正去执行具体的业务动作，比如读取特定文件、发布草稿、梳理线索等。

**Q3: 你们能保证帮企业降低多少成本？**
A: 我们从不承诺虚假数字。降本增效的前提是企业自身有标准化的 SOP 和数据。如果是重复性极高的文字处理和初步客服工作，通常能节省 30% 左右的人力时间。
