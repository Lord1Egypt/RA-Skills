## 模板一：通用面试评分表（30s+3min+15min）

**适用场景：** 所有岗位的初次面试

### 前30秒：静态观察评分

| # | 观察点 | 3分（好） | 2分（正常） | 1分（警惕） | 得分 |
|---|---|---|---|---|---|
| 1 | 鞋履/指甲洁净度 | 细节到位，干净整洁 | 无明显问题 | 脏污/不修边幅 | /3 |
| 2 | 初始坐姿 | 舒展自然，不刻意 | 正常 | 过度蜷缩或过度瘫坐 | /3 |
| 3 | 眉头状态 | 自然舒展 | 无明显皱眉 | 习惯性紧锁 | /3 |
| 4 | 眼神基线 | 有光，自然扫视环境 | 正常 | 浑浊/涣散/过度警惕 | /3 |
| 5 | 握手力度 | 力度适中+眼神接触 | 正常 | 绵软/过猛/躲闪 | /3 |

**第一阶段小计：__/15**
*如果第一阶段≤6分，后续面试仍可进行，但需在最终决策时标注"静态气质初筛预警"*

### 前3分钟：压力测试评分

**测试问题：** "请用1分钟说下你最失败的一个项目/经历"

| # | 观察点 | 3分（好） | 2分（正常） | 1分（警惕） | 得分 |
|---|---|---|---|---|---|
| 6 | 语速与流畅度 | 流畅有停顿，有具体细节 | 基本流畅 | 卡顿/改口/模糊 | /3 |
| 7 | 眼神（此问题中） | 自然平视，不躲闪不死盯 | 基本正常 | 飘忽/过度对视/频繁眨眼 | /3 |
| 8 | 归因模式 | 承认自身责任+系统改进 | 部分承认 | 全是他人/环境问题 | /3 |
| 9 | 被打断时的反应 | 倾听+点头+继续 | 基本接受 | 皱眉/抢话/急躁 | /3 |
| 10 | 安慰行为频率 | 几乎没有 | 1-2次 | 频繁（3次以上） | /3 |

**第二阶段小计：__/15**

### 前15分钟：四维深度评分

| 维度 | 观察要点 | 5分 | 4分 | 3分 | 2分 | 1分 | 得分 |
|---|---|---|---|---|---|---|---|
| 情绪稳定性 | 压力下的反应模式 | 极稳 | 稳 | 正常 | 偏弱 | 脆弱 | /5 |
| 掌控感 | 自信与担责意愿 | 松弛掌控 | 自信 | 正常 | 偏弱 | 自卑 | /5 |
| 诚实度 | 一致性+归因+透明度 | 完全可信 | 可信 | 正常 | 有疑点 | 不可信 | /5 |
| 共情能力 | 镜像反应+对他人态度 | 极强 | 强 | 正常 | 偏弱 | 极弱 | /5 |

**第三阶段小计：__/20**

### 六型人格定位（基于前三阶段综合判断）

| 判断项 | 结果 |
|---|---|
| 主型 | ☐ 将帅 ☐ 先锋 ☐ 谋士 ☐ 基石 ☐ 成长 ☐ 跨界（___×___） |

## 模板三：面相变化追踪表

**适用场景：** 内部员工/合伙人的长期追踪（季度/半年一次）

| 观察点 | 上次状态 | 本次状态 | 变化方向 | 解读 |
|---|---|---|---|---|
| 面色 | | | ☐变好 ☐不变 ☐变差 | |
| 眼神 | | | ☐变定 ☐不变 ☐变飘 | |
| 眉心 | | | ☐变舒展 ☐不变 ☐变紧 | |
| 嘴角 | | | ☐变上扬 ☐不变 ☐变下垂 | |
| 姿态 | | | ☐变舒展 ☐不变 ☐变缩 | |
| 整体 | | | ☐面相在变好 ☐不变 ☐面相在变差 | |

**解读指南：**
- **面相在变好：** 正在正向成长，"善能改相"在发生→值得投资
- **不变：** 状态稳定，正常
- **面相在变差（1-2个信号）：** 关注，增加沟通频率
- **面相在变差（3个以上信号）：** 高度关注，主动沟通了解情况——可能在酝酿离职/做错事/承受过度压力

## 模板四：客户/伙伴评估清单

**适用场景：** 判断客户靠谱度/合作风险/长期价值

| 维度 | 评估项 | 5分 | 3分 | 1分 | 得分 |
|---|---|---|---|---|---|
| 诚信 | 谈资金/付款时有无安慰行为 | 无 | 偶有 | 频繁 | /5 |
| 诚信 | 对合同/约定是否认真 | 逐条看 | 大致看 | 不看就签 | /5 |
| 实力 | 面色/眼神状态（财务健康度） | 清亮有光 | 正常 | 暗沉浑浊 | /5 |
| 实力 | 团队/员工状态 | 积极舒展 | 正常 | 压抑紧绷 | /5 |
| 格局 | 对供应商/合作伙伴的评价 | 客观公正 | 中规中矩 | 贬低吐槽 | /5 |
| 格局 | 是否愿意让你有合理利润 | 主动提 | 可接受 | 反复压价 | /5 |
| **总分** | | | | | **__/30** |

**评级：**
- 25-30：S级客户/伙伴，长期深度合作
- 20-24：A级，正常合作
- 15-19：B级，控制合作规模和回款周期
- 10-14：C级，谨慎，先小规模试
- <10：D级，不建议合作

## 模板使用说明

1. **面试时不要当面打分**——对方会感知到你在"打分"而进入防御状态。面试后在5分钟内完成评分（记忆新鲜）。
2. **照片辅助：** 如果可能，面试时在不打扰的前提下拍一张自然状态的照片，后续和入职后的照片对比"面相变化追踪"。
3. **多人交叉验证：** 重要岗位至少2人独立打分后对比。如果两个人的评分差异超过10分→安排第三轮面试。
4. **定期校准：** 每半年回顾一次之前的评分 vs 实际表现，校准自己的判断标准。
