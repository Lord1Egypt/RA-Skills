SimpliXio Decision Loop is an OpenClaw skill that helps agents reduce messy context into:

- **what matters**
- **why it matters**
- **what to do next**
- **what to ignore**

Built for founders, product engineers, AI builders, and anyone drowning in too many tasks, notes, ideas, and signals.

## Use it

```text
Use the SimpliXio Decision Loop to decide what I should do next.
```

```text
Use Weekly Review Mode and tell me what repeated, what mattered, and what I should build next.
```

```text
Use Marketing Automation Mode and turn the latest product output into proof-based social drafts.
```

## Modes

| | |
|---|---|
| 🎯 | **Decision Brief**: 3 priorities, why, action |
| 📅 | **Weekly Review**: repeated signals, next focus |
| 📣 | **Marketing Automation**: product-proof drafts |
| 🧹 | **Ignore For Now**: remove low-value noise |

## Install

```bash
mkdir -p ~/.openclaw/skills
cp -R simplixio-decision-loop ~/.openclaw/skills/
```

Then refresh OpenClaw skills.
