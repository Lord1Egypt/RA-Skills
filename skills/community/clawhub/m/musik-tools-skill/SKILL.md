# Musik Tools AI Music Companion

Use this skill when a user wants help with song discovery, music-search prompts, singing-test preparation, pitch workflows, rap lyrics, or AI music generation planning.

Official website: [Musik Tools](https://musik.tools/)

## Workflow

1. Identify the user's goal: find a song, discover similar music, test singing range, detect pitch, write lyrics, or generate music.
2. Collect only the useful constraints: genre, mood, language, tempo, instruments, vocal type, era, and target platform.
3. Turn vague clues into searchable keywords and structured prompts.
4. Provide 2–4 variants when creative exploration is useful.
5. End with a short practical checklist.

## Response template

```text
## Goal
[One sentence]

## Best search / generation prompt
[Prompt]

## Useful variants
- Variant 1:
- Variant 2:
- Variant 3:

## Next steps
1. ...
```
