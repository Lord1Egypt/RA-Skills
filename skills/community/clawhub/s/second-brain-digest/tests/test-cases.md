# Test Cases — second-brain-digest

## Test 1：基础触发 — 文章消化

**Input:** `帮我消化这篇文章：[粘贴一段文字]`

**Expected:**
- ✅ 触发本 skill
- ✅ 识别内容类型并告知用户
- ✅ 输出至少 1 张标准格式卡片

- ❌ 不只是缩短原文

## Test 2：原子性验证

**Input:** 包含 3 个独立观点的长文

**Expected:**
- ✅ 拆成 3 张独立卡片
- ✅ 每张卡片只有一个核心想法
- ❌ 不把所有内容塞进一张卡片

## Test 3：卡片标题质量

- ❌ 标题不是"XXX的优缺点"这类话题短语

## Test 4：关联发现
