# 销售与接单流程调研指南

## 调研目标
- 了解从线索到回款(Lead-to-Cash)的完整销售与接单流程
- 识别订单全生命周期管理中的痛点
- 评估销售预测与需求管理能力

## 调研重点

### 1. 订单管理流程
- 订单从哪里来？（ERP/CRM/邮件/微信/电话？）
- 订单输入方式（人工录入/系统对接/EDI？）
- 订单确认流程（谁确认交期？确认依据是什么？）
- 订单变更管理（变更流程？变更频次？如何通知相关部门？）
- 急单/插单处理机制

### 2. 销售预测
- 是否做销售预测？预测周期多长？
- 预测准确率如何评估？
- 预测如何传递给计划和采购？
- 有没有滚动预测机制？

### 3. 客户与产品信息
- 客户ABC分类管理
- 产品品类与复杂度
- 非标/定制比例
- 客户价格体系

### 4. 订单交期管理
- 标准交期如何定义？
- 交期承诺流程（销售直接承诺/需计划确认？）
- 交期达成率(OTD)现状
- 订单延期的主要原因

### 5. 信息系统
- CRM系统使用情况
- ERP订单模块使用深度
- 订单进度是否可追溯（客户能否查？）

## 常见问题诊断库

| 问题现象 | 可能根因 | 典型影响 |
|---------|---------|---------|
| 交期承诺不准确 | 销售直接承诺/无产能数据支撑 | 交货违约/加急成本 |
| 订单变更频繁 | 客户管理不规范/设计变更流程缺失 | 计划失效/库存呆滞 |
| 预测准确率低 | 预测方法粗放/无反馈闭环 | 库存积压/缺料并存 |
| 急单插单多 | 接单规则缺失/无优先级评估 | 计划频繁调整/效率下降 |
| 订单进度不透明 | 无订单追踪系统/信息孤岛 | 客户满意度低 |
| 订单录入错误多 | 手工录入/无校验/重复录入 | 生产错误/返工 |

## 数据采集项
- 年度销售额、订单数量
- 月平均订单数
- 急单占比（数量/金额）
- 订单变更率
- 交期达成率(OTD)
- 产品分类（标品vs非标比例）
