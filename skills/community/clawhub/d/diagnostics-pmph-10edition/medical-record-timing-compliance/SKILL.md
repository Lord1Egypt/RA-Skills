# 医疗文书时限合规书写

## 何时使用
- 患者完成门（急）诊就诊后
- 抢救措施实施结束后
- 患者办理住院手续后

## 执行步骤
1. **识别病历类型**：确认当前需书写的病历属于以下哪一类：
   - 门（急）诊病历
   - 抢救记录
   - 入院记录

2. **确定书写时限**：
   - **门（急）诊病历**：在患者就诊当时或结束后立即书写。
   - **抢救记录**：在抢救结束后6小时内据实补记。
   - **入院记录**：在患者入院后24小时内完成。

3. **规范时间标注**：
   - 使用阿拉伯数字书写日期和时间（例如：2024-06-15 14:30）。
   - 采用24小时制，禁止使用“上午/下午”或12小时制。
   - 时间精确到分钟；抢救等关键事件需精确到秒。

4. **处理延迟书写**：
   - 仅在紧急抢救等特殊情况下允许事后补记。
   - 必须同时注明事件实际发生时间与记录书写时间。
   - 由具备执业资格的医务人员签名确认。

5. **验证完整性**：确保记录包含该类型病历所需的全部法定内容项（如入院记录需含主诉、现病史、体格检查、初步诊断等）。

> 注意：本技能不适用于非正式医疗笔记或个人备忘录。
