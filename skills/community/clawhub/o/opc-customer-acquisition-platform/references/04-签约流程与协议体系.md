# 签约流程与协议体系

> 本文档详细说明SPV协议适配、必签附件清单、签约SOP等签约相关内容。

---

## 一、协议体系架构

### 1.1 协议层级

```
┌─────────────────────────────────────────────────────────────┐
│                    协议签署体系                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【主协议】SPV合作协议                                        │
│  ├── 签约方：全体项目成员                                     │
│  ├── 内容：贡献权重、治理机制、分账规则                        │
│  ├── 效力：整个项目周期                                       │
│  └── 模板：contracts/05-SPV项目合作协议-获客分包版.md         │
│                                                              │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │  【附件体系】                                            │ │
│  ├─────────────────────────────────────────────────────────┤ │
│  │  ├── 【附件1】分包协议（各分包模块）                      │ │
│  │  │   └── 签约方：挂帅人+各接包人                         │ │
│  │  │                                                           │ │
│  │  ├── 【附件2】保密协议                                     │ │
│  │  │   └── 签约方：全体成员                                 │ │
│  │  │                                                           │ │
│  │  ├── 【附件3】分账协议                                     │ │
│  │  │   └── 签约方：全体成员                                 │ │
│  │  │   └── 内容：分账比例、支付时间、税务处理               │ │
│  │  │                                                           │ │
│  │  ├── 【附件4】质量承诺书                                   │ │
│  │  │   └── 签约方：各执行成员                               │ │
│  │  │                                                           │ │
│  │  └── 【附件5】里程碑验收标准                               │ │
│  │      └── 签约方：全体成员                                 │ │
│  │      └── 内容：各里程碑目标+验收标准                       │ │
│  └─────────────────────────────────────────────────────────┘ │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 协议类型与项目匹配

| 项目类型 | 推荐协议模式 | 说明 |
|---------|------------|------|
| 短期项目（<30天） | 简化SPV协议 | 精简版，聚焦交付 |
| 中期项目（30-90天） | 标准SPV协议 | 完整版，含治理+分账 |
| 长期项目（>90天） | 完整SPV协议 | 完整版+动态调整机制 |
| 联合体投标 | 联合体SPV协议 | 多SPV协作模式 |
| 持续合作 | 框架协议+单次SPV | 年度框架+项目分签 |

## 二、必签附件清单

### 2.1 附件清单总表

| 附件 | 必须签署 | 适用场景 | 模板来源 | 签约方 |
|------|---------|---------|----------|--------|
| **保密协议** | ✅ 必须 | 所有项目 | contracts/06-保密协议.md | 全体成员 |
| **分账协议** | ✅ 必须 | 有收益分配的项目 | contracts/09-分账确认书.md | 全体成员 |
| **质量承诺书** | ✅ 必须 | 有交付物的项目 | contracts/07-质量承诺书.md | 各执行成员 |
| **里程碑验收标准** | ✅ 必须 | 有阶段交付的项目 | contracts/08-里程碑验收表.md | 全体成员 |
| **分包协议** | ✅ 必须 | 有分包的项目 | contracts/04-分包协议.md | 挂帅人+接包人 |
| **共管账户协议** | 建议签署 | 涉及资金的项目 | 股权合作Skill | 财务相关方 |
| **退出清算协议** | 建议签署 | 长期合作项目 | 股权合作Skill | 全体成员 |

### 2.2 各附件签署时机

| 附件 | 签署时机 | 前置条件 |
|------|----------|----------|
| **SPV主协议** | 项目启动前 | 挂帅人确定 |
| **分包协议** | 分包确定后 | 接包人确认 |
| **保密协议** | 项目启动前 | 全体成员确认 |
| **分账协议** | 项目启动前 | 分账方案确定 |
| **质量承诺书** | 执行启动前 | 各执行成员确认 |
| **里程碑验收标准** | 项目启动前 | 里程碑计划确定 |

## 三、签约SOP

### 3.1 标准签约流程

```
┌─────────────────────────────────────────────────────────────┐
│                    签约标准流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【步骤1】签约发起                                            │
│  ├── 挂帅人发起签约流程                                      │
│  ├── 生成协议草稿                                            │
│  └── 通知各方准备签署                                        │
│                                                              │
│  【步骤2】协议审核                                           │
│  ├── 各方确认协议内容                                        │
│  ├── 提出修改意见（如有）                                    │
│  └── 汇总修改意见                                            │
│                                                              │
│  【步骤3】意见处理                                           │
│  ├── 合理意见纳入协议                                        │
│  ├── 不采纳需说明原因                                        │
│  └── 更新协议版本                                            │
│                                                              │
│  【步骤4】定稿确认                                           │
│  ├── 各方确认最终版本                                        │
│  ├── 生成正式协议文件                                        │
│  └── 各方电子签章                                            │
│                                                              │
│  【步骤5】见证生效                                           │
│  ├── 三长老见证签字（如需）                                  │
│  ├── 协议上链存证                                            │
│  └── 协议正式生效                                            │
│                                                              │
│  【步骤6】档案归档                                           │
│  ├── 协议分发各参与方                                        │
│  ├── 协议归档存储                                            │
│  └── 通知项目启动                                            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 签约时效要求

| 协议类型 | 审核时间 | 签署时间 | 超时处理 |
|----------|----------|----------|----------|
| **SPV主协议** | 3天内 | 收到后2天内 | 催促/升级处理 |
| **分包协议** | 1天内 | 收到后1天内 | 影响项目启动 |
| **保密协议** | 即时 | 即时 | 影响参与资格 |
| **分账协议** | 2天内 | 收到后1天内 | 冻结分账 |
| **质量承诺书** | 即时 | 即时 | 影响执行资格 |

## 四、SPV协议适配指南

### 4.1 获客分包版SPV协议特点

| 对比项 | 股权合作版 | 获客分包版 |
|--------|-----------|-----------|
| **合作基础** | 技能/资源/资金出资 | 项目制分包 |
| **收益模式** | 贡献权重分配 | 项目制分账 |
| **合作周期** | 中长期合作 | 项目周期为主 |
| **治理结构** | 三长老治理 | 简化治理（可选项） |
| **退出机制** | 复杂退出清算 | 项目结束即终止 |
| **灵活性** | 动态调整权重 | 固定分账比例 |

### 4.2 协议适配要点

**必须修改的内容**：

| 条款类型 | 股权合作版 | 获客分包版 |
|----------|-----------|-----------|
| **贡献权重** | 动态调整权重 | 固定分账比例（项目期间不变） |
| **分账方式** | 按月/按季分配 | 按里程碑分配 |
| **治理结构** | 完整三长老 | 可简化为"挂帅人负责制" |
| **退出机制** | 详细退出清算 | 简化为"项目终止+尾款结算" |
| **竞业限制** | 项目期间+2年 | 仅项目期间 |
| **知识产权** | 共有/约定 | 默认归客户，可约定保留 |

**可选调整的内容**：

| 条款类型 | 说明 |
|----------|------|
| **里程碑数量** | 根据项目规模调整 |
| **付款节点** | 根据客户付款节奏调整 |
| **争议解决** | 可选择快速仲裁 |

### 4.3 简化版SPV协议（短期项目）

对于D级/C级短期项目（周期<30天），可使用简化版：

```
简化版SPV协议要素：
├── 1. 项目基本信息
├── 2. 分包模块清单
├── 3. 分账比例
├── 4. 里程碑验收标准
├── 5. 保密条款
├── 6. 争议解决
└── 7. 签署页
```

## 五、三长老见证规则

### 5.1 见证适用范围

| 项目等级 | 见证要求 | 见证长老 |
|----------|----------|----------|
| **D级** | 无需见证 | - |
| **C级** | 可选见证 | 导师 |
| **B级** | 必须见证 | 盟主+导师 |
| **A级** | 必须见证 | 三长老全体 |
| **S级** | 必须见证 | 三长老+理事会代表 |

### 5.2 见证流程

```
1. 签约发起 → 通知见证长老
2. 协议审核 → 见证长老确认协议内容
3. 见证签署 → 见证长老签字见证
4. 见证存档 → 见证记录归档
```

### 5.3 见证记录要素

| 要素 | 内容 |
|------|------|
| 项目信息 | 项目名称、编号、金额 |
| 签约方 | 各方名称、签约时间 |
| 协议内容 | 协议版本号、主要条款 |
| 见证意见 | 见证长老意见 |
| 见证签字 | 见证长老签字 |
| 见证时间 | 见证完成时间 |

## 六、快速检查清单

签约前，确认以下事项：

```
□ SPV主协议已准备好
□ 附件清单已确认
□ 各方信息已收集完整
□ 三长老见证安排已确定
□ 电子签章工具已就绪
□ 协议存证方案已确定
□ 各方签约时间已协调
□ 签约提醒已发送
```
