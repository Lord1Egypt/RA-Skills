# Qualitative and Mixed Methods Thesis Guidance

This reference summarizes the qualitative and mixed-methods guidance extracted and paraphrased from the local file `qua.txt`. Use it to strengthen undergraduate thesis advice when the user mentions interviews, focus groups, observations, case studies, phenomenology, ethnography, grounded theory, narrative/discourse analysis, document analysis, or mixed methods.

## Core Orientation

Qualitative research is appropriate when the purpose is to understand a social or human problem in depth, especially participants' experiences, meanings, actions, records, language, interactions, and contexts.

Use qualitative methods when:

- The user asks "what is the experience like", "how do participants understand", "what meanings emerge", or "what process occurs".
- The topic needs rich description rather than statistical testing.
- The sample is small, purposive, and context-specific.
- The expected output is themes, categories, cases, narratives, process models, or theory-building propositions.

Do not present qualitative research as easier than quantitative research. It requires careful design, interviewing/observing skill, systematic coding, reflexivity, and cautious interpretation.

Good qualitative conclusions are contextualized. They should not overgeneralize to a population unless the design and evidence justify transferability.

## Complete Qualitative Proposal Output

When a user provides a research theme and research content and asks for a qualitative thesis plan, produce these sections by default:

1. 推荐题目
   - Give one best title and 3-5 alternatives.
   - Match the design: use "质性研究", "现象学研究", "个案研究", "焦点小组研究", "文本分析", or "混合研究" accurately.
2. 研究类型与适配理由
   - Explain why qualitative research is suitable for the user's purpose.
   - State the recommended design and why it is feasible for an undergraduate.
3. 研究目的
   - Explain what experience, meaning, process, case, or context the study aims to understand.
4. 研究内容
   - Break the topic into 3-5 concrete tasks, such as concept clarification, participant experience, influencing context, coping process, support needs, or practical suggestions.
5. 研究问题
   - Write 3-5 open questions using "如何", "哪些", "怎样理解", "呈现出什么主题".
6. 研究对象与抽样
   - Specify participant/material source, inclusion/exclusion criteria, sample size range, and sampling strategy.
7. 资料收集方案
   - Specify interview/focus group/observation/document source, procedure, recording/transcription, and a preliminary protocol or interview outline.
8. 资料分析方案
   - Specify thematic analysis/coding/constant comparison/cross-case analysis/narrative/discourse analysis as appropriate.
9. 研究可信度与伦理
   - Include triangulation, reflexivity, participant verification where feasible, thick description, anonymization, informed consent, withdrawal, and risk referral.
10. 毕业论文可续写章节规划
   - Use the qualitative chapter table from `chapter-plans.md`, adapted to the specific topic.
11. 可行性、创新点与局限
   - Keep claims realistic and undergraduate-appropriate.
12. 开题报告可直接使用的表述
   - Provide short thesis-ready wording for method and technical route.

Do not omit the chapter plan when the user asks whether the plan can support continued thesis writing.

## Choosing the Right Qualitative Design

| Design | Use When | Typical Data | Undergraduate-Friendly Topic Pattern |
|---|---|---|---|
| Phenomenological study | The user wants to describe participants' lived experience of a phenomenon | Semi-structured interviews, focus groups, reflective texts, participant observation | 《大学生考研压力体验的质性研究》 |
| Case study | The user wants to examine a bounded individual, group, class, program, organization, or event | Interviews, observation, documents, demographic/background narrative | 《某高校心理健康课程实施的个案研究》 |
| Multiple-case study | The user can compare several cases around the same phenomenon | Separate case data plus cross-case comparison | 《三名大学生学习倦怠形成过程的多案例研究》 |
| Ethnographic study | The user can enter a group setting and examine norms, culture, language, practices, and artifacts | Fieldwork, participant observation, interviews, documents, photos/videos where ethical | Usually too time-consuming for undergraduates unless scope is very small |
| Grounded theory | The user wants to build a theory or process model from data | Iterative interviews/observations/documents, theoretical sampling, coding and constant comparison | Often advanced; can be simplified as "探索性质性研究" unless theory-building is real |
| Narrative analysis | The user wants to study how participants make meaning through stories | Life stories, illness/recovery narratives, educational experience narratives | 《心理咨询初学者专业成长叙事研究》 |
| Discourse analysis | The user wants to examine language use in social context | Texts, conversations, media posts, classroom dialogue, policy documents | 《高校心理健康宣传文本的话语分析》 |
| Document analysis | The user has documents rather than participants | Records, policy documents, reflective journals, social media texts, newsletters, photos/videos | 《大学生心理求助经验网络文本的主题分析》 |
| Action research | The user is improving a local practice while cycling through planning, action, observation, and evaluation | Field notes, reflective journals, interviews, surveys, process records | Only recommend when the user can actually participate in or implement a local improvement process |

Default undergraduate recommendation:

- Prefer phenomenological interviews, small case studies, document/text analysis, or simple focus groups.
- Avoid full ethnography and full grounded theory unless the user has enough time, access, and advisor support.

## Research Questions

Qualitative research questions should be open-ended and exploratory:

- "……群体如何体验……？"
- "……在……过程中形成了哪些主要感受、困难和应对方式？"
- "……案例中……问题是如何发生和变化的？"
- "……文本中呈现出哪些主题、意义和话语特征？"
- "……实践中参与者如何理解……？"

Avoid:

- "……是否显著影响……？"
- "……与……是否相关？"
- "……能否预测……？"

Those are quantitative questions.

## Sampling and Participants

Most qualitative undergraduate studies use purposive sampling.

Possible sampling strategies:

- Purposeful sampling: recruit participants who directly experienced the phenomenon.
- Convenience sampling: easiest access, but weakest rationale; describe limitations.
- Typical case sampling: choose cases that represent common conditions.
- Extreme/deviant case sampling: choose unusual or intense cases to understand boundaries.
- Critical case sampling: choose a case that is especially informative for a claim or decision.
- Snowball sampling: recruit through referrals from initial participants.
- Stratified purposeful sampling: include participants from different grades, genders, majors, roles, or experience levels.

Case-study case types:

- Extreme/deviant case: unusual, intense, or boundary case.
- Critical case: especially informative case; if a phenomenon appears here, it likely has wider significance.
- Convenience case: selected because accessible; usable but methodologically weaker.
- Typical case: represents ordinary or common conditions.
- Politically/organizationally important case: selected because it is sensitive or practically consequential.

Suggested sample sizes:

- Individual interviews: 6-15 participants for undergraduate work, depending on saturation and depth.
- Focus groups: 1-3 groups, usually 4-8 participants per group; 4-12 is a broad acceptable range, but large groups are harder to moderate.
- Case study: 1 case for single-case, 2-4 cases for undergraduate multiple-case comparison.
- Document analysis: define a clear corpus, such as 30-100 posts/documents depending on length and depth.

Write inclusion criteria clearly:

> 研究对象需符合以下标准：第一，……；第二，具有……经历；第三，自愿参加访谈并同意录音；第四，能够较完整地表达个人经验。

Write exclusion criteria when needed:

> 若参与者存在严重心理危机、急性精神症状或访谈可能明显加重其痛苦，应不纳入研究，并提供专业求助或转介信息。

## Data Collection Methods

### Semi-Structured Interview

Use for most psychology undergraduate qualitative topics.

Before the interview:

- Choose a quiet and minimally distracting location or secure online meeting format.
- Explain the purpose, time length, confidentiality, voluntary participation, withdrawal rights, and data use.
- Obtain informed consent and recording permission.
- Prepare a face-validated interview protocol when possible.
- Test recording equipment and prepare note-taking.

During the interview:

- Start with simple factual/warm-up questions.
- Ask present-experience questions before past or future questions when appropriate.
- Use probes to deepen answers, such as "能否举一个具体例子？" and "当时你是怎么想的？"
- Attend to nonverbal cues and emotional changes.
- Avoid leading, judging, or giving advice during data collection.
- End by asking whether the participant wants to add anything.

Interview guide structure:

1. Basic background questions
2. Experience description
3. Feelings and meanings
4. Influencing factors
5. Coping or change process
6. Support needs and suggestions
7. Closing/additional comments

Method wording:

> 本研究采用半结构式访谈收集资料。访谈提纲围绕……设计，主要包括基本情况、……经历、情绪体验、影响因素、应对方式和支持需求等方面。经参与者同意后进行录音，并在访谈结束后转写为文字资料。

### Focus Group

Use when group interaction can generate useful discussion, such as shared student experiences, course feedback, school adaptation, or intervention needs.

Design notes:

- Usually 4-8 participants for novice moderators; 4-12 is possible but harder to manage.
- Prepare about 6-10 questions from general to specific.
- Establish a nonthreatening atmosphere and group confidentiality agreement.
- Expect that participants build on each other's comments; this is normal and can enrich data.

Method wording:

> 本研究采用焦点小组访谈了解……。每组约……人，由研究者依据访谈提纲主持讨论，问题由一般经验逐步过渡到具体困难和支持需求。讨论全程在知情同意基础上录音并转写。

### Observation

Use when behavior, interaction, setting, and context matter.

Observation types:

- Participant observation: researcher enters the setting and participates to some degree.
- Direct/nonparticipant observation: researcher observes without intervening.

Observe and record:

- Physical environment
- People and roles
- Activities and events
- Interaction patterns
- Hierarchies and informal norms
- Language and communication channels
- Nonverbal or unspoken communication
- Researcher's own thoughts and emotional responses

Method wording:

> 本研究采用非参与式观察，围绕场地环境、互动对象、活动过程、语言交流、非语言行为和研究者反思等方面记录观察笔记。研究者不干预现场活动，仅对自然发生的行为和互动进行记录。

### Document or Text Analysis

Use when data are records, reflective journals, policy documents, counseling notes with permission, public social-media posts, videos/photos, newsletters, or course materials.

Procedure:

1. Define purpose and research question.
2. Define the document corpus and inclusion/exclusion criteria.
3. Label and organize documents.
4. Read repeatedly and code meaningful segments.
5. Group codes into categories and themes.
6. Compare different documents or perspectives around central issues.

Method wording:

> 本研究采用文本分析法，对……资料进行整理和编码。研究者首先依据纳入标准建立文本资料库，再通过反复阅读、初始编码、类别归纳和主题提炼，分析文本中呈现的主要意义模式。

### Case Study Data Collection

Use multiple sources where feasible:

- Background narrative or demographic/context information
- History of the person, group, program, organization, or event
- Interview data
- Observation notes
- Relevant documents or artifacts
- Photos/videos only when ethically appropriate and permitted

Writing rule:

> 个案研究需要把读者带入个案情境，提供足够的背景、过程和细节描述。多案例研究应先分别分析每个案例，再进行跨案例比较。

### Discourse, Conversation, and Narrative Analysis

Use sparingly for undergraduate work.

- Discourse analysis studies language use in social context, including how language constructs identities, relationships, power, and meanings.
- Conversation analysis studies naturally occurring interaction, turn-taking, sequence organization, gestures, and conversational repair.
- Narrative analysis studies how people tell and retell experience, often focusing on setting, characters, conflicts, evaluation, and resolution.

For most undergraduate psychology theses, "主题分析" or "叙事分析" is easier to defend than advanced discourse/conversation analysis.

## Data Analysis

Default undergraduate approach: thematic analysis or coding analysis.

Basic process:

1. Transcribe interviews or organize documents.
2. Read the material repeatedly.
3. Mark meaning units.
4. Generate initial codes.
5. Merge similar codes into categories.
6. Develop themes and subthemes.
7. Compare themes across participants, documents, or cases.
8. Select short anonymized excerpts as evidence.
9. Link themes to theory and literature.

Constant comparison:

- Compare answers to common questions.
- Compare different participants' perspectives on the same issue.
- Compare documents/cases to identify similarities, differences, and anomalies.

Cross-case analysis:

- Analyze each case separately first.
- Then compare cases to identify shared themes, unique features, and contextual conditions.
- Use matrices, tables, and figures to show similarities and differences.

Grounded theory analysis:

- Identify concepts from incidents/events in the data.
- Group concepts into higher-level categories.
- Build propositions or relationships between categories.
- Keep data collection and analysis iterative.
- Avoid claiming grounded theory if the study does not truly generate a theory.

Result-writing pattern:

> 通过对访谈资料的编码分析，本研究形成了……个主题：第一，……；第二，……；第三，……。其中，主题一包括……和……两个子主题。

Avoid writing only long interview excerpts. Use excerpts as evidence after analytic claims.

## Trustworthiness and Rigor

Use several of these strategies:

- Triangulation: combine different data sources, investigators, theories, or methods.
- Member checking/participant verification: ask participants to verify summaries or themes where feasible.
- Thick description: give enough context for readers to judge transferability.
- Reflexivity: report how the researcher's identity, experience, assumptions, and relationship with participants may shape the study.
- Audit trail: keep records of recruitment, interview protocol, coding decisions, theme development, and revisions.
- Negative/deviant case analysis: report cases that do not fit the main pattern.
- Peer debriefing: discuss coding/themes with advisor or peer researcher.

Classical quality criteria that can be translated for thesis writing:

- Construct validity: define procedures and concepts clearly.
- Internal validity/credibility: show how interpretations are grounded in the data.
- External validity/transferability: describe context so readers can judge where findings may apply.
- Reliability/dependability: make data collection and analysis procedures traceable and repeatable.

Triangulation types:

- Data triangulation: time, place, participant, or document-source variation.
- Investigator triangulation: more than one coder/observer.
- Theory triangulation: more than one theoretical perspective.
- Methodological triangulation: interviews plus observations/documents/questionnaires.

Reflexivity section wording:

> 研究者在资料收集与分析过程中记录反思笔记，反思自身专业背景、价值判断和与研究对象的关系可能对资料理解产生的影响，并在编码和主题提炼过程中尽量保持对原始资料的贴近。

Researcher's perspective wording:

> 由于质性研究中研究者不可避免地参与意义建构，本研究在方法部分说明研究者立场，并通过访谈提纲审阅、资料反复阅读、参与者确认和同伴讨论等方式提高研究可信度。

Suggested subsection heading:

> 研究者立场与反思

Include:

- Researcher's professional background or relationship to the topic.
- Possible assumptions before data collection.
- How reflexive notes were kept.
- How coding decisions were checked.

## Ethics

Always include:

- Voluntary participation
- Informed consent
- Permission for recording
- Confidentiality and anonymization
- Right to withdraw
- Secure storage of audio/transcripts
- Avoidance of harm
- Referral information for psychological risk

For vulnerable topics, add:

> 若访谈过程中参与者出现明显不适，研究者将暂停或终止访谈，并提供学校心理咨询中心或专业机构的求助信息。

## Method Section Templates

### Phenomenological Interview Study

> 本研究采用现象学取向的质性研究方法，旨在深入理解……群体在……中的主观体验和意义建构。研究采用目的抽样招募……名具有相关经历的参与者，通过半结构式访谈收集资料。访谈资料经录音转写后，采用主题分析方法进行编码、归类和主题提炼，并通过研究者反思、参与者确认和同伴讨论提高研究可信度。

### Case Study

> 本研究采用个案研究法，以……为研究对象，围绕……问题收集访谈、观察和文件资料。研究首先对个案背景进行描述，其次对资料进行编码和主题归纳，最后从……角度分析个案中问题形成、发展和应对的过程。

### Focus Group

> 本研究采用焦点小组访谈法，围绕……主题组织……组讨论，每组……人。访谈提纲包括……等方面。研究者在知情同意基础上进行录音和转写，并采用主题分析方法归纳参与者共同关注的问题、差异性观点和支持需求。

### Document Analysis

> 本研究采用文本分析法，以……为资料来源，根据……标准筛选文本。研究者通过反复阅读、开放编码、类别归纳和主题提炼，分析文本中关于……的主要表达方式和意义结构。

### Mixed Methods

> 本研究采用混合研究方法。定量部分通过问卷调查了解……的总体状况及变量关系；质性部分通过半结构式访谈进一步解释问卷结果背后的主观体验和现实原因。最后将定量结果与质性主题进行整合，以形成对……问题的综合理解。

## Mixed Methods Design

Use mixed methods when the study genuinely benefits from combining quantitative and qualitative evidence.

Common designs:

- Sequential explanatory: questionnaire/statistics first, then interviews explain the results.
- Sequential exploratory: interviews first, then questionnaire explores how common the themes are.
- Convergent parallel: collect quantitative and qualitative data in the same phase, then compare and integrate.
- Embedded design: one type of data supports a dominant design, such as interviews embedded in an intervention feasibility study.

Eight-step planning logic:

1. Determine the research question.
2. Decide whether mixed methods are necessary.
3. Select the mixed-methods design.
4. Collect quantitative and qualitative data.
5. Analyze each data type.
6. Interpret each data type.
7. Legitimate/integrate the evidence.
8. Draw conclusions and write the final report.

Integration strategies:

- Connect: use survey results to select interviewees.
- Build: use interview themes to design questionnaire items.
- Merge: compare numeric results and themes in the discussion.
- Explain: use themes to interpret surprising statistical results.
- Joint display: table aligning quantitative findings, qualitative themes, and integrated interpretation.

Undergraduate warning:

> 混合研究的工作量明显高于单一方法。若时间有限，建议选择"简单问卷 + 少量补充访谈"的顺序解释型设计，或直接改为单一问卷/单一质性研究。

## Advising Rules

- If the user says they cannot collect data, do not recommend qualitative interviews or focus groups.
- If the user can interview but cannot run statistics, recommend qualitative interview, focus group, case study, or document analysis.
- If the user wants "影响因素" but only has interview data, phrase the goal as "影响因素的质性探索", not causal testing.
- If the user wants clinical or traumatic topics, narrow the topic and add ethics/referral safeguards.
- If the user is an undergraduate, prefer designs that can be completed with limited participants and clear analysis.
