# 暴发型流脑休克型抢救流程

## 触发条件
IF 流脑患儿在数小时内出现以下三联征：
- 血压下降
- 意识障碍
- 皮肤瘀点瘀斑迅速融合成片
THEN 立即启动本抢救流程。

> **注意**：本流程仅适用于暴发型流脑休克型患儿，不适用于非感染性休克或非流脑引起的弥散性血管内凝血（DIC）。

## 抢救步骤

### 1. 尽早应用有效抗菌药物
- 首选第三代头孢菌素：
  - 头孢噻肟：200 mg/(kg·d)，静脉滴注
  - 或头孢曲松钠：100 mg/(kg·d)，静脉滴注
- 疗程：5～7天

### 2. 迅速纠正休克
- **扩容纠酸**：
  - 首剂：生理盐水 10～20 ml/kg，10～20 分钟内快速静推
  - 若循环无改善，可重复 1～2 次，总量 ≤ 40～60 ml/kg
  - 目标：6 小时内稳定血流动力学、改善组织灌注、重建氧平衡，维持血糖正常
- **血管活性药物**（在充分扩容纠酸基础上使用）：
  - 山莨菪碱（654-2）：0.5～1 mg/kg/次，每 10～15 分钟静注 1 次，至四肢温暖、血压回升后逐渐减量（不可骤停）
  - 可选多巴胺或间羟胺

### 3. 应用糖皮质激素
- 地塞米松：0.6 mg/(kg·d)，分 2 次静脉给药，疗程 ≤ 3 天
- 作用：纠正休克、减轻毒血症、降低颅内压

### 4. DIC 的识别与治疗
- **指征**：瘀斑迅速融合 + 血小板进行性减少
- **肝素治疗**：
  - 剂量：0.5～1 mg/kg/次，加入 10% 葡萄糖 100 ml 静滴
  - 间隔：每 4～6 小时可重复，多数 1～2 次即见效后停用
  - **过量处理**：若出现大量新鲜出血或 APTT 明显延长，立即停用，并按最后一次肝素剂量给予等量硫酸鱼精蛋白中和
- **凝血因子补充**（高凝状态纠正后）：输注新鲜血浆、纤维蛋白原或凝血酶原复合物

### 5. 重要脏器功能保护
- **呼吸衰竭**：及时给予辅助通气
- **难治性心力衰竭**：评估并考虑 ECMO 支持
- **肝肾功能衰竭或毒素蓄积**：采用血液净化技术（如 CRRT）

## 预期目标
在 24 小时内稳定生命体征，防止多器官功能衰竭，为后续治疗争取时间。
