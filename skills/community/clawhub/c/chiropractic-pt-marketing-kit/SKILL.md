## The 4 Prompts

### Prompt 1 — New Patient Acquisition Campaign (FREE TIER)
**What it generates:**
- Grand opening / new-location email to community list (350–450 words)
- Google Business Profile optimization (business description 750 chars, 5 service descriptions, 3 Q&As with structured answer format)
- Facebook/Instagram ad set (2 new patient offer variants: free consultation + 3 paid variants — all with FTC disclaimer language for health claims)
- Google Search RSA (3 headlines 30-char max, 2 descriptions 90-char max, health claim guardrails)
- Nextdoor neighborhood intro post (500–700 chars, neighbor-tone, no clinical superiority claims)
- Personal injury / auto accident referral outreach letter to 3 attorney types (PI, workers' comp, auto defense)
- Google Review request sequence (3-touch: in-office verbal script + text + email, HIPAA-safe — no condition/treatment reference)

**Compliance enforced:**
- FTC health claims: no "cure," "eliminate," "guaranteed relief" — outcome language softened to "may help," "designed to support," "many patients report"
- HIPAA: all patient communication templates use first name only, no diagnosis, no treatment detail in subject lines
- State board: no "best," "most effective," "leading" superlatives without substantiation
- ACA/APTA: no claim of superiority over medical doctors; no anti-medication language in ad copy

### Prompt 2 — Patient Communication & Retention System
**What it generates:**
- New patient welcome sequence (5 emails): appointment confirmation → intake prep → first-visit follow-up → care plan check-in → milestone celebration
- Reactivation campaign for lapsed patients (3-touch: 6-month lapse email + phone script + 12-month final attempt)
- Front desk objection scripts (5 objections, word-for-word with redirect):
  1. "I don't have insurance coverage" → cash-pay option, payment plan framing
  2. "I need a referral from my doctor first" → direct access laws, when referral is/isn't required
  3. "I've tried chiropractic/PT before and it didn't work" → empathy + differentiation framework
  4. "I just need some exercises I can do at home" → scope clarification + home program partnership offer
  5. "My pain is mostly gone, I think I'm fine" → maintenance care value framing, discharge vs. wellness plan
- Appointment reminder system (48-hr email + 2-hr text, HIPAA-safe — no condition named)
- Birthday/wellness check-in (automated touchpoint, opt-in language compliant)
- End-of-care transition script: clinical discharge → wellness/maintenance → self-pay options

**HIPAA compliance layer:**
- Email subject lines: NEVER include diagnosis, body part, treatment type, or insurance status
- Phone scripts: verification protocol before discussing any clinical information
- Text messages: 160-char limit, appointment info only, STOP opt-out required

## Compliance Reference

| Rule | Source | What It Means for Marketing |
|------|--------|----------------------------|
| HIPAA Privacy Rule | 45 CFR §164 | No PHI in marketing without written authorization; email/text must not include diagnosis or treatment |
| FTC Health Claims | FTC Act §5 | Testimonials must disclose if results are not typical; no absolute cure claims |
| ACA Code of Ethics | Section B | No comparative superiority claims without substantiation; no disparagement of other providers |
| APTA Code of Ethics | Principle 5 | Truthful advertising; no claims that exceed scope of physical therapy practice |
| State Board Rules | Varies by state | NV: NAC 634 prohibits "best," "leading," "most effective" without substantiation |
| CAN-SPAM | 15 U.S.C. §7701 | Unsubscribe in all commercial email; physical address required |
| TCPA | 47 U.S.C. §227 | Prior written consent for marketing texts; STOP opt-out in every SMS |
