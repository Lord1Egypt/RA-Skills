# Automation

This directory contains the documentation and flow definitions for the Playwright-based Juejin Draft Publisher automation.

**Important Note:** 
The automation provided in this Skill is strictly limited to **draft creation**. It will NOT click the final "Publish" button. This ensures that a human always reviews the content before it goes live on the Juejin platform.

## Contents
- `playwright_draft_flow.md`: Details the exact steps the Playwright script takes to fill out the Juejin editor.
- `safety_rules.md`: The security and safety boundaries that the automation script MUST adhere to.
