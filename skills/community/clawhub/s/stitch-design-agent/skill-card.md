## Description: <br>
Skill for an agent that integrates designs generated by Google Stitch directly into an app under development by authenticating, requesting a design, retrieving generated code, and integrating it into the active codebase. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[duvanCode](https://clawhub.ai/user/duvanCode) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers use this skill to turn a natural-language UI request into Google Stitch-generated component code and integrate it into an existing React, NestJS, or similar application. It also guides token setup, project placement, dependency checks, and build verification. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can use broad Google credentials. <br>
Mitigation: Use least-privilege Google credentials, avoid broad or long-lived tokens, and store tokens only in approved secret mechanisms. <br>
Risk: The skill can modify the active codebase and install packages. <br>
Mitigation: Run it on a branch, review planned file changes and diffs before writing, and approve each npm install or npm script command before execution. <br>
