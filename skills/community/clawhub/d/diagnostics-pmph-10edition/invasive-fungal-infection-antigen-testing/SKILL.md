# 侵袭性真菌病抗原检测方法选择

## 触发条件
当临床怀疑患者存在侵袭性真菌感染，并已获取血清或体液标本时，根据疑似病原体种类选择对应的抗原检测方法。

## 执行步骤

1. **判断疑似病原体类型**：明确是否为念珠菌、曲霉、隐球菌或接合菌（如毛霉、根霉等）。

2. **根据病原体选择检测方法**：
   - **若怀疑曲霉感染**：优先选择 **GM试验**（半乳甘露聚糖检测），因其专用于侵袭性曲霉病的早期诊断。
   - **若怀疑念珠菌或其他非隐球菌/非接合菌真菌感染**：选用 **G试验**（(1,3)-β-D-葡聚糖检测），适用于除隐球菌和接合菌外的多数真菌。
   - **若怀疑隐球菌感染**：必须进行 **隐球菌抗原检测**，通常检测脑脊液或血清中的荚膜多糖抗原。
   - **若怀疑接合菌感染**：**不可依赖G试验或GM试验**，应结合影像学、组织病理学或宏基因组测序（mNGS）等其他方法。

3. **说明检测局限性**：
   - G试验不能确定具体菌种；
   - GM试验仅对曲霉有效；
   - 隐球菌和接合菌不适用G试验。

4. **输出推荐方案**：明确推荐的检测方法，并简要解释其适用性与局限性。
