# Acupuncture Point Selection Principles

在明确病变部位与病机后，依据以下四类原则选择腧穴，可单独或联合应用，并结合“三因制宜”（因时、因地、因人）进行调整。

## 1. 近部选穴
- **适用条件**：病变位于体表或局部组织。
- **操作方法**：选取病变局部或邻近部位的腧穴。
- **理论依据**：腧穴能治疗局部和邻近部位病变。

## 2. 远部选穴
- **适用条件**：需通过经络调节远端病位。
- **操作方法**：选取远离病变但在相关经络上的腧穴。
- **理论依据**：“经络所过，主治所及”；参考《灵枢·终始》“病在上者下取之，病在下者高取之”。

## 3. 辨证选穴
- **适用条件**：已明确中医证型（如肝阳上亢、肾阴不足等）。
- **操作方法**：根据病因病机选择对应功效的腧穴。

## 4. 对症选穴（经验取穴）
- **适用条件**：存在特定症状（如哮喘、疳积、腰痛等）。
- **操作方法**：选用具有针对性疗效的经验穴或奇穴。

## Execution Steps
- 确保穴位组合覆盖病位、病性与主症。
- 遵循“理、法、方、穴、术”逻辑链条。
- 严格规避禁忌（如孕妇禁针三阴交、合谷；小儿囟门未闭者禁针囟门附近穴）。
