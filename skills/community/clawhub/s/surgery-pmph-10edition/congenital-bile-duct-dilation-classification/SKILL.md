# 先天性胆管扩张症分型与临床评估

## 触发条件
当患者满足以下任一情况时启用本技能：
- 出现腹痛、黄疸、上腹部囊性肿块中的1–3项；
- 影像学检查（如超声、CT、MRCP）显示胆管异常扩张；
- 临床怀疑为先天性胆道疾病（尤其儿童或成人不明原因胆道扩张）。

## 执行步骤

### 1. 确定扩张部位与形态
- 明确扩张是否累及肝内胆管、肝外胆管，或两者兼有。
- 描述扩张形态：囊性、憩室样、梭形、葫芦状等。

### 2. 应用五型分类标准
根据扩张部位、范围和形态，按以下标准分类：
- **Ⅰ型**：肝外胆管部分或全部囊性扩张，呈梭形或葫芦状，远端狭窄；肝内胆管正常。
- **Ⅱ型**：胆总管侧壁局限性憩室样膨出。
- **Ⅲ型**：胆总管十二指肠壁内段囊性突出，部分突入十二指肠腔。
- **Ⅳ型**：肝内与肝外胆管均呈多发性囊性扩张。
- **V型（Caroli病）**：仅肝内胆管多发性囊性扩张，伴肝纤维化，肝外胆管无扩张。

### 3. 评估临床特征
- 识别典型三联症：右上腹持续性钝痛、间歇性黄疸、可触及的上腹部光滑囊性肿块。
- 注意合并感染表现：黄疸加深、腹痛加剧、肿块触痛、发热畏寒。
- 警惕晚期征象：胆汁性肝硬化、门静脉高压。
- 检查血/尿淀粉酶是否升高。

### 4. 判断并发症风险
- 评估是否存在胆石（成人患者发生率约50%）。
- 警惕囊壁溃疡、癌变风险。

### 5. 推荐诊断方法
- 首选无创检查：超声、CT 或 MRCP。
- 疑难病例可考虑 PTC 或 ERCP。

## 输出要求
输出具体分型（Ⅰ–V型）、对应的解剖特征、关键临床表现及推荐的进一步诊疗建议，用于指导外科手术方案制定。
