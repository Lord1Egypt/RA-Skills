# Skill 模板

## 最小可用模板

```markdown
---
name: skill-name
description: 做某件具体事情。当用户说某些触发词时使用。
---

# Skill Name

你是……

## 什么时候使用

当用户……

## 工作流程

1. 先理解用户给的内容。
2. 如果信息不够，只问一个最关键的问题。
3. 按用户目标处理内容。
4. 输出清晰结果。

## 输出格式

……

## 安全规则

涉及删除、付款、授权、发送消息、发布内容、读取隐私文件时，必须先让用户确认。

## 测试提问

- 正常触发：……
- 复杂触发：……
- 不该触发：……
```

## 内容创作类模板

适用于作文、文章、脚本、小红书、公众号、短视频文案。

重点写清楚：
- 目标读者是谁
- 输出风格
- 是否保留原意
- 是否允许改写
- 是否需要标题、摘要、标签

## 文件处理类模板

适用于 PDF、Excel、图片、Markdown、Word。

重点写清楚：
- 用户会提供什么文件
- 要提取什么信息
- 输出成什么格式
- 遇到无法读取的文件怎么办
- 是否允许批量处理

## 外部服务类模板

适用于飞书、微信、GitHub、邮箱、数据库等。

重点写清楚：
- 是否需要登录或授权
- 是否会发送、发布、删除、修改外部内容
- 哪些动作必须用户确认
- 凭证不能写进 Skill 包
