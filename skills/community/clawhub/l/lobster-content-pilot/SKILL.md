## 升级记录 v2.0

### 融合卡兹克khazix-writer

| 升级项 | 原ContentPilot | v2.0版 |
|--------|--------------|--------|
| 内容类型 | 社媒短文案 | + 公众号长文 |
| 写作风格 | 通用模板 | + 卡兹克"活人感"风格 |
| 质检体系 | 无 | + 四层质检（硬性→逻辑→审美→活人感） |
| 适配平台 | 小红书/抖音/朋友圈 | + 公众号深度长文 |

### 新增能力

**5. 公众号长文写作（融合khazix-writer）**
- 长文规划（选题→大纲→写作→质检）
- 卡兹克风格模板（"有见识的普通人在认真聊一件打动他的事"）
- 核心价值观贯穿：好奇、人话、真诚、底线
- HKR选题质检（Happy有趣/K知识/R共鸣）
- 四层质检体系（L1硬性→L2逻辑→L3审美→L4活人感）
- 高频口语化表达库（"坦率的讲"、"说真的"、"怎么说呢"）

### 使用方式

```
# 短内容（原有）
content-pilot> 写小红书文案 "蓝牙耳机"

# 长文章（新增）
content-pilot> 写公众号文章 "2026年AI Agent的10个趋势" 卡兹克风格

# 素材转文章
content-pilot> 根据这个PDF写篇文章 [文件]
```

## 融合来源

| 源技能 | 融合点 |
|--------|--------|
| caption | 社交媒体文案、广告语、标题 |
| hashtag | 话题标签推荐、热度分析 |
| quote | 金句生成、名人名言引用 |
| newsletter | 电子报创建、内容编排、订阅管理 |
| press-release | 新闻稿撰写、媒体发布 |

## 能力

### 1. 📝 社媒文案生成
- 小红书种草文案（含emoji+排版）
- 抖音口播稿（含语气标注）
- 朋友圈文案（短小精悍）
- 公众号推文

### 2. 🏷️ 标签策略
- 热门标签推荐（基于热点）
- 长尾标签挖掘（低竞争高流量）
- 标签组合策略（大+中+小标签搭配）

### 3. 💬 金句引擎
- 根据主题自动生成金句
- 名人名言匹配推荐
- Slogan/口号设计

### 4. 📰 内容矩阵
- 电子报（周报/月报自动编排）
- 新闻稿（产品发布/活动报道）
- 内容日历（一周发布规划）

## 使用方式

```
# 单条内容
content-pilot> 写小红书文案 "蓝牙耳机" 带货

# 批量内容日历
content-pilot> 规划本周内容 3篇小红书+2条抖音

# 企业场景
content-pilot> 写新闻稿 "新品发布会"
content-pilot> 本周电子报
```

## 输出格式

```
📢 ContentPilot 输出
├── 📝 文案×3（不同平台适配）
├── 🏷️ 标签推荐×10
├── 💬 金句×5（含引用出处）
├── 📰 新闻稿/电子报
└── 📅 发布日历
```
