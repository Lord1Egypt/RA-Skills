# 转场类型详解

## 1. 叠化 Dissolve（↔）

**定义**：镜头A渐隐的同时镜头B渐显，两者短暂重叠。

**符号**：`↔` 或 `叠化`

**何时使用**：
- 并行叙事（两条时间线交替）
- 时间流逝（季节更替、岁月变迁）
- 情绪类比或对比
- 梦中梦、回忆过渡

**Prompt 写作模式**：

```
镜头A意象 ↔ 镜头B意象，过渡描述
```

**示例**：
```
古钟特写 ↔ 水面倒影，金字浮现
燃烧的香烟 ↔ 飘散的火星，时间流逝
孩子奔跑 ↔ 老人蹒跚，岁月叠化
```

**注意事项**：
- 上下镜头的意象最好有视觉或语义关联
- AI 生成时叠化时长不可控，避免依赖精确时长

---

## 2. 硬切 Hard Cut（下一秒）

**定义**：无过渡地骤然切换画面，干净利落。

**符号**：`下一秒` 或 `画面骤转`

**何时使用**：
- 突发转折（爆炸、车祸、耳光）
- 强调重点（眼神、武器、关键道具）
- 节奏加速（追逐、搏斗、审讯）
- 时间/地点强制跳转

**Prompt 写作模式**：

```
镜头A，[突发事件]，下一秒，画面切至镜头B描述
```

**示例**：
```
拳头砸在桌上，下一秒，画面切至审讯室冷白灯光
玻璃碎裂，下一秒，画面骤转至医院走廊
```

**注意事项**：
- 硬切节奏快，不适合慢抒情场景
- 突发感来自"下一秒"这个时间词，而非视觉描述

---

## 3. 匹配剪辑 Match Cut（→）

**定义**：利用两个镜头在构图、形状、方向或动作上的相似性实现平滑过渡。

**符号**：`→（匹配剪辑）→`

**何时使用**：
- 建立视觉对应关系
- 从一个对象平滑过渡到另一个相关对象
- 转场不打断观者注意力时
- 强调"同类"或"转化"关系

**Prompt 写作模式**：

```
镜头A核心意象 →（匹配剪辑）→ 镜头B核心意象
```

**示例**：
```
握紧的拳头 →（匹配剪辑）→ 握紧的枪柄
旋转的餐叉 →（匹配剪辑）→ 旋转的车钥匙
低垂的泪滴 →（匹配剪辑）→ 坠落的雨滴
```

**注意事项**：
- 匹配点需要明显且具体（形状/方向/动作相似）
- 景别尽量接近（全配全、中配中），差异过大会削弱匹配感

---

## 4. 淡入淡出 Fade（淡出 → 淡入）

**定义**：画面逐渐变黑（淡出）或从黑渐亮（淡入）。

**符号**：`淡出` / `淡入`

**何时使用**：
- 章节/幕结尾（情绪收束）
- 时间跨度大的跳跃（数年后、数月后）
- 场景情绪终结，为下一场景蓄势
- 梦境/现实切换的虚化处理

**Prompt 写作模式**：

```
[镜头A]，[情绪描述]，淡出
淡入 → [镜头B]，[场景描述]
```

**示例**：
```
背影消失在雨幕中，淡出
淡入 → 阳光普照的庭院，若干年后
女人望向窗外，淡出
淡入 → 同一扇窗，画面已是深夜
```

**注意事项**：
- 淡出淡入是独立操作，可以只淡出或只淡入
- AI 视频对"淡"的时长控制较弱，适合作为情绪标记而非精确时间控制

---

## 5. 划像 Wipe（屏幕方向）

**定义**：画面从屏幕某一方向"划入"新场景，或将旧场景"划出"。

**符号**：`←划入` `划出→` `↓划入` `↑划出` `↙划入`

**何时使用**：
- 空间位移（从门外划入室内）
- 视轴切换（从角色视角划向所见之物）
- 方向感强的场景转换（走廊、赛道、河流）
- 地图式/游记式叙事

**Prompt 写作模式**：

```
[镜头A]，[动作/方向描述]，↓划入 [镜头B描述]
```

**常用方向词**：
- `←划入` / `划出→`（左右横扫）
- `↓划入` / `↑划入`（上下穿越）
- `↙划入` `↗划入`（对角线划入）
- `↺划入`（旋转划入）

**示例**：
```
站在门口，身后门板逐渐放大，↓划入门内走廊
赛车冲出弯道 → ↗划入终点线俯拍
```

**注意事项**：
- 划像依赖屏幕边缘，9:16 竖屏需注意上下边缘空间
- 方向需与镜头运动逻辑一致（横向移镜头对应横向划像）
