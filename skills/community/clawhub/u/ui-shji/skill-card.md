## Description: <br>
Guides an agent through a five-stage UI design workflow from inspiration gathering to design-system and prototype deliverables. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[mark197011](https://clawhub.ai/user/mark197011) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Designers, developers, and product builders use this skill to turn a UI idea or redesign request into references, design direction, a style guide, an interactive HTML prototype, and handoff documentation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The workflow can optionally expand inspiration search to the web when built-in references are insufficient. <br>
Mitigation: Ask for user consent before broader web search and cite or summarize sources used for design inspiration. <br>
Risk: The workflow creates HTML and Markdown deliverables in a fixed output directory, which could overwrite existing design files. <br>
Mitigation: Review existing files in /workspace/projects/workspace/output/ui-design/ before writing or replacing generated deliverables. <br>
