## Known Risks and Mitigations: <br>
Risk: Generated mini-game code may use payment, login, location, clipboard, uploads, analytics, ads, or cloud-storage APIs that affect real users. <br>
Mitigation: Review generated code before implementation, confirm user consent and platform requirements, and test these APIs in the intended Douyin mini-game environment. <br>
Risk: The skill is a documentation-only reference and may not reflect project-specific compliance, monetization, or data-handling requirements. <br>
Mitigation: Validate sensitive flows such as payments, ads, identity, and user data handling against current Douyin platform documentation and the release team's policies. <br>
