# 义乌外贸风险防控指南

## 一、风险全景图

```
客户风险 → 违约赖账、欺诈陷阱、资信不良
货物风险 → 质量问题、数量短缺、侵权产品
物流风险 → 延误丢失、损坏破损、单证错误
合规风险 → 海关查验、政策变动、侵权纠纷
市场风险 → 竞争加剧、价格波动、汇率风险
```

---

## 二、客户风险防控

### 2.1 资信背调

**必查项目：**
- 公司注册信息（天眼查/邓白氏）
- 信用报告
- 历史交易记录
- 银行资信证明

### 2.2 高危信号

```
🚨 一级警报（坚决不合作）：
- 拒绝提供公司信息
- 要求发货到个人地址
- 首单金额异常大

⚠️ 二级警报（谨慎合作）：
- 新注册公司（<6个月）
- 要求超长账期
- 价格异常敏感
```

### 2.3 付款方式风险

| 方式 | 风险 | 适用 |
|------|------|------|
| 100%前T/T | 最低 | 首次合作、小单 |
| T/T定金+尾款见提单 | 低 | 常规合作 |
| D/P | 高 | 谨慎使用 |
| O/A赊销 | 最高 | 仅限长期信任客户 |

## 三、货物风险防控

### 3.1 质量控制

```
入库质检：抽检5-10%，不良率<3%
出货质检：抽检10-20%，首单加倍
第三方质检：大额订单建议使用SGS/TUV
```

### 3.2 侵权排查

**查询工具：**
- 商标：中国商标网、WIPO、USPTO
- 专利：Google Patents、中国专利网

**合规要点：**
- ✅ 自有品牌注册
- ✅ 获得品牌授权
- ✅ 原创设计
- ❌ 仿牌、高仿产品

---

## 四、物流风险防控

### 4.1 货运险

- 投保比例：货值110%
- 险种：一切险ICC(A)
- 保费率：0.3-0.8%
- 理赔：出险后72小时内报案

### 4.2 包装标准

- 外箱：五层瓦楞纸箱
- 内衬：气泡膜/珍珠棉
- 标识：易碎、防潮、向上

---

## 五、合规风险防控

### 5.1 1039合规红线

```
❌ 低报货值（<实际成交价70%）
❌ 虚假贸易（无真实货物）
❌ 拆分结汇（规避监管）
❌ 地下钱庄换汇
❌ 非义乌市场货物

✅ 采购地在义乌市场
✅ 货值真实申报
✅ 收汇合规渠道
✅ 单证留存3年
```

## 六、风险案例警示

### 案例1：客户赖账

**情况**：中东客户，$50,000，O/A 60天，货到后失联
**教训**：新客户不接受O/A，购买出口信用保险

### 案例2：侵权被扣

**情况**：印卡通图案T恤，目的国海关扣押销毁
**教训**：选品必须排查侵权风险

### 案例3：货物损坏

**情况**：玻璃制品，破损率30%，客户拒收
**教训**：易碎品加固包装，购买货运险
