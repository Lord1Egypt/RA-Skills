Mitigation: Install only in a controlled workspace and add explicit approval requirements before code or deployment changes, public communications, account or configuration changes, credential use, data access or export, deletion or mutation of business data, and sharing sensitive context with other agents. <br>

Mitigation: Define workspace-specific approval gates, delegation limits, and sensitive-data handling rules before allowing the skill to coordinate business workflows. <br>
