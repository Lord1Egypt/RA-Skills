# 参考区间与特殊情况处理

## ABO血型对vWF水平的影响
- **O型血**：vWF:A 正常参考范围为 38%～125.2%
- **非O型血**：vWF:A 正常参考范围为 49.2%～169.7%

## 进一步鉴别试验指引

### vWD 2A/2B/2M亚型区分
- **瑞斯托霉素诱导血小板聚集试验（RIPA）**：
  - 2B型：低浓度瑞斯托霉素即可诱导聚集
  - 2A/2M型：聚集反应减弱或缺失
- **vWF多聚体分析**：
  - 2A型：高分子量多聚体缺失
  - 2B型：高分子量多聚体选择性缺失
  - 2M型：多聚体结构正常但功能异常

### vWD 2N vs 血友病A鉴别
- 加测 **FVIII抗原（FVIII:Ag）**：
  - **血友病A**：FVIII:Ag 与 FVIII:C 均显著降低
  - **vWD 2N**：FVIII:Ag 正常或轻度降低，FVIII:C 明显降低

## 比值阈值说明
- **0.7** 是国际通用的关键分界点，用于区分数量型（1型）与功能型（2型）vWD
- 比值计算应使用同一实验室、同一批次校准的检测结果，避免系统误差

## 注意事项
- 急性期反应（如感染、炎症、妊娠）可导致vWF水平暂时升高，可能掩盖vWD表现
- 多次检测结果不一致时，建议在稳定状态下重复检测
