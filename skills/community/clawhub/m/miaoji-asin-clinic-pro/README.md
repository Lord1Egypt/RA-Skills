# 秒技·ASIN一键诊所Pro版

## 快速开始

### 基本使用

1. 打开技能
2. 输入ASIN和产品品类
3. 选择模式：竞品对标模式/90天行动计划/季节性优化
4. 获取完整的视觉策略报告

### 输入示例

```
ASIN：B08XXXXX
品类：露营帐篷
阶段：成长期
竞品ASIN列表：B09YYYYY, B07ZZZZZ
```

## 功能概览

### 竞品对标模式
输入3个同类产品ASIN，获取：
- 五维评分差距矩阵
- 赶超路线图
- 优先修复方向

### 90天行动计划
获取：
- 按周拆解的修复任务
- 每周明确交付物
- 月度复诊模板

### 季节性优化日历
获取：
- Prime Day D-60/D-30/D-7冲刺清单
- Q4旺季作战时间表
- 节日节点提醒

## 输出示例

技能会输出完整的诊断报告、90天路线图、本周任务卡等交付物。
