# 应用改良Duke标准诊断儿童感染性心内膜炎

## 何时使用
- 患者为儿童或新生儿，临床怀疑感染性心内膜炎（IE）；
- 已完成血培养和超声心动图检查；
- 存在以下任一表现：发热、新发或变化的心脏杂音、栓塞事件、免疫复合物相关征象；
- 且患者有基础心脏病、近期心脏手术、中心静脉置管等易感因素。

> **注意**：若已有其他明确诊断可完全解释所有临床表现，或抗生素治疗≤4天后症状消失且无病理证据，则不应使用本标准诊断IE。

## 执行步骤

1. **确认是否满足确诊IE的任一组合条件**（满足其一即可确诊）：
   - 2项主要指标；
   - 1项主要指标 + 3项次要指标；
   - 超声心动图显示心内膜受累证据 + 2项次要指标；
   - 5项次要指标；
   - 病理学证据（赘生物或心内组织检出病原体或证实活动性心内膜炎）。

2. **识别主要指标**：
   - **血培养阳性**：两次分离出相同IE常见病原体（如甲型溶血性链球菌、金黄色葡萄球菌、凝固酶阴性葡萄球菌、肠球菌等）；
   - **心内膜受累证据（超声心动图）**：
     - 赘生物附着于瓣膜、瓣膜装置、心内膜或人工材料；
     - 腱索断裂、瓣膜穿孔、人工瓣膜或补片新裂开；
     - 心腔内脓肿。

3. **识别次要指标**（最多6项）：
   - 易感条件（基础心脏病、心脏手术史、心导管操作、中心静脉置管等）；
   - 发热≥38℃伴贫血；
   - 原有杂音加重、新出现杂音或心功能不全；
   - 血管征象（动脉栓塞、感染性动脉瘤、瘀斑、脾大、颅内出血、Janeway斑等）；
   - 免疫征象（肾小球肾炎、Osler结节、Roth斑、类风湿因子阳性）；
   - 微生物学证据（血培养阳性但不符合主要标准）。

4. **判断排除IE的情况**（满足任一即排除）：
   - 有其他明确诊断可完全解释所有临床表现；
   - 抗生素治疗≤4天后临床表现完全消失；
   - 抗生素治疗≤4天后手术或尸检未发现IE病理证据。

5. **处理高度怀疑但未达确诊标准者**：
   - 启动经验性抗感染治疗；
   - 继续密切观察并重复检查（如重复血培养、复查超声心动图）。

## 输出结果
根据上述评估，输出以下三类结论之一：
- `确诊IE`
- `疑似IE需治疗`
- `可排除IE`
