# Skill: knowledge-bridge

知识桥 — 洞察引擎到知识库的自动管道

## Functionality
- Auto-ingest insights from insight engine into SQLite + LlamaIndex KB
- Auto-categorize by domain (AI/技术, 商业/变现, 教育/学习, etc.)
- Auto-summarize from title/URL
- Dedup via MD5 hash
- Search and explain knowledge entries
- Backfill all historical cycles

## Invocation
- `run("ingest")` — ingest latest insight cycle
- `run("search <query>")` — search knowledge base
- `run("stats")` — show KB statistics
- `run("backfill")` — backfill all historical cycles
