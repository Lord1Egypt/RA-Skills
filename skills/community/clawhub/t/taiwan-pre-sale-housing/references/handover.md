# 交屋與管委會設立

## 交屋流程

```
貸款核撥 → 繳清尾款 → 產權移轉登記 → 實體交屋 → 交屋保證金返還
```

### Step 1：銀行貸款核撥
- 銀行撥款至建商帳戶
- 撥款後即開始計算利息

### Step 2：繳清尾款
- 扣除貸款之剩餘款項
- 交屋保留款（總價5%）在驗收完成後付清

### Step 3：產權移轉登記
| 項目 | 時程 | 備註 |
|------|:----:|------|
| 申請移轉登記 | 繳清尾款後 | 代書辦理 |
| 土地增值稅申報 | 同時 | 代書辦理 |
| 產權移轉完成 | 約10~14天 | — |
| 他項權利設定 | 同時 | 貸款銀行設定 |

### Step 4：實體交屋
- 現場核對鑰匙數量
  - 大門鑰匙
  - 信箱鑰匙
  - 社區門禁卡/感應扣
  - 車道遙控器
- 核對各項文件
  - 使用執照影本
  - 水電瓦斯管路證明
  - 保固書
- 現場點交設備
- 水電瓦斯度數記錄

### Step 5：交屋保證金返還
- 管委會成立後
- 確認無應負擔費用
- 返還交屋保證金（如有）

## 交屋前應取得的文件

| 文件 | 說明 |
|------|------|
| 使用執照影本 | 建築物合法證明 |
| 土地/建物所有權狀 | 產權登記完成證明 |
| 房屋稅籍證明 | 課稅資料 |
| 水電瓦斯過戶單 | 過戶至買方 |
| 社區規約草案 | 管委會成立前先行參閱 |
| 保固書 | 結構15年+設備1年 |
| 設備操作手冊 | 各項設備操作說明 |
| 鑰匙清冊 | 數量盤點 |
| 管理費繳費單 | 起算日期 |

## 管委會設立

### 第一次區分所有權人會議
- **時機**：交屋後**6個月內**召開
- 由建商召集（或賣方）
- 選出管理委員
- 通過社區規約
- 遴選物業管理公司

### 管委會職責
- 公共區域維護管理
- 管理費收支管理
- 社區規約執行
- 保全/清潔管理
- 設備保養維護

### 管理費
- 一般住宅管理費：**50~120元/坪/月**
- 車位管理費：**300~1,000元/個/月**
- 管理費起算點（依契約約定）
  - 一般：**交屋日起算**
  - 有些：使用執照取得日起算

---

## 交屋後1年內注意事項

### 保固期內
- 任何非人為損壞，應書面通知建商修繕
- 保留缺失照片+書面通知記錄
- 建商應於合理期間內完成（一般7~30天）

### 管理委員會正式運作
- 向區公所報備成立
- 開立管委會銀行帳戶
- 簽訂物業管理合約
- 公共設施點交
