# 特殊穴位禁忌与安全操作技能

## When to Use
- 患者为孕妇，需避免堕胎风险穴位
- 操作部位涉及眼部、耳部、面部或关节区域
- 针刺井穴（如少泽、商阳、隐白）需控制深度或采用点刺
- 穴位邻近重要神经（如尺神经、正中神经）
- 患者存在体位配合障碍（如无法张口）

## Execution Steps

### 1. 孕妇相关禁忌
- **绝对禁针**：合谷（LI4）与三阴交（SP6），因有堕胎风险
- **谨慎操作**：至阴（BL67）可灸用于胎位不正，但针刺仍需评估风险

### 2. 井穴操作规范
- **定位**：指甲根角侧上方0.1寸
- **操作**：仅浅刺0.1～0.2寸，或使用三棱针点刺出血以泄热开窍

### 3. 面部及特殊部位操作
- **睛明（BL1）**：患者闭目，沿眶内侧壁缓慢进针0.3～1.0寸；**禁止提插捻转，禁灸**
- **听宫（SI19）**：必须在患者**张口状态下取穴**，直刺0.5～1.0寸；深刺可能损伤颞颌关节或面神经
- **攒竹（BL2）、头维（ST8）**：采用平刺0.5～0.8寸；**禁灸**

### 4. 神经规避要点
- **小海（SI8）**：直刺0.3～0.5寸，避免损伤尺神经
- **内关（PC6）**：限制大幅度提插捻转，以防伤及正中神经

### 5. 关节部位操作
- **下关（ST7）**：治疗下颌脱位时，需配合手法复位
- **犊鼻（ST35）**：患者屈膝并垫托，向后内斜刺

## Execution Steps
- **患者状态**：是否为孕妇？能否配合张口？是否存在昏迷或局部肿胀？
- **穴位类型**：是否为井穴、面部穴、耳周穴或神经邻近穴？
- **操作方式**：应选择浅刺、点刺、平刺、斜刺还是透刺？是否禁灸？

> ⚠️ 若任一禁忌条件触发，立即调整方案或终止操作，并提示医疗风险。
