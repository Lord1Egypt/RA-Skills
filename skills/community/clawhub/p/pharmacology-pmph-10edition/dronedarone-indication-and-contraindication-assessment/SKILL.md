# 决奈达隆适应症与禁忌风险评估

## 何时使用
- 患者被诊断为心房颤动（AF）或心房扑动（AFL）；
- 需要维持窦性心律；
- 尚未确定是否存在严重心力衰竭或左心室收缩功能不全。

## 执行步骤
1. **确认心律失常类型**：核实患者是否确诊为心房颤动或心房扑动。
2. **评估心力衰竭状态**：判断患者是否属于严重心力衰竭（如NYHA III-IV级）。
3. **评估左心室收缩功能**：检查左心室射血分数（LVEF），若LVEF <35%则视为功能不全。
4. **判断禁忌情况**：
   - 若存在严重心力衰竭 **或** 左心室收缩功能不全，则**禁止使用**决奈达隆；
   - 若两者均不存在，则**可考虑使用**决奈达隆以维持窦性心律。
5. **注意事项**：
   - 决奈达隆结构类似胺碘酮但不含碘，甲状腺毒性较低；
   - 仍需定期监测心脏功能，因其在禁忌人群中可能显著增加死亡风险。

## 输出结果
明确给出是否启用决奈达隆治疗的建议：符合条件则推荐使用，否则禁用并建议选择其他抗心律失常药物。
