Mitigation: Use only the included cleaning and analysis scripts unless additional tools are verified in a later release. <br>
Risk: The included scripts read input files and the cleaning script can write to a user-specified output path. <br>
Mitigation: Run the scripts only on files intended for processing and review output paths before execution. <br>
