# 白三烯调节药物治疗选择

## 何时使用本技能
- 患者确诊为支气管哮喘、风湿性关节炎、肾小球肾炎、痛风、溃疡性膀胱炎或缺血性心血管疾病
- 疾病机制明确涉及白三烯（LTs）通路
- 需要长期控制气道或系统性炎症
- **排除**：急性哮喘发作抢救期、非白三烯主导的炎症性疾病

## 执行步骤
1. **确认白三烯类型参与**：
   - 若以支气管收缩、黏液分泌、肺水肿为主 → 考虑半胱氨酸白三烯（CysLTs：LTC4/LTD4/LTE4）
   - 若以白细胞趋化、组织浸润为主 → 考虑LTB4

2. **匹配疾病与药物类别**：
   - **支气管哮喘长期控制**：优先选用CysLT1受体阻断药（如孟鲁司特、扎鲁司特）
   - **多通路白三烯合成抑制需求**：考虑5-脂氧合酶（5-LOX）抑制剂（如齐留通）

3. **排除禁忌场景**：
   - 不用于急性哮喘发作的急救
   - 对非白三烯通路主导的疾病（如纯IgE介导的过敏）效果有限，应避免使用

4. **输出治疗建议**：
   - 明确药物类别（CysLT1拮抗剂 或 5-LOX抑制剂）
   - 推荐具体药物（如“选用孟鲁司特进行哮喘长期控制”）
