# Seller Audit Quality Checklist

---

## Pre-Audit Setup

- [ ] Logged into all platforms being audited
- [ ] Account Health / Seller Central dashboard loaded
- [ ] Previous audit report available for comparison
- [ ] Full listing export pulled from each platform
- [ ] Last 90 days customer feedback / return data pulled

## Account Health Metrics

- [ ] All health metrics recorded with exact values (not just "green/red")
- [ ] Any metric within 20% of threshold flagged for immediate action
- [ ] Metric trends noted (improving, stable, declining)
- [ ] Platform policy newsletter reviewed this month

## Listing Compliance

- [ ] All suppressed/inactive listings identified
- [ ] Root cause of each suppression documented
- [ ] Prohibited claim scan completed on supplement/health listings
- [ ] Main images checked for text overlay and white background
- [ ] Backend keywords scanned for competitor brand names
- [ ] A+ content / Enhanced Brand Content status checked

## Policy and Violations

- [ ] All open violation notices reviewed
- [ ] Any unactioned notices from previous audit addressed
- [ ] Previous violation root causes confirmed as resolved (not just patched)
- [ ] No pending intellectual property complaints outstanding

## Customer Feedback Review

- [ ] Last 90-day negative feedback categorized by type
- [ ] Return/refund root causes analyzed
- [ ] Any product with >10% return rate flagged for listing review
- [ ] Any systemic "item not as described" pattern investigated

## Financial Health

- [ ] Disbursement schedule confirmed as normal
- [ ] Any reserve holds investigated and documented
- [ ] Fee changes in last 90 days reviewed and acknowledged

## Risk Scoring and Actions

- [ ] Each audit category scored 1–5 for risk
- [ ] All scores 4–5 have action plans with named owners and deadlines
- [ ] Previous audit actions confirmed complete (or re-escalated)
- [ ] Next audit date scheduled and on calendar
