## Description: <br>
Generates podcast audio from scripts through SkillBoss model access and a SkillBoss API key. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and developers use this skill to generate podcast-style audio from scripts, compare model options, and configure SkillBoss API access for the workflow. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill requires a paid SkillBoss API key that may enable access to many APIs beyond podcast audio generation. <br>
Mitigation: Use a restricted key or billing limits where available, and require explicit confirmation before any non-podcast capability is used. <br>
Risk: Podcast scripts may contain sensitive or unpublished content that would be sent to a third-party API. <br>
Mitigation: Review scripts before submission and avoid sending confidential, regulated, or proprietary content unless the selected provider and account terms are approved. <br>
