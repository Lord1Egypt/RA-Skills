## 一、义乌商贸城老档口的选品秘诀

### 1.1 选品核心原则

```
┌─────────────────────────────────────────────────────────────────┐
│                      义乌选品四大原则                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ① 轻小薄 → 运费省，利润空间大                                  │
│  ② 耐运输 → 破损率低，售后少                                    │
│  ③ 低侵权 → 避开知识产权风险                                    │
│  ④ 通用性 → 刚需产品，复购率高                                  │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 选品数据维度

| 维度 | 指标 | 参考标准 |
|------|------|----------|
| **体积重量** | 体积<30cm³ | 单件运费<5%货值 |
| **耐摔程度** | 材质 | 塑料/金属/玻璃分级 |
| **侵权风险** | 品牌识别 | 无明显品牌logo |
| **市场需求** | Google Trends | 搜索量稳定 |
| **竞争程度** | 供应商数量 | <100家 |
| **利润空间** | 毛利率 | >30% |
| **复购周期** | 使用频次 | <6个月复购 |

### 1.3 老档口的选品直觉

> "我做了20年饰品生意，选品主要看三点：一是重量，同等价值的东西，越轻越好；二是通用性，太小众的东西不好卖；三是更新速度，跟上潮流才能赚钱。"
> —— 义乌饰品档口 陈老板

**实战经验：**

1. **试水原则**
   - 新品类先小批量试水（100-500件）
   - 观察2-3周的销量再决定是否追加
   - 第一批不赚钱正常，关键是测试市场

2. **爆款嗅觉**
   - 每天逛商贸城，观察哪些档口人多
   - 关注网红/爆款视频，同款快速跟进
   - 跟着展会趋势走，广交会上的新品

3. **组合策略**
   - 主打产品（有量）+ 利润产品（有利润）+ 新奇特（引流）
   - 产品线不超过20个SKU
   - 每年淘汰20%老品，更新20%新品

### 1.4 选品禁区

| 禁区类型 | 风险 | 案例 |
|----------|------|------|
| 明显仿牌 | 海关扣货+法律风险 | 仿LV包被没收 |
| 敏感材质 | 出口限制 | 含磁物品空运受限 |
| 季节性强 | 库存风险 | 圣诞装饰过季滞销 |
| 体积过大 | 运费吞噬利润 | 大件家具运费比货值高 |
| 更新太快 | 压货风险 | 手机壳款式淘汰快 |

## 二、老外贸的议价技巧

### 2.1 议价三步法

```
┌─────────────────────────────────────────────────────────────────┐
│                      议价三步法                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Step 1: 了解底牌                                               │
│  ─────────────────────────────────────────────────────          │
│  "您能告诉我您期望的价格范围吗？"                               │
│  "除了价格，还有哪些因素对您很重要？"                           │
│                                                                  │
│  Step 2: 价值锚定                                               │
│  ─────────────────────────────────────────────────────          │
│  先展示价值，再谈价格                                           │
│  "我们的产品包含..."                                            │
│  "市场上同类产品的价格是..."                                    │
│                                                                  │
│  Step 3: 交换让步                                               │
│  ─────────────────────────────────────────────────────          │
│  降价可以，但要有交换条件                                       │
│  "如果我们给您5%的折扣，您能接受50%的预付款吗？"                │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 价值传递话术

**成本透明法：**
```
"让我给您算一下这个产品的成本构成：
- 原材料成本：$2.5
- 人工成本：$1.0
- 包装成本：$0.3
- 运输成本：$0.5
- 其他费用：$0.2
- 利润：$0.5
─────────────────
总计：$5.0

您看，我们的利润只有10%，再低就真的要亏本了。"
```

**对比锚定法：**
```
"市场上同类产品的价格区间是这样的：
- 普通品质：$4.5-5.0
- 中等品质：$5.5-6.0
- 优质品质：$6.5以上

我们的定位是中等偏上品质，价格$5.8，已经是很有竞争力的了。"
```

**附加价值法：**
```
"除了这个价格，您还能获得：
✓ 免费更换少量不良品
✓ 优先排单生产
✓ 灵活的付款方式
✓ 全程中文客服支持

这些服务的价值大约$0.3/件，所以实际上我们的性价比是很高的。"
```

### 2.3 老外贸的议价经验

> "议价最忌讳的就是客户一还价就答应，这样客户反而会觉得还有空间。正确的做法是先坚守价格，等客户亮出真正的底牌后再适度让步，而且让步一定要有条件。"
> —— 义乌机械出口 张总

**实战技巧：**

1. **分拆报价法**
   - 把总价拆成单价，让客户感知每个部分的价值
   - 提供阶梯报价，让客户感觉买越多越划算

2. **终极让步法**
   - 不要一开始就亮底价
   - 预留2-3次小幅让步空间
   - 最后一次让步要显得很为难

3. **条件交换法**
   - 降价=换条件（数量/预付款/交期）
   - 让客户感觉是在"谈判"而非"讨价还价"
   - 达成双方都满意的交易

├─────────────────────────────────────────────────────────────────┤

│         ┌────────────────────┬────────────────┐               │
│         │                    │                │               │
│         │   A级供应商        │   B级供应商    │               │
│         │   战略合作        │   重点培养     │               │
│         │                    │                │               │
│         ├────────────────────┼────────────────┤               │
│         │                    │                │               │
│         │   C级供应商        │   D级供应商    │               │
│         │   观察改进        │   淘汰替换     │               │
│         │                    │                │               │

### 3.2 供应商关系维护

**A类供应商（战略合作）：**
- 每月面对面沟通
- 年度框架协议
- 优先保障产能
- 联合开发新品
- 账期支持

**B类供应商（重点培养）：**
- 季度沟通
- 季度订单计划
- 品质提升辅导
- 逐步加大订单

**C类供应商（观察改进）：**
- 问题反馈与跟踪
- 改进要求明确
- 缩小订单量

**D类供应商（淘汰替换）：**
- 停止新订单
- 逐步清尾库存
- 寻找替代供应商

### 3.3 老档口的供应商管理

> "我档口边上有个专门做收纳盒的工厂，合作了8年。他的东西我放心，我的订单他优先。有时候资金紧张，他也愿意给我30天账期，这就是长期合作的好处。"
> —— 义乌日用百货档口 李老板

**实战经验：**

1. **筛选标准**
   - 看工厂规模（太大太小都不好）
   - 看老板为人（实在最重要）
   - 看配合意愿（愿意配合的优先）
   - 看响应速度（回复快的加分）

2. **合作技巧**
   - 定期拜访维护关系
   - 订单稳定时提前告知计划
   - 出现问题先沟通而非直接换供应商
   - 付款及时，树立信用

3. **风险分散**
   - 同类产品至少2家供应商
   - 不把所有订单压给一家
   - 旺季提前备货，避免产能不足

### 4.2 客户关系维护技巧

> "我做外贸十几年，最大的心得就是：要把客户当朋友。很多客户跟我合作了七八年，从未见过面，但信任度比见面的还高。关键就是真诚和靠谱。"
> —— 义乌某外贸公司负责人 王总

**实战技巧：**

1. **日常维护**
   - 节日祝福（不仅仅是春节）
   - 市场动态分享
   - 新品推荐
   - 订单状态更新

2. **关键时刻**
   - 订单完成后的感谢
   - 发货时的期待表达
   - 客户生日/公司周年祝福
   - 客户遇到困难时的关心

3. **信任建立**
   - 承诺必兑现
   - 问题不推诿
   - 主动告知风险
   - 超出期望的服务

### 5.2 老外贸的风险经验

> "做了这么多年外贸，见过太多骗子。最重要的经验就是：不见兔子不撒鹰。不管客户说得多好，不付款就不发货，宁可丢单也不冒险。"
> —— 义乌工量刃具出口 刘总

**实战经验：**

1. **付款安全**
   - 首单要求30-50%预付款
   - 大额订单分批出货
   - 正本提单在手，等尾款到账
   - 水单必须核对

2. **合同安全**
   - 所有重要条款书面确认
   - 保存完整沟通记录
   - 关键节点截图存档
   - 合同条款明确

3. **货物安全**
   - 出货前确认付款
   - 验货报告留存
   - 保险足额投保
   - 物流轨迹追踪

### 5.3 风险应对原则

| 风险类型 | 预防措施 | 应对策略 |
|----------|----------|----------|
| 支付诈骗 | 预付款+水单核对 | 立即停止+报警 |
| 货款纠纷 | 合同规范+证据留存 | 协商优先+法律途径 |
| 货物损失 | 足额保险+规范包装 | 保险索赔+物流追责 |
| 政策风险 | 合规经营+政策跟踪 | 调整策略+专业咨询 |
| 汇率风险 | 分批结汇+锁汇 | 调整报价+成本对冲 |

## 六、方法论总结

### 6.1 选品方法论

```
选品决策矩阵：

            高利润
               │
    新奇特 ◀───┼───▶ 刚需品
               │
    低频次 ────┼───▶ 高频次
               │
            高竞争
```

**最佳选择**：高利润+刚需品+高频次+低竞争

### 6.2 议价方法论

```
议价成功 = 价值感知 + 底线坚守 + 条件交换
```

**核心**：让客户感觉在谈判中获得了价值，而非单纯的价格降低

### 6.3 供应商管理方法论

```
供应商关系 = 筛选 + 培养 + 分级 + 维护
```

**核心**：找到愿意配合的供应商，长期合作，共同成长

### 6.4 客户维护方法论

```
客户忠诚 = 产品价值 + 服务体验 + 关系温度
```

**核心**：把客户当朋友，真诚靠谱，超出期望

### 6.5 风险防控方法论

```
风控原则 = 预防为主 + 信号警觉 + 底线坚守
```

**核心**：不见兔子不撒鹰，宁可丢单不冒险

## 七、经验传承路径

### 7.1 新人学习路径

```
第一阶段（1-3个月）：
├── 熟悉产品 → 掌握产品知识和卖点
├── 熟悉流程 → 跟单全流程操作
├── 熟悉客户 → 从维护老客户开始
└── 积累经验 → 在实战中学习

第二阶段（3-6个月）：
├── 独立开发 → 尝试开发新客户
├── 独立谈判 → 参与议价谈判
├── 独立处理 → 独立解决问题
└── 形成方法 → 总结自己的经验

第三阶段（6-12个月）：
├── 带教新人 → 传承经验
├── 优化流程 → 改进不足
├── 客户深耕 → 维护核心客户
└── 经验输出 → 分享经验教训
```

### 7.2 知识沉淀机制

| 形式 | 内容 | 受众 |
|------|------|------|
| SOP文档 | 标准化流程 | 新人 |
| 话术手册 | 场景话术 | 销售 |
| 案例库 | 成功/失败案例 | 全员 |
| 经验分享 | 月度/季度分享 | 全员 |
| 师徒制 | 老带新 | 新人 |
