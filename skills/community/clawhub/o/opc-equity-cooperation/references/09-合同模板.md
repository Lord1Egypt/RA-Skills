# 合同模板：合作协议、服务采购协议模板

> 本文件提供OPC分布式协作中台的标准化合同模板，包括联盟合作公约、项目合作协议、服务采购协议模板等。

---

## 一、合同模板清单

| 模板名称 | 用途 | 使用场景 |
|---------|------|---------|
| 《社区生意联盟合作公约》 | 联盟成员基础协议 | 联盟型SPV |
| 《项目软SPV合作协议》 | 项目成员合作协议 | 项目型SPV |
| 《服务采购协议》 | 各方与牵头方协议 | 所有类型 |
| 《贡献权重协议》 | 贡献权重确认协议 | 调整权重时 |
| 《共管账户管理协议》 | 账户管理协议 | 资金管理 |
| 《退出清算协议》 | 退出时的清算协议 | 成员退出 |

## 二、《社区生意联盟合作公约》模板

### 模板正文

```
【社区生意联盟合作公约】

甲方（盟主）：________________
乙方（成员）：________________

签订日期：____年__月__日

---

一、联盟宗旨

甲乙双方本着"合作共赢、贡献共享"的原则，共同参与[联盟名称]社区生意联盟（以下简称"本联盟"），通过资源整合、能力互补，实现共同发展。

二、联盟服务内容

甲方为本联盟成员提供以下服务：
1. 品牌赋能：使用[品牌名称]品牌进行市场推广
2. 供应链支持：对接优质供应链资源
3. 获客支持：提供客户引流和转化支持
4. 结算服务：提供项目结算和分账服务

三、成员权利与义务

1. 乙方权利：
   - 享有使用联盟品牌和资源的权利
   - 享有按贡献权重获得收益分配的权利
   - 享有参与联盟事务的投票权（按约定）
   - 享有对联盟运营的知情权

2. 乙方义务：
   - 按约定完成分配的任务和工作
   - 如实报告业务情况和贡献
   - 遵守联盟章程和决议
   - 维护联盟品牌和声誉
   - 保守联盟商业秘密

四、收益分配

1. 分配原则：按乙方贡献权重进行收益分配
2. 分配比例：甲方收取服务费[5]%，乙方按贡献权重分配剩余收益
3. 分配周期：按项目/按月结算，具体以项目约定为准
4. 税费处理：各自承担税费，可由甲方代扣代缴

五、信用管理

1. 联盟建立成员信用档案
2. 信用等级分为S/A/B/C/D五级
3. 信用分影响收益分配系数
4. 具体规则详见《信用评级管理办法》

六、争议解决

1. 争议发生后，各方应友好协商解决
2. 协商不成的，提交[仲裁机构/法院]解决
3. 争议解决期间，各方应继续履行无争议部分的义务

七、协议变更与解除

1. 协议变更需经双方书面同意
2. 任何一方可提前[30]天书面通知解除协议
3. 协议解除后，按贡献情况进行清算

八、其他

1. 本协议自签订之日起生效，有效期[1]年
2. 本协议一式两份，甲乙双方各执一份
3. 本协议未尽事宜，由双方另行协商解决

---

甲方（签字）：________  乙方（签字）：________
日期：________         日期：________
```

## 三、《项目软SPV合作协议》模板

### 模板正文

```
【项目软SPV合作协议】

项目名称：________________
项目编号：________________

甲方（牵头方）：________________
乙方（成员方）：________________
丙方（成员方）：________________
丁方（成员方）：________________

签订日期：____年__月__日

---

一、项目概述

1.1 项目内容：[具体描述项目内容和目标]

1.2 项目周期：____年__月__日至____年__月__日

1.3 项目预算：[项目总预算]元

二、SPV成员及贡献权重

| 成员 | 贡献类型 | 贡献描述 | 贡献权重 |
|------|---------|---------|---------|
| 甲方 | [技能/资源/资金] | [具体贡献] | [X]% |
| 乙方 | [技能/资源/资金] | [具体贡献] | [X]% |
| 丙方 | [技能/资源/资金] | [具体贡献] | [X]% |
| 丁方 | [技能/资源/资金] | [具体贡献] | [X]% |
| **合计** | - | - | **100%** |

三、贡献类型说明

3.1 技能贡献：[描述技能贡献的具体内容]

3.2 资源贡献：[描述资源贡献的具体内容]

3.3 时间贡献：[描述时间贡献的具体内容]

3.4 资金贡献：[描述资金贡献的具体内容]

四、里程碑与验收

| 里程碑 | 计划完成时间 | 验收标准 | 对应收益 |
|--------|-------------|---------|---------|
| 里程碑1 | | | |
| 里程碑2 | | | |
| 里程碑3 | | | |
| 最终交付 | | | |

五、收益分配

5.1 分账流程：
   - 项目回款进入共管账户
   - 扣除税费预留
   - 扣除平台服务费[5]%
   - 扣除实际成本
   - 按贡献权重分配剩余收益

5.2 分账优先级：
   1. 税费预留（按国家规定比例）
   2. 平台服务费
   3. 实际成本报销
   4. 按贡献权重分配

5.3 信用系数：
   - 分配时考虑各成员信用系数
   - 信用分对应系数：S级1.2、A级1.0、B级0.9、C级0.7、D级0.5

六、牵头方职责

甲方作为本项目牵头方，负责：
1. 对外签署项目合同
2. 管理共管账户
3. 开具发票、税务处理
4. 分账执行
5. 档案保管

七、三长老治理

7.1 盟主：[指定盟主姓名]
   - 负责日常运营决策
   - 协调成员关系

7.2 合规长老：[指定合规长老姓名]
   - 负责合规审查
   - 风险监控

7.3 监事长老：[指定监事长老姓名]
   - 账务监督
   - 发起弹劾

八、争议解决

8.1 争议发生后，各方应友好协商解决

8.2 协商不成的，依次按以下程序处理：
   - 一级：快速协调（24小时内）
   - 二级：盟主+监事调解（48小时内）
   - 三级：全体公投裁决

8.3 仍无法解决的，提交[仲裁机构/法院]解决

九、退出与清算

9.1 成员退出需提前[15]天书面申请

9.2 退出清算：
   - 按退出时的贡献权重计算应得收益
   - 扣除未完成任务的扣减
   - 完成所有交接工作

9.3 退出后不享有未来收益

十、其他

10.1 本协议自签订之日起生效

10.2 本协议一式[份数]份，各方各执一份

10.3 本协议未尽事宜，由各方另行协商签订补充协议

---

各方签字：

甲方（牵头方）：________________  日期：________
乙方（成员方）：________________  日期：________
丙方（成员方）：________________  日期：________
丁方（成员方）：________________  日期：________
```

8.2 本协议一式三份，甲乙丙三方各执一份

8.3 本协议未尽事宜，由三方另行协商解决

## 七、合同签署注意事项

### 7.1 签署前检查

| 检查项 | 说明 |
|--------|------|
| 各方身份确认 | 核实签约方的真实身份 |
| 权限确认 | 确认签约人有签约权限 |
| 内容核对 | 确认合同内容与约定一致 |
| 敏感词检查 | 确认无违规敏感词 |
| 格式检查 | 确认合同格式规范 |

### 7.2 签署后管理

| 管理项 | 说明 |
|--------|------|
| 合同归档 | 所有合同扫描存档 |
| 上链存证 | 合同关键信息上链 |
| 履行跟踪 | 跟踪合同履行情况 |
| 变更记录 | 变更内容补充协议 |
