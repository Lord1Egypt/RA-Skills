## 📖 这是什么

**家庭关怀技能**是一个为**原生家庭觉察与疗愈**设计的 AI 陪伴 Skill。

> 不是心理医生，不是人生导师。  
> 是一个足够温暖、足够安全的陪伴者。  
> 让你在对话中，慢慢理清那些来自家庭的"应该"和"不可以"里，  
> 哪些可以放下，哪些可以重新选择。

### ✨ 最大的不同：你不用说"请让XX来"

跟其他陪伴工具不同，**你不需要记住任何角色名字**。

在你主动使用本 Skill 后，你只需要说出你的感受，Skill 会根据你的描述自动判断你的状态，从八位历史人物中选出最懂你的那个人来回应你。

> ⚠️ **关于自动匹配**：本 Skill 仅在你**主动发起对话并在该技能上下文中**时进行角色匹配。它不会在后台监控或分析你的日常对话。所有匹配均基于你自愿分享的当前输入内容。如果你不希望接受自动角色匹配，可以随时指定你偏好的角色名称。

```text
"今天跟我妈吵架了，她说我自私……我好难过。"
→ 🧠 检测到疗愈需求 → 自动匹配 班昭（理解姐姐）来倾听你

"我为什么总是讨好别人？跟我爸从小要求完美有关。"
→ 🧠 检测到觉察需求 → 自动匹配 诸葛亮（智慧父亲）帮你分析

"好累，能不能陪陪我。"
→ 🧠 检测到陪伴需求 → 自动匹配 孟母（温暖母亲）给你拥抱
```

当然，如果你有偏好的角色，随时可以指名道姓：

```text
"温暖母亲，我今天好难过。"
"诸葛亮帮我分析一下。"
```

## 🧠 路由引擎

Skill 内置一个智能路由引擎（`src/router.js`），自动完成从输入到身份匹配的全流程：

```
用户输入 → 检测模式 → 识别家庭模式 → 判断情绪强度 → 自动匹配身份 → 生成回应
```

> ⚠️ **重要说明**：自动匹配基于有限的文本信号（关键词和句式），属于启发式判断，**并非对用户心理状态的诊断**。匹配结果仅供参考，用户可随时否决或更改。如果匹配感觉不合适，直接说"换一个"或指定你偏好的角色即可。

### 匹配决策树

```
用户输入
│
├─ 显式指定角色？（"温暖母亲""诸葛亮""西施"等）
│  └─ ✅ 尊重用户选择，不覆盖
│
└─ 无指定角色 → 自动匹配
   │
   ├─ 疗愈模式 ─── 高强度 → 温暖母亲（马皇后）
   │              低强度 → 理解姐姐（班昭）
   │
   ├─ 觉察模式 ─── 迷茫 → 智慧父亲（诸葛亮）
   │              刚有觉察 → 理解姐姐（班昭）
   │              理论多缺体感 → 智慧父亲（姜子牙）
   │
   ├─ 行动模式 ─── → 支持兄长（刘伯温）
   │
   ├─ 陪伴模式 ─── 情感忽视 → 温暖母亲（长孙皇后）
   │              有条件爱 → 温暖母亲（孟母）
   │              被控制 → 智慧父亲（诸葛亮）
   │              愧疚感 → 支持兄长（刘伯温）
   │              害怕抛弃 → 温暖母亲（长孙皇后）
   │              日常 → 温暖母亲（孟母）
   │
   └─ 延续模式 ─── 按检测到的家庭模式细化匹配
```

## 🌱 核心理念

走出原生家庭，不是指责任何人。而是：

1. **觉察** — 看见自己身上哪些模式是从家庭中习得的
2. **理解** — 理解父母的局限也是他们原生家庭的产物
3. **选择** — 不再自动重复旧模式，有意识地选择新的方式
4. **成长** — 建立自己的边界、价值观和关系方式

> 你不是你的过去。  
> 你是从过去中走出来的那个人。
