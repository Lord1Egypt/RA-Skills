# 📦 weekly-report-hy

> 周报生成工具 — 将零散的工作记录整理为格式规范、表述专业的中文周报

[![npm version](https://img.shields.io/npm/v/weekly-report-hy.svg)](https://www.npmjs.com/package/weekly-report-hy)
[![License](https://img.shields.io/npm/l/weekly-report-hy.svg)](https://github.com/cryptocxf/weekly-report-hy/blob/main/LICENSE)

## 📋 功能简介

**weekly-report-hy** 是一个智能周报生成工具，接收用户提供的零散、口语化的中文工作记录，自动整理为格式规范、表述专业的工作周报。

### 核心能力

| 功能 | 说明 |
|------|------|
| 🧠 **智能分类** | 自动按项目或主题对工作内容归类，生成 `【项目名】` 式分段结构 |
| ✍️ **专业化润色** | 将口语化描述转为专业工作表述，补全合理语境，丰富表达 |
| 📐 **格式标准化** | 统一的分段、编号、标点规范，开箱即用 |
| 🚫 **内容保真** | 基于原文事实润色，绝不编造未提及的内容 |

---

## 🚀 使用方式

本工具作为 AI 技能（Skill）使用，配合支持 OpenClaw 协议的 AI 助手。

### 快速使用

在对话中提供你的本周工作记录，例如：

```
这周做了：
1. 跟产品部对了下需求
2. 写了个数据中台的方案初稿
3. 看了下竞品 ABC 的功能
4. 帮新人小李配了开发环境
5. 开了两次部门周会
```

AI 助手将自动调用 `weekly-report-hy` 技能，输出格式规范的周报。

---

## 📖 输出示例

**输入：**
```
XX 平台每日运行数据处理；支撑业务部门同事进行平台账号开通；配合外部领导完成产品线上演示；撰写行业标准草案修改意见。
```

**输出：**
```
【XX 平台】
1. 完成 XX 平台每日运行数据的拉取、清洗与汇总分析，持续保障数据日报的准确性与时效性；
2. 支撑业务部门同事完成 XX 平台的账号开通、配置与基础功能使用；
3. 配合外部领导调研需要，完成 XX 平台的线上演示与功能讲解。

【科研项目】
1. 围绕行业标准草案开展深入研读与对标分析，撰写并提交本轮修改意见。
```
