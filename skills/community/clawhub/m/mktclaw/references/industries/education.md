## 1. 行业概览

### 1.1 市场规模与增长

| 细分品类 | 市场规模(2025) | 年增速 | 主要趋势 |
|---------|:------------:|:-----:|---------|
| K12 课外培训 | ≈ 4000 亿 | -10% (政策影响) | 素质教育转型、AI 伴学 |
| 职业培训 | ≈ 6000 亿 | 10-15% | 技能重塑、考公考研热 |
| 成人英语 | ≈ 600 亿 | 5-8% | 实用英语、口语外教 |
| 留学咨询 | ≈ 500 亿 | 8-12% | 低龄化、多国联申 |
| 编程/IT 培训 | ≈ 1500 亿 | 15-20% | AI 课程、就业导向 |
| 兴趣教育（音乐/美术） | ≈ 2500 亿 | 8-12% | 成人兴趣、在线化 |

### 1.2 大促节点

| 节点 | 重要性 | 关键动作 |
|------|:-----:|---------|
| 寒假（1-2 月） | ⭐⭐⭐⭐ | 春季班招生 |
| 开学季（2-3 月 / 8-9 月） | ⭐⭐⭐⭐⭐ | K12 招生高峰 |
| 暑期（6-8 月） | ⭐⭐⭐⭐⭐ | 全年最大投放窗口 |
| 618 / 双11 | ⭐⭐⭐ | 在线课程促销 |
| 考研季（9-12 月） | ⭐⭐⭐⭐ | 考研培训冲刺 |
| 公考（10-11 月） | ⭐⭐⭐⭐ | 公考培训 |

### 1.3 竞争格局特征

- **政策敏感**：双减后 K12 转型素质教育，行业格局重塑
- **强监管**：广告限制多，资质要求严
- **长决策周期**：高客单 + 长周期 + 强试听
- **口碑驱动**：老学员推荐、口碑转化率高
- **私域+社群**：社群运营是核心转化场

## 5. 合规与监管

### 5.1 广告法专属限制

| 限制类型 | 禁用 |
|---------|------|
| 升学率承诺 | "包过""保过""100% 上岸""升学率 99%" |
| 名校保证 | "保录 XX 大学""保进 XX 班" |
| 教师资质虚假 | 不得虚标名师/教研团队 |
| 价格欺诈 | "原价 X 万，现价 Y" 必须有真实销售记录 |

### 5.2 双减政策专属红线（K12）

| 红线 | 说明 |
|------|------|
| 学科类培训广告 | 寒暑假/国家法定节假日/休息日不得发布 |
| 学龄前培训 | 不得开展面向学龄前儿童的线上培训 |
| 应试导向 | 不得宣传"刷题""押题""应试技巧" |

### 5.3 行业特殊资质

| 业务类型 | 必备资质 |
|---------|---------|
| K12 非学科 | 民办学校办学许可证 + 营业执照 |
| 职业培训 | 民办职业培训学校许可证 |
| 留学咨询 | 教育部留学中介资质 |
| 在线教育 | ICP 经营许可证 + 网络文化经营许可证 |

### 5.4 平台审核红线

| 平台 | 红线 |
|------|------|
| 抖音 | "保过""升学率"、制造教育焦虑 |
| 小红书 | 虚假"上岸经验"、软广未声明 |
| 微信 | 多级分销、虚假宣传 |

## 7. 常见雷区

| 雷区 | 严重度 | 规避方法 |
|------|:-----:|---------|
| 升学率承诺 | 🔴 致命 | 用"提升""进步"代替 |
| 制造教育焦虑 | 🔴 高 | 正向激励为主 |
| 名师资质虚假 | 🔴 致命 | 真实展示教师证书 |
| 学科类违规时段投放 | 🔴 致命 | 严格遵守双减 |
| 试听课质量与正课不一致 | 🟡 中 | 试听即正课片段 |

| 方法论 | 适用场景 |
