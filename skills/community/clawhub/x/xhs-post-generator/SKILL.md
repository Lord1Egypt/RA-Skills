# XHS Post Generator

Generate Xiaohongshu (Little Red Book) style social media posts. 
Auto-formats content with emoji, line breaks, hashtags, and platform-appropriate tone.

## Features
- Topic to full post generation
- Emoji-enhanced formatting
- Hashtag suggestion
- Multiple tones (casual, professional, storytelling)

## Usage

User says: "Write a XHS post about [topic]"
Skill generates: Formatted post with emoji, sections, and hashtags

## Requirements
- Should only run when user wants to post to Xiaohongshu
- Keep posts under 1000 characters
- Use 3-5 relevant hashtags
