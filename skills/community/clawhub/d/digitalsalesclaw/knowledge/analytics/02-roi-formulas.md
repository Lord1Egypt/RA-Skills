# ROI 公式体系

> 本文档定义 DigitalSalesClaw 中所有效果衡量的 ROI/CAC/ROAS 计算公式，
> 供归因分析、智能投放优化、预算分配使用。

---

## 一、基础公式

### 1.1 ROAS（广告支出回报率）
```
ROAS = 广告带来的转化收入 / 广告支出
     = (转化数 × 客单价) / 广告花费

判断标准:
  ROAS ≥ 3.0  → 优秀，可追加预算
  ROAS ≥ 1.5  → 合格，维持现状
  ROAS < 1.5  → 亏损，需优化或暂停
```

### 1.2 ROI（投资回报率，含利润）
```
ROI = (转化收入 - 成本) / 总投入 × 100%
    = (GMV × 利润率 - 总投入) / 总投入 × 100%

注意: ROI 可以超过 100%（当利润 > 投入时）
```

### 1.3 CAC（客户获取成本）
```
CAC = 总营销费用 / 新增客户数

分解:
  CAC = (广告花费 + 人工成本 + 内容制作费) / 新增患者数

各渠道 CAC 对比:
  抖音CAC = 抖音广告花费 / 抖音新患者
  小红书CAC = 小红书广告花费 / 小红书新患者
```

### 1.4 LTV（客户终身价值）
```
LTV = 平均客单价 × 平均购买频次 × 平均客户生命周期

简化公式:
  LTV = 平均订单金额 × 月均购买次数 × 平均生命周期(月)

LTV参考:
  糖尿病慢病产品: LTV约 ¥3000-8000/年
  急性病产品: LTV约 ¥200-800
```

### 1.5 LTV/CAC 比值
```
LTV/CAC = 客户终身价值 / 客户获取成本

判断标准:
  LTV/CAC ≥ 3  → 优秀，业务可持续
  LTV/CAC ≥ 1  → 可行，需优化获客效率
  LTV/CAC < 1  → 亏损，需立即调整
```

## 二、渠道效果公式

### 2.1 CPE（互动成本）
```
CPE = 广告花费 / 总互动数（点赞+评论+收藏+转发）
```

### 2.2 CPM（千次曝光成本）
```
CPM = (广告花费 / 曝光量) × 1000
```

### 2.3 CPC（单次点击成本）
```
CPC = 广告花费 / 点击数
```

### 2.4 CPA（单次行动成本）
```
CPA = 广告花费 / 转化数（注册/留资/下单）
```

### 2.5 CVR（转化率）
```
CVR = 转化数 / 点击数 × 100%
    = (广告转化数 / 广告点击数) × 100%
```

### 2.6 CTR（点击率）
```
CTR = 点击数 / 曝光量 × 100%
```

## 三、归因模型公式

### 3.1 First-Touch 归因
```
首次触达渠道获客权重 = 100%
其他渠道 = 0%
```

### 3.2 Last-Touch 归因
```
末次触达渠道获客权重 = 100%
其他渠道 = 0%
```

### 3.3 Linear 归因
```
各触达渠道权重 = 1 / 触达渠道数量
```

### 3.4 Time-Decay 归因（基于时间衰减）
```
渠道权重 = e^(-λ × 天数距转化的时间差)
其中 λ 为衰减系数（通常取 0.1-0.3）
```

### 3.5 Position-Based 归因
```
首触权重 = 40%
末触权重 = 40%
中间渠道均分剩余 20%
```

## 四、库存与供应链公式

### 4.1 安全库存
```
安全库存 = (最大日消耗量 - 平均日消耗量) × 补货周期
```

### 4.2 库存周转率
```
库存周转率 = 出库量 / 平均库存
周转天数 = 365 / 库存周转率
```

### 4.3 库存满足率
```
库存满足率 = 有货订单数 / 总订单数 × 100%
```

---

## 五、内容效果公式

### 5.1 爆款率
```
爆款率 = 爆款内容数 / 总发布内容数 × 100%
（爆款定义: 互动 > 品类平均的 3倍）
```

### 5.2 内容互动率
```
互动率 = (点赞+评论+收藏+分享) / 曝光量 × 100%
```

### 5.3 KOL 合作效率
```
KOL合作效率 = GMV(带货款) / (坑位费 + 佣金)
```
