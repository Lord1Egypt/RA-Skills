## 1. DCF 估值模型

### 1.1 概述

DCF（Discounted Cash Flow，折现现金流）是通过预测公司未来自由现金流并折现到当前的估值方法，得到公司的内在价值。

**核心公式**：
```
企业价值 = Σ(FCFₜ / (1+WACC)^t) + 终值 / (1+WACC)^n
```

**适用场景**：
- 现金流稳定、可预测的成熟公司
- 有足够历史财务数据（3-5年）
- 用户需要判断当前股价高估/低估

**不适用场景**：
- 尚未盈利的初创公司
- 周期性极强的公司（需要正常化盈利）
- 金融类公司（银行、保险）— 使用 DDM 或 P/B 更合适

### 1.2 历史财务分析

分析最近 3-5 年历史数据，建立预测基础：

| 指标 | 计算方法 | 关注要点 |
|------|---------|---------|
| 营收 CAGR | (末年/初年)^(1/n) - 1 | 增长趋势是否可持续 |
| 毛利率趋势 | 毛利润/营收 | 稳定性、行业对比 |
| EBIT 利润率 | 营业利润/营收 | 经营杠杆效应 |
| D&A/营收 | 折旧摊销/营收 | 资本密集度 |
| CapEx/营收 | 资本开支/营收 | 增长投入力度 |
| NWC/营收变动 | ΔNWC/Δ营收 | 营运资本效率 |
| ROIC | NOPAT/(权益+有息负债) | 资本回报是否>WACC |

### 1.3 营收预测（三场景）

**预测期**：5年（标准），高增长公司可延长至 7-10 年。

**增长率框架**：
```
Year 1-2: 较高增长（近期可见度高）
Year 3-4: 向行业平均回归
Year 5:   接近终值增长率
```

**三场景设定**：

| 场景 | 增长假设 | 利润率假设 |
|------|---------|-----------|
| Bear（保守） | 历史增速 × 0.6-0.8 | 利润率持平或小幅下降 |
| Base（基准） | 历史增速持平或参考共识 | 利润率温和扩张 |
| Bull（乐观） | 历史增速 × 1.1-1.3 | 利润率显著扩张 |

**数据来源优先级**（cocode 环境）：
1. yfinance 财务数据（已通过 `fetch_stock_data.py --type financial` 采集）
2. `web_search` 搜索机构共识预期
3. 基于历史趋势外推

### 1.4 自由现金流推算

```
EBIT = 预测营收 × EBIT 利润率
税后 NOPAT = EBIT × (1 - 有效税率)
(+) D&A = 营收 × D&A 占比
(-) CapEx = 营收 × CapEx 占比
(-) ΔNWC = Δ营收 × NWC 占比
= 无杠杆自由现金流 (UFCF)
```

**关键假设**：
- 有效税率：中国企业通常 15-25%（高新技术 15%，一般 25%）
- D&A 占营收比：通常 2-8%，参考历史均值
- CapEx 占营收比：维护性 2-3% + 成长性 2-5%
- ΔNWC 占收入变动比：通常 -2% 到 +5%

### 1.5 WACC 计算

```
WACC = Ke × (E/(D+E)) + Kd × (1-t) × (D/(D+E))
```

**权益成本 (Ke) — CAPM**：
```
Ke = Rf + β × ERP

Rf  = 10年期国债收益率（通过 web_search 获取最新值）
β   = yfinance 提供或 web_search 查询
ERP = 5.0-6.0%（市场标准）
```

**债务成本 (Kd)**：
```
税前 Kd = 利息费用 / 总负债（从财报计算）
税后 Kd = Kd × (1 - 税率)
```

**资本结构权重**：
```
E = 市值 = 当前股价 × 总股本
D = 净负债 = 总负债 - 现金
权益占比 = E / (E + D)
负债占比 = D / (E + D)
```

**典型 WACC 区间**：
- 大型稳定蓝筹：7-9%
- 成长型企业：9-12%
- 高风险/小盘：12-15%

### 1.6 终值计算

**永续增长法**（首选）：
```
终值 = 末年FCF × (1 + g) / (WACC - g)

g = 永续增长率（通常 2-3%，不超过 GDP 增速）
```

**退出倍数法**（备选/交叉验证）：
```
终值 = 末年EBITDA × 退出EV/EBITDA倍数

退出倍数来源：同行业交易倍数或当前交易倍数
```

**终值折现**（中年惯例）：
```
PV(终值) = 终值 / (1 + WACC)^(n-0.5)
```

**合理性检查**：
- 终值占企业价值的 50-70% → 正常
- > 75% → 过度依赖终值假设，需审慎
- < 40% → 终值假设可能过于保守

### 1.7 估值汇总

```
(+) Σ FCF 折现值 = $X
(+) 终值折现值 = $Y
= 企业价值 (EV) = $Z
(-) 净负债 [或 + 净现金] = $A
= 股权价值 = $B
÷ 总股本 = C
= 每股内在价值 = $D
vs 当前股价 = $E
隐含回报 = (D/E - 1) × 100%
```

### 1.8 敏感性分析

构建两个敏感性矩阵：

**矩阵1：WACC vs 终值增长率 → 每股价值**

|  | g=1.5% | g=2.0% | g=2.5% | g=3.0% | g=3.5% |
|--|--------|--------|--------|--------|--------|
| WACC 8% | | | | | |
| WACC 9% | | | | | |
| WACC 10% | | | | | |
| WACC 11% | | | | | |
| WACC 12% | | | | | |

**矩阵2：营收增速 vs EBIT 利润率 → 每股价值**

|  | Margin 20% | Margin 25% | Margin 30% | Margin 35% | Margin 40% |
|--|-----------|-----------|-----------|-----------|-----------|
| Growth 5% | | | | | |
| Growth 10% | | | | | |
| Growth 15% | | | | | |
| Growth 20% | | | | | |
| Growth 25% | | | | | |

## 2. 可比公司分析 (Comps)

### 2.1 概述

通过对比同行业公司的运营指标和估值倍数，判断目标公司的相对估值水平。

**核心原则**："同类公司应有相似的估值倍数，差异反映基本面差距。"

### 2.2 可比公司选择标准

选择 3-5 家公司，优先匹配：

| 维度 | 优先级 | 说明 |
|------|--------|------|
| 行业/子行业 | **必须** | 同一细分行业 |
| 商业模式 | 高 | 收入结构、盈利模式相似 |
| 市值规模 | 中 | 同一量级（大盘配大盘） |
| 增长阶段 | 中 | 高增长配高增长 |
| 地域市场 | 低 | 同一市场更具可比性 |

**A 股常见可比组**（示例）：
- 白酒：贵州茅台 / 五粮液 / 泸州老窖 / 山西汾酒
- 银行：招商银行 / 宁波银行 / 兴业银行 / 平安银行
- 新能源车：比亚迪 / 理想汽车 / 蔚来 / 小鹏

### 2.3 运营指标对比

**核心指标**（必须包含）：

| 指标 | 公式 | 意义 |
|------|------|------|
| 营收增长率 | (本年营收-去年营收)/去年营收 | 成长能力 |
| 毛利率 | 毛利润/营收 | 产品竞争力 |
| EBITDA 利润率 | EBITDA/营收 | 核心盈利能力 |
| 净利率 | 净利润/营收 | 底线盈利 |
| ROE | 净利润/股东权益 | 股东回报 |
| 资产负债率 | 总负债/总资产 | 财务健康度 |

**可选指标**（根据行业选择）：
- SaaS：ARR、NRR、CAC、LTV/CAC
- 零售：同店增长、坪效
- 制造业：产能利用率、存货周转
- 金融：不良贷款率、净息差

### 2.4 估值倍数对比

| 倍数 | 公式 | 适用场景 |
|------|------|---------|
| PE (TTM) | 市值/净利润 | 盈利稳定的公司 |
| PE (Forward) | 市值/预测净利润 | 增长型公司 |
| PB | 市值/净资产 | 重资产/金融行业 |
| PS | 市值/营收 | 亏损公司/高增长 |
| EV/EBITDA | 企业价值/EBITDA | 跨资本结构对比 |
| EV/Revenue | 企业价值/营收 | 早期/高增长公司 |
| PEG | PE/净利润增长率 | 成长型估值 |

### 2.5 统计基准

对每个估值倍数计算：
```
最大值 (Maximum)
75 分位 (75th Percentile)
中位数 (Median) ← 核心参考
25 分位 (25th Percentile)
最小值 (Minimum)
```

**隐含估值计算**：
```
目标隐含价值 = 同行中位数倍数 × 目标公司对应指标
溢价/折价 = (当前股价/隐含价值 - 1) × 100%
```

### 2.6 相对估值判断

| 位置 | 判断 | 可能原因 |
|------|------|---------|
| > 75 分位 | 显著高估 | 龙头溢价 / 增速更快 / 泡沫 |
| 中位数~75 分位 | 略高估 | 质量溢价 / 品牌价值 |
| 25 分位~中位数 | 略低估 | 增速较慢 / 暂时性困难 |
| < 25 分位 | 显著低估 | 被市场忽视 / 基本面恶化 |

## 3. 三表财务建模

### 3.1 概述

分析利润表、资产负债表、现金流量表的联动关系，评估公司财务健康度并预测未来。

### 3.2 利润表 (Income Statement) 分析

**关键利润率层次**：
```
营收 (Revenue)
- 营业成本 → 毛利润 → 毛利率
- 销售费用
- 管理费用
- 研发费用
→ EBITDA → EBITDA 利润率
- D&A
→ EBIT → EBIT 利润率
+/- 投资收益/财务费用
→ 税前利润
- 所得税
→ 净利润 → 净利率
```

**分析要点**：
- 费用率趋势（销售/管理/研发占收入比是否下降 = 经营杠杆）
- 毛利率稳定性（竞争环境变化的信号）
- 非经常性损益（剔除后的核心利润）
- 同比 / 环比增速

### 3.3 资产负债表 (Balance Sheet) 分析

**资本结构**：
```
总资产 = 流动资产 + 非流动资产
总负债 = 流动负债 + 非流动负债
股东权益 = 总资产 - 总负债

净负债 = 有息负债 - 现金
净负债/EBITDA = 杠杆水平
```

**流动性指标**：
| 指标 | 公式 | 健康标准 |
|------|------|---------|
| 流动比率 | 流动资产/流动负债 | > 1.5 |
| 速动比率 | (流动资产-存货)/流动负债 | > 1.0 |
| 利息保障倍数 | EBIT/利息费用 | > 3 |

**运营效率（现金循环）**：
```
DSO (应收周转天数) = 应收账款/(营收/365)
DIO (存货周转天数) = 存货/(营业成本/365)
DPO (应付周转天数) = 应付账款/(营业成本/365)
CCC (现金循环天数) = DSO + DIO - DPO
```

CCC 越短，运营效率越高。

### 3.4 现金流量表 (Cash Flow Statement) 分析

**现金流质量评估**：

| 指标 | 公式 | 健康标准 |
|------|------|---------|
| 经营现金流/净利润 | OCF/NI | > 1.0（盈利质量高） |
| 自由现金流 | OCF - CapEx | > 0（可自我造血） |
| FCF/营收 | FCF/Revenue | > 5%（良好） |
| CapEx/营收 | CapEx/Revenue | 行业对比 |
| 分红/FCF | Dividend/FCF | < 70%（分红可持续） |

**现金流分类**：
- **经营活动**：核心业务产生的现金（最重要）
- **投资活动**：CapEx + 投资行为
- **筹资活动**：借贷 + 发股 + 分红

### 3.5 三表联动验证

**必须通过的检查**：

```
✅ BS 平衡：总资产 = 总负债 + 股东权益（误差 < 0.1%）
✅ 现金联动：期末现金 = 期初现金 + 三类现金流净额
✅ 利润联动：留存利润变动 ≈ 净利润 - 分红
✅ 折旧联动：CF 中的 D&A = IS 中的 D&A
✅ CapEx 联动：CF 中的 CapEx ≈ 固定资产净增加 + D&A
```

### 3.6 预测方法

**收入预测**：历史增长率 + 行业趋势 + 公司指引
**费用预测**：历史费用率 + 规模效应调整
**资产预测**：基于周转率假设
**负债预测**：基于资本结构目标
**现金流**：由 IS 和 BS 预测自动推导

## 4. 竞争格局分析

### 4.1 概述

评估公司在行业中的竞争地位、护城河宽度和可持续性。

### 4.2 行业分析框架

**行业概览**：
- 市场规模（TAM/SAM/SOM）
- 增长率（CAGR）
- 生命周期阶段（导入/成长/成熟/衰退）
- 关键驱动因素

**行业经济学**：
- 价值链结构和各环节利润分配
- 进入壁垒高低
- 供应商/买方议价能力
- 替代品威胁

### 4.3 竞争者分析

对 3-5 个主要竞争者建立画像：

| 维度 | 分析内容 |
|------|---------|
| 基础信息 | 市值、营收、员工数 |
| 市场地位 | 市场份额、行业排名 |
| 财务表现 | 增长率、利润率、ROE |
| 产品策略 | 产品线、差异化、定价策略 |
| 竞争优势 | 核心壁垒、独特资源 |

### 4.4 护城河评估

| 护城河类型 | 评估标准 | 示例 |
|-----------|---------|------|
| 品牌壁垒 | 品牌溢价、消费者忠诚度 | 贵州茅台、可口可乐 |
| 规模壁垒 | 成本优势、网络效应 | 阿里巴巴、亚马逊 |
| 技术壁垒 | 专利数量、研发投入占比 | 华为、台积电 |
| 转换成本 | 客户迁移难度、生态锁定 | 微软、苹果 |
| 牌照/监管 | 准入门槛、政策保护 | 银行、电信运营商 |

**护城河评级**：
- **宽护城河** ⭐⭐⭐：多重壁垒叠加，10年以上可持续
- **窄护城河** ⭐⭐：单一壁垒，5-10年可持续
- **无护城河** ⭐：竞争激烈，壁垒薄弱

### 4.5 SWOT 分析

```
         有利的              不利的
内  |  Strengths (S)  |  Weaknesses (W)  |
部  |  优势            |  劣势              |
    |                  |                    |
外  |  Opportunities(O)|  Threats (T)      |
部  |  机会            |  威胁              |
```

**输出要求**：每个象限列出 3-5 个要点，每个要点有数据支撑。

## 5. 综合估值判断

### 5.1 多方法交叉验证

将 DCF 和 Comps 结果综合：

| 估值方法 | 每股价值 | 权重 |
|---------|---------|------|
| DCF (Base Case) | $XX | 40% |
| DCF (Bear Case) | $XX | 15% |
| DCF (Bull Case) | $XX | 15% |
| Comps 中位数 PE | $XX | 15% |
| Comps 中位数 EV/EBITDA | $XX | 15% |
| **加权平均** | **$XX** | **100%** |

### 5.2 估值区间

```
低估区间：DCF Bear ~ Comps 25分位
合理区间：DCF Base ~ Comps 中位数
高估区间：DCF Bull ~ Comps 75分位
```

### 5.3 投资建议映射

| 隐含回报 | 建议 | 说明 |
|---------|------|------|
| > +20% | 强烈买入 | 显著低估，安全边际充足 |
| +10% ~ +20% | 买入 | 低估，有上行空间 |
| -10% ~ +10% | 持有 | 估值合理 |
| -20% ~ -10% | 减持 | 高估，上行空间有限 |
| < -20% | 卖出 | 显著高估 |

## 数据源适配说明

本 skill 的专业分析模块**不依赖 MCP 数据源**，而是使用以下数据获取方式：

| 数据需求 | 获取方式 |
|---------|---------|
| 历史财务数据 | `fetch_stock_data.py --type financial`（yfinance） |
| 基本面指标 | `fetch_stock_data.py --type fundamental`（yfinance） |
| 实时价格/市值 | `fetch_stock_data.py --type realtime`（腾讯 API） |
| 无风险利率 | `web_search`（搜索10年期国债收益率） |
| Beta 系数 | yfinance `info.beta` 或 `web_search` |
| 机构共识预期 | `web_search`（搜索机构评级/目标价） |
| 行业数据 | `web_search`（搜索行业报告） |
| 竞争者信息 | `web_search` + 对竞争者执行 `fetch_stock_data.py` |

**注意**：A 股通过 yfinance 获取的财务数据可能不完整，此时应：
1. 通过 `web_search` 搜索最新财报摘要
2. 用户可手动提供财务数据
3. 基于有限数据做保守估计，并明确标注数据局限性
