## Description: <br>
Convert Markdown text to beautifully styled, self-contained HTML with embedded CSS. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[claudiodrusus](https://clawhub.ai/user/claudiodrusus) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and content authors use this skill to convert Markdown files or piped Markdown text into responsive, self-contained HTML documents for documentation, reports, newsletters, and email templates. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated HTML from untrusted Markdown can run unsafe browser content when opened, published, emailed, or processed downstream. <br>
Mitigation: Use trusted Markdown sources only, or sanitize generated HTML before browser use, publication, email distribution, or handling user-submitted content. <br>
