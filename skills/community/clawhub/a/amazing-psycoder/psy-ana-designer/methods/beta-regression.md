# Beta 回归

## 概述

Beta回归用于建模取值在(0,1)区间内的连续比例数据,如准确率(非0/1试次)、注意力分配比例、资源分配比例。

**典型场景**: 比较条件间注意力分配比例; 建模准确率(每被试每条件一个比例值,远离0和1)。

## 何时使用

| 条件 | 要求 |
|------|------|
| 设计类型 | 被试间或被试内设计均可; 因变量为连续比例数据 |
| 因变量类型 | 连续比例值, 取值范围严格在 (0, 1) 开区间内 |
| 最小样本量 | 每组(条件)至少 20-30 个观测点; 参数数量不宜超过 N/10 |
| 因变量分布 | 比例数据, 无堆积在 0 或 1 边界; 可接受异方差 (边界方差小, 中心方差大) |
| 关键假设 | 观测独立性; 正确指定链接函数 (默认 logit); 无零/一膨胀 (若存在需改用 ZOIB 模型); 模型在子样本上系数稳定 |
| 典型数据形态 | 试次已按被试/条件聚合; 每个被试在每个条件下仅提供一个比例值 |

## 何时用 vs glmer

| 方法 | 数据 | 何时用 |
|------|------|--------|
| Beta回归 | 比例(0,1), 每被试每条件一个值 | 试次已均值化,比例在0.1-0.9 |
| glmer(binomial) | 0/1试次, 每行一个试次 | 有原始试次数据 |

## R代码

```r
library(betareg)
model <- betareg(accuracy ~ condition, data=data_agg)
summary(model)
```

## 好处

- 自然处理比例数据的异方差(接近0或1时方差小,0.5时方差大)
- 比arcsine转换更直观(估计量在原始比例尺度)

## 报告格式 (APA 7th)

**示例**:

> 以条件(一致 vs. 不一致)为自变量, 注意力分配比例(0-1)为因变量, 采用 Beta 回归(logit 链接函数)进行分析。结果显示, 在一致条件下注意力分配比例显著高于不一致条件, *b* = 0.42, *SE* = 0.15, *z* = 2.80, *p* = .005。模型伪决定系数 (pseudo *R*^2) 为 .18, 表明条件解释了注意力分配比例 18% 的变异。

**关键报告要素**: (1) 所用链接函数 (通常为 logit); (2) 回归系数 *b*、标准误 *SE*、*z* 值及 *p* 值; (3) 效应量指标 (如 pseudo *R*^2 或 odds ratio); (4) 若对边界值做过转换 (如 (y*(n-1)+0.5)/n), 需在方法部分披露。
