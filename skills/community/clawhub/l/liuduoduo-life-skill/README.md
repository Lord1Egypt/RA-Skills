# liuduoduo

Local life service skill: find handyman, find restaurant, find hotel. Ask and get results with phone numbers.

## Features

- **Find Handyman**: plumbing, electrician, locksmith, moving, appliance repair, AC cleaning, waterproofing
- **Find Restaurant**: local cuisine, farmhouse dining, group dinner recommendations (filter by budget, party size, scenery)
- **Find Hotel**: filter by price, location, breakfast included

## Usage

Install this Skill in CodeBuddy / WorkBuddy, then ask AI:

- "My drain is clogged, find me a plumber"
- "6 people, 500 yuan budget, where to eat? Want nice scenery"
- "Recommend a hotel near Sky Theater, 400/night, with breakfast"

AI will call tools automatically and return merchant name, phone, price, address.

## Tech Stack

- Framework: FastAPI + Uvicorn
- Protocol: MCP (JSON-RPC 2.0)
