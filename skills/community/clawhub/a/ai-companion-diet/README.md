# AI智能陪伴减肥助手

> 基于热量差=目标步数的核心理念，通过定时提醒、饮食打卡、步数督促、数据分析、激励机制陪伴用户健康减肥。

## 🌟 功能特点

### 🧮 智能热量计算
- **BMR基础代谢率计算**：基于性别、年龄、身高、体重科学计算
- **TDEE总能量消耗**：结合活动水平精准估算每日消耗
- **目标步数换算**：自动计算达成热量差目标所需步数
- **食物热量估算**：内置常见食物热量表，快速估算摄入

### ⏰ 定时提醒机制
- 🌅 早餐提醒 (7:30-8:30)
- ☀️ 午餐提醒 (11:30-12:30)  
- 🌙 晚餐提醒 (17:30-18:30)
- 🏃 步数督促 (10:00, 15:00, 20:00)
- 💤 睡前总结 (22:00)
- 💧 饮水提醒 (每2小时)

### 📝 饮食打卡系统
- ✅ 支持早、午、晚三餐记录
- 🍪 加餐单独统计
- 📊 自动累加每日总热量
- 🎯 实时进度反馈

### 📈 体重管理分析
- 📉 体重趋势图表展示
- 🔍 平台期智能识别分析
- 🎯 BMI计算与健康分类
- 🔮 减重趋势预测

### 💪 激励机制
- 🎉 每日达标正向反馈
- 🏆 周度总结进步表彰
- 🔥 连续打卡奖励机制
- 💬 个性化鼓励话语

## 🚀 快速开始

### 1. 初始化用户数据
```bash
init-user-data
```
按照提示输入：性别、年龄、身高、初始体重、目标体重、活动水平

### 2. 开始今日计划
```bash
start-daily-plan
```
系统会根据你的基础数据生成今日目标

### 3. 日常记录
```bash
# 记录饮食
record-breakfast "牛奶+面包" 200
record-lunch "米饭+鸡胸肉+青菜" 400
record-dinner "粥+小菜" 250
record-snack "苹果" 150

# 记录运动
record-steps 8000

# 记录体重
record-weight 68.5
```

### 4. 查看总结
```bash
# 今日总结
daily-summary

# 体重趋势
weight-trend

# 平台期分析
plateau-analysis

# 减重预测
predict-weight-loss
```

## ⚙️ 8+16断食模式

支持8+16轻断食模式：
- **进食窗口**：12:00 - 20:00（8小时）
- **禁食窗口**：20:00 - 次日12:00（16小时）

> 💡 建议：循序渐进，从12+12开始，逐步过渡到8+16。身体适应期间如有不适，请及时恢复正常饮食。

## 📊 健康标准

### BMI参考标准
| BMI范围 | 分类 | 建议 |
|---------|------|------|
| < 18.5 | 偏瘦 | 适当增加营养摄入 |
| 18.5 - 24 | 正常 | 继续保持健康生活 |
| 24 - 28 | 超重 | 建议开始控制饮食+运动 |
| ≥ 28 | 肥胖 | 建议在专业指导下减重 |

### 健康减重速度
- ✅ **推荐**：每周减重 0.5-1 kg
- ⚠️ **注意**：过快减重可能影响健康
- 💡 **建议**：设置合理的目标，循序渐进

## 🔧 技术实现

本技能使用纯Bash脚本开发，配合Python进行JSON数据处理，无需额外依赖，可在任何Linux/Unix环境运行。

核心组件：
- `calculator.sh` - 热量计算引擎
- `recorder.sh` - 数据记录模块
- `analyzer.sh` - 数据分析模块
- `notifier.sh` - 提醒通知模块
