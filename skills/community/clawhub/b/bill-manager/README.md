# Bill Manager 账单管理助手

帮你管理日常账单、追踪开支、设置缴费提醒——把零散的账单整理成可掌控的家庭财务记录。

## What it does
- 记录和分类日常开支（食品、住房、交通等）
- 生成月度/年度消费分析报告
- 设置水电煤气、房租、信用卡等周期性缴费提醒
- 订阅服务审计和预算优化建议

## Example scenarios

**1. 记账和分类**
> 👤 I spent ¥50 on lunch today and ¥200 on groceries.
> 🤖 已记录：午餐 ¥50（餐饮-外食）、食品杂货 ¥200（餐饮-超市）。今日餐饮合计 ¥250。需要继续记录其他开销吗？也可以直接说"记账 + 金额"，我来帮你分类。

**2. 月度消费分析**
> 👤 Show me my spending breakdown for this month.
> 🤖 本月消费结构——固定支出占比 55%（房租 ¥5000、水电 ¥320、话费 ¥99），变动支出占比 45%（餐饮 ¥2800、交通 ¥400、购物 ¥1200、娱乐 ¥600）。相比上月总支出下降 8%，但外卖占比上升了 15%，这是个可以关注的优化点。

**3. 家庭账单统一管理**
> 👤 家里水电气网费物业费每个月谁想起谁交，经常忘交被罚款，有没有办法统一管？
> 🤖 统一管理方案：第一步，把家里所有账单汇总——水、电、燃气、宽带、物业、房租/房贷、车位费，每项的缴费周期和金额列清楚；第二步，能线上缴的优先用支付宝/微信的"生活缴费"统一入口，绑定自动扣款更好；第三步，在日历上给每项账单设置到期前 3 天提醒。推荐用鲨鱼记账或随手记统一跟踪。
