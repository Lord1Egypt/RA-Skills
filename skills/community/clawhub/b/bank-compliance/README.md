# Bank Compliance Review Assistant
# 银行合规审查助手

> *AI-powered bank compliance review — screen marketing materials, KYC/AML checks, and CBIRC regulatory compliance in one workflow.*

> AI驱动的银行合规审查工具——营销材料审核、KYC/AML筛查、CBIRC监管合规，一站式完成。

---

## Overview

This skill provides comprehensive compliance review workflows for bank compliance officers. It covers marketing material screening, customer due diligence (KYC), anti-money laundering (AML) alert investigation, and CBIRC regulatory compliance checks.

本工具为银行合规人员提供全面的合规审查工作流，涵盖营销材料审核、客户尽职调查(KYC)、反洗钱(AML)预警调查及CBIRC监管合规检查。

## Installation

```bash
clawhub install bank-compliance-review
```

## Core Capabilities

| Capability | Description |
|------------|-------------|
| Marketing Material Review | Screen promotional content for CBIRC compliance |
| KYC Due Diligence | Customer identity verification, UBO analysis, risk scoring |
| AML Alert Investigation | Red flag indicator screening, suspicious transaction review |
| CBIRC Regulation Check | 资管新规/理财销售/消保/数据安全 checklist |
| Regulatory Reporting | Generate standard regulatory report drafts |

## Use Cases

- **Product marketing review** (产品宣传合规审查)
- **New customer onboarding KYC** (新客户开户KYC筛查)
- **AML alert investigation** (反洗钱预警核查)
- **Agreement compliance assessment** (合作协议合规审查)

## Target Users

- Bank compliance officers (合规管理员)
- Legal counsel (法律顾问)
- KYC/AML analysts (反洗钱专员)
- Product managers (产品经理)

## Compliance Framework

Aligned with CBIRC regulations: 资管新规、理财销售管理办法、消费者权益保护管理办法、反洗钱规定.
