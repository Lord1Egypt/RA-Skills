# PPT 配色方案参考

## 主题配色方案

### 1. 商务专业
- **主色**：深蓝 #1E40AF（稳重、专业）
- **辅色**：浅蓝 #3B82F6（现代感）
- **强调色**：金色 #F59E0B（成就、价值）
- **背景**：白色 #FFFFFF / 浅灰 #F3F4F6
- **适用**：企业汇报、商务提案、年度总结

### 2. 科技创新
- **主色**：深紫 #5B21B6（创新、前沿）
- **辅色**：青色 #06B6D4（科技感）
- **强调色**：亮绿 #10B981（活力）
- **背景**：深色 #0F172A / 渐变
- **适用**：产品发布、技术分享、创新项目

### 3. 教育培训
- **主色**：橙色 #EA580C（活力、热情）
- **辅色**：黄色 #FACC15（温暖）
- **强调色**：绿色 #16A34A（成长）
- **背景**：米白 #FFFBEB / 浅黄
- **适用**：培训课程、教学演示、知识分享

### 4. 自然环保
- **主色**：森林绿 #166534（自然、健康）
- **辅色**：天蓝 #0EA5E9（清新）
- **强调色**：土黄 #CA8A04（大地）
- **背景**：浅绿 #ECFDF5
- **适用**：环保主题、健康医疗、生态项目

### 5. 创意设计
- **主色**：玫红 #DB2777（大胆、个性）
- **辅色**：紫色 #9333EA（创意）
- **强调色**：黄色 #EAB308（跳跃）
- **背景**：深色 #18181B 或白色
- **适用**：设计提案、品牌展示、创意汇报

### 6. 医疗健康
- **主色**：医疗蓝 #0284C7（专业、信任）
- **辅色**：薄荷绿 #5EEAD4（健康）
- **强调色**：珊瑚红 #F87171（警示）
- **背景**：纯白 #FFFFFF
- **适用**：医疗报告、健康科普、医药展示

### 7. 金融财务
- **主色**：藏青 #1E3A5F（稳健、可靠）
- **辅色**：金色 #D4AF37（财富）
- **强调色**：红色 #DC2626（涨/警示）
- **背景**：浅灰 #F8FAFC
- **适用**：财务报表、投资分析、银行演示

### 8. 极简现代
- **主色**：黑色 #0A0A0A（经典）
- **辅色**：中灰 #525252
- **强调色**：任意亮色（按需选择）
- **背景**：纯白 #FFFFFF
- **适用**：极简风格、高端汇报、设计展示

## 配色心理学

| 颜色 | 心理暗示 | 适用场景 |
|------|----------|----------|
| 蓝色 | 信任、专业、冷静 | 商务、科技、医疗 |
| 绿色 | 成长、健康、自然 | 环保、教育、健康 |
| 红色 | 激情、紧迫、警示 | 营销、重要提示 |
| 黄色 | 活力、温暖、乐观 | 教育、创意、儿童 |
| 紫色 | 创新、高端、神秘 | 科技、设计、奢侈品 |
| 橙色 | 活力、友好、创意 | 营销、教育、创意 |
| 黑色 | 高端、权威、经典 | 奢侈品、高端商务 |
| 白色 | 简洁、纯净、现代 | 极简设计、科技 |

## 配色对比度建议

### 文字与背景对比
- **深色文字 + 浅色背景**：对比度 ≥ 7:1（最佳可读性）
- **浅色文字 + 深色背景**：对比度 ≥ 4.5:1（演示环境）

## 配色禁忌

1. **避免过多颜色**：单页PPT不超过3-4种颜色
2. **避免高饱和度组合**：如红配绿、蓝配橙（易视觉疲劳）
3. **避免低对比度**：如浅灰文字配白背景
4. **避免颜色含义冲突**：如用红色表示正向增长
5. **考虑色盲友好**：避免仅用颜色区分信息
