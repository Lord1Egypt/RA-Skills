# 政府信息公开申请书

## 适用场景
需从政府机关获取特定信息（政策文件、审批结果、规划信息、统计数据等），依法向制作或保存该信息的行政机关提出公开申请。

## 提交机构
制作或保存所需信息的行政机关（国务院部门、地方政府、各级机关等）

## 模板正文

```
                    政府信息公开申请书

申请人：姓名______，身份证号：__________________
        联系地址：__________________，联系电话：____________
        电子邮箱：__________________（便于电子答复）

申请公开的政府信息内容描述：
（请尽量详细、具体地描述所需的信息，便于行政机关查找）
1. ___________________________________________________；
2. ___________________________________________________；
3. ___________________________________________________。

申请公开政府信息的形式要求：
□ 纸质文本（复印件/摘录件）
□ 电子数据（电子邮件发送/光盘）
□ 查阅/摘抄
□ 其他：________________________

获取政府信息的方式：
□ 邮寄送达（邮寄地址：________________________）
□ 电子邮件发送（邮箱：________________________）
□ 自行领取
□ 当场查阅

用途说明（可选）：
本申请信息将用于______（如：学术研究/维护自身权益/了解公共政策等）。

法律依据：
根据《中华人民共和国政府信息公开条例》（2019年修订）
第27条、第29条之规定，公民、法人或者其他组织可以向行政机关
申请获取相关政府信息。

请贵机关在收到本申请之日起20个工作日内依法予以答复。

此致
____________（行政机关名称）

附：
1. 申请人身份证明复印件；
2. ______。

                                    申请人（签名）：
                                    ______年______月______日
```

## 注意事项
- **关键填写点**：信息描述越具体越好（文件名称、文号、时间段、关键词），避免笼统描述；明确选择获取方式（邮寄需写清地址）；用途说明有助于审批但不是法定必需的
- **常见错误**：申请"国家秘密"类信息（必然被拒）；信息描述过于宽泛（如"所有关于XX的文件"）；未提供身份证明复印件
- **份数要求**：正本1份即可，建议自留1份作为存证
- **答复时限**：行政机关应在收到申请之日起20个工作日内答复；如需延期，最长不超过20个工作日
- **收费**：原则上免费，但申请公开政府信息的数量、频次明显超过合理范围的，可收取信息处理费
- **救济途径**：对答复不服，可申请行政复议或提起行政诉讼
