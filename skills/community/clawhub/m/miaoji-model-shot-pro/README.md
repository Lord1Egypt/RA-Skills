# 秒技·模特拍摄Pro版

## 快速开始

### 基本使用

1. 打开技能
2. 输入产品和拍摄需求
3. 选择季节和时间
4. 获取完整拍摄计划

### 输入示例

```
产品名称：XXX户外背包
产品品类：户外运动
目标人群：户外爱好者
拍摄需求：全场景
同类产品ASIN：B09YYYYY, B07ZZZZZ
当前季节：秋季
拍摄预算：$3000-5000
```

## 功能概览

### 完整拍摄计划
获取：
- 场景→模特→道具→光线→角度全链路方案
- 模特引导指南
- 拍摄产出清单

### 季节性拍摄日历
获取：
- 春夏秋冬拍摄建议
- 节日节点规划
- 提前量提醒

### 多场景组合方案
获取：
- 基础/标准/完整套组建议
- Amazon图片矩阵组合
- 社交媒体场景扩展

### 同类视觉反超
获取：
- TOP3同类产品视觉分析
- 弱点识别
- 差异化策略
