# douyin-scraper

🎬 抖音内容爬虫 - 支持自然语言直接搜索视频内容

## 特性

- ✅ **纯自然语言驱动** - 直接说"搜索一下海鲜视频"即可
- ✅ 基于浏览器自动化，稳定可靠
- ✅ 自动提取视频标题、作者、点赞数据
- ✅ 支持会话持久化，跳过重复登录
- ✅ 零代码配置，开箱即用

## 安装

```bash
# 1. 安装 skill（通过 clawhub）
clawhub install douyin-scraper

# 2. 安装依赖
npm install -g agent-browser
agent-browser install
```

## 使用方法

**就这么简单：**

```
搜索一下海鲜视频
帮我找美食探店视频
搜索搞笑段子
查找Python编程教程
找一找旅行vlog
```

不需要记住复杂的命令格式，想说什么就说什么！

## 支持的句式

| 用户输入 | 解析结果 |
|---------|---------|
| 搜索一下海鲜视频 | 关键词：海鲜 |
| 帮我搜美食探店 | 关键词：美食探店 |
| 查找搞笑段子 | 关键词：搞笑段子 |
| 找一找Python教程 | 关键词：Python教程 |
| 健身视频 | 关键词：健身 |

更多示例参见：`examples/search_requests.txt`

## 工作流程

```
用户自然语言输入
    ↓
意图识别 + 关键词提取
    ↓
打开抖音搜索页面
    ↓
自动输入关键词 + 搜索
    ↓
抓取并解析搜索结果
    ↓
格式化输出
```

## 高级配置

### 登录获取完整内容

抖音部分内容需要登录后查看。第一次使用时可以手动登录，然后保存状态：

```bash
agent-browser --session douyin state save douyin-auth.json
```

下次使用会自动加载认证状态。

### 滚动获取更多结果

默认抓取第一页结果。需要更多内容时可以说："往下翻页加载更多"

## 注意事项

1. 首次运行需要下载 Chromium，耗时约 30 秒
2. 建议配合代理使用，避免访问频率限制
3. 抓取内容仅供个人学习使用，请遵守抖音平台协议
