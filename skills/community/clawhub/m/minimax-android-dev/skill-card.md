## Known Risks and Mitigations: <br>
Risk: Suggested Gradle or adb commands could modify builds, install apps, or run code from an untrusted Android project. <br>
Mitigation: Review target build files and dependencies before execution, and run commands only in a trusted workspace. <br>
Risk: BuildConfig and API URL examples could lead developers to place real secrets in source-controlled Android configuration. <br>
Mitigation: Keep real secrets out of Gradle and BuildConfig examples; use secure, environment-specific configuration instead. <br>
