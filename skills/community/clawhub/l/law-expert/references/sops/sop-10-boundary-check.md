# SOP-10: 权益边界速查

## 适用场景
普通人遇到日常场景，不确定这事是否合法/是否侵权，需要快速判断。

## 触发关键词
「合法吗」「能这样吗」「这事犯法吗」「有没有违法」「XX可以吗」「XX必须吗」「XX合理吗」

## 执行步骤

### Step 1: 场景识别
匹配以下预制高频场景，或动态分析用户描述。

#### 预制场景库（30个，每个说明：合法边界在哪里 + 法律依据）

**劳动就业类：**

1. **加班没有加班费** → 合法边界：每日加班不超过3小时/每月不超过36小时，超出部分须支付加班费（《劳动法》第41条、第44条）
2. **试用期不交社保** → 合法边界：用工之日起30日内必须办理社保登记（《社会保险法》第58条）
3. **被无故辞退** → 合法边界：须有法定事由（严重违纪、不能胜任等），否则须支付经济补偿（《劳动合同法》第39条、第46条、第47条）
4. **竞业限制过宽** → 合法边界：仅限高级管理人员/高级技术人员/其他负有保密义务的人员，期限不超过2年，须按月支付经济补偿（《劳动合同法》第23条、第24条）
5. **公司不发年终奖** → 合法边界：如劳动合同或规章制度明确约定，则公司必须按约发放；如仅属福利性质，则公司有裁量权
6. **实习生无薪/低薪** → 合法边界：试用期工资不低于劳动合同约定工资的80%，且不低于当地最低工资标准（《劳动合同法》第20条）
7. **公司监控员工聊天** → 合法边界：须事先告知员工并获得同意，监控目的须合理（如信息安全），不能侵犯员工隐私（《个人信息保护法》第13条）
8. **调岗降薪未经同意** → 合法边界：变更劳动合同须协商一致并采用书面形式（《劳动合同法》第35条）；单方调岗须符合法定条件（如不能胜任工作、医疗期满等）

**消费/服务类：**

9. **网购不退货** → 合法边界：消费者有权自收到商品之日起7日内无理由退货（特殊商品如定制、鲜活易腐、数字化商品等除外）（《消费者权益保护法》第25条）
10. **假货** → 合法边界：经营者提供商品有欺诈行为的，消费者可要求退一赔三，赔偿金额不足500元的为500元（《消费者权益保护法》第55条）
11. **健身房/美发店跑路** → 合法边界：经营者以预收款方式提供商品/服务，未按约定提供的，应退预付款并承担预付款利息（《消费者权益保护法》第53条）
12. **办卡后商家关门** → 合法边界：预付卡消费受《消费者权益保护法》保护，商家停业应提前30日公告并退还余额（《单用途商业预付卡管理办法》第21条）
13. **直播间买到假货** → 合法边界：网络直播营销平台/直播间运营者不能提供销售者真实信息的，消费者可要求其赔偿；明知或应知销售者侵权的，承担连带责任（《最高人民法院关于审理网络消费纠纷案件适用法律若干问题的规定》）
14. **快递丢失不赔** → 合法边界：未保价快递按实际损失赔偿，但最高不超过收取资费的倍数（通常3-5倍）；保价快递按声明价值赔偿（《快递暂行条例》第27条、《快递服务国家标准》）
15. **外卖吃出异物** → 合法边界：生产不符合食品安全标准的食品，消费者可要求价款10倍或损失3倍的赔偿金，增加赔偿金额不足1000元的为1000元（《食品安全法》第148条）

**住房/物业类：**

16. **房东不退押金** → 合法边界：承租人无违约行为、无房屋损毁时，出租人须全额退还押金（《民法典》第714条、第715条）
17. **房东随意涨租** → 合法边界：租赁期内出租人不得单方提高租金，须双方协商一致（《民法典》第703条）
18. **物业不作为** → 合法边界：物业服务人应按约定提供服务，否则承担违约责任（《民法典》第942条）
19. **邻居噪音扰民** → 合法边界：超过国家标准（居住区域昼间55分贝/夜间45分贝）构成噪声污染侵权（《噪声污染防治法》第31条、《民法典》第288条相邻关系规定）

**网络/数字类：**

20. **平台封号/禁言** → 合法边界：平台须有合理的社区规范作为依据，须告知用户具体违规理由，须提供申诉渠道（《网络信息内容生态治理规定》）
21. **照片/视频被盗用** → 合法边界：未经同意使用他人肖像构成侵犯肖像权；未经许可使用他人摄影/视听作品构成侵犯著作权（《民法典》第1019条、《著作权法》第52条）
22. **AI换脸/声音被盗用** → 合法边界：侵犯肖像权/声音权/个人信息权，可能构成民事侵权甚至刑事犯罪（《民法典》第1019条、第1023条、《个人信息保护法》第4条）
23. **被网暴/诽谤** → 合法边界：侵犯名誉权，情节严重的可自诉诽谤罪（《民法典》第1024条、《刑法》第246条）
24. **个人信息泄露** → 合法边界：处理个人信息须告知并取得同意；发生泄露须立即采取补救措施、通知个人并报告主管部门；造成损害的须赔偿（《个人信息保护法》第17条、第57条、第69条）

**借贷/金融类：**

25. **网贷高利息** → 合法边界：民间借贷利率超过合同成立时一年期LPR四倍的部分，人民法院不予支持（《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》第25条）
26. **暴力催收** → 合法边界：以威胁、恐吓、骚扰等方式催收债务构成违法，情节严重可构成催收非法债务罪/寻衅滋事罪（《刑法修正案（十一）》第34条）
27. **信用卡被盗刷** → 合法边界：发卡行对持卡人账户资金安全负有保障义务，因技术漏洞等导致盗刷的，银行应承担赔偿责任

**其他：**

28. **公司不发工资** → 合法边界：用人单位须按时足额支付工资，拖欠劳动报酬的应加付赔偿金（《劳动法》第50条、《劳动合同法》第85条）
29. **交通事故对方全责不赔** → 合法边界：被侵权人可要求对方保险公司在交强险责任限额内先行赔付，不足部分由商业三者险赔付（《道路交通安全法》第76条）
30. **遗嘱被藏匿/篡改** → 合法边界：藏匿/篡改/销毁遗嘱侵犯继承权，情节严重的丧失继承权（《民法典》第1125条）

### Step 2: 合法/不合法判断
- 对比用户场景与预制场景边界
- 对未匹配的场景，基于法律原则进行动态分析
- 输出：✅ **合法** / ❌ **不合法** / ⚠️ **视情况**（需明确判断条件）

### Step 3: 如果非法 → 启动A层权益感知器 → B层权利告知
若判断为不合法或视情况：
- **A层**：调用权益感知器，感知用户在此场景下拥有的各项权利（财产权、人身权、知情权、救济权等）
- **B层**：调用权利告知书，告知用户具体可主张的权利、赔偿范围、行动路径

### Step 4: 推荐下一步链路
根据具体场景推荐最适合的后续SOP链路：
- 涉及合同的 → SOP-01 合同审查
- 涉及劳动争议 → SOP-04 维权全流程
- 涉及消费纠纷 → SOP-04 维权全流程
- 涉及网络侵权 → SOP-07 知识产权与数字权益
- 涉及合同审核/文书起草 → SOP-05 法律文书起草
- 涉及税务问题 → SOP-02 税务速扫
- 无匹配链路则进入 → SOP-08 法律问答

[✅合法 / ❌不合法 / ⚠️视情况]

## 关键注意事项
- 本链路用于快速边界判断，不做深入法律分析
- 判断为非法或有风险时，必须自动衔接A层 → B层链路
- 预制场景库覆盖不了的情况，基于法律原则给出谨慎判断
- 所有判断应标注法律依据，不可凭主观臆断
- 诉讼时效提醒：一般民事纠纷为3年（《民法典》第188条），劳动争议仲裁时效为1年（《劳动争议调解仲裁法》第27条）
