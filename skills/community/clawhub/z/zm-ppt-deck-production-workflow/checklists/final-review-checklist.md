# 最终审核清单

## 内容审核

- [ ] 页数与内容大纲一致。
- [ ] 页面顺序一致。
- [ ] 主标题、副标题、核心结论不偏离。
- [ ] 每页只传递一个核心认知。
- [ ] 页面不是讲稿，没有大段堆字。

## 视觉审核

- [ ] 黑金风格统一。
- [ ] 无蓝紫霓虹、廉价粒子、俗套机器人/芯片滥用。
- [ ] 字号、留白、对齐符合规范。
- [ ] 同构页位置、裁切、编号、标签一致。
- [ ] 无锯齿、模糊、低清压缩痕迹。

## 素材审核

- [ ] Logo 使用真实资产。
- [ ] 产品/软件界面使用真实截图或授权素材。
- [ ] 人物/本人视频/二维码/数据没有被模型伪造。
- [ ] 占位符已移除或明确标记为非正式版。

## 后期元素审核

- [ ] Logo 左上角固定。
- [ ] Part 标签右上角固定。
- [ ] 页脚左下角固定。
- [ ] 页码右下角固定。
- [ ] 主视觉图中没有内置 Logo/页脚/页码/Part 标签。

## 文件审核

- [ ] PPTX 可打开。
- [ ] 渲染图完整。
- [ ] PDF/预览图已生成。
- [ ] 高画质版本和知识库安全版本用途清晰。
- [ ] 上传记录完整。
- [ ] 交付包结构完整。
