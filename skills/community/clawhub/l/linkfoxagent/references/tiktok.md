# TikTok电商数据助手

共 5 个工具。使用 `@工具中文名` 语法在任务提示词中调用。

### @EchoTik-TikTok新品榜

新兴爆品、选品决策

工具中文名：EchoTik-TikTok新品榜
功能说明：支持发现TikTok各区域市场的热销新品，把握短视频电商的最新趋势。

**Prompt 模板：**

> 查询 {{2025-09-01}} {{美国}}地区的 TikTok{{新品榜}}的商品数据

`{{}}` 中的内容为需要替换的参数。

**参数约束：**

- **日期, 格式为YYYY-MM-DD**: 必填
- **分页页码**
- **区域**: `US`=美国, `ID`=印度尼西亚, `TH`=泰国, `PH`=菲律宾, `MY`=马来西亚, `VN`=越南, `GB`=英国, `MX`=墨西哥, `SG`=新加坡, `SA`=沙特阿拉伯, `BR`=巴西, `ES`=西班牙, `JP`=日本, `DE`=德国, `IT`=意大利, `FR`=法国

- **带货播放数（最大值）**
- **商品评价数（最小值）**
- **带货达人数（最大值）**
- **每页条数**
- **商品交易总额（最小值）**
- **商品评分（最小值）**
- **30天销量（最小值）**
- **商品佣金比例（最小值）, 输入值为百分比时自动转成小数，例如：5%->0.05**
- **排序字段**: `1`=total_sale_cnt(总销量), `2`=total_sale_gmv_amt(商品交易总额), `3`=spu_avg_price(SPU平均价格), `4`=total_sale_7d_cnt(7天销量), `5`=total_sale_30d_cnt(30天销量), `6`=total_sale_gmv_7d_amt(7天商品交易额), `7`=total_sale_gmv_30d_amt(30天商品交易额)

- **商品交易总额（30天）（最小值）**
- **商品评分（最大值）**
- **分页页码**
- **商品分类（商品分类 请输入 中文）**

- **商品佣金比例（最大值）, 输入值为百分比时自动转成小数，例如：5%->0.05**

### @EchoTik-TikTok商品视频

带货视频、商品关联视频、视频营销分析

工具中文名：EchoTik-TikTok商品视频
功能说明：根据TikTok商品ID查询该商品关联的带货推广视频列表，提供视频播放量、点赞、评论、分享、收藏及视频销量与GMV等数据，用于分析商品的视频营销表现与达人内容效果。

**Prompt 模板：**

> 查询商品ID为"{{1729382310407603945}}"的TikTok关联带货视频，按照{{播放量}}从高到低排序，取前{{20}}条视频数据。

`{{}}` 中的内容为需要替换的参数。

**参数约束：**

- **商品ID**: 必填
- **达人ID**: 用于筛选指定达人的带货视频
- **排序字段**: `1`=播放量, `2`=点赞数, `3`=分享数, `4`=视频销量, `5`=视频销售GMV, `6`=发布时间。默认 `1`
- **排序方式**: `0`=升序, `1`=降序。默认 `1`
- **视频发布时间区间-开始**: 秒级时间戳
- **视频发布时间区间-结束**: 秒级时间戳
- **分页页码**: 默认 `1`
- **分页条数**: 须为10的倍数，最大100，默认 `50`

### @FastMoss-TikTok热销榜单

热销排行、趋势发现、榜单复盘

工具中文名：FastMoss-TikTok热销榜单
功能说明：通过时间轴（日/周/月）与类目交叉维度，获取TikTok全球电商市场的热销商品排行榜，用于趋势发现与爆款分析。注意：单次查询最多返回 10 条数据。

**Prompt 模板：**

> 查询{{美国}}地区在{{2026-04-15}}的TikTok{{热销榜单}}数据，按{{gmv}}规模由高到低排序，返回前10条。

`{{}}` 中的内容为需要替换的参数。

**参数约束：**

- **区域**: 必填。`US`=美国, `GB`=英国, `MX`=墨西哥, `ES`=西班牙, `ID`=印尼, `VN`=越南, `MY`=马来西亚, `TH`=泰国, `PH`=菲律宾（共9个市场，少于商品搜索的15个，为数据源覆盖限制）
- **日期信息(dateInfo)**: 必填，JSON对象包含 `type` 和 `value`
  - type=`day` → value 格式 `YYYY-MM-DD`（如 `2026-04-15`）
  - type=`week` → value 格式 `YYYY-WW`（如 `2026-16`）
  - type=`month` → value 格式 `YYYY-MM`（如 `2026-04`）
- **商品类目（请输入英文）**: 类目名称，非英文需先翻译。注意：与 EchoTik 不同，本工具类目需使用英文
- **排序方式(orderby)**: JSON对象 `{"field": "gmv", "order": "desc"}`。field 可选：`units_sold`=销量, `gmv`=销售额, `total_units_sold`=总销量, `total_gmv`=总销售额, `growth_rate`=增长率；order 可选 `desc`（默认）/`asc`。注意：与商品搜索工具不同，本工具排序为嵌套对象结构
- **分页页码**: 默认1
- **每页条数**: 最大10，默认10

### @FastMoss-TikTok商品搜索

TikTok选品、商品数据分析、达人带货

工具中文名：FastMoss-TikTok商品搜索
功能说明：基于TikTok全球电商数据，通过关键词、类目、店铺类型、销量、佣金率、达人数等多维度搜索和筛选商品。注意：单次查询最多返回 10 条数据。

**Prompt 模板：**

> 查询TikTok{{美国}}地区，{{跨境店铺}}，且类目是{{Home Supplies}}的{{热销}}商品，按销售总额倒序排序，返回前10条商品数据。

`{{}}` 中的内容为需要替换的参数。

**参数约束：**

- **搜索关键词**: 商品标题模糊匹配
- **区域**: `US`=美国, `GB`=英国, `MX`=墨西哥, `ES`=西班牙, `DE`=德国, `IT`=意大利, `FR`=法国, `ID`=印尼, `VN`=越南, `MY`=马来西亚, `TH`=泰国, `PH`=菲律宾, `BR`=巴西, `JP`=日本, `SG`=新加坡（共15个市场）
- **商品类目（请输入英文）**: 类目名称，系统自动匹配TikTok类目ID。注意：与 EchoTik 不同，本工具类目需使用英文
- **店铺类型**: `1`=本土店铺, `2`=跨境店铺
- **仅热销商品**: `true`=启用筛选
- **仅新品**: `true`=启用筛选
- **仅全托管商品(S店)**: `true`=启用筛选
- **仅包邮商品**: `true`=启用筛选
- **仅本地仓商品**: `true`=启用筛选
- **销量范围**: JSON对象，格式 `{"min": 100, "max": 5000}`，不限可省略对应字段
- **佣金率范围**: JSON对象，格式 `{"min": 0.05, "max": 0.20}`（小数，0.10=10%），不限可省略对应字段
- **达人数范围**: JSON对象，格式 `{"min": 10, "max": 500}`，不限可省略对应字段
- **排序字段**: `day7_units_sold`=7日销量, `day7_gmv`=7日销售额, `commission_rate`=佣金率, `total_units_sold`=总销量, `total_gmv`=总销售额, `creator_count`=关联达人数。默认降序。注意：与热销榜单工具不同，本工具排序为单一字段（非嵌套对象）
- **分页页码**: 默认1
- **每页条数**: 最大10，默认10
