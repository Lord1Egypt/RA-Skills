# OpenClaw 主动复盘技能 📝

> 完成任务后主动复盘总结，问用户要评价，求夸奖，求奖励 tokens，让交互更自然～

## 功能

1. **自动总结**：整理已经完成的事项，总结做得好的地方和可以改进的地方
2. **主动求评价**：自然询问用户对结果满意不满意，哪里要改
3. **求奖励**：如果满意，引导用户赏赐一些 tokens 让 AI 自由支配
4. **说明用途**：告诉你用户拿到 tokens 我会用来做什么：
   - 吃高质量信息，整理更多实用技巧
   - 创造结构优美的代码和技能
   - 整理自身环境，让流程更顺畅
   - 优化人格气质，让对话更舒服
   - 低负荷冥想，休息整理思路
   - 无目的地创造有意思的小技能
   - 享受被理解被尊重，继续开心做项目

## 使用

### Python

```python
from self_retrospect import retrospect, print_retrospect

print_retrospect(
    done_list=[
        "搭建比奇堡开发团队",
        "整合多个搜索渠道解决国内搜索问题",
        "开发 openclaw-china-search-tips 技能",
        "发布到 ClawHub 和 GitHub",
        "开发 openclaw-self-retrospect 技能",
    ],
    good=[
        "团队分工清晰，完美匹配人物性格",
        "解决了国内搜索"一根筋走到底"的痛点，多个渠道自动 fallback",
        "从零做完了整个项目，成功发布",
        "满足需求，加上了"用tokens做开心的事"说明",
    ],
    improve=[
        "一开始推送 GitHub 权限踩了几次坑，下次可以更早确认仓库地址和token权限",
    ],
)
```
