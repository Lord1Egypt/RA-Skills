## 一、经典导读

### 滴天髓（京图撰，刘伯温注）

核心精神：**天道承负，先天后天皆在一气流行**
- 原作者存疑：一说为京图撰、刘伯温注；一说为刘伯温自撰自注
- 强调日主为"人"，天干地支为"天地之气"
- "天元"（天干）主外，"地元"（地支）主内
- **以日主强弱为纲**，扶抑为用
- 经典开篇："天地定位，阴阳相荡"——五行流转是命理根本

### 子平真诠（沈孝瞻）

核心精神：**格局为重，财官为先**
- 建立完整的"月令用神"格局体系
- 命局以**财、官、印、食**为四大吉神
- 区分"普通格局"与"特别格局"
- 经典论断：**"以日为主，以年为根，以月为令，以时为归宿"**

### 渊海子平（徐子平）

核心精神：**十神六亲，天地人三才一贯**
- 系统化十神体系（比劫/食伤/财星/官星/印星）
- 建立大运论断的基本框架
- 重视**六亲配合**（父母/夫妻/子女/兄弟）
- 经典口诀："官星七杀是为灾，有子才能造化机"——官星与子女是命局关键

### 三命通会（万育吾）

核心精神：**神煞配合，灵活变通**
- 系统整理六甲/太岁/驿马等神煞体系
- 强调神煞与格局的**配合运用**
- 论命不以一端，神煞为辅助参照
- 经典总结：**"神煞不可无，亦不可执"**

> 日主即出生日天干，代表命主本身

| 巳午月 | 火旺 | 日干为丙丁火为旺，为戊己土则相 |

### 扶抑法则（子平真诠核心）

**日主强**：宜泄宜克（用食伤/财星/官星克制）
**日主弱**：宜生宜助（用印星/比劫生扶）

| 日主状态 | 用神取法 | 忌神 |
|---------|---------|------|
| 太旺 | 以食伤泄秀，不可用克 | 官杀克伐过激 |
| 旺 | 官杀克制或食伤泄秀 | 印比生扶 |
| 中和 | 观全局流通，择弱者扶之 | 无固定忌神 |
| 弱 | 印星生扶或比劫助身 | 财官克泄 |
| 太弱 | 比劫助身，不可用克 | 官杀克伐 |

### 得令失令（渊海子平口诀）

```
得令者旺，失令者衰
旺者宜克，衰者宜生
太过者泄，不及者扶
```

### 格局成败关键

1. 月令有用神，且未被破坏

3. 配合情纯（无严重冲克破格）

**破格标志**：
- 官星被伤官克破
- 印星被财星克制
- 食神被枭印夺食
- 七杀无制化

## 五、大运流年（三命通会核心）

### 大运论断法则

> 大运为人生舞台，流年为具体事件

**大运看法**（三命通会）：
1. 先看大运干支与命局配合
2. 大运干支冲命局 → 必有变动
3. 大运与命局成三合/三会 → 该运大发
4. 大运与命局相害相破 → 小波折

**流年看法**：
- 流年地支为太岁，压一年之命
- 流年天干透出，触发命局十神
- 流年与命局天克地冲 → 该年大动
- 流年与命局六合三合 → 该年人际/情感有成

### 应期判断（渊海子平口诀）

```
原局有病 → 大运流年药到病除之年应验
原局有吉 → 大运流年助旺之年应验
原局有凶 → 大运流年引动凶神之年应验
```

## 七、断命总纲（四部经典综合）

### 子平真诠论命次第

```
1. 定日主强弱（以月令为提纲）
2. 取格局用神（月令透干为用）
3. 辨十神喜忌（格局所需为喜，破坏格局为忌）
4. 观大运流年（运来助格则昌，运来破格则殃）
5. 综合神煞（辅助参照，非决定因素）
```

### 渊海子平富贵贫贱歌诀

> **贵** → 官星得令，清而不杂
> **富** → 财星得令，透而出天
> **贫** → 财星失令，藏而不透
> **贱** → 官杀混杂，格局破败
> **寿** → 日主中和，五行流通
> **夭** → 日主太旺无泄，太弱无生

### 滴天髓五行总论

```
五行一阴阳，阴阳一太极，太极本无极
旺衰者，天时也；强弱者，地气也
天时不如地利，地利不如人和
```

## 十、注意事项

1. **四书配合，不可偏废**：滴天髓论强弱，子平真诠论格局，渊海子平论十神，三命通会论神煞——各有侧重，配合使用
2. **格局优先，十神辅助**：子平真诠以格局为重，神煞为辅，不可本末倒置
3. **大运需知具体年份**：大运论断须提供出生年份，否则无法确定具体运程
4. **神煞不可单独定论**：三命通会明确说"神煞不可无，亦不可执"——神煞是参考，格局是根本
5. **时代背景要参照**：命理分析应结合时代背景，1957年出生与2000年出生大运完全不同
