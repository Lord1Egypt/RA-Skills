# 糖原贮积症Ia型诊断流程

## 何时使用
当患儿满足以下任一条件时启动本技能：
- 婴儿期出现空腹低血糖、显著肝大、生长迟缓
- 合并高尿酸血症、高乳酸血症、高脂血症或酮症酸中毒
- 临床初步怀疑为代谢性肝病，需排除糖原贮积症Ia型

> 注意：本技能仅适用于疑似Ia型糖原贮积症，不适用于其他类型（如III、IV、VI、IX型）或其他肝糖原累积病。

## 执行步骤

### 1. 识别典型临床表现
确认患儿是否存在以下特征：
- 生长缓慢
- 肝脏显著肿大
- 空腹血糖降低
- 高尿酸血症、高乳酸血症、高脂血症
- 酮症酸中毒

### 2. 进行肝脏病理形态学检查
- **光镜观察**：肝细胞弥漫肿胀，胞质空亮（着色浅），核居中，形似植物细胞
- **PAS染色**：肝细胞内见紫红色糖原颗粒，且对淀粉酶消化呈稳定反应（不被消化）
- **电镜检查**：胞质内糖原颗粒明显增多、大片堆积

> 警告：仅凭病理形态不足以确诊，必须结合功能或基因检测。

### 3. 获取确诊依据
选择以下至少一种方法进行确诊：
- **酶活性检测**：测定葡萄糖-6-磷酸酶（G-6-Pase）活性，若显著降低或缺失可支持诊断
- **基因检测**：确认G6PC基因存在致病性突变

### 4. 排除其他亚型
与其他肝脏为主型糖原贮积症（如III、IV、VI、IX型）进行鉴别，确保诊断特异性。

## 输出结果
若同时满足：
- 典型临床表现
- 特征性病理改变
- G-6-Pase酶活性缺失或G6PC基因致病突变

则确诊为糖原贮积症Ia型。
