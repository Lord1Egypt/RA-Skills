# 信息学奥赛知识点体系

## 知识点分类与权重

| 分类 | 主题 | 典型考点 | 建议难度 |
|------|------|----------|----------|
| basics | 计算机基础 | 二进制、存储单位、硬件组成 | 1 |
| hardware | 硬件设备 | 输入/输出设备、CPU、内存 | 1-2 |
| os | 操作系统 | Windows文件路径、文件管理、回收站 | 1-2 |
| python | Python编程 | 变量命名、运算符、输入输出、条件、循环 | 1-3 |
| binary | 进制转换 | 二进制↔十进制、位运算基础 | 2-3 |
| scratch | Scratch编程 | 广播、角色、事件、循环 | 1-2 |
| software | 应用软件 | Word/Excel/PPT基本操作 | 1 |
| network | 网络基础 | IP地址、互联网概念、浏览器 | 1-2 |
| security | 信息安全 | 密码、病毒、网络安全行为 | 1-2 |
| algorithm | 算法入门 | 排序思想、查找、流程图 | 2-3 |

## 题型与出题模板

### 选择题模板
- 概念辨析型："以下关于X的说法，正确的是？"
- 计算型："二进制数Y对应的十进制数是？"
- 代码追踪型："以下Python代码的输出结果是？"
- 应用场景型："要实现X功能，应使用哪个函数/操作？"

### 判断题模板
- 正向陈述："X是Y的核心功能。"
- 反向陷阱："X可以做Y。"（实际不能）
- 条件限定："在Z条件下，X等于Y。"（条件是否正确？）

### 编程填空题模板
- 代码填空：给出完整代码框架，关键位置留空
- 常考模式：
  - 累加/累乘（for循环）
  - 条件判断（if/elif/else）
  - 字符串操作
  - 列表遍历
  - 函数定义与调用

## 难度分级标准
- **难度1**：基础概念识记（占40%）
- **难度2**：理解与应用（占40%）
- **难度3**：分析与综合（占20%）

## 扩展学科建议
- 数学：计算题、几何证明、应用题
- 英语：语法选择、阅读理解、完形填空
- 科学：实验设计、现象解释、数据分析
