# 申请中流程指南

> 本指南涵盖专利/商标/软著申请流程、代理所选择标准、代理合同模板、申请材料清单。

---

## 第1页/共5页

## 一、专利申请三轨制详解

### 1.1 三种专利类型对比

| 类型 | 保护对象 | 审查周期 | 保护期限 | 官费(费减前) | 官费(费减85%后) |
|------|---------|---------|---------|-------------|----------------|
| **发明专利** | 产品或方法 | 2-3年 | 20年 | 3450元(实审) | 560元 |
| **实用新型** | 产品形状/结构 | 6-8月 | 10年 | 500元 | 75元 |
| **外观设计** | 产品外观 | 6-8月 | 15年 | 500元 | 75元 |

### 1.2 发明专利申请流程

```
时间节点          关键动作                    注意事项
─────────────────────────────────────────────────────────────────
Day 1            提交申请                     准备好全部材料
Day 1~7          受理通知书                   核对申请号和费减证明
Day 1~3月        初审阶段                     形式问题补正在此阶段处理
Day 18月         早期公布(可请求提前公开)     提前公开可缩短实审等待
Day 18~36月      实质审查等待期               可提PPH加快审查
Day 36~72月      实审阶段                     收到审查意见必须答复
收到授权通知     缴纳当年年费和登记费          2个月内完成，否则视为撤回
授权公告         领取证书，正式生效            可开始使用®标记
```

### 1.3 实用新型/外观设计流程

```
时间节点          关键动作                    注意事项
─────────────────────────────────────────────────────────────────
Day 1            提交申请                     材料要求相对简单
Day 1~7          受理通知书                   
Day 1~6月        初审阶段                     主要检查形式问题
                            ↓
                     授权/补正/驳回
                            ↓
收到授权通知     缴纳年费和登记费(2个月内)     
授权公告         领取证书，正式生效            
```

### 1.4 专利费用减缓政策

**减缴条件**（满足其一即可）：
- 个人：上年度月收入≤5000元
- 企业：上年度应纳税所得额≤100万元

**减缴比例**：
| 主体类型 | 减缴比例 |
|---------|---------|
| 个人 | 85% |
| 企业/事业单位 | 70%（单个申请人）/ 60%（两个及以上） |

## 二、商标申请流程详解

### 2.1 商标注册流程

```
时间节点          关键动作                    注意事项
─────────────────────────────────────────────────────────────────
Day 1            提交申请                     尼斯分类选对类别
Day 1~30天       受理通知书                   受理不代表注册成功
Day 1~4月        形式审查                    检查申请材料完整性
Day 4~7月        实质审查                    核心阶段，决定是否驳回
Day 4月          初审公告(3个月)              公告期内他人可提异议
Day 7月          注册公告                    获得注册证书
Day 7~8月        颁发证书                    正式获得商标专用权
```

### 2.2 尼斯分类概览（常用类别）

| 类别 | 内容 | 适用场景 |
|------|------|---------|
| 第1类 | 化学原料 | 化工原料、肥料 |
| 第9类 | 科学仪器 | 软件/APP/电子设备 |
| 第35类 | 广告销售 | 电商/零售/服务 |
| 第36类 | 金融物管 | 金融/保险/不动产 |
| 第38类 | 通讯服务 | 即时通讯/互联网服务 |
| 第41类 | 教育娱乐 | 培训/娱乐/体育 |
| 第42类 | 科技服务 | 软件开发/技术研究 |
| 第45类 | 社会服务 | 法律/安保/婚介 |

### 2.3 商标申请费用

## 三、代理所选择标准与避坑指南

### 3.1 专利代理机构选择checklist

| 评估项 | 权重 | 具体评估要点 |
|-------|-----|-------------|
| 资质证书 | ⭐⭐⭐ | 专利代理机构注册证（在有效期内） |
| 专业领域匹配 | ⭐⭐⭐ | 代理师是否有相关技术背景 |
| 代理师经验 | ⭐⭐⭐ | 执业年限、成功案例数量 |
| 客户评价 | ⭐⭐ | 通过公开渠道查看评价 |
| 流程透明度 | ⭐⭐ | 是否提供申请号追踪、节点通知 |
| 报价透明度 | ⭐⭐ | 官费+代理费分开报价，无隐藏费用 |
| 合同规范 | ⭐⭐ | 合同条款明确，退费机制清晰 |

### 3.2 商标/软著代理选择要点

**商标代理**：
- 必须具有商标代理机构备案
- 优先选择专注特定行业的代理
- 查看其代理商标的类别分布

**软著代办**：
- 无特殊资质要求，但需有代理委托书
- 重点考察：材料准备规范度、递交效率
- 加急服务是否靠谱（10/5/3工作日）

### 3.3 不良代理识别特征

| 危险信号 | 说明 | 风险等级 |
|---------|------|---------|
| 承诺100%通过 | 商标/专利审查有主观因素，不可能100% | ⛔极高 |
| 报价极低 | 发明专利代理费<2000元，可能低质量服务 | ⛔极高 |
| 不做技术沟通 | 直接报价，无视技术方案差异 | 🔴高 |
| 一次性收全款 | 不设退款机制，无质量保证 | 🔴高 |
| 频繁要求补材料 | 通过补正赚取额外费用 | 🟡中 |
| 无资质证明 | 无法提供代理资质 | ⛔极高 |
| 催促签约 | 制造紧迫感，不给考虑时间 | 🟡中 |

### 3.4 优质代理特征

| 特征 | 说明 |
|------|------|
| 先技术沟通 | 了解技术方案后才报价 |
| 风险明确告知 | 说明驳回风险和应对方案 |
| 流程可追溯 | 提供申请号、节点通知 |
| 修改有响应 | 及时响应审查意见 |
| 退费机制清晰 | 明确什么情况下退费 |

### 4.2 合同谈判要点

| 谈判要点 | 建议争取 | 底线 |
|---------|---------|------|
| 付款方式 | 分期付款（提交50%+授权50%） | 不超过三七开 |
| 修改次数 | ≥3次免费修改 | 至少2次 |
| 退费机制 | 明确退费比例 | 非甲方原因应退50%+ |
| 质量保证金 | 通过后再付尾款 | 尾款≤30% |
| 响应时间 | 工作日4小时回复 | 24小时内 |
| 优先服务 | 审查意见优先答复 | 不得拖延 |

### 4.3 "包授权"vs"按次收费"模式对比

| 模式 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| **包授权** | 风险转移给代理所 | 价格较高（通常是正常2-3倍） | 技术方案新颖、预算充足 |
| **按次收费** | 价格透明 | 风险自担 | 常规技术、有经验团队 |
| **混合模式** | 平衡风险和成本 | 条款复杂 | 建议采用此模式 |

## 五、申请材料清单与常见缺陷

### 5.1 发明专利申请材料

| 材料名称 | 要求 | 常见缺陷 |
|---------|------|---------|
| 请求书 | 标准表格，填写规范 | 申请人/发明人信息不一致 |
| 说明书 | 技术领域+背景+内容+实施例+附图说明 | 技术问题描述不清、实施例不足 |
| 权利要求书 | 独立+从属权利要求 | 保护范围过宽或过窄 |
| 摘要 | ≤300字，概括技术方案 | 与权利要求重复 |
| 摘要附图 | 从附图中选最能说明技术特征的图 | 图不清晰 |
| 费减备案证明 | 上一年度收入证明/纳税证明 | 未办理费减备案 |

### 5.2 商标注册申请材料

| 材料名称 | 要求 | 常见缺陷 |
|---------|------|---------|
| 商标注册申请书 | 标准表格，一类一申请书 | 类别选错/漏选 |
| 商标图样 | 指定颜色需彩色图样，尺寸5×5~10×10cm | 图样不清晰/尺寸不符 |
| 身份证明 | 个人身份证/企业营业执照副本 | 复印件不清晰/过期 |
| 代理委托书 | 签字/盖章原件 | 委托书过期或签字不一致 |
| 商品/服务清单 | 按尼斯分类规范填写 | 商品名称不规范 |

### 5.3 软著登记申请材料

| 材料名称 | 要求 | 常见缺陷 |
|---------|------|---------|
| 软件著作权登记申请表 | 在线填写，打印签字 | 信息填错/签字不一致 |
| 软件源代码 | ≥3000行，前后半部分各1500行 | 代码行数不足/格式混乱 |
| 软件说明书 | 功能介绍+技术特点 | 描述过于简单/与代码不符 |
| 身份证明 | 个人身份证/企业营业执照 | 复印件不清晰 |
| 版本说明 | 说明版本号 | 版本号与代码不一致 |

### 5.4 材料缺陷补救方案

| 缺陷类型 | 补救方案 | 时效 |
|---------|---------|------|
| 代码行数不足 | 补充完整代码或撰写详细设计文档说明 | 答复期内 |
| 功能描述模糊 | 重新撰写功能说明书 | 答复期内 |
| 附图不规范 | 按要求重新绘制 | 答复期内 |
| 申请人信息不一致 | 核对并提交变更申请 | 尽快 |
| 类别选错 | 撤回后重新申请或分案 | 初审公告前可撤回 |
