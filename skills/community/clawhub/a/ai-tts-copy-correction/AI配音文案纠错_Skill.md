# AI配音文案纠错 Skill

> 适用场景：中文AI配音（TTS）前的文案预处理
> 核心目标：消除AI味、避免读错、停顿自然、逻辑通顺
> 基于实践验证：音箱/投影仪/相机/工学椅/小冰箱/呼吸机/蛋白粉等多品类文案优化经验

---

## 一、数字读法（最易翻车）

AI对阿拉伯数字的转换极不稳定，必须人工干预。

### 1.1 量词前的数字
| 错误写法 | AI读法 | 正确写法 | 规则 |
|---------|--------|---------|------|
| 2磅 | 二磅（或英文two） | **两磅** | 量词前"2"必须写"两" |
| 2升 | 二升 | **两升** | 同上 |
| 2克 | 二克 | **两克** | 同上 |
| 2个 | 二个 | **两个** | 同上 |
| 2台 | 二台 | **两台** | 同上 |
| 2个月 | 二个月 | **两个月** | 同上 |
| 1个 | 一个/壹个 | **一个** | 1通常保留，但"一"更自然 |
| 3k | 三凯/三千 | **三千** | 价格缩写必须全写 |
| 1w | 一瓦/一万 | **一万** | 同上 |

### 1.2 年份与长数字
| 错误写法 | AI读法 | 正确写法 | 规则 |
|---------|--------|---------|------|
| 2025 | 二零二五/两千零二十五 | **二零二五**（年份）<br>**两千零二十五**（数量） | 年份可读二零二五，普通数量读两千零二十五 |
| 1000-2000 | 一千减两千 | **一千到两千** | 减号必须改为"到"或"至" |
| 3-5年 | 三减五年 | **三到五年** | 同上 |
| 5-6年 | 五减六年 | **五到六年** | 同上 |
| 5000+ | 五千加 | **五千以上** | 加号必须改写 |

### 1.3 小数与分数
| 错误写法 | 正确写法 |
|---------|---------|
| 1.2W | 1.2瓦 → **一点二瓦** |
| 0.3% | **百分之零点三**（或保留0.3%，AI通常能读） |
| 4.5W | **四点五瓦** |

## 二、单位与符号（必改项）

AI读字母/符号极易翻车，以下必须替换：

### 2.1 物理单位
| 符号 | AI易错读法 | 正确写法 |
|-----|-----------|---------|
| W | 达不溜/瓦 | **瓦** |
| g | 克（注意：AI可能读成"吉"） | **克** |
| kg | 千克/公斤 | **千克** 或 **公斤** |
| L | 字母el/升 | **升** |
| mAh | 毫安时 | **毫安时** |
| dB | 分贝 | **分贝** |
| Hz | 赫兹 | **赫兹** |
| mm | 毫米 | **毫米** |
| cm | 厘米 | **厘米** |
| nm | 纳米 | **纳米** |
| p（帧率） | 屁/拍 | **帧**（60p→60帧） |
| % | 百分之X | **百分之X**（小数字建议写中文，如0.3%→百分之零点三） |

### 2.2 价格与容量
| 符号 | 正确写法 |
|-----|---------|
| k（价格） | **千**（3k→三千） |
| w（价格） | **万**（1w→一万） |
| 元 | 数字后紧跟，无空格（800 元→800元） |

## 三、英文术语与缩写

### 3.1 必须改为中文
| 原文 | 正确写法 | 原因 |
|-----|---------|------|
| APP | **应用** | AI读A-P-P，极生硬 |
| AI | **人工智能** | 视平台而定，部分AI读"爱" |
| OS | **系统** | 读O-S |
| Hi-Res | **高解析度** | 读Hi-Res像英文 |
| BCAAs | **支链氨基酸** | 读B-C-A-A-s |
| 200目 | **200目筛网** | 防止读成"200眼睛" |
| 0反式 | **零反式** | 0读"零"更自然 |

### 3.2 可保留英文（品牌/认证名）
| 原文 | 处理方式 |
|-----|---------|
| CPAP/APAP/ST | 保留，读字母是行业标准 |
| NSF | 保留 |
| Informed-Choice | 保留 |
| ON（蛋白粉品牌） | 保留，读O-N |
| JBL/SONY/BOSE | 保留 |
| aptX HD | 保留或改为aptX高清 |
| MIUI TV | 保留 |

### 3.3 必须加拼音标注（多音字品牌）
| 品牌 | 正确标注 |
|-----|---------|
| 美的 | **美的（měi dì）** | 否则AI读成"美丽的"第三声 |
| 乐（yuè）视 | 标注拼音 |
| 长（cháng）安 | 标注拼音 |

## 四、空格清理（停顿异常元凶）

**原则：AI配音的停顿靠标点，不靠空格。**

### 4.1 必须删除的空格
- 数字 + 空格 + 中文：`800 元` → `800元`
- 中文 + 空格 + 中文：`它 是` → `它是`
- 英文 + 空格 + 中文：`USB 供电` → `USB供电`
- 中文 + 空格 + 英文：`采用 APP` → `采用APP`
- 标点 + 空格：`。 如果` → `。如果`

### 4.2 例外（可保留）
- 品牌型号内部：`小明 Q6 Neo`、`极米 Z6X Pro`（空格用于分隔品牌与型号，AI读时自然停顿）
- 但建议统一为无空格：`小明Q6 Neo`、`极米Z6X Pro`

## 五、标点与停顿优化

### 5.1 顿号 vs 逗号
- **参数/功能并列**：用顿号（、），AI读时短促不拖沓
  - 正确：`2.1声道、28瓦总功率、5英寸低音单元`
  - 错误：`2.1声道，28瓦总功率，5英寸低音单元`（停顿太长）
- **叙事转折**：用逗号（，），保留自然呼吸

### 5.2 停顿标记（如平台支持 `<#x#>`）
| 标记 | 时长 | 使用场景 |
|-----|------|---------|
| `<#0.3#>` | 0.3秒 | 句内呼吸点、并列项之间 |
| `<#0.5#>` | 0.5秒 | 产品名+价格报幕、转折词前 |
| `<#0.8#>` | 0.8秒 | 产品切换、段落小结 |
| `<#1.0#>` | 1.0秒 | 大价位段切换、开场/结尾 |

**语气词（laughs/sighs）**：投影仪等严肃导购内容禁用；生活vlog可酌情使用，但极度谨慎。

## 六、逻辑与事实检查清单

配音前必须过一遍：

- [ ] **参数矛盾**：同一产品前后参数是否一致（如功率、容量）
- [ ] **医学分类**：呼吸机单水平/双水平/ST是否混淆
- [ ] **价格梯度**：是否递增，无倒挂
- [ ] **功能定位**：是否自相矛盾（如"海绵坐垫"又说"网布松弛"）
- [ ] **错别字**："页面尺寸"→"体积"、"的却"→"的确"、"做为"→"作为"
- [ ] **重复描述**：同一产品是否出现两次且描述冲突
- [ ] **绝对化表述**："90%普通人"等是否过度

## 七、快速处理流程

```
1. 提取原文 → 2. 数字转中文（重点2→两，年份/区间改写）
         → 3. 单位替换（W/g/L/dB/Hz等）
         → 4. 英文术语替换（APP/AI/OS等）
         → 5. 空格清理（数字+中文/中文+中文）
         → 6. 顿号优化（参数并列处逗号改顿号）
         → 7. 多音字标注（美的měi dì等）
         → 8. 逻辑检查（参数/医学/价格）
         → 9. 输出终稿
```

## 八、各品类特殊注意点

| 品类 | 特殊易错点 |
|-----|-----------|
| **音箱** | W→瓦、Hz→赫兹、dB→分贝、声道（2.0/2.1）、蓝牙版本（5.0/5.3） |
| **投影仪** | ANSI流明→流明（去掉ANSI）、APP→应用、AI→人工智能、6向→六向 |
| **相机** | g→克、mm→毫米、ISO（保留）、fps→帧每秒 |
| **工学椅** | W型→波浪型、3D→三维、kg→公斤、1000-2000→一千到两千 |
| **小冰箱** | L→升、直冷/风冷/风冷无霜、双温/双门/两门区分 |
| **呼吸机** | CPAP/APAP/ST保留、延迟/延时升压统一、单水平/双水平医学区分正确 |
| **蛋白粉** | g→克、%→百分之X、BCAAs→支链氨基酸、200目→200目筛网 |

## 九、常见翻车案例

| 原文 | AI实际读法 | 修正后 |
|-----|-----------|-------|
| 7W的功率 | 七达不溜的功率 | **七瓦的功率** |
| 3k出头 | 三凯出头 | **三千出头** |
| 美的冰箱 | 美丽的冰箱（第三声） | **美的（měi dì）冰箱** |
| 60p | 六十屁 | **六十帧** |
| 3000ANSI流明 | 三千A-N-S-I流明 | **三千流明** |
| 5 磅 | 五（停顿）磅 | **五磅** |
| 2磅装 | 二磅装 | **两磅装** |
| 2025中国职业赛 | 二零二五（或读错） | **二零二五中国职业赛**（年份） |
| 0.3% | 零点三百分之 | **百分之零点三** |
| APP看报告 | A-P-P看报告 | **应用看报告** |
