# 来源标注与脚注规范

## 触发标注的内容类型

凡出现以下内容，Markdown 中必须用 `[^N]` 脚注标注来源：

| 触发类型 | 示例 |
|---------|------|
| 明确数字/统计数据 | "市场规模达2,300亿元" |
| 增长率/预测数据 | "CAGR预计达18.5%" |
| 市场份额 | "XX占据35%市场份额" |
| 排名/榜单 | "位列行业前三" |
| 政策法规引用 | "根据XX政策规定" |
| 权威机构结论 | "麦肯锡报告指出" |
| 证据性语气 | "研究表明"、"数据显示"、"据报道" |

## 无来源数据处理

- Web Search 未返回可靠来源 → 必须标注 `[推断]`
- 报告开头加入免责声明：`⚠️ 标注 [推断] 的数据为 AI 基于行业背景知识生成，未经第三方数据源验证，使用前请自行核实。`

## 参考文献规则

- Markdown 脚注 `[^N]` 与报告末尾参考文献列表一一对应
- 有多少数据点就列多少条，不设上限也不设最低数量
- 可信度分级：🟢 高（政府/权威咨询/财报）· 🟡 中（行业媒体/协会）· 🔴 低/推断

## 参考文献格式

```
[^N]: **{{机构名}}**．《{{报告/文章标题}}》．{{发布日期}}．{{URL}} 🟢 高可信度
```

## 高质量链接来源优先级

搜索时优先从以下来源获取延伸阅读文章：

1. 行业协会官网（中国XX协会、国际XX联合会）
2. 权威咨询报告（麦肯锡、BCG、艾瑞咨询、IDC、Gartner）
3. 学术数据库（知网、万方、ResearchGate、SSRN）
4. 政府/监管机构（工信部、发改委、SEC、欧盟报告）
5. 财经媒体深度报道（36氪研究院、虎嗅、Harvard Business Review、FT）
