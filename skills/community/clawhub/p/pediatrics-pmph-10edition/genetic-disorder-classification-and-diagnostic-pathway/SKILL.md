# 遗传性疾病分类与诊断路径

## 何时使用
当患儿出现以下任一表现时启动本技能：
- 先天性多发畸形
- 智力或运动发育迟缓
- 特殊面容或体征（如异常肤纹、身材比例失调）
- 阳性家族史提示遗传倾向
- 存在代谢异常气味（汗液或尿液）

**注意**：本技能不适用于获得性非遗传性疾病，且必须结合临床评估与实验室证据共同判断。

## 执行步骤

### 第一步：按五大类别对遗传病进行初步分类
1. **染色体病**：由染色体数目（如三体）或结构异常（缺失、易位）引起。常染色体病约占2/3，典型表现为“三联症”（畸形、智力落后、特殊肤纹）；性染色体病约占1/3，主要表现为性发育异常。
2. **单基因病**：由单个基因突变导致，按遗传方式分为：
   - 常染色体显性（如结节性硬化症）
   - 常染色体隐性（如苯丙酮尿症；近亲婚配风险升高）
   - X连锁显性（如抗维生素D佝偻病；女性症状较轻）
   - X连锁隐性（如血友病；男性发病，女性多为携带者）
   - Y连锁（父传子，如SRY突变致性反转）
3. **线粒体病**：由mtDNA突变引起，呈母系遗传，累及多系统（如肌病、脑病、耳聋），临床表现高度异质。
4. **基因组印记病**：同一染色体区域因亲源不同而表达差异，例如15q11-13缺失：
   - 父源缺失 → Prader-Willi综合征
   - 母源缺失或UBE3A突变 → Angelman综合征
5. **多基因病**：由多个微效基因与环境因素共同作用（如糖尿病、高血压），具有家族聚集性、性别/种族差异，群体患病率约为0.1%–1%。

### 第二步：执行四步诊断流程
1. **病史采集**：
   - 绘制三代家系图，明确遗传模式线索
   - 详细记录母亲妊娠史（如糖尿病、羊水异常）
   - 询问孕期暴露史（感染、药物、酒精等）
2. **体格检查**：
   - 评估特殊面容、身材比例（上/下部量）、指距、多指/并指
   - 检查皮肤毛发色素、掌纹、外生殖器发育
   - 注意肝脾肿大、心脏杂音、神经系统体征及异常气味
3. **选择实验室诊断技术**：
   - **染色体核型分析**：用于检出染色体数目及大片段结构异常
   - **FISH**：靶向检测已知微缺失/重复区域
   - **aCGH（分子核型）**：全基因组高分辨筛查拷贝数变异（CNV），但无法检出平衡易位
   - **DNA测序（含NGS）**：用于检出点突变、小片段插入/缺失及部分CNV
   - **生化/酶学检测**：如串联质谱（MS/MS）、气相色谱-质谱（GC/MS）用于代谢病筛查；细胞酶活性测定用于确诊
   - **辅助检查**：根据需要安排肌活检、脑电图、影像学等
4. **整合诊断思路**：
   - 以典型临床表型为核心起点
   - 结合家系分析推断可能的遗传模式
   - 若存在代谢或酶学异常，优先进行靶向基因检测
   - 最终通过基因检测确认致病突变，完成精准诊断

## 输出目标
明确遗传病的具体类型、致病机制及遗传模式，为后续治疗干预、预后评估和遗传咨询提供依据。
