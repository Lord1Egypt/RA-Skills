# /scan — 快速排雷

对指定A股公司执行快速排雷分析，5分钟内输出风险等级与预警信号清单。

## 执行步骤

1. 通过 MCP 拉取最近一期三表数据、核心财务指标、行业基准值
2. 按 `references/core/fraud-detection.md` 的 70+ 预警信号逐项扫描
3. 与同行业75分位/中位/25分位对标
4. 评定风险等级（低/中/高/极高）并给出回避/观望/可关注建议

## 输入格式

- 股票代码：如 `600519`、`000002`
- 公司名称：如 "贵州茅台"、"万科A"
- 或两者同时提供

## 输出要求

按 SKILL.md 中"工作流一"的输出格式模板输出，必须包含：
- 风险等级（带颜色标记）
- 触发预警信号表格（类别/信号/实际值/阈值/偏离）
- 行业对标摘要表
- 明确建议
