## Description: <br>
Self Learning helps an agent run a self-learning and iteration workflow for reflection, error prevention, progress tracking, and learning by analogy. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
External users and developers use this skill to guide an AI assistant through task reflection, learning progress tracking, recurring-error prevention, and generalized solution patterns across related problems. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill gives broad automatic and persistent self-reflection behavior that may activate unexpectedly or retain project context. <br>
Mitigation: Before using it on sensitive projects, confirm how to disable it, when it activates, whether it writes or remembers project context, and prefer explicit commands over casual control phrases. <br>
