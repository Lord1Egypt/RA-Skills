## Description: <br>
Search and discover OpenClaw skills from various sources when users need available skills, specific functionality, or new skills to install. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and agent users use this skill to find relevant OpenClaw skills, compare options, and prepare installation guidance for common tasks. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Install examples may lead users to add third-party skills globally or silently without reviewing the source. <br>
Mitigation: Review each third-party skill source, marketplace entry, and prerequisites before installation; prefer trusted publishers and isolated testing before production use. <br>
