# 各平台内容风格指南

本指南帮助针对不同平台优化内容风格。

## 微信公众号

### 风格特点
- 正式但不呆板
- 专业但不枯燥
- 有深度但不晦涩
- 允许长文，但要有阅读节奏

### 标题特点
- 数字标题："XXX个技巧/方法/秘密"
- 疑问标题："为什么XXX？"
- 痛点标题："XXX的10个误区"
- 金句标题："XXX，才是最好的XXX"

### 内容特点
- 开头要抓人，建立阅读动机
- 中间要有节奏感，段落不要太长
- 结尾要有行动召唤或引发思考
- 允许使用表情符号增加亲近感

### 典型结构
```
开篇：痛点/共鸣/故事
引出主题
正文：2-4个核心观点
结尾：总结/升华/行动召唤
```

### 禁止事项
- 不要过度标题党
- 不要传播未经证实的信息
- 不要太长（建议1500-3000字）
- 不要堆砌专业术语

## 小红书

### 风格特点
- 轻松活泼
- 贴近生活
- 带有个人特色
- 有温度有感情

### 标题特点
- 感叹句："绝了！""真的哭死！"
- 疑问句："真的假的？"
- 感叹+数字："XXX个坑，你踩了几个？"
- 种草式："都给我冲！""按头安利！"

### 内容特点
- 口语化
- 有表情符号
- 有个人经历/感受
- 有实用价值
- 鼓励互动（评论区见）

### 典型结构
```
开场：吸引眼球（美图/痛点/疑问）
正文：干货+感受
结尾：总结/种草/互动
```

### 禁止事项
- 不要太正式
- 不要纯干货没有温度
- 不要过度营销
- 不要内容与标题不符

## 微博

### 风格特点
- 简短精炼
- 金句频出
- 适合传播
- 可以蹭热点

### 标题特点（微博话题）
- 简洁有力
- 有争议性
- 引发讨论

### 内容特点
- 140字以内最佳
- 一条微博说一件事
- 要有观点/态度
- 可以蹭热点

### 典型结构
```
观点/金句 + 简短说明 + #话题#
```

### 禁止事项
- 不要太长
- 不要没有观点
- 不要传播谣言
- 不要过度营销

## 知乎

### 风格特点
- 深度分析
- 逻辑严谨
- 有理有据
- 专业但易懂

### 标题特点
- 专业型："XXX是什么体验/原理"
- 对比型："XXX和XXX有什么区别"
- 经验型："如何XXX"
- 盘点型："XXX的N个XXX"

### 内容特点
- 开篇表明观点
- 中间有理有据
- 可以引用数据/案例
- 结尾可以留讨论空间

### 典型结构
```
开篇：表明观点
论证：2-4个论据
结论：总结观点
```

### 禁止事项
- 不要没有观点
- 不要情绪化
- 不要传播未经证实的信息
- 不要太学术晦涩

## 抖音/视频号文案

### 风格特点
- 开头3秒抓人
- 节奏快
- 有钩子
- 容易传播

### 标题特点
- 悬念型："最后一件事太重要了"
- 数字型："XXX个XXX，第三个最关键"
- 冲突型："XXX居然XXX"
- 疑问型："你知道XXX吗？"

### 内容特点
- 开头要钩子
- 节奏要快
- 信息密度要高
- 结尾要有互动引导

### 典型结构
```
0-3秒：钩子
3-30秒：核心内容
30-60秒：互动引导
```

### 禁止事项
- 不要铺垫太久
- 不要信息太密
- 不要没有钩子
- 不要结尾太突然
