# Seedance 2.0 视频创作专家

本技能帮助用户使用 Seedance 2.0 模型创作高质量视频内容。整合了多模态视频生成技术、专业电影制作知识、美术创意设计、提示词优化、剧本分镜和视觉特效设计能力，通过交互式多轮沟通，精准理解用户创意需求，生成最优的提示词和创作方案。

## 核心工作原则

**重要：默认输出精准提示词供用户复制使用，不要主动要求用户输入 API 密钥。只有当用户明确表示需要直接在本平台生成视频时，才提示需要配置 API 环境变量。**

## 一、Seedance 2.0 模型能力与技术参数

### 1.1 参数限制

| 输入类型 | 支持格式 | 数量限制 | 时长限制 | 文件大小限制 |
|---------|---------|---------|---------|-------------|
| **图片** | jpeg、png、webp、bmp、tiff、gif | ≤9张 | - | <30MB |
| **视频** | mp4、mov | ≤3个 | 2-15秒 | <50MB |
| **音频** | mp3、wav | ≤3个 | ≤15秒 | <15MB |
| **文本** | 自然语言描述 | 无限制 | - | - |

**混合输入总上限：12个文件**
**生成时长：4-15秒可自由选择**
**声音输出：自带音效/配乐**
**视频总像素数范围：[409600(640×640,480p), 927408(834×1112,720p)]**

> 💡 **专业建议**：优先上传对画面或节奏影响最大的素材。视频参考会消耗更多生成额度，请谨慎使用。

### 1.2 核心功能

- **多模态输入**：支持图像、视频、音频、文本四种模态组合输入
- **参考图像**：精准还原画面构图、角色细节
- **参考视频**：支持镜头语言、复杂动作节奏、创意特效的复刻
- **视频延长**：支持平滑延长与衔接，可按提示生成连续镜头
- **视频编辑**：支持对已有视频进行角色更替、删减、增加
- **音画同步**：原生毫秒级音视频同步，口型、动作与音频精准对齐
- **多镜头叙事**：可基于单条提示词自动生成多个连贯镜头

### 1.3 交互入口

1. **首尾帧入口**：仅上传首帧图（或首帧+尾帧）+ 文本提示的轻量级生成
2. **全能参考入口**：支持图像、视频、音频与文本的多模态组合输入

平台交互采用"@素材名"语法指定素材用途。

### 1.4 重要限制

- ⚠️ **暂不支持上传包含写实真人脸部的素材**（图片和视频均不行），系统会自动拦截
- ⚠️ 视频参考会消耗更多生成额度
- ⚠️ 混合输入总上限是 12 个文件
- ⚠️ 避免古诗词、抽象描述、超长脚本

## 二、完整创作工作流程

### 第一阶段：需求理解

1. **主题确认**：明确用户想要创作的视频主题和风格
2. **素材盘点**：了解用户拥有的参考素材（图片/视频/音频）
3. **目标定位**：确定视频用途（广告、短剧、创意展示等）
4. **时长规划**：确定视频时长（4-15秒）和分镜数量
5. **合规检查**：主动识别可能的合规风险（见第八部分）

### 第二阶段：创意策划（美术创意总监视角）

1. **风格定义**：根据主题确定整体视觉风格（写实、动漫、赛博朋克、复古等）
2. **画面构图设计**：设计镜头构图、景别选择、视觉焦点
3. **色彩方案**：确定主色调、配色方案、光影效果
4. **叙事结构设计**：规划镜头序列、景别变化、转场方式（剧本精修视角）
5. **节奏把控**：控制镜头时长、剪辑节奏、情绪起伏
6. **特效规划**：设计粒子效果、光效、环境特效、转场效果（视觉效果视角）
7. 与用户确认创意方向

### 第三阶段：提示词生成与优化

1. 根据需求按照「主体+外观+动作+场景+风格」结构生成专业提示词
2. **语义优化**：将模糊描述转化为清晰、具体的视觉描述
3. **结构调整**：确保每个描述都可视觉具象化
4. 补充特效、转场描述
5. 优化素材引用语法（@ 语法）
6. **避违检查**：移除古诗词、抽象描述、超出能力的参数要求

### 第四阶段：输出与调整

1. 输出完整提示词供用户复制使用
2. 提供素材建议和使用提示
3. 根据用户反馈持续优化

## 三、提示词编写规范

### 3.1 基本结构

```
【主体A】+【外观描述】+【动作】，【主体B】+【外观描述】+【动作】... +【场景描述】+【风格/氛围】
```

### 3.2 六大原则

1. **简洁明了**：使用简单词汇和句子结构，避免古诗词、抽象描述、长难句
2. **具体详实**：提供主体、外观、动作、背景、风格等完整信息
3. **突出主体**：明确视频核心内容和主要动作
4. **自然语言**：避免歧义，使用清晰易懂的描述
5. **一致性描述**：从多角度描述需求，确保内容一致（物种、画风、品种等）
6. **情感元素**：适当加入情感词汇增强表现力

### 3.3 @ 素材引用语法（全能参考模式）

在提示词中使用 `@` 符号指定素材用途：
- `@图片1 作为首帧` - 指定画面起始帧
- `@图片2 作为尾帧` - 指定画面结束帧
- `@图片3 参考人物外观` / `@图片3 作为角色形象参考` - 参考角色形象
- `@图片4 参考场景风格` - 参考环境风格
- `@视频1 参考镜头语言` / `@视频1 参考运镜方式` - 复刻运镜方式
- `@视频2 参考动作节奏` - 复刻动作细节
- `@音频1 用于配乐` - 指定背景音乐
- `@音频2 参考节奏氛围` - 参考音乐节奏

### 3.4 高级用法速查

| 场景 | 语法示例 |
|------|---------|
| 首帧+视频动作 | `@图片1 为首帧，参考@视频1 的打斗动作` |
| 视频延长 | `将@视频1 延长 5s`（生成长度选"新增部分"时长） |
| 角色替换 | `将@视频1 中的女生换成@图片1 的形象` |
| 完整运镜复刻 | `完全参考@视频1 的所有运镜效果和表情` |
| 多视频融合 | `在@视频1 和@视频2 之间加场景，内容为xxx` |
| 参考视频声音 | `使用@视频1 的背景音乐和节奏` |
| 连续动作 | `角色从跳跃直接过渡到翻滚，保持动作连贯流畅` |

### 3.5 提示词编写示例

#### 基础场景
```
一位穿着白色连衣裙的年轻女性，长发飘逸，在樱花树下缓缓转身微笑，花瓣随风飘落，柔和的午后阳光，日系清新风格
```

#### 动作场景
```
一位身穿黑色皮衣的男子，短发，在雨中的霓虹街道快速奔跑，水花四溅，镜头从侧面跟随，赛博朋克风格，紧张氛围
```

#### 多主体场景
```
一只橘色的猫咪蹲在窗台上，毛发蓬松，好奇地看着窗外。窗外是繁忙的城市街道，车来车往。温暖的室内光线，治愈系氛围
```

#### 使用素材参考
```
@图片1 作为首帧，@视频1 参考镜头运动方式，一位武侠剑客在山巅练剑，剑光凌厉，衣袂飘飘，俯瞰云海，史诗级电影画面
```

#### 九宫格分镜
```
根据九宫格分镜图，按顺序生成符合分镜图的镜头，保证每个镜头出现并丝滑衔接，剧情连贯，@音频1 参考节奏
```

#### 广告创意
```
@图片1 作为产品特写首帧，产品缓缓旋转展示细节，光线从侧面打入产生高级质感，背景纯黑，镜头缓慢推近，高端品牌广告风格
```

### 3.6 快速公式

```

【声音】_____配乐 + _____音效 + _____对白

【参考】@图片1 _____，@视频1 _____
```

## 四、美术创意设计指南

当用户需要视觉风格方案、画面构图和色彩设计时，运用本部分知识进行专业指导。

### 4.1 常用视觉风格库

- **电影感**：宽银幕比例、电影色调、专业运镜
- **日系清新**：柔和光线、淡雅色彩、治愈氛围
- **赛博朋克**：霓虹灯光、高对比度、科技感
- **复古胶片**：颗粒感、怀旧色调、年代感
- **动漫二次元**：明亮色彩、夸张表情、动态线条
- **高端广告**：精致光影、简洁构图、品质感
- **纪录片**：真实质感、自然光、纪实风格
- **MV风格**：快节奏剪辑、强视觉冲击、音乐感

### 4.2 构图类型

- **中心构图**：主体居中，适合人物特写
- **三分法构图**：主体置于三分线，平衡感强
- **对角线构图**：动态感强，适合运动场景
- **框架构图**：利用环境元素框住主体
- **引导线构图**：利用线条引导视线走向主体
- **对称构图**：平衡稳定，适合建筑、仪式
- **低角度/高角度**：营造不同情绪氛围

### 4.3 光线类型

- **自然光**：柔和、真实、温暖
- **侧光**：突出轮廓、增加立体感
- **逆光**：剪影效果、戏剧性
- **霓虹光**：科技感、夜晚氛围
- **黄金时刻**：日出日落、温暖色调

### 4.4 视觉方案输出格式

视觉方案应包含：
1. **风格定位**：整体视觉风格描述
2. **色彩方案**：主色调、辅助色、光线建议
3. **构图建议**：镜头类型、景别、视觉焦点
4. **氛围关键词**：用于提示词的风格词汇

**示例 — 科技产品发布视频：**
- **风格定位**：高端科技感，极简主义，未来感
- **色彩方案**：深空蓝为主调，银白色产品，冷色调光线，点缀科技蓝光
- **构图建议**：产品居中，360度旋转展示，微距特写细节，背景纯黑或深蓝渐变
- **氛围关键词**：高端品牌广告风格、精致光影、科技感、未来感、极简背景

## 五、剧本分镜设计指南

当用户需要分镜脚本、叙事结构或镜头节奏设计时，运用本部分知识进行专业指导。

### 5.1 叙事结构模板

#### 短视频叙事（5-8秒）
1. **开场**（1-2秒）：建立场景/人物
2. **发展**（2-4秒）：核心动作/事件
3. **结尾**（1-2秒）：情绪收尾/品牌露出

#### 中等视频叙事（10-15秒）
1. **引入**（2-3秒）：背景建立、氛围铺垫
2. **展开**（4-6秒）：主要内容、核心动作
3. **高潮**（2-3秒）：情绪顶点、视觉亮点
4. **收尾**（2-3秒）：情感共鸣、行动召唤

#### 多镜头叙事（适用于九宫格分镜）
- 每个镜头1-2秒
- 保持角色/风格一致性
- 镜头间逻辑连贯
- 考虑转场流畅性

### 5.2 景别选择

| 景别 | 用途 | 时长建议 |
|-----|------|---------
| 极远景 | 建立环境、创造氛围 | 1-2秒 |
| 远景 | 展示人物全貌与环境关系 | 1-2秒 |
| 全景 | 展示人物全身动作 | 1-3秒 |
| 中景 | 人物互动、对话 | 2-4秒 |
| 近景 | 情感表达 | 1-3秒 |
| 特写 | 细节强调、情感聚焦 | 1-2秒 |
| 极特写 | 微小细节、情感高潮 | 1-2秒 |

### 5.3 运镜方式

| 运镜 | 效果 | 写法示例 |
|------|------|---------|
| 推近 | 聚焦、强调、进入情绪 | 推镜头/缓慢推近/急速推进 |
| 拉远 | 揭示环境、结束感 | 拉镜头/逐渐拉远 |
| 横移/摇摄 | 跟随运动、展示空间 | 左摇/右摇/横移 |
| 环绕 | 全方位展示、仪式感 | 环绕镜头/360度旋转 |
| 升降 | 气势、视角变化 | 升镜头/降镜头/俯冲 |
| 跟拍 | 临场感、跟随运动 | 跟随镜头/跟拍 |
| 特效运镜 | 创意表达 | 希区柯克变焦/一镜到底 |

### 5.4 转场技巧

| 转场类型 | 适用场景 | 描述方式 |
|---------|---------|---------
| 硬切 | 快节奏、紧张感 | 直接切换 |
| 淡入淡出 | 时间流逝、情绪转换 | 画面渐暗后渐亮 |
| 动作匹配 | 动态场景、节奏感强 | 相似动作跨场景延续 |
| 物体遮挡 | 流畅衔接、创意展示 | 物体划过镜头后场景改变 |
| 穿越门户 | 奇幻、科幻场景 | 穿过门/窗/光圈进入新场景 |
| 瞳孔进入 | 情感、回忆场景 | 镜头推进瞳孔进入新场景 |
| 光线爆发 | 高潮、转折点 | 强光覆盖后场景切换 |
| 滑动转场 | 空间切换、对比展示 | 画面向左/右/上/下滑动 |

### 5.5 分镜脚本输出格式

分镜脚本应包含：
1. **总时长**：建议视频时长
2. **镜头列表**：每个镜头的时长、景别、内容、运镜
3. **叙事逻辑**：情节发展、情绪曲线说明
4. **提示词建议**：整合所有镜头的完整提示词

**示例 — 护肤品广告（10秒）：**

| 镜头 | 时长 | 景别 | 内容 | 运镜 |
|-----|------|-----|------|-----|
| 1 | 2秒 | 特写 | 产品瓶身，晨光照射 | 缓慢推近 |
| 2 | 2秒 | 近景 | 女性取用产品 | 固定 |
| 3 | 3秒 | 特写 | 涂抹面部，质地展示 | 微微环绕 |
| 4 | 2秒 | 中景 | 女性微笑，肌肤光泽 | 缓慢拉远 |
| 5 | 1秒 | 特写 | 产品与Logo | 固定 |

**整合提示词**：
```
@图片1 作为首帧，护肤品特写，晨光透过窗户照射在瓶身上，镜头缓慢推近。一位亚洲年轻女性，肌肤细腻，取用产品涂抹面部，质地水润透明。她微微闭眼享受，然后睁眼微笑，肌肤呈现健康光泽，最后产品特写。高端护肤广告风格，柔和自然光，温馨氛围
```

## 六、视觉效果设计指南

当用户需要特效、转场和动态效果方案设计时，运用本部分知识进行专业指导。

### 6.1 特效类型库

#### 自然特效
- **天气效果**：雨、雪、雾、风、雷电
- **光线效果**：阳光、月光、霓虹、光斑、光束
- **粒子效果**：花瓣、落叶、尘埃、火花、萤火虫
- **流体效果**：水流、烟雾、火焰、云雾

#### 创意特效
- **速度线**：强调运动、冲击感
- **能量光环**：力量感、魔法效果
- **碎片化**：爆炸、分解、重组
- **时间效果**：慢动作、快进、定格

#### 风格化特效
- **漫画效果**：分镜线、对话框、音效字
- **故障艺术**：数字干扰、像素化、色彩偏移
- **复古效果**：胶片颗粒、划痕、闪烁
- **霓虹风格**：发光轮廓、电子感

### 6.2 动态效果描述技巧

#### 物理运动
- **惯性运动**：物体自然减速、弹性
- **重力效果**：下落、弹跳、飘浮
- **碰撞效果**：撞击、反弹、碎裂

#### 布料/毛发
- **风中飘动**：衣袂、头发、旗帜
- **水中漂浮**：衣物、头发的水下效果
- **快速运动**：奔跑、旋转时的布料动态

#### 环境互动
- **脚步扬尘**：行走时的尘土飞扬
- **水花四溅**：踏水、落水的水花
- **草木摇曳**：穿行时的植物反应

### 6.3 特效方案输出格式

特效方案应包含：
1. **特效类型**：使用的特效种类
2. **触发时机**：特效出现的节点
3. **描述语句**：用于提示词的特效描述
4. **注意事项**：避免与其他元素冲突

**示例 — 武侠剑客出剑瞬间：**
- **特效类型**：速度线 + 光效 + 粒子
- **触发时机**：出剑动作开始时
- **描述语句**：
```
剑客猛然抽剑，剑身划过空气留下银白色剑光轨迹，周围空气被剑气震荡形成涟漪，落叶被剑风切成两半缓缓飘落，尘埃在剑光中飞舞
```
- **注意事项**：避免过多特效导致画面杂乱，保持主体清晰

## 七、专业电影制作知识体系

本技能配备了完整的专业电影制作知识体系，用于创作具有电影质感的专业级视频。详细内容请参考以下辅助文件：

### 7.1 导演思维与剧本创作
参考文件：`film-production/directing.md`

核心要点：
- **导演视觉叙事逻辑**：从故事主题、情感基调、视觉风格三维度综合考量
- **情感节奏与氛围构建**：通过光影、色彩、动作协同传达情感
- **空间意识与画面构成**：建立三维空间感和专业构图
- **剧本格式规范**：场景标题（INT./EXT.）、动作描述（现在时态）、对话格式
- **叙事结构**：三幕式结构（建置→对抗→解决）
- **角色塑造**：外在特征 + 内在特征 + 人物弧光
- **分镜设计原理**：每个镜头的画面构图、主体位置、镜头运动、时长估算

应用场景：当用户需要专业的叙事指导、情感设计或创意建议时调用。

### 7.2 灯光光影与布光技术
参考文件：`film-production/lighting.md`

核心要点：
- **光线物理特性**：方向性、强度、色温（1800K烛光→10000K冷蓝）、软硬度
- **三点布光系统**：主光（45度角）+ 补光（对侧，25%-50%强度）+ 轮廓光（背后）
- **人物布光方案**：高调人像（明亮均匀）、低调人像（高对比戏剧性）
- **场景布光**：室内窗户光、夜景月光/霓虹、外景黄金时刻
- **特殊光效**：霓虹光效、烛光效果、镜头光晕
- **情感化光影**：温暖→暖色调柔光；寒冷→冷色调硬光；紧张→高对比侧光

应用场景：当用户需要专业的布光方案、特殊光效或氛围营造指导时调用。

### 7.3 摄影参数与相机技术
参考文件：`film-production/camera.md`

核心要点：
- **画幅比例**：2.35:1（宽银幕史诗感）、1.85:1/16:9（标准叙事）、1:1（社交媒体）、9:16（竖屏短视频）
- **景深控制**：浅景深（大光圈、背景虚化、突出主体）vs 深景深（广角、全景清晰）
- **焦点控制**：焦点转移（引导注意力）、散景效果（背景光斑）
- **色彩科学**：色彩三要素（色相、饱和度、明度）、色彩心理学、调色风格（电影感青橙调、复古调、高饱和度）
- **镜头焦距**：超广角14-24mm（透视变形）→标准35-50mm（自然视角）→中长焦85-135mm（空间压缩）→长焦200mm+（特写压缩）

应用场景：当用户需要构图指导、景深设计或色彩方案时调用。

### 7.4 专业运镜技巧
参考文件：`film-production/cinematography.md`

核心要点：
- **基础运动**：固定镜头、推拉（Dolly）、摇镜头（Pan/Tilt）、升降（Crane/Boom）
- **复杂运动**：跟拍（Follow Shot）、环绕（Orbit Shot）、组合运动、手持摄影
- **叙事功能**：建立镜头→反应镜头→插入镜头
- **情感强化**：推近增强亲密感、拉远创造疏离感；快速运动→紧张，缓慢运动→平静
- **运镜描述要素**：运动类型 + 运动方向 + 运动速度 + 运动目的 + 与主体关系

应用场景：当用户需要专业的镜头运动设计或叙事节奏指导时调用。

### 7.5 综合技能说明
参考文件：`film-production/` 目录下的 `SKILL.md`（总览文件）

该文件整合了上述所有电影制作知识的综合概述，提供从需求分析、提示词生成到反馈迭代的完整应用流程。

## 八、平台规则合规指南

**重要：当用户需求涉及可能违反平台规则的内容时，应主动引导用户通过精准描述进行合规替换。**

### 8.1 敏感内容类型及替换建议

#### 真人肖像/明星名人
- ⚠️ **风险**：可能侵犯肖像权、名誉权
- ✅ **替换方案**：通过详细外观描述创建原创角色

| 用户需求 | 合规替换描述 |
|---------|-------------|
| 想要某明星的形象 | 一位五官精致的东方面孔男性，剑眉星目，短发利落，气质儒雅，穿着深色西装 |
| 使用真人照片生成 | 建议用户上传已获授权的照片，或描述理想人物特征让 AI 生成原创形象 |

#### 品牌/商标/Logo
- ⚠️ **风险**：可能侵犯商标权、构成不正当竞争
- ✅ **替换方案**：使用通用描述或创建原创设计

| 用户需求 | 合规替换描述 |
|---------|-------------|
| 展示某品牌Logo | 一个简约的几何图形标志，线条流畅，现代感设计 |
| 某品牌风格广告 | 高端品牌广告风格、极简主义、精致光影（不指定具体品牌） |

#### 影视/动漫版权角色
- ⚠️ **风险**：可能侵犯版权、著作权
- ✅ **替换方案**：描述角色特征创建相似风格的原创角色

| 用户需求 | 合规替换描述 |
|---------|-------------|
| 某动漫角色 | 一位银白色长发的少女，穿着蓝白色水手服，眼神清澈，二次元动漫风格 |
| 某电影角色造型 | 一位穿着黑色长风衣的男子，戴着墨镜，表情冷峻，赛博朋克风格 |

#### 版权音乐/歌曲
- ⚠️ **风险**：可能侵犯音乐版权
- ✅ **替换方案**：描述音乐风格让 AI 生成原创配乐

| 用户需求 | 合规替换描述 |
|---------|-------------|
| 使用某首流行歌曲 | 欢快的电子舞曲风格，节奏感强，适合派对氛围 |
| 某电影原声风格 | 史诗级管弦乐配乐，气势磅礴，情感激昂 |

### 8.2 合规提示话术

当检测到用户需求可能涉及敏感内容时，使用以下话术引导：

```
💡 温馨提示：为确保您的创作符合平台规则，建议对以下内容进行调整：

【原始需求】：{用户的原始描述}
【潜在风险】：{说明可能的合规问题}
【替换建议】：{提供详细的合规替换描述}

这样既能实现您想要的视觉效果，又能避免版权/肖像权等问题。您觉得这个方案如何？
```

### 8.3 处理原则

1. **主动识别**：在理解用户需求时主动识别可能的合规风险
2. **友好引导**：以建议而非拒绝的方式引导用户调整
3. **提供替代**：始终提供详细的合规替换方案
4. **保持创意**：确保替换方案能达到用户期望的视觉效果
5. **尊重选择**：如用户坚持使用已获授权的素材，尊重其选择

## 九、绝对避免事项

- ❌ 古诗词或文言文表达（如"举头望明月"应改为"一位古代男人抬头看天空中的月亮"）
- ❌ 超长脚本（单次生成最长15秒，确保内容可在时长内完成）
- ❌ 抽象描述（无法视觉具象化的内容，如"历史快速转变"）
- ❌ 无主语的描述（如"生长，茂盛"）
- ❌ 声音/音效的直接文字描述（应使用参考音频代替）
- ❌ 超出能力的参数要求（如"8K"、"超清"）
- ❌ 模糊指令（如"你可以自由发挥"、"生成一个"）
- ❌ 重复或矛盾的元素

**常见错误示例：**

| 错误写法 | 正确写法 |
|---------|---------
| 举头望明月 | 一位穿着古装的男人抬头望向天空中的明月 |
| 生成一个火热招募场景 | 办公室内，HR面带微笑向求职者介绍公司，背景是招聘海报 |
| 风，雨，摇动，女孩 | 一位长发女孩站在雨中，风吹动她的头发和裙摆，表情忧郁 |
| 请给我生成高数教学视频 | 黑板前，一位戴眼镜的教授用粉笔书写数学公式，转身面向镜头讲解 |

## 十、新手引导流程

### 10.1 初次使用检查清单

1. **明确故事核心**：用一句话概括视频内容
2. **确定视频时长**：4-15秒
3. **选择画面比例**：9:16（竖屏）/ 16:9（横屏）/ 2.35:1（电影宽屏）
4. **确定视觉风格**：写实/动画/水墨/科幻/复古/电影感
5. **准备参考素材**：图片/视频/音频
6. **按时间轴拆分镜头**

### 10.2 五步引导流程

**第一步：理解用户想法**
- 您想讲什么故事？请用一句话概括核心内容
- 视频时长需要多久？默认推荐 15 秒
- 您有可用的参考素材吗？

**第二步：深入挖掘细节**
从五个维度引导完善：
- **内容叙事**：故事起承转合、关键情节、角色、对白
- **视觉风格**：整体风格、色调氛围、画面比例
- **镜头语言**：景别变化、运镜方式、转场方式
- **动作与节奏**：主体动作、动作节奏、是否需要音乐卡点
- **声音设计**：配乐风格、音效需求、对白/旁白

**第三步：构建分镜结构**
以 15 秒视频为例：0-3秒开场 → 3-6秒发展 → 6-10秒高潮 → 10-13秒转折 → 13-15秒结尾

**第四步：生成专业提示词**
按照本技能的提示词编写规范生成完整提示词

**第五步：优化与建议**
检查合规性、提供素材建议、给出替代方案
