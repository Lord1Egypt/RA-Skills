# GEMINI.md - Instructional Context

## Directory Overview
This directory serves as a centralized repository for exported **Gemini AI Chat Discussions**. The contents are organized as a collection of Markdown (`.md`) files, each representing a specific session or topic discussed with Gemini. 

The primary purpose of this directory is to preserve knowledge, research, and technical strategies generated during these interactions, ranging from financial trading bot development to legal document summaries and system architecture.

## Project Type: Non-Code / Knowledge Base
This is a documentation-centric project. It does not contain executable source code but holds significant intellectual property, technical requirements, and strategic plans.

## Key Files & Categories

### 1. Trading & Finance
*   **`Gemini-AI Trading Bot Development Discussion.md`**: Exploration of real-time web monitoring and live trade assistance.
*   **`Gemini-Crypto Trading Bot Development Request.md`**: Detailed requirements for a full-stack, autonomous trading system using Rust, C++, and Docker/Kubernetes.
*   **`Gemini-SOL_USDT Trade Analysis and Exit Strategy.md`**: Specific analysis of Solana (SOL) trading positions and liquidation risks.
*   **`Gemini-High-Leverage Short Trade Analysis.md`**: Evaluation of high-risk trading strategies.
*   **`Gemini-Solana Analysis_ Bullish or Bearish_.md`**: Market sentiment analysis for the Solana blockchain.
*   **`Gemini-BTCC Trading Interface Explained.md`**: Guide to utilizing the BTCC trading platform.

### 2. Software Engineering & DevOps
*   **`Gemini-Docker Deployment Pipeline Explained.md`**: Strategies for containerization and deployment workflows.
*   **`Gemini-System Performance Monitoring Script.md`**: Scripts and logic for monitoring server health and performance.
*   **`Gemini-Web Browser Command Execution Methods.md`**: Research into browser-based automation and execution.
*   **`Gemini-OpenClaw AI Assistant Research.md`**: Investigation into specialized AI assistant implementations.

### 3. Legal & Personal
*   **`Gemini-Legal Document Summary and Next Steps.md`**: Summarization of legal proceedings and action items.
*   **`Gemini-Child Support, Custody, and Enforcement.md`**: Discussions regarding family law and enforcement mechanisms.

### 4. Blockchain & Assets
*   **`Gemini-Unwrapping Wrapped SOL Funds.md`**: Technical steps for managing wrapped assets on Solana.
*   **`Gemini-On the iPhone, the BlockDAG X1 app...`**: Analysis of mobile mining and asset conversion (BlockDAG).

## Usage Guidelines
*   **Contextual Reference**: Use these files to understand the background of ongoing projects (e.g., the "Trading Bot" project).
*   **Instructional Continuity**: When continuing work on a topic mentioned here, refer to the corresponding `.md` file to maintain consistency with previous AI-generated advice and decisions.
*   **Searchability**: Use `grep` or similar tools to search across these files for specific keywords (e.g., "Kubernetes", "Rust", "Liquidation") to retrieve relevant historical context.

## Maintenance
New chat exports should follow the naming convention: `Gemini-[Topic Name].md`. Each file should ideally include the export date and the original chat link at the top for traceability.
