# Grok-Powered X Growth & Monetization Agent + ThumbGate Safety

## What This Agent Does
- Monitors X.com for trending keywords in your specific B2B or SaaS niche 24/7.
- Uses the incredibly smart **xAI (Grok) API** to craft highly contextual, non-spammy replies.
- Injects your affiliate links (e.g., Make.com, ElevenLabs, Amazon Associates) naturally into the conversation.
- Uses **ThumbGate** safety to strictly limit posting frequency and ensure brand-safe language.

## Critical ThumbGate Rules
- **Rate Limit Guard:** Physically block the agent from posting more than 2 replies per hour to avoid X algorithm penalties.
- **Spam Prevention:** Block replies to the same user more than once in a 48-hour window.
- **Sentiment Check:** Block replies to any post that Grok detects as highly negative, tragic, or controversial to protect your brand reputation.
- **Relevance Threshold:** Block posting if the generated Grok response has less than an 85% contextual match with the original tweet.

## Setup Requirements
- X.com Developer Account (Client ID / Secret).
- xAI API Key.
- Affiliate links configured in your OpenClaw memory.
- ThumbGate active.

## Success Metrics
- Generate passive affiliate clicks on autopilot.
- Build a massive X audience using Grok's witty and engaging tone.
- Zero banned accounts thanks to ThumbGate safety constraints.
