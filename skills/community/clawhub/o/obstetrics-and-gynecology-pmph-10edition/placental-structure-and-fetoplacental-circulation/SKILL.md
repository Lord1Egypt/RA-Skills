# 胎盘结构组成与胎儿-胎盘循环建立机制

## 适用场景
- 分析正常妊娠中胎盘形成过程
- 评估胎盘循环建立是否完成
- 解释子痫前期或FGR的病理生理基础
- 判断子宫螺旋动脉重塑异常的临床意义

## 执行步骤

1. **识别胎盘三部分构成**：
   - 胎儿部分：羊膜（无血管、神经、淋巴，表面具微绒毛）和叶状绒毛膜
   - 母体部分：底蜕膜（胎盘附着处的子宫内膜）

2. **追踪叶状绒毛膜发育三阶段**：
   - 初级绒毛：合体滋养细胞形成小梁，细胞滋养细胞伸入形成中心索
   - 次级绒毛：胚外中胚层长入中心索形成间质
   - 三级绒毛：受精后第15–17日，胚胎血管长入间质，形成绒毛内血管

3. **确认胎儿叶与小叶结构**：
   - 1个初级绒毛干及其分支 = 1个胎儿叶（共60–80个）
   - 1个次级绒毛干及其分支 = 1个胎儿小叶（约200个）

4. **建立胎儿-胎盘循环路径**：
   - 脐动/静脉分支进入绒毛干 → 毛细血管延伸至三级绒毛
   - 子宫螺旋动脉破裂，母血直接注入绒毛间隙（IVS）
   - 母胎物质交换发生在游离绒毛表面，通过胎盘屏障（绒毛毛细血管壁 + 间质 + 滋养细胞层），胎儿血与母血不直接相通

5. **评估子宫螺旋动脉重塑过程**：
   - 间质滋养细胞穿透蜕膜及子宫肌层1/3，聚集于螺旋动脉周围
   - 血管内滋养细胞逆行迁移取代内皮，将肌性狭窄血管转变为低阻力扩张血管
   - 妊娠早期形成栓子堵塞动脉末端，早孕期末栓子消失，循环正式建立
   - 正常妊娠重塑率达96%；重度子痫前期合并FGR时仅约10%

> **注意**：本技能不适用于非妊娠状态或胎盘发育异常（如完全性葡萄胎）。
