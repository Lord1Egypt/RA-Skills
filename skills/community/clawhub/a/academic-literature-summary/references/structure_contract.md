# 学术文献精读总结 —— 结构合约

## 参考PDF结构分析

### 来源文献特征
- **期刊**: Journal of Environmental Psychology (Elsevier)
- **类型**: 实证研究论文（在线实验）
- **语言**: 中文译文（混合中英文引用）
- **排版**: 双栏学术期刊格式

### 原文献章节结构映射

| 原文章节 | 映射到输出结构 |
|----------|---------------|
| Abstract / 摘要 | → 📚 综述（核心背景提炼） |
| 1. Introduction / 导言 | → 📚 综述（研究背景+文献脉络） |
| 1.1-1.5 子主题 | → 📚 综述（理论框架细分） |
| 1.6 研究问题 | → 🔬 研究假设 |
| 2. Methods / 方法 | → 🔍 各研究详情（参与者、程序、方法、材料） |
| 2.6 统计分析 | → 🔍 各研究详情（方法子项） |
| 3. Results / 结果 | → 🔍 各研究详情（结果子项） |
| 3.4 中介分析 | → 🗺️ 研究理论模型图 + 🔍 结果 |
| 4. Discussion / 讨论 | → 💬 总讨论 |
| 4.1-4.4 子主题 | → 💬 总讨论（研究结果+局限+启示+结论） |
| References / 参考文献 | → 引用保留在各节中，PDF输出时单独列出 |

### 引用格式特征

原文献采用APA风格引用，中文译文中的引用格式：
- `(Author, Year)` — 英文作者
- `(作者, 年份)` — 中文译名
- `Author et al., Year` — 三人以上
- `Author & Author, Year` — 两人

**输出时必须保留原始引用格式，不可转换或省略。**

## PDF/DOCX输出时的页面结构

当输出为PDF/DOCX格式时，页面结构须遵循以下组织：

### 首页结构
1. **期刊品牌横幅**（顶部）
   - Elsevier logo + ScienceDirect横幅 + 期刊名称 + 期刊主页URL
2. **文章标题**
3. **作者信息**（姓名、单位、通讯邮箱）
4. **文章元数据**（DOI、收稿日期、修回日期、接受日期）
5. **摘要**（双栏小字号）
6. **关键词**
7. **正文起始**（双栏）

### 正文页结构
1. **页眉**（running header）
   - 左：作者姓名
   - 右：期刊引用信息
2. **正文内容**（双栏）
3. **图表**（根据大小单栏、跨栏或整页）
4. **页脚**（页码居中）

### 参考文献页结构
1. **页眉**（与正文页一致）
2. **参考文献标题**（如 "References"）
3. **参考文献列表**（双栏，小字号，悬挂缩进）
4. **页脚**（页码居中）

## 内容提取优先级

### 高优先级（必须保留）
1. 研究问题/假设的精确表述
2. 样本量、人口学特征
3. 关键统计结果（B值、CI、p值、效应量）
4. 量表名称及来源
5. 核心变量定义
6. 所有引用

### 中优先级（精简保留）
1. 实验流程的时间线
2. 刺激材料描述
3. 统计方法名称
4. 子研究间的差异

### 低优先级（可删除）
1. 过渡性语句
2. 与主题关联较弱的背景描述
3. 重复的论证
4. 详细的软件版本信息

## 统计结果保留规范

- 保留原始效应量符号和方向（正/负）
- 保留置信区间边界
- 保留显著性标记（*, **, ***）
- 保留具体数值（如 B = 0.58, CI = 0.46, 0.69）

## 图表编号规范（PDF/DOCX输出）

- 图表使用连续编号：图1、图2、表1、表2...
- 图片标题位于图片下方，格式："图1。标题文字"
- 表格标题位于表格上方，格式："表2 标题文字"
- 文中引用图表时使用"见图1"、"（表2）"等格式
