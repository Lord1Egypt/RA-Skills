# C3 用户增长与活动运营 — 增长引擎

C2C本地生活服务平台增长引擎集群，覆盖用户获取、裂变拉新、补贴策略、活动运营、数据评估全链路。核心机制：三级裂变推荐体系（L1→L2→L3），平台抽佣1%分润至推荐池30%（L1:70% / L2:30% / L3:0%，严格三级封顶防范传销风险），虚拟货币"易点"贯穿激励与消费闭环。
