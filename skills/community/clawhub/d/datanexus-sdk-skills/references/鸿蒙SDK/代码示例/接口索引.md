# 鸿蒙 SDK 接口索引

## 初始化与启动接口

| 接口 | 说明 |
:|------|------|
| `dnSDK.init({ context, user_action_set_id, secret_key, user_unique_id?, show_log? })` | SDK 初始化（必填：`context`、`user_action_set_id`、`secret_key`） |
| `dnSDK.start()` | 启动数据采集，**必须在 `init()` 之后尽快调用** |

## 行为上报接口

| 接口 | 说明 |
:|------|------|
| `dnSDK.logAction(actionType, actionParam?)` | **通用行为上报（唯一的上报方法）** |

> **重要：** SDK 不提供任何预定义上报方法，所有行为均通过 `logAction` 上报。

### logAction 使用示例

```typescript
dnSDK.logAction('REGISTER', { method: 'huawei_account', is_success: true });
dnSDK.logAction('PURCHASE', { value: 9900, currency: 'CNY' });
dnSDK.logAction('VIEW_CONTENT', { content_id: 'article_123' });
```

### actionParam 约束

- **Key：** 字母、数字、下划线，必须以字母开头，最长 **64** 个字符
- **Value：** 支持 String / Number / Boolean / JSONArray

## 用户标识接口

| 接口 | 说明 |
:|------|------|
| `dnSDK.setUserUniqueId(userId)` | 设置自定义用户唯一标识 |

## 自动采集事件

| 事件名 | 说明 |
|--------|------|
| `START_APP` | 应用启动 |
| `TICKET` | 票据相关 |
| `ENTER_FOREGROUND` | 应用进入前台 |
| `ENTER_BACKGROUND` | 应用进入后台 |
| `APP_QUIT` | 应用退出 |

## 调试日志

- TAG：`[@dn-sdk/harmony v1.x.x]`
- 上报成功：`logAction success xxxx`
- 上报失败：`logAction failed xxxx`
