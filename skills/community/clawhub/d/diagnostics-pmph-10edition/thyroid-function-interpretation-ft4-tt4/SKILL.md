# 甲状腺功能评估中FT₄与TT₄的临床解读

## 适用场景
- 已测定TT₄、FT₄及TSH
- 患者疑似甲亢或甲减，但TT₄与TSH或临床症状矛盾
- 存在可能影响TBG水平的因素（如妊娠、雌激素使用、肝肾疾病等）

## 执行步骤

### 1. 判断是否适用本技能
- 确认患者**非严重非甲状腺疾病状态**（如危重症未校正）
- 若已知TBG异常，则**不可单独依赖TT₄**进行诊断

### 2. 解读TT₄结果（仅作参考）
- **TT₄升高**可能见于：甲亢、妊娠、口服雌激素/避孕药、原发性胆汁性胆管炎、TBG增多症、严重感染、心衰、肝肾疾病等
- **TT₄降低**可能见于：甲减、缺碘性甲状腺肿、慢性淋巴细胞性甲状腺炎、TBG减少症、糖尿病酮症酸中毒、恶性肿瘤等
- **注意**：TT₄受TBG显著影响（99.5% T₄与TBG结合），不能单独用于功能判断

### 3. 优先依据FT₄评估功能状态
- **FT₄升高**：高度提示甲亢（灵敏度优于TT₄），也可见于甲亢危象、甲状腺激素不敏感综合征、多结节性甲状腺肿
- **FT₄降低**：主要见于甲减；也可由抗甲状腺药物、糖皮质激素、苯妥英钠、多巴胺或肾病综合征引起

### 4. 联合判断策略
- 当TT₄与TSH或临床表现矛盾时，**直接依赖FT₄**作为主要判断依据
- 若有条件，测定TBG并计算 **T₄(μg/L) / TBG(mg/L) 比值** 辅助判断：
  - **3.1～4.5**：甲状腺功能正常
  - **0.2～2.0**：提示甲减
  - **7.6～14.8**：提示甲亢

> 提示：若无法测定TBG，应以FT₄为核心指标，并结合用药史和合并症综合分析。
