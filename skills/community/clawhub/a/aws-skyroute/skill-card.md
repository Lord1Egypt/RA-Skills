## Description: <br>
3D globe visualization of global traffic flowing to AWS regions, with real-time user connections, latency-colored arcs, CloudFront edge panels, P95/P99 metrics, error rate alerts, and sparkline trends. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and operations teams use this skill to create or adapt a browser-based AWS traffic monitoring dashboard for visualizing global user distribution, latency, request volume, and alert states. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The dashboard may display simulated AWS health and traffic data in a way that looks like live operational telemetry. <br>
Mitigation: Label simulated views clearly and connect production deployments only to verified observability data sources with transparent data provenance. <br>
