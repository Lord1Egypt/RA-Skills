# D07 法律领域任务要求

## 使用说明

本文件定义法律领域48个任务类型的组件要求。每个任务包含：
- **必选组件**: 完成任务必须具备的输入/资源
- **可选组件**: 增强任务质量的补充输入
- **组装顺序**: 组件的使用顺序
- **约束**: 任务执行的限制条件
- **格式**: 输出的标准格式

## L1 法律风险

### L1-01 活动风险识别
- **必选组件**: 活动类型描述、活动主体信息、活动范围和地域、涉及的法律关系
- **可选组件**: 历史类似案例、行业惯例、监管政策
- **组装顺序**: 分析活动性质→识别法律关系→确定适用法律→识别风险点→评估风险等级
- **约束**: ⚡校验: 风险识别必须基于现行法律法规，考虑法律环境变化趋势
- **格式**: 风险清单(风险点/风险等级/法律依据/防控建议)

### L1-02 合规差距分析
- **必选组件**: 合规基准要求、现状描述、差距识别标准
- **可选组件**: 行业最佳实践、监管检查结果、第三方评估报告
- **组装顺序**: 确定合规基准→评估现状→识别差距→分析差距原因→制定改进方向
- **约束**: ⚡校验: 合规基准必须基于最新法规，差距分析必须客观公正
- **格式**: 差距分析报告(合规领域/基准要求/现状/差距/改进建议)

### L1-03 法律环境扫描
- **必选组件**: 扫描范围(地域/行业/法律领域)、扫描时间范围、关注重点
- **可选组件**: 政策解读、司法判例、监管动态
- **组装顺序**: 确定扫描范围→收集法律法规→分析政策趋势→识别法律风险→提出应对建议
- **约束**: ⚡校验: 信息来源必须权威可靠，扫描必须覆盖相关法律领域
- **格式**: 法律环境报告(法律领域/现行规定/变化趋势/风险提示)

### L1-04 合同风险审查
- **必选组件**: 合同文本、合同背景信息、审查重点
- **可选组件**: 类似合同案例、行业标准条款、谈判地位信息
- **组装顺序**: 审查合同主体→审查合同条款→识别法律风险→评估风险等级→提出修改建议
- **约束**: ⚡校验: 审查必须基于合同法及相关法规，风险评估必须客观
- **格式**: 合同审查报告(条款/风险点/风险等级/修改建议)

### L1-05 知识产权风险
- **必选组件**: 知识产权类型、使用方式、使用范围
- **可选组件**: 知识产权检索结果、许可协议、侵权案例
- **组装顺序**: 确定知识产权类型→检索权利状态→分析使用方式→评估侵权风险→提出防控建议
- **约束**: ⚡校验: 风险评估必须基于知识产权法，考虑不同法域差异
- **格式**: 知识产权风险评估(知识产权类型/权利状态/使用方式/风险等级/建议)

### L1-06 劳动法风险
- **必选组件**: 用工模式、劳动合同条款、规章制度
- **可选组件**: 劳动争议案例、行业惯例、员工手册
- **组装顺序**: 分析用工模式→审查劳动合同→评估规章制度→识别法律风险→提出改进建议
- **约束**: ⚡校验: 风险评估必须基于劳动法及地方规定，考虑劳动仲裁实践
- **格式**: 劳动法风险评估(用工环节/风险点/法律依据/改进建议)

### L1-07 数据隐私风险
- **必选组件**: 数据类型、数据处理方式、数据主体
- **可选组件**: 隐私政策、数据安全措施、用户同意机制
- **组装顺序**: 识别数据类型→分析处理方式→评估合规要求→识别隐私风险→提出改进建议
- **约束**: ⚡校验: 风险评估必须基于个人信息保护法等法规，考虑不同法域隐私要求
- **格式**: 数据隐私风险评估(数据类型/处理方式/合规要求/风险点/改进建议)

## L2 合同管理

### L2-01 合同需求分析
- **必选组件**: 交易目的、交易对方信息、核心商业条款
- **可选组件**: 历史合同、行业标准、谈判地位
- **组装顺序**: 明确交易目的→分析交易对方→确定核心条款→识别法律要求→确定合同类型
- **约束**: ⚡校验: 需求分析必须符合法律规定，考虑合同可行性
- **格式**: 合同需求说明书(交易目的/对方信息/核心条款/法律要求)

### L2-02 合同起草
- **必选组件**: 合同需求说明书、合同模板、法律要求
- **可选组件**: 行业标准条款、类似合同范本、谈判要点
- **组装顺序**: 选择合同模板→填写基本信息→起草核心条款→添加法律条款→审核合同文本
- **约束**: ⚡校验: 合同条款必须合法有效，语言表达必须准确清晰
- **格式**: 合同文本(含所有必要条款)

### L2-03 合同审查
- **必选组件**: 合同文本、合同需求、审查标准
- **可选组件**: 法律意见、类似合同案例、风险清单
- **组装顺序**: 审查合同主体→审查合同条款→评估法律风险→提出修改建议→出具审查意见
- **约束**: ⚡校验: 审查必须基于合同法及相关法规，审查必须全面细致
- **格式**: 合同审查意见(条款/问题/风险/修改建议)

### L2-04 合同谈判
- **必选组件**: 合同审查意见、谈判目标、谈判底线
- **可选组件**: 谈判策略、替代方案、谈判记录
- **组装顺序**: 制定谈判策略→确定谈判要点→进行谈判→记录谈判结果→修改合同文本
- **约束**: ⚡校验: 谈判必须在法律框架内进行，谈判结果必须合法有效
- **格式**: 谈纪要(谈判要点/各方立场/达成共识/待决事项)

### L2-05 合同签署
- **必选组件**: 最终合同文本、签署主体资格证明、授权文件
- **可选组件**: 公证文件、见证文件、签署仪式安排
- **组装顺序**: 确认合同最终版本→核实签署主体资格→准备签署文件→组织签署→归档保管
- **约束**: ⚡校验: 签署程序必须符合法律规定，签署文件必须完整有效
- **格式**: 签署确认书(签署时间/地点/参与人/文件清单)

### L2-06 合同履行监控
- **必选组件**: 合同文本、履行计划、监控指标
- **可选组件**: 履行记录、沟通记录、变更文件
- **组装顺序**: 制定履行计划→设定监控指标→跟踪履行情况→记录履行问题→处理履行偏差
- **约束**: ⚡校验: 监控必须基于合同约定，问题处理必须及时有效
- **格式**: 履行监控报告(履行情况/问题/处理措施)

### L2-07 合同纠纷处理
- **必选组件**: 纠纷事实、合同约定、证据材料
- **可选组件**: 纠纷历史、沟通记录、调解方案
- **组装顺序**: 分析纠纷原因→评估法律风险→制定解决策略→进行协商→执行解决方案
- **约束**: ⚡校验: 纠纷处理必须基于合同和法律，考虑诉讼时效
- **格式**: 纠纷处理方案(纠纷事实/法律分析/解决策略/执行计划)

## L3 合规管理

### L3-01 法规梳理
- **必选组件**: 业务范围、行业类型、地域范围
- **可选组件**: 法规数据库、监管指南、行业标准
- **组装顺序**: 确定业务范围→识别适用法规→分类整理法规→分析法规要求→建立法规清单
- **约束**: ⚡校验: 法规梳理必须全面准确，必须包含最新法规
- **格式**: 法规清单(法规名称/发布日期/适用范围/主要要求)

### L3-02 合规制度设计
- **必选组件**: 法规清单、组织架构、业务流程
- **可选组件**: 合规政策、操作规程、合规手册
- **组装顺序**: 分析合规要求→设计制度框架→制定具体制度→审核制度内容→发布实施
- **约束**: ⚡校验: 制度必须符合法规要求，制度必须可操作可执行
- **格式**: 合规制度文件(制度名称/适用范围/具体要求/执行程序)

### L3-03 合规培训
- **必选组件**: 合规制度、培训对象、培训内容
- **可选组件**: 培训材料、案例库、考核标准
- **组装顺序**: 确定培训需求→设计培训内容→准备培训材料→实施培训→评估培训效果
- **约束**: ⚡校验: 培训内容必须准确完整，培训必须覆盖关键岗位
- **格式**: 培训方案(培训目标/内容/方式/考核)

### L3-04 合规检查
- **必选组件**: 合规制度、检查标准、检查范围
- **可选组件**: 检查清单、检查工具、历史检查结果
- **组装顺序**: 制定检查计划→设计检查清单→实施检查→记录检查结果→分析检查发现
- **约束**: ⚡校验: 检查必须客观公正，检查必须覆盖关键领域
- **格式**: 检查报告(检查范围/检查发现/问题分析/改进建议)

### L3-05 合规报告
- **必选组件**: 检查结果、合规数据、报告要求
- **可选组件**: 趋势分析、标杆对比、改进计划
- **组装顺序**: 汇总合规数据→分析合规趋势→编制合规报告→审核报告内容→提交报告
- **约束**: ⚡校验: 报告必须真实准确，报告必须及时提交
- **格式**: 合规报告(合规状况/问题分析/改进建议)

### L3-06 整改方案
- **必选组件**: 合规问题、整改要求、责任部门
- **可选组件**: 整改资源、时间计划、验收标准
- **组装顺序**: 分析问题原因→制定整改措施→明确责任分工→制定时间计划→设定验收标准
- **约束**: ⚡校验: 整改方案必须切实可行，整改必须彻底有效
- **格式**: 整改方案(问题描述/整改措施/责任分工/时间计划/验收标准)

### L3-07 合规持续改进
- **必选组件**: 整改结果、改进目标、改进资源
- **可选组件**: 最佳实践、改进工具、效果评估
- **组装顺序**: 评估整改效果→识别改进机会→制定改进计划→实施改进措施→评估改进效果
- **约束**: ⚡校验: 改进必须基于事实数据，改进必须持续有效
- **格式**: 改进报告(改进目标/改进措施/实施效果/后续计划)

## L4 知识产权

### L4-01 专利策略
- **必选组件**: 技术方案、商业目标、竞争环境
- **可选组件**: 专利检索结果、技术发展趋势、专利布局分析
- **组装顺序**: 分析技术方案→评估商业价值→检索现有专利→制定申请策略→规划专利布局
- **约束**: ⚡校验: 专利策略必须符合专利法，考虑专利审查实践
- **格式**: 专利策略报告(技术分析/商业价值/申请策略/布局规划)

### L4-02 商标注册
- **必选组件**: 商标图样、商品/服务类别、申请人信息
- **可选组件**: 商标检索结果、商标使用证据、优先权证明
- **组装顺序**: 设计商标图样→确定注册类别→进行商标检索→准备申请材料→提交注册申请
- **约束**: ⚡校验: 商标必须符合商标法要求，考虑商标审查标准
- **格式**: 商标注册申请文件(商标图样/类别/申请人信息)

### L4-03 著作权保护
- **必选组件**: 作品类型、作品内容、权利归属
- **可选组件**: 作品完成时间证据、首次发表证据、权利转让文件
- **组装顺序**: 确定作品类型→收集权利证据→进行著作权登记→建立保护机制→监控侵权行为
- **约束**: ⚡校验: 著作权保护必须符合著作权法，考虑作品独创性要求
- **格式**: 著作权保护方案(作品信息/权利证明/保护措施)

### L4-04 商业秘密保护
- **必选组件**: 秘密信息范围、保密措施、保密协议
- **可选组件**: 保密制度、保密培训、访问控制
- **组装顺序**: 确定秘密范围→制定保密措施→签订保密协议→建立保密制度→监控保密执行
- **约束**: ⚡校验: 保密措施必须合理有效，考虑商业秘密构成要件
- **格式**: 商业秘密保护方案(秘密范围/保密措施/保密协议/监控制度)

### L4-05 知识产权许可
- **必选组件**: 知识产权清单、许可方式、许可条件
- **可选组件**: 许可合同模板、许可费标准、使用限制
- **组装顺序**: 确定许可标的→选择许可方式→确定许可条件→起草许可合同→签订许可协议
- **约束**: ⚡校验: 许可必须符合知识产权法，许可条件必须明确具体
- **格式**: 知识产权许可合同(许可标的/许可方式/许可条件/使用限制)

### L4-06 知识产权维权
- **必选组件**: 侵权证据、权利证明、维权目标
- **可选组件**: 侵权分析报告、维权策略、诉讼材料
- **组装顺序**: 收集侵权证据→分析侵权行为→制定维权策略→采取维权措施→评估维权效果
- **约束**: ⚡校验: 维权必须基于有效权利，维权手段必须合法
- **格式**: 知识产权维权方案(侵权分析/维权策略/执行计划/效果评估)

### L4-07 知识产权评估
- **必选组件**: 知识产权清单、评估目的、评估基准日
- **可选组件**: 评估方法、可比交易数据、收益预测
- **组装顺序**: 确定评估范围→选择评估方法→收集评估数据→进行价值评估→出具评估报告
- **约束**: ⚡校验: 评估必须符合评估准则，评估方法必须适当
- **格式**: 知识产权评估报告(评估对象/评估方法/评估过程/评估结论)

## L5 公司法务

### L5-01 公司设立
- **必选组件**: 公司类型、股东信息、注册资本
- **可选组件**: 公司章程、出资协议、设立方案
- **组装顺序**: 确定公司类型→准备设立文件→办理工商登记→获取营业执照→办理后续手续
- **约束**: ⚡校验: 设立程序必须符合公司法，文件必须合法有效
- **格式**: 公司设立文件清单(设立文件/登记程序/注意事项)

### L5-02 股权架构
- **必选组件**: 股东信息、出资比例、公司治理需求
- **可选组件**: 股权激励计划、投资人要求、控制权安排
- **组装顺序**: 分析股东需求→设计股权结构→考虑控制权安排→设计激励机制→制定股权架构方案
- **约束**: ⚡校验: 股权架构必须符合公司法，考虑税务影响
- **格式**: 股权架构方案(股东结构/持股比例/控制权安排/激励机制)

### L5-03 股东协议
- **必选组件**: 股东信息、股权架构、核心条款
- **可选组件**: 股东权利义务、决策机制、退出机制
- **组装顺序**: 确定协议主体→明确股权结构→约定权利义务→设计决策机制→约定退出机制
- **约束**: ⚡校验: 协议必须符合公司法，条款必须公平合理
- **格式**: 股东协议(股东信息/股权结构/权利义务/决策机制/退出机制)

### L5-04 公司治理
- **必选组件**: 公司章程、治理结构、决策程序
- **可选组件**: 董事会议事规则、监事会议事规则、经营管理层职责
- **组装顺序**: 设计治理结构→制定公司章程→建立决策程序→明确职责权限→建立监督机制
- **约束**: ⚡校验: 治理结构必须符合公司法，决策程序必须合法有效
- **格式**: 公司治理方案(治理结构/决策程序/职责权限/监督机制)

### L5-05 融资法律
- **必选组件**: 融资方式、融资金额、融资条件
- **可选组件**: 融资协议、法律意见书、尽职调查报告
- **组装顺序**: 确定融资方式→准备融资文件→进行法律尽职调查→起草融资协议→完成融资交割
- **约束**: ⚡校验: 融资必须符合证券法等法规，融资文件必须合法有效
- **格式**: 融资法律文件清单(融资协议/法律意见/尽职调查报告)

### L5-06 并购法律
- **必选组件**: 并购方式、目标公司信息、并购条件
- **可选组件**: 并购协议、尽职调查报告、法律意见书
- **组装顺序**: 确定并购方式→进行法律尽职调查→起草并购协议→进行并购谈判→完成并购交割
- **约束**: ⚡校验: 并购必须符合公司法、证券法等法规，并购文件必须合法有效
- **格式**: 并购法律文件清单(并购协议/尽职调查报告/法律意见书)

### L5-07 公司解散
- **必选组件**: 解散原因、清算方案、债权债务处理
- **可选组件**: 清算报告、注销文件、员工安置方案
- **组装顺序**: 确定解散原因→成立清算组→进行清算→处理债权债务→办理注销登记
- **约束**: ⚡校验: 解散程序必须符合公司法，债权债务必须妥善处理
- **格式**: 公司解散方案(解散原因/清算方案/债权债务处理/注销程序)

## L6 纠纷解决

### L6-01 纠纷评估
- **必选组件**: 纠纷事实、法律关系、证据材料
- **可选组件**: 类似案例、法律意见、解决方案
- **组装顺序**: 了解纠纷事实→分析法律关系→评估证据材料→分析法律风险→制定解决策略
- **约束**: ⚡校验: 评估必须基于事实和法律，考虑诉讼时效
- **格式**: 纠纷评估报告(纠纷事实/法律分析/风险评估/解决策略)

### L6-02 协商调解
- **必选组件**: 纠纷评估报告、调解意愿、调解方案
- **可选组件**: 调解人信息、调解记录、调解协议
- **组装顺序**: 确定调解意愿→选择调解方式→准备调解方案→进行调解→达成调解协议
- **约束**: ⚡校验: 调解必须自愿合法，调解协议必须公平合理
- **格式**: 调解方案(调解意愿/调解方式/调解方案/预期结果)

### L6-03 仲裁准备
- **必选组件**: 仲裁协议、仲裁申请、证据材料
- **可选组件**: 仲裁规则、仲裁员信息、仲裁策略
- **组装顺序**: 确认仲裁协议→准备仲裁申请→收集证据材料→选择仲裁员→准备庭审
- **约束**: ⚡校验: 仲裁必须基于有效仲裁协议，仲裁程序必须符合仲裁法
- **格式**: 仲裁准备文件清单(仲裁申请/证据清单/仲裁策略)

### L6-04 诉讼准备
- **必选组件**: 起诉状、证据材料、诉讼请求
- **可选组件**: 诉讼策略、证人信息、法律依据
- **组装顺序**: 确定诉讼请求→准备起诉状→收集证据材料→确定管辖法院→提交起诉材料
- **约束**: ⚡校验: 诉讼必须符合民事诉讼法，注意诉讼时效
- **格式**: 诉讼准备文件清单(起诉状/证据清单/法律依据)

### L6-05 证据收集
- **必选组件**: 证据清单、证据来源、收集方式
- **可选组件**: 证据保全申请、公证取证、鉴定申请
- **组装顺序**: 确定证据需求→识别证据来源→收集证据材料→保全证据→整理证据清单
- **约束**: ⚡校验: 证据收集必须合法，证据必须真实有效
- **格式**: 证据清单(证据名称/证据来源/证明目的/收集方式)

### L6-06 庭审应对
- **必选组件**: 庭审提纲、证据材料、法律依据
- **可选组件**: 庭审策略、质证提纲、辩论提纲
- **组装顺序**: 准备庭审提纲→整理证据材料→准备质证意见→准备辩论意见→进行庭审
- **约束**: ⚡校验: 庭审必须遵守法庭纪律，陈述必须真实准确
- **格式**: 庭审准备文件(庭审提纲/证据清单/质证意见/辩论意见)

### L6-07 执行与上诉
- **必选组件**: 判决书/裁决书、执行申请、上诉材料
- **可选组件**: 执行策略、上诉理由、和解方案
- **组装顺序**: 分析判决结果→决定是否上诉→准备执行申请→申请强制执行→处理执行异议
- **约束**: ⚡校验: 执行必须符合执行程序，上诉必须在法定期限内
- **格式**: 执行/上诉方案(判决分析/执行策略/上诉理由/执行计划)

## L7 传承法务

### L7-01 遗嘱起草
- **必选组件**: 遗嘱人信息、遗产清单、继承人信息
- **可选组件**: 遗嘱执行人、遗嘱见证人、特别指示
- **组装顺序**: 确定遗嘱人意愿→梳理遗产清单→确定继承人→起草遗嘱→办理遗嘱公证
- **约束**: ⚡校验: 遗嘱必须符合继承法，遗嘱形式必须合法
- **格式**: 遗嘱文件(遗嘱人信息/遗产清单/继承人/分配方案)

### L7-02 信托设立
- **必选组件**: 信托目的、信托财产、信托当事人
- **可选组件**: 信托合同、信托监察人、信托管理方案
- **组装顺序**: 确定信托目的→确定信托财产→确定信托当事人→起草信托合同→设立信托
- **约束**: ⚡校验: 信托必须符合信托法，信托财产必须合法
- **格式**: 信托文件(信托目的/信托财产/信托当事人/信托条款)

### L7-03 赠与安排
- **必选组件**: 赠与人信息、受赠人信息、赠与财产
- **可选组件**: 赠与合同、赠与条件、税务安排
- **组装顺序**: 确定赠与意愿→确定赠与财产→确定受赠人→起草赠与合同→办理赠与手续
- **约束**: ⚡校验: 赠与必须符合合同法，考虑税务影响
- **格式**: 赠与文件(赠与人/受赠人/赠与财产/赠与条件)

### L7-04 遗产税规划
- **必选组件**: 遗产清单、继承人信息、税务规定
- **可选组件**: 税务筹划方案、免税额度、税务优惠
- **组装顺序**: 梳理遗产清单→评估遗产价值→分析税务规定→制定税务规划→实施税务安排
- **约束**: ⚡校验: 税务规划必须合法合规，考虑税法变化
- **格式**: 遗产税规划方案(遗产清单/税务分析/规划方案/实施计划)

### L7-05 继承纠纷预防
- **必选组件**: 继承人信息、遗产清单、继承意愿
- **可选组件**: 继承协议、调解方案、法律意见
- **组装顺序**: 了解继承人意愿→梳理遗产清单→沟通继承方案→签订继承协议→办理继承手续
- **约束**: ⚡校验: 继承安排必须符合继承法，继承协议必须公平合理
- **格式**: 继承纠纷预防方案(继承人/遗产清单/继承方案/预防措施)

### L7-06 跨境传承
- **必选组件**: 跨境遗产清单、继承人信息、适用法律
- **可选组件**: 跨境法律意见、税务安排、遗产管理方案
- **组装顺序**: 梳理跨境遗产→分析适用法律→制定传承方案→协调跨境执行→处理税务问题
- **约束**: ⚡校验: 跨境传承必须考虑多国法律，考虑国际条约
- **格式**: 跨境传承方案(跨境遗产/适用法律/传承方案/执行计划)
