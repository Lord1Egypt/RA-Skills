# 股东会决议 / 董事会决议

## 适用场景
公司召开股东会或董事会后，对会议讨论通过的事项形成书面决议，作为办理工商变更、银行开户、对外签约等的有效依据。

## 提交机构
公司内部留存；涉及工商变更登记的提交市场监督管理局

## 模板正文

```
               ______有限责任公司
              股东会决议 / 董事会决议

        会议时间：______年______月______日
        会议地点：____________________________
        出席人员：全体股东/全体董事共___人
        主持人：______
        记录人：______

本次会议依法召开，符合《中华人民共和国公司法》及公司章程的规定。
经与会股东/董事讨论和表决，通过如下决议：

一、关于____________________的决议
同意________________________。
（表决结果：赞成___票，反对___票，弃权___票）

二、关于____________________的决议
同意________________________。
（表决结果：赞成___票，反对___票，弃权___票）

三、关于____________________的决议
同意________________________。
（表决结果：赞成___票，反对___票，弃权___票）

（以下无正文）

全体股东/董事签名（或盖章）：

1. ______（签名/盖章）
2. ______（签名/盖章）
3. ______（签名/盖章）
4. ______（签名/盖章）
5. ______（签名/盖章）

                   ______有限责任公司（盖章）
                   日期：______年______月______日
```

## 注意事项
- **关键填写点**：决议事项需逐一列明并注明表决结果；签字人数必须满足法定或章程规定的表决权比例（一般事项过半，重大事项三分之二）；签章必须为有权签字的股东/董事本人
- **常见错误**：未达法定出席人数（应到与实到人数不符）；表决结果表述含糊（"多数通过"而非"X票赞成"）；签字人与实际持股/任职不符（如工商变更后未更新签字人）；决议事项超出职权范围
- **份数要求**：公司存档1份正本，工商提交1份（可为复印件加盖章），相关各方各1份
- **常见用途**：变更法定代表人、增减注册资本、修改章程、股权转让、贷款担保、公司合并分立等
- **决议效力**：决议内容违反法律或章程的，股东可自决议作出之日起60日内请求法院撤销
