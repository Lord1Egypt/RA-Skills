## Description: <br>
Generate valid Mermaid diagrams such as flowcharts, sequence diagrams, mind maps, ER diagrams, and user journeys from text or product descriptions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[pupujanet-eng](https://clawhub.ai/user/pupujanet-eng) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Product managers, developers, and agents use this skill to convert product descriptions, PRDs, workflow notes, and architecture descriptions into Mermaid diagram code. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill may activate on broad diagram-related requests and generate a diagram type the user did not intend. <br>
Mitigation: Check that the generated output matches the requested diagram type before using or sharing it. <br>
Risk: Generated Mermaid syntax can contain diagram-format mistakes or labels that render poorly. <br>
Mitigation: Review the Mermaid code block and render it in a Mermaid-compatible viewer before relying on the diagram. <br>
