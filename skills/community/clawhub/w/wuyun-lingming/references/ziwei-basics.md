## 一、紫微斗数概述

### 1.1 什么是紫微斗数

紫微斗数，又称"紫微术"，是中国古代术数体系中的一种**命理学派别**，被命理界称为"三大神数之首"。

**核心原理**：以人的农历出生年、月、日、时为基础，依固定公式排列命盘，分列十二宫位，将诸星曜分列各宫位，用以推研人生造化。

**命名由来**：
- "斗"指南北二斗（北斗、南斗）
- "数"指吉凶祸福的气数
- 紫微星为诸星之首，故称"紫微斗数"

### 1.2 历史渊源

| 时期 | 发展 |
|------|------|
| 上古 | 雏形可追溯至上古时期的星象观测术 |
| 唐代 | 道士吕洞宾《紫微斗数秘本》奠定理论基础 |
| 宋代 | 陈希夷（陈抟）系统整理，编撰《紫微斗数全书》 |
| 明清 | 从宫廷秘术走向民间实用化 |
| 现代 | 形成三合派、四化派等主流派别，AI排盘工具出现 |

### 1.3 两大主流派别

| 派别 | 特点 | 擅长 |
|------|------|------|
| **三合派** | 注重星曜组合与宫位联动 | 性格分析、全盘平衡 |
| **四化派** | 以"禄权科忌"飞星为核心 | 事件推演（如何时升迁） |

## 二、星曜系统

### 2.1 星曜分类（类似代码的类继承关系）

```
星曜体系
├── 甲级星（主星）- 权重最高，影响命运主轴
│   ├── 十四正星
│   └── 四化星
├── 乙级星（辅星）- 次要影响力
│   ├── 佐曜：文昌、文曲、天马、禄存
│   └── 辅曜：左辅、右弼、天魁、天钺
├── 丙级星（杂曜）- 辅助参考
└── 丁级星（煞星）- 触发条件
    ├── 四煞：铃星、火星、擎羊、陀罗
    └── 六煞：四煞 + 地空、地劫
```

### 2.2 十四主星（核心星曜）

紫微斗数的十四主星分为**紫微星系**和**天府星系**两大集团：

#### 紫微星系（帝星集团，逆时针分布）

| 主星 | 五行 | 象征意义 | 特点 |
|------|------|----------|------|
| **紫微星** | 阴土 | 尊贵、权威 | 官禄主，帝星，命主多具领导欲 |
| **天机星** | 阴木 | 智慧、变动 | 谋士星，命主从事技术、策划工作 |
| **太阳星** | 阳火 | 光明、奉献 | 贵星，适合教育、公益行业 |
| **武曲星** | 阴金 | 财星、刚毅 | 正财星，适合金融、刚毅果断者 |
| **天同星** | 阳水 | 福星、温和 | 享受型，温和重感情 |
| **廉贞星** | 火 | 次桃花 | 感情复杂，易有情感纠葛 |

#### 天府星系（财库集团，顺时针分布）

| 主星 | 五行 | 象征意义 | 特点 |
|------|------|----------|------|
| **天府星** | 阳火 | 库星、稳重 | 财库守门人，稳重保守 |
| **太阴星** | 阴水 | 母星、柔情 | 女性倾向，细腻温柔 |
| **贪狼星** | 木水 | 桃花、欲望 | 正桃花星，多才多艺但易贪欲 |
| **巨门星** | 阴水 | 暗星、口才 | 口才出众，但易有是非 |
| **天相星** | 阴水 | 印星、辅佐 | 服务型，适合幕僚 |
| **天梁星** | 阳土 | 荫星、慈善 | 逢凶化吉，有长辈缘 |
| **七杀星** | 火金 | 将星、刚烈 | 创业型，适合开创事业 |
| **破军星** | 水 | 先锋、破坏 | 冲锋陷阵，变革型 |

### 2.3 四化星（动态能量）

四化是紫微斗数的**动态核心**，代表星曜在不同年份的能量变化：

| 四化 | 五行 | 含义 | 象征 |
|------|------|------|------|
| **化禄** | 阴土 | 财富、机遇 | "得到"与"满足" |
| **化权** | 阳火 | 权力、竞争 | "强化"与"主导" |
| **化科** | 阳木 | 名声、才艺 | "显现"与"认可" |
| **化忌** | 阴水 | 阻碍、消耗 | "压力"与"缺失" |

**触发机制**：四化依据十天干（甲至癸）触发，不同天干使特定主星产生相应变化。

## 三、命盘结构

### 3.1 三个盘（类似软件的三层架构）

```
┌─────────────────────────────────────┐
│            命盘三层结构              │
├─────────────────────────────────────┤
│ 天盘（主盘）  │ 以命宫纳音五行为依据 │ 展示先天命运趋势
│ 人盘（运限）  │ 大限、小限、流年等   │ 展示各阶段沉浮曲线
│ 地盘（根盘）  │ 以身宫纳音五行为依据 │ 展示潜在性情根基
└─────────────────────────────────────┘
```

| 6 | **疾厄宫** | 健康状况 | 印星，先天体质 |
| 7 | **迁移宫** | 外出发展与际遇 | 驿马，外部机遇 |
| 8 | **交友宫** | 人际关系 | 下属、合作伙伴 |
| 9 | **官禄宫** | 事业成就 | 官杀，工作事业 |

## 四、排盘基础（手工排盘十六步）

### 4.1 排盘前准备

**工具**：
- 万年历（查农历生辰）
- 空白命盘图

**基础概念**：
```
十天干：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
十二地支：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥
六十甲子：天干地支组合循环（类似进制转换）
```

### 4.2 手工排盘步骤（简明版）

#### 第一步：定农历生辰
- 年：以春节为分界
- 月：以农历月为准，闰月以下个月计算
- 日：农历生日数
- 时：23:00-01:00 为子时（不分早晚子时）
- **注意**：以北京时间为准，需考虑真太阳时

#### 第二步：画命盘，填十二地支
```
        命盘十二宫排列（逆时针）：
        
            巳 午 未 申
           ┌──┬──┬──┬──┐
          辰│    卯│
           ├──┼──┼──┤
          卯 │    寅│
           ├──┼──┼──┤
          子 │    丑│
           └──┴──┴──┘
            亥 戌 酉 申
```

#### 第三步：起寅首天干（五虎遁）
```
甲己年 → 丙寅
乙庚年 → 戊寅
丙辛年 → 庚寅
丁壬年 → 壬寅
戊癸年 → 甲寅
```
顺时针排列：天干+地支

#### 第四步：定命宫与身宫
```
口诀：寅宫起正月，顺数生月宫，逆生时为命，顺生时是身

示例：农历3月辰时生
1. 从寅宫起正月 → 卯(2月) → 辰(3月)
2. 从辰宫起子时
3. 命宫：逆数到辰时 → 寅宫为命宫
4. 身宫：顺数到辰时 → 卯宫为身宫
```

#### 第五步：定五行局
根据命宫天干地支查表确定

#### 第六步：安紫微星（关键步骤）
根据五行局和出生日数，查表确定紫微星位置

#### 第七步：安十四正星

**紫微星系（逆行口诀）**：
```
紫微天机逆行旁，隔一阳武天同当，
又隔二位廉贞地，空三复见紫微郞。
```

**天府星系（顺行口诀）**：
```
天府太阴与贪狼，巨门天相及天梁，
七杀空三破军位，八星顺数细推详。
```

#### 第八步：安四化星
按年干查表标注

#### 第九步：安其他星曜
按生月、生时、年干等依次安放

### 4.3 快速查表法要点

现代排盘多使用**软件或在线工具**，但理解原理有助于深入学习：

| 排盘要素 | 确定依据 |
|----------|----------|
| 命宫位置 | 生月 + 生时 |
| 身宫位置 | 生月 + 生时 |
| 紫微星位置 | 五行局 + 出生日数 |
| 十四正星 | 紫微星位置 |
| 四化星 | 出生年干 |
| 文昌文曲 | 出生时支 |

### 4.4 出生时辰的特殊处理

**闰月**：排命盘时按下一个农历月计算

**时辰不明**：可参考
- 睡眠姿势：仰卧→子午卯酉；侧卧→寅申巳亥；俯卧→辰戌丑未
- 小指长度对比判断
