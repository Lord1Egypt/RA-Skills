# 抖音脚本合规红线

## 禁用词（绝对不能出现）

以下词汇触犯广告法，脚本中严禁出现：

- 国家级、最高级、最佳、第一、顶级、极品、万能
- 纯天然、100%、全网第一、全球第一、世界顶级、世界领先
- 绝对、终极、至极、史上最强、无敌、最强

## 极限词（尽量避免，使用时需替换方案）

| 极限词 | 建议替代表达 |
|--------|-------------|
| 最、极 | 非常、相当、格外 |
| 超级、超 | 很、十分、特别 |
| 巨、爆、炸、逆天 | 惊喜、好用、亮点 |
| 秒杀、必买、必囤 | 值得一试、推荐入手 |
| 人手一件、人手一瓶 | 热门款、口碑款 |
| 抢、囤、all in | 入手、选择、尝试 |

## 医疗相关敏感词（禁止使用）

以下词汇涉及医疗宣传，禁止在产品推广脚本中使用：

- **治疗类**：治疗、治愈、医治、疗效、药效、处方、医院
- **疾病类**：疾病、病症
- **药品功效类**：消炎、抗菌、杀毒、防疫

> ⚠️ 护肤/保健品脚本尤其注意，不能暗示疾病治疗效果，只能描述使用感受与外观改善。

## 食品饮料特别注意

- 不得宣称减肥、降血糖、降血压等医疗功效
- 保健食品不得宣称治疗作用
- 代餐类产品避免使用"绝食""断食"等极端表达

## 合规表达技巧

- 用**使用感受**替代功效宣称：「用完皮肤感觉水润」而非「修复受损肌肤」
- 用**场景描述**替代效果保证：「早上喝一杯，整天状态都好」而非「必定提神醒脑」
- 用**用户口吻**软化宣传：「我用了一周，感觉...」而非「经科学验证...」
