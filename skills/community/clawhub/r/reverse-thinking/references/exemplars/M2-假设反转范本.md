# M2 假设反转域范本

## M2-01 反转策略

### 输入示例
假设：教师中心假设、课堂固定假设、考试评估假设

### 处理过程
1. 策略目标：反转核心假设
2. 策略方法：逐一反转每个假设
3. 策略内容：为每个假设设计反转策略
4. 策略验证：验证策略的可行性

### 输出示例
```markdown
## 反转策略报告

### 策略目标
- **目标**：反转核心假设
- **原则**：推到逻辑极端

### 策略方法
- **方法**：逐一反转每个假设
- **反转**：教师中心→学生中心，课堂固定→随时随地，考试评估→过程评估

### 策略内容
| 假设 | 反转策略 | 反转结果 |
|------|----------|----------|
| 教师中心假设 | 反转为学生中心 | 学生成为知识的主要来源 |
| 课堂固定假设 | 反转为随时随地 | 学习不受空间限制 |
| 考试评估假设 | 反转为过程评估 | 过程评估取代考试 |

### 策略验证
- **验证方法**：检查策略的可行性
- **验证结果**：策略可行
```

1. 执行计划：按策略执行反转
2. 执行步骤：逐一反转每个假设
3. 执行控制：控制反转质量
4. 执行验证：验证反转成功

```markdown
### 执行计划
- **计划**：按策略执行反转

| 3 | 反转考试评估假设 | 完成 |

### 执行控制
- **控制目标**：确保反转质量
- **控制方法**：检查反转合理性

### 执行验证
- **验证方法**：检查反转结果
- **验证结果**：反转成功
```
