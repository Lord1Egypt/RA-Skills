## Known Risks and Mitigations: <br>
Risk: Screenshots may contain credentials, private customer data, confidential UI, or unreleased designs and may be sent to a vision model for analysis. <br>
Mitigation: Use the skill only with screenshots whose data handling is acceptable, and redact sensitive UI content before analysis. <br>
Risk: Generated UI code may not perfectly preserve behavior, accessibility, or visual details from the screenshot. <br>
Mitigation: Review, test, and adapt generated code before using it in production. <br>
