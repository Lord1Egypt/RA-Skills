## 🇨🇳 中国 — GB/T 1.1 体系

### 适用场景
- 国家标准（GB/GB/T）起草与审查
- 行业标准（如 JR/T 金融标准）编制
- 地方标准（DB）规范性检查
- 团体标准（T/）自我声明前审查
- 企业标准（Q/）备案前自查

### 使用方法
```
用户：请用 GB/T 1.1-2020 检查这个标准草案
技能加载：li_git1.1_checker
→ Phase 1：80+ 项 checklist 快速扫描
→ Phase 2：vision_analyze 读取标准原文逐条对照
→ Phase 3：输出结构化检查报告
```

### 关键检查点（中国）
| 检查维度 | 重点条款 | 常见问题 |
|----------|----------|----------|
| 封面 | 8.1 | ICS/CCS 号缺失 |
| 前言 | 8.3 | 缺少起草规则说明 |
| 范围 | 8.5 | 包含要求性表述 |
| 术语 | 8.7 | 循环定义 |
| 编号 | 6.1 | 层次超过五级 |
| 文体 | 9.1 | "应/宜/可"混用 |

### 输出报告格式（中国）
```markdown
# GB/T 1.1-2020 标准文档检查报告
## 文档信息
- 文档名称：[标准名称]
- 标准类型：国家标准/行业标准/地方标准/团体标准/企业标准
- 检查日期：[日期]
```

## 🇺🇸 美国 — ANSI/ASTM 体系

### 适用场景
- 美国国家标准（ANSI）草案审查
- ASTM 标准结构规范性检查
- 企业标准与 ANSI 结构对标
- 出口产品标准文件合规检查

### 中美标准对比
| 要素 | 中国 GB/T 1.1 | 美国 ANSI/ASTM |
|------|---------------|----------------|
| 封面 | ICS + CCS | ANSI 编号体系 |
| 前言 | 必须包含起草说明 | 必须包含 Scope 和 Referenced Documents |
| 范围 | 第1章 | Section 1 - Scope |
| 术语 | 第3章 | Section 3 - Terminology |
| 文体 | 应/宜/可 | shall/should/may |

### 使用方法
```
用户：请检查这个标准是否符合 ANSI 结构要求
技能：使用 li_git1.1_checker 进行结构对标检查
注意：对照时将 GB/T 1.1 条款映射到 ANSI 相应章节
```

### 关键映射
- GB/T 1.1 范围 → ANSI Section 1 (Scope)
- GB/T 1.1 术语 → ANSI Section 3 (Terminology)
- GB/T 1.1 要求 → ANSI Section 4+ (Requirements)

## 🌍 国际 — ISO/IEC 导则体系

### 适用场景
- ISO/IEC 国际标准草案审查
- 中国标准国际化对标
- 参与国际标准制定的文件准备
- ISO/IEC 导则 Part 2 培训

### 国际与中国标准对比
| 要素 | 中国 GB/T 1.1 | ISO/IEC 导则 Part 2 |
|------|---------------|---------------------|
| 编号 | GB/T XXXXX | ISO/IEC XXXXX:XXXX |
| 前言 | 起草说明 | Foreword |
| 范围 | 第1章 | 1 Scope |
| 引用 | 第2章 | 2 Normative references |
| 文体 | 应/宜/可 | shall/should/may |

### 使用方法
```
用户：请用 ISO/IEC 导则 Part 2 检查这个国际标准草案
技能：li_git1.1_checker
说明：GB/T 1.1-2020 等同采用 ISO/IEC 导则 Part 2:2018，可直接用于 ISO 标准结构检查
```

### ISO 标准检查重点
- Foreword 固定格式（第1条、第2条...）
- Scope 规范性（不应包含要求）
- Normative references 版本注明
- Terms and definitions 定义格式
- 附录标注（normative/informative）
- 参考文献格式（ISO 690）

### GB/T 1.1 与 ISO/IEC 导则的关系
```
GB/T 1.1-2020 ← 等同采用 → ISO/IEC Directives, Part 2:2018
```
因此，使用 `li_git1.1_checker` 检查 GB/T 1.1 合规性，等同于检查 ISO/IEC 导则 Part 2 合规性。

## 通用使用流程

### Phase 1: Checklist 快速扫描
```
输入：待检查文档
输出：80+ 项检查点快速扫描结果
  ✅ 通过项
  ⚠️ 警告项
  ❌ 错误项
  ➖ 不适用项
```

### Phase 2: 标准原文深度审查
```
输入：Phase 1 发现的问题
处理：vision_analyze 读取标准原文逐条对照
输出：具体问题 + 标准条款引用 + 修改建议
```

### Phase 3: 结构化报告
```
输出格式：
  一、总体评价
  二、结构检查
  三、表述规范检查
  四、问题汇总（严重/一般/轻微）
  五、检查依据
```
