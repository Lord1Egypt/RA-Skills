## 🌏 出发前30天

### 签证与证件
- [ ] 确认护照有效期（≥6个月）
- [ ] 申请签证（或确认免签资格）
- [ ] 准备签证材料（机票、酒店预订单、在职证明）
- [ ] 护照复印件（2-3份，分开存放）
- [ ] 签证页复印件
- [ ] 证件照（白底，2寸，3-5张）
- [ ] 身份证复印件（国内出发）

### 机票与行程
- [ ] 预订国际往返机票（Trip.com/携程/官网）
- [ ] 确认行李额度（免费托运20-46kg不等）
- [ ] 打印机票确认单（电子版手机存）
- [ ] 规划境内交通（高铁/航班）
- [ ] 预订第一晚酒店

### 财务准备
- [ ] 通知银行出国计划（避免卡被锁）
- [ ] 确认卡片境外网上支付已开通
- [ ] 申请境外取现额度提升
- [ ] 准备现金：€200-500 或 $200-500 USD
- [ ] 确认信用卡（Visa/MasterCard全币种）

### 通讯与网络
- [ ] 确认手机开通国际漫游（或准备境外套餐）
- [ ] 准备VPN（必须在境外先注册好！）
- [ ] 下载离线地图（高德/百度）
- [ ] 下载翻译APP（Google Translate/腾讯翻译君）

### 数字准备（离境前必做）
- [ ] 支付宝/微信支付账号注册并绑定境外卡（Visa/MasterCard/JCB）
- [ ] 确认支付宝/微信支付境外交易功能已开通
- [ ] 下载好离线内容（音乐、剧集、电子书）
- [ ] VPN客户端已安装并测试可用
- [ ] 手机设置好时区（中国时区UTC+8）
- [ ] 保存好重要联系人（酒店、使领馆、保险公司）

### 保险
- [ ] 购买旅行保险（医疗+行李+航班延误）
- [ ] 打印保险单
- [ ] 记录保险公司热线

## ✈️ 出发前7天

### 行李准备
- [ ] 打包行李（注意航空公司限制）
- [ ] 随身带：护照、机票、手机、钱包
- [ ] 随身带：充电宝（≤100Wh，标注金额）
- [ ] 随身带：水杯（机场过安检前需清空）
- [ ] 随身带：外套/围巾（机上冷）
- [ ] 托运：衣物、洗漱用品、药品

### 清单确认
- [ ] 确认VPN可用
- [ ] 支付宝/微信支付绑定境外卡测试
- [ ] 预订门票（故宫、兵马俑等需提前预约）
- [ ] 预约接机/送机

### 家中准备
- [ ] 取消订阅报纸/牛奶
- [ ] 设置不在家自动回复邮件
- [ ] 告知家人朋友行程
- [ ] 倒垃圾、浇花等

### 健康准备
- [ ] 准备常备药（感冒药、止泻药、创可贴）
- [ ] 准备个人洗漱用品（酒店提供但不习惯）
- [ ] 准备拖鞋（飞机/酒店用）

---

## 🛬 落地后24小时

### 机场必做
- [ ] 填写入境卡（飞机上会发）
- [ ] 排队通关（准备好护照、签证、机票）
- [ ] 提取行李
- [ ] 通过海关（免税烟酒限额、水果禁止）
- [ ] 换少量人民币（机场汇率差，够当天即可）

### 落地后首日
- [ ] 购买当地手机卡（或开通国际漫游）
- [ ] 注册支付宝/微信支付
- [ ] 确认酒店预订
- [ ] 确认第二天的交通方式
- [ ] 报平安（家人/朋友）

### 🏨 住宿安排（2026年3月新规）
> ⚠️ **2024年5月起，公安部要求所有酒店必须接受外宾**（不可拒绝）
> ⚠️ **2026年3月20日起，7省可在线登记非酒店住宿**（河北、辽宁、浙江、湖北、广西、重庆、四川）

- [ ] 优先选择连锁/国际品牌酒店（万豪、希尔顿、亚朵、汉庭、宜必思、Trip.com）
- [ ] 预订前确认酒店"接待外宾"资质（Booking.com筛选器可用）
- [ ] 保留中文酒店名+地址（截图给出租车司机）
- [ ] **如果住在民宿/短租/亲友家**，且在7个试点省份，可通过NIA在线登记（无需到派出所）
- [ ] 民宿建议提前与房东确认接待外宾经验

## 📅 每日检查清单

### 出门前
- [ ] 护照在身上（最强警告！）
- [ ] 钱包里有足够人民币
- [ ] 手机电量充足
- [ ] 酒店名片（在口袋里）
- [ ] 天气预报（决定是否带伞/外套）

### 晚上回酒店
- [ ] 确认明天行程
- [ ] 预订明天门票（部分需提前）
- [ ] 确认酒店叫早（如果有早班机）
- [ ] 充电宝+手机+钱包放枕头边
- [ ] 把大额现金藏好（保险箱或分开放）

### 退房前
- [ ] 检查房间（尤其浴室、枕头下）
- [ ] 带走所有充电器
- [ ] 确认没有遗留物品
- [ ] 拿到发票/收据

---

## 🏨 酒店检查

### 入住时
- [ ] 确认房型和预订一致
- [ ] 检查空调/热水
- [ ] 测试门锁
- [ ] 确认Wi-Fi密码
- [ ] 找到逃生通道

### 安全检查
- [ ] 门链/门锁反锁
- [ ] 可疑小卡片不要理会
- [ ] 不开门给陌生人
- [ ] 贵重物品放保险箱

## 📝 随身包清单

### 绝对随身带
- ✅ 护照（复印件放行李箱）
- ✅ 手机+充电宝
- ✅ 钱包（少量现金+1张卡）
- ✅ 酒店名片
- ✅ 机票/火车票订单
- ✅ 保险单（电子版可）

### 建议随身带
- 💊 常备药（感冒药、止泻药）
- 🧴 湿巾/免洗洗手液
- 🕶️ 墨镜+帽子
- 📱 充电线
- 📓 小本子+笔

### 建议放行李箱
- 衣物（按天数+1套备份）
- 洗漱用品（酒店有但不习惯）
- 吹风机（部分酒店差）
- 转换插头（如果需要）
- 睡衣
- 泳装（部分酒店有泳池）
