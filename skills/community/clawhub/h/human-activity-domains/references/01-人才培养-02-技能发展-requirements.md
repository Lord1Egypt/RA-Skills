## 技能发展生成要求

技能发展每种任务类型的"零件清单"——必选/可选组件、组装顺序、领域约束。按清单逐项产出。

---

### 1 技能评估

#### T1-1 现状诊断

- **必选组件**: 技能测评工具（笔试/实操/面试）、能力维度清单、评分标准、诊断报告模板
- **可选组件**: 360度评估、自评问卷、过往项目分析
- **组装顺序**: 确定评估维度 → 选择测评工具 → 执行测评 → 收集数据 → 生成诊断报告
- **约束**: 测评工具必须有效可靠；评估维度必须覆盖核心技能；结果必须客观量化
- **格式**: 技能诊断报告（Markdown）

#### T1-2 差距分析

- **必选组件**: 目标岗位能力标准、当前技能水平、差距清单、优先级排序
- **可选组件**: 行业对标数据、竞品分析、发展趋势预测
- **组装顺序**: 获取目标标准 → 对比当前水平 → 识别差距 → 分析原因 → 排序优先级
- **约束**: 目标标准必须来自可靠来源；差距识别必须具体可量化；优先级必须基于重要性和紧迫性
- **格式**: 差距分析报告（Markdown）

#### T1-3 目标设定

- **必选组件**: 短期目标（1-3个月）、中期目标（3-6个月）、长期目标（6-12个月）、关键指标
- **可选组件**: 里程碑节点、奖励机制、风险预案
- **组装顺序**: 分析差距清单 → 设定短期目标 → 设定中期目标 → 设定长期目标 → 定义关键指标
- **约束**: 目标必须SMART；长短期目标必须衔接；关键指标必须可衡量
- **格式**: 技能发展目标文档（Markdown）

#### T1-4 路径规划

- **必选组件**: 学习路径图、时间节点、资源清单、里程碑
- **可选组件**: 备选路径、加速方案、风险应对
- **组装顺序**: 确定学习顺序 → 分配时间 → 识别资源 → 设定里程碑 → 绘制路径图
- **约束**: 路径必须合理可行；时间必须充裕；资源必须可获得
- **格式**: 技能提升路径图（Markdown/图表）

---

### 2 技能训练

#### T2-1 训练计划

- **必选组件**: 训练内容清单、训练方法选择、时间安排、资源准备
- **可选组件**: 个性化方案、弹性安排、监督机制
- **组装顺序**: 分析学习目标 → 设计训练内容 → 选择训练方法 → 安排时间 → 准备资源
- **约束**: 内容必须覆盖目标；方法必须匹配内容；时间必须合理分配
- **格式**: 训练计划表（Markdown表格）

#### T2-2 知识学习

- **必选组件**: 学习资料（教材/文档/视频）、学习笔记、知识测验、学习总结
- **可选组件**: 在线课程、专家讲座、学习社群
- **组装顺序**: 获取学习资料 → 系统学习 → 记录笔记 → 完成测验 → 总结归纳
- **约束**: 资料必须权威可靠；学习必须系统完整；测验必须覆盖重点
- **格式**: 学习笔记（Markdown）

#### T2-3 实操练习

- **必选组件**: 练习项目、操作指南、练习记录、成果作品
- **可选组件**: 模拟环境、沙盒平台、真实项目
- **组装顺序**: 选择练习项目 → 学习操作指南 → 动手实操 → 记录过程 → 产出成果
- **约束**: 项目必须贴近实际；操作必须可重复；成果必须可展示
- **格式**: 实操练习记录（Markdown）

#### T2-4 反馈指导

- **必选组件**: 导师/同伴、反馈渠道、反馈记录、改进计划
- **可选组件**: 一对一辅导、小组讨论、在线答疑
- **组装顺序**: 寻求反馈 → 接受指导 → 记录反馈 → 制定改进计划 → 执行改进
- **约束**: 反馈必须及时具体；指导必须专业有效；改进必须可执行
- **格式**: 反馈指导记录（Markdown）

---

### 3 技能认证

#### T3-1 认证标准

- **必选组件**: 认证机构信息、认证标准文档、考核要求、报名条件
- **可选组件**: 认证对比分析、备考建议、费用预算
- **组装顺序**: 了解认证机构 → 获取认证标准 → 分析考核要求 → 确认报名条件 → 制定备考计划
- **约束**: 标准必须来自官方渠道；要求必须完整准确；条件必须核实
- **格式**: 认证标准分析（Markdown）

#### T3-2 考核评估

- **必选组件**: 考核方案（理论/实操/综合）、备考资料、模拟测试、应试技巧
- **可选组件**: 考前辅导、模拟考场、心理辅导
- **组装顺序**: 制定备考计划 → 准备备考资料 → 执行模拟测试 → 总结应试技巧 → 参加正式考核
- **约束**: 备考必须充分；模拟必须真实；技巧必须实用
- **格式**: 考核评估报告（Markdown）

#### T3-3 证书颁发

- **必选组件**: 证书申请材料、申请流程、颁发时间、证书样本
- **可选组件**: 证书验证方式、电子证书、证书展示
- **组装顺序**: 准备申请材料 → 提交申请 → 等待审核 → 领取证书 → 验证真伪
- **约束**: 材料必须真实完整；流程必须规范；证书必须妥善保管
- **格式**: 证书颁发记录（Markdown）

#### T3-4 资质维护

- **必选组件**: 继续教育要求、更新周期、维护记录、续期流程
- **可选组件**: 继续教育资源、更新提醒、维护费用
- **组装顺序**: 了解维护要求 → 制定继续教育计划 → 执行学习 → 记录学时 → 办理续期
- **约束**: 学时必须达标；记录必须完整；续期必须及时
- **格式**: 资质维护记录（Markdown）

---

### 4 持续提升

#### T4-1 进阶学习

- **必选组件**: 进阶学习内容、学习资源、学习计划、学习成果
- **可选组件**: 高级课程、专家指导、学习社群
- **组装顺序**: 确定进阶方向 → 获取学习资源 → 制定学习计划 → 执行学习 → 产出成果
- **约束**: 方向必须符合职业规划；资源必须高质量；成果必须可展示
- **格式**: 进阶学习计划（Markdown）

#### T4-2 技能拓展

- **必选组件**: 相关技能清单、拓展计划、学习资源、整合应用
- **可选组件**: 跨领域学习、技能组合创新、复合型人才培养
- **组装顺序**: 识别相关技能 → 评估拓展价值 → 制定拓展计划 → 学习新技能 → 整合应用
- **约束**: 拓展必须有战略价值；学习必须高效；整合必须产生协同效应
- **格式**: 技能拓展方案（Markdown）

#### T4-3 经验积累

- **必选组件**: 项目实践、经验总结、案例库、个人品牌
- **可选组件**: 经验分享、知识输出、行业影响
- **组装顺序**: 参与项目实践 → 总结经验教训 → 建立案例库 → 输出分享 → 形成品牌
- **约束**: 实践必须真实有效；总结必须深入；输出必须有价值
- **格式**: 经验积累档案（Markdown）

#### T4-4 职业发展

- **必选组件**: 职业规划、发展路径、行动计划、成果展示
- **可选组件**: 职业咨询、人脉拓展、机会把握
- **组装顺序**: 评估职业现状 → 设定职业目标 → 规划发展路径 → 制定行动计划 → 执行并展示成果
- **约束**: 规划必须基于技能优势；目标必须现实可行；行动必须持续执行
- **格式**: 职业发展规划（Markdown）
