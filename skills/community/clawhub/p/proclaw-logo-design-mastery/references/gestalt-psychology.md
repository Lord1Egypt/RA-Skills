## 概览

格式塔心理学（Gestalt Psychology）是 20 世纪初在德国兴起的心理学派派，研究人类如何感知和组织视觉信息。

格式塔的核心观点是"整体大于部分之和"（The whole is greater than the sum of its parts），即人类倾向于将视觉元素组织成有意义的整体。

在 Logo 设计中，格式塔心理学帮助我们：
- 理解视觉组织机制
- 优化视觉层次和结构
- 提升认知效率
- 创造更有效的 Logo

## 格式塔原理详解

### 接近性原理

**定义：**

接近性原理（Principle of Proximity）指出，彼此接近的元素会被感知为一组或一个整体。

**视觉表现：**

```
[元素 A] [元素 B] [元素 C]  →  被感知为一组
[元素 D] [元素 E] [元素 F]  →  被感知为一组

[元素 A] [元素 B]
    [元素 C]  →  被感知为三个独立元素
    [元素 D] [元素 E]
```

**Logo 设计应用：**

- 相关元素靠近，形成视觉组
- 无关元素远离，形成独立单元
- 通过间距控制分组

**例子：**
- Google Logo：字母紧密排列，形成一个单词
- IBM Logo：条纹紧密排列，形成一个整体
- FedEx Logo：字母紧密排列，形成一个单词

### 相似性原理

**定义：**

相似性原理（Principle of Similarity）指出，相似的元素会被感知为一组或一个整体。

**相似性类型：**

**类型 1：形状相似**

- 相同或相似的形状
- 被感知为一组

**例子：**
- IBM Logo：相同形状的条纹
- Google Logo：相同字体的字母
- Target Logo：相同形状的圆形

**类型 2：色彩相似**

- 相同或相似的色彩
- 被感知为一组

**例子：**
- Google Logo：重复使用的色彩（两个 o 是相同的橙色）
- Microsoft Logo：四个不同的色彩方块，但整体形成一组
- Apple Logo：单色，所有元素色彩相同

**类型 3：大小相似**

- 相同或相似的大小
- 被感知为一组

**例子：**
- Google Logo：字母大小相同
- IBM Logo：条纹大小相同
- Target Logo：圆形大小相同

**类型 4：方向相似**

- 相同或相似的方向
- 被感知为一组

**例子：**
- IBM Logo：条纹方向相同（垂直）
- NBC Logo：羽毛方向不同，但整体形成一组
- Pepsi Logo：圆形方向不同，但整体形成一组

### 闭合性原理

**定义：**

闭合性原理（Principle of Closure）指出，人类倾向于将不完整的形状感知为完整的形状。

**视觉表现：**

```
[不完整圆形]  →  被感知为完整圆形
[不完整方形]  →  被感知为完整方形
[不完整三角形] →  被感知为完整三角形
```

**Logo 设计应用：**

- 使用不完整的形状，让大脑自动补全
- 创造参与感和趣味性
- 简化设计，降低复杂度

**例子：**
- NBC Logo：六个羽毛形成不完整圆形，被感知为完整圆形
- WWF Logo：熊猫的负空间形成不完整形状，被感知为完整熊猫
- Apple Logo：咬痕创造不完整圆形，被感知为完整圆形

### 连续性原理

**定义：**

连续性原理（Principle of Continuity）指出，人类倾向于沿着平滑的曲线和直线感知视觉元素，而不是突兀的转折。

**视觉表现：**

```
[平滑曲线]  →  被感知为连续的曲线
[突兀转折]  →  被感知为断裂的线条
```

**Logo 设计应用：**

- 使用平滑的曲线和直线
- 避免突兀的转折和断裂
- 创造流动和连贯的视觉体验

**例子：**
- Nike Swoosh：平滑的曲线
- Coca-Cola Logo：平滑的曲线
- Pepsi Logo：平滑的曲线

### 图形-背景原理

**定义：**

图形-背景原理（Figure-Ground Principle）指出，人类倾向于将视觉场景分为图形（焦点）和背景。

**视觉表现：**

```
[图形] [背景]  →  被感知为图形和背景
[图形1] [图形2]  →  被感知为两个图形
```

**Logo 设计应用：**

- 创造清晰的图形-背景关系
- 利用负空间创造图形
- 避免图形-背景混淆

**例子：**
- FedEx Logo：E 和 x 之间的负空间形成箭头（负空间即图形）
- WWF Logo：熊猫的负空间形成熊猫形状（负空间即图形）
- Apple Logo：苹果的负空间（即苹果形状）vs 背景

### 同域原理

**定义：**

同域原理（Principle of Common Region）指出，被同一区域包围的元素会被感知为一组或一个整体。

**视觉表现：**

```
[元素 A] [元素 B] [元素 C]  →  被感知为一组
    （在同一区域内）

[元素 D] [元素 E] [元素 F]  →  被感知为一组
    （在同一区域内）
```

**Logo 设计应用：**

- 使用边框或背景包围相关元素
- 创造视觉区域和分组
- 强化元素之间的关系

**例子：**
- Target Logo：圆形边框包围目标，形成整体
- Olympic Rings：五个圆环，每个圆环包围一个区域
- Pepsi Logo：圆形包围 Pepsi 文字，形成整体

### 简单性原理

**定义：**

简单性原理（Principle of Simplicity）指出，人类倾向于将复杂的视觉元素感知为最简单的形式。

**视觉表现：**

```
[复杂形状]  →  被感知为简单形状
[复杂图案]  →  被感知为简单图案
```

**Logo 设计应用：**

- 简化复杂形态为简单形状
- 追求简洁和清晰
- 降低认知负荷

**例子：**
- Apple Logo：复杂的苹果插画 → 简化的苹果图形
- Nike Swoosh：复杂的翅膀 → 简化的曲线
- Shell Logo：复杂的贝壳 → 简化的贝壳

### 经验原理

**定义：**

经验原理（Principle of Past Experience）指出，过去的经验和知识会影响视觉感知。

**视觉表现：**

```
[熟悉符号]  →  被快速识别
[不熟悉符号] →  被缓慢识别
```

**Logo 设计应用：**

- 使用熟悉的符号和文化元素
- 考虑目标受众的经验和背景
- 避免文化误解

**例子：**
- Apple Logo：苹果是熟悉的符号
- Nike Swoosh：曲线象征翅膀（需要文化背景）
- FedEx Logo：箭头是熟悉的符号

## 视觉组织法则

### 分组原则

**分组原则：**

基于格式塔原理，将视觉元素分组。

**分组方法：**

**方法 1：接近性分组**

- 相关元素靠近
- 无关元素远离
- 通过间距控制分组

**例子：**
- Google Logo：字母紧密排列，形成一个单词
- IBM Logo：条纹紧密排列，形成一个整体

**方法 2：相似性分组**

- 相似元素成组
- 不相似元素分开
- 通过形状、色彩、大小分组

**例子：**
- Google Logo：相同字体的字母成组
- IBM Logo：相同形状的条纹成组

**方法 3：闭合性分组**

- 不完整元素形成完整形状
- 被感知为一组
- 通过负空间分组

**例子：**
- NBC Logo：六个羽毛形成不完整圆形，被感知为一组
- WWF Logo：熊猫的负空间形成熊猫，被感知为一组

### 层次原则

**层次原则：**

基于格式塔原理，创造视觉层次。

**层次方法：**

**方法 1：大小层次**

- 主元素最大
- 次元素中等
- 辅助元素最小

**例子：**
- Google Logo：字母大小相同（无层次，强调平等）
- Apple Logo：苹果图形比文字大（图形为主）

**方法 2：色彩层次**

- 主元素色彩最强
- 次元素色彩中等
- 辅助元素色彩最弱

**例子：**
- FedEx Logo：Fed（紫色）vs Ex（橙色）色彩对比强烈
- Google Logo：四色对比强烈，创造层次

**方法 3：位置层次**

- 主元素在焦点位置
- 次元素在次要位置
- 辅助元素在边缘位置

**例子：**
- Target Logo：圆形在中心位置（焦点）
- Apple Logo：苹果图形在中心位置（焦点）

### 平衡原则

**平衡原则：**

基于格式塔原理，创造视觉平衡。

**平衡类型：**

**类型 1：对称平衡**

- 左右对称或上下对称
- 创造稳定和秩序

**例子：**
- Target Logo：左右对称
- Chanel Logo：左右对称
- Olympic Rings：左右对称

**类型 2：非对称平衡**

- 通过大小、位置、色彩调节平衡
- 创造动感和现代感

**例子：**
- Nike Swoosh：非对称平衡
- Apple Logo：非对称平衡（咬痕）
- FedEx Logo：非对称平衡（箭头）

**类型 3：放射平衡**

- 从中心向外辐射
- 创造动态和统一

**例子：**
- NBC Logo：从中心向外辐射
- Olympic Rings：从中心向外辐射
- Target Logo：从中心向外辐射

## 格式塔与认知效率

**认知效率：**

格式塔心理学强调认知效率，即如何以最小的认知负荷获取最多的信息。

**格式塔与认知效率的关系：**

**1. 降低认知负荷**

- 简化设计（简单性原理）
- 创造分组（接近性、相似性原理）
- 创造层次（大小、色彩、位置层次）

**2. 提升识别速度**

- 使用熟悉符号（经验原理）
- 创造清晰分组（接近性、相似性原理）
- 创造清晰层次（大小、色彩、位置层次）

**3. 增强记忆性**

- 创造参与感（闭合性原理）
- 创造惊喜（图形-背景原理）
- 创造趣味（连续性原理）

**格式塔优化的评估标准：**

**标准 1：认知负荷是否合理？**

- 元素是否过多？
- 元素是否过于复杂？
- 元素是否过于混乱？

**标准 2：识别速度是否快速？**

- 3 秒内能否识别？
- 1 秒内能否识别？
- 0.5 秒内能否识别？

**标准 3：记忆性是否强？**

- 能否记住 Logo？
- 能否记住 Logo 的特征？
- 能否记住 Logo 的意义？

**格式塔优化的方法：**

**方法 1：应用接近性原理**

- 相关元素靠近
- 无关元素远离
- 通过间距控制分组

**方法 2：应用相似性原理**

- 相似元素成组
- 不相似元素分开
- 通过形状、色彩、大小分组

**方法 3：应用闭合性原理**

- 使用不完整的形状
- 让大脑自动补全
- 创造参与感和趣味性

**方法 4：应用连续性原理**

- 使用平滑的曲线和直线
- 避免突兀的转折和断裂
- 创造流动和连贯的视觉体验

**方法 5：应用图形-背景原理**

- 创造清晰的图形-背景关系
- 利用负空间创造图形
- 避免图形-背景混淆

**方法 6：应用简单性原理**

- 简化复杂形态为简单形状
