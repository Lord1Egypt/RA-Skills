# C4 数据分析与决策 — 详细单元规格

## C4-01 基础数据处理与可视化

| 字段 | 内容 |
|------|------|
| **输入** | 必选：原始数据或数据源描述；可选：分析维度、图表偏好 |
| **输出** | 数据处理报告（清洗后数据+统计摘要+可视化图表），Markdown |
| **依赖** | 无 |
| **AI自治度** | ⬛ 全自动 |
| **组合接口** | →C4-05, →C4-06, →C4-08, →C4-19, →C4-31, →C4-40 |

**执行要点**：
1. 数据清洗（缺失值/异常值/格式统一）
2. 描述性统计（均值/中位数/分布/趋势）
3. 选择合适的可视化方式
4. 标注数据质量和局限
