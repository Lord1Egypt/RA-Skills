# 儿童期前收缩的分类、诊断与治疗策略

## 使用前提
- 已完成12导联心电图检查
- 已评估是否存在器质性心脏病

## 禁忌场景
- 心悸由非心律失常原因引起（如焦虑、甲状腺功能亢进）
- 单纯偶发、无症状的期前收缩

## 执行步骤

### 1. 分类依据
根据异位起搏点位置，将期前收缩分为：
- 房性期前收缩
- 交界性期前收缩
- 室性期前收缩（最常见）

### 2. 心电图诊断标准
- **房性期前收缩**：提前P'波（可与前一心动T波重叠）、P'R间期正常、代偿间歇不完全；若QRS宽大畸形，提示心室内差异传导。
- **交界性期前收缩**：QRS提前且形态正常，QRS前后可见逆行P'波（P'R < 0.10秒），常与QRS重叠，代偿间歇多不完全。
- **室性期前收缩**：QRS提前、其前无P波、QRS宽大畸形、T波与主波方向相反，代偿间歇多完全。

### 3. 病因评估
- **无器质性心脏病**：疲劳、精神紧张、自主神经不稳定
- **器质性心脏病**：心肌炎、先天性心脏病、风湿性心脏病
- **其他诱因**：药物（洋地黄、奎尼丁等）、电解质紊乱（低钾）、缺氧、酸碱失衡、心导管或心脏手术

### 4. 治疗决策
- **无需治疗**：无症状、期前收缩次数少、运动后减少或消失、形态一致
- **需药物治疗**：有自觉症状；存在器质性心脏病基础上出现；心电图呈多源性或频发联律
  - **房性**：首选普罗帕酮或普萘洛尔；无效时考虑洋地黄
  - **室性**：急性期用利多卡因；口服可选美西律或莫雷西嗪
- **同时治疗原发病**（如心肌炎、电解质紊乱等）

### 5. 预后判断
多数为良性，部分可持续多年但可自行消退；药物治疗通常效果良好。
