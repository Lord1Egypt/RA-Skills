# 妇产科学核心技能合集

本技能合集基于权威教材《8F279867-Cfda-4144-Bab7-B1Fe2C18C363》，系统整合了妇产科学领域共187项临床与学术核心技能，涵盖女性全生命周期健康管理、妊娠与分娩管理、妇科肿瘤诊疗、生殖内分泌疾病、辅助生殖技术、围产期并发症处理、感染性疾病防控、盆底功能障碍、遗传咨询与产前诊断等多个关键方向。内容兼顾国际指南与中国临床实践（如PCOS中国2011诊断标准），适用于医学生、住院医师、专科医师及临床研究人员快速查阅与技能提升。
