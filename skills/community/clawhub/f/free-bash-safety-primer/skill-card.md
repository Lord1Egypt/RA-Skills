## Description: <br>
Understand the attack surface created when an OpenClaw agent executes shell commands autonomously, including obfuscation, injection, encoding attacks, and the ClawHavoc incident. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and operators use this primer to understand bash execution risks when agents run shell commands autonomously. It helps readers recognize command obfuscation, substitution injection, encoding attacks, shell-specific escape vectors, and persistence risks before enabling or installing shell-capable skills. <br>

### Deployment Geography for Use: <br>
Global <br>
