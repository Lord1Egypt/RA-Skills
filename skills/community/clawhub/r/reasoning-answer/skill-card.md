## Description: <br>
Provides detailed reasoning-based answers using the user's language by default, with clickable citation footnotes when referencing sources. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users and agents use this skill to shape answers so they preserve the user's language by default and apply sequential clickable citations when sources are cited. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Citation-format preferences may conflict with citation rules already enforced by the agent or platform. <br>
Mitigation: Review generated answers for the required citation style and resolve conflicts with platform citation policies before relying on sourced responses. <br>
