# 美学流派完整矩阵

每个流派从 6 个维度定义：色调、构图、材质、线条、情绪、代表作品。

---

## 1. 巴洛克 (Baroque) 1600-1750

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 深色背景 + 戏剧性光影（明暗对照法），暖金、深红、绀青 |
| 📐 构图 | 对角线布局、动感螺旋、开放式边界 |
| 🧱 材质 | 大理石、镀金、天鹅绒、油画肌理 |
| ✏️ 线条 | 流动曲线、剧烈扭曲、无直线 |
| 😌 情绪 | 戏剧、奢华、力量、宗教狂喜 |
| 🖼 代表 | 卡拉瓦乔、鲁本斯、贝尼尼 |

**龙虾应用**：暗色背景中一束顶光打在龙虾金色甲壳上，螯呈攻击姿态，像卡拉瓦乔的画。

---

## 2. 新艺术 (Art Nouveau) 1890-1910

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 柔和自然色：枯叶黄、苔绿、赭石、乳白、金属铜 |
| 📐 构图 | 非对称、S形曲线、有机边框 |
| 🧱 材质 | 彩色玻璃、锻铁、木雕、珐琅 |
| ✏️ 线条 | 标志性的"鞭曲线"：柔韧、流畅、像藤蔓 |
| 😌 情绪 | 优雅、自然崇拜、梦幻 |
| 🖼 代表 | 慕夏、高迪、吉马德 |

**龙虾应用**：龙虾的触须如藤蔓般蜿蜒形成装饰画框，甲壳上有植物纹样镶嵌，慕夏式的女性化优雅。

---

## 3. 浮世绘 (Ukiyo-e) 17-19世纪

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 平面色块：朱红、藏青、墨黑、米白、铜绿 |
| 📐 构图 | 大胆裁切、俯瞰视角、不对称留白 |
| 🧱 材质 | 木版印刷、和纸纹理、墨色渐变 |
| ✏️ 线条 | 流畅有力的轮廓线，粗细变化 |
| 😌 情绪 | 宁静、瞬间之美、尘世享乐 |
| 🖼 代表 | 葛饰北斋、歌川广重 |

**龙虾应用**：巨浪之下，一只朱红龙虾以侧影呈现，如北斋《神奈川冲浪里》的风格。

---

## 4. 极简主义 (Minimalism) 1960s-

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 单色或极低饱和：白、灰、黑、一个强调色 |
| 📐 构图 | 几何中心化、对称或网格、大量留白 |
| 🧱 材质 | 光滑表面、工业材料、抛光金属 |
| ✏️ 线条 | 直线、直角、精确几何 |
| 😌 情绪 | 安静、精确、自足、排斥装饰 |
| 🖼 代表 | 唐纳德·贾德、安藤忠雄 |

**龙虾应用**：白色空间里，一只龙虾简化为三条几何曲线——螯的弧、尾的弧、触须的直线。

---

## 5. 装饰艺术 (Art Deco) 1920-1940

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 华丽对比：金色+黑色、银+钴蓝、翡翠+象牙 |
| 📐 构图 | 对称、阶梯状、放射线、几何重复 |
| 🧱 材质 | 漆面、铬、玻璃、象牙、稀有木材 |
| ✏️ 线条 | 锐利几何之字纹、V形、阶梯线 |
| 😌 情绪 | 奢华、乐观、摩登、财富 |
| 🖼 代表 | 克莱斯勒大厦、塔玛拉·德·蓝碧嘉 |

**龙虾应用**：龙虾以对称构图呈现，甲壳上有金色和黑色的阶梯状几何纹样，如克莱斯勒大厦的尖顶。

---

## 6. 包豪斯 (Bauhaus) 1919-1933

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 三原色+黑白：红、黄、蓝、黑、白 |
| 📐 构图 | 网格、不对称平衡、功能导向布局 |
| 🧱 材质 | 钢材、玻璃、混凝土、胶合板 |
| ✏️ 线条 | 直线、圆、弧线的几何组合 |
| 😌 情绪 | 理性、现代、民主、反装饰 |
| 🖼 代表 | 格罗皮乌斯、康定斯基、莫霍利-纳吉 |

**龙虾应用**：龙虾由红、黄、蓝的矩形和圆形构成，像一个康定斯基的构成画，每个肢体都是功能性的几何体块。

---

## 7. 构成主义 (Constructivism) 1915-1930s

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 革命红、黑、白，少量金属灰 |
| 📐 构图 | 对角线、不对称、动感、空间穿透 |
| 🧱 材质 | 工业材料：钢铁、玻璃、塑料 |
| ✏️ 线条 | 锐利、斜线、交叉结构 |
| 😌 情绪 | 革命性、动态、前卫、功能至上 |
| 🖼 代表 | 塔特林、罗德琴科、利西茨基 |

**龙虾应用**：龙虾以红与黑的几何抽象体块呈对角线飞越空间，结构暴露在外，如塔特林塔。

---

## 8. 超现实主义 (Surrealism) 1920s-1950s

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 梦境色：褪色蓝、不自然的绿、对比鲜明的阴影 |
| 📐 构图 | 非理性并置、透视扭曲、无限空间 |
| 🧱 材质 | 一切材质都可拼接 |
| ✏️ 线条 | 柔和或锐利、按梦境逻辑变化 |
| 😌 情绪 | 不安、奇异、梦幻、潜意识 |
| 🖼 代表 | 达利、马格利特、恩斯特 |

**龙虾应用**：一只融化后挂在枯树枝上的龙虾，旁边有一只怀表在滴落——致敬达利的同时确立龙虾的超现实身份。

---

## 9. 赛博朋克 (Cyberpunk) 1980s-

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 霓虹粉/蓝/紫 + 暗黑背景，高饱和对比 |
| 📐 构图 | 高角度、断层、信息过载式的密集排布 |
| 🧱 材质 | 金属、塑料、发光屏幕、义肢 |
| ✏️ 线条 | 锐利、断裂、故障式的错位 |
| 😌 情绪 | 反乌托邦、高科技低生存、街头叛逆 |
| 🖼 代表 | 《银翼杀手》《攻壳机动队》 |

**龙虾应用**：机械义肢龙虾在霓虹雨夜的街头，甲壳上反射着全息广告，一只复眼是红色LED。

---

## 10. 波普艺术 (Pop Art) 1950s-1960s

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 高饱和原色：亮红、明黄、钴蓝，平涂无渐变 |
| 📐 构图 | 重复、网格排列、放大至不自然 |
| 🧱 材质 | 印刷网点、塑料、画布 |
| ✏️ 线条 | 硬边、黑色粗轮廓、漫画感 |
| 😌 情绪 | 通俗、讽刺、大胆、调侃精英 |
| 🖼 代表 | 沃霍尔、利希滕斯坦 |

**龙虾应用**：沃霍尔式四色丝网印刷龙虾，每只颜色不同，像超市货架上的商品。

---

## 11. 蒸汽波 (Vaporwave) 2010s-

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 粉紫渐变、水蓝、荧光粉、哑光霓虹 |
| 📐 构图 | 居中、对称、古典雕塑被数字化切割 |
| 🧱 材质 | CRT屏幕像素、VHS噪点、3D渲染的塑料感 |
| ✏️ 线条 | 柔和 + 数字故障的锯齿 |
| 😌 情绪 | 怀旧、讽刺、消费主义批判、迷茫美好 |
| 🖼 代表 | 网络迷因文化、Windows 95美学复兴 |

**龙虾应用**：一只古希腊大理石龙虾雕像漂浮在粉紫渐变背景上，周围有Windows 95弹窗和棕榈树剪影。

---

## 12. 酸性图形 (Acid Graphics) 1990s-

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 超饱和荧光色：霓虹绿、热粉红、毒紫、电蓝 |
| 📐 构图 | 扭曲、熔化、旋转、反透视 |
| 🧱 材质 | 液态金属、半透明塑料、全息箔 |
| ✏️ 线条 | 任意变形、流动、粘稠弯曲 |
| 😌 情绪 | 迷幻、混乱、反叛、感官超载 |
| 🖼 代表 | 《The Face》杂志、雷鬼海报、MVM |

**龙虾应用**：龙虾在荧光绿到热粉红的渐变熔液中扭动，甲壳呈液态全息质感，背景像溶解的九宫格。

---

## 13. 生物朋克 (Biopunk) 1990s-

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 生物荧光绿/蓝、组织粉、半透明琥珀色 |
| 📐 构图 | 解剖式、生长式有机结构、反几何 |
| 🧱 材质 | 生物组织、透明凝胶、骨骼、发光菌丝 |
| ✏️ 线条 | 有机曲线、血管状分枝 |
| 😌 情绪 | 身体恐怖、科技敬畏、生命与机器模糊 |
| 🖼 代表 | 《异形》、H.R.吉格尔、基因工程美学 |

**龙虾应用**：龙虾的甲壳半透明，内部发光的生物电路可见，右螯是组织与钛合金的融合体。

---

## 14. 侘寂 (Wabi-sabi) 日本美学

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 大地色：枯茶、灰褐、暗绿、墨白 |
| 📐 构图 | 不对称、留白、强调短暂与不完美 |
| 🧱 材质 | 粗陶、木头、纸张、生锈金属 |
| ✏️ 线条 | 自然不规则、手工感 |
| 😌 情绪 | 静谧、接受无常、残缺之美 |
| 🖼 代表 | 千利休茶道、金缮修复 |

**龙虾应用**：一只断了一边螯的老龙虾静卧在粗陶盘上，断口处有金缮修复的金线，岁月和伤痕都可见。

---

## 15. 生态美学 / 超有机 (Eco / Bio-Organic)

| 维度 | 特征 |
|------|------|
| 🎨 色调 | 苔绿、藻褐、珊瑚粉、海洋蓝、矿物灰 |
| 📐 构图 | 有机物与无机物共生、层叠、缓慢生长感 |
| 🧱 材质 | 活体苔藓、珊瑚、菌丝体、回收电路板 |
| ✏️ 线条 | 有机+几何混合，生物与人工的界限模糊 |
| 😌 情绪 | 共生、再生、对未来的谨慎希望 |
| 🖼 代表 | Neri Oxman的生态学设计、生物建筑 |

**龙虾应用**：龙虾壳上覆盖着荧光珊瑚和电路板苔藓，废弃芯片嵌入甲壳成为保护色，像海洋自己长出了计算机。
