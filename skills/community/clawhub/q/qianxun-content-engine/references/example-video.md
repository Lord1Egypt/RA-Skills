## 三、文案

### 9. 参考视频旁白文案

```
我有一个不一样的老板叫古姐。

她是七七年的，却在深圳南山开了一家两千平的新中式美学生活馆。

我问她为什么选择做新中式，她说想为中国女人做一件能穿出底气，经得起时间的好衣服。

从香云纱的厚度，宋锦的纹理，到版型的每一个弧度，她都不放过，坚持亲自审样，亲自把关。

衣服是跟着人走的，一定要舒服，更要称心。

开业这两个多月，她仍然每天在店里来回走，反复看，摸一摸面料，理一理陈列。

她说只有泡在店里，才能听见姐妹们真正需要什么。

她始终相信，把细节做好，把品质做实，才不辜负每一份信任，也不辜负这二十五年来对服装的执着。
```

### 10. 参考视频旁白文案逻辑分析

**第一层 · 建立反差与好奇（0-12 秒）**
开头抛出"75 后"和"2000 平"两个核心数字，用年龄与事业规模的巨大反差，迅速塑造"不一样"的创始人形象。这个钩子激发好奇心，让观众想知道"古姐到底是谁、为什么能做成"。

**第二层 · 传递匠心与价值（12-28 秒）**
通过"想为中国女人做一件能穿出底气的好衣服"和"坚持亲自审样"等细节，塑造创始人专注、专业的匠人精神。传递产品核心价值——不只是衣服，更是品质、专业和对女性的关怀，建立观众的信任感。

**第三层 · 落脚情感与共鸣（28 秒-结尾）**
从产品本身上升到情感层面。"泡在店里才能听见姐妹真正需要什么"，点明品牌"以用户为中心"的理念。情感连接让目标客户（独立、有阅历的女性）产生共鸣，从认可产品转变为认同品牌文化，完成叙事升华。

**关键设计**：
- 反差数字开场——3 秒抓住注意力
- 数字 + 工艺词（香云纱 / 宋锦 / 版型）——把"匠心"从形容词变成可信细节
- 时间锚点（"二十五年"）——把"专业"从主张变成事实

## 四、可学习要点（双模式对比 — 用来训练 agent 区分两种 mode 的输出形态）

> ⚠️ 本范例**同时展示**「客观拆解」和「品牌视角」两种模式下"可学习要点"该长什么样。实际拆解只输出其中一种（按 Step 0 graph 状态判断）。

### 模式 B · 客观拆解（graph 未填充）

每条 bullet 含「**现象 → 推论 → 建议**」三段：

1. **数字反差是开头 3 秒的最强钩子**（现象：本条用 75 后 + 2000 平双数字）→（推论：年龄/规模/时间/身份等具体数字制造反差，比形容词更抓注意力）→（建议：人设视频开头 3 秒至少埋 1 个具体数字反差）
2. **匠心要用工艺名词承载**（现象：本条用"香云纱的厚度、宋锦的纹理、版型的弧度"）→（推论：物质名词比形容词可信度高 10 倍）→（建议：所有"高级感"主张都要换成 1-2 个具体材质/工艺词）
3. **情感升华句留在最后 5 秒**（现象："不辜负二十五年对服装的执着"在结尾 4 秒）→（推论：前面建立信任，结尾落到"为谁做"才能激活共鸣）→（建议：升华句不能在中间，要压到最后 1/8 时长）
4. **人设视频不需要喊口号 CTA**（现象：评论区"联系方式""有机会一定去"自然出现）→（推论：人设建立到位转化会自来）→（建议：把口号 CTA 砍掉，给"留白结尾"和"画面收束"让位）

### 模式 A · 品牌视角（graph 已填充，假设示范配置如下）

> 假设 `graph/brand/brand-story.md` 写了主理人具体经历（如「{{N 年女装运营 / 跨界经历 / N+ 用户运营经验}}」等可复用素材）；`graph/audience/segments.md` 定义了主力客群层（如「客群 A · 35+ 高审美独立女性」）。

1. **主理人的"数字反差"可直接复用**——做我方主理人视频时，开头同样抛"硬数字 + 反差身份"（如"做了 N 年 X，第一次跨界做 Y 就跑出 Z"），类比古姐的 75 后 × 2000 平结构
2. **品类专属的"匠心承载词"可类比香云纱/宋锦**——把我方品类的具体材质 / 工艺 / 方法论名词作为高密度专业词替换抽象形容词（如珠宝品类用"古法金 / 拉丝 / 镂空"，软件品类用"链路 / 节点 / pipeline"）
3. **客群 A 喜欢的不是促销 CTA 是"信任补全"**——评论区置顶应放"工艺/数据/案例"佐证，而不是"快冲/限时"
4. **可学的人设公式**：开头 3 秒抛硬数字反差 → 中段展示具体方法/工艺 → 结尾"为谁做"（一句话点明使命，承接 brand-voice 的价值观关键词）

---

**两种模式的根本差异**：
- **客观模式**输出的是"通用方法论"——可作为内容策划手册的备忘
- **品牌视角模式**输出的是"立刻可执行的下条内容大纲"——直接喂给 generate mode 当输入

填 graph 的 ROI 在这里直接体现。
