# 酶决定代谢反应途径与去向

## 何时使用
- 分析遗传性代谢疾病中因酶缺陷导致的代谢阻断
- 预测外源底物（如药物、营养素）在细胞内的转化路径
- 解释为何同一底物在不同组织或条件下产生不同代谢产物
- 设计代谢工程策略时评估关键酶对通量分布的影响

## 执行步骤
1. **确认底物可进入细胞**：验证原初底物A是否可通过膜转运系统（如具酶活性的TA）进入胞内。
2. **识别关键催化酶**：列出参与该代谢网络的所有特异性酶（如E_AB、E_BC、E_CD等），明确其对应催化步骤。
3. **评估酶存在性与活性**：基于实验数据、基因型信息或调控条件，判断各酶是否表达且具有功能活性。
4. **推导可行路径**：
   - 若E_AB、E_BC、E_CD均存在且有活性，则主路径 A → B → C → D 可行；
   - 若其中任一关键酶缺失或失活，则主路径中断；
   - 若存在替代酶（如催化A→F的酶），则旁路 A → F → G 可能被激活。
5. **输出路径选择结果**：明确指出实际发生的代谢路径及其终产物，强调酶是决定代谢流向的核心因素。

> 注意：本技能仅适用于酶促反应，不适用于非酶促反应或单纯物理扩散过程。
