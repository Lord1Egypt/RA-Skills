# 心房颤动的听诊诊断标准

## 何时使用
- 对成年患者进行心脏听诊时，怀疑存在心律失常
- 听诊中观察到心律明显不规则
- 需在心电图检查前进行初步临床判断

## 执行步骤
1. **规范进行心脏听诊**：使用听诊器在心尖区清晰听取心音，重点评估以下三项特征：
   - **心律规则性**：判断心跳间隔是否“绝对不规则”（即完全无周期性、无规律可循）；
   - **第一心音（S₁）强度**：注意S₁是否呈现“强弱不等”（因每次心室充盈量不同导致收缩力度差异）；
   - **脉搏短绌**：同步触诊桡动脉（或其他外周动脉）计数脉率，同时听诊计数心率，确认脉率是否显著低于心率。

2. **综合判断**：若上述三项特征同时存在，则发出“高度怀疑心房颤动”警报。

3. **后续行动**：立即建议行12导联心电图检查以确诊，并评估是否需紧急干预。

## 注意事项
- 本技能仅用于初步筛查，**不能替代心电图确诊**；
- 不适用于窦性心律不齐等生理性节律变化；
- 听诊无法区分心房颤动与其他导致绝对不规则心律的罕见心律失常（如多源性房性心动过速），需结合病史与辅助检查综合判断。
