# 免疫组织化学染色阳性信号定位判读

## 何时使用
- IHC染色已完成，且阳性/阴性对照结果符合预期
- 显色系统为HRP-DAB等常规方法
- 观察到清晰、低背景、定位明确的染色信号
- 需排除内源性酶干扰和非特异性背景染色

## 判读步骤

1. **确认染色质量**：确保信号清晰、背景低、定位准确，且对照有效。

2. **按亚细胞定位判读阳性信号**：
   - **细胞膜阳性**：着色局限于细胞膜（如CD系列、E-cadherin）。
   - **细胞质阳性**：
     • 近膜胞质阳性（如CK、Hep Par-1、vimentin）；
     • 弥漫性胞质阳性（如Bcl-2）。
   - **细胞核阳性**：信号位于细胞核内（如Ki-67、TTF-1、ER、PR）。

3. **识别复合表达模式**：部分抗原（如EMA、CD15、CD30）可同时呈现多种定位模式，需综合判断。

4. **结合抗体生物学特性**：仅依据已知抗原的典型亚细胞定位判读，避免将非特异性沉积误判为阳性。

5. **支持半定量分析**：可结合图像分析系统（如激光扫描共聚焦）提高定位精确度，并记录信号强度等级。

> 注意：非特异性着色、内源性酶干扰或背景染色不得作为阳性判读依据。
