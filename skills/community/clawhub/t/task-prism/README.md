# Task Prism · 任务拆解与技能画像专家

> 像棱镜折射白光一样，将模糊复杂的业务需求分解为清晰的十二维可执行方案。

## ✨ 核心能力

- **十二维拆解**：假设 → WBS → 技能 → 干系人 → 沟通 → 岗位 → 流程 → 交付 → 成本 → 风险 → 变更 → 复盘
- **四种工作模式**：快速模式 / 完整模式 / 敏捷模式 / 跨国跨文化模式
- **专业方法论**：PERT 三点估算 · CPM 关键路径 · RACI 矩阵 · MoSCoW 优先级 · SMART 校验 · KPT 复盘
- **质量自检**：15 项核心检查 + 按模式扩展（敏捷/跨国/完整），聚焦实际关键点
- **Delta 修订模式**：支持基于上轮输出的增量更新，无需全量重生成

## 🚀 快速开始

在 AI Agent 中安装后，直接对 AI 说：

- `"帮我拆解这个项目的任务"`
- `"做个项目计划，预算50万，3个月"`
- `"用敏捷模式帮我拆解一个官网改版项目"`

## 🔧 四种工作模式

| 模式 | 触发场景 | 特点 |
|:-----|:---------|:-----|
| **快速模式** | 需求已明确，直接拆解 | 跳过澄清，十维直出 |
| **完整模式** | 需求有模糊点 | 先澄清再拆解，两阶段 |
| **敏捷模式** | 软件产品 / 需求不确定 | Sprint 周期拆解，可演示增量 |
| **跨国模式** | 多国协作 / 跨时区 / 多法规 | 时区协调 + 法规合规 + 文化适配 |

## 📊 方法论覆盖

| 方法 | 用途 | 所在维度 |
|:-----|:-----|:---------|
| PERT 三点估算 | 工期估算（乐观/最可能/悲观） | WBS |
| CPM 关键路径 | 识别零浮动任务链 + 压缩选项 | WBS |
| RACI 矩阵 | 责任分配（执行/负责/咨询/知会） | 岗位匹配 |
| MoSCoW | 多目标冲突优先级排序 | 需求澄清 |
| SMART | 目标声明五维校验 | 需求澄清 |
| Power/Interest Grid | 干系人分类管理 | 干系人分析 |
| Escalation Matrix | 信息升级路径（L1-L4） | 沟通管理 |
| CCB 变更控制 | 正式变更请求流程 | 变更管理 |
| KPT 模型 | 项目复盘（Keep/Problem/Try） | 复盘框架 |
| Hofstede 模型 | 跨文化沟通适配 | 跨国模式 |
