# 麦角生物碱类药物用于产后子宫止血与复旧

## 适用场景
仅限以下情况使用：
- 患者已分娩（产后状态）
- 存在子宫收缩乏力引起的产后出血或子宫复旧延迟

## 执行步骤
1. **确认适应证**：确保患者处于产后状态，且出血或复旧问题由子宫收缩乏力引起。
2. **选择药物**：优先选用麦角新碱或甲麦角新碱，二者对子宫平滑肌具有选择性兴奋作用，起效快、作用强。
3. **理解药理机制**：稍大剂量即可引起子宫体与子宫颈强直性收缩，通过机械压迫血管实现止血。
4. **严格排除禁忌**：
   - 绝对禁用于催产或引产（可致胎儿窘迫甚至死亡）
   - 妊娠毒血症产妇慎用
   - 血管硬化、冠心病患者忌用
5. **监测不良反应**：用药后观察是否出现恶心、呕吐、血压升高；警惕过敏反应（如呼吸困难、血压骤降）。
6. **避免长期使用**：含麦角胺制剂（如麦角流浸膏）长期使用可损伤血管内皮，引发肢端干性坏疽或血栓。

## 关键注意事项
- 妊娠后期子宫对此类药物敏感性显著增强，故仅限产后使用。
- 剂量控制至关重要：稍大剂量即可能引发强直收缩，需严格遵医嘱。
- 使用前必须评估患者心血管状况及妊娠并发症史。
