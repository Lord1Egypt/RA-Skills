# 微信AI接入顾问

> 微信小程序AI怎么接，先别急着写代码

## 这是什么

帮小程序商家、运营团队和开发者理解微信小程序 AI，选择开发模式/自动模式，把商品、服务、门店、下单、预约、支付、配送、会员等能力整理成 AI 能理解、能调用、能推荐的结构化形态。

直接说你的行业和业务，就能拿到接入方案、行业案例、GEO 数据清单、开发模式三件套草案、完整代码骨架、测试矩阵和上线检查。

## 快速开始

在 Claude 或 WorkBuddy 中安装后，直接对话即可：

```
我是奶茶店，有5个门店，怎么接微信AI？
帮我生成完整项目代码
帮我检查 mcp.json 规范
```

## 核心能力

| 能力 | 说明 |
|------|------|
| AI 能力清单 | 原子接口、原子组件、半屏页面、知识库、文字链、交互API |
| 12个行业案例 | 咖啡奶茶、本地团购、预约服务、零售电商、会员优惠、医疗健康、教育培训、政务办事、汽车服务、宠物服务、家政保洁、婚庆摄影 |
| 代码骨架生成 | app.json + AGENTS.md + SKILL.md + mcp.json + index.js + apis/ + components/ + page-meta.json |
| GEO 字段补全 | 别名、场景词、标签、Agent 可理解描述、质量评分 |
| 最佳实践检查 | 信息源注意力权重、接口description写法、content事实+动作两段式 |
| 测试矩阵 | 20条测试问题、失败兜底、上线检查 |
