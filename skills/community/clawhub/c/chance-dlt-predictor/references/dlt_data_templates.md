## 五、常用快捷指令说明

用户可直接发送以下指令触发对应功能：

| 指令 | 触发功能 |
|------|---------|
| `选5注` / `推荐5注` | 生成5注智能选号 |
| `稳健型` / `均衡型` | 使用稳健选号策略 |
| `冷号型` / `激进型` | 使用冷号回补策略 |
| `遗漏分析` | 输出当前遗漏值分析（需历史数据） |
| `频率分析` | 输出历史热号/冷号统计 |
| `过滤缩水` + 号码 | 对指定复式号码执行缩水过滤 |
| `大乐透规则` | 输出玩法规则和奖级说明 |
| `蒙特卡洛` | 运行蒙特卡洛模拟选号算法 |
| `代码` / `Python代码` | 输出完整Python分析代码 |

## 六、注意事项清单

选号前检查以下7项：

- [ ] 和值在 75-115 区间
- [ ] 奇数个数 1-4个（非全奇/全偶）
- [ ] 大号个数 1-4个（非全大/全小）
- [ ] 三区（01-11, 12-23, 24-35）各有≥1个号码
- [ ] 跨度（最大号-最小号）在 15-30 之间
- [ ] 连号不超过1组2连（无3连）
- [ ] 后区两球差值 2-9

通过全部7项 → 优质注数 ✅
