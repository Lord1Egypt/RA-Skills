# 膀胱尿道镜检查适应症与禁忌证评估

## 何时使用
当出现以下任一临床需求时，启动本评估流程：
- 需对尿道或膀胱内部进行直接可视化检查；
- 需获取膀胱或尿道组织进行病理活检；
- 需经输尿管口插入导管实施逆行肾盂造影；
- 需收集肾盂尿液送检；
- 需行输尿管套石术或放置双J形输尿管支架；
- 需使用电切镜进行尿道、膀胱或前列腺的治疗性手术（如肿瘤切除）。

## 执行步骤

### 1. 确认前提条件
确保患者满足以下基本前提：
- 血流动力学稳定；
- 无活动性泌尿道感染；
- 膀胱容量足够（通常 ≥100 mL）。

### 2. 排查绝对禁忌证
若存在以下任一情况，**禁止实施**膀胱尿道镜检查：
- **尿道狭窄**：无法顺利插入镜体，强行操作可致尿道破裂；
- **急性膀胱炎**：易加重感染或诱发严重出血；
- **膀胱容量过小**（<100 mL）：视野受限，穿孔风险显著增加。

### 3. 操作注意事项
- 插入前充分润滑镜体；
- 动作轻柔，避免暴力推进；
- 若遇明显阻力，立即停止操作并重新评估解剖条件。

### 4. 特殊说明
电切镜属于膀胱尿道镜的延伸应用，适用于复杂治疗性操作，但仍受上述相同禁忌证限制。

## 输出结果
- 若存在任一绝对禁忌证，发出“禁止检查”警告；
- 若符合所有适应症且无禁忌证，可安全实施检查以达成诊断或治疗目的。
