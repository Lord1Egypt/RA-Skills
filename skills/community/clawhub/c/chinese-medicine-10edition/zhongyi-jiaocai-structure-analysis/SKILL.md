# Zhongyi Jiaocai Structure Analysis

## When to Use
- 正在使用《中医学》第10版进行自学或教学
- 需要快速掌握教材整体架构以制定学习/授课计划
- 想了解教材如何体现“中西医并重”及新时代中医药发展成果
- 对教材内容范围或局限性存在疑问

## Execution Steps

### 1. 理解三篇结构
**上篇：中医基础理论**
- 聚焦中医学源流、哲学基础（如阴阳五行）与核心理论体系（藏象、经络、病因病机等）
- 强调“原汁原味”传承传统理论

**中篇：方药与针灸**
- 系统介绍中药、方剂及针灸疗法
- 突出历史成就与现代创新发展成果
- 强调治疗手段的科学性与实用性

**下篇：临床部分**
- 聚焦中医药优势病种（如肿瘤、风湿、消化病等）
- 病种分类注重与现代医学知识体系衔接
- 提升系统化临床思维能力

### 2. 掌握编写宗旨
- 继承前九版精华，吸纳师生反馈
- 融入新时代中医药新知识、新实践、新成果
- 贯彻“中西医并重”国家战略
- 配套《中医学习指导与习题集》（第4版）及数字资源，强化学习效果

### 3. 注意局限性
- 本教材不适用于非临床医学专业学生
- 不涵盖全部中医药学体系，仅为入门与核心框架
- 鼓励用户提供教学反馈以持续完善内容

## Output
构建系统、规范、与时俱进的中医学知识框架，支撑医学生掌握中医药基本能力。
