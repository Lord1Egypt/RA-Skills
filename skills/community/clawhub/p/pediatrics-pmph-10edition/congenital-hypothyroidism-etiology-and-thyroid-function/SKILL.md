# 先天性甲状腺功能减退病因与甲状腺激素作用

## 适用场景
- 新生儿甲状腺功能筛查异常
- 存在发育迟缓、黄疸延长、喂养困难等症状

## 执行步骤
1. **理解甲状腺激素生理作用**：
   - 促进胎儿及婴儿脑发育（不可逆）；
   - 促进生长、骨发育、蛋白质合成、糖脂代谢；
   - 影响消化（不足致便秘）、循环（过多致心动过速）、肌肉（过多致震颤）。
2. **先天性甲减病因分类**：
   - **散发性**（90%为甲状腺发育异常，如缺如、异位）；
   - **激素合成障碍**（常染色体隐性遗传）；
   - **中枢性**（TSH/TRH缺乏，常合并其他垂体激素缺乏）；
   - **暂时性**（母亲抗甲状腺药物或抗体影响）。
3. **地方性甲减**：孕妇缺碘所致，随碘盐普及已少见。

## 输出结果
提示需进行甲状腺超声、基因检测或母亲病史追溯以明确病因，并强调早期替代治疗对脑发育的关键性。
