# 文献综述写作指南

## 一、文献综述不是「书单」

**错误示范**：
```
关于这个话题，A 研究了... B 研究了... C 研究了...
```

**正确示范**：
```
现有研究主要围绕三个维度展开：（一）...（二）...（三）...
其中，第一个维度的研究以 A 和 B 为代表，他们共同的看法是...，但在...问题上存在分歧...
```

**核心原则**：综述要呈现「学界对话」，而非「学者名单」。

## 二、文献综述的三种组织方式

### 方式一：按主题线索（最常用）

适用于：研究议题涉及多个侧面

**结构**：
```
现有研究主要从 N 个角度探讨了...问题：

（一）角度一的梳理
  - 代表学者及其观点
  - 该角度的贡献与局限

（二）角度二的梳理
  - ...

（三）角度三的梳理
  - ...

（四）现有研究的不足
  - 尚未关注的问题
  - 已有观点的盲点
```

**示例**：
> 关于《红楼梦》的研究，学界大致从三个维度展开：作者考证、文本阐释与文化意义。在作者考证方面，以胡适、周汝昌为代表的研究者通过文献考证，基本确定了曹雪芹的作者身份，但关于后四十回的作者问题仍无定论...

### 方式二：按时间脉络

适用于：议题有较清晰的发展阶段

**结构**：
```
关于...的研究经历了几个发展阶段：

第一阶段（时间段）：奠基期
  - 主要学者和代表性观点
  - 该阶段的总体特征

第二阶段（时间段）：深化期
  - ...

第三阶段（时间段）：转向期
  - ...
```

**示例**：
> 明清之际思想史研究大致经历了三个阶段：1980 年代的「启蒙」叙事、1990 年代的「内在理路」转向，以及 2010 年以来的「全球史」视野...

### 方式三：按论争焦点

适用于：学界对某个问题有明显分歧

**结构**：
```
学界关于...问题存在几种主要观点：

观点 A：...
  - 支持者：...
  - 论据：...

观点 B：...
  - 支持者：...
  - 论据：...

观点 C：...
  - 支持者：...
  - 论据：...

三种观点的分歧核心在于...，尚未达成共识的方面...
```

**示例**：
> 关于晚明思想变革的性质，学界存在三种主要观点：以余英时为代表的「内在理路」说认为...；以沟口雄三为代表的「中国基体」说主张...；近年来有学者提出...

## 三、文献综述的写作步骤

### Step 1：收集与分类

1. 通读每篇文献，做笔记：核心论点、使用的材料、研究方法
2. 给每篇文献贴标签（可一篇多标签）：
   - 主题标签：#政治史 #思想史 #文学史
   - 方法标签：#考证 #阐释 #比较
   - 立场标签：#支持X #反对Y #补充Z
3. 按标签归类，形成综述的初步结构

### Step 2：寻找对话关系

**问自己**：
- 这些学者是在回答同一个问题吗？
- 他们之间有哪些共识？
- 他们在哪些点上争论？
- 谁继承了谁？谁反驳了谁？
- 有没有被忽视的声音？

**常用句式**：
- 「A 的研究开创了...，但其局限在于...」
- 「B 在此基础上推进了...，但仍未解决...」
- 「C 的研究与 A 形成对话，他认为...」
- 「值得注意的是，D 从...角度提出了不同看法」

### Step 3：定位研究空白

**空白类型**：
- **材料空白**：某个重要文本/史料尚未被充分研究
- **视角空白**：已有研究都从 X 角度切入，没人从 Y 角度
- **方法空白**：学界多用 A 方法，B 方法的潜力未被发掘
- **论题空白**：某个子问题一直被忽视

**表达研究空白**：
- 「然而，已有研究尚未充分关注...」
- 「值得注意的是，关于...的讨论仍显不足」
- 「现有研究多聚焦于...，而对...则缺乏深入探讨」
- 「上述研究为本文奠定了基础，但它们共同留下了一个问题：...」

### Step 4：说明你的位置

在综述末尾，明确说明：
- 你的研究如何回应这些文献
- 你站在哪一派/哪条脉络上
- 你的新贡献是什么

**句式**：
- 「基于上述回顾，本文拟从...角度切入，探讨...」
- 「不同于 A 的...观点，本文主张...」
- 「借鉴 B 的...方法，本文将其运用于...」

## 四、文献综述的常见错误

| 错误 | 修正 |
|------|------|
| 罗列式：A 说... B 说... C 说... | 按主题/脉络组织，呈现对话关系 |
| 只述不评：只总结观点，不做评价 | 每段都要指出贡献、局限或关联 |
| 引用过旧：大量引用几十年前的文献 | 平衡经典文献与前沿研究 |
| 忽略相反观点 | 主动呈现论争，再说明你的立场 |
| 综述与研究脱节 | 每部分都要回扣到你的研究问题 |
| 引用二手文献代替一手 | 尽量引用原始出处 |

## 五、字数与比例

- **绪论中的文献综述**：硕士 2000-3000 字，博士 3000-5000 字
- **独立文献综述章节**：硕士 5000-8000 字，博士 8000-15000 字
- **占全文比例**：约 15-25%

## 六、学术诚信

- 综述中的每个观点都要注明出处
- 概述他人观点时，准确反映原意，不歪曲
- 区分「直接引用」、「间接引述」、「观点概述」
- 不编造文献信息，不虚构学者观点
- 转述时使用自己的语言，避免改写式抄袭

## 七、进阶技巧

**1. 地图意识**：把文献综述想象成绘制一幅「学界地图」，标注主要山脉（主流观点）、河流（发展脉络）、空白地带（研究空白）。

**2. 漏斗结构**：从宽到窄。先综述最宏观的讨论，逐步聚焦到与你的具体议题直接相关的文献。

**3. 批判性阅读**：不是「他研究了什么」，而是「他为什么这样研究」、「他预设了什么」、「他遗漏了什么」。

**4. 反向利用**：找到批评某篇文献的后续研究，它们往往指出了最有价值的空白。
