# 修繕分類與處理

## 修繕責任歸屬（依租賃專法+民法）

| 項目 | 誰負責 | 說明 |
|------|:------:|------|
| 房屋結構 | 房東 | 樑柱、地板、屋頂、外牆 |
| 水電管線 | 房東 | 水管、電線、瓦斯管（牆內） |
| 冷氣 | 房東 | 維修費用，若房客使用不當則房客負責 |
| 熱水器 | 房東 | 維修費用 |
| 馬桶/洗手台 | 房東 | 堵塞若因房客使用不當，房客負責 |
| 燈泡 | 房客 | 消耗品 |
| 電池（遙控器/門鎖） | 房客 | 消耗品 |
| 紗窗破損 | 視原因 | 正常老化→房東，人為破壞→房客 |
| 水管堵塞 | 視原因 | 老舊管線→房東，異物堵塞→房客 |
| 牆面油漆 | 房東（自然褪色） | 小面積可自理，大面積房東出 |
| 窗簾/窗軌 | 視原因 | 一般由房東 |
| 門鎖更換 | 房東 | 房客搬入時建議更換新鎖芯 |

## 修繕處理時效

| 狀況 | 處理時效 | 備註 |
|------|:--------:|------|
| 馬桶堵塞 | 24小時內 | 基本居住功能 |
| 無熱水 | 24小時內 | 基本居住功能（冬季）|
| 漏水 | 48小時內 | 需先止漏 |
| 冷氣故障 | 3-7天（旺季）| 先提供電扇應急 |
| 燈泡更換 | 3天內 | 由房客自行更換 |
| 紗窗破損 | 7天內 | 季節性可協調 |
| 油漆剝落 | 15天內 | 自然耗損 |

## 修繕管理SOP

### 收到通報
1. 確認問題內容與類型
2. 判斷責任歸屬
3. 回覆房客處理時間

### 安排廠商
```json
{
  "常用廠商清單": {
    "水電": ["聯絡人A 0912-xxx-xxx", "聯絡人B 0988-xxx-xxx"],
    "冷氣": ["聯絡人C 0933-xxx-xxx"],
    "清潔": ["聯絡人D 0977-xxx-xxx"],
    "門窗": ["聯絡人E 0922-xxx-xxx"]
  }
}
```

### 修繕完成
1. 確認修繕完成
2. 拍照存證
3. 請房客確認
4. 保留單據

### 費用處理
- 房東負責：取得發票/收據，完全吸收
- 房客負責：告知金額，由房客繳納或從押金扣除
- 爭議：保留維修單據，協商解決

## 修繕協商技巧

### 叫修必備資訊
```
位置：○○路XX號X樓 ○○房間
問題：馬桶無法沖水
狀況描述：按壓沖水鈕無反應，水箱內有水
聯絡方式：房客手機
時段偏好：平日晚上7點後
```

### 降低修繕糾紛的方法
1. **定期檢修**：每季檢查一次，提早發現問題
2. **廠商價格透明**：先報價再施工
3. **給房客選擇權**：由房東安排vs.房客自理報銷
4. **維修紀錄留存**：建立物件維修履歷
5. **消耗品區分清楚**：簽約時明確定義誰負責
