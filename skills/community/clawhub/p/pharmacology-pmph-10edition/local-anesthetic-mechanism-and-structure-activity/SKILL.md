# 局部麻醉药作用机制与构效关系

## 何时使用
- 比较普鲁卡因与利多卡因的起效速度和穿透力
- 解释为何细纤维比粗纤维更易被阻滞
- 优化局麻方案（如表面麻醉 vs 神经阻滞）

## 执行步骤

### 1. 分析化学结构
- **共同结构**：芳香环（脂溶性）+ 中间链 + 胺基团（弱碱性）
- **中间链类型**：
  - **酯类**（如普鲁卡因）：被血浆酯酶快速水解，作用时间短
  - **酰胺类**（如利多卡因）：经肝代谢，作用时间较长

### 2. 计算非解离型比例
- 使用Henderson-Hasselbalch方程：
  ```
  [B]/[BH⁺] = 10^(pH - pKa)
  ```
- **pKa越接近生理pH（7.4）** → 非解离型（B）比例越高 → **穿透力越强**
  - 利多卡因（pKa=7.9）穿透力好
  - 普鲁卡因（pKa=8.9）穿透力差

### 3. 应用作用机制
- **非解离型（B）**：跨膜进入神经细胞
- **解离型（BH⁺）**：结合电压门控Na⁺通道内口（仅在通道开放时）
- **结果**：阻滞Na⁺内流 → 抑制动作电位 → 阻断传导

### 4. 预测敏感性顺序
- **纤维敏感性**：细 > 粗；无髓鞘 > 有髓鞘
- **阻滞顺序**：钝痛 → 锐痛 → 温觉 → 触觉 → 压觉 → 运动麻痹
- **恢复顺序相反**

### 5. 考虑使用依赖性
- 高频刺激时Na⁺通道开放增多 → BH⁺结合更多 → 阻滞增强
- 故兴奋中的神经更敏感

> 注意：本机制不适用于全身麻醉药。
