# 肾上腺素剂量依赖性血管效应与α阻断药翻转机制

## 何时使用
- 预测肾上腺素对特定器官（如皮肤、骨骼肌、肾脏）血管的影响
- 解释为何小剂量肾上腺素舒张压不变而大剂量升高
- 警示在使用酚妥拉明等α阻断药后误用肾上腺素可能导致低血压

## 核心逻辑

### 1. 肾上腺素的双重受体作用
- 同时激动 **α₁受体**（血管收缩 → 升压）和 **β₂受体**（血管舒张 → 降压）
- 最终效应取决于：
  - 血管床中α₁与β₂受体密度比值
  - 药物剂量

### 2. 不同血管床的反应
| 血管类型       | 主导受体 | 效应               |
|----------------|----------|--------------------|
| 皮肤、黏膜、肾 | α₁       | 收缩               |
| 骨骼肌、肝脏   | β₂       | 小剂量时舒张       |
| 冠状动脉       | β₂       | 舒张（多重机制）   |
| 脑、肺         | 弱反应   | 被动舒张           |

### 3. 剂量影响
- **小剂量/皮下注射**：β₂效应主导 → 舒张压不变或下降，脉压增大
- **大剂量/静脉注射**：α₁效应主导 → 收缩压和舒张压均升高

### 4. α受体阻断药导致的“翻转”
- **前提**：已预先给予酚妥拉明等非选择性α阻断药
- **机制**：α₁被阻断 → β₂舒血管效应无对抗 → 净效应为降压
- **特异性**：
  - 仅见于**肾上腺素**（因其兼具α和β活性）
  - **去甲肾上腺素**无翻转（主要作用于α）
  - **异丙肾上腺素**不受影响（仅作用于β）

> 临床警示：在α阻断药治疗期间，避免使用肾上腺素，以防意外低血压。
