# 心肌缺血的ST-T心电图改变分析

## 适用条件
- 患者存在典型胸痛或心肌缺血症状
- 已排除电解质紊乱、药物影响等非缺血性因素
- 心电图出现ST段偏移或T波形态异常

## 分析步骤

### 1. 判断T波改变类型
- **心内膜下缺血**：面向缺血区导联（如Ⅱ、Ⅲ、aVF或V₁～V₆）出现**高大直立T波**。
- **心外膜下缺血（含透壁性）**：面向缺血区导联出现**深尖、双肢对称的倒置T波**（“冠状T波”）。

### 2. 判断ST段改变类型
- **心内膜下损伤**：常规导联出现**水平型或下斜型ST段压低 ≥0.1mV**。
- **心外膜下损伤**：相关导联出现**ST段抬高**（如变异型心绞痛）。
- 注意是否存在**镜像性改变**（如前壁ST抬高伴下壁ST压低）。

### 3. 结合临床表现
- 典型心绞痛发作时：ST段压低 ≥0.1mV 和/或T波倒置。
- 静息状态下已有持续性ST-T改变（≥0.05mV压低或T波低平/双向），发作时加重或出现“伪性改善”。
- 变异型心绞痛：一过性ST段抬高 + 高耸T波 + 对应导联ST压低，提示严重缺血，需紧急扩冠治疗。

### 4. 鉴别诊断要点
- **脑血管意外**：宽深倒置T波 + QT显著延长。
- **肥厚型心肌病**：V₃～V₅深倒置T波，易误诊为缺血。
- **继发性ST-T改变**：见于左室肥厚、束支阻滞、预激综合征等，需结合QRS波群特征鉴别。

## 输出建议
根据ST-T改变类型判断缺血层次与部位，决定是否需紧急干预（如硝酸酯类、钙通道阻滞剂）或安排冠脉造影等进一步检查。
