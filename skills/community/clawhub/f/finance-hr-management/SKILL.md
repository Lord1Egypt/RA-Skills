## 1. 招聘管理

### 1.1 招聘流程（标准七步法）

```
需求提出 → 岗位审批 → 招聘渠道选择 → 简历筛选 → 面试评估 → Offer发放 → 体检入职
```

| 步骤 | 负责部门 | 周期 | 关键动作 |
|------|----------|------|----------|
| 1. 需求提出 | 用人部门 | 即时 | 填写《人员需求申请表》 |
| 2. 岗位审批 | HR + 管理层 | 1-3天 | 岗位JD确认、薪资范围确定 |
| 3. 渠道选择 | HR | 即时 | 依据预算和岗位选择渠道 |
| 4. 简历筛选 | HR | 3-5天 | 匹配度评估、邀约面试 |
| 5. 面试评估 | 用人部门+HR | 5-10天 | 初试→复试→终试 |
| 6. Offer发放 | HR | 1-2天 | 背景调查、薪酬谈判 |
| 7. 体检入职 | HR + 行政 | 7-15天 | 入职材料准备、合同签署 |

### 1.2 招聘渠道对比分析

| 渠道类型 | 适用岗位 | 成本 | 周期 | 效果 |
|----------|----------|------|------|------|
| **BOSS直聘** | 中低端岗位 | 低 | 快 | ⭐⭐⭐⭐ |
| **猎聘网** | 中高端岗位 | 中 | 中 | ⭐⭐⭐⭐ |
| **领英LinkedIn** | 海外/高端人才 | 高 | 慢 | ⭐⭐⭐⭐⭐ |
| **智联招聘** | 通用岗位 | 中 | 中 | ⭐⭐⭐ |
| **前程无忧** | 通用岗位 | 中 | 中 | ⭐⭐⭐ |
| **58同城** | 基层/蓝领 | 低 | 快 | ⭐⭐ |
| **猎头服务** | 高管/稀缺人才 | 很高 | 慢 | ⭐⭐⭐⭐⭐ |
| **内推** | 所有岗位 | 低 | 快 | ⭐⭐⭐⭐⭐ |
| **校园招聘** | 应届生 | 中 | 周期性强 | ⭐⭐⭐⭐ |
| **招聘会** | 基层岗位 | 中 | 快 | ⭐⭐⭐ |

### 1.5 招聘成本核算公式

```
# 单次招聘总成本
招聘总成本 = 直接成本 + 间接成本

# 直接成本
直接成本 = 广告费用 + 猎头费用 + 招聘会费用 + 简历下载费 + 体检费 + 差旅费

# 间接成本（人力成本）
间接成本 = 面试官时薪 × 面试时长 × 面试人数 × 候选人数量

# 单位招聘成本
单位招聘成本 = 招聘总成本 ÷ 成功录用人数

# 招聘周期
平均招聘周期 = Σ(每位员工从需求到入职天数) ÷ 录用总人数

# 招聘渠道ROI
渠道ROI = 该渠道录用人数 × 平均月薪 ÷ 该渠道投入成本
```

### 2.2 试用期管理规定

| 合同期限 | 试用期上限 | 工资要求 |
|----------|------------|----------|
| 不满3个月 | 不得约定试用期 | - |
| 3个月-1年（不含1年） | ≤1个月 | ≥约定工资80% |
| 1年-3年（不含3年） | ≤2个月 | ≥约定工资80% |
| 3年以上/无固定期限 | ≤6个月 | ≥约定工资80% |

**重要提醒**：试用期包含在劳动合同期限内，试用期工资不得低于转正后工资的80%，且不得低于当地最低工资标准。

### 2.5 经济补偿金计算公式

**基本公式**：
```
经济补偿金 = N × 月工资
```

**N 的计算标准**：

| 工作年限 | N值 |
|----------|-----|
| 不满6个月 | 0.5 |
| 6个月以上不满1年 | 1 |
| 1年以上 | 每满1年支付1个月 |

**特殊规定**：
- 6个月以上不满1年：按1年计算
- 不满6个月：支付半个月工资
- 最高年限：12年（高薪员工，月工资超过当地社平工资3倍）

**月工资定义**：
```
月工资 = 劳动合同解除或终止前12个月的平均工资
       = 前12个月应发工资总额 ÷ 12
```

**示例计算**：

**示例1**：员工工作5年3个月，月薪15000元
```
工作年限：5年3个月 → 5.5年
N = 5.5
经济补偿金 = 5.5 × 15000 = 82,500元
```

**示例2**：员工工作8个月，月薪20000元
```
工作年限：8个月 → 1年
N = 1
经济补偿金 = 1 × 20000 = 20,000元
```

**示例3**：员工工作3年5个月，月薪50000元，当地社平工资10000元
```
月工资超过社平3倍（30000元），按30000计算
工作年限超过12年，按12年计算
N = 12
经济补偿金 = 12 × 30000 = 360,000元
```

### 2.6 竞业限制规定

| 项目 | 规定 |
|------|------|
| 适用对象 | 高级管理人员、高级技术人员、负有保密义务的人员 |
| 最长期限 | 不超过2年 |
| 补偿标准 | 不低于当地最低工资标准（通常为月薪的30%-50%） |
| 生效条件 | 离职后实际履行 |
| 违约责任 | 按约定支付违约金 |

**计算公式**：
```
竞业限制补偿金总额 = 月补偿标准 × 限制月数
```

## 3. 薪酬管理

### 3.1 薪酬结构设计

```
月度薪酬包
├── 基本工资（Base Salary）
│   └── 占总薪酬的 40-60%
├── 岗位工资（Position Wage）
│   └── 占总薪酬的 10-20%
├── 绩效工资（Performance Pay）
│   └── 占总薪酬的 10-30%
├── 津贴补贴（Allowances）
│   ├── 餐补/交通补贴
│   ├── 通讯补贴
│   ├── 住房补贴
│   └── 高温/取暖补贴
└── 其他
    ├── 全勤奖
    ├── 工龄工资
    └── 专项奖励
```

### 3.2 工资计算公式

**月薪计算（日工资制）**：
```
月计薪天数 = 21.75天

日工资 = 月薪 ÷ 21.75
小时工资 = 月薪 ÷ 21.75 ÷ 8

应付月薪 = 月薪 ÷ 21.75 × 实际出勤天数
实际出勤天数 = 月应出勤天数 - 缺勤天数
```

**示例计算**：

某员工月薪10000元，10月应出勤21天，实际出勤18天（请假3天）

```
日工资 = 10000 ÷ 21.75 = 459.77元/天
应付月薪 = 10000 ÷ 21.75 × 18 = 8275.86元
```

### 3.3 加班费计算

**法定标准**：
| 加班类型 | 倍数 | 说明 |
|----------|------|------|
| 工作日加班 | 1.5倍 | 超过8小时/天或40小时/周 |
| 休息日加班 | 2倍 | 不能安排补休时 |
| 法定节假日加班 | 3倍 | 当日不得以补休代替 |

**计算公式**：
```
工作日加班费 = 小时工资 × 1.5 × 加班小时数
休息日加班费 = 日工资 × 2 × 加班天数
法定节假日加班费 = 日工资 × 3 × 加班天数
```

**示例计算**：

员工月薪6900元（假设当地最低工资符合标准），小时工资 = 6900÷21.75÷8 = 39.66元

1. 工作日加班2小时：
   ```
   加班费 = 39.66 × 1.5 × 2 = 118.97元
   ```

2. 周末加班1天（无法补休）：
   ```
   加班费 = 6900÷21.75 × 2 = 634.48元
   ```

3. 法定节假日加班1天：
   ```
   加班费 = 6900÷21.75 × 3 = 951.72元
   ```

### 3.4 个税计算（累计预扣法）

**税率表（2019年后适用）**：

| 级数 | 累计预扣预缴应税所得额 | 预扣率 | 速算扣除数 |
|------|------------------------|--------|------------|
| 1 | ≤36000元 | 3% | 0 |
| 2 | 36000-144000元 | 10% | 2520 |
| 3 | 144000-300000元 | 20% | 16920 |
| 4 | 300000-420000元 | 25% | 31920 |
| 5 | 420000-660000元 | 30% | 52920 |
| 6 | 660000-960000元 | 35% | 85920 |
| 7 | >960000元 | 45% | 181920 |

**专项附加扣除标准**：
| 项目 | 标准 |
|------|------|
| 子女教育 | 每个子女1000元/月 |
| 继续教育 | 学历400元/月，职业证书3600元 |
| 大病医疗 | 80000元/年上限 |
| 住房贷款利息 | 1000元/月 |
| 住房租金 | 800-1500元/月（按城市） |
| 赡养老人 | 2000元/月（独生子女） |

**计算公式**：
```
累计预扣预缴应税所得额 = 累计收入 - 累计免税收入 - 累计减除费用 
                        - 累计专项扣除 - 累计专项附加扣除 - 累计其他扣除

本期应预扣预缴税额 = 累计预扣预缴应税所得额 × 预扣率 - 速算扣除数 
                    - 累计已预扣预缴税额
```

**示例计算**（2024年，月薪20000元，专项附加扣除2500元/月，社保公积金2000元）：

```
每月减除费用：5000元
每月专项扣除（三险一金个人部分）：2000元
每月专项附加扣除：2500元

每月应纳税所得额 = 20000 - 5000 - 2000 - 2500 = 10500元
累计应纳税所得额 = 10500 × 月份数

月度应预扣税额计算：
第1月：10500 × 3% = 315元
第2月：21000 × 3% - 315 = 315元
第3月：31500 × 3% - 630 = 315元
...
累计超过36000后按10%税率计算
```

## 4. 五险一金

### 4.1 社保公积金缴费比例一览表

**2024年全国通用标准（部分城市有差异）**：

| 险种 | 单位缴费比例 | 个人缴费比例 | 备注 |
|------|-------------|-------------|------|
| 养老保险 | 16% | 8% | 进入统筹账户/个人账户 |
| 医疗保险 | 8-10% | 2% | 部分城市有差异 |
| 失业保险 | 0.5-1% | 0.5% | 各地标准不同 |
| 工伤保险 | 0.2-1.9% | 0% | 按行业风险等级 |
| 生育保险 | 0.5-1% | 0% | 已并入医疗保险 |
| **住房公积金** | 5-12% | 5-12% | 单位和个人同比例 |

**合计缴费比例（单位）**：约26.7-31.4%
**合计缴费比例（个人）**：约15.5-21.5%

### 4.2 工伤保险行业风险分类

| 行业类别 | 基准费率 | 适用行业 |
|----------|---------|----------|
| 一类 | 0.2% | 软件和信息技术服务业、银行业等 |

| 八类 | 1.9% | 石油和天然气开采业等 |

### 4.3 缴费基数规定

```
缴费基数上下限 = min(max(上年度月平均工资, 社平工资×60%), 社平工资×300%)

即：下限为社平工资60%，上限为社平工资300%
```

**示例**（假设当地社平工资为8000元）：
```
最低缴费基数 = 8000 × 60% = 4800元
最高缴费基数 = 8000 × 300% = 24000元

若员工月薪30000元 → 缴费基数按24000元封顶
若员工月薪3500元 → 缴费基数按4800元保底
```

### 4.4 社保公积金计算示例

**员工月薪15000元，当地社平工资8000元，公积金缴存比例12%**：

```
确定缴费基数：
缴费基数 = min(max(15000, 4800), 24000) = 15000元

单位缴纳社保：
- 养老保险 = 15000 × 16% = 2,400元
- 医疗保险 = 15000 × 8% = 1,200元
- 失业保险 = 15000 × 0.5% = 75元
- 工伤保险 = 15000 × 0.4% = 60元
- 生育保险 = 15000 × 0.5% = 75元
单位社保合计 = 3,810元

个人缴纳社保：
- 养老保险 = 15000 × 8% = 1,200元
- 医疗保险 = 15000 × 2% = 300元
- 失业保险 = 15000 × 0.5% = 75元
个人社保合计 = 1,575元

公积金：
单位公积金 = 15000 × 12% = 1,800元
个人公积金 = 15000 × 12% = 1,800元
公积金合计 = 3,600元

单位总支出 = 月薪15000 + 社保3810 + 公积金1800 = 20,610元
个人实扣 = 社保1575 + 公积金1800 = 3,375元
```

## 5. 考勤与假期

### 5.1 工时制度类型

| 类型 | 标准工时 | 加班规定 |
|------|----------|----------|
| **标准工时制** | 每天8小时，每周40小时 | 超时按加班费计算 |
| **综合计算工时制** | 以周/月/年为周期，平均每天8小时 | 超周期结算，超时按1.5倍 |
| **不定时工作制** | 无固定上下班时间 | 法定节假日工作需支付加班费 |

**适用范围**：
- 综合工时制：交通、铁路、邮电、旅游等季节性行业
- 不定时工时制：高管、外勤、司机、特殊岗位

### 5.2 考勤异常处理

| 异常类型 | 处理方式 | 是否扣款 |
|----------|----------|----------|
| 迟到≤30分钟 | 提醒警告 | 视公司制度 |
| 迟到>30分钟 | 扣半天工资 | ✅ 是 |
| 早退 | 参照迟到处理 | ✅ 是 |
| 旷工半天 | 扣1天工资+警告 | ✅ 是 |
| 旷工1天 | 扣3天工资+严重警告 | ✅ 是 |
| 连续旷工3天 | 可按严重违规解除 | ✅ 是 |

### 5.3 法定节假日（11天/年）

| 节日 | 天数 | 日期 |
|------|------|------|
| 元旦 | 1天 | 1月1日 |
| 春节 | 3天 | 农历正月初一至初三 |

| 中秋节 | 1天 | 农历中秋当日 |
| 国庆节 | 3天 | 10月1日至3日 |

### 5.4 带薪年休假规定

**年休假天数（累计工作年限）**：

| 累计工龄 | 年休假天数 |
|----------|-----------|
| 满1年不满10年 | 5天 |
| 满10年不满20年 | 10天 |
| 20年以上 | 15天 |

**计算公式**：
```
年休假天数 = 当年度应出勤天数 ÷ 365 × 应享天数（取整，不足1天不计）
```

**示例**：员工6月15日入职，累计工龄5年，当年应出勤天数180天
```
应享年休假 = 180 ÷ 365 × 5 = 2.47天 → 2天
```

**工资计算**：
```
年休假工资 = 日工资 × 应休未休天数 × 3倍
（其中已包含正常工资，实际只需支付2倍差额）
```

### 5.5 其他假期规定

| 假期类型 | 天数/规定 | 工资标准 |
|----------|----------|----------|
| **病假** | 视病情，最长24个月 | 不低于当地最低工资80% |
| **婚假** | 1-30天（各省不同） | 全额工资 |
| **产假** | 98天+延长60天（各地不同） | 生育津贴或全额工资 |
| **陪产假** | 7-30天（各省不同） | 全额工资 |
| **流产假** | 15-75天 | 全额工资 |
| **节育假** | 按手术类型 | 全额工资 |
| **丧假** | 1-5天（直系亲属） | 全额工资 |
| **工伤假** | 认定之日起至康复 | 原工资福利待遇 |

### 5.6 产假生育津贴计算

```
生育津贴 = 用人单位月缴费平均工资 ÷ 30 × 产假天数

生育医疗费用 = 顺产/剖宫产定额 + 特殊情况费用
```

**示例**：
```
单位月缴费平均工资：8000元
产假天数：98天（国家）+ 60天（延长）= 158天

生育津贴 = 8000 ÷ 30 × 158 = 42,133.33元
```

## 6. 劳动合同管理

### 6.1 劳动合同类型

| 类型 | 适用场景 | 试用期规定 |
|------|----------|------------|
| **固定期限劳动合同** | 有明确起止时间的工作 | 可约定试用期 |
| **无固定期限劳动合同** | 长期稳定岗位 | 可约定试用期 |
| **以完成一定工作任务为期限** | 项目制/季节性工作 | 不得约定试用期 |

### 6.2 试用期详细规定

| 合同期限 | 试用期上限 | 最长月薪 |
|---------|-----------|----------|
| 不满3个月 | 0 | - |
| 3个月-1年 | 1个月 | 2个月 |
| 1年-3年 | 2个月 | 4个月 |
| 3年以上 | 6个月 | 12个月 |

**试用期不得约定事项**：
- ❌ 单独约定试用期（试用期包含在合同期内）
- ❌ 延长试用期
- ❌ 试用期内不缴社保
- ❌ 工资低于转正后80%或当地最低工资

### 6.3 劳动合同变更

**可变更情形**：
- 双方协商一致
- 员工岗位/职务调整
- 工作地点调动（30公里内合理范围）
- 薪酬调整

**变更程序**：
```
1. 提出变更意向（书面）
2. 双方协商
3. 签署变更协议
4. 各执一份
```

### 6.4 劳动合同解除与终止

**协商解除**：
```
由单位提出 → 支付经济补偿
由员工提出 → 不支付经济补偿
```

**单位单方解除（过失性）**：
- 试用期内不符合录用条件
- 严重违反规章制度
- 严重失职、营私舞弊
- 被依法追究刑事责任

**单位单方解除（非过失性）**：
- 员工患病或非工伤后不能从事原工作
- 员工不能胜任工作（经培训或调岗仍不能胜任）
- 客观情况发生重大变化

**劳动合同终止**：
- 合同到期
- 员工开始享受基本养老保险待遇
- 员工死亡或被宣告死亡/失踪
- 单位被吊销/责令关闭/撤销
- 法律规定的其他情形

### 6.5 劳动仲裁时效

```
普通时效：1年（从知道或应当知道权利被侵害之日起）
特殊时效：劳动关系存续期间不受限制
```

```markdown
# 年龄结构

基层员工比例 = 基层员工人数 ÷ 总人数 × 100%
```

```
# 月离职率

# 年离职率
年离职率 = 全年离职人数 ÷ 年初在职人数 × 100%

# 主动离职率
主动离职率 = 主动辞职人数 ÷ 总离职人数 × 100%

# 试用期离职率
试用期离职率 = 试用期离职人数 ÷ 试用期总人数 × 100%
```

### 8.4 人效指标

```markdown
# 人均产值
人均产值 = 企业总收入 ÷ 员工人数

# 人均利润
人均利润 = 企业净利润 ÷ 员工人数

# 人均销售额
人均销售额 = 销售总额 ÷ 销售人数

# 人均成本
人均成本 = 人工成本总额 ÷ 员工人数
```

### 8.5 人工成本分析

```
# 人工成本构成
人工成本总额 = 薪酬总额 + 社保公积金单位部分 + 培训成本 + 招聘成本 + 其他

# 人事费用率
人事费用率 = 人工成本总额 ÷ 营业收入 × 100%

# 社保公积金占人工成本比
社保占比 = (社保+公积金)单位部分 ÷ 人工成本总额 × 100%

# 劳动分配率
劳动分配率 = 人工成本总额 ÷ 企业增加值 × 100%
```

**示例**：
```
某公司年营业收入5000万，人工成本总额1500万，员工200人

人事费用率 = 1500万 ÷ 5000万 × 100% = 30%
人均产值 = 5000万 ÷ 200 = 25万元/人
```

```excel
Sub CalculateSocialSecurity()
    Dim ws As Worksheet
    Set ws = ThisWorkbook.Sheets("工资表")
    
    Dim lastRow As Long
    lastRow = ws.Cells(ws.Rows.Count, "A").End(xlUp).Row
    
    Dim i As Long
    For i = 2 To lastRow
        Dim baseSalary As Double
        baseSalary = ws.Cells(i, 5).Value ' 月薪
        
        ' 缴费基数（上下限处理）
        Dim minBase As Double: minBase = 4800
        Dim maxBase As Double: maxBase = 24000
        Dim insurBase As Double
        insurBase = WorksheetFunction.Max(minBase, WorksheetFunction.Min(baseSalary, maxBase))
        
        ' 单位社保
        ws.Cells(i, 11).Value = insurBase * 0.16 ' 养老
        ws.Cells(i, 12).Value = insurBase * 0.08 ' 医疗
        ws.Cells(i, 13).Value = insurBase * 0.005 ' 失业
        ws.Cells(i, 14).Value = insurBase * 0.004 ' 工伤
        
        ' 个人社保
        ws.Cells(i, 15).Value = insurBase * 0.08
        ws.Cells(i, 16).Value = insurBase * 0.02
        ws.Cells(i, 17).Value = insurBase * 0.005
        
        ' 公积金
        ws.Cells(i, 18).Value = insurBase * 0.12 ' 单位
        ws.Cells(i, 19).Value = insurBase * 0.12 ' 个人
    Next i
End Sub
```

## 附录：常用计算公式速查表

### 薪酬相关
```
日工资 = 月薪 ÷ 21.75
小时工资 = 月薪 ÷ 21.75 ÷ 8
加班费(工作日) = 小时工资 × 1.5 × 加班小时
加班费(休息日) = 日工资 × 2 × 加班天数
加班费(法定假日) = 日工资 × 3 × 加班天数
```

### 离职补偿相关
```
N = 工作年限（6个月以上按1年，不满6个月按0.5）
经济补偿金 = N × 月工资（月工资超社平3倍按3倍，年限最多12年）
代通知金 = 1个月工资
```

### 社保相关
```
社保个人 = 缴费基数 × (8%+2%+0.5%)
公积金个人 = 缴费基数 × (5%-12%)
单位总社保成本 = 月薪 + 社保单位部分 + 公积金单位部分
```

### 假期相关
```
年休假工资 = 日工资 × 3 × 未休天数（实际支付2倍差额）
病假工资 = 当地最低工资 × 80% × 病假天数
生育津贴 = 月缴费基数 ÷ 30 × 产假天数
```

### 人效相关
```
离职率 = 离职人数 ÷ 期初人数 × 100%
人事费用率 = 人工成本 ÷ 营业收入 × 100%
人均产值 = 总产值 ÷ 员工人数
```
