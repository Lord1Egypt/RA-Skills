# 急性与慢性肾盂肾炎病理诊断

## 何时使用
- 患者出现**发热、寒战、腰痛、尿频/尿急/尿痛**，且尿检显示**脓尿、菌尿、白细胞管型** → 考虑**急性肾盂肾炎**
- 患者有**反复尿路感染史**，出现**多尿、夜尿、高血压、贫血**，影像学显示**肾脏不对称缩小伴瘢痕、肾盂肾盏变形** → 考虑**慢性肾盂肾炎**
- 需与膀胱炎、急性肾小球肾炎、其他间质性肾病鉴别

## 执行步骤

### 1. 判断感染类型与病程阶段
- **急性期**：起病急，全身炎症反应明显，肾功能通常正常
- **慢性期**：起病隐匿或反复发作，伴肾小管功能障碍及结构破坏

### 2. 分析病理特征
#### 急性肾盂肾炎
- **肉眼观**：肾脏肿大、充血，表面散在黄白色脓肿，切面髓质见黄色条纹；肾盂黏膜充血水肿，有脓性渗出
- **镜下**：
  - 灶状间质化脓性炎或脓肿
  - 肾小管腔内中性粒细胞集聚（形成**白细胞管型**，具诊断特异性）
  - 肾小管坏死
- **感染途径**：
  - 上行性（经尿道→膀胱→输尿管→肾盂）：常单侧，先累及肾盂
  - 下行性（血源性）：多双侧，先累及肾皮质

#### 慢性肾盂肾炎
- **肉眼观**：肾脏缩小，表面不规则瘢痕（多位于上下极），双侧常不对称；肾盂肾盏变形、黏膜粗糙
- **镜下**：
  - 间质纤维化 + 淋巴细胞/浆细胞浸润
  - 肾小管部分萎缩、部分扩张，扩张管腔内含**均质红染胶样管型**（形似甲状腺滤泡）
  - 肾小球晚期可继发玻璃样变
  - 急性发作时可见中性粒细胞浸润
- **病因分型**：
  - 反流性肾病（儿童多见，膀胱输尿管反流）
  - 慢性阻塞性肾盂肾炎（尿路梗阻致反复感染）

### 3. 关联临床表现
- **急性**：
  - 发热、寒战、腰痛、肾区叩痛
  - 尿检：脓尿、菌尿、白细胞管型（关键诊断依据）
  - 肾小球通常不受累 → 无高血压、氮质血症
- **慢性**：
  - 肾小管浓缩功能↓ → 多尿、夜尿
  - 电解质丢失 → 低钠、低钾、代谢性酸中毒
  - 肾缺血 → 肾素↑ → 高血压
  - 晚期 → 氮质血症、尿毒症

### 4. 排除禁忌情况
- 不适用于：
  - 非感染性肾小管间质疾病
  - 原发性肾小球疾病（如IgA肾病、膜性肾病）
  - 急性肾损伤（非感染性）

### 5. 输出诊断结论
- **急性**：结合症状、尿检（尤其白细胞管型）及影像学确诊
- **慢性**：依据病史、尿异常、电解质紊乱及肾盂造影（显示瘢痕与变形）综合判断
