# cn-sort-lines

文本行排序工具。支持升序、降序、按长度排序。

## 使用方法

```bash
python3 scripts/sort_lines.py --text "行C\n行A\n行B"
python3 scripts/sort_lines.py --text "行C\n行A\n行B" --desc
python3 scripts/sort_lines.py --text "行C\n行A\n行B" --by-length
```

纯Python标准库。
