# TikTok Shop Listing SEO Quality Checklist

Validate every listing optimization against this checklist before publishing. Each item must pass. Items marked with [CRITICAL] will significantly harm search performance if missed.

---

## 1. Title Optimization (7 items)

- [ ] [CRITICAL] Primary keyword appears within the first 3 words of the title
- [ ] [CRITICAL] Title is between 60 and 120 characters (target 80-100)
- [ ] Secondary keyword is present in the title and positioned before character 80
- [ ] Title includes the specific product type name that buyers search for (e.g., "face serum" not "skincare product")
- [ ] No ALL CAPS words, excessive punctuation, or promotional phrases (e.g., "BEST SELLER," "HOT DEAL," "#1")
- [ ] Brand name is not positioned at the start of the title unless the brand has proven TikTok Shop search volume
- [ ] Title reads naturally as a product description, not as a keyword list

---

## 2. Description Quality (7 items)

- [ ] [CRITICAL] Description is at least 300 words (target 500-700)
- [ ] [CRITICAL] Primary keyword appears 2-3 times across the full description, distributed across different sections
- [ ] Description uses short paragraphs (2-3 sentences maximum) for mobile readability
- [ ] Description includes at least 3 distinct sections (e.g., benefits, how to use, specifications, what is included)
- [ ] Bullet points are used for scannable information (benefits, features, package contents)
- [ ] Specific numbers and measurements are included (dimensions, quantities, percentages, durations)
- [ ] No hashtags, non-functional URLs, or HTML formatting in the description text

---

## 3. Tag Strategy (5 items)

- [ ] [CRITICAL] All 7 tag slots are filled
- [ ] [CRITICAL] Every tag is a multi-word phrase (2-4 words minimum; no single-word tags)
- [ ] At least one tag is an exact match for the primary keyword
- [ ] Tags do not duplicate phrases already prominent in the title
- [ ] At least one tag targets a long-tail or use-case-specific search query

---

## 4. Category and Attributes (5 items)

- [ ] [CRITICAL] Product is placed in the most specific leaf-level subcategory available
- [ ] All mandatory backend attribute fields are completed
- [ ] All optional backend attribute fields are completed with keyword-rich values
- [ ] Attribute values list all applicable options (e.g., multiple skin types, multiple use cases) rather than selecting only one
- [ ] Category path has been verified by browsing the full category tree, not just searching for the category name

---

## 5. Image Optimization (5 items)

- [ ] [CRITICAL] Main image (slot 1) shows the product clearly on a clean background, identifiable at thumbnail size
- [ ] At least 7 of 9 image slots are used (target all 9)
- [ ] Image set includes at least one lifestyle or in-use photo
- [ ] Image set includes a size reference or scale comparison shot
- [ ] No watermarks, promotional text overlays, or logos covering the product in any image

---

## 6. Competitive Alignment (4 items)

- [ ] Listing price is within 20% of the average price for top 10 search results for the primary keyword
- [ ] Title structure follows the dominant pattern observed in top-ranking competitor listings
- [ ] Description length meets or exceeds the average description length of top 5 competitors
- [ ] At least 2 keywords are targeted that top competitors are not optimizing for (keyword gaps identified in competitive analysis)

---

## 7. Keyword Coverage and Distribution (5 items)

- [ ] [CRITICAL] Primary keyword appears in at least 3 of these 4 fields: title, description, tags, backend attributes
- [ ] Each secondary keyword appears in at least 2 listing fields
- [ ] No single keyword phrase is repeated more than 3 times across the entire listing
- [ ] Keywords are sourced from TikTok Shop search autocomplete (not assumed or borrowed from other platforms)
- [ ] Keyword map has been completed showing every target keyword and its placement across all listing fields

---

## 8. Post-Publish Verification (3 items)

- [ ] After publishing, search for the primary keyword on TikTok Shop and confirm the listing appears in results (may take 24-48 hours)
- [ ] Listing renders correctly on mobile (check description formatting, image display, and title truncation in search results)
- [ ] Seller Center analytics are configured to track search impressions, CTR, and conversion rate separately from other traffic sources

---

## Scoring Guide

Count the total number of checked items.

| Score | Rating | Action |
|---|---|---|
| 38-41 | Excellent | Publish and monitor. Schedule 30-day review. |
| 32-37 | Good | Address any unchecked CRITICAL items before publishing. Non-critical gaps can be improved in the next optimization cycle. |
| 25-31 | Needs Work | Do not publish until all CRITICAL items pass and at least 32 total items are checked. |
| Below 25 | Incomplete | Significant optimization gaps remain. Return to the workflow and complete missing steps before running this checklist again. |

Total CRITICAL items: 9. All 9 must be checked regardless of total score.
