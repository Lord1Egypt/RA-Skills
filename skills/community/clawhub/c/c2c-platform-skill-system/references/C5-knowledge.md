## 步骤5.1 知识库建设与管理 (C5-01 to C5-07)

### C5-01 平台运营SOP编写

- **输入**: 平台运营流程清单（服务商入驻审批流程、条目上下架流程、订单状态机流转规则、易点虚拟币充值/消费/提现流程、1%平台佣金结算时序）、现有运营操作记录、业务方访谈纪要
- **输出**: 标准操作手册（每份SOP包含：流程目的、适用范围、角色职责矩阵、步骤编号与操作说明、异常分支处理、关键校验点），Markdown格式，每步骤配流程图Mermaid代码
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C5-08(FAQ素材)、C5-10(培训课程SOP模块)、C5-22(知识更新审批对象)

### C5-02 服务标准与规范编写

- **输入**: 本地生活服务品类目录（家政/维修/教培/美业/跑腿等）、行业服务标准参考、平台服务等级协议(SLA)目标（如响应时间≤30min、履约完成率≥95%）、完成码验证规则、差评处理时效要求
- **输出**: 服务标准文档（品类×维度矩阵：响应时效、到岗时效、服务流程规范、验收标准、禁止行为清单），PDF+Markdown双格式，每品类独立章节
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C5-03(入驻指南引用标准)、C5-15(条目发布规范引用标准)、C5-16(履约流程规范引用标准)

### C5-03 商家入驻指南编写

- **输入**: 服务商入驻流程（注册→实名认证→资质上传→审核→签约→开店）、所需证照清单（营业执照/行业许可/ICP备案）、入驻审核标准、平台服务协议条款、保证金/备付金缴纳标准、易点账户开通流程
- **输出**: 图文入驻指南（分步操作手册+截图标注+常见驳回原因Top10+审核周期说明），支持Web端与小程序端双版本截图
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C5-08(FAQ入驻类问题)、C5-11(服务商培训入驻模块)、C7-01(入驻审核参考标准)

### C5-04 争议处理手册编写

- **输入**: 历史争议案例库（服务不符/未到岗/加价/毁损/虚假宣传等类型）、争议仲裁流程（举证→调解→仲裁→执行）、退款规则（全退/部分退/不退判定标准）、完成码争议场景、平台介入判定原则、易点虚拟币退款规则
- **输出**: 争议处理手册（争议类型分类树→判定标准→处理流程→话术模板→案例解析），每类争议配决策树图
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C6-10(投诉处理流程参考)、C6-11(争议仲裁流程细化)、C6-29(纠纷举证材料模板)

### C5-05 支付合规指引编写

- **输入**: 易点虚拟币体系规则（1易点=1分钱、充值/消费/提现链路）、平台佣金1%计算与扣除规则、二清合规要求（资金不经过平台自有账户）、提现审核规则（T+1到账、大额提现风控）、发票开具规则、支付渠道费率表
- **输出**: 支付合规指引（虚拟币流转全景图、资金链路合规说明、二清规避方案、提现规则明细、税务处理指引），含Mermaid时序图
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C6-05(交易合规检查参考)、C6-06(二清合规评估输入)、C6-28(保证金管理规则)

### C5-06 用户协议解读文档

- **输入**: 《用户服务协议》《隐私政策》《服务商合作协议》法律文本、监管要求（个人信息保护法/电商法/反不正当竞争法）、平台特殊条款（3级分销上限条款、易点虚拟币使用条款、完成码确认条款）
- **输出**: 通俗解读文档（逐条原文→白话解读→用户影响说明→典型案例），分用户版/服务商版/运营版三套
- **依赖**: 无
- **AI自治度**: ⬛
- **组合接口**: → C6-07(隐私保护合规对照)、C6-25(协议更新参考)、C6-27(传销防控3级上限条款解读)

### C5-07 技术架构文档维护

- **输入**: 系统架构图（微服务划分：用户中心/订单中心/支付中心/风控中心/内容中心等）、API文档、数据库ER图、部署拓扑、关键技术选型说明（完成码验证机制、订单状态机实现、信用分计算引擎、3级分销关系链存储方案）
- **输出**: 技术架构文档（架构总览→服务拓扑→核心流程时序图→数据模型→接口规范→变更记录），Markdown+Mermaid格式，版本化管理
- **依赖**: 无
- **AI自治度**: ⬛
