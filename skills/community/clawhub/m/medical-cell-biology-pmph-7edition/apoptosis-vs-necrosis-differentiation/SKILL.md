# 细胞凋亡与细胞坏死的核心区别

## 判别流程
1. **起因判断**：
   - 生理性/轻度刺激 → 凋亡。
   - 剧烈损伤/严重病理 → 坏死。
2. **死亡范围**：
   - 单个散在细胞 → 凋亡。
   - 大片组织 → 坏死。
3. **膜完整性**：
   - 完整至形成凋亡小体 → 凋亡。
   - 破损、内容物外泄 → 坏死。
4. **核变化**：
   - 染色质半月状凝聚、DNA梯状片段化 → 凋亡。
   - 染色质絮状、DNA涂抹状降解 → 坏死。
5. **炎症反应**：
   - 无炎症 → 凋亡。
   - 有炎症 → 坏死。

## 输出结果
返回分类：'细胞凋亡' 或 '细胞坏死'，并附关键判别依据。

## 注意事项
- 不适用于原核细胞。
- 程序性坏死（如坏死性凋亡）需额外分子标志物确认。
