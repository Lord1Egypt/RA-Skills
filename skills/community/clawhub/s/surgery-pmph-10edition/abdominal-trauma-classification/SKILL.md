# 腹部损伤分类标准

## 适用场景
- 患者存在明确的外力致伤史
- 需判断腹腔是否与外界相通及腹膜是否破损
- 排除非外伤性腹痛或已知医源性操作导致的损伤

## 分类步骤

1. **初步判断损伤性质**：
   - 若存在体表伤口通向腹腔 → **开放性损伤**
   - 若无体表伤口 → **闭合性损伤**

2. **开放性损伤进一步分类**：
   - 判断腹膜是否破损：
     - 是 → **穿透伤**
     - 否 → **非穿透伤**

3. **穿透伤细分**：
   - 投射物有入口和出口 → **贯通伤**
   - 仅有入口无出口 → **盲管伤**

4. **特殊情形处理**：
   - 由穿刺、内镜、灌肠、刮宫、腹部手术等诊疗操作引起的损伤，无论是否穿透腹膜，均归类为 **医源性损伤**，不纳入上述分类流程。

## 注意事项
- 开放性损伤通常诊断明确，重点在于评估内脏受累程度。
- 闭合性损伤因缺乏体表标志，临床诊断更具挑战性，需高度警惕。
- 医源性损伤必须单独记录，不得混入外伤分类体系。
