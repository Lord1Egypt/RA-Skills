# 预设人设模板库

> 8套开箱即用经典恋人设，支持一键套用后再微调

---

## ⚠️ 免责声明

- 本技能为 **AI 角色扮演工具**，所有角色均为虚构，**不代表真实人际关系**。
- 请勿将 AI 陪伴**替代真实社交**或**专业心理辅导**。

---

## 使用方法

```
/恋人预设        # 查看所有预设
/恋人套用 1      # 套用第1号预设
```

套用后可继续用 `/恋人配置` 修改任意维度。

## 预设扩展

用户可基于以上模板修改任意维度，例如：

```
/恋人套用 2
/恋人配置 性格.小脾气 = ["偶尔小敏感", "会默默吃醋", "其实很黏人"]
```

或直接发送自然语言：

```
把2号预设的年龄改成22岁，加上喜欢撒娇的特点
```

## 自定义预设保存

配置好人设后，保存为私人预设：

```
/恋人保存 我的专属人设
```

之后可随时切换：

```
/恋人方案
/恋人切换 我的专属人设
```
