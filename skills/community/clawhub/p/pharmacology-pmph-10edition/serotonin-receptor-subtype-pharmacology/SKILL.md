# 5-羟色胺受体亚型与药理作用对应关系

## 何时使用
- 解释静脉注射5-HT引起的三相血压反应
- 分析5-HT在胃肠道或支气管的作用机制
- 理解虫咬或植物刺激引发痒痛的神经药理基础

## 决策流程

### 1. 心血管系统效应（静脉注射数微克5-HT）
- **第一相（短暂降压）**：5-HT₃受体激动 → 心脏负性频率作用
- **第二相（持续升压）**：5-HT₂受体激动 → 肾、肺血管收缩
- **第三相（长时间低血压）**：骨骼肌血管舒张（需内皮参与）
- **附加效应**：5-HT₂受体激动 → 血小板聚集

### 2. 平滑肌效应
- **胃肠道**：
  - 5-HT₂受体（直接作用于平滑肌）→ 张力↑
  - 5-HT₄受体（作用于肠壁神经节）→ 蠕动↑
- **支气管**：
  - 兴奋平滑肌，哮喘患者高度敏感

### 3. 神经系统效应
- **中枢**：侧脑室注射 → 镇静、嗜睡、体温调节紊乱
- **外周**：作用于感觉神经末梢 → 引起痒痛（如虫咬反应）

> 注意：5-HT本身无临床治疗价值，主要用于研究或解释病理生理现象。
