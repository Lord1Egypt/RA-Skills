## [0.1.0] - 2026-05-24
Initial release. Six-phase workflow that turns site / target data, tree biometrics, and observed defects into a DRAFT Basic Level 2 tree risk assessment aligned to the ISA *Best Management Practices — Tree Risk Assessment* (BMP-TRA, 2nd ed.), the ISA TRAQ matrix, and ANSI A300 Part 9 — covering scope-of-assessment intake (Level 1 limited visual / Level 2 basic / Level 3 advanced — defaulting to Level 2), site and target characterization (occupancy frequency, target type, target value), tree biometrics and species traits (DBH, height, crown class, lean, species-specific failure profile), systematic defect inspection (root collar / buttress roots / lower trunk / upper trunk / scaffold limbs / branch unions / canopy) with defect tables, load-factor capture (wind exposure, recent storms, edge effects, soil saturation), likelihood-of-failure rating (Improbable / Possible / Probable / Imminent) over the specified time frame, likelihood-of-impact rating (Very low / Low / Medium / High), consequences rating (Negligible / Minor / Significant / Severe), the BMP-TRA two-step matrix that combines failure-and-impact to Likelihood and Likelihood-and-Consequences to Risk (Low / Moderate / High / Extreme), mitigation options (move target, remove tree, prune, cable / brace, monitor, accept risk) with residual-risk re-rating, an assessor-limitations and disclaimer block, a photo-and-evidence index, and an open-questions list — for ISA TRAQ-qualified arborist review and sign-off before any communication to the tree owner / risk manager.
