# 完全性大动脉转位存活依赖混合通路评估

## 何时使用
- 新生儿确诊为完全性大动脉转位（TGA）；
- 出现严重青紫、低氧血症或代谢性酸中毒；
- 需快速判断是否具备维持氧合的血流混合通路以指导急救处理。

## 执行步骤

1. **确认解剖结构**：验证主动脉发自右心室、肺动脉发自左心室，形成两个并行循环（体循环输送静脉血，肺循环输送动脉血）。

2. **评估混合通路存在性**：检查是否存在以下至少一种有效混合通路：
   - **心内交通**：卵圆孔未闭（PFO）、房间隔缺损（ASD）、室间隔缺损（VSD）；
   - **心外交通**：动脉导管未闭（PDA）、体-肺侧支血管。

3. **根据合并畸形分类血流动力学表现**：
   - **室间隔完整型**：仅依赖PFO和PDA混合 → 青紫严重，缺氧显著；
   - **合并VSD**：心内混合增加 → 青紫减轻，但肺血增多 → 易致心力衰竭；
   - **合并VSD + 肺动脉狭窄**：类似法洛四联症 → 青紫明显，心衰少见。

4. **判断临床紧急程度**：若未发现有效混合通路，动脉血氧饱和度极低，应视为急症，立即启动干预措施。

5. **启动治疗启示**：
   - 使用前列腺素E1维持PDA开放；
   - 必要时行Rashkind球囊房间隔造口术以扩大心房水平交通，提高全身氧合，为根治手术争取时间。

> **注意**：本技能不适用于伴有单心室或复杂畸形无法建立混合通路的患儿。
