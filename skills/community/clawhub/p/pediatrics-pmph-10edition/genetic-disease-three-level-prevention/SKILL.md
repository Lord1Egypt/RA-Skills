# 遗传性疾病三级预防体系实施

## 适用场景
- 制定区域或机构层面的遗传病防控策略
- 设计婚前/孕前健康咨询与筛查项目
- 规划产前诊断服务流程
- 建立新生儿遗传代谢病筛查与早期干预机制

## 执行步骤

1. **一级预防（孕前阶段）**
   - 对目标人群开展隐性遗传病携带者筛查
   - 根据筛查结果提供个性化婚育指导
   - 对高风险夫妇建议进行胚胎植入前遗传学诊断（PGD）或产前诊断

2. **二级预防（孕期阶段）**
   - 对高风险孕妇实施产前诊断，包括：
     - 胎儿超声表型评估
     - 染色体核型分析或染色体微阵列（CMA）
     - 无创产前检测（NIPT）或羊水穿刺基因检测
   - 确诊后提供遗传咨询与妊娠决策支持

3. **三级预防（新生儿期）**
   - 在出生后72小时内采集足跟血进行群体筛查
   - 应用串联质谱、基因测序等技术检测可干预的遗传代谢病
   - 对阳性病例启动快速确诊与早期治疗，防止不可逆器官损伤

4. **整合与优化**
   - 确保三个阶段信息互通、转诊顺畅
   - 结合本地医疗资源评估技术可行性与成本效益
   - 定期评估防控效果并迭代优化流程

## 使用前提
- 具备相应的筛查与诊断技术能力
- 目标疾病具有明确诊断标准和有效干预措施
- 当地医疗体系可支持多阶段协同干预

## 限制条件
- 不适用于无法在症状前检测或缺乏有效治疗手段的遗传病
- 需根据地区经济水平、实验室能力和伦理规范调整实施方案
