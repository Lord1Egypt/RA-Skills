# 卡培他滨的体内代谢活化路径

## 何时使用
- 患者正在接受卡培他滨治疗，需解释其作用机制
- 评估患者是否存在影响该代谢路径的酶缺陷或肝功能障碍
- 分析为何卡培他滨具有肿瘤组织选择性
- 判断药物相互作用或个体差异对疗效的影响

## 执行步骤
1. **确认给药方式**：确保为口服给药，这是启动该代谢路径的前提。
2. **第一步：肝脏水解**：卡培他滨在肝脏中被60kDa羧酸酯酶水解为5'-脱氧-5-氟胞苷（5'-DFCR）。
3. **第二步：组织脱氨**：5'-DFCR在大多数组织（包括肿瘤）中被胞苷脱氨酶转化为5'-脱氧-5-氟尿苷（5'-DFUR）。
4. **第三步：肿瘤局部活化**：5'-DFUR在胸苷磷酸化酶（dThdPase）作用下生成具有细胞毒性的氟尿嘧啶（5-FU）。
5. **评估选择性机制**：确认肿瘤组织中胸苷磷酸化酶是否高表达，以解释5-FU在局部富集、提高疗效并降低全身毒性的原理。

## 关键约束
- 不适用于缺乏羧酸酯酶、胞苷脱氨酶或胸苷磷酸化酶的个体或组织
- 肝功能严重障碍可能阻碍第一步代谢，影响整体活化效率

> 注意：此路径是理解卡培他滨临床应用和不良反应管理的核心依据。
