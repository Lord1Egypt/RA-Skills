# Sector Overview — 行业概览与轮动分析 Skill

行业涨跌排行、轮动判断、见底信号匹配一体化分析工具。

## 功能

- 📊 行业涨跌排行：5日/20日/60日涨跌幅、PE分位、ETF资金流向
- 🔄 轮动分析：相对沪深300超额收益、进攻/防御/均衡阶段判断
- ✅ 见底信号匹配：对照5大集群配置文件逐条验证
- 💡 配置建议：基于轮动信号的行业配置建议

## 触发方式

- 说 "行业分析" / "板块分析" / "行业轮动" / "哪个板块有机会"
- 说 "分析XX行业"

## 依赖

- `neodata-financial-search` skill

## 输出

Markdown 行业概览报告，保存至 `sector-overview/reports/YYYY-MM-DD-sector-{行业名}.md`
