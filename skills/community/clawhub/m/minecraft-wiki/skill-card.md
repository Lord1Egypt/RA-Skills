## Description: <br>
Answers Minecraft Java Edition gameplay questions about facts, mechanics, strategies, recipes, item properties, mob behaviors, and version differences using an offline reference set. <br>

## Use Case: <br>
External users and developers use this skill to answer general Minecraft Java Edition gameplay questions from local markdown references. It is intended for game knowledge, strategy, troubleshooting, and version-aware explanations rather than live game-state inspection. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill is focused on Minecraft Java Edition and may not apply to Bedrock, modded, or server-plugin behavior. <br>
Mitigation: Clarify the requested edition or environment before applying guidance outside Java Edition. <br>
Risk: The skill does not include a bridge for current inventory, position, nearby entities, or world state. <br>
Mitigation: Use it for general knowledge and request live-game-state tooling when the answer depends on the user's current world. <br>
