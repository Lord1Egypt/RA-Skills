# 中胚层三分区与分化命运分析

## 触发条件
当中胚层在脊索两侧形成时，按其位置划分为三个区域，并启动相应的分化程序。

## 执行步骤

1. **确认前提条件**：
   - 三胚层胚盘已建立
   - 脊索已形成
   - 胚胎处于第4～8周（不适用于胚外中胚层）

2. **识别中胚层三分区**（由内向外）：
   - **轴旁中胚层**（paraxial mesoderm）：紧邻脊索两侧
     - 迅速增殖形成纵行细胞索，裂为块状体节（somite）
     - 体节左右成对，从颈部向尾部依次形成
     - 第5周共形成42～44对（可作为胚龄判断指标）
     - 分化为：背侧皮肤真皮、骨骼肌、中轴骨骼（如脊柱）
   - **间介中胚层**（intermediate mesoderm）：位于轴旁与侧中胚层之间
     - 分化为泌尿生殖系统主要器官（如肾、生殖腺等）
   - **侧中胚层**（lateral mesoderm）：最外侧部分
     - 内部小腔隙融合形成胚内体腔（intraembryonic coelom），并与胚外体腔相通
     - 分为两层：
       - **体壁中胚层**（parietal mesoderm）：贴外胚层，分化为胸腹四肢的真皮、骨骼肌、骨骼、血管
       - **脏壁中胚层**（visceral mesoderm）：贴内胚层，覆盖原始消化管外，分化为消化/呼吸系统的肌组织、血管、结缔组织、间皮

3. **追踪胚内体腔分化**：
   - 胚内体腔从头至尾分化为三大体腔：心包腔、胸膜腔、腹膜腔

4. **输出结果**：
   - 确定目标组织的中胚层来源类型与位置，用于器官原基追踪或发育异常分析
