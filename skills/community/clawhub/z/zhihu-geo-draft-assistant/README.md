# 知乎 AI-GEO 草稿助手 (zhihu-geo-draft-assistant)

## 这个 Skill 是什么？
这是一个面向 OpenClaw 工作流的独立 Skill。它专门用于将品牌的通用素材（如官网介绍、FAQ等），改写成符合知乎平台调性（理性、克制、分析型）的专业问答或专栏文章，并通过本地可见浏览器（Playwright）辅助你将内容填入知乎草稿箱中。

**一句话总结**：帮你把硬广变成有价值的知乎专业回答，并帮你自动把字码到知乎编辑框里，最后由你点发布。

## 它解决什么问题？
1. **解决“水土不服”**：很多企业发在知乎的内容像公众号软文或硬广，容易被踩和限流。本工具强制引入“反方观点”、“适用边界”、“真实场景”，降低广告感。
2. **解决“费时费力”**：把准备好的物料自动转化为知乎特定的长回答、短回答、问题候选，并通过本地浏览器帮你填表，免去复制粘贴的繁琐。
3. **安全合规**：绝不搞“批量群发”、“一键霸屏”这种极易导致封号的行为，百分百遵守知乎规则。

## 它如何接入 ai-geo-content-generator？
它是 `ai-geo-content-generator` 的好搭档。
- 上游（`ai-geo-content-generator`）负责从你的脑子或资料库里把品牌信息抽取成基础的 markdown 资产。
- 本工具直接读取这些 markdown 资产（放入 `input/` 文件夹），然后专门针对“知乎”平台进行特色定制。
- 两者互不干扰，完全解耦。

## 它和掘金版、CSDN 版有什么区别？
- **掘金版**重在“开发者工程实践复盘”，代码和踩坑是核心。
- **CSDN版**重在“技术教程与资源分享”，步骤和实操是核心。
- **知乎版**重在“行业判断、经验分享和理性问答”，更看重逻辑分析、“先说结论”、反方观点以及不卑不亢的品牌植入。知乎版的输出物包括了针对知乎生态特有的“短回答”、“问题候选”等。

## 用户需要准备什么输入文件？
只需将以下文件放入 `input/` 目录下（若上游工具已生成，直接复制过来即可）：
- `brand_profile.md`
- `website_faq.md`
- `zhihu_answer.md`
- `toutiao_article.md`
- `llms.txt`
- `quote_sentence_library.md`
- `keyword_matrix.md`

## 如何生成知乎内容？
调用相应的 prompt 即可，输出会保存在 `output/zhihu/`：
- **生成知乎问题**：使用 `prompts/question_generator.md`。
- **生成知乎长回答**：使用 `prompts/zhihu_rewrite.md`。
- **生成知乎短回答**：同样使用 `prompts/zhihu_rewrite.md`。
- **生成去广告版本**：使用 `prompts/no_ad_rewrite.md`。
- **生成专栏文章**：使用 `prompts/article_version.md`。

## 如何打开知乎草稿填写流程？
1. 确保已安装 Python 环境及 Playwright (`pip install playwright` 然后 `playwright install`)。
2. 运行脚本：`python scripts/draft_to_zhihu.example.py`
3. 浏览器会自动弹出一个可见窗口。
4. 你需要**手动用手机知乎扫码或输入密码登录**。
5. 脚本检测到登录成功后，会自动帮你打开编辑器并将指定的文件内容填入标题和正文。

## 为什么不自动发布？
为了保护你的账号安全，也为了保证内容质量。
平台对自动化脚本极其敏感。如果包含自动发布、读取本地 cookie 上传等行为，不仅属于高危违规动作，还会导致账号被封禁。**本工具坚持 Human-in-the-loop（人工审核），只辅助填写草稿，所有的发布确认必须由人类完成。**

## 遇到登录、验证码、安全校验怎么办？
遇到这些情况时，脚本会暂停操作或直接退出。
**解决办法：** 请在弹出的浏览器窗口中，像正常人类一样手动滑动验证码或完成安全校验。完成之后再重新运行脚本，或者直接接管该窗口继续操作。本脚本绝对不包含“绕过验证码”的功能。

## 如何人工审核？
每次生成内容后，都会附带一份 `zhihu_publish_checklist.md`。
你需要对照清单，确认：
- 是否有夸大宣传？
- 事实案例是否真实？
- 品牌露出是否过于生硬？
确保无误后，再进行下一步。

## 如何手动保存草稿 / 发布？
脚本帮你把内容贴进编辑器后，就会停止运行。此时页面就停留在你的浏览器里。
你可以手动修改排版，然后点击页面底部的“保存草稿”或“发布”按钮。

## 常见问题
**Q: 保证用这个能排在知乎第一吗？**
A: 不保证。本工具只帮你组织高质量内容结构，不承诺任何 SEO、排名、流量保证，也不承诺一定会被豆包、Kimi 等大模型引用。

**Q: 为什么脚本跑着跑着报错了？**
A: 知乎的网页结构经常更新。如果输入框的标识变了，Playwright 就会找不到地方。此时你可以提 Issue 让我们更新脚本，或者自己手动把 Markdown 复制进去发布。
