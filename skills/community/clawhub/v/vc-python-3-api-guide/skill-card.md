## Use Case: <br>
Developers and automation engineers use this skill to answer questions, draft examples, and review usage patterns for the Visual Components 5.0 Python 3 API. It is especially useful for module lookup, async programming guidance, practical simulation examples, and Python 2 to Python 3 migration support. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Copied Visual Components examples may start or stop simulations, move robots, toggle conveyors, or open and save layouts when run. <br>
Mitigation: Review generated or copied sample code in a test layout before running it in an active project. <br>
