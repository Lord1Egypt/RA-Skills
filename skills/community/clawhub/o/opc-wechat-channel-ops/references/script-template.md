# 脚本模板库：老胡视频号内容模板

## 一、通用脚本结构

### 标准90秒脚本模板

```markdown
【开场 Hook - 10秒】
大家好，我是老胡。
今天咱们聊一个很多人都在问的问题：[具体问题]
（停顿1秒）
这个问题，处理不好可能要亏几十万。

【背景引入 - 15秒】
最近[时间/事件]，很多人都在讨论这个。
有人说[观点A]，也有人说[观点B]。
那到底怎么回事？
（停顿1秒）
根据我十几年经验，我来说说我的看法。

【核心内容 - 50秒】
第一，[要点1]
（展开说明，用1-2句话）

第二，[要点2]
（展开说明，用1-2句话）

第三，[要点3]
（展开说明，用1-2句话）

【老胡观点 - 15秒】
我的看法是：[核心观点，一句话]
因为[原因简述]。

【行动号召 CTA - 5-10秒】
觉得有用的，点个赞。
还想听什么，评论区告诉我。
关注老胡，懂技术转化。
我是老胡，咱们下期见！
```

## 二、具体类型脚本模板

### 1. 政策解读类

```markdown
【开场 Hook】
大家好，我是老胡。
[日期]出了一个政策，很多人都没注意到，
但它可能直接影响你的[业务/项目]。

【政策简介】
这个政策叫[政策名称/文号]，
核心内容就三句话：
第一，[核心变化]
第二，[核心变化]
第三，[核心变化]

【老胡解读】
那这个政策意味着什么？
我的理解是：[核心观点]

【实操建议】
如果你正在做[相关业务]，要注意这几点：
1. [要点1]
2. [要点2]
3. [要点3]

【互动结尾】
这个政策你怎么看？
评论区聊聊，关注老胡，下期告诉你怎么落地。

【片尾】
我是老胡，懂技术转化。
```

### 2. 案例分享类

```markdown
【开场 Hook】
大家好，我是老胡。
最近帮一个[客户类型]做[服务类型]，
整个过程非常曲折，
今天把核心经验分享给大家。

【案例背景】
这家公司/这位创始人[背景描述]，
遇到了一个[核心问题]，
这个问题如果解决不了，[后果]。

【解决过程】
我是怎么处理的？三步走：
第一步，[做了什么]，目的是[达到什么]
第二步，[做了什么]，关键是[注意什么]
第三步，[做了什么]，最终[结果如何]

【核心经验】
总结下来，有三点最重要的经验：
第一，[经验1]，因为[原因]
第二，[经验2]，否则[后果]
第三，[经验3]，特别是[注意事项]

【观众提醒】
如果你也在做[相关业务]，
千万记住这几点：[重复核心要点]

【互动结尾】
你们有没有遇到过类似的问题？
评论区说说，咱们一起交流。
```

### 3. 行业洞察类

```markdown
【开场 Hook】
大家好，我是老胡。
最近[行业/领域]出了一个大事，[事件描述]，
很多人问我怎么看，
今天说说我的一家之言。

【事件分析】
首先，这件事为什么重要？
[核心原因分析]

其次，对谁影响最大？
[影响群体分析]

第三，接下来会怎么发展？
[趋势预判]

【老胡观点】
我的判断是：[核心观点]
理由有三：[理由1]、[理由2]、[理由3]

【应对建议】
对于[相关从业者]，我的建议是：
短期内：[建议]
中长期：[建议]

【互动结尾】
这个判断对不对，咱们拭目以待。
想持续关注这个话题，点个关注。
我是老胡，下期接着聊。
```

```markdown
1. [步骤1]
2. [步骤2]
3. [步骤3]

【互动结尾】

关注老胡，咱们下期见！
```

### 5. 干货教程类

```markdown
【开场 Hook】
大家好，我是老胡。

【效果展示】
先说效果：[具体效果描述]
（停顿）
这个方法，我已经帮[多少人/多少企业]用过了。

【步骤讲解】
具体怎么操作？三步：

第一步：[操作要点]
注意：[注意事项]

第二步：[操作要点]
关键是：[关键点]

第三步：[操作要点]
记住：[重点提醒]

【常见错误】
还有几个坑千万不要踩：
[错误1] → 应该[正确做法]
[错误2] → 应该[正确做法]

【行动号召】
赶紧去试试，用完有问题评论区告诉我。
```

## 三、爆款脚本公式

### 公式一：痛点+方案

```
【痛点开场】
你是不是也遇到过[痛点问题]？
这个问题困扰了很多人，包括我自己。

【解决方案】
后来我发现了一个方法，[解决方案简述]。

【具体步骤】
很简单，就三步：[步骤123]

【效果承诺】
用这个方法，[效果描述]。

【行动号召】
赶紧试试，有问题评论区见。
```

### 公式二：反常识+真相

```
【颠覆认知】
你知道吗？[反常识观点]

【解释原因】
大多数人都以为[错误认知]，
但真相是[真实情况]。

【案例支撑】
我见过一个例子：[简述案例]

【总结升华】
所以，[核心结论]。

【引导行动】
这个知识点，收藏起来！
```

### 公式三：数字清单

```
【数字开场】
做[某事]，只需要注意这[数字]点！
第[数字]点，99%的人都忽略了。

【逐条展开】
第一点：[内容]
第二点：[内容]
……
第N点：[内容]

【总结强化】
记住这[数字]点，你也能[价值]。

【互动结尾】
你做到了几点？评论区告诉我。
```

| 制造悬念 | "最后一点最重要，往下看。" |

## 六、脚本使用注意事项

1. **不要死记硬背**：理解核心逻辑，用自己的话表达
2. **不要照读**：多次练习，减少念稿感
3. **加入个人风格**：老胡的特色是实战、专业、有温度
4. **灵活调整**：根据实际情况增减内容
5. **持续优化**：每条视频后复盘，不断改进
