# 永久记忆系统 · 使用说明

## 功能

- ✅ 每次对话自动记录，不遗漏任何重要信息
- ✅ 语义搜索，说"记得上次..."即可召回
- ✅ 重启不丢，永远在 MEMORY.md 中等你
- ✅ 只追加不覆盖，记忆永不丢失
- ✅ 按日期分片，文件永不过载

## 安装

将 `longterm-memory/` 目录放入 `~/.openclaw/skills/`：

```bash
cp -r longterm-memory ~/.openclaw/skills/
```

## 使用

**无需手动调用。** 技能自动生效：

- 告诉太初重要的事 → 自动写入记忆
- 说"记得上次..." → 自动搜索记忆
- 新会话开始 → 自动加载记忆上下文

**手动触发：**
```
记住我的项目叫 xxx
上次我们讨论的那个数据库方案是什么？
忘了关于 xx 的记忆
```

## 配置

| 环境变量 | 默认值 | 说明 |
|---------|-------|------|
| `MEMORY_PATH` | `~/.openclaw/workspace/MEMORY.md` | 记忆文件路径 |

## 原理

- `MEMORY.md` — 提炼后的长期记忆（<50KB，永远不过载）
- `memory/YYYY-MM-DD.md` — 原始对话日记（按日期归档）

## 常见问题

**Q: 记忆会无限增长吗？**
A: 不会。MEMORY.md 有自动压缩机制，超过 50KB 自动压缩最旧记忆。

**Q: 可以删除某条记忆吗？**
A: 可以告诉太初"删除关于 xxx 的记忆"，但系统永远不会自动删除，只会标记过期。
