# antibacterial-combination-synergistic-effect-assessment

## 何时使用
- 需联合使用 ≥2 种抗菌药物；
- 感染严重、病原未明、混合感染或需长期治疗（如结核、心内膜炎）；
- 已知各抗菌药物的作用机制类别。

## 药物分类
按作用性质分为四类：
- **I类：繁殖期杀菌药**（如青霉素、头孢菌素等β-内酰胺类）；
- **II类：静止期杀菌药**（如氨基糖苷类、多黏菌素类）；
- **III类：快速抑菌药**（如四环素类、大环内酯类）；
- **IV类：慢速抑菌药**（如磺胺类）。

## 联合效应判断规则
| 组合类型 | 效应       | 示例与说明 |
|----------|------------|-----------|
| I + II   | 协同       | 青霉素 + 庆大霉素 → 用于肠球菌心内膜炎 |
| I + III  | **拮抗**   | 青霉素 + 四环素 → 应避免，因抑制细菌繁殖削弱杀菌效果 |
| I + IV   | 相加       | 青霉素 + 磺胺 → 可用于流行性脑膜炎 |
| II + III | 相加/协同  | 视具体药物而定 |
| III + IV | 相加       | 如四环素 + 磺胺 |

## 执行步骤
1. 确定每种抗菌药物所属类别（I–IV）；
2. 根据组合类型查表判断联合效应；
3. 若为拮抗组合（尤其 I + III），除非有特殊循证依据，否则应避免；
4. 优先选择协同或相加组合，并结合感染类型、病原学证据调整方案。

> 警告：禁止在无充分理由下将繁殖期杀菌药与快速抑菌药联用。
