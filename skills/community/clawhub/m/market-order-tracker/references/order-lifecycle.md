## 异常处理手册

### 异常分类

| 级别 | 定义 | 处理时限 | 升级条件 |
|-----|------|---------|---------|
| P0 | 严重异常，影响交付 | 2小时 | 跟单员→业务员 |
| P1 | 重要异常，影响交期 | 4小时 | 跟单员→主管 |
| P2 | 一般异常，需协调 | 24小时 | 自行处理 |
| P3 | 轻微异常，记录跟踪 | 72小时 | 记录即可 |

### 常见异常处理

#### 生产延期
1. 评估影响：计算延期天数和影响订单
2. 备选方案：加急/换货/协商延期
3. 客户沟通：提前告知，提供替代方案
4. 记录归档：异常记录和解决方案存档

#### 验货不合格
1. 评估不合格程度：AQL范围内/超出
2. 处理方案：返工/挑选/降价/赔偿
3. 客户沟通：如实告知，共同商定
4. 预防措施：改进质量控制流程

#### 物流异常
1. 确认异常类型：延误/破损/丢失
2. 联系货代：启动理赔程序
3. 客户沟通：说明情况，提供方案
4. 后续跟进：确保货物安全到达

## 客户通知模板

### 关键节点通知

| 节点 | 触发时机 | 内容要点 |
|-----|---------|---------|
| 接单确认 | 收到定金 | 订单确认+交期承诺 |
| 生产进度 | 每50% | 进度更新+照片 |
| 生产完成 | 100% | 完成通知+验货报告 |
| 装船通知 | 开船日 | 提单信息+预计到港 |
| 到港通知 | 清关完成 | 到港确认+提货指引 |
| 交付确认 | 客户签收 | 完成感谢+复购邀请 |

## 时间节点计算

### 交期计算公式

```
发货日期 = 客户要求交期 - 物流时间 - 清关时间 - 验货时间
生产完成 = 发货日期 - 装卸时间
生产开始 = 生产完成 - 生产周期
```
