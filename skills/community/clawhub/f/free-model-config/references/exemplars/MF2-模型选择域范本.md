### MF2-04 参数设置范本

**必选组件**：

1. **参数列表**：

| 参数 | 类型 | 默认值 | 有效范围 | 说明 |
|------|------|--------|---------|------|
| max_tokens | integer | 100 | 1-4096 | 最大生成长度 |
| temperature | float | 0.7 | 0-1 | 生成随机性 |
| top_p | float | 0.9 | 0-1 | 采样概率 |

2. **参数值**：

| 参数 | 设置值 | 说明 |
|------|--------|------|
| max_tokens | {最大长度} | 根据需求设置 |
| temperature | {随机性} | 根据需求设置 |
| top_p | {采样概率} | 根据需求设置 |

3. **参数说明**：

- **max_tokens**：控制生成文本的最大长度，值越大生成内容越长
- **temperature**：控制生成随机性，值越大越随机，值越小越确定
- **top_p**：控制采样概率，值越大越多样，值越小越集中

**可选组件**：

**参数默认值**：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| max_tokens | 100 | 默认生成长度 |
| temperature | 0.7 | 默认随机性 |
| top_p | 0.9 | 默认采样概率 |

**参数范围**：

| 参数 | 最小值 | 最大值 | 说明 |
|------|--------|--------|------|
| max_tokens | 1 | 4096 | 生成长度范围 |
| temperature | 0 | 1 | 随机性范围 |
| top_p | 0 | 1 | 采样概率范围 |

**参数建议**：

- 如果需要确定性输出，建议设置 temperature 为 0.1-0.3
- 如果需要创造性输出，建议设置 temperature 为 0.7-0.9
- 如果需要多样性强的输出，建议设置 top_p 为 0.9-1.0

**视频参数自动推荐规则**：

| 用户意图 | 分辨率 | 帧数 | 帧率 | 预计时长 |
|---------|--------|------|------|---------|
| 最长视频 | 480p (854×480) | 961 | 24fps | ~40 秒 |
| 高质量 | 720p (1088×832) | 409 | 24fps | ~17 秒 |
| 快速生成 | 1080p (1920×1080) | 169 | 24fps | ~7 秒 |
| 默认 | 720p (1088×832) | 121 | 24fps | ~5 秒 |

向用户展示推荐参数和理由，询问是否接受或自定义。

**组装顺序**：参数识别 → 参数设置 → 参数验证

**约束**：参数必须在有效范围内

**格式**：参数配置（JSON）

**输出示例**：

```json
{
  "max_tokens": 100,
  "temperature": 0.7,
  "top_p": 0.9
}
```
