# 伯氨喹在疟疾治疗中的规范使用与G-6-PD筛查

## 适用场景
- 患者确诊为间日疟或卵形疟
- 需要根治（清除肝内休眠子）或阻断传播（杀灭配子体）
- 肝功能正常

## 执行步骤
1. **确认适应症**：仅用于间日疟或卵形疟的病因性治疗，不可用于红细胞内期急性发作的单药治疗。
2. **强制筛查G-6-PD活性**：
   - 询问患者溶血病史
   - 检测葡萄糖-6-磷酸脱氢酶（G-6-PD）活性
   - 若G-6-PD缺乏，禁止使用伯氨喹
3. **确定给药方案**：
   - 标准剂量：每日一次，疗程通常为14天
   - 若怀疑肝内虫株耐药：将剂量加倍并维持14天疗程
4. **监测安全性**：
   - 观察是否出现胃肠道反应（通常轻微且可逆）
   - 警惕大剂量下高铁血红蛋白血症（表现为发绀）
5. **输出决策**：
   - G-6-PD缺乏 → 禁用伯氨喹
   - 耐药可能 → 加倍剂量 + 延长疗程至14天

## 关键约束
- 对红细胞内期疟原虫无效
- G-6-PD缺乏者绝对禁用
- 不可用于急性发作期的单药治疗
