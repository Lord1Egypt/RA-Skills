# SEO内容优化器 🔍

## 核心流程（8 Steps）

### Step 1: 内容与关键词分析
- 提取主关键词、长尾关键词、LSI关键词
- 分析搜索意图（信息型/导航型/交易型/商业调查型）
- 评估关键词难度(KD)和竞争度，确定布局策略
- 竞品SERP分析：排名前5的内容特征（字数/结构/更新频率）
- 输入校验：内容<200字提示扩充

### Step 2: 标题标签优化
- 主关键词前置，控制在28-32字（中文）/50-60字符（英文）
- 添加年份/数字/修饰词提升CTR（点击率提升3-5%）
- 避免关键词堆砌，使用情感词/ power words
- 提供3个标题方案供选择（保守/均衡/激进）

### Step 3: Meta描述优化
- 核心信息+差异化+行动引导，80-120字（中文）/150-160字符（英文）
- 包含主关键词1-2次和长尾关键词
- 添加CTA引导词（"立即了解""点击查看""免费试用"）
- 提供2个Meta描述方案

### Step 4: 内容结构优化
- H1包含主关键词（仅1次），H2/H3布局长尾关键词
- 首段100字内包含主关键词
- 关键词密度2-3%，LSI关键词自然穿插
- 段落长度控制（3-5句/段），提升可读性
- 添加FAQ模块（抢占精选摘要/PAA）
- 内容层次：倒金字塔结构（结论先行）

### Step 5: 页面元素优化
- 图片ALT标签：描述内容+含关键词，8-20字
- 内链策略：相关概念指向站内相关文章（3-5条内链）
- 外链建议：权威来源引用（提升E-E-A-T）
- 表格/列表使用：提升结构化展示概率
- URL优化：简短/含关键词/连字符分隔

### Step 6: 平台特色优化
- 百度：长内容(1200字+)+时效性(年份)+百度百科引用
- Google：用户意图匹配+E-E-A-T信号+Core Web Vitals
- Bing：社交信号+精确关键词+多媒体内容
- AI搜索适配：结构化问答+清晰事实陈述+引用来源（适配SGE/AI Overview）

### Step 7: Schema结构化数据
- 推荐Schema类型：Article/FAQ/HowTo/Product/Review/Breadcrumb
- 生成JSON-LD代码片段
- 验证建议：Google Rich Results Test / 百度结构化数据检测
- 适配AI Overview的关键标记

### Step 8: 完整优化报告输出
- 关键词策略、TDK优化、内容结构、技术优化、Schema标记
- 行动清单（高/中/低优先级）
- 预期效果（排名/流量/CTR估算）
- 竞品对比分析

## 边界约束
1. **内容限制**：内容需至少200字，建议800字以上
2. **关键词密度**：控制在2-3%，避免堆砌（密度>5%有惩罚风险）
3. **平台差异**：百度偏好长内容+时效性，Google重视用户意图+E-E-A-T
4. **不支持**：技术SEO(服务器/sitemap)、外链建设、付费广告、网站架构
5. **质量标准**：H标签层级清晰、有FAQ模块、可读性良好
6. **AI搜索**：适配Google SGE/AI Overview、百度AI摘要等新型搜索形态

## 输入参数

| 参数 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| content | 必须 | - | 需要优化的文章/网页内容 |
| target_keyword | 可选 | - | 主关键词 |
| platform | 可选 | baidu | 目标平台：baidu/google/bing/all |
| content_type | 可选 | article | 内容类型：article/product_page/landing_page/blog |
| language | 可选 | zh | 内容语言：zh/en |
| competitor_url | 可选 | - | 竞品参考URL |
