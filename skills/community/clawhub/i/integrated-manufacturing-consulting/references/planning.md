# 生产计划与物控调研指南

## 调研目标
- 评估企业生产计划体系的成熟度（MPS/MRP/排产）
- 识别计划-执行闭环中的断点与障碍
- 评估物料控制能力（库存/齐套/缺料管理）

## 调研重点

### 1. 生产计划体系
- 计划编制流程（从接单到排产）
- MPS（主生产计划）编制方法与频率
- 计划颗粒度（年/月/周/日）
- 产能规划是否纳入计划编制
- 计划变更/调整机制

### 2. 排产管理
- 排产工具（Excel/ERP/MES/APS？）
- 排产考虑的约束条件
- 排产频率（日排/周排/滚动排产？）
- 瓶颈工序识别与管理
- 排产方案评审与下发流程

### 3. 物料需求计划(MRP)
- MRP运算方式（系统自动/手工计算？）
- BOM准确性（BOM准确率？）
- 是否考虑安全库存/批量规则
- 采购计划生成流程

### 4. 库存管理
- 原材料/在制品/成品库存水平
- 库存周转天数
- ABC分类管理
- 呆滞料管理机制
- 盘点制度与准确率

### 5. 齐套管理
- 生产前是否做齐套检查？
- 缺料处理机制
- 替代料管理

## 常见问题诊断库

| 问题现象 | 可能根因 | 典型影响 |
|---------|---------|---------|
| 计划赶不上变化 | 计划编制周期长/缺乏滚动机制 | 现场频繁调整/救火式管理 |
| 库存高但还缺料 | 安全库存设置不合理/数据不准 | 资金占用/停线风险并存 |
| BOM不准导致计划错误 | BOM维护滞后/ECN执行不到位 | 物料错买/漏买/呆滞 |
| 排产凭经验 | 缺乏数字化排产工具 | 产能利用不均衡/交期延误 |
| 齐套率低 | 无齐套检查流程/信息不同步 | 产线待料/换线频繁 |
| 库存周转慢 | 批量过大/安全库存过高/呆滞未处理 | 资金占用/仓储成本高 |
| 计划与执行脱节 | 无报工反馈/数据滞后 | 计划失效/进度失控 |
| 呆滞料占比高 | BOM变更未同步/采购策略问题 | 资金沉淀/仓储压力 |

## 数据采集项
- MPS编制周期与颗粒度
- 排产工具与方式
- BOM准确率（抽样）
- 库存周转天数
- 齐套率
- 缺料停线次数/时长
- 呆滞料金额与占比
- 计划达成率
