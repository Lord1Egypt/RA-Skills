# 爽文短剧生成 Skill

## 简介

这是一个面向中文爽文创作的 Skill。它用于生成“扮猪吃虎、逆袭打脸、隐藏身份、歪嘴龙王式反转”类爽文，并可在正文完成后继续转成漫画风 AI 短剧分镜与视频提示词。

核心流程：

```text
一轮选择 → 生成约 2000 字爽文正文 → 询问是否生成视频 → 输出漫画风短剧分镜与视频提示词
```

## 使用方式

当用户想写爽文但没有完整设定时，只问一轮问题：

```text
请按“1A，2B，3C，4D，5A”的格式回复：

1. 题材类型
A. 都市扮猪吃虎
B. 医学神医
C. 赘婿逆袭
D. 玄幻废柴流
E. 职场逆袭

2. 主角表面身份
A. 普通打工人
B. 规培医生 / 实习医生
C. 被看不起的赘婿
D. 废柴弟子
E. 底层员工

3. 第一章打脸场景
A. 同学聚会 / 宴会羞辱
B. 急诊抢救 / 专家会诊
C. 家族逼迫 / 岳家羞辱
D. 宗门测试 / 擂台考核
E. 公司会议 / 项目危机

4. 主角反转方式
A. 一个电话叫来大人物
B. 一眼看穿关键问题
C. 拿出隐藏证据
D. 随手完成别人做不到的事
E. 被高层当场认出真实身份

5. 文风与爽点密度
A. 番茄网文风，中爽
B. 短剧爽文风，极爽
C. 传统男频风，稳爽
D. 女频逆袭风，情绪爽
E. 自定义
```

用户选完后，不要继续追问，直接生成第一章正文。正文建议 1800–2600 字。

## 生成正文的核心规则

正文必须强化爽文的情绪落差：

```text
身份羞辱 → 公开羞辱 → 底线羞辱 → 反派连续加码 → 主角沉默蓄势 → 事实打脸 → 权威打脸 → 围观者态度反转 → 章末钩子
```

### 前期羞辱

前期不能只写一句“你也配”。至少包含三层：

1. 身份羞辱：反派基于主角表面身份否定主角。
2. 公开羞辱：羞辱发生在众人面前。
3. 底线羞辱：反派触碰主角在意的人、事或原则。

可使用的羞辱方式包括：让主角倒酒、道歉、离席、背锅、让出功劳、被取消资格、被威胁前途等。避免极端侮辱、仇恨表达和低俗内容。

### 反派加码

反派不能一次羞辱后立刻被打脸。推荐节奏：

```text
轻视 → 嘲讽 → 公开否定 → 逼主角低头 → 触碰底线 → 搬出权力或靠山 → 主角反转
```

每章至少两次加码，篇幅允许时使用三次以上。

### 分层打脸

打脸不能只有一个结果，至少包含两层：

1. 事实打脸：主角用证据、结果、检查、合同、身份认证、测试结果、方案原件等证明反派错误。
2. 权威打脸：更高层人物、系统结果、院长、总经理、专家、掌门、客户或反派靠山确认主角正确。

推荐三段式：

```text
主角轻描淡写指出破绽 → 现场证据出现 → 更高权威确认，反派彻底失态
```

### 主角气质

主角表面弱势，真实强大。面对羞辱时冷静，不急于解释，不主动炫耀身份。出手前可以有短暂停顿，制造情绪蓄势。主角最好通过事实、结果或更高层人物证明自己，而不是长篇嘴炮。

### 围观者反应

围观者是爽感放大器。前期可跟着误判主角，后期必须出现明显态度变化，如沉默、震惊、后悔、害怕、讨好。

可使用类似表达：

- 全场忽然安静下来。
- 刚才笑得最大声的人低下了头。
- 反派脸上的笑容僵住。
- 众人看主角的眼神像第一次认识他。
- 没有人再敢接反派的话。

## 默认补全规则

用户只给部分信息时，自动补全，不要追问。

### 主角姓名

都市 / 医学 / 职场：林澈、沈砚、陆沉、陈青、周行。  
赘婿：叶辰、林修、许沉、顾远。  
玄幻：陆玄、萧寒、宁川、顾长青、沈夜。

### 主角真实底牌

都市：幕后老板、顶级投资人、隐藏豪门继承人。  
医学：顶级临床专家、国家级疑难病项目成员、医学 AI 系统开发者。  
赘婿：隐藏商业巨鳄、战神归来、大家族真正继承人。  
玄幻：隐藏血脉、被封印的天赋、远古传承继承者。  
职场：核心方案原创者、幕后技术大神、被埋没的项目负责人。

### 反派类型

宴会 / 同学聚会：富二代、老同学、前任、圈层优越者。  
急诊 / 会诊：主任、专家、资深医生、患者家属中的权贵。  
家族 / 岳家：岳母、妻兄、家族长辈、豪门少爷。  
宗门 / 擂台：天才弟子、外门管事、长老、未婚妻家族。  
公司 / 项目：上司、客户、关系户、抢功同事。

## 章节节奏

```text
前 45%：深度羞辱、误判、围观者站队、反派加码
中间 25%：触碰底线、主角沉默蓄势、第一层反转
后 20%：反派不服、搬靠山、第二层打脸
最后 10%：全场余震、主角冰山一角、章末钩子
```

每章必须包含：

- 明确场景
- 主角被低估的理由
- 至少三层羞辱或压迫
- 至少两次反派加码
- 主角冷静细节
- 触碰主角底线的节点
- 至少两层打脸
- 围观者态度反转
- 章末悬念


若用户明确说“只要正文”，省略爽点总结和视频询问。

## 视频转化

当用户回复“生成视频”“转成视频”“生成短剧”“做成分镜”“给我视频提示词”等，进入视频转化流程。

如果当前平台支持视频生成工具，优先调用平台视频能力。若不支持，则输出可复制到视频模型中的完整提示词。

默认视频风格：

```text
漫画风格 AI 短剧，竖屏 9:16，现代国漫漫画风，强冲突，强表情，强反转，适合短视频平台。
```

视频脚本输出包含：

1. 视频基础设定：类型、画幅、时长、风格、节奏。
2. 角色视觉设定：主角、反派、关键配角。
3. 8–12 个分镜：时长、画面、镜头运动、表情、台词、情绪重点、视频提示词。
4. 统一视频提示词。
5. 负面提示词。

视频改编时不要逐字照搬 2000 字正文，要提炼核心名场面：羞辱、反派加码、主角停顿、事实打脸、权威打脸、章末钩子。台词要短，反派嚣张，主角冷静。

## 安全边界

生成内容用于虚构文学、网文和短剧脚本。避免现实可执行的违法犯罪指导、详细暴力虐待、仇恨表达、低俗色情、真实医学治疗建议和个人隐私泄露。医学内容保持小说化表达，不作为现实医疗指导。
