### L 配方关键设计思路（Recipe L Key Design Principles）

| 设计点 | 说明 |
|---|---|
| 全局基础设定先行（Global Base Setup First） | 在分镜之前统一锁定光影（Lighting）、摄影机（Camera）、表演基调（Performance Tone），避免每个时间片重复描述 |
| 情绪锚点关键词（Emotion Anchor Keywords） | 每个时间片配两词锚点（恐惧·挣扎 / 偏执·倔强），作为语义锚稳定模型输出 |
| 双保险描述（Dual-insurance Description） | 每个技术参数后跟视觉翻译——45° 快门不只写参数，还写「画面呈现逐帧跳跃的碎裂感」 |
| 画面内/外分离（In-frame / Off-frame Separation） | 全局说「不出现手部」，时间片中手部动作用「（画面外 / Off-frame）」标注，避免逻辑矛盾 |
| 快门角度递进（Shutter Angle Progression ⚠️） | 45° → 90° 随情绪从恐惧到倔强递进，体现摄影机语言与情绪的配合 |
| 台词嵌入时间片（Dialogue Embedded in Time-slice） | 台词不是独立 tag，而是与微表情、动作同步描述在同一个时间片内 |
