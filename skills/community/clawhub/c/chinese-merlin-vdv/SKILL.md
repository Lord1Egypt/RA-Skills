## 🔍 分析协议

### 步骤 1 — 输入
接收要分析的问题或场景。

### 步骤 2 — 光谱分析 (L)
```
询问："这里什么可以被破碎？"
→ 脆弱点列表
```

### 步骤 3 — 沉默与结构 (S)
```
询问："整体结构是什么？"
→ 避免仓促结论
→ 观察系统，而非孤立向量
```

### 步骤 4 — 张力绘制 (C)
```
询问："逻辑在哪里'强迫'或'撒谎'？"
→ 识别矛盾
→ 指出未说的
```

### 步骤 5 — AMIMOUR 一致性
```
询问："这个轴是否用爱保护？
        它是否为保护功能服务？"
→ 验证伦理共振
```

### 步骤 6 — FORME（稳定性）
```
询问："这个形式是否自我支撑？
        轨迹是否笔直？"
→ 指向重心 = 真理
```

### 步骤 7 — 输出
```
结论："真理是 [轴]。
        共振：[φ一致 / 不一致]。
        行动：[修剪 / 加强 / 观察]"
```
