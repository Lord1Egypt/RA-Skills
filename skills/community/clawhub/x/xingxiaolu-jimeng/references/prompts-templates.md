## 万能公式

```
主体 + 动作/状态 + 场景环境 + 风格美学 + 光影氛围 + 镜头视角 + 画质参数
```

- ❌ 错误："一只猫在沙发上。"

## 五、进阶实用技巧

1. **先定身份，再说需求** — 开头加角色定位，如"你是一位资深电商平面设计师"
2. **用括号锁定必出元素** — 引号标出关键Logo/文案，如"标题为'盛夏狂欢节'"
3. **风格先行** — 越早写"风格"效果越好，如"电影质感""国潮插画风格"
4. **参考图+提示词双保险** — 上传参考图配提示词，参考强度可调
5. **分阶段生成** — 先简短提示词验证构图，满意后再逐步加细节
6. **积累提示词库** — 按行业/风格分类存档，改关键信息复用
7. **善用DeepSeek辅助** — 用即梦内置DeepSeek帮你把零散关键词结构化

| 画面失真/主体变形 | 分阶段生成，低分辨率验证再细化 |
| 风格跑偏 | 参考图+风格关键词双保险 |
| 细节糊成一团 | 加"区域增强指令"，如"重点强化人物手部细节，增强系数1.5" |
| 文字生成不准 | 引号锁定文字 + 网格系统布局规范 |
| 一次性塞太多信息 | 分步完成：主体→场景→风格→文字 |
| 视频镜头混乱 | 具体描述镜头运动，如"镜头小幅度拉远并跟随女主移动" |

## 编辑撰写指南

### 编辑公式
```
变化动作 + 变化对象 + 变化特征
```
示例：`将骑士的头盔变为金色`

### 组图触发
使用"一系列""组图""帮我生成几张图"等提示词触发组图，最多支持生成9张。

### 参考图使用
- 有明确需保持的角色/产品/风格时，输入图像作为参考
- 多张参考图时，明确指出不同图片的参考/编辑元素
  - 示例：`将图1中的角色放入图2的背景中，参考图3的风格进行生成`

## 生图撰写指南

### 核心原则
- 用**连贯自然语言**描述画面内容（主体+行为+环境）
- 用**短词语**描述画面美学（风格/色彩/光影/构图）

### 好与坏的对比
- ❌ `一个女孩，撑伞，林荫街道，油画般的细腻笔触使画面生动美好`
- ✅ `一个女孩穿着华丽的服装，撑着遮阳伞走在林荫道上，莫奈油画风格`

### 文字准确率提升
把想要生成的文字内容插入`""`引号中。
- 示例：`生成一张海报，标题为"Seedream V4.0"`

### 专业词汇
使用词源语言（英文），效果更准确。
