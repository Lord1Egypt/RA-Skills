# 基本面分析详细指南

## 一、财务指标速查表

### 利润表
| 指标 | 含义 | 怎么看 |
|------|------|--------|
| 营业收入 | 卖了多少东西 | 持续增长=市占率提升 |
| 营业成本 | 成本控制能力 | 增速<营收增速=好 |
| 毛利润 | 营收-成本 | 毛利率稳定或提升 |
| 销售费用 | 卖东西花了多少 | 费率下降=品牌力强 |
| 管理费用 | 管理效率 | 费率下降=管理好 |
| 研发费用 | 技术创新投入 | 科技公司不能少 |
| 净利润 | 最终赚了多少 | 扣非净利润更有参考价值 |

### 资产负债表
| 指标 | 含义 | 怎么看 |
|------|------|--------|
| 货币资金 | 现金+存款 | 越多越安全 |
| 应收账款 | 别人欠的钱 | 越少越好，大量=回款难 |
| 存货 | 库存商品 | 太多=卖不动 |
| 固定资产 | 厂房设备 | 重资产行业特点 |
| 短期借款 | 一年内要还的 | 别太多 |
| 长期借款 | 长期负债 | 合理杠杆没问题 |
| 所有者权益 | 真正属于股东的钱 | 持续增长 |

### 现金流量表
| 指标 | 含义 | 怎么看 |
|------|------|--------|
| 经营现金流 | 主业赚的钱 | 长期为负=危险 |
| 投资现金流 | 投资花/赚的 | 正=卖资产，负=在投资 |
| 筹资现金流 | 融资/还钱 | 借=负，还=正 |

## 二、ROE杜邦分析

ROE = 净利率 × 资产周转率 × 权益乘数

- **高净利率型**（茅台）：品牌溢价，毛利率高
- **高周转型**（沃尔玛）：薄利多销
- **高杠杆型**（银行）：借钱赚钱

## 三、成长性判断

| 营收增速 | 利润增速 | 判断 |
|:--------:|:--------:|------|
| >30% | >30% | 🟢 高速成长 |
| 15-30% | 15-30% | 🟡 稳定增长 |
| <15% | <15% | 🔶 低速增长 |
| 负增长 | 下降 | 🔴 衰退 |

## 四、估值方法

### PE（市盈率）= 股价 ÷ 每股收益
- 适用于：盈利稳定的公司
- 和行业平均对比
- 和历史分位对比

### PB（市净率）= 股价 ÷ 每股净资产
- 适用于：银行、保险、周期股
- <1 破净 = 可能被低估

### PEG = PE ÷ 利润增速
- <1 = 低估
- 1-2 = 合理
- >2 = 高估

## 五、常见财务排雷

| 危险信号 | 原因 |
|----------|------|
| 应收账款暴增 | 收入可能是假的 |
| 存货暴增无解释 | 产品卖不动 |
| 经营现金流远低于净利润 | 利润质量差 |
| 频繁变更会计政策 | 可能在调节利润 |
| 大额关联交易 | 利益输送嫌疑 |
| 大股东高比例质押 | 资金链紧张 |
| 商誉占比过高 | 收购可能减值 |
| 突然更换审计机构 | 审计不配合 |
