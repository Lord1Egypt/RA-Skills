# 淋巴组织的分类与结构特征

## 触发条件
当组织被确认为参与免疫应答的淋巴样结构时，应用本技能进行分类和结构描述。

## 前提条件
- 组织具有网状组织支架
- 网孔内含有大量淋巴细胞及其他免疫细胞

## 约束条件
- 不适用于非免疫性结缔组织

## 执行步骤

1. **确认淋巴组织基本特征**：检查组织是否以网状组织为支架，且网孔内富含淋巴细胞和其他免疫细胞。

2. **初步分类**：根据组织边界和结构明确性，分为以下两类：
   - **弥散淋巴组织**：无明确界限；含有高内皮微静脉（HEV，直径约30～50μm），其内皮细胞为柱状，可分泌黏附分子，促进淋巴细胞从血液迁移至组织；抗原刺激后可扩大并形成淋巴小结。
   - **淋巴小结（淋巴滤泡）**：呈直径1～2mm的球形小体，界限较清晰；受抗原刺激后增大并形成生发中心。

3. **进一步细分淋巴小结**：
   - **初级淋巴小结**：体积较小，无生发中心。
   - **次级淋巴小结**：由抗原刺激诱导形成，具有明显的生发中心。

4. **描述生发中心结构（如存在）**：
   - **暗区（dark zone）**：较小区域，主要含B淋巴细胞和Th细胞，细胞体积大、嗜碱性强、染色深。
   - **明区（light zone）**：较大区域，含B淋巴细胞、Th细胞、滤泡树突状细胞（FDC）和巨噬细胞。
   - **小结帽（cap）**：位于生发中心周边，由密集的小淋巴细胞构成，呈新月形，顶部最厚。

5. **输出结果**：返回该淋巴组织的具体类型（弥散淋巴组织 / 初级淋巴小结 / 次级淋巴小结）及其关键结构特征，用于组织学识别和免疫状态评估。
