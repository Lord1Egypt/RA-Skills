### 阶段1：选题分析与爆点设计

#### 1.1 选题评分（5维，每项1-5分）

| 维度 | 评分标准 | 权重 |
|------|----------|------|
| **收藏价值** | 用户看完会不会收藏/截图保存？ | ×2 |
| **情绪强度** | 能不能引发「太对了」「先收藏」的情绪？ | ×1.5 |
| **争议空间** | 有没有安全的争议点引发讨论？ | ×1.5 |
| **搜索需求** | 主题搜索量大不大？ | ×1 |
| **差异化空间** | 能不能做出明显不同的角度？ | ×1 |

**总分≥22分才值得做。**

#### 1.2 年龄段锁定

精确到：`0-3月` / `4-6月` / `7-12月` / `1-2岁` / `2-3岁` / `3-4岁` / `4-6岁` / `6-8岁` / `8-10岁` / `10-12岁`

**宁可窄到只服务一个年龄段，也不要泛泛。**

#### 1.3 爆点设计（至少1个）

| 爆点类型 | 示例 |
|----------|------|
| **反常识** | 「退烧药这两个不能一起吃，90%家长不知道」 |
| **可截图** | 可直接保存打印的清单/表格/流程图 |
| **对比冲击** | 正确做法 vs 错误做法 并排展示 |
| **情感共鸣** | 「凌晨3点第5次量体温，我崩溃了」 |
| **懒人福音** | 「一张图搞定，阿姨照着做就行」 |
| **数据支撑** | 「儿科医生协会最新指南：XXX」 |

#### 1.4 知识挖掘（至少5条，分级）

- ⭐ **必存级**：硬核实用信息（≥2条）
- 💬 **讨论级**：有讨论空间的观点（≥2条）
- 🔥 **传播级**：让人想转发/截图的（≥1条）

#### 1.5 输出

```
选题评分：XX/35 | 年龄段：[精确年龄段]
爆点类型：[主要爆点] + [辅助爆点]
知识挖掘：
  ⭐ 必存级：3条
  💬 讨论级：2条
  🔥 传播级：1条
情绪锚点：[具体描述]
```

## 边界情况处理

- **用户没给选题**：先执行0.4时令检测，结合季节/节气/节日自动推荐选题方向，无需反问
- **用户没指定人格**：按关键词规则自动判定
- **用户没指定平台**：默认产出四个平台全部文案
- **用户只要单平台**：按需产出，完成后询问是否需要扩展到其他平台
- **选题超出0-12岁**：提醒专注0-12岁
- **首次使用**：走阶段0完整流程，默认学霸妈妈
- **已有系列**：自动识别进度，续期编号
- **路径未指定**：默认 `../assets/output/`，告知用户
- **目录已存在**：追加日期后缀 `_0615`
