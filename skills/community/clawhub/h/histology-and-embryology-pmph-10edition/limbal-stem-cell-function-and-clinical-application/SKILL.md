# 角膜缘干细胞功能与临床应用

## 何时使用
- 患者确诊为严重眼表疾病（如化学烧伤、Stevens-Johnson综合征）
- 出现角膜上皮持续缺损、混浊或视力下降
- 怀疑角膜缘干细胞缺乏（LSCD）

## 执行步骤

### 1. 定位角膜缘干细胞
- 确认其位于角膜与巩膜交界带（宽约1～2mm）的上皮基底层。

### 2. 识别形态特征
- 上皮厚度 >10 层细胞
- 细胞体积小、核深染
- 基底层细胞呈矮柱状，排列成栅栏样
- 含黑素细胞，不含杯状细胞

### 3. 判断功能机制
- 干细胞持续增殖
- 子代细胞向角膜中央定向移行
- 分化为角膜上皮各层细胞（扁平→多边形→柱状）
- 支持角膜上皮每7天一次的快速更新

### 4. 评估临床意义
- 若角膜缘干细胞缺失，角膜表面将被结膜上皮侵入（含血管和杯状细胞），导致角膜混浊、视力下降
- 确诊后推荐自体或异体角膜缘干细胞移植（limbal stem cell transplantation）

### 5. 实施治疗
- 移植后干细胞重建角膜上皮屏障，恢复角膜透明性
- 注意：角膜缘也是眼球前部手术常用切口部位

## 禁忌与限制
- 不适用于巩膜或结膜上皮再生
- 需确保角膜缘结构完整且干细胞微环境正常作为前提条件
