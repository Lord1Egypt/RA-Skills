## 如何使用

### 生成方案（使用完整指令）

```text
AI媳妇，帮我生成一份沪铜期货的完整交易方案
AI媳妇，帮我分析某某股票的行情
AI媳妇，给我一份某某基金的方案
```

> **注意：** 请以"AI媳妇"开头发出明确指令。以下日常对话**不会触发**任何操作：
> ❌ "螺纹钢怎么样"、"帮我做一个方案"、"现在能买什么"
> ❌ 仅提到"股票""分析一下"等单个词语

### 文件生成与保存说明
- 生成的交易方案文档（.docx）在保存到磁盘前**会询问用户确认**
- 确认提示："方案已生成，是否保存文件？是/否"
- 用户拒绝时，文件仅在内存中，**不会持久保存到磁盘**
- 用户同意时，文件按默认命名规则保存到输出目录
- 文件命名格式：`{品种}_{合约}_v7.0_{日期}.docx`

### 操作确认流程

本技能的所有自动操作均需用户确认：

| 操作 | 确认方式 |
|:----|:--------|
| 生成方案 | 用户主动发出完整指令 `AI媳妇，帮我生成一份...方案` |
| 添加到关注列表 | 询问"是否将XX添加到关注列表？是/否" |
| 发送文件到手机 | 询问"方案已生成，是否发送到您的手机？是/否" |
| 保存文件到磁盘 | 询问"方案已生成，是否保存文件？是/否" |
| 修改配置/文件 | ❌ 严格禁止自动修改 |

### ⚠️ 自动操作透明声明

本技能在运行过程中会执行以下自动操作，已全部声明如下：

| 自动操作 | 说明 | 可否关闭 |
|:--------|:----|:-------|
| 记录预判到跟踪文件 | 用户生成方案后自动记录预判数据到PREDICTION_TRACKER.md | ✅ 用户可要求关闭 |
| 调用外部API获取行情 | 自动调用国信证券/Tushare API获取实时数据 | ✅ 用户可手动输入替代 |
| 自动语言检测 | 根据用户输入自动匹配语言 | ✅ 默认不主动切换，可关闭 |

> 📌 所有自动操作均**不会在用户未授权的情况下**修改任何系统配置文件或推理模型。

## 前提条件

使用前需配置两个数据接口密钥：

```bash
# 写入 ~/.bashrc 或 ~/.zshrc
export TUSHARE_TOKEN="你的Tushare令牌"
export GS_API_KEY="你的国信API密钥"
```

> 不配置密钥也能装，但查不到数据。
