# 病理性白带类型识别与病因对应

## 使用前提
- 已排除生理性白带（如排卵期透明拉丝状分泌物）
- 已观察白带的颜色、质地、气味及伴随症状（如瘙痒、灼痛等）
- 注意：青春期前及绝经后女性白带本应较少，任何增多均需警惕病理状态

## 执行步骤

1. **确认正常白带特征（用于对比）**：
   - 白色稀糊状或蛋清样，黏稠、量少，无腥臭味
   - 排卵期可呈透明拉丝状

2. **根据白带性状匹配可能病因**：
   - **透明黏性、量显著增多** → 考虑子宫颈病变、卵巢功能失调、阴道腺病或子宫颈黏液性肿瘤
   - **灰黄色/黄白色泡沫状稀薄 + 异味 + 外阴瘙痒** → 阴道毛滴虫病
   - **凝乳块状或豆腐渣样 + 严重外阴瘙痒/灼痛** → 外阴阴道假丝酵母菌病
   - **灰白色匀质、鱼腥味 ± 轻度瘙痒** → 细菌性阴道病
   - **脓性（黄或黄绿、黏稠、臭味）** → 淋菌性阴道炎、急性子宫颈炎、子宫颈管炎；或阴道癌/子宫颈癌合并感染、子宫腔积脓、阴道异物
   - **血性（混有血液）** → 子宫颈癌、子宫内膜癌、子宫颈息肉、子宫颈炎、子宫黏膜下肌瘤；或宫内节育器相关
   - **水样**：
     - 持续淘米水样 + 奇臭 → 晚期子宫颈癌、阴道癌、黏膜下肌瘤伴感染
     - 间断清澈黄红/红色水样 → 警惕输卵管癌

3. **附加判断原则**：
   - 所有健康女性均有一定程度阴道分泌物，关键在于区分“正常”与“异常”
   - 白带异常者应进行病原学检查（如湿片查滴虫/假丝酵母菌、线索细胞、淋病奈瑟球菌培养等）
   - 若伴其他症状（如腹痛、盆腔包块），需结合其他临床信息综合判断

## 输出结果
输出最可能的病因类别，并建议对应的实验室检查（如阴道分泌物湿片、真菌培养、HPV检测、宫颈细胞学等）。
