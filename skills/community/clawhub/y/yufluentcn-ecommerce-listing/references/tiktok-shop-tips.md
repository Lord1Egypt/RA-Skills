# TikTok Shop 平台规则库（Harness 权威源 · RAG 数据源）

> 更新日期：2026-05-28（v1.2 扩库）  
> 用途：Listing / SEO / 客服 scene 注入；超长时由关键词 RAG 按 query 选段。  
> **注意**：`tiktok` = TikTok **Shop 商品**；广告投放规则见 `platforms/tiktok_ads/rules.md`（`ad_optimize` scene）。  
> 数据来源：TikTok Shop Seller Center 政策摘要 + 短视频带货实践（须人工核对，勿编造认证）。

---

## 发现与推荐（For You / 搜索 / Shop Tab）

### 相关性信号

- **商品标题**约 **150 字符**内（含空格）；前 40 字符应含核心品类词（搜索与卡片展示）。
- **类目**选择错误会导致流量错位与审核驳回；文案须与类目属性、资质一致。
- **主图/视频封面**：高对比、单一主体；与标题卖点一致，避免封面党。
- **销量、评价、发货时效**影响推荐；文案应降低预期落差以减少差评。

### 转化信号

- 短视频 **3 秒钩子（hook）** 与标题利益点一致。
- 价格带、包邮门槛在 Listing 与直播间话术一致。
- **评价与退货率**：尺码/颜色/功效夸大是差评主因。

## 商品标题（Title）

### 通用限制

- 建议 **≤150 字符**；可适度使用 **1–2 个 emoji**（📦✨等），勿整行 emoji。
- 结构：`[核心品类词] + [关键卖点/规格] + [场景] · [品牌]`。
- **禁止**：全大写、连续「!!!」、未授权「官方」「旗舰店」宣称（除非品牌授权）。

### 标题禁用模式

- 「全网最低价」「销量第一」「网红同款」**无依据时禁止**。
- 堆砌 #话题 在标题内（话题应放 tags 字段，非标题）。
- 未授权影视/游戏/明星 IP、球队、品牌联名表述。
- 把直播限时价写进长期标题——用活动工具管理。

### 子类目标题差异

- **3C**：容量、mAh、协议、兼容型号；禁「军工」「监听」类词。
- **服装**：尺码范围、面料；禁「显瘦 10 斤」等不可验证表述。
- **美妆**：ml 容量、肤质；禁医疗功效。
- **食品**：保质期、过敏原；禁减肥治愈宣称。

## 商品描述与 Bullet（短描述）

### 格式要求

- **短描述**：2–4 段口语化段落，像朋友种草；每段 1–3 句，避免长墙字。
- **3 条 bullet**（若模板要求）：  
  1. 一句话核心卖点  
  2. 场景/效果 + 可验证参数  
  3. 价格/套装/售后（合规范围内，不虚假限时）
- 总长度建议 **400–1200 字**；可含换行，避免 HTML 复杂标签。

### 内容结构建议

1. **Hook 复述**：承接短视频开场问题。  
2. **痛点 → 解决方案**：谁买、解决什么。  
3. **参数与证据**：尺寸、材质、检测认证（有则写）。  
4. **使用场景**：通勤、健身、宿舍等具体画面。  
5. **下单须知**：规格选择、发货地、退换概要（与店铺政策一致）。

### 描述禁用内容

- 站外导流（微信/WhatsApp/独立站链接）——违反平台引流政策。
- 医疗宣称：cure, treat, 100% effective, prescription。
- 绝对化：best, #1, 第一, 最好（平台敏感）。
- 索评、返现改评、私下退款承诺。

## 话题标签（Hashtags）与 hook

- 提供 **3–5 个 `#话题`**：2 个品类大词 + 2 个场景长尾 + 1 个品牌/系列（若有）。
- 话题须与实物相关，禁止蹭无关热点政治/敏感事件。
- **hook**（3 秒口播/字幕首句）：疑问句或反差句，例：「还在为耳机掉跑步烦恼？」
- 直播话术与 PDP **价格、赠品、库存** 一致，避免「直播专享」在录播中永久展示。

---

## 短视频与直播协同（文案侧）

- **9:16 竖屏**优先；字幕可读，关键卖点前 3 秒出现。
- **演示用法**优于纯口播；3C 需展示真实开机/佩戴效果（勿夸大）。
- **音乐版权**：使用平台音乐库；文案不提「无版权风险」误导。
- **未成年人**：不得向未成年推销违禁品；玩具需适龄警示。
- **直播违禁**：虚假库存、演戏砍价、医疗诊断均为高风险。

## 禁用词汇与合规（全局）

### 绝对化与促销类

| 禁用 | 替代表述 |
|------|----------|
| 第一 / 最好 / 全网最低 | 热销款 / 高口碑 / 限时优惠（须有活动） |
| 100% 有效 / 包治 | 帮助改善…外观 / 个人体验因人而异 |
| 官方正品（无授权） | 品牌直发 / 授权经销商（须有证明） |
| 网红同款（无授权） | 相似风格 / 灵感款 |

### 医疗、美妆、食品

- 护肤：禁止「7 天去皱」「祛斑」；用「改善肌肤状态」「提亮观感」。
- 减肥食品：禁止「快速瘦身」「替代药物」。
- 医疗器械：需类目资质；consumer 文案不得写 prescription。

### 认证与资质

- **CCC、CE、FCC、FDA** 等：仅在有文件时写；跨境站点规则不同。
- 化妆品备案号（中国）、SCPN（欧）等按站点要求，勿虚构编号。

### 侵权

- 禁止 counterfeit、未授权 logo、影视台词商品化。
- 「Compatible with」需真实，不暗示品牌官方。

## 类目合规要点（与 categories/*.md 协同）

### 电子产品（electronics）

- 锂电池：Wh、UN38.3；航空限制说明。
- 无线：频段合规；IP 等级需报告。
- 兼容：列出型号年份，降低退货。

### 服装鞋帽（apparel）

- 尺码表 mandatory；面料成分百分比。
- 色差：说明拍摄光线可能导致轻微色差。

### 家居（home）

- 尺寸 cm/inch；承重测试值；安装工具是否含。

### 美妆个护（beauty）

- 全成分或关键成分；过敏原提示。
- 防晒 SPF 须备案；染发剂警示语。

## 跨境与站点差异

- **美国 / 英国 / 东南亚** 等站点：语言、货币、禁售清单不同；`lang` 参数决定输出语言。
- **税务与关税**：正文不写「免税到手价」除非运营确认；Harness 不生成税务建议。
- **本地仓 / 跨境直邮**：发货时效须在描述中与物流模板一致。
- **多语言 Listing**：核心卖点翻译一致，禁词表各语言均需检查。

---

## 图片、视频与 Listing 一致性

- 主图：清晰白底或场景图；勿过度滤镜导致货不对板。
- 视频封面文字 ≤ 少量关键词，避免遮挡产品。
- 文案不得描述视频/主图未展示的功能或赠品。
- **套装**：件数、颜色组合与 SKU 一致。

---

## 常见下架与限流原因

1. 类目错放或资质缺失（美妆、食品、3C）。  
2. 医疗/减肥绝对化宣称。  
3. 侵权 IP 或假冒。  
4. 站外导流与私下交易。  
5. 虚假宣传销量/评价。  
6. 未成年人不当营销。  
7. 电池/液体/刀具等受限品未申报。  
8. 直播与 PDP 价格不一致引发投诉。

## 2025–2026 TikTok Shop 趋势（运营关注）

- **内容电商**：短视频 + 直播权重高于纯图文；Listing 为转化落地页。  
- **搜索电商**：标题与 tags 对 Shop 内搜索越来越重要。  
- **合规加严**：功效宣称、达人带货 disclosure（#ad）在部分市场强制。  
- **全托管 / 半托管**：库存与发货承诺须与模式一致。  
- **AI 生成**：卖家对准确性负责；禁止深度伪造代言人。

## 关键词与 tags 策略（与 seo_keywords scene 协同）

- 核心词：标题前部 + 第一条 bullet。  
- 长尾：tags + 描述场景句。  
- 蹭热词：仅在与产品真实相关时；禁止敏感政治/灾难营销。  
- 与 Amazon/Shopify 同源 SKU：可复用卖点但**必须改写句式**并遵守本平台禁词。

## 搜索权重策略（2025+ 重要）

### 搜索排名因子

1. **标题匹配度**（权重最高）：前 40 字符必须含核心搜索品类词
2. **商品标签（hashtags）**：3–5 个标签中至少 2 个为高搜索量品类词
3. **类目准确度**：错放类目会被搜索降权
4. **销量与评价**：高销量 SKU 搜索排名加权（冷启动期靠标题优化）
5. **商品描述关键词密度**：首 2 段内容应含长尾搜索词

### 标题优化策略

- 格式：[核心搜索词] + [差异化卖点] + [场景] · [品牌]
- 例：`便携咖啡机 户外露营 15Bar高压萃取 · BrandX`
- 不要在前 40 字符放品牌名（除非品牌认知度高）
- 核心搜索词与长尾场景词须与 `{language_prompt}` 输出语言一致（中文站点用中文搜索词）
