# 颅骨整体结构与临床应用

## 何时使用
- 进行颅脑外伤初步评估时
- 规划神经外科开颅手术入路时
- 解读头颅CT/MRI等影像资料时
- 判断婴幼儿年龄（通过囟门或颅缝闭合状态）时

## 执行步骤
1. **确认颅骨整体性**：除下颌骨（可活动）和舌骨（游离）外，其余颅骨通过纤维膜和软骨牢固连接成一个整体结构。
2. **识别关键形态特征**：
   - 顶面呈卵圆形（前窄后宽）
   - 后面可见枕外隆凸及上、下项线
   - 内面有脑回压迹与脑膜中动脉沟
   - 底面呈三级阶梯状：颅前窝、颅中窝、颅后窝
3. **定位主要颅缝**：
   - 冠状缝：额骨与顶骨之间
   - 矢状缝：两侧顶骨之间（后份常有顶孔）
   - 人字缝：顶骨与枕骨之间
4. **应用临床逻辑**：
   - 局部颅骨骨折可能因整体性影响邻近区域，需全面评估
   - 颅缝闭合时间可用于法医学或儿科年龄推断（如新生儿前囟未闭）
   - 手术入路选择需避开重要血管沟及脑功能区对应的颅骨内面标志

> **注意**：本技能不适用于下颌骨运动分析或舌骨独立功能评估。
