# 示例输出对比

## 示例 1：小红书风格

**输入（AI 原文）：**
```
首先，在当今竞争激烈的职场环境中，持续学习新技能是至关重要的。
其次，良好的人际关系网络也是职业发展的关键因素。
最后，保持积极的工作态度同样不可忽视。
总而言之，只有将这三者有机结合，才能在职场中立于不败之地。
```

**输出（小红书版）：**
```
职场混得好的人，早就不信"努力就能赢"那套了💡

技能得一直更新，没错
但更重要的是——别一个人闷头干
认识对的人，比死磕更有用
心态别崩，槽可以吐，活不能停

三者攒齐，机会自然来

#职场成长 #打工人日常 #升职加薪
```

## 示例 2：知乎风格

**输入（AI 原文）：**
```
人工智能技术正在深刻地改变着我们的生活方式。
首先，在医疗领域，AI辅助诊断系统已经能够帮助医生更准确地识别疾病。
其次，在教育领域，个性化学习系统可以根据学生的特点提供定制化教学内容。
最后，在交通领域，自动驾驶技术也在稳步推进中。
```

**输出（知乎版）：**
```
AI 到底改变了什么？说几个我亲眼看到的。

1. 看病这事，AI 比很多新手医生靠谱
   我了解到的一些医院，AI 看 CT 片的准确率已经超过 90%。
   你说医生会失业？我觉得不会，但不会用 AI 的医生可能会。

2. 教育终于开始"因材施教"了
   不是概念，是真的有产品在做了。每个学生做题错了哪道，系统自动分析薄弱点，推对应的练习。我表弟在用，数学从 70 分提到了 85。

3. 自动驾驶比你想象中更近
   虽然离 L5 还远，但高速路段的辅助驾驶已经很成熟了。

说说你的看法？你觉得 AI 哪个领域落地最快？
```

## 示例 4：公众号风格

**输入（AI 原文）：**
```
不可否认的是，品牌建设对于企业长期发展具有至关重要的战略意义。
一个优秀的品牌不仅能够帮助企业获取更高的溢价空间，更能在竞争激烈的市场环境中构建起难以逾越的护城河。
因此，每一个企业家都应该将品牌建设纳入企业的核心战略。
```

**输出（公众号版）：**
```
品牌到底值不值得砸钱做？说三个真实案例。

第一个，做消费品的 A 公司，同样的产品换了个包装和定位，「价格翻了 3 倍，销量没降反而涨了」——这就是品牌的力量。

第二个，B 公司走纯性价比路线，没有品牌投入。疫情期间竞争对手品牌强，消费者宁可多等两天也要买有品牌的，B 公司直接掉了一半营收。

第三个，C 公司老板自己就是品牌——每天发朋友圈、拍短视频，公司没花一分钱广告费，获客全靠他个人 IP。

品牌不是大公司的专利。普通人也可以从「让别人记住你是谁」开始。
```
