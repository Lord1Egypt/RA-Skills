## Description: <br>
Search the web using Cloudsways SmartSearch API, returning relevant results with dynamic summaries, snippets, and optional full-text content for up-to-date information, news, or deep research. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[prismheart](https://clawhub.ai/user/prismheart) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agents use this skill to run web searches through Cloudsways SmartSearch and retrieve snippets, dynamic key fragments, or optional full-text content for current-information and research workflows. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Search queries and requested page-content fetches are sent to an external Cloudsways search provider. <br>
Mitigation: Avoid sending secrets, credentials, private customer data, confidential project names, or other sensitive internal information in queries. <br>
