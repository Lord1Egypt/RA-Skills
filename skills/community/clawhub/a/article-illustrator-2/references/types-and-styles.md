# Illustration Types and Styles

## Type

| Value | Best For |
| --- | --- |
| `infographic` | data, metrics, and conceptual explanation |
| `scene` | narrative or emotional scenes |
| `flowchart` | processes, steps, and workflows |
| `comparison` | side-by-side options and trade-offs |
| `framework` | models, systems, and methodology |
| `timeline` | history, evolution, and progression |
| `mixed` | multiple illustration types within one article |

## Style

| Value | Best For | Recommended Mapping |
| --- | --- | --- |
| `minimal` | clean, professional explanatory writing | `editorial` |
| `notion` | tool-like, knowledge-card explanations | `flat-illustration` |
| `blueprint` | technical diagrams and structures | `infographic` |
| `watercolor` | softer narrative or human-centered topics | `watercolor` |
| `elegant` | editorial and brand-oriented visuals | `editorial` |
| `poster` | stronger visual punch | `poster` |

## Density

| Value | Meaning |
| --- | --- |
| `minimal` | 1 to 2 images in the article |
| `balanced` | 2 to 4 images in the article |
| `per-section` | one image for each major section |
| `rich` | frequent, highly visual explanation throughout |
