# 合同解除通知书

## 适用场景
因对方违约、不可抗力或其他法定/约定事由，单方面通知对方解除合同，作为诉讼或仲裁前的法定前置步骤。

## 提交机构
无需提交机构，直接发送给对方当事人（建议通过EMS/挂号信保留送达证据）

## 模板正文

```
                    合同解除通知书

致：______（合同相对方全称/姓名）

发件人：姓名/名称______
        身份证号/统一社会信用代码：________________
        联系电话：__________________

鉴于：

双方于______年______月______日签订了《______合同》
（合同编号：______）。合同约定：________________________。

合同履行过程中，贵方存在以下违约行为：
1. ______年______月______日，________________________；
2. ___________________________________________________。

该行为已构成：
□ 根本违约（《民法典》第563条）
□ 迟延履行经催告后仍未履行（《民法典》第563条第3项）
□ 符合合同第______条约定的解除条件

______年______月______日，发件人已向贵方发出书面催告，
要求贵方在______日内纠正违约行为，但贵方至今未采取任何补救措施。

根据《中华人民共和国民法典》第563条（或第565条）之规定，
以及双方合同第______条之约定，发件人现正式通知贵方：

自本通知送达贵方之日起，《______合同》正式解除。

请贵方于收到本通知之日起______日内：
1. 返还已收取的发件人支付的款项______元；
2. 支付违约金______元；
3. 赔偿因违约给发件人造成的损失______元；
4. 完成已交付财产的清点和返还。

逾期未履行上述义务，发件人将依法追究贵方的违约责任。

特此通知。

                                    通知人（签名/盖章）：
                                    ______年______月______日
```

## 注意事项
- **关键填写点**：准确描述违约事实和对应的法定/约定解除依据；若有催告程序须写明催告时间和未履行事实；解除后果（返还款项、赔偿损失等）要明确具体
- **常见错误**：不符合解除条件擅自解除构成违约；未保留送达证据导致无法证明合同已解除；解除通知发送后仍继续履行合同造成默认追认
- **份数要求**：正本发送对方，发件人自留1份（附送达回执）
- **送达方式**：EMS/挂号信邮寄，保留邮寄凭证和签收回执；也可当面送达并取得签收确认
- **法律效果**：合同自通知到达对方时解除；对方有异议的可请求法院/仲裁机构确认解除效力
