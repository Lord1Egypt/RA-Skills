## Description: <br>
Lists the last 10 emails from the user's inbox using Himalaya for quick recent email review. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[shrivastavarohit](https://clawhub.ai/user/shrivastavarohit) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
People who use Himalaya for email can ask an agent to quickly list the most recent 10 message envelopes from their mailbox. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill reads recent email envelopes from the user's mailbox, and broad activation wording could run it when a user casually asks to check email. <br>
Mitigation: Review the skill before installing, use explicit prompts for checking recent email, and run it only where Himalaya mailbox access is acceptable. <br>
