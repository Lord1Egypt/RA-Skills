# 开发任务清单与依赖拓扑

智能硬件开发按产品生命周期组织的任务类型清单，附任务间依赖关系和UTOS元操作映射提示。

**域间逻辑流**：需求(H1) → 架构(H2) → 硬件(H3) → 固件(H4) → 结构(H5) → 原型(H6) → 量产(H7) → 运维(H8)

项目管理H9贯穿全流程。

## 依赖拓扑摘要

以下为任务间的主要依赖链路，UTOS执行轴可据此自动编排管线：

**需求-架构链**: H1-01 → H1-06 → H2-01 → H2-02 → H2-03 / H1-02 → H2-02
**架构-硬软并行链**: H2-01 → [H3-01 ‖ H4-01 ‖ H5-01] （硬软结构三路并行）
**硬件设计主链**: H3-01 → H3-02 → H3-03 → H3-04/H3-05/H3-06 → H3-09 → H3-10
**固件开发主链**: H4-01 → H4-02 → H4-03 → H4-04 → H4-05/H4-06 → H4-07
**结构设计链**: H5-01 → H5-02 → H5-03/H5-04 → H5-05 → H5-06
**原型三阶段链**: H6-01 → H6-02(EVT) → H6-03(DVT) → H6-04(PVT) → H6-05
**量产准备链**: H7-01 → H7-02 → H7-04 → H7-06 → H7-08
**供应链支撑链**: H3-02 → H7-03 → H7-06 → H7-08
**运维服务链**: H8-01 → H8-02 → H8-03/H8-04 → H8-05
**项目管控链**: H9-01 → H9-02 → H9-03贯穿全流程 → H9-04(各阶段) → H9-06
**认证合规链**: H3-09 → H6-03 → H6-06 → [证书获取] → H7-08
**问题闭环链**: H6-07 → [持续追踪] → H8-05 → H4-04(修复) → H8-04(发版)

更多组合由UTOS根据具体任务动态推导。
