# douyin-scraper 自然语言搜索示例
# ==============================================
# 以下所有句式都可以直接使用，Skill 会自动识别并执行搜索

# ===== 标准句式 =====
搜索一下海鲜视频
搜索一下美食探店视频
搜索一下搞笑段子
搜索一下健身视频
搜索一下化妆教程
搜索一下Python教程
搜索一下摄影技巧

# ===== 变体句式 =====
帮我搜一下海鲜视频
帮我找美食探店视频
帮我查找搞笑内容
找一找做饭教程
找一下跳舞视频

# ===== 极简句式 =====
海鲜视频
美食探店
搞笑段子
健身
化妆教程
Python

# ===== 更口语化的句式 =====
我想看海鲜视频
给我搜搜美食怎么做
有没有好玩的段子
我想学习一下Python
帮我找点健身的视频看看

# ===== 带修饰的句式 =====

搜索一下高赞的搞笑视频
