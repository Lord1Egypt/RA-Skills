### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Normal runs contact SkillPay and can trigger a 0.002 USDT charge. <br>
Mitigation: Use --test while evaluating the skill and install it only if the pay-per-use model is acceptable. <br>
Risk: The skill depends on a local OpenClaw/Sloan setup to generate output. <br>
Mitigation: Confirm the local OpenClaw Gateway and Sloan agent are trusted and available before operational use. <br>
Risk: Dependency lock data references HTTP npm mirror URLs. <br>
Mitigation: Regenerate dependencies from an HTTPS npm registry in stricter environments. <br>
