# 债务周期识别与分析指南

> 基于达利欧《债务危机》（A Template for Understanding Big Debt Crises）

---

## 债务周期的两种类型

### 短期债务周期（Business Cycle）
- **时长**：约 5-8 年
- **驱动**：信贷收缩/扩张
- **特征**：通常以衰退/复苏形式体现

### 长期债务周期（Long-term Debt Cycle）
- **时长**：约 75-100 年
- **驱动**：储备货币信誉的积累与消耗
- **终点**：货币体系重置（如布雷顿森林体系终结，1971）

---

## 长期债务周期六阶段

### 阶段一：早期扩张（Early Expansion）
**特征**：
- 债务/GDP 处于低位（<60%）
- 信用良好，借贷成本低
- 经济增长强劲，通胀温和

**资产配置**：重仓股票 > 债券 > 商品

---

### 阶段二：泡沫积累（Bubble Building）
**特征**：
- 债务增速 > GDP 增速
- 资产价格脱离基本面
- 投资者风险偏好极高

**预警信号**：
- 杠杆贷款激增
- 首次公开募股（IPO）数量创历史纪录
- 普通投资者大规模入市

**资产配置**：开始减仓、增加抗通胀资产

---

### 阶段三：顶部与收紧（Peak & Tightening）
**特征**：
- 央行开始加息
- 通胀压力上升
- 资产价格见顶

**触发因素**：
- 通胀持续超过目标
- 就业市场过热
- 货币信誉开始受质疑

**资产配置**：大宗商品、TIPS、短期债券

---

### 阶段四：去杠杆（Deleveraging）
**特征**：
- 债务违约增加
- 信贷紧缩
- 资产被抛售
- 失业率上升

**去杠杆四种工具**（达利欧分类）：
1. **紧缩**：削减支出（通缩性）
2. **债务重组**：违约/减记（通缩性）
3. **财富再分配**：税收政策（中性）
4. **货币印发**：QE（通胀性）

> 达利欧认为"美好的去杠杆"需要四种工具的平衡使用，避免通缩或通胀走极端。

**资产配置**：现金、黄金、短期国债

---

### 阶段五：滞胀（Stagflation）
**特征**：
- 高通胀 + 低增长（甚至衰退）同时存在
- 央行两难：降息助通胀，加息伤增长
- 实际收益率为负

**当前（2026年美国）处于此阶段的证据**：
- 财政赤字占 GDP 约 7%
- 通胀仍高于目标
- 经济增长放缓
- 美债海外需求下降

**资产配置**：
```
黄金          +20%（超配）
大宗商品      +15%（超配）
TIPS          +10%（超配）
股票          -10%（低配）
长期国债      -20%（大幅低配）
```

---

### 阶段六：新货币秩序（New Monetary Order）
**特征**：
- 旧货币体系失信或崩溃
- 新储备货币/体系形成
- 极度痛苦后的重建期

**历史案例**：
- 1933 年美国放弃金本位
- 1944 年布雷顿森林体系建立
- 1971 年尼克松关闭黄金窗口
- 下一次重置：？（达利欧认为正在逐步发生）

## 债务周期定位工具

### 快速判断清单

**当以下条件 ≥5 个满足时，处于债务周期后期（高风险区）**：

- [ ] 政府债务/GDP > 100%
- [ ] 财政赤字/GDP > 5%
- [ ] 利息支付/政府收入 > 15%
- [ ] 经常账户赤字/GDP > 3%
- [ ] 通胀 > 央行目标 1.5 倍以上
- [ ] 央行资产负债表仍处于历史高位
- [ ] 实际利率长期为负
- [ ] 国债外国持有比例持续下降
- [ ] 货币信誉开始受到质疑
- [ ] 黄金储备增速加快（各国央行）

### 美国当前得分（2026 年 4 月）

| 指标 | 状态 | 是否触发 |
|------|------|---------|
| 债务/GDP ~130% | ✓ | 是 |
| 赤字/GDP ~7% | ✓ | 是 |
| 利息/收入 ~18% | ✓ | 是 |
| 通胀仍高于目标 | ✓ | 是 |
| 美联储资产负债表仍大 | ✓ | 是 |
| 实际利率偏低 | 部分 | 是 |
| 国债海外需求下降 | ✓ | 是 |

**触发数量：7/10 → 处于债务周期高风险后期**

## 货币信誉框架

储备货币地位的三个核心支柱：

1. **经济体量**：GDP、贸易量、金融市场深度
2. **信誉与稳定**：通胀历史、制度稳定性、法治程度
3. **网络效应**：使用习惯、结算基础设施、锚定国数量

**信誉侵蚀的四大信号**：
1. 债务货币化（印钱还债）
2. 赤字持续扩大无法约束
3. 地缘冲突导致制裁工具滥用
4. 竞争货币/资产崛起（人民币、黄金、数字货币）

> 达利欧：**"储备货币的本质，是别人愿不愿意持有你发行的债务。"**
