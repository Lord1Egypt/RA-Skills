## Known Risks and Mitigations: <br>
Risk: Some Git examples can discard work, delete references, alter remotes, or affect collaborators if run in the wrong repository or branch. <br>
Mitigation: Review the repository, branch, remote target, and command intent before execution; use preview commands such as git clean -n where available. <br>
