# 纹身卫生安全、护理与遮盖修改指南

本文档涵盖纹身全流程的卫生安全标准、术后护理、以及遮盖/修改技术。
当用户讨论纹身安全、护理、遮盖或修改时加载本文档。

---

## 一、卫生安全标准

### 1.1 纹身前安全检查清单

在进入任何纹身工作室前，确认以下事项：

**工作室资质**:
- [ ] 营业执照（个体工商户/公司）在显著位置公示
- [ ] 《医疗机构执业许可证》或有纹身经营范围的营业执照
- [ ] 公共场所卫生许可证
- [ ] 从业人员健康证在有效期内

**一次性耗材**:
- [ ] 针头：必须一次性，当场拆封
- [ ] 针嘴/针管：一次性使用，当场拆封
- [ ] 色料杯（ink cap）：一次性
- [ ] 手套：纹身师全程佩戴一次性医用手套，接触非消毒物后立即更换
- [ ] 擦拭纸/棉片：一次性，不得重复蘸取色料

**消毒设备**:
- [ ] 高压蒸汽灭菌器（autoclave）：非一次性器械（手柄、管子）必须经此消毒
- [ ] 灭菌指示条/包内卡：证明达到灭菌温度（121°C/134°C）
- [ ] 紫外线消毒灯：工作台面消毒
- [ ] 消毒液：75% 酒精或含氯消毒剂在现场

**环境**:
- [ ] 纹身区域与非纹身区域物理分隔
- [ ] 操作台面干净、有一次性台布覆盖
- [ ] 无宠物在纹身区域活动
- [ ] 空气流通但不直吹纹身区域

**纹身师操作规范**:
- [ ] 操作前洗手消毒
- [ ] 全程戴手套
- [ ] 剃毛刀片一次性
- [ ] 色料倒入一次性杯子使用，不得从瓶中直接蘸取
- [ ] 使用过的针头放入锐器盒

### 1.2 绝对禁忌：以下情况不能纹身

| 禁忌情况 | 原因 |
|----------|------|
| 孕期/哺乳期 | 感染风险 + 色料成分不确定对胎儿/婴儿影响 |
| 糖尿病未控制 | 伤口愈合困难，感染风险极高 |
| 血友病/凝血功能障碍 | 大量出血风险 |
| 严重心脏病 | 疼痛刺激 + 感染可能引发心脏问题 |
| 癫痫频繁发作期 | 疼痛可能诱发发作 |
| 免疫系统疾病（如红斑狼疮、HIV未控制） | 感染风险极高 |
| 皮肤病活动期（牛皮癣、湿疹、白癜风活动期） | 可能导致 Koebner 现象（新皮损） |
| 过敏性体质（对色料成分过敏） | 严重过敏反应，可能需手术移除 |
| 服用抗凝药物（阿司匹林/华法林等） | 大量出血 + 影响上色 |
| 饮酒后 | 血液稀释 + 判断力下降 |
| 18 周岁以下 | 中国法律规定，纹身店不得为未成年人纹身 |

### 1.3 纹身前个人准备

- 充足睡眠（前一晚 7-8 小时）
- 纹身前吃饭（不能空腹，防止低血糖晕厥）
- 纹身前洗澡（保持皮肤清洁）
- 纹身区域皮肤无破损、晒伤、皮疹
- 穿着宽松舒适衣物（方便露出纹身区域且不摩擦）
- 不饮酒（前一天 + 当天）
- 不服用阿司匹林/布洛芬等抗凝药（至少 48 小时前停用）
- 女性避开月经期（痛感升高 + 凝血功能下降）

## 二、纹身后护理（Aftercare）

### 2.1 第一阶段：愈合期（1-3天）

**敷料处理**:
- 保鲜膜/医用敷料：纹身后 2-4 小时取下（不要超过 6 小时，否则滋生细菌）
- 二合一修复膜（Second Skin/Tegaderm 等）：可保留 24-72 小时，按产品说明

**首次清洗** (取下敷料后):
1. 用温水冲洗（不要直接对纹身冲，让水流过）
2. 使用无香精、无酒精的温和皂液
3. 用手指肚轻揉，不要用指甲/毛巾/浴花
4. 冲洗干净后，用一次性面巾纸轻轻蘸干（never 来回擦）
5. 晾干 5-10 分钟（自然风干）

**注意事项**:
- 前 48 小时组织液渗出为正常现象
- 微红、微肿为正常炎症反应
- 如持续大量渗出、异常红肿热痛→立即就医

### 2.2 第二阶段：结痂/脱皮期（3-14天）

**涂抹护理**:
- 薄涂专用修复膏（推荐：Aquaphor / Hustle Butter / Redemption / 纹身师推荐的品牌）
- 每天 2-3 次，薄薄一层即可（过度涂抹会堵塞毛孔、延迟愈合）
- 涂抹前必须洗手

**禁止事项**:
- 🚫 绝对不能抠痂皮！（抠掉痂皮 = 抠掉色料，留下白斑）
- 🚫 不能泡澡/游泳/桑拿/温泉（至少 2-4 周）
- 🚫 不能暴晒（至少 4 周，实际上永远不建议暴晒）
- 🚫 不能穿紧身衣物摩擦纹身区域
- 🚫 不能用酒精/双氧水消毒（会破坏正在生长的皮肤）
- 🚫 不能用凡士林/婴儿油（过于厚重，堵塞毛孔）
- 🚫 不能宠物舔舐纹身
- 🚫 不能剧烈运动大量出汗（至少 1 周）

**正常 vs 异常**:

| 正常愈合现象 | 异常信号（需就医） |
|-------------|-------------------|
| 轻微瘙痒 | 剧烈瘙痒+皮疹（过敏/感染） |
| 结薄痂 | 厚黄色痂（感染） |
| 轻微发红 | 红热蔓延超出纹身边缘（蜂窝织炎） |
| 少量组织液 | 脓液/恶臭分泌物（感染） |
| 微肿 | 严重肿胀（过敏反应） |

### 2.3 第三阶段：成熟期（2周-3个月）

- 皮肤表面已愈合，但深层仍在修复
- 纹身可能看起来"发雾"或比预期浅——这是皮肤最外层再生正常现象
- 继续保持保湿
- 可恢复正常洗澡/运动，但仍避免长时间浸泡

### 2.4 长期维护

- **防晒是关键**：紫外线是纹身褪色的头号敌人，户外活动必须涂 SPF 50+ 防水防晒
- 保持皮肤滋润（日常 body lotion 即可）
- 定期检查纹身，发现褪色及时补色
- 一般纹身 5-10 年后可能需要 touch-up

## 三、遮盖纹身（Cover-Up）

### 3.1 遮盖基本原理

**遮盖 ≠ 删除**：遮盖是用新纹身盖住旧纹身，本质是"用更深的颜色压住更浅的颜色"

**核心规则**:
- 新图案必须大（通常比原纹身大 30-50%）
- 新图案必须深（黑/深蓝/深紫是常规遮盖色）
- 新图案必须密集（需要复杂纹理来分散视线对旧纹身的注意）
- 无法用浅色遮盖深色（白色不能遮盖黑色）

### 3.2 遮盖策略分级

| 难易度 | 原纹身特征 | 遮盖策略 |
|--------|-----------|----------|
| ★ 容易 | 浅色/小面积/线条细/已褪色 | 几乎任何深色图案都可以 |
| ★★ 中等 | 中等大小/中等颜色/有明显褪色 | 深色花卉/动物/几何可遮盖 |
| ★★★ 困难 | 大面积/深色/线条粗/黑色为主 | 需更大型深色图案，通常需要 2-3 次（先做淡化和/或激光淡化后再遮盖） |
| ★★★★ 极其困难 | 纯黑大面积（Blackout）/色彩极深 | 通常需先激光淡化 3-6 次，再考虑遮盖 |
| ★★★★★ 不可遮盖 | 极端情况 | 建议激光去除 |

### 3.3 最佳遮盖风格

| 风格 | 遮盖能力 | 原因 |
|------|---------|------|
| Blackwork / 部落 | ★★★★★ | 纯黑大色块，最强遮盖力 |
| 日式传统 | ★★★★ | 背景（波浪/云）可有效遮盖，主图案分散注意力 |
| 写实（暗色调） | ★★★★ | 复杂光影纹理可"淹没"旧纹身线条 |
| 新传统（深色） | ★★★ | 复杂色彩可混淆视线 |
| 几何 Dark | ★★★ | 密集图案可分散注意 |
| 水彩 | ★ | 透明感无法遮盖，反而会让旧纹身显露 |
| 极简 / Fine line | ★ | 无法遮盖任何明显旧纹身 |

### 3.4 遮盖前必须告知用户

- 遮盖纹身必然比原纹身大（通常大一圈）
- 遮盖后可能仍能看到原纹身的某些痕迹（尤其在特定角度/光线下）
- 激光淡化 + 遮盖是最佳组合方案（成本更高但效果更好）
- 不是所有纹身都能遮盖——诚实评估并告知

## 四、修改/修复纹身（Touch-Up & Rework）

### 4.1 Touch-Up（补色）

**适用场景**: 纹身局部褪色、颜色不均、线条断开

**最佳时机**: 
- 新纹身：愈合后 3-6 个月（完全稳定后）
- 旧纹身：任何时间

**注意事项**:
- 多数纹身师对自家作品提供首次免费 touch-up（6 个月内）
- 过度补色会积累 scar tissue（疤痕组织），影响后续修改
- 一般同一位置 touch-up 不超过 3 次

### 4.2 Rework（重做/升级）

**适用场景**: 旧纹身过时、技术差、想升级为更好质量

**策略**:
1. **添加背景**: 在原有纹身周围添加日式波浪/云朵/几何背景，提升整体感
2. **增加细节**: 在原有基础上加深阴影、丰富层次
3. **扩展延伸**: 把孤立的小图案扩展为 sleeve/大图的一部分
4. **风格转换**: 在原有基础上叠加新风格元素（e.g. 把老学校 roses 升级为写实花卉）

### 4.3 激光去除

**作为遮盖的辅助手段**:
- 推荐先激光淡化 3-6 次再遮盖（而非直接遮盖）
- 激光后需等 6-8 周皮肤恢复再纹身
- 黑色最容易淡化，浅色/彩色难淡化

**全去除**:
- 通常需要 6-12 次，间隔 6-8 周
- 完全去除率约 70-90%（因色料、深浅、肤色而异）
- 费用：是纹身价格的 5-10 倍
- 过程疼痛感 > 纹身

## 五、纹身感染应急处理

### 识别感染症状

如果出现以下任一症状，立即就医（皮肤科）：
- 红肿范围不断扩大，超出纹身边缘
- 脓液分泌物（黄色/绿色）
- 恶臭
- 纹身区域异常发热
- 发烧 38°C 以上
- 淋巴结肿大（腋下/腹股沟）

### 应急处理（就医前）

1. 停止使用任何护理产品
2. 用生理盐水轻轻冲洗
3. 不要自行涂抹抗生素软膏（可能引起过敏或耐药）
4. 拍下感染区域照片，记录变化
5. 立即就医

## 六、过敏反应

### 常见过敏色料

- 红色颜料（最常见过敏源，含汞/镉成分）
- 黄色颜料（含镉）
- 某些廉价品牌的彩色色料

### 过敏症状

- 纹身区域隆起、瘙痒
- 仅特定颜色区域出现反应（而非整个纹身）
- 可能出现在纹身数月甚至数年后

### 处理

- 轻微：外用皮质类固醇药膏
- 严重：就医皮肤科，可能需要口服抗组胺药或手术移除
