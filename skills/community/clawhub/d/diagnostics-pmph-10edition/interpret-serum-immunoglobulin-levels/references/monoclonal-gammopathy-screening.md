# 单克隆丙种球蛋白病筛查要点

当某一类免疫球蛋白（尤其是IgG、IgA或IgM）显著升高，而其他类型正常或降低时，应高度怀疑单克隆丙种球蛋白病（如多发性骨髓瘤、华氏巨球蛋白血症）。

## 初步筛查建议
1. **血清蛋白电泳（SPEP）**：观察是否存在M峰（单克隆条带）。
2. **免疫固定电泳（IFE）**：确定M蛋白的轻链和重链类型（如IgG-κ）。
3. **游离轻链检测**：评估κ/λ比值是否异常。
4. **影像学检查**：如全身低剂量CT或MRI，排查溶骨性病变。
5. **骨髓穿刺**：若怀疑骨髓瘤，需评估浆细胞比例及形态。

## 常见单克隆病类型
- **IgG型多发性骨髓瘤**：最常见，常伴贫血、肾功能损害。
- **IgA型多发性骨髓瘤**：易出现高黏滞血症。
- **IgM型巨球蛋白血症（Waldenström）**：表现为高IgM、贫血、出血倾向、视力障碍。
- **轻链型骨髓瘤**：血清总Ig可能不高，但尿本周蛋白阳性。

> 注意：良性单克隆丙种球蛋白病（MGUS）也可能导致轻度Ig升高，需长期随访。
