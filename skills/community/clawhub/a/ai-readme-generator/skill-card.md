## Description: <br>
Generate AI-readable project documentation for OpenClaw/Cursor-style coding workflows. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[zjuncher](https://clawhub.ai/user/zjuncher) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and coding agents use this skill to scan an existing codebase and generate maintainable AI-facing project documentation, including AGENTS.md and Cursor rule files. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated project documentation may contain inaccurate conclusions or accidentally expose sensitive implementation details. <br>
Mitigation: Review generated files for accuracy and sensitive content before committing them or allowing future AI tools to rely on them. <br>
