## 二、档口拿货SOP

### 2.1 询价比价

```
比价原则：
1. 同款至少问3家
2. 记录价格区间
3. 问清是否含税含运

询价话术：
- "老板，这个多少起批？"
- "量大有优惠吗？"
- "能开发票吗？"
- "现货还是订做？"
```

### 2.2 拿样试单

```
拿样流程：
1. 挑选样品（1-5件）
2. 支付样品费（通常可抵后续货款）
3. 检查样品质量
4. 确认大货品质标准

样品检查要点：
- 外观：有无瑕疵、色差
- 材质：是否与描述一致
- 功能：是否正常使用
- 包装：是否完整
```

### 2.3 议价锁价

```
议价技巧：
- 阶梯报价：量大返点
- 付款方式：现款优惠
- 长期合作：年度返点

锁价策略：
- 签订供货协议
- 锁定价格周期（1-3个月）
- 约定调价机制
```

### 2.4 质检收货

```
现场质检：
- 抽检比例：5-10%
- 检查外观、数量、规格
- 发现问题当场退换

出货抽检：
- 发货前二次检查
- 不良率控制：<3%
- 问题产品隔离处理
```

## 四、混拼柜策略

### 4.1 拼柜原则

```
体积控制：
- 20尺柜：28-30立方米
- 40尺柜：56-58立方米
- 40尺高柜：65-68立方米

重量控制：
- 20尺柜：18-20吨
- 40尺柜：22-25吨
- 注意：海运按体积重/实重取大计费
```

### 4.2 混搭组合

```
场景：发往欧洲，目标货值$15000

组合方案：
- 饰品：$3000（体积小、价值高）
- 文具：$2000（体积中、价值中）
- 玩具：$4000（体积中、价值中）
- 箱包：$3000（体积大、价值中）
- 家居：$3000（体积大、价值中低）

总体积：约12立方米
运费：$280/立方米 × 12 = $3360
运费占比：22.4%（可接受）

优化：
- 减少箱包、家居比例
- 增加饰品、文具比例
- 目标运费占比：<15%
```

### 4.3 拼柜服务商

```
选择标准：
1. 有市场采购贸易操作经验
2. 能提供拼柜方案优化
3. 有海外代理网络
4. 价格透明、无隐形收费

义乌主流拼柜服务商：
- 可通过Chinagoods平台推荐
- 可咨询档口老板推荐
- 建议实地考察仓库
```

## 五、供应链管理

### 5.1 供应商分类管理

| 类型 | 特点 | 合作策略 |
|------|------|----------|
| 工厂直营档口 | 价格最低、可控性高 | 优先合作、深度绑定 |
| 批发商档口 | 货源稳定、品类多 | 备选合作、比价参考 |
| 贸易公司 | 服务好、加价率高 | 特殊品类补充 |

### 5.2 供应商评估表

```
评估维度：
1. 价格竞争力（30分）
   - 同类产品价格水平
   - 阶梯报价合理性

2. 质量稳定性（30分）
   - 不良率历史
   - 质量波动情况

3. 供货能力（20分）
   - 现货库存量
   - 补货周期

4. 服务配合度（20分）
   - 响应速度
   - 售后态度

评分≥80分：优质供应商
评分60-80分：合格供应商
评分<60分：淘汰或观察
```

### 5.3 断货应急预案

```
断货原因：
- 供应商缺货
- 生产周期延误
- 质量问题返工

应对方案：
1. 备选供应商（至少2家同类）
2. 提前2周预警库存
3. 与客户沟通调整交期
4. 寻找替代产品
```

## 七、成本核算模板

### 7.1 单品成本核算

```
产品：某款耳环
采购价：¥3.5/件
起订量：12件
汇率：1 USD = 7.2 CNY

成本构成：
- 采购成本：¥3.5 = $0.49
- 包装成本：¥0.3 = $0.04
- 国内运费：¥0.2 = $0.03
- 国际运费：¥0.5 = $0.07（按体积分摊）
- 报关费用：¥0.1 = $0.01

总成本：$0.64
建议售价：$1.5-2.0
毛利率：57-68%
```

### 7.2 订单成本核算

```
订单：发往美国，总货值$5000

成本构成：
- 货物成本：$3500
- 国际运费：$800（海运拼箱）
- 报关费用：$100
- 保险费用：$30
- 其他费用：$50

总成本：$4480
毛利润：$520
毛利率：10.4%

优化空间：
- 提高货值密度
- 寻找更优运费
- 增加货量谈判
```
