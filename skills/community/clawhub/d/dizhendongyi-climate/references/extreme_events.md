# 极端事件预警体系

## 一、三阶预警框架

### 核心逻辑：触发因素 → 传播路径 → 系统响应

以地质记录中的突变事件为"自然实验"，通过解析生物地球化学循环波动、海气/陆气耦合过程，构建三级预警体系。

## 二、预警等级

| 等级 | 颜色 | 触发条件 | 响应时长 |
|------|------|---------|---------|
| 正常 | 🟢 Green | 指标在历史范围内 | 无 |
| 关注 | 🟡 Yellow | 偏离正常值 5-10% | 100-500年 |
| 警戒 | 🟠 Orange | 偏离正常值 10-25% | 50-150年 |
| 紧急 | 🔴 Red | 偏离正常值 > 25% | < 50年 |

## 三、核心监测指标

### 1. AMOC（大西洋经向翻转环流）
- 触发阈值：强度偏差 > 20%
- 前兆：北大西洋淡水通量上升 + AMOC强度下降
- 历史案例：8,200年前冷事件（降温2-4°C，持续150-200年）

### 2. 冰盖稳定性
- 触发阈值：格陵兰冰盖物质平衡亏损 > 15%
- 前兆：北大西洋IRD（冰筏碎屑）通量上升
- 历史案例：Heinrich事件（末次冰期内6次，降温2-5°C）

### 3. 辐射强迫偏移
- 触发阈值：65°N夏季辐射变化 > 8 W/m²
- 前兆：植被覆盖度变化 + 粉尘输送减少
- 历史案例：Green Sahara结束（降水从700mm降至550mm）

### 4. 碳循环稳定性
- 触发阈值：深海δ¹³C异常 > 0.3‰
- 前兆：表层海水pH下降 + 碳酸盐饱和度降低
- 历史案例：PETM事件（CO₂从890升至1980ppm）

## 四、预警评估流程

```
1. 数据采集：AMOC强度、土壤湿度、SST梯度等监测指标
2. 基准对比：与历史案例库中的相似度匹配
3. 前兆指标评估：对照预警指标体系
4. 等级判定：正常/关注/警戒/紧急
5. 建议输出：持续监测频率与预期响应时长
```
