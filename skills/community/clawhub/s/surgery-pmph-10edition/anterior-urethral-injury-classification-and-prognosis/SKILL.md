# 前尿道损伤的病理分型与愈后评估

## 适用对象
- 男性前尿道
- 尿道球部
- 尿道阴茎部

## 使用前提
- 患者存在外伤史（如骑跨伤）或医源性操作史（如反复插导尿管）

## 禁忌场景
- 后尿道损伤
- 非创伤性尿道病变

## 执行步骤

1. **判断损伤类型**：
   - **挫伤**：仅局部水肿和出血，尿道壁结构完整；通常不遗留尿道狭窄。
   - **裂伤**：部分尿道壁撕裂但未完全离断；愈合易形成瘢痕，导致「瘢痕性尿道狭窄」。
   - **断裂**：尿道完全离断，断端退缩分离；血液及尿液可渗入周围组织。

2. **评估尿液外渗范围**：
   - **尿道球部损伤**：
     - 血液及尿液渗入会阴浅筋膜包绕的会阴浅袋，引起会阴、阴囊、阴茎肿胀，有时向上扩展至腹壁。
     - 因解剖限制，尿液**不会外渗到两侧股部**。
   - **尿道阴茎部损伤**：
     - 若阴茎筋膜完整，外渗局限于筋膜内，仅表现为阴茎肿胀。
     - 若阴茎筋膜破裂，尿液外渗范围扩大，表现与尿道球部损伤相同。

3. **输出预后判断**：
   - 明确是否可能发生尿道狭窄。
   - 描述尿液外渗的解剖范围。
   - 为后续治疗（如导尿、尿道成形术等）提供依据。
