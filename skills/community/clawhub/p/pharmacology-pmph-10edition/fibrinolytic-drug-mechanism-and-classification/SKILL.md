# 纤维蛋白溶解药的作用机制与分类

## 何时使用
- 患者确诊为急性血栓栓塞性疾病（如急性心肌梗死、大面积肺栓塞、急性缺血性脑卒中）；
- 发病时间在6小时内（部分情况可延长至12小时，需结合指南）；
- 无活动性出血或其他禁忌证；
- 需快速恢复血流且无法立即进行介入治疗。

## 如何执行

### 1. 理解纤溶过程三阶段
- **激活阶段**：前激活剂（如t-PA、尿激酶、链激酶）转化为纤溶酶原激活物；
- **转化阶段**：纤溶酶原在精氨酸-缬氨酸肽键处断裂，生成纤溶酶；
- **降解阶段**：纤溶酶降解纤维蛋白（及纤维蛋白原）为可溶性纤维蛋白降解产物（FDP），实现血栓溶解。

### 2. 根据药物代际选择合适溶栓药
- **第一代（非选择性）**：
  - 链激酶（SK）：间接激活纤溶酶原，无纤维蛋白特异性，全身纤溶风险高；
  - 尿激酶：直接激活，无抗原性，但半衰期短（≈16分钟），需大剂量给药。
- **第二代（较具选择性）**：
  - 阿替普酶（rt-PA）：优先结合血栓表面纤维蛋白，局部激活纤溶酶原，出血风险较低；
  - 阿尼普酶（APSAC）：复合物形式，在血栓表面缓慢释放活性，减少全身效应。
- **第三代（改良型）**：
  - 瑞替普酶（r-PA）：基因工程改造，半衰期延长，给药简便（固定剂量，无需按体重调整）。

### 3. 应用临床原则
- **给药时机**：越早越好，理想窗口为血栓形成后6小时内；
- **主要适应证**：急性心肌梗死、高危肺栓塞、符合条件的急性缺血性卒中；
- **监测与应对**：密切观察出血征象；若发生严重出血，立即停药并使用抗纤溶药（如氨甲环酸）拮抗。

### 4. 排查禁忌证
禁用于以下情况：
- 活动性内出血或出血性疾病；
- 近期重大创伤、手术或消化道溃疡；
- 未控制的严重高血压（如收缩压 >180 mmHg）；
- 对所选溶栓药过敏；
- 颅内肿瘤、动脉瘤或近期脑卒中史。

> 提示：具体用药方案应结合最新临床指南（如ACC/AHA、ESC）及患者个体情况综合判断。
