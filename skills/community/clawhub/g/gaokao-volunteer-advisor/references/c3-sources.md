# 权威数据源清单

## 第一梯队：官方来源（最权威）

| 数据类型 | 来源 | 网址 |
|---|---|---|
| 各省招生计划 | 省教育考试院 | 各省不同，搜索"XX省教育考试院" |
| 一分一段表 | 省教育考试院 | 高考出分后发布 |
| 批次线 | 省教育考试院 | 高考后发布 |
| 院校招生章程 | 院校本科招生网 | 各校官网 |
| 院校/专业库 | 阳光高考平台 | gaokao.chsi.com.cn |
| 选科要求 | 省教育考试院 | 各省发布的选科指引 |

## 第二梯队：院校官方

| 数据类型 | 来源 |
|---|---|
| 就业质量报告 | 学校就业信息网（每年发布） |
| 保研/推免公示 | 学校教务处或研究生院官网 |
| 转专业政策 | 学校教务处 |
| 学费标准 | 学校财务处或招生章程 |
| 宿舍条件 | 学校后勤处（部分有官网公示） |

## 第三梯队：可信参考

| 数据类型 | 来源 | 说明 |
|---|---|---|
| 在校生真实反馈 | 知乎、贴吧、小红书 | 需交叉验证，区分情绪和事实 |
| 安全事件 | 裁判文书网、媒体报道 | 已证实的信息 |
| 行业就业数据 | 招聘平台（BOSS直聘、猎聘） | 薪资参考，非精确数据 |
| 学科评估 | 教育部学科评估结果 | 反映学科实力 |

## 信源分级规则

- **已证实**：裁判文书、警方通报、官方公告、学校官方文件
- **高可信**：官方媒体报道、学校就业质量报告、教育部数据
- **中可信**：知乎/贴吧实名认证用户（如在校教师、辅导员）的分享
- **待核**：匿名爆料、情绪化吐槽、未经证实的传言

所有"待核"信息在报告中必须标注"待核"，不能当作事实使用。
