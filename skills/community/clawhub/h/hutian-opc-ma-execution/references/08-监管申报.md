# 并购监管申报指引

> ⚠️ **法律声明**：本指引为通用框架，实际申报要求需根据具体交易情况和最新监管规定确定，建议在专业顾问指导下完成。

---

## 监管申报类型概览

### 主要申报类型

| 申报类型 | 适用情形 | 主管部门 | 典型时限 |
|---------|---------|---------|---------|
| 上市公司重大资产重组 | 上市公司收购 | 证监会 | 45工作日 |
| 要约收购豁免 | 触发要约收购 | 证监会 | 20工作日 |
| 经营者集中审查 | 营业额达标 | 反垄断局 | 30日/90日 |

| 外国投资者并购 | 外商投资企业 | 商务部门 | 30日 |

## 上市公司重大资产重组

### 适用标准

```
【重大资产重组定义】

上市公司及其控股或者控制的公司购买、出售资产，达到下列标准之一的，构成重大资产重组：

（一）购买、出售的资产总额占上市公司最近一个会计年度经审计的合并财务会计报告期末资产总额的比例达到50%以上；

（二）购买、出售的资产在最近一个会计年度所产生的营业收入占上市公司同期经审计的合并财务会计报告营业收入的比例达到50%以上；

（三）购买、出售的资产净额占上市公司最近一个会计年度经审计的合并财务会计报告期末净资产额的比例达到50%以上，且超过5000万元人民币。

【计算规则】
购买资产为股权的，其资产总额以被投资企业的资产总额与该持股比例的乘积为准；
营业收入以被投资企业的营业收入与该项投资所占股权比例的乘积为准。
```

### 申报流程

```
【第一阶段：停牌与内幕信息管理】

1. 筹划阶段
   - 避免股票异常波动
   - 严格保密
   - 登记内幕信息知情人

2. 停牌申请（如需）
   - 公告停牌原因
   - 承诺停牌时间

【第二阶段：方案设计与内部审批】

1. 聘请中介机构
   - 独立财务顾问
   - 律师事务所
   - 会计师事务所
   - 资产评估机构

2. 尽职调查
   - 财务尽调
   - 法律尽调
   - 业务尽调

3. 方案设计
   - 交易结构
   - 定价机制
   - 业绩承诺

4. 内部审批
   - 董事会决议
   - 股东大会决议

【第三阶段：申报证监会】

1. 提交材料
   - 重组报告书
   - 财务顾问意见
   - 法律意见书
   - 审计报告
   - 评估报告

2. 证监会审核
   - 受理
   - 初审
   - 反馈
   - 落实反馈
   - 并购重组委审核
   - 核准/不予核准

【第四阶段：实施与持续披露】

1. 实施阶段
   - 资产过户
   - 发行股份
   - 变更登记

2. 持续披露
   - 持续督导
   - 定期报告披露
```

### 审核关注要点

```
【审核关注重点】

1. 交易必要性
   - 是否符合国家产业政策
   - 是否有利于提高上市公司质量
   - 协同效应分析

2. 交易合规性
   - 是否涉及借壳上市
   - 是否涉及发行股份购买资产
   - 业绩承诺合规性

3. 估值定价
   - 评估方法合理性
   - 评估假设合理性
   - 溢价合理性

4. 盈利能力
   - 历史业绩真实性
   - 盈利预测可靠性
   - 业绩承诺可实现性

5. 标的资产质量
   - 资产权属清晰性
   - 业务独立性
   - 可持续经营能力

6. 信息披露
   - 披露完整性
   - 披露及时性
   - 风险揭示充分性
```

## 经营者集中申报

### 申报标准

```
【申报触发条件】

经营者集中是指下列情形：
（一）经营者合并；
（二）经营者通过取得股权或者资产的方式取得对其他经营者的控制权；
（三）经营者通过合同等方式取得对其他经营者的控制权或者能够对其他经营者施加决定性影响。

【营业额门槛】

参与集中的经营者上一会计年度在全球范围内的营业额合计超过100亿元人民币，
并且其中至少两个经营者上一会计年度在中国境内的营业额均超过4亿元人民币；

或者参与集中的经营者上一会计年度在中国境内的营业额合计超过20亿元人民币，
并且其中至少两个经营者上一会计年度在中国境内的营业额均超过4亿元。

【例外情形】

符合以下情形之一的，经营者可以不申报：
（一）参与集中的经营者为关联企业
（二）特定金融机构资产并购
（三）股权收购不涉及控制权变更
（四）集中水平微小
```

### 申报流程

```
【第一阶段：判断是否需要申报】

1. 计算各方营业额
   - 全球营业额
   - 中国境内营业额

2. 判断控制权变更
   - 股权比例变化
   - 董事会构成变化
   - 实际控制人变更

3. 确定申报义务人
   - 合并：双方
   - 收购：买方
   - 新设合营：各方

【第二阶段：准备申报材料】

1. 基本材料
   - 申报书（标准格式）
   - 交易合同、协议
   - 交易各方营业执照
   - 财务审计报告

2. 附加材料
   - 市场界定说明
   - 竞争影响分析
   - 集中对消费者的影响
   - 集中对行业发展的影响

【第三阶段：提交申报】

1. 向反垄断执法机构提交
   - 国家市场监督管理总局
   - 省级市场监管部门（初步审查转报）

2. 申报时间
   - 实施集中前申报
   - 不得在申报前实施集中

【第四阶段：审查程序】

1. 初步审查（30日）
   - 立案
   - 初步审查
   - 决定：允许/进一步审查

2. 进一步审查（90日/180日）
   - 深入调查
   - 决定：允许/禁止/附加条件

3. 审查决定
   - 无条件批准
   - 附条件批准（行为/结构性）
   - 禁止集中
```

### 申报注意事项

```
【常见问题】

1. 营业额计算
   - 关联公司营业额需合并计算
   - 境外营业额需计入
   - 避免遗漏关联方

2. 申报时间
   - 必须在实施前申报
   - 申报等待期不得实施
   - 违规实施将面临处罚

3. 材料准备
   - 市场界定需充分论证
   - 竞争分析需客观
   - 避免夸大集中益处

【OPC建议】
- 提前评估申报必要性
- 预留充足审查时间
- 准备多种方案（附条件/禁止情形）
```

## 外商投资安全审查

### 适用情形

```
【审查范围】

外国投资者并购在中国境内进行的下列交易，需进行安全审查：

1. 军工及军工配套企业
2. 重点、敏感军事设施周边企业
3. 国防科技工业领域
4. 关系国家安全的重要农产品、重要能源和资源、重要信息技术产品和服务
5. 关键技术、重要互联网产品和服务
6. 重要金融服务
7. 其他重要领域

【触发条件】

- 外国投资者取得实际控制权
- 外国投资者持股比例超过一定门槛（如特定行业）
- 涉及关键领域
```

### 申报流程

```
【安全审查程序】

1. 申报触发
   - 交易各方自行判断
   - 或联席会议办公室告知

2. 提交材料
   - 安全审查申报表
   - 交易基本情况
   - 投资方背景
   - 对国家安全的影响分析

3. 审查流程
   - 联席会议办公室审查
   - 一般审查（30个工作日）
   - 特别审查（45个工作日，可延长）

4. 审查决定
   - 通过
   - 附条件通过
   - 禁止
```

## 各类申报对比

| 项目 | 重大资产重组 | 经营者集中 | 外商投资安全审查 |
|------|-------------|-----------|-----------------|
| 主管部门 | 证监会 | 反垄断局 | 发改委/商务部 |
| 申报主体 | 上市公司 | 交易各方 | 外国投资者 |
| 申报时间 | 实施前 | 实施前 | 实施前 |
| 审查时限 | 45工作日 | 30-180日 | 30-45工作日 |
| 主要关注 | 上市公司质量 | 市场竞争 | 国家安全 |
| 违规后果 | 行政处罚/无效 | 罚款/禁止实施 | 禁止/附条件 |

## OPC特别注意事项

```
【技术方申报注意要点】

1. 知识产权涉密审查
   - 技术是否涉及国家秘密
   - 是否需要通过军品审查
   - 出口管制合规

2. 外国投资者审查
   - 买方是否为外国投资者
   - 是否触发安全审查
   - 是否需要科技部审查

3. 行业特殊审批
   - 金融行业：银保监/证监会
   - 医药行业：药监局
   - 互联网行业：网信办
   - 教育行业：教育部

4. 审批时间预留
   - 上市公司重组：3-6个月
   - 反垄断审查：1-6个月
   - 安全审查：1-3个月

5. 并行申报策略
   - 反垄断与安全审查可并行
   - 上市公司重组需单独审批
   - 提前与中介机构沟通
```
