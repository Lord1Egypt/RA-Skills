## ⚙️ 高级配置（可选）

### ☁️ 云端备份到 IMA

**适用场景**：多设备同步、永久保存重要知识

**配置步骤**：

1. 获取 IMA 凭证
   - 打开 https://ima.qq.com 并登录
   - 进入个人中心获取 `client_id` 和 `api_key`

2. 配置凭证
   ```bash
   # 复制以下命令并替换你的凭证
   echo "你的client_id" > ~/.config/ima/client_id
   echo "你的api_key" > ~/.config/ima/api_key
   ```

3. 完成！AI 会自动同步记忆到 IMA

## ❓ 常见问题

**Q: 记忆保存在哪里？**

**Q: 隐私安全吗？**

## 🔄 从旧版迁移

如果你之前使用过旧版 yaoyao-memory，需要运行迁移脚本：

```bash
cd ~/.openclaw/workspace/skills/yaoyao-memory-v2
python3 migrate.py --dry      # 先预览
python3 migrate.py --force   # 执行迁移
```

**迁移特性**：
- 自动备份所有数据
- 每步可回滚
- 中断后可继续

**迁移后检查**：
```bash
python3 install_modules.py --verify
```
