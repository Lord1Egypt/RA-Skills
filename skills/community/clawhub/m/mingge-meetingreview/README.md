# Mingge_MeetingReview

> 内训课内容加工工作流 v3.3

## 简介

一个用于飞书内训课内容自动加工的 Agent Skill。从一篇飞书课程文档（包含参与用户表 + 评论区 + 逐字稿）出发，自动生成 4 个不同用途的版本：

| 版本 | 用途 |
|------|------|
| **蒸馏版** | 去水词、保留全部有效内容 |
| **总结版** | 核心观点 + 案例 + 高频问题 |
| **复盘版** | 六个办公室主任维度深度分析 |
| **3 点改进稿** | 针对 3 个环节的重写逐字稿 |

## 核心亮点

- 🧠 **六个办公室主任分析框架**：用 C 总、杏总、鸡哥、多多哥、海马体、导爷 6 个拟人化角色评估课程效果
- 📊 **七阶情绪升温模型**：从"听得懂"到"决策"的完整情绪路径
- ⚡ **一键生成**：提供飞书文档链接即可自动加工
- 📁 **自动归档**：生成的文档自动创建为飞书知识库子节点
