## Description: <br>
Generates Taiyi Shenshu charts with accumulated-year, Yin/Yang Dun, sixteen-deity placement, host/guest calculations, and interpretive readings for macro-style divination questions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External users can use this skill to ask an agent for Taiyi-style cultural divination charts and textual readings by date or question type. Outputs should be treated as entertainment or cultural interpretation, not as real-world forecasts for high-stakes decisions. <br>

## Known Risks and Mitigations: <br>
Risk: The skill frames cultural divination output as disaster, financial, medical, political, military, or public-safety predictions. <br>
Mitigation: Use it only for cultural or entertainment-style interpretation, and do not rely on its predictions for real-world high-stakes decisions. <br>
Risk: The documentation references a Python helper command that was not present in the reviewed artifact. <br>
Mitigation: Verify the installed package contents before running documented commands, and avoid assuming missing helper scripts are available. <br>
Risk: The artifact uses simplified calculation logic, so generated readings may not match traditional Taiyi methods or the documented examples. <br>
Mitigation: Treat readings as approximate generated guidance and review any chart logic before using it for study or publication. <br>
