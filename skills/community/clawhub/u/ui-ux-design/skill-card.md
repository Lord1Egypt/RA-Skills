## Description: <br>
Provides practical UI/UX guidance for web and mobile application design, including responsive layouts, color systems, typography, accessibility, Tailwind CSS, Shadcn/ui patterns, and micro-interactions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[itsjustdri](https://clawhub.ai/user/itsjustdri) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers, designers, and product builders use this skill to plan and review modern UI implementation choices before or during front-end development. It is especially useful for mobile-first layouts, component patterns, accessibility checks, and Tailwind CSS or Shadcn/ui projects. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Optional npm/npx setup commands may modify the current project or pull changing package versions. <br>
Mitigation: Run setup commands only in the intended project directory, review generated files, and pin package versions when reproducible supply-chain behavior matters. <br>
Risk: Design guidance may be applied without project-specific accessibility validation. <br>
Mitigation: Use the included WCAG 2.2 checks and contrast-testing references during implementation review. <br>
