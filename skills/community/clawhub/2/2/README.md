# Number Two (2) Skill

> A comprehensive prompt-based skill about the number **2** — the smallest prime, the only even prime, and the foundation of the binary world.

## Overview

This skill provides rich, structured knowledge about the number 2, covering its mathematical properties, history & etymology, the binary system, scientific applications, philosophical symbolism, music theory, language & idioms, sports, computing, famous pairs, and fun facts. It is designed as a **natural-language, prompt-based skill** that activates whenever a user mentions or implies topics related to the number 2.

## Activation

The skill activates when the user:

- Directly asks about the number 2 or the concept of "two"
- Discusses prime numbers (2 as the only even prime)
- Asks about the binary numeral system (base-2)
- Inquires about duality, pairs, or balance in philosophy
- Mentions 2 in scientific contexts (Helium, DNA double helix, Newton's 2nd law, etc.)
- Refers to 2 in computing (binary, boolean, two's complement)
- Discusses famous duos, pairs, or twin concepts
- Asks about idioms or expressions involving the number 2

## Quick Start

This is a **prompt-based skill**. Simply use natural language to interact:

### 🟢 Example Prompts

**Mathematics:**
```
Why is 2 the only even prime number?
```

**Binary System:**
```
How does the binary (base-2) system work?
```

**Science:**
```
What is the element with atomic number 2?
```

**Philosophy:**
```
What does duality mean across different cultures?
```

**Computing:**
```
Explain two's complement in binary arithmetic.
```

**Culture:**
```
What are some famous duos in history?
```

**Language:**
```
What English idioms involve the number 2?
```

**Fun Facts:**
```
Tell me some fun facts about the number 2.
```

**Comprehensive:**
```
Give me a full overview of the number 2.
```
