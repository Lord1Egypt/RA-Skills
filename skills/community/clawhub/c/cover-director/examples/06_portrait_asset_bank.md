# 示例 06：同一人物封面素材库

## 输入

- 用户上传一张人像
- 目标：生成 8–12 张同一人物不同表情手势素材
- 平台：抖音 / 小红书 / 视频号
- 比例：9:16

## 第一轮策略

不要一次生成 12 张。先生成 4 张，筛最像的 1–2 张，再继续扩展。

## 第一轮 4 张

1. 自信微笑 + 指向侧边标题区。
2. 严肃提醒 + 单手停止。
3. 疑惑思考 + 托下巴。
4. 惊喜兴奋 + 双手轻微张开。

## 统一身份锁定提示词

```text
基于用户上传的参考人像图，生成同一个人物的短视频封面人像素材。必须高度保留脸型、五官比例、眼睛形状、鼻子形状、嘴型、眉毛、发型、发色、肤色、年龄感、面部轮廓、气质和神态特征。可以改变表情、手势、头部角度和身体姿态，但不能改变人物身份。画面为 9:16 竖版，半身近景，人物占画面主体，干净纯色或渐变背景，明亮棚拍光线，真实商业摄影质感，高清锐利，预留标题空间。不要文字、水印、logo、多人、换脸、年龄变化、畸形手、多余手指、遮挡脸部、模糊、过度磨皮、卡通化。
```

## 筛选标准

- 第一优先：像同一个人。
- 第二优先：脸清晰、五官稳定。
- 第三优先：动作适合封面。
- 第四优先：手部自然。
