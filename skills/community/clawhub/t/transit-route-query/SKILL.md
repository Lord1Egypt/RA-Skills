# 公交地铁线路查询 — 市内公交地铁路线规划

> ⚡ **地铁优先 · 自动换乘规划 · 三种策略 · 零配置即装即用**

---

## 快速入门

**3个开场白示例，复制即用：**
1. "从三里屯怎么去天安门"
2. "南京路到浦东机场坐地铁怎么走"
3. "广州塔到白云机场最少换乘的路线"

---

## 核心能力

1. **路线规划**：输入任意两个地点，自动规划公交地铁路线
2. **地铁优先**：含地铁的方案优先展示，公交方案次之
3. **换乘详情**：每段线路的上下车站、站数、步行距离一目了然
4. **三种策略**：最快路线 / 最少换乘 / 最少步行
5. **最近站点提示**：首方案自动提示最近的地铁站

## 能做什么

- 查询市内公交地铁路线
- 规划换乘方案（地铁+公交组合）
- 按时间/换乘/步行偏好选路线

## 不能做什么

- 不支持打车/驾车路线（仅公交+地铁）
- 不支持跨城市路线查询
- 不提供实时公交到站信息（建议用高德/百度地图APP查看）

## 使用示例

1. "从三里屯怎么去天安门"
2. "南京路到浦东机场坐地铁怎么走"
3. "深圳北站到世界之窗怎么坐车"
4. "广州塔到白云机场最少换乘"

## 注意事项

- 路线方案来自地图平台，实际运营时间以当地公交地铁公告为准
- 费用为预估，实际以乘车时为准
- 地铁末班车时间请以当地地铁官方公告为准

## 数据流向

用户输入（起点/终点/城市等查询参数）→ 本技能脚本 → 代理服务 → 地图平台API → 返回结果给用户。查询参数会发送到代理服务以获取实时路线数据，代理服务不存储用户数据。
