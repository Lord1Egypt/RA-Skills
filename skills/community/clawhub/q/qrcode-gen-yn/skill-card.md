## Description: <br>
Generate QR codes from URLs or text using a pre-built Python script with the qrcode library. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[whubbt](https://clawhub.ai/user/whubbt) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and other agent users can use this skill to turn text or URLs into local PNG QR-code images without writing new QR-code generation code. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill saves a PNG to a user-supplied path, which could overwrite an existing file or create directories in an unintended location. <br>
Mitigation: Use normal output locations such as Desktop or Downloads, choose a new filename, and avoid paths that could overwrite important files. <br>
