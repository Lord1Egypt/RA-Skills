# 运动神经末梢分类与结构分析

## 适用对象
- 骨骼肌
- 心肌
- 平滑肌
- 腺体

## 使用前提
- 存在运动神经元轴突终末
- 明确效应器细胞类型

## 执行步骤

1. **识别效应器类型**：
   - **骨骼肌** → 躯体运动神经末梢
   - **心肌、平滑肌或腺体** → 内脏运动神经末梢

2. **分析结构特征**：
   - **躯体运动末梢**：
     - 来源于脊髓前角或脑干运动神经元
     - 轴突脱髓鞘后反复分支，末端呈葡萄状
     - 形成典型突触结构——运动终板（神经肌连接）
     - 突触后膜有皱褶，突触小体富含乙酰胆碱囊泡
   - **内脏运动末梢**：
     - 纤维较细，无髓鞘
     - 末梢呈串珠样膨体（varicosity）
     - 贴附于肌细胞表面或穿行于腺细胞间
     - 无典型突触结构

3. **明确作用机制**：
   - **躯体运动**：释放乙酰胆碱 → 激活突触后膜受体 → 肌膜去极化 → 收缩
   - **内脏运动**：释放乙酰胆碱或去甲肾上腺素 → 弥散作用于效应细胞 → 调控收缩或分泌

4. **关联功能单位**（如适用）：
   - 对骨骼肌，说明运动单位概念（一个运动神经元+所支配肌纤维）

## 输出要求
区分两类运动末梢的结构、分布和作用机制，用于理解神经对效应器的调控方式。
