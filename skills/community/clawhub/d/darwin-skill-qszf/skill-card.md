## Description: <br>
darwin-skill 2.0 helps an agent evaluate, improve, test, and either retain or roll back changes to existing skill files using a nine-dimensional scoring workflow and independent validation. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>

## Use Case: <br>
Developers and skill maintainers use this skill to assess and improve existing agent skills while preserving only validated improvements. The workflow is intended for iterative skill optimization with human confirmation before finalizing changes. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Unreviewed edits to other skill files can introduce incorrect, misleading, or unwanted guidance. <br>
Mitigation: Use the skill only on skill directories that are acceptable to change, review proposed edits, and keep the human confirmation step enabled. <br>
Risk: Automated optimization can favor score improvements that do not match the maintainer's practical expectations. <br>
Mitigation: Use the independent validation and rollback workflow described by the skill, then accept only changes that pass human review. <br>
