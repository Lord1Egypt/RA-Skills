# Herpetic Keratitis Staged Treatment

## When to Use
- 单纯疱疹病毒性角膜炎
- 黑睛星翳患者

## When to Use
- 黑睛浅层多个细小星翳（针尖或秤星样灰白色混浊）
- 伴沙涩疼痛、羞明流泪
- 已结合现代眼科检查排除其他非星翳性角膜病变

## Execution Steps

1. **确认诊断**：核实是否存在黑睛星翳及眼部刺激症状，注意是否单眼或双眼发病、是否易复发。

2. **辨病因类型（四类）**：
   - **风热客目**：恶风发热、头痛鼻塞、舌红苔薄黄、脉浮数；
   - **肝胆火炽**：口苦咽干、胁痛、溺赤、舌红苔黄、脉弦数；
   - **湿热犯目**：头重胸闷、口黏纳呆、便溏、苔黄腻、脉濡数；
   - **阴虚夹风**：口干咽燥、舌红少津、脉细数。

3. **审病程阶段**：
   - **新起（急性期）**：以祛邪为主；
   - **日久或反复发作（慢性期）**：正虚邪恋，需扶正祛邪。

4. **确定治疗总则**：
   - 风热者：疏风散热；
   - 肝火者：清肝泻火；
   - 湿热者：清热除湿；
   - 阴虚者：滋阴祛风；
   - 所有证型均需配合“退翳明目”法。

5. **输出结果**：根据病因与病程阶段选择相应治法，并推荐对应方药（如银翘散、龙胆泻肝汤、三仁汤、加减地黄丸等）。
