## Description: <br>
Learn and look up how to use ant-design-vue v4 components for Vue 3, including variants, runnable SFC examples, Vue 3 idioms, and key props, events, and slots. <br>

This skill is ready for commercial/non-commercial use. <br>

[junwugit](https://clawhub.ai/user/junwugit) <br>

## Use Case: <br>
Developers and engineers use this skill to answer ant-design-vue component questions, generate minimal Vue 3 SFC examples, and port React antd patterns to ant-design-vue. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated npm commands or UI examples may be inappropriate for a production project if applied without review. <br>
Mitigation: Review commands and generated Vue code before running them or merging them into a real application. <br>
Risk: Sample upload, delete, modal, and form flows can affect user data or application behavior when adapted directly. <br>
Mitigation: Validate destructive or data-handling examples against project requirements, permissions, and test data before use. <br>
