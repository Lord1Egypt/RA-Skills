# D11 精神领域任务目录

## 领域定义

精神领域(D11)关注个体内在生命、价值观、意义感和超越性体验的探索与发展。核心驱动需求：N5自我实现。

## 领域边界

| 范围内 | 范围外 |
|--------|--------|
| 个人价值观探索 | 宗教机构管理 |
| 人生意义建构 | 神学理论研究 |
| 精神修行实践 | 宗教传播活动 |
| 存在困境应对 | 哲学学术研究 |
| 精神遗产规划 | 文化遗产保护 |
| 临终精神关怀 | 医疗临终护理 |
| 代际智慧传承 | 家族企业传承 |
