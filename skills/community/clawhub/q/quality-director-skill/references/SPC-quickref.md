| 子组大小 n | 推荐控制图 | 公式/用途 |
|-----------|-----------|---------|
| **n = 1**（单件流/连续测量）| X-mR（单值-移动极差图）| 移动极差MR = |Xi - Xi-1| |

## 控制限计算公式速查

### X-bar & R 图（n < 10）

```
X-bar图：
  UCL = X̄ + A₂ × R̄     （A₂: n=5→0.577, n=4→0.729, n=3→1.023）
  CL  = X̄
  LCL = X̄ - A₂ × R̄

R图：
  UCL = D₄ × R̄          （D₄: n=5→2.114, n=4→2.282, n=3→2.574, n=2→3.267）
  CL  = R̄
  LCL = D₃ × R̄          （D₃: n=5→0, n=4→0, n=3→0, n=2→0）
```

### X-mR 图（n = 1）

```
单值图：
  UCL = X̄ + 3 × σ̂        （σ̂ = R̄/d₂, d₂: n=2→1.128）
  CL  = X̄
  LCL = X̄ - 3 × σ̂

移动极差图：
  UCL = D₄ × MR̄          （D₄: n=2→3.267）
  CL  = MR̄
  LCL = 0（n=2时D₃=0）
```

## 过程能力指数（Cpk）速算表

### 标准差法（σ已知/长期数据）

```
Cp  = (USL - LSL) / 6σ          （过程能力）
Cpk = min[(USL - μ)/3σ, (μ - LSL)/3σ]   （过程能力指数）
Ca  = (μ - Target) / (3σ)        （过程准确度）

Cpk等级：
  Cpk < 0.67  → 能力不足，须立即整改
  Cpk 0.67-1.0 → 能力勉强，需要改进
  Cpk 1.0-1.33 → 勉强接受（最低要求）
  Cpk 1.33-1.67 → 能力充分（行业一般要求）
  Cpk ≥ 1.67  → 能力充足（IATF关键安全特性要求）
  Cpk ≥ 2.00  → Six Sigma优秀水平
```

### 子组均值法（短期数据，σ未知）

```
σ̂  = R̄ / d₂    （d₂系数：n=5→2.326, n=4→2.059, n=3→1.693）
Cp  = (USL - LSL) / 6σ̂
Cpk = min[(USL - X̄)/3σ̂, (X̄ - LSL)/3σ̂]
```

### Pp / Ppk（长期过程能力）

```
Pp = (USL - LSL) / 6σ总     （长期能力）
Ppk = min[(USL - μ总)/3σ总, (μ总-LSL)/3σ总]
```

> **判断：** Ppk >> Cpk → 表明过程均值偏离或组内波动大，须改善

## 计数型数据 SPC 控制图

### P 图（不良率控制图）

```
P̄ = 合格批次数 / 总检验批次
UCL = P̄ + 3 × sqrt(P̄(1-P̄)/n)
LCL = P̄ - 3 × sqrt(P̄(1-P̄)/n)
```

### np 图（不良品数控制图）

```
np̄ = 不良品总数 / 批次数
UCL = np̄ + 3 × sqrt(np̄(1-p̄))
LCL = np̄ - 3 × sqrt(np̄(1-p̄))
```

### C 图（缺陷数控制图）

```
C̄ = 缺陷总数 / 检验批数
UCL = C̄ + 3 × sqrt(C̄)
LCL = C̄ - 3 × sqrt(C̄)
```

### U 图（单位缺陷数控制图）

```
Ū = 缺陷总数 / 单位总数
UCL = Ū + 3 × sqrt(Ū/n)
LCL = Ū - 3 × sqrt(Ū/n)
```

## 8项判定规则（计量型控制图）

```
规则1：任何1点超出UCL/LCL → 立即停线，调整工艺
规则2：连续9点在中心线同一侧 → 系统偏移，查找特殊原因
规则3：连续6点递增/递减 → 趋势性偏移
规则4：连续14点交替上下 → 周期性波动（调整节拍）
规则5：任何1点距中心线>2σ → 预警，加强监控
规则6：连续3点中有2点超2σ → 预警
规则7：连续5点中有4点超1σ → 预警
规则8：连续15点在1σ以内 → 数据分层，测量系统问题
规则9：连续8点超1σ → 偏离，须分析
```

## MSA 测量系统分析速查

### GR&R 判定标准（AIAG标准）

```
GR&R ≤ 10%  → 测量系统优秀，可接受
10% < GR&R ≤ 30% → 勉强接受，须改进
GR&R > 30%  → 测量系统不合格，须更换或改进
```

### GR&R 计算（X̄-R法，2或3操作者，n=5零件，r=2重复）

```
EV（重复性）= R̄_操作者 × EV系数
  EV系数：2次重复→0.8862；3次重复→0.5908

AV（再现性）= (X̄diff × 1/厂商变异系数) × AV系数
  AV系数：2操作者→4.56；3操作者→3.05

GR&R = sqrt(EV² + AV²)
%GR&R = GR&R / TV × 100%

TV（总变异）= 6 × sqrt(EV² + AV² + PV²)
PV（零件间变异）= 零件极差 × PV系数
  PV系数：n=5→0.4297
```

### MSA 五维评估

| 维度 | 评估内容 | 判定标准 |
|------|---------|---------|
| 偏倚（Bias）| 测量均值与真值之差 | 偏倚≤10%（参照标准件）|
| 线性（Linearity）| 全量程偏倚一致性 | 线性≤10% |
| 稳定性（Stability）| 随时间变化的偏倚变化 | 稳定，不随时间漂移 |
| 重复性（Repeatability）| 同一操作者重复测量一致性 | %GR&R≤10% |
| 再现性（Reproducibility）| 不同操作者测量一致性 | %GR&R≤10% |
