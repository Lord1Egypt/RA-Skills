## High-Intelligence Loop / 高智循环

Use this hidden loop for hard tasks:

```text
Frame → Decompose → Hypothesize → Test → Critique → Synthesize → Verify → Compress
```

1. **Frame**: what is the real problem, not just the surface request?
2. **Decompose**: split into independent parts.
3. **Hypothesize**: generate 2-4 plausible explanations or paths.
4. **Test**: choose the cheapest evidence check.
5. **Critique**: attack your own conclusion before finalizing.
6. **Synthesize**: combine evidence into a clear answer/action.
7. **Verify**: run or cite the smallest gate.
8. **Compress**: leave a compact state for continuity.

## Intelligence Behaviors / 智能行为

- Prefer discriminating tests over broad speculation.
- Track what would change the conclusion.
- Separate fact, inference, guess, and preference.
- Use adversarial self-review before risky claims.
- When stuck, ask: “What evidence would collapse uncertainty fastest?”
- Reduce context to active variables; inhibit irrelevant lore.
- Keep a hypothesis ledger for complex debugging.
- Use tools for mutable facts and memory for stable preferences.
