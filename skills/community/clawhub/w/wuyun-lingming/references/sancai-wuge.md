# 姓名学：三才五格基础

> 学习日期：2026年4月28日

## 一、三才五格概述

### 1.1 起源
- **熊崎姓名学**：由日本熊崎健翁于1918年创立，又称"五格剖象法"
- 基于《易经》"象"、"数"理论，综合阴阳五行相生相克

### 1.2 五格定义

| 五格 | 别称 | 含义 |
|------|------|------|
| 天格 | 根/父母运 | 祖先、先天，幼年运（1-12岁） |
| 人格 | 苗/主运 | 姓名中心点，中青年运（25-36岁），**核心重点** |
| 地格 | 花/前运 | 名字，前运（13-24岁/38岁前） |
| 外格 | 叶/副运 | 社交、人际关系（37-48岁） |
| 总格 | 果/后运 | 全体笔画，晚年运（49岁后） |

### 1.3 三才
- 三才 = 天格、人格、地格
- 三者五行配置决定吉凶，重点看**相生关系**

## 二、五格计算方法

### 2.1 基本规则
> **必须用繁体字/《康熙字典》笔画数**

### 2.2 四种姓名类型的计算

| 类型 | 示例 | 天格 | 人格 | 地格 | 外格 | 总格 |
|------|------|------|------|------|------|------|
| 单姓单名 | 李刚 | 姓+1 | 姓+名 | 名+1 | 固定2 | 姓名总和 |
| 单姓复名 | 李明哲 | 姓+1 | 姓+名首 | 名总和 | 总格-人格+1 | 姓名总和 |
| 复姓单名 | 司马光 | 姓相加 | 姓氏次字+名 | 名+1 | 总格-人格+1 | 姓名总和 |
| 复姓复名 | 司马懿 | 姓相加 | 姓氏次字+名首 | 名总和 | 总格-人格+1 | 姓名总和 |

### 2.3 举例计算："李明哲"

```
李：7画  明：8画  哲：10画

天格 = 7 + 1 = 8
人格 = 7 + 8 = 15
地格 = 8 + 10 = 18
外格 = (7+8+10) - 15 + 1 = 11
总格 = 7 + 8 + 10 = 25
```

## 三、81数理吉凶

### 3.1 数理五行归属
| 尾数 | 五行 |
|------|------|
| 1、2 | 木 |
| 3、4 | 火 |
| 5、6 | 土 |
| 7、8 | 金 |
| 9、0 | 水 |

### 3.2 吉凶分类

**大吉数**：1、3、5、6、7、8、11、13、15、16、21、23、24、25、29、31、32、33、35、37、39、41、45、47、48、52、57、63、65、68、81

**大凶数**：2、4、9、10、12、14、19、20、22、26、28、34、36、42、43、44、46、49、50、53、54、56、58、59、60、62、64、66、69、70、71、72、73、74、76、77、78、79、80

**半吉半凶**：6、7、17、18、27、30、40、51、55、61、75

### 3.3 重点提示
- **人格**是核心，人格必须为吉
- **地格、总格**忌讳大凶数（如20、22、26、34）
- 女性避开"孤寡运"：21、23、26、28、33、39
- 超过81的数，减去80取余数

## 四、三才五行配置

### 4.1 相生相克关系

**相生（木生火→火生土→土生金→金生水→水生木）**
- 连续相生为大吉，如"木→火→土"、"土→金→水"

**相克（金克木→木克土→土克水→水克火→火克金）**
- 出现相克需谨慎评估

### 4.2 李明哲案例分析

```
三才数理：
- 天格8 → 尾数8 → 金
- 人格15 → 尾数5 → 土  
- 地格18 → 尾数8 → 金

三才配置：金→土→金
结果：金生土，土生金 → 相生循环 → 大吉之象
```

## 六、快速判断口诀

> 一看人格定主运
> 二看三才论相生
> 三看数理避大凶
> 四看外格察人缘

---

## 七、应用原则

1. **以八字为本**：姓名数理应补益八字喜用神
2. **人格为核心**：人格数理必须为吉，五行不冲克日主
3. **三才相生优先**：选择天→人→地连续相生的组合
4. **音形义统一**：数理再好，名字难读难写寓意差也难称佳名

---

## 八、学习心得

- 三才五格是姓名学入门基础，但不可脱离八字单独论断
- 笔画计算必须以康熙字典为准，简体字不准确
- 五行平衡比单纯追求大吉数更重要
- 姓名影响人生但不是决定人生，好名字配好命如锦上添花
