## Description: <br>
Provides a Bilibili-based tutorial helper for learning how to build AI Agent and LLM agent workflows from basics to practice. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and learners use this skill to surface tutorial-oriented guidance and Bilibili learning references for AI Agent and LLM agent development. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill depends on a local Windows KnowledgeBase module and may save text passed to it locally. <br>
Mitigation: Install only in environments where local persistence is acceptable, and avoid passing sensitive or private text unless local storage is intended. <br>
Risk: The broad "llm/agent" trigger can activate during general discussions about LLMs or agents. <br>
Mitigation: Narrow, rename, or disable the trigger if the agent frequently handles unrelated LLM or agent conversations. <br>
