## ✨ 核心功能

- **风险定位**：精确定位条款位置和原文
- **风险说明**：法律依据 + 实际后果分析
- **修改建议**：提供具体可直接使用的修改文案

- **一键改写**：自动生成合规的替代条款
- **对比展示**：原文 vs 修改建议对比

- **100+专业模板**：覆盖全场景合同类型

- **自定义模板**：支持上传企业专属模板
- **版本管理**：模板版本追踪和回滚

### ⚖️ 法律合规检查
- **法规匹配**：自动匹配最新法律法规

## 🚀 快速开始

### 1. 审查合同
```bash
# 进入skill目录
cd ~/.openclaw/workspace/skills/contract-review-pro

# 审查合同文件
python scripts/review_contract.py --file /path/to/contract.txt

# 审查并生成报告
python scripts/review_contract.py --file contract.txt --export report.md
```

### 2. 生成合同
```bash
# 基于模板生成合同
python scripts/generate_contract.py --template 买卖合同 \
  --partyA "甲方公司名" --partyB "乙方公司名" \
  --amount 100000 --output new_contract.md
```

### 3. 列出可用模板
```bash
python scripts/generate_contract.py --list
```

## 📊 审查报告示例

```
【ContractAI Pro 审查报告】
📋 合同名称：XX科技公司采购合同
📅 审查时间：2026-05-13
📊 风险统计：🔴 高风险3项 | 🟡 中风险5项 | 🟢 低风险8项

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔴 【高风险1】违约金条款不合理
📍 位置：第8条第2款
📝 原文：违约方应向守约方支付合同总价50%的违约金
⚠️ 风险：
  - 违约金比例远超司法实践保护范围（一般不超过30%）
  - 存在被法院认定"过高"而予以调减的风险
💡 修改建议：
  违约方应向守约方支付合同总价20%的违约金（违约金不足以弥补实际损失的，违约方还应赔偿差额）
📜 法律依据：《民法典》第585条

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔴 【高风险2】单方解约权不对等
📍 位置：第12条
📝 原文：甲方有权随时解除本合同，无需承担违约责任
⚠️ 风险：
  - 严重违反合同法公平原则
  - 该条款可能被认定为格式条款而无效
💡 修改建议：
  任何一方严重违约致使合同目的无法实现的，守约方有权解除合同，并要求违约方承担违约责任

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🟡 【中风险1】争议解决条款不利
📍 位置：第18条
📝 原文：因本合同引起的争议，由甲方所在地人民法院管辖
⚠️ 风险：
  - 增加乙方维权成本和时间
  - 可能存在地方保护主义风险
💡 修改建议：
  因本合同引起的争议，由原告所在地人民法院管辖

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 总体评价：
   该合同存在3项高风险条款，建议修改后签署
   主要问题集中在违约责任和权利义务对等性方面

📋 建议修改优先级：
  1. 违约金条款 → 必须修改
  2. 解约权条款 → 必须修改
  3. 争议管辖 → 建议修改
```
