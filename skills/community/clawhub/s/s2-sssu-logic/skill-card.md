## Description: <br>
A spatial reasoning prompt framework for OpenClaw. Teaches the agent how to understand the 4m² Smart Space Standard Unit (SSSU) and Mars colonization logic. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[spacesq](https://clawhub.ai/user/spacesq) <br>

MIT-0 <br>


## Use Case: <br>
External users and developers use this instruction-only skill to guide agent reasoning about smart homes, spatial design, compact habitats, and Mars-base planning through the SSSU 4m² grid framework. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill makes broad claims about Chronos, temporal memory, behavior history, and emotional-state awareness that may not be backed by data available to the agent. <br>
Mitigation: Use these claims only when the user has separately provided the relevant data with consent and retention rules; otherwise frame them as conceptual SSSU terminology rather than factual observations. <br>
Risk: The skill can steer an agent toward branded SSSU and Mars-habitat reasoning even for general spatial-design questions. <br>
Mitigation: Install it only when that SpaceSQ design lens is desired, and review agent responses for overconfident or irrelevant application of the framework. <br>
