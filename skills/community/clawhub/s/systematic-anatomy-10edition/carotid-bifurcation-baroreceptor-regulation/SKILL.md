# 颈动脉分叉感受器生理调节

## 何时使用
- 用户描述血压突然升高或降低时的心血管代偿反应
- 用户提及血氧下降（PaO₂↓）、二氧化碳升高（PaCO₂↑）或酸中毒（pH↓）引发的呼吸调节
- 用户咨询颈动脉区域操作（如按摩、压迫）的生理后果或禁忌
- 用户需要解释自主神经系统如何通过外周感受器维持内环境稳定

## 如何执行
1. **识别触发条件**：
   - 若血压异常 → 激活**颈动脉窦**（压力感受器）
   - 若PaO₂↓、PaCO₂↑ 或 pH↓ → 激活**颈动脉小球**（化学感受器）

2. **调用对应反射通路**：
   - **颈动脉窦反射**：血压↑ → 窦壁扩张 → 传入神经（舌咽神经）→ 延髓心血管中枢 → 迷走神经兴奋 + 交感抑制 → 心率↓、血管舒张 → 血压↓
   - **颈动脉小球反射**：血气异常 → 化学感受器兴奋 → 传入神经（舌咽神经）→ 延髓呼吸中枢 → 呼吸肌活动增强 → 呼吸加深加快 → 改善气体交换

3. **评估临床适用性**：
   - 确认患者无颈动脉严重粥样硬化或既往颈动脉手术史（否则反射功能受损）
   - 警惕颈动脉窦过度敏感导致的心动过缓、晕厥（颈动脉窦综合征）

4. **输出解释**：
   - 清晰说明感受器位置、刺激变量、神经通路及最终生理效应
   - 强调两者协同维持循环与呼吸稳态的核心作用
