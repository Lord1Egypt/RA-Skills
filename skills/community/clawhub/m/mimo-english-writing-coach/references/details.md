# english-writing-coach 详细参考

## 一、写作类型深度指南

### 1.1 学术论文写作规范
```
特点：正式、客观、引用规范、逻辑严谨
时态：方法-过去时，结果-过去时，讨论-现在时
语态：Methods/Results倾向被动，结果讨论可用主动
句式：复杂句为主，长短句结合
常见问题：
- "I think/believe" → "This study suggests/indicates"
- "a lot of" → "numerous/significant"
- "doesn't work" → "failed to demonstrate significance"
```

### 1.2 商务邮件写作规范
```
特点：简洁、专业、行动导向、时间敏感
结构：结论先行（0-10秒原则）
称呼：Dear Mr./Ms. [Name], / Hello [Name], / Hi [Name],
结尾：期待回复/感谢/下一步行动
常见问题：
- 过长段落 → 拆分，每段一要点
- 模糊时间 → "by Friday EOD" / "by June 15"
- 被动攻击 → 直接表达需求
```

### 1.3 创意写作规范
```
特点：生动、有风格、吸引人、情感共鸣
技巧：
- 感官细节：show don't tell
- 对话推进：推动情节发展
- 节奏变化：长短句交替
- 修辞手法：比喻、拟人、夸张
常见问题：
- 陈词滥调 → 寻找独特表达
- 过度描写 → 删减不影响情节的描写
```

## 二、语法检查清单

### 2.1 时态系统
| 错误类型 | 错误示例 | 正确写法 |
|---------|---------|---------|
| 时态跳跃 | "I studied hard and realize the importance" | "I studied hard and realized the importance" |
| 现在完成时滥用 | "I have studied yesterday" | "I studied yesterday" |
| 过去完成时遗漏 | "I reached the station, the train left" | "I reached the station, the train had left" |
| 条件句时态 | "If I will have money, I buy a car" | "If I have money, I will buy a car" |

### 2.2 主谓一致规则
```
单数+each/every/no → 动词用单数
Each student has to submit their assignment.
No one knows the answer.

集体名词（英美差异）
- 美式：The team is playing well. (视为整体)
- 英式：The team are playing well. (视为个体)

数字/量词 + of → 谓语与of后的名词一致
A number of students are late. (students是复数)
The number of students is increasing. (number是单数)

None的用法
- None + 复数名词：None of them are ready. / None of them is ready. (均可接受)
- None + 单数不可数：None of the money was recovered.
```

### 2.3 冠词使用规则
```
可数vs不可数
- 可数：I need a solution. / The solution is clear.
- 不可数：I need information. / The information is useful.

抽象名词
- 一般不加：Life is beautiful. / Happiness is key.
- 特指时加：The life I want. / The happiness she brings.

固定搭配
- at university (泛指去上大学) vs at the university (具体地点)
- in school / go to school vs in the school / go to the school
- watch TV / go to bed / at home (不加the)
```

### 2.4 介词搭配速查
```
时间
- at: at 9 o'clock, at night, at the weekend (英) / on the weekend (美)
- in: in the morning, in June, in 2024, in summer
- on: on Monday, on June 15, on a cold morning

地点
- at: at home, at school, at work, at the bus stop
- in: in the room, in the city, in China
- on: on the table, on the wall, on the second floor

动词+介词
- depend on / believe in / listen to / wait for
- agree with / argue with / succeed in / worry about

形容词+介词
- good at / interested in / afraid of / proud of
- similar to / different from / famous for / aware of
```

### 2.5 标点符号规范
```
逗号使用
- 连接两个独立分句：I came home, and I found a letter.
- 非限制性定语从句：My brother, who lives in Tokyo, is visiting.
- 句中状语：After dinner, we went for a walk.
- 避免：连接两个简单词不加逗号

分号使用
- 两个独立分句（不用连词）：I love coffee; she prefers tea.
- 复杂列表中的分隔：There are three main points: first, the cost; second, the time; third, the effort.

引号
- 美式：双引号" " 内用单引号' '
- 英式：单引号' ' 内用双引号" "
- 引用句号位置：美式句号在引号内，英式引号外
```

## 三、表达优化词库

### 3.1 弱动词替换表
```
通用提升
- make → create/generate/produce/establish/build
- do → perform/execute/conduct/implement/carry out
- get → obtain/acquire/secure/achieve/gain
- have → possess/hold/own/maintain/retain

具体场景
- find out → discover/investigate/determine/ascertain
- look for → seek/search for/seek out/pursue
- think about → consider/reflect on/contemplate/deliberate
- try to → attempt to/endeavor to/strive to/seek to
```

### 3.2 冗余表达消除表
```
常见冗余 → 简洁表达
- in order to → to
- due to the fact that → because (of)
- at this point in time → now/currently/presently
- in the event that → if/should
- a large number of → many/numerous/substantial
- for the purpose of → to/for
- in spite of the fact that → although/though
- with regard to → about/regarding/concerning
- on a daily basis → daily/every day
- in the near future → soon/shortly/upcoming
- it is important to note that → note that/importantly
```

### 3.3 过渡词分类表
```
递进/补充
furthermore, moreover, in addition, additionally, besides, also, what's more

转折
however, nevertheless, nonetheless, on the other hand, conversely, whereas, although, even though

因果
therefore, consequently, thus, hence, as a result, accordingly, because of this

举例
for example, for instance, such as, including, namely, in particular

总结
in conclusion, to summarize, overall, in summary, to conclude, all in all

强调
indeed, certainly, definitely, obviously, clearly, undoubtedly, particularly, especially

对比
similarly, likewise, in the same way, on the contrary, in contrast, unlike
```

## 四、风格调整指南

### 4.1 正式度分级
```
非常正式（学术论文、法律文档）
- 不使用缩写：cannot, do not, it is
- 使用完整表达：in accordance with, with regard to
- 避免第一人称：This study was conducted...

半正式（商务邮件、报告）
- 可用缩写：can't, don't, it's
- 保持专业语气
- 适度使用第一人称

非正式（日常沟通、社交媒体）
- 可用口语：gonna, wanna, kinda
- 使用缩略语
- 允许感叹号和表情
```

### 4.2 语态选择原则
```
主动优先场景
- 强调动作执行者
- 句子更简洁有力
- 主语明确

被动适用场景
- 动作接受者更重要：The data was analyzed.
- 未知执行者：The window was broken.
- 保持客观性（学术）
- 避免提及主语

主动改被动练习
- "We conducted the experiment." → "The experiment was conducted (by us)."
- "The software processes data." → "Data is processed (by the software)."
```

### 4.3 句式变化技巧
```
长句构建
- 使用从句：because/although/when/if + 完整句子
- 使用分词结构：Having finished..., The data collected...
- 使用插入语：The results, as shown in Figure 1, indicate...

短句使用
- 强调关键点
- 制造节奏感
- 避免连续使用3个以上长句

修辞疑问句
- "Isn't it obvious?" = "It is obvious."
- "Who wouldn't want this?" = "Everyone wants this."
```

## 五、评分标准详解

### 5.1 语法评分维度（10分制）
```
9-10：基本无语法错误，或仅有minor issues
7-8：少量错误（3-5处），不影响理解
5-6：常见错误（6-10处），可能影响理解
3-4：错误较多（10+处），理解困难
1-2：语法混乱，难以理解
```

### 5.2 表达评分维度（10分制）
```
9-10：词汇精准丰富，句式多样地道
7-8：表达准确，偶有用词不当
5-6：表达基本准确，但缺乏变化
3-4：词汇重复，句式单一
1-2：表达混乱，难以传达意思
```

### 5.3 风格评分维度（10分制）
```
9-10：风格统一得体，适合目标读者
7-8：风格基本合适，偶有偏差
5-6：风格选择基本正确，执行有偏差
3-4：风格与场景不匹配
1-2：完全不适合目标场景
```

## 六、常见错误案例解析

### 6.1 Chinglish 典型错误
```
中文思维 → 英文表达
我认为... → I think... (非 I think that...)
非常感谢... → Thank you very much (非 Very thank you...)
我的意思是... → What I mean is... (非 My meaning is...)
打开电脑 → Turn on the computer (非 Open the computer)
学习知识 → Learn knowledge / Gain knowledge (可接受，但Acquisition of knowledge更学术)
```

### 6.2 易混词辨析
```
affect (v.影响) vs effect (n.效果)
- The weather affects my mood. / The effect was significant.

advice (n.) vs advise (v.)
- I need your advice. / I advise you to...

principal (adj.主要的/ n.校长) vs principle (n.原则)
- The principal reason / The school principal / a matter of principle

farther vs further
- Farther: 物理距离更远 / Further: 程度更进一步（两者均可用于距离）

imply (暗示) vs infer (推断)
- His words implied distrust. / I inferred from his words that...
```

### 6.3 常见拼写错误
```
- accommodate (双c双m)
- occurrence (双r双s)
- necessary (一 burr y)
- occurrence vs. occurrences
- embarrassment (双r三s)
- definitely (不是 definately)
- separate (不是 seperate)
- occasion (不是 occassion)
- recommend (不是 reccommend)
```
