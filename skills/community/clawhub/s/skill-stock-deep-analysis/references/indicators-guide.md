# 股票财务指标评价参考指南

## 盈利能力指标

### ROE（净资产收益率）
| 范围 | 评级 | 说明 |
|------|------|------|
| > 20% | ✅ 优秀 | 股东回报极佳 |
| 15-20% | ✅ 良好 | 良好水平 |
| 8-15% | ⚠️ 一般 | 行业平均水平 |
| < 8% | ❌ 较差 | 低于均值，需关注 |

### ROA（资产收益率）
| 范围 | 评级 | 说明 |
|------|------|------|
| > 8% | ✅ 优秀 | 资产利用效率高 |
| 4-8% | ✅ 良好 | 正常水平 |
| 1-4% | ⚠️ 一般 | 偏低 |
| < 1% | ❌ 较差 | 资产利用效率低 |

### 净利润率
| 范围 | 评级（消费/互联网） |
|------|------|
| > 20% | ✅ 优秀 |
| 10-20% | ✅ 良好 |
| 5-10% | ⚠️ 一般 |
| < 5% | ❌ 较低 |

### 毛利率
| 范围 | 评级（互联网平台） |
|------|------|
| > 60% | ✅ 优秀（平台型） |
| 40-60% | ✅ 良好 |
| 20-40% | ⚠️ 一般（电商型） |
| < 20% | ❌ 较低（重资产） |

## 估值指标

### PE（市盈率）
| 范围 | 评级 | 说明 |
|------|------|------|
| < 10 | ✅ 极低 | 价值洼地或困境 |
| 10-15 | ✅ 低估 | 合理偏低 |
| 15-25 | ⚠️ 合理 | 市场平均 |
| 25-40 | ⚠️ 偏高 | 成长股正常区间 |
| > 40 | ❌ 高估 | 需高增长支撑 |

### Forward PE（预期 PE）
同 PE，但使用未来12个月预期 EPS

### PB（市净率）
| 范围 | 评级 |
|------|------|
| < 1 | ✅ 破净（资产型） |
| 1-3 | ✅ 合理 |
| 3-5 | ⚠️ 偏高 |
| > 5 | ❌ 高估 |

### PS（市销率，互联网公司）
| 范围 | 评级 |
|------|------|
| < 3 | ✅ 低估 |
| 3-8 | ⚠️ 合理 |
| 8-15 | ⚠️ 偏高 |
| > 15 | ❌ 高估 |

---

## 财务健康指标

### 资产负债率
| 范围 | 评级（互联网） |
|------|------|
| < 30% | ✅ 优秀（低杠杆） |
| 30-50% | ✅ 良好 |
| 50-70% | ⚠️ 一般 |
| > 70% | ❌ 高风险 |

### 流动比率
| 范围 | 评级 |
|------|------|
| > 2 | ✅ 优秀 |
| 1.5-2 | ✅ 良好 |
| 1-1.5 | ⚠️ 尚可 |
| < 1 | ❌ 短期偿债风险 |

### 速动比率
| 范围 | 评级 |
|------|------|
| > 1 | ✅ 健康 |
| 0.5-1 | ⚠️ 尚可 |
| < 0.5 | ❌ 流动性紧张 |

### 债务/权益比
| 范围 | 评级 |
|------|------|
| < 30% | ✅ 低杠杆 |
| 30-60% | ⚠️ 中等 |
| 60-100% | ⚠️ 偏高 |
| > 100% | ❌ 高杠杆 |
