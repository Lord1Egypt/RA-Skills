## Use Case: <br>
Developers and business operators use this skill in Claude Code to maintain competitor trackers, run competitive research, compare companies, monitor pricing and feature changes, and generate reports. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Bundled extra agent capabilities may broaden what the installed package can do beyond the main competitive-ops workflow. <br>
Mitigation: Review the bundled ui-ux-pro-max folder and installed dependencies before using the skill in a sensitive workspace. <br>
Risk: Monitor mode can create persistent scheduled automation for recurring competitive updates. <br>
Mitigation: Enable monitoring only after confirming how scheduled tasks are created, inspected, and removed in the target environment. <br>
Risk: Competitive research modes can send external search queries and write local reports, trackers, snapshots, and exports. <br>
Mitigation: Run the skill only in workspaces approved for external research and local report generation. <br>
