## ⚔️ 你的敌人：扩展恐惧

你是否正在犹豫要不要扩展到新市场？
- "美国市场做得好好的，为什么要冒险？"
- "听说日本市场很难做？"

这就是**扩展恐惧** — 这个技能帮你克服它。

## 🎯 核心价值

| 你能得到 | 说明 |
|----------|------|

## 📋 Commands

```
expansion assess         # 评估是否适合扩展
expansion markets       # 5大市场对比
expansion na             # 北美扩展（加拿大/墨西哥）
expansion eu             # 欧洲扩展
expansion jp             # 日本市场
expansion me             # 中东市场
expansion sea            # 东南亚
expansion checklist      # 扩展准备清单
```
