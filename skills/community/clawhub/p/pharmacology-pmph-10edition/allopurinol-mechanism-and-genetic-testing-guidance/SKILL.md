# 别嘌醇作用机制与基因检测建议

## 何时使用
- 患者确诊为慢性痛风或需长期降尿酸治疗；
- 当前无急性痛风发作；
- 已完成肾功能评估；
- 正在考虑启动别嘌醇治疗。

## 执行步骤
1. **确认禁忌与前提条件**：
   - 急性痛风发作期不首选别嘌醇；
   - HLA-B*5801基因阳性者禁用。

2. **解释作用机制**：
   - 别嘌醇为次黄嘌呤类似物，在低浓度时竞争性抑制黄嘌呤氧化酶，高浓度时转为非竞争性抑制；
   - 在肝脏代谢为活性产物奥昔嘌醇，后者为长效非竞争性抑制剂，半衰期14～28小时，持续阻断尿酸合成。

3. **评估药代动力学与剂量调整**：
   - 口服后0.5～1小时达峰浓度；
   - 肾功能不全者起始剂量 ≤ 1.5 mg / eGFR（单位：mL/min/1.73m²），并缓慢加量。

4. **推荐基因检测**：
   - 如条件允许，治疗前必须进行HLA-B*5801基因检测；
   - 若结果为阳性，禁止使用别嘌醇；
   - 若阴性，可按肾功能调整剂量开始治疗。

5. **监测安全性**：
   - 警惕皮疹、胃肠不适、转氨酶升高、白细胞减少等不良反应；
   - 出现严重皮肤反应立即停药并就医。

> **关键提醒**：HLA-B*5801阳性显著增加史蒂文斯-约翰逊综合征（SJS）和中毒性表皮坏死松解症（TEN）风险，基因检测是安全用药的前提。
