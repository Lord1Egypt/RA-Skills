# Pathogen-Vital Qi Dynamic Analysis

## When to Use
- 分析疾病发展阶段或证候性质时
- 判断虚证实证及其真假时
- 评估预后或疾病转归趋势时
- 制定或调整治疗策略（扶正 vs 祛邪）时
- 评估患者整体抗病能力与病理反应状态时

## Execution Steps

### 1. 明确基本概念
- **正气**：指人体的生理功能、抗病及康复能力。
- **邪气**：泛指一切致病因素（外感六淫、内伤七情、痰饮瘀血等）。
- 核心原则：“正气存内，邪不可干”；“邪之所凑，其气必虚”。

### 2. 判断邪正力量对比
根据临床表现，确定当前状态属于以下哪一类：
- 邪盛正未衰（多见于实证）
- 正虚邪不盛（多见于虚证）
- 邪正相持 / 正虚邪恋 / 邪去正未复（多见于慢性或迁延期）

### 3. 辨识证候虚实
- **实证特征**：功能亢进或障碍、气血壅滞、病势急、体质尚可；常见于外感病初中期或痰食水血内停。
- **虚证特征**：功能减退、精气血津液亏少、病程久、体质虚弱；常见于先天不足、久病、大出血后。
- **注意虚实真假**：
  - 真虚假实（“至虚有盛候”）：本质为虚，但出现类似实证的表现（如腹胀但喜按）。
  - 真实假虚（“大实有赢状”）：本质为实，但出现类似虚证的表现（如神疲但腹痛拒按）。

### 4. 分析疾病转归趋势
- **正胜邪退** → 病情好转或痊愈。
- **邪胜正衰** → 病情恶化，甚至危及生命。
- **邪正相持/正虚邪恋/邪去正未复** → 疾病迁延、转为慢性或遗留后遗症。

### 5. 观察病邪出入动态
- **表邪入里**：提示正不胜邪，病情加重（如风温表证转为肺热里证）。
- **里邪出表**：提示正气恢复，邪有出路，预后良好（如温病汗出热退、斑疹外透）。

### 6. 指导治疗策略
- 邪盛为主 → 以祛邪为先。
- 正虚为主 → 以扶正为要。
- 虚实夹杂 → 扶正与祛邪并用，分清主次缓急。

> ⚠️ **限制条件**：本技能不适用于无正邪斗争过程的纯器质性损伤（如截肢、外伤骨折无感染等）。
