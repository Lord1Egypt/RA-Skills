# Skill: auto-insight-engine

自动洞察引擎 — 从6个来源采集洞察：HackerNews/GitHub/Wikipedia/RSS/ArXiv/扩展搜索

## Functionality
- 6 sources: HN 15 + GitHub 8 + Wikipedia 5 + 4 RSS feeds 20 + ArXiv API + extended search
- ~48 items per cycle
- 5 deep-analysis per cycle via NVIDIA NIM
- Trending keyword tracking with rising trend detection
- Auto-KB-ingest integration with knowledge bridge
- 6-phase engine loop

## Invocation
- `run("run")` — execute one insight collection cycle
- `run("status")` — show engine status
- `run("trends")` — show trending keywords from trends.json
