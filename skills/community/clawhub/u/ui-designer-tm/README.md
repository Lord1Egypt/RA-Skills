# 🎨 UI Designer

> Transform your interface ideas into pixel-perfect, accessible user interfaces — powered by OpenClaw.

**UI Designer** is an OpenClaw Skill that brings expert visual design knowledge to your workspace. Whether you're building a login form, a dashboard, or a full design system, this skill helps you create consistent, accessible, beautiful interfaces without starting from scratch.

---

## ✨ Features

### Design Token System
- Complete CSS custom property framework (colors, typography, spacing, shadows, transitions)
- Light & dark theme support built in
- Semantic color tokens (success, warning, error, info)

### Component Patterns
- Button (primary, secondary, ghost, danger — all sizes + states)
- Form inputs with validation states
- Cards, badges, loading skeletons
- Responsive layout patterns

### Accessibility (WCAG AA)
- Color contrast compliance baked into every token
- Focus ring visibility on all interactive elements
- Keyboard navigation support
- ARIA labels and screen reader announcements
