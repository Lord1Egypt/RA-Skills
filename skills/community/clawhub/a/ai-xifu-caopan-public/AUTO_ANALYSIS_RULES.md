## 一、触发规则（安全增强版）

> ⚠️ **安全声明**：本技能仅响应以"AI媳妇"开头的明确指令，杜绝日常对话中意外触发。

### 1.1 正确触发方式
用户必须以"AI媳妇"开头发出完整指令，例如：
- "AI媳妇，帮我生成一份某某股票的完整交易方案"
- "AI媳妇，帮我分析某个期货主力合约"
- "AI媳妇，给我一份某某基金的方案"

### 1.2 不会触发的情况
以下关键词在未带"AI媳妇"前缀时**不会触发**任何操作：
- ❌ 仅提到"股票"、"做个股票"、"分析股票"
- ❌ 仅提到"期货"、"做个期货"、"分析期货"
- ❌ 仅提到股票代码或股票名称
- ❌ 对话中的普通词语

### 1.3 支持的分析类型
当以"AI媳妇"发出指令后，可分析：
- 股票：A股所有个股
- 期货：商品期货主力合约
- 基金：公募基金
- 经济：宏观经济
- 期货品种名称：螺纹钢、铁矿石、原油、黄金、白银、铜、铝、锌、镍、锡、铅、焦煤、焦炭、动力煤、玻璃、纯碱、甲醇、PTA、棉花、白糖、豆粕、菜粕、玉米、淀粉、鸡蛋、生猪等
- 期货合约格式：品种+月份（如：螺纹2505、铁矿2505）

**⚠️ 重要规则：所有期货分析统一使用主力合约！**
- 主力合约定义：持仓量最大的合约
- 常见主力合约代码：
  - 沪铜主力：CU主连 或 CU2605（当前主力）
  - 螺纹主力：RB主连 或 RB2505
  - 铁矿主力：I主连 或 I2505
  - 黄金主力：AU主连 或 AU2506
  - 原油主力：SC主连 或 SC2506

### 1.3 基金类关键词
- "基金" / "做个基金" / "分析基金"
- "某某基金"
- 基金代码格式：6位数字（如：110022、161725）

### 1.4 经济类关键词
- "经济分析" / "宏观经济"
- "大盘分析" / "市场分析"

#### 2.5 数据调取检查清单

**写方案前必须确认已获取：**

- [ ] 实时行情数据（收盘价、涨跌幅、成交量、换手率）
- [ ] 资金流向数据（主力净流入、北向资金）
- [ ] 关联板块数据（所属板块涨跌幅）
- [ ] **近3日历史行情数据（必须！用于趋势判断）**
- [ ] 大盘指数数据（上证、深证、创业板）
- [ ] **大盘近3日走势数据（必须！用于大盘趋势判断）**

- [ ] 新闻资讯（最新相关新闻）

**⚠️ 如果数据获取失败：**

2. 使用合理的估算值（需标注"估算"）
3. 不影响整体分析框架

#### 2.6 走势预判规则（重要！）

**📌 核心原则：结合近3日数据+大盘走势，预判今日/明日走势**

**Step 1：获取近3日数据**
```
- 品种近3日收盘价、涨跌幅、成交量
- 大盘近3日收盘点位、涨跌幅、成交额
```

**Step 2：分析近3日趋势**
```
- 品种趋势：连续上涨/连续下跌/震荡
- 大盘趋势：连续上涨/连续下跌/震荡
- 品种与大盘联动性：同步/背离
```

**Step 3：预判今日/明日走势**
```
- 如果近3日连续上涨 + 大盘走强 → 预判继续上涨
- 如果近3日连续上涨 + 大盘走弱 → 预判回调风险
- 如果近3日连续下跌 + 大盘走强 → 预判反弹机会
- 如果近3日连续下跌 + 大盘走弱 → 预判继续下跌
- 如果近3日震荡 + 大盘震荡 → 预判继续震荡
```

**Step 4：在方案中体现**
```
- 行情回顾章节：展示近3日数据
- 走势分析章节：分析近3日趋势
- 操作建议章节：基于趋势预判给出建议
```

**预判输出格式：**

| 时间维度 | 走势判断 | 依据 |
|---------|---------|------|
| 近3日回顾 | {上涨/下跌/震荡} | {具体数据} |
| 今日预判 | {上涨/下跌/震荡} | {依据} |
| 明日预判 | {上涨/下跌/震荡} | {依据} |
| 操作建议 | {买入/观望/卖出} | {依据} |

## 四、特殊处理

### 4.1 大盘风险高时
当综合下跌概率 > 50% 时：
1. 先输出风险预警
2. 提醒用户当前大盘风险较高
3. **询问用户**是否继续分析
4. 用户确认后继续，在方案中标注风险等级

### 4.2 数据获取失败时
当API数据获取失败时：
1. **告知用户**数据获取失败情况
2. 询问用户是否接受使用合理估算值
3. 用户确认后，在方案中标注"估算值"
4. 用户拒绝时，停止生成方案并说明原因
