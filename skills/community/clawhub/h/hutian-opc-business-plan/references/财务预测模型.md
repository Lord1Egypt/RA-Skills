## 四、单位经济模型（LTV/CAC）

### LTV/CAC分析框架

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│           LTV (客户生命周期价值)                         │
│               ┌───────────────┐                        │
│               │   毛利贡献     │                        │
│               │  100元/年 ×    │                        │
│               │   5年生命周期  │                        │
│               │   = 500元      │                        │
│               └───────────────┘                        │
│                      │                                 │
│                      ▼                                 │
│               ┌───────────────┐                        │
│               │   LTV/CAC     │                        │
│               │    > 3.0      │  ← 优秀                 │
│               │    2.0-3.0    │  ← 良好                 │
│               │    1.0-2.0    │  ← 警告                 │
│               │    < 1.0      │  ← 危险                 │
│               └───────────────┘                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 单位经济指标详解

**CAC（客户获取成本）**
```
CAC = 营销总投入 / 新增付费客户数

示例：
- 月营销投入：30万元
- 月新增付费客户：30个
- CAC = 30万 / 30 = 1万元/客户
```

**LTV（客户生命周期价值）**
```
LTV = (客户年均收入 × 毛利率) × 平均客户生命周期

示例：
- 年均收入：5万元
- 毛利率：70%
- 平均生命周期：5年
- LTV = (5万 × 70%) × 5 = 17.5万元
```

**Payback Period（回收周期）**
```
Payback Period = CAC / (月均毛利贡献)

示例：
- CAC：1万元
- 月均毛利贡献：1,000元
- Payback Period = 10个月
```

### Burn Rate与Runway计算

```
月Burn Rate = (期初现金 - 期末现金) / 月份数

示例：
- 期末现金：1,350万
- 期初现金：500万
- 月Burn Rate = (1,350 - 500) / 12 = 70.8万/月

Runway = 期末现金 / 月Burn Rate

示例：
- 期末现金：1,350万
- 月Burn Rate：70.8万
- Runway = 1,350 / 70.8 = 19个月
```

## 六、估值模型

### 估值方法选择矩阵

| 估值方法 | 适用场景 | 优势 | 劣势 | 估值结果 |
|---------|---------|-----|-----|---------|
| DCF | 现金流稳定 | 理论基础强 | 假设主观 | 较保守 |
| P/E倍数 | 盈利企业 | 简单直观 | 忽视成长 | 中性 |
| P/S倍数 | 高增长企业 | 适合早期 | 忽视盈利 | 较激进 |
| EV/用户 | 互联网企业 | 反映规模 | 忽视变现 | 因人而异 |
| 融资金额反推 | 早期项目 | 操作简单 | 精度低 | 参考 |

### SaaS企业估值参考

```
估值倍数参考（基于ARR增长率）

| ARR增长率 | 低市销率 | 中市销率 | 高市销率 |
|----------|---------|---------|---------|
| < 50%    | 3-5x    | 5-7x    | 7-10x   |
| 50-100%  | 5-8x    | 8-12x   | 12-18x  |
| 100%+    | 8-12x   | 12-20x  | 20-30x  |
| 200%+    | 12-18x  | 18-30x  | 30-50x  |
```

### 估值计算示例

**P/S倍数法**
```
假设：
- 当前ARR：500万
- 预期3年后ARR：3,000万
- 行业P/S倍数：10x
- 折现率：20%

当前估值 = 500万 × 10x = 5,000万
3年后估值 = 3,000万 × 10x = 3亿
现值 = 3亿 / (1.2^3) = 1.74亿

合理估值区间：1.5-2.5亿
```
