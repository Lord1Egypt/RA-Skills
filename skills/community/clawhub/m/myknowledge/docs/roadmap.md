## 版本发布节奏

- **主版本（X.0.0）**：不兼容 API 变更，约 6-12 个月一次
- **次版本（1.X.0）**：向下兼容功能添加，约 1-2 个月一次
- **修订号（1.1.X）**：向下兼容问题修复，约 1-2 周一次
