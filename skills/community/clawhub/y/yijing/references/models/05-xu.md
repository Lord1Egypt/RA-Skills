# 05 ䷄ 需卦 — 时机博弈模型

## 卦象结构

![卦象图](../hexagrams/05-xu.svg)
- 下乾 ☰ 上坎 ☵（水天需）
- 内部状态：有实力、有动力、想往前冲（乾为健）
- 外部环境：前方有风险，需要等待（坎为险）

## 情境识别

什么时候你会用到这个模型？
- 决定融资时机：现在融还是等数据更好？
- 等待市场窗口：行业即将爆发但还没到最佳切入点
- 新产品发布时间：技术已成熟，但市场教育还不够
- 竞争对手正在犯错，你在等最佳反击时机
- 任何"不是不做，是时候未到"的决策时刻

## 核心原则

1. **需于郊，利用恒**：等待是一个主动行为，不是消极不作为。在等待中保持准备，保持节奏。
2. **需于沙，小有言**：等待过程中会有噪音——别人融到钱了、别人发布了、别人比你快。别被噪音牵着走。
3. **需于泥，致寇至**：等得太久，泥足深陷，机会窗口可能关闭。等待也有成本。
4. **需于血，出自穴**：最关键的等待时刻，需要付出重大代价（裁员、砍业务），但这正是"破穴而出"的前夜。
5. **需于酒食**：等待期间保持组织的士气和健康。"酒食"不是享乐，是维持正常的组织营养。

## 阶段地图（六爻 → 时机管理阶段）

- **初九·需于郊，利用恒，无咎** → 远距等待：离战场还很远，不急躁。保持节奏，持续准备。这是最从容的等待状态。
- **九二·需于沙，小有言，终吉** → 靠近战场：已经靠近水边（沙=水边），开始听到各种噪音（小有言）。竞争对手的动作、市场的波动、投资人的催促。坚持住。
- **九三·需于泥，致寇至** → 陷得太深：等太久了，开始陷进泥里。这时风险最大——"寇"不仅是外部对手，更是内部士气流失和人才出走。
- **六四·需于血，出自穴** → 关键时刻：最艰难的等待——可能需要放弃一些业务、裁一些人、接受一个不太理想的估值。但这正是突破的前夜。
- **九五·需于酒食，贞吉** → 等待中的滋养：最高明的等待——边等边建设团队、打磨产品、提升组织能力。不是在"熬"，是在"养"。
- **上六·入于穴，有不速之客三人来** → 等待结束：窗口终于打开，但"不速之客"（竞争者、抄底者）也同时出现。机会窗口打开时，往往也是竞争最激烈时。

## 关键警示

- ❌ 最常见的错误：把"等待"当成"不作为"——需卦的等是 active waiting，每时每刻都在准备
- ❌ 等得太久（需于泥）变成惰性——"再等等看"成了永远的借口
- ❌ 等到了但没准备好（上六"不速之客"）——窗口来了，但你的人、钱、能力都没准备好

## 关联模型（格栅连接）

- ䷄ **讼卦（综卦+后卦）**：需是"等"，讼是"争"。你等了半天，窗口打开时竞争者同时涌入，等就变成了争。
- ䷥ **睽卦（互卦）**：需的内部机制是睽 → 等待过程中团队容易出现分歧。有人等不及想冲，有人更保守。管理好等待期的内部对齐。
- ䷢ **晋卦（错卦）**：需是"等待时机"，晋是"趁势而上"。错卦关系提醒你：有时候不需要等，直接上反而是对的。
- ䷂ **屯卦**：同样是"不能冒进"，屯是"力量不够"，需是"时机不对"。别搞混——没钱是屯的问题，有钱但不对时候是需的问题。
