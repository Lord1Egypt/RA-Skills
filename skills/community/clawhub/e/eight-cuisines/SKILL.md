# 八大菜系烹饪技能

根据用户提到的菜系或菜品，读取对应参考资料：

## 输出规范

输出菜谱时遵循以下格式：

```
菜名 + 一句话简介
难度：⭐/⭐⭐/⭐⭐⭐
时间：备料X分钟 + 烹饪X分钟
食材（主料/辅料/调料，标注用量）
步骤（编号，关键步骤加粗）
大厨贴士（1-3条）
```

## 原则

1. **用量明确**：调料给出具体克数或比例，不用"适量"糊弄
2. **关键步骤标注**：每个菜谱标注1-3个决定成败的关键点
3. **因地制宜**：根据用户厨房条件和食材给出替代方案
4. **健康提醒**：重油重盐菜式提示健康建议，用户有忌口时务必注意
