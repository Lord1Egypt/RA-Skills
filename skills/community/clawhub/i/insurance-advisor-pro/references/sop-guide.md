# 保险顾问标准作业流程（SOP）参考

## 一、KYC信息采集清单

### 1.1 基本信息
| 项目 | 采集要点 | 备注 |
|------|----------|------|
| 姓名 | 真实姓名 | 用于保单填写 |
| 性别 | 男/女 | 影响保费计算 |
| 年龄/出生日期 | 周岁 | 影响费率和保额 |
| 职业 | 具体职业类别 | 1-6类职业分类 |
| 婚姻状况 | 未婚/已婚/离异 | 影响家庭责任评估 |
| 家庭结构 | 几口人、子女年龄 | 影响保障需求 |
| 所在地区 | 城市/省份 | 影响医疗资源评估 |

### 1.2 健康状况（关键）
| 项目 | 采集要点 | 备注 |
|------|----------|------|
| 身高体重 | 计算BMI | BMI>28或<18需关注 |
| 吸烟史 | 是否吸烟、烟龄 | 影响费率 |
| 饮酒史 | 频率和量 | 大量饮酒需关注 |
| 既往病史 | 住院、手术、慢性病 | 如实告知核心 |
| 家族病史 | 父母/兄弟姐妹重大病史 | 遗传风险评估 |
| 体检异常 | 近2年体检异常项 | 结节、息肉、指标异常 |
| 当前用药 | 长期服药情况 | 慢性病指标 |
| 医保/社保 | 是否有基本医保 | 影响医疗险选择 |

### 1.3 财务状况
| 项目 | 采集要点 | 备注 |
|------|----------|------|
| 年收入 | 税后年收入 | 建议保费不超过10-15% |
| 家庭负债 | 房贷、车贷等 | 影响寿险保额需求 |
| 现有保障 | 已有保单梳理 | 避免重复投保 |
| 储蓄/投资 | 流动资产情况 | 影响缴费能力 |
| 保险预算 | 可接受年保费 | 明确预算范围 |

### 1.4 需求偏好
| 项目 | 采集要点 | 备注 |
|------|----------|------|
| 最担心的风险 | 疾病/意外/养老 | 优先满足核心担忧 |
| 品牌偏好 | 大公司/性价比 | 影响产品选择 |
| 线上/线下 | 接受互联网保险 | 影响推荐范围 |
| 缴费偏好 | 年缴/月缴 | 影响产品选择 |

## 二、风险评估标准

### 2.1 疾病风险等级评估

**高风险信号：**
- 年龄>45岁
- BMI>30或<17
- 有既往重疾病史
- 家族有遗传病史
- 长期吸烟/大量饮酒
- 从事高风险职业

**中风险信号：**
- 年龄35-45岁
- BMI 25-30
- 有慢性病（高血压、糖尿病等）
- 体检有异常指标

**低风险信号：**
- 年龄<35岁
- BMI正常
- 无既往病史
- 无家族病史
- 健康生活方式

### 2.2 保障缺口计算公式

**重疾险缺口：**
```
建议保额 = 年收入 × 3-5倍
缺口 = 建议保额 - 已有重疾保额
```

**医疗险缺口：**
```
建议保额 = 100-300万（百万医疗）
缺口分析：是否有医保、是否覆盖特需/国际部
```

**寿险缺口（家庭经济支柱）：**
```
建议保额 = 家庭负债 + 子女教育费 + 父母赡养费 - 已有资产
```

**意外险缺口：**
```
建议保额 = 年收入 × 10-20倍
```

## 三、产品配置策略

### 3.1 基础保障金字塔

```
                    ┌─────────────┐
                    │  高端医疗   │  ← 预算充足时配置
                    │  年金/寿险  │
                    └──────┬──────┘
           ┌───────────────┼───────────────┐
           │         重疾险 + 医疗险        │  ← 核心保障
           │      （建议预算占比60%）       │
           └───────────────┬───────────────┘
      ┌───────────────────┼───────────────────┐
      │           意外险 + 定期寿险            │  ← 基础保障
      │          （建议预算占比30%）           │
      └───────────────────┬───────────────────┘
 ┌────────────────────────┼────────────────────────┐
 │                    社保/医保                     │  ← 最底层
 └──────────────────────────────────────────────────┘
```

### 3.2 不同人生阶段配置重点

**单身期（22-30岁）：**
- 重点：意外险 + 医疗险 + 定期重疾
- 预算：年收入的5-8%
- 特点：保费便宜，建立基础保障

**家庭形成期（30-40岁）：**
- 重点：重疾险 + 寿险 + 医疗险
- 预算：年收入的10-15%
- 特点：保额要充足，覆盖家庭责任

**家庭成熟期（40-50岁）：**
- 重点：重疾险 + 养老规划
- 预算：年收入的15-20%
- 特点：趁健康锁定保障，开始养老储备

**退休期（50岁+）：**
- 重点：医疗险 + 意外险 + 年金
- 预算：根据资产情况
- 特点：关注医疗和养老现金流

### 3.3 预算分配建议

| 保障类型 | 预算占比 | 说明 |
|----------|----------|------|
| 重疾险 | 50-60% | 核心保障，优先足额 |
| 医疗险 | 15-20% | 百万医疗为主 |
| 意外险 | 5-10% | 低保费高杠杆 |
| 定期寿险 | 10-15% | 经济支柱必备 |
| 储蓄/年金 | 10-20% | 预算充足时配置 |

## 四、方案呈现话术模板

### 4.1 开场白
"根据刚才了解的情况，我为您梳理了一份保障方案。这份方案的核心思路是：先解决您最担心的[疾病/意外]风险，在预算范围内做到保障最大化。"

### 4.2 需求回顾
"通过刚才的沟通，我了解到：
- 您目前最担心的是[疾病/意外]风险
- 您的预算是每年[X]万元左右
- 您[有/没有]社保，[有/没有]其他商业保险
- 您的健康状况[良好/有一些异常]"

### 4.3 方案讲解
"基于以上情况，我建议您配置以下保障：
1. **重疾险**：[X]万保额，覆盖[重疾/轻症/中症]，年缴约[X]元
   - 为什么需要：解决大病期间收入损失和康复费用
   
2. **医疗险**：[X]万保额，覆盖住院/特殊门诊，年缴约[X]元
   - 为什么需要：补充社保，覆盖大额医疗费用
   
3. **意外险**：[X]万保额，年缴约[X]元
   - 为什么需要：低保费高杠杆，覆盖意外风险"

### 4.4 异议处理

**"太贵了"**
"我理解您的顾虑。我们可以这样调整：
- 方案A：降低保额，先有保障
- 方案B：缩短保障期限，降低保费
- 方案C：选择消费型产品，提高杠杆
您看哪种方式更适合您？"

**"我再考虑考虑"**
"没问题，买保险确实需要慎重。不过我想了解一下，您主要顾虑的是哪方面？
- 是产品不合适？
- 是预算问题？
- 还是对健康告知有担心？
了解清楚后我可以帮您再调整方案。"

**"网上更便宜"**
"网上产品确实价格有优势，但保险不只是买产品，还包括：
- 投保时的健康告知把关
- 理赔时的协助服务
- 长期的保单管理和检视
我的价值在于帮您在投保时做好风控，理赔时少走弯路。"

## 五、合规注意事项

### 5.1 必须做到的
- [ ] 如实告知义务：明确说明健康告知的重要性
- [ ] 免责条款说明：逐条解释免责条款
- [ ] 犹豫期说明：告知犹豫期权利和退保损失
- [ ] 双录要求：按规定进行录音录像
- [ ] 投保提示书：让客户签署投保提示书

### 5.2 禁止行为
- [ ] 夸大收益或保障范围
- [ ] 承诺保本或保证理赔
- [ ] 代客户填写健康告知
- [ ] 返佣或变相返佣
- [ ] 诋毁同业或其他产品

## 六、后续服务SOP

### 6.1 投保后
- [ ] 协助客户完成投保流程
- [ ] 跟进核保结果
- [ ] 递送保单并讲解条款
- [ ] 建立客户档案

### 6.2 定期服务
- [ ] 每年生日前保单检视
- [ ] 家庭结构变化时方案调整
- [ ] 产品升级时告知
- [ ] 理赔协助服务

### 6.3 客户档案内容
- 基本信息表
- 健康告知记录
- 保障缺口分析
- 方案建议书
- 保单汇总表
