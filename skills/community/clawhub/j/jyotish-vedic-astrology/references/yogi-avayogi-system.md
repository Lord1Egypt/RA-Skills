## 1. 概念总览

Yogi 系统是一套基于**上升点（Lagna）** 计算的特殊行星指派体系，用于判断一个人命盘中的：

| 行星角色 | 吉凶 | 核心功能 |
|---------|------|---------|
| **Yogi** | 最吉祥 | 财富/好运的核心来源；接近神灵的方式 |
| **Duplicate Yogi** | 次吉祥 | 辅助 Yogi，强化正面效果 |
| **Ava Yogi** | 最凶 | 障碍、损耗、困难的来源 |

**关键原则**：Yogi/Ava Yogi 不是天生的吉凶星（天然益星/天然凶星），而是基于命盘动态计算的——同一颗行星对某人可能是 Yogi（极吉），对另一人可能是 Ava Yogi（极凶）。

## 2. 计算方法

### Step 1：计算 Yogi 点

```
Yogi 点 = 上升点经度 + 93°20'（即上升点 + 第6个Nakshatra的起始度数）
```

> 93°20' = 6 × 13°20'，即从上升点所在Nakshatra往前数6个Nakshatra的起始位置。

### Step 2：确定三个行星

1. **Yogi** = Yogi 点所落入 Nakshatra 的主星
2. **Duplicate Yogi** = Yogi 点所落入 Rashi（星座）的主星
3. **Ava Yogi** = 从 Yogi 所在 Nakshatra 往后数 **第6个** Nakshatra 的主星

> 注：如果 Yogi 和 Duplicate Yogi 是同一颗行星，则 Duplicate Yogi 改取 Yogi 点所在星座的另一主星（若有）或遵循其他争议规则。

### 计算示例

上升点在 Ashwini（Aries 0°-13°20'）：
- Yogi 点 = 0° + 93°20' = Gemini 6°40'（落在 Mrigashira Nakshatra 的最后一段）
- Yogi = Mrigashira 的主星 = **Mars**
- Duplicate Yogi = Gemini 的主星 = **Mercury**
- Ava Yogi = 从 Mrigashira 数6个 Nakshatra → Purva Phalguni → 主星 = **Venus**

## 3. 四大影响因子

### 3.1 财富/好运领域（Wealth Field）

Yogi 行星所主宰的宫位和所落入的宫位，指示财富来源。例如：
- Yogi = Jupiter 落入 2 宫 → 财富来自教育、演讲、出版
- Yogi = Venus 落入 7 宫 → 财富来自伴侣、合作、艺术
- Yogi = Mars 落入 10 宫 → 财富来自竞争性行业、军事、工程

### 3.2 与神灵的距离（Spiritual Distance）

Yogi 行星还决定一个人接近神灵/灵性的方式：
- Yogi = Sun → 通过王道、正法（Dharma）接近神灵
- Yogi = Moon → 通过慈悲、母性、情感接近
- Yogi = Mars → 通过勇气、保护、战斗接近
- Yogi = Mercury → 通过知识、学习、沟通接近
- Yogi = Jupiter → 通过智慧、教导、传统接近
- Yogi = Venus → 通过美、艺术、奉献接近
- Yogi = Saturn → 通过苦行、忍耐、纪律接近
- Yogi = Rahu → 通过非传统途径、异端、探索接近

### 3.3 社交圈类型（Sambandha Type）

Yogi 行星的天然属性决定你被什么样的人吸引：
- Sun → 权威人物、政府官员、领导者
- Moon → 母性人物、照顾者、公众人物
- Mars → 军人、运动员、竞争者
- Mercury → 学者、商人、作家
- Jupiter → 老师、导师、宗教人士
- Venus → 艺术家、音乐家、伴侣
- Saturn → 老年人、劳动者、纪律者
- Rahu → 革新者、外国人、非主流人士

### 3.4 支持者类型（Support Network）

与 Yogi 有 Sambandha（联系）的行星代表支持来源。联系类型：
- **合相**（Conjunction）→ 直接支持
- **相位**（Aspect）→ 间接但持续支持
- **交换**（Parivartana）→ 互相支持
- **Nakshatra 联系** → 深层业力支持

Ava Yogi 的支持者则代表**阻碍者类型**——应避免或谨慎对待的人群。

## 4. 特殊判定规则

### 4.1 上升点与 Yogi 点同 Nakshatra

如果 Yogi 点落入与上升点相同的 Nakshatra，此人具有**天生富裕业力**（born with affluent karma）—— 即使其他指标不佳，也能在关键时刻获得财富机遇。

### 4.2 Yogi 行星的状态评估

Yogi 行星的尊严状态直接影响财富质量：
- **入庙/耀升** → 财富丰厚，来路正当
- **落陷**（无 Neecha Bhanga）→ 财富困难，需要更多努力
- **落陷但有 Neecha Bhanga** → 早年辛苦，后期逆转
- **燃烧**（Combust）→ 财富被父亲/权威人物压制
- **逆行**（Retrograde）→ 财富延迟但更持久

### 4.3 Ava Yogi 识别法则

Ava Yogi 行星不仅影响自身，还影响与其有联系的行星：
- Ava Yogi 合相某行星 → 该行星领域也受负面影响
- Ava Yogi 相位某宫 → 该宫事务受阻
- Dasha 触发 Ava Yogi → 整个大运期间困难

## 5. Dasha 激活机制

### 5.1 Yogi Dasha（吉运）

当 Vimshottari Dasha 或 Antar Dasha 的主星是 Yogi 时：
- 该期间**财富机遇显著增加**
- 适合启动新项目、投资、创业
- 与 Yogi 行星有 Sambandha 的行星的 Antar 也激活正面效果

### 5.2 Ava Yogi Dasha（凶运）

当 Dasha 主星是 Ava Yogi 时：
- 该期间**障碍增多，需保守策略**
- 不宜重大投资或风险决策
- 如果同时有过境凶星激活，困难加倍

### 5.3 混合情况

Dasha 主星是 Yogi，Antar 是 Ava Yogi（或反之）→ 该 Antar 期内有矛盾体验：大方向利好但有具体障碍。

## 6. Transit（过境）激活

### 6.1 Yogi 点的过境触发

- 木星过境 Yogi 点所在星座/Nakshatra → 财富机遇期
- 土星过境 Yogi 点 → 财富领域考验期（考验通过后有更大收获）
- Rahu 过境 Yogi 点 → 非传统财富机遇（可能突然、意外）
- 太阳每年过境 Yogi 点所在星座 → 年度财富激活窗口

### 6.2 Ava Yogi 点的过境

- 凶星过境 Ava Yogi 点 → 障碍加剧
- 吉星过境 Ava Yogi 点 → 凶性被部分化解

## 7. 宝石激活疗法

### 7.1 Yogi 宝石（推荐佩戴）

| Yogi 行星 | 推荐宝石 | 替代宝石 |
|----------|---------|---------|
| Sun | 红宝石(Ruby) | 黄色拓帕石 |

| Mars | 红珊瑚(Red Coral) | 红玉髓 |
| Mercury | 祖母绿(Emerald) | 橄榄石 |
| Jupiter | 黄宝石(Yellow Sapphire) | 黄色拓帕石 |
| Venus | 钻石(Diamond) | 白宝石 |
| Saturn | 蓝宝石(Blue Sapphire) | 青金石 |
| Rahu | 蜜蜡(Gomed) | 锆石 |

### 7.2 Ava Yogi 宝石（应避免）

**严禁佩戴** Ava Yogi 行星对应的宝石——会放大障碍和困难。

### 7.3 宝石佩戴规则

1. Yogi 宝石佩戴在**合适的指头**上（通常为右手）
2. 金属框架应与 Yogi 行星对应（Sun/Mars/Jupiter→金，Moon/Venus→银，Mercury/Saturn→银/合金）
3. 宝石重量有最低要求（通常 3-5 克拉以上）
4. 需在 Yogi 行星的日子/时辰首次佩戴

## 11. 注意事项

1. **Rahu/Ketu 特殊性**：Rahu 可以作为 Yogi 或 Ava Yogi，但 Ketu **不参与**此系统（Ketu 没有 Nakshatra 主星身份）
2. **争议点**：不同学派对 Duplicate Yogi 的计算有分歧，部分学者认为应取 Yogi 点所在星座的主星，无论是否与 Yogi 重叠
3. **与 Tithi Lord 的区别**：Tithi Lord 基于出生月亮的 Tithi，Yogi 基于上升点——两者独立计算，可交叉验证
4. **时效性**：Yogi/Ava Yogi 是本命盘固定指标，但其激活时机取决于 Dasha 和 Transit
