# drug-interaction-risk-assessment

## 触发条件
当患者同时或先后使用两种及以上具有明确药理活性的药物时，必须执行本评估流程。

## 执行步骤

### 1. 判断是否适用
- 排除完全无药理活性的安慰剂组合
- 排除无重叠药代或药效通路的药物组合

### 2. 分析药效动力学相互作用（不改变血药浓度但改变效应）
- **协同作用**：判断是否存在效应增强（如氟烷增强β受体敏感性）
- **拮抗作用**：判断是否存在效应减弱

### 3. 分析药代动力学相互作用（改变作用部位药物浓度）
- **吸收**：评估是否因胃排空延迟等影响吸收速度
- **分布**：检查高蛋白结合率药物是否竞争结合位点，导致游离药物浓度升高
- **代谢**：识别是否存在酶抑制或诱导（如吉非贝齐抑制紫杉醇代谢）
- **排泄**：判断是否竞争肾小管分泌通路（如丙磺舒抑制青霉素排泄）

### 4. 识别高风险药物
重点关注以下治疗指数低或药效曲线陡峭的药物类别：
- 强心药
- 抗凝药
- 抗心律失常药
- 抗癫痫药
- 抗躁狂药
- 抗肿瘤药
- 免疫抑制药

### 5. 输出结果
发出药物相互作用预警，并建议：
- 调整剂量
- 更换药物
- 加强治疗药物监测或不良反应观察
