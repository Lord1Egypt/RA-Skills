# 免疫病理反应的分类与成因分析

## 何时使用
- 患者存在免疫功能异常（过度激活或功能低下）的临床证据
- 需区分免疫介导与非免疫介导的病理过程
- 为制定免疫调节治疗策略（抑制或增强）提供依据

## 执行步骤

1. **确认免疫病理反应的存在**
   - 排除非免疫因素（如外伤、毒素、代谢紊乱）
   - 确认症状与免疫系统功能障碍相关

2. **判断免疫功能状态**
   - **免疫功能过度增强**：表现为攻击自身组织、无害抗原或移植器官
   - **免疫功能低下**：表现为反复感染、肿瘤发生或免疫监视失效

3. **按功能状态分类疾病类型**
   - **免疫亢进相关**：
     - 超敏反应（I–IV型）
     - 自身免疫性疾病（如系统性红斑狼疮、类风湿关节炎）
     - 免疫增殖性疾病
     - 移植排斥反应
   - **免疫低下相关**：
     - 原发性或继发性免疫缺陷（如AIDS）
     - 肿瘤（因免疫监视功能减弱）

4. **评估临床后果严重程度**
   - 轻度：慢性炎症、局部组织损伤
   - 重度：多器官功能障碍或衰竭

5. **提出初步治疗方向**
   - 免疫亢进 → 考虑免疫抑制治疗（如糖皮质激素、钙调磷酸酶抑制剂）
   - 免疫低下 → 考虑免疫增强治疗（如干扰素、白细胞介素-2）

> 注意：本技能仅用于指导初步分类与治疗方向判断，具体用药需结合实验室检查与临床指南。
