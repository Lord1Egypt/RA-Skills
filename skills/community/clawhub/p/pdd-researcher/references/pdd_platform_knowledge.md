# 拼多多平台知识库

## 平台特征

拼多多是中国第二大电商平台，以"社交+低价"模式崛起，核心特征：
- **用户画像**：下沉市场为主，价格敏感度高，女性用户占比约60%
- **流量机制**：搜索流量+推荐流量+社交裂变（拼团/砍一刀）
- **核心指标**："已拼数"替代传统销量显示，强调拼团完成量
- **百亿补贴**：平台补贴品牌商品，建立正品心智
- **多多买菜**：社区团购业务，高频带低频

## 拼多多特有数据指标

| 指标 | 说明 | 分析价值 |
|------|------|---------|
| 已拼N件 | 累计拼团完成量 | 评估商品真实热度 |
| 关注人数 | 店铺/商品收藏数 | 衡量用户兴趣度 |
| 券后价 | 领券后的实际到手价 | 真实价格锚点 |
| 百亿补贴标签 | 平台补贴认证 | 正品+低价双信号 |
| 品牌黑标 | 品牌官方认证 | 区分白牌/品牌 |
| DSR评分 | 描述/服务/物流三维 | 店铺综合服务水平 |
| 退货包运费 | 平台退货保障 | 降低用户决策门槛 |

## 搜索排名因素

拼多多搜索排名核心影响因素（推测）：
1. 销量/拼单量（近期权重高）
2. 价格竞争力（比同行低加分）
3. DSR评分（描述相符/服务态度/物流速度）
4. 转化率（点击→下单→支付）
5. 商品标题关键词匹配度
6. 店铺等级与活跃度
7. 是否参与平台活动（百亿补贴/限时秒杀/9块9特卖）
8. 退货率/纠纷率（负向指标）

## 营销活动体系

- **百亿补贴**：平台补贴品牌正品，核心流量入口
- **限时秒杀**：超低价限时限量，引流利器
- **9块9特卖**：低价专区，适合清仓/引流
- **断码清仓**：品牌尾货处理
- **多多果园**：游戏化留存工具
- **砍价免费拿**：社交裂变获客
- **领现金/天天领现金**：拉新激励

## 价格带规律

拼多多商品价格带分布（2025年估算）：
- 0-9.9元：超低价引流款，占比约25%
- 10-49元：主力价格带，占比约40%
- 50-99元：品质升级带，占比约20%
- 100-299元：品牌/大件带，占比约10%
- 300元以上：高客单带，占比约5%

## 类目特征

### 高竞争类目
- 女装/内衣、家居日用、食品保健、母婴用品
- 特征：SKU海量、价格战严重、同质化高

### 新兴机会类目
- 数码配件、宠物用品、个护美妆、运动户外
- 特征：增速快、品牌化空间大、消费升级

### 平台扶持类目
- 农产品/生鲜（乡村振兴）、工厂直销（C2M）、国货品牌
- 特征：流量倾斜、政策支持

## 数据获取方式

### 方法1：WebSearch 公开信息
- 搜索 `{产品名} 拼多多 热销 多少钱`
- 搜索 `{产品名} 拼多多 评价 口碑`
- 搜索 `{类目} 拼多多 销量排行`
- 优势：无需任何权限，覆盖面广
- 局限：数据精度有限，需二次验证

### 方法2：WebFetch 抓取页面
- 目标：拼多多商品详情页/搜索页
- 注意：拼多多反爬严格，WebFetch 成功率有限
- 备选：使用移动端 H5 页面（yangkeduo.com）抓取

### 方法3：第三方数据工具
- 电霸（dianba6.com）：拼多多专用数据分析
- 多多参谋：多维度数据监控
- 多多情报通：类目/竞品/关键词分析
- 注意：第三方工具数据可能有时延，以实际页面为准

### 数据优先级
1. 优先使用 WebSearch 快速获取结构化信息
2. 尝试 WebFetch 抓取详情页补充数据
3. 引用第三方工具的公开报告/分析文章
4. 标注所有数据的来源和时效性

## 竞品分析框架

### 竞品识别
- 直接竞品：同品类、同价格带、同目标人群
- 间接竞品：替代品类、相邻价格带
- 标杆竞品：类目TOP10，学习对象

### 竞品分析维度
1. **商品力**：标题/主图/详情页质量
2. **价格力**：定价策略/优惠券/活动参与
3. **流量力**：搜索排名/推荐曝光/广告投放
4. **转化力**：评价质量/DSR评分/售后服务
5. **品牌力**：品牌认知/复购率/粉丝数

### 竞品监控指标
- 日销量/周销量变化
- 价格变动频率
- SKU增减策略
- 活动参与频次
- 评价增长趋势
