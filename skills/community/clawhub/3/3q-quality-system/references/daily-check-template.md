# 3Q 日检模板 — 每日 5 分钟质量自查

> 每天早上或下班前跑一次。让你的 AI 用这个模板对你今天的工作做一次 3Q 检查。

---

## 第一步：预检（30 秒）

开始今天的工作前，对今天的计划任务跑三问：

```
Q1: 今天最可能出什么错？
Q2: 如果出错，影响什么？
Q3: 怎么在执行中发现它出错？
```

**输出格式**：
```
📋 今日预检：[任务名]
Q1: {预判风险}
Q2: {影响范围}
Q3: {检测手段}
判定：✅ 可执行 / ❌ 先补信息
```

---

## 第二步：快速 3Q（1 分钟）

对今天已完成的工作跑快速三问：

```
① 做全了吗？ → 检查步骤完整性
② 做对了吗？ → 检查数据准确性
③ 有反常的吗？ → 检查异常点
```

**输出格式**：
```
📋 快速 3Q：[产出名]
完整性：✅ / ❌ {缺失项}
准确性：✅ / ⚠️ {偏差}
异常：✅ / ❌ {异常项}
判定：✅ 通过 / ⚠️ 升级深度 3Q
```

---

## 第三步：深度 3Q（3 分钟）

对今天**最重要的一份产出**做完整七维评分。

| 维度 | 评分（1-10） | 说明 |
|------|-------------|------|
| 元认知 Q | /10 | 任务理解是否准确？ |
| 标杆 Q | /10 | 是否对标了领域最佳实践？ |
| 挑战 Q | /10 | 是否质疑了自己的假设？ |
| 逻辑 Q | /10 | 论证是否自洽？有无矛盾？ |
| 用户 Q | /10 | 用户价值是否清晰？ |
| 竞争 Q | /10 | 差异化是否明确？ |
| 效果验证 | /10 | 是否实测对比了 with/without？ |

**总分**: /70
**判定**: ✅ ≥ 70 通过 / ❌ < 70 驳回

---

## 第四步：自修正（30 秒）

回顾今天的 3Q 检查：

```
同类问题重复出现 ≥ 3 次？ → 是 → 建议 AI 添加一条新规则
今天 3Q 发现了什么之前没注意到的盲点？ → 记录
有什么可以固化到日常流程中的？ → 记录
```

---

## 完整示例

**场景**：今天写了一篇市场分析文章

```
📋 今日预检：市场分析文章
Q1: 数据过时 + 风格不对（too salesy）
Q2: 读者觉得信息不准确
Q3: 交叉验证数据来源 + 发前让 AI 审风格
判定：✅ 可执行

📋 快速 3Q：市场分析 draft
完整性：✅ 
准确性：⚠️ 一个数据用了 3 天前的版本
异常：✅ 
判定：⚠️ 升级深度 3Q

📋 深度 3Q：市场分析 v2
元认知 Q：8/10 — 理解正确，但深挖不够
标杆 Q：6/10 — 没对标同类文章写法
挑战 Q：7/10 — 质疑了"看多"假设
逻辑 Q：9/10 — 自洽
用户 Q：8/10 — 有个段落偏 technical
竞争 Q：7/10 — 差异化不突出
效果验证：7/10 — A/B 对比显示 v2 比 v1 好但不够

总分：52/70 ❌ 驳回 → 修改后重审

📋 自修正
重复问题：数据时效性 — 第 3 次犯了
新盲点：差异化不够 → 下次写作先做竞争Q
```
