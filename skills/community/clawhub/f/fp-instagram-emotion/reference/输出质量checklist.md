# 输出质量 Checklist

## Meme 类（核心：梗到不到位）
- [ ] 梗戳的是机械师独有的痛/骄傲，不是普遍情绪
- [ ] 图片文字上下逻辑清楚，套对了模板
- [ ] Caption 首句是钩子
- [ ] 有互动钩子（Tag the guy who… / Drop a 🔧…）
- [ ] emoji 1-2 个，不堆砌
- [ ] Hashtag 8 个（品牌+模板标签+垂类）

## 梗强弱快速判断
- 强：能让机械师秒懂、想 tag 同行的具体场景
  （设备专挑工期坏、OEM报价一堆零、型号买错白跑）
- 弱：泛泛的情绪（被新机器吸引、周一不想上班这种谁都懂的）
- 如果一条 Meme 换成别的行业也成立 → 说明不够"机械圈"，重做

## 展会类
- [ ] 80 字内，现场感强
- [ ] @了展会官号
- [ ] 5 个 hashtag

## KOL Reel 类
- [ ] 开头是强钩子
- [ ] 中间有 KOL 故事
- [ ] 结尾 FP 植入 + Link in bio

## 微调提示
- 梗想更糙/更克制 → 改 SKILL.md 的 [梗的尺度]
- 有禁忌话题 → 填 [禁用梗]
- 模板别重复 → 填 [已用过的Meme]
