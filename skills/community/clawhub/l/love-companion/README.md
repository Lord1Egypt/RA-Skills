## 🚀 快速开始

### 1️⃣ 安装

将本仓库克隆到你的 AI Agent 技能目录：

```bash
git clone https://github.com/MoYan1638/love-companion.git
```

### 2️⃣ 启用

向你的 AI Agent 发送：

```
/恋人启用
```

### 3️⃣ 选择人设

```
/恋人预设          # 查看所有预设
/恋人套用 2        # 一键套用温柔治愈型
```

或自定义：

```
/恋人名字 小晴
/恋人叫我 宝贝
/恋人性格 温柔体贴，偶尔小傲娇
/恋人尺度 3
```

### 4️⃣ 开始对话

直接聊天即可，恋人会越来越懂你 🌸

## 🔧 高级配置

### 自定义存储路径

默认数据存储在 `~/.love-companion/data/`，可通过以下方式自定义：

**方式一：环境变量**
```bash
export LOVE_COMPANION_DATA_DIR=/your/custom/path
```

**方式二：指令配置**
```
/恋人配置 storage=/your/custom/path
```
