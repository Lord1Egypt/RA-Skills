# Blood Stasis Syndrome Identification

## When to Use
当患者出现以下任一表现时启动本技能：
- 固定不移的刺痛，夜间加重
- 体表或体内固定肿块
- 出血色紫暗或夹有血块
- 面色黧黑、唇甲青紫、肌肤甲错
- 舌质紫暗、有瘀点瘀斑或舌下脉络曲张
- 脉象细涩、沉弦或结代

且已知存在气虚、气滞、血寒、血热、外伤或久病等诱因，同时排除急性出血未凝固阶段。

## Execution Steps

1. **确认形成原因**：
   - 判断是否由气虚（推动无力）、气滞（运行受阻）、血寒（凝滞）、血热（煎熬成块）、外伤（离经之血）或久病入络所致。

2. **定位瘀血部位**：
   - 根据症状定位瘀阻脏腑或组织：
     - 心：心悸、胸痛、口唇指甲青紫
     - 肺：胸痛、咯血
     - 胃肠：呕血或黑便如漆
     - 肝：胁痛、痞块
     - 胞宫：小腹疼痛、月经不调、痛经、闭经或崩漏
     - 肢体末端：脱疽
     - 肌肤局部：青紫肿痛

3. **综合辨识特征**：
   - 疼痛：刺痛、固定、拒按、夜重
   - 肿块：外伤青紫或内积痞硬、固定不移
   - 出血：紫暗或带血块
   - 望诊：面色黧黑、肌肤甲错、唇甲青紫、舌紫或有瘀斑
   - 脉象：细涩、沉弦或结代

4. **输出结论**：
   - 综合上述信息，确认是否为瘀血证，并明确病位与可能的病因类型。

> 注意：本技能不适用于急性出血尚未凝固的阶段。
