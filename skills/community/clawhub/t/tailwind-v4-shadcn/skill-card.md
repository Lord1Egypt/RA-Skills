## Description: <br>
Configures Tailwind v4 with shadcn/ui using a required CSS variable theming pattern for automatic dark mode and fewer setup errors. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wpank](https://clawhub.ai/user/wpank) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers use this skill when adding Tailwind CSS v4 and shadcn/ui to React/Vite projects, migrating from Tailwind v3, or debugging theme, dark mode, and build issues. It provides setup steps, templates, migration guidance, and corrections for common v3-to-v4 mistakes. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Setup commands and templates can change a frontend project's dependencies, configuration, and source files. <br>
Mitigation: Run commands only in the intended project, review package-manager and shadcn commands before execution, and inspect generated changes before committing. <br>
Risk: Migration steps may remove or replace existing Tailwind configuration. <br>
Mitigation: Back up or review existing Tailwind, PostCSS, and shadcn configuration before applying delete or replacement steps. <br>
