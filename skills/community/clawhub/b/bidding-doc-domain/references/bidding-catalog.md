# 招投标文档编制任务清单与依赖拓扑

**域间逻辑流**：B1 招标解析 → B2 商务标编制 → B3 技术标编制 → B4 报价标编制 → B5 废标风控 → B6 全局校验

---

## B1 招标解析

| ID | 任务类型 | 说明 | 依赖 | UTOS映射提示 |
|----|---------|------|------|-------------|
| B1-01 | 招标文件解析 | 提取项目名称、编号、预算、工期、资质要求等关键信息 | 无（入口） | S→C |
| B1-02 | 评分项提取 | 提取评分标准、分值分配、加分项、扣分项 | B1-01 | S→C |
| B1-03 | 禁止条款识别 | 识别废标条款、否决条款、限制性条款 | B1-01 | S→C→G |
| B1-04 | 响应要求梳理 | 梳理必须响应的技术/商务/报价要求清单 | B1-01, B1-02 | C |

## 依赖拓扑摘要

**主链路**: B1-01 → B1-02/B1-03/B1-04(并行) → B2-01/B3-01(并行) → B3-02 → B3-03 → B4-01 → B4-02 → B4-03 → B5-01 → B5-02 → B5-03 → B6-01 → B6-02 → B6-03

**商务标链**: B1-01 → B2-01 → B2-02 → B2-03 → B2-04

**技术标链**: B1-01 → B1-04 → B3-01 → B3-02 → B3-03 → B3-04 → B3-05

**报价标链**: B1-01 → B1-04 → B4-01 → B4-02 → B4-03

**风控链**: B5-01 → B5-02 → B5-03 → B6-01 → B6-02 → B6-03

更多组合由UTOS根据具体任务动态推导。
