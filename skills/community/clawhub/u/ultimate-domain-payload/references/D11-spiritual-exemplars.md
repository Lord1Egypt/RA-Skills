## 范本1：中年意义危机应对

### 题目
45岁企业高管的中年意义危机应对全流程

### 背景
李先生，45岁，某科技公司副总裁，事业成功但感到内心空虚。最近完成一个重要项目后，突然感到人生失去方向。开始质疑工作的意义，对家庭生活也感到不满。睡眠质量下降，经常感到疲惫。希望通过精神探索重新找到人生意义。

### Step 1: 价值观探索阶段

#### 1.1 核心价值观识别 (SP1-01)
**输入**: 自我反思引导问题，价值观清单
**执行**:
1. 完成价值观清单初选：从50个价值观中选出10个重要价值观
2. 反思人生关键时刻：回忆大学选择专业、第一份工作、结婚、升职等关键时刻
3. 评估价值观对行为的影响：分析每个价值观如何影响过去决策
4. 识别核心价值观：创新、家庭、成就、自由、成长
5. 验证价值观的真实性：通过行为一致性检验

**输出**: 核心价值观列表（创新、家庭、成就、自由、成长）

#### 1.2 信念系统梳理 (SP1-02)
**输入**: 核心价值观列表，信念探索问题
**执行**:
1. 从核心价值观推导深层信念：
   - 创新 → "我必须不断突破才能证明价值"
   - 家庭 → "我必须为家人提供最好的生活"
   - 成就 → "成功等于职位和收入"
2. 识别信念来源：父亲是企业家，从小被灌输成功观念
3. 分类信念：关于自我（必须成功）、关于他人（依赖认可）、关于世界（竞争激烈）
4. 评估信念影响：正面（驱动努力）、负面（压力过大）
5. 标记需要调整的信念："成功等于职位和收入"

**输出**: 信念系统分析报告

#### 1.3 价值观排序 (SP1-03)
**输入**: 核心价值观列表，排序标准
**执行**:
1. 评估每个价值观的重要性：创新9分，家庭8分，成就7分，自由6分，成长8分
2. 分析价值观关系：创新和成长高度相关，家庭和自由有冲突
3. 识别潜在冲突：追求创新需要时间，影响家庭
4. 进行优先级排序：家庭 > 创新 = 成长 > 自由 > 成就
5. 验证排序合理性：回忆最近决策是否符合排序

**输出**: 价值观优先级排序

**校准点1**: 价值观排序确认
- 确认排序反映真实偏好
- 确认识别出价值观冲突
- 确认需要调整的信念

### Step 2: 意义建构阶段

#### 2.1 人生意义探索 (SP2-01)
**输入**: 价值观排序，意义探索引导问题
**执行**:
1. 反思人生各领域满意度：工作6分，家庭5分，健康4分，社交3分，精神2分
2. 识别已有意义来源：工作成就、家庭责任
3. 探索潜在意义领域：精神成长、社区贡献、个人创造
4. 定义个人生命意义："通过创新解决问题，同时为家人创造幸福生活"
5. 验证意义真实性：回忆过去感到充实的时刻

**输出**: 个人生命意义定义

#### 2.2 叙事身份建构 (SP2-02)
**输入**: 人生关键事件清单，叙事结构框架
**执行**:
1. 收集人生关键事件：童年贫困、大学奋斗、创业失败、职业成功、中年危机
2. 识别事件联系：从贫困到成功的奋斗主题
3. 提炼身份主题："奋斗者"、"保护者"
4. 建构连贯叙事：从贫困童年到成功高管，但内心始终渴望更多
5. 评估叙事对身份认同的影响：叙事强化了"必须成功"的信念

**输出**: 人生叙事初稿

#### 2.3 意义危机应对 (SP2-06)
**输入**: 意义危机症状，应对策略库
**执行**:
1. 识别意义危机症状：工作失去动力、家庭关系疏远、睡眠问题、存在空虚感
2. 评估危机严重程度：中度危机，尚未达到临床抑郁
3. 选择应对策略：
   - 短期：减少工作压力，增加家庭时间
   - 中期：探索精神修行，寻找新意义来源
   - 长期：重构人生叙事，调整价值观排序
4. 实施危机干预：请假一周，与家人旅行
5. 重建意义基础：开始冥想练习，阅读哲学书籍

**输出**: 意义危机应对计划

**校准点2**: 意义危机评估
- 确认危机严重程度评估准确
- 确认应对策略全面可行
- 确认有专业支持资源

### Step 3: 精神修行阶段

#### 3.1 冥想入门 (SP3-01)
**输入**: 冥想基础理论，入门技术指导
**执行**:
1. 了解冥想基本概念：专注、觉察、放下
2. 选择冥想类型：正念冥想（适合初学者）
3. 学习基础技术：呼吸观察、身体扫描
4. 开始短时间练习：每天5分钟
5. 逐步延长练习时间：每周增加5分钟，达到每天20分钟

**输出**: 冥想练习计划

#### 3.2 精神阅读 (SP3-06)
**输入**: 精神成长书单，阅读方法指导
**执行**:
1. 选择书籍：《活出生命的意义》（维克多·弗兰克尔）、《当下的力量》（埃克哈特·托利）
2. 制定阅读计划：每天30分钟，周末1小时
3. 采用反思式阅读：每章结束后写阅读感悟
4. 记录阅读感悟：创建阅读笔记
5. 将洞见应用到生活中：实践"活出意义"的理念

**输出**: 精神阅读计划

#### 3.3 修行习惯养成 (SP3-07)
**输入**: 修行习惯清单，习惯养成策略
**执行**:
1. 确定修行习惯：每日冥想、每周精神阅读、每月静修
2. 分解习惯为微小步骤：冥想从5分钟开始
3. 设计习惯触发机制：早晨起床后冥想，晚上睡前阅读
4. 开始实践并追踪进度：使用习惯追踪APP
5. 根据反馈调整策略：发现晚上阅读效果不好，改为早晨

**输出**: 修行习惯养成计划

**校准点3**: 精神修行启动
- 确认冥想练习正常进行
- 确认阅读计划可行
- 确认习惯养成策略有效

### Step 4: 存在困境应对

#### 4.1 死亡觉察 (SP4-01)
**输入**: 死亡教育材料，觉察练习指导
**执行**:
1. 了解死亡觉察意义：通过觉察死亡，珍惜生命
2. 学习死亡教育知识：阅读相关书籍，观看纪录片
3. 进行死亡觉察练习：写自己的墓志铭、想象最后一天
4. 处理相关情绪反应：允许恐惧和悲伤，与家人分享
5. 整合到生命态度中：更珍惜家庭时间，减少无意义工作

**输出**: 死亡觉察实践记录

#### 4.2 自由与责任 (SP4-03)
**输入**: 自由概念框架，责任伦理指导
**执行**:
1. 理解自由哲学含义：存在主义自由观
2. 学习责任伦理原则：对自我、家庭、社会的责任
3. 分析自由与责任关系：自由选择带来责任
4. 在决策中实践平衡：选择减少工作，承担更多家庭责任
5. 发展负责任的自由态度：在自由选择中承担责任

**输出**: 自由与责任平衡指南

### Step 5: 精神遗产规划

#### 5.1 人生回顾 (SP5-01)
**输入**: 人生阶段划分框架，回顾引导问题
**执行**:
1. 划分人生阶段：童年、青年、成年早期、中年
2. 回顾关键事件：贫困童年、奋斗青年、成功中年
3. 识别成长点：从物质追求到精神追求
4. 提炼人生主题：从生存到意义
5. 记录回顾感悟：写人生回顾日记

**输出**: 人生回顾报告

#### 5.2 智慧提炼 (SP5-02)
**输入**: 人生经历素材，智慧提炼框架
**执行**:
1. 回顾人生重要经历：创业失败、职业成功、中年危机
2. 识别经历教训：失败是成长机会，成功不等于幸福
3. 提炼普遍性智慧："真正的成功是内心的平静"
4. 选择表达方式：写给儿子的信
5. 创作智慧作品：《致儿子的信：关于成功与幸福》

**输出**: 智慧作品

**校准点4**: 精神遗产完成
- 确认人生回顾全面客观
- 确认智慧提炼真实深刻
- 确认精神遗产有价值

## 范本2：丧亲后的精神重建

### 题目
55岁女性丧偶后的精神重建全流程

### 背景
王女士，55岁，中学教师，丈夫因癌症去世6个月。虽然时间流逝，但她仍然深陷哀伤，无法集中精力工作，对生活失去兴趣。子女已成年独立，但她感到孤独和无助。希望通过精神探索重建生活意义。

### Step 1: 哀伤处理阶段

#### 1.1 丧失与哀伤 (SP4-06)
**输入**: 哀伤阶段理论，哀伤辅导技术
**执行**:
1. 理解哀伤自然过程：否认、愤怒、讨价还价、抑郁、接受
2. 评估当前哀伤阶段：处于抑郁阶段，开始接受现实
3. 学习哀伤应对技术：情绪日记、哀伤仪式、支持小组
4. 建设支持系统：加入丧偶支持小组，定期与子女沟通
5. 逐步重建生活：制定每日活动计划，逐步恢复社交

**输出**: 哀伤应对计划

#### 1.2 苦难的意义化 (SP2-03)
**输入**: 苦难经历描述，意义化框架
**执行**:
1. 承认和接纳苦难：丈夫的死亡是不可改变的事实
2. 探索苦难影响：失去伴侣、生活改变、情感创伤
3. 寻找可能的意义：珍惜生命、帮助他人、精神成长
4. 建构意义化叙事："通过失去，我更懂得珍惜"
5. 整合到人生故事：成为有智慧的长者

**输出**: 苦难意义化叙事

**校准点1**: 哀伤处理进展
- 确认哀伤阶段评估准确
- 确认应对策略有效
- 确认意义化过程健康

### Step 2: 价值观重构阶段

#### 2.1 核心价值观识别 (SP1-01)
**输入**: 自我反思引导问题，价值观清单
**执行**:
1. 完成价值观清单：家庭、健康、服务、智慧、精神成长
2. 反思人生关键时刻：丈夫生病期间的价值观体现
3. 评估价值观影响：家庭第一，服务他人
4. 识别核心价值观：家庭、服务、智慧、精神成长
5. 验证真实性：回忆过去行为是否一致

**输出**: 新的核心价值观列表

#### 2.2 价值观演化 (SP1-05)
**输入**: 价值观历史记录，关键转折点
**执行**:
1. 回顾价值观形成：年轻时重视事业，中年重视家庭
2. 识别转折点：丈夫生病、丈夫去世
3. 分析变化原因：经历死亡，重新审视生命
4. 记录当前状态：更重视精神成长和服务他人
5. 预测未来方向：向智慧长者发展

**输出**: 价值观演化时间线

#### 2.3 价值观与行为对齐 (SP1-06)
**输入**: 核心价值观，日常行为记录
**执行**:
1. 记录典型日常：教学、家务、社交、独处
2. 分析行为动机：教学出于责任，家务出于习惯
3. 评估一致性：教学与服务价值观一致，家务与家庭价值观一致
4. 识别不一致：社交活动与孤独感冲突
5. 制定改进计划：增加有意义社交，减少无意义活动

**输出**: 价值观行为对齐分析

**校准点2**: 价值观重构完成
- 确认新价值观反映真实需求
- 确认价值观演化健康
- 确认行为调整可行

### Step 3: 意义重建阶段

#### 3.1 人生意义探索 (SP2-01)
**输入**: 价值观重构结果，意义探索引导
**执行**:

2. 识别意义来源：教学、子女、精神成长
3. 探索新意义领域：志愿服务、写作、灵性修行
4. 定义新生命意义："通过教学和服务，传递爱与智慧"
5. 验证真实性：回忆过去感到充实的时刻

**输出**: 新生命意义定义

#### 3.2 叙事身份建构 (SP2-02)
**输入**: 人生关键事件，叙事框架
**执行**:
1. 收集关键事件：成长、教育、结婚、生育、丧偶
2. 识别联系：从被爱到爱人的转变
3. 提炼身份主题："照顾者"、"传递者"
4. 建构新叙事：从幸福妻子到智慧长者
5. 评估影响：叙事给予新身份认同

**输出**: 新人生叙事

#### 3.3 意义重构 (SP2-07)
**输入**: 人生转折点，意义重构框架
**执行**:
1. 识别转折点：丈夫去世
2. 评估旧意义适用性：以家庭为中心的意义需要调整
3. 探索新意义可能：服务、创造、精神成长
4. 建构新意义框架：从家庭幸福到生命智慧
5. 整合到新阶段：成为智慧传递者

**输出**: 意义重构计划

**校准点3**: 意义重建完成
- 确认新意义真实可行
- 确认叙事身份健康
- 确认意义重构成功

### Step 4: 精神修行深化

#### 4.1 冥想入门 (SP3-01)
**输入**: 冥想基础，技术指导
**执行**:
1. 了解冥想类型：慈悲冥想（适合哀伤处理）
2. 学习基础技术：慈悲观想、自我慈悲
3. 开始练习：每天10分钟
4. 逐步延长：达到每天30分钟
5. 整合到生活：在感到悲伤时练习

**输出**: 慈悲冥想练习计划

#### 4.2 正念练习 (SP3-02)
**输入**: 正念理论，练习方法
**执行**:
1. 理解正念核心：活在当下，不评判
2. 学习技术：正念呼吸、正念行走
3. 日常练习：吃饭时正念，走路时正念
4. 培养觉察：觉察情绪而不被带走
5. 整合到生活：成为生活方式

**输出**: 正念练习指南

#### 4.3 静默练习 (SP3-05)
**输入**: 静默理论，练习方法
**执行**:
1. 理解静默意义：在静默中遇见自己
2. 选择形式：每日静默时间
3. 准备环境：安静角落，舒适坐垫
4. 开始练习：每天15分钟
5. 延长时间：周末参加静修

**输出**: 静默练习计划

**校准点4**: 精神修行深化
- 确认冥想练习进展顺利
- 确认正念融入生活
- 确认静默练习有效

### Step 5: 精神遗产传承

#### 5.1 人生回顾 (SP5-01)
**输入**: 人生阶段框架，回顾引导
**执行**:
1. 划分阶段：童年、青年、婚姻、母亲、丧偶
2. 回顾关键事件：每个阶段的重要时刻
3. 识别成长：从依赖到独立，从被爱到爱人
4. 提炼主题：爱的传递
5. 记录感悟：写人生回顾信

## 范本3：退休后的人生意义重构

### 题目
60岁男性退休后的人生意义重构全流程

### 背景
陈先生，60岁，刚从工程师岗位退休。工作35年后突然离开职场，感到迷茫和失落。妻子仍在工作，子女在外地。他不知道如何安排大量空闲时间，担心失去社会价值。希望通过精神探索找到退休后的人生意义。

### Step 1: 价值观重新评估

#### 1.1 核心价值观识别 (SP1-01)
**输入**: 价值观清单，反思引导
**执行**:
1. 完成价值观清单：技术、成就、家庭、健康、贡献
2. 反思职业生涯：工程师生涯的价值体现
3. 评估价值观影响：技术成就驱动职业发展
4. 识别核心价值观：技术、贡献、家庭、健康、学习
5. 验证真实性：回忆退休前行为

**输出**: 退休后核心价值观

#### 1.2 价值观排序 (SP1-03)
**输入**: 核心价值观，排序标准
**执行**:
1. 评估重要性：健康9分，家庭8分，贡献7分，学习7分，技术6分
2. 分析关系：健康是基础，贡献需要技术
3. 识别冲突：技术追求与健康维护
4. 排序：健康 > 家庭 > 贡献 = 学习 > 技术
5. 验证：是否符合退休后需求

**输出**: 退休后价值观排序

#### 1.3 价值观与行为对齐 (SP1-06)
**输入**: 价值观排序，行为记录
**执行**:
1. 记录当前行为：看电视、散步、偶尔聚会
2. 分析动机：打发时间，保持基本社交
3. 评估一致性：行为与价值观严重不一致
4. 识别问题：缺乏贡献和学习活动
5. 制定改进：增加志愿服务、学习新技能

**输出**: 行为调整计划

**校准点1**: 价值观重新评估
- 确认新价值观适合退休生活
- 确认排序反映真实需求
- 确认行为调整可行

### Step 2: 意义探索阶段

#### 2.1 人生意义探索 (SP2-01)
**输入**: 价值观评估，意义探索引导
**执行**:
1. 反思人生满意度：工作8分，家庭7分，健康6分，精神5分
2. 识别意义来源：工作成就、家庭责任
3. 探索新意义：志愿服务、兴趣培养、社区参与
4. 定义新意义："通过技术经验和智慧，帮助年轻人成长"
5. 验证：回忆过去帮助他人的满足感

**输出**: 退休人生意义定义

#### 2.2 叙事身份建构 (SP2-02)
**输入**: 人生关键事件，叙事框架
**执行**:
1. 收集事件：求学、工作、结婚、育儿、退休
2. 识别联系：从学习者到专家到导师
3. 提炼身份："工程师"、"导师"
4. 建构叙事：从技术专家到人生导师
5. 评估影响：新身份给予新方向

**输出**: 退休身份叙事

#### 2.3 意义重构 (SP2-07)
**输入**: 退休转折点，重构框架
**执行**:
1. 识别转折点：退休
2. 评估旧意义：以工作为中心的意义失效
3. 探索新意义：服务、学习、创造
4. 建构新框架：从职业成就到人生智慧
5. 整合到新阶段：成为智慧长者

**输出**: 意义重构计划

**校准点2**: 意义探索完成
- 确认新意义真实可行
- 确认身份叙事健康
- 确认意义重构成功

### Step 3: 精神修行开始

#### 3.1 冥想入门 (SP3-01)
**输入**: 冥想基础，技术指导
**执行**:
1. 了解类型：专注冥想（适合工程师思维）

**输出**: 冥想练习计划

#### 3.2 呼吸法 (SP3-03)

**执行**:
1. 了解呼吸影响：减压、提升专注
2. 学习技术：腹式呼吸、4-7-8呼吸
3. 选择适合：4-7-8呼吸（简单有效）
4. 开始练习：每天2次，每次5分钟
5. 效果评估：睡眠改善，压力减轻

**输出**: 呼吸练习计划

#### 3.3 身体觉察 (SP3-04)
**输入**: 身体扫描技术，觉察指导
**执行**:
1. 学习身体扫描：从头到脚的觉察
2. 开始练习：每周3次，每次15分钟
3. 识别信号：肩颈紧张（压力）、腰部不适（久坐）
4. 发展智慧：通过身体了解情绪
5. 整合到生活：定期检查身体状态

**输出**: 身体觉察练习计划

**校准点3**: 精神修行开始
- 确认冥想练习正常
- 确认呼吸法有效
- 确认身体觉察进展
