## Copy Generation Framework

### Headline Formulas

```text
OUTCOME-LED:
[Specific Result] for [Specific Person]
"Cut my audit time by 60% — here's the workflow"

PAIN-POINT-DRIVEN:
[Pain Point] + [Solution] + [Proof]
"Stop losing deals to this one overlooked landing page mistake"

SOCIAL PROOF ANCHORED:
[Number] + [Specific Claim] + [Who It's For]
"327 security researchers use Atlas to cut triage time in half"

OBJECTION-CRUSHING:
[Address the Doubt] + [Proof It Won't Happen]
"No finding goes undocumented — or you don't pay"

BENEFIT-FIRST:
[Who It's For] + [What They Get] + [Why It Matters]
"For bug bounty teams: first-pass audits in 20 minutes, not 20 hours"
```

### CTA Formulas

```text
ACTION-VERB + OUTCOME:
"Get your free audit" / "Start your trial" / "See the report"

BENEFIT-LED:
"Get the exact checklist that found $2M in DeFi exploits"

RISK-REVERSAL:
"Start free — cancel anytime" / "Audit risk-free for 30 days"

SOCIAL PROOF:
"Join 300+ researchers" / "Used by top Code4rena wardens"

URGENCY + SPECIFICITY:
"Get your copy variants in 10 minutes"
```

## A/B Test Planning Template

```markdown
## A/B Test Plan: <Page / Flow>

### Test 1: <Element Being Tested>
- **Hypothesis:** Changing [X] will [increase/remove friction] because [reason]
- **Control:** <current version>
- **Variant:** <proposed change>
- **Primary metric:** <conversion metric being measured>
- **Secondary metric:** <guardrail metric to watch>
- **Minimum sample:** <statistical significance threshold>
- **Expected lift:** <conservative estimate>
- **Priority:** HIGH / MEDIUM / LOW

### Test 2: <Element Being Tested>
... (same structure)

### Test Sequencing Recommendation
1. Run [highest priority, easiest test] first
2. Then [medium effort, medium impact]
3. Then [highest effort,验证 needed]
```

## CRO Roadmap Template

```markdown
## CRO Roadmap: <Product / Page>

### Quick Wins (Days 1–3)
- [ ] <specific change with measurable impact>

### Medium Tests (Week 1–2)
- [ ] <A/B test with full brief>
- [ ] <copy revision with rationale>

### High-Effort Experiments (Weeks 3–4)
- [ ] <larger changes requiring design/dev>
- [ ] <full landing page rebuild if warranted>

### Success Metrics
- Primary: <metric and target>

```

## Growth Experiment Brief

```markdown
## Growth Experiment: <Experiment Name>

**Type:** [Activation / Retention / Referral / Revenue]
**Hypothesis:** If we [change X], then [Y will happen] because [reasoning].
**Current baseline:** [metric value]
**Target:** [metric improvement]
**Minimum detectable effect:** [smallest meaningful change]
**Required sample size:** [per calculator]
**Run time:** [days/weeks]
**Status:** [Idea / Planned / Running / Analyzed]
```

## Limitations

This is a **copy and strategy generation tool**. It helps you create high-quality test plans, copy variants, and CRO roadmaps quickly. Actual conversion lifts require A/B testing, measurement, and iteration. This skill generates the foundation — you still need to run the experiments.
