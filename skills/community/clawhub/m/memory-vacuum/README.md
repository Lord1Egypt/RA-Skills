# 记忆轻量化系统 · 使用说明

## 功能

- ✅ 每日自动分片，文件不膨胀
- ✅ 语义压缩，只保留必要信息
- ✅ 0 内存占用，按需加载
- ✅ 重启后记忆完整不丢失
- ✅ 永不在需要时找不到信息

## 安装

```bash
cp -r memory-vacuum ~/.openclaw/skills/
```

## 核心机制

### 分片策略
```
memory/
├── 2026-05-01.md   ← 5月1日所有对话
├── 2026-05-02.md   ← 5月2日所有对话
├── ...
└── MEMORY.md        ← 精华，< 50KB
```

### 压缩规则
- MEMORY.md > 50KB → 自动压缩最旧的一半
- 单日日记 > 100KB → 拆分成月份子文件
- 重复信息 → 自动合并
- 过时但重要的信息 → 标记为"历史参考"但不删除

## 使用

无需手动操作，完全自动运行。

**手动触发：**
```
整理一下记忆
压缩一下记忆文件
检查记忆有没有重复
```

## 0 内存原理

- 太初不预加载任何记忆文件
- 每次需要时按需读取
- MEMORY.md < 50KB，加载速度毫秒级
- 原始对话在当日日记，按日期分流，互不干扰

## 与永久记忆系统的区别

| 技能 | 职责 |
|------|------|
| longterm-memory | 写入：什么要记住 |
| memory-vacuum | 管理：怎么让记忆系统不崩溃 |
