## Known Risks and Mitigations: <br>
Risk: The skill may activate on broad local-AI terms and may propose setup commands. <br>
Mitigation: Use it intentionally and review setup, service, or installation commands before running them. <br>
Risk: Changing Ollama bind addresses, firewall rules, or proxy settings can expose a local inference service beyond localhost. <br>
Mitigation: Keep Ollama localhost-only unless the user explicitly approves a scoped remote-access plan with rollback notes. <br>
Risk: Model pulls and optional cloud API use can send model identifiers, prompts, or attachments to Ollama infrastructure. <br>
Mitigation: Confirm downloads or cloud execution with the user and keep inference, embeddings, and RAG storage local when privacy matters. <br>
