# Algorithm Page Template

Use this template when generating a single algorithm teaching page.

## Table of Contents
1. Goal
2. Section structure
3. Required content
4. Output checklist

---

## 1. Goal

A single algorithm page should help the learner:
- understand what the algorithm does
- understand why its formulas are used
- see how values are computed
- interact with the algorithm where useful
- interpret the result clearly

---

## 2. Section Structure

### Standard structure
1. hero intro
2. algorithm overview
3. problem definition
4. input and parameter area
5. core formulas
6. step-by-step derivation or computation
7. numerical example
8. interactive controls
9. charts
10. result summary
11. auto-generated explanation or conclusion

---

## 3. Required Content

A strong algorithm page should include:
- at least one intuition section
- at least one formula explanation block
- at least one numerical substitution block
- at least one result interpretation section
- charts if the algorithm output is numeric or iterative

### For dynamic pages
Include at least one of:
- slider
- stepper
- dropdown
- algorithm mode toggle

---

## 4. Output Checklist

Before finalizing the page, verify:
- the formulas match the narrative
- the numerical example matches the data shown
- the interactions update meaningful content
- the charts reflect the algorithm's behavior
- the html is complete and runnable
