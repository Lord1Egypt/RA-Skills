# 胸膜腔穿刺术的适应证与禁忌证判断

## 触发条件
当患者存在胸腔积液、气胸或脓胸，且无明确禁忌证时，可考虑行胸膜腔穿刺术。

## 执行步骤

1. **确认适应证类型**：
   - **诊断性**：抽取50～100ml胸腔积液，用于常规、生化、微生物学及细胞学检测，以明确积液性质和病因。
   - **治疗性**：
     - 抽出积液或气体以减轻肺压迫，促进肺复张，缓解呼吸困难；
     - 抽吸脓液并冲洗治疗脓胸；
     - 向胸膜腔注入药物（如抗生素、抗肿瘤药、硬化剂）。

2. **逐一排除禁忌证**（共6类）：
   - 体质衰弱、病情危重难以耐受穿刺；
   - 对麻醉药物（如2%利多卡因）过敏；
   - 凝血功能障碍或严重出血倾向未纠正；
   - 精神疾病或无法配合操作；
   - 疑为胸膜腔棘球蚴病（穿刺可致感染扩散）；
   - 穿刺部位或附近存在皮肤或软组织感染。

3. **术前准备确认**：
   - 明确存在胸腔积液或气胸（影像学支持）；
   - 完成术前评估（包括生命体征、凝血功能、过敏史等）；
   - 患者或法定代理人已签署知情同意书。

4. **特殊注意事项**：
   - 包裹性积液必须结合X线或超声精确定位；
   - 首次减压抽液量不超过600ml，后续每次不超过1000ml；
   - 若注入硬化剂（如米诺环素500mg加生理盐水20～30ml），注药后需变换体位2～4小时，使药物均匀涂布胸膜腔。

> 提示：若存在任何禁忌证，应暂缓或取消操作，并寻求替代诊疗方案。
