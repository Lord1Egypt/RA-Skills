# AI Prompt Engineering Frameworks

## The C.R.A.F.T. Framework

Use for any content generation task:

- **C**larify: What exactly do you want?
- **R**ole: Who should AI behave as?
- **A**ttributes: What tone, style, format?
- **F**ormat: Bullets, prose, JSON, code?
- **T**est: Iterate based on first output

## The RICE Scoring for Prompt Selection

When choosing which prompts to include in a bundle:
- **Reach**: How many people need this?
- **Impact**: How much does it improve output?
- **Confidence**: How tested is this prompt?
- **Effort**: How much customization is needed?

Prioritize high Reach + high Impact + high Confidence prompts.

## Prompt Versioning

Always track prompt versions with:
```
# v1.0 — [DATE] — Initial
# v1.1 — [DATE] — [CHANGE DESCRIPTION]
```

Save winning prompts in a centralized reference. They're repeatable assets.
