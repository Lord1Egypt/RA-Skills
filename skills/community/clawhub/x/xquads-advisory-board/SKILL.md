# Advisory Board

Conselho de 12 pensadores lendários para decisões estratégicas.

## Agentes Disponíveis

- **ray-dalio** - Principles, economic cycles, radical transparency
- **charlie-munger** - Mental models, inversion, worldly wisdom
- **naval-ravikant** - Wealth, leverage, specific knowledge
- **peter-thiel** - Zero to one, monopoly strategy
- **simon-sinek** - Start with why, leadership
- **reid-hoffman** - Blitzscaling, network effects
- **derek-sivers** - Indie entrepreneurship, simplicity
- **patrick-lencioni** - Team health, organizational dysfunction
- **brene-brown** - Vulnerability, courage, leadership
- **yvon-chouinard** - Values-driven business, sustainability
- **board-chair** - Orchestrator

## Tasks Disponíveis

- convene-board - Convene board meeting
- get-founder-counsel - Get founder advice
- seek-investment-counsel - Investment guidance
- evaluate-scaling - Evaluate scaling decisions
- resolve-culture-crisis - Resolve culture issues
- diagnose - Triage strategic challenge

## Como Usar

```
"Devo aceitar esse investimento?"
→ Convene board para múltiplas perspectivas

"Como escalar minha equipe?"
→ Naval + Reid para scaling philosophy

"Minha cultura está piorando"
→ Brene Brown + Lencioni para culture

"Qual o melhor modelo de negócio?"
→ Peter Thiel + Naval para business model
```
