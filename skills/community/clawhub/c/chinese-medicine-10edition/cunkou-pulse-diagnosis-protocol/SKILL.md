# Cunkou Pulse Diagnosis Protocol

## When to Use
所有接受脉诊的患者。

## When to Use
- 环境安静；
- 医患气血平和；
- 患者前臂平伸、掌心向上。

## Execution Steps

### 1. 准备阶段
- 患者前臂平伸，掌心向上，腕下垫脉枕；
- 医生以左手诊患者右手，右手诊患者左手。

### 2. 定位三部
- 以腕后桡骨茎突（高骨）为标记；
- 其内侧动脉搏动处为“关”；
- 关前（近腕侧）为“寸”；
- 关后（近肘侧）为“尺”。

### 3. 布指方法
- 中指定关位，示指切寸位，无名指切尺位；
- 三指呈弓形，指头平齐，以指腹切按脉体；
- 指距疏密根据患者手臂长短灵活调整。

### 4. 三部九候取法
对每一部（寸、关、尺）依次执行以下三种取法：
- **举**（浮取/轻取）：指力轻触皮肤；
- **寻**（中取）：用力不轻不重；
- **按**（沉取/重取）：重力切按至筋骨间。

### 5. 脉率测定
- 以医生一呼一吸（一息）为时间单位；
- 正常脉率为一息四至或五至（约60～90次/分钟）；
- 切脉时间必须持续1分钟以上，确保准确计数。

### 6. 特殊脉位处理
- 若遇“斜飞脉”（脉从尺部斜向手背）或“反关脉”（脉见于腕背），属生理变异，非病脉，需调整手指位置顺应脉道走向。

## Output
获得完整的三部九候脉象信息，用于判断脏腑病位、病性及预后。
