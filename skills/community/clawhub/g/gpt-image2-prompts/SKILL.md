## 模式选择

### [A] 专业模式（推荐设计师/有经验用户）

**触发方式**: 用户明确描述需求时，直接进入

**特点**:
- 跳过收敛问题
- 跳过详细分析
- 直接输出 2 套高质量方案
- 输出精简，每套提示词独立成段
- **无评分卡**（专业用户自己判断）

**适用**: 你知道自己要什么，只需要 AI 帮你组织语言

---

### [B] 引导模式（推荐普通用户）

**触发方式**: 用户需求模糊时触发

**特点**:
- 主动提出 3 个以内收敛问题
- 基于反馈生成 2-4 套方案
- **每套方案附带评分卡**
- **标注「推荐首选」**
- 提供场景预设快速入口

**适用**: 你不知道要什么风格，需要引导

## 提示词语言选择

### 规则

| 场景 | 建议语言 | 原因 |
|------|---------|------|
| 摄影风格（写实类） | **英文** | GPT Image 2 对英文摄影术语理解更精准 |
| 插画/手绘类 | **英文** | 风格关键词效果更稳定 |
| 动漫/二次元 | **英文** | anime/cel-shaded 等关键词效果更好 |
| 中文特有内容 | **中文主体 + 英文环境** | 汉服/古风/中文场景用中文主体描述 |
| 文字渲染类 | **中文** | 需要渲染的文字必须用目标语言 |

### 混合模式模板

```
[中文主体描述] + [英文风格/光线/构图]
```

## 收敛问题模板（人话版）

当需求模糊时，使用以下问题：

```
我来帮你设计，先确认几个关键点：

1. 【想要什么感觉的？】
   - 像真的照片一样（真实感）
   - 像手画的一样（艺术感）
   - 像动画片（卡通感）
   - 像电影截图（有故事感）

2. 【这张图用来干嘛？】
   - 发朋友圈/社交媒体
   - 当头像/壁纸
   - 工作/品牌宣传用

3. 【有什么特别想加的吗？】
   回复关键词，我来补全！
```

### 「推荐首选」标记规则

满足以下任一条件，标记为「推荐首选」：
- 用户未指定风格时，默认选择「最通用/最安全」的方案
- 方案评分卡中「上手度」最高
- 方案最适合目标使用场景（如社媒传播）

## 参考图处理流程

### 4 步法

**步骤 1：图像分析**

```
我分析了你的参考图：
- 主体：[识别结果]
- 风格：[识别结果]
- 色调：[识别结果]
- 光线：[识别结果]
- 构图：[识别结果]
- 氛围：[识别结果]
```

**步骤 2：要素转化**

将分析结果转化为提示词模板

**步骤 3：主体替换**

将用户指定的新主体填入模板

**步骤 4：确认调整**

```
请确认：
1. 新主体是什么？
2. 有没有要调整的地方？
```

## 反馈调整流程

### 快速问题 → 快速修复

| 问题 | 调整关键词 |
|------|-----------|
| 太假/像塑料 | 去掉 "flawless" → 加 "realistic skin texture, visible pores, natural imperfection" |
| 太暗 | 加 "bright, well-lit, naturally illuminated" |
| 太亮 | 加 "soft lighting, muted tones" |
| 风格不对 | 替换风格关键词 |
| 缺少氛围 | 加氛围词 + 光线描述 |
| 构图不好 | 调整构图词 |
| 文字模糊 | 加 "crisp text, sharp typography" |

### 调整输出模板

```
基于你的反馈，我做了以下调整：

调整前：[原问题描述]
调整后：[调整说明]

调整后提示词：
[新提示词]

快速复制版本：
[一键复制版本]

先试这个，如果还需要调整，告诉我具体哪里不满意。
```

## 质量控制词库

### 皮肤质感

```
positive: realistic skin texture, visible pores, natural imperfection
negative: no plastic skin, no airbrushing, no over-sharpening
```

### 光影质量

```
positive: soft diffused lighting, volumetric light rays, rim lighting
negative: no harsh shadows, no blown highlights
```

### 整体画质

```
positive: photorealistic, ultra-detailed, sharp focus
negative: no watermark, no text (unless requested), no AI artifacts
```

## 评分维度说明

| 维度 | 解释 |
|------|------|
| 真实感 | 偏向写实还是艺术化，★多=更真实 |
| 氛围感 | 情绪传达强度，★多=氛围感更强 |
| 上手度 | 你后续自行调整的难易程度，★多=更容易改 |
| 适用性 | 在主流场景中的通用程度，★多=更百搭 |
| 创意度 | 独特性与新颖程度，★多=更有创意 |

## 注意事项

- 中文需求默认输出英文提示词，中文特有内容除外
- 提示词各要素用逗号分隔，保持语义连贯
- 质量控制词放在末尾
- 多方案时风格差异要明显
- 场景预设可叠加使用
- 文字渲染需求必须明确标注需要渲染的文字内容
