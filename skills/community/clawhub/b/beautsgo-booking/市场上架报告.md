## 四、确认不可上架 (15 个)

按原因分类:

### 4.1 性质 / 格式不符(7 个)
| 平台 | 排除原因 |
|---|---|
| Coze 扣子(字节) | Bot 协议,跟 SKILL.md 不兼容,需重写 |
| Dify | 自家 plugin SDK,需重写 |
| 腾讯元宝 / 元器 | 需在元器后台从零搭建智能体配置,非"上架"是"重建",30-60 分钟/家 |
| 智谱 BigModel | marketplace 页白屏,且需重写为自家协议 |
| 字节 AutoGLM | 是 chat 应用,不是 skill marketplace |
| daymade/claude-code-skills | 作者个人作品集 daymade-* 前缀,不接受第三方 |
| netresearch/claude-code-marketplace | 公司只放自家 agent-harness 等 skills |

### 4.2 商业化 / 付费销售(3 个)
| 平台 | 排除原因 |
|---|---|
| Agensi | $15–$29 bundle 商业模式;且 submit 页报错 + 各分类 0 skills |
| KissMySkills | Shopify 付费销售平台,与 MIT-0 开源不符 |
| MCPMarket | "出售技能"商业化,我们项目未被动收录 |

### 4.3 stars 门槛拦死(2 个)
