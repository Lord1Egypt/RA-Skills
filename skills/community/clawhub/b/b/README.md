# Letter B Skill

> A comprehensive prompt-based skill about the letter **B** — the second letter of the Latin alphabet.

## Overview

This skill provides rich, structured knowledge about the letter B, covering its history, phonetics, typography, usage in science & technology, music theory, cultural significance, and fun facts. It is designed as a **natural-language, prompt-based skill** that activates whenever a user mentions or implies topics related to the letter B.

## Activation

The skill activates when the user:

- Directly asks about the letter "B" or "b"
- Discusses alphabet history or Latin script origins
- Inquires about phonetics / pronunciation of /b/
- Asks about typography or font design of the letter B
- Mentions B in scientific contexts (Boron, Byte, magnetic field, B vitamins, etc.)
- Refers to B in music theory (the note B, B major/minor)
- Discusses grading systems, ratings, or codes involving B

## Quick Start

This is a **prompt-based skill**. Simply use natural language to interact:

### 🟢 Example Prompts

**History:**
```
What is the origin of the letter B?
```

**Phonetics:**
```
How is the letter B pronounced in Spanish vs. English?
```

**Science:**
```
What does B stand for in the periodic table?
```

**Computing:**
```
What's the difference between B and b in data measurement?
```

**Music:**
```
What is the note B in German musical notation?
```

**Fun Facts:**
```
Tell me some fun facts about the letter B.
```

**Comprehensive:**
```
Give me a full overview of the letter B.
```
