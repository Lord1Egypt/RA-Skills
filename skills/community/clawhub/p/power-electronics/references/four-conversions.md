# 四大电力变换总览

电力电子核心任务：**电压/电流/频率/相数的变换**，通过功率开关器件的导通与关断实现。

| 变换 | 能量流向 | 典型拓扑 | 主要控制量 |
|------|----------|----------|------------|
| AC-DC 整流 | 交流→直流 | 不可控/半控/全控桥 + LC 滤波；PFC Boost | 触发角 α、滤波参数 |
| DC-AC 逆变 | 直流→交流 | 单相 H 桥 SPWM；三相两电平 + SVPWM | 调制比 M、频率 fo、死区 |
| DC-DC 斩波 | 直流→直流 | Buck / Boost / Buck-Boost / Flyback / LLC | 占空比 D、频率 fsw |
| AC-AC 交流变换 | 交流→交流 | 相控调压；交流斩波；矩阵式；交交变频 | 触发角 α 或占空比 D |

## 选型决策树

```
输入是 AC 还是 DC？
├─ AC 输入
│   ├─ 需要 DC 输出？ → AC-DC（整流 + 滤波 [+ PFC]）
│   └─ 需要 AC 输出（同频或变频）？ → AC-AC（相控 / 斩波 / 交交变频 / 矩阵）
└─ DC 输入
    ├─ 需要 AC 输出？ → DC-AC（逆变）
    └─ 需要 DC 输出？ → DC-DC（斩波 / 隔离变换）
```

## 各变换典型考试/工程题型

| 变换 | 常见题型 |
|------|----------|
| AC-DC | 求 Vd、Id、功率因数；α 范围；电容/电感滤波设计；逆变工况判定 |
| DC-AC | SPWM/SVPWM 调制比与输出电压；扇区与时间；THD 与过调制 |
| DC-DC | CCM/DCM 判定；L/C 设计；占空比与增益；器件应力 |
| AC-AC | 相控调压输出有效值；交流斩波 D 与 Vo_rms；交交变频输出频率范围 |

## 脚本入口

统一使用 `scripts/power_calc.py --transform <类型>`，详见 SKILL.md。
