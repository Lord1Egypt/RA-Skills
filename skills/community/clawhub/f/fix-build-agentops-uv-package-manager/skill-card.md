## Use Case: <br>
Developers and engineers use this skill to set up Python projects, manage dependencies and virtual environments, install Python versions, migrate from pip or Poetry workflows, and optimize CI, Docker, and monorepo dependency workflows with uv. <br>

### Deployment Geography for Use: <br>

## Known Risks and Mitigations: <br>
Risk: Quick installer examples use remote shell or PowerShell script execution, which can be unsafe in sensitive environments if copied without review. <br>
Mitigation: Prefer pip, Homebrew, cargo, or a verified release; when script installers are necessary, download and inspect the installer first and confirm it comes from the official uv source. <br>
