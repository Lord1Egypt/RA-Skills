## Description: <br>
Generates customized, phased data breach incident response playbooks with tool recommendations and compliance checklists based on an organization's profile. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[krishnakumarmahadevan-cmd](https://clawhub.ai/user/krishnakumarmahadevan-cmd) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Security teams, incident responders, and compliance officers use this skill to generate data breach response plans tailored to company profile, handled data types, existing security tools, and compliance obligations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Users may submit sensitive live incident details, company names, security tools, compliance obligations, or session identifiers to an external API. <br>
Mitigation: Verify that the API operator is trusted and minimize or sanitize inputs before use, especially during an active breach. <br>
