# Price Intel Quality Checklist

---

## Data Quality

- [ ] Competitive set is current (refreshed within 30 days)
- [ ] All 5-8 competitors are still active listings
- [ ] Price data is from today or within the last 48 hours
- [ ] Promotional context is noted (platform event active?)
- [ ] Bundle contents are documented for all competitors
- [ ] Review counts and ratings are logged

---

## Competitor Classification

- [ ] Each competitor classified: EDLP / Flash / High-Low / Event-Only
- [ ] New entrants in last 30 days identified and assessed
- [ ] Competitive set refreshed within 90 days

---

## Price Event Assessment

- [ ] Current price drop classified: flash / event / structural
- [ ] Historical precedent checked
- [ ] Platform event calendar checked
- [ ] 72-hour wait rule applied

---

## Value Gap Analysis

- [ ] Bundle value mapped (your bundle vs. top 3 competitors)
- [ ] Non-price value assets listed
- [ ] Value gap quantified in dollar terms
- [ ] Differentiation options identified before price-match considered

---

## Pricing Decision Gates

- [ ] Price floor confirmed (cost + minimum margin)
- [ ] Price change impact modeled
- [ ] Response type decided: hold / value-add / differentiation / price adjust
- [ ] Decision documented in Price Intel Report Template

---

## Post-Decision Review

- [ ] Decision and rationale logged with date
- [ ] Outcome metrics identified to track
- [ ] Review date set (7 days post-decision)
- [ ] Competitor reversion check scheduled
- [ ] Learnings captured for next quarterly review
