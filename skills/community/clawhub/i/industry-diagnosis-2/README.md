# 行业战略诊断系统 (Industry Diagnosis)

> 基于《五力模型·战略力》框架的智能诊断工具

---

## 简介

输入任意行业/品类名称，自动采集数据并生成专业诊断报告（MD格式）。

**核心价值**：不是给"行业分析报告"，而是给"**能不能做+怎么做**"的决策依据。

---

## 基于理论

《连锁品牌：让复制不变形的5力模型》战略力章节

作者：位道策划 杨首位

---

## 功能特性

### 8大诊断模块

1. **市场容量分析** - 赛道规模、增长率、容量评分
2. **行业趋势分析** - 趋势四指标、交叉信号、窗口期判断
3. **竞争格局分析** - 头部品牌、CR3集中度、生存现实
4. **差异化机会分析** - 三层次突破口、三道检验
5. **综合评估** - 四维评分、风险评级、投资建议
6. **切入策略建议** - 定位三策、扩张节奏
7. **风险警示** - 扩张开关闭合红线、操盘手战略手记
8. **发展机遇** - 具体机遇、窗口期、潜力评估

### 核心特色

- **结论前置**：每个模块开头先给核心结论+关键数字
- **数据口径分离**：区分"核心战力"与"生态网络"，避免战略误判
- **诊断-策略闭环**：策略必须引用前文诊断数据，形成完整链条
- **操盘手战略手记**：风险章节固定模板，强化战略纪律

## 安装使用

### OpenClaw用户

```bash
# 从ClawHub安装（推荐）
openclaw skills install ysw-2025/industry-diagnosis

# 或从GitHub安装
openclaw skills install git:ysw-2025/industry-diagnosis
```

### 手动安装

1. 下载 `industry-diagnosis.skill` 文件
2. 解压到 `~/.openclaw/skills/` 目录
3. 重启OpenClaw或执行 `/new` 刷新会话

## 技术架构

### 工作流程

```
输入行业名称
    ↓
并行数据采集（8组搜索关键词）
    ↓
数据整合与校验（多源验证+口径分离）
    ↓
四维量化评分（容量/趋势/竞争/差异化）
    ↓
生成MD诊断报告（8大模块）
    ↓
报告质量自检（5项检查）
```

### 文件结构

```
industry-diagnosis/

└── references/

```
