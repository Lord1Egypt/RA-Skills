# ROS介导的多靶点细胞损伤分析

## 触发条件
当满足以下条件时激活本技能：
- ROS生成显著增加（如由缺血、炎症、辐射、化学毒物或吞噬作用引起）
- 抗氧化系统（如SOD、谷胱甘肽过氧化物酶、维生素E）功能不足或被耗竭

## 执行步骤
1. **确认ROS类型**：识别涉及的活性氧种类，包括自由基类（超氧自由基 O₂·⁻、羟自由基 ·OH、次氯酸自由基 ·OCl、一氧化氮自由基 NO·）和非自由基类（过氧化氢 H₂O₂）。
2. **评估抗氧化状态**：判断超氧化物歧化酶（SOD）、谷胱甘肽过氧化物酶、过氧化氢酶及维生素E等是否足以清除ROS。
3. **识别四大损伤靶点**：
   - **生物膜脂质过氧化**：检查膜磷脂是否发生氧化，导致膜流动性下降和通透性增加。
   - **线粒体直接损伤**：评估线粒体呼吸链功能是否受损，ATP合成是否减少。
   - **DNA损伤**：检测是否存在单链断裂、碱基修饰或氧化产物（如8-OHdG）。
   - **蛋白质交联与降解**：观察含硫氨基酸残基是否形成二硫键交联，或多肽是否断裂成碎片。
4. **判断链式放大效应**：确认是否存在自由基引发的连锁反应，即一个ROS分子引发多个新自由基生成。
5. **关联终末病理变化**：将上述机制与水肿、脂肪变性、细胞坏死等组织学改变联系起来。

## 输出提示
若检测到丙二醛（MDA）、DNA断裂标志物或蛋白质交联产物，应提示“ROS介导的细胞损伤已发生”，并建议干预抗氧化通路或清除ROS源头。
