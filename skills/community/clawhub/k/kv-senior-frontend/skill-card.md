## Description: <br>
Builds, optimizes, and reviews React and Next.js applications with TypeScript and Tailwind CSS. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to scaffold React or Next.js projects, generate frontend components and hooks, analyze bundle size, and review accessibility, performance, and TypeScript/Tailwind code quality. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Scripts can write project, component, hook, test, story, and configuration files when invoked. <br>
Mitigation: Run scripts only in the intended project directory and review generated files before committing. <br>
Risk: Sample analytics, chat-widget, web-vitals, and client-side token-storage patterns may need privacy and security review before production use. <br>
Mitigation: Review these patterns against the target application's privacy, security, and data-handling requirements before adopting them. <br>
Risk: Frontend recommendations or generated code may be incomplete for a specific codebase or framework version. <br>
Mitigation: Validate generated code with the project's tests, linting, accessibility checks, and manual review before release. <br>
