# PC Repair Skill

电脑维修智能助手 - WorkBuddy AI 技能

## 📖 简介

这是一个为 WorkBuddy AI 助手开发的电脑维修智能诊断技能。当用户描述电脑故障时，技能会自动进行交互式诊断，提供维修方案建议和成本估算。

## ✨ 功能特性

- 🔍 **智能故障诊断** - 通过交互式问诊定位故障根因
- 🛠️ **三层维修方案** - 紧急自救 / DIY维修 / 专业维修
- 💰 **成本估算** - 基于2024年市场价格的维修成本参考
- 🛒 **配件购买指导** - 硬件兼容性检查和购买建议
- 🛡️ **防坑指南** - 维修店常见套路及应对方法
- 📊 **可视化报告** - 生成详细的维修决策报告

## 🎯 适用场景

- 台式机/笔记本电脑故障诊断
- Windows/macOS/Linux 系统问题
- 硬件兼容性检查和升级建议
- 维修成本估算和方案对比
- 配件购买指导

## 🚀 安装使用

### 前提条件

- 已安装 WorkBuddy AI 助手
- WorkBuddy 版本 >= 1.0.0

### 安装方法

1. **下载技能包**
   ```bash
   # 从 GitHub 克隆
   git clone https://github.com/bettermen/pc-repair.git ~/.workbuddy/skills/pc-repair
   ```

2. **或下载 ZIP 包**
   - 下载最新的 `pc-repair.zip`
   - 解压到 `~/.workbuddy/skills/pc-repair/` 目录

3. **重启 WorkBuddy**
   - 技能会自动加载

**触发方式1：直接描述故障**
```

AI：[自动加载 pc-repair 技能，开始诊断]
```

**触发方式2：明确请求**
```

AI：[加载 pc-repair 技能]
```

**触发方式3：维修咨询**

## 🔧 支持的故障类型

### 1. 无法开机类
- 完全无反应（电源/主板/CPU问题）
- 有反应但黑屏（显卡/内存/显示器问题）
- 开机卡LOGO（硬盘/BIOS问题）

### 2. 系统运行类
- 蓝屏死机（硬件兼容性/驱动/系统文件）
- 卡顿缓慢（硬盘/内存/CPU/软件）
- 频繁重启（散热/电源/内存）

### 3. 硬件故障类
- 存储问题（硬盘坏道/数据恢复）
- 显示问题（花屏/闪屏/偏色）
- 声音问题（无声/杂音/风扇狂转）
- 外设问题（键盘/鼠标/USB/网络）

### 4. 性能升级类
- 内存升级（兼容性检查/安装指导）
- 硬盘升级（SSD迁移/系统转移）
- 清灰保养（拆机指导/硅脂更换）

## 💡 常见使用场景

### 场景1：无法开机故障

**用户输入**：
```
我的台式机昨天还好好的，今天早上按电源键没反应，
风扇不转，键盘灯也不亮，屏幕黑的。
```

**技能输出**：
- 故障诊断：电源故障
- 紧急处理：检查插座/电源线
- 维修方案：DIY更换电源 或 送修
- 成本估算：200-500元

### 场景2：电脑卡顿故障

**用户输入**：
```
我的笔记本用了5年了，最近特别卡，
开机要2分钟，打开软件也要等半天。
```

**技能输出**：
- 故障诊断：硬盘老化 + 内存不足
- 升级建议：换SSD + 加内存
- 成本估算：500-800元
- 性能提升：开机10秒 + 软件秒开
