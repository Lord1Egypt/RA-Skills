# 新生儿Rh溶血病换血治疗

## 何时使用
当新生儿符合以下任一条件时，立即执行换血治疗：
- ≥35周新生儿在强光疗4～6小时后，血清总胆红素（TSB）未下降甚至上升，或免疫性溶血患儿TSB降幅<2～3 mg/dl；
- 出生时脐血胆红素>4.5 mg/dl（76 μmol/L），血红蛋白<110 g/L，并伴有水肿、肝脾大或心力衰竭；
- 已出现急性胆红素脑病的临床表现（无论TSB是否下降）。

> 注意：非免疫性高胆红素血症通常不需换血；操作前需排除禁忌证。

## 执行步骤

### 1. 确认指征与术前准备
- 对符合指征者，在准备血源的同时给予强光疗4～6小时（如适用）。
- 排除换血禁忌证。

### 2. 选择合适血源
- **Rh溶血病**：Rh血型与母亲同型，ABO血型与患儿同型；紧急情况下可使用O型血。
- **ABO溶血病**（母O型，子A/B型）：推荐使用AB型血浆 + O型红细胞混合血。
- **伴明显贫血或心衰**：使用血浆减半的浓缩血。

### 3. 确定换血量
- 换血总量为患儿血容量的2倍，约150～180 ml/kg。
- 此剂量可换出约85%致敏红细胞、60%胆红素及抗体。

### 4. 选择换血途径
- 常用脐静脉或其他大静脉单通道换血；
- 也可采用脐动脉+脐静脉双通道，或外周动静脉同步换血。

### 5. 理解换血作用机制
- 换出部分游离抗体和致敏红细胞，减轻溶血；
- 清除大量胆红素，预防胆红素脑病；
- 纠正贫血，改善携氧能力，防止心力衰竭。

### 6. 术后监测
- 持续监测生命体征、电解质及胆红素水平变化。

> 本技能适用于Rh溶血病新生儿及严重ABO溶血病新生儿，不适用于非免疫性高胆红素血症。
