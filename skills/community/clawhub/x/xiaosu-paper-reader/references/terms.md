# 常用术语对照表

| 英文术语 | 推荐中文 | 说明 |
|----------|----------|------|
| hippocampus | 海马 / 海马体 | 参与空间记忆和导航的大脑区域 |
| place cells | 位置细胞 | 在特定位置放电的神经元 |
| place coding | 位置编码 | 对空间位置的神经表征 |
| gaze | 注视 | 动物视觉朝向某目标 |
| gaze tuning | 注视调谐 | 神经元对不同注视目标的选择性 |
| head saccades | 头部扫视 | 快速、弹道式转头动作 |
| gaze fixation | 注视固定 | 扫视之间的稳定注视期 |
| lateral gaze | 侧向注视 | 用单眼侧向观察目标 |
| frontal gaze | 前向注视 | 喙朝向目标，双眼前向观察 |
| dash | 冲刺式移动 | 动物快速直接移动到目标 |
| contralateral gaze | 对侧注视 | 与记录半球相对一侧眼的注视 |
| ipsilateral gaze | 同侧注视 | 与记录半球同侧眼的注视 |
| interneurons | 中间神经元 | 调控网络兴奋和抑制的神经元 |
| theta oscillation | theta 节律 / theta 振荡 | 海马中常见的节律性神经活动 |
| catch trials | 捕捉试次 | 预期刺激出现但实际不出现的试次 |
| inhibitory neurons | 抑制性神经元 | 降低网络活性的神经细胞 |
| excitatory neurons | 兴奋性神经元 | 提高网络活性的神经细胞 |
| spatial memory | 空间记忆 | 对空间位置和环境的记忆 |
| navigation | 导航 | 从一处移动到另一处的能力 |
| synaptic plasticity | 突触可塑性 | 神经元连接强度可变化的能力 |
| firing rate | 放电率 | 神经元单位时间内的放电次数 |
| tuning curve | 调谐曲线 | 神经元对某一参数（如位置）的响应曲线 |
| optogenetic | 光遗传学的 | 用光控制神经元的技术 |
| in vivo | 体内 | 在活体中进行的实验 |
| in vitro | 体外 | 在培养皿或切片中进行的实验 |
