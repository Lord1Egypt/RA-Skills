# 物理致癌因素的作用机制

## 何时使用
- 用户描述个体有**长期或高强度暴露于紫外线（如日晒）或电离辐射（如医疗照射、职业暴露）** 的情况。
- 用户关注**皮肤癌（如基底细胞癌、鳞状细胞癌、黑色素瘤）或白血病**的成因。
- 用户需要解释**DNA损伤类型（如嘧啶二聚体、染色体断裂）** 与特定辐射源之间的关联。

## 如何执行
1. **识别辐射类型**：
   - 若为**紫外线（UV）**，聚焦于皮肤表层细胞；
   - 若为**电离辐射（如X线、γ射线、β粒子）**，考虑其对全身组织（尤其是骨髓等分裂活跃组织）的影响。

2. **分析DNA/染色体损伤机制**：
   - **UV机制**：检查是否形成**环丁烷嘧啶二聚体**，该结构阻碍DNA复制，引发突变。
   - **电离辐射机制**：评估是否存在**染色体断裂、易位或点突变**，可能导致原癌基因激活或抑癌基因失活。

3. **关联靶组织与癌症类型**：
   - UV主要致**皮肤细胞**损伤，关联**皮肤癌**；
   - 电离辐射可穿透组织，显著增加**造血细胞**突变风险，导致**白血病**。

4. **输出风险提示**：
   - 明确指出辐射类型、典型DNA损伤形式、主要靶组织及对应癌症风险。

> 注意：本技能不适用于化学或生物致癌机制；若辐射剂量低于阈值，可能不致病，需结合暴露强度与持续时间综合判断。
