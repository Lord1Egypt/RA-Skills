# TCM Diagnosis Guide (诊断指南)

Four examinations (四诊) guide for systematic data collection.

---

## 1. Inspection (望诊)

### Tongue Diagnosis (舌诊) — Critical diagnostic tool

| Tongue Body (舌质) | Indication |
|-------------------|------------|
| Pale (淡白) | Blood deficiency, Yang deficiency |
| Red (红) | Heat (excess or deficiency) |
| Deep red/Crimson (绛) | Heat entering Ying/Blood level |
| Dark/Purple (紫暗) | Blood stasis, cold |
| Swollen/Puffy (胖大) | Dampness, Spleen deficiency |
| Thin/Small (瘦薄) | Yin deficiency, Blood deficiency |
| Teeth marks (齿痕) | Spleen Qi deficiency |
| Cracked (裂纹) | Yin deficiency, Blood deficiency |
| Spots (瘀点/瘀斑) | Blood stasis |

| Tongue Coating (舌苔) | Indication |
|----------------------|------------|
| White (白) | Cold, Exterior |
| Yellow (黄) | Heat |
| Grey/Black (灰黑) | Extreme cold or heat |
| Thin (薄) | Exterior, mild condition |
| Thick (厚) | Interior, pathogen flourishing |
| Greasy (腻) | Phlegm-dampness, food stagnation |
| Dry (燥/干) | Heat damaging fluids |
| Wet/Moist (润) | Normal or Cold-damp |
| No coating (剥苔/无苔) | Yin deficiency, Stomach Qi deficiency |
| Sticky/Curdy (腐苔) | Food stagnation |

### Complexion (面色)

| Color | Indication |
|-------|------------|
| Pale white (白) | Deficiency, Cold, Blood deficiency |
| Red (赤) | Heat |
| Yellow (黄) | Spleen deficiency, Dampness, Jaundice |
| Green/Blue (青) | Cold, Pain, Blood stasis, Convulsions |
| Dark/Black (黑) | Kidney deficiency, Cold, Blood stasis |

### Body & Movement (形体动态)

- Emaciation (消瘦): Yin deficiency, chronic disease
- Obesity (肥胖): Phlegm-dampness, Qi deficiency
- Tremors (震颤): Liver wind
- Edema (水肿): Water retention, Kidney/Spleen deficiency

## 2. Listening & Smelling (闻诊)

### Voice & Sounds

| Sound | Indication |
|-------|------------|
| Loud voice (声高) | Excess |
| Weak voice (声低) | Deficiency |
| Frequent sighing (善叹息) | Liver Qi stagnation |
| Cough (咳嗽) | Lung Qi rebellious |
| Wheezing (哮) | Phlegm blocking airways |
| Vomiting sounds (呕吐) | Stomach Qi rebellious |
| Hiccup (呃逆) | Stomach Qi disorder |
| Borborygmi (肠鸣) | Phlegm-fluid retention, Dampness |

### Odors

| Odor | Indication |
|------|------------|
| Foul breath (口臭) | Stomach heat, food stagnation |
| Sour belching/vomiting (酸腐) | Food stagnation |
| Foul stools (大便臭秽) | Heat/Damp-heat |
| Foul vaginal discharge (带下臭) | Damp-heat |
| Fishy odor | Cold-damp |

## 3. Inquiry (问诊) — Ten Questions (十问歌)

The systematic Ten Questions framework for data collection:

### 一问寒热 (Chills & Fever)

| Pattern | Meaning |
|---------|---------|
| Aversion to cold + fever | Exterior pattern |
| Aversion to cold, no fever | Cold pattern (deficiency or exterior) |
| Fever, no aversion to cold | Interior heat |
| Alternate chills and fever (寒热往来) | Shaoyang pattern |
| Tidal fever (潮热) | Yangming bowel pattern or Yin deficiency |
| Fever only in afternoon | Yin deficiency or Damp-heat |
| Fever worse at night | Ying-level heat or Yin deficiency |

### 二问汗 (Perspiration)

| Pattern | Meaning |
|---------|---------|
| Sweating + fever + aversion to wind | Exterior deficiency (桂枝汤 pattern) |
| No sweating + fever + body aches | Exterior excess (麻黄汤 pattern) |
| Spontaneous sweating (自汗) | Qi deficiency |
| Night sweats (盗汗) | Yin deficiency |
| Profuse cold sweat | Yang collapse |
| Oily/greasy sweat | Yin exhaustion (critical) |

### 三问头身 (Head & Body)

| Complaint | Pattern |
|-----------|---------|
| Headache with stiff neck | Exterior (Taiyang) |
| Migraine / temporal headache | Shaoyang or Liver Yang |
| Vertex headache | Liver or Jueyin |
| Occipital headache | Taiyang |
| Dizziness | Liver Yang, Phlegm, Blood deficiency |
| Body aches | Exterior, Dampness |
| Heavy sensation | Dampness |
| Cold lower back/knees | Kidney deficiency |

### 四问便 (Stools & Urine)

**Stools:**
| Pattern | Meaning |
|---------|---------|
| Constipation + abdominal pain | Excess heat (Yangming) |
| Constipation + dry stools | Yin deficiency, blood deficiency |
| Diarrhea with foul odor | Damp-heat |
| Diarrhea with undigested food | Spleen Yang deficiency |
| Watery diarrhea | Kidney Yang deficiency |
| Tenesmus (里急后重) | Damp-heat in intestines |

**Urine:**
| Pattern | Meaning |
|---------|---------|
| Scanty dark urine | Heat |
| Clear profuse urine | Cold |
| Frequent urination (clear) | Kidney Qi deficiency |
| Frequent urgent urination (yellow) | Damp-heat in bladder |
| Difficult/painful urination | Damp-heat (Lin syndrome) |

### 五问饮食 (Appetite & Thirst)

| Pattern | Meaning |
|---------|---------|
| Thirst with desire for cold drinks | Interior heat |
| Thirst with desire for warm drinks | Dampness, Yang deficiency |
| No thirst | Cold, Dampness |
| Dry mouth, no desire to drink | Damp-heat or Ying-level heat |
| Great appetite (消谷善饥) | Stomach heat |
| Poor appetite | Spleen deficiency |
| Hunger without eating | Stomach Yin deficiency |

### 六问胸腹 (Chest & Abdomen)

| Pattern | Meaning |
|---------|---------|
| Chest pain (stabbing) | Blood stasis in chest |
| Chest/rib distension | Liver Qi stagnation |
| Epigastric fullness (痞满) | Middle burner Qi stagnation |
| Abdominal pain (warm relief) | Deficiency cold |
| Abdominal pain (cold relief) | Excess heat |
| Lower abdominal pain | Blood stasis, Liver cold |

### 七问聋 (Hearing)

| Pattern | Meaning |
|---------|---------|
| Sudden tinnitus (roaring) | Liver fire, Liver Yang |
| Gradual tinnitus (low) | Kidney deficiency |
| Hearing loss | Kidney deficiency |
| Sudden deafness | Wind-heat, Liver fire |

### 八问渴 (Thirst — already covered in #5)

### 九问旧病 (Past History)

- Previous illnesses
- Surgeries
- Allergies
- Current medications
- Family history

### 十问困 (Causal Factors)

- Dietary habits
- Emotional state
- Work/lifestyle
- Sleep quality
- Menstrual history (women)

### Additional Gynecological Questions (女子尤必问经期)

| Finding | Meaning |
|---------|---------|
| Delayed menses | Blood cold, Blood deficiency |
| Early menses | Blood heat |
| Irregular menses | Liver Qi stagnation |
| Painful menses (relieved by warmth) | Cold in uterus |
| Painful menses (fixed stabbing) | Blood stasis |
| Heavy bleeding | Qi deficiency, Blood heat |
| Scant bleeding | Blood deficiency, Blood stasis |
| Menopause | Natural, or Kidney deficiency |
| Vaginal discharge (white/clear) | Cold-damp |
| Vaginal discharge (yellow/foul) | Damp-heat |

## 4. Palpation (切诊)

### Pulse Diagnosis (脉诊)

| Pulse Quality (脉象) | Sensation | Indication |
|:--------------------:|-----------|------------|
| 浮 (Floating) | Felt on light touch | Exterior pattern |
| 沉 (Deep) | Felt only on heavy pressure | Interior pattern |
| 迟 (Slow) | < 60 beats/min | Cold pattern |
| 数 (Rapid) | > 90 beats/min | Heat pattern |
| 滑 (Slippery) | Rolling, smooth | Phlegm, food stagnation, pregnancy |
| 涩 (Choppy) | Rough, difficult | Blood stasis, Essence deficiency |
| 弦 (String-like) | Straight, taut | Liver disorder, Pain |
| 紧 (Tight) | Tense, like twisted rope | Cold, Pain |
| 缓 (Moderate) | 70-80 beats/min | Normal, or Dampness |
| 细 (Thin) | Thin as a thread | Deficiency, Yin/Blood |
| 弱 (Weak) | Deep + thin + soft | Qi/Blood deficiency |
| 虚 (Vacuous) | Empty on pressure | Deficiency |
| 实 (Excess/Forceful) | Full, strong | Excess |
| 洪 (Surging) | Surging then fading | Heat |
| 微 (Faint) | Barely palpable | Yang exhaustion (critical) |
| 促 (Knotted-rapid) | Irregular with faster rate | Heat, Blood stasis |
| 结 (Knotted-slow) | Irregular with slower rate | Cold, Blood stasis |
| 代 (Regularly intermittent) | Regular pauses | Organ damage, severe deficiency |
| 濡 (Soft) | Floating + thin + soft | Dampness, Deficiency |

**Pulse position correlations (寸关尺):**

| Position (Left) | Organ | Position (Right) | Organ |
|:---------------:|:-----:|:----------------:|:-----:|
| 寸 (Cun) | Heart | 寸 (Cun) | Lung |
| 关 (Guan) | Liver/Gallbladder | 关 (Guan) | Spleen/Stomach |
| 尺 (Chi) | Kidney (Yin) | 尺 (Chi) | Kidney (Yang) |

### Abdominal Palpation (腹诊)

| Finding | Indication |
|---------|------------|
| Abdominal distension with tenderness | Excess |
| Abdominal distension with no tenderness | Deficiency |
| Epigastric firmness (心下痞硬) | Xiexin Tang patterns |
| Lower abdominal fullness/pain | Blood accumulation |
| Masses that come and go | Qi stagnation |
| Fixed masses | Blood stasis |
| Cold abdomen | Yang deficiency |
| Warm abdomen | Heat |
| Splashing sound (振水音) | Phlegm-fluid retention |
