# 视频制作规格 - Video Production Specs

## 推荐分辨率

| 用途 | 分辨率 | 比例 | 编码 | 码率 |
|------|--------|------|------|------|
| 产品演示（通用） | 1920×1080 | 16:9 | H.264 | 8-12 Mbps |
| 抖音/快手 | 1080×1920 | 9:16 | H.264 | 4-8 Mbps |
| 微信视频号 | 1080×1230 | 4:5 | H.264 | 4-6 Mbps |
| 官网嵌入 | 1280×720 | 16:9 | H.264 | 3-5 Mbps |
| 展会大屏 | 3840×2160 | 16:9 | H.265 | 20-30 Mbps |

## 音频设置

| 参数 | 推荐值 | 备注 |
|------|--------|------|
| 编码格式 | AAC | 兼容性最好 |
| 采样率 | 44100 Hz | CD 品质 |
| 码率 | 192 kbps | 旁白/配音适用 |
| 声道 | Stereo | 双声道立体声 |

## TTS 配音规范

### 中文产品旁白
- 语音：zh-CN-XiaoxiaoNeural（女声，自然温暖）
- 语速：0%（正常速度）
- 字数估算：每分钟约 180-200 字
- 分段：每段不超过 500 字

### 英文产品旁白
- 语音：en-US-JennyNeural（女声，自然专业）
- 语速：0%（正常速度）
- 字数估算：每分钟约 150 字
- 分段：每段不超过 1000 字

### 旁白脚本模板
```
[开场] 萤火虫空压机，专注压缩空气系统节能十五年。
[产品介绍] 今天为您介绍的是{{产品名称}}，{{一句话卖点}}。
[技术亮点] 它采用{{核心技术1}}和{{核心技术2}}，实现了{{关键指标}}。

[结尾] 萤火虫空压机 — 节能自然萤火虫。咨询热线：13825202084
```

## FFmpeg 常用预设

### 通用产品演示视频
```bash
ffmpeg -i input.mp4 \
  -c:v libx264 -preset medium -crf 22 \
  -c:a aac -b:a 192k -ar 44100 \
  -movflags +faststart \
  output.mp4
```

### 竖版短视频（抖音）
```bash
ffmpeg -i input.mp4 \
  -vf "crop=ih*9/16:ih,scale=1080:1920" \
  -c:v libx264 -preset fast -crf 23 \
  -c:a aac -b:a 128k \
  output.mp4
```

### 视频 + TTS 配音合成
```bash
ffmpeg -i video.mp4 -i voiceover.mp3 \
  -c:v copy -c:a aac -b:a 192k \
  -map 0:v:0 -map 1:a:0 \
  -shortest \
  output.mp4
```
