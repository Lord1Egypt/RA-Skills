# 女性不孕症病因分类与筛查优先级

## 适用条件
- 患者已被诊断为女性不孕症
- 已完成基础病史采集和初步检查
- 已排除男方因素为主导的情况
- 不适用于不明原因性不孕的初始归类

## 执行步骤

1. **评估盆腔因素（约占全部不孕因素的35%）**：
   - 检查是否存在输卵管病变（如梗阻、粘连、积水、功能受损）
   - 评估子宫体病变（如内膜异常、肌瘤、腺肌病、宫腔粘连）
   - 排查子宫颈因素（如先天异常、闭锁/狭窄、黏液异常）
   - 判断子宫内膜异位症对排卵、输卵管功能、受精及内膜容受性的影响
   - 筛查生殖器官发育异常（如纵隔子宫、双角子宫、双子宫、输卵管发育异常）

2. **评估排卵障碍（占25%～35%）**：
   - 检查下丘脑病变（如低促性腺激素性无排卵）
   - 评估垂体病变（如高催乳素血症）
   - 分析卵巢病变（如多囊卵巢综合征、早发性卵巢功能不全、先天性性腺发育不全、放化疗致卵巢衰竭）
   - 排查其他内分泌疾病（如先天性肾上腺皮质增生症、甲状腺功能异常）

3. **评估卵巢生殖功能衰老**：
   - 若女性年龄 ≥35～37岁，考虑FSH敏感卵泡耗竭及衰老卵泡比例增加
   - 判断是否因卵母细胞数量减少、质量下降导致不孕

4. **综合判断**：
   - 注意多种病因可能共存，需系统评估
   - 根据病因分类结果，确定优先排查顺序并推荐后续检查（如输卵管造影、激素测定、超声监测等）

> **提示**：本技能输出为病因分类结构及检查指导，不替代临床诊断，需结合辅助检查确认具体病因。
