# Six Climatic Pathogen Characteristics

## When to Use
- 用户提及疾病由风、寒、暑、湿、燥、火等外邪引起；
- 疾病表现为外感病（如感冒、泄泻、痹证等）；
- 需要鉴别是否属于六淫致病，而非内伤杂病或内生五邪。

## Execution Steps

1. **确认前提条件**：
   - 确保六淫已形成并侵入人体；
   - 排除内伤杂病或内生五邪（如内风、内寒等）的情况。

2. **逐一检查五大共同特点**：

   ### (1) 外感性
   - 判断病邪是否从肌表、口鼻而入；
   - 若符合，则归为“外感病”。

   ### (2) 季节性
   - 核对发病季节：
     - 春 → 风病；
     - 夏 → 暑病；
     - 长夏 → 湿病；
     - 秋 → 燥病；
     - 冬 → 寒病。

   ### (3) 环境性
   - 分析患者所处地理或职业环境：
     - 西北高寒地区 → 多寒、燥病；
     - 东南湿热地区 → 多湿、热病；
     - 久居湿地/水上作业 → 易患湿病；
     - 高温作业 → 易患燥热病。

   ### (4) 相兼性
   - 判断是否存在两种及以上邪气合病：
     - 如风寒感冒、湿热泄泻、风寒湿痹等。

   ### (5) 转化性
   - 观察病程中证候性质是否发生变化：
     - 如“寒邪入里化热”、“暑湿日久化燥”；
     - 注意：转化反映的是证候性质变化，非邪气本身转变；
     - 结合患者体质（如阳盛者易从寒化热）综合判断。

3. **输出结论**：
   - 若符合一个或多个特点，提示该病可能由六淫所致；
   - 同时提醒：内生五邪不属于六淫范畴，需排除。

> **注意**：本技能仅适用于外感病辨证，不适用于内伤杂病。
