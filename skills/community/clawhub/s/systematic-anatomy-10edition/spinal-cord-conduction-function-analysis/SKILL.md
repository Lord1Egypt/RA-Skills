# 脊髓传导功能分析

## 何时使用
- 当需阐明感觉信息从躯干、四肢或内脏经脊神经后根传入脊髓并上传至脑的路径时
- 当需说明脑发出的运动指令如何通过下行纤维束传导至脊髓前角或侧角以控制效应器时
- 在评估脊髓白质（上行/下行纤维束）结构完整性对神经功能的影响时
- **注意**：不适用于脊髓完全横断后的急性期（脊髓休克阶段）

## 执行步骤
1. **确认前提条件**：
   - 脊神经后根与前根结构完整
   - 上行与下行纤维束未受损

2. **分析感觉传导路径**：
   - 定位感觉冲动来源（躯干、四肢或部分内脏）
   - 追踪其经脊神经后根进入脊髓
   - 识别所涉上行纤维束（如脊髓丘脑束、薄束、楔束）
   - 确认信息最终上传至脑（如丘脑、大脑皮质）

3. **分析运动传导路径**：
   - 确定运动指令起源于脑
   - 追踪其经下行纤维束（如皮质脊髓束）下行至脊髓
   - 区分目标神经元位置：
     - 前角 → 支配骨骼肌（躯体运动）
     - 侧角 → 支配平滑肌、心肌及腺体（内脏运动）

4. **整合功能意义**：
   - 强调脊髓作为脑与外周之间双向信息通道的核心作用
   - 指出该功能是随意运动和意识性感觉的解剖生理基础

## 关键依赖结构
- **上行纤维束**：负责感觉信息上传（如脊髓丘脑束、薄束、楔束）
- **下行纤维束**：负责运动指令下传（如皮质脊髓束）
- **脊髓白质完整性**：两者均依赖于此
