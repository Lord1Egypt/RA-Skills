# 预算管理指南

## 核心概念

预算帮你在花钱之前先规划好各分类可以花多少，超出时提前预警，而不是事后后悔。

## 基础操作

### 设定单个分类预算

```bash
python scripts/budget.py 你的账本.csv --set 餐饮 2000
python scripts/budget.py 你的账本.csv --set 交通 800
```

### 查看所有预算

```bash
python scripts/budget.py 你的账本.csv --list
```

### 查看执行进度

```bash
python scripts/budget.py 你的账本.csv --progress 2026-05
```

### 超支预警

```bash
python scripts/budget.py 你的账本.csv --alert 2026-05
```

### 移除预算

```bash
python scripts/budget.py 你的账本.csv --remove 餐饮
```

## 预算模板

### 50/30/20 法则（推荐）

基于月收入自动生成分类预算：
- 50% 必要支出（居住、餐饮、交通、通讯、医疗）
- 30% 弹性支出（购物、娱乐、教育、社交）
- 20% 储蓄/投资

```bash
python scripts/budget.py 你的账本.csv --template 15000
```

### 自定义比例

```bash
# 需求40% / 想要30% / 弹性20% / 其他10%
python scripts/budget.py 你的账本.csv --template 15000 --ratio 40 30 20 10
```

## 预算判断标准

| 进度 | 状态 | 建议 |
|------|------|------|
| < 75% | ✅ 正常 | 放心花 |
| 75-90% | 🔸 注意 | 控制消费节奏 |
| 90-100% | ⚠️ 快超了 | 谨慎支出 |
| > 100% | ❌ 已超支 | 本月不要再花了 |

## 实用建议

1. **先有收入再定预算** — 先确认月收入，再反推支出结构
2. **先紧后松** — 预算设低一点，实际超了比计划超了心理负担小
3. **重点盯一项** — 同时盯 10 个分类很累，先盯最重要的 2-3 个
4. **按月清零** — 每月重新评估预算，不建议跨月滚动
5. **把储蓄当支出** — 每月固定往存款账户"花"一笔，预算才算完整
