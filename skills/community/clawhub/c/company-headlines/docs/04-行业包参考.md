# 行业包参考

每个行业包为品牌配置向导的"问题1"提供默认值，包括语调、关键词域、禁用词增补和典型标题模式。

---

## 制造业

| 维度 | 内容 |
|------|------|
| **文案特征** | 重事实、重工艺、忌浮夸。用数据和产线语言说话 |
| **高频关键词** | 产线、良率、工艺、交付、智造、品控、产能、交期 |
| **禁用词增补** | "黑科技""重新定义制造" |
| **推荐语调配比** | 稳健可靠 80% / 技术自信 20% |

### 典型叙事角度
- 从"为什么这个工艺难"写起
- 用产线故事代替技术说明书
- 客户考察 → 写对方对工艺细节的关注

### 标题模式
- 《良率提升2.3%，答案在第三道工序里》
- 《交付周期从45天缩到30天，我们砍掉了什么》
- 《XX客户在产线前站了半小时，问了三个问题》

## 新能源/电池

| 维度 | 内容 |
|------|------|
| **文案特征** | 技术+使命双重驱动。数据和愿景并重 |
| **高频关键词** | 储能、零碳、循环、安全、密度、充放电、电芯 |
| **禁用词增补** | "弯道超车""降维打击" |
| **推荐语调配比** | 稳健可靠 60% / 绿色使命 25% / 技术自信 15% |

### 典型叙事角度
- 安全是底线叙事，不是卖点
- 零碳是长期叙事，不是口号
- 技术突破 → 从"行业痛点"切入，不说"我们很厉害"

### 标题模式
- 《一度电的安全，裹了多少层？》
- 《从惠州到全球：一块电池的零碳旅程》
- 《储能不是比谁装得多，是比谁不出事》

## 财税

| 维度 | 内容 |
|------|------|
| **文案特征** | 精准、合规、信任感。每一句话都要经得起推敲 |
| **高频关键词** | 合规、风控、降本、数字化、筹划、审计、申报 |
| **禁用词增补** | "节税XX万"（除非有审计报告支撑）、"避税" |
| **推荐语调配比** | 稳健可靠 90% / 温度 10%（仅在节日/文化场景） |

### 典型叙事角度
- 从"客户遇到了什么麻烦"写起
- 专业 = 不犯错，不是炫技
- 数字要精确，模糊的数字在财税文里是致命伤

### 标题模式
- 《30家门店的对账，从3天到40分钟》
- 《端午的"账"，财税人怎么算？》
- 《合规不是成本，是护城河》

## 科技/SaaS

| 维度 | 内容 |
|------|------|
| **文案特征** | 快节奏、数据导向、有观点。但不要飘 |
| **高频关键词** | 赋能、效率、场景、协作、AI、自动化、闭环 |
| **禁用词增补** | "All in""十倍速" |
| **推荐语调配比** | 技术自信 60% / 稳健可靠 30% / 温度 10% |

### 典型叙事角度
- 从"用户的真实痛点"切入（不说"行业痛点"这种空词）
- 功能不堆砌，讲一个场景把它讲透
- 数据要对比：用之前 vs 用之后

### 标题模式
- 《一个会计的月底：从加班三天到准点下班》
- 《我们做了一个不酷的功能。客户说好用》
- 《XX轮融资后，我们最在意的数字不是估值》

## 电商/消费

| 维度 | 内容 |
|------|------|
| **文案特征** | 活泼、追热点、用户视角。可以轻松但不能轻浮 |
| **高频关键词** | 体验、爆款、服务、升级、生态、用户、品质 |
| **禁用词增补** | "白菜价""秒杀一切" |
| **推荐语调配比** | 新锐活力 50% / 温暖使命 30% / 稳健可靠 20% |

### 典型叙事角度
- 从用户评价/体验切入
- 产品背后的人的故事
- 节日是和用户沟通的机会（不是促销）

### 标题模式
- 《一个差评，让我们改了半年》
- 《这件爆款背后，是XX个人的XX天》
- 《今日XX｜给每一位挑剔的你》
