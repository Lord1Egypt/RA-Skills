# 卵泡发育阶段识别

## 适用对象
- 卵巢卵泡
- 女性生殖系统组织切片或影像

## 使用前提
- 存在初级卵母细胞
- 具备卵泡细胞
- 卵泡未闭锁或退化

## 执行步骤

1. **检查卵泡整体结构与位置**：观察卵泡位于卵巢皮质浅层还是深部，是否突向表面。
2. **评估卵泡细胞形态与层数**：
   - 单层扁平 → 考虑原始卵泡
   - 多层立方/柱状 → 进入初级或更晚阶段
3. **确认是否存在卵泡腔**：
   - 无腔 → 原始或初级卵泡
   - 有腔 → 次级或成熟卵泡
4. **测量关键参数（若数据可用）**：
   - 初级卵母细胞直径（125–150 μm提示次级卵泡）
   - 透明带厚度（约5 μm）
   - 成熟卵泡直径（可达2 cm）
5. **识别特有结构**：
   - 透明带、皮质颗粒 → 初级以上
   - 放射冠、卵丘 → 次级卵泡
   - 颗粒层变薄、卵丘游离 → 成熟卵泡
6. **判断减数分裂状态（如信息提供）**：
   - 出现第一极体且次级卵母细胞停滞于MII期 → 成熟卵泡（排卵前36–48小时）

## 输出
返回卵泡所处的具体发育阶段（原始卵泡、初级卵泡、次级卵泡或成熟卵泡）及支持判断的关键结构特征。
