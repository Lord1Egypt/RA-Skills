# 腹前外侧群肌的组成与功能

## 适用对象
- 腹外斜肌
- 腹内斜肌
- 腹横肌
- 腹直肌

## 前提条件
- 需了解腹壁基本解剖结构
- 需能识别各肌肉的起止点

## 使用限制
- 不适用于后腹壁肌（如腰方肌）
- 不涵盖上肢或下肢肌肉

## 执行步骤

1. **确认腹前外侧群肌的组成**：包括四块肌肉——腹外斜肌（浅层）、腹内斜肌（中层）、腹横肌（深层）和腹直肌（正中）。

2. **识别各肌起止点**：
   - **腹外斜肌**：起自下位8对肋骨外面，止于髂嵴前部、腹股沟韧带、白线；
   - **腹内斜肌**：起自胸腰筋膜、髂嵴和腹股沟韧带外侧1/2，止于白线及下位3肋；
   - **腹横肌**：起自下位6对肋软骨内面、胸腰筋膜、髂嵴和腹股沟韧带外侧1/3，止于白线；
   - **腹直肌**：起自耻骨联合和耻骨嵴，止于胸骨剑突和第5～7肋软骨前面。

3. **整合功能分析**：
   - 共同保护腹腔脏器并维持腹内压；
   - 收缩时增加腹压，协助排便、呕吐、咳嗽及分娩；
   - 参与脊柱前屈、侧屈及旋转；
   - 降肋以辅助呼气。

4. **识别特殊衍生结构**：
   - 腹外斜肌腱膜形成腹股沟韧带、腔隙韧带、耻骨梳韧带及腹股沟管浅环；
   - 腹内斜肌与腹横肌下部共同构成腹股沟镰（联合腱）和提睾肌；
   - 腹直肌被3～4条腱划分隔成多个肌腹，腱划与鞘前层紧密结合，后层不愈合，故后面游离。
