## Description: <br>
Reverse image search that helps find image sources, visually similar images, and authenticity context using Yandex, Google, and Bing for image URLs or local files without an API key. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[gushu333](https://clawhub.ai/user/gushu333) <br>

### License/Terms of Use: <br>


## Use Case: <br>
External users and developers use this skill to investigate where an image came from, locate visually similar images, or gather authenticity context from public reverse image search engines. Submitted image URLs or files may be processed by Yandex, Google, or Bing. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Submitted image URLs or local files may be processed by external reverse image search providers. <br>
Mitigation: Avoid using private photos, internal documents, IDs, screenshots, or confidential images unless sharing them with the selected provider is acceptable. <br>
