## Description: <br>
An OpenClaw skill that turns Hermes-style self-learning practices into a repeatable workflow for post-task review, experience logging, error-driven improvement, and periodic skill refinement. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[simonstang](https://clawhub.ai/user/simonstang) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and agent users use this skill to turn complex task outcomes, user corrections, and recurring workflows into concise memory notes and refined skills. It supports ongoing knowledge hygiene through scheduled review, selective loading, and error-driven updates. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can encourage persistent memory or skill changes that preserve sensitive details or private task context. <br>
Mitigation: Require explicit approval before writing memory or skill files, and redact secrets, customer data, private prompts, exact file paths, credentials, and sensitive task details. <br>
Risk: Automatically extracted lessons can encode incorrect or outdated guidance. <br>
Mitigation: Review proposed memory entries and skill diffs before accepting them, and periodically prune or correct stale guidance. <br>
