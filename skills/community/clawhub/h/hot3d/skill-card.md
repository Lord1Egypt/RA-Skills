## Description: <br>
HOT3D helps agents guide multi-view egocentric 3D hand and object tracking workflows for Aria and Quest smart-glasses video using Meta's HOT3D dataset and tools. <br>

This skill is for research and development only. <br>

## Use Case: <br>
External researchers and XR developers use this skill to plan HOT3D dataset access, multi-view hand-object tracking setup, visualization, and benchmark-oriented analysis for egocentric video. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The HOT3D dataset and tools are governed by non-commercial and dataset access terms. <br>
Mitigation: Confirm license, registration, and dataset terms before use, especially for commercial or externally shared work. <br>
Risk: Egocentric smart-glasses video can include sensitive personal or bystander data. <br>
Mitigation: Use consent, access controls, retention limits, and isolated processing environments for video and derived tracking data. <br>
Risk: Using upstream HOT3D code and dependencies may introduce environment or supply-chain risk. <br>
Mitigation: Review the upstream repository and dependency list, then install in an isolated Python environment or container. <br>
