# TLICS评分指导胸腰椎骨折治疗决策

## 何时使用
- 患者被诊断为**急性胸腰椎创伤性骨折**
- 需要依据标准化评分系统决定**手术或非手术治疗**
- 需要根据神经状态和后方韧带复合体（PLC）完整性**选择手术入路**
- **不适用于**病理性骨折、陈旧性骨折或非创伤性压缩骨折

## 执行步骤

### 1. 确认前提条件
- 明确损伤机制（如压缩、旋转、牵张）
- 完成MRI检查以评估后方韧带复合体（PLC）完整性
- 进行神经系统检查，明确神经损伤程度

### 2. 计算TLICS总分
对以下三项分别评分并求和（总分范围通常为1–10分）：
- **损伤机制**：不同机制对应不同分值
- **PLC完整性**：完整（低分） vs 损伤（高分）
- **神经损伤情况**：
  - 无症状 → 低分
  - 根性症状 → 中分
  - 不完全脊髓/马尾损伤 → 高分
  - 完全性损伤 → 最高分

### 3. 制定治疗方案
- **TLICS ≥5分**：建议手术治疗
- **TLICS ≤3分**：建议非手术治疗
- **TLICS =4分**：手术或非手术均可，需结合患者全身状况个体化决策

### 4. 选择手术入路（仅适用于手术患者）
- **无神经症状或仅有根性症状**：无论PLC状态，均推荐**后路手术**
- **不完全性脊髓或马尾损伤**：
  - PLC损伤 → 推荐**前后联合入路**
  - PLC完整 → 推荐**前路手术**
- **完全性脊髓或马尾损伤**：
  - PLC损伤 → 推荐**后路或前后联合入路**
  - PLC完整 → **前路或后路均可**

### 5. 特殊人群处理
- **高龄骨质疏松患者**因轻微外伤所致骨折：优先考虑微创术式（如PVP或PKP），即使TLICS评分较高也需谨慎评估手术指征

> 注意：最终决策必须结合患者年龄、合并症、全身状况及影像学综合判断，不可仅依赖TLICS评分。
