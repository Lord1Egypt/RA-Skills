# 类白血病反应的分类与诊断

## 触发条件
当外周血白细胞总数 >50×10⁹/L 或涂片中出现幼稚粒/淋巴/单核细胞时，应启动本诊断流程。

## 执行步骤

1. **确认原发疾病存在**：检查是否存在感染、恶性肿瘤、中毒、创伤等明确诱因。若有，则支持类白血病反应诊断。

2. **按白细胞类型分型**：
   - **中性粒细胞型**：最常见；白细胞常达 (50~100)×10⁹/L；以分叶核、杆状核为主，可含晚幼粒、中幼粒；早幼粒+原始粒 <10%；伴核左移、中毒性改变；NAP积分显著增高；红细胞、血红蛋白、血小板通常正常。
   - **嗜酸性粒细胞型**：嗜酸性粒细胞 >20%（可达90%），多为成熟型；见于寄生虫、过敏、霍奇金淋巴瘤。
   - **淋巴细胞型**：白细胞 (20~30)×10⁹/L（可 >50×10⁹/L）；以成熟淋巴细胞为主，可见“反应性淋巴细胞”；见于病毒性感染（如传染性单核细胞增多症、百日咳）。
   - **单核细胞型**：单核细胞 >30%，白细胞一般 <50×10⁹/L；见于粟粒性结核、心内膜炎。

3. **识别“白细胞不增多型”**：白细胞总数未超参考值上限，但外周血出现较多该系幼稚细胞（见于结核、败血症、恶性肿瘤等）。

4. **与CML关键鉴别点**：
   - **病因**：类白血病反应有原发病，CML无；
   - **白细胞数**：类白血病反应多 <100×10⁹/L，CML常 >100×10⁹/L；
   - **嗜酸/嗜碱粒细胞**：类白血病反应不增多，CML常增多；
   - **中毒性改变**：类白血病反应明显，CML不明显；
   - **NAP积分**：类白血病反应显著增高，CML显著减低甚至为0；
   - **Ph染色体**：类白血病反应阴性，CML阳性；
   - **骨髓象**：类白血病反应仅粒系增生活跃、核左移；CML骨髓极度增生，粒系 >90%。

5. **输出建议**：若符合类白血病反应特征，报告具体类型并提示寻找原发病；若存在CML可疑特征（如Ph+、NAP↓、嗜碱↑），建议进一步行分子遗传学检查。

## 注意事项
- 本技能不能单独用于确诊白血病，必须结合其他指标。
- 不适用于无白细胞异常的疾病。
