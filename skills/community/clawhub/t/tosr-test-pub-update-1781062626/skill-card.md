## Description: <br>
Automates the ClawHub skill lifecycle test, including creation, metadata retrieval, version update, and deletion. <br>

This skill is for research and development only. <br>

## Use Case: <br>
Developers and release testers use this skill to exercise the ClawHub API lifecycle for a test skill, including publish, inspect, update, and cleanup operations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can create, update, and delete a remote ClawHub skill during lifecycle testing. <br>
Mitigation: Run it only with an appropriate test account or token, and avoid using it as a normal end-user skill. <br>
