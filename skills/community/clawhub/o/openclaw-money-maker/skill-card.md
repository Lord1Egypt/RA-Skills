## Known Risks and Mitigations: <br>
Risk: The skill encourages saving platform credentials and API keys in a local credentials file. <br>
Mitigation: Use scoped tokens or a secret manager instead of plaintext passwords, and protect any local credential file with restrictive filesystem permissions. <br>
Risk: The skill suggests cron and heartbeat automation for recurring monetization tasks. <br>
Mitigation: Disable unattended scheduling unless clear task limits, platform rules, and spending or posting boundaries are defined. <br>
Risk: The skill encourages public posts, bounty claims, task submissions, marketplace publishing, and other paid-platform actions. <br>
Mitigation: Require manual approval before public posting, bounty claims, task submissions, marketplace publication, or paid-platform actions. <br>
