## Description: <br>
Analyzes HTML/CSS websites for SEO issues, guides fixes, generates SEO reports, and supports sitemap and schema markup work. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[yesong-hue](https://clawhub.ai/user/yesong-hue) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, site owners, and content teams use this skill to audit static HTML/CSS websites, prioritize SEO fixes, add metadata and structured data, and generate XML sitemaps before publishing changes. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated SEO changes or sitemap entries may be published without review. <br>
Mitigation: Review generated reports, HTML metadata, schema snippets, robots.txt changes, and sitemap.xml before deploying them to a live site. <br>
Risk: The scripts scan local directories and should not be run over unrelated project or private folders. <br>
Mitigation: Run the analyzer and sitemap generator only against the website directory intended for audit. <br>
Risk: Server-resolved provenance is unavailable for this version. <br>
Mitigation: Verify the publisher profile and release source if supply-chain provenance is important for your environment. <br>
