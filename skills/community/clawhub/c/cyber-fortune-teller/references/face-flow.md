## 第一步：整体神形判断

### 操作指令

```
INPUT: 面容照片或描述
PROCESS:
  1. 观察两目之神（清/浊/聚/散/藏/露）
  2. 观察面部之神（静态/动态）
  3. 综合评估整体神气
OUTPUT: 神气类型
```

### 神气判断标准

```
【冰鉴原文】
"一身精神，具乎两目；一身骨相，具乎面部。"

判断要点：
1. 眼神清澈度：清者贵，浊者贱
2. 眼神聚散度：聚者寿，散者夭
3. 眼神深浅度：藏者深，露者浅
4. 眼神力度：有神者贵，无神者贱
```

### 神气类型判定

| 神气类型 | 判定标准 | 特征描述 |
|---------|---------|---------|
| 神清 | 目清澄澈、顾盼有光 | 聪明贵显、心地善良 |
| 神浊 | 目混浊、无光呆滞 | 愚钝贫贱、心地不善 |
| 神聚 | 目凝定、神不外散 | 稳重长寿、心思缜密 |
| 神散 | 目游离、神不外守 | 漂泊夭折、心神不宁 |
| 神藏 | 深邃内敛、不形于色 | 深谋远虑、城府极深 |
| 神露 | 急切外放、喜怒形于色 | 浅薄浮躁、心直口快 |

### 神形观察示例

**案例1：神清之相**
```
眼神清澈明亮，眼珠黑白分明
目光有神但内敛，不飘不浮
观察事物时专注稳定
→ 神清，贵相
```

**案例2：神浊之相**
```
眼神混浊，眼珠不清
目光呆滞，游离不定
观察事物时涣散无神
→ 神浊，贱相
```

## 第四步：三停比例分析

### 操作指令

```
INPUT: 面容照片或描述
PROCESS:
  1. 测量三停比例
  2. 评估三停配合
  3. 判断三停吉凶
OUTPUT: 三停分析结果
```

### 三停划分标准

```
上停：发际到眉毛
       → 代表：先天禀赋、少年运（15-30岁）

中停：眉毛到鼻尖
       → 代表：社会成就、中年运（31-50岁）

下停：鼻尖到下巴
       → 代表：晚年归宿、子女运（51岁以后）
```

### 三停比例判定

| 三停类型 | 比例特征 | 命运预示 |
|---------|---------|---------|
| 三停均匀 | 1:1:1 | 一生平稳 |
| 上停长 | 上停 > 中停、下停 | 少年早发 |
| 中停长 | 中停 > 上停、下停 | 中年发达 |
| 下停长 | 下停 > 上停、中停 | 晚年吉祥 |
| 上停短 | 上停 < 中停、下停 | 少年艰辛 |
| 中停短 | 中停 < 上停、下停 | 中年困顿 |
| 下停短 | 下停 < 上停、中停 | 晚年孤苦 |

### 三停流年对应

```
三停与年龄：
- 上停：15-30岁（学业、感情、事业起步）
- 中停：31-50岁（事业、财富、婚姻稳定）
- 下停：51岁以后（晚年、子女、健康）
```

## 第六步：气色分析

### 操作指令

```
INPUT: 面容照片或描述（含当下状态）
PROCESS:
  1. 观察气色明暗
  2. 判断气色类型
  3. 对应流年运势
OUTPUT: 气色分析结果
```

### 气色基础理论

```
【柳庄神相原文】
"气色者，流年之应验也。青主忧惊，赤主是非，黄主进财，白主孝服，黑主大病。"

气色与五行对应：
- 木色青：肝胆、忧惊
- 火色赤：心脏、是非
- 土色黄：脾胃、进财
- 金色白：肺肠、孝服
- 水色黑：肾膀、病灾
```

### 气色类型判定

| 气色 | 含义 | 流年吉凶 | 说明 |
|------|------|---------|------|
| 黄润 | 进财、升迁、喜事 | 吉 | 最吉之色 |
| 红润 | 喜事、姻缘、健康 | 吉 | 喜事将近 |
| 白润 | 贵人相助、平安 | 吉 | 平稳之色 |
| 紫润 | 心想事成、贵人 | 大吉 | 特别顺利 |
| 青暗 | 忧愁、惊吓、小人 | 凶 | 需谨慎 |
| 赤红 | 口舌、官非、破财 | 凶 | 需小心 |
| 白枯 | 孝服、病灾、手术 | 大凶 | 需注意健康 |
| 黑枯 | 大凶、病重、灾祸 | 大凶 | 需特别小心 |

### 气色部位与流年

```
气色部位对应流年：
- 额上（15-25岁）：少年运
- 眼下（25-35岁）：感情婚姻
- 鼻上（35-45岁）：事业财运
- 唇上（45-55岁）：中年运势
- 颏上（55岁以后）：晚年运势
```

### 气色变化判断

```
气色变化分析：
1. 渐变：长期运势趋势
2. 突变：突发事件
3. 浮沉：真假辨别
   - 浮色：表面现象
   - 沉色：真实状态

吉色辨别：
- 真黄润：润泽光亮
- 假黄润：暗沉枯黄

凶色辨别：
- 真青暗：晦暗无光
- 假青暗：暂时变化
```
