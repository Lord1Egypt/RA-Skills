# C3 活动运营与执行 — 详细单元规格

## C3-01 市场机会评估

| 字段 | 内容 |
|------|------|
| **输入** | 必选：目标市场/区域；可选：历史活动数据、预算范围 |
| **输出** | 市场机会评估报告（市场潜力/竞争强度/资源匹配度/优先级），Markdown |
| **依赖** | 无 |
| **AI自治度** | 🟨 半自动（需人工确认评估维度和结论） |
| **组合接口** | →C3-02, →C4-33 |

**执行要点**：
1. 基于行业校准确定评估维度
2. 评估目标市场的潜力和竞争强度
3. 匹配自身资源和能力
4. 输出优先级排序
