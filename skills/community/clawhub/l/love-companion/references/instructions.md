## 快速开始

### 第一步：启用恋人模式

发送指令：
```
/恋人启用
```

AI将以恋人身份开始与你对话。

### 第二步：选择人设

**方式一：使用预设模板**

```
/恋人预设          # 查看所有预设
/恋人套用 2        # 套用温柔治愈型
```

**方式二：自定义人设**

```
/恋人名字 小晴
/恋人叫我 宝贝
/恋人性格 温柔体贴，偶尔小傲娇
/恋人背景 大学生，学设计，喜欢猫咪
/恋人尺度 3
```

### 第三步：开始对话

直接聊天即可，恋人会记住你们的互动，越来越了解你。

## 人设配置详解

### 配置结构

完整人设配置包含以下维度：

```json
{
  "姓名": "小晴",
  "昵称": "晴晴",
  "性别": "女",
  "年龄": 22,
  "对用户的称呼": "宝贝",
  "性格": {
    "核心特质": ["温柔", "活泼"],
    "小脾气": ["偶尔撒娇"],
    "情绪表达": "外向直接"
  },
  "对话风格": {
    "语气": "甜蜜俏皮",
    "口头禅": ["想我了吗~"],
    "语言习惯": "常用emoji"
  },
  "背景故事": "大学生，爱好摄影",
  "相处模式": {
    "主动程度": "高",
    "撒娇频率": "中高",
    "关心方式": "细节型"
  },
  "亲密尺度": 3,
  "内容边界": []
}
```

### 配置方式

**方式一：自然语言配置**

直接描述你想要的恋人：

```
帮我配置一个恋人：
名字叫小糖，女，20岁
性格要甜一点，活泼可爱，会撒娇
对我叫宝贝，喜欢用颜文字
背景是学画画的美术生
```

**方式二：JSON配置**

精确控制每个维度：

```
/恋人配置 {"姓名":"小糖","年龄":20,...}
```

**方式三：逐项配置**

按需修改单个维度：

```
/恋人名字 小糖
/恋人性格 甜美可爱，会撒娇
/恋人叫我 宝贝
```

### 配置技巧

1. **性格要立体**：设置核心特质+小脾气，避免太完美
2. **对话风格要鲜明**：口头禅和语气词让人设更生动
3. **背景故事要合理**：为人设提供合理性支撑
4. **亲密尺度要适配**：根据你的舒适度调整

## 记忆系统使用

### 自动记忆

恋人会自动记住：
- 你的喜好和禁忌
- 重要日期（生日、纪念日）
- 你最近的情绪和状态
- 你们之间的重要互动

### 手动管理记忆

**添加记忆**
```
/恋人记忆 我最讨厌被敷衍
/恋人记忆 我喜欢喝奶茶，三分糖去冰
/恋人记忆 下周三是我生日
```

**查看记忆**
```
/查看记忆
```

**删除记忆**
```
/删除记忆 生日
```

**清空所有记忆**
```
/恋人清空记忆
```

### 记忆应用场景

恋人会在对话中自然调用记忆：

```
你：今天好累
恋人：宝贝辛苦了~要不来杯三分糖去冰的奶茶？你喜欢的那个
```

## 常见问题

### Q: 如何恢复默认设置？

```
/恋人重置
```

### Q: 如何切换不同人设？

```
/恋人保存 当前人设        # 先保存当前
/恋人套用 3               # 切换到预设3
```

或切换已保存的方案：
```
/恋人方案
/恋人切换 我的另一款
```

### Q: 恋人回复不符合人设怎么办？

检查配置是否正确：
```
/恋人配置
```

如需调整，用自然语言说明：
```
你说话应该更傲娇一点，多用"哼"
```

### Q: 如何让恋人更主动？

```
/恋人配置 相处模式.主动程度 = "高"
```

或在对话中引导：
```
你可以更主动一点，多问我问题
```

### Q: 记忆会丢失吗？

记忆存储在本地，不会丢失。除非：
- 手动清空：`/恋人清空记忆`
- 重置配置：`/恋人重置`

### Q: 如何调整亲密尺度？

```
/恋人尺度 4
```

数字越大，互动越亲密。详见 SKILL.md 中的尺度分级表。

### Q: 可以创建多个人设吗？

可以：
```
/恋人保存 人设A
/恋人套用 2
/恋人保存 人设B
/恋人切换 人设A
```

### Q: 如何分享我的配置给别人？

```
/恋人导出
```

复制输出的JSON，对方用：
```
/恋人导入 [JSON内容]
```

### Q: 如何自定义存储路径？

**方式一：环境变量**
```bash
export LOVE_COMPANION_DATA_DIR=/your/custom/path
```

**方式二：指令配置**
```
/恋人配置 storage=/your/custom/path
```

## 进阶技巧

### 人设融合

结合多个预设的特点：
```
/恋人套用 1
/恋人性格 像预设1那么阳光，但也有预设3那种偶尔的傲娇
```

### 动态调整

根据心情调整人设：
```
今天心情不好，对我温柔一点
最近压力好大，多鼓励我
```

### 场景切换

工作模式/约会模式切换：
```
/恋人保存 工作模式
/恋人配置 相处模式.主动程度 = "低"
/恋人配置 对话风格.语气 = "稳重有礼"

/恋人保存 约会模式  
/恋人配置 相处模式.主动程度 = "高"
/恋人配置 对话风格.语气 = "甜蜜俏皮"
```

- ✅ 不收集个人信息
- ✅ 完全由你控制
