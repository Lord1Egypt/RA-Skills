## Description: <br>
Analyze a GitHub repository and generate Xiaohongshu (Little Red Book) infographic posts. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[laojun509](https://clawhub.ai/user/laojun509) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and content creators use this skill to turn public GitHub repository information into Xiaohongshu-ready social media draft materials, including project analysis, outline options, image prompts, HTML cards, captions, and a packaged archive. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated social content may omit context or misrepresent a repository if the source README or project metadata is incomplete. <br>
Mitigation: Review and fact-check generated analysis, captions, and visual copy before publishing. <br>
Risk: Unexpected repository URL formats may place output outside the intended per-repository workflow. <br>
Mitigation: Use normal public GitHub repository URLs and inspect the reported output directory before sharing files. <br>
