## 十一、阅读推荐

### 入门读物（初次接触）

| 推荐顺序 | 书目 | 原因 |
|----------|------|------|
| 1 | 《三字经》《弟子规》 | 蒙学入门，了解基础 |
| 2 | 《论语》选读 | 儒家核心，影响深远 |
| 3 | 《诗经》选读 | 文学源头，优美动人 |
| 4 | 《世说新语》 | 趣味性强，易于理解 |
| 5 | 《古文观止》 | 散文精品，文笔典范 |

### 进阶研读（深入学习）

| 类别 | 书目 |
|------|------|
| 儒家 | 《四书章句集注》《传习录》 |
| 道家 | 《道德经》《庄子》 |
| 史学 | 《史记》《资治通鉴》 |
| 文学 | 《文心雕龙》《诗品》 |
| 思想 | 《韩非子》《墨子》 |

### 研究参考（学术深度）

| 类别 | 书目 |
|------|------|
| 经学 | 十三经注疏 |
| 史学 | 二十四史 |
| 子学 | 诸子集成 |
| 文学 | 中国历代文学作品选 |
