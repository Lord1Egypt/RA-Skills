# Core docs / forms map

This file is intentionally mostly structural until FY-linked details are verified.

## Employer / salary lifecycle
- IT declaration
- Form 12BB — official salaried-individual help text describes this as the employee-to-employer form for claims relevant to TDS on salary, including items like HRA, LTC, home-loan interest, and eligible tax-saving claims/investments.
- Form 16 — official salaried-individual help text describes this as the employer-issued salary TDS certificate at year end, carrying salary income, deductions/exemptions, and tax deducted at source for computing tax payable/refundable.
- salary slips / payroll breakup

## Tax data reconciliation
- AIS — official help page describes AIS as a comprehensive taxpayer information view intended to show information before filing, support feedback, and aid prefilling.
- TIS — official AIS help page describes TIS as the category-wise aggregated summary derived from AIS information.
- 26AS — official AIS help page says 26AS shows TDS/TCS-related tax statement data, while AIS covers broader information.

## Return filing
- applicable ITR form(s) for resident salaried users
- supporting documents checklist

## Banking / interest
- Form 15G — official salaried-individual help text describes this as a declaration by an eligible resident taxpayer (not company/firm) to request non-deduction of TDS on certain receipts like interest, subject to conditions such as income being below the basic exemption limit.
- Form 15H — official salaried-individual help text describes this as a declaration by a resident individual aged 60 or above for certain receipts without TDS, again subject to conditions.
- bank interest certificate
- FD / RD statements
- TDS certificates if available

## Investments / capital markets
- broker capital-gains statement
- dividend statement / annual tax statement
- mutual fund CAS / tax P&L if available

## Loan / deduction support
- education loan certificate
- home loan interest/principal certificate
