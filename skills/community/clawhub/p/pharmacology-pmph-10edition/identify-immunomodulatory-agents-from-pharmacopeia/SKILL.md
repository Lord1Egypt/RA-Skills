# 从药典识别免疫增强剂与生物调节剂

## 触发条件
当满足以下任一条件时启用本技能：
- 药物名为“卡介苗”或“bacillus Calmette-Guérin vaccine”且出现在第467页；
- 药物出现在第467–469页，且名称包含“转移因子”、“胸腺素”、“匹多莫德”、“异丙肌苷”或“泰它西普”。

## 执行步骤
1. **确认药物条目位置**：检查药物是否明确列于参考书第467–469页。
2. **匹配关键词**：识别是否包含以下任一药物名称：
   - 卡介苗（bacillus Calmette-Guérin vaccine）
   - 转移因子（transfer factor）
   - 胸腺素（thymosin）
   - 匹多莫德（pidotimod）
   - 异丙肌苷（inosine pranobex）
   - 泰它西普（telitacicept）
3. **判断作用类型**：
   - 若药物旨在**提升或恢复**整体免疫应答能力（如激活巨噬细胞、NK细胞、T细胞），则归类为**免疫增强剂**；
   - 若药物通过**精准靶向调节**过度或失衡的免疫反应（如抑制BLyS/APRIL通路），则归类为**免疫调节生物制剂**。
4. **验证适用对象**：
   - 免疫增强剂适用于免疫低下人群、反复呼吸道感染患者或慢性病毒感染者；
   - 免疫调节剂（如泰它西普）适用于特定自身免疫病（如系统性红斑狼疮）。
5. **排除不适用情况**：
   - 活动性自身免疫病或高免疫状态患者禁用免疫增强剂；
   - 免疫缺陷患者禁用卡介苗等活疫苗类增强剂；
   - 抗生素或抗病毒药即使有间接免疫影响，若未列于467–469页，则不纳入本类。
6. **输出建议**：根据分类结果，推荐将该药物视为免疫增强策略或免疫调节方案的一部分，并注明其作用机制与临床场景。

> 注意：卡介苗是唯一列入此范围的减毒活疫苗类免疫增强剂，其余多为小分子肽、合成二肽或融合蛋白。
