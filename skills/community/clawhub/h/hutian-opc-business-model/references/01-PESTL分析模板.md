# PESTL分析框架

## 概述

PESTL分析是一种宏观环境分析框架，用于评估影响行业和企业的外部环境因素。

## 分析维度

### P-政治环境（Political）
**关注点**：
- 国家政策导向
- 政府态度与支持力度
- 行业监管趋势
- 产业扶持政策
- 国际贸易政策

**分析问题**：
- 政府对目标行业的态度是支持还是限制？
- 有哪些政策红利可以借助？
- 监管政策走向如何？
- 是否存在政策风险？

### E-经济环境（Economic）
**关注点**：
- 宏观经济走势（GDP、CPI、PPI）
- 行业发展周期
- 目标客户消费能力
- 融资环境与成本
- 通货膨胀与汇率

**分析问题**：
- 宏观经济增长预期如何？
- 目标市场的规模和增速？
- 客户支付能力和意愿？
- 资本市场的活跃度？

### S-社会文化环境（Social）
**关注点**：
- 人口结构与分布
- 消费习惯与偏好
- 价值观念变化
- 教育水平与职业结构
- 城镇化进程

**分析问题**：
- 目标人群的规模和特征？
- 消费习惯有哪些变化趋势？
- 社会接受度和教育成本？
- 文化因素如何影响需求？

### T-技术环境（Technological）
**关注点**：
- 技术成熟度与迭代速度
- 替代技术的发展
- 研发投入与专利布局
- 技术壁垒与门槛
- 数字化转型趋势

**分析问题**：
- 目标技术处于什么发展阶段？
- 存在哪些替代技术或颠覆性风险？
- 技术壁垒有多高？
- 研发投入是否可持续？

### L-法律环境（Legal）
**关注点**：
- 行业相关法律法规
- 资质许可要求
- 知识产权保护
- 行业标准与规范
- 合规成本

**分析问题**：
- 进入行业需要哪些资质？
- 知识产权保护是否完善？
- 行业标准是什么？
- 合规成本有多高？

## 评估标准

### 影响评估
| 评估 | 含义 |
|------|------|
| 正面 | 有利于商业模式发展 |
| 负面 | 不利于商业模式发展 |
| 中性 | 影响不明显或双向 |

### 影响程度
| 程度 | 含义 |
|------|------|
| 高 | 对商业模式有决定性影响 |
| 中 | 对商业模式有显著影响 |
| 低 | 对商业模式影响有限 |

## 输出格式

### PESTL分析表
```
| 维度 | 关键发现 | 影响评估 | 影响程度 |
|------|----------|----------|----------|
| P | | | |
| E | | | |
| S | | | |
| T | | | |
| L | | | |
```

### 维度详情模板
```
## P-政治环境

### 政策支持
- 利好政策：[列出相关政策]
- 政策影响：[分析影响]

### 政府态度
- 态度倾向：[支持/中立/限制]
- 政府采购：[是否有相关机会]

### 监管趋势
- 监管方向：[趋严/放松/稳定]
- 合规要求：[具体要求]

### 综合评估
- 影响评估：[正面/负面/中性]
- 影响程度：[高/中/低]
- 关键风险：[如有]
- 应对建议：[如有]
```

## 商业计划书提取指南

### 政治环境提取关键词
- 政策、扶持、补贴、政府、监管、法规、许可、资质、标准

### 经济环境提取关键词
- 市场、规模、增长、投资、融资、收入、利润、成本、定价

### 社会环境提取关键词
- 用户、客户、需求、消费、习惯、年龄、收入、偏好

### 技术环境提取关键词
- 技术、专利、研发、创新、壁垒、优势、成熟、迭代

### 法律环境提取关键词
- 合规、许可、资质、知识产权、商标、专利、标准、规范

## 注意事项

1. **数据来源**：所有分析必须有据可查，引用原文
2. **时效性**：注意政策的最新变化
3. **地域性**：不同地区的政策可能不同
4. **行业差异**：不同行业的PESTL关注点不同
5. **不编造**：无法提取的信息如实标注"未提取到"
