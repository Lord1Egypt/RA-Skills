# 血细胞发育过程中形态演变的一般规律

## 适用条件
- 已制备Wright或Giemsa染色的骨髓涂片
- 在光学显微镜下观察造血系统细胞（不适用于退化细胞或非造血系统细胞如网状细胞、内皮细胞等）

## 执行步骤

### 1. 观察细胞体积变化
- 大多数血细胞：胞体由大变小
- 例外：巨核细胞由小变大；早幼粒细胞较原始粒细胞稍大
- 形态同步变化：巨核细胞、单核细胞由圆形/椭圆形变为不规则形

### 2. 观察细胞质变化
- **量**：由少逐渐增多（淋巴细胞变化不大）
- **染色**：由深蓝色 → 浅染 → 淡红色；红细胞系最终呈橙红色
- **颗粒**：
  - 原始细胞：无颗粒
  - 早幼粒细胞：出现嗜天青颗粒（非特异性）
  - 中幼及以后：出现特异性颗粒（中性、嗜酸性、嗜碱性）
  - 例外：幼红细胞始终无颗粒；淋巴细胞除NK细胞外无颗粒

### 3. 观察细胞核变化
- **大小与形状**：由大变小、规则→不规则→分叶；但巨核细胞核由小变大；红细胞系核变小且规则，成熟后脱核
- **染色质**：由细致疏松 → 粗糙致密 → 凝集成块；着色由浅变深
- **核仁**：由清晰可见 → 模糊 → 消失
- **核膜**：由不明显 → 明显

### 4. 判断核质比（N/C ratio）
- 一般趋势：由大变小（核大质少 → 核小质多）
- 例外：巨核细胞核质比随发育而增大

## 输出结果
根据上述四项指标综合判断细胞类型及其发育阶段。
