# Miki 语言发育评估

**使用方式：** 输入宝宝月龄+会说的词汇+理解能力 → Miki评估发育状态+给出促进方案

---

## 语言发育标准

### 12-15个月
- **词汇量：** 1-10个
- **理解能力：** 理解"不"、简单指令（给我/拿来）、认识家人
- **典型行为：** 叫爸妈（特指）、用声音+手势沟通、尝试模仿新发音
- **预警：** 12个月仍无任何有意义发音

### 15-18个月
- **词汇量：** 10-25个
- **理解能力：** 指认身体部位、理解"不可以"、认识图片
- **典型行为：** 词汇快速增长（每周1-3个新词）、用单词表达完整意思
- **预警：** 18个月词汇<5个、无手指指物

### 18-24个月
- **词汇量：** 50-100个（爆发期）
- **理解能力：** 两步指令、理解大小/颜色概念、听懂简单故事
- **典型行为：** 词汇爆发期、开始2词组合、问"这是什么"
- **预警：** 24个月词汇<30个、无2词组合

### 24-30个月
- **词汇量：** 200-300个
- **理解能力：** 理解空间概念（上面/下面/里面）、认识2种颜色
- **典型行为：** 3-4词短句、开始用"我/你/他"、提问"为什么"
- **预警：** 30个月仍无3词句

## 输入模板

```
【Miki语言评估】

月龄：X个月

会说的词汇：（尽量列出所有）
如：妈妈、爸爸、奶奶、饭饭、车车、狗狗、拿、抱...

理解能力表现：
- 听到名字有反应？
- 能指认身体部位？
- 能听懂简单指令？
- 会用手指点东西？
- 会用哭以外方式表达需求？

沟通行为：
- 会主动叫人？
- 想吃/要抱/想要什么怎么表达？
- 会摇头/点头表示拒绝/要？

主要担忧：（自由描述）
```

## 发育迟缓警示

| 月龄 | 警示信号 |
|------|---------|
| 12个月 | 不会用声音/手势沟通 |
| 15个月 | 不会手指指物，不叫爸妈 |
| 18个月 | 词汇<5个，不理解简单指令 |
| 24个月 | 词汇<30个，无2词组合 |
| 30个月 | 词汇<50个，无3词短句 |

**发现预警 → 建议咨询儿童保健科或语言发育门诊**
