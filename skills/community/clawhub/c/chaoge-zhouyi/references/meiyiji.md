# 梅花易数核心要诀

> 邵雍（康节）创立，以"体用"为核心，结合时间、方位、象数进行占卜
> 用于：模拟起卦、体用分析、应期判断

---

## 一、起卦法（核心六种）

### 1. 时间起卦法（最常用）

以**年＋月＋日**之和除以8取上卦，**年＋月＋日＋时**之和除以8取下卦：
- 年用**地支序数**（子1 丑2 寅3 卯4 辰5 巳6 午7 未8 申9 酉10 戌11 亥12）
- 月、日直接取**农历**数字
- 上卦：(年+月+日)÷8取余数（余0作8）→ 对应**先天八卦数**（乾1兑2离3震4巽5坎6艮7坤8）
- 下卦：(年+月+日+时辰数)÷8取余数（余0作8）→ 对应先天八卦数
- 动爻：(年+月+日+时辰数)÷6取余数（余0作6）→ 动爻

**⚠️ 注意**：起卦只用先天八卦数，与后天方位数完全不同，切勿混淆！

**时辰数**：子1 丑2 寅3 卯4 辰5 巳6 午7 未8 申9 酉10 戌11 亥12

**示例**：甲子年正月初七辰时
- 年支1+月1+日7=9 → 上卦：9÷8余**1**→ 乾
- 9+时辰5=14 → 下卦：14÷8余**6**→ 坎
- 动爻：14÷6余**2**→ 第2爻

### 2. 数字起卦法

报两个数（或多个数）：
- 第一个数÷8余数 → 上卦
- 第二个数÷8余数 → 下卦
- 两数之和÷6余数 → 动爻

### 3. 方位起卦法

以**来人方向＋问事时间**起卦：
- 方位用先天卦数（乾1兑2离3震4巽5坎6艮7坤8）对应方位
- 时间取**农历时刻数**
- 余数起卦法同上

> ⚠️ 方位起卦中，乾=西北，兑=西，离=南，震=东，巽=东南，坎=北，艮=东北，坤=西南。数字与先天数一一对应，而非方位序数。

### 4. 尺寸起卦法

以**尺寸数**起卦：
- 尺寸数÷8余数 → 上卦
- 尺寸数÷6余数 → 动爻
- 或以寸为上卦、尺为下卦

### 5. 物象起卦法

见物即卦：
- 动物 → 先天卦数（乾1兑2离3震4巽5坎6艮7坤8）
- 静物 → 按方位所属卦
- 以见到时的**状态和时辰**定动爻

### 6. 字画起卦法

字以笔画起卦：
- 奇数笔画 → 单卦（字÷8余数）
- 偶数笔画 → 重卦（前半÷8余上卦，后半÷8余下卦，笔画和÷6余动爻）

## 二、体用分析法（核心）

### 体用定义

- **体**（我方/主卦）：占卜者自身、问事主体、现状
- **用**（对方/变卦）：所占之事、对方、变化趋势

## 三、互卦（中间过程）

### 互卦定义

互卦是从主卦中提取的隐藏卦象，代表事物发展过程中的**中间状态、隐藏因素、内在趋势**。

- 互卦与体用分析同样重要
- 互卦揭示主卦与变卦之间的过渡
- 互卦生体则过程顺利，互卦克体则过程有阻

### 互卦取法

六画卦中，取**二三四爻**为下卦，**三四五爻**为上卦，合成互卦。

```
主卦六爻：  上爻
            五爻 ←┐
            四爻 ←┤→ 互卦上卦（三四五爻）
            三爻 ←┤
            二爻 ←┘→ 互卦下卦（二三四爻）
            初爻
```

**示例**：地水师䷆（上坤☷下坎☵）
- 六爻排布：上阴、五阴、四阴、三阴、二阳、初阴
- 互卦上卦 = 三四五爻 = 阴阴阴 = 坤☷
- 互卦下卦 = 二三四爻 = 阳阴阴 = 震☳
- 互卦 = 地雷复䷗

### 互卦在梅花易数中的角色

| 维度 | 主卦 | 互卦 | 变卦 |
|------|------|------|------|
| 时间 | 起始/现状 | 过程/中间 | 结果/终局 |
| 含义 | 事情开端 | 隐藏因素 | 变化结局 |
| 占比 | 判定根本 | 参考辅助 | 判定归趋 |

### 互卦断法

1. 互卦生体 → 过程有人助，暗中得力
2. 互卦克体 → 过程有阻碍，须防暗算
3. 互卦比和 → 过程平稳，无大波澜
4. 体克互卦 → 过程可掌控，虽有波折能化解
5. 体生互卦 → 过程耗心力，需付出始得通

## 四、核心口诀

### 1. 卦象吉凶总诀

```
阳卦：乾震坎艮（奇数爻为阳）
阴卦：坤巽离兑（奇数爻为阴）
主卦旺相吉，衰败凶
体克用为吉，用克体为凶
体生用泄气凶，用生体得助吉
比和相助，诸事顺遂
```

### 2. 动静口诀

```
一动为用，二动为变，三动为悔
静卦看体用生克，旺衰定论
动爻变化，体用随之转移
卦变生克，先天后天有别
```

### 3. 旺相休囚死（得令为旺）

```
旺：与月令相同五行
相：月令所生之五行（如木月火为相）
休：生月令之五行（如木月水为休）
囚：克月令之五行（如木月金为囚）
死：月令所克之五行（如木月土为死）
```

### 4. 先天数（起卦用）

```
乾1 兑2 离3 震4 巽5 坎6 艮7 坤8
```

### 5. 洛书数/后天方位数（辅助参考，非起卦用）

> ⚠️ 这些是洛书九宫数，**起卦只用先天数**，不要与先天八卦数混淆

```
坎1  坤2  震3  巽4  中5  乾6  兑7  艮8  离9
```

## 四、应期判断

### 时间应期

- 动爻属**乾** → 乾卦数9或1，或一、六期
- 动爻属**坤** → 坤卦数10或2，或二、八期
- 动爻属**震** → 震卦数4，或三、七期
- 动爻属**巽** → 巽卦数5，或三、八期
- 动爻属**坎** → 坎卦数6，或一、六期
- 动爻属**离** → 离卦数3，或二、七期
- 动爻属**艮** → 艮卦数8，或七、八期
- 动爻属**兑** → 兑卦数7，或四、九期

### 应期远近

- 卦气盛旺 → 应期近（近日）
- 卦气衰弱 → 应期远（迟久）
- 动爻旺相 → 速应
- 动爻休囚 → 缓应
- 体用比和 → 近期；体用克泄 → 远期

### 断语模式

```
近事当日起，
远事数十日，
大事数月年。
```

## 五、趋避法则

### 一般原则

1. **宜静守** → 体克用吉，体生用凶
2. **宜行动** → 用克体凶，用生体吉
3. **宜速** → 动爻旺相，卦气盛
4. **宜缓** → 动爻休囚，卦气衰
5. **宜改** → 遇凶卦，当变通以化之

### 变通之法

- 遇凶卦 → 静心敛神，转气场
- 遇阻滞 → 调整方位、时间
- 遇分散 → 聚合收束
- 遇刑冲 → 化解调和

### 方位调整

根据体卦所在方位，结合用卦五行生克，选择有利方位出行或回避不利方位。

## 六、常见事项断法

| 事项 | 体用关系 | 吉凶判断 |
|------|---------|---------|
| 求财 | 体生用→耗财，用生体→得财，体克用→得财，用克体→破财 | 财爻旺相吉 |
| 出行 | 用卦为所往，体用比和吉，忌用克体 | 宜生旺忌死绝 |
| 婚姻 | 男得震巽离兑吉，女得乾坤艮兑吉，体用比和婚成 | 比和为上 |
| 疾病 | 体卦旺吉，用卦旺凶，体用比和吉，用克体凶 | 医药有效否 |
| 诉讼 | 体克用吉，用克体凶，体用比和平局 | 宜和解不宜争 |
| 考试 | 离卦为文，离旺则吉，坎衰则困 | 文昌旺则利 |
