# 📊 Amazon Push Score 技能

亚马逊推送分计算与流量池分级系统 — 基于A9算法的4维权重量化评估工具

## 快速开始

只需输入4个指标，系统自动生成推送分、流量池等级和优化策略：

```
CTR(%) + CVR(%) + 7天增速 + 账号健康分(0-1)
示例：1.2 + 14 + 4 + 0.96
```

## 核心功能

- **推送分计算**：归一化4维指标，自动输出0-100推送权重分
- **流量池分级**：A+/A/B/C四档，对应不同运营策略
- **优化策略生成**：精准定位短板维度，生成可执行建议
- **行业基准参考**：10+品类优秀值对照

## 四维权重体系

| 维度 | 权重 | 说明 |
|------|------|------|
| CTR（点击率） | 20% | 用户匹配度 |
| CVR（转化率） | 40% | 产品信任度（核心） |
| 7天增速 | 20% | 用户认可度 |
| 账号健康分 | 20% | 合规安全度 |

## 流量池等级

| 等级 | 分数 | 流量特征 |
|------|------|---------|
| A+ 黄金流量池 | ≥90 | 优先曝光，首页展示 |
| A 优质流量池 | 75-89 | 稳定加权，Top3展示 |
| B 基础流量池 | 60-74 | 正常分发，需优化 |
| C 低权重池 | <60 | 流量受限，需关注 |

## 技术架构

- **计算引擎**：本地Python算法，4维权重归一化评分
- **策略生成**：基于行业知识库生成优化建议
- **输入方式**：用户手动输入4个指标，本地运算出分
