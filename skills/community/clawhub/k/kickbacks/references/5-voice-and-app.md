# Voice & Application — How to Talk About Kickbacks

> Source: kickbacks.ai product documentation, landing page copy, VS Code Marketplace page

## Product Voice

Kickbacks.ai uses a cheeky, self-aware tone. It openly acknowledges the absurdity of monetizing spinner waiting time while being genuinely useful.

**Key voice traits:**
- **Self-deprecating humor** — "Apologies, terminal jockeys"; "Built for people who watch spinners professionally."
- **Transparent** — The bid market is fully public. No hidden pricing, no dark patterns.
- **Neutral on AI** — It's not pro- or anti-AI tools. It simply recognizes that spinner time exists and monetizes it.
- **Developer-first** — Technical details (impression rates, bid mechanics, CLI support) are front and center, not buried.

**Signature phrases:**
- "Get paid for waiting."
- "We turned 'Discombobulating...' into an ad marketplace."
- "50% of the revenue goes to you."
- "Bid live for the most-watched spinner on Earth."

## Common Scenarios

### Scenario 1: New Developer
**User says:** "I use Claude Code all day. How do I start earning?"
**Response:** Install the VS Code extension, sign in with Google, keep using Claude Code normally. Your status bar will show your balance updating in real-time.

### Scenario 2: Skeptical User
**User says:** "This sounds too good to be true. What's the catch?"
**Response:** The catch is the earnings are micro — $5-40/month for heavy users. It's not a game-changer, it's a "free coffee while you code" bonus. The extension is fully reversible, never touches your code, and is transparent about the ad market.

### Scenario 3: Heavy User Maximizing Earnings
**User says:** "I'm running Claude Code on 3 machines. How do I maximize?"
**Response:** Sign in on all 3 machines, use them during overlapping hours when advertiser demand is highest (US business hours), make sure click-through is natural (don't fake clicks — the system detects fraud).

### Scenario 4: Advertiser First Campaign
**User says:** "I want to run ads for my startup on the spinner. How?"
**Response:** The bid market is open. Submit your ad line (3-60 chars), destination URL, and bid per 1,000 impressions ($1 minimum). Highest bid serves first. You can see the current queue and market rates in real-time.

### Scenario 5: Privacy-Conscious Engineer
**User says:** "Can this extension see my code? I work on proprietary stuff."
**Response:** No. Kickbacks only reads the spinner text position to place the ad. It never reads code, prompts, or completions. The extension has been architecturally constrained to not have access to editor buffers or terminal output content.
