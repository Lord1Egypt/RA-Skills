## Description: <br>
Use when onboarding a repository into Claude Code by creating or improving CLAUDE.md, and optionally identifying useful skills, hooks, and verifier setup. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wimi321](https://clawhub.ai/user/wimi321) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and engineering teams use this skill to analyze a repository and create or improve a practical CLAUDE.md with architecture notes, commands, conventions, and future-agent guidance. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated repository guidance may expose internal commands, paths, or operational details if copied from a private repository. <br>
Mitigation: Review the generated CLAUDE.md before committing or sharing it, especially for private or sensitive repositories. <br>
Risk: Incorrectly inferred build, test, or run commands could mislead future agent sessions. <br>
Mitigation: Verify commands and conventions against repository documentation and tooling before relying on the generated guidance. <br>
