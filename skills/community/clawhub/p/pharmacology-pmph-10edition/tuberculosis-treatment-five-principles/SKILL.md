# 抗结核治疗的五大应用原则

在制定或评估结核病化疗方案时，严格遵循以下五项核心原则，以确保治疗成功、防止耐药并减少复发风险。

## 1. 早期用药
- 一旦确诊为活动性结核病，立即启动抗结核治疗。
- **理由**：早期病灶处于渗出阶段，细菌活跃、药物敏感性高，局部血运丰富，有利于药物渗透和痰菌转阴。
- 避免延误至病灶纤维化、干酪化或空洞形成阶段，此时药物渗透差，疗效显著降低。

## 2. 联合用药
- 至少联合使用两种以上抗结核药物。
- **目的**：增强杀菌效果、延缓耐药性产生、降低单一药物的毒性风险。
- **典型方案**：
  - 轻症：异烟肼 + 利福平（二联）
  - 中重度或复治：四联方案（如加用吡嗪酰胺、乙胺丁醇）

## 3. 适量用药
- 根据患者体重、肝肾功能等个体因素调整剂量。
- **剂量不足** → 组织药物浓度不足 → 易诱发耐药 → 治疗失败。
- **剂量过大** → 增加严重不良反应风险 → 可能导致治疗中断。

## 4. 规律用药
- 每日按时服药，不得随意更改剂量、漏服或自行停药。
- 不规律用药可导致细菌再度繁殖、病情迁延，是治疗失败的重要原因。

## 5. 全程用药
- **疗程参考**：
  - 轻症肺结核：9～12个月
  - 中重度或复杂病例：18～24个月
- 疗程应根据病情动态评估调整，但**严禁过早停药**，否则极易导致复发或耐药。
