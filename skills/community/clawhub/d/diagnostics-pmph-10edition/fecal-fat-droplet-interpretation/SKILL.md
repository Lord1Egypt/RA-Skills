# 粪便脂肪小滴增多临床判读

## 触发条件
当粪便显微镜检查（经油红O或苏丹III染色）显示脂肪小滴数量 ≥6个/高倍视野（HPF）时，启动本判读流程。

## 执行步骤
1. **计数并分级脂肪小滴**：
   - 若 >6个/HPF：判定为“脂肪排泄增多”，提示可能存在消化或吸收障碍。
   - 若 >60个/HPF：诊断为“脂肪泻”（steatorrhea）。

2. **结合其他粪便成分综合评估**：
   - 同时观察淀粉颗粒、肌肉纤维等食物残渣，辅助判断胰腺外分泌功能状态。

3. **排除干扰因素**：
   - 确认患者近期无高脂饮食史，避免假阳性。
   - 建议结合粪便弹性蛋白酶等客观指标进行验证。

4. **列出可能病因供临床鉴别**：
   - 急性或慢性胰腺炎
   - 胰头癌
   - 吸收不良综合征
   - 胆汁淤积性黄疸

## 注意事项
- 仅适用于经染色确认的标本，不适用于未染色标本的粗略估计。
- 判读结果应作为警示信号，需进一步结合临床和其他检查综合判断。
